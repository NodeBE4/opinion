    <em>
     “最后，让我们记住的不是敌人的话，而是盟友的沉默。” — — 马丁路德金
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p">
   在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/3%e3%80%81%e7%9c%9f%e6%ad%a3%e7%9a%84%e8%a1%8c%e5%8a%a8%e5%8f%96%e5%86%b3%e4%ba%8e%e4%bd%a0%e7%9a%84%e7%9b%ae%e6%a0%87%e7%9a%84%e5%8f%8d%e5%ba%94%ef%bc%9a%e6%88%98%e7%95%a5%e6%9f%94%e6%9c%af/" href="https://www.iyouport.org/3%e3%80%81%e7%9c%9f%e6%ad%a3%e7%9a%84%e8%a1%8c%e5%8a%a8%e5%8f%96%e5%86%b3%e4%ba%8e%e4%bd%a0%e7%9a%84%e7%9b%ae%e6%a0%87%e7%9a%84%e5%8f%8d%e5%ba%94%ef%bc%9a%e6%88%98%e7%95%a5%e6%9f%94%e6%9c%af/" rel="noopener noreferrer" target="_blank">
    此前的文章中
   </a>
   我们强调过：在反抗运动的世界里，被捕并不是结束，完全可以是一个更大规模的崭新的开始。
  </p>
  <p class="graf graf--p">
   💡一场聪明的被捕可以让你成就反抗史上最出色的行动，并激发活动家们更多的创造力。
  </p>
  <p class="graf graf--p">
   2008年12月，土地管理局（BLM）拍卖在犹他州国家公园附近的石油和天然气非法开采权。这次拍卖是当时的总统布什送给他在工业界的好朋友的礼物。
  </p>
  <p class="graf graf--p">
   学生活动家 Tim DeChristopher 出发准备干扰这次活动。但是，当他走过大门口时，一名服务员问他是否是来竞标的，他感觉很惊讶。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “是的，是的，我是“，他回答到。然后服务员给了他一个竞标的牌子。
  </p>
  <p class="graf graf--p">
   用 Tim 的话说就是：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “一旦我在那里，我意识到任何形式的言论或破坏行动都不会非常有效。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    但我很快就看到了如何才能对这场活动产生重大影响。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    我花了一点时间来建立勇气，我知道后果会是什么 — 然后我开始参加竞标，并努力抬高价格。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    但我知道我可以做得更多。所以，我开始赢得竞标，并尽可能清楚地破坏它。
   </em>
   ”
  </p>
  <p class="graf graf--p">
   Tim 连续赢了十几个 — — 直到拍卖师意识到出了问题，暂停了程序，Tim 被捕。
  </p>
  <p class="graf graf--p">
   在奥巴马就职后，他的政府对“违规行为”的拍卖进行了调查，联邦法官取消了销售。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Tim 的行动 — — 单枪匹马地拯救了许多珍贵的犹他州荒地免受破坏 — — 是近年来美国历史上最具启发性和最成功的公民不服从行为之一。
   </strong>
  </p>
  <p class="graf graf--p">
   在他的量刑听证会上，Tim
   <a class="markup--anchor markup--p-anchor" data-href="https://www.commondreams.org/views/2011/07/27/i-do-not-want-mercy-i-want-you-join-me" href="https://www.commondreams.org/views/2011/07/27/i-do-not-want-mercy-i-want-you-join-me" rel="noopener noreferrer" target="_blank">
    向主审法官发表讲话
   </a>
   ，解释他的行为。
  </p>
  <p class="graf graf--p">
   他以下面这段话结束发言：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    我希望你[法官]和我一起支持公民挑战政府的权利和责任。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    我希望你和我一起重视这个国家丰富的非暴力公民不服从历史。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    如果你认可这些价值，但是认为我的策略是错误的，你有权判罚我；
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    你可以判我广泛的社区服务工作，这将以不同的方式实践使我对健康和公正的世界的承诺。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    你也可以让我与那些陷入困境的青少年一起工作，就如我在职业生涯的大部分时间都在做的那样…
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    如果你同意，你可以引导这一承诺，但你无法扼杀它。这不会消失。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    在难以想象的威胁的笼罩下，这就是希望的样子；在道德败坏的政府已经卖光其原则的时代，这就是爱国主义的样子；无数生命在线，这就是爱的样子，它只会成长壮大。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    你今天做出的选择是你站在哪一边
   </em>
   。✨✨
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    关键战术：创造性破坏
   </strong>
  </p>
  <p class="graf graf--p">
   Tim 成功地干预了整个程序。他找到了一种有效的方法来确保拍卖无法进行（基本上是当场发明了一种新的创造性破坏），然后毫不妥协地为这一行动辩护。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    关键原理：以聪明的方式被捕
   </strong>
  </p>
  <p class="graf graf--p">
   针对 Tim 的案件恰恰为他提供了一个非常大的宣传平台，以呼吁进一步的公民不服从行动。
  </p>
  <p class="graf graf--p">
   💡
   <strong class="markup--strong markup--p-strong">
    Tim 和他的盟友非常及时且聪明地利用他案件的每一步来攻击允许气候变化发生的资本主义政治制度和经济利益
   </strong>
   。
  </p>
  <p class="graf graf--p">
   他向法庭作出的有力的最终陈述、以及他随后服刑的监狱时间，都是非常清楚的例子。
  </p>
  <p class="graf graf--p">
   即使有机会，Tim 也没有直接站起来发表演讲讨论那些石油寡头，
   <strong class="markup--strong markup--p-strong">
    在权势面前反对的理论基本无效，他很明白这种破坏方式是徒劳的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   相反，他选择做一些看似合规的事情，但最终会造成极大的破坏性：他直接玩弄了竞标过程，直到很明显他无意支付他赢得的所有租约。
  </p>
  <p class="graf graf--p">
   💡然后，他把注意力转向另一个更加分散的受众面上：受到他的榜样启发的活动家们，以及那些因同情而更可能转向支持公民抗议行动的公众。
  </p>
  <p>
   <strong>
    记住 Tim 这句话：“I do not want mercy, I want you to join me（我不想要怜悯，我想要你加入）”
   </strong>
  </p>

  <p class="graf graf--p">
   ⚠️这并不是说鼓励人们被捕，肯定不是：如果你判断自己不能利用被捕成就更大的事业，那么你就需要努力保护自己和队友。
  </p>
  <p class="graf graf--p">
   显然，如果没有足够的盟友跟进行动，Tim 的策略效应就会大打折扣。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    在中国，很多情况下被捕“意味着结束”的原因正是因为缺乏充分的行动策略和盟友跟进。
   </strong>
  </p>
  <p class="graf graf--p">
   想想马丁路德金那句话。
  </p>
  <p class="graf graf--p">
   ⚠️一场运动很难通过直接压倒你的对手阵营而获胜 — — 因为反抗者永远都是少数甚至极少数，从基数上注定势单力薄。
  </p>
  <p class="graf graf--p">
   但是，要知道，
   <strong class="markup--strong markup--p-strong">
    你可以通过从下层转移支持而获胜
   </strong>
   。
  </p>
  <p class="graf graf--p">
   💡使用一系列盟友分析来识别受你的问题影响的社会群体（学生、工人），并找到这些群体，这样你就可以集中精力改变这些群体，以令其更加接近你的立场。
  </p>
  <p class="graf graf--p">
   识别更加特定的利益相关者（例如，不仅仅是学生、而且是公立大学的学生；不仅仅是工人，而且是特定领域的工人阶层）💡可以帮助您确定将不同社会群体拉近您的立场的最有效方式，以赢得您的行动目标。
  </p>
  <p class="graf graf--p">
   在制定行动时，将社会视为特定社区、集团或网络的集合，是有用的，其中一些是
   <strong class="markup--strong markup--p-strong">
    机构
   </strong>
   （工会、教堂、学校），其他一些
   <strong class="markup--strong markup--p-strong">
    不太明显或有不太有凝聚力
   </strong>
   ，如亚文化热衷者或人口统计分组。
  </p>
  <p class="graf graf--p">
   只要你可以越精确地识别利益相关者和受影响的社区，你就能越好地准备说服这些团体或个人更接近你的立场。然后，你可以权衡关注不同联盟的相对成本和收益。
  </p>
  <p class="graf graf--p">
   ⚠️评估你的盟友范围可以帮助你避免一些常见的陷阱。
  </p>
  <p class="graf graf--p">
   例如，一些激进组织只关注他们的最活跃盟友，这些盟友冒着“向大众讲道”的风险 — — 建立其他人难以理解的边缘亚文化。这样也许有效，但是很明显，你忽略了你真正需要说服的人。
  </p>
  <p class="graf graf--p">
   其结果只能是越来越不被理解，在不被理解中加速边缘化，从而失去声音的影响力。
  </p>
  <p class="graf graf--p">
   更常见的是，活动家表现得就好像每个不同意自己立场的人都是一个积极的对手，扮演着“正义少数人的角色”，就好像整个世界都在反对他们一样。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    自己把自己孤立起来了。这就是很多中国的活动家正在做的事
   </strong>
   。
  </p>
  <p class="graf graf--p">
   然而，也有人采取的是“说实话”的方法，通过道德诉求或逻辑论证的力量，他们或许可以以某种方式赢得他们最根深蒂固的积极对手。
  </p>
  <p class="graf graf--p">
   但是，其前提是真的有足够多的人愿意且有耐心去倾听。事实上你完全无法确保这点，大众最擅长的是“排异“。
  </p>
  <p class="graf graf--p">
   ⚠️很显然，上述这三种方法实际上都能保证失败。
  </p>
  <p class="graf graf--p">
   因为它们用错了原理。
  </p>
  <p class="graf graf--p">
   💡原理并不是通过压倒你的积极反对者来赢得运动和竞选活动，而是，
   <strong class="markup--strong markup--p-strong">
    通过将每个群体转移到你自己的频谱范围内 — — 从被动盟友变为主动盟友、从中立变为被动盟友、从被动对手变为中立者 — — 从而增加人们的力量、有利于支持变革和削弱那些反对你的意见
   </strong>
   。
  </p>
  <p class="graf graf--p">
   最经典的例子，在1964年美国，学生非暴力协调委员会（SNCC）是反种族隔离的南方非裔美国民权运动的主要推动者，他们意识到，为了赢得非裔美国人的平等人权和投票权，他们需要与富有同情心的北方白人建立积极的盟友。
  </p>
  <p class="graf graf--p">
   北方有许多学生具有同情心，但是没有进入该运动的切入点。
  </p>
  <p class="graf graf--p">
   ⚠️他们需要的不是接受教育或说服，他们需要被邀请加入斗争。
  </p>
  <p class="graf graf--p">
   或者说，在盟友模式中，他们需要从被动盟友转变为活跃的盟友。
  </p>
  <p class="graf graf--p">
   并且，正是这些白人学生可以帮助扩大白人家庭和朋友的社区，后者没有受到非裔美国南方人的斗争的直接影响。
  </p>
  <p class="graf graf--p">
   随着斗争的升级，这些团体最有可能会从中立转变为被动盟友、甚至是活跃的盟友。
  </p>
  <p class="graf graf--p">
   💡以上这些都是通过细致的调查得出了背景分析 — —
   <strong class="markup--strong markup--p-strong">
    绝非简单的推测
   </strong>
   。
  </p>
  <p class="graf graf--p">
   根据这一分析，SNCC 做出了一项战略决策：通过让有同情心的白人学生参加他们的自由暑期计划，专注于在北方人中建立中立的白人社区。
  </p>
  <p class="graf graf--p">
   大批的学生前往南方协助选民登记，许多人在这个过程中深受激励。
  </p>
  <p class="graf graf--p">
   他们亲眼目睹了私刑、暴力警察的虐待、和愤怒的白人暴徒，这些都是对南方黑人只是试图行使投票权的回应。
  </p>
  <p class="graf graf--p">
   许多人写信将这些故事告诉他们的父母 — — 他们突然与斗争有了个人联系。
  </p>
  <p class="graf graf--p">
   这引发了另一个理想的转变：
   <strong class="markup--strong markup--p-strong">
    他们的家庭成为了被动的盟友，进而带动了他们的工作场所和社交网络
   </strong>
   ；与此同时，学生们在秋季作为积极的盟友返回学校，并开始在自己的校园里进行组织 — — 推动向民权方向的转变。
  </p>
  <p class="graf graf--p">
   其结果是：政治格局发生了深刻的变革。
  </p>
  <p class="graf graf--p">
   💡这一巨大的立场转变并不是自发的：这是一项蓄意的运动战略的一部分，直到今天，还在为很多其他新的公民抵抗运动带来深刻的经验。
  </p>
  <p class="graf graf--p">
   中国活动家的牺牲层出不穷，但每一次都没能引发全社会的轰动和由此带来的行动跟进 — — 以至于当局可以大模大样地进行央视认罪，而不会惧怕民众的奋起。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这才是最大的失败（而非被捕和牺牲本身）。令人惋惜
   </strong>
   。
  </p>
  <p class="graf graf--p">
   请不必声称“民众的奴性”，在从未完整尝试之前您至少没有证据证明一个社会的基因中不存在反抗意识和同理心。
  </p>
  <p class="graf graf--p">
   ⚠️最重要的是意识。
  </p>
  <p class="graf graf--p">
   当中国最著名的诺贝尔和平奖得主去世时，有活动家认为“这是机会”（而非终结），却被其他学者抨击 — — 为什么？因为该学者缺乏这一意识，于是无法理解*死亡并非结束*的反抗运动原理。
  </p>
  <p class="graf graf--p">
   显然，这需要花时间去积累。一场成功的行动肯定不是在社交网络上拉一堆签名这么简单。
  </p>
  <p class="graf graf--p">

  </p>
