  <p class="graf graf--p">
   如今，所有人都以不同的方式使用智能手机。
  </p>
  <p class="graf graf--p">
   我们中的一些人几乎不怎么使用它们，而其他人则经常使用它们。有些人甚至将智能手机当成迷你电脑用，而其他人只用它们进行通信。
  </p>
  <p class="graf graf--p">
   然而，研究人员意识到，通过研究智能手机的加速度计，就可以根据手机的使用情况来推测人们的个性。
  </p>
  <p class="graf graf--p">
   你知道你的智能手机都说出了什么吗？也许比你想象中要多得多。
  </p>
  <p class="graf graf--p">
   皇家墨尔本理工学院的研究人员开发了一种方法，可以根据手机的加速度计检查手机的使用情况，从而推测人们的个性。
  </p>
  <p class="graf graf--p">
   ⚠️如果您关注过IYP的“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
    社交工程
   </a>
   ”栏目就能明白这件事有多危险：
   <strong class="markup--strong markup--p-strong">
    识别一个人的个性特点是发起最有效攻击的入口。
   </strong>
  </p>
  <p class="graf graf--p">
   测试对象填写了一个包含外向、亲和性、责任心、情绪不稳定性和经验开放性五个主要特性的人格调查。
  </p>
  <p class="graf graf--p">
   这些人随后被给予了 Android 手机软件，这些软件偶尔会从他们的手机里收集数据，例如他们何时拨打电话和发短信，以及手机加速度计显示的身体活动状况。
  </p>
  <p class="graf graf--p">
   然后，将两个区域进行匹配，参与者的已知个性和智能手机加速度计的数据，以寻找趋势和模式。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们假设可以通过研究参与者的电话活动和身体活动强度来推测人类个性，”该研究报告称。
  </p>
  <p class="graf graf--p">
   这些研究人员能够确定的是，平日晚上一直呈现运动状态的人更容易内向，而外向者的电话使用模式看起来更随机。
  </p>
  <p class="graf graf--p">
   您是否将自己视为一个容易相处的人并且更容易保持乐观？这种特性在电话使用中就会看起来更随机，虽然也表明使用者在周末和工作日的晚上比较忙。
  </p>
  <p class="graf graf--p">
   尽职尽责、井井有条的人往往不会经常在很短的时间内联系同一个人，好像他们的思想组织得足够好，根本不需要发送多个文本。
  </p>
  <p class="graf graf--p">
   研究人员能够对这些信息做些什么？
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “社交媒体中就有这种技术的应用，包括朋友推荐、在线约会和有针对性的广告，但我认为最令人兴奋的部分是我们可以了解自己的内容”，首席研究作者和博士生 Nan Gao 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们的许多习惯和行为都是无意识的，但经过分析后，这些数据就会告诉我们很多关于我们自己的真实情况，”她说。
  </p>
  <p class="graf graf--p">
   当然，对于任何知晓 Facebook / 剑桥分析公司丑闻的人来说，这可能都不是新闻。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这就是这项研究带来危险的地方。
   </strong>
  </p>
  <p class="graf graf--p">
   也许这项研究通过电话使用了解个性，可以确定一些很棒的事，就像研究者建议的那样；但是，我只能看到各种各样的恶意行为者正在开发收集这些数据的应用程序，并将其通过恶意软件添加到智能手机，收集我们的个性数据，为的是以某种方式伤害我们。
  </p>
  <p class="graf graf--p">
   此前报道过，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%ad%e5%9b%bd%e8%bf%ab%e4%bd%bf%e6%b8%b8%e5%ae%a2%e5%ae%89%e8%a3%85%e7%9b%91%e8%a7%86%e6%81%b6%e6%84%8f%e8%bd%af%e4%bb%b6/" href="https://www.iyouport.org/%e4%b8%ad%e5%9b%bd%e8%bf%ab%e4%bd%bf%e6%b8%b8%e5%ae%a2%e5%ae%89%e8%a3%85%e7%9b%91%e8%a7%86%e6%81%b6%e6%84%8f%e8%bd%af%e4%bb%b6/" rel="noopener noreferrer" target="_blank">
    中国边境警卫在一些游客的手机上安装监控应用程序
   </a>
   ，作为政府不断增强的对新疆大规模监控的一部分。这表明追踪智能手机活动是一个日益严重的隐私问题。
  </p>
  <p class="graf graf--p">
   在欧洲，监管机构最近宣布对亚马逊使用敏感消费者数据进行反垄断调查。与此同时，谷歌正在面对14.9亿欧元的罚款，因涉嫌参与违反欧盟反垄断规定的有针对性的广告行为。⚪️
