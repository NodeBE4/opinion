  <p class="graf graf--p">
   你肯定有权力；而且通过巧妙地使用，它可以发挥很好的作用。
  </p>
  <p class="graf graf--p">
   本文推荐一个小工具。
  </p>
  <p class="graf graf--p">
   自20世纪80年代以来该工具一直在被行动主义广泛使用。
  </p>
  <p class="graf graf--p">
   💡
   <strong class="markup--strong markup--p-strong">
    要赢得一场运动，您需要正确地识别谁有权解决您想要解决的问题
   </strong>
   。
  </p>
  <p class="graf graf--p">
   然后你需要针对性地给他们施加压力，以令其做出正确的决定。
  </p>
  <p class="graf graf--p">
   绘制权力地图是一种工具，不仅可以识别谁拥有这种权力，而且至关重要的是，谁拥有对该人的影响力，即 你的直接行动的矛头指向的那个人。
  </p>
  <p class="graf graf--p">
   正确完成的权力地图可以揭示这些权力关系和动态，并帮助您为运动设计制胜战略。（参见 原理4:
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%8d%e8%a6%81%e8%ae%a9%e4%bd%a0%e7%9a%84%e5%af%b9%e6%89%8b%e5%ae%9a%e4%b9%89%e4%bd%a0%ef%bc%88%e4%ba%8c%ef%bc%89/" href="https://www.iyouport.org/%e4%b8%8d%e8%a6%81%e8%ae%a9%e4%bd%a0%e7%9a%84%e5%af%b9%e6%89%8b%e5%ae%9a%e4%b9%89%e4%bd%a0%ef%bc%88%e4%ba%8c%ef%bc%89/" rel="noopener noreferrer" target="_blank">
    明智地选择你的目标
   </a>
   ）
  </p>
  <p class="graf graf--p">
   假设一家矿业公司正试图从您所在社区周围的土地中提取矿产。
  </p>
  <p class="graf graf--p">
   结果是 土地受到污染，严重影响了您的家人和邻居的健康。
  </p>
  <p class="graf graf--p">
   那么，你的目标是谁？
  </p>
  <p class="graf graf--p">
   该公司？如果是这样，您是否针对该公司所在区域主管或国际首席执行官？
  </p>
  <p class="graf graf--p">
   谁最终有权关闭矿井？
  </p>
  <p class="graf graf--p">
   你可以利用什么样的力量让他们做到这一点？
  </p>
  <p class="graf graf--p">
   制作整个情况的权力地图可以帮助您回答所有这些问题。
  </p>
  <p class="graf graf--p">
   事实上，⚠️这张图可能会告诉您，您不应该以公司为目标，因为您无法对它们建立足够的直接影响力。
  </p>
  <p class="graf graf--p">
   相反，权力图可能表明你应该针对当地政府。
  </p>
  <p class="graf graf--p">
   但是，到底针对谁呢？从当地市长到国家领导人的每个人在这种情况下都有一定程度的权力，以及彼此不同程度的影响力。
  </p>
  <p class="graf graf--p">
   💡权力图可以帮助阐明这些关系并提出最佳的进攻方向。
  </p>
  <p class="graf graf--p">
   在开始行动之前制作权力图至关重要。
  </p>
  <p class="graf graf--p">
   ⚠️如果瞄准错误的目标可能会损害您的行动力和资源！
  </p>
  <p class="graf graf--p">
   所以，在开始之前确保您走在正确的轨道上这至关重要。
  </p>
  <p class="graf graf--p">
   你可能会发现，即使在正确识别你的目标并对其进行一段时间的压力行动之后，你也无法动员足够的力量直接压迫他们解决问题。
  </p>
  <p class="graf graf--p">
   那时你需要集中精力对权力图中呈现的“次要目标”施加压力 —— 即：能够影响你的主要目标的权力持有者。
  </p>
  <p class="graf graf--p">
   如果让他们感到足够的压力，他们就有可能策动主要目标来满足您的需求。
  </p>
  <p class="graf graf--p">
   💡权力图可帮助您绘制主要目标与所涉及的所有其他利益相关方（包括您自己）之间的所有影响力关系。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    —— 如何使用该工具 ——
   </strong>
  </p>
  <p class="graf graf--p">
   第1步：确定您要解决的问题。
  </p>
  <p class="graf graf--p">
   扩展您的情报，直到您对问题和所涉及的权力关系有广泛的了解。
  </p>
  <p class="graf graf--p">
   第2步：确定主要利益相关者。
  </p>
  <p class="graf graf--p">
   这些利益相关者通常包括：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    那些制造问题的人;
   </li>
   <li class="graf graf--li">
    那些有能力解决问题但却没有这样做的人;
   </li>
   <li class="graf graf--li">
    与该问题的所在地相关的人;
   </li>
   <li class="graf graf--li">
    那些正在努力解决问题的人;
   </li>
   <li class="graf graf--li">
    不要忘了，还包括你和你的团队。
   </li>
  </ul>
  <p class="graf graf--p">
   您最终会得到一系列与您的问题相关的机构（正式和非正式的）、组织、有影响力的人物、媒体和各种各样的人。
  </p>
  <p class="graf graf--p">
   请保持这个清单清晰可用。
  </p>
  <p class="graf graf--p">
   第3步：研究利益相关者。
  </p>
  <p class="graf graf--p">
   你的名单上有一些机构，但是
   <strong class="markup--strong markup--p-strong">
    机构不做决定，是人在做
   </strong>
   。
  </p>
  <p class="graf graf--p">
   所以，你必须找出谁在这些机构做出决定。那就是你的目标人。
  </p>
  <p class="graf graf--p">
   然后，你必须尝试回答一些关于这些目标人的问题：他们在这个问题上究竟是同意还是不同意你？他们对处理这个问题有多大的权力？
  </p>
  <p class="graf graf--p">
   画出下面这张图：
  </p>
  <p class="graf graf--p">

   </noscript>
  </p>
  <p class="graf graf--p">
   第4步：绘制所有利益相关者的立场。
  </p>
  <p class="graf graf--p">
   可以直接在权力图上绘制。
  </p>
  <p class="graf graf--p">
   将每个利益相关者放在
   <strong class="markup--strong markup--p-strong">
    正确的位置
   </strong>
   上。然后，根据他们的支持程度，以及他们拥有的对问题本身的权力，您可以直观地看清楚应该如何行动。
  </p>
  <p class="graf graf--p">
   第5步：确定您的主要目标。
  </p>
  <p class="graf graf--p">
   当你弄清楚谁对这个问题影响最大、谁最有可能给你你想要的东西时，你的目标就突出出来了。
  </p>
  <p class="graf graf--p">
   完美的目标既强大又具有支持性（或者至少可以轻松说服其转向支持你）。显然，这种情况在现实生活中很少有。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    最难以推动的目标、正是你经常需要面对并被迫接受的那种，就是一个拥有很多权力的人，并且强烈反对你
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果您无法决定单个主要目标，但只能专注于一个主要目标，请选择权力图中最有可能的人。
  </p>
  <p class="graf graf--p">
