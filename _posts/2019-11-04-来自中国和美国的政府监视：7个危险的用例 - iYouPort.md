  <p class="graf graf--p">
   毫无疑问，“人工智能政府监控”的谷歌搜索结果中有很多涉及中国。
  </p>
  <p class="graf graf--p">
   人们的共识是，中国不仅在人工智能研究中追赶或超越美国，而且，对于使用人工智能令政府实现对自己的人民无限监视的权力方面是非常开放的。
  </p>
  <p class="graf graf--p">
   虽然它可能是地球上最具侵略性的监视状态 ——
   <strong class="markup--strong markup--p-strong">
    但中国并不是唯一一个将 AI 用于监视目的的国家
   </strong>
   —— 本文将探索和对比中国和美国监控技术新的和独特的用途、以及人工智能在全球的间谍应用。
  </p>
  <p class="graf graf--p">
   美国在政府监视方面有着悠久的历史，从1928年开始就是窃听大国。
  </p>
  <p class="graf graf--p">
   许多人还记得2013年国家安全局举报人 Edward Snowden 的惊人揭露。其实在监视的历史上那不是新鲜事。
  </p>
  <p class="graf graf--p">
   推荐一本书《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e9%97%b4%e8%b0%8d%e3%80%81%e5%92%8c%e5%8c%bf%e5%90%8d%e7%9c%9f%e7%9b%b8/" href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e9%97%b4%e8%b0%8d%e3%80%81%e5%92%8c%e5%8c%bf%e5%90%8d%e7%9c%9f%e7%9b%b8/" rel="noopener noreferrer" target="_blank">
    美国间谍、和匿名真相
   </a>
   》。
  </p>
  <p class="graf graf--p">
   此外在这里看到完整版揭露文件《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" rel="noopener noreferrer" target="_blank">
    互联网霸主的时间线
   </a>
   》；以及为什么揭露并没能阻止监视的继续（并且部分还升级了）《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/6%e5%b9%b4%ef%bc%8c%e6%94%b9%e5%8f%98%e4%ba%86%e4%bb%80%e4%b9%88%ef%bc%9f/" href="https://www.iyouport.org/6%e5%b9%b4%ef%bc%8c%e6%94%b9%e5%8f%98%e4%ba%86%e4%bb%80%e4%b9%88%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    6年，改变了什么？
   </a>
   》
  </p>
  <p class="graf graf--p">
   在俄罗斯、英国和澳大利亚以及阿联酋，政府监视的威胁也越来越受到关注。
  </p>
  <p class="graf graf--p">
   与大多数技术一样，人工智能同时具有优点和缺点，并且如果能通过正确的应用，可以证明其超过成本的效益。
  </p>
  <p class="graf graf--p">
   在这里看到一本书的介绍，关于为什么平衡监视和安全非常难《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E5%9C%A8%E6%AD%A3%E5%8F%8D%E4%B9%8C%E6%89%98%E9%82%A6%E4%B9%8B%E9%97%B4/" href="https://www.iyouport.org/%E5%9C%A8%E6%AD%A3%E5%8F%8D%E4%B9%8C%E6%89%98%E9%82%A6%E4%B9%8B%E9%97%B4/" rel="noopener noreferrer" target="_blank">
    在正反乌托邦之间
   </a>
   》
  </p>
  <p class="graf graf--p">
   然而，人工智能技术通常难以规范，因此误用和滥用是一个真正的问题。
  </p>
  <p class="graf graf--p">
   视频监控是基于人工智能的政府监控的最普遍和众所周知的用途。
  </p>
  <p class="graf graf--p">
   无论是通过闭路电视摄像机（CCTV）还是无人机，如今每个人都知道，几乎在任何时候都有一只眼睛在看着你，特别是在公共场所和机场等区域。
  </p>
  <p class="graf graf--p">
   今天的监控技术有很多种形式 —— 远远超出商场或机场的固定摄像头。
  </p>
  <p class="graf graf--p">
   本文将探索
   <strong class="markup--strong markup--p-strong">
    全球普遍存在的
   </strong>
   七个特定的、独特的AI监控用例：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    机器鸟
   </li>
   <li class="graf graf--li">
    智能眼镜
   </li>
   <li class="graf graf--li">
    支持AI的随身摄像头
   </li>
   <li class="graf graf--li">
    EEG头带
   </li>
   <li class="graf graf--li">
    带监视器的制服
   </li>
   <li class="graf graf--li">
    腕带
   </li>
   <li class="graf graf--li">
    社会信用评分
   </li>
  </ul>
  <p class="graf graf--p">
   首先介绍的是本列表中一个相比下更独特的用例：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    机器鸟
   </strong>
  </p>
  <p class="graf graf--p">
   虽然人们广泛地讨论了机器人和无人机作为秘密民事监视工具的问题，但在这件事上，机器鸟是一个纯粹的小聪明。
  </p>
  <p class="graf graf--p">
   一些看起来像鸟、移动状态也像鸟的东西，但是这种鸟的工作就是盯着所有人。
  </p>
  <p class="graf graf--p">
   鸟可以无处不在，大多数人都没有给予它们太多的关注。这一事实引起了人们对使用鸟形监视器的兴趣。
  </p>
  <p class="graf graf--p">
   如今说起来，这不是新鲜事。
  </p>
  <p class="graf graf--p">
   自
   <strong class="markup--strong markup--p-strong">
    2011年推出
   </strong>
   以来，机器鸟已经能够自主并在空中停留超过几分钟，其中
   <strong class="markup--strong markup--p-strong">
    Festo 公司的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.festo.com/group/en/cms/10238.htm" href="https://www.festo.com/group/en/cms/10238.htm" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Smartbird
    </strong>
   </a>
   看起来可以像鲱鱼一样灵活移动。
  </p>
  <p class="graf graf--p">
   中国西安工业大学的顶尖科学家团队恰当地称其为鸽子项目，由
   <a class="markup--anchor markup--p-anchor" data-href="https://biography.omicsonline.org/china/northwest-polytechnic-institute/song-bifeng-869010" href="https://biography.omicsonline.org/china/northwest-polytechnic-institute/song-bifeng-869010" rel="noopener noreferrer" target="_blank">
    Song Bifeng
   </a>
   领导，开发了这种鸟形的无人机，可在中国的多个地区部署，特别是新疆地区。虽然该计划仍处于早期阶段，
  </p>
  <p class="graf graf--p">
   但该团队
   <a class="markup--anchor markup--p-anchor" data-href="https://www.scmp.com/news/china/society/article/2152027/china-takes-surveillance-new-heights-flock-robotic-doves-do-they" href="https://www.scmp.com/news/china/society/article/2152027/china-takes-surveillance-new-heights-flock-robotic-doves-do-they" rel="noopener noreferrer" target="_blank">
    认为
   </a>
   ：“这种监视在满足军用和民用领域对无人机的需求方面具有一些独特的优势。” 特别是这些无人机的能力：
   <strong class="markup--strong markup--p-strong">
    甚至通过雷达也很难发现
   </strong>
   。
  </p>
  <p class="graf graf--p">
   以下是解析图：

  <p class="graf graf--p">
   据报道，
   <strong class="markup--strong markup--p-strong">
    美国陆军在2013年就投资了鸟形无人机技术，从
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://www.prioria.com/" href="http://www.prioria.com/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Prioria Robotics
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    购买了名为
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://www.prioria.com/maveric/" href="http://www.prioria.com/maveric/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Maveric
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    的30个微型无人机
   </strong>
   。
  </p>
  <p class="graf graf--p">
   并且，美国的机器鸟比中国“鸽子”更加厚实，一家公司
   <a class="markup--anchor markup--p-anchor" data-href="https://www.vice.com/en_us/article/wnxebn/drone-company-misled-military-into-buying-uavs-that-were-basically-toys-lawsuit" href="https://www.vice.com/en_us/article/wnxebn/drone-company-misled-military-into-buying-uavs-that-were-basically-toys-lawsuit" rel="noopener noreferrer" target="_blank">
    声称
   </a>
   中国那种监视鸟总体上是一个不合格的产品。
  </p>
  <p class="graf graf--p">
   此后美国方面的研发就没有详细消息了。
  </p>
  <p class="graf graf--p">
   最后一次看到这种空中无人机监视是在纽约市新年庆祝活动期间 —— 这场活动被雨水挫败了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    智能眼镜
   </strong>
  </p>
  <p class="graf graf--p">
   政府监视其公民的另一种方式是通过配备面部识别软件的“智能”眼镜来扫描面部，并将其与数据库进行快速匹配。
  </p>
  <p class="graf graf--p">
   这些增强现实（AR）眼镜（例如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.xloong.com/product/techlens_t2" href="https://www.xloong.com/product/techlens_t2" rel="noopener noreferrer" target="_blank">
    AR Boss Smart Glasses Techlens T2

   ）由
   <a class="markup--anchor markup--p-anchor" data-href="https://www.xloong.com/" href="https://www.xloong.com/" rel="noopener noreferrer" target="_blank">
    Xloong
   </a>
   开发，为执法部门提供了一种快速巡视大规模人群的方法。
  </p>
  <p class="graf graf--p">
   随着中国计划建立一个14亿公民的面部识别数据库，这种眼镜绝对是应运而生的。
  </p>

   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%bf%e7%94%a8%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%97%b6%e4%bd%a0%e5%bf%85%e9%a1%bb%e6%b3%a8%e6%84%8f%e4%bb%80%e4%b9%88%ef%bc%9f%e2%80%8a/" href="https://www.iyouport.org/%e4%bd%bf%e7%94%a8%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%97%b6%e4%bd%a0%e5%bf%85%e9%a1%bb%e6%b3%a8%e6%84%8f%e4%bb%80%e4%b9%88%ef%bc%9f%e2%80%8a/" rel="noopener noreferrer" target="_blank">
    如何自由地表达而不必担心被捕：使用社交媒体的注意事项
   </a>
   》。
  </p>
  <p class="graf graf--p">
   如此全面地扫描每个人的私密个人信息当然是非常令人恐惧的，尤其是它掌握在专制国家的手中时。
  </p>
  <p class="graf graf--p">
   并且，用于确定排名的算法也完全没有透明度。从一个非常真实的意义上说，政府可以完全控制一个人的社会经济是成功还是失败。
  </p>
  <p class="graf graf--p">
   France 24 从中国公民的角度制作了一个4分钟的视频，探索社会评分系统：
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/NXyzpMDtpSE" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   政府追求让14亿人保持一致虽然表面上看起来很难，但
   <strong class="markup--strong markup--p-strong">
    中国的社会信用分数（加上
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://emerj.com/ai-power/the-usa-china-ai-race/" href="https://emerj.com/ai-power/the-usa-china-ai-race/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     更顺从的儒家文化
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ）
   </strong>
   可能有助于执政党做到这一点。
  </p>
  <p class="graf graf--p">
   包括 Emerj 创始人 Daniel Faggella 在内的一些专家认为，中国的监控技术将基于人工智能的监控提升到了一个新的“正常”水平 —— 很可能会将这些技术的更具侵入性的使用案例传播到西方国家。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    包括英国在内的欧洲国家也正在考虑为其公民实施类似的评分计划
   </strong>
   ，尽管调查结果显示大多数人（78％）或参与者认为这是一个坏主意。
  </p>
  <p class="graf graf--p">
   简单说，现在只能寄希望于文化 —— 中国容易顺从，英国人不容易顺从 —— 而无法寄希望于取消该技术本身。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    关于人工智能进行政府监视的结论
   </strong>
  </p>
  <p class="graf graf--p">
   当权者永远会以所谓的“国家安全”为由要求1984。
  </p>
  <p class="graf graf--p">
   然而，识别安全性和操控之间的细微差别并不总是容易的。
  </p>
  <p class="graf graf--p">
   也就是说，必须有一条线将倡导与偏执真正切分。关键是在任何一种情况下都要达到适当的平衡，具体取决于具体情况。
  </p>
  <p class="graf graf--p">
   英国和中国在许多当前用于政府监视的人工智能应用中占有突出地位，而且往往是负面的。
  </p>
  <p class="graf graf--p">
   西方一些人
   <a class="markup--anchor markup--p-anchor" data-href="https://www.csis.org/analysis/new-china-data-privacy-standard-looks-more-far-reaching-gdpr" href="https://www.csis.org/analysis/new-china-data-privacy-standard-looks-more-far-reaching-gdpr" rel="noopener noreferrer" target="_blank">
    甚至认为
   </a>
   ，当涉及到政府监视和数据保护的人工智能时，中国对人工智能的最佳接触方式的“理解更好”。
  </p>
  <p class="graf graf--p">
   正如丹尼尔·法格拉（Daniel Faggella）所指出的那样，“西方需要一种新战略来维持自身的价值观，并在国际舞台上展开竞争。如果西方希望确保其价值和繁荣继续下去，就必须不能违规”。
  </p>
  <p class="graf graf--p">
   但是，正如文章开头所申明的，
   <strong class="markup--strong markup--p-strong">
    政府监视的最终可能性程度将取决于民间的反抗精神，而非对权势会否善意的期待
   </strong>
   。⚪️
  </p>
