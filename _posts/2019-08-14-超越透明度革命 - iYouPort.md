      </strong>

  <p class="graf graf--p">
   Jesselyn Radack, Thomas Drake, Chelsea Manning, Edward Snowden — — 这些来自权力大厅内部的举报人警告我们所有人关于权力被滥用的方式，就如普罗米修斯的圣火向人们传递被当权者禁止的信息。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    仅仅是举报人能拯救我们吗？仅仅举报就足够了吗？基于举报的社会变革战略中缺少了什么，以及超越透明度革命还需要做些什么？
   </strong>
  </p>
  <p class="graf graf--p">
   这篇文章再次强调了 IYP 在一年多前答读者问中对透明度革命的反思和担忧：中国难以实现透明度革命触发的变革，并非源于缺乏具有勇气智慧和技能的英雄举报人，而是，缺少透明度革命最重要中一个步骤：即 行动主义的智慧和识别机遇的能力。
  </p>
  <p class="graf graf--p">
   当然，举报揭露是有充分理由的而且是极其重要的，只有当内部知情人说出真相时，局外人才能获得清醒。
  </p>
  <p class="graf graf--p">
   然而，
   <strong class="markup--strong markup--p-strong">
    揭露暴君是一回事 — — 废除它们是另一回事
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Snowden 曾经亲口说过：“我最害怕的是……没有任何改变。人们会在媒体上看到所有这些披露，他们将知道政府如何单方面授予自己无限的权力，以便更深入地控制美国社会和全球社会。
   <strong class="markup--strong markup--p-strong">
    但是人们不愿意承担站起来与之战斗所必需的风险
   </strong>
   。“
  </p>
  <p class="graf graf--p">
   透明度革命中隐含的社会变革理论是，政府的罪行一旦被揭露，民众的愤怒将迫使政府做出改变。Snowden 说，“我相信只要美国国家安全局对美国人进行违宪的大规模监视被揭露，它无法在法院、国会和人民的审查中存活下来”。然而，
   <strong class="markup--strong markup--p-strong">
    斯诺登最大的恐惧已经成为现实：限制国家安全局监控计划的改革
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://www.theguardian.com/us-news/2014/nov/18/usa-freedom-act-republicans-block-bill" href="https://www.theguardian.com/us-news/2014/nov/18/usa-freedom-act-republicans-block-bill" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     已被挫败
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   斯诺登和其他透明度革命战士成功地揭穿了政府的阴谋，但并未阻止奥威尔国家的扩张。他们让我们所有人知道统治者是如何侵入性和不负责任的，但他们没有警告我们
   <strong class="markup--strong markup--p-strong">
    必须反击。
   </strong>
  </p>
  <p class="graf graf--p">
   透明度革命是以民主语话为前提的：如果人们足够了解，他们就可以“向权力讲真话”，而这种言论本身也会以某种方式催化变革。 当然，这假定了一种基于对话的政治制度。
  </p>
  <p class="graf graf--p">
   但这是非常理想化的。建立这种监视基础设施的部门 — — 现在试图通过强制力而不是说服性的论证，将斯诺登与切尔西曼宁一起监禁。这足够证明，
   <strong class="markup--strong markup--p-strong">
    仅仅说真话是不够的 — — 我们不是在对话，而是处在权力斗争中。
   </strong>
  </p>
  <p class="graf graf--p">
   同样，将政治家和官僚的幕后阴谋视为一种透明和平等的秩序中的“暂时性失误”是
   <strong class="markup--strong markup--p-strong">
    严重错误
   </strong>
   的。
   <strong class="markup--strong markup--p-strong">
    这些可怕的阴谋并不是过度行为，而是常规行为
   </strong>
   。自20世纪70年代透明度革命的鼎盛时期以来，人们一直在将所有这些真相视为“异常”- 意味着国家本身具有合法性，并且“只是需要改革”。但是，
   <strong class="markup--strong markup--p-strong">
    认为公民可以监管国家是非常天真的。国家越强大，它对公民的威力就越大
   </strong>
   — — 更不用说其他人了。
  </p>
  <p class="graf graf--p">
   将举报人定位为社会变革的主要角色还有其他弊端。这不仅意味着系统从根本上被认为是“合法的”，它还将那些在系统中拥有特权地位的人视为变革的推动者。
  </p>
  <p class="graf graf--p">
   然而，
   <strong class="markup--strong markup--p-strong">
    在大多数情况下，体制内的人是最不可能说真话的人；有成千上万的理由和激励机制可以确保他们不易受到良心危机的影响
   </strong>
   。毫无疑问，曼宁和斯诺登斯是如此的罕见，相对于那些在国家机器运作中勾结的不露面的成千上万人而言。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    问题并不在于人类自然是自私的或麻木的，而是，权力制度的基础设施促进了自私和麻木。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    将人类的未来置于权力大厅内部的人身上是错误的。相反，我们应该问各行各业的人们如何共同努力、以禁用压迫性的基础设施
   </strong>
   。
  </p>
  <p class="graf graf--p">
   像斯诺登这样的系统管理员确实影响了我们所有人的认知，但系统管理员无法自己创造解决方案。把变革的期望寄托在极少数计算机专家身上，只会掩盖无数大众的力量，大众的参与对于任何解放运动来说都至关重要。
  </p>
  <p class="graf graf--p">
   这种疏忽解释了 Julian Assange 和 Jacob Appelbaum
   <a class="markup--anchor markup--p-anchor" data-href="https://www.youtube.com/watch?v=johT3i6XFfc" href="https://www.youtube.com/watch?v=johT3i6XFfc" rel="noopener noreferrer" target="_blank">
    在2013年混沌通信大会上的讲话
   </a>
   中所暗示的失望，当时他们将系统管理员描述为一个应该组织起来捍卫自己利益的阶级，并警告说必须阻止数字专制的不断深化在为时已晚之前。
  </p>
  <p class="graf graf--p">
   事实上，无论互联网上的权力如何巩固，权力机构以外的人都将继续与不公正作斗争 — — 许多人坦白地说别无选择。
   <strong class="markup--strong markup--p-strong">
    越来越多的被边缘化、失业和被压迫者必须成为任何变革战略的中心，与编程阶层的叛逃者一道。
   </strong>
  </p>
  <p class="graf graf--p">
   如果程序员将他们的利益概念化为与其他人不同，并组织起来捍卫这些利益而不是参与比自己更大的斗争，他们将与其他物种一起注定失败。程序员不应该把自己组织成一个阶级 — — 他们应该参与在更大规模的阶层斗争中。
  </p>
  <p class="graf graf--p">
   正如斯诺登所担心的那样，
   <strong class="markup--strong markup--p-strong">
    在没有关于如何对抗它的提议的情况下，对国家监视的揭露只会加剧老大哥本就打算实现的寒蝉效应
   </strong>
   。
  </p>
  <p class="graf graf--p">
   一般的报纸读者在得知美国国家安全局一直在通过自己的智能手机和信用卡追踪自己的整个生命后，不太可能在愤怒中走上街头，而是变得更加谨慎和顺从。然而，
   <strong class="markup--strong markup--p-strong">
    沉默和顺从不可能保护我们：它只会鼓励当权者更加肆无忌惮地瞄准更广泛的潜在人民。
   </strong>
  </p>
  <p class="graf graf--p">
   加密和其他安全措施也不足以充分保证我们的安全：政府将始终拥有更强大的技术以击毁我们的保护壳。因为他们比我们更有钱，资源和权力在同一条裤子里。
   <strong class="markup--strong markup--p-strong">
    如果我们将抵制概念化为仅仅是技术问题，那么我们将从一开始就被打败。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    加密非常重要，但加密是为了实现我们能够实现的唯一真正的安全措施 — — 强大的社会运动，足以调动国家所针对的任何人站起来抵抗
   </strong>
   。无论政府机构收集多少情报，他们只能在能够承担政治后果的情况下利用它。
   <strong class="markup--strong markup--p-strong">
    我们越早与他们进行公开斗争，我们所有人就都会越安全。
   </strong>
  </p>
  <p class="graf graf--p">
   让我们回到透明度革命本身。理想的英雄就像我们一样：一个普通人，具有强大的勇气和智慧。
   <strong class="markup--strong markup--p-strong">
    英雄代表我们能采取的一步 — — 但不是我们由于自己害怕和感觉自己没有天赋而不得不转移期待的那个目标。这正是英雄之概念的危险之处：他们倾向于让那些相信他们的人变成纯粹的旁观者，置之不理
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这绝不是诋毁英雄。斯诺登和曼宁从最无私的慷慨中赐予一切真相的良心。但是，作为我们这些人，
   <strong class="markup--strong markup--p-strong">
    尊重他们的勇气和牺牲的最好方法是走上同样的反抗道路，而不仅仅是坐在那里点赞。
   </strong>
  </p>
  <p class="graf graf--p">
   他们给予我们的不仅仅是如生命一般珍贵的信息和公民权，而且，最重要的是，他们的行为本身，他们决心为了人民而奋斗的勇气。
  </p>
  <p class="graf graf--p">
   对于我们其他人来说，站出来抵制我们自己参与的权力结构这意味着什么？无论我们在权力架构中的哪个位置，这都是我们每个人都可以采取的步骤。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    单一的举报不会带来社会变革。变革需要的是直接的行动
   </strong>
   。请记住，弗格森并没有举报人 — — 这并不是警察不端行为的一个揭露，而它引发了近二十年来最重要的抗议警察暴行的浪潮，这正是因为人们对他们的愤怒采取了行动。对迈克尔·布朗的杀戮在全国范围内被理解为悲剧，
   <strong class="markup--strong markup--p-strong">
    因为人们抗议了
   </strong>
   ，不是因为视频记录了这件事，也不是因为内部人员透露凶手违反了某些法规。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    反对政府的运动是因为政府是非法的或腐败的，他们使我们无法通过合法的方式抵抗一切形式的残暴和虐待。无论法律如何，我们都需要培养广大公民对抗当权者的能力。否则，世界上所有举报都将是徒劳的。
   </strong>
  </p>
  <p class="graf graf--p">
   今天，我们并不缺乏对监视状态的认识，但是我们依旧缺乏如何对其采取行动的具体例子。本着
   <a class="markup--anchor markup--p-anchor" data-href="https://freejeremy.net/" href="https://freejeremy.net/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Jeremy Hammond
    </strong>
   </a>
   的精神，
   <strong class="markup--strong markup--p-strong">
    我们可能会假设我们所需要的不仅仅是揭示国家的不端行为，而是要确定其战略脆弱性。
   </strong>
  </p>
  <p class="graf graf--p">
   从保护突尼斯活动人士免受监视到揭露三K党成员的名字，Anonymous 已经证明了与社会运动协同作战的黑客攻击的战术优势。
   <a class="markup--anchor markup--p-anchor" data-href="http://www.theguardian.com/commentisfree/2010/dec/17/anonymous-wikileaks-protest-amazon-mastercard" href="https://www.theguardian.com/commentisfree/2010/dec/17/anonymous-wikileaks-protest-amazon-mastercard" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Richard Stallman
    </strong>
   </a>
   曾指出，拒绝服务行动只是一种新的封锁形式 — — 就像纽约到湾区的抗议者阻挡了州际高速公路一样，在线的活动家封锁了信息高速公路。结合在线和离线直接行动的抗议活动为跨越阶级、种族和地理的新联盟提供了机会。
  </p>
  <p class="graf graf--p">
   与此同时，在旧金山以谷歌为目标的抗议者的带领下，我们可以想象离线示威者在斗争中开辟了新的战线。也许这可以让那些认为自己是数字世界大师的人摆脱桌面的舒适区。
  </p>
  <p class="graf graf--p">
   今天，
   <strong class="markup--strong markup--p-strong">
    任何真正的实用主义者都必须承认，拆除所有不负责任的政权机构比改革它们更容易。获得隐私并处于和平状态的简单愿望使我们与全球网络化的国家权力发生直接冲突。
   </strong>
  </p>
  <p class="graf graf--p">
   这是一个令人生畏的前景，但同时也是一个好的时机，因为全球数百万其他人正在被这个头重脚轻的权力结构所产生的生态、经济和种族化危机推向同一个冲突中。
  </p>
  <p class="graf graf--p">
   在这里，我们到达了问题的核心 — —
   <strong class="markup--strong markup--p-strong">
    类似国家安全局这样的老大哥他们的主要目标从来就不是所谓的“恐怖分子”，而是挑战权力分配的基层运动
   </strong>
   。将监视覆盖面扩大到全社会甚至全世界的每个人根本不足为奇。监视机构的真正目标一直是突尼斯的活动家、埃及的革命者、希腊的无政府主义者，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wix.com/dashboard/6fc0666c-c125-4701-93b5-303f8db9aa33/blog/5cb1e5a90204460015084a4f/edit/search/.hash.m15" href="https://www.wix.com/dashboard/6fc0666c-c125-4701-93b5-303f8db9aa33/blog/5cb1e5a90204460015084a4f/edit/search/.hash.m15" rel="noopener noreferrer" target="_blank">
    ＃M15
   </a>
   ，
