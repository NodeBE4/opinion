   <strong class="markup--strong markup--p-strong">
    1、绕过 LinkedIn 的更改继续使用电子邮件地址挖掘目标情报的方法
   </strong>
  </h3>
  <p class="graf graf--p">
   对开源情报调查来说，在 LinkedIn 中搜索链接到电子邮件地址的任何帐户都是一种很有价值的工具。 研究人员一直在依靠许多不同的方法来执行此操作，包括URL操纵和浏览器加载项。最常用的是
   <a class="markup--anchor markup--p-anchor" data-href="https://business.linkedin.com/sales-solutions/compare-plans/sales-navigator-for-gmail" href="https://business.linkedin.com/sales-solutions/compare-plans/sales-navigator-for-gmail" rel="noopener noreferrer" target="_blank">
    Sales navigator for gmail
   </a>
   。
  </p>
  <p class="graf graf--p">
   Sales Navigator 是一种营销工具，旨在允许销售团队在 LinkedIn 上建立和培养客户关系。
  </p>
  <p class="graf graf--p">
   由于使用率低，从2020年3月起，Sales Navigator 已经不再在 Gmail 中运行。尽管销售团队可能并未按预期的方式使用 Sales Navigator，但开源情报的从业人员在进行人员搜索时每天都在使用它。
  </p>
  <p class="graf graf--p">
   目前为止拥有 LinkedIn 帐户的任何人都可以操纵URL，以便为他们提供 Sales Navigator 搜索电子邮件地址的结果，而不必为 Sales Navigator 付费。
  </p>
  <p class="graf graf--p">
   只需更改URL末尾的电子邮件地址，您就可以自由地向 LinkedIn 查询任何电子邮件地址，以识别是否已链接到 LinkedIn 帐户。查询如下：
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.linkedin.com/sales/gmail/profile/viewByEmail/intelligencewithsteve@protonmail.com" href="https://www.linkedin.com/sales/gmail/profile/viewByEmail/intelligencewithsteve@protonmail.com" rel="nofollow noopener noreferrer" target="_blank">
    https://www.linkedin.com/sales/gmail/profile/viewByEmail/
   </a>
   电邮地址
  </p>
  <p class="graf graf--p">
   但是在找到预期结果之前您会收到一个提示：“Something went wrong while displaying this profile. Please reload or try again later”。
  </p>
  <p class="graf graf--p">
   幸运的是，还有另一种通过电子邮件地址搜索 LinkedIn 的方法。自2017年以来，Sales Navigator 已与 Outlook 集成。Microsoft 同时拥有 Outlook 和 LinkedIn，并且许多公司都使用 Outlook 进行日常活动，因此与 Outlook 而不是 Gmail 集成是有意义的。
  </p>

  </p>

    https://account.microsoft.com/account?lang=en-us
   </a>
