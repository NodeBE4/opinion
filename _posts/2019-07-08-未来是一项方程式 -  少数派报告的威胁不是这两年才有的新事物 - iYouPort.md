  <p class="graf graf--p">
   画外音里，一个警察高兴地说：“从前，我们追捕犯了罪的罪犯，而现在，我们可以在他们犯罪之前阻止他们”。这个声音来自近10年前的孟菲斯。
  </p>
  <p class="graf graf--p">
   这里是美国犯罪率最高的城市之一。电视上，IBM公司正在为它们的一款名为
   <a class="markup--anchor markup--p-anchor" data-href="http://prasenjitchaudhuri.blogspot.com/2010/08/blue-crush-ibms-predictive-crime.html" href="https://prasenjitchaudhuri.blogspot.com/2010/08/blue-crush-ibms-predictive-crime.html" rel="noopener noreferrer" target="_blank">
    Blue Crush
   </a>
   的犯罪分析软件做广告，夸耀其效果。在2010年，市政府决定裁员25%的警察，为了不降低工作效能，孟菲斯市政府向 IBM 求助。
  </p>
  <p class="graf graf--p">
   从此，警察在执勤之前都会从手机上或是汽车配备的电脑上收到标明热点的地图，在这些“热点”区域，未来24小时被认为“有可能发生犯罪活动”。根据命令，警察部队会向闪着红光的热点地区集中。
  </p>
  <p class="graf graf--p">
   Blue Crush 软件由大数据公司的数学家、信息专家和人类学家共同研发。首先，它录入了警察局所有电子档案：出警报告、笔录、报警电话记录等等，然后通过算法，根据时间、地点、类型的不同，将犯罪进行归类。
  </p>
  <p class="graf graf--p">
   Blue Crush 软件每天24小时不间断与警察局电脑网络连接，可以随着警察录入的新情况、以及装在巡逻警车上和分布于城市各处的其他500个摄像头传回的画面，永远及时更新数据库。
  </p>
  <p class="graf graf--p">
   Blue Crush 据称可以根据定位软件计算出发生犯罪活动的概率。之后 IBM 又研发出
   <a class="markup--anchor markup--p-anchor" data-href="http://adimen.si.ehu.es/~rigau/teaching/EHU/TAIA/Curs2017-2018/Homework1/Team%2012%20-%20E.%20Perez%20and%20C.%20Mayor/Team12Report.pdf" href="http://adimen.si.ehu.es/~rigau/teaching/EHU/TAIA/Curs2017-2018/Homework1/Team%2012%20-%20E.%20Perez%20and%20C.%20Mayor/Team12Report.pdf" rel="noopener noreferrer" target="_blank">
    更高一层的版本 PredPol
   </a>
   ，这个版本已经在洛杉矶、亚特兰大和纽约的警察局使用。从此，
   <strong class="markup--strong markup--p-strong">
    算法牢牢地掌握着
   </strong>
   通过统计对比归类的潜在所谓犯罪人群的肖像。司法部科技部门前负责人曾经对算法的偏见问题发出过堵死异议的高调宣称：“显然大数据中含有偏见成分，但今后，偏见也可以通过数学的方式检测出来”。
  </p>
  <p class="graf graf--p">
   可以处理以前的犯罪信息、更可以预测下一次“犯罪”将在何时何地发生的少数派报告监视系统，很快在欧洲市场上极具竞争力。在英格兰肯特郡的警察局购买了 PredPol 后，德国模型预测科技学院的科学家们也研发出了
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Precobs" href="https://en.wikipedia.org/wiki/Precobs" rel="noopener noreferrer" target="_blank">
    自己的预测软件 Precobs
   </a>
   ，从2015年开始就在慕尼黑、纽伦堡、科隆、以及瑞士的苏黎世和巴塞尔进行了测试。而在法国，国家犯罪观测所在2016年也进行了同样的预测犯罪系统测试。
  </p>
  <p class="graf graf--p">
   少数派报告描述的是2054年发生的同样的事，但显然，Philip Kindred Dick 的噩梦提前几十年就被实现了。
   <strong class="markup--strong markup--p-strong">
    少数派报告是大数据最纯粹的噩梦
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在大数据科研实验室里，科学家们对 PredPol 及其他预测软件进行着进一步的完善，目的就是使这些软件在未来不仅能够预测犯罪何时何处发生，还能判断出谁是始作俑者。
  </p>
  <p class="graf graf--p">
   Nate Silver 曾经兴奋地表示：
   <strong class="markup--strong markup--p-strong">
    “未来是一项方程式”
   </strong>
   。这位被称为美国预测技术神算子的人在奥巴马入驻白宫的竞争中就负责主持预测分析工作。
  </p>
  <p class="graf graf--p">
   对于高科技企业来说，人类的绝大多数行为都是可以预测到的。今天，大数据时代的预言家不需要使用咖啡渣来占卜了，他们有更可靠的软件可以处理海量数据，答案就在其中。
  </p>
  <p class="graf graf--p">
   2007年，美国国土安全部发布了一项研究计划，旨在确立“潜在的恐怖分子”，也就是说，
   <strong class="markup--strong markup--p-strong">
    这些人今天是完全无罪的，但是算法认为他们在未来的某一天有可能成为“恐怖分子”。这项名为
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Future_Attribute_Screening_Technology" href="https://en.wikipedia.org/wiki/Future_Attribute_Screening_Technology" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     “未来属性筛选”技术（FAST）
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    的计划
   </strong>
   ，通过对所有相关元素进行筛查，包括个人行为、肢体语言、生理外貌等特点。一旦筛出“可疑的人”，就会借助面部识别系统对其进行密切跟踪 — — 请注意，这可不是最近两年的事，FAST 始于2008年。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    相关最近的报道（关于该技术有多么的偏见）：《
    <a class="markup--anchor markup--li-anchor" data-href="https://medium.com/@iyouport/hr-a388e3ff004c" href="https://www.iyouport.org/%e5%b8%ae%e6%b4%be%e6%95%b0%e6%8d%ae%e5%ba%93%e6%98%af%e4%bb%80%e4%b9%88%ef%bc%9f%e4%b8%ba%e4%bb%80%e4%b9%88%e6%8d%ae%e6%ad%a4%e5%b0%b1%e5%8f%af%e4%bb%a5%e7%9b%b4%e6%8e%a5%e7%bb%99/" rel="noopener noreferrer" target="_blank">
     “帮派数据库”是什么？为什么据此就可以直接给无辜的市民定下重罪？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这是监视摄像头被用于分析人类行为 — 这一目前最热门老大哥姿势 — 的开端
   </strong>
   。中国的步态识别实际上是很老土的东西，这些监视技术不仅可以识别身份，还能识别情绪和心理活动，比如通过对出汗和血液流动速度变化等指标的读取，觉察出人的心理反应。
  </p>
  <p class="graf graf--p">
   在伦敦郊区的卢顿，警察对街上所有摄像头进行了测试，当这些摄像头监测到数据库中录入的50种非法行为中的一种时，就会自动发出警报；而在尼斯，915个智能摄像头对人行道进行密切监控，
   <strong class="markup--strong markup--p-strong">
    自动定位那些在人群中表现焦躁不安或是过于僵化的人，并视其为“可疑”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   没错，不仅美国和中国，欧盟也对此上瘾。2011年，欧盟委员会就实施了一项名为
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/INDECT" href="https://en.wikipedia.org/wiki/INDECT" rel="noopener noreferrer" target="_blank">
    Indect 的计划
   </a>
   。他们挑选了不少于十七支科研院队，专门研究大数据如何在城市里发现“隐藏的非正常行为”。很快，如果你在大街上跑步、在人群中逆行、行进过快、当别人坐下的时候站起来、在商店蹲下系鞋带、在机场大厅拍照、甚至戴口罩……都会被电脑判定为可疑人物。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    更多详见最近的消息：《
    <a class="markup--anchor markup--li-anchor" data-href="https://medium.com/@iyouport/newsletters-04172019-ea4957a28a3f" href="https://www.iyouport.org/%e5%b0%91%e6%95%b0%e6%b4%be%e6%9a%b4%e8%99%90/" rel="noopener noreferrer" target="_blank">
     少数派暴虐
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   不仅如此，在公共空间，算法正在不知不觉中通过行为分析摄像头，
   <strong class="markup--strong markup--p-strong">
    将新的行为代码强加给人类
   </strong>
   。如果你不遵循这个代码，那么在电脑的记忆里，你的行为就会被贴上“可疑分子”的标签。
  </p>
  <p class="graf graf--p">
   大数据收集的信息量远远超过了现实世界里个人想要得到的所有知识。海量数据使得预测分析成为可能。
   <strong class="markup--strong markup--p-strong">
    不仅能预测个人行为，还能预测群体事件 — — 我敢说后者是当权者更感兴趣的东西。
   </strong>
  </p>
  <p class="graf graf--p">
   大数据渴望着未来能检测出人类的心理偏差、自杀或是犯罪意图，因为它认为，行为在发生之前，信息总会产生预兆、或多或少反复导致的必然性迹象，事前没有这些迹象的行为是极其罕见的。
  </p>
  <p class="graf graf--p">
   就像少数派报告中描绘的世界一样，
   <strong class="markup--strong markup--p-strong">
    人们更注重意图犯罪
   </strong>
   ，这是人们倾向于采取的所谓反恐方式。借着反恐的旗号，将所有人的观念从“有罪”过渡到了“危险性”，如今这个概念已经充分被推广，并蔓延全球。更多详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://steemit.com/privacy/@iyouport/2uxfge" href="https://www.iyouport.org/%e5%8f%af%e6%80%95%e7%9a%84%e8%bf%9e%e7%82%b9%e6%88%90%e7%ba%bf-%e5%92%8c%e4%ba%92%e8%81%94%e7%bd%91%e5%ae%a1%e6%9f%a5-%e7%9b%91%e8%a7%86%e4%b9%8b%e6%81%b6%ef%bc%88/" rel="noopener noreferrer" target="_blank">
    可怕的“连点成线” 和互联网审查 — — 监视之恶（四）“反恐”歧途
   </a>
   》
  </p>
  <p class="graf graf--p">
   比如在法国，最新的法律中就引入了“意图犯罪行为”的概念。现在，只要发现某人在家中有发动袭击的意图，警察就可以把他带上法庭。最近一个邪门的案例是这样的：
   <strong class="markup--strong markup--p-strong">
    如果你因任何原因需要一把锤子，英国的反恐警察部队就会认为你是一个潜在的”恐怖分子”
   </strong>
   。 他们还明确要求群众斗群众：任何看到某人做出可疑事的人 — — 比如购买锤子，就立即向执法者报告。详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/newsletters-04112019-3f47185495bd" href="https://www.iyouport.org/%e4%b9%b0%e4%b8%80%e6%8a%8a%e9%94%a4%e5%ad%90%e5%b0%b1%e4%bc%9a%e8%83%bd%e8%ae%a9%e4%bd%a0%e5%8f%98%e6%88%90%e6%81%90%e6%80%96%e5%88%86%e5%ad%90/" rel="noopener noreferrer" target="_blank">
    恐怖锤子
   </a>
   》。中国的菜刀实名制就不需要过多解释了。
  </p>
  <p class="graf graf--p">
   如今当权者更加相信，借助大数据的工具，司法可以更加依赖预测学；警察再也不用惩罚犯罪分子，而是惩罚犯罪的意图 — — 简单说就是*你想也别想*。
  </p>
  <p class="graf graf--p">
   这一诱惑是巨大的。为了保护有关大数据的法律，内政部长 Bernard Cazeneuve 抬出了与国家机器签署的自动识别“恐怖活动”的文件为自己的老大哥上瘾做辩护。刑法是建立在通过证据构成的有罪基础上的，而现在，大数据完全破坏了这一基础。
  </p>
  <p class="graf graf--p">
   私法和犯罪科学教授 米蕾耶马蒂对此很担忧，他说，人们是在从大数据中发现潜在可疑目标，而不是从潜在可以目标中找出数据，对现实的恐惧重振了大数据对未来的预测。（法国世界报/“老大哥怀里的民主”，2015年6月）
  </p>
  <p class="graf graf--p">
   人类身处怀疑的时代，这是一个被大数据占据的时代，这个时代被认为消除了世界的不稳定。然而，事实却相反。大数据公司纷纷在美国建立了实验室，然而，美国却是全球暴力泛滥排行榜上的前几位国家。仅仅在奥尔良40万居民中，2015年第一季度就有92起被记录的谋杀事件；这个数字相当于拥有200万居民的巴黎地区在2013年全年发生的谋杀数量。新奥尔良凶杀案不断攀升，一直达到了巴黎的40倍。
  </p>
  <p class="graf graf--p">
   为了使人们安心，大数据跨国公司建议人们通过它们研发的算法来消除社会不稳定因素。米蕾耶马蒂曾指出这种危害：
   <strong class="markup--strong markup--p-strong">
    大数据希望预测行为、发现企图，这已经成为一种失去人性的方式，因为人性的魅力在于其不确定性，如果人类失去了这一特性，将无法负责任何事物
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如今，大数据公司开拓的新市场是出售预测。谷歌是最早起步的，该公司尝试通过搜索引擎发现患流感的家庭。通过实行5亿次计算，谷歌公司的工程师确定了流行病发作地区上网者最常使用的45个搜索关键词；接着，位于山景城的这家大数据公司的分析师开发出了一种算法，能够先于全世界发现感染流行病的家庭所在的地区和城市。
  </p>
  <p class="graf graf--p">
   这些数据对医药公司来说尤其珍贵，因为它们可以借此在适当的时候销售药物。谷歌公司同样可以将信息传递到房地产市场，通过处理上网者的搜索行为，搜索引擎能够对下个月的房地产市场波动进行建模，这要比美国全国房地产经纪联合会的数据更快、更丰富。
  </p>
  <p class="graf graf--p">
   如今已经将这些信息发送到了房地产信息预测平台 Auction，从而预测产生了直接效益，谷歌仅在这点上就赚了五千万美元。
  </p>
  <p class="graf graf--p">
   同样，英国伯明翰大学研究专家宣称，通过我们所有人的智能手机跟踪数据，就可以预测出未来24小时我们都在哪儿，误差可以缩小至20米以内。更多详见一个专利介绍：《
   <a class="markup--anchor markup--p-anchor" data-href="https://iyouport.weebly.com/short-stories-enthusiast/facebook3070727" href="https://www.iyouport.org/facebook%e4%b8%8d%e4%bb%85%e7%9f%a5%e9%81%93%e4%bd%a0%e7%8e%b0%e5%9c%a8%e5%9c%a8%e5%93%aa%ef%bc%8c%e5%ae%83%e8%bf%98%e6%83%b3%e7%9f%a5%e9%81%93%e4%bd%a0%e6%9c%aa%e6%9d%a5%e4%bc%9a%e5%9c%a8%e5%93%aa/" rel="noopener noreferrer" target="_blank">
    Facebook 不仅知道你现在在哪，它还想知道你未来会在哪
   </a>
   》
  </p>
  <p class="graf graf--p">
   以色列数学家 Kira Radinsky 被称为“互联网时代的女预言家”，她声称借助她自己开发的算法可以预测金融市场的崩溃或即将发生的动荡，准确率超过90%。在日常生活中，她的创业公司
   <a class="markup--anchor markup--p-anchor" data-href="https://www.crunchbase.com/organization/salespredict#section-overview" href="https://www.crunchbase.com/organization/salespredict#section-overview" rel="noopener noreferrer" target="_blank">
    SalesPredict
   </a>
   在各个领域向大公司提供市场预测信息。
  </p>
  <p class="graf graf--p">
   大数据公司需要消除日常生活中的不确定性，才能擦亮占卜的水晶球。哲学家 Roger Pol-Droit 说过，为了最终建造一个完全彻底的所谓幸福世界，我们今后的目标是消除偶然性。在想象的天堂里，人们可以控制和掌握偶然性，并完全彻底将其消除……（“哲学不能造就幸福”/2015）
  </p>
  <p class="graf graf--p">
   为了不惜一切代价排除偶然性，人们再一次冒险抹去了人类本质上不确定性的那部分。
   <strong class="markup--strong markup--p-strong">
    正由于不可预见性，人类的思想才变得丰富，在出乎意料面前，人类思想才能真的富有创造力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在整个人类历史上，如果没有这些偶然性和意外，大部分的伟大发现就不可能存在。就如你所知道的那样，哥伦布是由于错误计算才导致发现了美洲新大陆；弗莱明也一样，他发现了一种霉菌，把他之前培育的葡萄球菌全部杀死了，后来他把这种霉菌命名为青霉素；物理学家贝克勒尔把铀盐撒到了相片底片上时，发现底片被感光了，就像在阳光照射下一样，从而发现了天然放射性……
  </p>
  <p class="graf graf--p">
   人类的发展历史本身就充满了偶然性和不可靠因素。父母的基因编码互相结合所产生的结果本身就是不可预见的；而由0和1决定的数字结果却是全然相反的，它热衷于把最难以置信的偶然性源头完全转化为方程式。
  </p>
  <p class="graf graf--p">
   这种偶然性也包括两个人的相逢。“因为爱情并不源于偶然性” — — 这是 Parship 的广告词。Parship 是一款交友应用程序，让人们在此找到情投意合的人。和其他同类软件一样，它会不断提醒你：一个完全符合你期望的人、可以马上约会的人，目前正在距离你几条街之外的某地…
  </p>
  <p class="graf graf--p">
   算法甚至能够决定我们与他人的交流联络，这是诸如 Facebook 之类的社交网络的观点。Facebook 通过 EdgeRank 算法计算出人与人之间情投意合的指数。该公司负责人解释说，Facebook 致力于实现“只和你觉得亲近的公司或个人进行交流。
  </p>
  <p class="graf graf--p">
   但有一点要注意，
   <strong class="markup--strong markup--p-strong">
    由于只和自己观点相似的人进行交流，所有人的观念正在变得越来越封闭、想法僵化、不再能碰撞出头脑风暴。互联网已不再是*交流和辩论的思想市场*
   </strong>
   。
  </p>
  <p class="graf graf--p">
   大数据向我们出售最小化的时间，获取最大化的效率，这保证了其良好的竞争力。阿尔巴尼亚著名诗人 Ismail Kadare 在谈论他的著作“梦幻宫殿”时说：“长久以来，我梦想建立一个地狱”。在这个想象的国度，无处不在的管理部门夜以继日地搜集居民的梦想，将其挑选、分类、破译，是为了看清这个国度的未来。人类正在把太多的权力拱手相让于大数据，不仅有解读梦想的权力，还有看清未来的权力…… ⚪️
  </p>
