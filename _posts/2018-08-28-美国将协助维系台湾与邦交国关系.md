title: 美国将协助维系台湾与邦交国关系

</div>
<p>
<img class="aligncenter" src="https://www.rfa.org/mandarin/yataibaodao/gangtai/hx-08232018101712.html/hx823.jpg"/>
</p>
<p>
  台湾的总统蔡英文8月23日会见美国在台协会台北办事处新任处长郦英杰
 </p>
<p>
</p>
<p>
<span style="font-size: 12pt;">
   在萨尔瓦多与台湾断交，转与北京建交后，美国参议员要求即将派驻中南美洲的大使，帮助台湾维系邦交国关系。美国在台协会台北办事处新任处长也表示，美国对萨尔瓦多感到失望，会重新评估与萨国关系，并支持台湾扩大对国际的参与和贡献。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   美国国会参议院外交委员会22日进行美国驻苏利南、洪都拉斯与尼加拉瓜三个中南美洲国家大使提名人听证会，多名参议员担忧萨尔瓦多跟台湾断交，会引发骨牌效应，要求美国外派大使帮助台湾维系友邦。美国驻尼加拉瓜大使提名人苏利文承诺，将敦促尼国维持与台湾关系；驻洪都拉斯大使提名人巴梅瑞也承诺，将台洪关系列为优先工作项目。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   前美国副国务卿阿米塔吉22日也发声明表示，台湾不应遭受胁迫，并应有保护主权与公民生计的权利。美国应在现有“一中政策”架构下扩大与台湾互动。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   在台湾，蔡英文总统23号接见美国在台协会台北办事处新任处长郦英杰。郦英杰表示，美国对萨尔瓦多的决定非常失望，如同白宫声明所说，萨尔瓦多接受中国明显干预西南半球国家的国内事务，这对美国是很大的忧虑，美国会重新评估和萨尔瓦多的关系。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   郦英杰说，中国试图片面改变现状是无益的，且伤害区域数十年来的和平、稳定和发展架构，美国会持续支持台湾扩大对国际的贡献，并支持台湾持续反抗这种制约台湾参与国际社会的方式。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   蔡英文强调，中国对台压迫力道持续加大，且无所不用其极，不只影响台湾的国际空间，也冲击区域和平及稳定。台湾需要理念相近国家共同关切中国在区域内种种压迫台湾、改变现状、破坏国际规则，甚至干预各国内部事务的行为。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   蔡英文说：“美国政府强调台湾是一个民主成功的故事，更是美国可信赖的伙伴，是国际良善的力量的说法，正是台湾人民所期待的正义之声。我们一定会坚守主权立场、坚持国家的尊严，也会持续强化自我防御能力、捍卫自由跟民主的生活方式，这是我们全体台湾人民的底线，也是无可挑战的普世公理。”
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   台湾蓝营人士把萨尔瓦多和“台湾”断交怪罪于蔡英文，批评她任内接连失掉五个友邦，是民进党搞台独，先放弃“中华民国”继承权，造成外交崩盘。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   对此，外交部发言人李宪章接受自由亚洲电台采访说：“我觉得不用去相信语言上这种故意含混。台湾、中华民国台湾，这是一个如假包换的国家，你要称呼宪法上规定明文是‘中华民国’国名，可是全世界都知道‘台湾’，几百年来都是这样子‘福尔摩沙’，我们国人有时候为了出国方便，我们的护照上后来也加注了台湾，所以基本上中华民国跟台湾是等同。”
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   萨尔瓦多国会第一副议长及国家团结联盟阵线（GANA）党主席葛耶哥斯在台湾与萨国断交后表示，若GANA赢得下次总统大选，将尽快恢复台萨邦交。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   李宪章说，台萨邦谊85年，对各方友台作为表示感谢。他提及，早期两岸曾在外交上恶性竞争，有些国家在两岸之间犹疑，出现过断交、复交情况，目前没有这样的条件跟气候。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   中国大陆国台办主任刘结一23号表示，“一中原则”是“大势所趋，不可阻挡”，“还有一些国家”会不断做出“正确决定”。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
