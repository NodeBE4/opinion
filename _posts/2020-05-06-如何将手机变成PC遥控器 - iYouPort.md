  <p class="graf graf--p">
   如今很少有人看电视了，越来越多的人选择通过电脑进行在线娱乐。这也意味着人们坐在电脑桌的时间可能比以前更长。
  </p>
  <p class="graf graf--p">
   当然，有些人可以访问连接到其主要电视或智能电视的游戏机，他们可以使用这些游戏机直接观看电影和节目。但是，并非每个人都如此幸运。
  </p>
  <p class="graf graf--p">
   尽管您可能没有 Sony PlayStation 4 或三星最新的 Ultra HD 之一，但您很有可能拥有智能手机。这就够了。
  </p>
  <p class="graf graf--p">
   用您的手机就可以整合一个以计算机为中心的家庭影院。
  </p>
  <p class="graf graf--p">
   只需下载以下其中一个应用，您最终就可以将手机用作PC遥控器，从舒适的沙发上浏览计算机屏幕。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    将手机变成PC遥控器
   </strong>
  </p>
  <p class="graf graf--p">
   在 Google Play 和 Apple App Store 中有很多PC远程控制应用可供选择。大多数都可以在 Windows 和 Mac 操作系统上使用。
  </p>
  <p class="graf graf--p">
   还有一些应用程序可以将您的手机变成一个通用遥控器，该遥控器可以控制您家中从冰箱到空调的所有事物 —— 听起来很可怕啊。

   <a class="markup--anchor markup--p-anchor" data-href="https://remotemouse.net/" href="https://remotemouse.net/" rel="noreferrer noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Remote Mouse
    </strong>
   </a>
  </p>
  <p class="graf graf--p">
   Remote Mouse（RM）是您会发现的最容易使用的PC远程控制应用程序之一。它是100％免费的，可将您的 Android 或 iOS 智能手机转换为计算机的无线输入设备。
  </p>
  <p class="graf graf--p">
   RM 有超过一百万的活跃用户，并支持 Android、iOS、Windows 和 Mac 操作系统。
  </p>
  <p class="graf graf--p">
   1、分别通过 Google Play 或 App Store 在 Android 或 iOS 设备上下载并安装 Remote Mouse 客户端。
  </p>
  <p class="graf graf--p">
   2、您还需要将服务器下载到您的计算机上。不要担心它相对较小，仅为706 KB，而手机应用程序为9 MB。
  </p>
  <p class="graf graf--p">
   3、安装后，计算机和手机都必须连接到同一 WiFi 热点或路由器。这将允许两个设备同步。
  </p>
  <p class="graf graf--p">
   4、建立连接后，请同时运行服务器和客户端应用程序。
  </p>
  <p class="graf graf--p">
   5、从客户端应用程序上，您应该能够看到当前连接到同一WiFi网络的所有设备的列表。
  </p>
  <p class="graf graf--p">
   6、点击您要使用该应用程序控制的设备的名称。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    如果未显示您要查找的设备，请点击+图标，然后选择 Connect by IP。在提供的文本字段中输入设备的IP地址。
   </li>
  </ul>
  <p class="graf graf--p">
   7、从电话到计算机的连接成功后，客户端应用程序将打开其主要用户界面。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    该界面包含六个功能图标，一个“设置”菜单和一个“电源”按钮。
   </li>
   <li class="graf graf--li">
    图标正上方是模仿计算机鼠标的三个按钮（左键单击，中键单击/滚动和右键单击）
   </li>
  </ul>

   <a class="markup--anchor markup--p-anchor" data-href="https://play.google.com/store/apps/details?id=wowappz.kiwimote" href="https://play.google.com/store/apps/details?id=wowappz.kiwimote" rel="noreferrer noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     KiwiMote
    </strong>
   </a>
  </p>
  <p class="graf graf--p">
   这绝对是您在Google Play商店中找到的评价最高的应用之一。
  </p>
  <p class="graf graf--p">
   此应用支持所有4.0.1或更高版本的 Android，并且需要安装适用于 Windows、Mac 和 Linux 操作系统的计算机端服务器。
  </p>
  <p class="graf graf--p">
   该服务器仅占用2 MB的硬盘驱动器存储空间，但需要在系统上安装 Java 才能正常工作。
  </p>
  <p class="graf graf--p">
   必须使用 WiFi 连接才能将 KiwiMote 从手机连接到计算机。较酷的功能之一是 KiwiMote 允许您通过使用手机扫描计算机上的QR码来建立连接。或者，您必须输入IP地址，端口号和唯一的PIN才能建立连接。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11115 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-12.png?resize=1100%2C677&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-12.png?w=1186&amp;ssl=1 1186w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-12.png?resize=300%2C185&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-12.png?resize=1024%2C630&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-12.png?resize=768%2C473&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-12.png?resize=1100%2C677&amp;ssl=1 1100w" data-recalc-dims="1" height="677" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-12.png?resize=1100%2C677&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11115" data-recalc-dims="1" height="677" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-12.png?resize=1100%2C677&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-12.png?w=1186&amp;ssl=1 1186w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-12.png?resize=300%2C185&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-12.png?resize=1024%2C630&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-12.png?resize=768%2C473&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-12.png?resize=1100%2C677&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   您将获得键盘、鼠标和游戏手柄等基本的PC远程控制功能，但该应用程序不允许您查看计算机屏幕。
  </p>
  <p class="graf graf--p">
   取而代之的是，您可以使用针对多种桌面应用程序的一些用户友好界面，例如 Adobe PDF Reader 和 VLC Media Player。
  </p>
  <p class="graf graf--p">
   KiwiMote 由广告资助，因此完全免费下载和使用。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11116 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-12.png?resize=1100%2C627&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-12.png?w=1136&amp;ssl=1 1136w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-12.png?resize=300%2C171&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-12.png?resize=1024%2C584&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-12.png?resize=768%2C438&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-12.png?resize=1100%2C627&amp;ssl=1 1100w" data-recalc-dims="1" height="627" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-12.png?resize=1100%2C627&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11116" data-recalc-dims="1" height="627" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-12.png?resize=1100%2C627&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-12.png?w=1136&amp;ssl=1 1136w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-12.png?resize=300%2C171&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-12.png?resize=1024%2C584&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-12.png?resize=768%2C438&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-12.png?resize=1100%2C627&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.unifiedremote.com/" href="https://www.unifiedremote.com/" rel="noreferrer noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Unified Remote
    </strong>
   </a>
  </p>
  <p class="graf graf--p">
   Unified Remote 是最有趣的远程应用程序之一，并且是在此列表中唯一具有付费选项的应用程序。
  </p>
  <p class="graf graf--p">
   与本文中的其他两个类似，它需要一个手机端应用程序和一个计算机端服务器才能工作。
  </p>
  <p class="graf graf--p">
   它不仅适用于 Windows、Mac 和 Linux 操作系统，而且还可以控制其他设备，包括 Raspberry Pi 和 ArduinoYún。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11117 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-12.png?resize=1100%2C634&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-12.png?w=1218&amp;ssl=1 1218w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-12.png?resize=300%2C173&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-12.png?resize=1024%2C590&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-12.png?resize=768%2C443&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-12.png?resize=1100%2C634&amp;ssl=1 1100w" data-recalc-dims="1" height="634" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-12.png?resize=1100%2C634&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11117" data-recalc-dims="1" height="634" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-12.png?resize=1100%2C634&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-12.png?w=1218&amp;ssl=1 1218w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-12.png?resize=300%2C173&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-12.png?resize=1024%2C590&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-12.png?resize=768%2C443&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-12.png?resize=1100%2C634&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   免费版的 Unified Remote 随附标准必需品（键盘、鼠标和音量），明暗主题可供选择，并提供18个免费遥控器，并支持单点和多点触摸鼠标控制。
  </p>
  <p class="graf graf--p">
   它与其他两个设置一样简单，提供自动服务器检测和密码保护。
  </p>
  <p class="graf graf--p">
   如果您决定以$ 3.99的价格购买完整版，则可以使用90多个遥控器，可以自定义自己的遥控器以及使用语音命令的功能。
  </p>
  <p class="graf graf--p">
   最后，在不用的时候不要忘了把Wi-Fi关掉，它不安全。见《

    Wi-Fi是如何出卖你的？
