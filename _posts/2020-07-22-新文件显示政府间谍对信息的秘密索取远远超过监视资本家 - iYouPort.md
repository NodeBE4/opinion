   </noscript>
  </p>
  <p class="graf graf--p">
   最近发布的文件显示，FBI 近几年已经向包括电信提供商、技术公司和信贷机构在内的120多家公司发出了大量的秘密书面信息索取要求，这引发了人们对政府悄悄收集公民数据能力的新疑问。
  </p>
  <p class="graf graf--p">
   这些数据索取是在
   <strong class="markup--strong markup--p-strong">
    没有法院监督的情况下
   </strong>
   发出的（称为国家安全信函或NSL），通常伴随着针对公司的禁言令。被拿走数据的人很少了解 NSL，也没有机会在法庭上质疑其要求。
  </p>
  <p class="graf graf--p">
   公民自由组织电子前沿基金会通过《信息自由法》诉讼发布的新文件强调了对2015年法律的有效性的持续质疑，该法律表面上旨在部分地为NSL的发送和执行带来一定的透明度。
  </p>
  <p class="graf graf--p">
   文件显示，收到NSL的公司排名前位的是信贷机构 Equifax、Experian 和 TransUnion；电信提供商 AT＆T、T-Mobile 和 Verizon；金融服务公司 美国银行、西联汇款和 Capital One；科技寡头 Google、Microsoft 和 Facebook。其中有些收到几十个NSL，其他的收到几个。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     在这里看到这些公司如何出卖公民反抗者
    </strong>
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bb%96%e4%bb%ac%e6%98%af%e5%a6%82%e4%bd%95%e6%8a%93%e6%8d%95%e8%af%b4%e7%9c%9f%e8%af%9d%e7%9a%84%e4%ba%ba%ef%bc%9f%e2%9a%a0%ef%b8%8f%e7%81%be%e9%9a%be%e5%92%8c%e6%95%99%e8%ae%ad/" href="https://www.iyouport.org/%e4%bb%96%e4%bb%ac%e6%98%af%e5%a6%82%e4%bd%95%e6%8a%93%e6%8d%95%e8%af%b4%e7%9c%9f%e8%af%9d%e7%9a%84%e4%ba%ba%ef%bc%9f%e2%9a%a0%ef%b8%8f%e7%81%be%e9%9a%be%e5%92%8c%e6%95%99%e8%ae%ad/" rel="noopener noreferrer" target="_blank">
     他们是如何抓捕说真话的人？⚠️灾难和教训
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   以上这些公司没有任何一家对这些披露发表评论。
  </p>
  <p class="graf graf--p">
   该法律特别允许 FBI 随意向这些行业的公司发送 NSL 数据索取要求。纽约时报首次报道了EFF获取的新文件的出版情况。
  </p>
  <p class="graf graf--p">
   一些技术公司已公开其收到的NSL信息，在某些情况下，还发布了经过编辑的NSL，以提高人们对这种做法的认识，在定期报告中发布了其数量表，或在法庭上与禁言令作斗争。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    其他行业的公司，例如 电信和银行业，通常都已经满足政府间谍对信息的需求
   </strong>
   ，政府官员声称，“这对进行恐怖主义或情报问题的调查至关重要”。
  </p>
  <p class="graf graf--p">
   自9/11以来，政府已经发布了成千上万的 NSL，这促使政府从私人公司保存的记录中收集数据的范围不断扩大。斯诺登的书中对此有更多详细说明。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    在这里下载斯诺登的书 **完整中文版**：
    <a class="markup--anchor markup--li-anchor" data-href="https://t.me/iyouport/6935" href="https://t.me/iyouport/6935" rel="nofollow noopener noreferrer" target="_blank">
     https://t.me/iyouport/6935
    </a>
   </li>
  </ul>
  <p class="graf graf--p">
   EFF公布的文件记录显示，FBI 在审查过程中考虑了将近12,000个禁言令，并取消了其中的6％。FBI拒绝对文件发表评论，他们每年发出数千份新的NSL。
  </p>
  <p class="graf graf--p">
   总部位于旧金山的EFF资深律师 Andrew Crocker 说：“发送这些信函的速度比取消它们的速度快得多；所以整体是增加而不是减少。”
  </p>
  <p class="graf graf--p">
   NSL 是 FBI 发出的书面指令，要求提供私密的通信记录或财务交易记录。它们最常用于获取电话和电子通讯记录，也用于获取财务记录和消费者信用信息。
  </p>
  <p class="graf graf--p">
   尽管NSL的历史可追溯到1980年代，但FBI在2001年911之后将其用作调查工具的次数激增。
  </p>
  <p class="graf graf--p">
   彻底改革NSL的使用源于广泛的公众反对，自从前国家安全局承包商 Edward Snowden 揭露了大规模监视暴力的内幕以来。斯诺登提供的文件显示，世界各地的人们共享的电子邮件、电话聊天或与亲人进行的视频通话，每时每刻都被政府全面监视着。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    在这里看到斯诺登文件中文完整版介绍《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" rel="noopener noreferrer" target="_blank">
     互联网霸主的时间线 — — 他们将互联网视为间谍、破坏和战争的主场
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   与国家安全局的技术数据收集工具相比，NSL 是一种更有针对性的信息收集方式，但长期以来，由于缺乏司法监督和透明性，隐私权和公民自由倡导者对此不断提出指责。
  </p>
  <p class="graf graf--p">
   搜查令需要得到法官的批准。但NSL的法律标准很低：只要能说明所寻求的记录与调查有关就行。
  </p>
  <p class="graf graf--p">
   为NSL辩护的官员说，他们允许调查人员在敏感案件中追踪线索，而目标却不知道自己正在被视为嫌疑而受到监视，这种策略可能会加快逮捕速度……⚪️
  </p>
  <p class="graf graf--p">
