   <h1>
    梁小斌：《母语》·赵野：《汉语》
   </h1>
   <ul class="post-entry-meta">
    <li>
     By:
     <a href="http://www.ideobook.com/about-ideobook/" title="查看 爱德布克 的作者主页">
      爱德布克
     </a>
     . 2004-7-9.
     <a href="http://www.ideobook.com/372/post-views-count/" title="统计说明">
      6,599
     </a>
    </li>
   </ul>
  </header>
  <div class="post-entry-content">
   <blockquote>
    <p>
     <strong>
      母语
     </strong>
     ——梁小斌
    </p>
    <p>
     我用我们民族的母语写诗
     <br/>
     母语中出现土地  森林
     <br/>
     和最简单的火
     <br/>
     有些字令我感动
     <br/>
     但我读不出声
     <br/>
     我是一个见过两块大陆
     <br/>
     和两种文字相互碰撞的诗人
     <br/>
     为了找水
     <br/>
     我曾经忘却了我留在沙滩上的
     <br/>
     那些图案
     <br/>
     母语河流中的扬子鳄
     <br/>
     不会拖走它岸边的孩子
     <br/>
     如今，我重新指向那些象形文字
     <br/>
     我还在沙滩上画出水在潺潺流动
     <br/>
     的模样
     <br/>
     我不用到另一块大陆去寻找点滴
     <br/>
     还有太阳
     <br/>
     我是活在我们民族母语中的
     <br/>
     一个象形文字
     <br/>
     我活着
     <br/>
     我写诗
    </p>
   </blockquote>
   <blockquote>
    <p>
     <strong>
      汉语
     </strong>
     ——赵野
    </p>
    <p>
     一
    </p>
    <p>
     在这些矜持而没有重量的符号里
     <br/>
     我发现了自己的来历
     <br/>
     在这些秩序而威严的方块中
     <br/>
     我看到了汉族的命运
     <br/>
     节制，彬彬有礼，仿佛
     <br/>
     雾中的楼台，霜上的人迹
     <br/>
     是我们不致远行千里
     <br/>
     或者死于异地的疾病
    </p>
    <p>
     二
    </p>
    <p>
     祖先的语言，载着一代代的歌舞华筵
     <br/>
     值得我们青丝白发
     <br/>
     每个词都被锤炼千年，犹如
     <br/>
     每片树叶每天改变质地
     <br/>
     它们在笔下，在火焰和纸上
     <br/>
     仿佛刀锋在孩子的手中
     <br/>
     鱼倒挂树梢，鸟儿坠入枯井
     <br/>
     人头雨季落地，悄无声息
    </p>
   </blockquote>
