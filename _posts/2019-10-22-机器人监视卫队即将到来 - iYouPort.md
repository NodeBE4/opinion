   <img alt="" class="aligncenter size-full wp-image-5939 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/报告.png?resize=1100%2C737&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/报告.png?w=1200&amp;ssl=1 1200w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/报告.png?resize=300%2C201&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/报告.png?resize=768%2C515&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/报告.png?resize=1024%2C686&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/报告.png?resize=1100%2C737&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/报告.png?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="737" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/报告.png?resize=1100%2C737&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5939" data-recalc-dims="1" height="737" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/报告.png?resize=1100%2C737&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/报告.png?w=1200&amp;ssl=1 1200w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/报告.png?resize=300%2C201&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/报告.png?resize=768%2C515&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/报告.png?resize=1024%2C686&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/报告.png?resize=1100%2C737&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/报告.png?resize=272%2C182&amp;ssl=1 272w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   我们周围都是监控摄像头，可以随时记录我们。
  </p>
  <p class="graf graf--p">
   但在大多数情况下，当这些摄像机正在观察我们时，没有人在看那些摄像机观察或记录的东西，因为没有人会为这种耗时且单调的任务所需的安保人员支付费用。
  </p>
  <p class="graf graf--p">
   但想象一下，所有视频都在被密切观看 —— 数以百万计的监视者全天候监控它们；想象一下，这支军队是一群不需要报酬、从不厌倦、从不睡觉、从不错过任何细节的家伙，并且他们可以做到对所看到的一切完整回忆分毫不差。
  </p>
  <p class="graf graf--p">
   这样一支监视者部队可以仔细检查他们看到的每个人是否有“可疑”行为的迹象。
  </p>
  <p class="graf graf--p">
   通过无限的时间和无止境的注意力，他们还可以记录他们所看到的所有人的详细信息 —— 人们的衣着特征、人们的表情和情感、肢体语言，谁与谁正在联系，以及人们的每一项活动和动作。
  </p>
  <p class="graf graf--p">
   这种情况很快就会到来了。当然，这些守卫不会是人类 —— 他们是AI特工。
  </p>
  <p class="graf graf--p">
   ACLU
   <a class="markup--anchor markup--p-anchor" data-href="https://www.aclu.org/report/dawn-robot-surveillance" href="https://www.aclu.org/report/dawn-robot-surveillance" rel="noopener noreferrer" target="_blank">
    正在发布一份价值32亿美元的行业报告
   </a>
   ，该行业正在建立一种被称为“视频分析”的技术，该技术开始
   <strong class="markup--strong markup--p-strong">
    在全球范围内增强监控摄像头的能力，并将它们变成那种不眨眼的监视者。
   </strong>
  </p>
  <p class="graf graf--p">
   该技术使用尖端的、基于深度学习的人工智能，发展速度是如此之快，以至于
   <strong class="markup--strong markup--p-strong">
    这项技术的早期版本已经开始进入所有人的生活。
   </strong>
  </p>
  <p class="graf graf--p">
   一些汽车现在已经配备了
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Driver_Monitoring_System" href="https://en.wikipedia.org/wiki/Driver_Monitoring_System" rel="noopener noreferrer" target="_blank">
    仪表板摄像头
   </a>
   ，当驾驶员看起来昏昏欲睡时可以发出警报；
  </p>
  <p class="graf graf--p">
   当陌生人出现在家门口时，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/technology/2019/01/31/doorbells-have-eyes-privacy-battle-brewing-over-home-security-cameras/" href="https://www.washingtonpost.com/technology/2019/01/31/doorbells-have-eyes-privacy-battle-brewing-over-home-security-cameras/" rel="noopener noreferrer" target="_blank">
    门铃摄像机
   </a>
   会提醒 主人；
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2018/01/21/technology/inside-amazon-go-a-store-of-the-future.html" href="https://www.nytimes.com/2018/01/21/technology/inside-amazon-go-a-store-of-the-future.html" rel="noopener noreferrer" target="_blank">
    无收银台的商店
   </a>
   使用支持 AI 的摄像头监控客户，并在他们从货架上挑选物品时自动跟踪他们……
  </p>
  <p class="graf graf--p">
   在报告中，我们研究了这项技术的部署地点，以及公司声称可以提供的功能。
  </p>
  <p class="graf graf--p">
   我们还审查了计算机视觉科学家和其他研究人员的大量论文，以了解正在设想和开发的能力类型。
  </p>
  <p class="graf graf--p">
   我们发现，
   <strong class="markup--strong markup--p-strong">
    如果应用于监视和营销，计算机科学家所追求的能力将创造一个令人恐惧的洞察性监视世界。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️收集和存储视频以备不时之需的相机正在转变为可以主动监视所有人的设备，而且通常都是实时的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    一台无比强大的监控机器已经在我们周围成长起来，但是曾经，在很大程度上它是愚蠢和惰性的 —— 然而现在，从某种意义上来说，它正在“醒来”。
   </strong>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5940 jetpack-lazy-image" data-lazy-sizes="(max-width: 1080px) 100vw, 1080px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/识别.png?resize=1080%2C420&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/识别.png?w=1080&amp;ssl=1 1080w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/识别.png?resize=300%2C117&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/识别.png?resize=768%2C299&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/识别.png?resize=1024%2C398&amp;ssl=1 1024w" data-recalc-dims="1" height="420" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/识别.png?resize=1080%2C420&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1080"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5940" data-recalc-dims="1" height="420" sizes="(max-width: 1080px) 100vw, 1080px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/识别.png?resize=1080%2C420&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/识别.png?w=1080&amp;ssl=1 1080w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/识别.png?resize=300%2C117&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/识别.png?resize=768%2C299&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/识别.png?resize=1024%2C398&amp;ssl=1 1024w" width="1080"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   计算机正在变得越来越“好”，例如，所谓的“人类行为识别”功能。人工智能培训的数据集包括数以千计的计算机可识别行为 —— 例如戴上帽子、摘下眼镜、手伸进口袋、喝啤酒等等。
  </p>
  <p class="graf graf--p">
   研究人员也在努力创造在所谓的“
   <a class="markup--anchor markup--p-anchor" data-href="http://openaccess.thecvf.com/content_cvpr_2018/papers/Sultani_Real-World_Anomaly_Detection_CVPR_2018_paper.pdf" href="http://openaccess.thecvf.com/content_cvpr_2018/papers/Sultani_Real-World_Anomaly_Detection_CVPR_2018_paper.pdf" rel="noopener noreferrer" target="_blank">
    异常检测”
   </a>
   方面表现更好的人工智能技术 ——
   <strong class="markup--strong markup--p-strong">
    对任何“不寻常”、“异常”、“离经叛道”、或“非典型”的人发出警报；
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️它识别我们的情感、检测我们的感知、对我们行为的物理和社会背景深度理解，以及对我们运动模式广泛跟踪。
  </p>
  <p class="graf graf--p">
   想想这些技术的一些含义，特别是与人脸识别等其他技术结合使用时。
  </p>
  <p class="graf graf--p">
   例如，不难想象未来一位腐败的市长会对助手说：“这是我政府的敌人名单。让相机向我们发送这些人亲吻另一个人的所有实例，以及他们亲吻的人的身份。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    政府和公司可以使用AI代理来根据衣着、姿势、不寻常的特征或行为以及情绪来追踪谁是“可疑的”。
   </strong>
  </p>
  <p class="graf graf--p">
   那些以某种方式脱颖而出并吸引如此警惕的相机注意力的人，可能会发现自己立刻受到了骚扰、审讯、被驱逐出商店，或更糟糕。
  </p>
  <p class="graf graf--p">
   这些技术中的许多或大部分介于不可靠和完全虚假之间。
  </p>
  <p class="graf graf--p">
   然而，根据经验，这往往不会阻止这些技术被部署 —— 并且也不会避免伤害无辜的人民。
  </p>
  <p class="graf graf--p">
   而且，
   <strong class="markup--strong markup--p-strong">
    像许多技术一样，这些新监视能力的危险性将不可避免地落在已经处于不利地位的人群身上：有色人种、穷人、政治异议人士、和边缘人。
   </strong>
  </p>
  <p class="graf graf--p">
   目前仍处于计算机视觉革命的早期阶段，我们不知道人工智能将如何发展，但我们需要记住，人工智能的进步最终可能会非常迅速。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在不久的将来，我们可能最终生活在机器人军队的严密监视之下，他们比人类更危险
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这些人工智能监视者，如果不加以控制，可能会迅速增加到数十亿，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.aclu.org/blog/privacy-technology/consumer-privacy/coming-power-struggles-over-internet-things" href="https://www.aclu.org/blog/privacy-technology/consumer-privacy/coming-power-struggles-over-internet-things" rel="noopener noreferrer" target="_blank">
    代表企业和官僚权力扩展到我们生活的方方面面
   </a>
   ，密切跟踪每个人、并不断操纵每个人的行为。⚪️
  </p>
  <p class="graf graf--p">
   在这里阅读完整报告：

    https://www.aclu.org/sites/default/files/field_document/061819-robot_surveillance.pdf
   </a>
  </p>
