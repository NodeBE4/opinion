     <em>
      如何让艺术在社会运动中发挥推动作用？一位长期协助反抗运动的艺术家讲述了自己的故事……
     </em>

  <p class="graf graf--p graf--startsWithDoubleQuote">
   “青年的力量。教育者的力量。劳动者的力量。社区的力量。艺术建设的力量。” — — Keith Brown 奥克兰教育协会主席
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “文化是一个我们可以引入思想、将情感与具体变化联系在一起并赢得对我们价值观的热情的空间。艺术是我们改变叙事的地方，因为人们可以想象变化的外观和感觉。” — — Favianna Rodriguez 为奥克兰教师罢工贡献艺术品的艺术家之一。
  </p>
  <p class="graf graf--p">
   漂亮的艺术画横幅、丝网印花织物制作的旗帜以及罢工歌曲被一百名教师和支持者包围着，奥克兰教师工会主席 Keith Brown 召开新闻发布会
   <a class="markup--anchor markup--p-anchor" data-href="https://www.facebook.com/KatBrezler37/videos/698147043913221/" href="https://www.facebook.com/KatBrezler37/videos/698147043913221/" rel="noopener noreferrer" target="_blank">
    宣布
   </a>
   ，奥克兰教师将举行罢工投票。工会大厅办公室和停车场都挤满了人们制作的艺术品、并播放着艺术家制作的歌曲。这是罢工能胜利的一个关键部分。
  </p>
  <p class="graf graf--p">
   上个月已经胜利的洛杉矶教师罢工也举行了类似的前期艺术动员。教师和工会艺术组织者 Joe Brusky 说：“为洛杉矶创造的艺术在赢得胜利方面发挥了重要作用。”
  </p>
  <p class="graf graf--p">
   在过去的两个月里，奥克兰教育协会、当地艺术家和艺术组织者团队之间的大规模＃strike-ready 艺术制作合作一直在建立势头，参与并创作了数千件在公共场合使用的手工艺术品。导致罢工的行动将在警戒线上出现。
  </p>
  <p class="graf graf--p">
   旧金山湾区的社会参与艺术家 Favianna Rodriguez，Micah Bazant，Miriam Stahl，Eric Norbert，Melanie Cervantes 和 Jesus Baraza，Kim Cozier，Emory Douglas，Claudio Martinez，Jeanette Arellano 和 Paul Kjelland 都为参与罢工的教师和学生们提供了设计。
  </p>
  <p class="graf graf--p">
   Joe 与来自密尔沃基的艺术家组织者团队（Kim Cozier，Nicolas Lampert，Paul Kjelland，Claudio Martinez 和 Josie Osborne）一起为教师工会开创了这个雄心勃勃的大型艺术建筑模型。他们继续共同组织洛杉矶和奥克兰的艺术建筑，在那里我是奥克兰教师的当地艺术组织者。
  </p>
  <p class="graf graf--p">
   摄影记者 Joe 是洛杉矶大学 UTLA 教师工会官方摄影师，他在 Facebook，Instagram 和 Twitter 等社交媒体上记录和推出图像和故事。诗人 Lu Aya 与教师们一起编写并排演罢工歌曲。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-1240 jetpack-lazy-image" data-lazy-sizes="(max-width: 1067px) 100vw, 1067px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-139.png?resize=1067%2C705&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-139.png?w=1067&amp;ssl=1 1067w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-139.png?resize=300%2C198&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-139.png?resize=768%2C507&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-139.png?resize=1024%2C677&amp;ssl=1 1024w" data-recalc-dims="1" height="705" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-139.png?resize=1067%2C705&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1067"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-1240" data-recalc-dims="1" height="705" sizes="(max-width: 1067px) 100vw, 1067px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-139.png?resize=1067%2C705&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-139.png?w=1067&amp;ssl=1 1067w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-139.png?resize=300%2C198&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-139.png?resize=768%2C507&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-139.png?resize=1024%2C677&amp;ssl=1 1024w" width="1067"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   艺术家们为运动制作了各种横幅和标示，更多的是他们将教师、学生、家长和社区聚集在一起，共同奋斗。OEA 艺术构建没有什么不同，OEA 联合空间和他们的停车场被改造成了一个艺术化的空间，有数百人参加，人们经常停留六到八小时或更长时间。“
  </p>
  <p class="graf graf--p">
   二十年来，我一直在组织各种运动中的艺术构建，从1999年西雅图世界贸易组织的关闭和其他全球司法动员到摩洛哥工人联盟农场工人的胜利，到全球气候正义运动，将艺术带入我们的组织、教育和行动的中心。
  </p>
  <p class="graf graf--p">
   这种艺术构建对我们的工会来说是变革性的。除了建立大量的标志、海报和横幅广告外，我们还为工会运动带来了许多新的活动家和艺术家。我们动员了许多设计图像的当地艺术家，他们开始参加学校组织的会议、公开听证会以及围绕我们的问题组织的行动。
  </p>
  <p class="graf graf--p">
   艺术很受欢迎，我们收到了来自全国各地的要求，使用艺术并分享想法。这些艺术作品伴随着州内的听证会一起铺展各地，我们看到了在密尔沃基制作的最先进的照片，所以影响效果是巨大的。
  </p>
  <p class="graf graf--p">
   NEA 注意到我们在明尼阿波利斯举办的第一次 MKE 艺术构建和随后的各种艺术活动，并希望尝试在明尼阿波利斯为其全国种族和社会正义会议复制一些想法。我们在会议楼层画了15个降落伞横幅，为期两天，现场有来自全国各地的参与者。 NEA 非常喜欢这个想法，艺术的魅力调动了高度的参与。
  </p>
  <p class="graf graf--p">
   他们知道他们想要在全国范围内扩大这个想法，并且L.A.罢工恰好在同一时间出现。并且，从艺术构建中创建的照片可以用于在事件完成后生成社交媒体内容。由于L.A.需要大量的标示、横幅和海报，艺术构建的想法被全面使用，我们已经看到这些艺术作品为L.A.教师建立了一个成功的胜利样本。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-1241 jetpack-lazy-image" data-lazy-sizes="(max-width: 1067px) 100vw, 1067px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-140.png?resize=1067%2C708&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-140.png?w=1067&amp;ssl=1 1067w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-140.png?resize=300%2C199&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-140.png?resize=768%2C510&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-140.png?resize=1024%2C679&amp;ssl=1 1024w" data-recalc-dims="1" height="708" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-140.png?resize=1067%2C708&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1067"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-1241" data-recalc-dims="1" height="708" sizes="(max-width: 1067px) 100vw, 1067px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-140.png?resize=1067%2C708&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-140.png?w=1067&amp;ssl=1 1067w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-140.png?resize=300%2C199&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-140.png?resize=768%2C510&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-140.png?resize=1024%2C679&amp;ssl=1 1024w" width="1067"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   我认为为洛杉矶创作的艺术在赢得胜利方面发挥了重要作用。 Nisha Sethi“教师的：我们只能为人民服务”和 Ernesto Yerena 的“与L.A.教师站在一起”，都是根据实际的加州教育家的肖像设计的，真正有助于人性化教育工作者的斗争。
  </p>
  <p class="graf graf--p">
   我参加的每一场行动都可以看到这两个图像。这些设计经常能够解决非常复杂的教育问题，并使其易于理解和共情。产生的艺术品使所有人都能轻松理解。另一个强大的组成部分是激活当地艺术家的斗争。设计图像的艺术家可能是公共教育的支持者，通过制作设计，他们公开宣称他们的主张是争取变革。 L.A.图像已经具有标志性，永远具有特殊含义。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-1242 jetpack-lazy-image" data-lazy-sizes="(max-width: 1067px) 100vw, 1067px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-141.png?resize=1067%2C705&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-141.png?w=1067&amp;ssl=1 1067w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-141.png?resize=300%2C198&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-141.png?resize=768%2C507&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-141.png?resize=1024%2C677&amp;ssl=1 1024w" data-recalc-dims="1" height="705" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-141.png?resize=1067%2C705&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1067"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   如果没有记录，艺术构建和后续行动就会失效。文档至关重要。我使用社交媒体来讲述艺术构建的故事，因为它可以实时展开。这些故事不仅使那些参与者人性化，而且将艺术和斗争带入全世界关注者的视野中。
  </p>
  <p class="graf graf--p">
   我们的文档也引起了其他团体的兴趣，并吸引了可能以前从未支持过这场运动的人群。艺术版本提供了令人惊叹的吸引力，可以在社交媒体上轻松传播，并有助于围绕问题或事件组织评论和传播。
  </p>
  <p class="graf graf--p">
   不仅是工会分享故事和视觉效果，而且其他参与者也在创造自己的社交媒体故事。这会以指数级速度增加人们谈论和分享您的问题的数量。
  </p>
  <p class="graf graf--p">
   我希望人们看到我的照片时能感觉就像是自己成为了行动的一部分。我希望我的照片能够捕捉到当下的精髓，同时也要吸引那些等待被邀请加入的人。社交媒体必须在艺术品构建或罢工之前、期间和之后起到辅助作用。
  </p>
  <p class="graf graf--p">
   由于我在我的照片上放置了非商业公共许可证，因此其他工会、非营利组织和独立媒体都可以使用。我想为这些群体提供他们可以用来帮助建立无论身在何处都可以充分体验这场运动的高质量内容。
  </p>
  <p class="graf graf--p">
   这篇文章是一位美国的艺术家写的，希望能给中国的社运动员带来一点灵感。IYP 一直在强调联合的重大作用，各领域和社区之间的联盟将能使成功更容易获得。
  </p>
