  <p>
   <img alt="" class="aligncenter size-full wp-image-4377 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/数字脱衣.png?resize=1100%2C616&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/数字脱衣.png?w=1646&amp;ssl=1 1646w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/数字脱衣.png?resize=300%2C168&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/数字脱衣.png?resize=768%2C430&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/数字脱衣.png?resize=1024%2C574&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/数字脱衣.png?resize=1100%2C616&amp;ssl=1 1100w" data-recalc-dims="1" height="616" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/数字脱衣.png?resize=1100%2C616&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  </p>
  <p class="graf graf--p">
   全身扫描仪现在是机场的常规视线。大多数人都对这些简直是羞辱的搜索感到不满，就如上图所示，全裸。
  </p>
  <p class="graf graf--p">
   但似乎这种全裸的脱衣搜索在全世界变得越来越普遍了。最重要的是，不仅机场。
  </p>
  <p class="graf graf--p">
   例如，已经发现的，英国的犯罪受害者 — 不是罪犯，是受害者— 被要求移交他们的移动电话和社交媒体账户，以便警方可以下载搜索“相关”信息的所有内容。
  </p>
  <p class="graf graf--p">
   据
   <a class="markup--anchor markup--p-anchor" data-href="https://www.independent.co.uk/news/uk/crime/rape-victims-phones-police-investigation-disclosure-forms-cps-a8888376.html" href="https://www.independent.co.uk/news/uk/crime/rape-victims-phones-police-investigation-disclosure-forms-cps-a8888376.html" rel="noopener noreferrer" target="_blank">
    “独立报”报道
   </a>
   ：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    如果投诉人拒绝交出他们的数字设备，或者试图阻止任何个人信息被共享，警方称 “调查或起诉可能无法继续”……
   </em>
  </p>
  <p class="graf graf--p">
   还有人敢报案吗？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    特别值得关注的是涉嫌强奸的案件，在这些案件中，受害人一旦透露私密生活细节，很容易被辩护律师用来对付他们
   </strong>
   。
  </p>
  <p class="graf graf--p">
   一位女士在“卫报”中写道，她的案件被撤销了，并不是因为不太可能被定罪，而是因为，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/commentisfree/2019/apr/29/sexual-assault-case-dropped-refused-police-phone-rape" href="https://www.theguardian.com/commentisfree/2019/apr/29/sexual-assault-case-dropped-refused-police-phone-rape" rel="noopener noreferrer" target="_blank">
    她拒绝交出她的手机
   </a>
   以便让警方将其数据全部下载：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    我认为这一要求严重侵犯了我的人权。我手机上的内容是私密的，与案件无关。但我知道这些私人信息有可能被用来在立场上羞辱我和诋毁我。它将是“短裙”的数字版本 —— 那些支持强奸者的人经常说的那句话“因为你穿得太露了才会被强奸”
   </em>
  </p>
  <p class="graf graf--p">
   一份请愿书
   <a class="markup--anchor markup--p-anchor" data-href="https://you.38degrees.org.uk/petitions/stop-forcing-sexual-assault-survivors-to-hand-in-their-phones-in-investigations" href="https://you.38degrees.org.uk/petitions/stop-forcing-sexual-assault-survivors-to-hand-in-their-phones-in-investigations" rel="noopener noreferrer" target="_blank">
    已经开始呼吁
   </a>
   英国警方放弃这种做法。
  </p>
  <p class="graf graf--p">
   ⚠️但隐私国际组织的
   <a class="markup--anchor markup--p-anchor" data-href="https://privacyinternational.org/press-release/1700/new-report-reveals-uk-police-are-secretly-downloading-content-suspects-mobile" href="https://privacyinternational.org/press-release/1700/new-report-reveals-uk-police-are-secretly-downloading-content-suspects-mobile" rel="noopener noreferrer" target="_blank">
    一份报告显示
   </a>
   ，英国警方一直在“大规模”地从私人手机上下载内容。
  </p>
  <p class="graf graf--p">
   使用信息自由法请求，Privacy International 发现47名英国警察中有26名承认他们正在使用
   <strong class="markup--strong markup--p-strong">
    手机提取技术
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这份报告还提供了有关所用设备的一些细节。
  </p>
  <p class="graf graf--p">
   两家公司在这个*新兴领域*特别活跃。
   <strong class="markup--strong markup--p-strong">
    一个是 MSAB，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.msab.com/products/xry/" href="https://www.msab.com/products/xry/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     声称
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    其 XRY 软件可以访问超过 27,000 个设备和应用程序，甚至包括无人机
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在英国，该公司表示，97％以上的警察部队已经获得了XRY系统来提取数据 —— 远高于将其告诉隐私国际组织的 55％。
  </p>
  <p class="graf graf--p">
   MSAB
   <a class="markup--anchor markup--p-anchor" data-href="https://www.msab.com/company/" href="https://www.msab.com/company/" rel="noopener noreferrer" target="_blank">
    在美国
   </a>
   也很活跃。⚠️ 2010年，该公司就收到了美国政府有史以来最大的订单。订单量增加一倍，意味着美国已经成为MSAB的单一最大市场。
  </p>
  <p class="graf graf--p">
   另一个重要的玩家是 Cellebrite。据美国有线电视新闻网报道，⚠️
   <a class="markup--anchor markup--p-anchor" data-href="https://money.cnn.com/2016/03/31/technology/cellebrite-fbi-phone/" href="https://money.cnn.com/2016/03/31/technology/cellebrite-fbi-phone/" rel="noopener noreferrer" target="_blank">
    “Cellebrite 是联邦调查局的首选手机黑客技术来源
   </a>
   ”。
  </p>
  <p class="graf graf--p">
   ⚠️其产品的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cellebrite.com/en/products/ufed-ultimate/" href="https://www.cellebrite.com/en/products/ufed-ultimate/" rel="noopener noreferrer" target="_blank">
    一项关键功能是
   </a>
   声称能够绕过模式、密码或 PIN 锁定，并绕过手机上的加密。设备上保存的数据与社交媒体上的公共数据相结合，以提供更丰富的数据集：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    轻松访问、查看并将公共领域社交媒体和基于云的数据整合到您的调查中，例如 位置信息、个人资料、图像、文件和来自热门应用程序（包括 Facebook，Twitter 和 Instagram 等）的所有社交通信。
   </em>
  </p>
  <p class="graf graf--p">
   美国最高法院在对 Riley 诉加利福尼亚案中进行了具有里程碑意义的裁决，虽然手机上的数据不能免于搜查，但通常需要先发出逮捕令，即使与逮捕相关也是如此。
  </p>
  <p class="graf graf--p">
   然而，最近另一个法院
   <a class="markup--anchor markup--p-anchor" data-href="https://www.techdirt.com/articles/20190331/18014441916/7th-circuit-punts-border-smartphone-searches-says-riley-decision-doesnt-affect-anything.shtml" href="https://www.techdirt.com/articles/20190331/18014441916/7th-circuit-punts-border-smartphone-searches-says-riley-decision-doesnt-affect-anything.shtml" rel="noopener noreferrer" target="_blank">
    裁定
   </a>
   ，这不适用于美国边境。
  </p>
  <p class="graf graf--p">
   不止英美。
  </p>
  <p class="graf graf--p">
   加拿大当局也采取同样的做法。
  </p>
  <p class="graf graf--p">
   当一名加拿大商人最近降落在多伦多机场时，⚠️边防官员要求他交出密码以便搜索他的手机和笔记本电脑。
  </p>
  <p class="graf graf--p">
   当这名商人拒绝时，
   <strong class="markup--strong markup--p-strong">
    边防官员直接没收了他的设备，并说设备将被送到政府实验室试图获取访问权限 —— 显然，将会使用 MSAB 和 Cellebrite 提供的那种工具，绕过密码保护
   </strong>
   。
  </p>
  <p class="graf graf--p">
   正如加拿大广播公司报道的那样，该国的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cbc.ca/news/business/cbsa-boarder-security-search-phone-travellers-openmedia-1.5119017" href="https://www.cbc.ca/news/business/cbsa-boarder-security-search-phone-travellers-openmedia-1.5119017" rel="noopener noreferrer" target="_blank">
    边境服务部门声称，他们有权在没有手令的情况下搜查过境的任何电子设备
   </a>
   ，以获取所谓的犯罪证据，“就像他们对行李的搜查一样”。然而，一些加拿大律师认为这是违宪的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    无论当地法律如何，这里都存在一个与全球相关的更大问题
   </strong>
   。
  </p>
  <p class="graf graf--p">
   智能手机不再仅仅是一个拨打电话的设备。在硬件方面，它是一个口袋大小的超级计算机、照相机、录像机和录音机、GPS设备……以及更多。
  </p>
  <p class="graf graf--p">
   将此与通常可用于主平台的数百万个应用程序相结合，它是有史以来最强大的个人设备。
  </p>
  <p class="graf graf--p">
   智能手机不仅仅是日常生活的一部分，而是生活本质的存储。
   <strong class="markup--strong markup--p-strong">
    当局抓住你的手机时，他们抓住的是你整个数字自我。
   </strong>
  </p>
  <p class="graf graf--p">
   臭名昭著的红衣主教黎塞留曾说过：“给我这个世界上任何人写下的六行字，我肯定能从中找到绞死他的理由”。
  </p>
  <p class="graf graf--p">
   现在已经不是仅仅六行字了。想象一下，从您的手机中提取的六千兆字节最私密的数据可以做些什么？
  </p>
  <p class="graf graf--p">
   鉴于这种材料令人难以置信的丰富性，毫无疑问当局有明确的动机想要通过移动设备尽可能多地下载和使用尽可能多的数据。
  </p>
  <p class="graf graf--p">
   但是，这种丰富性意味着必须提高移动设备的标准安全性，以便那些蛮力提取工具不起作用，并在政府设法获得访问权限时对其内容进行法律保护。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.privateinternetaccess.com/blog/2019/05/digital-strip-searches-police-increasingly-downloading-personal-data-from-smartphones-including-from-crime-victims/" href="https://www.privateinternetaccess.com/blog/2019/05/digital-strip-searches-police-increasingly-downloading-personal-data-from-smartphones-including-from-crime-victims/" rel="noopener noreferrer" target="_blank">
    “Digital strip-searches”: police increasingly downloading everything from smartphones — including from crime victims
   </a>
  </p>
