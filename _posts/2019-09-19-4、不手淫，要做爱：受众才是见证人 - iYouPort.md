  <p class="graf graf--p">
   一些活动人士变得越来越封闭，这可不是好事；人们各自在“自己的领域”里奔忙，在团队中彼此照应，这都很好，但是，缺失了一个重要的东西 —— 一种能让你有所作为的效应。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    要知道，在评估特定行动的成功与否时，您对您的创意海报、或新闻稿、或公民不服从的看法，并不重要。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    重要的是观众的想法。
   </strong>
  </p>
  <p class="graf graf--p">
   抗议只是为了自我表达和自我满足吗？这是手淫的政治等同物。经过精心设计和执行以引起目标受众的充分反应或回应的政治行为 —— 那才叫做爱。
  </p>
  <p class="graf graf--p">
   如果你已经想出了一些非常棒的，即时的行动，现在正试图找出你想要用它来达到的目标，那么你就是在倒退。
  </p>
  <p class="graf graf--p">
   💡创造性政治行动的关键不仅仅是创造性，而是要对特定受众产生预期的影响。
  </p>
  <p class="graf graf--p">
   首先确定您的目标受众，然后集中讨论行动以有效传达您的信息。
  </p>
  <p class="graf graf--p">
   贾斯汀比伯怎么样？效果肯定很棒，唯一的条件在于，你的预期受众是美国退休人员协会的成员。
  </p>
  <p class="graf graf--p">
   Andy Warhol 也许会令你和很多人陶醉，如果你的预期受众是中国年轻人，那他真的不如二次元。
  </p>
  <p class="graf graf--p">
   ⚠️请记住，没有最合适的受众群体，只有适合您特定目标的受众群体。
  </p>
  <p class="graf graf--p">
   💡请尝试这个基本公式：如果他们相信C，我们可以让A做B，A是你的受众，B是你的目标，C是你的信息。设计一个或多个行动，让A相信C。
  </p>
  <p class="graf graf--p">
   如果您的核心策略和行动没有明确的和战略性地设计，以便对目标受众产生
   <strong class="markup--strong markup--p-strong">
    预期的
   </strong>
   影响，那么您就不具备战略性。
  </p>
  <p class="graf graf--p">
   贾斯汀比伯可能很惊人，并且 YouTube 上的点击量是天文数字，但它能否触动你的目标受众呢？这才是你唯一需要考虑的问题。
  </p>
  <p class="graf graf--p">
   传统艺术家不一定会担心他们的观众体验。对他们来说，创造性的自我表达可能就足够了。
  </p>
  <p class="graf graf--p">
   但是，⚠️对于政治艺术家来说，观众就是一切。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    政治艺术的目的就是体验它的人的反应
   </strong>
   。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4465 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中.jpg?resize=1100%2C733&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中.jpg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中.jpg?resize=768%2C512&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="733" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中.jpg?resize=1100%2C733&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4465" data-recalc-dims="1" height="733" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中.jpg?resize=1100%2C733&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中.jpg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中.jpg?resize=768%2C512&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中.jpg?resize=272%2C182&amp;ssl=1 272w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   互联网新媒体时代、尤其是算法操纵的回音壁时代，也许你会认为“定位受众”可以轻而易举？但并非如此。
  </p>
  <p class="graf graf--p">
   如果人们上网的90%动机是娱乐（的确是，而且实际上很可能高于90%），那么你的任何严肃行动都将被娱乐化，从而过目即忘。更多参见这里《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%9c%a8%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e4%b8%8a%e6%8a%97%e8%ae%ae%e6%9c%89%e4%bb%80%e4%b9%88%e9%97%ae%e9%a2%98%ef%bc%9f/" href="https://www.iyouport.org/%e5%9c%a8%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e4%b8%8a%e6%8a%97%e8%ae%ae%e6%9c%89%e4%bb%80%e4%b9%88%e9%97%ae%e9%a2%98%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    在社交媒体上”抗议”有什么问题？
   </a>
   》
  </p>
  <p class="graf graf--p">
   简单说，
   <strong class="markup--strong markup--p-strong">
    利用社交媒体时，你不一定就真的能*抵达*你的受众。甚至你都不知道真正的受众在哪。
   </strong>
  </p>
  <p class="graf graf--p">
   但是，如果媒体“是你的”呢？会不会不一样……
  </p>
  <p class="graf graf--p">
   这就是媒体劫持。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    媒体劫持是指为了自己的目的而颠覆对手的奇观。
   </strong>
  </p>
  <p class="graf graf--p">
   政治家、巨头企业和游说者拥有更大的公共关系预算和品牌吸引力，可以吸引并控制媒体参与其分阶段的宣传活动。
  </p>
  <p class="graf graf--p">
   💡然而，通过精心策划的创意干预，作为反对派，您也可以重新聚焦事物并突出故事的另一面。
  </p>
  <p class="graf graf--p">
   有几种不同的方法来设计成功的媒体劫持。
  </p>
  <p class="graf graf--p">
   首先是
   <strong class="markup--strong markup--p-strong">
    最简单地征服媒体
   </strong>
   。
  </p>
  <p class="graf graf--p">
   1991年第一次海湾战争期间，其中一个最直接和大胆的例子发生在抗艾滋病组织 ACT UP 在黄金时段的新闻直播期间，抗议者突然进入了CBS电视工作室，并接管了该节目，只有几秒钟的时间，他对着镜头高呼：“打击艾滋病，而不是阿拉伯人”！
  </p>
  <p class="graf graf--p">
   这是一个伟大的经典镜头。想想看，您所在的国家有哪些收视率非常高的大型直播节目吗……
  </p>
  <p class="graf graf--p">
   如果你觉得这太难了，“戒备森严的地方即便有内应也难以成功”……不，别把它想得这么复杂，事实上有很多机会可以实现媒体劫持。
  </p>
  <p class="graf graf--p">
   另一个选择是，
   <strong class="markup--strong markup--p-strong">
    使用你的对手的平台来讲述你自己的故事。
   </strong>
  </p>
  <p class="graf graf--p">
   2007年，Kleenex 搞了一个昂贵的公关特技，他们在街上采访人们为了他们正在制作的商业广告，让参与者哭着说：“我需要面巾纸。”
  </p>
  <p class="graf graf--p">
   绿色和平组织的积极人士得知这个消息后就去悄悄地排队，以接受采访，但是不哭，以严肃的表情反对伐木造纸。
  </p>
  <p class="graf graf--p">
   他们在当天剩下的时间里成功地关闭了那场拍摄，并且有关该行动的视频变成了病毒式传播的效果。
  </p>
  <p class="graf graf--p">
   💡
   <strong class="markup--strong markup--p-strong">
    具有创造力的媒体劫持是使用你的对手自己的故事来攻击他们，在假设的角度破坏它们
   </strong>
   ，详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/4%e3%80%81%e5%88%ab%e6%8a%8a%e6%b3%a8%e6%84%8f%e5%8a%9b%e9%9b%86%e4%b8%ad%e5%9c%a8%e6%8a%b5%e6%8a%97%e4%b8%8a%ef%bc%9a%e9%a2%84%e8%ae%be%e6%80%a7%e5%b9%b2%e9%a2%84%e6%88%98%e6%9c%af/" href="https://www.iyouport.org/4%e3%80%81%e5%88%ab%e6%8a%8a%e6%b3%a8%e6%84%8f%e5%8a%9b%e9%9b%86%e4%b8%ad%e5%9c%a8%e6%8a%b5%e6%8a%97%e4%b8%8a%ef%bc%9a%e9%a2%84%e8%ae%be%e6%80%a7%e5%b9%b2%e9%a2%84%e6%88%98%e6%9c%af/" rel="noopener noreferrer" target="_blank">
    别把注意力集中在抵抗上：预设性干预战术
   </a>
   》。
  </p>
  <p class="graf graf--p">
   我们讲述过这样的成功案例：共和党作秀被活动家劫持的特技，在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%8a%97%e8%ae%ae%e6%98%af%e9%9d%9e%e6%b3%95%e7%9a%84%ef%bc%8c%e9%82%a3%e5%b0%b1%e8%ae%a9%e6%97%a5%e5%b8%b8%e7%94%9f%e6%b4%bb%e6%88%90%e4%b8%ba%e6%8a%97%e8%ae%ae%ef%bc%88%e4%b8%80/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%8a%97%e8%ae%ae%e6%98%af%e9%9d%9e%e6%b3%95%e7%9a%84%ef%bc%8c%e9%82%a3%e5%b0%b1%e8%ae%a9%e6%97%a5%e5%b8%b8%e7%94%9f%e6%b4%bb%e6%88%90%e4%b8%ba%e6%8a%97%e8%ae%ae%ef%bc%88%e4%b8%80/" rel="noopener noreferrer" target="_blank">
    为你的对手制造决策困境
   </a>
   》。活动家的表演成功地将 “税收=压迫”这一信息深入人心了。
  </p>
  <p class="graf graf--p">
   同样，在2006年，热带雨林行动网络的积极人士制作了假的新闻通行证，穿上西装，潜入了洛杉矶车展。
  </p>
  <p class="graf graf--p">
   当时，通用汽车（General Motors）首席执行官里克•瓦格纳（Rick Wagoner）正在就通用的“环保”汽车如何了不起发表主题演讲。
  </p>
  <p class="graf graf--p">
   演讲全是废话，但无所谓，关键在于，RAN 的活动家们大模大样地走上了舞台，接过了麦克风，假装自己是主持人。
  </p>
  <p class="graf graf--p">
   他们向瓦格纳表示祝贺，然后告诉观众，他们很高兴地宣布，通用汽车准备以书面形式落实承诺，并展开了他们要求他签署的超大“承诺”。
  </p>
  <p class="graf graf--p">
   瓦格纳只有两种选择：1）签名，给活动家想要得到东西，或者 2）拒绝，证明他自己只是个骗子。
  </p>
  <p class="graf graf--p">
   他选择了第二种。一众媒体都疯了。
   <strong class="markup--strong markup--p-strong">
    超过700家媒体刊登了有关通用汽车公开撒谎的报道
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    关键原则 ——
   </strong>
  </p>
  <p class="graf graf--p">
   媒体报道为活动家提供了一个独特的机会，不仅可以让对手参与活动家的比赛，而且可以实际发号施令并重新讨论。
  </p>
  <p class="graf graf--p">
   通过将目标放在媒体面前的现场，活动家可以快速聚焦受众眼球，重塑公众如何看待“好人”和“坏人”，并将他们的对手的故事扔上头条丑闻。
  </p>
  <p class="graf graf--p">
   💡当你在盛大的戏剧表演中推倒一棵树时，要确保正确的人看到了它，并且他们会听到一声巨响。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    注意事项 ——
   </strong>
  </p>
  <p class="graf graf--p">
   媒体是一个极其不平衡的竞争领域。准确而富有同情心的媒体报道通常基于活动家与记者保持的良好关系。因此请注意，要让您的行为不会疏远您依仗媒体专业人士。
  </p>
  <p class="graf graf--p">
   此外，关联我们早期的分析《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ae%a9%e5%ae%83%e6%b0%91%e4%b8%bb%ef%bc%9a%e5%9c%a8%e7%9c%9f%e7%9b%b8%e4%ba%8b%e4%b8%9a%e4%b8%ad%e6%9e%84%e6%83%b3%e6%9c%aa%e6%9d%a5%e5%9b%be%e6%99%af%ef%bc%88%e4%b8%8a%ef%bc%89/" href="https://www.iyouport.org/%e8%ae%a9%e5%ae%83%e6%b0%91%e4%b8%bb%ef%bc%9a%e5%9c%a8%e7%9c%9f%e7%9b%b8%e4%ba%8b%e4%b8%9a%e4%b8%ad%e6%9e%84%e6%83%b3%e6%9c%aa%e6%9d%a5%e5%9b%be%e6%99%af%ef%bc%88%e4%b8%8a%ef%bc%89/" rel="noopener noreferrer" target="_blank">
    让它民主：在真相事业中构想未来图景（上）
   </a>
   》《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ae%a9%e5%ae%83%e6%b0%91%e4%b8%bb%ef%bc%9a%e6%9b%bf%e4%bb%a3%e5%aa%92%e4%bd%93%e4%bd%9c%e4%b8%ba%e6%8a%97%e4%ba%89%e4%b9%8b%e6%ba%90%ef%bc%88%e4%b8%8b%ef%bc%89/" href="https://www.iyouport.org/%e8%ae%a9%e5%ae%83%e6%b0%91%e4%b8%bb%ef%bc%9a%e6%9b%bf%e4%bb%a3%e5%aa%92%e4%bd%93%e4%bd%9c%e4%b8%ba%e6%8a%97%e4%ba%89%e4%b9%8b%e6%ba%90%ef%bc%88%e4%b8%8b%ef%bc%89/" rel="noopener noreferrer" target="_blank">
    替代媒体作为抗争之源（下）
   </a>
   》⚪️
  </p>
