  <p class="graf graf--p">
   你能想象一个美国的城市秘密建立一个可追踪每个人的中国式公共监控网络吗？
  </p>
  <p class="graf graf--p">
   最好想象一下，因为它已经发生了。
  </p>
  <p class="graf graf--p">
   当我去年写下“
   <a class="markup--anchor markup--p-anchor" data-href="https://massprivatei.blogspot.com/2018/10/covert-facial-recognition-street-lights.html?q=spying+street+lights" href="https://massprivatei.blogspot.com/2018/10/covert-facial-recognition-street-lights.html?q=spying+street+lights" rel="noopener noreferrer" target="_blank">
    隐藏的面部识别路灯来到你附近的城市
   </a>
   ”时，还从未想过我的文章会如此迅速地成为现实。
  </p>
  <p class="graf graf--p">
   San Diego Reader
   <a class="markup--anchor markup--p-anchor" data-href="https://www.sandiegoreader.com/news/2019/mar/11/hacker-stalls-port-san-diegos-video-spy-network/" href="https://www.sandiegoreader.com/news/2019/mar/11/hacker-stalls-port-san-diegos-video-spy-network/" rel="noopener noreferrer" target="_blank">
    最近的一篇文章
   </a>
   揭示了黑客如何发现圣地亚哥港和 BriefCam 之间的电子邮件。这些电子邮件显示，
   <strong class="markup--strong markup--p-strong">
    执法部门秘密地使用了400个面部识别监控摄像头网络来识别每个人
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在这里看到视频：
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="314" src="//www.youtube.com/embed/JxgxBH0JkkU" width="560">
   </iframe>
  </p>
  <p class="graf graf--p">
   去年，BriefCam 宣布了实时面部识别监控技术的“突破性”发展：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    强大的多摄像头搜索功能可以快速准确地识别男性，女性，儿童和车辆，使用25个类别和属性，人脸识别、外观相似性、肤色、体量大小、速度、路径、方向和停留时间
   </em>
   。
  </p>
  <p class="graf graf--p">
   根据
   <a class="markup--anchor markup--p-anchor" data-href="https://www.sandiegoreader.com/news/2019/feb/20/san-diegos-street-lights-spy/" href="https://www.sandiegoreader.com/news/2019/feb/20/san-diegos-street-lights-spy/" rel="noopener noreferrer" target="_blank">
    另一篇文章
   </a>
   ，圣地亚哥市正在使用GE的 CityIQ 路灯来窃听每个人。
  </p>
  <p class="graf graf--p">
   在2017年，民权倡导者已经致函市长和市议会，要求该市不要安装GE的路灯：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    能够监视和记录居民一举一动的设备严重侵犯隐私权、危害言论自由，并且不成比例地影响有色人种社区
   </em>
   。
  </p>
  <p class="graf graf--p">
   但正如文章所揭示的那样，
   <strong class="markup--strong markup--p-strong">
    圣地亚哥忽视了公众的担忧，并秘密安装了3200个间谍GE路灯
   </strong>
   。
  </p>
  <p class="graf graf--p">
   GE的间谍路灯实际上“将城市变成了一个隐形的实验室，用于嵌入式智能设备，采用 DEA，ICE 和其他维稳机构经常使用的设备进行收集。”（要了解有关圣地亚哥间谍路灯的更多信息，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.sandiegoreader.com/news/2019/mar/11/ticker-public-meeting-san-diegos-surveillance/" href="https://www.sandiegoreader.com/news/2019/mar/11/ticker-public-meeting-san-diegos-surveillance/" rel="noopener noreferrer" target="_blank">
    请点击此处
   </a>
   。）
  </p>
  <p class="graf graf--p">
   来自 BriefCam 西部地区销售总监 Erik Wade 的电子邮件揭示了谁在背后真正支持圣地亚哥的公共监控网络。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “我目前正在与 SDPD 合作，在整个城市的新实时犯罪中心（RTCC）部署 BriefCam，这将极大地帮助你，因为这些监视相机的覆盖会互相受益”，Wade 说
   </em>
   。
  </p>
  <p class="graf graf--p">
   那么，BriefCam 和 DHS 到底有多接近？
  </p>
  <p class="graf graf--p">
   去年，BriefCam 宣布他们赢得了2018年’ASTORS’国土安全奖的金奖，称其为“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.briefcam.com/company/press-releases/briefcam-honored-with-2018-gold-astors-homeland-security-award/" href="https://www.briefcam.com/company/press-releases/briefcam-honored-with-2018-gold-astors-homeland-security-award/" rel="noopener noreferrer" target="_blank">
    著名的计划
   </a>
   ，旨在尊重杰出的政府和供应商解决方案。”
  </p>
  <p class="graf graf--p">
   我希望我能说，我很惊讶私营公司正在与国土安全部合作，但这种状况并不令人吃惊，因为自9/11以来一直都是如此。
  </p>
  <p class="graf graf--p">
   令人惊讶的是了解到了圣地亚哥执法部门是如何秘密创建公共监视网络的。BriefCam 的“突破性”声明是一个令人难以置信的承认。
  </p>
  <p class="graf graf--p">
   圣地亚哥的执法部门正在使用 BriefCam 创建“精确的人脸识别”，使用从视频、外部图像源和预定义的监视列表中提取的数字图像，快速准确地识别任何当局感兴趣的人。
  </p>
  <p class="graf graf--p">
   WatchListing 是 BriefCam 的主要卖点，他们说：“我们的可扩展监视列表管理实现了快速而强大的规则配置。” 也就是中国的监视黑名单咯。
  </p>
  <p class="graf graf--p">
   马萨诸塞州伊普斯威奇镇
   <a class="markup--anchor markup--p-anchor" data-href="https://www.briefcam.com/company/press-releases/the-city-of-ipswich-drives-citywide-transformation-with-briefcam-video-analytics/" href="https://www.briefcam.com/company/press-releases/the-city-of-ipswich-drives-citywide-transformation-with-briefcam-video-analytics/" rel="noopener noreferrer" target="_blank">
    最近宣布
   </a>
   ，他们已经成为美国第一个建立公共监视网络的城镇。
  </p>
  <p class="graf graf--p">
   将数十万儿童及其家人列入监视黑名单是一件大事。
  </p>
  <p class="graf graf--p">
   去年，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.briefcam.com/company/press-releases/axis-communications-briefcam-lenel-milestone-and-ruckus-collaborate-to-bring-greater-security-and-connectivity-to-the-2018-little-league-baseball-world-series/" href="https://www.briefcam.com/company/press-releases/axis-communications-briefcam-lenel-milestone-and-ruckus-collaborate-to-bring-greater-security-and-connectivity-to-the-2018-little-league-baseball-world-series/" rel="noopener noreferrer" target="_blank">
    BriefCam 宣布
   </a>
   他们使用监控摄像头识别2018年在宾夕法尼亚州南威廉斯波特举行的世界少棒联盟系列赛中的所有儿童及其家人。“每年都有数十万人前往威廉斯波特参加这场赛事”。
  </p>
  <p class="graf graf--p">
   就如 Vanity Fair 所警告的：
   <strong class="markup--strong markup--p-strong">
    将我们的城市变成中国镜像的真正目的是“
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.vanityfair.com/news/2018/07/china-surveillance-state-artificial-intelligence" href="https://www.vanityfair.com/news/2018/07/china-surveillance-state-artificial-intelligence" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     让人们更加顺从
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ”，简而言之就是：监视政治
   </strong>
   。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    S
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.activistpost.com/2019/03/san-diego-has-been-turned-into-a-massive-chinese-style-public-surveillance-network.html" href="https://www.activistpost.com/2019/03/san-diego-has-been-turned-into-a-massive-chinese-style-public-surveillance-network.html" rel="noopener noreferrer" target="_blank">
    an Diego Has Been Turned Into A Massive Chinese-Style Public Surveillance Network:
   </a>
   the real purpose behind turning our cities into a mirror image of China is to “make people more obedient” and that is surveillance politics in a nutshell.
  </p>
