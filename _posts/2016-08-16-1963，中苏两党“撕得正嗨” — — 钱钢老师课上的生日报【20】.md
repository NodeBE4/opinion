      文／宋建锋
     </p>
<blockquote class="iz ja jb">
<p class="ib ic jc id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="ec9c">
<strong class="id jd">
        钱钢老师附言
       </strong>

<p class="ib ic jc id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="d5c8">
       在北大深圳研究生院财经传媒硕士班，宋建锋同学放了大招，他的生日报演示，竟是一个自己制作的动画短片：《1963，我想和老爸谈谈》。片子是对1963国际国内的鸟瞰。整理图文版时，他继续研究旧报，又有新的发掘。我们先看片子，再读他带来的中苏恶斗的故事。
      </p>
</blockquote>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="966c">
      我的爸爸出生在1963年9月5日的开封，一个曾为国都省府的城市。作为老一辈的开封人，他会沿着郑开大道，时不时地来到这些街上走走转转，儿时记忆中开封的繁荣浮现在脑际，一些老建筑映入眼帘，恍若回到一个甲子之前的斑驳岁月。
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="8f68">
      久居闹市，我突然很想回到爸爸出生的那个年代，特意找来了1963年9月5日出版的《人民日报》。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="79f2">
      和光一一样，我首先看到的是头版文章《首都盛会热烈欢迎印度尼西亚共产党主席艾地和代表团》，印尼共和中共在1963年的关系之亲密可窥一斑。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="118a">
      然而再往下翻，到了第六版，我发现全版刊登着一篇题为《请看苏联报刊是怎样诽谤和攻击中国》（下文省略为《请看》）的文章。这篇长文洋洋洒洒，竟然占据了6、7、8整整三个版面。
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="b13e">
      搜索资料，我发现除了这一天，9月2日和3日的人民日报也发表了“苏报刊攻击和污蔑中国共产党”的一部分社论和文章。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="03c5">
      “诽谤”和“攻击”听上去着实严重，中国到底为什么要跟“老大哥”苏联杠上呢？
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="9c8e">
      文章开头提到，一切的缘起是一封“苏共中央的公开信”。这封信由苏联共产党中央委员会发表于当年7月14日莫斯科中苏会谈期间。当时距苏共二十大赫鲁晓夫批判斯大林已经过去七年，中苏在共产主义运动的路线和策略上分歧渐生，在此信前，也早已有多次“探讨式”的“信件往来”。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="d781">
      但文章表示，自这封公开信发布以后，“苏联领导人就开动了他们的宣传机器，发动了一个锣鼓喧天的反华运动”。看来，友好讨论的氛围已经消失，“唇枪舌剑”开始了。十五天里，“苏联中央一级的报刊上，就刊登了二百八十六篇攻击中国共产党的社论、文章和其他形式的材料”。
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="ccad">
      这些材料的原文，我没有搜到 — — 但没关系，人民日报编者“不惜篇幅”，“为了让读者看一看这些谣言和诽谤制造者堕落到什么地步”，选择了发表在苏联《共产党人》杂志上一篇典型社论，让我们得以一窥究竟。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="59f4">
      这篇题为《争取创造性的马克思列宁主义的胜利,反对修正世界共产主义运动的方针》（下文省略为《方针》）的文章提出了与中共的四个分歧，分别是：战争与和平，世界社会主义体系的发展和作用，反对“个人迷信”的意识形态和实践，和世界工人运动和民族解放斗争的战略和策略。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="e382">
<strong class="id jd">
       要战争还是要和平？
      </strong>
</p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="0e45">
      冷战期间战争与和平的问题，自然是美苏间核战的威胁。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="037f">
      苏联文章《方针》说道：“实际局势表明,在我们时代，拯救人类免于热核战争的斗争任务有着头等的意义” 。这与今天我们说，“和平与发展”是世界的主题，似乎是一个道理。而“中国同志持有另一种观点：‘原子弹是纸老虎’，它‘并不可怕’”，甚至有“某些负责的中国领导人声称,在消灭帝国主义的战争中有可能牺牲一半中国人民,甚至牺牲人类的半数,这也不是偶然的。”
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="6520">
      “原子弹是纸老虎”这句话，大概是来自毛主席“帝国主义和一切反动派都是纸老虎”的著名论断。据苏联领导人赫鲁晓夫的回忆录称，“牺牲半数”的话是1957年毛在莫斯科说的。人民网也曾报道过这段史实。
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="bba7">
      1963年，中国距第一颗原子弹成功爆炸仅一年之遥。作为一个正在加紧研制原子弹的国家，此类言论造成的国际影响，不啻于今天有个正在发展核武器的国家说，原子弹试验成功后第一个攻击美国。然而，对于这样的说法，《请看》一文在开头表示，都是“放肆的攻击”和“污蔑”。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="8e09">
<strong class="id jd">
       什么是世界社会主义体系？它和工人运动、民族解放斗争的关系是什么？
      </strong>
</p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="678a">
      当中国提出对“世界革命进程的主导力量”的不同意见时，苏联作为社会主义阵营的领头羊，显然感到了在理论层面捍卫自身权威的重要性。《方针》强调道，“社会主义体系，是世界革命运动的决定性力量”。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="f89e">
      苏共的理论中，这一体系要包括五大洲的所有无产阶级力量。文章批评，中国 “纠缠不休地提出”，世界革命过程的“震心”是“展开民族解放运动的亚洲、非洲和拉丁美洲地区”，强调领导亚非拉地区人民起来革命的重要性，“否认发达的资本主义国家中工人运动的革命可能性，并且硬说什么世界革命的命运决定性地只取决于民族解放运动的结局，这是根本不正确的”。他们认为，中共这样做，是为了“讨好民族解放运动,是想强迫它接受自己的领导权”。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="5ead">
      苏共还提出，社会主义要与帝国主义进行力量对比，其决定因素是经济因素。因此，“两个世界体系在人类活动的决定性领域 — — 物质生产领域，和在经济领域中的斗争，在现阶段具有特别的意义。”因此它也指责中共在“故意回避关于社会主义国家经济成就的国际意义的问题”。这不由让我想到邓小平在改革开放时的名言，“发展才是硬道理”。
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="c219">
      看来，苏共一心想谋经济发展，而中共则要将革命进行到底。一个指责对方“背离了无产阶级的阶级态度，而代之以小资产阶级的态度”；另一个将这样的指控称为“扣帽子”，并认为“这些对中国的攻击，同苏共中央的公开信一样，采用了资产阶级政客的颠倒是非、造谣惑众的手法。”
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="1ec4">
<strong class="id jd">
       “个人崇拜”要不要？
      </strong>
</p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="62aa">
      反对“个人迷信”，是苏共二十大决议中的倡议内容。大会上，赫鲁晓夫发表秘密报告《关于个人崇拜及其后果》，抨击了前任苏共领导人斯大林在位期间的严重错误。
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="2f00">
      《方针》引用人民日报编辑部1956年12月发表的文章《再论无产阶级专政的历史经验》，论述中国领导人曾承认反对个人迷信的必要性，现在“却公开维护个人迷信，并且谴责反对个人迷信的斗争”。
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="52b5">
      而实际上，这篇由中共中央授意的文章虽然称赞“苏联共产党第二十次代表大会，在破除关于斯大林的迷信、揭露斯大林错误的严重性、消除斯大林错误的后果方面，表现了巨大的决心和勇气”，但更重要的，却是指出了斯大林尽管在后期犯了一些严重错误，他的一生仍是“伟大的马克思列宁主义革命家的一生”，并提出“要正确对待斯大林以及一切在他的影响下犯了类似错误的同志”，分清“敌我矛盾”和“人民内部矛盾”。
     </p>
