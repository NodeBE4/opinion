  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e5%92%8c%e4%b8%ad%e5%9b%bd%e5%9c%a8%e7%9b%91%e8%a7%86%e5%85%ac%e6%b0%91%e6%96%b9%e9%9d%a2%e5%bc%95%e9%a2%86%e5%85%a8%e7%90%83/" href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e5%92%8c%e4%b8%ad%e5%9b%bd%e5%9c%a8%e7%9b%91%e8%a7%86%e5%85%ac%e6%b0%91%e6%96%b9%e9%9d%a2%e5%bc%95%e9%a2%86%e5%85%a8%e7%90%83/" rel="noopener noreferrer" target="_blank">
    中国以拥有典型大规模的监视状态而闻名
   </a>
   ，众所周知，通过中国银行系统进行的交易可以并且将由政府进行直接监视。
  </p>
  <p class="graf graf--p">
   但是，该国两个主要的移动支付平台支付宝和微信支付在技术层面上是私人公司，因此它们
   <strong class="markup--strong markup--p-strong">
    被要求与政府合作以时刻监视公民的交易
   </strong>
   。
  </p>
  <p class="graf graf--p">
   为了说明这些平台的规模，支付宝声称每秒处理多达 256,000 次付款，相比下 VISA 为每秒 65,000 次。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    所谓的国家加密货币可以使政府直接观察公民的交易，也就是拿走你全部生活的内幕。
   </strong>
  </p>
  <p class="graf graf--p">
   中国政府认识到了分散化货币的力量，它赋权给个人以远离老大哥的眼睛。
  </p>
  <p class="graf graf--p">
   中国央行支付结算司副司长穆长春在网上演讲中强调，更多的货币竞争可以使人民币合法化，他们称之为“货币主权”。
  </p>
  <p class="graf graf--p">
   他说：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    如果天秤座人人都接受并成为一种广泛使用的支付工具，那么一段时间后，它很可能会发展成为一种全球性的，超主权的货币。我们需要提前计划以保护我们的货币主权……
   </em>
  </p>
  <p class="graf graf--p">
   有趣的是，中国政府声称将把 “隐私” 换成 “可控制的匿名性”，这意味着政府允许“隐私”，正如中国央行的穆先生所解释的那样。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    你知道 “可控” 是什么意思 —— 就如您已经了解 “VPN可控” 的概念，这是一样的东西。
   </strong>
  </p>
  <p class="graf graf--p">
   穆说：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “只要你不犯法，不想进行那种不希望被人知道的购买，我们仍然希望保护你的隐私 ……”
   </em>
  </p>
  <p class="graf graf--p">
   太可笑了。
   <strong class="markup--strong markup--p-strong">
    就如斯诺登揭露后国安局的自辨：“如果你不干什么坏事，就不用担心监控” —— 当权者的通用逻辑。请时刻记得，究竟什么是“好事”、什么是“坏事”，永远是他们说了算的。
   </strong>
  </p>

  <p class="graf graf--p">
   总之，⚠️国家发行的电子货币将帮助中国政府更多地了解公民的花钱方式，比之前要多出很多很多。
  </p>
  <p class="graf graf--p">
   自 Facebook 推出 Libra 计划以来，还不到半年的时间，这家科技巨头进军金融业的尝试就遭遇了一连串的质疑。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    在这里看到天秤座有什么问题《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%9d%80%e6%ad%bb%e5%a4%a9%e7%a7%a4%e5%ba%a7%e5%9c%a8%e8%90%8c%e8%8a%bd%e7%8a%b6%e6%80%81/" href="https://www.iyouport.org/%e6%9d%80%e6%ad%bb%e5%a4%a9%e7%a7%a4%e5%ba%a7%e5%9c%a8%e8%90%8c%e8%8a%bd%e7%8a%b6%e6%80%81/" rel="noopener noreferrer" target="_blank">
     杀死天秤座在萌芽状态
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   对北京而言，Libra 提供了另一个推动人民币数字化的迫切动机。中国在其境内屏蔽 Facebook 平台，但中国领导人认为，Libra有可能开启一个新的世界金融体系，一个可以撼动政府和央行传统权威的体系 — — 中国的也包括在内。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果 Libra 被大家所接受，变成一个通行的支付工具，那么再过一段时间它会发展成一个世界级的超主权货币”，在中国央行担任要职的穆长春在近期一次在线讲座中说，“为了保护自己的货币主权和法币地位，我们需要未雨绸缪。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    建立
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2016/10/12/business/dealbook/central-banks-consider-bitcoins-technology-if-not-bitcoin.html" href="https://www.nytimes.com/2016/10/12/business/dealbook/central-banks-consider-bitcoins-technology-if-not-bitcoin.html" rel="nofollow noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     国家虚拟货币
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    的想法并不新鲜。英国、新加坡和加拿大的央行行长们就如何将数字代币融入金融体系进行了实验。欧洲中央银行(European Central Bank)董事会成员也称，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://uk.reuters.com/article/uk-facebook-cryptocurrency-france/france-germany-blast-facebooks-libra-back-public-cryptocurrency-idUKKCN1VY1EH" href="https://uk.reuters.com/article/uk-facebook-cryptocurrency-france/france-germany-blast-facebooks-libra-back-public-cryptocurrency-idUKKCN1VY1EH" rel="nofollow noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Libra 是唤醒银行电子支付努力的一记警钟
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p">
   北京一直希望人民币在国际贸易和金融中得到更多使用。如果Libra的货币跨境流通方便，那么它将成为许多国家的货币选择，尤其是那些经济不稳定的国家。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://finance.sina.com.cn/money/forex/rmb/2019-07-10/doc-ihytcerm2650777.shtml" href="https://finance.sina.com.cn/money/forex/rmb/2019-07-10/doc-ihytcerm2650777.shtml" rel="nofollow noopener noreferrer" target="_blank">
    在7月的一次讲话中
   </a>
   ，前央行行长周小川说，美元的主导地位已经侵蚀了那些“弱势”货币国家的经济，他警告说，Libra和其他未来的货币可能有一天也会这样做。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们需要提前做好准备，使人民币成为强势货币，”他表示。
  </p>
  <p class="graf graf--p">
   中国领导人也希望能有一个SWIFT以外的选择。SWIFT是西方主导的电讯网络，帮助资金跨境和跨银行流动。西方政府此前曾限制伊朗的SWIFT的接入，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2012/03/16/world/middleeast/crucial-communication-network-expelling-iranian-banks.html" href="https://www.nytimes.com/2012/03/16/world/middleeast/crucial-communication-network-expelling-iranian-banks.html" rel="nofollow noopener noreferrer" target="_blank">
    以惩罚其核计划
   </a>
   。
  </p>
  <p class="graf graf--p">
   数字化的人民币不太可能在一夜之间成为全球最受欢迎的新货币。中国严格监管人民币兑换其他货币，以保持汇率稳定，类似的限制措施可能也会约束“加密人民币”。⚪️
