  <p class="graf graf--p">
   当政府的搜查行为从手机转移到所谓的云端时：云提取技术和 “移动取证的未来” 是什么样的？
  </p>
  <p class="graf graf--p">
   手机仍然是执法调查中最常用和最重要的数字资源。然而，
   <strong class="markup--strong markup--p-strong">
    警察不仅仅要掌握手机上实际存储的内容，还想要从中了解到目标人曾经访问过什么，主要是通过存储在云中的数据
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Cellebrite 是用于从手机中提取数据的领先监控技术供应商，在其年度趋势调查中指出，在所有调查中大约一半证据都 “出现’’ 了云数据，并且 “通常’’ 涉及社交媒体或应用程序 —— 那些并不在物理设备上的数据。
  </p>
  <p class="graf graf--p">
   数据 “不驻留在物理设备上” 表示执法部门正在转向 “云提取”：对存储在第三方服务器上的用户数据进行取证分析，那些通常由设备和应用程序制造商备份的数据。
  </p>
  <p class="graf graf--p">
   然而，随着执法部门越来越多地将云提取用于从应用程序中获取数据，大众对此的警惕性完全没有跟上节奏。YouGov 的一项调查显示，在英国，有45.6％的人没有考虑过手机上应用程序创建的数据存储在哪里，而44.3％的人完全不知道或不认为他们手机上的应用程序使用云存储。
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   随着人们花更多的时间使用社交媒体、消息收发应用、使用 Dropbox 和 Google Drive 之类的东西来存储文件 …… 随着电话变得更加安全、锁定的设备更难以破解，并且基于文件的加密变得越来越普遍 …… 正如一位著名的业内人士所言，“
   <strong class="markup--strong markup--p-strong">
    可以说云提取决定了移动取证的未来
   </strong>
   。’’
  </p>
  <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote">
   <p>
    “基于私有云的数据代表了取证调查人员挖掘潜在证据的虚拟金矿。”
   </p>
  </blockquote>
  <p class="graf graf--p">
   隐私国际组织从取证学角度反复提出了对手机提取风险的担忧，并强调了缺乏有效的隐私和安全保护措施的危险。云提取则将风险更进了一步 —— 它不仅可以访问手机中包含的内容，还可以访问目标人在其中使用过的任何内容。
  </p>

  </p>

   云提取技术的部署几乎没有透明度，而且是在公众了解非常有限的背景下进行的：本报告将隐私国际组织的开源研究、技术分析和信息自由法案的结果汇总在一起，以尝试揭露这一
   <strong class="markup--strong markup--p-strong">
    新出现的对人权的紧急威胁
   </strong>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     什么是手机提取
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   手机提取工具是允许警察从手机下载数据的设备和软件，包括：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    联系人
   </li>
   <li class="graf graf--li">
    通话数据 —— 你打电话给谁、何时通话、以及持续多长时间
   </li>
   <li class="graf graf--li">
    短信
   </li>
   <li class="graf graf--li">
    存储的文件 —— 照片、视频、音频文件、文档等
   </li>
   <li class="graf graf--li">
    应用程序数据 —— 你使用什么应用程序、以及存储在其中的数据
   </li>
   <li class="graf graf--li">
    位置数据 —— 近乎实时的跟踪
   </li>
   <li class="graf graf--li">
    Wi-fi 网络连接 —— 可以显示您连接到 wi-fi 的任何地方的位置，例如曾经访问过的工作场所和居所
   </li>
  </ul>
  <p class="graf graf--p">
   移动电话提取需要与要分析的目标移动设备建立物理连接，以提取、分析并显示电话中包含的数据。
  </p>
  <p class="graf graf--p">
   但是，它不仅可以提供设备本身所包含的内容，而且还可以成为通向云和外部信息源的入口。如果警察从检查的目标设备中提取登录名、密码和令牌，就可以使用它们来验证凭据以提取云存储的数据。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     什么是云提取
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   云提取（或云分析）是一种访问、提取、分析、和保留存储在云中的数据的能力，这是技术公司广泛使用的术语，通常是指从第三方公司的服务器上提取。示例包括 Dropbox、Slack、Instagram、Twitter、Facebook、Google 产品如 My Activity、还有 Uber 和 Hotmail 等。
  </p>
  <p class="graf graf--p">
   随着云存储越来越多地用于社交媒体、与互联网连接的设备和应用程序，云提取为挖掘大量敏感个人信息打开了大门。
  </p>
  <p class="graf graf--p">
   在有关基于云的数据爆炸的相关报告中，有数据显示到2025年，将有49％的数据存储在公有云环境中。思科全球云指数预测了全球数据中心和基于云的IP流量的增长，并预测到2021年公有云数据中心的使用将继续大幅增加。
  </p>

  <p class="graf graf--p">
   例如，Cellebrite 的 UFED Cloud Analyzer 使用可以从设备中提取的登录凭据然后
   <strong class="markup--strong markup--p-strong">

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     这种取证是如何工作的
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   有多种方法可以 “绕开移动设备的状态或配置” 访问云数据，这从取证的角度来看很有吸引力。
  </p>
  <p class="graf graf--p">
   第一种涉及应用由个人提供的已知用户凭证，即 当个人自愿提交其登录详细信息时。第二种是从电话中提取数据，然后使用在该设备上或在其他设备（如笔记本电脑）上找到的令牌，用户可能在其中通过浏览器保存了身份验证令牌。第三种方法涉及在公共领域中收集数据。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “当用户成功通过应用或云服务进行身份验证时，该服务会返回一个令牌，该令牌用于使用户无需再次输入用户名和密码即可访问该服务。令牌就像一个通行证，例如当您打开 Gmail 帐户并使用它登录时不需要您进行任何交互。大多数令牌在身份验证时都设置了有效期限，这会因应用程序或云服务器而异。有些仅适用于单次登陆，另一些则适用于两个星期，还有一些适用于30天，如果用户在同一移动设备上使用该应用程序，则永久有效。
   </em>
  </p>
  <p class="graf graf--p">
   令牌的使用避免了登录时触发的双因素身份验证（2FA），后者通常会阻止对数据的访问。
  </p>
  <p class="graf graf--p">
   在2FA中，提示用户确认发送到独立设备（例如 移动电话）的代码的过程是一项关键的安全功能。但是，
   <strong class="markup--strong markup--p-strong">
    即使触发了 2FA，Oxygen Forensics Cloud Extractor 仍可以通知政府间谍，并且 “提供了多个选项来绕开附加步骤。”
   </strong>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     用于在移动设备之外获取令牌的工具
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、Elcomsoft 的 GTEX 工具可以在计算机上搜索身份验证令牌 ——
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果用户的计算机上安装了 Google Chrome 浏览器，并且用户通过浏览器登录了至少一项 Google 服务，就可以对 Google 帐户进行无密码身份验证。
  </p>
  <p class="graf graf--p">
   新的 Google 令牌提取器（GTEX）工具会自动在用户计算机上搜索由 Google Chrome 浏览器保存的身份验证令牌。
  </p>
  <p class="graf graf--p">
   用户在浏览器中登录自己的 Google 帐户后，这些令牌可实现对 Google 服务的无缝访问，而无需重新输入密码。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、Cellebrite 的 PC Cloud Collector ——
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “是一个独立的工具，可以使用浏览器中的 Cookie 和该PC上安装的应用程序从目标PC上创建令牌。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、UFED Cloud Analyzes 7.6 ——
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “扩展了其密码收集器功能，以包括保存在移动网络浏览器中的密码。审查员现在可以使用密码收集器从各个站点检索密码登录信息，以收集有关嫌疑人或受害者的最大数据量。这是通过利用用户访问在线帐户时保存在浏览器中的登录详细信息来实现的。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、另一个类似的工具是 Oxygen Forensics 的 KeyScout，可在PC上查找密码和令牌 ——
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “ KeyScout 安装了闪存卡并从 Windows PC 收集凭据。然后可以将收集到的凭据导入 Oxygen Forensic Cloud Extractor 中，以立即使用。”
  </p>

   </noscript>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     面部识别和云提取
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   使用云提取技术从手机和其他设备提取的数据分析越来越多地包括面部识别功能的使用。
  </p>
  <p class="graf graf--p">
   👉如果我们考虑可以从包含面部图像的 Instagram、Google照片、iCloud 等基于云的资源中获取大量个人数据的现实，那么在大量数据上使用面部识别的能力就很重要。
  </p>
  <p class="graf graf--p">
   毫无疑问，它可能被用于大量的云存储数据而没有任何透明度和问责制。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    早在2017年8月
   </strong>
   ，Cellebrite 为其分析平台引入了所谓的 “先进机器学习技术”，该技术可用于分析从云中提取的数据，其中包括面部识别和匹配。
  </p>
  <p class="graf graf--p">
   从2019年7月开始，Oxygen Forensic Detective 内置的 Oxygen Forensics JetEngine 模块提供了对人脸进行分类的功能。
  </p>
  <p class="graf graf--p">
   👉Oxygen 不仅可以在提取的数据中提供面部的分类和匹配，而且面部分析还
   <strong class="markup--strong markup--p-strong">
    可以基于性别、种族、和情绪识别进行分类
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Oxygen首席运营官 Lee Reiber 表示，该工具可以 “在证据集中搜寻特定的面孔，或将同一个人的图像聚合在一起。还可以按种族或年龄组以及 “喜悦” 和 “愤怒” 等情绪来过滤面孔。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12930 jetpack-lazy-image" data-lazy-sizes="(max-width: 429px) 100vw, 429px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/12-1.png?resize=429%2C568&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/12-1.png?w=429&amp;ssl=1 429w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/12-1.png?resize=227%2C300&amp;ssl=1 227w" data-recalc-dims="1" height="568" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/12-1.png?resize=429%2C568&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="429"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12930" data-recalc-dims="1" height="568" sizes="(max-width: 429px) 100vw, 429px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/12-1.png?resize=429%2C568&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/12-1.png?w=429&amp;ssl=1 429w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/12-1.png?resize=227%2C300&amp;ssl=1 227w" width="429"/>
   </noscript>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     持续追踪
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   获得用户的凭据后，不仅可以获取其基于云的数据，还可以使用其基于云的帐户来跟踪他们。
  </p>
  <p class="graf graf--p">
   例如，Cellebrite 的 Cloud Analyzer 的功能就包括：
  </p>
  <p class="graf graf--p">
   👉“跟踪在线行为，分析帖子、点赞、事件和联系，以更好地了解嫌疑人或受害者的兴趣、人际关系、观点和日常活动。”
  </p>
  <p class="graf graf--p">
   这提供了对个人生活的非常侵入性的了解。
   <strong class="markup--strong markup--p-strong">
    个人本身永远不会知
   </strong>
   道有人可以访问并正在使用他们的云数据。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12931 jetpack-lazy-image" data-lazy-sizes="(max-width: 974px) 100vw, 974px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/13-2.png?resize=974%2C497&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/13-2.png?w=974&amp;ssl=1 974w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/13-2.png?resize=300%2C153&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/13-2.png?resize=768%2C392&amp;ssl=1 768w" data-recalc-dims="1" height="497" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/13-2.png?resize=974%2C497&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="974"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12931" data-recalc-dims="1" height="497" sizes="(max-width: 974px) 100vw, 974px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/13-2.png?resize=974%2C497&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/13-2.png?w=974&amp;ssl=1 974w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/13-2.png?resize=300%2C153&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/13-2.png?resize=768%2C392&amp;ssl=1 768w" width="974"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   短期或长期的活动监视（特别是没有手机或在设备之外时）这非常具有侵犯性，并为云提取功能带来了另一个令人担忧的方面。
  </p>
  <p class="graf graf--p">
   ⚠️监视者不仅可以随时通过其登录凭据或访问其基于云的帐户的功能来跟踪和监视目标人的行为、消息和位置数据，而且还可以发送消息，模仿目标人，将包含非法内容的信息发送给其他人。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     结论
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   由于没有有关使用云提取技术的详细信息，因此不清楚这是否合法，也不清楚如何保护个人免受滥用其数据的危害。（我们会坚持建议您尽可能不要使用云）
  </p>
  <p class="graf graf--p">
   可以从云服务中提取的数据量，包括分析图像的面部识别技术，以及，即使仅提取与一个人有关的云数据，也将产生获得
   <strong class="markup--strong markup--p-strong">
    更大量周边人的个人数据
   </strong>
   的影响，足以表明这种技术应该有更大的透明度和责任感。
  </p>
  <p class="graf graf--p">
   这是执法机构的危险趋势的一部分，希望在全球范围内确保其使用的新技术形式的透明度和问责制。
  </p>
  <p class="graf graf--p">
   以下是隐私国际组织的建议。
  </p>
  <p class="graf graf--p">
   搜索一个人的基于云的数据比搜索他们的家更具侵入性，不仅是信息的数量和详细程度，还有可回溯性和继续挖掘未来数据的能力。 当权者不应无拘无束地获得公民的全部生活，使用云提取需要最严格的保护。
  </p>
  <p class="graf graf--p">
   因此提出以下建议：
  </p>
  <p class="graf graf--p">
   1、对警察和边境管理部门对云提取技术的使用加强独立调查；并征询公众、民间社会、工业界的意见。
  </p>
  <p class="graf graf--p">
   2、在对任何基于云的数据进行取证检查或以其他方式访问其中存储的任何内容或通信数据之前，警察必须获得法官基于合理怀疑而签发的逮捕令。
  </p>
  <p class="graf graf--p">
   3、必须有明确的法律依据来检查、收集、存储和分析基于云的服务中的数据，必须提供足够的保障以确保仅在必要且相称时才使用这种侵入式权力。
  </p>
  <p class="graf graf--p">
   4、必须发布针对公众的权利及其提取内容的指南，并将其提供给要对其设备进行分析的人员。
  </p>
  <p class="graf graf--p">
   5、必须告知目标人其基于云的数据已被提取、分析和保留。
  </p>
  <p class="graf graf--p">
   6、检查过基于云的数据的任何人都应该可以使用有效的补救措施，从而可以提出有关合法性的任何担忧。
  </p>
  <p class="graf graf--p">
   7、执法人员必须对合法使用这些权力的遵守情况接受独立监督。
  </p>
  <p class="graf graf--p">
   8、网络安全标准应达成共识并分发，规定必须如何存储数据、将其保留多长时间、何时必须删除数据，以及谁可以访问它。
  </p>
  <p class="graf graf--p">
   9、使用这些权力的所有主管部门都必须通过公共领域的采购渠道购买相关取证工具，并定期更新其已购买工具的清单，包括有关所拥有工具、制造商、和支出金额的详细信息。
  </p>
  <p class="graf graf--p">
   10、创建并遵循技术标准以确保进行验证和确认。例如，这应该伴随有明确记录的过程。
  </p>
  <p class="graf graf--p">
   11、需要足够的技术技能。因为拥有如此空前的数据量，必需由具备高技能的取证调查员来执行。
  </p>
  <p class="graf graf--p">
   12、云提取技术的测试、试验和部署必须伴随准确的评估、适当的保障措施、以及公众和民间社会的参与。
  </p>
