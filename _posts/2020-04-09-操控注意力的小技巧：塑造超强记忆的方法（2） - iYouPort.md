   </a>
   我们提到专注力，它是记忆力的关键；正因此
   <strong class="markup--strong markup--p-strong">
    既是侦察的关键、也是反侦察的关键
   </strong>
   。
  </p>
  <p class="graf graf--p">
   一般人能记住的对象（比如词语、数字、项目或想法）的数量在大约5～9之间，这是注意力的一个主要特点。
  </p>
  <p class="graf graf--p">
   要想超过这个数字似乎很难，但也有一些方法可以突破极限 —— 你要做的就是对记忆对象进行分组。
  </p>
  <p class="graf graf--p">
   以数字为例，比如电话号码，中国的手机号码有11位数字，是全球最长的手机号码。前3位是网络识别号、4–7位是地区编码 (HLR归属位置寄存器)、8–11位是用户号码(随机分配)。这就可以把它分为3条记忆信息，就是这么简单。
  </p>
  <p class="graf graf--p">
   很多时候为了你的需求，大脑在自动寻找拆分记忆信息的方法；但如果你知道拆分的意义，就如电话号码这样，印象就会更深刻。
  </p>
  <p class="graf graf--p">
   这无需多说，背诵外文单词的人都明白这个道理 —— 先记住词根。学中文也是一样，汉字是象形的，它的偏旁部首与它的含义有关，如果你能知道这个字是怎么来的，就不会再写错字。
  </p>
  <p class="graf graf--p">
   所以你发现了，
   <strong class="markup--strong markup--p-strong">
    这一切的根本在于你对注意力的控制能力 —— 如果能掌握自己的注意力，你就能调试自己的记忆和反应速度；相反，如果你能控制对手的注意力，你就能操纵他们以满足你的目的。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️注意力不是发呆，不是旁若无人，这里的关键字是：掌握，也就是由你来控制它。
  </p>
  <p class="graf graf--p">
   所以练习 “掌握” 的关键在于，让它动起来 —— 在你的控制下自如地移动，也就是
   <strong class="markup--strong markup--p-strong">
    转换注意力
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “提高” 注意力并不是最重要的，重要的是转换。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     对注意力的练习
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   同时执行两项任务可以提高转换注意力的能力。
  </p>
  <p class="graf graf--p">
   比如同时阅读两本书，一本读完一段后，换成另一本再读一段，如此反复进行。或者在听广播的时候每隔几秒钟切换一次频道，同时收听两个新闻节目，再或者同时观看两个电视节目。
  </p>
  <p class="graf graf--p">
   当你这样尝试的时候请试着
   <strong class="markup--strong markup--p-strong">
    通过上下文再现缺失的信息
   </strong>
   。
  </p>
  <p class="graf graf--p">
   慢慢你就会发现，两本书/两个电视节目你都能理解其意思。也就是说，💡不论在哪一边停留哪怕再短的时间，你都能控制注意力在那个位置上，尤其是可以
   <strong class="markup--strong markup--p-strong">
    自如地转移它到一个新的位置，并保持强度不变
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是不要沉迷！同时执行多项任务虽然对训练注意力很有帮助，但它不是同时成功完成大量任务的有效方法。
  </p>
  <p class="graf graf--p">
   转移注意力练习是为应对现实中的复杂环境。
  </p>
  <p class="graf graf--p">
   你练习过散打吗？当一人对多人比赛时，那一人必须能以最快速度控制并转移自己的注意力，因为威胁来自多个不同的位置，对每一个位置都需要全力以赴地应对，并且更重要的是，需要灵活自如地转移注意力到新的位置上。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在这种情况下要求注意力 “集中”在一点上，不仅没有帮助，甚至会让你送命。
   </strong>
  </p>
  <p class="graf graf--p">
   互联网就类似于散打的环境 —— 你需要应对几乎同时来自不同方向的信息轰炸，如果胡乱应对它们，最终你什么都记不住，连续阅读一整天几乎一无所获。
  </p>
  <p class="graf graf--p">
   💡相反，如果你能做到自如地转移注意力，就能在其中迅速发现对自己最有价值的内容，并捕获它们。
  </p>
  <p class="graf graf--p">
   我们已经
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/iyps-telegram%e9%a2%91%e9%81%93%e6%9c%ac%e5%b9%b4%e5%ba%a6%e9%83%a8%e5%88%86%e5%86%85%e5%ae%b9%e6%9b%b4%e6%96%b0%e6%b1%87%e6%80%bb/" href="https://www.iyouport.org/iyps-telegram%e9%a2%91%e9%81%93%e6%9c%ac%e5%b9%b4%e5%ba%a6%e9%83%a8%e5%88%86%e5%86%85%e5%ae%b9%e6%9b%b4%e6%96%b0%e6%b1%87%e6%80%bb/" rel="noopener noreferrer" target="_blank">
    在 telegram 频道中
   </a>
   上传过两次
   <strong class="markup--strong markup--p-strong">
    游击战相关的技巧内容
   </strong>
   。游击队成员最关键的技能之一就是敏锐自如的注意力转换，你需要比你的对手更快，才能抢先获胜。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     舒尔特表训练法（5✖️5）
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   舒尔特表有助于加速心理感知能力的发展，包括你的
   <strong class="markup--strong markup--p-strong">
    周边视觉和注意力、自我控制力、以及聚焦能力
   </strong>
   —— 这几点是关键。
  </p>
  <p class="graf graf--p">
   下面是一个 5✖️5 的表格，每个单元格中随机填写了 1～25 之间的任意数字。你的任务是以最快的速度按顺序指出每个数字的位置。
  </p>
  <p class="graf graf--p">
   这里面其实是有技巧的。不要将目光从一个单元格移动到下一个，那样会非常慢，也不要大声读出来、不要在心里默念这些数字。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    将你的视线固定在表格的正中央，使用你的 **周边视觉** 来搜索数字。
   </strong>
  </p>
  <p class="graf graf--p">
   人只有两只眼睛，而且全长在脑袋前面，物理角度上看它似乎无法兼顾四面八方？其实不然。因为你可以训练出更强的周边视觉，
   <strong class="markup--strong markup--p-strong">
    尤其是在现实中反侦察行动中，周边视觉能帮你快速反应以识别威胁。
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    在这里回顾《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%9c%a8%e4%bd%a0%e7%9a%84%e6%97%a5%e5%b8%b8%e7%94%9f%e6%b4%bb%e4%b8%ad%e5%ba%94%e7%94%a8%e9%9a%90%e8%94%bd%e9%87%8e%e6%88%98%e6%8a%80%e6%9c%af/" href="https://www.iyouport.org/%e5%9c%a8%e4%bd%a0%e7%9a%84%e6%97%a5%e5%b8%b8%e7%94%9f%e6%b4%bb%e4%b8%ad%e5%ba%94%e7%94%a8%e9%9a%90%e8%94%bd%e9%87%8e%e6%88%98%e6%8a%80%e6%9c%af/" rel="noopener noreferrer" target="_blank">
     在你的日常生活中应用隐蔽野战技术：简单方式抵御敌对监视

   </li>

  <p class="graf graf--p">
   从你的舒尔特表练习开始。最初可能会感觉很难，但不要放弃继续这样做，不断打破排序模式，反复这样做，就能看到成效。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这项技能能帮助你完成一系列任务，从观察、到迅速识别风险、到速读、到速记，无所不包
   </strong>
   。
  </p>
  <p class="graf graf--p">
   以下面这个表格为例，目标是，完成数字 1～25 的顺序所花费的时间最终减少到12秒，或者更短。

    <span style="color: #ff6600;">
     工作记忆
    </span>
   </strong>
  </h3>
  <p class="graf graf--p">
   在上一篇文章中我们介绍了感觉记忆、短期记忆、和长期记忆。其实还有一种被称为 “工作记忆” 的特殊类别。它用于储存数据以便立刻处理。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    工作记忆与其他记忆的主要区别在于，工作记忆与智力水平密切相关
   </strong>
   。
  </p>
  <p class="graf graf--p">
   显然，如果一个人能够在工作记忆中储存更多的数据，那么他就会更擅长注意外部关系，以及形成新的认知，这就是智力测试所关注的能力。
  </p>
  <p class="graf graf--p">
   此外，工作记忆容量也和注意力有关，一个人的工作记忆容量越大，他能够同时关注的对象就越多。
  </p>
  <p class="graf graf--p">
   一般人的工作记忆容量在5～9个单位之间。
   <strong class="markup--strong markup--p-strong">
    工作记忆容量小的人在控制注意力和组织行为等方面会遇到更多困难。
   </strong>
  </p>
  <p class="graf graf--p">
   不要着急，它可以通过训练获得提升。
  </p>
  <p class="graf graf--p">
   想确定自己的工作记忆容量是多少吗？做一个小测试即可。
  </p>
  <p class="graf graf--p">
   请注意下面列表中的数字，并把他们写在一张纸上。不要将这些数字每两个或三个一组记忆。举个例子，将1234表示为 “1–2–3–4”，而不是 “12–34”。当你记到列表末尾时，
   <strong class="markup--strong markup--p-strong">
    计算一下自己正确记录的最后一个数的位数，那就是你的工作记录容量。
   </strong>

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     练习方法
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   找一副扑克牌。从任意两个花色中依次数出6张对应的牌，比如 红桃 A、2、3、4、5、6，和梅花 A、2、3、4、5、6。把牌的顺序打乱，然后将它们以四排三列的布局放在桌子上，保持排面朝下。
  </p>
  <p class="graf graf--p">
   开始用秒表计时⌛️。
  </p>
  <p class="graf graf--p">
   每次你可以翻开任意两张牌，如果牌面上的数字相同，就让牌面朝上，如果不相同，就再把它们朝下翻过去。
  </p>
  <p class="graf graf--p">
   你的任务就是尽快找到所有数字匹配的牌。越快越好。
  </p>
  <p class="graf graf--p">
   因为，如果你能记住牌面朝下的牌，就能更快地完成任务。
  </p>
  <p class="graf graf--p">
   把这些牌面朝下的牌想象成已经翻转过来被你看到的数字的样子。
   <strong class="markup--strong markup--p-strong">
    除非你找到了与之匹配的牌，否则尽可能限制自己不要随意把某张牌再次翻开。这很重要。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这个练习旨在培养你的工作记忆力和专注力，加速感知过程。当你学会相信自己的直觉并增强工作记忆之后，就不需要再反复核查已经看过的牌了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   工作记忆的能力在日常生活中很重要。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    如果你发现有车子跟踪你，在甩掉它之前迅速记住那辆车子的外观和车牌会很有帮助；
   </li>
   <li class="graf graf--li">
    如果你的队友快要被捕了，在最后的关键时刻他耳语你一个账户和密码，那里面保存着关键证据，你能够迅速记住它们吗？
   </li>
   <li class="graf graf--li">
    如果政府间谍与你侧肩而过，你迅速认出他们吗？……
   </li>
  </ul>
  <p class="graf graf--p">
   这些都是行动力的关键部分。你的对手很狡猾，而且资金雄厚人手充足，这些你都没有，你能做到的做好的事就是从个人能力方面胜过他们。庞大笨重的大象可能玩不过一只灵巧的猴子。
  </p>
  <p class="graf graf--p">
   💡这一训练还能帮助你在现实中应对很多嘈杂和复杂的状况，迅速抓取并记住最重要的事物、更快地了解对方、更准确地评估和预测面临的各种状况。你会发现自己越来越擅长与各种人攀谈，因为你能迅速掌握对方的心理模式；并且能比其他人更快地学到东西。
  </p>
  <p class="graf graf--p">
   随意在一个陌生的城市里逛一小时，同伴一起，回来后你和同伴可以互相测试，互相提问：面包店的标志是什么颜色的？书店的门上贴着的那张告示上的主要内容是什么？餐馆的桌布上是什么图案？老板娘带了一副耳环，它是什么形状的？……
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    你们彼此都不知道对方即将提问什么，于是在整个逛街过程中你们都需要努力地观察和妥善地记忆。这是极好的训练
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong>
    <span style="color: #ff0000;">
     快速回答：
    </span>
   </strong>
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    关于那副扑克牌，我举例的两种花色是什么？不要向前翻看！
   </span>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    黑桃和方块
   </li>
   <li class="graf graf--li">
    红桃和黑桃
   </li>
   <li class="graf graf--li">
    方块和梅花
   </li>
   <li class="graf graf--li">
    红桃和梅花
   </li>
  </ul>
  <p class="graf graf--p">
   逛街是个好主意，因为街道上包含无数细节，其中隐藏着最关键的东西 —— 如果你想要在最需要的情况下及时想起它们来，你就需要随时的记忆。这是一个能力训练。
  </p>
  <p class="graf graf--p">
   我们将在下一次的文章中具体介绍训练观察能力和记忆力结合的方法。希望您已经答对了上面两个问题。⚪️
  </p>
