  <p class="graf graf--p graf-after--figure" id="7dac">
   <strong class="markup--strong markup--p-strong">
    编者按
   </strong>
   ：当技术落到政权手里无疑将变成路西法。无人机用以镇压群体抗议的事件已经出现，然而国际标准却没能跟上邪恶的膨胀速度，以至于它目前为止甚至仍是合法的。这篇文章将提醒所有人警惕，当我们计划反抗时，需要首先准备更多的知识和技术，以应对高科技的镇压模式。
  </p>
  <p class="graf graf--p graf-after--p" id="e441">
   ***
  </p>
  <p class="graf graf--p graf-after--p" id="8cf3">
   视频从一个可见的多旋翼无人机开始，飞在湛蓝的天空上。一群头戴蓝色头盔的电视记者站在下面，抬头看着它。它挂在那里有一秒钟的时间，然后瞬间喷发出催泪瓦斯，挥舞着白烟的尾巴。一片骚乱，记者们开始朝着各个方向奔跑，咳嗽。镜头再次仰望蓝天，那个小小的黑点神秘地退到了远处。
  </p>
  <p class="graf graf--p graf-after--p" id="583f">
   欢迎来到无人机革命的下一阶段。
  </p>
  <p class="graf graf--p graf-after--p" id="2f3e">
   3月12日，以色列可能成为了有史以来第一个在现实世界中使用武装消费无人机的国家，上述是当时黎巴嫩新闻网播放的录像，一个催泪瓦斯对付加沙抗议者的录像。
  </p>
  <p class="graf graf--p graf-after--p" id="657d">
   <strong class="markup--strong markup--p-strong">
    部署无人机对抗抗议者 — 甚至是武装部队 — 引发了关于合法性、身份和目的的一系列令人担忧的新问题。
   </strong>
   这种用法可能是不可避免的，但国际社会至少需要尽快制定并执行法律和道德标准。
  </p>
  <p class="graf graf--p graf-after--p" id="d6c2">
   这些
   <strong class="markup--strong markup--p-strong">
    催泪瓦斯无人机
   </strong>
   的起源并不是什么秘密。它们由一家名为 ISPRA 的以色列国防公司制造，该公司专门从事“防暴装置、人群管理、反恐设备和警用装备”的制造。
  </p>
  <p class="graf graf--p graf-after--p" id="26de">
   该公司的网站包含一个关于气旋防暴无人机系统的
   <a class="markup--anchor markup--p-anchor" data-href="http://www.ispraltd.com/image/users/423329/ftp/my_files/Riot%20Control%20Drone%20-%20Cyclone-1.pdf?id=30746915" href="http://www.ispraltd.com/image/users/423329/ftp/my_files/Riot%20Control%20Drone%20-%20Cyclone-1.pdf?id=30746915" rel="noopener nofollow noreferrer" target="_blank">
    信息手册
   </a>
   。除以色列之外，越来越多的武装团体和其他非国家行为者在战区使用消费型无人机。伊斯兰国、逊尼派武装、菲律宾极端主义分子等都在使用小型无人机摧毁爆炸物并进行监视。
   <strong class="markup--strong markup--p-strong">
    美国军方
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wired.com/story/army-dji-drone-ban/" href="https://www.wired.com/story/army-dji-drone-ban/" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     一直在使用
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    中国公司 DJI 制造的消费类无人机，直到去年因安全问题而暂停。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="bfb9">
   ISPRA Cyclone 也是一种消费无人机，几乎可以肯定是 DJI S1000 + octocopter 模型的改版，它经常被摄影师使用，看起来非常像那些地面上的民用无人机。Cyclone 和 IDF 选择使用它，突显
   <strong class="markup--strong markup--p-strong">
    消费无人机和军用无人机之间界限的模糊
   </strong>
   ，
   <strong class="markup--strong markup--p-strong">
    以及它们在国际人道法下的不确定地位
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="4bcf">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.files.ethz.ch/isn/195222/Tear%20Gassing%20By%20Remote%20Control%20Report.pdf" href="https://www.files.ethz.ch/isn/195222/Tear%20Gassing%20By%20Remote%20Control%20Report.pdf" rel="noopener nofollow noreferrer" target="_blank">
    至少从 2001 年起
   </a>
   ，通过无人机输送催泪瓦斯一直是被国防承包商积极试验的领域：到 2016 年，多家公司将它们
   <a class="markup--anchor markup--p-anchor" data-href="https://www.popularmechanics.com/flight/drones/g2445/tear-gas-capable-drones/" href="https://www.popularmechanics.com/flight/drones/g2445/tear-gas-capable-drones/" rel="noopener nofollow noreferrer" target="_blank">
    提供给
   </a>
   国防和执法部门的采购商。至少从2015年1月起，ISPRA 的专业人群控制无人机已经可用，该公司
   <a class="markup--anchor markup--p-anchor" data-href="http://www.ispraltd.com/eee.html" href="http://www.ispraltd.com/eee.html" rel="noopener nofollow noreferrer" target="_blank">
    在网站上
   </a>
   称自己是以色列国防和警察部队的“首选供应商”。
  </p>
  <p class="graf graf--p graf-after--p" id="0ce2">
   以色列国防军对消费无人机技术也不陌生：
   <strong class="markup--strong markup--p-strong">
    2017 年，他们在整个军队中分发了由中国 DJI 制造的可折叠 Mavic 无人机
   </strong>
   。尽管以色列军方消息人士告诉媒体说，通过无人机方式进行的新型人群控制手段是“实验性”的，并且在3月22日3月22日尚未投入使用，边防警察副专员 Yaakov Shabtai
   <a class="markup--anchor markup--p-anchor" data-href="https://www.timesofisrael.com/israel-to-deploy-tear-gas-dropping-drones-on-planned-gaza-border-protests/" href="https://www.timesofisrael.com/israel-to-deploy-tear-gas-dropping-drones-on-planned-gaza-border-protests/" rel="noopener nofollow noreferrer" target="_blank">
    对“Hadashot”电视新闻说
   </a>
   ， “这些装备帮助部队抵消了很多危险。”
  </p>
  <p class="graf graf--p graf-after--p" id="85a1">
   但是，3月22日以后，无人驾驶飞机开始越来越多地出现。随着巴勒斯坦权力组织在3月30日以后恢复抗议，自组装军备竞赛开始了：巴勒斯坦示威者开始放飞风筝，薄薄的风筝带着燃烧着的物质穿过加沙和以色列之间的边界屏障（尽管它们常常被强风阻挡）。根据
   <a class="markup--anchor markup--p-anchor" data-href="https://www.haaretz.com/israel-news/israel-enlists-amateur-drone-racers-to-stop-kite-bombs-from-gaza-1.6077983" href="https://www.haaretz.com/israel-news/israel-enlists-amateur-drone-racers-to-stop-kite-bombs-from-gaza-1.6077983" rel="noopener nofollow noreferrer" target="_blank">
    “国土报”报道
   </a>
   ，5月11日，以色列国防军招募无人机爱好者来对付风筝。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.haaretz.com/israel-news/israel-enlists-amateur-drone-racers-to-stop-kite-bombs-from-gaza-1.6077983" href="https://www.haaretz.com/israel-news/israel-enlists-amateur-drone-racers-to-stop-kite-bombs-from-gaza-1.6077983" rel="noopener nofollow noreferrer" target="_blank">
    无人机配备
   </a>
   了锋利的鱼钩，以切断风筝的绳子。
  </p>
  <p class="graf graf--p graf-after--p" id="6bcd">
   一些巴勒斯坦人称，以色列国防军无人机正在使用比通常针对加沙地区人群部署的化学物质
   <strong class="markup--strong markup--p-strong">
    更有效的化学剂
   </strong>
   。5月14日，目击者
   <a class="markup--anchor markup--p-anchor" data-href="https://www.apnews.com/b83dac724dc04a699d3861cf740dbf45" href="https://www.apnews.com/b83dac724dc04a699d3861cf740dbf45" rel="noopener nofollow noreferrer" target="_blank">
    告诉美联社
   </a>
   ，以色列无人驾驶飞机已经“放弃了燃烧材料，并布置了用于计划中对付加沙边境抗议活动的燃烧轮胎”。从那天起，一段
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/Islam_Abdu80/status/996067552882241537" href="https://twitter.com/Islam_Abdu80/status/996067552882241537" rel="noopener nofollow noreferrer" target="_blank">
    Twitter 视频
   </a>
   似乎描绘了一架无人机投下一个装置击穿了帐篷，然后爆炸起火。
  </p>
  <p class="graf graf--p graf-after--p" id="33f7">
   巴勒斯坦红新月会应急和灾害响应部门负责人 Mohammed Abu Mosabeh 说，人们仍然习惯于催泪弹从枪支中发射出来，而不是瓦斯从天而降。Mosabeh说，
   <strong class="markup--strong markup--p-strong">
    这种气体引起了强烈的反应，比催泪瓦斯更加高效：“有些人出现大面积惊厥反应，整个身体像触电一样震动。其他人晕厥20或30分钟。”（这个从4月8日开始的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.google.com/url?q=https://www.youtube.com/watch?v%3DRltdLF3HYYk&amp;sa=D&amp;ust=1526932680293000&amp;usg=AFQjCNFT7tTIvmSv63_qthn0n4v6NGe2jg" href="https://www.google.com/url?q=https://www.youtube.com/watch?v%3DRltdLF3HYYk&amp;sa=D&amp;ust=1526932680293000&amp;usg=AFQjCNFT7tTIvmSv63_qthn0n4v6NGe2jg" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     视频显示
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    由这种气体引起的惊厥症状）。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="5bf8">
   这些报告非常令人担忧，虽然尚未得到外部研究人员的证实。国际特赦组织的一位研究人员说，这种气体可能只是特别有效的催泪瓦斯，这本身就是危险的。
  </p>
  <p class="graf graf--p graf-after--p" id="0c37">
   不管是什么，
   <strong class="markup--strong markup--p-strong">
    催泪瓦斯和无人机都已经不仅仅是加沙的问题，而是整个世界范围内的。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="ca60">
   我们可以将两种法律框架应用于小型武器化无人机 — 国际人道主义法和国际人权法 — 并且激烈的争论在加沙适用。4月，以色列和巴勒斯坦维权团体向以色列最高法院
   <a class="markup--anchor markup--p-anchor" data-href="https://www.i24news.tv/en/news/israel/173622-180430-israel-s-open-fire-rules-on-gaza-border-challenged-in-court" href="https://www.i24news.tv/en/news/israel/173622-180430-israel-s-open-fire-rules-on-gaza-border-challenged-in-court" rel="noopener nofollow noreferrer" target="_blank">
    提出请愿
   </a>
   ：这些团体认为，根据国际人权法，执法行动不允许以色列对加沙示威者使用致命火力。（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.haaretz.com/israel-news/.premium-red-cross-slams-israel-for-misuse-of-report-to-justify-gaza-actions-1.6091737" href="https://www.haaretz.com/israel-news/.premium-red-cross-slams-israel-for-misuse-of-report-to-justify-gaza-actions-1.6091737" rel="noopener nofollow noreferrer" target="_blank">
    红十字
   </a>
   国际委员会和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amnesty.org/en/latest/news/2018/03/israelopt-stop-the-use-of-lethal-and-other-excessive-force-and-investigate-deaths-of-palestinian-protesters/" href="https://www.amnesty.org/en/latest/news/2018/03/israelopt-stop-the-use-of-lethal-and-other-excessive-force-and-investigate-deaths-of-palestinian-protesters/" rel="noopener nofollow noreferrer" target="_blank">
    国际特赦组织
   </a>
   等都同意这种解释）
  </p>
  <p class="graf graf--p graf-after--p" id="7243">
   作为回应，以色列政府就其在加沙边界的行动披露了自己的法律立场。该国认为，边界抗议活动构成以色列和哈马斯之间的武装冲突，这受到武装冲突法的约束，也被称为国际人道主义法。有争议的是，以色列认为（正如 Eliav Lieblich 在 Just Security 网站上的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.justsecurity.org/56346/collectivizing-threat-analysis-israels-legal-claims-resort-force-gaza-border/" href="https://www.justsecurity.org/56346/collectivizing-threat-analysis-israels-legal-claims-resort-force-gaza-border/" rel="noopener nofollow noreferrer" target="_blank">
    一篇杰出文章中所述
   </a>
   那样）其在加沙的活动属于国际人道主义法中的一个单独的执法制度，该制度“受到鼓励”，但与国际人道主义法的执法机制实际上并不相同 — 从而使以色列在这种情况下对抗议者使用致命武力是合法的。
  </p>
  <p class="graf graf--p graf-after--p" id="a9c9">
   根据国际人道主义法和国际人权法，以色列对消费无人机的使用存在问题，撇开该国对规则的争议性解释。
  </p>
  <p class="graf graf--p graf-after--p" id="7713">
   如果国际人道法适用于加沙局势，那么以色列使用催泪瓦斯可能就是非法的。1993年“化学武器公约”
   <a class="markup--anchor markup--p-anchor" data-href="https://www.opcw.org/chemical-weapons-convention/" href="https://www.opcw.org/chemical-weapons-convention/" rel="noopener nofollow noreferrer" target="_blank">
    禁止
   </a>
   大多数情况下在战争中使用催泪瓦斯。以色列
   <a class="markup--anchor markup--p-anchor" data-href="https://www.haaretz.com/opinion/.premium-israel-must-sign-the-chemical-weapons-convention-1.6009076" href="https://www.haaretz.com/opinion/.premium-israel-must-sign-the-chemical-weapons-convention-1.6009076" rel="noopener nofollow noreferrer" target="_blank">
    尚未批准
   </a>
   “化学武器公约”，但这并不能保护它免受国际人道主义法的批评。
  </p>
  <p class="graf graf--p graf-after--p" id="a5bb">
   另一方面，
   <strong class="markup--strong markup--p-strong">
    如果国际人道主义法不适用于这种情况，则允许使用催泪瓦斯：“化学武器公约”是执法的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://nwp.ilpi.org/?p=5667#The_Chemical_Weapons_Convention_and_law_enforcement" href="http://nwp.ilpi.org/?p=5667#The_Chemical_Weapons_Convention_and_law_enforcement" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     例外
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，这就是为什么
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/news/morning-mix/wp/2014/08/14/tear-gas-is-a-chemical-weapon-banned-in-war-but-ferguson-police-shoot-it-at-protesters/?utm_term=.4c9b72e7b37e" href="https://www.washingtonpost.com/news/morning-mix/wp/2014/08/14/tear-gas-is-a-chemical-weapon-banned-in-war-but-ferguson-police-shoot-it-at-protesters/?utm_term=.4c9b72e7b37e" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     催泪弹的使用是美国抗议活动的常规特征
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="f2a4">
   以色列国防军的无人机似乎并没有任何标识能将其明确为军用飞机或以色列所有。它们似乎也没有携带转发器或其他电子识别手段，能将其识别为军用飞机。这是另一个潜在的法律问题。国际法的
   <a class="markup--anchor markup--p-anchor" data-href="https://books.google.com/books?id=1dwUDAAAQBAJ&amp;pg=PA311&amp;lpg=PA311&amp;dq=%22military+aircraft+must+bear+both+nationality+marking+and+marking+as+a+military+aircraft%22&amp;source=bl&amp;ots=XGbZXUJgKm&amp;sig=onENlWaF_ovmlr7F_-yOUplGOHg&amp;hl=en&amp;sa=X&amp;ved=0ahUKEwi5spq" href="https://books.google.com/books?id=1dwUDAAAQBAJ&amp;pg=PA311&amp;lpg=PA311&amp;dq=%22military+aircraft+must+bear+both+nationality+marking+and+marking+as+a+military+aircraft%22&amp;source=bl&amp;ots=XGbZXUJgKm&amp;sig=onENlWaF_ovmlr7F_-yOUplGOHg&amp;hl=en&amp;sa=X&amp;ved=0ahUKEwi5spq" rel="noopener nofollow noreferrer" target="_blank">
    大多数解释
   </a>
   都指出，军用飞机必须同时拥有国籍标志和标识，将其标识为军用飞机。
  </p>
  <p class="graf graf--p graf-after--p" id="6fe3">
   这些标记使他们能够根据区分原则将平民区分开来，确保其他战斗人员和平民能够将其区别于民用飞机。根据国际人道法，无人驾驶飞行器（由人控制，而不是自主控制）受制于与载人飞机相同的规则和要求。我们可以将其解释为意味着以色列有义务将无人机标记为军用而不是民用飞机 — 就像以色列标记其他飞机一样。
  </p>
  <p class="graf graf--p graf-after--p" id="c1bf">
   然而，与有人驾驶飞机不同，无人机没有标准或习惯性的标记 — 而适用于更大型有人驾驶飞机的标志标准需要修改才能为其适用。仍然没有更好的方法通过电子或无线电方式在空中区分小型无人机; 虽然许多研究人员
   <a class="markup--anchor markup--p-anchor" data-href="https://utm.arc.nasa.gov/" href="https://utm.arc.nasa.gov/" rel="noopener nofollow noreferrer" target="_blank">
    正在研究
   </a>
   适合小型无人机的空域交通管理系统，但这项工作还需要很多时间才能见效。
  </p>
  <p class="graf graf--p graf-after--p" id="ec63">
   在不久的将来，在军事或警察无人机更频繁出现的情况下，将使得此事变得特别棘手。新闻记者正在越来越多地使用无人机来记录事件，而援助工作人员目前正在利用无人机来获得对灾难的态势感知，并且可能会在不久的将来使用它们提供医疗用品和其他救助资源。
  </p>
  <p class="graf graf--p graf-after--p" id="bbf0">
   这很危险，
   <strong class="markup--strong markup--p-strong">
    一位年轻的巴勒斯坦无人机记者 Yaser Murtaja 在4月份死亡的事实强调了为什么危险。
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/global-development/2018/apr/09/yasser-murtajas-last-video-footage-protests-soldiers-black-smoke" href="https://www.theguardian.com/global-development/2018/apr/09/yasser-murtajas-last-video-footage-protests-soldiers-black-smoke" rel="noopener nofollow noreferrer" target="_blank">
    他最后一部片子
   </a>
   是关于边界示威的录像，画面中抗议者大面积流泪，片子鸟瞰现场混乱的局面，从奔跑的抗议者到广阔的田野和燃烧的轮胎。Murtaja 几天后将被赶出现场。
  </p>
  <p class="graf graf--p graf-after--p" id="9573">
   Murtaja 在被枪杀时没有放飞无人机，但在他去世后的头几天里，许多人认为他做过。“我不知道他是不是摄影师 — 无论谁在以色列国防士兵头顶上操作无人机，都需要明白他在自找麻烦“，以色列国防部长 Avigdor Lieberman 于4月9日
   <a class="markup--anchor markup--p-anchor" data-href="https://www.haaretz.com/israel-news/.premium-no-evidence-to-lieberman-s-claim-that-slain-gaza-journalist-had-drone-1.5979315" href="https://www.haaretz.com/israel-news/.premium-no-evidence-to-lieberman-s-claim-that-slain-gaza-journalist-had-drone-1.5979315" rel="noopener nofollow noreferrer" target="_blank">
    告诉国土报
   </a>
   ，暗示 Murtaja 已经将自己置于危险之中。
  </p>
  <p class="graf graf--p graf-after--p" id="6ab6">
   Lieberman 无意中捕捉到了关于无人机麻烦的中心问题。在民用无人机普及之前，很容易认为在冲突地区驾驶无人机的每个人都可能是战斗人员。也很容易争辩说，平民知道无人机本质上是危险的，因此没有必要将它们标记为军用飞机。在近乎无处不在的消费类无人机时代，情况就不再如此了。
   <strong class="markup--strong markup--p-strong">
    技术再次超越了法律和我们的文化。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="2fc5">
   我们没有时间来巩固对冲突中小型无人机的集体文化和法律反应。
   <strong class="markup--strong markup--p-strong">
    以色列不太可能成为最后一个使用无人驾驶飞机对付示威者的国家 — 而且武器制造商也不可能停止使用催泪弹
   </strong>
   。以杜克机器人公司的
   <a class="markup--anchor markup--p-anchor" data-href="https://dukeroboticsys.com/" href="https://dukeroboticsys.com/" rel="noopener nofollow noreferrer" target="_blank">
    Tikad
   </a>
   为例，这是一种可携带机枪、榴弹发射器和其他攻击性武器的多旋翼无人机。根据 Defense One 记录，2015年 Tikad 消费型无人机原型就杀死了一个目标，尽管 Tikad 似乎还没有用于战斗，但那只是时间问题。2017年，康涅狄格立法者提出了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nbcnews.com/news/us-news/weaponized-drones-connecticut-bill-would-allow-police-use-lethal-force-n745291" href="https://www.nbcnews.com/news/us-news/weaponized-drones-connecticut-bill-would-allow-police-use-lethal-force-n745291" rel="noopener nofollow noreferrer" target="_blank">
    一项法案
   </a>
   ，允许警方在其无人驾驶飞机上增加致命武器，而北达科他州则在2015年将小型无人机上的非致命武器
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nbcnews.com/news/us-news/weaponized-drones-connecticut-bill-would-allow-police-use-lethal-force-n745291" href="https://www.nbcnews.com/news/us-news/weaponized-drones-connecticut-bill-would-allow-police-use-lethal-force-n745291" rel="noopener nofollow noreferrer" target="_blank">
    合法化
   </a>
   了。
  </p>
  <p class="graf graf--p graf-after--p" id="2696">
   无人驾驶飞机应该为受益于他们的平民提供服务，但自由必须与安全平衡。首先，国际社会应该根据国际人道主义法建立标准化的国际标识系统，光线模式或颜色，以指定不同种类的无人机 — 例如蓝色是新闻媒体，红色是援助组织、黑色是武装团体。这些标记的建立将使记者和援助人员等平民能够将他们的无人机与冲突中其他行为体所使用的无人机区分开来，从而降低发生致命身份错误的风险。
  </p>
  <p class="graf graf--p graf-after--p" id="7b19">
   这将是一个开始，但还明显不够。所有在冲突和其他复杂环境中操作无人机的各方最终都需要标准化的可靠电子解决方案，以在空域内明确识别无人机。这可能会采取小型 ADS-B 转发器的形式，如有人驾驶飞机使用的转发器或“
   <a class="markup--anchor markup--p-anchor" data-href="https://techcrunch.com/2017/10/25/djis-electronic-license-plate-helps-authorities-identify-drones-in-mid-flight/" href="https://techcrunch.com/2017/10/25/djis-electronic-license-plate-helps-authorities-identify-drones-in-mid-flight/" rel="noopener nofollow noreferrer" target="_blank">
    电子车牌
   </a>
   ”那样。
  </p>
  <p class="graf graf--p graf-after--p" id="d5b5">
   还有另外一个法规和法律压力。Omega 研究基金会的研究人员在2015年
   <a class="markup--anchor markup--p-anchor" data-href="https://remotecontrolproject.org/wp-content/uploads/2015/12/Tear-Gassing-By-Remote-Control-Report.pdf" href="https://remotecontrolproject.org/wp-content/uploads/2015/12/Tear-Gassing-By-Remote-Control-Report.pdf" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     呼吁
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    禁止化学武器组织对无人机散布化学品进行法律审查
   </strong>
   。立法者在国内和国际层面都应遵循澄清用于催泪瓦斯的无人机的法律地位以及其他执法目的，无论是致命的还是其他目的。
  </p>
  <p class="graf graf--p graf-after--p" id="633c">
   我们已经进入了无人机的时代。然而，我们仍然不知道无人机适用于国际习惯法，还是适合我们现有的道德和操作系统。无人机可能有摄像头，但地上的人可没有识别能力。
  </p>
  <p class="graf graf--p graf-after--p" id="3eb6">
   <a class="markup--anchor markup--p-anchor" data-href="http://foreignpolicy.com/2018/05/22/drones-dont-wear-uniforms-they-should/" href="http://foreignpolicy.com/2018/05/22/drones-dont-wear-uniforms-they-should/" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Drones Don’t Wear Uniforms. They Should.
    </strong>
   </a>
  </p>
  <p class="graf graf--p graf-after--p" id="20a7">
   Israel’s use of consumer drones against protesters heralds a dangerous, lawless age of conflict.
  </p>
