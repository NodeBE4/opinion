   您所做的一切都会在某处生成数据。这些数据在收集和分析后成为

  </p>
  <p class="graf graf--p">
   随着时间的推移，收集到的信息完全可以告诉某人有关您的一切，并且可以用来操纵和伤害您。该数据称为元数据。
  </p>
  <ul class="postList">
   <li class="graf graf--li">

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    什么是元数据？
   </strong>
  </h3>
  <p class="graf graf--p">
   元数据是关于数据的数据。是关于单个数据的事实。可以用在 Microsoft Word 中写信的示例来说明这一点。
  </p>
  <p class="graf graf--p">
   您写了一封信，将发生在工作中的不道德行为甚至可能是非法行为发送给领导。是的，它马上就会变得很戏剧化。
  </p>
  <p class="graf graf--p">
   这封信是由您按特定顺序键入的字符组成的信息。您键入的字符由一堆的1和0生成。1和0是组成信息的数据。
  </p>
  <p class="graf graf--p">
   当“ 1”和“ 0”转换为一封信时，还有其他有关创建这封信的数据。其中一些元数据包括写这封信的时间、谁写的这封信、上次保存的时间、在哪个版本的 Word 中创建的等等，所有这些都是有关数据的数据。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    元数据做什么？
   </strong>
  </h3>
  <p class="graf graf--p">
   它可以帮助您找到文档的最新版本或查看文档的创建者，从而可以向他们询问有关此文档的问题。它有助于跟踪文档的编辑或注释。
  </p>
  <p class="graf graf--p">

   Office 程序和其他程序也使用它来处理文档。例如，Windows 资源管理器使用该信息对文档进行分类和排序。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    为什么我要删除元数据？
   </strong>
  </h3>
  <p class="graf graf--p">
   回到您发送给领导的这封信中。您之所以匿名写信，是因为您害怕受到报复，或者只是不想引起他人的注意。
  </p>
  <p class="graf graf--p">
   您全力以赴，获得了一个临时的电子邮件地址，然后从图书馆的公共计算机上发送该电子邮件，以掩盖您的数字足迹。
  </p>
  <p class="graf graf--p">
   但是，由于存在元数据，文档中可能包含可用于将其关联到您的真实身份的信息。它甚至可能仍带有您的名字。
  </p>
  <p class="graf graf--p">
   更糟糕的是，您对文档所做的更改虽然不再对您自己可见，但仍可能存在于文档中。
  </p>
  <p class="graf graf--p">
   如果您写了一段包含特定内容的段落，但由于这一段路可以用来识别您的身份于是您将其删除了，那么它仍可能以元数据的形式成为文件的一部分。
  </p>

    <img alt="" class="aligncenter size-full wp-image-12111" data-recalc-dims="1" height="912" sizes="(max-width: 992px) 100vw, 992px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/12.png?resize=992%2C912&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/12.png?w=992&amp;ssl=1 992w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/12.png?resize=300%2C276&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/12.png?resize=768%2C706&amp;ssl=1 768w" width="992"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   完成后您将看到有关找到的数据类型的信息。圆圈中的绿色对勾表示未找到该类型的数据。红色的感叹号表示它找到了该类型的数据。在该数据类型的说明旁边，您会看到全部删除按钮。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12112 jetpack-lazy-image" data-lazy-sizes="(max-width: 994px) 100vw, 994px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/13.png?resize=994%2C900&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/13.png?w=994&amp;ssl=1 994w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/13.png?resize=300%2C272&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/13.png?resize=768%2C695&amp;ssl=1 768w" data-recalc-dims="1" height="900" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/13.png?resize=994%2C900&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="994"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12112" data-recalc-dims="1" height="900" sizes="(max-width: 994px) 100vw, 994px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/13.png?resize=994%2C900&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/13.png?w=994&amp;ssl=1 994w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/13.png?resize=300%2C272&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/13.png?resize=768%2C695&amp;ssl=1 768w" width="994"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   单击该按钮以删除该类型的所有数据。这些按钮可能有多个，因此请向下滚动以确保不要遗漏。
  </p>
  <p class="graf graf--p">
   删除元数据后，您可能要单击“重新检查”按钮，以确保它不会丢失任何内容。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12113 jetpack-lazy-image" data-lazy-sizes="(max-width: 994px) 100vw, 994px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/14.png?resize=994%2C210&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/14.png?w=994&amp;ssl=1 994w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/14.png?resize=300%2C63&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/14.png?resize=768%2C162&amp;ssl=1 768w" data-recalc-dims="1" height="210" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/14.png?resize=994%2C210&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="994"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12113" data-recalc-dims="1" height="210" sizes="(max-width: 994px) 100vw, 994px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/14.png?resize=994%2C210&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/14.png?w=994&amp;ssl=1 994w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/14.png?resize=300%2C63&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/14.png?resize=768%2C162&amp;ssl=1 768w" width="994"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   立即保存您的文档，以确保不会重新输入数据。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如何确定元数据已删除
   </strong>
  </h3>
  <p class="graf graf--p">
   请按照上述步骤操作，以在 Word、Excel 或 PowerPoint 中查看所有元数据。检查 core.xml、document.xml 和 people.xml 文件后，您应该看到其中不再有个人数据。
  </p>
  <p class="graf graf--p">
   如果将扩展名从.zip更改回.docx，则可以再次在Word中正常打开文件。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如何在 Windows 资源管理器中删除元数据
   </strong>
  </h3>
  <p class="graf graf--p">
   如果要快速从多个文件中剥离元数据，这是一个好方法。只需几秒钟即可完成对
   <strong class="markup--strong markup--p-strong">
    2个或更多文件
   </strong>
   的操作。
  </p>
  <p class="graf graf--p">
   打开Windows资源管理器，然后导航到要从中删除元数据的文件。右键单击该文件，然后单击“属性”。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12114 jetpack-lazy-image" data-lazy-sizes="(max-width: 934px) 100vw, 934px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/15.png?resize=934%2C740&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/15.png?w=934&amp;ssl=1 934w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/15.png?resize=300%2C238&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/15.png?resize=768%2C608&amp;ssl=1 768w" data-recalc-dims="1" height="740" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/15.png?resize=934%2C740&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="934"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12114" data-recalc-dims="1" height="740" sizes="(max-width: 934px) 100vw, 934px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/15.png?resize=934%2C740&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/15.png?w=934&amp;ssl=1 934w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/15.png?resize=300%2C238&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/15.png?resize=768%2C608&amp;ssl=1 768w" width="934"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在“属性”窗口中，点击“详细信息”选项卡，然后点 “删除属性和个人信息”。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12115 jetpack-lazy-image" data-lazy-sizes="(max-width: 974px) 100vw, 974px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/16.png?resize=974%2C1046&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/16.png?w=974&amp;ssl=1 974w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/16.png?resize=279%2C300&amp;ssl=1 279w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/16.png?resize=954%2C1024&amp;ssl=1 954w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/16.png?resize=768%2C825&amp;ssl=1 768w" data-recalc-dims="1" height="1046" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/16.png?resize=974%2C1046&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="974"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12115" data-recalc-dims="1" height="1046" sizes="(max-width: 974px) 100vw, 974px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/16.png?resize=974%2C1046&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/16.png?w=974&amp;ssl=1 974w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/16.png?resize=279%2C300&amp;ssl=1 279w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/16.png?resize=954%2C1024&amp;ssl=1 954w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/16.png?resize=768%2C825&amp;ssl=1 768w" width="974"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   您可以通过
   <strong class="markup--strong markup--p-strong">
    两种方式
   </strong>
   删除信息：可以从原始文件中删除元数据，也可以制作没有任何元数据的文件副本。
  </p>
  <p class="graf graf--p">
   方法1：选择“从此文件中删除以下属性：” 然后仅选中所需的框，或单击 “全选” 按钮。然后单击确定。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12116 jetpack-lazy-image" data-lazy-sizes="(max-width: 642px) 100vw, 642px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/17.png?resize=642%2C838&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/17.png?w=642&amp;ssl=1 642w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/17.png?resize=230%2C300&amp;ssl=1 230w" data-recalc-dims="1" height="838" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/17.png?resize=642%2C838&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="642"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12116" data-recalc-dims="1" height="838" sizes="(max-width: 642px) 100vw, 642px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/17.png?resize=642%2C838&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/17.png?w=642&amp;ssl=1 642w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/17.png?resize=230%2C300&amp;ssl=1 230w" width="642"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   方法2：这将复制文件，并在文件名的末尾添加单词 Copy。该副本将没有任何与之关联的元数据。
  </p>
  <p class="graf graf--p">
   在 “删除属性” 窗口中，选择 “创建一个删除了所有可能属性的副本”，然后单击 “确定” 按钮。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12117 jetpack-lazy-image" data-lazy-sizes="(max-width: 646px) 100vw, 646px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/18.png?resize=646%2C844&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/18.png?w=646&amp;ssl=1 646w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/18.png?resize=230%2C300&amp;ssl=1 230w" data-recalc-dims="1" height="844" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/18.png?resize=646%2C844&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="646"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12117" data-recalc-dims="1" height="844" sizes="(max-width: 646px) 100vw, 646px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/18.png?resize=646%2C844&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/18.png?w=646&amp;ssl=1 646w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/18.png?resize=230%2C300&amp;ssl=1 230w" width="646"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   比较原件和副本的属性以查看区别。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12118 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/19.png?resize=1100%2C819&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/19.png?w=1236&amp;ssl=1 1236w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/19.png?resize=300%2C223&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/19.png?resize=1024%2C762&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/19.png?resize=768%2C572&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/19.png?resize=1100%2C819&amp;ssl=1 1100w" data-recalc-dims="1" height="819" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/06/19.png?resize=1100%2C819&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    最后
   </strong>
  </h3>
  <p class="graf graf--p">
   这是否意味着已经与您完全？无法从文档中识别出您吗？很难说。接下来对文档的处理将确定这一点。
  </p>
  <p class="graf graf--p">
   ⚠️对文档进行任何进一步的数字处理（例如通过电子邮件发送）都可以将元数据添加回链中。一个可行的选择是打印文档并将其邮寄。因为很难从纸张中获取元数据。⚪️
  </p>
