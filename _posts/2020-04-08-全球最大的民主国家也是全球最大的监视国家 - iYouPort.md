  <p class="graf graf--p">
   印度是世界上最大的民主国家，也是最大的监视国家之一。

   </a>
   ，在监视公民方面，该国仅次于俄罗斯和中国。
  </p>
  <p class="graf graf--p">
   在该研究公司的隐私指数排名中，印度在5分中得分2.4，表明 “系统性的维护（隐私）保护措施失败”。

  <p class="graf graf--p graf--startsWithDoubleQuote">
   “有多个国家正在创建只能被描述为监视国家的状态，而隐私权几乎不被考虑。”
  </p>
  <p class="graf graf--p">
   对于印度来说尤其如此。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    印度为什么会变成这样？
   </strong>
  </p>
  <p class="graf graf--p">
   印度宪法保障基本的隐私权，但是，由于多种因素，印度人实际享有的隐私水平非常低。
  </p>
  <p class="graf graf--p">
   1、臭名昭著的
   <strong class="markup--strong markup--p-strong">
    Aadhaar 监视系统
   </strong>
   ：在 Aadhaar 下，公民获得唯一的识别号。该系统拥有全球最大的生物特征数据库，即12.3亿人的数据。
  </p>
  <p class="graf graf--p">
   这个有争议的系统过去曾因各种数据泄露而成为头条新闻。Aadhaar 数据库还包含购买、银行帐户和保险等私密信息。
  </p>
  <p class="graf graf--p">
   2、
   <strong class="markup--strong markup--p-strong">
    数据保护法案
   </strong>
   ：“该法案尚未生效，并且还没有一个数据保护机构，这意味着目前隐私保护非常薄弱，” Compritech 指出。
  </p>
  <p class="graf graf--p">
   3、
   <strong class="markup--strong markup--p-strong">
    监控 WhatsApp
   </strong>
   ：调查发现，政府使 Facebook 拥有的 WhatsApp 通过在发送的每条消息中添加数字指纹而使其消息变得可追溯，严重侵犯隐私。
  </p>
  <p class="graf graf--p">
   4、
   <strong class="markup--strong markup--p-strong">
    CCTV
   </strong>
   ：有关闭路电视监视器（CCTV）的法规含糊不清，任何与之相关的隐私法均开放解释。
  </p>
  <p class="graf graf--p">
   5、
   <strong class="markup--strong markup--p-strong">
    其他原因
   </strong>
   ：印度在安装高科技边境监视装置方面的努力以及与美国频繁共享信息的做法严重侵犯了隐私权。
  </p>
  <p class="graf graf--p">
   值得注意的是，印度还与不同国家签订了多项司法互助条约。Compritech 还表示，最近
   <strong class="markup--strong markup--p-strong">
    印度已允许10个政府机构授权解密、监视和拦截任何计算机上的数据
   </strong>
   ，如果服务提供商未能提供拦截功能，甚至会面临入狱长达七年。
  </p>
  <p class="graf graf--p">
   Compritech 指出，印度的法律现在已经开始声称保护数据隐私，但对成功与否定持怀疑态度。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “新的数据保护法生效后，Covert 监视也将被禁止。但是，由于监视策略和生物识别技术已经取得了令人难以置信的进步，因此，法律究竟能改变什么，在很大程度上令人怀疑。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    安全港
   </strong>
  </p>
  <p class="graf graf--p">
   在非欧盟国家中，隐私评分为3.1的挪威位居榜首。
  </p>
  <p class="graf graf--p">
   然而，根据 Compritech 的说法，基于生物特征识别技术的引入可能使其处于危险之中。
  </p>

  </p>
  <p class="graf graf--p">
   在欧盟国家中，爱尔兰、法国、葡萄牙和丹麦是拥有适当保障措施的前四名。排名第五的马耳他的保障措施已经遭到削弱。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “对前五名（唯一获得“足够”分数的国家）来说，有趣的是，它们全部受GDPR法律管辖，” Compritech 指出。GDPR 法律确保欧盟和欧洲经济区（EEA）所有公民的数据保护和隐私。它还解决了这些区域之外的个人数据传输。
  </p>
