  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     科技巨头微软一直在与一所中国军事大学进行人工智能研究合作，最近有报道称该研究可用于监视和审查
    </em>
   </strong>
   。
  </p>
  <p class="graf graf--p">
   科技巨头微软一直在与中国军事大学合作研究AI。据英国
   <a class="markup--anchor markup--p-anchor" data-href="https://www.ft.com/content/9378e7ee-5ae6-11e9-9dde-7aedca0a081a" href="https://www.ft.com/content/9378e7ee-5ae6-11e9-9dde-7aedca0a081a" rel="noopener noreferrer" target="_blank">
    “金融时报”报道
   </a>
   ，这可用于监视和审查。3月至11月期间，共发表了三项科学研究，这些研究由北京微软亚洲研究院的学者共同撰写。参与这项研究的其他研究人员与中国国防科技大学（NUDT）有着密切的联系。
  </p>
  <p class="graf graf--p">
   新美国智囊团高级研究员和中国技术专家 Samm Sacks 评论了微软与中国军事大学之间合作的可能性问题，并指出，最近发表的论文因为其主题技术的敏感性质而引发了“危险信号”，加之我们对中国如何部署这项技术的了解。“ Sacks 补充说：“[中国]政府正在利用这些技术建立监控系统并拘留少数民族人口[在新疆]。”
  </p>
  <p class="graf graf--p">
   外交关系委员会网络空间政策主任 Adam Segal 对此次合作发表评论说：“由于联邦调查局专注于学生和科学家以及国防部的间谍威胁，美中学术伙伴关系日益显现。 [侧重于]前沿技术可能最终进入解放军[中国军方]的可能性。“
  </p>
  <p class="graf graf--p">
   麻省理工学院最近切断了与中国电信集团华为的联系，并开始对其中国项目进行“高风险”审查程序，表明对中国技术和监控问题的审查越来越严格。
  </p>
  <p class="graf graf--p">
   令人担忧的是，
   <strong class="markup--strong markup--p-strong">
    微软研究人员最近与 NUDT 合作开展的一些项目有直接的监控用例
   </strong>
   。其中一篇由微软和 NUDT 研究人员共同撰写的论文研究了一种新的A.I. 通过分析专家认为可直接应用于监视的人脸来重建详细环境地图的方法。
  </p>
  <p class="graf graf--p">
   华盛顿大学 AI 教授 Pedro Domingos 评论这些申请时说：“假设我是一个情报机构，我有感兴趣的人的照片; 我就可以使用这个系统来描述他们所处的任何地方，因为他们没有意识到自己已经暴露了。“
  </p>
  <p class="graf graf--p">
   微软研究人员与 NUDT 研究人员合作的其他论文涉及机器学习，这是一个可以用于审查的技术领域。中国军事技术专家兼智囊团新美国安全中心的研究员 Elsa Kania 在这方面评论说：“机器阅读可以用于审查，这是中国政府的利益。他们将自然语言处理视为实现大规模审查的一种方式。“
  </p>
  <p class="graf graf--p">
   微软评论了这个问题，称其“通常是学者的研究人员与来自世界各地的顶尖学者和专家进行基础研究，以促进我们对技术的理解。” 该公司补充说：“在每种情况下，研究都遵循我们的原则，完全符合美国和当地法律，并且 ……确保透明度，以便每个人都能从我们的工作中受益。“
  </p>
  <p class="graf graf--p">
   没错，每个人都受益了，尤其是那些专制政权，他们一直都热衷于监视审查。
  </p>
  <p class="graf graf--p">

    Tech giant Microsoft has reportedly been working on artificial intelligence research with a Chinese military-run university, recent reports claim that the research could be used for both surveillance and censorship
   </a>
   .
  </p>
