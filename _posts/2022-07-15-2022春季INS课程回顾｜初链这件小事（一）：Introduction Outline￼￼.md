  <p class="has-normal-font-size">
   初链这件小事是2022年春季，中国美术学院网络社会研究所开设的新课程，本课程的主要目的是让非理工科背景的同学能够理解区块链与元宇宙基础知识，让大家能基于现有的开源工具与平台和基础的编程知识最大程度地参与到现有区块链与元宇宙项目中，思考新技术所带来的社会与伦理问题。
  </p>
  <p class="has-normal-font-size">
   本课程主要分为三个部分：1. 基础背景知识回顾与掌握 2. 区块链的原理与生态 3. 元宇宙的愿景与现实。这个系列的文章由课堂录音整理而来，全系列共9讲。
  </p>

  <h2 class="wp-block-heading" style="font-size:38px">
   内容总结：
  </h2>
  <p>
  </p>
  <p class="has-vce-cat-230-color has-text-color has-normal-font-size">
   <strong>
    课程背景：
   </strong>
  </p>
  <p class="has-normal-font-size">
   一方面，一些区块链公司正在大张旗鼓进入我们的生活，另一方面，很多商业公司也开始进军区块链。这就抛出一个问题，区块链行业或者技术是不是真的代表了一种未来？
  </p>
  <p class="has-vce-cat-230-color has-text-color has-normal-font-size">
   <strong>
    ​​课程目标：
   </strong>
  </p>
  <p class="has-normal-font-size">
   在区块链这个领域有一定的知识积累，掌握一些基础知识，会使用一些基础技能和工具；理解什么是WEB3; get your hands dirty，通过实践入门。
  </p>
  <p class="has-vce-cat-230-color has-text-color has-normal-font-size">
   <strong>
    工具介绍：
   </strong>
  </p>
  <ul class="has-normal-font-size">
   <li>
    搜索引擎
    <ul>
     <li>
      通过Google trends分析web3、metaverse的历史趋势和热度
     </li>
    </ul>
    <ul>
     <li>
      增加关键词和限定符，缩小检索范围（可自行搜索“高级搜索技巧”进行系统学习）
     </li>
    </ul>
   </li>
   <li>
    社交媒体
    <ul>
     <li>
      Twitter、Reddit
     </li>
    </ul>
    <ul>
     <li>
      如何有选择地关注（好的博主有自己的思考，喜欢转发抽奖的没什么干货，可以适当关注获取行业信息）
     </li>
    </ul>
   </li>
   <li>
    笔记工具
    <ul>
     <li>
      Notion、Hackmd（Codimd）
     </li>
    </ul>
    <ul>
     <li>
      学会markdown语法，方便排版
     </li>
    </ul>
   </li>
   <li>
    即时讯息
    <ul>
     <li>
      Discord、telegram
     </li>
    </ul>
    <ul>
     <li>
      加入不同discord观察社区，参与社区建设
     </li>
    </ul>
   </li>
  </ul>
  <p class="has-vce-cat-230-color has-text-color has-normal-font-size">
   <strong>
    研究方法：
   </strong>
  </p>
  <ul class="has-normal-font-size">
   <li>
    分析方法
    <ul>
     <li>
      定性分析
     </li>
    </ul>
    <ul>
     <li>
      定量分析
     </li>
    </ul>
    <ul>
     <li>
      定型与定量结合
     </li>
    </ul>
   </li>
   <li>
    数据种类
    <ul>
     <li>
      观察数据
     </li>
    </ul>
    <ul>
     <li>
      实验数据
     </li>
    </ul>
    <ul>
     <li>
      模拟数据
     </li>
    </ul>
    <ul>
     <li>
      派生数据
     </li>
    </ul>
   </li>
  </ul>
