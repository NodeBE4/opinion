  <p class="graf graf--p">
   这里是旧金山 —— 这个一直被认为是与特朗普截然相反的进步堡垒 —— 当地警察在民选官员的帮助下，瞄准一名记者、并以一种让特朗普嫉妒的热情侵犯他的权利。
  </p>
  <p class="graf graf--p">
   旧金山的居民应该感到愤怒，这个城市如此严重地侵犯了他们曾经引以为傲的新闻自由。
  </p>
  <p class="graf graf--p">
   不到两周前，旧金山的执法当局突袭了自由撰稿人 Bryan Carmody 的家。
  </p>
  <p class="graf graf--p">
   这些条子挥舞着他们的枪，给 Carmody 带上了手铐，并没收了数千美元的电子设备。
  </p>
  <p class="graf graf--p">
   他一直被锁了六个小时的手铐，当局搜查他的房子，并试图以“泄密”为幌子审问他。
  </p>
  <p class="graf graf--p">
   这是因为几个星期前，Carmody 出版了
   <a class="markup--anchor markup--p-anchor" data-href="https://apnews.com/b015a5c1e80c483f961ca29bf019724c" href="https://apnews.com/b015a5c1e80c483f961ca29bf019724c" rel="noopener noreferrer" target="_blank">
    一篇文章
   </a>
   —— 基于泄露的警察报告、录像和一些采访 —— 关于当地公共辩护人的死亡真相。
  </p>
  <p class="graf graf--p">
   这是对第一修正案骇人听闻的违反。
  </p>
  <p class="graf graf--p">
   但
   <strong class="markup--strong markup--p-strong">
    当地立法者的初步反应
   </strong>
   —— 其中许多人经常反对特朗普侵犯公民自由 ——
   <strong class="markup--strong markup--p-strong">
    几乎与此行为本身一样糟糕
   </strong>
   。
  </p>
  <p class="graf graf--p">
   政治领导人本能地为袭击辩护，并对报道和第一修正案的实际运作方式
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/AASchapiro/status/1131304732478197761" href="https://twitter.com/AASchapiro/status/1131304732478197761" rel="noopener noreferrer" target="_blank">
    表现出令人尴尬的无知
   </a>
   。
  </p>
  <p class="graf graf--p">
   甚至就连旧金山市长 London Breed 也
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/yashar/status/1130849091871760385" href="https://twitter.com/yashar/status/1130849091871760385" rel="noopener noreferrer" target="_blank">
    发表声明称
   </a>
   ，“我支持这一决定” 袭击 Carmody 的家，辩称警察经历了“适当的法律程序”。
  </p>
  <p class="graf graf--p">
   越来越多的抗议使得 Breed 最终改变了她的说辞。
  </p>
  <p class="graf graf--p">
   她在推特上
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/LondonBreed/status/1130128639230562304" href="https://twitter.com/LondonBreed/status/1130128639230562304" rel="noopener noreferrer" target="_blank">
    写道
   </a>
   ，她“感觉警方突击搜查记者不合适”，但执法部门选择了变本加厉。
  </p>
  <p class="graf graf--p">
   警方在新闻发布会上宣布，他们正在对 Carmody 的“泄密事件”进行刑事调查，而不是道歉。
  </p>
  <p class="graf graf--p">
   与此同时，他们完全回避了关于突袭记者几乎肯定是非法的这一根本性问题。
  </p>
  <p class="graf graf--p">
   加利福尼亚州曾经制定了“保护法”，旨在保护记者不至于在这些类型的情况下泄露其信息来源。
  </p>
  <p class="graf graf--p">
   法规的语言很明确：在任何情况下，警方都不能获得逮捕令以强迫记者暴露其信息来源。在进行任何搜查之前需要获得搜查令，记者可以在法庭上对其提出质疑。
  </p>
  <p class="graf graf--p">
   但是，获得逮捕令，他们真的做到了。
  </p>
  <p class="graf graf--p">
   不幸的是，
   <strong class="markup--strong markup--p-strong">
    我们不知道警方是如何说服法官同意他们的抄家计划的，因为法院到目前为止一直保密逮捕令的细节
   </strong>
   。
  </p>
  <p class="graf graf--p">
   正如第一修正案联盟的 David Snyder 所质问的那样：“这些法官是否了解 Carmody 作为记者的身份？警察局是否隐瞒了这一身份？即使他们知道 Carmody 是一名记者，法官是否依旧授权搜查令？”
  </p>
  <p class="graf graf--p">
   公众应该知道这些问题的答案。
  </p>
  <p class="graf graf--p">
   公共辩护人 Jeff Adachi 长期以来一直是警方机构的眼中钉，而且人们普遍认为警察部门内的任何人都有可能向 Carmody 泄露了报告，这可能是为了让 Adachi 的死亡看起来很糟糕。（警方报告称，Adachi 被发现携带毒品，并包含一些其他私人的不怎么雅观的细节）
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    泄密者的动机应该与 Carmody 作为记者的权利完全无关。这才是重点。
   </strong>
  </p>
  <p class="graf graf--p">
   警察局长声称 Carmody “越界”，但是
   <strong class="markup--strong markup--p-strong">
    没有提供一丝证据
   </strong>
   表明他做了任何违法行为。
  </p>
  <p class="graf graf--p">
   这位官员似乎并不理解
   <strong class="markup--strong markup--p-strong">
    获取当局可能不希望公开的重要信息正是记者的职责。
   </strong>
  </p>
  <p class="graf graf--p">
   令人遗憾的是，旧金山警察局 —— 以及为他们辩护的当地政界人士 ——
   <strong class="markup--strong markup--p-strong">
    正在向特朗普政府提交一份危险的行动路线图，以便可能大模大样地起诉记者。
   </strong>
  </p>
  <p class="graf graf--p">
   特朗普政府正以前所未有的速度起诉记者，而特朗普本人则表示他希望看到记者入狱。
  </p>
  <p class="graf graf--p">
   正是这种阴谋论 —— 记者为与机密信息持有者交谈或要求提供机密信息是“犯罪行为”的想法 —— 让新闻自由倡导者无比担心。
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    <strong class="markup--strong markup--blockquote-strong">
     新闻工作永远不会让当权者感觉舒服，但这并不意味着民选官员应该放弃保护我们宪法权利的义务。
    </strong>
   </p>
  </blockquote>
  <p class="graf graf--p">
   特朗普不是第一个走这条邪路的人。
  </p>
  <p class="graf graf--p">
   在奥巴马政府执政期间，司法部甚至将福克斯新闻的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/commentisfree/2013/may/20/obama-doj-james-rosen-criminality" href="https://www.theguardian.com/commentisfree/2013/may/20/obama-doj-james-rosen-criminality" rel="noopener noreferrer" target="_blank">
    记者 James Rosen 列为“间谍活动的同谋”
   </a>
   ，该案件涉及他的前消息来源，国务院官员 Stephen Kim。
  </p>
  <p class="graf graf--p">
   司法部甚至拿出了授权，可以搜查记者 Rosen 的私人电子邮件。
  </p>
  <p class="graf graf--p">
   在公众强烈反对这种威胁行为之后，司法部收回了刑事诉讼。
  </p>
  <p class="graf graf--p">
   当时的司法部长 Eric Holder 后来说这件事是他上任六年来
   <a class="markup--anchor markup--p-anchor" data-href="https://townhall.com/tipsheet/katiepavlich/2014/10/30/eric-holders-claims-biggest-regret-is-targeting-fox-news-james-rosen-n1911942" href="https://townhall.com/tipsheet/katiepavlich/2014/10/30/eric-holders-claims-biggest-regret-is-targeting-fox-news-james-rosen-n1911942" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     最大的遗憾之一
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p">
   特朗普显然没有这种顾虑。
  </p>
  <p class="graf graf--p">
   事实上，特朗普的司法部门以前所未有的举动向 Wikileaks 记者 Julian Assange 提起了十多项指控，控告他“获取”机密信息并将其公之于众。
  </p>
  <p class="graf graf--p">
   许多新闻自由倡导者认为，
   <strong class="markup--strong markup--p-strong">
    这可能会使新闻采访和出版过程的关键工作变成了刑事犯罪
   </strong>
   ，并使得像纽约时报和华盛顿邮报这样的报纸的数十名调查记者非常容易遭受起诉。
  </p>
  <p class="graf graf--p">
   旧金山居然给了特朗普这样一个可以直接起诉记者的蓝图，震惊了所有良知。
  </p>
  <p class="graf graf--p">
   不论是自由派政客还是保守派政客，新闻业永远不会让当权者感到舒服 —— 但这并不意味着民选官员应该放弃保护我们宪法权利的义务。只是因为记者发表了他们可能不喜欢的东西。⚪️
  </p>
  <p class="graf graf--p">

    Journalism Is Under Unprecedented Assault
   </a>
   . Recent controversies involving both Julian Assange and the San Francisco police show the First Amendment is under dire threat
  </p>
