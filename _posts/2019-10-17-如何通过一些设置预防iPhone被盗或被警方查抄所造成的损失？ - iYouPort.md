  <p class="graf graf--p">
   如今我们的电话已经不仅仅是一个电话，说它是身家性命都不为过。您一定理解这个道理。
  </p>
  <p class="graf graf--p">
   手机是警方当局和犯罪分子瞄准的重点目标。
  </p>
  <p class="graf graf--p">
   为了防止这种情况发生，你应该采取预防措施，越早越好。
  </p>
  <p class="graf graf--p">
   提前声明，以下步骤的适用范围不包含您的对手是NSO那类公司的客户时的状况。一般情况下您也很难遇到那种状况（至少他们的间谍工具非常贵，并非随便哪个小偷和警察部门就能买得起的）
  </p>
  <p class="graf graf--p">
   以下是一些步骤，是您最基本要做到的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    提示一：设置生物识别身份验证
   </strong>
  </p>
  <p class="graf graf--p">
   请注意，这是防小偷的，有可能不防警察（防警察的办法后面说）尤其是在警方可以强迫按压您的手指解锁和/或扫描你的面部而不受惩罚的国家。
  </p>
  <p class="graf graf--p">
   您也许曾经被建议设置指纹而不是面部ID；因为面部识别可以在不接触的情况下解锁，似乎更不安全。
  </p>
  <p class="graf graf--p">
   但是这不确定。
  </p>
  <p class="graf graf--p">
   由于禁用很容易，于是，⚠️这需要您根据自己面对的实际风险作出尽可能准确的评估。
  </p>
  <p class="graf graf--p">
   是的，您的密码依旧是一层安全保护，对某些人而言可能已经足够了，但是 Touch ID 可以增强您的安全性并使窃贼和黑客更难进入您的物理设备。
  </p>
  <p class="graf graf--p">
   下一步更最重要。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5830 jetpack-lazy-image" data-lazy-sizes="(max-width: 1070px) 100vw, 1070px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1、生物识别.png?resize=1070%2C1046&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1、生物识别.png?w=1070&amp;ssl=1 1070w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1、生物识别.png?resize=300%2C293&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1、生物识别.png?resize=768%2C751&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1、生物识别.png?resize=1024%2C1001&amp;ssl=1 1024w" data-recalc-dims="1" height="1046" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1、生物识别.png?resize=1070%2C1046&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1070"/>

    <img alt="" class="aligncenter size-full wp-image-5830" data-recalc-dims="1" height="1046" sizes="(max-width: 1070px) 100vw, 1070px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1、生物识别.png?resize=1070%2C1046&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1、生物识别.png?w=1070&amp;ssl=1 1070w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1、生物识别.png?resize=300%2C293&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1、生物识别.png?resize=768%2C751&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1、生物识别.png?resize=1024%2C1001&amp;ssl=1 1024w" width="1070"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    提示二：了解如何快速禁用生物识别技术
   </strong>
  </p>
  <p class="graf graf--p">
   现在您已经设置了面部ID或触控ID，这是一件非常重要的事：如果抢劫者或执法部门拿走您的 iPhone，它们可能会被用来对付您。
  </p>
  <p class="graf graf--p">
   犯罪分子和警察当局可以强行使用脸部或指纹解锁你的 iPhone，后者甚至还有权在合法的情况下没收你的设备。
  </p>
  <p class="graf graf--p">
   请注意，⚠️掌握时机是非常重要的！
  </p>
  <p class="graf graf--p">
   只要您知道如何操作，您就可以在瞬间让 Face ID 或 Touch ID 失效，而不会弄乱屏幕上的任何内容。
  </p>
  <p class="graf graf--p">
   只需按一下按钮即可锁定生物识别访问。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2.1 快速禁用面部ID
   </strong>
  </p>
  <p class="graf graf--p">
   要禁用面部ID，只需同时按住侧面按钮，同时按下开关键音量调高或音量调低按钮。如果你愿意的话，你甚至可以按住所有三个。保持约两秒钟。
  </p>
  <p class="graf graf--p">
   ⚠️一定要保持足够长时间；您需要能看到“滑动关机”开关出现。一旦出现该屏幕，就放手。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5831 jetpack-lazy-image" data-lazy-sizes="(max-width: 1066px) 100vw, 1066px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2、滑动关机.png?resize=1066%2C1124&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2、滑动关机.png?w=1066&amp;ssl=1 1066w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2、滑动关机.png?resize=285%2C300&amp;ssl=1 285w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2、滑动关机.png?resize=768%2C810&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2、滑动关机.png?resize=971%2C1024&amp;ssl=1 971w" data-recalc-dims="1" height="1124" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2、滑动关机.png?resize=1066%2C1124&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1066"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    重要提示
   </strong>
   ：如果您的脸部在按下按钮组合之前已经解锁屏幕，或者您已经在主屏幕或应用程序中处于解锁状态，则必须单击侧面按钮一次以关闭屏幕或让它自己睡着。如果不这样做警方依然可以访问您的手机，因为它目前已经解锁。
  </p>
  <p class="graf graf--p">
   第二个选项。
  </p>
  <p class="graf graf--p">
   连续5次单击侧面按钮（也称为睡眠/唤醒（电源）按钮）将激活紧急SOS屏幕，这将禁用 Touch ID，直到输入密码为止。
  </p>
  <p class="graf graf--p">
   使用 iPhone X，X S，X S Max和X R，快速单击Side按钮五次也将激活紧急SOS；它会直接进入倒计时模式，而不是“滑动关机”，所以你必须手动阻止它调用911。但是这样做将有效地禁用 Face ID。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5832 jetpack-lazy-image" data-lazy-sizes="(max-width: 1066px) 100vw, 1066px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3、911.png?resize=1066%2C724&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3、911.png?w=1066&amp;ssl=1 1066w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3、911.png?resize=300%2C204&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3、911.png?resize=768%2C522&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3、911.png?resize=1024%2C695&amp;ssl=1 1024w" data-recalc-dims="1" height="724" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3、911.png?resize=1066%2C724&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1066"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5832" data-recalc-dims="1" height="724" sizes="(max-width: 1066px) 100vw, 1066px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3、911.png?resize=1066%2C724&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3、911.png?w=1066&amp;ssl=1 1066w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3、911.png?resize=300%2C204&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3、911.png?resize=768%2C522&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3、911.png?resize=1024%2C695&amp;ssl=1 1024w" width="1066"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    重要提示
   </strong>
   ：和前面的选项一样，如果您的脸部在执行五次点击之前已经解锁了屏幕，或者您已经在主屏幕或应用程序中处于解锁状态，那么您必须单击侧面按钮一次以关闭屏幕或让它自己入睡，否则无效。
  </p>
  <p class="graf graf--p">
   如果您不喜欢这种方法，或者您害怕可能不小心拨打了911，可以在“设置”应用中禁用这一快捷方式。
  </p>
  <p class="graf graf--p">
   此外还有一个选项，在上述两个选项都不合适您的情况下。
  </p>
  <p class="graf graf--p">
   也就是制造*多次识别失败*和关闭或重新启动设备。
  </p>
  <p class="graf graf--p">
   如果您只想暂时禁用 Face ID ，关闭并重新启动 iPhone X，X S，X S Max和X R的方法就有点过了，因为您必须执行与选项1相同的操作，而且要步骤更多。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    警告
   </strong>
   ：注意力检测功能是 iPhone X，X S，X S Max和X R的设置，可防止脸部ID在您直接面对iPhone之前解锁iPhone。
  </p>
  <p class="graf graf--p">
   设置需要您的全脸 — 包括眼睛 — 以解锁您的设备。也就是说，角度是自然的 时就能解锁。但是这个角度很有限。
  </p>
  <p class="graf graf--p">
   如果没有激活此注意力检测功能，有人可能会在您不看屏幕时（甚至在您睡觉时）将 iPhone 指向您的脸部来解锁 iPhone。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2.2 快速禁用指纹ID
   </strong>
  </p>
  <p class="graf graf--p">
   在美国，执法部门可以让您通过指纹解锁智能手机，但他们不能强迫您输入密码或PIN，这将违反您的第五修正案权利。
  </p>
  <p class="graf graf--p">
   在中国，没有第五修正案，他们什么都可以做。所以您需要能快速禁用指纹解锁，并且尽可能争取10次密码 错误以自动擦除手机。
  </p>
  <p class="graf graf--p">
   iOS 11及以上都可以做到让您在几秒钟内禁用Touch ID。
  </p>
  <p class="graf graf--p">
   想要暂时禁用Touch ID，你只需快速按下电源按钮五次即可，这时屏幕会显示“紧急SOS”页面，你可以呼叫紧急联系人，但同时，这样也可以防止你的iPhone在没有密码的情况下被解锁，也就是说此时已无法用指纹解锁设备。
  </p>

    提示3：更强的密码
   </strong>
  </p>
  <p class="graf graf--p">
   如果您在第一次打开 iPhone 时设置了密码，则可能是四位数或六位数字代码。
  </p>
  <p class="graf graf--p">
   尽管具有一百万种六位数代码的可能组合比仅有10,000种的四位数代码的可能组合更安全，但它仍然相当弱。
  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   如果您想要在 iPhone 上获得最佳安全性，则应切换到自定义字母数字或纯数字密码。
  </p>
  <p class="graf graf--p">
   两种类型都允许比通常的密码选项更多的数字/字符。如果你想坚持全数字的模式，将你的密码长度达到10位数就可以给它带来1亿种不同的组合。

    提示5：备份数据
   </strong>
  </p>
  <p class="graf graf--p">
   危险随时都会发生，而且基本不会让您事先收到什么提示。这就是为什么您应该随时做好防备。
  </p>
  <p class="graf graf--p">
   现在您不再需要使用 iTunes 将设备备份到计算机上 —— 您可以直接从 iPhone 进行操作。
  </p>
  <p class="graf graf--p">
   在“设置”中点击“iCloud”，然后点击“iCloud Backup”。
  </p>
  <p class="graf graf--p">
   启用“iCloud Backup”（如果尚未启用），然后点击“立即备份”以备份您的设备。（再一次：需要您不在“云上贵州”）
  </p>
  <p class="graf graf--p">
   如果启用了“iCloud Backup”，则当 iPhone 连接到电源并且您已连接到 Wi-Fi 时，将自动备份您的数据。

  </p>
  <p class="graf graf--p">
   ⚠️犯罪分子和警察可以利用此功能绕过您的锁屏密码并进入手机 —— 这就是您应该从锁定屏幕禁用USB附件的原因。
  </p>
  <p class="graf graf--p">
   在“设置”中，转到“面部识别码和密码”或“触摸ID和密码”，输入您的密码，然后一直向下滚动。确保“USB附件”旁边的拨动开关关闭（不是绿色）。

    提示7：关闭锁定屏幕消息回复
   </strong>
  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   并且还可以采取其他社交工程攻击。
  </p>
  <p class="graf graf--p">
   要防止出现这种情况，只需在“面部识别码和密码”或“触摸ID和密码”设置中禁用“回复消息”功能即可。
  </p>
  <p class="graf graf--p">
   虽然这样做您自己也无法通过锁定屏幕回复消息，但最好先花点时间解锁您的设备，而不是让您的设备更容易被犯罪分子和警察攻击。

    提示8：将iPhone设置为自动擦除
   </strong>
  </p>
  <p class="graf graf--p">
   一个非常幸运的犯罪分子可以猜出你的密码，但你可以让这个过程变得更难。稍微有点难就行。
  </p>
  <p class="graf graf--p">
   iOS上的“擦除数据”功能会在10次尝试解锁失败后擦除设备上的所有数据，这使得犯罪分子和警方几乎无法访问您珍贵的数据。
  </p>
  <p class="graf graf--p">
   只需确保准备好 iCloud 备份，以便以后您自己仍可以访问它。
  </p>
  <p class="graf graf--p">
   再次在“面部识别码和密码”或“触摸ID和密码”设置中，打开“擦除数据”旁边的切换按钮，然后点击“启用”将其激活。
  </p>
  <p class="graf graf--p">
   如果您是经常忘记密码的人，即使您启用了面部识别码或触摸ID，也可能不想这样做，因为您有时仍需要密码。所以，请斟酌自己的情况。

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    提示9：设置查找我的 iPhone
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “查找我的iPhone”不仅会向您显示设备的确切位置，还可以让您远程擦除设备，并在您的设备上留下消息和联系信息锁屏。
  </p>
  <p class="graf graf--p">
   要启用“查找我的iPhone”，请在“设置”应用中点击您的姓名，然后点按“iCloud”并向下滚动，直至看到“查找我的iPhone”。
  </p>
  <p class="graf graf--p">
   在这里，您可以启用“查找我的iPhone”以及“ 发送最后位置 ”；当电池电量非常低时，它会自动将 iPhone 的位置发送到“查找我的iPhone”。
  </p>
