  <p class="graf graf--p">
   不幸的是，对儿童下手的监视资本主义数据收集并不是新鲜事，尽管反对声越来越高，但这种收集仍在继续。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e6%ad%a3%e5%9c%a8%e5%af%b9%e4%bd%a0%e7%9a%84%e5%ad%a9%e5%ad%90%e4%b8%8b%e6%89%8b/" href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e6%ad%a3%e5%9c%a8%e5%af%b9%e4%bd%a0%e7%9a%84%e5%ad%a9%e5%ad%90%e4%b8%8b%e6%89%8b/" rel="noopener noreferrer" target="_blank">
     谷歌正在对你的孩子下手
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   据当地媒体报道，由于父母对隐私的担忧、以及担心自己的孩子被控制，一项涉及小学生的头戴式设备以监控注意力范围的试验已经在中国被叫停。
  </p>
  <p class="graf graf--p">
   据称，该设备监控脑电波，并将数据发送到浙江省金华市中心小学的计算机系统，并且一些数据已与父母共享。
  </p>
  <p class="graf graf--p">
   报道引用了一位老师的话说，
   <strong class="markup--strong markup--p-strong">
    该项实验已经进行了一年，而且数据也已经转移到了公司服务器上
   </strong>
   。一位姓张的老师说，“学生们报告说该设备没有副作用，并且没有与父母共享全部数据”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种头戴式监视设备是由总部位于美国波士顿、中国区总部位于浙江杭州的公司
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.instagram.com/brainco_tech/?hl=en" href="https://www.instagram.com/brainco_tech/?hl=en" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     BrainCo
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    开发的
   </strong>
   ，该公司的网站上描述自己称 “BrainCo是首家入选哈佛大学创新实验室(Harvard Innovation Lab)的华人团队，是全球领先的脑科技术解决方案的供应商，在教育、医疗等领域具有领先优势。”
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.brainco.cn/company/detail/company-intro" href="https://www.brainco.cn/company/detail/company-intro" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     该公司网站
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    的 “核心团队” 栏目中有一位叫 Samuel Prentice 的人，他的简介中写着：曾担任美国国防部高级研究计划局（DARPA）项目主程序员。
   </strong>
  </p>
  <p class="graf graf--p">
   BrainCo 中国区的一位代表也姓张，他对中国媒体表示，学生每周两次佩戴该设备，每次30分钟。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    美国的校园监视《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%9b%91%e7%8b%b1%e5%bc%8f%e6%95%99%e5%ad%a6/" href="https://www.iyouport.org/%e7%9b%91%e7%8b%b1%e5%bc%8f%e6%95%99%e5%ad%a6/" rel="noopener noreferrer" target="_blank">
     监狱式教学
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   随着中国各地许多儿童使用智能手表、移动电话和其他跟踪位置和个人信息的设备，父母越来越关注这些数据的使用方式以及谁可以访问这些数据。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “目前，[上述学校]是中国唯一使用这种监控设备的学校。而这些监控产品已经在美国得到了更广泛的使用”，浙江 BrainCo 技术公关总监说。
  </p>
  <p class="graf graf--p">
   哇哦 …… 这种东西也在美国使用？美国的父母对此怎么看？
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/fQ3eW3qQ2pk" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   就算您不关心监视问题，至少，研究已经确定，暴露于电磁辐射中可能会
   <a class="markup--anchor markup--p-anchor" data-href="https://www.saferemr.com/" href="https://www.saferemr.com/" rel="noopener noreferrer" target="_blank">
    造成长期伤害
   </a>
   。包括
   <a class="markup--anchor markup--p-anchor" data-href="https://ehtrust.org/american-academy-pediatrics-recommendations-cell-phones-cell-towers-wireless-radiation/" href="https://ehtrust.org/american-academy-pediatrics-recommendations-cell-phones-cell-towers-wireless-radiation/" rel="noopener noreferrer" target="_blank">
    美国儿科学会
   </a>
   在内的健康专家纷纷警告说，
   <a class="markup--anchor markup--p-anchor" data-href="https://ehtrust.org/policy/fcc-safety-standards/" href="https://ehtrust.org/policy/fcc-safety-standards/" rel="noopener noreferrer" target="_blank">
    儿童更容易
   </a>
   受到手机和WiFi辐射的伤害，我们
   <a class="markup--anchor markup--p-anchor" data-href="https://www.activistpost.com/2019/08/hospital-reduces-childrens-exposure-to-wifi-radiation-because-they-are-more-vulnerable-to-it.html" href="https://www.activistpost.com/2019/08/hospital-reduces-childrens-exposure-to-wifi-radiation-because-they-are-more-vulnerable-to-it.html" rel="noopener noreferrer" target="_blank">
    应该减少
   </a>
   儿童对电子设备的接触，而不是增加接触。⚪️
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.activistpost.com/2019/11/parents-stop-school-mounting-electromagnetic-devices-on-students-heads-to-collect-brainwave-data.html" href="https://www.activistpost.com/2019/11/parents-stop-school-mounting-electromagnetic-devices-on-students-heads-to-collect-brainwave-data.html" rel="noopener noreferrer" target="_blank">
