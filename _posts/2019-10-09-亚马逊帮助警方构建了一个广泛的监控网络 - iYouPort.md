    <span style="color: #00ccff;">
     <em>
      <strong>
       如果有人免费送你或大幅优惠给你一个监视器，你愿意帮助警察构建广泛的监视网络吗？
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   这种诱导式的助纣的确值得警惕。想想看，这个东西很贵你买不起 + 此前你被宣传称安装这种东西代表“安全的现代化生活”你很羡慕，这种时候有人要白送你一个……谁会拒绝呢？
  </p>
  <p class="graf graf--p">
   俗话说得好，天下只有老鼠夹子上的奶酪是免费的。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5568 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/ring-2.jpeg?resize=1100%2C618&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/ring-2.jpeg?w=1960&amp;ssl=1 1960w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/ring-2.jpeg?resize=300%2C169&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/ring-2.jpeg?resize=768%2C432&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/ring-2.jpeg?resize=1024%2C576&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/ring-2.jpeg?resize=1100%2C618&amp;ssl=1 1100w" data-recalc-dims="1" height="618" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/ring-2.jpeg?resize=1100%2C618&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   如果你在新泽西州的布卢姆菲尔德（Bloomfield）散步，千万要小心，因为你很有可能被偷拍监视。但不是公司办公室或仓库安全摄像头的捕捉镜头 —— 而是亚马逊制造的Ring门铃。
  </p>
  <p class="graf graf--p">
   虽然住宅区通常没有安全摄像头，但
   <strong class="markup--strong markup--p-strong">
    智能门铃的受欢迎程度基本上创造了由亚马逊提供支持并由警察部门推动的大型私人监控网络
   </strong>
   。
  </p>
  <p class="graf graf--p">
   全国各地的警察部门，从休斯顿等主要城市到人口不到30,000人的小城镇，
   <strong class="markup--strong markup--p-strong">
    都向公民提供免费或打折的 Ring 门铃
   </strong>
   ，有时使用纳税人的资金来支付亚马逊的产品。
  </p>
  <p class="graf graf--p">
   虽然理论上 Ring 所有者应该可以选择是否将视频提供给警察，但是，在某些赠品的条款中，警察要求保持在任何时候翻看镜头的权力。
  </p>
  <p class="graf graf--p">
   Ring 说它将开始打击附带条件。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “客户应该可以控制他们的视频。Ring 已经向 Neighbor 的执法合作伙伴捐赠了设备，供他们向社区成员提供，“Ring 在一份声明中说，“Ring 不支持要求接收者提供录像的程式，也不支持共享来自 Ring 设备的镜头作为捐赠设备的条件。我们正积极与合作伙伴合作，以确保这反映在他们的程式中。”
  </p>
  <p class="graf graf--p">
   但是你信吗？鉴于亚马逊如此专注于向警方推荐他们的产品，他们会帮你阻止警方的使用方式？
  </p>
  <p class="graf graf--p">
   虽然街区的监控录像可以帮助警方调查犯罪事件，但亚马逊Ring业务运营的摄像机数量庞大，引发了涉及执法和科技巨头的双重隐私问题。
  </p>
  <p class="graf graf--p">
   你可能会认为亚马逊是一个获得廉价优惠的地方，但批评者已经指出，这家零售巨头为警方提供面部识别工具。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这些摄像机可以让多方面受益
   </strong>
   ：警方可以收集更多的视频，而亚马逊可以为新的 Ring 拥有者收取高达每月3美元的智能门铃用户费。与此同时，居民可以放心，特别是 Neighbors 应用程序，本质上是一个社交网络共享摄像头。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们这片社区现在完全被监视相机覆盖，“布卢姆菲尔德警察局侦探局指挥官 Vincent Kerney 上尉说，”我们管辖区内的每个城镇都有许多 Ring 摄像机”。
  </p>
  <p class="graf graf--p">
   布卢姆菲尔德的警察部门没有收到任何来自 Ring 的免费摄像头，但这个摄像头已经在大约5万人的镇上流行。
  </p>
  <p class="graf graf--p">
   在过去的两年里，美国
   <strong class="markup--strong markup--p-strong">
    有超过50个当地警察部门与Ring合作
   </strong>
   ，称赞亚马逊拥有的产品如何允许他们访问通常没有摄像头的区域的监视视频。
  </p>
  <figure aria-describedby="caption-attachment-5569" class="wp-caption aligncenter" id="attachment_5569" style="width: 664px">
   <img alt="" class="size-full wp-image-5569 jetpack-lazy-image" data-lazy-sizes="(max-width: 664px) 100vw, 664px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/地图.png?resize=664%2C1144&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/地图.png?w=664&amp;ssl=1 664w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/地图.png?resize=174%2C300&amp;ssl=1 174w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/地图.png?resize=594%2C1024&amp;ssl=1 594w" data-recalc-dims="1" height="1144" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/地图.png?resize=664%2C1144&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="664"/>

  <p class="graf graf--p">
   隐私倡导者指出，这种伙伴关系给执法部门提供了前所未有的监视能力。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们在这里拥有的是执法部门和世界上最大的公司之一的完美结合，为一个很少有人愿意参与的监视社会创造参与条件，”南加州 ACLU 的律师 Mohammad Tajsar 说。
  </p>
  <p class="graf graf--p">
   Ring 还在
   <a class="markup--anchor markup--p-anchor" data-href="https://blog.ring.com/2019/02/14/how-rings-neighbors-creates-safer-more-connected-communities/" href="https://blog.ring.com/2019/02/14/how-rings-neighbors-creates-safer-more-connected-communities/" rel="noopener noreferrer" target="_blank">
    博客文章
   </a>
   中提到了它如何处理与警方合作关系的隐私问题。
  </p>
  <p class="graf graf--p">
   该公司称：“我们的客户和 Neighbors 应用程序用户信任我们，以帮助保护他们的家园和社区，我们非常认真地承担这一责任”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Neighbors 如何工作
   </strong>
  </p>
  <p class="graf graf--p">
   亚马逊在2018年以10亿美元的价格收购了Ring，而智能门铃和安全摄像头的制造商帮助扩大了该零售巨头的智能家居业务推广。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种情况发生在消费者对新连网设备的兴趣激增之际
   </strong>
   ，从灯泡、到电视、到安全摄像头。
  </p>
  <p class="graf graf--p">
   不止亚马逊，像谷歌这样的公司更热衷于此，2014年谷歌以32亿美元收购 Nest，一个为家庭提供安全摄像头的公司。 Strategy Analytics 预计
   <strong class="markup--strong markup--p-strong">
    2018年会售出
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.strategyanalytics.com/access-services/devices/connected-home/smart-home/reports/report-detail/smart-home-surveillance-camera-market-analysis-and-forecast" href="https://www.strategyanalytics.com/access-services/devices/connected-home/smart-home/reports/report-detail/smart-home-surveillance-camera-market-analysis-and-forecast" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     超过340万个可视门铃
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p">
   在被亚马逊收购之前，Ring 一直在追求当地警察部门的垂青。
  </p>
  <p class="graf graf--p">
   警方最感兴趣的是 Ring 的 Neighbors 应用程序，免费下载的，人们可以在这里分享、查看和评论他们附近的犯罪信息，以及上传来自Ring门铃的视频剪辑。然后警方鼓励公众购买 Ring。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们鼓励 Mountain Brook 的居民购买这种类型的技术，并与应用程序一起使用，”阿拉巴马州 Mountain Brook 警察局局长 Ted Cook 说，“我们认为它正在努力创造一种数字邻里观察”。
  </p>
  <p class="graf graf--p">
   不是观察，是圆形监狱。
  </p>
  <p class="graf graf--p">
   当警察与 Ring 合作时，他们可以访问执法仪表板，在那里他们可以对地区进行地理围栏定位或寻求在特定时间内拍摄的镜头。如果居民选择发送视频，执法部门只需要从应用程序中就能获取镜头。否则，警察需要传唤Ring。
  </p>
  <p class="graf graf--p">
   警方表示，该应用程序帮助他们解决了犯罪问题，因为居民通常会记录窃贼的镜头，或者录制下经过附近的可疑汽车。
  </p>
  <figure aria-describedby="caption-attachment-5570" class="wp-caption aligncenter" id="attachment_5570" style="width: 1508px">

  <p class="graf graf--p">
   人们甚至认为该计划为警方提供了方便，于是能“获得更多安全”。
  </p>
  <p class="graf graf--p">
   约翰杰伊刑事司法学院副教授 Eric Piza 说：“投资 Ring 的人显然关心自己的安全和财产。”
  </p>
  <p class="graf graf--p">
   但是，尽管有各种好处，警察部门和Ring之间的关系无疑引起了人们对监控和隐私的担忧，因为
   <strong class="markup--strong markup--p-strong">
    亚马逊正在与执法机构合作，用监视器覆盖社区
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Ring 有自己的隐私问题。去年12月的报道揭露，乌克兰的工作人员在没有客户知情的情况下在公共应用上观看私密的监控视频。在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E7%BE%8E%E5%9B%BD%E7%9A%84%E6%96%AF%E5%A1%94%E8%A5%BF%E5%9C%A8%E4%B9%8C%E5%85%8B%E5%85%B0/" href="https://www.iyouport.org/%E7%BE%8E%E5%9B%BD%E7%9A%84%E6%96%AF%E5%A1%94%E8%A5%BF%E5%9C%A8%E4%B9%8C%E5%85%8B%E5%85%B0/" rel="noopener noreferrer" target="_blank">
    美国的斯塔西在乌克兰
   </a>
   》。
  </p>
  <p class="graf graf--p">
   该公司在报告发表后
   <a class="markup--anchor markup--p-anchor" data-href="https://techcrunch.com/2019/01/10/amazon-ring-privacy-concerns/" href="https://techcrunch.com/2019/01/10/amazon-ring-privacy-concerns/" rel="noopener noreferrer" target="_blank">
    向 TechCrunch 发表声明
   </a>
   称，“我们非常重视客户个人信息的隐私和安全。”
  </p>
  <p class="graf graf--p">
   电子前沿基金会的高级调查研究员 Dave Maass 说：“从本质上讲，我们正在创造一种文化，每个人都是用双筒望远镜望着窗外的爱管闲事的邻居。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “它创造了这一巨大的数据库，使政府能够分析我们的一举一动，无论是否犯下了罪行。”
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    无处不在
   </strong>
  </p>
  <p class="graf graf--p">
   在布卢姆菲尔德的热图上，新泽西城镇几乎没有任何区域看不到Ring相机。
  </p>
  <p class="graf graf--p">
   Kerney 说，智能门铃已经在社区中非常流行，并且与Ring合作执法仪表板是有意义的。
  </p>
  <p class="graf graf--p">
   从Ring进入城镇之前该镇的监控录像数量已经大幅增加可供布卢姆菲尔德警察局使用。
  </p>
  <p class="graf graf--p">
   Kerney 说他在2017年开始了志愿监视登记。任何有安全摄像头的地方都可以报名并向警方提供录像。
  </p>
  <p class="graf graf--p">
   Kerney回忆说，大约有442个登记的地方。主要是企业，因为许多私人住宅当时没有安全摄像头。但与 Ring 相比，私人和企业的比例正好相反。
  </p>
  <p class="graf graf--p">
   他说：“Ring 摄像机的安装数量可能是其他任何东西的10倍。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    大规模安装令 Ring 和 Google 的 Nest 等受欢迎的产品制造了一个监控系统。它们不仅仅适用于企业：到2023年，智能家居安防摄像机的市场预计将
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://news.strategyanalytics.com/press-release/intelligent-home/smart-home-surveillance-camera-market-surpass-97-billion-2023-says" href="https://news.strategyanalytics.com/press-release/intelligent-home/smart-home-surveillance-camera-market-surpass-97-billion-2023-says" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     超过97亿美元
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   警察部门正在 Ring 构建的监视网络上建立他们自己的监控网络。
  </p>
  <p class="graf graf--p">
   在弗吉尼亚州的汉普顿，警方在3月与该公司合作后收到了15台免费的Ring摄像机。警察部门仍然在弄清楚他们要将这些摄像机分配给哪些社区。
  </p>
  <p class="graf graf--p">
   汉普顿警方公共信息官 Ashley Jenrette 表示，部分原因包括与犯罪分析部门合作，确定哪些街区最需要使用这些摄像机。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    付出代价
   </strong>
  </p>
  <p class="graf graf--p">
   Ring可帮助警方避免出现监控技术的障碍，无论是缺乏资金还是公众对隐私的担忧。
  </p>
  <p class="graf graf--p">
   EFF 的 Maass 说，“我们有理由推动要求警察部门在获得任何监控技术之前获得市议会批准的条例。”
  </p>
  <p class="graf graf--p">
   多个城市都有法律要求公共程序来讨论警察如何使用和购买监控技术。社区活动家也一直在努力反击面部识别和自动车牌阅读器等工具。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    当警察和亚马逊说服私人居民购买这些相机时，它基本上绕过了整个公民监督过程，同时省了不少钱
   </strong>
   。Ring 摄像机的价格在99美元到500美元之间。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们在全市都没有安装监视摄像头，从本质上讲，Ring 具有在全市范围内创建监视技术的能力。我们呼吁公民参与、自己购买”，Cook 说。
  </p>
  <p class="graf graf--p">
   一些警察部门做的可不仅仅是呼吁。
  </p>
  <p class="graf graf--p">
   印第安纳州、新泽西州、加利福尼亚州和其他州的警察已经为Ring相机提供折扣，有时高达125美元。在某些情况下，这些折扣来自纳税人的钱。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “简单说就是，
   <strong class="markup--strong markup--p-strong">
    公众在财政上补贴了在他们的社区中侵犯他们自己隐私的东西
   </strong>
   ，”Tajsar 说。
  </p>
  <p class="graf graf--p">
   今年4月，印第安纳州哈蒙德市宣布有37,500美元的资金来补贴 Ring 设备 —— 其中一半来自 Ring。哈蒙德警方公共信息官 Steve Kellogg 说，其他18,750美元来自该市纳税人。
  </p>
  <p class="graf graf--p">
   这个城市有500台摄像机，大约一周后，它们全部售罄。Kellogg 表示，市政府推出了更多的优惠计划，在该市推出了600多台 Ring 摄像机。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “街上会有更多的摄像头，这真是一个明智的选择。”
  </p>
  <p class="graf graf--p">
   其他城市将提供赠品，无论是抽奖赠品还是作为提供犯罪线索的奖励，如南加州城市
   <a class="markup--anchor markup--p-anchor" data-href="https://www.facebook.com/CityofElMonte/posts/2340642139328181" href="https://www.facebook.com/CityofElMonte/posts/2340642139328181" rel="noopener noreferrer" target="_blank">
    El Monte 所做的那样
   </a>
   。
  </p>
  <figure aria-describedby="caption-attachment-5571" class="wp-caption aligncenter" id="attachment_5571" style="width: 1400px">
   <img alt="" class="size-full wp-image-5571 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/按门铃.png?resize=1100%2C735&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/按门铃.png?w=1400&amp;ssl=1 1400w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/按门铃.png?resize=300%2C200&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/按门铃.png?resize=768%2C513&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/按门铃.png?resize=1024%2C684&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/按门铃.png?resize=1100%2C735&amp;ssl=1 1100w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/按门铃.png?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="735" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/按门铃.png?resize=1100%2C735&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-5571">
    Police across the country have been giving away Ring doorbells or subsidizing them at a heavy discount.
   </figcaption>
  </figure>
  <p class="graf graf--p">
   虽然理论上警察要想获得镜头需要被许可，但休斯顿的赠品条件可以确保执法部门能够获得所需的任何视频。休斯敦警方在去年三月的
   <a class="markup--anchor markup--p-anchor" data-href="https://nextdoor.com/agency-post/tx/houston/houston-police/ring-doorbell-cameras-106854768/" href="https://nextdoor.com/agency-post/tx/houston/houston-police/ring-doorbell-cameras-106854768/" rel="noopener noreferrer" target="_blank">
    赠品帖
   </a>
   中写道，获奖者会同意让休斯顿警方在要求时使用相机。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这种模式是最令人不安的，这基本上是把人们的家用作了执法的监视前哨，”Tajsar 说。
  </p>
  <p class="graf graf--p">
   休斯顿警方没有回应评论请求。 Ring 表示，它不支持这种模式，并且它正在向警察合作伙伴商谈，以确保这不是Ring赠品的要求。
  </p>
  <p class="graf graf--p">
   在多个城市里，每20个注册该应用的人就会捐赠一台Ring相机。这就是为什么一些警察部门一直在推动更多居民去注册。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    警方推广监视相机也有助于亚马逊的利润。即使Ring正在免费提供相机或提供补贴，它也很快就能获得投资回报
   </strong>
   。
  </p>
  <p class="graf graf--p">
   您不被强迫成为Ring会员，但这是您可以存储从摄像机录制的镜头的唯一方法。最便宜的计划起价为每月3美元。
  </p>
  <p class="graf graf--p">
   即使亚马逊向哈蒙德的补贴计划捐赠了价值18,750美元，它也可以在不到一年的时间里通过600个新的会员来弥补所有这一切。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “随着警务变得更加以技术为导向，我们将这一新的警察问题视为商业企业的利益，”Piza 说。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    结合技术
   </strong>
  </p>
  <p class="graf graf--p">
   Kerney 说，尽管布卢姆菲尔德充满了Ring摄像机，但人们可能没有为警察提供大量监视视频。
  </p>
  <p class="graf graf--p">
   侦探局指挥官说，他在过去两周内发出了大约10项请求，涉及盗窃等案件，但大多数都没有得到答复。
  </p>
  <p class="graf graf--p">
   当 Neighbors 应用程序中的人没有响应时，警察就会走上街头，开始敲门，要求亲自获取拍摄镜头。
  </p>
  <p class="graf graf--p">
   他说，当官员到他们家门口询问Ring监视视频时，人们会更加合作。民间倡导者认为，
   <strong class="markup--strong markup--p-strong">
    人们并没有真正的选择
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “当你看到一个警察在你旁边开车的时候，你会改变你开车的方式。如果警察出现在你家门口并要求你做什么怎么办？”Tajsar 说， “即使你是最明确的公民自由主义者，你也会感到正在被迫将这些镜头交出来。”
  </p>
  <p class="graf graf--p">
   而 Ring 并不仅限于亚马逊自己的技术，更多精通技术的警察部门也发现了这一点。
  </p>
  <p class="graf graf--p">
   去年12月，当 Ring 正考虑对门铃相机加入面部识别技术时，它面临强烈的反对，警方能够使用他们自己的算法利用居民提供的镜头。
  </p>
  <p class="graf graf--p">
   根据Ring摄像机的设置方式，它可以捕捉街道上的任何运动，比如经过的汽车。 Kellogg 指出，Hammond 有自动车牌阅读器，可以使用Ring相机的镜头来追踪车辆。
  </p>
  <p class="graf graf--p">
   他补充说，警方可以输入在Ring镜头中捕获的汽车的详细信息，在车牌阅读器系统中搜索，并找出汽车的所有者和地址。
  </p>
  <p class="graf graf--p">
   居民可能不会意识到这一点。 Ring视频的请求通常出现在 Neighbors 应用程序中，只是询问与报告事件相关的证据，
   <strong class="markup--strong markup--p-strong">
    没有关于剪辑将用于什么的详细信息。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果公众要与警方分享这段视频，他们至少需要知道它将被用于干什么，”Maass 说。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnet.com/features/amazons-helping-police-build-a-surveillance-network-with-ring-doorbells/" href="https://www.cnet.com/features/amazons-helping-police-build-a-surveillance-network-with-ring-doorbells/" rel="noopener noreferrer" target="_blank">
    Amazon’s helping police build a surveillance network with Ring doorbells
   </a>
   <br/>
   Its popular Ring smart doorbells mean more cameras on more doorsteps, where surveillance footage used to be rare.
  </p>
