   提前发布这篇文章，因为刚得到消息：中国将在两周后发放5G手机号：

  <p>
   <img alt="" class="aligncenter size-full wp-image-8575 jetpack-lazy-image" data-lazy-sizes="(max-width: 1080px) 100vw, 1080px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5G.jpeg?resize=1080%2C1350&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5G.jpeg?w=1080&amp;ssl=1 1080w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5G.jpeg?resize=240%2C300&amp;ssl=1 240w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5G.jpeg?resize=768%2C960&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5G.jpeg?resize=819%2C1024&amp;ssl=1 819w" data-recalc-dims="1" height="1350" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5G.jpeg?resize=1080%2C1350&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1080"/>

  <p class="graf graf--p">
   2018年1月，国家安全委员会战略规划高级主管 Robert Spalding 在白宫街对面的艾森豪威尔行政大楼办公室里看到了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.axios.com/trump-team-debates-nationalizing-5g-network-f1e92a49-60f2-4e3e-acd4-f3eb03d910ff.html" href="https://www.axios.com/trump-team-debates-nationalizing-5g-network-f1e92a49-60f2-4e3e-acd4-f3eb03d910ff.html" rel="noopener noreferrer" target="_blank">
    Axios
   </a>
   的突发新闻警报。
  </p>
  <p class="graf graf--p">
   这则快讯的标题是：“特朗普政府考虑将5G网络国有化”。
  </p>
  <p class="graf graf--p">
   这位空军准将 Spalding 已在军队服役了近三十年，曾经作为国防部长驻北京工作。在国安委，他正在研究如何确保5G可以防止网络攻击。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我并不是在政策角度上考虑这个问题的，”他说，“这个问题是关于物理学的，关于什么是可能的。”
  </p>
  <p class="graf graf--p">
   令 Spalding 感到意外的是，Axios 的报道是基于他在一年中大部分时间里一直在研究的一份报告的泄露。
  </p>
  <p class="graf graf--p">
   两个词可以解释我们当前的无线网络与5G之间的区别：速度和延迟。 5G预计会快一百倍，如果你相信炒作的话。（可以在不到4秒的时间内下载两小时的电影）该速度将减少并可能消除延迟。
  </p>
  <p class="graf graf--p">
   再次，如果你相信炒作的话，这将导致一个全新的物联网，从烤面包机到狗项圈、从透析泵到跑鞋的所有东西，都将被连接起来。
  </p>
  <p class="graf graf--p">
   远程机器人手术将成为常规，军方将开发高端超音速武器，自动驾驶汽车将沿着智能高速公路快速行驶……
  </p>
  <p class="graf graf--p">
   延伸《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%ba%e5%b7%a5%e6%99%ba%e8%83%bd%e6%96%b0%e5%86%b7%e6%88%98%e2%80%8a-%e2%80%8a-ai%e5%92%8c5g%e5%a6%82%e4%bd%95%e5%8f%98%e6%88%90%e4%b8%80%e5%9c%ba%e5%99%a9%e6%a2%a6/" href="https://www.iyouport.org/%e4%ba%ba%e5%b7%a5%e6%99%ba%e8%83%bd%e6%96%b0%e5%86%b7%e6%88%98%e2%80%8a-%e2%80%8a-ai%e5%92%8c5g%e5%a6%82%e4%bd%95%e5%8f%98%e6%88%90%e4%b8%80%e5%9c%ba%e5%99%a9%e6%a2%a6/" rel="noopener noreferrer" target="_blank">
    人工智能新冷战 — — AI和5G如何变成一场噩梦
   </a>
   》
  </p>
  <p class="graf graf--p">
   赌注非常高。据估计，到2035年，5G将为全球经济注入12万亿美元，仅在美国就能增加2,200万个新工作岗位。我们被告知，这个5G世界将迎来
   <strong class="markup--strong markup--p-strong">
    第四次工业革命。
   </strong>
  </p>
  <p class="graf graf--p">
   一个完全连通的世界也将特别容易遭受网络攻击。
  </p>
  <p class="graf graf--p">
   甚至在引入5G网络之前，骇客就已经攻破了市政大坝系统的控制中心，阻止了一辆联网汽车（当时它沿着州际公路行进），并破坏了家用电器。
  </p>
  <p class="graf graf--p">
   勒索软件、恶意软件、身份盗窃和数据泄露，已经变得如此普遍，以至于更多的美国人
   <a class="markup--anchor markup--p-anchor" data-href="https://thebestvpn.com/cyber-security-statistics-2019/" href="https://thebestvpn.com/cyber-security-statistics-2019/" rel="noopener noreferrer" target="_blank">
    害怕成为网络犯罪而不是暴力犯罪的受害者
   </a>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    向在线世界添加更多设备注定会创造更多的遭受破坏的机会
   </strong>
   。 “5G可不仅适用于冰箱，” Spalding 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “它几乎是一切 —— 可以真正杀死人的。这是一种我们以前从未体验过的完全不同的威胁。”
  </p>
  <p class="graf graf--p">
   Spalding 的解决方案是从头开始构建5G网络，将网络防御融入其设计中。
  </p>
  <p class="graf graf--p">
   因为这将是一项艰巨的任务，他最初建议联邦政府支付一笔费用，并且基本上将任务出租给电信公司。
  </p>
  <p class="graf graf--p">
   但他现在已经放弃了这个想法。他说，后来的草案提出了主要的电信公司 — Verizon、AT＆T、Sprint 和 T-Mobile 这几大巨头组成一个独立的公司，共同构建该网络并共享它。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “它本来是一个全国性的网络，而不是一个国有化的网络”，Spalding 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他们可以建立这个网络，然后向他们的零售客户出售带宽。政府永远不会拥有这个网络。它始终是关于，我们如何让行业真正保护系统？”
  </p>
  <p class="graf graf--p">
   2017年，Verizon 宣布将在十一个城市推出5G，包括达拉斯、安娜堡、迈阿密和丹佛。AT&amp;T也正在十几个城市测试其服务。T-Mobile专注于斯波坎。
  </p>
  <p class="graf graf--p">
   在大多数情况下，他们在现有基础设施之上构建新服务，并继承原有的漏洞。克莱姆森大学教授 Thomas Hazlett 告诉我，“这只是过渡部分。会进行各种各样的实验，在市场上进行试验，并且进行各种部署，这些部署可以与旧系统真正区分开来。”
  </p>
  <p class="graf graf--p">
   与此同时，运营商之间开始争夺地位。 Sprint 和 T-Mobile 提起了诉讼，声称AT＆T的5GE服务（其中“E”代表“进化”）仅仅是4G的另一个名字。
  </p>
  <p class="graf graf--p">
   根据 Spalding 的说法，当运营商听说政府正在考虑将其行业的未来“国有化”时，他们很快就会动员起来共同反对这一提议。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “随后我跟人们谈过，他们说他们从未见过这个行业可以如此迅速地团结起来，” Spalding 说。
  </p>
  <p class="graf graf--p">
   他们在政府、国会和官僚机构中得到了支持，他们拥有如此庞大的政治游说队伍，而且是全面而迅速的采取了行动。”
  </p>
  <p class="graf graf--p">
   Axios 的报道在周日发布。第二天，联邦通信委员会主席
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnn.com/2018/01/28/politics/trump-nationalize-5g/index.html" href="https://www.cnn.com/2018/01/28/politics/trump-nationalize-5g/index.html" rel="noopener noreferrer" target="_blank">
    Ajit Pai 彻底拒绝
   </a>
   任何将互联网国有化的想法，称 “市场，而不是政府，最有利于推动创新和投资。”
  </p>
  <p class="graf graf--p">
   到了周三，Spalding 失业了。 “对我长期以来的贡献没有任何感谢，” Spalding 告诉我。 “只是让我滚出去，别再回来了”。
  </p>
  <p class="graf graf--p">
   华为是中国消费电子和电信设备制造商，目前是5G技术的全球领导者。其首席执行官任正非被华盛顿（尤其是特朗普）指责为中国情报的通道。
  </p>
  <p class="graf graf--p">
   在“华盛顿邮报”的一篇专栏文章中，阿肯色州的共和党参议员 Tom Cotton 和德克萨斯州的 John Cornyn 表示，该公司是由中国政府补贴资助的公司，作为一种特洛伊木马可以“让中国有效”地控制数字制高点。
  </p>
  <p class="graf graf--p">
   他们讲述了在亚的斯亚贝巴总部安装华为服务器的非洲联盟遭遇监视的故事，他们发现这些服务器每晚都在向中国发送敏感数据。
  </p>
  <p class="graf graf--p">
   参议员指出，虽然华为强烈否认它是中国政府的代理人，但该公司受制于中国法律，这些法律要求公司与国家情报机构合作。
  </p>
  <p class="graf graf--p">
   伦敦时报报道了CIA声称有证据表明，华为已从解放军以及中国情报部门的分支机构取款。
  </p>
  <ul>
   <li class="graf graf--p">
    延伸：《

     华为之战的虚伪：技术的巴尔干化
    </a>
    》

   <li>
    《
    <a href="https://www.iyouport.org/5g%e6%8a%80%e6%9c%af%e7%9a%84%e9%9a%90%e7%a7%98%e5%86%9b%e4%ba%8b%e7%94%a8%e9%80%94/" rel="noopener noreferrer" target="_blank">
     5G技术的隐秘军事用途
    </a>
    》
   </li>
   <li>
    《
    <a href="https://www.iyouport.org/%e6%ad%a6%e5%99%a8%e5%8c%96%e5%9f%8e%e5%b8%82/" rel="noopener noreferrer" target="_blank">
     武器化城市
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   不过，欧盟准备拒绝美国的恳求，葡萄牙和德国等国家也表示愿意使用华为设备（原因详见上面的延伸阅读，那是来自欧洲的分析）。
  </p>
  <p class="graf graf--p">
   加拿大已经依靠华为进行过至少一次5G试验。甚至美国自己的 AT&amp;T 也
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wsj.com/articles/the-u-s-wants-to-ban-huawei-at-t-mexico-relies-on-it-11555407001" href="https://www.wsj.com/articles/the-u-s-wants-to-ban-huawei-at-t-mexico-relies-on-it-11555407001" rel="noopener noreferrer" target="_blank">
    继续在墨西哥使用华为设备
   </a>
   。
  </p>
  <p class="graf graf--p">
   华为设备比其西方竞争对手的设备更便宜，创新委员会（dib）的研究人员说。在很多情况下该委员会为国防部长提供新技术方面的建议。
  </p>
  <p class="graf graf--p">
   截至今年年初，华为占据了全球电信设备市场近30％的份额，其收入比前一年高出百分之三十九。
  </p>
  <p class="graf graf--p">
   据该公司称，其持续增长“将使中国能够推广其首选的5G网络标准和规范，并将在未来塑造全球5G产品市场。”
  </p>
  <p class="graf graf--p">
   有很好的理由可以让一家公司看起来像工业网络间谍。
   <strong class="markup--strong markup--p-strong">
    但禁止华为硬件无法保护这些网络。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    即使在没有华为设备的情况下，该系统仍然可能依赖于在中国开发的软件，并且软件可以被恶意行为者远程重新编程
   </strong>
   。
  </p>
  <p class="graf graf--p">
   连接到5G网络的每台设备都可能仍然容易被骇客攻击。
  </p>
  <p class="graf graf--p">
   目前在R街研究所负责国家安全计划的总法律顾问、前FBI官员 James Baker 说，“有人担心，从网络角度来看，连接到5G网络的设备不会非常安全。这给系统带来了巨大的漏洞，因为这些设备可以随时变成肉鸡，
   <strong class="markup--strong markup--p-strong">
    你可以拥有一个用来攻击不同部分的庞大僵尸网络”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   今年1月，奥巴马时代联邦通信委员会主席 Tom Wheeler 在纽约时报发表了一篇题为 “如果5G如此重要为什么不安全？”
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2019/01/21/opinion/5g-cybersecurity-china.html" href="https://www.nytimes.com/2019/01/21/opinion/5g-cybersecurity-china.html" rel="noopener noreferrer" target="_blank">
    的文章
   </a>
   。特朗普政府已经把 Wheeler 辞退了。
  </p>
  <p class="graf graf--p">
   最值得注意的是，在最近的国际标准谈判中，
   <strong class="markup--strong markup--p-strong">
    美国取消了5G的技术规范中包括网络防御的要求
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这是有史以来第一次，”Wheeler 写道，“在设计每一种新的网络标准时，网络安全都需要作为一种预见 —— 直到特朗普的FCC废除它”。
  </p>
  <p class="graf graf--p">
   该机构还拒绝了建立和运营美国数字网络的公司负责监督其安全性的观点。这可能是预期内的，但目前的FCC确实不认为网络安全是其领域的一部分。
  </p>
  <p class="graf graf--p">
   特朗普政府热衷于赢得它所谓的“5G竞赛”，可能更有兴趣试图制止华为 — 以及中国的技术发展。新FCC主席 Ajit Pai 最近宣布该委员会将阻止另一家中国公司中国移动在美国的运营，并再次引用安全问题。
  </p>
  <p class="graf graf--p">
   经济学家兼技术政策研究所主席 Scott Wallsten 表示，“如果我们没有与中国有其他贸易问题，那么接受[政府]的安全声明就更容易了。
   <strong class="markup--strong markup--p-strong">
    但当它与所有其他贸易问题混淆时，就会让人更加怀疑
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   特朗普签署了一份关于“为美国的未来制定可持续战略”的备忘录，几周后，FCC拍卖了新的无线电频谱（还有另一场拍卖计划在今年晚些时候举行）。
  </p>
  <p class="graf graf--p">
   开辟新频谱对于实现5G承诺的超高速度至关重要。
  </p>
  <p class="graf graf--p">
   大多数美国运营商正在计划将其服务迁移到该频谱的较高部分，可以允许巨大的数据流通过（较低的部分速度没那么快，但可靠）。
  </p>
  <p class="graf graf--p">
   直到最近，这些被称为毫米波的高频频段还无法用于互联网传输，但天线技术的进步将使其成为可能，至少在理论上如此。
  </p>
  <p class="graf graf--p">
   在实践中，毫米波是很挑剔的：它们只能短距离传输 —— 大约一千英尺 —— 并且会被墙壁、树叶、人体和下雨所阻碍。
  </p>
  <p class="graf graf--p">
   为了适应这些限制，至少必须在建筑物内和每个城市街区安装大量的5G蜂窝式中继。例如，
   <strong class="markup--strong markup--p-strong">
    安装在1300万个电线杆上的中继将为美国一半以上人口提供5G速度，安装成本约为4,000亿美元
   </strong>
   。
  </p>
  <p class="graf graf--p">
   农村地区有太多树木和太少的人口，可能不适用。
  </p>
  <p class="graf graf--p">
   部署数以百万计的无线中继而且彼此如此靠近，因此已经引发了人类身体健康方面的担忧。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    两年前，来自36个国家的180名科学家和医生呼吁欧盟
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.jrseco.com/european-union-5g-appeal-scientists-warn-of-potential-serious-health-effects-of-5g/" href="https://www.jrseco.com/european-union-5g-appeal-scientists-warn-of-potential-serious-health-effects-of-5g/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     暂停对5G的采用
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，直到研究得出切实结论关于低水平辐射增加所产生的影响。
   </strong>
  </p>
  <p class="graf graf--p">
   今年2月，来自康涅狄格州的民主党参议员
   <a class="markup--anchor markup--p-anchor" data-href="https://www.blumenthal.senate.gov/newsroom/press/release/at-senate-commerce-hearing-blumenthal-raises-concerns-on-5g-wireless-technologys-potential-health-risks" href="https://www.blumenthal.senate.gov/newsroom/press/release/at-senate-commerce-hearing-blumenthal-raises-concerns-on-5g-wireless-technologys-potential-health-risks" rel="noopener noreferrer" target="_blank">
    Richard Blumenthal
   </a>
   参加了FCC和FDA在
   <strong class="markup--strong markup--p-strong">
    没有评估
   </strong>
   其健康风险的情况下推进5G的任务。
   <strong class="markup--strong markup--p-strong">
    “我们对此视而不见”
   </strong>
   ，他总结说。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    基于数百万个中继、天线和传感器的系统也提供了以前无法想象的监控潜力。
   </strong>
  </p>
  <p class="graf graf--p">
   电信公司已向营销人员出售位置数据，执法部门也使用类似数据跟踪抗议者。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5G将前所未有地准确记录某人来自哪里、他们去了哪里、以及他们在做什么。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    与面部识别和人工智能相结合，5G的数据流和定位功能将使匿名彻底成为历史文物。
   </strong>
  </p>
  <p class="graf graf--p">
   在中国，已经安装了35万个5G中继，再加上一个广泛的监控摄像头网络，每个监视器都配备了面部识别技术，使得当局能够跟踪任何人。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这种做法使中国成为了应用下一代技术监视其所有人口的先驱，开启了自动化种族主义的新时代。”
  </p>
  <p class="graf graf--p">
   美国至少目前还没到这个地步。但是随着5G开始推出，捕获和利用来自个人、企业和政府的新数据流的争议将变得更加激烈。
  </p>
  <p class="graf graf--p">
   建立对系统的保障似乎是一个明显和必要的目标。
  </p>
  <p class="graf graf--p">
   Spalding 现在是哈德森研究所的高级研究员，并就企业和其他机构对中国提出的网络安全威胁提供建议。他警告说，危险并不仅限于单一的民族国家。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <strong class="markup--strong markup--p-strong">
    这允许极权政权 — 或任何政府 — 完全了解你所做的一切
   </strong>
   ，”他说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “因为当权者永远想要规范你的思考方式、你的行为方式、你做的一切事。问题在于，大多数人仍没有意识到这个世界正在变得非常危险”。⚪️
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.newyorker.com/news/annals-of-communications/the-terrifying-potential-of-the-5g-network" href="https://www.newyorker.com/news/annals-of-communications/the-terrifying-potential-of-the-5g-network" rel="noopener noreferrer" target="_blank">
    The Terrifying Potential of the 5G Network
   </a>
  </p>
