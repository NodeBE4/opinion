  <p class="graf graf--p">
   谈到中国时，人们往往会提到难以联合。这是一个明显的弱点。
  </p>
  <p class="graf graf--p">
   但有些时候，活动家会将联合视为几百万人的规模；这的确是很理想的状态，但也有可能是一时下无法实现的。
  </p>
  <p class="graf graf--p">
   没有哪种行动会彻底无效，除了错误的策略之外。
  </p>
  <p class="graf graf--p">
   小规模也一样可能成功。Baraza 就是这样一种机制。
  </p>
  <p class="graf graf--p">
   Baraza 是斯瓦希里语，用于社区聚会，它很有传统，早在欧洲殖民统治之前就已成为东非乡村生活的一部分。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    类似的聚会一直在世界各地以其他各种形式发生，特别是在全球南方国家。
   </strong>
  </p>
  <p class="graf graf--p">
   以社区集会的形式讨论重要事项、分享信息，并让政客负责。
  </p>
  <p class="graf graf--p">
   特别是在东非农村地区，Baraza 经常吸引数百甚至数千名当地居民。
  </p>
  <p class="graf graf--p">
   运动和组织能够利用这些空间来提出关键的问题，例如，教育人们减轻气候变化的影响、提请注意家庭暴力的增加、或揭发当地领导人的腐败内幕。
  </p>
  <p class="graf graf--p">
   不称职的公务员、贪婪的商人和撒谎的政客通常拒绝出现在 Baraza，因为他们害怕被羞辱；没皮没脸抛头露面的政客会被当众要求为他们的行为负责。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Baraza 并不总是善与恶的战斗。
   </strong>
  </p>
  <p class="graf graf--p">
   在这里，通常会摆明影响社区的问题并进行讨论，面向与会者收集解决方案。
  </p>
  <p class="graf graf--p">
   💡达成一致后，责任将指定给特定的团队实施。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这就是 barazas 的性质，它是一个非常横向的、直接民主的社区组织机制
   </strong>
   。
  </p>
  <p class="graf graf--p">
   对于缺乏民主经验的国家地区来说，barazas 作为演练是一个好办法。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    请注意，这与社交媒体上对特定社会问题的讨论截然不同
   </strong>
   ，解释详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%9c%a8%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e4%b8%8a%e6%8a%97%e8%ae%ae%e6%9c%89%e4%bb%80%e4%b9%88%e9%97%ae%e9%a2%98%ef%bc%9f/" href="https://www.iyouport.org/%e5%9c%a8%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e4%b8%8a%e6%8a%97%e8%ae%ae%e6%9c%89%e4%bb%80%e4%b9%88%e9%97%ae%e9%a2%98%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    在社交媒体上”抗议”有什么问题？
   </a>
   》。
  </p>
  <p class="graf graf--p">
   在乌干达的阿比姆，barazas 被用来评估公务员使用地方税收的合理性。这导致基础设施得到了改善，包括道路和桥梁。
  </p>
  <p class="graf graf--p">
   1986年至1992年期间，科威特的宪法运动使用了diwanyat 组织 ——
   <strong class="markup--strong markup--p-strong">
    以家庭为单位的当地组织，因为当时街头抗议活动是非法的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   而由于文化规范，警察永远不会进入 diwanya。
  </p>
  <p class="graf graf--p">
   Diwanyat 实现了群众动员，为最终恢复宪法的运动提供了动力。
  </p>
  <p class="graf graf--p">
   💡世界各地的人们都在以各种名义组织类似 Diwanyat 的聚会：新英格兰的 “town halls”，马德里的 “indignados”，阿拉伯世界的大部分地区，都是以 “diwaniyat”的形式组织的。
  </p>
  <p class="graf graf--p">
   barazas 是人类存在方式的基础，在努力建设一个更好的社区的过程中，需要首先将人们聚集在一起。
  </p>
  <p class="graf graf--p">
   有了组织才有运动。
  </p>
  <p class="graf graf--p">
   它没有固定的模版，
   <strong class="markup--strong markup--p-strong">
    需要依据您的当地文化作出最有效的策划
   </strong>
   。更多详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/1%e3%80%81%e4%ba%86%e8%a7%a3%e4%bd%a0%e7%9a%84%e5%9c%b0%e5%bd%a2%e6%98%af%e7%ad%96%e5%88%92%e8%a1%8c%e5%8a%a8%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" href="https://www.iyouport.org/1%e3%80%81%e4%ba%86%e8%a7%a3%e4%bd%a0%e7%9a%84%e5%9c%b0%e5%bd%a2%e6%98%af%e7%ad%96%e5%88%92%e8%a1%8c%e5%8a%a8%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" rel="noopener noreferrer" target="_blank">
    了解你的地形是策划行动的第一步
   </a>
   》。
  </p>
  <p class="graf graf--p">
   ⚠️基于你的文化的 baraza 应该是什么样的？你所在的社区中自然存在的直接民主传统是什么？你怎么才能做到利用它来改善现状？
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-6107 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-15.png?resize=1100%2C691&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-15.png?w=1166&amp;ssl=1 1166w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-15.png?resize=300%2C188&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-15.png?resize=768%2C482&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-15.png?resize=1024%2C643&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-15.png?resize=1100%2C691&amp;ssl=1 1100w" data-recalc-dims="1" height="691" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-15.png?resize=1100%2C691&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如何使用 ——
   </strong>
  </p>
  <p class="graf graf--p">
   Barazas 的组织方式不同，并且根据具体情况以各种名称命名。
  </p>
  <p class="graf graf--p">
   但组织它们的
   <strong class="markup--strong markup--p-strong">
    几个基本步骤
   </strong>
   通常都适用：
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    准备。协调团队确定 baraza 的时间、地点和议程。
   </li>
   <li class="graf graf--li">
    宣布。公众将收到通知。在集体主义社会的农村地区，公告被钉在树上、告示板或中央地区；加上口口相传，通常足以确保足够多的参与。全球北方国家的常见做法包括创建 Facebook 活动社区，以及向学龄儿童的家长发送公告。
   </li>
   <li class="graf graf--li">
    召开。通常明智的做法是假设已经到场的人都是合作者。
    <strong class="markup--strong markup--li-strong">
     促进 baraza 不是制造紧张气氛，而是热情
    </strong>
    ；它的确依赖与会者的讨论和输入，但是作为组织者，对你的议程进行总体概述是有用的。
   </li>
   <li class="graf graf--li">
    继续。不要因为探讨出结论就停止。如果相关，请告知参与者下一个 baraza 的时间和地点，并鼓励他们邀请其他人前往。
   </li>
  </ol>
  <p class="graf graf--p">
   这里有一个人权 baraza 的方案：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.undp.org/content/dam/uganda/img/Research%20and%20Publications/BARAZA%20BOOK.pdf" href="https://www.undp.org/content/dam/uganda/img/Research%20and%20Publications/BARAZA%20BOOK.pdf" rel="noopener noreferrer" target="_blank">
    关于开展社区公众会议的手册
   </a>
   》，活动家可以参考。
  </p>
  <p class="graf graf--p">
   ⚠️潜在风险：
  </p>
  <p class="graf graf--p">
   Barazas 可能被社区的反对者篡夺，令活动家失去对自己组织机制的控制权。
  </p>
  <p class="graf graf--p">
   很多国家政府擅长于此，包括中国，鉴于中国政府主导的社会运动的有效性，这不容忽视。
  </p>
  <p class="graf graf--p">
   在肯尼亚，政府在最低行政单位召集 Barazas，向居民提供即将到来的疫苗接种计划、新农业实践和其他政府计划的最新信息。
  </p>
  <p class="graf graf--p">
