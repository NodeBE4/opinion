  <p class="graf graf--p">
   先说句题外话。自去年以来很多人正在为深度造假（deepfake）技术的飞跃而发愁，看起来这种技术有可能明显影响到开源情报事业。（在这里看到关于深度造假的分析报告《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bd%93%e6%81%b6%e6%84%8f%e8%80%85%e6%8e%8c%e6%8f%a1%e6%8a%80%e6%9c%af%ef%bc%9a%e4%bf%a1%e6%81%af%e6%88%98%e7%9a%84%e6%9c%aa%e6%9d%a5%e7%a9%b6%e7%ab%9f%e6%9c%89%e5%a4%9a%e5%8f%af%e6%80%95%ef%bc%9f/" href="https://www.iyouport.org/%e5%bd%93%e6%81%b6%e6%84%8f%e8%80%85%e6%8e%8c%e6%8f%a1%e6%8a%80%e6%9c%af%ef%bc%9a%e4%bf%a1%e6%81%af%e6%88%98%e7%9a%84%e6%9c%aa%e6%9d%a5%e7%a9%b6%e7%ab%9f%e6%9c%89%e5%a4%9a%e5%8f%af%e6%80%95%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    当恶意者掌握技术：信息战的未来究竟有多可怕？
   </a>
   》）
  </p>
  <p class="graf graf--p">
   也许在较小一部分程度上的确如此，但在很多情况下，至少目前为止不会影响开源情报调查。这是因为造假内容不会存在于一个完美连贯的逻辑链条上 —— 造假者可以修改逻辑链上的某些节点，但无法修改全部，并且，更有可能的是，造假者不能完全了解调查人员会采取什么样的取证思路。
  </p>
  <p class="graf graf--p">
   正如我们曾
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%b7%b1%e5%ba%a6%e9%80%a0%e5%81%87%e7%9c%9f%e7%9a%84%e6%98%af%e4%b8%80%e5%9c%ba%e7%9c%9f%e7%9b%b8%e5%b7%b2%e6%ad%bb%e7%9a%84%e5%8d%b1%e6%9c%ba%e5%90%97%ef%bc%9f%e6%88%91%e4%bb%ac%e5%a6%82%e4%bd%95/" href="https://www.iyouport.org/%e6%b7%b1%e5%ba%a6%e9%80%a0%e5%81%87%e7%9c%9f%e7%9a%84%e6%98%af%e4%b8%80%e5%9c%ba%e7%9c%9f%e7%9b%b8%e5%b7%b2%e6%ad%bb%e7%9a%84%e5%8d%b1%e6%9c%ba%e5%90%97%ef%bc%9f%e6%88%91%e4%bb%ac%e5%a6%82%e4%bd%95/" rel="noopener noreferrer" target="_blank">
    经论述过的
   </a>
   ，深度造假真的不是什么新鲜事，人类历史上充满了欺骗性的做法，由于照片编辑技术，过去几十年来对摄影的信任已经受到明显侵蚀。但是，真相并没有被消灭 —— 因为社会已经重新适应了这一局面。
  </p>
  <p class="graf graf--p">
   照片造假并非造成了真相的死亡，而是对新技术的认识鼓励人们采取新的指标，以做出关于所呈现的图像是否真实的明智决定。
  </p>
  <p class="graf graf--p">
   deepfake 技术只是提出了一个新的挑战，甚至有希望促进更多真相捍卫者的诞生。只要您想要真相，并不难获得。
  </p>
  <p class="graf graf--p">
   本网“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">

  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    照片的拍摄时间是非常重要的真伪判断元素
   </strong>
   ，当消息是突发新闻时，它还很可能具有强大的政治意义。
  </p>
  <p class="graf graf--p">
   检验照片拍摄时间对于开源情报来说一直是一个挑战；我们曾经演示过一些做法，比如《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ae%a9%e9%98%b4%e5%bd%b1%e8%af%b4%e8%af%9d%ef%bc%9a%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e5%b0%8f%e6%b8%b8%e6%88%8f/" href="https://www.iyouport.org/%e8%ae%a9%e9%98%b4%e5%bd%b1%e8%af%b4%e8%af%9d%ef%bc%9a%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e5%b0%8f%e6%b8%b8%e6%88%8f/" rel="noopener noreferrer" target="_blank">
    让阴影说话：开源调查小游戏
   </a>
   》等等。本文是另一个演示。
  </p>
  <p class="graf graf--p">
   工具并不复杂，💡希望您能从这些演示中学到真伪检验的思考方式，
   <strong class="markup--strong markup--p-strong">
    这是公民记者的基本功之一
   </strong>
   。
  </p>
  <p class="graf graf--p">
   2014年10月11日，一个名为 “Ruslan Boshirov” 的人在 Facebook 张贴了布拉格的一张照片，显示了交通繁忙的旧城广场和天文钟。
  </p>
  <p class="graf graf--p">
   该账户就是毒杀前俄罗斯间谍 Skripal 的嫌疑人 —— 俄罗斯军事情报局 GRU 的一名装饰精良的上校 Anatoliy Chepiga 的社交媒体身份。
  </p>
  <p class="graf graf--p">
   这张照片的上传日期恰逢捷克出版物
   <a class="markup--anchor markup--p-anchor" data-href="https://www.irozhlas.cz/zpravy-domov/double-agent-sergey-skripal-followed-prague-russians_1810101220_miz" href="https://www.irozhlas.cz/zpravy-domov/double-agent-sergey-skripal-followed-prague-russians_1810101220_miz" rel="noopener noreferrer" target="_blank">
    Radiožurnal 的报道
   </a>
   ，“Boshirov”（Chepiga）于2014年10月11日访问了布拉格。
  </p>
  <p class="graf graf--p">
   每日邮报
   <a class="markup--anchor markup--p-anchor" data-href="https://www.dailymail.co.uk/news/article-6135859/Novichok-suspect-ONE-friend-Facebook-Ukrainian-woman-met-five-years-ago-Prague.html" href="https://www.dailymail.co.uk/news/article-6135859/Novichok-suspect-ONE-friend-Facebook-Ukrainian-woman-met-five-years-ago-Prague.html" rel="noopener noreferrer" target="_blank">
    报道
   </a>
   ，Boshirov / Chepiga 创建了这个 Facebook 简介，并在布拉格咖啡馆与一位乌克兰女性见面后，将她作为“朋友”添加到了 Facebook。
  </p>
  <p class="graf graf--p">
   这位女士接受了“每日邮报”的采访，表示她记得“在2013年或2014年”与 Boshirov / Chepiga 会面，并且他创建了这个
   <strong class="markup--strong markup--p-strong">
    专门与她联系的
   </strong>
   Facebook 页面。
  </p>
  <p class="graf graf--p">
   她是他唯一的 Facebook “朋友”。
  </p>
  <p class="graf graf--p">
   但是，这个 Facebook 档案真的属于 Chepiga / Boshirov 吗？还无法确定；每日邮报和捷克出版物的报道都没有做到检验这点。但公民记者可以提供对拍摄照片时间的验证报告以做出判断。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6816 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-9.jpg?resize=1100%2C1201&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-9.jpg?w=1405&amp;ssl=1 1405w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-9.jpg?resize=275%2C300&amp;ssl=1 275w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-9.jpg?resize=768%2C839&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-9.jpg?resize=938%2C1024&amp;ssl=1 938w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-9.jpg?resize=1100%2C1201&amp;ssl=1 1100w" data-recalc-dims="1" height="1201" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-9.jpg?resize=1100%2C1201&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   下面来演示这个操作。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    寻找时间细节
   </strong>
  </p>
  <p class="graf graf--p">
   将这张照片地理定位到布拉格市中心，这对于任何熟悉这个城市的人都不费吹灰之力（因为有地标），但确定具体时间则是一个更有趣的挑战。
  </p>
  <p class="graf graf--p">
   提出的问题是：该照片是否真的拍摄于2014年10月11日左右，该照片的上传日期与 Chepiga 访问布拉格的未经证实的报道是否相符？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    基于可观察到的细节，可以以不同的精度确定照片或视频的拍摄时间
   </strong>
   。
  </p>
  <p class="graf graf--p">
   照片中呈现的天气和季节性细节提供了一个认识 —— 可以假设这张照片是在一个寒冷的雨天拍摄的，基于场景中可见的外衣和雨伞 —— 但这还远不够精确。
  </p>
  <p class="graf graf--p">
   由于这个场景是整个欧洲被拍摄最多的著名景点之一，因此有大量的在线参考资料可以比较。
  </p>
  <p class="graf graf--p">
   但是，⚠️
   <strong class="markup--strong markup--p-strong">
    如果没有特定的时间细节，所有这些参考资料都是无用的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   通过最简单的搜索就可以知道，这个天文钟经常进行翻新，但在拍摄这张照片的时候显然没有正在进行的翻新工作（该时钟在2005、2017年和2018年进行了明显的翻新，但不是2014年 —— 这些资料可以很容易在网上查到，即便您不了解布拉格）。
  </p>
