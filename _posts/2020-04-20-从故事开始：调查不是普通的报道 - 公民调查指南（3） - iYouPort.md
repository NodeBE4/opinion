  <p class="graf graf--p">
   以往大多数的调查手册都把大部分笔墨集中于哪里能找到信息。他们认为一旦发现了要找的信息，就可以完成一个可行的故事。
  </p>
  <p class="graf graf--p">
   但是，巴黎著名学者和调查记者 Mark Lee Hunter 不同意这点，他指出，
   <strong class="markup--strong markup--p-strong">
    唯一的问题并不是寻找信息；相反，核心任务应该是讲故事
   </strong>
   。
  </p>
  <p class="graf graf--p">
   用故事来粘合调查过程中的每一个步骤：从设想到研究、写作、质量控制以及发表。他将这种方法称为
   <strong class="markup--strong markup--p-strong">
    “以假设为基础的调查”，以强调故事在经过验证之前只是一个假说
   </strong>
   。
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   💡通过验证或否定一个故事，公民记者可以更容易地明确自己所要寻找的信息，以及如何解释它。
  </p>
  <p class="graf graf--p">
   随着研究的深入，公民记者或独立调查团队可以组织材料进行写作，形成最后报道的某些特定部分。
  </p>
  <p class="graf graf--p">
   这可以使质量控制更为便利，并且可以洞察报道是否符合社会利益和道德标准。
  </p>
  <p class="graf graf--p">
   调查过程最终将形成一个故事，它可以被概括为几个“切中要害”的句子，从而使这个故事易于推广和被人记住。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     调查性报道不是普通的报道
    </strong>
   </span>
  </h3>
  <blockquote class="graf graf--blockquote">
   <p>
    给自己一个明确的理念，你在做什么，为什么而做。
   </p>
  </blockquote>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    调查性报道所要向公众揭露的事实，要么是被具有某种权力的人故意遮蔽的，要么是偶然的，因为混乱的事实和环境造成的认识模糊。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    所以它同时需要使用秘密和公开两种消息源和文献。
   </strong>
  </p>
  <p class="graf graf--p">
   常规的新闻报道在很大程度上几乎完全是依赖别人提供的素材（如警察、政府、公司等），即使不是完全
   <strong class="markup--strong markup--p-strong">
    被动
   </strong>
   的，也基本上只是一种反应。
  </p>
  <p class="graf graf--p">
   与此相反，💡调查性报道所依赖的素材是记者通过自己的
   <strong class="markup--strong markup--p-strong">
    主动
   </strong>
   调查搜集和产生的。
  </p>
  <p class="graf graf--p">
   常规的新闻报道目的在于如实地反映这个世界的客观形象，而调查性报道使用任何理性的观察者都会认可的客观真实的材料，为的是实现
   <strong class="markup--strong markup--p-strong">
    改造世界的主观目标
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这并不是说允许在正当的理由下说谎。而是了解事实真相，从而能够改变世界的一种职责。
  </p>
  <p class="graf graf--p">
   与一些专业人士常说的相反，调查性报道并不仅仅是把优秀的老式新闻做得更加出色。
  </p>
  <p class="graf graf--p">
   的确，这两种新闻形式都关注“谁”、“发生了什么”、“在哪里”、“什么时候”这些要素，但是，传统报道中的第五个要素“为什么”，
   <strong class="markup--strong markup--p-strong">
    在调查中变成了“如何”。
   </strong>
  </p>
  <p class="graf graf--p">
   其他几个元素不仅在量的方面，而且在质的方面都得到了进一步开发。
  </p>
  <p class="graf graf--p">
   “谁”不只是一个名字和一个头衔，而是一个有着个性特征和行事风格的人；“什么时候”不再是新闻的现在状态，而是一个历史连续性的叙述；“发生了什么”，并不只是一个事件，而是一个原因和后果的现象；“在哪里”不仅仅是一个地址，而是一种设置，在那里某些特定的事情变得或多或少得变得可能。
  </p>
  <p class="graf graf--p">
   这些元素和细节最大限度地增强了调查性报道的感观效应，从而增强了对情绪的影响。
  </p>

  <p class="graf graf--p">
   总之，尽管公民记者可能既从事日常报道，也从事调查性工作，但是，必须说，这两种角色有时涉及完全不同的技能、工作习惯、流程和目标。这些差异，详列于下表。
  </p>
  <p class="graf graf--p">
   不过它们并不是泾渭分明的、不可调和的对立面，而是
   <strong class="markup--strong markup--p-strong">
    一种判断 ——
   </strong>
   当情势更加倾向于表格的左边，说明你正在做的是常规的报道；当情势向表格的右面转化时，说明你开始了调查行动。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10883 jetpack-lazy-image" data-lazy-sizes="(max-width: 896px) 100vw, 896px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-29.png?resize=896%2C1194&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-29.png?w=896&amp;ssl=1 896w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-29.png?resize=225%2C300&amp;ssl=1 225w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-29.png?resize=768%2C1023&amp;ssl=1 768w" data-recalc-dims="1" height="1194" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-29.png?resize=896%2C1194&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="896"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10883" data-recalc-dims="1" height="1194" sizes="(max-width: 896px) 100vw, 896px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-29.png?resize=896%2C1194&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-29.png?w=896&amp;ssl=1 896w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-29.png?resize=225%2C300&amp;ssl=1 225w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-29.png?resize=768%2C1023&amp;ssl=1 768w" width="896"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   调查性工作看起来是不是远远多于普通的新闻工作？
  </p>
  <p class="graf graf--p">
   事实上，即使你能够愉快地有效率地投入其中，过程中的每一步仍然需要更多的工作。当然，它也有更多的回报，无论是对于公众、对于你的组织，还是对于你本人。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     挑选一个故事来展开调查
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   开始的时候，人们经常会问：“你是如何选择一个故事展开调查的？”他们在找故事的过程中经常遇到困难。
  </p>
  <p class="graf graf--p">
   但素材是无处不在的。问题是如何去发现它。幸运的是，有许多方法可以注意到那些能引发调查的故事。
  </p>
  <p class="graf graf--p">
   其中一个方法就是关注媒体。
  </p>
  <p class="graf graf--p">
   一般来说，
   <strong class="markup--strong markup--p-strong">
    监看某个特定的领域
   </strong>
   是一个好主意，这样你可以开始识别一些模式，可以意识到这个领域内有什么不正常的事情发生了。
  </p>
  <p class="graf graf--p">
   如果你看完一个故事并开始思考：“为什么会出现这种情况”，这意味着一个好的机会，因为有更多需要去调查。
  </p>
  <p class="graf graf--p">
   另一种方法是注意你周围的环境发生了怎样的变化，⚠️而不是想当然。
  </p>
  <p class="graf graf--p">
   最出色的比利时记者 Chris de Stoop 上班经过某个街区时察觉到，原先在那里的比利时妓女已经让位于外国妇女，他想知道这是为什么，从而开始了一个具有里程碑意义的关于贩卖妇女的调查。
  </p>
  <p class="graf graf--p">
   第三种方法是倾听人们的抱怨。
  </p>
  <p class="graf graf--p">
   ⚠️为什么事情必须这样呢？就没有什么可以做的吗？
  </p>
  <p class="graf graf--p">
   任何地方人群聚集的地方，比如乡村市场，网络论坛，晚宴等等，你都会听到许多听起来奇怪的，令人震惊的，或引发好奇的事情。
  </p>
  <p class="graf graf--p">
   最后，不要只看到那些涉及不当行为的事情。
  </p>
  <p class="graf graf--p">
   报道某些正面的事物 — — 理解一个新的人才、一个能实现伟大目标的开发项目，一个创造财富和就业机会的公民组织，往往更为困难。
  </p>
  <p class="graf graf--p">
   从中发现不可替换的成功要素或“最佳做法”，这对于你的受众来说也是一种有价值的服务。
  </p>
  <p class="graf graf--p">
   我们经常会感慨于那些优秀的活动家 “只有当他们入狱后才被媒体和大众所关注”，而在他们从事最有价值的事业的高峰期，却不为人知。这是一种痛苦的遗憾。⚠️
   <strong class="markup--strong markup--p-strong">
    公民记者应该摆脱职业记者的 “吸睛度” 思考方式，真正从社会利益角度开展你的工作。
   </strong>
  </p>
  <p class="graf graf--p">
   还要记住：尤其是当你刚开始着手的时候，根本就没有小调查这回事。你在遥远的村庄展开调查所需要的技能和你日后在首府所需要的技能是一样的。
  </p>
  <p class="graf graf--p">
   这不是一个理论，而是经验。
   <strong class="markup--strong markup--p-strong">
    用你现在身边出现的故事来构建这些技能，不要等到你去参与一个高风险的调查时再想要去弄明白自己在做什么
   </strong>
   。
  </p>
  <p class="graf graf--p">
   最后，也是最重要的一点，跟着你的激情走，这个原则体现为以下两个方面：
  </p>
  <p class="graf graf--p">
   第一个方面就是我们所说的
   <strong class="markup--strong markup--p-strong">
    “断腿综合症”
   </strong>
   。我们给它起了这个名字是因为，直到我们中的某个人摔断了腿，才会注意到有多少人一瘸一拐。
  </p>
  <p class="graf graf--p">
   ⚠️一般来说，我们不会注意到某些现象，除非我们已经对它们很敏感。
  </p>
  <p class="graf graf--p">
   所以，💡请让你现有的热情带领你去敏锐地发现那些别人没有认真对待的故事。
  </p>
  <p class="graf graf--p">
   第二个方面是，如果一个故事不能让你着迷，或者让你愤怒，或者给你强烈的愿望去弄明白发生了哪些变化，那么你应该把它留给别人。
  </p>
  <p class="graf graf--p">
   同样，如果你是一名编辑，请注意你的记者是否把一项调查仅仅当作任务。如果是这样，把任务收回，让别人去做。
  </p>
  <p class="graf graf--p">
   为什么要这样呢？请记住：调查涉及到许多额外的工作。⚠️如果你不关心一个故事，你就不会做这些工作。当然，你会使用批判的态度去完成；当然，你的态度可能在所有情况下都保持专业。但是，
   <strong class="markup--strong markup--p-strong">
    如果这个故事没有触动你的情感，无论怎样你都会以失败告终
   </strong>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     这个故事值得吗？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   很多调查是由于错误的理由开展的。
  </p>
  <p class="graf graf--p">
   虽然激情很重要，但是复仇也是一种激情，有一些记者和媒体利用调查来实现个人的复仇。
  </p>
  <p class="graf graf--p">
   ⚠️虽然调查是辛勤的工作，但其中有些故事被报道只是因为它们最容易获得。有很多调查者甚至没有考虑这个故事对于读者来说是否是重要的、为什么重要……
  </p>
  <p class="graf graf--p">
   所以当你遇到一个故事，还不确定它是否值得你付出努力的时候，
   <strong class="markup--strong markup--p-strong">
    问问自己以下几个问题：
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    有多少人会受到影响？（我们称其为“野兽的大小”。）
   </li>
   <li class="graf graf--li">
    他们受到影响的程度有多大？（质量和数量是同样重要的。哪怕只有一个人死亡，或者他或她的生活被毁了，这个故事就是重要的）
   </li>
   <li class="graf graf--li">
    如果他们受到的影响是积极的，导致积极影响的因素可以在其它地方复制吗？
   </li>
   <li class="graf graf--li">
    或者，这些人是受害者吗？
   </li>
   <li class="graf graf--li">
    他们的苦难可以避免吗？
   </li>
   <li class="graf graf--li">
    我们能展示出如何避免吗？
   </li>
   <li class="graf graf--li">
    是否有做坏事的人必须受到惩罚？或者至少，受到谴责？
   </li>
   <li class="graf graf--li">
    把发生的事情说出来，为了让它再次发生或者不会再发生，是否在任何情况下都是重要的？
   </li>
  </ul>
  <p class="graf graf--p">
   这个世界充满了痛苦，但是有些痛苦是无用的，仅仅是恶习和错误的结果。
  </p>
  <p class="graf graf--p">
   任何能够减轻苦难、残忍和愚蠢的事情都是值得做的。
  </p>
  <p class="graf graf--p">
   调查可以进一步实现这一目标。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    要把这个服务目标放在第一位，而不是简单地利用调查来推进你的事业
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️永远不要忘记调查是一种武器，你使用它的时候会对别人造成伤害，可能是故意的，也可能由于你的疏忽。 （水门事件中的名人伍德沃德和伯恩斯坦自己也承认，由于事实没有弄清楚，毁掉了尼克松身边几个无辜人的饭碗）
  </p>
  <p class="graf graf--p">
   在你的职业生涯中，你可能会成为某些人遭遇到的最好的事，也可能是某些人遭遇到的最糟糕的事。
   <strong class="markup--strong markup--p-strong">
    注意你在扮演着什么角色、对谁而言、以及为什么。
   </strong>
  </p>
  <p class="graf graf--p">
   在对别人进行调查之前，仔细审视一下你自己的动机。
   <strong class="markup--strong markup--p-strong">
    如果这个故事对于别人来说没有对于你自己那么重要，你可能不应该报道它
   </strong>
   。
  </p>
  <p class="graf graf--p">
   无论你想要说什么，最好是发自内心的 —— 更重要的是，这些话对于你的讲述对象是有意义的。
   <strong class="markup--strong markup--p-strong">
    人们不喜欢记者，原因之一就是他们不信任记者的动机。希望公民记者能有助于改变这种状况 ——为了民主为了知情权而工作。
   </strong>
  </p>
  <p>

   </noscript>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     使用假设：调查方法的核心
    </strong>
   </span>
  </h3>
  <blockquote class="graf graf--blockquote">
   <p>
    假设是一个故事和检验它的方法
   </p>
  </blockquote>
  <p class="graf graf--p">
   记者们总是抱怨编辑拒绝他们伟大的故事设想。当然，这种事会发生。但通常的情况是，编辑拒绝的并不是一个故事，而是被拖下水的邀请。
  </p>
  <p class="graf graf--p">
   计划不周的调查等于是在为了一个不确定的结果耗费时间和金钱。
  </p>
  <p class="graf graf--p">
   例如，说：“我想调查腐败”，这对于一个编辑来说并不是伟大的构想。当然，腐败在世界各地都存在。如果花足够的时间去寻找，你会找到一大堆腐败现象。
  </p>
  <p class="graf graf--p">
   但是腐败本身是一个主题，它不是一个故事，而记者所要做的是讲故事。
  </p>
  <p class="graf graf--p">
   如果你追求一个主题，而不是一个故事，你可以成为这个问题的专家，但要花费很多的时间、金钱和精力。
  </p>
  <p class="graf graf--p">
   这就是为什么任何一个有头脑的编辑都会告诉你，“不要做”。
  </p>
  <p class="graf graf--p">
   相反，如果你说“家长们期望子女们过上更好的生活，但学校的制度腐败已经摧毁了他们的希望。”那么你就是在讲一个具体的故事。这已经变得更加有趣了。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    无论你是否知道，你都是在提出一个假设。因为你还没有证明你的故事是真的。
   </strong>
   您现在提出学校里存在腐败，并且它已经对至少两个群体 — — 家长和儿童具有毁灭性的影响。这可能是真的，也可能是假的，你还需要掌握事实。
  </p>
  <p class="graf graf--p">
   在此期间，如果你不确定这个故事是否有意义，你就必须回答假设所定义的几个具体问题。
  </p>
  <p class="graf graf--p">
   这些具体的问题产生于我们对假设的分解，看这个假设可以分为几个独立的具体的要求。然后，可以去逐个核实这些要求。
  </p>
  <p class="graf graf--p">
   此外，还要看一下用来讲故事的这些词的具体含义，因为你必须发现和界定其含义。
  </p>
  <p class="graf graf--p">
   以下是
   <strong class="markup--strong markup--p-strong">
    这个过程的示意图
   </strong>
   —— 首先从假设出发＝》现在分解成不同的概念＝》然后更加严密地界定这些概念，看它会产生什么问题：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10884 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-21.png?resize=1100%2C569&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-21.png?w=1472&amp;ssl=1 1472w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-21.png?resize=300%2C155&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-21.png?resize=1024%2C530&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-21.png?resize=768%2C398&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-21.png?resize=1100%2C569&amp;ssl=1 1100w" data-recalc-dims="1" height="569" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-21.png?resize=1100%2C569&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10884" data-recalc-dims="1" height="569" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-21.png?resize=1100%2C569&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-21.png?w=1472&amp;ssl=1 1472w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-21.png?resize=300%2C155&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-21.png?resize=1024%2C530&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-21.png?resize=768%2C398&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-21.png?resize=1100%2C569&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   你可以以任何顺序回答这些问题，但最明智的顺序几乎总是你最容易实现的那个。
  </p>
  <p class="graf graf--p">
   任何调查迟早都会遇到困难，因为它涉及大量的事实、大量的资料来源，这意味着你需要组织大量的素材。并且，在拿你的名誉冒险之前，你还要担心你的故事是否准确。
  </p>
  <p class="graf graf--p">
   在上述案例中，一旦你找到
   <strong class="markup--strong markup--p-strong">
    至少四个消息源向你确认
   </strong>
   学校确实存在腐败现象（四个以下的消息来源说明调查的基础还不牢靠），你可以开始研究学校系统是如何运作的。
  </p>
  <p class="graf graf--p">
   你需要研究它的规则、程序，及其既定的理想和使命。当你知道系统如何运作，你将看到腐败可以滋生的灰色和黑色地带。然后，您可以将你所听到的和发现的事实同这个系统宣称的理念进行对比。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     用假设推动调查的优点 ——
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   以上的例子听起来是不是和许多工作相似？这是因为它和那些工作本来就是一致的，它只是有别于大多数新闻报道的写作，那些报道的方法不过是同一两个消息源交谈或者是改写一篇见报的报道。
  </p>
  <p class="graf graf--p">
   不过只要你将假设的方法同其它调查的方法进行比较，就会发现其节省人力的优点是非常明显的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    🌟1、假设给你一个需要验证的东西，而不是去发现一个秘密。
   </strong>
  </p>
  <p class="graf graf--p">
   如果没有合理的理由，人们通常不会透露他们的秘密。
  </p>
  <p class="graf graf--p">
   他们更可能证实你已经掌握的信息，只是因为大多数人不愿意说谎。
  </p>
  <p class="graf graf--p">
   假设可以让你去向他们核实你的信息，而不是直接去询问信息。
  </p>
  <p class="graf graf--p">
   假设还会让你处于一个开放的探询的立场，因为你愿意接受那些超出你一开始所怀疑的事实。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    🌟2、假设还会增加发现秘密的机会。
   </strong>
  </p>
  <p class="graf graf--p">
   很多我们所说的“秘密”只是一些从来没有人问起的事实。假设会对你产生一种心理影响，从而使你对素材更为敏感，所以你可以问出这些问题。
  </p>
  <p class="graf graf--p">
   正如法国的调查人员 Edwy Plenel 所说：“如果你想有所发现，你就必须一直处于寻找的状态。” 还要补充一点，如果一直在寻找，你会发现你所找到的比你预期的还要多。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    🌟3、假设会让你的项目管理更容易
   </strong>
  </p>
  <p class="graf graf--p">
   一经确定你要找的是什么，以及从哪里开始寻找，你就可以估算调查的开头几个步骤需要多少时间。
  </p>
  <p class="graf graf--p">
   这是把调查作为一个可以管理的项目的第一步。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    🌟4、假设是一种可以反复使用的工具
   </strong>
  </p>
  <p class="graf graf--p">
   当你能有条不紊地工作时，你的职业生涯将会发生改变。更重要的是，你本人也会将会改变。
  </p>
  <p class="graf graf--p">
   你将不再需要有人告诉你去做什么，而是会看到需要去做什么和这个世界上的混乱和痛苦做斗争，并且你将能够实现它。
  </p>
  <p class="graf graf--p">
   这不正是你要成为一个公民记者的首要理由吗？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    🌟5、假设可以切实保证你将提供的是一个故事，而不是一大堆的数据
   </strong>
  </p>
  <p class="graf graf--p">
   假设大大增加了产出的可能性。它使你能够预测到你工作所产生的最小的和最大的积极结果，以及最糟糕的情况。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     假设可能是危险的 ——
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   有经验的记者都知道，当你报道了一个错误的故事时，你可能会被起诉，有时还会被送入监狱，无论是对是错。
  </p>
  <p class="graf graf--p">
   但是，还有一个不显而易见的后果 — — 讲述一个不真实的故事会让这个世界变得更加丑陋和无望。
  </p>
  <p class="graf graf--p">
   所以请记住这一点：⚠️如果你只是试图不惜一切代价地证明你的假设是真实的，甚至忽略证据，那么你将加入世界职业骗子的行列。比如那些制裁无辜的警察，那些贩卖战争如同贩卖肥皂的政客。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    调查可不是为了证明你是对的，而是要寻找真相
   </strong>
   。
  </p>
  <p class="graf graf--p">
   以假设为基础的调查是一种工具，可以挖出很多真相，但它也同样可以挖出无辜者的坟墓。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-10886 jetpack-lazy-image" data-lazy-sizes="(max-width: 1051px) 100vw, 1051px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-6.jpeg?resize=1051%2C700&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-6.jpeg?w=1051&amp;ssl=1 1051w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-6.jpeg?resize=300%2C200&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-6.jpeg?resize=1024%2C682&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-6.jpeg?resize=768%2C512&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-6.jpeg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="700" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-6.jpeg?resize=1051%2C700&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1051"/>

  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     假设是如何运行的 ——
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   👉1、为什么第一个假设是否真实并不重要
  </p>
  <p class="graf graf--p">
   把调查建构成假设是一个像科学一样古老的程序，并且它已经成功地在各个领域运用，从警务工作到商业咨询。（事实上，这个方法最近才被借鉴到新闻学领域是不正常的）
  </p>
  <p class="graf graf--p">
   从本质上讲，
   <strong class="markup--strong markup--p-strong">
    它基于一种心理技巧
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在了解一定信息的基础之上，你建立一个你对事实看法的陈述，然后进一步寻求信息来证明或反驳你的陈述。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这是一个核实过程
   </strong>
   。即使整个假设无法被确认，某个单独的概念依然可以被验证。
  </p>
  <p class="graf graf--p">
   如果还是不行，返回步骤之一，提出新的假设。因为一个整体和局部都无法验证的假设仅仅是猜测。
  </p>
  <p class="graf graf--p">
   如果证据支持你的陈述，很棒，你已经有故事了。不显而易见的是，即使陈述被证明是错误的，也是很好的事，因为这意味着可能有一个比你原来想象的更好的故事。
  </p>
  <p class="graf graf--p">
   👉2、成功在于构建假设
  </p>
  <p class="graf graf--p">
   最初的假设应该不超过三句话，理由有两点。如果太长，你就无法解释给别人听。
  </p>
  <p class="graf graf--p">
   更重要的是，如果太长，你可能自己都不明白它到底什么意思。
  </p>
  <p class="graf graf--p">
   假设被陈述成一个故事。这点很重要，因为这意味你要像讲故事那样，结尾时要呼应开头 。
  </p>
  <p class="graf graf--p">
   公民记者不只是收集事实，而是在讲故事，讲那些你希望可以改变世界的故事。
  </p>
  <p class="graf graf--p">
   假设将帮助你把这个故事解释给别人听，从你的合作者、编辑和出版商开始，然后是公众。
  </p>
  <p class="graf graf--p">
   按照最基本的形式，
   <strong class="markup--strong markup--p-strong">
    故事一般可以阐述为以下三句话
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    我们正面临一种形势，它造成了极大的痛苦（或者这个事情是一个很好
    <br/>
    的例子，值得被更多的人知晓）；
   </li>
   <li class="graf graf--li">
    我们是如何走到这一步的；
   </li>
   <li class="graf graf--li">
    如果什么都不改变，将会发生什么……我们如何做可以让事情有所转变。
   </li>
  </ul>
  <p class="graf graf--p">
   👉3、体会以下这些句子
  </p>
  <p class="graf graf--p">
   它们有一个隐含的时间顺序。但似乎并不明显，因为顺序并不是从过去到未来一条直线。而是告诉我们以下内容：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    关于这个问题的新闻，这是目前进展；
   </li>
   <li class="graf graf--li">
    造成这个问题的原因，这是过去的；
   </li>
   <li class="graf graf--li">
    要解决问题必须改变什么，这是未来的。
   </li>
  </ul>
  <p class="graf graf--p">
   因此，当构建假设的时候，您已经开始构建叙事。
  </p>
  <p class="graf graf--p">
   一个故事涉及到在特定地点和时间中移动的人物。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    调查中最难的事情就是保持对叙事的专注，而不被大量的事实所淹没
   </strong>
   。
  </p>
  <p class="graf graf--p">
   假设可以帮助你做得这一点。当你感到不知所措的时候，停止挖掘，想想这些事实是要告诉你一个什么样的故事。
  </p>
  <p class="graf graf--p">
   如果不符合原来的假设，改变它。毕竟，这只是一个假设。
  </p>
  <p class="graf graf--p">
   顺便提一句，想要展示如何解决一个现有的问题是非常非常困难的。有时候，你所能做到的就是谴责不公平。但是往往和你故事相关的某个人已经找到了问题的解决方案，不要忽视寻找这个人。
  </p>
  <p class="graf graf--p">
   👉4、让假设生效的四个要点
  </p>
  <p class="graf graf--p">
   使用假设并不是一项复杂的技巧，但是除非你比我们更有天赋（我们接受这种可能），否则在你对这种方法习以为常之前，可能需要进行几次尝试。
  </p>
  <p class="graf graf--p">
   以下是你需要记住的四个要点。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    # 富有想象力
   </strong>
  </p>
  <p class="graf graf--p">
   通常记者对形势做出反应。他们报道他们看到的，或听到的，或读到的，或跟进昨天的新闻。
  </p>
  <p class="graf graf--p">
   而
   <strong class="markup--strong markup--p-strong">
    调查员试图揭示某些未知的东西
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️公民调查记者并不不仅仅是报道新闻，而是制造新闻。
  </p>
  <p class="graf graf--p">
   因此必须进行必要的飞跃，进入一个不确定的未来。
  </p>
  <p class="graf graf--p">
   这意味着要试着描画一个故事，这是一项创造性的工作。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    # 要非常精确
   </strong>
  </p>
  <p class="graf graf--p">
   如果你在假设中使用“房子”这个词，它是一个别墅，还是阁楼，还是窝棚？
  </p>
  <p class="graf graf--p">
   答案很重要。你假设的事实越准确，就越容易验证。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    # 使用你的经验
   </strong>
  </p>
  <p class="graf graf--p">
   如果你已经认识到，世界在以某种方式运转，这可能适用于你试图验证的故事。你的经验可以帮助你完善假设。
  </p>
  <p class="graf graf--p">
   不过请记住，即使是最有经验的人也会为从未见过的东西感到惊奇，甚至十分自信的人也会怀疑自己的亲身经历。
  </p>

  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     专注于故事
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   请牢记：公民记者提出的每一个假设必须被当作一个真实的故事来建构。
  </p>
  <p class="graf graf--p">
   它包含新闻、原因、和解决办法。
  </p>
  <p class="graf graf--p">
   这意味着，为了让假设看上去牢不可破，您关注的重点是故事，而不
   <br/>
   仅仅是事实。
  </p>
  <p class="graf graf--p">
   事实可能是你故事的基础，但他们本身并不讲述故事，而是故事在讲述事实。
  </p>
  <p class="graf graf--p">
   ⚠️没有人能记得住他地址簿上的前三行，但每个人都记得他们地址簿中关于每个名字的故事。
  </p>
  <p class="graf graf--p">
   从一开始就把你的调查建构成一个故事（记住，这个故事不一定是真实的），这不只是帮助你最终的读者或观众记住它，还是在
   <strong class="markup--strong markup--p-strong">
    帮助自己记住它
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这就是调查最困难的部分  — — 在事实叠加的时候要牢记故事。
  </p>
  <p class="graf graf--p">
   花时间努力成为这个方法的专家。在每次调查过程中实践它。它将使你更幸运，并且重复你的运气。 `
  </p>

   下一篇文章中您将能看到在哪可以找到公开的挖掘线索，或者用老前辈们习惯的称呼 —— “没锁的门”，以及处理技巧。下次见。⚪️
  </p>
