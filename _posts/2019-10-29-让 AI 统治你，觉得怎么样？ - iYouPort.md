  </p>
  <p class="graf graf--p">
   有些人可能会说政客们不会变得更糟，于是IFLScience.com提了一个问题：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    如果你有机会投票给一个你完全信任的政客，你肯定他们没有隐藏的议程、会真正代表选民的观点，你会投，对吗？
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    如果这位政治家是机器人怎么办？不是具有机器个性的人，而是真正的人工智能机器人
   </em>
   。
  </p>
  <p class="graf graf--p">
   其实IYP去年介绍过一本书，其中的主题就是这个问题，详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E6%94%BF%E6%B2%BB%E7%9A%84%E6%9C%AA%E6%9D%A5-%E5%BD%93%E6%9C%BA%E5%99%A8%E5%86%B3%E5%AE%9A-%E6%B0%91%E4%B8%BB-f71326b9c35f" href="https://medium.com/@iyouport/%E6%94%BF%E6%B2%BB%E7%9A%84%E6%9C%AA%E6%9D%A5-%E5%BD%93%E6%9C%BA%E5%99%A8%E5%86%B3%E5%AE%9A-%E6%B0%91%E4%B8%BB-f71326b9c35f" rel="noopener noreferrer" target="_blank">
    政治的未来：当机器决定“民主”？
   </a>
   》
  </p>
  <p class="graf graf--p">
   据一项令人惊讶的新调查显示，
   <strong class="markup--strong markup--p-strong">
    超过四分之一的欧洲人宁愿让他们国家的人工智能做出重要的政治决定，而不是把决策权交给他们当选的和未经选举的人类官员
   </strong>
   。
  </p>
  <p class="graf graf--p">
   四分之一的欧洲人表示他们“有点或完全赞成让人工智能做出关于他们国家运作的重要决定”，这个数字在荷兰、英国和德国上升到了三分之一。这是来自西班牙 IE 大学的以技术为重点的研究小组“变革治理中心”的调查。
  </p>
  <p class="graf graf--p">
   这些数字在教育水平、性别和政治派别上没有不同，表明欧洲人对他们的新机器人霸主非常欢迎 — — 或者说至少他们已经厌倦了人类。
  </p>
  <p class="graf graf--p">
   正如 Helen Buyniski 所报告的那样，该调查揭示了人们的高度技术焦虑，甚至担心被机器人偷走一个人的工作的大趋势也不会影响欧洲人对他们的政治主人的反感程度，那些人类当权者已经表明他们都非常愿意抛弃他们的选民 — — 无论是法国的伊曼纽尔·马克龙试图扼杀抗议活动，还是禁止他的新闻发布会上的关键媒体，或者特蕾莎·梅在反向心理学方面多次提供的令人厌恶的英国脱欧协议。
  </p>
  <p class="graf graf--p">
   该调查的结果描述为“对欧洲代议制民主模式的重大质疑”并声称它“挑战了人民主权的概念”，CGC 执行董事 Diego Rubio 避免提及越来越多的欧洲公民发现了欧盟本身可能已经挑战了“人民主权的概念。”
  </p>
  <p class="graf graf--p">
   尽管试图将这种“欧洲怀疑主义”归咎于俄罗斯、意大利的五星运动、和荷兰的PVV等政党，但像马克龙、梅和德国的安吉拉默克尔这样的现状支持者却在创纪录的低支持率中萎靡不振。
  </p>
  <p class="graf graf--p">
   尽管对人类统治的未来前景有些暗淡的看法，依旧有“超过三分之二的各个年龄段的欧洲人认为，如果不加以适当的控制，新技术将在未来十年对社会造成更大的伤害，”
  </p>
  <p class="graf graf--p">
   IE 大学的调查发现，与此同时，有三分之二到四分之三的受访者都支持“限制自动化的强有力的政策措施”。
  </p>
  <p class="graf graf--p">
   研究人员发现，这种威胁远不是抽象的 — — 大多数欧洲人担心机器人接管大部分人类工作，而近一半人担心他们的雇主会在新的硅统治者崛起之前堕落。大多数人认为他们的教育也没有为技术变革的快节奏做好充分的准备。
  </p>
  <p class="graf graf--p">
   超过三分之二的人“担心”或“非常担心”他们的社交生活也会完全数字化 — — 在一个日益严格的在线言论规则的时代，这是一个令人沮丧的前景。有趣的是，除西班牙和葡萄牙外，每个国家都有超过一半的受访者支持禁止社交网络中的“政治和意识形态内容”，“以保护民主”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “人们对政府感到失望，但同时要求他们解决新兴技术可能带来的社会和经济负面影响，”作者写道，指出结果的“矛盾”性，由于那些相同的新兴技术包括人工智能，欧洲人是否期望他们的机器人大师一旦掌权就会自我调节呢？
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap graf--startsWithDoubleQuote">
   <span class="graf-dropCap">
    <span class="graf-dropCapQuote">
     “
    </span>
    P
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.zerohedge.com/news/2019-03-25/sick-human-politicians-quarter-europeans-prefer-ai-government" href="https://www.zerohedge.com/news/2019-03-25/sick-human-politicians-quarter-europeans-prefer-ai-government" rel="noopener noreferrer" target="_blank">
    eople are disillusioned with governments, yet at the same time ask them to tackle the societal and economic negative effects that emerging technologies might have
   </a>
   ”. Sick Of Human Politicians? A Quarter Of Europeans Prefer AI Government
  </p>
