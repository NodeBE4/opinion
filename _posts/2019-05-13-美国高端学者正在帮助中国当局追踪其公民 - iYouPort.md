 <div class="entry-content">
  <p class="graf graf--p">
   英国“金融时报 ”发布的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.ft.com/content/41be9878-61d9-11e9-b285-3acd5d43599e" href="https://www.ft.com/content/41be9878-61d9-11e9-b285-3acd5d43599e" rel="noopener noreferrer" target="_blank">
    一份报告显示
   </a>
   ，西方学者正在帮助中国政府监测其公民的一举一动，其中越来越先进的监控技术以人工智能为基础。
  </p>
  <p class="graf graf--p">
   英国“金融时报” Madhumita Murgia 的报告指出，至少有九篇学术论文是由美国学者和那些向中国政府出售监控技术的公司、或者与国家军队有联系的机构 如中国国防科技大学（NUDT）共同撰写的。
  </p>
  <p class="graf graf--p">
   中国公民是世界上最受关注的人群之一。 全国有
   <a class="markup--anchor markup--p-anchor" data-href="https://techcrunch.com/2017/12/13/china-cctv-bbc-reporter/" href="https://techcrunch.com/2017/12/13/china-cctv-bbc-reporter/" rel="noopener noreferrer" target="_blank">
    超过1亿台
   </a>
   闭路电视摄像机，政府使用它们来跟踪和控制人们去了哪里以及做了什么。
  </p>
  <p class="graf graf--p">
   人权活动人士对中国政府如何利用监控技术瞄准新疆地区的维吾尔人提出了特殊的问题，人们普遍担心北京在监视方面走得太远 —— 变成了老大哥之国。
  </p>
  <p class="graf graf--p">
   英国“金融时报”确定的研究伙伴关系揭示，
   <strong class="markup--strong markup--p-strong">
    西方学者正在与中国当局在侵犯人权方面同谋。
   </strong>
  </p>
  <p class="graf graf--p">
   这涉及诺基亚贝尔实验室、普林斯顿大学、麻省理工学院、伊利诺伊大学厄巴纳 — 香槟分校、和澳大利亚悉尼大学等组织，据英国“金融时报”报道，他们都拒绝发表评论。
  </p>
  <p class="graf graf--p">
   有趣的是，
   <strong class="markup--strong markup--p-strong">
    有四位美国学者与谷歌相关
   </strong>
   。
  </p>
  <p class="graf graf--p">
   谷歌高级职员研究科学家托马斯·法克豪瑟（Thomas Funkhouser）去年在普林斯顿大学担任客座教授期间，曾与两名NUDT科学家一起工作。三人组研究了计算机视觉如何用于无人驾驶飞机和无人水下航行器。他们的论文可以
   <a class="markup--anchor markup--p-anchor" data-href="https://arxiv.org/pdf/1803.08407.pdf" href="https://arxiv.org/pdf/1803.08407.pdf" rel="noopener noreferrer" target="_blank">
    在这里找到
   </a>
   。 Funkhouser 拒绝发表评论。
  </p>
  <p class="graf graf--p">
   据报道，
   <strong class="markup--strong markup--p-strong">
    谷歌的工作人员与中国的研究人员和企业合作
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Google 的一位发言人就此辩称，这些是大学研究人员撰写的学术论文 — “谷歌没有参与这些项目，也没有与中国大学建立合作关系”。
  </p>
  <p class="graf graf--p">
   美国和中国正在主宰世界人工智能“竞赛”，但中国正在对此大力投资以获得优势。中国还将本国科学家派往西方大学，以获得海外人工智能方面的专业知识。
  </p>
  <p class="graf graf--p">
   在澳大利亚战略政策研究所研究中国共产党的研究员 Alex Joske 说，
   <strong class="markup--strong markup--p-strong">
    近年来有数千名科学家与中国军队有联系。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “就如在普林斯顿大学的那些人，是在过去十年中被派往国外作为博士生或访问学者的数千名[中国军队]官员和干部。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我认为这真的令人担忧，因为大学无法确保他们帮助这些[实体]发展和改进的技术将以何种方式方式被使用。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.forbes.com/sites/samshead/2019/04/21/researchers-at-us-universities-are-reportedly-helping-china-to-track-its-citizens/#58e8e81a61cd" href="https://www.forbes.com/sites/samshead/2019/04/21/researchers-at-us-universities-are-reportedly-helping-china-to-track-its-citizens/#58e8e81a61cd" rel="noopener noreferrer" target="_blank">
    Researchers At U.S. Universities Are Reportedly Helping China To Track Its Citizens
   </a>
  </p>
