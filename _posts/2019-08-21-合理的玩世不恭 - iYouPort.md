   合理的玩世不恭

      人们为什么要支持特朗普，而且很可能会再次支持他？民主党人必须承认自己的错误才能改变这一现状，为什么？
     </em>

   </noscript>
  </p>
  <p class="graf graf--p">
   有一种新兴的观点认为，民主党在2020年获胜的方式是转移到中间并获得一些中间派选票。这是被新自由主义机构、主流媒体、以及大部分*付费政治专家*推动的观点。
  </p>
  <p class="graf graf--p">
   但是，这里有两个问题。
  </p>
  <p class="graf graf--p">
   首先，没有什么“中间”，或者更确切地说，它
   <a class="markup--anchor markup--p-anchor" data-href="https://www.people-press.org/2019/03/14/political-independents-who-they-are-what-they-think/" href="https://www.people-press.org/2019/03/14/political-independents-who-they-are-what-they-think/" rel="noopener noreferrer" target="_blank">
    是微不足道的
   </a>
   。太多的权威人士将无党派人士与中间派混为一谈了，事实上，绝大多数专家都倾向于这种误区。
  </p>
  <p class="graf graf--p">
   如果你把左倾的无党派人士和民主党人加起来，他们相当于48％的选民，而右倾无党派人士和共和党人则占选民总数的39％。真正的“中间派”只占7％左右。
  </p>
  <p class="graf graf--p">
   其次，选举政治中的真正奖项是没有表现。要知道为什么会这样，我们可以检查最近的两次选举。2016年，约有58％的合格选民参加了投票，特朗普获胜，这得益于选举团；而希拉里克林顿获得了大约28％的选民。来看看2016年的数字：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    6580万人投票给克林顿;
   </li>
   <li class="graf graf--li">
    有6290万人投票支持特朗普；
   </li>
   <li class="graf graf--li">
    690万投票给第三党候选人;
   </li>
   <li class="graf graf--li">
    9600万人没有投票。
   </li>
  </ul>
  <p class="graf graf--p">
   美国有
   <a class="markup--anchor markup--p-anchor" data-href="http://www.electproject.org/2016g" href="http://www.electproject.org/2016g" rel="noopener noreferrer" target="_blank">
    近2.31亿合格选民
   </a>
   ，但2016年选举中只有1.35亿选民投票。做一个简单的计算。一方面，你可以放弃你的价值观并“转移”到“中间”，以赢得1600万真正的中间派中的一部分；另一方面，你可以坚持自己的价值观，并追踪 9600 万没有投票的人中的一部分。
  </p>
  <p class="graf graf--p">
   2018年的选举是一个自1914年以来最大的投票率 — — 50.1％ — — 很大程度上是因为民主党人在宣传其进步的价值观，而且因为特朗普引起了某种恐惧和厌恶感。但是你想要赢就必须赢得49.9％的人参加。
  </p>
  <p class="graf graf--p">
   如果你检查
   <a class="markup--anchor markup--p-anchor" data-href="https://www.vox.com/policy-and-politics/2018/11/19/18103110/2018-midterm-elections-turnout" href="https://www.vox.com/policy-and-politics/2018/11/19/18103110/2018-midterm-elections-turnout" rel="noopener noreferrer" target="_blank">
    最近的选民投票率
   </a>
   统计数据，有两件事情会变得清晰。民主党在投票率很高的时候获胜，共和党在投票率低的时候获胜。实际上，这些记录还显示了另一件事 — — 共和党的胜利预示着创纪录的赤字和经济衰退。
  </p>
  <p class="graf graf--p">
   因此，你会认为民主党领导层将采取一种专注于针对失约者的战略？但是你错了。为什么呢？
  </p>
  <p class="graf graf--p">
   为了获得一些尚未投票的人的支持，你必须开展一个以人民利益为中心的运动，而不是利益集团的利益。
  </p>
  <p class="graf graf--p">
   现实情况是，人们对民主党的态度是合理的玩世不恭 — — 实际上对两党都是。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.huffpost.com/entry/its-candidate-smith-by-a-_b_5552229" href="https://www.huffpost.com/entry/its-candidate-smith-by-a-_b_5552229" rel="noopener noreferrer" target="_blank">
    Smith Project
   </a>
   揭示了人们玩世不恭的深度。例如：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    百分之八十六的选民认为政治领导人更有兴趣保护自己的权力，而不是为美国人民做正确的选择。
   </li>
   <li class="graf graf--li">
    83％的人认为，这个国家由现任政治家、媒体专家、游说者和其他利益集团联盟经营，以谋取利益。
   </li>
   <li class="graf graf--li">
    此外，79％的人认为，华尔街银行对公司、工会和 PAC 有强大的兴趣以使用竞选活动和游说资金来操纵系统为自己的利益服务，并且他们以牺牲每个美国人为代价来掠夺国库。
   </li>
  </ul>
  <p class="graf graf--p">
   如果你想了解特朗普在哪里得到的支持，Smith Project 的一个统计数据应该是显而易见的。大约77％的美国人喜欢“将政治精英和特殊利益”带到那些符合既定意识形态的候选人。特朗普的选民没有投票任何东西，他们投反对现状的票。
  </p>
  <p class="graf graf--p">
   所有那些愤世嫉俗者都是对的。这是系统性问题。例如，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.motherjones.com/politics/2016/12/america-income-inequality-wealth-net-worth-charts/" href="https://www.motherjones.com/politics/2016/12/america-income-inequality-wealth-net-worth-charts/" rel="noopener noreferrer" target="_blank">
    在1980年至2015年期间
   </a>
   ，最高的.01％极富人群的收入增长了322％，而最低的90％的收入仅增长了0.03％。
  </p>
  <p class="graf graf--p">
   而且不仅仅是经济。
   <a class="markup--anchor markup--p-anchor" data-href="https://scholar.princeton.edu/sites/default/files/mgilens/files/gilens_and_page_2014_-testing_theories_of_american_politics.doc.pdf" href="https://scholar.princeton.edu/sites/default/files/mgilens/files/gilens_and_page_2014_-testing_theories_of_american_politics.doc.pdf" rel="noopener noreferrer" target="_blank">
    正如 Gilens 和 Paige 所表明的那样
   </a>
   ，在公共政策方面，人民的意志几乎无关紧要。实际上，他们得出的结论是，美国不再是一个民主国家，而是一个寡头集团，其中富人和特殊利益集团的意志始终胜过人民的利益。
  </p>
  <p class="graf graf--p">
   难怪人们愤世嫉俗。难怪“第二次世界大战”以来“失约者”赢得了每一次投票。
  </p>
  <p class="graf graf--p">
   这种合理的玩世不恭的解药是一场以价值观而非战术为主题的运动。是的，特朗普很可恶。但是揭露他的糟糕程度并不会削弱他的支持者人数，也不会让失约者出现。而且即便赢得足够的7％中间派也不会获得胜利。
  </p>
  <p class="graf graf--p">
   大多数美国人压倒性地支持进步政策，包括绿色新政、以及全民医保。但是，新自由主义者并没有采取明显的制胜战略，而是在对进步人士发达战争。
  </p>
  <p class="graf graf--p">
   Nancy Pelosi 的工作人员向医疗保健和制药说客保证，全民医保是不可能的；Pelosi 则自己嘲笑绿色新政。美国进步中心 — 新自由主义的堡垒 — 出版了一系列关于全民医保和伯尼桑德斯的热门文章。
  </p>
  <p class="graf graf--p">
   看起来就像 Pelosi，Schumer 等人正在把赌注押在揭示揭露特朗普有多糟糕上。试图获得他的纳税申报表，希望穆勒报告能够爆炸 — — 以及全力以赴地获得神话般的中间派 — — 似乎是他们2020年的战略。
  </p>
  <p class="graf graf--p">
   而问题就在这里。特朗普的支持者并不关心他有多糟糕。他们只关心他正在攻击那些已经非常顽固的政治权力。
  </p>
  <p class="graf graf--p">
   仅仅因为特朗普比一般政治家更糟糕，那些心怀不满的未投票者也不会出现在民意调查中。有太多原因了，两党不过是半斤八两的区别。
  </p>
  <p class="graf graf--p">
   可能有一些新自由主义者实际上认为，走向“中间”是赢得胜利的方式，但是对大多数人来说，新自由主义制度并没有采取明显的战略来赢回白宫和参议院。保留众议院很简单。
  </p>
  <p class="graf graf--p">
   他们是通过服务于富人、精英和大公司的利益来达成权力目的的。声称为人民服务的政策却将不利于他们选取的人排除出去。
  </p>
  <p class="graf graf--p">
   这将意味着所有人将冒险经历特朗普的连任。从字面上看，这是一种存在主义的威胁。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.commondreams.org/views/2019/04/15/why-us-elected-despot-and-why-its-poised-do-it-again" href="https://www.commondreams.org/views/2019/04/15/why-us-elected-despot-and-why-its-poised-do-it-again" rel="noopener noreferrer" target="_blank">
    Why the US Elected a Despot, and Why It’s Poised To Do It Again. Chasing the center and ignoring the real electoral prize
   </a>
   . Eighty-six percent of all voters believe political leaders are more interested in protecting their power than in doing what’s right for the American people. To get some of the no-shows on board, you’d have to run a campaign centered on the people’s interest, not the monied interests.
  </p>
