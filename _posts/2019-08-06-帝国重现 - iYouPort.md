  <p class="graf graf--p">
   苏维埃帝国的影子又在重现了。但这次不是社会主义的，而是资本主义的。普京曾公开承认俄罗斯是资本主义的；而且这次是更阴暗的手段：砸钱。
  </p>
  <p class="graf graf--p">
   今年1月，马达加斯加为总统安德里·拉乔利纳揭幕，他在两轮投票中击败了35名对手。根据BBC俄语部的一份
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bbc.com/russian/features-47851658" href="https://www.bbc.com/russian/features-47851658" rel="noopener noreferrer" target="_blank">
    新报告
   </a>
   ，至少有六名候选人 — — 可能包括 Rajoelina — — 从俄罗斯获得了竞选资金。
  </p>
  <p class="graf graf--p">
   BBC的调查记者发现，
   <strong class="markup--strong markup--p-strong">
    大量俄罗斯政治专家卷入了马达加斯加2018年的总统选举
   </strong>
   。英国新闻媒体没有具体说明有多少俄罗斯顾问活跃，但上个月
   <a class="markup--anchor markup--p-anchor" data-href="https://www.proekt.media/investigation/prigozhin-afrika/?utm_source=tlgrm&amp;utm_medium=chnl&amp;utm_campaign=mdgskr" href="https://www.proekt.media/investigation/prigozhin-afrika/?utm_source=tlgrm&amp;utm_medium=chnl&amp;utm_campaign=mdgskr" rel="noopener noreferrer" target="_blank">
    Proekt 网站
   </a>
   的一份报告引用了一位消息来源称有15至20名政治战略家在从事这项工作。
  </p>
  <p class="graf graf--p">
   据报道，俄罗斯政治人员的工作于2018年3月开始，持续了将近一年。据BBC报道，俄罗斯的战略家们
   <strong class="markup--strong markup--p-strong">
    作为游客和选举观察员进入该国
   </strong>
   ，然后与高级政治官员合作。两位前候选人现在公开谈论他们与俄罗斯政治顾问的合作：Andre Mailhol（在第一轮投票中获得第四名）和前总理Omer Beriziky（获得第24名）。
  </p>
  <p class="graf graf--p">
   Mailhol 说，他与来自俄罗斯的三名战略家 — — Andrey Kramar，Roman Pozdnyakov 和 Vladimir Boyaryshev 合作 — — 称
   <strong class="markup--strong markup--p-strong">
    他们为他的竞选活动捐赠了12,000欧元（13,515美元），为他的电视广告提供资金，并向他递交了5000欧元（5,630美元）的现金
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Mailhol 说他当时并不具体知道谁在为他的竞选工作，或者他们代表谁。他告诉记者，他认为俄罗斯顾问在此期间支持了八到九名不同的候选人。透明国际组织验证了许多候选人从国外捐助者那里获得了竞选资金。与此同时，英国广播公司表示，只有六名候选人与俄罗斯政治战略家合作。
  </p>
  <p class="graf graf--p">
   前总理 Omer Beriziky 说，俄罗斯顾问向他承诺了200万美元，尽管他从未收到过这笔钱中的大部分。发言人表示，他与俄罗斯战略家 Maxim Shugalei 合作，据称
   <strong class="markup--strong markup--p-strong">
    他承诺提供“技术支持”，并指示 Beriziky 如何控制选举。
   </strong>
  </p>
  <p class="graf graf--p">
   Andrey Kramar, Roman Pozdnyakov,和 Vladimir Boyaryshev 在一次被阻止的抗议活动后离开了马达加斯加。几位参与者说他们是有偿参加的。示威发生后，地方当局驱逐了几名俄罗斯国民，理由是他们在马达加斯加伪装游客参与工作。Kramar，Pozdnyakov 和 Boyaryshev 拒绝与记者交谈，但新闻媒体 Openmedia.io表示，Kramar 和 Pozdnyakov 曾经在乌克兰的 Donbas 地区工作。
  </p>
  <p class="graf graf--p">
   英国广播公司说，俄罗斯战略家最初帮助了多名候选人，但当 Andry Rajoelina 成为明确的领跑者时，他们将支持重定向到了 Andry Rajoelina。据报道，在竞争的这个阶段，俄罗斯顾问还建议 Andre Mailhol 和 Omer Beriziky 支持 Rajoelina。 Mailhol 说，当他拒绝时，战略家们就放弃了他。
  </p>
  <p class="graf graf--p">
   在选举期间，Rajoelina 告诉BBC他不认为自己是俄罗斯的候选人。“我不是任何国家的候选人 — 我是马达加斯加全体人民的候选人，”他解释说，拒绝透露他的竞选是否得到了俄罗斯的援助。与此同时，BBC指出，没有证据表明 Rajoelina 曾接受过俄罗斯的资助。
  </p>
  <p class="graf graf--p">
   据英国广播公司报道，
   <strong class="markup--strong markup--p-strong">
    圣彼得堡寡头组织 Evgeny Prigozhin 负责为在非洲旅行的俄罗斯政治战略家团队提供资金，为不同的候选人提供资金和建议。
   </strong>
  </p>
  <p class="graf graf--p">
   去年11月
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bloomberg.com/news/features/2018-11-20/putin-chef-yevgeny-prigozhin-is-now-meddling-in-africa" href="https://www.bloomberg.com/news/features/2018-11-20/putin-chef-yevgeny-prigozhin-is-now-meddling-in-africa" rel="noopener noreferrer" target="_blank">
    彭博社还报道说
   </a>
   ，“普京最臭名昭著的厨师现在正在非洲干涉其政治。”
   <a class="markup--anchor markup--p-anchor" data-href="https://www.proekt.media/investigation/prigozhin-afrika/?utm_source=tlgrm&amp;utm_medium=chnl&amp;utm_campaign=mdgskr" href="https://www.proekt.media/investigation/prigozhin-afrika/?utm_source=tlgrm&amp;utm_medium=chnl&amp;utm_campaign=mdgskr" rel="noopener noreferrer" target="_blank">
    Proekt 网站称
   </a>
   ，Prigozhin 的顾问与马达加斯加的八位总统候选人合作，其中包括 Andry Rajoelina。
  </p>
  <p class="graf graf--p">
   但 Prigozhin 否认这点，他的代表告诉记者，他“与任何可能在非洲提供政治建议的项目、或与记者提到的任何战略家都无关。但是，谁知道呢……
  </p>
  <p class="graf graf--p">
   请注意马达加斯加的战略地位：中国和美国都在非洲争抢影响力。
  </p>
