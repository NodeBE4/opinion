  <p class="graf graf--p">
   你经常能听到这样的荒谬观点：“我不需要隐私，因为我没什么可隐藏的”。
   <strong class="markup--strong markup--p-strong">
    这一观点至今都在全球流行，而不仅仅是中国
   </strong>
   。
  </p>
  <p class="graf graf--p">
   那么它是怎么来的？
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    简介
   </strong>
  </h3>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “没什么可隐藏” 的人认为，政府监视是为了发现 “非法活动”，它们不会威胁到隐私，只要你循规蹈矩；如果政府监视程序发现了非法活动，则实施这些活动的人无权将其保密。
  </p>
  <p class="graf graf--p">
   ⚠️赞成这种说法的人可能会说 “我没有什么可隐藏的”，以此来表示自己支持政府的监视，声明自己是 “好人”。
  </p>
  <p class="graf graf--p">
   于是持这种观点的人声称 “只要你没什么可隐藏的，就不必担心政府的监视”。
  </p>
  <ul class="postList">
   <li class="graf graf--li graf--startsWithDoubleQuote">
    “Nothing to Hide: The False Tradeoff between Privacy and Security” by Daniel J. Solove
   </li>
  </ul>
  <p class="graf graf--p">
   在英国推行闭路电视监视系统的时候，政府就以这句话作为座右铭：“如果你没有什么可隐瞒的，你就不必害怕”。
  </p>
  <p class="graf graf--p">
   Upton Sinclair 在他1919年的《黄铜支票》中写道（指1914年列克星敦大道爆炸案）：
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    从头到尾我什么都没有隐藏，因此我什么都不怕，这对报纸来说是众所周知的，对爆炸案进行调查的警察也是如此。
   </p>
  </blockquote>
  <ul class="postList">
   <li class="graf graf--li">
    《The Brass Check》Sinclair, Upton (1919). p. 194.
   </li>
  </ul>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这一愚蠢的观点从来不是新鲜事
   </strong>
  </h3>
  <p class="graf graf--p">
   该论点通常用于有关隐私的讨论中。法律学者 Geoffrey Stone 说，这种说法的使用 “太普遍了”
  </p>
  <p class="graf graf--p">
   数据安全专家和密码学家 Bruce Schneier 将其描述为“针对隐私倡导者的最常见的反驳”。
  </p>
  <p class="graf graf--p">
   The Privacy Advocates 的作者 Colin J. Bennett 说，保护隐私的倡导者常常 “不得不不断驳斥” 这一论点。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《The privacy advocates: resisting the spread of surveillance》by Colin J. Bennett
   </li>
  </ul>
  <p class="graf graf--p">
   Bennett 解释说，
   <strong class="markup--strong markup--p-strong">
    尽管 “有充分的证据表明，所有人都在遭受监视，但人们经常认为监视过程不是针对自己，而是针对那些不法分子”
   </strong>
   ，认为 “监视机制是针对其他人的”。
  </p>
  <p class="graf graf--p">
   这一论点 “已经在隐私文献中得到了充分的记载，成为发展实用的隐私保护策略的绊脚石，它也与 ‘隐私’ 一词本身的含糊性和象征性有关。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    隐私是一个抽象的概念，人们只有在隐私消失后才开始关注它
   </strong>
   。就如人们知道臭氧消耗和全球变暖是不利的发展，但是与 “驾驶汽车的直接便利” 相比，前者就变得无所谓了。面对隐私问题时也是类似逻辑”。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    心理学角度《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e4%ba%ba%e4%bb%ac%e5%af%b9%e5%9c%a8%e7%ba%bf%e9%9a%90%e7%a7%81%e5%a6%82%e6%ad%a4%e8%99%9a%e4%bc%aa%ef%bc%9f/" href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e4%ba%ba%e4%bb%ac%e5%af%b9%e5%9c%a8%e7%ba%bf%e9%9a%90%e7%a7%81%e5%a6%82%e6%ad%a4%e8%99%9a%e4%bc%aa%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     为什么人们对在线隐私如此虚伪？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   由 Ana Viseu、Andrew Clement 和 Jane Aspinal 进行的一项关于将在线服务整合到日常生活中的民族志研究于2004年在《信息、通信与社会》杂志上发表，题为 “在线隐私的处境：复杂的感知和日常实践”。
  </p>
  <p class="graf graf--p">
   调查发现，充分就业的中上中等收入者表达了类似的信念，即 他们不应该成为监视目标；而那些没有表示担忧的其他受访者（低收入人群）则认为，“自己没有做错任何事，或者没有什么可隐藏的。
  </p>
  <p class="graf graf--p">
   在参与者样本中只有一位参与者报告说使用了隐私增强技术。Viseu 说：“受试者对隐私的感知和实践最明显的特征之一就是他们对问题的消极态度。被动性源于 “没什么可隐藏的” 论点。
  </p>
  <p class="graf graf--p">
   在2003年左右对英国政府进行的定性研究中，Perri 博士提出了四种隐私风险观点，并映射了八个框架以显示隐私态度分布。根据这项研究，被调查者最初使用 “没什么可隐藏” 的观点，然后转向认为监视是一种令人讨厌的事情，而不是威胁。
  </p>
  <h3 class="graf graf--p">

  </p>

  <p class="graf graf--p">
   在讨论 MAINWAY 计划时，前美国参议院多数党领袖 Trent Lott 说：“人们担心什么？有什么问题？难道你在做不应该做的事吗？”
  </p>
  <ul class="postList">
   <li class="graf graf--li graf--startsWithDoubleQuote">
    “
    <a class="markup--anchor markup--li-anchor" data-href="http://www.cnn.com/2006/POLITICS/05/15/bellsouth.nsa/" href="https://www.cnn.com/2006/POLITICS/05/15/bellsouth.nsa/" rel="noopener noreferrer" target="_blank">
     BellSouth denies giving records to NSA

    ”

  <p class="graf graf--p">
   英国作家 Johann Hari 辩称，“没什么可隐藏” 的论点与在公共场所布满闭路电视摄像机的措施无关，因为摄像机安置在公共场所，与而不是在 “你可以隐藏的地方”。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <a class="markup--anchor markup--li-anchor" data-href="https://www.independent.co.uk/voices/commentators/johann-hari/johann-hari-this-strange-backlash-against-cctv-796807.html" href="https://www.independent.co.uk/voices/commentators/johann-hari/johann-hari-this-strange-backlash-against-cctv-796807.html" rel="nofollow noopener noreferrer" target="_blank">

   </strong>
   ，措辞为：“如果你没有什么可隐藏的，那就没有什么可担心的”。
  </p>
  <p class="graf graf--p">
   前保守党外交大臣 William Hague 在2013年
   <strong class="markup--strong markup--p-strong">

  </p>

  <p class="graf graf--p">
   ⚠️如上，你看到，所有支持者都是寡头和政客。他们推动了恶意的愚蠢。
  </p>

   <strong class="markup--strong markup--p-strong">
    反对者 ——
   </strong>
  </p>
  <p class="graf graf--p">
   Edward Snowden ：“争辩说您不关心隐私权是因为您没有什么可隐藏的，就如同说您不在乎言论自由是因为您无话可说。”
  </p>

   <li class="graf graf--li graf--startsWithDoubleQuote">

  <p class="graf graf--p graf--startsWithDoubleQuote">
   “当您说 ‘我无话可说’ 时，您是在说 ‘我不在乎这项权利’。权利的运作方式是，政府必须证明侵犯您权利的理由。”（现在你帮他们证明了）
  </p>
  <p class="graf graf--p">
   Daniel J. Solove 在《高等教育纪事》的一篇文章中说，他反对这一观点。他说，政府可能泄漏有关某人的信息并对该人造成严重伤害，或者即使某人实际上并没有从事不正当行为，也可能受到惩罚，政府的任何失误都能毁掉一个人的生活。
  </p>
  <p class="graf graf--p">
   Solove 写道：“没什么可隐藏的观点可能会令人反感，因为它迫使辩论集中于对隐私的极为狭隘的理解上。但是，当面对政府数据收集和使用所带来的众多隐私问题而不仅仅是监视和泄露时，没什么可隐藏的观点最终无话可说。”
  </p>
  <p class="graf graf--p">
   社交媒体研究人员 danah boyd（小写的名字）也反对该观点，他指出，
   <strong class="markup--strong markup--p-strong">
    “人们常常没能感受到遭遇国家监视的危险，只因为他们太循规蹈矩令当权者满意了”，对你了解越多的人越容易损害你的声誉、诽谤和操纵你
   </strong>
   。
  </p>

  </ul>
  <p class="graf graf--p">
   《Privacy Rights: Moral and Legal Foundations》的作者 Adam D. Moore 说，“这种观点认为人权不如成本/利益甚至后果。我们必须拒绝这样一种观点，即 认为隐私利益是可以为了安全而进行交易的事物。”
  </p>
  <p class="graf graf--p">
   他还表示，监视可以根据外观，种族和宗教对社会中的某些群体产生不成比例的影响。
  </p>
  <p class="graf graf--p">
   Moore 认为，“没什么可隐藏” 的论点至少还有其他三个问题。首先，如果个人拥有隐私权，那么使用 “没什么可隐藏的” 观点就是无效的。隐私被理解为控制对空间、位置和个人信息的访问和使用的权利，这意味着由谁来确定访问权限。
  </p>
  <p class="graf graf--p">
   为了说明这一点，Moore 提供了以下情况：“想象一下，有一天你离开家时，发现有人在垃圾桶中搜寻，将那些撕碎的便条和文件重新拼在一起。为应对您的震惊，他宣布 ‘您不必担心，没什么可隐藏的对吗？”
  </p>
  <p class="graf graf--p">
   其次，个人可能希望掩盖主流文化不接受的尴尬行为或举止。 “考虑某人的性病或病史。想象有人到图书馆去学习大多数人不接受的另类生活方式。”
  </p>
  <p class="graf graf--p">
   如果 “没什么可隐藏” 的观点合理，那么国家安全局的特工、政客、警察局长和首席执行官，也没什么可隐藏的，他们应该像我们其他人一样完全透明。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “但是正相反，他们没有，而且这些政客、警察局长或首席执行官几乎总是认为他们自己去偷窥别人是一件好事。”
  </p>
  <p class="graf graf--p">
   计算机安全专家和密码学家 Bruce Schneier 引用红衣教主的名言说：“如果给我六句由最诚实的人写的文字，我肯定能从中发现绞死他的理由”，指的是
   <strong class="markup--strong markup--p-strong">
    国家政府可以在一个人的生活中找到任何方面以起诉或勒索该人
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️ “太多人错误地将辩论固定在安全和隐私之间的权衡上。但事实上，真正的选择应该是自由还是成为奴隶。”
  </p>
  <p class="graf graf--p">
   哲学家和心理分析家 Emilio Mordini 认为，“没什么可隐藏” 的观点本质上是自相矛盾的。人们不需要具有 “可隐藏的东西” 即可隐藏 “某些东西”。重要的不是隐藏什么东西，而是应该存在一个可能用来隐藏的私密区域的经验。
   <strong class="markup--strong markup--p-strong">
    从心理学上讲，我们之所以成为 *个人* 是通过我们发现自己可以向他人隐藏某些东西。
   </strong>
  </p>
  <p class="graf graf--p">
   Julian Assange 指出：“There is no killer answer yet”。Jacob Appelbaum 做出了明智的回应，要求持有这种观点的人将手机解锁并脱下裤子。
  </p>
  <p class="graf graf--p">
   ⚠️“从哲学上讲，真正的答案是：大规模监视是一种大规模的结构性变化。当社会陷入困境时，即使你是地球上最卑鄙的人，也会深陷其中。”
  </p>
  <ul class="postList">
   <li class="graf graf--li graf--startsWithDoubleQuote">
    <a class="markup--anchor markup--li-anchor" data-href="http://wikileaksetc.blogspot.nl/2015/04/courage-foundation-reddit-ama.html" href="https://wikileaksetc.blogspot.nl/2015/04/courage-foundation-reddit-ama.html" rel="nofollow noopener noreferrer" target="_blank">
     “Courage Foundation: Reddit AMA”
    </a>
   </li>
  </ul>
  <p class="graf graf--p">
   Harvey A. Silverglate 估计，普通人平均每天可以在完全不知情的情况下犯下三项重罪。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    Three Felonies a Day: How the Feds Target the Innocent. Encounter Books. 2011. ISBN 9781594032554
   </li>
  </ul>

   关于监视究竟是什么，我们曾经发布的

    “
   </strong>

  </p>
  <p class="graf graf--p">
   现在您至少明白了，这一流行全球的愚蠢论调究竟是怎么来的；尤其是，为什么它必须被抵制，以及如何抵制。⚪️
