   所有需要电话号码注册和验证的在线应用都是政府间谍那种追踪者的盛宴。
  </p>
  <p class="graf graf--p">
   Facebook 确实想要您的电话号码，你一加入，它就会提醒你这点。理论上这不是一件坏事，因为可以通过两步验证来帮助保护您的帐户。但是，
  </p>

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/facebook-%e4%bb%a5%e5%ae%89%e5%85%a8%e6%80%a7%e4%b8%ba%e5%b9%8c%e5%ad%90%e9%aa%97%e5%8f%96%e4%bd%a0%e7%9a%84%e6%89%8b%e6%9c%ba%e5%8f%b7%e7%a0%81/" href="https://www.iyouport.org/facebook-%e4%bb%a5%e5%ae%89%e5%85%a8%e6%80%a7%e4%b8%ba%e5%b9%8c%e5%ad%90%e9%aa%97%e5%8f%96%e4%bd%a0%e7%9a%84%e6%89%8b%e6%9c%ba%e5%8f%b7%e7%a0%81/" rel="noopener noreferrer" target="_blank">
     <strong class="markup--strong markup--li-strong">

    </strong>
   </li>

  <p class="graf graf--p">
   另一方面，这使得揭露 Facebook 上几乎所有人的私人电话号码的侦查追踪变得非常容易，包括名人和政客。
  </p>
  <p class="graf graf--p">
   本文将研究如何做到这点，以及您应该如何保护自己。

  <p class="graf graf--p">
   许多 Facebook 用户已经忘记了自己的私人电话号码，甚至没有意识到他们的私人电话号码已连接到 Facebook 帐户。
  </p>
  <p class="graf graf--p">
   Facebook 不允许简单地从手机中提取您的电话号码，但是他们可以在每次启动 Facebook 时反复要求您确认并保存您的电话号码，从而实现本文想要强调的：“相当于网络欺凌的应用程序”。
  </p>
  <p class="graf graf--p">
   Facebook 上的默认隐私设置允许任何人在您添加电话号码后通过您的电话号码对您进行搜索。这不是一个新问题。
  </p>

   当然，名人和政客比其他人更注重在线隐私。但是，
   <strong class="markup--strong markup--p-strong">
    任何人都可能成为网络跟踪者或黑客为目标。
   </strong>
  </p>
  <p class="graf graf--p">
   一旦追踪者有了电话号码和您的名字，他们便可以快速使用
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    开源情报 工具
   </a>
   来获取更多的数据，例如职业、雇主、配偶、亲戚、以及任何其他被公开的信息。
  </p>
  <p class="graf graf--p">
   追踪者还可以通过直接致电您以利用这些信息进一步进行
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
    社交工程攻击
   </a>
   。甚至盗用你的身份让你背锅。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     追踪者实际上将如何找到您的电话号码？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   从理论上讲，如果他们有足够多时间的话可以直接搜索所有9,999,999,999个潜在数字，直到偶然发现您的。显然，这不是很有效，所以让我们来看看正确的方法。
  </p>
  <p class="graf graf--p">
   对于实践主题，本文将使用DC前市长 Muriel Bowser 作为案例。在以下示例中编辑了她的电话号码以保护其真实电话号码。主要为展示追踪者的方式。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     第一步：使用区号
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   如果您将目标的电话号码视为所有可能的10位美国电话号码之一，则可以很快看到100亿个北美电话号码，它的列表太大了，无法有效地进行搜索。
  </p>
  <p class="graf graf--p">
   幸运的是，由于有了
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/North_American_Numbering_Plan" href="https://en.wikipedia.org/wiki/North_American_Numbering_Plan" rel="noopener noreferrer" target="_blank">
    北美编号计划（NANP）
   </a>
   ，该计划为美国的电话号码制定了指南，因此可以避免这一繁琐。
  </p>
  <p class="graf graf--p">
   举个例子：234–235–5678。看一下 NANP，可以看到前三个数字（234）是区号，并且该计划允许第一个数字为 2–9，第二个和第三个数字为 0–9。仅此信息就消除了攻击名单中十亿个可能的数字。
  </p>
  <p class="graf graf--p">
   如果追踪者知道或可以对您的住所进行有根据的猜测，那么他们也可以快速利用此功能，因为这很容易进行 Google 搜索。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11135 jetpack-lazy-image" data-lazy-sizes="(max-width: 670px) 100vw, 670px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-1.jpg?resize=670%2C373&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-1.jpg?w=670&amp;ssl=1 670w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-1.jpg?resize=300%2C167&amp;ssl=1 300w" data-recalc-dims="1" height="373" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-1.jpg?resize=670%2C373&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="670"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11135" data-recalc-dims="1" height="373" sizes="(max-width: 670px) 100vw, 670px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-1.jpg?resize=670%2C373&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-1.jpg?w=670&amp;ssl=1 670w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-1.jpg?resize=300%2C167&amp;ssl=1 300w" width="670"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在本示例（235）中，区号后面的三个数字是中心局号。同样，该计划要求第一个数字为2–9，第二个和第三个数字为0–9，但有一个警告。
  </p>
  <p class="graf graf--p">
   在第二个数字为1的区号中，第三个数字不能也是1。这再次从可能列表中删除了大量的电话号码。电话号码的最后四位数字是线路号，在这种情况下为5678。
  </p>
  <p class="graf graf--p">
   推测是DC的市长会有DC的区号，并且追踪者还可以查找目标的 Facebook 帐户，并可能找到目标居住或工作的家乡或当前城市。
  </p>
  <p class="graf graf--p">
   一些较大的城市（例如洛杉矶）将在其中包含多个区号，但是无论其中有多少个“拆分”区号，它都仍然大大减少了可能的号码。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     第二步：获取最后的数字
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   既然已经知道了目标号码是 202-???-????，那么就要尝试删除尽可能多的问号，以便可以更轻松地进行 Facebook 搜索。
  </p>
  <p class="graf graf--p">
   值得庆幸的是，在使用区号之后 Facebook 就支持了追踪，这已成为第二个最简单的步骤 —— 为了获得最后两位数字，只需要在密码重置过程中走几步即可。
  </p>

   <li class="graf graf--li">

   为此，追踪者转到 Facebook 主页，然后单击 “忘记帐户” 以开始该过程。
  </p>

   </noscript>
  </p>
  <p class="graf graf--p">
   接下来，追踪者输入要记住的目标名字，然后单击“搜索”按钮。

   <span style="color: #ff6600;">

     第四步：暴力破解
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在这一点上，追踪者完全可能直接将号码扔到 Facebook 搜索栏中，但这仍然没有那么高效。
  </p>
  <p class="graf graf--p">
   那么，一个懒惰的追踪者会做些什么呢？他们会利用 Facebook 的功能，可让您进行
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Bisection_method" href="https://en.wikipedia.org/wiki/Bisection_method" rel="noopener noreferrer" target="_blank">
    括号搜索
   </a>
   。
  </p>
  <p class="graf graf--p">
   Facebook 允许您以CSV格式上传联系人列表，然后告诉您联系人是否在 Facebook 上，以便您可以将其添加为朋友。于是，
   <strong class="markup--strong markup--p-strong">
    通过构建自己的潜在号码联系人列表，追踪者可以快速排除大量错误号码
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在这种情况下，已经知道这个号码必须在 202–000–6969 到 202–999–6969 之间。通过将其切成两半并创建一个从 202–000–6969 到 202–500–6969 的数字列表，可以有效地排除列表的一半，因为目标将仅位于创建的两个列表之一中。然后就可以上传列表，并立即确定它们是否在列表中。
  </p>
  <p class="graf graf--p">
   要创建此列表，转到
   <a class="markup--anchor markup--p-anchor" data-href="https://contacts.google.com/" href="https://contacts.google.com/" rel="noopener noreferrer" target="_blank">
