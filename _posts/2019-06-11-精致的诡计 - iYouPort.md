      “进攻性信息战”是什么？主流媒体称国安局关闭了曾经震惊世界的电话监听计划，但这究竟是不是真的？
     </em>

   </li>

  <p class="graf graf--p graf--startsWithDoubleQuote">
   “纽约时报”称，国家安全局已“悄悄”关闭了斯诺登在2013年揭露的电话记录监控计划。很多人都在庆祝这件事。但是，这是真的吗？
  </p>
  <p class="graf graf--p">
   当美国国家安全局局长 Paul Nakasone 拒绝证实或否认这个报道时，人们对纽约时报产生了进一步的怀疑。Nakasone 说，该机构仍在“审议过程中”，关于是否要使用经过改进的电话监听数据库。
  </p>
  <p class="graf graf--p">
   然而，所有这些都是一个精心设计的诡计。根据英国“金融时报”采访的两位前国家安全局高级官员的说法，没有任何可信的理由相信国家安全局的电话监控真的被关闭了。
  </p>
  <p class="graf graf--p">
   当纽约时报的报道引起舆论爆炸时，我联系了两名前NSA高级官员，Russ Tice 和 Thomas Drake，以了解他们的想法。
  </p>
  <p class="graf graf--p">
   他们两人告诉我，
   <strong class="markup--strong markup--p-strong">
    美国国家安全局关闭这一特定计划并不意味着结束对全民电话的监控，恰恰相反 — — 该监视计划已被优质技术所取代
   </strong>
   。
  </p>
  <p class="graf graf--p">
   曾任美国空军海军情报和国防情报局办公室负责人的前高级情报分析师 Russ Tice 表示，美国国家安全局正在关闭电话监控的最新说法很难令人信服，“为什么会有人信这种话呢？”Tice 也是2005年5月首位暴露非法监视和窃听美国公民的国家安全局举报人。
  </p>
  <p class="graf graf--p">
   他说：
   <strong class="markup--strong markup--p-strong">
    “他们过去曾多次撒谎，现在他们很可能继续撒谎。他们一直在收集元数据和内容，逐字逐句，包括语音和文字。
   </strong>
   “
  </p>
  <p class="graf graf--p">
   我问 Tice 他有多肯定国家安全局仍在对全民进行电话监控。 “当然，NSA 仍在进行针对电话和计算机通信的监控，是的，’更广泛的计划’继续进行，一个更有效的新计划可能已经实施了，”他说。当前老计划退位的真正原因是现在有了更好的技术可用于更先进的监控。
  </p>
  <p class="graf graf--p">
   美国国家安全局的前高级主管 Thomas Drake 曾在斯诺登揭露的前几年在该机构内严重批评非法的大规模监视，浪费资源和管理不善，他表示，国家安全局的电话监视“肯定会以其他形式继续”。
  </p>
  <p class="graf graf--p">
   2015年的“美国自由法案”即将到期，但 Drake 表示，国家安全局根本不需要“自由法案”来授权继续对美国公民进行无证电话监控。他说，监控仍然可以继续进行，使用其他特殊的、紧急的计划 — 包括第702条和第12333号行政命令 — 隐藏在所谓的机密中。
  </p>
  <p class="graf graf--p">
   Drake 承认，国家安全局也正在转向适应广泛的技术使用的变化，这很可能使旧的监视计划不再那么有用。“使用传统的固定电话和手机的通信如今越来越多地发生在其他消息应用程序上”。
  </p>
  <p class="graf graf--p">
   在关于这个问题的一系列推文中，Drake 警告说，美国国家安全局最近的历史几乎没有证明该机构的说法：“所有这些国内间谍程序从根本上违反了宪法、FISA和第4修正案，然后在秘密FISC 批准的掩护下，通过违宪的事后法律，使其“合法”。“
  </p>
  <p class="graf graf--p">
   Tice 和 Drake 的观点证实了 The Register 的一项独立分析：如果国家安全局提出自愿放弃其电话元数据收集，这为该机构提供了几个机会。首先，它没有必要解释秘密法律依据是什么，因此可以继续使用这种方法；其次，它可以重复与2015年相同的壮举 — 让国会支持所谓的“安全”；第三，它可以继续完成它正在做的，同时向所有其他人宣称监视“缩减”。
  </p>
  <p class="graf graf--p">
   根据Tice的说法，非法的NSA监视是针对联邦政府的，目的是确保政治杠杆。他将这种监视的使用描述为
   <strong class="markup--strong markup--p-strong">
    一种反民主的工具，允许该机构向众议院和参议院的当选领导人施加压力。这就是为什么美国人称情报部门为隐形政府
   </strong>
   ，关于这点详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/sc-b6b87c0487d0" href="https://medium.com/@iyouport/sc-b6b87c0487d0" rel="noopener noreferrer" target="_blank">
    国王们的晚宴
   </a>
   》。
  </p>
  <p class="graf graf--p">
   他说，在2002年至2005年期间，他发现美国国家安全局一直针对性监视国会、FISA、最高法院、五角大楼高级官员、媒体、甚至未来总统候选人的所有通讯。
  </p>
  <p class="graf graf--p">
   但美国国家安全局的监视范围远远超出了这一范围，至今仍未公开。
  </p>
  <p class="graf graf--p">
   Tice说，由于他在国家安全局的角色，他意识到一个高度机密的活动计划，以前从未透露过，涉及“
   <strong class="markup--strong markup--p-strong">
    特殊访问计划（SAP）进攻性信息战
   </strong>
   ”。
  </p>
  <p class="graf graf--p">
   SAP 就是 Tice 所描述的“黑色世界”，这是一个完全不负责任的项目领域，用于开发具有可怕的触及范围和功率的技术。
   <strong class="markup--strong markup--p-strong">
    “这些计划远远超出了收集通信的范围 — 它们可以触及你，并造成巨大的，甚至是严重的、有形的伤害。
   </strong>
   ”
  </p>
  <p class="graf graf--p">
   当我让 Tice 提供有关这些进攻性信息战能力的进一步细节时，他拒绝详细说明。
  </p>
  <p class="graf graf--p">
   但他确实强调，
   <strong class="markup--strong markup--p-strong">
    斯诺登对大规模监视的揭露只代表了国家安全局一直在做的行为的冰山一角。他表示还存在一个完全黑暗的秘密计划世界，主要由美国国防部失踪的数万亿美元资助
   </strong>
   。
  </p>
  <p class="graf graf--p">
   有三种不同形式的 SAP，Tice 说：已承认、未承认和“幽灵”黑色程式。在国防和情报委员会中国会和参议院的代表都知道“已承认”的计划。幽灵计划国会不知道，“这些计划中有许多涉及极其昂贵的尖端技术。我是这其中的高级技术分析师之一。“
  </p>
  <p class="graf graf--p">
   在 Tice 任职期间，这些计划仅在美国境外运作。 “真正的问题是，
   <strong class="markup--strong markup--p-strong">
    美国国家安全局现在是否已经开始让美国公民接受其强大的黑色世界进攻性信息战能力？
   </strong>
   当国防部武器在国内对抗我们所有人时，没人能对老大哥行使追索权。“
  </p>
  <p class="graf graf--p">
   十二年前，Tice 在众议院作证。他告诉委员会，尽管美国国家安全局的特殊访问计划可能违反了数百万美国公民的宪法权利，但委员会成员甚至国家安全局检查员都没有必要的安全许可而无法了解这些计划，更不用说监督了。
  </p>
  <p class="graf graf--p">
   一周后，Tice 计划在参议院军事委员会面前就 NSA 监视的“不同角度”作证，但他被制止了。 Tice 说，当时五角大楼和国家安全局“告诉国会他们无法与我交谈，因为他们没有适当的安全许可。”
  </p>
  <p class="graf graf--p">
   简而言之：
   <strong class="markup--strong markup--p-strong">
    不要相信主流媒体宣传的国家安全局正在关闭其大规模监视计划的内容。离真相还很远。该机构只是在不断发展和重新校准，而且像往常一样，这些主流媒体不会对国会和政府提出迫切需要提出的尴尬问题
   </strong>
   。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    W
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.blacklistednews.com/article/71585/whistleblowers-say-nsa-still-spies-on-american-phones-in-hidden.html" href="https://www.blacklistednews.com/article/71585/whistleblowers-say-nsa-still-spies-on-american-phones-in-hidden.html" rel="noopener noreferrer" target="_blank">
    HISTLEBLOWERS SAY NSA STILL SPIES ON AMERICAN PHONES IN HIDDEN PROGRAM
   </a>
   . the NSA’s shut down of this particular program did not imply an end to domestic phone record surveillance, but quite the opposite — that the program had been superseded by superior technology.
  </p>
