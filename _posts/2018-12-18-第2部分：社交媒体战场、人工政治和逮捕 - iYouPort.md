  <h4 class="graf graf--h4 graf-after--figure" id="f996">
   – VI — 社交媒体大战：巨魔游戏
  </h4>
  <p class="graf graf--p graf-after--h4" id="bda2">
   互联网及其架构的最初概念向我们提出了一种分散化和民主化的可能性，即每个人都是一个媒介。但 50 年后的今天，这一梦想已经彻底破灭。实际上，基础设施和服务变得高度集中，由互联网服务提供商和巨头互联网公司（如谷歌或 Facebook）控制。在这里阅读 IYP 的警告：《
   <a class="markup--anchor markup--p-anchor" data-href="https://steemit.com/internet/@iyouport/7jsklp" href="https://steemit.com/internet/@iyouport/7jsklp" rel="noopener nofollow noreferrer" target="_blank">
    大型“云”供应商时代的审查制度：集中化毁掉互联网自由
   </a>
   》。
  </p>
  <p class="graf graf--p graf-after--p" id="716b">
   是的，我们仍然有机会成为个人媒体，但在大多数情况下，我们只能在更大的社交和媒体结构中存在，由其他人拥有和控制。但是，即使在这种情况下，政府或政治行为者对媒体的控制也要比20年前有难度，因为他们需要控制的节点不再高度集中在几家国家电视台和报纸上。从这个意义上讲，
   <strong class="markup--strong markup--p-strong">
    这是对个别节点（人）及其社交图谱的统治战
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="0819">
   <strong class="markup--strong markup--p-strong">
    通过工具化和征服单个节点，他们能够干扰和影响人们的社交图谱
   </strong>
   （参见：
   <a class="markup--anchor markup--p-anchor" data-href="https://labs.rs/en/facebook-algorithmic-factory-human-data-banks-and-algorithmic-labour/" href="https://labs.rs/en/facebook-algorithmic-factory-human-data-banks-and-algorithmic-labour/" rel="noopener nofollow noreferrer" target="_blank">
    Human Data Banks and Algorithmic Labor
   </a>
   ，SHARE Labs 2016）这一图谱由人们的社交圈子 — 数百名朋友、同事和亲戚组成。 也就是说，通过代理征服他人的信息流。
  </p>
  <p class="graf graf--p graf-after--p" id="3ec0">
   <strong class="markup--strong markup--p-strong">
    社交网络生态系统是针对反对者的不同形式的虚假信息或诽谤运动的肥沃土壤，或仅仅是一种啦啦队活动，就像中国那样（详见上一篇文章），这取决于政治战争的风格
   </strong>
   。在这样的环境中，政治宣传（为了加速伤害机构、事业或个人，而传播符合当权者利益的思想、信息或谣言），可以通过匿名或微小的单个节点执行，将他们的现实身份直接连接到政党。
  </p>

  <p class="graf graf--p graf-after--p" id="8d07">
   在查看显示的图表时，应该记住一些事项。网络中的节点是人/实体（组织），边缘是它们之间的交互：
  </p>
  <ol class="postList">
   <li class="graf graf--li graf-after--p" id="a016">
    节点的大小反映了特定节点如何进入的，交互量（推文、回复）是传入和传出交互的总和；
   </li>
   <li class="graf graf--li graf-after--li" id="183b">
    根据他们所代表的政治结构，我们为政治组织的最大节点着色（因此用相同颜色的不太密集的变体着色他们在网络中的
    <span class="markup--quote markup--li-quote is-other" data-creator-ids="beb5bd62510c">
     近邻）。
    </span>
   </li>
  </ol>
  <p class="graf graf--p graf-after--li" id="0254">
   这张图表在很多方面反映了塞尔维亚社会、或者至少是塞尔维亚 Twitter 社区的现实。它表现为强烈的两极分化，执政党在一边（蓝色节点），其余的在另一边 — —
   <strong class="markup--strong markup--p-strong">
    是的，这就是为什么要以此为案例，因为与中国极其相似
   </strong>
   ；有几个节点在边缘，但并不意味着它们是中性的，而是，它们有一个自己的微观宇宙，它们在其中运行。
  </p>
  <p class="graf graf--p graf-after--p" id="6844">
   在塞尔维亚的政治言论中，有一段时间内，“我们”与“他们”的语话基调非常明显的存在 ——
   <strong class="markup--strong markup--p-strong">
    是的，再次和中国一样
   </strong>
   。这是由于自我受害和民粹主义的必要性成为了为现代塞尔维亚政治家最强大的特征之一，2016 年竞选活动的总体基调方面，“对抗对手“比“对自己的计划和承诺“的肯定更为卖力。
  </p>
  <p class="graf graf--p graf-after--p" id="3be2">
   我们可以在此图上轻松发现
   <strong class="markup--strong markup--p-strong">
    两种不同类型的巨魔领主活动。
   </strong>
  </p>
  <ol class="postList">
   <li class="graf graf--li graf-after--p" id="4d27">
    <strong class="markup--strong markup--li-strong">
     攻击者
    </strong>
    ：第一个位于图表的右侧，在“蓝色团队”（执政党）的核心，有一个大的淡黄色节点（@zeljkocenej）。这个用户被其他蓝队节点连接（转发、点赞），但在他的推文中，他经常攻击黄色团队（@demokrate），这就是为什么 SNA 算法给他一个淡黄色的原因（
    <strong class="markup--strong markup--li-strong">
     因为他们之间有明显的连接
    </strong>
    ）。根据这张图可以确定，他是最强大的巨魔领主，是蓝队宣传机器的引擎之一。
   </li>
   <li class="graf graf--li graf-after--li" id="c3c9">
    <strong class="markup--strong markup--li-strong">
     Bridge
    </strong>
    ：另一个有趣的例子可以在图的中下部分看到。账户 @Kika_Bibic 是一座桥梁，对反对党官方节点上的各种蓝队攻击进行了推动（转发和回复），与反对派节点和媒体账户相关联的个人账户（例如 @ n1infobg）
   </li>
  </ol>
  <p class="graf graf--p graf-after--li" id="bf67">
   <strong class="markup--strong markup--p-strong">
    Twitter 上的人工政治
   </strong>
   HASH
   <strong class="markup--strong markup--p-strong">
    案例
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="3094">
   除了真正的人工巨魔领主之外，通过使用社交网络分析，我们还可以发现试图参与信息战的原始人工行为者的痕迹。
  </p>
  <p class="graf graf--p graf-after--p" id="8df8">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wix.com/dashboard/6fc0666c-c125-4701-93b5-303f8db9aa33/blog/5be81d5c7ce1d7001aa0c8ca/edit/search/posts?query=%23slozno" href="https://www.wix.com/dashboard/6fc0666c-c125-4701-93b5-303f8db9aa33/blog/5be81d5c7ce1d7001aa0c8ca/edit/search/posts?query=%23slozno" rel="nofollow noopener noreferrer" target="_blank">
    #slozno
   </a>
   （“unanimous”）标签的情况，这是选举名单“SložnozaSrbiju”（“塞尔维亚的一致”）的竞选口号。该列表在社交媒体上没有得到公众的好评，尤其是在 Twitter 上，即使是列表中的成员本身也不够活跃，也没有社区讨论他们的建议或计划（在活动的背景下）。然而，有一条推文使用了这个标签，显然是个假帐户叫“@najludja”（在发布有关推文前几个小时被创建）。
  </p>
  <p class="graf graf--p graf-after--p" id="6407">
   这条推文被一些通常不参与竞选活动的账户转发了一千多次，这个数字甚至在整个塞尔维亚都是非常惊人的。
   <strong class="markup--strong markup--p-strong">
    这是一个“外包”支持者的例子
   </strong>
   ，它完全可以作为一个测试案例，对于那些想要在选举过程中在线看到人工认知营销效果的人来说，非常有用。之后很快推文就被删除了，但它已经被我们用于实时数据收集的软件抓取。
  </p>

   Facebook 水族馆内的政治泡沫
  </h4>
  <p class="graf graf--p graf-after--h4" id="fc8f">
   世界上最大的社交网络，在我们之前的研究中得到了深刻的解剖，其中有超过一半的塞尔维亚公民，这里很可能是信息战中最大的生态系统之一，相当于中国的微博。无意深入探讨 Facebook 的主体是非活跃的代理人、非中立的玩家，有能力塑造、交替或审查政治讨论，我们要说的重点是，
   <strong class="markup--strong markup--p-strong">
    Facebook 背后的界面、结构和算法正在决定这场信息战的规则
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="673d">
   Eli Pariser 在他的著作“过滤泡泡”中指出，用户很难接触到冲突性观点，并且在 Facebook 自己的信息泡沫中被算法隔离，算法根据用户信息选​​择性地猜测用户想要看到的内容 — — ⚠️
   <strong class="markup--strong markup--p-strong">
    请注意，这并不仅仅是在迎合个人的偏好，更多还有 Facebook 这些阴暗的巨头希望人们的认知变成什么样！这才是最大的威胁。也就是我们在《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://pao-pao.net/article/1023" href="https://pao-pao.net/article/1023" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     定制人将终结民主
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》一文中主要阐述的
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="418e">
   这种现象也被称为“回声室” ，是对一种情形的隐喻描述，即 其中信息、思想或信仰通过“封闭”系统内的传播和重复，而被放大或加强，其中不同的或相互竞争的观点被审查、禁止或否认，以至于明显的代表性不足。
  </p>
  <p class="graf graf--p graf-after--p" id="4dfa">
   <span class="markup--quote markup--p-quote is-other" data-creator-ids="4a91706e921f">
    换句话说，这创造了一个高度政治化的空间，思想市场已经不复存在，并且与此同时，由用户部分算法管理的泡沫很容易成为具有相同政治背景的人只是在彼此之间进行的讨论，而不会将其暴露于其他政治观点中，不会形成辩论和交锋。 是的，
    <strong class="markup--strong markup--p-strong">
     这就是为什么中国这类追求“大一统”的国家会非常热衷于 FATBAG（Facebook Amazon Tencent Baidu Alibaba Google）这个中美均分的数字极权黑手整体在此完全一致，只有它们才能提供政府期待的“和谐”
    </strong>
    。
   </span>
  </p>

   “他们已经到了一个没有人敢说出自己的想法的时候，当凶狠咆哮的狗到处游荡，当你不得不看到你的同志在承认令人震惊的罪行后被撕成了碎片。” — — 《动物农场》乔治奥威尔
  </p>
  <p class="graf graf--p graf-after--p" id="313a">

  </p>
  <p class="graf graf--p graf-after--p" id="7b17">
   关于法律规定的迷糊和含混是否有意或无意都不重要了，它已经被证明是实现在线噤声的便利工具。尽管它们的状态是在私人社交媒体账户上发布的，并且没有引起实际的恐慌，但在被拘留者中，有一个18岁的男人，一个两个孩子的父亲，一个时尚化妆师……总而言之，第一波逮捕
   <a class="markup--anchor markup--p-anchor" data-href="http://www.blic.rs/vesti/tema-dana/na-saslusanje-zbog-fejsbuka/swjln7k" href="http://www.blic.rs/vesti/tema-dana/na-saslusanje-zbog-fejsbuka/swjln7k" rel="noopener nofollow noreferrer" target="_blank">
    至少有 15 起案件涉及贝尔格莱德地区
   </a>
   。在接下来的几个月里，有报道称，该国境内的城镇和小型社区的一些当地在线和广播记者也受到了威胁，这些威胁都与洪水有关。
  </p>
  <p class="graf graf--p graf-after--p" id="3af8">
   到同年秋天，调查报告中心公布了被
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cins.rs/srpski/research_stories/article/ispovesti-uhapsenih-zbog-komentara-na-fejsbuku" href="https://www.cins.rs/srpski/research_stories/article/ispovesti-uhapsenih-zbog-komentara-na-fejsbuku" rel="noopener nofollow noreferrer" target="_blank">
    拘留的人的证词

   <h4 class="graf graf--h4 graf-after--figure" id="7e17">
    后果
   </h4>
   <p class="graf graf--p graf-after--h4" id="1cc9">
    网络攻击对塞尔维亚网络媒体和记者的真正后果是什么？大多数内容被消失和 DDoS 攻击的案例都没有与内容本身相关的长期后果。正如 EFF 的创始人之一约翰吉尔摩（John Gilmore）所说的那样，“互联网将审查制度视为故障，并绕道而行”。从网络上取下的内容通常会在不同的地方成倍地增加，由其他博客或在线媒体网站重新发布，吸引更多的读者。
   </p>
   <p class="graf graf--p graf-after--p" id="9e87">
    但显然，这与人们的勇气直接相关。
   </p>
   <p class="graf graf--p graf-after--p" id="6309">
    <strong class="markup--strong markup--p-strong">
     不安全和恐惧
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="1fd9">
    我们可以说，这些袭击的主要后果是不安全和恐惧的加剧，从而对在线言论自由产生寒蝉效应。
   </p>
   <p class="graf graf--p graf-after--p" id="a2a4">
    事实是，发布批评权力结构（政府、犯罪集团或任何其他权力）的内容可能导致网站的破坏、阻塞或暂时下线，随后是大量的压力和昂贵的工作时间来恢复系统，这明显会影响人们自由表达自己的意愿。
   </p>
   <p class="graf graf--p graf-after--p" id="436b">
    <strong class="markup--strong markup--p-strong">
     在网络空间，防御通常比攻击更昂贵
    </strong>
    。对于那些无法负担昂贵的网络安全专家或技术解决方案来保护自己的小型独立媒体和公民媒体而言，这可能是非常令人沮丧的。根据莫罗佐夫的说法，DDoS 攻击给内容制作者带来了沉重的心理压力，迫使他们担心各种机构间问题，例如他们与互联网托管公司之关系的未来，以及网站不可用对其在线社区产生的破坏性影响等。
   </p>
   <p class="graf graf--p graf-after--p" id="e7f8">
    这种趋势的一个有点积极的结果是一些更专业的媒体组织实施了 DDoS 缓解系统，例如 CloudFlare。DDoS 攻击仍在继续，但与 2014 年相比，这些攻击已经变得更为复杂了，使用大型且更持久的僵尸网络，已经能够绕过免费版本的 CloudFlare 并关闭服务器。
   </p>
   <p class="graf graf--p graf-after--p" id="30a3">
    <strong class="markup--strong markup--p-strong">
     对公众的寒蝉效应
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="73fb">
    由于他们的博客、评论或其他形式的在线写作而被逮捕的个人案例，不仅对记者和在线媒体组织产生了寒蝉效应，而且对塞尔维亚的一般在线用户群体也产生了影响，占塞尔维亚公民的60％。
   </p>
   <p class="graf graf--p graf-after--p" id="8f90">
    因此，公民在数字环境中不会感到被赋予权力和受到保护，这便减少了新技术的潜在使用。预计过去一年国家开展的众多法律诉讼将进一步加剧对网络言论的寒蝉效应。
   </p>
   <p class="graf graf--p graf-after--p" id="5c46">
    <strong class="markup--strong markup--p-strong">
     隐私侵犯和监视
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="d808">
    针对个人和专业组织的通信以及电子邮件、在线文档和数据库等工作工具的针对性攻击可能会危害来源的匿名性，泄露调查计划，或通过发布私人信息以及身份盗用来诋毁受害者。
    <strong class="markup--strong markup--p-strong">
     达到必要的数字安全水平通常意味着复杂的程序，改变与使用技术相关的常规习惯，当然这是必要的，但同时也可能导致记者和组织的整体效率降低
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="4443">
    <strong class="markup--strong markup--p-strong">
     阻止公众对话
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="c9cd">
    使用由政党成员策划的技术工具操纵公众舆论的规模，导致了对主要新闻门户网站和社交网络的评论和使用泛滥，
    <strong class="markup--strong markup--p-strong">
     集中化愈加严重
    </strong>
    ，这一状况正在改变开放空间，使对话和在线观点表达只在一个的领域可以接收到意见，
    <strong class="markup--strong markup--p-strong">
     从而造成了舆论的整体错误形象
    </strong>
    。这种人为的噪音使得个人的真实声音几乎不可能被听到，这阻碍了对社会重要的话题的对话。
   </p>
   <p class="graf graf--p graf-after--p" id="ce0a">
    <strong class="markup--strong markup--p-strong">
     无法保护
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="802c">
    显然，政府未能保护塞尔维亚的在线媒体和公民记者。相关的国家机构在某些情况下对于更有效的反应只具有有限的技术和组织能力。然而，真正危险的是，相关公共机构（警察和司法机构）的反应因案例而异，有时它们非常有效，有时则非常缓慢且没有适当的反应。
   </p>
   <p class="graf graf--p graf-after--p" id="9ff0">
    在大多数情况下，国家当局极其缓慢或完全没有反应，这显然与对网络媒体、调查记者和批评政府的公民媒体的网络攻击有关。任何问题都不会得到解决。
   </p>
   <p class="graf graf--p graf-after--p" id="e534">
    另一方面，相关的公共机构在针对社交媒体用户和博客的逮捕和司法诉讼中反而被证明非常有效（如 Malagurski 案和在洪水期间引发“恐慌”的案件）。所有这些方面共同导致了该领域缺乏法律确定性，法治水平无法令人满意。
   </p>
   <p class="graf graf--p graf-after--p" id="427b">
    总之，公民社会应该努力起到援助这些危机的功能，我们无法指望政府的帮助，正相反很多问题就是政府造成的。我们演示这一调查思路，就是希望能在中国看到公民社会参与这项工作，除了你们，没有其他人能在此给出有效的帮助。制图推荐：

    </a>

   </p>
