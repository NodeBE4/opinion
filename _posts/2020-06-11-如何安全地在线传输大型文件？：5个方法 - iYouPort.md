  <p class="graf graf--p">
   由于电子邮件服务顽固地坚持其Web 1.0 25 MB 附件大小的限制，因此如果用户希望传输更大的文件，必须另想方法。
  </p>
  <p class="graf graf--p">
   当然，如果你完全不考虑安全性问题，总会有云存储可以用；但是，
   <strong class="markup--strong markup--p-strong">
    如果您需要一种安全的方式传输文件，那么就必须另想办法。
   </strong>
  </p>
  <p class="graf graf--p">
   本文将介绍五种安全的文件传输选项，它们都是免费的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、WeTransfer
   </strong>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://wetransfer.com/" href="https://wetransfer.com/" rel="noopener noreferrer" target="_blank">
    WeTransfer
   </a>
   的优势不仅是因为它非常简单，而且因为它具有快速的上传和下载速度。
  </p>
  <p class="graf graf--p">
   有免费和付费选项。免费选项允许最大2GB的文件传输，而无需注册帐户。只需上传文件，然后添加您自己的电邮和和个性化的消息即可。您可以使用一次性电邮服务。
  </p>
  <p class="graf graf--p">
   您可以通过电子邮件发送下载链接，也可以使用唯一的加密链接来复制和粘贴聊天消息。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12122 jetpack-lazy-image" data-lazy-sizes="(max-width: 528px) 100vw, 528px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-6.png?resize=528%2C758&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-6.png?w=528&amp;ssl=1 528w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-6.png?resize=209%2C300&amp;ssl=1 209w" data-recalc-dims="1" height="758" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-6.png?resize=528%2C758&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="528"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12122" data-recalc-dims="1" height="758" sizes="(max-width: 528px) 100vw, 528px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-6.png?resize=528%2C758&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-6.png?w=528&amp;ssl=1 528w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-6.png?resize=209%2C300&amp;ssl=1 209w" width="528"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   付费版计划提供了更多的功能（每月12美元或每年120美元），例如20GB文件传输、100GB文件存储空间、文件传输中的密码和有效期，甚至可以设置和设计自己的 WeTransfer 页面和URL 。
  </p>
  <p class="graf graf--p">
   但是，除非您要传输大型视频和音频文件，否则一般文件的大小永远不可能超过2GB。因此免费计划就足够了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、Firefox Send
   </strong>
  </p>

   <a class="markup--anchor markup--p-anchor" data-href="https://send.firefox.com/" href="https://send.firefox.com/" rel="noreferrer noopener" target="_blank">
    Firefox Send
   </a>
   可以让您在不注册的情况下进行1GB的文件传输、在注册免费帐户后可使用2.5GB的文件传输。
  </p>
  <p class="graf graf--p">
   另一端的人下载文件后，链接会自动失效，因此您不必担心该链接会被偷走或文档会被重新下载。
  </p>
  <p class="graf graf--p">
   但是，如果您使用 Firefox 帐户发送2.5GB的文件，则可以将这些链接保持活动状态长达一周，并与更多人共享该链接。

   <img alt="" class="aligncenter size-full wp-image-12125 jetpack-lazy-image" data-lazy-sizes="(max-width: 928px) 100vw, 928px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-3.png?resize=928%2C634&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-3.png?w=928&amp;ssl=1 928w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-3.png?resize=300%2C205&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-3.png?resize=768%2C525&amp;ssl=1 768w" data-recalc-dims="1" height="634" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-3.png?resize=928%2C634&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="928"/>

   </noscript>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、RiseUp
   </strong>
  </p>
  <p class="graf graf--p">
   这是我们使用的在线协作平台，可以说比较理想。
  </p>
  <p class="graf graf--p">
   它用来文件传输的空间有限，上限仅为50MB。但是对于我们来说，在线协作中传输文件是很方便的。
  </p>
  <p class="graf graf--p">
   您只需要将文件拖到窗口中，就会生成一个加密链接，该链接将持续一周。或者，如果其他人已经告诉您他们拥有该文件，则可以立即删除自己的文件。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12126 jetpack-lazy-image" data-lazy-sizes="(max-width: 818px) 100vw, 818px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-3.png?resize=818%2C104&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-3.png?w=818&amp;ssl=1 818w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-3.png?resize=300%2C38&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-3.png?resize=768%2C98&amp;ssl=1 768w" data-recalc-dims="1" height="104" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-3.png?resize=818%2C104&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="818"/>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、Onionshare
   </strong>
  </p>
  <p class="graf graf--p">
   这当然是最好的。Onionshare 的优势在于，您的文件未存储在任何等待有人下载的中央服务器上。
  </p>
  <p class="graf graf--p">
   相反，其他人需要Tor浏览器，并且当两个浏览器都打开时，会形成加密连接。然后，文件会自动从你的计算机移到他们的计算机上。
  </p>
  <p class="graf graf--p">
   这是所有方法中最安全的一种。一个等效的P2P平台是 Resilio，您也可以尝试。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、Signal
   </strong>
  </p>
  <p class="graf graf--p">
   Signal 是目前最安全的消息传递应用程序。因此，理论上通过Signal发送文件也将足够安全。
  </p>
  <p class="graf graf--p">
   在该应用程序上，点击要发送文件的联系人，然后点击左侧的“ +”图标。
  </p>
  <p class="graf graf--p">
