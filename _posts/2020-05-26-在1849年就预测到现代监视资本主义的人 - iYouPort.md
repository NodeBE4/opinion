   这一切问题都有一个共同的根源。
  </p>
  <p class="graf graf--p">
   在法国政治思想家和历史学家亚历克西斯·德·托克维尔（Alexis de Toqueville）的书中有一段引人入胜的章节，题为：“民主国家必须惧怕哪种专制主义？” 在其中，作者描述了一种真正非同寻常的状况 —— 一种人类从未遇到过的社会状况。
  </p>
  <p class="graf graf--p">
   那是一百七十多年前，Toqueville 试图用手指触摸尚不存在的东西，但在他非凡的政治想象力中，他能够以惊人的清晰度预见到这一点。
  </p>
  <p class="graf graf--p">
   已经有充分的理由担心，由 Google 开创的、并由 Facebook 等寡头公司纷纷采用的商业监视业务模型，正在破坏我们的民主基础。

   <strong class="markup--strong markup--p-strong">
    这是一种新的、阴险的、微妙的、几乎看不见的征服，你我就是这一征服的主题，这种征服正是 Tocqueville 在1849年以不可思议的能力预见到的。
   </strong>

  <p class="graf graf--p">
   一百七十多年前他就写明了这点：“民主制度下的人民所遭受的那种压迫不会像在世界上进行的任何压迫一样。”
  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">

  </p>

   这里的过滤气泡是一个有用的概念：爱沙尼亚生物学家 Jakob vonUexküll 的研究核心就是 “气泡” 和 “周围世界” 的概念，
   <strong class="markup--strong markup--p-strong">
    每个生物为自己创造了一个 “气泡”，并以此作为他们对 “世界” 的体验。
   </strong>
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    One of the best ways to manipulate attention is to appeal to outrage and fear, emotions that increase engagement. Facebook’s algorithms give users what they want —— Roger McNamee, Time Magazine
   </p>

  <p class="graf graf--p">
   umwelt 是“在人类（和动物）的交流和指征研究中最关键的生物学基础”。该术语通常被翻译为 “以自我为中心的世界”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    对于每种生物而言，umwelt 都是其整体的全部。我们作为人类的独特之处在于我们的 umwelt 不是固定的、不动的、刚性的或静态的。
   </strong>
  </p>
  <p class="graf graf--p">
   我们了解 Facebook 算法操纵对用户的影响的一种方式就是，可见的，
   <strong class="markup--strong markup--p-strong">
    每个用户 umwelt 都在有效地收缩：变得越来越小，越来越钙化。彼此间无法再沟通。
   </strong>
  </p>
  <p class="graf graf--p">
   作者 Zeynep Tufekci 在 “ Facebook的算法如何抑制内容多样性以及新闻提要如何控制您的点击” 一书中已经明确了这点。通过确保每个人仅仅接触到其个人只可能会赞成和赞同的事物，我们的 umwelt（即 社会现实）就会大大减少、并同质化。
  </p>

    IYP 警告的《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bc%82%e8%ae%ae%e7%9a%84%e6%b6%88%e4%ba%a1%ef%bc%9a%e4%b8%ba%e4%bb%80%e4%b9%88%e7%bd%91%e7%bb%9c%e5%8f%98%e5%be%97%e5%90%8c%e8%b4%a8%e5%8c%96/" href="https://www.iyouport.org/%e5%bc%82%e8%ae%ae%e7%9a%84%e6%b6%88%e4%ba%a1%ef%bc%9a%e4%b8%ba%e4%bb%80%e4%b9%88%e7%bd%91%e7%bb%9c%e5%8f%98%e5%be%97%e5%90%8c%e8%b4%a8%e5%8c%96/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     异议的消亡：为什么网络变得同质化
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Facebook 的商业模式具有深远的影响，尤其是在我们对他人（与自己不相像或思想不相似的人）产生同情的能力的方面。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    IYP 2017 年的科幻小故事讽刺了这件事《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%bb%bf%e8%89%b2%e5%85%94%e5%ad%90%ef%bc%88%e7%a7%91%e5%b9%bb%e6%95%85%e4%ba%8b%ef%bc%89/" href="https://www.iyouport.org/%e7%bb%bf%e8%89%b2%e5%85%94%e5%ad%90%ef%bc%88%e7%a7%91%e5%b9%bb%e6%95%85%e4%ba%8b%ef%bc%89/" rel="noopener noreferrer" target="_blank">
     绿色兔子
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   这在缅甸产生了毁灭性的后果，在那里，Facebook 成为了针对罗兴亚人进行种族清洗的工具。
  </p>
  <p class="graf graf--p">
   Facebook 已经成为纯粹的
   <strong class="markup--strong markup--p-strong">
    “部落主义论坛”
   </strong>
   ，提倡所谓的 “简化版’社区’”，同时可以说是 “损害民主、科学和公共健康”的罪魁祸首 —— 正如 Siva Vaidhyanathan 在《
   <a class="markup--anchor markup--p-anchor" data-href="https://t.me/iyouport/6775" href="https://t.me/iyouport/6775" rel="noopener noreferrer" target="_blank">
    反社会的媒体
   </a>
   》中所抨击的： Facebook 切断了我们之间的联系、并破坏了民主（牛津大学出版社，2018年）。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%a4%be%e5%8c%ba%e5%ba%94%e8%af%a5%e6%98%af%e4%bb%80%e4%b9%88%ef%bc%9f%e6%98%af%e8%b0%81%e5%9c%a8%e7%8e%a9%e5%9d%8f%e8%bf%99%e4%b8%aa%e6%a6%82%e5%bf%b5%ef%bc%9f/" href="https://www.iyouport.org/%e7%a4%be%e5%8c%ba%e5%ba%94%e8%af%a5%e6%98%af%e4%bb%80%e4%b9%88%ef%bc%9f%e6%98%af%e8%b0%81%e5%9c%a8%e7%8e%a9%e5%9d%8f%e8%bf%99%e4%b8%aa%e6%a6%82%e5%bf%b5%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     社区应该是什么？是谁在玩坏这个概念
    </a>
    ？》
   </li>
  </ul>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11667 jetpack-lazy-image" data-lazy-sizes="(max-width: 655px) 100vw, 655px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-17.png?resize=655%2C430&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-17.png?w=655&amp;ssl=1 655w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-17.png?resize=300%2C197&amp;ssl=1 300w" data-recalc-dims="1" height="430" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-17.png?resize=655%2C430&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="655"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11667" data-recalc-dims="1" height="430" sizes="(max-width: 655px) 100vw, 655px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-17.png?resize=655%2C430&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-17.png?w=655&amp;ssl=1 655w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-17.png?resize=300%2C197&amp;ssl=1 300w" width="655"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   大部分研究表明，
   <strong class="markup--strong markup--p-strong">
    同理心和我们创造性地重建另一个 umwelt 的能力之间有着密切的关系。
   </strong>
  </p>
  <p class="graf graf--p">
   尽管无法共享他人或自己的 umwelt —— 我们每个人都像以前一样一直在自己构建的气泡中 —— 我们只能参加一个共同的 umwelt，在许多方面，这是社交媒体的既定目标。
  </p>
  <p class="graf graf--p">
   具有讽刺意味的是，声称拥有最重要的连通性的 Facebook 实际上促成了相反的结果 —— 它让每个人都陷入了僵化僵硬的孤立泡沫中。
  </p>
  <p class="graf graf--p">
   面对政府日渐加强的无能局面，我们必须认识到
   <strong class="markup--strong markup--p-strong">
    Facebook、Google 和 Amazon 是新的利维坦
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    下载国际特赦组织的报告《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%9b%bd%e9%99%85%e7%89%b9%e8%b5%a6%e7%bb%84%e7%bb%87%e7%a7%b0google%e5%92%8cfacebook%e4%b8%ba%e4%be%b5%e7%8a%af%e4%ba%ba%e6%9d%83%e8%80%85/" href="https://www.iyouport.org/%e5%9b%bd%e9%99%85%e7%89%b9%e8%b5%a6%e7%bb%84%e7%bb%87%e7%a7%b0google%e5%92%8cfacebook%e4%b8%ba%e4%be%b5%e7%8a%af%e4%ba%ba%e6%9d%83%e8%80%85/" rel="noopener noreferrer" target="_blank">
     Google 和 Facebook 是人权侵犯者
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   Facebook 只向用户提供与他们个人共同意见的帖子，就像 Tocqueville 所说的“ tutelary’s power”（托克维尔的 “tutelary’s power”），“它将意志的行动限制在一个很小的空间内，并且一点一点地窃取了每个公民自由意志的使用”。
  </p>
  <p class="graf graf--p">
   这些公司不仅仅希望自动化信息：正如 Zuboff 所说，
   <strong class="markup--strong markup--p-strong">
    “现在的目标是使我们每个人自动化 …… 通过规避个人意识来产生无知，从而消除任何自决的可能性。”
   </strong>
  </p>
  <p class="graf graf--p">
   把所有人变成木偶。
  </p>
  <p>
   <strong>
    Julian Assange 在2011年就声明过：Facebook是一台恐怖的间谍机器
   </strong>
   《
   <a href="https://www.cnet.com/news/assange-facebook-is-an-appalling-spy-machine/" rel="noopener noreferrer" target="_blank">
    Assange: Facebook is an ‘appalling spy machine’
   </a>
   》。
  </p>
  <figure aria-describedby="caption-attachment-11668" class="wp-caption aligncenter" id="attachment_11668" style="width: 881px">
   <img alt="" class="size-full wp-image-11668 jetpack-lazy-image" data-lazy-sizes="(max-width: 881px) 100vw, 881px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-6-scaled.jpeg?resize=881%2C2048&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-6-scaled.jpeg?w=881&amp;ssl=1 881w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-6-scaled.jpeg?resize=129%2C300&amp;ssl=1 129w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-6-scaled.jpeg?resize=440%2C1024&amp;ssl=1 440w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-6-scaled.jpeg?resize=768%2C1786&amp;ssl=1 768w" data-recalc-dims="1" height="2048" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-6-scaled.jpeg?resize=881%2C2048&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="881"/>
   <noscript>
    <img alt="" class="size-full wp-image-11668" data-recalc-dims="1" height="2048" sizes="(max-width: 881px) 100vw, 881px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-6-scaled.jpeg?resize=881%2C2048&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-6-scaled.jpeg?w=881&amp;ssl=1 881w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-6-scaled.jpeg?resize=129%2C300&amp;ssl=1 129w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-6-scaled.jpeg?resize=440%2C1024&amp;ssl=1 440w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-6-scaled.jpeg?resize=768%2C1786&amp;ssl=1 768w" width="881"/>
   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-11668">
    Mr. Robot 的台词，讲述的也是同一件事：寡头公司至高无上的权力正在操纵整个世界，同理心的消融、定制人的囚禁、umwelt，将一切商品化，包括异议
   </figcaption>
  </figure>
  <p class="graf graf--p">
   Facebook 的商业模式代表了一种
   <strong class="markup--strong markup--p-strong">
    新的隐性征服形式
   </strong>
   ，这种形式并不构成专制，但是，正如 Vaidhyanathan 所观察到的那样，
   <strong class="markup--strong markup--p-strong">
    “它阻碍、妥协、激化、压制、令人迷惑，最终使每个国家的所以人口均沦为仅由政府管理的怯懦而机械性的动物。”
   </strong>
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    it hinders, compromises, enervates, extinguishes, dazes, and finally reduces each nation to being nothing more than timid and industrious animals of which the government is the shepherd —— Vaidhyanathan
   </p>
  </blockquote>
  <p class="graf graf--p">
   Facebook 为认知规范的恶化做出了自己的贡献，并帮助迎来了所谓的后真相时代。
  </p>
  <p class="graf graf--p">
   这种对真理的轻视背后的动机一直是相同的 —— 即，它服务于钱。
  </p>
  <p class="graf graf--p">
   正如 McNamee 所说：“在 Facebook 上，
   <strong class="markup--strong markup--p-strong">
    普通信息和虚假信息看起来是一样的；唯一的区别是，虚假信息产生了更多的收入，因此得到了更好的待遇
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   与许多其他人一样，McNamee 建议用户应控制自己的数据和元数据 —— 说的就好像数据所有权问题是监视资本主义祸害的解决方案一样。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     《
    </strong>
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%95%b0%e6%8d%ae%e6%89%80%e6%9c%89%e6%9d%83%e4%b8%8d%e4%bb%85%e6%97%a0%e6%b3%95%e8%a7%a3%e5%86%b3%e7%8e%b0%e6%9c%89%e9%97%ae%e9%a2%98%ef%bc%8c%e5%ae%83%e8%bf%98%e5%88%b6%e9%80%a0/" href="https://www.iyouport.org/%e6%95%b0%e6%8d%ae%e6%89%80%e6%9c%89%e6%9d%83%e4%b8%8d%e4%bb%85%e6%97%a0%e6%b3%95%e8%a7%a3%e5%86%b3%e7%8e%b0%e6%9c%89%e9%97%ae%e9%a2%98%ef%bc%8c%e5%ae%83%e8%bf%98%e5%88%b6%e9%80%a0/" rel="noopener noreferrer" target="_blank">
     <strong class="markup--strong markup--li-strong">
      “数据所有权”不仅无法解决现有问题，它还制造了新的问题
     </strong>
    </a>
    <strong class="markup--strong markup--li-strong">
     》
    </strong>
   </li>
  </ul>
  <p class="graf graf--p">
   这不是真的。这种想法的问题在于，它没有问到更根本的问题，即 这些数据是否应该存在。
  </p>
  <p class="graf graf--p">
   正如 Zuboff 所说：
   <strong class="markup--strong markup--p-strong">
    “这就像去谈判一个7岁的孩子一天应该工作多少个小时，而不是在谈论童工的基本合法性。”
   </strong>
  </p>
  <p class="graf graf--p">
   监视资本主义代表了一种新的专制主义，这种专制主义正在损害我们的个人自治能力，以便行为数据可以不受阻碍地继续生成，从而为市场以及 Google 和 Facebook 的真实客户也就是那些广告商提供服务。
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    We are becoming the kind of solipsistic and atomistic society that Tocqueville foresaw, “an enumerable crowd of like and equal men who revolve on themselves without repose … each of them, withdrawn, and apart, is like a stranger to the destiny of all —— Sam Ben-Meir
   </p>
  </blockquote>
  <p class="graf graf--p">
   我们正在成为 Tocqueville 所预见的那种唯我论的和原子主义的社会，
   <strong class="markup--strong markup--p-strong">
    “一群毫无区别的[定制]人，他们自发地生活着 …… 冷漠，疏远，彼此分隔，就像其他所有人的命运的陌生人一样。 …
   </strong>
  </p>
  <p class="graf graf--p">
   Alexis de Tocqueville 警告人们，压迫完全可能采取温和的、静悄悄的形式，但仍然可以摧毁真正的自由。
  </p>
  <p class="graf graf--p">
   要充分应对该问题，不仅需要更大的隐私或数据所有权，并且，还需要对我们的基本假设提出根本性的质疑，并对资本主义监视时代的民主意味着什么这个问题，需要有新的理解。
  </p>
  <p class="graf graf--p">
   IYP 需要补充一句：
   <strong class="markup--strong markup--p-strong">
    解决方案必须是技术的民主化，只有技术的基础设施是民主的，社会才能是民主的
   </strong>
   。监视资本主义的解毒剂就是如此。⚪️
  </p>
  <p class="graf graf--p">

    How Alexis De Tocqueville Predicted Surveillance Capitalism Over 175 Years Ago

  </p>
