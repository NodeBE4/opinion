  </p>
  <p class="graf graf--p graf-after--p" id="1973">
   <strong class="markup--strong markup--p-strong">
    1. OSINT Framework
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="77f3">
   虽然
   <a class="markup--anchor markup--p-anchor" data-href="http://osintframework.com/" href="http://osintframework.com/" rel="noopener nofollow noreferrer" target="_blank">
    OSINT Framework
   </a>
   不是在您的服务器上运行的工具，但它是通过查询网络上公开提供的免费搜索引擎、资源和工具来获取有价值信息的一种非常有用的方法。它专注于为 OSINT 数据源提供最佳链接。
  </p>
  <p class="graf graf--p graf-after--p" id="6e75">
   虽然这个 Web 应用程序最初是专注于 IT 安全性而创建的，但随着时间的推移，它现在也可以从其他行业获得其他类型的信息。它用于查询信息的大多数网站都是免费的，但有些网站可能需要支付较低的费用。
  </p>

   <p class="graf graf--p graf-after--figure" id="a7b7">
    <strong class="markup--strong markup--p-strong">
     2. CheckUserNames
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="3b2c">
    <a class="markup--anchor markup--p-anchor" data-href="http://checkusernames.com/" href="http://checkusernames.com/" rel="noopener nofollow noreferrer" target="_blank">
     CheckUserNames
    </a>
    是一个在线工具，可以帮助您
    <strong class="markup--strong markup--p-strong">
     在 170 多个社交网络中查找用户名
    </strong>
    。如果您正在运行调查以确定在不同社交网络上使用相同用户名的目标，则此功能尤其有用。
   </p>
   <p class="graf graf--p graf-after--p" id="d448">
    它还可以用于检查品牌公司名，而不仅仅是个人。
   </p>
   <p class="graf graf--p graf-after--p" id="1704">
    <strong class="markup--strong markup--p-strong">
     3. HaveIbeenPwned
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="f559">
    <a class="markup--anchor markup--p-anchor" data-href="https://haveibeenpwned.com/" href="https://haveibeenpwned.com/" rel="noopener nofollow noreferrer" target="_blank">
     HaveIbeenPwned
    </a>
    可以帮助您检查您的帐户过去是否遭到入侵。该网站由 Troy Hunt 开发，Troy Hunt 是该市场最受尊敬的 IT 安全专业人士之一，多年来一直提供准确的报告。
   </p>
   <p class="graf graf--p graf-after--p" id="5efa">
    如果您怀疑自己的帐户遭到入侵，或者想要验证外部帐户的第三方妥协，这是一个完美的工具。它可以追踪来自 Gmail，Hotmail，雅虎帐户，以及 LastFM，Kickstarter，
    <a class="markup--anchor markup--p-anchor" data-href="http://wordpress.com/" href="https://wordpress.com/" rel="noreferrer noopener nofollow" target="_blank">
     WordPress.com
    </a>
    ，Linkedin 和许多其他热门网站来源的网络攻击。
   </p>

   <p class="graf graf--p graf-after--figure" id="d7a9">
    <strong class="markup--strong markup--p-strong">
     4. BeenVerified
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="48af">
    <a class="markup--anchor markup--p-anchor" data-href="https://www.beenverified.com/" href="https://www.beenverified.com/" rel="noopener nofollow noreferrer" target="_blank">
     BeenVerified
    </a>
    是另一种类似的工具，当您需要在公共互联网记录上搜索目标人时使用它。当您进行 IT 安全调查并且目标是未知人员时，获取世界上任何人的更多有价值信息可能非常有用。
   </p>
   <p class="graf graf--p graf-after--p" id="4a64">
    完成后，
    <strong class="markup--strong markup--p-strong">
     将显示结果页面，其中包含与该姓名匹配的所有人员，以及他们的详细信息，地理位置，电话号码等
    </strong>
    。一旦找到，您就可以构建自己的报告。
   </p>
   <p class="graf graf--p graf-after--p" id="6e22">
    关于 BeenVerified 的惊人之处在于它还包括有关犯罪记录和官方政府信息的数据。
   </p>
   <p class="graf graf--p graf-after--p" id="1c89">
    BeenVerified 的背景报告可能包括来自多个数据库的记录，
    <strong class="markup--strong markup--p-strong">
     破产记录、职业历史、社交媒体配置文件、甚至在线照片的信息
    </strong>
    ！
   </p>
   <p class="graf graf--p graf-after--p" id="287f">
    <strong class="markup--strong markup--p-strong">
     5. Censys
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="e7fb">
    <a class="markup--anchor markup--p-anchor" data-href="https://censys.io/" href="https://censys.io/" rel="noopener nofollow noreferrer" target="_blank">
     Censys
    </a>
    是一个很棒的搜索引擎，用于获取有关连接到互联网的任何设备的最新和最准确的信息，它可以是服务器或域名。
   </p>
   <p class="graf graf--p graf-after--p" id="a0ba">
    您将能够找到有关
    <strong class="markup--strong markup--p-strong">
     在任何服务器上运行的 80 和 443 端口的完整地理位置和详细技术信息
    </strong>
    ，以及目标网站的 HTTP / S 正文内容和 GET 响应，Chrome TLS 握手，完整的 SSL 证书链信息以及 WHOIS 信息。
   </p>
   <p class="graf graf--p graf-after--p" id="3a38">
    <strong class="markup--strong markup--p-strong">
     6. BuiltWith
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="e152">
    <a class="markup--anchor markup--p-anchor" data-href="https://builtwith.com/" href="https://builtwith.com/" rel="noopener nofollow noreferrer" target="_blank">
     BuiltWith
    </a>
    是一种很酷的方式，可以检测互联网上任何网站使用的技术。
   </p>
   <p class="graf graf--p graf-after--p" id="b625">
    它包括有关使用的完整详细信息，如 WordPress，Joomla，Drupal 等，以及全面深入的 Javascript 和 CSS 库，如 jquery，bootstrap / foundation，external fonts，Web 服务器类型（Nginx，Apache，IIS 等），SSL 提供商以及使用的网络托管服务。
   </p>
   <p class="graf graf--p graf-after--p" id="3532">
    BuiltWith 还可以让您找到目前最流行的技术，或者哪些技术正在趋势化。
   </p>
   <p class="graf graf--p graf-after--p" id="65c6">
    毫无疑问，它是一个非常好的工具，可以收集有关任何网站的所有可能的技术细节。
   </p>
   <p class="graf graf--p graf-after--p" id="ebd5">
    <strong class="markup--strong markup--p-strong">
     7. Google Dorks
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="aa31">
    在调查目标人或公司时，许多 IT 安全新手都忘记了使用传统搜索引擎进行侦察和收集的重要性。在这种情况下，Google Dorks 可以成为您最好的朋友。他们自2002年以来一直在那里，可以帮助你进行在线侦察。
   </p>
   <p class="graf graf--p graf-after--p" id="61cf">
    Google Dorks 只是向 Google 查询某些可能对您的安全调查有用的信息的方法。
   </p>
   <p class="graf graf--p graf-after--p" id="785e">
    搜索引擎会索引有关互联网上几乎所有内容的大量信息，包括个人、公司及其数据。
   </p>
   <ul class="postList">
    <li class="graf graf--li graf-after--p" id="95a2">
     Filetype：您可以使用此 dork 查找任何文件类型。
    </li>
    <li class="graf graf--li graf-after--li" id="4ca8">
     Ext：可以帮助您查找具有特定扩展名的文件（例如.txt，.log等）。
    </li>
    <li class="graf graf--li graf-after--li" id="defb">
     Intext：有助于搜索任何页面内的特定文本。
    </li>
    <li class="graf graf--li graf-after--li" id="7d04">
     Intitle：它将搜索页面标题内的任何特定单词。
    </li>
    <li class="graf graf--li graf-after--li" id="bdd9">
     Inurl：将在任何网站的 URL 内寻找提到的单词。
    </li>
   </ul>

    <a class="markup--anchor markup--p-anchor" data-href="https://www.shodan.io/" href="https://www.shodan.io/" rel="noopener nofollow noreferrer" target="_blank">

    </a>

   </p>

    </strong>

    <strong class="markup--strong markup--p-strong">
     15. Nmap
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="0dd6">
    <a class="markup--anchor markup--p-anchor" data-href="https://www.scrt.ch/en/attack/downloads/webshag" href="https://www.scrt.ch/en/attack/downloads/webshag" rel="noopener nofollow noreferrer" target="_blank">
     Nmap
    </a>
    是最受欢迎和最广泛使用的安全审计工具之一，其名称的意思是“网络映射器”。是一个免费的开源实用程序，用于跨本地和远程主机进行安全审核和网络探索。
   </p>
   <p class="graf graf--p graf-after--p" id="cab8">
    一些主要功能包括：
   </p>
   <ul class="postList">
    <li class="graf graf--li graf-after--p" id="cd9c">
     主机检测：Nmap 能够识别任何网络中具有某些端口打开的主机，或者能够发送对 ICMP 和 TCP 数据包的响应的主机。
    </li>
    <li class="graf graf--li graf-after--li" id="d184">
     IP 和 DNS 信息检测：包括设备类型，Mac 地址甚至反向 DNS 名称。
    </li>
    <li class="graf graf--li graf-after--li" id="2243">
     端口检测：Nmap 可以检测目标网络上打开的任何端口，并让您知道其上可能正在运行的服务。
    </li>
    <li class="graf graf--li graf-after--li" id="563c">
     操作系统检测：获取所有连接主机的完整操作系统版本检测和硬件规格。
    </li>
    <li class="graf graf--li graf-after--li" id="097b">
     版本检测：Nmap 还可以获取应用程序名称和版本号。
    </li>
   </ul>

    <strong class="markup--strong markup--p-strong">
     16. WebShag

   </p>
   <p class="graf graf--p graf-after--p" id="42be">
    <a class="markup--anchor markup--p-anchor" data-href="https://www.scrt.ch/en/attack/downloads/webshag" href="https://www.scrt.ch/en/attack/downloads/webshag" rel="noopener nofollow noreferrer" target="_blank">
     WebShag
    </a>
    是一个很棒的服务器审计工具，用于扫描 HTTP 和 HTTPS 协议。与其他工具相同，它是 Kali Linux 的一部分，可以帮助您进行 IT 安全研究和渗透测试。
   </p>
   <p class="graf graf--p graf-after--p" id="0a70">
    您将能够启动简单的扫描，或使用高级方法（如通过代理或 HTTP 身份验证）。
   </p>
   <p class="graf graf--p graf-after--p" id="92e6">
    用 Python 编写，它可以是审计系统时最好的盟友之一。
   </p>
   <p class="graf graf--p graf-after--p" id="ff39">
    主要功能包括：
   </p>
   <ul class="postList">
    <li class="graf graf--li graf-after--p" id="2c4b">
     端口扫描
    </li>
    <li class="graf graf--li graf-after--li" id="6cd8">
     网址扫描
    </li>
    <li class="graf graf--li graf-after--li" id="5a90">
     文件模糊查找
    </li>
    <li class="graf graf--li graf-after--li" id="c581">
     网站抓取
    </li>
   </ul>
   <p class="graf graf--p graf-after--li" id="d810">
    为了避免被远程服务器安全系统阻止，它通过每个 HTTP 代理服务器启动随机请求来使用智能 IDS 规避系统，因此您可以继续审核服务器而不会被禁止。
   </p>
   <p class="graf graf--p graf-after--p" id="c4ab">
    <strong class="markup--strong markup--p-strong">
     17. OpenVAS
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="ac6d">
    <a class="markup--anchor markup--p-anchor" data-href="http://www.openvas.org/" href="http://www.openvas.org/" rel="noopener nofollow noreferrer" target="_blank">
     OpenVAS
    </a>
    （开放式漏洞评估系统）是一个安全框架，包括针对信息安全专业人员的特定服务和工具。
   </p>
   <p class="graf graf--p graf-after--p" id="f719">
    这是一个开源漏洞扫描程序和安全管理器，是在著名的 Nessus 从开源转换为私有之后构建的。然后，Nessus 漏洞扫描程序的原始开发人员决定分割原始项目并创建 OpenVAS。
   </p>
   <p class="graf graf--p graf-after--p" id="137b">
    虽然它比旧的 Nessus 设置起来有点麻烦，但在使用它来分析远程主机的安全性时却非常有效。
   </p>
   <p class="graf graf--p graf-after--p" id="d5c6">
    OpenVAS 中包含的主要工具是 OpenVAS Scanner，这是一种高效的代理，可以在目标计算机上执行所有网络漏洞测试。
   </p>
   <p class="graf graf--p graf-after--p" id="f402">
    另一方面，另一个主要组件称为 OpenVAS Manager，它基本上是漏洞管理解决方案，允许您将扫描数据存储到 SQLite 数据库中，因此您可以以一种简单易用的方式搜索、过滤和排序扫描结果。
   </p>

   </p>
   <p class="graf graf--p graf-after--p" id="aca6">
    <a class="markup--anchor markup--p-anchor" data-href="https://github.com/mschwager/fierce" href="https://github.com/mschwager/fierce" rel="noopener nofollow noreferrer" target="_blank">
     Fierce
    </a>
    是一种用 PERL 编写的 IP 和 DNS 侦察工具，很著名，帮助 IT 专业人员找到与域名相关的目标 IP。
   </p>
   <p class="graf graf--p graf-after--p" id="6f43">
    它最初是由 RSnake 和旧http://ha.ckers.org/的其他成员一起编写的。它主要用于本地和远程企业网络。
   </p>
   <p class="graf graf--p graf-after--p" id="ea41">
    一旦定义了目标网络，它将针对所选域启动多次扫描，然后它将尝试查找配置错误的网络和易受攻击的点，这些点可能泄露私人的和有价值的数据。
   </p>
   <p class="graf graf--p graf-after--p" id="747d">
    结果将在几分钟内准备就绪。
   </p>
   <p>
    <img alt="" class="aligncenter size-full wp-image-4789 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-147.png?resize=1100%2C551&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-147.png?w=1275&amp;ssl=1 1275w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-147.png?resize=300%2C150&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-147.png?resize=768%2C385&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-147.png?resize=1024%2C513&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-147.png?resize=1100%2C551&amp;ssl=1 1100w" data-recalc-dims="1" height="551" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-147.png?resize=1100%2C551&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
    <noscript>
     <img alt="" class="aligncenter size-full wp-image-4789" data-recalc-dims="1" height="551" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-147.png?resize=1100%2C551&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-147.png?w=1275&amp;ssl=1 1275w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-147.png?resize=300%2C150&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-147.png?resize=768%2C385&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-147.png?resize=1024%2C513&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-147.png?resize=1100%2C551&amp;ssl=1 1100w" width="1100"/>
    </noscript>
   </p>
   <p class="graf graf--p graf-after--figure" id="ae5f">
    <strong class="markup--strong markup--p-strong">
     19. Unicornscan
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="2dd8">
    <a class="markup--anchor markup--p-anchor" data-href="https://github.com/dneufeld/unicornscan" href="https://github.com/dneufeld/unicornscan" rel="noopener nofollow noreferrer" target="_blank">
     Unicornscan
    </a>
    是用于安全研究的顶级收集工具之一。它还有一个内置的关联引擎，旨在同时实现高效、灵活和可扩展的能力。
   </p>
   <p class="graf graf--p graf-after--p" id="a147">
    主要功能包括：
   </p>
   <ul class="postList">
    <li class="graf graf--li graf-after--p" id="ca91">
     完整的 TCP / IP 设备/网络扫描。
    </li>
    <li class="graf graf--li graf-after--li" id="d26b">
     异步无状态 TCP 扫描（包括所有 TCP 标志变体）。
    </li>
    <li class="graf graf--li graf-after--li" id="df42">
     异步 TCP 标志检测。
    </li>
    <li class="graf graf--li graf-after--li" id="dc42">
     UDP 协议扫描。
    </li>
    <li class="graf graf--li graf-after--li" id="6eaa">
     A / P OS识别。
    </li>
    <li class="graf graf--li graf-after--li" id="4507">
     应用和组件检测。
    </li>
    <li class="graf graf--li graf-after--li" id="088c">
     支持 SQL 关系输出。
    </li>
   </ul>
   <p class="graf graf--p graf-after--li" id="1971">
    <strong class="markup--strong markup--p-strong">
     20. Foca
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="fb17">
    FOCA (Fingerprinting Organizations with Collected Archives) 是 ElevenPaths 编写的工具，可用于扫描、分析、提取和分类来自远程 Web 服务器及其隐藏的信息。
   </p>
   <p class="graf graf--p graf-after--p" id="eca4">
    Foca 能够分析和收集来自 MS Office 套件，OpenOffice，PDF 以及 Adobe InDesign 和 SVG 和 GIF 文件的有价值数据。此安全工具还可以与 Google，Bing 和 DuckDuckGo 搜索引擎一起积极地从这些文件中收集其他数据。获得完整文件列表后，它会开始提取信息，以尝试从文件中识别更有价值的数据。
   </p>
   <p class="graf graf--p graf-after--p" id="42bd">
    好啦，正如您所看到的，事实上有很多侦察和收集工具。在这篇文章中，我们只提到了20种最受欢迎​​的工具，如果您有相关经验，欢迎留言介绍您认为最顺手的工具给大家。⚪️
   </p>
