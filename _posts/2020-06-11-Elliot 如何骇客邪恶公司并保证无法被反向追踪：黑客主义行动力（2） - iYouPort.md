  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    代理如何工作
   </strong>
  </h3>
  <p class="graf graf--p">
   如您所知，每当您通过互联网访问（或黑客入侵）网站或服务器时，您唯一的IP地址都会紧跟着您。因此，仅通过跟踪源IP地址来跟踪攻击的源头是很容易的。
  </p>
  <p class="graf graf--p">
   几乎所有黑客都会使用代理来隐藏或模糊其IP地址；也就是将流量发送到中间代理，代理再将流量发送到目标，用其自身替换源IP地址。
  </p>
  <p class="graf graf--p">
   这样，恶意流量看起来似乎来自代理而不是原始发送者。
  </p>
  <p class="graf graf--p">

   <img alt="" class="aligncenter size-full wp-image-12138 jetpack-lazy-image" data-lazy-sizes="(max-width: 716px) 100vw, 716px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6.jpg?resize=716%2C151&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6.jpg?w=716&amp;ssl=1 716w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6.jpg?resize=300%2C63&amp;ssl=1 300w" data-recalc-dims="1" height="151" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6.jpg?resize=716%2C151&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="716"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12138" data-recalc-dims="1" height="151" sizes="(max-width: 716px) 100vw, 716px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6.jpg?resize=716%2C151&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6.jpg?w=716&amp;ssl=1 716w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6.jpg?resize=300%2C63&amp;ssl=1 300w" width="716"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   相反，来尝试一下俄罗斯代理的列表。
  </p>
  <p class="graf graf--p">
   👉由于欧盟和美国执法机构对俄罗斯没有管辖权，因此许多黑客使用俄罗斯代理。这意味着很少或根本没有机会追踪其身份。这也算是地缘政治的技术性玩法了。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12139 jetpack-lazy-image" data-lazy-sizes="(max-width: 711px) 100vw, 711px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7.jpg?resize=711%2C532&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7.jpg?w=711&amp;ssl=1 711w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7.jpg?resize=300%2C224&amp;ssl=1 300w" data-recalc-dims="1" height="532" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7.jpg?resize=711%2C532&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="711"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12139" data-recalc-dims="1" height="532" sizes="(max-width: 711px) 100vw, 711px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7.jpg?resize=711%2C532&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7.jpg?w=711&amp;ssl=1 711w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7.jpg?resize=300%2C224&amp;ssl=1 300w" width="711"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   可以看到，该列表中俄罗斯有5个“高匿名”代理。那就选其中之一吧。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第四步：配置代理链
   </strong>
  </h3>
  <p class="graf graf--p">
   现在有了潜在的匿名代理列表，您必须配置代理链来使用它。
  </p>
  <p class="graf graf--p">
   与几乎所有 Linux / Unix 应用一样，配置由纯文本文件完成。这些文件通常位于
   <em class="markup--em markup--p-em">
    /etc
   </em>
   目录中。在这种情况下，可以在以下位置找到代理链的配置文件：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     /etc/proxychains.conf
    </em>
   </strong>
  </p>
  <p class="graf graf--p">
   可以使用任何文本编辑器打开它，这里使用的是 Leafpad。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     kali &gt; leafpad /etc/proxychains.conf
    </em>
   </strong>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12140 jetpack-lazy-image" data-lazy-sizes="(max-width: 567px) 100vw, 567px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/8.jpg?resize=567%2C647&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/8.jpg?w=567&amp;ssl=1 567w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/8.jpg?resize=263%2C300&amp;ssl=1 263w" data-recalc-dims="1" height="647" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/8.jpg?resize=567%2C647&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="567"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12140" data-recalc-dims="1" height="647" sizes="(max-width: 567px) 100vw, 567px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/8.jpg?resize=567%2C647&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/8.jpg?w=567&amp;ssl=1 567w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/8.jpg?resize=263%2C300&amp;ssl=1 263w" width="567"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   默认情况下，代理链设置为使用Tor。正如 Elliot 在第一集中所述，控制 Tor 出口节点的人就可以控制流量。当然政府间谍和警察也知道这一点，所以使用Tor不是好办法。
  </p>
  <p class="graf graf--p">
   要禁用 Tor，只需在64行之前添加一个注释标记（＃）。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12141 jetpack-lazy-image" data-lazy-sizes="(max-width: 544px) 100vw, 544px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/9.jpg?resize=544%2C89&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/9.jpg?w=544&amp;ssl=1 544w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/9.jpg?resize=300%2C49&amp;ssl=1 300w" data-recalc-dims="1" height="89" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/9.jpg?resize=544%2C89&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="544"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12141" data-recalc-dims="1" height="89" sizes="(max-width: 544px) 100vw, 544px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/9.jpg?resize=544%2C89&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/9.jpg?w=544&amp;ssl=1 544w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/9.jpg?resize=300%2C49&amp;ssl=1 300w" width="544"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   现在，要使用上述俄罗斯代理之一，只需要添加该代理的类型（http）以及IP地址和端口，就像上面所做的那样。然后，保存
   <em class="markup--em markup--p-em">
    proxychain.conf
   </em>
   文件并关闭它。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第五步：通过代理发送流量
   </strong>
  </h3>
  <p class="graf graf--p">
   只需键入以下内容即可打开 Iceweasel 浏览器：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     kali &gt; proxychains iceweasel
    </em>
   </strong>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12142 jetpack-lazy-image" data-lazy-sizes="(max-width: 540px) 100vw, 540px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/10.jpg?resize=540%2C21&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/10.jpg?w=540&amp;ssl=1 540w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/10.jpg?resize=300%2C12&amp;ssl=1 300w" data-recalc-dims="1" height="21" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/10.jpg?resize=540%2C21&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="540"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12142" data-recalc-dims="1" height="21" sizes="(max-width: 540px) 100vw, 540px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/10.jpg?resize=540%2C21&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/10.jpg?w=540&amp;ssl=1 540w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/10.jpg?resize=300%2C12&amp;ssl=1 300w" width="540"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   现在，您的所有流量都将通过该代理，并且如果有人检查流量，它会看上去似乎来自该代理。
  </p>
  <p class="graf graf--p">
   就是这么简单！您学会了吗？⚪️
