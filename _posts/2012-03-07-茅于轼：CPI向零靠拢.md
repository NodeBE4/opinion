    《环球人物》杂志
   </span>
   <span style="font-size: 9pt">
    记者
   </span>
   <span style="font-size: 9pt">
    白菊梅
   </span>
   <span style="font-size: 9pt">
    (
   </span>
   <span style="font-size: 9pt">
    整理
   </span>
   <span style="font-size: 9pt">
    )
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    中国眼下最热门的经济术语大概就是
   </span>
   <span style="font-size: 9pt">
    CPI(
   </span>
   <span style="font-size: 9pt">
    消费物价指数
   </span>
   <span style="font-size: 9pt">
    )
   </span>
   <span style="font-size: 9pt">
    了，粮价、油价、衣服价，什么都在涨。过去
   </span>
   <span style="font-size: 9pt">
    30
   </span>
   <span style="font-size: 9pt">
    多年中，物价总体上涨了
   </span>
   <span style="font-size: 9pt">
    5.4
   </span>
   <span style="font-size: 9pt">
    倍。不过，这中间不同商品的上涨幅度大不相同，个别品种甚至一点都没涨，比如手表，同样质量的手表不但没涨，还落了价。自行车大约涨了一倍半，粮食涨了六七倍，猪肉涨了约
   </span>
   <span style="font-size: 9pt">
    10
   </span>
   <span style="font-size: 9pt">
    倍，衣服鞋帽涨了四五倍。涨的最多的大概是看电影，过去
   </span>
   <span style="font-size: 9pt">
    5
   </span>
   <span style="font-size: 9pt">
    毛钱一张票，现在是
   </span>
   <span style="font-size: 9pt">
    50
   </span>
   <span style="font-size: 9pt">
    块，涨了
   </span>
   <span style="font-size: 9pt">
    100
   </span>
   <span style="font-size: 9pt">
    倍。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    百姓都怕通货膨胀，因为这会使自己腰包的钱"毛"了，购买力减少了。
   </span>
   <span style="font-size: 9pt">
    CPI
   </span>
   <span style="font-size: 9pt">
    的重要意义就是体现货币购买力的变化。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    我的理想中国里，
   </span>
   <span style="font-size: 9pt">
    CPI
   </span>
   <span style="font-size: 9pt">
    应该为零，货币的购买力既不要上升，也不要下降，稳定最好。如果货币的购买力下降即通货膨胀，手中有货币
   </span>
   <span style="font-size: 9pt">
    (
   </span>
   <span style="font-size: 9pt">
    或银行存款
   </span>
   <span style="font-size: 9pt">
    )
   </span>
   <span style="font-size: 9pt">
    的人都会遭受损失。如果货币的购买力上升，手持货币的人得到好处，可受损失的是作为生产者的企业，同样不利于全社会。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    从过去的经验看，货币购买力上升的事例很少，而下降的事例很多。因为当政府有权发行货币时，能通过印钞票获得购买力，一旦出现赤字，也可以通过印钞票来弥补。历史上发生过多次因为政府乱印钞票造成的超级通货膨胀，一年中物价上涨几百倍甚至成千上万倍，都不算稀奇。为了防止这种情况的出现，很多国家的中央银行都独立于政府，自己有权决定印发多少钞票。目前在我国，还没有实现央行的独立。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    中央银行最重要的任务是保持币值的稳定，但是这个任务并不容易完成，因为谁也不知道发行多少货币是正好的。有人认为，可以通过观察物价的起落决定货币的发行量，物价涨时减少货币发行，物价落时增加货币发行。这个方法看似有理，但货币数量影响物价有一个很长的滞后期，一般要一年左右，等到物价涨时再来减少货币发行的数量，已经太晚。至今还没有一个简单可靠的方法来决定货币发行量的多少，所以国家要有货币委员会，综合专家的意见决定。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    如果
   </span>
   <span style="font-size: 9pt">
    CPI
   </span>
   <span style="font-size: 9pt">
    超过
   </span>
   <span style="font-size: 9pt">
    2%
   </span>
   <span style="font-size: 9pt">
    ，就应该引起注意。当这一数字高到
   </span>
   <span style="font-size: 9pt">
    10%
   </span>
   <span style="font-size: 9pt">
    左右，就必须采取强有力的反通胀措施。最常用的办法是提高银行利息率，通过高利率抵消货币损失的购买力，以稳定大众的心理。我国
   </span>
   <span style="font-size: 9pt">
    1994
   </span>
   <span style="font-size: 9pt">
    年
   </span>
   <span style="font-size: 9pt">
    CPI
   </span>
   <span style="font-size: 9pt">
    高达
   </span>
   <span style="font-size: 9pt">
    24%
   </span>
   <span style="font-size: 9pt">
    ，银行就采取了利息率和
   </span>
   <span style="font-size: 9pt">
    CPI
   </span>
   <span style="font-size: 9pt">
    挂钩的办法。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    其实，经济运行的规律并不像表面所显示的那样，而是需要专业知识来洞察背后的机制，仅凭常识往往会把事情搞糟。比如：有人认为，通胀时应该提高工资，但这种方法会造成货币发行的循环加速，工资越高货币发得越多，通胀也就越严重。真正有效的办法是全社会咬咬牙，顶住个人所受的损失，坚持严控货币发行量，不到一年
   </span>
   <span style="font-size: 9pt">
    CPI
   </span>
   <span style="font-size: 9pt">
    肯定能恢复常态。另外，在通胀面前，百姓还往往希望政府能管住物价。其实，此时最好的办法，是政府取消价格管制，改善资源配置。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    通胀固然可怕，但是真正对生活好坏起作用的还是
   </span>
   <span style="font-size: 9pt">
    GDP
   </span>
   <span style="font-size: 9pt">
    的增长。改革开放
   </span>
   <span style="font-size: 9pt">
    30
   </span>
   <span style="font-size: 9pt">
    多年，物价涨了
   </span>
   <span style="font-size: 9pt">
    5
   </span>
   <span style="font-size: 9pt">
    倍多，而我们的财富生产总量提高了近
   </span>
   <span style="font-size: 9pt">
    15
   </span>
   <span style="font-size: 9pt">
    倍
   </span>
   <span style="font-size: 9pt">
    (
   </span>
   <span style="font-size: 9pt">
    同时人口增加了
   </span>
   <span style="font-size: 9pt">
    40%)
   </span>
   <span style="font-size: 9pt">
    ，温和的通货膨胀并不可怕。更需要我们关心的，是财富创造的增加和财富分配的合理。只有真正实现了这两方面，物价才可能达到我们的理想水平。
   </span>
  </div>
