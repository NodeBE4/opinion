  <p class="graf graf--p">
   泄露的文件详细介绍了该网站的审查指南，关于这家中国人拥有的社交网站 TikTok 指示其版主审查涉及天安门广场、西藏独立或被禁止的宗教团体法轮功的视频。
  </p>
  <p class="graf graf--p">
   卫报首次披露的文件阐明了拥有 TikTok 的总部位于北京的科技公司 ByteDance 如何通过该应用程序推进中国的对外政策目标。
  </p>
  <p class="graf graf--p">
   越来越多的审查是出于政治原因，有关香港抗议的讨论也正在 TikTok 受到审查：华盛顿邮报的一篇报道
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/technology/2019/09/15/tiktoks-beijing-roots-fuel-censorship-suspicion-it-builds-huge-us-audience/" href="https://www.washingtonpost.com/technology/2019/09/15/tiktoks-beijing-roots-fuel-censorship-suspicion-it-builds-huge-us-audience/" rel="noopener noreferrer" target="_blank">
    指出
   </a>
   ，在抖音搜索香港显示“几乎看不到抗议的迹象”。
  </p>
  <p class="graf graf--p">
   该审查指南将禁止使用的材料分为两类：某些内容被标记为“违规”，这表示该内容已从网站中完全删除，并可能导致用户被禁止使用该服务。
  </p>
  <p class="graf graf--p">
   另一种是，较小的违规行为被标记为“仅对发布者自己可见”，这将保留内容，但是通过 TikTok 的算法限制了内容的分发。
  </p>

  <p class="graf graf--p">
   后一种实施技术 —— 即 算法专制 —— 意味着用户可能不知道自己发布的内容是否已经被删除，他们会误认为自己的帖子“没有足够的影响力”，才导致无人回应。
  </p>
  <p class="graf graf--p">
   大部分审查指导方针都包含在“仇恨言论和宗教”一节中。在每种情况下，都将它们放在旨在使规则看起来通用而不是特例的上下文中。
  </p>
  <p class="graf graf--p">
   例如，禁止对中国的社会主义制度进行批评的一般禁令原文是这样的“：“禁止对任何政策、任何国家的社会规则（例如君主立宪制、君主制、议会制、三权分立、社会主义制度等）进行批评/攻击”。
  </p>
  <p class="graf graf--p">
   另一个禁令包括 “禁止对本国或其他国家历史的妖魔化或扭曲，例如1998年5月的印度尼西亚暴动、柬埔寨的种族灭绝、天安门事件等。”
  </p>
  <p class="graf graf--p">
   更具通用性的规则禁止 “极富争议的话题，例如分裂主义、宗教派别冲突、族群之间的冲突，例如夸大伊斯兰教派冲突，煽动北爱尔兰、车臣共和国、西藏和台湾的独立、以及夸大黑人和白人之间的冲突”。
  </p>
  <p class="graf graf--p">
   以上所有所谓的违规行为都会导致帖子被标记为“仅对发布者自己可见”。
  </p>
  <p class="graf graf--p">
   但是，宣传法轮功的帖子被标记为“违反规则”。法轮功自1999年以来一直受到北京的镇压，但自2001年以来发生的一次事件，当时有五人在天安门广场自焚，那之后该组织一直被禁止。
  </p>
  <p class="graf graf--p">
   奇怪的规则也可以在该审查指南的其他地方找到。
  </p>
  <p class="graf graf--p">
   例如，该服务关于所谓 “未成年人色情内容” 的政策明确规定了四类未成年人用户：一岁以下的婴儿； 1–8岁的孩子; 青春期未成年人; 未满18岁的任何人。
  </p>
  <p class="graf graf--p">
   但是，如果“不清楚”用户是否未满18岁，则指南明确建议版主“作为成年人对待”。
  </p>
  <p class="graf graf--p">
   该服务还禁止20名“外国领导人或敏感人物”的特定名单，包括金正日，金日成，圣雄甘地，弗拉基米尔·普京，唐纳德·特朗普，巴拉克·奥巴马，金正恩，安倍晋三，朴槿惠，佐科·维多多和纳伦德拉·莫迪。
  </p>
  <p class="graf graf--p">
   名单上显然没有的是中国国家主席习近平。很可能因为中国官方也使用该平台进行政治宣传。
  </p>
  <p class="graf graf--p">
   Bytedance 说，《卫报》看到的文件版本已于5月作废，当时香港抗议活动还没有开始，目前的准则并未提及具体的国家或问题。
  </p>
  <p class="graf graf--p">
   该公司说：“在 TikTok 成立之初，我们采取了一种直率的方法来最大程度地减少平台上的冲突，并且我们的审核准则允许对诸如煽动宗教派别或族裔群体之类的冲突的内容进行处罚 ……
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “随着 TikTok 去年开始在全球范围内流行，我们认识到这不是正确的方法，并开始致力于增强对每个市场都有细微了解的本地[审查]团队。随着我们的成长，我们已经在产品、团队、政策制定等各个方面实施了这种本地化方法 ……
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “有问题的旧指南已过时，不再使用。今天，我们采用了本地化的方法，包括本地版主、本地内容和审核策略，全局策略的本地优化等等。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    所谓的本地化，就是硅谷的口头禅：“我们在遵守当地法律”。
   </strong>
  </p>
  <p class="graf graf--p">
   TikTok 于2017年推出，不久就与一家美国公司 Musical.ly 合并，据报道并购价格10亿美元（8亿英镑），以促进该应用程序的增长。
  </p>
  <p class="graf graf--p">
   类似的，仅在中国使用的应用程序 Douyin 于2016年推出，到2017年底，用户人数增加到全中国人口的十分之一。
  </p>
  <p class="graf graf--p">
   TikTok 是2018年上半年全球 iOS App Store 上下载次数最多的商品，自那以来一直在其特别是25岁以下的核心用户群中广受欢迎。
  </p>
  <p class="graf graf--p">
   但是这种受欢迎程度是昂贵的：据报道，
   <strong class="markup--strong markup--p-strong">
    ByteDance 已经在 Facebook 广告上花费了
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/technology/2019/jul/05/why-tiktok-is-facing-greater-scrutiny-video-sharing-app-child-safety" href="https://www.theguardian.com/technology/2019/jul/05/why-tiktok-is-facing-greater-scrutiny-video-sharing-app-child-safety" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     10亿美元
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，以保持高增长
   </strong>
   。⚪️
  </p>
