      </strong>
     </em>

   <noscript>

   </noscript>

  <p class="graf graf--p">
   2017年出版的两本书《Social Movements and Civil War》和《Civil Resistance in the Arab Spring: Triumphs and Disasters》考察了“阿拉伯之春”的余波，得出的结论是，如果抵抗运动不成熟，可能导致暴力上升、威权主义和民主化失败。
  </p>
  <p class="graf graf--p">
   正如其编辑 Adam Roberts 所说：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    公民抵抗运动可能在更大的悲剧中发挥作用 —— 如果它在没有提供有效继承的情况下取代现有政权，则可能导致权力真空和无人管理的空间，从而造成灾难性的后果。它甚至可以导致战争的开端 —— 叙利亚就是一个特别令人担忧的典型例子。
   </em>
  </p>
  <p class="graf graf--p">
   然而，现有的关于公民抵抗运动的短期和长期影响的研究基于一个多世纪的大量代表性案例，提出了不同的重点。
  </p>
  <p class="graf graf--p">
   尽管认识到内战可能发生在公民抵抗运动之后，但定量研究指出，
   <strong class="markup--strong markup--p-strong">
    非暴力运动通常是民主化的积极力量，可以帮助建立较少暴力的政治秩序这一事实。
   </strong>
  </p>
  <p class="graf graf--p">
   至少有八本已发表的著作强调了抵抗运动取得积极成果的重大可能性。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    公民抵抗与民主
   </strong>
  </h3>
  <p class="graf graf--p">
   1、Karatnycky 和 Ackerman（2005）进行的
   <a class="markup--anchor markup--p-anchor" data-href="https://freedomhouse.org/sites/default/files/How%20Freedom%20is%20Won.pdf" href="https://freedomhouse.org/sites/default/files/How%20Freedom%20is%20Won.pdf" rel="noopener noreferrer" target="_blank">
    一项研究
   </a>
   使用了来自 Freedom House 组织的数据，并分析了1972年至2005年之间的67个政治过渡。他们发现，非暴力抵抗推动的过渡是：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    成功实现民主化的可能性是掌权者推动的自上而下的政治过渡的
    <strong class="markup--strong markup--li-strong">
     四倍以上
    </strong>
    ；
   </li>
   <li class="graf graf--li">
    巩固民主成果的可能性是经历过一些反对派暴力的政治过渡的
    <strong class="markup--strong markup--li-strong">
     三倍以上
    </strong>
    。
   </li>
  </ul>
  <p class="graf graf--p">
   此外，以公民抵抗为主导的过渡表现出自由得分的最高增长，因此，即使在一个国家在政治过渡后没有完全民主化的情况下，平均而言，与其他类型的过渡相比，非暴力公民抵抗还是获得民主收益增加最多的主要原因。
  </p>
  <p class="graf graf--p">
   2、Chenoweth 和 Stephan（2011）
   <a class="markup--anchor markup--p-anchor" data-href="https://cup.columbia.edu/book/why-civil-resistance-works/9780231156820" href="https://cup.columbia.edu/book/why-civil-resistance-works/9780231156820" rel="noopener noreferrer" target="_blank">
    分析了
   </a>
   1900年至2006年之间323次暴力和非暴力运动的数据，结果表明：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    在一次成功的运动结束后的五年内，对于取得民主结果的可能性，非暴力的运动 (57%) 几乎是暴力抵抗运动(6%)的十倍。
   </li>
   <li class="graf graf--li">
    即使在非暴力抵抗运动未能实现其既定目标的情况下，五年后，一个国家仍然有35％的机会实现民主结果。这表明，
    <strong class="markup--strong markup--li-strong">
     即使某运动未能实现其近期目标，其抵抗运动固有的动力也可能有助于民主化
    </strong>
    。相比之下，暴力抵抗运动失败后只有4％的机会获得这种结果。
   </li>
  </ul>
  <p class="graf graf--p">
   3、研究人员 Celestino 和 Gleditsch（
   <a class="markup--anchor markup--p-anchor" data-href="http://journals.sagepub.com/doi/abs/10.1177/0022343312469979" href="http://journals.sagepub.com/doi/abs/10.1177/0022343312469979" rel="noopener noreferrer" target="_blank">
    2013）
   </a>
   使用了由 Chenoweth 和 Stephan 创建的非暴力和暴力运动与结果数据集（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.du.edu/korbel/sie/research/chenow_navco_data.html" href="https://www.du.edu/korbel/sie/research/chenow_navco_data.html" rel="noopener noreferrer" target="_blank">
    NAVCO
   </a>
   ），研究了非暴力运动对独裁制到民主制过渡的影响，并得出结论，
   <strong class="markup--strong markup--p-strong">
    “非暴力抗议运动大大增加了过渡到民主的可能性。”
   </strong>
  </p>
  <p class="graf graf--p">
   他们还发现，
   <strong class="markup--strong markup--p-strong">
    “暴力直接行动在破坏独裁统治的整体方面效果不佳，而向新独裁政权过渡的可能性更大。”
   </strong>
  </p>
  <p class="graf graf--p">
   4、Bayer、Bethke 和 Lambach（2016）的
   <a class="markup--anchor markup--p-anchor" data-href="http://journals.sagepub.com/doi/abs/10.1177/0022343316658090" href="http://journals.sagepub.com/doi/abs/10.1177/0022343316658090" rel="noopener noreferrer" target="_blank">
    研究证明
   </a>
   ，当非暴力抵抗是政治过渡的主要推动力时，民主制度在冲突后的稳定和持久性将大大提高。
  </p>
  <p class="graf graf--p">
   在暴力抵抗运动导致政治过渡的国家，冲突后的民主政府的平均成活率为五年。相比之下，经历了过渡而根本没有基层抵抗的政府的平均成活率为九年。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    由非暴力抵抗运动引导的民主政府的存活率为：47年。
   </strong>
  </p>
  <p class="graf graf--p">
   换句话说，如果政治变革是由非暴力运动推动的话，民主的生存时间会更长，民主倒退的机会将大大减少。
  </p>
  <p class="graf graf--p">
   5、Bethke 和 Pinckney（2016）
   <a class="markup--anchor markup--p-anchor" data-href="https://www.v-dem.net/files/45/Users%20Working%20Paper%203.pdf" href="https://www.v-dem.net/files/45/Users%20Working%20Paper%203.pdf" rel="noopener noreferrer" target="_blank">
    的研究
   </a>
   利用1945年至2010年间101个政权的样本，并采用精密的方法来隔离导致民主化的其他因素，他们发现，
   <strong class="markup--strong markup--p-strong">
    民主的质量（主要在言论自由领域）得到了显著改善。相对于没有非暴力抵抗的情况，只有通过非暴力抵抗激发的变革才能有这样的结果。
   </strong>
  </p>
  <p class="graf graf--p">
   6、Pinckney（2014）
   <a class="markup--anchor markup--p-anchor" data-href="http://digitalcommons.du.edu/cgi/viewcontent.cgi?article=1516&amp;context=etd" href="http://digitalcommons.du.edu/cgi/viewcontent.cgi?article=1516&amp;context=etd" rel="noopener noreferrer" target="_blank">
    在一篇硕士论文中
   </a>
   的研究显示，由非暴力运动推动的政治过渡机制（例如谈判和选举）导致未来的民主率比诸如政变或外部军事干预等暴力过渡机制
   <strong class="markup--strong markup--p-strong">
    高出一倍
   </strong>
   。
  </p>
  <h3>
   <img alt="" class="aligncenter size-full wp-image-13571 jetpack-lazy-image" data-lazy-sizes="(max-width: 1050px) 100vw, 1050px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-6.jpeg?resize=1050%2C701&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-6.jpeg?w=1050&amp;ssl=1 1050w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-6.jpeg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-6.jpeg?resize=1024%2C684&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-6.jpeg?resize=768%2C513&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-6.jpeg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="701" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-6.jpeg?resize=1050%2C701&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1050"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13571" data-recalc-dims="1" height="701" sizes="(max-width: 1050px) 100vw, 1050px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-6.jpeg?resize=1050%2C701&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-6.jpeg?w=1050&amp;ssl=1 1050w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-6.jpeg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-6.jpeg?resize=1024%2C684&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-6.jpeg?resize=768%2C513&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-6.jpeg?resize=272%2C182&amp;ssl=1 272w" width="1050"/>
   </noscript>
  </h3>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    公民抵抗与经济和社会发展
   </strong>
  </h3>
  <p class="graf graf--p">
   7、
   <a class="markup--anchor markup--p-anchor" data-href="http://onlinelibrary.wiley.com/doi/10.1111/j.1468-0130.2010.00643.x/full" href="http://onlinelibrary.wiley.com/doi/10.1111/j.1468-0130.2010.00643.x/full" rel="noopener noreferrer" target="_blank">
    Johnstad（2010）
   </a>
   的一项研究从自由之家的数据中得出，发现在随后的过渡时期，非暴力抵抗运动的国家比自上而下由当权者主导的政治过渡时期，更有可能实现更高的经济增长。
  </p>
  <p class="graf graf--p">
   8、为了进一步了解社会经济影响， Stoddard（2013）
   <a class="markup--anchor markup--p-anchor" data-href="https://www.stabilityjournal.org/articles/10.5334/sta.bx/" href="https://www.stabilityjournal.org/articles/10.5334/sta.bx/" rel="noopener noreferrer" target="_blank">
    的研究
   </a>
   专注于预期寿命，作为该国总体健康和生活状况的指标，在​​成功的非暴力运动和暴力运动结束五到十年后。
  </p>
  <p class="graf graf--p">
   她发现，所有受影响的国家的预期寿命都在最初出现了下降，但是，在十年之内，通过非暴力反抗而上台的政府能够做出积极的改变，帮助其人口赶上甚至超过了世界平均寿命预期。
  </p>
  <p class="graf graf--p">
   这八项研究的结果证实，总体而言，公民抵抗运动推动了民主化。
  </p>
  <p class="graf graf--p">
   大量的历史案例给这方面带来了希望的有力证据，也包含关于公民抵抗运动潜在的 “大规模沦为政治暴力的风险” 的有力说法，需要根据大量案例中的其余数据所显示的内容来考虑(Della Porta, 2017)。对于阿拉伯之春运动的有限例子，也应谨慎对待。
  </p>
  <p class="graf graf--p">
   同时，公民抗动运动的确带有风险（例如1979年的伊朗非暴力革命）。
  </p>
  <p class="graf graf--p">
   实际上，Chenoweth 和 Stephan（2011）发现，运动结束后的十年内，非暴力运动导致内战的可能性为28％。尽管可能性仍很大，但是
   <strong class="markup--strong markup--p-strong">
    这种可能性明显低于暴力叛乱后发生内战的机会。
   </strong>
  </p>
  <p class="graf graf--p">
   这些新研究的关键线索可能在于，
   <strong class="markup--strong markup--p-strong">
    公民抵抗运动期间各运动进行的内部纪律、计划和战略制定以及体制建设，如何能够进一步增加其以后成功进行民主过渡的机会，同时最大程度地降低未来引发内战的可能性。
   </strong>
  </p>
  <p class="graf graf--p">
   👉 于是问题在于：从长远来看，如何改变运动方式以增加产生积极和平变化的可能性。
  </p>
  <p>

   </noscript>
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    为什么有些运动无法带来积极的结果，又该如何改变？
   </strong>
  </h3>
  <p class="graf graf--p">
   在某些情况下，非暴力运动成功地结束了现任独裁者的统治，⚠️ 但无法巩固收益，反而使局势恶化。
  </p>
  <p class="graf graf--p">
   可以在三种类型的运动中观察到这种影响：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、邪恶运动，或受到邪恶变革推动者严重影响的运动。
   </strong>
  </p>
  <p class="graf graf--p">
   这种运动至少在起步阶段是非暴力的 —— 由于其表面上是和平的行动。
  </p>
  <p class="graf graf--p">
   但是，它们的基本议程和目标是基于和促进不容忍、歧视和暴力。
  </p>
  <p class="graf graf--p">
   例如，德国佩吉达（Pegida）的反难民抗议运动；美国的右翼运动；Ayatollah Khomenei 作为1979年伊朗非暴力革命的象征领袖；或1934年纳粹对犹太商店和企业的非暴力抵制。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、胜利后没有足够能力维持积极变化的良性运动。
   </strong>
  </p>
  <p class="graf graf--p">
   例如，2004年乌克兰的橙色革命迫使库奇马政权接受重新举行总统选举。选举反过来确保了反对派候选人的胜利 —— 结果只是革命后的乌克兰的改革停滞不前，威权主义随之而来，大规模的政治腐败很快又重新出现。
  </p>
  <p class="graf graf--p">
   在2011年的一次民众起义中，埃及人确立了透明、团结、民主和平等的原则和规范，这些大原则指导着他们占领塔里尔广场长达11天。但是，他们
   <strong class="markup--strong markup--p-strong">
    未能
   </strong>
   确保自己的理想也能在整个埃及的工作场所、大学、公共机构、和农村社区中存活和塑造现实。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、战略上有缺陷的运动
   </strong>
  </p>
  <p class="graf graf--p">
   这些运动带有崇高和进步的思想，但实现它们的战略手段却与积极目标的本质相矛盾。
  </p>
  <p class="graf graf--p">
   至少可以识别出两类这种情况的运动：
  </p>
  <p class="graf graf--p">
   ▶️
   <strong class="markup--strong markup--p-strong">
    与反民主人士勾结的运动
   </strong>
  </p>
  <p class="graf graf--p">
   一个典型的例子是埃及的 Tamarod，他们收集了超过2000万个签名，以支持穆尔西总统的免职，并在2013年6月引发了大规模的反政府抗议活动。
  </p>
  <p class="graf graf--p">
   该运动在战略上与埃及军队保持一致，并支持了导致穆尔西垮台的2013年政变。
  </p>
  <p class="graf graf--p">
   然而，此后的当年8月，穆斯林兄弟会支持者遭到屠杀，埃及的专制政权最终得以重建。
  </p>
  <p class="graf graf--p">
   ▶️
   <strong class="markup--strong markup--p-strong">
    使用暴力（甚至只是小规模暴力）来挑战压迫的运动
   </strong>
  </p>
  <p class="graf graf--p">
   总部位于美国的反法西斯运动，愿意对极右翼活动分子采取暴力行动，并以制止法西斯主义的崇高理由为其辩解，尽管
   <a class="markup--anchor markup--p-anchor" data-href="https://www.revealnews.org/episodes/street-fight-a-new-wave-of-political-violence/" href="https://www.revealnews.org/episodes/street-fight-a-new-wave-of-political-violence/" rel="noopener noreferrer" target="_blank">
    它也承认
   </a>
   种族主义和暴力极端主义的最终失败需要来自政治群众的动员和参与。
  </p>
  <p class="graf graf--p">
   由于这些缺陷考虑了运动本身的实际特性，因此，如果运动的内部工作方式发生了变化，它们的轨迹实际上就很可能会发生变化。
  </p>
  <p class="graf graf--p">
   换句话说，
   <strong class="markup--strong markup--p-strong">
    活动家具有一定程度的代理权并能控制这些变化
   </strong>
   。
  </p>
  <p class="graf graf--p">
   👉那么，活动家需要注意哪些因素，以增加内在抵抗力长期产生积极变化的可能性，以及如何做到？
  </p>
  <p class="graf graf--p">
   分析认为，有关运动的两个方面在起作用：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、尊重人权
   </strong>
  </p>
  <p class="graf graf--p">
   👌 将运动的行为建立在尊重普遍人权的基础上，可以帮助运动坚持非暴力纪律，并在斗争中和斗争后预防或减少暴力。
  </p>
  <p class="graf graf--p">
   Elizabeth A. Wilson 在她最近出版的著作《People Power Movements and International Human Rights》中将其称为 “人权精神”。她确定了非暴力运动可以用来衡量其人权精神的四项原则：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    不歧视，体现为对他人的平等对待
   </li>
   <li class="graf graf--li">
    不压迫，其目标是促进所有人的政治权利和政治自治
   </li>
   <li class="graf graf--li">
    不剥削，表现为与受迫害者团结一致，提供互助，并接触社会各个阶层，以建立共同的理解和信任
   </li>
   <li class="graf graf--li">
    非暴力策略，体现为不威胁或不伤害他人的行为。
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、建设性抵抗的长期策略
   </strong>
  </p>
  <p class="graf graf--p">
   👌 为了维持自己的利益并增强执行持久积极变革的能力，各运动都必须秉持
   <strong class="markup--strong markup--p-strong">
    长期建设性抵抗
   </strong>
   的战略，以尽可能压制破坏性战略。
  </p>
  <p class="graf graf--p">
   即：以建立网络和联盟为基础，并建立替代和并行的流程，实践，规范和机构。
  </p>
  <p class="graf graf--p">
   👉 运动可以通过建立自力更生、自我组织和自治的抵制来增强其维持胜利的能力，
   <strong class="markup--strong markup--p-strong">
    并在该现实成为国家实践的新规范之前就构建一个新的积极的社会和政治现实。
   </strong>
  </p>
  <p class="graf graf--p">
   民间的基层机构为遵循更可行和更民主的过渡奠定了基础。
  </p>
  <p class="graf graf--p">
   👉 他们包括地下自由新闻和未经审查的通讯网络，替代性的学校和教育系统，经济合作社，以及平行的文化机构，这些都是在抵抗压迫期间建立的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种建设性的抵抗也有助于在过渡之前建立积极的社会资本
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在公民反抗精神的支持下，为有需要的人和受压迫者提供各种服务的密集协作网络，有利于发展真正的公共服务和公共问责文化。还可以帮助减少随后的过渡期间的腐败。
  </p>
  <p class="graf graf--p">
   反抗是一项系统工作，它需要丰富的知识、技能、毅力和精神、组织协作能力和不断学习的意识。我们要的是民主的世界，给我们自己和后代带来更美好的生活，它是为终极目标，它应该指导我们的一切行动策略。
  </p>
  <p class="graf graf--p">
   这就是为什么IYP要支持直接行动，而非诉求。
  </p>
  <p class="graf graf--p">
   希望这篇文章能有所帮助。⚪️
  </p>
