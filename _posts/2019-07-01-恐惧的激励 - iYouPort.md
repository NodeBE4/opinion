  <h1 class="entry-title">
   恐惧的激励
  </h1>
 </header>

  <span class="cat-links">

   <a href="https://www.iyouport.org/category/%e5%ae%a1%e6%9f%a5%e5%92%8c%e8%87%aa%e6%88%91%e5%ae%a1%e6%9f%a5/" rel="category tag">
    审查和自我审查
   </a>
  </span>
  <span class="published-on">
   <time class="entry-date published" datetime="2019-07-02T00:03:11+08:00">

   0 Minutes
  </span>
 </div>
 <div class="entry-content">
  <p class="graf graf--p">
   俄罗斯在互联网上打击任何言论自由的努力仍然有增无减。普京政府在过去几年中已经有效地使
   <a class="markup--anchor markup--p-anchor" data-href="https://www.techdirt.com/articles/20170609/09213837550/russia-stumbles-forth-quest-to-ban-vpns-private-messenger-apps.shtml" href="https://www.techdirt.com/articles/20170609/09213837550/russia-stumbles-forth-quest-to-ban-vpns-private-messenger-apps.shtml" rel="noopener noreferrer" target="_blank">
    VPN 和加密通讯应用程序非法化
   </a>
   了。虽然政府公开坚称这些措施对于“保护国家安全”是必要的，但实际的激励因素与我们几十年来在美国和世界其他地方看到的那些古老的激励因素一样：那就是，恐惧和控制。俄罗斯不希望人们私下组织、讨论或挑战政府越来越专制的状态。
  </p>
  <p class="graf graf--p">
   在瞄准 VPN 之后，普京3月又签署了两项新的法案，这些法案极大地阻碍了言论，特别是在线言论。
  </p>
  <p class="graf graf--p">
   其中一项法律专门针对所谓的“虚假新闻”这一模糊概念，特别惩罚任何“公然不尊重俄罗斯社会、政府、官方标志、宪法或政府机构”的在线资料。换句话说，俄罗斯希望禁止对普京及其腐败政府的任何批评，专家告诉“华盛顿邮报”：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “检察官可以向国家提出有关在线媒体的投诉，如果不删除违规材料，当局可以阻止访问网站。专家说，这是新的规定。 “检察官办公室现在可以在司法判决之前阻止这些新闻来源。
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     它为检察官办公室提供了极高的权力，几乎完全取消了俄罗斯法院
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    ，“欧洲政策分析中心的兼职研究员 Maria Snegovaya 在一封电子邮件中写道
   </em>
   。
   <em class="markup--em markup--p-em">
    “换句话说，它显著扩大了俄罗斯镇压机器的镇压力量。已经可以与斯大林的“三驾马车”进行比较，“她补充说
   </em>
   。
  </p>
  <p class="graf graf--p">
   现在在俄罗斯传播所谓“虚假新闻”的网站（翻译一下就是批评普京及其流氓寡头联盟的任何内容）会突然面临高达150万卢布（22,900美元）的罚款。
  </p>
  <p class="graf graf--p">
   普京同时签署的另一项配套法案同样存在问题; 它将更新现有法律，将侮辱俄罗斯政府或政治领导人的行为定为犯罪。重复违反该法律的人将面临高达300,000卢布（4,700美元）的罚款 — — 以及15天的监禁：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “修改现有信息法案的新法案在不到两个月的时间内就以压倒多数的支持率通过了俄罗斯议会两院。观察员和一些立法者批评立法的模糊性语言和扼杀言论自由的可能性。该立法将对传播 “明显不尊重俄罗斯社会、政府、官方政府标示、宪法或政府机构”的信息进行惩罚
   </em>
   。
  </p>
  <p class="graf graf--p">
   尽管遭到了广泛的谴责，并且有100多名记者和学者将这一提议视为愚蠢的威权主义，但这两项法律更新在不到两个月内迅速通过了俄罗斯议会。
   <strong class="markup--strong markup--p-strong">
    这是一个可怕的升级，来自一个长期在网上推销仇恨和愚蠢的虚假信息的政府，同时假装对真实的新闻报道和批判性思维发动战争
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这显然是普京希望出口到众多国家的典范。
  </p>
  <p class="graf graf--p">
   虽然俄罗斯将此作为一种力量的表现，但它实际上是一种恐惧的表现。这与俄罗斯脱离互联网的努力相结合，使人们非常清楚政府不仅害怕言论自由，还害怕自己的人民。俄罗斯潜在的经济问题非常明显，普京不希望这些状况在网上被突出显示。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    T
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.techdirt.com/articles/20190319/07324341822/terrified-internet-putin-signs-laws-making-it-illegal-to-criticize-government-leaders-online.shtml" href="https://www.techdirt.com/articles/20190319/07324341822/terrified-internet-putin-signs-laws-making-it-illegal-to-criticize-government-leaders-online.shtml" rel="noopener noreferrer" target="_blank">
    errified Of The Internet, Putin Signs Laws Making It Illegal To Criticize Government Leaders Online
   </a>
   . This, combined with Russia’s efforts to disconnect itself from the internet makes it abundantly clear how afraid the government is of not only free speech, but its own people too.
  </p>
