  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    恶意闪存驱动器
   </strong>
  </h3>
  <p class="graf graf--p">
   Elliot 在首次入侵监狱的尝试中，他让 Darlene 假装 “意外” 将受感染的闪存驱动器丢到监狱外面。
  </p>
  <p class="graf graf--p">
   这里的策略是，如果监狱网络中的某人将其捡起、并将它插入他们自己的计算机系统中，就可以注入恶意软件为 Elliot 提供外部连接。
  </p>

  <p class="graf graf--p">
   正如预期的那样，一个昏昏欲睡的狱警确实捡起了那个USB并将其插入了自己的计算机。理论上 Elliot 可以建立SSH连接，但是在执行任何操作之前，AV软件会检测到它并断开 Elliot 的连接。
  </p>
  <p class="graf graf--p">
   Elliot 指责 Darlene 不懂行，仅仅使用 “ Rapid9” 中的著名恶意软件，而不是开发新的利用方法，而 Darlene 为自己辩护说 “我只有一​​个小时。（她可能用 Veil-Evasion 重新编码了，也许不会被检测的）
  </p>
  <p class="graf graf--p">
   一些人质疑这种方法是否行得通。在现代操作系统上禁用自动运行功能之前，您应该会在闪存驱动器上有一个可以自动执行的EXE文件。（在现代操作系统上，默认情况下禁用自动运行）
  </p>
  <p class="graf graf--p">
   我们可能会假设这台机器启用了自动运行功能，或者更可能是 Darlene 已将恶意软件安装在已重新编程为模拟USB键盘的闪存驱动器上。
  </p>
  <p class="graf graf--p">
   也就是说，将闪存驱动器安装在系统上后，操作系统会将它识别为USB键盘，授予其已登录用户的权限，然后将其恶意代码注入操作系统。
  </p>
  <p class="graf graf--p">
   因此，如果 Darlene 用 Veil-Evasion 重新编码了恶意软件，那么此方法可能行得通。
  </p>

   <img alt="" class="aligncenter size-full wp-image-12609 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpg?resize=1100%2C694&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpg?w=1117&amp;ssl=1 1117w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpg?resize=300%2C189&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpg?resize=1024%2C646&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpg?resize=768%2C485&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpg?resize=1100%2C694&amp;ssl=1 1100w" data-recalc-dims="1" height="694" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpg?resize=1100%2C694&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

    <img alt="" class="aligncenter size-full wp-image-12609" data-recalc-dims="1" height="694" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpg?resize=1100%2C694&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpg?w=1117&amp;ssl=1 1117w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpg?resize=300%2C189&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpg?resize=1024%2C646&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpg?resize=768%2C485&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpg?resize=1100%2C694&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   这促使 Elliot 采取了一种新策略，即 入侵蓝牙，并通过警车与监狱之间的专用蜂窝连接进入监狱的计算机系统。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    破解蓝牙键盘
   </strong>
  </h3>
  <p class="graf graf--p">
   Elliot 在这里使用的策略是欺骗警车的蓝牙连接到他的键盘。
  </p>
  <p class="graf graf--p">
   如果他能使笔记本电脑相信他自己的键盘实际上是警察的键盘，那么他就可以控制警察的笔记本电脑并进入监狱的网络。
  </p>
  <p class="graf graf--p">
   一旦进入网络，他就可以上传恶意软件以控制监狱的数字控制系统。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第一步：启用蓝牙
   </strong>
  </h3>
  <p class="graf graf--p">
   首先要做的就是在他的 Linux 黑客系统上启动蓝牙服务：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    kali &gt; service bluetooth start
   </strong>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12610 jetpack-lazy-image" data-lazy-sizes="(max-width: 533px) 100vw, 533px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-7.jpg?resize=533%2C39&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-7.jpg?w=533&amp;ssl=1 533w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-7.jpg?resize=300%2C22&amp;ssl=1 300w" data-recalc-dims="1" height="39" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-7.jpg?resize=533%2C39&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="533"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12610" data-recalc-dims="1" height="39" sizes="(max-width: 533px) 100vw, 533px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-7.jpg?resize=533%2C39&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-7.jpg?w=533&amp;ssl=1 533w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-7.jpg?resize=300%2C22&amp;ssl=1 300w" width="533"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   接下来，他需要激活蓝牙设备：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    kali &gt; hciconfig hci0 up
   </strong>
  </p>
  <p class="graf graf--p">
   然后，通过键入以下内容来检查它是否真正起作用以及其属性：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    kali &gt; hciconfig hci0
   </strong>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12611 jetpack-lazy-image" data-lazy-sizes="(max-width: 645px) 100vw, 645px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-5.jpg?resize=645%2C133&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-5.jpg?w=645&amp;ssl=1 645w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-5.jpg?resize=300%2C62&amp;ssl=1 300w" data-recalc-dims="1" height="133" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-5.jpg?resize=645%2C133&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="645"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12611" data-recalc-dims="1" height="133" sizes="(max-width: 645px) 100vw, 645px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-5.jpg?resize=645%2C133&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-5.jpg?w=645&amp;ssl=1 645w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-5.jpg?resize=300%2C62&amp;ssl=1 300w" width="645"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   请注意第二行中的 “BD Address”，这是蓝牙设备的MAC地址。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第二步：扫描蓝牙设备
   </strong>
  </h3>
  <p class="graf graf--p">
   如果仔细查看 Elliot 的屏幕，您会发现他正在使用 hcitool，这是 Kali Linux 中内置的蓝牙工具。
  </p>
  <p class="graf graf--p">
   尽管此方法有效，但使用 btscanner 可以更容易，btscanner 是具有基本GUI的内置蓝牙扫描仪。要使用它，请简单输入：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    kali &gt; btscanner
   </strong>
  </p>
  <p class="graf graf--p">
   然后选择 “i” 以启动查询扫描。您可以在下面看到结果。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12612 jetpack-lazy-image" data-lazy-sizes="(max-width: 728px) 100vw, 728px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7-6.jpg?resize=728%2C355&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7-6.jpg?w=728&amp;ssl=1 728w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7-6.jpg?resize=300%2C146&amp;ssl=1 300w" data-recalc-dims="1" height="355" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7-6.jpg?resize=728%2C355&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="728"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12612" data-recalc-dims="1" height="355" sizes="(max-width: 728px) 100vw, 728px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7-6.jpg?resize=728%2C355&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7-6.jpg?w=728&amp;ssl=1 728w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7-6.jpg?resize=300%2C146&amp;ssl=1 300w" width="728"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   使用 btscanner，可以获得范围内所有蓝牙设备的列表。这里有一个MAC地址和一个“Tyler”的名字 —— 要欺骗该设备就必须欺骗MAC地址和设备名。
  </p>
  <p class="graf graf--p">
   这就是 Elliot 如何获取警车中MAC地址和蓝牙设备名的方式。请记住，蓝牙是一种低功耗协议，范围仅约10米（使用定向天线最大距离达到100米）。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第三步：欺骗键盘的MAC地址
   </strong>
  </h3>
  <p class="graf graf--p">
   现在，Elliot 有了警察键盘的名称和MAC地址，他将需要通过使用此信息克隆警察键盘来进行欺骗。
  </p>
  <p class="graf graf--p">
   Kali Linux 有一个旨在欺骗蓝牙设备的工具，称为 spooftooph。可以使用它来欺骗键盘，类似于以下命令：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    kali &gt; spooftooph -i hci0 -a A0:02:DC:11:4F:85 -n Car537
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     -i
    </strong>
    指定设备，在这种情况下为 hci0
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     -a
    </strong>
    指定要欺骗的MAC地址
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     -n
    </strong>
    指定要欺骗的设备名，在这种情况下为 “Car537”
   </li>
  </ul>
  <p class="graf graf--p">
   如果做对了，就是下面这样：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12613 jetpack-lazy-image" data-lazy-sizes="(max-width: 578px) 100vw, 578px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/8-4.jpg?resize=578%2C110&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/8-4.jpg?w=578&amp;ssl=1 578w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/8-4.jpg?resize=300%2C57&amp;ssl=1 300w" data-recalc-dims="1" height="110" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/8-4.jpg?resize=578%2C110&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="578"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12613" data-recalc-dims="1" height="110" sizes="(max-width: 578px) 100vw, 578px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/8-4.jpg?resize=578%2C110&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/8-4.jpg?w=578&amp;ssl=1 578w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/8-4.jpg?resize=300%2C57&amp;ssl=1 300w" width="578"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   为了检查是否成功，可以使用 hciconfig，如果成功，它将具有与警察的蓝牙设备相同的MAC地址和名称。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    kali &gt; hciconfig hci0 name
   </strong>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12614 jetpack-lazy-image" data-lazy-sizes="(max-width: 637px) 100vw, 637px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/9-5.jpg?resize=637%2C96&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/9-5.jpg?w=637&amp;ssl=1 637w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/9-5.jpg?resize=300%2C45&amp;ssl=1 300w" data-recalc-dims="1" height="96" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/9-5.jpg?resize=637%2C96&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="637"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12614" data-recalc-dims="1" height="96" sizes="(max-width: 637px) 100vw, 637px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/9-5.jpg?resize=637%2C96&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/9-5.jpg?w=637&amp;ssl=1 637w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/9-5.jpg?resize=300%2C45&amp;ssl=1 300w" width="637"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   现在已经有了一个蓝牙设备，它是警车的蓝牙键盘的完美克隆。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第四步：时间……
   </strong>
  </h3>
  <p class="graf graf--p">
   现在出现了现实与 Mr. Robot 剧情发生分歧的地方。这部剧整体上很真实，但导演也的确加入了艺术化成分，这当然是允许的。创意作品不应受到现实的限制。
  </p>
  <p class="graf graf--p">
   为了让 Eliot 现在连接到警车的笔记本电脑，他需要在键盘和笔记本电脑上的蓝牙适配器之间进行交换的链路密钥（就是识别先前配对的蓝牙设备的钥匙）。
  </p>
  <p class="graf graf--p">
   他可以猜测或破解它，但是肯定不会像剧情中显示的那样快。另一种可能性是，当系统重启或键盘断开连接时，Elliot 可以连接到笔记本电脑，因为它是警察键盘的克隆。
  </p>
  <p class="graf graf--p">
   无论哪种情况，黑入警察的蓝牙键盘都需要比剧情中显示的更多的时间。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第五步：入侵监狱系统
   </strong>
  </h3>
  <p class="graf graf--p">
   在最后一步中，Elliot 使用警察的被黑的计算机通过FTP上传恶意软件，这将使他能够控制监狱牢房的门。
  </p>
  <p class="graf graf--p">
   也许很少有人会意识到监狱和其他通常称为SCADA（数据采集与监控系统）的工业系统很容易被黑客入侵。而这种事其实经常发生。
  </p>
  <p class="graf graf--p">
   针对伊朗铀浓缩设施的震网病毒袭击与此非常相似。这些工业系统都具有基本上是数字控制器的PLC。
  </p>
  <p class="graf graf--p">
   据推测，这所监狱中有PLC控制监狱牢房的门（这是一个非常合理的假设），Elliot 的恶意软件感染并控制了它们，使他能够打开所有牢房，释放了 Vera 和所有其他囚犯。⚪️
  </p>
