   ，这些小游戏将非常适合您开拓思路，不论是调查挖掘还是验证信息，都需要一个好的思考方式；尤其是对OSINT初学者来说，构建合理的思考方式比获得技术工具更重要。

   快速浏览一下法兰克福的Eintracht赛程列表显示，尽管法兰克福队通常在德甲联赛中打大部分比赛，但他们本赛季也一直参加欧洲联赛。
  </p>
  <p class="graf graf--p">
   此外，他们本赛季仅打了14场比赛，其中一半是客场（球队名单左侧的队伍总是主队）。这意味着已经可以将可能的场合缩小到仅仅7场比赛。
  </p>
  <p class="graf graf--p">
   照片拍摄时球队还没有出场，于是无法知道这场比赛的对手是谁。照片左上角走进球场的人不是球员，很明显。
  </p>
  <p class="graf graf--p">
   还有一些其他独特的特征需要考虑：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    人群中的金色和黑色旗帜。
   </li>
   <li class="graf graf--li">
    右边的横幅上写着“wir stehen ……”（德语，大概是“我们大胜”的意思）
   </li>
   <li class="graf graf--li">
    球场左侧上的物体可能是某种横幅或标志。
   </li>
  </ul>
  <p class="graf graf--p">
   我决定从法兰克福最近的欧洲足联比赛开始向前推进，直到找到能够帮助确定这场比赛的确切时间的细节。
  </p>
  <p class="graf graf--p">
   和很多照片定位挑战一样，很难直接通过图中呈现的线索找到准确位置。于是不如从其他角度下手：这里尝试首选新闻和YouTube，而不是照片本身。
  </p>
  <p class="graf graf--p">
   如上这里检查的第一次比赛是法兰克福对切尔西，2/5/19：
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="393" src="//www.youtube.com/embed/cmAvYFgge28" width="700">
   </iframe>
  </p>
  <p class="graf graf--p">
   见上面视频。开球前大量的体育场镜头都非常有用。但是没有任何金色和黑色旗帜的迹象，或者“wir stehen ……”的旗帜。
  </p>
  <p class="graf graf--p">
   很简单不是这里。
  </p>
  <p class="graf graf--p">
   那么继续尝试：Eintracht Frankfurt v Benfica 18/4/19:

   这个视频的相似性就比较多了；但是依旧没有找到金色旗帜，也看不到“wir stehen”横幅。
  </p>
  <p class="graf graf--p">
   当然，不可否认，这段视频只有几秒钟，并不是100％的结论，但在完全否定之前，需要继续检查欧足联之前的比赛。
  </p>
  <p class="graf graf--p">
   就是这个了。只需要看前五秒就知道这是正确的：

  </p>
  <p class="graf graf--p">
   这也表明，在照片中左上角走进球场的人不是球员，而是将大型UEFA横幅带到场地上的啦啦队。
  </p>
  <p class="graf graf--p">
   这也有助于确定时间。因为开赛时间是确定的。
  </p>
  <p class="graf graf--p">
   再看一下
   <a class="markup--anchor markup--p-anchor" data-href="https://spieledb.eintracht.de/spielplan/europa-league" href="https://spieledb.eintracht.de/spielplan/europa-league" rel="noopener noreferrer" target="_blank">
    赛事列表
   </a>
   ，启动时间是2019年3月7日18:55。这必定意味着照片是在开球前拍摄的，可能是在18：45–18：50左右。
  </p>
  <p class="graf graf--p">
   现在找到门路了。开始做一些图像搜索，验证一下在视频中找到的东西。
  </p>

  <p class="graf graf--p">
   通过将所有照片细节与体育场座位计划图进行比较，看起来 Lars 坐在41K、41F或40D区域。

  <p class="graf graf--p">
   Lars 确认他坐在40D区。
  </p>
  <p class="graf graf--p">
   这个挑战中也呈现了最常用的思考方式，就如我们曾经演示的大多数深度调查中所展现的那样：首先抓取几个主要观测点，然后开始与其他来源进行对照，继续提出合理的假设，验证这些假设，最终接近最合理的那一个。
  </p>
  <p class="graf graf--p">
   在这里看到我们曾经介绍过的详细思路，关于
   <span style="color: #993366;">
    《

     什么是调查，如何验证
    </a>

   </span>
   。
  </p>
  <p class="graf graf--p">
   Qiztime 挑战于2017年开始，当时德国记者 Julia Bayer 在她的 Twitter 帐户上发布了一张照片，显示科隆一个多雨的十字路口，并问道：“我在哪里？找到确切的位置”。这是她给学员记者在培训课上的练习。但就是因为她贴出来了，引起了一大群人的兴趣，人们纷纷加入了挑战游戏。
  </p>

   从那之后 Bayer 每周都会发布一个测试；并邀请粉丝们也这样做。粉丝中包括一些调查记者、公民记者、还有OSINT专家；他们希望能磨练自己的技巧，都是对真相感兴趣的人。
  </p>
  <p class="graf graf--p">
   这些帖子的评论非常有趣，人们纷纷写下自己挖到的线索，
   <strong class="markup--strong markup--p-strong">
    线索经常不是唯一的，结论将通过多个线索综合得出
   </strong>
   ，发布者也会和回复者互动，以纠正那些跑偏了的人。于是你会看到一个非常有趣的探讨场面。
  </p>
  <p class="graf graf--p">
   找到答案的人会单独发帖，而不是在回复栏中，所以你任何时间加入都可以，不会被剧透；而且即使你没有时间参与游戏，也可以通过浏览留言发现有趣的思考方式。
  </p>
  <p class="graf graf--p">
   测验的难度各不相同。
   <strong class="markup--strong markup--p-strong">
