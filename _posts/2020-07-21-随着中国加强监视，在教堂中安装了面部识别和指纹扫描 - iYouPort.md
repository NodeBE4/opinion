  <p class="graf graf--p">
   包括面部和指纹扫描设备在内的高科技监视设备现已在中国各地的教堂中安装。
  </p>
  <p class="graf graf--p">
   根据探索中国人权和宗教自由的出版物
   <a class="markup--anchor markup--p-anchor" data-href="https://bitterwinter.org/want-to-enter-church-scan-your-fingerprints-and-face/" href="https://bitterwinter.org/want-to-enter-church-scan-your-fingerprints-and-face/" rel="nofollow noopener noreferrer" target="_blank">
    Bitter Winter
   </a>
   的报告，一种面部识别系统首先被放置在新疆维吾尔族首府乌鲁木齐的政府控制的教会 —— 三自教会中 —— 现在正在中国其他地区引入。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “去年10月6日，湖北黄石的牧羊堂二楼就设有两个生物识别设备。从那之后，信徒必须排成一列，被扫描面部和指纹，然后才被允许进入教堂。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “与此同时，在该市的国营天恩堂中，也安装了面部识别设备，以检查参加聚会的每一个信徒。”
  </p>
  <p class="graf graf--p">
   几周前，一名基督徒告诉该杂志，地方当局要求黄石市三自教会建立的每个聚会场所都必须扫描信徒的指纹，并存档他们的个人和家庭信息。
  </p>
  <p class="graf graf--p">
   上教堂的人们对该命令感到不安，因为它不仅要求对教会成员进行不断的跟踪和监视，而且还要求了解他们的家人和亲戚的所有联系。
  </p>
  <p class="graf graf--p">
   其他当地的信徒也报告说，自10月以来，黄石的多个国家批准的教堂开始使用指纹传感器和面部扫描仪记录出席的所有人。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “ 9月下旬，在黄石市南竹林社区的一个三自教友聚会场所的成员被记录下来。场地负责人告诉他们，所有人员都必须扫描指纹才能参加礼拜天的服务。”
  </p>
  <p class="graf graf--p">
   黄石市南竹林社区一个教堂的负责人向信徒解释说：“就像员工上班一样。通过这种方式，教会可以清楚地知道谁参加了礼拜，谁没有参加。”
  </p>
  <p class="graf graf--p">
   中国基督徒认为，
   <strong class="markup--strong markup--p-strong">
    这是政府权力不断增加的又一个例子
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在中国，国家对教堂的监视并不是什么新鲜事。所谓的 “雪亮工程计划” 长期存在于国营教堂中，甚至在某些礼拜场所的洗手间都装有摄像头，以确保 “全面监控”。
  </p>
  <p class="graf graf--p">
   《南华早报》说，监视计划的名称似乎源于毛泽东时代的口号，该口号旨在敦促人们谴责那些未能遵循共产党信条的人：“人民的眼睛是雪亮的。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    监视计划旨在覆盖所有地区，目标是到2020年实现 “无盲区监视”
   </strong>
   —— “覆盖所有区域，在所有网络之间共享，随时可用，并在所有地方受到控制”。
  </p>
  <p class="graf graf--p">
   去年早些时候有报道称，山东青岛市的国营教堂被勒令停止唱诗，而被命令唱那些经国家批准的赞美诗。新批准的歌曲仅以爱国、及尊重老人为主题。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “政府要求的赞美诗只宣传政治、世俗的内容。所有信徒都不愿唱。” 国家认可的教堂的一位负责人说，以其中一首赞美诗的合唱为例，歌词是：“中国是美丽的；中国很棒。中国的儿女爱中国。 ……上帝保佑中国。” ⚪️
  </p>
