   如您所知，防御是一切的基础。也是IYP的主题之一。
  </p>
  <p class="graf graf--p">
   依旧有很多人不甚了解这个互联网具有多么大的伤害性，于是我们
   <strong class="markup--strong markup--p-strong">
    采取了
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">

     激进的方法
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    —— 用演示追踪攻击来提醒防御
   </strong>
   。
  </p>
  <p class="graf graf--p">
   我知道这也许会吓到您，但恰好说明您已经开始警觉；
   <strong class="markup--strong markup--p-strong">
    请千万不要停留在警觉层面上，您应该快速采取行动，以避免恶性事件发生在您身上
   </strong>
   。
  </p>
  <p class="graf graf--p">
   当中国互联网用户看到张贾龙案中被起诉 “推特转发” 政府不喜欢的内容时，才意识中国当局的监控不仅限于墙内，而是全球。于是人们纷纷采取措施删除自己的旧推文。
  </p>
  <p class="graf graf--p">
   我们已经介绍过如何批量删除推文，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%b8%85%e6%b4%81%e5%b7%a5%e5%85%b7%ef%bc%9a%e8%87%aa%e5%8a%a8%e5%88%a0%e9%99%a4%e6%82%a8%e7%9a%84%e6%97%a7%e6%8e%a8%e6%96%87%ef%bc%8c%e4%bd%86%e6%82%a8%e8%a6%81%e4%bf%9d%e7%95%99%e7%9a%84%e6%8e%a8/" href="https://www.iyouport.org/%e6%b8%85%e6%b4%81%e5%b7%a5%e5%85%b7%ef%bc%9a%e8%87%aa%e5%8a%a8%e5%88%a0%e9%99%a4%e6%82%a8%e7%9a%84%e6%97%a7%e6%8e%a8%e6%96%87%ef%bc%8c%e4%bd%86%e6%82%a8%e8%a6%81%e4%bf%9d%e7%95%99%e7%9a%84%e6%8e%a8/" rel="noopener noreferrer" target="_blank">
    清洁工具：自动删除您的旧推文，但您要保留的推文除外
   </a>
   》。
  </p>
  <p class="graf graf--p">
   同类工具只能帮您完全删除一切，或者最多是定时删除所有；但您需要保留的内容也会随之不见。这个方法的优势就在于，它能让你选择性删除内容，不仅可以保留您希望保留的东西，
   <strong class="markup--strong markup--p-strong">
    尤其是，不会在监视者眼中看起来很可疑（如果您已成为目标的话）。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️但是，您必须知道，上述只是一小部分。
  </p>
  <p class="graf graf--p">
   如果您发布的内容曾经被病毒式转发，仅仅删推文可能是无效的，您的内容依旧被保留在互联网上。
  </p>
  <p class="graf graf--p">
   尤其是，推特对整个互联网世界来说只是一个小院；对您的威胁会来自整个世界。
  </p>
  <p class="graf graf--p">
   于是本文的重点就在于，帮您解决各种隐患。
  </p>
  <p class="graf graf--p">
   👉不论您是担心间谍、警察、网监、骇客、还是社交工程恶意行为者，或者仅仅是一个曾经说了后悔话的普通人，这些方法您都值得了解。
  </p>
  <p class="graf graf--p">
   您曾经想过从互联网上抹去自己泄漏过的愚蠢记录吗？您并不孤单 —— 因为 “从互联网上抹去自己99%内容” 的推特话题成为了病毒式传播的主题，短时间内就有
   <strong class="markup--strong markup--p-strong">
    十几万人转发
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Twitter 用户 Liam 介绍了一些简单的方法 —— 事实上
   <strong class="markup--strong markup--p-strong">
    这些方法我们全部介绍过
   </strong>
   ，只是解释得非常详细，以至于分布在很多篇文章中。本文算是一个简明扼要的总结，以便您可以看清楚应该怎么做。
  </p>

   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     第一步，您需要跟踪自己旧的电子邮件登录。
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   仔细查找您在过去十年中使用过的每一个电子邮件。如果您失去了访问权限，就需要想办法恢复它们。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     第二步，您需要仔细查看已注册的所有服务的电子邮件，从那些已经被遗忘的服务中删除旧的帐户。
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   使用电子邮件上的搜索功能，查找诸如“注册”或“欢迎”之类的短语；这样您会更容易找到那些东西。

   现在，通过搜索 “删除帐户” +“ &lt;服务&gt;”，在服务中查找删除帐户功能。
  </p>
  <p class="graf graf--p">
   👉如果没有找到，请通过 google 或各种其他搜索引擎寻找支持的电子邮件，以便联系平台请求他们删除您的帐户。
  </p>
  <p class="graf graf--p">
   对于某些服务，您可能希望在删除帐户之前清除所有内容和消息，因为该帐户可能已存档，并且骇客或外部实体很可能会在以后访问到此信息。
  </p>
  <p class="graf graf--p">
   这是要记住的事。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     第三步：检查您的信息是否已经被泄露。
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">

   <strong class="markup--strong markup--p-strong">
    布尔搜索
   </strong>
   的方法来正确使用Google，以定位这些信息。
  </p>

  </p>

   </li>
