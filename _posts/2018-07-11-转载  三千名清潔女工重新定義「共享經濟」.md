     <p class="graf graf--p graf-after--h3" id="81ae">
      十五年前，當多明格斯（
      <a class="markup--anchor markup--p-anchor" data-href="https://www.pdf-18.com/speaker/cirenia-dominguez/" href="https://www.pdf-18.com/speaker/cirenia-dominguez/" rel="noopener" target="_blank">

      </a>
      ）從墨西哥來到紐約，她想的是，能否多打掃幾戶人家，為孩子的教育多掙一些錢。
     </p>
     <p class="graf graf--p graf-after--p" id="be37">
      十五年後，在個人民主論壇的台上，她與臉書、微軟、美國官員同台，以創業家的身份，告訴人們經濟弱勢者怎麼逆轉生活，科技，如何創造更包容的經濟，「平台合作社主義」，是她找到的答案。

     <p class="graf graf--p graf--leading" id="ebab">
      川普上任要滿兩年，入夏的紐約法學院，科技人、記者、官員、學者、喜劇演員等，齊聚《個人民主論壇》，滿滿兩天的演講、對談、工作坊中，他們找一個答案，如何修補民主？
     </p>
     <p class="graf graf--p graf-after--p" id="543f">
      民主破了哪些洞，是個很大的題目。臉書的早期創辦團隊成員之一、歐巴馬選舉操盤手 Chris Hughes 在台上拋出
      <a class="markup--anchor markup--p-anchor" data-href="https://basicincome.org/news/2018/05/economic-security-for-all-questions-about-chris-hughes-guaranteed-income-proposal/" href="https://basicincome.org/news/2018/05/economic-security-for-all-questions-about-chris-hughes-guaranteed-income-proposal/" rel="noopener" target="_blank">
       「保障收入」（Guaranteed income）
      </a>
      作為解方，希望能解決貧富差距造成的社會崩解；同樣是歐巴馬任內的幕僚、官員則提出如何用科技解決政府效能問題，以及如何立出數據使用規範，提防人工智慧的不當發展，也有講者提醒，當一切聯網之後，政府可能對人民的大規模監控，是否進一步打壓弱勢族群的權益等；最終，壓軸的拉美裔喜劇演員
      <a class="markup--anchor markup--p-anchor" data-href="http://neginfarsad.com/" href="http://neginfarsad.com/" rel="noopener" target="_blank">
       Negin Farsad
      </a>
      ，以脫口秀的方式，以她自身的程長經歷，點出美國社會的大熔爐中，膚色如何成牆，那些衝突種子如何被埋下、又該如何被清除。
     </p>

     <p class="graf graf--p graf-after--figure" id="bf3d">
      今年的論壇主題定為「我們如何創善？（ How we make good?）」，菁英們針對經濟、文化到科技政策，提出不同觀點與討論，站在金字塔頂端的他們，有惶恐、憤怒，更有急迫。
     </p>
     <p class="graf graf--p graf-after--p" id="3e6a">
      但在現場，一名來自底層的講者，卻為眾人帶來最激勵人心的答案。
     </p>
     <p class="graf graf--p graf-after--p" id="9b88">
      需要翻譯協助溝通的多明格斯，是一位來自墨西哥的母親，現職為專業的清潔工，她也是講者們口中的「需要被包容的多元文化」、「美國經濟需要的力量」、「要給予基本人權的外來移民」，以及「貧富差距下被遺忘的失聲公民」。當她站上台上，她提出的不是美國政府應該幫她什麼。在台上，她分享了自己所完成的。
     </p>
     <p class="graf graf--p graf-after--p" id="016a">
      一個三千名清潔女工共創的平台，一家新型態的合作社：
      <a class="markup--anchor markup--p-anchor" data-href="https://brightly.coop/" href="https://brightly.coop/" rel="noopener" target="_blank">
       Brightly.coop
      </a>
      。
     </p>
     <p class="graf graf--p graf-after--p" id="8017">
      Brightly.coop 是一個結合開源共享平台技術的合作社，每一個清潔工都是會員、都是股東，她們靠著開源軟體Up&amp;Go的幫助，打造自己的數位平台，消費者可以透過平台直接雇用他們的清潔服務。
     </p>
     <p class="graf graf--p graf-after--p" id="cd85">
      兩年前，Brightly.coop 展開與 Up&amp;Go 平台的合作，介面的設計以增加信任為目標，消費者直接了解每一個清潔服務的提供者，和他們互動，「在這個時代，人們沒有看到你的社交檔案，很難相信你，很難讓你進到他們家裡面去打掃。」多明格斯說。雖然一樣掛著共享經濟之名，但在 Up&amp;Go 的平台上，一切都以第一線勞動者的需求為出發，因為，他們正是這平台的擁有者。
     </p>
     <p class="graf graf--p graf-after--p" id="e480">
      分潤機制上也有不同。每一筆消費，平台只收 5％，其中 2％ 作為平台維護營運的成本，3％ 回到合作社，剩下，全都歸勞動者所有。
     </p>
     <p class="graf graf--p graf-after--p" id="3adb">
      「雖然我們的定價比其他清潔公司稍微高了一些，但（一年來）我們發現，人們在我們這邊消費是開心的，」多明格斯說。在她們進入市場之前，消費者面對清潔服務平台，每筆消費收取 20％ 到 50％ 的抽傭，有的採取個案外包的方式，在預定之前，看不見清潔工、無法保證得到的服務是什麼，甚至，大部分人必須靠著朋友打聽，靠口耳相傳，才能找到足以信任的清潔服務。
     </p>
     <p class="graf graf--p graf-after--p" id="4c38">
      但在 Up&amp;Go 上，因為勞動者就是平台的擁有者，他們想的不是一次性的生意，而是人與人之間的關係建立，於是定價透明、服務細項清楚列出，消費者在消費發生前，就能看得見自己的付出將得到什麼，能改變什麼。
     </p>
     <p class="graf graf--p graf-after--p" id="8174">
      在 Up&amp;Go 平台上的三千個清潔工，平均時薪比市場行情多出 5 美元，若與他們加入合作社前的收入相比，則多出 10 美元。
     </p>
     <blockquote class="graf graf--blockquote graf-after--p" id="6145">
      <p>
       「我們還採取環境友善的清潔方式，也把消費者的身體健康都想好了，」多明格斯說。她談起清潔服務時，就像任何一個專業工作者談起自身的事業那樣，「我們以自己的專業和得到的信任為榮，顧客看待我們就像看待專業的電子技工、髮型師一樣。」
      </p>
     </blockquote>
     <p class="graf graf--p graf-after--blockquote" id="1562">
      讓消費者得到在地、可信賴、對社會有幫助的服務，讓勞動者得到自我認同、該有的權益，以及在經濟弱勢族群中建立起組織、發聲權，這就是平台合作社主義（Platform Cooperativism）的宗旨。
     </p>
     <h3 class="graf graf--h3 graf-after--p" id="cc1c">
      <strong class="markup--strong markup--h3-strong">
       重獲新生的平台合作社主義
      </strong>
     </h3>
     <p class="graf graf--p graf-after--h3" id="da93">
      平台合作社主義在 2014 年由紐約新學院（The New School）助理教授休茲（
      <a class="markup--anchor markup--p-anchor" data-href="http://www.pdf-18.com/speaker/trebor-scholz/" href="http://www.pdf-18.com/speaker/trebor-scholz/" rel="noopener" target="_blank">
       Trebor Scholz
      </a>
      ）提出。看著多明格斯在台上分享過去一年的經驗，休茲笑說，這樣的故事在世界各地吸引大量的注意，「我每天都收到信，從南非UBER 的司機到洛杉磯的遛狗員、印度的軟體工程師，他們都在問該怎麼做、下一步是什麼？」
     </p>
     <p class="graf graf--p graf-after--figure" id="2536">
      休茲解釋，當 Uber、Airbnb 等各種共享經濟平台走向世界，近年來看見的，卻是平台業者手握利潤和使用者的數據，不斷提高自身的市場估值；另一面，卻是勞動者不斷下降的平均工資和勞動權益。
     </p>
     <p class="graf graf--p graf-after--p" id="7d74">
      2014年，
      <a class="markup--anchor markup--p-anchor" data-href="https://dataspace.princeton.edu/jspui/bitstream/88435/dsp010z708z67d/5/587.pdf" href="https://dataspace.princeton.edu/jspui/bitstream/88435/dsp010z708z67d/5/587.pdf" rel="noopener" target="_blank">
       普林斯頓大學針對二十個城市的 Uber 司機展開調查
      </a>
      ，以洛杉磯為例，他們每小時時薪收入中位數約為 17美元（UberX），還需要扣掉汽油、保險、汽車維護等成本，
      <a class="markup--anchor markup--p-anchor" data-href="https://www.govdocs.com/l-county-minimum-wages-goes-live-july-1/" href="https://www.govdocs.com/l-county-minimum-wages-goes-live-july-1/" rel="noopener" target="_blank">
       不一定能達成落杉磯訂定的每小時最低工資13美元
      </a>
      。
     </p>
     <p class="graf graf--p graf-after--p" id="b42f">
      共享經濟雖有「共享」二字，但主要獲利的卻是平台業者和投資人，勞動者有可能反而掉入勞動權益的黑洞。
      <a class="markup--anchor markup--p-anchor" data-href="http://www.nber.org/papers/w22667.pdf" href="http://www.nber.org/papers/w22667.pdf" rel="noopener" target="_blank">
       經濟學家 Lawrence. F. Katz 和 Alan. B. Krueger 研究發現
      </a>
      ，2005 年到 2015 年的十年間，在全美新增的近1000萬的工作崗位中，幾乎全部都是臨時工、合同工或是自由擇業的散工。地球另一端的德國也出現相同趨勢。網絡社會的新型經濟不但沒有縮小貧富差距，反而令各種工作更零散、更原子化。科技獨角獸一隻隻誕生，大部分勞工的收入卻更低、更不穩定。
     </p>
     <p class="graf graf--p graf-after--p" id="c50f">
      「真正的共享經濟正在衰竭」休茲說，他想用擁有兩百多年的合作社概念，結合科技，拯救共享經濟。四年多來，全球十四個教研機構、國際非政府組織加入，研究、推廣，由下而上的打造平台合作社運動。
     </p>
     <p class="graf graf--p graf-after--p" id="c71e">
      他觀察過去四年來自世界各地的經驗，指出「整體是技術、文化、政治、社會的變化，不只是技術的終極追求。」最大的考驗是如何創造一個全新的所有權機制，實踐民主價值，平台技術不是難關，但重塑社群內的關係、決策機制，以及如何對外溝通、找尋夥伴，才是成功與否的關鍵。
     </p>
     <p class="graf graf--p graf-after--p" id="4187">
      <a class="markup--anchor markup--p-anchor" data-href="http://www.rosalux-nyc.org/wp-content/files_mf/scholz_platformcoop_5.9.2016.pdf" href="http://www.rosalux-nyc.org/wp-content/files_mf/scholz_platformcoop_5.9.2016.pdf" rel="noopener" target="_blank">
       他提出十大原則
      </a>
      ，成為平台合作社主義發展的準則以及檢視的標準：
     </p>
     <ol class="postList">
      <li class="graf graf--li graf-after--p" id="7689">
       所有權：平台合作社歸屬於創造大部分平台價值的人們所有，可以改變一般人對自己與網路間關係的認知。
      </li>
      <li class="graf graf--li graf-after--li" id="1104">
       體面的工資和收入保障：讓隱形的數位經濟工人，拿回該有的權利跟生活條件。
      </li>
      <li class="graf graf--li graf-after--li" id="6837">
       透明性與數據公開：合作社的財務等最透明，對使用者來說，拿取什麼數據、如何搜集、向誰出售、如何使用，都要透明。
      </li>
      <li class="graf graf--li graf-after--li" id="10bb">
       尊重與認同：確保基本的勞權，並在社群中建立健康的自我認同。
      </li>
      <li class="graf graf--li graf-after--li" id="709c">
       共同決策：從平台的設計跟打造起，勞工就應該參與集中。
      </li>
      <li class="graf graf--li graf-after--li" id="b570">
       法律保障的框架：對內、對外的保障都是。
      </li>
      <li class="graf graf--li graf-after--li" id="0380">
       不被企業綁定的工人保障和福利：不屬於單一企業的勞工，必須尋找出新的制度，持續的給予社會福利。對臨時工、全職員工必須一視同仁。
      </li>
      <li class="graf graf--li graf-after--li" id="e509">
       保護免受霸凌：平台經濟下，勞工受到平台的不公平對待必須消除。個人事業聲譽、評等等「資本」，不應該被綁定、被控制。
      </li>
      <li class="graf graf--li graf-after--li" id="f20d">
       拒絕工作場所的不斷監視。
      </li>
      <li class="graf graf--li graf-after--li" id="c4c3">
       消失權：數位經濟下的勞工，仍必須確保下線、登出的權利。
      </li>
     </ol>

        </p>
        <p>
         <img alt="" class="aligncenter wp-image-2774 size-full" data-attachment-id="2774" data-comments-opened="0" data-image-description="" data-image-meta='{"aperture":"0","credit":"","camera":"","caption":"","created_timestamp":"0","copyright":"","focal_length":"0","iso":"0","shutter_speed":"0","title":"","orientation":"0"}' data-image-title="0_I7J3ma-M6FDPi-8s" data-large-file="https://i1.wp.com/www.caa-ins.org/wp-content/uploads/2018/07/0_I7J3ma-M6FDPi-8s.png?fit=810%2C810&amp;ssl=1" data-medium-file="https://i1.wp.com/www.caa-ins.org/wp-content/uploads/2018/07/0_I7J3ma-M6FDPi-8s.png?fit=300%2C300&amp;ssl=1" data-orig-file="https://i1.wp.com/www.caa-ins.org/wp-content/uploads/2018/07/0_I7J3ma-M6FDPi-8s.png?fit=960%2C960&amp;ssl=1" data-orig-size="960,960" data-permalink="https://www.caa-ins.org/archives/2770/0_i7j3ma-m6fdpi-8s" data-recalc-dims="1" height="810" loading="lazy" sizes="(max-width: 810px) 100vw, 810px" src="https://i2.wp.com/caa-ins.org/wp-content/uploads/2018/07/0_I7J3ma-M6FDPi-8s.png?resize=810%2C810&amp;ssl=1" srcset="https://i1.wp.com/www.caa-ins.org/wp-content/uploads/2018/07/0_I7J3ma-M6FDPi-8s.png?w=960&amp;ssl=1 960w, https://i1.wp.com/www.caa-ins.org/wp-content/uploads/2018/07/0_I7J3ma-M6FDPi-8s.png?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.caa-ins.org/wp-content/uploads/2018/07/0_I7J3ma-M6FDPi-8s.png?resize=300%2C300&amp;ssl=1 300w, https://i1.wp.com/www.caa-ins.org/wp-content/uploads/2018/07/0_I7J3ma-M6FDPi-8s.png?resize=768%2C768&amp;ssl=1 768w, https://i1.wp.com/www.caa-ins.org/wp-content/uploads/2018/07/0_I7J3ma-M6FDPi-8s.png?resize=810%2C810&amp;ssl=1 810w" width="810"/>
        </p>
       </div>
      </div>
      <figcaption class="imageCaption">
      </figcaption>
     </figure>
     <h3>
     </h3>
     <h3 class="graf graf--h3 graf-after--figure" id="a3ac">
      <strong class="markup--strong markup--h3-strong">
       以人為本的民主精神
      </strong>
     </h3>
     <p class="graf graf--p graf-after--h3" id="9890">
      以全新的制度經營事業，勞動者扮演的角色也完全不同。多明格斯笑稱，她以前只要做好媽媽、姊姊、清潔工的角色即可，現在，她必須不斷精進自己，學習會計、行銷、管理，參加工作坊。
     </p>
     <p class="graf graf--p graf-after--p" id="960b">
      「沒有人可以搭便車，每個人都是擁有者，要學會技術、要有責任，也了解到彼此是生命共同體。」她點出最基本的考驗。
     </p>
     <p class="graf graf--p graf-after--p" id="12dd">
      像是在集體追求一個理想生活一般，三千個清潔女工一起創業，要有共同的方向、想法，不是件容易的事。他們先組成小團體，每兩週碰面一次，每個團體推代表，從中，選出執行董事會，每個月與合作夥伴 Up&amp;Go 碰面。「很難每一個人都同意，尤其這些人以前都是管好自己就好的人。⋯⋯每個人要以團隊利益優先，有時候是很難的事情。」多明格斯坦承，但他們確定的是，既然決定是合作社了，那麼凡事考量先以人的角度，然後才是以企業的營運獲利。
     </p>
     <p class="graf graf--p graf-after--p" id="426d">
      以人為本的核心概念，也進入平台設計的過程中，每項功能的開發順序、方向，都以各合作社的獨特文化與特性為主，非技術出身的清潔工為優先，開發者反而是次要的角色。
     </p>
     <p class="graf graf--p graf-after--p" id="8d13">
      以人為本、講求民主，從總體規模跟平台本身來看，平台合作社的進展慢、規模擴大慢、獲利最大化的可能低，但若回到勞動者的角度，實質收入高、自我實踐、社群的凝聚等，不再走向負循環。另一方面，因為有了實踐民主的經驗，移民社群、經濟弱勢的勞動者，也同時養成參與當地公共事務的習慣，增加多元族群的發聲。
     </p>
     <blockquote class="graf graf--blockquote graf-after--p" id="31ac">
      <p>
       「過去，很多人只把我們當做掃地的機器，我們在這個社會中是最底層的人。我們希望讓大家看到，我們提供的服務中帶有的人性成分，也想讓我們重新找回生活品質，我們要證明自己的專業，即使這只是清潔服務。」
      </p>
     </blockquote>
     <p class="graf graf--p graf-after--blockquote graf--trailing" id="44d7">
      十五年前，那個沒有資本、沒有科技，沒有尊嚴，在大街一個人一個人問需不需要幫忙打掃的多明格斯，如今不但能夠許下改善產業未來的願，靠著平台合作社，她還有能力使其成真。她笑稱，或許這就是人們所稱的美國夢吧。
     </p>
    </div>
   </div>
