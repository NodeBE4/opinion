      </strong>

  <p class="graf graf--p">
   在2018年，特朗普试图抛弃他的前任在2015年与伊朗签署的具有里程碑意义的核协议，他当时正在寻找方法去压倒持怀疑态度的新闻。
  </p>
  <p class="graf graf--p">
   白宫声称，核协议允许伊朗增加军事预算，华盛顿邮报记者 Salvador Rizzo 和 Meg Kelly
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/news/fact-checker/wp/2018/05/09/fact-checking-president-trumps-reasons-for-leaving-the-iran-nuclear-deal/?noredirect=on&amp;utm_term=.5f4c2b7f622f" href="https://www.washingtonpost.com/news/fact-checker/wp/2018/05/09/fact-checking-president-trumps-reasons-for-leaving-the-iran-nuclear-deal/?noredirect=on&amp;utm_term=.5f4c2b7f622f" rel="noopener noreferrer" target="_blank">
    要求提供信源
   </a>
   。
  </p>
  <p class="graf graf--p">
   作为回应，白宫拿出了一篇名为 Heshmat Alavi 的作家在福布斯新闻上发表的一篇文章。
  </p>
  <p class="graf graf--p">
   这就是故事的关键。
  </p>
  <p class="graf graf--p">
   Rizzo 和 Kelly 这两位记者报道说，根据福布斯撰稿人 Heshmat Alavi 在白宫官员发给我们的一篇文章中所说，“伊朗目前的预算主要通过’石油、税收、增加债券、[并]消除伊朗人的现金补贴”来资助。
  </p>
  <p class="graf graf--p">
   他们完全没有怀疑这篇文章。
  </p>
  <p class="graf graf--p">
   而白宫也曾使用 Alavi 的文章 —— 部分来自伊朗消息来源 —— 以证明其终止协议的决定是正当的。
  </p>
  <p class="graf graf--p">
   但是，有一个问题：
   <strong class="markup--strong markup--p-strong">
    ⚠️Heshmat Alavi 这个“人”并不存在。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Alavi 的角色是由伊朗反对派组织 Mojahedin-e-Khalq 开展的信息战活动制造出来的，该组织以其缩写 MEK 闻名
   </strong>
   ，两名消息人士告诉 The Intercept。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “Heshmat Alavi 是一个由 MEK 政治部门人员组成的角色扮演，”来自 MEK 的高级叛逃者 Hassan Heyrani 说，他说他对这次行动有直接的了解。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “他们写下指挥官要求的任何东西，并用这个虚拟人物的名字在报刊上刊登大量文章。这不是、从来都不是个真正的人。”
   </strong>
  </p>
  <p class="graf graf--p">
   Heyrani 表示，⚠️这个假人形象是由阿尔巴尼亚的一个 MEK 特工团队管理的，该团队有一个基地，专门用于在线信息战。
  </p>
  <p class="graf graf--p">
   Heyrani 的说法和 Sara Zahiri 一致，Sara Zahiri 是一位专注于 MEK 的波斯语研究员。
  </p>
  <p class="graf graf--p">
   在伊朗政府网络安全官员中有消息来源的 Zahiri 说，Alavi 在伊朗内部被称为一个由 MEK 成员团队管理的“团体账户”，并且 Alavi “本人”并不存在。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “Alavi”，这个在福布斯网站上的撰稿的作者在文章旁边的简历中自称是“一名对平等权利充满热情的伊朗活动家”。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️过去几年里，这个假人一直多家主流媒体如福布斯、Hill、Daily Caller、The Federalist、阿拉伯卫星电视台英文版等，发表了大量关于伊朗的文章。
  </p>
  <p class="graf graf--p">
   以“Alavi”的名义发布的文章、以及这个假人的社交媒体的存在，似乎对 MEK 来说是一个福音。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    MEK 以其精巧的宣传而闻名 —— 在伊朗臭名昭著。
   </strong>
  </p>
  <p class="graf graf--p">
   过去十年来，
   <strong class="markup--strong markup--p-strong">
    MEK 已经将信息战的注意力转向英语受众
   </strong>
   ——
   <strong class="markup--strong markup--p-strong">
    特别是在美国、加拿大、和英国等国，这些国家的外交政策是 MEK 计划推翻伊朗政权并取而代之的核心目标。
   </strong>
  </p>
  <p class="graf graf--p">
   这个假人 Alavi 的角色由三个 MEK 成员管理。
  </p>
  <p class="graf graf--p">
   Heyrani 曾一度帮助协调该组织的在线活动。他指出了 MEK 政治部门的个人和指挥官，他们负责以 Heshmat Alavi 的名义撰写英文文章和推文，并与 The Intercept 分享了这些人的照片和名字。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他们是我的朋友。我们是亲密的朋友”，Heyrani说，“我们在一起工作。”
  </p>
  <figure aria-describedby="caption-attachment-5802" class="wp-caption aligncenter" id="attachment_5802" style="width: 768px">
   <img alt="" class="size-full wp-image-5802 jetpack-lazy-image" data-lazy-sizes="(max-width: 768px) 100vw, 768px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/Hassan-Heyrani.jpg?resize=768%2C1024&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/Hassan-Heyrani.jpg?w=768&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/Hassan-Heyrani.jpg?resize=225%2C300&amp;ssl=1 225w" data-recalc-dims="1" height="1024" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/Hassan-Heyrani.jpg?resize=768%2C1024&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="768"/>

   </figcaption>

  <p class="graf graf--p">
   Heyrani 解释说，MEK 领导层对于以自己的名义操作信息战角色的流利英语使用者不会表现得很友好。
  </p>
  <p class="graf graf--p">
   他说，
   <strong class="markup--strong markup--p-strong">
    普通成员不被鼓励拥有突出的公众形象 —— 这反映了许多评论家所指出的 MEK 邪教般的操作原则。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “该组织的领导者不允许任何人使用自己的真实姓名，”Heyrani 说，“因为领导者是组织中的第一人，所有事情都在他们的阴影之下运行。”
  </p>
  <p class="graf graf--p">
   ⚠️MEK 进行了无情的在线信息战宣传活动，利用机器人网军以“群体”的外观形象抨击有关伊朗的在线辩论。
  </p>
  <p class="graf graf--p">
   Heyrani 说，管理 Heshmat Alavi 帐户的 MEK 团队的目标之一是以 Alavi 的名义专门在美国媒体上发布文章。
  </p>
  <p class="graf graf--p">
   Intercept 要求对 MEK 政治部门发表评论的请求、以及对 Alavi 角色操作员的采访请求，都没有得到答复。
  </p>
  <p class="graf graf--p">
   另一位现居加拿大的前 MEK 成员 Reza Sadeghi 证实，Heyrani 确定的三人组合的确参与了该组织的在线信息战操作。
  </p>
  <p class="graf graf--p">
   Sadeghi 是 MEK 的成员工作直到2008年，
   <strong class="markup--strong markup--p-strong">
    他也参与了对美国的游说活动，以及 MEK 在伊拉克阿什拉夫营地的行动。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    他描述了一个由该集团管理的不断增长的在线宣传中心，旨在影响有关伊朗的在线讨论。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️“我们一直在积极将虚假新闻传播到外国媒体和伊朗，” Sadeghi 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <strong class="markup--strong markup--p-strong">
    在阿什拉夫营地，有一些计算机可以进行在线信息战操作。多年来，随着 Facebook 和 Twitter 等社交媒体平台的流行，[信息战]活动变得更加激烈。”
   </strong>
  </p>
  <p class="graf graf--p">
   MEK 是政治反对派中
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2015/02/26/long-march-yellow/" href="https://theintercept.com/2015/02/26/long-march-yellow/" rel="noopener noreferrer" target="_blank">
    最具争议的组织
   </a>
   之一。
  </p>
  <p class="graf graf--p">
   虽然今天它主要涉及政治活动和游说，但
   <strong class="markup--strong markup--p-strong">
    该组织也有暴力史
   </strong>
   。
  </p>
  <p class="graf graf--p">
   从1997年到2012年，MEK 被美国国务院列为外国恐怖主义组织，这一地位最终被奥巴马政府外交协议部分撤销。
  </p>
  <p class="graf graf--p">
   该组织最后一次声称为暴力袭击负责是在2001年。
  </p>
  <p class="graf graf--p">
   MEK 最初支持伊斯兰革命，但是，在以文职为首的伊斯兰共和国成立后不久就失宠了。
  </p>
  <p class="graf graf--p">
   随后的镇压迫使该团体流亡，在法国和伊拉克之间运作 —— 由于萨达姆侯赛因的慷慨，该团体占领了阿什拉夫营地，作为参与两伊战争中转站。
  </p>
  <p class="graf graf--p">
   美国入侵伊拉克之后的几年对于 MEK 来说
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2015/02/26/long-march-yellow/" href="https://theintercept.com/2015/02/26/long-march-yellow/" rel="noopener noreferrer" target="_blank">
    是非常痛苦的
   </a>
   ，而被列入恐怖主义组织则让状况更加复杂了。
  </p>
  <p class="graf graf--p">
   当美国撤军时，他们将 MEK 伊拉克基地的安全工作交给伊拉克政府；接下来是另一轮暴力镇压。
  </p>
  <p class="graf graf--p">
   2012年将 MEK 从美国的恐怖组织名单中删除的协议促成了数千名 MEK 成员从伊拉克迁移到阿尔巴尼亚，该组织被安置在一个新的秘密建筑物中。
  </p>
  <p class="graf graf--p">
   根据 MEK 叛逃者 Heyrani 的说法，
   <strong class="markup--strong markup--p-strong">
    就是阿尔巴尼亚的这个基地，一些管理 Alavi 假人物角色的 MEK 成员在这里工作。
   </strong>
  </p>

  <p class="graf graf--p">
   ⚠️Alavi 的文章倾向于将伊朗政府严厉的谴责与不那么微妙的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.forbes.com/sites/heshmatalavi/2018/03/21/trump-iran-mek-albania-nowruz/#52789f472adf" href="https://www.forbes.com/sites/heshmatalavi/2018/03/21/trump-iran-mek-albania-nowruz/#52789f472adf" rel="noopener noreferrer" target="_blank">
    建议
   </a>
   混为一谈 —— 以争取令现任政府被 MEK 及其领导人 Maryam Rajavi 所取代。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该团体似乎利用 Alavi 取得了巨大成功，特别是在美国主流媒体福布斯新闻上。
   </strong>
  </p>
  <p class="graf graf--p">
   Intercept 向 Alavi 在过去几年发表文章的主流媒体的编辑们致电，以询问假人的问题。
  </p>
  <p class="graf graf--p">
   但是，这些主流媒体都没有能够证实他们曾与“Alavi”交谈过或遇到过 Alavi。
  </p>
  <p class="graf graf--p">
   根据这些出版物的发言人的说法，
   <strong class="markup--strong markup--p-strong">
    这些稿子是免费的
   </strong>
   ，他们没有为这个假人给福布斯、Daily Caller、或 the Diplomat 的写作付钱。
  </p>
  <p class="graf graf--p">
   虽然“Alavi”主要在一些主要是右倾的出版物上发表关于伊朗的文章，但到目前为止，他的文章最常出版的地方是福布斯。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在2017年4月至2018年4月的一年中，Alavi 为福布斯网站发布了惊人的61篇文章。
   </strong>
  </p>
  <p class="graf graf--p">
   2014年以 Alavi 的名义创建的 Twitter 帐户拥有超过30,000名粉丝，⚠️其中包括一些记者和基于华盛顿的保守派智囊团员工。
  </p>
  <p class="graf graf--p">
   该帐户经常
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/HeshmatAlavi/status/1096523048654651392" href="https://twitter.com/HeshmatAlavi/status/1096523048654651392" rel="noopener noreferrer" target="_blank">
    分享赞美
   </a>
   Rajavi 的文章和标签，并分享 MEK 举办的抗议和活动的镜头。
  </p>
  <p class="graf graf--p">
   Alavi 这个假人似乎在华盛顿的右翼圈子中获得了一定程度的认同。
  </p>
  <p class="graf graf--p">
   除了福布斯和其他网站发表的许多文章之外，Alavi 还推出了一个名为“
   <a class="markup--anchor markup--p-anchor" data-href="https://irancommentary.wordpress.com/" href="https://irancommentary.wordpress.com/" rel="noopener noreferrer" target="_blank">
    伊朗评论
   </a>
   ”的博客，该博客的
   <a class="markup--anchor markup--p-anchor" data-href="https://freebeacon.com/national-security/report-senior-irgc-commander-flees-iran-with-secret-intel/" href="https://freebeacon.com/national-security/report-senior-irgc-commander-flees-iran-with-secret-intel/" rel="noopener noreferrer" target="_blank">
    描述中
   </a>
   自称其关注“与伊朗和中东有关的问题”。
  </p>
  <p class="graf graf--p">
   以 Alavi 的名义出版的作品整体上对伊朗政府和哈桑·鲁哈尼持一贯的强硬态度；尤其是，
   <strong class="markup--strong markup--p-strong">
    这些文章也混淆了对伊朗和美国政策的批评，并公开宣传 MEK。
   </strong>
  </p>
  <p class="graf graf--p">
   他在 Daily Caller、The Hill和其他媒体网站上的作品 —— 虽然比他对福布斯的贡献少，但是都采用了类似的对伊朗政权的宣传和对 MEK 的赞扬。
  </p>
  <p class="graf graf--p">
   ⚠️虽然众所周知 MEK 在伊朗人中间受到广泛厌恶，但 Alavi 在
   <a class="markup--anchor markup--p-anchor" data-href="https://dailycaller.com/2017/02/03/language-iran-understands-loud-and-clear/" href="https://dailycaller.com/2017/02/03/language-iran-understands-loud-and-clear/" rel="noopener noreferrer" target="_blank">
    2017年写给 Daily Caller 的文章
   </a>
   中自称该组织为“伊朗主要反对派组织”。
  </p>
  <p class="graf graf--p">
   The Diplomat 是一个外交政策网站，在2017年发布了一些 Alavi 的作品，他们说 Alavi 通过 Gmail 帐户给编辑发送了文章草稿。
  </p>
  <p class="graf graf--p">
   据称 Alavi 发送了数十篇文章，但最终只有少数文章被接受。
  </p>
  <p class="graf graf--p">
   一位要求匿名讨论内部事务的消息人士表示，外交官在确定他的文章不符合出版标准后停止接受 Alavi 的推销。
  </p>
  <p class="graf graf--p">
   Daily Caller 还告诉 The Intercept，他们已经停止发布 Alavi 的文章，因为对他提交的文章质量表示担忧。
  </p>
  <p class="graf graf--p">
   TheHill、al-Arabiya English 和 The Federalist 没有回复评论请求。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们在2018年初终止了与 Heshmat Alvi 的关系，” 福布斯发言人在给 The Intercept 的一份声明中表示。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">

   </strong>

  <p class="graf graf--p">
   该组织已派出杰出的政治人物发表演讲和新闻发布会、
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2015/02/26/long-march-yellow/" href="https://theintercept.com/2015/02/26/long-march-yellow/" rel="noopener noreferrer" target="_blank">

  </p>
  <p class="graf graf--p">
   在2018年，MEK 的社交媒体业务是半岛电视台“Listening Post”
   <a class="markup--anchor markup--p-anchor" data-href="https://www.aljazeera.com/programmes/listeningpost/2018/09/faking-online-debate-iran-180915121527882.html" href="https://www.aljazeera.com/programmes/listeningpost/2018/09/faking-online-debate-iran-180915121527882.html" rel="noopener noreferrer" target="_blank">
    纪录片的主题
   </a>
   。
  </p>
  <p class="graf graf--p">
   该宣传战小组利用这些公关努力实现其政策目标。
  </p>
  <p class="graf graf--p">
   直到2012年，MEK 主要
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2015/02/26/long-march-yellow/" href="https://theintercept.com/2015/02/26/long-march-yellow/" rel="noopener noreferrer" target="_blank">
    专注于脱离美国的“恐怖组织”名单
   </a>
   。

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    而在接下来的几年里，该组织专注于攻击伊朗与美国之间的核外交，并在2015年之后攻击该协议本身。
   </strong>
  </p>
  <p class="graf graf--p">
   在整个过程中，MEK 的信息战强调政权更迭 —— 并试图将 MEK 作为伊朗伊斯兰共和国领导层的可行替代方案，并提供了作为该组织公众形象十五年的 Rajavi 作为领导国家的替代方案。
  </p>
  <p class="graf graf--p">
   Alavi 在福布斯的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.forbes.com/sites/heshmatalavi/2018/02/01/irans-future-in-its-own-words/#26644d841c17" href="https://www.forbes.com/sites/heshmatalavi/2018/02/01/irans-future-in-its-own-words/#26644d841c17" rel="noopener noreferrer" target="_blank">
    一篇文章
   </a>
   中呼吁国际支持伊朗的政权更迭，他写道：

  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    现在是搁置伊朗“改革派”海市蜃楼的时候了。几十年来，作为伊朗国家抵抗委员会主席的玛丽亚姆·拉贾维（Maryam Rajavi）正在为伊朗提供唯一的、现实的替代方案，其十点计划得到了全球数千名当选官员的支持。
   </em>
  </p>
  <p class="graf graf--p">
   就像 MEK 提升 Rajavi 的目标一样，Alavi 的宣传战消息也与该组织攻击伊朗核协议的努力联系在一起。
  </p>
  <p class="graf graf--p">
   在2017年至2018年期间，当 Alavi 的文章出现在福布斯上时，特朗普正在采取措施将美国从这笔交易中解脱出来，尽管欧洲盟友和前奥巴马政府官员
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2017/11/13/iran-nuclear-deal-trump-eu-european-union/" href="https://theintercept.com/2017/11/13/iran-nuclear-deal-trump-eu-european-union/" rel="noopener noreferrer" target="_blank">
    都反对
   </a>
   。
  </p>
  <p class="graf graf--p">
   Alavi 的文章努力怂恿政府，其中包括 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.forbes.com/sites/heshmatalavi/2017/10/06/iran-trump-nuclear-deal-jcpoa/" href="https://www.forbes.com/sites/heshmatalavi/2017/10/06/iran-trump-nuclear-deal-jcpoa/" rel="noopener noreferrer" target="_blank">
    伊朗感受特朗普对核交易的热情
   </a>
   ”、以及“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.forbes.com/sites/heshmatalavi/2017/08/24/how-trump-can-correctly-approach-irans-nuclear-deal/#15ef6d217505" href="https://www.forbes.com/sites/heshmatalavi/2017/08/24/how-trump-can-correctly-approach-irans-nuclear-deal/#15ef6d217505" rel="noopener noreferrer" target="_blank">
    特朗普如何正确地接近伊朗的核协议
   </a>
   ”等项目。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在2018年5月，特朗普宣布美国将退出协议 —— 此刻就在 Alavi 的上一篇文章发表在福布斯的一个月之后
   </strong>
   。
  </p>
  <p class="graf graf--p">
   白宫在2018年向华盛顿邮报提出的以退出核协议为理由的 Alavi
   <a class="markup--anchor markup--p-anchor" data-href="https://www.forbes.com/sites/heshmatalavi/2018/03/01/a-close-look-at-irans-budget/#20cbef616e1e" href="https://www.forbes.com/sites/heshmatalavi/2018/03/01/a-close-look-at-irans-budget/#20cbef616e1e" rel="noopener noreferrer" target="_blank">
    文章
   </a>
   引用了伊朗政府的半官方消息来证明鲁哈尼政府增加的军费开支。
  </p>
  <p class="graf graf--p">
   Alavi 的文章典型地夸夸其谈，称赞特朗普政府结束对伊朗的“绥靖”政策，并指责欧洲“与凶残的毛拉政权并肩作战，违背伊朗人民的意愿”。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    这个“Alavi” 对美国政治话语的压力，而不是伊朗本身的压力 —— 似乎是 MEK 战略的一部分。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “该组织几乎没有使用波斯语制作什么内容。他们似乎已经放弃了在伊朗国内获得影响力。他们现在的主要目标是影响英语世界的人”
   </strong>
   ，马苏德·霍达班德（Massoud Khodabandeh）说，他是 MEK 情报部门的前成员，他于1996年离开该组织。

   <strong class="markup--strong markup--p-strong">
    “他们的在线战略只在华盛顿有效；在德黑兰不起作用。”
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️这场信息战除了社交媒体战略和定期文章外，MEK 还参与了更高风险的信息活动。
  </p>

   2002年，MEK 揭露在

  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “纽约客”杂志2006年的一篇文章也表明，该情报可能是被以色列情报部门传递给该组织的，这使人质疑 MEK 声称它在伊朗境内经营着一个强大的间谍网络。
  </p>
  <p class="graf graf--p">
   不仅如此，在其他情况下，MEK 的信息也不太可靠，引起许多西方国家安全分析师的怀疑。
  </p>
  <p class="graf graf--p">
   在2015年的新闻发布会上，MEK 官员
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/world/national-security/exile-group-accuses-iran-of-secret-nuclear-weapons-research/2015/02/24/ad8d64d6-bc5a-11e4-8668-4e7ba8439ca6_story.html?noredirect=on&amp;utm_term=.2498b0b617c4" href="https://www.washingtonpost.com/world/national-security/exile-group-accuses-iran-of-secret-nuclear-weapons-research/2015/02/24/ad8d64d6-bc5a-11e4-8668-4e7ba8439ca6_story.html?noredirect=on&amp;utm_term=.2498b0b617c4" rel="noopener noreferrer" target="_blank">
    声称
   </a>

  </p>
  <p class="graf graf--p">
   这一说法部分被自由派网站 Daily Kos 的博主揭穿。

   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.dailykos.com/stories/2015/02/25/1366697/-BREAKING-Amateur-hour-at-the-Pro-war-media-Latest-allegations-against-Iran-FABRICATED" href="https://www.dailykos.com/stories/2015/02/25/1366697/-BREAKING-Amateur-hour-at-the-Pro-war-media-Latest-allegations-against-Iran-FABRICATED" rel="noopener noreferrer" target="_blank">

  </p>
  <p class="graf graf--p">
   任何有关该组织甚至伊朗政治的言论都可以通过推特和其他社交媒体上的回复评论来满足。
  </p>
  <p class="graf graf--p">
   许多亲 MEK 账户将重复相同的信息，通常是逐字逐句、汇聚任何评论员的提及。
  </p>
  <p class="graf graf--p">
   社交媒体专家、社交媒体制图公司 SocialCartograph 的创始人 Geoff Golberg 特别注意到 Alavi 的推特账户，他指出，
   <strong class="markup--strong markup--p-strong">

   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该帐户受到其他亲 MEK 账户的大力推动，其他推动者还包括对伊朗对抗政策的支持者。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️对于不经意的观察者来说，这些群体的在线活动似乎显示了大量的伊朗人对 MEK 正在推广的东西充满热情 —— 事实上都是假象。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “Heshmat Alavi 账户是一组账户的一部分，这些账户多年来一直参与协调的虚假信息战行为”
   </strong>
   ，Golberg 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">

   <strong class="markup--strong markup--p-strong">

   除了他的社交媒体、他的文章、和他给编辑的电子邮件之外，Alavi 也在网上留下了一些痕迹。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    一张照片、一张经过严格过滤的侧脸照片，用于所有 Alavi 的作者个人资料、他的 LinkedIn 页面和 Twitter 帐户
   </strong>
   。

   ⚠️请注意：在这里照片是侧脸而且低头，并且经过润饰，这种拍摄方法可以在照片的角度上抵制多数人脸识别技术。
  </p>
  <p class="graf graf--p">
   至少，有强烈迹象表明这个假人 Alavi 的“人格”不是它声称的那样。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    近年来，使用假身份进行政治宣传已经变得很普遍。
   </strong>
  </p>
  <p class="graf graf--p">
   2018年英国广播公司新闻调查了另一个虚假的网络“名人”：一个名为“Sarah Abdallah”的 Twitter 账户，在叙利亚战争的在线辩论中大量混淆。
  </p>
  <p class="graf graf--p">
   这个假人 Sarah Abdallah 的说法在某些方面与政治领域相反，这点和假人 Alavi 一样；
   <strong class="markup--strong markup--p-strong">
    Abdallah 是伊朗亲密盟友巴沙尔·阿萨德的叙利亚政府的声音支持者。
   </strong>
  </p>
  <p class="graf graf--p">
   一家在线研究公司确定 Abdallah 的账户是“关于叙利亚在线对话中最具影响力的社交媒体账户之一”。
  </p>
  <p class="graf graf--p">
   虽然英国广播公司的调查引起了人们对
   <strong class="markup--strong markup--p-strong">
    数百名真正的记者居然关注阴暗在线假账户所影响的严重担忧
   </strong>
   ，但它没有得出结论认为 Abdallah 由前线组织操作。
  </p>
  <p class="graf graf--p">
   然而，尽管这个“她”具有强大的在线影响力，
   <strong class="markup--strong markup--p-strong">
    Sarah Abdallah 却从来没有能够取得假人“Heshmat Alavi”那样的成功，后者的文章发表在多家美国主流媒体上，并在白宫被宣读。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️对于那些不熟悉伊朗内部政治的人来说，这个假人 Alavi 可能的确像是“一位对平等权利充满激情的伊朗活动家” —— 这是“他”在其简介中所写的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    前 MEK 成员 Heyrani 说，这个幻觉正是该信息战团队希望与假角色一起创造的效果。
   </strong>
  </p>
  <p class="graf graf--p">
   Hejarani 说：“Mojahedin” —— 伊朗 MEK 的名字 —— “希望向全世界展示他们的叙述得到了支持，即使是那些不是该组织直接成员的人也是如此。”
  </p>
  <p class="graf graf--p">
   Heshmat Alavi 的帐户目前已经被 Twitter 冻结。
  </p>

   此后不久，在另一个社交媒体网站 QuodVerum 上又出现了 Alavi 名字的账号，并发表了一篇帖子，驳斥真相的揭露者。
  </p>
  <p class="graf graf--p">
   上述发布该假人文章的主流媒体部分已经关闭了其专栏页面，其他媒体撤下了部分文章。⚪️
