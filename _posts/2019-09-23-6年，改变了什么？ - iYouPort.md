  <p class="graf graf--p">
   这篇文章是一本新书的节选。也是比此前发布的同类文章更为痛苦的反思：它将矛头指向了我们所有人。
  </p>
  <p class="graf graf--p">
   去年的这个时候 IYP 发布过一篇纪念文章《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/snowden-%e5%90%af%e7%a4%ba%e4%ba%94%e5%91%a8%e5%b9%b4%ef%bc%9a%e6%88%91%e4%bb%ac%e7%a6%bb%e8%87%aa%e7%94%b1%e8%bf%98%e6%9c%89%e5%a4%9a%e8%bf%9c%ef%bc%9f/" href="https://www.iyouport.org/snowden-%e5%90%af%e7%a4%ba%e4%ba%94%e5%91%a8%e5%b9%b4%ef%bc%9a%e6%88%91%e4%bb%ac%e7%a6%bb%e8%87%aa%e7%94%b1%e8%bf%98%e6%9c%89%e5%a4%9a%e8%bf%9c%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    Snowden 启示五周年：我们离自由还有多远？
   </a>
   》。现在是第六年 —— 斯诺登揭露的惊人的全球性间谍活动，到今天为止没有任何一项被停止了。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4579 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/头-5.png?resize=1100%2C715&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/头-5.png?w=1486&amp;ssl=1 1486w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/头-5.png?resize=300%2C195&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/头-5.png?resize=768%2C499&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/头-5.png?resize=1024%2C666&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/头-5.png?resize=1100%2C715&amp;ssl=1 1100w" data-recalc-dims="1" height="715" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/头-5.png?resize=1100%2C715&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p graf--startsWithDoubleQuote">
   “占领华尔街”（Occupy Wall Street）那段时间的照片已经开始困扰我了。这张被用作杂志封面的照片，乍一看似乎捕获了一个犀利的起义画面。
  </p>
  <p class="graf graf--p">
   大批的抗议者拥挤在高大的围栏旁，被一群可怕的防暴警察阻挡，警方在捍卫空荡荡的 Duarte 广场，这是曼哈顿市中心一片单调的混凝土建筑。
  </p>
  <p class="graf graf--p">
   然而，仔细观察，你会发现，一个不同的场景成为焦点：只有少数几个抗议者实际上正在推动围栏。
  </p>
  <p class="graf graf--p">
   而其他人紧紧地挤在一起，高举着手机，互相录制对方和场景的视频，也许是准备分享给朋友和家人。
  </p>
  <p class="graf graf--p">
   Duarte 广场的围栏在2011年的12月日的抗议高峰时段几乎没有任何破坏。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4580 jetpack-lazy-image" data-lazy-sizes="(max-width: 791px) 100vw, 791px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/占领华尔街.jpg?resize=791%2C1024&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/占领华尔街.jpg?w=791&amp;ssl=1 791w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/占领华尔街.jpg?resize=232%2C300&amp;ssl=1 232w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/占领华尔街.jpg?resize=768%2C994&amp;ssl=1 768w" data-recalc-dims="1" height="1024" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/占领华尔街.jpg?resize=791%2C1024&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="791"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   这张照片抓住了近乎膝跳反射的倾向 —— 许多参与大规模抗议活动的人都在社交媒体上讲述每一个动作，并认为这本身就是“大胆的和激进的”；将抗议的叙述置于我们自己的手中，我们自己的广播设备，拒绝依赖传统媒体机构。
  </p>
  <p class="graf graf--p">
   无论你在哪里讨论社交媒体平台是否有助于、或阻碍了、或以其他方式塑造了抗议运动（或上述所有方面），都应该认识到，在斯诺登揭露之后的几年里，这些被上传网络的抗议现场照片已经呈现出完全不同的效应。
  </p>
  <p class="graf graf--p">
   对于上面那张照片中的一众手机来说，
   <strong class="markup--strong markup--p-strong">
    已经不仅是 Duarte 广场群体努力的障碍了；并且更重要的是，它们是监视设备
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这一点最迟在2014年已经变得无可否认：我们依赖的设备和平台 —— 沟通、收集信息和建立团结的基本工具 —— 已经变成了掐住我们所有人的黑手，使我们不得不接受持续不间断的监控。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果没有易于追踪所有公民的方法，监视国家就无法维持
   </strong>
   。
  </p>
  <p class="graf graf--p">
   人们避免深度谈论这件事，避免承认自己对自己的*可监控性*之参与有多么的深刻 —— 不论是智能手机还是智能家居，都是人们兴高采烈地买回来的，而不是被谁强迫安置的 —— 这就是我们的生活方式。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4581 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞1.jpg?resize=1100%2C622&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞1.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞1.jpg?resize=300%2C170&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞1.jpg?resize=768%2C434&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞1.jpg?resize=1024%2C579&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞1.jpg?resize=1100%2C622&amp;ssl=1 1100w" data-recalc-dims="1" height="622" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞1.jpg?resize=1100%2C622&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4581" data-recalc-dims="1" height="622" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞1.jpg?resize=1100%2C622&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞1.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞1.jpg?resize=300%2C170&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞1.jpg?resize=768%2C434&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞1.jpg?resize=1024%2C579&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞1.jpg?resize=1100%2C622&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   几年前，我们对于了解被揭露的大规模监视状态的程度感到震惊，这似乎是古怪的（虽然视其为古怪并不是一件好事，要知道
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e7%a9%b6%e7%ab%9f%e6%98%af%e4%bb%80%e4%b9%88%ef%bc%9f/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e7%a9%b6%e7%ab%9f%e6%98%af%e4%bb%80%e4%b9%88%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    这个集中化的互联网本身就建立在大规模监视的欲求之上
   </a>
   ）。
  </p>
  <p class="graf graf--p">
   但美国国家安全局的泄密事件在当时无疑是一个重要的启示 —— 它揭示了政府、通信和科技巨头之间可怕的联系；并且提出了一个警告，
   <strong class="markup--strong markup--p-strong">
    在我们所有人同谋的控制系统中
   </strong>
   所面临的挑战。
  </p>
  <p class="graf graf--p">
   在这里看到对斯诺登文件的最完整汇总《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" rel="noopener noreferrer" target="_blank">
    互联网霸主的时间线 — — 他们将互联网视为间谍、破坏和战争的主场
   </a>
   》。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在对斯诺登泄密事件即时的愤怒反应中，有一种愚蠢的、但也是一种值得称赞的乐观态度
   </strong>
   。记者和活动家们在拼命想要抓住一个东西，一个反派。
  </p>
  <p class="graf graf--p">
   究竟谁应该受到谴责？坏人在哪儿？我们如何反击？
  </p>
  <p class="graf graf--p">
   有明显的罪魁祸首需要谴责：无论是时任国家情报总监的詹姆斯克拉珀、当时的国家安全局局长基思亚历山大、谷歌、AT＆T、微软、还是整个PRISM数据收集和监视计划 ……所有人都想要指责
   <strong class="markup--strong markup--p-strong">
    那种可以被孤立定位的人和事
   </strong>
   。
  </p>
  <p class="graf graf--p">
   政治家和活动家的努力集中在自上而下的国家安全局改革和对科技巨头的透明化要求上。因此，
   <strong class="markup--strong markup--p-strong">
    他们错过了所涉危险的细微差别和严重性
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这是两党哑剧的重要时刻。民主党人和共和党议员齐聚一堂，表达了对美国国家安全局大量收集美国人通信数据的要求。
  </p>
  <p class="graf graf--p">
   在2015年6月签署成为法律的具有荒谬名称的“美国自由法案”中，主要是在这一点上获得了吸引力。
  </p>
  <p class="graf graf--p">
   它对美国通信的大量数据收集提出了一些限制，但是，
   <strong class="markup--strong markup--p-strong">
    它也恢复了乔治·W·布什2001年签署的臭名昭著的“爱国者法案”中一些最糟糕的规定
   </strong>
   。
  </p>
  <p class="graf graf--p">
   奥巴马政府组建了咨询委员会，他们正式发布了冗长的报告，并承诺将会有更多的评论。也许最糟糕的是，
   <strong class="markup--strong markup--p-strong">
    作为“好人”之地位的争夺战，包括谷歌和 Facebook 在内的技术巨头们公开宣称要提高所谓的透明度
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在 Facebook 不透明的政策和产品帮助诸如剑桥分析公司这类的机构操弄选举被曝光之后，所谓的透明度现在看起来很可笑；尽管如此，每周你仍然会从硅谷大型集中化“云”的神秘堡垒中听到所谓“透明度”的新承诺。
  </p>
  <p class="graf graf--p">
   （顺便说，剑桥分析公司远非独一无二，还有一大群同类公司与其做着一模一样的事，在这里看到揭露报告《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" rel="noopener noreferrer" target="_blank">
    心理操纵的秘密：”行为经济学”+PR=？
   </a>
   》《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a7%98%e5%af%86%e5%ae%a3%e4%bc%a0%e7%bd%91%e7%bb%9c/" href="https://www.iyouport.org/%e7%a7%98%e5%af%86%e5%ae%a3%e4%bc%a0%e7%bd%91%e7%bb%9c/" rel="noopener noreferrer" target="_blank">
    秘密宣传战网络
   </a>
   》）
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    斯诺登揭露的所有恐怖间谍行为没有任何一个真的被停止了。六年。
   </p>
  </blockquote>
  <p class="graf graf--p">
   对国家监视的大胆抵制和立法改革的斗争很少。
  </p>
  <p class="graf graf--p">
   政府机构
   <strong class="markup--strong markup--p-strong">
    仍在使用
   </strong>
   像 PRISM（棱镜）这样的计划，该计划于2007年启动，允许国家安全局大量收集和存储数据，与几乎所有主要科技公司合作，包括在没有授权的情况下访问所有人的私人通信。
  </p>
  <p class="graf graf--p">
   在斯诺登揭露的所有恐怖中没有任何一个真的被停止了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    通过仅仅专注于清晰可辨认的权力人物，激进组织和愤怒的政治参与者在很大程度上回避了有关受监视的主体如何“维护”他们的被监视之地位的问题。
   </strong>
  </p>
  <p class="graf graf--p">
   很多关于政府监视的讨论是反事实的：如果我们知道自己作为数字居民注册了什么，我们是否*同意*我们目前遭受的大规模监视水平？
  </p>
  <p class="graf graf--p">
   2014年，詹姆斯克拉珀承认国家安全局应该更公开地向公众宣布无处不在的监视一直在囤积他们的私密通信。
  </p>
  <p class="graf graf--p">
   但他调转了语风，称：“如果该计划（棱镜）在9/11袭击事件发生后立即公开引入，大多数美国人可能都会支持它，”他说。
  </p>
  <p class="graf graf--p">
   克拉珀忍不住诉诸于一种不正常的条件逻辑，在这种逻辑中公众会“同意”他们实际上不能同意的事。
  </p>
  <p class="graf graf--p">
   正如斯诺登的法律顾问、美国公民自由联盟隐私和技术项目主任 Ben Wizner 在回应克拉珀的言论时所评论的那样，“我们当时是否会同意这一点将永远不会被人知道。” 我们还没有同意接受这种不间断的监视，即使我们的生活方式已经产生了它。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4582 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞2.jpg?resize=1100%2C733&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞2.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞2.jpg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞2.jpg?resize=768%2C512&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞2.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞2.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞2.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="733" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞2.jpg?resize=1100%2C733&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4582" data-recalc-dims="1" height="733" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞2.jpg?resize=1100%2C733&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞2.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞2.jpg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞2.jpg?resize=768%2C512&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞2.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞2.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞2.jpg?resize=272%2C182&amp;ssl=1 272w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   自2014年以来，通过技术资本主义进行监视的对话已经从违宪的政府间谍计划转移到了关于技术巨头如何以及企业和政治目的如何提取和使用我们的数据的问题。
  </p>
  <p class="graf graf--p">
   或者更确切地说，这些巨头公司如何使用（和加工）我们的数据 —— 毕竟数据不是我们的。
  </p>
  <p class="graf graf--p">
   这种话语转变是有道理的；如果主要关注国家安全局，会错误地将当代监视视为一个不受欢迎的、压迫性的政府审查问题。
  </p>
  <p class="graf graf--p">
   它远远不是。
  </p>
  <p class="graf graf--p">
   毫无疑问，政府审查是一个问题 —— 任何被秘密放入禁飞名单上的人都会证明这一点。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    斯诺登透露的政府计划是在大规模监视已成为社会参与基准的基础上进行的
   </strong>
   ：通过几乎所有可跟踪的互动，这些互动都是我们自己提供的，由少数极其强大的公司运作。
  </p>
  <p class="graf graf--p">
   我们真正“想要”和“接受”这些条件的程度并没有实际意义。虽然我们毫无疑问是维护监视状态的*积极参与者*，但是，称我们因此“同意”，则是夸大了我们对此事的选择能力。
  </p>
  <p class="graf graf--p">
   就算作为经济必需品，也并非每个人都需要依赖于具有监控功能的设备和接口（尽管许多工作者依赖如 Uber 和 TaskRabbit 这样的东西）。
  </p>
  <p class="graf graf--p">
   然而，对于那些遵守现在的社会和经济精神的人来说，*参与*监控系统是不可避免的，因为
   <strong class="markup--strong markup--p-strong">
    这些硅谷拥有的网络和接口已成为社会、私人和商业—— 甚至政治和革命 —— 的主要舞台
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Paul Virilio 是技术世界最有先见之明的思想家之一，他谈到了每项技术从发明的那一刻起是如何发生事故的（意外事故）—— 它将一种事故、一种灾难的规模引入了以前不存在的世界。
  </p>
  <p class="graf graf--p">
   正如他所说，“当你发明这艘船时，你也发明了沉船；当你发明飞机时，你也发明了坠机；当你发明电力时，你会发明触电……每项技术都有自己的消极性，这与技术进步同时被发明出来”。
  </p>
  <p class="graf graf--p">
   正如 Virilio 所看到的那样，包含自身意外事故的技术“非常明显，不得不重复这一点，这表明我们在多大程度上被关于进步的宣传所疏远。”
  </p>
  <p class="graf graf--p">
   Virilio 的框架拒绝非黑即白 —— 我们不一定必须认为我们当前和正在开发的技术是“好”还是“坏”。相反，它需要一个恒定的道德计算：我们必须要求技术可能性包含它的潜在事故以及它们是否可以容忍。
  </p>
  <p class="graf graf--p">
   这点并非总能非常清楚地知道；但总是可以提问的。而且要求提问早一点而不是晚一点。
  </p>
  <p class="graf graf--p">
   正如 Virilio 在1995年所指出的那样，高速列车成为可能，因为旧的铁路技术产生了一种新的交通控制形式，使列车能够越来越快，而不会有灾难性的碰撞风险。事故已经考虑过了。
  </p>
  <p class="graf graf--p">
   但是，“当今的信息技术没有交通控制系统，”他当时写道，最近又补充说：“我们仍然不知道虚拟事故是什么样的。”
  </p>
  <p class="graf graf--p">
   然而，他知道，
   <strong class="markup--strong markup--p-strong">
    这次事故将是“不可分割的全球性的”；这就是在线网络与日常生活融为一体的全部，所有这一切都在瞬间被加速
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们受到压迫，互相压迫 / 我们将立刻被告知 / 发生的任何事” —— 诗人 George Oppen 在他1968年的伟大作品 Of Being Numerous 中就预见到了今天。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4583 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞3.jpg?resize=1100%2C825&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞3.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞3.jpg?resize=300%2C225&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞3.jpg?resize=768%2C576&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞3.jpg?resize=1024%2C768&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞3.jpg?resize=1100%2C825&amp;ssl=1 1100w" data-recalc-dims="1" height="825" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞3.jpg?resize=1100%2C825&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4583" data-recalc-dims="1" height="825" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞3.jpg?resize=1100%2C825&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞3.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞3.jpg?resize=300%2C225&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞3.jpg?resize=768%2C576&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞3.jpg?resize=1024%2C768&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/黑洞3.jpg?resize=1100%2C825&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   人们与监控状态的设备的接触比他们使用的技术工具更深入 —— 实际上这些不仅仅是工具而是设备。
  </p>
  <p class="graf graf--p">
   对于哲学家
   <a class="markup--anchor markup--p-anchor" data-href="https://soundenvironments.files.wordpress.com/2011/11/agamben-what-is-and-apparatus.pdf" href="https://soundenvironments.files.wordpress.com/2011/11/agamben-what-is-and-apparatus.pdf" rel="noopener noreferrer" target="_blank">
    Giorgio Agamben
   </a>
   来说，这些装置不仅仅是一种技术装置，而是“在某种程度上具有捕获、定向、确定、拦截、建模、控制或保护手势、行为、观点、或话语之能力的东西”。他写到了他对手机的“不可饶恕的仇恨”，他想要摧毁所有手机并惩罚它们的用户。
  </p>
  <p class="graf graf--p">
   但是，后来他指出，这不是正确的解决方案。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “设备”不可能简单地被隔离，因为设备由使用它们的主体所塑造；
   <strong class="markup--strong markup--p-strong">
    破坏设备将需要在某些方面破坏创建它们和被其创建的主体。
   </strong>
  </p>
  <p class="graf graf--p">
   无可否认，使我们成为可监控目标的设备也是我们通过社交媒体和可追踪的在线交流创建的自我和关键的社区。
  </p>
  <p class="graf graf--p">
   想要粉碎所有手机、笔记本电脑等GPS设备的大规模 Luddite 运动将忽略这样一个事实，即：
   <strong class="markup--strong markup--p-strong">
    我们数百万人选择的并不是偶然的“历史事故”，而是生活方式，虽然是在多种因素影响下的选择
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “加利福尼亚的意识形态”（在20世纪90年代早期由少数富有的白人男性创造出来的）这一梦想，将互联网描述为权力下放的民主化力量 ——
   <strong class="markup--strong markup--p-strong">
    始终是神话思维的呓语，它未能考虑到互联网就诞生于资本主义体系和残酷的社会等级制度之内，而不是超越它。
   </strong>
  </p>
  <p class="graf graf--p">
   公众对信息的访问的确范围很广，但是，与此同时，信息网络上的权力整合却令人震惊。正如作家和神经科学家 Aaron Bornstein 指出的那样：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    每年都有更多的数据产生 出来 — 并且，与更廉价的存储和收集文化一起保存 —— 超越互联网出现之前的所有人类历史。而事实上，人类的记录比以往任何时候都要少，因为每个人都被迫将这些记录交给了国家
   </em>
   。
  </p>
  <p class="graf graf--p">
   在 2009 年，当时谷歌的CEO埃里克施密特说：“如果你有什么东西不想让任何人知道，也许你一开始就不该这样做”，他这句话足够说明硅谷的特权。它巩固了不受约束的全方位监控系统。
  </p>
  <p class="graf graf--p">
   在这种危险的谷歌道德规定下，并非每个人都有机会将其生活、个人记录、偏好和历史的详细信息提供给公众消费而不会产生任何后果的。一位高薪的硅谷工程师可以随意抛出他对火人节的偏见；但无证移民、性工作者和被监禁的个人 —— 仅举几个边缘化社区的案例 —— 并不一定具有这种特权。
  </p>
  <p class="graf graf--p">
   除此之外，
   <strong class="markup--strong markup--p-strong">
    对大规模监视的抵制不仅是我们可能或不可能隐藏的问题，而是，我们的生活如何被组织和易受社会的控制、我们如何像福柯所说的那样成为了“温顺的身体”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在大多数情况下，对监视的抵制是以隐私为框架的 —— 无论是坚持让科技公司和国家推行更好的隐私政策、还是鼓励个人更好的隐私做法；或者通过混淆保护：加密服务、洋葱路由 ……但
   <strong class="markup--strong markup--p-strong">
    充其量只是改革派的和被动的选择
   </strong>
   。
  </p>
  <p class="graf graf--p">
   政府执行的间谍行为和全方面控制是长期持续的；而在由非政府行为者执行的网络中的干扰如黑客和破坏，往往都是短暂的。这就是权力的不对称。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    我们目前虚弱的集体抵抗足够说明监控技术已经在多大程度上被深度嵌入并塑造了日常生活的结构。
   </strong>
  </p>
  <p class="graf graf--p">
   为了参与社交媒体的小利益、快乐、必需品和不可避免的能力，从维特根斯坦使用的术语来看，
   <strong class="markup--strong markup--p-strong">
    监视是一种“生活形式”：可监视性是使这些“愉快的交互和体验”成为可能的背景
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    监视不是社交媒体生活的附带现象 —— 它是它的基石。在晚期资本主义之下。
   </strong>
  </p>
  <p class="graf graf--p">
   这就是曾经 Oppen 所预料到的。半个世纪以前，虽然他没有暗示通过社交媒体进行全面监控的未来，但是，他谈到了我们当前条件下的相互关系。
  </p>
  <p class="graf graf--p">
   我们参与社会秩序、我们避免了自己仍然痴迷的单一自我，但我们还没有（至少现在没有）选择集体或团结。
  </p>
  <p class="graf graf--p">
   相反，我们选择了一大堆东西：作为在线自我；也因此把自己变成了监视数据点。
  </p>
  <p class="graf graf--p">
   我们依靠并通过数字生活：点赞、点击、互粉。然而，Oppen 写道，至关重要的是，我们所选择的是众多的“意义” —— 希望一起选择不同的“意义”。
  </p>
  <p class="graf graf--p">
   通过沉迷于数字饲养的生活，数百万人已经选择了以前所未有的方式成为“大众”。
  </p>
  <p class="graf graf--p">
   参与社交媒体会产生并要求复杂的关系、联系、真理和幻想；它有时会被革命主题所推动，反过来又产生了革命性的主题。
  </p>
  <p class="graf graf--p">
   例如，毫无疑问的，Twitter 揭示了2009年的伊朗革命，2010年的突尼斯革命、2011年的埃及革命的新亮点。社交媒体既是创造这些起义可能性条件的产物，也是其中的一个因素。
  </p>
  <p class="graf graf--p">
   推文塑造了他们的叙述，因为推文就是他们的叙述。在过去的七年左右，我参加的大多数街头抗议活动都是通过社交媒体进行组织的。
  </p>
  <p class="graf graf--p">
   但这并不意味着我们可以谈论“推特革命”。作为社会运动工具的 Hashtags 具有深刻的局限性，但所有其他策略也都是如此。
  </p>
  <p class="graf graf--p">
   社交媒体在革命斗争和抗议中的有用性，使我们是否能够容忍“事故”的问题进一步复杂化了 —— 即 监视。更不用说这些平台提供平等机会的“意外”，法西斯主义者也可以在特定地区之外成为“众多人”。
  </p>
  <p class="graf graf--p">
   我们是否可以容忍我们的数字习惯和依赖中必然包含对我们而言的事故？这对控制数字基础设施的人来说是理想的结果吗？
  </p>
  <p class="graf graf--p">
   在2000年的采访中，Virilio 说，“抵抗永远是可能的”，并首先要求发展“民主的技术文化”。
  </p>
  <p class="graf graf--p">
   正如他所说：“技术科学智能目前在整个社会中传播不足，以至于我们难以解释今天正在形成的各种技术科学进步。”
  </p>
  <p class="graf graf--p">
   他是对的，但答案并不是要所有人都去学习如何编码和hack（尽管学了肯定不会有害）。现在的问题是，我们的
   <strong class="markup--strong markup--p-strong">
    社交媒体生活基础设施的技术科学力量比企业和政府对城市街道基础设施的控制更加顽固和等级化
   </strong>
   。
  </p>
  <p class="graf graf--p">
   很难看出摆脱困境的方法。但这并不是说潜在的抵抗者不会尝试。
  </p>
  <p class="graf graf--p">
   例如，文化评论家本·塔诺夫（Ben Tarnoff）提出了在民主民族主义模板下将我们的数据社会化的想法，作为将权力从技术巨头手中下放给数字工地居民的途径。
  </p>
  <p class="graf graf--p">
   正如他在 Logic 杂志中解释的那样：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    无需将数据中心国有化以将数据国有化。公司可以继续在民主决定的规则下提取和改进数据 —— 但是，关键区别在于，他们是在代表我们这样做，为了我们的利益…… 为了获得提取和完善我们的数据的许可，公司将被要求将其一定比例的数据收入支付交给主权财富基金，无论是现金还是股票。
   </em>
  </p>
  <p class="graf graf--p">
   数据应该是私有的 —— 如果我们能够真正民主地确定管理公司如何提取数据的规则以及为什么提取 —— 社交媒体网站将看起来非常不同。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    只要大数据仍然没有民主化，我们就无法谈论“民主化的技术文化”，单靠监管不会让我们实现民主化
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Tarnoff 强调了民主化数据重组所面临的困难和风险：“透明度、协调性、自动化 —— 如果这些都具有民主可能性，那么它们也具有专制的可能性。” 这样的想法本身就是考虑了可能的技术未来的道德规范及其所包含的事故。
  </p>
  <p class="graf graf--p">

   而这本书更接地气、也更伤心，它立足于斯诺登启示六周年后一切都没能改变的残酷事实，指出：今天的状况是我们所有人的共谋 — — 最初也许是一无所知的，但当知道了一切之后仍没有改观，不仅是一直以来的抵抗方式出现了错误，并且，是因为回避了我们自己与这个互联网的关系。⚪️
