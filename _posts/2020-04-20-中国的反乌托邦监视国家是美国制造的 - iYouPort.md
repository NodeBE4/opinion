   如今，也许这个标题已经不会再让你感觉奇怪了。IYP有一个栏目：“

  </p>
  <p class="graf graf--p">
   以下是 Jevan Hutson 的文章，他只是从一个角度上解释了这件事。Hutson 是技术政策倡导者和HCI研究员，他的领域是面部识别和AI，于是他从自己的角度进行了一些总结，并发表了意见。
  </p>
  <p class="graf graf--p">
   在文章开头，他特别指出：“本文只代表作者的意见，而非美国政府任何机构的意见”。
  </p>
  <p class="graf graf--p">
   2019年10月7日，美国商务部将28家中国科技巨头和政府实体列入黑名单，理由是这些公司在“残酷镇压中国少数民族”中发挥的作用。
  </p>
  <p class="graf graf--p">
   被禁止的公司中包括该行业领先的人工智能和面部识别技术公司。
  </p>
  <p class="graf graf--p">
   面部识别与监控摄像机的镜头、社交媒体上发布的照片​​、及政府签发的各种证件上的身份照片等海量数据基础设施相结合，无疑会
   <strong class="markup--strong markup--p-strong">
    威胁到基本的人身自由，并促进新形式的专制控制和压迫
   </strong>
   。
  </p>
  <p class="graf graf--p">
   媒体经常报道这些新兴技术及其与中国威权监视系统的集成：中国的社会信用体系似乎直接取自美剧《黑镜》；抗议者在香港拆除了面部识别摄像机；对维吾尔族人口的技术型压迫，唤起了人们对集中营恶劣条件的记忆……
  </p>
  <p class="graf graf--p">
   ⚠️这些故事都是真实的。但是，
   <strong class="markup--strong markup--p-strong">
    媒体没有说出的是：美国研究人员和组织在其中发挥着积极的共谋作用。
   </strong>

   在被列入黑名单之前，SenseTime 在去年早些时候因其与 Leon Technology 在新疆的合资企业而受到批评，
   <strong class="markup--strong markup--p-strong">
    该公司在中国
   </strong>

     新疆的监视项目
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    里仅仅一年就从所谓的“安全”业务中获得了
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://web.archive.org/web/20180703011001/http:/www.xj.gov.cn/info/10701/67479.htm" href="https://web.archive.org/web/20180703011001/http:/www.xj.gov.cn/info/10701/67479.htm" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     42％的收入
    </strong>
   </a>
   —— 在新疆针对数百万少数民族人口。
  </p>
  <p class="graf graf--p">
   麻省理工学院的 SenseTime 基金得到了被美国商务部列入黑名单的另一家中国监控技术巨头 iFlyTek 大量捐款的支持，而人权观察组织在2017年就表达了对该公司在新疆警方监视计划中
   <a class="markup--anchor markup--p-anchor" data-href="https://www.reuters.com/article/us-china-xinjiang-mit-tech-insight/risky-partner-top-u-s-universities-took-funds-from-chinese-firm-tied-to-xinjiang-security-idUSKCN1TE04M" href="https://www.reuters.com/article/us-china-xinjiang-mit-tech-insight/risky-partner-top-u-s-universities-took-funds-from-chinese-firm-tied-to-xinjiang-security-idUSKCN1TE04M" rel="noopener noreferrer" target="_blank">
    收集海量数据
   </a>
   的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hrw.org/sites/default/files/supporting_resources/201710asia_china_iflytek.pdf" href="https://www.hrw.org/sites/default/files/supporting_resources/201710asia_china_iflytek.pdf" rel="noopener noreferrer" target="_blank">
    关注
   </a>
   。
  </p>
  <p class="graf graf--p">
   关于中国
   <strong class="markup--strong markup--p-strong">
    监控技术巨头 iFlyTek
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%b8%ad%e5%9b%bd%e5%9c%a8%e8%a5%bf%e8%97%8f%e6%9e%84%e5%bb%ba%e4%ba%86%e5%bc%ba%e6%82%8d%e7%9a%84%e6%95%b0%e5%ad%97%e9%9a%94%e7%a6%bb%e5%a2%99/" href="https://www.iyouport.org/%e4%b8%ad%e5%9b%bd%e5%9c%a8%e8%a5%bf%e8%97%8f%e6%9e%84%e5%bb%ba%e4%ba%86%e5%bc%ba%e6%82%8d%e7%9a%84%e6%95%b0%e5%ad%97%e9%9a%94%e7%a6%bb%e5%a2%99/" rel="noopener noreferrer" target="_blank">
     中国在西藏构建了强悍的数字隔离墙
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%ba%e5%b7%a5%e6%99%ba%e8%83%bd%e6%96%b0%e5%86%b7%e6%88%98%e2%80%8a-%e2%80%8a-ai%e5%92%8c5g%e5%a6%82%e4%bd%95%e5%8f%98%e6%88%90%e4%b8%80%e5%9c%ba%e5%99%a9%e6%a2%a6/" href="https://www.iyouport.org/%e4%ba%ba%e5%b7%a5%e6%99%ba%e8%83%bd%e6%96%b0%e5%86%b7%e6%88%98%e2%80%8a-%e2%80%8a-ai%e5%92%8c5g%e5%a6%82%e4%bd%95%e5%8f%98%e6%88%90%e4%b8%80%e5%9c%ba%e5%99%a9%e6%a2%a6/" rel="noopener noreferrer" target="_blank">

  <p class="graf graf--p">
   从美国政策的角度来看，使情况更糟的是，
   <strong class="markup--strong markup--p-strong">
    与针对维吾尔族种族镇压有关的这些公司的资金已与美国军事机构（如陆军研究实验室和海军研究实验室）的资金集中在一起，用于
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://arxiv.org/pdf/1904.06317v1.pdf" href="https://arxiv.org/pdf/1904.06317v1.pdf" rel="noopener noreferrer" target="_blank">

     具体

   <strong class="markup--strong markup--p-strong">
    的
   </strong>

    <strong class="markup--strong markup--p-strong">
     项目
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    中
   </strong>
   。
  </p>
  <p class="graf graf--p">
   去年十月份被列入黑名单后，麻省理工学院自称 “正在审查” 其与 SenseTime 的关系，但是，此后没有任何后续行动的报道。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    麻省理工学院并不是美国机构与协助针对维吾尔族人口镇压的公司有联系的唯一例子，美国政府机构也在其中参与
   </strong>
   。
  </p>
  <p class="graf graf--p">
   伊利诺伊大学厄巴纳-香槟分校计算机科学系的
   <a class="markup--anchor markup--p-anchor" data-href="https://arxiv.org/pdf/1811.10144v1.pdf" href="https://arxiv.org/pdf/1811.10144v1.pdf" rel="noopener noreferrer" target="_blank">
    另一个例子表明
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    美国一家主要情报机构和中国监视公司 CloudWalk 共同筹集了资金 —— 这是一家致力于根据表型
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2019/04/14/technology/china-surveillance-artificial-intelligence-racial-profiling.html" href="https://www.nytimes.com/2019/04/14/technology/china-surveillance-artificial-intelligence-racial-profiling.html" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     自动检测维吾尔族和藏族人口
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    的中国科技创业公司。
   </strong>
  </p>
  <p class="graf graf--p">
   这些例子说明，美国机构在开发和出口用于压迫维吾尔族人民的强大监视工具方面同谋，这应该引起我们集体的恐惧和愤怒。
  </p>
  <p class="graf graf--p">
   但是，如果我们不同时审视美国境内相同技术的部署，这种愤怒是无效的。
  </p>
  <p class="graf graf--p">
   许多
   <strong class="markup--strong markup--p-strong">
    相同的机构
   </strong>
   都对面部识别和其他信息技术在我们自己国家内部实现的有组织的系统性压迫负有责任。
  </p>
  <p class="graf graf--p">
   国土安全部（DHS）、移民和海关执法局（ICE）、帕兰提尔、和亚马逊合作开发技术，以加强
   <a class="markup--anchor markup--p-anchor" data-href="https://mijente.net/2018/10/23/whos-behind-ice-the-tech-companies-fueling-deportations/" href="https://mijente.net/2018/10/23/whos-behind-ice-the-tech-companies-fueling-deportations/" rel="noopener noreferrer" target="_blank">
    大规模驱逐出境
   </a>
   的行为。（CIA是帕兰提尔公司的资助人，也是唯一的客户）
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    我们自己的政府机构收集了大规模的个人信息（例如车牌）、并收集生物特征信息（例如 指纹、虹膜扫描和面部扫描），以前所未有的速度识别、分类和监视人民。
   </strong>
  </p>
  <p class="graf graf--p">
   这些系统使用自动化来加快
   <strong class="markup--strong markup--p-strong">
    种族化警察国家
   </strong>
   的速度和规模，这具有放大受益于此类系统的人的权力、同时又起到了剥夺边缘化群体权利的作用。
  </p>
  <p class="graf graf--p">
   从斯诺登（Snowden）曝光的大规模监视、到地方警察的军事化、全国各地政府机构的过失和虐待 —— 导致民权主义者和政策制定者在华盛顿、加利福尼亚、马萨诸塞州和田纳西州通过了严格的地方监视法规，以及国家措施和运动。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%ad%a6%e5%af%9f%e5%9b%bd%e5%ae%b6%e7%9a%84%e6%a2%a6%e6%83%b3/" href="https://www.iyouport.org/%e8%ad%a6%e5%af%9f%e5%9b%bd%e5%ae%b6%e7%9a%84%e6%a2%a6%e6%83%b3/" rel="noopener noreferrer" target="_blank">
     警察国家的梦想
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%b8%8d%e8%ae%ba%e6%94%bf%e5%ba%9c%e5%85%b3%e9%97%ad%e4%b8%8e%e5%90%a6%ef%bc%8c%e8%ad%a6%e5%af%9f%e5%9b%bd%e5%ae%b6%e9%83%bd%e5%b0%86%e7%bb%a7%e7%bb%ad%e8%93%ac%e5%8b%83%e5%8f%91%e5%b1%95%e3%80%82/" href="https://www.iyouport.org/%e4%b8%8d%e8%ae%ba%e6%94%bf%e5%ba%9c%e5%85%b3%e9%97%ad%e4%b8%8e%e5%90%a6%ef%bc%8c%e8%ad%a6%e5%af%9f%e5%9b%bd%e5%ae%b6%e9%83%bd%e5%b0%86%e7%bb%a7%e7%bb%ad%e8%93%ac%e5%8b%83%e5%8f%91%e5%b1%95%e3%80%82/" rel="noopener noreferrer" target="_blank">

    </a>

   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%ad%a6%e5%af%9f%e5%a6%82%e4%bd%95%e6%89%bc%e6%9d%80%e6%88%91%e4%bb%ac%e8%af%b4%e5%ae%9e%e8%af%9d%e7%9a%84%e6%9d%83%e5%88%a9/" href="https://www.iyouport.org/%e8%ad%a6%e5%af%9f%e5%a6%82%e4%bd%95%e6%89%bc%e6%9d%80%e6%88%91%e4%bb%ac%e8%af%b4%e5%ae%9e%e8%af%9d%e7%9a%84%e6%9d%83%e5%88%a9/" rel="noopener noreferrer" target="_blank">
     警察如何扼杀我们说实话的权利
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   被列入黑名单的中国公司（如果总部设在美国）即使是协助针对移民、黑人活动家、和其他民间活动人士进行监视的情况下，也被视为‘主要创新者’。
  </p>
  <p class="graf graf--p">
   亚马逊首席执行官杰夫·贝索斯（Jeff Bezos）最近也不得不承认了面部识别是“经典的军民两用技术”。
   <strong class="markup--strong markup--p-strong">
    这应该拉响警报，因为美国军事资金已经与黑名单上从事面部识别的中国公司的资金集中在一起
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是，仅商务部对该技术本身的规定还远远不够。我们必须停止面部识别技术的研究、开发、采购和部署，因为这些技术令人讨厌，而且国家行为者不适当地使用它们进行监视和征服。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    我们必须营造一种环境，在这种环境下，在智力上全社会不宜为面部识别项目做出贡献，在政治上也不应该允许美国政府将有限的财政资源用于这些令人不快的技术。
   </strong>
  </p>
  <p class="graf graf--p">
   美国研究资助机构必须放弃面部识别研究，并且必须禁止面部识别技术 —— 即使是美国公司开发的这类技术也一样。⚪️
  </p>
