 <h1 class="post-head">
  中朝关系六十年：从难以捉摸的盟友到尴尬的邻居
 </h1>

 <div class="post-body">
  <p>
   <strong>
    □“政见”观察员 刘冉
   </strong>
  </p>
  <div>
   <a href="http://cnpolitics.org/wp-content/uploads/2013/04/75feb99b6203.jpg">

  </div>
  <div>
  </div>
  <div>
   <div>
    <p>
     作为社会主义阵营的
     <span style="font-family: 'Times New Roman';">
      “
     </span>
     <span style="font-family: 宋体;">
      兄弟
     </span>
     <span style="font-family: 'Times New Roman';">
      ”
     </span>
     <span style="font-family: 宋体;">
      ，中朝关系始终吸引着国际社会的目光，也被认为对东亚政治的风云变幻构成举足轻重的影响。如今朝鲜屡屡下出险棋，中国的反应至关重要；而要理解和预测两国关系的发展，对历史的长期分析必不可少。
     </span>
    </p>
    <p>
     近日，两名韩国学者
     <span style="font-family: Georgia;">
      Jae Ho Chung
     </span>
     <span style="font-family: 宋体;">
      与
     </span>
     <span style="font-family: Georgia;">
      Myung-hae Choi
     </span>
     <span style="font-family: 宋体;">
      在《
     </span>
     <span style="font-family: Georgia;">
      The Pacific Review
     </span>
     <span style="font-family: 宋体;">
      》学刊上发表研究，对
     </span>
     <span style="font-family: Georgia;">
      1949-2010
     </span>
     <span style="font-family: 宋体;">
      年的中朝关系进行分析，试图回答其中的多个谜团：两国关系经历了怎样的历史演变？两国之间是否存在有效的沟通？中朝是否构成互相信任的同盟？这些年来，中国究竟是否成功地影响了朝鲜的行动，使其符合中国自身利益？
     </span>
    </p>
    <p>
     两位研究者认为，从地缘政治来看，中国很难将朝鲜置于敌对位置；与此同时，
     <span style="font-family: Georgia;">
      1956
     </span>
     <span style="font-family: 宋体;">
      年朝鲜共产党内部的派系清洗以及朝鲜对中国长久以来的疑心至今仍影响着两国关系。中朝两国的同盟关系从一开始就充满不确定性，传统的
     </span>
     <span style="font-family: 'Times New Roman';">
      “
     </span>
     <span style="font-family: 宋体;">
      大国
     </span>
     <span style="font-family: Georgia;">
      –
     </span>
     <span style="font-family: 宋体;">
      小国
     </span>
     <span style="font-family: 'Times New Roman';">
      ”
     </span>
     <span style="font-family: 宋体;">
      分析模式并不适用
     </span>
     <span style="font-family: 'Times New Roman';">
      ——
     </span>
     <span style="font-family: 宋体;">
      事实上，作为大国的中国始终处于两难局面中，面对朝鲜的过激行为却难以态度强硬，教条化的回应往往并不构成压力；与此同时，朝鲜却成为一个
     </span>
     <span style="font-family: 'Times New Roman';">
      “
     </span>
     <span style="font-family: 宋体;">
      非典型小国
     </span>
     <span style="font-family: 'Times New Roman';">
      ”
     </span>
     <span style="font-family: 宋体;">
      ，不受强大的中国邻居摆布。
     </span>
    </p>
    <p>
     <strong>
      中朝是否有良好的沟通渠道？
     </strong>
    </p>
    <p>
     从历史来看，尽管朝鲜半岛长期将中国视为宗主国和保护国，但二者之间并非简单的庇护关系，而是
     <span style="font-family: 'Times New Roman';">
      “
     </span>
     <span style="font-family: 宋体;">
      唇齿相依
     </span>
     <span style="font-family: 'Times New Roman';">
      ”
     </span>
     <span style="font-family: 宋体;">
      和
     </span>
     <span style="font-family: 'Times New Roman';">
      “
     </span>
     <span style="font-family: 宋体;">
      唇亡齿寒
     </span>
     <span style="font-family: 'Times New Roman';">
      ”
     </span>
     <span style="font-family: 宋体;">
      的对等关系
     </span>
     <span style="font-family: 'Times New Roman';">
      ——
     </span>
     <span style="font-family: 宋体;">
      这意味着
     </span>
     <span style="font-family: 'Times New Roman';">
      “
     </span>
     <span style="font-family: 宋体;">
      牙齿
     </span>
     <span style="font-family: 'Times New Roman';">
      ”
     </span>
     <span style="font-family: 宋体;">
      也有咬到
     </span>
     <span style="font-family: 'Times New Roman';">
      “
     </span>
     <span style="font-family: 宋体;">
      嘴唇
     </span>
     <span style="font-family: 'Times New Roman';">
      ”
     </span>
     <span style="font-family: 宋体;">
      的可能。
     </span>
    </p>
    <p>
     1940-50
     <span style="font-family: 宋体;">
      年代，大批参加过中国共产党及军队的朝鲜族人在金日成的要求下被派遣回朝鲜支援建设，形成朝鲜共产党内部重要的
     </span>
     <span style="font-family: 'Times New Roman';">
      “
     </span>
     <span style="font-family: 宋体;">
      延安派
     </span>
     <span style="font-family: 'Times New Roman';">
      ”
     </span>
     <span style="font-family: 宋体;">
      ，并与中共保持密切互动。在中共的支持下，延安派迅速崛起，这很快引发了金日成的疑虑。
     </span>
     <span style="font-family: Georgia;">
      1956
     </span>
     <span style="font-family: 宋体;">
      年，金日成对包括延安派在内的党内派系进行清洗，随即宣布朝鲜不接受任何国外势力的干涉，并要求中国军队撤出。这无疑毁掉了中朝之间最有效的沟通渠道，也为其后中朝关系的发展造成挥之不去的阴影。
     </span>
    </p>
    <p>
     金日成在
     <span style="font-family: Georgia;">
      1949-1977
     </span>
     <span style="font-family: 宋体;">
      年间委任的七任朝鲜驻中国大使和四任外交部部长全部与延安派毫无干系，说明他对中国及延安派始终怀有严重的疑心。正因如此，中国不得不试图采用其他途径与朝鲜高层沟通。
     </span>
    </p>
    <p>
     1958
     <span style="font-family: 宋体;">
      年，周恩来访朝期间，双方达成高层互访共识，这意味着两国最高领导人将保持密切往来，并就重要事件保持直接联系。因此，其后每当金日成和金正日访问中国，几乎都能够得到与全体政治局常委直接会晤的机会。
     </span>
    </p>
    <p>
     这种非常规的高层沟通渠道使外交部等正式部门在中朝关系上沦为附庸，就连被认为在对朝关系中起到重要作用的中共中央对外联络部事实上也受到很大限制。与此同时，中国驻朝领事机构在情报收集工作方面也不尽人意。
    </p>
    <p>
     在这种情况下，高层互访几乎成为中国判断朝鲜意图的关键手段，这导致中国屡屡受朝鲜误导，作出错误分析，因而对朝鲜的多次意外举措未能及时获悉，有时消息甚至不如西方国家灵通。
    </p>
    <p>
     <strong>
      中朝是否是相互信任的盟友？
     </strong>
    </p>
    <p>
     作者认为，中朝两国从来都不是互相信任的盟友。
    </p>
    <p>
     中国与朝鲜最初达成同盟主要是为了应对苏联的威胁，但当时，朝鲜却需要同时依赖苏联去对抗美国。
     <span style="font-family: Georgia;">
      60
     </span>
     <span style="font-family: 宋体;">
      年代朝鲜对美国的几次挑衅都得到了中国台面上的支持，但与此同时，中国与美国已在秘密计划恢复外交关系。
     </span>
     <span style="font-family: Georgia;">
      70
     </span>
     <span style="font-family: 宋体;">
      年代后，中朝两国的政策更是背道而驰，同盟关系进一步削弱。
     </span>
     <span style="font-family: Georgia;">
      80-90
     </span>
     <span style="font-family: 宋体;">
      年代，中国通过睦邻友好外交政策在亚洲交到了不少朋友，使朝鲜愈发失去了原先重要的战略地位。
     </span>
     <span style="font-family: Georgia;">
      1992
     </span>
     <span style="font-family: 宋体;">
      年的中韩关系正常化更是给朝鲜当头一棒。
     </span>
    </p>
    <p>
     长久以来的猜疑使朝鲜无法将中国视为可靠的保护者，因此在这段时间内致力于发展核武器和远距离武器，并与第三世界国家的潜在盟友发展关系，甚至开始对西欧和日本有限度开放。
     <span style="font-family: Georgia;">
      1994
     </span>
     <span style="font-family: 宋体;">
      年，朝鲜未与中国商议便自行宣布退出板门店停战协定，随即要求中国也退出。朝鲜半岛第一次核危机后，朝鲜与美国发布联合宣言，这一过程中，中国的角色大受限制。
     </span>
    </p>
    <p>
     不过，即使事情发展到如此地步，中朝也并未废止或修改过被视为两国同盟关系根基的《中朝友好合作互助条约》。对中国而言，这一方面是因为美韩同盟的存在，另一方面也是为了通过条约中
     <span style="font-family: 'Times New Roman';">
      “
     </span>
     <span style="font-family: 宋体;">
      就重要事件互相协商
     </span>
     <span style="font-family: 'Times New Roman';">
      ”
     </span>
     <span style="font-family: 宋体;">
      的条款而对朝鲜保留一定的影响力。
     </span>
    </p>
    <p>
     <strong>
      中国对朝鲜的影响力有多大？
     </strong>
    </p>
    <p>
     关于中国是否能够影响朝鲜政策，学界有着不同的看法。
    </p>
    <p>
     一派认为中国的影响力本身有限，虽然在经济方面受到朝鲜依赖，却难以将影响转化到其他领域；另一派则认为，由于朝鲜对外来干涉极其敏感，加之中国希望保持稳定的地区环境，因此应主动选择对朝鲜采取相对
     <span style="font-family: 'Times New Roman';">
      “
     </span>
     <span style="font-family: 宋体;">
      温柔
     </span>
     <span style="font-family: 'Times New Roman';">
      ”
     </span>
     <span style="font-family: 宋体;">
      的态度，以保证长久的影响力。
     </span>
    </p>
    <p>
     两位韩国研究者则指出，要想全面分析中朝关系，应当着重看待朝鲜如何回应中国对朝鲜半岛事务的提议。
    </p>
    <p>
     1970
     <span style="font-family: 宋体;">
      年代以来，中国开始尝试敦促朝鲜与美韩改善关系并展开谈判，但朝鲜坚持拒绝将韩国视为有权参与谈判的一方。
     </span>
     <span style="font-family: Georgia;">
      1980
     </span>
     <span style="font-family: 宋体;">
      年代，中国希望朝鲜参与在北京举行的三边谈判，但朝鲜阳奉阴违，在仰光对金斗焕实行爆炸刺杀，使朝鲜半岛局势降至冰点，中国失去了从中斡旋的机会。
     </span>
     <span style="font-family: Georgia;">
      1996
     </span>
     <span style="font-family: 宋体;">
      年，中国以大量军事、粮食与经济援助诱使朝鲜参与四边会谈，但朝鲜坚持所有会议在日内瓦而非北京举行，并明确宣布
     </span>
     <span style="font-family: 'Times New Roman';">
      “
     </span>
     <span style="font-family: 宋体;">
      朝鲜民族内部事务绝不应由四边会谈决定
     </span>
     <span style="font-family: 'Times New Roman';">
      ”
     </span>
     <span style="font-family: 宋体;">
      。
     </span>
     <span style="font-family: Georgia;">
      1998
     </span>
     <span style="font-family: 宋体;">
      年，朝鲜试射大浦洞飞弹，导致四边会谈破裂，最终也未能达成实质性进展。
     </span>
     <span style="font-family: Georgia;">
      2003
     </span>
     <span style="font-family: 宋体;">
      年以来的六边会谈中，中国十分认真且积极地希望扮演重要的协调角色，无奈朝鲜不领情，屡次以核试验来加强筹码，意图与美国直接叫板，排斥和淡化中国的作用。
     </span>
    </p>
    <p>
     作者统计了
     <span style="font-family: Georgia;">
      2003-2011
     </span>
     <span style="font-family: 宋体;">
      年中朝两国领导人会面记录，指出：在
     </span>
     <span style="font-family: Georgia;">
      23
     </span>
     <span style="font-family: 宋体;">
      次会面中，只有
     </span>
     <span style="font-family: Georgia;">
      8
     </span>
     <span style="font-family: 宋体;">
      次对敦促朝鲜参与三边或六边谈判产生了实质影响。
     </span>
     <span style="font-family: Georgia;">
      2007
     </span>
     <span style="font-family: 宋体;">
      年之后，该
     </span>
     <span style="font-family: 'Times New Roman';">
      “
     </span>
     <span style="font-family: 宋体;">
      影响率
     </span>
     <span style="font-family: 'Times New Roman';">
      ”
     </span>
     <span style="font-family: 宋体;">
      更是由
     </span>
     <span style="font-family: Georgia;">
      2003-06
     </span>
     <span style="font-family: 宋体;">
      年间的
     </span>
     <span style="font-family: Georgia;">
      58%
     </span>
     <span style="font-family: 宋体;">
      迅速降低至不足
     </span>
     <span style="font-family: Georgia;">
      10%
     </span>
     <span style="font-family: 宋体;">
      。这有可能意味着：朝鲜对其核能力的自信获得增长，也因此愈来愈不买中国的面子。
     </span>
    </p>
    <p>
     <strong>
      非典型
      <span style="font-family: 'Times New Roman';">
       “
      </span>
      <span style="font-family: 宋体;">
       大国
      </span>
      <span style="font-family: Georgia;">
       –
      </span>
      <span style="font-family: 宋体;">
       小国
      </span>
      <span style="font-family: 'Times New Roman';">
       ”
      </span>
      <span style="font-family: 宋体;">
       关系
      </span>
     </strong>
    </p>
    <p>
     作者总结道，作为一个
     <span style="font-family: 'Times New Roman';">
      “
     </span>
     <span style="font-family: 宋体;">
      非典型小国
     </span>
     <span style="font-family: 'Times New Roman';">
      ”
     </span>
     <span style="font-family: 宋体;">
      ，朝鲜非常清楚如何利用其在地缘政治中的重要地位，并在两个超级大国之间走钢丝，巧妙地以十分危险的举动达到自己的目的而不受惩罚。
     </span>
    </p>
    <p>
     与此同时，中国屡次希望朝鲜参与多边谈判的尝试均宣告失败，其与朝鲜的关系却没有发生过实质改变，而是陷入教条式的循环往复。
    </p>
    <p>
     作者最后指出，正在发生的又一次朝鲜半岛危机恐怕也很难打破中朝关系的怪圈，这无疑会令国际社会深感失望；但可以肯定的是，朝鲜将愈来愈不再是中国可以信任的战略盟友，而是成为令中国如坐针毡的恼人邻居。
    </p>
