  <p class="graf graf--p">
   中国社会最根源的问题是封闭。从国际角度上来看，不论是政府的民族主义信息战，还是民间遭受的人权苦难，其面目都是一种受害者的形象，似乎迫切需要的只是被关心。事实上，中国社会的反抗真正需要的是力量的增强、是参与，是与国际反抗力量的并肩作战。
  </p>
  <p class="graf graf--p">
   美国的公民社会活动家有很多技巧值得中国社会学习。希望这篇文章能带来帮助。
  </p>
  <p class="graf graf--p">
   目前，美国的进步政治活动正在上升。在充满活力的当地社区中，问题是切实的而不是抽象的。
   <strong class="markup--strong markup--p-strong">
    年轻人和那些刚刚接触激进主义的人们有机会迅速发挥重要作用
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在这里，经常会将人们拉开差距的僵化意识形态界限并不那么重要，人们有机会在一个共同的事业中作为邻居聚集在一起。非常令人羡慕，这就是
   <strong class="markup--strong markup--p-strong">
    IYP 一直期待的去意识形态隔膜
   </strong>
   ，我们的团结应该建立在共同目标的基础上，而不是任何一种派系斗争。
  </p>
  <p class="graf graf--p">
   在美国，许多地方问题仍然存在于更大的问题之中。从污染到警察暴力，社区面临的挑战是如何更好地与更大范围的力量联系在一起，并且有充分的战略理由将两者之间的点联系起来。
  </p>
  <p class="graf graf--p">
   让我们的战斗力变得更强大，增强我们的力量。把国内反抗和全球力量结合起来，展示受其影响的真实的人和地区，将非常有益于推动本地运动。
  </p>
  <p class="graf graf--p">
   那么，我们怎样才能使地方激进主义战斗变得更大呢？根据美国公民社会40多年来的经验，涉及不同问题和地点的激进主义有三个基本策略。
  </p>
  <p class="graf graf--p">
   以下每一点都非常重要。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1. 扩大叙事范围
   </strong>
  </p>
  <p class="graf graf--p">
   强大的激进运动的根本力量是一个强大的故事。它必须清晰、引人注目且
   <strong class="markup--strong markup--p-strong">
    准确
   </strong>
   。然后它必须将本地故事与更大的全球图景联系起来。
  </p>
  <p class="graf graf--p">
   曾经的中国社会的激进力量一般程度上可以做到清晰，甚至引人注目，但是它经常不够准确，与缺乏本地独立媒体以及独立调查跟进不足有关，也与参与者的经验不足有关。突发事件出现的同时就是假消息盛行的开始，这在任何国家都不是新奇的事，但是，作为活动家和其组织，必需能够控制真相在手，并完成尽可能准确的叙事。这对你们的影响力来说是至关重要的。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    延伸阅读：《
    <a class="markup--anchor markup--li-anchor" data-href="https://medium.com/@iyouport/https-medium-com-iyouport-disinformation-thinking-eed8534b04ac" href="https://www.iyouport.org/%E4%B8%BA%E4%BB%80%E4%B9%88%E8%99%9A%E5%81%87%E4%BF%A1%E6%81%AF%E4%BB%A4%E7%8B%AC%E8%A3%81%E6%94%BF%E6%9D%83%E5%8F%98%E5%BE%97%E6%9B%B4%E5%BC%BA%E5%A4%A7%EF%BC%8C%E4%BB%A4%E6%B0%91%E4%B8%BB%E5%9B%BD-2/" rel="noopener noreferrer" target="_blank">
     为什么维护正义需要足够的诚实
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   Cochabamba 的故事是这样的：在20世纪90年代末，世界银行强迫玻利维亚将 Cochabamba 的公共供水系统私有化，使其成为帮助扩大获得清洁水的条件。美国企业巨头柏克德（Bechtel）的子公司控制了水资源，并在一夜之间将贫困家庭的支出提高了
   <a class="markup--anchor markup--p-anchor" data-href="https://democracyctr.org/archive/the-water-revolt/cochabambas-water-bills-from-bechtel/" href="https://democracyctr.org/archive/the-water-revolt/cochabambas-water-bills-from-bechtel/" rel="noopener noreferrer" target="_blank">
    一倍甚至更多
   </a>
   。
  </p>
  <p class="graf graf--p">
   大规模的公众起义令这个拥有50万人口的城市全面调动起来了，最终迫使该公司离开。为了报复，Bechtel 在世界银行的一个贸易法庭上起诉 Cochabamba 人民，这是一项价值
   <a class="markup--anchor markup--p-anchor" data-href="https://democracyctr.org/archive/the-water-revolt/bechtel-vs-bolivia-details-of-the-case-and-the-campaign/" href="https://democracyctr.org/archive/the-water-revolt/bechtel-vs-bolivia-details-of-the-case-and-the-campaign/" rel="noopener noreferrer" target="_blank">
    5000 万美元的诉讼
   </a>
   。
  </p>
  <p class="graf graf--p">
   Lockport 的故事是这样的：为了应对枪击事件，全国各地的学区都在忙着做一些关于安全的工作。看到机会后，一家加拿大赌场安保公司说服 Lockport 学区在一个高科技面部识别摄像系统上
   <a class="markup--anchor markup--p-anchor" data-href="https://www.lockportjournal.com/news/local_news/trying-for-more-secure-schools-lockport-district-turning-to-facial/article_f1cc9cfa-0898-5da0-ac5d-d600df21bed7.html" href="https://www.lockportjournal.com/news/local_news/trying-for-more-secure-schools-lockport-district-turning-to-facial/article_f1cc9cfa-0898-5da0-ac5d-d600df21bed7.html" rel="noopener noreferrer" target="_blank">
    投入超过 270 万美元
   </a>
   的技术教育资金，该系统与该国其他任何其他地方使用的都不同，引发了严重的隐私担忧。
  </p>
  <p class="graf graf--p">
   就是这两个故事。
  </p>
  <p class="graf graf--p">
   在上述两种情况下，
   <strong class="markup--strong markup--p-strong">
    战略是相同的 — — 将实地故事与较大的参与者和问题联系起来
   </strong>
   。开始，考虑以下问题：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    是否有公司或更大的组织在幕后推动决策？
   </li>
   <li class="graf graf--li">
    哪些州或联邦法律影响当地问题？
   </li>
   <li class="graf graf--li">
    当地的战斗是否反映了全国甚至全球的争论？
   </li>
  </ul>
  <p class="graf graf--p">
   这就是*连接*。
  </p>
  <p class="graf graf--p">
   活动家应该将这些问题的答案
   <strong class="markup--strong markup--p-strong">
    作为叙述的核心部分
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2. 加宽注意力
   </strong>
  </p>
  <p class="graf graf--p">
   接下来，这个故事需要被广泛分布的媒体所吸取，这是
   <strong class="markup--strong markup--p-strong">
    借力
   </strong>
   ，比一个社区的单打独斗更重要。
  </p>
  <p class="graf graf--p">
   如果这是一个影响力较大的好故事，就像棱镜文件或者巴拿马文件，那么就并不需要做太多，如果不是这种程度的故事（绝大多数时候都不是），那么继续以下步骤：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    充分的独立研究，查找报道这个故事的记者并与他们分享您的研究成果；
   </li>
   <li class="graf graf--li">
    将记者与可以作为文章中可靠来源的当地居民联系起来；
   </li>
   <li class="graf graf--li">
    自己撰写文章，目标是获取覆盖当地社区
    <strong class="markup--strong markup--li-strong">
     之外的
    </strong>
    受众的出版物发表。
   </li>
  </ul>
  <p class="graf graf--p">
   回头看中国社会的问题在于两点：1、公民社会组织的运作不够积极，尤其是社区间协作不佳；2、天然地误认为媒体就应该主动去抓取“我们的故事”。
   <strong class="markup--strong markup--p-strong">
    这两点都是很糟糕的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   活动家的工作就是要动起来，你需要认识到媒体 — 尤其是那些霸占了90%市场份额的主流媒体 — 并不会因为你的苦难足够深刻而主动跑来关心你，他们考虑的唯一价值是报道能吸引多少流量。这是他们要的，而不是用来接受我们的道德评价的。就如主流媒体关心马云和范冰冰可以赚取充分的社会眼球，关注普通老百姓的奋斗则很难满足广告商的需求。于是
   <strong class="markup--strong markup--p-strong">
    活动家的工作
   </strong>
   就是要说服这些记者让他们相信，这个故事如果被发表可以做到足够轰动。
  </p>
  <p class="graf graf--p">
   在 Bolivia，通过在远离 Cochabamba 的地区的出版物中撰写有关本地故事的文章，例如“旧金山纪事报”、 In These Times，以及民主中心的全球
   <a class="markup--anchor markup--p-anchor" data-href="https://democracyctr.org/archive/the-water-revolt/dispatches-from-the-streets/" href="https://democracyctr.org/archive/the-water-revolt/dispatches-from-the-streets/" rel="noopener noreferrer" target="_blank">
    电子邮件简报
   </a>
   ，以此来帮助
   <strong class="markup--strong markup--p-strong">
    使本地故事成为全球性的故事
   </strong>
   。
  </p>
  <p class="graf graf--p">
   该文章很快引起了对新兴全球化问题感兴趣的其他记者的关注，并引发了纽约客和 PBS 以及一些纪录片的报道。
  </p>
  <p class="graf graf--p">
   所有这一切都巩固了水起义作为一个全球性故事的力量。
  </p>
  <p class="graf graf--p">
   在 Lockport 监控摄像头的灾难上，最初在当地日报上多次撰写这个问题。并使用这些文章去说服 Buffalo News 的记者让他们相信这是一个非常好的地区故事，然后该记者的作品引起了国际关注，因为来自 intercept、华盛顿邮报和 NPR 的转述报道接踵而来了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    获得这种更大的覆盖范围也会更容易在当地建立你的权力，这是一个反作用力，因为它使地方官员受到更广泛的审查，使他们感到紧张
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3. 拓宽你的联盟
   </strong>
  </p>
  <p class="graf graf--p">
   本国的和全球的倡导团体需要本地故事，以真实性和权威注入其行为。你可以通过与他们结盟来帮助他们完成任务和在当地的努力。请记住，这些人是资源有限的人，因此
   <strong class="markup--strong markup--p-strong">
    请关注
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    寻找符合您的问题的组织；
   </li>
   <li class="graf graf--li">
    与那些对您的问题表现出兴趣的组织内的个人联系。说服他们，您需要像这样的内部倡导者将您的故事推到优先级列表中；
   </li>
   <li class="graf graf--li">
    分享您的故事的
    <strong class="markup--strong markup--li-strong">
     简明并引人注目
    </strong>
    的摘要，以便潜在的盟友可以毫不费力地看出它是否适合他们的倡导工作。
   </li>
  </ul>

   <noscript>
    <img alt="" class="size-full wp-image-2315" data-recalc-dims="1" height="797" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0.jpeg?resize=1100%2C797&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0.jpeg?w=1100&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0.jpeg?resize=300%2C217&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0.jpeg?resize=768%2C556&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0.jpeg?resize=1024%2C742&amp;ssl=1 1024w" width="1100"/>
   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-2315">
    Protesters in Cochabamba, Bolivia, in 2000. (Photo by Thomas Kruse)
   </figcaption>
  </figure>
  <p class="graf graf--p">
   在 Cochabamba 水起义的情况下，Bolivia 的团体在政策研究所、加拿大人委员会、 Public Citizen、地球正义等组织中找到了极好的盟友。他们的工作对于推动针对全球参与者（如世界银行和 Bechtel）的活动至关重要。
  </p>
  <p class="graf graf--p">
   在 Lockport，与监控摄像机的斗争赢得了纽约公民自由联盟（NYCLU）的大力支持。该组织成为了与州教育当局之间的桥梁，带来了法律专业知识和联系。它还提出了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nyclu.org/en/news/we-asked-answers-facial-recognition-schools-our-questions-remain" href="https://www.nyclu.org/en/news/we-asked-answers-facial-recognition-schools-our-questions-remain" rel="noopener noreferrer" target="_blank">
    “信息自由法”的请求
   </a>
   ，记录了该计划背后的阴暗交易。
  </p>
  <p class="graf graf--p">
   中国的社区间协作一直都挺令人遗憾的，不仅难以跨地区，甚至难以跨行业，这是一个基本层面上的失败。一场运动发生时（不论是线上还是线下）人们往往会首先去判断它属于草根维权？中产阶级？是女权？还是LGBT？是工人阶级？还是老兵？等等，认为自己不属于该群体的人们将以旁观者的角度去品评，而不是加盟。行动者本身也经常忽略联合的重要价值，这便很难扩充力量，同时也为当局的打压提供了精准的定位。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-2316 jetpack-lazy-image" data-lazy-sizes="(max-width: 952px) 100vw, 952px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-1.jpeg?resize=952%2C772&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-1.jpeg?w=952&amp;ssl=1 952w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-1.jpeg?resize=300%2C243&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-1.jpeg?resize=768%2C623&amp;ssl=1 768w" data-recalc-dims="1" height="772" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-1.jpeg?resize=952%2C772&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="952"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-2316" data-recalc-dims="1" height="772" sizes="(max-width: 952px) 100vw, 952px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-1.jpeg?resize=952%2C772&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-1.jpeg?w=952&amp;ssl=1 952w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-1.jpeg?resize=300%2C243&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-1.jpeg?resize=768%2C623&amp;ssl=1 768w" width="952"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    最后
   </strong>
  </p>
  <p class="graf graf--p">
   水起义不仅取得了一次胜利，还取得了两次胜利。首先，玻利维亚人赶走了 Bechtel 公司，随后全球团结运动迫使该公司放弃其 5000 万美元的法律诉讼。
  </p>
  <p class="graf graf--p">
   Lockport 的案子目前还在继续中。它和水起义不同，有一个时间错位，当活动家们发现这件事时地区官员已经完成了购买监视摄像机的计划。然而，由于活动家和公民社会组织及媒体的联合，促使纽约州教育部警告学区不要将该系统上线，直到官员们解决所涉及的巨大隐私问题，并且，Lockport 已经成为关于借口所谓的安全问题以实施侵权监视的国家警示故事。
  </p>
  <p class="graf graf--p">
   假设透明度革命工作足够强劲，内部举报能够提早将问题呈现在活动家和社区面前，Lockport 的案子很可能会得到更棒的结果，从根本上阻止购买。
  </p>
  <p class="graf graf--p">
   这两个截然不同的地方运动经验提供了关于如何将地方、国家和全球战斗之间的点联系起来的良好实践课程。
  </p>
  <p class="graf graf--p">
   这些相同的见解可以应用于美国的气候变化问题、拉丁美洲的企业环境破坏问题、中国的民间社会维权、以及其他各种紧急战斗。
  </p>
  <p class="graf graf--p">
   作为活动家，您应该改写陈旧的口号：在当地采取行动，但是同时要尽可能将这一斗争转化为更大区域内的一部分。您需要让反抗“走出来”，而不是原地等待聚光灯。
  </p>
