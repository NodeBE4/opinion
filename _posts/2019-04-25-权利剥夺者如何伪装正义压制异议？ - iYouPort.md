     <strong>
      <em>
       权力斗争如何以公民权利为代价。这篇文章中给出的信息和线索值得人们警惕，即 审查封锁可以完全伪装为正义和高尚，在政治宣传的掩护下公然违宪。究竟什么是事实？如果”事实”只能被当权者塑造，公民该如何保持清醒？
      </em>

  <p class="graf graf--p">
   今年10月，Facebook 和 Twitter 删除了数百名用户的账户，其中包括美国公民维护的许多自媒体，揭露警察暴行和干预主义的热门网站，如 The Free Thought Project, Anti-Media, 和 Cop Block，甚至
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/RachBlevins/status/1050487416488890370" href="https://twitter.com/RachBlevins/status/1050487416488890370" rel="noopener noreferrer" target="_blank">
    Rachel Blevins
   </a>
   等记者的页面 。
  </p>
  <p class="graf graf--p">
   Facebook 声称这些网页“违反了我们针对垃圾邮件的规则”。然而，像 The Free Thought Project 这样的网站已被 Facebook 认证，并拥有广泛的信誉作为合法的新闻和观点的来源。为自由思想做出贡献的独立记者约翰·维贝斯（John Vibes）
   <a class="markup--anchor markup--p-anchor" data-href="https://sputniknews.com/us/201810121068814924-Reporters-Pages-Shut-Down-By-Facebook/" href="https://sputniknews.com/us/201810121068814924-Reporters-Pages-Shut-Down-By-Facebook/" rel="noopener noreferrer" target="_blank">
    指责
   </a>
   Facebook
   <strong class="markup--strong markup--p-strong">
    “偏爱主流资源，并摒弃其他声音”。
   </strong>
  </p>
  <p class="graf graf--p">
   在这些首次发表的评论中，一位新保守派华盛顿内部人士显然声称对最近的大清洗给予支持 — ，并承诺在不久的将会有更多的内容审查。
  </p>
  <p class="graf graf--p">
   Jamie Fly 说，“这[审查]只是一个开始”。他是有影响力的智库德国马歇尔基金会亚洲项目的高级研究员和主任，该基金会由
   <a class="markup--anchor markup--p-anchor" data-href="http://www.gmfus.org/our-partners" href="http://www.gmfus.org/our-partners" rel="noopener noreferrer" target="_blank">
    美国政府和北约组织资助
   </a>
   。
  </p>
  <p class="graf graf--p">
   Fly 继续抱怨说人们建立一个 Facebook 或 Twitter 帐户太容易了，“你需要的只是一封电子邮件”，他认为网站对公众开放的可访问性是危险的。他预测在全球范围内需要长期努力来解决问题，并指出“需要时刻保持警惕”。
  </p>
  <p class="graf graf--p">
   Fly 向 Jeb Sprague 发表了这些惊人的评论，Sprague 是加州大学圣巴巴拉分校社会学访问学者，也是本文的合著者。两人在午餐休息时间交流，这是德国柏林的 Stiftung Wissenschaft und Politik 组织的亚洲安全会议。
  </p>
  <p class="graf graf--p">
   Fly 的言论 — “[审查]只是刚刚开始” — 似乎证实了小众在线媒体社区最担心的问题。即
   <strong class="markup--strong markup--p-strong">
    最近的大清洗是出于政治目的，而不是什么垃圾邮件的预防，并受到了强烈的利益驱使，这种利益非常不利于持不同政见者的观点。
   </strong>
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Jamie Fly 一个新保守派政客的崛起
   </strong>
  </p>
  <p class="graf graf--p">
   Jamie Fly 是一位有影响力的外交政策强硬派，他去年一直在游说对社交媒体上所谓的“边缘观点”的审查。多年来，他一直主张对伊朗进行军事攻击、对叙利亚进行政权更迭战，并将军费开支提升到前所未有的水平。他是新保守派政客的化身。
  </p>
  <p class="graf graf--p">
   像许多第二代新保守主义者一样，Fly 通过钻营乔治·W·布什的国家安全委员会和国防部的中级职位而
   <a class="markup--anchor markup--p-anchor" data-href="https://rightweb.irc-online.org/profile/fly_jamie/" href="https://rightweb.irc-online.org/profile/fly_jamie/" rel="noopener noreferrer" target="_blank">
    进入政府
   </a>
   。
  </p>
  <p class="graf graf--p">
   2009 年，他被任命为外交政策倡议（FPI）的主任，这是 Bill Kristol 的新美国世纪项目（PNAC）的重新命名版本。后者是一群由新保守派活动家组成的小组，他们首先提出入侵伊拉克的提案。到 2011 年，Fly
   <a class="markup--anchor markup--p-anchor" data-href="http://www.atimes.com/atimes/Middle_East/MJ19Ak03.html" href="http://www.atimes.com/atimes/Middle_East/MJ19Ak03.html" rel="noopener noreferrer" target="_blank">
    通过主张
   </a>
   要对伊朗进行军事打击，推进了 PNAC 蓝图的下一阶段；一年后，Fly
   <a class="markup--anchor markup--p-anchor" data-href="http://www.aei.org/publication/the-case-for-regime-change-in-iran/" href="https://www.aei.org/publication/the-case-for-regime-change-in-iran/" rel="noopener noreferrer" target="_blank">
    再次敦促
   </a>
   美国“将[伊朗]核计划之外的目标清单扩大到共和国卫队和情报部的关键指挥和控制部门，以及与其他主要政府官员相关的设施。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    PropOrNot 引发了非主流媒体的恐慌
   </strong>
  </p>
  <p class="graf graf--p">
   在克林顿竞选失败后不到三个星期，华盛顿邮报的 Craig Timberg 发表了一份“可疑来源”的报道，标题为“俄罗斯的宣传努力有助于传播‘假新闻‘”。这篇文章大肆宣传麦卡锡的努力，匿名运营名为 PropOrNot 的组织，借此
   <strong class="markup--strong markup--p-strong">
    将大约 200 家美国非主流媒体列入了黑名单
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://www.propornot.com/p/the-list.html" href="http://www.propornot.com/p/the-list.html" rel="noopener noreferrer" target="_blank">
    PropOrNot 黑名单
   </a>
   上就包括最近被 Facebook 和 Twitter 清除的一些媒体组织，如 The Free Thought Project 和 Anti-Media.。PropOrNot 呼吁美国政府正式调查“被列入黑名单的网点“。
  </p>
  <p class="graf graf--p">
   华盛顿邮报记者克雷格·蒂姆伯格（Craig Timberg）被民主党人广泛称赞，前奥巴马白宫助手 Dan Pfeiffer 称这篇问责为“世界上最大的故事”。但是在其他地方遭遇了广泛的痛批，包括
   <a class="markup--anchor markup--p-anchor" data-href="https://www.newyorker.com/news/news-desk/the-propaganda-about-russian-propaganda" href="https://www.newyorker.com/news/news-desk/the-propaganda-about-russian-propaganda" rel="noopener noreferrer" target="_blank">
    “纽约客”的页面
   </a>
   也对其做出了严肃批评，结果，该文章被修改成了一份编辑说明：“[华盛顿]邮报……本身
   <strong class="markup--strong markup--p-strong">
    并不保证
   </strong>
   PropOrNot 关于任何个别媒体的调查结果的有效性。 ”
  </p>
  <p class="graf graf--p">
   PropOrNot 已经被曝光为 McCarthyite 的假货，但其背后的概念 — 即 将主流不喜欢的在线美国媒体全部说成是俄罗斯宣传，这一奇怪的审查论调 — 继续蓬勃发展。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “俄罗斯机器人跟踪器“的诞生
   </strong>
  </p>
  <p class="graf graf--p">
   到8月份，一个新的、严重相关的新举措突然出现了，该机构被称为
   <a class="markup--anchor markup--p-anchor" data-href="https://securingdemocracy.gmfus.org/" href="https://securingdemocracy.gmfus.org/" rel="noopener noreferrer" target="_blank">
    安全民主联盟
   </a>
   （ASD），具有每日“ Hamilton 68 ”在线仪表板，通过易于消化的图表突出显示所谓的机器人活动。更方便的是，该网站避免了声称要跟踪的任何数字克里姆林宫影响力帐户。审查者正在迅速进化。
  </p>

   <a class="markup--anchor markup--p-anchor" data-href="https://www.politico.com/magazine/story/2018/02/26/trump-russia-putin-bots-global-politico-transcript-217082" href="https://www.politico.com/magazine/story/2018/02/26/trump-russia-putin-bots-global-politico-transcript-217082" rel="noopener noreferrer" target="_blank">

   在 2018 年3月的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.c-span.org/video/?c4718707/russian-disinformation-efforts" href="https://www.c-span.org/video/?c4718707/russian-disinformation-efforts" rel="noopener noreferrer" target="_blank">
    C-Span 采访
   </a>

   <strong class="markup--strong markup--p-strong">
    如果禁止“另类观点”，岂不是意味着中国式的“大一统”？
   </strong>

  <p class="graf graf--p">
   并且必须知道，在 PropOrNot 的黑名单中的网站，很少会在最近的 Facebook 大清洗中被删除，而 Fly 和他的同事们则明显认为那些都是俄罗斯账户。被封锁的网站可能被指控的唯一违规行为就是发布了 Fly 和他的同伙们所认为的“另类”观点。
  </p>
  <p class="graf graf--p">
   更重要的是，
   <strong class="markup--strong markup--p-strong">
    ASD 已经被迫承认，最初被认定为“俄罗斯机器人”的 Twitter 账户有很多不一定是机器人 — 也可能并不是俄罗斯人
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “我不相信关于这个机器人的事”
   </strong>
  </p>
  <p class="graf graf--p">
   本文的共同作者 Max Blumenthal 于 2017 年11月 进行的一项调查发现，ASD 的“Hamilton 68”仪表板是“在国安机构内运作的一些顶尖人物的支持下的一系列歪曲、反击、在线骚扰者和偏执狂的集合”。
   <strong class="markup--strong markup--p-strong">
    它已经对著名异议人士发起了很多抹黑和攻击，包括普利策奖得主、the intercept 编辑 Glenn Greenwald。
   </strong>
  </p>
  <p class="graf graf--p">
   该计划的支持者中包括 Andrew Weisburd 和 Clint Watts，他们被引用为华盛顿邮报推广 PropOrNot 的专家。这个 Watts 曾多次在国会大喊大叫，要求政府“平息信息叛乱”。
  </p>
  <figure aria-describedby="caption-attachment-506" class="wp-caption aligncenter" id="attachment_506" style="width: 1024px">
   <img alt="" class="wp-image-506 size-large jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-37.jpeg?resize=1024%2C576&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-37.jpeg?resize=1024%2C576&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-37.jpeg?resize=300%2C169&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-37.jpeg?resize=768%2C432&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-37.jpeg?resize=1100%2C619&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-37.jpeg?w=1280&amp;ssl=1 1280w" data-recalc-dims="1" height="576" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-37.jpeg?resize=1024%2C576&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>

  <p class="graf graf--p">
   然而，
   <a class="markup--anchor markup--p-anchor" data-href="https://grayzoneproject.com/2018/07/02/clint-watts-world-how-americas-top-russian-disinformation-expert-pushes-disinformation-to-justify-censorship/" href="https://grayzoneproject.com/2018/07/02/clint-watts-world-how-americas-top-russian-disinformation-expert-pushes-disinformation-to-justify-censorship/" rel="noopener noreferrer" target="_blank">
    在 Grayzone 撰稿人 Ilias Stathatos 的公开活动中
   </a>
   <strong class="markup--strong markup--p-strong">
    Watts 承认他的大部分证词都是错误的
   </strong>
   ，并且拒绝提供证据来支持他对俄罗斯恶意僵尸活动的一系列最丰富多彩的主张。
  </p>
  <p class="graf graf--p">
   在一次与 Buzzfeed 的单独采访中 ，Watts 似乎
   <strong class="markup--strong markup--p-strong">
    完全否认 Hamilton 68 机器人追踪器是一种合法的工具
   </strong>
   。“我不相信这个机器人的事，” Watts 说。他甚至称这个叙述是他帮助制造的“过度”概念，并
   <strong class="markup--strong markup--p-strong">
    承认“Hamilton 68”追踪的记录不一定是俄罗斯情报人员
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们根本不认为他们在被俄罗斯完全指挥。我们认为他们中的一些人是合法的热情的公民，只是他们的观点对俄罗斯有利，“ Watts 承认。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    但这些惊人的承认并没有减缓即将到来的内容审查大清洗势头。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    进入大西洋理事会
   </strong>
  </p>
  <p class="graf graf--p">
   在与 Sprague 的谈话中，德国马歇尔基金组织的 Fly 表示他正在与大西洋理事会合作，从 Facebook 等社交媒体平台清除非主流意见。
  </p>
  <p class="graf graf--p">
   大西洋理事会是华盛顿的另一个智囊团，是新保守派和干预主义者在全球范围内推动军事侵略的聚集点。它由北约和
   <a class="markup--anchor markup--p-anchor" data-href="http://www.atlanticcouncil.org/support/supporters" href="http://www.atlanticcouncil.org/support/supporters" rel="noopener noreferrer" target="_blank">
    压制性政权资助
   </a>
   ，包括沙特阿拉伯、巴林和土耳其，以及乌克兰寡头，如 Victor Pynchuk。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    大西洋理事会的 DFRLab 因其对合法在线异议与非法俄罗斯活动的严重混淆而臭名昭著
   </strong>
   ，采用与 PropOrNot 和 ASD 相同的策略。今年5月，
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/dfrlab/why-were-partnering-with-facebook-on-election-integrity-19f0ca39db2e" href="https://medium.com/dfrlab/why-were-partnering-with-facebook-on-election-integrity-19f0ca39db2e" rel="noopener noreferrer" target="_blank">
    Facebook 宣布
   </a>
   与大西洋理事会的数字取证研究实验室（DFRLab）建立合作伙伴关系。
  </p>
  <p class="graf graf--p">
   Ben Nimmo 是一名 DFRLab 研究员，他在冲刷克里姆林宫影响力网络方面建立了自己的声誉， 但今年开始了一系列令人尴尬的事件，他承认自己错误地认定了几个人，受害者包括
   <a class="markup--anchor markup--p-anchor" data-href="https://www.youtube.com/watch?v=NOUHXqGsJTs" href="https://www.youtube.com/watch?v=NOUHXqGsJTs" rel="noopener noreferrer" target="_blank">
    Mariam Susli
   </a>
   — 一位著名的叙利亚-澳大利亚社交媒体人物，还有著名的乌克兰音乐会钢琴家
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/vallisitsa/status/978559232508416000?lang=en" href="https://twitter.com/vallisitsa/status/978559232508416000?lang=en" rel="noopener noreferrer" target="_blank">
    Valentina Lisitsa
   </a>
   ，以及一位名叫 Ian Shilling 的英国养老金领取者。
  </p>
  <p class="graf graf--p">
   在接受 Sky News 的采访中，Shilling 对他的指控者进行了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.youtube.com/watch?time_continue=300&amp;v=EJogpc3Uy34" href="https://www.youtube.com/watch?time_continue=300&amp;v=EJogpc3Uy34" rel="noopener noreferrer" target="_blank">
    令人难忘的评论
   </a>
   ：“
   <strong class="markup--strong markup--p-strong">
    我没有克里姆林宫的任何联系; 我不认识任何俄罗斯人，我与俄罗斯政府没有任何联系、与他们没有任何关系，“他大声说道，“我是一名普通的英国公民，只是恰好正在研究当前在叙利亚发生的新型战争。
   </strong>
   ”
  </p>
  <p class="graf graf--p">
   随着最新的 Facebook 和 Twitter 审查大清洗，像 Shilling 这样的普通公民正在被公开瞄准，而且没有道歉。
  </p>
  <p class="graf graf--p">
   对自媒体报道的大规模删除说明了来自德国马歇尔基金会和大西洋理事会的国家安全强硬派（以及支持 PropOrNot 的人）是如何通过政治宣传帮助制造所谓的“俄罗斯“恐慌，以便
   <strong class="markup--strong markup--p-strong">
    为更广泛的媒体审查制度骗取公众支持。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    编者按
   </strong>
   ：这种现象 — — 即 利用地缘政治斗争宣传，实现对本国异议人士不分青红皂白地监视和审查行为，并不是新鲜事。对异议人士的抹黑运动也一直都在其中。尤其是当下，这种监视和审查正在“外包”给大型科技公司，这是非常值得警惕的 — IYP 已经不止一次强调过这件事，因为巨头们不透明的裁决让公民失去了监督问责的权利，它们可以轻松违宪而免于追究。
  </p>
  <p class="graf graf--p">
   如您所知，我们警惕政治宣传水军（见“
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E5%B7%A8%E9%AD%94%E5%B4%9B%E8%B5%B7-%E6%94%BF%E5%BA%9C%E8%B5%84%E5%8A%A9%E7%9A%84%E7%BD%91%E7%BB%9C%E6%B0%B4%E5%86%9B%E6%AD%A3%E9%81%8D%E5%B8%83%E5%85%A8%E7%90%83-a45710e8e431" href="https://medium.com/@iyouport/%E5%B7%A8%E9%AD%94%E5%B4%9B%E8%B5%B7-%E6%94%BF%E5%BA%9C%E8%B5%84%E5%8A%A9%E7%9A%84%E7%BD%91%E7%BB%9C%E6%B0%B4%E5%86%9B%E6%AD%A3%E9%81%8D%E5%B8%83%E5%85%A8%E7%90%83-a45710e8e431" rel="noopener noreferrer" target="_blank">
    巨魔崛起
   </a>
   ”）但更要警惕以反水军为由针对异议人士采取的审查和封锁（见下图）尤其提醒中国读者警惕，因为是权力而不是公民，掌握着对该定义的解释权。
  </p>
  <p>
   <img alt="" class="aligncenter wp-image-507 size-large jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-38.jpeg?resize=1024%2C870&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-38.jpeg?resize=1024%2C870&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-38.jpeg?resize=300%2C255&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-38.jpeg?resize=768%2C652&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-38.jpeg?w=1062&amp;ssl=1 1062w" data-recalc-dims="1" height="870" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-38.jpeg?resize=1024%2C870&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>

    <img alt="" class="aligncenter wp-image-507 size-large" data-recalc-dims="1" height="870" sizes="(max-width: 1024px) 100vw, 1024px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-38.jpeg?resize=1024%2C870&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-38.jpeg?resize=1024%2C870&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-38.jpeg?resize=300%2C255&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-38.jpeg?resize=768%2C652&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-38.jpeg?w=1062&amp;ssl=1 1062w" width="1024"/>

  <p class="graf graf--p">
   对于中国读者来说，应该理解到，所有这些监视资本主义巨头 — 不管是百度、腾讯、谷歌还是 Facebook 和 Twitter，都不会实现真正的言论自由，言论自由并不是曾经不能在微信说的话现在可以在 Twitter 说，不是对任何一种霸权的迎合，而是，任何时候你都可以无阻碍地表达自己的观点。是*自己的*。
