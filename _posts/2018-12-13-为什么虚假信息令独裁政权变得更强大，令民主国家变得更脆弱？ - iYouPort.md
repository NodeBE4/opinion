  <p class="graf graf--p graf-after--figure" id="8f59">
   一些中国的高级知识分子告诉我，他们喜欢假消息。这听起来似乎很惊人。但我被告知，假消息在中国是“一种反抗精神”，是对当局操控的“辟谣”的反叛。我能了解这其中所描述的现象，因为它在任何国家都不是新闻；但我无法理解这一逻辑，它直接玷污了反抗者的概念。
  </p>
  <p class="graf graf--p graf-after--p" id="4b64">
   很多主流媒体都会告诉你，是特朗普所表达的对记者的抨击和仇恨导致了新闻自由度暴跌；不过如果你了解民粹主义政客的社交媒体运作方式就应该能明白，这些标志性言论建立在大众的既有认知之上 — 它并非凭空编造。
  </p>
  <p class="graf graf--p graf-after--p" id="4e81">
   美国公众对主流媒体的厌恶已经由来已久，IYP 曾经在这篇文章中《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E5%AA%92%E4%BD%93-%E4%BD%A0%E7%9C%9F%E7%9A%84%E7%9F%A5%E9%81%93%E4%BB%80%E4%B9%88%E6%98%AF%E4%BA%8B%E5%AE%9E%E5%90%97-f86286d2b040" href="https://medium.com/@iyouport/%E5%AA%92%E4%BD%93-%E4%BD%A0%E7%9C%9F%E7%9A%84%E7%9F%A5%E9%81%93%E4%BB%80%E4%B9%88%E6%98%AF%E4%BA%8B%E5%AE%9E%E5%90%97-f86286d2b040" rel="noopener noreferrer" target="_blank">
    媒体，你真的知道什么是事实吗？
   </a>
   》部分解释了这一背景，特朗普自己如何认为媒体是次要的，重要的是他知道能够通过抨击媒体赢得支持者的再一次兴奋。
  </p>
  <p class="graf graf--p graf-after--p" id="63c0">
   特朗普在此没有任何原创，他只是制造了一个新词“假新闻”，而政客对自己不喜欢的一切消息都称之为“假”这种做法有着悠久的历史。“Shooting the Messenger” 就是这样一个隐喻，用来形容类似中文中所说的“睡不着觉埋冤枕头”。在现代电信出现之前，消息通常由人类信使传递。例如在战争中，一个信使将消息从一个营地带到另一个营地。如果消息引起了厌倦和惶恐，接收信息的一方就可能会将这种坏消息归咎于信使，并且把愤怒施加在信使身上。
  </p>
  <p class="graf graf--p graf-after--p" id="c52c">
   中国也是这样做的。当局的“谣言”论只是在驳斥那些让当权者感觉不舒服的消息。事实上
   <a class="markup--anchor markup--p-anchor" data-href="https://steemit.com/media/@iyouport/4yzpnz" href="https://steemit.com/media/@iyouport/4yzpnz" rel="noopener nofollow noreferrer" target="_blank">
    全球新闻自由指数连续数年持续滑坡
   </a>
   在很大程度上由于各国的当权者更加热衷于“Shooting the Messenger” 了。
  </p>
  <p class="graf graf--p graf-after--p" id="b422">
   但本文希望讲述的并不只有这种被政客武器化了的“假消息”，更有真正的假消息，即 部分中国高级知识分子和公民群体将其期待为“反抗行为艺术”的东西。前后两者看起来是有区别的，但是在本质层面上它们完全一致。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="06e0">
   “虚假消息”通常携带着强烈的仇恨和激发严重暴力的引导，它们已经在很多国家造成了真实的杀戮和迫害、操纵了选民的认知、分裂了社会、让专制政客得利其中，民主摇摇欲坠。相比下，中国社会弥漫的往往是基于流量经济和内容营销而制造的假消息，则直接起到了阻碍民主变革的作用。
  </p>
  <p class="graf graf--p graf-after--p" id="cfb5">
   <strong class="markup--strong markup--p-strong">
    撒谎的权力
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="faff">
   专制权力建立在谎言之上，谎言 — 或者准确说是操纵现实的能力 — 让权力得以实现并不断被稳固和加强。只要你能让人们信任你所描述的东西，人们就会拥护你赋予你权力、满足你想要的目的。
  </p>
  <p class="graf graf--p graf-after--p" id="c360">
   虚假信息宣传活动体现了美国社会的分歧 — — 选民欺诈、疫苗阴谋以及关于移民问题、乔治索罗斯和 Black Lives Matter 的种族主义阴谋，这只是仅举几例而已 — — 而这些鲜明的分歧恰恰是俄罗斯等独裁政体的力量源泉，
   <strong class="markup--strong markup--p-strong">
    正是由于人们对哪些是事实、哪些是被政权制造出来的“事实”之判断缺乏共识，从而加强了普京及其寡头的权力
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="2711">
   英国的脱欧公投是异曲同工的被虚假信息宣传活动推动的结果。两国的民粹主义政客不仅从对移民的妖魔化中获益 — 特别是来自叙利亚的移民 — 而且还获益于微妙的反犹主义和赤裸的对伊斯兰教的仇视。两者都导致了种族主义、不容忍和针对少数民族和有色人种的仇恨犯罪“严重增加”。有利于英国脱欧和特朗普胜选的投票与经济焦虑或贫困的关系并不大（是的主流媒体的叙述有误），更多的是与文化焦虑和种族怨恨有关。任何能够在最大程度上调用这种社会情绪的人都能实现操纵民意和稳固权力。
  </p>
  <p class="graf graf--p graf-after--p" id="2b5b">
   冷战式的国家安全是围绕冷战思想的，如进攻 — 防御平衡，传统威慑理论和对威慑的抵制，而这些都不足以思考关于虚假信息的攻击问题; 与此同时，信息安全被局限于关于“服务器和个人网络”的思考层面，而不是“攻击更广泛的民主社会结构的后果“。显然这是不充分的。
  </p>

  <p class="graf graf--p graf-after--figure" id="eb58">
   <strong class="markup--strong markup--p-strong">
    为什么专制国家能通过撒谎获得稳定？而民主国家通过撒谎获得的是混乱和危机？
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="4a7f">
   政治学家 Henry Farrell 在
   <a class="markup--anchor markup--p-anchor" data-href="https://papers.ssrn.com/sol3/papers.cfm?abstract_id=3273111" href="https://papers.ssrn.com/sol3/papers.cfm?abstract_id=3273111" rel="noopener nofollow noreferrer" target="_blank">
    一篇新论文
   </a>
   中提出了一个很好的问题：他指出应该通过信息安全的视角来看待民主，以便了解当前互联网虚假信息攻击的浪潮。具体来说，需要能解释为什么在俄罗斯发挥稳定作用的同样的虚假宣传活动在美国造成的却是不稳定？
  </p>
  <p class="graf graf--p graf-after--p" id="461a">
   答案将围绕着独裁政体和民主国家作为信息系统运作的不同方式。
  </p>
  <p class="graf graf--p graf-after--p" id="8cf4">
   我们首先要区分两种不同社会在其政治制度中对共同政治认识的使用，这是指人们在一个社会中普遍认同的信息体。人们同意谁作为统治者以及他们对合法性的主张是什么。人们普遍同意他们的政府如何运作，即使其中有些人不喜欢这种运作方式。在一个民主国家里是指，人们同意选举如何运作：如何创建和界定选区，如何选择候选人，以及他们的投票数量 — 即使只是粗略的和不完美的。
  </p>
  <p class="graf graf--p graf-after--p" id="d511">
   这似乎是很基本的东西，但当我们在独裁和民主国家中对比这两种形式的认识时，它就会变得很有趣了。
   <strong class="markup--strong markup--p-strong">
    这两种不同形式的政府对共同的和有争议的政治认识有着完全相反的需求
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="9fcf">
   例如民主国家，利用其人民中存在的分歧来解决问题。不同的政治团体对如何治理有着不同的看法，这些团体通过说服他们的选民以争夺自己的政治影响力。关于谁将负责并能够制定政策目标，也存在长期的不确定性。
   <strong class="markup--strong markup--p-strong">
    理想情况下，这是一种机制，通过这种机制，政体可以利用其成员的多样性观点来更好地解决复杂的政策问题
   </strong>
   。如果没有人知道下次选举后将换谁来负责，不同的政党和候选人就会争先恐后地去说服选民不同政策提案的好处。
  </p>
  <p class="graf graf--p graf-after--p" id="1103">
   但是要知道，
   <strong class="markup--strong markup--p-strong">
    为了实现这一目标，就需要人民对政府职能以及如何选择政治领导人有共同的认识
   </strong>
   。还需要了解政治参与者是谁，他们和他们的政党代表什么，以及他们如何相互冲突。此外，这种认识在各种各样的行为者中分散化 — 这是一个基本要素，因为普通公民在政治决策中发挥着重要作用。
  </p>
  <p class="graf graf--p graf-after--p" id="1845">
   那么将民主与专制对照一下。在专制国家，关于谁负责长期统治以及他们的政策目标是什么的共同政治认识，是稳定的基本条件。
   <strong class="markup--strong markup--p-strong">
    独裁政权不需要关于选举效力和公平性的共同政治认识，并且要努力保持对其他任何形式的共同政治认识的垄断
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="7722">
   政权需要积极地压制社会中各种潜在群体之间形成某种共同政治认识、他们的民众支持水平、以及他们如何形成彼此的联盟。这不需要过多的解释，你能在中国看到当权者如何长期以来受益于原子化的社会和高度的互信危机（封闭的和
   <a class="markup--anchor markup--p-anchor" data-href="https://pao-pao.net/article/773" href="https://pao-pao.net/article/773" rel="noopener nofollow noreferrer" target="_blank">
    隧道化的小圈子
   </a>
   ，以彼此的轻蔑来巩固自身的凝聚力；谣传弥漫、群众斗群众，你
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E4%B8%BA%E4%BB%80%E4%B9%88%E6%88%91%E4%BB%AC%E4%BA%92%E4%B8%8D%E4%BF%A1%E4%BB%BB-51432855663e" href="https://medium.com/@iyouport/%E4%B8%BA%E4%BB%80%E4%B9%88%E6%88%91%E4%BB%AC%E4%BA%92%E4%B8%8D%E4%BF%A1%E4%BB%BB-51432855663e" rel="noopener noreferrer" target="_blank">
    很难找到真正的同盟
   </a>
   ）。
  </p>
  <p class="graf graf--p graf-after--p" id="ce02">
   另一方面，
   <span class="markup--quote markup--p-quote is-other" data-creator-ids="e5835fc6e3b2">
    专制政权受益于各种非政府组织和社会行为者之间有争议的政治认识
   </span>
   。因为如果没有人真正知道还可能形成哪些其他政党、他们可能代表什么、以及他们可能得到什么样的支持，对那些反对执政党的组织和潜在有可能形成的党派来说这本身就是一个重大的障碍。
  </p>
  <p class="graf graf--p graf-after--p" id="913c">
   这就是为什么 IYP 从来无视异议群体内部的观点分歧。意见不同、思考方式不同是非常正常的，完全不会影响结盟，
   <strong class="markup--strong markup--p-strong">
    只要所有人的目标是一致的
   </strong>
   。然而也许是
   <span class="markup--quote markup--p-quote is-other" data-creator-ids="e5835fc6e3b2">
    “赶上了”
    <a class="markup--anchor markup--p-anchor" data-href="https://pao-pao.net/article/824" href="https://pao-pao.net/article/824" rel="noopener nofollow noreferrer" target="_blank">
     超个人主义时代
    </a>
    ，自我表现变成了
    <a class="markup--anchor markup--p-anchor" data-href="https://pao-pao.net/article/919" href="https://pao-pao.net/article/919" rel="noopener nofollow noreferrer" target="_blank">
     绝大多数人的最高追求
    </a>
    ，而不是通过讨论以获得共识。如果您还能记得关于“暴力/非暴力”等月经之辩，现在就应该可以理解到，一直以来究竟是谁在从这些无意义的口水战中获利。
   </span>
  </p>
  <p>
   <img alt="" class="alignleft wp-image-5602 jetpack-lazy-image" data-lazy-sizes="(max-width: 400px) 100vw, 400px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-269.jpeg?resize=400%2C266&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-269.jpeg?w=777&amp;ssl=1 777w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-269.jpeg?resize=300%2C200&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-269.jpeg?resize=768%2C511&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-269.jpeg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="266" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-269.jpeg?resize=400%2C266&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="400"/>
   <noscript>
    <img alt="" class="alignleft wp-image-5602" data-recalc-dims="1" height="266" sizes="(max-width: 400px) 100vw, 400px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-269.jpeg?resize=400%2C266&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-269.jpeg?w=777&amp;ssl=1 777w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-269.jpeg?resize=300%2C200&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-269.jpeg?resize=768%2C511&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-269.jpeg?resize=272%2C182&amp;ssl=1 272w" width="400"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="6761">
   如上，两种政体之间的这种差异对安全性有着重要的影响。专制政权将容易受到这样的信息攻击 — — 即 这些信息攻击能挑战政权对共同政治认识的垄断；他们容易受到外部信息的攻击 — — 这些信息可以表明政府正在为自己的利益操纵共同的政治认识；
   <strong class="markup--strong markup--p-strong">
    这些攻击可以将有争议的政治认识转化为共同的政治认识
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="6313">
   因此，
   <strong class="markup--strong markup--p-strong">
    专制政权最害怕的是人民的联盟、很容易受到允许人们更容易沟通和形成组织的工具的影响，以及那些为公民提供外部信息和观点的工具
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="730c">
   简而言之，能冲击专制政权的是真相、是社会互信、是社团联合，是信息自由，而不是表达另一种立场的虚假消息如宫廷八卦。这在任何一个专制国家都一样。
  </p>
  <p class="graf graf--p graf-after--p" id="8439">
   在阿拉伯之春的第一次爆发之前，突尼斯政府就对共同认识有了广泛的控制。它要求每个人公开支持该政权，使公民们很难知道究竟有多少人讨厌该政权，并阻止任何潜在的反政权联盟组织的发起。然而，它没有及时关注 Facebook，这让公民更容易谈论他们有多么厌恶统治者，并且在最初的事件引发抗议时，人们迅速组织起了反对政权的群众示威活动（抗议者们由于很快手机和在线系统就被禁用，从而逃脱了数字监控）。而像俄罗斯这样的国家将信息自由议程视为他们喉咙里的刀片、包括澳大利亚在内的很多国家将加密技术视为威胁，其原因是显而易见的。
  </p>
  <p class="graf graf--p graf-after--p" id="86e1">
   如上，这种差异解释了同一措施如何能够增加一种政权形式的稳定性、并降低另一种政体的稳定性。
   <strong class="markup--strong markup--p-strong">
    同时我们也已经看到，开放自由的信息流动能使民主国家受益，同时威胁到独裁政权
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="668e">
   <strong class="markup--strong markup--p-strong">
    总之
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="2313">
   民主国家依赖于人们对其他人观点的良好了解，最明显的是通过自由和公平的选举来体现，公民可以了解他们邻居的观点，因此有动力找到他们知道的广泛有效的解决方案。
  </p>
  <p class="graf graf--p graf-after--p" id="56e1">
   而专制权力的稳定需要公众互相之间不知道其他人的感受 — — 需要人们对哪些机构、团体和观点是真实的以及哪些是阴谋、欺诈或权力争夺战，
   <strong class="markup--strong markup--p-strong">
    保持持续的混淆
   </strong>
   。需要人们的互不信任，需要人们整天思考的是哪个家伙是当局的线人、谁在钓鱼、谁在利用大众的思维缺陷实现他们自己的名利欲……简单说，“我正在被谁利用”，而不是“我能和谁结盟”。
  </p>
  <p class="graf graf--p graf-after--p" id="c142">
   而当部分异议人士以虚假信息支撑自己的政治立场时 — — 哪怕这一立场绝对正义 — — 其所造成的结果只有社会进一步地失去互信。造假太拙略了，人们很快就会发现自己拿不到任何确证，进而陷入失望和愤怒。
   <strong class="markup--strong markup--p-strong">
    当你听到越来越多的人们开始讽刺“民逗”的时候你应该感觉悲哀，应该还能记得民主派人士和群体曾经是聚拢社会共同认识的新希望
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="71ee">
   反过来，一旦公众发现并确定周围有多少人对专制政权的抵制是存在共识的、对恶行的揭露是握有证据的，他们就会大胆地站起来反对。民间独立团体和社群应该能够给民众以“找到组织了”的感觉，这是不需要你分发红包以换取的“点赞和转发”。支持和协作建立在信任之上。
  </p>
  <p class="graf graf--p graf-after--p" id="37cd">
   事实上这一切应该早已是常识，绝大部分人都知道专制政权如何视“机密”为生命一般、如何暗箱操作并保持神秘感，专制社会的人们是如何的自我封闭、如何与全球反对力量脱节，以及多么的难以形成本地联盟。
  </p>
  <p class="graf graf--p graf-after--p" id="47d1">
   <strong class="markup--strong markup--p-strong">
    信息自由是改变这一切的基础
   </strong>
   。人们应该能够无阻碍地访问互联网，应该能够绕避一切审查获得表达的自由，应该能够对真相保有坚定的热情，这就是为什么
   <a class="markup--anchor markup--p-anchor" data-href="https://steemit.com/advocacy/@iyouport/is-democracy-in-its-death-throes-what-should-we-do" href="https://steemit.com/advocacy/@iyouport/is-democracy-in-its-death-throes-what-should-we-do" rel="noopener nofollow noreferrer" target="_blank">
    “Open China”
   </a>
   要建立于这样一个理念之上，
   <strong class="markup--strong markup--p-strong">
    打破封闭、保护隐私、去中性化、OSINT 并展示匿名协作的强大无国界能力
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="b5a2">
   本地联盟已经不够，互联网最大的效用是全球被压迫者的联盟。当你坚信“中国特殊论”时你是孤单的，当你知道还有其他很多国家和中国有着同样的甚至更糟糕的恶劣环境、但那些国家的人民一直在顽强地抵抗时，你将变得比以往更加自信和强大。⚪️
  </p>
