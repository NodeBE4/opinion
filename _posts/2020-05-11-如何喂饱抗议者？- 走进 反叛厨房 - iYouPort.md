  <p class="graf graf--p">
   运行在抗议营地中间的厨房带来了一些不寻常的操作挑战。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “目前我们正在现场烹饪大部分热食 …… 但警察不断跑来收拾我们的东西”，George Coiley 说，他带领我经过沸腾的炉灶水壶、锅具和两名志愿者准备的史诗般的水果沙拉。
  </p>
  <p class="graf graf--p">
   这是 Coiley 经营的第四个 “反叛厨房”，它是大约30名志愿者组成的轮值小组，
   <strong class="markup--strong markup--p-strong">
    每天24小时为抗议者和其他需要的人提供食物和热饮
   </strong>
   。
  </p>
  <p class="graf graf--p">
   所有食物都是纯素食主义风格的，都是通过捐赠筹集的。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “这本身就是一种政治行动，”
   </strong>
   Coiley 说，他是个聪明的年轻人，他正在攻读可持续食品专业博士学位。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这里有些人不认为自己是维权人士，只是觉得自己走进这个厨房感觉很舒适。听起来很基本，但我认为这确实令人鼓舞。”
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11216 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-3.jpg?resize=1100%2C619&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-3.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-3.jpg?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-3.jpg?resize=1024%2C576&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-3.jpg?resize=768%2C432&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-3.jpg?resize=1100%2C619&amp;ssl=1 1100w" data-recalc-dims="1" height="619" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-3.jpg?resize=1100%2C619&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   与他们试图解决的巨大的生存问题相比，如何喂饱一群抗议者，或者甚至是成千上万的激进主义者，似乎显得微不足道。但是它们并非无关紧要。
  </p>
  <p class="graf graf--p">
   我们的饮食与气候危机密不可分：人类排放的大约30％与从生产到消费的食物有关。
   <strong class="markup--strong markup--p-strong">
    我们的饮食选择本身就是政治性的。
   </strong>
  </p>
  <p class="graf graf--p">
   愤世嫉俗的人经常以食物为主题对反叛厨房活动家吹毛求疵。愚蠢的挑剔可能会突显出有关个人责任重要性的关键问题，即 您想要在世界上看到的变化与全局结构变化之间的关系。
  </p>
  <p class="graf graf--p">
   Coiley 不论是在学术工作中还是在特拉法加广场（Trafalgar Square）的 “反叛厨房” 做饭期间都在考虑这一点。
  </p>
  <p class="graf graf--p">
   他说：“我们正在努力帮助人们看到一个不同的世界，在这里，我们所有人都可以食用营养丰富且可持续的食物，这种食物既无害，也可以获得享受。”
  </p>
  <p class="graf graf--p">
   Coiley 认为良好的烹饪可以表明 “我们不需要牛排就能度过美好的时光。”
  </p>
  <p class="graf graf--p">
   反叛厨房在四个凉亭中，这些凉亭有定制的篷布和许多胶带。挂在外面的白板上写着愿望清单，上面列出了橘子、香蕉、咸味小吃、鹰嘴豆泥和自制素食蛋糕及零食。
  </p>
  <p class="graf graf--p">
   人们通过手机拍照，然后捐赠。大箱子专用于从调味料到干果、豆类到蔬菜、以及各种纯素食和非纯素食饼干的形形色色的产品。
  </p>
  <p class="graf graf--p">
   在帐篷的服务端，小扁豆和蔬菜咖喱装在20升的保温瓶中，还有很多面包和沙拉用的平底锅。后面还有一个由志愿者经营的洗手台。这里禁止一次性餐具以避免白色污染。
  </p>

  <p class="graf graf--p">
   当我们走出厨房进入特拉法加广场时，那里的场面完全不像抗议，而是像某种庆典。色彩鲜艳的衣服、旗帜和传单照亮了灰色人行道和灰色天空之间的一切。人群散发出正义的愤怒和热情。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《2、
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/2%e3%80%81%e5%88%86%e5%b8%83%e5%bc%8f%e8%a1%8c%e5%8a%a8%e7%9a%84%e8%a1%97%e5%a4%b4%e6%b4%be%e5%af%b9/" href="https://www.iyouport.org/2%e3%80%81%e5%88%86%e5%b8%83%e5%bc%8f%e8%a1%8c%e5%8a%a8%e7%9a%84%e8%a1%97%e5%a4%b4%e6%b4%be%e5%af%b9/" rel="noopener noreferrer" target="_blank">
     分布式行动的街头派对
    </a>
    》
   </li>

  <p class="graf graf--p">
   营地包括由反叛活动家占领的一百多个帐篷，以及专用于分享有关气候变化、维权人士福祉、和被捕者支持信息的大型帐篷。
  </p>
  <p class="graf graf--p">
   根据最新统计，这场抗议活动中在伦敦已有1300多人被捕。
  </p>
  <p class="graf graf--p">
   这里有一个书店帐篷、还有一个艺术帐篷，您可以在这里把 Extinction Rebellion 的标志印在衣服上，一个手鼓，一群草裙舞、还有一群人在唱歌，向人群传递着激昂的声音。当然，还有一个十英尺高的粉红色章鱼。
  </p>
  <p class="graf graf--p">
   在特拉法加广场营地之外，Extinction Rebellion 组织了整个伦敦的一系列自发性抗议行动，通常涉及示威者将自己绑在门、建筑物甚至飞机上。
  </p>
  <p class="graf graf--p">
   反叛厨房负责向这些抗议者提供食品，包括谷物棒、苹果、年糕、饼干、花生酱、干果和面包卷。骑自行车的志愿者跑来跑去，运送热食和热茶，在这个经常下雨的城市里很受欢迎。
  </p>
  <p class="graf graf--p">
   就在米尔班克河对岸，离军情五处大楼不远的地方，几百名活动家封锁了街道以示抗议。
  </p>
  <p class="graf graf--p">
   一名活动家 Miranda 是戴着墨镜的中年妇女，站在马路边吃着椰枣，看着她的同伴爬上一辆警车。
  </p>
  <p class="graf graf--p">
   她对在示威期间提供食物的经验非常积极。她说：“我是素食主义者，而我一生中从未吃过如此多的素食。这些食物太棒了 … 我体重增加了。”
  </p>
  <p class="graf graf--p">
   另一名抗议者解释说，一些激进活动家通过完全不吃东西来抗议。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <strong class="markup--strong markup--p-strong">
    今天有些人开始绝食，但现在他们称之为空腹行动
   </strong>
   ，” Emma 说，她正在寻找其他抗议者，他们被困在兰贝斯桥北侧。7月，38岁的医生 Cliff Kendall 完成了为期15天的绝食抗议，以支持这项运动。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11218 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-3.jpg?resize=1100%2C619&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-3.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-3.jpg?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-3.jpg?resize=1024%2C576&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-3.jpg?resize=768%2C432&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-3.jpg?resize=1100%2C619&amp;ssl=1 1100w" data-recalc-dims="1" height="619" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-3.jpg?resize=1100%2C619&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11218" data-recalc-dims="1" height="619" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-3.jpg?resize=1100%2C619&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-3.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-3.jpg?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-3.jpg?resize=1024%2C576&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-3.jpg?resize=768%2C432&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-3.jpg?resize=1100%2C619&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    抗议期间提供充足的食物也可以鼓舞士气
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “在这里你能感受到很好的照顾，” 与一群音乐家站在一起的另一名抗议者 Leonie 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “我们在这里待了两个星期，人们把自己的整个年假都放在抗议现场了。但是大家需要食物，热食，否则人们会变得脾气暴躁。我们很多人不想在这里，但我们更害怕不在这里
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   标准激进主义者的燃料似乎严重依赖便携的谷物棒，但是持续时间更长的抗议部署又该如何呢？
  </p>
  <p class="graf graf--p">
   就如绿色和平组织那样的抗议活动。从攀登伦敦碎片大厦到占领石油钻塔，绿色和平组织以其大胆的直接行动任务而闻名；
   <strong class="markup--strong markup--p-strong">
    该慈善机构的新闻官 Graham Thompson 曾经在西伯顿（West Burton）发电站的烟囱上驻扎了一个星期。
   </strong>
  </p>
  <p class="graf graf--p">
   Thompson 说，在这类行动中，自加热式军用配给包提供了一种高科技（且营养丰富）的解决方案。绿色和平组织始终使用纯素食食品：典型的包装包括蔬菜咖喱、蔬菜辣椒或蔬菜面食。
  </p>
  <p class="graf graf--p">
   Thompson 笑着说：“我想说的是，这些食物的味道比你想象的要差很多。”他补充说，
   <strong class="markup--strong markup--p-strong">
    你可以携带多少水经常就是维权人士可以坚持多久的限制因素。
   </strong>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11219 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-4.jpg?resize=1100%2C619&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-4.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-4.jpg?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-4.jpg?resize=1024%2C576&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-4.jpg?resize=768%2C432&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-4.jpg?resize=1100%2C619&amp;ssl=1 1100w" data-recalc-dims="1" height="619" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-4.jpg?resize=1100%2C619&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11219" data-recalc-dims="1" height="619" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-4.jpg?resize=1100%2C619&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-4.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-4.jpg?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-4.jpg?resize=1024%2C576&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-4.jpg?resize=768%2C432&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-4.jpg?resize=1100%2C619&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   绿色和平组织的成员完全是素食主义者，大部分（如果不是全部的话）是绝对素食主义者，向工作人员提供补给的主要是素食主义者午餐，但是也有一些肉食者在那里工作。
  </p>
  <p class="graf graf--p">
   Thompson 说：“人们一直在说减少个人碳足迹是不可思议的事，但是这不是他们那些解决方案可以解决的问题。您需要最高级别的改变。”
  </p>
  <p class="graf graf--p">
   查塔姆研究所主任 Tim Benton 同意这一立场，他也是今年政府间气候变化专门委员会关于食物、土地和气候的报告的作者。他说，“我们确实必须改变饮食习惯，因为 Extinction Rebellion 组织的鼓声已经在后台响起”，我们都是其他人或过去的政策制定者设计的系统中的参与者。
  </p>
  <p class="graf graf--p">
   无论是激进主义的燃料、象征着生活方式的选择、出于宗教或政治目的、还是仅仅出于个人品味，食物都绝不仅仅是我们所吃的东西。并且，无论您关注 Extinction Rebellion 组织的动态快讯，还是着眼于喂饱其活动家的系统，该运动都为所有人提供了很多的思考机会。⚪️
  </p>

  </p>
