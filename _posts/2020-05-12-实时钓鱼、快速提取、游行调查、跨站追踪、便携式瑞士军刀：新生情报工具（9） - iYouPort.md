  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、实时钓鱼数据库
   </strong>
  </p>
  <p class="graf graf--p">
   这是一个实时网络钓鱼数据库，该数据库从多个来源收集网络钓鱼URL。
  </p>
  <p class="graf graf--p">
   Telegram 和 Twitter 上都有一个机器人，机器人会发布最近几个小时添加到数据库中的最新网络钓鱼URL。
  </p>
  <p class="graf graf--p">
   它们提供的API才是真正的强大功能，您可以在其中搜索特定的文本、ASN、IP地址、国家等。
  </p>
  <p class="graf graf--p">
   您可以直接在浏览器中运行它，并以JSON格式显示所需的数据。
  </p>
  <p class="graf graf--p">
   该数据库中有超过300万条记录，并且一直在不断增长。
   <strong class="markup--strong markup--p-strong">
    在政府间谍软件泛滥的时代，这类研究很有价值
   </strong>
   （无交互攻击依旧过于昂贵因此相对罕见，而大多数攻击都是交互性的）。
  </p>
  <p class="graf graf--p">
   Link:
   <a class="markup--anchor markup--p-anchor" data-href="https://phishstats.info/" href="https://phishstats.info/" rel="nofollow noopener noreferrer" target="_blank">
    https://phishstats.info/
   </a>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、Instagram 开源情报工具
   </strong>
  </p>
  <p class="graf graf--p">
   对于 Instagram、抖音这类平台进行的调查往往会更加容易（相比Twitter的话），因为在这些平台上的人大多倾向于暴露自己真实的一面，不论是脸还是行动、简介还是位置。
  </p>
  <p class="graf graf--p">
   一个简单的工具，用于对 Instagram 帐户进行一些基本的检查。
  </p>
  <p class="graf graf--p">
   该脚本易于运行，并返回您可以从 Instagram 获取的所有基本信息，例如用户名、个人资料名称、关注者、帖子、个人简历等。
  </p>
  <p class="graf graf--p">
   它还可以检索有关其是否为注册企业帐户、与 Facebook 的连接、以及其他基本信息。它不仅在屏幕上显示信息，而且还以JSON格式将数据以及头像保存在单独的目录中给你。
  </p>
  <p class="graf graf--p">
   必须使用python3.6或更高版本。
  </p>
  <p class="graf graf--p">
   Link:
   <a class="markup--anchor markup--p-anchor" data-href="https://github.com/sc1341/InstagramOSINT" href="https://github.com/sc1341/InstagramOSINT" rel="nofollow noopener noreferrer" target="_blank">
    https://github.com/sc1341/InstagramOSINT
   </a>
  </p>

  <p class="graf graf--p">
   我们不断强调过基本的安全须知：您不能在多个平台上使用相同的用户名，这是很危险的，追踪者很容易据此对您进行全网捕捉。
  </p>
  <p class="graf graf--p">
   这就是跨平台追踪的方式，实际上有很多快捷的工具可以支持这点。在下面看到介绍和演示案例：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e6%9e%84%e5%bb%ba%e8%87%aa%e5%ae%9a%e4%b9%89%e7%9a%84%e7%94%a8%e6%88%b7%e5%90%8d%e4%be%a6%e6%9f%a5%e5%b7%a5%e5%85%b7%ef%bc%9a%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e6%8a%80%e6%9c%af/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e6%9e%84%e5%bb%ba%e8%87%aa%e5%ae%9a%e4%b9%89%e7%9a%84%e7%94%a8%e6%88%b7%e5%90%8d%e4%be%a6%e6%9f%a5%e5%b7%a5%e5%85%b7%ef%bc%9a%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e6%8a%80%e6%9c%af/" rel="noopener noreferrer" target="_blank">
     如何构建自定义的用户名侦查工具：开源调查技术
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%b0%bd%e5%8f%af%e8%83%bd%e5%a4%9a%e5%9c%b0%e4%ba%86%e8%a7%a3%e4%bd%a0%e7%9a%84%e4%be%a6%e6%9f%a5%e7%9b%ae%e6%a0%87%ef%bc%8c%e5%b9%b6%e4%b8%8d%e8%ae%a9%e5%af%b9%e6%96%b9%e5%8f%91%e8%a7%89%e7%9a%84/" href="https://www.iyouport.org/%e5%b0%bd%e5%8f%af%e8%83%bd%e5%a4%9a%e5%9c%b0%e4%ba%86%e8%a7%a3%e4%bd%a0%e7%9a%84%e4%be%a6%e6%9f%a5%e7%9b%ae%e6%a0%87%ef%bc%8c%e5%b9%b6%e4%b8%8d%e8%ae%a9%e5%af%b9%e6%96%b9%e5%8f%91%e8%a7%89%e7%9a%84/" rel="noopener noreferrer" target="_blank">
     尽可能多地了解你的侦查目标，并不让对方发觉的方法
    </a>
    》
   </li>
   <li class="graf graf--li">
    演示《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%bf%bd%e8%b8%aa%e6%94%bf%e5%ba%9c%e5%ae%a3%e4%bc%a0%e6%88%98%e7%9a%84%e5%86%85%e5%b9%95%ef%bc%9a%e8%b7%a8%e5%b9%b3%e5%8f%b0%e7%bd%91%e7%bb%9c%e5%88%86%e6%9e%90%e7%9a%84%e6%95%b0%e5%ad%97%e5%8f%96/" href="https://www.iyouport.org/%e8%bf%bd%e8%b8%aa%e6%94%bf%e5%ba%9c%e5%ae%a3%e4%bc%a0%e6%88%98%e7%9a%84%e5%86%85%e5%b9%95%ef%bc%9a%e8%b7%a8%e5%b9%b3%e5%8f%b0%e7%bd%91%e7%bb%9c%e5%88%86%e6%9e%90%e7%9a%84%e6%95%b0%e5%ad%97%e5%8f%96/" rel="noopener noreferrer" target="_blank">

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    请注意，跨站追踪并非仅仅是恶意者使用的，对于公民记者来说也是非常重要的工具
   </strong>
   —— 您需要在深入调查之前对目标有一个充分的了解。这些工具可以帮助您快速做到这点。
  </p>
  <p class="graf graf--p">
   今天要介绍的工具是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.namecheckr.com/" href="https://www.namecheckr.com/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     namecheckr
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.namecheckr.com/" href="https://www.namecheckr.com/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Namecheckr
    </strong>
   </a>
   是一个可以简单地在许多网站上检查用户名的服务。
  </p>
  <p class="graf graf--p">
   你可以尝试使用目标人拥有的所有各种电子邮件和用户名进行查找。
  </p>
  <p class="graf graf--p">
   互联网鼓励的是个人营销，于是大多数人倾向于跨站使用相同的名字，以积累个人影响力和声誉；即便不是如此，也经常出现跨站发布相同内容的情况。于是这些工具往往会比较好用。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    7、便携式瑞士军刀
   </strong>
  </p>
  <p class="graf graf--p">
   POCKINT（又名 Pocket Intelligence）是 DFIR / OSINT 专业人员的瑞士军刀。它被设计为轻巧便携的GUI程序（可以放在USB或VM中），以紧凑的形式为用户提供基本的开源情报调查功能。
  </p>
  <p class="graf graf--p">
   您可以从
   <a class="markup--anchor markup--p-anchor" data-href="https://github.com/netevert/pockint/releases" href="https://github.com/netevert/pockint/releases" rel="noopener noreferrer" target="_blank">
    发布页面
   </a>
   获取最新版本。
  </p>
  <p class="graf graf--p">
   POCKINT 作为单个可执行文件提供，可以在计算机上的任何位置存储和运行。POCKINT 可用于 Windows 和 Linux 平台。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    简单
   </strong>
   ：实际上有很多很棒的开源情报工具。但问题是它们要么要求分析人员一定程度上熟悉命令行操作（例如 pOSINT），要么给您太多的功能（例如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b%e5%b7%a5%e5%85%b7%e7%94%a8%e4%ba%8e%e5%bc%80%e6%ba%90%e6%83%85%e6%8a%a5%e8%b0%83%e6%9f%a5%ef%bc%9a0%e5%9f%ba%e7%a1%80%e5%88%9d%e5%ad%a6%e8%80%85%e6%8c%87%e5%8d%97/" href="https://www.iyouport.org/%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b%e5%b7%a5%e5%85%b7%e7%94%a8%e4%ba%8e%e5%bc%80%e6%ba%90%e6%83%85%e6%8a%a5%e8%b0%83%e6%9f%a5%ef%bc%9a0%e5%9f%ba%e7%a1%80%e5%88%9d%e5%ad%a6%e8%80%85%e6%8c%87%e5%8d%97/" rel="noopener noreferrer" target="_blank">
    Maltego
   </a>
   ）。而 POCKINT 则注重简单性，它是通过简单的界面快速轻松获得结果的理想工具。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    便携
   </strong>
   ：大多数工具都需要安装，许可证或配置。而 POCKINT 可以随时随地出发。将其放入您的USB中、VM里、或笔记本电脑中，就可以立即运行。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    强大
   </strong>
   ：POCKINT 将廉价的开源情报源与专用API的功能结合在一起。从一开始您就可以使用一组内置转换。添加多个API密钥，您就可以解锁更多专门的数据挖掘功能。
  </p>

    https://github.com/netevert/pockint
