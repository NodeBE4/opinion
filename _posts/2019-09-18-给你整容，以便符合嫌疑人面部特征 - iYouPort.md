   </noscript>
  </p>
  <p class="graf graf--p">
   纽约警察局滥用其面部识别系统，编辑嫌疑人的照片，以确定犯罪嫌疑人，这是研究人员在一份关于执法部门使用监控技术的调查
   <a class="markup--anchor markup--p-anchor" data-href="https://www.flawedfacedata.com/" href="https://www.flawedfacedata.com/" rel="noopener noreferrer" target="_blank">
    报告
   </a>
   中做出的谴责。
  </p>
  <p class="graf graf--p">
   调查结果来自与纽约警察局长达两年的法律纠纷中获得的文件，其中包括乔治城隐私和技术中心对全国警察使用面部识别的调查。
  </p>
  <p class="graf graf--p">
   该报告在一些城市（包括旧金山）的呼吁下发布，这些城市正在努力禁止警方使用该技术，这是对评论家们认为可能增加错误逮捕风险的秘密技术增强抵制的一部分。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <strong class="markup--strong markup--p-strong">
    如果警方将非常主观的、高度编辑过的或错误的信息放入他们的系统，那么面部识别算法的准确程度就并不重要了
   </strong>
   ，”该报告的作者、隐私和技术中心的高级助理 Clare Garvie 表示。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他们不会得到好的信息。他们不会获得有价值的线索。错误识别的风险很高。如果他们使用它并且不与辩护律师共享，则会违反正当程序。”
  </p>
  <p class="graf graf--p">
   面部识别算法将身份不明的人的图像与嫌疑人照片、预订照片和驾驶执照照片进行比较。
  </p>
  <p class="graf graf--p">
   警方正在采用该技术进行常规调查，称这有助于他们解决本来会很棘手的案件。他们说把这种技术当作一种调查工具，“逮捕某人并不是唯一的理由”。
  </p>
  <p class="graf graf--p">
   乔治城的这份报告并未专注于纽约警方。还有亚利桑那州马里科帕县、俄勒冈州华盛顿县、和佛罗里达州皮内拉斯县。
  </p>
  <p class="graf graf--p">
   NYPD在报告中占据最重要的位置。
  </p>
  <figure aria-describedby="caption-attachment-4413" class="wp-caption aligncenter" id="attachment_4413" style="width: 1110px">
   <img alt="" class="size-full wp-image-4413 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1、.png?resize=1100%2C622&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1、.png?w=1110&amp;ssl=1 1110w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1、.png?resize=300%2C170&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1、.png?resize=768%2C435&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1、.png?resize=1024%2C579&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1、.png?resize=1100%2C622&amp;ssl=1 1100w" data-recalc-dims="1" height="622" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1、.png?resize=1100%2C622&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="size-full wp-image-4413" data-recalc-dims="1" height="622" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1、.png?resize=1100%2C622&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1、.png?w=1110&amp;ssl=1 1110w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1、.png?resize=300%2C170&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1、.png?resize=768%2C435&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1、.png?resize=1024%2C579&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1、.png?resize=1100%2C622&amp;ssl=1 1100w" width="1100"/>

   <figcaption class="wp-caption-text" id="caption-attachment-4413">
    A document obtained by the ACLU describes a “case study” of using a sketch for facial recognition in Washington County, Oregon Public records obtained by ACLU Oregon &amp; Northern California
   </figcaption>
  </figure>
  <p class="graf graf--p">
   研究人员为了获取有关其面部识别系统如何运作的信息，与 NYPD 进行过冗长的法庭冲突。
  </p>
  <p class="graf graf--p">
   该部门最终交出了数千页的文件，但此后一直试图让研究人员归还其中一些文件，称这些文件
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nydailynews.com/new-york/ny-nypd-facial-recognition-disclosures-20190414-ifwgro76cje5tgiyq6wqsty7ou-story.html" href="https://www.nydailynews.com/new-york/ny-nypd-facial-recognition-disclosures-20190414-ifwgro76cje5tgiyq6wqsty7ou-story.html" rel="noopener noreferrer" target="_blank">
    是保密的
   </a>
   并且是“无意中”共享的。
  </p>
  <p class="graf graf--p">
   纽约警察局发言人在一份声明中为其部门的做法辩护。该发言人对乔治城的报道没有任何异议，但表示该部门正在审查其“面部识别协议”。
  </p>
  <p class="graf graf--p">
   乔治城大学的研究人员表示，他们发现了一个案例，其中纽约警察局的调查人员试图使用监控录像找出一名从CVS偷啤酒的男子。
  </p>
  <p class="graf graf--p">
   研究人员引用侦探关于此案的内部报告称，监控图像的质量不是很高，并且在面部识别系统中没有产生任何潜在的匹配。
  </p>
  <p class="graf graf--p">
   但是一名侦探指出，嫌犯“看起来像”是演员伍迪哈里森，所以在嫌疑人的位置上提交了哈里森的高分辨率图片。然后导致逮捕。
  </p>
  <figure aria-describedby="caption-attachment-4415" class="wp-caption aligncenter" id="attachment_4415" style="width: 1120px">
   <img alt="" class="size-full wp-image-4415 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-5.png?resize=1100%2C581&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-5.png?w=1120&amp;ssl=1 1120w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-5.png?resize=300%2C159&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-5.png?resize=768%2C406&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-5.png?resize=1024%2C541&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-5.png?resize=1100%2C581&amp;ssl=1 1100w" data-recalc-dims="1" height="581" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-5.png?resize=1100%2C581&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

    <img alt="" class="size-full wp-image-4415" data-recalc-dims="1" height="581" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-5.png?resize=1100%2C581&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-5.png?w=1120&amp;ssl=1 1120w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-5.png?resize=300%2C159&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-5.png?resize=768%2C406&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-5.png?resize=1024%2C541&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-5.png?resize=1100%2C581&amp;ssl=1 1100w" width="1100"/>
   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-4415">
    An internal NYPD document, left, describes using celebrity comparisons for its facial recognition system, a technique that included the use of a photo of Woody Harrelson (not the photo pictured here).NYPD / AP File
   </figcaption>
  </figure>
  <p class="graf graf--p">
   研究人员还发现了一些证据，⚠️证明纽约警察局的调查人员篡改了嫌疑人的照片，使他们看起来更像是另一些人 —— 包括用一张闭着嘴的照片取代张着嘴的照片，从谷歌上搜的照片剪下嘴部贴在嫌疑人的照片上。
  </p>
  <p class="graf graf--p">
   研究人员表示，该部门还使用3D建模软件填补不完整的图像或旋转面部，使照片中的面部面向前方。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这些技术相当于面部识别点的制作：相比下较好的做法是创建从来不存在的信息（至少不会牵连无辜的人），最坏的情况则是引入完全无辜的其他人的信息作为证据，”报告说。
  </p>
  <p class="graf graf--p">
   该报告还详细介绍了纽约警察局的其他可疑方法，包括未能使用传统的调查技术来确认面部识别结果。
  </p>
  <p class="graf graf--p">
   在第二份报告中，乔治城的研究人员描述了底特律和芝加哥的计划 —— 通过将软件连接到位于城市各处的监控摄像头来
   <strong class="markup--strong markup--p-strong">
    创建实时的面部识别系统
   </strong>
   。
  </p>
  <p class="graf graf--p">
   批评者认为，这种做法是政府扩大公众监视能力的进一步。
  </p>
  <p>
   ⚠️ 想想看，这种做法还能怎么用？把当局不喜欢的反对派领导人的脸放在杀人现场的监视视频中？如果没有独立调查机构及时揭露呢？会有多少冤狱甚至死刑？
  </p>
  <p>
   <a href="https://www.nbcnews.com/news/us-news/nypd-used-celebrity-doppelg-ngers-fudge-facial-recognition-results-researchers-n1006411" rel="noopener noreferrer" target="_blank">
    NYPD used celebrity doppelgängers to fudge facial recognition results, researchers say
   </a>
   . The NYPD is also accused of editing suspects’ photos to get results from its facial recognition system, a new report says.
  </p>
