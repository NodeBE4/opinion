  <p class="graf graf--p">
   同样是做调查，有些人很擅长技术，调查过程更轻松、更深入，最重要的是更安全；另一些人则不擅长技术，往往只能通过社交关系取得转述性的信息，并难以验证，而且自己和信息源都非常不安全。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    首先回顾一下《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e7%9f%a5%e9%81%93%e4%bb%80%e4%b9%88%e6%89%8d%e6%98%af%e7%9c%9f%e6%ad%a3%e7%9a%84%e8%b0%83%e6%9f%a5%e5%90%97%ef%bc%9f%e6%89%80%e8%b0%93%e7%9a%84%e8%b5%b7%e5%ba%95%e7%bb%8f%e5%b8%b8/" href="https://www.iyouport.org/%e4%bd%a0%e7%9f%a5%e9%81%93%e4%bb%80%e4%b9%88%e6%89%8d%e6%98%af%e7%9c%9f%e6%ad%a3%e7%9a%84%e8%b0%83%e6%9f%a5%e5%90%97%ef%bc%9f%e6%89%80%e8%b0%93%e7%9a%84%e8%b5%b7%e5%ba%95%e7%bb%8f%e5%b8%b8/" rel="noopener noreferrer" target="_blank">
     究竟什么是调查？（声称“起底”的大多是骗人的）
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   调查记者在技术能力方面落差极大，这点在全球很多国家都是如此。国际记者中心（ICFJ）去年发布过一份报告，证实了这个问题所引发的很多严重问题：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    技术人员在新闻编辑室中人数稀少
   </li>
   <li class="graf graf--li">
    在假新闻和黑客盛行的时代，只有极少数记者采取了预防措施
   </li>
   <li class="graf graf--li">
    超过一半的记者（54%）和新闻编辑室（52%）不能保护自己的信息
   </li>
   <li class="graf graf--li">
    只有不到一半（45%）的新闻编辑室有条件做日常数据分析
   </li>
   <li class="graf graf--li">
    只有40%的新闻编辑室提供数据分析方面的培训 ……
   </li>
  </ul>
  <p class="graf graf--p">
   事实上我们的
   <strong class="markup--strong markup--p-strong">
    “
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     开源情报
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ”栏目中介绍的所有工具、技巧和思考方式，都是用于调查的，不论是公民记者、NGO组织还是职业调查记者 —— 掌握信息验证、信息挖掘和数据分析的能力，都非常重要
   </strong>
   。
  </p>
  <p class="graf graf--p">
   熟悉本栏目的读者一定知道，我们提供过非常多的工具；但对于基本调查工作来说，您至少需要熟练掌握一些常规的工具 —— 所有工具就如一个超大的武器库，它能帮你完成几乎任何事；而常规工具就如多功能钥匙扣，每天都应该带在身上，因为大部分时间你都需要它们。
  </p>
  <p class="graf graf--p">
   本文来总结一下，作为调查者您的随身钥匙扣上应该装些什么。
  </p>
  <p class="graf graf--p">
   不要害怕，这些东西都非常简单，即便您没有经过任何培训也不难掌握（还记得你第一次拿到电脑时的样子吗？没人教你怎么谷歌，但你打开那个网页后立刻就学会了。当然，深度使用这些工具仍需要一定经验和知识）
  </p>
  <p class="graf graf--p">
   好的，现在开始。
  </p>

  <p class="graf graf--p">
   1、
   <strong class="markup--strong markup--p-strong">
    网页变更提醒
   </strong>
  </p>
  <p class="graf graf--p">
   作为调查记者，经常需要监测某些网页的内容变更情况。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://versionista.com/" href="https://versionista.com/" rel="noopener noreferrer" target="_blank">
    Versionista
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://visualping.io/" href="https://visualping.io/" rel="noopener noreferrer" target="_blank">
    VisualPing
   </a>
   这两款工具，能省去每天早上手动查看的麻烦。
  </p>
  <p class="graf graf--p">
   两者都十分简单易用：只需添加你想监测的网站以及您的个人邮箱地址，当目标网站出现变更时，你就能收到邮件提醒。
  </p>
  <p class="graf graf--p">
   这两款工具还会收藏目标网站的不同版本，您可查询某段时间内目标网站页面的变化。
  </p>
  <p class="graf graf--p">
   两者均提供免费的基本查询，付费可使用更多进阶功能。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10185 jetpack-lazy-image" data-lazy-sizes="(max-width: 1026px) 100vw, 1026px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-28.png?resize=1026%2C334&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-28.png?w=1026&amp;ssl=1 1026w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-28.png?resize=300%2C98&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-28.png?resize=768%2C250&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-28.png?resize=1024%2C333&amp;ssl=1 1024w" data-recalc-dims="1" height="334" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-28.png?resize=1026%2C334&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1026"/>

   </noscript>

  <p class="graf graf--p">
   2、
   <strong class="markup--strong markup--p-strong">
    文件管理神器
   </strong>
  </p>
  <p class="graf graf--p">
   调查记者经常需要处理海量文件，这方面被广泛推荐的工具之一是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/" href="https://www.documentcloud.org/" rel="noopener noreferrer" target="_blank">
    DocumentCloud
   </a>
   ，它可以将PDF文件转换成文本格式、统计用户上传文件的相关数据。
  </p>
  <p class="graf graf--p">
   但如果你需要分析上千份甚至更多的文件时，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.overviewdocs.com/" href="https://www.overviewdocs.com/" rel="noopener noreferrer" target="_blank">
    Overview
   </a>
   会更实用些。
  </p>
  <p class="graf graf--p">
   Overview 支持多种文件上传渠道，包括 DocumentCloud，并能处理少至几十页、多达百万页的文件。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e8%bf%8e%e6%8e%a5%e6%9b%b4%e5%a4%9a%e7%9a%84%e5%b7%b4%e6%8b%bf%e9%a9%ac%e6%96%87%e4%bb%b6%ef%bc%9f/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e8%bf%8e%e6%8e%a5%e6%9b%b4%e5%a4%9a%e7%9a%84%e5%b7%b4%e6%8b%bf%e9%a9%ac%e6%96%87%e4%bb%b6%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     如何迎接更多的“巴拿马文件”？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   此外，Overview 能以多种形式将文件集中的数据生成可视化、可生成词云或制作网络图。
  </p>
  <p class="graf graf--p">
   Overview 上的多款搜索工具还能帮你轻松筛选数据，以查询特定信息。
  </p>
  <p class="graf graf--p">
   其另一功能是按内容为你的文件自动分组，保存至不同的文件夹。
  </p>
  <p class="graf graf--p">
   与 DocumentCloud 类似，Overview 也配备了光学字符识别技术（OCR），不仅可查看文件的原始格式，还可看纯文字版。并且，你还可以在文件上添加标签和笔记。
  </p>
  <p class="graf graf--p">
   3、
   <strong class="markup--strong markup--p-strong">
    文件加密共享
   </strong>
  </p>
  <p class="graf graf--p">
   安全是最重要的事 —— 不仅是您的个人安全，还包括与您联系的信息提供者的安全；危险性和信息的重要程度成正比。
  </p>
  <p class="graf graf--p">
   您必须有一个可信赖的媒介作为资料互通、与信源联络以分享某些敏感文件和信息的通道。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    敏感文件的传送必须能做到避免被拦截，而且传递双方都应该保存可靠的匿名性
   </strong>
   ，那么最简单的利器就是
   <a class="markup--anchor markup--p-anchor" data-href="https://onionshare.org/" href="https://onionshare.org/" rel="noopener noreferrer" target="_blank">
    OnionShare
   </a>
   。
  </p>
  <p class="graf graf--p">
   恰好今天的另一篇文章介绍了 OnionShare 2.2 在这里《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%96%b0%e7%89%88-onionshare-%e4%bd%bf%e4%bb%bb%e4%bd%95%e4%ba%ba%e9%83%bd%e5%8f%af%e4%bb%a5%e8%bd%bb%e6%9d%be%e5%8f%91%e5%b8%83%e5%8c%bf%e5%90%8d%e7%9a%84%e3%80%81%e6%97%a0%e6%b3%95%e5%ae%a1/" href="https://www.iyouport.org/%e6%96%b0%e7%89%88-onionshare-%e4%bd%bf%e4%bb%bb%e4%bd%95%e4%ba%ba%e9%83%bd%e5%8f%af%e4%bb%a5%e8%bd%bb%e6%9d%be%e5%8f%91%e5%b8%83%e5%8c%bf%e5%90%8d%e7%9a%84%e3%80%81%e6%97%a0%e6%b3%95%e5%ae%a1/" rel="noopener noreferrer" target="_blank">
    新版 OnionShare 使任何人都可以轻松发布匿名的、无法审查的网站
   </a>
   》。
  </p>
  <p class="graf graf--p">
   OnionShare 是一款开源软件，适用于大多数操作系统，包括 Windows、Mac OS 和 Linux。
  </p>
  <p class="graf graf--p">
   进行文件共享时，OnionShare 会建立一个安全的Tor服务器并生成一个链接，文件接收方可通过该链接和匿名的Tor浏览器下载文件。下载完成后，OnionShare 默认马上关闭连接，这样他人就无法找到文件的踪迹了。
  </p>
