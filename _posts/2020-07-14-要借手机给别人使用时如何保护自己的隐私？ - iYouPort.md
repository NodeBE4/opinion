   要借手机给别人使用时如何保护自己的隐私？
  </h1>
 </header>

 <div class="entry-content">
  <p class="graf graf--p">
   Android 访客模式是一个选项，可让您隐藏自己的所有东西，但仍保持手机正常运行。
  </p>
  <p class="graf graf--p">
   切换到访客模式时，您将隐藏所有应用程序、历史记录、图片、消息等，同时允许其他人使用您的手机。
  </p>
  <p class="graf graf--p">
   使用 Android 访客模式就像在您的主要手机中拥有一部完全独立的手机一样。
  </p>
  <p class="graf graf--p">
   访客模式可以具有不同的应用程序、文件、主屏幕小部件、电子邮件等。访客用户仍可以拨打电话、安装应用程序和下载自己的文件，但不会与您的账户发生冲突。
  </p>
  <p class="graf graf--p">
   下面来看一下如何启用Android访客模式，以便您可以为朋友或家人在需要手机时使用这个简洁的备用帐户（或者在警察查手机的时候糊弄一下）。
  </p>
  <p class="graf graf--p">
   也许一个陌生人想要借用您的手机打个私人电话，而不希望被他们窥探您的短信或银行信息；或者您的孩子用您的手机看动画片，而您不希望孩子看到您的私人联系人跳出来；再或者，您的手机坏了，拿去送修，修理人员说 “过几个小时后再来取”…… 这些情况下最好的方法就是启动访客模式。
  </p>
  <p class="graf graf--p">
   如果您想要安静一段时间不被打扰，Android访客模式也很有用。由于您的常规应用未以访客模式安装，因此您不会在工作期间尝试打开游戏或与朋友外出时浏览社交媒体。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-10973 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/访客.jpg?resize=1100%2C769&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/访客.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/访客.jpg?resize=300%2C210&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/访客.jpg?resize=1024%2C716&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/访客.jpg?resize=768%2C537&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/访客.jpg?resize=1100%2C769&amp;ssl=1 1100w" data-recalc-dims="1" height="769" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/访客.jpg?resize=1100%2C769&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如何设置Android 访客模式
   </strong>
  </p>
  <p class="graf graf--p">
   开启Android的访客模式非常简单。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    转到“设置”&gt;“系统”&gt;“高级”&gt;“多个用户”，然后启用多个用户（如果尚未启用的话）。
   </li>
   <li class="graf graf--li">
    点击 Add guest 或 Guest
   </li>
  </ul>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10974 jetpack-lazy-image" data-lazy-sizes="(max-width: 874px) 100vw, 874px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-25.png?resize=874%2C722&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-25.png?w=874&amp;ssl=1 874w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-25.png?resize=300%2C248&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-25.png?resize=768%2C634&amp;ssl=1 768w" data-recalc-dims="1" height="722" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-25.png?resize=874%2C722&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="874"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10974" data-recalc-dims="1" height="722" sizes="(max-width: 874px) 100vw, 874px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-25.png?resize=874%2C722&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-25.png?w=874&amp;ssl=1 874w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-25.png?resize=300%2C248&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-25.png?resize=768%2C634&amp;ssl=1 768w" width="874"/>
   </noscript>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    如果要接听电话，请在多用户屏幕上点击访客选项旁边的设置按钮。否则，访客用户将无法拨打或接听电话。
   </li>
   <li class="graf graf--li">
    快速进入和退出Android访客模式的另一种方法是从屏幕顶部向下滑动以查看通知面板。将其完全展开以查看所有选项，然后选择头像按钮以选择Guest。
   </li>
  </ul>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10975 jetpack-lazy-image" data-lazy-sizes="(max-width: 708px) 100vw, 708px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-24.png?resize=708%2C554&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-24.png?w=708&amp;ssl=1 708w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-24.png?resize=300%2C235&amp;ssl=1 300w" data-recalc-dims="1" height="554" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-24.png?resize=708%2C554&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="708"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10975" data-recalc-dims="1" height="554" sizes="(max-width: 708px) 100vw, 708px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-24.png?resize=708%2C554&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-24.png?w=708&amp;ssl=1 708w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-24.png?resize=300%2C235&amp;ssl=1 300w" width="708"/>
   </noscript>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    在访客模式用完后，在您选择其他帐户进行切换之前，“访客” 按钮将变为 “删除访客”。
   </li>
  </ul>
  <p class="graf graf--p">
   这是一种简单的方法，可清除您在访客模式下所做的所有操作，以便下次访问该帐户时，它是一个全新的帐户。
  </p>
  <p class="graf graf--p">
   如果您不清除访客模式，则下次打开时仍可以执行。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    帐户之间共享什么
   </strong>
  </p>
  <p class="graf graf--p">
   手机上的每个帐户都共享对应用程序和设置进行的更新，例如Wi-Fi网络详细信息和已配对的蓝牙设备。
  </p>
  <p class="graf graf--p">
   这意味着当一个用户更新应用时，对所有用户来说都会更新该应用。无线设备也是如此。无论您还是访客用户都加入了一个无线网络，由于密码是在两个帐户之间共享的，因此你们都可以访问该网络。
  </p>
  <p class="graf graf--p">
   但是，访客帐户和主帐户之间不会共享存储在应用程序中的文本、文件、电子邮件、音乐、视频、文档、照片和数据。
  </p>
  <p class="graf graf--p">
   您可以开设电子邮件帐户、下载文档、拍照等，而其他帐户只有切换到您的帐户后才能看到。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    安卓手机上的访客与用户
   </strong>
  </p>
  <p class="graf graf--p">
   当切换到Android访客模式或启用多用户功能时，您可能已经注意到了添加用户的选项。
  </p>
  <p class="graf graf--p">
   用户和访客帐户实际上是相似的帐户，但有一个主要区别：擦除和重新启动访客帐户更容易。
  </p>
  <p class="graf graf--p">
   每次切换到访客模式时，系统都会询问您是否要继续使用上一次的操作，或者是否要重新开始。
  </p>
  <p class="graf graf--p">
   切换到用户模式时您不会看到这样的提示。这是因为Android将访客模式视为临时模式，您可能希望在每次使用新功能之前将其擦除，而用户帐户则可以长期使用其自己的应用程序和文件。
  </p>
  <p class="graf graf--p">
   另一个区别是您无法在Android访客模式下发送或接收文本。如果启用了通话功能，则可以使用电话，但无法打开短信功能。
  </p>
  <p class="graf graf--p">
   如果您想要另一个与主要帐户不同的用户帐户，并且希望它能够发送短信，那么必须使用另一个用户帐户而不是访客模式（然后为其启用短信功能）。
  </p>
  <p class="graf graf--p">
   两者都有其优点和缺点，因此，使用哪种方法完全取决于您当时的需求。⚪️
  </p>
