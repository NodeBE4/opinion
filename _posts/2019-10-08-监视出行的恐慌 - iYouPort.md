  <p class="graf graf--p">
   美国移民和海关执法局一直在使用强大的技术来追踪人们的每一步出行，最新揭露的文件再次强调了这点。
  </p>
  <p class="graf graf--p">
   这些文件是由美国公民自由联盟获得并发布的，显示有超过9,200名ICE员工使用了美国各地自动车牌阅读器的信息数据库。
  </p>
  <p class="graf graf--p">
   像ICE这样的机构会继续依赖监控技术，联邦支出法案似乎提供1亿美元以资助边境监控技术。亚马逊和微软等公司也向 ICE 提供面部识别等监视产品。
  </p>
  <p class="graf graf--p">
   这些监视工具让ICE可以大规模地针对全民进行监视，许多倡导团体强烈指责其对隐私的侵犯。
   <strong class="markup--strong markup--p-strong">
    由于软件显示它能够跟踪人们的一举一动，无论是否经过人们的同意，对监控的担忧都在增加
   </strong>
   。在中国，监控技术被用于监控公民和分配社会信用评分，这可能导致民事处罚。
  </p>
  <p class="graf graf--p">
   ACLU 表示，ICE 使用的数据库包含超过50亿个位置数据，由车牌读取器在停车场、道路、收费站和警车上收集。这些数据是在每个通过读卡器的牌照上收集的 — 这意味着每天通勤的无辜乘客都会被捕捉和被窥视他们的一举一动。
  </p>
  <p class="graf graf--p">
   文件显示，
   <strong class="markup--strong markup--p-strong">
    另有15亿个监视点来自数十个州的80多个当地警察局
   </strong>
   。 ACLU 表示，这种大规模监视违反了当地的隐私法。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “存储大量位置信息既是对隐私的重大侵犯，并且实时追踪是完全没有必要的，”北加州 ACLU 的律师 Vasudha Talla 周三在一篇文章中说。
  </p>
  <p class="graf graf--p">
   根据 ACLU 获得的文件，该数据库每月都会增加1.5亿至2亿次新的扫描数据。这些自动牌照阅读器可以每分钟拍摄数千个牌照。
  </p>
  <p class="graf graf--p">
   该数据来自 VIGilant Solutions，这是一家ICE于2018年开始合作的公司，其合同将在2020年到期。该公司表示，它拥有大约60％的美国人口的车牌数据，包括亚特兰大，芝加哥，克利夫兰，洛杉矶，纽约，旧金山和德克萨斯州奥斯汀等地区。
  </p>
  <p class="graf graf--p">
   数据收集的一个关键部分涉及向当地警察部门索取信息。 Vigilant 的文件包括如何与当地警察联系并向当地警方索取位置数据的分步指南。
  </p>
  <p class="graf graf--p">
   ICE 的隐私规则要求所有使用车牌数据的记录请求。但 ACLU 获得的日志显示这些请求通常是非正式的，通过电子邮件提供。
  </p>
  <p class="graf graf--p">
   就其本身而言，ICE在一份声明中表示，它并未使用车牌数据来“定位或跟踪与ICE调查或执法活动无关的个人”。
  </p>
  <p class="graf graf--p">
   尽管如此，ACLU 仍在呼吁当地执法部门停止与ICE分享人们的信息。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “令人震惊的是，ICE 已将这一大规模监视数据库添加到其武器库中，并且当地执法机构和私营公司正在帮助该机构进行监视工作，”Talla 在一份声明中说。
  </p>
