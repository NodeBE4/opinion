 <div class="entry-content">
  <p class="graf graf--p">
   如果您以某种方式忘记了锁定 Android 设备的模式、PIN或密码，您可能会认为自己运气不好并注定永远被锁定。
  </p>
  <p class="graf graf--p">
   或者你的设备被警察抢走了，而你觉得自己能扛得住酷刑不必交出密码就能获得安全。
  </p>
  <p class="graf graf--p">
   这都不可能。
  </p>
  <p class="graf graf--p">
   因为，在许多情况下，闯入锁定设备并不难做到。
  </p>
  <p class="graf graf--p">
   有多种不同的方法可以破解锁定的Android智能手机或平板电脑，但没有哪种方法比
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amazon.com.au/Recovery-Password-Better-Windows-Versions-7/dp/B01BO6604S/?tag=wghau-22" href="https://www.amazon.com.au/Recovery-Password-Better-Windows-Versions-7/dp/B01BO6604S/?tag=wghau-22" rel="noopener noreferrer" target="_blank">
    密码破解 USB 棒
   </a>
   更简单了。
  </p>
  <p class="graf graf--p">
   本文将介绍7种最有效的破解方法；在没有USB的情况下。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第一种：使用 Google 的“查找我的设备”功能
   </strong>
  </p>
  <p class="graf graf--p">
   对于大多数 Android 手机和平板电脑，都有“查找我的设备”的内置服务。
  </p>
  <p class="graf graf--p">
   只要您登录自己的 Google 帐户，就可以使用任何设备或计算机访问
   <a class="markup--anchor markup--p-anchor" data-href="https://www.google.com/android/find" href="https://www.google.com/android/find" rel="noopener noreferrer" target="_blank">
    此链接
   </a>
   提供的服务。
  </p>
  <p class="graf graf--p">
   测试中我们注意到这个方法在 Android 8.0 或更高版本上不起作用。但只要您的手机运行 Android 7.1.1 Nougat 或更低，它就可以解决问题。
  </p>
  <p class="graf graf--p">
   一旦使用“查找我的设备”在您的手机上获得了修复，
   <strong class="markup--strong markup--p-strong">
    请先单击“锁定”按钮
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果该服务在查找设备时遇到问题，请单击刷新按钮几次，如果您的手机兼容，则应在5次左右的尝试中建立连接。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6252 jetpack-lazy-image" data-lazy-sizes="(max-width: 880px) 100vw, 880px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-15.jpg?resize=880%2C621&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-15.jpg?w=880&amp;ssl=1 880w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-15.jpg?resize=300%2C212&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-15.jpg?resize=768%2C542&amp;ssl=1 768w" data-recalc-dims="1" height="621" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-15.jpg?resize=880%2C621&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="880"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6252" data-recalc-dims="1" height="621" sizes="(max-width: 880px) 100vw, 880px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-15.jpg?resize=880%2C621&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-15.jpg?w=880&amp;ssl=1 880w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-15.jpg?resize=300%2C212&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-15.jpg?resize=768%2C542&amp;ssl=1 768w" width="880"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   单击“锁定”按钮后，系统将提示您输入新密码，该密码将替换您忘记的模式、PIN或密码。
  </p>
  <p class="graf graf--p">
   输入两次新密码以确认您的选择，然后单击“锁定”按钮。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6253 jetpack-lazy-image" data-lazy-sizes="(max-width: 880px) 100vw, 880px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-10.jpg?resize=880%2C621&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-10.jpg?w=880&amp;ssl=1 880w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-10.jpg?resize=300%2C212&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-10.jpg?resize=768%2C542&amp;ssl=1 768w" data-recalc-dims="1" height="621" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-10.jpg?resize=880%2C621&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="880"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   密码最多可能需要5分钟才能切换。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第二种：使用三星的“查找我的手机”服务
   </strong>
  </p>
  <p class="graf graf--p">
   如果您有三星设备，那么可以使用名为“查找我的手机”的类似服务。
  </p>
  <p class="graf graf--p">
   首先从任何 Web 浏览器前往
   <a class="markup--anchor markup--p-anchor" data-href="https://findmymobile.samsung.com/login.do" href="https://findmymobile.samsung.com/login.do" rel="noopener noreferrer" target="_blank">
    此链接
   </a>
   ，然后登录到您的 Samsung 帐户。
  </p>
  <p class="graf graf--p">
   如果您从未设置过三星帐户，那么此方法将无法运行。此外，一些运营商，如 Sprint，锁定了这项服务。
  </p>
  <p class="graf graf--p">
   登录 Samsung 帐户后，单击左侧窗格中的“锁定我的屏幕”按钮。
  </p>
  <p class="graf graf--p">
   在此处，在第一个字段中输入新的 PIN，然后单击屏幕底部附近的“锁定”按钮。
  </p>
  <p class="graf graf--p">
   在一两分钟内，您的锁屏密码应更改为您刚刚输入的 PIN，您可以使用它来解锁设备。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6254 jetpack-lazy-image" data-lazy-sizes="(max-width: 975px) 100vw, 975px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-7.jpg?resize=975%2C619&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-7.jpg?w=975&amp;ssl=1 975w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-7.jpg?resize=300%2C190&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-7.jpg?resize=768%2C488&amp;ssl=1 768w" data-recalc-dims="1" height="619" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-7.jpg?resize=975%2C619&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="975"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第三种：使用“遗忘模式”功能
   </strong>
  </p>
  <p class="graf graf--p">
   如果您的设备运行的是 Android 4.4或更低版本，请尝试使用“遗忘模式”功能。
  </p>
  <p class="graf graf--p">
   在5次尝试解锁失败后，您会看到一条消息“30秒后再试一次”。
  </p>
  <p class="graf graf--p">
   显示此消息时，点击屏幕底部显示“遗忘模式”的按钮。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6255 jetpack-lazy-image" data-lazy-sizes="(max-width: 1064px) 100vw, 1064px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-11.png?resize=1064%2C656&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-11.png?w=1064&amp;ssl=1 1064w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-11.png?resize=300%2C185&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-11.png?resize=768%2C474&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-11.png?resize=1024%2C631&amp;ssl=1 1024w" data-recalc-dims="1" height="656" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-11.png?resize=1064%2C656&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1064"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   在此处，选择“输入 Google 帐户详细信息”（根据您的设备，您可以直接转到此选项），然后输入您的主要 Gmail 帐户和密码。
  </p>
  <p class="graf graf--p">
   Google 会向您发送包含解锁图案的电子邮件，或者您可以随时更改。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6256 jetpack-lazy-image" data-lazy-sizes="(max-width: 1066px) 100vw, 1066px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/5-9.png?resize=1066%2C876&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/5-9.png?w=1066&amp;ssl=1 1066w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/5-9.png?resize=300%2C247&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/5-9.png?resize=768%2C631&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/5-9.png?resize=1024%2C841&amp;ssl=1 1024w" data-recalc-dims="1" height="876" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/5-9.png?resize=1066%2C876&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1066"/>

   </noscript>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第四种：执行出厂重置
   </strong>
  </p>
  <p class="graf graf--p">
   如果您更关心的是进入手机而不是保留存储在手机上的任何数据，那么出厂重置应该适用于许多情况。
  </p>
  <p class="graf graf--p">
   但由于新的防盗功能称为出厂重置保护，如果手机在2016年或之后出厂，您需要知道自己的 Google 帐户密码才能使用此方法。
  </p>
  <p class="graf graf--p">
   此过程将根据您的设备类型而有所不同，但对于大多数手机，请先关闭设备电源。
  </p>
  <p class="graf graf--p">
   当屏幕变黑时，同时按住音量和电源按钮，这将打开 Android 的引导程序菜单。
  </p>
  <p class="graf graf--p">
   在此处，按两次音量降低按钮以突出显示“恢复模式”选项，然后按电源按钮将其选中。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6257 jetpack-lazy-image" data-lazy-sizes="(max-width: 1060px) 100vw, 1060px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/6-9.png?resize=1060%2C908&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/6-9.png?w=1060&amp;ssl=1 1060w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/6-9.png?resize=300%2C257&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/6-9.png?resize=768%2C658&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/6-9.png?resize=1024%2C877&amp;ssl=1 1024w" data-recalc-dims="1" height="908" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/6-9.png?resize=1060%2C908&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1060"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   接下来，按住电源按钮并点击音量增大按钮一次，然后您的手机应进入恢复模式。
  </p>
  <p class="graf graf--p">
   在此处，使用音量按钮突出显示“擦除数据/恢复出厂设置”选项，然后按电源按钮将其选中。
  </p>
  <p class="graf graf--p">
   完成此过程后，选择“立即重新启动系统”选项，手机就打开了。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6258 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-1.jpg?resize=1100%2C572&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-1.jpg?w=1114&amp;ssl=1 1114w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-1.jpg?resize=300%2C156&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-1.jpg?resize=768%2C399&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-1.jpg?resize=1024%2C532&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-1.jpg?resize=1100%2C572&amp;ssl=1 1100w" data-recalc-dims="1" height="572" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-1.jpg?resize=1100%2C572&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6258" data-recalc-dims="1" height="572" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-1.jpg?resize=1100%2C572&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-1.jpg?w=1114&amp;ssl=1 1114w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-1.jpg?resize=300%2C156&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-1.jpg?resize=768%2C399&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-1.jpg?resize=1024%2C532&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-1.jpg?resize=1100%2C572&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   如果是较新的手机，系统会提示您使用 Google 帐户和密码登录。
  </p>
  <p class="graf graf--p">
   此时只需重新登录您的 Google 帐户即可重新获得对手机的访问权限。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第五种：使用ADB删除密码文件
   </strong>
  </p>
  <p class="graf graf--p">
   这种方法仅在您之前已在手机上启用USB调试时才有效；并且，只有在您允许使用的计算机通过 ADB 连接时，它才会起作用。
  </p>
  <p class="graf graf--p">
   但如果您满足这些要求，这种方法是解锁设备的完美方式。
  </p>
  <p class="graf graf--p">
   请注意，默认情况下启用加密的模型可能与此方法不兼容。
  </p>
  <p class="graf graf--p">
   首先使用 USB 数据线将手机连接到计算机，然后在ADB安装目录中打开命令提示符窗口。在此处，键入以下命令，然后按Enter键。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     adb shell rm /data/system/gesture.key
    </strong>
   </li>
  </ul>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6259 jetpack-lazy-image" data-lazy-sizes="(max-width: 548px) 100vw, 548px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/8-3.jpg?resize=548%2C213&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/8-3.jpg?w=548&amp;ssl=1 548w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/8-3.jpg?resize=300%2C117&amp;ssl=1 300w" data-recalc-dims="1" height="213" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/8-3.jpg?resize=548%2C213&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="548"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6259" data-recalc-dims="1" height="213" sizes="(max-width: 548px) 100vw, 548px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/8-3.jpg?resize=548%2C213&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/8-3.jpg?w=548&amp;ssl=1 548w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/8-3.jpg?resize=300%2C117&amp;ssl=1 300w" width="548"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   接下来，重新启动手机，安全锁屏已经消失。
  </p>
  <p class="graf graf--p">
   但这只是暂时的，因此请确保立即设置新模式、PIN或密码。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第六种：启动到安全模式以绕过第三方锁屏
   </strong>
  </p>
  <p class="graf graf--p">
   如果您试图绕过的锁屏是第三方应用程序，则启动到安全模式是最简单的方法。
  </p>
  <p class="graf graf--p">
   对于大多数手机，您可以通过从锁定屏幕调出电源菜单，然后长按“关闭电源”选项启动到安全模式。
  </p>
  <p class="graf graf--p">
   选择“确定”，当此过程完成时，您的第三方锁屏应用将暂时禁用。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-6260 jetpack-lazy-image" data-lazy-sizes="(max-width: 1056px) 100vw, 1056px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9-7.png?resize=1056%2C916&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9-7.png?w=1056&amp;ssl=1 1056w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9-7.png?resize=300%2C260&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9-7.png?resize=768%2C666&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9-7.png?resize=1024%2C888&amp;ssl=1 1024w" data-recalc-dims="1" height="916" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9-7.png?resize=1056%2C916&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1056"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6260" data-recalc-dims="1" height="916" sizes="(max-width: 1056px) 100vw, 1056px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9-7.png?resize=1056%2C916&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9-7.png?w=1056&amp;ssl=1 1056w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9-7.png?resize=300%2C260&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9-7.png?resize=768%2C666&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9-7.png?resize=1024%2C888&amp;ssl=1 1024w" width="1056"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   从这里，只需清除第三方锁屏应用上的数据或将其卸载，然后重新启动手机即可退出锁定模式。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第七种：崩溃锁屏UI
   </strong>
  </p>
  <p class="graf graf--p">
   最后，如果您的设备已加密并运行 Android 5.0–5.1.1，也有办法可以绕过密码锁屏。
  </p>
  <p class="graf graf--p">
   首先，点击锁定屏幕上的“紧急呼叫”选项，然后使用拨号程序界面输入10个星号。
  </p>
  <p class="graf graf--p">
   在此处，双击该字段以突出显示输入的文本并选择“复制”，然后将其粘贴到同一字段中，以使输入字符的数量加倍。
  </p>
  <p class="graf graf--p">
   一直重复此复制和粘贴的相同过程，以添加更多字符，直到双击该字段不再能突出显示字符。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6262 jetpack-lazy-image" data-lazy-sizes="(max-width: 1074px) 100vw, 1074px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/10.png?resize=1074%2C564&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/10.png?w=1074&amp;ssl=1 1074w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/10.png?resize=300%2C158&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/10.png?resize=768%2C403&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/10.png?resize=1024%2C538&amp;ssl=1 1024w" data-recalc-dims="1" height="564" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/10.png?resize=1074%2C564&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1074"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6262" data-recalc-dims="1" height="564" sizes="(max-width: 1074px) 100vw, 1074px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/10.png?resize=1074%2C564&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/10.png?w=1074&amp;ssl=1 1074w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/10.png?resize=300%2C158&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/10.png?resize=768%2C403&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/10.png?resize=1024%2C538&amp;ssl=1 1024w" width="1074"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   接下来，返回锁定屏幕并打开相机快捷方式。在此处，下拉通知帘并点击“设置”图标，系统将提示您输入密码。
  </p>
  <p class="graf graf--p">
   长按输入字段并选择“粘贴”，然后再重复此过程多次。
  </p>
  <p class="graf graf--p">
   最后您的锁定屏幕将崩溃，这将允许您访问手机的其余界面。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6263 jetpack-lazy-image" data-lazy-sizes="(max-width: 1068px) 100vw, 1068px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/11.png?resize=1068%2C554&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/11.png?w=1068&amp;ssl=1 1068w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/11.png?resize=300%2C156&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/11.png?resize=768%2C398&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/11.png?resize=1024%2C531&amp;ssl=1 1024w" data-recalc-dims="1" height="554" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/11.png?resize=1068%2C554&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1068"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6263" data-recalc-dims="1" height="554" sizes="(max-width: 1068px) 100vw, 1068px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/11.png?resize=1068%2C554&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/11.png?w=1068&amp;ssl=1 1068w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/11.png?resize=300%2C156&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/11.png?resize=768%2C398&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/11.png?resize=1024%2C531&amp;ssl=1 1024w" width="1068"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   这是黑客常用的方法。
  </p>
  <p class="graf graf--p">
   您知道任何绕过 Android 锁屏的其他方法吗？欢迎留言。
  </p>
  <p class="graf graf--p">
   如果您的目标是避免被恶意解锁，那么需要了解上述解锁方法，并反用它们，以找到合适您所处的紧急状况的应对方法。⚪️
  </p>
