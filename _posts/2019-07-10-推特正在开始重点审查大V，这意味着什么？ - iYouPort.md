    <img alt="" class="aligncenter size-full wp-image-6387" data-recalc-dims="1" height="733" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头.jpeg?resize=1100%2C733&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头.jpeg?w=2048&amp;ssl=1 2048w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头.jpeg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头.jpeg?resize=768%2C512&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头.jpeg?resize=1024%2C682&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头.jpeg?resize=1100%2C733&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头.jpeg?resize=272%2C182&amp;ssl=1 272w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Twitter 正在推出新的通知，警告属于公众人物的账号违反社区规则的推文。
  </p>
  <p class="graf graf--p">
   现在，如果像特朗普这样的人物发布违反了 Twitter 规则的推文，该平台可以通知用户关于违规，并减少相关推文的覆盖范围。
  </p>
  <p class="graf graf--p">
   在最近的采访中，Twitter 的高管们暗示，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.vox.com/recode/2019/6/27/18760444/twitter-kayvon-beykpour-vijaya-gadde-kara-swisher-peter-kafka-code-conference-interview-transcript" href="https://www.vox.com/recode/2019/6/27/18760444/twitter-kayvon-beykpour-vijaya-gadde-kara-swisher-peter-kafka-code-conference-interview-transcript" rel="noopener noreferrer" target="_blank">
    这样的改变即将到来
   </a>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    此通知仅适用于属于政治人物、经过认证的用户、以及拥有超过100,000个关注者的帐户的推文
   </strong>
   。
  </p>
  <p class="graf graf--p">
   也就是说，如果有较低关注度但整体大量的水军出没，不在此通知范围内。
  </p>
  <p class="graf graf--p">
   如果推文被标记为违反平台规则，那么推特公司有一组人专门用来决定这是否是“公共利益问题”。
  </p>
  <p class="graf graf--p">
   在推文通知用户违规之前会出现一个浅灰色框，点击该框的用户仍然可以看到相关推文。
  </p>
  <p class="graf graf--p">
   理论上，这可以保留推文作为公共记录的一部分，但是不允许通过 Twitter 平台将其推广到新的受众。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6388 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1.jpeg?resize=1100%2C1100&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1.jpeg?w=2048&amp;ssl=1 2048w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1.jpeg?resize=150%2C150&amp;ssl=1 150w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1.jpeg?resize=300%2C300&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1.jpeg?resize=768%2C768&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1.jpeg?resize=1024%2C1024&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1.jpeg?resize=1100%2C1100&amp;ssl=1 1100w" data-recalc-dims="1" height="1100" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1.jpeg?resize=1100%2C1100&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6388" data-recalc-dims="1" height="1100" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1.jpeg?resize=1100%2C1100&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1.jpeg?w=2048&amp;ssl=1 2048w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1.jpeg?resize=150%2C150&amp;ssl=1 150w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1.jpeg?resize=300%2C300&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1.jpeg?resize=768%2C768&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1.jpeg?resize=1024%2C1024&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1.jpeg?resize=1100%2C1100&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “过去，我们允许某些违反我们规则的推文留在 Twitter 上，因为它们符合公众的利益，但我们不知道何时以及如何做出这些决定，” Twitter 的安全团队在一篇博文中写道。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “为公众对话提供服务包括为任何人提供谈论对他们重要的事情的能力；在与政府官员和政治人物交往时，这一点尤其重要”。
  </p>
  <p class="graf graf--p">
   收到此通知的推文将不再出现在搜索结果中、不在热门推文时间轴上、也不在直播活动页面、推荐推送通知，或通知选项卡页面中。
  </p>
  <p class="graf graf--p">
   媒体主要关注的是特朗普，他多次因违反 Twitter 规则而受到批评。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nbcnews.com/politics/donald-trump/president-trump-tweets-wwe-video-himself-attacking-cnn-n779031" href="https://www.nbcnews.com/politics/donald-trump/president-trump-tweets-wwe-video-himself-attacking-cnn-n779031" rel="noopener noreferrer" target="_blank">
    2017年
   </a>
   ，他在 Twitter 上发布了一张照片，上面显示自己正在与 CNN 搏斗，许多人认为这是针对媒体的暴力行为。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theverge.com/2018/1/5/16855900/twitter-donald-trump-ban-explanation" href="https://www.theverge.com/2018/1/5/16855900/twitter-donald-trump-ban-explanation" rel="noopener noreferrer" target="_blank">
    在2018年1月
   </a>
   ，Twitter 专门致电禁止特朗普使用该服务。
  </p>
  <p class="graf graf--p">
   该公司当时在一篇博客文章中表示，“封锁 Twitter 上的世界领导者或删除他们有争议的推文，会隐藏人们应该能够看到和加入辩论的重要信息。”
  </p>
  <p class="graf graf--p">
   于是推特最终采取了“通知违规”的办法。而不是删除或封锁。
  </p>
  <p class="graf graf--p">
   目前尚不清楚这种通知的使用频率会如何，Twitter 在其博客文章中写道，它预计很少会应用此通知。
  </p>
  <p class="graf graf--p">
   简单说，这一“通知”作为标记，仅仅能显示推特声称“我知道了、我在帮助减轻影响”，此外做不到什么了。
  </p>
  <p class="graf graf--p">
   尤其是，如果你考虑到“是否是公众利益问题”的最终判断是由高度集中且不透明的内部小组决定的，这显然有可能会是一项可疑的审查。
  </p>
  <p class="graf graf--p">
   一方面，人们有权看到他们选举的官员的真面目，不论有多坏。而另一方面，对公众人物的标准的确应该更高。这似乎不是一个简单的问题。⚪️
  </p>
