  <p class="graf graf--p">
   今年是 Edward Snowden 泄露有关大规模监控计划如何运作的详细信息后的第六年，现在，至少两个国家 — 英国和南非 — 已经公开承认使用了最危险的批量拦截能力。

  <p class="graf graf--p">
   上面这张图描述了通信是如何通过互联网传播的，这对于理解大规模拦截的监视机制来说至关重要。
  </p>
  <p class="graf graf--p">
   所有互联网通信都被分解为更小的片段，称为“数据包”；每个数据包都包含实际通信内容的一部分以及元数据；元数据是有关通信的信息，例如 发件人和收件人、发送日期和位置以及主题行。
  </p>
  <p class="graf graf--p">
   在这里看到检验和避免元数据的工具《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%83%bd%e6%98%af%e5%b7%a5%e5%85%b7%ef%bc%8c%e4%bb%8e%e8%87%aa%e6%88%91%e4%bf%9d%e6%8a%a4%e5%88%b0%e6%9c%89%e6%95%88%e5%88%a9%e7%94%a8%e2%80%8a-%e4%ba%a6%e6%ad%a3%e4%ba%a6%e9%82%aa/" href="https://www.iyouport.org/%e9%83%bd%e6%98%af%e5%b7%a5%e5%85%b7%ef%bc%8c%e4%bb%8e%e8%87%aa%e6%88%91%e4%bf%9d%e6%8a%a4%e5%88%b0%e6%9c%89%e6%95%88%e5%88%a9%e7%94%a8%e2%80%8a-%e4%ba%a6%e6%ad%a3%e4%ba%a6%e9%82%aa/" rel="noopener noreferrer" target="_blank">
    从自我保护到有效利用
   </a>
   》。
  </p>
  <p class="graf graf--p">
   组成单个通信的数据包不仅可以通过不同的路径到达目的地，还可以采用任何可行的路径。距离不是决定性因素。
  </p>
  <p class="graf graf--p">
   因此，同一城市中两个人之间的通信可能会在到达其接收者之前在世界各地“旅行”。
  </p>
  <p class="graf graf--p">
   数据包在互联网上的分散意味着我们的通信和数据更容易被外国政府拦截，外国政府可能会在全球范围内反弹的过程中将其捕获。
  </p>
  <p class="graf graf--p">
   数据包的捕获，特别是因为它们包含的元数据，是非常具有侵入性的。
  </p>
  <p class="graf graf--p">
   互联网极大地改变了我们的沟通方式，并在此过程中创建了有关我们沟通的额外个人数据。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%97%a0%e5%a4%84%e5%8f%af%e8%97%8f%e7%9a%84%e6%95%b0%e5%ad%97%e8%b6%b3%e8%bf%b9-%e4%ba%a6%e6%ad%a3%e4%ba%a6%e9%82%aa%e5%85%83%e6%95%b0%e6%8d%ae-%e4%b8%8a%e7%af%87/" href="https://www.iyouport.org/%e6%97%a0%e5%a4%84%e5%8f%af%e8%97%8f%e7%9a%84%e6%95%b0%e5%ad%97%e8%b6%b3%e8%bf%b9-%e4%ba%a6%e6%ad%a3%e4%ba%a6%e9%82%aa%e5%85%83%e6%95%b0%e6%8d%ae-%e4%b8%8a%e7%af%87/" rel="noopener noreferrer" target="_blank">
    元数据相当于
   </a>
   让贴身间谍随时跟踪您、记录您去了哪里以及和谁说过话。
  </p>
  <p class="graf graf--p">
   但在数字领域，元数据记录得东西更多 —— 例如，您的网络活动，可以显示购买的商品，访问的新闻网站，加入的论坛，阅读的书籍和观看的电影。
  </p>
  <p class="graf graf--p">
   这些数据中的每一个都可以洞察一个人的内心。
  </p>
  <p class="graf graf--p">
   这些数据拼凑在一起就可以对一个人的全部私人生活进行侵入性的和全面的监控，揭示他或她的身份、关系、兴趣、活动和位置等等一切。
  </p>
  <p class="graf graf--p">
   移动数字设备越来越普遍，每时每刻都在产生数量不断增长的新数据形式。
  </p>
  <p class="graf graf--p">
   与此同时，存储数据的成本一直都在急剧下降。
  </p>
  <p class="graf graf--p">
   最重要的是，结合数据集和分析这一庞大数据库的技术手段已经发展得非常迅速，以前被认为无意义或不连贯的类型和数量的数据现在可以产生令人难以置信的启示性分析。
  </p>
  <p class="graf graf--p">
   元数据的结构使得计算机可以更快，更有效地搜索你的行为模式，并比通过实际通信内容的类似搜索更多地了解你。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9151 jetpack-lazy-image" data-lazy-sizes="(max-width: 1000px) 100vw, 1000px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-5.png?resize=1000%2C500&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-5.png?w=1000&amp;ssl=1 1000w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-5.png?resize=300%2C150&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-5.png?resize=768%2C384&amp;ssl=1 768w" data-recalc-dims="1" height="500" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-5.png?resize=1000%2C500&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1000"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9151" data-recalc-dims="1" height="500" sizes="(max-width: 1000px) 100vw, 1000px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-5.png?resize=1000%2C500&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-5.png?w=1000&amp;ssl=1 1000w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-5.png?resize=300%2C150&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-5.png?resize=768%2C384&amp;ssl=1 768w" width="1000"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   第二张图分解了批量拦截过程的各个阶段。下面将进一步解释这一过程，以英国为例（英国真正做满了全套）。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    拦截 ——
   </strong>
  </p>
  <p class="graf graf--p">
   英国信号情报机构政府通信总部 “GCHQ”通过利用登陆英国的海底光缆进行大规模拦截。和美国一样，英国的地理位置使其成为许多这些电缆的自然枢纽。
  </p>
  <p class="graf graf--p">
   斯诺登文件表明，英国政府 — 经常与电信公司合作 — 已经在这些电缆上安装探测器以拦截其流量。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    萃取 ——
   </strong>
  </p>
  <p class="graf graf--p">
   英国政府将被拦截到的流量的副本重定向到缓冲区，临时存储空间，这些存储空间将所有内容保留三天，元数据保留30天。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    筛选 ——
   </strong>
  </p>
  <p class="graf graf--p">
   然后根据“选择标准”过滤此信息。就像中国审查的关键字过滤。
  </p>
  <p class="graf graf--p">
   虽然英国政府提供的常见示例是电子邮件地址和电话号码，但是并不清楚允许过滤的全部范围。
  </p>
  <p class="graf graf--p">
   这些过滤标准还可以包括与特定国家/地区相关的所有流量，特定的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%b5%8f%e8%a7%88%e5%8e%86%e5%8f%b2%e8%ae%b0%e5%bd%95%e6%9b%9d%e5%85%89%e4%bd%a0%e7%9a%84%e5%86%85%e5%bf%83%e5%85%a8%e6%99%af%e5%9b%be%ef%bc%9aosint-%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e6%bc%94/" href="https://www.iyouport.org/%e6%b5%8f%e8%a7%88%e5%8e%86%e5%8f%b2%e8%ae%b0%e5%bd%95%e6%9b%9d%e5%85%89%e4%bd%a0%e7%9a%84%e5%86%85%e5%bf%83%e5%85%a8%e6%99%af%e5%9b%be%ef%bc%9aosint-%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e6%bc%94/" rel="noopener noreferrer" target="_blank">
    搜索引擎查询
   </a>
   或
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bd%93%e6%81%b6%e6%84%8f%e8%bd%af%e4%bb%b6%e4%bc%aa%e8%a3%85%e7%bd%91%e8%b4%ad%e7%89%a9%e6%b5%81%ef%bc%9a%e6%9d%a5%e8%87%aa%e6%94%bf%e5%ba%9c%e7%9a%84%e7%9b%91%e6%8e%a7%e6%80%bb%e8%83%bd%e8%ae%a9/" href="https://www.iyouport.org/%e5%bd%93%e6%81%b6%e6%84%8f%e8%bd%af%e4%bb%b6%e4%bc%aa%e8%a3%85%e7%bd%91%e8%b4%ad%e7%89%a9%e6%b5%81%ef%bc%9a%e6%9d%a5%e8%87%aa%e6%94%bf%e5%ba%9c%e7%9a%84%e7%9b%91%e6%8e%a7%e6%80%bb%e8%83%bd%e8%ae%a9/" rel="noopener noreferrer" target="_blank">
    在线购买
   </a>
   ，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%8f%af%e4%bb%a5%e9%87%87%e5%8f%96%e5%93%aa%e4%ba%9b%e6%8e%aa%e6%96%bd%e6%9d%a5%e9%98%bb%e6%ad%a2-google-%e5%92%8c%e5%85%b6%e4%bb%96%e4%ba%ba%e8%b7%9f%e8%b8%aa%e6%82%a8%e7%9a%84%e6%89%8b%e6%9c%ba/" href="https://www.iyouport.org/%e5%8f%af%e4%bb%a5%e9%87%87%e5%8f%96%e5%93%aa%e4%ba%9b%e6%8e%aa%e6%96%bd%e6%9d%a5%e9%98%bb%e6%ad%a2-google-%e5%92%8c%e5%85%b6%e4%bb%96%e4%ba%ba%e8%b7%9f%e8%b8%aa%e6%82%a8%e7%9a%84%e6%89%8b%e6%9c%ba/" rel="noopener noreferrer" target="_blank">
    地理位置
   </a>
   或一系列IP地址等等。
  </p>
  <p class="graf graf--p">
   这些例子中的每一个都可以捕获许多许多人的私密信息。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    存储和分析 ——
   </strong>
  </p>
  <p class="graf graf--p">
   截获的信息存储在数据库中，分析人员可以随意进行查询、数据挖掘或调用信息以进一步检查。
  </p>
  <p class="graf graf--p">
   2015年9月，The Intercept 发布的斯诺登文件记录了三个GCHQ项目 —— Black Hole、MUTANT BROTH 和 KARMA POLICE —— 揭示了英国政府批量拦截元数据的方式。
  </p>
  <p class="graf graf--p">
   Black Hole 是一个元数据存储库，存储
   <strong class="markup--strong markup--p-strong">
    “电子邮件和即时消息记录，有关搜索引擎查询的详细信息，有关社交媒体活动的信息，与黑客操作相关的日志，以及人们使用匿名工具（如Tor）浏览互联网的数据。
   </strong>
   ”
  </p>
  <p class="graf graf--p">
   MUTANT BROTH 筛选与 cookies 相关的 Black Hole 数据 —— 存储在设备上数据以识别和跟踪浏览互联网的人 —— 监控互联网使用并发现在线身份。
  </p>
  <p class="graf graf--p">
   KARMA POLICE 的描述是，“旨在将每个被动[信号情报]可见的用户与他们访问的每个网站相关联，从而提供（a）互联网上每个可见用户的网络浏览资料；或（b）互联网上每个可见网站的用户个人资料。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    情报共享 ——
   </strong>
  </p>
  <p class="graf graf--p">
   GCHQ 会将截获的信息或分析结果分享给其他机构，如军情五处、军情六处、和国家犯罪局、还有政府。
  </p>
  <p class="graf graf--p">
   英国获得美国的大规模拦截 ——
  </p>
  <p class="graf graf--p">
   英国政府还可以通过各种方式获取美国政府批量拦截的信息。
  </p>
  <p class="graf graf--p">
   英国可以
   <strong class="markup--strong markup--p-strong">
    直接和不受限制地访问原始拦截材料
   </strong>
   ，然后可以提取、过滤、存储、分析和/或传播。还可以访问美国政府存储在数据库中的信息，然后可以分析和/或传播。或者访问美国政府的分析结果，例如情报报告，然后可以检查和/或传播。
  </p>
  <p class="graf graf--p">
   斯诺登文件披露了美国大规模监控的惊人范围。
  </p>
  <p class="graf graf--p">
   最早的揭露涉及一项名为 Upstream 的计划，同样利用了路经美国的海底光纤电缆的地理便利。
  </p>
  <p class="graf graf--p">
   同时，硅谷科技巨头的高度集中化意味着世界上许多通信 —— Gmail通信、Whatsapp文本、Facebook帖子等等 —— 都将在传输过程中到达美国的服务器。
  </p>
  <p class="graf graf--p">
   美国政府还在全球范围内进行全面的大规模拦截计划。RAMPART-A 就是与外国合作伙伴共同运营的全球范围大规模拦截计划。
  </p>
  <p class="graf graf--p">
   被曝光的NSA文件表明，RAMPART-A 可以拦截“全球范围内每秒超过3太比特的数据流，并涵盖所有通信技术，如语音、传真、电传、调制解调器、电子邮件、互联网聊天、虚拟专用网络（VPN） 、VoIP、和语音通话记录。”
  </p>
  <p class="graf graf--p">
   MUSCULAR 是一个与 GCHQ 共同运作的计划，
   <strong class="markup--strong markup--p-strong">
    该计划在谷歌和雅虎位于世界各地的私人数据中心之间直接拦截和提取数据。
   </strong>
  </p>
  <p class="graf graf--p">
   根据泄露的2013年文件，在一个为期30天的时间内，国家安全局传送了超过
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/world/national-security/nsa-infiltrates-links-to-yahoo-google-data-centers-worldwide-snowden-documents-say/2013/10/30/e51d661e-4166-11e3-8b74-d89d714ca4dd_story.html" href="https://www.washingtonpost.com/world/national-security/nsa-infiltrates-links-to-yahoo-google-data-centers-worldwide-snowden-documents-say/2013/10/30/e51d661e-4166-11e3-8b74-d89d714ca4dd_story.html" rel="noopener noreferrer" target="_blank">
    1.81亿条
   </a>
   记录 —— 包括内容和元数据 —— 返回到位于马里兰州米德堡总部的数据库。
  </p>
  <p class="graf graf--p">
   美国和英国政府长期以来一直安排彼此分享情报，以及澳大利亚、加拿大和新西兰 —— 也就是所谓的“五眼联盟” ，他们分享各种情报，特别是信号情报。
  </p>
  <p class="graf graf--p">
   1946年开始美国和英国政府就签署了UKUSA协议，这是一项战后“通信情报”共享协议，后来才扩展到包括其他三个五眼联盟成员国。
  </p>
  <p class="graf graf--p">
   UKUSA 协议的第3部分规定，“各方同意交换某些”与外国通信有关的业务“的产品”，包括“（1）流量收集; （2）获取通信文件和设备; （3）流量分析; （4）密码分析; （5）解密和翻译”。
  </p>
  <p class="graf graf--p">
   在某些情况下，NSA和GCHQ联合运行大规模拦截程序，如上面的MUSCULAR示例所示。但斯诺登文件显示，英国政府也可以获取通过各种美国大规模拦截计划截获的信息。
  </p>
  <p class="graf graf--p">
   英国政府也可以访问 Marina，这是NSA的元数据存储库。Marina 有能力回顾[信号情报]收集系统看到的最后365天的元数据，无论它们是否被赋予任务采集。

    之后呢？
   </strong>
  </p>
  <p class="graf graf--p">
   不止斯诺登，在过去十年或更长时间里，有很多举报人和记者一直在揭露美国、英国和全球其他政府的大规模拦截能力。
  </p>
  <p class="graf graf--p">
   2013年的斯诺登揭露仍然是历史上的关键时刻，这些指控得到了数千份文件的支持，这些文件以极其详细的方式展示了世界上一些最强大的国家的大规模监视计划 —— 其能力、功能和过程。
  </p>
  <p class="graf graf--p">
   但是，即使在斯诺登披露之后，政府仍然拒绝承认这些计划的存在，这可能会使全世界每个人都继续被置于国家监视的显微镜下。
  </p>
  <p class="graf graf--p">
   有无数民间反抗者加入了抵制运动。
  </p>
  <p class="graf graf--p">
   尽管南非情报机构相信大规模监控是一种普遍被接受的做法 —— 他们多次重复这点 —— 两国政府承认使用其大规模监视能力的情况表明，他们并不习惯于谈论这些做法。
  </p>
  <p class="graf graf--p">
   他们的承认被埋没在法律诉讼中。联合国人权委员会已经对英国和南非的大规模监视表示了担忧。

   英国
  </p>
  <p class="graf graf--p">
   在英国，该案件始于2013年，就在 Snowden 披露后不久。这项大规模拦截称为“Tempora”计划，政府坚持拒绝承认其存在。
  </p>
  <p class="graf graf--p">
   隐私国际和其他九个非政府组织在斯诺登披露后不久对这种做法的合法性提出了质疑。该案件已经通过国家诉讼程序，现已再次出现在欧洲最高人权法院 — 欧洲人权法院。
  </p>
  <p class="graf graf--p">
   法律投诉最初于2013年7月提交给英国调查权力法庭（IPT）。IPT认定英国大规模拦截制度是非法的，因为管理此类监控的法律框架是秘密的。
  </p>
  <p class="graf graf--p">
   欧洲法院认为英国的大规模拦截违反了“欧洲人权公约”第8条和第10条，这两条分别保护了隐私权和言论自由权。

   </p>
   <p class="graf graf--p">
    那是英国政府首次相对详细地描述了这项大规模监视计划，收集、过滤、“选择性审查”和监视批量截获的通信的过程。他们描述了“简单”和“复杂”查询之间的区别、使用的选择器、以及它们如何操作：

    <li class="graf graf--li">
     《
     <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/ibm%e5%92%8c%e8%b0%b7%e6%ad%8c%e5%a6%82%e4%bd%95%e9%80%9a%e8%bf%87%e4%b8%ad%e5%9b%bd%e5%8d%8f%e5%8a%a9%e7%8b%ac%e8%a3%81%e6%94%bf%e6%9d%83%e7%9a%84%e9%97%b4%e8%b0%8d%e8%a1%8c%e4%b8%ba%ef%bc%9f/" href="https://www.iyouport.org/ibm%e5%92%8c%e8%b0%b7%e6%ad%8c%e5%a6%82%e4%bd%95%e9%80%9a%e8%bf%87%e4%b8%ad%e5%9b%bd%e5%8d%8f%e5%8a%a9%e7%8b%ac%e8%a3%81%e6%94%bf%e6%9d%83%e7%9a%84%e9%97%b4%e8%b0%8d%e8%a1%8c%e4%b8%ba%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
      IBM和谷歌如何通过中国协助独裁政权的间谍行为？
     </a>
     》
    </li>
   </ul>
   <p class="graf graf--p">
    就如你所知道的，已曝光的一切仍没有被停止。
   </p>
   <p class="graf graf--p">
    揭露铺天盖地，人们甚至几乎不再为此“兴奋”。
   </p>
   <p class="graf graf--p">
    没有公开的辩论，没有公民的声音和行动 —— 是否允许当权者继续肆无忌惮地抽干所有人的自由和人权；如果不，如何阻止他们。
   </p>
   <p class="graf graf--p">
    曝光无法解决问题，透明度革命的成功与否根本在于
    <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%b6%85%e8%b6%8a%e9%80%8f%e6%98%8e%e5%ba%a6%e9%9d%a9%e5%91%bd/" href="https://www.iyouport.org/%e8%b6%85%e8%b6%8a%e9%80%8f%e6%98%8e%e5%ba%a6%e9%9d%a9%e5%91%bd/" rel="noopener noreferrer" target="_blank">
     大众的行动
    </a>
    。只有大众利用其绝对的庞大基数之优势去推翻统治机器，才能将人类转向健康的循环。⚪️
   </p>
