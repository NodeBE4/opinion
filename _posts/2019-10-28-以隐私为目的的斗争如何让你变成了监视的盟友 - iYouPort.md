 <div class="entry-content">
  <ul>
   <li class="graf graf--p">
    <span style="color: #00ccff;">
     <em>
      <strong>
       贿赂永远意味着，行贿者付出的东西需要受贿者以数倍甚至数百倍的代价来回报。现在我们所有人都是受贿者；我们已经为“回报”掏空了自己的一切……
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   上个月的内容中我们声明过这个问题，关于为什么必需认识到我们所有人是1984的同谋，详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/6%e5%b9%b4%ef%bc%8c%e6%94%b9%e5%8f%98%e4%ba%86%e4%bb%80%e4%b9%88%ef%bc%9f/" href="https://www.iyouport.org/6%e5%b9%b4%ef%bc%8c%e6%94%b9%e5%8f%98%e4%ba%86%e4%bb%80%e4%b9%88%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    6年，改变了什么？
   </a>
   》。
  </p>
  <p class="graf graf--p">
   但是下面这篇文章更深一步，它强调了以隐私为“终极目标”的斗争为什么注定失败。最重要的是，如何扭转这一局面。
  </p>
  <p class="graf graf--p">
   这篇文章令我们所有这些提供隐私保护技术和知识的人感到无比惭愧 —— 我们只是在被动斗争，只是在努力保护自己和盟友的安全，只是在等待他人的革命让我们摆脱灾难；我们没有真正站起来反击 —— 我们还在敌人的地盘上扑腾。
  </p>
  <p class="graf graf--p">
   这样是不会赢的。即便高超的防卫技术令队友可以存活下来；但是不会赢。
  </p>
  <p class="graf graf--p">
   要想赢，我们必须占据主场作战 —— 就如 Jathan Sadowski 所强调的：成为战斗者的增援。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6248 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/智能锁.png?resize=1100%2C725&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/智能锁.png?w=1398&amp;ssl=1 1398w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/智能锁.png?resize=300%2C198&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/智能锁.png?resize=768%2C507&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/智能锁.png?resize=1024%2C675&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/智能锁.png?resize=1100%2C725&amp;ssl=1 1100w" data-recalc-dims="1" height="725" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/智能锁.png?resize=1100%2C725&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   一个典型的案例提醒人们如何为特定的权利而战。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnet.com/news/tenants-win-rights-to-physical-keys-over-smart-locks-from-landlords/" href="https://www.cnet.com/news/tenants-win-rights-to-physical-keys-over-smart-locks-from-landlords/" rel="noopener noreferrer" target="_blank">
    法官命令
   </a>
   纽约的房东必须“向任何不想使用建筑物上安装的 Latch 智能锁的租户提供实物钥匙”。
  </p>
  <p class="graf graf--p">
   这项法律裁决是在曼哈顿一栋公寓楼的居民们联合起来并
   <a class="markup--anchor markup--p-anchor" data-href="https://onezero.medium.com/americas-favorite-door-locking-app-has-a-data-privacy-problem-f19169a8ab2e" href="https://onezero.medium.com/americas-favorite-door-locking-app-has-a-data-privacy-problem-f19169a8ab2e" rel="noopener noreferrer" target="_blank">
    成功起诉房东
   </a>
   之后做出的 —— 因为他们的房东用智能锁更换了大堂的实物锁。
  </p>
  <p class="graf graf--p">
   Latch 这类使用智能手机打开的类似无钥匙锁
   <strong class="markup--strong markup--p-strong">
    已经是美国各地的公寓楼越来越常见的特征。
   </strong>
  </p>
  <p class="graf graf--p">
   在反对这种强制升级的过程中，租户列出了为什么智能锁实际上没有提供他们的房东声称的“增强的便利性和安全性”的无数理由。
  </p>
  <p class="graf graf--p">
   租户提出的问题不仅包括一些老年居民根本没有智能手机，而且，更重要的是，
   <strong class="markup--strong markup--p-strong">
    围绕房东监控和骚扰租户的严重问题。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我说我不想被跟踪，房东笑了”，一名72岁的大楼居民
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnet.com/news/tenants-worry-smart-home-tech-could-be-abused-by-landlords/" href="https://www.cnet.com/news/tenants-worry-smart-home-tech-could-be-abused-by-landlords/" rel="noopener noreferrer" target="_blank">
    告诉 CNET
   </a>
   。
  </p>
  <p class="graf graf--p">
   我们都知道智能技术 —— 数据驱动、互联网连接、自动化 —— 充斥着隐私问题。有无数的例子，名单每天都在增长。
  </p>
  <p class="graf graf--p">
   智能锁带来了人权关注者最担心的问题。
  </p>
  <p class="graf graf--p">
   这种恐惧已经变为现实 —— 包括收集个人数据，然后用于（秘密）创建关于我们的身份、偏好、行为和惯例的
   <a class="markup--anchor markup--p-anchor" data-href="https://onezero.medium.com/americas-favorite-door-locking-app-has-a-data-privacy-problem-f19169a8ab2e" href="https://onezero.medium.com/americas-favorite-door-locking-app-has-a-data-privacy-problem-f19169a8ab2e" rel="noopener noreferrer" target="_blank">
    详细资料
   </a>
   。
  </p>
  <p class="graf graf--p">
   除了推动建立在“个性化”目标基础上的监视资本主义经济体制，并且，随着这种程度的跟踪加深，几乎使我们完全丧失了免受政府、公司和老板的严苛监控的能力。
  </p>
  <p class="graf graf--p">
   我们对所谓的智能玩意的怀疑往往始于隐私，这并不奇怪。
  </p>
  <p class="graf graf--p">
   然而，
   <strong class="markup--strong markup--p-strong">
    问题在于它通常也以隐私结束。⚠️这种框架限制了我们对正在发生的事情和利害攸关的事物的理解。
   </strong>
  </p>
  <p class="graf graf--p">
   这是错误的斗争模式。
  </p>
  <p class="graf graf--p">
   相反，当我们谈论技术时，我们应该首先考虑权力机制。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    有很多理由可以解释为什么科技公司很乐意讨论隐私问题
   </strong>
   ，他们资助以隐私为主题的奢侈会议、向倡导者提出一些观点、甚至创建像“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.palantir.com/pcl/" href="https://www.palantir.com/pcl/" rel="noopener noreferrer" target="_blank">
    隐私和公民自由工程
   </a>
   ”​​这样的新职业。
  </p>
  <p class="graf graf--p">
   这是因为：
   <strong class="markup--strong markup--p-strong">
    ⚠️这种以隐私为目的框架几乎没有什么可以挑战实际的政治经济体系的东西 —— 基于提取数据和扩大控制的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/?s=%E7%9B%91%E8%A7%86%E8%B5%84%E6%9C%AC%E4%B8%BB%E4%B9%89" href="https://www.iyouport.org/?s=%E7%9B%91%E8%A7%86%E8%B5%84%E6%9C%AC%E4%B8%BB%E4%B9%89" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     监视资本主义经济体系
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    —— 就是这些寡头建立的它，并且一直在从中受益。
   </strong>
  </p>
  <p class="graf graf--p">
   社会学家 Sami Coll 认为：
   <strong class="markup--strong markup--p-strong">
    隐私的概念现在已经变得更像是“
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.tandfonline.com/doi/full/10.1080/1369118X.2014.918636" href="https://www.tandfonline.com/doi/full/10.1080/1369118X.2014.918636" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     监视的盟友
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ”，而不是它的解毒剂。
   </strong>
  </p>
  <p class="graf graf--p">
   因为如今隐私已经由个人主义思想和个人解决方案来定义了，这些思想和解决方案的核心是负责实施良好网络卫生、负责任的数字消费者。
  </p>
  <p class="graf graf--p">
   但是，⚠️一个必须保护自己的勇敢的人、和一堆随时随地想要更深刻地窥探和挖掘所有人内心的强大机构之间的斗争，其输赢几乎没有多少悬念。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当我们接受一个对监视工业综合体友好 — 或者甚至已经被其吸收和挪用的框架时，我们最终只能按照*他们的*规则在*他们的*地盘上作战。
   </strong>
  </p>
  <p class="graf graf--p">
   我们可能会赢得一些小冲突，但是，最终我们将输掉整个斗争。
  </p>
  <p class="graf graf--p">
   因此，当租金稳定的公寓中的租户们试图阻止房东安装面部识别门锁时，他们不仅仅是“引发了关于隐私和监视的辩论”。
  </p>
  <p class="graf graf--p">
   不仅仅是人们想要躲避监视想要获得一分安全和宁静。
  </p>
  <p class="graf graf--p">
   正如房东所描述的那样，这场战斗也
   <strong class="markup--strong markup--p-strong">
    不是隐私与安全之间的选择
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这绝对不是一种下意识的恐慌反应，正如反监管亲企业智库梅卡图斯中心所说的那样。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    他们不仅仅是技术反叛者。他们正在数字资本主义的前线作战 —— 他们需要增援。
   </strong>
  </p>
  <p class="graf graf--p">
   最终，这个问题与权力有关：谁有权力、他们如何运用权力、他们获得了什么？反之亦然：谁被排除在外、谁被针对、被挤压？
  </p>
  <p class="graf graf--p">
   智能锁和面部识别的例子非常重要，突出了真正的利害关系 —— 因为它是一个明确的案例：一个不那么强大的团体（租户）抵制一个更强大的团体（房东）企图武装自己的危险武器库。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    租户甚至没有试图提升他们的地位 —— 他们只是试图不失势。
   </strong>
  </p>
  <p class="graf graf--p">
   与我们意识到的相比，这些权力机制更为常见，对新技术的设计和使用产生了更大的影响。
  </p>
  <p class="graf graf--p">
   ⚠️技术实现了权力，使其成为坚固耐用的形式。
  </p>
  <p class="graf graf--p">
   最重要的不是技术本身，而是技术背后的人、价值观和组织，以及通过技术引导其利益的方式。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    技术不是中立的、也不是不可避免的、或完全可塑的。
   </strong>
  </p>
  <p class="graf graf--p">
   隐藏在每项技术中的是一堆人类选择 —— 应该解决哪些问题、应该如何花费资源、应该进行权衡取舍、如何表达观点、应该使用何种技术以及其他许多可以解决的决策做 X，而不是Y或Z，才是关键。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    今天包围我们的大部分智能技术都源于一种利润和权力体系，这种体系通过历史学家和评论家 Lewis Mumford 在1970年所确定的“
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.boundary2.org/2018/07/loeb/" href="https://www.boundary2.org/2018/07/loeb/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     超级技术贿赂
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ”来使自己合法化。
   </strong>
  </p>
  <p class="graf graf--p">
   贿赂永远意味着，行贿者付出的东西需要受贿者以数倍甚至数百倍的代价来回报。
  </p>
  <p class="graf graf--p">
   这种贿赂的条件很简单：为了以最小的麻烦接受系统产生的东西，一些人被允许获得适度的便利和舒适的奢侈品，而许多其他人则受到更高程度的剥削、提取和排斥。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    一小部分的好处建立在对绝大部分的伤害之上。
   </strong>
  </p>
  <p class="graf graf--p">
   换句话说，我们的注意力会受到酷炫小工具的拥有*感*和定期升级的吸引，而监视资本主义的增压系统则融入了整个社会。
  </p>
  <p class="graf graf--p">
   我们都没有明确同意这笔交易，但就像软件许可或服务条款一样，我们别无选择 —— 只能“接受”。
  </p>
  <p class="graf graf--p">
   这种贿赂的例子随处可见。就如保险公司提供折扣设备和其他激励措施，是作为从智能家居和日常生活中收集大量有价值的个人数据的一种方式；
  </p>
  <p class="graf graf--p">
   更如，当科技公司出售优化城市系统和证券化城市空间的解决方案时，它们完全接管了城市运营、全面监视和所有权。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种贿赂之所以成功的基础源于我们所有人对一些平台的依赖
   </strong>
   —— Uber 的出游、DoorDash 的交付、和 Airbnb 的房间。
  </p>
  <p class="graf graf--p">
   它们看起来是如此的便宜和方便，我们这些人便就此忽略了那些背后的剥削问题、残酷的扩张、价值提取，以及使这些服务
   <a class="markup--anchor markup--p-anchor" data-href="https://onezero.medium.com/landlord-2-0-techs-new-rentier-capitalism-a0bfe491b463" href="https://onezero.medium.com/landlord-2-0-techs-new-rentier-capitalism-a0bfe491b463" rel="noopener noreferrer" target="_blank">
    成为可能的
   </a>
   数十亿风险投资。
  </p>
  <p class="graf graf--p">
   贿赂是一种腐败形式。随着社会根据数字资本主义的需求进行升级，利润和权力的必要性已经严重破坏了这些智能技术的源头。
  </p>
  <p class="graf graf--p">
   我们可能没有以任何有意义的方式接受贿赂，但尽管如此，它看起来依旧像是一种魅力。
  </p>
  <p class="graf graf--p">
   正如 Virginia Eubanks 在她的书 “Automating Inequality” 中所展示的那样，资本主义经常测试其最新的方法来提取、利用和排斥社会中最无力的、最边缘化的群体 —— 那些甚至没有得到贿赂的人 —— 在针对大多数人之前。
  </p>
  <p class="graf graf--p">
   就如那位每一步都在受到监视和操纵的女性对 Eubanks 所说的话：
   <strong class="markup--strong markup--p-strong">
    “你应该注意到发生在我们身上的事。你是下一个”。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️除非我们追随那些拒绝智能锁的租户及其实现的权力机制的脚步；理解到他们不仅仅是技术反叛者。他们正在数字资本主义的前线作战 ——
   <strong class="markup--strong markup--p-strong">
    他们需要增援
   </strong>
   。⚪️
  </p>
  <p class="graf graf--p">
   这里是一份论文《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.tandfonline.com/doi/full/10.1080/1369118X.2014.918636?scroll=top&amp;needAccess=true" href="https://www.tandfonline.com/doi/full/10.1080/1369118X.2014.918636?scroll=top&amp;needAccess=true" rel="noopener noreferrer" target="_blank">
    为什么应该将隐私理解为“监视的盟友”
   </a>
   》；并且本文的思考方式并非首次呈现，另见《
   <a href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e7%a9%b6%e7%ab%9f%e6%98%af%e4%bb%80%e4%b9%88%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    互联网究竟是什么？ — — 来自“监视谷”惊人的启示
   </a>
   》
  </p>
