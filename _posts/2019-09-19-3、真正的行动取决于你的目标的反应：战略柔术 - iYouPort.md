   <p class="graf graf--p graf--startsWithDoubleQuote">
    <em>
     “真正的行动在于敌人的反应。” —— Saul Alinsky
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p">
   我们在中国最经常看到的是诉苦、维权和找律师（因为行动者被捕），而不是借势将行动推进。似乎哪里出了问题。
  </p>
  <p class="graf graf--p">
   当然，人们会说“你能不关心受难者？” 当然不是。但关心受难者最好的方式是迅速接力以达成他们的目标，而不是痛哭流涕。
  </p>
  <p class="graf graf--p">
   不得不说中国学生协助的劳工维权行动是非常成功的，他们直接点中了执政党的要害。但是，非常可惜，在执政党做出了最愚蠢的反应时 —— 也就是已经快要赢了的时候，其他活动家陷入了沉默。没有任何人跟进追打。
  </p>
  <p class="graf graf--p">
   他们错失了一个大好机会。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果您认为被捕是结束，那就真的错了。在反抗运动的世界里，死亡都不是结束。
   </strong>
  </p>
  <p class="graf graf--p">
   💡在面对一个众所周知的大目标时，通常目标对你的行动的反应是你成功的关键。因此，你要做的是，预测目标的反应并将其写入你的脚本。
  </p>
  <p class="graf graf--p">
   简曰：是你在玩他们，而不是让他们玩你。
  </p>
  <p class="graf graf--p">
   在1930年的 Salt March “食盐进军”期间，印度的独立活动家们就是一直在走路 —— 长途跋涉，没有武装，沿途未受阻碍，直到陷入英国警察的野蛮打击之中。
  </p>
  <p class="graf graf--p">
   约2500个和平抗议者被当局攻击。直到1930年年底，约有6万人被捕入狱。
  </p>
  <p class="graf graf--p">
   结束了吗？并没有。
  </p>
  <p class="graf graf--p">
   重要的是，效应是鲜明的 ——
   <strong class="markup--strong markup--p-strong">
    积极人士们确保了这一点
   </strong>
   —— 记录了殖民政府如何以可预见的卑鄙耍流氓。
  </p>
  <p class="graf graf--p">
   印度国家先团结起来了。随后是世界舆论的凝结。
  </p>
  <p class="graf graf--p">
   组织者 Saul Alinsky 后来创造了“政治柔术”一词语来描述这样的抗议行动。
  </p>
  <p class="graf graf--p">
   ⚠️柔术的特色是重视地板扭斗，以杠杆原理控制对手的关节，来制服对手。即使是身材矮小的人，也可以透过这些技巧来打败对手。
  </p>
  <p class="graf graf--p">
   柔术练习者擅长将对手拖向地面，然后在地面上获得控制的姿势。
  </p>
  <p class="graf graf--p">
   柔术的技巧主要是位置控制以及各种降服技巧。
  </p>
  <p class="graf graf--p">
   💡像食盐进军这样的对抗使得资源不足的激进组织能够
   <strong class="markup--strong markup--p-strong">
    通过激发目标权势的反应来利用其强大的对手将其转变为自己的动力，然后在镜头前看着他们不论在字面意义上还是在隐喻层面上都被击得粉碎
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4461 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/食盐进军.png?resize=1100%2C758&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/食盐进军.png?w=1440&amp;ssl=1 1440w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/食盐进军.png?resize=300%2C207&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/食盐进军.png?resize=768%2C529&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/食盐进军.png?resize=1024%2C705&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/食盐进军.png?resize=1100%2C758&amp;ssl=1 1100w" data-recalc-dims="1" height="758" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/食盐进军.png?resize=1100%2C758&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4461" data-recalc-dims="1" height="758" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/食盐进军.png?resize=1100%2C758&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/食盐进军.png?w=1440&amp;ssl=1 1440w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/食盐进军.png?resize=300%2C207&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/食盐进军.png?resize=768%2C529&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/食盐进军.png?resize=1024%2C705&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/食盐进军.png?resize=1100%2C758&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   ⚠️在应用这一原则时，重要的是要明白，你不能仅仅希望目标以一种突出不公正的方式作出反应。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在可能的情况下，你需要事先计划目标的反应，鼓励他们的反应，并将其拖入你自己的行动
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果第一次不起作用，那么就调整计划，再来一次。
  </p>
  <p class="graf graf--p">
   💡
   <strong class="markup--strong markup--p-strong">
    确保从你的对手那里获得战略性有用反应的一个好方法就是：强迫他们进入“决策困境”
   </strong>
   —— 在这看到具体解释《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E5%A6%82%E6%9E%9C%E6%8A%97%E8%AE%AE%E6%98%AF%E9%9D%9E%E6%B3%95%E7%9A%84%EF%BC%8C%E9%82%A3%E5%B0%B1%E8%AE%A9%E6%97%A5%E5%B8%B8%E7%94%9F%E6%B4%BB%E6%88%90%E4%B8%BA%E6%8A%97%E8%AE%AE%EF%BC%88%E4%B8%80/" href="https://www.iyouport.org/%E5%A6%82%E6%9E%9C%E6%8A%97%E8%AE%AE%E6%98%AF%E9%9D%9E%E6%B3%95%E7%9A%84%EF%BC%8C%E9%82%A3%E5%B0%B1%E8%AE%A9%E6%97%A5%E5%B8%B8%E7%94%9F%E6%B4%BB%E6%88%90%E4%B8%BA%E6%8A%97%E8%AE%AE%EF%BC%88%E4%B8%80/" rel="noopener noreferrer" target="_blank">
    策略1.1 为你的对手制造决策困境
   </a>
   》—— 所有可用的选项都可以发挥作用。
  </p>
  <p class="graf graf--p">
   冒充陶氏化学发言人的 “Yes Men” 在英国广播公司电视台宣布：陶氏正在为博帕尔灾难道歉并拨出120亿美元赔偿受害者。这是一场经典的大戏，最终，道指股价暴跌，陶氏被迫做出反应。
  </p>
  <p class="graf graf--p">
   陶氏不得不发表一份声明，表示他们并没有为博帕尔灾难道歉，也不会赔偿受害者。一个巨大的打脸爆笑。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    你的对手的反应 —— 这就是最好的胜利。在这里，高明的行动策划再一次让目标变得无比失败。
   </strong>
  </p>
  <p class="graf graf--p">
   💡正如陶氏化学的案例所说明的那样，有些时候你根本不需要进行直接的身体对抗，就可以制造绝佳的大戏。
  </p>
  <p class="graf graf--p">
   2000年，布什竞选团起诉反对派活动家 Zack Exley，并试图关闭了他的恶作剧网站 GWBush.com（Exley 在竞选活动开始之前设法购买了这个域名）。
  </p>
  <p class="graf graf--p">
   显然这件事特别有吸睛度。媒体纷纷报道了这个故事，以及每一次新出现的法律攻击 ——布什曾一度说过“应该限制自由”，高端打脸。
  </p>
  <p class="graf graf--p">
   一个令人遗憾的提醒是，即使是最聪明的一次性战术，也不是一次成功的战役。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    流行的看法是“压制和镇压是失败和挫折”，不，恰恰相反，当权势威胁到你时，你应该庆祝
   </strong>
   ；这意味着他们做出了反应。这一反应越是激烈、愚蠢和疯狂，你的胜算就越大。
  </p>
  <p class="graf graf--p">
   这意味着你占据了上风。
  </p>
  <p class="graf graf--p">
   抓住你的对手最激烈和最愚蠢的对白，将它们编织成新闻稿，以全力宣传它们 —— 这样你就把你的对手投入到你的游戏中了。
  </p>
  <p class="graf graf--p">
   回看下，中国的抗议行动者错过了多少机会？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这就是柔术的逻辑。你借用了他们的一些力量，就如同抓住了他们的小辫子，顺势努力抽打他们的脸，让他们动弹不得
   </strong>
   。
  </p>
  <p class="graf graf--p">
   当歌利亚摔了个狗啃泥时，每个人都会跑来看热闹的。⚪️
  </p>
