   <h1 class="entry-title" itemprop="headline">
    風險觀察：新冷戰中求生 中共或會製造「珍珠港事件」
   </h1>
   <div class="entry-meta">
    <span class="posted-on">
     <a href="https://tw.sinoinsider.com/2019/06/%e6%96%b0%e5%86%b7%e6%88%b0%e4%b8%ad%e6%b1%82%e7%94%9f-%e4%b8%ad%e5%85%b1%e6%88%96%e6%9c%83%e8%a3%bd%e9%80%a0%e3%80%8c%e7%8f%8d%e7%8f%a0%e6%b8%af%e4%ba%8b/" rel="bookmark" title="20:31">

  <div class="entry-content" itemprop="text">
   <p>
    ◎中共可能使用的常規或「超限戰」策略和戰術來對付美國。
   </p>
   <p>
    <span id="more-2787">
    </span>
   </p>
   <hr/>
   <p>
    目前中共正面臨一系列前所未有的危機：
   </p>
   <ul>
    <li>
     習近平陣營與江派之間「你死我活」的
     <a href="https://tw.sinoinsider.com/politics/">
      派系鬥爭
     </a>
     不斷升級;
    </li>
    <li>
     迅速惡化的經濟；
    </li>
    <li>
     腐敗、疫情（豬瘟、禽流感等）和蟲害（秋季粘蟲）造成糧食危機;
    </li>
    <li>
     中國各地的社會動盪加劇。
    </li>
   </ul>
   <p>
    中美貿易戰和美國對中共政權的強硬立場加劇了中共的危機。
   </p>
   <ul>
    <li>
     今年5月，美國總統唐納德·川普宣佈計劃對所有中國商品徵收關稅（5月份對價值2000億美元的中國產品提高關稅;6月份對另外價值3000億美元的中國商品加徵25％的關稅）;
    </li>
    <li>
     美國商務部宣佈對
     <a href="https://tw.sinoinsider.com/2019/05/%e5%b0%81%e6%ae%ba%e8%8f%af%e7%82%ba%e8%83%bd%e9%80%bc%e4%b8%ad%e5%9c%8b%e5%9b%9e%e8%ab%87%e5%88%a4%e6%a1%8c%e5%97%8e%ef%bc%9f/">
      華為
     </a>
     實施禁令，這一舉措將對華為帶來毀滅性打擊;
    </li>
    <li>
     美國似乎正在對其它中國科技公司進行封鎖，特別是那些在大陸侵犯人權的公司;
    </li>
    <li>
     川普政府正在打
     <a href="https://tw.sinoinsider.com/2019/06/%e7%be%8e%e5%9c%8b%e5%b0%8d%e8%8f%af%e6%94%bf%e7%ad%96%e5%8d%87%e7%b4%9a%e5%8d%b1%e5%8f%8a%e4%b8%ad%e5%85%b1/">
      人權牌和台灣牌
     </a>
     ；
    </li>
    <li>
     美國正在和盟國加強在南海的軍事活動;
    </li>
    <li>
     美國正在加強印度-太平洋地區的聯盟和夥伴關係，以對抗中共政權。
    </li>
   </ul>
   <p>
    根據我們對中共的研究，我們認為，在派系鬥爭、經濟危機、糧食短缺、社會動盪和美國強力施壓這些因素的疊加下，中共及其政權無法長期生存。由於攸關生死，中共不會坐以待斃，會盡一切可能，不擇手段，求得生存。
   </p>
   <p>
    如果有足夠的時間，中共可以解決任何內部問題。因此，它將優先解決外部問題，以換取時間處理內部困境。美國和川普政府是中共外部威脅的首選。隨著6月底美國提高關稅的到來，我們認為中共將會盡快部署對策。
   </p>
   <p>
    下面我們列舉一些中共可能使用的常規或「超限戰」策略和戰術，製造類似「珍珠港」事件，圖謀在「新冷戰」中生存下來。
   </p>
   <p>
    <span style="color: #333399;">
     1.中共可能試圖破壞美國金融市場
    </span>
   </p>
   <ul>
    <li>
     中共宣傳最近開始對中美股市進行比較。其宣傳聲稱，美國市場存在較高風險，而中國市場在專制統治下更具「韌性」。
    </li>
    <li>
     資深中國官員和學者最近一直在討論貿易戰如何引發「全球金融危機」以及美國金融市場的問題
     <a href="#001">
      [1]
     </a>
     。
    </li>
    <li>
     中共可能利用其「
     <a href="https://tw.sinoinsider.com/2019/12/%e4%b8%96%e7%95%8c%e8%a2%ab%e4%b8%ad%e5%85%b1%e3%80%8c%e7%b4%85%e8%89%b2%e9%a7%ad%e5%ae%a2%e5%b8%9d%e5%9c%8b%e3%80%8d%e5%9c%8d%e5%9b%b0/">
      紅色駭客帝國
     </a>
     」來削弱投資者對美國市場和經濟的期望和信心。
    </li>
    <li>
     在海外上市的中國公司和中國主權財富基金，可能會在6月份美國加徵關稅的時間點上，製造股市恐慌。華爾街的親中派甚至可能與中共合作，從金融危機中獲利。
    </li>
    <li>
     如果能夠成功引發美國股市崩盤，中共可能推出一種跟黃金掛勾的貨幣來取代美元，從而贏得與美國的「貨幣戰爭」。
    </li>
   </ul>
   <p>
    <span style="color: #333399;">
     2.中共可能設法影響和干預美國的政治和社會
    </span>
   </p>
   <ul>
    <li>
     中共會動用一切資源干預美國內政，並將矛頭指向川普政府，分散川普在中國問題上的注意力，並影響2020年美國總統大選。
    </li>
    <li>
     為了破壞川普政府，中共可能設法影響兩個主要政黨中的政治家，給川普的執政或政策造成障礙。例如，中共希望川普的貿易和關稅權限受到限制，啟動彈劾程序，並看到川普的國內舉措（基礎設施、醫療保健等）在下屆大選前無法實施。
    </li>
    <li>
     為了轉移川普對中國的注意力，中共可能利用其「紅色駭客帝國」來指責川普政府實行「種族主義」，並試圖挑起「文明衝突」，同時強調其他「壞蛋」國家（俄羅斯、伊朗等）的威脅。
    </li>
    <li>
     為了影響2020年美國總統大選，中共可能設法支持其他候選人，同時製造或支持反川普的宣傳。
    </li>
    <li>
     為了分裂美國社會，中共可能在「
     <a href="https://tw.sinoinsider.com/2019/05/%e7%82%ba%e4%bb%80%e9%ba%bc%e4%b8%ad%e5%85%b1%e6%87%bc%e6%80%95%e6%84%8f%e8%ad%98%e5%bd%a2%e6%85%8b%e6%88%b0%e5%92%8c%e3%80%8c%e6%96%87%e6%98%8e%e8%a1%9d%e7%aa%81%e3%80%8d/">
      文明衝突
     </a>
     」上做文章，指責川普政府實行「排外主義」。
    </li>
   </ul>
   <p>
    <span style="color: #333399;">
     3.中共可能製造外國危機，轉移美國對中國的注意力
    </span>
   </p>
   <ul>
    <li>
     中國可能加強與俄羅斯的關係，煽動俄羅斯與美國對抗。
    </li>
    <li>
     中國可能設法引發伊朗與美國之間的軍事對抗。
    </li>
    <li>
     在朝鮮無核化問題上，中國可能唆使朝鮮對美國強硬，包括加強導彈試驗。
    </li>
    <li>
     中國可能暗中支持委內瑞拉的馬杜羅政權，阻礙該國擺脫獨裁統治。
    </li>
   </ul>
   <p>
    <span style="color: #333399;">
     4.中共試圖通過破壞美國和其盟友的關係，以避免美國的遏制
    </span>
   </p>
   <ul>
    <li>
     中國可能會積極
     <a href="https://tw.sinoinsider.com/2019/04/%e6%ad%90%e6%b4%b2%e8%b8%8f%e9%80%b2%e4%b8%ad%e5%85%b1%e7%9a%84%e3%80%8c%e8%ae%93%e6%ad%a5%e9%99%b7%e9%98%b1%e3%80%8d/">
      拉攏歐洲國家
     </a>
     ，分化美國的歐洲盟友。
    </li>
    <li>
     中國可能試圖說服印度-太平洋國家「反對美國霸權」或在美中之間搞平衡。這將阻止印度-太平洋國家與美國合作以應對中共的威脅。
    </li>
   </ul>
   <p>
    <strong>
     <span style="color: #ff6600;">
      我們的分析：
     </span>
    </strong>
    <br/>
    1.受傷的動物是最危險的。我們列出的中共對美國可能採取的策略和計謀並非詳盡無遺，我們不排除中共還會部署其他更加邪惡和更加難以想像的對策。
   </p>
   <p>
    2.美國遏制中共的堅定立場對美國和世界都是一個積極因素。然而，美國的強硬措施將激發中共的激烈反應，大幅提升全球政治、經濟和軍事風險。雖然我們認為美國目前正走向「新冷戰」勝利，但如果只採用目前的戰略，可能也會付出沉重的代價。
   </p>
   <p>
    3.根據我們的研究，我們認為中國在貿易談判中的出爾反爾與中共派系鬥爭的升級直接相關。中共派系鬥爭目前的狀態，為美國大幅縮減贏得「新冷戰」的時間和代價，打開了一個重要的機會窗口。川普政府要抓住這個天賜良機，首先必須區分政權中的改革者和極左強硬派，並對後者施加戰略壓力。
   </p>
   <p>
    <strong>
     <span style="color: #ff6600;">
      更多透視：
     </span>
    </strong>
    <br/>
    「透視中國」有精准預測中國的記錄。我們認為，黑天鵝將在2019年下半年降臨中國。在中國面臨巨變時，「透視中國」可以幫助企業、投資者和政府規避風險、抓住機遇。
   </p>
   <p>
    美國政府要避免與中共進行漫長且代價高昂的對抗，可以採用創新的解決方案補充現有戰略，化解「文明衝突」問題。「透視中國」有這些解決方案。
   </p>
   <p>
    <strong>
     <span style="color: #333399;">
      備注：
     </span>
    </strong>
    <br/>
    <a name="001">
    </a>
    [1]5月16日，中國人民大學教授金燦榮在接受大陸媒體採訪時表示，美國的股市問題比中國的嚴重。金形容中國股市崩盤相當於一個人從二樓跳樓，美國股市崩盤相當於從五十樓跳樓，結果是完全不一樣的——二樓跳樓可能會扭傷腳踝，五十樓跳樓是必死無疑。金補充說，華爾街的股市占美國GDP的26%，而中國股票僅佔中國GDP的4％，而且美國企業80%的融資渠道是靠股市，中國只佔到10%。
   </p>
   <p>
    5月31日，原中國人民銀行行長戴相龍在中美經貿關係高端研討會上表示，中美貿易戰「如果升級，可能會引發全球金融危機」。
   </p>
   <p>
    6月2日，中央電視台新聞聯播用4分鐘時間採訪中國證監會主席易會滿，談有關中國和美國的股市。易會滿主要的觀點有：
    <br/>
    • 面對美國的政策攻擊，中國的A股有足夠的韌性。A股震盪為主，而美股則是明顯下行；
    <br/>
    • 從兩國股市自身情況出發，美股處於十年牛市的頂部，A股處於2015年熊市以來的底部；
    <br/>
    • 美國家庭資金配置中，股票佔了35%左右，中國家庭資金配置裡，股票只有1.4%；
    <br/>
    • 「如果這兩個國家的股市下跌，誰有更多的迴旋餘地，誰會受到更多傷害？這是不言而喻的！」
   </p>
