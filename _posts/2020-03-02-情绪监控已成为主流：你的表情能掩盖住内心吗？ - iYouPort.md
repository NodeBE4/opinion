   </strong>
  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   开发广告和电影预告片的人多年来一直使用
   <a class="markup--anchor markup--p-anchor" data-href="https://www.fastcompany.com/1731055/rise-neurocinema-how-hollywood-studios-harness-your-brainwaves-win-oscars" href="https://www.fastcompany.com/1731055/rise-neurocinema-how-hollywood-studios-harness-your-brainwaves-win-oscars" rel="noopener noreferrer" target="_blank">

   </a>
   ，来监视被试对象的情绪反应。
  </p>
  <p class="graf graf--p">
   但这是麻烦且昂贵的。
  </p>
  <p class="graf graf--p">

   <a class="markup--anchor markup--p-anchor" data-href="https://www.affectiva.com/product/affdex-for-market-research/" href="https://www.affectiva.com/product/affdex-for-market-research/" rel="noopener noreferrer" target="_blank">

  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   它们很快会插入到我们家庭、办公室、和街道中最常见的那些摄像机里。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    人工智能如何读懂你的脸
   </strong>
  </h3>
  <p class="graf graf--p">
   这一技术“发展”并非完全突然。
  </p>
  <p class="graf graf--p">
   自2012年以来，人工智能
   <a class="markup--anchor markup--p-anchor" data-href="http://www.wired.co.uk/article/mit-fake-smile-detection" href="https://www.wired.co.uk/article/mit-fake-smile-detection" rel="noopener noreferrer" target="_blank">
    已经能够
   </a>
   比人类更好地将假笑与真正的笑容区分开；到2015年，
   <strong class="markup--strong markup--p-strong">
    人工智能测谎仪
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.newscientist.com/article/mg22830462-700-lie-detecting-algorithm-spots-fibbing-faces-better-than-humans/" href="https://www.newscientist.com/article/mg22830462-700-lie-detecting-algorithm-spots-fibbing-faces-better-than-humans/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     已经学会
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    了能胜过最好的人类审讯者
   </strong>
   。
  </p>
  <p class="graf graf--p">
   2017年出现的最新技术不仅能够区分真实的微笑和假笑，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.extremetech.com/extreme/250725-depressed-suicidal-ai-algorithm-detects-whats-hidden-beneath-smile?source=deep-learning" href="https://www.extremetech.com/extreme/250725-depressed-suicidal-ai-algorithm-detects-whats-hidden-beneath-smile?source=deep-learning" rel="noopener noreferrer" target="_blank">
    还可以区分
   </a>
   轻度抑郁者和自杀意念者的假笑。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    即使您可以谨慎地保持表情中立，这些新的监视工具也可以分析脸部皮肤下的血液流动，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://startupheretoronto.com/sectors/technology/could-your-face-be-a-window-to-your-health-u-of-t-startup-gathers-vital-signs-from-video/" href="http://startupheretoronto.com/sectors/technology/could-your-face-be-a-window-to-your-health-u-of-t-startup-gathers-vital-signs-from-video/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     以捕获您的心率和血压
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，从而提供丰富的关于您的情绪、心理压力和疲劳感的情报
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这是您掩饰不了的东西。
  </p>
  <p class="graf graf--p">
   ⚠️所有这些监视系统都可以在廉价的现成网络摄像头的视频源上运行，甚至可以
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amazon.com/Screw-Head-Mini-Hidden-Camera/dp/B01493DMX0" href="https://www.amazon.com/Screw-Head-Mini-Hidden-Camera/dp/B01493DMX0" rel="noopener noreferrer" target="_blank">
    很小足够伪装成螺丝钉
   </a>
   。
  </p>
  <p class="graf graf--p">
   这些工具具有巨大的潜力，既可以带来社会福利，也可以带来社会危害。
  </p>
  <p class="graf graf--p">
   可以预测自杀念头或精神病发作的系统可以帮助社会上最脆弱的成员及时获得帮助。但是，同样的工具也可能在不合适的情况下引发推测性诊断。
  </p>
  <p class="graf graf--p">
   操作人员如果不够专业会产生逆反作用，比如故意歧视或以不适当的广告和讯息针对弱势人群。
  </p>
  <p class="graf graf--p">
   无论技术的采用基于什么方向，它都可能快速加速。
   <strong class="markup--strong markup--p-strong">
    新的面部分析和情绪监测技术可以轻松地与现有的监测基础设施和渴望获得情绪数据的行业整合
   </strong>
   。
  </p>
  <p class="graf graf--p">
   网络摄像头不仅仅是你可见的那些，它们已经嵌入到城市街道和火车站上的许多数字广告屏幕中。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    在这里看到《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e8%a6%81%e4%bd%a0%e8%bf%9c%e7%a6%bb%e5%b9%bf%e5%91%8a%e7%89%8c-%e8%80%81%e5%a4%a7%e5%93%a5%e6%9d%a5%e4%ba%86/" href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e8%a6%81%e4%bd%a0%e8%bf%9c%e7%a6%bb%e5%b9%bf%e5%91%8a%e7%89%8c-%e8%80%81%e5%a4%a7%e5%93%a5%e6%9d%a5%e4%ba%86/" rel="noopener noreferrer" target="_blank">
     为什么让你远离广告牌
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   ⚠️它们在你完全不知情的情况下为你拍照，并在几秒钟内捕获详细信息，包括
   <a class="markup--anchor markup--p-anchor" data-href="http://orbscreen.com/" href="http://orbscreen.com/" rel="noopener noreferrer" target="_blank">
    一家公司的文献
   </a>
   中所指出的：“面部特征，例如眼镜、胡须、情绪、年龄、和性别等等 ……
  </p>

  <p class="graf graf--p">
   这个“情绪”部分已经包括分析微笑的长度和质量。尽管欧盟法律禁止这些系统将图像超时存储，但是它们完全可以且已经被用于瞄准特定人群。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    充分利用不安全感：监视越界
   </strong>
  </h3>
  <p class="graf graf--p">
   滥用的可能性很深，早期的信号已经令人不安。
  </p>
  <p class="graf graf--p">
   在社交媒体平台上，情感定位已经越来越流行。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    &lt;
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%83%85%e6%84%9f%e5%88%86%e6%9e%90%ef%bc%9a%e5%a6%82%e4%bd%95%e5%9c%a8%e7%a4%be%e4%ba%a4%e7%bd%91%e7%bb%9c%e4%b8%8a%e6%8a%93%e5%8f%96%e5%b9%b6%e5%88%86%e6%9e%90%e5%a4%a7%e4%bc%97%e5%af%b9%e6%9f%90/" href="https://www.iyouport.org/%e6%83%85%e6%84%9f%e5%88%86%e6%9e%90%ef%bc%9a%e5%a6%82%e4%bd%95%e5%9c%a8%e7%a4%be%e4%ba%a4%e7%bd%91%e7%bb%9c%e4%b8%8a%e6%8a%93%e5%8f%96%e5%b9%b6%e5%88%86%e6%9e%90%e5%a4%a7%e4%bc%97%e5%af%b9%e6%9f%90/" rel="noopener noreferrer" target="_blank">
     情感分析：如何在社交网络上抓取并分析大众对某些话题的情绪
    </a>
    ？&gt;
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%af%b9%e7%89%b9%e6%9c%97%e6%99%ae%e6%8e%a8%e6%96%87%e7%9a%84%e6%83%85%e6%84%9f%e5%88%86%e6%9e%90-python-%f0%9f%90%8d/" href="https://www.iyouport.org/%e5%af%b9%e7%89%b9%e6%9c%97%e6%99%ae%e6%8e%a8%e6%96%87%e7%9a%84%e6%83%85%e6%84%9f%e5%88%86%e6%9e%90-python-%f0%9f%90%8d/" rel="noopener noreferrer" target="_blank">
     对特朗普推文的情感分析: Python
    </a>
    🐍》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%bd%93%e4%ba%ba%e4%bb%ac%e4%b8%8d%e5%86%8d%e4%bf%a1%e4%bb%bb%ef%bc%8d%ef%bc%8d%e5%9f%8b%e8%97%8f%e5%9c%a8%e6%83%85%e7%bb%aa%e6%95%b0%e6%8d%ae%e4%b8%ad%e7%9a%84%e5%aa%92%e4%bd%93%e5%92%8c%e8%a2%ab/" href="https://www.iyouport.org/%e5%bd%93%e4%ba%ba%e4%bb%ac%e4%b8%8d%e5%86%8d%e4%bf%a1%e4%bb%bb%ef%bc%8d%ef%bc%8d%e5%9f%8b%e8%97%8f%e5%9c%a8%e6%83%85%e7%bb%aa%e6%95%b0%e6%8d%ae%e4%b8%ad%e7%9a%84%e5%aa%92%e4%bd%93%e5%92%8c%e8%a2%ab/" rel="noopener noreferrer" target="_blank">
     当人们不再信任－－埋藏在情绪数据中的媒体和被损伤的社会知觉
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   Facebook
   <a class="markup--anchor markup--p-anchor" data-href="https://arstechnica.com/information-technology/2017/05/facebook-helped-advertisers-target-teens-who-feel-worthless/" href="https://arstechnica.com/information-technology/2017/05/facebook-helped-advertisers-target-teens-who-feel-worthless/" rel="noopener noreferrer" target="_blank">
    已经告诉广告商
   </a>
   ，它可以识别青少年何时感到“一文不值”、“不安全”、“挫败”、或“焦虑”，并特别揭露他们的不安全感，例如“身体语言”。
  </p>
  <p class="graf graf--p">
   ⚠️如此来，团体就可以购买广告或发布消息，来精确地瞄准那些最易受伤害的年轻人。
  </p>
  <p class="graf graf--p">
   如今的每一位雇主都在渴望了解员工的私生活。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%8f%98%e6%80%81%e7%9a%84%e9%9b%87%e4%bd%a3%e5%85%b3%e7%b3%bb/" href="https://www.iyouport.org/%e5%8f%98%e6%80%81%e7%9a%84%e9%9b%87%e4%bd%a3%e5%85%b3%e7%b3%bb/" rel="noopener noreferrer" target="_blank">
     变态的雇佣关系
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://steemit.com/surveillance/@iyouport/276ycw" href="https://steemit.com/surveillance/@iyouport/276ycw" rel="noopener noreferrer" target="_blank">
     你正在为一个贴身老大哥努力工作：地狱的新常态
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   2016年，在跟踪金融交易者情绪的实验中就对这些技术
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bloomberg.com/news/articles/2016-09-01/wall-street-s-next-frontier-is-hacking-into-emotions-of-traders" href="https://www.bloomberg.com/news/articles/2016-09-01/wall-street-s-next-frontier-is-hacking-into-emotions-of-traders" rel="noopener noreferrer" target="_blank">
    进行了测试
   </a>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “试想一下，如果所有的交易员都需要佩戴跟踪腕表来监视人们的生理状况，并且有一个仪表板，可以实时告诉您谁在抓狂 …… ”。 —— Andrew Lo
   </em>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    从那以后的几年中，这些技术变得更加便宜，随便就能买得起，因此对其他行业更具吸引力了
   </strong>
   ，包括那些员工的薪水可能较低且工作安全性较差的行业，这可能使他们很难拒绝对这些工具的使用。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/news/business/wp/2016/09/07/this-employee-badge-knows-not-only-where-you-are-but-whether-you-are-talking-to-your-co-workers/" href="https://www.washingtonpost.com/news/business/wp/2016/09/07/this-employee-badge-knows-not-only-where-you-are-but-whether-you-are-talking-to-your-co-workers/" rel="noopener noreferrer" target="_blank">
    初创公司 Humaneyz
   </a>
   通过带有一大堆传感器的ID徽章，跟踪员工谈话的语气（虽然不是谈话内容）、监视您与之交谈的人说了多长时间、判断是不经意的还是积极的 …… 以映射活动和网络并优化团队绩效。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.techrepublic.com/article/ai-tool-for-slack-gives-managers-insight-into-employees-emotions/" href="https://www.techrepublic.com/article/ai-tool-for-slack-gives-managers-insight-into-employees-emotions/" rel="noopener noreferrer" target="_blank">
    AI 服务 Vibe
   </a>
   可以分析 Slack 上的工作场所对话，以了解您的 “幸福感、烦恼、不赞成、失望和压力”，如果员工士气低落，它会向经理发送通知。
  </p>
  <p class="graf graf--p">
   注意力跟踪和眼动跟踪已经用于职业培训； TSA 的新兵被通过 EEG 监控他们的注意力水平和眼动追踪，“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.eyetechds.com/dhs-baggage-screening.html" href="https://www.eyetechds.com/dhs-baggage-screening.html" rel="noopener noreferrer" target="_blank">
    以前所未有的方式获得受训人员一直以来无法被观察到的方面
   </a>
   。”
  </p>
  <p class="graf graf--p">
   在美国，健康计划通常包括生物特征监控。尽管这些通常是可选的，但它们可以带会带来可观的奖金并影响健康保险的选择。
  </p>
  <p class="graf graf--p">
   这些技术的普及似乎只会加速。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    他们监控我们的情绪、并针对情绪反应的监控结果来控制我们看到的内容和广告 —— 以此操纵每个人的选择
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果想要将这项技术用在公民福祉方面，就需要让大众深入了解其利弊各方面，并进行坦率的公开的辩论，这至关重要。⚪️
  </p>
