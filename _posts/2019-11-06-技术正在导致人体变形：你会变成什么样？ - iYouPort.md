   技术正在导致人体变形：你会变成什么样？

   <li class="graf graf--p">
    <span style="color: #00ccff;">
     <em>
      <strong>
       科学家们已经模拟出人类未来的样子 —— 而且看起来一点都不帅……
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   专家们研究了技术对人体形态的影响，研究表明人类的身体可能会发生巨大变化。
  </p>
  <p class="graf graf--p">
   创建一个名为“Mindy”的未来人类的3D模型，科学家们表示，在2100年的人类可能会因长时间坐在电脑前和看着智能手机而出现身体变形。
  </p>
  <p class="graf graf--p">
   这个模型还体现出更大的颈部肌肉以弥补不良姿势的影响、头骨会变厚以保护免受辐射、大脑会变小，这是因为过度久坐不动的生活方式而发生的萎缩。
  </p>
  <p class="graf graf--p">
   在不到100年的时间里，人类的手指也可能会出现卷曲，就像下图这样：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6602 jetpack-lazy-image" data-lazy-sizes="(max-width: 620px) 100vw, 620px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-4.jpg?resize=620%2C413&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-4.jpg?w=620&amp;ssl=1 620w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-4.jpg?resize=300%2C200&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-4.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="413" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-4.jpg?resize=620%2C413&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="620"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6602" data-recalc-dims="1" height="413" sizes="(max-width: 620px) 100vw, 620px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-4.jpg?resize=620%2C413&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-4.jpg?w=620&amp;ssl=1 620w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-4.jpg?resize=300%2C200&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-4.jpg?resize=272%2C182&amp;ssl=1 272w" width="620"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Maple Holistics 的健康与保健专家 Caleb Backe 表示：“每天花数小时低头看着你的手机会拉伤你的脖子并使你的脊椎失去平衡。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “因此，脖子上的肌肉不得不花费额外的努力来支撑你的头部。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “坐在办公室电脑前几个小时也意味着你的躯干是向前倾斜的，而不是和臀部在一条垂直线上。”
  </p>
  <p class="graf graf--p">
   托莱多大学的 Kasun Ratnayake 也表示，人体可能被改变，以限制眼睛受到的有害光线的数量 —— 可能导致进化出第二眼睑。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “人类可能会发展出更大的内眼睑以防止暴露在过度的有害光线下，或者眼睛的晶状体可能会进化发育，使其能够阻挡入射的蓝光，但不会阻挡其他高波长的光，如绿、黄或红，”他说。
  </p>
  <p class="graf graf--p">
   来自 Med Alert Help 的 Nikola Djordjevic 博士说：“我们拿着手机的方式会导致某些接触点的紧张 — 导致’鼠标手’和’90度肘’也被称为肘管综合征。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这种综合征是由尺骨神经的压力或伸展所引起的，这一尺神经在肘内侧的凹槽中延伸。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这会导致无名指和小手指麻木或刺痛感，前臂疼痛和手部无力 — 保持肘部弯曲时间过长导致的。”
  </p>
  <p class="graf graf--p">
   TollFreeForwarding.com 的负责人杰森奥布莱恩（Jason O’Brien）制作了这个模型，并且研究了其成因，描述了人类的任何身体变化，作为技术的“副作用”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “技术为我们提供了便利、连接、娱乐等等 — 但需要权衡利弊，”他解释道。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “过度暴露于技术环境有时会损害我们的健康。”⚪️
  </p>
