 </p>
 <p>
  蔡英文6月25日接受法新社采访时表示台湾是“坚韧之岛”
 </p>
 <p>
 </p>
 <p>
  <span style="font-size: 12pt;">
   台湾中华民国总统蔡英文周一接受法新社采访时呼吁国际社会一道制约中国霸权野心。起身保卫民主自由价值。蔡英文形容中国对民主制度构成重大威胁。
   <img alt="" border="0" height="1" src="https://logs1187.xiti.com/hit.xiti?pv=1&amp;from=p&amp;s=588369&amp;p=syndication::boxun&amp;s2=1&amp;x1=%5bTW%5d&amp;x6=%5barticle%5d&amp;x10=%5bsyndication%5d&amp;x17=%5bexterne%5d" width="1"/>
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   蔡英文在采访中认为，台湾正遭受中国巨大的压力，虽然有时觉得挫折，但台湾人民不能放弃。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   蔡英文在采访中用了相当篇幅谈论中国大陆与台湾的两岸关系，她表示，过去两年来台湾持续承受来自中国的压力，近来发生的很多事件的力道与强度已冲击台海现状。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   蔡英文表示，中国越来越具侵略性，企图压制台湾，让民主自由的台湾离中国越走越远。她希望国际社会体认中国有展现区域霸权的意图，希望国际社会一起行动，制约中国，减少或遏止中国霸权扩充。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   蔡英文对法新社说，防御中国扩张主义，“这不只是对台湾的挑战，而是对整个地区和世界的挑战，因为今天是台湾，明天可能就是另一个国家得面对中国扩张势力。”“我们必须共同合作，重申我们的民主自由价值，以约制中国，尽量防止他们扩张霸权势力”
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   不过，蔡英文也表示，尽管台海两岸紧张情势升高，她仍愿意在总统任期内，在对等和不设政治前提条件下与中国国家主席习近平会面。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   蔡英文谈到亚太区域议题时表示，中国扩张和朝鲜核武计划对这个区域同样是不稳定或对和平状态的威胁来源。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   蔡英文表示，台湾有一个很大的邻居中国，在中国持续打压下，台湾仍坚持民主与自由价值，并且愿意贡献国际社会。蔡英文形容台湾是一座“坚韧之岛”。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   蔡英文认为，“面对中国威胁，我们觉得有必要加强自卫能力。”
  </span>
 </p>
 <p>
