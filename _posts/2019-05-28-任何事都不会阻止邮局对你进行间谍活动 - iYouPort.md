  <h1 class="entry-title">
   任何事都不会阻止邮局对你进行间谍活动
  </h1>
 </header>
 <div class="entry-meta">
  <span class="byline">
   <a href="https://www.iyouport.org/author/don-evans/" rel="author" title="由McCaffrey发布">
    McCaffrey
   </a>

    监视与操纵
   </a>
  </span>
  <span class="published-on">

 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      你寄出的信件可能无法被送到目的地，因为邮政系统一直在监视你，你甚至根本无法察觉这种监视的存在，怎么回事？
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   你还记得去年最著名的邮包炸弹事件吗？炸弹爆炸后不到一个星期，一名嫌犯被捕了。几乎立刻，在特朗普的集会上的视频开始传播，那些人戴着“Make America Great Again”的帽子高呼口号。那名嫌疑人很快就被审判、定罪和监禁。
  </p>
  <p class="graf graf--p">
   不，故事并没有结束。对炸弹事件的调查应该
   <a class="markup--anchor markup--p-anchor" data-href="https://www.fastcompany.com/90257308/suspicious-packages-spotlight-vast-postal-surveillance-system-mail-covers" href="https://www.fastcompany.com/90257308/suspicious-packages-spotlight-vast-postal-surveillance-system-mail-covers" rel="noopener noreferrer" target="_blank">
    将注意力集中在
   </a>
   <strong class="markup--strong markup--p-strong">
    一个几乎不为人知的联邦监视计划上 — — 该计划对每个公民的隐私和宪法权利构成直接的威胁。它被称为“Mail Cover Program”，由美国邮政局（USPS）运营
   </strong>
   。是的，甚至连邮局都在暗中监视我们。
  </p>
  <p class="graf graf--p">
   该监视计划允许邮政员工拍摄并向联邦执法机构（FBI，DHS，特勤局等）发送邮局处理的每一封邮件的正面和背面。并且以数字方式保留信息，并在没有授权的情况下将其提供给任何想要它的政府机构。
  </p>
  <p class="graf graf--p">
   2015年，美国邮政总局检察长发布了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.uspsoig.gov/sites/default/files/document-library-files/2015/hr-ar-15-007_0.pdf" href="https://www.uspsoig.gov/sites/default/files/document-library-files/2015/hr-ar-15-007_0.pdf" rel="noopener noreferrer" target="_blank">
    一份报告称
   </a>
   ，“各机构必须展示合理的依据，向刑事调查服务中心发送申请表的硬拷贝以进行处理，并将邮件封面视为限制和保密……不应将邮件封面用作常规调查工具。对邮件封面计划的控制不足可能会妨碍邮政检查服务进行有效调查的能力，导致公众对邮件隐私的担忧，并损害邮政服务的品牌形象。“
  </p>
  <p class="graf graf--p">
   事实上这些警告不仅被完全忽略了，邮件封面监视计划在上述报告发布后还扩展了其规模。实际上，在该报告发布后的几个月内，有6,000个邮件封面收集请求。根据2019年2月版的“监狱法律新闻”（P.34–35），只有10个请求被拒绝。
  </p>
  <p class="graf graf--p">
   我对邮件封面监视计划有一些个人经验。我因试图揭露中央情报局的非法酷刑计划而被服刑23个月。在被关押了两个月之后，我决定委托一位囚犯为我妻子的40岁生日买一张卡片。
  </p>
  <p class="graf graf--p">
   我提前两周寄出的这张生日卡，但我的妻子从未收到过。最后，大约四个月后，卡片被送回给我，上面有一个黄色的“Return to Sender — Address Not Known”标签。但是那张贴纸下面是第二个黄色贴纸，上面写着：“不得交付。Hold For Supervisor. Cover Program.。“
  </p>
  <p class="graf graf--p">
   我为什么要接受邮政服务的监视？我不知道。我在法庭上度过了一天，案子就结束了。但请记住，邮政局不必回答任何人关于这个问题 — 包括我的律师、我的法官、甚至是他们自己的监察长。他们可以不需要逮捕令就可以监视我（或我的家人），甚至是如果国会议员询问为什么会有这样的间谍监视行为他们也不需要回答。
  </p>
  <p class="graf graf--p">
   问题不仅在于政府机构（或准政府机构）在没有任何理由或正当程序的情况下监视个人的险恶性质，尽管这些都是严重的问题。这是因为该计划的处理非常糟糕且非常随意，以至于在某些情况下，由于没有明显的执法理由而对个人进行了监视，并且监视工作是由邮政服务人员发起的，甚至没有被授权这样做。
  </p>
  <p class="graf graf--p">
   再一次，受害人没有追索权，因为被监视的人甚至都不知道什么时候这种情况正在发生。
  </p>
  <p class="graf graf--p">
   该计划的一个更令人不安的方面可能是，2000年至2012年期间，邮政服务平均每年发起8,000封邮件的封面请求。但在2013年，这个数字
   <a class="markup--anchor markup--p-anchor" data-href="https://www.uspsoig.gov/document/postal-inspection-service-mail-covers-program" href="https://www.uspsoig.gov/document/postal-inspection-service-mail-covers-program" rel="noopener noreferrer" target="_blank">
    跃升至49,000
   </a>
   。为什么？没有人知道，邮政局不必说。
  </p>
  <p class="graf graf--p">
   但问题不在于邮件封面监视计划下有多少案件，甚至没有多少案件要求提供信息。真正的问题是，“这对宪法意味着什么？”也许第二个问题是，“为什么没有人在法庭上质疑该计划？”
  </p>
  <p class="graf graf--p">
   一般来说，美国人不会 — 或者至少曾经没有反对逐渐丧失公民自由和宪法权利。现在这种情况必须被停止。即使邮局正在监视你，你也应该知道民主自由遇到了麻烦。
  </p>
  <p class="graf graf--p">
   作者 John Kiriakou 是前中央情报局反恐官员，也是参议院外交关系委员会的前高级调查员。由于他试图揭露布什政府的酷刑计划，被判服刑23个月。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    I
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://consortiumnews.com/2019/02/28/john-kiriakou-neither-rain-sleet-nor-snow-will-stop-the-post-office-from-spying-on-you/" href="https://consortiumnews.com/2019/02/28/john-kiriakou-neither-rain-sleet-nor-snow-will-stop-the-post-office-from-spying-on-you/" rel="noopener noreferrer" target="_blank">
    t’s called the “Mail Cover Program” and it’s run by the U.S. Postal Service (USPS). Yes, even the Post Office is spying on us, writes John Kiriakou
   </a>
   . When even the Post Office is spying on you, you know the republic is in trouble.
  </p>
