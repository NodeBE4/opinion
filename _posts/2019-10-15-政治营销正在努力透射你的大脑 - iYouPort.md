  <p class="graf graf--p">
   Facebook 和亚马逊的精神错乱似乎只会持续，并且不会很快放缓。
  </p>
  <p class="graf graf--p">
   现在，这两家大型企业巨头希望
   <strong class="markup--strong markup--p-strong">
    以他们自己的方式进入阅读人类情感的未知领域。
   </strong>
  </p>
  <p class="graf graf--p">
   Facebook 想要一个具有
   <a class="markup--anchor markup--p-anchor" data-href="https://translate.googleusercontent.com/translate_c?anno=2&amp;depth=1&amp;hl=sv&amp;rurl=translate.google.com&amp;sl=en&amp;sp=nmt4&amp;tl=sv&amp;u=https://www.engadget.com/2019/05/20/facebook-ai-robots-curiosity-tactile/&amp;xid=17259,15700022,15700186,15700191,15700256,15700259&amp;usg=ALkJrhgPdWSggfsG0CvwRhKYGRjk1NdnKw" href="https://translate.googleusercontent.com/translate_c?anno=2&amp;depth=1&amp;hl=sv&amp;rurl=translate.google.com&amp;sl=en&amp;sp=nmt4&amp;tl=sv&amp;u=https://www.engadget.com/2019/05/20/facebook-ai-robots-curiosity-tactile/&amp;xid=17259,15700022,15700186,15700191,15700256,15700259&amp;usg=ALkJrhgPdWSggfsG0CvwRhKYGRjk1NdnKw" rel="noopener noreferrer" target="_blank">
    五种感官
   </a>
   能够读懂人类情感的机器人。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.dailymail.co.uk/sciencetech/article-7085789/Facebook-wants-send-emotional-robots-explore-scan-faces-help-users-make-friends.html" href="https://www.dailymail.co.uk/sciencetech/article-7085789/Facebook-wants-send-emotional-robots-explore-scan-faces-help-users-make-friends.html" rel="noopener noreferrer" target="_blank">
    据
   </a>
   <em class="markup--em markup--p-em">
    Dailymail
   </em>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.dailymail.co.uk/sciencetech/article-7085789/Facebook-wants-send-emotional-robots-explore-scan-faces-help-users-make-friends.html" href="https://www.dailymail.co.uk/sciencetech/article-7085789/Facebook-wants-send-emotional-robots-explore-scan-faces-help-users-make-friends.html" rel="noopener noreferrer" target="_blank">
    报道
   </a>
   ，Facebook 希望具备“敏锐的情绪体察能力”的机器人可以探索世界、识别物品和人，并使其用户能够结交更多朋友。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    这些机器人将配备轮子或坦克式履带的装置，使其能够在大多数环境中行走。另外，Facebook 还在他们的专利中建议使用驱动系统，这些驱动系统可以让机器人在水下移动、在空中飞行、甚至漂浮在太空中
   </em>
   。
  </p>
  <p class="graf graf--p">
   我不确定为什么有人会再次信任 Facebook 管理数据，更不用说生物识别数据了，就在包括数据挖掘和操纵在内的众多丑闻被曝光之后。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5728 jetpack-lazy-image" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/亚马逊上.gif?resize=1100%2C788&amp;is-pending-load=1#038;ssl=1" data-recalc-dims="1" height="788" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/亚马逊上.gif?resize=1100%2C788&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5728" data-recalc-dims="1" height="788" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/亚马逊上.gif?resize=1100%2C788&amp;ssl=1" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   另一方面，据彭博社
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bloomberg.com/news/articles/2019-05-23/amazon-is-working-on-a-wearable-device-that-reads-human-emotions" href="https://www.bloomberg.com/news/articles/2019-05-23/amazon-is-working-on-a-wearable-device-that-reads-human-emotions" rel="noopener noreferrer" target="_blank">
    报道
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    亚马逊还在研究以一种完全不同的方式阅读人类情感的技术、利用语音激活的可穿戴设备，通过语调来读取佩戴者的心态
   </strong>
   。
  </p>
  <p class="graf graf--p">
   到目前为止，几乎每个人都熟悉国家监视设备的庞大规模了。问题已经不再是“我们是否受到监视”，而是，入侵还会进一步扩大到什么地步。
  </p>
  <p class="graf graf--p">
   当世界各地的政治家已经
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%B8%BA%E4%BB%80%E4%B9%88%E8%A6%81%E4%BD%A0%E8%BF%9C%E7%A6%BB%E5%B9%BF%E5%91%8A%E7%89%8C-%E8%80%81%E5%A4%A7%E5%93%A5%E6%9D%A5%E4%BA%86/" href="https://www.iyouport.org/%E4%B8%BA%E4%BB%80%E4%B9%88%E8%A6%81%E4%BD%A0%E8%BF%9C%E7%A6%BB%E5%B9%BF%E5%91%8A%E7%89%8C-%E8%80%81%E5%A4%A7%E5%93%A5%E6%9D%A5%E4%BA%86/" rel="noopener noreferrer" target="_blank">
    使用嵌入广告牌中的面部识别技术来
    <strong class="markup--strong markup--p-strong">
     判断和操纵
    </strong>
    那些视觉上对所显示信息作出反应的人的情绪
   </a>
   时，这个概念已经变得格外荒谬。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    它甚至还有自己的研究分支：被称为神经政治学
   </strong>
   。（详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" rel="noopener noreferrer" target="_blank">
    心理操纵的秘密：”行为经济学”+PR=？
   </a>
   》）
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “令人毛骨悚然”这个词如今在科技新闻中几乎遍地可见，虽然
   <strong class="markup--strong markup--p-strong">
    不断重复这个词并不能改变灾难的现状，甚至无法帮你深刻理解灾难
   </strong>
   ，
  </p>
  <p class="graf graf--p">
   然而，爱荷华大学的一个新项目实际上已经被正式标记为“令人毛骨悚然的研究”。更不用说负责实施这些技术的人根本不知道他们自己在做什么。
  </p>
  <p class="graf graf--p">
   隐藏在白色天花板瓷砖中的摄像机秘密扫描所有人的面孔，并且，能够立即计算出他们的心情 —— 读数快乐、沮丧、困惑、恐惧、愤怒和悲伤。
  </p>
  <p class="graf graf--p">
   Hedgcock 认为，爱荷华大学是
   <strong class="markup--strong markup--p-strong">
    世界上唯一
   </strong>
   一所拥有这种实时监控情感软件的商学院，
   <strong class="markup--strong markup--p-strong">
    它将人脸图像转换为不同情绪的阅读材料
   </strong>
   。⚠️专家们已经看到这种玩意儿在营销、广告、甚至政治活动的世界中的巨大潜力。
  </p>
  <p class="graf graf--p">
   这是一个完美的例子，足以说明为什么“令人毛骨悚然”的描述并没有能揭示它的真实含义。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    真实的含义是：他们把学生当成了试验品，不仅消除了隐私、还把人类本身货币化了，同时说服学生将自己定位为以人类行为操纵为目标的新行业的先驱
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这不是仅仅“令人毛骨悚然”能描述的，这是技术专家的恶意收购。
  </p>
  <p class="graf graf--p">
   目前，该系统声称准确率不超过35％，因为观察到的大多数情绪被归类为“中性”。
  </p>
  <p class="graf graf--p">
   但是，Dan Hill（一家致力于这种技术的公司的总裁）和 Hedgcock 教授关于意图的最终陈述中
   <strong class="markup--strong markup--p-strong">
    公开承认操纵是最终目标，并且没有任何悔意
   </strong>
   ：
  </p>
  <p class="graf graf--p">
   Hedgcock 说，⚠️政治战略家已经对神经营销充满兴趣 —— 神经营销是一种使用人体测量为手段的营销领域 ，包括脑部扫描、呼吸频率和皮肤变化测量，以此来衡量被营销对象的反应。
  </p>
  <p class="graf graf--p">
   老大哥们正努力假装深沉，他们只是做，并不想对此多说些什么。
  </p>
  <p class="graf graf--p">
   相比下 Hedgcock 是足够坦率的，⚠️他承认，“在神经营销领域的每一位学者都不会担心什么过度操纵的问题，我们唯一思考的是，如何让它更有效”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    现在应该担心你的家
   </strong>
  </p>
  <p class="graf graf--p">
   值得注意的是，至少 Facebook 和亚马逊这两家公司都拥有智能家居设备，在阅读完上述内容后，您应该担心附加在这些家居设备上的摄像头和麦克风所收集的信息……已经不仅仅是你和你的家人被卖掉这么简单了。
  </p>
  <p class="graf graf--p">
   在亚马逊方面，似乎该公司很可能希望在各种不同的数码产品中使用这项技术，从 Alexa 等个人助理到零售巨头目前正在开发的新技术。
  </p>
  <p class="graf graf--p">
   亚马逊已经宣布
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theverge.com/2018/4/23/17270002/amazon-robot-home-alexa-echo" href="https://www.theverge.com/2018/4/23/17270002/amazon-robot-home-alexa-echo" rel="noopener noreferrer" target="_blank">
    正在开发
   </a>
   一种个人辅助机器人，因此新的情感技术可以轻松地集成到这个家用机器人中。确实是一种可怕的恐怖思想。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    一个新市场
   </strong>
  </p>
  <p class="graf graf--p">
   亚马逊和 Facebook 并不是唯一利用人类情感的公司。其他寡头也在蠢蠢欲动。
  </p>
  <p class="graf graf--p">
   此前，Activist Post 报道称，沃尔玛还在考虑通过购物车手柄上的传感器监控您的生物识别数据、电脉冲和位置。（在这里看到中文版和视频《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%88%ab%e6%91%b8%e4%b8%9c%e8%a5%bf%ef%bc%81video/" href="https://www.iyouport.org/%e5%88%ab%e6%91%b8%e4%b8%9c%e8%a5%bf%ef%bc%81video/" rel="noopener noreferrer" target="_blank">
    别摸东西！
   </a>
   》）
  </p>
  <p class="graf graf--p">
   依旧有数百家零售商店 —— 很快就是成千上万家 —— 正在调查使用生物识别软件 FaceFirst 来建立一个商店扒手数据库，以帮助打击盗窃。
  </p>
  <p class="graf graf--p">
   FaceFirst 被设计为可以在50到100英尺范围内识别任何面部。当顾客走进商店入口时，摄像机会重复捕捉每个购物者的图像，并选择最清晰的存储。
  </p>
  <p class="graf graf--p">
   然后，软件分析该图像并将其与零售商编制的“坏蛋客户”数据库进行比较；如果匹配，软件会向员工发送警报，告知“高风险”客户已进入大门。
  </p>
  <p class="graf graf--p">
   直到目前面部识别技术甚至仍是可笑的：亚马逊自己的面部识别技术将28位国会议员标记为“犯罪嫌疑人”，是众所周知的闹剧。
  </p>
  <p class="graf graf--p">
   而且此前，英国的另一项面部识别技术测试，结果导致35次错误匹配、和1次错误逮捕。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    亚马逊是全球最大的间谍组织的最大技术供应商
   </strong>
   。
  </p>
  <p class="graf graf--p">
   2018年联邦调查局和其他间谍机构已经开始使用相同的亚马逊面部识别技术来筛选监测数据。
  </p>
  <p class="graf graf--p">
   Defense One
   <a class="markup--anchor markup--p-anchor" data-href="https://www.activistpost.com/2018/10/the-rise-of-facial-recognition-technology-is-now-inevitable.html" href="https://www.activistpost.com/2018/10/the-rise-of-facial-recognition-technology-is-now-inevitable.html" rel="noopener noreferrer" target="_blank">
    报道
   </a>
   ，“预测犯罪系统”正在拉斯维加斯举行的最近一次安全技术会议 ISC West 上展出。
  </p>
  <p class="graf graf--p">
   数以百计的零售商店想要面部识别技术的事实本身就是一个可怕的想法。
  </p>
  <p class="graf graf--p">
   而且还结合生物识别数据，对于我们未来的购物车而言，更是一个可怕的前景 —— 可以读取人类的情感数据，包括实时检测心理压力。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    我们所有人正在进入少数派报告的世界。
   </strong>
  </p>
  <p class="graf graf--p">
   公民被灌输了只有放弃实际的人权才能获得“安全”的价值观，一旦人们默许，就没有回头路了。
  </p>
  <p class="graf graf--p">
   正如本杰明富兰克林所说：“那些放弃必要的自由去交换一点临时安全的人，既不能得到自由也不会得到安全。”
  </p>
  <p class="graf graf--p">
   每一位读到本文的读者，请帮助您的身边人重新思考这笔交易。⚪️
  </p>

    the final statements from Dan Hill (president of one company working with this tech) and Professor Hedgcock about what is intended, is an open admission that manipulation is the end goal and there is no remorse about it
   </a>
