<p class="ib ic jc id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="9a70">
       我课上的生日报演示，课程结束时有选手互投。有趣的是，有两位冠军，港大慈美琳和北大韩晓天，选题都与毛著有关。这些个90后，冲浪网络，四处搜求，竟能把死板的题目做得妙趣横生（我还应小慈邀请唱了一段“再版前言”）。当然这不只是神剧重演，笑声中同学们触碰到致命病灶。这和出版、营销有半毛钱关系吗？是书的奇迹，还是劫难的先兆？如果再有领袖著作发行数千万乃至上亿的报道，相信这些年轻人不会跟着起哄。
      </p>
</blockquote>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="c65c">
      毛泽东74岁生日这天的报纸会是什么样子呢？带着一吨好奇，我翻开了《人民日报》、山东省机关报《大众日报》和当时离家乡最近的报纸《青岛日报》。那一日，三份报纸出奇地相似。
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="43b5">
      三份报纸的起题风格、艺术品位各不相同，但内容其实是同一篇通稿，介绍畅销书《毛泽东选集》今年的出版盛况。为何却未有提及毛的生日？钱钢老师告诉我，原来，毛曾立下规矩，党内不祝寿。不过，想来这份满载胜利喜悦的出版报告已是一份颇有分量的生日祝福，不论配上4*“万寿无疆”，抑或1966年8月新鲜出炉的口号“四个伟大”（伟大的导师、伟大的领袖、伟大的统帅、伟大的舵手）想必都是极好的。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="1943">
      翻阅几份报纸，我发现里面内容大同小异，大抵是在介绍两件事情。一，全国和全世界人民都很热爱毛主席；二，全国全世界人民都热爱毛泽东思想。这天的文章都紧紧围绕着这两个中心思想，多而不乱。
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="0dfa">
      那天的文章金句很多，以下几个可供感受一下氛围：
     </p>

      如果让我找一个词形容对这些话语的感受，大概会用“浪漫”吧。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="d994">
<strong class="id jd">
       毛系神书之毛选
      </strong>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="c9c5">
      回归正题，今期报纸头版的标题是毛选，副标题里也提到了毛语录等另外三本神书。
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="c4d7">
      这让我很疑惑。毛选出版了8000万部，远不及毛语录的三亿五千万册，为何却可以做主角？查阅资料发现，原来这年有一项重要的政治任务，1月4日时，周恩来曾在北京工人体育场宣布，当年要印制《毛选》8000万套，要求全国节约纸张，尤其公文用纸，集中力量印毛选。而1967年全年实际完成了9151万套《毛选》的出版，年底毛生日时，宣布这个光荣任务的完成再合适不过。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="79a4">
      毛选是一个我们从小常听到的词汇，但毛选到底是什么？什么来头？收录了哪些内容？带着这些问题，我继续查阅。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="8772">
      我发现，早在建国以前，毛选就出现了。1944年，时任《晋察冀日报》总编辑邓拓主持修订了第一部五卷本的《毛泽东选集》，选了28篇文章。1945年中共七大之后，由于毛泽东思想成为中共指导思想，各地更是纷纷出版选集，达21种之多，但都没有经过毛本人的审核。
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="1df9">
      北大深圳研究生院韩晓天同学在生日报演示时详细罗列了毛选版本。他发现，当时的毛选版本间差异较大，且受技术条件限制装订粗糙，甚至错误频繁。其中苏中出版社的编辑质量比较高，但所用纸张较差，排版和篇幅也跟其它版本差异较大，据说是当时根据地割裂，苏中的编辑工作独立，且都是隐蔽在芦苇荡中进行的缘故。而1948年晋冀鲁豫中央局编印的精装版是新中国成立前内容最为丰富的一版，共收录毛泽东著作61篇、共计95万字。但这个版本却删去了一封毛泽东写给林彪的、含有批评林彪内容的信。
     </p>

     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="b68c">
      为何战线拉得这么长？与毛系其他神书不同，毛选是由中共中央毛泽东选集编辑委员会编选，再经毛泽东亲自通读审查，最后由人民出版社出版的，是真正根正苗红的官方读本。
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="2145">
      文革前夕，为更好普及毛选，人民出版社在1964年出版了《毛泽东著作选读》，分甲、乙两种版本，其中甲种本面向干部、乙种本面向工农群众和青年知识分子。

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="119a">
      而我从家里翻出一本塑料封面、32开本的一卷本《毛泽东选集》，也是同年出版。这个版本是由中国人民解放军战士出版社翻印。听钱老师说我才知道，这个酷炫的一卷本刚出现时，只有军人和官员家里才有，非常珍贵。我家这本应当是后来买到的，几次搬家都未丢失，家人应也是十分珍视。

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="1aab">
      而爸爸出生的文革期间，毛泽东思想“脱销”，官方版本已无法满足群众的学习需求。这时，民间开始流行一种称为“万岁本”的毛著版本，这是一种非官方出版物，由一些红卫兵组织自发收集毛泽东的内部讲话和文稿、批示，编印成《毛泽东思想万岁》等各种版本。由于涉及大量未公开发表过的内容，出版不久便遭查禁。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="c588">
      当日的报纸上，有一段讲述日本人民与毛选的故事，摘录如下：“一位第一次来到中国的日本青年海员，心里想的是能够得到梦寐以求的宝书 — — 《毛泽东选集》。船刚靠岸，他就直奔国际海员俱乐部，一心一意想得到毛主席著作。当他在俱乐部里看到陈列着的《毛泽东选集》时，真高兴极了。他飞快地跑过去，双手捧起宝书，紧紧地贴在胸前。顿时，喜悦的热泪便夺眶而出，老半天才自言自语地说：‘哦，盼啊盼，今天总算把宝书盼到手了！’接着，他又仔细地端详着书中扉页上的毛主席像，沉浸在无限的幸福之中。他向服务员滔滔不绝地倾诉了日本革命人民热爱毛主席、渴望毛泽东思想的心情。他语重心长地说：‘啊，我们日本人民是多么急需这本革命的宝书啊！’”
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="5641">
<strong class="id jd">
       毛系神书之毛语录
      </strong>
</p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="f2d7">
      毛语录，因多采用红色塑料皮包装，江湖人称“红宝书”，又有名曰“红色圣经”，外国群众亲切地称它“Little Red Book”，它是当之无愧的时代标志。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="5363">
      毛语录的问世，离不开林彪的大力推动。1961年4月，林彪在部队视察时指示，为使战士在各个时期、各种情况下都能及时得到毛主席思想指导，《解放军报》应当经常选登毛主席的有关语录。同年劳动节开始，《解放军报》每日在报纸报眼放上一句毛主席的话，第一次选登的是，”整个革命历史证明，没有工人阶级的领导，革命就要失败，有了工人阶级的领导，革命就胜利了。在帝国主义时代，任何国家的任何别的阶级，都不能领导任何真正的革命达到胜利”。后来其他报纸也纷纷效法。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="eb59">
      第一本语录也是由《解放军报》编辑出版，最早的征求意见版出现在1964年，收录了毛的200句语录，印成52开本，方便战士们随时随地拿出学习。后来，毛语录又经历了全国发行、再版、修改、销往海外，成为了世界人民的革命宝书。
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="7e72">
      值得一提的是，1966年末，毛语录再版时，《人民日报》在头版刊出唯一副主席林彪写的《&lt;毛主席语录&gt;再版前言》，“毛泽东同志是当代最伟大的马克思列宁主义者，毛泽东同志天才地，创造性地，全面地继承，捍卫和发展了马克思列宁主义……”如此晦涩的一段话，竟又被改编为一首歌曲，请欣赏。

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="e675">
      后来，毛语录以至少48种文字在世界上传播，包括中文、8种少数民族文字、37种外文、汉英对照版和盲文版。十余年间，发行数量达到50亿本，而1977年时，全世界人口只有40.83亿。有机构统计，毛语录发行量仅低于《圣经》。
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="618f">
      那时，语录渗透进群众生活的方方面面，从每个人嘴里说出的话都姓毛。那时家长要打孩子时，孩子一旦机灵说出“要文斗不要武斗”，常能让爸妈下不去手。而行乞时，要记得说“最高指示:我们都是来自五湖四海,为了一个共同的革命目标走到一起来了……一切革命队伍的人都要互相关心，互相爱护，互相帮助”再伸出双手。

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="2e28">
      语录仗也是那个年代十分有趣的现象。在两派辩论或争吵中，若要想骂对方“不要脸”，可引这段：“只有不要脸的人们才说得出不要脸的话,顽固派有什么资格站在我们面前哼一声呢?”还有一句常用“语录”：“死皮赖脸，乱吹一顿，不识人间有羞耻事。”
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="5909">
      据说陈毅在一次接受造反派批判时，要造反派跟他一起翻开《毛主席语录》271页“共同学习最高指示”。他带头诵读道：“伟大领袖毛主席教导我们:‘陈毅是个好同志。’”可是《毛主席语录》并无271页，造反派发现上当，欲批判陈毅“伪造最高指示”，在会场上的周恩来却证实毛主席说过这句话。但是陈毅当年的秘书杜易在《大雪压青松──“文革”中的陈毅》一书中则表示并无此事。编造毛泽东语录是“罪上加罪”，陈毅不可能这么做。而这个故事之所以流传，正是说明当时人们认为陈毅是好同志，为他抱不平。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="1bde">
      伴随着文革的结束，毛语录的神话也终结了。1979年2月12日，中共中央宣传部发出的《关于停止发行〈毛主席语录〉的通知》中说：“林彪为捞取政治资本而搞的《毛主席语录》，断章取义，割裂毛泽东思想，自发现以來，危害很大，流毒甚广。為了肃清林彪、‘四人帮’的流毒，自即日起，新华书店、国际书店现存的中文版、民族版和外文版《毛主席语录》本一律停止发行。”
     </p>
