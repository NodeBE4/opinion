   <strong>
    导读部分
   </strong>
   ：
  </p>
  <ul>
   <li class="graf graf--p graf--startsWithDoubleQuote">
    “问题不在于夺取权力，而在于行使权力。” ——《
    <a href="https://www.iyouport.org/1%E3%80%81-%E5%85%B1%E8%AF%86%E5%8F%AA%E6%98%AF%E6%89%8B%E6%AE%B5%E8%80%8C%E4%B8%8D%E6%98%AF%E7%9B%AE%E7%9A%84/" rel="noopener noreferrer" target="_blank">
     共识只是手段而不是目的
    </a>
    》；
   </li>
   <li>
    “展示运动的广度、多样性和力量；你不需要把队友集中在同一个大目标上” ——《
    <a href="https://www.iyouport.org/2%E3%80%81%E5%88%86%E5%B8%83%E5%BC%8F%E8%A1%8C%E5%8A%A8%E7%9A%84%E8%A1%97%E5%A4%B4%E6%B4%BE%E5%AF%B9/" rel="noopener noreferrer" target="_blank">
     分布式行动的街头派对
    </a>
    》；
   </li>
   <li>
    “关于框架有一个基本的事实：如果你接受了别人的框架，你就输了” —— 《
    <a href="https://www.iyouport.org/3%E3%80%81%E6%A1%86%E6%9E%B6%E6%98%AF%E4%B8%80%E7%A7%8D%E6%9D%83%E5%8A%9B/" rel="noopener noreferrer" target="_blank">
     框架是一种权力
    </a>
    》；
   </li>
   <li>
    “你永远不可能仅仅通过对抗现有的现实来改变这一切。要改变某些东西，你必须建立一个新的模型，使现有模型过时” —— 《
    <a href="https://www.iyouport.org/4%E3%80%81%E5%88%AB%E6%8A%8A%E6%B3%A8%E6%84%8F%E5%8A%9B%E9%9B%86%E4%B8%AD%E5%9C%A8%E6%8A%B5%E6%8A%97%E4%B8%8A%EF%BC%9A%E9%A2%84%E8%AE%BE%E6%80%A7%E5%B9%B2%E9%A2%84%E6%88%98%E6%9C%AF/" rel="noopener noreferrer" target="_blank">
     预设性干预战术
    </a>
    》
   </li>

  <blockquote>
   <p class="graf graf--p graf--startsWithDoubleQuote">
    <em>
     “关于框架有一个基本的事实：如果你接受了别人的框架，你就输了” —— George Lakoff
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p">
   还记得我们在《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" rel="noopener noreferrer" target="_blank">
    互联网霸主的时间线
   </a>
   》那篇长报告中一开始所阐述的问题吗？没错：叙事。也就是讲故事。
   <strong class="markup--strong markup--p-strong">
    这非常重要
   </strong>
   。
  </p>
  <p class="graf graf--p">
   人类本身实际上是用叙事连接起来的。
  </p>
  <p class="graf graf--p">
   故事是我们生活的线索，是故事把人们编织在一起，形成人类文化的结构。
  </p>
  <p class="graf graf--p">
   故事可以是立即通知、也或者是欺骗，可以是启发、也或者是娱乐，再或者是所有上述这一切。我们生活在一个由故事塑造的世界中。
  </p>
  <p class="graf graf--p">
   传统的分析使组织者和活动家能够理解关键目标决策者和盟友之间的权力关系和制度动态。
  </p>
  <p class="graf graf--p">
   叙事分析则提供了一个框架，将权力分析扩展到叙事空间 —— 故事、思想和假设的无形领域，构成公众对情况和有关参与者的看法。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    叙事有助于定义什么是正常的，什么是合理的，以及政治上可能的限制。所有权力关系都有这样一个叙事组成部分
   </strong>
   。
  </p>
  <p class="graf graf--p">
   💡叙事的基础是认识到故事的价值不在于真理，而在于意义。
  </p>
  <p class="graf graf--p">
   也就是说，
   <strong class="markup--strong markup--p-strong">
    故事的强大之处在于，故事如何在听众的心灵和思想中创造意义。
   </strong>
  </p>
  <p class="graf graf--p">
   如你所知，说服人们的障碍通常不是由于他们尚不知道什么，而是他们已经知道的却没能理解的。
  </p>
  <p class="graf graf--p">
   换句话说，人们现有的假设和信念在充当叙事的过滤器，以防止他们去*理解*社会变化的信息。
  </p>
  <p class="graf graf--p">
   就如大多数人都知道“我的TL”和“他的TL”不一样，于是“我眼中的热点”和“他眼中的热点”不一样；这就是算法专制，但是，也正是大多数人依旧在假设这个互联网是信息自由的、或者干脆说依赖这个互联网带来信息自由，于是他们会下意识地拒绝去*理解*什么是算法专制。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    叙事权力分析旨在挖掘这些有害叙事的隐藏阻塞，以便解放性叙述可以更好地挑战它们
   </strong>
   。
  </p>
  <p class="graf graf--p">
   💡简单说，你的叙述不仅需要能挑战所面临的经济和政治力量，而且，还应该挑战支持这些力量的叙述 ——这种叙述自我合法化并允许构建它的权力发挥威胁性。
  </p>
  <p class="graf graf--p">
   目前的现实往往植根于压迫性叙事。作为变革推动者的角色就是要破坏这些叙述，并用有助于建立一个更公平、更自由的世界的新叙事取而代之。
  </p>
  <p class="graf graf--p">
   叙事最重要的是框架。
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    <em>
     赢得辩论的最简单方法是由你来定义辩论条款。
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p">
   重构是通过加宽框架、缩小框架或将框架完全移动到另一个场景，来用新的故事替换旧故事的过程。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当权者通常会以一种有利于他们利益的方式来制定他们的议程；你要做的就是打破他们的框架。
   </strong>
  </p>
  <p class="graf graf--p">
   就像相机的取景器一样，叙事的框架将公众聚焦于反映制定者利益的特定信息上。
  </p>
  <p class="graf graf--p">
   如何重新定义一个问题？第一步就是进行叙事权力分析，寻求确定对手的基本假设 —— 例如 “这是独一无二的”、“天下的天鹅都是白色的”、“C 国是世界上最伟大的国家”……
  </p>
  <p class="graf graf--p">
   根据你的叙事权力分析，提出另一个揭露错误假设的故事。
  </p>
  <p class="graf graf--p">
   ⚠️重构通常包括通过突出显示故事中已被排除在主线之外的方面来使隐藏的部分变为可见。
  </p>
  <p class="graf graf--p">
   接下来，设计一个重新定位故事的重构动作。将公众的注意力转移到犯罪现场，以揭露一个恶行。
  </p>
  <p class="graf graf--p">
   使用与历史叙事相关的标志性位置，例如纪念碑或公园，其名称在故事中很重要；将您的行动与即将备受关注的高调活动或重要日期联系起来，构建并预示围绕这些庆祝活动的公众对话。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “重要日期”甚至可以不是传统的，只要它是大众所熟知的 —— ⚠️请注意，是大众而不是小众，不能将反对派群体自己的“重要日期”理解为大众。
  </p>
  <p class="graf graf--p">
   如果您希望将重构行动扩展为一场运动，您可能会成功地将强大的新 meme 注入到媒体和政策语话中。芝加哥 Arise Workers Center 的 Adam Kader 提供了
   <a class="markup--anchor markup--p-anchor" data-href="http://www.inthesetimes.com/working/entry/6824/" href="http://www.inthesetimes.com/working/entry/6824/" rel="noopener noreferrer" target="_blank">
    这样一个例子
   </a>
   ：
  </p>
  <p class="graf graf--p">
   像劳工部和主流媒体这样的机构将剥削工人的现象称为“不支付工资”。
  </p>
  <p class="graf graf--p">
   但是，工人中心设计了“工资盗窃”的 meme，直接推翻了主导假设，即：工资是老板的财产，它只是在与工人分享。
  </p>
  <p class="graf graf--p">
   相反，
   <strong class="markup--strong markup--p-strong">
    在这个新的叙述中，工资被强调是工人的财产，并且被老板偷走了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这是一个成功的反击性叙事。
  </p>
  <p class="graf graf--p">
   ⚠️想想看，中国当局一直将可怕的社交媒体情报描述为“舆情分析”，暗示权势在通过社交媒体“了解民意”，事实上这是监视，只有“情报”的叙述才能体现出挖掘和操纵的阴暗。
   <strong class="markup--strong markup--p-strong">
    然而包括知识分子在内的几乎所有人都在沿用当权者的叙述 —— 依附于对手的框架
   </strong>
   ，导致大众一直对这种明目张胆的监视不以为然。
  </p>
  <p class="graf graf--p">
   现实中有很多这样的例子，活动家们应该努力开发它们，争取更全面地夺取叙事框架的权力。⚪️
  </p>
