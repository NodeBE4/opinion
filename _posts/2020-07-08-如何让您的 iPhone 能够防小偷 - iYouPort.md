    <img alt="" class="aligncenter size-full wp-image-12501" data-recalc-dims="1" height="1100" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-12.jpg?resize=1100%2C1100&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-12.jpg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-12.jpg?resize=300%2C300&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-12.jpg?resize=1024%2C1024&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-12.jpg?resize=150%2C150&amp;ssl=1 150w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-12.jpg?resize=768%2C768&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-12.jpg?resize=1100%2C1100&amp;ssl=1 1100w" width="1100"/>

  <p class="graf graf--p">
   丢失或被盗这种事可能随时会发生。被盗 iPhone 的OLED显示屏组件、框架和充电端口等零件在黑市上价格较高。更糟糕的是，如果有人要窃取您的手机，他们只需将其关闭即可避免使用诸如 “Find My iPhone” 之类的防盗功能。
  </p>
  <p class="graf graf--p">
   尽管您的 iPhone 防御也许能抵挡未经授权的访问，但 iOS 仍然存在可以被任何人关闭的固有弱点，这实际上使您无法拨打丢失的电话或跟踪手机。
  </p>
  <p class="graf graf--p">
   为了解决这个问题，
   <a class="markup--anchor markup--p-anchor" data-href="https://ios.gadgethacks.com/how-to/jailbreaking/" href="https://ios.gadgethacks.com/how-to/jailbreaking/" rel="noopener noreferrer" target="_blank">
    jailbreak
   </a>
   越狱社区进行了许多调整，其中一些可以保护您的 iPhone 免受意外关机的影响。挑选两个功能齐全的方法介绍一下。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    方法一：PowerGuard
   </strong>
  </h3>
  <p class="graf graf--p">
   PeterDev 的
   <a class="markup--anchor markup--p-anchor" data-href="https://repo.packix.com/package/com.peterdev.powerguard/" href="https://repo.packix.com/package/com.peterdev.powerguard/" rel="noopener noreferrer" target="_blank">
    PowerGuard
   </a>
   是一项简单的调整，可以保护 iPhone 免受未经授权的关机。它可以通过 Touch ID、FaceID、PIN 或密码来保护 iPhone 的电源菜单（“滑动以关闭电源” 栏）。
  </p>
  <p class="graf graf--p">
   安装后，在 “设置” 中的 “PowerGuard” 菜单中启用。之后，选择您的选项，它会自动进行调整，无需重新设置。
  </p>
  <p class="graf graf--p">
   不幸的是，这种调整并不能阻止其他人强制重启您的 iPhone，因此请记住这一点。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-12502 jetpack-lazy-image" data-lazy-sizes="(max-width: 946px) 100vw, 946px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-7-scaled.jpg?resize=946%2C2048&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-7-scaled.jpg?w=946&amp;ssl=1 946w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-7-scaled.jpg?resize=139%2C300&amp;ssl=1 139w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-7-scaled.jpg?resize=473%2C1024&amp;ssl=1 473w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-7-scaled.jpg?resize=768%2C1663&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-7-scaled.jpg?resize=709%2C1536&amp;ssl=1 709w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-7-scaled.jpg?resize=1100%2C2382&amp;ssl=1 1100w" data-recalc-dims="1" height="2048" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-7-scaled.jpg?resize=946%2C2048&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="946"/>

    <img alt="" class="aligncenter size-full wp-image-12502" data-recalc-dims="1" height="2048" sizes="(max-width: 946px) 100vw, 946px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-7-scaled.jpg?resize=946%2C2048&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-7-scaled.jpg?w=946&amp;ssl=1 946w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-7-scaled.jpg?resize=139%2C300&amp;ssl=1 139w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-7-scaled.jpg?resize=473%2C1024&amp;ssl=1 473w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-7-scaled.jpg?resize=768%2C1663&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-7-scaled.jpg?resize=709%2C1536&amp;ssl=1 709w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-7-scaled.jpg?resize=1100%2C2382&amp;ssl=1 1100w" width="946"/>
   </noscript>
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    方法二：PickPocket 3
   </strong>
  </h3>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://repo.ziph0n.com/depictions/?p=com.ziph0n.pickpocket11" href="http://repo.ziph0n.com/depictions/?p=com.ziph0n.pickpocket11" rel="noopener noreferrer" target="_blank">
    PickPocket 3
   </a>
   是另一个免费的解决方案，可以很好地保护 iPhone 免受未经授权的关机等影响。除了启动关机的标准 TouchID / FaceID 和密码要求之外，ziph0n 还具有巧妙的其他功能，可以进一步增强安全性。
  </p>
  <p class="graf graf--p">
   这些功能包括在控制中心内禁用对飞行模式的访问权限，并设置伪造的关机来欺骗可能的小偷。
  </p>
  <p class="graf graf--p">
   虽然它不能完全阻止硬重置，但它具有 “Fake Hard Reset” 选项，可以模拟操作以减少重置设备的机会。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12503 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-11.png?resize=1100%2C763&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-11.png?w=1202&amp;ssl=1 1202w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-11.png?resize=300%2C208&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-11.png?resize=1024%2C710&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-11.png?resize=768%2C533&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-11.png?resize=1100%2C763&amp;ssl=1 1100w" data-recalc-dims="1" height="763" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-11.png?resize=1100%2C763&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12503" data-recalc-dims="1" height="763" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-11.png?resize=1100%2C763&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-11.png?w=1202&amp;ssl=1 1202w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-11.png?resize=300%2C208&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-11.png?resize=1024%2C710&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-11.png?resize=768%2C533&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-11.png?resize=1100%2C763&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   PickPocket 3 也可以设置为响起警报或自定义语音、对小偷进行秘密拍照、如果您设置了这些它可以通过短信和电子邮件通知您，还有去除不必要的SIM卡、错误的密码或PIN尝试等等。
  </p>
  <p class="graf graf--p">
   您甚至可以在锁定屏幕上显示您的联系信息，以便当好心的人发现丢失的 iPhone 时可以与您联系。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-12504 jetpack-lazy-image" data-lazy-sizes="(max-width: 946px) 100vw, 946px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-5-scaled.jpg?resize=946%2C2048&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-5-scaled.jpg?w=946&amp;ssl=1 946w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-5-scaled.jpg?resize=139%2C300&amp;ssl=1 139w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-5-scaled.jpg?resize=473%2C1024&amp;ssl=1 473w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-5-scaled.jpg?resize=768%2C1663&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-5-scaled.jpg?resize=709%2C1536&amp;ssl=1 709w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-5-scaled.jpg?resize=1100%2C2382&amp;ssl=1 1100w" data-recalc-dims="1" height="2048" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-5-scaled.jpg?resize=946%2C2048&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="946"/>

   它是完全免费使用的，因此，如果您想尝试 PickPocket 3，请确保将 repo.ziph0n.com 添加到 Cydia 或 Sileo 的“源”页面中。⚪️
