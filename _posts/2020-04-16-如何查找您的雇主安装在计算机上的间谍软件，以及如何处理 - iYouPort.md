  <p class="graf graf--p">
   没错，《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e7%9a%84%e8%80%81%e6%9d%bf%e5%8f%af%e8%83%bd%e6%98%af%e6%af%94-facebook%e3%80%81%e8%b0%b7%e6%ad%8c%e6%9b%b4%e5%a4%a7%e7%9a%84%e8%80%81%e5%a4%a7%e5%93%a5/" href="https://www.iyouport.org/%e4%bd%a0%e7%9a%84%e8%80%81%e6%9d%bf%e5%8f%af%e8%83%bd%e6%98%af%e6%af%94-facebook%e3%80%81%e8%b0%b7%e6%ad%8c%e6%9b%b4%e5%a4%a7%e7%9a%84%e8%80%81%e5%a4%a7%e5%93%a5/" rel="noopener noreferrer" target="_blank">
    你的老板可能是比 Facebook、谷歌更大的老大哥
   </a>
   》。越来越多的雇主和IT部门采取极端方法来监视员工，这是当前劳工权益领域最大的议题，因为
   <strong class="markup--strong markup--p-strong">
    没有法律能限制雇主的间谍行为
   </strong>
   。
  </p>
  <p class="graf graf--p">
   作为雇员，您可能无法更改您所在公司的政策，但是至少可以自行检查是否受到了监视以及如何受到监视的。
  </p>
  <p class="graf graf--p">
   显然，这方面无法提供一份万能的指南，因为我们不了解每家公司的内部运作情况，但是我们可以告诉您一些最佳的方法，以了解您的公司是否在计算机上安装了任何间谍软件，以及处理这种情况的选项。
  </p>
  <p class="graf graf--p">
   无论您是否认为这是公平的，根据美国
   <a class="markup--anchor markup--p-anchor" data-href="https://www.worktime.com/usa-employee-monitoring-laws-what-can-and-cant-employers-do-in-the-workplace" href="https://www.worktime.com/usa-employee-monitoring-laws-what-can-and-cant-employers-do-in-the-workplace" rel="noopener noreferrer" target="_blank">
    法律
   </a>
   ，雇主可以在公司硬件上监视员工，甚至在某些情况下记录击键那种黑客行为也是合法的。
  </p>
  <p class="graf graf--p">
   虽然雇主有可能会通知员工正在进行的监视，但 “知情” 仅在少数地区是必需的，因此，监视很可能在您不知情的情况下发生。
  </p>
  <p class="graf graf--p">
   如果您已经被告知正在受到监视，那么就不需要本指南了。您甚至可以直接在屏幕一角看到正在运行的监视软件。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10839 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-27.png?resize=1100%2C598&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-27.png?w=1596&amp;ssl=1 1596w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-27.png?resize=300%2C163&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-27.png?resize=1024%2C557&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-27.png?resize=768%2C418&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-27.png?resize=1536%2C835&amp;ssl=1 1536w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-27.png?resize=1100%2C598&amp;ssl=1 1100w" data-recalc-dims="1" height="598" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-27.png?resize=1100%2C598&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   如果您没有被告知，检查的最佳方法就是查看Windows系统上正在运行的内容，您可以使用
   <strong class="markup--strong markup--p-strong">
    Ctrl + Shift + Esc
   </strong>
   进行调用（或通过在任务栏中进行搜索）。
  </p>
  <p class="graf graf--p">
   切换到 “
   <strong class="markup--strong markup--p-strong">
    Processes
   </strong>
   ” 选项卡：大多数有价值的间谍软件都可以在“应用程序”列表中隐藏，但请检查后台进程，尤其是对于占用大量系统资源的程序。
  </p>
  <p class="graf graf--p">
   通过网络快速搜索您不认识的任何东西，可能会发现一个或两个已知间谍软件应用程序的名称 —— 通过您的私人电话进行搜索，
   <strong class="markup--strong markup--p-strong">
    不能使用公司 wifi
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <strong class="markup--strong markup--p-strong">
    Start-up
   </strong>
   ” 选项卡也值得研究：每次计算机重新启动时，都需要将监视软件加载到内存中，因此应在此处列出。同样，您可以查找所有无法识别的晦涩名字，那也许是可疑的东西。
  </p>
  <p class="graf graf--p">
   通过 Windows 命令提示符，您可以检查哪些应用和实用程序正在访问Web。在任务栏中搜索“ cmd”，然后在结果中右键单击 “
   <strong class="markup--strong markup--p-strong">
    Command Prompt
   </strong>
   ”，选择“以管理员身份运行”（也许行、也可能无法在公司计算机上执行此操作）。
  </p>

  <p class="graf graf--p">
   现在您正在查看的是从计算机访问互联网的所有可执行文件（程序）。同样，可能需要做一些侦探工作才能识别出奇观的东西 —— 不仅是标有“ chrome.exe”或同样易于理解的内容。
  </p>
  <p class="graf graf--p">
   另一种选择是在计算机上运行某种按需的防病毒软件或反间谍软件程序包，但是如果在公司计算机上可以安装的内容有限，显然会遇到问题。并且，某些监控工具可以通过这种方式被检测到，而有些则无法检测；不论如何如果您有权安装的话，值得一试。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.emsisoft.com/en/home/emergencykit/" href="https://www.emsisoft.com/en/home/emergencykit/" rel="noopener noreferrer" target="_blank">
    Emsisoft Emergency Kit
   </a>
   是其中值得选择的一种，因为它免费、轻巧且易于运行。再比如趋势科技的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.trendmicro.com/en_gb/forHome/products/housecall.html" href="https://www.trendmicro.com/en_gb/forHome/products/housecall.html" rel="noopener noreferrer" target="_blank">
    HouseCall
   </a>
   也可以，但它更适合于试图造成实际损害的那种恶意软件，而不是我们在此谈论的雇主间谍软件。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.superantispyware.com/" href="https://www.superantispyware.com/" rel="noopener noreferrer" target="_blank">
    SuperAntiSpyware
   </a>
   是另一个有效的程序，它是免费的可运行手动扫描。
  </p>
  <p class="graf graf--p">
   如果您的雇主给您的机器不是 Windows 而是Mac，原理也是相同的。使用 Activity Monitor 代替 Task Manager，可以从 Spotlight 搜索中调用（点击
   <strong class="markup--strong markup--p-strong">
    Cmd + Space
   </strong>
   即可调出 Spotlight）。
  </p>

  <p class="graf graf--p">
   尽管网络也值得检查，但“CPU”选项卡是一个不错的起点：在这里，您可以看到在macOS上运行的所有内容，因此，您可以挑选出自己了解的所有应用和实用程序，以及那些可疑的应用程序和实用程序。
  </p>
  <p class="graf graf--p">
   如果其中有您无法确定的东西，请通过
   <strong class="markup--strong markup--p-strong">
    自己的手机（不要连接公司网络）或回到家后对它们进行搜索
   </strong>
   。
  </p>
  <p class="graf graf--p">
   要检查哪些应用程序正在使用macOS启动，您需要前往其他地方。在Apple菜单中，单击系统偏好设置，然后单击
   <strong class="markup--strong markup--p-strong">
    Users &amp; Groups
   </strong>
   . 在您的用户名下方，您会看到一个登录项标签，其中会显示在重启时始终启动的程序。
  </p>
  <p class="graf graf--p">
   间谍程序在这里并不总是可见的，但是您可能会看到一些东西。
  </p>
  <p class="graf graf--p">
   然后打开终端，在这里有很多选择，但是尤其需要提到的是
   <a class="markup--anchor markup--p-anchor" data-href="http://osxdaily.com/2011/04/29/show-what-apps-processes-are-use-internet-connection-command-line/" href="http://osxdaily.com/2011/04/29/show-what-apps-processes-are-use-internet-connection-command-line/" rel="noopener noreferrer" target="_blank">
    OSXDaily
   </a>
   一个简单而有用的选择：Type: [ lsof -nPi | cut -f 1 -d “ “| uniq | tail -n +2 ] 查看在线的每个应用程序的名称。
  </p>
  <p class="graf graf--p">
   与 Windows 一样，还有第三方应用程序可以帮助您进行间谍软件搜寻。
  </p>
  <p class="graf graf--p">
   前面提到的趋势科技
   <a class="markup--anchor markup--p-anchor" data-href="https://www.trendmicro.com/en_gb/forHome/products/housecall.html" href="https://www.trendmicro.com/en_gb/forHome/products/housecall.html" rel="noopener noreferrer" target="_blank">
    HouseCall
   </a>
   可在Mac上使用，并且可以为您进行简单的按需扫描，当然，免费的
   <a class="markup--anchor markup--p-anchor" data-href="https://objective-see.com/products/knockknock.html" href="https://objective-see.com/products/knockknock.html" rel="noopener noreferrer" target="_blank">
    KnockKnock
   </a>
   也值得拥护：它可以找到随macOS启动的应用程序，以及您可能不知道的插件、脚本、扩展和其他加载项，从而使其成为出色的间谍软件猎人。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10842 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-16.png?resize=1100%2C611&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-16.png?w=1594&amp;ssl=1 1594w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-16.png?resize=300%2C167&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-16.png?resize=1024%2C569&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-16.png?resize=768%2C427&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-16.png?resize=1536%2C854&amp;ssl=1 1536w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-16.png?resize=1100%2C611&amp;ssl=1 1100w" data-recalc-dims="1" height="611" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-16.png?resize=1100%2C611&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10842" data-recalc-dims="1" height="611" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-16.png?resize=1100%2C611&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-16.png?w=1594&amp;ssl=1 1594w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-16.png?resize=300%2C167&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-16.png?resize=1024%2C569&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-16.png?resize=768%2C427&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-16.png?resize=1536%2C854&amp;ssl=1 1536w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-16.png?resize=1100%2C611&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   还值得一提的是一个名为
   <a class="markup--anchor markup--p-anchor" data-href="https://apps.apple.com/us/app/etrecheck/id1423715984?mt=12" href="https://apps.apple.com/us/app/etrecheck/id1423715984?mt=12" rel="noopener noreferrer" target="_blank">
    EtreCheck
   </a>
   的很棒的小程序，它可以在Mac的内部进行窥视，以报告硬件和软件问题以及您可能不知道的任何不受欢迎的软件。它可以标记其他实用程序可能未发现的潜在可疑问题。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://macpaw.com/cleanmymac" href="https://macpaw.com/cleanmymac" rel="noopener noreferrer" target="_blank">
    CleanMyMac X
   </a>
   的价格为90美元（或每年40美元），但这是一个非常全面的清理、维护和安全程序，可以确保Mac安全并在未来几年内保持最高速度。
  </p>
  <p class="graf graf--p">
   但是，与Windows一样，您可能无法在公司Mac上安装自己的软件。这是个问题。
  </p>
  <p class="graf graf--p">
   雇主/雇员之间的关系可能很棘手，也许您仍然需要做很多工作来考虑这件事。如果您确实怀疑在不知情的情况下已被安装了某些内容，那么以上提示应该有助于您进行查找。
  </p>
  <p class="graf graf--p">
   必须说，如果您确实发现了间谍软件，那么您所处的位置可能比不知道时更加尴尬。
   <strong class="markup--strong markup--p-strong">
    因为在美国没有法律保护你，中国在这方面什么情况我们不知道，但任何情况都不值得惊讶了。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️您唯一能做的是就是不要把与自己私人的任何想法和需求带入公司的电子设备、包括网络！不要用公司网络连接私人手机！还有
   <strong class="markup--strong markup--p-strong">
    一系列自我保护措施
   </strong>
   ，详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e7%9a%84%e8%80%81%e6%9d%bf%e5%8f%af%e8%83%bd%e6%98%af%e6%af%94-facebook%e3%80%81%e8%b0%b7%e6%ad%8c%e6%9b%b4%e5%a4%a7%e7%9a%84%e8%80%81%e5%a4%a7%e5%93%a5/" href="https://www.iyouport.org/%e4%bd%a0%e7%9a%84%e8%80%81%e6%9d%bf%e5%8f%af%e8%83%bd%e6%98%af%e6%af%94-facebook%e3%80%81%e8%b0%b7%e6%ad%8c%e6%9b%b4%e5%a4%a7%e7%9a%84%e8%80%81%e5%a4%a7%e5%93%a5/" rel="noopener noreferrer" target="_blank">
    你的老板可能是比 Facebook、谷歌更大的老大哥
   </a>
   》。⚪️
  </p>
