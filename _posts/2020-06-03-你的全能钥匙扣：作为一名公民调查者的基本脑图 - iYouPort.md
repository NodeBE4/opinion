  <p class="graf graf--p">
   我们的 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    开源情报
   </a>
   ” 栏目推荐过很多工具，也许您已经有点眼花缭乱了。
  </p>
  <p class="graf graf--p">
   其实成功完成一项调查所需要的是一套组织良好的工具 —— 用科学的逻辑组织在一起，而不是一大堆看似很丰富但杂乱无章的东西。
  </p>
  <p class="graf graf--p">
   调酒师的钥匙扣上只有几个小玩意，但是它们能随时打开所有不同封装的酒瓶，完成绝大多数工作。作为公民调查者，这就是您应该追求的：不需要很多，但必须很趁手；不需要特别高级，但必须实用。
  </p>
  <p class="graf graf--p">
   这就需要您平日将最顺手的东西按照合理的逻辑组织在一起，也就是栓在您的钥匙扣上。
  </p>
  <p class="graf graf--p">
   本文将介绍几个平台，帮您打造这个钥匙扣。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、列表和电子表格
   </strong>
  </h3>
  <p class="graf graf--p">
   显然，电子表格通常是跟踪大量链接和工具的首选解决方案，因为很简单，所有人都会用。
  </p>
  <p class="graf graf--p">
   您可以根据主题（用户名、电话号码等）及其用途，在电子表格或Word中创建一系列工具。
  </p>
  <p class="graf graf--p">
   电子表格的缺点是维护有点累，一旦列表变大就会麻烦；而且你需要不断删除已经作废的工具和链接（因为技术在不断变化）。
  </p>
  <p class="graf graf--p">
   但如果您已经有了一系列理想的基本配套选项，电子表格会比较适合您。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11905 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-25.png?resize=1100%2C810&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-25.png?w=1157&amp;ssl=1 1157w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-25.png?resize=300%2C221&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-25.png?resize=1024%2C754&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-25.png?resize=768%2C566&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-25.png?resize=1100%2C810&amp;ssl=1 1100w" data-recalc-dims="1" height="810" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-25.png?resize=1100%2C810&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、个性化思维导图
   </strong>
  </h3>
  <p class="graf graf--p">
   您还可以使用个性化思维导图，就像 OSINT Framework 那样的方式存储工具。
  </p>
  <p class="graf graf--p">
   如果您还不熟悉开源情报框架，可以
   <a class="markup--anchor markup--p-anchor" data-href="https://osintframework.com/" href="https://osintframework.com/" rel="noopener noreferrer" target="_blank">
    在这里找到它
   </a>
   ，它本身就是一个很了不起的资源。
  </p>
  <p class="graf graf--p">
   使用思维导图的优势在于，非常直观，连工具带思路一眼就能看出来。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.xmind.net/" href="https://www.xmind.net/" rel="noopener noreferrer" target="_blank">
    Xmind
   </a>
   是创建思维导图的绝佳工具，它提供免费和付费版本。
  </p>
  <p class="graf graf--p">
   免费版本功能会受限，而且在导出的文件上带有水印，但如果您是个人调查使用，它就是很好的（写报告的话最好使用付费版）。
  </p>
  <p class="graf graf--p">
   这种方法的另一个好处是思维导图是可共享的，适合团队协作。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11906 jetpack-lazy-image" data-lazy-sizes="(max-width: 749px) 100vw, 749px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-19.png?resize=749%2C791&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-19.png?w=749&amp;ssl=1 749w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-19.png?resize=284%2C300&amp;ssl=1 284w" data-recalc-dims="1" height="791" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-19.png?resize=749%2C791&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="749"/>

    3、Pocket
   </strong>
  </h3>
  <p class="graf graf--p">
   另一个流行解决方案是
   <a class="markup--anchor markup--p-anchor" data-href="https://getpocket.com/" href="https://getpocket.com/" rel="noopener noreferrer" target="_blank">
    Pocket
   </a>

  </p>
  <p class="graf graf--p">
   该站点允许您将文章/链接保存到个性化列表中，以供日后阅读。
  </p>
  <p class="graf graf--p">
   缺点是免费版本限制了您可以保存到列表中的文章数量。
  </p>
  <p class="graf graf--p">
   这个网站似乎类似于 Pinterest，但 Pinterest 是免费的。您可以根据需求选择。

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、书签
   </strong>
  </h3>
  <p class="graf graf--p">
   另一种方法是在网络浏览器中简单地将它们添加书签。
  </p>
  <p class="graf graf--p">
   在 Buscador 或 Kali 中，打开浏览器并在书签中为每个调查方案创建一个文件夹。
  </p>
  <p class="graf graf--p">
   如果您安装了 Buscador，那肯定知道，已经有大量有用的工具在手边，而不必搜索长列表了。
  </p>
