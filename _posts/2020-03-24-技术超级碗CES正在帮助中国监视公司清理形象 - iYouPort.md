  <p class="graf graf--p">
   Ezviz 成立于2013年，被描述为一家智能家居安全公司，并在
   <a class="markup--anchor markup--p-anchor" data-href="https://ces20.mapyourshow.com/8_0/exhibitor/exhibitor-details.cfm?ExhID=T0004269" href="https://ces20.mapyourshow.com/8_0/exhibitor/exhibitor-details.cfm?ExhID=T0004269" rel="noopener noreferrer" target="_blank">
    CES目录
   </a>
   中列为位于加利福尼亚州的企业。
  </p>
  <p class="graf graf--p">
   它提供面部识别等功能来识别门外的人，并提供可与语音助手（包括亚马逊的Alexa）集成的视频门铃等小工具。
  </p>
  <p class="graf graf--p">
   该公司还提供可以创建您的朋友和家人数据库的技术。 Ezviz 在2016年表示
   <strong class="markup--strong markup--p-strong">
    拥有超过1000万用户
   </strong>
   。
  </p>

  <p class="graf graf--p">
   该公司在其网站上说：“ EZVIZ 通过其智能设备、基于云的平台和AI技术为用户创造了安全、便捷和智能的生活。”
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">

    </strong>
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e5%ae%b6%e5%86%b0%e7%ae%b1%e5%a6%82%e4%bd%95%e4%bd%9c%e4%b8%ba%e5%91%88%e5%a0%82%e8%af%81%e4%be%9b%ef%bc%9f/" href="https://www.iyouport.org/%e4%bd%a0%e5%ae%b6%e5%86%b0%e7%ae%b1%e5%a6%82%e4%bd%95%e4%bd%9c%e4%b8%ba%e5%91%88%e5%a0%82%e8%af%81%e4%be%9b%ef%bc%9f/" rel="noopener noreferrer" target="_blank">

  <p class="graf graf--p">
   Ezviz 对与母公司的关系持开放态度，但在
   <strong class="markup--strong markup--p-strong">
    其网站上并未提及海康威视在中国侵犯人权的历史。
   </strong>
  </p>
  <p class="graf graf--p">
   美国商务部于去年10月7日将海康威视与 iFlytek 一起列入了经济黑名单，理由是有证据表明他们正在帮助压迫新疆地区的维吾尔族穆斯林。
  </p>
  <p class="graf graf--p">
   商务部在一份声明中说：“特别是，这些实体牵涉到中国实施镇压运动、大规模任意拘留、以及对维吾尔人、哈萨克人和其他穆斯林少数民族成员进行高科技监视的运动中的侵犯人权行为。”
  </p>
  <p class="graf graf--p">
   海康威视已获得超过2.7亿美元的合同，在中国提供数千个监视摄像头，其中许多被安放在拥有超过100万维吾尔族穆斯林的清真寺和拘留营中。
  </p>
  <p class="graf graf--p">
   英国政府也知道海康威视的声誉。 3月，国会议员要求该公司提供在新疆和西藏的监视合同，称这是对人权的滥用。
  </p>
  <p class="graf graf--p">
   立法者批评海康威视的监视和面部识别能力，并写道：“这种技术的复杂性及其危险的使用是引起人们严重关注的严重原因。”
  </p>
  <p class="graf graf--p">
   据人权观察组织称，
   <strong class="markup--strong markup--p-strong">
    iFlytek 正在与中国警方合作建立语音识别数据库，警务人员可用来识别电话中任何目标人的身份
   </strong>
   。

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    CES 奖励赢家
   </strong>
  </p>
  <p class="graf graf--p">
   人权倡导者说，鉴于海康威视和 iFlytek 被公开的有关侵犯人权行为的历史，CES 决定为两家公司提供一个展示其技术并向其授予创新奖的平台的决定令人担忧。
  </p>
  <p class="graf graf--p">
   人权观察的报告发布两年后，iFlytek 在CES上获得了“科技促进更美好世界”类别的奖项。
  </p>
  <p class="graf graf--p">
   海康威视的子公司 Ezviz 在2018年凭借其所谓的智能门镜赢得了创新奖，该智能门镜使用户能够创建 “可信任访客” 的数据库。
  </p>
  <p class="graf graf--p">
   当在10月9日被问及有关 iFlytek 的担忧时，CTA表示 ‘需要时间采取行动“，因为美国政府的实体清单仅在两天前发布；但批评者在指出，Ezviz 和海康威视与侵犯人权的关系已经众所周知多年了。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “为一家据证明在中国受到严重压制的地区卷入侵犯人权行为的公司授予奖项，似乎是不合适的，我希望组织者对这一决定进行评估。”
  </p>
  <p class="graf graf--p">
   CTA没有评论授予 iFlytek 和 Ezviz 的奖项的问题，但是
   <strong class="markup--strong markup--p-strong">
    两家公司一直在使用该组织的认可来促进其业务发展。
   </strong>
  </p>
  <p class="graf graf--p">
   Ezviz 总经理 Albert Lin 在2018年获奖后的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.prnewswire.com/news-releases/ezviz-named-as-ces-2018-innovation-awards-honoree-300577995.html" href="https://www.prnewswire.com/news-releases/ezviz-named-as-ces-2018-innovation-awards-honoree-300577995.html" rel="noopener noreferrer" target="_blank">
    一份声明中说
   </a>
   ：“我们很荣幸获得CTA这样的组织的认可。EZVIZ 致力于提供直观的智能家居产品和应用程序，使客户能够看到、捕获、共享和保护对他们最有价值的东西。”
  </p>
  <p class="graf graf--p">
   iFlytek 在2019年1月告诉
   <a class="markup--anchor markup--p-anchor" data-href="http://www.xinhuanet.com/english/northamerica/2019-01/13/c_137739486.htm" href="http://www.xinhuanet.com/english/northamerica/2019-01/13/c_137739486.htm" rel="noopener noreferrer" target="_blank">
    一家中国新闻社
   </a>
   ，它认为CES “是展示我们创新能力的绝佳平台”。
  </p>
  <p class="graf graf--p">
   监控技术监督项目主管 Albert Fox Cahn 说，不应允许这些公司在大型技术展览会上展示其技术和产品，并清理其公众形象。
  </p>
  <p class="graf graf--p">
   Cahn 的非营利组织致力于制止歧视性监视。他说，“当一家公司出售这项技术以延续过去50年来最严重的侵犯人权行为之一时，它们应该在我们的消费者经济中失去任何地位。” ⚪️
  </p>
