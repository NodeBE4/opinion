     面部识别与图像搜索
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   这里的新事物是：Yandex 不仅可以用作开源情报调查中常规的反向图像搜索工具，还可以用来实施面部匹配。
  </p>
  <p class="graf graf--p">
   它找到的所有图像看起来都不像原始图像。不同的宽高比、不同的背景、不同的细节 …… 但是，

    脸部是匹配的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️它甚至通过本案例测试者的自拍照找到了其本人年轻时没留胡子的版本！还找到了目标人手捧盒子遮住部分脸部的版本！简直太疯狂了。
  </p>
  <p class="graf graf--p">
   那么其他反向图像搜索平台在面部识别角度上的效果如何呢？就如我们👆关联文章中所做的效果比较。
  </p>

  <p class="graf graf--p">
   百度的结果：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12481 jetpack-lazy-image" data-lazy-sizes="(max-width: 766px) 100vw, 766px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-3.jpg?resize=766%2C622&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-3.jpg?w=766&amp;ssl=1 766w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-3.jpg?resize=300%2C244&amp;ssl=1 300w" data-recalc-dims="1" height="622" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-3.jpg?resize=766%2C622&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="766"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12481" data-recalc-dims="1" height="622" sizes="(max-width: 766px) 100vw, 766px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-3.jpg?resize=766%2C622&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-3.jpg?w=766&amp;ssl=1 766w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-3.jpg?resize=300%2C244&amp;ssl=1 300w" width="766"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Bing 的结果：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12482 jetpack-lazy-image" data-lazy-sizes="(max-width: 765px) 100vw, 765px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7-3.jpg?resize=765%2C862&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7-3.jpg?w=765&amp;ssl=1 765w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7-3.jpg?resize=266%2C300&amp;ssl=1 266w" data-recalc-dims="1" height="862" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7-3.jpg?resize=765%2C862&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="765"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12482" data-recalc-dims="1" height="862" sizes="(max-width: 765px) 100vw, 765px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7-3.jpg?resize=765%2C862&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7-3.jpg?w=765&amp;ssl=1 765w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7-3.jpg?resize=266%2C300&amp;ssl=1 266w" width="765"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   看看图像中那些同样有着短头发的白人，
   <strong class="markup--strong markup--p-strong">
    全部都不是目标人
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这些算法推测还算是不错的，就是说，给了它们一个头像，然后它们发现了类似的头像：短发、男性、白人。但是，都不是目标人。
  </p>
  <p class="graf graf--p">
   👉只有 Yandex 找到了目标人的脸。 （可以说，TinEye 的0个结果是该查询的正确答案；因为这张图片并不在网上）
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     为什么这很重要
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   面部识别算法在日常隐私方面正处于重大变革的风口浪尖。AI 现在已经足够好，可以进行基本准确的面部识别。
   <strong class="markup--strong markup--p-strong">
    现代软件可以拍摄一张只有半边脸的模糊照片，依旧可以成功将其与同一个人的其他脸部照片相匹配。
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e9%81%ae%e4%bd%8f%e4%b8%80%e5%8d%8a%e8%84%b8%e6%b2%a1%e6%9c%89%e7%94%a8%ef%bc%9a%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e5%8d%87%e7%ba%a7%e7%89%88%e8%af%9e%e7%94%9f/" href="https://www.iyouport.org/%e9%81%ae%e4%bd%8f%e4%b8%80%e5%8d%8a%e8%84%b8%e6%b2%a1%e6%9c%89%e7%94%a8%ef%bc%9a%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e5%8d%87%e7%ba%a7%e7%89%88%e8%af%9e%e7%94%9f/" rel="noopener noreferrer" target="_blank">
     遮住一半脸没有用：“面部识别”升级版诞生
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   这是一个相当可怕的未来，尤其是像新疆这样的专制政府已经建立的未来，但这可能是难以阻挡的。
  </p>
  <p class="graf graf--p">
   重要的是，目前普通百姓仍然无法使用陌生人的照片并找到他或她的名字。也就是说，
   <strong class="markup--strong markup--p-strong">
    技术不是民主的 —— 它只掌握在当权者手里
   </strong>
   。也是为什么我们开始思考让技术民主化的可能性，如下：（既然难以阻止其发展）
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%ba%e8%84%b8%e8%af%86%e5%88%ab%e5%8f%af%e4%bb%a5%e8%87%aa%e5%88%b6%ef%bc%8c%e4%bd%a0%e7%9f%a5%e9%81%93%e5%90%97%ef%bc%9f%e8%80%8c%e4%b8%94%e9%9d%9e%e5%b8%b8%e7%ae%80%e5%8d%95/" href="https://www.iyouport.org/%e4%ba%ba%e8%84%b8%e8%af%86%e5%88%ab%e5%8f%af%e4%bb%a5%e8%87%aa%e5%88%b6%ef%bc%8c%e4%bd%a0%e7%9f%a5%e9%81%93%e5%90%97%ef%bc%9f%e8%80%8c%e4%b8%94%e9%9d%9e%e5%b8%b8%e7%ae%80%e5%8d%95/" rel="noopener noreferrer" target="_blank">
     人脸识别可以自制，你知道吗？而且非常简单
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e4%bd%bf%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e6%8a%80%e6%9c%af%e4%b8%ba%e5%85%ac%e6%b0%91%e6%89%80%e7%94%a8%ef%bc%9f-%e5%bc%80%e6%ba%90%e6%83%85%e6%8a%a5%e6%96%b9%e6%b3%95/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e4%bd%bf%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e6%8a%80%e6%9c%af%e4%b8%ba%e5%85%ac%e6%b0%91%e6%89%80%e7%94%a8%ef%bc%9f-%e5%bc%80%e6%ba%90%e6%83%85%e6%8a%a5%e6%96%b9%e6%b3%95/" rel="noopener noreferrer" target="_blank">
     如何使面部识别技术为公民所用？- 开源情报方法
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   但是，即便您在技术方面毫无基础，Yandex 也有可能帮你做到这点。
  </p>
  <p class="graf graf--p">
   Yandex 已有很长的历史了，是世界上为数不多的可与谷歌竞争的公司之一。他们的索引严重偏向东欧数据，但是，实验显示，他们有足够的全球数据来找到美国人 nelsonslog 和台湾裔 Andrew Yang。
  </p>
  <p class="graf graf--p">
   如果您使用 Google 相册或 Facebook，则很有可能已经领教了他们的面部识别能力。匹配效果足够惊人，而且杂音不多。
  </p>
  <p class="graf graf--p">
   Facebook 似乎只会为您显示与您有某种 Facebook 关系的人的名字。Google相簿也类似地提供随机名字。他们可以做得更多；例如，Facebook 可以将脸部与任何 Facebook 用户匹配。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    这两种服务都不能作为识别陌生人的通用面部识别服务
   </strong>
   。
  </p>
  <p class="graf graf--p">
   至少有一个可以做到同样事的公共服务：FindClone （它以前被称为 FindFace 和 SearchFace）。但是，仅限于来自俄罗斯 VKontakte 的照片。这的确是一个很大的数据集。但这并不像 Yandex 搜索引擎那样国际化。
  </p>
  <p class="graf graf--p">
   预计在几年之内，这种 “按照面孔查找名字” 的使用功能将和 “用电话号码查找名字” 一样的普遍。
  </p>
  <p class="graf graf--p">
   ⚠️ 所以，再次强调！除非您是明星，靠脸吃饭的那种，否则
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e6%9b%be%e7%bb%8f%e5%9c%a8%e7%a4%be%e4%ba%a4%e7%bd%91%e7%bb%9c%e4%b8%8a%e5%8f%91%e5%b8%83%e8%bf%87%e8%87%aa%e5%b7%b1%e7%9a%84%e7%85%a7%e7%89%87%ef%bc%9f%e4%bd%a0%e5%8d%b1%e9%99%a9%e4%ba%86/" href="https://www.iyouport.org/%e4%bd%a0%e6%9b%be%e7%bb%8f%e5%9c%a8%e7%a4%be%e4%ba%a4%e7%bd%91%e7%bb%9c%e4%b8%8a%e5%8f%91%e5%b8%83%e8%bf%87%e8%87%aa%e5%b7%b1%e7%9a%84%e7%85%a7%e7%89%87%ef%bc%9f%e4%bd%a0%e5%8d%b1%e9%99%a9%e4%ba%86/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     尽可能不要以任何方式将您的脸部照片/视频放到网上
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ！包括您认识的所有人发布的包括您的照片！这非常重要
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果您上传自己的脸部在 Yandex 搜到了您本人的任何信息，迅速删除它们所有。⚪️
  </p>
