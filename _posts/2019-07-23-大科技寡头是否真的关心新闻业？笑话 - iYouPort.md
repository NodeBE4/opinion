  <p class="graf graf--p">
   苹果在宣布 News+ 的时候，虚弱的传统媒体表现出一片兴高采烈，仿佛由此看到了活下来的希望。但这真的是好事吗？
  </p>
  <p class="graf graf--p">
   GOOGLE 宣布它将与 McClatchy 合作，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.axios.com/google-local-news-sites-funding-mcclatchy-df9988e5-b0ad-42dc-b579-ff014e98467b.html" href="https://www.axios.com/google-local-news-sites-funding-mcclatchy-df9988e5-b0ad-42dc-b579-ff014e98467b.html" rel="noopener noreferrer" target="_blank">
    为约50万人的社区中的三个新的本地新闻实体提供资金
   </a>
   。这标志着新闻业被吸收到大型科技企业边缘的速度发生了变化。
  </p>
  <p class="graf graf--p">
   有史以来的第一次，一家大型科技公司正在与新闻主管直接合作，建立一个本地新闻业务，并建议为此提供资金。
  </p>
  <p class="graf graf--p">
   McClatchy 首席执行官 Craig Forman 将这项努力描述为一个真正的“合作”，McClatchy 团队将与谷歌的专家一起工作。虽然谷歌“在财务上帮助支持这项工作，”Forman 在谷歌博客的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.blog.google/outreach-initiatives/google-news-initiative/mcclatchy-and-google-partner-experimental-lab-local-news/" href="https://www.blog.google/outreach-initiatives/google-news-initiative/mcclatchy-and-google-partner-experimental-lab-local-news/" rel="noopener noreferrer" target="_blank">
    新闻稿
   </a>
   中写道，“这些网站将100％由 McClatchy 拥有和运营，McClatchy 将保持对内容的唯一编辑控制权和所有权。
  </p>
  <p class="graf graf--p">
   如果有任何疑问，他重申“Google 将不会参与或干预任何编辑工作或决策。”
  </p>
  <p class="graf graf--p">
   很难知道谷歌专家与 McClatchy 员工合作而“没有任何编辑意见”会是什么样子。这种事是天方夜谭。
   <strong class="markup--strong markup--p-strong">
    在网上建立成功的新闻产品的每个人都知道，应用于新闻的技术架构、工具、软件和分析不可避免地最终会影响编辑内容的各个方面
   </strong>
   。
  </p>
  <p class="graf graf--p">
   实际上，众所周知，新闻编辑室中最常见的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cjr.org/tow_center/newsroom-consolidation-product.php" href="https://www.cjr.org/tow_center/newsroom-consolidation-product.php" rel="noopener noreferrer" target="_blank">
    “错误”之一
   </a>
   是未能将“产品”正确地整合到新闻编辑室中，或者没能恰当地考虑到他们出版的技术环境。这就是
   <strong class="markup--strong markup--p-strong">
    资助的绑架
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在过去十年中，没有哪家公司比谷歌更多地资助和“支持”新闻业。只要你参加新闻发布会、听到新闻编辑室的实验、甚至只是看新闻研究，就很难躲开谷歌的名字。
   <strong class="markup--strong markup--p-strong">
    谷歌的数字新闻计划从欧洲扩展到美国，该计划已经在欧洲的新闻创新上
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://blog.google/around-the-globe/google-europe/digital-news-innovation-fund-three-years-and-662-total-projects-supported/" href="https://blog.google/around-the-globe/google-europe/digital-news-innovation-fund-three-years-and-662-total-projects-supported/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     花费了超过一亿欧元
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，迄今为止已经对“美国报道” — — 以及现在的 McClatchy 计划 — — 进行了高调的投资
   </strong>
   。
  </p>
  <p class="graf graf--p">
   由于出版物仍然只能获得如此少的广告资金，而且来自读者的收入一直无法弥补这种亏空，
   <strong class="markup--strong markup--p-strong">
    新闻编辑室的创新工作越来越意味着要求思考“谷歌想要什么？”而不是公民想要什么
   </strong>
   — — 谷歌一直在影响新闻编辑室的选择，从虚拟现实到语音、再到照片库。
  </p>
  <p class="graf graf--p">
   但问题在于谷歌现在对新闻业的许多方面都不满意。该公司的引流能力明显高于一些竞争对手，尤其是 Facebook。这部分是因为它更加成熟，并且可以更好地处理与媒体的关系（例如，它
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2018/11/14/technology/facebook-data-russia-election-racism.html" href="https://www.nytimes.com/2018/11/14/technology/facebook-data-russia-election-racism.html" rel="noopener noreferrer" target="_blank">
    没有试图隐藏
   </a>
   自己的影响力宣传活动）。
  </p>
  <p class="graf graf--p">
   它也很能花钱。谷歌为新闻业提供的额外资金并不是直接购买或支持异议，但它肯定会让新闻CEO和编辑将谷歌与其他平台区别开。他们的态度是“Google get it”。
  </p>
  <p class="graf graf--p">
   在谷歌新闻的早期，当我负责卫报的网站和数字产品的编辑时，我们从与谷歌的密切关系中获的了利益，因为
   <strong class="markup--strong markup--p-strong">
    谷歌了解如何为搜索算法*优化*文章的元数据
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是，谷歌涉足本地新闻业不仅仅是一个“帮助”的问题。当该公司在欧洲推出 DNI 时，它直接回应了欧盟监管机构的压力。
   <strong class="markup--strong markup--p-strong">
    这笔钱是从营销预算中分配出来的，相当于游说活动
   </strong>
   。现在，谷歌正在将其直接资金投入美国。它是在2020年总统大选之前这样做的，此时民主党的领域正在集结（暗示该巨头的不透明算法具有党派偏好）。
  </p>
  <p class="graf graf--p">
   在总统竞选中，监管技术平台首次成为选民的问题，并且至少是一位认真的候选人伊丽莎白沃伦的议程，并且长期以来也是另一个人 Amy Klobuchar 感兴趣的领域。
  </p>
  <p class="graf graf--p">
   那些为了打破全球监视资本主义巨头 Facebook、亚马逊和 Alphabets（谷歌的控股公司）而重新考虑反托拉斯法案的人正在获得动力。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    这项开创性工作的智库 — — Open Markets Initiative 在2017年新美国基金会（New America Foundation）中
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2017/08/30/us/politics/eric-schmidt-google-new-america.html" href="https://www.nytimes.com/2017/08/30/us/politics/eric-schmidt-google-new-america.html" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     失去了
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    它的主导权，当时它引起了主要资助者谷歌不满的紧张局面
   </strong>
   。如果说谷歌增加对新闻业的资助完全与第一个监管技术平台的重大举措有关，有人信吗？是不是听起来像阴谋论呢？
  </p>
  <p class="graf graf--p">
   不过要知道，科技公司突然关注新闻业的金融稳定性并非巧合。正如我的同事 Mathew Ingram
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cjr.org/the_new_gatekeepers/facebook-journalism.php" href="https://www.cjr.org/the_new_gatekeepers/facebook-journalism.php" rel="noopener noreferrer" target="_blank">
    报道
   </a>
   的那样，Facebook 最近在丹佛召开了自己的本地新闻发布会（会议 WiFi 密码：m0vefast）。库克则在该公司的半年度
   <a class="markup--anchor markup--p-anchor" data-href="https://www.apple.com/apple-events/march-2019/" href="https://www.apple.com/apple-events/march-2019/" rel="noopener noreferrer" target="_blank">
    营销演讲
   </a>
   中花了数分钟的时间在宣布推出 Apple News +之前宣称他对新闻业的热爱，这一产品将你的8,000美元的杂志支出减少到10美元。
  </p>
  <p class="graf graf--p">
   我花了很多时间与为科技公司工作的新闻主管合作，我可以说有很多人对这些公司的新闻有所了解和关心。
   <strong class="markup--strong markup--p-strong">
    他们中的许多人都是聪明且有成就的记者，他们真正热衷于改善新闻业。但他们已经被降级到了边缘部门
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    平台的核心是软件工程；他们是新闻业的核心，而新闻业不是他们的核心
   </strong>
   。像 Newsgeist 和 Facebook 新闻项目会议这样的高级会议，在技术业务的中心位置，高管们通常不会那么关心新闻业。他们认为新闻业是太阳系中的冥王星 — — 他们所做的一部分，但相当小而且遥远。
  </p>
  <p class="graf graf--p">
   他们“关心新闻业的方式与我关心清洁水和飞机安全的方式差不多 — 深刻且经常 — 但是，这并不意味着我有参与其发展的资格。
  </p>
  <p class="graf graf--p">
   谷歌与美国国防部进行谈判以获得利益，并试图掩盖这种合作，Facebook 把发布批评内容的发布者列入黑名单，苹果公司与那些经常监禁新闻记者和建立种族集中营的政权合作。这三者都有管理媒体的策略，他们发布的数据很少，关于他们平台上发生的事或者它的影响是什么，我们一无所知，这些技术报告原本应该成为一种重要的问责形式，但并没有。
  </p>
  <p class="graf graf--p">
   虽然谷歌与 McClatchy 的合作是前所未有的，但非新闻组织直接担任出版商和资助新闻的角色并不新鲜。最新的是全行业接受谷歌和 Facebook 的资金支持，特别是在地方一级新闻业的资金支持方面。从技术中赚钱的个人有时会用它来支持新闻业; 杰夫贝索斯因为“拯救”和复活华盛顿邮报而受到欢迎，Laurene Powell Jobs 的艾默生集团正在支持和扩大大西洋月刊，Craig Newmark（Craigslist 的创立者）资助新闻学院和研究，包括 CJR 和 Tow 的中心部分。
  </p>
  <p class="graf graf--p">
   在所有这些行动中，透明度和对编辑独立于资金的承诺使得人际关系变得似乎有些舒适。但是，
   <strong class="markup--strong markup--p-strong">
    当谈到公司利益时，记者必须警惕与他们自己发生冲突的议程
   </strong>
   。
  </p>
  <p class="graf graf--p">
   事实上，以完全独立的方式支持新闻业是有可能的：它可以通过税收和公民媒体的扩展来实现；可以通过支付由不同机构管理的公平禀赋来实现；甚至可以通过改变自己平台上的激励结构来提升并将更多资金返还给新闻编辑室。
  </p>
  <p class="graf graf--p">
   而谷歌的赞助提供的可能性很小。
  </p>
