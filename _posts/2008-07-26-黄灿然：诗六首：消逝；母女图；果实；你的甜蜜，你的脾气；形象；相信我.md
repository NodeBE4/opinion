   <h1>
    黄灿然：诗六首：消逝；母女图；果实；你的甜蜜，你的脾气；形象；相信我
   </h1>
   <ul class="post-entry-meta">
    <li>
     By:
     <a href="http://www.ideobook.com/huangcanran/" title="查看 黄灿然 的作者主页">
      黄灿然
     </a>
     . 2008-7-26.
     <a href="http://www.ideobook.com/372/post-views-count/" title="统计说明">
      5,929
     </a>
    </li>
   </ul>
  </header>
  <div class="post-entry-content">
   <blockquote>
    <p>
     <strong>
      消 逝
     </strong>
     <br/>
     <br/>
     就在我眼前，山上那片树林
     <br/>
     枝叶如此清晰地晃动，在风中，
     <br/>
     每一瞬间的姿态都不一样——
     <br/>
     那不是不一样，那是快速地消逝。
    </p>
   </blockquote>
   <blockquote>
    <p>
     <strong>
      母女图
     </strong>
     <br/>
     <br/>
     那对母女坐在公园的长凳上谈心，
     <br/>
     隔着一个空位。她们放松地靠着椅背的姿态，
     <br/>
     她们闲闲地聊天的姿态
     <br/>
     给了我这颗近来不胜忧烦的心多少的安慰，
     <br/>
     虽然我知道，我知道，她们可能
     <br/>
     非常可能是在互相安慰。
    </p>
   </blockquote>
   <blockquote>
    <p>
     <strong>
      果 实
     </strong>
     <br/>
     <br/>
     平时我都是在下午的阳光中
     <br/>
     坐在茶餐厅外远远凝望你，
     <br/>
     看你枝叶翻飞。今夜，为了散心
     <br/>
     我过了天桥，来到小石路上，
     <br/>
     坐在你的与夜色一样暗的浓荫下。
     <br/>
     我真的想不到，真的想不到
     <br/>
     在你的繁枝茂叶下，在杈桠间，
     <br/>
     在粗壮、潮湿的躯干上
     <br/>
     有这么累累的果实。
    </p>
   </blockquote>
   <blockquote>
    <p>
     <strong>
      你的甜蜜，你的脾气
     </strong>
     <br/>
     <br/>
     当我从你的甜蜜里出来，
     <br/>
     走在大街上，看见来往行人的正面和背影，
     <br/>
     看见他们在阳光中而不知道自己在阳光中，
     <br/>
     在蓝天下而不抬头望望蓝天，
     <br/>
     <br/>
     我就想
     <br/>
     <br/>
     我为何这么幸福，
     <br/>
     而他们这么可怜和忙碌。
     <br/>
     <br/>
     而当我从你的脾气里出来，
     <br/>
     走在大街上，看见来往行人的正面和背影，
     <br/>
     看见他们在阳光中而不必知道自己在阳光中，
     <br/>
     在蓝天下而不必抬头望望蓝天，
     <br/>
     <br/>
     我就想
     <br/>
     <br/>
     我为何这么可怜，
     <br/>
     而他们这么幸福和忙碌。
    </p>
   </blockquote>
   <blockquote>
    <p>
     <strong>
      形 象
     </strong>
     <br/>
     <br/>
     我看见整个人类的形象
     <br/>
     是一个委身屈膝的顺从者，
     <br/>
     而诗人、艺术家、英雄
     <br/>
     和所有不屈不挠者，
     <br/>
     是他头上的短发
     <br/>
     在风中挺立。
    </p>
   </blockquote>
   <blockquote>
    <p>
     <strong>
      相信我
     </strong>
     <br/>
     <br/>
     就像你一定会有过这样的经历：翻看
     <br/>
     一本有人说他深受感动的书，或任何一本书，
     <br/>
     但你没有状态，看不下去，只感到昏昏欲睡，
     <br/>
     但有一天，当你充满生机，充满感觉，充满
     <br/>
     理解和同惰，你会从同一本书，或任何一本书
     <br/>
     读出生命的悲欢，并感到周身灵气流转。
     <br/>
     <br/>
     你身外的大千世界也是这样，一片风景，
     <br/>
     一块招牌，一棵树，一个公园，甚至一个个人，
     <br/>
     也像一行行、一段段死文字，只有当你充满生机，
     <br/>
     充满感觉，尤其是当你充满爱的力量，
     <br/>
     你才会领悟，并相信我，并像我一样
     <br/>
     为一块石头或一个不认识的人落泪。
    </p>
   </blockquote>
   <p>
    <br/>
    原载《天涯》2008年第4期
   </p>
