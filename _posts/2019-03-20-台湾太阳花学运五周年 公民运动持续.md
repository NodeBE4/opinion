 <p>
  76名台湾公民运动者3月18日集结台北宣示成立“台湾公民阵线”，守护台湾民众。站立者为中研院副研究员吴介民
 </p>
 <p>
  <span style="font-size: 12pt;">
   3月18日是台湾占领立法院、反对两岸服贸易协议的“太阳花运动 ”五周年。数十位活动人士当天在台北，发起成立“台湾公民阵线”。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   “2020公民先行、带领台湾前进！”
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   76位台湾公民运动者18日集结台北，宣示成立“台湾公民阵线”。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   经济民主连合顾问林秀幸宣读《台湾公民阵线成立声明》：“确保台湾的主权独立，只有确认主权，我们才能决定自己的命运。主权独立必须搭配民主政治，人民才能拿到定义国家的权力。”
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   中央研究院社会所副研究员吴介民表示，今天是“三一八太阳花学运”反两岸黑箱服贸协议五周年，中华人民共和国政权在三一八太阳花运动之后加速对台湾年轻人统战，利用台湾的自由开放夺走台湾的自由民主，反观台湾民主这五年在防卫立法上则毫无作为。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   吴介民提到，中研院社会所过去七年民调显示，问到中国大陆经济政治各方面发展到跟台湾差不多水平时，同不同意两岸就应统一？虽然去年仍有68%受访台湾人不愿和对岸进行统一，但要注意另一现象。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   吴介民说：“过去这几年，认同两岸条件拉齐的时候就应该统一的比例却有缓慢升高的趋势。在2013年是20%，可是到了去年2018年已经到了27%。没错，这些是相对少数一群人，他们大部份可能属于所谓统派支持者，但是这个人口群体就是中国政府的切入点，在其中发酵，扩大影响力，也就是产生锐实力的地盘。”
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   民进党立委补选胜选 学者称是选民改判民进党“缓刑”
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   吴介民说，去年底九合一选举民进党大败，反应基层民众失望疏离。而上周日立委补选，民进党保住两席，国民党籍高雄市长韩国瑜辅选的立委则败北，引发“韩流”是否退烧的议论。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   吴介民说：“民进党本身焦虑明年初的大选，会不会失去政权？反观国民党则是带着即将重获政权的焦躁跟轻浮，以为玩弄民粹就可以重返执政。我相信，民进党获得十个月‘缓刑’的意义，就在于我们台湾社会还存在一股头脑清楚的力量，要阻止亲中保守力量的复辟。”
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   提到强化台湾民主防卫机制。台湾人权促进会执行委员沈伯洋说，必须提防敌国发动资讯战成为混合战的一环。一次世界大战俄罗斯就是对波罗的海三小国和乌克兰使用资讯战的混合战战术，不费一兵一卒将这些国家并吞。 包括透过断交、提供经济诱因、破坏金融体制，加上资讯战陆面渗透，和空中社群网路当武器，进入各个异议分子的电脑系统作相关跳板攻击，以大量资讯传递方式阻挠、引导人民决定。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   台湾人权促进会执行委员沈伯洋呼吁严防中国大陆对台资讯战。
  </span>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   沈伯洋说：“资讯战的武器化在台湾已经变成一个非常严重的问题。如果说以美国跟瑞士现在最新对台湾的判断，台湾已经陷入俄罗斯的模式战争阶段的第三阶段。俄罗斯战争模式由中国所引用，以及中国所谓政战军官翻译的六阶段，这六个阶段战争模式的最后一阶段就是和平协议。所以如果台湾不阻挡相关资讯战的作为，很有可能在第六阶段的和平协议就已经无力回天。”
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   经济民主连合召集人赖中强表示，台湾公民阵线将针对2020大选及国家发展愿景，提出第一号、第二号、第三号等“公民政纲”，与各政党对话、谈判，促成各政党采纳公民政纲付诸实现。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>

 </p>
