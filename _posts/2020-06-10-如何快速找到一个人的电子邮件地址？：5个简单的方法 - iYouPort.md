  <p class="graf graf--p">
   查找某人的电子邮件地址可能很困难，因为与电话号码和实际地址不同，没有类似于电话簿的电子邮件地址数据库。但是，您可以使用一些技巧来简化电子邮件查找过程。
  </p>
  <p class="graf graf--p">
   本文介绍5个方法，以找到目标人的电邮地址。这些方法都很简单，它们强调了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%88%86%e8%a3%82%e4%ba%ba%e6%a0%bc%e7%9a%84%e9%ad%85%e5%8a%9b/" href="https://www.iyouport.org/%e5%88%86%e8%a3%82%e4%ba%ba%e6%a0%bc%e7%9a%84%e9%ad%85%e5%8a%9b/" rel="noopener noreferrer" target="_blank">
    分身的重要性
   </a>
   —— 不要将您的重要电邮地址与不重要的地址发生任何关联。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、使用搜索引擎
   </strong>
  </h3>
  <p class="graf graf--p">
   Google 或其他搜索引擎是查找各种信息的好地方，如果您使用的工具具有高级搜索功能，那么查找电子邮件地址会很容易。
  </p>
  <p class="graf graf--p">
   使用搜索引擎您可以立即找到并浏览庞大的网站数据库。这是查找任何人的电子邮件地址的最佳方法，因此它应该是您的第一种方法（如果它没找到，再试其他方法）。
  </p>
  <p class="graf graf--p">
   但是，几乎不可能手动浏览所有这些网站来找到每个电子邮件地址，然后对所有这些地址进行过滤以找到您想要的东西。因此，重要的是使用可用的任何工具来缩小结果范围。
  </p>
  <p class="graf graf--p">
   例如，使用 Google，您可以采取搜索运算符来真正了解所要寻找的人。如果您要查找 iyouport 的 Gmail 地址，请参见以下示例：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “@gmail.com” AND “iyouport”
   </strong>
  </p>
  <p class="graf graf--p">
   只是举个例子！这个例子可能不好，因为我们并没有gmail ；也就是说，“iyouport” 可以换成任何您想要的人。
  </p>
  <p class="graf graf--p">
   注意引号哦。
   <strong class="markup--strong markup--p-strong">
    这就是 Googledork
   </strong>
   。您可以在这里看到完整版用法介绍：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%94%bb%e9%98%b2%e7%9a%86%e5%8f%af%e7%94%a8%ef%bc%9agoogledorking-%e9%ab%98%e7%ba%a7%e8%bf%90%e7%ae%97%e7%ac%a6%e5%ae%8c%e6%95%b4%e5%88%97%e8%a1%a8/" href="https://www.iyouport.org/%e6%94%bb%e9%98%b2%e7%9a%86%e5%8f%af%e7%94%a8%ef%bc%9agoogledorking-%e9%ab%98%e7%ba%a7%e8%bf%90%e7%ae%97%e7%ac%a6%e5%ae%8c%e6%95%b4%e5%88%97%e8%a1%a8/" rel="noopener noreferrer" target="_blank">
    攻防皆可用：GoogleDorking 高级运算符完整列表
   </a>
   》。引号、冒号、没有或有空格，都很重要。
  </p>
  <p class="graf graf--p">
   当然，添加任何其他相关信息都可能会有所帮助。
  </p>
  <p class="graf graf--p">
   如果您知道此人的工作或生活地点，或与他或她有关的任何其他信息，都可以试着在此处使用。这会更精准定位。
  </p>
  <p class="graf graf--p">
   以下是使用 Google 查找任何人的电子邮件地址的其他示例：
  </p>
  <ul class="postList">
   <li class="graf graf--li graf--startsWithDoubleQuote">
    <strong class="markup--strong markup--li-strong">
     “@yahoo.com” AND “new york” AND “Pat Neises”
    </strong>
   </li>
   <li class="graf graf--li graf--startsWithDoubleQuote">
    <strong class="markup--strong markup--li-strong">
     “@aol.com” OR “@hotmail.com” “Colorado State University” AND “Elizabeth”
    </strong>
   </li>
  </ul>
  <p class="graf graf--p">
   如果您无法确定目标人使用的是哪个电子邮件提供商，那么可以分别尝试使用它们进行搜索，也可以使用如上所示的 OR 选项。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、在目标人的网站上搜索
   </strong>
  </h3>
  <p class="graf graf--p">
   目标人有自己的网站吗？也许你知道目标人曾就读某所学校，而您怀疑他们的电子邮件地址已经张贴在学校的网站上。那就去搜。
  </p>
  <p class="graf graf--p">
   有几种方法可以将您的电子邮件搜索范围缩小到一个特定的网站。
  </p>
  <p class="graf graf--p">
   第一种是使用搜索引擎或网站的搜索功能。例如，转到Google并输入
   <strong class="markup--strong markup--p-strong">
    site:example.com “@gmail.com” AND “Mary”
   </strong>
   以帮助查找 Mary 的 Gmail 帐户。
  </p>
  <p class="graf graf--p">
   此类搜索仅限于
   <em class="markup--em markup--p-em">
    example.com
   </em>
   ，因此您可以将其更改为可能列出其地址的任何网站。
  </p>
  <p class="graf graf--p">
   <p class="graf graf--p">
    从网站收集电子邮件地址的另一种方法是，使用像
    <a class="markup--anchor markup--p-anchor" data-href="https://www.voilanorbert.com/" href="https://www.voilanorbert.com/" rel="noopener noreferrer" target="_blank">
     <strong class="markup--strong markup--p-strong">
      VoilaNorbert
     </strong>
    </a>
    这样的电子邮件抓取工具。只需输入目标人的名字和姓氏以及网站 URL，即可获得匹配的电子邮件地址列表。您可以免费获得50个成功搜索。
   </p>

   </p>
   <p class="graf graf--p">
    当然，您可能已经考虑过这样做了，但是您应该知道
    <strong class="markup--strong markup--p-strong">
     一些技巧
    </strong>
    ，可帮助您更容易猜测某人的电子邮件地址。
   </p>
   <p class="graf graf--p">
    如果您已经知道目标人的用户名（@之前的部分），请尝试所有可以想到的域。
   </p>
   <p class="graf graf--p">

    </strong>

    <strong class="markup--strong markup--p-strong">

    <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%96%b0%e7%94%9f%e6%83%85%e6%8a%a5%e5%b7%a5%e5%85%b7%ef%bc%887%ef%bc%89%ef%bc%9a%e5%85%a8%e7%90%83%e6%90%9c%e4%ba%ba%e3%80%81%e5%85%8d%e8%b4%b9%e6%89%8b%e6%9c%ba%e7%9f%ad%e4%bf%a1%e3%80%81%e6%94%be/" href="https://www.iyouport.org/%e6%96%b0%e7%94%9f%e6%83%85%e6%8a%a5%e5%b7%a5%e5%85%b7%ef%bc%887%ef%bc%89%ef%bc%9a%e5%85%a8%e7%90%83%e6%90%9c%e4%ba%ba%e3%80%81%e5%85%8d%e8%b4%b9%e6%89%8b%e6%9c%ba%e7%9f%ad%e4%bf%a1%e3%80%81%e6%94%be/" rel="noopener noreferrer" target="_blank">

    </a>

   </p>
   <p class="graf graf--p">
    您所要做的就是创建一个免费用户帐户以查看结果。
   </p>
   <p class="graf graf--p">
    该工具可能会向您显示某人的电子邮件地址，并很可能会显示其在线状态，然后您可以使用该工具进行进一步的研究。
   </p>
   <p class="graf graf--p">
    使用 Pipl 找到的关于某人的全部信息可能包括他们的姓名、用户名或电话号码等等。位置会更有帮助，但这不是必需的，跟踪狂比较喜欢这个。执行搜索后，可以按位置和年龄过滤结果。
   </p>
   <h3 class="graf graf--p">
    <strong class="markup--strong markup--p-strong">
     5、查找包含其地址的过去电子邮件
    </strong>
   </h3>
   <p class="graf graf--p">
    不要忽视最明显的地方：打开目标人曾经发送给您的电子邮件。
   </p>
   <p class="graf graf--p">
    如果您想查找某人的电子邮件地址，以便可以向他们发送消息，但您的联系人列表中没有这个人，请快速搜索您的所有电子邮件，以查找该人的消息。
   </p>
   <p class="graf graf--p">
    大多数电子邮件提供商在页面顶部都有一个大搜索工具，您可以在其中输入有关所要查找邮件的详细信息，例如主题、收件人电子邮件签名中的详细信息、姓名等。
   </p>
   <p class="graf graf--p">
    输入您所知道的关于此人的任何信息，然后查看是否包含在旧消息中。
   </p>
   <p class="graf graf--p">
    您可以在其中查看电子邮件的到达时间、谁发送的、以及发送给谁。您要查找的是 “发件人” 地址。
   </p>
   <p class="graf graf--p">
    就是这么简单，您肯定学会了。⚪️
