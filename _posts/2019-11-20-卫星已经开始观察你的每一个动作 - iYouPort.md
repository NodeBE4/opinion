  <p class="graf graf--p">
   随着大规模监视成为一个恐怖的现实，卫星也不闲着了 —— 已经开始关注你的一举一动，特别是对爱好自由的人类来说，噩梦正在加深。
  </p>
  <p class="graf graf--p">
   根据麻省理工学院技术评论的一份报告，虽然商业卫星图像已经足够强大，例如，能看到是一辆汽车，但它的详细程度还不足以识别品牌和型号。
  </p>
  <p class="graf graf--p">
   但隐私和自由倡导者说，这无关紧要。问题是，我们已经被看到了。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnet.com/news/satellites-are-starting-to-watch-your-every-move/" href="https://www.cnet.com/news/satellites-are-starting-to-watch-your-every-move/" rel="noopener noreferrer" target="_blank">
    据 CNET 称
   </a>
   ，近10年来卫星成像技术的巨大进步让隐私权倡导者很担心24小时持续的监控。
  </p>
  <p class="graf graf--p">
   卫星公司声称他们将一个人的数据与任何识别特征分开，但是，包括安全世界基金会的 Peter Martinez 在内的许多人仍然关注人们
   <strong class="markup--strong markup--p-strong">
    被不断监测
   </strong>
   的问题。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这些风险不仅来自卫星图像本身，还来自地球观测数据与其他数据来源的融合，” Martinez 在一封电子邮件中说。
  </p>
  <p class="graf graf--p">
   问题在于卫星的数量很大。影像公司 Planet Labs 证实它目前有140颗成像卫星在轨道上运行。这足以在一天之内游遍地球上的每一个地方。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “即使使用 Planet 的最高分辨率图像（1米分辨率），仍然无法区分个人、车牌或其他识别性信息。我们的图像非常适合每天监控大规模的变更；这包括看到建筑物和道路、森林、农业、水体等地表的日常变化”，Planet Labs 的发言人在一封电子邮件中说。
  </p>
  <p class="graf graf--p">
   想想看，如果他们能看清，他们会承认吗？
  </p>
  <p class="graf graf--p">
   但肯定无疑的是，技术正在快速发展，并随之增加政府的监控能力。
  </p>
  <p class="graf graf--p">
   尽管许多人都知道自己一直在被持续不断地监控和追踪，但有些人并不完全理解这种情况有多糟糕。
  </p>
  <p class="graf graf--p">
   每个人都想要某种形式的隐私；然而，我们生活的世界提供的并不多。
  </p>
  <p class="graf graf--p">
   就如 Cloud Wards 的报告所展示的，如今很多国家都有大规模监控系统，但五眼、九眼和14眼联盟的国家基于情报合作，可以大规模地跨国共享数据，也就是形成跨国监控。无辜的人每天都在被监视。
  </p>
  <p class="graf graf--p">
   更多：全球大规模监视系统简要报告《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%82%a8%e5%ba%94%e8%af%a5%e4%ba%86%e8%a7%a3%e7%9a%84%e6%94%bf%e5%ba%9c%e7%9b%91%e8%a7%86%e9%a1%b9%e7%9b%ae/" href="https://www.iyouport.org/%e6%82%a8%e5%ba%94%e8%af%a5%e4%ba%86%e8%a7%a3%e7%9a%84%e6%94%bf%e5%ba%9c%e7%9b%91%e8%a7%86%e9%a1%b9%e7%9b%ae/" rel="noopener noreferrer" target="_blank">
    您应该了解的政府监视项目
   </a>
   》
  </p>
  <p class="graf graf--p">
   五眼、九眼和14眼情报联盟是全球监控游戏中最重要的角色。所涉及的每个国家都可以在特定地区进行监视，并与联盟中的任何其他国家分享。
  </p>
  <p class="graf graf--p">
   这就是绕开宪法制约的最佳方法。比如美国的宪法禁止情报机构监视本国公民，那么美国的间谍机构就可以让英国的间谍机构监视美国人，然后再与美国的情报机构共享。他们这样做已经有几十年了。
  </p>
  <p class="graf graf--p">
   在这里看到更多《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%9c%80%e4%bd%b3%e5%ba%87%e6%8a%a4%e6%89%80%e5%92%8c%e8%b7%a8%e5%9b%bd%e7%bb%b4%e7%a8%b3-%e9%80%8f%e6%98%8e%e5%ba%a6%e9%9d%a9%e5%91%bd%e7%9a%84%e6%9c%aa%e6%9d%a5/" href="https://www.iyouport.org/%e6%9c%80%e4%bd%b3%e5%ba%87%e6%8a%a4%e6%89%80%e5%92%8c%e8%b7%a8%e5%9b%bd%e7%bb%b4%e7%a8%b3-%e9%80%8f%e6%98%8e%e5%ba%a6%e9%9d%a9%e5%91%bd%e7%9a%84%e6%9c%aa%e6%9d%a5/" rel="noopener noreferrer" target="_blank">
    最佳庇护所和跨国维稳 — — 透明度革命的未来
   </a>
   》
  </p>
  <p class="graf graf--p">
   卫星监视的发展速度会有多快？看清全世界每一个人的脸和每一张车牌还需要多久？⚪️
  </p>
