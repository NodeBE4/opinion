  <p class="graf graf--p">
   您在 WhatsApp 和 Telegram 上看到的内容可能并不是您的联系人发送的内容，因为这两个应用程序存在一个漏洞。
  </p>
  <p class="graf graf--p">
   尽管安全消息传递应用程序的端到端加密理论上可以保护人们免受政府的监视，但是，赛门铁克的研究人员披露了可能允许潜在的骇客改变图像和音频文件的漏洞。
  </p>
  <p class="graf graf--p">
   虽然发件人可能发送了一张地图照片，但
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnet.com/tags/malware/" href="https://www.cnet.com/tags/malware/" rel="noopener noreferrer" target="_blank">
    恶意软件
   </a>
   可以利用 WhatsApp 和 Telegram 上的此漏洞替换照片并向收件人提供错误的指示。
  </p>
  <p class="graf graf--p">
   在另一个示例中，恶意软件可以改变发票照片中显示的数字，欺骗受害者向错误的人支付金钱。
  </p>
  <p class="graf graf--p">
   必需提醒这点，因为这两个消息应用程序都是活动家、记者和持不同政见者使用的重要工具，这些敏感人士很希望保持对话免受监视。
  </p>
  <p class="graf graf--p">
   Signal，WhatsApp，Telegram 等消息应用程序具有端到端加密功能，这意味着即使是公司本身也无法看到对话。
  </p>
  <p class="graf graf--p">
   虽然加密可以保护您的通信免受监控，但是，这并不意味着应用程序本身具有绝对强大的免疫力。
  </p>
  <p class="graf graf--p">
   今年5月就披露了 WhatsApp 的一个惊人的漏洞，该漏洞允许骇客通过简单的电话呼叫在设备上安装间谍软件 —— 甚至受害人无需接听就能完成。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6811 jetpack-lazy-image" data-lazy-sizes="(max-width: 940px) 100vw, 940px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/图-2.png?resize=940%2C882&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/图-2.png?w=940&amp;ssl=1 940w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/图-2.png?resize=300%2C281&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/图-2.png?resize=768%2C721&amp;ssl=1 768w" data-recalc-dims="1" height="882" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/图-2.png?resize=940%2C882&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="940"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6811" data-recalc-dims="1" height="882" sizes="(max-width: 940px) 100vw, 940px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/图-2.png?resize=940%2C882&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/图-2.png?w=940&amp;ssl=1 940w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/图-2.png?resize=300%2C281&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/图-2.png?resize=768%2C721&amp;ssl=1 768w" width="940"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   安全研究人员还在2017年披露了一项 telegram 漏洞，该漏洞允许骇客直接接管帐户。
  </p>
  <p class="graf graf--p">
   赛门铁克的研究人员称，最新披露的新漏洞虽然不允许帐户被劫持，但是,作为欺诈是非常成熟的。
  </p>
  <p class="graf graf--p">
   赛门铁克在一篇博客文章中表示，该漏洞源于媒体文件在 WhatsApp 和 Telegram 上的储存方式。
  </p>
  <p class="graf graf--p">
   当文件储存在外部存储上时，其他应用程序就可以访问和操作它们。这不具备任何安全性。
  </p>
  <p class="graf graf--p">
   在 WhatsApp 上，默认情况下文件存储在外部，而在 Telegram 上，如果启用了“Save to Gallery”，则会出现此漏洞。
  </p>
  <p class="graf graf--p">
   赛门铁克的研究人员对其创建的恶意软件进行了测试，以篡改通过 WhatsApp 和 Telegram 发送的图像和音频文件。
  </p>
  <p class="graf graf--p">
   在一个演示片段中，一个人发送了两个朋友的照片，收件人设备上的恶意软件自动将其替换成了演员 Nicolas Cage。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/FHvkGUh8S_c" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   WhatsApp 在一份声明中说，“WhatsApp 已经密切关注这个问题，它与之前关于移动设备存储影响应用生态系统的问题类似；WhatsApp 遵循操作系统提供的媒体存储最新实践，并期待提供符合 Android 持续开发的更新”。
  </p>
  <p class="graf graf--p">
   Telegram 没有回复评论请求。
  </p>
  <p class="graf graf--p">
   ⚠️如果您正在使用这些应用程序，请快速通过更改媒体存储设置来保护自己免受此风险。
  </p>
  <p class="graf graf--p">
   在 WhatsApp 上，您可以通过转到设置并切换“媒体可见性”来执行此操作。在 Telegram 上，您可以通过切换“保存到图库”来保护自己。
  </p>
  <p class="graf graf--p">
   但 WhatsApp 在一封电子邮件中表示，建议的更改可能会产生隐私问题并限制图像的共享方式。
  </p>
  <p class="graf graf--p">
   许多应用程序都将图像存储在外部存储中，因此即使在卸载应用程序时，人们也可以保存图片，并且大多数 Android 设备都没有提供足够的内部存储空间。这是个问题。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    一个冒名顶替的 Telegram
   </strong>
  </p>
  <p class="graf graf--p">
   赛门铁克的研究人员还发现了一个与 Telegram 有关的单独问题，在 Google Play 商店中有一个虚假版本的 Telegram 应用程序。
  </p>
  <p class="graf graf--p">
   这个名为 MobonoGram 的应用程序将自己升级为具有额外功能的 Telegram 的增强版。
  </p>
  <p class="graf graf--p">
   如果“额外功能”意味着在背景中访问色情和诈骗等恶意网站，那倒的确是真的。
  </p>
  <p class="graf graf--p">
   该欺诈性应用在从 Google Play 商店中被删除之前已经被下载了超过100,000次。谷歌证实它被删除并称它禁止了开发者。
  </p>
  <p class="graf graf--p">
   研究人员也发现了另一个名为 Whatsgram 的冒名顶级应用程序，该应用程序来自同一个开发人员。
  </p>
  <p class="graf graf--p">
   赛门铁克表示，从1月到5月，它已经阻止了1200多个与这些开发者相关的应用程序，大多数感染事件发生在美国、伊朗和印度。⚪️
  </p>
