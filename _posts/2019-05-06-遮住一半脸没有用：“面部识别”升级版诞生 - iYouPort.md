 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <strong>
      <em>
       遮住部分脸部已经无法逃离面部识别，甚至只需要拥有你童年时代的照片也能找到你现在的本人，俄罗斯最新被发现的面部识别技术已经完全超越 Facebook 的恐怖，这是谁干的？
      </em>
     </strong>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   俄罗斯互联网用户
   <a class="markup--anchor markup--p-anchor" data-href="https://habr.com/ru/post/439828/" href="https://habr.com/ru/post/439828/" rel="noopener noreferrer" target="_blank">
    发现
   </a>
   了一个网站
   <a class="markup--anchor markup--p-anchor" data-href="http://searchface.ru/" href="http://searchface.ru/" rel="noreferrer noopener" target="_blank">
    searchface.ru
   </a>
   ，允许任何人使用单个图像搜索俄罗斯热门社交媒体网络VKontakte。该网站的功能很简单：在上传包含某人脸部的照片后，用户就可以看到指向 VKontakte 页面的链接列表，其中包含描绘同一个人的照片。
  </p>
  <p class="graf graf--p">
   在社交媒体网
   <a class="markup--anchor markup--p-anchor" data-href="https://meduza.io/news/2019/02/12/vkontakte-podast-v-sud-na-razrabotchikov-sayta-searchface-kotoryy-pozvolyaet-nahodit-polzovateley-po-fotografiyam" href="https://meduza.io/news/2019/02/12/vkontakte-podast-v-sud-na-razrabotchikov-sayta-searchface-kotoryy-pozvolyaet-nahodit-polzovateley-po-fotografiyam" rel="noopener noreferrer" target="_blank">
    宣布将起诉
   </a>
   SearchFace“严重违反[VKontakte’s]规则”后，该算法的创建者从网站的搜索结果中
   <a class="markup--anchor markup--p-anchor" data-href="https://meduza.io/news/2019/02/13/servis-searchface-ubral-iz-poiskovoy-vydachi-ssylki-na-stranitsy-polzovateley-vkontakte" href="https://meduza.io/news/2019/02/13/servis-searchface-ubral-iz-poiskovoy-vydachi-ssylki-na-stranitsy-polzovateley-vkontakte" rel="noopener noreferrer" target="_blank">
    删除
   </a>
   了个人资料链接，但保留了其余所有部分的功能。
  </p>
  <p class="graf graf--p">
   在此之前，Meduza 记者 Sultan Suleimanov 通过将最近他自己的照片上传到网站上来尝试 SearchFace 的技术。
   <strong class="markup--strong markup--p-strong">
    他有意使用了一张脸部被扭曲的照片，但这似乎并没有影响网站的搜索结果
   </strong>
   。SearchFace 不仅能够找到 Suleimanov 的个人 VKontakte 页面，还能找到几个虚假账户，其创建者已经上传了他本人的照片。
  </p>
  <p class="graf graf--p">
   甚至，
   <strong class="markup--strong markup--p-strong">
    结果中还包括 Suleimanov 自己都从未见过的图像
   </strong>
   。那是在朋友的婚礼上拍摄的，并
   <strong class="markup--strong markup--p-strong">
    被贴在一个锁定的 VKontakte 帐​​户上
   </strong>
   。
  </p>
  <p class="graf graf--p">
   SearchFace 让人回想起另一个俄罗斯项目 FindFace，该项目于2016年由 N-Tech Lab 公司发布。 Meduza 无法比较两种服务的准确性，因为 FindFace 的公共访问在2018年秋季
   <a class="markup--anchor markup--p-anchor" data-href="https://meduza.io/news/2018/06/30/razrabotchiki-findface-ob-yavili-o-zakrytii-servisa" href="https://meduza.io/news/2018/06/30/razrabotchiki-findface-ob-yavili-o-zakrytii-servisa" rel="noopener noreferrer" target="_blank">
    已关闭
   </a>
   。其创建者决定将重点转向企业产品。
  </p>
  <p class="graf graf--p">
   在 SearchFace 发布消息开始传播之后，TJournal 的编辑设法与该产品的创作者联系。要求匿名的程序员表示，他的目标不是赚钱 — SearchFace 没有广告，也不需要注册。相反，他只是想研究他的算法的能力。具体来说，
   <strong class="markup--strong markup--p-strong">
    程序员说他希望在一组非常大的照片上测试面部识别工具，包括低质量的图像和隐藏部分人脸的图像
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Meduza 与同一个开发人员取得了联系，后者能够证明他参与了 SearchFace 项目。他解释说，该算法是在发布给 VKontakte 的5亿张照片的基础上运行的。这些基础数据是在2018年夏天收集的，之前社交媒体网络使用户能够通过锁定他们的个人资料，使信息不会公开显示。但是，
   <strong class="markup--strong markup--p-strong">
    SearchFace 来了，锁定不再管用
   </strong>
   。
  </p>
  <p class="graf graf--p">
   尽管如此，程序员向 Meduza 保证，SearchFace 不会保存其照片库或用户上传到 VKontakte 的任何其他图像。他说，允许网站搜索功能运行的神经网络已由他的三人团队完成开发，不需要额外的视觉输入。他还称 SearchFace 是新面部识别技术的演示版本，并称其“几乎意外地”可被公开访问。在该网站被发现后的第二天，有成千上万的用户访问了其主页。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    SearchFace 可以非常有效地识别个人的照片。例如一个人的童年照片，也能找到现在的本人；记者奥列格·卡申（Oleg Kashin）的一张照片中部分脸部隐藏在海报后面，但 SearchFace 也能找到其本人
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在 Kashin 和 Krasilshchik 的案例中，SearchFace 的表现甚至超过允许 Facebook 面部识别的算法。这是令人毛骨悚然的开始。
   <strong class="markup--strong markup--p-strong">
    穿着连帽衫的苏丹·苏莱曼诺夫（Sultan Suleimanov）无法被Facebook 的算法识别，但是对于 SearchFace 来说，这是小菜一碟
   </strong>
   。
  </p>
  <p class="graf graf--p">
   调查记者很快就开始利用这项技术了：一份出版物
   <a class="markup--anchor markup--p-anchor" data-href="https://tjournal.ru/money/87887-cherez-searchface-udalos-uznat-podrobnosti-zhizni-sozdatelya-azino777-blagodarya-odnomu-foto-iz-dela-ukrainskoy-policii" href="https://tjournal.ru/money/87887-cherez-searchface-udalos-uznat-podrobnosti-zhizni-sozdatelya-azino777-blagodarya-odnomu-foto-iz-dela-ukrainskoy-policii" rel="noopener noreferrer" target="_blank">
    成功找到
   </a>
   了网上赌场 Azino777 的创始人，并仅仅使用一张2004年的单张照片就可以详细研究他的个人生活。
  </p>
  <p class="graf graf--p">
   1月下旬，另一个旨在搜索 VKontakte 照片的网站vlicco.ru发布。与 SearchFace 一样，Vlicco 目前正处于初始测试阶段。它的创建者希望通过在网站上托管广告来获利。要访问 Vlicco，用户必须通过他们的 VKontakte 帐​​户登录并允许其访问他们自己的所有照片。该网站上可用的搜索查询数量有限，其搜索质量比 SearchFace 差很远。
  </p>
  <p class="graf graf--p">

    你曾经在社交网络上发布过自己的照片？你危险了！
   </a>
   》
  </p>
