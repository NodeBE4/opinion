   赋权还是剥削
  </h1>

  <p class="graf graf--p graf--startsWithDoubleQuote">
   “监狱劳工”通常与体力劳动有关，但芬兰两所监狱的囚犯正在进行一种新型劳动：对数据进行分类，以便为创业公司培训人工智能算法。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-3309 jetpack-lazy-image" data-lazy-sizes="(max-width: 1000px) 100vw, 1000px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-240.png?resize=1000%2C640&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-240.png?w=1000&amp;ssl=1 1000w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-240.png?resize=300%2C192&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-240.png?resize=768%2C492&amp;ssl=1 768w" data-recalc-dims="1" height="640" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-240.png?resize=1000%2C640&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1000"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-3309" data-recalc-dims="1" height="640" sizes="(max-width: 1000px) 100vw, 1000px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-240.png?resize=1000%2C640&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-240.png?w=1000&amp;ssl=1 1000w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-240.png?resize=300%2C192&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-240.png?resize=768%2C492&amp;ssl=1 768w" width="1000"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   虽然有问题的创业公司 Vainu 认为合伙企业是一种教授“有价值技能”的“监狱改革”，但其他专家可不这么看。
  </p>
  <p class="graf graf--p">
   联合创始人 Tuomas Rasila 表示，Vainu 正在建立一个全球公司的综合数据库，帮助企业找到合作伙伴。为了实现这一目标，人们需要阅读从互联网上删除的成千上万的商业文章，并标记每一篇文章，例如，这篇文章是关于 Apple 的科技公司还是一家名为“苹果”的水果公司。 （然后，此标记数据用于训练管理数据库的算法。）
  </p>
  <p class="graf graf--p">
   对于英文文章来说，这没有什么问题：Vainu 只需设置一个 Amazon Mechanical Turk 帐户，让人们完成这些小任务。但是，当你想用芬兰语做某事时，Mechanical Turk “并不是那么有用，”Rasila 说，而且该公司只有一名工作人员用芬兰语标记大量数据。这显然是不够的。
  </p>
  <p class="graf graf--p">
   Vainu 的办公室恰好与刑事制裁机构（CSA）的总部在同一栋楼内，该机构是监督芬兰监狱的政府机构，因此，Rasila 说，创始人有了一个想法：“嘿，我们实际上可以使用监狱劳工。”
  </p>
  <p class="graf graf--p">
   这种伙伴关系大约在三个月前开始，Vainu 现在与两个不同的监狱合作，一个在赫尔辛基，另一个在图尔库。Vainu 向这些监狱运送了10台电脑，并为囚犯完成的每项任务向 CSA 支付费用。这个金额与创业公司为 Mechanical Turk 所完成的任务支付的金额相当，尽管 CSA 负责确定囚犯中有多少人实际参与这项工作。
  </p>
  <p class="graf graf--p">
   芬兰以其先进的监狱政策而闻名。该国大约三分之一的监狱是所谓的开放式监狱，在那里，囚犯像普通公民一样生活和工作。惠灵顿维多利亚大学研究不同刑罚制度的研究员 John Pratt 指出，这里没有美国那种对安全性的强制要求。囚犯通常被要求工作或接受职业培训，甚至有时被允许在监狱外工作或保持自营职业。
  </p>
  <p class="graf graf--p">
   Rasila 表示，该机构的官员很高兴与他人合作，特别是因为除了笔记本电脑外，新工作不需要任何其他设施。“没有暴力风险，”他说，并补充说，当谈到其他形式的监狱劳工，如金属加工时，使用可以变成临时武器的工具有可能使监狱工作区“成为一个危险的地方”。
  </p>
  <p class="graf graf--p">
   Rasila 估计，目前，只有不到100名囚犯正在为Vainu的项目工作，每天工作几个小时。 （CSA 的代表没有回复评论请求。）
  </p>
  <p class="graf graf--p">
   目前，Vainu 和 CSA 根据任务数量签订了年度合同。 Vainu 团队希望将此工作扩展到芬兰的其他地方，以及其他很难找到愿意用当地语言进行此类工作的国家。对他们来说，这是一个“双赢的局面”。
  </p>
  <p class="graf graf--p">
   当然，囚犯的一个动机是赚钱，但“一个卖点是，人工培训 AI 的需求实际上在全球范围内都有显著增加，”Rasila 说。同样，CSA 在一份新闻稿中写道，该计划是其努力开展符合“现代工作生活要求”的活动的一部分。
  </p>
  <p class="graf graf--p">
   加利福尼亚大学圣地亚哥分校的通讯教授莉莉·伊朗（Lilly Irani）表示，这类工作的需求特别高不足为奇。她说，人工智能算法需要以文化特定的方式进行培训，而大多数 Mechanical Turk 都在美国。
  </p>
  <p class="graf graf--p">
   虽然 Rasila 说这是开发技能的一个例子，这可以在将来发挥作用，但他还说这些任务具有“零学习曲线”并且只需要识字就行，这会使人怀疑这项技能究竟有什么用。
  </p>
  <p class="graf graf--p">
   洛杉矶加利福尼亚大学信息科学教授莎拉·T·罗伯茨（Sarah T. Roberts）说，这类工作往往是“死记硬背的，重复性的”。它不需要建立高水平的技能，如果大学研究人员试图以同样的方式与监狱工作者合作，“不会通过伦理审查委员会的研究。”
  </p>
  <p class="graf graf--p">
   虽然囚犯获得与 Mechanical Turk 相同的工资这点是好的，但 Roberts 指出，Mechanical Turk 的工资无论如何都非常低。最近的一份研究报告发现，工人的平均工资为每小时2美元。
  </p>
  <p class="graf graf--p">
   对于 Irani 来说，在这个故事中人工智能没有什么特别之处。至少在美国，监狱劳工一直存在争议，有些人说它在经济上剥削工人，而另一些人认为监狱劳工有助于康复。
  </p>
  <p class="graf graf--p">
   对她而言，公共关系推动的合作比数字化工作已成为监狱劳工的一部分更令人惊讶。“这就是为什么流传这种炒作，以便可以将真正古老的劳动剥削形式伪装成’监狱改革’，”Irani 说。
  </p>

  </p>
