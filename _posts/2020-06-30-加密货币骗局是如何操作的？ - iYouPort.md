  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://navms.com/allegedly-head-of-the-bitconnect-cryptocurrency-scam-arrested-in-dubai/" href="https://navms.com/allegedly-head-of-the-bitconnect-cryptocurrency-scam-arrested-in-dubai/" rel="noopener noreferrer" target="_blank">
    成千上万
   </a>
   的加密货币投资者的钱被骗走了。到2018年，与加密货币有关的犯罪造成的损失已经高达
   <a class="markup--anchor markup--p-anchor" data-href="https://www.reuters.com/article/us-crypto-currency-crime/cryptocurrency-thefts-scams-hit-1-7-billion-in-2018-report-idUSKCN1PN1SQ" href="https://www.reuters.com/article/us-crypto-currency-crime/cryptocurrency-thefts-scams-hit-1-7-billion-in-2018-report-idUSKCN1PN1SQ" rel="noopener noreferrer" target="_blank">
    17亿美元
   </a>
   。犯罪分子使用老式策略和新技术的结合实施欺诈。
  </p>
  <p class="graf graf--p">
   通过研究区块链、加密货币和网络犯罪，可以看出一些加密货币欺诈者依靠的是久经考验的庞氏骗局，利用新参与者的收入向早期投资者支付回报。
  </p>
  <p class="graf graf--p">
   其他人则使用高度自动化和复杂的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.buzzfeednews.com/article/ryanmac/cryptocurrency-scammers-are-running-wild-on-telegram" href="https://www.buzzfeednews.com/article/ryanmac/cryptocurrency-scammers-are-running-wild-on-telegram" rel="noopener noreferrer" target="_blank">
    流程
   </a>
   ，包括与 Telegram 交互的自动化软件，Telegram 是一种基于互联网的即时消息传递系统，在对加密货币感兴趣的人群中很受欢迎。
  </p>
  <p class="graf graf--p">
   即使加密货币计划合法，欺诈者仍然可以在市场上
   <a class="markup--anchor markup--p-anchor" data-href="https://theconversation.com/how-can-criminals-manipulate-cryptocurrency-markets-97294" href="https://theconversation.com/how-can-criminals-manipulate-cryptocurrency-markets-97294" rel="noopener noreferrer" target="_blank">
    操纵其价格
   </a>
   。
  </p>
  <p class="graf graf--p">
   但是，一个更基本的问题是：毫无戒心的投资者如何加密货币欺诈者所吸引？
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    口若悬河的骗子
   </strong>
  </h3>
  <p class="graf graf--p">
   一些加密货币欺诈者勾引人们的贪婪，并承诺获得丰厚的回报。例如，一群不知名的企业家在运行诈骗机器人 iCenter，这是
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@nickcryptoltc/ponzi-investment-schemes-new-and-improved-on-the-blockchain-icenter-co-f9ee68f6c8fe" href="https://medium.com/@nickcryptoltc/ponzi-investment-schemes-new-and-improved-on-the-blockchain-icenter-co-f9ee68f6c8fe" rel="noopener noreferrer" target="_blank">
    针对
   </a>
   比特币和莱特币的庞氏骗局。
  </p>
  <p class="graf graf--p">
   它并没有提供投资策略的信息，但不知何故承诺投资者每日回报率为
   <a class="markup--anchor markup--p-anchor" data-href="https://www.buzzfeednews.com/article/ryanmac/cryptocurrency-scammers-are-running-wild-on-telegram" href="https://www.buzzfeednews.com/article/ryanmac/cryptocurrency-scammers-are-running-wild-on-telegram" rel="noopener noreferrer" target="_blank">
    1.2％
   </a>
   。
  </p>
  <p class="graf graf--p">
   iCenter 骗局通过 Telegram 上的群聊进行操作，它始于一小群骗子，加入的人会获得一个推荐码，通过与他人分享，在博客和社交媒体上，让更多人加入聊天。
  </p>
  <p class="graf graf--p">
   新进群的人开始受到群内骗子的蛊惑。一些新人决定投资，这时候骗子要求他们将比特币存入一个独立的钱包，并放置99天或120天 ，就可以获得高额的回报。
  </p>
  <p class="graf graf--p">
   在此期间，新进入的投资者经常使用社交媒体与朋友和联系人
   <a class="markup--anchor markup--p-anchor" data-href="https://thenextweb.com/hardfork/2018/12/17/cryptocurrency-italy-silly-scams/" href="https://thenextweb.com/hardfork/2018/12/17/cryptocurrency-italy-silly-scams/" rel="noopener noreferrer" target="_blank">
    分享
   </a>
   他们自己的推荐码，让更多人参与群聊和投资计划。
  </p>
  <p class="graf graf--p">
   这些资金在任何合法业务中都没有实际投资。相反，当新人加入时，招募他们的人获得新资金的一定比例，并且循环继续，新进投资者的资金将有一部分分配给早期参与者作为分红。
  </p>
  <p class="graf graf--p">
   一些成员特别努力地筹集新资金，发布教程视频和他们自己持有大量资金的图片
   <a class="markup--anchor markup--p-anchor" data-href="https://www.buzzfeednews.com/article/ryanmac/cryptocurrency-scammers-are-running-wild-on-telegram" href="https://www.buzzfeednews.com/article/ryanmac/cryptocurrency-scammers-are-running-wild-on-telegram" rel="noopener noreferrer" target="_blank">
    作为加入骗局的诱惑
   </a>
   。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    不断编造的谎言
   </strong>
  </h3>
  <p class="graf graf--p">
   一些诈骗者会直接欺骗，加密货币社区最知名的庞氏骗局 OneCoin 的创始人通过说服人们投资根本不存在的加密货币成功骗走了投资者
   <a class="markup--anchor markup--p-anchor" data-href="https://cointelegraph.com/news/us-district-attorney-charges-onecoin-founders-with-billions-in-alleged-fraud" href="https://cointelegraph.com/news/us-district-attorney-charges-onecoin-founders-with-billions-in-alleged-fraud" rel="noopener noreferrer" target="_blank">
    38亿美元
   </a>
   。
  </p>
  <p class="graf graf--p">
   其他骗局的基础是用行话或专业知识声称给潜在的受害者留下深刻印象。“全球贸易” 诈骗者声称他们利用各种加密货币交易所的价格差异从所谓的套利中获利，简单地以低价购买并以更高的价格出售。他们真的能通过这种方式骗走投资者的钱。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “全球贸易” 也使用了 Telegram 上的机器人，投资者可以发送余额查询消息并获得有关其账户中余额的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.buzzfeednews.com/article/ryanmac/cryptocurrency-scammers-are-running-wild-on-telegram" href="https://www.buzzfeednews.com/article/ryanmac/cryptocurrency-scammers-are-running-wild-on-telegram" rel="noopener noreferrer" target="_blank">
    虚假回复
   </a>
   ，有时甚至会看到余额在一小时内上涨
   <a class="markup--anchor markup--p-anchor" data-href="https://steemit.com/bitconnect/@jjona/global-trading-bot-promising-6-gains-per-day" href="https://steemit.com/bitconnect/@jjona/global-trading-bot-promising-6-gains-per-day" rel="noopener noreferrer" target="_blank">
    1％
   </a>
   。
  </p>
  <p class="graf graf--p">
   回报看起来
   <a class="markup--anchor markup--p-anchor" data-href="https://steemit.com/bitconnect/@jjona/global-trading-bot-promising-6-gains-per-day" href="https://steemit.com/bitconnect/@jjona/global-trading-bot-promising-6-gains-per-day" rel="noopener noreferrer" target="_blank">
    很像
   </a>
   是那么回事，谁还能责怪人们在社交媒体上与朋友和家人分享这些骗局计划呢？
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    剥削朋友和家人
   </strong>
  </h3>
  <p class="graf graf--p">
   一旦计划启动，至少在一段时间内它就会通过社交媒体保持活跃。一个人被加密货币投资的巨大回报承诺所吸引，就很可能会将这个骗局传播给
   <a class="markup--anchor markup--p-anchor" data-href="https://news.bitcoin.com/crypto-scams-comprise-0-6-fraud-australian-consumer-watchdog/" href="https://news.bitcoin.com/crypto-scams-comprise-0-6-fraud-australian-consumer-watchdog/" rel="noopener noreferrer" target="_blank">
    朋友和家人
   </a>
   。
  </p>
  <p class="graf graf--p">
   有时候知名 KOL 参与其中也会让骗局看起来更可信，例如，
   <a class="markup--anchor markup--p-anchor" data-href="https://entrackr.com/2019/04/amit-bhardwaj-bail-bitoin/" href="https://entrackr.com/2019/04/amit-bhardwaj-bail-bitoin/" rel="noopener noreferrer" target="_blank">
    GainBitcoin
   </a>
   背后的主要人物就能够做到令一些宝莱坞名人为他们背书。
  </p>
  <p class="graf graf--p">
   他甚至试图让自己成为一个名人，宣称自己是 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.prnewswire.com/in/news-releases/cryptocurrency-guru-amit-bhardwaj-launches-pioneering-e-book-632585663.html" href="https://www.prnewswire.com/in/news-releases/cryptocurrency-guru-amit-bhardwaj-launches-pioneering-e-book-632585663.html" rel="noopener noreferrer" target="_blank">
    加密货币大师
   </a>
   ”。因为他领导的
   <a class="markup--anchor markup--p-anchor" data-href="https://coinjournal.net/exit-scam-vietnamese-cryptocurrency-company-goes-dark-after-allegedly-duping-investors-of-us660m/" href="https://coinjournal.net/exit-scam-vietnamese-cryptocurrency-company-goes-dark-after-allegedly-duping-investors-of-us660m/" rel="noopener noreferrer" target="_blank">
    投资成本
   </a>
   在7.69亿美元到20亿美元之间。
  </p>
  <p class="graf graf--p">
   并非所有名人都知道自己已经被参与其中了。在一篇博客文章中，iCenter 推出了一个视频，据称是 Dwayne “The Rock” Johnson 的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.buzzfeednews.com/article/ryanmac/cryptocurrency-scammers-are-running-wild-on-telegram" href="https://www.buzzfeednews.com/article/ryanmac/cryptocurrency-scammers-are-running-wild-on-telegram" rel="noopener noreferrer" target="_blank">
    代言
   </a>
   ，手持一个标有 iCenter 标志的牌子。Justin Timberlake 和 Christopher Walken 的视频被欺骗性地编辑成了他们赞扬 iCenter。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/3yGlurBytwA" width="750">
   </iframe>
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    欺诈性的初始代币产品
   </strong>
  </h3>
  <p class="graf graf--p">
   另一种流行的诈骗技术被称为 “初始代币发行”。一个潜在合法的投资机会，一个初始代币提供本质上是一个初创加密货币公司通过用户筹集资金的方式：换取像比特币和以太币这样的活跃加密货币，向客户承诺未来代币上线的高额回报。
  </p>
  <p class="graf graf--p">
   许多最初的代币产品已被证明是骗局，组织者利用狡猾的话术，甚至租用假办公室和制造得看上去很好的营销材料。
  </p>
  <p class="graf graf--p">
   2017年，关于加密货币的大量炒作和媒体报道引发了大量初始代币欺诈行为。
  </p>
  <p class="graf graf--p">
   在2018年，大约1,000个最初的代币项目崩溃，支持者至少在这上面损失了1亿美元。其中许多项目没有任何创意 ，超过15％的项目仅仅是从其他加密货币抄袭代码及概念。
  </p>
  <p class="graf graf--p">
   寻求新技术领域的投资者仍然对区块链和加密货币感兴趣，但应该注意它们是复杂的投资产品，即使对于销售它们的人来说也是新的。新投资者甚至相关专家都成了骗局的牺牲品。
  </p>
  <p class="graf graf--p">
   在当前的加密货币市场这样的环境中，潜在的投资者应该非常小心地研究自己投入的资金，并确保找出谁参与其中以及实际赚钱的计划是什么。⚪️
  </p>
