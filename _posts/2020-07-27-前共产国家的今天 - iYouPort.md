  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这只是政治”，马其顿共和国首都斯科普里的出租车司机 Branko 说。
  </p>
  <p class="graf graf--p">
   这是对马其顿窃听丑闻的普遍反应。
  </p>
  <p class="graf graf--p">
   此前一年，反对派领导人 Zoran Zaev 在网上发布了一系列他称为“爆炸性”的窃听程序 —— 这些信息表明，⚠️
   <strong class="markup--strong markup--p-strong">
    多年来，大约有20,000名活动家、律师、反对派成员、记者、公务员、商人，甚至政府成员中的异议，都受到非法监视
   </strong>
   。别忘了，马其顿是一个很小的国家。
  </p>
  <p class="graf graf--p">
   由欧洲委员会任命的高级法治专家
   <a class="markup--anchor markup--p-anchor" data-href="http://ec.europa.eu/enlargement/news_corner/news/news-files/20150619_recommendations_of_the_senior_experts_group.pdf" href="http://ec.europa.eu/enlargement/news_corner/news/news-files/20150619_recommendations_of_the_senior_experts_group.pdf" rel="noopener noreferrer" target="_blank">
    进行的评估
   </a>
   证实了许多指控。
  </p>
  <p class="graf graf--p">
   出租车司机 Branko 的观点是可以理解的，但可以说是被操纵了。
  </p>
  <p class="graf graf--p">
   多年来，马其顿的媒体格局一直受到合并运动的影响，媒体自由整体受到限制，编辑政策被所有权、财务关系和与高级职员的私人关系所控制。
  </p>
  <p class="graf graf--p">
   马其顿在《无国界记者》的2015年新闻自由指数中
   <a class="markup--anchor markup--p-anchor" data-href="https://index.rsf.org/#%21/index-details/MKD" href="https://index.rsf.org/#%21/index-details/MKD" rel="noopener noreferrer" target="_blank">
    排名第117位
   </a>
   ，仅次于塔吉克斯坦和卡塔尔。在2007年，它还排名第36位。
  </p>
  <p class="graf graf--p">
   反对派领导人声称，窃听是由当时的总理 Nikola Gruevski 和他的堂兄 Saso Mijalkov 进行的，他当时是马其顿情报机构 “安全与反间谍局”（UBK）的负责人。
  </p>
  <p class="graf graf--p">
   Gruevski 辩称窃听是由外国情报机构进行的，目的是“破坏国家稳定”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在这种受控的媒体空间中，很难知道可以信任谁。谣言盛行
   </strong>
   。
  </p>
  <p class="graf graf--p">
   丑闻的直接影响是深远的：大规模抗议导致欧盟促成一项协议，最终总理及其堂兄辞职，并于2016年6月举行了新的选举。
  </p>
  <p class="graf graf--p">
   然而，个人和马其顿社会必定会在未来的多年中继续“体验”这种大规模窃听的长期影响。
  </p>

  <h3 class="graf graf--p graf--startsWithDoubleQuote">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     “我们在倒退”
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   对于曾经居住在前南斯拉夫的人们来说，严密控制的气氛是熟悉的。
  </p>
  <p class="graf graf--p">
   马其顿是前社会主义联邦的成员之一，共产党在该国对媒体施加了严格的控制，并定期监视其人民，直到90年代初期血腥分裂为止。
  </p>
  <p class="graf graf--p">
   1976年，剧院导演 Vladimir Milchin 被执政党定为国家敌人，“无政府自由主义者”，并遭受到定点监视。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    直到2000年，他才开始能够读取到那些监听文件
   </strong>
   —— 被执政党记录了五年的他的私人谈话，他与家人、朋友、及所有人的谈话记录。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    而今天，他再次收到了同样的监听记录
   </strong>
   。但是这次是CD形式 ——
   <strong class="markup--strong markup--p-strong">
    监视不是共产主义国家进行的，而是由现代欧洲民主共和国的情报机构进行的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   2015年2月，反对党马其顿社会民主联盟（
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Social_Democratic_Union_of_Macedonia" href="https://en.wikipedia.org/wiki/Social_Democratic_Union_of_Macedonia" rel="noopener noreferrer" target="_blank">
    SDSM
   </a>
   ）开始向受害者提供他们自己被截获的电话的录音，以CD的形式。
  </p>
  <p class="graf graf--p">
   Vladimir 解释说：“我没有感到震惊。” 他已经担任马其顿开放社会研究所的执行主任，这是一个支持自由公民社会的国际基金会，由亿万富翁乔治·索罗斯（George Soros）资助。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “令人失望的是，这个国家现在与一党专制的共产主义国家一样，当年我就被共产党认为是内部敌人。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “今年我在街上遭到一名戴头巾的男子的袭击 …… 我收到了死亡威胁。有一次我被告知我的尸体也不会被允许埋在马其顿 ……这就是这个国家对待政治对手的方式。”
   </strong>
  </p>
  <p class="graf graf--p">
   Violeta Gligoroska 是终身活动家和新闻工作者，她还在开放社会研究所工作，她在五月份也收到了类似的私人对话窃听记录。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “感觉就像我被强奸了一样，就像我被国家强奸了一样。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我的父亲是被共产党开除党籍的，因为他是持不同政见者。当然，他受到了监视。我从没索要过那些监听文件。由于父亲已经去世，我没有勇气去寻求帮助。我希望内政部离我远点。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “不幸的是，我自己也没能幸免。”
  </p>
  <p class="graf graf--p">
   他们的基金会同事 Slavica Indjevska 解释说，他经常被亲政府的媒体妖魔化，被攻击为外国特工。这些媒体贬义地称他为“Sorosoids’’。
  </p>
  <p class="graf graf--p">
   上述这些人并不是特例。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    即使是针对性最强的监视也会影响到与目标接触的每个人。这就是现代监控的主要目标和问题之一
   </strong>
   。
  </p>
  <p class="graf graf--p">
   基金会工作人员的工作是关注社会和政治发展，并与记者和非政府组织（NGO）洽谈他们的项目、计划和意见。由于基金会遭到监视，因此政府可以直接监视到他们交谈和接触的每个人。
  </p>
  <p class="graf graf--p">
   Slavica 仍不敢在自己的办公室里收听那张监听录音的CD。星期天晚上，当她接到电话被告知她可以拿到那张监听CD时，她正在为穿越马其顿的过境难民准备救助包。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这是一种痛苦的感觉，当有人未经您同意就进入您的私人空间时，确实是一种可怕的感觉。就像有人潜入了你的房子，钻进了你的耳朵里……”。
  </p>
  <p class="graf graf--p">
   经验丰富的激进主义活动家 Violeta 说：“这是该国历史上最糟糕的时期。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “即使在共产主义时代，至少我们还知道自己一直在被监视…… 而如今的监视居然是秘密进行的”。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “就像东德一样；通过恐惧和宣传来毒害人们，以控制人们的思想和言谈。”
  </p>
  <p class="graf graf--p">
   Slavica 回顾了在独立公投，该公投见证了马其顿在90年代初离开南斯拉夫。“我带走了儿子，我认为对他来说这可能是光明的未来。但是，似乎哪里出了问题 …… 在90年代，我们见证了希望和变化，人们被激励着去超越自己的个人立场。但是现在，我们却倒退了。”
  </p>
  <p class="graf graf--p">
   她对未来并不乐观。她说，
   <strong class="markup--strong markup--p-strong">
    “我能看到国家拥有多少铁幕控制、压制，花了如此多的金钱，我能看到人们的恐惧，由于经济上的弱势而对强权的假设；人们很容易受到操纵，尤其是当你走投无路时”
   </strong>
   。
  </p>

  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   监视丑闻和相关抗议导致由欧盟委员会任命的一组高级法治专家对马其顿进行了调查，并在两个主要当事方之间达成了由欧盟代理的解决方案，即 所谓的 Przino 协议。
  </p>
  <p class="graf graf--p">
   Katica Janeva 得到了所有议会政党的认可，并被任命为特别检察官，负责调查窃听行为。
  </p>
  <p class="graf graf--p">
   但是，亲政府媒体已经对该任命提出了批评，而执政党
   <a class="markup--anchor markup--p-anchor" data-href="https://euobserver.com/opinion/131798" href="https://euobserver.com/opinion/131798" rel="noopener noreferrer" target="_blank">
    则表示
   </a>
   ，它“不再相信她的独立性”，并对“她的行为合法性严重保留意见”。
  </p>
  <p class="graf graf--p">
   特别检察官的主要任务之一是确定这种广泛的非法窃听是如何发生的并如何持续了如此长的时间。
  </p>
  <p class="graf graf--p">
   1991年《马其顿宪法》第17条明确保障了通讯的隐私性，其中指出“通讯和通讯方式的自由和保密需要得到保障。在刑事调查必不可少或为维护共和国的利益而必需的情况下，只有法院的决定才可以授权侦听……”。
  </p>
  <p class="graf graf--p">
   2005年，该国首次引入了《电子通信法》，以管理电信运营商与有权访问被截取的通信的政府机构之间的关系，此后发生了多处变化，
   <a class="markup--anchor markup--p-anchor" data-href="http://www.analyticamk.org/images/Files/impossible_en_final_9af93.pdf" href="http://www.analyticamk.org/images/Files/impossible_en_final_9af93.pdf" rel="noopener noreferrer" target="_blank">
    直到2014年
   </a>
   颁布了一项新法律。其中
   <a class="markup--anchor markup--p-anchor" data-href="http://www.pravo.org.mk/documentlaws.php?%20name=%D0%B5%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BD%D1%81%D0%BA%D0%B8+%D0%BA%D0%BE%D0%BC%D1%83%D0%BD%D0%B8%D0%BA%D0%B0%D1%86%D0%B8%D0%B8" href="http://www.pravo.org.mk/documentlaws.php?%20name=%D0%B5%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BD%D1%81%D0%BA%D0%B8+%D0%BA%D0%BE%D0%BC%D1%83%D0%BD%D0%B8%D0%BA%D0%B0%D1%86%D0%B8%D0%B8" rel="noopener noreferrer" target="_blank">
    第175条规定
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    运营商必须安装用于实时拦截通信所必需的技术
   </strong>
   。
  </p>
  <p class="graf graf--p">
   2010年，交通运输部提出了一项法律修正案，强制电信运营商向内政部提供直接且不受阻碍的流量和其他数据访问权限，而无需事先通知或法院命令。
  </p>
  <p class="graf graf--p">
   该法案获得通过后，
   <strong class="markup--strong markup--p-strong">
    马其顿的宪法法院在非政府组织的请愿书逼迫之下废除了该修正案。
   </strong>
  </p>
  <p class="graf graf--p">
   但是，非政府组织 Metamorphosis 的项目主管 Filip Stojanovski 当时曾对该修正案提出异议，他现在认为该丑闻表明，
   <strong class="markup--strong markup--p-strong">
    根本没有遵循法院的裁决
   </strong>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     直接访问
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   向政府机构提供不受限制的直接访问电信网络的特权，是电信行业内的一个主要但模糊的问题，这也是它们必须面对的问题。
  </p>
  <p class="graf graf--p">
   在马其顿，电信行业被以 T-Mobile 品牌运营的全球公司所主导。
  </p>
  <p class="graf graf--p">
   前国有的 Makedonski Telekom 负责提供固定电话服务。他们的股份由匈牙利运营商 Magyar Telecom 拥有51％，后者本身是德国电信的子公司，后者是全球领先的电信公司之一，也是 T-Mobile 品牌背后的实体。
  </p>
  <p class="graf graf--p">
   经过数年的努力，2015年7月1日，Makedonski Telekom 与子公司 T-Mobile Macedonia
   <a class="markup--anchor markup--p-anchor" data-href="http://www.telekom.hu/about_us/press_room/press_releases/2015/june_17" href="http://www.telekom.hu/about_us/press_room/press_releases/2015/june_17" rel="noopener noreferrer" target="_blank">
    合并
   </a>
   ，后者是马其顿领先的移动运营商。
  </p>
  <figure aria-describedby="caption-attachment-9778" class="wp-caption aligncenter" id="attachment_9778" style="width: 1300px">
   <img alt="" class="size-full wp-image-9778 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-6.png?resize=1100%2C727&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-6.png?w=1300&amp;ssl=1 1300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-6.png?resize=300%2C198&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-6.png?resize=768%2C507&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-6.png?resize=1024%2C677&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-6.png?resize=1100%2C727&amp;ssl=1 1100w" data-recalc-dims="1" height="727" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-6.png?resize=1100%2C727&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="size-full wp-image-9778" data-recalc-dims="1" height="727" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-6.png?resize=1100%2C727&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-6.png?w=1300&amp;ssl=1 1300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-6.png?resize=300%2C198&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-6.png?resize=768%2C507&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-6.png?resize=1024%2C677&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-6.png?resize=1100%2C727&amp;ssl=1 1100w" width="1100"/>
   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-9778">
    Skopje city centre, 2015. Privacy International
   </figcaption>
  </figure>
  <p class="graf graf--p">
   关于运营商向国家机构提供直接访问权限的普遍性究竟有多少，鲜为人知。
  </p>
  <p class="graf graf--p">
   虽然欧洲有一些由“欧洲电信标准协会”（ETSI）编纂的“合法拦截”技术标准，而其他标准，例如俄罗斯的“ SORM”，则适用于不同的规范。
  </p>
  <p class="graf graf--p">
   在俄罗斯和许多以前的社会主义国家/地区，其电信和侦听体系结构都基于SORM，
   <a class="markup--anchor markup--p-anchor" data-href="http://iks.sut.ru/publications/zakonnyy-perehvat-soobshcheniy-podhody-etsi-calea-i-sorm/" href="http://iks.sut.ru/publications/zakonnyy-perehvat-soobshcheniy-podhody-etsi-calea-i-sorm/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     该系统更具侵入性
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，因为电信公司几乎没有任何有意义的机会来控制国家机构的拦截活动、和/或调解国家机构使用其网络对个人数据的访问权限。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    全球最大的运营商之一
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://www.theguardian.com/business/2014/jun/06/vodafone-reveals-secret-wires-allowing-state-surveillance" href="https://www.theguardian.com/business/2014/jun/06/vodafone-reveals-secret-wires-allowing-state-surveillance" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     沃达丰（Vodafone）曾透露
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    一些政府也可以直接访问其网络，而不受沃达丰（Vodafone）的监督
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    在一些国家/地区，沃达丰（Vodafone）可以对用于接受代理机构或授权机构要求的“合法监听”的技术基础设施保持完全的运营控制。但是，在其他国家/地区，法律规定特定的机构和当局可以直接访问运营商的网络，而绕过运营商对合法拦截的任何形式的控制……
   </em>
  </p>
  <p class="graf graf--p">
   通常，运营商在由交换机、路由器以及爱立信、思科、华为或诺基亚等大型设备供应商提供的其他节点组成的基础架构上运行网络，该设施通常设计为符合合法的拦截标准。
  </p>
  <p class="graf graf--p">
   例如，用于确保电信网络正常运行的集成交换机本身通常用于将拦截的数据或内容转发到监视设备。
  </p>
  <p class="graf graf--p">
   一些公司提供了专门为合法拦截而设计的服务和系统，例如监视中心（执法机构在其中接收被拦截的材料），以及专门为合法拦截而设计的保留和调解套件。
  </p>
  <p class="graf graf--p">
   其他公司设计并销售探针，这些探针可以被动地收集拦截并将其转发给代理人，而无需使用网络节点，而其他公司则提供了用于随意收集大规模的被动数据的系统。
  </p>
  <p class="graf graf--p">
   尽管根据欧盟的调查已确定马其顿的情报机构UBK可以直接进入，但尚未确定如何在技术上进行、以及哪些公司提供了必要的设备。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     谁在提供监视技术？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   特别检察官的工作是查明如何从技术上进行拦截、使用的是哪种技术、以及由谁提供的拦截技术。
  </p>
  <p class="graf graf--p">
   在进行的第一批调查中显示，
   <strong class="markup--strong markup--p-strong">
    进行窃听所需的监视设备是由一家以色列公司提供的。
   </strong>
  </p>
  <p class="graf graf--p">
   去年4月，反对派释放出一份内部录音，音频表明，UBK负责人要求以色列国防部代表支付500,000欧元以进行购买，总价约为1400万欧元。
  </p>
  <p class="graf graf--p">
   马其顿内政部公开宣称，这笔钱仅仅是“我们在以色列政府中的合作伙伴提供的捐款”。也就是说他们承认存在这笔钱而且数额是对的。
  </p>
  <p class="graf graf--p">
   这份录音仍可以在网上找到，其中一些是英文的。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/otmi1qPXzpU" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   斯洛文尼亚一家鲜为人知的电信和监视设备制造商一直在马其顿从事各种合法拦截项目。
  </p>
  <p class="graf graf--p">
   马其顿的 Iskratel 实施了允许合法拦截 Makedonski Telekom 固网网络的项目，根据隐私国际组织的记录，该软件可用于可视化SORM3协议上的拦截数据。
  </p>
  <p class="graf graf--p">
   Iskratel 总部位于斯洛文尼亚 Kranj，为世界各地的客户开发和销售IP和固定线路合法侦听（LI）系统。
  </p>
  <p class="graf graf--p">
   Iskratel
   <strong class="markup--strong markup--p-strong">
    曾在2014年和2015年出席了国际上最臭名昭著的电子通讯监视技术非公开会议 ISS World，并在会议上介绍了其合法拦截解决方案
   </strong>
   。
  </p>
  <p class="graf graf--p">
   他们还邀请 ISS World 的代表参观他们的展台，称其“正在加强针对电信运营商和代理商的合法拦截领域的解决方案组合。
  </p>
  <p class="graf graf--p">
   下面是他们的宣传册内容：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    For lawful interception, Iskratel built a universal, highly flexible traffic-monitoring solution. The solution supports ETSI LI, SORM2 and SORM3 recommendations, and can be used in various IP-based networks — either fixed or mobile, either wired or wireless. The solution is based on the big-data concept — a concept that includes tools, processes and methods that a LEA needs to handle the new traffic types, large amounts of data and storage facilities.
   </em>
  </p>
  <p class="graf graf--p">
   SORM 标准的权威机构
   <a class="markup--anchor markup--p-anchor" data-href="http://www.agentura.ru/english/projects/Project_ID/masssurveillance/" href="http://www.agentura.ru/english/projects/Project_ID/masssurveillance/" rel="noopener noreferrer" target="_blank">
    Andrei Soldatov
   </a>
   表示，SORM2 是适用于所有IP流量的基于俄罗斯的拦截标准，而 SORM3 “负责收集所有通信、通信的长期存储、以及对所有用户数据的访问”。
  </p>
  <p class="graf graf--p">
   Iskratel 在其马其顿网站上将自己描述为 “过去60年马其顿共和国电信设备的主要提供商，并且是设计、实施和维护 ICT 解决方案的领先系统集成商。”
  </p>

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     如何解决这个问题
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   人权法专家、活动家和博客作者 Zarko Trajanovski 在研讨会上说：
   <strong class="markup--strong markup--p-strong">
    “没有任何法律机制可以解决这一问题 ……我们已经超越了菲德尔·卡斯特罗（Fidel Castro）时代的表现。”
   </strong>
  </p>
  <p class="graf graf--p">
   当然，没有唯一的解决方案可以确保在马其顿或其他任何地方不会再发生这种惊人的窃听丑闻。
  </p>
  <p class="graf graf--p">
   然而，Zarko 指出，
   <strong class="markup--strong markup--p-strong">
    任何解决方案都必须具有整体性，要求改革马其顿的所有当局和机构，并要求社会上所有团体采取有组织的行动。
   </strong>
  </p>
  <p class="graf graf--p">
   正如欧盟委员会高级专家小组的审查所表明的那样，问题并不在于管理马其顿的法律制度不完善，而是，
   <strong class="markup--strong markup--p-strong">
    “立法与实践之间存在相当大的差距”，尤其是“缺乏适当的管理、缺乏客观公正的实施。”
   </strong>
  </p>
  <p class="graf graf--p">
   Zarko 还同意，
   <strong class="markup--strong markup--p-strong">
    包括德国电信（Deutsche Telekom）在内的诸多外国公司在监视丑闻中所起的作用
   </strong>
   。
  </p>
  <p class="graf graf--p">
   大型跨国网络运营商必须采取强有力的一致行动，以确保它们不参与大规模的滥用行为。
  </p>
  <p class="graf graf--p">
   透明度报告是一个好的开始，但是必须优先考虑采取积极的行动，例如法律挑战和整个行业的举措。
  </p>
  <p class="graf graf--p">
   此外，必须更好地规范通常位于网络基础架构本身之上的合法拦截相关技术和其他监视技术的交易。
  </p>
  <p class="graf graf--p">
   如果一个国家的法律框架不充分，或者客户的记录表明其当权者参与了侵犯人权的行为，则不应为他们提供监视系统，因为有滥用和违反国际人权法的风险。
  </p>
  <p class="graf graf--p">
   正如马其顿正在进行的窃听事件所显示的那样，监视所产生的如此大规模的影响可能是毁灭性的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    执政党借助现代电信监视技术所获得的情报，能够对马其顿的每个部门和机构，包括媒体、公民社会、司法机构、和公务员，进行大量的政治控制
   </strong>
   。
  </p>
  <p class="graf graf--p">
   为确保在马其顿或其他地方不再发生这种情况，需要采取一种全面的、综合的方法，这要依靠确保存在符合国际人权标准的适当法律框架，各国在具有透明度机构和负责任的安全部门的情况下应该具有良好的治理水平，
   <strong class="markup--strong markup--p-strong">
    相关行业需要扮演积极的角色，个人可以访问安全的网络和设备，并且知道如何保护自己。
   </strong>
  </p>
  <p class="graf graf--p">
   禁止出口监视系统并提高行业透明度的保障措施也可以而且必须成为此解决方案的一部分。
  </p>
  <p class="graf graf--p">
   但是，在马其顿，
   <strong class="markup--strong markup--p-strong">
    需要更多实现的补救和改变 —— 需要来自本土的努力。现在，这是特别检察官、新闻记者、民间社会、政客及其公民的重要工作
   </strong>
   。⚪️
  </p>
