  <p class="graf graf--p">
   您能说出您所居住的城市的建筑有什么特点吗？颜色、风格、高度、布局都是什么样的？其他城市呢？您去过一次的地方、或者看过照片和视频的城市？
  </p>
  <p class="graf graf--p">
   很多人很难说清楚。因为平日没有注意观察过。我们在 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%99%9a%e5%81%87%e8%ae%b0%e5%bf%86%e5%92%8c%e8%ae%b0%e5%bf%86%e6%93%8d%e7%ba%b5%ef%bc%9a%e5%a1%91%e9%80%a0%e8%b6%85%e5%bc%ba%e8%ae%b0%e5%bf%86%e5%8a%9b%ef%bc%8811%ef%bc%89/" href="https://www.iyouport.org/%e8%99%9a%e5%81%87%e8%ae%b0%e5%bf%86%e5%92%8c%e8%ae%b0%e5%bf%86%e6%93%8d%e7%ba%b5%ef%bc%9a%e5%a1%91%e9%80%a0%e8%b6%85%e5%bc%ba%e8%ae%b0%e5%bf%86%e5%8a%9b%ef%bc%8811%ef%bc%89/" rel="noopener noreferrer" target="_blank">
    超强记忆力
   </a>
   ” 系列中介绍的诸多练习和测试中很多都是在强调观察能力，观察是记忆的基础，记忆是直觉的来源。
  </p>
  <p class="graf graf--p">
   也许没有哪个开源情报培训计划会建议您仅凭直觉就得出结论。这不是坏事，因为
   <strong class="markup--strong markup--p-strong">
    调查人员得出的任何结论都必须得到客观证据的支持
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是，任何有经验的研究人员将始终严重依赖于直觉，即使他们没有意识到这一点。原因很简单，专业人士的直觉是可以很强大的。
  </p>
  <p class="graf graf--p">
   不到一秒钟内大脑就会根据以前的经验进行许多无意识的计算，以便快速做出决定。直觉不仅是无意识的反应，就像
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amazon.co.uk/Blink-Power-Thinking-Without/dp/0141014598" href="https://www.amazon.co.uk/Blink-Power-Thinking-Without/dp/0141014598" rel="noopener noreferrer" target="_blank">
    Malcolm Gladwell
   </a>
   所说的那样， “不加思考地思考”。无视自己对某些事物的直觉是愚蠢的。
  </p>
  <p class="graf graf--p">
   对于开源情报调查也是如此。
  </p>
  <p class="graf graf--p">
   在本案例的侦探挑战演示中，nixintel 就采取了捕捉直觉的方法：首先判断该地点在东欧，完全是基于对照片外观的感觉。
  </p>
  <p class="graf graf--p">
   你也会这样做的 —— 如果一张照片中出现了糖葫芦和红灯笼，基本不会有人认为那有可能是北非或南美，对吗？
  </p>
  <p class="graf graf--p">
   这只是一个极为简单的类比。而
   <strong class="markup--strong markup--p-strong">
    在实际操纵中，直觉必须与广博的知识面密切相关，因为您的调查目标经常不会与您同一个城市
   </strong>
   。
  </p>
  <p class="graf graf--p">
   以下是 nixintel 的演示，在这里看到原文《
   <a href="https://nixintel.info/osint/where-in-the-world-global-and-local-identifiers-quiztime-12th-sept-2019/" rel="noopener noreferrer" target="_blank">
    Where In The World? Global and Local Identifiers – Quiztime 12th Sept 2019
   </a>
   》
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    全球与本地 —— 从大到小
   </strong>
  </h3>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12549 jetpack-lazy-image" data-lazy-sizes="(max-width: 707px) 100vw, 707px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1.jpeg?resize=707%2C943&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1.jpeg?w=707&amp;ssl=1 707w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1.jpeg?resize=225%2C300&amp;ssl=1 225w" data-recalc-dims="1" height="943" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1.jpeg?resize=707%2C943&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="707"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12549" data-recalc-dims="1" height="943" sizes="(max-width: 707px) 100vw, 707px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1.jpeg?resize=707%2C943&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1.jpeg?w=707&amp;ssl=1 707w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1.jpeg?resize=225%2C300&amp;ssl=1 225w" width="707"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   题目就是上面这张照片。这是一个很有挑战性的测试题目：其目标
   <strong class="markup--strong markup--p-strong">
    不仅在于要找到拍摄照片的具体位置，而且还要找出是从哪层建筑物拍摄到的照片
   </strong>
   。
  </p>
  <p class="graf graf--p">
   并且没有什么特别明显的迹象可以显示出它是在哪里被拍摄的，更不用说哪个城市或建筑物了。
  </p>
  <p class="graf graf--p">
   背景细节太模糊，无法用传统方法进行裁剪和反向图像搜索，前景中的椅子和桌子也并不是很独特。
  </p>
  <p class="graf graf--p">
   那么从哪里开始呢？nixintel 的直觉显示这是在东欧，尽管无法解释原因。事后看来，这一直觉完全是因为那个混凝土高层建筑物。它与西欧的任何地方都不像，而且植被和房屋对于亚洲或非洲来说也都不适合。
  </p>
  <p class="graf graf--p">
   还记得上一次的侦探挑战中介绍的
   <strong class="markup--strong markup--p-strong">
    思考方式技巧
   </strong>
   吗？《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a9%ba%e8%8d%a1%e8%8d%a1%e7%9a%84%e8%93%9d%e5%a4%a9%e4%b8%8a%e6%8e%a0%e8%bf%87%e4%b8%80%e6%9e%b6%e9%a3%9e%e6%9c%ba%e4%bd%a0%e8%83%bd%e7%9f%a5%e9%81%93%e5%9c%b0%e7%90%86%e4%bd%8d/" href="https://www.iyouport.org/%e7%a9%ba%e8%8d%a1%e8%8d%a1%e7%9a%84%e8%93%9d%e5%a4%a9%e4%b8%8a%e6%8e%a0%e8%bf%87%e4%b8%80%e6%9e%b6%e9%a3%9e%e6%9c%ba%e4%bd%a0%e8%83%bd%e7%9f%a5%e9%81%93%e5%9c%b0%e7%90%86%e4%bd%8d/" rel="noopener noreferrer" target="_blank">
    空荡荡的蓝天上掠过一架飞机……你能知道地理位置吗？- 史上最难的侦探挑战
   </a>
   》没错：从大到小。本案例中就采取了类似的方法。
  </p>
  <p class="graf graf--p">
   nixintel 使用 “全球” 和 “本地” 的概念来代替“大”和“小”。全球线索可能会形象地告诉您这一位置可能在哪；一旦使用全球线索来找到大概的区域，本地线索就可以帮您找到精确的位置。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12550 jetpack-lazy-image" data-lazy-sizes="(max-width: 707px) 100vw, 707px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/07/2.png?resize=707%2C943&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/07/2.png?w=707&amp;ssl=1 707w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/07/2.png?resize=225%2C300&amp;ssl=1 225w" data-recalc-dims="1" height="943" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/07/2.png?resize=707%2C943&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="707"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12550" data-recalc-dims="1" height="943" sizes="(max-width: 707px) 100vw, 707px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/07/2.png?resize=707%2C943&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/07/2.png?w=707&amp;ssl=1 707w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/07/2.png?resize=225%2C300&amp;ssl=1 225w" width="707"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   如上图，绿框中的项目被归类为 “全球” 指标 —— 如果可以识别它们，然后将它们放在一起，大致就能显示出这是哪个城市或城镇。
  </p>
  <p class="graf graf--p">
   红色框中的项目是 “本地” 指标：它们的独特性或突出性不足以帮助您在全球范围内找到位置，但是，如果可以使用全球指标找到正确的近似区域，那么就可以将本地指标用于正确的区域内。
  </p>
  <p class="graf graf--p">
   以下是一个摘要：
  </p>
  <p class="graf graf--p">
   全球指标 ——
  </p>
  <p class="graf graf--p">
   1）带有红色和白色相间的单层巴士。具体说它的下面是红色，其他地方都是白色。
  </p>
  <p class="graf graf--p">
   2）一组独特的高层建筑物，比周围的建筑物大得多。其中一个看起来像是黄色的。
  </p>
  <p class="graf graf--p">
   3）看起来像绿色的高速公路标志指向出口。似乎显示一位数的道路编号。
  </p>
  <p class="graf graf--p">
   本地指标 ——
  </p>
  <p class="graf graf--p">
   1）再次考虑高速公路标志。显示此位置会在高速公路旁，可能在出口处。
  </p>
  <p class="graf graf--p">
   2）一个奇怪的白色篷状屋顶。
  </p>
  <p class="graf graf--p">
   3）桌子和椅子。这可能是最 “本地” 的标识了，仅对验证最终的精确位置有用。
  </p>
  <p class="graf graf--p">
   因此，
   <strong class="markup--strong markup--p-strong">
    此位置必须符合以下条件
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    这个城市必须拥有红白相间的那种巴士
   </li>
   <li class="graf graf--li">
    这座城市的主要特征将是一座高大的黄色建筑
   </li>
   <li class="graf graf--li">
    黄色的高楼将在目标位置范围内
   </li>
   <li class="graf graf--li">
    该位置靠近可能只有一位数字的高速公路
   </li>
   <li class="graf graf--li">
    在该位置旁边，将有一个奇怪的白色帐篷状屋顶
   </li>
   <li class="graf graf--li">
    建筑物内将有一个咖啡馆或餐厅，其椅子和桌子看起来像照片中那样
   </li>
   <li class="graf graf--li">
    还有直觉显示可能是在东欧，尽管目前还没有真正的客观理由确定这点
   </li>
  </ul>
  <p class="graf graf--p">
   就是这样。下面逐一梳理这些线索。请注意，在梳理之前您并不知道其中哪个线索是最关键的，虽然一些经验显示公交车的外观是重要线索，但您需要有心理准备，因为经验会遭遇例外。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    公共汽车
   </strong>
  </h3>
  <p class="graf graf--p">
   从公共汽车开始，因为经验显示这将是找到国家和城市的最佳机会。
  </p>
  <p class="graf graf--p">
   根据直觉，首先需要将重点放在东欧，所以搜索方法是使用 Google 图片并搜索
   <strong class="markup--strong markup--p-strong">
    country AND red and white bus
   </strong>
   。有点粗糙，但有效。
  </p>
  <p class="graf graf--p">
   返回的结果首先是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.google.com/search?biw=1920&amp;bih=954&amp;tbm=isch&amp;sa=1&amp;ei=6SmFXdziJIaUlwTenbjYDg&amp;q=red+and+white+bus+%2B+poland&amp;oq=red+and+white+bus+%2B+poland&amp;gs_l=img.3...13203.19402..19899...0.0..0.101.1013.15j1......0....1..gws-wiz-img.......0i67j0j0i8i30j0i30j0i24.JBOYg_JfSGw&amp;ved=0ahUKEwic-6LWkuDkAhUGyoUKHd4ODusQ4dUDCAY&amp;uact=5" href="https://www.google.com/search?biw=1920&amp;bih=954&amp;tbm=isch&amp;sa=1&amp;ei=6SmFXdziJIaUlwTenbjYDg&amp;q=red+and+white+bus+%2B+poland&amp;oq=red+and+white+bus+%2B+poland&amp;gs_l=img.3...13203.19402..19899...0.0..0.101.1013.15j1......0....1..gws-wiz-img.......0i67j0j0i8i30j0i30j0i24.JBOYg_JfSGw&amp;ved=0ahUKEwic-6LWkuDkAhUGyoUKHd4ODusQ4dUDCAY&amp;uact=5" rel="noopener noreferrer" target="_blank">
    波兰
   </a>
   。这里有一些红色和白色的巴士，但它们很旧了，看起来不像图中的那辆。
