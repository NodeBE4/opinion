   <span class="s2">
    Wikileaks
   </span>
   <span class="s1">
    创始人的律师告诉法庭，
   </span>
   <span class="s2">
    Julian Assange
   </span>
   <span class="s1">
    病得非常严重，无法通过视频链接参加有关他可能被引渡到美国的最新听证会。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    此次听证会是自
   </span>
   <span class="s2">
    5
   </span>
   <span class="s1">
    月
   </span>
   <span class="s2">
    2
   </span>
   <span class="s1">
    日以来的第一起听证会，当时美国政府的律师开始对其案件进行审理，试图引渡他面临审判。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    但是，他的律师
   </span>
   <span class="s2">

   </span>
   <span class="s1">

   </span>
   <span class="s2">
    Assange
   </span>
   <span class="s1">

   </span>
  </p>
  <p class="p1">
   <span class="s1">
    下次听证会的日期确认为
   </span>
   <span class="s2">
    6
   </span>
   <span class="s1">
    月
   </span>
   <span class="s2">
    12
   </span>
   <span class="s1">
    日。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    高级地区法官
   </span>
   <span class="s2">
    Emma Arbuthnot

   <span class="s1">

   </span>
   <span class="s2">
    Belmarsh
   </span>
   <span class="s1">
    监狱举行，
   </span>
   <span class="s2">
    Assange
   </span>
   <span class="s1">
    被关押在那里。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    美国已要求引渡
   </span>
   <span class="s2">
    Assange
   </span>
   <span class="s1">
    ，他于
   </span>
   <span class="s2">
    4
   </span>
   <span class="s1">
    月
   </span>
   <span class="s2">
    11
   </span>
   <span class="s1">
    日被强行从厄瓜多尔驻伦敦大使馆带走。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    当时他被控串谋犯计算机入侵，最高刑罚为五年。
   </span>
  </p>

   <span class="s1">

    <b>

    </b>
   </span>
   <span class="s2">
    <b>
     Wikileaks
    </b>
   </span>
   <span class="s1">
    <b>

   </span>
   <span class="s2">

   <span class="s1">
    间谍法
   </span>
   <span class="s2">

  </p>
  <p class="p1">
   <span class="s1">
    <b>
     检察官宣布对
    </b>
   </span>
   <span class="s2">
    <b>
     Assange
    </b>

   <span class="s1">

    </b>
   </span>
   <span class="s2">
    <b>

    </b>
   </span>
   <span class="s1">
    <b>

    </b>

   </span>

   </span>
   <span class="s2">

   <span class="s1">

  </p>
  <p class="p1">
   <span class="s2">
    Wikileaks
   </span>
   <span class="s1">

   <span class="s2">

  <p class="p1">
   <span class="s2">
    “

   <span class="s1">

    </b>
    。
   </span>
   <span class="s2">
    ”
   </span>
  </p>
  <p class="p1">
   <span class="s1">

   </span>

   <span class="s2">

   </span>
   <span class="s1">

   <span class="s2">
    Assange

   <span class="s2">

   </span>
   <span class="s1">
    年
   </span>
   <span class="s2">

   <span class="s1">
    但是，
   </span>
   <span class="s2">
    Manning
   </span>
   <span class="s1">
    一再指出，她
    <b>
     并没有
    </b>

   </span>
   <span class="s2">
    2010
   </span>
   <span class="s1">

   <span class="s2">

   <span class="s1">
    她将伊拉克和阿富汗的
    <b>
     军事非法屠杀报告

  </p>
  <p class="p1">
   <span class="s1">

  </p>
  <p class="p1">
   <span class="s1">
    为了对
   </span>
   <span class="s2">

   <span class="s1">

   <span class="s2">

   <span class="s2">

   <span class="s1">

   <span class="s2">

  </p>
  <p class="p1">
   <span class="s2">

   <span class="s1">
