  <p class="graf graf--p">

  <p class="graf graf--p">
   有的时候，你对目标人的“了解”仅仅是一个电子邮件地址，这种时候可以怎么办？
  </p>
  <p class="graf graf--p">
   也许你第一步要做的就是，查看此人的社交媒体账户，以便从中查找到更多调查线索。
  </p>
  <p class="graf graf--p">
   有多种方法可以检查电子邮件地址是否链接到 Twitter 帐户。
  </p>
  <p class="graf graf--p">
   其中一个最简单的方法就是，通过 Gmail 添加目标，然后将其同步到您的 Twitter 帐户。这样，您将立即查看您的目标电子邮件地址是否已链接到 Twitter 上的帐户。
  </p>
  <p class="graf graf--p">
   本指南简单说说这件事怎么做。
  </p>
  <p class="graf graf--p">
   请按照以下步骤执行此检查。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    步骤1：在 Gmail 上使用假帐户登录
   </strong>
  </p>

  </p>
  <p class="graf graf--p">

   </strong>
   的帐户。
  </p>

   通过使用伪造的帐户，可以减少损害和被追踪的风险。避免被您调查的人了解到您的目的。

  </p>
  <p class="graf graf--p">
   在上一步打开的屏幕中，您可以查看您的联系人。
  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   要添加联系人，请点击屏幕左上方或屏幕底部中间的“创建联系人”。
  </p>
  <p class="graf graf--p">
   请注意：⚠️如果您的列表中已有其他联系人，我们建议您先将其删除。否则，在下一步中同步联系人时，您可能会得到令人困惑的结果。
