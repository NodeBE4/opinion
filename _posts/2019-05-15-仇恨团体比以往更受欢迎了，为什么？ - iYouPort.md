   仇恨团体比以往更受欢迎了，为什么？
  </h1>

 <div class="entry-content">
  <p class="graf graf--p">
   2018年，有11人在一家
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnn.com/2018/10/28/us/72-hours-of-hate-in-america/index.html" href="https://www.cnn.com/2018/10/28/us/72-hours-of-hate-in-america/index.html" rel="noopener noreferrer" target="_blank">
    犹太教堂被杀
   </a>
   ；两个人
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2018/11/02/us/tallahasee-shooting-yoga-florida.html" href="https://www.nytimes.com/2018/11/02/us/tallahasee-shooting-yoga-florida.html" rel="noopener noreferrer" target="_blank">
    在 Kroger 超市被杀
   </a>
   ；其他人
   <a class="markup--anchor markup--p-anchor" data-href="https://www.adl.org/media/12480/download" href="https://www.adl.org/media/12480/download" rel="noopener noreferrer" target="_blank">
    在瑜伽馆被杀
   </a>
   。这只是一年中极端主义暴力达到几十年来
   <a class="markup--anchor markup--p-anchor" data-href="https://www.adl.org/media/12480/download" href="https://www.adl.org/media/12480/download" rel="noopener noreferrer" target="_blank">
    最高峰
   </a>
   的一周。
  </p>
  <p class="graf graf--p">
   根据追踪美国极端主义行为的南方贫困法律中心的一份新报告，今年活跃的仇恨团体数量连续第四年上升至1,020个。根据 SPLC 的数据，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.splcenter.org/fighting-hate/intelligence-report/2019/year-hate-rage-against-change" href="https://www.splcenter.org/fighting-hate/intelligence-report/2019/year-hate-rage-against-change" rel="noopener noreferrer" target="_blank">
    2018年至少40起仇恨犯罪谋杀案
   </a>
   ，在自2017年以来被记录的数据基础上增加了一倍以上。
  </p>
  <p class="graf graf--p">
   正如 Liam Stack 在“纽约时报”对 SPLC 数据的报道中
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2019/02/20/us/hate-groups-rise.html" href="https://www.nytimes.com/2019/02/20/us/hate-groups-rise.html" rel="noopener noreferrer" target="_blank">
    所写道的
   </a>
   ，“这与其他令人担忧的事态发展相呼应，其中包括向 F.B.I 报告的仇恨犯罪数量增加了30％，从2015年到2017年。“
  </p>
  <p class="graf graf--p">
   反诽谤联盟（Anti-Defamation League）也进行过类似的研究，该联盟报告的数字甚至比 SPLC 还要高，至少为50起。ADL 还强调，这些谋杀“绝大多数都与右翼极端分子有关”，而不是圣战组织： “白人至上主义者对绝大多数杀人案负有责任，这种情况通常都是如此。”
  </p>
  <p class="graf graf--p">
   至于肇事者如何处理他们的罪行，ADL 写道：“枪支仍然是极端分子杀人的首选武器。 2018年50人的死亡案中有42人被枪支打死，其次是刀片或锋利武器。“
  </p>
  <p class="graf graf--p">
   正如 SPLC 指出的那样，黑人民族主义团体“支持反白人、反犹太主义或反同性恋和反变性观点”的数量也在2018年明显增长，从233增加到264，但事实上他们的影响力和行动已经变得苍白无力，与白人至上主义者、右翼仇恨团体相比。正如 SPLC 情报项目主任 Heidi Beirich 所写的那样，“与黑人民族主义团体不同，白人至上主义者团体在主流中有稳固的立足点。”
  </p>
  <p class="graf graf--p">
   Beirich 断言，这一*主流*包括特朗普政府及其支持者。“总统在福克斯新闻的帮助和怂恿下继续将他有害的反移民和反穆斯林思想推向公众意识，这助长了对即将到来的种族主义的担忧。”
  </p>
  <p class="graf graf--p">
   在特朗普当选之前，仇恨团体的数量在过去三年间实际上已经有所下降。现在，Beirich 在接受“纽约时报”采访时解释说：“现在有了更多的仇恨团体，更多的仇恨犯罪和更多的国内恐怖主义。这是一个令人不安的状况。“
  </p>
  <p class="graf graf--p">
   Beirich 强调特朗普只是一个更大的仇恨生态系统的一部分，“他是这种动态的关键方面，但是，他并不是唯一的原因。仇恨团体的行列正在增长。在网络空间传播仇恨的能力是关键。“
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.splcenter.org/sites/default/files/intelligence_report_166.pdf" href="https://www.splcenter.org/sites/default/files/intelligence_report_166.pdf" rel="noopener noreferrer" target="_blank">
    在这里
   </a>
   读到完整 SPLC 报告，以及
   <a class="markup--anchor markup--p-anchor" data-href="https://www.adl.org/media/12480/download" href="https://www.adl.org/media/12480/download" rel="noopener noreferrer" target="_blank">
    在这里
   </a>
   读到完整 ADL 报告。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    R

   <a class="markup--anchor markup--p-anchor" data-href="https://www.truthdig.com/articles/report-in-2018-hate-groups-are-more-popular-than-ever/" href="https://www.truthdig.com/articles/report-in-2018-hate-groups-are-more-popular-than-ever/" rel="noopener noreferrer" target="_blank">
    eport: In 2018, Hate Groups Are More Popular Than Ever.
   </a>
   Hate crime murders, at least 40 in 2018, according to the SPLC, have more than doubled since 2017.
  </p>
