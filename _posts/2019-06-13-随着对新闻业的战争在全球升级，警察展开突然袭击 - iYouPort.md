  <p class="graf graf--p">
   澳大利亚联邦警察在短短两天的时间内对记者进行了两次突然袭击，并查封了调查资料。
  </p>
  <p class="graf graf--p">
   5号，新闻集团澳大利亚记者 Annika Smethurst 的家
   <a class="markup--anchor markup--p-anchor" data-href="https://www.smh.com.au/politics/federal/afp-raid-home-of-news-corp-journalist-over-national-security-reporting-20190604-p51ub0.html" href="https://www.smh.com.au/politics/federal/afp-raid-home-of-news-corp-journalist-over-national-security-reporting-20190604-p51ub0.html" rel="noopener noreferrer" target="_blank">
    遭到澳大利亚联邦警察局的突袭
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    警方寻找与她去年所做的调查报告有关的信息
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该报告暴露了这样一个事实，即 澳大利亚政府一直在讨论给予自己前所未有的权力来监视公民的可能性
   </strong>
   。
  </p>
  <p class="graf graf--p">
   次日，警方又
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bbc.com/news/world-australia-48522729" href="https://www.bbc.com/news/world-australia-48522729" rel="noopener noreferrer" target="_blank">
    突袭了澳大利亚广播公司的悉尼总部
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    查抄了有关澳大利亚军队在阿富汗涉嫌战争罪的2017年调查报告的相关信息。
   </strong>
  </p>
  <p class="graf graf--p">
   另外，另一名澳大利亚记者
   <a class="markup--anchor markup--p-anchor" data-href="https://www.news.com.au/national/politics/australian-federal-police-raid-political-editor-annika-smethursts-home-over-spy-story/news-story/135c27ced2becde0333c0ef61d901007" href="https://www.news.com.au/national/politics/australian-federal-police-raid-political-editor-annika-smethursts-home-over-spy-story/news-story/135c27ced2becde0333c0ef61d901007" rel="noopener noreferrer" target="_blank">
    昨日透露
   </a>
   ，民政部已开始调查他所做的有关寻求庇护者的报道，这甚至有可能导致刑事案，他表示自己被迫披露信息来源。
  </p>

  <p class="graf graf--p">
   在悉尼突袭期间，澳大利亚天空新闻的
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/David_Speers/status/1136087799554174976" href="https://twitter.com/David_Speers/status/1136087799554174976" rel="noopener noreferrer" target="_blank">
    政治编辑 David Speers 发布推文
   </a>
   ： “Annika Smethurst 和ABC的调查报道中是否真的出现了新的证据？！”
  </p>
  <p class="graf graf--p">
   The Conversation 首席政治记者 Michelle Grattan 说：“如果这些袭击之间没有联系，就如AFP所说这是一个非同寻常的巧合；那么，为什么会在报道发布后长达两年的时间才开始突袭？这不是维稳效率太低了吗？必须有一个揭示，否则这是非常奇怪的”。
  </p>
  <p class="graf graf--p">
   确实奇怪。
  </p>
  <p class="graf graf--p">
   确实，AFP已经正式否认这两次袭击之间存在任何联系，事实上很难想象
   <strong class="markup--strong markup--p-strong">
    两次突袭具有有一个共同的主题，即：揭露政府希望保密的渎职行为。
   </strong>
  </p>
  <p class="graf graf--p">
   如果两次突袭之间真的“没有联系”，那么，这种恐慌带来了什么？

   <strong class="markup--strong markup--p-strong">

  <p class="graf graf--p graf--startsWithDoubleQuote">
   “对 Assange 的逮捕和间谍指控只是一个开始，正如媒体中的许多人、甚至那些讨厌 Assange 的人所担心的那样；澳大利亚主流记者的家遭到警方袭击。只因为她的调查，她的本职工作”，联合新闻主编
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/unjoe/status/1136114487755399168" href="https://twitter.com/unjoe/status/1136114487755399168" rel="noopener noreferrer" target="_blank">
    Joe Lauria 说
   </a>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">

   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/johnpilger/status/1136222003436933120" href="https://twitter.com/johnpilger/status/1136222003436933120" rel="noopener noreferrer" target="_blank">

   </a>
   ：
   <strong class="markup--strong markup--p-strong">
    “一个逮捕令就可以允许警方在ABC新闻的电子设备上添加、复制、删除或修改’ 文件。对 Julian Assange 的攻击是对我们所有人的明确警告：这只是一个开始。“
   </strong>
  </p>
  <p class="graf graf--p">
   如果你考虑一下，如果两次袭击之间存在联系，那么它将远远不如替代可能性那么令人不安，因为替代可能性更加险恶：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    澳大利亚政府对新闻自由的态度已经发生了变化。他们已经在这样做了，正如澳大利亚已经做了几十年的那样 —— 与其他帝国的行为保持一致
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在一份题为《Assange被起诉后警方针对更多调查记者发起攻击行动》的文章中，Joe Lauria 提醒我们，澳大利亚并不是西方权力联盟中第一个被看到以这种范式升级的国家 —— 就在英国非法抓捕 Julian Assange 之后。
  </p>

  </p>
  <p class="graf graf--p">
   这种趋势背后不一定有任何中央计划的阴谋；它可以非常简单地只是一个生病的帝国的自然结果，它需要更多的战争、谎言和欺骗，以防止崩溃，并相应地作出反应。
  </p>
  <p class="graf graf--p">
   ⚠️一旦 Assange 这条线被跨越，它就可以成为一个先例，帝国内其他政府就可以毫无障碍地开始做他们一直想要做的事。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5246 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4.png?resize=1100%2C1240&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4.png?w=1146&amp;ssl=1 1146w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4.png?resize=266%2C300&amp;ssl=1 266w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4.png?resize=768%2C866&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4.png?resize=908%2C1024&amp;ssl=1 908w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4.png?resize=1100%2C1240&amp;ssl=1 1100w" data-recalc-dims="1" height="1240" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4.png?resize=1100%2C1240&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5246" data-recalc-dims="1" height="1240" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4.png?resize=1100%2C1240&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4.png?w=1146&amp;ssl=1 1146w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4.png?resize=266%2C300&amp;ssl=1 266w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4.png?resize=768%2C866&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4.png?resize=908%2C1024&amp;ssl=1 908w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4.png?resize=1100%2C1240&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Julian Assange 是一个历史性重要问题末尾的问号，我们现在都在问这个问题。
  </p>
  <p class="graf graf--p">
   即：人类是否希望建立一个基于真相并掌握权力的社会、或者它要的东西是否完全相反？
  </p>
  <p class="graf graf--p">
   到目前为止，对这个问题的普遍共识是这样的：“对奥威尔式反乌托邦的猛烈抨击真的让我们感觉很好，谢谢。”
  </p>
  <p class="graf graf--p">
   但是，随着这个答案的含义变得更加清晰和透彻，我们可能会在为时已晚之前看到另一个方向。⚪️
  </p>
