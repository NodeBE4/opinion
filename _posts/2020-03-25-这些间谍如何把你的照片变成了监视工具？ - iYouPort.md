  <p class="graf graf--p">

   但是委员会中没有任何人质问这位亿万富翁首席执行官关于利用 Facebook 上的数据牟利的监控公司的信息。

   <strong class="markup--strong markup--p-strong">

   </strong>
  </p>
  <p class="graf graf--p">
   该数据库构成了面部识别服务 Face-Int 的核心，该面部识别服务在2017年收购了鲜为人知的监视公司 Terrogence 之后，现在由以色列供应商 Verint 拥有。
  </p>
  <p class="graf graf--p">
   ⚠️Verint 和 Terrogence 长期以来一直是美国政府的供应商，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wired.com/2012/04/shady-companies-nsa/" href="https://www.wired.com/2012/04/shady-companies-nsa/" rel="noopener noreferrer" target="_blank">
    向NSA
   </a>
   、美国海军、
   <strong class="markup--strong markup--p-strong">
    以及无数其他情报和间谍机构提供尖端间谍技术
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.terrogence.com/capabilities/biometric-database-enhancement/" href="https://www.terrogence.com/capabilities/biometric-database-enhancement/" rel="noopener noreferrer" target="_blank">
    Terrogence 网站
   </a>
   上所述，该数据库包含数千个 “从 YouTube、Facebook 和 全球开放和封闭论坛等在线资源中收集” 的嫌疑人的面部数据。这就是 “社交媒体情报”。
  </p>
  <p class="graf graf--p">
   根据互联网档案馆 Wayback Machine 的数据，该营销页面在2013年上线，
   <strong class="markup--strong markup--p-strong">
    表明该产品至少已有五年历史了
   </strong>
   。
  </p>
  <p class="graf graf--p">

    </strong>
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" rel="noopener noreferrer" target="_blank">
     互联网霸主的时间线
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   该公司的联合创始人 Arbel 曾是以色列军队的一名情报官员，但他拒绝回答有关该技术如何运作的问题，尽管他在短信中称 Face-Int 令人称奇，并确认其在 Verint 的领导下仍在继续运作。
  </p>
  <p class="graf graf--p">
   Facebook 的发言人表示，Terrogence 的产品 “似乎违反了其政策”，其中包括禁止使用从社交网络获取的数据来提供监视工具，Facebook 利用自己的面部识别技术来识别整个平台中用户的照片。但是 Facebook 难道有可能没有从中获利？这一猜想并不符合监视资本家的思维逻辑。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    社交媒体监视器
   </strong>
  </p>
  <p class="graf graf--p">
   没有证据表明美国已经购买了 Face-Int。但是它从 Terrogence 建立的其他情报服务中受益。
  </p>
  <p class="graf graf--p">
   根据公开记录，
   <strong class="markup--strong markup--p-strong">
    该间谍技术供应商与美国政府至少签订了两份合同，均与美国海军签订，总价值为148,000美元。
   </strong>
  </p>
  <p class="graf graf--p">
   这些合同分别来自2014年和2015年，用于订购该公司的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.terrogence.com/files/3014/7878/9778/Mobius_Catalogue_Update_2013-May_2015.pdf" href="https://www.terrogence.com/files/3014/7878/9778/Mobius_Catalogue_Update_2013-May_2015.pdf" rel="noopener noreferrer" target="_blank">
    Mobius
   </a>
   和 TGAlertS 产品。
  </p>
  <p class="graf graf--p">
   Mobius 包含有关简易爆炸装置（IED）及其恐怖分子策略的最新趋势的报告，这是该公司的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/4438368-Terrogence-Brochure-Email-Version.html" href="https://www.documentcloud.org/documents/4438368-Terrogence-Brochure-Email-Version.html" rel="noopener noreferrer" target="_blank">
    小册子
   </a>
   上写的。这些报告是基于从各种社交媒体平台收集的情报得出的。
   <strong class="markup--strong markup--p-strong">
    TGAlertS 同时提供 Terrogence 员工大范围监视时发现的 “几乎实时” 的信息。
   </strong>
  </p>
  <p class="graf graf--p">
   这些员工部分地通过虚假的账户在线收集信息。
   <strong class="markup--strong markup--p-strong">
    也就是
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/?s=%E8%A2%9C%E5%AD%90%E6%9C%A8%E5%81%B6" href="https://www.iyouport.org/?s=%E8%A2%9C%E5%AD%90%E6%9C%A8%E5%81%B6" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     袜子木偶
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p">
   正如另一本小册子中指出的那样，⚠️他们 “
   <strong class="markup--strong markup--p-strong">
    通过仔细引导在线讨论来获取信息，经常引起目标人的兴趣、并通过在单个操作中使用多个虚拟账户来促进交流
   </strong>
   ”。
  </p>
  <p class="graf graf--p">
   ⚠️再次提醒，社交媒体上当任何陌生人试图进一步 “探讨”时，你应该小心自己说出的每一句话。不论该陌生人的个人简介写得多漂亮，因为有可能是假的。
  </p>
  <p class="graf graf--p">
   Arbel 在监视工业园区中的第一个牛仔竞技是他与其他人共同创立的 SenseCy，该公司于2017年被 Verint 收购。它还专门建立了 “虚假实体” 来收集情报。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “ SenseCy 经过多年的实践，拥有数十个虚假实体，结合了强大而可信的封面故事以及完善的Web交互方法，并从所有相关的Web平台中获取了宝贵的情报” ，其网站上的一位人士说。（虚假实体就是袜子木偶）
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    黑名单的私有化
   </strong>
  </p>
  <p class="graf graf--p">
   Terrogence 并非仅仅专注于恐怖主义，其在政治方面的业务应该令人恐惧，其面部识别技术可能已经席卷了大量人群。
  </p>
  <p class="graf graf--p">
   Stanley 警告说，这带来了 Terrogence 业务的另一个特别令人担忧的方面：“黑名单私有化的开端”。
  </p>
  <p class="graf graf--p">
   他说：“多年来，我们一直在就这些黑名单的正当程序与政府进行斗争……无辜的人被放在这些黑名单上而没有被告知原因，也无法确定当权者会如何使用这些黑名单。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当权者如何使用这些黑名单？也许并不难知道，以下是最基本的3个例子
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    当活动家和异议人士被纳入“恐怖分子”黑名单《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%88%91%e8%a2%ab%e8%ad%a6%e5%af%9f%e6%8a%98%e7%a3%a8%e8%87%b3%e7%b2%be%e7%a5%9e%e5%b4%a9%e6%ba%83%ef%bc%9a%e7%8e%b0%e5%9c%a8%e6%88%91%e5%bf%85%e9%a1%bb%e7%88%ac%e8%b5%b7%e6%9d%a5%e6%88%98%e6%96%97/" href="https://www.iyouport.org/%e6%88%91%e8%a2%ab%e8%ad%a6%e5%af%9f%e6%8a%98%e7%a3%a8%e8%87%b3%e7%b2%be%e7%a5%9e%e5%b4%a9%e6%ba%83%ef%bc%9a%e7%8e%b0%e5%9c%a8%e6%88%91%e5%bf%85%e9%a1%bb%e7%88%ac%e8%b5%b7%e6%9d%a5%e6%88%98%e6%96%97/" rel="noopener noreferrer" target="_blank">
     我被警察折磨至精神崩溃：现在我必须爬起来战斗
    </a>
    》
   </li>
   <li class="graf graf--li">
    当无辜的平民被政府间谍记错了电话号码《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%9a%97%e6%9d%80%e8%ae%a1%e5%88%92/" href="https://www.iyouport.org/%e6%9a%97%e6%9d%80%e8%ae%a1%e5%88%92/" rel="noopener noreferrer" target="_blank">
     暗杀计划
    </a>
    》
   </li>
   <li class="graf graf--li">
    当记者被政府纳入《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%88%ba%e6%9d%80%e5%90%8d%e5%8d%95/" href="https://www.iyouport.org/%e5%88%ba%e6%9d%80%e5%90%8d%e5%8d%95/" rel="noopener noreferrer" target="_blank">
     刺杀名单
    </a>
    》……
   </li>
  </ul>
  <p class="graf graf--p">
   去年底，Verint
   <a class="markup--anchor markup--p-anchor" data-href="https://www.verint.com/press-room/2018-press-releases/verint-introduces-facial-recognition-technology-built-for-real-world-environments.html" href="https://www.verint.com/press-room/2018-press-releases/verint-introduces-facial-recognition-technology-built-for-real-world-environments.html" rel="noopener noreferrer" target="_blank">
    推出
   </a>
   了看起来完全独立的面部识别产品 FaceDetect。它声称可以打破伪装而识别个人，并 “允许操作员立即将嫌疑人加入重点监视黑名单”。
  </p>
  <p class="graf graf--p">
   Stanley 还质疑 Facebook 所谓的用户控制个人资料照片的政策。
  </p>
  <p class="graf graf--p">
   他说，该社交网络拥有全世界最大的面孔数据集合，但个人头像在一定程度上无法完全锁定（每个人都可以看到你的头像，如果你的头像是真人照片，有可能很危险）。
  </p>
  <p class="graf graf--p">
   像ACLU这样的隐私倡导组织现在希望看到用户获得了对这些图像的更多控制权。但是我们真的能寄希望于监视资本家的 “开明” 吗？或者指望狼忘了怎么吃肉？你知道这有多难（即便不是不可能的话），我们必须先自己保护自己，才有机会为改变而战斗。⚪️
  </p>
