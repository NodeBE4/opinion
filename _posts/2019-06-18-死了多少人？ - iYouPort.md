  <p class="graf graf--p">
   人们需要关注这场大规模战争正在发生的事实。
  </p>
  <p class="graf graf--p">
   Halima Farah 56岁，是六个孩子的母亲，像她这样的人回到索马里绝不是什么好的选择。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这是一次自杀任务，”她
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/world/2019/03/08/kenyan-refugee-camp-rep-ilhan-omar-sparks-dreams-drawings-not-controversy/?utm_term=.3316f32008c2" href="https://www.washingtonpost.com/world/2019/03/08/kenyan-refugee-camp-rep-ilhan-omar-sparks-dreams-drawings-not-controversy/?utm_term=.3316f32008c2" rel="noopener noreferrer" target="_blank">
    告诉华盛顿邮报
   </a>
   。这是因为这个位于肯尼亚和埃塞俄比亚之间的非洲之角的国家，已经成为三十年持久战争的战场 — — 而且只会变得更糟。几年前，她逃到了肯尼亚的一个难民营，以摆脱战争的噩梦。
  </p>
  <p class="graf graf--p">
   根据“纽约时报”的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2019/03/10/us/politics/us-somalia-airstrikes-shabab.html" href="https://www.nytimes.com/2019/03/10/us/politics/us-somalia-airstrikes-shabab.html" rel="noopener noreferrer" target="_blank">
    一篇文章
   </a>
   ，美国军方已经加剧了对青年党的战争，这是一个与基地组织有联系的极端组织。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “纽约时报”的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2019/03/10/us/politics/us-somalia-airstrikes-shabab.html" href="https://www.nytimes.com/2019/03/10/us/politics/us-somalia-airstrikes-shabab.html" rel="noopener noreferrer" target="_blank">
    深度报道显示
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    美军在索马里执行的无人机袭击造成的死亡人数已连续多年创下历史新高。据美国国防部获得的数据显示，去年的47起袭击事件导致死亡人数达到326人。据报道，今年的战争有望超过这些数字，仅在1月至2月期间就有225人在24次披露的袭击中丧生
   </strong>
   。
  </p>
  <p class="graf graf--p">
   像 ACLU 这样的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/commentisfree/2018/sep/17/the-cia-is-back-in-the-drone-business-trump-is-unshackling-americas-drones-thanks-to-obamas-weakness" href="https://www.theguardian.com/commentisfree/2018/sep/17/the-cia-is-back-in-the-drone-business-trump-is-unshackling-americas-drones-thanks-to-obamas-weakness" rel="noopener noreferrer" target="_blank">
    人权组织说
   </a>
   ，平民 — 包括儿童 — 在多年来的空袭中被大批地杀害，包括在奥巴马任期内。但“华盛顿邮报”这样的官方媒体
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/world/2019/03/08/kenyan-refugee-camp-rep-ilhan-omar-sparks-dreams-drawings-not-controversy/?utm_term=.3316f32008c2" href="https://www.washingtonpost.com/world/2019/03/08/kenyan-refugee-camp-rep-ilhan-omar-sparks-dreams-drawings-not-controversy/?utm_term=.3316f32008c2" rel="noopener noreferrer" target="_blank">
    称
   </a>
   ，由于“在战斗中无法进入索马里的大部分地区”，因此很难量化这些说法。
  </p>
  <p class="graf graf--p">
   国际危机组织（International Crisis Group）的负责人布列塔尼•布朗（Brittany Brown）是一个致力于减少致命冲突的非营利组织，他将索马里的战争称为“全自动操作”。在奥巴马和特朗普政府期间，布朗还在国家安全委员会的索马里政策上工作。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “人们需要关注这场大规模战争正在发生的事实，”她告诉“纽约时报”。
  </p>
  <p class="graf graf--p">
   2017年，白宫批准了针对索马里武装分子的战斗扩张。据
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bbc.com/news/world-africa-46612542" href="https://www.bbc.com/news/world-africa-46612542" rel="noopener noreferrer" target="_blank">
    英国广播公司报道
   </a>
   ，该批准取消了曾经对青年党进行袭击所需的高级别审查。
  </p>
  <p class="graf graf--p">
   General Thomas D. Waldhauser 将军
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2017/03/30/world/africa/trump-is-said-to-ease-combat-rules-in-somalia-designed-to-protect-civilians.html" href="https://www.nytimes.com/2017/03/30/world/africa/trump-is-said-to-ease-combat-rules-in-somalia-designed-to-protect-civilians.html" rel="noopener noreferrer" target="_blank">
    在新闻发布会上说
   </a>
   ：“在决策过程中，我们拥有更多的灵活性，更多的时效性，这非常重要，也非常有帮助。” “这使我们能够以更快的方式起诉目标。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    特朗普政府的另一项改变意味着平民死亡人数将被少报
   </strong>
   。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bbc.com/news/world-us-canada-47480207" href="https://www.bbc.com/news/world-us-canada-47480207" rel="noopener noreferrer" target="_blank">
    正如英国广播公司所指出的那样
   </a>
   ，特朗普撤销了巴拉克•奥巴马（Barack Obama）通过2016年行政命令实施的政策，该政策要求中央情报局记录在阿富汗、巴基斯坦和索马里等国家进行无人机袭击，包括战区以外的袭击，以公布被杀害的平民人数，报道还说，该命令“并没有推翻国会为军方设定的平民死亡报告要求”。
  </p>
  <p class="graf graf--p">
   据报道，特朗普政府称奥巴马的行政命令“多余”，令人分心。
  </p>
  <p class="graf graf--p">
   这绝不是第一个执行无人机袭击的美国总统，甚至不是第一个为袭击申辩的政府。
  </p>
  <p class="graf graf--p">
   正如BBC新闻白宫记者 Tara McKelvey 指出的那样，奥巴马政府也因空袭造成大批平民死亡而受到批评。但她表示，随着数字以前所未有的速度增长，这个问题已经加剧了。
  </p>
  <p class="graf graf--p">
   根据研究人员的说法，“在奥巴马上任八年期间，已经进行了1,878次无人机袭击”，McKelvey
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bbc.com/news/world-us-canada-47480207" href="https://www.bbc.com/news/world-us-canada-47480207" rel="noopener noreferrer" target="_blank">
    写道
   </a>
   。 “自特朗普于2016年当选以来，已有2,243架无人机袭击。共和党总统也进行了一些行动，即战区之外的行动，更加隐秘。因此，今天的情况有所不同：在特朗普的领导下，有更多的无人机袭击 — 并且透明度更低。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.teenvogue.com/story/trump-administration-drone-war-somalia-air-strikes" href="https://www.teenvogue.com/story/trump-administration-drone-war-somalia-air-strikes" rel="noopener noreferrer" target="_blank">
    he Trump Administration Is Waging a Drone War in Somalia With Increased Air Strikes.
   </a>
   “People need to pay attention to the fact that there is this massive war going on.”
  </p>
