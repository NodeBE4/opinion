  <p class="graf graf--p">
   也许您早已有这样的体会：在互联网上，只有那些口水和浅薄的（一次性的、没有保留价值的）内容会被更多人分享，但是，实用性的知识被分享程度会极低。
  </p>
  <p class="graf graf--p">
   曾经很多人对这种现象做出过自己的解读，从虚拟空间的娱乐化到行动力差、从算法操纵到回音壁气泡 …… 也许，各种原因都有，但是，
   <strong class="markup--strong markup--p-strong">
    一些更明显的原因被忽略了。
   </strong>
  </p>
  <p class="graf graf--p">
   我们先抛开互联网媒介。如果你曾经在一个团队中与其他人协同工作，不论是公司还是基层组织，也许你已经有这样的体验，即：某些工作只能某个人来完成，而此人往往不会倾向于教会更多其他人关于如何完成这件事的技术和经验，即便你很想学，也经常会遭到委婉的拒绝。
  </p>
  <p class="graf graf--p">
   这是为什么呢？
  </p>
  <p class="graf graf--p">
   这个问题的答案与开头第一个问题的答案是一致的：
   <strong class="markup--strong markup--p-strong">
    大多数人不会倾向于分享高质量的知识 —— 也就是那些你无法从教科书中获得的东西，它必须来自大量的实践。于是它往往是私人的。
   </strong>
  </p>
  <p class="graf graf--p">
   这是件麻烦事。不论您是公司管理人员，还是公民社会组织者，您都会期待您的下属和队友彼此分享知识；因为，毕竟，这样可以为个人、团队和组织带来更大的创造力、更多的创新和更好的性能。
  </p>
  <p class="graf graf--p">
   然而，尽管组织者试图鼓励知识共享，但许多成员还是隐瞒了自己所知道的东西 ——
   <strong class="markup--strong markup--p-strong">
    这不是罕见的，这种现象被称为知识囤积或知识隐藏
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在需要分享知识的时候，人们会下意识装傻，假装不知道某些东西，或者承诺分享但从来不这样做，或者直接告诉别人不可能共享。
  </p>
  <p class="graf graf--p">
   《组织行为学杂志》上的
   <a class="markup--anchor markup--p-anchor" data-href="https://onlinelibrary.wiley.com/doi/full/10.1002/job.2364" href="https://onlinelibrary.wiley.com/doi/full/10.1002/job.2364" rel="noopener noreferrer" target="_blank">
    一篇研究
   </a>
   发现，工作的设计方式会影响成员是否向同事共享知识或向队友隐藏知识。
  </p>
  <p class="graf graf--p">
   具体来说，认知上更为复杂的工作（人们需要处理大量信息并解决复杂的问题）这种情况下倾向于促进更多的知识共享，而提供更多自治的工作也有可能促进知识共享。
  </p>
  <p class="graf graf--p">
   👉而
   <strong class="markup--strong markup--p-strong">
    在竞争环境下，最主要的就是抑制成员之间的知识共享
   </strong>
   。
  </p>
  <p class="graf graf--p">
   现在你知道的了，由于互联网鼓励的是竞争，人们彼此间更多是为了争夺更大的自身影响力；这种情况会明显压制知识的流动，并鼓励那些肤浅的观点占据大部分流量 —— 人们倾向于转发后者是由于它们代表 “
   <strong class="markup--strong markup--p-strong">
    我的
   </strong>
   立场”，积累 “
   <strong class="markup--strong markup--p-strong">
    我的
   </strong>
   ” 声誉。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%99%be%e5%bf%a7%e8%a7%a3%e6%97%b6%e4%bb%a3%ef%bc%88%e4%b8%80%ef%bc%89%e5%8f%98%e8%b4%a8%e7%9a%84%e7%ab%9e%e4%ba%89%e5%92%8c%e6%97%b6%e4%bb%a3%e6%80%a7%e7%84%a6%e8%99%91/" href="https://www.iyouport.org/%e7%99%be%e5%bf%a7%e8%a7%a3%e6%97%b6%e4%bb%a3%ef%bc%88%e4%b8%80%ef%bc%89%e5%8f%98%e8%b4%a8%e7%9a%84%e7%ab%9e%e4%ba%89%e5%92%8c%e6%97%b6%e4%bb%a3%e6%80%a7%e7%84%a6%e8%99%91/" rel="noopener noreferrer" target="_blank">
     百忧解时代（一）变质的竞争和时代性焦虑
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   这项研究收集了来自澳大利亚各个组织的394名知识工作者和中国一家出版公司的195名知识工作者的数据。
  </p>
  <p class="graf graf--p">
   研究人员对参与者进行了调查，以了解他们在工作上的认知需求、他们的自主权以及同事之间在工作上的依赖程度。还询问了他们与他人分享知识的动机。
  </p>
  <p class="graf graf--p">
   几个月后，研究人员要求参与者报告与同事共享知识的频率、以及他们认为该知识有用的程度。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    分析得出了三个关键发现
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    首先，人们出于不同原因共享和隐藏知识；
   </li>
   <li class="graf graf--li">
    其次，如上所述，当人们从事对认知有要求的工作并拥有很多自主权时，他们会更愿意分享知识；
   </li>
   <li class="graf graf--li">
    第三，如果他们认为同事过于依赖他们，就更有可能隐藏知识。
   </li>
  </ul>
  <p class="graf graf--p">
   下面逐一介绍一下。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、是什么促使人们分享或隐藏知识？
   </strong>
  </h3>
  <p class="graf graf--p">
   在分析有关动机的数据时，研究人员将其参与者的回答归类为 “自主动机”（这意味着做某事是因为它有意义或令人愉快）、或“受控动机”（这意味着要做某事是为了获得奖励或避免惩罚）。
  </p>
  <p class="graf graf--p">
   研究结果表明，当成员具有自主动机时，知识共享就更有可能 —— 例如，他们同意 “与队友分享我所知道的事可能很重要”、或 “谈论我所知道的事很有趣” 这样的想法。
  </p>
  <p class="graf graf--p">
   相反，当人们的动力是由外部压力驱动时（比如 “我不想受到批评” 或 “不这样做的话我可能会被排斥”），人们反而更有可能隐藏自己的知识。
  </p>
  <p class="graf graf--p">
   👉这意味着，*迫使*人们分享知识而不是让他们看到知识分享本身的价值，就不会有好的结果。
  </p>
  <p class="graf graf--p">
   如果你的队友不了解共享知识以达到组织目标的重要性，那么他们共享该知识的可能性就很小。
  </p>
  <p class="graf graf--p">
   并且，如果你的队友被迫分享自己的知识（比如你把这条写入组织规范了，并添加了隐藏知识的惩罚方式），那么就很可能适得其反。
  </p>
  <p class="graf graf--p">
   👉如上所述，如果人们害怕失去竞争优势，他们就更不愿意透露信息了。
  </p>
  <p class="graf graf--p">
   但要知道，这其中可能有社会文化方面的作用。在这项研究的中国样本中，上述 “受控动机” 与知识共享的频率增加有关，但与共享知识的更大效用却没有关系 —— 简单说，在中国的案例中，人们更可能会在惩罚压力下共享知识，而非对共享本身的价值的认同。
  </p>
  <p class="graf graf--p">
   所以，在组织运作中，作为领导者您应该考虑文化因素。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、什么样的工作会促进知识共享或导致隐藏？
   </strong>
  </h3>
  <p class="graf graf--p">
   对认知的要求很高的工作可能会更加有趣和刺激，并且也更加困难和充满挑战，于是您会希望队友既喜欢共享信息、又能看到共享的更大利好。
  </p>
  <p class="graf graf--p">
   同样，由于在工作中拥有更大的自主权会导致发现它更有意义，所以您会希望看到同样的分享倾向。
  </p>
  <p class="graf graf--p">
   这项研究的结果支持了这些想法。
  </p>
  <p class="graf graf--p">
   👉当人们的工作涉及较高的认知要求和自主权时，他们会更频繁地共享有用的知识。澳大利亚和中国的知识工作者在此都一样。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、当人们依赖你时会发生什么？
   </strong>
  </h3>
  <p class="graf graf--p">
   知识工作在很大程度上取决于组织中各个成员所拥有的有形和无形知识（例如，数据和专有技术），这会在员工之间造成相互依赖性。
  </p>
  <p class="graf graf--p">
   因此，一个人可能需要来自各方的信息才能有效完成其工作。
  </p>
  <p class="graf graf--p">
   如果您依靠同事的知识来完成工作，那么可能会通过与他人共享知识来使自己更有可能做出回报。这是一种凝聚力。
  </p>
  <p class="graf graf--p">
   除了询问受访者有关他们如何与同事分享和隐藏知识外，研究还询问他们是否依靠同事来完成工作。研究人员希望看到如果受访者认为他们的同事依赖于他们，他们将更愿意分享知识，而不太可能隐藏它。
  </p>
  <p class="graf graf--p">
   但令人惊讶的是，
   <strong class="markup--strong markup--p-strong">
    结果完全相反
   </strong>
   。
  </p>
  <p class="graf graf--p">
   👉当人们意识到他人依赖自己时，他们会感到压力（动机的受控类型），从而更加倾向于隐藏知识。
  </p>
  <p class="graf graf--p">
   这也许是队友的依赖性占据了他们太多时间，时间是一直稀缺的资源。这种情况下人们甚至会撒谎，假装自己没什么可以提供的。
  </p>
  <p class="graf graf--p">
   当然，与所有研究一样，这项研究也有局限性。
  </p>
  <p class="graf graf--p">
   首先，没有考虑参与者共享知识的性质。有人可能会说，如果队友喜欢分享自己的知识（自主动机），这可能会导致他们分享更多的隐性知识（更难以记录和转让的内在专有技术），而不是显性知识（清晰的教科书信息） 。
  </p>
  <p class="graf graf--p">
   其次，研究数据仅限于员工的自我报告。如果将来的研究可以收集其他数据，例如队友之间的共享数据、有用性、拒绝次数以及原因，结果会更好。
  </p>
  <p class="graf graf--p">
   👉有效的知识共享对于所有组织来说都是必不可少的，但是许多组织都在努力劝导成员这样去做，却往往事与愿违。
  </p>
  <p class="graf graf--p">
   这项的结果表明，作为组织者，如果您想要鼓励队友之间更多的共享，那就
   <strong class="markup--strong markup--p-strong">
    首先需要设计您的工作本身
   </strong>
   ，以便人们想要讨论他们所知道的东西。
  </p>
  <p class="graf graf--p">
   不论是社区、团队、组织或者公司，管理都是一项复杂的工作；您可以在实践中获得经验，虽然这会导致太多的试错，当然也可以通过科学研究的结论找到灵感。希望这篇文章对你有用。
  </p>
