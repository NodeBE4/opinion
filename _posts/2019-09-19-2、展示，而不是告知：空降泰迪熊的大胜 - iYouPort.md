  <p class="graf graf--p">
   这是异议和反对派活动家根本性的区别。如果活动家陷入了后者，不仅会失败，而且会失去这一身份的意义。

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    任何时候不要高估人们对你的观点和理论的兴趣
   </strong>
   。
  </p>
  <p class="graf graf--p">
   曾经与几位中国南方农村地区的小企业主交流。他们给我们分析：“乱了怎么办，没乱怎么办”。
  </p>
  <p class="graf graf--p">
   喜忧参半。一方面看起来人们知道有可能“乱”- 出现社会反抗或动荡，另一方面，也没有准备加入抗议，而是在努力想着如何保住自己的小日子。
  </p>
  <p class="graf graf--p">
   如果你是一位致力于劝导人们加入你的活动家，在这种情况下没准已经火冒三丈了。“什么叫乱？乱是变革的契机！难道你不想改变糟糕的现状吗？”……一肚子话对不对？
  </p>
  <p class="graf graf--p">
   我们什么也没说。相反，我们拿出了一些北欧社会有趣的照片，关于抗议行动策划者主持的别具一格的街头派对。我们讲了一些好玩的故事。
  </p>
  <p class="graf graf--p">
   没人会把一场欢快的派对称之为“乱”。
  </p>
  <p class="graf graf--p">
   一张图片可能胜过千言万语。
  </p>
  <p class="graf graf--p">
   在今天的形象驱动的新闻周期和大众媒体文化中，这句话比以往任何时候都更加真实。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    有效的创作活动也必须以形象为导向。换句话说，展示，而不是告知
   </strong>
   。并且有很多方法可以做到这一点。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    以故事为主导，而不是事实
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这句话我们在本系列中有多次提到。因为它在策略方面很重要。
  </p>
  <p class="graf graf--p">
   事实很少能说明一切。⚠️虽然你的信息的事实准确性是绝对绝对必不可少的，但是，事实应该只是故事的支持细节，而不是使故事引人注目的钩子。
  </p>
  <p class="graf graf--p">
   如果你想传达失业的破坏性，不要带着一大堆统计数据。相反，讲述一个关于一个人的引人入胜的故事。然后指出：那里有成千上万个像她一样的人。
  </p>
  <p class="graf graf--p">
   让它视觉化。
  </p>
  <p class="graf graf--p">
   很多重要的东西很难用文字说清楚 —— 它们太宏大了、太遥远了、太抽象、或者太复杂。如果你的听众祖上三代都没有经历过民主自由的失去，你大声讲述民主自由的利好，在听者看来就像是在讲述遥远的天际有一只三条腿的公鸡。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    很多问题都是如此。作为理论的卓越者，你很可能一眼就能看到深刻的根源，但是，你肯定很难让你的听众与你感同身受。
   </strong>
  </p>
  <p class="graf graf--p">
   💡要想做到后者，就需要首先用你的听众的理解方式去理解你自己的样子。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    道具、视觉效果、和具体与针对性的语言策略，都可以帮助将宏大的事物降低到*人的尺度*
   </strong>
   。
  </p>
  <p class="graf graf--p">
   举一个美国的例子，以最经典的经济不平等为例。
  </p>
  <p class="graf graf--p">
   你可以很容易就迷失在税法的细节里，但是，当亿万富翁沃伦巴菲特说他的秘书支付的税金超过他时，这明显是错误的，无可辩驳的。
  </p>
  <p class="graf graf--p">
   为了引起人们对首席执行官和工人薪酬之间日益扩大的差异的关注，一个活动家小组展示了华盛顿纪念碑的一个小型复制品，缩小了419倍 —— 他们此前在真实的纪念碑前举行的新闻发布会。
  </p>
  <p class="graf graf--p">
   每个人都知道这是什么意思。你不需要多说任何一句话。
  </p>
  <p class="graf graf--p">
   使用强大的隐喻。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    通过隐喻，你可以充分展示它的内容，而不必解释它。
   </strong>
  </p>
  <p class="graf graf--p">
   要找到令人信服的比喻，请寻找体现您想要传达的内容的东西。
  </p>
  <p class="graf graf--p">
   比如，美国的关于移民的辩论就采用了候鸟的比喻 —— “迁徙的鸟类是否也需要护照？” —— 效果非常好，充分地指出了情况的荒谬性，请注意，活动家们甚至没有关注任何具体的政策或立法。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    用行动来说话，而不是告知。
   </strong>
  </p>
  <p class="graf graf--p">
   做出你想要说的东西。
  </p>
  <p class="graf graf--p">
   在抗议活动中，每当有警察保护银行时，就会自然形成一个反映现实情况的比喻：国家为我们其他人保护富人。
  </p>
  <p class="graf graf--p">
   有时景观和画面可以自然地呈现这一点 —— 或者，有时候，💡你可以把它人为地策划出来。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    精心设计的动作自己就可以可以解释自己
   </strong>
   。
  </p>
  <p class="graf graf--p">
   理想的情况下可以提供多种方式来解决问题。⚠️请注意一个关键：是你希望你的观众得出自己的结论，而不是你觉得他们有必要被告知思考什么。
  </p>
  <p class="graf graf--p">
   说教很难产生说服力。
  </p>
  <p class="graf graf--p">
   无论我们是在讲故事、想象一个场景、提供一个比喻、以身作则、还是让我们的行动自己说话，有数以百万计的方式可以传达我们的信息和价值观，而不会引发政治诽谤。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4457 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、故事-空降泰迪熊.png?resize=1100%2C725&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、故事-空降泰迪熊.png?w=1216&amp;ssl=1 1216w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、故事-空降泰迪熊.png?resize=300%2C198&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、故事-空降泰迪熊.png?resize=768%2C507&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、故事-空降泰迪熊.png?resize=1024%2C675&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、故事-空降泰迪熊.png?resize=1100%2C725&amp;ssl=1 1100w" data-recalc-dims="1" height="725" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、故事-空降泰迪熊.png?resize=1100%2C725&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4457" data-recalc-dims="1" height="725" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、故事-空降泰迪熊.png?resize=1100%2C725&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、故事-空降泰迪熊.png?w=1216&amp;ssl=1 1216w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、故事-空降泰迪熊.png?resize=300%2C198&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、故事-空降泰迪熊.png?resize=768%2C507&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、故事-空降泰迪熊.png?resize=1024%2C675&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、故事-空降泰迪熊.png?resize=1100%2C725&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   下面讲一个精彩的故事。
  </p>
  <p class="graf graf--p">
   这是反全球化运动高峰期的典型峰会抗议活动。
  </p>
  <p class="graf graf--p">
   来自美洲各地的34位国家元首正聚集在魁北克市，对美洲自由贸易区（FTAA）进行谈判，
   <strong class="markup--strong markup--p-strong">
    这是一项涉及深刻反民主条款的全面贸易协议。
   </strong>
  </p>
  <p class="graf graf--p">
   抗议活动已被召集，预计将有数万人涌入街道 —— 于是，⚠️在旧城周围建立了由数千名防暴警察捍卫的巨大围栏，以迫使抗议者远离会议中心。
  </p>
  <p class="graf graf--p">
   随着峰会越来越紧密，如何最有效地反对 FTAA 的辩论进入激烈状态，一个自称为“解构主义超现实拓扑研究所（DIST）”的活动家团体散发了一本
   <strong class="markup--strong markup--p-strong">
    宣传更具创造性的抗议策略
   </strong>
   的小册子：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “对于那些每次看到另一个 Black Bloc 时就开始打哈欠的人，解构主义超现实拓扑研究所提出了这个简要的备选方案清单，给他们注入一点创造力和嘲弄的笑声。”
   </em>
  </p>
  <p class="graf graf--p">
   黑群（Black Bloc）是一种抗议游行活动中常见的战术，其特征是每个人都穿着黑色服装，使用围巾、墨镜、滑雪面罩、带衬料的摩托车头盔或其他隐藏和保护面部的物品。简单说就是，一眼看上去就是来抗议的。
  </p>
  <p class="graf graf--p">
   就是下面这样：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4456 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图：BB.png?resize=1100%2C628&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图：BB.png?w=1258&amp;ssl=1 1258w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图：BB.png?resize=300%2C171&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图：BB.png?resize=768%2C438&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图：BB.png?resize=1024%2C584&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图：BB.png?resize=1100%2C628&amp;ssl=1 1100w" data-recalc-dims="1" height="628" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图：BB.png?resize=1100%2C628&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4456" data-recalc-dims="1" height="628" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图：BB.png?resize=1100%2C628&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图：BB.png?w=1258&amp;ssl=1 1258w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图：BB.png?resize=300%2C171&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图：BB.png?resize=768%2C438&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图：BB.png?resize=1024%2C584&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图：BB.png?resize=1100%2C628&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   DIST 的想法是打破这种沉闷的标签。
  </p>
  <p class="graf graf--p">
   他们提出了几个建议：1、Gary Coleman Bloc（Gary 是著名谐星）：就是让人们不断地朝着警察走过去，一边喊：“威尔斯你在说什么？”也就是模仿 Gary；2、Mascot Bloc：让抗议者穿着紧身衣和粉色的芭蕾舞裙，跟随那些 Black Bloc 并无情地取笑他们。
  </p>
  <p class="graf graf--p">
   DIST 还开玩笑地提议用 Monty-Pythonesque（英国著名的戏剧团体）风格的 Mascot Bloc 挑战会议区周围的围栏：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <em class="markup--em markup--p-em">
    如果这个家伙要把山顶变成一个堡垒，那么 Mascot Bloc 很有兴趣地围攻它。”
   </em>
  </p>
  <p class="graf graf--p">
   一位支持他们的公众人物联系到了该组织，对他们说，“如果你能找到一个人来制造弹射器，我来付钱。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    消息一推出，渥太华的一群弹射爱好者马上就活跃起来了。这种装置平时是被禁止的，因为它可以用作武器，而 DIST 将其走私进入该城市
   </strong>
   。
  </p>
  <p class="graf graf--p">
   游行当天，活动家们把全尺寸的弹射器拉到了围栏旁，开始朝着防爆警察的队伍弹射 —— 泰迪熊布偶。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    你没看错，并不是任何武器，而是玩具熊
   </strong>
   。他们收集到大批的玩具熊。
  </p>
  <p class="graf graf--p">
   与此同时，其他活动人士用螺栓切割机拆除了有问题的围栏。
  </p>
  <p class="graf graf--p">
   特技表演完成后，活动人士禁用了道具，并将其丢给了挥舞着催泪瓦斯前进的警察部队。
  </p>
  <p class="graf graf--p">
   每个人都认为这就结束了，但是，警察们可不甘心自己在这种荒谬的前线上被包围：他们通过派遣便衣警察
   <strong class="markup--strong markup--p-strong">
    抓捕了一名与弹射器毫无关系的杰出活动家
   </strong>
   Jaggi Singh 以进行报复，并指控他拥有一种“危险武器”：Jaggi Singh 被关押了17天。
  </p>
  <p class="graf graf--p">
   真正策划和做这件事的活动家们一个都没被抓到。
  </p>
  <p class="graf graf--p">
   💡很显然，警方输了。虚假的所谓武器指控只会加剧 DIST 的火力，他们趁机掀起了新一轮的新闻发布会和媒体特技，以嘲弄这些维稳机构。
  </p>
  <p class="graf graf--p">
   活动人士将他们的“填充物同志” ——即 泰迪熊 ——交给了全国各地的当地警察局，还将熊送到了加拿大总理办公室，以抗议荒谬的指控。
  </p>
  <p class="graf graf--p">
   非常精彩。
  </p>
  <p class="graf graf--p">
   💡弹射器的道具不仅仅是一场好的戏剧，而且，它是切实有效的行动主义。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    它不仅在物理层面上、而且在象征意义上攻击了使民间社会远离贸易协定谈判的那个围栏，这种谈判会影响到每个人
   </strong>
   。
  </p>
  <p class="graf graf--p">
   最后，抗议活动取得了最终成功：那场会议是加拿大政府的公关噩梦，公众对抗议者表示同情，半球范围内的贸易协议从未签署过。
  </p>
  <p class="graf graf--p">
   💡空降泰迪熊表面上的目标是防暴警察及其背后的政客，但是其真正的目标不在于围栏。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    首先
   </strong>
   ，这一行动以一种媒体奇观的方式准确吸引了公众的想象，暴露了政客的荒谬，这些政客被公民提出的合理问题“围困”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    其次
   </strong>
   ，这一行动让其他积极人士得到了两个重要的信息：第一，不要害怕面对国家权力；第二，当你这样做时，不要失去幽默感、不要忽视你行动的更广泛的道具。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    关键原则 ——
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、用道具说话
   </strong>
   ：不论是超大的股票碎纸机还是泰迪熊弹射器，精心挑选的瞩目级道具都可以帮助你创造一个媒体奇观、并讲述一个深入人心的故事。
  </p>
  <p class="graf graf--p">
   通过选择一个荒诞的攻城引擎，DIST 巧妙地暴露了更大局势的荒谬性：政客在做出非常不受欢迎的决定时被迫被选民“围攻”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、使用荒谬来破坏权威的光环
   </strong>
   ：要运作，权力就需要权威的光环。
  </p>
  <p class="graf graf--p">
   穿着制服或西装的男人掌控着一切。他是清醒的、认真的、也许最重要的是，被需要的。
  </p>
  <p class="graf graf--p">
   💡没有什么能比笑声这样的东西更容易破坏这种光环的了（以及随之而来的国家暴力的理由），特别是在权势不知道该如何处理的荒谬情境中。
  </p>
  <p class="graf graf--p">
   如果权势按照他们的正常逻辑做出反应，他们就会看起来非常荒谬和/或偏执 —— 就如加拿大警方没收泰迪熊作为“危险武器”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、使用手头的材料
   </strong>
   ：泰迪熊行动取得了成功，正因为所涉及的人对所出现的意外机会作出了明智的和创造性的回应：第一，为回应荒谬的建议提供了严肃的资金，第二，警察的过度反应进一步强调了这种情况的荒谬性。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、展示，而不是告知
   </strong>
   ：加拿大维稳部门希望通过激起对暴力抗议的恐惧以证明其前所未有的动员。但是，还有什么东西能比玩具熊更加非暴力？
  </p>
  <p class="graf graf--p">
   通过建立一个真正的战争引擎、并选择轻轻地弹射泰迪熊，DIST 找到了一种有趣的和意想不到的方式来展示他们对非暴力的承诺，并将政府捏造的恐惧视为毫无根据。
  </p>
  <p class="graf graf--p">
   还是那句话，反抗可以很好玩，可以充满乐趣；它既不是“乱”而不是危机，正相反，它能带来希望。🐻
  </p>
