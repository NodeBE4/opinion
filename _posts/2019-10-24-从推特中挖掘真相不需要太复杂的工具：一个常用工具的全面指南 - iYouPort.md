  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   它的能力其实可以很强大。不论您是希望追踪热点话题、查找特定人物、还是挖掘目标人和组织的背景，它都能做到。
  </p>
  <p class="graf graf--p">
   💡本指南尝试总结一下 Tweetdeck 的能力，也许其中很多是您没想到的。（Tweetdeck 的用户量很大，想必推特不至于关闭它。f**k
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/facebook-%E6%AD%A3%E5%9C%A8%E4%BB%A4%E7%9C%9F%E7%9B%B8%E6%8C%96%E6%8E%98%E4%BA%8B%E4%B8%9A%E9%99%B7%E5%85%A5%E5%9B%B0%E5%A2%83/" href="https://www.iyouport.org/facebook-%E6%AD%A3%E5%9C%A8%E4%BB%A4%E7%9C%9F%E7%9B%B8%E6%8C%96%E6%8E%98%E4%BA%8B%E4%B8%9A%E9%99%B7%E5%85%A5%E5%9B%B0%E5%A2%83/" rel="noopener noreferrer" target="_blank">

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    对于深度调查来说非常有用。TweetDeck 可以极大地简化和组织您的研究，同时允许您以更少的时间和精力收集更多的信息
   </strong>
   。
  </p>
  <p class="graf graf--p">
   调查人员经常处于一次打开五十多个选项卡的疯狂切换状态。而且，每当您想要更改搜索细节时，都必须重新开始。
  </p>
  <p class="graf graf--p">
   如果存在一个平台，可以将所有这些搜索保留在一个地方，随意操作调配并保存，以便您可以在一天甚至一个月内检查相同的搜索，以跟进事件查看新出现的信息流，这该多方便呢。那么您等的就是 TweetDeck。
  </p>
  <p class="graf graf--p">
   尤其是大型事件，比如抗议活动、选举、灾难、战争等等，调查人员和记者总是希望有一个“上帝视角”，能从最外层统揽全局；TweetDeck 就能做到。
  </p>
  <p class="graf graf--p">
   在这里调查人员可以更轻松地实时收集重要信息、并对搜索进行排序，对其进行扩充、以查找更相关的内容并消除不可避免的噪音。
  </p>
  <p class="graf graf--p">
   对于实时检测的预期来说，基本没有比 TweetDeck 更好的免费工具了。
  </p>
  <p class="graf graf--p">
   ⚠️作为免责声明，请记住：在进行在线调查时要始终注意保护自己。
  </p>
  <p class="graf graf--p">
   ⚠️虽然 Twitter 是查看信息的一种相当温和的方式，但是要知道，
   <strong class="markup--strong markup--p-strong">
    如果你不小心，一次错误的点击就可以揭示你的真实身份。
   </strong>
  </p>

  <h3 class="graf graf--p">
   <span style="font-family: impact, sans-serif;">
    <strong class="markup--strong markup--p-strong">
     基础知识：搜索列和过滤器
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   下载 TweetDeck 访问
   <a class="markup--anchor markup--p-anchor" data-href="http://www.tweetdeck.com" href="https://www.tweetdeck.com" rel="nofollow noopener noreferrer" target="_blank">
    www.tweetdeck.com
   </a>
   并使用您的 Twitter 账户登录即可开始 —— 调查人员建议使用单独的角色扮演账户，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%88%86%e8%a3%82%e4%ba%ba%e6%a0%bc%e7%9a%84%e9%ad%85%e5%8a%9b/" href="https://www.iyouport.org/%e5%88%86%e8%a3%82%e4%ba%ba%e6%a0%bc%e7%9a%84%e9%ad%85%e5%8a%9b/" rel="noopener noreferrer" target="_blank">
    具体做法详见这里
   </a>
   。
  </p>
  <p class="graf graf--p">
   您可以创建的最基本的列是搜索列。
  </p>
  <p class="graf graf--p">
   这本质上是一个关键字搜索 —— 过滤每秒种输入互联网的6,000条推文，包含您感兴趣的单词（或短语）的内容。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    单独使用关键字通常不足以对存在的大量无关内容进行排序。
   </strong>
  </p>
  <p class="graf graf--p">
   通过过滤搜索，您可以定位搜索工作，并收集更多的相关信息。
  </p>
  <p class="graf graf--p">
   搜索栏上的过滤器包括（并不限于）：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    按推文类型过滤：包含图片、视频、GIF等的推文；
   </li>
   <li class="graf graf--li">
    按日期和时间过滤：您可以创建自己的日期范围，可以追溯到2010年！如果您想要寻找在一天中非常特定的小时范围内发送的推文，也可以这样做；
   </li>
   <li class="graf graf--li">
    按照语种过滤推文；
   </li>
   <li class="graf graf--li">
    包括或排除结果中的转推；
   </li>
   <li class="graf graf--li">
    通过推文发布者的过滤：您可以筛选仅仅包含推荐的 Twitter 用户，列表成员、个人；
   </li>
   <li class="graf graf--li">
    以及更多可能性！
   </li>
  </ul>

   这一功能是 TweetDeck 的支柱。

  <p class="graf graf--p">
   💡强烈建议您使用过滤器亲测它的潜力，通过实践掌握如何最好地操纵搜索。
  </p>
  <h3 class="graf graf--p">
   <span style="font-family: impact, sans-serif;">
    <strong class="markup--strong markup--p-strong">
     Twitter Lists
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   什么是 Twitter Lists？
  </p>
  <p class="graf graf--p">
   Twitter 列表是 Twitter 用户的汇编，将其视为 Twitter 上存在的源列表。
  </p>
  <p class="graf graf--p">
   例如，如果您对华盛顿特区的新闻感兴趣，您可能会列出所有D.C.通讯员的名单。
  </p>
  <p class="graf graf--p">
   然后，当华盛顿特区发生突发新闻事件时，您可以转到该列表内，这里只能看到来自其成员的推文。
  </p>
  <p class="graf graf--p">
   这可以将注意力转入感兴趣的特定主题上，消除时间线上的任何杂音。
  </p>
  <p class="graf graf--p">
   最重要的是，
   <strong class="markup--strong markup--p-strong">
    您无需关注任何这些 Twitter 用户，即可将其添加到列表中
   </strong>
   。（斯诺登最初注册推特时中文新闻使用的标题是“他只关注了一个账户NSA”。其实他不需要关注任何账户也能及时查看重要消息。关于NSA是鄙视的姿态）
  </p>
  <p class="graf graf--p">
   使用 Twitter Lists。您可以很容易将它们与 Twitter Feed 完全分开，并将内容分类。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    请注意这绝不是鼓励切断连接关系
   </strong>
   ，而是说，作为调查者，您肯定会使用与身份完全无关的独立账户，而且您不希望任何人看一眼您的账户的关注者就能了解您的兴趣和意向。
  </p>
  <p class="graf graf--p">
   这种时候 Twitter Lists 就很重要了。
  </p>
  <h3 class="graf graf--p">
   <span style="font-family: impact, sans-serif;">
    <strong class="markup--strong markup--p-strong">
     创建自己的列表
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   列表会显示在用户的个人资料中。创建自己的列表，您创建的列表可以是公共列表或私有列表。如果将其公开，则添加到其中的某人将会收到通知。
  </p>
  <p class="graf graf--p">
   如果设置为私人，除了你和所有推特工程师之外，没有人会知道它们存在。
  </p>
  <p class="graf graf--p">
   虽然从头开始创建自己的列表是一项值得尊敬的任务，但是，有可能已经存在您想要寻找的列表（比如您可能不是唯一对华盛顿特区新闻感兴趣的人；或者对OSINT感兴趣，或者对猫咪……）。
  </p>
  <p class="graf graf--p">
   💡
   <strong class="markup--strong markup--p-strong">
    要查找这些列表，您有几个选择。
   </strong>
  </p>
  <p class="graf graf--p">
   首先，你可以找到一个有信誉的新闻机构、记者或调查员。这些用户中的大多数人都会有大量的列表可供您查看；
  </p>
  <p class="graf graf--p">
   或者你可以通过谷歌获得。在您的 Google 搜索栏中，输入 site：twitter.com / * / lists“LISTNAME”。
  </p>
  <p class="graf graf--p">
   这会能显示各种公共 Twitter 列表，与您想要的主题相匹配。
  </p>
  <p class="graf graf--p">
   然后，您可以单击这些结果中的任何一个，它将带您进入 Twitter 上的列表。
  </p>

   <strong class="markup--strong markup--p-strong">
    列表创建者将收到您已订阅的通知，并且您将无法操纵列表
   </strong>

  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   但是，您需要登录 Twitter 才能使用它，这样一来
   <strong class="markup--strong markup--p-strong">
    工具的创建者就可以访问您的帐户。
   </strong>
  </p>
  <p class="graf graf--p">
   我认识的大多数研究人员都使用这个工具（
   <strong class="markup--strong markup--p-strong">
    除非

  </p>
  <p class="graf graf--p">
   该工具对于我们许多人来说都是不可或缺的，因为它允许您将所有账户从某人的 Twitter 列表复制到您自己的列表中。

   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️这里要小心，如果您正在试图隐身状态下进行研究，请确保在将成员复制到其中之前，您自己的列表是私密的。
  </p>
  <p class="graf graf--p">
   然后，因为结果列表是属于您的（从其他人那里借来的成员），您
   <strong class="markup--strong markup--p-strong">
    可以
   </strong>
   通过添加或删除其中的成员来操作和更改该列表。
  </p>
  <h3 class="graf graf--p">
   <span style="font-family: impact, sans-serif;">
    <strong class="markup--strong markup--p-strong">
     将列表添加到 TweetDeck
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   好的，现在你有名单了！这对研究来说至关重要，因为这些人的推文很有可能包含您感兴趣并且可以信任的信息。
  </p>
  <p class="graf graf--p">
   困难的部分结束了，现在您只需要在 TweetDeck 中添加一个列，它可以让您实时查看来自列表成员的推文。
  </p>
  <p class="graf graf--p">
   转到 TweetDeck 的左侧，点击加号，然后从选项中选择“lists”。然后，选择您感兴趣的列表，就完成了！
  </p>

  <p class="graf graf--p">
   如果你想真正过滤来自列表的推文，我会推荐一个看似有点违反直觉的方法。
  </p>
  <h3 class="graf graf--p">
   <span style="font-family: impact, sans-serif;">
    <strong class="markup--strong markup--p-strong">
     在 TweetDeck 上搜索列表
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   首先，要搜索列表并过滤 TweetDeck 上列表的结果，请不要浏览 list column。
  </p>
  <p class="graf graf--p">
   相反，请浏览基本搜索列。
  </p>
  <p class="graf graf--p">
   在基本搜索列中的过滤器选项 “Tweet Authors” 下，选择 “List of members …”，然后键入您感兴趣的列表，使用具体的格式，“@username / list-name”。
  </p>
  <p class="graf graf--p">
   用户名是指拥有该列表的人，如果您正在尝试搜索自己的列表，请将您的ID放在那里。
  </p>
  <p class="graf graf--p">
   不是只能搜自己哦。
   <strong class="markup--strong markup--p-strong">
    您可以在此列中搜索任何人的列表。
   </strong>
  </p>
  <p class="graf graf--p">
   您可以通过转到 Twitter 上的列表页面并复制网址末尾的内容，以您想要的确切格式找到列表名称。

  <h3 class="graf graf--p">
   <span style="font-family: impact, sans-serif;">
    <strong class="markup--strong markup--p-strong">
     按位置搜索
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   重要提示：有些人可能已经阅读过 Twitter 位置标记的最新更改。截至目前，以下详述的执行过程结果没有出现重大差异。
  </p>
  <p class="graf graf--p">
   目前仍然可以进行这样的搜索。如果将来发生变化，我们会及时通知您。
  </p>
  <h3 class="graf graf--p">
   <span style="font-family: impact, sans-serif;">
    <strong class="markup--strong markup--p-strong">
     位置搜索的基础知识
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   对于许多研究人员来说，使用 TweetDeck 最有用的方法之一就是单独按地理位置搜索推文。
  </p>
  <p class="graf graf--p">
   ⚠️即使没有任何关键字，您也可以按照发送推文的地理位置过滤 Twitter，并及时收集有关某个区域的信息。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    尤其是在大型群体事件、灾难和暴力的调查中，从一开始确认推文来自*真正的目击者*，可以帮助您省下很多时间。
   </strong>
  </p>
  <p class="graf graf--p">
   要按位置搜索，请在 TweetDeck 上打开常规搜索。输入 “geocode:coordinate,coordinate,Xkm” ，其中“coordinate,coordinate”是您要搜索的位置，Xkm 是您要搜索的特定位置的半径范围。
  </p>
  <p class="graf graf--p">
   ⚠️请注意，搜索项中的任何内容之间都没有空格，包括逗号和第一个经度坐标之间（当复制和粘贴坐标时，会自动存在一个需要手动删除的空格）。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6130 jetpack-lazy-image" data-lazy-sizes="(max-width: 529px) 100vw, 529px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-5.png?resize=529%2C575&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-5.png?w=529&amp;ssl=1 529w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-5.png?resize=276%2C300&amp;ssl=1 276w" data-recalc-dims="1" height="575" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-5.png?resize=529%2C575&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="529"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6130" data-recalc-dims="1" height="575" sizes="(max-width: 529px) 100vw, 529px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-5.png?resize=529%2C575&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-5.png?w=529&amp;ssl=1 529w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-5.png?resize=276%2C300&amp;ssl=1 276w" width="529"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   查找要搜索的任何特定位置的坐标的最简单方法之一是访问 Google 地图，搜索您想要的位置，右键单击地图上的位置，然后点击“What’s here？”
  </p>
  <p class="graf graf--p">
   坐标将显示在地图的底部，复制它们就可以了。
  </p>
  <p class="graf graf--p">
   此方法适用于几乎所有位置，因在发送推文时会选择用户手机的位置设置。
  </p>
  <p class="graf graf--p">
   如果用户启用了位置服务，就像大多数人做的那样，他们的推文将出现在此搜索中。
  </p>
  <p class="graf graf--p">
   如果它不起作用并且 TweetDeck 告诉你“没有最近的推文”，⚠️请仔细检查你是否输入了完全正确的格式，然后尝试扩大搜索范围。
  </p>
  <p class="graf graf--p">
   您可能已经注意到搜索栏的过滤器选项中存在“位置”选项。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    为了使用它，您需要考虑关键字，并且不能被动地监视特定位置的所有推文。
   </strong>
  </p>
  <p class="graf graf--p">
   使用此“位置”过滤器选项，您还需要使用 Twitter 可识别的位置名称，例如大城市的名字。
  </p>
  <p class="graf graf--p">
   它不适用于坐标，这对于对世界偏远地区感兴趣的研究人员更有用。
  </p>
  <p class="graf graf--p">
   一般来说使用地理编码搜索技术就已经足够了，可以忽略此过滤器选项。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    将位置搜索与更多功能结合起来
   </strong>
  </p>
  <p class="graf graf--p">
   现在您已经拥有了地理编码搜索列，您的 TweetDeck 应该可以实时地从该区域获取所有推文。
  </p>
  <p class="graf graf--p">
   这可能是压倒性的信息流，每隔几秒就有一条推文出现。
  </p>
  <p class="graf graf--p">
   这里面也许有很多杂音，对您的研究并没有太大作用（即便是剧烈的突发事件现场也是如此）。
  </p>
  <p class="graf graf--p">
   因此，除了位置搜索之外，您还需要
   <strong class="markup--strong markup--p-strong">
    缩小搜索范围
   </strong>
   ，以便仅查找与调查相关的结果。
  </p>
  <p class="graf graf--p">
   由于地理编码搜索位于普通搜索列中，因此您可以使用所有常用搜索过滤器对其进行过滤，包括日期、列表成员、语言等等。
  </p>
  <p class="graf graf--p">
   💡如果您没有找到能够连接所有推文的搜索关键词，这种方法会非常有用。
  </p>
  <p class="graf graf--p">
   例如，如果您只是在一个区域中实时监控推文，过滤它们可能比使用必须存在于所有推文中的关键词搜索更有用。
  </p>

  <p class="graf graf--p">
   除了 TweetDeck 之外，还有其他有趣的 Twitter 定位工具，它们使用上面相同的搜索功能，但是，可以为您
   <strong class="markup--strong markup--p-strong">

   </strong>

  </p>

   </a>
   。此站点允许您输入位置的坐标并指定半径范围。
  </p>
  <p class="graf graf--p">
   然后，它会在您选择的位置的地图上覆盖推文结果。
  </p>
  <p class="graf graf--p">
   位置不一定完全正确，因为无法确定 Twitter 如何准确提取和传输。但是，这可能是一种
   <strong class="markup--strong markup--p-strong">
    用可视化数据补充调查
   </strong>
   的好办法。
  </p>
  <p class="graf graf--p">
   可推荐的另一个工具是
   <a class="markup--anchor markup--p-anchor" data-href="https://onemilliontweetmap.com/" href="https://onemilliontweetmap.com/" rel="nofollow noopener noreferrer" target="_blank">
    https://onemilliontweetmap.com/
   </a>
   。
  </p>
  <p class="graf graf--p">
   这个工具类似的地方在于它也是覆盖地图，但是您
   <strong class="markup--strong markup--p-strong">
    可以按照关键字而不是坐标来搜索数据库
   </strong>
   ，然后可以放大地图上的某个位置以查看是否有任何带有该关键词的推文。
  </p>
  <p class="graf graf--p">
   此工具还允许您
   <strong class="markup--strong markup--p-strong">
    按时间范围进行过滤
   </strong>
   （仅限最近20小时），而 qtrtweets 仅按时间顺序显示结果。
  </p>
  <p class="graf graf--p">
   Onemilliontweetmap 还允许您在登录时保存搜索，如果您想要随时检查某些内容，这可能很有用。
   <strong class="markup--strong markup--p-strong">
    比如跟进持续多日发展的事件
   </strong>
   。
  </p>
  <p class="graf graf--p">
   💡这两个工具都给您显示推文链接，这样您就可以在 Twitter 上找到它们并存档或进一步调查。
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">

   <a class="markup--anchor markup--p-anchor" data-href="http://www.instagram.com" href="https://www.instagram.com" rel="nofollow noopener noreferrer" target="_blank">
    www.instagram.com
   </a>
   ”，然后输入您想要的任何搜索关键词。

  <p class="graf graf--p">
   这将能够生成带有搜索关键词的 Instagram 帖子。这也适用于个人在 Instagram 上为自己标记的位置。
