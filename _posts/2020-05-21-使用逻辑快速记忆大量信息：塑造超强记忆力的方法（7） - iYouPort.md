  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习（等级1）
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在网上照一张地图，从中找到你熟悉的城市。
  </p>
  <p class="graf graf--p">
   在地图上为自己选一条路线，努力记住这条路线，
   <strong class="markup--strong markup--p-strong">
    用脑而不是用笔
   </strong>
   ；然后关闭地图，努力想象自己正走在这条路线上。
  </p>
  <p class="graf graf--p">
   你在沿途看到了什么？路的两侧都有什么？尽可能在脑海中把它们画出来。越详细越好。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     记忆和识别
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   确保你已经记住某件事的唯一方法是：在没有任何帮助或提示的情况下完整地复述它。然后将它与原始材料进行对比。
  </p>
  <p class="graf graf--p">
   不要只在主观上 “感觉” 自己已经记住了，也不要在别人的故事或文本中记住某件事，而是要把它写下来，并与原始材料进行对比。
  </p>
  <p class="graf graf--p">
   👉这很重要。尤其是对于行动者来说，
   <strong class="markup--strong markup--p-strong">
    你需要掩饰身份或处理任务时，这一能力非常重要 —— 所有内容都要完整复述，并通过检查
   </strong>
   。
  </p>
  <p class="graf graf--p">
   其实生活中也是如此。比如准备考试，你觉得自己已经把材料背的滚瓜烂熟了，但是真的拿到考卷时就发现想不起来。
  </p>
  <p class="graf graf--p">
   这意味着其实你并没有真的 *记住*，你只是 *识别* 了这些信息，于是产生了一种 “已经记住” 的错觉。
  </p>
  <p class="graf graf--p">
   能 “认出”，并不代表能记住，也不会意味着在用的时候能想起来。
  </p>
  <p class="graf graf--p">
   ⚠️如果你是团队组织者，正在向你的队友发布指令，那么请队友重复这些指令。如果你告诉对方的是行动流程，那么必须让他以正序和倒序两种方式重复指令。
  </p>
  <p class="graf graf--p">
   如果你是学生，正在准备考试，你需要把要回忆的所有材料写在一张纸上。光有记住和识别的主观感觉是不够的。
  </p>
  <p class="graf graf--p">
   👉记住这条记忆规律：仅对内容进行部分复述只会让记忆变得更糟。最好是回想全部内容。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     报告和演讲
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   你一定见过那种拿着稿子的演讲方式？很令人沮丧吧。原本是言辞激昂的内容，演讲者的眼睛却不断看着手稿…… 就像断气一样。
  </p>
  <p class="graf graf--p">
   说实话脱稿可能并不容易。公共场合的演讲会面对大批听众的压力，在人类恐惧症榜单上始终排名前列。但您依旧有太多理由追求脱稿模式，因为
   <strong class="markup--strong markup--p-strong">
    只有这样才能调动听众的热情 —— 尤其是在动员中。
   </strong>
  </p>
  <p class="graf graf--p">
   演讲者是第一批使用助记法技巧的人之一。脱稿会让你更自信，也能更好地应对自身压力。
  </p>
  <p class="graf graf--p">
   你没有必要全文背诵演讲词。记住要点和要点之间的联系就足够了。这样你不会忘记内容，还有充分的临场发挥空间，可以与听众沟通的空间。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    步骤如下
   </strong>
   ：
  </p>
  <p class="graf graf--p">
   1、将你的演讲分解为不同的语义单元。没必要分太细，对于小型演讲来说，10个左右的单元就足够了。
  </p>
  <p class="graf graf--p">
   2、构想论点（每个篇章的主要思想）—— 用3～4个单词组成的句子表示即可。
  </p>
  <p class="graf graf--p">
   3、用一个能够明确阐述你的陈述的关键词，对每一个论点进行编码。
  </p>
  <p class="graf graf--p">
   4、使用故事法或地点法帮助记住所有关键词的列表。可以用自己的身体部位作为助记空间。
  </p>
  <p class="graf graf--p">
   5、排练。复述或写下整篇演讲的论点，通过记忆复述关键词。正序做一遍，倒序再做一遍。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   阅读一篇很短的文章。将它分解为不同的语义单位。用提纲的形式写下这些语义单位的主要思想。
  </p>
  <p class="graf graf--p">
   确认关键词并记住它们。
  </p>
  <p class="graf graf--p">
   然后试着复述文章，最后与原文进行对比，检查准确度如何。
  </p>
  <p class="graf graf--p">
   使用不同主题的文章反复进行这项练习，不仅可以锻炼你的记忆力，还可以锻炼留存、处理和压缩信息的能力。
  </p>
  <p class="graf graf--p">
   👉快速从大量信息中准确提取重要信息的能力对于互联网人来说尤其重要。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     巧妙使用结构
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   要想记住大量信息，最好能将它们组织起来。
  </p>
  <p class="graf graf--p">
   组织不同资料最有效的方法之一就是使用树形结构对其进行分类。
  </p>
  <p class="graf graf--p">
   比如你需要记住桌面上的所有物品，以便外出回来后能迅速知道是否有人动了你的东西。
  </p>
  <p class="graf graf--p">
   你的桌子上放着下面这些东西：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    一台笔记本电脑
   </li>
   <li class="graf graf--li">
    一支钢笔
   </li>
   <li class="graf graf--li">
    一个笔记本，第15页上有书签
   </li>
   <li class="graf graf--li">
    几张白纸
   </li>
   <li class="graf graf--li">
    一个卷笔刀
   </li>
   <li class="graf graf--li">
    一台无线信号探测器
   </li>
   <li class="graf graf--li">
    一本各国索引
   </li>
   <li class="graf graf--li">
    一块移动硬盘
   </li>
   <li class="graf graf--li">
    一个相框照片
   </li>
   <li class="graf graf--li">
    一张储存卡
   </li>
   <li class="graf graf--li">
    一支铅笔
   </li>
   <li class="graf graf--li">
    一本心理学书
   </li>
   <li class="graf graf--li">
    一个手机充电器
   </li>
  </ul>
  <p class="graf graf--p">
   以原始形式记住所有东西可能不太容易。但是你可以把物品分类，比如分为电子设备和其他物品两类。那么清单就变成这样：
  </p>
  <p class="graf graf--p">
   1、电子设备：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    笔记本电脑
   </li>
   <li class="graf graf--li">
    无线信号探测器
   </li>
   <li class="graf graf--li">
    移动硬盘
   </li>
   <li class="graf graf--li">
    储存卡
   </li>
   <li class="graf graf--li">
    手机充电器
   </li>
  </ul>
  <p class="graf graf--p">
   2、其他物品：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    钢笔
   </li>
   <li class="graf graf--li">
    笔记本，第15页上有书签
   </li>
   <li class="graf graf--li">
    白纸
   </li>
   <li class="graf graf--li">
    卷笔刀
   </li>
   <li class="graf graf--li">
    各国索引
   </li>
   <li class="graf graf--li">
    相框照片
   </li>
   <li class="graf graf--li">
    铅笔
   </li>
   <li class="graf graf--li">
    心理学书
   </li>
  </ul>
  <p class="graf graf--p">
   这样一来清单就小很多了。但还可以细分，比如：电子设备分为存储设备和其他设备；其他物品分为书写材料、书籍和装饰品。
  </p>
  <p class="graf graf--p">
   这就有了一个二级分类的树形结构列表。
  </p>
  <p class="graf graf--p">
   👉已经很多实验表明，记忆有序的信息比记忆杂乱的信息效果要高出数十倍。
  </p>
  <p class="graf graf--p">
   试着在脑子里画图表，将信息分区保存。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     思维导图
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   俗称脑图。技术人员都擅长这个。它是以视觉形式呈现资料的结构。
  </p>
  <p class="graf graf--p">
   思维导图是英国心理学家 Tony Buzan 开发的记忆技巧，Buzan 是世界记忆力锦标赛的创始人。这一赛事会挑选全球最优秀的记忆专家同台竞技，现场比拼记忆力准确度。
  </p>
  <p class="graf graf--p">
   对于任何人来说都是如此。当你需要快速掌握新领域的知识时，思维导图是必不可少的工具。你可以将分散的信息链接在一起，找出看似独立的事件之间的联系。
  </p>
  <p class="graf graf--p">
   制作思维导图其实很简单。有一大堆软件可以做到，比如 MarginNote、Mind Vector、SimpleMind …… 有些是要花钱的。其实不需要那么费劲（多下载一个APP就会多一分被攻击的风险）
   <strong class="markup--strong markup--p-strong">
    一张白纸就能解决问题
   </strong>
   。
  </p>
  <p class="graf graf--p">
   1、拿出一张16开白纸，把它横过来。
  </p>
  <p class="graf graf--p">
   2、在纸的中央标出思维导图的主题。可以是图片、文字或短语，任何你想要的形式都可以。最重要的是你应该理解并记住它。
  </p>
  <p class="graf graf--p">
   3、对主题进行延展和诠释。将主题作为起始点，开始绘制线条，并用解释性的关键词或图片进行标注。
  </p>
  <p class="graf graf--p">
   考虑因果关系 —— 哪些因素会影响你的导图主题？它们可能会导致什么样的后果？可能带来什么危险？导图包含哪些组成部分？
  </p>
  <p class="graf graf--p">
   问题取决于导图的主题是什么，而且问题可能不会少。不要因此被吓倒，坚持一边画一边思考，你会发现很快就能梳理清楚这些逻辑。
  </p>
  <p class="graf graf--p">
   在制图过程中一些线条会聚集在一起，另一些则会留下空白。这些情况都有可能发生。
  </p>
  <p class="graf graf--p">
   4、对每一行做同样的操作：将每一个关键词诠释为一个单独的主题。在每行的尾端绘制新的分支线条，并用关键词或图像对其进行标记。
  </p>
  <p class="graf graf--p">
   5、继续按照从中心到外围的方向画出新的分支，直到你所有的想法全部展示在导图上为止。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    为了让导图更有用，请注意以下几点
   </strong>
   ：
  </p>
  <p class="graf graf--p">
   1、即使你不擅长绘画，最好还是用图像对想法和文字进行编码。这样你会更好地记住它们。
  </p>
  <p class="graf graf--p">
   2、如果你使用词语进行描述，那么一个概念只写一个词即可。否则你的导图会变得混乱。
  </p>
  <p class="graf graf--p">
   3、离中心词越远，词语和图像的尺寸就越小，线条也越细，这样就能显示出知识的等级层次；并且要把最重要的信息和最不重要的信息区分开。
  </p>
  <p class="graf graf--p">
   4、使用不同颜色的笔、区分大小写字母，以便让图更容易一眼读懂。
  </p>
  <p class="graf graf--p">
   5、用线和箭头连接相关概念，或将它们组合成块。
  </p>
  <p class="graf graf--p">
   6、一个概念尽量不要创建7个以上的分支。如果需要展示更多信息，你可以创建另一个层次结构。
  </p>
  <p class="graf graf--p">
   思维导图的优势在于易于观察 —— 所有信息都展示在页面上，很容易看出信息结构和元素之间的联系。
  </p>
  <p class="graf graf--p">
   尤其是，导图很适合助记法的两个原则：关联性和情绪（图的颜色、底纹和书写符号都反映了作者的态度）。
  </p>
  <p class="graf graf--p">
   ⚠️切记要保密哦！别忘了在记住内容后销毁你的思维导图！这也是为什么不建议您使用软件，而是使用白纸的原因。（鬼知道那些公司会拿你的信息去做什么）
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   你昨天一整天都做了什么？明天准备干什么？用思维导图的形式把它们详细画出来。
  </p>
  <p class="graf graf--p">
   用导图中的元素大小来体现耗时的长短，用不同的颜色来凸显任务的重要性。试试看。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习 —— 地图（等级2）
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在另一张地图上创建一条较长的路线，并进行记忆。为了辅助记忆可以选择您比较熟悉的区域。
  </p>
  <p class="graf graf--p">
   给自己一分钟时间记下路线方向。沿着地图上标出的路线走，想象沿途左右两边的公园、池塘、河流、商店 …… 在十字路口停下来，进入小巷。记住你所见的一切。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     生活方式和记忆
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   这不是关于养生保健的话题。而是希望解释一些良好的生活方式可以帮助更好地行动和工作。
  </p>
  <p class="graf graf--p">
   👉不注重健康、特别是不注重安全的人，请勿与之合作。这点非常重要 —— 你愿意让一个连续熬夜2天的人开车带你长途旅行吗？哪怕此人熬夜完全是为了帮你做事，也绝对不行。他的精神状态会杀死你们所有人。
  </p>
  <p class="graf graf--p">
   我们在行动技巧中特别强调过：赴死不是英雄，正相反是一种法西斯主义倾向。
   <strong class="markup--strong markup--p-strong">
    请警惕那种讲求舍身的人，避免与其合作
   </strong>
   ，那种人不是骗子就是傻子。
  </p>
  <p class="graf graf--p">
   👉你需要的是周密的计划、丰富的经验和值得骄傲的技能，以最低耗损为目标完成任务。否则你不仅无法联盟他人，而且会折损你自己，并无法获得成功。
  </p>
  <p class="graf graf--p">
   与助记相关的生活方式包括：均衡的营养、体育运动、优化的时间表、以及自我减压的能力。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     时间表和记忆
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   什么早睡早起、8小时等都是养生讲求的；对于助记来说，您需要做的不是遵循这些，而是为自己计划一个
   <strong class="markup--strong markup--p-strong">
    最适合自己身体节奏
   </strong>
   的安排。
  </p>
  <p class="graf graf--p">
   如果你是早睡型人，那就不要计划熬夜工作，相反如果你是起型人，把重要工作安排在黄昏前后也许更好。
  </p>
  <p class="graf graf--p">
   身体的自然节奏不是很稳定的，它容易被外界环境的改变而影响，就如倒时差的人经常会发现的那样。所以，您需要做的是尽可能稳定身体的自然节奏，即便在周末和假日也保持这一作息习惯。
  </p>
  <p class="graf graf--p">
   睡眠会极大程度上影响个人表现和心理状态；审讯者经常利用剥夺睡眠的方法逼疯嫌疑人，并操纵他们说出重要信息。缺乏睡眠会导致烦躁，犯更多的错误。
  </p>
  <p class="graf graf--p">
   如果实在没有足够的睡眠时间，可以试试拆分睡眠的方法；也就是小睡20分钟，让自己舒舒服服地享受这20分钟 —— 注意不要超过这个时间。
  </p>
  <p class="graf graf--p">
   入睡后，大脑立即进入轻度睡眠阶段，在脑造影照片上可以清楚地看到高振幅的 θ波。
  </p>
  <p class="graf graf--p">
   入睡20分钟后缓慢的 δ波逐渐占据主导地位，深度睡眠阶段随之开始。在慢波睡眠阶段醒来的人会感到迟钝和沮丧。
  </p>
  <p class="graf graf--p">
   将闹钟设置为 20分钟，或者请其他人准时叫醒你。在大多数情况下，这种短暂的睡眠足以消除疲劳，并让你能够完成紧急的工作。醒来后可以喝上一杯浓茶。
  </p>
  <h4 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习 —— 火柴造型（等级4）
    </strong>
   </span>
  </h4>
  <p class="graf graf--p">
   加大难度啦：现在主要关注火柴摆放的形状、以及火柴头所指的方向，还要准确记住它们的位置。
  </p>
  <p class="graf graf--p">
   记忆的时间越短越好，接近过目不忘。
  </p>
  <p class="graf graf--p">
   预告：下一部分内容中我们将介绍一些现场实用的记忆方法，它可以辅助社交工程、反情报等方面的需求，也可以帮助您构建良好的组织环境。⚪️
