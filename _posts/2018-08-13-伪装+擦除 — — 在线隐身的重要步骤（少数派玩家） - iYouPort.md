  <p class="graf graf--p graf-after--figure" id="525d">
   美国有很多公司从公共记录和营销公司那里收集人们的个人信息，然后在方便的包装中转售。他们收集公共记录数据，如房地产交易、逮捕记录、法庭案件、婚姻、离婚等，并将其与从网页上收集的在线数据相结合。而且这些数据只需 3.95 美元即可购买！
  </p>
  <p class="graf graf--p graf-after--p" id="f8bc">
   要是想删除，你就需要付钱，很不公平是吧，他们收集的时候根本没经过你的允许。不过好在这些公司都比较好找，你可以提出申请然后交钱删除。而且这是个非常繁琐的工作，即便你全完成了也无法改变被政府控制的数据，你只能是让这些数据的获取变得麻烦一些。
  </p>
  <p class="graf graf--p graf-after--p" id="10b6">
   在中国就不同了，你很可能根本不知道自己的个人信息掌握在谁手里，更不用说找回来。
  </p>
  <p class="graf graf--p graf-after--p" id="4815">
   不过中国人还是有另外一些办法限制个人数据被滥用的。比如在线的。
  </p>
  <p class="graf graf--p graf-after--p" id="b4d0">
   如果搜索某人的姓名，几乎可以在 Google 或其他搜索引擎上找到有关他们的惊人数量的信息。要删除此类信息通常很困难。除非在非常特殊的情况下，否则大多数对搜索引擎结果的更改请求都将被忽略。一般来说，搜索公司会告诉您接收托管相关网页的网站 — 他们建议您向托管网页的网站提交电子邮件或信件。实际上，这通常有可能是最简单的路径。
  </p>
  <p class="graf graf--p graf-after--p" id="f001">
   如果您有足够的资金，一家名为 Reputation 的公司能尝试为您删除任何不需要的搜索引擎结果。他们通过创建更讨人喜欢的网页，并尝试使其成为最佳结果来实现这一目标。他们的服务可不便宜哦 —
   <strong class="markup--strong markup--p-strong">
    每年的价格在3000美元到5000美元左右！您现在知道数据有多值钱了？
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="0b8e">
   还是自己动手吧。下面说说如何从 Google 删除个人信息。
  </p>
  <p class="graf graf--p graf-after--p" id="b6dd">
   你在互联网上投放的任何内容最终都有可能落在 Google 手里，是任何。也就是说任何人都有机会获取它们，所以你应该谨慎在互联网放东西。
  </p>
  <p class="graf graf--p graf-after--p" id="641d">
   &gt; Google 电话簿：Google 电话簿使用第三方数据填充其列表，在网络搜索的结果旁边有一个电话图标。你可以
   <a class="markup--anchor markup--p-anchor" data-href="https://support.google.com/websearch/?visit_id=636698443596397190-2827289089&amp;hl=en&amp;rd=2#topic=3285072" href="https://support.google.com/websearch/?visit_id=636698443596397190-2827289089&amp;hl=en&amp;rd=2#topic=3285072" rel="noopener nofollow noreferrer" target="_blank">
    在这里
   </a>
   要求不把自己的号码列在里面。但即便从谷歌电话簿中删除了，你的电话号码也可能依旧会显示在 Google 搜索结果中，因为这些信息可能来自网页。
  </p>
  <p class="graf graf--p graf-after--p" id="566f">
   &gt; Google 地图： Google 地图也会使用第三方数据，有时会显示个人联系的详细信息以代替商家详细信息。如果您是经过验证的企业所有者，则可以更正此信息。如果其他企业显示了您的个人联系信息，您可以
   <a class="markup--anchor markup--p-anchor" data-href="http://maps.google.com/support/bin/answer.py?hl=en&amp;answer=171429" href="https://maps.google.com/support/bin/answer.py?hl=en&amp;answer=171429" rel="noopener nofollow noreferrer" target="_blank">
    直接编辑信息或使用报告问题链接
   </a>
   来请求更改。
  </p>
  <p class="graf graf--p graf-after--p" id="1389">
   &gt; Blogger： Blogger 是一个博客平台，因此 Google 通常不会从中移除内容。但是，您可以请求删除违反 Blogger 服务条款的内容。例如某人使用您的真实姓名冒充您，可以
   <a class="markup--anchor markup--p-anchor" data-href="http://www.google.com/support/blogger/bin/answer.py?answer=82112" href="https://www.google.com/support/blogger/bin/answer.py?answer=82112" rel="noopener nofollow noreferrer" target="_blank">
    要求进行调查
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="0d7f">
   &gt; Google 街景： Google 街景视图显示地图相关数据（例如街道和商家）的照片，通常包括住宅、车辆和人员。谷歌有时候会试图模糊识别细节，例如面孔和车牌号码。您可以
   <a class="markup--anchor markup--p-anchor" data-href="http://maps.google.com/help/maps/streetview/privacy.html" href="https://maps.google.com/help/maps/streetview/privacy.html" rel="noopener nofollow noreferrer" target="_blank">
    要求完全删除您的房屋和汽车的照片
   </a>
   。只需转到 Google 地图并放大包含个人照片的区域即可。点击“报告问题”链接并选择“隐私问题。
  </p>
  <p class="graf graf--p graf-after--p" id="9cb3">
   事实上最麻烦的是
   <strong class="markup--strong markup--p-strong">
    社交网络
   </strong>
   ，一切都是公开的，从注册到发帖都要非常小心。使用化名（回看本文第一部分）、避免在任何时候显示地理位置、避免谈论自己、发布任何照片抹去元数据，等等。使用多个电子邮件地址，把重要的和次要的分开，只用来注册帐户的邮件地址让它单独存在。
  </p>
  <p class="graf graf--p graf-after--p" id="0cb7">
   此外，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.safeshepherd.com/" href="https://www.safeshepherd.com/" rel="noopener nofollow noreferrer" target="_blank">
    Safe Shepard
   </a>
   是一项免费的服务，可以从各种“人物搜索”数据库（例如 PeekYou）中删除您的姓名；还有一个网站
   <a class="markup--anchor markup--p-anchor" data-href="http://www.accountkiller.com/en/" href="https://www.accountkiller.com/en/" rel="noopener nofollow noreferrer" target="_blank">
    Account Killer
   </a>
   ，提供了从绝大多数热门网站上删除个人信息的方法。
  </p>
  <p class="graf graf--p graf-after--p" id="3e6c">
   好啦，恭喜您完成我们的《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E9%9A%90%E8%BA%AB%E6%B8%B8%E6%88%8F-%E5%A6%82%E4%BD%95%E5%9C%A8%E7%9B%91%E8%A7%86%E9%81%8D%E5%9C%B0%E7%9A%84%E7%A4%BE%E4%BC%9A%E9%87%8C%E8%AE%A9%E8%87%AA%E5%B7%B1%E5%AE%8C%E5%85%A8%E6%B6%88%E5%A4%B1-c8ff94612663" href="https://medium.com/@iyouport/%E9%9A%90%E8%BA%AB%E6%B8%B8%E6%88%8F-%E5%A6%82%E4%BD%95%E5%9C%A8%E7%9B%91%E8%A7%86%E9%81%8D%E5%9C%B0%E7%9A%84%E7%A4%BE%E4%BC%9A%E9%87%8C%E8%AE%A9%E8%87%AA%E5%B7%B1%E5%AE%8C%E5%85%A8%E6%B6%88%E5%A4%B1-c8ff94612663" rel="noopener noreferrer" target="_blank">
    隐身游戏
   </a>
   》第一步，也是很重要的一步。要知道，
   <strong class="markup--strong markup--p-strong">
    在线数据信息是一个非常关键的参数，在现实生活中那些摄像头和生物识别技术很多依赖的就是在线生成的数据，政府当局也会参照这些数据以决定要不要瞄准您
   </strong>
   。
  </p>
