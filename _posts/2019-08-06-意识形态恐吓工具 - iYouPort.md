  <p class="graf graf--p">
   这是一个
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2019/04/07/world/asia/china-xi-jinping-study-the-great-nation-app.html" href="https://www.nytimes.com/2019/04/07/world/asia/china-xi-jinping-study-the-great-nation-app.html" rel="noopener noreferrer" target="_blank">
    苹果商店中国地区下载量最大的应用程序，拥有超过1亿用户
   </a>
   ，它可以阅读文章、可以参加测验、了解社会主义理论、观看习近平和共产党的政治宣传视频。“学习强国”是由中共中宣部和阿里巴巴科技公司的舆情研究部门开发的，它被比作“毛主席的红宝书”。
  </p>
  <p class="graf graf--p">
   在这里看到关于“学习强国”的更多分析《
   <a class="markup--anchor markup--p-anchor" data-href="https://iyouport.weebly.com/short-stories-enthusiast/7633210" href="https://iyouport.weebly.com/short-stories-enthusiast/7633210" rel="noopener noreferrer" target="_blank">
    强国之控
   </a>
   》。
  </p>
  <p class="graf graf--p">
   这个应用程序已经不仅仅是“红宝书”了 — — 一个被许多人视为60年代和70年代中国文化大革命期间的生存工具 — — 因为“学习强国”在国内的广泛分布，并且是在政党的压力下让人们参与其内容的。
  </p>
  <p class="graf graf--p">
   用户只需登录和阅读文章即可获得积分，但要真正收到有意义的分数，需要协同努力（或作弊）。它还禁止你谈论任何有关对应该用程序或对政府的批评。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2019/04/04/opinion/xi-jinping-thought-app.html" href="https://www.nytimes.com/2019/04/04/opinion/xi-jinping-thought-app.html" rel="noopener noreferrer" target="_blank">
    一位评论该应用程序的记者表示
   </a>
   ，当她被提示发表关于她对这种体验的看法的评论时，该应用程序显示：“只有有效的观点才能获得积分”，“好的评论将优先被展示。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们必须爱我们的国家，”35岁的退伍军人 Jiang Shuiqiu
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2019/04/07/world/asia/china-xi-jinping-study-the-great-nation-app.html" href="https://www.nytimes.com/2019/04/07/world/asia/china-xi-jinping-study-the-great-nation-app.html" rel="noopener noreferrer" target="_blank">
    告诉纽约时报
   </a>
   ，“我们越来越强大了。” 据报道他是长沙的最佳射手。
  </p>
  <p class="graf graf--p">
   报道说，学校、企业和政府办公室都在使用该应用程序的分数系统作为一种评判方式，充其量只能激励人们，更糟糕的是，惩罚他们。
  </p>
  <p class="graf graf--p">
   但报告中说明的例子表明了
   <strong class="markup--strong markup--p-strong">
    该应用程序如何被用作意识形态恐吓的工具
   </strong>
   — — 公司员工被按其应用程序的使用情况进行排名，据报道，一些雇主强迫员工每天向领导层发送分数截图。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们被告知必须每天至少得30分，”华南地区的一名政府雇员
   <a class="markup--anchor markup--p-anchor" data-href="https://www.scmp.com/news/china/politics/article/3004007/mao-zedongs-little-red-book-gets-modern-twist-mobile-app" href="https://www.scmp.com/news/china/politics/article/3004007/mao-zedongs-little-red-book-gets-modern-twist-mobile-app" rel="noopener noreferrer" target="_blank">
    告诉南华早报
   </a>
   ，“我可以通过点击文章和视频来获得该分数，但我必须调整设置以避免超时。”
  </p>
  <p class="graf graf--p">
   北京的两名公务员称，在应用程序上的使用情况方面他们很少受到上级的监督，但是，“纽约时报”报道说，有一些政府办公室组织了研究会议，让那些没有保持‘接受分数的‘工人写检讨书。
  </p>
  <p class="graf graf--p">
   并且，
   <strong class="markup--strong markup--p-strong">
    该应用程序的使用量激增并不仅仅是暴虐上司的结果 — — 据报道，获得足够积分的人也可以使用其分数来兑换某些商家的折扣或者进入一些旅游景点。
   </strong>
  </p>
  <p class="graf graf--p">
   目前还不清楚将“学习强国”应用程序纳入常规例程会对个人用户产生哪些影响，“分数不够”的人会不会被雇主追究责任？或以获得折扣为动机被强制？据“纽约时报”说，是
   <strong class="markup--strong markup--p-strong">
    中国宣传部负责收集和分析用户数据，包括他们的名字、照片、视频和联系人。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    用户还可以与应用程序内的人进行文本、呼叫和视频聊天 — — 它可以与阿里巴巴的消息应用程序 DingTalk 一起使用 — — 这意味着该国的政党可以访问一些潜在的私密关系数据和其他致命信息。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “你不能转移注意力，”澳大利亚皇家墨尔本理工大学研究中国媒体的教授 Haiqing Yu 告诉纽约时报，“这是一种数字监控。它已经将数字独裁带到了一个全新的高度。“
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    C
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://gizmodo.com/chinas-little-red-book-like-app-is-blowing-up-the-chart-1833886502" href="https://gizmodo.com/chinas-little-red-book-like-app-is-blowing-up-the-chart-1833886502" rel="noopener noreferrer" target="_blank">
    hina’s ‘Little Red Book’-Like App Is Blowing Up the Charts
   </a>
   . the examples illustrated in reports indicate how the app is used as a tool for ideological intimidation
  </p>
