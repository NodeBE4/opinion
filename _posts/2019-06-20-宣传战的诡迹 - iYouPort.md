  <p class="graf graf--p">
   不止是当权者采取的政治宣传，在党派/派系斗争中也经常采用以下这些方法。认清它们，可以避免被误导 — —
   <strong class="markup--strong markup--p-strong">
    在这种宣传战中你支持任何一方都只能是被利用
   </strong>
   。
  </p>
  <p class="graf graf--p">
   1、
   <strong class="markup--strong markup--p-strong">
    诉诸人身
   </strong>
   ：指借由与当前论题无关的个人特质，如人格、动机、态度、地位、阶级或处境等，作为驳斥对方或支持己方论证的理据。诉诸人身又称作“对人不对事”、“因人废言”或“因人设事”、“以人废言”。
  </p>
  <p class="graf graf--p">
   诉诸人身可诉诸负面特质（也称作人身攻击谬误）、正面特质、或中立特质（比如诉诸权威）。诉诸人身推论指向特定人物的某些属性并声称这些属性可以影响其可靠性。正面诉诸人身，意在提高对方的可靠性。反面诉诸人身，是贬低其可靠性。
  </p>
  <p class="graf graf--p">
   正面特质，也称因人纳言，是指仅根据论证者个人的品德高尚、出众才华、处境优越或自己对论证者的好感，就轻率地肯定其论断或观点，而不考虑其论断的内容是否真实或其论证过程是否正确，这是一种常见的谬误，比如上级、领导、老师、学长等。人身保护的谬误属于典型的因人纳言，具体包括两类：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    人格人身保护是以自己或他人的人格高尚为理由，诱使他人相信其论题为真，其实人格优秀的人坚持的看法未必就是正确的；
   </li>
   <li class="graf graf--li">
    处境人身保护是以自己或他人处境优越为理由，诱使他人相信其论题为真。但是要知道，即便处于同样的出境中也会有完全不同的理解。
   </li>
  </ul>
  <p class="graf graf--p">
   负面特质，也称因人废言、人身攻击谬误，是指仅仅根据论证者在品质、名声方面的缺陷，所处环境的特殊性，以往有过错等方面的原因或自己对立论者的厌恶态度，而不考虑立论者的论断内容是否真实，也不根据逻辑反驳的规则和要求，就对立论者的论点加以否定而表现出来的一种谬误。例如说“支持**的人都没读过书”、“支持**的人都思想落伍，不愿接受新资讯”等就是这类的谬误。
  </p>
  <p class="graf graf--p">
   多伦多大学学者道格拉斯·沃尔顿指出，常规上，诉诸人身可以分类为：辱骂性(Abusive)、情境性(Circunstantial)、扣帽子以及诉诸伪善。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    扣帽子、人身攻击：将各种不利的条件或观点强加至对方身上
   </li>
   <li class="graf graf--li">
    秽言谬误：以不雅的言词攻击对方
   </li>
   <li class="graf graf--li">
    诉诸动机：质疑对方有某种立场或地位，因此其提出之主张是基于不良的动机、企图。
   </li>
   <li class="graf graf--li">
    你也一样：对方批评某件事不对时，批评对方自己也做了那件不对的事。
   </li>
   <li class="graf graf--li">
    诉诸智能：批评对方智能不足或精神有问题，因而其主张不可采。
   </li>
   <li class="graf graf--li">
    罪恶关联：指控支持某观点的人必然归属于某个形象不好的团体。
   </li>
   <li class="graf graf--li">
    诉诸成就：主张由于某人或某人的团队没有达成某项成就，因而其针对某事的主张不可信。
   </li>
   <li class="graf graf--li">
    诉诸富贵：断言富足者的观点较正确，或断言贫困者的观点不正确。
   </li>
   <li class="graf graf--li">
    诉诸贫困：断言贫困者的观点正确，或断言富足者的观点不正确。
   </li>
   <li class="graf graf--li">
    诉诸好人：宣称某人有某些美德，因而其主张可采。
   </li>
  </ul>
  <p class="graf graf--p">
   如果当前论题确实与对方有关，那么并不是诉诸人身。
  </p>
  <p class="graf graf--p">
   例如，参选人A：“不要投票给参选人B，因为他上次当选后，很多选举承诺都没有兑现。”
  </p>
  <p class="graf graf--p">
   说明：一个人是否有信用，是选举中选民是否该选他的重要依据，因此选举时提出诚信问题，并不是犯下谬误；而在讨论一个人的诚信时，把此人过去的信用纪录拿出来是与讨论的话题直接相关的，因此不算是人身攻击。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、诉诸反复
   </strong>
   ：是指借由宣称某个观点已经被充分讨论多次、至今没有人反驳过我的观点、反方的论点之前都已经驳斥过等等，而回避对某个断言提出的举证。
  </p>
  <p class="graf graf--p">
   举例：甲：死刑是错的，我们应该废除死刑；乙：为什么死刑是错的；甲：这问题学界已经讨论过千百遍了，死刑是不对的。
  </p>
  <p class="graf graf--p">
   甲：酒驾不能算谋杀，顶多算是有认识的过失；乙：可是大家都认为酒驾就是谋杀，为什么不能算？；甲：这已经被讨论过无数次了，我不想再说。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-3653 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/88.jpg?resize=1100%2C859&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/88.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/88.jpg?resize=300%2C234&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/88.jpg?resize=768%2C600&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/88.jpg?resize=1024%2C800&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/88.jpg?resize=1100%2C859&amp;ssl=1 1100w" data-recalc-dims="1" height="859" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/88.jpg?resize=1100%2C859&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、诉诸权威
   </strong>
   ：或称伪托权威、援假权威，是一种特殊类型的归纳论证谬误，通常以统计三段论的形式来表达。虽然有时某些类别的诉诸权威能够成为有力的归纳论证，但是这种论证通常都存在被误用的情况。
  </p>
  <p class="graf graf--p">
   根据统计三段论，诉诸权威的表达方式为：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    专家甲在乙领域说的任何事情都是正确的。
   </li>
   <li class="graf graf--li">
    专家甲在乙领域提出了丙理论。
   </li>
   <li class="graf graf--li">
    因此，丙理论是正确的。
   </li>
  </ul>
  <p class="graf graf--p">
   这种归纳的论证方式是基于以下两项因素：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    专家甲是乙领域的专家。
   </li>
   <li class="graf graf--li">
    丙理论是在乙领域的专家们的讨论下得出的共识。
   </li>
  </ul>
  <p class="graf graf--p">
   这些条件也可以加到论证本身的结构中，形式如下
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    X认为A是正确的。
   </li>
   <li class="graf graf--li">
    X是该领域的专家。
   </li>
   <li class="graf graf--li">
    专家们（或经典文献）认同X。
   </li>
   <li class="graf graf--li">
    因此，假定A是正确的。
   </li>
  </ul>
  <p class="graf graf--p">
   诉诸不相关领域的权威也是一种常见的诉诸权威错误。假设专家甲是领域乙的权威，他对领域乙的发言大都是可靠的，但如果将专家甲在领域丙的言论也视为可靠，而专家甲并非领域丙的权威，就是属于一种错误的诉诸权威。但
   <strong class="markup--strong markup--p-strong">
    非常能唬人，网络大V名人使用的就是这种宣传方法。
   </strong>
  </p>
  <p class="graf graf--p">
   另一种诉诸权威的谬误是将权威说的话都当成正确。从刚才的例子可得知，虽然专家甲是领域乙的权威，但不代表他在领域乙的发言就一定是正确。相关推论的正确性应该由理据的充分性来决定，尤其是对于专家彼此争辩中的、尚未取得共识的议题。如果只是质疑提出问题的人的发言权，无论他是不是权威，都犯下了诉诸人身的谬误。
  </p>
  <p class="graf graf--p">
   甚至只要称之为权威，不署名都是经常被滥用的。例如，“有位心理学家说过，每人都有犯罪倾向”、“许多研究（而没有具体提及相关研究）都认为，对于谋杀，死刑不比无期徒刑有更强的吓阻效果”、“历年来的研究（而没有具体提及相关研究）显示，严刑峻罚不能减少犯罪”、“多数的研究（而没有具体提及相关研究）都认为对儿童诗用体罚会导致更多的偏差行为。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、诉诸恐惧
   </strong>
   ：这是一种逻辑谬误，同时也是一种宣传和广告最常用的技巧，意图让人产生恐惧或偏见，进而采取某些抵抗恐惧的措施、或接受某些信念。
  </p>
  <p class="graf graf--p">
   恐惧是一种人类最原始的情感之一，也是人类生存的一种直觉反应，当人类感到恐惧的时候，经常会做出一些可怕或不寻常的事。正由于这种天性，恐惧是最常被宣传者使用的技巧之一，而且十分有效。
  </p>
  <p class="graf graf--p">
   诉诸恐惧最常见的例子，是宣传者或广告会警告阅听人可能面临的危险或灾难，如果不立刻跟随或改变信念，就会身陷于危难当中，希望人们借此改变行为或信念，以减少恐惧，例如：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    你再继续酗酒，就会跟你爸一样早死！
   </li>
   <li class="graf graf--li">
    你如果高中毕不了业，就只好穷一辈子！
   </li>
   <li class="graf graf--li">
    把票投给他们就等于把票投给恐怖分子！
   </li>
  </ul>
  <p class="graf graf--p">
   宣传者在上面的例子中都以恐惧来强化自己的讯息，但所诉诸的恐惧是否真的与讯息相关，往往值得存疑。
  </p>
  <p class="graf graf--p">
   诉诸恐惧还有几种特殊的类型，例如恐外症（xenophobia）、对恐怖主义的恐惧、对犯罪的恐惧、对经济困顿、环境灾难、天灾、人口爆炸、侵犯个人隐私或歧视等恐惧。宣传者可以从如此宽广的恐惧光谱中挑选相关的恐惧置入讯息当中。
   <strong class="markup--strong markup--p-strong">
    当恐惧的背景资讯不是很充分的时候，诉诸恐惧的效果可以随着消息的不确定性与猜疑而倍增。当这种不确定推至极致时，就成了阴谋论
   </strong>
   。
  </p>
  <p class="graf graf--p">
   伊拉克战争前，美国政府强力宣称伊拉克拥有大规模毁灭性武器，借此合理化出兵伊拉克的行动。而许多广告、政治宣传、网络谣言及都会传奇也大量了使用诉诸恐惧的技巧。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、诉诸群众
   </strong>
   ：这也是一种逻辑谬误、一种社会心理的状态、也是一种宣传的技巧，常被称为“从众”，代表人类害怕在社会中被孤立，因而向社会其他多数靠拢的一种过程。
  </p>
  <p class="graf graf--p">
   人类经常会有一种倾向，去从事或相信其他多数人从事或相信的东西，就是所谓的“从众效应”。为了不让自己在社会中孤立，所以社会个体常常不经思考就选择与大多数人相同的选择，而从众效应就是诉诸群众的谬误和宣传的基础。
  </p>
  <p class="graf graf--p">
   从众效应衍生出谬误，又被称为“从众谬误”，就是将许多人或所有人所相信的事视为真实，例如“大家都这么说，一定不会错”！例如“很多人都有吸烟、喝酒或犯法，因此这些都是对的或不危害健康的”、“欧洲所有的国家都禁止体罚，我国也该禁止体罚”、“美国都有棱镜计划，我们也应该有”……
  </p>
  <p class="graf graf--p">
   但许多事实证明，多数或所有人相信的事情，在当下或经过了时间的演进，也并不一定就是对的，比方说，西方古时候几乎所有人都相信我们所处的世界是宇宙的中心的地心说，日心说甚至被天主教会禁止。
  </p>
  <p class="graf graf--p">
   建构于从众效应的宣传手法则是更常见的宣传法，宣传者营造出一种“加入我们，否则就是与大家作对”的气氛，要求听者接受某种仿佛大家都接受的想法。诉诸群众的宣传也暗示听者：“与胜利者站在同一边，不要去计较是非！”
  </p>
  <p class="graf graf--p">
   从众效应也常常与其他的手法合并使用，例如在广告中经常可以看到类似“每五个医师中就有四个推荐某种牌子的口香糖……”的文案，这种文案同时利用了从众效应及“诉诸权威”两种宣传手法或谬误。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    6、非黑即白
   </strong>
   ：是指提出少数选项（一般是两个，但有可能是三个或更多，要你从中选一个，但这些选择并未涵盖所有的可能性。非黑即白是一种非形式谬误。现实生活中很多事都有中间地带，很多时候中间解是更好的解，因此要人在少数选项中选一个的做法往往涉及假两难。
  </p>
  <p class="graf graf--p">
   非黑即白是基于对排中律的误用。排中律只适合衡量非此即彼的二元观念（例如“对与错”、“真与假”等等），用于其他范畴未必恰当。要破解此类谬误，可证明除了论证中提出的选项外，还有其他可能。就像“你要不就是跟我们一伙的，要不就是与我等为敌的”（You’re either with us, or against us）这类的讲法在某些情况下，会被视为假两难或非黑即白的二分法之一的例子。
  </p>
  <p class="graf graf--p">
   有时假两难是被刻意提出以迫使他人做出选择的，但其他时候它是因不小心忽略其他可能性，而非蓄意造成的。
  </p>
  <p class="graf graf--p">
   例如，A：“你不要整天吃东西！” B：“都不吃的话，我饿死怎么办？”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “整天吃”和“不吃”只是两个极端，中间还有适度进食的可能，但是被主动忽略。
  </p>
  <p class="graf graf--p">
   再比如，“你要么是外向的，要么是内向的。”
  </p>
  <p class="graf graf--p">
   内外向牵涉到很多的情境和变数，同一个人在不同情境下的表现会有所不同，可能在情境甲之下比较外向，在情境乙之下比较内向，因此内外向不是截然二分的，而是一个连续体，多数的人都介于完全的内向与完全的外向之间。
  </p>
  <p class="graf graf--p">
   再比如：“我一直把你当朋友，但你竟然在我面前帮那个坏人说话，你这是不把我当朋友，所以从今以后，你就是我的敌人！“
  </p>
  <p class="graf graf--p">
   把所有帮对方说话的人都当成敌人，是在强迫他人选边站，是在消灭中间立场。
  </p>
  <p class="graf graf--p">
   如果有充分的理由证明一件事情就是非此即彼、没有别的可能性，那就没有犯下假两难谬误。
  </p>
  <p class="graf graf--p">
   一个不是谬误的例子：“假设一辆电车驾驶面对两个轨道，只能决定走其中之一；有五个人在其中一条轨道上工作，在另一条轨道上只有一个人；电车进入的轨道上，如果有任何人，都会注定被杀。”
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-3654 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/77.jpg?resize=1100%2C737&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/77.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/77.jpg?resize=300%2C201&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/77.jpg?resize=768%2C515&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/77.jpg?resize=1024%2C686&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/77.jpg?resize=1100%2C737&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/77.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="737" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/77.jpg?resize=1100%2C737&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    7、个人崇拜
   </strong>
   ：指以大规模宣传手段将某个人在一个社群中塑造成崇拜对象，通常都通过媒体手段将其人格形象理想化、英雄化甚至神化。
  </p>
  <p class="graf graf--p">
   当政府利用大众媒体、宣传及其他方式予以不质疑的奉承和频繁的颂扬、美化等等，将某一在世（或已身故）的政治领袖塑造为英雄化、神化、理想化的公众形象，则自然会产生个人崇拜的群众现象。这些崇拜与人类本身具有的英雄仰慕等心理类似，但区别在于个人崇拜是非自主性的仰慕，必须同时伴随大量可被限制的单一媒介，将一位普通人物，进行一边倒的形象包装、过度高估的领袖特质等荒腔走板的宣传，在常态下通常人都有缺点，而个人崇拜的民众却说无法被说出来。此外，对偶像的负面批评通常是不被容忍的和不配存在的，会被视为错误、反动思想，甚至到了不配合表达效忠（如统一佩戴胸章）就会被整肃的程度，这点是两者之间的最大差别。
  </p>
  <p class="graf graf--p">
   通常个人崇拜现象有以下表征，但下列特点既非判定个人崇拜的必要条件，也非充分条件。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    崇拜对象个人及其理论被绝对化、神圣化、教条化
   </li>
   <li class="graf graf--li">
    崇拜对象以“救世主”或“解放者”的形象出现，人们对其不再具有批判、质疑的权力
   </li>
   <li class="graf graf--li">
    崇拜的程度达到迷信的程度
   </li>
   <li class="graf graf--li">
    由政客群体、社会群体共同参与的，有时甚至包括所有社会成员
   </li>
   <li class="graf graf--li">
    被崇拜者通过一切手段来巩固和强化，并利用人们的崇拜来维系他的地位与权利
   </li>
   <li class="graf graf--li">
    对异见者处以逮捕、酷刑、禁锢甚至死刑
   </li>
  </ul>
  <p class="graf graf--p">
   个人崇拜需要不断地统一宣传并排斥不协调意见的流行，这需要一些机构和政治群体来维护，当这些机构和群体是个人崇拜的受益者时，他们有足够的动力维持现状。但当崇拜对象死亡或发生意外时，容易使得政治形势剧烈变化而发生动荡，这是个人崇拜不如传统宗教神权稳定的主要原因。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    8、妖魔化
   </strong>
   ：是一种将敌手宣传为只有破坏性目标的邪恶侵略者的国家政治宣传技巧。妖魔化是最古老的宣传技巧，旨在激发对敌人的仇恨使之更易被打击，起到保护和动员盟友，使敌人沮丧的效果。
  </p>
  <p class="graf graf--p">
   由于术语妖魔化的频繁误用, 学者已不再分析它的可能语气。这就是为什么朱尔斯·博伊科夫定义了妖魔化的四个准则：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    媒体和国家都使用框架来描绘主要在道德层面的所谓的敌人的固有本性。
   </li>
   <li class="graf graf--li">
    敌人的性格是以摩尼教的方式描绘的, 正如正义与邪恶的较量。
   </li>
   <li class="graf graf--li">
    国家是这种妖魔化描绘的起源。
   </li>
   <li class="graf graf--li">
    没有来自国家的重大反诉。
   </li>
  </ul>
  <p class="graf graf--p">
   将敌人妖魔化的战略不可避免地导致了暴行的恶性循环，许多作家都对此进行了阐述，其中包括卡尔·冯·克劳塞维茨。将敌人妖魔化使得外交解决途径不可行，不可避免地导致了战争或关系恶化。尤其是把敌人描绘成邪恶力量，更容易激起杀心。
  </p>
  <p class="graf graf--p">
   《黑镜》中有一集“Men Against Fire”，将边缘人群宣传为“蟑螂”，激发战士的杀戮欲望。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    9、假情报
   </strong>
   ：这种非常常见的，又称为战略性欺骗。即 故意传播的虚假情报或信息。造谣可能以伪造的文档、手稿、照片或者使用谣言和编造的情报进行宣传。在信息战中，经常使用间谍故意散播谣言来使敌人处于某种不利的地位上。
  </p>
  <p class="graf graf--p">
   造谣和宣传不同之处在于造谣的来源是隐秘的，而宣传则是大张旗鼓的。
  </p>
  <p class="graf graf--p">
   我们后面会有专门的系列文章分析*战略性欺骗*，因为它不仅有利于防御，也有益于进攻。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    10、以退为进
   </strong>
   ：一个主要出现在社会心理学中的讨价方式。劝说者通过提出一个会被拒绝的离谱要求来让被劝说者同意第二个较为合理的请求，较之单独提出“合理的请求”更容易获接受。因为这一理论中提出离谱要求就像被人关在门外，而在英语称为“door-in-the-face”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    11、得寸进尺：
   </strong>
   是一种通过先提出一个简单的小请求来说服被劝说者同意一个较大请求的劝说方法。得寸进尺法得益于被社会学家称为“连续渐进”（successive approximations）的一项人类基本特点。该特点的大意是，如果向对方提出小的请求或作出小的行为越多，对方越有可能按照计划的方向转变自己的态度、行为，并渐渐感觉自己有必要准许那些要求更多的请求。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    12、片面真理
   </strong>
   ：一种欺骗性的陈述，其中包括一部分真理。陈述可能是部分正确的，该语句本身可能完全真实的，但只包括全部真相中的一部分，也可能利用一些欺骗性的元素，如不当的标点符号、双关语，特别是当其目的是欺骗、逃避、指责或歪曲事实时。在逻辑中，则有似真似假。主流媒体经常这么做。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    13、不相干谬误
   </strong>
   ：是指论证的前提和结论毫无逻辑关联的不当推理方式，这种情况又称不相干的结论（irrelevant conclusion）或制造伪冒理据。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    转移注意力（红鲱鱼、烟雾弹）：提出非关争议重点的言论（通常乍看有关），以转移他人的注意力
   </li>
   <li class="graf graf--li">
    打稻草人：攻击和对方论述无关的论点（通常乍看有关）
   </li>
   <li class="graf graf--li">
    我有权发表意见：被他人批评时，主张自己有权利发表意见。然而，是否有发表意见的权利和意见是否正确可取（或他人是否可批评）是不相干的。
   </li>
   <li class="graf graf--li">
    废话谬误：废话谬误的其中一种形式是说出模棱两可的、无论如何总是正确的语句，却没有真正回应主题，亦是不相干的谬误。
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    14、标语口号
   </strong>
   ：在政治、社会、商业、军事或是宗教等范筹上所使用的一句容易记忆的格言或者宣传句子，主要用作反复表达一个概念或者目标。标语的种类多不胜数，由书面至颂唱、禅理到通俗都有。很多时候它们简单的修辞特色为详情留下很少的想像空间，因此它们多被用作一个统一目的的社会表达，甚于为特定对象而作的映射。
  </p>
  <p class="graf graf--p">
   口号很有效，动员中只要能够一句合适的口号，效果会倍增。但是必须提醒，口号永远都是偏激的，致力于触发大众激烈情绪的，而不是合理的，在接受口号之前请仔细分析其是否基于正义。
  </p>
  <p class="graf graf--p">
   推荐一本书：对当今的宣传有一个很好的分析《The Propaganda Model Today: Filtering Perception and Awarenesses》在这里下载：https://t.me/iyouport/6705
  </p>
