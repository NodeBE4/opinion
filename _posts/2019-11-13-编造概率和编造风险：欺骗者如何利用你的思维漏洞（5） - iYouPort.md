  <p class="graf graf--p">
   人的大脑就像一部巨大的模式探测器，它力求从往往看似随机的布局中提取秩序和结构。
  </p>
  <p class="graf graf--p">
   很多人喜欢看星星，但你有没有想过，我们之所以能从夜空中看到“猎户座”，并不是因为这些星星就是以某种形式组织在一起的，而是，人类的大脑能够把某种模式投射到散乱无序的事物当中。
  </p>
  <p class="graf graf--p">
   掌握这点的骗子是非常容易得手的。
  </p>
  <p class="graf graf--p">
   更普遍的例子，比如，当你想念某位朋友的时候，那位朋友就“恰好”打电话来。
  </p>
  <p class="graf graf--p">
   想必很多人都遇到过这种巧合，它令你惊喜。
  </p>
  <p class="graf graf--p">
   正因为巧合令人惊喜，所以大脑就对它留下了深刻印象。
  </p>
  <p class="graf graf--p">
   但是，当你没有想到那个人的时候，那个人打电话来了，大脑就不会永远保留这样的印象。
  </p>
  <p class="graf graf--p">
   还有更多类似的案例。是否有人仔细分析过它们呢？因为，
   <strong class="markup--strong markup--p-strong">
    理清这一问题是防骗的关键
   </strong>
   。现在我们来试一试。做一个四重表。
  </p>
  <p class="graf graf--p">
   假设你在周末接到了20个电话，其中两个电话是长时间没有联系过你的朋友打来的。你之前有一阵子没想到过他们了。
  </p>
  <p class="graf graf--p">
   但就在你开始想他们时，他们在10分钟之内就打来了电话。
  </p>
  <p class="graf graf--p">
   以上这些信息就构成了四重表上第一行的内容：20个来电、其中2个是你的“超感觉信号”召唤来的，另外18个你没有“召唤”。
  </p>
  <p class="graf graf--p">
   等一下……我们还需要填写这个四重表下面一行的内容：你想念人们但人们并没打电话给你的情况有多少次？；和，你没有想念他们并且他们也没有打电话给你的情况又有多少次？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这才是关键部分
   </strong>
   。因为基本没人会去想这些问题。这正是骗子得手的微妙之处。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6960 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-4.jpeg?resize=1100%2C383&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-4.jpeg?w=1331&amp;ssl=1 1331w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-4.jpeg?resize=300%2C105&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-4.jpeg?resize=768%2C268&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-4.jpeg?resize=1024%2C357&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-4.jpeg?resize=1100%2C383&amp;ssl=1 1100w" data-recalc-dims="1" height="383" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-4.jpeg?resize=1100%2C383&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   为了填写这个表格的其他部分，我假设一周之中你想念他人的总次数为52次，一周中没有想念他人的总次数为930次（将一周的168小时分割为10分钟的小片段，总计大约就是980次想念）。
  </p>
  <p class="graf graf--p">
   现在我们已经知道，其中有50次属于人们没有打电话给你的情况（2次打电话），所以剩下的930次就是你在想其他事而不是想人的次数。
  </p>
  <p class="graf graf--p">
   也许这里的估计有点保守，但是把任何你想要的合理数字放在这里都同样能说明问题 —— 您可以自己试一下。
  </p>
  <p class="graf graf--p">
   如上面的表格，
   <strong class="markup--strong markup--p-strong">
    大脑真的只会注意到左上部的方格，而忽略掉其他3个方格 —— 这对于逻辑的培养非常有害
   </strong>
   ，但却能鼓励奇幻的思维。关于这点的详细分析见：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%a7%a3%e9%87%8a%e6%97%a0%e6%b3%95%e8%a7%a3%e9%87%8a%e7%9a%84%e8%a1%8c%e4%b8%ba%e5%bf%83%e7%90%86%e5%ad%a6%ef%bc%9a%e5%bd%93%e9%80%bb%e8%be%91%e5%a4%b1%e8%b4%a5%ef%bc%8c%e8%bf%b7%e4%bf%a1%e5%88%b0/" href="https://www.iyouport.org/%e8%a7%a3%e9%87%8a%e6%97%a0%e6%b3%95%e8%a7%a3%e9%87%8a%e7%9a%84%e8%a1%8c%e4%b8%ba%e5%bf%83%e7%90%86%e5%ad%a6%ef%bc%9a%e5%bd%93%e9%80%bb%e8%be%91%e5%a4%b1%e8%b4%a5%ef%bc%8c%e8%bf%b7%e4%bf%a1%e5%88%b0/" rel="noopener noreferrer" target="_blank">
    解释无法解释的行为心理学：当逻辑失败，迷信到来
   </a>
   》。
  </p>
  <p class="graf graf--p">
   在你订机票去拉斯维加斯玩俄罗斯轮盘赌之前，可以先玩一个数字游戏。
  </p>
  <p class="graf graf--p">
   假设你刚才想念了某个人，那么他给你打电话的概率是多少呢？：大概是2/52 或4%。就是这样。
  </p>
  <p class="graf graf--p">
   当你想念某人的时候他恰好给你打电话的概率是4%。这给人的印象还不算太深刻。
  </p>
  <p class="graf graf--p">
   当这种巧合发生时，有什么可以用来解释4%的概率呢？
  </p>
  <p class="graf graf--p">
   物理学家可能会援引你的四重表中的1000个事件，并且指出其中只有两个事件（0.2%）看似比较“怪异”，所以你应该把它看作偶然事件。
  </p>
  <p class="graf graf--p">
   社会心理学家可能想要知道，是否存在着某些
   <strong class="markup--strong markup--p-strong">
    外部事件
   </strong>
   ，它们导致你和你的朋友互相想念对方，才引起了打电话的行为。
  </p>
  <p class="graf graf--p">
   你一定读过基督城恐怖袭击事件的新闻报道；你想起了几年前和同学经常谈论去新西兰旅行的事。关于恐袭的新闻让你感觉惊讶。正因为你的朋友和你都记得当年关于新西兰旅行的交谈，这就是此时他打电话给你的原因 —— 通过这份记忆想起了你。
  </p>
  <p class="graf graf--p">
   这就是
   <strong class="markup--strong markup--p-strong">
    错觉关联
   </strong>
   。
  </p>
  <p class="graf graf--p">
   错觉关联是行为遗传学家就行为融合这种奇怪现象而提出的一种标准化解释。
  </p>
  <p class="graf graf--p">
   大脑对上述表格中的左上部方格
   <strong class="markup--strong markup--p-strong">
    非常痴迷
   </strong>
   ，就如对双胞胎兄弟的共同点非常关注；但却对不同点视而不见。
  </p>
  <p class="graf graf--p">
   很多骗子在利用这点控制你的思考方式，为他们所用。
  </p>
  <p class="graf graf--p">
   为了避免被欺骗，您可以利用这个四重表反驳任何逻辑缺失的误导。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    编造概率
   </strong>
  </p>
  <p class="graf graf--p">
   你在巴黎出差，站在蒙娜丽莎的画像前，忽然一个熟悉的声音叫出了你的名字。
  </p>
  <p class="graf graf--p">
   抬头一看，是多年前的舍友张三。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “真是难以置信啊，这可是巴黎，我们居然在蒙娜丽莎的画像前相遇了，这样的事发生的概率是多少！一定是几千万分之一吧！”
  </p>
  <p class="graf graf--p">
   是的，在蒙娜丽莎“面前”遇到老舍友的概率有可能是几千万分之一（准确计算很难，但可能性微乎其微是肯定的）。
  </p>
  <p class="graf graf--p">
   但是，这种编造概率的做法却靠不住。
  </p>
  <p class="graf graf--p">
   可以退一步思考这个问题。
  </p>
  <p class="graf graf--p">
   假如你和张三相遇的地点不是蒙娜丽莎画像前，而是维纳斯前呢？或者在酒店大厅？卫生间？咖啡馆？或者埃菲尔铁塔脚下？情况又会如何？
  </p>
  <p class="graf graf--p">
   你同样会感到他乡遇故知的惊喜，与蒙娜丽莎面前的感受分毫不差；这种偶遇又为什么只能限于巴黎呢？它完全可以发生在马德里的出差途中、克利夫兰转机的机场上、图森的温泉浴场里……
  </p>
  <p class="graf graf--p">
   请用这种方式重新表述以上事件的概率：你会在自己意料之外的任何地方与你认识的人相遇。很明显这种相遇感觉很好，但是人的大脑并不会主动这样去思考。
  </p>
  <p class="graf graf--p">
   认知科学已经表明，通过培训使自己避免这种拖泥带水的思维方式有多么重要。
   <strong class="markup--strong markup--p-strong">
    不论是政治还是经济方面，骗子最擅长利用这点对你下手
   </strong>
   。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    编造风险
   </strong>
  </p>
  <p class="graf graf--p">
   如上，偶遇的例子只是编造概率中一个最无害的例子。但是，要知道。编造概率可以非常危险，因为它阻止你对风险做出理性的分析。
  </p>
  <p class="graf graf--p">
   最著名的例子是：飞机恐惧。
  </p>
  <p class="graf graf--p">
   即便把911事件都算进去，乘坐飞机仍然是相对最安全的交通方式。铁路安全性紧随其后。
  </p>
  <p class="graf graf--p">
   尽管如此，911事件后很多人都会在出行时避免乘坐飞机，甚至宁可选择高速公路的汽车旅行。正因此，在911后几个月内，美国死于汽车事故的人数急剧飙升。
  </p>
  <p class="graf graf--p">
   因为人们只能用直觉去感受，而非理性分析，于是无法知道自己承担的风险其实是增加了而非减少。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    新闻故事中误导性的文字比比皆是
   </strong>
   。随便搜索你就会看到这样的文字：“2014年死于飞机事故的人比1960年时更多”。
  </p>
  <p class="graf graf--p">
   读到这句话的人也许倾向得出结论：乘坐飞机已经“更加不安全了”。
  </p>
  <p class="graf graf--p">
   虽然这条统计内容正确，但是，
   <strong class="markup--strong markup--p-strong">
    它不具有相关性
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果你想弄清楚的问题是乘坐飞机的安全性，那么统计死亡总数并不能给你想要的答案 —— 你应该做的是统计死亡率。
  </p>
  <p class="graf graf--p">
   即 每英里飞行距离的死亡人数，或每次飞行的死亡人数，抑或相当于统计单位基准的其他内容。
  </p>
  <p class="graf graf--p">
   虽然1960年时的航班数量没有现在多，但毫无疑问，那时候的
   <a class="markup--anchor markup--p-anchor" data-href="https://edition.cnn.com/interactive/2014/07/travel/aviation-data/" href="https://edition.cnn.com/interactive/2014/07/travel/aviation-data/" rel="noopener noreferrer" target="_blank">
    风险更大
   </a>
   。
  </p>
  <p class="graf graf--p">
   同样逻辑，你可以说，在高速公路上，下午5点～7点比凌晨2点～4点死亡人数更多；所以“你应该避免在下午5点～7点出行”。
  </p>
  <p class="graf graf--p">
   但实际上，选择在下午5点～7点出行的总人数是选择在凌晨2点～4点出行的总人数的好几倍，却被忽略了 —— 你应该做的是分析死亡率，而不是原始数字。
  </p>
  <p class="graf graf--p">
   死亡率会告诉你，凌晨2点～4点更危险，很可能与酒驾或疲劳驾驶相关。
  </p>
  <p class="graf graf--p">
   巴黎恐袭发生后，CNN说，“在整个欧洲反难民情绪不断高涨的背景下，进入欧盟国家的恐怖分子至少有一半是难民身份；长久以来，欧洲反难民积极分子都在呼吁欧盟实施更严格的边界管控政策……”。
  </p>
  <p class="graf graf--p">
   本文不关心社会和政治，但数字可以更好地说明问题。
  </p>
  <p class="graf graf--p">
   巴黎恐袭死亡人数130人；如果不允许被战火蹂躏的地区高达数百万难民跨越边界，毫无疑问会造成至少数万人失去生命。对于那些对数字无感的人来说，诸如“难民中只要有一名是恐怖分子”这样的标题非常容易点燃反移民人群的情绪，同时忽略了移民政策也挽救过很多人的生命。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    恐怖分子期望你相信的谎言就是：你此刻正在面临被灭亡的险境。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这就是编造风险
   </strong>
   。这种欺骗的使用非常普遍。尤其是营销方面。
  </p>
  <p class="graf graf--p">
   比如你收到了一家生产家庭用监视摄像头公司发来的推销邮件，其中基本会有这样一句话：“90%的入室抢劫案都依靠业主提供的监视视频得以破获”。
  </p>
  <p class="graf graf--p">
   听起来很棒吧，多有实证性啊是吧？
  </p>
  <p class="graf graf--p">
   其实这句话严重有问题。请检查它的合理性。
  </p>
  <p class="graf graf--p">
   忽略掉“监视视频”，只看基本框架：“90%的入室抢劫案都依靠 ……得以破获”。这看起来合理吗？
  </p>
  <p class="graf graf--p">
   不需要查阅实际统计数据，只需借助你在现实生活中的知识即可：90%的入室抢劫都得以破获，这显然非常不真实。因为对任何警察部门来说，90%的破案率都堪称神奇。
  </p>
  <p class="graf graf--p">
   在网上搜索一下，很快找到联邦调查局网页报告，其中写道：大约30%的盗窃案件得到了“清除”，也就是破案。
  </p>
  <p class="graf graf--p">
   于是你能知道，那句推销辞令是极度夸张的，不属实的。这家公司的意思更可能是：“已经破获的案件”是借助业主提供的视频破获的。
  </p>
  <p class="graf graf--p">
   这两者可
   <strong class="markup--strong markup--p-strong">
    不是一回事
   </strong>
   。
  </p>
  <p class="graf graf--p">
   因为样本池不同。
  </p>
  <p class="graf graf--p">
   在第一种情况下，我们关注的是所有入室抢劫；在第二种情况下，我们考虑的仅仅是已经被破获的案件，这些案件的数量要少得多。
  </p>
  <p class="graf graf--p">
   那么，这是不是说，如果我安装了监视摄像头，警察侦破发生正在我家的盗窃案件的概率就是 90%呢？
  </p>
  <p class="graf graf--p">
   绝对不是。
  </p>
  <p class="graf graf--p">
   你所知道的仅仅是：如果一个案件得以破获，那么警察在破案过程中得到视频帮助的概率为90%。
  </p>
  <p class="graf graf--p">
   但如果你认为现在已经得到了充分的信息可以回答关于“有没有必要安装监视摄像头”的问题，那就真的错了 —— 再回到那个四重表中就可以直观地看清楚。
  </p>
  <p class="graf graf--p">
   表格中第一行所需要的数字已经有了：就是破获案件中家庭装有摄像头的百分比；但是，要填写下面的行列，还需要知道：那些没有被侦破的案件中家庭装有摄像头的比例。
  </p>
  <p class="graf graf--p">
   要知道，
   <strong class="markup--strong markup--p-strong">
    P（盗窃案破获︱装有家庭摄像机） ≠ P（装有家庭摄像机︱盗窃案破获）。
   </strong>
  </p>
  <p class="graf graf--p">
   这家生产家庭监视摄像头的公司之所以编造风险数据，是因为其目的在于激起你的情感波澜，从而促使你购买它的产品。
  </p>
  <p class="graf graf--p">
   然而，这种产品实际上根本起不到宣传中所承诺的作用。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    甚至很容易被骇客入侵而变成随时监视你自己私生活的设备
   </strong>
   。
  </p>

   <strong class="markup--strong markup--p-strong">
    信念固执
   </strong>
  </p>
  <p class="graf graf--p">
   人类认知方面的一个奇怪特点就在于：一旦形成了某种信念或接受了某种说法，即便在面对压倒性的证伪证据时都无法改变固有的想法。
  </p>
  <p class="graf graf--p">
   这就是为什么虚假宣传明明逻辑混乱、证据缺失，却依旧可以骗倒很多人。就如我们在《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%ba%e6%a0%bc%e6%9a%97%e6%9d%80%e5%92%8c%e6%8a%b9%e9%bb%91%e8%bf%90%e5%8a%a8%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%8f%97%e7%9b%8a%ef%bc%9f/" href="https://www.iyouport.org/%e4%ba%ba%e6%a0%bc%e6%9a%97%e6%9d%80%e5%92%8c%e6%8a%b9%e9%bb%91%e8%bf%90%e5%8a%a8%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%8f%97%e7%9b%8a%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    人格暗杀
   </a>
   》中所分析的那样：人格暗杀操纵者胡言乱语毫无根据，但不会影响其暗杀效应。
  </p>
  <p class="graf graf--p">
   是人类思维固有的缺陷给了宣传者和所有骗子以强大的信心。
  </p>
  <p class="graf graf--p">
   中英文都将宣传称之为洗脑，这很形象，但有些人对洗脑的理解依旧不充分。
  </p>
  <p class="graf graf--p">
   要知道，洗脑并非人们能认识到的那些“谎言和恶意欺骗”，
   <strong class="markup--strong markup--p-strong">
    相反，洗脑恰恰是人们无法认识到的、无法理解到是被灌输的东西，误认为它们是来自自己“内心的想法”。
   </strong>
   反洗脑非常难，因为很少有人能知道该从哪开始。
  </p>
  <p class="graf graf--p">
   这就是为什么IYP曾经多次对此做出分析。关键部分：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    旨在改变观点和态度的直接宣传必须具有先前宣传，这种宣传具有社会性、缓慢性、一般性，目的是寻求创造气氛，有利于初步态度的氛围。没有预先宣传，任何直接宣传都不会有效……《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%93%8d%e7%ba%b5%e5%bc%8f%e5%ae%a3%e4%bc%a0%ef%bc%9a%e4%ba%94%e4%b8%aa%e8%bf%87%e6%bb%a4%e5%99%a8%e5%92%8c%e8%af%86%e5%88%ab%e7%ba%bf%e7%b4%a2/" href="https://www.iyouport.org/%e6%93%8d%e7%ba%b5%e5%bc%8f%e5%ae%a3%e4%bc%a0%ef%bc%9a%e4%ba%94%e4%b8%aa%e8%bf%87%e6%bb%a4%e5%99%a8%e5%92%8c%e8%af%86%e5%88%ab%e7%ba%bf%e7%b4%a2/" rel="noopener noreferrer" target="_blank">
     操纵式宣传：五个过滤器和识别线索
    </a>
    》
   </li>
   <li class="graf graf--li">
    如果你认为所有事情至少都是预先确定的，那么你就不必为所发生的事的任何特定部分承担责任。 更重要的是，你不必记住真实过去发生的事，因为那些替代品已经死了。你认为那些东西不可能回来。《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%9c%a8%e7%ba%bf%e5%ae%a3%e4%bc%a0%e6%88%98%e7%9a%84%e7%bb%93%e6%9e%9c/" href="https://www.iyouport.org/%e5%9c%a8%e7%ba%bf%e5%ae%a3%e4%bc%a0%e6%88%98%e7%9a%84%e7%bb%93%e6%9e%9c/" rel="noopener noreferrer" target="_blank">
     在线宣传战的结果
    </a>
