     <em>
      人工智能是当今技术中被滥用程度最高的术语之一，一项新的研究证实了该技术的炒作方式。想想看，中国究竟有多少这类假货？
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   人工智能是当今技术中被滥用程度最高的术语之一，一项新的研究显然证实了该技术的炒作方式。
  </p>
  <p class="graf graf--p">
   根据伦敦风险投资公司
   <a class="markup--anchor markup--p-anchor" data-href="https://www.mmcventures.com/wp-content/uploads/2019/02/The-State-of-AI-2019-Divergence.pdf" href="https://www.mmcventures.com/wp-content/uploads/2019/02/The-State-of-AI-2019-Divergence.pdf" rel="noopener noreferrer" target="_blank">
    MMC 的调查
   </a>
   ，40％ 被归类为人工智能公司的欧洲创业公司实际上并未以对其业务“重要”的部分使用人工智能。MMC 在13个欧盟国家研究了大约2,830个人工智能创业公司，以便对每个公司的“活动、重点和资金”进行审查。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “在40％的案例中我们都没有发现人工智能的证据，”编制该报告的 MMC 研究负责人 David Kelnar
   <a class="markup--anchor markup--p-anchor" data-href="https://www.forbes.com/sites/parmyolson/2019/03/04/nearly-half-of-all-ai-startups-are-cashing-in-on-hype/#74804801d022" href="https://www.forbes.com/sites/parmyolson/2019/03/04/nearly-half-of-all-ai-startups-are-cashing-in-on-hype/#74804801d022" rel="noopener noreferrer" target="_blank">
    告诉福布斯新闻
   </a>
   。 Kelnar 说，这意味着“人们认为和思考的公司可能不是AI公司。”
  </p>
  <p class="graf graf--p">
   正如福布斯所指出的那样，这些创业公司是“人工智能公司”的说法并不一定来自公司本身。第三方分析站点通常负责这种分类，并且从 MMC 的报告中不清楚它确定的假冒 AI 初创公司中在多大程度上积极误导了客户。
  </p>
  <p class="graf graf--p">
   但该报告的调查结果显示，如果被错误分类，公司就会有明显的激励措施。
   <strong class="markup--strong markup--p-strong">
    “人工智能”一词显然是对投资者的触动，MMC 发现声称AI的创业公司比其他公司多吸引了15％到50％的资金。
   </strong>
  </p>
  <p class="graf graf--p">
   正如 Kelnar 对福布斯所说的那样：“我认为在大多数情况下[初创公司]都会意识到他们是如何被归类的。”他们只是不在乎纠正这些记录。
  </p>
  <p class="graf graf--p">
   MMC 的报告发现，当公司部署人工智能和机器学习时，用例通常非常平庸。初创公司调查使用AI的一些最流行的方式包括聊天机器人（26％的公司）和欺诈检测（21％）。
  </p>
  <p class="graf graf--p">
   在这两种情况下，判断这项技术对客户有多大帮助是很棘手的。聊天机器人通常很烦人，可以只是消除人力客户支持的成本。虽然欺诈检测对客户和企业来说无疑是有用的，但它更像是辅助服务而非中心卖点。
  </p>
  <p class="graf graf--p">
   但是，正如我们从
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theverge.com/circuitbreaker/2017/1/4/14164206/smart-toothbrush-ara-ai-kolibree" href="https://www.theverge.com/circuitbreaker/2017/1/4/14164206/smart-toothbrush-ara-ai-kolibree" rel="noopener noreferrer" target="_blank">
    人工智能牙刷
   </a>
   这样的小工具的扩散中学到的，仅仅是因为某些东西说它是人工智能，并不意味着它重要。人工智能是一个术语，涵盖了广泛的技术，显然，其中包含了大量的炒作。
  </p>
  <p class="graf graf--p">
   想想看，中国究竟有多少这类假货？
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    F
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theverge.com/2019/3/5/18251326/ai-startups-europe-fake-40-percent-mmc-report" href="https://www.theverge.com/2019/3/5/18251326/ai-startups-europe-fake-40-percent-mmc-report" rel="noopener noreferrer" target="_blank">
    orty percent of ‘AI startups’ in Europe don’t actually use AI, claims report
   </a>
   . Companies want to take advantage of the AI hype
  </p>
