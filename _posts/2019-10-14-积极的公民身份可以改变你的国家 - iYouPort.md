  <p class="graf graf--p">
   教育和民主公民身份之间的关系是一个重要的议题。
  </p>
  <p class="graf graf--p">
   政治变革可能被很多种因素所促进，但变革的结果究竟是不是走向民主、尤其是如何维护住一个新生的民主制度的生命力，需要做的工作比推动变革本身要更多。
  </p>
  <p class="graf graf--p">
   其中教育是一个极为重要的部分。
  </p>
  <p class="graf graf--p">
   民主的维护在于公民的积极参与和认知的清晰；新生民主社会往往不具备这一基础。如果能从变革前开始着手培养积极的公民意识，胜算也许更大。
  </p>
  <p class="graf graf--p">
   这就是翻译这篇文章的原因。
  </p>
  <p class="graf graf--p">
   本文来自 Kurt Lewin 基金会的 KatalinSzéger 与 Open Society Think Tank Fund 的 Andrej Nosko 进行的讨论，关于：积极的公民身份。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    什么是积极的公民身份？
   </em>
  </p>
  <p class="graf graf--p">
   积极的公民身份意味着人们参与当地社区和各级民主，从城镇到城市，再到全国范围的社会活动。
  </p>
  <p class="graf graf--p">
   积极的公民身份可以像义务清理街道的运动一样小，也可以像教育年轻人的民主价值观、技能和参与能力一样大。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    积极的公民身份是迈向健康社会的最重要步骤之一
   </strong>
   ，特别是对于新诞生的民主国家来说，比如匈牙利。
  </p>
  <p class="graf graf--p">
   Kurt Lewin基金会在位于匈牙利东部的四所学校开展了一项为期一年半的研究和发展计划，研究主题是学校与民主公民之间的关系。
  </p>
  <p class="graf graf--p">
   年龄在17至19岁之间（11至12年级）的学生和他们的老师参加了这项研究；
   <strong class="markup--strong markup--p-strong">
    该研究考察了积极的公民身份、民主价值观以及社会和政治态度在教育中的地位和影响
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.opensocietyfoundations.org/reports/youth-and-active-citizenship" href="https://www.opensocietyfoundations.org/reports/youth-and-active-citizenship" rel="noopener noreferrer" target="_blank">
    该报告
   </a>
   旨在研究学校如何作为培养学生积极公民身份的基地。
  </p>
  <p class="graf graf--p">
   虽然匈牙利教育政策强调了以活动、责任和民主价值为基础的公民身份概念，但
   <strong class="markup--strong markup--p-strong">
    匈牙利年轻一代的观点和意见似乎正朝着不利于加强积极公民身份的方向发生变化
   </strong>
   。
  </p>
  <p class="graf graf--p">
   因此，Kurt Lewin 基金会建立了一个研究和教育项目，一方面通过各种发展活动努力提高积极的公民身份，另一方面围绕积极的公民身份问题收集研究数据。
  </p>
  <p class="graf graf--p">
   正如我们所理解的那样，
   <strong class="markup--strong markup--p-strong">
    积极的公民身份是知识、态度、技能和行动的结合，旨在为建设和维护民主社会作出贡献
   </strong>
   。
  </p>
  <p class="graf graf--p">
   积极公民身份支持民主合作，其基础是接受普遍的人权和法治、价值观多样性，并包容整个社区。
  </p>
  <p class="graf graf--p">
   事实上在我们共同的社会生活的各个层面上都能获得这种知识、发展和实践技能的教育，但学校在其中发挥着非常重要的作用。
  </p>
  <p class="graf graf--p">
   我们越来越多地看到，在涉及到欧洲的问题时，需要采用极端的解决方案。
  </p>
  <p class="graf graf--p">
   在这种情况下，积极的公民身份可以发挥关键作用，使公民能够审视问题的真正解决办法，而不是沉溺于极端者兜售的替罪羊、歧视和其他策略。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    您的报告侧重于学校在积极公民身份中的作用；研究发现了什么？
   </em>
  </p>
  <p class="graf graf--p">
   我们关注学校组织文化和隐藏课程的影响如何着眼于学生的积极公民身份（隐藏课程指的是一套规则、规范、态度和行为，它不属于国家课程，但是学生可以通过社区/机构的生活和学习而获得，这是学校教育的一种副作用）。
  </p>
  <p class="graf graf--p">
   我们在两所语言学校（一所由教会维护、另一所由政府维护）以及两所职业中学中进行研究。
  </p>
  <p class="graf graf--p">
   一般来说，
   <strong class="markup--strong markup--p-strong">
    学生重视与同学的关系，但只有不到一半的学生能够与老师分享他们的问题
   </strong>
   。
  </p>
  <p class="graf graf--p">
   对于少数民族来说，学生们
   <strong class="markup--strong markup--p-strong">
    没有表达出高度的团结。他们对政治、民主和社会问题感到失望。
   </strong>
  </p>
  <p class="graf graf--p">
   学生们似乎更喜欢不那么民主、更强硬、和更专制的方法。
  </p>
  <p class="graf graf--p">
   他们对任何政党都不感兴趣。
  </p>
  <p class="graf graf--p">
   尽管如此，
   <strong class="markup--strong markup--p-strong">
    激进的极端政治观点通常比主流政党更容易被接受
   </strong>
   。
  </p>
  <p class="graf graf--p">
   与学生相比，教师们似乎认为民主价值观更重要，更有可能拒绝反民主活动。但是，尽管如此，
   <strong class="markup--strong markup--p-strong">
    关于教师的调查所呈现的结果仍然是对政治的不信任和失望。
   </strong>
  </p>
  <p class="graf graf--p">
   尽管有这些发现，总体上学生们仍然认为参与和承诺很重要，他们会欢迎学校
   <strong class="markup--strong markup--p-strong">
    更强大的社区生活
   </strong>
   。
  </p>
  <p class="graf graf--p">
   通过我们的研究，我们发现
   <strong class="markup--strong markup--p-strong">
    教师 —— 出于许多不同的原因 —— 经常会回避讨论社会和政治问题。
   </strong>
  </p>
  <p class="graf graf--p">
   如果没有一个开放的和鼓励性的环境让学生们讨论社会及政治经验和问题，这只能播下不安全感、恐惧感和混乱感的种子，并为学校以外经常提供的*偏执答案*开辟侵入途径。
  </p>
  <p class="graf graf--p">
   公民教育在这里发挥着重要作用。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5665 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/投票.jpg?resize=1100%2C825&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/投票.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/投票.jpg?resize=300%2C225&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/投票.jpg?resize=768%2C576&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/投票.jpg?resize=1024%2C768&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/投票.jpg?resize=1100%2C825&amp;ssl=1 1100w" data-recalc-dims="1" height="825" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/投票.jpg?resize=1100%2C825&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    您为提高积极的公民身份提出了哪些建议？
   </em>
  </p>
  <p class="graf graf--p">
   需要制定教育计划以提高合作和沟通等能力，同时努力提高批判能力，减少偏见，建立宽容，理解，同理心和对多样性的开放环境。
  </p>
  <p class="graf graf--p">
   在研究期间，开发了
   <strong class="markup--strong markup--p-strong">
    量身定制的计划
   </strong>
   来解决这些问题，并得到参与者的积极响应。现在需要更广泛的认可和实施。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    世界上有哪些国家是积极公民身份的领导者或榜样？是什么让他们值得被效仿？
   </em>
  </p>
  <p class="graf graf--p">
   尽管不同国家和文化中的积极公民教育的定义存在差异，但我们可以说，在德国，尤其是西德和不来梅等城市，
   <strong class="markup--strong markup--p-strong">
    由于对积极的公民身份的促进，已经建立了至关重要的公民社会和政治文化
   </strong>
   。
  </p>
  <p class="graf graf--p">
   有影响力的机构努力谴责排他性或极端主义态度。
  </p>
  <p class="graf graf--p">
   在公共部门（学校、青年中心等）工作人员与年轻人联合，以打击专制态度并支持跨文化和民主能力的发展，在法律上和经济上都有强有力的官方支持。
  </p>
  <p class="graf graf--p">
   虽然公民社会一方面致力于积极地影响政治，但
   <strong class="markup--strong markup--p-strong">
    政治组织和民间组织之间也在通过合作开展这些项目。
   </strong>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    您说最近关于青年和积极公民身份的报告部分是为了回应“反民主思想”的增长 —— 这指的是什么？它们的影响是什么？
   </em>
  </p>
  <p class="graf graf--p">
   在匈牙利，近年来我们目睹了不容忍现象的急剧增加。
  </p>
  <p class="graf graf--p">
   反罗姆人，反犹太主义，民族主义，种族主义和反同性恋的言论和活动都有所增加。
  </p>
  <p class="graf graf--p">
   这些趋势的影响可以在很多年轻人的生活中找到。
  </p>
  <p class="graf graf--p">
   我们最需要关注的是那些
   <strong class="markup--strong markup--p-strong">
    缺乏
   </strong>
   围绕年轻人所经历的各种体验进行讨论和组织教育计划的学校。
  </p>
  <p class="graf graf--p">
   在我们的计划中，除了研究结果外，我们还提供可以在学校和政策制定层面提供帮助的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.opensocietyfoundations.org/publications/policy-recommendation-development-education-democratic-citizenship-elementary-and" href="https://www.opensocietyfoundations.org/publications/policy-recommendation-development-education-democratic-citizenship-elementary-and" rel="noopener noreferrer" target="_blank">
    解决方案
   </a>
   。⚪️
  </p>
  <p class="graf graf--p">
   **在上面链接中可下载36页的解决方案建议。希望能为中国的活动家和非政府组织提供一些灵感（据说中国的“在家教育”服务正在兴起，我们认为这是一个很好的趋势）。
  </p>
