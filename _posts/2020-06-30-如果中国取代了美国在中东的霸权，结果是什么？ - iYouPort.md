  <p class="graf graf--p">
   中国声称其外交政策是基于“非干预主义”的，以区别于过去几十年来美国在中东外交政策中所体现的鹰派态度。

   </li>
  </ul>
  <p class="graf graf--p">
   幸运的是至少目前为止中国政府尚且没有直接参与中东战争。北京的参与主要取决于所谓的 “一带一路” 倡议或急需从该地区进口石油等经济投资。
  </p>
  <p class="graf graf--p">
   但是，这些关系可能会导致中国在该地区发挥政治作用，无论它是否喜欢。

  <p class="graf graf--p">
   经过数年适得其反的干预，美国正在从中东撤退。特朗普成功地基于非干预主义外交政策开展了他自己的竞选活动，承诺将军队从饱受战争蹂躏的地区带回来（虽然他显然做不到）。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    并非巧合
   </strong>
   的是，美国的能源供应也越来越自给自足，其石油消费量占其自身消耗的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.eia.gov/energyexplained/oil-and-petroleum-products/imports-and-exports.php" href="https://www.eia.gov/energyexplained/oil-and-petroleum-products/imports-and-exports.php" rel="noopener noreferrer" target="_blank">
    86％
   </a>
   。预计明年这一数字将再次
   <a class="markup--anchor markup--p-anchor" data-href="https://www.reuters.com/article/us-usa-oil-eia-outlook/us-oil-output-to-hit-new-record-above-13-million-bpd-in-2020-eia-idUSKCN1Q12C3" href="https://www.reuters.com/article/us-usa-oil-eia-outlook/us-oil-output-to-hit-new-record-above-13-million-bpd-in-2020-eia-idUSKCN1Q12C3" rel="noopener noreferrer" target="_blank">
    上升
   </a>
   。美国对参与中东石油开采不再感兴趣。
  </p>
  <p class="graf graf--p">
   另一方面，
   <strong class="markup--strong markup--p-strong">
    中国则越来越依赖国外进口能源
   </strong>
   。中国几乎
   <a class="markup--anchor markup--p-anchor" data-href="http://www.worldstopexports.com/top-15-crude-oil-suppliers-to-china/" href="http://www.worldstopexports.com/top-15-crude-oil-suppliers-to-china/" rel="noopener noreferrer" target="_blank">
    一半
   </a>
   的石油供应来自中东，沙特阿拉伯是仅次于俄罗斯的第二大出口国，位居榜首。
  </p>
  <p class="graf graf--p">
   沙特阿拉伯在该地区的头号敌人 —— 伊朗 —— 在对中国的石油出口国排名中仅排第七位。因此，即使德黑兰希望中国与伊朗建立对抗美国制裁的特殊关系，
   <strong class="markup--strong markup--p-strong">
    北京也不太可能
   </strong>
   就伊朗与海湾的利益成为伊朗的盟友。
  </p>
  <p class="graf graf--p">
   中国在该地区的基础设施项目对当前的冲突也保持中立。中国的新丝绸之路经济带穿过伊朗，而海上丝绸之路则从阿曼、沙特和埃及穿过波斯湾。
  </p>

  <p class="graf graf--p">
   北京声明，其目标是与它所参与的所有国家建立互惠互利的协议，而不管它们之间可能存在的争议。
  </p>
  <p class="graf graf--p">
   中国声称没有在国外复制美国的干预项目，而是声称希望在不影响其他国家内政的情况下 “带来发展”。
  </p>
  <p class="graf graf--p">
   但是，在由宗派形式的战争所界定的地区中，中国要保持中立地位
   <strong class="markup--strong markup--p-strong">
    将变得越来越困难
   </strong>
   ，尤其是在美国准备退出并因此出现权力真空的情况下。
  </p>
  <p class="graf graf--p">
   迄今为止，中国政府一直没有在中东地区的众多冲突中表达任何官方立场。但是，中立性可能伴随着代价，迫使中国无意中采取了行动。海湾国家已经怀疑北京与伊朗的友好关系。
  </p>
  <p class="graf graf--p">
   但是，尽管德黑兰希望与中国结成联盟，但北京在激怒其邻国以及挑战美国目前的霸权方面仍然保持谨慎。
  </p>
  <p class="graf graf--p">
   中国的国有石油公司由于美国的制裁，已于去年10月
   <a class="markup--anchor markup--p-anchor" data-href="https://www.scmp.com/news/china/diplomacy/article/3031787/iran-says-chinese-state-oil-firm-withdraws-us5-billion-deal" href="https://www.scmp.com/news/china/diplomacy/article/3031787/iran-says-chinese-state-oil-firm-withdraws-us5-billion-deal" rel="noopener noreferrer" target="_blank">
    撤回
   </a>
   了一笔50亿美元的交易，这笔钱是用来开发伊朗大规模海上天然气的一部分；同时，
   <strong class="markup--strong markup--p-strong">
    中国不愿向伊朗提供任何重要的军事技术，而
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.albawaba.com/news/china-selling-autonomous-weaponized-drones-saudi-arabia-and-pakistan-1321951?fbclid=IwAR0InutAYVLzPODkQ1jxu5XGaqMwngQDdfKp-TVhsigsVRU2Rlq6K2pOgGw" href="https://www.albawaba.com/news/china-selling-autonomous-weaponized-drones-saudi-arabia-and-pakistan-1321951?fbclid=IwAR0InutAYVLzPODkQ1jxu5XGaqMwngQDdfKp-TVhsigsVRU2Rlq6K2pOgGw" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     据称
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    希望向巴基斯坦和沙特阿拉伯等中东其他国家提供 “杀手级无人机”
   </strong>
   。
  </p>

  <p class="graf graf--p">
   美国已经向其在中东的盟国施加压力，以限制他们与中国的接触。五角大楼负责该地区的最高官员 Michael Mulroy 警告说，北京在中东的影响力可能会破坏美国与其海湾盟国之间的军事合作。
  </p>
  <p class="graf graf--p">
   担心沙特阿拉伯这样的海湾国家可能对美国的支持失去信心，并开始自己的核计划，伊朗也会秘密地这样做。据
   <a class="markup--anchor markup--p-anchor" data-href="https://edition.cnn.com/2019/06/05/politics/us-intelligence-saudi-arabia-ballistic-missile-china/index.html" href="https://edition.cnn.com/2019/06/05/politics/us-intelligence-saudi-arabia-ballistic-missile-china/index.html" rel="noopener noreferrer" target="_blank">
    报道
   </a>
   ，美国政府已经获得情报，表明利雅得正在北京的支持下发展弹道导弹计划。
  </p>
  <p class="graf graf--p">
   从今天的情况来看，无论是在经济上还是军事上，沙特和更广泛的海湾国家似乎都是中国在中东的主要区域盟友。
  </p>
  <p class="graf graf--p">
   这种关系将如何发展，尤其是中国声称的不就其支持如何影响该地区采取任何立场，尚有待观察。⚪️
  </p>
