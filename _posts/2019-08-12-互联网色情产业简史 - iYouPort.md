   人类历史上是什么站在推动新媒介的前沿？是色情。互联网色情业究竟有多赚钱？或许超乎你的想象……这篇文章来自一本新书的节选 The Players Ball: A Genius, a Con Man, and the Secret History of the Internet’s Rise， by David Kushner。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “万能的上帝，万物之主，我们赞美你在我们这个时代享受的计算机化通信的进步。然而，令人遗憾的是，有些人在淫乱、不雅和破坏性的色情内容中污染了这条信息高速公路“……

  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   他在那里敦促参议员们通过他和印第安纳州参议员 Dan Coats 对“

    通信规范法案
   </a>
   ”（CDA）的修正案，该法案将现有的猥亵和反猥亵法扩展到了新兴互联网时代的“交互式计算机服务”领域。 他在祈祷中说，“考虑控制计算机通信污染的方法以及如何保护我们最大的资源之一：我们孩子的思想以及我们国家的未来和道德力量。阿门“。
  </p>
  <p class="graf graf--p">
   Exon 举起了一个蓝色的活页夹，他警告说，这里面充满了“变态色情”，在网上“只需点击几下”。“我不能也不会向参议院展示这些照片，”他说，但是“我希望所有同事，如果他们感兴趣的话，都会来我这里拿走看看这种令人作呕的材料。“
  </p>
  <p class="graf graf--p">
   他们的确很感兴趣。
  </p>
  <p class="graf graf--p">
   他们一个接一个地翻阅了这些“怪诞的东西”，就像 Coats 所说的那样，创新得到了培养。他引用了一项数据 — 虽然很可疑 — 来自一项研究，该研究发现在线的色情图像超过450,000，去年的访问量约为640万次。主要来源是免费的

  <p class="graf graf--p graf--startsWithDoubleQuote">

  </p>
  <p class="graf graf--p">
   然而，对色情和新兴技术的这种关注远非新鲜。研究技术文化影响的哥伦比亚大学研究员约翰·蒂尔尼（John Tierney）追溯他称之为“色情技术冲动”的东西到了至少27000年前 — — 从那个时代发现的第一批粘土烧制的人物形象中，就存在大乳房和大屁股的性感女性。

   <strong class="markup--strong markup--p-strong">
    “几乎总是这样，从石器时代的雕塑到电脑公告栏，色情一直是新媒体的首批用途之一。”
   </strong>
  </p>

   尽管有公民自由主义者和怀疑论者的强烈抗议，Rimm 的研究最终依旧成为了“沟通规范法”提案的基础。（Rimm 自己也暗示他能够确定“任何特定日期在 Usenet 上可用的所有图像的色情内容百分比”是纯粹的幻想）

  <p class="graf graf--p">
   然而，到了那年的夏天，该法律的基础已经彻底失去了信誉。 Rimm 的论文受到评论家的抨击，被发现是在没有同行评议的情况下发表的，评论家们认为这是反色情活动家的阴谋。“纽约时报”将这项研究视为“扼杀者”，充斥着“误导性的分析、含糊不清的定义和不受支持的结论”。在互联网 trolls 的攻击下，Rimm 躲了起来。但他和参议员们的工作“已经完成了“。

   整个世界都注意到了在线色情爆炸的现象。虽然许多其他同类网站都在疯狂地发布，但成人网站每月收入依旧高达数百万美元。

  <p class="graf graf--p">
   色情创新者做的不仅仅是在线张贴图片。色情作家已经在互联网营销中找到了极好的创新，创造了在网页前“弹出”广告的模式；他们还创建了新的交付机制，安全的信用卡支付和实时视频。正如报道所说的，“

  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   距离曼斯菲尔德在西雅图的豪宅几英里，一位名叫 Danni Ashe 的雄心勃勃的年轻脱衣舞娘在海滩度假期间读了一本关于 HTML 编程的书。她于1995年在网上推出了自己的粉丝网站 Danni’s Hard Drive，作为展示自己宣传照片的地方。然后，Ashe 就找到了一个更有利可图的想法 — — 收取会员费，那在当时仍然是一个新想法。
  </p>
  <p class="graf graf--p">
   她雇佣模特，张贴图片，采访的音频和视频，然后以每月15美元的价格收费 — — 成为了互联网上首批订阅网站之一，另一家是“华尔街日报”。不久之后，Ashe 每年都能赚到250万美元，据报道其使用的带宽比整个中美洲都多。
  </p>
  <p class="graf graf--p">
   随着色情大亨成为了互联网上令人羡慕的赢家，联邦政府很合时宜地改变了态度。1997年6月26日，经过一年多的关于审查互联网的激烈辩论，美国最高法院以违反第一修正案为由废除了“通信规范法”。这是一项具有里程碑意义的决定，保护年轻媒介免受政府监管。
  </p>
  <p class="graf graf--p">
   正如滑稽演员 Dave Barry 在参观了当年的 AdultDex 大会之后所说的，“这个快速增长的数十亿美元的行业无疑将提出更新更好的方式搏得头筹。”无论好坏，在线色情就此延续下来了。
