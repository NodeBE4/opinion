  <p class="graf graf--p">
   一份新的报告显示，世界各地许多政府一直在利用社交媒体来颠覆民主选举、并监视其本国公民。
  </p>
  <p class="graf graf--p">
   最新一版的《网络自由度》报告再一次证实，这个集中化的
   <strong class="markup--strong markup--p-strong">
    互联网不是充当提高透明度和选举公开性的媒介，正相反，是被用来破坏民主进程的工具。
   </strong>
  </p>
  <p class="graf graf--p">
   自由之家负责人 Mike Abramowitz 对记者说：“政府和民粹主义运动正在利用社交媒体大规模地操纵选举，而政府正在利用技术以前所未有的规模监视其本国公民。”
  </p>
  <p class="graf graf--p">
   这可不是新闻。
  </p>

  </p>
  <ul class="postList">
   <li class="graf graf--li">

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%95%b0%e5%ad%97%e5%a8%81%e6%9d%83%e4%b8%bb%e4%b9%89%e7%9a%84%e5%85%b4%e8%b5%b7-%e7%81%be%e5%b9%b4%e4%b9%8b%e7%95%94/" href="https://www.iyouport.org/%e6%95%b0%e5%ad%97%e5%a8%81%e6%9d%83%e4%b8%bb%e4%b9%89%e7%9a%84%e5%85%b4%e8%b5%b7-%e7%81%be%e5%b9%b4%e4%b9%8b%e7%95%94/" rel="noopener noreferrer" target="_blank">

  <p class="graf graf--p">
   下降的主要原因有两个：更多的在线选举干预、和更多的政府监视 —— 两者的共同点是：都在利用社交媒体的力量。
  </p>
  <p class="graf graf--p">
   报告发现，如今，
   <strong class="markup--strong markup--p-strong">
    将近30亿人受到政府和执法机构的监视，部分原因是监视技术变得越来越先进，价格越来越便宜
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Abramowitz 说：“政府正在使用社交媒体收集和分析全体公民的大量个人数据。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “许多政府使用人工智能来识别潜在威胁、并沉默反对派。随着这种监视技术变得越来越便宜，越来越多的执法机构正在使用几乎没有监督或负责的大规模监视。”
  </p>
  <p class="graf graf--p">
   该报告着眼于全球65个国家/地区的互联网自由度调查，涵盖了全球87％的互联网用户。
  </p>
  <p class="graf graf--p">
   中国连续第四年成为世界上最严重的互联网自由威胁者，而冰岛则是最佳的保护者。
  </p>
  <p class="graf graf--p">
   在美国，互联网自由度连续第三年下降，这是由于执法部门对社交媒体的加大力度的监视、以及移民官员对穿越南部边境的所有人的监视。
  </p>
  <p class="graf graf--p">
   去年早些时候发布的文件显示，移民和海关执法局（ICE）
   <a class="markup--anchor markup--p-anchor" data-href="https://www.brennancenter.org/issues/protect-liberty-security/social-media/government-social-media-surveillance" href="https://www.brennancenter.org/issues/protect-liberty-security/social-media/government-social-media-surveillance" rel="noopener noreferrer" target="_blank">
    一直在监控2018年夏季参加纽约市反特朗普抗议活动的所有参与者的社交媒体账户
   </a>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “国土安全部的特工针对的不仅是移民，而且，还有对政府的移民政策和其他政策主题进行和平抗议的所有人，所有人都在遭受社交媒体监控工具的压制，美国人原本受宪法保护的活动正在被审查
   </strong>
   ”，该报告的一位作者 Adrian Shahbaz 告诉记者。
  </p>
  <p class="graf graf--p">
   该报告揭示了另一个令人不安的发展：
   <strong class="markup--strong markup--p-strong">
    民主国家越来越多地使用在线虚假信息
   </strong>
   。虽然在线虚假宣传运动已经存在多年了，但报告显示，它们不再是独裁政权的专利 —— 民主国家也痴迷信息战（这也不是新闻啊）。
  </p>
