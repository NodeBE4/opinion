  <p class="graf graf--p">
   想干点正经事其实并不难，手头的工具大把，只需要合理的搭配。津巴布韦的经历和中国很相似，但相比中国，津巴布韦有诸多的优势，我们在文末具体分析。
  </p>
  <p class="graf graf--p">
   根据 Access Now 的数据，已被记录的互联网关闭数量从2016年的75次飙升至2018年的188次，其中大部分影响了亚洲和非洲的用户。
  </p>
  <p class="graf graf--p">
   最近，津巴布韦当局在燃料成本上涨导致人们走上街头后关闭了互联网 — — 导致对抗议者的镇压、以及该国历史上第一次被观察到互联网完全关闭。在津巴布韦高等法院裁定关闭非法之后，该国互联网服务中断于1月21日结束; 然而，信息、宣传和广播服务部副部长 Energy Mutodi 最近表示，政府会毫不犹豫地再次实施这些措施。
  </p>
  <p class="graf graf--p">
   在看到 VPN 搜索量增加了1500％之后，我们与津巴布韦的合作伙伴进行了交谈，以了解有关如何讨论和分享VPN的更多信息，以及 meme 在推广日常使用中可以发挥的作用。
  </p>
  <p class="graf graf--p">
   这不是津巴布韦第一次故意破坏他们的互联网服务。2016年，当罗伯特穆加贝仍然掌权时，抗议者集体走上街头表达他们对该国经济形势的不满，并遭遇了 WhatsApp 等社交媒体网站的关闭。
   <strong class="markup--strong markup--p-strong">
    由于此前的经验，许多人可以预计到部分互联网中断的状况，但对现政府采取的更为激烈的措施毫无准备
   </strong>
   。根据创新与技术中心和本地化实验室的贡献者之一 Sean Ndlovu 的说法，当政府实施全面的互联网关闭时，他措手不及：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “当停止关闭的呼吁开始时，数字安全领域的一些人开始告诉人们做好准备。但在我们看来，我们认为这将是社交媒体、而不是互联网的彻底关闭。人们并没有真正期望互联网会关闭，因为我记得
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     人们鼓励其他人下载 Psiphon 和其他 VPN 时，每个人都回答说“不，没事的。总统不会因为这个问题搞乱他自己的名声
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    ，“你知道，一切都会按照计划进行，但是，唉，他们确实关闭了互联网。
   </em>
   ”
  </p>
  <p class="graf graf--p">
   对于斯特拉斯莫尔大学的高级研究员 Arthur Gwagwa 来说，意外情绪是一样的：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “总统一直在给国际社会一个面子，让他们认为他不会做这样的事 — — 所以我们从未见过这种状况。即使在2016年的抗议活动期间，互联网也并没有像现在这样完全关闭。
   </em>
   “
  </p>
  <p class="graf graf--p">
   互联网关闭被定义为“故意破坏互联网或电子通信，使其无法访问或无法用于特定人群或某个地点，通常是为了控制信息流。” — — 这在中国非常常见。
  </p>
  <p class="graf graf--p">
   这可能意味着，互联网限制（或故意放慢互联网带宽以使其基本无法使用）、部分互联网关闭（如 阻止特定社交媒体网站）、以及完全互联网关闭（也称为 blackouts ）。根据 OONI Probe 的一份报告，在津巴布韦的情况下，政府实施了所有这些策略 — — 从限制开始，然后继续阻止 Twitter 和 WhatsApp 等社交媒体网站，直到他们最终完全关闭互联网。
  </p>
  <figure class="wp-block-image">
   <img alt="" class="wp-image-3230 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-218.png?fit=1024%2C827&amp;ssl=1&amp;is-pending-load=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-218.png?w=1556&amp;ssl=1 1556w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-218.png?resize=300%2C242&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-218.png?resize=768%2C620&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-218.png?resize=1024%2C827&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-218.png?resize=1100%2C888&amp;ssl=1 1100w" height="1256" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-218.png?fit=1024%2C827&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1556"/>

  <p class="graf graf--p">
   从1月15日开始到1月21日，津巴布韦经历了一些形式的访问中断，最严重的断网措施表明在16日和18日的部分时段发生。据 Gwagwa 说：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “互联网阻塞是逐渐发生的。社交媒体和应用程序关闭然后又打开了。在开放期间里，人们可以与居住在国外的其他活动家和津巴布韦人交流。由于试图保护自己的国际声誉，该国政府在完全关闭互联网时犹豫不决，开放和关闭交替发生。这个窗口期让津巴布韦人有机会相互交谈，了解到底发生了什么。
   </em>
   “
  </p>
  <p class="graf graf--p">
   然而，尽管津巴布韦人可以在互联网开放的窗口期内分享信息，但关于国内发生了什么事的错误信息也正在努力传播，最值得一提的是副部长 Mutodi 的虚假宣传：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “一般用户不了解一开始的情况。新闻部副部长在国家电视台上说，互联网很拥挤，这就是为什么它速度很慢。人们开始分享这个信息……所以现在你会看到有两个叙述。 [政府]正在拼命兜售他们自己的宣传。
   </em>
   “
  </p>
  <p class="graf graf--p">
   数字安全培训师和 LocLab Shona 语言协调员 Chido Musodza 认为，在其中一家主要互联网服务提供商向其客户发出信息并承认他们阻止互联网后，大多数人都不可能相信政府的言论：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “我们收到了来自 Econet 的短信，告诉我们他们接到了政府要求关闭互联网的指示。在我们听到国家电视台信息部长发来的信息说互联网只是拥堵时我们是不会相信的”
   </em>
   。
  </p>
  <p class="graf graf--p">
   根据 BestVPN.com的数据，津巴布韦的 VPN 搜索在最近的关闭期间飙升了1,500％。这与在推特和其他社交媒体网站上可信来源倡导使用 VPN 的民间社会组织的兴起同时发生：
  </p>
  <figure class="wp-block-image">
   <img alt="" class="wp-image-3231 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-219.png?fit=1024%2C830&amp;ssl=1&amp;is-pending-load=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-219.png?w=1488&amp;ssl=1 1488w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-219.png?resize=300%2C243&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-219.png?resize=768%2C622&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-219.png?resize=1024%2C830&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-219.png?resize=1100%2C892&amp;ssl=1 1100w" height="1206" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-219.png?fit=1024%2C830&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1488"/>

    <img alt="" class="wp-image-3231" height="1206" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-219.png?fit=1024%2C830&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-219.png?w=1488&amp;ssl=1 1488w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-219.png?resize=300%2C243&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-219.png?resize=768%2C622&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-219.png?resize=1024%2C830&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-219.png?resize=1100%2C892&amp;ssl=1 1100w" width="1488"/>

  <p class="graf graf--p">
   在津巴布韦内外，人们开始通过社交媒体向用户发送 VPN 建议。关闭期间 VPN 的普及可以通过普遍数量的推文、meme 和消息看到：
  </p>
  <p class="graf graf--p">
   虽然在互联网关闭期间人们对 VPN 的兴趣在增加，但由于与该国互联网基础设施相关的现有限制，使用 VPN 会带来不同的问题。
  </p>
  <p class="graf graf--p">
   津巴布韦的宽带速度在200个国家中排名125，其用户支付全非洲最高价格 — — 使得下载既昂贵又缓慢。作为回应，许多用户转向了流行的文件共享应用程序 SHAREit，它通过Wi-Fi直接工作。
  </p>
  <p class="graf graf--p">
   当人们在互联网关闭期间无法下载新应用时，那些已经在手机上拥有 VPN 的用户可以使用 SHAREit 将他们的 VPN 发送给其他社区成员。 Gwagwa 是一个利用这种方法的用户：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     我们是一个共同的国家 — — 我们总是相互沟通。这里的每个人都很亲密。当断网发生时，我就到附近去把我信任的社区 VPN 分享给人们
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    。
   </em>
   “
  </p>
  <p class="graf graf--p">
   然而，尽管人们试图推动 VPN 的使用，但似乎普遍缺乏对 VPN 如何工作以及与 VPN 相关危险的错误信息的理解，导致在断网的最关键时刻出现混乱。据 Ndlovu 说：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “人们通过社交媒体倡导 VPN。首先是 Twitter，然后是 Facebook，然后是 WhatsApp。但是当谈到 VPN 时出现了一个问题 — — 大多数人只是选择他们在 Playstore 或 App Store 中找到的任何 VPN。因此，这成为了一项挑战，因为数字安全领域的人们正在推动像 Psiphon 或 Tunnelbear 这样的可靠 VPN，但后来有人说他们正在使用完全不同的其他不安全 VPN。在整个互联网关闭后，人们不明白，即使您有VPN也没用，因为您无法访问外部世界。所以再次，人们完全感到困惑，因为他们一直以为使用 VPN 是万能的，因此存在一些基本误解。
   </em>
   “
  </p>
  <p class="graf graf--p">
   不幸的是，
   <strong class="markup--strong markup--p-strong">
    这些误解会导致真正的数字安全问题 — — 特别是在从不值得信任的站点下载 VPN 时。
   </strong>
   对于伪装成数字安全应用程序的骗局来说，它们一直在吸引毫无戒心的用户放弃个人隐私权。在像津巴布韦这样的环境中，
   <strong class="markup--strong markup--p-strong">
    当人们迫切想要重新上线时，下载不安全 VPN 的风险要高得多
   </strong>
   。 Musodza 的一些联系人就是这种情况：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “在某些情况下，我们意识到人们手机上最多有21个VPN。而且没有一个是应该被推荐的。我们一直在努力推动像 Psiphon 和 TunnelBear 这样值得信赖的VPN。但人们正在下载各种各样的不安全的东西“
   </em>
   。
  </p>
  <p class="graf graf--p">
   根据 Ndlovu 的说法，这一状况发生的原因在于数字安全教育通常不容易获得：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “我们需要写文章。我们需要一个广告系列来告诉人们 VPN 是什么。
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     当您尝试解释 VPN 时，无论您绘制多少个图表，人们都不会太了解。但是现在，在这种简单有趣的背景下，人们开始明白了。
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    但是，我们需要更多地了解 VPN 的优势，它是什么以及如何使用它。我们还需要人们知道，这不仅仅是为了规避政府的限制，而且也是为了您自己的隐私。这是我们需要向人们传播的信息。
   </em>
   “
  </p>
  <p class="graf graf--p">
   Ndlovu 认为这些消息应该包括让 VPN 成为津巴布韦日常生活的一部分的重要性，而克服的第一个障碍是
   <strong class="markup--strong markup--p-strong">
    确保人们在断网结束后不要删除他们的 VPN
   </strong>
   。
  </p>
  <p class="graf graf--p">
   手机内存通常是津巴布韦人关注的重点问题，由于空间有限，人们需要从手机中删除经常不使用的应用程序。 Gwagwa 正在推动数字安全培训师和互联网研究人员
   <strong class="markup--strong markup--p-strong">
    利用当前的形势，提高人们对 VPN 使用的认识
   </strong>
   ：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “我们需要利用该国目前的情绪。现在，它不是一个选择。 VPN 应该是人们在屏幕上看到的第一个东西。有些人会因为手机空间而删除 VPN。我们需要更多地传播这些知识
   </em>
   。“
  </p>
  <p class="graf graf--p">
   对于 Gwagwa 来说，解决方案的一部分可能依赖于社交媒体 — — 更具体地说是笑话和meme。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “我们需要让工具更容易获取。让它们更容易被理解。
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     由于笑话和 meme 一直在帮助人们提高对 VPN 的认识。他们可以通过将其简化为更巧妙的语言来提供帮助。津巴布韦人有很强的幽默感，即使在逆境中，我们也喜欢笑
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    。
   </em>
   “
  </p>
  <figure class="wp-block-image">
   <img alt="" class="wp-image-3232 jetpack-lazy-image" data-lazy-sizes="(max-width: 1034px) 100vw, 1034px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-220.png?fit=854%2C1024&amp;ssl=1&amp;is-pending-load=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-220.png?w=1034&amp;ssl=1 1034w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-220.png?resize=250%2C300&amp;ssl=1 250w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-220.png?resize=768%2C921&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-220.png?resize=854%2C1024&amp;ssl=1 854w" height="1240" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-220.png?fit=854%2C1024&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1034"/>
   <noscript>
    <img alt="" class="wp-image-3232" height="1240" sizes="(max-width: 1034px) 100vw, 1034px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-220.png?fit=854%2C1024&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-220.png?w=1034&amp;ssl=1 1034w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-220.png?resize=250%2C300&amp;ssl=1 250w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-220.png?resize=768%2C921&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-220.png?resize=854%2C1024&amp;ssl=1 854w" width="1034"/>
   </noscript>
  </figure>
  <p class="graf graf--p">
   根据技术专家和 Memes to Movements 的作者 An Xiao Mina 的说法，利用 meme 和笑话的力量传播有关 VPN 使用的信息是让人们对这项技术感兴趣的可行策略：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “meme 是传播信息的关键载体 — — 无论是好还是坏 — — 通过数字媒体。因此，通过 meme 传播信息是有道理的，最好的方法是创建引人入胜的幽默内容，引起人们的注意，并希望能够借此推动人们采用良好的 VPN。
  </p>
  <p class="graf graf--p">
   当然，
   <strong class="markup--strong markup--p-strong">
    风险是 meme 也可能传播有关 VPN 的错误信息。因此，最有效的 meme 方法将 meme 的幽默感和参与其他形式的教育、以及关于正确使用 VPN 的知识相结合
   </strong>
   。“
  </p>
  <p class="graf graf--p">
   如果以正确的方式完成，幽默可能只是让更多人下载和使用安全的 VPN 应用程序的关键。但正如我们在津巴布韦互联网关闭期间看到的那样，当政府决定关闭互联网时，这仍然无济于事。在这些情况下，Ndlovu 说我们可能需要向互联网自由社区寻求帮助：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “我们需要一个安全指南，
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     提供离线解决方案
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    ，以确保我们可以信任和使用关于互联网关闭的调查，这样人们就不会错过他们能在互联网上听到的任何工具。我认为我们将来会有更多的抗议活动，这意味着[在津巴布韦]更多关闭互联网的可能性很高，我们现在需要这些解决方案。
   </em>
   “
  </p>
  <p class="graf graf--p">
   更多详见 IYP 去年的报告《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E5%A6%82%E4%BD%95%E7%94%A8%E6%8C%87%E6%A0%87%E9%A2%84%E6%B5%8B%E4%BA%92%E8%81%94%E7%BD%91%E4%B8%AD%E6%96%AD-%E4%B8%80%E6%97%A6%E6%96%AD%E7%BD%91%E8%BF%98%E5%8F%AF%E4%BB%A5%E6%80%8E%E4%B9%88%E5%8A%9E-d6ecac8eb93c" href="https://medium.com/@iyouport/%E5%A6%82%E4%BD%95%E7%94%A8%E6%8C%87%E6%A0%87%E9%A2%84%E6%B5%8B%E4%BA%92%E8%81%94%E7%BD%91%E4%B8%AD%E6%96%AD-%E4%B8%80%E6%97%A6%E6%96%AD%E7%BD%91%E8%BF%98%E5%8F%AF%E4%BB%A5%E6%80%8E%E4%B9%88%E5%8A%9E-d6ecac8eb93c" rel="noopener noreferrer" target="_blank">
    如何用指标预测互联网中断？一旦断网还可以怎么办？
   </a>
   》
  </p>
  <p class="graf graf--p">
   得到一些线索：1、关于没有预测到严重性 — — 误认为政治变革（改朝换代）可以改变互联网管制的状况。这在中国极有可能出现，需要提起警惕。对任何政权保持警惕是作为公民的基本要务。
  </p>
  <p class="graf graf--p">
   2、政府使用互联网管制的机会兜售当权者的政治宣传，这在中国也很常见，当信息匮乏的时候，大众很容易极力捕捉信息而不区分来源，平日对政治宣传的警惕性会明显弱化。这种时候必须有强大的独立媒体和民间组织后盾来传播真相，而独立组织的知名度和信誉需要长期不懈的积累。
  </p>
  <p class="graf graf--p">
   3、在互联网关闭期间无法下载新应用时所使用的方法也非常值得中国的借鉴。
  </p>
  <p class="graf graf--p">
   4、津巴布韦有连接强度较大的社会关系，而中国大部分地区没有这样的社会连接。要想在中国这种互信度低下的社会保持传输通畅，就需要一些覆盖面广泛的社区组织。
  </p>
  <p class="graf graf--p">
   5、缺乏对技术的理解这一问题在中国也非常普遍。虽然中国有GFW，但人们依旧认为只要能翻墙就是“好的”VPN。事实上完全不是。如果告知人们关于这一问题，而不需要从技术层面上解释以产生门槛感，津巴布韦的经验值得学习。
  </p>
  <p class="graf graf--p">
   6、对 VPN 的理解错误导致很多人误认为它只是一个翻墙工具，于是出国的人会感叹“终于不再需要 VPN了”，这是严重错误的。只要您还在这个互联网上，就必须时刻保持自己的隐私状态，VPN 应该是您开机时第一个看到的东西。
  </p>
  <p class="graf graf--p">
   7、津巴布韦的公民宣传有效地借助了社会情绪，这点也值得中国的倡导者学习。不仅是如何借助的问题，更有如此保持住这份情绪状态的问题。⚪️
  </p>
