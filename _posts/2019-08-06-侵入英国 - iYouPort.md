       英国议会以及全国各地的警察、医院、学校和大学，布满了中国海康威视的产品，该公司是如何侵入英国的？
      </strong>

  <p>

  <p class="graf graf--p">
   这是一家涉嫌侵犯人权的中国国有企业。但这并没有阻止海康威视在英国获得一个主要的立足点。根据消息来源和采购记录，这家总部位于中国杭州的监视技术公司
   <strong class="markup--strong markup--p-strong">
    通过一个企业合作伙伴网络，为英国议会以及全国各地的警察、医院、学校和大学提供监控系统
   </strong>
   。
  </p>
  <p class="graf graf--p">
   海康威视的技术是美国政府最近禁止联邦机构采购的，但丝毫没有影响这家年收入高达数百万美元的老大哥公司将其技术出售给英国警方、政府和组织。
  </p>
  <p class="graf graf--p">
   与此同时，该公司一直在帮助在中国政府在新疆地区建立一个压迫性的*监视国家*，维吾尔族少数民族被关押在秘密的拘留营里。
  </p>
  <p class="graf graf--p">
   一些英国政客正在提出对这项技术的担忧 — — 并且正在呼吁对进一步购买该技术实施禁运 — — 理由是海康威视是侵犯人权的同谋，也是国家安全风险，因为担心中国情报机构可能会获取英国敏感位置的监控数据。
   <strong class="markup--strong markup--p-strong">
    该公司的一些监控摄像机会记录音频并连接到互联网，这意味着它们可以在世界上任何地方进行监控
   </strong>
   。
  </p>
  <p class="graf graf--p">
   根据英国议会第二议院议长 Alan West 勋爵的说法，今年1月，这些摄像机计划安装在伦敦的 Portcullis House 内。Portcullis House 是威斯敏斯特的一座办公大楼，这里由200多名议员和400名员工用于日常工作，这些工作通常涉及国家安全、经济和外交政策等敏感问题的讨论。
  </p>
  <p class="graf graf--p">
   West 告诉 The Intercept，一些“担心这种情况正在发生”的人给了他一份合同，合同上显示该建筑物将配备的是海康威视的监控设备。他说，他随后向议会当局抱怨此事。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “在我看来这件事非常令人担心 — — 这更像是能让 Mata Hari 进入每个办公室，”他说，指的是荷兰异国情调的舞者，被指控在第一次世界大战期间为德国进行间谍活动。“我们是否确定对中国的监视摄像机在议会办公室里感到满意？让他们听取我们的政府部门员工对选民说的话？窃听部长说的话？在他们的办公桌上拍摄机密文件？“
  </p>
  <p class="graf graf--p">
   议会发言人否认存在涉及海康威视的合同，并表示没有计划“今年在 Portcullis House 安装任何额外的摄像头”。
  </p>
  <p class="graf graf--p">
   然而，熟悉议会大楼部分监控系统的消息来源表示，
   <strong class="markup--strong markup--p-strong">
    除了 Portcullis House 外，还包括威斯敏斯特宫、诺曼肖建筑和大本钟，海康威视的设备过去“绝对”在那里使用过
   </strong>
   。
  </p>
  <p class="graf graf--p">
   该消息人士表示，他们无法确认海康威视的监控相机目前是否处于活动状态，因为该地区所有议会和政府大楼内及周围都安装了数百台摄像机。
  </p>
  <p class="graf graf--p">
   据以前的估计，在整个英国，有超过120万台海康威视的监控相机。The Intercept 审查的采购记录和政府合同表明，
   <strong class="markup--strong markup--p-strong">
    该公司 — — 截至2018年6月，由中国独裁的共产党政权拥有40％的股权 — — 已向全国各地的各种组织和公司提供监控系统
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这些摄像机已广泛安装在伦敦，包括哈克尼、肯辛顿、切尔西、哈默史密斯和富勒姆；并且已被吉尔福德、南凯斯特文、瑟罗克、斯托克顿、北泰恩赛德、阿伯丁郡、福尔柯克、西萨福克和肯特郡的当地政府当局采购。
  </p>
  <p class="graf graf--p">
   去年威尔士警方开始将中国的监控摄像机安装在17个城镇。
   <strong class="markup--strong markup--p-strong">
    在北爱尔兰，海康威视的监控设备已安装在300多辆公交车内！
   </strong>
   并且，这些摄像机已安装在汉普郡、兰开夏郡、肯特郡、北安普顿、康沃尔郡、坎布里亚郡和约克郡的医院内；还有被安装在萨里、德文郡、伯明翰和普利茅斯的学校里。这些监视相机也已经部署在商店里：位于巴斯的 Southgate 购物中心、伦敦的 Gallions Reach 购物中心，以及 Tesco 超市和汉堡王快餐店，全有。
  </p>
  <p class="graf graf--p">
   海康威视的营销材料表示，它的相机可以
   <strong class="markup--strong markup--p-strong">
    与面部识别软件一起使用，并链接到一个集中的照片数据库里
   </strong>
   。该公司声称，该技术可以区分已知面孔和陌生人，并在未知人员进入建筑物或办公室时触发警报。它表示，其企业使命是“共同努力，加强安全，推动全球可持续发展。”
  </p>
  <p class="graf graf--p">
   在中国，海康威视一直在帮助政府实施一个名为“天网”的全国性监控网络。近年来，这项努力一直专注于新疆地区，共产党政府正在以“打击恐怖主义”为借口，对穆斯林少数民族维吾尔人进行镇压。
  </p>
  <p class="graf graf--p">
   在新疆，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.reuters.com/article/us-china-rights-un-idUSKBN1KV1SU" href="https://www.reuters.com/article/us-china-rights-un-idUSKBN1KV1SU" rel="noopener noreferrer" target="_blank">
    估计有
   </a>
   100万维吾尔人 — — 包括儿童，孕妇，老人和残疾人 — — 被关押在拘留营中。在这些神秘的设施中，维吾尔人被迫接受“再教育”过程，包括强制性收看共产党政治歌曲和演讲。据说那些抵抗者会面临惩罚，例如殴打和单独监禁。
  </p>
  <p class="graf graf--p">
   根据人权观察组织的报告，中国当局“几十年内在新疆实施的人权侵犯行为已达到一定程度”。该组织在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hrw.org/report/2018/09/09/eradicating-ideological-viruses/chinas-campaign-repression-against-xinjiangs" href="https://www.hrw.org/report/2018/09/09/eradicating-ideological-viruses/chinas-campaign-repression-against-xinjiangs" rel="noopener noreferrer" target="_blank">
    2018年的一份报告
   </a>
   中说，对新疆维吾尔人的镇压最令人不安的一个方面就是大规模监视系统。
  </p>
  <p class="graf graf--p">
   报告说：“新疆当局对声音样本和 DNA 等生物识别数据进行强制性大规模收集，并使用人工智能和大数据来识别、剖析和跟踪每一个人。” 更多相关信息（不仅是中国的生物识别监控方法和解析）详见
   <a class="markup--anchor markup--p-anchor" data-href="https://docs.google.com/document/d/1tmU2VmsdHRo79NniIM85yWJrS-A_GIhku_0yXZOg8Fk/edit?usp=sharing" href="https://docs.google.com/document/d/1tmU2VmsdHRo79NniIM85yWJrS-A_GIhku_0yXZOg8Fk/edit?usp=sharing" rel="noopener noreferrer" target="_blank">
    IYP 分类知识列表-2 中的“生物识别和对抗方法”
   </a>
   部分。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “当局已将这些系统设想为一系列’过滤器’，挑选出具有某种行为特征的人，他们认为这些行为或特征对共产党的统治构成了威胁。”
  </p>
  <p class="graf graf--p">
   至少自2010年以来，海康威视一直在帮助在新疆建立庞大的监控网络，警方正在利用这些摄像机监视少数民族。 2013年，海康威视的公共安全经理吹嘘该公司的技术*使​​安全部队能够跟踪和描述任何人*。他说：“我们可以通过查看某个人来自哪个家庭来帮助保持稳定，说服他们的亲属阻止他们的有害行为，就像法轮功那种行为，”这是一个被禁止的精神团体。
  </p>
  <p class="graf graf--p">
   随着中国加大对新疆的打击力度，海康威视已经获得了丰厚的财务回报。
  </p>
  <p class="graf graf--p">
   据报道，
   <strong class="markup--strong markup--p-strong">
    该公司在该地区拥有超过
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://ipvm.com/reports/xinjiang-dahua-hikvision" href="https://ipvm.com/reports/xinjiang-dahua-hikvision" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     10亿美元的业务份额
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，仅2017年就有5份合同，价值约2.77亿美元。这些合同包括向地方政府机构提供监视系统的交易
   </strong>
   ，用于拘留营、街道以及清真寺、学校和办公室内。
  </p>
  <p class="graf graf--p">
   海康威视拒绝评论这份报告。该公司过去曾试图淡化与中国政权的关系，将自己描绘成一家“独立的”公司。然而，该公司自己的财务记录显示其控股股东是中国政府所有的实体，称为中国电子科技集团。
  </p>
  <p class="graf graf--p">
   根据视频监控行业出版物
   <a class="markup--anchor markup--p-anchor" data-href="https://ipvm.com/reports/sasac-hik-xi" href="https://ipvm.com/reports/sasac-hik-xi" rel="noopener noreferrer" target="_blank">
    IPVM 翻译的政府新闻稿
   </a>
   ，2018年9月，中国政府官员 Weng Jieming 宣布领导层“被纳入海康威视公司的治理结构”。Weng 称赞该公司，称其“坚决贯彻执行国家主席习近平的重要指示精神”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在英国，海康威视不直接向其客户提供监控相机；相反，它通过批发商和分包商网络销售其监视设备
   </strong>
   。该公司最新的英国财务记录显示，
   <strong class="markup--strong markup--p-strong">
    从2017年开始，年度总利润为262万美元，营业额为655万美元。根据其宣传材料，该公司的全球销售总收入高达66.5亿美元。
   </strong>
  </p>
  <p class="graf graf--p">
   海康威视在英国设有三个办事处，去年该公司还宣布计划在伦敦希思罗机场附近的英国总部内建立一个新的研发中心。该研发部门由 Pu Shiliang 领导，据报道，此人
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wsj.com/articles/surveillance-cameras-made-by-china-are-hanging-all-over-the-u-s-1510513949" href="https://www.wsj.com/articles/surveillance-cameras-made-by-china-are-hanging-all-over-the-u-s-1510513949" rel="noopener noreferrer" target="_blank">
    还曾在政府公安部工作
   </a>
   ，该部门是一个专门以活动家和政治对手为目标的间谍组织。
  </p>
  <p class="graf graf--p">
   对于任何从事安全行业的公司来说，英国是一个很有吸引力的前景。
   <strong class="markup--strong markup--p-strong">
    英国是世界上使用监视技术最多的国家之一，在整个城镇和城市中，估计有600万个摄像头，平均每11个人一个摄像头。
   </strong>
  </p>
  <p class="graf graf--p">
   海康威视通过大幅削减其欧洲竞争对手，成功进入了利润丰厚的英国市场。根据政府采购文件，可以以1,000英镑（1,310美元）购买基本的海康威视监控系统；相比之下，德国博世制造的类似规格的系统成本为3,000英镑（3,930美元）。
  </p>
  <p class="graf graf--p">
   部分英国政客对中国政府参与该国的关键基础设施表示担忧。去年12月，国防部长 Gavin Williamson
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bbc.com/news/uk-46690627" href="https://www.bbc.com/news/uk-46690627" rel="noopener noreferrer" target="_blank">
    表示
   </a>
   ，他将密切关注中国公司华为将英国移动网络从4G升级到5G的角色。然而，海康威视在英国不断增长的存在并没有引起同样的审查。
  </p>
  <p class="graf graf--p">
   在美国，海康威视没有这么轻松的旅程。去年8月，“国防授权法”增加了一项修正案，禁止美国军方和政府购买海康威视的技术。撰写该修正案的众议员 Vicky Hartzler 表示，中国政府试图通过扩大中国公司在美国国内通信和公共安全领域的作用来“瞄准美国”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “中国公司销售的视频监控和安全设备使美国政府面临重大漏洞，”她说，“我起草的修正案将确保中国不能在联邦机构内建立视频监控网络。” 该禁令
   <a class="markup--anchor markup--p-anchor" data-href="https://ipvm.com/reports/ban-law" href="https://ipvm.com/reports/ban-law" rel="noopener noreferrer" target="_blank">
    最终签署
   </a>
   为法律。
  </p>
  <p class="graf graf--p">
   英国工党议员 Karen Lee 告诉 The Intercept，她敦促英国政府考虑抵制海康威视产品，特别是用于公有建筑的情况。“当外国政治进程中的数字干扰越来越多地被用来破坏其他国家的稳定时，我们必须警惕海康威视或任何外国公司可能对英国国家安全造成的任何风险，”Lee 说。
  </p>
  <p class="graf graf--p">
