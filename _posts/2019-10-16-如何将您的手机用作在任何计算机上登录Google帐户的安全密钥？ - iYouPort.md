  <p class="graf graf--p">
   许多在线帐户现在都增加了双因素身份验证（2FA）功能，以帮助确保我们的数据安全。
  </p>
  <p class="graf graf--p">
   这实际上意味着在满足一组特定要求之前，任何人都无法访问该帐户。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    短信验证肯定是不安全的
   </strong>
   。在这里看到为什么《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e4%bd%bf%e7%94%a8%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b%e6%9d%a5%e6%94%bb%e5%87%bb-2fa%ef%bc%9f//" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e4%bd%bf%e7%94%a8%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b%e6%9d%a5%e6%94%bb%e5%87%bb-2fa%ef%bc%9f//" rel="noopener noreferrer" target="_blank">
    如何使用社交工程来攻击 2FA？
   </a>
   》而且还容易被骗，比如《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/facebook-%E4%BB%A5%E5%AE%89%E5%85%A8%E6%80%A7%E4%B8%BA%E5%B9%8C%E5%AD%90%E9%AA%97%E5%8F%96%E4%BD%A0%E7%9A%84%E6%89%8B%E6%9C%BA%E5%8F%B7%E7%A0%81/" href="https://www.iyouport.org/facebook-%E4%BB%A5%E5%AE%89%E5%85%A8%E6%80%A7%E4%B8%BA%E5%B9%8C%E5%AD%90%E9%AA%97%E5%8F%96%E4%BD%A0%E7%9A%84%E6%89%8B%E6%9C%BA%E5%8F%B7%E7%A0%81/" rel="noopener noreferrer" target="_blank">
    Facebook 以”安全性”为幌子骗取你的手机号码
   </a>
   》。
  </p>
  <p class="graf graf--p">
   手机号码本身就很危险，在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e5%8f%aa%e7%9f%a5%e9%81%93%e4%b8%80%e4%b8%aa%e7%94%b5%e8%af%9d%e5%8f%b7%e7%a0%81%ef%bc%8c%e4%bd%a0%e8%83%bd%e6%8c%96%e5%87%ba%e5%a4%9a%e5%b0%91%e6%9c%89%e6%95%88%e4%bf%a1%e6%81%af/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e5%8f%aa%e7%9f%a5%e9%81%93%e4%b8%80%e4%b8%aa%e7%94%b5%e8%af%9d%e5%8f%b7%e7%a0%81%ef%bc%8c%e4%bd%a0%e8%83%bd%e6%8c%96%e5%87%ba%e5%a4%9a%e5%b0%91%e6%9c%89%e6%95%88%e4%bf%a1%e6%81%af/" rel="noopener noreferrer" target="_blank">
    如果只知道一个电话号码，你能挖出多少有效信息
   </a>
   ？》。
  </p>
  <p class="graf graf--p">
   这就是为什么推荐物理密钥。
  </p>
  <p class="graf graf--p">
   但是物理密钥你需要买。
  </p>
  <p class="graf graf--p">
   现在，不花钱购买额外设备也可以做到。
  </p>
  <p class="graf graf--p">
   用手机来代替。因为如今几乎没有人不使用智能手机，如果能把智能手机变成物理密钥的话就能省下额外的开销。
  </p>
  <p class="graf graf--p">
   具体来说，大多数2FA系统在您登录之前需要两件事的组合：您知道的东西、和您拥有的东西。
  </p>
  <p class="graf graf--p">
   您的密码是“您知道的东西”，Google 现在拥有一个系统，当您在计算机上登录 Google 服务时，您的手机可以作为“您拥有的东西”。
  </p>
  <p class="graf graf--p">
   设置完成后，除非潜在的攻击者除了知道您的密码外、还可以直接实际访问您的手机，否则他们将无法登录。
  </p>
  <p class="graf graf--p">
   您现在可以使用任何 Android 手机来保护您的 Google 帐户，使用他们所谓的两步验证（2SV）系统。
  </p>
  <p class="graf graf--p">
   这可以被认为是如上所述的2FA方法的一种形式，因为它需要两个单独的因素。
  </p>
  <p class="graf graf--p">
   新增的2SV功能意味着您不再需要随身携带单独的物理密钥来保护您的帐户。
  </p>
  <p class="graf graf--p">
   基本要求 ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <a class="markup--anchor markup--li-anchor" data-href="https://android.gadgethacks.com/how-to/android-basics-tell-what-android-version-build-number-you-have-0168050/" href="https://android.gadgethacks.com/how-to/android-basics-tell-what-android-version-build-number-you-have-0168050/" rel="noopener noreferrer" target="_blank">
     Android 7.0 Nougat or higher
    </a>
   </li>
   <li class="graf graf--li">
    Windows, Mac, or Chrome OS computer
   </li>
   <li class="graf graf--li">
    <a class="markup--anchor markup--li-anchor" data-href="https://www.google.com/chrome/" href="https://www.google.com/chrome/" rel="nofollow noopener noreferrer" target="_blank">
     Google Chrome installed
    </a>
    on phone and computer
   </li>
  </ul>
  <p class="graf graf--p">
   下面说说怎么做。
  </p>
  <p class="graf graf--p">
   第一步：在您的计算机上启用蓝牙
  </p>
  <p class="graf graf--p">
   首先，您需要在计算机上启用蓝牙。
  </p>
  <p class="graf graf--p">
   如果您的计算机没有内置蓝牙功能，您只需
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amazon.com.au/s/?tag=wghau-22&amp;k=bluetooth+dongle" href="https://www.amazon.com.au/s/?tag=wghau-22&amp;k=bluetooth+dongle" rel="noopener noreferrer" target="_blank">
    8美元即可从亚马逊购买蓝牙适配器
   </a>
   ，然后只需将其插入计算机的USB端口即可添加蓝牙功能。
  </p>
  <p class="graf graf--p">
   要在 Windows 10 PC 上启用蓝牙，请转到“开始”菜单（单击左下角的 Windows 图标），然后单击“设置”齿轮。
  </p>
  <p class="graf graf--p">
   接下来，选择“设备”，然后确保“蓝牙”下面的切换开关打开。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5771 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-6.jpg?resize=1100%2C715&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-6.jpg?w=1456&amp;ssl=1 1456w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-6.jpg?resize=300%2C195&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-6.jpg?resize=768%2C500&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-6.jpg?resize=1024%2C666&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-6.jpg?resize=1100%2C715&amp;ssl=1 1100w" data-recalc-dims="1" height="715" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-6.jpg?resize=1100%2C715&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5771" data-recalc-dims="1" height="715" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-6.jpg?resize=1100%2C715&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-6.jpg?w=1456&amp;ssl=1 1456w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-6.jpg?resize=300%2C195&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-6.jpg?resize=768%2C500&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-6.jpg?resize=1024%2C666&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-6.jpg?resize=1100%2C715&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   对于 Windows 8 用户，将鼠标指针移动到右下角，将开始显示几个图标（“Charm Bar”）。
  </p>
  <p class="graf graf--p">
   向上移动鼠标以显示 Charms，然后单击“设置”。
  </p>
  <p class="graf graf--p">
   从此处转到更改PC设置&gt;无线&gt;无线设备，然后启用蓝牙切换。
  </p>
  <p class="graf graf--p">
   如果您使用的是Mac，请单击任意屏幕左上角的Apple图标，然后选择“系统偏好设置”。
  </p>
  <p class="graf graf--p">
   从那里打开“蓝牙”菜单，然后单击标有“打开蓝牙”的按钮。
  </p>
  <p class="graf graf--p">
   如果此按钮变为显示“关闭蓝牙”，则蓝牙已启用，您无需更改任何内容。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5772 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-4.jpg?resize=1100%2C709&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-4.jpg?w=1322&amp;ssl=1 1322w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-4.jpg?resize=300%2C193&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-4.jpg?resize=768%2C495&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-4.jpg?resize=1024%2C660&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-4.jpg?resize=1100%2C709&amp;ssl=1 1100w" data-recalc-dims="1" height="709" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-4.jpg?resize=1100%2C709&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   对于 ChromeOS 用户，只需点击右下角的时间，然后选择“蓝牙”即可。
  </p>
  <p class="graf graf--p">
   第二步：在手机上启用蓝牙和位置
  </p>
  <p class="graf graf--p">
   现在，从手机屏幕顶部向下滑动以展开“快速设置”菜单，然后点击蓝牙切换以启用此功能。
  </p>
  <p class="graf graf--p">
   或者，您可以在手机的主设置应用中搜索“蓝牙”一词，然后从那里打开开关。
  </p>
  <p class="graf graf--p">
   接下来，转到手机的主设置应用，然后打开“位置”菜单。从那里，启用页面顶部的切换以确保打开位置服务。
  </p>
  <p class="graf graf--p">
   这是确保您的手机在登录时物理上靠近计算机所必需的。
  </p>

   <strong class="markup--strong markup--p-strong">
    注意：蓝牙和位置不要一直处于开启状态，就在您登录帐户时开一瞬间！
   </strong>

  <p class="graf graf--p">
   如果您还没有这样做或者想要在特定设备上保护多个帐户，可以通过转到此菜单轻松添加它们：
  </p>
  <pre class="graf graf--pre"><code>Settings &gt; Accounts &gt; Add account &gt; Google</code></pre>
  <p class="graf graf--p">
   您可以在上面的位置找到向设备添加新帐户所需的一切。
  </p>
  <p class="graf graf--p">
   按照屏幕上的说明使用一个或多个Google帐户登录。
  </p>
  <p class="graf graf--p">

   <strong class="markup--strong markup--p-strong">
    潜在的问题和疑虑
   </strong>
  </p>
  <p class="graf graf--p">
   1、这适用于您的所有在线帐户还是仅适用于您的Google帐户？
  </p>
  <p class="graf graf--p">
   目前，这仅适用于您的Google帐户或访问任何Google服务，例如YouTube或Google Play。
  </p>
  <p class="graf graf--p">
   但应该不会停留在此处。如果能扩展的话是相当方便的。
  </p>
  <p class="graf graf--p">
   2、这是否适用于除谷歌浏览器之外的其他网络浏览器？
  </p>
  <p class="graf graf--p">
   至少目前还不行。谷歌Chrome是目前唯一支持此功能的浏览器，但随着时间的推移，相信其他流行的浏览器也应该可以做到。
  </p>
  <p class="graf graf--p">
   3、如果您无法通过手机验证连接以允许您登录，该怎么办？
  </p>
  <p class="graf graf--p">
   它可能并不完全适用于所有用户，因为使用手机作为安全密钥的新方法仍处于测试阶段，但预计手机上的 Google Play 服务会更新，以便尽快解决此问题。
  </p>
  <p class="graf graf--p">
   另外，请确保手机上已启用位置和蓝牙，并且只能在计算机上启用蓝牙功能。这样才能在两者之间建立联系。
  </p>
  <p class="graf graf--p">
   如果所有其他方法都失败了，请在登录时点击“检查手机2FA”提示上的“取消”，然后在登录页面上选择“尝试其他方式”。
  </p>
  <p class="graf graf--p">
   从那里，选择“输入您的一个8位替代码”，然后在出现提示时输入上述步骤5中生成的代码之一。
  </p>
  <p class="graf graf--p">
   4、如果您无法访问受信任的设备并被锁定，该怎么办？
  </p>
  <p class="graf graf--p">
   如果您无法访问受信任的设备，Google会提供许多不同的替代选项。
  </p>
  <p class="graf graf--p">
   如果所有其他方法都失败了，那么确保您不会被锁定帐户的最佳方法之一就是使用您的备用码。
  </p>
  <p class="graf graf--p">
   同样，要做到这一点，在要求检查手机时单击提示上的“取消”，然后在登录页面上选择“尝试其他方式”并选择“输入您的8位备用码之一”。
  </p>
  <p class="graf graf--p">
   5、这与Google旧的“使用您的手机登录”提示有何不同？
  </p>
  <p class="graf graf--p">
   简单说就是，以前的方法在您实施验证的时候，谷歌不会验证您的手机和您正在记录的设备是否在同一物理区域。
  </p>
  <p class="graf graf--p">
   这个旧系统完全基于Web，但由于新系统使用蓝牙和位置服务，它可以验证这两个设备在物理上彼此靠近，从而降低远程帐户入侵的可能性。
  </p>
