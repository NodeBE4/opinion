   </noscript>
  </p>
  <p class="graf graf--p">
   美国有线电视新闻网
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnn.com/2019/06/05/politics/us-intelligence-saudi-arabia-ballistic-missile-china/index.html" href="https://www.cnn.com/2019/06/05/politics/us-intelligence-saudi-arabia-ballistic-missile-china/index.html" rel="noopener noreferrer" target="_blank">
    报道称
   </a>
   ，美国情报显示沙特阿拉伯在中国政府的帮助下大大扩展了其弹道导弹计划。
  </p>
  <p class="graf graf--p">
   以前没有报道过的机密情报显示，该国最近从中国购买了一些资产，使其能够大幅扩展其导弹基础设施和技术。
  </p>
  <p class="graf graf--p">
   特朗普政府没有立即向国会的主要成员披露调查结果，导致一些民主党人得出结论认为这是故意遗漏。
  </p>
  <p class="graf graf--p">
   中央情报局和国家情报总监办公室没有立即回应评论请求。
  </p>
  <p class="graf graf--p">
   消息人士告诉CNN说，沙特的最终目标并不是很清楚，尽管看起来像是为核弹头所做的努力的一部分。
  </p>
  <p class="graf graf--p">
   沙特王储穆罕默德·本·萨勒曼（Mohammed bin Salman）曾经表示，如果伊朗地区对手发展核武器，该王国将努力获得核武器。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “毫无疑问，如果伊朗发展核弹，我们将尽快效仿”，MBS 去年对美国媒体说。
  </p>
  <p class="graf graf--p">
   关联此前的报道《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E6%9D%80%E4%BA%BA%E7%8B%82%E7%9A%84%E7%9B%9B%E5%AE%B4/" href="https://www.iyouport.org/%E6%9D%80%E4%BA%BA%E7%8B%82%E7%9A%84%E7%9B%9B%E5%AE%B4/" rel="noopener noreferrer" target="_blank">
    美国在将高度敏感的核技术卖给沙特
   </a>
   》
  </p>
  <figure aria-describedby="caption-attachment-5889" class="wp-caption aligncenter" id="attachment_5889" style="width: 908px">
   <img alt="" class="size-full wp-image-5889 jetpack-lazy-image" data-lazy-sizes="(max-width: 908px) 100vw, 908px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/核技术.png?resize=908%2C1350&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/核技术.png?w=908&amp;ssl=1 908w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/核技术.png?resize=202%2C300&amp;ssl=1 202w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/核技术.png?resize=768%2C1142&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/核技术.png?resize=689%2C1024&amp;ssl=1 689w" data-recalc-dims="1" height="1350" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/核技术.png?resize=908%2C1350&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="908"/>
   <noscript>
    <img alt="" class="size-full wp-image-5889" data-recalc-dims="1" height="1350" sizes="(max-width: 908px) 100vw, 908px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/核技术.png?resize=908%2C1350&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/核技术.png?w=908&amp;ssl=1 908w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/核技术.png?resize=202%2C300&amp;ssl=1 202w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/核技术.png?resize=768%2C1142&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/核技术.png?resize=689%2C1024&amp;ssl=1 689w" width="908"/>
   </noscript>

   </figcaption>
  </figure>
  <p class="graf graf--p">
   根据1987年导弹技术控制制度，沙特被禁止从美国购买弹道导弹，但至少在几十年前沙特已经从中国购买了弹道导弹，公开报告显示他们可能在2007年再次购买过。
  </p>
  <p class="graf graf--p">
   CNN称，美国一直试图通过销售美国军用飞机来阻止沙特阿拉伯寻求导弹，军用飞机也能确保沙特在该地区保持空中力量优势。
  </p>
  <p class="graf graf--p">
   华盛顿智库民主国家基金会的 Behnam Taleblu说：“沙特不需要与伊朗竞争生产或采购弹道导弹。它已具有重要的常规军事优势。”
  </p>
  <p class="graf graf--p">
   然而，最近几个月，美国与伊朗之间的紧张局势有所加剧，德黑兰通过缩减其部分承诺，回应美国退出核协议的行为。
  </p>
  <p class="graf graf--p">
   国会与特朗普政府在伊朗问题上的紧张关系也在爆发，最近几周国会议员对特朗普决定绕过国会向沙特和其他海湾盟友出售武器的决定感到愤怒。
  </p>
  <p class="graf graf--p">
   国会今年早些时候投票阻止美国支持沙特领导的也门战争。但是在通过众议院和参议院后特朗普否决了这项措施。⚪️
  </p>
