  <p class="graf graf--p graf-after--figure" id="ae81">
   大众操作系统，包括 Windows，Mac OS X 和 Linux，太容易破解了。稍不留神 — 比如打开了带有木马的电子邮件附件、安装了伪装成 Flash 的恶意软件、没有足够快地更新软件 等等 — 都会让你成为攻击者的盛宴。
  </p>
  <p class="graf graf--p graf-after--p" id="8541">
   攻击者可以访问您的所有文件，可以记录您的按键、截取屏幕快照，甚至可以通过麦克风窃听，并通过网络摄像头偷窥。
  </p>
  <p class="graf graf--p graf-after--p" id="4090">
   但是你可以将最要紧的文件和程序与计算机的其他部分隔离开来。包括即使你打开了可能是未知的附件，也可以保护你的系统。再比如访问不太确定的网站 — 色情网站，洪流网站，盗版电视和体育网站 — 或测试从网站随机下载的软件，虚拟化技术都可以将危险隔离出来。
  </p>
  <p class="graf graf--p graf-after--p" id="a895">
   你还可以通过它来
   <strong class="markup--strong markup--p-strong">
    保持在线匿名
   </strong>
   ，通过确保所有互联网流量通过它进行路由来保护 Tor 提供的隐私护盾 —
   <strong class="markup--strong markup--p-strong">
    即使软件，如 Tor Browser 或 Pidgin，也有可能被特意攻击，攻击者有可能绕过 Tor
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="7b3f">
   很多地方都会用到虚拟化技术，比如sandbox、程序保护壳等。
  </p>
  <blockquote class="graf graf--blockquote graf-after--p" id="82f4">
   本文是虚拟机上的简单入门，包括如何在其中一安装 Ubuntu Linux 发行版。再介绍一个
   <strong class="markup--strong markup--blockquote-strong">
    可以在虚拟机中运行的操作系统，以最大化你的在线匿名性
   </strong>
   ; 它是保持秘密身份的理想选择。此外还会介绍一种比目前大多数可用的操作系统
   <strong class="markup--strong markup--blockquote-strong">
    更安全的操作系统，并提升隔离安全性
   </strong>
   。
  </blockquote>
  <p class="graf graf--p graf-after--blockquote" id="8d07">
   <strong class="markup--strong markup--p-strong">
    虚拟机 VM
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="81df">
   每个虚拟机在运行时都会使用计算机内存的一大块，并拥有自己的虚拟硬盘，这只是真实硬盘上的一个文件。可以在其中安装操作系统和运行软件，可以在执行具有潜在危险的操作之前保存快照，并在完成后还原快照，从而将 VM 恢复到先前的状态。
  </p>
  <p class="graf graf--p graf-after--p" id="fcdf">
   在虚拟化术语中，您现在运行的操作系统称为“主机”，您运行的每个 VM 都是“来宾／访客”。如果来宾虚拟机被骇客入侵，您的主机将保持安全。
  </p>
  <p class="graf graf--p graf-after--p" id="78c5">
   出于这个原因，安全研究人员经常使用 VM 来研究病毒：在来宾虚拟机上释放病毒，以安全地监控它们正在尝试做什么以及如何工作，他们将病毒与其他计算机“隔离”起来。
  </p>
  <p class="graf graf--p graf-after--p" id="7613">
   本文使用名为
   <a class="markup--anchor markup--p-anchor" data-href="https://www.virtualbox.org/" href="https://www.virtualbox.org/" rel="nofollow noopener noreferrer" target="_blank">
    VirtualBox
   </a>
   的虚拟化软件。它是开源的，可以免费下载。VirtualBox 适用于 Windows，Mac OS X 和 Linux。如果您尝试，可以在这里
   <a class="markup--anchor markup--p-anchor" data-href="https://www.virtualbox.org/wiki/Downloads" href="https://www.virtualbox.org/wiki/Downloads" rel="nofollow noopener noreferrer" target="_blank">
    下载并安装副本
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="3cf6">
   一般来说，首先在虚拟机中安装 Linux 发行版更简单。可以根据需要随意安装任意数量的 Linux 虚拟机 — 轻松设置。
  </p>
  <p class="graf graf--p graf-after--p" id="fd50">
   如果想要在虚拟机内测试 Windows 或 Mac OSX 的软件以查看它是否恶意，还可以在 VM 中安装这些操作系统。但是有限制。例如，虽然 OS X 可以免费安装在最多两台虚拟机上，但是当您这样做时必须是在 Mac 上；在 Windows 上，您可能需要为每个 VM 购买单独的 Windows 许可证。以下是在 VM 中
   <a class="markup--anchor markup--p-anchor" data-href="https://ntk.me/2012/09/07/os-x-on-os-x/" href="https://ntk.me/2012/09/07/os-x-on-os-x/" rel="nofollow noopener noreferrer" target="_blank">
    安装 Mac OS X
   </a>
   以及在 VM 中
   <a class="markup--anchor markup--p-anchor" data-href="http://download.cnet.com/blog/download-blog/how-to-run-windows-10-on-a-virtual-machine" href="https://download.cnet.com/blog/download-blog/how-to-run-windows-10-on-a-virtual-machine" rel="nofollow noopener noreferrer" target="_blank">
    安装 Windows 10
   </a>
   的说明。
  </p>
  <p class="graf graf--p graf-after--p" id="d67e">
   不必担心破坏任何东西，你可以随时删除您的 VM 重新开始。这就是虚拟机的好处：用电子游戏的说法就是，你获得了无限的生命。
  </p>
  <p class="graf graf--p graf-after--p" id="3a4d">
   <strong class="markup--strong markup--p-strong">
    创建 VM 并安装 Ubuntu
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="2f6d">
   希望您已经按照上面的说明
   <a class="markup--anchor markup--p-anchor" data-href="https://www.virtualbox.org/wiki/Downloads" href="https://www.virtualbox.org/wiki/Downloads" rel="nofollow noopener noreferrer" target="_blank">
    下载并安装了
   </a>
   VirtualBox。接下来，跳到 Ubuntu 的网站并
   <a class="markup--anchor markup--p-anchor" data-href="http://www.ubuntu.com/download/desktop" href="https://www.ubuntu.com/download/desktop" rel="nofollow noopener noreferrer" target="_blank">
    下载
   </a>
   Ubuntu 的副本。
  </p>
  <p class="graf graf--p graf-after--p" id="4a68">
   现在打开 VirtualBox 并单击“新建”以创建新 VM。下面是正在调用虚拟机“ubuntu-test”。
  </p>
  <div class="wp-block-image">
   <figure class="aligncenter">

  <p class="graf graf--p graf-after--figure" id="89a3">
   您可以选择新 VM 将具有多少内存，然后为其创建新的虚拟硬盘。您分配给 VM 的资源将无法用于计算机上的其他程序。最后，单击“创建”。
  </p>
  <p class="graf graf--p graf-after--p" id="bc4c">
   下一步是安装 Ubuntu。选择“ubuntu-test” VM 后，单击“开始”启动它。由于虚拟机是全新的，它会提示插入操作系统安装盘。当然实际上并不需要“磁盘”，相反可以找到并选择磁盘映像文件，然后单击“开始”。
  </p>
  <div class="wp-block-image">
   <figure class="aligncenter">

  <p class="graf graf--p graf-after--figure" id="100b">
   现在 VM 开始启动到 Ubuntu 磁盘。
   <strong class="markup--strong markup--p-strong">
    注意
   </strong>
   ，如果您单击虚拟机窗口，VirtualBox 会警告您 VM 将“捕获”您的鼠标和键盘输入，这意味着当您移动鼠标并在键盘上键入时，您将在您的来宾虚拟机中执行此操作，而不是您的主机上。
  </p>
  <p class="graf graf--p graf-after--p" id="0c16">
   您可以按“主机键”使鼠标和键盘再次控制常规计算机。在 Mac 上，主机键是左“Command”键，在 Windows 和 Linux 上，它是右键“Ctrl”键。
  </p>

  <p class="graf graf--p graf-after--figure" id="65d4">
   Ubuntu 磁盘已完成启动。点击“安装 Ubuntu ”并按照简单的说明操作。
  </p>
  <p class="graf graf--p graf-after--p" id="7f6f">
   选择“擦除磁盘并安装Ubuntu”（不用担心，只是擦除虚拟机的虚拟磁盘）。编写一个用户名和密码登录到这个 VM，然后将让它完成安装。当它最终完成后，VM 将重新启动到刚安装的操作系统中（安装 Ubuntu 后，VM 在重启时也许无法完全关闭。如果您也遇到这种情况，请单击“机器”菜单并选择“重置”以强制重新启动 VM）
  </p>
  <p class="graf graf--p graf-after--p" id="8bab">
   <strong class="markup--strong markup--p-strong">
    更新 VM 内的软件
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="53d8">
   要及时更新软件哦，养成习惯，在虚拟机中也坚持这样做。
  </p>
  <p class="graf graf--p graf-after--p" id="6d09">
   要更新 Ubuntu 中的所有软件，运行“软件更新程序”，输入密码，然后等着它干活儿。由于你刚刚安装了此操作系统并且从未进行过更新，因此下载和安装所有内容可能需要一段时间。
  </p>

   </noscript>

  <p class="graf graf--p graf-after--figure" id="6096">
   <strong class="markup--strong markup--p-strong">
    Guest Additions
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="cd88">
   现在就是安装 VirtualBox Guest Additions 了。Guest Additions 不是必需的，但它允许你做一些便捷的操作，比如调整 VM 窗口大小，在主机和来宾机之间共享剪贴板，设置共享文件夹，以便来宾机 VM 可以访问主机上的特定文件。
  </p>
  <p class="graf graf--p graf-after--p" id="28a3">
   要安装 Guest Additions，需要将包含该软件的虚拟 CD 插入 VM。您可以通过单击屏幕顶部的“设备”菜单，从 VirtualBox 程序中选择“插入Guest Additions CD图像“。它会弹出一个对话框，要求获得安装许可。单击“运行”，VirtualBox 将打开一个显示安装进度的新窗口。
  </p>
  <p class="graf graf--p graf-after--p" id="7610">
   完成后，重新启动 VM。您可以通过单击右上角的齿轮，点“关机”，然后单击“重新启动”来执行此操作。
  </p>
  <p class="graf graf--p graf-after--p" id="f1e4">
   你能看到上面菜单中的选项分别是：“已禁用”，“主机到访客”，“访客到主机”或“双向”。
   <strong class="markup--strong markup--p-strong">
    除非您需要在访客虚拟机和主机之间进行复制和粘贴，否则最好将此设置保持为“已禁用”。您可以随时暂时启用剪贴板共享，完成后再次禁用它。⚠️ 这个很重要哦
   </strong>
   。

   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="6564">

   <strong class="markup--strong markup--p-strong">
    隔离 VM 内部的危险行为
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="dac7">
   在做一些您认为可能会破坏 VM 或可能会遭恶意软件感染的动作之前，您可能希望保存它的快照，以便在完成后恢复它。可以通过单击屏幕顶部的 VirtualBox 的“计算机”菜单并选择“拍摄快照”来保存快照。
  </p>
  <p class="graf graf--p graf-after--p" id="9679">
   &gt;
   <strong class="markup--strong markup--p-strong">
    打开您不信任的文档
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="9c2d">
   被骇客攻击的最简单方法之一就是打开恶意文档。攻击者可能会通过电子邮件向您发送一个陷阱。如果你点开它了，该恶意文件将利用操作系统或 Adobe Reader 或 Microsoft Word 等软件中的缺陷，从而允许攻击者接管您的计算机。
  </p>
  <p class="graf graf--p graf-after--p" id="d376">
   我们并不总是清楚哪些文件是安全的，哪些是恶意的。一个聪明的攻击者可以假装邀请你参加你感兴趣的会议，并附上一个伪装成会议日程的恶意文件，或者他们可以假装招聘你提供你的梦想工作，并附上一些看起来像是职位的描述 —
   <strong class="markup--strong markup--p-strong">
    总之他们对你调查很久了，完全可以提供你最感兴趣的东西吸引你。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="ee96">
   最安全的方法是不打开任何附件或点击电子邮件中的任何链接，但这是完全不可行的，特别是对于积极征求消息来源的记者或活动家来说。
  </p>
  <p class="graf graf--p graf-after--p" id="9f6d">
   您可能还会在网上找到您真正想要查看的文件，但不确定是否安全。例如，
   <a class="markup--anchor markup--p-anchor" data-href="https://firstlook.org/theintercept/2015/07/07/leaked-documents-confirm-hacking-team-sells-spyware-repressive-countries/" href="https://firstlook.org/theintercept/2015/07/07/leaked-documents-confirm-hacking-team-sells-spyware-repressive-countries/" rel="nofollow noopener noreferrer" target="_blank">
    Hacking Team 电子邮件归档
   </a>
   中的文档。你可能不应该相信那些，但无论如何你可能想看看它们。
  </p>
  <p class="graf graf--p graf-after--p" id="8142">
   这是一封电子邮件，黑客团队的员工似乎在讨论如何向埃及国防承包商提供他们的黑客服务演示。附件被称为“Exploit.docx”，似乎合法？
  </p>
  <p class="graf graf--p graf-after--p" id="2414">
   如果我尝试在 Chrome 中打开这个可疑文件，浏览器会抛出安全警告，这是有充分理由的。从 Hacking Team 存档下载的任何附件可能会让人被入侵。
  </p>

   </strong>

  <p class="graf graf--p graf-after--p" id="6877">
   <strong class="markup--strong markup--p-strong">
    &gt; 访问奇怪的网站
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="9bb0">
   在日常使用的普通计算机上，强化 Web 浏览器始终是个好主意。安装阻止广告和恶意软件的浏览器插件，对于阻止可能试图通过 Web 浏览器接管您的计算机的软件大有帮助。
  </p>
  <p class="graf graf--p graf-after--p" id="7d19">
   但即使做了所有这些事，也无法保证只是通过加载网站就不会被黑客入侵。如果您打算访问一个您认为可能会遭受黑客攻击的网站，通过 VM。
  </p>
  <p class="graf graf--p graf-after--p" id="5be0">
   您甚至可以为此目的设置专用 VM。 （如果在上一步中关闭了网络，可以通过单击“设备”菜单，转到“网络”并选中“连接网络适配器”再次启用它）
  </p>
  <p class="graf graf--p graf-after--p" id="9f83">
   <strong class="markup--strong markup--p-strong">
    &gt; 运行易受攻击的软件
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="efa0">
   所有程序都包含 bug，这些 bug 可能被利用来接管计算机。不被骇客攻击的最简单彻底的方法是不使用计算机，但这肯定不是一种选择; 我们仍然需要运行程序。
  </p>
  <p class="graf graf--p graf-after--p" id="46f2">
   <strong class="markup--strong markup--p-strong">
    有些程序的攻击面比其他程序大得多
   </strong>
   。例如，libpurple，为加密聊天程序 Pidgin 和 Adium 提供支持的底层代码，因其最初写于1998年的旧的臃肿的，可能有缺陷的源代码而受到严厉批评（近年来许多关键的 libpurple 错误得到了修复，所以它目前的形状要好得多）但是，如果您想在计算机上进行加密聊天对话，那么除了使用它之外，您没有很多选择。
  </p>
  <p class="graf graf--p graf-after--p" id="4cba">
   如果您依赖某个软件，但是您认为在主机上运行它会增加被黑客攻击的可能性，您可以设置专用 VM 来运行该程序。
  </p>
  <p class="graf graf--p graf-after--p" id="1666">
   例如，⚠️ 如果您的专用聊天虚拟机通过 Pidgin 攻击被入侵，攻击者将能够监视你在 Pidgin 中加密的聊天对话，但即便如此，🔐 他们也无法访问您计算机上的其他文件。他们将无法查看您在网络浏览器中保存的密码，或通过麦克风窃听，也无法阅读电子邮件或其他任何密码。
  </p>
  <h3 class="graf graf--h3 graf-after--p" id="477e">
   <strong class="markup--strong markup--h3-strong">
    你还是要小心
   </strong>
  </h3>
  <p class="graf graf--p graf-after--h3" id="4421">
   所有软件都有错误，当然也包括虚拟化软件。虽然隔离 VM 内部的危险活动大大降低，但从来没有绝对安全。
  </p>
  <p class="graf graf--p graf-after--p" id="5592">
   ⚠️ 如果您的虚拟机被攻击，那么攻击者可以逃脱您的虚拟机，以便在您的主机上自由运行和更改程序。为此攻击者必须利用虚拟化软件。这些错误很少见，但
   <a class="markup--anchor markup--p-anchor" data-href="http://www.zdnet.com/article/researcher-critical-vulnerability-found-in-vmwares-desktop-apps/" href="https://www.zdnet.com/article/researcher-critical-vulnerability-found-in-vmwares-desktop-apps/" rel="nofollow noopener noreferrer" target="_blank">
    确实发生过
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="c35e">
   您还应该小心如何使用上面描述的 VirtualBox 剪贴板共享和文件共享功能。
  </p>
  <p class="graf graf--p graf-after--p" id="0471">
   例如，如果有人攻击了共享剪贴板设置为“Host to Guest”或“Bidirectional”的 VM，则攻击者可能会监视您复制到主机上剪贴板的内容 — 例如密码。
  </p>
  <p class="graf graf--p graf-after--p" id="37e5">
   💡
   <strong class="markup--strong markup--p-strong">
    Whonix 以保持匿名
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="6975">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.whonix.org/" href="https://www.whonix.org/" rel="nofollow noopener noreferrer" target="_blank">
    Whonix
   </a>
   是一个操作系统，您可以在 VirtualBox 内部的现有计算机上安装，这会强制所有网络流量通过匿名网络 Tor。
  </p>
  <p class="graf graf--p graf-after--p" id="2fd8">
   关于 Tor 我们介绍过很多啦，可以在
   <a class="markup--anchor markup--p-anchor" data-href="https://steemit.com/@iyouport" href="https://steemit.com/@iyouport" rel="nofollow noopener noreferrer" target="_blank">
    这里
   </a>
   找到。但 Tor 浏览器和所有其他软件一样，都有漏洞，是的他们更新是很快的。
   <strong class="markup--strong markup--p-strong">
    如果您访问 Tor 浏览器中的网站，该网站可以假装利用严重的错误强迫您的计算机与 Tor 网络外的攻击者建立互联网连接，让他们了解您的真实 IP 地址和身份。这正是
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://www.wired.com/2013/09/freedom-hosting-fbi/" href="https://www.wired.com/2013/09/freedom-hosting-fbi/" rel="nofollow noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     联邦调查局在2013年9月时抓人用的歪点子
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="e31e">
   🔐 Whonix 使用两个虚拟机，称为 Whonix-Gateway 和 Whonix-Workstation，以最大限度地保护匿名。
  </p>
  <p class="graf graf--p graf-after--p" id="15e7">
   网关 VM 充当工作站 VM 的上游互联网提供商，它强制所有网络流量通过 Tor 网。工作站 VM 是您使用 Tor 浏览器的地方。如果你被攻击，
   <strong class="markup--strong markup--p-strong">
    例如使用类似 FBI 用过的 Tor 浏览器漏洞攻击，攻击者不仅被包含在此 VM 中无法访问您的主机，也无法打破匿名化。攻击者所做的所有网络连接都将通过网关 VM，迫使他们通过 Tor。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="6272">
   Whonix 非常棒，因为您可以确信在工作站 VM 中所做的一切都是匿名通过 Tor 网的。这意味着攻击者无法对您验明身份，除非他们可以逃离您的虚拟机。
  </p>
  <p class="graf graf--p graf-after--p" id="9491">
   您可以使用 XChat 等聊天软件匿名连接到 IRC 服务器，或者使用 Pidgin 连接到 Jabber 服务器进行匿名加密聊天，或使用 Icedove 和 Enigmail 发送匿名加密电子邮件。
  </p>
  <p class="graf graf--p graf-after--p" id="8048">
   ⚠️ 但请记住，Whonix 与其他基于虚拟机的安全性一样，如果您的主机遭到攻击或被占用，它是无法保护您的。如果您使用 Whonix 匿名向记者发送文档，并且被专制政府怀疑为嫌疑人，您的 Whonix VM 中可能包含可用于对您定罪的证据！⚠️
  </p>
  <p class="graf graf--p graf-after--p" id="13d0">
   <strong class="markup--strong markup--p-strong">
    安装和配置 Whonix
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="09ff">
   转到 Whonix VirtualBox
   <a class="markup--anchor markup--p-anchor" data-href="https://www.whonix.org/wiki/VirtualBox#First_time_user.3F" href="https://www.whonix.org/wiki/VirtualBox#First_time_user.3F" rel="nofollow noopener noreferrer" target="_blank">
    下载页面
   </a>
   并下载 Whonix-Gateway 和 Whonix-Workstation 的副本（总共3.1GB，因此可能需要一些时间）。验证 PGP 签名也是一个好主意，我们以前介绍过。
  </p>
  <p class="graf graf--p graf-after--p" id="7f62">
   下载完成后，打开 VirtualBox，单击顶部的“文件”菜单，然后单击“导入设备”。浏览刚刚下载的 Whonix-Gateway 文件，然后单击“继续”。
  </p>
  <div class="wp-block-image">
   <figure class="aligncenter">
    <img alt="" class="wp-image-3460 jetpack-lazy-image" data-lazy-sizes="(max-width: 540px) 100vw, 540px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-75.jpeg?resize=540%2C399&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-75.jpeg?w=540&amp;ssl=1 540w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-75.jpeg?resize=300%2C222&amp;ssl=1 300w" data-recalc-dims="1" height="399" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-75.jpeg?resize=540%2C399&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="540"/>
    <noscript>
     <img alt="" class="wp-image-3460" data-recalc-dims="1" height="399" sizes="(max-width: 540px) 100vw, 540px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-75.jpeg?resize=540%2C399&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-75.jpeg?w=540&amp;ssl=1 540w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-75.jpeg?resize=300%2C222&amp;ssl=1 300w" width="540"/>
    </noscript>
   </figure>
  </div>
  <p>
   现在单击“导入”，阅读警告，然后单击“同意”。您的 Whonix 网关 VM 将自动设置。使用 Whonix-Workstation 重复这些相同的步骤。完成后，VirtualBox 中将有两个新 VM。
  </p>
  <div class="wp-block-image">
   <figure class="aligncenter">
    <img alt="" class="wp-image-3461 jetpack-lazy-image" data-lazy-sizes="(max-width: 540px) 100vw, 540px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-76.jpeg?resize=540%2C359&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-76.jpeg?w=540&amp;ssl=1 540w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-76.jpeg?resize=300%2C199&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-76.jpeg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="359" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-76.jpeg?resize=540%2C359&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="540"/>
    <noscript>
     <img alt="" class="wp-image-3461" data-recalc-dims="1" height="359" sizes="(max-width: 540px) 100vw, 540px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-76.jpeg?resize=540%2C359&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-76.jpeg?w=540&amp;ssl=1 540w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-76.jpeg?resize=300%2C199&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-76.jpeg?resize=272%2C182&amp;ssl=1 272w" width="540"/>
    </noscript>
   </figure>
  </div>
  <p class="graf graf--p graf-after--figure" id="c0f5">
   启动 Whonix-Gateway 和 Whonix-Workstation。您需要在后台打开网关 VM，否则工作站 VM 将无法访问 Internet，但您将在工作站中完成大部分工作。
  </p>
  <p class="graf graf--p graf-after--p" id="7d54">
   当网关 VM 第一次完成引导时，您需要对其进行配置。单击“Whonix 安装向导”以启用 Tor 和自动更新。
  </p>
  <p class="graf graf--p graf-after--p" id="a781">
   单击工作站 VM 中的“Whonix 安装向导”。在两个 VM 中，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.whonix.org/wiki/Post_Install_Advice#On_Whonix-Gateway_and_Whonix-Workstation" href="https://www.whonix.org/wiki/Post_Install_Advice#On_Whonix-Gateway_and_Whonix-Workstation" rel="nofollow noopener noreferrer" target="_blank">
    更改
   </a>
   默认密码并
   <a class="markup--anchor markup--p-anchor" data-href="https://www.whonix.org/wiki/Security_Guide#Recommendation_to_install_latest_security_updates_on_all_systems" href="https://www.whonix.org/wiki/Security_Guide#Recommendation_to_install_latest_security_updates_on_all_systems" rel="nofollow noopener noreferrer" target="_blank">
    更新
   </a>
   软件。
  </p>
  <p class="graf graf--p graf-after--p" id="ff39">
   现在是时候开始使用 Whonix 了。在工作站 VM 中，继续并打开 Tor Browser。它会在您第一次尝试打开时自动下载并安装它。一旦打开，您可以
   <strong class="markup--strong markup--p-strong">
    匿名
   </strong>
   浏览网页，
   <strong class="markup--strong markup--p-strong">
    即使 Tor 浏览器被攻击，也可以保持匿名
   </strong>
   。
  </p>
  <figure class="wp-block-image">
   <img alt="" class="wp-image-3462 jetpack-lazy-image" data-lazy-sizes="(max-width: 1000px) 100vw, 1000px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-77.jpeg?resize=1000%2C673&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-77.jpeg?w=1000&amp;ssl=1 1000w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-77.jpeg?resize=300%2C202&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-77.jpeg?resize=768%2C517&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-77.jpeg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="673" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-77.jpeg?resize=1000%2C673&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1000"/>
   <noscript>
    <img alt="" class="wp-image-3462" data-recalc-dims="1" height="673" sizes="(max-width: 1000px) 100vw, 1000px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-77.jpeg?resize=1000%2C673&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-77.jpeg?w=1000&amp;ssl=1 1000w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-77.jpeg?resize=300%2C202&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-77.jpeg?resize=768%2C517&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-77.jpeg?resize=272%2C182&amp;ssl=1 272w" width="1000"/>
   </noscript>
  </figure>
  <p class="graf graf--p graf-after--figure" id="73db">
   <strong class="markup--strong markup--p-strong">
    Qubes：隔离的安全性
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="6d5e">
   由于所有软件都有错误，隔离自己的 VM 中的每个程序是不是最安全呢？
   <a class="markup--anchor markup--p-anchor" data-href="https://www.qubes-os.org/" href="https://www.qubes-os.org/" rel="nofollow noopener noreferrer" target="_blank">
    Qubes
   </a>
   是一个操作系统，可以做到这一点，并且比传统操作系统中使用 VirtualBox 或 VMWare 等虚拟化软件更加可用和安全。
  </p>
  <p class="graf graf--p graf-after--p" id="bab9">
   在 Qubes 中，您的主机运行图形桌面环境，就是这样 — 主机甚至无法访问 Internet。您可以在 Linux 或 Windows VM 中运行所有其他软件。 Qubes 也对 Whonix 有很大的支持。如果在 Qubes 中使用 Whonix，则与使用传统操作系统时相比，主机的受攻击面要小得多。
  </p>
  <p class="graf graf--p graf-after--p" id="5162">
   <strong class="markup--strong markup--p-strong">
    Qubes 可以轻松管理不同“安全域”的单独 VM
   </strong>
   。例如，您可以创建用于检查工作电子邮件并登录工作相关帐户的工作 VM，以及您使用的单独个人 VM 登录 Facebook。您可以创建用于日常 Web 冲浪中的不受信任的 VM，以及用于存储敏感文件（如密码数据库）或您正在处理的机密文档的 Vault VM。您可以右键单击任何文档以在“一次性 VM”中打开它，这是一个仅为查看此文档而创建的 VM，然后在关闭文档时再次删除。
  </p>
  <figure class="wp-block-image">
   <img alt="" class="wp-image-3463 jetpack-lazy-image" data-lazy-sizes="(max-width: 1000px) 100vw, 1000px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-78.jpeg?resize=1000%2C563&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-78.jpeg?w=1000&amp;ssl=1 1000w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-78.jpeg?resize=300%2C169&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-78.jpeg?resize=768%2C432&amp;ssl=1 768w" data-recalc-dims="1" height="563" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-78.jpeg?resize=1000%2C563&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1000"/>
   <noscript>
    <img alt="" class="wp-image-3463" data-recalc-dims="1" height="563" sizes="(max-width: 1000px) 100vw, 1000px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-78.jpeg?resize=1000%2C563&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-78.jpeg?w=1000&amp;ssl=1 1000w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-78.jpeg?resize=300%2C169&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-78.jpeg?resize=768%2C432&amp;ssl=1 768w" width="1000"/>
   </noscript>
  </figure>
  <p class="graf graf--p graf-after--figure" id="f5ba">
   虽然它拥有一个活跃的开发社区和不断增长的用户群，但 Qubes 对于非高级用户来说并不容易使用。除非您已经习惯从命令行解决 Linux 问题。在 Qubes 中，诸如如何安装新程序或截取屏幕截图这样的简单问题可能会让初学者产生陡峭的学习曲线。
  </p>
  <p class="graf graf--p graf-after--p" id="5674">
   🔐
   <strong class="markup--strong markup--p-strong">
    但总而言之，如果您使用 Qubes，您可以将计算机变成一个令人难以置信的复杂而安全的堡垒，这与传统操作系统不同。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="07e1">
   最后再唠叨一句，所有软件都有 bug，包括 Qubes 和 Xen，这是为 Qubes 提供动力的虚拟化软件。即使您正在运行 Qubes，并立即更新所有软件，并仔细隔离所有软件，只打开一次性 VM 中的文档，并在 Whonix 工作站内的 Tor Browser 中进行所有浏览，您的主机仍然可能被攻击，只要攻击者拥有大量资源、耐心以及掌握零日漏洞。
   <a href="https://www.iyouport.org/%e9%9b%b6%e6%97%a5%e6%bc%8f%e6%b4%9e%e5%92%8c-cia-%e9%bb%91%e5%ae%a2%e5%b7%a5%e5%85%b7/" rel="noopener noreferrer" target="_blank">
    这里是关于零日漏洞的白话级介绍
   </a>
   。情报机构花大价钱买这东西，足见他们是有多坏。⚪️
  </p>
