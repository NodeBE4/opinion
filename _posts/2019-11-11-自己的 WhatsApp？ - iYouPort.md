  <p class="graf graf--p">
   数据必需是民主的，否则任何主权都不是解决霸权的正确思考方式。
  </p>
  <p class="graf graf--p">
   印度正在紧跟法国的脚步，准备建立自己的通信应用程序，并要求政府雇员将其用于官方通信。
  </p>
  <p class="graf graf--p">
   据“经济时报”报道，新德里政府正在考虑是否需要使用自己开发的电子邮件和聊天应用程序。
  </p>
  <p class="graf graf--p">
   此举背后的理由是减少对外国实体的依赖，这种担忧在中美争霸中的体现已经非常明显。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们需要使我们的沟通变得独立，” 该报援引一位不愿透露姓名的政府高级官员的话说。
  </p>
  <p class="graf graf--p">
   该官员说，通过将中国巨头华为列入实体名单，美国已经“对新德里敲响了警钟”。
  </p>
  <p class="graf graf--p">
   6月份，印度与特朗普直接发生了一个持续的贸易紧张局势，此前印度没有向特朗普保证将“提供公平合理的市场准入”。
  </p>
  <p class="graf graf--p">
   印度称这一举动是一种“不幸”，几周后美国一些出口产品的关税上调。
  </p>
  <p class="graf graf--p">
   如果实现这一目标，离开国外通信应用程序的举动将
   <strong class="markup--strong markup--p-strong">
    不会是一个国家第一次试图谨慎地限制外国玩家运行本国政府部门使用的流行消息应用程序
   </strong>
   。
  </p>
  <p class="graf graf--p">
   法国今年早些时候在其政府办公室推出了一个加密聊天应用程序 — 称为 Tchap。
  </p>
  <p class="graf graf--p">
   只有法国政府办公室的雇员才可以注册使用该服务，尽管该国已开放应用程序的代码供全世界查看和审核。
  </p>
  <p class="graf graf--p">
   当然，Tchap 的安全漏洞在发布后的头24小时内就曝光了。
  </p>
  <p class="graf graf--p">
   安全性是政府必须解决的真正挑战，
   <strong class="markup--strong markup--p-strong">
    小国政府可能没有最好的资源 —— 人才、预算和专业知识 —— 来应对这一挑战
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这是挑战高堡奇人的最大难点之一。
  </p>
  <p class="graf graf--p">
   中国限制了许多外国公司在其国内的经营，同时也保留了用于政府办公室的定制版本的流行操作系统。朝鲜也是这么做的。
  </p>
  <p class="graf graf--p">
   对印度来说，这也不是前所未有的一步。
  </p>
  <p class="graf graf--p">
   由于大多数政府机构继续使用微软的 Windows 操作系统，该国多年来一直试图建立和扩展自己的基于 Linux 的桌面操作系统 BOSS，但收效甚微。
  </p>
  <p class="graf graf--p">
   即使印度已成为世界第三大创业中心，该国也未能为许多受欢迎的服务建立本地替代品。
  </p>
  <p class="graf graf--p">
   Facebook 的 WhatsApp 已经在印度的通信中无处不在，而谷歌的 Android 和微软的 Windows 在印度大多数智能手机和电脑都是如此地位稳固。
  </p>
