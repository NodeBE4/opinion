  <p class="graf graf--p">
   想象一个警察和维稳部门使用的应用程序，其中包含从你的手机上的内容、到你的用电量、到你的银行帐户交易等等一切信息。
  </p>
  <p class="graf graf--p">
   你没有想错，这种东西真的存在。
  </p>
  <p class="graf graf--p">
   这个应用程序并非来自电视剧《黑镜》，而是来自中国 —— 用于监视中国新疆地区的数百万公民。
  </p>
  <p class="graf graf--p">
   在同一地区，据信有超过一百万的维吾尔族穆斯林被拘留在秘密的 “再教育营” 里。
  </p>
  <p class="graf graf--p">
   但是，⚠️不仅跟踪穆斯林，而且，该地区的
   <strong class="markup--strong markup--p-strong">
    每个人
   </strong>
   都在受到被称为联合行动平台（IJOP）的精密的监视系统的监视。
  </p>
  <p class="graf graf--p">
   人权观察组织披露了该应用程序的工作方式，以及启用的极端监视级别。
  </p>
  <p class="graf graf--p">
   请注意，
   <strong class="markup--strong markup--p-strong">
    对于与负责该系统构建的中国电子科技集团公司（CETC）合作的澳大利亚组织，这应该引起警惕和疑问
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/_Hy9eIjkmOM" width="750">
   </iframe>
  </p>
  <h3 class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “全球110多个国家”
   </strong>
  </h3>
  <p class="graf graf--p">
   CETC 是一家中国国有的军事技术公司 —— 中国的科技巨头之一。它自称为 “国防、安全[维稳]和信息化领域最强大的国家重点公司”。
  </p>
  <p class="graf graf--p">
   人权观察的研究表明，CETC 最终负责该监视应用程序，该应用程序是通过其全资子公司河北远东通信系统工程有限公司开发的。
  </p>
  <p class="graf graf--p">
   去年发表的映射中国科技公司全球扩张的调查发现，
   <strong class="markup--strong markup--p-strong">
    CETC 的合作项目从巴布亚新几内亚一直延伸到俄罗斯再到南非
   </strong>
   。在下面看到：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%9b%be%e8%a7%a3%e4%b8%ad%e5%9b%bd%e7%a7%91%e6%8a%80%e5%b7%a8%e5%a4%b4-%e5%8d%8a%e5%a3%81%e9%ab%98%e5%a0%a1%e5%a5%87%e4%ba%ba/" href="https://www.iyouport.org/%e5%9b%be%e8%a7%a3%e4%b8%ad%e5%9b%bd%e7%a7%91%e6%8a%80%e5%b7%a8%e5%a4%b4-%e5%8d%8a%e5%a3%81%e9%ab%98%e5%a0%a1%e5%a5%87%e4%ba%ba/" rel="noopener noreferrer" target="_blank">
     图解中国科技巨头 — 半壁“高堡奇人”
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   CETC 在其网站上声称
   <strong class="markup--strong markup--p-strong">
    在全球110多个国家和地区开展业务
   </strong>
   。
  </p>
  <p class="graf graf--p">
   包括澳大利亚在内。
  </p>
  <p class="graf graf--p">
   2017年，CETC 与悉尼科技大学（UTS）签署了一项2000万澳元的协议，以资助一个研究创新中心。
  </p>
  <p class="graf graf--p">
   可以预见，该项目以后将扩展到包括更多其他澳大利亚和中国的大学。
  </p>
  <p class="graf graf--p">
   ASPI 的研究先前强调了有关中国政府如何使用该项目中开发的技术的担忧。
  </p>
  <p class="graf graf--p">
   研究员 Elsa Kania 的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.aspi.org.au/report/technological-entanglement" href="https://www.aspi.org.au/report/technological-entanglement" rel="noopener noreferrer" target="_blank">
    一份报告
   </a>
   特别提到了 UTS 与 CETC 在
   <strong class="markup--strong markup--p-strong">
    军民两用AI研究方面的合作
   </strong>
   ，这是一个特别令人关注的领域。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%ba%e5%b7%a5%e6%99%ba%e8%83%bd%e6%96%b0%e5%86%b7%e6%88%98%e2%80%8a-%e2%80%8a-ai%e5%92%8c5g%e5%a6%82%e4%bd%95%e5%8f%98%e6%88%90%e4%b8%80%e5%9c%ba%e5%99%a9%e6%a2%a6/" href="https://www.iyouport.org/%e4%ba%ba%e5%b7%a5%e6%99%ba%e8%83%bd%e6%96%b0%e5%86%b7%e6%88%98%e2%80%8a-%e2%80%8a-ai%e5%92%8c5g%e5%a6%82%e4%bd%95%e5%8f%98%e6%88%90%e4%b8%80%e5%9c%ba%e5%99%a9%e6%a2%a6/" rel="noopener noreferrer" target="_blank">
     人工智能新冷战 — — AI和5G如何变成一场噩梦

  <p class="graf graf--p">
   例如，在2018年，UTS 发布了与 CETC 有关的研究
   <a class="markup--anchor markup--p-anchor" data-href="https://web-tools.uts.edu.au/projects/detail.cfm?ProjectId=PRO18-5157" href="https://web-tools.uts.edu.au/projects/detail.cfm?ProjectId=PRO18-5157" rel="noopener noreferrer" target="_blank">
    信息
   </a>
   ，该研究涉及 “基于复杂数据条件的在线[监视]视频检索系统”，该项目具有明显的潜在大规模监视应用。
  </p>
  <p class="graf graf--p">
   人权观察所披露的信息加重了这些担忧。
  </p>
  <p class="graf graf--p">
   我们现在知道，CETC 的技术正在实现大规模监视，对少数民族的迫害和侵犯人权的行为。那些大学需要问自己这是否真的是他们想要合作的公司。

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    中国科技传播影响力
   </strong>
  </h3>
  <p class="graf graf--p">
   CETC
   <strong class="markup--strong markup--p-strong">
    并非
   </strong>
   唯一一家既支持新疆镇压又在澳大利亚开展业务的公司。
  </p>
  <p class="graf graf--p">
   华为正在为新疆警察提供设备、技术支持和培训，并将加入新疆首都 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.albawaba.com/news/long-form-china%E2%80%99s-global-surveillance-industrial-complex-1141152" href="https://www.albawaba.com/news/long-form-china%E2%80%99s-global-surveillance-industrial-complex-1141152" rel="noopener noreferrer" target="_blank">
    智能安防行业
   </a>
   ” 创新实验室。
  </p>
  <p class="graf graf--p">
   华为被禁止在澳大利亚提供5G设备，但是，
   <strong class="markup--strong markup--p-strong">
    华为仍继续持有其他主要公共合同，包括最近与西澳大利亚州政府签订的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.abc.net.au/news/2019-03-13/huawei-contract-for-perth-trains-confirmed-by-wa-government/10896220" href="https://www.abc.net.au/news/2019-03-13/huawei-contract-for-perth-trains-confirmed-by-wa-government/10896220" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     价值2亿美元
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    的珀斯火车系统数字无线电合同。
   </strong>
  </p>
  <p class="graf graf--p">
   同时，监视摄像机巨头海康威视向新疆的监视计划提供了价值数亿美元的摄像机。
   <strong class="markup--strong markup--p-strong">
    其中许多配备了先进的技术，包括面部识别、甚至窃听功能
   </strong>
   。
  </p>
  <p class="graf graf--p">
   公开招标记录显示，海康威视摄像机已被放置在新疆至少967座清真寺中，用于监视维吾尔族公民，并 “确保所有人奉行‘统一’的政府规章”。最令人不安的是，海康威视还直接向中国政府的“再教育营”提供设备和服务。
  </p>

   </li>
  </ul>
  <p class="graf graf--p">
   中国的监视技术也正在进入澳大利亚。
  </p>
  <p class="graf graf--p">
   去年，据透露，达尔文市议会计划从中国深圳引进实施所谓的 “智能城市” 技术，其中包括 “虚拟围栏”，一旦有人越过就会触发警报。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%99%ba%e8%83%bd%e5%9f%8e%e5%b8%82%e7%a9%b6%e7%ab%9f%e6%98%af%e4%b8%aa%e4%bb%80%e4%b9%88%e9%ac%bc%ef%bc%9f%ef%bc%81/" href="https://www.iyouport.org/%e6%99%ba%e8%83%bd%e5%9f%8e%e5%b8%82%e7%a9%b6%e7%ab%9f%e6%98%af%e4%b8%aa%e4%bb%80%e4%b9%88%e9%ac%bc%ef%bc%9f%ef%bc%81/" rel="noopener noreferrer" target="_blank">
     智能城市到底是什么？你被那些当权者剥夺了什么？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   该项目将包括 “装有扬声器、摄像头和 Wi-Fi 的杆子，
   <strong class="markup--strong markup--p-strong">
    这可以使用当权者获得有关多少人走在哪个人行道上、以及他们使用了哪些网站和应用程序的详细数据。”
   </strong>
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    澳大利亚研究可以重新定位
   </strong>
  </h3>
  <p class="graf graf--p">
   UTS表示，它与 CETC 的合作均未涉及“根据《人权观察》报告”在新疆地区使用的技术。
  </p>
  <p class="graf graf--p">
   但是，伙伴关系中的一些关键研究领域
   <strong class="markup--strong markup--p-strong">
    应受到特别关注
   </strong>
   。 UTS在新闻稿中表示，该项目将专注于视觉大数据分析、移动感应和通信、以及物联网。
  </p>
  <p class="graf graf--p">
   在UTS自己的网站上显示，合作包括对 “公共安全[大规模监视]在线视频检索” 系统的研究。这些
   <strong class="markup--strong markup--p-strong">
    正是
   </strong>
   CETC 在新疆的监视平台所采用的技术。
  </p>
  <p class="graf graf--p">
   IJOP系统的关键是新疆各地街道，公共建筑、学校、办公室和清真寺中无处不在的闭路电视监视摄像机网络。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这些监视相机中的许多都配备了面部识别、车牌识别和夜视功能
   </strong>
   。
  </p>
  <p class="graf graf--p">
   对于分析这些摄像机收集的大量数据的需求，催生了中国人工智能公司蜂拥而上，每个公司都在争夺政府合同，以帮助实现政府监控程序的自动化。
  </p>
  <p class="graf graf--p">
   考虑到从人权观察的调查中获得的这些新信息，澳大利亚组织需要问问自己，与像 CETC 这样的公司合作是否值得。⚪️
  </p>
  <p class="graf graf--p">

    Should Australia be partnering with the Chinese tech giants designing a surveillance state?
