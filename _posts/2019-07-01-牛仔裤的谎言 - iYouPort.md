     <strong>
      <em>
       操纵司法结论真是太容易了。你的牛仔裤能暴露你的身份信息，这是为什么？这种监视方法是谁发明的？
      </em>
     </strong>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-3064 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-3.png?resize=1100%2C744&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-3.png?w=1600&amp;ssl=1 1600w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-3.png?resize=300%2C203&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-3.png?resize=768%2C519&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-3.png?resize=1024%2C692&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-3.png?resize=1100%2C744&amp;ssl=1 1100w" data-recalc-dims="1" height="744" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-3.png?resize=1100%2C744&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-3064" data-recalc-dims="1" height="744" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-3.png?resize=1100%2C744&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-3.png?w=1600&amp;ssl=1 1600w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-3.png?resize=300%2C203&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-3.png?resize=768%2C519&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-3.png?resize=1024%2C692&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-3.png?resize=1100%2C744&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   一个多月前，techdirt 报道了由 Richard Vorder Bruegge 领导的 FBI 取证技术的一个超级奇怪的分支。Vorder Bruegge 声称牛仔裤等大量生产的服装可以与指纹和 DNA 一样独特。根据他的法医“专业知识”，可以使用低分辨率 CCTV 屏幕抓取图像和一大堆箭头的布局图进行匹配 — — 用来识别一个人的身份。就是上面图中那样。
  </p>
  <p class="graf graf--p">
   这种邪门的 FBI 取证方法仍在被使用中。 其发明者 Vorder Bruegge 已经上升到了可以将他可疑的专业知识传授给他人的叫兽级地位。甚至没有被 FBI 的法医实验室嘲笑！
  </p>
  <p class="graf graf--p">
   ProPublica 继续深入研究联邦调查局可疑的法医计划，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.propublica.org/article/fbi-scientists-statements-linked-defendants-to-crimes-even-when-his-lab-results-didnt#158261" href="https://www.propublica.org/article/fbi-scientists-statements-linked-defendants-to-crimes-even-when-his-lab-results-didnt#158261" rel="noopener noreferrer" target="_blank">
    并发现
   </a>
   Vorder Bruegge 现在位于该国法医组织排行榜的前列！
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    如今，Vorder Bruegge 是全美最具影响力的犯罪实验室科学家之一。他在法证科学标准委员会任职，该委员会为从 DNA 到指纹的每个领域制定规则。他是本周在巴尔的摩组织的美国法医学院会议的联合主席，该会议由数千名犯罪实验室专业人士、研究人员、律师和法官组成
   </em>
   。
  </p>
  <p class="graf graf--p">
   这是因为 Vorder Bruegge 的推荐
   <strong class="markup--strong markup--p-strong">
    数量，而不是质量
   </strong>
   。 ProPublica 引用2013年法律评论文章，称他是“无处不在”的专家证人。这并不是一项高质量的工作。但是，只
   <strong class="markup--strong markup--p-strong">
    要有足够的机会，Vorder Bruegge 就可以并且已经成功地将未经证实的关于衣服独特性的说法转变成了他所作证的人遭受多年监禁的证据。
   </strong>
  </p>
  <p class="graf graf--p">
   他的记录表明，如果能更好地为检方服务，他愿意改变他的专家意见。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    在他的报告中，Vorder Bruegge 写道，John Henry Stroman 和强盗有类似的“面部，鼻子，嘴巴，下巴和耳朵的整体形状。”但 Vorder Bruegge
   </em>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/4952729-Faviau-Stroman-Report.html#document/p4/a457673" href="https://www.documentcloud.org/documents/4952729-Faviau-Stroman-Report.html#document/p4/a457673" rel="noopener noreferrer" target="_blank">
    <em class="markup--em markup--p-em">
     没有宣布匹配
    </em>
   </a>
   <em class="markup--em markup--p-em">
    ，只说视频和图片是低分辨率的。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    尽管如此，检察官
   </em>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/5026383-Stroman-Videoevidence.html#document/p2/a464275" href="https://www.documentcloud.org/documents/5026383-Stroman-Videoevidence.html#document/p2/a464275" rel="noopener noreferrer" target="_blank">
    <em class="markup--em markup--p-em">
     在法庭文件中称
    </em>
   </a>
   <em class="markup--em markup--p-em">
    ，如果 Vorder Bruegge 采取立场，他将证明“这张照片具有足够的分辨率，可以明确说明强盗是 John Henry Stroman。”
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    这不是第一次，也不是最后一次，Vorder Bruegge 的实验室说了一件事，而法院被告知了一些完全不同的东西。法庭记录和 FBI 实验室档案显示检察官或 Vorder Bruegge 的陈述至少在三个案件中与他的原始结论完全不同
   </em>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这就是当你关心定罪时所发生的事，而不是那些关于科学时发生的事
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Vorder Bruegge 作为一名地质学家的专业背景当然没有让他为将来盯着嫌疑人所穿的衬衫的颗粒状照片做好准备。但是，这一切对 FBI 来说都不重要，后者发现他是伪科学的有用支持者，可以用来把人们扔进监狱。
  </p>
  <p class="graf graf--p">
   FBI 专家很容易影响陪审员和法官的决策，即使在交叉检查中暴露出数学上不可能的确定性水平，或者至少在一个案例中，Vorder Brugge 承认他没能达到检察官希望他达到的结论。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    像 Vorder Bruegge 这样的工作正是为什么一位著名的联邦法官在2015年从法医委员会
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.techdirt.com/articles/20150202/11152629883/judge-resigns-forensic-science-committee-calls-out-dojs-trial-ambush-tactics.shtml" href="https://www.techdirt.com/articles/20150202/11152629883/judge-resigns-forensic-science-committee-calls-out-dojs-trial-ambush-tactics.shtml" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     辞职的原因
    </strong>
   </a>
   。
   <strong class="markup--strong markup--p-strong">
    Jed Rakoff 法官承认司法部不想解决其法医调查中普遍存在的问题
   </strong>
   ，他们只是把污秽扫到地毯下面，表面上看起来光亮就行了。司法部过度投资于半生不熟的科学和“自学成才”的砖家，这意味着会有多少人承受冤狱的折磨？
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    F
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.techdirt.com/articles/20190303/13431341724/fbis-clothing-match-expert-changed-testimony-to-better-serve-prosecutors-co-chairs-natl-forensic-committee.shtml" href="https://www.techdirt.com/articles/20190303/13431341724/fbis-clothing-match-expert-changed-testimony-to-better-serve-prosecutors-co-chairs-natl-forensic-committee.shtml" rel="noopener noreferrer" target="_blank">
    BI’s ‘Clothing Match’ Expert Changed Testimony To Better Serve Prosecutors, Co-Chairs Nat’l Forensic Committee
   </a>
   . This is what happens when you care about convictions but not all that much about science.
  </p>
