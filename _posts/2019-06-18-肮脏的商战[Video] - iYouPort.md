 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      <strong>
       用间谍手段搞商业竞争这事不这么奇怪，不过有家公司在此做的很奇特，他们是怎么做的？
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   优步的一位高级消息人士向澳大利亚广播公司证实了一项肮脏的计划，优步澳大利亚非法部署了一个名为 Surfcam 的内部间谍软件，以窥探一家名为 Gocatch 的竞争对手公司的司机。
  </p>
  <p class="graf graf--p">
   优步能够编制对手司机的电子邮件列表、汽车注册号和其他详细信息，并使用这些私密信息挖走 Gocatch 司机并将其转换为优步司机。
  </p>
  <p class="graf graf--p">
   这不是第一次。优步此前曾在新加坡非法使用 Surfcam 间谍软件。一位公司发言人报告称，他们“对 Surfcam 的使用情况进行了审计并且已将其禁止”，并将该监视工具的使用扣在前任首席执行官头上，也就是被驱逐的 Travis Kalanick。
  </p>
  <p class="graf graf--p">
   值得注意的是，在澳大利亚政府为乘车共享系统的合法运营建立监管框架时 Gocatch 还没有启动；相比之下，优步在实践合法化之前就已经进入了市场。
  </p>
  <p class="graf graf--p">
   优步在内部开发 Surfcam 这种东西只是众多隐秘的肮脏技巧中的一个。2017年该公司就运行了一个叫
   <a class="markup--anchor markup--p-anchor" data-href="https://boingboing.net/2017/03/04/sounds-legit.html" href="https://boingboing.net/2017/03/04/sounds-legit.html" rel="noopener noreferrer" target="_blank">
    Greyball
   </a>
   的类似的东西。
  </p>
  <p class="graf graf--p">
   GoCatch 的联合创始人兼首席执行官安德鲁·坎贝尔告诉 Four Corners，当 GoCatch 幸存下来时，优步的策略破坏了该公司。“优步利用黑客技术窃取我们的数据和我们的司机，这简直骇人听闻，”他说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这种间谍活动对我们的业务产生了巨大的影响，并且为澳大利亚经济和澳大利亚企业树立了一个非常危险的先例。如果不制止它将暗示每个在澳大利亚做生意的跨国公司都可以遵循同样的做法。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “作为一家澳大利亚本土小企业、一家总部位于澳大利亚的技术创业公司正在提高出租车行业的效率和服务水平，然而却让一家美国公司跑到澳大利亚来做这种事……这真令人作呕。对 GoCatch 和其他业务造成的损害是巨大的，坦率地说它应该已经被停止。“
  </p>
  <p class="graf graf--p">
   提醒注意，该公司能用间谍软件监视对手公司就能监视任何人。您还敢使用它吗？
  </p>
