  <p class="graf graf--p">
   中国广泛的监视系统令人不安的内幕已经曝光。（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%B9%94%E6%B2%BB%E6%8B%9C%E7%99%BB%E7%9A%84%E5%84%BF%E5%AD%90%E6%8A%95%E8%B5%84%E4%B8%AD%E5%9B%BD%E8%80%81%E5%A4%A7%E5%93%A5%E7%9B%91%E8%A7%86%E6%8A%80%E6%9C%AF/" href="https://www.iyouport.org/%E4%B9%94%E6%B2%BB%E6%8B%9C%E7%99%BB%E7%9A%84%E5%84%BF%E5%AD%90%E6%8A%95%E8%B5%84%E4%B8%AD%E5%9B%BD%E8%80%81%E5%A4%A7%E5%93%A5%E7%9B%91%E8%A7%86%E6%8A%80%E6%9C%AF/" rel="noopener noreferrer" target="_blank">
    在这里看到中文版
   </a>
   ，文末有推荐解决方案）人权观察的报告详细介绍了中国警方在新疆使用的大规模监视应用程序，这里据说有数十万人被政治“再教育”。
  </p>
  <p class="graf graf--p">
   维吾尔人以惊人的速度被任意拘留，部分原因是这种监视APP的广泛覆盖范围。
  </p>
  <p class="graf graf--p">
   该应用程序汇总来自众多系统和来源的监视数据，包括闭路电视摄像机、本地窃听报告、以及像便利店一般无处不在的警察检查站。
  </p>
  <p class="graf graf--p">
   实际上，它允许警察监视中国公民的一举一动。
  </p>
  <p class="graf graf--p">
   当数据通过应用程序时，它会筛选和分析所谓的可疑活动。被认为“可疑”的活动中包括很多日常生活的简单活动。
  </p>
  <p class="graf graf--p">
   国际社会迅速谴责中国任意大规模拘留维吾尔人的行为。
  </p>
  <p class="graf graf--p">
   但是，国际社会对其他可能遭受中国利用新兴技术为工具进行压迫的人口，关注相对较少。
  </p>
  <p class="graf graf--p">
   无疑，中国对压迫性监视技术的拥抱肯定不会仅仅影响其穆斯林人口。另外一个特别脆弱的群体是朝鲜难民。
  </p>
  <p class="graf graf--p">
   虽然目前在中国的脱北者总数不详，但有些人估计目前仍有
   <a class="markup--anchor markup--p-anchor" data-href="https://www.refworld.org/pdfid/47a6eba20.pdf" href="https://www.refworld.org/pdfid/47a6eba20.pdf" rel="noopener noreferrer" target="_blank">
    10万至30万
   </a>
   脱北者躲在中国。
  </p>
  <p class="graf graf--p">
   大多数人最终能在韩国寻求庇护，在那里他们可以自动获得公民身份。
  </p>
  <p class="graf graf--p">
   然而，几乎所有来自朝鲜的难民都必须通过中国才能在韩国获得自由。
  </p>
  <p class="graf graf--p">
   这些难民依靠地下网络 — 主要由基督教传教士和走私者组成 — 开启一段危险旅程。
  </p>
  <p class="graf graf--p">
   逃离残酷的金家政权的可能性，取决于匿名、隐身和地下系统的使用。由于北京方面同平壤达成协议，遣返在中国境内发现的所有朝鲜人，因此脱北者的隐身至关重要。
  </p>
  <p class="graf graf--p">
   联合国正式将朝鲜脱北者归类为难民。因此，北京遣返朝鲜难民在技术上违反了中国对“联合国难民公约”的承诺。
  </p>
  <p class="graf graf--p">
   然而多年来，中国一直在遣返朝鲜脱北者，认为他们是经济移民，而不是难民。
  </p>
  <p class="graf graf--p">
   最近，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hrw.org/news/2019/05/14/china-protect-7-north-koreans-fleeing-oppression" href="https://www.hrw.org/news/2019/05/14/china-protect-7-north-koreans-fleeing-oppression" rel="noopener noreferrer" target="_blank">
    七名朝鲜逃亡者正面临着中国的遣返
   </a>
   —— 其中一人是一名10岁的女孩，她的母亲和父亲已经在韩国获得了自由。
  </p>
  <p class="graf graf--p">
   难民一旦返回平壤，将面临严重后果。
  </p>
  <p class="graf graf--p">
   联合国调查委员会2014年的一份报告记录了从中国被遣返回朝鲜的人所遭受的系统性镇压。
  </p>
  <p class="graf graf--p">
   大多数人被扔进普通的囚犯营地（교소））或政治监狱（관리소），他们很可能会遭受酷刑、营养不良和强迫劳动。
  </p>
  <p class="graf graf--p">
   许多怀孕的朝鲜妇女被迫流产，往往没有麻醉，有时候是让一名士兵站在孕妇的肚子上。
  </p>
  <p class="graf graf--p">
   如果孩子在堕胎中存活下来，母亲可能会被迫看着自己的孩子被窒息死亡。所有被遣返的难民的境遇都是残酷的，但对妇女来说尤其严重。
  </p>
  <p class="graf graf--p">
   大多数朝鲜难民得到了基督教传教士的支持。
  </p>
  <p class="graf graf--p">
   接受传教士帮助或在中国皈依基督教的朝鲜人，一旦被遣返会面临特别苛刻的待遇。
  </p>
  <p class="graf graf--p">
   调查委员会指出，难民经常会被询问是否与基督教传教士有过接触。基督徒会在朝鲜遭受独特的迫害。另一个组织，Open Doors USA，已经确定平壤是世界上最糟糕的基督徒迫害者。
  </p>
  <p class="graf graf--p">
   国际组织已经开始关注中国日益庞大的监视工程所带来的深远影响。
  </p>
  <p class="graf graf--p">
   除了对中国公民个人隐私构成的威胁之外，值得认真考虑这些政策的次要影响，包括它们可能对朝鲜难民产生的影响。
  </p>
  <p class="graf graf--p">
   注：这是 forbes 的文章。我们依旧认为监视本身就是迫害，不论受害者是谁。在其中区分“最脆弱”和“不怎么脆弱”的意义并不大。
  </p>
  <p class="graf graf--p">
   当然，我们绝对必然关心维族公民和脱北难民的人身安全，因为他们会直接因遭受监视而导致被残害；但我们认为需要有更多人站起来反抗才能结束这种所有人正在面对的威胁。
  </p>
  <p class="graf graf--p">
   这篇报道中提到的隐身，对所有人来说都很有必要；尤其是反抗者，反抗者的成功才是终结1984的希望。我们将在下个月开启新的系列话题，专门讲述隐身的小技巧。
  </p>
