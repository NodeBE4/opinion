   从”小房”运动中汲取经验

 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      <strong>
       给 NGO 组织一些灵感
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   几年前，一些住在北京高层公寓里的老年人谈起曾经居住的四合院生活，他们十分感慨，他们感觉到冷冰冰的公寓失去了邻里街坊之间互相协作的许多乐趣。
  </p>
  <p class="graf graf--p">
   近年来被日本政府高度重视的问题之一就是年轻人的宅文化，他们远离社区、逃避社会，躲进自己的小空间里，把公民义务锁在了门外……
  </p>
  <p class="graf graf--p">
   如今几乎每个人都羡慕“大house”，但正是这些一应俱全的大房子让人们彼此孤立起来了。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4245 jetpack-lazy-image" data-lazy-sizes="(max-width: 1020px) 100vw, 1020px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/小房子.png?resize=1020%2C684&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/小房子.png?w=1020&amp;ssl=1 1020w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/小房子.png?resize=300%2C201&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/小房子.png?resize=768%2C515&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/小房子.png?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="684" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/小房子.png?resize=1020%2C684&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1020"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4245" data-recalc-dims="1" height="684" sizes="(max-width: 1020px) 100vw, 1020px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/小房子.png?resize=1020%2C684&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/小房子.png?w=1020&amp;ssl=1 1020w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/小房子.png?resize=300%2C201&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/小房子.png?resize=768%2C515&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/小房子.png?resize=272%2C182&amp;ssl=1 272w" width="1020"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   我从“小房运动”中了解到一些令人惊讶的事。
  </p>
  <p class="graf graf--p">
   关于“小房运动”的陈述经常传达个人主义运动方面的意识，例如人们自己建造房屋或渴望在树林中隐居。
  </p>
  <p class="graf graf--p">
   这些故事与自给自足的新自由主义理想是一致的，将自己与更大的世界隔离开来。事实上并非如此。
  </p>
  <p class="graf graf--p">
   虽然这可能意味着那些在小房屋建筑和生活方面拥有更多经验的人可以更好地实现目标，但这也不是最重要的问题。
  </p>
  <p class="graf graf--p">
   相反，我从小房子的居民、建筑商、该运动的的倡导者和领导者那里学到了更多的东西，以及与我的家人一起尝试小房子生活的乐趣，
   <strong class="markup--strong markup--p-strong">
    真正重要的是一个小房子可以帮助促进些什么。
   </strong>
  </p>
  <p class="graf graf--p">
   在个人层面上，这可能意味着缩小规模（从更大的空间转移到更小的空间）、减少债务、增加储蓄、降低负担（能够减少无聊工作的时间以便参与更有意义的工作和活动） ，并在购物和消费之外创造意义。
  </p>
  <p class="graf graf--p">
   但是，在一个更具集体性和结构挑战性的层面上，小房子显然可以支持社区建设、反贫困和抵制住房不安全，甚至可以对威胁人民社区和生活方式的资本主义产业提出挑战。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    社区建设
   </strong>
  </p>
  <p class="graf graf--p">
   小房屋的住户必须和周边人互相依存，这就为社区的组织形成了一个天然便捷的环境。任何一个小房屋的住户都不能完成所有事，只有彼此联合起来才能做到。
  </p>
  <p class="graf graf--p">
   这实际上是好事。
  </p>
  <p class="graf graf--p">
   那些只有少数人住的大房子街区经常有各种各样的设施 —— 家庭健身房、家庭影院、有很多书的图书馆、超市 —— 所以住在里面的人根本没有必要走出来。
  </p>
  <p class="graf graf--p">
   如今的人们往往乐于拥护庆祝独立的文化，并将需要他人帮助的文化比作个人缺陷或弱点。
  </p>
  <p class="graf graf--p">
   但是，当你去当地的教会，电影院或公共图书馆时，就意味着走出了自己的社区，打破了隔离墙并有更多机会与他人建立关系。
  </p>
  <p class="graf graf--p">
   这是一种反对孤立感、疏离感、对陌生人的恐惧感、以及各种其他感受的重要方式，正是这些感受一直以来令人们深居简出缩在自己的家里。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    消除贫困和住房不安全
   </strong>
  </p>
  <p class="graf graf--p">
   小房子也起到了抵御贫困和住房不安全的作用。
  </p>
  <p class="graf graf--p">
   底特律是个典型案例，CASS 社区服务在这个城市解决了贫困和缺乏体面的经济适用房等重大问题。他们正在建造小房子，并将这些房子提供给低收入人群和曾经无家可归的人。
  </p>
  <p class="graf graf--p">
   他们不仅建造房屋，还在创造社区，他们为居民提供了摆脱住房不安全和贫困的重要途径。
  </p>
  <p class="graf graf--p">
   他们决定在地基上而不是轮子上建造小房子，这是有意而为的。就如中国的拆迁一样，底特律许多居民面临着严重的住房不安全感，可能已经习惯于随时拎起行李就可以离开不安全的地方。
  </p>
  <p class="graf graf--p">
   因而他们很难组织成任何社区。
  </p>
  <p class="graf graf--p">
   而在地基上建造的房子本身就在宣誓永久性和成为社区一部分的机会。
  </p>
  <p class="graf graf--p">
   在支付与房屋相关的账单满七年后（每月每平方英尺1美元），居民就可以直接拥有他们的房子。
  </p>
  <p class="graf graf--p">
   想象一下，对于居无定所的人来说拥有自己的房产是多么伟大的梦想。该计划及这些实物小房子正在为摆脱贫困的途径提供可能性。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-8842 jetpack-lazy-image" data-lazy-sizes="(max-width: 896px) 100vw, 896px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/小房子.png?resize=896%2C802&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/小房子.png?w=896&amp;ssl=1 896w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/小房子.png?resize=300%2C269&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/小房子.png?resize=768%2C687&amp;ssl=1 768w" data-recalc-dims="1" height="802" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/小房子.png?resize=896%2C802&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="896"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8842" data-recalc-dims="1" height="802" sizes="(max-width: 896px) 100vw, 896px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/小房子.png?resize=896%2C802&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/小房子.png?w=896&amp;ssl=1 896w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/小房子.png?resize=300%2C269&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/小房子.png?resize=768%2C687&amp;ssl=1 768w" width="896"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    小房运动作为社会抗议
   </strong>
  </p>
  <p class="graf graf--p">
   BC省的 Secwepemc Nation 采用了建造在轮子上的小房子的想法，并利用这些房子作为物理屏障，抵制拟议的跨山扩建管道通过他们的领土。
  </p>
  <p class="graf graf--p">
   该管道将​​从阿尔伯塔省埃德蒙顿运送原油到BC省的本拿比，对水路、鱼类和动物构成污染威胁，并威胁到作为 Secwepemc 人祖祖辈辈居住的土地。
  </p>
  <p class="graf graf--p">
   这些管道只会继续助长产能过剩、过度消费和浪费。
  </p>
  <p class="graf graf--p">
   不仅仅是一个物理屏障，这些小房子和为住户提供能源的太阳能电池板表明，我们需要创造可持续生活的机会。
  </p>
  <p class="graf graf--p">
   对于 Secwepemc 人来说，清洁的水和保护家园必需作为优先事项，高于不可持续的能源生产和运输，显然，最广泛意义上就是社区重塑了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    小房子=大变化
   </strong>
  </p>
  <p class="graf graf--p">
   我们目前面临着无数的社会问题：异化、住房不安全和无家可归，威胁土著社区的发展，更有威胁我们在地球上存在的气候变化问题。我并不是说小房子会能解决我们面临的所有问题，而是说我们需要寻找并接受挑战这些问题的创造性方法。
  </p>
  <p class="graf graf--p">
   最终，这些小房子将展示了我们如何在最广泛和最重要的意义上捍卫家园。
  </p>
