  <p class="p1">
   <span class="s2">
    “

   <span class="s1">

   </span>

   <span class="s2">

   </span>
  </p>
  <p class="p1">
   <span class="s2">
    “

   </span>
   <span class="s2">
    ”
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    事实证明，这位管理员和太阳报中引用的其他消息来源甚至比当时公开的消息更为准确。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    就在前一个月，国家安全局已经经历了一次大规模的停电，并且第一次被迫将其最关键的监控设施
   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">
    国家安全行动中心
   </span>
   <span class="s2">
    ” ——
   </span>
   <span class="s1">
    其神经中枢
   </span>
   <span class="s2">
    ——
   </span>
   <span class="s1">
    转移到了位于佐治亚州奥古斯塔的备用设施，该消息来自一份内部报告，被标记为
   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">
    机密
   </span>
   <span class="s2">
    ”
   </span>
   <span class="s1">
    。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    根据内部网站
   </span>
   <span class="s2">
    SIDtoday
   </span>
   <span class="s1">
    上发布的一篇文章，这场灾难的罪魁祸首是：炎热的天气和电力公司没有产生足够的电力。
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    对于国家安全局来说，这个相对平稳的交接是一个胜利。但是，
    <b>
     这一事件标志着一个重要的转折点，强调了国家安全局如何为其监视设施收集了太多信息
    </b>
    。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    该机构将继续在犹他州荒芜的沙漠地区建立一个庞大的数据中心，估计能够保存数十亿
   </span>
   <span class="s2">
    GB
   </span>
   <span class="s1">
    的信息。
   </span>
  </p>
  <p class="p4">
   <span class="s3">
    事实上，
   </span>
   <span class="s1">
    2006
   </span>
   <span class="s3">
    年

   <span class="s1">

   <span class="s3">

   </span>

    </b>
   </span>
   <span class="s2">

    </b>
   </span>
   <span class="s1">

    </b>
   </span>
   <span class="s2">
    <b>
     ”
    </b>
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    1966
   </span>
   <span class="s1">
    年，一个新的国家安全局项目开始出现，以确定为什么受监视的电子信号
   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">

   </span>
   <span class="s2">
    ”
   </span>
   <span class="s1">

   </span>
   <span class="s1">

   </span>
  </p>
  <p class="p1">
   <span class="s1">
    虽然技术本身以及
   </span>
   <span class="s2">
    NSA
   </span>
   <span class="s1">
    的使命将在未来
   </span>
   <span class="s2">

   </span>

    </b>

   </span>
