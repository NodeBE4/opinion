  <p class="graf graf--p">
   弗洛伊德认为，人类心理包含意识和无意识两个部分。两者都受到各自规律的控制。
  </p>
  <p class="graf graf--p">
   为了说明这一点，弗洛伊德使用了冰山的比喻：从外表看，观察者只能看到冰山露出水面的那一小块，但其90%的部分都隐藏在水下。同样，
   <strong class="markup--strong markup--p-strong">
    大部分的人类心理超越了意识，并且会隐秘地影响人的感受、思想和行为
   </strong>
   。
  </p>
  <p class="graf graf--p">
   弗洛伊德认为，无意识中存在着被压抑的欲望、感受和想法。造成这些压抑的情绪往往也会被转移。因此，人们可以通过忽视和遗忘来解决内心冲突。
  </p>
  <p class="graf graf--p">
   心理治疗师确认过很多起心因性失忆症的案例，即 病人会忘记他们经历过的极端境遇。对某些事的情绪变得无法忍受时，人们就会陷入遗忘状态。
  </p>
  <p class="graf graf--p">
   弗洛伊德的理论所引发的争议已经持续了一百多年。一些心理学家指责人类精神分析并不科学，而且没有经过实验证实；另一些专家则认识到了它的价值，并成功地将其用于临床实践。
  </p>
  <p class="graf graf--p">
   弗洛伊德认为，意识和无意识中被转移的内容之间的冲突会导致精神障碍，包括 焦虑、抑郁和强迫症。
  </p>
  <p class="graf graf--p">
   弗洛伊德提出，可以用精神分析法来治疗这些精神失调症。
  </p>
  <p class="graf graf--p">
   精神分析法即通过于精神分析专家进行特殊对话，以帮助人们了解自身内部冲突的原因。弗洛伊德断言，通过从无意识中找回导致痛苦症状的根源，足以缓解精神紧张并且达到治愈效果。
  </p>
  <p class="graf graf--p">
   他创造了几种分析无意识的方法。其中一种就是
   <strong class="markup--strong markup--p-strong">
    自由联想法
   </strong>
   ：病人自由地诉说心中想到的任何东西。
  </p>
  <p class="graf graf--p">
   在某些时候，病人的独白会中断，就像遇到某些障碍物那样。这意味着他们已经触碰到了意识中被转移的重要信息，并且已经回想起了那段记忆。
  </p>
  <p class="graf graf--p">
   另一种方法是
   <strong class="markup--strong markup--p-strong">
    释梦
   </strong>
   ，梦中的人物和情节显示了无意识的内容。通过解释梦境，分析师可以帮助病人了解被转移的感受、欲望和记忆。
  </p>
  <p class="graf graf--p">
   这个转移理论的重要结论是：人们往往会忘记不愉快的事情。人们经常会将那些令他们产生内疚和羞愧情绪的行为从记忆中抹去，比如 犯罪、欺骗和冷酷无情。
  </p>
  <p class="graf graf--p">
   人们会忘记小的债务、请求、命令、以及未履行的承诺。
   <strong class="markup--strong markup--p-strong">
    有些时候，人的记忆会对某些事情进行部分修改
   </strong>
   。比如，一个人会记住一种情境，记得别人的行为和话语，却不记得他自己做了什么。
  </p>
  <p class="graf graf--p">
   👉 调查人员经常不得不处理创伤性事件后被转移的记忆。为了帮助某个人回忆起这些内容，你可以使用精神分析法的技巧。
  </p>
  <p class="graf graf--p">
   建议对方说出提到这一事件时他脑海里能想到的所有东西，就算缺乏直接联系也没关系。分析这些独白。注意可能与事件直接或间接相关的所有内容。
  </p>
  <p class="graf graf--p">
   一旦对方终止诉说，或者完全停了下来，请他/她告诉你此刻正在想些什么。给对方以支持，明确表示你不会因为他/她告诉你的任何事而对其进行责备。让对方完全放松。
  </p>
  <p class="graf graf--p">
   从一个联想转到另一个联想。某些表达出来的想法很可能会提供你想要的信息。这一方法也可以对你自己使用。
  </p>
  <p class="graf graf--p">
   👉 这很重要，对于本文后面的内容来说。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     虚假记忆
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   什么是记忆？它是资料库还是资料再现的过程？
  </p>
  <p class="graf graf--p">
   现代心理学认为记忆是一个过程。每次触及过去的记忆时，我们都会对其进行再现。一方面，这意味着我们可以恢复被遗忘的信息：利用记忆中留存的碎片，可以重新创建整个图景。另一方面，
   <strong class="markup--strong markup--p-strong">
    每一次新的再现都将改变记忆
   </strong>
   。
  </p>
  <p class="graf graf--p">
   法国心理学 Jean Piaget 描述了自己生活中的一个案例。他记得大约两岁时，有人试图将他从婴儿车中偷走。Piaget 能够描述绑架现场的情形，他的保姆如何阻止了犯罪分子，以及犯罪分子在看到警察后如何逃跑。
  </p>
  <p class="graf graf--p">
   一直到15岁之前，他都对自己的这段记忆充满信心。但15岁那年，他的保姆开始信教，写信给 Jean 的父母并告诉他们，这个故事从头到尾都是她编造的。Jean 可能多次听到他的父母重新演绎这一传奇故事，
   <strong class="markup--strong markup--p-strong">
    直到自己将它进行视觉化再现
   </strong>
   ，并记忆为 “真实事件”。
  </p>
  <p class="graf graf--p">
   这个例子表明，
   <strong class="markup--strong markup--p-strong">
    一个人在记忆虚假事件的同时，也可以绝对确信自己记忆的可靠性。
   </strong>
  </p>
  <p class="graf graf--p">
   童年的记忆往往是不真实的。这可以归因于心理学家称之为 “童年失忆症” 的现象，即 人们会忘记自己生命中最初几年的经历。
  </p>
  <p class="graf graf--p">
   有好几种理论都涉及这一主题：从精神分析学到神经生物学。最具说服力的是对于与抽象思维和语言发育相关的长期记忆发育解释：儿童尚未学会足够多的概念，因而无法记住当时发生的事。
  </p>
  <p class="graf graf--p">
   尽管如此，许多人依旧认为自己记得童年早期出现的面孔、玩具、电视剧和事件。但是，一般来说，这些记忆是基于大人们提供的照片和故事形成的，也就是说，
   <strong class="markup--strong markup--p-strong">
    它不是自己的记忆，而是对他人描述的重新构建
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是，不要认为虚假记忆只是填补了童年失忆症相关的空白；
   <strong class="markup--strong markup--p-strong">
    它们也可能在认知错误的情况下出现
   </strong>
   ：👉人们只看到自己想要看到的东西，却忽视了与个人信仰和立场相矛盾的事物，并根据个人狭隘的信仰完成了整个图景。
  </p>
  <p class="graf graf--p">
   一个相信大多数犯罪分子都是无家可归者证人，会将犯罪分子描述为无家可归者。他们可能不会注意到富人的特征，而会用其他细节来对犯罪分子的描述进行补充。
  </p>
  <p class="graf graf--p">
   👉 他们会根据自己的刻板印象再现整个事件，同时也确信自己对所有事情的记忆 “准确无误”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    虚假记忆的另一个来源是顺从性。这是将态度、信仰和行为与群体社会规范相匹配的一种行为。
   </strong>
  </p>
  <p class="graf graf--p">
   许多心理学实验表明，人们在偶像的影响下，或者权威的压力下，会变得容易改变自己的观点。有些人甚至准备否认某些显而易见的事实 —— 包括在其他人故意 “证实” 一个谎言时，被试也跟着认为 “那就是真的”。
  </p>
  <p class="graf graf--p">
   👉对于那些容易受到他人影响的人来说，他们的记忆可能是受到他人暗示而形成的；如果这些暗示是来自权威阶层，其效应就更不用说了。
  </p>
  <p class="graf graf--p">
   但要知道，
   <strong class="markup--strong markup--p-strong">
    虚假的记忆和彻头彻尾的谎言是两码事
   </strong>
   。
  </p>
  <p class="graf graf--p">
   一个人在再现记忆的时候，他是真心地误会了。他自信地说着，因为他相信自己所说的话。他的记忆往往有一种内在的逻辑。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    调查人员经常会遇到虚假记忆的情况，因此原则是：始终核实
   </strong>
   。
  </p>
  <p class="graf graf--p">
   将真相与虚假分开的第一种方法是，比较不同来源的信息。如果来自不同的人的信息都彼此一致，那么这个信息就可能是正确的。如果信息出现差异，您需要评估哪个来源更可靠。
  </p>
  <p class="graf graf--p">
   另一种分离真实记忆的方式是，从始至终彻底地查明对方真正看到和听到的内容，以及他们在整个过程中的想法。
  </p>

   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">

    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在一个活动（庆典、派对、野餐）结束几天后，询问所有活动参与者当时都有哪些人在场？大家各自穿的是什么？每个人分别坐在什么位置？谈了什么内容？桌子上放的是什么？大家都是什么时候到达的？什么时候离开的？
  </p>
  <p class="graf graf--p">
   你会发现，不同人有着不同的版本，有些版本之间存在惊人的差异。
  </p>
  <p class="graf graf--p">
   如果可能的话，看看活动当天的照片和视频，并以此评估记忆的可靠性。

  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">

    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   探索你的童年时期最鲜活的记忆。将它们和父母及亲属的记忆中的版本相比较。查看当时的家庭照片和保存的视频。如果你发现了不同之处，试着弄清楚它们是怎么出现的。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">

    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   撰写自传。从出生到现在，逐年回顾自己的个人经历。记住曾经住过的地方，做了什么事，和谁交过朋友。询问认识你的人，将你的记忆和家庭照片或文档进行比较。寻找老朋友和同学，联系以前的同事。
  </p>
  <p class="graf graf--p">
   👉这是帮助行动者撰写令人信服的掩护身份的一个绝佳练习。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     操纵记忆
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   记忆的脆弱性可能会让我们对证人提供的证词和信息产生怀疑。由
   <strong class="markup--strong markup--p-strong">
    于误会或者别人特意灌输的暗示，回忆可能是虚假的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   👉 有经验的律师知道如何让目击证人的证词变得不可信 —— 通过询问与细节有关的复杂问题，他们让证人心中产生怀疑，从而得出结论：证人的所有证词都不可靠。
  </p>
  <p class="graf graf--p">
   比如，在被问到劫匪加速开往银行所驾驶的那辆雪铁龙汽车的颜色时，一名证人回答说“绿色”。在律师的压力下，他记住了汽车的颜色。他并没有看到汽车驶近银行的过程，也无法识别出汽车的样子。
  </p>
  <p class="graf graf--p">
   但是，在律师
   <strong class="markup--strong markup--p-strong">
    诱导性问题
   </strong>
   的帮助下，证人再现了自己的记忆，而且最重要的是，他开始相信律师。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “是的，这是一辆绿色的雪铁龙，它正全速驶向银行。”
  </p>
  <p class="graf graf--p">
   在听取完证词后，律师将其与视频记录进行比较，一辆旧的绿色福特慢慢地开到了银行。这些明显的差异让证人在法庭上的证词变得不可信，并使得他怀疑自己是否真的记得抢劫的情形。
  </p>
  <p class="graf graf--p">
   最重要的是，
   <strong class="markup--strong markup--p-strong">
    证人在指认劫匪时会产生自我怀疑，从而导致后者将有机会逃脱受罚
   </strong>
   。判刑证据和无罪证据都可能出现错误。
  </p>
  <p class="graf graf--p">
   👉 司法史中充满了冤假错案，虚假记忆导致无辜的人受到了有罪判决。儿童的证词尤其令人怀疑。
  </p>
  <p class="graf graf--p">
   强大的想象力和易受到暗示性这些特点，让他们成为可怜的证人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    记忆不是资料仓库，而是通过再现的过程构建的
   </strong>
   。
   <strong class="markup--strong markup--p-strong">
    在这个过程中受到的外部干预会影响结果。我们不仅可以改变或摧毁现有记忆，还可以引入新的记忆。
   </strong>
  </p>
  <p class="graf graf--p">
   实验表明，在大约 30％的情况下，成年人可以虚假地回忆童年生活。在某些案例中，这些记忆开始在他们的生活中 “活” 下来：它们不断变得血肉丰满，生动形象，感情充沛。
  </p>
  <p class="graf graf--p">
   👉 成年时期相关活动的虚假记忆也可能被有意灌输，当谎言与事实混在一起的时候更是如此。
  </p>
  <p class="graf graf--p">
   有时候，针对某个不存在的事件 “提醒” 某人的能力能够对情报员起到帮助作用，对那些法外之徒而言尤为如此。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-12438 jetpack-lazy-image" data-lazy-sizes="(max-width: 1050px) 100vw, 1050px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpeg?resize=1050%2C700&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpeg?w=1050&amp;ssl=1 1050w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpeg?resize=300%2C200&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpeg?resize=1024%2C683&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpeg?resize=768%2C512&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpeg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="700" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpeg?resize=1050%2C700&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1050"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12438" data-recalc-dims="1" height="700" sizes="(max-width: 1050px) 100vw, 1050px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpeg?resize=1050%2C700&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpeg?w=1050&amp;ssl=1 1050w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpeg?resize=300%2C200&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpeg?resize=1024%2C683&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpeg?resize=768%2C512&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-7.jpeg?resize=272%2C182&amp;ssl=1 272w" width="1050"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   对成年记忆进行成功暗示可能需要做更多的工作。
  </p>
  <p class="graf graf--p">
   💡以下是一些小窍门：
  </p>
  <p class="graf graf--p">
   1. 必须在事件发生很久之后进行暗示。
  </p>
  <p class="graf graf--p">
   👉为了在记忆中植入一件从未发生的事，最好让它 “发生” 在三到五年前。如果是植入真实事件的一些细节，时间范围可以缩短到几个月以内。
  </p>
  <p class="graf graf--p">
   2. 关于事件的描述必须做到真实可信。
  </p>
  <p class="graf graf--p">
   👉 必须把谎言和事实混在一起。一定要把一个不存在的事件与某个人生命中其他真实的场景联系起来。
  </p>
  <p class="graf graf--p">
   3. 如果故事包含很多细节，那么它将更具说服力。
  </p>
  <p class="graf graf--p">
   4. 如果故事中包含画面、声音和气味等不同的感官信息，那么它会更容易被人相信。
  </p>
  <p class="graf graf--p">
   5. 最好使用伪造的文件、照片或权威证人的证词来支撑你的故事。
  </p>
  <p class="graf graf--p">
   6. 植入虚假记忆需要花一些时间。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “暗示时间” 之间应该有间隔，这样一来植入的信息就能在目标对象的记忆中得到巩固。
  </p>
  <p class="graf graf--p">
   👉 就算某个人一开始就否认一切，但在间隔期间，他会开始怀疑。这将为后续的暗示创造基础。
  </p>
  <p class="graf graf--p">
   此外，有必要多次让对方重温记忆：他所听到的重复次数越多，暗示越会在他的记忆中深深扎根。
  </p>
  <p class="graf graf--p">
   7. 如果遭到了拒绝，你的策略应该是关注记忆的不可靠性：“请尽量去回想”“你只是忘记了，那不是真实的情况”、“这不可能，因为……” 等等。
  </p>
  <p class="graf graf--p">
   ⚠️ 诱导性问题的设计之妙就在于，它们已经包含了正确的答案，这也有一定帮助：“他没有胡子吗？” “那是在晚上。还记得吗，路灯亮着吗？”
  </p>
  <p class="graf graf--p">
   记忆操纵可以是完全无害的 — — “圣诞老人给你带来了一份礼物！” 但是，它也可能是非常危险的：
   <strong class="markup--strong markup--p-strong">
    政治宣传往往是谎言，试图重写一个国家的历史 — — 即 其公民的集体记忆。
   </strong>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在你的一个熟人头脑中植入一段虚假的记忆。
  </p>
  <p class="graf graf--p">
   比如，派对上出现的一个有趣的或者尴尬的状况。编造一个详细的故事，并制定暗示策略。
  </p>
  <p class="graf graf--p">
   如果有可能，准备 “证人” 和 “重要证据”。
  </p>
  <p class="graf graf--p">
   第一次被拒绝后，请不要放弃。让你的目标人物记住这个故事，并向他们提供更多的细节。
  </p>
  <p class="graf graf--p">
   也许这个故事会比你一开始想象的更加丰富多彩和生动有趣。
  </p>
  <p class="graf graf--p">
   要当心：在实验对象身上植入虚假记忆可能会伤害对方的心灵。一旦熟人开始相信你的故事了，马上承认你之前一直在玩恶作剧。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-12439 jetpack-lazy-image" data-lazy-sizes="(max-width: 1050px) 100vw, 1050px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-5.jpeg?resize=1050%2C700&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-5.jpeg?w=1050&amp;ssl=1 1050w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-5.jpeg?resize=300%2C200&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-5.jpeg?resize=1024%2C683&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-5.jpeg?resize=768%2C512&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-5.jpeg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="700" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-5.jpeg?resize=1050%2C700&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1050"/>

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     欺骗测谎仪
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   90％的测谎结果取决于操作者的知识和经验。
  </p>
  <p class="graf graf--p">
   首先，测试对象的反应取决于他被问到的问题。这些问题应该事先好好准备，并按照正确的顺序安排。
  </p>
  <p class="graf graf--p">
   其次，操作者需要了解谎言在个人情绪反应中是如何表现的。毕竟，由于害怕测试程序本身，测试对象即使在讲真话时也会担心不已。
  </p>
  <p class="graf graf--p">
   操作者最终决定测试对象说的到底是真相还是谎言，而这一断言在很多方面都是
   <strong class="markup--strong markup--p-strong">
    主观的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在解释生理指标时，操作员可能会犯以下两个错误：宣布虚假证词是真相，或者指责诚实的测试对象在撒谎。
  </p>
  <p class="graf graf--p">
   测谎仪的测试质量很大程度上取决于测试者所提出的问题。
  </p>
  <p class="graf graf--p">
   👉 一般而言，问题分为两类：相关性问题和控制性问题。第一类问题与测试目标直接相关。第二种类型有助于理解测试对象在说谎时身体会发生什么反应。选择控制性问题是为了让测试对象被迫撒谎，以维护自己的声誉。
  </p>
  <p class="graf graf--p">
   比如，相关性问题是这样的：“你是否曾向未经授权方透露过机密信息？”控制性问题则是这样的：“你是否曾挪用过他人财物？”
  </p>
  <p class="graf graf--p">
   一般来说，我们挪用他人财物的次数至少也有一次。但是，由于不想被人看作是小偷，测试对象可能会说谎。如此就会引发情绪反应，而
   <strong class="markup--strong markup--p-strong">
    这些反应在分析同一对象对相关性问题的反应时是必需的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   👉 由于情绪非常个人化，因此操作员解释的并非反应本身，而是不同反应之间的差异。
  </p>
  <p class="graf graf--p">
   如果测试对象对于相关性问题的反应强于其对于控制性问题的反应，那么他很可能是害怕被判有罪。如果像大多数人一样是相反的结果，就说明他关心的是维护自己的声誉。
  </p>
  <p class="graf graf--p">
   💡 测谎仪可能会被人骗，优秀的情报人员知道如何欺骗仪器。
   <strong class="markup--strong markup--p-strong">
    有很多方法可以通过机器测试，行家们会将它们进行组合使用
   </strong>
   。
  </p>
  <p class="graf graf--p">
   1、
   <strong class="markup--strong markup--p-strong">
    机械法
   </strong>
   是最简单的。你需要的就是引发疼痛，比如咬舌头或者脸颊，用力弯曲脚趾，或者将脚趾插入鞋子中的针上。
  </p>
  <p class="graf graf--p">
   疼痛和肌肉紧张会扰乱生理参数，让操作者难以判断。
  </p>
  <p class="graf graf--p">
   但是，有经验的检测员能在测试过程中注意到机械阻力，而这会对测试对象非常不利。
  </p>
  <p class="graf graf--p">
   2、
   <strong class="markup--strong markup--p-strong">
    药物法
   </strong>
   。服用镇静剂或兴奋剂，干扰检测者用来区分真相与谎言的数据。
  </p>
  <p class="graf graf--p">
   运用这种方法时要小心：使用精神活性物质可能会因为瞳孔缩小或放大，或者脉搏或肤色发生改变而露出马脚。
  </p>
  <p class="graf graf--p">
   此外，这种方法还存在药物过量的风险。
  </p>
  <p class="graf graf--p">
   3、
   <strong class="markup--strong markup--p-strong">
    行为法
   </strong>
   ：毫无分寸的亲密动作，与检测者闲聊，重复提问，表现得很愚蠢。过度兴奋会扰乱生理指标。
  </p>
  <p class="graf graf--p">
   如果判断是否说谎的依据是对相关性问题和控制性问题的不同反应，那么你可以对所有问题或者检测员说的其他事情都做出更加强烈的反应，这样一来问题就迎刃而解了。
  </p>
  <p class="graf graf--p">
   过于熟悉或放松的行为也会让检测员的工作更加困难。
  </p>
  <p class="graf graf--p">
   4、
   <strong class="markup--strong markup--p-strong">
    心理法
   </strong>
   是一组基于心理自我调节的技巧。
  </p>
  <p class="graf graf--p">
   这些技巧包括从瑜伽和其他东方习俗中借鉴过来的放松技巧，以及心算或对所有问题都增强反应等这些复杂的分散注意力的方法。
  </p>
  <p class="graf graf--p">
   这组方法非常有效，但需要认真准备。
  </p>
  <p class="graf graf--p">
   合理化这一方法也很好用。在了解测试目的之后，测试对象可以提前做好准备。为了不在相关性问题上表现出焦虑或兴奋之情，可以在脑海中为自己想掩饰的行为找一个合理的解释。
  </p>
  <p class="graf graf--p">
   比如，你泄漏了一个秘密，可以被重新**在内心**解释为这一信息本不应该被保密，泄漏它是出于公众利益目的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    设计恰当的合理化理由非常有效，因为它让人对自己的行为充满信心，并解决了谎言所造成的内部冲突
   </strong>
   。
  </p>
  <p class="graf graf--p">
   👉正因此，测谎仪无法识别虚假的记忆：测试对象相信自己说的绝对是真话。
  </p>
  <p class="graf graf--p">
   欺瞒测谎仪的这些方法没有一种是完全可靠的，因为没有一种绝对准确的方法来解释人体生理指标。
  </p>
  <p class="graf graf--p">
   但只有知识和训练才能帮助你通过测谎仪测试。
  </p>
  <p class="graf graf--p">
   👉 练习学会管理你的生理指标。你需要脉冲或直流电皮肤电阻传感器。把它们放在手指或耳垂上，然后与监视器相连，然后就能实时追踪自己的生理参数了。
  </p>
  <p class="graf graf--p">
   许多计算机程序可以帮助你学会有意识地管理自己的生理状态。
  </p>
  <p class="graf graf--p">
   这个技能不仅仅适用于应对测谎仪。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     训练— — 反义词
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   观察下列表格。在这个练习中，你需要大声说出与方格中词语意思相反的词，但要忽略表格中已有的内容。
  </p>
  <p class="graf graf--p">
   例如，如果你看到“ light”（明亮的）这个词，你应该说“ dark”（黑暗的）这个词。
  </p>
  <p class="graf graf--p">
   👉 不断重复这个练习以增强自己的洞察力和心理敏捷度。
  </p>

   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     最后
    </strong>

   终于完成了！相信您现在已经取得了很大的进步。

   您花了很多时间进行训练，这些练习都非常有用，您应该坚持。您会发现自己的记忆力在不断提高。同时，您处理信息和与他人沟通的能力都会显著增强。
  </p>
  <p class="graf graf--p">
   不要停留在这个阶段，继续前进，你可以获得更多进步。
   <strong class="markup--strong markup--p-strong">
    限制自己使用电子设备的时间，把省下来的时间用于锻炼自己的记忆力和分析能力 —— 始终保持高水平的心智。
