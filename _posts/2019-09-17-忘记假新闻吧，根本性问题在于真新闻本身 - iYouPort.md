      </strong>
     </em>

   </li>

  </p>
  <p class="graf graf--p">
   忘记假新闻吧，这是一个有毒的术语。真正的新闻本身才是更大的问题。但是却没人关心？
  </p>
  <p class="graf graf--p">
   捍卫新闻自由的组织只关心记者有没有被捕、被杀或者被威胁；监督机构只关心新闻有没有撒谎、有没有收取贿赂报道偏袒的消息；军事政治决策者只关心新闻有没有“泄漏机密”、是否为合格的信息战武器；大众只关心新闻是不是刺激，能不能提供一些龙门阵话题……
  </p>
  <p class="graf graf--p">
   然而很少有人会关心新闻业本身的问题 —— 以上这些都不是新闻业应有的职能、甚至不是底线所在；而正因为新闻本身出了问题，新闻业的职能和底线才被扭曲成现在的样子。
  </p>
  <p class="graf graf--p">
   这就是荷兰新闻平台 De Correspondent 想要改变的状况。他们承诺为其读者提供“日常新闻研究的解毒剂”。
  </p>
  <p class="graf graf--p">
   其主创 Rob Wijnberg 曾经是荷兰一家主要报纸的编辑。他准确地抓住了
   <strong class="markup--strong markup--p-strong">
    当代新闻最大的问题，概括起来就是四个词：轰动的、特殊的、否定性的、和时效性的。
   </strong>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter wp-image-4381 jetpack-lazy-image" data-lazy-sizes="(max-width: 700px) 100vw, 700px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1.png?resize=700%2C600&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1.png?w=1054&amp;ssl=1 1054w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1.png?resize=300%2C257&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1.png?resize=768%2C659&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1.png?resize=1024%2C878&amp;ssl=1 1024w" data-recalc-dims="1" height="600" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1.png?resize=700%2C600&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="700"/>
   <noscript>
    <img alt="" class="aligncenter wp-image-4381" data-recalc-dims="1" height="600" sizes="(max-width: 700px) 100vw, 700px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1.png?resize=700%2C600&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1.png?w=1054&amp;ssl=1 1054w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1.png?resize=300%2C257&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1.png?resize=768%2C659&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1.png?resize=1024%2C878&amp;ssl=1 1024w" width="700"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   这四个词正是现代新闻不再有意义的根本性原因。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    轰动的：如果你没能获得媒体认为当有的轰动效应，你就不会被关心；哪怕你的遭遇对他人来说具有确切的警示性效应；
   </li>
   <li class="graf graf--li">
    特殊的：意味着第一个人遭到勒索软件攻击时它值得报道的 ，但勒索软件泛滥时，大批的受害者都会被媒体无视，因为他们“不再特殊”。换个角度看，如果你作恶一次，肯定会被揭露；如果你持续地广泛地作恶 …… 很可能就被无视了。
   </li>
   <li class="graf graf--li">
    否定性的：你有没有注意到准确的分析（如果有的话）永远都是被报道的错误行为的陪衬？抗议者被关注的时候永远都是他们被捕的时候、甚至被杀？在他们真正成功的时候却没人知道他们……
   </li>
   <li class="graf graf--li">
    时效性的：这点无须解释。99%的新闻不会让你在第二天还想看它，剩下的1%不会让你在一周后还觉得它值得说两句。一切都是走马观花的，难以推动对现实的改变。
   </li>
  </ul>
  <p class="graf graf--p">
   就如 Rob Wijnberg 所指出的，从耸人听闻开始，新闻通常是令人震惊的、怪异的丑闻、或离奇的，足以唤起更广泛的评论。
   <strong class="markup--strong markup--p-strong">
    这个道理在互联网时代所有人都能理解
   </strong>
   ，人们会告诉你 “想火吗？你必须吵架，必须拉住名人吵架，不管你说些什么都能迅速提升知名度”。
  </p>
  <p class="graf graf--p">
   但这样究竟有什么意义？是否有人想过？
  </p>
  <p class="graf graf--p">
   以至于新闻基本不是现实，只不过是现实中极为荒诞的一些特例。
  </p>
  <p class="graf graf--p">
   经常有中国的朋友问我们：“欧洲遍地都是难民吗，把本国人的工作全抢走了？” 或者“极右翼是不是每天都横冲直撞？” … 真没有。但为什么人们会这么认为？因为新闻每天都在捕捉那些*特例*，而无视真正的社会。
  </p>
  <p class="graf graf--p">
   漫画家 Matt Wuerker 出色地捕捉到了这一新闻属性：当我们被数以百万计的和平爱好者、反仇恨活动家、倡导团结进步的同胞所包围时，只需要几个新纳粹、圣战者或 KKK 就能填满24小时的新闻周期。
  </p>
  <figure aria-describedby="caption-attachment-4382" class="wp-caption aligncenter" id="attachment_4382" style="width: 1000px">
   <img alt="" class="wp-image-4382 size-full jetpack-lazy-image" data-lazy-sizes="(max-width: 1000px) 100vw, 1000px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、漫画.png?resize=1000%2C817&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、漫画.png?w=1000&amp;ssl=1 1000w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、漫画.png?resize=300%2C245&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、漫画.png?resize=768%2C627&amp;ssl=1 768w" data-recalc-dims="1" height="817" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、漫画.png?resize=1000%2C817&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1000"/>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这不仅扭曲了我们对其他人的看法，新闻也让我们对那些没有例外的影响力视而不见
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这就是为什么我们经常听不到重大事态发展，直到发生极不可能的事……（塔勒布称之为“黑天鹅”）。
  </p>
  <p class="graf graf--p">
   例如，在雷曼兄弟投资银行申请破产之前，2008年的金融危机并没有成为一个巨大的新闻 —— 这是一个极不寻常的事件。
  </p>
  <p class="graf graf--p">
   银行一直在日复一日地将风险置于风险之上危如累卵 —— 但是，正由于发生的事情是逐渐的、没有所谓的新闻价值渴求的“突发性”，于是一直没有被新闻所呈现。直到彻底崩溃。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果它流血，它会导致”是一种新闻流行语。换句话说：好消息不是新闻。因此，那些试图通过新闻了解世界的人很快就会认为这个世界简直就是个地狱 —— 尽管事实恰恰相反。
  </p>
  <p class="graf graf--p">
   更重要的是，这个消息不断让我们觉得人们不可信任：他们犯了欺诈罪、他们腐败了、他们互相窃取、他们撒谎背叛 ……现实情况是，大多数人都是好人，并希望别人做对的事。但这不是新闻，对吗？
  </p>
  <p class="graf graf--p">
   几乎所有新闻都必须是刚刚发生的事。但最近的事情并不是最有影响力的事情，这是很简单的道理。
  </p>
  <p class="graf graf--p">
   世界上的一切都有历史。这段历史在很大程度上决定了为什么今天会发生某些事。
  </p>
  <p class="graf graf--p">
   然而，向人们通报随着时间的推移而增长的权力结构，如 种族主义的历史根源，或提醒人们逐渐变化的社会，如 经济的金融化，对于每日新闻的形式和节奏来说，根本不是自然的。
  </p>
  <p class="graf graf--p">
   最后，根本原因在于新闻主要围绕事件发生。新闻必须有一个钩子，使用新闻行话说就是：“现在报道它而不是以后报道的原因”。这听起来仿佛合理，但是，
   <strong class="markup--strong markup--p-strong">
    这意味着趋势很少会成为新闻。因为趋势不是实例
   </strong>
   。比如气候变化问题，你不能说：“今天气候发生了变化”，尽管它确实如此。
  </p>
  <p class="graf graf--p">
   人们将其称之为日历新闻。即 反复出现的、经常被计划的事件，这些事件可以作为将事物提升到新闻状态的借口。考虑新闻发布会、季度收入、智库报告、纪念日、或总统的推文 …… 这意味着你可以提前提取大部分新闻，使其成为一种根本不是“新”的东西。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4383 jetpack-lazy-image" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图3.png?resize=215%2C253&amp;is-pending-load=1#038;ssl=1" data-recalc-dims="1" height="253" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图3.png?resize=215%2C253&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="215"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4383" data-recalc-dims="1" height="253" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图3.png?resize=215%2C253&amp;ssl=1" width="215"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   当你把所有这些放在一起时，这意味着，
   <strong class="markup--strong markup--p-strong">
    新闻实际上根本无法实现其最大的承诺：告诉人们世界上正在发生的事。跟不用说捍卫民主了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   与虚假新闻形成鲜明对比，
   <strong class="markup--strong markup--p-strong">
    假新闻是误导性的，因为它纯粹是不真实的，而且不难辨认；但是，真正的新闻会以更微妙和根本的方式误导所有人，是持续的长期的误导，难以辨认。它使人们对概率、历史、进展、发展和相关性产生了深刻的偏见。
   </strong>
  </p>
  <p class="graf graf--p">
   俗话说：“如果你不读报纸，你就不知情；如果你读报纸，就会变得越来越糊涂。”
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4384 jetpack-lazy-image" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图4-1.png?resize=234%2C187&amp;is-pending-load=1#038;ssl=1" data-recalc-dims="1" height="187" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图4-1.png?resize=234%2C187&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="234"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4384" data-recalc-dims="1" height="187" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图4-1.png?resize=234%2C187&amp;ssl=1" width="234"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   需要明确的是，这里说的“新闻”并不是指“所有的新闻报道”。
  </p>
  <p class="graf graf--p">
   依旧有很多类型的新闻报道都是透彻的和信息丰富的，仍有成千上万的记者致力于公共服务，他们做了宝贵的工作。
  </p>
  <p class="graf graf--p">
   对新闻的批评并不是对新闻业的整体否定，现在人们普遍理解这句话：那些目前席卷美国和整个世界的对记者的不信任浪潮，很可能受到希望利用媒体怀疑论的政治精英的推动。
  </p>
  <p class="graf graf--p">
   但是，新闻完全可以做得好一点。尤其是在新闻更加无处不在的时代里，它在公民间的日常对话中占据主导地位，在很大程度上决定了政治议程，并严重影响了人们对人类和世界的看法。
  </p>
  <p class="graf graf--p">
   荷兰新闻平台 De Correspondent 致力于改变当下新闻的弊端；它背后的想法很简单：让我们一起重新定义新闻 —— 从耸人听闻到基础。
  </p>
  <p class="graf graf--p">
   当你真的着眼于现实时，你会发现你从来不会通过新闻发现的东西。
  </p>
  <p class="graf graf--p">
   雷曼兄弟的垮台、英国脱离欧盟、以及特朗普的选举，确实是引人注目的特殊事件，但是，
   <strong class="markup--strong markup--p-strong">
    它们也是缓慢的、不引人注目的系统性趋势的结果
   </strong>
   。（如下图）
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter wp-image-4385 jetpack-lazy-image" data-lazy-sizes="(max-width: 700px) 100vw, 700px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图5.png?resize=700%2C782&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图5.png?w=1062&amp;ssl=1 1062w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图5.png?resize=269%2C300&amp;ssl=1 269w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图5.png?resize=768%2C858&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图5.png?resize=917%2C1024&amp;ssl=1 917w" data-recalc-dims="1" height="782" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图5.png?resize=700%2C782&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="700"/>
   <noscript>
    <img alt="" class="aligncenter wp-image-4385" data-recalc-dims="1" height="782" sizes="(max-width: 700px) 100vw, 700px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图5.png?resize=700%2C782&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图5.png?w=1062&amp;ssl=1 1062w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图5.png?resize=269%2C300&amp;ssl=1 269w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图5.png?resize=768%2C858&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图5.png?resize=917%2C1024&amp;ssl=1 917w" width="700"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   现象不是今天而是每天都在发生，因此永远不会形成一个能使他们有资格作为新闻呈现的“钩子”；现象也太日常，无法产生轰动性的头条新闻或点击。
  </p>
  <p class="graf graf--p">
   其结果就是，
   <strong class="markup--strong markup--p-strong">
    人们忽然“惊讶地发现”，某件事已经发生了；但不知道是如何走到这一步的，更无从通过提前警惕而避免它
   </strong>
   。
  </p>
  <p class="graf graf--p">
   De Correspondent 则反其道而行。也就是说，新闻不是“今天的天气”，而是气候；让人们能通过新闻认识和描述我们这个时代真正有影响力的发展。
  </p>
  <p class="graf graf--p">
   用基础和最近的相关来取代耸人听闻。
  </p>
  <p class="graf graf--p">
   现在的大众新闻是一大群读者去了解一两名记者脑子里的东西；而 De Correspondent 则是，一大群读者和一大群记者协作出内容，以令更多人获益。
  </p>
  <p class="graf graf--p">
   例如，数百名教师、学生和学校校长帮助教育记者了解学校正在发生的事；数百名医生、邮递员和铁路职工帮助公共服务记者了解公共部门的问题……人们被问及，“你每天在工作或生活中遇到了什么，尤其是媒体不关心的东西，但你认为它真的应该出现在头版？” 他们的回答往往是记者永远不可能独自发现的东西。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种新闻报道模式，即 记者不仅仅是制作者、读者不仅仅是消费者，最终植根于一种潜在的信念：通过彼此分享知识和经验，可以让世界变得比我们发现的更好
   </strong>
   。 换句话说，De Correspondent 基于对进步的信念。
  </p>
  <p class="graf graf--p">
   这种信念不是毫无根据的希望; 它甚至不是政治立场。相信进步是一个理性的、事实的结论。因为人类的历史是一个进步的历史。
  </p>
  <p class="graf graf--p">
   推动进步的是知识和经验的分享，而不是恐惧和愤怒。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果你不满意当下，那就去尝试你想要的改变
   </strong>
   。
  </p>
  <p class="graf graf--p">
   希望
   <a class="markup--anchor markup--p-anchor" data-href="https://thecorrespondent.com/" href="https://thecorrespondent.com/" rel="noopener noreferrer" target="_blank">
    De Correspondent
   </a>
   的尝试能为中国的公民记者提供一些灵感。⚪️
  </p>
