 <div class="entry-content">
  <ul>
   <li>
    <em>
     你家的所谓智能家居正在读取并操纵你的情绪，它能判断出你究竟是开心还是伤心，或者只是调侃，这是哪家巨头的黑科技？
    </em>
   </li>
  </ul>
  <p class="graf graf--p">
   随着网络扬声器市场的升温，以及谷歌和亚马逊、Harmon Kardon 和 Denon Heos 等高端品牌剥夺 Sonos 和 Apple 的市场份额，这家大型 iPhone 制造商已经开始研发新的 HomePod 扬声器，该产品据称可以钻进你的大脑，判断出你此刻的情绪是悲伤还是快乐，或者只是调侃。
  </p>
  <p class="graf graf--p">
   使用 Siri 语音软件的最初 Apple HomePod 版是一个失败，苹果消费者将其描述为“过于昂贵”并且其功能与其他制造商提供的相比下有限。
  </p>
  <p class="graf graf--p">
   现在苹果提出的新专利显示，这家大型 iPhone 公司正试图阻止其传统忠诚客户从 Apple 生态系统转向第三方产品。
  </p>
  <p class="graf graf--p">
   虽然该专利未提及 Apple 的智能扬声器，但应用程序中有足够的提示将其识别为主要的 HomePod 升级。
  </p>
  <p class="graf graf--p">
   观察家们确信第一个 HomePod 的“圆柱形”造型将继续存在。与第一个 HomePod 的不同之处是它可以显示包含文字的移动视觉信息的“发光二极管阵列。专利特别指出 LED 将位于侧壁，存在至少100个 LED 以显示视觉信息和文本，“包括歌曲标题”。
  </p>
  <p class="graf graf--p">
   ChannelNews 了解到新设备还将识别挥手或拍手等手势。该专利说它可能包含“一个检测三维手势的传感器”。这将是光学接近传感器和电容式接近传感器。
  </p>
  <p class="graf graf--p">
   目前尚不清楚这可能如何起作用，但这可能意味着你可以通过举起一个停止手势来结束音乐播放。
  </p>
  <p class="graf graf--p">
   与 Amazon Alexa 和 Google Voice 一样，Apple正在设计新的 HomePod，以便能够打开电视，该专利指出它可以向电、加热和照明控制器发送指令。
  </p>
  <p class="graf graf--p">
   它还将内置环境传感器来检测二氧化碳。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该专利还表明该设备具有面部识别功能，这意味着必须内置摄像头
   </strong>
   。Alexa 和谷歌都可以这样做。
  </p>
  <p class="graf graf--p">
   并且，专利还透露了一款全新的表情符号，通过扬声器表面展示。
  </p>
  <p class="graf graf--p">
   虽然未使用 HomePod 这个词，但 Apple 将新设备称为“设备10”。
  </p>
  <p class="graf graf--p">
   该专利中给出的描述称新设备可以分析用户的语音（例如，当用户向设备10提供诸如数据库查询和/或其他命令的语音命令时），如果在捕获的语音信息中检测到声音中的压力，则设备10可以调整呈现给用户的内容。
  </p>
  <p class="graf graf--p">
   例如，如果语音响度或压力模式指示用户有压力，则可以相应地调整设备10中的视觉输出图案的颜色和/或亮度（例如以反映升高的压力或试图通过营造平静的环境来应对压力）。
  </p>
  <p class="graf graf--p">
   简单说它想要读并介入以控制你的情绪。
  </p>
  <p class="graf graf--p">
   例如，如果在用户的语音中检测到悲伤，则该设备可以相应地改变（例如，改变为表示用户悲伤的悲伤表情符号、或者用于抵消检测到的悲伤的快乐表情符号）。还可以根据当前呈现给用户的内容的性质来改变，例如，如果用户向数字助理询问关于购买生日礼物的信息，则数字助理可以使用快乐的表情符号来呈现结果。
  </p>
  <p class="graf graf--p">
   该过程被描述为潜台词识别。翻译一下就是入侵大脑。
  </p>
