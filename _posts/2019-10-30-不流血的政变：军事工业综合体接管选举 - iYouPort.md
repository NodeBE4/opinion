  <p class="graf graf--p">
   五月份，科技巨头微软宣布了其“保护”美国选举免受干扰的解决方案，并将其命名为 “ElectionGuard”。
  </p>
  <p class="graf graf--p">
   该选举技术已经被
   <strong class="markup--strong markup--p-strong">
    半数投票机制造商
   </strong>
   和一些州政府用于2020年大选。
  </p>
  <p class="graf graf--p">
   虽然此后几周内这玩意得到了主流媒体大力推广，但是，这些报道都没有透露 ElectionGuard 有几个明显的利益冲突，极大地破坏了其旨在保护民主的主张。
  </p>
  <p class="graf graf--p">
   在这项调查中，MintPress 将揭示 ElectionGuard 是如何由与国防和情报界以及以色列军事情报机构有密切联系的公司开发的；以及为什么这项技术很难防止外国或国内对选举的操纵和其他方面的事实。
  </p>
  <p class="graf graf--p">
   选举取证分析师和作家 Jonathan Simon 以及调查记者 Yasha Levine，就军队长期以来寻求武器化包括互联网在内的公共技术之问题进行了广泛的分析，并就 ElectionGuard 与军事工业综合体的联系以及这些联系对美国民主的影响作为调查的一部分进行了解读。
  </p>
  <p class="graf graf--p">
   今年1月，MintPress 发布了一篇博文，揭露了一家名为 Newsguard 的新闻评级公司。
  </p>
  <p class="graf graf--p">
   该公司声称其旨在打击所谓的“假新闻”，而
   <strong class="markup--strong markup--p-strong">
    该公司与情报部门有密切联系，它是一个顶级的新保守主义智囊团，以及自我承认的政府宣传员，其真正意图是推动企业媒体并压制独立媒体。
   </strong>
  </p>
  <p class="graf graf--p">
   详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%aa%92%e4%bd%93%e5%b7%a8%e5%a4%b4%e5%92%8c%e7%a1%85%e8%b0%b7%e7%9a%84%e6%ad%a6%e5%99%a8%e5%8c%96%e7%ae%97%e6%b3%95%e6%b2%89%e9%bb%98%e5%bc%82%e8%ae%ae/" href="https://www.iyouport.org/%e5%aa%92%e4%bd%93%e5%b7%a8%e5%a4%b4%e5%92%8c%e7%a1%85%e8%b0%b7%e7%9a%84%e6%ad%a6%e5%99%a8%e5%8c%96%e7%ae%97%e6%b3%95%e6%b2%89%e9%bb%98%e5%bc%82%e8%ae%ae/" rel="noopener noreferrer" target="_blank">
    媒体巨头和硅谷的武器化算法沉默异议
   </a>
   》
  </p>
  <p class="graf graf--p">
   这个 Newsguard 正是微软所谓的“捍卫民主”计划的首批举措之一。
  </p>
  <p class="graf graf--p">
   通过与微软的合作，Newsguard
   <strong class="markup--strong markup--p-strong">
    已经安装
   </strong>
   在全国的公共图书馆和大学中，即使目前为止私营公司仍在努力避免采用这种有问题的浏览器插件。
  </p>
  <p class="graf graf--p">
   现在，微软正在推动一项新的倡议 —— 一项同样存在明显利益冲突的倡议 —— 以 Newsguard 永远无法实现的方式
   <strong class="markup--strong markup--p-strong">
    威胁美国民主
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ElectionGuard 被微软吹捧为一个系统，旨在“使投票更安全、更便捷、更高效；在美国和世界各地的国家使用。”
  </p>
  <p class="graf graf--p">
   自那时起，主流媒体和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.npr.org/2019/05/06/720071488/ahead-of-2020-microsoft-unveils-tool-to-allow-voters-to-track-their-ballots" href="https://www.npr.org/2019/05/06/720071488/ahead-of-2020-microsoft-unveils-tool-to-allow-voters-to-track-their-ballots" rel="noopener noreferrer" target="_blank">
    政府资助的
   </a>
   媒体
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cbsnews.com/news/microsoft-unveils-new-software-tools-it-claims-will-make-voting-secure/" href="https://www.cbsnews.com/news/microsoft-unveils-new-software-tools-it-claims-will-make-voting-secure/" rel="noopener noreferrer" target="_blank">
    就开始大力推动
   </a>
   它在2020年大选中的使用。
  </p>
  <p class="graf graf--p">
   更多：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%80%89%e4%b8%be%e8%bf%90%e5%8a%a8%e7%9a%84%e6%95%b0%e5%ad%97%e5%8c%96%e6%96%97%e4%ba%89-2020-presidential-race/" href="https://www.iyouport.org/%e9%80%89%e4%b8%be%e8%bf%90%e5%8a%a8%e7%9a%84%e6%95%b0%e5%ad%97%e5%8c%96%e6%96%97%e4%ba%89-2020-presidential-race/" rel="noopener noreferrer" target="_blank">
    选举运动的数字化斗争: 2020 Presidential Race
   </a>
   》
  </p>
  <p class="graf graf--p">
   然而，根据选举取证分析师、“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amazon.com/CODE-RED-Computerized-Elections-Democracy/dp/1719301646/ref" href="https://www.amazon.com/CODE-RED-Computerized-Elections-Democracy/dp/1719301646/ref" rel="noopener noreferrer" target="_blank">
    红色代码：计算机化选举和美国民主战争
   </a>
   ”的作者 Jonathan Simon 的说法，
   <strong class="markup--strong markup--p-strong">
    这场公关运动可能只是为了让更多内部人操纵美国选举。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “提议保护选举是对的，但不幸的是，提议的解决方案只是更多的计算机化和复杂性 —— 这将转化为专家和内部人员的更多控制”，他说。
  </p>
  <p class="graf graf--p">
   至于这些内部人士的可能身份 ——
   <strong class="markup--strong markup--p-strong">
    微软的 ElectionGuard 是与私人军事和情报承包商共同开发的，其唯一的投资者是美国国防部
   </strong>
   ，这提供了令人不安的线索。
  </p>
  <p class="graf graf--p">
   因此，ElectionGuard 对“安全”选举的承诺是可疑的，特别是考虑到微软本身就是大牌的军事承包商。
  </p>
  <p class="graf graf--p">
   此外，在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.haaretz.com/israel-news/facebook-busts-israel-based-campaign-to-disrupt-elections-in-african-asian-nations-1.7249342" href="https://www.haaretz.com/israel-news/facebook-busts-israel-based-campaign-to-disrupt-elections-in-african-asian-nations-1.7249342" rel="noopener noreferrer" target="_blank">
    以色列干涉外国选举
   </a>
   的
   <a class="markup--anchor markup--p-anchor" data-href="http://www.startribune.com/israeli-disinformation-campaign-targeted-nigerian-election/510069112/" href="http://www.startribune.com/israeli-disinformation-campaign-targeted-nigerian-election/510069112/" rel="noopener noreferrer" target="_blank">
    丑闻
   </a>
   中，微软与以色列军事情报局和以色列私人网络技术公司的关系日益密切，更令人担忧的是，ElectionGuard 的真正目的是
   <strong class="markup--strong markup--p-strong">
    “确保”能选举出于对该机构友好的候选人，尤其是军工综合体
   </strong>
   。
  </p>
  <p class="graf graf--p">
   延伸：《
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Foreign_electoral_intervention" href="https://en.wikipedia.org/wiki/Foreign_electoral_intervention" rel="noopener noreferrer" target="_blank">
    干预外国选举的状元是谁？
   </a>
   》
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-6370 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/选举.png?resize=1100%2C593&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/选举.png?w=1298&amp;ssl=1 1298w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/选举.png?resize=300%2C162&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/选举.png?resize=768%2C414&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/选举.png?resize=1024%2C552&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/选举.png?resize=1100%2C593&amp;ssl=1 1100w" data-recalc-dims="1" height="593" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/选举.png?resize=1100%2C593&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    分析这个 ElectionGuard
   </strong>
  </p>
  <p class="graf graf--p">
   根据微软副总裁 Tom Burt 于5月初发布的公告，ElectionGuard 被称为“一个免费的开源软件开发工具包（SDK）”、“将使投票更安全，更易于访问，更高效地被使用”。
  </p>
  <p class="graf graf--p">
   Burt 的声明进一步声称，ElectionGuard 系统“将实现选举的端到端验证，向第三方组织公开结果以进行安全验证，并允许个人选民确认其投票得到了正确计算。”
  </p>
  <p class="graf graf--p">
   虽然 ElectionGuard 似乎只关注电子选票，但该
   <a class="markup--anchor markup--p-anchor" data-href="https://gadgets.ndtv.com/apps/news/build-2019-microsofts-electionguard-software-to-offer-tools-to-secure-elections-2033954" href="https://gadgets.ndtv.com/apps/news/build-2019-microsofts-electionguard-software-to-offer-tools-to-secure-elections-2033954" rel="noopener noreferrer" target="_blank">
    公告声称
   </a>
   该系统 “旨在通过使用光学扫描仪与使用纸质选票的系统
   <strong class="markup--strong markup--p-strong">
    配合使用
   </strong>
   ”。
  </p>
  <p class="graf graf--p">
   值得注意的是，微软在选择宣布 ElectionGuard 之前，它已经与主要的选举技术供应商合作，他们正在探索将 ElectionGuard 整合到他们的投票系统中。
  </p>
  <p class="graf graf--p">
   Burt 进一步指出，微软现在与选举技术供应商建立了合作伙伴关系，这些供应商负责在美国销售超过一半的投票机。
  </p>
  <p class="graf graf--p">
   ElectionGuard 的合作伙伴公司包括 Democracy Live，Election Systems＆Software，Hart InterCivic，BPro，MicroVote 和 VotingWorks。
  </p>
  <p class="graf graf--p">
   微软公告中另一个有趣且令人深感不安的信息是，微软的 ElectionGuard 开发合作伙伴、
   <strong class="markup--strong markup--p-strong">
    位于波特兰的网络安全公司 Galois，“最近从五角大楼的国防高级研究计划局（DARPA）获得了1000万美元的资金
   </strong>
   ，用于建立一个示范投票系统，以帮助评估 DARPA 研究人员正在开发的硬件，作为单独的 DARPA 计划的一部分。”
  </p>
  <p class="graf graf--p">
   微软的声明随后指出，“该机构计划实施 ElectionGuard SDK，作为他们在未来版本的示范投票系统中实现端到端可验证组件的努力的一部分。”
  </p>
  <p class="graf graf--p">
   正如 DARPA 对 ElectionGuard 的1000万美元间接投资所显示的那样令人深感不安，但这只是表面上看，
   <strong class="markup--strong markup--p-strong">
    因为 Galois 本身实际上就是 DARPA 在私人网络安全行业的延伸。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这家所谓的“私人”公司的唯一投资者是五角大楼
   </strong>
  </p>
  <p class="graf graf--p">
   Galois 由 John Launchbury 于1999年创立，
   <a class="markup--anchor markup--p-anchor" data-href="https://galois.com/about/" href="https://galois.com/about/" rel="noopener noreferrer" target="_blank">
    很快就接近了众多政府机
   </a>
   构，现在 —— 根据 Galois 网站的介绍 —— 政府机构构成了该公司绝大部分客户。
  </p>
  <p class="graf graf--p">
   事实上，Galois 目前只在其“客户”部分列出
   <a class="markup--anchor markup--p-anchor" data-href="http://www.hoovers.com/company-information/cs/company-profile.galois_inc.34ac489b29ae2b98.html?aka_re=1" href="http://www.hoovers.com/company-information/cs/company-profile.galois_inc.34ac489b29ae2b98.html?aka_re=1" rel="noopener noreferrer" target="_blank">
    以下美国政府机构
   </a>
   ：DARPA、国防部、能源部、国土安全部、“情报社区”（即 中央情报局、国家安全局等）和美国宇航局。
  </p>
  <p class="graf graf--p">
   然而，Galois 的其他客户还包括美国顶级武器制造商通用动力公司。
  </p>
  <p class="graf graf--p">
   Galois 表示，该公司的重点是先进计算机科学的研究和开发；它还涉及人工智能、人机交互和机器学习。
  </p>
  <p class="graf graf--p">
   虽然它将自己描述为“私营公司”，但是，
   <strong class="markup--strong markup--p-strong">
    公开记录显示，Galois 的唯一投资者是 DARPA 和海军研究办公室（ONR）—— 两者都是国防部的部门
   </strong>
   。
  </p>
  <p class="graf graf--p">
   换句话说，虽然表面上是一家私营公司，但是它唯一的投资者是政府机构，更具体地说是五角大楼。
  </p>

  </p>

   <a class="markup--anchor markup--p-anchor" data-href="https://d60.darpa.mil/speakers/DrJohnLaunchbury.html" href="https://d60.darpa.mil/speakers/DrJohnLaunchbury.html" rel="noopener noreferrer" target="_blank">
    随后担任
   </a>
   DARPA 信息创新办公室主任，负责处理“国家级网络安全和人工智能投资”。
  </p>
  <p class="graf graf--p">
   2017年，他离开 DARPA，回到 Galois 工作，担任该公司的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.linkedin.com/in/johnlaunchbury/" href="https://www.linkedin.com/in/johnlaunchbury/" rel="noopener noreferrer" target="_blank">

   </a>
   。
  </p>

   <a class="markup--anchor markup--p-anchor" data-href="https://www.federaltimes.com/enterprise-view/2015/10/02/how-darpa-s-i2o-finds-innovation-on-the-edge/" href="https://www.federaltimes.com/enterprise-view/2015/10/02/how-darpa-s-i2o-finds-innovation-on-the-edge/" rel="noopener noreferrer" target="_blank">
    也侧重于
   </a>
   加强“人/机伙伴关系”。
  </p>
  <p class="graf graf--p">
   一家名为 Free＆Fair 的 Galois 子公司，专门开发选举技术，与微软合作生产 ElectionGuard。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://freeandfair.us/" href="https://freeandfair.us/" rel="noopener noreferrer" target="_blank">

   </a>

   <a class="markup--anchor markup--p-anchor" data-href="https://www.sos.state.mn.us/about-the-office/news-room/secretary-simon-announces-new-steps-to-enhance-election-cybersecurity/" href="https://www.sos.state.mn.us/about-the-office/news-room/secretary-simon-announces-new-steps-to-enhance-election-cybersecurity/" rel="noopener noreferrer" target="_blank">

   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e7%a9%b6%e7%ab%9f%e6%98%af%e4%bb%80%e4%b9%88%ef%bc%9f/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e7%a9%b6%e7%ab%9f%e6%98%af%e4%bb%80%e4%b9%88%ef%bc%9f/" rel="noopener noreferrer" target="_blank">

  <p class="graf graf--p">
   该单位还因为“胁迫目的” —— 即 收集讹诈信息 —— 以及通过与国家安全局的情报共享协议
   <a class="markup--anchor markup--p-anchor" data-href="https://www.npr.org/sections/thetwo-way/2014/09/21/350274814/snowden-reveal-makes-israeli-spies-protest-an-american-issue" href="https://www.npr.org/sections/thetwo-way/2014/09/21/350274814/snowden-reveal-makes-israeli-spies-protest-an-american-issue" rel="noopener noreferrer" target="_blank">
    对平民进行间谍活动
   </a>
   而闻名。
  </p>
  <p class="graf graf--p">
   然而，微软与8200的联系已经远远超出了 Adallom。另一个例子是微软对由 Team8 创建的以色列网络安全公司 Illusive Networks 的
   <a class="markup--anchor markup--p-anchor" data-href="https://techcrunch.com/2017/01/24/microsoft-ventures-backs-illusive-networks-a-cybersecurity-firm-that-protects-with-deception/" href="https://techcrunch.com/2017/01/24/microsoft-ventures-backs-illusive-networks-a-cybersecurity-firm-that-protects-with-deception/" rel="noopener noreferrer" target="_blank">
    大量投资
   </a>
   。
  </p>
  <p class="graf graf--p">
   Team8 的首席执行官和联合创始人是 Nadav Zafrir，他曾经领导过8200，该公司其他三位联合创始人中的两位也都是8200的老手。
   <strong class="markup--strong markup--p-strong">
    谷歌的前任首席执行官 Eric Sc​​hmidt，是 Team8 主要的支持者。
   </strong>
  </p>
  <p class="graf graf--p">
   Team8 已经与前美国国家安全局局长合作，例如，Zafrir 与前国家安全局局长臭名昭著的 Keith Alexander 一起做演讲。
  </p>
  <p class="graf graf--p">
   这些努力最终导致 Team8
   <a class="markup--anchor markup--p-anchor" data-href="https://www.prnewswire.com/news-releases/former-nsa-director-and-commander-of-us-cyber-command-mike-rogers-joins-cybersecurity-foundry-team8-as-a-senior-advisor-300733619.html" href="https://www.prnewswire.com/news-releases/former-nsa-director-and-commander-of-us-cyber-command-mike-rogers-joins-cybersecurity-foundry-team8-as-a-senior-advisor-300733619.html" rel="noopener noreferrer" target="_blank">
    招聘已退休的海军上将 Mike Rogers
   </a>
   ，也是前美国国家安全局和美国网络司令部主任，担任“高级顾问”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我过去曾与8200单位的高级人才资源合作过，所以当我有机会加入Team8时，我知道这是一个难得的宝贵机会，” Rogers 说。
  </p>
  <p class="graf graf--p">
   Rogers 被一家外国军事情报机构前任老板领导的公司聘用，引起了美国国家安全局退伍军人的尖锐批评。
  </p>
  <p class="graf graf--p">
   其中一位前 NSA 员工 —— 美国国家安全局定制访问操作黑客部门的资深人士 Jake Williams
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cyberscoop.com/mike-rogers-team8-nsa-board-criticism-unit-8200/" href="https://www.cyberscoop.com/mike-rogers-team8-nsa-board-criticism-unit-8200/" rel="noopener noreferrer" target="_blank">
    告诉 Cyber​​Scoop
   </a>
   ，“由于他的技术经验，Rogers 不值得这个角色……
   <strong class="markup--strong markup--p-strong">
    这纯粹是因为他对机密操作的了解和他能够影响美国政府和私营部门承包商的许多人。
   </strong>
   ”
  </p>
  <p class="graf graf--p">
   除了微软通过与 Adallom、Illusive Networks 和 Team8 的8200的密切关系之外，微软还与以色列军方
   <a class="markup--anchor markup--p-anchor" data-href="https://www.windowscentral.com/ar-war-microsoft-hololens-be-used-israeli-army-strategy-and-training" href="https://www.windowscentral.com/ar-war-microsoft-hololens-be-used-israeli-army-strategy-and-training" rel="noopener noreferrer" target="_blank">
    直接联系
   </a>
   ，IDF采用了该公司的 HoloLens 技术。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    ElectionGuard 为军事工业综合体制造了一场不流血的政变
   </strong>
  </p>
  <p class="graf graf--p">
   在2016年大选之后以及对“俄罗斯黑客”渗透选举系统的强烈关注之后，像国家安全局这样的联邦机构
   <strong class="markup--strong markup--p-strong">
    利用这种威胁来游说以加强对美国民主的控制
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Rogers —— 现在受雇于微软资助的以色列军事情报联盟公司 Team8 —— 为美国政府机构（包括国家安全局和国土安全部）的直接参与进行游说；ElectionGuard 将帮助这些机构特别容易地监视美国选举，特别是考虑到
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" rel="noopener noreferrer" target="_blank">
    微软过去与国家安全局的幕后合作
   </a>
   。
  </p>
  <p class="graf graf--p">
   鉴于目前所描述的 ElectionGuard 系统既不像微软声称的那样“安全”也不“可验证”，显然其开发人员的利益冲突、特别是他们与美国和以色列军队的关系，是灾难的一个重要因素。
   <strong class="markup--strong markup--p-strong">
    这等同于军工综合体对美国选举制度的接管
   </strong>
   。
  </p>
  <p class="graf graf--p">
   正如 Jonathan Simon 所说，“如果我们让计算机退出投票过程，并且愿意集体投入人类在公共场合可观察地计算投票所需的一小部分努力，那么我们就可以轻松地走向相反的方向，迅速解决选举安全的所有问题。曾经一直在这样做。如果民主不值得这样的努力，也许我们不值得得到它”。⚪️
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.mintpressnews.com/microsoft-electionguard-a-trojan-horse-for-a-military-industrial-takeover-of-us-elections/258732/" href="https://www.mintpressnews.com/microsoft-electionguard-a-trojan-horse-for-a-military-industrial-takeover-of-us-elections/258732/" rel="noopener noreferrer" target="_blank">
    Microsoft’s ElectionGuard a Trojan Horse for a Military-Industrial Takeover of US Elections
   </a>
  </p>
