  <h1 class="entry-title">
   道德的贬值
  </h1>
 </header>
 <div class="entry-meta">

 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      你会因为公司的缺德而辞职吗？
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   就业网站最近进行了一项调查，试图了解恶劣的丑闻如何影响科技工作者继续参与和申请在这些巨头公司工作，这些公司一直处于数据泄漏、性骚扰指控和欺骗性隐私实践等丑闻的中心。但是，结果可能会让你大吃一惊。
  </p>
  <p class="graf graf--p">
   事实上，调查中1000名科技工作者被询问了他们对整个监视资本主义的态度，他们对陷入丑闻的公司工作的愿望，是否曾在丑闻缠身的公司工作，是否考虑在其他地方找工作，等等问题。结果是……
  </p>
  <p class="graf graf--p">
   首先，结果显示大多数科技工作者（59％）并不认为科技行业的丑闻足以阻止他们在该行业工作。这并不是特别令人惊讶，因为工人们投入了时间和金钱以接受培训为了进入一个领域，有些人可能常常认为他们可以帮助从内部改变环境；事实上，只有6％的受访者表示，丑闻使他们对科技行业的工作兴趣减少。但仍然很奇怪，有34％的科技工作者说丑闻实际上让他们对这个领域更感兴趣了。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    这个答案在男性（41％）中比女性（26％）更常见，在千禧一代（41％）中更常见，相比退休的人（16％; 55岁及以上）
   </em>
   。
  </p>
  <p class="graf graf--p">
   当谈到为特定丑闻缠身的公司工作时，33％的受访者表示他们对为这样一家公司工作的兴趣不大。尽管女性更为谨慎（40％的人更愿意签约，相比之下，男性占27％），但是，在实践中，更多的男性表示他们实际上已经因为丑闻而退出了科技公司的工作（68％）与女性相比（50％）。然而，目前尚不清楚退出的员工在丑闻中扮演的角色（例如，如果一个人是肇事者，那就不那么高贵了）。
  </p>
  <p class="graf graf--p">
   根据调查，公司所涉及的丑闻类型会对员工的留下意愿产生影响。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    基于技术的丑闻，例如数据泄露或产品故障，是最有可能导致员工离职的争议类型，37％的受访者表示他们会“高度可能”或“有可能”这样做
   </em>
   。
  </p>
  <p class="graf graf--p">
   基于性别的丑闻，如性骚扰和雇佣偏见，也可能使员工离职，但不如科技丑闻；只有35％的受访者表示这些丑闻会影响他们的决定。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在参与丑闻的公司工作的人中，近四分之三（74％）的工人表示他们没有考虑离开。因此，上述数据中的良好意图并不总是能够实现
   </strong>
   。
  </p>
  <p class="graf graf--p">
   奇怪的是，65％接受调查的科技工作者仍然认为科技公司是道德的，尽管超过一半（53％）的人认为他们需要更多监管。再次，这种情况具有性别差异，57％的女性与71％的男性认为公司是“道德的”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    目前还不清楚在这个意义上“道德的”标准是什么，但具有讽刺意味的是，那些愿意离开这个领域的人最有兴趣进入金融领域，尽管金融领域是众所周知的道德失败
   </strong>
   。
  </p>
  <p class="graf graf--p">
   总之，以下是来自Indeed调查的一些关键要点：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     34％的受访者表示，由于丑闻，他们对科技行业的工作更感兴趣了
    </strong>
    ；
   </li>
   <li class="graf graf--li">
    32％的受访者表示他们更愿意申请参与丑闻的公司的工作；
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     31％的受访者表示，由于公共丑闻，他们更渴望在整个科技行业工作
    </strong>
    ；
   </li>
   <li class="graf graf--li">
    15％的受访者表示，他们曾在2018年成为公共丑闻一部分的公司工作；
   </li>
   <li class="graf graf--li">
    38％的员工表示他们可能会因为技术丑闻而离开公司。相比之下，35％的人愿意在性别丑闻后离开，而30％的人愿意在政治丑闻之后离开，例如有一位表达与自己相矛盾的政治观点的高管。
   </li>
   <li class="graf graf--li">
    48％的受访者表示，他们现在更愿意在2018年的公共丑闻之后举报不当行为；
   </li>
   <li class="graf graf--li">
    如果公司对丑闻透明并试图纠正，那么79％的员工可能会留在岗位上；
   </li>
   <li class="graf graf--li">
    根据调查，
    <strong class="markup--strong markup--li-strong">
     决定是否加入科技公司时最重要的因素是薪酬和福利
    </strong>
    。公众声誉和形象排在榜单的最底层，只有23％的受访者表示这很重要；
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     技术人员考虑完全转向另一个行业的首要原因是薪酬较高，而不是其他因素
    </strong>
    。
   </li>
  </ul>
  <p class="graf graf--p">
   尽管得到了一些令人沮丧的结果，但我们已经看到有技术人员开始要求一个更加道德和透明的工作场所，特别是在去年。谷歌和微软的员工迫使高管们放弃与国防部达成100亿美元合同的投标，因为他们不愿意将他们的工作武器化。450名亚马逊员工签署了一封信，要求首席执行官杰夫贝佐斯停止向执法机构出售其面部识别软件。超过1,400名谷歌员工签署了一封信，要求提高透明度和为中国政府设计的审查版搜索引擎的责任。这只是几个例子 — 尽管公平地说，这些只是少数几家科技公司。
  </p>
  <p class="graf graf--p">
   更多《
   <a class="markup--anchor markup--p-anchor" data-href="https://iyouport.weebly.com/short-stories-enthusiast/1866977" href="https://iyouport.weebly.com/short-stories-enthusiast/1866977" rel="noopener noreferrer" target="_blank">
    透明度革命要从内部爆发
   </a>
   》
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    科技行业面临的一个主要问题是，未来员工的教育中没有涉及道德培训。当公司或专业组织制定道德规范时，这些规范似乎对实际行为几乎没有影响
   </strong>
   。例如，北卡罗来纳州立大学的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.sciencedaily.com/releases/2018/10/181009113617.htm" href="https://www.sciencedaily.com/releases/2018/10/181009113617.htm" rel="noopener noreferrer" target="_blank">
    一项研究发现
   </a>
   ，道德规范似乎没有对计算机科学家进入该领域后做出的决定产生任何影响。
  </p>
  <p class="graf graf--p">
   由于我们许多人致力于在科技行业制定某种道德准则，很明显
   <strong class="markup--strong markup--p-strong">
    我们面临的挑战是人们是否真正关心公司的道德规范。就如微信和Facebook，丑闻不断和监视曝光都没能明显减少用户数量
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果我们自己都不关心自己的隐私和安全，我们怎么能期望公司表现得更好呢？如果我们继续简单地对道德教育嗤之以鼻，并且不对不良行为或甚至违法行为施加任何处罚，我们就相当于亲手打造一个更具欺骗性和危险性的未来。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    T
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.forbes.com/sites/jessicabaron/2019/03/04/tech-workers-are-still-willing-to-work-for-scandal-ridden-companies/#50ae0a903c29" href="https://www.forbes.com/sites/jessicabaron/2019/03/04/tech-workers-are-still-willing-to-work-for-scandal-ridden-companies/#50ae0a903c29" rel="noopener noreferrer" target="_blank">
    ech Workers Are Still Willing To Work For Scandal-Ridden Companies
   </a>
   . If we don’t care about our own privacy and safety, how can we expect companies to behave any better? If we continue to simply give lip service to ethics education and fail to enforce any penalties for bad behavior or even breaking the law, we may be implicitly sanctioning a more deceptive and dangerous future.
  </p>
