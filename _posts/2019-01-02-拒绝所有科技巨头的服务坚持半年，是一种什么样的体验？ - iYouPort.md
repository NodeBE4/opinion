  <p>
   <span style="color: #515151;">
    五月份，我决定从我的生活中剥离所谓的“五大”科技公司一个月。这意味着拒绝 Google，Microsoft，亚马逊，Facebook，Apple 或其任何子公司的任何服务。
   </span>
  </p>
  <p>
   <span style="color: #515151;">
    我想进行这项实验有很多原因，包括回收我的隐私，以及更多地了解我每天使用的技术是如何实际运作的。然而，主要原因是想看看五大服务是否真的有必要或仅仅是为了方便。
   </span>
  </p>
  <p>
   <span style="color: #515151;">
    我发现几乎所有五大公司的主要服务都有足够多的开源或独立替代品。在某些情况下，例如地图软件或社交媒体平台，五大与替代服务之间的差距如此之大，以至于它对我的生活产生了明显的影响。但在大多数情况下，开源或独立的替代方案工作得很好。
    <br/>
   </span>
  </p>
  <ul>
   <li>
    <span style="color: #515151;">
     延伸阅读：《
     <a href="https://www.iyouport.org/google-%e4%ba%a7%e5%93%81%e7%9a%84%e6%9b%bf%e4%bb%a3%e5%93%81-%e5%ae%8c%e6%95%b4%e5%88%97%e8%a1%a8-%e5%85%b3%e6%b3%a8%e9%9a%90%e7%a7%81%e6%9d%83%e7%9a%84%e4%bd%a0%ef%bc%8c%e9%a9%ac%e4%b8%8a/" rel="noopener noreferrer" target="_blank">
      Google 产品的替代品在这里
     </a>
     》
    </span>
   </li>
   <li>
    <span style="color: #515151;">
     《
     <a href="https://www.iyouport.org/%e5%ae%89%e5%85%a8%e6%89%8b%e5%86%8c%ef%bc%9a%e8%bf%99%e9%87%8c%e6%98%af%e4%bd%a0%e9%9c%80%e8%a6%81%e7%9a%84%e5%87%a0%e4%b9%8e%e6%89%80%e6%9c%89%e5%ae%89%e5%85%a8%e4%b8%8a%e7%bd%91%e5%b7%a5%e5%85%b7/" rel="noopener noreferrer" target="_blank">
      所有安全产品的工具箱在这里
     </a>
     》
    </span>
   </li>
  </ul>
  <p>
   <span style="color: #515151;">
    以下是一些总结 — –
   </span>
   <br/>
   <span style="color: #515151;">
    <br/>
    FACEBOOK
    <br/>
   </span>
  </p>
  <ul>
   <li>
    <span style="color: #515151;">
     在删除我的 Facebook 个人资料之前，我最关心的一个问题是它会不会破坏我的社交生活。然而，离开平台差不多一年后，我发现事实并非如此。Facebook 坚持认为它是社区和友谊的核心和灵魂，但这只是一种营销策略，而不是现实。如果有什么的话，离开社交媒体平台真的会减少我的错失恐惧症。
    </span>
   </li>
   <li>
    <span style="color: #515151;">
     如果您将其他第三方应用程序链接到了您的 Facebook 个人资料，那么退出社交媒体平台会更加困难。只需使用您的电子邮件地址和密码管理器即可登录这些应用程序。
    </span>
   </li>
   <li>
    <span style="color: #515151;">
     Facebook 很难从网站上批量下载高分辨率的照片副本。有脚本可以做到这一点，但它们需要很长时间并经常崩溃，所以如果你已经在 Facebook 上存储了很多照片，请准备花几个小时去删除照片。
    </span>
   </li>
  </ul>
  <p>
   <span style="color: #515151;">
    苹果
    <br/>
   </span>
  </p>
  <ul>
   <li>
    <span style="color: #515151;">
     您可以构建一台功能强大的计算机，其成本远低于 Apple 台式机或 MacBook 的成本。Apple 卖的只是广告。如果您真的担心功能性，您可以通过选择自己的组件并自行组装计算机，以较低的价格获得更好的 PC 性能。我保证它并不像听起来那么难。
    </span>
   </li>
  </ul>
  <p>
   <span style="color: #515151;">
    微软
    <br/>
   </span>
  </p>
  <ul>
   <li>
    <span style="color: #515151;">
     Linux 很棒。它也是免费的，比 Windows 便宜139美元。
    </span>
   </li>
  </ul>
  <p>
   <span style="color: #515151;">
    AMAZON
    <br/>
   </span>
  </p>
  <ul>
   <li>
    <span style="color: #515151;">
     事实证明，您可以在 IRL 商店获得您需要的大部分东西，其他所有内容都可以在专业网站上找到。奇怪吗，但这是真的！
    </span>
   </li>
  </ul>
  <p>
   <span style="color: #515151;">
    谷歌
    <br/>
   </span>
  </p>
  <ol>
   <li>
    <span style="color: #515151;">
     谷歌地图仍然没有很好的替代品，但是有非常接近的。
    </span>
   </li>
   <li>
    <span style="color: #515151;">
     设置您自己的“云”服务只需50美元即可完成，无需大量技术技能。
    </span>
   </li>
   <li>
    <span style="color: #515151;">
     就协作文档编辑而言，Google Drive 确实没有好的开源替代品。
    </span>
   </li>
   <li>
    <span style="color: #515151;">
     抛弃 Android 很难。Sailfish 几乎是唯一仍然存在的独立移动操作系统，但基本上不可能进入某些手机。虽然有一个充满活力的黑客社区正在开发他们自己的分叉版 Android，但谷歌正在打击这些非官方版本，无法通过非官方 Android 操作系统从其应用程序商店下载谷歌应用程序。但另一方面，使用 APK 文件将应用程序侧载到非官方 Android 版本上并不困难。
    </span>
   </li>
   <li>
    <span style="color: #515151;">
     DuckDuckGo 是谷歌搜索的一个很好的替代品，并表明广告支持的搜索引擎不需要侵犯用户的隐私。
    </span>
   </li>
  </ol>
  <ul>
   <li>
    <span style="color: #515151;">
     延伸阅读：《
     <a href="https://www.iyouport.org/%e5%96%9c%e6%ac%a2-android-%e4%bd%86%e8%ae%a8%e5%8e%8c-google-%e7%9a%84%e9%9a%90%e7%a7%81%e4%be%b5%e7%8a%af%ef%bc%8c%e5%8f%af%e4%bb%a5%e6%80%8e%e4%b9%88%e5%8a%9e%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
      喜欢 Android 但讨厌 Google 的隐私侵犯，可以怎么办？
     </a>
     》
    </span>
   </li>
  </ul>
  <p>
   <span style="color: #515151;">
    重要的是要意识到这些服务都不是必需的。我们可能已经开始过度依赖它们了，但事实上它们没那么重要。亚马逊只是与警察合作的老大哥。您的社区在 Facebook 出现之前就已经存在了。谷歌并不总是一个动词。我们有能力通过我们与他们互动的方式来改变这些公司 — 但只有在我们想要的时候。
   </span>
  </p>
