       这个该死的“即服务”句式，如今至少是诚实了一次
      </strong>
     </em>
    </span>
   </li>

  <p>
   <img alt="" class="aligncenter size-full wp-image-6470 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/亚马逊.jpg?resize=1100%2C618&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/亚马逊.jpg?w=2048&amp;ssl=1 2048w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/亚马逊.jpg?resize=300%2C169&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/亚马逊.jpg?resize=768%2C432&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/亚马逊.jpg?resize=1024%2C576&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/亚马逊.jpg?resize=1100%2C618&amp;ssl=1 1100w" data-recalc-dims="1" height="618" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/亚马逊.jpg?resize=1100%2C618&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6470" data-recalc-dims="1" height="618" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/亚马逊.jpg?resize=1100%2C618&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/亚马逊.jpg?w=2048&amp;ssl=1 2048w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/亚马逊.jpg?resize=300%2C169&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/亚马逊.jpg?resize=768%2C432&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/亚马逊.jpg?resize=1024%2C576&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/亚马逊.jpg?resize=1100%2C618&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   亚马逊可能还在纠结自己的监控功能不够广泛，尽管有其 Echo、Ring 和 Key 产品的广泛使用，更不用说亚马逊每时每刻都在收集客户的所有数据了；但显然该公司仍觉不够。
  </p>
  <p class="graf graf--p">
   亚马逊最近获得了美国专利，提供“监控即服务”。
  </p>
  <p class="graf graf--p">
   该专利是针对“无人驾驶飞行器” — 无人机的技术术语 — “可以在授权方的财产上执行监视行动”，并且可以“对监视目标进行成像以生成监视图像”。
  </p>
  <p class="graf graf--p">
   亚马逊在2015年6月12日提交并于今年6月4日被授权的
   <a class="markup--anchor markup--p-anchor" data-href="https://pdfpiw.uspto.gov/.piw?PageNum=0&amp;docid=10313638&amp;IDKey=E35B67ACE250%0D%0A&amp;HomeUrl=http%3A%2F%2Fpatft.uspto.gov%2Fnetacgi%2Fnph-Parser%3FSect1%3DPTO2%2526Sect2%3DHITOFF%2526u%3D%25252Fnetahtml%25252FPTO%25252Fsearch-adv.htm%2526r%3D108%2526f%3DG%2526l%3D50%2526d%3DPTXT%2526s1%3D%252522amazon%252Btechnologies%252522.ASNM.%2526p%3D3%2526OS%3DAN%2F%252522amazon%252Btechnologies%252522%2526RS%3DAN%2F%252522amazon%252Btechnologies%252522" href="https://pdfpiw.uspto.gov/.piw?PageNum=0&amp;docid=10313638&amp;IDKey=E35B67ACE250%0D%0A&amp;HomeUrl=http%3A%2F%2Fpatft.uspto.gov%2Fnetacgi%2Fnph-Parser%3FSect1%3DPTO2%2526Sect2%3DHITOFF%2526u%3D%25252Fnetahtml%25252FPTO%25252Fsearch-adv.htm%2526r%3D108%2526f%3DG%2526l%3D50%2526d%3DPTXT%2526s1%3D%252522amazon%252Btechnologies%252522.ASNM.%2526p%3D3%2526OS%3DAN%2F%252522amazon%252Btechnologies%252522%2526RS%3DAN%2F%252522amazon%252Btechnologies%252522" rel="noopener noreferrer" target="_blank">
    专利
   </a>
   中表示：基于无人机的监控将优于传统的视频摄像机，后者范围有限、容易遗漏、容易被操纵或被入侵者破坏。
  </p>
  <p class="graf graf--p">
   1 来自亚马逊的“监控即服务”专利的图片。
  </p>
  <p class="graf graf--p">
   基于无人机的监控对于亚马逊来说是合乎逻辑的下一步，亚马逊已经在这个方程式的两端都投入了大量资金。
  </p>
  <p class="graf graf--p">
   延伸：关于无人机监视的分析《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a4%a9%e7%a9%ba%e4%b8%ad%e7%9a%84%e7%9c%bc%e7%9d%9b/" href="https://www.iyouport.org/%e5%a4%a9%e7%a9%ba%e4%b8%ad%e7%9a%84%e7%9c%bc%e7%9d%9b/" rel="noopener noreferrer" target="_blank">
    天空中的眼睛
   </a>
   》。
  </p>
  <p class="graf graf--p">
   自2013年以来，该公司一直在谈论使用无人机投寄包裹，那时起就引起物流巨头联邦快递的巨大兴趣。
  </p>
  <p class="graf graf--p">
   而如今，亚马逊的计划对其竞争对手来说可能看起来不那么有趣了。
  </p>
  <p class="graf graf--p">
   亚马逊全球消费者业务首席执行官 Jeff Wilke 在6月初曾表示，该公司可能会在未来几个月内
   <a class="markup--anchor markup--p-anchor" data-href="https://qz.com/1636076/amazons-new-drone-may-be-just-months-away/" href="https://qz.com/1636076/amazons-new-drone-may-be-just-months-away/" rel="noopener noreferrer" target="_blank">
    开始使用其 Prime Air 无人机向客户提供套餐
   </a>
   。
  </p>
  <p class="graf graf--p">
   亚马逊最近的专利表示，在包裹交付后，监控将是这种无人机系统的“第二重要任务”。
  </p>
  <p class="graf graf--p">
   该公司建议用户“订阅监控系统以提供监控即服务”，这是对已建立的所谓“软件即服务”的一种即兴表达。
  </p>
  <p class="graf graf--p">
   在这里看到为什么“即服务”句式如此荒谬《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%9c%b0%e4%b8%bb2-0%ef%bc%9a%e7%a7%91%e6%8a%80%e7%9a%84%e6%96%b0%e9%a3%9f%e5%88%a9%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89/" href="https://www.iyouport.org/%e5%9c%b0%e4%b8%bb2-0%ef%bc%9a%e7%a7%91%e6%8a%80%e7%9a%84%e6%96%b0%e9%a3%9f%e5%88%a9%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89/" rel="noopener noreferrer" target="_blank">
    地主2.0：科技的新食利资本主义
   </a>
   》；《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e6%89%80%e6%9c%89%e6%98%82%e8%b4%b5%e7%9a%84%e7%94%b5%e5%ad%90%e8%ae%be%e5%a4%87%e4%ba%8b%e5%ae%9e%e4%b8%8a%e5%85%a8%e9%83%bd%e6%98%af%e7%a7%9f%e6%9d%a5%e7%9a%84/" href="https://www.iyouport.org/%e4%bd%a0%e6%89%80%e6%9c%89%e6%98%82%e8%b4%b5%e7%9a%84%e7%94%b5%e5%ad%90%e8%ae%be%e5%a4%87%e4%ba%8b%e5%ae%9e%e4%b8%8a%e5%85%a8%e9%83%bd%e6%98%af%e7%a7%9f%e6%9d%a5%e7%9a%84/" rel="noopener noreferrer" target="_blank">
    你所有昂贵的电子设备事实上全都是租来的
   </a>
   》。
  </p>
  <p class="graf graf--p">
   2018年2月，亚马逊向 Ring 支付了超过10亿美元，这是一个“智能门铃”的制造商，它发送视频信息，客户可以通过他们的智能手机查看和响应。
  </p>
  <p class="graf graf--p">
   在这里看到为什么它如此危险《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%9a%e9%a9%ac%e9%80%8a%e5%b8%ae%e5%8a%a9%e8%ad%a6%e6%96%b9%e6%9e%84%e5%bb%ba%e4%ba%86%e4%b8%80%e4%b8%aa%e5%b9%bf%e6%b3%9b%e7%9a%84%e7%9b%91%e6%8e%a7%e7%bd%91%e7%bb%9c/" href="https://www.iyouport.org/%e4%ba%9a%e9%a9%ac%e9%80%8a%e5%b8%ae%e5%8a%a9%e8%ad%a6%e6%96%b9%e6%9e%84%e5%bb%ba%e4%ba%86%e4%b8%80%e4%b8%aa%e5%b9%bf%e6%b3%9b%e7%9a%84%e7%9b%91%e6%8e%a7%e7%bd%91%e7%bb%9c/" rel="noopener noreferrer" target="_blank">
    亚马逊帮助警方构建了一个广泛的监控网络
   </a>
   》。
  </p>
  <p class="graf graf--p">
   在该交易发布三个月后，亚马逊推出了 Neighbors，这是一个鼓励群众斗群众的社交网络，让用户直接从他们的 Ring 监视摄像头上传视频，并使用“犯罪”、“安全”和“可疑”等标签标记帖子。
  </p>
  <p class="graf graf--p">
   美国50多个当地警察部门与 Ring 合作，以获取其所有者的监视镜头。
  </p>
  <p class="graf graf--p">
   亚马逊还提供 Amazon Key，这项服务让 Prime 会员可以将包裹直接送到家中 —— 需要他们在家门口安装智能锁，并在家中安装亚马逊监视摄像头。
  </p>
  <p class="graf graf--p">
   破折号后面是重点。
  </p>
  <p class="graf graf--p">
   同时，根据投资公司 Cowen 的调查数据，截至2018年第四季度，亚马逊的 Echo 和语音个人助理 Alexa 估计已经覆盖美国家庭的24％。
  </p>
  <p class="graf graf--p">
   今年4月，彭博报道亚马逊工作人员正在远程监听其 Echo 设备捕获的录音，详细《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e7%9a%84%e6%96%af%e5%a1%94%e8%a5%bf%e5%9c%a8%e4%b9%8c%e5%85%8b%e5%85%b0/" href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e7%9a%84%e6%96%af%e5%a1%94%e8%a5%bf%e5%9c%a8%e4%b9%8c%e5%85%8b%e5%85%b0/" rel="noopener noreferrer" target="_blank">
    美国的斯塔西在乌克兰
   </a>
   》。
  </p>
  <p class="graf graf--p">
   Alexa 的监听能力非常强大，华盛顿大学的一组研究人员开发了一种工具，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.technologyreview.com/s/613791/your-smart-speaker-could-know-if-youre-having-a-cardiac-arrest/" href="https://www.technologyreview.com/s/613791/your-smart-speaker-could-know-if-youre-having-a-cardiac-arrest/" rel="noopener noreferrer" target="_blank">
    据报道可以通过呼吸声听到人类心脏病发作的迹象
   </a>
   。
  </p>
  <p class="graf graf--p">
   简单说，它什么都能听到，它是斯塔西的现代版本。⚪️
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://qz.com/1648875/amazon-receives-us-patent-for-surveillance-as-a-service/" href="https://qz.com/1648875/amazon-receives-us-patent-for-surveillance-as-a-service/" rel="noopener noreferrer" target="_blank">
    Amazon wants to sell “surveillance as a service”
   </a>
  </p>
