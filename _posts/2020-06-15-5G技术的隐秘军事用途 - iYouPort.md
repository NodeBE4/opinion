  <p class="graf graf--p">
   虽然早先的技术已完美地用于创建越来越先进的智能手机，但5G不仅旨在提高其性能，而且还主要用于连接需要大量数据才能自动工作的数字系统。
  </p>
  <p class="graf graf--p">
   最重要的5G应用是在军事领域的。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    5G究竟是干什么用的？见《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/5g%e7%bd%91%e7%bb%9c%e7%9a%84%e5%8f%af%e6%80%95%e6%bd%9c%e5%8a%9b/" href="https://www.iyouport.org/5g%e7%bd%91%e7%bb%9c%e7%9a%84%e5%8f%af%e6%80%95%e6%bd%9c%e5%8a%9b/" rel="noopener noreferrer" target="_blank">
     5G网络的可怕潜力
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   由国防科学委员会出版的《 5G网络技术的国防应用》解释了这项新技术提供的可能性，该委员会为五角大楼提供科学建议，它写道：
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    5G技术的出现为国防部提供了机会，以最低的成本利用该系统的优势满足自身的运营需求。
   </p>
  </blockquote>
  <p class="graf graf--p">
   换句话说，由私人公司建立和激活的5G商业网络将由美国军队使用，其支出要比为了纯粹的军事目标而建立网络所需的支出低得多。
  </p>
  <p class="graf graf--p">
   军事专家预见到，5G系统将在使用高超音速武器方面发挥至关重要的作用 —— 导弹，包括带有核弹头的导弹，其飞行速度超过5马赫（音速的五倍）。
  </p>
  <p class="graf graf--p">
   为了在可变轨迹上引导它们，并在不到一秒的时间内改变方向以避免被拦截，它必须在非常短的时间内收集、识别和传输大量数据。
  </p>
  <p class="graf graf--p">
   如果使用此类武器发动攻击，则激活防御必须采取同样的措施 —— 因此唯一的可能性就是依靠5G自动系统。
  </p>
  <p class="graf graf--p">
   这项新技术还将在作战网络中发挥关键作用。借助在限定区域内同时连接数百万个收发器的能力，它将使军事人员 —— 部门和个人 —— 几乎实时地相互传送有关正在进行的操作的地图、照片和其他信息。
  </p>
  <p class="graf graf--p">
   5G对于特勤部门和特种部队来说也是极为重要的。这将使控制和间谍活动系统的效率大大超过今天使用的系统。
  </p>
  <p class="graf graf--p">
   通过赋予他们基于面部识别和其他特征识别以瞄准目标人的能力，它将提高杀手无人机和战争机器人的杀伤力。
  </p>
  <p class="graf graf--p">
   5G网络作为高科技的武器，也将成为使用新一代武器进行网络攻击和战争行动的目标。
  </p>
  <p class="graf graf--p">
   与美国一样，该技术也是中国和其他国家/地区的重点开发目标。
  </p>
  <p class="graf graf--p">
   因此，有关5G的国际竞争不仅是商业上的。
  </p>
  <p class="graf graf--p">
   5G的军事意义几乎被完全忽略了，批评者，包括许多科学家，仅仅着眼于其带来的辐射问题，而将注意力集中在对健康和环境的有毒影响上。
  </p>
  <p class="graf graf--p">
   对健康的影响当然是最重要的，但必须与普通用户间接资助的对该技术的军事用途的研究联系起来。⚪️
  </p>
