  <p class="graf graf--p">
   进步人士经常赞同“对权力说真话”的想法。但这个概念是危险的。如果从字面上理解并作为一种一心一意的策略进行部署，它只会转移注意力，将注意力从那些滥用权力的人身上夺走。如果政治运动依赖于吸引强权者的道德感，那么政治运动就不可能走得太远。
  </p>
  <p class="graf graf--p">
   虽然小马丁路德金博士注意到了“没有爱的权力是鲁莽和滥用的”，但他也指出，“没有权力的爱情是多愁善感的。” 很多时候，进步积极人士在依赖道德论证来说服当局时，并没有意识到自己的潜在权力。
   <strong class="markup--strong markup--p-strong">
    吸引那些经营无情系统的人的心是很少有效的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   由于经济不平等和军国主义的破坏性影响证明，人道原则在利润驱动的计划中是低优先级的。总的来说，贪婪的权力已经知道它正在做什么 — 从华尔街和大型企业的董事会到五角大楼和所谓“国家安全”的最高层。
  </p>
  <p class="graf graf--p">
   对权力说真话这点很好，但
   <strong class="markup--strong markup--p-strong">
    更重要的是 — — 专注于公共教育工作，动员和组织，在挑战权力的同时说出权力的真相。只有动员起来，我们才能切实有希望克服和摧毁主导权力结构
   </strong>
   。
  </p>
  <p class="graf graf--p">
   1967年，当King博士将“没有权力的爱”描述为“多愁善感”时，表达对越南人民的爱的最有意义的努力包括努力阻止美国政府杀害他们。对强大的精英说实话可能会有所帮助，但对于美国人来说，道德要求是在试图结束战争的过程中互相说真话。
  </p>
  <p class="graf graf--p">
   在2019年，战争系统正在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.commondreams.org/views/2018/12/24/bring-troops-home-also-stop-bombing" href="https://www.commondreams.org/views/2018/12/24/bring-troops-home-also-stop-bombing" rel="noopener noreferrer" target="_blank">
    轰炸许多国家
   </a>
   ，而掠夺性的经济政策正在
   <a class="markup--anchor markup--p-anchor" data-href="https://truthout.org/articles/wall-street-banks-and-angry-citizens-the-inequality-gap-is-growing/" href="https://truthout.org/articles/wall-street-banks-and-angry-citizens-the-inequality-gap-is-growing/" rel="noopener noreferrer" target="_blank">
    肆虐美国和全球大部分地区
   </a>
   。
   <strong class="markup--strong markup--p-strong">
    在不产生可信威胁的情况下推动人文变革就像恳求一样，无效
   </strong>
   。
  </p>
  <p class="graf graf--p">
   许多进步人士一直都能认识到依赖游说或传统的街头抗议活动的局限性。这些活动总是必要的，不应低估它们的重要性。但尽管如此，如果没有对推翻官员的主要挑战的可信威胁，他们仍然不会改变任何。增强的持续杠杆作用是建立选举力量的众多好处之一。
  </p>
  <p class="graf graf--p">
   现在已经有了来自 Rashida Tlaib，Ilhan Omar，Alexandria Ocasio-Cortez 和 Ayanna Pressley 等国会新成员的鼓舞人心的日常领导力。更重要的是，选举他们的活动家们已经展示了当运动变得严肃和有条不紊时可以做些什么。
  </p>
  <p class="graf graf--p">
   在联邦和州一级，选票通常包括党派标签，这种胜利发生在民主党初选中。进步人士并没有被标签剥夺权力，而是学习到如何赢得选举。
  </p>
  <p class="graf graf--p">
   获得权力的基层组织使得 Tlaib，Omar，Ocasio-Cortez 和 Pressley 的选举成为可能，这激励了数千名为经济正义、人权和可持续发展而奋斗的活动家。这四位女性 — 以及其他最近在国会中逐渐到来的女性，如 Pramila Jayapal 和 Ro Khanna- — 正在与全国志同道合的力量形成一个众议院权力集团的核心。
  </p>
  <p class="graf graf--p">
   国会进步核心小组与 Jayapal 共同主持的四届众议员 Mark Pocan 恰如其分地说：“
   <strong class="markup--strong markup--p-strong">
    DC的人认为我们是宇宙的中心，但我们不是 — 选举我们的人才是宇宙的中心
   </strong>
   。当你在各个地区有这种行动时，你真的会有影响力。“最大限度地发挥影响力的方法是在选举舞台内外加强全国各地的进步能力。
  </p>
  <p class="graf graf--p">
   获得选举权可以而且必须成为社会运动的一部分，这些社会运动在一系列行业中从权力当局手中夺取权力 — 同时培养替代机构。持久价值的进步力量源于社区的根深蒂固，
   <strong class="markup--strong markup--p-strong">
    社区具有削弱当前寡头政治的不民主力量的巨大潜力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   可以理解的是，左翼的许多人对权力感到不安，并且真的对想要获得权力感到不舒服。毕竟，普遍的权力往往是压迫的和非法的。激励进步权力的努力往往转化为有毒的机会主义和共同选择。但是，长期滥用权力并不能否定团结创造的权力的实际可能性，其目的是提升人道价值。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “权力永远是危险的，”Edward Abbey 说。 “权力吸引了最坏的情况并腐蚀了最好的。”然而，民主的权力整合提供了机会，
   <strong class="markup--strong markup--p-strong">
    将所有权力视为本身有害的。虽然有害的权力蔑视民主，但合法的权力需要它
   </strong>
   。
  </p>
  <p class="graf graf--p">
   迫在眉睫的转变远远超出了选举职位。经济权力经常归结为政治权力; 公司统治和非民主治理的结构陷入了同一个整体体系。出于必要，各种各样的进步运动现在正在挑战机构使精英权力和群众被剥夺权力永久化的构建。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “权力，正确理解，它是实现目标的能力，”King 博士写道。 “这是实现社会、政治或经济变革所需的力量。从这个意义上说，权力不仅是可取的，而且是必要的，以实现爱与正义的要求。
   </strong>
   “
  </p>
  <p class="graf graf--p">
   态度、假设和理解在很大程度上决定了治理方面急需的变革 — 从市政厅和州立法机构到国会和白宫。利害攸关远远超过谁赢得选举; 实际的民主要求获胜者在任职期间承担责任。
  </p>
  <p class="graf graf--p">
   除非人们能够巩固权力以服务于许多人的利益而不是少数特权阶层的利益，否则，“对权力说真话”并没有太大作用。从下往上运用，权力有可能使民主成为现实。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    社会正义、人权、环境保护、公民自由和和平的任务，需要高标准，只有基层政权才能实现
   </strong>
   。
  </p>
  <p class="graf graf--p">
   我们编译这些文章，希望能为中国的民主追求者提供一些经验 — 关于民主的思考方式。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap graf--startsWithDoubleQuote">
   <span class="graf-dropCap">
    <span class="graf-dropCapQuote">
     “
    </span>
    S
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://truthout.org/articles/speaking-truth-to-power-is-no-substitute-for-taking-power/" href="https://truthout.org/articles/speaking-truth-to-power-is-no-substitute-for-taking-power/" rel="noopener noreferrer" target="_blank">
    peaking Truth to Power” Is No Substitute for Taking Power
   </a>
   . Quests for social justice, human rights, environmental protection, civil liberties and peace will require high standards that only grassroots power can achieve.
  </p>
