  <p class="graf graf--p">
   亚马逊希望用他们的产品填你的家。该公司销售一个可以告诉你天气的声控智能扬声器（Echo），一个可以告诉你谁在门外的监视摄像头（Ring），以及一个可以在电视上播放电影和节目的流媒体（Fire TV）…它还销售其他几十种设备 — 插头，恒温器，灯，真空吸尘器，甚至是微波炉 — 里面装着健谈的语音助手 Alexa。让你家的每一个角落都站上一个老大哥。现在，亚马逊又希望拥有你的智能家居基础设施：路由器。
  </p>
  <p class="graf graf--p">
   亚马逊宣布收购 Eero，该公司生产的路由器系统旨在覆盖难以触及的 Wi-Fi 死角。这是亚马逊寻求拥有智能家居的最有力进攻，它可以借此获得更多更深入的用户隐私数据。
  </p>
  <p class="graf graf--p">
   2017年 Eero
   <a class="markup--anchor markup--p-anchor" data-href="https://blog.eero.com/eero-data-report-october-2017/" href="https://blog.eero.com/eero-data-report-october-2017/" rel="noopener noreferrer" target="_blank">
    报告
   </a>
   收集了来自“数十万”路由器的匿名数据，可以看到他们能收集哪些信息：显示一天中最活跃时段的流量
   <a class="markup--anchor markup--p-anchor" data-href="https://1ve19i29lklo3d9qyr26auc3-wpengine.netdna-ssl.com/wp-content/uploads/2017/10/activenetworks-768x964.jpg" href="https://1ve19i29lklo3d9qyr26auc3-wpengine.netdna-ssl.com/wp-content/uploads/2017/10/activenetworks-768x964.jpg" rel="noopener noreferrer" target="_blank">
    图表
   </a>
   （周日晚上9点到晚上10点），以及连接到 Eero 网络的
   <a class="markup--anchor markup--p-anchor" data-href="https://1ve19i29lklo3d9qyr26auc3-wpengine.netdna-ssl.com/wp-content/uploads/2017/10/devices_data_report-768x840.png" href="https://1ve19i29lklo3d9qyr26auc3-wpengine.netdna-ssl.com/wp-content/uploads/2017/10/devices_data_report-768x840.png" rel="noopener noreferrer" target="_blank">
    前40
   </a>
   个设备（首先是 iPhone，iPad 和 Sonos扬声器）。如果
   <a class="markup--anchor markup--p-anchor" data-href="https://eero.com/shop/home-wifi-system?cvosrc=organic.blog.image&amp;utm_source=blog&amp;utm_medium=organic&amp;utm_campaign=image&amp;cvo_campaign=image" href="https://eero.com/shop/home-wifi-system?cvosrc=organic.blog.image&amp;utm_source=blog&amp;utm_medium=organic&amp;utm_campaign=image&amp;cvo_campaign=image" rel="noopener noreferrer" target="_blank">
    你有超过三个 Eeros
   </a>
   ，公司也可以推断出你有一个大房子。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “一个实体可能拥有您家中的所有设备。在电子前沿基金会（EFF）的工作技术专家 Sydney Li 说，从个人隐私的角度来看，
   <strong class="markup--strong markup--p-strong">
    它可以形成你作为一个人在物理和数字世界中非常完整的形象，这特别令人担忧
   </strong>
   。
  </p>
  <p class="graf graf--p">
   只要您连接到该网络，路由器就可以访问您的浏览历史记录；每当您访问以HTTPS开头的安全网站时，路由器最多可以看到网站的域名（例如
   <a class="markup--anchor markup--p-anchor" data-href="http://google.com/" href="https://google.com/" rel="noreferrer noopener" target="_blank">
    google.com
   </a>
   ），但不能查看其内容，例如您要搜索的内容。“
   <strong class="markup--strong markup--p-strong">
    您的路由器还可以将其看到的网络活动和网络流量与特定设备ID相关联。如果你知道如何改变/欺骗这个ID（MAC地址）可以避免，但是大多数人都不知道该怎么做
   </strong>
   “。
  </p>
  <p class="graf graf--p">
   无论如何，亚马逊可以使用这些数据更好地了解其消费者，并密切关注哪些竞争对手在其客户家中。该公司一直在跟踪购物者的浏览和购买行为，通过 AmazonBasics 等私人品牌以低价销售自有品牌的产品。一些第三方卖家
   <a class="markup--anchor markup--p-anchor" data-href="https://www.buzzfeednews.com/article/leticiamiranda/amazon-sellers-say-the-tech-giant-is-crushing-them-with" href="https://www.buzzfeednews.com/article/leticiamiranda/amazon-sellers-say-the-tech-giant-is-crushing-them-with" rel="noopener noreferrer" target="_blank">
    表示
   </a>
   ，自有品牌计划正在破坏他们在平台上的业务。
  </p>
  <p class="graf graf--p">
   在过去，Eero 将其作为独立硬件公司的地位
   <a class="markup--anchor markup--p-anchor" data-href="https://blog.eero.com/a-peek-inside-the-connected-home/" href="https://blog.eero.com/a-peek-inside-the-connected-home/" rel="noopener noreferrer" target="_blank">
    吹捧为
   </a>
   “对消费者有利”，但这点可能很快就会改变。亚马逊正在销售广告业务，事实上，广告是其增长最快的部分，2018年第四季度的广告销售收入
   <a class="markup--anchor markup--p-anchor" data-href="https://www.mobilemarketer.com/news/amazons-ad-sales-jump-to-34b-in-q4-2018/547393/" href="https://www.mobilemarketer.com/news/amazons-ad-sales-jump-to-34b-in-q4-2018/547393/" rel="noopener noreferrer" target="_blank">
    达34亿美元
   </a>
   ，同比增长95％。
  </p>
  <p class="graf graf--p">
   亚马逊收购 Eero 也表明消费者越来越难以逃避巨头公司的掌控。亚马逊已经可以看到大多数人在网上做的任何事，因为亚马逊
   <a class="markup--anchor markup--p-anchor" data-href="https://gizmodo.com/how-one-little-amazon-error-can-destroy-the-internet-1792828399" href="https://gizmodo.com/how-one-little-amazon-error-can-destroy-the-internet-1792828399" rel="noopener noreferrer" target="_blank">
    为近一半的互联网
   </a>
   提供云托管服务，包括 Netflix 和 Pinterest。 Eero 可以为此提供更详细完整的画面。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    互联网巨头希望统治你周围的一切，并在整个过程中收集尽可能多的个人信息
   </strong>
   — 这种策略对他们来说非常有效。我们的数字生活已经被 Facebook，谷歌，微软，苹果和亚马逊所统治，试图逃避他们的数据收集几乎是不可能的。
  </p>
  <p class="graf graf--p">
   对于那些想要避开亚马逊或 Eero 的人来说，替代方案看起来并不乐观。互联网服务提供商通常向其客户提供跟踪分析数据并向广告商销售网络浏览历史记录和应用使用历史记录。Google 通过自己的网状网络路由器 Google Wi-Fi 来
   <a class="markup--anchor markup--p-anchor" data-href="https://support.google.com/wifi/answer/6246642?hl=en" href="https://support.google.com/wifi/answer/6246642?hl=en" rel="noopener noreferrer" target="_blank">
    收集设备信息
   </a>
   。所有连接的设备、信号强度和数据使用信息都将发送给 Google，但该公司在其隐私页面上称其路由器“不会跟踪您访问的网站或收集网络上任何流量”。但这只是因为它不需要路由器数据。 Google 已经通过其他产品（如 Chrome，Android，Gmail 及其广告网络）收集了大量此类信息。
  </p>
  <p class="graf graf--p">
   隐私专家非常担心，因为同类的，Facebook 收购消息应用程序两年后 WhatsApp 就改变了其隐私政策并开始共享用户信息给 Facebook；在被 Facebook 收购一年后 Instagram 就在该平台上推出了广告。Eero 在此不会有任何区别。
  </p>
  <p class="graf graf--p">
   除了获取更多用户数据之外，考虑到亚马逊的智能家居野心，收购 Eero 是一个切实可行的举措。 Eero 将帮助这家科技巨头尽可能多地在您的家中安插亚马逊品牌的硬件，它可以简化经常复杂的小工具安装，这是阻止消费者购买智能家居设备的最大痛点之一。Eero 可以确保客户的家庭网络具有足够的连接，以便所有亚马逊的设备都能可靠地运行。从理论上讲，亚马逊还可以将 Eero 技术集成到 Echo 设备中，将每个智能扬声器转变为Wi-Fi网状网络节点。
  </p>
  <p class="graf graf--p">
   亚马逊的 Eero 就像特洛伊木马。亚马逊将可以说服客户邀请路由器进入他们的家中，从而令您的家彻底布满老大哥。
  </p>
