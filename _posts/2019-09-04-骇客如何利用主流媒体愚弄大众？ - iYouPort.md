  <p class="graf graf--p">
   我们会在文末做出分析。
  </p>
  <p class="graf graf--p">
   乍看下这篇文章似乎是对“外交政策”杂志的真诚贡献 。
  </p>
  <p class="graf graf--p">
   这篇文章在2017年6月发布，它声称前中央情报局局长迈克尔海登批评在沙特的压力下驱逐穆斯林兄弟会和哈马斯，并且引用海登说美国不应该让一个没有经验的王储（指的是沙特王储 MBS）破坏中东的安全计划。
  </p>
  <p class="graf graf--p">
   但是，这篇文章是伪造的，这是一个与伊朗有联系的虚假信息网络制造的假货，冒充美国“外交政策”杂志，目的是在 2016 年开始并持续至今的
   <strong class="markup--strong markup--p-strong">
    信息战行动
   </strong>
   中诋毁德黑兰的竞争对手。
  </p>
  <p class="graf graf--p">
   根据一份报告，在这个庞大的、开创性的行动中，
   <strong class="markup--strong markup--p-strong">
    与伊朗有关的特工在数十个不同的领域创造了100多份假文章，其中许多都是冒充合法的新闻媒体
   </strong>
   ，推动袭击沙特阿拉伯、以色列和伊朗的虚构故事。这是来自研究机构 Citizen Lab 的
   <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.ca/2019/05/burned-after-reading-endless-mayflys-ephemeral-disinformation-campaign/#appendixa" href="https://citizenlab.ca/2019/05/burned-after-reading-endless-mayflys-ephemeral-disinformation-campaign/#appendixa" rel="noopener noreferrer" target="_blank">
    报告
   </a>
   。
  </p>
  <p class="graf graf--p">
   为了宣传他们的文章，该行动甚至依赖假的 Twitter 角色，在线与记者和研究人员进行交流，并发送链接引导目标到伪造的页面。
  </p>
  <p class="graf graf--p">
   公民实验室确定了11个这样的人物角色。其中之一，“Mona A. Rahman”，在Twitter上非常活跃，称自己是“政治分析家和作家”，并且似乎是一名反沙特活动家。
  </p>
  <p class="graf graf--p">
   那篇虚假的“外交政策”文章以及模仿真实新闻网站的数十个其他虚假网站目前已经不再存在。剩下的只能在网络存档的页面上查看，例如
   <a class="markup--anchor markup--p-anchor" data-href="http://web.archive.org/web/20170620122214/http://foriegnpolicy.net/2017/06/07/ex-cia-chief-michael-hayden-the-regional-arrangements-a-young-inexperienced-saudi/" href="https://web.archive.org/web/20170620122214/http://foriegnpolicy.net/2017/06/07/ex-cia-chief-michael-hayden-the-regional-arrangements-a-young-inexperienced-saudi/" rel="noopener noreferrer" target="_blank">
    这一页
   </a>
   ，显示了捏造的“外交政策”文章。
  </p>
  <p class="graf graf--p">
   在视觉上，
   <strong class="markup--strong markup--p-strong">
    该网站完全复制了“外交政策”的网站，但该文章中充满了语法错误。
   </strong>
  </p>
  <p class="graf graf--p">
   仅仅将注意力集中在俄罗斯身上是偏颇的，这份报告强调了其他国家如何利用宣传和虚假信息在网上传播他们喜欢的叙述方式。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果你把这次行动与我们已经看到的所有其他行动结合在一起，就会发现伊朗已经像俄罗斯一样成为了一个重要的虚假信息宣传战参与者，看起来它们似乎不会消失，”Ben Nimmo 说，他是研究虚假信息的大西洋理事会数字取证研究实验室的高级研究员。
  </p>
  <p class="graf graf--p">
   虽然“伊朗早期的行动相对粗暴，并利用社交媒体引导用户转向反叛内容的网站，”他说，随着他们背后的操作人员尝试传播亲伊朗信息的新方法，这些行动现在变得越来越复杂了。
  </p>
  <p class="graf graf--p">
   该报告的作者警告说，他们不能明确地将这场行动归咎于代表伊朗工作的骇客。
  </p>
  <p class="graf graf--p">
   无法识别运营该虚假网络的个人的身份，但他们在网上传播的信息始终与伊朗的利益和宣传密切相关。
  </p>
  <p class="graf graf--p">
   亲伊朗的宣传活动通过使用公民实验室研究人员称之为“短暂的虚假信息”的策略，在战略性欺骗领域开辟了一个新天地。
  </p>
  <p class="graf graf--p">
   运营者会创建假的新闻页面和假的网站，发布虚假文章，然后在文章开始于社交媒体上获得影响力后删除页面。
  </p>
  <p class="graf graf--p">
   该策略的重点似乎旨在将错误的叙述注入信息（舆论）生态系统，然后删除证据。
  </p>

  <p class="graf graf--p">
   其中提到的重点之一就是：利用欺骗目标人/群体的心理弱点定制最容易被接受的骗局；假象不能完全是假的，要有部分真实才更容易被信任。
  </p>
  <p class="graf graf--p">
   在本案例中，目标群体 — 即互联网人 — 的心理特征是寻求吸睛刺激、不求甚解、仅因信息刺激而乐于转发分享，这就是推动虚假信息传播的有利基础；并且，环境背景是，绝大多数人依旧通过主流媒体获取信息，认知一旦形成便很难更改（也就是说，骇客最终删除的页面并不会明显减轻对人们的影响）。
  </p>
  <p class="graf graf--p">
   请忽略这篇文章中暗示的地缘政治元素，
   <strong class="markup--strong markup--p-strong">
    不论您的政治立场是什么，都不能忽视这种操纵式宣传的危害。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️试想，如果中国骇客伪造了纽约时报的页面、安置了精心策划的并且吸睛度很高内容（比如权斗），是否会“刷爆朋友圈”呢？
  </p>
  <p class="graf graf--p">
   更重要的是，应该有更多人意识到，“主流媒体”本身也极有可能在任何时候成为宣传战的武器，并无需被骇客伪造，详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%aa%92%e4%bd%93%e6%93%8d%e6%8e%a7%ef%bc%9a%e6%88%98%e7%95%a5%e6%80%a7%e6%ac%ba%e9%aa%97%e7%9a%84%e9%87%8d%e5%a4%b4%e6%88%8f/" href="https://www.iyouport.org/%e5%aa%92%e4%bd%93%e6%93%8d%e6%8e%a7%ef%bc%9a%e6%88%98%e7%95%a5%e6%80%a7%e6%ac%ba%e9%aa%97%e7%9a%84%e9%87%8d%e5%a4%b4%e6%88%8f/" rel="noopener noreferrer" target="_blank">
    媒体操控：战略性欺骗的重头戏
   </a>
   》；所以，以渠道为参考条件的判断是远远不足以验证真伪的。
  </p>
  <p class="graf graf--p">
   您需要完善一套思考方式。
  </p>

   关于反欺骗的思考方式和基本注意事项详见上面链接中的解读。
