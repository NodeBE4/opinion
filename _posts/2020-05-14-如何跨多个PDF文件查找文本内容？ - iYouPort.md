  <p class="graf graf--p">
   PDF是绝大多数人共享文档最喜欢的格式。主要是因为它保留了其中所有的文本格式和图形。但PDF格式的唯一缺点是，您无法像使用其他文件一样轻松地在所有这些文件中搜索文本。
  </p>
  <p class="graf graf--p">
   但是，的确有几种方法可以让您执行PDF搜索操作，以便您可以在计算机上一次在多个PDF文件中搜索特定关键字。
  </p>
  <p class="graf graf--p">
   以下指南教您如何做到这一点。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     如何使用 Acrobat Reader 搜索多个PDF文件
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   搜索PDF或多个PDF文件的最流行的方法是使用计算机上的 Acrobat Reader 软件。以下是执行方式。
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    在计算机上打开 Acrobat Reader。
   </li>
   <li class="graf graf--li">
    单击编辑菜单，然后选择高级搜索。
   </li>
   <li class="graf graf--li">
    选择“所有PDF文档”，然后选择您的PDF文件夹。
   </li>
   <li class="graf graf--li">
    在输入框中输入搜索词。
   </li>
   <li class="graf graf--li">
    点击搜索按钮。
   </li>
   <li class="graf graf--li">
    在屏幕上的提示中点击允许。
   </li>
  </ol>
  <p class="graf graf--p">
   该软件将在指定文件夹中的所有PDF文件中搜索给定术语。您将在搜索窗格中直接看到结果。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     在多个PDF文件中搜索特定的搜索词
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   大多数PDF阅读器可让您搜索PDF文件中所需的任何文本。
  </p>
  <p class="graf graf--p">
   这是因为PDF是可扫描的文档，并且与 Microsoft Word 文件一样，它们的字符也可以由您的系统识别。
  </p>
  <p class="graf graf--p">
   不过，使用PDF文件不容易做到的是，您无法一次在多个文件中搜索特定术语。
  </p>
  <p class="graf graf--p">
   按下 Control + F 键只会搜索在软件中打开的当前文档，而不会搜索任何其他PDF文件。
  </p>
  <p class="graf graf--p">
   如果您有很多PDF文件，并且正在寻找特定的文件，最好的选择是使用 Acrobat Reader 的高级搜索功能。它使您可以在计算机上一个位置的所有PDF文件中查找特定的搜索词。
  </p>
  <p class="graf graf--p">
   使用 Acrobat Reader 的整个过程如下：
  </p>
  <p class="graf graf--p">
   如果您尚未在计算机上安装
   <a class="markup--anchor markup--p-anchor" data-href="https://get.adobe.com/reader/" href="https://get.adobe.com/reader/" rel="noopener noreferrer" target="_blank">
    Acrobat Reader
   </a>
   ，请下载并安装它。
  </p>
  <p class="graf graf--p">
   启动软件，您会看到最近的文件列表。您要查找的选项位于“编辑”菜单中。单击顶部的“编辑”菜单，然后选择“高级搜索”选项。或者，您可以按 Shift + Ctrl +F。
  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   在接下来的屏幕上，将选项设置为以下内容：
  </p>
  <p class="graf graf--p">
   您想在哪里搜寻？ —— 选择“所有PDF文档”，然后使用下拉菜单选择保存多个PDF的文件夹。
  </p>
  <p class="graf graf--p">
   您想搜索什么单词或短语？ —— 在PDF文件中输入要搜索的单词。
  </p>
  <p class="graf graf--p">
   您可以使用其他选项来自定义单词的搜索方式，例如在 “区分大小写” 复选框上打勾，以使搜索查询区分大小写，依此类推。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     使用 Windows 搜索在PDF文件中进行搜索
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   也可以使用 Windows 计算机上的默认搜索选项来搜索PDF文件。
  </p>
  <p class="graf graf--p">
   转到控制面板&gt;索引选项，然后单击高级。
  </p>

  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     使用 SeekFast 搜索PDF文件
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://seekfast.org/" href="https://seekfast.org/" rel="noopener noreferrer" target="_blank">
    SeekFast
   </a>
   还使您可以轻松搜索各种文件类型（包括PDF）中的关键字。运作方式如下。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    在计算机上下载并安装该软件。
   </li>
   <li class="graf graf--li">
    启动软件，在顶部的搜索框中输入搜索词，单击浏览按钮以选择您的PDF文件夹，最后点击搜索。
   </li>
  </ul>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11302 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/6-2.png?resize=1100%2C431&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/6-2.png?w=1388&amp;ssl=1 1388w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/6-2.png?resize=300%2C118&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/6-2.png?resize=1024%2C401&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/6-2.png?resize=768%2C301&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/6-2.png?resize=1100%2C431&amp;ssl=1 1100w" data-recalc-dims="1" height="431" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/6-2.png?resize=1100%2C431&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11302" data-recalc-dims="1" height="431" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/6-2.png?resize=1100%2C431&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/6-2.png?w=1388&amp;ssl=1 1388w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/6-2.png?resize=300%2C118&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/6-2.png?resize=1024%2C401&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/6-2.png?resize=768%2C301&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/6-2.png?resize=1100%2C431&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   它将在所选目录中的所有PDF文件中搜索关键字。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     使用 Foxit Reader 执行PDF搜索
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.foxitsoftware.com/pdf-reader/" href="https://www.foxitsoftware.com/pdf-reader/" rel="noopener noreferrer" target="_blank">
    Foxit Reader
   </a>
   还具有高级搜索功能，您可以使用它在多个PDF文件中查找所需内容。
  </p>
  <p class="graf graf--p">
   获取计算机上安装的软件，然后启动它。
  </p>
  <p class="graf graf--p">
   单击右上角菜单中搜索框旁边的搜索图标。它会打开高级搜索选项。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11303 jetpack-lazy-image" data-lazy-sizes="(max-width: 806px) 100vw, 806px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/7-2.png?resize=806%2C402&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/7-2.png?w=806&amp;ssl=1 806w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/7-2.png?resize=300%2C150&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/7-2.png?resize=768%2C383&amp;ssl=1 768w" data-recalc-dims="1" height="402" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/7-2.png?resize=806%2C402&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="806"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11303" data-recalc-dims="1" height="402" sizes="(max-width: 806px) 100vw, 806px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/7-2.png?resize=806%2C402&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/7-2.png?w=806&amp;ssl=1 806w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/7-2.png?resize=300%2C150&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/7-2.png?resize=768%2C383&amp;ssl=1 768w" width="806"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在以下屏幕上，从第一个下拉菜单中选择您的PDF文件夹，在搜索框中输入搜索词，如果要应用其他过滤器，请选中该标记，最后单击该“搜索”按钮。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11304 jetpack-lazy-image" data-lazy-sizes="(max-width: 580px) 100vw, 580px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/8-1.png?resize=580%2C790&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/8-1.png?w=580&amp;ssl=1 580w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/8-1.png?resize=220%2C300&amp;ssl=1 220w" data-recalc-dims="1" height="790" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/8-1.png?resize=580%2C790&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="580"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11304" data-recalc-dims="1" height="790" sizes="(max-width: 580px) 100vw, 580px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/8-1.png?resize=580%2C790&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/8-1.png?w=580&amp;ssl=1 580w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/8-1.png?resize=220%2C300&amp;ssl=1 220w" width="580"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   您的搜索结果现在应该在屏幕上。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     如何使用 UltraFinder 搜索PDF文件
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.ultraedit.com/downloads/ultrafinder.html" href="https://www.ultraedit.com/downloads/ultrafinder.html" rel="noopener noreferrer" target="_blank">
    UltraFinder
   </a>
   是Windows的高级搜索工具，也可用于在PDF文件中搜索文本。
  </p>
  <p class="graf graf--p">
   在 Windows 计算机上安装并启动该工具。
  </p>
  <p class="graf graf--p">
   如下设置选项，以便搜索PDF文件的内容。然后点击绿色的按钮开始搜索。
  </p>
  <p class="graf graf--p">
   Find matches by — 文件内容
  </p>
  <p class="graf graf--p">
   Find what — 在这里输入您的搜索词
  </p>
  <p class="graf graf--p">
   Find where — 添加PDF所在的目录
  </p>
  <p class="graf graf--p">
   In files/types — *.pdf
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11305 jetpack-lazy-image" data-lazy-sizes="(max-width: 902px) 100vw, 902px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/9-1.png?resize=902%2C1384&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/9-1.png?w=902&amp;ssl=1 902w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/9-1.png?resize=196%2C300&amp;ssl=1 196w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/9-1.png?resize=667%2C1024&amp;ssl=1 667w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/9-1.png?resize=768%2C1178&amp;ssl=1 768w" data-recalc-dims="1" height="1384" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/9-1.png?resize=902%2C1384&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="902"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11305" data-recalc-dims="1" height="1384" sizes="(max-width: 902px) 100vw, 902px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/9-1.png?resize=902%2C1384&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/9-1.png?w=902&amp;ssl=1 902w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/9-1.png?resize=196%2C300&amp;ssl=1 196w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/9-1.png?resize=667%2C1024&amp;ssl=1 667w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/9-1.png?resize=768%2C1178&amp;ssl=1 768w" width="902"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   它将列出在其中找到您的搜索词的所有PDF文件。
  </p>

   就是这样。很简单吧？希望对您有帮助。⚪️
