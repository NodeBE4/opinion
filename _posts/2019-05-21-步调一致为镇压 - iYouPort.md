 <div class="entry-content">
  <ul>
   <li class="graf graf--p">
    <span style="color: #00ccff;">
     <em>
      中国和日本究竟是不是敌对关系？不知道。但至少在一件事上，这俩国家异口同声地默契起来了，那就是：镇压。在哪里实现的？
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   当津巴布韦的军队将罗伯特穆加贝推下台时，欢呼的居民在哈拉雷的街道上庆祝；而上个月，在燃料价格翻倍的抗议活动发生数天之后，安全部队发起了大规模的镇压行动，造成12人死亡，600人被捕。津巴布韦政府还下令在全国范围内首次关闭互联网。
  </p>
  <p class="graf graf--p">
   该国最大的移动服务提供商 Econet Wireless 表示，在政府下令后，该服务已暂停。该公司在给客户的短信中说：“我们有义务在被指示的情况下采取行动，而且这件事是我们无法控制的。” 在一篇 Facebook 帖子中，1999年成立 Econet Wireless 之后发了财的流亡津巴布韦亿万富翁斯蒂夫·马西瓦（Strive Masiyiwa）称：“政府行为”迫使该公司关闭互联网连接。他补充说，他被告知“如果不遵守规定将导致我们的管理人员被监禁三年。”
  </p>
  <p class="graf graf--p">
   最近该国恢复了部分互联网服务，但 Facebook，WhatsApp 和 Twitter 等社交媒体应用和消息服务仍然被封锁。
  </p>
  <p class="graf graf--p">
   津巴布韦对互联网的攻击是源于政府最近开始试图将其意志强加给普通的津巴布韦人，引起了观察家和活动家的恐惧。一年前，即将上任的总统 Emmerson Mnangagwa 承诺启动一个“新民主国家”并结束该国的政治分歧。在“纽约时报”发表的一篇文章中，Mnangagwa 发誓要将津巴布韦变成一个“经济蓬勃发展、青年就业、投资者机会、民主和所有人平等权利”的国家。
  </p>
  <p class="graf graf--p">
   在穆加贝下台后的那一年 — — 曾经此人宣称“只有上帝会驱逐我” — — 津巴布韦的经济问题日益恶化，因为该国正面临巨额债务，基础设施受损，食品价格飙升，恶性通货膨胀和失业。
  </p>
  <p class="graf graf--p">
   由于未能带来繁荣，津巴布韦政府现在似乎决心在其数字和公共空间的各个方面建立统治权。结果是，
   <strong class="markup--strong markup--p-strong">
    该国邀请中国和日本的公司帮助建立一个监视国家，活动家和研究人员担心未来的情况可能比穆加贝时代更加糟糕
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “关闭互联网已成为一种先发制人的策略。罗伯特·穆加贝（Robert Mugabe）因侵犯人权行为而受到谴责，但他至少没有下令断网。而新政府没有表现出任何保护权利的政治意愿，“Jeffrey Smith 说，他是倡导开放民主基金会 Vanguard Africa 的创始人，对津巴布韦特别感兴趣。
  </p>
  <p class="graf graf--p">
   津巴布韦律师阿瑟·格瓦格（Arthur Gwagwa）是斯特拉斯莫尔大学法学院的网络威胁建模专家和开放技术基金会研究员，他说，最近互联网关闭可能是未来侵犯人权的预兆。“逮捕、折磨和殴打是在关闭互联网的阴影下进行的，所以人们都很害怕，”他说。
  </p>
  <p class="graf graf--p">
   津巴布韦对在线和离线空间的控制主要依赖于外国供应商，这些供应商为政府提供了大量新的网络控制工具。去年11月，
   <strong class="markup--strong markup--p-strong">
    日本驻津巴布韦大使 Toshiyuki Iwado 向津巴布韦财政部长 Mthuli Ncube 提供了价值360万美元的网络监视设备
   </strong>
   。据日本驻津巴布韦大使馆称，该补助金是日本“网络防范犯罪设备供应补助金项目”的一部分。
  </p>
  <p class="graf graf--p">
   津巴布韦的通信技术部长更加关注该技术的用途。Kazembe Kazembe 说，警方将部署它以推动数字取证和面部识别系统，并引入一个新的数据共享平台。该技术还将允许政府利用电子通信来防止犯罪和动物走私进入津巴布韦。
  </p>
  <p class="graf graf--p">
   但是，技术专家担心，日本的拨款可能会用来帮助政府打击异议。
  </p>
  <p class="graf graf--p">
   Gwagwa 说：“虽然日本并不是津巴布韦扼杀其人民的盟友，但日本可能无法控制其技术可能被采用的邪恶目标。” 他补充说，“政府眼中的民主活动家可能被归类为野生动物走私者的一部分。”
  </p>
  <p class="graf graf--p">
   并且，日本合同只是同一波进口监控技术的一部分。另一部分则是：中国。中国最近参与建立了国家人工智能面部识别数据库。根据官媒环球时报称，
   <strong class="markup--strong markup--p-strong">
    总部位于广州的创业公司
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.biometricupdate.com/201805/implementation-of-cloudwalk-facial-recognition-technology-in-zimbabwe-progressing-in-stages" href="https://www.biometricupdate.com/201805/implementation-of-cloudwalk-facial-recognition-technology-in-zimbabwe-progressing-in-stages" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     CloudWalk Technology
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    去年与津巴布韦签署了一项协议，提供全国范围内布设的面部识别监视软件
   </strong>
   。
  </p>
  <p class="graf graf--p">
   该协议是“一带一路”的一部分，这是中国的所谓软实力计划，将扩展到建设机场、铁路和公交车站以及智能金融系统等基础设施。中国历来是津巴布韦的亲密盟友，是该国陷入困境的经济中
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iol.co.za/business-report/chinas-influence-on-zimbabwe-is-going-nowhere-12109858" href="https://www.iol.co.za/business-report/chinas-influence-on-zimbabwe-is-going-nowhere-12109858" rel="noopener noreferrer" target="_blank">
    最大的金融投资者
   </a>
   。中国已投资数十亿美元用于钻石和铂金矿、新高速公路和发电大坝。
  </p>
  <p class="graf graf--p">
   这笔交易可能意味着津巴布韦
   <a class="markup--anchor markup--p-anchor" data-href="https://www.techzim.co.zw/2018/04/mnangagwas-govt-getting-facial-recognition-tech-from-china/" href="https://www.techzim.co.zw/2018/04/mnangagwas-govt-getting-facial-recognition-tech-from-china/" rel="noopener noreferrer" target="_blank">
    是该地区首批
   </a>
   采用这种技术的国家之一。CloudWalk 使用3D光线面部识别软件，在识别深色皮肤的脸部时比传统的面部识别技术效果更好。至关重要的是，
   <strong class="markup--strong markup--p-strong">
    这笔交易将让 CloudWalk 根据津巴布韦公民的生物数据训练算法。由此产生的信息将帮助中国建立世界上最全面和种族最多样化的面部识别数据库之一
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这让活动人士深感担忧。从本质上讲，津巴布韦可能会向一个不负责任的海外科技巨头赠送大量关于津巴布韦公民的私人数据。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “（这些交易）允许北京利用发展中国家作为实验室来改进其监测技术，”Gwagwa 说。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    关于这点的更多详细介绍，见去年的报道《
    <a class="markup--anchor markup--li-anchor" data-href="https://steemit.com/tech/@iyouport/dqcwj-news" href="https://steemit.com/tech/@iyouport/dqcwj-news" rel="noopener noreferrer" target="_blank">
     非洲人的脸被用来训练中国老大哥：北京正在努力将其监视技术输出到发展中国家
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   这是更大模式的一部分。Gwagwa 表示，对于中国而言，人工智能技术的出口有多种用途。“这是双赢的外交 — — 中国人工智能公司在收集和保留 DNA 的过程中，可以在肤色较深的非洲人身上训练他们的算法，同时建立数据中心以刺激数字贸易并帮助传统的经济交易。”
  </p>
  <p class="graf graf--p">
   CloudWalk 的官员没有回应 Coda 的采访请求，但它告诉该国的环球时报说，“津巴布韦政府并非纯粹是为人工智能或面部识别技术来广州的，而是对基础设施等领域有一个全面的一揽子计划，包括技术与生物学。“
  </p>
  <p class="graf graf--p">
   中国公司也在
   <a class="markup--anchor markup--p-anchor" data-href="http://www.eurasiainfo.ch/en/china-designed-big-data-system-aids-angolas-intelligent-governance/" href="http://www.eurasiainfo.ch/en/china-designed-big-data-system-aids-angolas-intelligent-governance/" rel="noopener noreferrer" target="_blank">
    安哥拉
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hrw.org/news/2014/03/25/ethiopia-telecom-surveillance-chills-rights" href="https://www.hrw.org/news/2014/03/25/ethiopia-telecom-surveillance-chills-rights" rel="noopener noreferrer" target="_blank">
    埃塞俄比亚
   </a>
   签署了类似的协议。
  </p>
  <p class="graf graf--p">
   另一项中国交易涉及津巴布韦共和国警方，该警察部队已经加入了专制技术行列，并将在哈拉雷引入强大的夜视监控摄像头；所有驾驶员被要求安装仪表板摄像头并将驾驶违规的视频
   <a class="markup--anchor markup--p-anchor" data-href="https://www.dropbox.com/request/T7zgjX6KQUxL7zhgwtKW" href="https://www.dropbox.com/request/T7zgjX6KQUxL7zhgwtKW" rel="noopener noreferrer" target="_blank">
    上传到 Dropbox 文件夹
   </a>
   。据警方称，目标是“遏制违规驾驶和犯罪行为”。在 Dropbox 中保存有捕捉到的驾驶者和公民的镜头、姓名和电子邮件地址等信息。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    中国最臭名昭著的海康威视也向警方提供高科技监控摄像头。海康威视一直在与新疆的大规模监控项目密切合作
   </strong>
   ，中国政府迫害维吾尔族穆斯林。海康威视在乌鲁木齐赢得的
   <a class="markup--anchor markup--p-anchor" data-href="http://www1.hikvision.com/cn/news_detail_63_i2394.html" href="http://www1.hikvision.com/cn/news_detail_63_i2394.html" rel="noopener noreferrer" target="_blank">
    一个项目
   </a>
   价值7900万美元，并包含约30,000个监视摄像头。海康威视的项目还利用视频分析中心、
   <a class="markup--anchor markup--p-anchor" data-href="http://www.sohu.com/a/205838858_389742" href="https://www.sohu.com/a/205838858_389742" rel="noopener noreferrer" target="_blank">
    大数据中心
   </a>
   、
   <a class="markup--anchor markup--p-anchor" data-href="http://www.bidchance.com/info.do?channel=calgg&amp;id=20166604" href="http://www.bidchance.com/info.do?channel=calgg&amp;id=20166604" rel="noopener noreferrer" target="_blank">
    警察检查站
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="http://www.xj.cei.gov.cn/info/10856/340992.htm" href="http://www.xj.cei.gov.cn/info/10856/340992.htm" rel="noopener noreferrer" target="_blank">
    无人机
   </a>
   。
  </p>
  <p class="graf graf--p">
   哈拉雷市议会还分别
   <a class="markup--anchor markup--p-anchor" data-href="https://www.newsday.co.zw/2018/08/chinese-company-to-mount-surveillance-cameras/" href="https://www.newsday.co.zw/2018/08/chinese-company-to-mount-surveillance-cameras/" rel="noopener noreferrer" target="_blank">
    花费200万美元
   </a>
   在整个城市的交通信号灯上安装海康威视的监控摄像头。“这些摄像机将帮助识别交通违规者，特别是那些阻碍交通畅通的人，”该市总工程师 George Munyonga 表示。摄像机将监控中央商务区内的所有道路和停车位。目前监视网络已经安装完成了70％。
  </p>
  <p class="graf graf--p">
   总部位于约翰内斯堡的数字技术中心 iAfrikan 的创始人兼首席执行官 Tefo Mohapi 担心津巴布韦越来越依赖中国创建的监控系统。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “有许多切实的担忧，特别是关于像津巴布韦这样的政府已经证明它可以用任何手段侵犯公民权利，”他说。“其中之一就是限制那些反对国家的人的行动。”
  </p>
  <p class="graf graf--p">
   津巴布韦国防军指挥官长期以来一直热衷于这种技术。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “作为一支军队，我们正在训练我们的军官，以便能够应对这种新的威胁，我们称之为网络战，其中武器不一定是枪支，但基本上是信息和通信技术，”Valerio Sibanda 说。
  </p>
  <p class="graf graf--p">
   在其他时候，军队充分利用了互联网接入的优势。至关重要的是，在政变期间，军队允许不受限制地访问网络，以确保全球接受军队占领权力的叙述。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    请注意这句话
   </strong>
   ：“军队不是削减对网络的访问权限，而是允许自由流动的在线信息；但是，精心设计的话语和意见，“Gwagwa 说。“它利用软实力，让公民享有社交媒体自由[的假象]，作为一种允许快速交换信息的大战略。“
  </p>
  <p class="graf graf--p">
   Gwagwa 将军队的战略与土耳其的例子进行了对比，土耳其总统雷杰普·埃尔多安使用 Facetime 来抵抗2016年的未遂政变。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <strong class="markup--strong markup--p-strong">
    当数字信息具有高价值时，权威主导信息操纵是高度动态的。在津巴布韦，军队通过精心设计的方案精心挑选群众，“Gwagwa 说。“公众很容易接受这些叙述
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   更令人担忧的是，他认为军队成功操纵数字技术以适应其描绘的“开放与和平”权力转移的目标，可以在非洲范围内广泛传播。
  </p>
