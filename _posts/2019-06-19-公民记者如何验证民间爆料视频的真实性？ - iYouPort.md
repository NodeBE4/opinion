  <p class="graf graf--p">
   经常有民间自称目击者的网友提供“现场视频”，并声称在某些时刻发生过什么。但它是真的吗？分辨真伪对公民记者来说是个强大的考验。
  </p>
  <p class="graf graf--p">
   大多数公民记者没有资金，难以亲赴现场验证；而有些内容可能是非常关键的证据，事关人命，于是需要技巧来确定它们是真实的。
  </p>
  <p class="graf graf--p">
   这可能是一个非常具有挑战性的过程。
  </p>
  <p class="graf graf--p">
   通过数字社交网络分享的错误信息对于任何现代新闻编辑室来说都是一个严重的问题。
  </p>
  <p class="graf graf--p">
   我们曾经介绍过验证视频和照片的方法，见这里《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E8%99%9A%E5%81%87%E4%BF%A1%E6%81%AF%E9%89%B4%E5%88%AB%E5%B7%A5%E5%85%B7%E7%AE%B1%EF%BC%9A%E5%A6%82%E4%BD%95%E9%AA%8C%E8%AF%81%E5%9B%BE%E7%89%87%E5%92%8C%E8%A7%86%E9%A2%91%E7%9A%84%E7%9C%9F%E4%BC%AA/" href="https://www.iyouport.org/%E8%99%9A%E5%81%87%E4%BF%A1%E6%81%AF%E9%89%B4%E5%88%AB%E5%B7%A5%E5%85%B7%E7%AE%B1%EF%BC%9A%E5%A6%82%E4%BD%95%E9%AA%8C%E8%AF%81%E5%9B%BE%E7%89%87%E5%92%8C%E8%A7%86%E9%A2%91%E7%9A%84%E7%9C%9F%E4%BC%AA/" rel="noopener noreferrer" target="_blank">
    虚假信息鉴别工具箱：如何验证图片和视频的真伪？
   </a>
   》，这里还有更全面的信息验证工具《超
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%F0%9F%92%8E-%E8%B6%85%E7%BA%A7%E6%83%85%E6%8A%A5%E6%94%B6%E9%9B%86%E5%B7%A5%E5%85%B7%E5%BA%93%EF%BC%9A%E5%BC%80%E6%BA%90%E9%AA%8C%E8%AF%81%E5%92%8C%E8%B0%83%E6%9F%A5%E5%B7%A5%E5%85%B7%E5%8F%8A/" href="https://www.iyouport.org/%F0%9F%92%8E-%E8%B6%85%E7%BA%A7%E6%83%85%E6%8A%A5%E6%94%B6%E9%9B%86%E5%B7%A5%E5%85%B7%E5%BA%93%EF%BC%9A%E5%BC%80%E6%BA%90%E9%AA%8C%E8%AF%81%E5%92%8C%E8%B0%83%E6%9F%A5%E5%B7%A5%E5%85%B7%E5%8F%8A/" rel="noopener noreferrer" target="_blank">
    级情报收集工具库：开源验证和调查工具及使用方法
   </a>
   》。
  </p>
  <p class="graf graf--p">
   本文介绍一个
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2018/09/04/reader-center/social-media-video-how-to-verify.html" href="https://www.nytimes.com/2018/09/04/reader-center/social-media-video-how-to-verify.html" rel="noopener noreferrer" target="_blank">
    纽约时报的案例
   </a>
   演示视频，使用的也是开源调查的方法，相对比较简单，公民记者应该掌握这些基本技巧。
  </p>
  <p class="graf graf--p">
   尤其是思路，比工具更重要。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    跟踪视觉效果
   </strong>
  </p>
  <p class="graf graf--p">
   记者经常会通过 Twitter、Facebook、YouTube、Snapchat 等等社交媒体平台获取信息，尤其是目击者传播的视频。所有这些内容都需要仔细验证。
  </p>
  <p class="graf graf--p">
   下面是一个通过 WhatsApp 传播的视频的案例。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5885 jetpack-lazy-image" data-lazy-sizes="(max-width: 750px) 100vw, 750px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-9.jpg?resize=750%2C1334&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-9.jpg?w=750&amp;ssl=1 750w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-9.jpg?resize=169%2C300&amp;ssl=1 169w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-9.jpg?resize=576%2C1024&amp;ssl=1 576w" data-recalc-dims="1" height="1334" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-9.jpg?resize=750%2C1334&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="750"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5885" data-recalc-dims="1" height="1334" sizes="(max-width: 750px) 100vw, 750px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-9.jpg?resize=750%2C1334&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-9.jpg?w=750&amp;ssl=1 750w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-9.jpg?resize=169%2C300&amp;ssl=1 169w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-9.jpg?resize=576%2C1024&amp;ssl=1 576w" width="750"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   使用 WhatsApp，六个消息来源之一的一名叙利亚医生证实了叙利亚化学武器袭击现场附近医院隧道入口的位置。
  </p>
  <p class="graf graf--p">
   验证过程分为两个一般步骤：首先，确定视频是否真的是新的，而不是旧视频声称新事件；接下来剖析每一个框架，以得出有关位置、日期和时间的准确信息，考证所涉及的参与者、以及究竟发生了什么。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、它是新的吗？
   </strong>
  </p>
  <p class="graf graf--p">
   在突发事件出现时，使用旧视频冒充新事件的情况是非常常见的。
  </p>
  <p class="graf graf--p">
   比如前两天的四川地震，社交网络上很多人使用的是台湾花莲地震的旧照片声称是四川。
  </p>
  <p class="graf graf--p">
   这点在任何国家都一样。
  </p>
  <p class="graf graf--p">
   当美国在4月13日晚对叙利亚发动空袭时，社交媒体上有一些小视频开始流传；其中一段视频声称在大马士革发生了一系列爆炸事件。然而验证显示，该视频是2015年2月乌克兰城市卢汉斯科发生的战斗。
  </p>
  <p class="graf graf--p">
   这是个影响严重的错误，
   <strong class="markup--strong markup--p-strong">
    一些大牌媒体也上当了
   </strong>
   ，他们发出了错误的报道。
  </p>
  <p class="graf graf--p">
   是因为他们不会验证吗？很可能并不是，因为验证一点都不难。这是抢首发的盲目造成的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    对于公民记者来说，真实性尤其重要，因为你们没有强大的政治后台、也没有几百万美元的宣传经费可用，真实性是你们唯一的公信力来源。
   </strong>
  </p>
  <p class="graf graf--p">
   如果你不想被撒谎的职业媒体压倒，就应该比他们技高一筹。
  </p>
  <p class="graf graf--p">
   方法和工具在前文那两个链接中。
  </p>
  <p class="graf graf--p">
   不论是暴力事件还是非暴力事件，都会有人使用陈旧的图片和视频冒充正在发生的事；一定要提高警惕，带着批判性意识采纳信息。
  </p>
  <p class="graf graf--p">
   尤其是，
   <strong class="markup--strong markup--p-strong">
    越激烈的、越戏剧化的视频越应该小心
   </strong>
   ，因为这样的视频由于画面刺激会更加高速传播，欺骗者经常利用这点来推动虚假信息。
  </p>
  <p class="graf graf--p">
   在完美的状况下，验证需要包括获取原始视频文件或查找在线共享该视频的第一个版本，审查上传者的数字足迹、并联系该人 —— 如果能这样做是最安全的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、逐帧分析
   </strong>
  </p>
  <p class="graf graf--p">
   如果初步认定视频是新的，上传者的地理位置的确在事发现场附近；下面要做的工作就是逐帧分析。
  </p>
  <p class="graf graf--p">
   但是你经常会遇到一些比较极端的情况，比如武装冲突、强大的国家审查压制，这会导致联系目击者变得很困难。
  </p>
  <p class="graf graf--p">
   于是公民记者应该拥有自己的一套技能和方法来独立确认或证实视频中可见的内容。
  </p>
  <p class="graf graf--p">
   在此案例中方法是很简单的，因为原始视频文件附带元数据 —— 嵌入文件的详细信息，包括使用的相机或手机型号、日期和时间，有时甚至是显示该地点的精确坐标。
  </p>
  <p class="graf graf--p">
   要知道这是极为幸运的事。很多时候，尤其是通过社交媒体如 Facebook 或 Twitter 、加密通讯如 WhatsApp、signal 等获取的视频，会被篡改或者抹去元数据。
  </p>
  <p class="graf graf--p">
   这种时候必须在视频中寻找有关位置和日期的视觉线索。
  </p>
  <p class="graf graf--p">
   在此案例中第一步查找事发地点。
  </p>
  <p class="graf graf--p">
   广角镜头的视频通常会显示道路、桥梁或不同建筑物等地标，或显示山脉或河流等地理特征。
  </p>
  <p class="graf graf--p">
   这就是线索。
  </p>
  <p class="graf graf--p">
   所有这些都可以与参考资料匹配，如卫星图像、街景和地理标记照片，以确定事件发生的大致或准确位置。
  </p>
  <p class="graf graf--p">
   这是一个定位过程的视频演示作为参考：
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="576" src="//player.vimeo.com/video/343148828?title=0&amp;byline=0&amp;portrait=0&amp;color=8dc7dc" width="700">
   </iframe>
  </p>
  <p class="graf graf--p">
   虽然这听起来很容易，但很多时候你在视频中看到的通常都是路灯、交通标志或树木。
  </p>
  <p class="graf graf--p">
   这就比较有挑战性了。但是即便是路灯那种简单的东西也可以做到验证线索。
  </p>
  <p class="graf graf--p">
   你看过《国土安全》那部电视剧吗？其中有一个细节：恐怖分子绑架了 Quinn，把他放在玻璃房子里做毒气攻击实验。你还记得 Carrie Mathiso 是怎么找到 Quinn 被绑架关押的具体位置以营救他的吗？
  </p>
  <p class="graf graf--p">
   没错，地砖。
  </p>
  <p class="graf graf--p">
   那是个室内视频，没有任何地标类的视觉线索；Carrie 是通过视频中露出地砖特点以找到具体地点的 —— 那个地砖是哪里生产的？主要在哪里销售？
  </p>
  <p class="graf graf--p">
   这是常用的思考方式。尼日利亚东北部 Maiduguri 的法外处决暴行视频证据，就是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amnesty.org/en/latest/news/2015/06/how-technology-helped-us-expose-war-crimes-in-nigeria/" href="https://www.amnesty.org/en/latest/news/2015/06/how-technology-helped-us-expose-war-crimes-in-nigeria/" rel="noopener noreferrer" target="_blank">
    通过路灯来验证的
   </a>
   。
  </p>
  <p class="graf graf--p">
   💡找到视频中任何一个细节的特点，抓住它，使用一切工具查找该特点有可能出现的地方。
  </p>
  <p class="graf graf--p">
   并且要注意音频，因为当地方言可能有助于证实一般位置；如果没有对话，请注意视频中出现的其他背景声音，比如汽车繁杂的声音意味着地理位置接近某高速公路；如果有飞机的声音，说明位置接近机场，等等，你一定对此很有经验。
  </p>
  <p class="graf graf--p">
   下一步是验证日期和时间。
  </p>
  <p class="graf graf--p">
   确定事件发生的确切日期和时间可能更具挑战性。
  </p>
  <p class="graf graf--p">
   可以使用历史天气数据来检测视频传播者所讲述的事发时间是否存在不一致性，但当然这并不能提供确切的日期。
  </p>
  <p class="graf graf--p">
   这就是阴影帮上忙的时候了（我们会
   <strong>
    在后面的文章计划中详细介绍阴影的使用方法
   </strong>
   ）。
  </p>
  <p class="graf graf--p">
   阴影是确定时间的好办法。一种名为
   <a class="markup--anchor markup--p-anchor" data-href="http://suncalc.net/#/51.508,-0.125,2/2019.06.19/07:58" href="http://suncalc.net/#/51.508,-0.125,2/2019.06.19/07:58" rel="noopener noreferrer" target="_blank">
    SunCalc
   </a>
   的工具可以提供帮助，就是这个链接，它的网站很容易操作，可以在给定的一天的给定位置上确定光线的位置。这样就很容易确定时间了。
  </p>
  <p class="graf graf--p">
   之后就是确定事件和人员。
  </p>
  <p class="graf graf--p">
   依旧是仔细观察视频中出现的细节，任何一个微小的细节都可以帮上忙。
  </p>
  <p class="graf graf--p">
   其中人员会佩戴徽章吗？他们穿着制服吗？周围的物品上有什么标志吗？
  </p>
  <p class="graf graf--p">
   在这里看到通过迷彩服图案确定战斗人员身份的演示《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E5%A6%82%E4%BD%95%E4%BD%BF%E7%94%A8%E6%95%B0%E5%AD%97%E6%8A%80%E5%B7%A7%E5%9C%A8%E7%BA%BF%E9%AA%8C%E8%AF%81%E4%B8%AD%E4%B8%9C%E5%86%B2%E7%AA%81%E4%B8%AD%E7%9A%84%E6%88%98%E6%96%97%E4%BC%99%E4%BC%B4/" href="https://www.iyouport.org/%E5%A6%82%E4%BD%95%E4%BD%BF%E7%94%A8%E6%95%B0%E5%AD%97%E6%8A%80%E5%B7%A7%E5%9C%A8%E7%BA%BF%E9%AA%8C%E8%AF%81%E4%B8%AD%E4%B8%9C%E5%86%B2%E7%AA%81%E4%B8%AD%E7%9A%84%E6%88%98%E6%96%97%E4%BC%99%E4%BC%B4/" rel="noopener noreferrer" target="_blank">
    如何使用数字技巧在线验证中东冲突中的战斗伙伴关系
   </a>
   》
  </p>
  <p class="graf graf--p">
   这里还有一个很棒的细节线索调查演示，验证大屠杀的证据《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E7%BB%86%E8%8A%82%E7%9A%84%E7%BA%BF%E7%B4%A2%E9%AD%85%E5%8A%9B%EF%BC%9A%E5%A6%82%E4%BD%95%E7%94%A8%E8%A7%86%E9%A2%91%E5%88%9B%E5%BB%BA%E5%85%A8%E6%99%AF%E5%9B%BE%EF%BC%9Fosint-%E5%BC%80%E6%BA%90/" href="https://www.iyouport.org/%E7%BB%86%E8%8A%82%E7%9A%84%E7%BA%BF%E7%B4%A2%E9%AD%85%E5%8A%9B%EF%BC%9A%E5%A6%82%E4%BD%95%E7%94%A8%E8%A7%86%E9%A2%91%E5%88%9B%E5%BB%BA%E5%85%A8%E6%99%AF%E5%9B%BE%EF%BC%9Fosint-%E5%BC%80%E6%BA%90/" rel="noopener noreferrer" target="_blank">
    细节线索的魅力
   </a>
   》
  </p>
  <p class="graf graf--p">
   还有一个在线侦查的演示案例，通过一个头像挖掘网络水军的线索《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E7%BD%91%E7%BB%9C%E6%B0%B4%E5%86%9B%E5%A6%82%E4%BD%95%E8%AF%95%E5%9B%BE%E5%BD%B1%E5%93%8D%E9%80%89%E4%B8%BE%EF%BC%9F-%E4%B8%80%E4%B8%AA%E8%B0%83%E6%9F%A5%E6%A1%88%E4%BE%8B%EF%BC%8C%E8%A7%82%E5%AF%9F/" href="https://www.iyouport.org/%E7%BD%91%E7%BB%9C%E6%B0%B4%E5%86%9B%E5%A6%82%E4%BD%95%E8%AF%95%E5%9B%BE%E5%BD%B1%E5%93%8D%E9%80%89%E4%B8%BE%EF%BC%9F-%E4%B8%80%E4%B8%AA%E8%B0%83%E6%9F%A5%E6%A1%88%E4%BE%8B%EF%BC%8C%E8%A7%82%E5%AF%9F/" rel="noopener noreferrer" target="_blank">
    网络水军如何试图影响选举？
   </a>
   》
  </p>
  <p class="graf graf--p">
   这些演示是希望介绍思路 ——
   <strong class="markup--strong markup--p-strong">
    掌握思路是比工具本身更重要的事
   </strong>
   。
  </p>
  <p class="graf graf--p">
   工具非常多，这里有一个汇总《超
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%F0%9F%92%8E-%E8%B6%85%E7%BA%A7%E6%83%85%E6%8A%A5%E6%94%B6%E9%9B%86%E5%B7%A5%E5%85%B7%E5%BA%93%EF%BC%9A%E5%BC%80%E6%BA%90%E9%AA%8C%E8%AF%81%E5%92%8C%E8%B0%83%E6%9F%A5%E5%B7%A5%E5%85%B7%E5%8F%8A/" href="https://www.iyouport.org/%F0%9F%92%8E-%E8%B6%85%E7%BA%A7%E6%83%85%E6%8A%A5%E6%94%B6%E9%9B%86%E5%B7%A5%E5%85%B7%E5%BA%93%EF%BC%9A%E5%BC%80%E6%BA%90%E9%AA%8C%E8%AF%81%E5%92%8C%E8%B0%83%E6%9F%A5%E5%B7%A5%E5%85%B7%E5%8F%8A/" rel="noopener noreferrer" target="_blank">
    级情报收集工具库
   </a>
   》，但是如果您没有经验缺乏思路，有可能不知道该从哪里开始。
  </p>
  <p class="graf graf--p">
   所以，多练习，多参考专业人士的演示，很快就可以积累经验。
  </p>
  <p class="graf graf--p">
   在全球范围内，公民媒体正在彻底改变人权事实的调查，为人权监督机构以及法院的调查开辟了新的可能性。
  </p>
  <p class="graf graf--p">
   随着公民新闻和新技术的出现呈指数增长，人权调查人员能够以前所未有的速度查找和挖掘战争罪和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%8d%b1%e5%ae%b3%e4%ba%ba%e7%b1%bb%e7%bd%aa%e7%9a%84%e6%83%a9%e6%b2%bb%e6%96%b9%e6%a1%88/" href="https://www.iyouport.org/%e5%8d%b1%e5%ae%b3%e4%ba%ba%e7%b1%bb%e7%bd%aa%e7%9a%84%e6%83%a9%e6%b2%bb%e6%96%b9%e6%a1%88/" rel="noopener noreferrer" target="_blank">
    危害人类罪
   </a>
   的证据。
  </p>
  <p class="graf graf--p">
   在中国发生的很多事都缺乏证据，以至于受害者难以申诉。我们认为这是不公平的。公民记者和人权活动家应该努力为捍卫真相加强技能。
  </p>

   IYP会尽可能提供知识技巧给您，关注我们的

    OSINT栏目

   以获取它们。该栏目会持续更新下去。⚪️
