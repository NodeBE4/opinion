   点击农场是什么？一个阴暗的互联网行业仍在中国蓬勃发展
  </h1>
 </header>

 <div class="entry-content">
  <p class="graf graf--p">
   我们曾经介绍过《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e4%b8%8a%e7%a9%b6%e7%ab%9f%e6%9c%89%e5%a4%9a%e5%b0%91%e4%b8%9c%e8%a5%bf%e6%98%af%e5%81%87%e7%9a%84%ef%bc%9f/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e4%b8%8a%e7%a9%b6%e7%ab%9f%e6%9c%89%e5%a4%9a%e5%b0%91%e4%b8%9c%e8%a5%bf%e6%98%af%e5%81%87%e7%9a%84%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    互联网上究竟有多少东西是假的？
   </a>
   》曾经看似确定无疑真实的所有东西现在似乎都是假的； 曾经看起来有点假的所有东西现在都具有真实的力量和存在。这并不奇怪。
  </p>
  <p class="graf graf--p">
   在中国，已经实现这种造假很久了。其中一个典型就是：点击农场。
  </p>
  <p class="graf graf--p">
   在中国这个拥有超过8亿用户的全球最大的智能手机市场，一个独特的农场在城市地区涌现。那里唯一的作物是：智能手机。
  </p>
  <p class="graf graf--p">
   这些操作被称为点击农场，可以在货架上容纳数百甚至数千部 iPhone 和 Android 手机。
  </p>
  <p class="graf graf--p">
   它们被插入并编程为一遍又一遍地搜索、单击和下载某个应用程序。目标是操纵应用商店排名、愚弄搜索结果。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8584 jetpack-lazy-image" data-lazy-sizes="(max-width: 618px) 100vw, 618px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-17.jpg?resize=618%2C298&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-17.jpg?w=618&amp;ssl=1 618w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-17.jpg?resize=300%2C145&amp;ssl=1 300w" data-recalc-dims="1" height="298" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-17.jpg?resize=618%2C298&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="618"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8584" data-recalc-dims="1" height="298" sizes="(max-width: 618px) 100vw, 618px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-17.jpg?resize=618%2C298&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-17.jpg?w=618&amp;ssl=1 618w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-17.jpg?resize=300%2C145&amp;ssl=1 300w" width="618"/>
   </noscript>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “假冒的观点如此广泛，”南华早报于2018年报道说，“
   <strong class="markup--strong markup--p-strong">
    视频网站上许多热门节目产生的观看次数中有90％都是假的
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   应用程序开发商购买这项“服务”以提高其产品的知名度 —— 争取在中国500亿美元的在线广告市场中占据更大份额。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    点击农场如何运作
   </strong>
  </p>
  <p class="graf graf--p">
   在流量驱动的数字经济中，点击农场的用例正在扩展，已经不仅仅是推广应用程序了。
  </p>
  <p class="graf graf--p">
   所谓的内容创业者也在使用这种欺骗的方法来夸大其内容的“知名度”以吸引广告客户，提升视频观看次数，并在电子商务平台上操纵搜索结果。
  </p>
  <p class="graf graf--p">
   延伸：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%9c%a8%e4%b8%ad%e5%9b%bd%ef%bc%9a%e5%86%85%e5%ae%b9%e5%88%9b%e4%b8%9a%e8%80%85%e7%9a%84%e8%83%8c%e9%9d%a2/" href="https://www.iyouport.org/%e5%9c%a8%e4%b8%ad%e5%9b%bd%ef%bc%9a%e5%86%85%e5%ae%b9%e5%88%9b%e4%b8%9a%e8%80%85%e7%9a%84%e8%83%8c%e9%9d%a2/" rel="noopener noreferrer" target="_blank">
    在中国：内容创业者的背面
   </a>
   》
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8585 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-17.jpeg?resize=800%2C532&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-17.jpeg?w=800&amp;ssl=1 800w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-17.jpeg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-17.jpeg?resize=768%2C511&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-17.jpeg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="532" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-17.jpeg?resize=800%2C532&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8585" data-recalc-dims="1" height="532" sizes="(max-width: 800px) 100vw, 800px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-17.jpeg?resize=800%2C532&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-17.jpeg?w=800&amp;ssl=1 800w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-17.jpeg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-17.jpeg?resize=768%2C511&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-17.jpeg?resize=272%2C182&amp;ssl=1 272w" width="800"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   点击农场使用自动化流程攻击正常的 App Store 优化（ASO）实践 —— 这需要开发人员在描述中使用某些关键字并通过成为有用的产品来吸引用户 —— 点击农场则是通过模仿真实用户来推动某些关键字、点击应用程序、下载、甚至撰写正面评论。
  </p>
  <p class="graf graf--p">
   操作这个农场需要硬件和软件的投资。要购买数百部 iPhone（主要是二手机）并不便宜。
  </p>
  <p class="graf graf--p">
   点击农场还使用可能入侵应用程序商店系统的服务器和软件。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8586 jetpack-lazy-image" data-lazy-sizes="(max-width: 1050px) 100vw, 1050px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-7.jpg?resize=1050%2C527&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-7.jpg?w=1050&amp;ssl=1 1050w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-7.jpg?resize=300%2C151&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-7.jpg?resize=768%2C385&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-7.jpg?resize=1024%2C514&amp;ssl=1 1024w" data-recalc-dims="1" height="527" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-7.jpg?resize=1050%2C527&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1050"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8586" data-recalc-dims="1" height="527" sizes="(max-width: 1050px) 100vw, 1050px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-7.jpg?resize=1050%2C527&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-7.jpg?w=1050&amp;ssl=1 1050w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-7.jpg?resize=300%2C151&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-7.jpg?resize=768%2C385&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-7.jpg?resize=1024%2C514&amp;ssl=1 1024w" width="1050"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   点击农场在中国被视为非法，并且一直存在被 Apple 等应用商店检测和阻止的风险。
  </p>
  <p class="graf graf--p">
   尽管如此，点击农场集群依旧被发现存在于深圳，这个中国南方大都市被称为电子产品中心。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    点击农场是有效的 — 特别是在众包的情况下
   </strong>
  </p>
  <p class="graf graf--p">
   对于点击农场运营商来说，这是与 Apple 不断发展的算法的持续战斗。
  </p>
  <p class="graf graf--p">
   在一则在线广告中，点击农场运营商声称他们已经更新了计划，因为 Apple 在2016年破解了这些行为，并给予实际下载更多的权重。
  </p>
  <p class="graf graf--p">
   视频：
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="618" src="//player.vimeo.com/video/353511749?title=0&amp;byline=0&amp;portrait=0&amp;color=8dc7dc" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   因此，点击农业进化了，并开始涉及人群众包。
  </p>
  <p class="graf graf--p">
   在中国流行的消息应用程序微信和QQ上，聚集在群聊中的人们声称他们的任务就是下载和查看某些固定的应用程序，并因此获得报酬。
  </p>
  <p class="graf graf--p">
   人工操作的成本通常高于机器操作，但众包看起来更像是有机流量，不太可能被应用商店阻止。
  </p>
  <p class="graf graf--p">
   尽管存在风险，但从点击农场购买服务仍被视为在短时间内推广应用程序的有效且相对经济的方式。
  </p>
  <p class="graf graf--p">
   例如，游戏应用程序是点击农场的主要客户，因为用户更有可能浏览排名靠前的新游戏。一些付费应用程序也可以用这种方法快速吸引更多下载。
  </p>
  <p class="graf graf--p">
   总之，你被操纵了，还误以为自己“赶上了潮流”而欣欣然得意着。⚪️
  </p>
