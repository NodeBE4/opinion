      <strong>
       不仅仅是读书， 您可以做任何事
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-10195 jetpack-lazy-image" data-lazy-sizes="(max-width: 1008px) 100vw, 1008px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/书.jpg?resize=1008%2C435&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/书.jpg?w=1008&amp;ssl=1 1008w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/书.jpg?resize=300%2C129&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/书.jpg?resize=768%2C331&amp;ssl=1 768w" data-recalc-dims="1" height="435" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/书.jpg?resize=1008%2C435&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1008"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10195" data-recalc-dims="1" height="435" sizes="(max-width: 1008px) 100vw, 1008px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/书.jpg?resize=1008%2C435&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/书.jpg?w=1008&amp;ssl=1 1008w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/书.jpg?resize=300%2C129&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/书.jpg?resize=768%2C331&amp;ssl=1 768w" width="1008"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   从在 Instagram 上浏览照片到在 Twitter 上分享文章，社交媒体已成为每个人生活中无处不在的一部分。
  </p>
  <p class="graf graf--p">
   大多数人浏览 Facebook 等应用程序所花掉的时间都比他们自己所认为的时间要长很多，最宝贵的精力就这么流失掉了。
  </p>
  <p class="graf graf--p">
   根据
   <a class="markup--anchor markup--p-anchor" data-href="https://www.globalwebindex.com/reports/social" href="https://www.globalwebindex.com/reports/social" rel="noopener noreferrer" target="_blank">
    GlobalWebIndex
   </a>
   的《 2019年旗舰报告》，全球数字消费者平均每天在社交媒体上花费2个小时23分钟。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10196 jetpack-lazy-image" data-lazy-sizes="(max-width: 850px) 100vw, 850px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-30.png?resize=850%2C320&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-30.png?w=850&amp;ssl=1 850w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-30.png?resize=300%2C113&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-30.png?resize=768%2C289&amp;ssl=1 768w" data-recalc-dims="1" height="320" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-30.png?resize=850%2C320&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="850"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10196" data-recalc-dims="1" height="320" sizes="(max-width: 850px) 100vw, 850px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-30.png?resize=850%2C320&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-30.png?w=850&amp;ssl=1 850w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-30.png?resize=300%2C113&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-30.png?resize=768%2C289&amp;ssl=1 768w" width="850"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   但是潮流似乎正在转变，该报告还显示了人们越来越多地意识到
   <strong class="markup--strong markup--p-strong">
    社交媒体的成瘾性是一个大问题
   </strong>
   。
  </p>
  <p class="graf graf--p">
   追踪社交媒体使用情况的新应用正在激增。其中之一是来自
   <a class="markup--anchor markup--p-anchor" data-href="https://www.omnicalculator.com/everyday-life/social-media-time-alternatives" href="https://www.omnicalculator.com/everyday-life/social-media-time-alternatives" rel="noopener noreferrer" target="_blank">
    Omni Calculator
   </a>
   的工具，它可以计算您的屏幕使用时间，并告诉您：如果把这些时间用来专注于阅读，您可以读完多少本书。
  </p>
  <p class="graf graf--p">
   计算器显示，如果每天能减少三次10分钟的社交媒体使用，意味着您每年可以阅读多达30本书。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10197 jetpack-lazy-image" data-lazy-sizes="(max-width: 344px) 100vw, 344px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-29.png?resize=344%2C346&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-29.png?w=344&amp;ssl=1 344w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-29.png?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-29.png?resize=298%2C300&amp;ssl=1 298w" data-recalc-dims="1" height="346" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-29.png?resize=344%2C346&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="344"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10197" data-recalc-dims="1" height="346" sizes="(max-width: 344px) 100vw, 344px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-29.png?resize=344%2C346&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-29.png?w=344&amp;ssl=1 344w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-29.png?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-29.png?resize=298%2C300&amp;ssl=1 298w" width="344"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   数字成瘾问题是关键的难题之一，于是数字监测工具也是当下的主要趋势。
  </p>
  <p class="graf graf--p">
   Google用户现在可以查看其屏幕时间的数据，Android 用户可以设置应用程序时间限制；苹果、Facebook 和 Instagram 都提供类似的功能。
  </p>
  <p class="graf graf--p">
   据谷歌
   <a class="markup--anchor markup--p-anchor" data-href="https://www.globalwebindex.com/hubfs/Downloads/2019%20Q1%20Social%20Flagship%20Report.pdf?utm_campaign=Social%20report%20July%202019&amp;utm_source=hs_automation&amp;utm_medium=email&amp;utm_content=74226065&amp;_hsenc=p2ANqtz-9QeYNRjY0iiRyeK3c_LfQ9mQpzMjPhHYhdCJo2aJtqqxqafLL5jsi4B8UyZW-iIb9zkaf-72Ze0qyIvjpPwVSRz0iqHFgGOmpKtMh3DG6PQMkOUk0&amp;_hsmi=74226065" href="https://www.globalwebindex.com/hubfs/Downloads/2019%20Q1%20Social%20Flagship%20Report.pdf?utm_campaign=Social%20report%20July%202019&amp;utm_source=hs_automation&amp;utm_medium=email&amp;utm_content=74226065&amp;_hsenc=p2ANqtz-9QeYNRjY0iiRyeK3c_LfQ9mQpzMjPhHYhdCJo2aJtqqxqafLL5jsi4B8UyZW-iIb9zkaf-72Ze0qyIvjpPwVSRz0iqHFgGOmpKtMh3DG6PQMkOUk0&amp;_hsmi=74226065" rel="noopener noreferrer" target="_blank">
    称
   </a>
   ，如果使用数字健康跟踪应用程序，超过75％的人会对自己的手机使用“感到更满意”。
  </p>
  <p class="graf graf--p">
   Omni 计算器还研究了其他方法，计算如果您可以重新利用在社交媒体上花费的时间去进行锻炼，可以燃烧多少卡路里，或实现可以掌握的其他技能。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    近年来，对抗数字成瘾问题的社会运动在全球各地发展
   </strong>
   ，因为人们真的意识到了这是一个非常严重的问题。关于为什么，见：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%9c%80%e5%ae%8c%e7%be%8e%e7%9a%84%e7%bb%91%e6%9e%b6/" href="https://www.iyouport.org/%e6%9c%80%e5%ae%8c%e7%be%8e%e7%9a%84%e7%bb%91%e6%9e%b6/" rel="noopener noreferrer" target="_blank">
    最完美的绑架
   </a>
   》。
  </p>
  <p class="graf graf--p">
   建议您关闭屏幕上显示的推送通知、删除一些应用程序，多与您的社区组织面对面交流而不是使用通讯工具，并有意识地限制自己的社交媒体时间。
  </p>
  <p class="graf graf--p">
   社交媒体也许能有很多好处，但是，集中化社交媒体的操纵只能是弊大于利，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%9a%90%e6%80%a7%e6%93%8d%e6%8e%a7%ef%bc%9a%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e5%b9%b3%e5%8f%b0%e6%ad%a3%e5%9c%a8%e4%bd%bf%e7%94%a8%e4%b8%8e%e8%b5%8c%e5%8d%9a%e5%85%ac%e5%8f%b8%e7%9b%b8%e5%90%8c/" href="https://www.iyouport.org/%e9%9a%90%e6%80%a7%e6%93%8d%e6%8e%a7%ef%bc%9a%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e5%b9%b3%e5%8f%b0%e6%ad%a3%e5%9c%a8%e4%bd%bf%e7%94%a8%e4%b8%8e%e8%b5%8c%e5%8d%9a%e5%85%ac%e5%8f%b8%e7%9b%b8%e5%90%8c/" rel="noopener noreferrer" target="_blank">
    隐性操控：社交媒体平台正在使用与赌博公司相同的技术只为让你上瘾
   </a>
   》。
  </p>
  <p class="graf graf--p">
   即使您只是能做到掌控自己的使用时间，对于摆脱 matrix 也是很有好处的。《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e4%bb%8e-matrix-%e4%b8%ad%e9%86%92%e6%9d%a5%ef%bc%9f/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e4%bb%8e-matrix-%e4%b8%ad%e9%86%92%e6%9d%a5%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    如何从 Matrix 中醒来？
   </a>
   》。⚪️
  </p>
