<div id="entry">
<div class="at-above-post addthis_tool" data-url="http://zhanlve.org/?p=6277">

<p>
<img alt="藏人行政中央司政洛桑森格在美国国家民主基金会发表演讲。(2018年11月13日)" class="aligncenter" src="https://boxun.com/news/images/2018/11/201811150104intl1.jpg"/>
</p>
<p>
  藏人行政中央司政洛桑森格在美国国家民主基金会发表演讲
 </p>
<p>
</p>
<p>
<span style="font-size: 12pt;">
   华盛顿 — 美国国家民主基金会（National Endowment for Democracy, NED）11月13日举办讨论会，邀请藏人行政中央司政洛桑森格与美国学者一同探讨中国的“一带一路倡议”对参与国可能带来的潜在风险。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   洛桑森格在研讨会上以中国政府在西藏建设基础设施为例，表示在1950年代时，有的西藏民众起初欢迎中国政府在当地进行建设公路的举措，但这位藏人流亡政府领导人称，中国政府随后对西藏进行了“殖民化”，大批汉人入藏，占据了当地的经济、政治等领域的主导地位，当地的自然资源也被开采出来。洛桑森格认为，公路等基础设施建设使藏人遭遇了损失，而不是从中获益。他表示，中国政府在处理西藏问题上的战略与其推广“一带一路”倡议背后的战略“惊人地相似”（Strikingly similar）。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   洛桑森格称，中国政府在西藏进行基础设施投资旨在保持对当地的“战略控制”。他说当年在西藏修建的一条公路现在已经变成了连接西藏各地的“一百条公路”，铁路线将拉萨与青海连通，将西藏北部和东南部地区与中国内陆连通，到2030年，中国还计划建成新的铁路线，将四川、甘肃、云南等地与西藏相连。此外，洛桑森格说，西藏现在已经建成有30座机场和六座军用机场。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   洛桑森格表示，中国的“一带一路倡议”在国际社会中已经迎来一些人士的强烈抵制。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   他说：“幸运的是，至少在民众的层面，已经出现了一些质疑和反弹。你在哈萨克斯坦、斯里兰卡、马来西亚，甚至巴基斯坦和尼泊尔都能看到这样的情况出现。”
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   洛桑森格称，多位欧盟的驻华大使也对中国的“一带一路”倡议表达了批评的意见。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   洛桑森格在接受美国之音采访时表示，他认为国际社会需要“睁大眼睛”审视“一带一路倡议”。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   他说： “其他国家应该睁大眼睛审视‘一带一路’倡议，并且以透明地方式谈判协议，以便更多地获益而不是更少。这是我想传递的信息。”
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   参与此次讨论的美国国家民主基金会“民主研究国际论坛”主任香缇·卡拉提认为，关注“一带一路”的民主国家除了提出批评意见外，还应该提供替代方案。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   她说：“那些有能力这么做的国家应该提供替代方案，不应仅仅说这样很糟，还应该说这些是我们寻求的条件：透明、负责任以及可以讨论所有这些倡议的一个自由、开放的公共空间。”
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   另一方面，中国政府说，“一带一路”是合作共赢，造福沿线各国人民。中国外交部发言人华春莹曾表示，中国始终坚持“平等开放透明，坚持以企业为主体的市场化运作，按照市场规律和通行的国际规则行事，每一个项目都是有关各方平等协商的结果”。她说，“一带一路”合作项目遵循“共商共建共享”的原则，已经为项目伙伴国带来22亿美元税收，创造20多万个就业岗位。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   华春莹说，“一带一路”获得了相关国家的“真心欢迎”。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   在西藏问题上，中国政府坚持西藏事务属于中国内政，不容外国干涉的立场。中国外交部发言人陆慷曾表示，西藏60多年来各项事业取得了“前所未有的成就”，社会经济取得了“长足发展”。陆慷强调，中国政府和人民捍卫国家主权、维护民族团结的意志“坚定不移”。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   随着中国“一带一路”倡议不断推进，中国的这一战略规划正在受到国际社会越来越多的关注。与此同时，有消息称，美国正计划设立一个名为“美国国际发展金融公司”（United States International Development Finance Corporation，IDFC）的新机构，为发展中国家提供贷款，协助这些国家发展基础设施项目。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   有分析人士认为，美国这一计划成立的新机构或将能够抗衡中国的“一带一路”倡议。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   洛桑森格对美国之音说，美国和欧洲国家应让贷款更加易得。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   他说： “我无法说这一新机构的目标是什么。但我确实认为美国和欧洲国家应让用于基础设施的贷款更加易得。基础设施建设非常重要，更多的贷款会更有助益。”
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
</span>
</p>
