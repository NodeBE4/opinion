  <p class="graf graf--p">
   中国的自动驾驶汽车正在美国为乘客服务。
  </p>
  <p class="graf graf--p">
   根据在加州公共事业委员会网站上发布的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cpuc.ca.gov/avcissued/" href="https://www.cpuc.ca.gov/avcissued/" rel="noopener noreferrer" target="_blank">
    通知
   </a>
   ，AutoX 和 Pony.ai 已经成为加州首批获准提供 robotaxis 的中国公司。
  </p>
  <p class="graf graf--p">
   AutoX 由普林斯顿大学教授肖健雄于2016年创立，被他的学生称为“ X教授”。
  </p>
  <p class="graf graf--p">
   AutoX 现在是中国资金最雄厚的自动驾驶初创企业之一，与 Pony.ai 一起成立。Pony.ai 是由百度的两位前高管于2016年共同创立的自动驾驶部门。
  </p>
  <p class="graf graf--p">
   AutoX 去年表示，它正在与投资者进行谈判，以筹集高达
   <a class="markup--anchor markup--p-anchor" data-href="https://www.scmp.com/tech/venture-capital/article/2181857/hong-kong-self-driving-start-autox-talks-raise-100-million" href="https://www.scmp.com/tech/venture-capital/article/2181857/hong-kong-self-driving-start-autox-talks-raise-100-million" rel="noopener noreferrer" target="_blank">
    1亿美元
   </a>
   的资金。截至去年4月，Pony.ai 已筹集了至少
   <a class="markup--anchor markup--p-anchor" data-href="https://techcrunch.com/2019/04/12/kunlun-invests-50m-in-pony-ai/" href="https://techcrunch.com/2019/04/12/kunlun-invests-50m-in-pony-ai/" rel="noopener noreferrer" target="_blank">
    2.14亿美元
   </a>
   的资金。
  </p>
  <p class="graf graf--p">
   虽然有超过62家公司持有在加利福尼亚州进行自动驾驶汽车测试的许可证，但是实际上很少有公司被允许使用这些汽车载人。
  </p>
  <p class="graf graf--p">
   Zoox 在去年获得了该州首个提供 robotaxi 服务的绿灯时，就跨越了一个新的里程碑。
  </p>
  <p class="graf graf--p">
   不过有一些问题。授予这三家公司的许可类型为 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cpuc.ca.gov/avcpilotinfo/" href="https://www.cpuc.ca.gov/avcpilotinfo/" rel="noopener noreferrer" target="_blank">
    Drivered AV Passenger Service,
   </a>
   ”，该服务禁止公司向乘客收取试乘费用，并要求人类驾驶者坐在驾驶座上。
  </p>
  <p class="graf graf--p">
   到目前为止，尚未允许任何实体在加利福尼亚州运营真正的无人驾驶客运服务，这表明监管机构尚未准备好让科技公司实施真正的无人运输。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-10412 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/11-3.png?resize=1100%2C387&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/11-3.png?w=1388&amp;ssl=1 1388w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/11-3.png?resize=300%2C105&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/11-3.png?resize=768%2C270&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/11-3.png?resize=1024%2C360&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/11-3.png?resize=1100%2C387&amp;ssl=1 1100w" data-recalc-dims="1" height="387" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/11-3.png?resize=1100%2C387&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10412" data-recalc-dims="1" height="387" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/11-3.png?resize=1100%2C387&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/11-3.png?w=1388&amp;ssl=1 1388w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/11-3.png?resize=300%2C105&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/11-3.png?resize=768%2C270&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/11-3.png?resize=1024%2C360&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/11-3.png?resize=1100%2C387&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   AutoX 已经开始使用自动驾驶汽车在圣何塞运送杂货，通过引入加利福尼亚州的第一个 robotaxi 服务，它已经抢占了先机。
  </p>
  <p class="graf graf--p">
   网站上的指示说，居住在圣何塞或圣塔克拉拉北部的人们现在可以申请加入该公司的早期运营计划并提供反馈。
  </p>
  <p class="graf graf--p">
   Pony.ai 的发言人告诉 TechCrunch，该公司在获得许可后也开始提供 robotaxi 服务。
  </p>
  <p class="graf graf--p">
   Alphabet 的 Waymo 于2018年已经在凤凰城启动了一项客运服务，并表示，如果需要，它将有训练有素的驾驶员为乘客提供帮助。
  </p>
  <p class="graf graf--p">
   Waymo 可以向旅客收费，但只能运送经过审查的人员，因此该计划并非适用于所有人。
  </p>
  <p class="graf graf--p">
   考虑到批评家提出的法律和道德问题，这些限制似乎是明智的。
  </p>
  <p class="graf graf--p">
   2018年优步（Uber）的自动驾驶测试车在坦普尔（Temple）撞死了一名妇女，促使该巨头暂停了其试驾。
  </p>
  <p class="graf graf--p">
   这一事件已成为该领域初创企业的警惕故事。以 Momenta 为例，这是中国第一家估值
   <a class="markup--anchor markup--p-anchor" data-href="https://techcrunch.com/2019/06/13/momenta-profile/" href="https://techcrunch.com/2019/06/13/momenta-profile/" rel="noopener noreferrer" target="_blank">
    超过10亿美元
   </a>
   的自动驾驶创业公司。其CEO要求所有高管亲自驾驶最少的自主里程，显示管理层对乘客安全的重视。
  </p>
  <p class="graf graf--p">
   中国初创公司渴望获得加州执照的原因有很多。
  </p>
  <p class="graf graf--p">
   最重要的是，自动驾驶汽车天生就需要更多的数据。全球只有极少数的城市允许在公共道路上使用自动驾驶，因此只要有许可，它总是有助于收集更多的数据。
  </p>
  <p class="graf graf--p">
   这些中国公司中的许多还在加利福尼亚州建立了研发中心，以吸引该地区的技术人才。例如，Pony .ai 在硅谷、北京和广州都部署了研发人员和办事处。 AutoX 今年初在深圳开设了研发中心，但仍将开发团队留在圣何塞。⚪️
  </p>

    California has let two Chinese startups offer robotaxis to the public
