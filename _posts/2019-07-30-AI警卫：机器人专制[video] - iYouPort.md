  <p class="graf graf--p">
   一个高度安全性的中国监狱已经在每个牢房、每个角落都养了一个守卫，他们不睡觉、不吃饭、不眨眼。因为他们是机器。
  </p>
  <p class="graf graf--p">
   Yancheng 监狱，由司法部直接管理的一个设施，通过采用人工智能管理囚犯，朝着机器人专制迈出了重要的一步。
  </p>
  <p class="graf graf--p">
   摄像机网络将能够监控囚犯的每一个动作，并标记任何被认为是异常的或令人类卫兵担心的事物。
  </p>
  <p class="graf graf--p">
   一些专家甚至认为该系统将使关押其中的人无法逃脱，因为即使囚犯能够贿赂警卫，他们也无法阻止系统触发警报。
  </p>
  <p class="graf graf--p">
   但是，尽管监狱对其囚犯进行了全天候的不断监视，但由于其内部相对舒适的条件，该监狱之前被描述为“豪华监狱”。
  </p>
  <p class="graf graf--p">
   这里有一些备受瞩目的被关押者，比如谷开来、芮成刚、以及铁道部副总工程师张曙光、前国家足协副主席南勇。
  </p>
  <p class="graf graf--p">
   该监狱还拥有数目不详的外国人，许多国家的外交官已被允许访问以检查条件。
  </p>
  <p class="graf graf--p">
   但在过去的一年里，该监狱的管理层一直面临着来自高层的压力，以改善其运营。
  </p>
  <p class="graf graf--p">
   12月，一个党派纪检小组警告说，该监狱没有正确理解“它在新时代的政治性”，并警告说，一些警卫“经常违反规则”。
  </p>
  <p class="graf graf--p">
   目前尚不清楚哪些具体规则被违反了，但警告称人类警卫的政治忠诚和职业道德受到质疑。
  </p>
  <p class="graf graf--p">
   监狱已经安装了人工智能网络，与人类卫兵不同，它应该可以完全“冷静地”对待囚犯。
  </p>
  <p class="graf graf--p">
   经过数月的密集建设，河北燕郊40公顷设施的监控系统升级工作即将完成，该项目涉及的几个消息来源于向南华早报确认。
  </p>
  <p class="graf graf--p">
   新的“智能监狱”系统涉及一个监控摄像头网络和隐藏式传感器网络，就像其中一个消息来源所说的那样，覆盖范围像“神经元纤维”。
  </p>
  <p class="graf graf--p">
   该监视网络持续收集数据并将数据流传输到“大脑”，这是一台快速运转的人工智能计算机，能够全天候识别、跟踪和监控每个犯人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在每天结束时，系统使用不同的 AI 功能（如面部识别和运动分析）为每个囚犯生成一份综合报告，包括行为分析。
   </strong>
  </p>
  <p class="graf graf--p">
   大多数情况下这些报告都会被存档，但如果检测到任何异常，机器将标记警报。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “例如，”该项目代表说，“如果发现一名囚犯在一个房间内踱步了一段时间，机器可能会认为这种现象是可疑的，并建议一名人类警员进行近距离检查”。
  </p>
  <p class="graf graf--p">
   他解释说，
   <strong class="markup--strong markup--p-strong">
    这种行为可能表明囚犯焦虑或兴奋或有可能以意想不到的方式行事。
   </strong>
  </p>
  <p class="graf graf--p">
   据中国媒体称，Yancheng 监狱去年为止有1600多名囚犯被关押。近年来，由于习近平发起的大规模反腐运动，许多高级官员最终进了监狱，这一数字一直在迅速增加。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该人工智能监视系统由天津大学等众多公共研究机构和位于天津的全球最大的监控技术公司之一 Tiandy 共同开发
   </strong>
   。Tiandy 的一名员工 Meng 声称，无论囚犯人数有多大，该系统都会知道每个人在哪里以及他或她在做什么 — — 并且不需要人类看守监视器。
  </p>
  <p class="graf graf--p">
   这部分归功于先进的面部识别技术，该技术能够同时处理大量的监视目标。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “最先进的技术允许每台摄像机同时跟踪多达200个面孔，”他说。所以囚犯可能会出现在人群中、走廊里、忙碌的食堂、运动会、甚至大规模的斗殴现场，但他们永远无法溜走。
   </strong>
  </p>
  <p class="graf graf--p">
   即使他们能够贿赂监狱看守以帮助他们逃脱，但 AI 系统仍会检测到这种情况，并引发警报。
  </p>
  <p class="graf graf--p">
   简单说就是，连狱警一起监视了。
  </p>
  <p class="graf graf--p">
   Meng 说，
   <strong class="markup--strong markup--p-strong">
    Tiandy 正在与一些南美国家进行谈判，将这项技术引入有暴力和安全漏洞历史的任何监狱
   </strong>
   ，但由于美国政府禁止使用中国的技术，他们进入美国市场的机会有限。
  </p>
  <p class="graf graf--p">
   关于 Tiandy，这里有一篇早前的报道：在中国，有一条迅速的《
   <a href="https://www.iyouport.org/%e5%9c%a8%e4%b8%ad%e5%9b%bd%ef%bc%9a%e8%87%b4%e5%af%8c%e7%9a%84%e6%8d%b7%e5%be%84/" rel="noopener noreferrer" target="_blank">
    致富捷径
   </a>
   》
  </p>
  <p class="graf graf--p">
   但是，每周七天，每天24小时使用高科技监测囚犯，引发了深刻的道德问题。
  </p>
  <p class="graf graf--p">
   联合国人权事务规则规定，囚犯“应该受到尊重以他们固有的尊严和作为人类的价值”。
  </p>
  <p class="graf graf--p">
   在北京师范大学极端环境中研究人类行为的生理学教授表示，摄像机和人工智能“肯定会影响”囚犯的生活和精神状态。
  </p>
  <p class="graf graf--p">
   他还认为，当局必须考虑用机器监视人类可能会以意想不到的方式适得其反。
  </p>
  <p class="graf graf--p">
   例如，一些知情和聪明的囚犯可能会学习到如何欺骗人工智能的新方法，并利用机器的弱点来实现其目的。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这项技术不应取代人类，”他说。
  </p>
  <p class="graf graf--p">
   但参与该项目的天津大学电气工程助理教授表示，囚犯无法击败人工智能。即使有人设法说服监视机器他们没有任何不妥之处，他们也无法离开监狱。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该系统不仅与摄像机相连，而且与光纤等传感器相连，能够检测和定位多种类型的地面运动。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “你可能会欺骗相机。你可能会欺骗传感器。但你无法同时欺骗两者，“他说。
  </p>
  <p class="graf graf--p">
   越狱在中国比较少见，没有官方数字，但确实不时发生。
  </p>
  <p class="graf graf--p">
   去年，两名囚犯逃离了辽宁省的一所监狱。根据“北京青年报”的报道，他们从警卫的储物柜里偷走了制服和出入卡，然后就走了出去。在涉及1200多名警察的追捕后被抓获。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    中国并不是唯一一个拥有人工智能监狱的国家。在英国，利物浦附近的 Altcourse 监狱在2016年就安装了人工智能
   </strong>
   ，希望它可以阻止违禁品走私到监狱。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    新加坡政府进行了一系列试验，看看技术能否帮助警卫更有效地运作，甚至还讨论了建立无卫监狱的可能性
   </strong>
   。在这里看到视频：
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="393" src="//www.youtube.com/embed/9ph_pTYN9uo" width="700">
   </iframe>
  </p>
  <p class="graf graf--p">
   正在樟宜监狱其中一间牢房试行的技术据称能减轻狱警这方面的工作量。这项原型影片分析技术名为人类行为检测系统（Human Behaviour Detection System），又称「头像」（Avatar），系统可以侦测到目标具有侵略性的行为。演算法会捕捉囚犯之间高强度、不规则的动作，以及各种互动点。分析系统如监测到具有侵略性的动作，便会向狱警传送警报。
  </p>
  <p class="graf graf--p">
   新加坡近年来正极力发展所谓的智能城市，不断推行「无人经济」。例如在2016年8月，政府推出全国首间「无人咖啡店」，顾客只需使用售卖机便可购买到轻食、甜品、饮品甚至熟食。当地部分超级市场亦逐一引入自助收银系统；很多餐厅已配备平板电脑让食客自行点餐；无人驾驶的士更有望在今年在新加坡全面推广。
  </p>
  <p class="graf graf--p">
   在这里看到更多介绍，对全球所谓的智能/监视城市的综合报告《
   <a href="https://www.iyouport.org/%e6%99%ba%e8%83%bd%e5%9f%8e%e5%b8%82%e7%a9%b6%e7%ab%9f%e6%98%af%e4%b8%aa%e4%bb%80%e4%b9%88%e9%ac%bc%ef%bc%9f%ef%bc%81/" rel="noopener noreferrer" target="_blank">
    “智能城市”究竟是个什么鬼？！
   </a>
   》
  </p>
  <p class="graf graf--p">
   该消息真正主题并不是关于监狱，而是，您应该从中了解到中国已经拥有来这样的技术、以及最重要的是，这种技术还会被如何使用。
  </p>
