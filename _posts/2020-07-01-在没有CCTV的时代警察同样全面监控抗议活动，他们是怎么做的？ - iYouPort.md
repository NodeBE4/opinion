  <p class="graf graf--p">
   仅仅因为警察没有佩戴
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ba%ab%e4%bd%93%e7%9b%b8%e6%9c%ba%e9%80%82%e5%be%97%e5%85%b6%e5%8f%8d%e5%90%97%ef%bc%9f%e4%b8%ad%e5%9b%bd%e6%ad%a3%e5%9c%a8%e5%bc%80%e5%a7%8b%e6%b5%81%e8%a1%8c%e8%bf%99%e7%a7%8d%e4%b8%9c%e8%a5%bf/" href="https://www.iyouport.org/%e8%ba%ab%e4%bd%93%e7%9b%b8%e6%9c%ba%e9%80%82%e5%be%97%e5%85%b6%e5%8f%8d%e5%90%97%ef%bc%9f%e4%b8%ad%e5%9b%bd%e6%ad%a3%e5%9c%a8%e5%bc%80%e5%a7%8b%e6%b5%81%e8%a1%8c%e8%bf%99%e7%a7%8d%e4%b8%9c%e8%a5%bf/" rel="noopener noreferrer" target="_blank">
    身体摄像机
   </a>
   、没有安装
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%9c%a8%e5%95%86%e5%ba%97%e9%87%8c%ef%bc%8c%e7%a7%98%e5%af%86%e7%9b%91%e8%a7%86%e5%a6%82%e4%bd%95%e8%b7%9f%e8%b8%aa%e4%bd%a0%e7%9a%84%e6%af%8f%e4%b8%80%e6%ad%a5%e8%a1%8c%e5%8a%a8%ef%bc%9f/" href="https://www.iyouport.org/%e5%9c%a8%e5%95%86%e5%ba%97%e9%87%8c%ef%bc%8c%e7%a7%98%e5%af%86%e7%9b%91%e8%a7%86%e5%a6%82%e4%bd%95%e8%b7%9f%e8%b8%aa%e4%bd%a0%e7%9a%84%e6%af%8f%e4%b8%80%e6%ad%a5%e8%a1%8c%e5%8a%a8%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    LinkNYC 亭
   </a>
   、或没使用
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ba%94%e4%bb%98%e6%94%bf%e5%ba%9c%e4%bd%bf%e7%94%a8%e9%bb%84%e8%b2%82%e9%b1%bc%e5%af%b9%e5%85%ac%e6%b0%91%e8%bf%9b%e8%a1%8c%e7%9a%84%e5%a4%a7%e8%a7%84%e6%a8%a1%e9%97%b4%e8%b0%8d/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ba%94%e4%bb%98%e6%94%bf%e5%ba%9c%e4%bd%bf%e7%94%a8%e9%bb%84%e8%b2%82%e9%b1%bc%e5%af%b9%e5%85%ac%e6%b0%91%e8%bf%9b%e8%a1%8c%e7%9a%84%e5%a4%a7%e8%a7%84%e6%a8%a1%e9%97%b4%e8%b0%8d/" rel="noopener noreferrer" target="_blank">
    黄貂鱼
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    并不意味着
   </strong>
   他们在整个上个世纪都没有监控公民的录像。
  </p>
  <p class="graf graf--p">
   在过去的一年中，纽约市市政档案馆一直在仔细地数字化 NYPD 监视部门在1960年至1980年代之间创建的长达140小时监视录像的胶片，从而创造了丰富的视频档案记录，供大家仔细研究。
  </p>
  <p class="graf graf--p">
   您可以
   <a class="markup--anchor markup--p-anchor" data-href="http://nycma.lunaimaging.com/luna/servlet/NYCMA~3~3" href="http://nycma.lunaimaging.com/luna/servlet/NYCMA~3~3" rel="noopener noreferrer" target="_blank">
    在这里
   </a>
   找到档案，也可以
   <a class="markup--anchor markup--p-anchor" data-href="http://nycma.lunaimaging.com/luna/servlet/browseByCategory" href="http://nycma.lunaimaging.com/luna/servlet/browseByCategory" rel="noopener noreferrer" target="_blank">
    在这里
   </a>
   按类别浏览。这是大量的镜头，但是
   <strong class="markup--strong markup--p-strong">
    有很多令人着迷的近期历史
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️ 其中一些镜头是由便衣警官秘密拍摄的 —— 和斯塔西一样，而其他镜头则是通过警车旁的胶片式摄影机公开拍摄的。
  </p>
  <p class="graf graf--p">
   ⚠️ 而且，许多录像片段都集中在纽约警察局所监视的抗议行动和游行上。其中包括1970年世界地球日游行的录像、伊斯兰民族集会、CORE 和 NAACP 抗议种族隔离的运动、人权组织 Young Lords 的抗议活动、同性恋权利倡导者的早期抗议、肯特州枪击事件后的大规模反战游行、1970年5月的示威游行等等。
  </p>
  <p class="graf graf--p">
   录像带作为具有里程碑意义的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nyclu.org/en/cases/handschu-v-special-services-division-challenging-nypd-surveillance-practices-targeting" href="https://www.nyclu.org/en/cases/handschu-v-special-services-division-challenging-nypd-surveillance-practices-targeting" rel="noopener noreferrer" target="_blank">
    Handschu 协议
   </a>
   的一部分发行：Barbara Handschu 和其他15个政治或意识形态协会和组织的其他成员于1971年起诉纽约警察局，理由是监视他们的聚会和活动侵犯了宪法保护的权利。
  </p>
  <p class="graf graf--p">
   此案于1985年解决，并产生了一项同意令，禁止NYPD间谍政治和宗教组织和团体，除非有 “特定证据” 将该团体与已经或即将实施的犯罪联系起来。
  </p>
  <p class="graf graf--p">
   除了新的监视指南（
   <strong class="markup--strong markup--p-strong">
    纽约警察局已经被指控
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://gothamist.com/news/nypd-ordered-to-turn-over-photos-intel-from-surveillance-of-black-lives-matter-protesters" href="https://gothamist.com/news/nypd-ordered-to-turn-over-photos-intel-from-surveillance-of-black-lives-matter-protesters" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     公然违反
    </strong>
   </a>
   ）之外，该协议还要求市政档案馆接公开有这些记录，以确定它们是否具有历史重要性。
  </p>
  <p class="graf graf--p">
   1971年提起诉讼的律师之一 Martin Stolar 说：“这场展示证明了为什么首先要提起诉讼。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “您可以观看所有视频，但我看不到其中有任何犯罪行为。
   <strong class="markup--strong markup--p-strong">
    警察们正在记录的是政治行为
   </strong>
   。警察局为什么要调查人民的政治活动？那根本不在他们的职责范围内，这就是我们提起诉讼的原因。
   <strong class="markup--strong markup--p-strong">
    他们之所以这样做，是为了确定参与政治运动的领导者和成员的身份。为了抓捕
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   纽约市记录和信息服务部
   <a class="markup--anchor markup--p-anchor" data-href="https://www.archives.nyc/blog/2019/11/1/nypd-surveillance-films" href="https://www.archives.nyc/blog/2019/11/1/nypd-surveillance-films" rel="noopener noreferrer" target="_blank">
    写道
   </a>
   ：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    纽约警察局针对被视为 “现状敌人” 的个人和反抗组织的监视
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     可以追溯到1900年代初
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    。在不同时期，重点是无政府主义者、劳工领袖、纳粹支持者、白人至上主义者、社会主义者和共产主义者。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    这部监视镜头可以追溯到1960年代和1970年代 BOSSI 小队的鼎盛时期，当时他们收集了关于
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     政治范围内所有个人和团体的情报
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    ，⚠️
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     尤其是民权组织、反战组织、和女权主义者。
    </em>
   </strong>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    这些监视录像记录了民权运动的顶峰，当时激发了 “黑人权力和自豪感”、“妇女平权运动”、“男女同性恋”、“囚犯权利” 等社运团体的反抗行动，以及对贫困、环境恶化和越南战争的抵抗运动。
   </em>
  </p>

  <p class="graf graf--p">
   民权律师 Gideon Oliver 表示：“观看这段录像使我想起，
   <strong class="markup--strong markup--p-strong">
    变化越多，它们保持不变的程度就越高
   </strong>
   。”（这是一句法语谚语。这里指的是
   <strong class="markup--strong markup--p-strong">
    当下的警察和政府间谍继续致力于监视打压公民权利行动，并且其侵入性和危害性已经呈指数级上升
   </strong>
   ）
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “尽管制定了 Handschu 准则，但纽约警察局
   <strong class="markup--strong markup--p-strong">
    仍继续进行监视、渗透和破坏社区和社会正义运动，其透明度极少，当然问责制也更少
   </strong>
   。
  </p>
  <p class="graf graf--p">
   不幸的是，构成所谓的 “犯罪活动” 的种类繁多而且门槛非常低，几乎任何事都能触发警察的监视：纽约警察局的立场是，正在计划游行的人、或在20人以上的城市公园集会，就可能涉嫌违反法律，于是参与者必须服从监视，以及在某些情况下的渗透和/或破坏。”
  </p>
  <p class="graf graf--p">
   您可以查看以下录像的一些示例，看看您认识的人甚至您自己，是否因行使和平集会的宪法权利而被纽约警察局的监视定位。
  </p>
  <p class="graf graf--p">
   首先，您会看到1969年8月2日在第33街和第7大街举行的一次反战集会：“到1969年，越南战争已成为引起广泛社会关注的焦点议题。在反战活动家中，有黑豹、同性恋活动家联盟、学生争取民主社会等著名组织的支持者。”
  </p>

  </p>

   下面是1970年5月11日的抗议活动，工会工人抗议当时的市长 John Lindsay：“安全帽动乱发生在200名工会建筑工人暴力破坏了肯特州大学枪击案团结集会的时候，该集会由市中心的大学生和高中生举行。这份监控录像描绘了随后几天举行的众多建筑工人游行之一，表明他们对尼克松总统的支持和对 Lindsay 市长的愤怒。”
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="618" src="//player.vimeo.com/video/370324665?title=0&amp;byline=0&amp;portrait=0&amp;color=8dc7dc" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   Stolar 指出，在此处包含的所有监视视频所描述的
   <strong class="markup--strong markup--p-strong">
    完全都是合法的活动
   </strong>
   ：“那么警方到底在录制什么并将其归档到警察档案中呢？这些监视视频本来就不应该存在 …… 但是很高兴他们现在公开了这些视频。这是警察部门或任何其他执法机构的丑闻。”
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    最后
   </strong>
  </h3>
  <p class="graf graf--p">
   人们总是在说 “技术鼓励了反乌托邦的地狱世界”，但是，这些资料的曝光显示，在没有高科技监视器材的年代里，警察和秘密政府间谍同样致力于压制和平抗议、阻碍公民行使宪法权利。
  </p>
  <p class="graf graf--p">
   监视如人类历史一般古老，它只是在不同的年代有着不同的形态、不同的操作方式。如果公民不能有效抵抗，并且持续抵抗（因为稍有松懈噩梦就极有可能卷土重来），它无疑将继续深化，直到我们再也无能为力。⚪️
  </p>
