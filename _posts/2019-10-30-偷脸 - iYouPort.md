  <p class="graf graf--p">
   当我们强调不要在互联网上发布照片时已经暂时忽略了现实中的偷拍，也就是这篇报道中显示的状况 —— 政府资助的大学研究通过偷拍训练算法。
  </p>
  <p class="graf graf--p">
   于是它提醒了一件事：即便没有安装面部识别技术的监视摄像头也具有协助面部识别技术的“功能”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    反对者辩论的着眼点应该放在“公共场合是否应该有合理的隐私期待”这个问题上，这是监视者狡辩的重点
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这一辩论将不仅涉及政府对民众的监视，也包括人与人之间的监视 —— 你可能很难阻止陌生人拍摄你并放到互联网上；最重要的如何抵制这种情况，也就是辩论方法。正如我们此前专门分析过的，详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%80%e4%b8%aa%e6%84%9a%e8%a0%a2%e5%b8%a6%e6%9d%a5%e7%9a%84%e5%90%af%e7%a4%ba/" href="https://www.iyouport.org/%e4%b8%80%e4%b8%aa%e6%84%9a%e8%a0%a2%e5%b8%a6%e6%9d%a5%e7%9a%84%e5%90%af%e7%a4%ba/" rel="noopener noreferrer" target="_blank">
    一个愚蠢带来的启示
   </a>
   》。
  </p>
  <p class="graf graf--p">
   科罗拉多大学科罗拉多斯普林斯分校的一位教授领导了一个项目，秘密偷脸长达六年 ——
   <a class="markup--anchor markup--p-anchor" data-href="https://megapixels.cc/datasets/uccs/" href="https://megapixels.cc/datasets/uccs/" rel="noopener noreferrer" target="_blank">
    拍摄了超过
   </a>
   1700多名学生、教职员工和其他人在公共场合散步的照片，用来训练面部识别技术。
  </p>
  <p class="graf graf--p">
   这些照片目前已经作为数据集在线发布，从2016年到今年4月的部分可供下载。
  </p>
  <p class="graf graf--p">
   虽然 Terrance Boult 教授和CU官员为该项目拼命辩护，但丹佛大学法学教授 Bernard Chao 质疑这是否是技术跨越道德底线的又一个例子。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这是我们看到隐私入侵的另一个领域，”Bernard Chao 说道，他在DU教授法律与技术的交叉，并且之前在硅谷执业近20年。
  </p>
  <p class="graf graf--p">
   科罗拉多斯普林斯独立报
   <a class="markup--anchor markup--p-anchor" data-href="https://www.csindy.com/coloradosprings/uccs-secretly-photographed-students-to-advance-facial-recognition-technology/Content?oid=19664437" href="https://www.csindy.com/coloradosprings/uccs-secretly-photographed-students-to-advance-facial-recognition-technology/Content?oid=19664437" rel="noopener noreferrer" target="_blank">
    首次报道
   </a>
   的这件事，该项目于2012年开始，⚠️
   <strong class="markup--strong markup--p-strong">
    资金来自美国各种情报和军事机构，包括海军研究办公室、特种作战司令部和国家情报局局长办公室
   </strong>
   。
  </p>
  <p class="graf graf--p">
   目前尚不清楚该项目从政府机构获得了多少资金。
  </p>
  <p class="graf graf--p">
   Boult 的研究最初旨在分析面部识别算法，以确定它们是否适合美国海军使用。但事实证明，这项技术并不像海军想要的那样高效。
  </p>
  <p class="graf graf--p">
   Boult 说：“如果你想要将两张护照照片放在一起比较，那并不难，但如果你想要识别100米开外的人，就没那么简单了。”
  </p>
  <p class="graf graf--p">
   Boult 和他的团队做了更多高级研究，试图改进面部识别技术。
  </p>
  <p class="graf graf--p">
   Boult说：“这项研究正在努力使面部识别变得更好，特别是在远距离识别或监视应用中。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们希望收集一些人们在公共场合自然行为的数据集，因为这是最终尝试使用面部识别的方式。”
  </p>
  <p class="graf graf--p">
   面部识别技术正在被越来越多地使用，包括让 Facebook 在图片中标记人物，让政府机构检查护照或签证等等。
  </p>
  <p class="graf graf--p">
   为了进行这项研究，Boult 在离科罗拉多斯普林斯校区西草坪约150米的办公室窗户内设置了一个远程监控摄像机，他认为这是一个公共区域，路人不会有合理的隐私期望。
  </p>
  <p class="graf graf--p">
   在2012年和2013年春季学期的某些日子，这台监视相机一直在偷偷拍摄西草坪地区散步的人们。
  </p>
  <p class="graf graf--p">
   相机拍到的人大多低头看着手机、有些模糊不清，要么已经走出了监视区。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    总共拍摄了超过16,000张照片，产生了1,732种独特身份
   </strong>
   。Boult 说，他等了五年才公开发布该数据集“是为了保护隐私”。
  </p>
  <p class="graf graf--p">
   该校的发言人 Jared Verner 说，他们“致力于学术自由和教师学习和研究各种主题的能力，同时也认真对待学生的隐私”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “研究方案由 UCCS 机构审查委员会进行分析，该委员会确保在研究中保护人类受试者的权利和福利，”Verner 在一份声明中写道。
  </p>
  <p class="graf graf--p">
   DU 的 Chao 说，即便该研究得到了大学的机构审查委员会的批准，CU Colorado Springs 确定对个人受到的伤害没有实质性的担忧，但该项目仍然是“令人惊讶”的。
  </p>
  <p class="graf graf--p">
   Boult 说，照片数据集于4月15日在互联网上发布，但不是因为隐私问题。因为金融时报发布了一篇文章，题为“谁在用你的脸？关于面部识别的丑陋真相”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他们在文章中提供的信息比我们预期的要多”， Boult 说。
  </p>
  <p class="graf graf--p">
   发布的信息包括拍摄照片的日期和时间，Boult 表示这挫败了试图随机化数据集中的照片的预期目的。他说他正在考虑公开发布另一个版本的数据来解决这个问题。
  </p>
  <p class="graf graf--p">
   如果学生反对在不知情的情况下参与研究，他说他会试图“弥补”。
  </p>
  <p class="graf graf--p">
   Boult 说：“如果有人想来我的实验室浏览成千上万的照片并说’那个人就是我’，我们很乐意将它们从数据集中删除。”
  </p>
  <p class="graf graf--p">
   但他依旧辩称，学生的面孔正在被用来“带来更大的好处”，他说，他“平衡了”学生的隐私和改善面部识别系统的需求。
  </p>
  <p class="graf graf--p">
   Chao 反驳说，
   <strong class="markup--strong markup--p-strong">
    Boult 的推理假设政府或联邦机构正在做的事是社会希望他们做的事，而事实并非如此。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他可能正在帮助他们做一些不正确的事，”Chao 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我认为正确的回应需要是，’也许我们不应该这样做’”。⚪️
  </p>
