   <h1>
    诗：Song, by Christina Georgina Rossetti
   </h1>
   <ul class="post-entry-meta">
    <li>
     By:
     <a href="http://www.ideobook.com/about-ideobook/" title="查看 爱德布克 的作者主页">
      爱德布克
     </a>
     . 2007-5-14.
     <a href="http://www.ideobook.com/372/post-views-count/" title="统计说明">
      9,235
     </a>
    </li>
   </ul>
  </header>
  <div class="post-entry-content">
   <blockquote>
    <p>
     <strong>
      Song
     </strong>
    </p>
    <p>
     <a href="https://amzn.to/2OyF2GJ">
      <img alt="" src="https://images-na.ssl-images-amazon.com/images/I/51m3Altnx7L._SX324_BO1,204,203,200_.jpg" style="float:right;margin:5px 0px 35px 45px"/>
     </a>
     by
     <strong>
      <a href="https://amzn.to/2OyF2GJ">
       Christina Georgina Rossetti
      </a>
     </strong>
     (1830-1894)
    </p>
    <p>
     When I am dead, my dearest,
     <br/>
     Sing no sad songs for me;
     <br/>
     Plant thou no roses at my head,
     <br/>
     Nor shady cypress tree:
     <br/>
     Be the green grass above me
     <br/>
     With showers and dewdrops wet;
     <br/>
     And if thou wilt, remember,
     <br/>
     And if thou wilt, forget.
    </p>
    <p>
     I shall not see the shadows,
     <br/>
     I shall not feel the rain;
     <br/>
     I shall not hear the nightingale
     <br/>
     Sing on, as if in pain:
     <br/>
     And dreaming through the twilight
     <br/>
     That doth not rise nor set,
     <br/>
     Haply I may remember,
     <br/>
     And haply may forget.
    </p>
   </blockquote>
   <p>
    可能很多人和我一样，最先是从罗大佑谱曲的徐志摩译作知道Rossetti这首诗的。《歌》是一首不错的歌，尽管罗大佑同志的嗓音实在不怎么样。（听说这首歌还有张艾嘉演唱的版本，可惜我没听到过。）近日又读到飞白的译本，论准确，似乎比徐译稍准一点，可就总体诗意和意境而言，比徐志摩的翻译可是差多了。信译不美，美译不信。
   </p>
   <blockquote>
    <p>
     歌
    </p>
    <p>
     我死了的时候，亲爱的，
     <br/>
     别为我唱悲伤的歌；
     <br/>
     我坟上不必安插蔷薇，
     <br/>
     也无须浓荫的柏树；
     <br/>
     让盖着我的青青的草
     <br/>
     零着雨，也沾着露珠；
     <br/>
     假如你愿意，请记着我，
     <br/>
     要是你甘心，忘了我。
    </p>
    <p>
     我再不见地面的青荫，
     <br/>
     觉不到雨露的甜蜜；
     <br/>
     再听不见夜莺的歌喉，
     <br/>
     在黑夜里倾吐悲啼，
     <br/>
     在悠久的昏暮中迷惘，
     <br/>
     阳光不升起，也不消翳；
     <br/>
     我也许，也许我记得你，
     <br/>
     我也许，我也许忘记。
     <br/>
     （徐志摩 译）
    </p>
   </blockquote>
   <blockquote>
    <p>
     歌
    </p>
    <p>
     在我死后，亲爱的，
     <br/>
     不要为我唱哀歌；
     <br/>
     不要在我头边种蔷薇，
     <br/>
     也不要栽翠柏。
     <br/>
     让青草把我覆盖，
     <br/>
     再洒上雨珠露滴；
     <br/>
     你愿记得就记得，
     <br/>
     你愿忘记就忘记。
    </p>
    <p>
     我不再看到荫影，
     <br/>
     我不再感到雨珠，
     <br/>
     我不再听到夜莺
     <br/>
     唱得如泣如诉。
     <br/>
     我将在薄暮中做梦——
     <br/>
     这薄暮不升也不降；
     <br/>
     也许我将会记得，
     <br/>
     也许我将会相忘。
     <br/>
     （飞白 译）
    </p>
   </blockquote>
