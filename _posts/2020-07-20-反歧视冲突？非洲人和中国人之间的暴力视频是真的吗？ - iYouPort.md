  <p class="graf graf--p">
   几个月前中国网络上出现了越来越多的视频。显示居住在中国的黑人受到歧视。这些消息自4月初开始流传，它们指称非洲社区是中国第二波Covid-19大流行的原因。
  </p>
  <p class="graf graf--p">
   此后，一种新的视频出现在社交媒体上：其中显示非洲人在报复中国人。
  </p>
  <p class="graf graf--p">
   但是，许多这些视频都是虚假的 —— 与Covid-19大流行完全无关。
  </p>
  <p class="graf graf--p">
   自4月初以来，由于谣传非洲社区将新冠状病毒病例带到中国，许多居住在中国广州的非洲人被禁止离开家或被迫驱逐出境。
  </p>
  <p class="graf graf--p">
   一些据称发生在广州的对非洲人表现出暴力和敌意的视频，或者相反，非洲人对歧视作出愤怒反应的视频，在诸如 WhatsApp 等社交媒体和消息服务上广泛流传。
  </p>
  <p class="graf graf--p">
   而这些视频中有很多都是假的。
  </p>
  <p class="graf graf--p">
   以下是 FRANCE24 揭穿的与此主题有关的三个热门视频。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     1、尼日利亚的中国商店被纵火了吗？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   自从4月19日首次上传以来，几段视频在 Facebook、WhatsApp 和 Twitter 上获得了数十万的观看次数。这些视频显示一栋建筑物正在燃烧，而人们在街上奔跑以远离浓烟。
  </p>
  <p class="graf graf--p">
   视频中一个人用英语喊着：“Save your life, get out!” 这些视频的标题声称这些是在尼日利亚被烧毁的中国商店。其中不少评论还声称，这些大火是为了报复尼日利亚人在中国遭受的歧视。
  </p>
  <p class="graf graf--p">
   事实证明，这场大火确实于4月15日在尼日利亚西南部奥约州首府伊巴丹的一个市场上发生。核查人员检查了视频中显示的建筑物和其他元素，以及出现在
   <a class="markup--anchor markup--p-anchor" data-href="http://www.vanguardngr.com/2020/04/fire-guts-popular-dugbe-market-in-ibadan/" href="http://www.vanguardngr.com/2020/04/fire-guts-popular-dugbe-market-in-ibadan/" rel="noopener noreferrer" target="_blank">
    尼日利亚媒体发布的
   </a>
   同一场大火的照片中的建筑物和其他元素。
  </p>
  <p class="graf graf--p">
   通过识别这些景观细节，可以使用
   <a class="markup--anchor markup--p-anchor" data-href="http://www.google.com/maps/@7.3886826,3.8810479,3a,60y,45h,90t/data=!3m6!1e1!3m4!1sR2thG6Q68z2DOw0Zwd7imw!2e0!7i13312!8i6656" href="https://www.google.com/maps/@7.3886826,3.8810479,3a,60y,45h,90t/data=!3m6!1e1!3m4!1sR2thG6Q68z2DOw0Zwd7imw!2e0!7i13312!8i6656" rel="noopener noreferrer" target="_blank">
    Google街景视图
   </a>
   来确定火灾的确切位置：它发生在 Iya Olobe Oketedo 街的 Dugbe 市场上。
  </p>

  <p class="graf graf--p">
   在找到火源后，调查人员联系了 Omoloye Mattheux Bamidele，他是一名店主，销售手机配件，他的店也遭受了火灾。
  </p>
  <p class="graf graf--p">
   他说：“发生火灾的15家商店全部归尼日利亚人所有。该地区没有中国人经营的商店。大火始于一家销售床垫的商店，这家商店距离我的店很近。大火是中午左右开始的，我们不知道是怎么造成的。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    结论
   </strong>
   ：2020年4月确实在尼日利亚发生了大火，但并不是什么报复性袭击，烧毁的也不是中国人拥有的商店。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     2、非洲人在中国被殴打？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   不少读者通过 WhatsApp 给 FRANCE24 的调查人员发送了一个视频，该视频在 Facebook 上获得了数以万计的观看次数。视频显示，一群亚洲男子在购物街上殴打一名黑人。
  </p>
  <p class="graf graf--p">
   传播该视频的大部分帖子都在4月中旬发布，并声称该视频显示了谣言引发的暴力，因为谣传中国黑人社区正在传播Covid-19。
  </p>
  <p class="graf graf--p">
   调查人员使用 InVid 工具通过反向图像搜索来查找此视频。InVid会找到目标素材是否曾经出现过以及何时出现过。它返回了一篇刊登在尼日利亚网站上的文章中的视频，而日期是
   <strong class="markup--strong markup--p-strong">
    2015年6月
   </strong>
   。
  </p>
  <p class="graf graf--p">
   根据该网站的文章，事件发生在香港，被殴打的受害者是来自伊博部落的商人（这是居住在尼日利亚的一个民族）。
  </p>
  <p class="graf graf--p">
   AFP Fact Check 的事实检查小组证实，视频中的那个人在说伊博语。
  </p>
  <p class="graf graf--p">
   但是，当使用中国的百度地图来定位事发地点的购物街时显示，事件并非在香港发生。实际上发生在中国广东省佛山市。
  </p>

   <strong class="markup--strong markup--p-strong">
    结论
   </strong>
   ：调查人员无法确定这场打斗背后的故事。但是它肯定是在中国发生，只是，至少在Covid-19大流行之前的五年。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     3、一名黑人在当街与亚洲裸男打斗？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   这个视频在 Twitter 上被观看了数千次，并在 WhatsApp 上被广泛传播，其中显示，两名亚洲男子、其中一名全裸，在街中央与一名黑人打斗。
  </p>
  <p class="graf graf--p">
   依旧是通过反向图像搜索来查找该视频，并迅速确定它是旧视频 —— 从
   <strong class="markup--strong markup--p-strong">
    2019年7月
   </strong>
   开始在 Reddit 论坛上找到了关于它的讨论。
   <strong class="markup--strong markup--p-strong">
    这场打斗不是在中国发生的，而是在美国费城
   </strong>
   。（
   <a class="markup--anchor markup--p-anchor" data-href="http://www.google.com/maps/place/815%20Snyder%20Ave,%20Philadelphia,%20PA%2019148,%20%C3%89tats-Unis/@39.9231812,-75.1604508,3a,75y,37.47h,87.43t/data=!3m6!1e1!3m4!1sYz3nUzVS1v6OMcJJna2YpQ!2e0!7i16384!8i8192!4m5!3m4!1s0x89c6c60153514313:0x591e00e7c332177!8m2!3d39.9232904!4d-75.160341" href="https://www.google.com/maps/place/815%20Snyder%20Ave,%20Philadelphia,%20PA%2019148,%20%C3%89tats-Unis/@39.9231812,-75.1604508,3a,75y,37.47h,87.43t/data=!3m6!1e1!3m4!1sYz3nUzVS1v6OMcJJna2YpQ!2e0!7i16384!8i8192!4m5!3m4!1s0x89c6c60153514313:0x591e00e7c332177!8m2!3d39.9232904!4d-75.160341" rel="noopener noreferrer" target="_blank">
    在Google街景图上的确切位置
   </a>
   ）
  </p>
  <p class="graf graf--p">
   FRANCE24 联系了拍摄视频的人（他也出现在镜头中）但是没有得到回复。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    结论
   </strong>
   ：目前还无法确定发生这种冲突的原因。但是，完全可以确认该视频已存在至少9个月了，因此与当前的Covid-19大流行完全无关。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     注

  <p class="graf graf--p">
   这些事实核查都非常简单，是最常见的谣传而非精心制作的高技术含量深度造假。但正是这类简易粗陋的假消息，能欺骗如此多的人。
  </p>
