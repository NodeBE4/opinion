  <p class="graf graf--p">
   我知道那周我们在商店买了核桃，我想在燕麦片里加些核桃。我喊妻子，问她把核桃放哪了。她正在浴室里洗脸，水龙头开着，而且一定听不到我的声音 —— 她没有回答。
  </p>
  <p class="graf graf--p">
   我在没有她帮助的情况下找到了那袋坚果，然后放了一点在我的碗里。我的手机正在桌子上充电。很无聊，我拿起手机来检查应用程序是否可以无线获取我一个月前开始佩戴的健身手环的数据。
  </p>
  <p class="graf graf--p">
   我看到上面显示我在前一天晚上睡了将近八个小时，但是只有两个小时的“深度睡眠”。我发现我已经达到13,000步/天之目标的30％。
  </p>
  <p class="graf graf--p">
   然后，我注意到在一个小窗口中保留了一条消息，这个位置用于保留其他健康提示。上面写着“核桃”。
   <strong class="markup--strong markup--p-strong">
    它告诉我多吃核桃
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这可能是巧合，是偶然。尽管如此，它还是让我低头看了一下腕带，然后看了一眼手机，这是一种全新的型号，具有许多未知的未经测试的功能。
  </p>
  <p class="graf graf--p">
   我的手机是否已通过麦克风窃听了我说的话、然后以某种方式将其中继到腕带，然后向应用程序发出信号？
  </p>
  <p class="graf graf--p">
   这些设备背着我窃窃私语（我知道它们 “配对” 时会是什么样），但是突然之间我对它们之间的关系感到毛骨悚然。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    它们还和谁窃窃私语？还有什么？它们之间的谈话发生了什么？它们是临时存档的、及时清理的、还是永久合并到名称太过令人迷惑的“幽灵”实体中？
   </strong>
  </p>
  <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote">
   <p>
    “我认为它正在扫描我们，” Dalton 说，有件事告诉我他是对的。
   </p>
  </blockquote>
  <p class="graf graf--p">
   那是2013年的冬天，这些 “核桃时刻” 一直在增加，每当我上网时都会发生些微刺激。
  </p>
  <p class="graf graf--p">
   去年夏天的一个晚上，我开车去好莱坞一家艺术馆认识一个朋友，这是我多年来第一次访问艺术馆。
  </p>
  <p class="graf graf--p">
   而就在第二天早上，在我的收件箱中，有好几封垃圾邮件催促我投资艺术品。这很容易弄清楚：我
   <strong class="markup--strong markup--p-strong">
    在 Google Maps 中输入了画廊的名称以导航
   </strong>
   。
  </p>
  <p class="graf graf--p">
   另一个简单的跟踪记录是自从我查阅了洛杉矶（地区）戒酒
   <strong class="markup--strong markup--p-strong">
    匿名互助组
   </strong>
   的在线日历以来一直受到的毒品和酒精康复中心的邀请。
  </p>
  <p class="graf graf--p">
   由于AA的会员身份应该是机密的，因此这些电子邮件使我感到恼火。他们自以为是的贴心令人厌恶。
  </p>
  <p class="graf graf--p">
   而且，这些让我感到不安的提示中有些很难解释。
  </p>
  <p class="graf graf--p">
   例如，在我的 Facebook 页面上，“我可能认识的人” 标题下出现的一位加利福尼亚音乐家，这个人是我在一次
   <strong class="markup--strong markup--p-strong">
    私人住宅里的匿名戒酒互助组
   </strong>
   上碰到过几次的人。
  </p>
  <p class="graf graf--p">
   按照匿名互助的惯例，他从未告诉过我他的姓氏，也从未询问过我的名字。据我所知，我们只有一个共同的朋友，一位特别孤独的老年小说家，他完全避免使用计算机。
  </p>
  <p class="graf graf--p">
   我在一个在线技术论坛上进行了一些研究，得知：这位音乐家将我的电话号码输入了智能手机的通讯录中（编制电话清单以备不时之需是AA的仪式）很可能触发了将其全名和照片发送到我的 Facebook 页面上的机制（Facebook 在读取你的手机通讯录）。
  </p>

  <p class="graf graf--p">
   然后是那种特殊的心理入侵。
  </p>
  <p class="graf graf--p">
   一个晚上，大约在我的手机建议我多吃核桃之前的一年，我正在研究现代间谍软件话题为一本我想写的书，当时我偶然看到了一部令人毛骨悚然的 YouTube 视频。
  </p>
  <p class="graf graf--p">
   视频中包括来自中东一家旅馆的监控录像，据称代表以色列行事的秘密特工暗杀了哈马斯的一名高级官员。
  </p>
  <p class="graf graf--p">
   我看着特工们如何跟踪他们的目标，目标显然是在他的房间外被谋杀的，然后他们又重新出现在走廊上，就像没事一样按下电梯。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    由于其中一位特工是女性，因此我在浏览器的搜索栏中输入了这些词：“摩萨德勾引技术”
   </strong>
   。

    <strong class="markup--strong markup--li-strong">
     间谍机构早已开始使用电视机植入恶意软件以监视目标人了
    </strong>
    ，这种电视无法实际关机，你以为它关上了，但它一直在监听监视《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e9%9b%b6%e6%97%a5%e6%bc%8f%e6%b4%9e%e5%92%8c-cia-%e9%bb%91%e5%ae%a2%e5%b7%a5%e5%85%b7/" href="https://www.iyouport.org/%e9%9b%b6%e6%97%a5%e6%bc%8f%e6%b4%9e%e5%92%8c-cia-%e9%bb%91%e5%ae%a2%e5%b7%a5%e5%85%b7/" rel="noopener noreferrer" target="_blank">
     零日漏洞和 CIA 黑客工具
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   在展览内部，一个穿着整洁的推销员站在一个女人旁边，后者盯着他，表情像偶像崇拜一样。
  </p>
  <p class="graf graf--p">
   他向我们展示了一系列枪支弹药，旨在用数十个细小的多面刀片切碎人类目标。其中一个弹壳的两端都精确地束着一卷捆扎好的金属丝，这样一来，当发射时它会解开并伸展，将目标切成碎片。
  </p>
  <p class="graf graf--p">
   这名推销员说：重要的是在自动取款机停止工作、杂货店被抢劫一空后，要活下去…… （危机意识和灾难心理有多么的流行。人们为什么会深深感到不安全？）
  </p>
  <p class="graf graf--p">
   这两个人并没有将商品推销给我们，只是推销了他们的外表。当他们得知我们来自蒙大拿州时，他们问我们是否曾看到过fema营地，据推测有成千上万的外国部队驻扎在这里。
  </p>
  <p class="graf graf--p">
   推销员担心这些部队会 “劫持我们的妇女”，因此他推荐了一个播客：The Common Sense Show，由一个叫戴夫·霍奇斯（Dave Hodges）的人主持，
   <strong class="markup--strong markup--p-strong">
    警告我们为即将来临的攻城做好准备。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该名男子说话时眼神不停扫向周边，仿佛在警惕潜伏着的特工
   </strong>
   。后来，我得知他的担心并非完全没有根据。
  </p>
  <p class="graf graf--p">
   去年1月，美国公民自由联盟（ACLU）发了一封电子邮件，描述了一项联邦计划，以扫描枪械展示场外停放的车辆的车牌。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    枪展主要不是关于武器的，而是关于自治
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在这种情况下，枪展被示为有权反对一个傲慢的、侵入性的新秩序，就如我前一天晚上亲眼目睹的镇压和控制手段。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    对它激起的无能为力的感觉似乎没有理性的回应。你只能选择是忽略还是发疯，至少在某种程度上
   </strong>
   。
  </p>
  <p class="graf graf--p">
   出售的突击步枪和榴弹发射器是一部想象中的抵抗戏剧中的道具，其中个人奋起为自己的安全而抗争。
  </p>
  <p class="graf graf--p">
   具有讽刺意味的是，这可能是为这些人的防御做准备的唯一方式，即这些人知道如何通过策划反冲行动和囤积弹药来解决安全问题，而技术霸主也恰恰以此来证明自己的间谍行为是正当的。
  </p>
  <p class="graf graf--p">
   在这场史诗般的冲突中，可能的战斗人员之间的联合可能比他们所欣赏的那种更为紧密。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11241 jetpack-lazy-image" data-lazy-sizes="(max-width: 960px) 100vw, 960px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-4.jpg?resize=960%2C640&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-4.jpg?w=960&amp;ssl=1 960w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-4.jpg?resize=300%2C200&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-4.jpg?resize=768%2C512&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-4.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="640" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-4.jpg?resize=960%2C640&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="960"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11241" data-recalc-dims="1" height="640" sizes="(max-width: 960px) 100vw, 960px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-4.jpg?resize=960%2C640&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-4.jpg?w=960&amp;ssl=1 960w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-4.jpg?resize=300%2C200&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-4.jpg?resize=768%2C512&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-4.jpg?resize=272%2C182&amp;ssl=1 272w" width="960"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   PA系统上的声音宣布该展会将在15分钟内结束，导致供应商大幅降低价格，
   <strong class="markup--strong markup--p-strong">
    客户用迷彩连身裤、太阳能收音机和各种世界末日露营装备塞满了行李
   </strong>
   。
  </p>
  <p class="graf graf--p">
   我们在汽车中，沿着I-15公路向北行驶，我戴上了新的防弹射击眼镜，而 Dalton 则将手机插入立体声音响，并播放了《The Common Sense Show》的一集。它暗淡的地下室般的声音显示它显然是
   <strong class="markup--strong markup--p-strong">
    在防辐射棚中
   </strong>
   播放的。
  </p>
  <p class="graf graf--p">
   像我这样仍在学习如何应对监视的神经病来说容易产生不安和忐忑，但对于 The Common Sense Show 来说，可能会感到奇怪的镇定。
  </p>
  <p class="graf graf--p">
   播客带来了适合长途驾驶的状态。几个月后，当我读到一个雄心勃勃的意大利外科医生打算完善移植人头的手术时，在我眼中重新出现的器官超人的景象才终于取代了整体数据中心的回忆。
  </p>
  <p class="graf graf--p">
   我们在黄昏时分跨入爱达荷州南部，在熔岩温泉（Lava Hot Springs）旁，熔岩温泉是一个偏僻的山区小镇，以治疗性热水池而闻名。
  </p>
  <p class="graf graf--p">
   我想把脑子里的黑色直升机洗掉。在数据中心附近偷偷摸摸地与那些抽搐的枪支表演者混在一起，削弱了我的心理免疫系统。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    偏执是一种地狱般的苦难，一旦被它抓住就很难将其扑灭，尤其是在每个星期都有政府间谍和商业间谍的最新消息闪现的时代里。
   </strong>
  </p>
  <p class="graf graf--p">
   AT&amp;T 和美国国家安全局串通出卖用户；FBI正在操作塞斯纳斯的飞行器配备了监视摄像机和手机扫描仪，遍布所有美国城市；谷歌有能力通过其搜索算法来决定下届总统大选的胜者……
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    一旦您对想要间谍您的一切的人了解得很少，那么日常经验就会开始失去其纯真度，任何细微的小事都会变成超级大事的引线。（这种感受中国读者应该最熟悉了，你能看到人们总是倾向于捕风捉影，因为人们只能感受到恐惧，而对幕后的庞大黑手一无所知）
   </strong>
  </p>
  <p class="graf graf--p">
   我与一个十几岁的年轻人进行了交谈，他在一年前辍学，对自己的前景感到沮丧。他告诉我，他没有找到人类能比机器人做得更好的工作。
  </p>
  <p class="graf graf--p">
   当我告诉他我的NSA游览时，他叹了口气，摇了摇头。他说，监视是没有意义的，完全是浪费。相反，
   <strong class="markup--strong markup--p-strong">
    权力可以诱导人们自愿承认自己的秘密
   </strong>
   。他设想了配备麦克风和耳机的庞大中心，人们可以在其中详尽地谈论自己的经历、思想和感受，以独白的形式完全传达给窃听者，比零星收集快多了。（这不就是社交媒体吗？）
  </p>
  <p class="graf graf--p">
   在泳池中，沉浸在形成神秘感的蒸汽云中，我想知道发现隐私已经过时的这一代人是否可能正在发生巨大变化，从而放弃了无法再保证其神圣性的内部心理领域。
  </p>
  <p class="graf graf--p">
   掩盖自己的内心曾经是人类社会生活的基本任务，然而这项任务如今已经变得更加艰巨，甚至可疑；为什么不能退出战斗？
  </p>
  <p class="graf graf--p">
   社交媒体引发的对自我披露的热潮也许来自对入侵者的漠视：认为免费提供的东西不会被盗。我的朋友已经退出了。
  </p>
  <p class="graf graf--p">
   不久前，我妻子出差去城里，我给她发短信说晚安。我写道：“睡个好觉，别让臭虫咬。” 而第二天早晨，当我在电子邮件列表上发现驱虫除臭剂的信息时，我感到不安。
  </p>
  <p class="graf graf--p">
   如果几年前有人告诉我这不是巧合，那我会对那个人感到怀疑。但现在，我对仍然信任那是巧合的人们表示怀疑。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    我们身边有太多的鬼魅 —— 它们的位置如此隐蔽，它们的方法如此巧妙，其动机如此难以理解 —— 以至于除非你被困扰其中否则不会醒来。
   </strong>
  </p>
  <p class="graf graf--p">
   这就是为什么偏执狂，即使是极端的形式，在我看来也是正常的认知方式，是一种具有令人印象深刻的先验记录的认知方式。
  </p>
  <p class="graf graf--p">
   我们曾经嘲笑你是偏执狂，非常抱歉。我们担心你疯了，但现在我们也疯了，这意味着我们已经准备好了倾听，我们谈谈吧。
  </p>
  <p class="graf graf--p">
   是时候了。早就是时候了。让我们彼此了解。悄无声息地，在黑暗中，留给我们的私人空间已经如此之小，现在几乎消失了。⚪️
  </p>
  <p class="graf graf--p">

    If You’re Not Paranoid, You’re Crazy
   </a>
  </p>
