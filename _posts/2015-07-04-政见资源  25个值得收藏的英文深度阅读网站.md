 <!--p class="post-lead">别以为全世界人都在手机上进行碎片化阅读，至少在英文世界，深度阅读并未消失。这里是政见团队准备的一份英文深度阅读网站名录和简介，包含了主要的社会政治类网站。</p-->
 <div class="post-body">
  <figure>
   <img src="http://cnpolitics.org/wp-content/uploads/2015/07/83.pic_.jpg" width="566"/>
  </figure>
  <p>
   在这个两千个汉字都考验阅读耐心的微信年代，国内的深度阅读氛围正在被短平快的创作和各种复制粘贴行为逆淘汰。但至少在英语世界，很多好网站并没有消失。深度阅读作为一种文化，不仅依赖于老牌杂志的更新换代，还在新兴独立媒体的涌现和新创作平台的尝试中得到了强化。
  </p>
  <p>
   以下是政见（
   <a href="http://cnpolitics.org">
    http://cnpolitics.org
   </a>
   ）准备的一份英文深度阅读网站名录和简介，其中包含了主要的社会政治类网站，不包括文学艺术色彩浓厚的长文站点。那些付费墙标准极高、需要订阅才可以看内容的媒体也没有被收入其中。由于相关媒体总数极大，以下列表难免有所遗漏，欢迎大家在评论中补充。虽然我们推荐的媒体中左派较多，但本文的目的在于推广深度阅读，而不是传达特有的政治观念。
  </p>
  <p>
   现在就把这些站点放入收藏夹，并把这份名单分享到朋友圈吧。如果想要转载，请一定记得联系我们获得授权。
  </p>
  <p>
   一、经典杂志
  </p>
  <p>
   New Yorker：
   <a href="http://www.newyorker.com/">
    http://www.newyorker.com/
   </a>
   <br>
    作为文化指南的《纽约客》就不多介绍了吧，不过付费墙出现后，读者每个月只能免费阅读6篇文章了（如果你不做手脚的话）。
   </br>
  </p>
  <p>
   The Atlantic：
   <a href="http://www.theatlantic.com/">
    http://www.theatlantic.com/
   </a>
   <br>
    快160年历史的《大西洋月刊》，似乎也不用多花笔墨。
   </br>
  </p>
  <p>
   The New Republic：
   <a href="http://www.newrepublic.com/">
    http://www.newrepublic.com/
   </a>
   <br>
    杜威、凯恩斯、奥威尔等人撰稿过的新共和，早期属于进步左翼，80年代后转向保守，但总体依然较为自由派。值得一提的八卦是，去年底《新共和》杂志改版，导致编辑和作者用总辞抗议，《
    <a href="http://www.newyorker.com/news/news-desk/inside-collapse-new-republic ">
     纽约客
    </a>
    》和《
    <a href="http://www.theatlantic.com/politics/archive/2014/12/how-the-new-republic-lost-its-place-on-the-american-left/383544/ ">
     大西洋月刊
    </a>
    》都报道了。
   </br>
  </p>
  <p>
   Harper’s：
   <a href="http://harpers.org/">
    http://harpers.org/
   </a>
   <br>
    文化期刊《哈泼斯》是美国第二古老的月刊，仅次于《科学美国人》。政见团队（http://cnpolitics.org）顺便为你推荐个旧文吧，作家
    <a href="http://harpers.org/archive/2007/02/the-ecstasy-of-influence/">
     Jonathan Lethem论抄袭的神文
    </a>
    。
   </br>
  </p>
  <p>
   New York Review of Books：
   <a href="http://www.nybooks.com/">
    http://www.nybooks.com/
   </a>
   <br>
    纽约书评诞生于1963年纽约的一场罢工，之后成为自由派大本营。比如最近一期，《21世纪资本论》作者
    <a href="http://www.nybooks.com/articles/archives/2015/jun/25/practical-vision-more-equal-society/ ">
     皮凯蒂评论经济学阿特金森的新书
    </a>
   </br>
  </p>
  <p>
   The Nation：
   <a href="http://www.thenation.com/">
    http://www.thenation.com/
   </a>
   <br>
    美国最古老的左派杂志，今年刚庆祝150岁生日。
    <a href="http://www.thenation.com/article/200785/150th-anniversary-issue ">
     150周年特刊下载
    </a>
   </br>
  </p>
  <p>
   Dissent：
   <a href="http://www.dissentmagazine.org/">
    http://www.dissentmagazine.org/
   </a>
   <br/>
   《异议》成长于麦卡锡时期，由批评家欧文豪与社会学家刘易斯科塞创立。杂志经常会有关于中国和香港的文章出现。如果你所在的机构订阅了Muse数据库，那就读一下
   <a href="https://muse.jhu.edu/journals/dissent/toc/dss.61.1.html">
    60周年的纪念特刊
   </a>
   吧，它完整回顾了杂志的历史 ，看不到的可以读
   <a href="http://www.newyorker.com/news/daily-comment/a-modest-utopia-sixty-years-of-dissent ">
    纽约客的纪念短文
   </a>
   。
  </p>
  <p>
   New Left Review：
   <a href="http://newleftreview.org/">
    http://newleftreview.org/
   </a>
   <br/>
   来自伦敦的左派双月刊。缺点是，这群马克思主义者们设计的网站界面实在不太美观……他们曾经
   <a href="http://newleftreview.org/II/92/joshua-wong-scholarism-on-the-march ">
    访问过黄之锋
   </a>
   。
  </p>
  <p>
   Boston Review：
   <a href="http://bostonreview.net/">
    http://bostonreview.net/
   </a>
   <br/>
   双月刊《波士顿评论》也已经有四十年的历史。政见团队（http://cnpolitics.org）为你推荐其中一篇文章：
   <a href="http://bostonreview.net/arts-culture/hal-stucker-sixties ">
    我们如何高估了60年代青年抗争的颠覆性
   </a>
   。
  </p>
  <p>
   Harvard Political Review：
   <a href="http://harvardpolitics.com/">
    http://harvardpolitics.com/
   </a>
   <br/>
   哈佛本科生运营的杂志，立场中立，接受哈佛所有学生的投稿。推荐一篇：
   <a href="http://harvardpolitics.com/books-arts/politics-race-rap/">
    论嘻哈乐中的种族政治
   </a>
   。
  </p>
  <p>
   二、新兴杂志
  </p>
  <p>
   Slate：
   <a href="http://www.slate.com/">
    http://www.slate.com/
   </a>
   <br/>
   1996年由《新共和》前主编Kinsley成立的在线杂志，关于广义的政治和文化议题。一个坏消息：由于网站盈利来自广告，目前网站对美国以外的读者实行了付费墙，国际读者一个月只能免费读五篇文章。政见团队（http://cnpolitics.org）曾经介绍的《
   <a href="http://cnpolitics.org/2014/11/nazis/">
    从保龄球馆飘出的纳粹幽灵
   </a>
   》里的主要参考文献，
   <a href="http://www.slate.com/articles/business/the_dismal_science/2013/06/the_rise_of_nazi_germany_did_the_prevalence_of_hunting_and_singing_clubs.single.html">
    Slate也第一时间介绍过
   </a>
   。
  </p>
  <p>
   Aeon：
   <a href="http://aeon.co/magazine/">
    http://aeon.co/magazine/
   </a>
   <br/>
   大名鼎鼎的文化思想杂志“万古”。即使你没听过它，也可能通过很多渠道间接阅读过了，比如译言有专门的翻译频道，微信号“利维坦”也刊登过好多次万古杂志的文章。例子：来个欢乐（其实很严肃）的话题，
   <a href="http://aeon.co/magazine/psychology/does-too-much-porn-numb-sexual-pleasure/">
    色情影片的社会后果
   </a>
   。
  </p>
  <p>
   Baffler：
   <a href="http://www.thebaffler.com/ ">
    http://www.thebaffler.com/
   </a>
   <br/>
   1988年创办的文化批评杂志，目前文章几乎都可以通过网站阅读。其团队的自我介绍说，我们是“一群松散的，无组织的文化评论人，知识民工，诗人，插画家和柜子里的乌托邦主义者（closet utopians）。”他们架势高冷，深感美国的信息帝国经济正受到金融欺诈，环境威胁和思想腐化的威胁…如果你还不知道这群疯子在说什么，就直接点击吧。例子：女性主义科技批判神文《
   <a href="http://www.thebaffler.com/salvos/dads-tech ">
    科技的父亲们
   </a>
   》。
  </p>
  <p>
   Jacobin：
   <a href="https://www.jacobinmag.com/">
    https://www.jacobinmag.com/
   </a>
   <br/>
   成立才五年的《雅各宾》，已经成为美国左翼的重要标杆杂志，它也立志要在年轻人中传承和更新老左派的传统。去年引发巨大争议的文章《
   <a href="https://www.jacobinmag.com/2014/12/foucault-interview/">
    我们能否批判福柯
   </a>
   》，认为福柯有新自由主义倾向。
  </p>
  <p>
   N+1：
   <a href="http://nplusonemag.com/">
    http://nplusonemag.com/
   </a>
   <br/>
   这个诡异的杂志名由来暂时不可考。N+1一年只出版三期，不过每一期内容都够看很久。杂志需要订阅，其他读者只能享用有限的阅读配额。例文：
   <a href="https://nplusonemag.com/issue-21/the-intellectual-situation/too-fast-too-furious/ ">
    加速的社会
   </a>
   。
  </p>
  <p>
   Discover Society：
   <a href="http://discoversociety.org/">
    http://discoversociety.org/
   </a>
   <br/>
   英国非营利组织主办，偏向政策与行动社会学的在线杂志，关注平权，环境，公众记忆等，已经出版了20期。撰稿人几乎全都是学术研究者。例子：
   <a href="http://discoversociety.org/2015/03/01/race-talk-from-biology-to-social-science-and-back-again/ ">
    种族研究的趋势
   </a>
   。
  </p>
  <p>
   Model View Culture：
   <a href="https://modelviewculture.com/">
    https://modelviewculture.com/
   </a>
   <br/>
   一个成立才一年多的科技批判杂志，平均三周更新一期，完全依赖读者捐款。角度刁钻，言辞犀利，以小见大。缺点是专业词汇过多，阅读难度较大，普通读者即使英文程度较高，也要做好不停谷歌的准备。例子：福柯即视感的《
   <a href="https://modelviewculture.com/pieces/being-seenbeing-watched-surveillance-technology-and-madness">
    监控，科技与疯狂
   </a>
   》。
  </p>
  <p>
   三、新兴平台
  </p>
  <p>
   Vox：
   <a href="http://www.vox.com/">
    http://www.vox.com/
   </a>
   <br/>
   融合了即时新闻和深度阅读的站点，一大特色是旗下的“地图”栏目，推出过很多又话题有趣又一目了然的地图，比如这篇：
   <a href="http://www.vox.com/2014/8/26/6063749/38-maps-that-explain-the-global-economy ">
    解释世界经济的38张图片
   </a>
   。
  </p>
  <p>
   Medium：
   <a href="https://medium.com/">
    https://medium.com/
   </a>
   <br/>
   个人创作自媒体网站，文章质量参差不齐，但编辑们会推荐优秀文章，也确实有很多牛人写作长文。网站的tag也可以帮助用户搜寻和关注特定主题和作者的更新。例子：本政见观察员非常欣赏的行动社会学家
   <a href="https://medium.com/@zeynep ">
    Zeynep Tufekci作者页面
   </a>
   。
  </p>
  <p>
   The Awl：
   <a href="http://www.theawl.com/">
    http://www.theawl.com/
   </a>
   <br/>
   2009年创立的深度阅读网站，话题偏流行文化，政治议题相对较少。例子：
   <a href="http://www.theawl.com/2014/11/the-new-identity-media-manifesto ">
    新“身份媒体”宣言
   </a>
   。
  </p>
  <p>
   JSTOR Daily：
   <a href="http://daily.jstor.org/">
    http://daily.jstor.org/
   </a>
   <br/>
   这是一个跟政见（http://cnpolitics.org）很像的网站，依托JSTOR数据库丰富的学术资源，JSTOR每日网站将新闻和已有的研究连接起来。例子：
   <a href="http://daily.jstor.org/uber-sales-door-door-vacuum-salesmen/">
    文献告诉你历史上的Uber模式
   </a>
   。
  </p>
  <p>
   The New Rambler：
   <a href="http://newramblerreview.com/">
    http://newramblerreview.com/
   </a>
   <br/>
   芝加哥大学赞助的深度书评网站，有很多学术大牛贡献内容。虽然由于定位问题，书评一般局限在各大学出版社出版的偏学术书籍上，但是很多评论都相当有现实意义。例子：
   <a href="http://newramblerreview.com/book-reviews/law/ethics-on-the-run ">
    西北大学法学教授Steven Lubet抨击威斯康星麦迪逊社会学助理教授Alice Goffman
   </a>
   。
  </p>
  <p>
   Timeline：
   <a href="https://timeline.com/">
    https://timeline.com/
   </a>
   <br/>
   这主要是一个今年新推出的App，致力于提供新闻的历史背景，短至几十年，长可达千年。呈现的方式是通过用户的下拉操作，不断出现一个个历史节点的介绍。这是将热点和历史结合起来的创新尝试。难能可贵的是，每篇文章后面还有参考文献和延伸阅读，方便读者挖掘进一步的信息。例子：
   <a href="https://timeline.com/stories/the-gay-rights-movements-long-historic-march-to-marriage-equality ">
    关于同性恋权利运动的历史
   </a>
   。
  </p>
  <p>
   Longform：
   <a href="http://longform.org/">
    http://longform.org/
   </a>
   <br/>
   深度阅读综合平台，集成了大部分上述站点的订阅，还包括很多新闻网站的评论版块内容。这个站点的特色在于你不仅可以关注某个网站，还可以关注你喜爱的长文作者。网站的iOS app也很不错，可以直接转存文章到Evernote等。
  </p>
  <p>
   Longreads：http://longreads.com/
   <a href="http://longreads.com/">
   </a>
   <br/>
   深度阅读综合平台，特色是编辑每周会评选五大长文，给出文章摘要字数等基本信息，读者点击后会被跳转到原文网站进行阅读。
  </p>
