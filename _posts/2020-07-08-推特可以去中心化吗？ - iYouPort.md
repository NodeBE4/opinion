  <p>
   <img alt="" class="aligncenter size-full wp-image-12498 jetpack-lazy-image" data-lazy-sizes="(max-width: 780px) 100vw, 780px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-28.jpeg?resize=780%2C520&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-28.jpeg?w=780&amp;ssl=1 780w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-28.jpeg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-28.jpeg?resize=768%2C512&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-28.jpeg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="520" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-28.jpeg?resize=780%2C520&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="780"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   Jack Dorsey 的计划雄心勃勃，但细节不足。
  </p>
  <p class="graf graf--p">
   去年底，Jack Dorsey 宣布了 Bluesky，该项目旨在为 Twitter 和社交媒体格局开发开放的和去中心化的标准。
  </p>
  <p class="graf graf--p">
   当前，社交媒体用户只能与同一平台内的其他成员进行交互。Dorsey 建议，他的新标准将使交流能够跨社交网络进行，类似于电子邮件的工作原理。
  </p>
  <p class="graf graf--p">
   不过，除此之外，Dorsey 并未确切说明这意味着什么。
  </p>
  <p class="graf graf--p">
   如果 Twitter 真的想去中心化，它可以利用现有的生态系统。去中心化社交媒体的一系列服务已经集成在
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@VirtualAdept/a-friendly-introduction-to-the-fediverse-5b4ef3f8ed0e" href="https://medium.com/@VirtualAdept/a-friendly-introduction-to-the-fediverse-5b4ef3f8ed0e" rel="noopener noreferrer" target="_blank">
    Fediverse
   </a>
   中，这是可互操作的社交网络的集合，这些社交网络使用开放的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.w3.org/TR/social-web-protocols/" href="https://www.w3.org/TR/social-web-protocols/" rel="noopener noreferrer" target="_blank">
    标准和协议
   </a>
   相互对话。
  </p>
  <p class="graf graf--p">
   但是 Dorsey 希望花费数年时间开发新标准，而不是与当前解决方案集成。有充分的理由对此表示怀疑。
  </p>
  <p class="graf graf--p">
   Twitter 当前提供了一个集中式平台，该平台可通过对其用户进行有针对性的监视来获利。相比之下，分散式社交网络旨在最大程度上减少数据收集，减少广告投放，并使社交网络的所有权和控制民主化。
  </p>
  <p class="graf graf--p">
   Mastodon 社交网络拥有超过200万的用户，可一窥其工作原理。它与 Twitter 非常相似，不同之处在于
   <a class="markup--anchor markup--p-anchor" data-href="https://www.msn.com/en-in/news/newsindia/why-are-indias-twitter-users-moving-to-mastodon/ar-BBWsDgi" href="https://www.msn.com/en-in/news/newsindia/why-are-indias-twitter-users-moving-to-mastodon/ar-BBWsDgi" rel="noopener noreferrer" target="_blank">
    用户
   </a>
   可以在服务器上托管自己的小型社交网络（称为“实例”），可以与其他 Mastodon 实例上的用户进行对话。
  </p>
  <p class="graf graf--p">
   创建实例的人将设置自己的行为准则。例如，一个人可以创建一个专门用于幼犬的 Mastodon 实例；另一个实例可能更通用，但禁止仇恨言论和暴力。用户可以在其他情况下与其他人交谈，但是如果他们违反了规则，那么版主可以阻止他们或将他们踢出去。
  </p>
  <p class="graf graf--p">
   Mastodon 不仅赋予社区对自己空间的控制权。它还提供了宝贵的隐私选项。实例可以设为私有，并且用户可以注册而无需透露真实姓名或将其帐户链接到个人身份信息。
  </p>
  <p class="graf graf--p">
   服务器管理员可以在网络上看到用户的通信，因此对管理员的信任对用户至关重要。也没有针对性或其他方式的广告。
  </p>
  <p class="graf graf--p">
   由于 Mastodon 是分散式的，因此没有中央控制的实例，并且可以与 Fediverse 上的其他社交网络和网站进行互操作：Mastodon 和 Pleroma 用于社交网络，PeerTube 用于视频共享，PixelFed 用于图像共享（类似于 Instagram）。
  </p>
  <p class="graf graf--p">
   因此，用户可以发布视频之类的内容，并在网站和服务之间进行交互，而不必分别登录每个服务。
  </p>
  <p class="graf graf--p">
   相比之下，Facebook 和 Twitter 之类的大型社交网络旨在将用户锁定在自己的公司范围内（因为用户是监视资本家的资产）。这些公司选择不允许其用户与其他网络的成员对话。
  </p>
  <p class="graf graf--p">
   相比之下，分散式服务旨在将社交媒体世界打造成社区拥有和控制的网络的分散格局。它们依靠开放协议（用于传输数据的一组规则和过程）来促进跨服务的交互。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.w3.org/TR/activitypub/" href="https://www.w3.org/TR/activitypub/" rel="noopener noreferrer" target="_blank">
    ActivityPub
   </a>
   是去中心化社交网络的一种流行协议，而诸如
   <a class="markup--anchor markup--p-anchor" data-href="https://matrix.org/faq/" href="https://matrix.org/faq/" rel="noopener noreferrer" target="_blank">
    Matrix
   </a>
   之类的其他协议则用于聊天室和聊天应用程序。
  </p>
  <p class="graf graf--p">
   这就是 Twitter 的公告变得有趣的地方。
  </p>
  <p class="graf graf--p">
   去中心化社交媒体提供了数千个互联网络，旨在提供无广告、尊重隐私的体验。如果用户发现网络正在滥用其数据，则可以转移到另一服务。
  </p>
  <p class="graf graf--p">
   而寡头公司的社交媒体服务可以随意监视用户，因为每个人的朋友都被锁定在他们的集中化网络中。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a4%a7%e5%9e%8b%e4%ba%91%e4%be%9b%e5%ba%94%e5%95%86%e6%97%b6%e4%bb%a3%e7%9a%84%e5%ae%a1%e6%9f%a5%e5%88%b6%e5%ba%a6%ef%bc%9a%e9%9b%86%e4%b8%ad%e5%8c%96%e6%af%81%e6%8e%89%e4%ba%92/" href="https://www.iyouport.org/%e5%a4%a7%e5%9e%8b%e4%ba%91%e4%be%9b%e5%ba%94%e5%95%86%e6%97%b6%e4%bb%a3%e7%9a%84%e5%ae%a1%e6%9f%a5%e5%88%b6%e5%ba%a6%ef%bc%9a%e9%9b%86%e4%b8%ad%e5%8c%96%e6%af%81%e6%8e%89%e4%ba%92/" rel="noopener noreferrer" target="_blank">
     集中化毁掉互联网自由
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   很难想象 Dorsey 对 “去中心化 Twitter” 的设想。真正去中心化 Twitter 的本质会削弱该公司的基本商业模式。
  </p>
  <p class="graf graf--p">
   他简要
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/jack/status/1204952252747661312" href="https://twitter.com/jack/status/1204952252747661312" rel="noopener noreferrer" target="_blank">
    提到
   </a>
   了与 ActivityPub 集成的可能性，但没有说明他是否会像在该网络上的其他活动一样停止投放广告并停止通过用户数据获利。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/video%e5%a6%82%e4%bd%95%e6%89%8d%e8%83%bd%e6%8c%91%e6%88%98-facebook-%e5%b9%b6%e5%8f%96%e5%be%97%e8%83%9c%e5%88%a9%ef%bc%9f-distributed-social-network/" href="https://www.iyouport.org/video%e5%a6%82%e4%bd%95%e6%89%8d%e8%83%bd%e6%8c%91%e6%88%98-facebook-%e5%b9%b6%e5%8f%96%e5%be%97%e8%83%9c%e5%88%a9%ef%bc%9f-distributed-social-network/" rel="noopener noreferrer" target="_blank">
     如何才能挑战 Facebook 并取得胜利？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   去中心化社交网络可以追溯到 Diaspora。 Diaspora 由四个纽约大学的学生于2010年创立，其灵感来自哥伦比亚大学法学教授 Eben Moglen 的演讲，他提议创建一个称为 “
   <a class="markup--anchor markup--p-anchor" data-href="https://freedombox.org/" href="https://freedombox.org/" rel="noopener noreferrer" target="_blank">
    freedom box
   </a>
   ” 的个人云服务器，以帮助互联网分散化。
  </p>
  <p class="graf graf--p">
   此后不久启动了 FreedomBox 项目。至关重要的是，它为社交网络、电子邮件、聊天应用程序、和日历等分散服务提供了基础架构。
  </p>
  <p class="graf graf--p">
   用户可以通过可信途径将数据发送给朋友。这样就消除了诸如 Facebook 或 Twitter 之类的监视中介，否则它们始终在人们使用其平台时监视每个人。
  </p>
  <p class="graf graf--p">
   Diaspora 最初引起了轰动，但最终失去了动力。在随后的几年中，其他技术人员承担了任务，而 Mastodon 是迄今为止最大的成功案例。
  </p>
  <p class="graf graf--p">
   目前，这些项目主要托管在社区控制的服务器上，而不是 Moglen 设想的个人云设备。但是，FreedomBox 正在进行 Fediverse 集成，
   <a class="markup--anchor markup--p-anchor" data-href="https://freedombone.net/" href="https://freedombone.net/" rel="noopener noreferrer" target="_blank">
    FreedomBone
   </a>
   等类似软件为 PeerTube 和 Friendica 等服务提供了应用程序。
  </p>
  <p class="graf graf--p">
   Twitter 可以与 Fediverse 融合，也可以研究比现有解决方案更加分散和注重隐私的方法。但是显然不太可能，因为这样做会破坏其基本商业模式。
  </p>
  <p class="graf graf--p">
   Dorsey 的真正动机可能是通过自我监管来抵制政府监管行动。例如，立法者
   <a class="markup--anchor markup--p-anchor" data-href="https://www.fastcompany.com/90273352/maybe-its-time-to-take-away-the-outdated-loophole-that-big-tech-exploits" href="https://www.fastcompany.com/90273352/maybe-its-time-to-take-away-the-outdated-loophole-that-big-tech-exploits" rel="noopener noreferrer" target="_blank">
    正在考虑
   </a>
   修订《通信规范法》第230条，该条限制了用户发布的有害内容的中介责任。将内容审核置于最终用户、小型社区和推荐算法手中的新标准可以减轻 Twitter 对诽谤、仇恨言论和耸人听闻的内容的责任。
  </p>
  <p class="graf graf--p">
   也可能是 Dorsey 希望在社会对社交媒体监视和操纵的不满情绪日益浓厚的时候推出 “激进的新解决方案”。
  </p>
  <p class="graf graf--p">
   对于人权倡导者而言，分散化社交网络是一个崇高的目标，但由于 Facebook 和 Twitter 等平台的根深蒂固的主导地位，很难扩大规模。 Twitter 似乎不太可能在没有被政府强迫的情况下做出这一改变。
  </p>
  <p class="graf graf--p">
   社交网络巨头能否在基于成千上万个可互操作网络的去中心化生态系统中创造丰厚利润？
  </p>
  <p class="graf graf--p">
   当 Twitter 和所有监视资本主义寡头一样收入来源是监视、操纵和广告时，如何才能信任 Twitter？
  </p>
  <p class="graf graf--p">
   如果 Dorsey 希望他的建议得到认真对待，他需要开始回答这些问题。⚪️
  </p>
