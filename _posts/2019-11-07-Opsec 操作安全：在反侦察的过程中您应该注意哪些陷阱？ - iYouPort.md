   对于挖掘真相的人们来说，保持匿名通常是一个重要的问题。
  </p>
  <p class="graf graf--p">
   尤其是你调查的对象非常敏感时（通常都很敏感，否则也不值得调查）就需要尽可能保证在调查完成之前不要被抓住。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">

   。

   <strong class="markup--strong markup--p-strong">
    在使用袜子木偶时仍然有可能被识别的几种途径
   </strong>
   —— 简单说：如何避免被识别；以及为什么数字指纹的防御应该是任何身份隐藏工作的一部分。
  </p>

  <p class="graf graf--p">
   并且，更普遍的是，即便您不是调查人员、对真相毫无兴趣，
   <strong class="markup--strong markup--p-strong">
    作为对自身的保护免于被操纵甚至迫害，您也值得了解这些知识
   </strong>
   ：关于 selfcare。
  </p>
  <p class="graf graf--p">
   监视资本主义寡头 FATBAG（Facebook Amazon Tencent Baidu Alibaba Google） —— 以及无数其他中间商 —— 为营销目的跟踪和分析我们所有人的绝大部分网络行为，以操纵我们的行为。
  </p>
  <p class="graf graf--p">
   ⚠️除非您对隐私非常积极主动，否则您的大多数在线活动都可以被收集、分析、出售、再出售。
  </p>
  <p class="graf graf--p">
   即便您是欧盟公民也一样，⚠️GDPR 也几乎没有提供对这种侵入式数据收集的真正保护。
  </p>
  <p class="graf graf--p">
   网络公司和政府合作收集有关您和您的设备的数据的能力意味着任何保持假名的努力都可能被轻松推翻。
  </p>
  <p class="graf graf--p">
   如果您登录到自己的 Facebook 袜子木偶帐户，然后注销并切换到您的个人帐户（或其他袜子木偶账户）—— 整个过程对 Facebook 完全透明，然后 Facebook 就可以在您的真实和假人物之间建立关联。
  </p>

   <strong class="markup--strong markup--p-strong">
    您就暴露了
   </strong>
   。
  </p>

  </p>
  <p class="graf graf--p">
   本文假设您对自我保护一无所知，于是以下内容将适用于任何首次接触这些知识的用户：没有门槛。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    IP地址
   </strong>
  </p>
  <p class="graf graf--p">
   当您连接到互联网时，您的计算机具有唯一的公共IP地址，该地址会向您访问的网站公开。
  </p>
  <p class="graf graf--p">
   这是众所周知的，并且常见的 Opsec 实践是使用Tor或VPN来伪装您的真实IP地址。我们也多次介绍过（在本网搜索“洋葱头会让谁流泪”）
  </p>
  <p class="graf graf--p">
   技术角度上这样做很有效，但是
   <strong class="markup--strong markup--p-strong">
    它有很多限制
   </strong>
   ：使用Tor浏览器限制了许多网站的功能，而且，⚠️Tor不掩饰你正在使用Tor的事实 —— 每个人都可以看到你正在使用Tor，即使他们不知道你是谁。
  </p>
  <p class="graf graf--p">
   调查人员都知道，由于谷歌吸取的数据量异常庞大，于是谷歌应用对于深度挖掘的开源情报来说非常有效。
  </p>

   一个好的 VPN 服务是一个很好的 Opsec 操作安全的重要工具，但
   <strong class="markup--strong markup--p-strong">
    Tor 或 VPN 都不是完全神奇的工具，“完全神奇”意味着你不必关注其他领域只需要使用它们就够了 —— 绝对不够
   </strong>
   。
  </p>
  <p class="graf graf--p">
   我还认为 IP 地址对谷歌或亚马逊等数据监控公司来说并不一定非常重要。
  </p>
  <p class="graf graf--p">
   根据您访问网络的方式和位置，您的IP地址可能会定期更改，或者可能会与许多其他人共享，因此，对于想要确切知道您是谁以及您希望在互联网上做什么的监视者来说，它的价值有限。
  </p>
  <p class="graf graf--p">
   但是，这并不意味着您不应该考虑您的IP地址可能会揭示您的身份，但重要的是要注意它不是完整的生活全景图。

   <strong class="markup--strong markup--p-strong">
    浏览器指纹识别
   </strong>
  </p>
  <p class="graf graf--p">
   浏览器指纹识别是一种比IP地址或 cookie 更新的跟踪概念。
  </p>
  <p class="graf graf--p">
   它并不是众所周知的，但它更具侵入性。
  </p>
  <p class="graf graf--p">
   因此对希望保持谨慎的任何调查人员来说都提出了
   <strong class="markup--strong markup--p-strong">
    更大的挑战
   </strong>
   。
  </p>
  <p class="graf graf--p">
   有一篇很棒的
   <a class="markup--anchor markup--p-anchor" data-href="https://securehomes.esat.kuleuven.be/~gacar/persistent/the_web_never_forgets.pdf" href="https://securehomes.esat.kuleuven.be/~gacar/persistent/the_web_never_forgets.pdf" rel="noopener noreferrer" target="_blank">
    学术论文
   </a>
   概述了这方面的一些研究。
  </p>
  <p class="graf graf--p">
   浏览器指纹识别的基本前提是可以将浏览器和计算机的独特功能结合起来，并将它们组合成一个独特的值，然后可以使用它
   <strong class="markup--strong markup--p-strong">
    在网上跟踪你的一举一动
   </strong>
   。
  </p>
  <p class="graf graf--p">
   具体说就是，通过查看浏览器软件版本、操作系统、系统字体、图形卡、屏幕分辨率和许多其他变量等信息，可以构建一个独特的浏览器“指纹”，然后用来在整个网络中跟踪您。
  </p>
  <p class="graf graf--p">
   从跟踪效果上对比，⚠️浏览器指纹识别比 IP 地址或 cookie 更有效和持久。
  </p>
  <p class="graf graf--p">
   💡有几种很好的工具可用于分析您自己的浏览器指纹。
  </p>
  <p class="graf graf--p">
   我最喜欢的两个是 EFF 的
   <a class="markup--anchor markup--p-anchor" data-href="https://panopticlick.eff.org/" href="https://panopticlick.eff.org/" rel="noopener noreferrer" target="_blank">
    Panopticlick
   </a>
   工具和
   <a class="markup--anchor markup--p-anchor" data-href="https://browserleaks.com/" href="https://browserleaks.com/" rel="noopener noreferrer" target="_blank">
    BrowserLeaks
   </a>
   网站。
  </p>

   浏览器指纹识别最有效的方面是画布指纹。
  </p>
  <p class="graf graf--p">
   已知使用画布指纹识别的
   <a class="markup--anchor markup--p-anchor" data-href="https://multilogin.com/sites-track-canvas/" href="https://multilogin.com/sites-track-canvas/" rel="noopener noreferrer" target="_blank">

  </p>
  <p class="graf graf--p">
   画布指纹识别的工作原理是获取有关浏览器显示字体和图形的方式的信息，然后将它们组合成一个
   <strong class="markup--strong markup--p-strong">
    可用于识别您身份的唯一
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Hash_function" href="https://en.wikipedia.org/wiki/Hash_function" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     哈希
    </strong>
   </a>
   。
  </p>

  </p>
  <p class="graf graf--p">
   此哈希是您计算机的唯一数字指纹，网站就是用这个东西来识别您。
  </p>
  <p class="graf graf--p">
   ⚠️即使你已经制作了非常好的袜子木偶，如果你从同一台机器上运行所有帐户，那么它们对于监视者来说一眼就能看出是您的操作。您的身份就暴露了。
  </p>

   <img alt="" class="aligncenter size-full wp-image-6753 jetpack-lazy-image" data-lazy-sizes="(max-width: 600px) 100vw, 600px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/19.png?resize=600%2C39&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/19.png?w=600&amp;ssl=1 600w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/19.png?resize=300%2C20&amp;ssl=1 300w" data-recalc-dims="1" height="39" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/19.png?resize=600%2C39&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="600"/>

    <img alt="" class="aligncenter size-full wp-image-6753" data-recalc-dims="1" height="39" sizes="(max-width: 600px) 100vw, 600px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/19.png?resize=600%2C39&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/19.png?w=600&amp;ssl=1 600w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/19.png?resize=300%2C20&amp;ssl=1 300w" width="600"/>

  </p>
  <p class="graf graf--p">
   相当不同了！
  </p>
  <p class="graf graf--p">
   我的浏览器不再是将独特的哈希值呈现给网站，而是呈现随机值，因此无法轻松地在网络上跟踪我。
  </p>
  <p class="graf graf--p">
   我也尝试过 Canvas Defender，它与 Panopticlick 或 BrowserLeaks 之间没有任何区别。
  </p>
  <p class="graf graf--p">
   对于 Firefox，我推荐使用 CanvasBlocker，Canvas Fingerprint Defender 或 Canvas Blocker。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    结论
   </strong>
  </p>
  <p class="graf graf--p">
   希望这篇文章能清楚地解释：
   <strong class="markup--strong markup--p-strong">
    “要创建一个真正有效的OSINT袜子木偶帐户、或者在任何情况下保护您自己的真实身份，您需要的不仅仅是一个VPN、一个假名和随机照片（
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%ae%8c%e7%be%8e%e6%9b%bf%e8%ba%ab-video/" href="https://www.iyouport.org/%e5%ae%8c%e7%be%8e%e6%9b%bf%e8%ba%ab-video/" rel="noopener noreferrer" target="_blank">

     假脸
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ）
   </strong>
   。
  </p>
  <p class="graf graf--p">
   还必须考虑网站及其背后的海量数据公司正在以什么方式盯着你。
