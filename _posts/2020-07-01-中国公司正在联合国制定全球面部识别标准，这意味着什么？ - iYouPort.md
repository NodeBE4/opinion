  <p class="graf graf--p">
   尽管人们对人脸识别技术的准确性和公平性以及政府如何使用它来监视公民一举一动的危险性而感到担忧，但它的使用范围仍在快速
   <a class="markup--anchor markup--p-anchor" data-href="https://www.engadget.com/2019/10/31/aclu-sues-over-fbi-secrecy-with-facial-recognition/" href="https://www.engadget.com/2019/10/31/aclu-sues-over-fbi-secrecy-with-facial-recognition/" rel="noopener noreferrer" target="_blank">
    扩大
   </a>
   。
  </p>
  <p class="graf graf--p">
   金融时报的一份报告
   <a class="markup--anchor markup--p-anchor" data-href="https://www.ft.com/content/c3555a3c-0d3e-11ea-b2d6-9bf4d1957a67" href="https://www.ft.com/content/c3555a3c-0d3e-11ea-b2d6-9bf4d1957a67" rel="noopener noreferrer" target="_blank">
    显示
   </a>
   ，中国公司在制定有关该技术的国际标准方面具有重要影响力，这导致担忧已经加剧。
  </p>
  <p class="graf graf--p">
   该报告详细介绍了中兴通讯、大华和中国电信等中国公司如何向联合国国际电信联盟（ITU）提出有关面部识别标准的建议。ITU 是负责电信行业全球技术标准的机构。
  </p>
  <p class="graf graf--p">
   通常，国际电联制定的标准本质上是技术性的，但是人权倡导者说，在这种情况下正在讨论的提案更像是政策建议。
  </p>
  <p class="graf graf--p">
   ⚠️ 提议的标准甚至包括针对用例的建议 —— 建议警察可以使用面部识别在人群中发现特定目标、雇主可以使用面部识别来监视员工……
  </p>
  <p class="graf graf--p">
   令人担忧的是，该技术标准将被发展中国家采用，尤其是那些缺乏资源来制定自己的标准的非洲国家。
  </p>
  <p class="graf graf--p">
   这使中国有能力控制技术市场。
  </p>
  <p class="graf graf--p">
   例如，中兴通讯和中国移动于去年6月提出了一项智能路灯标准，该标准反映了中兴通讯的智能路灯产品设计，包括：在路灯柱上添加视频监控功能的选项。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%b7%af%e7%81%af%e9%97%b4%e8%b0%8d/" href="https://www.iyouport.org/%e8%b7%af%e7%81%af%e9%97%b4%e8%b0%8d/" rel="noopener noreferrer" target="_blank">
     路灯间谍
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   在香港，类似的技术已经被用来镇压民主抗议者。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e9%a6%99%e6%b8%af%e7%9a%84%e6%8a%97%e8%ae%ae%e6%b4%bb%e5%8a%a8%e4%b8%ad%e5%86%8d%e6%ac%a1%e5%bc%ba%e8%b0%83%e4%ba%86%e4%bd%a0%e7%9a%84%e8%84%b8%e5%a6%82%e4%bd%95%e6%88%90%e4%b8%ba%e6%ad%a6%e5%99%a8/" href="https://www.iyouport.org/%e9%a6%99%e6%b8%af%e7%9a%84%e6%8a%97%e8%ae%ae%e6%b4%bb%e5%8a%a8%e4%b8%ad%e5%86%8d%e6%ac%a1%e5%bc%ba%e8%b0%83%e4%ba%86%e4%bd%a0%e7%9a%84%e8%84%b8%e5%a6%82%e4%bd%95%e6%88%90%e4%b8%ba%e6%ad%a6%e5%99%a8/" rel="noopener noreferrer" target="_blank">
     香港的抗议活动中再次强调了你的脸如何成为武器
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   中国是面部识别世界的主要大国，该技术已被中国被执法部门广泛使用，并用于监视公租房项目。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%b8%ad%e5%9b%bd%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e5%b7%a8%e4%ba%ba%e5%b4%9b%e8%b5%b7%e7%9a%84%e8%83%8c%e5%90%8e/" href="https://www.iyouport.org/%e4%b8%ad%e5%9b%bd%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e5%b7%a8%e4%ba%ba%e5%b4%9b%e8%b5%b7%e7%9a%84%e8%83%8c%e5%90%8e/" rel="noopener noreferrer" target="_blank">
     中国面部识别巨人崛起的背后
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   这导致了对侵犯人权的指控 —— 面部识别被用来监测少数民族，这是新疆对穆斯林少数民族的持续镇压的一部分。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    中国手机强制实施面部识别
   </strong>
  </h3>
  <p class="graf graf--p">
   中国要求所有新手机用户都必须提交面部扫描的要求已经生效。国际批评家称此举是 “唤醒世界各地人们的警钟”。
  </p>
  <p class="graf graf--p">
   中国工信部宣布了新规定，声称 “保护公民在网络空间中的权益和防止欺诈”，强行收集所有人的生物识别数据。新的面部扫描要求已经于去年生效。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这是一种侵入性的监视形式，这种技术也在美国的执法用途中被广泛普及。
   </strong>
  </p>
  <p class="graf graf--p">
   面部识别是监视的独特危险形式。它使政府能够对整个人口进行侵入性和无处不在的监控，非营利性数字权利倡导组织 “为未来而战” 的副主任 Evan Greer 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “没有证据表明这种技术可以改善公共安全，但它绝对是专制政权控制全社会的理想选择，”
  </p>
  <p class="graf graf--p">
   Greer 说，尽管西方人感觉使用生物识别监视技术很 “诱人”，并且可以方便地 “指着中国”，但现实是，面部监视程序也在美国和欧洲迅速传播。”
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e5%92%8c%e4%b8%ad%e5%9b%bd%e5%9c%a8%e7%9b%91%e8%a7%86%e5%85%ac%e6%b0%91%e6%96%b9%e9%9d%a2%e5%bc%95%e9%a2%86%e5%85%a8%e7%90%83/" href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e5%92%8c%e4%b8%ad%e5%9b%bd%e5%9c%a8%e7%9b%91%e8%a7%86%e5%85%ac%e6%b0%91%e6%96%b9%e9%9d%a2%e5%bc%95%e9%a2%86%e5%85%a8%e7%90%83/" rel="noopener noreferrer" target="_blank">
     美国和中国在监视公民方面引领全球
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   华盛顿邮报于去年7月份进行的一项调查
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/technology/2019/07/07/fbi-ice-find-state-drivers-license-photos-are-gold-mine-facial-recognition-searches/" href="https://www.washingtonpost.com/technology/2019/07/07/fbi-ice-find-state-drivers-license-photos-are-gold-mine-facial-recognition-searches/" rel="noopener noreferrer" target="_blank">
    显示
   </a>
   ，美国移民和海关执法人员使用面部识别技术通过州驾驶执照数据库进行搜捕。
  </p>
  <p class="graf graf--p">
   同样在7月，一份基于英国的报告发现，被伦敦市警察局使用面部识别技术标记的嫌疑人中有
   <a class="markup--anchor markup--p-anchor" data-href="https://abcnews.go.com/International/80-facial-recognition-suspects-flagged-londons-met-police/story?id=64129255" href="https://abcnews.go.com/International/80-facial-recognition-suspects-flagged-londons-met-police/story?id=64129255" rel="noopener noreferrer" target="_blank">
    80％是无辜的
   </a>
   。
  </p>
  <p class="graf graf--p">
   Greer 和为未来而战组织提倡彻底禁止出于监视目的的面部识别技术。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “中国对这项技术的实施应该是为世界各地敲响的警钟。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “面部扫描技术的警察专用监视摄像机在我们的街道上不应该存在”，北加州美国公民自由联盟的技术和公民自由律师在声明中说。
  </p>
  <p class="graf graf--p">
   而中美的民族主义竞争中，欧洲的监视技术公司正在努力将他们的技术输出到中国。具体报道：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%ac%a7%e6%b4%b2%e5%af%b9%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e5%92%8c%e5%85%b6%e4%bb%96%e4%b8%ad%e5%9b%bd%e6%8a%80%e6%9c%af%e7%9a%84%e5%85%b4%e8%b6%a3%e6%ad%a3%e5%9c%a8%e5%a1%ab%e8%a1%a5%e7%be%8e/" href="https://www.iyouport.org/%e6%ac%a7%e6%b4%b2%e5%af%b9%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e5%92%8c%e5%85%b6%e4%bb%96%e4%b8%ad%e5%9b%bd%e6%8a%80%e6%9c%af%e7%9a%84%e5%85%b4%e8%b6%a3%e6%ad%a3%e5%9c%a8%e5%a1%ab%e8%a1%a5%e7%be%8e/" rel="noopener noreferrer" target="_blank">
    欧洲对面部识别和其他中国技术的兴趣正在填补美国缺席的深圳展会
   </a>
   》。⚪️
  </p>
