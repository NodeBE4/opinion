   </noscript>
  </p>
  <p class="graf graf--p">
   虽然民主党精心挑选了23名竞争者，但特朗普的竞选活动正在致力于加强数字工具，希望能借此在2020年获胜。
  </p>
  <p class="graf graf--p">
   数字工具操纵民意不是新闻，但是
   <strong class="markup--strong markup--p-strong">
    本次的规模似乎超出了过往
   </strong>
   。
  </p>
  <p class="graf graf--p">
   特朗普的数字化运动已经全速运转。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    同一个人，使用的是四年前相同的战术，但是这次它似乎表现更出色 —— 能够更好的定位、更好地关注再次获胜所必需的选民群体
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在 2008 年和 2012 年，每个人都惊讶于一个名为 Blue State Digital 的小型组织的表现力，正是他们，在巴拉克奥巴马的选举中发挥了至关重要的作用。
  </p>
  <p class="graf graf--p">
   对于这件事的详细以及数字化选举信息战的全景图介绍在这里
   <strong class="markup--strong markup--p-strong">
    《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%9b%bd%e7%8e%8b%e4%bb%ac%e7%9a%84%e6%99%9a%e5%ae%b4-%e8%bf%99%e6%98%af%e4%b8%80%e5%9c%ba%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e6%94%bf%e5%8f%98/" href="https://www.iyouport.org/%e5%9b%bd%e7%8e%8b%e4%bb%ac%e7%9a%84%e6%99%9a%e5%ae%b4-%e8%bf%99%e6%98%af%e4%b8%80%e5%9c%ba%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e6%94%bf%e5%8f%98/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     国王们的晚宴 — — 这是一场看不见的政变
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》
   </strong>
   。
  </p>
  <p class="graf graf--p">
   而今天，民主党陷入了困境。
  </p>
  <p class="graf graf--p">
   被 WPP 收购的 Blue State 已经不再是政治游戏的一部分，而且，总的来说，民主党似乎已经被共和党的数字机器所超越。
  </p>
  <p class="graf graf--p">
   来看看他们是怎么做的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、抢占先机
   </strong>
  </p>
  <p class="graf graf--p">
   特朗普几乎在他进入白宫的那天开始就启动了他的竞选活动。
  </p>
  <p class="graf graf--p">
   事实上，美国近代史上
   <strong class="markup--strong markup--p-strong">
    没有任何总统
   </strong>
   在前两年的任期中就为下一次竞选筹集和花费了如此之多，如华尔街日报的这张图表所显示的（这些数字还
   <strong class="markup--strong markup--p-strong">
    不包括
   </strong>
   PAC 筹集和支出的资金）：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6319 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-19.png?resize=1100%2C712&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-19.png?w=1174&amp;ssl=1 1174w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-19.png?resize=300%2C194&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-19.png?resize=768%2C497&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-19.png?resize=1024%2C663&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-19.png?resize=1100%2C712&amp;ssl=1 1100w" data-recalc-dims="1" height="712" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-19.png?resize=1100%2C712&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   早在2016年就建立了特朗普数字战略的 Brad Parscale 在今年年初就被任命为竞选经理。
  </p>
  <p class="graf graf--p">
   根据他的说法，把握时机是较大计划的一部分，正如他在1月份对福克斯新闻的描述（强调“我的”）：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “……这就是为什么总统如此早地宣布任命我的原因。就如你知道的，人们经常是三个月过去了、三年过去了，却不知道自己在干什么。
   <strong class="markup--strong markup--p-strong">
    我们正在进行顶级垂直营销，我们需要准确地定位每一个会投票给总统的人，立即去找他们。如今作精准定位已经便宜很多了，现在就要开始
   </strong>
   ，而不是在媒体获得所有广告并且广告费用更高而我们不得不急于找到他们的时候再下手……”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、开销
   </strong>
  </p>
  <p class="graf graf--p">
   如下图所示，特朗普数字化的竞选活动开销 —— 为830万美元 —— 已经超过了民主党人的5倍，这还是与最接近的候选人相比（伊丽莎白沃伦170万美元）。
  </p>
  <p class="graf graf--p">
   以下数据来自 Bully Pulpit，显示了1月1日至5月25日期间的媒体宣传活动开销，很直观能看到差距。Facebook（浅色）和谷歌（深色）。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6320 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-18.png?resize=1100%2C351&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-18.png?w=1600&amp;ssl=1 1600w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-18.png?resize=300%2C96&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-18.png?resize=768%2C245&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-18.png?resize=1024%2C326&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-18.png?resize=1100%2C351&amp;ssl=1 1100w" data-recalc-dims="1" height="351" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-18.png?resize=1100%2C351&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    以上只是表面支出，并不是整个数字信息战中的一切
   </strong>
   。 2016年的时候，数字支出就达到了6.23亿美元（所有媒体的合计为
   <strong class="markup--strong markup--p-strong">
    90亿美元
   </strong>
   ）;
  </p>
  <p class="graf graf--p">
   56％的投资来自民主党人，31.3％来自共和党人，12.3％来自独立候选人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    不要看比例；最大的区别是策略
   </strong>
   。
  </p>
  <p class="graf graf--p">
   以下是 Parscale 在布加勒斯特最近一次会议上解释的2016年竞选活动最后四个月的一个例子：
  </p>
  <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote">
   <p>
    “希拉里克林顿的团队制作了66,000 个不同的广告；我的团队制作了
    <strong class="markup--strong markup--blockquote-strong">
     590万
    </strong>
    个。
    <strong class="markup--strong markup--blockquote-strong">
     这些广告是以人们*想要消费的方式*直接面向目标人群的广告
    </strong>
    。
   </p>
  </blockquote>
  <blockquote class="graf graf--blockquote">
   <p>
    我不再把人视为人口统计数字、群体、或人物角色。
   </p>
  </blockquote>
  <blockquote class="graf graf--blockquote">
   <p>
    我们要把目标视为个体，考察他们每一个人是如何行动的。因为两个看起来非常相同的人也可能会采取完全不同的行动。
    <strong class="markup--strong markup--blockquote-strong">
     这就是政治营销的未来
    </strong>
    。”
   </p>
  </blockquote>
  <p class="graf graf--p">
   在某些时候，特朗普的竞选活动是
   <strong class="markup--strong markup--p-strong">
    每天
   </strong>
   测试 100,000 个不同的广告，在上次总统辩论期间达到了峰值 ——
   <strong class="markup--strong markup--p-strong">
    175,000个变化
   </strong>
   。
  </p>
  <p class="graf graf--p">
   此外，特朗普竞选活动的
   <strong class="markup--strong markup--p-strong">
    每10条信息中就有超过8条是关于直接采取行动的
   </strong>
   —— 而希拉里竞选活动中只有一半信息关乎行动。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、2020年的活动：范围更广
   </strong>
  </p>
  <p class="graf graf--p">
   今年，Brad Parscale 以非常简单的方式解释了他的战略蓝图，就像一个月前对 CBS 的采访、以及1月份对福克斯新闻的采访中所说的那样。
  </p>
  <p class="graf graf--p">
   以下是关键部分摘录：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    关于数字营销活动的规模 ——
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “作为现任，我想说我们现在正在建立一个与2016年不同的业务。
  </p>
  <p class="graf graf--p">
   在2016年，这项业务是非常基层的，如你所知，该活动有时最好的运作状态中也只是一小部分有效。
  </p>
  <p class="graf graf--p">
   这是动态的。它正在改变。它也正在适应。当时的服务目标是平生第一次参选的人。而在2020年，运作会有所不同。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这将是一个更大规模的地面战。这将需要新技术、新事物
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Facebook 仍然会是是个好地方，但
   <strong class="markup--strong markup--p-strong">
    我们已经从 Facebook 收获了我们想要的东西
   </strong>
   ” —— Face the Nation
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <strong class="markup--strong markup--p-strong">
    正在进行的这次选举非常关注营销、广告，理解数据和分析数据，建立基础结构，以了解所需要的一切信息以及如何实现赢的目标
   </strong>
   。
  </p>
  <p class="graf graf--p">
   有很多东西来自对如何构建基础设施以便能够与数千万人进行沟通的理解，以及
   <strong class="markup--strong markup--p-strong">
    如何有效地花费五亿六千万美元的宣传预算 [后来，Parscale 承认支出可能达到了10亿美元！
   </strong>
   ]” —— Face the Nation
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “2016年，我们有大约70万名志愿者。现在，2020年的选举有160万名志愿者提供服务，我们计划
   <strong class="markup--strong markup--p-strong">
    更多利用手机上的技术、应用程序和其他技术开发
   </strong>
   。
  </p>
  <p class="graf graf--p">
   有些人可能负责街头聚会、有些人可能会参与社交媒体运营、有些人可能会挨家挨户敲门。
  </p>
  <p class="graf graf--p">
   在每一个坐标中我们都在考虑如何比我们在2016年做得更大、更好” —— Face the Nation
  </p>
  <p class="graf graf--p">
   另一个指标也可以说明两次营销业务之间的差异：2016年，Parscale 在全国拥有3,000名团队领导者。这次他将有 90,000 名。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    目标定位 ——
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “现在，这些政策[消息]有一系列的东西。在 1300 Elm Street 的那个人可能真的很关心移民问题；而在 1305 Elm Street 的人可能关心的是税收政策。
  </p>
  <p class="graf graf--p">
   现在，这并不意味着我们正在改变总统的信息对他们的影响。而是，
   <strong class="markup--strong markup--p-strong">
    我们必须向每一位选民提供真正能触动他们的信息
   </strong>
   ” —— Fox News
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这是明尼苏达州的选民，只要增加 26,000 名新支持者我就可以扭转局面。所以，我现在就去找他们。
  </p>
  <p class="graf graf--p">
   我们每个月花费数百万美元，比历史上的任何一次战役都更加领先，为我们
   <strong class="markup--strong markup--p-strong">
    找到需要针对性推销的人
   </strong>
   、找到我们需要了解的内容、我们需要对他们说什么、以及如何
   <strong class="markup--strong markup--p-strong">
    准确地影响他们
   </strong>
   ，奠定基础” —— Face The Nation
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们必须拿到可以能直接联系到人们的东西。手机号码、电子邮件地址，等等。
  </p>
  <p class="graf graf--p">
   一个好的候选人可能在选举日会有四到五百万票。我们可能会有6000万 [关于背景： 2016年有1.38亿人投票，其中投特朗普的人是 6300万] 在选举日，我们可能会让所有我们直接联系过的人都投票给总统。
  </p>
  <p class="graf graf--p">
   这就是我们在整个时间里花费的精力。” —— Fox News
  </p>
  <p class="graf graf--p">
   Parscale 已经公开声明了
   <strong class="markup--strong markup--p-strong">
    要为关键的蓝州部署这些营销策略
   </strong>
   ：科罗拉多州、新墨西哥州、明尼苏达州、内华达州。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、Facebook 的嵌入问题
   </strong>
  </p>
  <p class="graf graf--p">
   没有人知道像 Facebook 或谷歌这样的大型平台是否会有“嵌入”，即 分配给每个信息战活动的顾问。
  </p>
  <p class="graf graf--p">
   事实是，Parscale 和他的100多名队友已经不再需要 Facebook 了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这已经不是2015年
   </strong>
   ，当时他要求 Facebook 的人向他解释网络无可挑剔的广告机器的每个功能。
  </p>
  <p class="graf graf--p">
   无论 Facebook 如何公开谈论政治广告，该寡头都会
   <strong class="markup--strong markup--p-strong">
    毫无疑问地采取一切措施来支持可能带来超过10亿美元收入的业务
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、WhatsApp 未知
   </strong>
  </p>
  <p class="graf graf--p">
   目前为止 WhatsApp 已经在多次选举中展示了其
   <strong class="markup--strong markup--p-strong">
    信息战的强大能力
   </strong>
   ，巴西和印度是最有说服力的例子。
  </p>
  <p class="graf graf--p">
   然而，WhatsApp 在美国的受欢迎程度远低于亚洲：虽然 Facebook 在美国有2.21亿用户，而 Facebook Messenger 有1.3亿用户，但是，WhatsApp 在美国只有2300万用户。
  </p>
  <p class="graf graf--p">
   但是，该应用程序的使用量可以快速增长，
   <strong class="markup--strong markup--p-strong">
    并且已经被特朗普的信息战活动所使用。
   </strong>
  </p>
  <p class="graf graf--p">
   WhatsApp 是虚假信息和诽谤信息的理想载体：它模仿一对一的对话，并且无法跟踪来源。
  </p>
  <p class="graf graf--p">
   还有，Mark Zuckerberg 已经承诺，在他所谓的“隐私聚焦愿景”中，要使 WhatsApp 可以与SMS互操作。
  </p>
  <p class="graf graf--p">
   对于刚刚超过 10,000 个虚假陈述标记的总统来说，这可能是一个十分诱人的工具。⚪️
  </p>
  <p class="graf graf--p">
   **注：自2016年至今，IYP 曾经做过两个较大的预测。这两个预测都不详。
  </p>
  <p class="graf graf--p">
   第一个就是“贸易战对改变中国没有任何作用”，因为中美是高堡奇人的关系 —— 高堡奇人不仅意味着长期持久的竞争，而且意味着双方谁也不会击败谁，因为他们互相依存。
  </p>
  <p class="graf graf--p">
   这点已经在上个月的G20会议期间得到了证实。
  </p>
  <p class="graf graf--p">
   由于这一预言没能令人们明显放弃对美国的“期待”；后来我们干脆将
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e9%ab%98%e5%a0%a1%e5%a5%87%e4%ba%ba/" href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e9%ab%98%e5%a0%a1%e5%a5%87%e4%ba%ba/" rel="noopener noreferrer" target="_blank">
    高堡奇人
   </a>
   放入了网站栏目，汇集所有美国和中国之间的不为人知的合作和竞争的另一面。
  </p>
  <p class="graf graf--p">
   这绝非是我们所希望看到的。
  </p>
  <p class="graf graf--p">
   另一个预测就是，下一次特朗普还会赢。这是更大的不希望；但是民主党在很多方面仍然不力。
  </p>
  <p class="graf graf--p">
   目前距离大选还有一年的时间，希望这个预测能够被推翻（我们对两大党派都不看好）。
  </p>
  <p class="graf graf--p">
   此外，关于美国大选，有一篇文章能够解释一些问题《
