  <p class="XzvDs _208Ie _2Dym_ blog-post-text-font blog-post-text-color _2p1aK _2R0Lu _2Dym_">
   防病毒软件使人们能够轻松使用手机、平板电脑和计算机，能够在网络上更安全地浏览，而不用担心将私人信息传播给其他人（或任何病毒）。
  </p>
  <p class="XzvDs _208Ie _2Dym_ blog-post-text-font blog-post-text-color _2p1aK _2R0Lu _2Dym_">
   一些好的防病毒软件还包含对来自其他计算机的威胁（如垃圾邮件，网上银行攻击，恶意URL，骗局和钓鱼攻击，在线身份（隐私），社交工程技术，高级持续威胁（APT）和僵尸网络DDoS攻击 的保护措施。
  </p>
  <p class="XzvDs _208Ie _2Dym_ blog-post-text-font blog-post-text-color _2p1aK _2R0Lu _2Dym_">
   以下是免费开源的、且轻量级的前 5 个 #AntiVirus 工具列表。无论您是初学者还是高级用户，这些都可以帮助您保持计算机的安全：
  </p>

  <p class="XzvDs _208Ie _2Dym_ blog-post-text-font blog-post-text-color _2p1aK _2R0Lu _2Dym_">
   2/ 🔐
   <strong>
    ClamAV
   </strong>
   Clam AntiVirus 或
   <a class="_2qJYG blog-link-hashtag-color _3sz0l" href="https://www.clamav.net/" rel="noopener noreferrer" target="_blank">
    <u class="sDZYg">
     ClamAV
    </u>
   </a>
   是一款免费的，跨平台开源防病毒软件工具包，用C和C ++编写，并在 GNU 通用公共许可证下授权。4月9日发布，不仅用于 Unix，还适用 AIX，BSD，HP-UX，Linux，macOS，OpenVMS，OSF（Tru64）和Solaris
  </p>
  <p class="XzvDs _208Ie _2Dym_ blog-post-text-font blog-post-text-color _2p1aK _2R0Lu _2Dym_">
   它可以检测许多类型的恶意软件和病毒，并且可以在
   <strong>
    邮件服务器上
   </strong>
   将其用作服务器端电子邮件病毒扫描程序。从版本 0.97.5 开始，ClamAV 在 Microsoft Windows 上构建并运行，并且可以免费下载。
  </p>
  <p class="XzvDs _208Ie _2Dym_ blog-post-text-font blog-post-text-color _2p1aK _2R0Lu _2Dym_">
   ClamAV 包含许多实用程序，如命令行扫描程序，自动数据库更新程序和可扩展的多线程守护程序，它们在共享库的反病毒引擎上运行。
  </p>

  <p class="XzvDs _208Ie _2Dym_ blog-post-text-font blog-post-text-color _2p1aK _2R0Lu _2Dym_">
   3/ 🔐
   <a class="_2qJYG blog-link-hashtag-color _3sz0l" href="http://www.clamwin.com/" rel="noopener noreferrer" target="_blank">
    <strong>
     <u class="sDZYg">
      ClamWin
     </u>
    </strong>
   </a>
   用于 Windows 的免费和开源防病毒工具，用 C ++和 Python 编写，为 Clam AntiVirus 引擎提供图形用户界面。它是根据 GNU 通用公共许可证授权的。
  </p>
  <p class="XzvDs _208Ie _2Dym_ blog-post-text-font blog-post-text-color _2p1aK _2R0Lu _2Dym_">
   其功能包括用户登录时的扫描调度程序、自动病毒库更新、以及用于 Windows 资源管理器的具有上下文菜单集成的独立病毒扫描程序。
  </p>

    5/ 🔐
    <a class="_2qJYG blog-link-hashtag-color _3sz0l" href="https://sourceforge.net/projects/moonav/" rel="noopener noreferrer" target="_blank">
     <strong>
      <u class="sDZYg">
       Moon Secure AV
      </u>
     </strong>
    </a>
    一款基于 GNU/GPL 协议的免费软件，使用 clamav engine 引擎，功能比较全，查、杀、防病毒，反木马和间谍软件，有防火墙，其功能还包括按需扫描恶意软件，实时保护，实时更新，资源占用率低，可在 XP + 的所有 Windows 上运行。
   </p>
   <p class="XzvDs _208Ie _2Dym_ blog-post-text-font blog-post-text-color _2p1aK _2R0Lu _2Dym_">
    在这个数字时代，越来越多的人正在使用计算机作为研究和交流的有力工具。但是由于这个革命性的发明和不断的创新，有时我们会觉得交换信息或电子邮件时不再安全。但也正是由于这种进步，创建有助于抵御各种恶意软件的防病毒工具已经成为可能。这些工具不仅能带给你在线安全，并且，

    。⚪️
