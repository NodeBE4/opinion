      <strong>
       普京曾经说，”成为 AI 领域领导者的人将成为整个世界的统治者。” 俄罗斯在此是怎么做的？为什么与北京如此相似？
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-3693 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-312.png?resize=1100%2C737&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-312.png?w=1266&amp;ssl=1 1266w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-312.png?resize=300%2C201&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-312.png?resize=768%2C514&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-312.png?resize=1024%2C686&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-312.png?resize=1100%2C737&amp;ssl=1 1100w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-312.png?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="737" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-312.png?resize=1100%2C737&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-3693" data-recalc-dims="1" height="737" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-312.png?resize=1100%2C737&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-312.png?w=1266&amp;ssl=1 1266w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-312.png?resize=300%2C201&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-312.png?resize=768%2C514&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-312.png?resize=1024%2C686&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-312.png?resize=1100%2C737&amp;ssl=1 1100w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-312.png?resize=272%2C182&amp;ssl=1 272w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   这很可能是普京的主意。他曾经对AI有过
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/newsletters-07302019-b66dc307cf8b" href="https://medium.com/@iyouport/newsletters-07302019-b66dc307cf8b" rel="noopener noreferrer" target="_blank">
    明确的说法
   </a>
   ：”成为这个领域领导者的人将成为整个世界的统治者。”
  </p>
  <p class="graf graf--p">
   莫斯科政府部长 Eduard Lysenko 告诉我，“俄罗斯传统上一直是开拓者之一”；“感谢有才华的员工队伍和悠久的数学传统。在我们看来，我们已经开发了许多突破性的解决方案，使俄罗斯能够跻身顶级 AI 国家行列。”
  </p>
  <p class="graf graf--p">
   Lysenko 是莫斯科信息技术部的负责人，这次“独家”采访是因为他想谈谈5G以及该市的部署计划。鉴于美国提出的安全担忧，他知道我想讨论
   <strong class="markup--strong markup--p-strong">
    莫斯科的面部识别和莫斯科部署华为5G网络的决定。
   </strong>
  </p>
  <p class="graf graf--p">
   具有讽刺意味的是，你很快就会发现，莫斯科的监控剧本可能是从北京拷过来的。
  </p>
  <p class="graf graf--p">
   在去年的世界杯期间，俄罗斯部署了著名的大规模面部识别技术，来帮助警察监视此次活动的全局。“面部识别系统使得警方可以将联邦通缉名单上的12名罪犯、8名扒手、以及60名被禁止观看比赛的球迷逮捕扣留。”
  </p>
  <p class="graf graf--p">
   Lysenko 解释说，
   <strong class="markup--strong markup--p-strong">
    “目前面部识别系统已经安装在1,500个视频流的试验模式中，占莫斯科167,000个监视摄像机的1.5％。今年，我们打算将数量增加到105,000，
   </strong>
   这意味着所有安装在公共区域的摄像机都可以优化系统的使用。“
  </p>
  <p class="graf graf--p">
   莫斯科已经大量使用了其笼罩全市的监控系统，该系统现在用于调查莫斯科70％的犯罪案件。最近，视频监控还被用来寻找从国家特列季亚科夫画廊偷走 Arkhip Kunidzhi 画作的人。“
  </p>
  <p class="graf graf--p">
   该系统每年捕获12亿小时的视频，每天发出 45,000 个交通罚款，并帮助建立了一个统一的系统，当局声称该系统将首次响应时间减少了20％。
  </p>
  <p class="graf graf--p">
   但这不仅仅是CCTV。还记得中国PR机器、配备人工智能眼镜的警察等报道吗？嗯，莫斯科的计划是一样的。“将来，5G 将允许我们使用面部识别眼镜等新设备为警察扩展系统。我们
   <strong class="markup--strong markup--p-strong">
    正在与 Ntechlab 公司一起测试具有嵌入式面部识别功能的增强现实眼镜，该公司以创建面部识别工具 FindFace 而闻名。
   </strong>
   “
  </p>
  <p class="graf graf--p">
   FindFace 是一款高度精确的面部识别引擎，在这个以其在太空中的实力而闻名的国家中脱颖而出。但 Lysenko 还谈到了
   <strong class="markup--strong markup--p-strong">
    国际合作，中国是一个明显的起点
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们的任务是使用智能视频监控系统来监控城市流程并确保公民的安全。我们的闭路电视系统使用俄罗斯公司开发的识别技术。当然，
   <strong class="markup--strong markup--p-strong">
    我们有兴趣与包括中国在内的外国同事交流经验，中国在部署面部识别系统方面比其他国家更先进
   </strong>
   。“
  </p>
  <p class="graf graf--p">
   这引起了对5G的讨论。莫斯科已经开始使用5G并计划到2019年进行全面实验。“在2018年世界杯期间，”Lysenko 告诉我，“MegaFon 使用诺基亚5G设备来演示 VR 广播技术。五十人使用VR眼镜观看广播。体育场摄像机广播到5G蜂窝塔，蜂窝塔传输到连接 VR 眼镜的智能手机，每台设备使用高达 35 Mbps 的速度。不过，现在，“MegaFon 已经表示同意与华为在俄罗斯制定并实施 5G 标准。”
  </p>
  <p class="graf graf--p">
   这使得讨论进入了美国和英国提出的问题，尤其是与华为相关的安全风险，包括有关该公司可能为北京情报收集提供便利的指控。在这个问题上，Lysenko 的回应很简短：“俄罗斯联邦有严格的信息安全法规，我们一直遵守这些法规。”华盛顿怀疑华为与北京的情报联系，也许俄罗斯和华盛顿对各自国家安全的威胁有着不同的看法。
  </p>
  <p class="graf graf--p">
   无论如何，华为都不是全部。 “还有其他公司的合作伙伴关系，”Lysenko 说，“俄罗斯电信运营商将根据其采购政策选择5G供应商用于试验区，现在正在为5G技术的实施准备技术标准。之后，这个过程将加速。
   <strong class="markup--strong markup--p-strong">
    我们与许多设备供应商合作，包括华为，诺基亚，爱立信，高通和IBM。
   </strong>
   “
  </p>
  <p class="graf graf--p">
   那么俄罗斯自己的设备制造商呢？
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这些决定是由[网络运营商]而不是政府做出的，”但“我们希望当地的俄罗斯供应商能够参加试点项目，以展示他们的进展。”
  </p>
  <p class="graf graf--p">
   目的是让2019年的试点参与者在世界杯[监视]试验成功的基础上再接再厉，“2020–2020年预计将实现5G的全面商用”。 2019年的参与者将瞄准莫斯科的关键地区。“这些领域分为两大类，”Lysenko 解释说，“拥挤的地方，如公园和中央街道，将举行消费者技术5G测试和示范，然后是创新中心和技术园区，技术公司将能够测试工业5G。”
  </p>
  <p class="graf graf--p">
   而对于 Lysenko 来说，这完全是关于工业和自动化的。莫斯科的IT部门“拥有自己的人工智能团队，致力于在城市管理中实施人工智能。例如，文件分类和机器人操作员的自动化处理了对城市呼叫中心的所有请求的 40％ — — 机器人与城市信息系统集成并且可以提供62个典型问题的答案，它能够识别情绪和自主学习。”
  </p>
  <p class="graf graf--p">
   除了与通常的大学研究合作外，还“与莫斯科物理科技大学和国立科技大学’MISiS’等世界一流大学合作，”人们认识到需要与行业合作才能取得成功。“该市正在通过为技术公司的发展创建基础设施来推动创新，包括技术园区和公司财务支持系统。我们还支持像 Yandex 这样的大公司与他们交换开发新服务所需的数据。 “
  </p>
  <p class="graf graf--p">
   上个月，俄罗斯互联网巨头 Yandex 宣布与现代摩比斯“联手”自动驾驶汽车。 “Yandex 的自动驾驶汽车已在莫斯科、特拉维夫和拉斯维加斯的街道上成功驾驶，”当时 Yandex 首席执行官 Arkady Volozh
   <a class="markup--anchor markup--p-anchor" data-href="https://yandex.com/company/press_center/press_releases/2019/2019-03-19" href="https://yandex.com/company/press_center/press_releases/2019/2019-03-19" rel="noopener noreferrer" target="_blank">
    解释说
   </a>
   ，“这意味着它可以扩展到任何地方。我们花了两年时间才从最初的基本测试转变为完整的公共自动驾驶服务。现在，由于我们与现代摩比斯的协议，我们将能够更快地行动。“
  </p>
  <p class="graf graf--p">
   Lysenko 告诉我，“自2018年12月以来，所有自动驾驶汽车开发商都可以在莫斯科提交测试请求。经过实验室测试后就可以进行户外测试。到目前为止，最成功的是 Yandex，其自动驾驶汽车已经在莫斯科街头进行了测试。 Yandex 还在 Skolkovo 地区提供自动驾驶出租车服务。大约30家其他公司有可能在不久的将来开始测试，包括 KAMAZ，其自动驾驶公交车和自动驾驶卡车现在处于实验室测试阶段。“
  </p>
  <p class="graf graf--p">
   不仅仅是小汽车。 “莫斯科政府和俄罗斯开发商 Cognitive Technologies 正联合开发俄罗斯的全自动电车。电车目前正在车厂进行测试。它将很快会在一条路线上进行试运行。
   <strong class="markup--strong markup--p-strong">
    预计到 2021 年至 2222 年将实现完全的自主驾驶电车。莫斯科地铁最近宣布，自动驾驶列车可能会在大约5年内出现在环线上。
   </strong>
   目前莫斯科地铁正在分析国际经验并寻找适合莫斯科的解决方案。“
  </p>
  <p class="graf graf--p">
   除了交通，Lysenko 想要关注的领域还有医疗保健，5G 和物联网将带来的批量变革。他解释说，“世界杯结束后不久，莫斯科的信息技术和医疗部门与电信运营商一起组织了5G医疗保健可能性的示范。华为 CPE 已经用于远程超声诊断和基因测序，测试表明，响应时间足以让健康专家进行舒适的远程工作。“
  </p>
  <p class="graf graf--p">
   鉴于其在世界杯上的经验，莫斯科拥有一些真实的5G数据，以帮助指导部署和管理期望。“5G测试显示一个扇区的最大速度为7.5 Gbps，延迟时间为4 ms。理论上，标准情况下的5G记录可以是35 Gbps，这意味着下载高清电影只需要不到3秒。”
  </p>
  <p class="graf graf--p">
   也就是说，在美国进行的测试表明，部分5G试验和供应商的声称给消费者设定了错误的预期。莫斯科仍有世界杯5G设备，将重复使用，“该网络目前尚未运营。但该设备也尚未拆除，并将于11月成为5G试点网络的一部分。”
  </p>
  <p class="graf graf--p">
   莫斯科是一个人口规模与洛杉矶相似的城市，比巴黎和伦敦都要大。
   <strong class="markup--strong markup--p-strong">
    Lysenko 说，“我们的目标是与纽约或北京的城市管理者一样
   </strong>
   ，使用技术为市民提供更好的服务，让城市以更有效的方式运作。为实现这一目标，
   <strong class="markup--strong markup--p-strong">
    我们与全球其他城市的代表自由交流智能城市发展的想法和经验，包括人工智能应用
   </strong>
   。我们广泛使用像 GitHub 这样的公共平台 — — 与世界各地的开发者分享我们的算法。“
  </p>
  <p class="graf graf--p">
   他已经看到这个宏伟计划“很好地”登陆了一个城市，75％的公民使用在线公共服务，超过200万人在电子投票系统上注册，可以针对特定的城市问题进行投票。他说，“莫斯科市民普遍对新技术感兴趣，他们正在等待世界上所有用户的5G商业运营。”
  </p>
  <p class="graf graf--p">
   然而，这些市民对全市105,000台面部识别监视摄像机的反应仍有待观察。
  </p>
