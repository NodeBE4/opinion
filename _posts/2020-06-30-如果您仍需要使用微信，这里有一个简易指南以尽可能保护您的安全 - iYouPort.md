  <p class="graf graf--p">
   如您所知，IYP 的
   <strong class="markup--strong markup--p-strong">
    读者来稿通道主要支持解决方案的思考
   </strong>
   ，即 根据您和您的周边人所面临的难题、困境，寻找可能的方法和措施。它可以是技术角度的、可以是思想角度的、可以是行动经验的延展、也可以是前所未有的创造力。唯一的期待是，实用价值。
  </p>
  <p class="graf graf--p">
   我们在这里，随时恭候您的智慧。
  </p>
  <p class="graf graf--p">
   【读者来稿】 IYP 的中文读者都知道微信是国家级的大规模监控工具，但在中国，很少有人能够避免使用它，包括在华的外国人。
  </p>
  <p class="graf graf--p">
   当需要和家人、朋友、同事、客户等等社会关系建立便捷的即时通讯渠道，微信几乎是普通中国人的 “唯一选择”。
  </p>
  <p class="graf graf--p">
   如果您也未能免俗，不妨思考如何在减少隐私暴露的前提下，尽可能 “安全”地使用微信。“戴着镣铐跳舞”，这是在集权统治下技术公民必须学会和经常做的事。
  </p>
  <p class="graf graf--p">
   言归正传，微信的隐私威胁主要来自两个方面：
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    搜集和分析聊天数据，基于大数据分析向用户推送广告和向政府报告敏感内容。
   </li>
   <li class="graf graf--li">
    利用手机广泛搜集元数据，并结合其他数据集（如搜索关键词、社交媒体帐号、浏览内容、其他APP信息等），精准详细地分析描绘出用户的数字生活乃至日常生活全景。
   </li>
  </ol>
  <p class="graf graf--p">
   在上述危险中，1 是您无法避免的，只能通过减少使用来减少损失。
  </p>
  <p class="graf graf--p">
   微信聊天内容对政府是完全透明的，您当然知道只在微信中保留必要的少数几个联系人，
   <strong class="markup--strong markup--p-strong">
    不使用
   </strong>
   “钱包” 和 “朋友圈” 等多余功能，聊天时只谈工作、生活上不敏感的事，这样即使您是警察重点关注对象，也能将信息风险控制在可以接受的范围内。
  </p>
  <p class="graf graf--p">
   有些朋友可能认为使用境外手机号注册微信能避免被审查。一定程度上的确如此，比如香港话题、新疆话题，中国手机号注册的账户不能说，或者消息无法显示，但境外手机号注册的账户就可以显示这类内容。但是，境外手机号注册时会被监控，监控的结果被用来审查所有中国微信使用者。
  </p>
  <p class="graf graf--p">
   （
   <strong class="markup--strong markup--p-strong">
    编者注
   </strong>
   ：这是加拿大公民实验室最近的调查所揭示的，IYP不久后将发布具体解读，关于微信如何操作这种监视）
  </p>
  <p class="graf graf--p">
   您能做的，是通过技术手段来避免上述 2，即：减少微信的元数据和手机数据搜集。
  </p>
  <p class="graf graf--p">
   忘记那些复杂难懂的权限管理APP和技巧吧，智能手机就是为搜集数据而生的间谍工具，将您的手机调教成“乖孩子”不过是痴心妄想。
  </p>
  <p class="graf graf--p">
   丢掉手机，
   <strong class="markup--strong markup--p-strong">
    笔记本电脑运行虚拟机是您使用微信的更好选择
   </strong>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     请按照以下步骤操作 ——
    </strong>
   </span>
  </h3>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第一步：
   </strong>
  </h3>
  <p class="graf graf--p">
   4G随身路由器，电量越大越好，能够满足6小时开机，某些型号甚至可刷 openwrt 固件，能改MAC地址，安全性更佳。
  </p>
  <p class="graf graf--p">
   使用电话卡流量上网，推荐购买 “已实名” 的流量黑卡，物联网卡更佳，因为缺少一些普通手机卡的电信指令，更难被定位和监控。
  </p>

   </noscript>
  </figure>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第二步：
   </strong>
  </h3>
  <p class="graf graf--p">
   一台笔记本电脑，最好是超级本，有较大的内存，超长的续航，可一整天开机使用。
  </p>
  <p class="graf graf--p">
   如有较高安全需求，可将摄像头遮蔽，麦克风模块取掉，被称为 mic blocker 的小工具也可屏蔽麦克风。
  </p>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第三步：
   </strong>
  </h3>
  <p class="graf graf--p">
   安装
   <a class="markup--anchor markup--p-anchor" data-href="https://virtualbox.org/wiki/Downloads)" href="https://virtualbox.org/wiki/Downloads%29" rel="noopener noreferrer" target="_blank">
    virtualbox
   </a>
   ，如需剪贴板共享等功能，请为 virtualbox安装拓展包。
  </p>
  <p class="graf graf--p">
   然后安装
   <a class="markup--anchor markup--p-anchor" data-href="https://genymotion.com/download/" href="https://genymotion.com/download/" rel="noopener noreferrer" target="_blank">
    genymotion Desktop
   </a>
   ，注册后（
   <a class="markup--anchor markup--p-anchor" data-href="https://t.co/uKfXK3QsC8?amp=1" href="https://t.co/uKfXK3QsC8?amp=1" rel="noopener noreferrer" target="_blank" title="https://www.genymotion.com/account/create/">
    https://genymotion.com/account/create/
   </a>
   ）使用免费版本，注意填写有效的邮箱以接收验证码。
  </p>
  <p class="graf graf--p">
   下面在 Linux 系统下演示，通常认为 Linux 的安全性和性能优于 Windows 等闭源操作系统。
  </p>
  <p class="graf graf--p">
   但本文所演示内容在 windows 下也有效。
  </p>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第四步：
   </strong>
  </h3>
  <p class="graf graf--p">
   创建虚拟机。
  </p>
  <p class="graf graf--p">
   在 genymotion 界面点击+号，列表中选择要虚拟的设备。
  </p>
  <p class="graf graf--p">
   推荐使用8.0版本的 Android API，设备选择手机而非平板，因为虚拟平板在使用微信时存在屏幕90度翻转且无法调节的bug。
  </p>
  <p class="graf graf--p">
   由于微信需要arm指令集支持，您还需要下载
   <a class="markup--anchor markup--p-anchor" data-href="https://github.com/m9rco/Genymotion_ARM_Translation" href="https://github.com/m9rco/Genymotion_ARM_Translation" rel="noopener noreferrer" target="_blank">
    相应版本的补丁
   </a>
   ，将zip文件拖到 genymotion 虚拟设备界面即可安装。
  </p>
  <p class="graf graf--p">
   然后安装微信，您可以选择apk安装包，也可通过
   <a class="markup--anchor markup--p-anchor" data-href="https://opengapps.org" href="https://opengapps.org" rel="noopener noreferrer" target="_blank">
    安装OpenGapps框架
   </a>
   后在谷歌商店安装。
  </p>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第五步：
   </strong>
  </h3>
  <p class="graf graf--p">
   配置代理服务器。
  </p>
  <p class="graf graf--p">
   花些小钱购买一个 VPS 是必要的，搭建 shadowsocks/v2ray/trojan/ 迷雾通等可以 “长连接” 的 socks 代理服务（教程很多，请自行搜索学习）。
  </p>
  <p class="graf graf--p">
   如果有他人提供的可靠的代理服务器也可以用。
  </p>
  <p class="graf graf--p">
   在运行虚拟机的笔记本上运行代理软件客户端，安装并配置 privoxy 软件，修改 config 文件，找到 “forward-socks5t” 这一行，修改为 “forward-socks5 / 127.0.0.1:[代理软件的本地端口] ”，重启 privoxy。
  </p>

   </noscript>
  </figure>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第六步：
   </strong>
  </h3>
  <p class="graf graf--p">
   运行虚拟设备。
  </p>
  <p class="graf graf--p">
   在 android 设置界面依次：点击WIFI — — 长按热点名称 — — 修改 — — 高级选项 — — 代理方式手动 — — 主机名10.0.3.2，端口8118 — — 点击保存。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    您的虚拟设备可以全局代理上网了，运行并登录您的微信吧！
   </strong>
  </p>
