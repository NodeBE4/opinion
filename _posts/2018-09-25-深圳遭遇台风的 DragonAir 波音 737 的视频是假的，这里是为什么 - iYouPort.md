  <p class="graf graf--p graf-after--figure" id="6da4">
   一则显示在“深圳”台风中被发现的商用客机的视频正在网上流传。视频中，客机被认为是在进行低空飞行。
  </p>
  <p class="graf graf--p graf-after--p" id="a9fe">
   视频附带的说明声称“无论台风如何，飞行员都能安全降落” — DragonAir B-737 hit by typhoon in Shen Zhen. But managed to land safely。视频的后半部分显示人们在着陆后从“同一架飞机”撤离的场景。
  </p>
  <p class="graf graf--p graf-after--p" id="3208">
   如果你看出这个视频有点问题，那么恭喜，你猜对了。因为显示飞机正在执行看似不可能的低空行驶的视频的前半部分是被数字化创建的，并且与人们撤离时画面 即视频的后半部分所示的背景完全不同。
  </p>
  <p class="graf graf--p graf-after--p" id="5493">
   仔细观察你会发现，碰上了台风的飞机实际上来自从 2017 年6月14日的 MeniThings YouTube 频道的视频（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.youtube.com/watch?v=AgvzhJpyn10" href="https://www.youtube.com/watch?v=AgvzhJpyn10" rel="noopener nofollow noreferrer" target="_blank">
    视频在这里
   </a>
   ）。MeniThings 是一家制作公司，除其他事项外，该公司还生产一些有趣的视觉效果。他们的网站上就直接这样写着：
  </p>
  <blockquote class="graf graf--blockquote graf-after--p" id="c6a0">
   <p>
    <em class="markup--em markup--blockquote-em">
     MeniThings 是一家全功能的制作公司，支持导演 Aristomenis Tsirbas 的原创动画和现场表演作品，其中包括故事片、电视节目、音乐视频和视觉效果。
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p graf-after--blockquote" id="0fcc">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.youtube.com/channel/UCgVtFgr_DYgF3lA-aKeanCw" href="https://www.youtube.com/channel/UCgVtFgr_DYgF3lA-aKeanCw" rel="noopener nofollow noreferrer" target="_blank">
    该公司的 YouTube 频道
   </a>
   充满了 CGI 飞机特技、不明飞行物的视频，甚至还有几次对外星人的采访（Haha）。因此，在台风中捕获的飞机视频显然是 CGI 创建的视频，并没有描绘真实的飞机。
  </p>
  <p class="graf graf--p graf-after--p" id="d232">
   视频的后半部分显示一架飞机上的乘客正在疏散，看起来很像是前半部分那个安全着陆的飞机？
  </p>
  <p class="graf graf--p graf-after--p" id="d8d1">
   但完全不是这样。在视频的下半部分显示人们从中撤离的这架飞机，是来自于
   <a class="markup--anchor markup--p-anchor" data-href="https://www.dailymail.co.uk/news/article-6105287/Chinas-Shenzhen-airport-shuts-runway-briefly-emergency-landing.html" href="https://www.dailymail.co.uk/news/article-6105287/Chinas-Shenzhen-airport-shuts-runway-briefly-emergency-landing.html" rel="noopener nofollow noreferrer" target="_blank">
    2018 年8月28日发生的事件
   </a>
   ，当时从北京到澳门的一架飞机因发动机和起落架问题而不得不转移到深圳机场，而不是任何与台风相关的问题。
  </p>
  <p class="graf graf--p graf-after--p" id="ef24">

   你可以清楚地看到，飞机的尾部有不同的标志。尽管该视频声称这架飞机是 DragonAir Boeing 737，但空中的 CGI 飞机是加拿大航空公司 Jetz Boeing 737（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.youtube.com/watch?v=AgvzhJpyn10" href="https://www.youtube.com/watch?v=AgvzhJpyn10" rel="noopener nofollow noreferrer" target="_blank">
    在这里通过原始视频

  </p>
  <p class="graf graf--p graf-after--p" id="6cfa">
   一架飞机在遭遇台风之后以这种方式表演出令人难以置信的绝技，还能以某种方式奇迹般安全降落，肯定能吸引主流媒体的关注吧。唉，没有，因为视频明显是假的。

   <p class="graf graf--p graf-after--p" id="75f9">
    至于为什么真相如此重要？为什么如今人们普遍不关心真相？为什么这样的现实正是民主最大的杀器？详见这里的分析：《真相衰变》。
   </p>
   <p class="graf graf--p graf-after--p" id="cb1c">
    虽然谎言的历史几乎与人类的起源同龄，但当下，人们正面临着前所未有的欺诈世界：人工智能的进步已经开创一个假视频和音频的新时代。
   </p>
   <p class="graf graf--p graf-after--p" id="7554">
    GAN（生成性对抗网络）是一种用于执行无监督机器学习的 AI 。在 GAN 中，相反的神经网络协同工作以制造越来越逼真的音频、图像和视频内容。 GAN 中的一个神经网络充当箔片，推动其他网络产生更高保真度的结果。神经网络还可以更容易伪造音频，可以将源音频的元素转换为统计属性，并且可以重新排列这些属性以制作高超的假音频剪辑。
   </p>

    </p>
    <p class="graf graf--p graf-after--p" id="c20c">
     斯坦福大学的研究人员发表了早期结果，表明可以实时改变一个人预先录制的面部画面来模仿出另一个人的表情。
    </p>
    <p class="graf graf--p graf-after--p" id="922b">
     从本质上讲，想要冒充目标人的演员可以通过将脸部变成网络摄像头来创建数字人类木偶。 目标面部的数字再现将实时模仿演员的面部。
    </p>
    <p class="graf graf--p graf-after--p" id="6fda">
     目前，这种技术仍然需要数小时的预先录制的目标视频片段才能让它看起来真实。不幸的是，强大的公众人物经常可被追踪，有充足的历史录像带足够显示他们在现实生活中的谈话和移动，基础素材极为丰富。
    </p>
    <p class="graf graf--p graf-after--p" id="bc0a">

       <img alt="" class="size-full wp-image-4674" data-recalc-dims="1" height="573" sizes="(max-width: 1024px) 100vw, 1024px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-422.png?resize=1024%2C573&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-422.png?w=1024&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-422.png?resize=300%2C168&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-422.png?resize=768%2C430&amp;ssl=1 768w" width="1024"/>
      </noscript>
      <figcaption class="wp-caption-text" id="caption-attachment-4674">
       Face2Face, Stanford University
      </figcaption>
     </figure>
     <p>
      在威斯康星大学进行的奥巴马假视频实验是由三星、谷歌、Facebook、Intel 和该大学的动画研究实验室资助，研究人员使用了神经网络，首先从一个音频文件转换成声音的基本口形。然后系统将这些口型移植并混合到现有的目标视频上并调整时间以创建新的逼真的唇形同步视频。
     </p>
     <figure aria-describedby="caption-attachment-4675" class="wp-caption aligncenter" id="attachment_4675" style="width: 1024px">
      <img alt="" class="size-full wp-image-4675 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-423.png?resize=1024%2C1274&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-423.png?w=1024&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-423.png?resize=241%2C300&amp;ssl=1 241w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-423.png?resize=768%2C956&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-423.png?resize=823%2C1024&amp;ssl=1 823w" data-recalc-dims="1" height="1274" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-423.png?resize=1024%2C1274&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>
      <noscript>
       <img alt="" class="size-full wp-image-4675" data-recalc-dims="1" height="1274" sizes="(max-width: 1024px) 100vw, 1024px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-423.png?resize=1024%2C1274&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-423.png?w=1024&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-423.png?resize=241%2C300&amp;ssl=1 241w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-423.png?resize=768%2C956&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-423.png?resize=823%2C1024&amp;ssl=1 823w" width="1024"/>
      </noscript>
      <figcaption class="wp-caption-text" id="caption-attachment-4675">
       University of Washington’s sketch of the process that created the fake Obama video
      </figcaption>
     </figure>
     <p class="graf graf--p graf-after--figure" id="bc92">
      制作假音频也变得越来越容易和消费化。 例如，加拿大创业公司 Lyrebird 正在开发一种技术，可以从某人的声音中录制一分钟的音频，从而以相同的声音生成更长的音频片段。同样，在2016 年，Adobe 就推出了名为 Project VoCo 的原型（也被称为 “Photoshop for voice”）。该项目旨在让用户编辑人类语音，就像使用 PS 编辑数字图片一样简单……
     </p>
     <p class="graf graf--p graf-after--p" id="a76b">
      显然，人工智能对视频和音频干预的广泛采用意味着对“高价值目标”的声誉攻击也将相应增加。
      <strong class="markup--strong markup--p-strong">
       声誉攻击可以诋毁一个人的人格，使其不可信，甚至可以因此制造出一个逮捕的理由！
      </strong>
     </p>
     <p class="graf graf--p graf-after--p" id="8c23">
      这是危险的。但没人能挡得住技术的发展，于是，
      <span class="markup--quote markup--p-quote is-other" data-creator-ids="anon">
       <strong class="markup--strong markup--p-strong">
        更危险的在于，公民对技术的陌生
       </strong>
       ，无法使用技术反击、因无力鉴别而被欺骗。技术被掌握在恶意者的手中就是民主的灾难。
      </span>
     </p>
     <p class="graf graf--p graf-after--p" id="0e73">
      希望能有更多公民至少熟悉鉴别真伪的技术，这具有重大的意义。⚪️
     </p>
