  <p class="graf graf--p">
   如果您正在进行开源深度调查、或信息验证，很可能经常会遇到需要确定图像是否被篡改过的情况。
  </p>
  <p class="graf graf--p">
   篡改图像的动机有多种，比如：
  </p>
  <p class="graf graf--p">
   1、
   <strong class="markup--strong markup--p-strong">
    政治抹黑
   </strong>
   ，包括地缘政治冲突中的《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%be%9f%e8%b0%a3%e5%be%80%e5%be%80%e6%97%a0%e6%95%88%e9%99%a4%e9%9d%9e%e8%87%aa%e6%88%91%e6%89%b9%e9%a9%b3-%e8%af%86%e5%88%ab%e6%88%98%e7%95%a5%e6%80%a7%e6%ac%ba%e9%aa%97%e7%9a%84%e4%b8%bb/" href="https://www.iyouport.org/%e8%be%9f%e8%b0%a3%e5%be%80%e5%be%80%e6%97%a0%e6%95%88%e9%99%a4%e9%9d%9e%e8%87%aa%e6%88%91%e6%89%b9%e9%a9%b3-%e8%af%86%e5%88%ab%e6%88%98%e7%95%a5%e6%80%a7%e6%ac%ba%e9%aa%97%e7%9a%84%e4%b8%bb/" rel="noopener noreferrer" target="_blank">
    战略性欺骗
   </a>
   》；也包括本国内部当权者为镇压异议人士而采取的《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%ba%e6%a0%bc%e6%9a%97%e6%9d%80%e5%92%8c%e6%8a%b9%e9%bb%91%e8%bf%90%e5%8a%a8%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%8f%97%e7%9b%8a%ef%bc%9f/" href="https://www.iyouport.org/%e4%ba%ba%e6%a0%bc%e6%9a%97%e6%9d%80%e5%92%8c%e6%8a%b9%e9%bb%91%e8%bf%90%e5%8a%a8%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%8f%97%e7%9b%8a%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    人格暗杀
   </a>
   》。
  </p>
  <p class="graf graf--p">
   2、为符合所谓的
   <strong class="markup--strong markup--p-strong">
    宣传口径
   </strong>
   以“引导舆论”，而实施的《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%af%92%e9%9c%be%e2%80%8a-%e2%80%8a-%e7%9b%91%e6%8e%a7%e5%ae%a1%e6%9f%a5%e7%bd%91%e7%bb%9c%e7%9a%84%e5%85%a8%e7%90%83%e8%94%93%e5%bb%b6%ef%bc%9a%e4%b8%ad%e5%9b%bd%e5%92%8c%e4%bf%84%e7%bd%97/" href="https://www.iyouport.org/%e6%af%92%e9%9c%be%e2%80%8a-%e2%80%8a-%e7%9b%91%e6%8e%a7%e5%ae%a1%e6%9f%a5%e7%bd%91%e7%bb%9c%e7%9a%84%e5%85%a8%e7%90%83%e8%94%93%e5%bb%b6%ef%bc%9a%e4%b8%ad%e5%9b%bd%e5%92%8c%e4%bf%84%e7%bd%97/" rel="noopener noreferrer" target="_blank">
    战略信息散发
   </a>
   》；它的目的是《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%93%8d%e7%ba%b5%e5%bc%8f%e5%ae%a3%e4%bc%a0%ef%bc%9a%e4%ba%94%e4%b8%aa%e8%bf%87%e6%bb%a4%e5%99%a8%e5%92%8c%e8%af%86%e5%88%ab%e7%ba%bf%e7%b4%a2/" href="https://www.iyouport.org/%e6%93%8d%e7%ba%b5%e5%bc%8f%e5%ae%a3%e4%bc%a0%ef%bc%9a%e4%ba%94%e4%b8%aa%e8%bf%87%e6%bb%a4%e5%99%a8%e5%92%8c%e8%af%86%e5%88%ab%e7%ba%bf%e7%b4%a2/" rel="noopener noreferrer" target="_blank">
    操纵式宣传
   </a>
   》，以引发《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%9c%a8%e7%ba%bf%e5%ae%a3%e4%bc%a0%e6%88%98%e7%9a%84%e7%bb%93%e6%9e%9c/" href="https://www.iyouport.org/%e5%9c%a8%e7%ba%bf%e5%ae%a3%e4%bc%a0%e6%88%98%e7%9a%84%e7%bb%93%e6%9e%9c/" rel="noopener noreferrer" target="_blank">
    信息战
   </a>
   》。
  </p>
  <p class="graf graf--p">
   3、社交网络《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bc%95%e6%b5%81%e6%9a%b4%e6%94%bf%e5%92%8c%e7%94%9f%e5%ad%98%e5%9b%b0%e6%83%91/" href="https://www.iyouport.org/%e5%bc%95%e6%b5%81%e6%9a%b4%e6%94%bf%e5%92%8c%e7%94%9f%e5%ad%98%e5%9b%b0%e6%83%91/" rel="noopener noreferrer" target="_blank">
    引流暴政
   </a>
   》引发的
   <strong class="markup--strong markup--p-strong">
    虚假痴迷
   </strong>
   —— 虚假信息由于其精心策划以满足大众的情感预期，从而能获得远高于真相的流量，《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%81%87%e6%96%b0%e9%97%bb%e4%ba%a7%e4%b8%9a%e8%a7%a6%e7%9b%ae%e6%83%8a%e5%bf%83-%e4%ba%92%e8%81%94%e7%bd%91%e4%ba%ba%e7%9c%9f%e7%9a%84%e5%a6%82%e6%ad%a4%e5%a5%bd%e9%aa%97%ef%bc%9f/" href="https://www.iyouport.org/%e5%81%87%e6%96%b0%e9%97%bb%e4%ba%a7%e4%b8%9a%e8%a7%a6%e7%9b%ae%e6%83%8a%e5%bf%83-%e4%ba%92%e8%81%94%e7%bd%91%e4%ba%ba%e7%9c%9f%e7%9a%84%e5%a6%82%e6%ad%a4%e5%a5%bd%e9%aa%97%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    互联网人很难静下心来思考问题
   </a>
   》。
  </p>
  <p class="graf graf--p">
   当然还有其他可能性，但
   <strong class="markup--strong markup--p-strong">
    不论任何动机，它们的结果都是一样的
   </strong>
   ，即：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e8%99%9a%e5%81%87%e4%bf%a1%e6%81%af%e4%bb%a4%e7%8b%ac%e8%a3%81%e6%94%bf%e6%9d%83%e5%8f%98%e5%be%97%e6%9b%b4%e5%bc%ba%e5%a4%a7%ef%bc%8c%e4%bb%a4%e6%b0%91%e4%b8%bb%e5%9b%bd-2/" href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e8%99%9a%e5%81%87%e4%bf%a1%e6%81%af%e4%bb%a4%e7%8b%ac%e8%a3%81%e6%94%bf%e6%9d%83%e5%8f%98%e5%be%97%e6%9b%b4%e5%bc%ba%e5%a4%a7%ef%bc%8c%e4%bb%a4%e6%b0%91%e4%b8%bb%e5%9b%bd-2/" rel="noopener noreferrer" target="_blank">
    损伤民主最重要的根基
   </a>
   》。
  </p>
  <p class="graf graf--p">
   这就是为什么要呼吁所有支持民主的公民加入到反击战中来。
  </p>
  <p class="graf graf--p">
   要加入这场抵制虚假的战役，您必须拥有一些至少是基本的武器。
  </p>
  <p class="graf graf--p">
   本指南列出了一些最常用的工具，以及如何使用它们来分析数码照片中的篡改。这些工具是作为公民所必备的傍身武器。
  </p>
  <p class="graf graf--p">
   如果您还记得的话，我们曾经介绍过一些检验图像造假的知识，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%99%9a%e5%81%87%e4%bf%a1%e6%81%af%e9%89%b4%e5%88%ab%e5%b7%a5%e5%85%b7%e7%ae%b1%ef%bc%9a%e5%a6%82%e4%bd%95%e9%aa%8c%e8%af%81%e5%9b%be%e7%89%87%e5%92%8c%e8%a7%86%e9%a2%91%e7%9a%84%e7%9c%9f%e4%bc%aa/" href="https://www.iyouport.org/%e8%99%9a%e5%81%87%e4%bf%a1%e6%81%af%e9%89%b4%e5%88%ab%e5%b7%a5%e5%85%b7%e7%ae%b1%ef%bc%9a%e5%a6%82%e4%bd%95%e9%aa%8c%e8%af%81%e5%9b%be%e7%89%87%e5%92%8c%e8%a7%86%e9%a2%91%e7%9a%84%e7%9c%9f%e4%bc%aa/" rel="noopener noreferrer" target="_blank">
    虚假信息鉴别工具箱：如何验证图片和视频的真伪？
   </a>
   》。
  </p>
  <p class="graf graf--p">
   今天的指南将系统介绍相关知识 —— 最基本的知识，无需技术基础。
  </p>
  <p class="graf graf--p">
   只需要追求真相的一点点精神，虚假就很难盛行。

    1、反向图像搜索
   </strong>
  </p>
  <p class="graf graf--p">
   反向图像搜索是一种最傻瓜的和最易于使用的工具，每个人都会用，您可能想知道为什么本指南依然要包含它。
  </p>
  <p class="graf graf--p">
   这是因为，对于初学者来说刚刚接触信息验证时，你可能不知道如何执行正确的反向图像搜索。
  </p>
  <p class="graf graf--p">
   于是本指南将演示一个案例，介绍最容易理解的反向图像搜索思考方式。
  </p>
  <p class="graf graf--p">
   这里采取的案例是奥巴马的可疑照片。
  </p>
  <p class="graf graf--p">
   您要做的就是获取照片，然后转到 Google、Bing或 Yandex。
  </p>

   <strong class="markup--strong markup--p-strong">

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、EXIF/Metadata
   </strong>
  </p>
  <p class="graf graf--p">
   EXIF 代表可交换图像文件，它用于在使用 JPG 压缩的数码摄影中存储信息。
  </p>
  <p class="graf graf--p">
   EXIF 文件中包含的信息包括相机信息，例如光圈、闪光灯、ISO编号、曝光、快门速度、镜头、分辨率、白平衡和GPS坐标。
   <strong class="markup--strong markup--p-strong">
    这些数据可以成为任何开源情报调查的关键。
   </strong>
  </p>
  <p class="graf graf--p">
   也许您想知道相机的类型以便可以入侵它，或者您想要知道照片的GPS坐标以跟踪某人的位置……
  </p>
  <p class="graf graf--p">
   存在许多查看 EXIF 数据的工具，包括右键单击图像并转到详细信息。

  <p class="graf graf--p">
   有时，使用差异功能可以帮您捕获到通常很难被看到的微小修改。
  </p>
  <figure aria-describedby="caption-attachment-9268" class="wp-caption aligncenter" id="attachment_9268" style="width: 439px">
   <img alt="" class="size-full wp-image-9268 jetpack-lazy-image" data-lazy-sizes="(max-width: 439px) 100vw, 439px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/18-2.png?resize=439%2C356&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/18-2.png?w=439&amp;ssl=1 439w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/18-2.png?resize=300%2C243&amp;ssl=1 300w" data-recalc-dims="1" height="356" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/18-2.png?resize=439%2C356&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="439"/>
   <noscript>
    <img alt="" class="size-full wp-image-9268" data-recalc-dims="1" height="356" sizes="(max-width: 439px) 100vw, 439px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/18-2.png?resize=439%2C356&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/18-2.png?w=439&amp;ssl=1 439w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/18-2.png?resize=300%2C243&amp;ssl=1 300w" width="439"/>
   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-9268">
    Difference
   </figcaption>
  </figure>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、你的眼睛 —— 随身携带的识别工具
   </strong>
  </p>
  <p class="graf graf--p">
   这真的不是玩笑。
  </p>
  <p class="graf graf--p">
   有许多微妙的线索可以表明图像已被数字化篡改。
  </p>
  <p class="graf graf--p">
   有时候您无需使用任何特殊工具即可直接看到许多被篡改的痕迹。
  </p>
  <p class="graf graf--p">
   下图是一个著名的骗局图像：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9269 jetpack-lazy-image" data-lazy-sizes="(max-width: 1008px) 100vw, 1008px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/19.jpg?resize=1008%2C705&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/19.jpg?w=1008&amp;ssl=1 1008w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/19.jpg?resize=300%2C210&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/19.jpg?resize=768%2C537&amp;ssl=1 768w" data-recalc-dims="1" height="705" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/19.jpg?resize=1008%2C705&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1008"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9269" data-recalc-dims="1" height="705" sizes="(max-width: 1008px) 100vw, 1008px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/19.jpg?resize=1008%2C705&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/19.jpg?w=1008&amp;ssl=1 1008w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/19.jpg?resize=300%2C210&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/19.jpg?resize=768%2C537&amp;ssl=1 768w" width="1008"/>

  <p class="graf graf--p">
   只要稍微仔细观察就会发现一些非常明显的迹象表明该图像已被篡改。
  </p>
  <p class="graf graf--p">
   放大后可以清楚地看到，普京脸部的对比度比其他部分要粗糙得多，阴影也不正确。
  </p>
  <p class="graf graf--p">
   同样，缩小时，您可以看到他的身体轮廓与周围的轮廓明显不同。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9270 jetpack-lazy-image" data-lazy-sizes="(max-width: 752px) 100vw, 752px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/20-2.png?resize=752%2C495&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/20-2.png?w=752&amp;ssl=1 752w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/20-2.png?resize=300%2C197&amp;ssl=1 300w" data-recalc-dims="1" height="495" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/20-2.png?resize=752%2C495&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="752"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9270" data-recalc-dims="1" height="495" sizes="(max-width: 752px) 100vw, 752px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/20-2.png?resize=752%2C495&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/20-2.png?w=752&amp;ssl=1 752w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/20-2.png?resize=300%2C197&amp;ssl=1 300w" width="752"/>

  <p class="graf graf--p">
   关于
   <strong class="markup--strong markup--p-strong">
    对比度、纹理差异和不匹配的阴影作为识别线索
   </strong>
   ，另一个例子也非常著名：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9271 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/21.jpg?resize=1100%2C817&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/21.jpg?w=1297&amp;ssl=1 1297w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/21.jpg?resize=300%2C223&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/21.jpg?resize=768%2C570&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/21.jpg?resize=1024%2C760&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/21.jpg?resize=1100%2C817&amp;ssl=1 1100w" data-recalc-dims="1" height="817" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/21.jpg?resize=1100%2C817&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   如上图，您可以看到所有人的阴影如何与爱因斯坦身上的阴影方向不匹配。
  </p>
  <p class="graf graf--p">
   此外，爱因斯坦与其他人之间的对比度和纹理感也不同，并且头发的边缘看起来非常不自然，那是剪裁的结果。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    最后
   </strong>
  </p>
  <p class="graf graf--p">
   事实上有大量有用的工具可以帮助分析数码照片是否被篡改，以进行验证。
  </p>
  <p class="graf graf--p">
   本指南介绍的都是
   <strong class="markup--strong markup--p-strong">
    最最基本的方法
   </strong>
   ，只需要您花一两分钟的思考就可以识别造假。您可以挑选最合适您的任何工具。
  </p>
  <p class="graf graf--p">
   就个人而言，我经常将 Start.me 与
   <a class="markup--anchor markup--p-anchor" data-href="https://start.me/p/ZGAzN7/verification-toolset" href="https://start.me/p/ZGAzN7/verification-toolset" rel="noopener noreferrer" target="_blank">
    Verification Toolset
   </a>
   一起使用，后者提供了大量验证措施。
  </p>
  <p class="graf graf--p">
   请记得，作为拒绝被蒙骗的普通公民，您不一定需要精美的工具来分析数码照片，许多验证工具都是免费的，易于使用的，并且随时可用！
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    不要再被骗了，别让自己看起来很傻
   </strong>
   。那真的特别不可爱。⚪️
  </p>
