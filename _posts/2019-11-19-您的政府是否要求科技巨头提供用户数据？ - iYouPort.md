  <p class="graf graf--p graf--startsWithSingleQuote">
   ‘用户数据’，这是我们许多人曾经一直以来没有意识到的概念，直到最近法律的变化，使它成为全球人不能忽视的热门话题。
  </p>
  <p class="graf graf--p">
   新的数据保护法律迫使技术公司更加透明地展示他们使用您的数据的方式、以及他们如何与全球政府分享数据。
  </p>
  <p class="graf graf--p">
   虽然这不能解决问题，也无法阻止监视资本主义的继续，但至少通过一些报告能让更多人提起警惕。
  </p>
  <p class="graf graf--p">
   ⚠️请记住，透明度报告不是万能的，有可能并不够透明，而且《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%94%bf%e5%ba%9c%e7%9a%84%e5%8e%8b%e5%8a%9b%e5%b0%86%e9%80%8f%e6%98%8e%e5%ba%a6%e6%8a%a5%e5%91%8a%e4%bb%8e%e8%a8%80%e8%ae%ba%e8%87%aa%e7%94%b1%e7%9a%84%e5%ba%86%e5%85%b8%e5%8f%98%e6%88%90%e4%ba%86/" href="https://www.iyouport.org/%e6%94%bf%e5%ba%9c%e7%9a%84%e5%8e%8b%e5%8a%9b%e5%b0%86%e9%80%8f%e6%98%8e%e5%ba%a6%e6%8a%a5%e5%91%8a%e4%bb%8e%e8%a8%80%e8%ae%ba%e8%87%aa%e7%94%b1%e7%9a%84%e5%ba%86%e5%85%b8%e5%8f%98%e6%88%90%e4%ba%86/" rel="noopener noreferrer" target="_blank">
    政府的压力已经将透明度报告从言论自由的庆典变成了审查的庆典
   </a>
   》。
  </p>
  <p class="graf graf--p">
   并且，对于那些不是这方面专家的人来说，这些报告往往是难以理解的。因此本文创建了一系列可视化来解决这些问题。
  </p>
  <p class="graf graf--p">
   不论如何，有数字总比没有的好。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    谁发送了你的数据、发送到哪里？
   </strong>
  </p>
  <p class="graf graf--p">
   这些地图显示了科技巨头发送数据的位置。
  </p>
  <p class="graf graf--p">
   Facebook 名列前茅，获得了 138,317 个用户信息请求。涉及数据的 64,351 个实例被发送到美国政府，美国在2010年至2018年期间向 Facebook 提出了 75,208 个请求。
  </p>
  <p class="graf graf--p">
   相比之下，德国是数据的第二大请求者，在2010年至2018年间，Facebook 获得了来自德国的6,970个数据请求。
  </p>
  <p class="graf graf--p">
   但是，德国似乎对 Apple 的用户数据更感兴趣，在过去8年中
   <strong class="markup--strong markup--p-strong">
    请求并接收
   </strong>
   了22,080个用户数据实例。
  </p>
  <p class="graf graf--p">
   谷歌怎么样？好吧。他们向不同国家披露了 70,908 个数据实例，其中近一半（30,332个）进入美国政府。
  </p>

    惊人的全球性监视
   </strong>
   ，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" rel="noopener noreferrer" target="_blank">
    互联网霸主的时间线
   </a>
   》；也同时意味着
   <strong class="markup--strong markup--p-strong">
    无限规模的欺骗和操纵
   </strong>
   ，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" rel="noopener noreferrer" target="_blank">
    看不见的独裁
   </a>
   》。
  </p>
  <p class="graf graf--p">
   那么这点如何反映在政府对这些公司的数据请求数量上呢？
  </p>
  <p class="graf graf--p">
   该图表显示，与世界其他地区相比，美国提出了更多的要求。
  </p>
  <p class="graf graf--p">
   自2010年以来，所有请求中有32.17％至40.65％来自美国。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-7128 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/666.png?resize=1100%2C1100&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/666.png?w=1200&amp;ssl=1 1200w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/666.png?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/666.png?resize=300%2C300&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/666.png?resize=768%2C768&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/666.png?resize=1024%2C1024&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/666.png?resize=1100%2C1100&amp;ssl=1 1100w" data-recalc-dims="1" height="1100" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/666.png?resize=1100%2C1100&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-7128" data-recalc-dims="1" height="1100" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/666.png?resize=1100%2C1100&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/666.png?w=1200&amp;ssl=1 1200w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/666.png?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/666.png?resize=300%2C300&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/666.png?resize=768%2C768&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/666.png?resize=1024%2C1024&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/666.png?resize=1100%2C1100&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   互联网已经改变了所有人的生活方式，从买书到与朋友聊天，我们在网上做的所有事都会产生数据。
  </p>
  <p class="graf graf--p">
   必须承认，直到目前为止仍没有彻底曝光这些数据的所有使用方式。只能期待透明度革命继续前进。虽然已经曝光的部分已经足够惊人。
  </p>
  <p class="graf graf--p">
   想想看，您所在国家的政府使用这些数据做了什么？能做些什么？还有哪些可疑的使用方式尚未获得证据？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    方法
   </strong>
  </p>
  <p class="graf graf--p">
   为了创建这些可视化，首先需要生成一个寡头公司的列表，包括顶级技术巨头和最流行的社交媒体平台。
  </p>
  <p class="graf graf--p">
   然后，检查每家公司是否都有易于访问的透明度报告。
  </p>
  <p class="graf graf--p">
   ⚠️请注意，这仅仅来自于透明度报告，也就是这些寡头公司愿意提供出来的东西；而
   <strong class="markup--strong markup--p-strong">
    远非
   </strong>
   完整的真相（完整的只会更惊人）。
  </p>
  <p class="graf graf--p">
   也包含研究人员从2010年到2018年收集到的所选公司的透明度信息。
  </p>
  <p class="graf graf--p">
   不同公司会以不同的方式显示他们的数据，将事物分成不同的类别或使用不同的概念来描述相同类型的数据请求，因此必须解码这些信息，并确保它们都是标准化的。
  </p>
