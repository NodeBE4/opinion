   现在，我标记的某些网址可能无害，并且无法通过网络跟踪用户。我没有深入了解每个 API 或他们收集的数据，我为 Facebook 建立了以下查询：
  </p>
  <p class="XzvDs _208Ie _3_7DB blog-post-text-font blog-post-text-color _2p1aK _158eo _3_7DB">
   SELECT pages.rank, pages.url, requests.url, ext FROM httparchive.runs.latest_requests_mobile requests JOIN ( SELECT rank, pageid, url FROM httparchive.runs.latest_pages_mobile) pages ON pages.pageid = requests.pageid WHERE (requests.url CONTAINS “
   <a class="_2qJYG blog-link-hashtag-color y_1_u" href="https://facebook.com/tr" rel="noreferrer noopener" target="_top">
    facebook.com/tr
   </a>
   ”|| requests.url CONTAINS “
   <a class="_2qJYG blog-link-hashtag-color y_1_u" href="https://graph.facebook.com/" rel="noreferrer noopener" target="_top">
    graph.facebook.com
   </a>
   ”|| requests.url CONTAINS “
   <a class="_2qJYG blog-link-hashtag-color y_1_u" href="https://facebook.com/impression" rel="noreferrer noopener" target="_top">
    facebook.com/impression
   </a>
   ” || requests.url CONTAINS “
   <a class="_2qJYG blog-link-hashtag-color y_1_u" href="https://facebook.com/connect" rel="noreferrer noopener" target="_top">
    facebook.com/connect
   </a>
   ”|| requests.url CONTAINS “
   <a class="_2qJYG blog-link-hashtag-color y_1_u" href="https://connect.facebook.net/" rel="noreferrer noopener" target="_top">
    connect.facebook.net
   </a>
   ”|| requests.url CONTAINS “
   <a class="_2qJYG blog-link-hashtag-color y_1_u" href="https://connect.facebook.com/" rel="noreferrer noopener" target="_top">
    connect.facebook.com
   </a>
   ”|| requests.url CONTAINS “
   <a class="_2qJYG blog-link-hashtag-color y_1_u" href="https://facebook.com/brandlift" rel="noreferrer noopener" target="_top">
    facebook.com/brandlift
   </a>
   ” ) ORDER BY rank ASC
  </p>
  <p class="XzvDs _208Ie _3_7DB blog-post-text-font blog-post-text-color _2p1aK _158eo _3_7DB">
   所以，Facebook 跟踪了你多少？
  </p>
  <p class="XzvDs _208Ie _3_7DB blog-post-text-font blog-post-text-color _2p1aK _158eo _3_7DB">
   我在数据库中发现了超过 147.5k 站点的 805k Facebook 追踪请求。这意味着
   <strong>
    平均每页5个 Facebook 追踪者，约占整个互联网的 33％。
   </strong>
  </p>
  <p class="XzvDs _208Ie _3_7DB blog-post-text-font blog-post-text-color _2p1aK _158eo _3_7DB">
   有趣的是，按排名显示，排名前100位的网站使用远远低于 Facebook 的追踪数据（11％） – 可能是因为 Facebook 的许多竞争对手都位列前100名。
  </p>
