<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="fab8">
      七个多月的生日报连载，到了尾声。62篇文字，书写从反右到文革结束将近20年的历史。每个人的故事看似独立，实则互相勾连、印证，串起中国在毛泽东时代的执政逻辑。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="65c4">
      今年1月，刚接到生日报的编辑任务时，我期待却忐忑。期待从大家大国的主线里打捞出那些浮浮沉沉的个人故事。忐忑着虽尽全力不知能否交出满意答卷。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="32f2">
      “生日报”项目的所有作者，都曾上过钱钢老师的中国传媒史课程。对我，这堂课是一次全面补习，梳理建国后的传媒发展，政治是主线，也兼及经济。学生们印象深刻的是不同领导人治下“
      <a class="ck ho iz ja jb jc" href="http://mp.weixin.qq.com/s?__biz=MjM5OTY3MTY4MA==&amp;mid=201474193&amp;idx=1&amp;sn=9d878ad106e6b2e3ad450ffd303d923f&amp;scene=21#wechat_redirect" rel="noopener nofollow">
       五种力
      </a>
      ”的博弈图。五种力指的是“政治权力”、“市场经济”、“新闻媒体”、“资讯科技”和“公民社会”。
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="fcc7">
      接受这一结论是简单的，但如何从父母生日报的材料中求证？从几十篇报道中抽出值得深挖的信息，需要探测用语的变化、判断宣传力度的强弱，注意一些语词的出现和消失 — — 做到这一点也许不需要严谨的学术训练，却依赖于对历史的充分敏感，和细致入微的洞察力。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="2c31">
<strong class="id ju">
       找准节点，回到历史现场
      </strong>
</p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="6171">
      学会提问，是回到历史现场的第一步。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="5207">
      1960年的内蒙古报纸，为什么
      <a class="ck ho iz ja jb jc" href="http://mp.weixin.qq.com/s?__biz=MjM5OTY3MTY4MA==&amp;mid=407508656&amp;idx=1&amp;sn=a483c2cda9aa639382948a622e687d43&amp;scene=21#wechat_redirect" rel="noopener nofollow">
       把“红旗”写作“红旂”
      </a>
      ？来自内蒙古的女生注意到《昭乌达报》上奇特的繁简混合用法，得出地方报纸比中央报纸在内容和形式上都要“慢半拍”的结论；一只
      <a class="ck ho iz ja jb jc" href="http://mp.weixin.qq.com/s?__biz=MjM5OTY3MTY4MA==&amp;mid=407274711&amp;idx=1&amp;sn=c10c077cd36dd3a4f2c3bc45289ca1f7&amp;scene=21#wechat_redirect" rel="noopener nofollow">
       “童子牛”怀孕
      </a>
      ，为什么能登上1961年的《解放日报》？上海姑娘由这则“喜讯”生疑，读出大饥荒时连牛都营养不良，无法生育的讯息。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="5823">
      有些问题，同学们在浏览报纸时自然而然会提出；而有一些信息的重要性，则需钱老师提点才能意识到。比如，担任了55年人大代表、为“妇女解放”代言的
      <a class="ck ho iz ja jb jc" href="http://mp.weixin.qq.com/s?__biz=MjM5OTY3MTY4MA==&amp;mid=408154039&amp;idx=1&amp;sn=956ed20a45d166d89ef9977b867182e4&amp;scene=21#wechat_redirect" rel="noopener nofollow">
       申纪兰
      </a>
      ，为什么每逢政治运动都会在报纸上接受采访或者发表文章“表态”？结合她“从来不投反对票”的行为，可推演出什么样的逻辑？又比如，
      <a class="ck ho iz ja jb jc" href="http://mp.weixin.qq.com/s?__biz=MjM5OTY3MTY4MA==&amp;mid=2657330825&amp;idx=1&amp;sn=4c0f0254fb2e8aa3e3b2b76024984c49&amp;scene=21#wechat_redirect" rel="noopener nofollow">
       马来亚共产党
      </a>
      原本要就要放弃武装斗争，为什么领袖陈平访华后，路线来了个180度大转弯？
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="b12e">
      在钱老师的指引下，每个人都在生日报上找到了进入历史的独特节点。而通过资料检索，同学们梳理、核对基本史实，还历史以清晰的本来面貌。这些史料，包括《炎黄春秋》杂志文章、文革学者宋永毅主编的当代政治史资料库、《毛泽东文选》等等。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="60be">
<strong class="id ju">
       积累历史感
      </strong>
</p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="7021">
      仅仅回到历史现场还不够。面对历史材料时，我常感到自己“只见树木，不见森林”：只看到事件本身，却不知如何分析事件背后的逻辑链条。相信这也是很多同学的困惑。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="f33a">
      尤其当遇到一些离我们遥远而陌生的词语。教育革命、四清运动、斗私批修、批林批孔……这些文革中掀起一阵阵巨浪的运动，究竟目的何在？它们的轮番击打，对国家和个人的命运造成了什么样的影响？而运动频发的事实背后，是什么样的执政思路？
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="30f1">
      要了解这些，除了理清语词本身的小脉络，更需摸透家事国事背后的大脉络。以1960年代中苏交恶为例，自赫鲁晓夫在苏共二十大上推倒斯大林的神像后，中苏关系就起了微妙变化。
      <a class="ck ho iz ja jb jc" href="http://mp.weixin.qq.com/s?__biz=MjM5OTY3MTY4MA==&amp;mid=408529158&amp;idx=1&amp;sn=1b1fb0547dce65c4486777cd377d2ccb&amp;scene=21#wechat_redirect" rel="noopener nofollow">
       浙江同学发现
      </a>
      ，苏联专家于1959年停止向中国提供原子弹样品和技术资料，中国自此只能靠一己之力，把原子弹试验推上日程；
      <a class="ck ho iz ja jb jc" href="http://mp.weixin.qq.com/s?__biz=MjM5OTY3MTY4MA==&amp;mid=2657330592&amp;idx=1&amp;sn=51dc44ebbf53a0862d524d69367ef696&amp;scene=21#wechat_redirect" rel="noopener nofollow">
       来自钢城武汉的妹子
      </a>
      则注意到，1953年就开始援助武钢的苏联，突然在1964年拒绝了中国工会的“五一”邀请，由此牵出武钢与苏联的一场“爱恨情仇”。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="6b77">
      当争论逐渐公开化，中苏在明面上撕破脸时，来自古城开封的男生发现1963年的《人民日报》上整整三版文章，竟然全文刊载苏联对中国的攻击文字，正自疑惑中共的报刊为何如此信心满满；另一位北京同学就看到了第二日
      <a class="ck ho iz ja jb jc" href="http://mp.weixin.qq.com/s?__biz=MjM5OTY3MTY4MA==&amp;mid=2657330370&amp;idx=1&amp;sn=d3d3f1ace18bdf63e2f4121cf8dff13c&amp;scene=21#wechat_redirect" rel="noopener nofollow">
       中共报纸发动的大规模反击
      </a>
      ：《9评苏共中央公开信》的第一篇正式亮相。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="3d1d">
      外交布局也因此改变。湖南的女生惊讶于
      <a class="ck ho iz ja jb jc" href="http://mp.weixin.qq.com/s?__biz=MjM5OTY3MTY4MA==&amp;mid=2657330600&amp;idx=1&amp;sn=f57d987f4837d8d5c5b7ca52e2306828&amp;scene=21#wechat_redirect" rel="noopener nofollow">
       六十年代中国对阿尔巴尼亚的援助
      </a>
      异乎寻常地慷慨。原来，阿尔巴尼亚对待赫鲁晓夫的态度和中国一致，曾在1960年的一场社会主义国家会议上公开支持中国，和苏联唱反调；贵州同学发现母亲生日那天原定举行的
      <a class="ck ho iz ja jb jc" href="http://mp.weixin.qq.com/s?__biz=MjM5OTY3MTY4MA==&amp;mid=2657330465&amp;idx=1&amp;sn=253fc94d21cfad82dd62c9d33c4068f7&amp;scene=21#wechat_redirect" rel="noopener nofollow">
       “第二次亚非会议”无限延期
      </a>
      ，看到了和“老大哥”闹翻后，中国外交面临的四面楚歌和举步维艰。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="f8f6">
      看似毫不相关的角度，却能归因于一个主题。历史感的积累，来源于这样一步步的探索。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="af43">
<strong class="id ju">
       回归家庭故事
      </strong>
</p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="2df1">
      真正赋予生日报生动色彩的，是每个人的家庭故事。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="19be">
      有一位来自内蒙古的同学，在爸爸生日前几天的报纸上看到关于“爱国卫生运动”的报道，和奶奶一聊才知道，爸爸作为早产儿诞生，正是因为奶奶在这场运动中打扫卫生时，从小凳子上摔了下来。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="923c">
      另一位来自河北邢台的同学，则发现自己的奶奶之所以精神失常，是因为1963年的邢台水灾后，遭遇了以“反资产阶级复辟”为旗号的抢粮。而这一切，要追溯到
      <a class="ck ho iz ja jb jc" href="http://mp.weixin.qq.com/s?__biz=MjM5OTY3MTY4MA==&amp;mid=407878052&amp;idx=1&amp;sn=c00967b636c1544c45df5b854c7c42d9&amp;scene=21#wechat_redirect" rel="noopener nofollow">
       八届十中全会
      </a>
      提出的“千万不要忘记阶级斗争”。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="f5af">
      还有一位同学来自香港，他从旧报上读到往来港澳的
      <a class="ck ho iz ja jb jc" href="http://mp.weixin.qq.com/s?__biz=MjM5OTY3MTY4MA==&amp;mid=2657330921&amp;idx=1&amp;sn=5adf1856c0eb17f5b212bb5642bfa2bd&amp;scene=21#wechat_redirect" rel="noopener nofollow">
       “佛山轮”沉没的消息
      </a>
      ，由此展开探索，想知道这艘他爷爷曾经工作并随之沉没的轮船，有着什么样的故事。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="614e">
      因为生日报，同学们与家中老人展开未曾料想过的对话。这些话，如果我们不去问起，也许只能停留在他们记忆中，随时光消散。我们用录音笔、微信语音记下这一场场回到过去的时光旅行，也记下与长辈相处的珍贵日子。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="66a1">
      也因为这一个个故事，我们终于感到自己和50多年前的这段历史有了不可分割的血脉联系。我们的父辈诞生于此。我们虽然不是这段历史的见证者，却有幸成为它的记录者，甚至讲述者。我们仿佛进入一条历史暗河，用自己的一点微芒照亮水面。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="5d27">
      1958年到1976年，这段历史离我们不再遥远。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="942e">
      1985年，邓小平会见津巴布韦总理穆加贝时，用不到300个字总结了这段历史。他说，“一九五七年开始有一点问题了，问题出在一个‘左’字上。”文化大革命则更是“走到了左的极端，极左思潮泛滥”。直到今天，这话仍是黄钟大吕，时时警醒后人。
     </p>
