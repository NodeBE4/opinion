  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     摘要
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   中国和俄罗斯信息控制的科技和技术在全球扩散已经成为主要国际报纸的显著标题。
  </p>
  <p class="graf graf--p">
   然而，对这种扩散的驱动力以及结果却很少有系统性的分析。
  </p>
  <p class="graf graf--p">
   本报告针对这一主题的研究发现，俄罗斯和中国的信息控制正在更有效地扩散至非常多的国家 —— 尤其是那些与中俄有关系的国家。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    中国的信息控制较易于扩散到其“一带一 路”沿线的国家；俄罗斯之控制散布于独联体各国。
   </strong>
  </p>
  <p class="graf graf--p">
   在研究得出此一结论过程中，本报告将首先定义出俄罗斯和中国的信息控制模式，然后追踪这两国的信息控制在其各自的「科技圈」(technosphere) 所及共 110 个国家中的扩散 —— 这指的既是地理区域，也是中国与俄罗斯信息控制科技、信息处理技术以及法律扩散所及的影响范围。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     前言
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   北京和莫斯科的审查监控技术 (例如，出口审查设备、监视城市等)、和技术(例如 由其他国家模仿的中／俄监视法律)，已走向全球，其影响既能扩及到加勒比地区仅有 8 万或 10 万居民的小岛国家，如安提瓜和巴布达，也能到达人口超过 10 亿的南亚国家（如 印度）。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    来自巴哈马、莱索托和秘鲁的记者参加了在北京的宣传培训，中国的监视装备已经用在巴西东部的军事指挥部，以及约旦的议会大厦
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%80%81%e5%a4%a7%e5%93%a5%e7%9a%84%e8%81%94%e7%9b%9f/" href="https://www.iyouport.org/%e8%80%81%e5%a4%a7%e5%93%a5%e7%9a%84%e8%81%94%e7%9b%9f/" rel="noopener noreferrer" target="_blank">
     老大哥的联盟
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    俄罗斯的监视设备部署在其邻国
   </strong>
   ，如白俄罗斯、哈萨克斯坦、和乌克兰，也部署在遥远的阿尔及利亚、古巴、墨西哥、和巴勒斯坦。
  </p>
  <p class="graf graf--p">
   研究此类重大扩散之时，
   <strong class="markup--strong markup--p-strong">
    很重要的一点
   </strong>
   就是了解：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     为什么是俄罗斯的监视审查、而非中国的监视审查，能在一些国家扩散，而非别的国家？
    </strong>
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     是什么促使这种控制能在不同地区扩散？
    </strong>
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     如果在一个国家中扩散的是某一套监视审查技术，而非另一套，它们之间的差别是什么？
    </strong>
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     这种监视审查的国际繁衍如何使这两国受益匪浅？
    </strong>
   </li>
  </ul>

  <p class="graf graf--p">
   为了回答上述问题，本报告首先建立了*信息控制类型学*。
  </p>
  <p class="graf graf--p">
   第二节则引用信息控制类型，检视俄罗斯和中国信息控制的做法。
  </p>
  <p class="graf graf--p">
   第三节揭示如何测量扩散，第四节说明扩散的原因。
  </p>
  <p class="graf graf--p">
   第五节追踪和分析中国与俄罗斯的此类监视审查科技，包括技术与法律的模仿以及培训，在国际上如何扩散。
  </p>
  <p class="graf graf--p">
   第六节明确指出中国和俄罗斯因监控审查技术出口而获得的政治、经济和情报优势，以及这种扩散对进口国的影响。
  </p>
  <h3>
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--li-strong">
     1、信息控制类型学
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   对信息控制的定义，学者间意见殊异。
  </p>
  <p class="graf graf--p">
   一些学者指出，信息控制的概念为“在网络空间内且通过网络空间进行的行动，以寻求阻绝、中断、操纵和塑造信息和通信，达到战略和政治目的。”
  </p>
  <p class="graf graf--p">
   另外一些学者
   <strong class="markup--strong markup--p-strong">
    将信息控制分类为恐惧(即，自我审查)、摩擦 (类似审查)、和泛滥 (近似战络信息散发)
   </strong>
   。
  </p>
  <p class="graf graf--p">
   有别于这些学说，本文认定信息控制的核心概念为“监视”(surveillance)，并分析
   <strong class="markup--strong markup--p-strong">
    如何借助监视影响管理信息
   </strong>
   的其他形式。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    信息控制可以有不同形式：包括监视、审查 (censorship)、自我审
    <br/>
    查 (self-censorship)、和战略信息散发 (strategic information dissemination)。
   </strong>
  </p>
  <p class="graf graf--p">
   当虑及一个国家使用各种各样信息控制形式的程度，并评估其控制信息的选项(工具)数量，即可呈现形形色色的信息控制模式。
  </p>
  <h3 class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e4%b8%8e%e6%93%8d%e7%ba%b5/" href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e4%b8%8e%e6%93%8d%e7%ba%b5/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     —— 监视 (surveillance)
    </strong>
   </a>
  </h3>
  <p class="graf graf--p">
   要对信息控制有全盘概念，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e4%b8%8e%e6%93%8d%e7%ba%b5/" href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e4%b8%8e%e6%93%8d%e7%ba%b5/" rel="noopener noreferrer" target="_blank">
    监视
   </a>
   至关紧要。
  </p>
  <p class="graf graf--p">
   ⚠️有了监视，才可以进而做到审查、自我审查、和战略信息散发。
  </p>
  <p class="graf graf--p">
   没有监视就没有威吓，而政府也不知道要阻绝哪些网站，当局也无从知晓哪些网络对话需要接受所谓的 “政治指导”。
  </p>
  <p class="graf graf--p">
   ⚠️监视可被观察出来，也可能没被观察到。
  </p>
  <p class="graf graf--p">
   当一般民众查觉有监视时，就会引起
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%ae%a1%e6%9f%a5%e5%92%8c%e8%87%aa%e6%88%91%e5%ae%a1%e6%9f%a5/" href="https://www.iyouport.org/category/%e5%ae%a1%e6%9f%a5%e5%92%8c%e8%87%aa%e6%88%91%e5%ae%a1%e6%9f%a5/" rel="noopener noreferrer" target="_blank">
    自我审查
   </a>
   。
  </p>
  <p class="graf graf--p">
   很明显的，只需展现出有监视、可以不必实际进行监视，就能获得监视之效果。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    一个政府完全可能安装假的监视相机，就足以吓到市民。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️即使监视不被民众查觉，它仍然产生影响。当监视未被察觉时，依旧可以用来阻挡特定内容 —— 审查，或指向特定目标 —— 散发战略信息。
  </p>
  <ul class="postList">

     被监视噤声的作家：大数据时代的自我审查
    </a>

   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    虽然恐惧感可能是推动自我审查的主要因素之一，但并非每一次自我审查的行为都伴有恐惧感
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️当一个人面对着可能有潜在敌意的说话对象时，有很多原因会使这个
   <br/>
   人保留自己的意见，比如可能是想避免口角之争，或担心冒犯了别人或伤了别人的心，或担心受到报复，如丢掉工作、或有被打的危险，或担心被当成异类。
  </p>
  <p class="graf graf--p">
   ⚠️这就是为什么在互联网上中文群组中几乎看不到有效的对话，原因之一 —— 任何中文群组，包括匿名群组。另一个原因则是，社会长期缺乏民主辩论实践以及社区协作的经验。
  </p>
  <p class="graf graf--p">
   有鉴于此，
   <strong class="markup--strong markup--p-strong">
    本文仅定义“自我审查”为：“以为听我说话的对方会跟自己的真实意见相左时，就保留自己的意见。”
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️一个政府手中可任意运用的监视工具越多，就越容易驱使民众走向自我约束。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    ——
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%ae%a1%e6%9f%a5%e5%92%8c%e8%87%aa%e6%88%91%e5%ae%a1%e6%9f%a5/" href="https://www.iyouport.org/category/%e5%ae%a1%e6%9f%a5%e5%92%8c%e8%87%aa%e6%88%91%e5%ae%a1%e6%9f%a5/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     审查 (censorship)
    </strong>
   </a>
  </h3>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">

   就本报告的讨论范围，虚假信息的例子之一就是，俄罗斯假造“新闻”称：艾滋病是美国政府实验的结果，并大肆传播。
  </p>

  </p>
  <p class="graf graf--p">
   ⚠️一个国家对其人民知道的越多，操纵人民的选项就越多。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     使用程度 ——
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   形形色色信息控制的使用程度经常取决于许多具体情境因素。
  </p>
  <p class="graf graf--p">
   各国在某种程度上都受限于能掌握的工具。
  </p>
  <p class="graf graf--p">
   例如，比起中国，俄罗斯想实施严格网上审查制度的选项就相对较少，因为其互联网科技的监控基础设施较不完备。
  </p>
  <p class="graf graf--p">
   因此，俄罗斯主要是借助
   <strong class="markup--strong markup--p-strong">
    威吓人民
   </strong>
   (例如 监禁反对派候选人)，以及透过宣传与虚假消息散发来
   <strong class="markup--strong markup--p-strong">
    操纵人民
   </strong>
   。
  </p>
  <p class="graf graf--p">
   俄罗斯仰赖这些技术，可能是因为这些做法不需要太复杂的科技。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     —— 选项 ——
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   例如，使用间谍软件渗透手机的某一国家可能借此读取某些信息，用以恐吓反对派人士。
  </p>
  <ul class="postList">
   <li class="graf graf--li">

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e2%9a%a0%ef%b8%8f-%e5%a4%a7%e5%9e%8b%e7%bd%91%e7%bb%9c%e9%92%93%e9%b1%bc%e6%94%bb%e5%87%bb%e6%b4%bb%e5%8a%a8%e4%b8%93%e9%97%a8%e9%92%88%e5%af%b9%e6%b3%a8%e9%87%8d%e9%9a%90%e7%a7%81%e7%9a%84%e7%94%a8/" href="https://www.iyouport.org/%e2%9a%a0%ef%b8%8f-%e5%a4%a7%e5%9e%8b%e7%bd%91%e7%bb%9c%e9%92%93%e9%b1%bc%e6%94%bb%e5%87%bb%e6%b4%bb%e5%8a%a8%e4%b8%93%e9%97%a8%e9%92%88%e5%af%b9%e6%b3%a8%e9%87%8d%e9%9a%90%e7%a7%81%e7%9a%84%e7%94%a8/" rel="noopener noreferrer" target="_blank">

  <p class="graf graf--p">
   然而，一个国家如果没有深度包检测工具(用来网络过滤与监视)，即无
   <br/>
   法取得其他信息，因此就限制了这个国家锁定其目标异议人士的能力。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    更高的监视能力并不一定能转成更有效的信息控制，但当局如果拥有复杂的监控技术，即能有多样的工具箱，以选择执行信息控制。
   </strong>
  </p>
  <p class="graf graf--p">
   因此，将监视纳入信息控制类型，可对不同国家的信息控制做出更扎实的比较。
  </p>
  <p class="graf graf--p">
   例如，中国拥有比俄罗斯更高的监视能力，因此有更多选项来实施信息控制，这将在下一节深入讨论。
  </p>
  <p class="graf graf--p">
   因此，⚠️监视工具的数量和多样性是理解中国和俄罗斯信息控制模式各自特色的一个主要因素。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    监视、自我审查、审查、和战略信息散发，共同构成了信息控制的机制与形式。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    一个国家依赖每种机制的强度，结合它实施每种机制的选项数目，界定了该国的信息控制模式。
   </strong>
  </p>
  <p class="graf graf--p">
   下节将介绍，俄罗斯仰赖遍布式的监视、自我审查、和战略信息散发，以维系国内稳定，而中国以广泛的监视为基础，大规模使用审查和战略信息散发。
  </p>
  <p class="graf graf--p">
   中国与俄罗斯两种模式有重迭之处。中国在某种程度上与俄罗斯一样，以法律规定网络平台实名制，以及把 VPN 拥有者关进监狱，进而驱使自我审查。
  </p>

  <p class="graf graf--p">
   俄罗斯也与中国一样，制定禁用网站黑名单，实施审查制度。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     2. 中国与俄罗斯信息控制模型
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在比较俄罗斯和中国的信息控制模式时，一种有效的方式就是检视他们依赖各种机制到何种程度，以及各自能支配哪些工具，以实施控制。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     俄罗斯的做法 ——
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    —— 监视
   </strong>
  </p>
  <p class="graf graf--p">
   俄罗斯 1995 年的“行动调查法”(Law on Operational Investigation)为实现广泛的监视框架建立了法源，该法允许“俄罗斯联邦安全局”(Federal Security Service of the Russian Federation，FSB) 实施网络监控“行动调查活动系统”(System for Operative Investigative Activities)，简称为 SORM。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%99%ae%e4%ba%ac%e7%9a%84%e7%a7%98%e5%af%86%e6%9c%8d%e5%8a%a1%ef%bc%9a%e5%85%8b%e9%87%8c%e5%a7%86%e6%9e%97%e5%ae%ab%e5%a6%82%e4%bd%95%e6%8e%a7%e5%88%b6-fsb-%e4%bf%84%e7%bd%97/" href="https://www.iyouport.org/%e6%99%ae%e4%ba%ac%e7%9a%84%e7%a7%98%e5%af%86%e6%9c%8d%e5%8a%a1%ef%bc%9a%e5%85%8b%e9%87%8c%e5%a7%86%e6%9e%97%e5%ae%ab%e5%a6%82%e4%bd%95%e6%8e%a7%e5%88%b6-fsb-%e4%bf%84%e7%bd%97/" rel="noopener noreferrer" target="_blank">

    </a>

   </li>
  </ul>
  <p class="graf graf--p">
   SORM-1 专门用在拦截电话与手机通话。为因应互联网普及率的飙升，就成立了 SORM-2，监视互联网流量。
  </p>
  <p class="graf graf--p">
   之后，SORM-3 提升与增加了当时监视系统的功能，如监看社交媒体和 Wi-Fi。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    —— 自我审查
   </strong>
  </p>
  <p class="graf graf--p">
   在 2000 年早期，新的“俄罗斯联邦的信息安全准则” (Information Security
   <br/>
   Doctrine of the Russian Federation) 出台，将信息安全界定为国家安全事务。
  </p>
  <p class="graf graf--p">
   随后，多项限制跟进，如 2014 年的“博客法” (blogger’s law)与“反加密法”(antiencryption law)，及 2016 年的一项立法，规定数据存储需
   <strong class="markup--strong markup--p-strong">
    本地化
   </strong>
   存于俄罗斯境内。
  </p>
  <p class="graf graf--p">
   这些法律足以展示俄罗斯对其人民遍布式的监视，并促使自我审查。
  </p>
  <p class="graf graf--p">
   同时，俄罗斯当局威吓艺术家、新闻记者、科技主管、和反对派代表人士。反对派的 Boris Nemtsov 被杀，政治行动者 Alexei Navalny 多次入狱，而 VK 电报信使 (VK and Telegram Messenger) 的创办人 Pavel Durov 在政府持续高压下，只得流亡。
  </p>
  <p class="graf graf--p">
   更微细的恐吓措施也产生了深远的影响。
  </p>
  <p class="graf graf--p">
   俄罗斯国家主导的全国性的公示，不断强调互联网的危险，经研究发现，这也导致自我审查增高。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    —— 战略信息散发
   </strong>
  </p>
  <p class="graf graf--p">
   在俄罗斯，宣传与虚假信息的重点是发送有利于政权的信息。
  </p>
  <p class="graf graf--p">
   学者 Gallagher 与 Fredheim 指出，“互联网研究机构”(Internet Research Agency) 是个俄罗斯编造厂，
   <strong class="markup--strong markup--p-strong">
    对国内，就是统一言论口径，在国外，就是制造分裂话语。
   </strong>
  </p>
  <p class="graf graf--p">
   从 Twitter 上发给国内民众的消息就是强调应该支持俄罗斯介入乌克兰和叙利亚，并夸大西方阵营的分歧。
  </p>
  <p class="graf graf--p">
   俄罗斯编造厂散发讯息时，有时也会走官方媒体频道，如 Channel
   <br/>
   One、Russia One、NTV、Komsomolskaya Pravda、与 Izvestia。这些动作有助于扩大克里姆林宫战略信息的散发范围。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “互联网研究机构”付费的博主，和另一个宣传组织 Nashi 青年运动的付费博主，主要是由秘密雇用的民众组成。其中有些人全职工作，以 12 小时为一班上班。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    —— 审查制度
   </strong>
  </p>
  <p class="graf graf--p">
   俄罗斯政府一直都较不重视审查制度。跟中国不同的是，俄罗斯一直未能建立可以替代外国媒体平台的本国平台。
  </p>
  <p class="graf graf--p">
   而俄罗斯的科技公司，包括本国人经营的 app 应用程序商店、搜索引擎、或社交媒体平台等，都未能有足够的国内市占率，可以排除或取代外国竞争者。
  </p>
  <p class="graf graf--p">
   此外，俄罗斯的网络过滤采用了一个全国性阻挡清单(blocklist)，并使用深度包检测工具等其他先进技术，维持高度监视。
  </p>
  <p class="graf graf--p">
   但是，俄罗斯过滤的遍布程度不如中国。
  </p>
  <p class="graf graf--p">
   例如，Alexei Navalny 经常抨击政府，但他的 YouTube 频道仍在营运。
  </p>
  <p class="graf graf--p">
   最近，俄罗斯进一步严打网络环境，试图封锁 Telegram Messenger，但是政府封锁方法太粗糙，导致俄罗斯大片互联网停摆。
  </p>
  <p class="graf graf--p">
   所以，Telegram Messenger 仍然在应用程序商店可以取得，政府如果想封锁，技术设备远远不足。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     中国的做法 ——
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    —— 监视
   </strong>
  </p>
  <p class="graf graf--p">
   与俄罗斯相比，中国在监视民众时，有更多的选项，因为监视装置已经遍布现今的中国社会每一个角落。
  </p>
  <p class="graf graf--p">
   举例而言，在新疆，当地人受到无所不在的监看，监视系统结合了侵入式应用程序、面部识别摄像头、和其他技术。
  </p>
  <p class="graf graf--p">
   监视设备已经布建全面、深入社会。这在中国的教育体系内也处处可见，学生在教室上课，由教室中的面部识别系统监看；学生到哪里都可透过手环追踪，上面记录他们的地理位置。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    在美国在此也是如此，见《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%9b%91%e7%8b%b1%e5%bc%8f%e6%95%99%e5%ad%a6/" href="https://www.iyouport.org/%e7%9b%91%e7%8b%b1%e5%bc%8f%e6%95%99%e5%ad%a6/" rel="noopener noreferrer" target="_blank">

    </a>

   </li>
  </ul>
  <p class="graf graf--p">
   虽然这类监看是由学校执行的，但这是
   <strong class="markup--strong markup--p-strong">
    政府的“智能教育”计划
   </strong>
   所支持布建的。
  </p>
  <p class="graf graf--p">
   政府也有各种方案，鼓励
   <strong class="markup--strong markup--p-strong">
    监视大军中的民间参与者
   </strong>
   （朝阳群众）推动民众进行自我审查，这些民间参与者就在政府的监视努力中扮演了补强的角色。
  </p>
  <ul class="postList">
   <li class="graf graf--li">

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    —— 审查
   </strong>
  </p>
  <p class="graf graf--p">
   在中国，最明显的审查事例就是其防火墙 (Great Firewall)，它禁止中国人使用 Google、Facebook、与 Twitter等服务。
  </p>
  <p class="graf graf--p">
   在中国境内，政府内部和私营公司有数以千计的审查员，在大流量信息中筛选出新的关键词设定禁用。
  </p>
  <p class="graf graf--p">
   其中大部分的审查仍然是人工操作，但有一部分筛选工作已经是自动化过滤，如使用已知关键词阻挡 (known-keyword blocking)。
  </p>
  <p class="graf graf--p">
   学者 Ray Bradbury 在《华氏 451》(Fahrenheit 451)一书中提出的大哉问又可再次提出，
   <strong class="markup--strong markup--p-strong">
    “在你烧掉的所有书中，你曾经读过其中任何一本吗？”
   </strong>
  </p>
  <p class="graf graf--p">
   换言之，审查员真的读了他们审查掉的内容吗？
  </p>
  <p class="graf graf--p">
   在未来，他们可能连读都不必了，因为大部分过滤都可由人工智能自动化处理。
  </p>
  <p class="graf graf--p">
   最近，中国甚至对那些试图连接外国网站的精英与热衷科技的公民加大审查。
  </p>
  <p class="graf graf--p">
   还有一些法规出台，使政府不批准的 VPN 在 app 商店被下架，导致在中国 Apple App Store 与腾讯 App Store 这两个主要应用程序商店可以取得的 VPN 极其有限。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    —— 战略信息散发
   </strong>
  </p>
  <p class="graf graf--p">
   中国与俄罗斯一样，战略信息散发的重要角色之一就是为政府策略性的把敏感政治议题转移焦点。
  </p>
  <p class="graf graf--p">
   在中国，这种操作的目的是不加入争议性的话题，而是
   <strong class="markup--strong markup--p-strong">
    把争议的焦点从争议移转到亲政府的贴文
   </strong>
   。
  </p>
  <p class="graf graf--p">
   早在 2004 年，中国当局就已经雇佣大批人力用来摇摆网络上的民意。他们认知到，只做到决定哪些信息可以提供是不够的，形塑仍留在网络上的信息同样重要。
  </p>
  <p class="graf graf--p">
   为此，当局聘用了数以千计的互联网评论员，专门做贴文，俗称为五毛党。据估计，网络上这样的佣兵现在高达 200 万名，每年制造约 4.5 亿个贴文。
  </p>

   <strong class="markup--strong markup--p-strong">
    无薪转发
   </strong>

  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    —— 自我审查
   </strong>
  </p>
  <p class="graf graf--p">
   中国在毛泽东时代，以恐吓逼使自我审查是一种非常普遍的做法。之后，这种做法转为以审查和战略信息散发为主的信息控制方式。
  </p>
  <p class="graf graf--p">
   目前，政府针对精英阶层培养自我审查，而用审查和战略信息散发对付一般民众(这种策略控制普通民众最为有效)。
  </p>
  <p class="graf graf--p">
   诱使自我审查措施的一个例子是逮捕提供 VPN 服务的人(作为对其他 VPN 供货商的警告)，或者强制规定社交媒体上有影响力者必须实名登记。
  </p>
  <p class="graf graf--p">
   除此之外，为鼓励自我审查，中国的互联网法律往往故意规范模糊，在该罚与不罚间制造模棱两可的空间。

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     3. 研究方法：测量扩散
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   针对中国与俄罗斯信息控制科技与技术的扩散，本报告的研究方法是
   <strong class="markup--strong markup--p-strong">
    开源追踪法
   </strong>
   ，分析从公司报告、技术网络测量、报纸或期刊文章、以及政府法规中收集的数据资料。
  </p>
  <p class="graf graf--p">
   本研究以三个指标来衡量扩散：
   <strong class="markup--strong markup--p-strong">
    科技、模仿、与培
   </strong>
   训。
  </p>
  <p class="graf graf--p">
   用这三个指标的目的是衡量中国与俄罗斯对其他国家运用的战术分布所及的广度和深度。
  </p>
  <p class="graf graf--p">
   广度指的是有多少国家暴露在至少一个指标之下。
  </p>
  <p class="graf graf--p">
   深度指的是单一国家采用了几个指标。
  </p>
  <p class="graf graf--p">
   如果一个国家采用了一项扩散指标，就定位在“科技圈”的外缘。
  </p>
  <p class="graf graf--p">
   如果一个国家被标上最多指标 (俄罗斯的两个指标，中国的三个指标)，则在这一国的扩散就被视为深广，该国就落在“科技圈”的中心。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    指标 1：科技
   </strong>
  </h3>
  <p class="graf graf--p">
   扩散的第一个指标是科技。
  </p>
  <p class="graf graf--p">
   就本文的研究目的，这里的焦点放在中国与俄罗斯的
   <strong class="markup--strong markup--p-strong">
    监视和审查科技出口到海外实体
   </strong>
   ，这些外国实体可控制大量的人口或公共建筑，可以假设是由警察或其他政府主管单位经营。
  </p>
  <p class="graf graf--p">
   此类科技的实例包括安装在监狱、港口、城市、火车站、机场中的摄像机，以及由海外互联网服务供货商部署的过滤设备。
  </p>
  <p class="graf graf--p">
   此一指标尚
   <strong class="markup--strong markup--p-strong">
    不包括监视系统出口给非警察或非政府单位的情况
   </strong>
   ，如海康威视出口给日本一间公立大学的闭路电视监控设备。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    大型图文报告《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%9b%be%e8%a7%a3%e4%b8%ad%e5%9b%bd%e7%a7%91%e6%8a%80%e5%b7%a8%e5%a4%b4-%e5%8d%8a%e5%a3%81%e9%ab%98%e5%a0%a1%e5%a5%87%e4%ba%ba/" href="https://www.iyouport.org/%e5%9b%be%e8%a7%a3%e4%b8%ad%e5%9b%bd%e7%a7%91%e6%8a%80%e5%b7%a8%e5%a4%b4-%e5%8d%8a%e5%a3%81%e9%ab%98%e5%a0%a1%e5%a5%87%e4%ba%ba/" rel="noopener noreferrer" target="_blank">
     图解中国科技巨头 — 半壁“高堡奇人”
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   除了阅览公司声明和媒体报导进行追踪监视出口项目外，还有透过网络测量，辨识出一组特定的监视中间盒。
  </p>
  <p class="graf graf--p">
   我们使用 OONI Explorer 和 Censys，来寻找有“V2R2C00-IAE/1.0” 指纹的华为中间盒。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这个指纹或标头出现在华为的 eSight 产品，这个对象能够分析网络流量以及发现 IpSec VPN 服务 — 这就可以用来监控和审查流量。
   </strong>
  </p>
  <p class="graf graf--p">
   在检视了 OONI Explorer 和 Censys 的数据后，显示出华为中间盒已经用在哥伦比亚、古巴、意大利、墨西哥、尼日利亚、巴基斯坦、西班牙和土耳其。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    指标 2：模仿
   </strong>
  </h3>
  <p class="graf graf--p">
   就本文的研究目的，模仿的定义为：另一个国家复制中国或俄罗斯的信息控制法律和技术。
  </p>
  <p class="graf graf--p">
   例子之一是乌兹别克斯坦仿效俄罗斯的国家监视法；而坦桑尼亚和津巴布韦政府官员宣布，他们打算模仿中国用本土的内容提供商取代外国网络内容提供商，以强化控制。
  </p>
  <p class="graf graf--p">
   本文收集的数据显示，⚠️如果没有同时配合科技和培训的扩散，模仿几乎不会单独发生，但科技和培训扩散指标可以单独存在。
  </p>
  <p class="graf graf--p">
   因此，模仿显示为深度扩散。很显然，所有模仿俄罗斯或中国的国家即处于其各自“科技圈”的核心。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    指标 3：培训
   </strong>
  </h3>
  <p class="graf graf--p">
   此一指标包含
   <strong class="markup--strong markup--p-strong">
    两类培训：
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    一、对执法人员、政府官员、以及私营公司(经常为政府实施信息控制的公司)的培训，
   </li>
   <li class="graf graf--li">
    二、对新闻记者的培训。
   </li>
  </ul>
  <p class="graf graf--p">
   对第一类对象，就是教他们如何扩大搜取有关民众的信息 (数字取证、安全城市项目)，或如何更有效地实施审查。
  </p>
  <p class="graf graf--p">
   所有所谓的智能城市的落实或销售深度封包检测技术都几乎一定配有相
   <br/>
   关技术操作培训。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%99%ba%e8%83%bd%e5%9f%8e%e5%b8%82%e7%a9%b6%e7%ab%9f%e6%98%af%e4%b8%aa%e4%bb%80%e4%b9%88%e9%ac%bc%ef%bc%9f%ef%bc%81/" href="https://www.iyouport.org/%e6%99%ba%e8%83%bd%e5%9f%8e%e5%b8%82%e7%a9%b6%e7%ab%9f%e6%98%af%e4%b8%aa%e4%bb%80%e4%b9%88%e9%ac%bc%ef%bc%9f%ef%bc%81/" rel="noopener noreferrer" target="_blank">

  <p class="graf graf--p">
   本文仅纳入有书面文件明文提到向特定国家提供培训的案例。
  </p>
  <p class="graf graf--p">
   举例而言，一宗只报导了华为在西班牙建设一个监控城市的新闻，即不纳入此一培训指标。
  </p>
  <p class="graf graf--p">
   而第二类，即：
   <strong class="markup--strong markup--p-strong">
    在中国培训海外记者，可能各国政府都没有注意到，因为其中一些培训是由私人机构或新闻机构举办的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   此类培训旨在增加海外对中国的正面报导。许多案例显示，
   <strong class="markup--strong markup--p-strong">
    这种培训达到了预期的政治效果
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “培训”一词易被误解。在西方世界，这个字的意思是：将某人提升到一定标准
   <br/>
   的熟练度，很可能是某种学术水平。然而，在中国，媒体培训实际上是公共关系目的的免费到中国参访，行程安排都是亲政府议程。……
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种对媒体的安排，实为中国软实力战略的核心
   </strong>
   。
  </p>
  <p class="graf graf--p">
   因此，中国计划在未来几年内“培训”数百名拉丁美洲记者，最佳的解读应该是，
   <strong class="markup--strong markup--p-strong">
    让有影响力的意见领袖暴露在北京宣传之下的一种做法
   </strong>
   。
  </p>
  <p class="graf graf--p">
   此外，记者培训通常包含亲身体验信息控制方法的实际运作方式。
  </p>
  <p class="graf graf--p">
   由广西的共产党人事部门运营的广西 “百色干部学院” (Baise Leadership Academy) 就是一个旗舰范例，对来自东南亚的官员和记者的培训，就是教他们如何在网上“引导民意”。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     4. 信息控制扩散的原因
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   本节将提供信息控制扩散的实证数据，并分析为什么相较之下
   <strong class="markup--strong markup--p-strong">
    某些国家更易于
   </strong>
   模仿、购买技术、或接受培训。
  </p>
  <p class="graf graf--p">
   根据附录 A 中的实证数据，本文主张，一国的政府体制以及国与国之间的相互关联度可以解释，为什么信息控制扩散在某些情况中较易发生。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    解释变量 1: 政权体制类型
   </strong>
  </h3>
  <p class="graf graf--p">
   在追踪扩散指标时，政权类型是*模仿*的一个主要变量，但对科技与培训的解释力较弱。
  </p>
  <p class="graf graf--p">
   本文中政权类型的定义是依据经济学人信息部 (Economist Intelligence
   <br/>
   Unit) 的民主指数 (Democracy Index)，分成“民主制” (democracy)、“混合制政权” (hybrid regime)、和“威权式政权” (authoritarian regime)。
  </p>
  <p class="graf graf--p">
   本研究收集的数据显示，威权式政权和混合制更适于模仿俄罗斯和中国。在模仿中国或俄罗斯的 19个国家中(包括俄罗斯与中国相互模仿)，58% 是威权式政权，37% 是混合制， 5%是民主制。
  </p>
  <p class="graf graf--p">
   为什么混合制和威权式政权更可能模仿中国和俄罗斯？理由之一或许是独裁国家间倾向更愿意相互学习，因为彼此都面临着类似的生存威胁。
  </p>
  <p class="graf graf--p">
   这种倾向使独裁者分享各种科技与政策。早在 1848 年，不同政体的独裁主政者就相互讨论和彼此分享如何对付在欧洲的革命趋势。
  </p>
  <p class="graf graf--p">
   异曲同工，在 2010 代初期的“阿拉伯之春”盛行之后，独裁者就参与了“精英学习”，让他们更了解如何塑造更充分的政策。
  </p>
  <p class="graf graf--p">
   有些状况下，独裁者之间的互助十分微细，如交换知识，但有时是明目张
   <br/>
   胆、无所忌讳的。
  </p>
  <p class="graf graf--p">
   在 2011 年有一例，就是沙特阿拉伯积极帮助巴林镇压异议人士。
  </p>

   <strong class="markup--strong markup--p-strong">

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    解释变量 2: 互联度 (interconnectedness)
   </strong>
  </h3>
  <p class="graf graf--p">
   政治实体之间的双边经济、政治、历史和社会的相互关联可以解释为什么中国和俄罗斯的信息控制更易于扩散到某些国家，而非其他国家。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    中俄与另一个国家之间的相互联系程度越强，信息控制传到对方的可能性就越大。
   </strong>
  </p>
  <p class="graf graf--p">
   此一发现有别于但是也类似于学者 Levitsky 和 Way 所提出的“联系”(linkage)概念。
  </p>
  <p class="graf graf--p">
   两位学者研究在后冷战后期，西方国家与某些威权倾向政权间的关系，检视关系中所谓“联系”的概念，因而提出民主措施扩散的一项假设：扩散是建立于国与国之间关系密度的基础之上。
  </p>
  <p class="graf graf--p">
   他们发现，当“高度联系”存在，民主化即更有可能会发生。
  </p>
  <p class="graf graf--p">
   学者 Ambrosio 与 Weyland 也思考了“联系”和地理邻近是与威权扩散情境的两项因素。本文的论点以这些学者的研究为基础，但在关键领域，提出相当差异的论点。
  </p>
  <p class="graf graf--p">
   例如，Levitsky 和 Way 主张，国与国间建立紧密关系的最重要因素是地理邻近性。
  </p>
  <p class="graf graf--p">
   然而，中国与国外的互联性却延伸的愈来愈远 —— 可能部分原因在于该国的经济崛起。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    海外融资是一个可以说明的实例。
   </strong>
  </p>
  <p class="graf graf--p">
   从中国对不同地区的官方融资列表可以显示，非洲位居首位，其次是中欧和东欧 (包括俄罗斯)，以及拉丁美洲。
  </p>
  <p class="graf graf--p">
   这些地区之后，名单上才出现南亚、东南亚、中亚和东北亚。本文提出信息控制扩散的数据也描述了同一趋势。
  </p>
  <p class="graf graf--p">
   虽然俄罗斯的信息控制已经深入扩散到独联体中的数国，但俄罗斯信息控制所扩散到的其他许多国家并非其邻国。
  </p>
  <p class="graf graf--p">
   中国的信息控制在全世界扩散既深且广，因此削弱了“地理邻近是建立相互联系，因而促使推动信息控制扩散的最强因素”此一假设。
  </p>
  <ul class="postList">
   <li class="graf graf--li">

  </p>
  <p class="graf graf--p">

   </strong>

  </p>
  <p class="graf graf--p">
   最后，扩散的主要原因不太可能是杠杆作用，亦即，透过制裁、军事威胁、或撤出外援这种外部压力。相反，
   <strong class="markup--strong markup--p-strong">

   </strong>
   才是更为合理的假设。
  </p>
  <p class="graf graf--p">
   因此，本文提出了“互联度”一词，以阐述信息控制国际扩散背后的原因。

   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     互联度的替代 ——
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在本文中，“一带一路倡议” (Belt and Road Initiative, BRI)和独联体 (CIS) 也当成显示与中国和俄罗斯有高度互联度的另一种量测替代项。
  </p>
  <p class="graf graf--p">
   就这个倡议或组织，参与其一或都参与的国家更有可能使用中国或俄罗斯信息控制。
  </p>
  <p class="graf graf--p">
   之所以如此，是因为一方面，“一带一路”和独联体基本上是中国和俄罗斯间实质经济、外交、技术官僚、社会、信息和民间社会关系的替代项，以及另一方面，这是其他参与国家之间这些关系的替代项。
  </p>
  <p class="graf graf--p">
   许多因素促成了这种对照。
  </p>
  <p class="graf graf--p">
   第一、这两个倡议都可以上溯到历史记载，中俄在古早时期与其他国家曾有紧密的互联度。“一带一路”奠基在古代丝路的复兴。另一方面，独联体植基于前苏联的传统，力图庚续旧张。
  </p>
  <p class="graf graf--p">
   其次，对这两项计划，俄罗斯和中国都积极投入。俄罗斯经常拉拢东欧和中亚，视其为势力范围之内，而中国鼓励“一带一路”沿线各国交换思想、物品、和社会互动(这些互动都在双边谅解备忘录中明文推崇)。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在“一带一路”框架中，中国还把记者和媒体培训编了进去，科技和服务的销售也一并纳入
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在一方面，中国企业力图寻找新的出口市场与利润。另一方面，对外贸易在一定程度上是配合政府的协调。

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     5. 北京和莫斯科的核心科技圈
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   以下重点说明“互联度”的强度如何导致深广的扩散，并且形成了核心科技圈。
  </p>
  <p class="graf graf--p">
   与俄罗斯和中国互联度较低的国家，亦即那些不属于“一带一路”或独联体的国家，就较不可能受到信息控制的广泛扩散，故处于俄罗斯和中国科技圈的外缘。
  </p>
  <h3 class="graf graf--p">
   —— 独联体国家
  </h3>
  <p class="graf graf--p">
   独联体是 1991 年 12 月苏联解体后成立的，包括亚美尼亚、阿塞拜疆、白俄罗斯、哈萨克斯坦、吉尔吉斯斯坦、摩尔多瓦、塔吉克斯坦、和乌兹别克斯坦，其目的为鼓励俄罗斯与这些新成立的国家继续在政治、经济、和文化层面合作。乌克兰最初是其中一员，但在 2018 年退出。
  </p>
  <p class="graf graf--p">
   本研究发现，
   <strong class="markup--strong markup--p-strong">
    俄罗斯的信息控制扩散到 28 个国家
   </strong>
   ，这些国家的地理位置相差很远，如巴勒斯坦和墨西哥天各一方。
  </p>
  <p class="graf graf--p">
   独联体的国家占俄罗斯信息控制出口的25%，即在 28 个国家中占 7 国。阿塞拜疆、白俄罗斯、哈萨克斯坦、吉尔吉斯斯坦、乌克兰、和乌兹别克斯坦位于俄罗斯科技圈的核心。
  </p>
  <p class="graf graf--p">
   这其中每一个国家，不论曾经是或现在仍是独联体国家，
   <strong class="markup--strong markup--p-strong">
    都已经模仿了
   </strong>
   俄罗斯的网络监视法律。
  </p>
  <p class="graf graf--p">
   这些独联体国家的许多监视法律都带有一种
   <strong class="markup--strong markup--p-strong">
    可疑的氛围，制造了威吓异议份子的空间
   </strong>
   ，此即复制了俄罗斯信息控制的做法。
  </p>
  <p class="graf graf--p">
   然而，如果没有俄罗斯提供的监视技术，仅仅是这些法律与恐惧的环境就没有太大的威力。
  </p>
  <p class="graf graf--p">

  </p>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    —— 重迭的科技圈
   </strong>
  </h3>
  <p class="graf graf--p">
   中国科技圈和俄罗斯科技圈在总共 20 个国家发生重迭。有趣的是，深处于俄斯科技圈的每一个国家，也部分处于中国的科技圈内。
  </p>
  <p class="graf graf--p">
   中国在白俄罗斯、哈萨克斯坦、吉尔吉斯斯坦、乌克兰、和乌兹别克斯坦都很活跃。
  </p>
  <p class="graf graf--p">
   另一方面，俄罗斯在其邻国以外，只出现在中国科技圈核心诸国中的一国；在泰国，有一家专门进行声音取证的俄罗斯公司 Speech Technology Center 一直在提供设备。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     6. 扩散的影响
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   本文讨论扩散的主要结果是创建了“科技圈”，这是一个地理区，却给信息控制出口国取得了各种先占优势。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这些优势可分成政治、经济、和情报相关的优势
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在政治上
   </strong>
   ，信息控制的增生强化了专制政权。例如，俄罗斯信息安全观的广泛采用，有助于俄国政府对内巩固其合法性，因为政府可以宣传其模式受到
   <br/>
   国外模仿的此一事实。
  </p>
  <p class="graf graf--p">
   俄罗斯把信息控制系统扩散海外，俨然奠定其规范制定者的形象。
  </p>
  <p class="graf graf--p">
   中国对外国政府官员和记者提供培训，与海外领导人物打好关系，让他们了解中国所认知威权信息环境应如何运作。
  </p>
  <p class="graf graf--p">
   可能会有人质疑，培训某一国中的一些政府官员或记者会有影响吗，但是，在全球层次，培训数以千计的人可能会带来大幅的结构变化，进而使各国只有更依赖中国的设备和技术。
  </p>
  <p class="graf graf--p">
   ⚠️美亚柏科的信息安全学院已经培训了 1000 多名海外执法人员进行数字取证，而中国公共外交协会计划到 2020 年每年培训 1,500 名外国记者。
  </p>

   换言之，
   <strong class="markup--strong markup--p-strong">
    中国对津巴布韦出口的提升，使该国有了更多的信息控制选项
   </strong>
   。
  </p>
  <p class="graf graf--p">
   同时，信息控制的使用可能随扩散而改变。
  </p>
  <p class="graf graf--p">
   津巴布韦可能原本只选择仰赖审查和战略信息散发，但与中国的互动肯定使此一非洲国家更能全面采用此一做法。
  </p>
  <p class="graf graf--p">
   津巴布韦与中国信息管理方式逐渐融合，在前者明言要模仿后者之时，已经不言可喻。
  </p>
  <figure aria-describedby="caption-attachment-9246" class="wp-caption aligncenter" id="attachment_9246" style="width: 1208px">
   <img alt="" class="size-full wp-image-9246 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/12-4.png?resize=1100%2C1313&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/12-4.png?w=1208&amp;ssl=1 1208w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/12-4.png?resize=251%2C300&amp;ssl=1 251w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/12-4.png?resize=768%2C917&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/12-4.png?resize=858%2C1024&amp;ssl=1 858w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/12-4.png?resize=1100%2C1313&amp;ssl=1 1100w" data-recalc-dims="1" height="1313" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/12-4.png?resize=1100%2C1313&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   <figcaption class="wp-caption-text" id="caption-attachment-9246">
    扩散对津巴布韦的影响。全面深入研究津巴布韦信息控制超出了本文的
    <br/>
    范围。因此，上述描述中国信息控制在该国扩散之前与之后的四图是假设的。
   </figcaption>
  </figure>
  <p class="graf graf--p">
   然而，在扩散过程中，该国向中国模式趋同并非假设，而是基于本文收集的证据。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     结论
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   本文综观大约十三年以来中国和俄罗斯的信息控制出口。
  </p>
  <p class="graf graf--p">
   在此期间，扩散的科技在本质上发生了巨变。
  </p>
  <p class="graf graf--p">
   最初，大多是粗糙的审查和监视科技往国际扩散。这包括了中国装置，出口到津巴布韦干扰无线电广播，以及俄罗斯提供给前苏联国家的过滤和语音取证设备。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    逐渐，这些出口展现多样化，早期的初级品变得精良
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️现在我们正在研究扩散的是具面部识别技术的闭路电视摄像机、形形色
   <br/>
   色的数字搜证工具、智能国家身份证、政府智能数据库和所谓的智能城市。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在这个瞬息万变的环境中，重要的是追踪俄罗斯和中国正在从事何种开发和部署，以便预测国外明天会发生何事。
   </strong>
  </p>
  <p class="graf graf--p">
   当然，伊朗、泰国、和委内瑞拉等国家都在开发各自的技术和科技，然而，了解俄罗斯和中国信息控制在各国醒目的存在，是辨识它们扩散全球趋势的重要研究起点。
  </p>
  <p class="graf graf--p">
   那么，面对这些趋势，该如何因应？不幸的是，没有简单的答案 — 部
   <br/>
   分原因是难以阻止军民两用信息控制科技和技术的散布 ——
   <strong class="markup--strong markup--p-strong">
    更，民主国家自己也热衷购买信息控制设备。
   </strong>
  </p>
  <p class="graf graf--p">
   当培训被描述成善意提供，而非战略信息扩散技术的分享渠道时，即不能受到无的放矢。
  </p>
  <p class="graf graf--p">
   尽管如此，民主国家仍然可以采取行动。在国内，民主国家不必部署所有具高度监视功能的工具。
  </p>
  <p class="graf graf--p">
   马萨诸塞州正在考虑立法，限制面部识别的使用。旧金山已经通过了此一措施。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     不过需要注意的是《
    </strong>
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%a6%81%e6%ad%a2%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e5%8f%af%e4%bb%a5%e5%81%9a%e5%88%b0%e4%bb%80%e4%b9%88%e3%80%81%e4%b8%8d%e8%83%bd%e5%81%9a%e5%88%b0%e4%bb%80%e4%b9%88/" href="https://www.iyouport.org/%e7%a6%81%e6%ad%a2%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e5%8f%af%e4%bb%a5%e5%81%9a%e5%88%b0%e4%bb%80%e4%b9%88%e3%80%81%e4%b8%8d%e8%83%bd%e5%81%9a%e5%88%b0%e4%bb%80%e4%b9%88/" rel="noopener noreferrer" target="_blank">
     <strong class="markup--strong markup--li-strong">
      禁止面部识别可以做到什么、不能做到什么
     </strong>
    </a>
    <strong class="markup--strong markup--li-strong">
     》
    </strong>
   </li>
  </ul>
  <p class="graf graf--p">
   民主国家还应努力使个人装置和通信渠道更能抵抗过滤或监视设备。然而，现状是正如其反。
  </p>
  <p class="graf graf--p">
   最近，澳大利亚通过了一项法律，扩张监视，侵害了点对点的加密信息，而这种做法比较在威权政权中常见。
  </p>
  <p class="graf graf--p">
   加拿大、新西兰、英国、和美国打算让主管当局能够进入存取点。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    在这里看到《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e5%8f%af%e8%83%bd%e7%a6%81%e6%ad%a2%e6%97%a0%e6%b3%95%e8%a2%ab%e7%a0%b4%e8%a7%a3%e7%9a%84%e7%ab%af%e5%88%b0%e7%ab%af%e5%8a%a0%e5%af%86%e9%80%9a%e4%bf%a1/" href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e5%8f%af%e8%83%bd%e7%a6%81%e6%ad%a2%e6%97%a0%e6%b3%95%e8%a2%ab%e7%a0%b4%e8%a7%a3%e7%9a%84%e7%ab%af%e5%88%b0%e7%ab%af%e5%8a%a0%e5%af%86%e9%80%9a%e4%bf%a1/" rel="noopener noreferrer" target="_blank">

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%bf%99%e4%ba%9b%e4%bd%bf%e7%94%a8%e6%9a%97%e7%bd%91%e7%9a%84%e4%ba%ba%e6%98%af%e5%a6%82%e4%bd%95%e8%a2%ab%e6%8a%93%e5%88%b0%e7%9a%84%ef%bc%9f/" href="https://www.iyouport.org/%e8%bf%99%e4%ba%9b%e4%bd%bf%e7%94%a8%e6%9a%97%e7%bd%91%e7%9a%84%e4%ba%ba%e6%98%af%e5%a6%82%e4%bd%95%e8%a2%ab%e6%8a%93%e5%88%b0%e7%9a%84%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     这些使用暗网的人是如何被抓到的？
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bb%96%e4%bb%ac%e6%98%af%e5%a6%82%e4%bd%95%e6%8a%93%e6%8d%95%e8%af%b4%e7%9c%9f%e8%af%9d%e7%9a%84%e4%ba%ba%ef%bc%9f%e2%9a%a0%ef%b8%8f%e7%81%be%e9%9a%be%e5%92%8c%e6%95%99%e8%ae%ad/" href="https://www.iyouport.org/%e4%bb%96%e4%bb%ac%e6%98%af%e5%a6%82%e4%bd%95%e6%8a%93%e6%8d%95%e8%af%b4%e7%9c%9f%e8%af%9d%e7%9a%84%e4%ba%ba%ef%bc%9f%e2%9a%a0%ef%b8%8f%e7%81%be%e9%9a%be%e5%92%8c%e6%95%99%e8%ae%ad/" rel="noopener noreferrer" target="_blank">
     他们是如何抓捕说真话的人？⚠️灾难和教训
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   最后，无庸置疑，在民主国家，公权力使用监视装置时，
   <strong class="markup--strong markup--p-strong">
    应
   </strong>
   遵循透明、由公众问责的程序。例如，应订定制衡机制，使只能让当地警察接触的监视设备，不被国家情报机构取得。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    现在的状况是，完全相反。
   </strong>
  </p>
  <p class="graf graf--p">
   虽然这些建议本身都不能阻止监控技术在全球散布，但是全面的防范可以降低未来可能的滥用。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    全球言论表达自由和人权状况逐渐恶化。迄今为止，超过一百多个国家已经购入与模仿俄罗斯和中国的信息控制，并且接受过这两国的培训
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    希望本报告能帮助更多人了解这一局面 —— 它之所以如此糟糕来自当权者之间的合谋，不论是哪种政治体制。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️我们必须让各国公民了解这些知识，让更多人加入到反抗中来 —— 因为作为被监视对象，我们除了彼此之外，别无依靠。⚪️
  </p>
