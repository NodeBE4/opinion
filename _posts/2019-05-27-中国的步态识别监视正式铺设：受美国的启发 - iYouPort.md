  <p class="graf graf--p">
   你走路的姿势可以告诉老大哥很多事。
  </p>
  <p class="graf graf--p">
   中国人工智能初创公司 Watrix 表示，它的软件可以识别50米开外的人 — 即使他们已经遮住了脸、或者背对着摄像机走路。
  </p>
  <p class="graf graf--p">
   被称为步态识别的监视技术通过分析关于人的步行的数千个度量，从身体轮廓到手臂运动的角度到内八字或外八字姿势，然后建立数据库来工作。
  </p>
  <p class="graf graf--p">
   Watrix 的联合创始人兼首席执行官 Huang Yongzhen 在北京接受采访时说，他们的技术不需要脸部数据。他说像这样的特征让 Watrix 在监视识别方面具有特别的优势，
   <strong class="markup--strong markup--p-strong">
    因为人们无法避开这样的监视
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Huang 说，
   <strong class="markup--strong markup--p-strong">
    北京、上海和重庆街头的警察已经开始进行步态识别技术试验了
   </strong>
   ，该公司3月正式推出了2.0版本，支持分析大城市级别的实时摄像头。
  </p>
  <p class="graf graf--p">
   这个 Huang 还说，“我们
   <strong class="markup--strong markup--p-strong">
    目前正与警方合作
   </strong>
   进行刑事调查，例如在抢劫现场跟踪嫌疑人，目前，中国有大约30万名通缉犯，[我们的软件]数据库包括那些有先前步态记录的人。”
  </p>
  <p class="graf graf--p">
   Watrix 是臭名昭著的 SenseTime 和 Megvii Inc 等多家中国老大哥初创企业之一，推出了人工智能的创新应用，人工智能被称为第四次工业革命，预计将为工业和社会带来变革。北京正在将面部识别推向人们生活的许多角落，从揭露 jaywalkers 到拯救卫生纸，从发现票贩子到启用无密码付款。
  </p>
  <p class="graf graf--p">
   随着与美国的贸易和技术竞争加剧，一些学者认为，在相对宽松的隐私保护法的帮助下，中国能够产生和挖掘庞大的数据库，这将使其在人工智能方面具有优势。
  </p>
  <p class="graf graf--p">
   2017年7月，中国国务院公布了该国的“下一代人工智能发展计划” — 到2030年人工智能占主导地位的三步路线图。
  </p>
  <p class="graf graf--p">
   美国也不甘示弱，特朗普3月发布了一项行政命令，以维持“美国在人工智能方面的领导地位”，旨在优先投资该领域的研发。牛津大学2018年的一份报告发现，除了“获取数据”之外，中国在人工智能增长的三个核心领域落后于美国：硬件研究和算法开发以及行业的商业化。
  </p>
  <p class="graf graf--p">
   Huang 说，
   <strong class="markup--strong markup--p-strong">
    虽然步态技术背后的研究是在中国科学院的孵化下进行的，但它首先受到了美国国防高级研究计划局（DARPA）在2000年代开展的相关研究的启发
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “DARPA 就像是技术领域的风向标。无论它搞什么样的项目都能为国家利益服务”，Huang 说。 Huang 是前CAS研究员，于2016年离开学术界，共同创立了 Watrix。
  </p>
  <p class="graf graf--p">
   他还说，如今，这家总部位于北京的公司已成为世界上步态识别的第一名。 “我们在实验室水平的准确率超过96％，而 Watrix 在中国和美国拥有50多项专利。”
  </p>
  <p class="graf graf--p">
   据称这种技术针对最具创造力的监视目标进行测试。
   <strong class="markup--strong markup--p-strong">
    伪装跛行是无法欺骗它的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Huang 说，自从 Watrix 去年10月推出其软件的第一个版本以来，他已经收到了数十个有关该技术的询问。该公司目前正在与来自新加坡、印度、俄罗斯、荷兰和捷克共和国的安全公司就合同进行谈判，其中步态识别可以与面部扫描一起用于实时或离线视频监控。
  </p>
  <p class="graf graf--p">
   Watrix 还提供面部识别技术，并结合这两项技术可以提高整体效果。
  </p>
  <p class="graf graf--p">
   除了监视之外，步态识别还可用于所谓的智能家居，养老院，以便在老年人摔倒或即将跌倒时向工作人员发出警报，在医院诊断神经问题，以及进行理疗和运动训练。
  </p>
  <p class="graf graf--p">
   该公司于10月份完成了一轮1亿元人民币（2000万新元）的A轮融资，并计划在最新一轮融资中筹集2亿至3亿元人民币，以加快该技术的研究和业务采用。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    C
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.asiaone.com/china/chinese-police-test-cutting-edge-gait-recognition-technology-identifies-people-based-how-they" href="https://www.asiaone.com/china/chinese-police-test-cutting-edge-gait-recognition-technology-identifies-people-based-how-they" rel="noopener noreferrer" target="_blank">
    hinese police test cutting-edge gait recognition technology that identifies people based on how they walk
   </a>
   . Although the research behind gait technology was incubated under the Chinese Academy of Sciences it was first inspired by a US Defence Advanced Research Projects Agency (DARPA) study initiated in the 2000s, said Huang.
  </p>
