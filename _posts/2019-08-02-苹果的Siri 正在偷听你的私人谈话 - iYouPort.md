  <p class="graf graf--p">
   最近的新闻报道证实了许多苹果用户可能已经在怀疑的东西：苹果公司的语音助手Siri有能力记录私人谈话。
  </p>
  <p class="graf graf--p">
   这些音频片段并不总是存储在一个服务器上 —— 许多样本都是传递给第三方人力承包商的，他们付钱让人力监听这些音频。
  </p>
  <p class="graf graf--p">
   这并不像语音助手监听其用户那么简单：报告称，苹果公司的承包商将这些片段作为公司质量控制措施的一部分而进行监听，确定Siri是否被意外触发或故意触发，以及是否回应完全正确。
  </p>
  <p class="graf graf--p">
   但是这种做法在 Apple 面向客户的隐私文档中并不明确，并且，”拉链的声音”都有可能触发 Siri 错误地激活。
  </p>
  <p class="graf graf--p">
   承包商最终能听到很多敏感的私人谈话，包括：业务会议、性取向、私人医疗预约、和各种私密的交易。
  </p>
  <p class="graf graf--p">
   从某种意义上说，这个消息并不令人震惊；很明显，任何使用Siri或任何语音助手的用户都必须允许这些设备听取并分析他们的声音。
  </p>
  <p class="graf graf--p">
   语音助理一直在录制和窃听用户这不是新闻，那么新闻是什么？
  </p>
  <p class="graf graf--p">
   这里有一个更微妙的事实值得考虑：人工智能的“语音助手”，被称为高效和轻松使用的家伙，甚至未能回答基本问题，并且，经常在不适当的时间意外被激活（众所周知的案例包括国防部长在下议院对叙利亚问题发表讲话时
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/politics/video/2018/jul/03/gavin-williamson-heckled-siri-commons-speech-video" href="https://www.theguardian.com/politics/video/2018/jul/03/gavin-williamson-heckled-siri-commons-speech-video" rel="noopener noreferrer" target="_blank">
    被Siri 直接打断
   </a>
   、Siri 还在新闻发布会上
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/technology/video/2015/sep/11/siri-interupts-white-house-press-briefing-video" href="https://www.theguardian.com/technology/video/2015/sep/11/siri-interupts-white-house-press-briefing-video" rel="noopener noreferrer" target="_blank">
    打断了记者的提问
   </a>
   、Siri 甚至直接
   <a class="markup--anchor markup--p-anchor" data-href="https://www.youtube.com/watch?v=NxyrQeHFJyY" href="https://www.youtube.com/watch?v=NxyrQeHFJyY" rel="noopener noreferrer" target="_blank">
    打断了新闻直播
   </a>
   ）
  </p>
  <p class="graf graf--p">
   如今已经众所周知这些产品甚至不是100％自动化的 —— 在闪闪发光的接口背后是低薪的和过度工作的人工承包商。
  </p>
  <p class="graf graf--p">
   这些人力往往被剥夺了完全的就业权利，并且对他们所工作的公司没什么忠诚度，但却被雇用来填补人工智能的空白。
  </p>
  <p class="graf graf--p">
   这是迄今为止最具反乌托邦风格的元素。技术与时尚或快餐的生产没有什么不同 — 大部分繁重的工作都是在“血汗工厂”中完成的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在语音功能加速普及的同时，人类的声音已经成为监视的重头戏

   。见：

  <p class="graf graf--p">
   这是技术水平与日俱增的时代；这些服务很快就会变得更加深入和高明，并且可以完全自动化。
  </p>
  <p class="graf graf--p">
   而且，就如 Apple 等这些寡头公司，他们的软件是专有的，人们别无选择。
  </p>
  <p class="graf graf--p">
   必须能让更多人联合起来以打破这种阴暗。第一步就是让更多人了解这些事。⚪️
