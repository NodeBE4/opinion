    <span style="color: #00ccff;">
     <em>
      你经常会看到活动家、主流媒体、警方和当局对游行人数的报告存在很大的差异，严重干扰了行动者的影响力。这种情况下怎么办？本文介绍一个小技巧，用一个最简单的开源调查以反驳主流媒体和警方当局的掩盖
     </em>

   </li>

  <p class="graf graf--p">
   俄罗斯议会目前正在讨论一项将在俄罗斯建立“自治互联网”的新法律。该网络将不再依赖在其他国家/地区托管的服务器和服务。
  </p>
  <p class="graf graf--p">
   俄罗斯的反对者就此提出了抗议。英国广播公司报道说：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    俄罗斯有数千人抗议在互联网上实施更严格限制的计划。上个月议会支持这项备受争议的法案后，莫斯科的大规模集会以及其他两个城市的类似示威活动开启。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    政府表示，该法案允许它将俄罗斯的互联网服务与世界其他地区隔离开来，这将“改善网络安全”。但是活动人士说，这是一种增加审查制度和扼杀异议的尝试。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    活动人士说，周日有超过15,000人聚集在莫斯科，这是警方估计的数字的两倍
   </em>
   。
  </p>
  <p class="graf graf--p">
   路透社的数字略有不同：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    据参加集会的非政府组织 White Counter 称，这次集会聚集了大约15,300人。莫斯科警方将这一数字定为6,500
   </em>
   。
  </p>
  <p class="graf graf--p">
   这些报告通常用于宣传目的，并且通常是错误的。 2月22日，英国寡头 Richard Brenson 在哥伦比亚库库塔组织了一场音乐会。虽然哥伦比亚当局声称“有超过40万人参加”，但“新闻报”报道超过30万人，华盛顿邮报则称有20万人在场。
  </p>
  <p class="graf graf--p">
   通过使用航拍图片和科学人群密度测量，blacklistednews 证明不到2万人参加了音乐会。之后，“华盛顿邮报”默默地删除了其20万的报道。
  </p>
  <p class="graf graf--p">
   对于俄罗斯这场运动，英国广播公司提供了两张照片。
  </p>

  <p class="graf graf--p">
   请注意，两张照片展示的都是相对紧凑的正面镜头。
   <strong class="markup--strong markup--p-strong">
    从正面检查人群大小是不可能的
   </strong>
   。曼彻斯特城市大学人群科学教授 G. Keith 也
   <a class="markup--anchor markup--p-anchor" data-href="http://www.gkstill.com/Support/crowd-density/CrowdDensity-1.html" href="http://www.gkstill.com/Support/crowd-density/CrowdDensity-1.html" rel="noopener noreferrer" target="_blank">
    注意到了
   </a>
   密度估计：CCTV角度越低，密度越高。
  </p>
  <p class="graf graf--p">
   2015年的一个案例
   <a class="markup--anchor markup--p-anchor" data-href="https://www.moonofalabama.org/2015/01/paris-a-photo-op-for-the-hypocrits.html" href="https://www.moonofalabama.org/2015/01/paris-a-photo-op-for-the-hypocrits.html" rel="noopener noreferrer" target="_blank">
    已经显示了
   </a>
   这种正面照片是多么具有误导性。就是下面这样：
  </p>
  <p>
   <img alt="" class="aligncenter wp-image-2394 jetpack-lazy-image" data-lazy-sizes="(max-width: 700px) 100vw, 700px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-19.png?resize=700%2C399&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-19.png?w=764&amp;ssl=1 764w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-19.png?resize=300%2C171&amp;ssl=1 300w" data-recalc-dims="1" height="399" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-19.png?resize=700%2C399&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="700"/>

  </p>

   <img alt="" class="aligncenter size-full wp-image-2395 jetpack-lazy-image" data-lazy-sizes="(max-width: 872px) 100vw, 872px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-20.png?resize=872%2C634&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-20.png?w=872&amp;ssl=1 872w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-20.png?resize=300%2C218&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-20.png?resize=768%2C558&amp;ssl=1 768w" data-recalc-dims="1" height="634" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-20.png?resize=872%2C634&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="872"/>

    <img alt="" class="aligncenter size-full wp-image-2395" data-recalc-dims="1" height="634" sizes="(max-width: 872px) 100vw, 872px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-20.png?resize=872%2C634&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-20.png?w=872&amp;ssl=1 872w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-20.png?resize=300%2C218&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-20.png?resize=768%2C558&amp;ssl=1 768w" width="872"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   该图片允许非常准确的人数估计。因为主干道车道的宽度为3.3至3.6米（11–12英尺）。照片中的街道有十条车道和一条1米的中央分隔带，因此该游行的正前方是37米。
  </p>
  <p class="graf graf--p">
   可以使用与车道宽度的比较来测量人群的密度。细节图片分析显示，在前7米处有一个中等密度的人群，每平方米约2.5人，接下来的7米，每平方米的轻中密度为1.5人。除此之外的人数可能总共约300人。
  </p>
  <p class="graf graf--p">
   可以这样计算：
  </p>
  <p class="graf graf--p">
   37m * 7m * 2.5 ppl/m2 + 37m * 7m * 1.5 ppl/m2 + 300 ppl = 1.336 人
  </p>
  <p class="graf graf--p">
   即使我们假设前区的密度高达 3.5 ppl / m2，后面区域的密度为 2.5 ppl / m2，而后方区域有500人混合，总共只有2,054人。
  </p>
  <p class="graf graf--p">
   莫斯科市拥有1700万人口，周围还有300万人口。 2,000名抗议者占其人口的0.01％。将密度增加三倍才能达到警方估计的6.500人、或莫斯科居民的0.0325％。
  </p>
  <p class="graf graf--p">
   通常情况下克里姆林宫总会把数字往少处说，为什么这次不同？可能需要对当局的意向有所怀疑。
  </p>
