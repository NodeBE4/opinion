 <p>
  <span style="font-size: 12pt;">
   原标题 :
   <b>
    维吾尔人站起来，发出你的怒吼！
   </b>
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   近几个月来，美国几大媒体，如《华尔街日报》、《美联社》、《纽约时报》、《外交政策》等，一个接着一个，报道了有关海外维吾尔人被中共骚扰、施压的情况。
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   从各个真名实姓、或匿名陈述者报道来看，情况及其严重；海外维吾尔人在国内的亲人已被中共当局当作了人质。中共以拘押、抓捕，甚至判刑入狱海外维吾尔人在国内亲人之极端恐怖主义做法，施压海外维吾尔人，企图使海外维吾尔人胆怯、退让，甚至投降，在海外做中共外宣之帮凶，甘当中共奴才！
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   实际上，中共这种做法其实也并不新鲜，也不是昨天或前天才开始的做法；很多东突厥斯坦复国运动海外骨干，各国维吾尔人权活动先锋，他们的亲人早已经被抓的抓、判的判，和国内亲人失去联络、联系的时间，早已经是很长了。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   “株连九族”是中国统治者一贯的做法，中国共产党也不例外。共产党自建立以来，清洗运动不断；毛泽东为首共产党自窃取中国统治权以来，更是以各种名目的运动，对处于其统治的老百姓轮番进行镇压。尤其是，由毛泽东一手发动的所谓的“文化大革命”，以及其前后的镇反、反右、反地方民族主义等运动，在东突厥斯坦，针对的就是那些在历史上参加过东突厥斯坦各突厥民族复兴运动的倡导者，以及两次东突厥斯坦独立建国运动的先锋。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   凡是倡导东突厥斯坦独立，倡导民族复兴，倡导民族文化、身份、信仰，倡导民族平等的爱国者；无论其来自何民族，无论其为维吾尔、哈萨克、乌兹别克，还是蒙古、锡伯、回，都被中共政权当作了敌人，被中共在历次运动中，整的他们家破人亡、妻离子散！
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   现在，这种绑架国内家人做人质，施压海外亲人就范的恐怖主义做法，只不过是被习近平更为赤裸裸、更为明目张胆地使用到每一个海外维吾尔人，图伯特人，汉民主、民运人士身上而已；无论其是倡导独立系民族复兴的，倡导人权的，还是倡导民主、平等、法治的！
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   甚至，那些平时远离维吾尔社区，因惧怕中共而以自欺欺人的不参与政治为口号，躲避维吾尔人各类活动，包括一些纯娱乐性活动；只顾低头挣钱，不管他人生死，等待时机准备衣锦还乡，光宗耀祖的个别维吾尔人，这次居然也成为了中共借国内父母亲人打压的目标，也无法幸免于难。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   但令人惊讶的是，在这种全体维吾尔民族已经被中共以其极端恐怖主义镇压，民族清洗之政策、做法，宣示为中国之敌人之时刻；维吾尔民族作为一个民族之生存处于极其危机、危难的时刻；居然，还有维吾尔人还在继续犹豫、胆怯、退让、观望；仍然，还有维吾尔人极其幼稚的对中共抱希望。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   中国人有句老话，说的是：“人找老实的欺，柿子找软的捏”；越是犹豫、软弱、退让，越是害怕、胆怯、恐惧，中共就越会步步紧逼、得寸进尺；害怕者、甘愿受辱者在祖国的亲人，就更易被中共认为是有用人质；中共更会任意拿捏、欺辱其国内亲人，以便从海外的这些胆小鬼们得到更多的情报，更进一步强化对海外维吾尔人的控制。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   我以为，现在是不仅是我们整个东突厥斯坦的各民族联合起来的时刻已经来临，而且是整个海外倡导民主、自由、平等的，反对独裁、专制的各派、各民族团结起来，共同对敌的时刻已经来临；我们海外各派、各民族，必须不惧强权，不畏邪恶，坚决的向中共恐怖主义极权统治说不，以大无畏的勇敢精神斩断中共伸向海外的邪恶之手，与习近平为首独裁政权之恐怖主义做法做坚决的斗争，绝不妥协；和中共做最后斗争的时刻已经来临，这不是我们的选择，而是中共独裁政权将我们都大家逼上了梁山！
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   害怕强权、畏惧邪恶，向绑架我们亲人的中共恐怖主义低头，与习近平独裁政权妥协；只能把我们引向泯灭人性、失去人格尊严的万丈深渊；使我们更加胆小如鼠、是非不分，成为邪恶中共的帮凶、奴才；使我们尽管生活在自由的文明世界，灵魂却在独裁者的奴役下呻吟、挣扎，良心在强权蹂躏中变态、堕落！
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   那些只顾自己个人利益的维吾尔人，图伯特人、汉人，请不要再沉默，不要再躲避，不要再昧着良心，甘心堕落、助纣为虐；站起来，勇敢地面对中共恐怖主义，面对中共邪恶极权，面对习近平独裁；勇敢地面对那些绑架我们亲人，以我们的亲人做人质，威胁我们的自由、威胁良知的黑暗恶势力，发出怒吼，加入抗争运动！
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   维吾尔人，图伯特人、汉人，不要再害怕，不要再畏惧，不要再退让、犹豫；实际上，独裁者的胆子最小，极权政府最不稳定，作恶者心最虚；独裁者、极权政府、作恶者，最害怕的是民众，害怕的是民众的团结一心，走上街头，参加游行，要求自由、民主与平等。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   4月27日全世界维吾尔人汇聚比利时布鲁塞尔欧盟议会前，举行5000人大游行，将不仅是海外维吾尔人再一次，也是继3月15日全世界维吾尔妇女以“同一个步伐，共同的呼声”为口好发起大游行之后，为显示维吾尔民族之团结，显示海外维吾尔人为拯救危难中之东突厥斯坦父母乡邻及各民族同胞之决心；同时，也应该是海外各派、各民族，显示其勇于放弃争论，走到一起；以敢于牺牲一切之精神，献身自由、民主与平等的时刻！
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   请站出来，维吾尔人、图伯特人、汉人，发出怒吼，向极权恐怖主义说不！向有辱人类良心、尊严之中共伸向海外的黑手说不！
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   倾巢之下无完卵，再不站出来，可能再没有机会站起来了！
  </span>
 </p>
