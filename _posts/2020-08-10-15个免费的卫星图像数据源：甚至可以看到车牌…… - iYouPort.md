   <strong class="markup--strong markup--p-strong">
    1、USGS Earth Explorer
   </strong>
  </h3>
  <p>

  <p class="graf graf--p">
   无论您身在何处，
   <a class="markup--anchor markup--p-anchor" data-href="http://earthexplorer.usgs.gov/" href="http://earthexplorer.usgs.gov/" rel="noopener noreferrer" target="_blank">
    USGS Earth Explorer
   </a>
   都可以提供大量的卫星图像资源：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    LANDSAT：
   </strong>
   想象一下，在任何给定地点都能回顾40多年；在任何给定时间段内，您都可以看到发生了什么变化。那有多值钱？
  </p>
  <p class="graf graf--p">
   这是 Landsat 卫星影像的骄傲。它具有近40年的历史，是任何其他卫星图像所无法比拟的。通过 USGS Earth Explorer 上的 Landsat 您可以免费下载这个星球的历史。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    高程：
   </strong>
   想要高程吗？ASTER 和 SRTM 是世界著名的数字高程模型。通过 USGS Explorer，在地球上的任何地方找到你想要的东西。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    高光谱
   </strong>
   ：探索来自 Hyperion 的高光谱图像。只有通过这个平台您才能拥有如此难得的机会。
  </p>
  <p class="graf graf--p">
   USGS Earth Explorer 是免费卫星图像的世界一流来源。无论您身在何处，都可以随时使用它。
  </p>
  <p class="graf graf--p">
   这里有一个有关如何从 USGS Earth Explorer 下载免费 Landsat 影像的
   <a class="markup--anchor markup--p-anchor" data-href="https://gisgeography.com/usgs-earth-explorer-download-free-landsat-imagery/" href="https://gisgeography.com/usgs-earth-explorer-download-free-landsat-imagery/" rel="noopener noreferrer" target="_blank">
    入门指南
   </a>
   。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、Sentinel Open Access Hub
   </strong>
  </h3>
  <p>
   <img alt="" class="aligncenter size-full wp-image-13244 jetpack-lazy-image" data-lazy-sizes="(max-width: 861px) 100vw, 861px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-10.png?resize=861%2C396&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-10.png?w=861&amp;ssl=1 861w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-10.png?resize=300%2C138&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-10.png?resize=768%2C353&amp;ssl=1 768w" data-recalc-dims="1" height="396" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-10.png?resize=861%2C396&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="861"/>

    <img alt="" class="aligncenter size-full wp-image-13244" data-recalc-dims="1" height="396" sizes="(max-width: 861px) 100vw, 861px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-10.png?resize=861%2C396&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-10.png?w=861&amp;ssl=1 861w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-10.png?resize=300%2C138&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-10.png?resize=768%2C353&amp;ssl=1 768w" width="861"/>

  <p class="graf graf--p">
   Sentinel-2 是开放式卫星图像新的令人兴奋的时代的开始。
  </p>
  <p class="graf graf--p">
   不仅仅是因为其覆盖范围遍及全球；而且，Sentinel-2 是当今可下载的最清晰的免费卫星图像工具。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://scihub.copernicus.eu/dhus/" href="https://scihub.copernicus.eu/dhus/" rel="noopener noreferrer" target="_blank">
    Sentinels Open Access Hub
   </a>
   是下载 Sentinel 图像的官方总站。
   <a class="markup--anchor markup--p-anchor" data-href="https://gisgeography.com/how-to-download-sentinel-satellite-data/" href="https://gisgeography.com/how-to-download-sentinel-satellite-data/" rel="noopener noreferrer" target="_blank">
    这里有一份指南
   </a>
   ，可以帮您了解如何下载 Sentinel-2 图像。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    SENTINEL-2
   </strong>
   ：两颗卫星组成 Sentinel-2。由于其受欢迎程度，未来几年将再增加2颗卫星。
  </p>
  <p class="graf graf--p">
   Sentinel-2 具有10米的分辨率 (红绿蓝、近红外)。它的12个光谱带包括SWIR。SWIR 波段具备更强的大气穿透能力，有助于对地物景观进行较以往更深入的分析判读。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    SENTINEL-1
   </strong>
   ：如果需要合成孔径雷达（SAR）图像，则 Sentinel-1 为C波段。 SAR的潜力是无限的。但是您不会得到像 Sentinel-2 这样的漂亮图片。
  </p>
  <p class="graf graf--p">
   下载 Sentinel 图像后，如何使用它？下载ESA的开源
   <a class="markup--anchor markup--p-anchor" data-href="https://sentinel.esa.int/web/sentinel/toolboxes/sentinel-2" href="https://sentinel.esa.int/web/sentinel/toolboxes/sentinel-2" rel="noopener noreferrer" target="_blank">
    Sentinel-2 Toolbox
   </a>
   。所有工具都为您提供了包含丰富的分析、可视化和处理工具集的框架。
  </p>

    3、NASA Earthdata Search

  </h3>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    LAND COVER:

   NASA Earthdata 还是全球土地使用和覆盖的绝佳来源。它具有专业的卫星数据，如永久冻土和湿地类型。
  </p>
  <p class="graf graf--p">
   建议从一个简单的搜索开始。然后，更改时间范围标准以缩小搜索范围。最后下载免费的卫星图像。经过几次尝试您就能掌握其窍门。👇这里有一份使用指南将有所帮助。

   <strong class="markup--strong markup--p-strong">
    4、NOAA Data Access Viewer
   </strong>
  </h3>
  <p class="graf graf--p">
   您可以在这里发现沿海地区的权威数据集。但这不仅适用于卫星图像。
  </p>
  <p class="graf graf--p">
   数据来自美国国防部（DOD）极地轨道运行环境卫星（POES）、NOAA的地球静止运行环境卫星（GOES）和其他衍生数据。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    航空/卫星影像
   </strong>
   ：NOAA 数据访问查看器可保存卫星，航空和 LiDAR 图像。首先，输入您感兴趣的区域。所有可用的数据集将显示在右侧窗格中。从这里开始，只需下载。

   创建 VITO Vision 帐户需要花费一些时间和精力。该界面直观易用，可随时提供免费的低分辨率卫星数据。这种数据类型对于不需要更详细信息的大型应用程序非常有用。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    13、NOAA Digital Coast
   </strong>
  </h3>
  <p>
   <img alt="" class="aligncenter size-full wp-image-13253 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/13.png?resize=1100%2C449&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/13.png?w=1350&amp;ssl=1 1350w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/13.png?resize=300%2C122&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/13.png?resize=1024%2C418&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/13.png?resize=768%2C313&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/13.png?resize=1100%2C449&amp;ssl=1 1100w" data-recalc-dims="1" height="449" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/13.png?resize=1100%2C449&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13253" data-recalc-dims="1" height="449" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/13.png?resize=1100%2C449&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/13.png?w=1350&amp;ssl=1 1350w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/13.png?resize=300%2C122&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/13.png?resize=1024%2C418&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/13.png?resize=768%2C313&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/13.png?resize=1100%2C449&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   当您在
   <a class="markup--anchor markup--p-anchor" data-href="https://coast.noaa.gov/digitalcoast/" href="https://coast.noaa.gov/digitalcoast/" rel="noopener noreferrer" target="_blank">
    NOAA’s Digital Coast
   </a>
   上探索数据时，这一切都与美丽的海滨有关。您将在此图像平台上获得沿海数据。
  </p>
  <p class="graf graf--p">
   要下载数据，请选择您感兴趣的区域。定义要下载的数据集。并筛选结果。这里有底栖生物、海拔、图像、土地覆盖和社会经济数据。
  </p>
  <p class="graf graf--p">
   您可以从一系列免费的卫星图像中进行选择，例如雷达，红外和真彩色复合图像。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    14、Satellite Land Cover
   </strong>
  </h3>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://landcover.org/" href="http://landcover.org/" rel="noopener noreferrer" target="_blank">
    Landcover.org
   </a>
   列出了从陆地卫星、MODIS 和 AVHRR 获得的全球土地覆盖数据。
  </p>
  <p class="graf graf--p">
   利用这个工具，研究工作可以量化全球土地覆盖和土地覆盖变化。这包括地表的植被、地质、农业、水文和城市地区。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    15、UNAVCO
   </strong>
  </h3>
  <p class="graf graf--p">
   如果你不知道 UNAVCO 是啥，这里有一个快速而有用的3分钟视频可以了解它。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/yxLMk120vMU" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   他们的主要目标是使用大地测量技术来支持科学研究。洪水、板块构造、地震 — — 这个世界是相通的。UNAVCO 通过提供对大地测量数据的访问来促进研究。他们用自己的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.unavco.org/data/imaging/sar/sar.html" href="https://www.unavco.org/data/imaging/sar/sar.html" rel="noopener noreferrer" target="_blank">
    UNAVCO SAR Archive Search User Interface
   </a>
   。
  </p>
  <p class="graf graf--p">
   该网站充斥着来自卫星的合成孔径雷达。但是首先，您需要一个用户名和密码。然后就可以使用数据了。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    最后
   </strong>
  </h3>
  <p class="graf graf--p">
   卫星数据为我们如何了解地球开辟了新的渠道。它是调查研究的刚需。
  </p>
  <p class="graf graf--p">
   你已经知道，并不需要花几天的时间去寻找一个好的地图，你可以利用这个免费的卫星图像数据列表。
  </p>
  <p class="graf graf--p">
   现在您可以从全球各地获得免费的卫星图像了 —— 找到您想要的东西，不再被宣传的谎言所误导。
  </p>
  <p class="graf graf--p">
   让公民掌握真相，是那些 “空中的眼睛” 能做到的最正确对的事。
