  <p class="graf graf--p">
   讨论？这对推文来说几乎不可能。而且一直都是如此 —— 您如何将至少2500个单词提炼成280个字符？
  </p>
  <p class="graf graf--p">
   更重要的是，您如何将一个有板有眼的论述变成某种有趣的东西？—— 这里的“有趣”指的是*可分享性*，“可分享性”指的是能带来吸睛度（转发量），能到来吸睛度和转发量的东西必须是极化的、充满争议的和刺激的。
  </p>
  <p class="graf graf--p">
   在这里，论述一钱不值。
  </p>
  <p class="graf graf--p">
   说 “Todd Phillips 是个大傻逼”，这是一回事；而你至少需要一千字来说明为什么《 Joker》这部作品缺乏深度，那就是另一回事了。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “从字面上看，就像 ‘我们用真正的预算制作了一部真实的电影，我们称之为他妈的小丑’一样，’” 这是该片导演在去年9月对 The Wrap
   <a class="markup--anchor markup--p-anchor" data-href="https://www.thewrap.com/joker-director-todd-phillips-rebuffs-criticism-of-dark-tone-we-didnt-make-the-movie-to-push-buttons-exclusive/" href="https://www.thewrap.com/joker-director-todd-phillips-rebuffs-criticism-of-dark-tone-we-didnt-make-the-movie-to-push-buttons-exclusive/" rel="noopener noreferrer" target="_blank">
    说的
   </a>
   ，他是指他的电影故意美化了一个被
   <a class="markup--anchor markup--p-anchor" data-href="https://time.com/5666055/venice-joker-review-joaquin-phoenix-not-funny/" href="https://time.com/5666055/venice-joker-review-joaquin-phoenix-not-funny/" rel="noopener noreferrer" target="_blank">
    许多人
   </a>
   认为是反英雄的角色（当时的恶评是如此泛滥）。
  </p>
  <p class="graf graf--p">
   不只是批评。还记得在《蝙蝠侠：黑暗骑士崛起》2012年放映期间发生的 Aurora 枪击案吗？当时的受害者甚至要求片方向幸存者赔钱，并进行枪支暴力干预计划 ——
   <strong class="markup--strong markup--p-strong">
    人们认为是这部电影传播的意识形态导致了疯狂的杀戮
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Phillips 显然对此争议感到困惑。
  </p>
  <p class="graf graf--p">
   他问道：“ 让这些电影引起就暴力问题进行的讨论不是很好吗？如果电影确实引起了人们的议论，为什么这是一件坏事呢？”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这本身的确不是一件坏事，除了它压根儿无法进行讨论之外。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    要进行讨论，您就需要双方都至少保持一点谦逊的态度，也就是说，双方都需要了解自己认为可能是错误的东西，才能真正听取反对意见
   </strong>
   。
  </p>
  <ul>
   <li>
    &lt;“
    <a href="https://www.iyouport.org/%e6%88%91%e6%89%8d%e6%98%af%e5%af%b9%e7%9a%84%ef%bc%9a%e5%93%b2%e5%ad%a6%e5%a6%82%e4%bd%95%e5%b8%ae%e5%8a%a9%e4%bd%a0%e8%a7%a3%e5%86%b3%e7%a1%ae%e8%ae%a4%e5%81%8f%e8%af%af/" rel="noopener noreferrer" target="_blank">
     我才是对的”：哲学如何帮助你解决确认偏误
    </a>
    &gt;
   </li>
  </ul>
  <p class="graf graf--p">
   事实上 Phillips 和他所质疑的“社交媒体暴民们”都没有过真正的讨论。很难责怪谁，因为
   <strong class="markup--strong markup--p-strong">
    阻碍讨论的东西与其说是社交媒体媒介、不如说是用户本身
   </strong>
   。
  </p>
  <p class="graf graf--p">
   可能责怪 Phillips 更容易些。他通过简单化的比喻试图其转变为探索当代恐惧的深刻艺术品，但事实上，该电影只是在暗示深度、但又停留在表面上。
  </p>
  <p class="graf graf--p">
   在类似的矛盾表现中，既然他的小丑电影正被卷入一个复杂的话题，Phillips 拒绝参与其中，而是选择简单地打发那些评论者，反映了他对在线批评的
   <strong class="markup--strong markup--p-strong">
    公开鄙视
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9971 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-1.jpg?resize=1100%2C740&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-1.jpg?w=2048&amp;ssl=1 2048w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-1.jpg?resize=300%2C202&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-1.jpg?resize=768%2C517&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-1.jpg?resize=1024%2C689&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-1.jpg?resize=1100%2C740&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-1.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="740" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-1.jpg?resize=1100%2C740&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>

  <p class="graf graf--p">
   社交媒体网络由同心圆组成，从中心向外依次是：现实生活中的朋友、在线朋友、在线熟人、陌生人。
  </p>
  <p class="graf graf--p">
   我们离中心越远，就会变得越不文明。
  </p>
  <p class="graf graf--p">
   康涅狄格大学教授玛丽·K·沙纳汉（Marie K. Shanahan）是《新闻、在线评论和公共语话的未来》一书的作者，她说，这被称为
   <strong class="markup--strong markup--p-strong">
    “在线去抑制效应”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   她说，没有人可以调查，“我可以投下炸弹并逃跑。我不必处理后果。”
  </p>
  <p class="graf graf--p">
   这无疑是引发很多令人困惑的争论的场所。
   <strong class="markup--strong markup--p-strong">
    那些平日在现实生活中处于憋屈中的人们，在社交媒体上总是看起来特别强势
   </strong>
   ，也许因为 “这里没有第二天还不得不继续相处的人”？
  </p>
  <p class="graf graf--p">
   必须说，确实有一些机器人在故意“惹恼他人而引起混乱”，那是一种
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e6%88%98%e5%92%8c%e4%bf%a1%e6%81%af%e6%88%98/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e6%88%98%e5%92%8c%e4%bf%a1%e6%81%af%e6%88%98/" rel="noopener noreferrer" target="_blank">
    信息战
   </a>
   的标准模式。但是大部分环境依旧是人类构成的。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%88%86%e8%be%a8%ef%bc%9a%e6%9c%ba%e5%99%a8%e4%ba%ba%e3%80%81%e5%83%b5%e5%b0%b8%e7%bd%91%e7%bb%9c%e5%92%8c-trolls/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%88%86%e8%be%a8%ef%bc%9a%e6%9c%ba%e5%99%a8%e4%ba%ba%e3%80%81%e5%83%b5%e5%b0%b8%e7%bd%91%e7%bb%9c%e5%92%8c-trolls/" rel="noopener noreferrer" target="_blank">
     如何分辨：机器人、僵尸网络和 trolls
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   社交媒体无法支持关于《小丑》的辩论（实际上是无法支持关于任何问题的辩论）这本身就是
   <strong class="markup--strong markup--p-strong">
    社交媒体基因的一部分。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “流”是一种参与形式，在被视频游戏和社交媒体采用之前它最初是作为一种心理学设想的参考，旨在使人们从一件事飞向另一件事，不断的寻找但从未实现满足。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    其目的是使人们继续关注肤浅的内容、发表那些没有任何反思性的评论，然后迅速跳到下一个meme。
   </strong>
  </p>
  <p class="graf graf--p">
   就如 Jay David Bolter 在他的新书《
   <a href="https://t.me/iyouport/6459" rel="noopener noreferrer" target="_blank">
    The Digital Plenitude
   </a>
   》中所说的，“游戏既不会太困难和令人沮丧，也不会太容易和无聊”。他指的是视频游戏，当然也包括社交媒体。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这便能在最大程度地提高参与者对当下的瞬间的满足感和对权力的幻想，这两者都充满了不真实甚至自相矛盾，但无所谓，因为没有人想要那些等待足够长的时间才能找到的真相
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在 David Barsamian 1998年的著作《The Common Good》中就描述了这种狭窄的范围以及该范围内的局限性如何使人们处于被动状态。他说，“这给了人们一种自由思考正在进行的
   <strong class="markup--strong markup--p-strong">
    错觉
   </strong>
   ，但障碍总是会被辩论范围的限制所加强”。
  </p>
  <p class="graf graf--p">
   那么，当你无能为力却又充满愤怒时，会发生什么呢？
  </p>
  <p class="graf graf--p">
   社交媒体总是可以让您释放野蛮的数字嗥叫；每个人都争先恐后地以一种或另一种形式喊出自己的挫败感，对头条新闻、对刷屏的 threads、对恼人的历史、甚至现实事实进行确认或否认以强调自己的偏见 ……
  </p>
  <p class="graf graf--p">
   但是，“有人真的在听吗？” Shanahan 问。这可能是最大的问题。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    熟悉社交媒体的人都知道，几乎任何群组（而不仅是公开平台）最终都会变成时间线
   </strong>
   —— 人们各自发言、转发，没有人会去浏览其他人发布的内容。经常的，一模一样的内容会被不断重复发布。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%9c%a8%e4%b8%ad%e5%9b%bd%ef%bc%9a%e4%ba%92%e8%81%94%e7%bd%91%e4%b8%8a%e7%9a%84%e6%94%bf%e6%b2%bb%e4%b8%ba%e4%bd%95%e5%a6%82%e6%ad%a4%e5%b1%80%e9%99%90%ef%bc%88%e4%b8%8a%ef%bc%89/" href="https://www.iyouport.org/%e5%9c%a8%e4%b8%ad%e5%9b%bd%ef%bc%9a%e4%ba%92%e8%81%94%e7%bd%91%e4%b8%8a%e7%9a%84%e6%94%bf%e6%b2%bb%e4%b8%ba%e4%bd%95%e5%a6%82%e6%ad%a4%e5%b1%80%e9%99%90%ef%bc%88%e4%b8%8a%ef%bc%89/" rel="noopener noreferrer" target="_blank">
     在中国：互联网上的政治为何如此局限（上）
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%a4%e6%b5%81%e5%87%ba%e7%8e%b0%e4%ba%86%e4%bb%80%e4%b9%88%e9%97%ae%e9%a2%98%ef%bc%88%e4%b8%8a%ef%bc%89%e4%b8%ba%e4%bb%80%e4%b9%88%e6%80%bb%e6%98%af%e9%9a%be%e4%bb%a5%e8%af%b4%e6%9c%8d%e4%bb%96/" href="https://www.iyouport.org/%e4%ba%a4%e6%b5%81%e5%87%ba%e7%8e%b0%e4%ba%86%e4%bb%80%e4%b9%88%e9%97%ae%e9%a2%98%ef%bc%88%e4%b8%8a%ef%bc%89%e4%b8%ba%e4%bb%80%e4%b9%88%e6%80%bb%e6%98%af%e9%9a%be%e4%bb%a5%e8%af%b4%e6%9c%8d%e4%bb%96/" rel="noopener noreferrer" target="_blank">
     交流出现了什么问题（上）为什么总是难以说服他人？
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%a4%e6%b5%81%e5%87%ba%e7%8e%b0%e4%ba%86%e4%bb%80%e4%b9%88%e9%97%ae%e9%a2%98%ef%bc%88%e4%b8%ad%ef%bc%89%e7%bd%91%e7%bb%9c%e8%88%86%e8%ae%ba%e5%9c%ba%e7%9a%84%e4%ba%a4%e6%b5%81%e9%9a%9c%e7%a2%8d/" href="https://www.iyouport.org/%e4%ba%a4%e6%b5%81%e5%87%ba%e7%8e%b0%e4%ba%86%e4%bb%80%e4%b9%88%e9%97%ae%e9%a2%98%ef%bc%88%e4%b8%ad%ef%bc%89%e7%bd%91%e7%bb%9c%e8%88%86%e8%ae%ba%e5%9c%ba%e7%9a%84%e4%ba%a4%e6%b5%81%e9%9a%9c%e7%a2%8d/" rel="noopener noreferrer" target="_blank">
     交流出现了什么问题（中）网络舆论场的交流障碍和解决办法
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%a4%e6%b5%81%e5%87%ba%e7%8e%b0%e4%ba%86%e4%bb%80%e4%b9%88%e9%97%ae%e9%a2%98%ef%bc%88%e4%b8%8b%ef%bc%89%e6%9d%a5%ef%bc%8c%e5%ae%9e%e7%8e%b0%e6%b2%9f%e9%80%9a/" href="https://www.iyouport.org/%e4%ba%a4%e6%b5%81%e5%87%ba%e7%8e%b0%e4%ba%86%e4%bb%80%e4%b9%88%e9%97%ae%e9%a2%98%ef%bc%88%e4%b8%8b%ef%bc%89%e6%9d%a5%ef%bc%8c%e5%ae%9e%e7%8e%b0%e6%b2%9f%e9%80%9a/" rel="noopener noreferrer" target="_blank">
     交流出现了什么问题（下）来，实现沟通
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   UConn 研究员 Michael P. Lynch 在《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2019/09/23/opinion/fake-news.html" href="https://www.nytimes.com/2019/09/23/opinion/fake-news.html" rel="noopener noreferrer" target="_blank">
    纽约时报
   </a>
   》的专栏中提出了一个答案。
  </p>
  <p class="graf graf--p">
   他引用了一项研究，表明
   <strong class="markup--strong markup--p-strong">
    至少有60％在社交媒体上被分享出来的新闻根本没有被分享者阅读
   </strong>
   ；
   <strong class="markup--strong markup--p-strong">
    情绪上的刺激性是最受欢迎的分享指标，但道德化甚至更好（分享机会增加了20％）也就是指责或暗示他人不道德
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Lynch 的答案是，社交媒体用户的感觉[而不是思考]比听的还多（还记得
   <strong class="markup--strong markup--p-strong">
    “真相感”
   </strong>
   那个词吗？—— 在这里回顾《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%aa%92%e4%bd%93%ef%bc%8c%e4%bd%a0%e7%9c%9f%e7%9a%84%e7%9f%a5%e9%81%93%e4%bb%80%e4%b9%88%e6%98%af%e4%ba%8b%e5%ae%9e%e5%90%97%ef%bc%9f/" href="https://www.iyouport.org/%e5%aa%92%e4%bd%93%ef%bc%8c%e4%bd%a0%e7%9c%9f%e7%9a%84%e7%9f%a5%e9%81%93%e4%bb%80%e4%b9%88%e6%98%af%e4%ba%8b%e5%ae%9e%e5%90%97%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    什么是真相感？大众媒体如何愚弄我们
   </a>
   ？》）。
  </p>
  <p class="graf graf--p">
   他写道：“能激发在线共享内容的行为的最稳定因素是表达我们自己的情绪，这似乎是合理的，我们经常在发帖时表示自己的愤慨，并希望其他人也能分享。这是建立部落和执行社会规范的一种方式”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这也是资本主义对所谓的成功的执着追求的方式。通过贬低他人增强自己的优势。
   </strong>
  </p>
  <p class="graf graf--p">
   Shanahan 说，不要试图将所有内容都固定在“极右翼” 的类型上：“事实上每个人都在做这件事。”
  </p>
  <p class="graf graf--p">
   不管是否有意识，Phillips 都用这种迷惑性来丢弃几乎所有对他电影的批评。他说，“最令我困惑的是，
   <strong class="markup--strong markup--p-strong">
    当符合他们的议程时，极左和极右看起来基本完全一样
   </strong>
   ”…… 确实，两极总会指向同一个方向。
  </p>
  <p class="graf graf--p">
   在社交媒体上，您肯定不会因为表现得客观而得到回报，也不会因为理性获得更多转发和评论。当然，也不会有更多的言论自由。
  </p>
  <p class="graf graf--p">
   正如 Kovie Biakolo 在2013年的《 Thought Catalog》中写道：“自由参与的公共辩论被误解了，当人们认为他们有权简单地表达意识形态，尤其是当这些意识形态必须仅根据表达能力来考虑时。 ”
  </p>
  <p class="graf graf--p">
   自由言论的论点倾向于忽略所有语境，因为它不承认
   <strong class="markup--strong markup--p-strong">
    并非所有声音都平
   </strong>
   等，所有主题或所有媒体也一样。
  </p>
  <p class="graf graf--p">
   NPR 的 Gene Demby 最近在种族对话中也谈到了类似的疏忽：“似乎人们希望通过讨论，我们可以设计一个独立的空间，让人们在那里进行交谈，然后聚在一起，但现实不是这么简单的，任何讨论的开端都必然来自公共场域……
  </p>
  <p class="graf graf--p">
   当然，人们已经在社交媒体上投入了大量的精力，但这与在实际对话中投入的东西完全不同。
  </p>
  <p class="graf graf--p">
   因此，似乎正在发生的情况是，
   <strong class="markup--strong markup--p-strong">
    对话正越来越多地从庞大的、不受限制的互联网公共广场转移开了。
   </strong>
  </p>
  <p class="graf graf--p">
   志趣相投的人会在小圈子中与想法相似的人展开对话，这可能反映了在线回声室，但至少它们不怎么暴力。
  </p>
  <p class="graf graf--p">
   Shanahan 说：“有很多人不参与在社交媒体上的辩论，是因为社交媒体不是一个安全的空间。”（这里的安全指的是各种方面）
  </p>
  <p class="graf graf--p">
   最近在《国际传播杂志》上发表的一项针对近200名大学生的研究证实，尽管他们消费并共享了在线政治“讨论”，但是学生还是选择了离线，为了限制他们自己受到的威胁并更好地理解和控制讨论。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9973 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-3.jpg?resize=1100%2C733&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-3.jpg?w=1440&amp;ssl=1 1440w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-3.jpg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-3.jpg?resize=768%2C512&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-3.jpg?resize=1024%2C683&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-3.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-3.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="733" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-3.jpg?resize=1100%2C733&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9973" data-recalc-dims="1" height="733" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-3.jpg?resize=1100%2C733&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-3.jpg?w=1440&amp;ssl=1 1440w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-3.jpg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-3.jpg?resize=768%2C512&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-3.jpg?resize=1024%2C683&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-3.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-3.jpg?resize=272%2C182&amp;ssl=1 272w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Phillips 说：“很难与 Twitter 上的3000万人辩论。你就是做不到，对吧？所以你就滚出了，就像我这样”。
  </p>
  <p class="graf graf--p">
   问题不在于 Phillips 拒绝招待所谓的 Twitter 暴民，而是，
   <strong class="markup--strong markup--p-strong">
    他对善意批评的回应就像社交媒体攻击一样空洞而感性
   </strong>
   —— 似乎要想被其他人“听取”的唯一方法就是充分放大社交媒体的*不可讨论性*。
  </p>
  <p class="graf graf--p">
   就像很多人一直在做的那样。这是个恶性循环。
  </p>
  <p class="graf graf--p">
   Phillips 试图向人们展示破坏性社会如何在没有实际拥有决定权的情况下塑造破坏性的个人；但他显然不愿参加有关他的电影的论述。
  </p>
  <p class="graf graf--p">
   他甚至无法在屏幕上或电视上与观众互动，这进一步反映了
   <strong class="markup--strong markup--p-strong">
    他是多么地鄙视社交媒体的肤浅性
   </strong>
   。
  </p>
  <p class="graf graf--p">
   具有讽刺意味的是，Phillips 不仅担心在线对话也担心离线对话，显然面对面 时人们总会比在线状态下友好得多。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    但是要想真正能够进行讨论，他将不得不抛弃自己的情感。这对每个人来说都是如此。
   </strong>
  </p>
  <p class="graf graf--p">
   1985年出版的《
   <em class="markup--em markup--p-em">
    The Ending of Time
   </em>
   》是物理学家戴维·博姆（David Bohm）与哲学家吉德·克里希那穆提（Jiddu Krishnamurti）的对话，介绍了一种试图超越冲突的更先进的讨论方法。
  </p>
  <p class="graf graf--p">
   Bohm Dialogue 是这样一种对话模式：即 在不较真的情况下进行思想交流也许会更容易彼此理解。部分原因是
   <strong class="markup--strong markup--p-strong">
    承认您的感觉如何阻止了您听取对方的声音
   </strong>
   ，从而放弃了对话需要解决的想法。
  </p>
  <p class="graf graf--p">
   具有讽刺意味的是，Bohm 依旧是使用了“流”的隐喻来传达人与人之间的思想互通，尽管 Bohm Dialogue 的定义暗示了社交媒体方法的对立面 —— 即 与 Phillips 方法的对立面。
  </p>
  <p class="graf graf--p">
   Bohm 在1987年的《科学，秩序与创造力》中解释说：“在对话中，人们必须能够直面分歧而没有对抗，并愿意探索他们不认同的观点。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果人们能够在不回避并避免情绪化的情况下进行这样的对话，他们会发现，
   <strong class="markup--strong markup--p-strong">
    放下顽固的即有认知是很重要的，否则会以破坏对话本身为代价
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   这是个值得思考的问题。当我们一直在强调社交媒体的设计模式根本性不适合对话时，请不要忘了，对话的始终是人。⚪️
  </p>
