   3K受害党
  </h1>

 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <strong>
      <em>
       允许他们监视的结果就是这样：活动家被指为”恐怖分子”；新纳粹被称为”受害者”，这是哪国的规矩？专注于监视活动家的人都是谁？
      </em>
     </strong>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   新文件显示，联邦调查局
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/us-news/california" href="https://www.theguardian.com/us-news/california" rel="noopener noreferrer" target="_blank">
    对加利福尼亚州
   </a>
   的一个民权组织进行了“国内恐怖主义”调查，并在2016年抗议新纳粹分子后将活动家们标记为“极端分子”。
  </p>
  <p class="graf graf--p">
   根据“卫报”获得的文件，联邦当局对“By Any Means Necessary ”（Bamn）进行了监视行动，监视这个左翼团体的一举一动，此事是在 Bamn 的一名成员在白人至上主义集会上被
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/us-news/2017/jul/25/yvette-felarca-trump-protest-charges-activism" href="https://www.theguardian.com/us-news/2017/jul/25/yvette-felarca-trump-protest-charges-activism" rel="noopener noreferrer" target="_blank">
    刺伤
   </a>
   后进行的。联邦调查局的 Bamn 档案显示：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    联邦调查局调查了 Bamn 对“三K党”和白人至上主义者“权利”的潜在“阴谋”；
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     联邦调查局认为 KKK 是受害者，左翼抗议者是潜在的恐怖威胁
    </strong>
    ，写道：“KKK 由一些人认为支持白人至上主义议程的成员组成”；
   </li>
   <li class="graf graf--li">
    联邦调查局的监视包括针对人员的监视，
    <strong class="markup--strong markup--li-strong">
     该机构将 Bamn 倡导的反对“强奸和性侵犯”和“警察暴行”作为恐怖主义调查的证据
    </strong>
    ……
   </li>
  </ul>
  <p class="graf graf--p">
   前联邦调查局特工和极右翼专家 Mike German 表示，联邦调查局长达46页的关于 Bamn 的报告对KKK提出了“非常令人惊讶的”描述。该报告忽略了“100年的 Klan 恐怖主义已经杀死了数千名美国人并继续使用暴力直至现今。对KKK的这种描述应该是联邦调查局领导层的耻辱。”
  </p>
  <p class="graf graf--p">
   Bamn 的全国主席 Shanta Driver 在向卫报发表的一份声明中强烈批评该监视，他说：“联邦调查局对 BAMN 的兴趣是长期政策的一部分……
   <strong class="markup--strong markup--p-strong">
    从他们迫害和诽谤马丁路德金的行动开始，该部门有一个以和平民权和反种族主义组织为针对目标的卑劣历史，同时没有采取任何行动起诉袭击 King 及其建立的运动的种族主义者和法西斯主义者
   </strong>
   。“
  </p>
  <p class="graf graf--p">
   在2016年6月在萨克拉门托举行的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/us-news/2016/jun/26/white-nationalist-rally-stabbings-california" href="https://www.theguardian.com/us-news/2016/jun/26/white-nationalist-rally-stabbings-california" rel="noopener noreferrer" target="_blank">
    新纳粹集会
   </a>
   上，持有刀具的白人至上主义者与包括 Bamn 活动家在内的数百名反抗议者面对面后，联邦调查局发起了对 Bamn 的“恐怖主义”调查和监视。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    虽然许多新纳粹分子被怀疑在混战中刺杀了至少七名反法西斯活动家，甚至有些人受到
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2018/may/25/neo-nazi-rally-california-stabbing-police-target-black-activist" href="https://www.theguardian.com/world/2018/may/25/neo-nazi-rally-california-stabbing-police-target-black-activist" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     致命伤害
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，但是，联邦调查局却选择对左翼抗议者的活动进行调查
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这些文件虽然经过了严格修改，但并未包含联邦调查局的任何结论，即 不存在 Bamn 违反法律或构成持续威胁的任何证据。其成员尚未面临联邦起诉。FBI 拒绝对 Bamn 发表评论。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “很明显，联邦调查局放弃了调查，没有任何不法行为的证据。这一监视本来就不应该存在“。
  </p>
  <p class="graf graf--p">
   2016年的集会是由两个白人至上主义团体组织的：传统工人党（TWP）和附属加利福尼亚的金州光头党。加利福尼亚州
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2018/feb/09/california-police-white-supremacists-counter-protest" href="https://www.theguardian.com/world/2018/feb/09/california-police-white-supremacists-counter-protest" rel="noopener noreferrer" target="_blank">
    执法部门随后与新纳粹分子一起
   </a>
   辨认抗议者、追捕被刺伤的受害者和反法西斯活动家，并
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/us-news/2019/jan/25/california-police-neo-nazis-antifa-protest" href="https://www.theguardian.com/us-news/2019/jan/25/california-police-neo-nazis-antifa-protest" rel="noopener noreferrer" target="_blank">
    决定不起诉极右翼的任何凶手
   </a>
   。
  </p>
  <p class="graf graf--p">
   FBI 似乎采用过类似的方法。在2016年10月编辑的一份文件中，FBI 将其 Bamn 调查标记为“DT [国内恐怖主义] — ANARCHIST EXTREMISM” 案。联邦调查局的旧金山办事处写道，它正在调查有关“Bamn 成员参加 Ku Klux Klan 集会并袭击纳粹支持者”的指控。它以这种方式总结了萨克拉门托事件：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    2016年，执法部门了解到3K党将在州议会大厦举行集会…… KKK 由一些人认为支持白人至上主义议程的成员组成。作为回应，一些团体动员起来抗议集会。发布的传单要求人们参加，以便阻止该集会
   </em>
   。
  </p>
  <p class="graf graf--p">
   KKK 和传统工人党有着相似的意识形态，但却是不同的群体。目前还不清楚为什么联邦调查局将这次集会称为 KKK 事件。
  </p>
  <p class="graf graf--p">
   Bamn 长期以来一直倡导种族公正和移民权利，经常组织公共活动和抗议集会。
  </p>
  <p class="graf graf--p">
   联邦调查局的报告称，“某些 BAMN 成员的行为可能超出受保护活动的界限，可能构成对联邦法律的侵害”。作为理由联邦调查局引用了 Bamn 的网站，该网站鼓励支持者抗议 KKK，其中包括 “SMASH FASCISM！”和“NO ‘FREE SPEECH’ FOR FASCISTS!”等口号。联邦调查局列举的“证据”还包括 Bamn 页面的屏幕截图，其中引用了该组织的其他一些倡导问题，包括反对“强奸和性侵犯”、反对“警察暴行”的运动。
  </p>
  <p class="graf graf--p">
   联邦调查局的档案还包括对 Yvette Felarca 的提及，这是一名 Bamn 成员，她在集会中被刺伤了，但现在她面临国家指控的袭击和骚乱（她的律师在法庭上辩称，
   <strong class="markup--strong markup--p-strong">
    警察调查员和检察官对反法西斯主义者有偏见，并努力保护新纳粹分子
   </strong>
   ）。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “联邦调查局没有找到殴打反种族主义抗议者的人，而是选择瞄准 BAMN，他们自己也承认活动家享有受第一修正案保护的示威游行权利。”
  </p>
  <p class="graf graf--p">
   Bamn 的档案显示，
   <strong class="markup--strong markup--p-strong">
    2016年11月，FBI 对伯克利学区外的抗议活动也进行了监视
   </strong>
   。由于被修改过，目前还不清楚联邦调查局正在监视谁，尽管报告指出 FBI 观察到“几个孩子……坐在外面……旁边有标志”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Bamn 案件涉及最近围绕 FBI 针对左翼团体的强烈争议，其中包括
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/us-news/2017/feb/10/standing-rock-fbi-investigation-dakota-access" href="https://www.theguardian.com/us-news/2017/feb/10/standing-rock-fbi-investigation-dakota-access" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     对 Standing Rock 活动家的所谓“恐怖主义”调查
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2018/may/11/rakem-balogun-interview-black-identity-extremists-fbi-surveillance" href="https://www.theguardian.com/world/2018/may/11/rakem-balogun-interview-black-identity-extremists-fbi-surveillance" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     对黑人活动家的监视
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，以及
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/us-news/2018/dec/13/fbi-climate-change-protesters-iowa-files-monitoring-surveillance-" href="https://www.theguardian.com/us-news/2018/dec/13/fbi-climate-change-protesters-iowa-files-monitoring-surveillance-" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     对和平气候变化抗议者进行的间谍活
    </strong>
    动
   </a>
   。
  </p>
  <p class="graf graf--p">
   Shapiro 说，即使很多人都知道联邦调查局一直在监视活动人士，但该报告仍然令人震惊。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “一群反法西斯主义者出现在纳粹集会上并遭到纳粹分子的袭击，该局的反应却是对反法西斯主义者展开国内恐怖主义调查 ……就其本身而言，联邦调查局一直是一支主要针对左派的政治警察部队。”
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    R
   </span>

    evealed: FBI investigated civil rights group as ‘terrorism’ threat and viewed KKK as victims
   </a>
   . Bureau spied on California activists, citing potential ‘conspiracy’ against the ‘rights’ of neo-Nazis
