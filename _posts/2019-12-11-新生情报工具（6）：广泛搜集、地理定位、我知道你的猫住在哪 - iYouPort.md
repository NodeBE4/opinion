  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Th3inspector
   </strong>
  </p>
  <p class="graf graf--p">
   这是个相对比较知名的信息收集工具，现在有了更新版，速度更快了。
  </p>
  <p class="graf graf--p">
   如果您还不了解这个工具，下面是一些简单的介绍；关于它能干什么，以及视频操作演示。
  </p>
  <p class="graf graf--p">
   用法 ——
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    电话号码信息：-n （-number）
   </li>
   <li class="graf graf--li">
    查找IP地址和电子邮件服务器：-mx（-mailserver）
   </li>
   <li class="graf graf--li">
    域名Whois查询：-w （-whois）
   </li>
   <li class="graf graf--li">
    查找网站/ IP地址位置：-l （-location）
   </li>
   <li class="graf graf--li">
    域名年龄检查器：-a （-age）
   </li>
   <li class="graf graf--li">
    用户代理信息：-ua（useragent）
   </li>
   <li class="graf graf--li">
    信用卡 Bin Checker：-b（bin）
   </li>
   <li class="graf graf--li">
    子域扫描程序：-s（-subdomain）
   </li>
   <li class="graf graf--li">
    电子邮件地址检查：-e（-email）
   </li>
   <li class="graf graf--li">
    内容管理系统检查：-cms（-cms）
   </li>
   <li class="graf graf--li">
    显示帮助消息并退出：-h（-help）
   </li>
  </ul>
  <p class="graf graf--p">
   示例 ——
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    列出所有基本选项和开关：
   </li>
  </ul>
  <pre class="graf graf--p">perl Th3inspector.pl -h</pre>
  <ul class="postList">
   <li class="graf graf--li">
    获取网站信息：
   </li>
  </ul>
  <pre class="graf graf--p">perl Th3inspector.pl -i example.com</pre>
  <ul class="postList">
   <li class="graf graf--li">
    获取电话号码信息：
   </li>
  </ul>
  <pre class="graf graf--p">perl Th3inspector.pl -n xxxxxxx</pre>
  <ul class="postList">
   <li class="graf graf--li">
    查找IP地址和电子邮件服务器：
   </li>
  </ul>
  <pre class="graf graf--p">perl Th3inspector.pl -mx example.com</pre>
  <ul class="postList">
   <li class="graf graf--li">
    查找网站或IP地址位置：
   </li>
  </ul>
  <pre class="graf graf--p"><span style="font-family: georgia, palatino, serif;"><code>perl Th3inspector.pl -l example.com</code></span></pre>
  <p class="graf graf--p">
   安装 —— （ Linux）
  </p>
  <pre class="graf graf--pre">git clone https://github.com/Moham3dRiahi/Th3inspector.git
cd Th3inspector
chmod +x install.sh &amp;&amp; ./install.sh</pre>
  <p class="graf graf--p">
   （ Windows）
  </p>
  <pre class="graf graf--pre">Download Perl
Download Th3inspector
cpan install JSON
Extract Th3inspector into Desktop
Open CMD and type the following commands:
cd Desktop/Th3inspector-master/
perl Th3inspector.pl</pre>

   因为它之所以能存在，正是因为隐私权倡导者多年来关于自我保护的警告失败了 —— 依旧有太多人乐于在互联网上暴露自己最致命的部分。
  </p>
  <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote">
   <p>
    “如果你在社交媒体网站上启用了基于位置的服务 —— 你只是在邀请罪犯到你家” —— 塔斯卡卢萨警长办公室 Lt. Andy Norris
   </p>
  </blockquote>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8997 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/14.png?resize=1100%2C527&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/14.png?w=1300&amp;ssl=1 1300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/14.png?resize=300%2C144&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/14.png?resize=768%2C368&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/14.png?resize=1024%2C491&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/14.png?resize=1100%2C527&amp;ssl=1 1100w" data-recalc-dims="1" height="527" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/14.png?resize=1100%2C527&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8997" data-recalc-dims="1" height="527" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/14.png?resize=1100%2C527&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/14.png?w=1300&amp;ssl=1 1300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/14.png?resize=300%2C144&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/14.png?resize=768%2C368&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/14.png?resize=1024%2C491&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/14.png?resize=1100%2C527&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   欢迎来到今天的互联网 — 您可以购买任何东西，每个网站都在跟踪您的一举一动，任何地方您都可以找到猫的视频和图像。
  </p>
  <p class="graf graf--p">
   目前，在公共图像托管网站上有1500万个标有“猫”字样的图像，每天还有数千个图像从地球上的任意位置上传上来。
  </p>
  <p class="graf graf--p">
   iknowwhereyourcatlives.com 是一个数据可视化实验，它通过嵌入在元数据中的纬度和经度坐标，在世界地图上定位一百万只猫的公开图像样本。
  </p>
  <p class="graf graf--p">
   该项目探讨了互联网的两种用途：一种是促进对驯化猫科动物的社交和幽默欣赏的共享；而另一种是，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e6%8e%a7%e5%88%b6%e7%9a%84%e4%b8%96%e7%95%8c%e5%92%8c%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89-2/" href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e6%8e%a7%e5%88%b6%e7%9a%84%e4%b8%96%e7%95%8c%e5%92%8c%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89-2/" rel="noopener noreferrer" target="_blank">
    监视资本主义
   </a>
   的现实。
  </p>
  <p class="graf graf--p">
   构建：
  </p>
  <p class="graf graf--p">
   HTML5, CSS3, Twitter Bootstrap 3, Javascript / jQuery, history.js, Google Map API, MarkerClustererPlus, MarkerWithLabel, PHP 5, MySQL, Apache, Python, SciPy K-means clustering
  </p>
  <p class="graf graf--p">
   对于开源情报调查来说：如果你的追踪目标是个喜欢晒的人，你就有福了。
  </p>
  <p class="graf graf--p">
   视频：
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="618" src="//player.vimeo.com/video/99867948?title=0&amp;byline=0&amp;portrait=0&amp;color=8dc7dc" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    新的图谱搜索工具
   </strong>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/facebook-%e6%ad%a3%e5%9c%a8%e4%bb%a4%e7%9c%9f%e7%9b%b8%e6%8c%96%e6%8e%98%e4%ba%8b%e4%b8%9a%e9%99%b7%e5%85%a5%e5%9b%b0%e5%a2%83/" href="https://www.iyouport.org/facebook-%e6%ad%a3%e5%9c%a8%e4%bb%a4%e7%9c%9f%e7%9b%b8%e6%8c%96%e6%8e%98%e4%ba%8b%e4%b8%9a%e9%99%b7%e5%85%a5%e5%9b%b0%e5%a2%83/" rel="noopener noreferrer" target="_blank">
    Facebook 关闭其图谱搜索
   </a>
   后，全球的开源情报社区开发者都在尽各种努力补救这一损失。
  </p>
  <p class="graf graf--p">
   最近一些越南开发人员已经创建了一个新的 Facebook 搜索工具，实际上看起来效果很好，并以 JSON 格式返回查询。
  </p>
  <p class="graf graf--p">
   旧的图谱搜索中采取的运算符，在这里基本都可以使用。
  </p>
  <p class="graf graf--p">
   您需要构建的URL如下：
  </p>
  <pre class="graf graf--p">graphsearch.thao.pw/api/graphSearch?q={ID}/{operator}&amp;amount=10</pre>
  <p class="graf graf--p">
   其中{ID}是您要查询的用户的 Facebook ID，{operator}是您要执行的特定搜索；’amount = x’ 指定了需要返回的结果数量。
  </p>
  <p class="graf graf--p">
   例如，如果想找出在壳牌公司工作的前100名员工（id = 200969413280005），只需要请求’employees’，这样：
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://graphsearch.thao.pw/api/graphSearch?q=200969413280005/employees&amp;amount=100" href="https://graphsearch.thao.pw/api/graphSearch?q=200969413280005/employees&amp;amount=100" rel="noopener noreferrer" target="_blank">
    https://graphsearch.thao.pw/api/graphSearch?q=200969413280005/employees&amp;amount=100
   </a>
  </p>
  <p class="graf graf--p">
   就是这么简单！
  </p>
  <p class="graf graf--p">
   查询工具：
   <a class="markup--anchor markup--p-anchor" data-href="https://graphsearch.thao.pw" href="https://graphsearch.thao.pw" rel="nofollow noopener noreferrer" target="_blank">
    https://graphsearch.thao.pw
   </a>
  </p>
  <p class="graf graf--p">
   GUI版本：
   <a class="markup--anchor markup--p-anchor" data-href="https://fbinsight.thao.pw/" href="https://fbinsight.thao.pw/" rel="nofollow noopener noreferrer" target="_blank">
    https://fbinsight.thao.pw/
   </a>
  </p>

   富有创造力的开发者社区总是如此令人激动。如果您也对开源情报工具开发感兴趣，欢迎与我们联系。
