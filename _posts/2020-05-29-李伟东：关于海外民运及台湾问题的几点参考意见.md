  关于海外民运及台湾问题的几点参考意见
  <br/>
  李伟东
  <br/>
  （2020/2/8）
  <br/>
  一、关于针对海外民运之批评意见的几点辨析
  <br/>
  近三年来，由于郭文贵污蔑海外民运三十年来一事无成，对民运的批评声音多起来了。民运内部也有很多互相指责的声音，而且组织机构也发生了若干动荡。在郭文贵的闹剧事件中，我对民运也有很多批评，最后汇总成一篇文章：《集体乌龙球》，大家可以参考。
  <br/>
  我批评民运，绝不是郭的逻辑～说一事无成加泼脏水。民运并非一事无成，而且做了很多很有价值的工作。评价民运的标准不能设定在是否推翻中共上～以中共之强大，设定短期推翻目标并不现实。民运做成、做对和需要继续坚持的事，在我看来有如下几点：1，解构中共神话，始终告诉世人和西方，中共是个邪恶政权，人民有权力推翻和打倒。2，始终与国内的各种反抗运动站在一起，支持和声援他们。3，为各种遭受迫害的维权个人或反共英雄提供帮助。4，为新疆、西藏、香港、台湾的各种正义抗争发声支援～是支持他们的民主权利而非盲目支持独立。5，与美国及西方政府密切沟通，请求他们支援中国的民主运动。6，在各种有关中国民主的重大纪念日，用纪念和抗议，保持历史记忆和继续努力的希望。7，用各种街头抗议，唤醒来海外旅游的中国人和西方民众。8，总体上保持了多个民运组织的协同合作和一致对敌，为中国的民主运动一直保存着组织化的火种和持续提供动力。
  <br/>
  以上八条，是我正式肯定的看法。
  <br/>
  但我几年来也一直批评民运的路线问题：1，完全无视中国近年的巨大发展，也不分析它的原因，反对路线仍然建立在中共明年垮台的假设上，并且在舆论上不停地做这个假设，最后年复一年地成为舆论笑柄，失去听众。似乎推翻中共是个指日可待的短期行为，从未对斗争的长期性复杂性提出有价值的路线安排～在这一点上应好好学习共产党，看看他们是如何艰苦卓绝地打败国民党的。2，一切行动如果只建立在短期设计上，势必年年遭受挫折感，不仅失去信众，而且会把自己的焦虑投射到行动上，变得越来越极端，这是海外各种独运兴起的原因，民运如果支持独运，就等于自我否定全中国民主化的目标，是自己毁掉了民运。3，越来越不接地气～中国现实发展的地气，并以此提出新的战略和策略。越来越脱离中国民众和海外华人民众，海外民运如果连海外华人都动员不了，并让他们越来越远离你们。不是中共收买问题，而是民运关于中国的描述～明年崩溃论越来越失去听众。4，民运三十年来从未检讨过自己的路线问题，固步自封，那谈何成功？
  <br/>
  我与民运的总体路线分歧及我的主张，已经发表在给冯崇义教授的公开信中，供各位参考。
  <br/>
  二、关于台湾的前途问题
  <br/>
  台湾刚刚经历了大选。台湾大选，牵动了所有民运人士的心，基本是一边倒地支持民进党，甚至几位“民国派”的著名人士都反对国民党。民进党在大选中成功地将国民党抹红，指称他们卖台。并以香港反送中事件为例，将大选宣扬为阻挡大陆侵蚀和变成第二个香港的阻击战。民进党这个选战的策略取得了巨大成功，以八百多万票对五百多万票的战绩赢得了大选。但民进党刚刚宣布胜选，大家就在蔡英文总统的胜选讲话中，体会到了另外的味道：她使用民主手段赢得了中华民国总统席位，却一上台就在讲话中修改了中华民国的国号，变成了“中华民国台湾”。即大家都以为自己在选一位保持中华民国法统的总统，但事实上选出了一位要以渐进台独为手段、最终实现“台湾国”建国理念的总统。而这一点，在她的大多数支持者中实际是心知肚明的。这意味着，台湾的未来及台海关系迎来了一个转折点。
  <br/>
  而国民党继70年前输掉了大陆，70年后又在台湾输光了民国～因为国号已改，成了“中华民国台湾”了。这不仅仅是一次败选。
  <br/>
  我在价值观上不反对任何独立理念。但民进党的渐进台独操作，对台湾人民是福还是祸却是一个需要认真讨论的问题，也是民运需要正视和表明立场的问题——民运是要完成整个中国的民主化，包括在这个过程中应用台湾的民主经验及宪法并将台湾纳入民主统一的轨道？还是一边支持大陆民主，同时支持台独？如果采取后一种立场，民运是否还能得到大陆人民的支持？在这个问题上的立场，实际上关乎民运的生死存亡。
  <br/>
  仅就台湾的前途而言，是渐进台独操作能最终使台湾摆脱“大陆魔掌”，还是努力促使和参与大陆民主化才是台湾长治久安的保障？民进党似乎认为渐进台独没问题，因为有美国的《台湾关系法》保障，大陆不敢怎么样。赖清德副总统说：台海的定海神针铁不是九二共识，而是美国的与台湾关系法。可见民进党的台独定力，完全是建立在美国帮忙基础上的。但在我看来，这是一种不靠谱的一种心理依仗，最终必将给台湾带来灾难。
  <br/>
  台湾长治久安的唯一出路就是促成大陆民主化——这也应是民运的立场。任何悄悄逃离的操作，最终都无法成功并带来灾难。大陆的统一决心和能力是显而易见的，美国的驰援最终一定是靠不住的。
  <br/>
  为了实现台湾的长久安全，我对台湾提出了以攻为守的建议——也是给国民党的起死回生建议。
  <br/>
  以攻为守：大陆提一国两制，台湾就应该向对岸提出一国一制，即一个中华民国，一个以47宪法为基础的民主制度。即在对岸同意以47宪法精神（此宪法本来就是国民党、共产党和民盟三党联合起草的）改造大陆宪法的前提下，同意大陆承接中华民国招牌和法统。在一定过渡期和国际检验及见证下实现统一。民进党也可以用上述建议，将本党的宗旨改成“民主统一也是未来选项之一”。
  <br/>
  上述建议一提出，就立刻化解了大陆武力攻台的理由，并被迫要考虑“民主统一”的条件。从而无法坚持专制大一统的蛮横统一。台湾只要提出以大陆民主化作为统一的前提，而且最好由国民党和民进党联合提出，台海两岸统一议题的球就踢回给大陆了。而且一定会得到世界舆论的普遍支持。这个以攻为守的策略，是台湾目前的最佳选择。就看岛内两党是否有这个大智慧了。
  <br/>
  为此，我在2019年10月发表的《中华民国丢失大陆70年祭及战略检讨》一文的后面加了一个附件：《大中华邦联方案》，这个方案已经将我提出的以攻为守的策略方案化了，供参考。
 </p>
