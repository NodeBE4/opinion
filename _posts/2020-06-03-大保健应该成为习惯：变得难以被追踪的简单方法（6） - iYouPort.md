  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在数据泄露时代维护自己
   </strong>
  </h3>
  <p class="graf graf--p">
   在数据泄露时代，我们如何保护自己的个人身份信息（PII）？
  </p>
  <p class="graf graf--p">
   为了在似乎每周都有新数据泄露的情况下保持低调，您必须加倍谨慎，
   <strong class="markup--strong markup--p-strong">
    尽可能不要将个人信息放在网上
   </strong>
   。
  </p>
  <p class="graf graf--p">
   👉做到这一点的方法是尽可能地在线使用有关您自己的虚假信息 ——
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%bf%ab%e9%80%9f%e5%88%9b%e5%bb%ba%e4%b8%80%e4%b8%aa%e5%81%87%e4%ba%ba%ef%bc%9f%e7%ae%80%e6%98%93%e7%89%88%e5%88%86%e8%ba%ab%e6%9c%af%e2%80%8a-%e2%80%8a%e8%b0%83%e6%9f%a5%e4%ba%ba/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%bf%ab%e9%80%9f%e5%88%9b%e5%bb%ba%e4%b8%80%e4%b8%aa%e5%81%87%e4%ba%ba%ef%bc%9f%e7%ae%80%e6%98%93%e7%89%88%e5%88%86%e8%ba%ab%e6%9c%af%e2%80%8a-%e2%80%8a%e8%b0%83%e6%9f%a5%e4%ba%ba/" rel="noopener noreferrer" target="_blank">
    袜子木偶
   </a>
   、
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%a7%92%e8%89%b2%e6%89%ae%e6%bc%94%ef%bc%9a%e8%a1%8c%e5%8a%a8%e8%80%85%e5%88%9a%e9%9c%80video/" href="https://www.iyouport.org/%e8%a7%92%e8%89%b2%e6%89%ae%e6%bc%94%ef%bc%9a%e8%a1%8c%e5%8a%a8%e8%80%85%e5%88%9a%e9%9c%80video/" rel="noopener noreferrer" target="_blank">
    角色扮演
   </a>
   、
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%88%86%e8%a3%82%e4%ba%ba%e6%a0%bc%e7%9a%84%e9%ad%85%e5%8a%9b/" href="https://www.iyouport.org/%e5%88%86%e8%a3%82%e4%ba%ba%e6%a0%bc%e7%9a%84%e9%ad%85%e5%8a%9b/" rel="noopener noreferrer" target="_blank">
    分身
   </a>
   。链接中是具体方法。
  </p>
  <p class="graf graf--p">
   为什么要使用您的真实姓名、家庭住址和电话号码呢？您知道恶意者可以用这些信息造成多么大的损失吗？很多，完全可以超乎你的想象。
  </p>
  <p class="graf graf--p">
   已经2020年，但你仍然可以在网上找到大量的个人身份信息，几乎俯拾皆是。这是多么恐怖？
  </p>
  <p class="graf graf--p">
   这个世界是一个巨大的舞台，对于那些有足够感知力的人来说，到处都是数字面包屑证据、到处都是微妙的线索、随之而来的是数字足迹。
  </p>
  <p class="graf graf--p">
   许多不同类型的人出于多种不同目的使用
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    开源情报工具
   </a>
   、方法和技术。有些是网络威胁猎人，有些是罪犯分子，有些将其用于数字取证目的，其他人可能只是想追踪过去的敌人，或
   <strong class="markup--strong markup--p-strong">
    用来确定有多少关于自己的个人信息在线 —— 用敌人的眼睛看自己。
   </strong>
  </p>
  <p class="graf graf--p">
   我们提供开源情报技术为的正是最后一种目的。
  </p>
  <p class="graf graf--p">
   开源情报的的光圈可以根据您的需求任意扩大或缩小。有时搜索范围很广且很深，而其他时候搜索范围又很窄也很浅。
  </p>
  <p class="graf graf--p">
   开源情报就是要知道要为特定任务使用哪些工具，这些工具将产生最佳结果，或者要使用哪些工具的最佳组合。毕竟，您可能不会使用高射炮打鸟。开源情报也一样，
   <strong class="markup--strong markup--p-strong">
    根据您想要定位的信息，某些工具会比其他工具更好
   </strong>
   。
  </p>
  <p class="graf graf--p">
   开源情报经历了一波高潮，导致大量新软件工具和技术涌现，几乎任何人都可以使用它来收集关于另一个人或组织的“情报”。
  </p>
  <p class="graf graf--p">
   有很多软件工具和资源可用于此类调查活动，可以参见
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    我们的专栏
   </a>
   ，非常多。
   <strong class="markup--strong markup--p-strong">
    但是，某些最好的开源情报工具并不是完全设计为工具的 —— 社交媒体本身就可以是最出色的开源情报工具之一
   </strong>
   。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    社交媒体
   </strong>
  </h3>
  <p class="graf graf--p">
   如果您重视数字隐私，并且使用 Linkedin 之类的社交媒体网站与上班族联系，那么您面临的侵犯隐私权的可能性比您想象的要大得多。
  </p>
  <p class="graf graf--p">
   在 Linkedin 上，除非您正在积极发布有关您自己的
   <strong class="markup--strong markup--p-strong">
    虚假信息
   </strong>
   以摆脱追踪者的困扰，否则，任何人都可以通过您向雇主转发的照片、帖子或资料来查找您的私密信息并可能对您发起攻击，以试图使您被解雇或至少受到谴责。
  </p>
  <p class="graf graf--p">
   您为什么要给一个您甚至都不知道是谁的人如何严重的机会？
  </p>
  <p class="graf graf--p">
   根据您的个性，与社交媒体上的某人进行火拼可能只是时间问题。
  </p>
  <p class="graf graf--p">
   您可能是最好、最可爱的人，但是，如果某人不喜欢您写的或发布的内容，那么对您发起战争就是他们的首选项。
  </p>
  <p class="graf graf--p">
   他们将不惜一切代价破坏您的整个生存。只要看看最近的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.huffingtonpost.com/entry/deadly-prank-endangering-lives_us_5b17fca6e4b09578259e132b" href="https://www.huffingtonpost.com/entry/deadly-prank-endangering-lives_us_5b17fca6e4b09578259e132b" rel="noopener nofollow noreferrer" target="_blank">
    Swatting
   </a>
   攻击如何导致无辜受害者死亡就可以明白了。
  </p>
  <p class="graf graf--p">
   当事情趋向于升级时，您认为那些讨厌您的人会转向什么呢？毫无疑问。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%bd%93%e4%b8%80%e4%b8%aa%e9%99%8c%e7%94%9f%e4%ba%ba%e6%83%b3%e8%a6%81%e6%af%81%e6%8e%89%e4%bd%a0%e7%9a%84%e7%94%9f%e6%b4%bb%e6%9d%a5%e8%87%aa%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93/" href="https://www.iyouport.org/%e5%bd%93%e4%b8%80%e4%b8%aa%e9%99%8c%e7%94%9f%e4%ba%ba%e6%83%b3%e8%a6%81%e6%af%81%e6%8e%89%e4%bd%a0%e7%9a%84%e7%94%9f%e6%b4%bb%e6%9d%a5%e8%87%aa%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93/" rel="noopener noreferrer" target="_blank">

  <p class="graf graf--p">
   坦白地说，您无需花大量的网络经验就可以了解开源情报的信息收集工具和技术。换句话说，您**不必**成为开源情报界专家就可以在短期内让某个人再也爬不起来。所以，您觉得其他人不会这样做吗？
  </p>
  <p class="graf graf--p">
   易于追踪就是这个互联网的工作方式，如果您必须使用它，请在这里至少以一个完全不同的身份存在。

  <p class="graf graf--p">
   <strong>
    1、Freenet
   </strong>
  </p>
  <p class="graf graf--p">
   每当人们想到匿名浏览（尤其是对于暗网）时，就会想到 Tor。但是，还有其他选择，例如 I2P 和
   <a class="markup--anchor markup--p-anchor" data-href="https://freenetproject.org/pages/about.html" href="https://freenetproject.org/pages/about.html" rel="noopener noreferrer" target="_blank">
    Freenet Project
   </a>
   。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11930 jetpack-lazy-image" data-lazy-sizes="(max-width: 1036px) 100vw, 1036px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3.png?resize=1036%2C835&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3.png?w=1036&amp;ssl=1 1036w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3.png?resize=300%2C242&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3.png?resize=1024%2C825&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3.png?resize=768%2C619&amp;ssl=1 768w" data-recalc-dims="1" height="835" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3.png?resize=1036%2C835&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1036"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11930" data-recalc-dims="1" height="835" sizes="(max-width: 1036px) 100vw, 1036px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3.png?resize=1036%2C835&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3.png?w=1036&amp;ssl=1 1036w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3.png?resize=300%2C242&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3.png?resize=1024%2C825&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3.png?resize=768%2C619&amp;ssl=1 768w" width="1036"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Freenet 是一种免费软件，可让您匿名共享文件、浏览和发布 “freesites”（只能通过 Freenet 访问的网站）以及在论坛上聊天，而不必担心审查。
  </p>
  <p class="graf graf--p">
   Freenet 节点之间的通信经过加密，并通过其他节点进行路由，这使得确定谁在请求信息及其内容是什么变得极为困难。
  </p>
  <p class="graf graf--p">
   文件会根据其受欢迎程度自动保留或删除，而最不受欢迎的文件将被丢弃，以便为更新或更受欢迎的内容让路。
  </p>
  <p class="graf graf--p">
   文件是加密的，通常难以追踪；聊天论坛，网站和搜索功能均基于此分布式数据存储构建。
  </p>
  <p class="graf graf--p">
   请注意：即使加密并仅临时存储在计算机硬盘上，它仍然可以导致严重后果；您如何向政府间谍解释，在您家门口搜出的所谓的非法内容不是您的？
  </p>
  <p class="graf graf--p">
   您是否仍然天真地认为检察官或法官会关心 Freenet 分布式数据共享功能的微妙错综复杂性？您是否不同意 Freenet 的使用条款并自行承担下载软件的风险？最好先想清楚这个问题。
  </p>

    2、I2P

  <p class="graf graf--p">
   I2P 与 Tor 十分相似，您可将其视为“匿名覆盖网络 — 网络中的网络”。它旨在保护通信不受诸如ISP之类的第三方进行的窥探和监视。
  </p>
  <p class="graf graf--p">
   I2P 被许多关心隐私的人使用：活动家、被压迫的人、新闻工作者、透明度革命英雄、以及想要人权的普通人。
  </p>
  <p class="graf graf--p">
   任何网络都不能 “绝对匿名”。I2P 的持续目标是使攻击越来越难以进行。随着网络规模的扩大以及正在进行的学术审查，其匿名性将越来越强。
  </p>
  <p class="graf graf--p">
   I2P可在台式机，嵌入式系统（如 Raspberry Pi）和 Android 手机上使用。
  </p>
  <p class="graf graf--p">
   如果您不喜欢Tor，可以考虑尝试使用I2P。
  </p>

   我们在 《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%bb%ba%e7%ab%8b%e4%b8%80%e4%b8%aa%e5%8c%bf%e5%90%8d-facebook-%e8%b4%a6%e6%88%b7%e4%bb%a5%e4%bf%9d%e6%8a%a4%e4%b8%aa%e4%ba%ba%e6%95%b0%e6%8d%ae%e7%9a%84%e7%a7%81%e5%af%86%e6%80%a7/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%bb%ba%e7%ab%8b%e4%b8%80%e4%b8%aa%e5%8c%bf%e5%90%8d-facebook-%e8%b4%a6%e6%88%b7%e4%bb%a5%e4%bf%9d%e6%8a%a4%e4%b8%aa%e4%ba%ba%e6%95%b0%e6%8d%ae%e7%9a%84%e7%a7%81%e5%af%86%e6%80%a7/" rel="noopener noreferrer" target="_blank">
    如何建立一个匿名社交媒体账户以保护个人数据的私密性？
   </a>
   》介绍过更详细的方法。
  </p>
  <p class="graf graf--p">
   很多网站要求您提供名字和电子邮件地址。要知道，没人强制要求您提供真实的名字和电邮，所以，你为什么要给他们真的呢？
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    无人能对攻击免疫
   </strong>
  </h3>
  <p class="graf graf--p">
   良好的生活习惯可以降低您无意间感染流行病的风险；这在互联网上也一样，良好的行为习惯能帮助您避免很多常见的攻击。
  </p>
  <p class="graf graf--p">
   请记住，没有人能对网络攻击完全免疫，这里没有100%的安全，但安全是一个相对的概念，您只需要做到让自己比其他很多人都更安全，就可以了。
  </p>
  <p class="graf graf--p">
   以下几个方法
   <strong class="markup--strong markup--p-strong">
    应该成为您的日常习惯
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    A：学会跟进
   </strong>
  </p>
  <p class="graf graf--p">
   识别网络攻击已经变得越来越困难。带有恶意链接的消息
   <strong class="markup--strong markup--p-strong">
    并不
   </strong>
   总是来自陌生人的。
  </p>
  <p class="graf graf--p">
   它们可能看起来就像是例行交流，或者似乎起源于您认识或与之合作的人。
  </p>
  <p class="graf graf--p">
   攻击使用的是人类微妙的但根深蒂固的认知偏见 —— 人性的 bugs。这就是为什么我们要强调您理解 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
    社交工程
   </a>
   ” 栏目中介绍的内容，了解人类固有的弱点，以及防御方法。
  </p>
  <p class="graf graf--p">
   还记得这个案例吗？👉攻击者伪装成国际大牌媒体记者和著名人权组织工作人员，通过加密通讯攻击敏感人士，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ad%a6%e5%91%8a%ef%bc%9awhatsapp%e6%96%87%e6%9c%ac%e6%b5%81%e4%b8%ad%e7%9a%84%e6%81%b6%e6%84%8f%e9%93%be%e6%8e%a5/" href="https://www.iyouport.org/%e8%ad%a6%e5%91%8a%ef%bc%9awhatsapp%e6%96%87%e6%9c%ac%e6%b5%81%e4%b8%ad%e7%9a%84%e6%81%b6%e6%84%8f%e9%93%be%e6%8e%a5/" rel="noopener noreferrer" target="_blank">
    警告：WhatsApp文本流中的恶意链接
   </a>
   》。
  </p>
  <p class="graf graf--p">
   值得庆幸的是，您可以使用一种简单的低技术方法来阻止这些攻击：在信任之前先跟进。
  </p>
  <p class="graf graf--p">
   具体说就是：如果信息显示它来自您认识的人，去给那个人打个电话，或直接走到他/她面前，问一句：“这个是你发给我的吗？”
  </p>
  <p class="graf graf--p">
   如果是您不认识的人，一概不接受就行了。
  </p>
  <p class="graf graf--p">
   也是为什么我们说，
   <strong class="markup--strong markup--p-strong">
    您和您的队友之间应该有多种不同的联系方式
   </strong>
   ，包括不同的通讯平台、电话、
   <strong class="markup--strong markup--p-strong">
    只有你们双方知道的
   </strong>
   会面地点和暗号等等。攻击者可能霸占其中一个，但很难霸占全部。
  </p>
  <p class="graf graf--p">
   如果消息是真实的，那么请过几分钟再进行仔细检查，以确保没有任何危害。
  </p>
  <p class="graf graf--p">
   如果不是这样，您应该立刻发出警报：您的熟人已经被入侵，以提醒相关的其他人小心。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    B：主动使用并鼓励其他人使用端到端加密
   </strong>
  </p>
  <p class="graf graf--p">
   仅仅您一个人使用安全通信是没用的，您应该鼓励您的朋友、同事、亲属都切换到 Signal 等应用程序。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    没有人能独自一个人做到安全
   </strong>
   ，《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%a6%81%e6%83%b3%e4%bf%9d%e6%8a%a4%e6%af%8f%e4%b8%aa%e4%ba%ba%e7%9a%84%e4%ba%ba%e6%9d%83%e5%92%8c%e5%ae%89%e5%85%a8%ef%bc%8c%e6%88%91%e4%bb%ac%e5%b0%b1%e9%9c%80%e8%a6%81%e8%81%94%e5%90%88/" href="https://www.iyouport.org/%e8%a6%81%e6%83%b3%e4%bf%9d%e6%8a%a4%e6%af%8f%e4%b8%aa%e4%ba%ba%e7%9a%84%e4%ba%ba%e6%9d%83%e5%92%8c%e5%ae%89%e5%85%a8%ef%bc%8c%e6%88%91%e4%bb%ac%e5%b0%b1%e9%9c%80%e8%a6%81%e8%81%94%e5%90%88/" rel="noopener noreferrer" target="_blank">
    要想保护每个人的人权和安全，我们就需要联合
   </a>
   》。
  </p>
  <p class="graf graf--p">
   这并不意味着您必须完全停止使用其他通信方法，而是将其视为层次结构。
  </p>
  <p class="graf graf--p">
   使用 Signal 表示应受信任的重要消息，仅对 “无关紧要” 的通信使用所有其他消息传递方法，例如SMS或社交网站。
  </p>
  <p class="graf graf--p">
   所以，
   <strong class="markup--strong markup--p-strong">
    如果看起来 *很重要* 的请求或链接通过被定位为 *不重要* 的渠道出现在您面前，您应该立刻亮起红灯
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    C：不要随便把什么东西插在自己的电子设备上
   </strong>
  </p>
  <p class="graf graf--p">
   如果您在人行道的垃圾箱里捡到一个牙刷，您会把它塞进自己嘴里吗？
  </p>
  <p class="graf graf--p">
   USB设备也是同样的道理。
  </p>
  <p class="graf graf--p">
   也许您捡到一个拇指驱动器时会反思一下要不要插进自己的设备，但是，
   <strong class="markup--strong markup--p-strong">
    绝大多数人仍然不会怀疑公用的移动设备充电器，和路边小摊卖的USB电缆，这是很奇怪的事。
   </strong>
  </p>
  <p class="graf graf--p">
   与电子邮件和一些文件共享服务会在文件到达计算机之前对其进行扫描和过滤不同，通过USB插入是直接的且不受保护的。
  </p>
  <p class="graf graf--p">
   建立此连接后，用户无需执行任何其他操作即可发生许多可能的灾难。
  </p>
  <p class="graf graf--p">
   通过USB连接，恶意软件和勒索软件等问题很容易感染您的设备。
  </p>
  <p class="graf graf--p">
   无需宣誓USB连接的便利性，也不必完全避免使用这些设备。但您在任何一次插入之前都应该三思。如果要插入计算机，请确保格外小心。
  </p>
  <p class="graf graf--p">
   请直接从制造商或有信誉的公司或具有供应链控制权的经销商处购买USB电缆，而不是街边小店和淘宝。充电时应该使用带有USB端口的壁式充电器。
  </p>
  <p class="graf graf--p">
   很有必要养成健康的生活习惯，这是防卫的最佳姿势 —— 将一些认真的网络安全实践纳入您的新10年决议中吧，立即开始实施它们。⚪️
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    —— 未完待续 ——
   </em>
  </p>
