  <p class="graf graf--p">
   示威者用贴纸覆盖身体以确保展示的乳头是“男性”，符合 Facebook 政策。
  </p>
  <p class="graf graf--p">
   有些毛茸茸的，有些尖尖，有些是深褐色的，有些是淡粉色的……但周日在 Facebook 纽约市总部门前展出的数百个乳头在技术上都是“男性”，尽管事实上有些乳头是女性抗议者。
  </p>
  <p class="graf graf--p">
   超过100人裸体躺在人行道上，要求改变 Facebook 公司的审查政策。
  </p>
  <p class="graf graf--p">
   这项名为 #wethenipple 的行动是由艺术家 Spencer Tunick 和全国反审查联盟组织的。
  </p>
  <p class="graf graf--p">
   示威者用男性乳头贴纸盖住自己的身体，以抗议 Facebook 和 Instagram 禁止平台上的“女性乳头照片”。
  </p>
  <p class="graf graf--p">
   活动人士说，禁令会伤害艺术家和用户对性别和身份的探索。
  </p>
  <p class="graf graf--p">
   在2014年被 Facebook 封锁主页后，以裸体摄影而闻名的 Tunick 多年来一直在与这些政策作斗争。
  </p>
  <p class="graf graf--p">
   他过去曾组织过类似的抗议模式，其中包括2007年的一项行动，其中一系列志愿者裸体站在阿尔卑斯山的冰川上，作为抵抗全球变暖运动的一部分。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “作为一个21世纪的艺术家，我依靠Instagram。这是世界杂志，任何审查都会挫败我的锐气”，他说。
  </p>
  <p class="graf graf--p">
   在这里看到抗议视频：
   <a class="markup--anchor markup--p-anchor" data-href="https://vimeo.com/341151424" href="https://vimeo.com/341151424" rel="nofollow noopener noreferrer" target="_blank">
    https://vimeo.com/341151424
   </a>
  </p>
  <p class="graf graf--p">
   抗议者持有的乳头图像取自男性名人和艺术家“捐赠”的照片，包括 Bravo 的 Andy Cohen，艺术家 Andres Serrano，演员摄影师 Adam Goldberg，Red Hot Chili Peppers 鼓手 Chad Smith 和 Tunick 本人。
  </p>
  <p class="graf graf--p">
   全国反审查联盟向 Facebook 发送了一封信，其中概述了其要求，这封信由250多个激进组织和博物馆签署，包括电子前沿基金会、学术博物馆、和画廊协会、以及 Leslie-Lohman 男女同性恋艺术博物馆。
  </p>
  <p class="graf graf--p">
   他们呼吁 Facebook 和 Instagram 制定类似于 YouTube 的政策，该政策在十多年前改变了规则，允许一些艺术裸体展出。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “裸体禁令阻止了许多艺术家在网上分享他们的作品，”NCAC 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这特别伤害了那些以自己的身体为重点的艺术家，包括 queer 和非常规性别的艺术家，以及他们社区中的人。博物馆和画廊甚至在推广以裸体为主题的展览时都会受到限制。”
  </p>
  <p class="graf graf--p">
   用户过去曾对 Facebook 和 Instagram 的政策提出过质疑，质疑该公司如何定义“女性乳头”，并将男性乳头PS到自己的裸体照片上。
  </p>
  <p class="graf graf--p">
   Facebook 没有回复评论请求。
  </p>
  <p class="graf graf--p">
   该公司近年来修改了其政策，允许一些图像，“包括那些描绘抗议行为的女性、积极参与母乳喂养的女性，以及乳房切除术后瘢痕形成的照片”。
  </p>
  <p class="graf graf--p">
   截至报道发布时，这场抗议行动的照片和视频仍保留在 Instagram 上。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “Facebook 或 Instagram 没有任何理由审查这个视频或阻止主题标签，”Tunick 在 Instagram 标题中说。
  </p>
