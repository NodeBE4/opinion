  <p class="graf graf--p">
   2018年8月的一个晚上，年仅21岁的 Mikhail Aksel 踏上了莫斯科地铁 Sportivnaya 车站壮丽的大理石台阶。
  </p>
  <p class="graf graf--p">
   Aksel 是公民运动组织 “另一个俄罗斯” 的资深活动家，这是一个与著名作家流亡作家 Eduard Limonov 有联系的小规模但华丽的反对党，他们对与警察发生冲突并不陌生。
  </p>
  <p class="graf graf--p">
   但即便如此，当警察把他带到派出所并要求查看他的证件时，Aksel 仍感到惊讶。他被告知，
   <strong class="markup--strong markup--p-strong">
    该站的安全系统已将他识别为 “通缉犯”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   当 Aksel 对此表示抗议说他没有做错任何事时，他被送到派出所的警察办公室，
   <strong class="markup--strong markup--p-strong">
    那里有一个大屏幕 —— 上面显示了他的个人资料，其中详细列出了他的姓名、出生日期和住址
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li graf--startsWithDoubleQuote">
    “一整面墙上布满了显示器。那些数据被整合成了一个详细的人格分析，以此来预测逃离者最有可能做出什么样的选择，然后提前下手围堵。这些构成你的人格分析图解的数据都来自于公开信息，来自你平时完全没有留意到就泄漏出去了的信息。它们最终会变成等着你的手铐”……《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e9%9a%90%e8%ba%ab%e6%b8%b8%e6%88%8f/" href="https://www.iyouport.org/%e9%9a%90%e8%ba%ab%e6%b8%b8%e6%88%8f/" rel="noopener noreferrer" target="_blank">
     隐身游戏：如何在监视遍地的社会里让自己完全消失
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   个人资料中显示没有案件编号、没有调查人员、也没有指控。唯一提供的其他信息是，内务部 “打击极端主义中心” 的一名官员已将 Aksel 的名字添加到了系统的数据库中。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     把公民活动家标记为 “极端主义分子” 可不是俄罗斯的专利
    </strong>
    ，见《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%88%91%e8%a2%ab%e8%ad%a6%e5%af%9f%e6%8a%98%e7%a3%a8%e8%87%b3%e7%b2%be%e7%a5%9e%e5%b4%a9%e6%ba%83%ef%bc%9a%e7%8e%b0%e5%9c%a8%e6%88%91%e5%bf%85%e9%a1%bb%e7%88%ac%e8%b5%b7%e6%9d%a5%e6%88%98%e6%96%97/" href="https://www.iyouport.org/%e6%88%91%e8%a2%ab%e8%ad%a6%e5%af%9f%e6%8a%98%e7%a3%a8%e8%87%b3%e7%b2%be%e7%a5%9e%e5%b4%a9%e6%ba%83%ef%bc%9a%e7%8e%b0%e5%9c%a8%e6%88%91%e5%bf%85%e9%a1%bb%e7%88%ac%e8%b5%b7%e6%9d%a5%e6%88%98%e6%96%97/" rel="noopener noreferrer" target="_blank">
     我被警察折磨至精神崩溃：现在我必须爬起来战斗
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   Aksel 回忆说：“警察告诉他，看，如果只是行政逮捕，你的详细信息将在此处显示为灰色。但是在这里，已经以红色突出显示，并带有警告提示”。不过一个匆忙的电话后，Aksel 被释放了。
  </p>
  <p class="graf graf--p">
   就这样 Mikhail Aksel 偶然发现了俄罗斯正在酝酿的面部识别监控系统，这是一个由AI连接的摄像头网络，预计很快将成为
   <strong class="markup--strong markup--p-strong">
    全世界最大的
   </strong>
   监视摄像头网络之一。
  </p>
  <p class="graf graf--p">
   Aksel 被逮捕的那个地铁站 Sportivnaya 通向莫斯科的旗舰卢日尼基（Luzhniki）体育场。在2018年FIFA世界杯前夕，该站成为俄罗斯在全国范围内推出人脸识别技术的原爆点。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    普京曾经说，”成为 AI 领域领导者的人将成为整个世界的统治者。” 俄罗斯在此是怎么做的？为什么与北京如此相似？《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%8c%97%e4%ba%ac%e6%a8%a1%e5%bc%8f/" href="https://www.iyouport.org/%e5%8c%97%e4%ba%ac%e6%a8%a1%e5%bc%8f/" rel="noopener noreferrer" target="_blank">
     北京模式
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   在世界杯期间，从繁华的莫斯科和圣彼得堡到萨兰斯克（人口：320,000）的11个主办城市的体育场内及周围，
   <strong class="markup--strong markup--p-strong">
    都使用了神经网络图像处理技术来识别、跟踪、并将任何可疑人与黑名单相比较
   </strong>
   。
  </p>
  <p class="graf graf--p">
   据报道，在通过面部识别算法识别出超过180名规则破坏者后，他们被逐出世界杯比赛看台。
  </p>
  <p class="graf graf--p">
   甚至在世界杯结束粉丝离开后，该监视技术系统仍然悄悄地保留在原地。实际上，
   <strong class="markup--strong markup--p-strong">
    世界杯开始了俄罗斯对面部识别监视技术的大量投资
   </strong>
   。
  </p>
  <p class="graf graf--p">
   2018年，莫斯科地铁宣布对面部识别摄像头进行试验，以监视火车和车站上的乘客。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    到2020年，这项技术将更加深入，可以识别出进入地铁的任何乘客，并在输出车票之前直接从其银行账户中扣款。
   </strong>
  </p>
  <p class="graf graf--p">
   在地铁站外，途径首都主要火车站和多莫杰多沃国际机场的任何莫斯科人已经在神经网络的监视之下。
  </p>
  <p class="graf graf--p">
   根据俄罗斯中央银行2018年
   <a class="markup--anchor markup--p-anchor" data-href="https://www.planetbiometrics.com/article-details/i/7273/desc/russian-central-bank-starts-using-biometric-customer-database/" href="https://www.planetbiometrics.com/article-details/i/7273/desc/russian-central-bank-starts-using-biometric-customer-database/" rel="noopener noreferrer" target="_blank">
    宣布的一项计划
   </a>
   ，面部识别软件将控制消费者在全国范围内对银行分支机构、在线银行和政府服务的访问
   <strong class="markup--strong markup--p-strong">
    权限
   </strong>
   ，例如 税收处理、社保付款和护照续签。
  </p>
  <p class="graf graf--p">
   同时，该国自己的面部识别监控摄像头网络正在迅速发展。面部识别试验于2017年在莫斯科开始。
  </p>
  <p class="graf graf--p">
   不到两年后，市政府认为该实验成功了，声称已抓获了200多名通缉犯。2019年5月，莫斯科宣布招标，在全市多达 200,000 个监控摄像头中安装面部识别软件，到2019年底将连接 105,000 个。
  </p>
  <p class="graf graf--p">
   综上所述，这将使俄罗斯成为全球公认的最大的面部识别摄像头网络之一。
  </p>
  <p class="graf graf--p">
   根据一些预测，它甚至可能
   <strong class="markup--strong markup--p-strong">
    大于中国的2亿个摄像头系统
   </strong>
   。
  </p>
  <p class="graf graf--p">
   驻莫斯科的中国独立观察家和技术分析师 Leonid Kovachich 说：“不可能推测中国或俄罗斯的能力究竟谁更大。因为没有人真正知道中国实际上有多少台监视摄像机与面部识别技术相连。”
  </p>

  <p class="graf graf--p">
   尽管俄罗斯对面部识别技术的关注远没有中国的监视体系结构引起的重视更多，但莫斯科处于充分利用大规模监视手段的主要位置。
  </p>
  <p class="graf graf--p">
   俄罗斯在数学上有卓越的历史，苏联和俄罗斯的数学家被授予9次菲尔兹奖章，仅次于法国和美国。这种
   <strong class="markup--strong markup--p-strong">
    专业知识
   </strong>
   使俄罗斯成为拥有世界一流的本土人工智能和面部识别部门的极少数国家之一。
  </p>
  <p class="graf graf--p">
   但是，俄罗斯在AI方面的成功也是国家对该领域感兴趣的产物。克里姆林宫多年来一直意识到该部门的潜在安全优势：
   <strong class="markup--strong markup--p-strong">
    早在2011年
   </strong>
   ，莫斯科地铁就进行了面部识别技术的试验。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    人脸识别技术最初是迪士尼主题公园用的，用来帮游客挑照片做纪念品。而
    <strong class="markup--strong markup--li-strong">
     在占领华尔街运动其间，这项技术已经被用来抓捕抗议者了
    </strong>
    。《

  <h3>
   <span style="color: #339966;">
    <strong>
     自产AI
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   同时，
   <strong class="markup--strong markup--p-strong">
    克里姆林宫将整个人工智能领域
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.defenseone.com/technology/2019/09/whats-russias-national-ai-strategy/159740/" href="https://www.defenseone.com/technology/2019/09/whats-russias-national-ai-strategy/159740/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     确定为
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    俄罗斯可以在某些国家援助下成功与外国竞争对手竞争的领域
   </strong>
   。
  </p>
  <p class="graf graf--p">
   2017年，弗拉基米尔·普京（Vladimir Putin）
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theverge.com/2017/9/4/16251226/russia-ai-putin-rule-the-world" href="https://www.theverge.com/2017/9/4/16251226/russia-ai-putin-rule-the-world" rel="noopener noreferrer" target="_blank">
    告诉
   </a>
   一群俄罗斯学生，成为世界AI领导者的国家将成为“世界的统治者”。
  </p>
  <p class="graf graf--p">
   NIST在人脸识别算法的全球排名中被广泛认为是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nist.gov/news-events/news/2018/11/nist-evaluation-shows-advance-face-recognition-softwares-capabilities" href="https://www.nist.gov/news-events/news/2018/11/nist-evaluation-shows-advance-face-recognition-softwares-capabilities" rel="noopener noreferrer" target="_blank">
    行业标准
   </a>
   ，⚠️
   <strong class="markup--strong markup--p-strong">
    在前十名中有两家俄罗斯公司 —— NTech Lab 和 VisionLabs —— 是仅有的来自中国或美国以外的第三国，VisionLabs 经常与中国的 DeepGlint 竞争第一名
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这些公司之一，NTech Lab，曾经在全球享有盛名。
  </p>
  <p class="graf graf--p">
   2016年，NTech Lab 发布了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.digitaltrends.com/photography/findface-social-networks-detect-people-public-with-70-reliability/" href="https://www.digitaltrends.com/photography/findface-social-networks-detect-people-public-with-70-reliability/" rel="noopener noreferrer" target="_blank">
    FindFace 应用程序
   </a>
   ，这是一款面向消费者的面部识别服务，可从俄罗斯的 Facebook 等价物 VKontakte 中提取数据，并可以使用手机摄像头通过将其与 VKontakte 的
   <strong class="markup--strong markup--p-strong">
    2亿个人资料
   </strong>
   进行匹配来识别人脸。
  </p>
  <p class="graf graf--p">
   在短暂的宣传风暴后，NTech 退出了 FindFace 的公共访问，并宣布将基础技术重定向到“安全和零售领域的全球项目”。
  </p>
  <p class="graf graf--p">
   最近，NTech 与 VisionLabs 一起开始了各种私营面部识别监视项目。
  </p>
  <p class="graf graf--p">
   俄罗斯的两家领先的面部识别公司 VisionLabs 和 NTech Lab
   <strong class="markup--strong markup--p-strong">
    都获得了大型国有公司的帮助
   </strong>
   —— 国有银行 Sberbank 在2017年购买了 VisionLabs 25％的股份，而 NTech 的12.5％的股份由俄罗斯国防部拥有的工业集团 Rostec 持有。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    私人专业知识和国家慷慨投资的结合，在不受外国竞争对手（尤其是中国竞争对手）影响的情况下，培育并保护了俄罗斯面部识别程序的基础技术
   </strong>
   。
  </p>
  <p class="graf graf--p">
   中国观察家 Leonid Kovachich 表示：“虽然其他前苏联国家（如乌兹别克斯坦和塔吉克斯坦）一直在购买完整的中国面部识别解决方案，而俄罗斯依靠中国提供硬件，但仅使用本地的算法和软件。他们倾向于认为，即便从国家安全的角度来看，购买外国硬件也不是什么威胁。”
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     未来的形状
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   但是，尽管俄罗斯有意限制中国在面部识别领域的影响力，但两个AI巨头之间还是有很多相似之处的。
  </p>
  <p class="graf graf--p">
   在这两个国家中，欠发达的数据保护法律都意味着相关研究变得更加容易，因为AI公司能够
   <strong class="markup--strong markup--p-strong">
    购买或挖掘大量数据来训练其算法
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在俄罗斯，VKontakte的隐私政策（比其他社交网络要宽松得多）为本地AI研究人员
   <strong class="markup--strong markup--p-strong">
    提供了大量的个人数据。就如中国的微信和微博
