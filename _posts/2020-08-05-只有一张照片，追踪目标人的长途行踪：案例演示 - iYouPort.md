  <p class="graf graf--p">
   社交媒体情报是开源情报的重头戏，它以其低技术性和广泛的工具利用，成为大多数初学者的入门课 — — 请注意，这其中包括政府间谍。

   <strong class="markup--strong markup--p-strong">

   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%83%85%e6%8a%a5%e5%92%8c%e5%8f%8d%e6%83%85%e6%8a%a5%e5%9f%ba%e6%9c%ac%e5%b7%a5%e5%85%b7%e6%89%8b%e5%86%8c%ef%bc%9a%e8%87%aa%e6%88%91%e4%ba%ba%e8%82%89%ef%bc%882/" href="https://www.iyouport.org/%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%83%85%e6%8a%a5%e5%92%8c%e5%8f%8d%e6%83%85%e6%8a%a5%e5%9f%ba%e6%9c%ac%e5%b7%a5%e5%85%b7%e6%89%8b%e5%86%8c%ef%bc%9a%e8%87%aa%e6%88%91%e4%ba%ba%e8%82%89%ef%bc%882/" rel="noopener noreferrer" target="_blank">
    社交媒体情报和反情报工具手册
   </a>
   》；用户名不是唯一的要害。您的简介、您的照片、您的跨平台信息共享、特定偏好、甚至语言风格，都是线索。
  </p>
  <p class="graf graf--p">
   这些追踪演示体现的是 *
   <strong class="markup--strong markup--p-strong">
    对抗性思考方式
   </strong>
   *，这是开源情报给普通人的最好的礼物。
  </p>
  <p class="graf graf--p">
   Benjamin Strick 演示了一个很棒的追踪监视，
   <strong class="markup--strong markup--p-strong">
    完全通过社交媒体情报；简单说是仅仅通过目标人晒出来的照片。
   </strong>
   在这里看到原文《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@benjamindbrown/finding-mcafee-a-case-study-on-geoprofiling-and-imagery-analysis-6f16bbd5c219" href="https://medium.com/@benjamindbrown/finding-mcafee-a-case-study-on-geoprofiling-and-imagery-analysis-6f16bbd5c219" rel="noopener noreferrer" target="_blank">
    Finding McAfee: A Case Study on Geoprofiling and Imagery Analysis
   </a>
   》。这次的追踪目标是著名企业家 John McAfee，以体现时间线上两个点的相对地理位置如何在两者之间给出可能的路径和可能的位置；即 追踪路线图。
  </p>
  <p class="graf graf--p">
   这里只有很普通的社交媒体照片。只用它来做到这点。
  </p>
  <p class="graf graf--p">
   这项追踪演示分为4个部分：
  </p>

   <li class="graf graf--li">

   <li class="graf graf--li">

   </li>
   <li class="graf graf--li">
    Point A：目标人何时何地开始旅程
   </li>
   <li class="graf graf--li">
    对 Point B 图像进行地理定位
   </li>
  </ul>
  <p class="graf graf--p">
   本案例演示中所使用的工具是完全免费的，
   <strong class="markup--strong markup--p-strong">

   </strong>
