      </strong>

  <p class="graf graf--p">
   反监视资本主义的倡导者总是指出 FATTBAG（Facebook Amazon Tencent Twitter Baidu Alibaba Google）而不是“中国还是美国” —— 因为这些寡头没有任何区别。
  </p>
  <p class="graf graf--p">
   而我们在中国听到的故事总是呈现 “Twitter/Facebook/Google 有多么自由”。当然这其中透露着很多委屈。
  </p>
  <p class="graf graf--p">
   但我们必须说，如果您真的想要获得自由，就有必要认识到强权的本性；认识到对于奴隶来说，换一个主人并不能改变被奴役的命运。
  </p>
  <p class="graf graf--p">
   有读者留言询问，“对 Twitter 封锁中国虚假消息宣传账户这件事怎么看”。事实上这件事非常简单，但
   <strong class="markup--strong markup--p-strong">
    不是这件事本身看起来的那样，而是人们应该认识到整体所呈现的样子
   </strong>
   。
  </p>
  <p class="graf graf--p">
   首先看背景，然后看细节，尤其是被大多数人所忽略的细节。
  </p>
  <p class="graf graf--p">
   抛开一些荒谬和无知的建议，许多人似乎一直在（不正确地）相信各种互联网公司拒绝压制任何东西，因为这“违背了他们的利益”。
  </p>
  <p class="graf graf--p">
   你可以从一些美国政客那里听到了这一点 —— 特别是那些寻求改变《通讯规范》法案230条款的人，他们（再一次，错误地）认为，因为第230条，某种程度上
   <a class="markup--anchor markup--p-anchor" data-href="https://www.techdirt.com/articles/20190614/20280842406/congress-now-creating-moral-panic-around-deepfakes-order-to-change-cda-230.shtml" href="https://www.techdirt.com/articles/20190614/20280842406/congress-now-creating-moral-panic-around-deepfakes-order-to-change-cda-230.shtml" rel="noopener noreferrer" target="_blank">
    没有动机
   </a>
   来缓和平台上的内容。
  </p>
  <p class="graf graf--p">
   这在多个级别上都是错误的。
  </p>
  <p class="graf graf--p">
   对于寡头来说，
   <strong class="markup--strong markup--p-strong">
    在这些平台上缓和内容存在巨大的商业、政治、道德和社会压力
   </strong>
   。如果他们弄错了，就会受到批评。他们可能会失去用户。
  </p>
  <p class="graf graf--p">
   而且（重要的是）他们可能会失去广告商、合作伙伴、客户和投资者。而230条允许他们进行相应的实验和调节。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这就是为什么
   </strong>
   Twitter 和 Facebook 都大张旗鼓地宣布封锁了似乎是滥用两个平台以推动针对香港抗议者的虚假信息的协同企图。
  </p>
  <p class="graf graf--p">
   简单说，这些寡头公司本身有动力这样做，或者有动力不去做，而不是为了香港或者人权。真的不是。
  </p>
  <p class="graf graf--p">
   尽管人们普遍认为两家公司都在努力发现并阻止这些宣传 —— 尤其是中文用户尤其这样认为。
  </p>
  <p class="graf graf--p">
   当然从来没有完美，因为大规模的内容审核
   <strong class="markup--strong markup--p-strong">
    不可能
   </strong>
   做得好。误报和漏报的错误都是不可避免的。
  </p>
  <p class="graf graf--p">
   但是，如果有人认为修改第230条会让公司在这方面做得更好，那么他们就真的错了。
  </p>
  <p class="graf graf--p">
   接下来请注意细节。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.ibtimes.co.uk/twitter-promotes-chinese-made-ads-against-hong-kong-protestors-report-1668023" href="https://www.ibtimes.co.uk/twitter-promotes-chinese-made-ads-against-hong-kong-protestors-report-1668023" rel="noopener noreferrer" target="_blank">
    Twitter 仍然允许攻击香港抗议者的用户购买其公告
   </a>
   推送服务。
  </p>
  <p class="graf graf--p">
   由中国政府喉舌新华社制作的有争议的广告正在 Twitter 上运行，公开攻击抗议引渡法案的公民。
  </p>
  <p class="graf graf--p">
   Twitter 早些时候
   <strong class="markup--strong markup--p-strong">
    已经被指责让其平台被用来影响世界各地的选举，现在正处于新风暴的中心
   </strong>
   —— 就在中文用户感激 Twitter 的同时。
  </p>
  <p class="graf graf--p">
   这不奇怪，所有寡头都在这么做，不同的只是有些寡头可以做得非常隐秘，不为人知。请参阅这一深度报告《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E7%9C%8B%E4%B8%8D%E8%A7%81%E7%9A%84%E7%8B%AC%E8%A3%81/" href="https://www.iyouport.org/%E7%9C%8B%E4%B8%8D%E8%A7%81%E7%9A%84%E7%8B%AC%E8%A3%81/" rel="noopener noreferrer" target="_blank">
    看不见的独裁
   </a>
   》。
  </p>
  <p class="graf graf--p">
   Twitter 上的煽动性广告表明香港的抗议活动是“暴力升级”。此外，广告要求“恢复秩序”。广告还强调对中国“祖国”的支持，并试图引导其受众关注香港的经济困境。
  </p>
  <p class="graf graf--p">
   该社交媒体巨头尚未就此事发表评论，但其他平台如
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/Pinboard/status/1162711159000055808" href="https://twitter.com/Pinboard/status/1162711159000055808" rel="noopener noreferrer" target="_blank">
    Pinboard 已正式指责
   </a>
   Twitter 支持中国当局。
  </p>
  <p class="graf graf--p">
   Twitter 现在面临着两难的局面，可能需要明确立场 —— 要么继续运营这些广告以支持中国政府，要么失去其声誉。
  </p>
  <p class="graf graf--p">
   如果它继续推广这些广告将具有讽刺意味，因为该社交媒体巨头在中国实际上被禁止，并且一直被中国国营的新浪微博取代。
  </p>
  <p class="graf graf--p">
   早些时候，Twitter 也被广泛地指责无视仇恨言论和政治宣传，现在，它被指责推动实际的政治宣传。
  </p>
  <p class="graf graf--p">
   自有争议的引渡法案宣布以来，香港在过去三个月中一直在进行抗议活动。抗议者一直在要求政府推翻该法案。
  </p>
  <p class="graf graf--p">
   但以上并不是全部 。
   <strong class="markup--strong markup--p-strong">
    关于 Twitter 和中国的关系，不仅仅基于香港抗议活动，还包括对维族穆斯林人口的人权侵犯
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2019/08/19/twitter-ads-china-uighurs/" href="https://theintercept.com/2019/08/19/twitter-ads-china-uighurs/" rel="noopener noreferrer" target="_blank">
    Intercept 报道
   </a>
   ：公司的广告记录审查显示，
   <strong class="markup--strong markup--p-strong">
    Twitter 促进了中国政府对新疆地区有争议的拘留营的政治宣传和虚假信息
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在新疆这个中国西部省份，联合国估计有100万维吾尔族穆斯林人口 — 包括儿童、孕妇、老人和残疾人 —— 被当局以“打击极端主义”为借口拘留。
  </p>
  <p class="graf graf--p">
   今年6月至8月期间对 Twitter 广告的审查显示，这家社交媒体巨头推动了来自中国官方媒体机构环球时报的50多条英文推文。
  </p>
  <p class="graf graf--p">
   其中一些推文故意掩盖了新疆局势的真相，并抨击了该国的政治批评者。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    环球时报支付推特广告费，向社交媒体平台上超过3亿活跃用户中的一部分推广其推文。
   </strong>
  </p>
  <p class="graf graf--p">
   这些推文出现在用户的时间线上，无论他们是否关注了环球时报帐户，都能看到。
  </p>
  <p class="graf graf--p">
   7月，就在全球谴责政府当局对新疆维吾尔族人口待遇的情况下，Twitter开 始推广多篇中国环球时报关于新疆地区的推文。
  </p>
  <p class="graf graf--p">
   其中一则
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/globaltimesnews/status/1149337184555548673" href="https://twitter.com/globaltimesnews/status/1149337184555548673" rel="noopener noreferrer" target="_blank">
    来自7月11日起被推广的推文
   </a>
   ，包括一个嵌入式视频，内容中环球时报的总编辑声称，将新疆设施称为“大规模拘留营”的人“是在抹黑，职业教育和培训中心”旨在帮助人们避免极端主义。”
  </p>
  <p class="graf graf--p">
   他继续攻击“欧洲政客和媒体工作者”，并声称“他们曾试图捍卫新疆的恐怖活动”，还补充道，“他们的手被在暴徒袭击中死亡的中国人民的鲜血染红了。 ”
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/globaltimesnews/status/1146997128608268289" href="https://twitter.com/globaltimesnews/status/1146997128608268289" rel="noopener noreferrer" target="_blank">
    7月4日的另一条推文
   </a>
   中包含了据称在新疆拍摄的视频，其中显示人们可以在街上购物，在餐馆吃饭，欣赏钢琴音乐……
  </p>
  <p class="graf graf--p">
   该视频描述了2009年发生在新疆首都乌鲁木齐的骚乱，并指出那里的居民“现在过着幸福安宁的生活”，因为他们共同努力打击恐怖主义和极端主义。视频中完全没有提到大规模拘留营。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Twitter 协助宣传的
   </strong>
   其他中国环球时报广告都遵循类似的主题，将该地区描述为一个幸福和平的地方，“没有发生”任何侵犯人权的行为。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/globaltimesnews/status/1146970999180578816" href="https://twitter.com/globaltimesnews/status/1146970999180578816" rel="noopener noreferrer" target="_blank">
    一则推文
   </a>
   还包括一名老妇人的视频，描述她收到政府官员提供的一揽子医疗用品，然后在欢乐中感激涕零。
  </p>
  <p class="graf graf--p">
   该推文称该地区的贫困状况已得到缓解，因为当地居民可以“获得高质量的医疗保健和负担得起的药品”。
  </p>
  <p class="graf graf--p">
   大赦国际（Amnesty International）中国研究员帕特里克潘（Patrick Poon）表示，他发现推特宣传这些广告“令人震惊”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这是 Twitter 需要解决的一个非常重要的严重问题，” Poon 说。 “Twitter 正在帮助推广虚假信息和政府宣传。允许这样的广告设置了令人震惊的先例。”
  </p>
  <p class="graf graf--p">
   周一，Twitter表示将不再接受来自国家控制的媒体的广告。你信吗？
  </p>
  <p class="graf graf--p">
   Twitter 推动中国政府宣传似乎与其自身的政策相矛盾，后者声称平台上的广告必须“诚实”。
  </p>
  <p class="graf graf--p">
   这些广告还破坏了 Twitter 首席执行官杰克·多尔西自己的声明，他去年告诉参议院情报委员会，该公司正在努力打击“通过机器人和人类协调制作的错误信息宣传”。
  </p>
  <p class="graf graf--p">
   像许多西方科技公司一样，Twitter 与中国有着复杂的关系。Twitter 在该国被封锁，如果不使用审查规避技术（例如 VPN 或代理服务），则无法访问该社交媒体平台。
  </p>
  <p class="graf graf--p">
   然而，
   <strong class="markup--strong markup--p-strong">
    与此同时，Twitter 在中国产生了大量的广告收入，并在该国的业务持续增长。
   </strong>
  </p>
  <p class="graf graf--p">
   据报道，去年7月，推特在中国的董事
   <a class="markup--anchor markup--p-anchor" data-href="https://pandaily.com/twitters-greater-china-team-triples-in-size/" href="https://pandaily.com/twitters-greater-china-team-triples-in-size/" rel="noopener noreferrer" target="_blank">
    表示
   </a>
   ，该公司在中国的团队人数于去年增加了两倍，成为该公司发展最快的部门。
  </p>
  <p class="graf graf--p">
   今年5月，这家社交媒体巨头在北京举办了“推特营销人员”会议。
  </p>
  <p class="graf graf--p">
   与此同时，大赦研究员 Poon 表示，近几个月来，中国警方越来越多地针对在 Twitter 上活跃的中国人权倡导者，迫使他们删除他们的帐户或删除批评政府的特定帖子。
  </p>
  <p class="graf graf--p">
   据说这些案件已经向 Twitter 报道，但该公司尚未采取任何行动。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “推特允许中国政府宣传其谎言，同时对那些遭受中国政权迫害的人充耳不闻。我们很想了解 Twitter 如何解释这个问题”， Poon 说。
  </p>
  <p class="graf graf--p">
   总之，对强权保持持续的高度警惕，是作为一名公民的最基本意识。⚪️
  </p>
