   现代战争：以心理操纵为武器
  </h1>

 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      集中化的互联网媒介给了欺诈战略以前所未有的独特空间，令其变得更加高效、并难以被察觉。这对反抗者来说尤其关键，为什么？
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   俗话说兵不厌诈。欺诈和操纵自古以来都是战争中最常用的也是最有效的武器。古老的欺诈战略形形色色多样且丰富，我们会逐步介绍这些知识，
   <strong class="markup--strong markup--p-strong">
    因为它们在当下仍非常有效。最重要的是，集中化的互联网媒介给了欺诈战略以前所未有的独特空间，令其变得更加高效，和难以被察觉
   </strong>
   。
  </p>
  <p class="graf graf--p">
   与北约有关联的一个研究小组发现，他们能够在士兵中“灌输不良行为”，并获得有关他们的“非常详细”的个人信息。
  </p>
  <p class="graf graf--p">
   北约战略传播中心于1月份发布的一项研究，探讨了“数字环境中可用的用户数据类型，并演示了恶意行为者如何在军事演习中利用这些数据“，报告的摘要是这样写的。
  </p>
  <p class="graf graf--p">
   该报告指出了社交媒体心理操纵对士兵 — 战斗带来的影响。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “结果……表明，在当前的数字领域，对手能够通过收集足够多的士兵个人数据，以精确创建有针对性的信息，便可以成功地影响操作者选择的目标受众，以实现操作者想要的行为，”摘要说。
  </p>
  <p class="graf graf--p">
   研究人员表示，操作者试图通过创建虚假账户，在社交平台上使用社交工程伎俩与士兵交朋友、以及创建可以向他们做广告的假网页和群组，以影响战斗部队整体。由于安全问题他们无法透露确切方法。
  </p>
  <p class="graf graf--p">
   请回忆一下，中国政府
   <strong class="markup--strong markup--p-strong">
    在多年前开始
   </strong>
   就确立了*严控*部队战士对社交媒体和应用的使用，随后很快又开始严控官媒记者对社交媒体的使用 — 要求上交账户以登记，这不仅仅是“防止他们说出不该说的话”，更有，防止他们被其他思想所影响。足见，
   <strong class="markup--strong markup--p-strong">
    那些认为当权者很愚蠢的反抗者太过轻敌了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   该研究指出，社交媒体平台，特别是 Facebook，能够对抗一些针对部队的操纵活动。研究表示该平台在大约两周内取消或暂停了“蜜罐页面”，在一两个小时内封锁了模仿真实页面的假页面，以及删除了一些在不同时期冒充真实人物的个人资料。但没有关闭假的聊天群组。
  </p>
  <p class="graf graf--p">
   但是，这不过是由于Facebook 对战争的支持，与此同时这些社交工程伎俩一直在大规模地被用于针对人权人士、活动家和记者实施的攻击和操纵，Facebook 会关心吗？
  </p>
  <p class="graf graf--p">
   这份研究说：“武装部队必须加强监测和对策，以降低社交媒体被用来收集任务敏感信息的风险”。
   <strong class="markup--strong markup--p-strong">
    这句话尤其重要！对反抗者非常重要！我们经常能看到中国的反抗者在线发布了大量高度危险的私密信息，这种情况下你们不是在作战，不客气的说变成了通敌
   </strong>
   。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    N
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://thehill.com/policy/technology/431077-nato-experimented-with-using-social-media-to-influence" href="https://thehill.com/policy/technology/431077-nato-experimented-with-using-social-media-to-influence" rel="noopener noreferrer" target="_blank">
    ATO study experimented with using social media to influence soldiers
   </a>
   . A research group affiliated with with the North Atlantic Treaty Organization (NATO) found it was able to “instill undesirable behavior” in soldiers and gain “very detailed” personal information about them.
  </p>
