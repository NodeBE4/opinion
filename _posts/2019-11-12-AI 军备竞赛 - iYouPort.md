  <p class="graf graf--p">
   美国国防部的国防高级研究计划局 DARPA 计划在未来5年内投入20亿美元用于人工智能研究和开发。
  </p>
  <p class="graf graf--p">
   顺便说，当年就是因 DARPA 的情报需求并大力资助才诞生了谷歌，该计划被称为大规模数字数据系统（MDDS）项目。在这里看到详细介绍《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e7%9a%84%e7%9c%9f%e9%9d%a2%e7%9b%ae%e2%80%8a-%e2%80%8a-theyre-always-watching/" href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e7%9a%84%e7%9c%9f%e9%9d%a2%e7%9b%ae%e2%80%8a-%e2%80%8a-theyre-always-watching/" rel="noopener noreferrer" target="_blank">
    谷歌的真面目
   </a>
   》。
  </p>
  <p class="graf graf--p">
   这一次是建立在联邦政府已经花费在基于人工智能的技术研发的
   <strong class="markup--strong markup--p-strong">
    20亿美元
   </strong>
   之上。
  </p>
  <p class="graf graf--p">
   毫无疑问，美国对用于国防和军事用途的各种人工智能技术的深度和潜力非常感兴趣，这是朝着与中国进行技术军备竞赛的方向迈出的重要一步。但是，这可能还不够。
  </p>
  <p class="graf graf--p">
   美国空军情报、监视和侦察副参谋长 VeraLinn Jamieson 中将
   <a class="markup--anchor markup--p-anchor" data-href="https://www.af.mil/About-Us/Biographies/Display/Article/108431/lieutenant-general-veralinn-dash-jamieson/" href="https://www.af.mil/About-Us/Biographies/Display/Article/108431/lieutenant-general-veralinn-dash-jamieson/" rel="noopener noreferrer" target="_blank">
    表示
   </a>
   ，中国已在2017年花费了120亿美元用于人工智能研发，估计到2020年可能高达
   <strong class="markup--strong markup--p-strong">
    700亿美元
   </strong>
   。
  </p>
  <p class="graf graf--p">
   中国政府投资的主要受益者是人工智能创业公司，这在面部识别领域最为明显。
  </p>
  <p class="graf graf--p">
   监视相机在中国无处不在，公民受到几乎不断的监控。基于 AI 的面部识别算法的发展已经广泛开始。
  </p>
  <p class="graf graf--p">
   但是，DARPA 存在问题。虽然美国
   <a class="markup--anchor markup--p-anchor" data-href="https://www.datainnovation.org/2019/01/survey-few-americans-want-government-to-limit-use-of-facial-recognition-technology-particularly-for-public-safety-or-airport-screening/" href="https://www.datainnovation.org/2019/01/survey-few-americans-want-government-to-limit-use-of-facial-recognition-technology-particularly-for-public-safety-or-airport-screening/" rel="noopener noreferrer" target="_blank">
    大多数人
   </a>
   对在机场、零售商店和公共场所等一些民用领域使用面部识别持宽容态度，但军事用途却是另一回事。
  </p>
  <p class="graf graf--p">
   许多人对在军事环境中使用面部识别和其他基于AI的技术持谨慎态度。
  </p>
  <p class="graf graf--p">
   谨慎源于捕获图像或视频的目的。例如，在零售环境中，摄像机捕获图像和素材以跟踪人们的需求。零售商对个人并不感兴趣，而是对他们的购买行为感兴趣。
  </p>
  <p class="graf graf--p">
   然而在军事背景下，其目的是识别、分类、验证、并在必要时反击任何感知到的威胁。出于安全考虑，这似乎是一种合理的应用。但是，面部识别的一个主要问题是它的准确性。
  </p>
  <p class="graf graf--p">
   美国国家标准与技术研究院（NIST）对不同的面部识别算法
   <a class="markup--anchor markup--p-anchor" data-href="https://iapp.org/news/a/this-little-known-facial-recognition-accuracy-test-has-big-influence/" href="https://iapp.org/news/a/this-little-known-facial-recognition-accuracy-test-has-big-influence/" rel="noopener noreferrer" target="_blank">
    进行了准确度测试
   </a>
   ，结论称所测试的28种算法的失败率仅为0.2％。
  </p>
  <p class="graf graf--p">
   然而，其他研究则表明，该技术非常不准确，不能正确识别人，并表现出一些明显的偏见，特别是对于深色皮肤的人。
  </p>
  <p class="graf graf--p">
   当与自动杀伤性武力相结合时，这种不准确性可能导致致命的错误。有关方面正在呼吁制定法规，但这些形式将受到激烈辩论。
  </p>
  <p class="graf graf--p">
   这些担忧让谷歌员工对 Project Maven 做出了强烈反应，抗议爆发。这是与美国国防部进行基于人工智能的无人机监控的持续合同。
  </p>
  <p class="graf graf--p">
   抗议运动促使公司决定在2019年3月到期时不再续约。许多员工认为公司应避免参与 “战争业务”。
  </p>
  <p class="graf graf--p">
   这一信念也迫使谷歌暂时放弃竞购一个更大的单一承包商 DoD 项目，该项目被称为五角大楼的联合企业防御基础设施（JEDI）。
  </p>
  <p class="graf graf--p">
   亚马逊、微软和 IBM 等其他科技巨头
   <strong class="markup--strong markup--p-strong">
    仍然在
   </strong>
   争夺五角大楼的数据云存储以及其他指定使用面部识别技术的利润丰厚的合同。
  </p>
  <p class="graf graf--p">
   一提到军事工业综合体就会说到谷歌，然而却忽略了许多科技寡头一直在与军方合作。
  </p>
  <p class="graf graf--p">
   除了大型科技公司之外，军方还在寻找来自小型私营公司和学术界的人才，以应对基于人工智能的
   <strong class="markup--strong markup--p-strong">
    新战场
   </strong>
   的挑战。
  </p>
  <p class="graf graf--p">
   美国陆军于2019年2月1日正式成立了人工智能特别工作组，包含私人企业和学术机构的人工智能专家网络。该工作组的总部设在卡内基梅隆大学。
  </p>
  <p class="graf graf--p">
   本文将讨论面部识别在军队中的当前应用。这些应用将解决识别或匹配方面的常见错误源，包括角度、比例、照明和分辨率的差异，以及训练数据的稀缺性。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    自适应技术与检测感知威胁
   </strong>
  </p>
  <p class="graf graf--p">
   面部识别是一种用于身份搜索和验证的非接触式方法，可以在不与主体交互的情况下捕获图像和视频并进行识别，然而，面部识别并不总是准确的。
  </p>
  <p class="graf graf--p">
   准确性取决于软件检测和匹配面部节点的能力。大多数面部识别算法需要大量数据来训练它们，并且对存储图像和目标图像之间的差异具有敏感性。
  </p>
  <p class="graf graf--p">
   当数据不可用于训练算法、或目标图像模糊、或在不利条件下拍摄时，会损害软件获得高精度的能力。这些是现实世界中的问题。
  </p>
  <p class="graf graf--p">
   为了解决这个问题，美国海军空战中心武器部在加州中国湖开发了自适应面部识别软件。
  </p>
  <p class="graf graf--p">
   这是 Katia Estabrides 的心血结晶，该软件只需要较少的数据就可以训练算法，并且有能力整合和使用新数据。
  </p>
  <p class="graf graf--p">
   该软件还能够容忍存储图像和目标图像之间的差异，例如照明、角度和比例，尤其是当多个传感器发挥作用时。
  </p>
  <p class="graf graf--p">
   该自适应技术需要至少一个与分类处理工具相关联的电子处理器，一个包含已知对象若干图像的数据库，以及一个未知对象的​​测试图像。
  </p>
  <p class="graf graf--p">
   这类软件的一个典型用途是识别敌人。军方通常会保留多个人员身份识别记录，其中包括照片。
  </p>
  <p class="graf graf--p">
   该软件可以快速识别不在其数据库中的面孔，对其进行标记，并向监视人员发出有关未授权者存在的警报。这可以成为活跃战场中的资产。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    热成像和卷积神经网络
   </strong>
  </p>
  <p class="graf graf--p">
   在低光照或夜间条件下识别人脸一直是军方的重点需求。
  </p>
  <p class="graf graf--p">
   标准面部识别软件依赖于图像或视频中的可见细节来进行匹配，即便如此，某些算法也无法获得很准确的结果。
  </p>
  <p class="graf graf--p">
   当由于缺少光源导致拍摄对象的脸部处于阴影中或根本不可见时，缺乏细节将使得面部识别非常不准确。
  </p>
  <p class="graf graf--p">
   军方利用热成像来检测人的存在，并且可以捕获可识别面部的图像。
  </p>
  <p class="graf graf--p">
   然而，传统的面部识别仍然无法进行准确匹配，因为无法获得可见的细节。
  </p>
  <p class="graf graf--p">
   为了解决这个问题，美国陆军研究实验室的科学家提出了一种利用可见光谱和现有面部识别软件从热图像中生成可见空间的方法。热图像绘制了面部的冷热区域，足以为正确的合成方法生成重要信息。
  </p>
  <p class="graf graf--p">
   下面是一段40秒的视频，演示了该方法的工作原理：
   <a class="markup--anchor markup--p-anchor" data-href="https://youtu.be/ItoSJeVilMg" href="https://youtu.be/ItoSJeVilMg" rel="nofollow noopener noreferrer" target="_blank">
    https://youtu.be/ItoSJeVilMg
   </a>
  </p>
  <p class="graf graf--p">
   科学家使用卷积神经网络（CNN），一种深度学习算法，在面部的局部区域（即眼睛、鼻子、嘴巴）中放入缺失的细节，以合成一张脸。
  </p>
  <p class="graf graf--p">
   CNN的工作方式类似于人脑，因为它可以通过为不完整图像的某些方面赋值并进行连接，从少量数据中推断出图像整体。
  </p>
  <p class="graf graf--p">
   虽然结果不是如照片般逼真的，但是有足够的关键点或标志可以在许多情况下为面部识别软件进行精确匹配。
  </p>
  <p class="graf graf--p">
   Benjamin Riggan 是提出这种方法的研究人员之一。他表示，“当使用热成像仪捕捉面部图像时，主要的挑战是捕获的热图像必须与监视名单或图库相匹配，而该名单或图库仅包含来自已知感兴趣的人的常规可见图像。”
  </p>
  <p class="graf graf--p">
   通过使用不同的综合方法，Riggan 及其同事帮助军队有可能在黑暗中识别面部。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    无人机和视频素材分析
   </strong>
  </p>
  <p class="graf graf--p">
   无人机由于各种原因已被军方广泛使用。
  </p>
  <p class="graf graf--p">
   延伸：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%b6%88%e8%b4%b9%e6%97%a0%e4%ba%ba%e6%9c%ba%e5%af%b9%e6%8a%97%e7%a4%ba%e5%a8%81%e8%80%85%e9%a2%84%e7%a4%ba%e4%ba%86%e4%b8%80%e4%b8%aa%e5%8d%b1%e9%99%a9%e7%9a%84%e3%80%81%e6%97%a0%e6%b3%95%e6%97%a0/" href="https://www.iyouport.org/%e6%b6%88%e8%b4%b9%e6%97%a0%e4%ba%ba%e6%9c%ba%e5%af%b9%e6%8a%97%e7%a4%ba%e5%a8%81%e8%80%85%e9%a2%84%e7%a4%ba%e4%ba%86%e4%b8%80%e4%b8%aa%e5%8d%b1%e9%99%a9%e7%9a%84%e3%80%81%e6%97%a0%e6%b3%95%e6%97%a0/" rel="noopener noreferrer" target="_blank">
    消费无人机对抗示威者预示了一个危险的、无法无天的冲突时代
   </a>
   》
  </p>
  <p class="graf graf--p">
   随着 MQ-1“捕食者”无人机的发明，无人机很快成为军事行动的重要组成部分，并于2001年底在阿富汗使用，允许在无法进入或危险的地区进行侦察、监视和情报收集。
  </p>
  <p class="graf graf--p">
   美国军方库存中的大约8,000架无人机能够拍摄视频，而一些较大的无人机甚至可以拍摄高分辨率的照片。
  </p>
  <p class="graf graf--p">
   问题在于如何及时处理所有这些视频和照片。
  </p>
  <p class="graf graf--p">
   人类分析师需要花费很长时间才能完成处理这些无人机拍摄的大量视频和照片，而且实际情况可能会在那段时间内发生变化。
  </p>
  <p class="graf graf--p">
   为了提高分析效率，军方采用人工智能技术来应对这一挑战。
  </p>
  <p class="graf graf--p">
   军方特意与谷歌签订协议，使用其 TensorFlow AI 系统构建AI软件，作为算法战争跨职能团队或 Project Maven 的一部分。
  </p>
  <p class="graf graf--p">
   军事合同涉及使用人工智能软件分析镜头，检测威胁并指定感兴趣的物体供人类分析人员审查，这将成为制定军事决策的基础。
  </p>
  <p class="graf graf--p">
   国防部声称从机器学习辅助分析中获得的信息可以“帮助减少附带损害”，减轻威胁并使士兵保持在地面上安全。
  </p>
  <p class="graf graf--p">
   永不疲惫的哨兵和军事基地
  </p>
  <p class="graf graf--p">
   大多数人认为军事基地拥有超级严密的安全性是理所当然的。然而这需要一些工作，因为大多数这些基地的绝对规模之大意味着对人员的重大投资。人工智能也许可以做出部分改变。
  </p>
  <p class="graf graf--p">
   今天的主要军事基地充分利用技术进步来确保人身安全，包括人工智能。
  </p>
  <p class="graf graf--p">
   关键在于广泛的传感器和摄像机网络的战略布局，这些传感器和摄像机连接到机器视觉软件，旨在发现镜头中的异常，从而警告人类操作员。例如，在爱德华兹空军基地，一个地面雷达系统可以扫过其308,000英亩的土地。
  </p>
  <p class="graf graf--p">
   人类操作员需要不断地对这些进行监控，但是在长时间班次结束时或在瞬间分心时很容易错过一些细节。人工智能软件可以解决这个问题，提高人工操作员的效率。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    特勤局和白宫试验
   </strong>
  </p>
  <p class="graf graf--p">
   面部识别作为机场的有效安全功能正在迅速受到青睐；这些机场使用的所有商业人工智能软件似乎都能实现他们的承诺。
  </p>
  <p class="graf graf--p">
   似乎美国政府可能会利用这些现成的安检工具在军事基地等关键领域实施？奇怪的是，事实并非如此。
  </p>
  <p class="graf graf--p">
   根据国防生物识别系统（DBIDS）网站，军事设施只在入口处使用身份证和指纹生物识别技术。
  </p>
  <p class="graf graf--p">
   然而，美国特勤局尝试了一些新的东西。他们使用来自白宫现有的闭路电视（CCTV）监视系统的镜头测试了面部识别软件，其中包括特勤局志愿者的测试人群。
  </p>
  <p class="graf graf--p">
   该测试是面部识别试点或FRP计划的一部分，以确定软件在识别公共场所志愿者方面的准确性。志愿者代表了软件最终必须识别的已知“感兴趣的主题”。
  </p>
  <p class="graf graf--p">
   以下是来自 NowThisNews 的3分钟视频简介和对此试验的评论：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    2018年11月的测试是白宫计划实施面部识别技术的第一阶段，以取代目前的军警部门官员和特工。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    进一步的测试将持续到2019年8月，届时将确定是否推进面部识别技术。该文件未指定试验中使用的特定面部识别软件。
   </em>
  </p>
  <p class="graf graf--p">
   以上是这场军备竞赛美国部分的大致内容。
  </p>
  <p class="graf graf--p">
   没有人知道谁会领先。
  </p>
  <p class="graf graf--p">
   对手在军备竞赛中的能力的不确定性可以成为一种危险的促进因素，促使政府投入更多资源并接受这些技术的更多风险应用。
  </p>
  <p class="graf graf--p">
   在冷战期间，不完全的信息和相互猜疑推动了核武器的储存越来越高。保密和混淆导致其他人的能力被夸大，并激发了严厉的反应。
  </p>
  <p class="graf graf--p">
   然而，当今政策制定者不会对攻击者的能力感到困惑，而是在经过多年研究和开发后对其未来能力感到困惑。如果人工智能达到超人的智力水平，它将彻底改变全球力量平衡，并将失败者降级为永久的二等身份。
  </p>
  <p class="graf graf--p">
   这也许会是一场无休止的竞技。⚪️
  </p>
