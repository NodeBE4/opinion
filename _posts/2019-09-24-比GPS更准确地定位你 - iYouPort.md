   比GPS更准确地定位你
  </h1>
 </header>

  <span class="byline">
   <a href="https://www.iyouport.org/author/don-evans/" rel="author" title="由McCaffrey发布">
    McCaffrey

    技术防身/自我保护方法
   </a>
  </span>

 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      有一种方法可以比GPS更准确地定位你，而且这一定位数据正在被运营商使用私下卖给出价最高的人
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   正如主板调查显示的那样，蜂窝运营商经常在个人用户蜂窝位置数据的海洋上徘徊，将这些宝贵的数据出售给任何出价高的公司，这其间几乎没有任何有意义的监督。除了直接从手机信号塔收集的位置数据外，一项新的主板调查还显示，手机运营商也一直在向数据经纪人、赏金猎人和相关企业提供 A-GPS 数据。
  </p>
  <p class="graf graf--p">
   A-GPS 数据有助于加快传统 GPS 技术的首次定位时间（TTFF），使得确定用户或设备的位置比单独使用 GPS 更快更准确。在传统卫星信号较弱或存在明显干扰的情况下，也可以使用 A-GPS。
  </p>
  <p class="graf graf--p">
   这被证明在无线技术中特别有用。在2007年的
   <a class="markup--anchor markup--p-anchor" data-href="https://assets.documentcloud.org/documents/5729080/FCC-07-166A1-1.pdf" href="https://assets.documentcloud.org/documents/5729080/FCC-07-166A1-1.pdf" rel="noopener noreferrer" target="_blank">
    报告和命令中
   </a>
   ，FCC 强制要求蜂窝电话运营商提供紧急呼叫调度员可用的手机位置数据。
  </p>
  <p class="graf graf--p">
   CTIA（无线行业的主要政策和游说组织）
   <a class="markup--anchor markup--p-anchor" data-href="https://ecfsapi.fcc.gov/file/60001013580.pdf" href="https://ecfsapi.fcc.gov/file/60001013580.pdf" rel="noopener noreferrer" target="_blank">
    提交的政府文件
   </a>
   显示，A-GPS 技术能够在50米范围内的任何地方确定设备或用户的位置。使用 MAC 地址和蓝牙公共设备地址（BT-PDA）数据可以帮助更精确地确定用户的位置。
  </p>
  <p class="graf graf--p">
   必须警惕的是，
   <strong class="markup--strong markup--p-strong">
    这种在细微的细节内跟踪用户的能力清楚地提示了蜂窝运营商 — 以及一群阴暗的中间人 — 这是另一个诱人的收入来源
   </strong>
   。正如 Motherboard 早前的调查所指出的那样，这些数据已经被未授权访问它的赏金猎人所使用。
  </p>
  <p class="graf graf--p">
   鉴于美国联邦通信委员会已经制定了具体规则来防止这种情况发生，主板的调查结果尤其令人不安。
  </p>
  <p class="graf graf--p">
   2015年，美国联邦通信委员会发布了收集911相关
   <a class="markup--anchor markup--p-anchor" data-href="https://assets.documentcloud.org/documents/5729259/FCC-15-9A1-2.pdf" href="https://assets.documentcloud.org/documents/5729259/FCC-15-9A1-2.pdf" rel="noopener noreferrer" target="_blank">
    数据的规则
   </a>
   ，并为遵守这些准则的行业“路线图”奠定了基础。规则声明此类敏感数据应在国家紧急地址数据库（NEAD）中收集和保护，不得用于（或出售）除911服务或其他有效的法律目的之外的任何其他目的。
  </p>
  <p class="graf graf--p">
   同年，ACLU，EFF 和 Public Knowledge 等消费者团体联盟在
   <a class="markup--anchor markup--p-anchor" data-href="https://assets.documentcloud.org/documents/5729301/Privacy-Letter.pdf" href="https://assets.documentcloud.org/documents/5729301/Privacy-Letter.pdf" rel="noopener noreferrer" target="_blank">
    一封公开信中特别警告
   </a>
   FCC，需要解决这种精确位置数据收集的潜在隐私后果。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “开发出符合新规定的高精度定位技术将引发一系列的隐私问题，这些问题在E911程序中尚未得到充分解决，”该组织表示。 “公共安全不应以牺牲消费者隐私为代价 — 也不应该出现这类问题。”
  </p>
  <p class="graf graf--p">
   美国联邦通信委员会表示，“除非法律要求，否则提供商必须证明他们不会将 NEAD 或相关数据用于除响应911电话之外的任何其他目的。”
  </p>
  <p class="graf graf--p">
   但是，主板对收集和销售 A-GPS 数据的调查发现，用户 A-GPS 数据最近于2017年被出售给了第三方。主板咨询的多名法律专家证实，联邦通信委员会有充分的权力惩罚运营商和数据经纪人，他们试图从这些数据中获利。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “美国联邦通信委员会有能力对 CPNI（用户专有网络信息）违规行为处以罚款，”Public Knowledge 的 Dylan Gilbert 在一封电子邮件中告诉 Motherboard。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “正如2017年11月的订单所述，该计划承诺个人提交给 NEAD 数据库的信息将不会被分享，除非法律要求，因此，这似乎明显违反了联邦通信委员会在2015年批准 NEAD 和一般高级911任务时所要求的内容。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “隐私专家 Guarav Laroia 告诉 Motherboard，这些公司正在促进向赏金猎人出售位置信息，这是不合情理的。“调查航空公司在这方面的作用完全属于联邦通信委员会的范围，如果委员会不认真对待这些问题，委员会的法定职责就会失败。”
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    W
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://motherboard.vice.com/en_us/article/j575dg/what-a-gps-data-is-and-why-wireless-carriers-most-definitely-shouldnt-be-selling-it" href="https://motherboard.vice.com/en_us/article/j575dg/what-a-gps-data-is-and-why-wireless-carriers-most-definitely-shouldnt-be-selling-it" rel="noopener noreferrer" target="_blank">
    hat A-GPS Data Is (and Why Wireless Carriers Most Definitely Shouldn’t Be Selling It)
   </a>
   A Motherboard investigation found that telecom companies are selling their customers’ “assisted GPS” data, which is intended for first responders answering 911 calls.
  </p>
