  <p class="graf graf--p">
   大多数开源情报调查都会涉及至少一个电子邮件地址。有些调查之初，你甚至
   <strong class="markup--strong markup--p-strong">
    只有
   </strong>
   一个电子邮件地址，此外什么都没有。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    不要错过《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e8%b0%83%e6%9f%a5%e4%b9%8b%e5%88%9d%e4%bd%a0%e5%8f%aa%e6%9c%89%e4%b8%80%e4%b8%aa%e7%94%b5%e9%82%ae%e5%9c%b0%e5%9d%80%e5%8f%af%e4%bb%a5%e6%80%8e%e4%b9%88%e5%8a%9e/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e8%b0%83%e6%9f%a5%e4%b9%8b%e5%88%9d%e4%bd%a0%e5%8f%aa%e6%9c%89%e4%b8%80%e4%b8%aa%e7%94%b5%e9%82%ae%e5%9c%b0%e5%9d%80%e5%8f%af%e4%bb%a5%e6%80%8e%e4%b9%88%e5%8a%9e/" rel="noopener noreferrer" target="_blank">
     如果调查之初你只有一个电邮地址……可以怎么办？– 开源情报经验
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   电子邮件地址有时可能会有些挑战，但它们也可以提供有关目标的大量信息。这篇文章的其余部分将研究可用于最大程度上利用电子邮件地址的各种不同工具和技术。
  </p>
  <p class="graf graf--p">
   有关特定电子邮件地址的可用信息量可能相差很大。
  </p>
  <p class="graf graf--p">
   这取决于许多不同的因素，例如，电子邮件地址的年代有多久远、所有者在互联网上发布该电子邮件的范围、提供商是否是像 Gmail 或 Protonmail 这样的大众电子邮件、或者电子邮件地址绑定到自己的公司/组织的域名了吗。
  </p>
  <p class="graf graf--p">
   下面我们来看看您可以怎么做。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     1、Google
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11436 jetpack-lazy-image" data-lazy-sizes="(max-width: 768px) 100vw, 768px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-14.png?resize=768%2C394&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-14.png?w=768&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-14.png?resize=300%2C154&amp;ssl=1 300w" data-recalc-dims="1" height="394" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-14.png?resize=768%2C394&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="768"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11436" data-recalc-dims="1" height="394" sizes="(max-width: 768px) 100vw, 768px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-14.png?resize=768%2C394&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-14.png?w=768&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-14.png?resize=300%2C154&amp;ssl=1 300w" width="768"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Google 是进行搜索的理想之地，但有时查找电子邮件地址的价值可能有限。
  </p>
  <p class="graf graf--p">
   主要原因是 Google 无法访问人们使用其电子邮件地址（例如帐户登录页面）的地方。
  </p>
  <p class="graf graf--p">
   但是，仍然存在一些有用的方法来查找Google能为其建立索引的电子邮件地址。
  </p>
  <p class="graf graf--p">
   使用引号仅返回完全匹配项。也就是说，搜索
   <strong class="markup--strong markup--p-strong">
    “
    <a class="__cf_email__" data-cfemail="6e1b1d0b1c2e0a01030f0700400d0103" href="/cdn-cgi/l/email-protection">
     [email protected]
    </a>
    ”
   </strong>
   比仅搜索
   <a class="__cf_email__" data-cfemail="0b7e786e794b6f64666a626525686466" href="/cdn-cgi/l/email-protection">
    [email protected]
   </a>
   更精确。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    intext
   </strong>
   搜索运算符还可用于查找包含该电子邮件地址的网页。
  </p>
  <p class="graf graf--p">
   当与
   <strong class="markup--strong markup--p-strong">
    site：
   </strong>
   结合使用时，也就是说在与您的目标组织相关的网站内进行搜索时，尤其有效。例如：
   <strong class="markup--strong markup--p-strong">
    site:targetcompany intext:
    <a class="__cf_email__" data-cfemail="b9cdd8cbdedccdf9dcd4d8d0d597dad6d4" href="/cdn-cgi/l/email-protection">
     [email protected]
    </a>
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    在这里看到
    <strong class="markup--strong markup--li-strong">
     所有运算符和使用方法汇总
    </strong>
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%94%bb%e9%98%b2%e7%9a%86%e5%8f%af%e7%94%a8%ef%bc%9agoogledorking-%e9%ab%98%e7%ba%a7%e8%bf%90%e7%ae%97%e7%ac%a6%e5%ae%8c%e6%95%b4%e5%88%97%e8%a1%a8/" href="https://www.iyouport.org/%e6%94%bb%e9%98%b2%e7%9a%86%e5%8f%af%e7%94%a8%ef%bc%9agoogledorking-%e9%ab%98%e7%ba%a7%e8%bf%90%e7%ae%97%e7%ac%a6%e5%ae%8c%e6%95%b4%e5%88%97%e8%a1%a8/" rel="noopener noreferrer" target="_blank">
     攻防皆可用：GoogleDorking 高级运算符完整列表
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   您甚至可以通过以下搜索字段来调整该技术，这样：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    site:organisation.com intext:@organisation.com
   </strong>
  </p>
  <p class="graf graf--p">
   这将返回公司网站内的所有可索引的电子邮件地址。本示例显示您可以使用以下查询来查找 bbc.co.uk 域中列出的所有电子邮件地址：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    site:bbc.co.uk intext:@bbc.co.uk
   </strong>
  </p>
  <p class="graf graf--p">
   另一种真正有效的技术是使用
   <strong class="markup--strong markup--p-strong">
    filetype:
   </strong>
   搜索运算符可找到目标的电子邮件地址。这可以找到隐藏在PDF或其他文件类型中的目标电子邮件地址。
  </p>
  <p class="graf graf--p">
   这可以显示公司文件、发票、会议记录、体育俱乐部设施、或任何其他类型的文件。例如这样的搜索：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    intext:”
    <a class="__cf_email__" data-cfemail="84e6ebf6edf7aaeeebeceaf7ebeaaae9f4c4f4e5f6e8ede5e9e1eaf0aaf1ef" href="/cdn-cgi/l/email-protection">
     [email protected]
    </a>
    ” filetype:pdf
   </strong>
  </p>
  <p class="graf graf--p">
   这样将找到包含鲍里斯·约翰逊议会电子邮件地址的所有PDF。
  </p>
  <p class="graf graf--p">
   当搜索链接到网络上有大量文档的组织（例如政府机构或大学）的电子邮件时，此功能特别有效。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11437 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-10.png?resize=1100%2C226&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-10.png?w=1585&amp;ssl=1 1585w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-10.png?resize=300%2C62&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-10.png?resize=1024%2C211&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-10.png?resize=768%2C158&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-10.png?resize=1536%2C316&amp;ssl=1 1536w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-10.png?resize=1100%2C226&amp;ssl=1 1100w" data-recalc-dims="1" height="226" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-10.png?resize=1100%2C226&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   在这一点上也值得一提
   <a class="markup--anchor markup--p-anchor" data-href="https://www.faganfinder.com/filetype/" href="https://www.faganfinder.com/filetype/" rel="noopener noreferrer" target="_blank">

   。它的工作方式与 Google filetype:类似，但它允许将不同的文件类型与更广泛的搜索引擎结合使用。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     2、用户名
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   某人的电子邮件地址与其用户名之间通常存在一些联系。尝试的一种好方法是获取电子邮件地址的第一部分并通过多个用户名搜索引擎运行它。
  </p>
  <p class="graf graf--p">
   比如，如果您想了解有关
   <a class="__cf_email__" data-cfemail="5635242f2622392535373b3b332460606016313b373f3a7835393b" href="/cdn-cgi/l/email-protection">
    [email protected]
   </a>
   的更多信息，则可以使用用户名 cryptoscammer666 运行搜索。
  </p>
  <p class="graf graf--p">
   电子邮件名越独特，您就越有可能找到匹配项。有许多基于浏览器的工具可以执行此操作，但是到目前为止，我最喜欢的工具是

  </p>

   不过请注意。用户名的归属和关联远非确定。仅仅因为两个帐户或电子邮件地址具有相同的用户名，并不意味着它们已被链接。

   如有可能，应进一步确认。比如上图，有好几个 “nixintel”，其实真实的只有一个。

  <h3 class="graf graf--p">
   <span style="color: #339966;">

    毫无疑问，由于匿名服务和 GDPR 之类的立法的兴起，WhoIs 作为开源情报的用途远不如以前了。
   </p>
   <p class="graf graf--p">
    但是，仍有大量电子邮件地址链接到 WhoIs 的域和IP，无论是注册人、技术支持还是滥用联系。
   </p>
   <p class="graf graf--p">
    有一些工具可以搜索 WhoIs 记录，但是

     ViewDNS
    </a>
    在此处提供了一个很好的简单界面，用于根据注册人信息检查电子邮件。
   </p>
