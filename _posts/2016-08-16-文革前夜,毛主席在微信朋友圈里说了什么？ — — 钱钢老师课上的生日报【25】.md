        钱钢老师附言

      <p class="ib ic jc id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="0d6c">
       寻找一个年龄相距百岁的“隔世纪同龄人”，是啥感觉？哈哈，90后的雷菲菲，瞄准了同为90后的毛泽东，还让他刷起了微信朋友圈。这淘气的生日报演示方式，说的却是严肃得不能再严肃的政治：在毛的微信里，分明已听到大风暴前的隐隐雷声。
      </p>
     </blockquote>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="da33">
      我的爸爸妈妈都出生于 1964 年11 月，相差仅半月，一个1号，一个16号。按照生日报的常规玩法，我跑去《人民日报》资料库瞄了几眼。爸爸出生那天的头版上一堆”查希尔””阿富汗””赞比亚”，妈妈出生那天的头版上一堆”冬麦””甘蔗””甜菜”。各自的画风倒是挺统一，然而可惜并没有戳中我的好奇心……
     </p>

     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="2ce0">
      一个被《人民日报》无聊到的我于是习惯性地拿出手机刷起了微信朋友圈。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="0949">
      等等！如果 1964 年也有微信的话，会是什么光景？71 岁的社交网瘾老年患者毛泽东，天天在朋友圈里指点江山激扬文字，点评时事新闻，得瑟诗词书画，党内高层或许还得违心地给他点赞……再说，老毛也是”90 后”呀（生于 1893 年），假设他也像我们一样，热衷于在朋友圈里直播生活，说不定还能从他的某些状态里拼凑出整个中国的历史命脉呢！
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="e47d">
      据说老毛晚年回顾人生，感叹自己这辈子就干了两件大事，一是建立了新中国，二是发动了文化大革命。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="4c71">
      于是我决定探究一下老毛在 1964 年发的两条朋友圈是怎么为两年后的文化大革命埋下伏笔的。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="716c">
      <strong class="id jd">
       “裴多菲俱乐部”的中国版
      </strong>
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="582e">
      1964 年 5 月，中宣部文艺处写出了一个《关于全国文联和各协会整风情况的报告（草稿）》。当时江青看见了这个文件，顺手就把链接转发给了老毛。6 月 27 日，老毛看完了这篇报告，于是分享到了朋友圈，留了一大段评语，还@了一大堆人示意他们过目。
     </p>

     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="caa2">
      这段点评里最耐人寻味的是最后一句：如不认真改造，势必在将来的某一天，要变成匈牙利裴多菲俱乐部那样的团体。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="5ac5">
      “匈牙利裴多菲俱乐部”是个什么东西？
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="bfb9">
      1956 年 2 月，赫鲁晓夫在苏共二十大揭露了斯大林的罪行，给全世界都敲了一记警钟，掀起一波“反对个人迷信”和“反对左倾极端主义”的大潮。匈牙利当时的最高领导人拉科西活脱脱就是个“斯大林第二”，赫鲁晓夫的报告于是更加坚定了匈牙利国内的反对派们把拉科西拽下马的决心。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="80f2">
      因此，他们急需一个能公开发表政见和揭露拉科西罪行的讲坛，来进行知识普及和舆论建设 — — 这就有了裴多菲俱乐部。
     </p>

     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="486f">
      苏共二十大刚结束，裴多菲俱乐部马上就活跃起来，先后举办了一系列大型研讨会，讨论经济、哲学、历史、新闻等等议题，邀请来的讲者包括全国知名的作家学者、社会名流、知识分子、人民军军官，以及那些在人民群众心中有威望的党内改革派。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="7a07">
      研讨会所关注的问题越来越尖锐，这样的讲坛形式也从首都布达佩斯蔓延至全国，拉科西当时忧心忡忡地说：“我感到自己像坐在一只随时可能爆炸的火药桶上。”
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="cd07">
      到 1956 年 10 月 23 日，这个火药桶果然炸了。布达佩斯超过十万群众举行游行示威，反对拉科西政策，随后还发展为武装暴动。
     </p>

     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="74cf">
      匈牙利不是唯一的学习苏联模式的国家。苏共二十大之后，坐在火药桶上的人也不只拉科西一个……
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="8b2b">
      据说赫鲁晓夫曾经说过：“我一看到毛就像看到斯大林一样。一个模子里浇出来的。”所以你们懂的。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="2855">
      但老毛怎么可能允许“裴多菲俱乐部”这样的团体诞生在中国的土地上呢？绝对要防微杜渐，不能由着知识分子们去给群众们叨逼叨。所以，意识形态上面的斗争一天都不能松，一定要死死把握住领导权。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="76bd">
      实际上，文艺界的整风运动 1963 年就已经开始了，当时已经风声鹤唳、人人自危。到 1964 年 6 月，老毛又作了这个批示，8 月下旬，就成立了一个“文艺界整风指挥部”。用“指挥部”这样一个军事术语称谓的机构来解决思想问题，你想想，老毛脑子里对于“意识形态”这根弦绷得是有多紧啊。（毕竟在火药桶上坐了这么多年。）
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="45eb">
      对文艺界来说，文化大革命的序幕这时候其实已经悄然拉开了。文化部副部长夏衍、齐燕铭，作家协会党组书记邵荃麟，全国文联副主席阳翰笙，全国剧协主席田汉等人先后受到了批判。《林家铺子》《不夜城》等十多部影片被打成“大毒草”。“阶级斗争”、“路线斗争”之风随即又从文艺界刮到了教育界……
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="e082">
      虽然标志着文革正式启动的事件发生在 1966 年，但是，冰冻三尺非一日之寒。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="a4a6">
      <strong class="id jd">
       “党内头号走资派”
      </strong>
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="73ee">
      1964 年 12 月 28 日，刚过完 71 岁生日两天的老毛发了这样一条朋友圈。他 po 出了两张图，一张是党章封面，一张是宪法封面，并说：“我是党员，我是公民。你们一个不让我参加党的会议，违反党章；一个不让我发言，违反宪法。”以及皮笑肉不笑的呵呵脸。
     </p>

          <img alt="Image for post" class="t u v cx aj" height="1202" sizes="700px" src="https://miro.medium.com/max/1400/1*nXqn1YIJQg9QSG8b20InXQ.jpeg" srcset="https://miro.medium.com/max/552/1*nXqn1YIJQg9QSG8b20InXQ.jpeg 276w, https://miro.medium.com/max/1104/1*nXqn1YIJQg9QSG8b20InXQ.jpeg 552w, https://miro.medium.com/max/1280/1*nXqn1YIJQg9QSG8b20InXQ.jpeg 640w, https://miro.medium.com/max/1400/1*nXqn1YIJQg9QSG8b20InXQ.jpeg 700w" width="700"/>
         </noscript>
        </div>
       </div>
      </div>
     </figure>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="d0dc">
      这种不点名的吐槽真是隐隐透着一股危险的气息。整个朋友圈被老毛的这条吓得大气也不敢出，没人敢评论，没人敢点赞。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="5806">
      我们如果把时间轴往后拨几年，就能证实这条朋友圈的恐怖之处。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="c12b">
      “一个不让我参加党的会议”这位，后来在文革中被打成“党内二号走资派” — — 邓小平。“一个不让我发言”这位，后来在文革中被打成“党内头号走资派” — — 刘少奇。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="963c">
      “走资派”这个词是“走资本主义道路的当权派”的简称，在老毛的字典里和“叛徒”一个意思。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="f8e3">
      这条朋友圈背后的故事是这样的。1964 年 12 月，中共中央政治局开工作会议。邓小平当时考虑的是，不过是进行一般的工作汇报，无需惊动老毛。于是汇报的时候就跟老毛说：如果工作忙，就不必参加了。而在另外一次会议上，老毛只讲了几句，就被刘少奇打断了。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="114d">
      这两个没有眼力见儿的家伙，有意无意剥夺老毛的存在感，老毛怎么能忍。篇幅有限，这里不展开讲毛邓二人的恩怨，主要捋一捋刘少奇是怎么变成老毛的头号眼中钉的。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="7d70">
      首先来看看 1957 年到 1967 年间毛刘二人上《人民日报》头版标题的次数。
     </p>

      </figcaption>

     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="ee78">
      1959 年庐山会议后，老毛开始“退居二线”，刘少奇作为一个冉冉升起的接班人，理所当然地开始上头版，刷存在感，为继位预热。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="c664">
      但是，老毛慢慢发现，这个钦定的接班人太不乖了。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="0e27">
      上过高中历史课的童鞋们肯定都知道，中国从 1959 年到 1961 年被称为”三年困难时期”。西方学者把这段时期称为“中国大饥荒”，当时饿死的人数到现在史学家还在考证。按照中国作家杨继绳在《墓碑》里统计的数据，当时非正常死亡人数达到了 3600 万。（而上文提到的被赫鲁晓夫揭老底的斯大林在大清洗中造成的死亡人数也是以“百万”来计，你们感受一下。）
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="a6be">
      之所以会造成这么严重的局面，刘少奇认为，“三分天灾，七分人祸”，觉得是“大跃进”和人民公社化运动中的左倾错误所导致。据王光美、刘源所著的《你所不知道的刘少奇》一书记载，当时刘少奇曾激动地对老毛讲：“饿死这么多人，历史要写上你我的，人相食，要上书的！”
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="4802">
      至于老毛的回应，则是他在八届十中全会上说的那样，“我周游了全国，从中南到西南，找各大区的同志谈话，每个省都说去年比前年好，今年比去年好，看来并非一片黑暗。有的同志把情况估计得过分黑暗了。”
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="a4a8">
      嗯，懂得读气氛的你们一定能听出来，这显然是一副要撕的节奏了。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="3c16">
      而压死刘少奇的最后一根稻草，大概可以从“四清运动”说起。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="22ae">
      “四清运动”是 1962 年底老毛开始在中国农村逐步推开的一场政治运动，主要目的是“反修防修”。
     </p>

     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="f2ee">
      关于运动的开展，毛刘二人在三个问题上谈不拢。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="e568">
      第一个分歧：“根子”在哪儿。老毛觉得是中央高层的修正主义倾向。而刘少奇觉得是基层干部的贪污腐化、蜕化变质。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="4cde">
      第二个分歧：调查方法。老毛觉得要靠开座谈会、听汇报、做记录，然后归纳总结，以此开展调查研究。刘少奇则觉得要“蹲点”，也就是深入基层。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="c3ae">
      第三个分歧：主要矛盾。老毛认为，四清运动是要解决社会主义与资本主义的矛盾，运动的重点是要整肃党内的走资派。而刘少奇则认为，关键在于“四清”和“四不清”的矛盾，党内外矛盾交叉，或者是敌我矛盾和人民内部矛盾交叉。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="eb9a">
      这么说吧，在这个运动的开展过程中，毛刘二人压根儿就没有能谈拢的地方……
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="674f">
      咱们再来看看实例。刘少奇在四清运动期间南下视察开座谈会的时候，说过这么一段话：“现在调查研究，按毛主席的办法不行了……层层听汇报，或者看报表，不行了。这种领导工作、领导革命的方法不行了。现在这种靠会议、报表的领导方法，一定要亡国！”
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="a0eb">
      老毛从井冈山时代就创立并推行的座谈会调查研究方法，被刘少奇一口一个“不行了”，到末了还干脆形容成“要亡国”，换作你是老毛，你一定也觉得心头有千万只草泥马奔腾而过。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="97a7">
      况且，此时的老毛，不是个普通的老毛，这是个坐在火药桶上神经紧绷内忧外患的老毛啊 — —
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="2d15">
      他担心国外的修正主义之风要吹到中国来，他担心国内的知识分子要搞中国版的裴多菲俱乐部，他更担心中共党内会出一个赫鲁晓夫式的人物、像对付斯大林那样来对付自己 — — 而眼下这个跟自己对着干的刘少奇，真是怎么看怎么像赫鲁晓夫啊！
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="7af2">
      后来，老毛对斯诺说，在 1965 年 1 月的时候，他就已经下定了决心，刘少奇必须要下台。你这么一捋老毛的心路历程，就觉得没什么好奇怪的了。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="4d55">
      面对来自国外、国内、党内的威胁，老毛不得不采取非常手段来保证中国的政治取向。大概，在老毛看来，文化大革命是在为人类摸索一条通往共产主义的新路吧。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="4191">
