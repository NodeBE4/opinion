   斯塔西的世界
  </h1>

 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <strong>
      <em>
       这些人以朋友甚至亲人的角色隐藏在被监视的群众身边，也就是那些对政府不满意的人、对西方表现出太多喜爱的人、或者以某种方式脱颖而出的人……这是个很容易被滥用的线人系统，人们利用这个系统公报私仇，压制竞争对手和他们不喜欢的其他人…
      </em>
     </strong>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   69年前，即1950年2月8日，刚刚起步的德意志民主共和国正式推出了其维稳部门，致力于监测和防止一切外部影响 — 尤其是来自西方的影响。
  </p>
  <p class="graf graf--p">
   德国人永远不会忘记“斯塔西”时代的恐怖。苏联为这个新组织提供了培训和设备，一切都是为了根除不同意见的来源，并找出可能违背“社会主义梦想”的所谓“颠覆性个人”。
  </p>
  <p class="graf graf--p">
   斯塔西雇佣了成千上万的士兵、分析员、官员和技术专家作为其在东德的大型监视系统的一部分，因此几乎每个工作场所、公共场所和教育机构都以某种方式受到监视和窃听。
  </p>
  <p class="graf graf--p">
   这种监视是通过微型摄像头或麦克风进行的 — 在某些方面，东德人的老大哥能力远远超过了他们那个时代。2006年的电影 “Das Leben der Anderen”（“其他人的生活”）令人难忘地描述了这种对监视的痴迷。
  </p>
  <p class="graf graf--p">
   在这里看到视频：
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="314" src="//www.youtube.com/embed/n3_iLOp6IhM" width="560">
   </iframe>
  </p>
  <p class="graf graf--p">
   也许更有效 — 也更可怕的 — 是为斯塔西工作的庞大的
   <strong class="markup--strong markup--p-strong">
    非正式线人网络
   </strong>
   ，这些人以朋友甚至亲人的角色隐藏在被监视的群众身边，也就是那些对政府不满意的人、对西方表现出太多喜爱的人、或者以某种方式脱颖而出的人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这是个很容易被滥用的线人系统，人们利用这个系统公报私仇，压制竞争对手和他们不喜欢的其他人
   </strong>
   。绝大多数人都有竞争对手，以及竞争中的阴暗心理，于是他们乐于为斯塔西服务。
  </p>
  <p class="graf graf--p">
   BStU（斯塔西记录委员会）承认，
   <a class="markup--anchor markup--p-anchor" data-href="http://www.bstu.bund.de/EN/MinistryOfStateSecurity/UnofficialCollaboratorsOfMfS/_node.html" href="https://www.bstu.bund.de/EN/MinistryOfStateSecurity/UnofficialCollaboratorsOfMfS/_node.html" rel="noopener noreferrer" target="_blank">
    1989年约有189,000名非正式的斯塔西线人
   </a>
   ，换言之，每90名公民中就有一名线人。而这其中甚至没有包括全职的线人！在整个民主德国40年的生命周期中，这个数字也各不相同，在政治危机期间不断上升。但是当 Wikileaks 在2007年透露，BStU
   <a class="markup--anchor markup--p-anchor" data-href="https://wikileaks.org/wiki/Stasi_still_in_charge_of_Stasi_files" href="https://wikileaks.org/wiki/Stasi_still_in_charge_of_Stasi_files" rel="noopener noreferrer" target="_blank">
    至少聘用了79名前斯塔西成员
   </a>
   时，BStU 本身的可信度受到了质疑。简单说就是，即便是上面这个惊人的数字也很可能是低于事实。
  </p>
  <p class="graf graf--p">
   加上这种近乎无所不能的老大哥系统是
   <strong class="markup--strong markup--p-strong">
    “Zersetzung(剥离)”的阴险实践。这是一种心理战的形式，通过将目标人孤立，使脱离支持的目标人士气低落，从而弱点毕露
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithSingleQuote">
   ‘Zersetzung’的效果可能就像官僚主义对你不利一样简单。在像DDR这样的国家，数百万人的日常生活取决于发放许可证和遵循官方制定的程序。有目的的延误、“误报”和其他干扰都可能会严重挫败国家政治对手的生活。
  </p>
  <p class="graf graf--p">
   这种手段的升级版会涉及
   <strong class="markup--strong markup--p-strong">
    捏造的证据
   </strong>
   ，并将其发送给“Zersetzung”目标对象的家人和朋友，以
   <strong class="markup--strong markup--p-strong">
    诋毁其声誉和公信力
   </strong>
   。记录显示已经有许多家庭被这些行为摧毁。
  </p>
  <p class="graf graf--p">
   终极的、也许是最阴险的阶段，将涉及对目标人家中物品的微妙操纵 — — 少一只袜子、时钟被停摆，家具被重新排列 …… 这是刺激一种深度的恐惧，让人陷入怀疑自己精神错乱的无助，最终一事无成。
  </p>
  <p class="graf graf--p">
   尽管德意志民主共和国解散已经将近30年，但今天仍然有一些受害者还在积极寻求赔偿。奇怪的是，
   <strong class="markup--strong markup--p-strong">
    前斯塔西官员仍在领取退休金
   </strong>
   。
  </p>
  <p class="graf graf--p">
   直到1989年，斯塔西一直在东德人的生活中存在，并在试图诋毁那些希望争取更大自由的人方面发挥了重要作用。后期的斯塔西开始监禁一些持不同政见者，许多人都写过关于他们在柏林 — 霍恩施豪森等监狱的经历 — 这里曾经是一个悲惨的地方，现在提供给游客参观。
  </p>
  <p class="graf graf--p">
   随着柏林墙的倒塌，统一进程被点燃，斯塔西开始摧毁数百万与他们对东德人口的监视有关的档案。1990年1月15日，大规模的抗议活动、以及对柏林 Ruschestraße 的 Stasi 总部的猛攻，阻止了更多资料被送进碎纸机。
  </p>
  <p class="graf graf--p">
   今天，前斯塔西总部是一个博物馆，不仅讲述了该组织的故事，而且还讲述了许多被斯塔西作为目标和受害者的故事，“Zersetzung”的幸存者和被监禁的人们经常举行会谈。
  </p>
  <p class="graf graf--p">
   关于斯塔西的更多信息可以在柏林的 DDR 博物馆学习到，这有助于帮助后人理解这种被政权不断监视而造成的深度恐惧和焦虑。
  </p>
  <p class="graf graf--p">
   如今仍有数十万建在的人经历过斯塔西时代的恐怖事件，那个时代对他们的生活产生了非常现实的影响。
   <strong class="markup--strong markup--p-strong">
    当下，在这个大规模深度监视变得如此简单和容易操作的世界里，回忆那个年代的灾难具有重大的意义
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在第二次世界大战结束时，德国被分为四个军事占领区：西部的美国、英国和法国，东部的苏联。最初的目标是共同努力帮助重建德国，同时也确保它再也不会成为对世界和平的威胁。
  </p>
  <p class="graf graf--p">
   但是，冷战很快就阻止了这一切。德意志民主共和国（GDR）是苏联于1949年10月7日在东部成立的，其他三个地区已经在此前几个月组成了德意志联邦共和国。
  </p>
  <p class="graf graf--p">
   经过25年的统一，民主德国对许多人来说是遥远的记忆，但它仍然影响着现代德国。
  </p>
  <p class="graf graf--p">
   以下是关于这个非常独特的国家的一些有趣的事实 — — 它们至今都非常重要，是因为它们仍然与现实密切相关：
  </p>
  <p class="graf graf--p">
   东德政府密切关注早年牛仔裤受欢迎程度的增长；舞厅里禁止所谓的“铆钉裤”，在学校穿着这样衣服的孩子会被送回家。国家将这些服饰视为“西方资本主义和反叛”的表达。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    这点上与现实的相关性详见：今天的朝鲜如何禁止牛仔裤《
    <a class="markup--anchor markup--li-anchor" data-href="https://medium.com/@iyouport/%E6%9C%9D%E9%B2%9C-%E6%94%B9%E5%BC%80-%E7%9A%84%E6%96%B0%E9%9D%A2%E7%9B%AE-%E7%9B%91%E8%A7%86%E8%AE%BE%E5%A4%87%E7%9A%84%E5%85%B4%E8%B5%B7%E5%8F%8A%E7%94%9F%E6%B4%BB%E7%8A%B6%E5%86%B5%E7%9A%84%E5%8F%98%E5%8C%96-%E5%A4%9A%E5%9B%BE-9b15dfd31136" href="https://medium.com/@iyouport/%E6%9C%9D%E9%B2%9C-%E6%94%B9%E5%BC%80-%E7%9A%84%E6%96%B0%E9%9D%A2%E7%9B%AE-%E7%9B%91%E8%A7%86%E8%AE%BE%E5%A4%87%E7%9A%84%E5%85%B4%E8%B5%B7%E5%8F%8A%E7%94%9F%E6%B4%BB%E7%8A%B6%E5%86%B5%E7%9A%84%E5%8F%98%E5%8C%96-%E5%A4%9A%E5%9B%BE-9b15dfd31136" rel="noopener noreferrer" target="_blank">
     朝鲜“改开”的新面目：监视设备的兴起及生活状况的变化
    </a>
    》
   </li>
   <li class="graf graf--li">
    并且，美国正在如何监视那些穿着某些服装的人《
    <a class="markup--anchor markup--li-anchor" data-href="https://medium.com/@iyouport/hr-a388e3ff004c" href="https://medium.com/@iyouport/hr-a388e3ff004c" rel="noopener noreferrer" target="_blank">
     “帮派数据库”是什么？为什么据此就可以直接给无辜的市民定下重罪？
    </a>
    》
   </li>
   <li class="graf graf--li">
    此外，中国对嘻哈文化的管控已经众所周知。
   </li>
  </ul>
  <p class="graf graf--p">
   所有人都熟悉分裂东柏林和西柏林的那个臭名昭着的墙，但它并不是德国唯一一个用混凝土拆分社区的隔离物。
  </p>
  <p class="graf graf--p">
   MödlareuthamTannbach 是一个约50人的小村庄，位于图林根和巴伐利亚的边界。由于这也是德国内美国和苏联管辖区之间的边界，美国士兵
   <a class="markup--anchor markup--p-anchor" data-href="http://www.bundesstiftung-aufarbeitung.de/uploads/pdf-2011/flyer_maueraustellung_shortguide.pdf" href="http://www.bundesstiftung-aufarbeitung.de/uploads/pdf-2011/flyer_maueraustellung_shortguide.pdf" rel="noopener noreferrer" target="_blank">
    称其为“小柏林”
   </a>
   。从20世纪60年代开始，围栏穿过其中间变成了坚固的墙，直到1989年。
  </p>
  <p class="graf graf--p">
   现在，你在柏林及其他地区都能看到一种非常常见的汽车品牌叫特拉贝特，当地人亲切地称其为“Trabi”，它是民主德国制造的汽型。这是
   <strong class="markup--strong markup--p-strong">
    “Ostalgie”的一个很好的例子 — 这个词指的是1990年以后出现的对前国家的怀旧情绪。
   </strong>
  </p>
  <p class="graf graf--p">
   这种车子很廉价，并且以行动缓慢、噪音很大和极端污染而闻名。汽车外壳实际上是使用再生塑料制成料。但东德人想要这种车子。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “在东德时代，每个人都试图买一辆特拉贝特。因为它来之不易，你必须排号，有时必须等待很多年才能拿到这辆车”，柏林特拉贝特导游 Johannes Drexler 告诉德国之声。想想中国的摇号？
  </p>
  <p class="graf graf--p">
   来之不易的东西通常被人们认为是“最好的东西”，而且记忆非常牢固，以至于如今遍地跑敞篷巴士和自动驾驶的时代里，依旧有很多人钟情于这种老笨拙劣的“Trabi”。
  </p>
  <p class="graf graf--p">
   不仅这些，而且孩子们依旧喜欢收集瓶子、金属、纸张、罐头壳、甚至是废弃的相机胶卷，他们可以卖掉这些收集以挣到一些钱。
  </p>
  <p class="graf graf--p">
   前两年我们途径中国的时候在豪华的大城市街道上看到不止一位翻找垃圾箱的老年人。他们手里提着一个简易的布袋或大号塑料袋，里面塞满了他人丢弃的饮料瓶，他们在垃圾箱里翻找着其他瓶子和易拉罐。他们的穿着基本体面，看起来并不像无家可归者，如果不是如此专注于垃圾箱，你可能会认为他们和其他工薪族没什么两样。
  </p>
  <p class="graf graf--p">
   回到德国。即使在四分之一个世纪之后，一些前共产主义品牌仍然能在德国超市的货架上随处可见。这是 “Ostalgie”的另一个很好的例子，某些食品品牌在柏林墙倒塌后，甚至在资本主义竞争激烈的情况下，仍然保持着它们的知名度。
  </p>
  <p class="graf graf--p">
   最著名的例子包括 ‘Rotkäppchen’（Red Riding Hood），这是一种起泡酒的品牌，其成功在1989年之后持续发展，直到他们能够购买西德 Mumm 酒庄的程度。
  </p>
  <p class="graf graf--p">
   其他包括1957年首次生产的碳酸饮料“Vita Cola”，如今仍可以在当地的 Aldi 购买，还有 Spreewald 泡菜，这些泡菜在地理上受到欧盟传统特色的保护。
  </p>
  <p class="graf graf--p">
   1989年后，东德仍然拥有60家母乳库。这些母乳库是第一次世界大战结束时在德国引进的，为那些本来无法获得母乳的儿童提供母乳。
  </p>
  <p class="graf graf--p">
   虽然在20世纪60年代和70年代接管西方的“女权主义”概念对于东德女性来说是相当陌生的，但是这里也有一种自己的“女权主义运动”。
  </p>
  <p class="graf graf--p">
   它是由政府按照“马克思主义女权主义”的方针设计的，引入了一个全面的系统来帮助妇女工作。他们为职业女性制定了配额，并引入了全面的法律来保护职业母亲、提供儿童保育。
  </p>
  <p class="graf graf--p">
   据估计，1990年约有90％的妇女在东德就业，而前联邦共和国只有55％。即使在今天，正如联邦统计局 Destatis 最近计算的那样，性别工资差距仍然显著存在：前共产主义国家的性别工资差距约为8％，而西方国家则为23％。
  </p>
  <p class="graf graf--p">
   甚至还有一种 ‘Ostalgie’ 酒店。该地点位于前东德的 ‘Plattenbauten’（预制建筑）之一，在这里你将会被传送回40年前。
  </p>
  <p class="graf graf--p">
   除了床垫和床单外，这家酒店的一切都是正宗的复古。而且，当然，每个房间都有一个前民主德国领导人 Erich Honecker 的肖像，它时刻监视着你……
  </p>
  <p class="graf graf--p">
   中国的 ‘Ostalgie’ 有哪些？文革复古跨包？军用水壶？搪瓷缸子？毛崇拜主题餐馆？汉服？…
  </p>
  <p class="graf graf--p">
   没有人想要回去那个时代；但在技术的推动下，今天的斯塔西3.0进化版正在以一种不可控的和变本加厉的威胁卷土重来。人性中最邪恶的一面被重新激发了。今天再次回顾这些悲惨的往事有着特别的意义，关于我们想要一个什么样的世界的思考。
  </p>
