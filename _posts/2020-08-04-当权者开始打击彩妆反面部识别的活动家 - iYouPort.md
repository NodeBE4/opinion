   <li class="graf graf--p">
    <span style="color: #00ccff;">
     <em>
      <strong>
       随着视频监控技术在世界范围内变得越来越普遍和日趋复杂，一些人正在寻找应对方法。但当局的反应似乎很快 ……
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-13094 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/0-1.jpg?resize=1024%2C576&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/0-1.jpg?w=1024&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/0-1.jpg?resize=300%2C169&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/0-1.jpg?resize=768%2C432&amp;ssl=1 768w" data-recalc-dims="1" height="576" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/0-1.jpg?resize=1024%2C576&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13094" data-recalc-dims="1" height="576" sizes="(max-width: 1024px) 100vw, 1024px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/0-1.jpg?resize=1024%2C576&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/0-1.jpg?w=1024&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/0-1.jpg?resize=300%2C169&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/0-1.jpg?resize=768%2C432&amp;ssl=1 768w" width="1024"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   想要保护自己隐私的激进抗议者正在使用化妆术、特殊的衣服、和配饰来混淆面部识别技术。
   <strong class="markup--strong markup--p-strong">
    具体方法相见：
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://start.me/p/xbYXdR/iyp-1" href="https://start.me/p/xbYXdR/iyp-1" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     列表-1 “生物识别和对抗方法”
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   在俄罗斯，一位专门从事反体制工作的活动家和艺术家创建了围绕这些反监视技术的在线社区。然后她被捕了 ……
  </p>
  <p class="graf graf--p">
   在 Facebook 和 Telegram 上发布的许多照片中，粗细相间的黑色和红色线条在俄罗斯年轻男女的脸上纵横交错。他们加入了 “Sledui” 运动（follow的意思）。
  </p>
  <p class="graf graf--p">
   艺术家和活动家 Ekaterina Nenasheva 于2月发起了这项运动。她曾因令人震惊的艺术表演和对俄罗斯政府的公开批评而闻名。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-13095 jetpack-lazy-image" data-lazy-sizes="(max-width: 996px) 100vw, 996px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-4.png?resize=996%2C994&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-4.png?w=996&amp;ssl=1 996w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-4.png?resize=300%2C300&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-4.png?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-4.png?resize=768%2C766&amp;ssl=1 768w" data-recalc-dims="1" height="994" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-4.png?resize=996%2C994&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="996"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13095" data-recalc-dims="1" height="994" sizes="(max-width: 996px) 100vw, 996px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-4.png?resize=996%2C994&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-4.png?w=996&amp;ssl=1 996w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-4.png?resize=300%2C300&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-4.png?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-4.png?resize=768%2C766&amp;ssl=1 768w" width="996"/>
   </noscript>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们不希望未经我们同意而被置于无处不在的视频监控摄像机镜头中。我们使用彩妆在几分钟内就可以使自己免受监视和面部识别技术的侵害，该彩妆成为反抗的象征” ，Nenasheva 在她的 Facebook 页面上写道。
  </p>
  <p class="graf graf--p">
   这个反抗组织在其
   <a class="markup--anchor markup--p-anchor" data-href="http://t.me/follow_channel?fbclid=IwAR0Ky3mszeJfk9RRnWQczdhgGduZPHGODKc4P7GW7bTy2esHfgzXb0F_riI" href="https://t.me/follow_channel?fbclid=IwAR0Ky3mszeJfk9RRnWQczdhgGduZPHGODKc4P7GW7bTy2esHfgzXb0F_riI" rel="noopener noreferrer" target="_blank">
    Telegram 频道
   </a>
   上发布了各种化妆的照片，随后有850多人关注。
  </p>
  <p class="graf graf--p">
   该运动还拥有自己的聊天机器人，可以为受众提供如何化妆的说明，以及对不同可能模式的解释：那些实际上会使人脸识别技术产生混乱的方法，以及使其根本上失效的方法。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13096 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-1.jpg?resize=1024%2C576&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-1.jpg?w=1024&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-1.jpg?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-1.jpg?resize=768%2C432&amp;ssl=1 768w" data-recalc-dims="1" height="576" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-1.jpg?resize=1024%2C576&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13096" data-recalc-dims="1" height="576" sizes="(max-width: 1024px) 100vw, 1024px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-1.jpg?resize=1024%2C576&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-1.jpg?w=1024&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-1.jpg?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-1.jpg?resize=768%2C432&amp;ssl=1 768w" width="1024"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   1月底，英国反抗运动 Dazzle Club 在伦敦组织了以城市监视为主题的无声的、精心编排的散步。参与者使用的正是与 Sledui 运动相似的图案彩妆。
  </p>
  <p class="graf graf--p">
   尽管英国反抗组织每个周末都能在伦敦格林威治街头露面，但是俄罗斯激进抗议者却不太幸运。
  </p>
  <p class="graf graf--p">
   2月9日，Ekaterina Nenasheva 发布了一系列彩妆的新照片 —— 但是，这一次，她是在警车里。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13097 jetpack-lazy-image" data-lazy-sizes="(max-width: 992px) 100vw, 992px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-4.png?resize=992%2C996&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-4.png?w=992&amp;ssl=1 992w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-4.png?resize=300%2C300&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-4.png?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-4.png?resize=768%2C771&amp;ssl=1 768w" data-recalc-dims="1" height="996" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-4.png?resize=992%2C996&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="992"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13097" data-recalc-dims="1" height="996" sizes="(max-width: 992px) 100vw, 992px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-4.png?resize=992%2C996&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-4.png?w=992&amp;ssl=1 992w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-4.png?resize=300%2C300&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-4.png?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-4.png?resize=768%2C771&amp;ssl=1 768w" width="992"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13098 jetpack-lazy-image" data-lazy-sizes="(max-width: 996px) 100vw, 996px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4-4.png?resize=996%2C1000&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4-4.png?w=996&amp;ssl=1 996w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4-4.png?resize=300%2C300&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4-4.png?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4-4.png?resize=768%2C771&amp;ssl=1 768w" data-recalc-dims="1" height="1000" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4-4.png?resize=996%2C1000&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="996"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13098" data-recalc-dims="1" height="1000" sizes="(max-width: 996px) 100vw, 996px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4-4.png?resize=996%2C1000&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4-4.png?w=996&amp;ssl=1 996w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4-4.png?resize=300%2C300&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4-4.png?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4-4.png?resize=768%2C771&amp;ssl=1 768w" width="996"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   这位艺术家在她的 Facebook 页面上写道：“他们昨天因为我的化妆而逮捕了我们，现在他们指责我们参加了未经授权的活动 —— 只是因为化妆。”
  </p>
  <p class="graf graf--p">
   莫斯科警察局将她和另外两位化了妆的人拘留了多个小时后才放行。⚪️
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    注：
   </strong>
   曾经很多国家确立了 “遮盖脸部违法” 的管制条例，而现在，化妆也变成了逮捕的理由。俄罗斯的案件不会是特例，这种镇压方式有可能很快会在全球其他国家出现，取决于当地活动家的反抗模式。活动家应该做好防御准备。
  </p>
