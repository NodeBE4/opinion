  <p class="graf graf--p">
   时尚很有趣，它也有助于回应我们生活的环境。
  </p>
  <p class="graf graf--p">
   天气转冷的时候，你会穿上厚重的衣服，随着温度的升高再变得轻盈；下雨或在耀眼的阳光下举起五彩缤纷的伞，在派对上戴上有趣的帽子……
  </p>
  <p class="graf graf--p">
   <strong>
    而当我们周围的社会转变为一个全景圆形监狱时，时尚会适应隐藏我们的身份。
   </strong>
  </p>
  <p class="graf graf--p">
   那个全能的监视国家正在从英国到美国到香港，一路挺进。
  </p>
  <p class="graf graf--p">
   世界各国政府和企业正在加紧使用监控摄像头和面部识别技术来识别抗议者、行人以及任何人，跟踪他们的一举一动。与此同时，行动主义创意者正在以创造性的方式
   <strong>
    予以反击
   </strong>
   。
  </p>
  <p class="graf graf--p">
   他们通过创新技术和企业家的帮助获得了很大的进展 ——
   <strong>
    他们看到了重视隐私的客户的庞大市场
   </strong>
   。
  </p>

  <p class="graf graf--p">
   所有东西都在被计算机视觉算法所监视，甚至我们服装上的图案也可能是颠覆性的。
  </p>
  <p class="graf graf--p">
   数字安全专业和时装设计师 Kate Rose 不只是设计纺织品图案。她通过制作
   <a class="markup--anchor markup--p-anchor" data-href="https://adversarialfashion.com/" href="https://adversarialfashion.com/" rel="noopener noreferrer" target="_blank">
    假车牌图案覆盖的衣服
   </a>
   ，尝试大规模污染车牌收集的数据库。
  </p>
  <p class="graf graf--p">
   如果你曾经在邮件中收到交通罚单（有时你甚至能看到你的车子的照片，照片中还有你自己和前排座位上的乘客），这表明你的车可能已经遇到了自动车牌阅读器或 ALPR。
  </p>
  <p class="graf graf--p">
   像如今城市的许多监控技术一样，ALPR 已经无处不在，无休止的永恒监视。
  </p>
  <p class="graf graf--p">
   它们适合安装在从公共电线杆到警车的所有东西上，但它们不仅仅是摄像机。ALPR 可捕获其范围内类似牌照的所有物品，每分钟可收集多达一千个牌照。
  </p>
  <p class="graf graf--p">
   因此，当您开车或乘坐共享时，这些系统会读取车牌并收集其 GPS 位置和登记信息，以及日期和时间。
  </p>
  <p class="graf graf--p">
   根据电子前沿基金会的说法，分享 ALPR 的不仅仅包括解决犯罪技术的执法机构，供应商也在其中：他们还有详细的监控系统，允许警察跟踪人们往返于康复中心和移民诊所等敏感地点的情况。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8713 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-33.png?resize=1100%2C672&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-33.png?w=1126&amp;ssl=1 1126w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-33.png?resize=300%2C183&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-33.png?resize=768%2C469&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-33.png?resize=1024%2C626&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-33.png?resize=1100%2C672&amp;ssl=1 1100w" data-recalc-dims="1" height="672" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-33.png?resize=1100%2C672&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   但是，与其他同类先进技术一样，它们很容易被看起来与真实物体足够接近的图案所愚弄。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “当一个 ALPR 环顾四周时，它会在相机视觉范围内寻找任何可能是矩形或字母数字组合的东西，这些字母数字表明对象是一个车辆牌照，” Rose说。系统会裁剪图像的该部分，然后与数据库中存储的牌照示例进行比较。
  </p>
  <p class="graf graf--p">
   如果系统确定目标物体是一个车牌，它就使用光学字符识别（OCR）来读取，识别和保存该牌照，以及它的 GPS 位置和被观察到的时间。
  </p>
  <p class="graf graf--p">
   Rose 的设计混乱了这个过程。她开发并测试了印制在面料上的重复图案，看起来尽可能接近真正的车牌，这样一来系统就可以将它保存为“真实的”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “把这种面料制成衣服，通过在街上穿戴这些设计，你可以帮助将垃圾数据引入监控系统 —— 可以大规模降低监视系统的运作效率，让老大哥运转起来更加昂贵，”她说。
  </p>
  <p class="graf graf--p">
   这就是反监视的行动主义。
  </p>
  <p class="graf graf--p">
   同样的思路，你可以以任何一种方式实践对大型监视系统的破坏；就如欧洲的活动家用合成人脸照片申请护照。
  </p>
  <p class="graf graf--p">
   Rose 在今年的拉斯维加斯举行的黑客大会 DEF CON 的演讲中介绍了她的工作。
  </p>
  <p class="graf graf--p">
   DEF CON 认为今年参加会议的女性人数
