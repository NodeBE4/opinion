  <p class="graf graf--p">
   集中营、种族主义、优生学、统治世界的野心。听起来有点熟吗？是的。有一个新的第三帝国正在出现，它就是中国。
  </p>
  <p class="graf graf--p">
   在地图上显示为中华人民共和国西北部的这个动荡地区的家庭正在被撕裂。被监禁的维吾尔族和哈萨克族父母的子女被“限制”在通过铁丝网和重型警察巡逻与外界隔离的“学校”里。他们被拒绝使用自己的语言教学，被迫学习普通话。这些控制措施是所谓的“汉化”政策的一部分，这是一种强制同化的计划。“汉”是中国主要民族的名称。
  </p>
  <p class="graf graf--p">
   由于维吾尔人和哈萨克人在营地中死亡人数相当多，北京
   <a class="markup--anchor markup--p-anchor" data-href="https://www.rfa.org/english/news/uyghur/crematoriums-06262018151126.html" href="https://www.rfa.org/english/news/uyghur/crematoriums-06262018151126.html" rel="noopener noreferrer" target="_blank">
    正在建造火葬场
   </a>
   ，以便在处置尸体，消除土葬的传统。
  </p>
  <p class="graf graf--p">
   这些集中营正在蔓延，这可能涉及危害人类罪。中国现在正在建设
   <a class="markup--anchor markup--p-anchor" data-href="https://theprint.in/defence/china-claims-it-has-no-gulags-but-satellite-imagery-shows-3-new-ones-coming-up-in-tibet/190940/" href="https://theprint.in/defence/china-claims-it-has-no-gulags-but-satellite-imagery-shows-3-new-ones-coming-up-in-tibet/190940/" rel="noopener noreferrer" target="_blank">
    类似的设施
   </a>
   ，在中国西南部的西藏，有各种委婉的名字，如“职业培训中心”。
  </p>
  <p class="graf graf--p">
   与此同时，北京正在重新尝试在全国范围内消除宗教信仰。正如佛教徒一样，基督徒在中国受到更大的攻击。中国的统治者要求五种公认的宗教“中国化” — — 官方认可是一种控制机制。
  </p>
  <p class="graf graf--p">
   作为这种无情努力的一部分，中国人正在摧毁清真寺和教堂，迫使虔诚的穆斯林
   <a class="markup--anchor markup--p-anchor" data-href="https://www.independent.co.uk/news/world/asia/china-muslims-xinjiang-pork-alcohol-lunar-new-year-spring-festival-uighur-islam-a8767561.html" href="https://www.independent.co.uk/news/world/asia/china-muslims-xinjiang-pork-alcohol-lunar-new-year-spring-festival-uighur-islam-a8767561.html" rel="noopener noreferrer" target="_blank">
    喝酒吃猪肉
   </a>
   ，派遣汉族官员住在穆斯林家中，并结束对未成年人的宗教教育。
  </p>
  <p class="graf graf--p">
   自习近平于2012年11月成为共产党总书记以来，这些在中国历史上有先例的尝试已经愈演愈烈。
  </p>
  <p class="graf graf--p">
   与此同时，习远远超过他的前任，一直在推动由一个主权国家、一
   <a class="markup--anchor markup--p-anchor" data-href="https://www.gatestoneinstitute.org/13497/china-sovereign-state" href="https://www.gatestoneinstitute.org/13497/china-sovereign-state" rel="noopener noreferrer" target="_blank">
    个中国国家统治的世界秩序的概念
   </a>
   。
  </p>
  <p class="graf graf--p">
   简而言之，至少除了大屠杀之外，习对世界的看法与第三帝国的看法非常相似。
  </p>
  <p class="graf graf--p">
   第三帝国与中国都有着致命的种族主义，在中国礼貌地称其为“大汉主义”。汉族据说包括中国人口的约92％。
  </p>
  <p class="graf graf--p">
   中国神话认为，所有中国人都是黄帝的后裔，中国人认为自己是与世界其他地方的人类有区别的，这种观点通过在学校灌输等方式得到加强。
  </p>
  <p class="graf graf--p">
   中国学者支持这种中国特殊性观念以及“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.scmp.com/magazines/post-magazine/long-reads/article/2154934/how-fossils-found-asia-could-rewrite-history" href="https://www.scmp.com/magazines/post-magazine/long-reads/article/2154934/how-fossils-found-asia-could-rewrite-history" rel="noopener noreferrer" target="_blank">
    北京人
   </a>
   ”的进化论，后者认为中国人与人类的其余部分没有共同的祖先。毫不奇怪，这种中国人独特进化的理论强化了种族主义的观点。
  </p>
  <p class="graf graf--p">
   由于种族主义，中国的许多人，包括官员，“相信自己与其他人类完全不同，并且暗示自己优于其他人类”，
   <span style="color: #993366;">
    “中国秩序：世界帝国与中国权力的本质”
   </span>
   的作者 Fei-Ling Wang 写道。
  </p>
  <p class="graf graf--p">
   因此，种族主义被制度化并公开宣传。去年中国中央电视台春节联欢晚会上的13分钟短剧中，这一点非常明显。一位中国女演员扮演肯尼亚母亲，她的胸部巨大​​，臀部大得可笑。更糟糕的是，她的搭档是一只人类大小的猴子。猴子和女人的结合是湖北省博物馆2017年的展览“
   <a class="markup--anchor markup--p-anchor" data-href="https://qz.com/africa/1101699/africans-in-china-are-infuriated-over-a-museum-exhibit-comparing-africans-to-animals/" href="https://qz.com/africa/1101699/africans-in-china-are-infuriated-over-a-museum-exhibit-comparing-africans-to-animals/" rel="noopener noreferrer" target="_blank">
    这是非洲
   </a>
   ”的回声，展示了非洲人在灵长类动物旁边的照片。
  </p>
  <p class="graf graf--p">
   近年来，中国媒体对非洲人进行了许多丑陋的描绘，虽然去年的短剧并不是最糟糕的，但也足够令人震惊，因为主要的国家广播公司通过播放给大约8亿观众，明确表示中国官员把非洲人视为嘲笑和非人的对象。
  </p>
  <p class="graf graf--p">
   在这种情况下，可以安全地假设这些观点是北京领导人所共有的，令人担忧的是，这种观点正在向所有华人 — — 而不仅仅是中国人 — — 提出更频繁的种族呼吁。
  </p>
  <p class="graf graf--p">
   然而，本世纪的帝国竞争有一个问题。中国现在是世界上人口最多的国家，但正面临人口迅速下降的问题。去年的出生率是1949年建国以来的最低水平。根据联合国人口司发布的“2017年世界人口前景”，该国的人口将在2029年达到峰值。
  </p>
  <p class="graf graf--p">
   但事实上联合国的数据是基于北京过于乐观的假设。中国的官方人口统计学家完全没有预见到去年出生率几乎崩溃。
  </p>
  <p class="graf graf--p">
   在2024年，将发生另一个重大事件。然后，至少300年来第一次 — — 也许是有史以来第一次 — — 中国将不会成为世界上人口最多的国家。这个荣誉将归于一个中国人普遍厌恶和恐惧的国家，印度。当印度在2061年达到巅峰时，其人口39888.8万将超过中国。
  </p>
  <p class="graf graf--p">
   一旦人口开始萎缩，中国将迅速萎缩。 2018年，中国的人口是美国的4.3倍；预计到2100年，中国的人口只会是2.3倍。
  </p>
  <p class="graf graf--p">
   中国的人口发展道路已经存在了几十年，它将对中国社会和国家的“综合国力”产生重大而极端不利的影响。也许这就是为什么北京看起来好像正在试图通过为“超级中国人”的种族奠定基础来弥补其人口崩溃的前景。
  </p>
  <p class="graf graf--p">
   深圳南方科技大学的贺建奎于11月宣布了他的基因编辑双胞胎女孩工程。他声称自己正在让婴儿能够更好地抵抗艾滋病毒，但有人猜测他也在努力增强智力。无论如何，这一声明引发了对纳粹优生学实验的联想，特别是因为有证据表明中国政府支持了他的“世界第一”实验，被认为是不道德的和危险的。
  </p>
  <p class="graf graf--p">
   习近平当然很危险。 “毛泽东可能在试图将前殖民地人民团结起来反对白人帝国主义者时，扮演了第三世界的种族怨恨，但他认为共产主义是一种全球现象，最终会到处找到一个家，而毛泽东的乌托邦在未来出现了，”哈德森研究所的 Charles Horner 告诉 Gatestone。 “习近平的中国共产党不是以这种方式全球性或乌托邦式的；相反，它似乎是一种被强调的’中国性’。”
  </p>
  <p class="graf graf--p">
   Horner 看到习近平的中国与20世纪30年代的帝国日本之间存在的令人不安的相似之处。 “就像当时的日本帝国一样，”Horner 说，“当一位善良的皇帝将整个世界聚集在一起沐浴在他的荣耀中并分享他的慷慨时，习近平和他的政党正向往这一神话般的过去。” ⚪️
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    C

    oncentration camps, racism, eugenics, ambitions of world domination. Sound familiar
   </a>
   ? There is a new Third Reich, and it is China.
  </p>
