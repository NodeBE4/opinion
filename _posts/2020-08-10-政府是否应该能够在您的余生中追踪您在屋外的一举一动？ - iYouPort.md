  <p class="graf graf--p">
   实际上，
   <strong class="markup--strong markup--p-strong">
    面部识别软件已经存在了近20年
   </strong>
   ，其熟练度和扩散性也在不断增长。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    人脸识别技术最初是迪士尼主题公园用的，用来帮游客挑照片做纪念品。而在占领华尔街运动其间，这项技术已经被用来抓捕抗议者了。如今，中国已经加入全球人脸识别的盛宴…… 详见《

  <p class="graf graf--p">
   执法只是冰山一角。营销、游戏和网络都是可以使用面部识别技术的领域。2016年，Pepperdine Law Review 进行的一项
   <a class="markup--anchor markup--p-anchor" data-href="http://pepperdinelawreview.com/wp-content/uploads/2016/06/Mohapatra%E2%80%94Final-No-ICR.pdf" href="http://pepperdinelawreview.com/wp-content/uploads/2016/06/Mohapatra%E2%80%94Final-No-ICR.pdf" rel="noopener noreferrer" target="_blank">
    研究
   </a>
   发现，人脸识别甚至可以用于医学中，通过评估面部结构来发现遗传状况。
  </p>
  <p class="graf graf--p">
   这是一个简单的道理：面部识别功能强大且有用。效果还不错。如果对话到此为止，那么它的快速发展似乎是天生的好事；但是，现实是：这项技术的开发和使用比规定其适当使用的法律的增长要快得多。对许多人来说，这是一个严重的问题。
  </p>
  <p class="graf graf--p">
   简答说，你无法限制它 *只* 用在正确的地方。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    缺乏适当的法律准则
   </strong>
  </h3>
  <p class="graf graf--p">
   关于如何使用人脸识别，尚无明确的界限。没有法律告诉面部识别使用者他们可以走多远，或者至少现在能走多远。
  </p>
  <p class="graf graf--p">
   在美国，人们所拥有的仅仅是一些最高法院的案件，这些案件为政府使用侵入性技术提供了基本准则。
  </p>
  <p class="graf graf--p">
   1968年的《窃听法》是源于 Katz 诉美国和 Berger 诉纽约的裁决，它阻止了执法人员窃听嫌疑人的电话。在2017年 Carpenter 诉美国一案中，最高法院裁定，政府在没有手令的情况下不能追踪公民的电话。
  </p>
  <p class="graf graf--p">
   就人脸识别而言，1974年的《隐私法》规定了专门针对FBI使用的人脸识别软件的限制规则。
  </p>
  <p class="graf graf--p">
   还有其他许多科技公司、组织、甚至州都已开始限制人脸识别软件的使用。
  </p>

  </p>
  <ul class="postList">
   <li class="graf graf--li">

    发现
   </a>
   ：“采用面部识别技术是在没有任何有意义的监督的情况下进行的 …… 并且没有颁布防止内部和外部滥用的法律保护措施。”
  </p>
  <p class="graf graf--p">
   在研究了有关人脸识别的法律先例后，波士顿大学期刊的作者 Sharon Nakar 和 Dov Beenbaum 得出
   <a class="markup--anchor markup--p-anchor" data-href="https://advance-lexis-com.ezproxy.phc.edu/document/?pdmfid=1516831&amp;crid=f627a500-7d97-4d1c-a414-6f26865a7ab1&amp;pddocfullpath=%2Fshared%2Fdocument%2Fanalytical-materials%2Furn%3AcontentItem%3A5NBW-3KK0-00B1-91DG-00000-00&amp;pddocid=urn%3AcontentItem%3A5NBW-3KK0-00B1-91DG-00000-00&amp;pdcontentcomponentid=148851&amp;pdteaserkey=sr0&amp;pditab=allpods&amp;ecomp=sp79k&amp;earg=sr0&amp;prid=b3fba66e-e4c3-4e00-8720-71611666fb43" href="https://advance-lexis-com.ezproxy.phc.edu/document/?pdmfid=1516831&amp;crid=f627a500-7d97-4d1c-a414-6f26865a7ab1&amp;pddocfullpath=%2Fshared%2Fdocument%2Fanalytical-materials%2Furn%3AcontentItem%3A5NBW-3KK0-00B1-91DG-00000-00&amp;pddocid=urn%3AcontentItem%3A5NBW-3KK0-00B1-91DG-00000-00&amp;pdcontentcomponentid=148851&amp;pdteaserkey=sr0&amp;pditab=allpods&amp;ecomp=sp79k&amp;earg=sr0&amp;prid=b3fba66e-e4c3-4e00-8720-71611666fb43" rel="noopener noreferrer" target="_blank">
    结论
   </a>
   ：“美国法院似乎在是否存在匿名权方面 —— 保护人们免受[面部识别技术追踪]方面 —— 的分歧最大。 …… 在这项技术变得更加牢固之前，需要做出新的努力以在所有利益相关者之间达成共识。”
  </p>
  <p class="graf graf--p">
   新泽西，EFF 和这些作者都关心同一件事：人脸识别将如何影响隐私的未来。谁可以使用面部识别技术？用户是否可以随时访问数据？如果您不希望自己的脸出现在可搜索的数据库中该怎么办？……
  </p>
  <p class="graf graf--p">
   这些是要考虑的许多不同组件。但是，它们都取决于同一场斗争：在公民的隐私人权与面部识别可能带来的潜在利益之间如何取得平衡。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    隐私人权和面部识别技术
   </strong>
  </h3>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “隐私权”一词从未出现在美国宪法中。然而，隐私权的概念已成为数百万人生活中的重要法律组成部分。
  </p>
  <p class="graf graf--p">
   因此，如果它不是来自宪法，那么隐私权又是哪里来的呢？在1965年的 Griswold 诉康涅狄格州一案中，最高法院裁定，隐私权不是显性权利，而是隐含在第一、第三、第四和第九修正案中。
  </p>
  <p class="graf graf--p">
   他们的推理是这样的：如果公民有权使用其言论自由，如果政府不能将军队部署在人们家中，如果公民有权保护自己的人员、文件和房屋免受不合理的搜查和没收，如果宪法中的权利可能并不详尽，宪法确实表明，公民与国家之间存在一定的空间，政府不得跨越。
  </p>
  <p class="graf graf--p">
   这就是政府即使不认为您可能在做违法的事，也常常不能在没有授权的情况下搜查您手机的原因。这就是政府不能使用热成像技术偷窥您房屋内部的原因。而且，如前所述，这就是政府不能使用您的电话数据来跟踪您的原因。
  </p>
  <p class="graf graf--p">
   但是，对于一张脸来说呢？政府可以使用脸孔来查找涉嫌犯罪的人的身份和一切在线信息吗？也许 …… 那要看你在哪里了。
  </p>
  <p class="graf graf--p">
   回想 Reynolds 的案子。在决定从 Acre 五金店偷两个烤架和一个吸尘器之前，Reynolds 只是一个普通的、无辜的佛罗里达人。即便如此，有一件事使她的脸可被搜索。她做了非法的事 —— 并且在公开场合做的。
  </p>
  <p class="graf graf--p">
   她做违法的事使警察有理由搜查她的脸，但是，这并不是重要的部分。
  </p>
  <p class="graf graf--p">
   这里的不同在于，Reynolds 当时不在私人空间中。
  </p>
  <p class="graf graf--p">
   警察认为，Reynolds 踏入公共场所的那一刻，她就失去了隐私权，否则本来可以保护自己的脸免受人脸识别搜索。这就是为什么。
  </p>
  <p class="graf graf--p">
   全世界的警察都这么认为。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    只要你走出这扇门，一切就结束了
   </strong>
  </h3>
  <p class="graf graf--p">
   1967年，最高法院在 “Katz诉美国” 案中
   <a class="markup--anchor markup--p-anchor" data-href="https://supreme.justia.com/cases/federal/us/389/347/" href="https://supreme.justia.com/cases/federal/us/389/347/" rel="noopener noreferrer" target="_blank">
    裁定
   </a>
   ，一个人有意向公众公开的物品不再受《第四修正案》保护。
  </p>
  <p class="graf graf--p">
   换句话说，对所有人都是一样的 —— “您不能在公共场合主张隐私权”。
  </p>
  <p class="graf graf--p">
   通常，这并不是真正的问题。毕竟，人们通常不会拿走你的地址，姓名和在线信息，到处传播。
  </p>
  <p class="graf graf--p">
   但是，每当您穿过购物中心、Ace五金店、或繁忙的街道旁时，您都愿意将自己的脸露出来。
   <strong class="markup--strong markup--p-strong">
    而只要有了一张照片或来自监视摄像机的录像，掌握人脸识别技术的任何人都有机会找到该面孔可能在线出现的任何地方
   </strong>
   。
  </p>
  <p class="graf graf--p">
   目前尚无明确答案。取缔拍照和监控摄像头实际上是不可行的。政府无法简单地禁止私人公司使用人脸识别，而且它太强大了，也无法从执法部门手中拿走。
  </p>
  <p class="graf graf--p">
   从政策的角度来看，要在人脸识别的好处和隐私人权之间找到平衡，这将是未来几年必须解决的问题，主要是通过反复试验。它使个人利益与更大社区的潜在利益相抵触。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    以所谓的国家安全的名义
   </strong>
  </h3>
  <p class="graf graf--p">
   与朋友见面在购物中心吃午饭听起来像是度过一个星期六下午的平凡模式。如果没有那两个细节的话。而对于 Alim 来说可能就是这种情况。Alim 恰好是一名维吾尔族穆斯林、新疆居民。
  </p>
  <p class="graf graf--p">
   根据NPR的说法，要进入中国购物中心，您必须经过一个安检站，该检查站看起来像是金属探测器和地铁入口那种安检机器的组合。
  </p>
  <p class="graf graf--p">
   您需要刷政府签发的身份证，用监视摄像头扫描自己的脸，才能进去。
  </p>
  <p class="graf graf--p">
   Alim 的朋友经过扫描并顺利通过。但是当 Alim 刷自己的身份证时，他得到了截然不同的结果。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我扫描了我的ID… 几乎瞬间橙色警报就响起来了，” Alim 对NPR的记者说。橙色警报是为识别到潜在的恐怖分子和罪犯嫌疑人的名字时的反应。
  </p>
  <p class="graf graf--p">
   片刻之后，警察就跑来把 Alim 带走了。
  </p>
  <p class="graf graf--p">
   经过短暂的审问，Alim 被释放，并建议他如果不想在其他安检站再次遭到拘留和审讯，那就最好留在家里。
  </p>
  <p class="graf graf--p">
   但是政府不需要检查站也能随时找到他。通过使用面部识别，中国执法人员可以在交通路口、便利店、银行或机场等任何地方找到 Alim。
  </p>
  <p class="graf graf--p">
   只要他出现在任何摄像机的范围内，无论他走到哪里，Alim 的“橙色警报”状态都会跟随着他。
  </p>
  <p class="graf graf--p">
   但是为什么要监视？Alim 家乡新疆的警察认为，存在所谓的 “三大弊端：极端主义，恐怖主义和分裂主义”。维族人被认为是不同的。他们是穆斯林。他们具有非常独特的文化身份。它们不是政府希望看到的“中国”版本。维吾尔族穆斯林被视为对中国同质化的挑战。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    被监视破坏的穆斯林公民的反抗案例，值得学习《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e9%ab%98%e6%98%8e%e7%9a%84%e5%8f%8d%e6%8a%97/" href="https://www.iyouport.org/%e9%ab%98%e6%98%8e%e7%9a%84%e5%8f%8d%e6%8a%97/" rel="noopener noreferrer" target="_blank">
     高明的反抗
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   卫报
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/news/2019/apr/11/china-hi-tech-war-on-muslim-minority-xinjiang-uighurs-surveillance-face-recognitionhttps://www.theguardian.com/news/2019/apr/11/china-hi-tech-war-on-muslim-minority-xinjiang-uighurs-surveillance-face-recognition" href="https://www.theguardian.com/news/2019/apr/11/china-hi-tech-war-on-muslim-minority-xinjiang-uighurs-surveillance-face-recognitionhttps://www.theguardian.com/news/2019/apr/11/china-hi-tech-war-on-muslim-minority-xinjiang-uighurs-surveillance-face-recognition" rel="noopener noreferrer" target="_blank">
    说
   </a>
   ，使用面部识别监视像 Alim 这样的人是中国版反恐战争的一部分，“目标不是外国人，而是国内少数族裔，这些族裔似乎在威胁中共的独裁统治。”
  </p>
  <p class="graf graf--p">
   即使 Alim 从未进过监狱、从未被被捕或触犯法律，他的种族和文化特征也足以使他成为全天候被监视的目标。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    别忘了《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%a5%bf%e6%96%b9%e5%ad%a6%e6%9c%af%e7%95%8c%e5%b8%ae%e5%8a%a9%e6%9e%84%e5%bb%ba%e4%ba%86%e4%b8%ad%e5%9b%bd%e7%9a%84%e8%87%aa%e5%8a%a8%e5%8c%96%e7%a7%8d%e6%97%8f%e4%b8%bb%e4%b9%89/https://www.iyouport.org/%e8%a5%bf%e6%96%b9%e5%ad%a6%e6%9c%af%e7%95%8c%e5%b8%ae%e5%8a%a9%e6%9e%84%e5%bb%ba%e4%ba%86%e4%b8%ad%e5%9b%bd%e7%9a%84%e8%87%aa%e5%8a%a8%e5%8c%96%e7%a7%8d%e6%97%8f%e4%b8%bb%e4%b9%89/" href="https://www.iyouport.org/%e8%a5%bf%e6%96%b9%e5%ad%a6%e6%9c%af%e7%95%8c%e5%b8%ae%e5%8a%a9%e6%9e%84%e5%bb%ba%e4%ba%86%e4%b8%ad%e5%9b%bd%e7%9a%84%e8%87%aa%e5%8a%a8%e5%8c%96%e7%a7%8d%e6%97%8f%e4%b8%bb%e4%b9%89/https://www.iyouport.org/%e8%a5%bf%e6%96%b9%e5%ad%a6%e6%9c%af%e7%95%8c%e5%b8%ae%e5%8a%a9%e6%9e%84%e5%bb%ba%e4%ba%86%e4%b8%ad%e5%9b%bd%e7%9a%84%e8%87%aa%e5%8a%a8%e5%8c%96%e7%a7%8d%e6%97%8f%e4%b8%bb%e4%b9%89/" rel="noopener noreferrer" target="_blank">
     西方学术界帮助构建了中国的自动化种族主义
    </a>
    》
   </li>
  </ul>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    于是您知道了
   </strong>
  </h3>
  <p class="graf graf--p">
   一切取决于当权者的价值观。这不仅是不公平的，而且非常危险。每一个当权者都会将威胁到自身权力稳固的一切视为敌人。
  </p>
  <p class="graf graf--p">
   面部识别技术本身很可能无法阻挡，但是避免被捕捉目前仍然是可行的。具体方法见
   <a class="markup--anchor markup--p-anchor" data-href="https://start.me/p/xbYXdR/iyp-1" href="https://start.me/p/xbYXdR/iyp-1" rel="noopener noreferrer" target="_blank">
    列表- 1 中的 “生物识别和防御方法”
   </a>
   。
  </p>
  <p class="graf graf--p">
   不过，我们不能一直停留在躲藏的层面上。
  </p>
  <p class="graf graf--p">
   那是那句话，如果技术不能是民主的，那么社会就无法是民主的。人脸识别也一样，它可以帮助NGO组织寻找失踪儿童、可以帮助公民调查人员追踪权力关系内幕、可以帮助记者寻求真相 ……
  </p>
  <p class="graf graf--p">
   于是重点在于，让公民掌握技术。
  </p>

   <li class="graf graf--li">
