  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     1、什么是 “黑钱”？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   竞选活动依靠金钱来进行。
  </p>
  <p class="graf graf--p">
   钱用来支付薪水，旅行，尤其是广告。个人不富裕的候选人依靠捐款来支付这些费用，或者依靠政党和其他政治团体的支持性支出。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e9%80%89%e4%b8%be%e8%bf%90%e5%8a%a8%e7%9a%84%e6%95%b0%e5%ad%97%e5%8c%96%e6%96%97%e4%ba%89-2020-presidential-race/" href="https://www.iyouport.org/%e9%80%89%e4%b8%be%e8%bf%90%e5%8a%a8%e7%9a%84%e6%95%b0%e5%ad%97%e5%8c%96%e6%96%97%e4%ba%89-2020-presidential-race/" rel="noopener noreferrer" target="_blank">
     特朗普的竞选经理承认花了10亿美元做广告
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   自70年代水门事件以来，联邦法律已对政治捐款施加了限制，并要求候选人向联邦选举委员会披露 “大多数” 用于竞选活动的捐款的来源。
  </p>
  <p class="graf graf--p">
   大多数州的选举都有类似的法律。
  </p>
  <p class="graf graf--p">
   在制定披露要求后的最初几年中，大多数联邦竞选支出都被披露了。但是，随着不断变化的竞选方式，尤其是，既不是候选人委员会也不是政党的外部团体的作用日益增强，使得一些大型捐助者得以掩盖更多的捐助。
  </p>
  <p class="graf graf--p">
   从2010年开始，阳光基金会的竞选财务观察员开始将其中一些不受监管的基金称为“黑钱”。这个词在 Jane Mayer 的2016年最畅销的书中得到了普及。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “黑钱”是指未披露来源的竞选资金。
  </p>
  <p class="graf graf--p">
   支出（例如，做那种批评对手的电视广告）通常会向FEC报告，但
   <strong class="markup--strong markup--p-strong">
    不会披露
   </strong>
   支付的人、公司或组织的身份。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e9%80%89%e4%b8%be%e5%a6%82%e4%bd%95%e6%88%90%e4%b8%ba%e4%b8%80%e9%97%a8%e7%94%9f%e6%84%8f%ef%bc%9a%e7%a7%81%e4%ba%ba%e6%83%85%e6%8a%a5%e5%b8%82%e5%9c%ba%e5%92%8c%e7%8e%b0%e5%ae%9e%e7%9a%84%e6%9c%89/" href="https://www.iyouport.org/%e9%80%89%e4%b8%be%e5%a6%82%e4%bd%95%e6%88%90%e4%b8%ba%e4%b8%80%e9%97%a8%e7%94%9f%e6%84%8f%ef%bc%9a%e7%a7%81%e4%ba%ba%e6%83%85%e6%8a%a5%e5%b8%82%e5%9c%ba%e5%92%8c%e7%8e%b0%e5%ae%9e%e7%9a%84%e6%9c%89/" rel="noopener noreferrer" target="_blank">
     选举如何成为一门生意：私人情报市场和现实的有偿操纵者
    </a>
    》
   </li>
  </ul>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     2、“黑钱” 有什么问题？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   缺乏信息披露使新闻记者、监管者和反对者更难以发现违反竞选财务法的行为，例如外国捐助者或政府承包商的非法捐款、以及超出法定限额的捐款。
  </p>
  <p class="graf graf--p">
   它还隐藏了来自不可靠来源的非法捐款比如 Harvey Weinstein 或 Bernie Madoff。
  </p>
  <p class="graf graf--p">
   而且这还不是全部。缺乏信息披露也使选民无法获得真正有价值的信息，也就难以准确判断。
  </p>
  <p class="graf graf--p">
   正如美国最高法院在
   <a class="markup--anchor markup--p-anchor" data-href="https://supreme.justia.com/cases/federal/us/424/1/" href="https://supreme.justia.com/cases/federal/us/424/1/" rel="noopener noreferrer" target="_blank">
    Buckley 诉 Valeo 案
   </a>
   中所显示的那样，1976年具有里程碑意义的裁决支持联邦竞选活动公开法，确定了候选人的财务支持者，“使投票人能看清候选人最有可能响应的利益”。
  </p>
  <p class="graf graf--p">
   当所有候选人都在同一个政党时，这在初选中尤为重要，而选民不能依靠政党的标签来决定要投票给谁。
  </p>
  <p class="graf graf--p">
   举例来说，在目前的民主党总统竞选中，公开允许选民核实候选人对捐赠者的要求，这表明前副总统乔·拜登（Joe Biden）
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2019/04/25/joe-biden-presidential-bid-lobbyists-fundraiser/" href="https://theintercept.com/2019/04/25/joe-biden-presidential-bid-lobbyists-fundraiser/" rel="noopener noreferrer" target="_blank">
    有多少说客
   </a>
   ，参议员伊丽莎白·沃伦（Elizabeth Warren）从大型
   <a class="markup--anchor markup--p-anchor" data-href="https://www.opensecrets.org/members-of-congress/summary?cid=N00033492" href="https://www.opensecrets.org/members-of-congress/summary?cid=N00033492" rel="noopener noreferrer" target="_blank">
    个人捐助者
   </a>
   那里获得了近30％的资金，以及某些
   <a class="markup--anchor markup--p-anchor" data-href="https://prospect.org/article/do-pete-buttigiegs-donors-know-him-better-we-do" href="https://prospect.org/article/do-pete-buttigiegs-donors-know-him-better-we-do" rel="noopener noreferrer" target="_blank">
    硅谷捐助者
   </a>
   正在支持南本德市长皮特·布蒂吉格（Pete Buttigieg）。
  </p>
  <p class="graf graf--p">
   了解这些细节有助于选民了解候选人当选后可能倾向什么利益。
  </p>
  <p class="graf graf--p">
   可以肯定的是，披露受到批评。去年夏天，由于众议员华金·卡斯特罗（Joaquin Castro）
   <a class="markup--anchor markup--p-anchor" data-href="https://thehill.com/homenews/house/456902-house-conservatives-call-for-ethics-probe-into-joaquin-castro-tweet" href="https://thehill.com/homenews/house/456902-house-conservatives-call-for-ethics-probe-into-joaquin-castro-tweet" rel="noopener noreferrer" target="_blank">
    在推特上
   </a>
   向特朗普推销大型捐助者的姓名和雇主而引起的争执 —— 所有这些都已按法律要求予以披露 —— 突显了某些人的担心，即 “披露是侵犯了捐助者的隐私”。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     3、究竟有多少黑钱？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   由于其真的非常 “黑暗”，因此很难知道究竟有多少黑钱，但是有理由相信这一数字很大、并且还在不断增长。
  </p>
  <p class="graf graf--p">
   请记住：有些黑钱支出是在未披露捐赠人的情况下向联邦选举委员会报告的，而有些黑钱支出则根本没有报告。
  </p>
  <p class="graf graf--p">
   竞选资金监管机构 Center for Responsive Politics
   <a class="markup--anchor markup--p-anchor" data-href="https://www.opensecrets.org/news/2016/11/1-4-billion-and-counting-in-spending-by-super-pacs-dark-money-groups/" href="https://www.opensecrets.org/news/2016/11/1-4-billion-and-counting-in-spending-by-super-pacs-dark-money-groups/" rel="noopener noreferrer" target="_blank">
    发现
   </a>
   ，黑钱团体报告称在2016年联邦选举中花费了1.81亿美元。在过去十年中，黑钱占候选人和政党以外的团体支出的近五分之一。
  </p>
  <p class="graf graf--p">
   但是，向FEC报告的支出很可能仅代表
   <a class="markup--anchor markup--p-anchor" data-href="https://www.opensecrets.org/news/2019/02/somp3-billion-dollar-dark-money-tip-of-the-iceberg/" href="https://www.opensecrets.org/news/2019/02/somp3-billion-dollar-dark-money-tip-of-the-iceberg/" rel="noopener noreferrer" target="_blank">
    “冰山一角”
   </a>
   。只要竞选活动的支出不予披露，黑钱的总数是不可知的。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     4、为什么可以隐藏捐款？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   黑钱源于竞选财务法律中的空白。
  </p>
  <p class="graf graf--p">
   联邦选举法最明确地涉及候选人、政党和主要是为候选人和政党提供支持的政治委员会的报告和披露。但是，不要忘了其他组织也会参与进来。
  </p>
  <p class="graf graf--p">
   这些组织包括美国商会等商业团体和贸易协会、美国全国步枪协会或保护选民联盟等会员组织、工会以及很多不同的意识形态团体。
  </p>
  <p class="graf graf--p">
   这些团体
   <strong class="markup--strong markup--p-strong">
    不必
   </strong>
   报告其捐助者，因为他们声称致力于解决问题，而不代表特定候选人。
  </p>
  <p class="graf graf--p">
   有一些例外。如果该团体在广告宣传运动上花费数额巨大，则必须报告该支出以及专门帮助购买该广告的任何捐助者。
  </p>
  <p class="graf graf--p">
   但是法律只涵盖了某些宣传活动。只有字面上要求击败明确指出的候选人的广告（
   <a class="markup--anchor markup--p-anchor" data-href="https://supreme.justia.com/cases/federal/us/424/1/" href="https://supreme.justia.com/cases/federal/us/424/1/" rel="noopener noreferrer" target="_blank">
    “express advocacy”
   </a>
   ）才是可以公开的竞选广告。
  </p>
  <p class="graf graf--p">
   这意味着组织可以对广告中的候选人进行猛烈抨击或热烈赞扬，但只要能避免呼吁人们如何投票就可以避免披露捐赠者。
  </p>
  <p class="graf graf--p">
   2002年，国会通过了麦凯恩-法因戈尔德法 (McCain-Feingold Act)，该法将披露范围扩大到包括广播广告，这些广播广告涉及在初选前30天和大选前60天提及候选人，但未覆盖竞选季初期播出的类似广告。
  </p>
  <p class="graf graf--p">
   由于许多名义上非政治性的组织间接为竞选广告提供资金，因此追随这笔钱变得更加困难。他们通过捐赠给另一个购买广告的团体来做到这一点。有时，即使是第二个团体，也要在购买广告之前将资金转移到第三家组织。
  </p>
  <p class="graf graf--p">
   这种 “俄罗斯套娃” 的做法绕开了任何透明度要求。
  </p>
  <p class="graf graf--p">
   即使实际进行支出的组织必须向联邦选举委员会披露其大额捐助者，但这些报告仅列出了该链条中下一个环节的贡献 —— 并未告诉选民任何有关
   <a class="markup--anchor markup--p-anchor" data-href="https://www.issueone.org/wp-content/uploads/2018/09/Dark-Money-Illuminated-Report.pdf" href="https://www.issueone.org/wp-content/uploads/2018/09/Dark-Money-Illuminated-Report.pdf" rel="noopener noreferrer" target="_blank">
    实际支付者
   </a>
   的信息。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     5、该怎么办？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   严重的 “黑钱” 问题也许有可能通过立法修正来解决。
  </p>
  <p class="graf graf--p">
   首先，可能要求所有组织（包括从事选举相关支出的公司、工会和非营利组织）披露其资金用于竞选广告的大型捐助方。
  </p>
  <p class="graf graf--p">
   其次，当披露的捐赠来自链条下游的组织时，披露的内容需要包括该组织的主要捐赠者。一些州，例如新泽西州和科罗拉多州，最近通过了要求提供该信息的法律。
  </p>
  <p class="graf graf--p">
   去年春季美国众议院通过了类似的立法，针对联邦选举中的大量黑钱捐献者和支出者，尽管参议院的共和党人基本不太可能接受。
  </p>
  <p class="graf graf--p">
   可以肯定的是，仍然存在一些宪法问题，尤其是与选举相关的广告的定义问题。但是由于披露不会限制或禁止竞选资金的使用并增加选民信息，法院经常认定披露与《第一修正案》相一致。
  </p>
  <p class="graf graf--p">
   换句话说，与竞选融资改革中的许多其他问题不同，改善披露的障碍是政治上的，而不是宪法上的。
  </p>
