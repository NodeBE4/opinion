  <p class="graf graf--p">
   西巴布亚地区历史悠久。这个前荷兰殖民地于1961年宣布独立，直到1969年被印度尼西亚吞并。
  </p>
  <p class="graf graf--p">
   联合国监督了关于西巴布亚独立的极富争议性的投票，这被称为《自由选择法》。据报道，只有1025人被印度尼西亚军方选中投票。尽管结果使印尼正式控制了该地区，但许多人批评这一过程存在严重缺陷。
  </p>
  <p class="graf graf--p">
   在2019年，分离主义团体继续为独立而战，并呼

    吁进行新的全民公投
   </a>

  </p>
  <p class="graf graf--p">
   当八月在巴布亚爆发数十年来最严重的暴力事件时，激发了种族歧视的指控和对独立的重新呼吁，

     封锁了对该省部分地区的互联网访问
    </strong>
   </a>

   </strong>
   以令当权者控制对“事实”的陈述。

  </p>
  <p class="graf graf--p">

   <img alt="" class="aligncenter size-full wp-image-9718 jetpack-lazy-image" data-lazy-sizes="(max-width: 1070px) 100vw, 1070px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/30.png?resize=1070%2C596&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/30.png?w=1070&amp;ssl=1 1070w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/30.png?resize=300%2C167&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/30.png?resize=768%2C428&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/30.png?resize=1024%2C570&amp;ssl=1 1024w" data-recalc-dims="1" height="596" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/30.png?resize=1070%2C596&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1070"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9718" data-recalc-dims="1" height="596" sizes="(max-width: 1070px) 100vw, 1070px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/30.png?resize=1070%2C596&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/30.png?w=1070&amp;ssl=1 1070w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/30.png?resize=300%2C167&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/30.png?resize=768%2C428&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/30.png?resize=1024%2C570&amp;ssl=1 1024w" width="1070"/>

  <p class="graf graf--p">
   这里使用的工具是 Twint。我们以前也介绍过，见：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%96%b0%e7%94%9f%e6%83%85%e6%8a%a5%e5%b7%a5%e5%85%b7%ef%bc%882%ef%bc%89%ef%bc%9a%e5%88%ae%e5%88%80%e3%80%81%e6%89%93%e7%a0%b4%e6%8e%a8%e7%89%b9%e9%99%90%e5%88%b6%e3%80%81%e6%83%85%e6%8a%a5%e8%b5%84/" href="https://www.iyouport.org/%e6%96%b0%e7%94%9f%e6%83%85%e6%8a%a5%e5%b7%a5%e5%85%b7%ef%bc%882%ef%bc%89%ef%bc%9a%e5%88%ae%e5%88%80%e3%80%81%e6%89%93%e7%a0%b4%e6%8e%a8%e7%89%b9%e9%99%90%e5%88%b6%e3%80%81%e6%83%85%e6%8a%a5%e8%b5%84/" rel="noopener noreferrer" target="_blank">
    新生情报工具（2）：刮刀、打破推特限制、情报资源机器人、和袜子木偶管家
   </a>
   》。
  </p>
  <p class="graf graf--p">
   使用以下代码提取所有推文：
  </p>
  <pre class="graf graf--p"><em class="markup--em markup--p-em">sudo twint -u bellanow1 –media -o papuabots1 –csv</em></pre>

  <p class="graf graf--p">
   现在拥有的电子表格包含 @ bellanow1 机器人帐户中的所有推文数据。命令 Twint 将其拉入csv文件时，它会逐项列出。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9720 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/32.png?resize=1100%2C502&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/32.png?w=1387&amp;ssl=1 1387w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/32.png?resize=300%2C137&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/32.png?resize=768%2C351&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/32.png?resize=1024%2C467&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/32.png?resize=1100%2C502&amp;ssl=1 1100w" data-recalc-dims="1" height="502" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/32.png?resize=1100%2C502&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   对 @ kevinma40204275 重复了相同的过程，以将两者匹配并进行比较。在将两者排列对比时可以看到该信息战活动指示自动化的流程。
  </p>
  <p class="graf graf--p">
   这些帐户以一种自动流程的方式彼此同步，以便在更大的机器人网络中以不同的时间发布内容。
  </p>
  <p class="graf graf--p">
   通过查看序列可以看到机器人在25日晚上进行了活动，活动时间为 23:32:52, 23:32:54, 0:32:54, 0:32:56, 3:32:54 和 3:32:55，整个时间线很有规律。
  </p>

   <img alt="" class="aligncenter size-full wp-image-9721 jetpack-lazy-image" data-lazy-sizes="(max-width: 422px) 100vw, 422px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/33.png?resize=422%2C497&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/33.png?w=422&amp;ssl=1 422w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/33.png?resize=255%2C300&amp;ssl=1 255w" data-recalc-dims="1" height="497" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/33.png?resize=422%2C497&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="422"/>

  <p class="graf graf--p">
   结论是，该机器人网络利用其他主要的社交媒体平台来传播有关印尼政府的宣传，并且淹没了任何反印尼政府的言论。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bellingcat.com/wp-content/uploads/2019/10/Investigating_Information_Operations_in_West_Papua.pdf" href="https://www.bellingcat.com/wp-content/uploads/2019/10/Investigating_Information_Operations_in_West_Papua.pdf" rel="noopener noreferrer" target="_blank">
    在这里看到完整版报告
   </a>
   。
  </p>
  <p class="graf graf--p">
   以上讲述呈现的是对机器人信息战网络的调查。这个机器人网络可以说稍有特殊（至少30万美元真没白花）因为他们获得了较大的影响力。
  </p>
  <p class="graf graf--p">
   而一般的信息战网络可以不需要宣传账户自身的影响力，他们只需要以大量的涌入来推动主流叙事向操纵者所希望的方向倾斜；当有人搜索相关关键字时，宣传性信息将被置于前列。所以，及时抓取、调查并排除信息战网络非常重要。
  </p>
  <p class="graf graf--p">
   就如我们
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    曾经介绍和演示的所有调查
   </a>
   一样，您可以举一反三应用在您身边最重要的事件上。尤其是其中涉及的工具和调查思路，它们可以用来进行很多实践。⚪️
  </p>
