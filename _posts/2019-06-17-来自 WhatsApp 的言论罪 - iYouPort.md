  <p class="graf graf--p">
   人权观察指，印度尼西亚司法委员会应审查泗水地方法院（Surabaya）起诉 Saidah Saleh Syamlan 的刑事诽谤罪名。
  </p>
  <p class="graf graf--p">
   2019年2月26日，法院以涉嫌诽谤罪判处 Syamlan 10个月的监禁，涉事的具体内容是通过 WhatsApp 向两家银行发送了四条关于公司业绩的信息。她否认发送消息。人权观察说，印尼政府应废除限制在线和平言论自由的刑事条款。
  </p>
  <p class="graf graf--p">
   印度尼西亚研究人员 Andreas Harsono 表示，“根据发送 WhatsApp 的信息批评公司的脆弱证据将某人送进监狱，将对言论自由产生灾难性的影响。” “Saidah Syamlan 的案件表明，刑事诽谤法可能会受到具有政治或财政权力的人的操纵，这些人可能会影响调查人员的行为。”
  </p>
  <p class="graf graf--p">
   Syamlan 的丈夫 Aziz Hamedan 长期担任 Pisma Textile 的财务总监，但在2016年退休。WhatsApp 的消息是用爪哇语发表的四个短句，分别于2017年6月23日发送给雅加达的两位银行家 — 印度尼西亚进出口银行和印度尼西亚国家银行的官员 — 内容是质疑 Pista Textile 的信誉，Pista Textile 是一家著名的纺织公司。
  </p>
  <p class="graf graf--p">
   2017年9月12日，
   <strong class="markup--strong markup--p-strong">
    Pisma Textile 的律师 Muhammad Bayu Kusharyanto 向泗水警方报告了“该手机号码的拥有者”，指控发件人诽谤该公司，并提供给警方聊天消息截图
   </strong>
   。
  </p>
  <p class="graf graf--p">
   2017年10月4日，泗水警方询问了 Hamedan 和 Syamlan。两人都否认发送消息。
  </p>
  <p class="graf graf--p">
   Syamlan 承认显然用于发送这些消息的号码是她的，但她说很长时间没有使用这部电话了。说，手机已经坏了，并且在2017年4月丢了。她说她之后使用了一个不同号码的新手机。
  </p>
  <p class="graf graf--p">
   Syamlan 本人是一名出生于雅加达的阿拉伯人，她在法庭上说她不讲爪哇语，尽管爪哇语在泗水广泛使用。大多数印度尼西亚公民使用本国语言，印度尼西亚语和他们各自的母语。
  </p>
  <p class="graf graf--p">
   2018年1月15日，警察搜查了 Hamedan 和 Syamlan 的家。根据 Syamlan 的说法，警方没有发现任何设备连接到这些消息。然而，在2018年3月，警方指控 Syamlan 犯有刑事诽谤罪。Syamlan 告诉泗水警察局的内部安全部门，她觉得自己受到了骚扰和恐吓，并向该部门投诉，但警方调查人员很快将她的案件送到了泗水检察院办公室。
  </p>
  <p class="graf graf--p">
   她的审判发生在2018年10月至2019年2月之间。根据第27（3）条和第45（3）条，检察官要求判处18个月徒刑和5亿印尼盾（35,500美元）的罚款，这些条款涉及“电子诽谤”罪名，来自2016年泰国的电子信息与交易法案（互联网法）。
  </p>
  <p class="graf graf--p">
   2019年2月26日，泗水地区法院判定 Syamlan 犯有诽谤 Pisma Textile 的罪行，判处她10个月监禁，并处以500万印尼盾（355美元）的罚款。她仍然可以上诉。
  </p>
  <p class="graf graf--p">
   2010年，人权观察发布了对印度尼西亚刑事诽谤法（包括2008年互联网法）的负面影响的分析，并敦促废除这些法律。
   <strong class="markup--strong markup--p-strong">
    这些法律可以被有权势的人利用，以报复那些指控腐败、欺诈或指出强大利益或政府官员的不当行为的人
   </strong>
   。
  </p>
  <p class="graf graf--p">
   2016年，众议院修改了2008年互联网法，将诽谤的监禁期限从六年减少到四年，但如果通过互联网传播所谓的“诽谤性言论”，则将保留刑事处罚。
  </p>
  <p class="graf graf--p">
   人权观察指出，印度尼西亚政府应废除包括在2016年“互联网法”中的刑事诽谤条款，将其替换为民事诽谤条款，其中包含足以保护言论自由免受不公正干涉的保障措施。
  </p>
  <p class="graf graf--p">
   Harsono 说：“令人震惊的是，像 Syamlan 这样的人可以根据模糊的和严重的刑事诽谤法入狱，足够证实这些法律本身就有问题。”“相反，政府应该制定民事而非刑事处罚，并且只适用于信息而不是意见的情况，只适用于造成真正伤害或损害的明显虚假声明的情况。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    此案提醒注意
   </strong>
   ：虽然我们必须强调使用加密通讯应用是第一步（不推荐WhatsApp）但您必须确保与之交流的人是具有充分安全意识的 — — 不会截图保留的人；并且，如果您必须使用signal等需要手机号注册的加密通讯应用，请避免使用自己的真实手机号、避免使用在其他平台使用的名字、头像和简介。更多详见《
