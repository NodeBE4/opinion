   “紧急状态”意味着什么
  </h1>

 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      “紧急状态”意味着什么？意味着总统有权控制你家一切通电的东西。为什么会这样？
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   1941年12月那个星期四的珍珠港仍然是美国历史上的一个重要时刻，就在那一天，富兰克林罗斯福总统告诉他的新闻秘书 Stephen T. Early，政府应该接管全国广播网络。
  </p>
  <p class="graf graf--p">
   早些时候通知联邦通信委员会（FCC）主席兼新成立的国防通信委员会（DCB）主任 James L. Fly 被点名了，罗斯福曾亲自指示 Fly 满足政府控制全国广播网络的需要。DCB 是为这样的国家紧急情况而创建的：它的任务是协调战争或其他国家紧急状态下的所有通信（军事和民用全包括）。美国联邦通信委员会和 DCB
   <a class="markup--anchor markup--p-anchor" data-href="https://www.law.cornell.edu/uscode/text/47/606" href="https://www.law.cornell.edu/uscode/text/47/606" rel="noopener noreferrer" target="_blank">
    都获得了1934年“通信法”第606条的授权
   </a>
   ，该法明确规定总统在这种情况下有权完全控制电子传输。
  </p>
  <p class="graf graf--p">
   特别是第606（c）赋予了总统完全的控制权，以暂停和征用该国的整个电子监管系统。“在总统宣布存在战争或战争威胁、公共危险或灾难或其他“国家紧急状态”时，总统就可以在他认为合适的任何时间内暂停或修改控制在美国管辖范围的任何电台或设备的规则和条例”，该法律说，总统有能力关闭任何无线电台 — 也就是“任何能够发射10 kilocycles 至100,000 megacycles 之间的电磁辐射的设备” — 并且可以征用其设备，“仅仅向业主赔偿即可。“
  </p>
  <p class="graf graf--p">
   Fly 完全可以完成这个任务，因为他
   <a class="markup--anchor markup--p-anchor" data-href="https://www.tandfonline.com/doi/abs/10.1080/08821127.2007.10678081" href="https://www.tandfonline.com/doi/abs/10.1080/08821127.2007.10678081" rel="noopener noreferrer" target="_blank">
    一直致力于广播宣传计划
   </a>
   ；然而 Fly
   <a class="markup--anchor markup--p-anchor" data-href="https://reason.com/archives/2016/03/19/the-man-j-edgar-hoover-blamed" href="https://reason.com/archives/2016/03/19/the-man-j-edgar-hoover-blamed" rel="noopener noreferrer" target="_blank">
    是一个公民自由主义者
   </a>
   ，他强烈反对白宫的命令。为了推动他自己的计划，他最终说服了政府的同事 Jack Benny, Bob Hope 和其他人。
  </p>
  <p class="graf graf--p">
   但是1941年12月11日早期和 Fly 之间的关系在美国媒体监管的历史上仍然很重要。第606条今天仍然有效，因为1996年的“电信法”纳入了1934年“原始通信法”的实质部分。
  </p>
  <p class="graf graf--p">
   当特朗普又宣布了
   <a class="markup--anchor markup--p-anchor" data-href="http://reason.com/blog/2019/02/15/trumps-emergency-action-stinks-all-the-w" href="https://reason.com/blog/2019/02/15/trumps-emergency-action-stinks-all-the-w" rel="noopener noreferrer" target="_blank">
    另一次国家紧急状态时
   </a>
   ，重新考虑第606条是很重要的。大多数人都不知道这些紧急状态声明在允许政府对我们的生活全面控制方面拥有如此广泛的权力。在宣布国家紧急状态后，特朗普就可以指挥美国的每一个电视和广播电台，并要求它按照他的指示行事。
  </p>
  <p class="graf graf--p">
   那太糟了。但是：还有更糟的！
  </p>
  <p class="graf graf--p">
   法律允许白宫接管任何发射射频传输的设备。在2019年，这就是意味着从植入式心脏起搏器装置到吹头发的吹风机的一切。包括你的电动运动器材，任何智能设备（也包括数字洗衣机、电冰箱等）和你的笔记本电脑 — 基本上你家里的所有通电的东西都包括在内。
  </p>
  <p class="graf graf--p">
   我们倾向于将美国联邦通信委员会视为电视广播领域的警察：该部门负责发放许可证、对发布违规内容的站点进行罚款，以及以其他方式管理广播和电视公司。但是，实际上它的任务范围更广。
  </p>
  <p class="graf graf--p">
   FCC 管理着所有可以传输电力的设备。即使您家中的灯开关也受到 FCC 的监管，因为它们在运行过程中会传输微量电力。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://ieeexplore.ieee.org/stamp/stamp.jsp?arnumber=7879929" href="https://ieeexplore.ieee.org/stamp/stamp.jsp?arnumber=7879929" rel="noopener noreferrer" target="_blank">
    家用电器通常属于
   </a>
   两种轻度管制类别中的一种: “incidental radiators” 或者 “unintentional radiators.”。您的手机、广播电台的传输塔，甚至您的蓝牙设备，都属于更加规范的类别：“intentional radiators”。
  </p>
  <p class="graf graf--p">
   但就总统的紧急状态权力而言，这些子类别基本上毫无意义。白宫在国家紧急状态下对电力进行大规模控制的权力仍然存在，并且已经由国会通过的立法授予了总统。
  </p>
  <p class="graf graf--p">
   所以请记住：只要他愿意，特朗普或任何其他总统都可以控制您的收音机、电视、笔记本电脑、甚至是电灯开关和微波炉。再一次，显而易见：不了解历史是很可怕的。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    In
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://reason.com/archives/2019/02/15/in-a-state-of-emergency-the-president-ca" href="https://reason.com/archives/2019/02/15/in-a-state-of-emergency-the-president-ca" rel="noopener noreferrer" target="_blank">
    a State of Emergency, the President Can Control Your Phone
   </a>
   , Your TV, and Even Your Light Switches. Under a little-known regulation that dates back to the 1930s, the president has legal power over electronic transmissions.
  </p>
