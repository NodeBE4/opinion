  <p class="graf graf--p">
   政治和企业经济正在推动“行为科学”的实施，包括“助推”（简称舆论操纵），以现任政府、决策者、官僚和公司的自我利益（和有限理性）为基础，但在很大程度上被忽视了，尽管我们中的一些人多年来一直对此表示担忧。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这些操纵者急于将人群描绘成非理性的和认知上有缺陷的
   </strong>
   ，行为经济学家、政府、官僚和大公司游说、公关、“战略传播”和所谓的“咨询”行业的阴暗黑社会，似乎忽视了一些
   <strong class="markup--strong markup--p-strong">
    他们自己的高涨的“认知偏见”
   </strong>
   。无论是否有任何道德界限，这一切都源于他们
   <strong class="markup--strong markup--p-strong">
    对利润和权力的强烈追求
   </strong>
   。
  </p>
  <p class="graf graf--p">
   一旦保守党在2010年随便宣布他们的“行为改变”议程，并设立了’Nudge Unit’，剑桥分析公司那类的丑闻也就不可避免了。
   <strong class="markup--strong markup--p-strong">
    人们怎么能指望一个专制政府会自行抵制利用所谓的“行为科学”伎俩来操纵公民的观念、认知、行为、选择以及最终投票决策的诱惑呢？
   </strong>
   — — 专制的定义就是抵制变革，行为操纵在此的性质也完全一样，把每个人变成当权者想要的样子。
  </p>
  <p class="graf graf--p">
   剑桥分析公司的商业副总裁 Richard Robinson 曾经说过，
   <strong class="markup--strong markup--p-strong">
    让某人投票和说服他们购买哪一种品牌的牙膏之间没有根本性区别
   </strong>
   。他补充说：
   <strong class="markup--strong markup--p-strong">
    “当人们处于那种特定的思维模式下时就会自动以你想要他们理解的方式去理解特定事物”
   </strong>
   。Robinson 声称，要做到这点只需要使用大数据来分析公民，将“人”叠加在数据之上 — — 这是以前广告商使用的方法，被称为“人性化营销”。
  </p>
  <p class="graf graf--p">
   我不同意这个概念。我将
   <strong class="markup--strong markup--p-strong">
    对公民决策的监控、入侵和微观操控视为操纵商业行为的商品化和市场化的一种形式
   </strong>
   。未经我们的同意，也不会与我们分享所产生的利润。这是非常“非人性化的营销”。 “（关联《
   <a href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e5%bf%85%e9%9c%80%e6%8b%92%e7%bb%9d%e5%a4%a7%e6%95%b0%e6%8d%ae-%e7%9b%91%e8%a7%86%e4%b9%8b%e6%81%b6%ef%bc%88%e4%ba%8c%ef%bc%89%e5%85%ac%e7%a7%81%e7%9b%91/" rel="noopener noreferrer" target="_blank">
    为什么必需拒绝大数据 — — 监视之恶（二）公私监控伙伴关系
   </a>
   》）
  </p>
  <p class="graf graf--p">
   我们的个人数据被用于构建“说服概况”，使用一组基于概率的估计 — — 关于特定影响策略对个人的有效性，这也是基于特定的人在过去对这些策略的反应。其中一些公司也
   <strong class="markup--strong markup--p-strong">
    正在试验生物识别技术
   </strong>
   。营销中的许多企业公开承认他们的目标是*
   <strong class="markup--strong markup--p-strong">
    实现行为改变
   </strong>
   *。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    私人公司和政府使用公民作为巴甫洛夫狗是不对的。这种针对性的说服策略严重破坏了人类的独立性，这是人类尊严和民主的核心。
   </strong>
  </p>
  <p class="graf graf--p">
   基于数据挖掘的大规模监视和心理剖析，
   <strong class="markup--strong markup--p-strong">
    互联网已迅速成为一个对公民和人口进行分类、分析、排序和“管控”的圆形监狱环境
   </strong>
   。
  </p>
  <p class="graf graf--p">
   数字跟踪和企业监控的强大工具，包括用于操纵意见和行为的技术，已经从公共产品服务营销领域转向
   <strong class="markup--strong markup--p-strong">
    以政治和选民为目标
   </strong>
   ，是自然而然的。个人数据市场也一直是
   <strong class="markup--strong markup--p-strong">
    行为控制的市场
   </strong>
   。行为控制市场由那些出售操纵行为的机会以获得巨额利润的人和购买此类机会的人组成。
  </p>
  <p class="graf graf--p">
   年初，政府的采购服务已经扩大了公共部门对所谓的“行为洞察”专家的选择。此前，行为洞察团队（Nudge Unit）是单一的供应商，但新的 Crown Commercial Service Behavioral Insights 框架将数量
   <strong class="markup--strong markup--p-strong">
    扩大到了六个
   </strong>
   。该框架保留了 Nudge Unit，新的供应商是CFE（研究和咨询），Kantar Public，McKinsey 和 Ipsos Mori。还有第六家供应商不知道是谁，因为还没有正式签署新合同。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    公关行业与营销、咨询、战略信息、行为经济学、舆论操纵等重叠，多年来一直在使用行为分析、心理分析和针对性宣传
   </strong>
   。很多企业一直在尽可能保密，这对研究构成了限制。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “数据密集型”公司以含混模糊的方式进行沟通，但在销售服务时它们更加大胆，在这种情况下，它们通过
   <strong class="markup--strong markup--p-strong">
    公开声明揭示内部实践
   </strong>
   。
  </p>
  <p class="graf graf--p">
   下面举几个比较典型的例子 — — 请记住这很可能不是全部。心理操纵市场已经如0day市场一样庞大和阴暗。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://www.thehuntingdynasty.com/" href="http://www.thehuntingdynasty.com/" rel="noopener noreferrer" target="_blank">
    <em class="markup--em markup--p-em">
     The Hunting Dynasty
    </em>
   </a>
   是一个“行为洞察和传播机构”，在他们的网站上写着：
   <em class="markup--em markup--p-em">
    “我们通过你的观众无法想象的方式来改善他们的行为，并且使用
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     健全的科学、心理学和虚拟技术
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    ，无可比拟地提高你的叙事”，然后继续说“欢迎来到消除破坏性行为的第一步”。 — —
   </em>
   请注意，“改善”和“破坏性”等文字都是狡辩性宣传措辞。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “我们独特的深度应用和学术行为知识的结合，以及
   </em>
   <a class="markup--anchor markup--p-anchor" data-href="http://www.thehuntingdynasty.com/#who-we-are-2" href="http://www.thehuntingdynasty.com/#who-we-are-2" rel="noopener noreferrer" target="_blank">
    <em class="markup--em markup--p-em">
     在世界上最大的广告代理商工作的经验
    </em>
   </a>
   <em class="markup--em markup--p-em">
    ，使我们每次都能提供强大的、全栈的开箱即用的 A +项目和保留服务。
   </em>
   ”
  </p>
  <p class="graf graf--p">
   该公司于2015年获得黄金零售’Nudge 奖’，并于2016年和2017年入围该奖项。（Nudge 的中文意思是“将特定的人向某一个特定的方向引领/助推”，你可以理解为中国当局追求的“舆论引导”，其目的是
   <strong class="markup--strong markup--p-strong">
    仅仅合乎操纵者的利益
   </strong>
   。当然这也是一个非常狡猾的概念，准确说它应该叫洗脑 — —
   <strong class="markup--strong markup--p-strong">
    你已经被操纵了，却还以为自己的立场出于自己的本心
   </strong>
   。这种邪恶的伎俩居然还有一个奖项，看起来有些人至少还需要重看《发条橙》）
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://www.beworks.com/?gclid=Cj0KCQjwy9LVBRDOARIsAGqoVnsl4GHaYSoYzmh5Gq1fekixgiSAOwK2FbNnck2KcmRr6jkVDv-ScPEaAlRjEALw_wcB" href="http://www.beworks.com/?gclid=Cj0KCQjwy9LVBRDOARIsAGqoVnsl4GHaYSoYzmh5Gq1fekixgiSAOwK2FbNnck2KcmRr6jkVDv-ScPEaAlRjEALw_wcB" rel="noopener noreferrer" target="_blank">
    <em class="markup--em markup--p-em">
     BeWorks
    </em>
    ,
   </a>
   是采用同类手段的公司的另一个例子，它将自己描述为
   <em class="markup--em markup--p-em">
    “第一家致力于将行为科学应用于战略、营销、运营和政策挑战的管理咨询公司”，同时“利用强大的力量”行为经济学的见解，以解决您最棘手的挑战。
   </em>
   “
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    他们为政府、能源行业、金融服务业、保险业和零售业工作
   </strong>
   ，自称“帮助组织将行为经济学融入他们的文化”。
  </p>
  <p class="graf graf--p">
   该公司表示：
   <em class="markup--em markup--p-em">
    “该团队
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     将认知和社会心理学，神经科学和市场营销领域
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    的领先学者与管理咨询专家相结合。我们的多学科专业知识使我们能够为客户提供最新的科学见解和战略性商业视角“
   </em>
   。
  </p>
  <p class="graf graf--p">
   他们还写了一篇文章，题为：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.scientificamerican.com/article/how-science-can-help-get-out-the-vote/" href="https://www.scientificamerican.com/article/how-science-can-help-get-out-the-vote/" rel="noopener noreferrer" target="_blank">
    科学如何帮助投票
   </a>
   》。他们声称“我们的科学家和商业专家团队提供了一种强大的方法，可以
   <strong class="markup--strong markup--p-strong">
    分析并可衡量地影响人们的决策
   </strong>
   ”。
  </p>
  <p class="graf graf--p">
   他们继续说“测量大脑活动和其他生物指标的神经营销研究是衡量真实情绪反应的另一种方式，而不是依赖于人们的感受。
   <strong class="markup--strong markup--p-strong">
    EEG帽和生物识别带是最常用的工具，但其他技术 — — 从阅读面部表情到测量反应时间的微小差异等伎俩 — — 也被使用
   </strong>
   。“
  </p>

   Cambridge Analytica / Facebook 丑闻仅仅是第一个’案例研究’。这是一个更基本问题的症状。⚠️ 大规模监视、数据剖析和行为改变战略，都嵌入在企业部门里，现在正以一种挑战自由民主社会政治传统的方式使用，其中*公民*在传统上由自决原则来定义 — — 心理操纵的诡计正在消除公民的概念。
  </p>
  <p class="graf graf--p">
   由于公民自决权被根本性侵蚀了，政治诚信和民主主权的未来正在遭受严重破坏。通过企业和政治实践的不透明性来创造、重建和放大权力不平衡，通过 监视、数据收集、心理剖析和心理定制的信息，旨在操纵公民的观念、决策和行为，最终严重限制了公民的选择。
