  <p class="graf graf--p">
   大多数人仍然认为通过所谓的GFW实施的只是针对中国国内的审查监控。也许曾经是，但如今已经不同。
  </p>
  <p class="graf graf--p">
   GFW是一种监视和阻塞性的系统，可防止中国公民查看国外的网站。中国政府提出使用该防火墙的理由是基于所谓的 “互联网主权” 的概念。中国很早就宣布“互联网在中国领土内属于中国主权管辖”。
  </p>

  <p class="graf graf--p">
   作为 “一国两制” 协议的一部分，香港基本上不在防火墙之内：那里可以使用 Twitter、Google 和 Facebook 等外国服务，而本地ISP则
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hkispa.org.hk/139-urgent-statement-of-hkispa-on-selective-blocking-of-internet-services.html" href="https://www.hkispa.org.hk/139-urgent-statement-of-hkispa-on-selective-blocking-of-internet-services.html" rel="noopener noreferrer" target="_blank">
    明确表示
   </a>
   反对对其开放式互联网进行的直接国家审查。
  </p>
  <p class="graf graf--p">
   但是，随着香港抗议活动的持续进行，以及中国大陆在全球范围内广泛破坏这一运动的尝试，中国正在试图将其影响范围扩大到防火墙之外。
  </p>
  <p class="graf graf--p">
   为了消灭防火墙外部的抗议活动的声息，
   <strong class="markup--strong markup--p-strong">
    中国在全世界范围内全面展示着自己的力量，并揭示了可以用来在世界范围内消除异议或批评的工具
   </strong>
   。
  </p>
  <p class="graf graf--p">
   其中一些工具（例如对包括美国公司NBA和暴雪在内的私人实体的压力）引起了一堆头条新闻，并激怒了这些公司的客户和员工。而其他的则更具技术性和隐秘性，经常被观察家们忽略。
  </p>
  <h3 class="graf graf--p graf--startsWithDoubleQuote">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     “大炮”瞄准防火墙外的站点
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   大炮是由中国ISP大规模部署的一种技术，用于将恶意 JavaScript 代码注入客户的不安全（HTTP）请求中。恶意代码为通过这些ISP的数百万个中国大陆互联网连接提供了武器。
  </p>
  <p class="graf graf--p">
   当用户访问不安全的网站时，他们的浏览器还将下载并运行政府的恶意 javascript，这将导致他们向“GFW”之外的站点发送更多流量，从而使这些网站对其他用户的运行速度变慢，甚至完全过载。
  </p>
  <p class="graf graf--p">
   比如，通过向百度注入恶意的 JavaScript 代码以使从中国大陆以外访问百度网站及广告的流量转换为DDoS攻击发送至目标。
  </p>
  <p class="graf graf--p">
   大炮（Great Cannon）在2015年的首次亮相时就摧毁了 Github，中国用户在该处托管反审查软件和《纽约时报》等被禁止的新闻媒体的镜像。在广泛的国际反对之后，这次袭击被制止。
  </p>
  <p class="graf graf--p">
   然而
   <strong class="markup--strong markup--p-strong">
    去年9月，大炮再次被激活，这次瞄准的是香港示威者
   </strong>
   。它短暂地捣毁了 LIHKG，这是香港热门的社交媒体平台，它对抗议活动至关重要。
  </p>
  <h3 class="graf graf--p">

   <strong class="markup--strong markup--p-strong">
    将监视软件植入到中国境外的所有华人社区和政治团体
   </strong>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     审查国外的中国应用程序
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在中国国内，政府当局不需要使用零日攻击在个人设备上安装恶意软件。因为中国的所有消息传递和浏览器应用程序的制造商都必须在其客户端上包括满足于政府要求的过滤。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这意味着，当您使用由中国大陆公司创建的任何应用时，该应用都可能包含旨在扫描和阻止所谓的 “违禁网站或关键字” 的代码。
   </strong>
  </p>

   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     这不是中国的特例

   狭隘的 “互联网主权” 论已被证明不足以满足中国的需求：国内政策目标现在要求在境内外均可控制互联网。
  </p>
  <p class="graf graf--p">
   需要明确的是，
   <strong class="markup--strong markup--p-strong">
    中国政府并不是控制互联网方面的特例：包括美国和欧盟在内的其他国家，都在忙于为各自的国界外行使自己的监视和审查能力提供各式各样的理由
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这一局面的最终结果会是什么？
  </p>
  <p class="graf graf--p">
   中国现在通过无限制使用恶意软件和国家支持的DDoS攻击，将其互联网力量投射到国外，还有授权的客户端过滤和监视、限制跨境言论自由的经济制裁、并迫使私人实体充当全球思想警察。
  </p>
  <p class="graf graf--p">
   除非立法者、公司和个人用户像香港反抗者一样勇敢地坚持抵制威权主义的行为，否则，我们将不得不看到所有国家都屈服于高堡奇人。⚪️
  </p>
