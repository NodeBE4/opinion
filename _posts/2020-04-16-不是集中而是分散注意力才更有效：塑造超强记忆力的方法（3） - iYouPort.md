  <p class="graf graf--p">
   在上一篇中我们介绍了 “走神” —— 关于那块手表的小练习，有趣吗？
  </p>
  <p class="graf graf--p">
   从小到大老师和家长都在敲你的脑袋大喊着 “集中注意力”！他们只是想让你注意到他们说的话，但是
   <strong class="markup--strong markup--p-strong">
    完全集中注意力并无助于您在现实中应对突发局面
   </strong>
   。
  </p>
  <p class="graf graf--p">
   当您的注意力集中在一个项目和事件上时，很可能会错过更重要的事；想象您正在街上注意一位超短裙美女，您太专注了以至于没有看到一辆制动失灵的大卡车正呼啸而来……哎呀。
  </p>
  <p class="graf graf--p">
   这是个玩笑。但是，
   <strong class="markup--strong markup--p-strong">
    在反侦察过程中恰恰会经常出现美女和大卡车的情况
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如您所知，现实中的间谍行动和社交工程攻击往往分组运作，最基本的是两人一组，其中一人负责操纵目标人的注意力，另一个人负责进攻。前者就是美女，后者就是大卡车。他们更多会选择嘈杂的环境，以进一步扰乱你的观察。
  </p>
  <p class="graf graf--p">
   而反侦察必备的技能之一就是：
   <strong class="markup--strong markup--p-strong">
    无重点观察
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️无重点观察就是它字面上的意思 —— 你必须能看到整个全局画面，而不是集中在任何一个细节上；不必担心错过东西，您的安防意识会带来敏锐的洞察力，以从中识别风险。
  </p>
  <p class="graf graf--p">
   在前文关于
   <strong class="markup--strong markup--p-strong">
    操纵对手的注意力
   </strong>
   的分析中，我们将注意力比作聚光灯；而无重点观察就如漫射的日光，您必须有足够大的覆盖面才能让操纵者难以隐形。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   尝试在不设定任何目标的情况下观察某个事物。
  </p>
  <p class="graf graf--p">
   举个例子。你坐在咖啡厅等人。你可以留心观察周边的环境，但不要专注于任何具体的事物。如果你看到有趣的东西，就记住它，但是不要因此缩小你的视野。
  </p>
  <p class="graf graf--p">
   继续观察周边环境，不要评论，不要赞美 —— 若无其事地看，就像在你和周围所有正在发生的事之间，隔着一个厚厚的玻璃墙。请找到那个感觉。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     游击战
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   游击战是非正规作战，几乎和人类文明一样悠久，但直到美索不达米亚和古埃及才有实据。
  </p>
  <p class="graf graf--p">
   游击战是在敌强我弱的情况下采取的方法。敌方强大到我方不能用其它正规战法与之相对抗、或用正规战法我方损失太大，大到无以为继，这种情况下就只能用游击战与之对抗。
  </p>
  <p class="graf graf--p">
   正因为敌方强大，具有大型部队，所以游击战多是在弱方的国土上进行，也因弱方熟悉该区的地形。
  </p>
  <p class="graf graf--p">
   游击战虽然不足以击垮传统部队，却可以制造军事上的僵局，而这种僵局最终又会导致政治解决。
  </p>
  <p class="graf graf--p">
   几乎在所有的例子里，在以政治方式解决时，总是传统部队那一方吃亏。
  </p>
  <p class="graf graf--p">
   游击战因兵力少而火力弱，很难独立定点进行长时间作战，必须有当地民众提供物资、情报等资源，并积极创造条件，利用天时加地形并搭配陷阱，以机动力及隐蔽性主动出击，遭遇强大敌人时化整为零，消耗敌人战力、拖延敌人行动、误导敌人方向，在敌人心理上制造极大压力，才可能取胜。
  </p>
  <p class="graf graf--p">
   独立战争期间，“一分钟人”（Minutemen，又作瞬息民兵）在北美享有盛誉，它指的是反抗英军、追求自治的武装村民，他们出则为兵，入则为民，因为他们行动特别迅速，只要一听到警报，在一分钟内就能集合起来，立即投入战斗，在整个战争期间四处袭扰英军，让英国人吃了不少苦头。
  </p>
  <p class="graf graf--p">
   越南战争中，越共成员先用游击战的型态攻击法国在越南的殖民地，获得了明显的成果；在美军大规模介入之后，由于不熟识越南环境和越共军队的游击战战略，造成了伤亡。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     丢失的秘密文件
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   1955年 3月 2日，莫斯科第 32警察局接到了一份关于 Simon Yakovlevich Bernstein 失踪的报告。
  </p>
  <p class="graf graf--p">
   Bernstein 生于1897年，是苏联科学院（ Academy of Sciences of USSR）档案馆员工。
  </p>
  <p class="graf graf--p">
   自 2月 21日以来，Bernstein 都未去过档案馆，但由于其此前因病频繁请假，且其公寓内未安装电话，因此失踪一周后才被人发现。
  </p>
  <p class="graf graf--p">
   2月28日和3月1日，同事试图去他家探望，但两次都无人应答。
  </p>
  <p class="graf graf--p">
   3月2日，向上级报告其失踪后，在辖区警官 A. P. Vasiliev 中尉的监督下，Bernstein 的房间（高尔基大街 22号第 15公寓）被打开。房间内空无一人。

     想象力
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   从进化角度来说，与理解和记忆言语的能力相比，人类感知和记忆图像的能力更加古老，并且更成熟。这仅仅是因为对于人类祖先来说，一只随时准备进攻的野兽比一个骂大街的邻居更危险。
  </p>
  <p class="graf graf--p">
   这意味着，
   <strong class="markup--strong markup--p-strong">
    与文字相比，所有图片都会更容易被感知和记忆，尤其是那些明亮鲜艳的动态图片
   </strong>
   。
  </p>
  <p class="graf graf--p">
   那么你可以如何利用这点来改善记忆呢？很简单，尝试把自己需要记住的东西形象化就行了。对于大脑来说，想象出来的东西与实际存在的东西没什么区别。
  </p>
  <p class="graf graf--p">
   图像比文字更容易记忆，如果你想象的图像鲜明、生动而细致，那就更加如此了。
  </p>
  <p class="graf graf--p">
   举个例子。新认识的朋友向你解释如何去她的公寓。她口述了一个地址，包括街道名、大楼、公寓号。你记住了吗？
  </p>
  <p class="graf graf--p">
   然后那位朋友告诉你，她住在一栋有一家宠物商店的大楼里，这就好多了！想象一下，你的朋友站在宠物商店柜台后面，向三只猫出售三条金鱼。公寓号是33。
  </p>
  <p class="graf graf--p">
   就是这么简单。越是鲜明和生动，就越容易记忆。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">

     练习
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   观察你桌子上的东西，注意所有细节！—— 所有。它们都放在什么位置？以什么角度放置？整体是什么布局？每件物品都有哪些特征：颜色、质地、磨痕、划痕？褶皱？破损？……
  </p>
  <p class="graf graf--p">
   现在闭上眼睛。首先想象这张桌子，然后开始逐个想象桌上的所有物品。想象它们的每一个细节。把它们以正确的角度放在正确的位置上。
  </p>
  <p class="graf graf--p">
   如果你做不到，那就睁开眼睛再看一下，观察一下哪些细节是你难以想象的，然后再次闭上眼睛，继续尝试。
  </p>
  <p class="graf graf--p">
   这个练习不仅适用于桌子上的物品，也适用于整个房间、窗外的风景、电影中的街道场景、旅途中坐在你对面的人……
  </p>

  <p class="graf graf--p">
   Solomon Shereshevsky，二十世纪初的一位俄罗斯记者，是个具有超强记忆力的人，他的故事曾经被收入了几乎所有心理学教科书里。
  </p>

  </p>
  <p class="graf graf--p">
   活动中禁止参加者做任何笔记。但在活动后他令人惊讶地一字不差地回忆起讲话的所有内容。他还以为这是人人都有的能力。
  </p>
  <p class="graf graf--p">
   心理学家卢里亚花了30年的时间研究 Solomon，将他标记为 “S”。
  </p>
  <p class="graf graf--p">
   在卢里亚的书中，S可以记忆任意数量的信息；他拥有看似无限的记忆力，可以回忆起任何事：图片、概念、文字、或字母组合的无意义的东西，他的背诵就如阅读一样容易。
  </p>
  <p class="graf graf--p">
   而且人们还发现，只要被他记住的东西，就永远不会被忘记。他甚至可以毫不费力地回忆起10～15年前的实验中别人对他发出的指令。
  </p>
  <p class="graf graf--p">
   卢里亚的研究试图描述S的记忆运作方式，以及记忆和重现信息的方式。他发现了以下几点：
  </p>
  <p class="graf graf--p">
   1、为了方便地储存，S将信息编码成图像。比如数字1是一个骄傲的体型匀称的男人。数字6是一个肿脚的男人，数字8是一个肥胖粗壮的女人……
  </p>
  <p class="graf graf--p">
   2、S有很强的通感 —— 即 各个感官的感觉交织在一起。他可以清楚地区分字母的颜色、感受声音的沙哑程度、品味不同的形状。在S的案例中，除了嗅觉之外，其他所有感官都是想通的。他使用五种感觉器官中的四种来创建生动而鲜明的记忆画面。
  </p>
  <p class="graf graf--p">
   3、为了记住长列表中的数字或项目顺序，S会在脑海中“走过”家乡一条熟悉的街道，并在沿途设置图像。有时他也会从列表中“丢失”一些东西，就像图像被放入了黑暗的地方、或者与背景融为一体了。他会想象那些图像在冒险，并由此组织成一个迷人的故事。

   当你在一个无法解决的问题上卡壳时，停下来，在脑子里挂起那块书写板，在上面写下问题，再擦除。它还会弹出来吗？擦掉它！又弹出来了？继续擦掉！几次之后大脑就会转换路线，开辟一个新的思路出来。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff0000;">
    测试！快速回答：
   </span>
  </h3>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    伯恩斯坦的案件为什么被反间谍部门直接接手？
   </span>
  </p>
  <p class="graf graf--p">
   A：伯恩斯坦在公务员系统中身居高位；
  </p>
  <p class="graf graf--p">
   B：伯恩斯坦保管着包含机密文件的失踪文件；
  </p>
  <p class="graf graf--p">
   C：伯恩斯坦是一名克格勃特工；
  </p>
  <p class="graf graf--p">
   D：警方无法找到伯恩斯坦；
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     游戏 —— 填字（4✖️4）
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   这个练习旨在训练想象力和视觉记忆力。
  </p>
  <p class="graf graf--p">
   只要你身边有填字游戏的废报纸就可以开始练习。
  </p>
  <p>
   <img alt="" class="size-full wp-image-10871 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/3-1.jpg?resize=800%2C410&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/3-1.jpg?w=800&amp;ssl=1 800w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/3-1.jpg?resize=300%2C154&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/3-1.jpg?resize=768%2C394&amp;ssl=1 768w" data-recalc-dims="1" height="410" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/3-1.jpg?resize=800%2C410&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>
   <noscript>
    <img alt="" class="size-full wp-image-10871" data-recalc-dims="1" height="410" sizes="(max-width: 800px) 100vw, 800px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/3-1.jpg?resize=800%2C410&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/3-1.jpg?w=800&amp;ssl=1 800w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/3-1.jpg?resize=300%2C154&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/3-1.jpg?resize=768%2C394&amp;ssl=1 768w" width="800"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在填字游戏的四个角各画一个由4行4列方格组成的正方形。盯着其中一个角，只关注黑白方格的布局。
  </p>
  <p class="graf graf--p">
   4～5秒后，把报纸翻过去，在另一张纸上画出黑白方格组成的图案。
  </p>
  <p class="graf graf--p">
   如果你的大脑能够“拍摄”这个矩阵照片，然后用它来填写单元格，这个练习就会变得非常容易。如果不能“拍摄”全景，可以将黑色方格组合成几何图形来记忆。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    空间推理技巧非常重要，在思考中你遇到的假设、理论和方案经常具有复杂的结构，因此，以图表的形式绘制并熟练地使用它们的能力，有助于将复杂任务相关的所有信息存放在正确的框架中
   </strong>
   。
  </p>
  <p class="graf graf--p">
   您答对了今天的测试题吗？
  </p>
  <p class="graf graf--p">
   别忘了那盒火柴和骰子。
  </p>
  <p class="graf graf--p">
   你一定有经常去的地方对吧，比如咖啡馆、书店、音乐厅。坐在家里回想这些地方的布局，墙壁、天花板、地板、装饰 …… 家具是如何摆放的？桌上和架子上有什么物品？想象自己坐在最常坐的位置上看上述一切，然后继续
   <strong class="markup--strong markup--p-strong">
    更换自己的位置，从一个新鲜的角度创新观察上述的一切
   </strong>
   。
  </p>
  <p class="graf graf--p">
   所有动作都在想象力的空间中完成。试试看，看自己能“观察”到什么？⚪️
  </p>
  <p class="graf graf--p">
