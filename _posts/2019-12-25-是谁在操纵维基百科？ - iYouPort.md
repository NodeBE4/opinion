      </strong>

   </noscript>
  </p>
  <p class="graf graf--p">
   问 Google 或 Siri：“台湾是什么？”
  </p>
  <p class="graf graf--p">
   它们会回答“一个国家，东亚国家”。
  </p>
  <p class="graf graf--p">
   但是在今年9月初，它们的回复是： “台湾是中华人民共和国的一个省”。
  </p>
  <p class="graf graf--p">
   实际上，
   <strong class="markup--strong markup--p-strong">
    许多搜索引擎、数字助理和手机语音助理的信息来源都指向一个地方：维基百科。而维基百科突然发生了变化
   </strong>
   。
  </p>
  <p class="graf graf--p">
   编辑被撤销，但很快又恢复；然后再次被撤销。就该在线百科而言，这已经变成了一场社论式的拔河比赛，导致台湾是什么这个问题的答案在一天之内不断变化。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “今年是非常疯狂的一年，” 台湾维基媒体协会董事会成员 Jamie Lin 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “许多台湾维基百科页面遭到攻击。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    编辑之战
   </strong>
  </p>
  <p class="graf graf--p">
   任何人都可以在 Wikipedia 上编写或编辑条目，并且在地球上几乎每个国家中，都存在“ Wikipedians”社区来保护和贡献内容。
  </p>
  <p class="graf graf--p">
   这是有史以来积累的最大的人类知识收藏，可以免费在线提供给所有人，可以说是数字时代的最大成就。但是，在 Lin 和她的同事们的眼中，维基百科正受到攻击。
  </p>
  <figure aria-describedby="caption-attachment-9647" class="wp-caption aligncenter" id="attachment_9647" style="width: 624px">
   <img alt="" class="size-full wp-image-9647 jetpack-lazy-image" data-lazy-sizes="(max-width: 624px) 100vw, 624px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1.jpg?resize=624%2C351&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1.jpg?w=624&amp;ssl=1 624w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1.jpg?resize=300%2C169&amp;ssl=1 300w" data-recalc-dims="1" height="351" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1.jpg?resize=624%2C351&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="624"/>
   <noscript>
    <img alt="" class="size-full wp-image-9647" data-recalc-dims="1" height="351" sizes="(max-width: 624px) 100vw, 624px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1.jpg?resize=624%2C351&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1.jpg?w=624&amp;ssl=1 624w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1.jpg?resize=300%2C169&amp;ssl=1 300w" width="624"/>

    Jamie Lin – seen on the left – is one of many Taiwanese Wikipedians concerned about changes being made to the online encylopedia
   </figcaption>
  </figure>
  <p class="graf graf--p">
   有关台湾的编辑之战只是维基百科广泛的多语种条目中爆发的所有战争之一。
  </p>
  <p class="graf graf--p">
   香港抗议的页面一天之内就发生了65次变化 —— 主要是定义问题。他们是抗议者吗？还是暴徒？
  </p>
  <p class="graf graf--p">
   尖阁诸岛的英文条目说它们是“东亚岛屿”，但今年早些时候，其普通话页面已被改为 “中国的固有领土”。
  </p>
  <p class="graf graf--p">
   1989年的天安门广场抗议活动的普通话维基百科页面也被更改，将其描述为“平息反革命暴动”的“ 64事件”。
  </p>
  <p class="graf graf--p">
   在英语版本中，达赖喇嘛是藏族难民；而普通话页面则标注为中国流亡者。
  </p>
  <p class="graf graf--p">
   愤怒的意见分歧一直在 Wikipedia 上发生。但是对 Lin 女士来说，情况稍有不同。
  </p>
  <p class="graf graf--p">
   她说：“这是由（中国）政府控制的。太可怕了。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “社会主义核心价值观”
   </strong>
  </p>
  <p class="graf graf--p">
   BBC Click 的调查发现了近1600次倾向性的编辑，跨越22篇带有政治敏感性的文章。
  </p>
  <p class="graf graf--p">
   调查无法验证是谁进行了这些编辑，为何进行编辑，或它们是否反映了更广泛的实践；然而，有些迹象表明，这些编辑不是随机的，它们有目的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    来自中国境内的官员和学者都已公开呼吁其政府和公民
   </strong>
   系统地“纠正”他们所认为的整个维基百科中普遍存在的严重的“反华偏见”。
  </p>
  <p class="graf graf--p">
   维基百科上发表了一篇名为《中国外交的机遇与挑战》的论文，并于今年发表在《社会科学杂志》上。
  </p>
  <p class="graf graf--p">
   其中，学者 Li-hao Gan 和 Bin-Ting Weng 认为：“由于外国媒体的影响，维基百科条目对中国政府有很多偏见”。
  </p>
  <p class="graf graf--p">
   他们继续说：
   <strong class="markup--strong markup--p-strong">
    “我们必须制定有针对性的对外交流策略，这不仅包括重建一组外部交流语话系统，还包括，在Wiki平台上培养有影响力的编辑。”
   </strong>
  </p>
  <p class="graf graf--p">
   这篇文章以行动号召结束。
  </p>
  <p class="graf graf--p">
   作者说，“中国迫切需要鼓励和培训中国网民，使其成为 Wikipedia 平台的舆论领袖和管理者……他们可以坚持社会主义核心价值观，并组成一些核心编辑团队。”
  </p>

   </li>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    观念转变
   </strong>
  </p>
  <p class="graf graf--p">
   另一篇文章是由中国共产党控制的中国国际出版集团的官员 Jie Ding 撰写的。文章辩称：“在维基百科关于中国的主要政治言论的内容缺乏系统的整理和维护”。
  </p>
  <p class="graf graf--p">
   这篇文章也敦促 “在编辑作品中反映我们[中国]的声音和观点，以客观、真实地反映中国道路和中国思想对其他国家和历史的影响” 的重要性。
  </p>
  <p class="graf graf--p">
   香港中文大学助理教授 Lokman Tsui 表示：“讲好中国故事” 这个[习近平提出的]概念在过去的两年中获得了极大的关注。“他们[中国领导人]认为，海外对中国的许多看法是一种误解。”
  </p>
  <p class="graf graf--p">
   Tsui 认为，
   <strong class="markup--strong markup--p-strong">
    随着中国动员其国内的在线控制系统延伸到境外，以应对海外存在的所谓误解，现在的局势正在发生重大转变
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这本身不是新鲜事。自成立以来，维基百科就一直面临破坏性行为的问题。所有人都可以编辑、可以破坏他人的编辑、可以锁定页面，该网站由机器人和人类编辑共同巡逻。
  </p>
  <p class="graf graf--p">
   一些人从一开始就尝试操纵 Wikipedia，而其他人则尽力阻止了这些操纵。
  </p>
  <figure aria-describedby="caption-attachment-9648" class="wp-caption aligncenter" id="attachment_9648" style="width: 624px">

  <p class="graf graf--p">
   但是，Lin 所描述的许多活动并非完全是故意破坏。有些行为（例如台湾的主权问题）是要推动一项有争议的主张高于其他主张。还有一些，关于对语言的修剪，尤其是普通话页面的修剪，从政治角度出发，仍然有些微妙。
  </p>
  <p class="graf graf--p">
   香港抗议活动应该被视为“反对”中国吗？您应该将社区称为“汉族台湾人”还是“台湾原住民的汉族分支”？
  </p>
  <p class="graf graf--p">
   许多激烈的战斗就是在这种语言领域的微妙层面进行的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    协调策略？
   </strong>
  </p>
  <p class="graf graf--p">
   攻击通常不是针对 Wikipedia 的内容，而是针对其 Wikipedians 社区。
  </p>
  <p class="graf graf--p">
   Lin 说：“有些人告诉我们，他们的个人信息被曝光，只因为他们持有不同的想法。”（也就是 doxing 攻击，在这里看到
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%8a%b5%e5%88%b6%e5%9c%a8%e7%ba%bf%e6%94%bb%e5%87%bb%ef%bc%9a%e4%b8%8d%e8%a6%81%e8%a2%ab-doxing-%e5%8e%8b%e5%9e%ae-2/" href="https://www.iyouport.org/%e6%8a%b5%e5%88%b6%e5%9c%a8%e7%ba%bf%e6%94%bb%e5%87%bb%ef%bc%9a%e4%b8%8d%e8%a6%81%e8%a2%ab-doxing-%e5%8e%8b%e5%9e%ae-2/" rel="noopener noreferrer" target="_blank">
    防御方法
   </a>
   ）
  </p>
  <p class="graf graf--p">
   甚至还有针对台湾维基百科的死亡威胁。在相关的公共 Wikimedia telegram 频道上有人写着：“警察会很喜欢你母亲的法医报告”。
  </p>
  <p class="graf graf--p">
   在选举拥有更大权力的维基百科管理员职位的过程中，地缘政治界限也同样清晰可见。
  </p>
  <p class="graf graf--p">
   将在线活动按照民族国家划分通常是不可能的，而且这些编辑与中国政府之间也没有直接的经过验证的联系。
  </p>
  <p class="graf graf--p">
   Tsui 说：“绝对可以想象，所谓的爱国华人正在编辑这些 Wikipedia 条目。但就此下结论会忽略政府必须操纵这些平台的更大结构的协调性战略。”
  </p>

    <img alt="" class="size-full wp-image-9649" data-recalc-dims="1" height="549" sizes="(max-width: 976px) 100vw, 976px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-3.png?resize=976%2C549&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-3.png?w=976&amp;ssl=1 976w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-3.png?resize=300%2C169&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-3.png?resize=768%2C432&amp;ssl=1 768w" width="976"/>
   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-9649">
    It has been argued that minor changes to Wikipedia entries can change readers’ perceptions
   </figcaption>
  </figure>
  <p class="graf graf--p">
   尽管未归因，但这些编辑行为确实是在包括中国在内的许多国家
   <strong class="markup--strong markup--p-strong">
    加大了系统性操纵在线平台的尝试
   </strong>
   的背景下进行的。
  </p>
  <p class="graf graf--p">
   这些民族国家已经在 Twitter 和 Facebook 上这么做很久了，世界各地的研究人员都警告说，政府支持的在线宣传战范围不断扩大。
  </p>
  <figure aria-describedby="caption-attachment-9651" class="wp-caption aligncenter" id="attachment_9651" style="width: 912px">
   <img alt="" class="size-full wp-image-9651 jetpack-lazy-image" data-lazy-sizes="(max-width: 912px) 100vw, 912px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-4.png?resize=912%2C1110&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-4.png?w=912&amp;ssl=1 912w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-4.png?resize=246%2C300&amp;ssl=1 246w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-4.png?resize=768%2C935&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-4.png?resize=841%2C1024&amp;ssl=1 841w" data-recalc-dims="1" height="1110" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-4.png?resize=912%2C1110&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="912"/>
   <noscript>
    <img alt="" class="size-full wp-image-9651" data-recalc-dims="1" height="1110" sizes="(max-width: 912px) 100vw, 912px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-4.png?resize=912%2C1110&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-4.png?w=912&amp;ssl=1 912w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-4.png?resize=246%2C300&amp;ssl=1 246w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-4.png?resize=768%2C935&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-4.png?resize=841%2C1024&amp;ssl=1 841w" width="912"/>
   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-9651">
    https://twitter.com/DonEvansWm/status/1180383760472141824
   </figcaption>
  </figure>
  <p class="graf graf--p">
   与几乎所有其他在线平台相比，维基百科是一个诱人的甚至显而易见的目标。
  </p>
  <p class="graf graf--p">
   新南威尔士大学数字文化高级讲师 Heather Ford 说：“我绝对不会感到惊讶。” 他的研究重点是维基百科的政治编辑。
  </p>
  <p class="graf graf--p">
   当然，每个国家都在乎其声誉。
  </p>
  <p class="graf graf--p">
   伦敦政治经济学院高级研究员 Shirley Ze Yu 说：“中国是世界第二大经济体，中国正在做的事是任何其他处于这种地位的国家都会寻求的东西。”
  </p>
  <p class="graf graf--p">
   台湾本身就陷入了与中国的信息战中，台湾有着自己的地缘政治观点，而且许多观点冲突可能是真实的，至少在编辑这些误解的人的眼中是如此。
  </p>
  <p class="graf graf--p">
   那么，这是否等于所谓的讲述中国故事或在线宣传？
  </p>
  <p class="graf graf--p">
   至少在 Wikipedia 的话题上，上述问题的答案将取决于人们对互联网用途的两种截然不同的想法。
  </p>
  <p class="graf graf--p">
   其中一种是开放知识、开源、志愿者主导的社区的哲学；但是，
  </p>
  <p class="graf graf--p">
   现在它可能会面临另一种力量：
   <strong class="markup--strong markup--p-strong">
    全球各国当权者都在努力定义“什么是真相”的在线力量日益强大
   </strong>
   ，已经扩展到了像 Wikipedia 这样的地方，这些地方非常重要，以至于它们无法被忽视。⚪️
  </p>
  <p class="graf graf--p">
   以上是BBC的文章，可以说比较客观；但是，IYP认为，
   <strong class="markup--strong markup--p-strong">
    从本质上，有计划有目的的协调性内容操纵与观点冲突是完全不同的，就如50C水军和天然愚蠢完全不同 —— 揭露操纵行为非常重要
   </strong>
   ，拿到执行者被雇佣的证据和行动模式，可以帮助包括维基百科在内的很多在线平台抵制操纵。⚪️
  </p>
