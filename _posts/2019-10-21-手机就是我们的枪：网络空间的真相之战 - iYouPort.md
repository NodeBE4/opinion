  <p class="graf graf--p">
   虚假消息可能是很严重的问题，事关生命威胁；至少，如我们分析过的《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%B8%BA%E4%BB%80%E4%B9%88%E8%99%9A%E5%81%87%E4%BF%A1%E6%81%AF%E4%BB%A4%E7%8B%AC%E8%A3%81%E6%94%BF%E6%9D%83%E5%8F%98%E5%BE%97%E6%9B%B4%E5%BC%BA%E5%A4%A7%EF%BC%8C%E4%BB%A4%E6%B0%91%E4%B8%BB%E5%9B%BD-2/" href="https://www.iyouport.org/%E4%B8%BA%E4%BB%80%E4%B9%88%E8%99%9A%E5%81%87%E4%BF%A1%E6%81%AF%E4%BB%A4%E7%8B%AC%E8%A3%81%E6%94%BF%E6%9D%83%E5%8F%98%E5%BE%97%E6%9B%B4%E5%BC%BA%E5%A4%A7%EF%BC%8C%E4%BB%A4%E6%B0%91%E4%B8%BB%E5%9B%BD-2/" rel="noopener noreferrer" target="_blank">
    为什么虚假信息令独裁政权变得更强大，令民主国家变得更脆弱？
   </a>
   》。但依靠寡头公司的审查封锁肯定不是解决方案。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5866 jetpack-lazy-image" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/头.gif?resize=1100%2C619&amp;is-pending-load=1#038;ssl=1" data-recalc-dims="1" height="619" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/头.gif?resize=1100%2C619&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    <em>
     十年前的缅甸，一张SIM卡售价高达2000美元；如今， Facebook 拥有1800万本地用户，是若开邦冲突的平行空间……
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p">
   2月20日，通过 WhatsApp 流传着一些照片。
  </p>
  <p class="graf graf--p">
   第一张照片是一位年轻女子，她的脸颊上小心翼翼地涂着传统的檀娜卡；用手托着下巴，甜甜地对着镜头微笑着：这是那种亲人可以保留为手机背景的照片。
  </p>
  <p class="graf graf--p">
   随后的照片显示她死在地板上，她的左脸颊被子弹击碎，脸上溅满了鲜血。
  </p>
  <p class="graf graf--p">
   根据发布者的说法，她的名字叫 Ma Ye Ye Soe，19岁，属于若开族佛教族群，住在若开邦北部 Rathedaung 乡偏远西部的 Myin Hpu 村。
  </p>
  <p class="graf graf--p">
   她被一颗流弹击中，据说是缅甸军队士兵在镇压阿拉干军队的过程中杀死了这个年轻人。
  </p>
  <p class="graf graf--p">
   另一名四岁女孩也受伤了。这些细节在一天后于当地媒体上得到了证实，所有这些都是真的，除了 Ma Ye Ye Soe 是18岁而不是19岁。
  </p>
  <p class="graf graf--p">
   此后，缅甸军队否认了对杀戮的责任，指责武装分子。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    若开邦在2016年和2017年是种族清洗的战场，对罗兴亚人的几次种族灭绝屠杀造成至少6700人死亡，导致70多万人逃往邻国孟加拉
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这引起了全世界的谴责，并呼吁国际刑事法院查找负责人，该国的形象以及领导人诺贝尔奖获得者昂山素季的形象瞬间黯然失色。
  </p>
  <p class="graf graf--p">
   若开族反叛者与缅甸军队或“缅甸国防军”之间的冲突酝酿已久。迄今为止，暴力事件已造成1万多人流离失所，并对中央政府管理该国冲突地区的能力造成严重破坏。
  </p>
  <p class="graf graf--p">
   Ye Ye Soe 的尸体照片在被触发后的几小时内开始在社交媒体上流传；若开族社区的愤怒情绪立即显而易见。
  </p>
  <p class="graf graf--p">
   在过去，若开邦缺乏连通性基础设施，这意味着不可能迅速记录侵犯人权和杀害无辜平民的行为。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    可以利用这些图像来获得对一场持续冲突的支持，这对缅甸军队和国家信息机构来说都不是新闻
   </strong>
   ：信息公关心理战部门一直试图通过向社交媒体大量发送印度教公民死者的照片来转移对若开邦罗兴亚人暴行的批评，他们声称这些公民被罗兴亚叛乱分子屠杀。
  </p>
  <p class="graf graf--p">
   对于今天在缅甸发生的每一场冲突，在网络空间、活动家、民族武装团体、政党、文职政府、和军队本身，都发生了相似的冲突。
  </p>
  <p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    连通性的新时代
   </strong>
  </p>
  <p class="graf graf--p">
   2007年，僧侣们第一次走上缅甸最大规模抗议活动的街头，这是一场即将被粉碎的民众起义，被称为“藏红花革命”；
   <strong class="markup--strong markup--p-strong">
    在残酷的军事独裁政权强加的严格审查制度的控制下，这个国家得到了全世界罕见的一瞥。
   </strong>
  </p>
  <p class="graf graf--p">
   随着抗议者人数增加到数万人 —— 这是自1988年民主运动以来最重大的起义 ——
   <strong class="markup--strong markup--p-strong">
    该运动被猛烈扼杀，成千上万的持不同政见者被判入狱
   </strong>
   。
  </p>
  <p class="graf graf--p">
   2007年标志性的镜头之一：安全部队成员近距离枪击日本摄影记者 Kenji Nagai的照片，是一位
   <strong class="markup--strong markup--p-strong">
    公民记者
   </strong>
   拍摄到的。
  </p>

    Only hours after the death of Kenji Nagai, the outside world could witness how the photographer was pushed to the ground, shot and killed.
   </figcaption>

  <p class="graf graf--p">
   这一照片被异议网络迅速上传并分发。
  </p>
  <p class="graf graf--p">
   当时，媒体出版前审查的措施已经确实存在，对出版许可和广播权的限制几乎完全是国家垄断。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    独立媒体被国家标记为叛徒
   </strong>
   ，外国媒体更是如此，其中一则特别值得火药味的宣传称英国广播公司为“广播杀手，漫天谎言”。
  </p>
  <p class="graf graf--p">
   当时，互联网接入是一种昂贵的、普遍过滤的和严重监视的商品，只有精英和那些勇敢的、技术娴熟且耐心的人，才可以使用在城市中心开始出现的网吧提供的缓慢且不可靠的连接。政治动荡时期的互联网中断非常常见。
  </p>
  <p class="graf graf--p">
   在20世纪90年代末，小规模的数字行动已经开始形成。尽管受到限制，流亡的持不同政见者、公民记者和国内活动家仍努力在线。
  </p>
  <p class="graf graf--p">
   对于这样一个国家来说，这是一个重大转变。
   <strong class="markup--strong markup--p-strong">
    军事政权要求在未事先获得部门批准的情况下禁止拥有计算机、调制解调器、传真机或计算机网络 —— 对违规行为的处罚为7至15年的监禁。
   </strong>
  </p>
  <p class="graf graf--p">
   十年前在缅甸，一张SIM卡价值高达2000美元，只有一个运营商没得选。
  </p>
  <p class="graf graf--p">
   而现在只需1.5美元，消费者可以从四个运营商中的任何一个获得高速4G，覆盖全国大部分地区。
  </p>
  <p class="graf graf--p">
   在这个连通性的新时代，曾经审查互联网和媒体、攻击持不同政见者、并对流亡者媒体进行DDOS攻击的军队，已经转向了对虚假信息的攻击。
  </p>
  <p class="graf graf--p">
   独立分析师 David Scott Mathieson 说：“破坏性的审查很快被社交媒体的喧嚣所取代，
   <strong class="markup--strong markup--p-strong">
    其中政治宣传可以更好的隐藏，实现更有效地阴险
   </strong>
   。”
  </p>

   虽然其他社交媒体平台如 Viber、WeChat 和 WhatsApp 在该国都可以使用，但
   <strong class="markup--strong markup--p-strong">
    只有 Facebook 享有至高无上的地位
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Facebook 在缅甸政治格局中占有独特的地位，拥有1800万用户。反叛组织和国家当局都使用它来发布公报。
  </p>
  <p class="graf graf--p">
   在资源匮乏的环境中，许多地方政府办公室仍在使用打字机，而 Facebook 允许政府和军方直接或间接地与公民进行沟通 。
  </p>
  <p class="graf graf--p">
   电信数据显示，Facebook 在其高峰时期占据了每日国内网络流量的80％，大多数浏览外部网站都是通过该平台进行的。
  </p>
  <p class="graf graf--p">
   Facebook 的巨大影响力吸引了宣传运动。
  </p>
  <p class="graf graf--p">
   在2018年末和2019年初，Facebook 宣布它已经封锁了广泛的可直接追溯到缅甸军方的“协调不真实行为”（CIB）活动（“虚假宣传”的含蓄说法）。
  </p>
  <p class="graf graf--p">
   第一次大清洗在缅甸封锁了18个帐户和52个主页，其中包括高级军事人员。其中有46个页面和12个从事秘密影响力活动的账户。
  </p>
  <p class="graf graf--p">
   这些页面看起来是独立的，但偶尔会推动军事路线，破坏昂山素季政府，并促进反罗兴亚情绪。
  </p>
  <p class="graf graf--p">
   总而言之，
   <strong class="markup--strong markup--p-strong">
    被禁的账户都有很大的影响力；他们拥有近1,200万关注者，约占缅甸所有 Facebook 用户的三分之二
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在被封锁之前，高级将军 Min Aung Hlaing 的两个主页共有400多万粉丝。
  </p>
  <p class="graf graf--p">
   Facebook 的一位资深人士非常熟悉 CIB 调查，他表示虽然这种“黑色宣传”并不新鲜，但它在社交媒体上的存在对那些必须处理它的人来说是一个独特的挑战。
  </p>
  <p class="graf graf--p">
   不只是军方被指责在线升级宣传；
   <strong class="markup--strong markup--p-strong">
    社会媒体上的仇恨言论也被联合国指出，“为冲突增添了动力”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   联合国实况调查团注意到，在2016年和2017年的罗兴亚危机期间，
   <strong class="markup--strong markup--p-strong">
    “民政当局散布了错误的叙述；否认了缅甸国防军的屠杀行为；封锁独立调查，包括联合国实况调查团都被封锁了；并且监督证据的销毁” ……结论认为，民政当局”推动了暴力罪行“。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “昂山素季的国家参赞信息委员会[Facebook]页面上充满了关于恐怖分子的歇斯底里，并暗示[援助机构]正在协助[罗兴亚叛乱分子]”，英国缅甸运动组织的 Mark Farmaner 说。
  </p>
  <p class="graf graf--p">
   Facebook 主页上发布的材料
   <a class="markup--anchor markup--p-anchor" data-href="https://www.facebook.com/InfomationCommittee/posts/660387290801047?__tn__=K-R" href="https://www.facebook.com/InfomationCommittee/posts/660387290801047?__tn__=K-R" rel="noopener noreferrer" target="_blank">
    指责
   </a>
   罗兴亚妇女声称“假强奸”，这些消息仍留在 Facebook 网站上。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5870 jetpack-lazy-image" data-lazy-sizes="(max-width: 1086px) 100vw, 1086px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-9.png?resize=1086%2C720&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-9.png?w=1086&amp;ssl=1 1086w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-9.png?resize=300%2C199&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-9.png?resize=768%2C509&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-9.png?resize=1024%2C679&amp;ssl=1 1024w" data-recalc-dims="1" height="720" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-9.png?resize=1086%2C720&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1086"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5870" data-recalc-dims="1" height="720" sizes="(max-width: 1086px) 100vw, 1086px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-9.png?resize=1086%2C720&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-9.png?w=1086&amp;ssl=1 1086w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-9.png?resize=300%2C199&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-9.png?resize=768%2C509&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-9.png?resize=1024%2C679&amp;ssl=1 1024w" width="1086"/>
   </noscript>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “手机就是我们的抢”
   </strong>
  </p>
  <p class="graf graf--p">
   全球范围内质疑社交媒体对民主进程的影响的呼声越来越高。
  </p>
  <p class="graf graf--p">
   在印度，WhatsApp 与一系列私刑和死亡有关；
  </p>
  <p class="graf graf--p">
   在菲律宾，协调的 Facebook 和 Twitter 洗脑运动推动了罗德里戈·杜特尔特总统的议程，并直接攻击批评政府的媒体；
  </p>
  <p class="graf graf--p">
   尽管有关滥用社交媒体平台的明显警告，科技公司的行动仍然缓慢。
  </p>
  <p class="graf graf--p">
   在亚洲，若开邦的暴力事件可能是自该国社交媒体出现以来最令人震惊的事件。
  </p>
  <p class="graf graf--p">
   在缅甸，Facebook 一直面临着对其未能在猖獗的仇恨言论中采取行动的强烈批评，联合国授权的事实调查组宣布 Facebook 上的内容在激起对罗兴亚穆斯林少数民族的仇恨方面发挥了“决定性作用”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该公司表示愿意遵守对保存数据的任何法律要求，
   </strong>
   并且
   <strong class="markup--strong markup--p-strong">
    发布到该网站的材料很可能作为任何国际刑事司法程序的证据
   </strong>
   。
  </p>
  <p class="graf graf--p">
   自从2011年宣布军统过渡到准平民民主选举的议会以来，政治上发挥的作用，与人类历史上最快的数字化推广同时发生。
  </p>
  <p class="graf graf--p">
   随着人们开始倒数2015年的选举，这是几十年来第一次可信的民主选举，手机信号塔建立起来了。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    数字技术的推出也引发了新的专制镇压
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “当人们发布的内容被认为损害当局的利益或诋毁当局时，网民因撰写对政府的批评而被判入狱。任何公民都有可能被起诉”，表达自由活动家 Maung Saungkha 说。
  </p>
  <p class="graf graf--p">
   若开邦州的暴行指控也使该国分裂，被审查的国内媒体报道了这一问题正面临的巨大压力 —— 2017年12月两名路透社记者被监禁的案例充分显示了这一点。
  </p>
  <p class="graf graf--p">
   记者 Wa Lone 和 Kyaw Soe Oo 的实地报道曝光了一个叫做 Inn Dinn 的村庄里发生的十名罗兴亚男子被大屠杀的真相。
  </p>
  <p class="graf graf--p">
   然而，说真话让两位记者被判处七年监禁。
  </p>
  <p class="graf graf--p">
   这两位获得今年普利策新闻奖的记者在上个月被释放，至此已经被关押了500天。
  </p>
  <p class="graf graf--p">
   然而，5月份，
   <strong class="markup--strong markup--p-strong">
    7名因 Inn Dinn 村杀人罪被判处10年徒刑的士兵在被监禁初期
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bbc.com/news/world-asia-48419595" href="https://www.bbc.com/news/world-asia-48419595" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     就被释放了
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   在2019年初，一个名为 Athan（音）的组织报告说，
   <strong class="markup--strong markup--p-strong">
    在2018年期间，有103起案件涉及言论自由，共有217人被起诉；其中9起案件涉及记者
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在缅甸，与其他地方一样，这种新的虚假信息、错误信息、和政治宣传动态对冲突局势产生的确切影响尚未得到人们的充分理解。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <strong class="markup--strong markup--p-strong">
    在社交媒体上使用虚假账户和虚假信息通常被用来操纵公众舆论，给人一种似乎存在普遍共识的幻象。这种虚假共识的制造可能会产生一种随波逐流的效应，即人们表现出支持一种观念或政治权力，只是因为他们害怕自己被边缘化
   </strong>
   ”，未来研究所数字情报实验室主任 Samuel Woolley 说。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%E5%B7%A8%E9%AD%94%E5%B4%9B%E8%B5%B7%EF%BC%9A%E6%94%BF%E5%BA%9C%E8%B5%84%E5%8A%A9%E7%9A%84%E7%BD%91%E7%BB%9C%E6%B0%B4%E5%86%9B%E6%AD%A3%E9%81%8D%E5%B8%83%E5%85%A8%E7%90%83/" href="https://www.iyouport.org/%E5%B7%A8%E9%AD%94%E5%B4%9B%E8%B5%B7%EF%BC%9A%E6%94%BF%E5%BA%9C%E8%B5%84%E5%8A%A9%E7%9A%84%E7%BD%91%E7%BB%9C%E6%B0%B4%E5%86%9B%E6%AD%A3%E9%81%8D%E5%B8%83%E5%85%A8%E7%90%83/" rel="noopener noreferrer" target="_blank">
     巨魔崛起：政府资助的网络水军正遍布全球
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “通常，这种依赖技术的宣传 —— 利用自动化和社交媒体试图引导政治沟通 —— 是为了大规模地强制放大一种观点，同时不成比例地抑制另一种观点”，他说。
  </p>
  <p class="graf graf--p">
   在一个新闻业几乎被淘汰的国家，社交媒体提供了对罗兴亚和若开族社区的实地情况的一点洞见。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    对于那些被困在冲突地区的平民来说，技术使他们有机会发出呼救声。
   </strong>
  </p>
  <p class="graf graf--p">
   正如一位反对派人士所说，“手机就是我们的枪。”
  </p>
  <p class="graf graf--p">
   在若开邦，暴力仍在继续。大赦国际的
   <a class="markup--anchor markup--p-anchor" data-href="https://news.sky.com/story/myanmar-military-accused-of-new-war-crimes-in-rakhine-state-11730978" href="https://news.sky.com/story/myanmar-military-accused-of-new-war-crimes-in-rakhine-state-11730978" rel="noopener noreferrer" target="_blank">
    一份新报告
   </a>
   指责缅甸军方在新行动中不分青红皂白地袭击平民，导致平民伤亡。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “自从世界对罗兴亚人遭受的大规模暴行感到愤怒不到两年后，缅甸军队
   <strong class="markup--strong markup--p-strong">
    再次对若开邦的族群进行了可怕的虐待
   </strong>
   ，”大​​赦国际东亚和东南区亚地区主任 Nicholas Bequelin 说。⚪️
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://codastory.com/authoritarian-tech/myanmar-facebook-conflict-rakhine/" href="https://codastory.com/authoritarian-tech/myanmar-facebook-conflict-rakhine/" rel="noopener noreferrer" target="_blank">
    The war for truth in Myanmar’s cyberspace
   </a>
   . A decade ago in Myanmar, a SIM card could cost as much as $2,000. Facebook now has 18 million local users and is a parallel space for the conflict in Rakhine State
  </p>
