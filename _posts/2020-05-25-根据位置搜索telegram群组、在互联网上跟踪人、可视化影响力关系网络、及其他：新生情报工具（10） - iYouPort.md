  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     1、如何根据位置搜索telegram群组（aware-online）
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   自从去年6月telegram更新以来，就可以根据其位置搜索用户和群组。这意味着您可以在搜索结果中从与自己相同位置的用户和群组中获得建议。
  </p>
  <p class="graf graf--p">
   因此，
   <strong class="markup--strong markup--p-strong">
    作为调查者，您可以简单地检查哪个用户和群组在哪个区域中处于活动状态
   </strong>
   。
  </p>
  <p class="graf graf--p">
   👉那么，您是否要不停地旅行才能将用户和所有群组映射到具体位置？当然不必，那也太费劲了。您可以通过简单的位置欺骗来解决此问题。
  </p>
  <p class="graf graf--p">
   也就是说，相当于手动将位置设置为其他位置。还记得我们介绍过的浏览器方法吗？在下面看到：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     《
    </strong>
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e4%bd%bf%e7%94%a8%e4%bd%8d%e7%bd%ae%e6%ac%ba%e9%aa%97%e6%89%a9%e5%b1%95%e8%bf%bd%e8%b8%aa%e7%ba%a6%e4%bc%9a%e7%bd%91%e7%ab%99%e4%b8%8a%e7%9a%84%e7%9b%ae%e6%a0%87%e7%a7%81%e5%af%86/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e4%bd%bf%e7%94%a8%e4%bd%8d%e7%bd%ae%e6%ac%ba%e9%aa%97%e6%89%a9%e5%b1%95%e8%bf%bd%e8%b8%aa%e7%ba%a6%e4%bc%9a%e7%bd%91%e7%ab%99%e4%b8%8a%e7%9a%84%e7%9b%ae%e6%a0%87%e7%a7%81%e5%af%86/" rel="noopener noreferrer" target="_blank">
     <strong class="markup--strong markup--li-strong">
      如何使用位置欺骗扩展追踪约会网站上的目标私密信息？
     </strong>
    </a>
    <strong class="markup--strong markup--li-strong">
     》
    </strong>
   </li>
  </ul>
  <p class="graf graf--p">
   那么，您需要的是什么？
  </p>
  <p class="graf graf--p">
   实际上只需要一部（虚拟）电话、Telegram 的移动应用程序和 Telegram 上的帐户。就行了。
  </p>
  <p class="graf graf--p">
   就如下面的示例中这样，使用“Android模拟器”，这是一种虚拟电话，其功能与普通电话大致相同。当然，你还可以为此使用 NoxPlayer、Bluestacks 或 Genymotion。选择哪种取决于自己的喜好。
  </p>
  <p class="graf graf--p">
   例如，您可以在虚拟机
   <a class="markup--anchor markup--p-anchor" data-href="https://www.virtualbox.org/" href="https://www.virtualbox.org/" rel="noopener noreferrer" target="_blank">
    Virtual Box
   </a>
   中创建 Genymotion，也或许您发现 NoxPlayer 可以更好或更快速地工作。当然您也可以使用预付费电话。
  </p>
  <p class="graf graf--p">
   好了电话已有，现在就下载 Telegram 吧。
  </p>
  <p class="graf graf--p">
   您可以通过
   <strong class="markup--strong markup--p-strong">
    Android 模拟器 + 伪造的 Gmail 帐户，通过 Google Playstore 进行此操作
   </strong>
   ，这取决于您所使用的设备。
