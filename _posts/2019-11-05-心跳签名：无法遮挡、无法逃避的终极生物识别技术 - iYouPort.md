      <strong>
       目前市面上几乎所有生物识别技术都可以通过伪造被欺骗。也就是说你还有出路。但是，现在出现了一种终极监视方式，无法遮掩、无法伪造、无需接触远程即可实现……
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-6509 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/心跳签名.png?resize=1100%2C640&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/心跳签名.png?w=1582&amp;ssl=1 1582w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/心跳签名.png?resize=300%2C174&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/心跳签名.png?resize=768%2C447&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/心跳签名.png?resize=1024%2C595&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/心跳签名.png?resize=740%2C430&amp;ssl=1 740w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/心跳签名.png?resize=1100%2C640&amp;ssl=1 1100w" data-recalc-dims="1" height="640" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/心跳签名.png?resize=1100%2C640&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6509" data-recalc-dims="1" height="640" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/心跳签名.png?resize=1100%2C640&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/心跳签名.png?w=1582&amp;ssl=1 1582w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/心跳签名.png?resize=300%2C174&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/心跳签名.png?resize=768%2C447&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/心跳签名.png?resize=1024%2C595&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/心跳签名.png?resize=740%2C430&amp;ssl=1 740w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/心跳签名.png?resize=1100%2C640&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   五角大楼推出了一款能够探测到人的独特心跳特征的远程激光设备。
  </p>
  <p class="graf graf--p">
   与以往一样，五角大楼宣称该技术“旨在挫败恐怖主义”，但是很显然，也会导致全民监视升级的新台阶：
   <strong class="markup--strong markup--p-strong">
    这是一种从远处即可进行生物识别的监视方法。
   </strong>
  </p>
  <p class="graf graf--p">
   科幻小说早就告诉我们，未来的战争将与激光战斗，我们只是假设它们是武器。现在看起来还可以作为监视设备存在。
  </p>
  <p class="graf graf--p">
   五角大楼的心跳探测激光器被称为 Jetson，它使用一种称为振动测量的技术来检测由于血液在整个循环系统中的运动而引起的人体微妙振动。
  </p>
  <p class="graf graf--p">
   这里的意义在于你的心跳是独一无二的 —— 不是心跳速度，而是峰谷，每个人都是不同的。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wired.com/insights/2014/06/heartbeat-may-soon-password/" href="https://www.wired.com/insights/2014/06/heartbeat-may-soon-password/" rel="noopener noreferrer" target="_blank">
    在这里看到具体解释
   </a>
   。
  </p>
  <p class="graf graf--p">
   该监视装置目前的有效射程为200米，理论上就像狙击武器一样。
  </p>
  <p class="graf graf--p">
   人类操作员使用激光器瞄准，在目前的技术程度上，需要30秒时间实现捕获。一旦目标被捕获心跳特征，激光系统就可以通过重复该过程来确认目标的身份。
  </p>
  <p class="graf graf--p">
   这意味着，例如，进行侦察的士兵可以在执行任务之前获得可疑目标的心跳特征。
  </p>
  <p class="graf graf--p">
   这样的情报将允许派遣任何攻击部队随后与目标群体接触，只瞄准那些预先标记好的人。
  </p>
  <p class="graf graf--p">
   同时也意味着，如果在目标群体中事先安插了己方的线人，那么狙击手可以通过识别他们的心跳来识别“是不是自己人”，避免向自己人开火。
  </p>
  <p class="graf graf--p">
   五角大楼认为这样的技术可以减少误伤。
  </p>
  <p class="graf graf--p">
   但是，也许对这项技术最令人疑惑的影响是其在军事领域之外的潜在用途。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    没有钥匙，没有密码，没有芯片 —— 实现监视
   </strong>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.researchgate.net/publication/45503358_Human_Microchip_Implantation_Human_Microchip_Implantation_Human_Microchip_Implantation" href="https://www.researchgate.net/publication/45503358_Human_Microchip_Implantation_Human_Microchip_Implantation_Human_Microchip_Implantation" rel="noopener noreferrer" target="_blank">
    大约70％的人不希望
   </a>
   自己的身体被植入芯片以存储数据和授予访问权限。
  </p>
  <p class="graf graf--p">
   这意味着要放弃密码、密钥、指纹登录和面部识别系统，还有很长的路要走。
  </p>
  <p class="graf graf--p">
   不幸的是人们容易忘记密码、钥匙丢失或被盗，一块透明胶就有可能欺骗指纹识别。虹膜扫描仪也并非不可欺骗，虽然需要一些技术能力和经费。
  </p>
  <p class="graf graf--p">
   在这里看到：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bd%93%e6%af%8f%e7%a7%8d%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab%e9%83%bd%e5%8f%af%e4%bb%a5%e4%bc%aa%e9%80%a0%e6%97%b6/" href="https://www.iyouport.org/%e5%bd%93%e6%af%8f%e7%a7%8d%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab%e9%83%bd%e5%8f%af%e4%bb%a5%e4%bc%aa%e9%80%a0%e6%97%b6/" rel="noopener noreferrer" target="_blank">
    当每种生物识别都可以伪造时
   </a>
   》
  </p>
  <p class="graf graf--p">
   但是，你的身体随着你的脉搏振动的方式这一特征是无法被模仿或被盗的。
  </p>
  <p class="graf graf--p">
   同样，
   <strong class="markup--strong markup--p-strong">
    你也不可能通过戴着手套或面具来掩饰心跳。
   </strong>
  </p>
  <p class="graf graf--p">
   从本质上讲，
   <strong class="markup--strong markup--p-strong">
    心跳签名是一种终极生物识别身份验证工具。
   </strong>
  </p>
  <p class="graf graf--p">
   你只是走到一扇门前，等待激光扫描你，然后进入。
  </p>
  <p class="graf graf--p">
   这种技术和其他生物识别一样可以被安装在你的手机上、便利店里等等。
  </p>
  <p class="graf graf--p">
   与华盛顿
   <a class="markup--anchor markup--p-anchor" data-href="https://thenextweb.com/artificial-intelligence/2019/07/02/inconvenience-store-tacoma-shops-facial-recognition-ai-is-a-racist-nightmare/" href="https://thenextweb.com/artificial-intelligence/2019/07/02/inconvenience-store-tacoma-shops-facial-recognition-ai-is-a-racist-nightmare/" rel="noopener noreferrer" target="_blank">
    塔科马的商店
   </a>
   中使用的面部识别系统被批评种族歧视，但心跳识别肯定不具有种族歧视性 —— 简单说就是，
   <strong class="markup--strong markup--p-strong">
    隐私保护者更难批评和阻止这种监视手段。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    苹果手表的逻辑扩展
   </strong>
  </p>
  <p class="graf graf--p">
   其实心跳签名的监视手段并不是新思路。苹果手表已经实现了心跳监控。
  </p>
  <p class="graf graf--p">
   由于 Apple Watch 等可穿戴技术，如今能够在发生心脏急症时被及时发现。
  </p>
  <p class="graf graf--p">
   但是显然，曾经您必须佩戴这样的设备或连接到医院的机器才能实现这点。
  </p>
  <p class="graf graf--p">
   想象一下，如果医院要实施这样一个系统，扫描每个进入急诊室的人，以确定他们是否患有心脏急症，那么将可以大大改善治疗效果。
  </p>
  <p class="graf graf--p">
   想象一下，养老院、康复中心和急救人员配备了这样的设备，似乎也会很方便。在未来，诊断分类可能是一个自动化的过程。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    执法部门的使用
   </strong>
  </p>
  <p class="graf graf--p">
   执法方面的使用范围可能是巨大的。
   <strong class="markup--strong markup--p-strong">
    他们要想找到他们的目标就需要大范围扫描人群。
   </strong>
  </p>
  <p class="graf graf--p">
   也许有些人会认为“精准的生物识别可以瞄准犯罪而不会误伤好人”？听起来的确有点像，但是，那是在完美社会中才能实现的。
  </p>
  <p class="graf graf--p">
   不幸的是，我们并不是生活在一个完美的社会中。
  </p>
  <p class="graf graf--p">
   无论是通过有偏见的技术还是故意滥用，无处不在的、不安全和不道德的监控技术已经受到了越来越多的抨击。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    五角大楼的 Jetson 激光器可以为警察提供一种非常高效的工具
   </strong>
   ，将警察当局怀疑的一切人收集下来，以便在任何时候盯防 —— 这些人会成为重点监视目标。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    延伸：被成为重点监视目标的结果是什么
   </strong>
   《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%88%91%e8%a2%ab%e8%ad%a6%e5%af%9f%e6%8a%98%e7%a3%a8%e8%87%b3%e7%b2%be%e7%a5%9e%e5%b4%a9%e6%ba%83%ef%bc%9a%e7%8e%b0%e5%9c%a8%e6%88%91%e5%bf%85%e9%a1%bb%e7%88%ac%e8%b5%b7%e6%9d%a5%e6%88%98%e6%96%97/" href="https://www.iyouport.org/%e6%88%91%e8%a2%ab%e8%ad%a6%e5%af%9f%e6%8a%98%e7%a3%a8%e8%87%b3%e7%b2%be%e7%a5%9e%e5%b4%a9%e6%ba%83%ef%bc%9a%e7%8e%b0%e5%9c%a8%e6%88%91%e5%bf%85%e9%a1%bb%e7%88%ac%e8%b5%b7%e6%9d%a5%e6%88%98%e6%96%97/" rel="noopener noreferrer" target="_blank">
    我被警察折磨至精神崩溃：现在我必须爬起来战斗
   </a>
   》。
  </p>
  <p class="graf graf--p">
   心跳签名与指纹或 DNA 不同，进行鉴定所需的数据无法由实验室捏造。
  </p>
  <p class="graf graf--p">
   当然，目前为止它依旧是早期技术。 Jetson 是一个原型。
  </p>
  <p class="graf graf--p">
   如上所述，扫描一个人大约需要30秒，并且目标必须在整个时间内保持静止。
  </p>
  <p class="graf graf--p">
   但是，无疑，它会迅速发展。蓝图已经在这里了。⚪️
  </p>
