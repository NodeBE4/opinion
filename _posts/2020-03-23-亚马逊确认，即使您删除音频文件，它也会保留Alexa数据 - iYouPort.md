  <p class="graf graf--p">
   亚马逊已经承认，即使用户选择擦除其帐户中的音频文件，亚马逊也不会删除用户与该公司的 Alexa 和 Echo 设备进行语音交互而产生的存储数据。
  </p>
  <p class="graf graf--p">
   亚马逊在给参议员 Chris Coons（D-DE）的一封信中明确概述了这点，进一步揭示了该公司关于其所谓的数字语音助手的隐私惯例。
  </p>
  <p class="graf graf--p">
   该回复是对 Coons 要求的后续行动，当时 Coons 质疑该公司究竟保留了 Echo 互动的语音记录和抄本多长时间。
  </p>
  <p class="graf graf--p">
   这封信中亚马逊证实了一些指控。信中写道：“我们保留客户的录音和文字副本，直到客户选择删除它们。”
  </p>
  <p class="graf graf--p">
   但在CNET调查之后，还有一个问题是：即使用户选择删除了该音频，亚马逊是否仍保留与 Alexa 进行语音交互的文字记录。
  </p>
  <p class="graf graf--p">
   亚马逊表示，确实没有删除其中一些文字记录或从文字记录中收集的信息，这是因为该公司 “不得不” 从其全球数据存储系统的各个部分中清理数据，于是在某些情况下，
   <strong class="markup--strong markup--p-strong">
    亚马逊选择保留该数据而不告诉用户
   </strong>
   。
  </p>
  <p class="graf graf--p">
   亚马逊公共政策副总裁布莱恩·侯赛曼（Brian Huseman）在回应中表示，该公司正在 “持续努力，以确保这些文字记录不会留在 Alexa 的其他存储系统中。”
  </p>
  <p class="graf graf--p">
   换句话说，即使用户手动删除了音频版本，某些文本版本仍会在未知长度的时间段内继续保存在单独的存储系统中。
  </p>
  <p class="graf graf--p">
   甚至，在某些情况下，亚马逊认为删除数据会阻碍 Alexa 的功能，因此该公司决定保留某些版本的数据。
  </p>
  <p class="graf graf--p">
   亚马逊声称它不会保留音频文件，但是例如，如果有人使用 Alexa 打电话给Uber或下订单，它将保留交易信息。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们不存储Alexa回应的声音。但是，我们可能仍会保留客户与Alexa互动的记录，包括Alexa为响应客户要求而采取的行动的记录”，Huseman 写道。
  </p>
  <p class="graf graf--p">
   这封信还指出，该公司甚至连 Alexa 技能的开发人员都可以记录用户使用 Echo 设备进行的每笔交易或日常活动。
  </p>
  <p class="graf graf--p">
   亚马逊表示，“这是为确保该任务易于重复且对用户方便”。—— 又是 “方便”。
  </p>
  <ul class="postList">
   <li class="graf graf--li graf--startsWithDoubleQuote">
    “方便” 究竟是一种什么样的深渊？《

     便利的暴政
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   彭博社的一份报告中概述了 Alexa 的内部工作方式，这已经引起了人们的极大关注 —— 其中数千人是合同工，有些甚至不是亚马逊直接雇用的人，他们都可以使用这些语音和文本数据，
   <strong class="markup--strong markup--p-strong">
    这些数据可以用于整理有关用户个人生活的全部图景
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    在这里看到《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e9%9b%86%e4%bd%93%e7%9b%91%e5%90%ac/" href="https://www.iyouport.org/%e9%9b%86%e4%bd%93%e7%9b%91%e5%90%ac/" rel="noopener noreferrer" target="_blank">
     集体监听
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   但是，由于缺乏关于亚马逊如何收集和存储这些数据的方式、以及到何种目的使用数据的清晰性、还有为什么难以从公司的服务器上彻底清除这些数据的困惑，这使人们重新审视了亚马逊所谓的 “行业标准做法”，这些做法对于构建AI-依赖的工具和服务。
  </p>
  <p class="graf graf--p">
   风险越来越高，因为 Alexa

   </a>
   处理敏感度最高的患者健康信息。

   亚马逊还受到儿童隐私权保护组织

    的抨击
   </a>
   ，该公司被批评通过使用其 Amazon Echo Dot Kids 设备收集和存储13岁以下儿童的数据，违反了《儿童在线隐私保护法》（COPPA）。
  </p>
  <p class="graf graf--p">
   Coon 在一份声明中说：“亚马逊的回应使我们了解到，即使用户删除了他或她的声音录音，也不会从所有亚马逊服务器上删除与Alexa的交互记录。公民应该了解科技公司如何使用他们的个人数据，我将继续与消费者和公司合作，以确定如何最好地保护公民的个人信息。”
  </p>
  <p class="graf graf--p">
   您知道中国小米的语音助手如何使用您的数据吗？您能记得自己都和该窃听器说过什么吗？您所在的国家是否有完善隐私保护法？公民权利组织是否有成熟的反抗行动和维权措施？如果答案是否定的，那么为什么要花钱邀请这种东西进入您最私密的空间？更多相见栏目 “
