  <p class="graf graf--p graf-after--figure" id="33eb">
   1月下旬，当 Strava 应用程序发布全球热图时，一切都崩溃了。你可以在上面看到这张地图，显示其所有用户的已记录活动。这是一张华丽的地图，可以看到每一个跑步的和骑自行车的人带有光闪闪的靓丽颜色。这东西是完全公开的：任何人都可以在网上查看它。
  </p>
  <p class="graf graf--p graf-after--p" id="74b4">
   大部分活动都在西方国家进行。在世界其他地方，Strava 的用户很少，地图基本上是空的。但是在这里和那里，有时候在沙漠地区或其他荒凉的地方也能找到一阵丰富的色彩。
  </p>
  <p class="graf graf--p graf-after--p" id="18eb">
   一些聪明的调查人员很快就会发现它更有趣的用途：
   <strong class="markup--strong markup--p-strong">
    军事基地，其中一些甚至是一直试图保持隐秘的军事基地
   </strong>
   。使用 Strava 的西方军事人员不知不觉地引起了全球人对他们自己和他们同事的关注。
  </p>
  <p class="graf graf--p graf-after--p" id="8c6c">
   怎么做？
  </p>
  <p class="graf graf--p graf-after--p" id="0f6a">
   <strong class="markup--strong markup--p-strong">
    第一阶段：
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="32d7">
   在荷兰，Foeke Postma 对此感兴趣。他是开源情报网站 Bellingcat 的死忠和志愿者。 4月，Postma 开始使用芬兰公司 Polar 制作的健身应用程序; 他很好奇应用程序将收集哪些关于他的数据。
   <strong class="markup--strong markup--p-strong">
    像许多其他用户一样
   </strong>
   ，他在出门前打开应用程序，并且在他回到家之前不会关闭。
  </p>
  <p class="graf graf--p graf-after--p" id="37eb">
   Polar 的地图向他展示了他家附近越来越多的跑步者。 Postma 意识到这意味着什么：如果他公开自己的个人资料，任何人都可以看到他住在哪里。
  </p>
  <p class="graf graf--p graf-after--p" id="6cbf">
   他的侦探心理变得活灵活现了。他滚动浏览 Polar 的地图，
   <strong class="markup--strong markup--p-strong">
    找到了 Volkel，这是一个储存核武器的荷兰军事基地
   </strong>
   。 Bam：“这里有一个跑步者，使用了真实姓名记录”。 Postma 缩小并找到慢跑者跑步的其他路线。大多数的记录开始和结束于附近城镇的房子，也就是他们就住在附近。
  </p>
  <p class="graf graf--p graf-after--p" id="7c32">
   使用 LinkedIn搜索，Postma 得知那个慢跑者是
   <strong class="markup--strong markup--p-strong">
    荷兰军方的高级军官
   </strong>
   。他现在知道了这个人的家庭住址。并且成功地在其他敏感地点重复了这一伎俩。
  </p>
  <p class="graf graf--p graf-after--p" id="6cb6">
   Postma 意识到他偶然发现了重大的安全漏洞。他联系了荷兰媒体 De Correspondent 并告诉媒体他所使用的简单技巧。De Correspondent 决定帮助他，因此进一步追究此事。
  </p>
  <p class="graf graf--p graf-after--p" id="f5e9">
   以下是如何精确定位家庭地址的四个步骤：
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4718 jetpack-lazy-image" data-lazy-sizes="(max-width: 490px) 100vw, 490px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-26.jpeg?resize=490%2C2048&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-26.jpeg?w=490&amp;ssl=1 490w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-26.jpeg?resize=72%2C300&amp;ssl=1 72w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-26.jpeg?resize=245%2C1024&amp;ssl=1 245w" data-recalc-dims="1" height="2048" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-26.jpeg?resize=490%2C2048&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="490"/>

  <p class="graf graf--p graf-after--figure" id="a816">
   起初人们觉得整件事都有些讽刺。记者们展示了特定敏感站点附近的跑道，例如军营，并示范了了追踪跑步者家庭住址的容易程度。到此为止仍然只是一种好奇心。
  </p>
  <p class="graf graf--p graf-after--p" id="ac47">
   当他们放大伊拉克北部的埃尔比勒国际机场时，情况发生了变化。这个机场是
   <strong class="markup--strong markup--p-strong">
    几个北约国家的军事基地的所在地，包括荷兰 — 正在帮助库尔德军队与伊斯兰国恐怖组织作战
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="1747">
   他们找到了四名荷兰士兵，包括一名被称之为“Tom”的士兵（这是记者采取的化名，后面部分都是如此），鉴于软件显示的他的
   <strong class="markup--strong markup--p-strong">
    快速心率和悠闲的步伐
   </strong>
   不怎么匹配，说明 Tom 不是埃尔比勒基地最适合的士兵。记者们甚至还找到了
   <strong class="markup--strong markup--p-strong">
    这名士兵的 Facebook 页面
   </strong>
   。他和妻子在 FB 分享的照片 — 与全世界其他人一样 —
   <strong class="markup--strong markup--p-strong">
    揭示了太多私人信息
   </strong>
   ， Tom 的孩子们已经二十出头了。几分钟后，记者们在荷兰北部的一个小镇上追查到了这名士兵的
   <strong class="markup--strong markup--p-strong">
    家庭住址
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="945d">
   记者们也偶然发现了化名为“John“的另一个人。”John 的 Facebook 页面没有提及他在埃尔比勒的使命。他一直小心翼翼地保密这些信息。但记者们确实找到了
   <strong class="markup--strong markup--p-strong">
    他的孩子的照片
   </strong>
   ，还是一个婴儿，他亲切地抱着他的肩膀。通过 Polar，记者们发现他住的房子位于一个中等规模的荷兰城市的工人阶级社区。
  </p>
  <p class="graf graf--p graf-after--p" id="24ae">
   当记者们找到 Tom 和 John 的家庭住址时，他们意识到自己正在处理的正是
   <strong class="markup--strong markup--p-strong">
    可能危及人们生命的信息
   </strong>
   。如果 ISIS 战士或同情者去寻找这些士兵，打算复仇的话可怎么办？如果骇客收集这些信息并将其卖给出价最高的人会怎么样？因为所有人都很容易找到这些信息。
  </p>
  <p class="graf graf--p graf-after--p" id="4463">
   是时候认真对待这件事了。记者们把自己想象成是那种想要播下恐惧、监视别人、讹诈别人进行间谍活动的人。然后开始系统地搜索目标。
  </p>
  <p class="graf graf--p graf-after--p" id="e679">
   <strong class="markup--strong markup--p-strong">
    第二阶段
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="07f9">
   对于 Postma 已经找到的 Volkel 基地的高级官员“Frank”来说（也是化名），
   <strong class="markup--strong markup--p-strong">
    间谍和勒索是潜在的严重问题。因为核武器储存在这个基地
   </strong>
   。有人可以使用 Frank 的身份收集有关基本安全的信息：例如拦截 Frank 的通信，或者榨取机密信息。
  </p>
  <p class="graf graf--p graf-after--p" id="115e">
   记者们在几分钟内就找到了 Frank 的家庭住址。还知道他至少有两个女儿。知道他的妻子在哪里工作，以及他们的爱好都是什么。甚至可以访问每个家庭成员的照片。
   <strong class="markup--strong markup--p-strong">
    这些愚蠢的人平日完全没有注意到自己把什么放在了网上。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="79b2">
   Frank 并不是唯一一个在 Volkel 军事基地附近进行健身训练的人。记者们还找到了另外三个人。每个家庭住址都是非常详细的。
  </p>

   <strong class="markup--strong markup--p-strong">
    驻扎在马里加奥军事基地的几名军事人员
   </strong>
   。他们大多数是德国人。一名士兵的父母住在德国南部，靠近奥地利边境，在一个迷人的小房子里。记者们还发现了另外
   <strong class="markup--strong markup--p-strong">
    两名荷兰士兵
   </strong>
   ，根据他们的锻炼数据判断，他们已经回到家中了。在马里时，他们从未离开基地。

     </p>
     <p class="graf graf--p graf-after--p" id="b356">
      这里有一点技术性。
     </p>
     <p class="graf graf--p graf-after--p" id="224a">
      假设您要查找 2017 年1月1日至4月1日期间在美国 Smallville 登录的所有健身者。您可以在 Polar 地图搜索框中键入 Smallville，就像在 Google 地图中一样。你会在左边看到一个菜单，让你输入你感兴趣的时间范围。你没看到的是这张地图 — 实际上是一个网络应用程序 — 创建了封装搜索查询的 URL 或 Web 地址。
     </p>
     <p class="graf graf--p graf-after--p" id="4736">
      Polar 使用其唯一 URL 来搜索用户数据库。生成的匹配将发送到前面的地图页面，现在将显示您的搜索结果。地图最多显示20个结果，并无法搜索超过六个月的时间段。
     </p>
     <p class="graf graf--p graf-after--p" id="1c51">
      以下是地图对数据库的请求，简而言之：可以访问数据库的页面的 URL +地理区域的 GPS 坐标+“从起始”日期和“到截止”日期。
     </p>
     <p class="graf graf--p graf-after--p" id="ef6e">
      记者发现
      <strong class="markup--strong markup--p-strong">
       完全可以手动创建这些 URL。这意味着可以直接向数据库询问想要的任何数据
      </strong>
      。根本不需要地图。更好的是：
      <strong class="markup--strong markup--p-strong">
       可以绕过几个地图的过滤器，并要求一次查看400个结果
      </strong>
      ，而不是地图会向我们显示的 20 个结果。还可以请求自 2014 年以来的所有活动 — 比地图允许的六个月最大值的时间范围大得多了。
     </p>
     <p class="graf graf--p graf-after--p" id="b8be">

      </li>
      <li class="graf graf--li graf-after--li" id="d9fc">

      </li>
      <li class="graf graf--li graf-after--li" id="30e8">
       将搜索（2014 年至今）的全部时间划分为三个月的分期;
      </li>
      <li class="graf graf--li graf-after--li" id="a604">
       并将所有这些组合转换为可以在数据库中触发的 URLs。
      </li>

      </strong>

        记者们甚至还发现了一些额外新事物：即，
        <strong class="markup--strong markup--p-strong">
         Strava 不仅可以识别军事基地和其他敏感地点，还可以识别个人及其地址
        </strong>
        。很快确定了以下信息：
       </p>
       <ul class="postList">
        <li class="graf graf--li graf-after--p" id="4b83">
         美国军方的一名检察长，他在关塔那摩湾附近跑了一圈;
        </li>
        <li class="graf graf--li graf-after--li" id="c229">
         伊拉克北部埃尔比勒国际机场的几名美国士兵;
        </li>
        <li class="graf graf--li graf-after--li" id="7367">
         在马里加奥的一个军事基地的几名法国士兵;
        </li>
        <li class="graf graf--li graf-after--li" id="8ca7">
         还有一位美国分析师在马里兰州的米德堡（Fort Meade）工作，国家安全局就在那里。
        </li>
       </ul>
       <p class="graf graf--p graf-after--li" id="eb8c">
        简而言之：Strava 完全没有解决它的隐私问题。
       </p>
       <p class="graf graf--p graf-after--p" id="9916">
        下面测试的另一个应用程序是来自 Under Armour 运动服装品牌的
        <strong class="markup--strong markup--p-strong">
         Endomondo
        </strong>
        。使用这个应用程序，很容易找到以下内容：
       </p>
       <ul class="postList">
        <li class="graf graf--li graf-after--p" id="b4cf">
         关塔那摩湾的三名警长和一名系统管理员;
        </li>
        <li class="graf graf--li graf-after--li" id="64cc">
         米德堡参谋长联席会议顾问;
        </li>
        <li class="graf graf--li graf-after--li" id="bc14">
         埃尔比勒国际机场的两名士兵。
        </li>
       </ul>
       <p class="graf graf--p graf-after--li" id="346a">
        最后记者们又测试了
        <strong class="markup--strong markup--p-strong">
         Runkeeper
        </strong>
        。相比下使用这个应用程序比较难确定身份，并且更难找到地址，但记者们依旧在 Gao (Mali)，Volkel 军事基地（荷兰）和关塔那摩湾找到了几个用户。
       </p>
       <p class="graf graf--p graf-after--p" id="d54c">
        <strong class="markup--strong markup--p-strong">
         不仅仅是隐私问题
        </strong>
       </p>
       <p class="graf graf--p graf-after--p" id="6706">
        这不是一个普通的隐私问题，无法使用几个软件补丁和更好的常见问题解答来修复它。这是一个安全问题。
       </p>
       <p class="graf graf--p graf-after--p" id="4846">
        为了保持安全，需要让自己和家人远离公众视线的人们最好避免使用这些类型的应用程序，或者至少仔细考虑它们泄露的信息你能否承受得了。
        <strong class="markup--strong markup--p-strong">
         即便是普通人，如果你不想因为网上晒生活而遭到入室抢劫的话，也最好别用这些应用程序。
        </strong>
       </p>
       <p class="graf graf--p graf-after--p" id="c83f">
        雇主们必须确保现有的安全措施不会被这种数字污染所破坏。科技公司必须意识到他们收集、存储和与世界其他地方分享的数据的敏感程度 — 以及随之而来的责任。
       </p>
