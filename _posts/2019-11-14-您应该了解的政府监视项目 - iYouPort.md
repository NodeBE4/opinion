   同类的大规模监视系统在很多国家都存在。

   做了一个简要的总结。
  </p>
  <p class="graf graf--p">
   政府监视项目正如其名称所暗示的那样：目的是大规模地有组织地监视当地人口。
  </p>
  <p class="graf graf--p">
   世界上有许多这样的监视项目，通常与美国国家安全局的当地监视项目
   <strong class="markup--strong markup--p-strong">
    等效
   </strong>
   。
  </p>
  <p class="graf graf--p">
   公民经常只能通过
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e9%80%8f%e6%98%8e%e5%ba%a6%e9%9d%a9%e5%91%bd/" href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e9%80%8f%e6%98%8e%e5%ba%a6%e9%9d%a9%e5%91%bd/" rel="noopener noreferrer" target="_blank">
    透明度革命
   </a>
   解密后才会看到这些监视项目的真面目。
  </p>
  <p class="graf graf--p">
   否则，它们的名字不为人所知。
  </p>
  <p class="graf graf--p">
   在有人进行调查之前，被监视的公民无法了解监视项目的日常工作情况。
  </p>
  <p class="graf graf--p">
   这就是为什么
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e9%80%8f%e6%98%8e%e5%ba%a6%e9%9d%a9%e5%91%bd/" href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e9%80%8f%e6%98%8e%e5%ba%a6%e9%9d%a9%e5%91%bd/" rel="noopener noreferrer" target="_blank">

   </strong>
   。

    仅仅

  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   至少在所有参与者都死了之前，没人会知道。
  </p>
  <p class="graf graf--p">
   尽管如此，人们仍然可以通过熟悉世界闻名的政府监视项目来
   <strong class="markup--strong markup--p-strong">
    了解当权者的欲望有多深
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    PRISM (USA)
   </strong>
  </p>
  <p class="graf graf--p">
   这可能是最著名的最家喻户晓的政府监视项目之一了。
  </p>
  <p class="graf graf--p">
   在这里看到完整版汇总《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" rel="noopener noreferrer" target="_blank">
    互联网霸主的时间线 — — 他们将互联网视为间谍、破坏和战争的主场
   </a>
   》。
  </p>
  <p class="graf graf--p">
   PRISM 是一个NSA监视项目，主要用于从提供互联网服务的公司收集信息。
  </p>
  <p class="graf graf--p">
   是的，这就是美国政府让私营互联网公司监视他们的客户然后将这些信息传递给他们的方式之一。是
   <strong class="markup--strong markup--p-strong">
    协同作恶
   </strong>
   。
  </p>
  <p class="graf graf--p">
   PRISM 于2007年推出，是目前为止生命力最强的最恐怖的监视项目。
  </p>
  <p class="graf graf--p">
   自从它上线以来，PRISM 一直都是NSA互联网数据收集的主要来源。
  </p>
  <p class="graf graf--p">
   NSA 在线收集到的所有信息中有90％都是来自于 PRISM。
  </p>
