   已经找到了一种方法：它在免费产品中为您提供了比许多竞争对手在付费版本中提供的功能更多的功能。
  </p>
  <p class="graf graf--p">
   按需扫描以清理受感染的系统？当然。实时保护首先防止攻击？没问题。浏览器过滤阻止访问危险站点？自然。家长控制、以定义您的孩子可以在线访问的内容？是的。
  </p>
  <p class="graf graf--p">
   你还可以用一个帐户覆盖最多三台 Mac 或 Windows 设备，并从中央网络控制台远程检查和管理它们的状态 — 在保护整个家庭时可能非常有用。
  </p>
  <p class="graf graf--p">
   AV Test 的2017年12月Mac检测率
   <a class="markup--anchor markup--p-anchor" data-href="https://www.av-test.org/en/news/news-single-view/put-to-the-test-antivirus-solutions-for-macos-sierra" href="https://www.av-test.org/en/news/news-single-view/put-to-the-test-antivirus-solutions-for-macos-sierra" rel="noopener noreferrer" target="_blank">
    测试表明
   </a>
   ，Sophos 提供的保护与 Avast，Bitdefender，Kaspersky 和其他大牌产品相同。
  </p>
  <p class="graf graf--p">
   Sophos Home Premium 确实有一些有价值的额外功能：高级勒索软件保护、监控网络摄像头和麦克风的访问、实时聊天和电子邮件支持，以及保护多达10个设备的选项。它功能强大，每年的价格为37.50英镑（48美元）。
  </p>
  <p class="graf graf--p">
   总之，选择什么仍然取决于你。也许问题不应该是反病毒软件是否物有所值，而是，你的资产是否比安全应用程序的成本更有价值。
  </p>
