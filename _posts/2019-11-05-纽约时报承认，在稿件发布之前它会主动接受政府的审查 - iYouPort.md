  <p class="graf graf--p graf--startsWithDoubleQuote">
   “纽约时报”公开承认，它在发布之前会将其部分文稿发交给政府审查，以获得“国家安全官员”的批准。
  </p>
  <p class="graf graf--p">
   这证实了像 James Risen 这样的资深纽约时报记者所指出过的：
   <strong class="markup--strong markup--p-strong">
    报纸经常与政府合作，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.democracynow.org/2018/1/5/how_the_ny_times_us_government" href="https://www.democracynow.org/2018/1/5/how_the_ny_times_us_government" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     以压制高层官员不希望公开报道的任何内容
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   延伸《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E7%94%B5%E5%AD%90%E9%82%AE%E4%BB%B6%E8%A2%AB%E6%9B%9D%E5%85%89%EF%BC%9A%E6%8F%AD%E7%A4%BA%E8%AE%B0%E8%80%85%E4%B8%BB%E5%8A%A8%E6%8E%A5%E5%8F%97%E6%83%85%E6%8A%A5%E9%83%A8%E9%97%A8%E7%9A%84%E5%AE%A1/" href="https://www.iyouport.org/%E7%94%B5%E5%AD%90%E9%82%AE%E4%BB%B6%E8%A2%AB%E6%9B%9D%E5%85%89%EF%BC%9A%E6%8F%AD%E7%A4%BA%E8%AE%B0%E8%80%85%E4%B8%BB%E5%8A%A8%E6%8E%A5%E5%8F%97%E6%83%85%E6%8A%A5%E9%83%A8%E9%97%A8%E7%9A%84%E5%AE%A1/" rel="noopener noreferrer" target="_blank">
    电子邮件被曝光：揭示记者主动接受情报部门的审查，这究竟是为什么？
   </a>
   》
  </p>
  <p class="graf graf--p">
   6月15日，纽约
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2019/06/15/us/politics/trump-cyber-russia-grid.html" href="https://www.nytimes.com/2019/06/15/us/politics/trump-cyber-russia-grid.html" rel="noopener noreferrer" target="_blank">
    时报报道称
   </a>
   ，美国政府正在加剧对俄罗斯电网的网络攻击。
  </p>
  <p class="graf graf--p">
   根据这篇文章，“特朗普政府正在更积极地部署网络战工具”，这是华盛顿和莫斯科之间更大的“数字冷战”的一部分。
  </p>
  <p class="graf graf--p">
   在回应这份报道时，特朗普在推特上骂
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/realDonaldTrump/status/1140065300186128384" href="https://twitter.com/realDonaldTrump/status/1140065300186128384" rel="noopener noreferrer" target="_blank">
    纽约时报
   </a>
   ，称这篇文章是“a virtual act of Treason”。
  </p>
  <p class="graf graf--p">
   纽约时报公关办公室从其官方推特账号中回复特朗普，捍卫这篇报道，并指出它在出版之前已经被美国政府批准了。
  </p>
  <p class="graf graf--p">
   纽约时报账户说：“我们在出版前向政府描述了这篇文章。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “正如我们的报道所指出的那样，特朗普总统自己的国家安全官员没有对此表示任何担忧”，纽约时报补充道。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6504 jetpack-lazy-image" data-lazy-sizes="(max-width: 1044px) 100vw, 1044px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-12.png?resize=1044%2C1044&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-12.png?w=1044&amp;ssl=1 1044w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-12.png?resize=150%2C150&amp;ssl=1 150w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-12.png?resize=300%2C300&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-12.png?resize=768%2C768&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-12.png?resize=1024%2C1024&amp;ssl=1 1024w" data-recalc-dims="1" height="1044" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-12.png?resize=1044%2C1044&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1044"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6504" data-recalc-dims="1" height="1044" sizes="(max-width: 1044px) 100vw, 1044px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-12.png?resize=1044%2C1044&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-12.png?w=1044&amp;ssl=1 1044w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-12.png?resize=150%2C150&amp;ssl=1 150w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-12.png?resize=300%2C300&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-12.png?resize=768%2C768&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-12.png?resize=1024%2C1024&amp;ssl=1 1024w" width="1044"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   事实上，“纽约时报”关于美国不断升级的针对俄罗斯的网络攻击的报道归因于“现任和前任官员”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这一报道正是来自这些官员，而非勇敢无畏的调查记者的曝光揭露
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    得到“国家安全”官员的批准
   </strong>
  </p>
  <p class="graf graf--p">
   新自由主义者自称“
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/ericgarland/status/1140241109786013696" href="https://twitter.com/ericgarland/status/1140241109786013696" rel="noopener noreferrer" target="_blank">
    Resistance
   </a>
   ”，特朗普肆无忌惮地指控
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/brianklaas/status/1140638004802924544" href="https://twitter.com/brianklaas/status/1140638004802924544" rel="noopener noreferrer" target="_blank">
    叛国罪
   </a>
   （民主联盟
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/TheDemCoalition/status/1140556027047403522" href="https://twitter.com/TheDemCoalition/status/1140556027047403522" rel="noopener noreferrer" target="_blank">
    自称
   </a>
   “我们帮助运行 #TheResistance”，并称特朗普为“普京的傀儡”）。其余的企业媒体都疯了。
  </p>
  <p class="graf graf--p">
   但完全被忽视的是“纽约时报”声明中最具启发性的东西：即 该报基本承认了它与政府有共生关系。
  </p>
  <p class="graf graf--p">
   事实上，
   <strong class="markup--strong markup--p-strong">
    一些著名的美国权威人士甚至坚持认为，这种与政府的共生关系“正是使某人成为记者的原因”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   5月，新保守派华盛顿邮报专栏作家
   <a class="markup--anchor markup--p-anchor" data-href="https://rightweb.irc-online.org/profile/marc-thiessen/" href="https://rightweb.irc-online.org/profile/marc-thiessen/" rel="noopener noreferrer" target="_blank">
    Marc Thiessen
   </a>
   （前总统乔治·W·布什的演讲撰稿人）
   <a class="markup--anchor markup--p-anchor" data-href="https://thegrayzone.com/2019/05/24/trump-charges-assange-press-freedoms-media-obama/" href="https://thegrayzone.com/2019/05/24/trump-charges-assange-press-freedoms-media-obama/" rel="noopener noreferrer" target="_blank">
    宣称
   </a>
   ，维基解密出版人和政治犯 Julian Assange “不是记者”；相反，他是一个“值得被监禁”的“间谍”。
  </p>
  <p class="graf graf--p">
   邮报专栏作家私自撤销 Assange 新闻资质的理由是什么？
  </p>
  <p class="graf graf--p">
   Thiessen 写道：与“信誉良好的新闻机构不同，Assange
   <strong class="markup--strong markup--p-strong">
    没有给美国政府机会审查 Wikileaks 计划发布的机密信息
   </strong>
   ，政府官员没有机会提出异议 …… 所以 [惩罚 Assange 的结果] 其他记者不必害怕”。
  </p>
  <p class="graf graf--p">
   邮报应该给 Thiessen 颁发一个“绝世猪队友”的锦旗了。
  </p>
  <p class="graf graf--p">
   换句话说，
   <strong class="markup--strong markup--p-strong">
    这位前美国政府演讲稿撰稿人认为，与政府合作、接受政府对你的报道的审查，才能在很大程度上决定你是一名记者
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这就是这位著名评论家表达的意识形态。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “纽约时报”编辑非常愿意与政府合作
   </strong>
  </p>
  <p class="graf graf--p">
   企业媒体和政府之间的共生关系已有很长一段时间了。
  </p>
  <p class="graf graf--p">
   情报机构就像一种乐器一样发挥作用 —— 选择性地在适当的时刻泄露信息，以推动政府的软实力并提升当权者的利益。
  </p>
  <p class="graf graf--p">
   但是一直以来很少有这种共生关系被如此随意和毫不知耻地承认。
  </p>
  <p class="graf graf--p">
   2018年，前“纽约时报”记者 James Risen 在“
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2018/01/03/my-life-as-a-new-york-times-reporter-in-the-shadow-of-the-war-on-terror/" href="https://theintercept.com/2018/01/03/my-life-as-a-new-york-times-reporter-in-the-shadow-of-the-war-on-terror/" rel="noopener noreferrer" target="_blank">
    intercept
   </a>
   ”发表了一篇15,000字的文章，进一步深入探讨了这一未说明的审查合作的运作方式。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6505 jetpack-lazy-image" data-lazy-sizes="(max-width: 1018px) 100vw, 1018px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-9.png?resize=1018%2C1080&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-9.png?w=1018&amp;ssl=1 1018w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-9.png?resize=283%2C300&amp;ssl=1 283w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-9.png?resize=768%2C815&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-9.png?resize=965%2C1024&amp;ssl=1 965w" data-recalc-dims="1" height="1080" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-9.png?resize=1018%2C1080&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1018"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6505" data-recalc-dims="1" height="1080" sizes="(max-width: 1018px) 100vw, 1018px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-9.png?resize=1018%2C1080&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-9.png?w=1018&amp;ssl=1 1018w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-9.png?resize=283%2C300&amp;ssl=1 283w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-9.png?resize=768%2C815&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-9.png?resize=965%2C1024&amp;ssl=1 965w" width="1018"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Risen 详细描述了他的编辑“非常愿意与政府合作”。
  </p>
  <p class="graf graf--p">
   事实上，
   <strong class="markup--strong markup--p-strong">
    一位中央情报局的高级官员甚至告诉 Risen，他批准秘密行动的经验法则是“这在纽约时报的头版上看起来怎么样？”
   </strong>
  </p>
  <p class="graf graf--p">
   Risen 解释说，国家和媒体之间存在“非正式安排”，
   <strong class="markup--strong markup--p-strong">
    美国政府官员“定期与新闻界进行谈判，试图阻止任何敏感信息的发布。”
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “当时，我通常会参与这些谈判”，这位前纽约时报记者说。他回忆起他在2001年9月11日袭击事件之前就阿富汗撰写的一个报道的例子。
   <strong class="markup--strong markup--p-strong">
    当时的中央情报局局长 George Tenet 亲自打电话给 Risen，并要求他封杀这个报道。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他告诉我，披露这些消息会威胁到中情局在阿富汗的官员的安全，” Risen 说， “然后我就同意了。”
  </p>
  <p class="graf graf--p">
   Risen 说他后来质疑这是否是正确的决定。
  </p>
  <p class="graf graf--p">
   他写道：
   <strong class="markup--strong markup--p-strong">
    “如果我在911之前发布了这篇报道，那么中央情报局会生气，但是，报道可以引起公众辩论美国是否足以捕获或杀死本·拉登” …… “那场公开辩论将可能迫使中央情报局更加认真地抓捕本·拉登。”
   </strong>
  </p>
  <p class="graf graf--p">
   这种困境导致 Risen 重新考虑回应美国政府对报道审查的要求。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这最终让我与纽约时报的编辑们发生了冲突，”他说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “在9/11袭击事件发生后，
   <strong class="markup--strong markup--p-strong">
    布什政府开始更频繁地要求新闻界封杀报道
   </strong>
   ，” Risen 继续说道。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他们经常这么做，
   <strong class="markup--strong markup--p-strong">
    以至于我确信政府是假借所谓的国家安全来摧毁那些仅仅在政治上令当权者尴尬的报道
   </strong>
   ”。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6506 jetpack-lazy-image" data-lazy-sizes="(max-width: 1044px) 100vw, 1044px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-9.png?resize=1044%2C1104&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-9.png?w=1044&amp;ssl=1 1044w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-9.png?resize=284%2C300&amp;ssl=1 284w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-9.png?resize=768%2C812&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-9.png?resize=968%2C1024&amp;ssl=1 968w" data-recalc-dims="1" height="1104" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-9.png?resize=1044%2C1104&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1044"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6506" data-recalc-dims="1" height="1104" sizes="(max-width: 1044px) 100vw, 1044px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-9.png?resize=1044%2C1104&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-9.png?w=1044&amp;ssl=1 1044w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-9.png?resize=284%2C300&amp;ssl=1 284w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-9.png?resize=768%2C812&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-9.png?resize=968%2C1024&amp;ssl=1 968w" width="1044"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在伊拉克战争前夕，Risen 经常与纽约时报编辑“发生冲突”，因为他对政府的谎言提出了质疑。
  </p>
  <p class="graf graf--p">
   但他的报道所质疑的 “特别是政府对伊拉克与基地组织之间存在联系的声称，都被砍掉了、埋葬了、被拒绝了。”
  </p>
  <p class="graf graf--p">
   纽约时报的执行编辑 Howell Raines “被许多人认为更喜欢支持战争的报道，”Risen 说。
  </p>
  <p class="graf graf--p">
   在另一则轶事中，Risen 回忆起他在一个拙劣的中情局情节中发现的一个独家新闻。
  </p>
  <p class="graf graf--p">
   布什政府听到了风声，并把他叫到了白宫，当时的国家安全顾问康多莉扎·赖斯命令“纽约时报”封杀这个报道。
  </p>
  <p class="graf graf--p">
   Risen 说赖斯告诉他 “要忘记这个故事，销毁我的笔记，不要再拨打电话与任何人讨论此事。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “布什政府成功地说服媒体搜查和封杀报道”，Risen 写道，而巴拉克•奥巴马（Barack Obama）政府随后加速了“对新闻界的战争”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    中央情报局媒体渗透和制造同意
   </strong>
  </p>
  <p class="graf graf--p">
   在著名的媒体研究报告 “制造同意：大众传媒的政治经济学”中，Edward S. Herman 和 Chomsky 阐述了一种“宣传模式” —— 展示了“媒体如何为强大的权势服务、制造符合权力机构政策的新闻价值定义”。
  </p>
  <p class="graf graf--p">
   但在某些情况下，情报机构与企业媒体之间的关系不仅仅是意识形态维稳、或间接压力，而是雇主关系的一种。
  </p>
  <p class="graf graf--p">
   20世纪50年代，中央情报局发起了一项名为 Project Mockingbird 的秘密行动，监视、影响和操纵记者和媒体的报道，明确是为了引导公众舆论。
  </p>
  <p class="graf graf--p">
   在这里看到介绍《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E5%86%B7%E6%88%98%E6%95%85%E4%BA%8B%EF%BD%9E-cia-%E5%A6%82%E4%BD%95%E6%93%8D%E7%BA%B5%E5%AA%92%E4%BD%93%E5%BD%B1%E5%93%8D%E8%88%86%E8%AE%BA%EF%BC%9Amockingbird-%E8%A1%8C%E5%8A%A8/" href="https://www.iyouport.org/%E5%86%B7%E6%88%98%E6%95%85%E4%BA%8B%EF%BD%9E-cia-%E5%A6%82%E4%BD%95%E6%93%8D%E7%BA%B5%E5%AA%92%E4%BD%93%E5%BD%B1%E5%93%8D%E8%88%86%E8%AE%BA%EF%BC%9Amockingbird-%E8%A1%8C%E5%8A%A8/" rel="noopener noreferrer" target="_blank">
    冷战故事～ CIA 如何操纵媒体影响舆论：Mockingbird 行动
   </a>
   》。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    传奇记者 Carl Bernstein 是华盛顿邮报的前记者，曾帮助揭开水门事件的丑闻
   </strong>
   ，他于1977年发表了一篇题为“
   <a class="markup--anchor markup--p-anchor" data-href="http://www.carlbernstein.com/magazine_cia_and_media.php" href="http://www.carlbernstein.com/magazine_cia_and_media.php" rel="noopener noreferrer" target="_blank">
    中央情报局和媒体
   </a>
   ”的滚石杂志封面故事。
  </p>
  <p class="graf graf--p">
   Bernstein 获得了中央情报局的文件，这些文件显示，
   <strong class="markup--strong markup--p-strong">
    过去25年中有400多名美国记者“秘密为中央情报局执行任务”。
   </strong>
  </p>
  <p class="graf graf--p">
   Bernstein 写道：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “记者提供了全方位的秘密服务 —— 从简单的情报收集到担任间谍的中间人。记者与中央情报局分享了他们的笔记 ……
  </p>
  <p class="graf graf--p">
   …… 大多数人都不那么崇高了：他们发现与情报机构的关系有助于他们的职业生涯 …… 甚至，全
   <strong class="markup--strong markup--p-strong">
    职的中央情报局员工伪装成国外记者
   </strong>
   。
  </p>
  <p class="graf graf--p">
   …… 中央情报局的文件显示，在许多情况下，在美国主要
   <strong class="markup--strong markup--p-strong">
    新闻机构管理层的同意下
   </strong>
   ，记者们参与了中央情报局的任务”。
  </p>
  <p class="graf graf--p">
   Bernstein 透露，
   <strong class="markup--strong markup--p-strong">
    几乎所有主流媒体都与中央情报局合作
   </strong>
   ，包括ABC，NBC，美联社，UPI，路透社，新闻周刊，赫斯特报业，迈阿密先驱报，星期六晚报和纽约先驱论坛报……
  </p>
  <p class="graf graf--p">
   然而，他补充说，“根据中央情报局官员的说法，这些媒体机构中最有价值的一直是纽约时报、CBS 和时代公司。”
  </p>
  <p class="graf graf--p">
   这些层面的国家操纵、审查、甚至新闻媒体自己也承认了这点，即 这些主流媒体实际上只是政府的发言人。⚪️
  </p>
  <p class="graf graf--p">
   延伸：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%93%8d%e7%ba%b5%e5%bc%8f%e5%ae%a3%e4%bc%a0%ef%bc%9a%e4%ba%94%e4%b8%aa%e8%bf%87%e6%bb%a4%e5%99%a8%e5%92%8c%e8%af%86%e5%88%ab%e7%ba%bf%e7%b4%a2/" href="https://www.iyouport.org/%e6%93%8d%e7%ba%b5%e5%bc%8f%e5%ae%a3%e4%bc%a0%ef%bc%9a%e4%ba%94%e4%b8%aa%e8%bf%87%e6%bb%a4%e5%99%a8%e5%92%8c%e8%af%86%e5%88%ab%e7%ba%bf%e7%b4%a2/" rel="noopener noreferrer" target="_blank">
    主流媒体为什么注定撒谎？
   </a>
   》；《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%ae%88%e9%97%a8%e4%ba%ba%e4%bd%9c%e4%b8%ba%e8%ae%a4%e7%9f%a5%e6%9d%80%e6%89%8b/" href="https://www.iyouport.org/%e5%ae%88%e9%97%a8%e4%ba%ba%e4%bd%9c%e4%b8%ba%e8%ae%a4%e7%9f%a5%e6%9d%80%e6%89%8b/" rel="noopener noreferrer" target="_blank">
    守门人作为认知杀手
   </a>
   》
  </p>
