  <p class="graf graf--p">
   中国外交在 Twitter 上有了新的声音 —— 而且不仅仅是外交。
  </p>
  <p class="graf graf--p">
   共产党政府最近拥抱了这个社交媒体平台，尽管GFW继续对其封锁，但当局仍部署了外交部和一支日益壮大的外交官队伍，旨在向全球听众宣传和捍卫其政策。
  </p>
  <p class="graf graf--p">
   一位外交官发布了在尼泊尔的艺术自拍照，中国驻南非特使引用西方诗歌以及日落和野生动植物的图片发帖，而驻英国大使刘晓明则利用该网站对陷入困境的中国电信巨头华为进行了强硬的辩护。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12904 jetpack-lazy-image" data-lazy-sizes="(max-width: 900px) 100vw, 900px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-9.png?resize=900%2C1302&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-9.png?w=900&amp;ssl=1 900w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-9.png?resize=207%2C300&amp;ssl=1 207w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-9.png?resize=708%2C1024&amp;ssl=1 708w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-9.png?resize=768%2C1111&amp;ssl=1 768w" data-recalc-dims="1" height="1302" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-9.png?resize=900%2C1302&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="900"/>

  <p class="graf graf--p">
   他们都是最近几个月来开设推特账户的十几位中国大使和总领事中的一员，通常采用的风格与外交传统大相径庭。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12905 jetpack-lazy-image" data-lazy-sizes="(max-width: 898px) 100vw, 898px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/2-12.png?resize=898%2C962&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/2-12.png?w=898&amp;ssl=1 898w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/2-12.png?resize=280%2C300&amp;ssl=1 280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/2-12.png?resize=768%2C823&amp;ssl=1 768w" data-recalc-dims="1" height="962" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/2-12.png?resize=898%2C962&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="898"/>

  </p>
  <p class="graf graf--p">
   现在，中国政府本身也参与其中了，外交部年初发表了第一条推文，用上了“LOL”、感叹号和主题标签，以抵制批判者并赞扬北京的世界观。
  </p>
  <p class="graf graf--p">
   外交部推特账户称，“有些人宁愿接受谎言也不愿相信权威信息”，这是对前中国间谍王立强曝光事件的评论推文。王立强去年底在澳大利亚寻求庇护。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12906 jetpack-lazy-image" data-lazy-sizes="(max-width: 904px) 100vw, 904px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/3-10.png?resize=904%2C1138&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/3-10.png?w=904&amp;ssl=1 904w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/3-10.png?resize=238%2C300&amp;ssl=1 238w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/3-10.png?resize=813%2C1024&amp;ssl=1 813w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/3-10.png?resize=768%2C967&amp;ssl=1 768w" data-recalc-dims="1" height="1138" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/3-10.png?resize=904%2C1138&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="904"/>

  <p class="graf graf--p">
   这些非正式的、有时甚至是对抗性的口气与政府通常镇定的正式声明相去甚远 —— 这种做法导致了偶尔的公众热议。
  </p>
  <p class="graf graf--p">
   外交部高级官员赵立坚去年与美国前驻联合国大使 Susan Rice 在网上发生了口角，后者称他为 “种族主义耻辱” 在他发布推文称华盛顿存在种族歧视之后。
  </p>
  <p class="graf graf--p">
   对社交媒体的推动之际，中国正面临着国际社会越来越大的压力，因为在西部地区新疆的穆斯林少数群体被大规模拘留、以及香港的民主抗议活动加剧，而与美国的贸易战仍在继续。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12907 jetpack-lazy-image" data-lazy-sizes="(max-width: 1050px) 100vw, 1050px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/4-2.jpg?resize=1050%2C583&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/4-2.jpg?w=1050&amp;ssl=1 1050w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/4-2.jpg?resize=300%2C167&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/4-2.jpg?resize=1024%2C569&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/4-2.jpg?resize=768%2C426&amp;ssl=1 768w" data-recalc-dims="1" height="583" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/4-2.jpg?resize=1050%2C583&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1050"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   英国利兹大学的媒体讲师 Yuan Zeng 说，中国官员和媒体长期以来难以令人信服地向全球受众传播他们的信息，他们 “迫切需要一种更有效的方法来释放中国的声音”。
  </p>
  <h3 class="graf graf--p graf--startsWithSingleQuote">
   <strong class="markup--strong markup--p-strong">
    ‘特朗普效应’
   </strong>
  </h3>
  <p class="graf graf--p">
   中国外交部在 Twitter 上的存在与美国总统多产的高音喇叭形成了鲜明的对比。特朗普一直利用该平台攻击对手并积极赞扬自己的政策。
  </p>
  <p class="graf graf--p">
   香港科技大学教授 Wenfang Tang 说，北京认为 “特朗普在社交媒体上非常受欢迎，西方媒体经常引用他的推文。所以从某种意义上讲，这是特朗普效应”。
  </p>
  <p class="graf graf--p">
   中国官员此前在社交媒体上一直保持低调，将发声机会留给国营媒体。
  </p>
  <p class="graf graf--p">
   但是咨询公司 China Circle 的媒体专家 Ardi Bouwers 表示，中国日益增长的政治和经济影响力已经促使外交官们在互联网上和现实中以越来越自信的和民族主义的声音说话。
   <a aria-label="propaganda (59个项目)" class="tag-cloud-link tag-link-29 tag-link-position-27" href="https://www.iyouport.org/wp-admin/post-new.php#" role="button">
    propaganda
   </a>
  </p>
  <p class="graf graf--p">
   Bouwers 说，这种举动还使他们能够证明其对习近平的忠诚。习本人“使用爱国语言、谈论自力更生、中华民族的伟大复兴，以粉碎分裂中国的企图”。
  </p>
  <p class="graf graf--p">
   外交部在一份声明中称，它已经在 Twitter 开设了帐户，“以便更好地与其他国家进行交流并更好地解释中国的情况和政策”。
  </p>
  <p class="graf graf--p">
   新闻发言人耿爽被问及鉴于该平台在中国境内被封锁，因此使用 Twitter 是否公平。“我们拥有世界上最大的互联网用户群体。同时，我们一直按照法律法规管理互联网。” 耿说。
  </p>
  <p class="graf graf--p">
   他说，“中国互联网是开放的”。
  </p>

  <h3 class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “不过是宣传”
   </strong>
  </h3>
  <p class="graf graf--p">
   在海外，对中国社交媒体宣传运动的反应好坏参半。
  </p>
  <p class="graf graf--p">
   北京驻尼泊尔大使侯艳琪在传统尼泊尔建筑的背景下建立了一个网上粉丝群，上面有时尚且经过精心构图的她自己的照片。
  </p>
  <p class="graf graf--p">
   她的照片已被 “点赞’’ 过数千次，并在很大程度上受到了本地 Twitter 用户的好评。
  </p>
  <p class="graf graf--p">
   但是对外交部推文的答复经常充斥着轻蔑和嘲弄，常常伴随着新闻报道，记录着中国的镇压政策，以及批评北京的讽刺漫画。
  </p>
  <p class="graf graf--p">
   西交利物浦大学国际关系学教授 Alessandra Cappelletti 对法新社表示：“所有这些努力都是针对部分 Twitter 用户的宣传。” “中国信息专家如何在受过良好教育的外国受众群体中树立自己的国家形象，还有待观察”。⚪️
  </p>
