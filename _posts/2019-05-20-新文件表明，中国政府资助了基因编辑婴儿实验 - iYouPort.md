   新文件表明，中国政府资助了基因编辑婴儿实验
  </h1>

 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      贺建奎究竟是不是独狼？有新文件显示他接受了中国当局的资助，是真的吗？
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   根据
   <a class="markup--anchor markup--p-anchor" data-href="https://www.statnews.com/2019/02/25/crispr-babies-study-china-government-funding/" href="https://www.statnews.com/2019/02/25/crispr-babies-study-china-government-funding/" rel="noopener noreferrer" target="_blank">
    STAT获得的文件
   </a>
   ，中国政府可能已经提供资金用于一个有争议的项目，即去年非常轰动的婴儿基因编辑实验。虽然目前尚不清楚政府官员是否密切关注资金的使用方式，但这些文件与中国科学家贺建奎被作为反叛者的表面叙述相矛盾，后者被认为是单独行动的并反对上层人士的意愿。
  </p>
  <p class="graf graf--p">
   早在去年11月，贺建奎在香港的一次会议上
   <a class="markup--anchor markup--p-anchor" data-href="https://www.technologyreview.com/s/612458/exclusive-chinese-scientists-are-creating-crispr-babies/" href="https://www.technologyreview.com/s/612458/exclusive-chinese-scientists-are-creating-crispr-babies/" rel="noopener noreferrer" target="_blank">
    宣布
   </a>
   ，他使用 CRISPR 工具对双胞胎女孩进行了基因编辑，使她们对艾滋病病毒产生更强的抵抗力。国际科学界迅速谴责这些行动，并说这不仅在道德上是错误的，编辑本身也是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theverge.com/2018/11/29/18116830/crispr-baby-he-jiankui-genetics-ethics-science-health-mutation" href="https://www.theverge.com/2018/11/29/18116830/crispr-baby-he-jiankui-genetics-ethics-science-health-mutation" rel="noopener noreferrer" target="_blank">
    草率的
   </a>
   ，可能没有用。当时，中国政府也迅速谴责贺，并开始进行一项调查，官方描绘了一位孤立科学家的形象，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theverge.com/2019/1/22/18192961/crispr-genetic-engineering-baby-ethics-scientist-china-investigation-he-jiankui" href="https://www.theverge.com/2019/1/22/18192961/crispr-genetic-engineering-baby-ethics-scientist-china-investigation-he-jiankui" rel="noopener noreferrer" target="_blank">
    称他
   </a>
   “无视政府禁令追求个人名利。”
  </p>
  <p class="graf graf--p">
   但 STAT 获得的文件显示，他的研究项目可能部分来自他以前的工作场所，即当地的深圳政府和大中华区的科技分支机构。然而，资金可能是以前的项目，而
   <a class="markup--anchor markup--p-anchor" data-href="https://www.statnews.com/2019/02/25/crispr-babies-study-china-government-funding/" href="https://www.statnews.com/2019/02/25/crispr-babies-study-china-government-funding/" rel="noopener noreferrer" target="_blank">
    接受采访的其他科学家表示
   </a>
   ，他有可能假装政府资助的可信度。
  </p>
  <p class="graf graf--p">
   STAT
   <a class="markup--anchor markup--p-anchor" data-href="https://www.statnews.com/2019/02/25/crispr-babies-study-china-government-funding/" href="https://www.statnews.com/2019/02/25/crispr-babies-study-china-government-funding/" rel="noopener noreferrer" target="_blank">
    报道说
   </a>
   ，中国科学部发表声明回应这些文件，声称“没有资助他的人类基因组编辑活动”。
  </p>
  <p class="graf graf--p">
   纽约大学的生物伦理学家
   <a class="markup--anchor markup--p-anchor" data-href="https://med.nyu.edu/faculty/arthur-l-caplan" href="https://med.nyu.edu/faculty/arthur-l-caplan" rel="noopener noreferrer" target="_blank">
    Arthur Caplan
   </a>
   说，尽管中国研究人员担心他的实验已经玷污了该国的科学声誉，但这些新文件并不一定意味着中国的信誉再次受到损害。Caplan 认为可能会有一些初步的批准，特别是因为中国根本没有明确禁止基因工程胚胎，这种研究不会引发与美国或其他国家相同研究的道德危险信号，即强大的生命权运动。但从那以后，“中国政府已退出并明确表示他们不想超越这一领域的规则，”他说。
  </p>
  <p class="graf graf--p">
   当然，国际科学界仍在急于弄清楚
   <a class="markup--anchor markup--p-anchor" data-href="https://fivethirtyeight.com/features/we-have-ways-to-stop-rogue-scientists-they-dont-always-work/" href="https://fivethirtyeight.com/features/we-have-ways-to-stop-rogue-scientists-they-dont-always-work/" rel="noopener noreferrer" target="_blank">
    如何防止
   </a>
   这些实验再次发生。中国前卫生部副部长建议，国家
   <a class="markup--anchor markup--p-anchor" data-href="http://www.globaltimes.cn/content/1129467.shtml" href="http://www.globaltimes.cn/content/1129467.shtml" rel="noopener noreferrer" target="_blank">
    应该建立
   </a>
   一个监督生物医学实验的组织，例如，世界卫生组织已经成立了一个委员会来
   <a class="markup--anchor markup--p-anchor" data-href="https://www.npr.org/sections/health-shots/2019/02/14/694710663/world-health-organization-forms-committee-to-guide-editing-of-human-genes" href="https://www.npr.org/sections/health-shots/2019/02/14/694710663/world-health-organization-forms-committee-to-guide-editing-of-human-genes" rel="noopener noreferrer" target="_blank">
    研究
   </a>
   婴儿基因编辑的指导方针。
  </p>
  <p class="graf graf--p">
   Caplan 认为有一些建议可以添加进去。首先，他说，每个听过这种项目的科学家都应该向他们的研究机构和可能的国立卫生研究院报告。其次，这种规模的研究不应该给媒体随意报道，任何公告都需要伴随一篇有数据的论文。
  </p>
  <p class="graf graf--p">
   与此同时，关于他和他的职业生涯的下一步以及这对双胞胎的表现将会
   <a class="markup--anchor markup--p-anchor" data-href="https://go.redirectingat.com/?id=66960X1514734&amp;xs=1&amp;url=https%3A%2F%2Fwww.nature.com%2Farticles%2Fd41586-019-00673-1" href="https://go.redirectingat.com/?id=66960X1514734&amp;xs=1&amp;url=https%3A%2F%2Fwww.nature.com%2Farticles%2Fd41586-019-00673-1" rel="noopener noreferrer" target="_blank">
    有很多问题
   </a>
   。他还承认编辑了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theverge.com/2019/1/22/18192961/crispr-genetic-engineering-baby-ethics-scientist-china-investigation-he-jiankui" href="https://www.theverge.com/2019/1/22/18192961/crispr-genetic-engineering-baby-ethics-scientist-china-investigation-he-jiankui" rel="noopener noreferrer" target="_blank">
    其他胚胎
   </a>
   ，目前还不清楚这些婴儿何时会出生。中国现在正受到所有关注，但另一个问题是 CRISPR 婴儿可能在哪里出生。法律背景仍然不确定，北海道大学的生物伦理学家石井哲
   <a class="markup--anchor markup--p-anchor" data-href="https://go.redirectingat.com/?id=66960X1514734&amp;xs=1&amp;url=https%3A%2F%2Fwww.nature.com%2Farticles%2Fd41586-019-00673-1" href="https://go.redirectingat.com/?id=66960X1514734&amp;xs=1&amp;url=https%3A%2F%2Fwww.nature.com%2Farticles%2Fd41586-019-00673-1" rel="noopener noreferrer" target="_blank">
    也表示
   </a>
   ，以色列、俄罗斯和西班牙等国家对基因编辑的政策都很不严格。
  </p>
