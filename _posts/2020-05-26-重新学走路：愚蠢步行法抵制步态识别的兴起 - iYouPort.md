  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">

   与人工智能（AI）软件结合使用时，即使隐藏了脸部，该技术仍然可以从50米外的普通监控摄像机镜头中识别出人。
  </p>
  <p class="graf graf--p">
   尽管步态识别（例如面部识别）不是一种新技术，但人工智能的进步使其更加有效，使监视者可以实时识别任何人。
  </p>
  <p class="graf graf--p">
   曼彻斯特大学的研究人员能够使用20,000英尺的步态数据来训练AI，以识别不同的因素，例如人们的步行节奏等等。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    随着这些形式的技术变得更先进，开发成本更低，它们也可能会变得更加商业化。
   </strong>
  </p>
  <p class="graf graf--p">
   该技术的开发者声称，他们的步态识别
   <strong class="markup--strong markup--p-strong">
    不能
   </strong>
   被假装跛脚、弯腰或故意外八字的走路方式所抵消。但这可能不会阻止希望避免监视的人们。
  </p>
  <p class="graf graf--p">
   除了已经在市场上投放的各种反监视服装之外，也许2020年，反监视技术也将像 Monty Python 式的愚蠢步行一样爆发。

   基于采用面部识别的方式，人们似乎也将越来越熟悉这些其他技术。如果我们要避免进入超级监视的新纪元，那么确保有效管理这些技术至关重要。
  </p>
  <p class="graf graf--p">
   大多数技术天生不是善恶分明的，而是取决于使用它的人。这意味着治理不仅对于保护隐私很重要，而且可以支持负责任地将技术用于社会公益事业。
  </p>
  <p class="graf graf--p">
   👉IYP 非常支持 Monty Python 式的愚蠢步行，并不是说确认它 “可以抵制” 步态识别，而是，它本身是一场反抗运动 —— 它采用的原理是 **让隐形可见**，以及 **将你的对手陷入愚蠢滑稽**。在下面回顾原理：
