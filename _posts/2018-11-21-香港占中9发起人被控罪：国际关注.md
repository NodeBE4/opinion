</p>

<p class="wp-caption-text">
<span style="font-size: 10pt; text-indent: 2em;">
    有“占中三子”之称的陈健民（左起）、戴耀廷和朱耀明于2018年11月19日进入法庭前呼喊口号
   </span>
</p>
</div>
<p>
</p>
<p>
<span style="font-size: 12pt;">
   4年前，香港的“占领行动”震撼国际社会。3名发起人以及另外6人被控公众妨扰等罪名，案件周一（11月19日）开审。案件已引起国际关注，有美国律师更警告，审讯结果可能引起美国国会更大反应，重新审视香港的地位。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   案件周一在香港西九龙法庭开审。“占领行动”三名发起人戴耀廷、陈健民和朱耀明，以及另外6名被告，包括立法会议员陈淑庄及邵家臻合共面对6项控罪。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   9人被指分别在2014年9月27到28日期间，在金钟地区煽惑其他人非法阻碍公众地方及道路。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   有“占中三子”之称的戴耀廷、陈健民和朱耀明，另外被控，在2013年至2014年串谋其他人阻碍中环及附近道路，妨扰公众，9名被告均否认控罪。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   控方宣读开审陈词，指3名“占领行动”发起人，在2013年3月召开记者会，提议以非法堵塞道路方式占据中环一带的车路，作为争取普选的手段，其后透过接受媒体访问，举办会议及筹款来宣扬信息，鼓励公众参与。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   控方又指2014年9月下旬，3人与陈淑庄、邵家臻等人，分别参与及主持集会，号召市民到场。他们的行为对交通造成不必要阻碍，对公众构成损害。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   在9月28日凌晨，戴、陈、朱3人宣布“占领中环正式启动”。其中戴耀廷要求政府回应，否则“占领行动”将会升级。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   案件预计审讯1个月，部分被告将会自辩。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   香港特首强硬回应国际社会关注
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   这次审讯引起国际社会高度关注。英国下议院多名议员动议谴责香港特区政府起诉参与者。香港特首林郑月娥则回应这是干预香港内部事务。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   林郑月娥: “这是香港在司法独立，在法治前提下，香港检控部门作出的决定，而“基本法”之下这个检控决定只属于律政司司长，我本人作为行政长官都不会干预，而外国政府或外国的国会议员竟然可以提出要求，我觉得这很明显是干预香港内部事务，是非常不可取的。”
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   美国国会上星期发表报告，批评香港法治和言论自由持续被侵蚀，反映“一国两制”下的香港越来越像中国大陆城市，建议检视美国科技出口政策中，把香港视为独立关税区的做法。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   熟悉美国国会运作、并且长期关注香港人权问题的美国律师方恩格（Ross Feingold）认为，随着民主党即将重掌国会众议院，相信这次审讯将受到极大关注。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   方恩格: “外交委员会主席将由民主党人出任。如果民主党觉得白宫或国务院对香港人权问题反应不够强，或许往后他们的反应会（比之前）更强烈。过去两三年，基本上任何香港人权问题都引起美国国会的关注，这次“占中”（人士）如果被判罪成，必定会有国会议员关注。”
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   他以民主共和两党国会议员近期去信总统特朗普，引用“马格尼茨基法案”，要求他彻查“华盛顿邮报”作家怀疑遭虐杀为例，认为在有关香港的问题上，不排除国会议员也会以类似手段，向商务部等行政部门施压。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   出处 ：RFA
