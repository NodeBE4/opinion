    <span style="color: #00ccff;">
     <em>
      <strong>
       多年前我们论述“反转GFW”的时候，其主题是提醒警惕中国BAT的全球扩张足迹；仅仅是技术和商业角度上的，也是不全面的。
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   <strong>
    事实上“反转GFW”的助推器还包括人 —— 也就是华人移民
   </strong>
   。就如人们在巴基斯坦看到的状况那样（详细《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%B8%AD%E5%9B%BD%E5%A6%82%E4%BD%95%E5%9C%A8%E5%B7%B4%E5%9F%BA%E6%96%AF%E5%9D%A6%E6%89%A9%E5%BC%A0%E5%85%B6%E8%B6%B3%E8%BF%B9%EF%BC%9F/" href="https://www.iyouport.org/%E4%B8%AD%E5%9B%BD%E5%A6%82%E4%BD%95%E5%9C%A8%E5%B7%B4%E5%9F%BA%E6%96%AF%E5%9D%A6%E6%89%A9%E5%BC%A0%E5%85%B6%E8%B6%B3%E8%BF%B9%EF%BC%9F/" rel="noopener noreferrer" target="_blank">
    中国如何在巴基斯坦扩张其足迹
   </a>
   》）。
  </p>
  <p class="graf graf--p">
   华人移民的特点是不论身在哪里，都基本不会放弃对微信的使用（其背后原因有点复杂，不仅是乡愁问题）。这便造成了其所在国家被微信所“渗透”；随着华人移民越来越多，很多国家的政客都难以忽视华裔社区的选票……其结果便是：
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4120 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1-6.png?resize=1100%2C724&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1-6.png?w=1400&amp;ssl=1 1400w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1-6.png?resize=300%2C198&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1-6.png?resize=768%2C506&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1-6.png?resize=1024%2C674&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1-6.png?resize=1100%2C724&amp;ssl=1 1100w" data-recalc-dims="1" height="724" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1-6.png?resize=1100%2C724&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   包括首相和反对党领袖在内的澳大利亚政界人士，正出现在中国最大的社交媒体平台微信上，他们使用的官方账户是中国公民设立的。
  </p>
  <p class="graf graf--p">
   这种做法似乎违反了该平台的条款和条件，并
   <strong class="markup--strong markup--p-strong">
    使澳大利亚政客进入了可以被中国政府随意审查的区域内。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️WeChat 声称它在澳大利亚拥有100万月度活跃用户。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    澳大利亚政界人士正在利用微信这样的平台笼络华裔澳大利亚选民的热情。
   </strong>
  </p>
  <p class="graf graf--p">
   总理 Scott Morrison 的微信官方账号于1月份以中国福建省一名不知名的中国公民的名义登记，而 Bill Shorten 和 ALP 的官方账户则登记在山东省一名身份不明的中国男子名下。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    ABC 已经确定了12名澳大利亚政客，他们正在使用以中国人的名义注册的微信账号，其中包括政府部长
   </strong>
   。
  </p>
  <p class="graf graf--p">
   根据微信的纸上规定，“初始注册申请人不得捐赠、借用、出租、转让或出售微信账户，也不允许任何非初始注册申请人使用微信账户。”
  </p>
  <p class="graf graf--p">
   微信的用户协议声明它有权禁止或取消违反规则的账户。
  </p>
  <p class="graf graf--p">
   但在澳大利亚的案例中，上述这些规定显然都不存在。
  </p>
  <p class="graf graf--p">
   在澳大利亚政客这事里，任何一方都不会透露哪些中国公民拥有这些账户。
  </p>
  <p class="graf graf--p">
   很明显，
   <strong class="markup--strong markup--p-strong">
    通过使用在中国注册的账户，澳大利亚政客已经受制于中国政府的政策，这些政策要求用户推动共产党的意识形态、维护公共秩序、追求中国的国家利益。
   </strong>
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4121 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-3.png?resize=1100%2C730&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-3.png?w=1398&amp;ssl=1 1398w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-3.png?resize=300%2C199&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-3.png?resize=768%2C510&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-3.png?resize=1024%2C680&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-3.png?resize=1100%2C730&amp;ssl=1 1100w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-3.png?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="730" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-3.png?resize=1100%2C730&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4121" data-recalc-dims="1" height="730" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-3.png?resize=1100%2C730&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-3.png?w=1398&amp;ssl=1 1398w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-3.png?resize=300%2C199&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-3.png?resize=768%2C510&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-3.png?resize=1024%2C680&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-3.png?resize=1100%2C730&amp;ssl=1 1100w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-3.png?resize=272%2C182&amp;ssl=1 272w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    账户背后的中国公民到底是谁？
   </strong>
  </p>
  <p class="graf graf--p">
   澳大利亚联邦工党有一个在中国注册的中心级账户 “Bill Shorten and Labor”.
  </p>
  <p class="graf graf--p">
   该账户在2017年5月之前还只是一个品茶主题的中国账户。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    ABC已经找到自由党候选人的11个微信官方账号
   </strong>
   ，其中包括城市部长 Alan Tudge，他的账户是在天津市的一名身份不明的男子名下登记的，还有移民部长 David Coleman，他在一位北京籍公民的身份下登记了账户。
  </p>
  <p class="graf graf--p">
   澳大利亚战略政策中心的网络政策专家 Fergus Ryan 表示，“我们的政治领导人需要做一些解释了”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果我们查看 Bill Shorten 的 Instagram 帐户或他的推特，或总理 Scott Morrison 的 Instagram 帐户或 Twitter 帐户，发现它是由爱达荷州的一位未透露姓名的男性注册的，我们肯定会认为这很奇怪。中国的微信也不例外，”Ryan 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “我认为首先我们应该知道这些账户注册人的姓名和他们的注册方式，以及这些人与有关政客之间的关系。”
   </em>
  </p>
  <p class="graf graf--p">
   总理办公室不会直接回答有关 Morrison 先生账户所有权的问题，并在一份书面声明中说：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <strong class="markup--strong markup--p-strong">
    总理设立微信账号是为了与使用该社交媒体网络的华裔澳大利亚社区的一些人进行交流。使用微信才能直接与社区讨论他们关心的问题
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   ALP 运动的发言人在一份声明中说：“该帐户的经营者是澳大利亚居民和澳大利亚工党雇员”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这是建立帐户的人，我们所有的社交媒体帐户都由反对党领导办公室和本党在澳大利亚管理。”
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4122 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpeg?resize=1100%2C1059&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpeg?w=1122&amp;ssl=1 1122w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpeg?resize=300%2C289&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpeg?resize=768%2C739&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpeg?resize=1024%2C986&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpeg?resize=1100%2C1059&amp;ssl=1 1100w" data-recalc-dims="1" height="1059" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpeg?resize=1100%2C1059&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4122" data-recalc-dims="1" height="1059" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpeg?resize=1100%2C1059&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpeg?w=1122&amp;ssl=1 1122w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpeg?resize=300%2C289&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpeg?resize=768%2C739&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpeg?resize=1024%2C986&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpeg?resize=1100%2C1059&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    澳大利亚政客必须遵守共产党的规则
   </strong>
  </p>
  <p class="graf graf--p">
   中国注册的账户所受到的审查比国际注册的账户更加严格。
  </p>
  <p class="graf graf--p">
   使用这些账户的澳大利亚政客必须遵守微信政策和中国共产党关于互联网使用的任何指导方针 —— 那是一套严格的规定，其名称听起来像是在中央政治局会议上讨论过的那种东西。
  </p>
  <p class="graf graf--p">
   它被称为“通过即时通讯工具提供的公共信息服务发展管理暂行规定”，该规定规范了中国的所有社交媒体。
  </p>
  <p class="graf graf--p">
   根据此政策，这些帐户的用户应该：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    维护公共秩序和社会道德
   </li>
   <li class="graf graf--li">
    追求中国的国家利益
   </li>
   <li class="graf graf--li">
    推进“社会主义制度”（请记得，中国讲的“社会主义”不是理论上的社会主义）
   </li>
  </ul>
  <p class="graf graf--p">
   位于多伦多的研究中心 The Citizen Lab 发现，在中国大陆注册的微信用户均被通过关键词过滤进行了严格的审查。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “公民实验室”报告的合著者 Lotus Ruan 说，微信过滤了几乎所有对中国政府和执政党领导人的批评，有关香港、台湾和民族的问题，以及天安门广场大屠杀等历史事件相关的内容。
  </p>
  <p class="graf graf--p">
   根据 Ruan 的说法，有时候用户不知道自己已被审查过。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果一条消息中包含某个已被黑名单的关键字，整个消息只有发布者自己可见，”她说。
  </p>
  <p class="graf graf--p">
   Fergus Ryan 认为，即使澳大利亚政客的信息没有受到审查，他们仍然需要通过自我审查才能继续使用微信去接触他们的关键选民。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他们可能已经意识到围绕微信的安全隐患和审查问题，但是他们也有很多的动机需要尝试与这些选民进行沟通，”他说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <strong class="markup--strong markup--p-strong">
    由于担心这条*重要的通道*被关闭，他们可能不想说任何被北京认为过于敏感的事和话题。”
   </strong>
  </p>
  <p class="graf graf--p">
   在四月举行的微信现场论坛上，Bill Shorten 被问及一系列有关华为、中国干涉澳大利亚政治、亿万富翁商人和政治捐助者黄祥模的问题，以及对中国共产党在澳大利亚之存在的负面看法。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    他没有回答任何这些问题
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Shorten 的竞选团队在一份声明说：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们不容忍任何旨在破坏我们自由和公平社会的外来干涉。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “工党从未在任何社交媒体平台上对我们的通信进行任何审查。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    为什么澳大利亚政客要在中国注册？
   </strong>
  </p>
  <p class="graf graf--p">
   自由党声称他们的微信活动扭转了2016年最后一次联邦选举中 Chisholm 的失败结果，其中自由党在全国范围内逆势而上并抢夺了工党的边缘席位。
  </p>
  <p class="graf graf--p">
   在中国注册的官方账户允许政客发布更多文章并向潜在选民发送更多通知。
  </p>

   微信提供订阅账户和服务账户 —— 但前者只适用于中国公民和企业。
  </p>
  <p class="graf graf--p">
   订阅帐户每天最多可向其关注者发送一个通知，每次“推送”最多8篇文章。
  </p>
  <p class="graf graf--p">
   非中国公民可以拥有的服务帐户每月可以发布四条消息。
