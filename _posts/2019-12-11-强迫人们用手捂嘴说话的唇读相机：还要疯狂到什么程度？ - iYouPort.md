     <em>
      <strong>
       必需捂住嘴巴才能说话的日子可能不远了
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-8972 jetpack-lazy-image" data-lazy-sizes="(max-width: 634px) 100vw, 634px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1.jpg?resize=634%2C422&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1.jpg?w=634&amp;ssl=1 634w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1.jpg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="422" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1.jpg?resize=634%2C422&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="634"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8972" data-recalc-dims="1" height="422" sizes="(max-width: 634px) 100vw, 634px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1.jpg?resize=634%2C422&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1.jpg?w=634&amp;ssl=1 634w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1.jpg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1.jpg?resize=272%2C182&amp;ssl=1 272w" width="634"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   监督机构警告说，公民遭受政府的监视有可能比“1984”中描述的国内间谍活动更具侵扰性。
  </p>
  <p class="graf graf--p">
   英国已经拥有比其他任何国家都更多的人均闭路电视摄像机比例，大都会警察局已经在伦敦试用了面部识别摄像头，并且，
   <strong class="markup--strong markup--p-strong">
    现在步态识别的技术已经出现，甚至监视器可以用读唇技术获知人们说了什么话。
   </strong>
  </p>
  <p class="graf graf--p">
   政府监控摄像技术专员 Tony Porter 表示，如果不采取任何措施来控制这些新技术的使用，很快人们就只能被迫“捂嘴”才能说话了，否则无法保持户外对话的私密性。
  </p>
  <p class="graf graf--p">
   他警告说，目前限制政府使用秘密监视的严格法律框架并不适用于可以部署在公共场所的新监视技术。
  </p>
  <p class="graf graf--p">
   （所谓的严格法律框架只是要求在执行秘密监视之前获得手令，在这里看到手令是多么容易获得《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%96%e4%bb%ac%e6%98%af%e5%a6%82%e4%bd%95%e6%8a%93%e6%8d%95%e8%af%b4%e7%9c%9f%e8%af%9d%e7%9a%84%e4%ba%ba%ef%bc%9f%e2%9a%a0%ef%b8%8f%e7%81%be%e9%9a%be%e5%92%8c%e6%95%99%e8%ae%ad/" href="https://www.iyouport.org/%e4%bb%96%e4%bb%ac%e6%98%af%e5%a6%82%e4%bd%95%e6%8a%93%e6%8d%95%e8%af%b4%e7%9c%9f%e8%af%9d%e7%9a%84%e4%ba%ba%ef%bc%9f%e2%9a%a0%ef%b8%8f%e7%81%be%e9%9a%be%e5%92%8c%e6%95%99%e8%ae%ad/" rel="noopener noreferrer" target="_blank">
    他们是如何抓捕说真话的人？⚠️灾难和教训
   </a>
   》；在美国如此，在五眼其他国家更是如此）
  </p>
  <p class="graf graf--p">
   Porter 表示，新技术可以为执法和公共安全带来重要好处，但他在接受伦敦晚报采访时警告说：“保护一个自由开放的社会非常重要，我们现在应该关注技术的影响；
  </p>
  <p class="graf graf--p graf--startsWithSingleQuote">
   ‘运行唇形同步技术以确定人们说了什么话的监视能力会产生非常大的抑制作用。它会改变我们社会的本质。人们不会再觉得自己可以自由谈话’。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8973 jetpack-lazy-image" data-lazy-sizes="(max-width: 634px) 100vw, 634px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2.jpg?resize=634%2C463&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2.jpg?w=634&amp;ssl=1 634w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2.jpg?resize=300%2C219&amp;ssl=1 300w" data-recalc-dims="1" height="463" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2.jpg?resize=634%2C463&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="634"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8973" data-recalc-dims="1" height="463" sizes="(max-width: 634px) 100vw, 634px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2.jpg?resize=634%2C463&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2.jpg?w=634&amp;ssl=1 634w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2.jpg?resize=300%2C219&amp;ssl=1 300w" width="634"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   他说，这项技术将意味着一个“影响深远、无所不在的监视社会”，没有任何人能在这样的社会里获得隐私权。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “现在是自乔治·奥威尔的《1984》发布以来的第70个年头，我们必须重新听取这些些警告，并宣布我们不想要那样的社会，我们对此能做些什么。”
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/70%e5%b9%b4%e5%90%8e%ef%bc%8c%e8%bf%98%e6%98%af1984/" href="https://www.iyouport.org/70%e5%b9%b4%e5%90%8e%ef%bc%8c%e8%bf%98%e6%98%af1984/" rel="noopener noreferrer" target="_blank">
     70年后，还是1984
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   大都会最近对面部识别相机进行的试验已经受到了强烈反对，与此同时，公民权利组织 Big Brother Watch 的一项调查揭示了英国私人拥有的面部识别技术“正在像瘟疫一样蔓延”。
  </p>
  <p class="graf graf--p">
   公民权利组织在英国各个角落都找到了使用该技术的实例，主要房地产开发商、购物中心、博物馆、会议中心、和赌场。
  </p>
  <p class="graf graf--p">
   他们的调查发现，去年在英国北部最大的购物中心之一谢菲尔德的 Meadowhall 进行的现场面部识别试验，可以扫描超过200万访客的面孔。
  </p>
  <p class="graf graf--p">
   Big Brother Watch 负责人 Silkie Carlo 说：“英国有一种面部识别流行病。
   <strong class="markup--strong markup--p-strong">
    警方和私营公司在构建这些监视网络中的勾结
   </strong>
   非常令人不安。
  </p>
  <p class="graf graf--p graf--startsWithSingleQuote">
   ‘面部识别是压迫的完美工具，其广泛的使用意味着我们正在面临隐私权的紧急情况。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “面部识别监控有可能使英国的隐私权灭绝”。
  </p>
  <p class="graf graf--p">
   视频：
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="618" src="//player.vimeo.com/video/357147460?title=0&amp;byline=0&amp;portrait=0&amp;color=8dc7dc" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   面部识别软件通过将实时图像与人脸照片数据库进行匹配来实现识别。
  </p>
  <p class="graf graf--p">
   在每个人脸部的眼睛、鼻子、脸颊和嘴巴上取大约80个独特的点，这些点间的距离使得一个人可以与另一个人区别开来。
  </p>
  <p class="graf graf--p">
   监视摄像机测量人脸上各点之间的距离，例如鼻子的宽度，眼窝的深度，眼睛之间的距离和下颌的形状。
  </p>
  <p class="graf graf--p">
   由此产生了唯一的数字代码，然后可以将其与从先前照片收集的匹配代码对比以形成联系。
  </p>
  <p class="graf graf--p">
   中国官员使用的面部识别系统连接数百万台闭路电视摄像机，并使用人工智能来挑选目标。
  </p>
  <p class="graf graf--p">
   专家认为，面部识别技术将很快超越指纹识别技术，成为识别人的最有效方式。
  </p>
  <p class="graf graf--p">
   视频：
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="618" src="//player.vimeo.com/video/357147987?title=0&amp;byline=0&amp;portrait=0&amp;color=8dc7dc" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   权利组织 Liberty 对高等法院南威尔士警方使用面部识别摄像机的问题提出了挑战，正在等待裁决。
  </p>
  <p class="graf graf--p">
   但 Porter 认为，与可安装在监控摄像头上的最新技术相比，面部识别“几乎已经是昨天的事了”。
  </p>
  <p class="graf graf--p">
   CCTV监视器和穿戴式监视摄像机的复杂程度，
   <strong class="markup--strong markup--p-strong">
    越来越可能与间谍技术具有相同的功能
   </strong>
   ，但其使用并未得到适当的控制。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “对于使用秘密监视有一个非常严格的监管锁定，但是，我正在谈论的技术类型 —— 比如读唇技术、步态识别技术等 —— 这些都不在严格监管范围内。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “鉴于这些技术的侵略性程度与秘密监视一样，至少必须回顾我们在社会中是如何开始容忍这种情况存在的。”
  </p>
  <p class="graf graf--p">
   然而目前警方仍在为其使用面部识别技术的行为做自辨。⚪️
  </p>
