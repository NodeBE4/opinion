   中国的“百度-魏则西”事件发生时，很多人由此了解到了什么是搜索引擎竞价排名。但这还远远不够，不够 ……因为搜索结果排名操纵本身是一项蓬勃发展的生意。

   众所周知，你的名声是其他人对你的评价，而不是你对自己的看法。
  </p>
  <p class="graf graf--p">
   而在线状态下，名声主要取决于人们在将个人或公司的名称输入Google等搜索引擎时第一眼所看到的结果。
  </p>
  <p class="graf graf--p">
   虽然您可以仔细策划自己的社交媒体资料，但是您的 Google 足迹仍然可以提供一种生动的外观，完整的瑕疵，从破产到逮捕，再到各种骂名。
  </p>
  <p class="graf graf--p">
   中国当局的水军在5年前就使用过这种这种方法，以此抹黑他们不喜欢的异议人士，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://pao-pao.net/article/180" href="https://pao-pao.net/article/180" rel="noopener noreferrer" target="_blank">

   <strong class="markup--strong markup--p-strong">
    由于有这么一群由公关顾问和专业垃圾邮件发送者组成的“行业”，你可以花钱在谷歌这个非常重要的第一页上控制信息
   </strong>

  <p class="graf graf--p">
   声誉管理（洗地）行业的发展突出了人们在数字世界中控制个人数据和信息的愿望之间出现的紧张关系，以及平台需要防止抑制或删除信息的企图，只因为那些信息造成不便或不讨人喜欢。
  </p>
  <p class="graf graf--p">
   Andy Beal 是一本关于搜索引擎优化和在线声誉管理的书籍的作者，他告诉 BuzzFeed 新闻，大多数声誉管理项目都涉及“利用搜索引擎优化来尝试将负面页面推出谷歌[搜索结果]”。
  </p>
  <p class="graf graf--p">
   为了实现这一目标，Beal 表示他致力于“为客户创建高质量的、积极的内容，以反映他们的身份”。他说，目标是为客户提供他们合理赚取的声誉。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">

  </p>

  <p class="graf graf--p">
   Chase Rubin 的名字被三个看似虚假的在线角色所使用：摄影师、房地产开发商、和私人金融家。
  </p>
  <p class="graf graf--p">
   他的兄弟 Blake Rubin 现在的在线角色是一位假的女性旅行摄影师、网络开发人员/技术工作者、和房地产开发商。所有
   <strong class="markup--strong markup--p-strong">
    假人
   </strong>
   都声称自己住在费城。
  </p>
  <p class="graf graf--p">
   这些角色都使用在线素材图片作为他们的个人资料照片，在某些情况下，不同的人物角色之间
   <strong class="markup--strong markup--p-strong">
    甚至相互链接
   </strong>
   。

  <p class="graf graf--p">
   在一个案例中，“房地产开发商” Blake Rubin 的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.facebook.com/blake.rubin.96" href="https://www.facebook.com/blake.rubin.96" rel="noopener noreferrer" target="_blank">
    Facebook 个人资料
   </a>
   居然链接到了“科技工作者” Blake Rubin 的
   <a class="markup--anchor markup--p-anchor" data-href="https://web.archive.org/web/20190626200952/https://www.blakerubin.net/sample-page/" href="https://web.archive.org/web/20190626200952/https://www.blakerubin.net/sample-page/" rel="noopener noreferrer" target="_blank">
    网站
   </a>

  <p class="graf graf--p">
   他们的父亲 Adrian Rubin 也有多个在线人物角色。其中一个角色是自由创意总监，“他”的网站描述他 “花了30多年的时间在博爱城工作（费拉德尔菲亚市的昵称）”。
  </p>
  <p class="graf graf--p">
   他最近在由著名报纸“费城周刊”（Philadelphia Weekly）发表的一篇赞助内容中（就是软文）为自由职业者提供健康提示。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在“他”的网站上他使用了一个白人的素材图像照片；而在 Twitter上，该角色使用的是亚洲人的素材头像
   </strong>
   ；标注的身份分别为费城
   <a class="markup--anchor markup--p-anchor" data-href="https://web.archive.org/web/20190612213044/https://www.adrianrubin.info/adrian-rubin-biography/" href="https://web.archive.org/web/20190612213044/https://www.adrianrubin.info/adrian-rubin-biography/" rel="noopener noreferrer" target="_blank">
    科技工作者

   Adrian Rubin 和费城
   <a class="markup--anchor markup--p-anchor" data-href="https://web.archive.org/web/20190612213221/https://www.talkgeo.com/know-how-adrian-rubin-has-reinforced-his-business-backbone-in-realty-industry" href="https://web.archive.org/web/20190612213221/https://www.talkgeo.com/know-how-adrian-rubin-has-reinforced-his-business-backbone-in-realty-industry" rel="noopener noreferrer" target="_blank">
    房地产开发商

  <p class="graf graf--p">
   然后是 Ian Leaf / Andrews。 2005年，他因从英国纳税人手中骗取了超过5000万美元的资金而被判犯有13项欺诈罪。
  </p>
  <p class="graf graf--p">
   在被引渡到英国之前，Leaf 住在瑞士的一座豪宅里，拥有两架私人飞机，并与他的第三任妻子一位前瑞典小姐结婚。
  </p>
  <p class="graf graf--p">
   媒体密切关注他的案子，导致大量关于他的奢侈生活方式的报道出现，以及他之前从意大利步行穿越阿尔卑斯山返回瑞士逃离引渡到英国的事实。
  </p>
  <p class="graf graf--p">
   据“每日镜报”
   <a class="markup--anchor markup--p-anchor" data-href="https://www.mirror.co.uk/news/uk-news/exclusive-54million-tax-tea-leaf-577846" href="https://www.mirror.co.uk/news/uk-news/exclusive-54million-tax-tea-leaf-577846" rel="noopener noreferrer" target="_blank">
    报道
   </a>
   ，他用他的私人飞机将他的孩子送到学校，他的妻子乘坐飞机从日内瓦到伦敦只为了做头发。所有这些钱都是骗来的。
  </p>
  <p class="graf graf--p">
   在服刑后，Leaf 在2013年
   <a class="markup--anchor markup--p-anchor" data-href="https://www.irishexaminer.com/lifestyle/features/humaninterest/return-of-the-playboy-fraudster-222847.html" href="https://www.irishexaminer.com/lifestyle/features/humaninterest/return-of-the-playboy-fraudster-222847.html" rel="noopener noreferrer" target="_blank">
    公开报道
   </a>
   中声称，他已将自己的名字改为 Ian Andrews，并仍然参与他的一家前公司 Home Funding Corporation（HFC）。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    早在2015年，使用他的两个名字的在线角色就开始在网站上发布内容
   </strong>
   ，例如ianleafreviews.com，ianandrewsfraudster.com 和 ianleaf.com。
  </p>
  <p class="graf graf--p">
   账户设置在
   <a class="markup--anchor markup--p-anchor" data-href="http://archive.is/CmmHK" href="https://archive.fo/CmmHK" rel="noopener noreferrer" target="_blank">
    Medium
   </a>
   、
   <a class="markup--anchor markup--p-anchor" data-href="https://web.archive.org/web/20190613141427/https://twitter.com/ianleafstrategy/" href="https://web.archive.org/web/20190613141427/https://twitter.com/ianleafstrategy/" rel="noopener noreferrer" target="_blank">
    Twitter
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.linkedin.com/in/ianleaf/" href="https://www.linkedin.com/in/ianleaf/" rel="noopener noreferrer" target="_blank">
    LinkedIn
   </a>
   上。
  </p>
  <p class="graf graf--p">
   在线的 Ian Leaf 角色将自己称为“欺诈观察者”（Fraud Watcher），自称是一名致力于 “防止每个行业的企业成为欺骗行为牺牲品”的顾问。
  </p>
  <p class="graf graf--p">
   该角色甚至还在亚马逊上进行了
   <a class="markup--anchor markup--p-anchor" data-href="https://web.archive.org/web/20190613145451/https://ideamensch.com/ian-leaf/" href="https://web.archive.org/web/20190613145451/https://ideamensch.com/ian-leaf/" rel="noopener noreferrer" target="_blank">
    采访
   </a>
   并出版了
   <a class="markup--anchor markup--p-anchor" data-href="https://web.archive.org/web/20190614184227/https://www.amazon.com/Ian-Leaf/e/B016P1H3E8/ref=dp_byline_cont_ebooks_1" href="https://web.archive.org/web/20190614184227/https://www.amazon.com/Ian-Leaf/e/B016P1H3E8/ref=dp_byline_cont_ebooks_1" rel="noopener noreferrer" target="_blank">
    书籍
   </a>
   ，有时将 Ian Andrews 列为
   <a class="markup--anchor markup--p-anchor" data-href="https://web.archive.org/web/20190614184242/https://www.amazon.com/Ian-Andrews/e/B01AHRRGPC/ref=dp_byline_cont_ebooks_2" href="https://web.archive.org/web/20190614184242/https://www.amazon.com/Ian-Andrews/e/B01AHRRGPC/ref=dp_byline_cont_ebooks_2" rel="noopener noreferrer" target="_blank">
    合著者
   </a>
   。
  </p>
  <p class="graf graf--p">
   ……
  </p>
  <p class="graf graf--p">
   上述任何一种做法违反了职业道德规范吗？并没有。因为，正如与 BuzzFeed News 交谈的声誉管理顾问所言，
   <strong class="markup--strong markup--p-strong">
    这是一个没有规则或商定标准的行业
   </strong>
   。道德纯粹是个人意见的问题。
  </p>
  <p class="graf graf--p">
   搜索引擎优化专家 Beal 称，假冒人物角色和奖学金等策略从未支付过行业中出现的“黑帽、不道德技术”的例子。
  </p>
  <p class="graf graf--p">
   Link Builders 的 Cuttonaro 承认为至少一位客户创造了在线角色，但表示这是由于“特殊情况”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “特殊”指的是，该客户表示他们正面临与过去涉及为美国政府工作相关的死亡威胁。 Cuttonaro 被要求制定一个策略，以他们的名字混淆搜索结果。
  </p>
  <p class="graf graf--p">
   （一位不愿透露姓名的联邦调查局特工告诉 BuzzFeed News，有关人员面临可信的持续威胁。因此他们需要在报道中匿名）
  </p>
  <p class="graf graf--p">
   虽然 Cuttonaro 说这种情况是一个例外并且与国家安全有关，但是，他还利用其中一个为假人物角色创建的网站为其他客户放置文章，并推广男性性功能增强补品。他的公司还为假人形成了奖学金，并成功将其添加到大学网站。
  </p>
  <p class="graf graf--p">
   Cuttonaro 最初表示，他不确定奖学金是否已经支付，或者他的公司是否参与了其创建。随后他又说自己想起来了曾经参与其中，并声明“我们从来没有得到任何申请……我不记得有过”。因此，从未付过钱，他说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我告诉你，我是一个好人，”他说。
  </p>
  <p class="graf graf--p">
   把坏人说成好人可能很困难，
   <strong class="markup--strong markup--p-strong">
    Rubin 的人物角色是所谓的声誉管理行业无法无天之性质的完美典范
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Hopkins 已有十多年的经营历史，他的 LinkedIn 个人资料显示他目前正在攻读硕士学位。
  </p>
  <p class="graf graf--p">
   他称，创造虚假的在线角色以影响搜索结果是“非常正常的事”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “一般来说，这就是客户要求的东西，”他说。
  </p>
  <p class="graf graf--p">
   但当被问及他自己关于 Rubins 和 Ian Leaf 的在线角色的工作时，Hopkins 不会承认任何参与。
  </p>
  <p class="graf graf--p">
   SSL证书、域名注册以及 BuzzFeed News 发现的其他信息都显示，Hopkins 与可疑的 Rubin 和 Ian Leaf / Andrews 角色相关。
  </p>
  <p class="graf graf--p">
   他的公司 AfterHim Media 被列为 Leaf 自己出版的两本书的虚构出版商。用于 Adrian、Chase 和 Blake Rubin 奖学金申请的 Google 表格也是由 AfterHim Media 创建的。
  </p>
  <p class="graf graf--p">
   另一个相关证据是，adrianrubinscholarship.com 的SSL证书也被 popkins.info 和 douglaspitassi.info 使用，这两个域名都由Hopkins拥有。
  </p>
  <p class="graf graf--p">
   但在电话采访中 Hopkins 拒绝承认和 Rubin 角色有任何关系。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “Adrian Rubin 是谁？” 美国地区法官 Eduardo Robreno 去年八月在量刑听证会上问道。
  </p>
  <p class="graf graf--p">
   Rubin 承认犯有一项串谋违反“诈骗影响和腐败组织法”的罪名，一项串谋罪和两项邮件欺诈罪。

   但法官不确定在他面前的懊悔是否是真实的。 1997年，Rubin 因逃税被判处一年徒刑后被释放。然而此后他继续犯罪。
  </p>

   就这么一个在现实中的真人都是假的的家伙（真绕嘴）还有一堆在线的假声誉炒作……真是很令人无语。
