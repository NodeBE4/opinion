 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      战争罪是可怕的、非人的，但其幕后的推手更可怕，那家伙是谁？……
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   在大赦国际组织调查进一步暗示海湾国家利用西方武器犯下战争罪行后，阿联酋继续与五角大楼的铁腕雷神公司签署了高达数十亿美元的新武器合同，从而加深了人道主义危机，在饱受战争蹂躏的也门加剧了大屠杀的威胁。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “对也门平民的持续屠杀 — — 包括沙特阿拉伯和阿联酋领导的联盟以及它所支持的民兵 — — 应该给所有供应武器的国家以严重警告，”军备控制和人权研究员 Patrick Wilcken 说。“阿联酋部队从西方国家和其他国家获得了价值数十亿美元的武器，只是为了供应也门的民兵，而这些民兵据信正在犯下战争罪。”
  </p>
  <p class="graf graf--p">
   呼吁停止向沙特和阿联酋领导的联盟供应武器的人权组织也注意到，在2015年爆发的冲突中“阿联酋已经加强了地面攻势”，并且，根据一些估计，杀死了6万多名也门人，数百万人流离失所，并处于饥荒的边缘。
  </p>
  <p class="graf graf--p">
   正如“纽约时报”最近报道的那样，“美国也参与了也门的空中战争”，并指出该国的主要武器销售和情报来源是沙特。考虑到各方在冲突期间犯下的战争罪行，平民伤亡人数增加、以及基础设施遭受破坏，希望有新立法阻止美国对战争的支持。
  </p>
  <p class="graf graf--p">
   不过，雷神国际首席执行官约翰·哈里斯（John Harris）已经完全无视该行业所面临的批评，他对 CNBC 说，“我们不制定政策。”
  </p>
  <p class="graf graf--p">
   因此，利润丰厚的交易仍在继续。
  </p>
  <p class="graf graf--p">
   据官方媒体报道，阿联酋宣布已签署了价值超过16亿美元的交易，用于购买美国军火巨头雷神公司的爱国者导弹发射器。
  </p>
  <p class="graf graf--p">
   在阿布扎比举行的国际军事展览的第二天，这笔交易是在阿联酋宣布将从雷神公司购买价值3.53亿美元的爱国者导弹后的第二天发布的。
  </p>
  <p class="graf graf--p">
   据国营的WAM通讯社报道，这个石油丰富的国家和西方公司在世博会期间签署了约32亿美元的交易。
  </p>
  <p class="graf graf--p">
   虽然中国、美国、英国、法国和其他欧洲国家都因为向联军提供武器而受到批评，但新的威胁正在出现。随着地面战争的演变，武器不仅被阿联酋部队在也门使用，而且还被传递给完全不负责任的联军民兵，其中一些人被指控犯有战争罪。
  </p>
  <p class="graf graf--p">
   该报告还引用了之前的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amnestyusa.org/wp-content/uploads/2018/07/%E2%80%98GOD-ONLY-KNOWS-IF-HE%E2%80%99S-ALIVE%E2%80%99-MDE-31.8682.2018.pdf" href="https://www.amnestyusa.org/wp-content/uploads/2018/07/%E2%80%98GOD-ONLY-KNOWS-IF-HE%E2%80%99S-ALIVE%E2%80%99-MDE-31.8682.2018.pdf" rel="noopener noreferrer" target="_blank">
    出版物（pdf）
   </a>
   ，揭示阿联酋正在训练也门民兵，其中发生了“一系列令人震惊的滥用行为”。
  </p>
  <p class="graf graf--p">
   由众议员罗·卡纳（D-Calif）提出的决议希望能阻止美国的参与，他在投票后称“我们比以往任何时候都更接近于结束我们在这场人道主义灾难中的共谋”。他引用了美国有线电视新闻网最近的一项调查，显示沙特据信向也门的基地组织部队提供美国产的武器。他说，“我们必须停止出售武器，停止帮助沙特和阿联酋继续使这场灾难性的战争永久化”。
  </p>
