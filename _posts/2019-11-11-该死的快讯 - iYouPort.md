   Breaking news, fake news 和一切病态的东西（不论现在你叫它什么）。
  </p>
  <p class="graf graf--p">
   在今天的超级连接世界中，每天、每时每刻都有一个看似无止境的庞大信息生态覆盖着每个人 —— 真实的和虚假的，混杂在一起。
  </p>
  <p class="graf graf--p">
   但是，这一切并非刚刚成为噪音。
  </p>
  <p class="graf graf--p">

    527,000张
   </a>
   照片分享到 Snapchat，在这样的一个世界中，还能有什么噪音令人震惊？
  </p>
  <p class="graf graf--p">
   新闻报道在这场眼球争夺战中的丑态真的很难以理解吗？

  <p class="graf graf--p">
   在所有这些丑态百出的比赛中，你唯一可以感觉到的就是，新闻议程在你有机会消费它之前就已经消失了；
   <strong class="markup--strong markup--p-strong">
    你无休止地追逐着那些风暴，但永远不会捕捉到任何东西
   </strong>
   。
  </p>
  <p class="graf graf--p">
   正如我们曾经分析过那样，新闻业正处一个泥泞的岔路口 —— 主流媒体大量裁员（如 BuzzFeed 和 Huffington Post 等），其他媒体悲惨消亡（如 The Pool 等），死的死伤的伤，这已经不是丑态能描述的境况了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这些“战败”者的共同特点都是，主要依靠广告和赞助来创造收入
   </strong>
   （尽管 The Pool 确实有一个’溢价’
   <a class="markup--anchor markup--p-anchor" data-href="https://mediatel.co.uk/newsline/2017/07/25/the-pool-to-launch-3-vip-subscription/?source=post_page---------------------------" href="https://mediatel.co.uk/newsline/2017/07/25/the-pool-to-launch-3-vip-subscription/?source=post_page---------------------------" rel="noopener noreferrer" target="_blank">
    付费通讯
   </a>
   ）。
  </p>
  <p class="graf graf--p">
   为什么依靠广告的你注定失败？详见解读《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bc%82%e8%ae%ae%e7%9a%84%e6%b6%88%e4%ba%a1%ef%bc%9a%e4%b8%ba%e4%bb%80%e4%b9%88%e7%bd%91%e7%bb%9c%e5%8f%98%e5%be%97%e5%90%8c%e8%b4%a8%e5%8c%96/" href="https://www.iyouport.org/%e5%bc%82%e8%ae%ae%e7%9a%84%e6%b6%88%e4%ba%a1%ef%bc%9a%e4%b8%ba%e4%bb%80%e4%b9%88%e7%bd%91%e7%bb%9c%e5%8f%98%e5%be%97%e5%90%8c%e8%b4%a8%e5%8c%96/" rel="noopener noreferrer" target="_blank">
    异议的消亡
   </a>
   》。
  </p>
  <p class="graf graf--p">
   当人们越来越愿意为他们喜欢的高质量新闻付费时，这也变成了一种奇怪的策略 — 而且往往也是发生在网上。
  </p>

  <p class="graf graf--p">
   事实上，目前的严峻状况的确前所未有。
  </p>
  <p class="graf graf--p">
   最近去世的受人尊敬的体育记者休·麦克尔万尼（Hugh McIlvanney）在他人生中最后一篇“星期日泰晤士报”专栏文章中哀叹到：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “我羡慕现代体育记者的年轻，但我不羡慕他们的经营状况。”
   </em>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    人们是否都厌倦了追逐即时新闻快讯风暴的疲惫？相反，是否还需要深思熟虑的新闻业？人们会愿意为此付费吗？
   </strong>
  </p>
  <p class="graf graf--p">
   如果答案是肯定的，那么慢新闻必需重新回归。
  </p>
  <ul class="postList">
   <li class="graf graf--li">

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%bf%98%e8%ae%b0%e5%81%87%e6%96%b0%e9%97%bb%e5%90%a7%ef%bc%8c%e6%a0%b9%e6%9c%ac%e6%80%a7%e9%97%ae%e9%a2%98%e5%9c%a8%e4%ba%8e%e7%9c%9f%e6%96%b0%e9%97%bb%e6%9c%ac%e8%ba%ab/" href="https://www.iyouport.org/%e5%bf%98%e8%ae%b0%e5%81%87%e6%96%b0%e9%97%bb%e5%90%a7%ef%bc%8c%e6%a0%b9%e6%9c%ac%e6%80%a7%e9%97%ae%e9%a2%98%e5%9c%a8%e4%ba%8e%e7%9c%9f%e6%96%b0%e9%97%bb%e6%9c%ac%e8%ba%ab/" rel="noopener noreferrer" target="_blank">

    </a>

   </li>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    什么是慢新闻？
   </strong>
  </p>
  <p class="graf graf--p">
   慢新闻是一种让作者有空间和时间去“徘徊”并触及故事的核心的做法，通常是在新闻议程开始之后。
  </p>
  <p class="graf graf--p">
   这并不是什么新鲜事 —— 过去几年中长篇报道的流行度增长基本上是持平的，尽管它们通常是对更传统的新闻做法的补充（参见卫报的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/news/series/the-long-read?source=post_page---------------------------" href="https://www.theguardian.com/news/series/the-long-read?source=post_page---------------------------" rel="noopener noreferrer" target="_blank">
    长篇部分
   </a>
   ）。
  </p>
  <p class="graf graf--p">
   现在我们看到专属网点的增加只会减缓新闻业的发展。
  </p>
  <p class="graf graf--p">
   也许最突出的是英国的 Delayed Gratification（DG）杂志。这是一本高质量的季刊，
   <strong class="markup--strong markup--p-strong">
    以深刻的分析重新审视新闻
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “我们不是在试图与传统新闻媒体竞争，” 该杂志编辑 Matthew Lee 在接受在线采访时表示。
   </em>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “慢新闻希望能激起读者的兴趣，并提供他们以前从未听过的新闻报道的观点。
   </em>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “但我们的主要目标是创造一本漂亮的、信息丰富的且有趣的杂志。”
   </em>
  </p>
  <p class="graf graf--p">
   自2011年以来，DG 一直活跃，并且已经 — 慢慢地、适当地 — 建立了24,000个打印版读者群，包括5,000名订阅者，截至去年3月。
  </p>
  <p class="graf graf--p">
   今年又出现了另一个慢新闻品牌：Tortoise。它将运作一个会员模型，旨在让人们
   <strong class="markup--strong markup--p-strong">
    真正了解事件的全貌而不会感觉到庞大信息量的压力
   </strong>
   （其座右铭是’减速、明智’）。
  </p>
  <p class="graf graf--p">
   而且它的编辑团队有一大批重要的实力派合作者。
  </p>
  <p class="graf graf--p">
   这是英国广播公司前新闻和“时报”编辑 Matthew Harding 的心血结晶，
   <strong class="markup--strong markup--p-strong">
    他还挖掘到了一大批顶级人才
   </strong>
   ，包括新闻之夜的前政策编辑 Chris Cook，加入他的旗舰公司。
  </p>
  <p class="graf graf--p">
   Tortoise 方法包括举办名为 ThinkIn 的新闻发布会，编辑团队在这里讨论他们将要调查的内容。
  </p>
  <p class="graf graf--p">
   最重要的是，邀请 Tortoise 成员参加这些会议并分享他们对正在讨论的故事和问题的看法。
  </p>
  <p class="graf graf--p">
   然后，这些内容会登陆每日一次的新闻摘要，每天早上11:00分享五到七个简明文章。
  </p>
  <p class="graf graf--p">
   该公司还计划出版一本季刊专门收集长篇报道 —— 精心报道的调查性新闻作品。即 “新闻背后的驱动力”。
  </p>
  <p class="graf graf--p">
   以上两个例子的共同点是希望深入地重新审视新闻故事 —— 根本不会考虑什么新闻周期（互联网新闻周期普遍只有2小时，最长不超过2天）。
  </p>
  <p class="graf graf--p">
   两者采取的都是订户/会员模型，都在避免任何形式的广告。
  </p>
  <p class="graf graf--p">
   显然有一个背景：Tortoise 在 Kickstarter 上获得了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.kickstarter.com/projects/tortoisemedia/tortoise-open-journalism-a-different-kind-of-newsr?source=post_page---------------------------" href="https://www.kickstarter.com/projects/tortoisemedia/tortoise-open-journalism-a-different-kind-of-newsr?source=post_page---------------------------" rel="noopener noreferrer" target="_blank">
    539,035英镑
   </a>
   的支持，而去年年底的目标是75,000英镑。看到会员数量在4月份正式推出后如何增长将会很有趣。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-6836 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/慢.jpg?resize=1100%2C792&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/慢.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/慢.jpg?resize=300%2C216&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/慢.jpg?resize=768%2C553&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/慢.jpg?resize=1024%2C738&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/慢.jpg?resize=1100%2C792&amp;ssl=1 1100w" data-recalc-dims="1" height="792" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/慢.jpg?resize=1100%2C792&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6836" data-recalc-dims="1" height="792" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/慢.jpg?resize=1100%2C792&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/慢.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/慢.jpg?resize=300%2C216&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/慢.jpg?resize=768%2C553&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/慢.jpg?resize=1024%2C738&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/慢.jpg?resize=1100%2C792&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    慢新闻的示例
   </strong>
  </p>
  <p class="graf graf--p">
   上面的两个案例并不是那些热衷于慢新闻的人的唯一选择。
  </p>
  <p class="graf graf--p">
   以下是目前活跃的慢速新闻运动参与者的部分列表：
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.slow-journalism.com/?source=post_page---------------------------" href="https://www.slow-journalism.com/?source=post_page---------------------------" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Delayed Gratification, UK
    </strong>
   </a>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.tortoisemedia.com/?source=post_page---------------------------" href="https://www.tortoisemedia.com/?source=post_page---------------------------" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Tortoise, UK
    </strong>
   </a>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.propublica.org/?source=post_page---------------------------" href="https://www.propublica.org/?source=post_page---------------------------" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     ProPublica, USA
    </strong>
   </a>
  </p>
  <p class="graf graf--p">
   ProPublica 是一个独立的非营利性新闻编辑室，以道德力量制作调查性新闻。
  </p>
  <p class="graf graf--p">
   它深入挖掘重要问题，揭示权力的滥用和公众信任的背叛。
  </p>
  <p class="graf graf--p">
   这是在2010年第一次赢得普利策奖的在线新闻来源 —— 这个故事记录了一家医院医生在被卡特里娜飓风洪水袭击时做出的紧急生死决定。自那以后，该平台再次获得了两次优质新闻奖项。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://magazine.atavist.com/?source=post_page---------------------------" href="https://magazine.atavist.com/?source=post_page---------------------------" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     The Atavist, USA
    </strong>
   </a>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “Atavist” 杂志每个月都会发表一篇长篇非虚构报道。旨在成为具有深度报道功能的电影版真实故事来源。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.longplay.fi/?source=post_page---------------------------" href="https://www.longplay.fi/?source=post_page---------------------------" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Long Play, Finland
    </strong>
   </a>
  </p>
  <p class="graf graf--p">
   2013年，八位芬兰自由撰稿人发起了 Long Play。
  </p>
  <p class="graf graf--p">
   这是一项旨在为严肃、高质量的新闻创建读者资助的出版模式的实验。
  </p>
  <p class="graf graf--p">
   其理念是：“生活中的重要问题很少有简单的答案。如果我们要真正了解当今的重大问题 —— 气候变化、健康和技术伦理、恐怖主义、种族主义 —— 我们的记者需要花费大量时间进行研究。”
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.slow-news.com/?source=post_page---------------------------" href="https://www.slow-news.com/?source=post_page---------------------------" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Slow News, Italy
    </strong>
   </a>
  </p>
  <p class="graf graf--p">
   Slow News 制作了两个专门的新闻通讯 —— Flow 和 Wolf —— 订阅者可以付费使用。
  </p>
  <p class="graf graf--p">
   订阅每月只需花费2欧元，Flow 每周出两次。
  </p>
  <p class="graf graf--p">
   每周三，它包含了由 Slow News 的编辑人员策划的五篇内容 —— 包括文章、故事、照片、视频。
  </p>
  <p class="graf graf--p">
   每星期日，它包含两个内容和一个未发表的文章框架，由出版商及其合作者设计和制作。
  </p>
  <p class="graf graf--p">
   与此同时，Wolf 提供有关通信、营销和数字新闻领域的报道和分析。
  </p>
  <p class="graf graf--p">
   由 Alberto Puliafito、Mafe de Baggis 和 Filippo Pretolani 编辑，每周出三次，每月收费10欧元。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.zetland.dk/?source=post_page---------------------------" href="https://www.zetland.dk/?source=post_page---------------------------" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Zetland, Denmark
    </strong>
   </a>
  </p>
  <p class="graf graf--p">
   Zetland 每天发布三到四篇新闻文章，重点关注长篇报道和深度文章。
  </p>
  <p class="graf graf--p">
   它是一个基于订阅的组织，截至2017年7月拥有超过8,500名订阅者。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    慢下来
   </strong>
  </p>
  <p class="graf graf--p">
   就如我们在《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e7%9f%a5%e9%81%93%e4%bb%80%e4%b9%88%e6%89%8d%e6%98%af%e7%9c%9f%e6%ad%a3%e7%9a%84%e8%b0%83%e6%9f%a5%e5%90%97%ef%bc%9f%e6%89%80%e8%b0%93%e7%9a%84%e8%b5%b7%e5%ba%95%e7%bb%8f%e5%b8%b8/" href="https://www.iyouport.org/%e4%bd%a0%e7%9f%a5%e9%81%93%e4%bb%80%e4%b9%88%e6%89%8d%e6%98%af%e7%9c%9f%e6%ad%a3%e7%9a%84%e8%b0%83%e6%9f%a5%e5%90%97%ef%bc%9f%e6%89%80%e8%b0%93%e7%9a%84%e8%b5%b7%e5%ba%95%e7%bb%8f%e5%b8%b8/" rel="noopener noreferrer" target="_blank">
    什么是调查
   </a>
   》中所强调的，
   <strong class="markup--strong markup--p-strong">
    快速的一切都不存在深刻性和全面性。调查性新闻永远不会快速。这也正是它的价值所在
   </strong>
   。
  </p>
  <p class="graf graf--p">
   慢新闻不是创新，而是回归，它就是新闻本来的样子；慢新闻从业者往往会强调不与快讯争夺市场，但这不是事实，正相反，只有成功地争夺市场才能让新闻业重回民主支柱的位置。
  </p>
  <p class="graf graf--p">
   以下这四个词正是现代快速新闻不再有意义的根本性原因：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    轰动的：如果你没能获得媒体认为当有的轰动效应，你就不会被关心；哪怕你的遭遇对他人来说具有确切的警示性效应；
   </li>
   <li class="graf graf--li">
    特殊的：意味着第一个人遭到勒索软件攻击时它值得报道的 ，但勒索软件泛滥时，大批的受害者都会被媒体无视，因为他们“不再特殊”。换个角度看，如果你作恶一次，肯定会被揭露；如果你持续地广泛地作恶 …… 很可能就被无视了。
   </li>
   <li class="graf graf--li">
    否定性的：你有没有注意到准确的分析（如果有的话）永远都是被报道的错误行为的陪衬？抗议者被关注的时候永远都是他们被捕的时候、甚至被杀？在他们真正成功的时候却没人知道他们……
   </li>
   <li class="graf graf--li">
    时效性的：这点无须解释。99%的新闻不会让你在第二天还想看它，剩下的1%不会让你在一周后还觉得它值得说两句。一切都是走马观花的，难以推动对现实的改变。
   </li>
  </ul>
  <p class="graf graf--p">
   新闻的问题并不在于虚假与否，而在于
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bf%98%e8%ae%b0%e5%81%87%e6%96%b0%e9%97%bb%e5%90%a7%ef%bc%8c%e6%a0%b9%e6%9c%ac%e6%80%a7%e9%97%ae%e9%a2%98%e5%9c%a8%e4%ba%8e%e7%9c%9f%e6%96%b0%e9%97%bb%e6%9c%ac%e8%ba%ab/" href="https://www.iyouport.org/%e5%bf%98%e8%ae%b0%e5%81%87%e6%96%b0%e9%97%bb%e5%90%a7%ef%bc%8c%e6%a0%b9%e6%9c%ac%e6%80%a7%e9%97%ae%e9%a2%98%e5%9c%a8%e4%ba%8e%e7%9c%9f%e6%96%b0%e9%97%bb%e6%9c%ac%e8%ba%ab/" rel="noopener noreferrer" target="_blank">
    该行业本身
   </a>
   。
  </p>
  <p class="graf graf--p">
   以上慢新闻的成功案例无疑是对现代新闻的一项重大改革尝试，它们的成功是令人钦佩的。但是，这只是一个开始。
  </p>
  <p class="graf graf--p">
   战胜快餐文化的根深蒂固并非易事。当读者抱怨我们的调查报告“太长、没法看”的时候，我们应该思考的不只有改革模式问题。⚪️
  </p>
