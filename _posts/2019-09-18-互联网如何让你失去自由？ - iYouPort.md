  <p class="graf graf--p">
   我们目前的年龄是由互联网定义的。
  </p>
  <p class="graf graf--p">
   这个互联网在人类生活中已经变得如此根深蒂固，以至于获取上网的权利被认为
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wired.com/2011/06/internet-a-human-right/" href="https://www.wired.com/2011/06/internet-a-human-right/" rel="noopener noreferrer" target="_blank">
    是一项基本人权
   </a>
   。
  </p>
  <p class="graf graf--p">
   作为创新的媒介，它为我们提供了诸如 Ub​​er Eats 和 Tinder 这样多样化的美好事物，更不用说促进工业和学术界的巨大飞跃了。
  </p>
  <p class="graf graf--p">
   然而，就像任何足够复杂的技术一样，互联网已经创造了一些严重且无法预见的外部性。
  </p>
  <p class="graf graf--p">
   其中最为紧迫的就是人类的自由潜在且不可逆转的下降 —— 互联网目前正在促进普通人生活能力的下降，这种能力本可以使其免于受到政府或企业的干预。
  </p>
  <p class="graf graf--p">
   这不是抽象的危言耸听 —— 互联网的社会重要性与对其使用方式施加限制所必需的政治意愿和技术并行增加。
  </p>
  <p class="graf graf--p">
   全球范围内对互联网的使用都存在着具体的局限性，西方国家很少有人完全理解这种情况有可能变得多么可怕。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    自由、宽松的定义，是指以你选择的方式行事的能力，没有强迫或干涉 —— 无论是来自国家的还是来自你的同胞的强迫或干涉，你可以做得越多，你就越自由
   </strong>
   。
  </p>
  <p class="graf graf--p">
   大多数人都支持言论自由，特别因为是言论自由有很多社会效用，但至少，你需要能知道经营一个政治博客不会导致你在劳教所里度过余生或者在被爆头中结束生命。
  </p>
  <p class="graf graf--p">
   不幸的是，根据2018年自由之家的年度报告，由于各种原因，包括内战、饥荒、不民主政权的崛起和根深蒂固的体制腐败，个人自由在全球范围内均受到了严重的压力。
  </p>
  <p class="graf graf--p">
   在这里看到上述报告的
   <strong class="markup--strong markup--p-strong">
    中文增补版
   </strong>
   《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E6%95%B0%E5%AD%97%E5%A8%81%E6%9D%83%E4%B8%BB%E4%B9%89%E7%9A%84%E5%85%B4%E8%B5%B7-%E7%81%BE%E5%B9%B4%E4%B9%8B%E7%95%94/" href="https://www.iyouport.org/%E6%95%B0%E5%AD%97%E5%A8%81%E6%9D%83%E4%B8%BB%E4%B9%89%E7%9A%84%E5%85%B4%E8%B5%B7-%E7%81%BE%E5%B9%B4%E4%B9%8B%E7%95%94/" rel="noopener noreferrer" target="_blank">
    数字威权主义的兴起: 灾年之畔
   </a>
   》。
  </p>
  <p class="graf graf--p">
   这不是巧合：
   <strong class="markup--strong markup--p-strong">
    信息和数据通过互联网传播的性质本身就有助于加快审查工具的发展和对某些信息的压制，以便为政治目标服务
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在这看到具体解释《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e6%8a%80%e6%9c%af%e6%9c%89%e5%88%a9%e4%ba%8e%e6%9a%b4%e6%94%bf%ef%bc%9f%e2%80%8a-%e2%80%8a-%e4%b8%80%e5%9c%ba%e4%bd%a0%e5%bf%85%e9%a1%bb%e5%8f%82%e4%b8%8e/" href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e6%8a%80%e6%9c%af%e6%9c%89%e5%88%a9%e4%ba%8e%e6%9a%b4%e6%94%bf%ef%bc%9f%e2%80%8a-%e2%80%8a-%e4%b8%80%e5%9c%ba%e4%bd%a0%e5%bf%85%e9%a1%bb%e5%8f%82%e4%b8%8e/" rel="noopener noreferrer" target="_blank">
    为什么技术有利于暴政？ — — 一场你必须参与的政治斗争
   </a>
   》。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4404 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/头-1.jpg?resize=1100%2C618&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/头-1.jpg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/头-1.jpg?resize=300%2C169&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/头-1.jpg?resize=768%2C431&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/头-1.jpg?resize=1024%2C575&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/头-1.jpg?resize=1100%2C618&amp;ssl=1 1100w" data-recalc-dims="1" height="618" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/头-1.jpg?resize=1100%2C618&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  </p>
  <p class="graf graf--p">
   任何对俄罗斯或中国政治有粗略了解的人都知道，持久的国家审查是一种隐含的政权目标，表面上是为了确保社会稳定。
  </p>
  <p class="graf graf--p">
   最引人注目的例子是中国的大防火墙：一种非常复杂的软件和人工审查的融合，与国家媒体协同运作，几乎可以完全控制中国公民可获得的信息的基调和类型。
  </p>
  <p class="graf graf--p">
   俄罗斯则通过将其所有互联网流量重定向到内部路由来实现类似的概念。
  </p>
  <p class="graf graf--p">
   虽然网络弹性是一个共同的安全目标，但所有外部流量（即俄罗斯与外部世界之间流动的信息）
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2019/feb/12/great-firewall-fears-as-russia-plans-to-cut-itself-off-from-internet" href="https://www.theguardian.com/world/2019/feb/12/great-firewall-fears-as-russia-plans-to-cut-itself-off-from-internet" rel="noopener noreferrer" target="_blank">
    必须通过政府控制的检查点
   </a>
   这一事实，
   <strong class="markup--strong markup--p-strong">
    几乎肯定是一种将被用于情报目的的机制
   </strong>
   —— 俄罗斯国家政权将能够以网络安全为借口拒绝接受非国家批准的任何材料。（更多详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%81%8f%e6%89%a7%e7%9a%84%e4%bf%9d%e5%af%86-%e6%ac%ba%e9%aa%97%e7%9a%84%e4%b8%bb%e8%a6%81%e6%96%b9%e6%b3%95%e4%b9%8b%e4%b8%80%ef%bc%884%ef%bc%89/" href="https://www.iyouport.org/%e5%81%8f%e6%89%a7%e7%9a%84%e4%bf%9d%e5%af%86-%e6%ac%ba%e9%aa%97%e7%9a%84%e4%b8%bb%e8%a6%81%e6%96%b9%e6%b3%95%e4%b9%8b%e4%b8%80%ef%bc%884%ef%bc%89/" rel="noopener noreferrer" target="_blank">
    战略性欺骗的主要方法
   </a>
   》）
  </p>
  <p class="graf graf--p">
   您可能想知道如果俄罗斯设法制造了比中国更具限制性的防火墙怎么办？没办法。只能确认这些是专制的、不自由的政权。
  </p>
  <p class="graf graf--p">
   然而，虽然许多压迫性政权过去曾试图制定全面的审查机制，但互联网审查却是一个完全不同的野兽。
  </p>
  <p class="graf graf--p">
   人类历史上第一次，通信将接近奇点，其中几乎所有非面对面的信息传输都将通过IP（互联网协议）发生。
  </p>
  <p class="graf graf--p">
   随着通信变得越来越集中，在技术上可以做到监视和控制几乎所有通信。
  </p>
  <p class="graf graf--p">
   根本问题是：如果您的政府无权（或不应该有权）监控您的物理通信，那么它在多大程度上有权监控您的数字通信？
  </p>
  <p class="graf graf--p">
   并且，对于在其宪法中提供言论自由的国家：在多大程度上可以在互联网上限制宪法允许的言论或内容？
  </p>
  <p class="graf graf--p">
   也许有人会争辩说，中国和俄罗斯的公民仍然可以表达对政权的不满，并讨论被禁止的政治话题，或者有效地选择退出受到国家或大公司审查的渠道。的确是。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    这未能解决问题的核心：当没有人能听到你说的话时，你真正拥有多少言论自由？
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    由于绝大多数有影响力的沟通都是以数字方式进行的，当您被排除在使用构成社会语话基础的任何数字平台之外时，您还有多少言论自由？
   </strong>
  </p>
  <p class="graf graf--p">
   使用代理绕过封锁、或者只能在加密通信中匿名发言，只是安全措施，并不是言论自由。就如下面这个故事所讽刺的：那个亭子不仅不是言论自由，而且是对言论自由的讽刺。
  </p>

   </noscript>
  </p>
  <p class="graf graf--p">
   公开反科技的西奥多·卡钦斯基（Theodore Kaczynski，又称 Unabomber）做出了有先见之明的观察，即*选择退出*论证在社会层面是错误的：在实践中，
   <strong class="markup--strong markup--p-strong">
    当采用该技术成为社会规范时，个人是否有权选择退出任何特定技术？
   </strong>
  </p>
  <p class="graf graf--p">
   就如同，每个人都可以自由地拒绝购买汽车，但交通基础设施严重偏向于适合汽车的道路建设和维护。
  </p>
  <p class="graf graf--p">
   选择不开车会就让你在到处是汽车的状态下处于极大的实际劣势。
  </p>
  <p class="graf graf--p">
   互联网是未来的基础设施 —— 在某种程度上，这个世界将不再可能避免使用互联网技术，并且，通过扩展、内容和语话控制的机制，即便是笨拙的和技术上文盲的国家也可以顺利使用。
  </p>
  <p class="graf graf--p">
   一段时间以来，美国一直在讨论相似的性质，特别是关于私人社交媒体拒绝尊重那些涉嫌违反其内部政策或指导方针的人的权利。（在这里看到一个案例《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E5%A6%82%E6%9E%9C%E5%85%A8%E7%BD%91%E5%B0%81%E9%94%81%E5%91%A8%E5%B0%8F%E5%B9%B3%EF%BC%9F%E4%B8%80%E4%B8%AA%E8%88%86%E8%AE%BA%E7%83%AD%E7%82%B9%E6%88%96%E8%AE%B8%E8%83%BD%E5%B8%AE%E5%8A%A9%E4%B8%AD/" href="https://www.iyouport.org/%E5%A6%82%E6%9E%9C%E5%85%A8%E7%BD%91%E5%B0%81%E9%94%81%E5%91%A8%E5%B0%8F%E5%B9%B3%EF%BC%9F%E4%B8%80%E4%B8%AA%E8%88%86%E8%AE%BA%E7%83%AD%E7%82%B9%E6%88%96%E8%AE%B8%E8%83%BD%E5%B8%AE%E5%8A%A9%E4%B8%AD/" rel="noopener noreferrer" target="_blank">
    如果全网封锁周小平？一个舆论热点或许能帮助中国社会更深层理解审查
   </a>
   》）。
  </p>
  <p class="graf graf--p">
   就如俄罗斯和中国通过在网络层面监控互联网来确定什么是可接受的内容，性质一样，西方国家的大多数公民已经恐慌于这样一个事实，即 审查已经演变成了 Facebook 和 Twitter 等集中化社交媒体怪物的核心功能。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    但关心这件事的人依旧太少，除非人们自己亲历账户被封锁、内容被删除。就如在中国讲述审查监视的效率一直很低，除非人们自己或自己的亲人遭遇了审查监视的威胁
   </strong>
   。
  </p>
  <p class="graf graf--p">
   亚历山大·布鲁姆（Alexander Blum）在最近的
   <a class="markup--anchor markup--p-anchor" data-href="https://areomagazine.com/2019/04/05/the-left-wing-case-against-censorship-of-social-media/" href="https://areomagazine.com/2019/04/05/the-left-wing-case-against-censorship-of-social-media/" rel="noopener noreferrer" target="_blank">
    一篇文章
   </a>
   中全面分析了反对社交媒体审查的论点：当公共广场由私人和基本上不负责任的实体控制时，谁能说清明天哪些形式的言论可能被任意禁止？
  </p>
  <p class="graf graf--p">
   Blum 认为，左翼进步公民应完全反对平台审查，因为他们自己未来可能会最先受到审查。
  </p>
  <p class="graf graf--p">
   然而，虽然私人社交媒体平台已经有了强大的自我审查，但他们也同样继续受到其运营所在的监管环境的审查。
  </p>
  <p class="graf graf--p">
   平台的级别仅仅是整个审查层次结构的一部分，从底层网络开始，以监视和整理来自多个私有平台的数据的监视系统结束。
  </p>
  <p class="graf graf--p">
   Facebook 目前正在进行自我监管，以禁止白人民族主义和白人至上主义内容，同时等待澳大利亚政府决定其监控和删除极端主义内容的法律义务的程度。就连 Facebook 自己也承认，很难弄清楚谁是白人至上主义者。
  </p>
  <p class="graf graf--p">
   一般而言，自我审查和被动政府干预之间的共生关系可能得到了 Blum 在其作品中所确定的进步人士的支持。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://quillette.com/2019/02/12/it-isnt-your-imagination-twitter-treats-conservatives-more-harshly-than-liberals/" href="https://quillette.com/2019/02/12/it-isnt-your-imagination-twitter-treats-conservatives-more-harshly-than-liberals/" rel="noopener noreferrer" target="_blank">
    对 Twitter 禁令所表明的政治偏见的分析
   </a>
   显示，Twitter（与硅谷出现的许多其他平台一样）
   <strong class="markup--strong markup--p-strong">
    具有内在的渐进式意识形态倾向，并且没有理由相信这种偏见会自我纠正。根本没有纠正的动机
   </strong>
   。
  </p>
  <p class="graf graf--p">
   最令人震惊的是，互联网还倾向于加速技术的发展，这些技术允许物理监视和与互联网本身无关的其他类型的监控。
  </p>
  <p class="graf graf--p">
   中国再次提供了一个反乌托邦的模版，其中包括哥特式的社会信用体系，允许政府对公民进行全方位的评估和评级，并惩罚任何所谓的反社会行为。
  </p>
  <p class="graf graf--p">
   Facebook 更厉害，它已经达到了反乌托邦的终极模式，详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/facebook-%e7%bb%99%e4%ba%86%e5%85%a8%e7%90%83%e5%bd%93%e6%9d%83%e8%80%85%e4%b8%80%e4%b8%aa%e7%bb%88%e6%9e%81%e6%9a%b4%e6%94%bf%e8%93%9d%e5%9b%be/" href="https://www.iyouport.org/facebook-%e7%bb%99%e4%ba%86%e5%85%a8%e7%90%83%e5%bd%93%e6%9d%83%e8%80%85%e4%b8%80%e4%b8%aa%e7%bb%88%e6%9e%81%e6%9a%b4%e6%94%bf%e8%93%9d%e5%9b%be/" rel="noopener noreferrer" target="_blank">
    Facebook 给了全球当权者一个终极暴政蓝图
   </a>
   》
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在互联网出现之前，这样的1984系统是完全不可行的：它需要大量的数据、以及以有意义和有效的方式，快速处理这些数据的计算能力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   互联网允许不同的中国政府部门相互沟通并更新社会信用评分。如果没有无处不在的瞬时传输大量数据的手段，就不可能有这种惩罚方法的存在。
  </p>
  <p class="graf graf--p">
   关联《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E8%A5%BF%E6%96%B9%E7%A4%BE%E4%BC%9A%E8%B7%9D%E7%A6%BB%E4%B8%AD%E5%9B%BD%E5%BC%8F%E7%9A%84%E7%A4%BE%E4%BC%9A%E4%BF%A1%E7%94%A8%E4%BD%93%E7%B3%BB%E7%9B%91%E6%8E%A7%E8%BF%98%E6%9C%89%E5%A4%9A/" href="https://www.iyouport.org/%E8%A5%BF%E6%96%B9%E7%A4%BE%E4%BC%9A%E8%B7%9D%E7%A6%BB%E4%B8%AD%E5%9B%BD%E5%BC%8F%E7%9A%84%E7%A4%BE%E4%BC%9A%E4%BF%A1%E7%94%A8%E4%BD%93%E7%B3%BB%E7%9B%91%E6%8E%A7%E8%BF%98%E6%9C%89%E5%A4%9A/" rel="noopener noreferrer" target="_blank">
    西方社会距离中国式的”社会信用体系”监控还有多远
   </a>
   ？》
  </p>
  <p class="graf graf--p">
   西方政府也尝试过类似的系统。最臭名昭著的爱国者法案授予美国情报机构广泛的权力，以收集所有美国公民的数据。
  </p>
  <p class="graf graf--p">
   事实证明美国的监视深入程度是极为惊人的。详见一个最全面的汇总报告《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" rel="noopener noreferrer" target="_blank">
    互联网霸主的时间线 — — 他们将互联网视为间谍、破坏和战争的主场
   </a>
   》。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4405 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-4.png?resize=1100%2C846&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-4.png?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-4.png?resize=300%2C231&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-4.png?resize=768%2C591&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-4.png?resize=1024%2C788&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-4.png?resize=1100%2C846&amp;ssl=1 1100w" data-recalc-dims="1" height="846" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-4.png?resize=1100%2C846&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4405" data-recalc-dims="1" height="846" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-4.png?resize=1100%2C846&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-4.png?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-4.png?resize=300%2C231&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-4.png?resize=768%2C591&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-4.png?resize=1024%2C788&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-4.png?resize=1100%2C846&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在没有政府监视的情况下在线交流和共享数据的权利已经越来越少，这一可怕的趋势或许是不可逆转的。
  </p>
  <p class="graf graf--p">
   在澳大利亚，当局已经提出立法，迫使加密通信服务提供商为政府机构
   <a class="markup--anchor markup--p-anchor" data-href="https://www.abc.net.au/radio/programs/am/tech-companies-renew-their-calls-for-encryption-bill-amendments/10947046" href="https://www.abc.net.au/radio/programs/am/tech-companies-renew-their-calls-for-encryption-bill-amendments/10947046" rel="noopener noreferrer" target="_blank">
    开发后门
   </a>
   ，从而使大量私人内容向情报机构开放监视。
  </p>
  <p class="graf graf--p">
   民主政府迫使互联网服务提供商或平台以这种方式行事的能力通常在功能上类似于专制国家政府的直接控制：国家监管的触角正慢慢地缠绕着日常的互联网使用。
  </p>
  <p class="graf graf--p">
   在线审查的程度也
   <strong class="markup--strong markup--p-strong">
    明显超过了
   </strong>
   所谓的改善国家安全所需的一切。
  </p>
  <p class="graf graf--p">
   欧洲指令
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wired.co.uk/article/what-is-article-13-article-11-european-directive-on-copyright-explained-meme-ban" href="https://www.wired.co.uk/article/what-is-article-13-article-11-european-directive-on-copyright-explained-meme-ban" rel="noopener noreferrer" target="_blank">
    第11条/第13条表明
   </a>
   ，在巨额罚款的威胁下，欧盟不会犹豫是否在互联网平台上施加巨大的（并且可以说是定义不明确的）义务：所谓的链接税和平台提供商现在需要的版权过滤在互联网时代开始时是不可想象的。
  </p>
  <p class="graf graf--p">
   臭名昭著的英国色情禁令表明了国家单方面实施内容限制的惊人能力 —— 就像在克赖斯特彻奇的悲剧之后，新西兰互联网服务提供商能做到迅速阻止视频内容的传播那样。
  </p>
  <p class="graf graf--p">
   虽然这些例子没有暗示任何形式的反自由阴谋，但它们表明了一个关键点：没有审查和监视的互联网已经不复存在了。
  </p>
  <p class="graf graf--p">
   在现有媒体和通信法的支持下，很多民主政府已经表现出采取被动行动并在没有民主协商的情况下实施审查制度的态度。
  </p>
  <p class="graf graf--p">
   互联网最初被视为对国家审查和控制的解毒剂，主要是因为它曾在功能上无法被监视或控制除物理访问互联网连接以外的任何内容，并且由于缺乏有关互联网技术使用的法规。
  </p>
  <p class="graf graf--p">
   然而，限制公民可以通过互联网访问的信息类型的意愿和能力从未如此强大 ，即使互联网本身从未如此重要。
  </p>
  <p class="graf graf--p">
   一些社交媒体平台选择参与政治化的自我审查，这只是一个更大、更可怕的趋势的冰山一角。
  </p>
  <p class="graf graf--p">
   更多详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E5%A4%A7%E9%83%A8%E5%88%86%E5%AE%A1%E6%9F%A5%E5%B9%B6%E9%9D%9E%E5%A6%82%E4%BD%A0%E5%B9%B3%E6%97%A5%E7%9A%84%E5%90%90%E6%A7%BD%E5%AF%B9%E8%B1%A1%E9%82%A3%E8%88%AC%E6%98%BE%E8%80%8C%E6%98%93%E8%A7%81/" href="https://www.iyouport.org/%E5%A4%A7%E9%83%A8%E5%88%86%E5%AE%A1%E6%9F%A5%E5%B9%B6%E9%9D%9E%E5%A6%82%E4%BD%A0%E5%B9%B3%E6%97%A5%E7%9A%84%E5%90%90%E6%A7%BD%E5%AF%B9%E8%B1%A1%E9%82%A3%E8%88%AC%E6%98%BE%E8%80%8C%E6%98%93%E8%A7%81/" rel="noopener noreferrer" target="_blank">
    大部分审查并非如你平日的吐槽对象那般显而易见 — — 为自由而战（二）
   </a>
   》
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    注：
   </strong>
   这篇文章写得不够透彻，我们增加了一些关联内容以便能被读懂。最透彻的应该是这本书《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%BA%92%E8%81%94%E7%BD%91%E7%A9%B6%E7%AB%9F%E6%98%AF%E4%BB%80%E4%B9%88%EF%BC%9F/" href="https://www.iyouport.org/%E4%BA%92%E8%81%94%E7%BD%91%E7%A9%B6%E7%AB%9F%E6%98%AF%E4%BB%80%E4%B9%88%EF%BC%9F/" rel="noopener noreferrer" target="_blank">
    互联网究竟是什么？ — — 来自“监视谷”惊人的启示
   </a>
   》。但我们依旧翻译它，至少能证明美国学者也亲口承认这个问题了。这可不怎么容易。
  </p>
