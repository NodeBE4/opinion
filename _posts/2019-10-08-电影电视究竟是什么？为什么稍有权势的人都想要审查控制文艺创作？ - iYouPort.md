  <p class="p1">
   <span class="s1">
    最近通过信息自由法案获得了
   </span>
   <span class="s2">
    200
   </span>
   <span class="s1">
    页文件，内容都是缉毒局和文艺创作者之间签订的合同。这些惊人的合同首次揭露了缉毒局
   </span>
   <span class="s2">
    ——
   </span>
   <span class="s1">
    这个被认为相对不怎么厉害的权力部门
   </span>
   <span class="s2">
    ——
    <span class="Apple-converted-space">
    </span>
   </span>
   <span class="s1">
    也一直在深入操纵电影电视的制作。
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    缉毒局（
   </span>
   <span class="s2">
    DEA
   </span>
   <span class="s3">
    ）在娱乐业中非常活跃；
    <b>
     该机构一直在严格控制纪录片、真人秀和电影电视剧中涉及该部门的所有内容
    </b>
    。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    通过多个项目，
   </span>
   <span class="s2">
    DEA
   </span>
   <span class="s1">
    会仔细检查他们自己的文件，挑选出使他们看起来很好的精选案例，然后要求文艺创作者制造出文艺作品。
   </span>
  </p>
  <p class="p4">
   <span class="s3">
    这些合同
   </span>
   <span class="s1">
    [
    <a href="https://www.scribd.com/document/411596720/DEA-Production-Assistance-Agreements" rel="noopener noreferrer" target="_blank">
     PDF
    </a>
    ]
   </span>
   <span class="s3">
    涵盖
   </span>
   <span class="s1">
    2011
   </span>
   <span class="s3">
    年至
   </span>
   <span class="s1">
    2017
   </span>
   <span class="s3">
    年。在此期间，
   </span>
   <span class="s1">
    DEA
   </span>
   <span class="s3">
    支持了数十个类似项目，包括
   </span>
   <span class="s1">
    “Cops and Coyotes”
   </span>
   <span class="s3">
    和
   </span>
   <span class="s1">
    “Drugs Inc.”
   </span>
   <span class="s3">
    中的多个剧集、以及
   </span>
   <span class="s1">
    “Gangsters: America’s Most Evil”
   </span>
   <span class="s3">
    等作品。
   </span>
  </p>
  <p class="p4">
   <span class="s3">
    他们
    <b>
     也在操纵纯属虚构的与毒品有关的电视剧
    </b>
    。其他被操纵的影视项目包括
   </span>
   <span class="s1">
    “Lethal Cargo”
   </span>
   <span class="s3">
    、
   </span>
   <span class="s1">
    “The Notorious Mr. Bout”
   </span>
   <span class="s3">
    和
   </span>
   <span class="s1">
    “Declassified: Untold Stories of American Spies”
   </span>
   <span class="s3">
    。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    他们
    <b>
     甚至与美国政府资助的中东广播网络合作
    </b>
    ，开展了一个以
   </span>
   <span class="s2">
    DEA
   </span>
   <span class="s1">
    博物馆为特色的项目。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    信息自由法案并不好用，不仅极为耗时（经常要等待数月甚至数年时间），并且，最终能获得的文件也是被编辑过的或残缺重要部分的。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    合同中奇怪的是
   </span>
   <span class="s2">
    “Narcos”
   </span>
   <span class="s1">
    和
   </span>
   <span class="s2">

   </span>
   <span class="s1">

   </span>
   <span class="s2">

   </span>
   <span class="s1">
    员工作为顾问。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    在回应独立
   </span>
   <span class="s2">
    FOIA
   </span>
   <span class="s1">
    要求询问他们对
   </span>
   <span class="s2">
    “Breaking Bad”
   </span>
   <span class="s1">
    的意见时，
   </span>
   <span class="s2">
    DEA
   </span>
   <span class="s1">
    声称他们
   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">
    无法找到任何记录
   </span>
   <span class="s2">
    ”
   </span>
   <span class="s1">
    。
   </span>
  </p>
  <p class="p4">
   <span class="s3">
    发布的文件中也没有包括
   </span>
   <span class="s1">

   </a>

   </span>

  <p class="p3">
   <span class="s3">

   </span>
   <span class="s2">
    <b>
     DEA
    </b>
   </span>
   <span class="s3">

    </b>

   </span>

   <span class="s1">

   </span>

   </span>

   </span>
   <span class="s2">

   </span>
   <span class="s1">
    现场的
   </span>
   <span class="s2">

   </span>
   <span class="s1">

   </span>
   <span class="s2">
    ”
   </span>
   <span class="s1">

   </span>

  <p class="p1">

  </p>

   <span class="s2">
    <b>
     DEA
    </b>
   </span>
   <span class="s3">
    <b>

    </b>

   </span>
  </p>
  <p class="p1">
   <span class="s1">

   <span class="s1">
    ：
   </span>

   </span>
  </p>
  <p class="p1">
   <span class="s1">
    <b>

    </b>
   </span>
   <span class="s2">

   <span class="s2">

   <span class="s1">

    ”

   <span class="s1">
    的
   </span>
   <span class="s2">
    DEA
   </span>
   <span class="s1">

   </span>
  </p>
  <p class="p1">
   <span class="s1">
    在所有受支持的项目中，
    <b>
     影视制作者必须授予
    </b>
   </span>
   <span class="s2">
    <b>
     “

   </span>
   <span class="s1">
    <b>

    </b>

   <span class="s2">

   </span>
   <span class="s1">

  <p class="p1">

   </span>
   <span class="s1">

  <p class="p4">

   <span class="s3">

  </p>

   </span>
  </p>
  <p class="p4">
   <span class="s1">
    <b>
     “
    </b>
   </span>
   <span class="s3">
    <b>
     这就是宣传的
    </b>
   </span>
   <span class="s1">
    <b>
     BS
    </b>
   </span>
   <span class="s3">
    <b>
     ，纯洁而简单
    </b>
    ，
   </span>
   <span class="s1">
    “Valentine
   </span>
   <span class="s3">
    说。
   </span>
  </p>
  <p class="p3">
   <span class="s2">
    <b>
     “
    </b>
   </span>
   <span class="s3">
    <b>
     我谈过很多关于英雄的神话。他们想把自己描绘成一个崇高事业的英雄。他们要求完全控制叙事
    </b>
   </span>
   <span class="s2">
    <b>
     ”
    </b>
   </span>
   <span class="s3">
    <b>
     。
    </b>
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    当被问及这些影视节目是否代表了对
   </span>
   <span class="s2">
    DEA
   </span>
   <span class="s1">
    间谍和毒贩之间真实关系的不准确呈现时，
   </span>
   <span class="s2">
    Valentine
   </span>
   <span class="s1">
    回答说，
   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">
    这取决于具体的间谍和贩运者，但是可以肯定，贩运者永远不会逮捕间谍。因此，间谍拥有法律的力量。这件事变得棘手的地方是线人、特殊雇员和秘密行动。也就是中央情报局的那些东西。
   </span>
   <span class="s2">
    ”
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    除了声明对影视制作的最终编辑的完全控制之外，这些合同文件还有条款声明
   </span>
   <span class="s2">
    DEA“
   </span>
   <span class="s1">
    保留进行背景调查的完整权利，并且如果必要的话，
   </span>
   <span class="s2">
    DEA
   </span>
   <span class="s1">
    可以拒绝任何被认为可能危及
   </span>
   <span class="s2">
    DEA
   </span>
   <span class="s1">
    操作的人。
   </span>
   <span class="s2">
    “
   </span>
  </p>
  <p class="p1">
   <span class="s4">
    ⚠️
   </span>
   <span class="s1">
    与联邦调查局一样，
   </span>
   <span class="s2">
    DEA
   </span>
   <span class="s1">
    不仅关注他们支持的影视作品的内容，而且还关注制作这些影视作品的人的背景（你可以理解为政审，形式差不多）。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    “
   </span>
   <span class="s1">
    这是确保完全控制的另一种方式。
   </span>
   <span class="s2">
    DEA
   </span>
   <span class="s1">
    痴迷于控制并将控制做为其优势力量，
   </span>
   <span class="s2">
    “ Valentine
   </span>
   <span class="s1">
    补充道。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    这些合同也进一步证明了
   </span>
   <span class="s2">
    DEA
   </span>
   <span class="s1">
    对控制的痴迷
   </span>
   <span class="s2">
    ——
   </span>
   <span class="s1">
    合同要求制作人在制作发布或播放后立即销毁
   </span>
   <span class="s2">
    DEA
   </span>
   <span class="s1">
    提供的任何材料（照片、文件、统计数据、
   </span>
   <span class="s2">
    b-roll
   </span>
   <span class="s1">
    素材等等）。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    制片人必须向
   </span>
   <span class="s2">
    DEA
   </span>
   <span class="s1">
    提供
   </span>
   <span class="s2">
    DVD
   </span>
   <span class="s1">
    副本和非独家许可，以便该机构可以使用由此产生的任何电影或电视节目，以
   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">
    招募、培训、专业发展、社区关系、或其他宣传性努力
   </span>
   <span class="s2">
    ”
   </span>
   <span class="s1">
    。
   </span>
   <span class="s4">
    ⚪️
   </span>
  </p>
