   <li>
    <span style="color: #00ccff;">
     <em>
      耐克鞋、热裤、哈利波特、新媒体、迷你裙……朝鲜的”文化革命”都有哪些亮点？
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   文化革命这个词是原文标题，或许对中国读者来说有些歧义，因为它不是中国的那种文革，准确说是我们曾经在长篇图文报告中所讲述的朝鲜“改开”，在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E6%9C%9D%E9%B2%9C-%E6%94%B9%E5%BC%80-%E7%9A%84%E6%96%B0%E9%9D%A2%E7%9B%AE-%E7%9B%91%E8%A7%86%E8%AE%BE%E5%A4%87%E7%9A%84%E5%85%B4%E8%B5%B7%E5%8F%8A%E7%94%9F%E6%B4%BB%E7%8A%B6%E5%86%B5%E7%9A%84%E5%8F%98%E5%8C%96-%E5%A4%9A%E5%9B%BE-9b15dfd31136" href="https://medium.com/@iyouport/%E6%9C%9D%E9%B2%9C-%E6%94%B9%E5%BC%80-%E7%9A%84%E6%96%B0%E9%9D%A2%E7%9B%AE-%E7%9B%91%E8%A7%86%E8%AE%BE%E5%A4%87%E7%9A%84%E5%85%B4%E8%B5%B7%E5%8F%8A%E7%94%9F%E6%B4%BB%E7%8A%B6%E5%86%B5%E7%9A%84%E5%8F%98%E5%8C%96-%E5%A4%9A%E5%9B%BE-9b15dfd31136" rel="noopener noreferrer" target="_blank">
    朝鲜“改开”的新面目：监视设备的兴起及生活状况的变化
   </a>
   》其实如果更准确说的话，“改开”也会有歧义，它与中国80年代的改开不完全一样，它还包括了中国当下的很多特征。
  </p>
  <p class="graf graf--p">
   穿着热裤的舞者、耐克的 Air Jordan 品牌，乍一看你会惊讶这是朝鲜。朝鲜流行文化长期被批评为对斯大林主义黑暗时代的拙劣回归，不过现在似乎有些不一样的东西出现了。
  </p>
  <p class="graf graf--p">
   不仅如此，从电视剧和动画节目到消费品的多样性和包装等各方面都有所变化。究竟是为了跟上韩国，还是金正恩愿意接受其前任鄙视的“资产阶级西方消费文化”，不清楚。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “对我们来说最重要的是生产出一种适合大众口味的产品，”朝鲜 Ryuwon 鞋厂的 Kim Kyong Hui 告诉美联社，该厂的陈列室里摆满了几十种跑步鞋、排球鞋、足球鞋、甚至乒乓球鞋，她说：“受人尊敬的领导人金正恩指示我们密切研究来自世界各地的鞋子，并学习他们榜样，” 她指着一双火焰红色的高帮篮球鞋。
  </p>
  <p class="graf graf--p">
   可以肯定的是，朝鲜仍然是全世界最孤立的国家之一。这一系列变化是非常谨慎的，与此同时更重要的是，任何公开批评政府或领导的人、或被当局视为威胁的人，都会受到严重惩罚。
   <strong class="markup--strong markup--p-strong">
    这点和中国一样：改开只是为稳固政权，而不是改革政体
   </strong>
   。
  </p>
  <p class="graf graf--p">
   最明显的变化是电视和普遍的宣传节目还有纪录片，要以表达对领导人的赞美为主题。
   <strong class="markup--strong markup--p-strong">
    这点也和中国一样，我们在景观政治的相关分析中介绍过
   </strong>
   ，详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://pao-pao.net/article/885" href="https://pao-pao.net/article/885" rel="noopener noreferrer" target="_blank">
    中国政府已攀上景观政治的边缘
   </a>
   》
  </p>
  <p class="graf graf--p">
   朝鲜国家主要电视网络正在停止播放电视剧的最新一集，这是一部历史剧集，讲述的是16世纪晚期，朝鲜抵制日本入侵的故事。
  </p>
  <p class="graf graf--p">
   和中国一样，反日、民族主义的主题在朝鲜并不是什么新鲜事。如今类似的主题有些改变，被用于“金正恩对电视剧产业的第一次重大贡献” — — 这是一部动画系列，从金正恩父亲的时代复兴了一部名为“少年将军”的流行漫画，该漫画在2015年首次亮相。这部动画的故事创作于高句丽时期，关于该民族抵制中国人的入侵，该剧在朝鲜非常热门，还因此为手机创建了一款 Boy General 游戏。据信即将发行新剧集。
  </p>
  <p class="graf graf--p">
   该剧中由朝鲜演员扮演的恶棍用日语进行对话，一般来说比较准确，尽管带有浓重的朝鲜口音。与此同时，
   <strong class="markup--strong markup--p-strong">
    Boy General 游戏甚至巧妙地运用了电脑特技效果，在视觉效果上甚至可以与世界上最好的动画相提并论
   </strong>
   。
  </p>
  <p class="graf graf--p">
   TD 文章认为金正恩意识到了朝鲜公众越来越熟悉外国流行文化，尽管严格限制使大多数人无法出国旅行或自由体验外国电影、音乐或书籍。
  </p>
  <p class="graf graf--p">
   对外国文化的熟悉程度尤其适用于朝鲜精英，他们习惯于在首都平壤的高档商店货架上看到 Dior 到索尼的品牌产品；来自中国的廉价仿冒品在朝鲜全国各地的市场中也很常见。
  </p>
  <p class="graf graf--p">
   观看韩国电影或收听韩国音乐依旧是非法的。但是很多东西都有所变化，官方认可的文化食品里也包含外国料理。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    宝莱坞电影在朝鲜国营电影院很受欢迎
   </strong>
   — 例如，2009年的“三白痴”与 Aamir Khan 最近在金日成广场对面的一家电影院放映。朝鲜的教育频道定期播放外国纪录片中的长篇剪辑；哈利波特的书是朝鲜最大的图书馆最受欢迎的书籍之一。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “朝鲜“对外国媒体涌入的态度是’现代化’的媒体制作，以提供一种有吸引力和有竞争力的新闻产品，老一代产品不再具有吸引力，” 一家新加坡的非营利组织的创始人 Geoffrey See 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “对于消费品而言，它还与国家政策相关，以鼓励更多的国内产品和进口替代品，”他说。
  </p>
  <p class="graf graf--p">
   这点也和中国一样。习的“融媒”指示是今年中国宣传业的一个最大改变， IYP 在中国年之前连续发布了四篇文章分析为什么“现代新闻”无益于民主。当中国的“融媒”转型彻底完成时（它很快就会完成）即便有了选票也很可能不会有民主，这就是为什么人们需要从现在开始警惕。
  </p>
  <p class="graf graf--p">
   金正恩第一次尝试更新流行文化的场景基本是在他于2011年底掌权后创立的莫兰邦乐队，一群女歌手和音乐家，他们是政权的“软实力”。
  </p>
  <p class="graf graf--p">
   虽然这些成员都属于朝鲜人民军，但他们以迷你裙表演和穿着时尚服饰而闻名。他们已经发行了数十首歌曲，所有这些歌曲都通过音乐会巡演、DVD和电视播放获得了大量观众。
  </p>
  <p class="graf graf--p">
   然而，目前他们开始看起来有点过时了。
  </p>
  <p class="graf graf--p">
   去年2月，
   <strong class="markup--strong markup--p-strong">
    朝鲜派遣了一些顶级音乐家，包括穿着黑色短裤的女声五重奏组，该乐团在韩国平昌冬季奥运会期间演出
   </strong>
   。两个月后，当韩国女子团体 Red Velvet 在平壤举行的第一场真正的 K-pop 表演时，金正恩就在观众席上。金正日上个月还将该乐团送到北京进行了另一次表演。
  </p>
  <p class="graf graf--p">
   尽管如此，穿着传统的“Choson-ot”礼服表演的军乐团和经过专业训练的歌手仍然是平壤音乐界的中流砥柱。这支女子乐队在北京的演出得到了国家军队合唱团和管弦乐队的支持。
  </p>
  <p class="graf graf--p">
   最重要的是，
   <strong class="markup--strong markup--p-strong">
    这一切变化都与政治紧密相关
   </strong>
   。
  </p>
  <p class="graf graf--p">
   据国家媒体报道，当乐团返回平壤时，金正恩敦促他们继续“以党的意识形态进行原创艺术活动”，并“勇敢地作为党的喉舌”。
  </p>
