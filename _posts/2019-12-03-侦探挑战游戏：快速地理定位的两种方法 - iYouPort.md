  <p class="graf graf--p">
   对于第一次看到本话题内容的读者说：这里是一系列小游戏；不定期更新。主要采用的是开源情报社区提供的“挑战测试”。它们能帮助您培养很好的观察和分析能力 —— 思考方式是比工具更重要的东西。
  </p>
  <p class="graf graf--p">
   今天的内容是两个挑战：这两张照片（题目）
   <strong class="markup--strong markup--p-strong">
    在所包含的信息类型方面非常不同，需要两种完全不同的方法才能成功地对图像进行地理定位
   </strong>
   。
  </p>
  <p class="graf graf--p">
   💡它将帮助您体验到灵活的思考方式对于情报分析工作来说有多重要。
  </p>
  <p class="graf graf--p">
   对一个调查起作用的东西对于另一个调查就不起作用 —— 死记硬背是不行的，只有那些使用了正确方法的参与者才能成功。
  </p>
  <p class="graf graf--p">
   本文介绍这两种不同的方法，可以快速进行地理定位挑战，尤其是，希望能强调：
   <strong class="markup--strong markup--p-strong">
    如何确定何时应用正确的方法
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    对象搜索和图像相似度
   </strong>
  </p>
  <p class="graf graf--p">
   题目是这样的：
  </p>
  <p class="graf graf--p">

   </noscript>
  </p>
  <p class="graf graf--p">
   有两种基本方法可以处理此图像。
  </p>
  <p class="graf graf--p">
   您可以通过视觉相似性进行搜索，也可以基于照片中的对象进行搜索。这些都是在地理地位中常用的思考方式。
  </p>
  <p class="graf graf--p">
   通过视觉相似性搜索基本上是涉及使用反向图像搜索工具来查找原始图像的全部或部分的匹配。
  </p>
  <p class="graf graf--p">
   对于这个题目来说，如果使用这种方法那么对象就是照片中那个 Metro 标志，但是这样一来最终你会得到很多匹配的东西：
  </p>

  <p class="graf graf--p">
   其实也并非特别糟糕。至少足以告诉你这是在巴黎拍摄的（尽管从原始照片中用眼就能看出来），但是没有足够的视觉匹配来给出具体位置。

  <p class="graf graf--p">
   成功的参与者似乎并不依赖于这种方法。基于对象的搜索是解决这一挑战的更好方法。
  </p>
  <p class="graf graf--p">
   原因是照片中存在
   <strong class="markup--strong markup--p-strong">
    大量可搜索的信息
   </strong>
   ，例如商店和街道标志，这些信息足以提供正确的位置，使得依赖于视觉相似性的搜索根本没必要。
  </p>
  <p class="graf graf--p">
   考虑此图像中可识别对象的数量：
  </p>
  <p class="graf graf--p">
   1) Metro 标志
  </p>
  <p class="graf graf--p">
   2) Royal *lly bar
  </p>
  <p class="graf graf--p">
   3) “Boulanger / Patissier”标志
  </p>
  <p class="graf graf--p">
   4) 两个 Loto 标志
  </p>
  <p class="graf graf--p">
   5) 部分模糊的路牌，上面写着* tgallet
  </p>
  <p class="graf graf--p">
   即使您不认识巴黎，也能从上述“3”中看出，这是法语；“1”则告诉您这可能是巴黎（
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Rapid_transit_in_France" href="https://en.wikipedia.org/wiki/Rapid_transit_in_France" rel="noopener noreferrer" target="_blank">

   这种情况是：图像本身提供了如此多的特定和可识别对象可以进行搜索，于是，与搜索视觉相似性相比，基于对象的搜索就是一种更有效的技术，可以快速地对图像进行地理定位。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    没有任何标志的时候 —— 视觉相似性
   </strong>
  </p>
  <p class="graf graf--p">
   第二个挑战是一种完全不同的图像类型，但它可以作为一个很好的例子，说明为一张照片工作的相同方法并不适用于另一张照片。

   <img alt="" class="aligncenter size-full wp-image-8682 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-1.jpg?resize=1024%2C311&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-1.jpg?w=1024&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-1.jpg?resize=300%2C91&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-1.jpg?resize=768%2C233&amp;ssl=1 768w" data-recalc-dims="1" height="311" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-1.jpg?resize=1024%2C311&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>

    <img alt="" class="aligncenter size-full wp-image-8682" data-recalc-dims="1" height="311" sizes="(max-width: 1024px) 100vw, 1024px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-1.jpg?resize=1024%2C311&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-1.jpg?w=1024&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-1.jpg?resize=300%2C91&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-1.jpg?resize=768%2C233&amp;ssl=1 768w" width="1024"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   哇，这个好厉害 —— 什么标志都没有；与上面第一个题目相比，同样的方法一点都用不上了。
  </p>
  <p class="graf graf--p">
   这张照片是在哪里拍摄的？
  </p>
  <p class="graf graf--p">
   考虑照片中可识别线索（线索都太大了，无法使用）：
  </p>
  <p class="graf graf--p">
   1）河
  </p>
  <p class="graf graf--p">
   2）丘陵
  </p>
  <p class="graf graf--p">
   3）红屋顶房屋
  </p>
  <p class="graf graf--p">
   4）盐盘
  </p>
  <p class="graf graf--p">
   5）河口
  </p>
  <p class="graf graf--p">
   你能看到，这里没有任何特定或独特的内容。
  </p>
  <p class="graf graf--p">
   在这种情况下，缺少特定对象意味着视觉相似性搜索更有效。
  </p>
  <p class="graf graf--p">
   使用“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e6%90%9c%e5%9b%be%e4%b8%8d%e5%a5%bd%e7%94%a8%e6%80%8e%e4%b9%88%e5%8a%9e%ef%bc%9a%e6%8e%a8%e8%8d%90%e5%8f%a6%e5%a4%96%e4%b8%a4%e4%b8%aa%e6%9b%b4%e5%a5%bd%e7%94%a8%e7%9a%84%e5%b7%a5/" href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e6%90%9c%e5%9b%be%e4%b8%8d%e5%a5%bd%e7%94%a8%e6%80%8e%e4%b9%88%e5%8a%9e%ef%bc%9a%e6%8e%a8%e8%8d%90%e5%8f%a6%e5%a4%96%e4%b8%a4%e4%b8%aa%e6%9b%b4%e5%a5%bd%e7%94%a8%e7%9a%84%e5%b7%a5/" rel="noopener noreferrer" target="_blank">
    按图搜索
   </a>
   ”，剪裁照片的中心区域，删除了一些天空和前景细节：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8683 jetpack-lazy-image" data-lazy-sizes="(max-width: 1080px) 100vw, 1080px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-12.png?resize=1080%2C635&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-12.png?w=1080&amp;ssl=1 1080w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-12.png?resize=300%2C176&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-12.png?resize=768%2C452&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-12.png?resize=1024%2C602&amp;ssl=1 1024w" data-recalc-dims="1" height="635" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-12.png?resize=1080%2C635&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1080"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8683" data-recalc-dims="1" height="635" sizes="(max-width: 1080px) 100vw, 1080px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-12.png?resize=1080%2C635&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-12.png?w=1080&amp;ssl=1 1080w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-12.png?resize=300%2C176&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-12.png?resize=768%2C452&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-12.png?resize=1024%2C602&amp;ssl=1 1024w" width="1080"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   然后使用 Yandex 进行快速反向图像搜索，在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.slobodnaevropa.org/a/27831258.html" href="https://www.slobodnaevropa.org/a/27831258.html" rel="noopener noreferrer" target="_blank">
    斯洛文尼亚的 Piranski 附近
   </a>
   进行了位置匹配。通过视觉相似性搜索对于这种地理定位更有效。
  </p>
  <p class="graf graf--p">
   ⚠️对于大多数类型的地理定位问题，有必要使用一系列技术来找到答案 —— 而不是单一技术。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    重要的是要记住，为每个图像自动使用相同的技术并不总是有效的，每个图像都需要根据自己的内容进行评估 —— 我实际看到的是什么？我能看到多少细节？
   </strong>
  </p>
  <p class="graf graf--p">
   一旦你回答了这个问题，就可以更好地决定：应该采用什么样的方法。
  </p>
  <p class="graf graf--p">
   请注意调大分辨率哦！尤其是来自社交媒体的照片，分辨率都较低，您需要首先恢复到原始分辨率才能看清其中的细节。具体做法在这里《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%88%ab%e8%ae%a9%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e7%9a%84%e8%87%aa%e7%84%b6%e6%a8%a1%e7%b3%8a%e9%98%bb%e6%8c%a1%e7%9c%9f%e7%9b%b8/" href="https://www.iyouport.org/%e5%88%ab%e8%ae%a9%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e7%9a%84%e8%87%aa%e7%84%b6%e6%a8%a1%e7%b3%8a%e9%98%bb%e6%8c%a1%e7%9c%9f%e7%9b%b8/" rel="noopener noreferrer" target="_blank">
    别让社交媒体的自然模糊阻挡真相
   </a>
   》。
  </p>
  <p class="graf graf--p">
   多练习，思路会更活跃；任何地方都可以练习，我们曾经演示过如何使用电影来练习情报分析，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%bf%e7%94%a8%e7%94%b5%e5%bd%b1%e5%81%9a%e4%be%a6%e6%8e%a2%e6%8c%91%e6%88%98%e6%bc%94%e7%a4%ba%ef%bc%9a%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e6%b8%b8%e6%88%8f/" href="https://www.iyouport.org/%e4%bd%bf%e7%94%a8%e7%94%b5%e5%bd%b1%e5%81%9a%e4%be%a6%e6%8e%a2%e6%8c%91%e6%88%98%e6%bc%94%e7%a4%ba%ef%bc%9a%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e6%b8%b8%e6%88%8f/" rel="noopener noreferrer" target="_blank">
    使用电影做侦探挑战演示：开源调查游戏
   </a>
   》。您完全可以在社交网络上随便寻找调查目标；做得越多就能越熟练。⚪️
  </p>
