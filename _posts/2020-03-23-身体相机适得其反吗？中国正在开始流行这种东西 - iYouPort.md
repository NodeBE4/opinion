   <li class="graf graf--p">
    <span style="color: #00ccff;">
     <em>
      <strong>
       身体相机本来的用途应该是满足问责制。但是现在发生了什么？
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   我们收到消息说中国一些城市的巡逻警察正在使用身体相机。
  </p>
  <p class="graf graf--p">
   事实上身体相机已经在全球很多国家流行多年了。一开始它的目的是问责制 —— 记录警方在执法过程中的行为，在受害人申诉的情况下可以拿出来作为证据。因此它曾经一直被民权组织支持。
  </p>
  <p class="graf graf--p">
   在英国，已有15万警官已经配备了身体相机，而且，医院和学校中也使用了大量的身体相机，一些地方当局还试图迫使出租车司机也使用这种监视设备……
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    多年来的事实证明，这种东西不仅换不来正义，反而成为了更严重的监视工具。
   </strong>
   就如中国读者普遍了解的状况：当需要伸张正义的时候摄像头总是不好用。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theatlantic.com/technology/archive/2019/11/border-patrol-weighs-body-cameras-face-recognition/600469/" href="https://www.theatlantic.com/technology/archive/2019/11/border-patrol-weighs-body-cameras-face-recognition/600469/" rel="noopener noreferrer" target="_blank">
    The Atlantic
   </a>
   关于这件事的思考也许会给中国的权利组织一些提示。
  </p>
  <figure aria-describedby="caption-attachment-10380" class="wp-caption aligncenter" id="attachment_10380" style="width: 900px">
   <img alt="" class="wp-image-10380 jetpack-lazy-image" data-lazy-sizes="(max-width: 900px) 100vw, 900px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-7.jpg?resize=900%2C506&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-7.jpg?w=720&amp;ssl=1 720w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-7.jpg?resize=300%2C169&amp;ssl=1 300w" data-recalc-dims="1" height="506" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-7.jpg?resize=900%2C506&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="900"/>

  <p class="graf graf--p">
   2014年，警务人员 Darren Wilson 在密苏里州弗格森的一条街道上枪杀了一名手无寸铁的黑人少年，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theatlantic.com/national/archive/2015/08/violence-in-ferguson-missouri-and-the-legacy-of-michael-browns-death/400901/" href="https://www.theatlantic.com/national/archive/2015/08/violence-in-ferguson-missouri-and-the-legacy-of-michael-browns-death/400901/" rel="noopener noreferrer" target="_blank">
    他名叫 Michael Brown
   </a>
   ，那时候警察的野蛮行径飙升至全国种族平权辩论的中心。
  </p>
  <p class="graf graf--p">
   激进主义者争辩说，执法需要更多的问责制，而身体相机就此成为了该州首选的纠正措施。
  </p>
  <p class="graf graf--p">
   奥巴马政府的司法部在 Michael Brown 去世后的2015年，为这种新的身体相机提供了超过
   <a class="markup--anchor markup--p-anchor" data-href="https://www.justice.gov/opa/pr/justice-department-awards-over-23-million-funding-body-worn-camera-pilot-program-support-law" href="https://www.justice.gov/opa/pr/justice-department-awards-over-23-million-funding-body-worn-camera-pilot-program-support-law" rel="noopener noreferrer" target="_blank">
    2300万美元
   </a>
   的赠款，在2016年又提供了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.justice.gov/opa/pr/department-justice-awards-over-20-million-law-enforcement-body-worn-camera-programs" href="https://www.justice.gov/opa/pr/department-justice-awards-over-20-million-law-enforcement-body-worn-camera-programs" rel="noopener noreferrer" target="_blank">
    2000万美元
   </a>
   。
  </p>
  <p class="graf graf--p">
   当时的候选人希拉里·克林顿（Hillary Clinton）呼吁在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnn.com/2015/04/29/politics/hillary-clinton-police-body-cameras/index.html" href="https://www.cnn.com/2015/04/29/politics/hillary-clinton-police-body-cameras/index.html" rel="noopener noreferrer" target="_blank">
    全国
   </a>
   范围内
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnn.com/2015/04/29/politics/hillary-clinton-police-body-cameras/index.html" href="https://www.cnn.com/2015/04/29/politics/hillary-clinton-police-body-cameras/index.html" rel="noopener noreferrer" target="_blank">
    强制性
   </a>
   使用这种身体相机。
  </p>
  <p class="graf graf--p">
   2018年，纽约法官要求所有NYPD警官都佩戴这种身体相机，以作为结束
   <a class="markup--anchor markup--p-anchor" data-href="https://www.govtech.com/public-safety/Body-Cams-Now-Worn-by-All-20K-New-York-City-Police-Officers.html" href="https://www.govtech.com/public-safety/Body-Cams-Now-Worn-by-All-20K-New-York-City-Police-Officers.html" rel="noopener noreferrer" target="_blank">
    “拦截询问和整顿”程序
   </a>
   的努力的一部分。警务的未来似乎已经到来。
  </p>
  <p class="graf graf--p">
   然而，不到五年之后，这种势头开始放缓。据称身体相机的地位目前已经一落千丈；由于
   <a class="markup--anchor markup--p-anchor" data-href="https://www.techdirt.com/articles/20181207/11242241179/atlanta-cops-caught-deleting-body-cam-footage-failing-to-activate-recording-devices.shtml" href="https://www.techdirt.com/articles/20181207/11242241179/atlanta-cops-caught-deleting-body-cam-footage-failing-to-activate-recording-devices.shtml" rel="noopener noreferrer" target="_blank">
    数据被删除或错误
   </a>
   标记。
  </p>
  <p class="graf graf--p">
   一份2018年的报告发现，
   <strong class="markup--strong markup--p-strong">
    大多数警方致命枪击事件中采取的身体相机镜头从未被公开过。截至去年6月，纽约警察局
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://brooklyneagle.com/wp-content/uploads/2019/07/20190710_boardmtg_BWC_memo-2-1.pdf" href="https://brooklyneagle.com/wp-content/uploads/2019/07/20190710_boardmtg_BWC_memo-2-1.pdf" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     已经积压了将近800个申请查看镜头的请求
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   现在，一些警务改革的拥护者认为，最近的技术进步意味着
   <strong class="markup--strong markup--p-strong">
    这些摄像机已越来越多地被用于监视公众，而非监督警察。
   </strong>
  </p>
  <p class="graf graf--p">
   录制的素材会被上传到所谓的云端，从而使警察可以保存更多的图像和视频，并保留更长的时间。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    还有物体识别功能，使警察可以在长达数小时的录像中通过快速搜索以找到感兴趣的任何物品（例如 “红色背包”）
   </strong>
   。
  </p>
  <p class="graf graf--p">
   通过串流直播，警察们可以将他们看到的一切
   <strong class="markup--strong markup--p-strong">
    几乎立即发送回总部
   </strong>
   。
  </p>
  <p class="graf graf--p">
   从理论上讲，曾经笨拙、缓慢且容易出错的
   <strong class="markup--strong markup--p-strong">
    面部识别现在可以从理论上将任何警员胸前佩戴的身体相机转变为复杂的监视设备
   </strong>
   —— 以识别路过的任何人。
  </p>
  <p class="graf graf--p">
   在公民自由组织集结起来阻止执法部门在旧金山和奥克兰等城市使用面部识别之后，加利福尼亚州也禁止了在身体相机中使用这项技术。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%a6%81%e6%ad%a2%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e5%8f%af%e4%bb%a5%e5%81%9a%e5%88%b0%e4%bb%80%e4%b9%88%e3%80%81%e4%b8%8d%e8%83%bd%e5%81%9a%e5%88%b0%e4%bb%80%e4%b9%88/" href="https://www.iyouport.org/%e7%a6%81%e6%ad%a2%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e5%8f%af%e4%bb%a5%e5%81%9a%e5%88%b0%e4%bb%80%e4%b9%88%e3%80%81%e4%b8%8d%e8%83%bd%e5%81%9a%e5%88%b0%e4%bb%80%e4%b9%88/" rel="noopener noreferrer" target="_blank">
     禁止面部识别可以做到什么、不能做到什么
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   美国公民自由联盟（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.aclu.org/blog/privacy-technology/surveillance-technologies/california-just-blocked-police-body-cam-use-face" href="https://www.aclu.org/blog/privacy-technology/surveillance-technologies/california-just-blocked-police-body-cam-use-face" rel="noopener noreferrer" target="_blank">
    ACLU
   </a>
   ）赞扬制止身体相机的法律，他们指出：
   <strong class="markup--strong markup--p-strong">
    身体相机已经 “被转变为可以追踪我们的面部、声音甚至是我们独特行走方式的移动监视设备”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   现在，警察问责制激进主义者与隐私激进主义者背道而驰了。
  </p>
  <p class="graf graf--p">
   隐私激进主义者担心不可避免的拉网监视会导致任何人都有被扫描、识别以及与犯罪或移民数据库相匹配的风险，而这些人甚至完全没有犯罪嫌疑。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “对于我们来说，戴在身上的相机是一种积极的产品。从倡导者的角度来看，情况要复杂得多。” 平民投诉审查委员会负责人 Jonathan Darche 说，该委员会调查了针对纽约警察局警官的不当行为指控。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “ 身体相机使我们能够比没有录像时更频繁地发现任何特定事件发生的情况。通过它，我们可以确定究竟是使用了 … 谨慎的态度还是使用了不适当的威胁，而这对于我们的机构而言是真正的游戏规则改变者。”
  </p>
  <p class="graf graf--p">
   CCRB 的首要目标是警察问责制。当某人对警察的不当行为提出指控时，该委员会将通过审查病历、获取视频证据，并与投诉人、作出回应的官员和潜在证人的访谈来进行调查。
  </p>
  <p class="graf graf--p">
   公用CCTV摄像机的质量可能很差，很少有音频，拍摄质量和声音对于一个声称口头骚扰或威胁的案例来说至关重要。
  </p>
  <p class="graf graf--p">
   旁观者通常仅在发现问题后才开始使用自己的手机录制视频，因此有可能错过有用的环境信息。
  </p>
  <p class="graf graf--p">
   另一方面，身体相机会自动保存警官按下“记录”之前30秒钟的镜头，Darche 说，这一进展对调查产生了很大影响。
  </p>
  <p class="graf graf--p">
   他说：“我们能够在人们意识到该事件需要被记录之前看到发生了什么。”
  </p>
  <p class="graf graf--p">
   但是，隐私权和移民权利倡导者则认为，有些时候，
   <strong class="markup--strong markup--p-strong">
    身体相机可以带来了解，却不能带来正义。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “您可以围绕 Eric Garner 案思考这件事，”Make the Road New York 的组织者 Kesi Foster 说，该组织着重于移民的权利和警察的责任。
  </p>
  <p class="graf graf--p">
   他说：“每个人都能看到该视频中发生的事，” 他指的是纽约警察局局长 Daniel Pantaleo 的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theatlantic.com/national/archive/2014/12/context-for-the-punishment-free-killing-of-eric-garner/383413/" href="https://www.theatlantic.com/national/archive/2014/12/context-for-the-punishment-free-killing-of-eric-garner/383413/" rel="noopener noreferrer" target="_blank">
    病毒式传播录像
   </a>
   ，他使 Eric Garner 处于
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theatlantic.com/politics/archive/2015/05/the-election-in-which-eric-garners-death-didnt-matter/392525/" href="https://www.theatlantic.com/politics/archive/2015/05/the-election-in-which-eric-garners-death-didnt-matter/392525/" rel="noopener noreferrer" target="_blank">
    致命的窒息状态
   </a>
   ， “尽管如此，仍然需要花五年的时间、在组织和社区的压力下…… 要求一名官员负责。”
  </p>
  <p class="graf graf--p">
   而且，越来越多的人担心身体相机被用来监视市民。
  </p>
  <p class="graf graf--p">
   最近，电子前沿基金会（EFF）向海关和边境保护局发布了信息自由请求（RFI），表明其对
   <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/6488270-CBP-BWC-and-FR-RFI-10-16-2019.html#document/p19/a531331" href="https://www.documentcloud.org/documents/6488270-CBP-BWC-and-FR-RFI-10-16-2019.html#document/p19/a531331" rel="noopener noreferrer" target="_blank">
    配备面部识别功能的身体相机
   </a>
   的兴趣。
  </p>
  <p class="graf graf--p">
   摄像头可以将人的面部与预先存在的图像数据库（例如，大头照或驾驶执照照片）实时匹配，例如，使 CBP 官员
   <strong class="markup--strong markup--p-strong">
    几乎可以通过扫描一个途经的人来即时确认此人的身份。
   </strong>
  </p>
  <p class="graf graf--p">
   面部识别技术的广泛采用意味着只要你出门，就会面临被扫描的风险 —— 这可能对生活在CBP例行巡逻的边境社区的移民产生巨大的寒蝉效应。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    CBP 等生物识别数据库也吸引了黑客，他们可以在暗网上出售有价值的信息。
   </strong>
  </p>
  <p class="graf graf--p">
   3月，美国海关边防总署宣布，黑客从分包商那里
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theatlantic.com/technology/archive/2019/06/travelers-images-stolen-attack-cbp/591403/" href="https://www.theatlantic.com/technology/archive/2019/06/travelers-images-stolen-attack-cbp/591403/" rel="noopener noreferrer" target="_blank">
    窃取了数千名旅行者的身份证照片和车牌图像
   </a>
   。
  </p>
  <p class="graf graf--p">
   如果CBP的数据库扩展到包括从其提议的身体相机项目中生成的图像，那么将
   <strong class="markup--strong markup--p-strong">
    有更多的人容易受到安全漏洞的攻击
   </strong>
   。
  </p>
  <p class="graf graf--p">
   EFF 的报告是提醒联邦机构试图将面部识别技术应用于其身体相机的首次迹象。没有CBP采用配备面部识别功能的身体相机的时间表，也没有迹象表明是否有任何公司对RFI做出回应。⚪️
  </p>
