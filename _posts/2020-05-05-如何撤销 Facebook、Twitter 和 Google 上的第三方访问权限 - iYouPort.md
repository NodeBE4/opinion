  <p class="graf graf--p">
   重申：这不是一次性工作，是需要您定期检查的。
  </p>
  <p class="graf graf--p">
   如果您懒得使用电子邮件地址注册服务，则可以使用社交媒体或Google帐户注册这些服务。那些按钮就摆在那里，基本所有人都知道它们。
  </p>
  <p class="graf graf--p">
   然后，这些网站与 Facebook、Google、Twitter 等寡头服务联系，授予您身份验证令牌，同时提供你的基本详细信息，例如你的姓名、照片和电子邮件地址等等。
  </p>
  <p class="graf graf--p">

   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11064" data-recalc-dims="1" height="634" sizes="(max-width: 646px) 100vw, 646px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-37.png?resize=646%2C634&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-37.png?w=646&amp;ssl=1 646w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-37.png?resize=300%2C294&amp;ssl=1 300w" width="646"/>

  </p>
  <p class="graf graf--p">
   问题在于，一旦您通过门户将社交媒体帐户授予第三方网站就可能带来风险。如果该网站有不良意图怎么办？如果他们尝试获取更多的用户信息（例如密码）怎么办？
  </p>
  <p class="graf graf--p">
   这就是为什么您必须定期检查哪些网站有权访问您的社交媒体帐户或Google帐户的原因。
  </p>
  <p class="graf graf--p">
   如果您不再使用该网站，请立即撤消其访问权限。以下是具体方法。
  </p>

   <span style="color: #ff6600;">
    <strong>
     Facebook
    </strong>

   <span style="color: #ff6600;">
    <strong>
     Twitter
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   登录您的 Twitter 帐户，然后在“设置”中
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/settings/applications" href="https://twitter.com/settings/applications" rel="noopener noreferrer" target="_blank">

   自几个月前的清理以来，我的 Twitter 帐户中又积累了多少。也是为什么说您应该定期做这项工作。
  </p>
  <p class="graf graf--p">
   只需单击要撤消的站点，然后就像 Facebook 那样，您将看到该网站的详细信息。
  </p>
  <p>

   点击底部的“撤消访问权限”即可。
  </p>
  <p class="graf graf--p">
   如果您突然改变主意，Twitter 不会像 Facebook 那样要求您确认，而是将撤消链接转换为 “撤消撤消访问权限” 链接。只需单击该按钮，即可将一切恢复原状。
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   返回您的“应用程序”页面，重复这套操作以删除您要删除的其他第三方。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong>
     Google
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   Google 要求您登录帐户，您可以
   <a class="markup--anchor markup--p-anchor" data-href="https://myaccount.google.com/u/0/security" href="https://myaccount.google.com/u/0/security" rel="noopener noreferrer" target="_blank">
    在此处
   </a>

  </p>

  <p class="graf graf--p">
