   击毙睡眠者
  </h1>

 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <strong>
      <em>
       一位20岁的说唱歌手，在路边的汽车里睡觉，然后，被警方击毙了…… Willie Bo 的遇害事件引发了很多丑闻的曝光
      </em>
     </strong>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   加利福尼亚州 Vallejo 的六名警察开枪打死了20岁的说唱歌手 Willie Bo，当时他正在塔可钟的车里睡觉。
  </p>
  <p class="graf graf--p">
   根据“卫报”的报道，在塔可钟的一名员工打电话报告一名男子在车上“瘫倒”后，警方到场了，
  </p>
  <p class="graf graf--p">
   六名警察向这名睡觉的歌手开了“数枪”。McCoy 的家人周二表示，警方对这位年轻的黑人严重地种族歧视，没有任何理由对一个正在睡觉的人使用致命武力。
  </p>
  <p class="graf graf--p">
   Willie 的哥哥 Marc McCoy 告诉“卫报”说：“并没有试图找到和平解决方案”；“警方的工作是逮捕违法者 — — 不要把法律掌握在自己手中。你不是法官、陪审团和刽子手……我们永远不会原谅这种事。“
  </p>
  <p class="graf graf--p">
   警方称，六名警官都在大约四秒钟内开枪，但他们没有透露袭击 Willie Bo 的子弹数量。在将他从车上移走并“提供医疗援助”之前，警察继续对他高喊。但他在现场就死了。警方尚未证实他的身份，并表示尸检“正在审理中”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Willie Bo 遇害事件引发了多次丑闻，涉及警方对旧金山东北30英里的瓦莱霍（Vallejo）黑人居民的野蛮暴行
   </strong>
   。
  </p>
  <p class="graf graf--p">
   一段
   <a class="markup--anchor markup--p-anchor" data-href="https://abc7.com/news/chief-responds-to-vallejo-police-brutality-accusations/1799465/" href="https://abc7.com/news/chief-responds-to-vallejo-police-brutality-accusations/1799465/" rel="noopener noreferrer" target="_blank">
    2017年的视频
   </a>
   拍摄了一名 Vallejo 警察摁倒一名男子并反复殴打他的脸部。另一起
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nbcbayarea.com/news/local/Vallejo-Police-Response-Video-Capturing-Suspect-Arrest-491790901.html" href="https://www.nbcbayarea.com/news/local/Vallejo-Police-Response-Video-Capturing-Suspect-Arrest-491790901.html" rel="noopener noreferrer" target="_blank">
    暴力逮捕
   </a>
   事件于去年传播开来，有一段视频显示一名军官用警棍猛烈
   <a class="markup--anchor markup--p-anchor" data-href="https://www.mercurynews.com/2018/08/25/vallejo-police-defend-officers-use-of-force-captured-in-video/" href="https://www.mercurynews.com/2018/08/25/vallejo-police-defend-officers-use-of-force-captured-in-video/" rel="noopener noreferrer" target="_blank">
    殴打倒在地面上
   </a>
   的男子。上个月，一名海军陆战队老兵
   <a class="markup--anchor markup--p-anchor" data-href="https://www.sfchronicle.com/crime/article/Police-investigate-video-of-alleged-assault-of-a-13579568.php" href="https://www.sfchronicle.com/crime/article/Police-investigate-video-of-alleged-assault-of-a-13579568.php" rel="noopener noreferrer" target="_blank">
    指控一名 Vallejo 警官
   </a>
   在他拍摄他的堂兄被捕时袭击了他。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “没有人相信 Vallejo 的警察，” Willie Bo 的堂兄说，“我们正在成为被攻击的目标……警方开展了一项运动，以确定符合当局所称的某种形象的年轻黑人。Willie 扮成了这个角色。他代表着嘻哈音乐。该形象被警察关注着。“
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    关于这一“关注明白”为什么会存在的更多详细报道：《
    <a class="markup--anchor markup--li-anchor" data-href="https://medium.com/@iyouport/hr-a388e3ff004c" href="https://medium.com/@iyouport/hr-a388e3ff004c" rel="noopener noreferrer" target="_blank">
     “帮派数据库”是什么？为什么据此就可以直接给无辜的市民定下重罪？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   Willie 的童年很艰难，年纪轻轻就失去了父母：“他对音乐非常有抱负。他充满激情。“ Willie 在一个名为 FBG 的团体中做歌手。Vallejo 是许多成功的说唱歌手和音乐家的故乡。
  </p>
  <p class="graf graf--p">
   已经有两起引人注目的案件，都是警察在受害者醒来后致命地开枪，引发民事诉讼。据旧金山纪事报
   <a class="markup--anchor markup--p-anchor" data-href="https://www.sfchronicle.com/crime/article/Family-of-man-killed-in-Oakland-police-shooting-13595064.php" href="https://www.sfchronicle.com/crime/article/Family-of-man-killed-in-Oakland-police-shooting-13595064.php" rel="noopener noreferrer" target="_blank">
    报道
   </a>
   ，去年，奥克兰的四名警官射杀了一名无家可归的男子，当时他正在两间房子之间睡觉，这引发了上周家属提起的过度武力诉讼。2015年，奥克兰警察杀死了一名在车内昏倒的男子。他的家人获得了120万美元的和解费。
  </p>
  <p class="graf graf--p">
   Willie 的家人表示，涉案人员应该面临刑事起诉。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们无法理解他们为什么要射杀他。这是毫无意义的……枪杀一个在车里睡觉的人！我们希望确保这种情况再也不会发生在另一个人身上。”
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    S
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/us-news/2019/feb/12/california-police-shooting-willie-mccoy?CMP=Share_AndroidApp_Google%2B" href="https://www.theguardian.com/us-news/2019/feb/12/california-police-shooting-willie-mccoy?CMP=Share_AndroidApp_Google%2B" rel="noopener noreferrer" target="_blank">
    ix California officers fire shots at rapper who had been asleep in car, killing him
   </a>
   . Willie McCoy was startled awake and fatally shot by Vallejo police at Taco Bell, prompting family to allege racial profiling
  </p>
