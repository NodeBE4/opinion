  <p class="graf graf--p">
   卫报发表了一封来自152名政治家、律师、学者、记者和活动家的联署公开信，呼吁警方停止将示威者列为“国内极端分子”。
  </p>
  <p class="graf graf--p">
   事情是这样的，《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ad%a6%e5%af%9f%e5%b0%86%e6%b4%bb%e5%8a%a8%e5%ae%b6%e7%a7%b0%e4%b8%ba%e5%9b%bd%e5%86%85%e6%9e%81%e7%ab%af%e5%88%86%e5%ad%90%e7%9a%84%e4%ba%94%e4%b8%aa%e8%8d%92%e8%b0%ac%e7%90%86/" href="https://www.iyouport.org/%e8%ad%a6%e5%af%9f%e5%b0%86%e6%b4%bb%e5%8a%a8%e5%ae%b6%e7%a7%b0%e4%b8%ba%e5%9b%bd%e5%86%85%e6%9e%81%e7%ab%af%e5%88%86%e5%ad%90%e7%9a%84%e4%ba%94%e4%b8%aa%e8%8d%92%e8%b0%ac%e7%90%86/" rel="noopener noreferrer" target="_blank">
    警察将活动家称为“国内极端分子”的五个荒谬理由
   </a>
   》
  </p>
  <p class="graf graf--p">
   我是签署这封信的人之一 —— 我就是警察称之为国内极端分子的人之一。
  </p>
  <p class="graf graf--p">
   我是一名反军售活动家。
  </p>
  <p class="graf graf--p">
   根据警方档案，我第一次进入所谓的 “国内极端分子”数据库是在1999年，当时我因试图阻止武器交易商前往交易会而被捕。
  </p>
  <p class="graf graf--p">
   我的切身体会足以了解这个标签所造成的损害，以及它应该被废除的原因。
  </p>
  <p class="graf graf--p">
   这封公开信的其他签署者也在那个国内极端主义数据库中。
  </p>
  <p class="graf graf--p">
   其中包括环保活动家
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/politics/2014/jun/15/green-party-peer-put-on-database-of-extremists-by-police" href="https://www.theguardian.com/politics/2014/jun/15/green-party-peer-put-on-database-of-extremists-by-police" rel="external noopener noreferrer" target="_blank">
    Jenny Jones
   </a>
   ，作家兼表演者
   <a class="markup--anchor markup--p-anchor" data-href="https://markthomasinfo.co.uk/domestic-extremist/" href="https://markthomasinfo.co.uk/domestic-extremist/" rel="external noopener noreferrer" target="_blank">
    Mark Thomas
   </a>
   和 John Catt,，John 是唯一一位成功的挑战者。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    John Catt 已经94岁，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.thecanary.co/trending/2019/01/24/a-94-year-old-just-won-an-7-year-legal-battle-over-the-polices-domestic-extremist-database/" href="https://www.thecanary.co/trending/2019/01/24/a-94-year-old-just-won-an-7-year-legal-battle-over-the-polices-domestic-extremist-database/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     在长达七年的法律战后才取得了这一成绩
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。Catt 参加了许多抗议活动，但从来没有刑事定罪，他发现警察已经66次保存了有关他的信息 。
   </strong>
  </p>
  <p class="graf graf--p">
   这封警察监测网络（Netpol）组织的公开信中写道：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    将合法的言论自由和异议归为“国内极端主义”，恐吓和阻止人们参加抗议活动，限制他们行使集会和结社自由权利的能力。这种现象不能继续存在了。
   </em>
  </p>
  <p class="graf graf--p">
   Netpol 组织目前正在开展一项废除国内极端主义标签的活动。
  </p>
  <p class="graf graf--p">
   正如 Netpol 组织的协调员凯文·布洛（Kevin Blowe）在“金丝雀”（Canary）的一篇客座
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ad%a6%e5%af%9f%e5%b0%86%e6%b4%bb%e5%8a%a8%e5%ae%b6%e7%a7%b0%e4%b8%ba%e5%9b%bd%e5%86%85%e6%9e%81%e7%ab%af%e5%88%86%e5%ad%90%e7%9a%84%e4%ba%94%e4%b8%aa%e8%8d%92%e8%b0%ac%e7%90%86/" href="https://www.iyouport.org/%e8%ad%a6%e5%af%9f%e5%b0%86%e6%b4%bb%e5%8a%a8%e5%ae%b6%e7%a7%b0%e4%b8%ba%e5%9b%bd%e5%86%85%e6%9e%81%e7%ab%af%e5%88%86%e5%ad%90%e7%9a%84%e4%ba%94%e4%b8%aa%e8%8d%92%e8%b0%ac%e7%90%86/" rel="noopener noreferrer" target="_blank">
    文章中所指出
   </a>
   的，
   <strong class="markup--strong markup--p-strong">
    活动家们非常容易被贴上国内极端主义标签。
   </strong>
   这是为什么我非常乐意签署 Netpol 公开信的众多原因之一。
  </p>
  <p class="graf graf--p">
   因为在 Blowe 描述的五个典型类别中我一个人就占据了三个：我被视为组织者、参与了挑战公司利益的活动、并参与了非暴力直接行动。
  </p>
  <p class="graf graf--p">
   请注意，这不仅仅意味着你的一切被纳入了一个糟糕的警察数据库，更有，警方会对所谓的“国内极端分子”做些什么。
  </p>
  <p class="graf graf--p">
   我曾多次写在文章中强调过这点。
   <strong class="markup--strong markup--p-strong">
    该标签基本上意味着你成为了警察的重点骚扰对象。意味着任意逮捕。意味着成为任意拦截和搜查的目标。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    多年来，我在参加抗议期间和之后一直在遭受警方的滋扰；警察出现在我家门口；高级警察威胁要‘把我抓进去关很久’。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    ⚠️并且，我最好的朋友之一竟然被发现是
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://domesticextremist.wordpress.com/2012/10/22/we-need-to-talk-about-martin/" href="https://domesticextremist.wordpress.com/2012/10/22/we-need-to-talk-about-martin/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     BAE系统的间谍
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    —— 我身边不同程度的朋友中至少有五名卧底警察。
   </strong>
  </p>
  <p class="graf graf--p">
   警方的暴力和骚扰已经严重影响了我的生活，至少两次危机。
  </p>
  <p class="graf graf--p">
   第一次，
   <strong class="markup--strong markup--p-strong">
    我最终在急诊室里，喝了太多导致吐血、出现危险的低血压，出现幻觉，把医护人员看成了警察……
   </strong>
  </p>
  <p class="graf graf--p">
   第二次
   <strong class="markup--strong markup--p-strong">
    我的身体彻底垮了。我的精神状况已经完了，对身体症状的反应使我不得不停止反抗事业。这是一种极大的折磨。
   </strong>
  </p>
  <p class="graf graf--p">
   并不是说在国内极端主义数据库中整理的信息甚至是准确的。
  </p>
  <p class="graf graf--p">
   我只看到了关于自己的部分文件，就从中发现了多个错误。例如，他们把我
   <strong class="markup--strong markup--p-strong">
    从来没有参加过的抗议活动
   </strong>
   也写入我的名下了。
  </p>
  <p class="graf graf--p">
   更重要的是，信息受到了操纵。
  </p>
  <p class="graf graf--p">
   叙述本身就不是中立的。
   <strong class="markup--strong markup--p-strong">
    它的目的是给我画上一个败坏的形象，作为一个积极的暴力麻烦制造者。
   </strong>
  </p>
  <p class="graf graf--p">
   因此，例如，我的档案中包含了几起涉嫌袭警的事件。但是，并没有写明事实是指控被撤销我被判无罪；也没有写明在那些事件我后成功起诉了警察；我收到的赔偿金以及警方对我们所做的事的道歉都不包括在内。
  </p>
  <p class="graf graf--p">
   该数据库只是警察创造的一个神话；他们创造的图片只是证明了他们的工作和标签，但与现实不符。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这非常危险。因为警方对我的叙述直接影响了我遭受监视的方式、如何进行保释评估、以及我在街头有可能遭遇的危险。它变成了一个恶性循环。
   </strong>
  </p>
  <p class="graf graf--p">
   在 Catt 案的法律挑战之后，Blowe 告诉 The Canary：
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    <em>
     抗议的自由受到不断的、不必要的和侵入性的监视的威胁，裁决承认这种监视会对社会参与产生寒蝉效应。
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p">
   ⚠️确实如此。我们更难以动员人们了。人们告诉我们，由于警方的监视，他们不再参加抗议活动了。
  </p>
  <p class="graf graf--p">
   人们看到了我过去遭受过的警察骚扰，他们害怕自己也遭受同样的事。
  </p>
  <p class="graf graf--p">
   但“令人不寒而栗”的影响则更进一步。
  </p>
  <p class="graf graf--p">
   阅读警方关于你的文件非常令人不寒而栗。当你意识到自己和警察在街上交谈了两分钟的时候，脊背都会发冷。你会怀疑一切都与警察有关。
  </p>
  <p class="graf graf--p">
   但是，⚠️
   <strong class="markup--strong markup--p-strong">
    当我意识到连我儿子的名字也被包含在关于我的警察档案中时，那种寒意直接变成了纯粹的冰。
   </strong>
  </p>
  <p class="graf graf--p">
   我不是罪犯。在那个数据库里我与一些我见过的最勇敢的人并排在一起。我们都没有犯下任何严重的罪行。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    我们被放入这个数据库只是因为我们都相信自己可以争取到一个更美好的世界，我们不能只是坐下来不做任何事就能等到一个美好的世界。
   </strong>
  </p>
  <p class="graf graf--p">
   但这并不是所有的厄运和沮丧。我们还可以反击。
  </p>
  <p class="graf graf--p">
   我们不仅可以反击，我们还要赢得这场战斗。
  </p>
  <p class="graf graf--p">
   就如 Blowe 所说：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    我们被问到，作为一个小型组织，要求警方完全停止对活动家使用“国内极端分子”标签，这在很大程度上是象征性的，我们是否真的有机会实现这种改变？
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    我坚信我们可以赢得这一点。仅仅内政部否定这个标签是不够的，我们必须更进一步，
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     我们停止警察利用反恐的借口打击异议
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    。反恐和针对合法政治异议的攻击之间不应该有任何联系。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    我们还需要更好的法律保护，让活动家能够避免在抗议活动中遭受监视。现在的警察几乎可以随心所欲。
   </em>
  </p>
  <p class="graf graf--p">
   我们可以阻止警察做“他们想要的任何事”。
   <strong class="markup--strong markup--p-strong">
    每个想要生活在真正民主中的人都应该参与这场运动。
   </strong>
  </p>
  <p class="graf graf--p">
   任何参加抗议的人都不应该遭受警察的监视和骚扰。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    抗议不是恐怖主义。试图让世界变得更美好的人不应该在警察的数据库里。
   </strong>
   ⚪️
  </p>
