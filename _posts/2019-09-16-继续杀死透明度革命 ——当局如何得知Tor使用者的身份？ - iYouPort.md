  <p class="graf graf--p">
   显然，政府对惩罚人民的兴趣远高于对自己的错误进行反思和修正的兴趣。司法部正在起诉另一位举报人。
  </p>
  <p class="graf graf--p">
   空军和国家安全局的一名前语言分析师被指控泄露文件，
   <strong class="markup--strong markup--p-strong">
    这些文件
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://shadowproof.com/2019/05/09/trump-administration-charges-alleged-drone-whistleblower-with-violating-espionage-act/" href="https://shadowproof.com/2019/05/09/trump-administration-charges-alleged-drone-whistleblower-with-violating-espionage-act/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     详细说明了政府无人机暗杀计划
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    影响
   </strong>
   。文件是泄露给独立媒体 intercept 的。
  </p>
  <p class="graf graf--p">
   31岁的 David Hale 是最近遭到司法部起诉的举报人。
  </p>
  <p class="graf graf--p">
   新闻自由基金会（Freedom of the Press Foundation）
   <a class="markup--anchor markup--p-anchor" data-href="https://freedom.press/news/trump-admin-pace-shatter-record-most-prosecutions-journalistic-sources/" href="https://freedom.press/news/trump-admin-pace-shatter-record-most-prosecutions-journalistic-sources/" rel="noopener noreferrer" target="_blank">
    评论指
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    此案使得特朗普政府成为了使用所谓的“间谍罪”起诉记者之消息来源的最大黑手。
   </strong>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    特朗普的司法部已经逮捕并指控前情报分析师 Daniel Everette Hale 因涉嫌与记者分享所谓的“国家安全信息”。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    在执政两年多的时间里，Hale 至少是
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     被特朗普政府指控的第六个新闻来源
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    ！
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    司法部此前曾表示
   </em>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2017/08/04/us/politics/jeff-sessions-trump-leaks-attorney-general.html" href="https://www.nytimes.com/2017/08/04/us/politics/jeff-sessions-trump-leaks-attorney-general.html" rel="noopener noreferrer" target="_blank">
    <em class="markup--em markup--p-em">
     正在进行
    </em>
    <strong class="markup--strong markup--p-strong">
     <em class="markup--em markup--p-em">
      数十
     </em>
    </strong>
   </a>
   <em class="markup--em markup--p-em">
    个“泄密调查”……
   </em>
  </p>
  <p class="graf graf--p">
   指控称：Hale 交给独立媒体 The Intercept 的文件为该网站的多部分“无人机暗杀报告”提供了基础。
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/drone-papers/" href="https://theintercept.com/drone-papers/" rel="noopener noreferrer" target="_blank">
    在这里看到*无人机暗杀报告*
   </a>
   ，非常惊人的揭露。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    这些无人机文件揭露了政府的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.techdirt.com/articles/20150427/16430130816/cia-will-keep-killing-civilians-with-drone-strikes-because-rules-drone-strikes-arent-actually-rules.shtml" href="https://www.techdirt.com/articles/20150427/16430130816/cia-will-keep-killing-civilians-with-drone-strikes-because-rules-drone-strikes-arent-actually-rules.shtml" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     法外、域外杀戮
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    计划惊人的广度。
   </strong>
  </p>
  <p class="graf graf--p">
   这包括在也门和索马里开展的行动，尽管不同地区在情报和监视方面存在长期差距，但是，在有针对性的杀戮方面，都是一样的。
  </p>
  <p class="graf graf--p">
   在这里看到两份早期的报告，关于直接的暗杀计划《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%B8%AD%E4%B8%9C%E5%90%9B%E4%B8%BB%E5%88%B6%E5%A6%82%E4%BD%95%E9%9B%87%E4%BD%A3%E7%BE%8E%E5%9B%BD%E6%88%98%E5%A3%AB%E4%BB%A5%E6%9D%80%E6%AD%BB%E5%85%B6%E6%94%BF%E6%95%8C-%E8%BF%99%E5%A4%A7/" href="https://www.iyouport.org/%E4%B8%AD%E4%B8%9C%E5%90%9B%E4%B8%BB%E5%88%B6%E5%A6%82%E4%BD%95%E9%9B%87%E4%BD%A3%E7%BE%8E%E5%9B%BD%E6%88%98%E5%A3%AB%E4%BB%A5%E6%9D%80%E6%AD%BB%E5%85%B6%E6%94%BF%E6%95%8C-%E8%BF%99%E5%A4%A7/" rel="noopener noreferrer" target="_blank">
    中东君主制如何雇佣美国战士以杀死其政敌? — 这大概就是战争未来的样子
   </a>
   》；关于间接的暗杀计划（仅举一例）《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%bb%ba%e7%ab%8b%e4%b8%80%e4%b8%aa%e9%97%b4%e8%b0%8d%e5%b8%9d%e5%9b%bd/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%bb%ba%e7%ab%8b%e4%b8%80%e4%b8%aa%e9%97%b4%e8%b0%8d%e5%b8%9d%e5%9b%bd/" rel="noopener noreferrer" target="_blank">
    雄厚的财力，深藏不露：美国间谍如何协助专制国家构建一个间谍帝国
   </a>
   》。
   <strong class="markup--strong markup--p-strong">
    这两份报告将责任人集中在“独立个人和团体”身上，显然是不完整的 —— Hale 揭露的真相则显示，暗杀是*正规任务*
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️并且，这些文件还显示军方将无人机袭击的所谓附带损害 —— 也就是杀死老百姓 —— 称为“在行动中遇害的敌人”，
   <strong class="markup--strong markup--p-strong">
    并未核实
   </strong>
   每个死者是否真的是战斗人员。（“附带伤害”是对平民死亡的常规辩词）
  </p>
  <p class="graf graf--p">
   Hale 的案例在被起诉的所有举报人中并不常见，因为
   <strong class="markup--strong markup--p-strong">
    对他的调查是更公开的方式推出的，相比大多数被起诉者。显然，这里面存在杀鸡的预期
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Hale 出现在无人机战争
   <a class="markup--anchor markup--p-anchor" data-href="https://kcts9.org/programs/independent-lens/episodes/1815" href="https://kcts9.org/programs/independent-lens/episodes/1815" rel="noopener noreferrer" target="_blank">
    纪录片“National Bird”
   </a>
   中，他在其中讨论了他是多么的不情愿参加当局的情报收集活动，这些情报是无人机空袭的根据，他还谈到了他从事情报工作的风险。
  </p>
  <p class="graf graf--p">
   ⚠️此处提请中国的独立纪录片制作者注意，在揭露真相的过程中一定要时刻注意保护信息来源的人身安全 —— 如果必须采访，不仅要做到脸部遮盖，而且要注意声纹编辑。
  </p>
  <p class="graf graf--p">
   Hale 的经历并没有太特殊。⚠️
   <strong class="markup--strong markup--p-strong">
    在2014年该纪录片拍摄的期间，他的家遭到 FBI 的袭击。
   </strong>
  </p>
  <p class="graf graf--p">
   根据
   <a class="markup--anchor markup--p-anchor" data-href="https://assets.documentcloud.org/documents/5996501/Hale-Indictment.pdf" href="https://assets.documentcloud.org/documents/5996501/Hale-Indictment.pdf" rel="noopener noreferrer" target="_blank">
    起诉书[PDF]
   </a>
   ，Hale 打印并删除了绝密文件，这些文件始于2013年初 ——
   <strong class="markup--strong markup--p-strong">
    注意这个时间，此时是第一次 Snowden 泄密之前的几个月。
   </strong>
  </p>
  <p class="graf graf--p">
   也就是说，Hale 比斯诺登更早。
  </p>
  <p class="graf graf--p">
   起诉书的时间表似乎是正确的。在关于斯诺登首次与调查记者 Glenn Greenwald 会面的纪录片 “Citizenfour（第四公民）” 中，Greenwald
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/local/public-safety/former-intelligence-analyst-charged-with-leaking-drone-details-to-news-outlet/2019/05/09/19eeb9e2-7258-11e9-8be0-ca575670e91c_story.html?noredirect=on&amp;utm_term=.df3cbf181c2f" href="https://www.washingtonpost.com/local/public-safety/former-intelligence-analyst-charged-with-leaking-drone-details-to-news-outlet/2019/05/09/19eeb9e2-7258-11e9-8be0-ca575670e91c_story.html?noredirect=on&amp;utm_term=.df3cbf181c2f" rel="noopener noreferrer" target="_blank">
    讨论了另一种泄漏源
   </a>
   ，最有可能是 Hale。
  </p>
  <p class="graf graf--p">
   在这里看到这部纪录片：https://vimeo.com/ondemand/citizen4
  </p>

  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    在纪录片“Citizenfour”结束时，Intercept 创始 Glenn Greenwald 告诉斯诺登，Intercept 有一个关于美国无人机计划的新机密信息来源。
   </em>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">

  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">

   <strong class="markup--strong markup--p-strong">
    这份起诉书提到了加密通信和 Tor 的可能用途，但是，似乎并没有阻止政府访问 Hale 发送给其他人的有关泄露文件的文本和聊天记录

  <p class="graf graf--p">
   我们希望能弄清楚，当局是如何在 Tor 中挖到举报人真实身份的。
  </p>
  <p class="graf graf--p">
   根据起诉书，Hale 总共面临五项与所谓的“间谍”相关的指控。令人感到意外并沮丧的是，这份起诉书到达的时间居然这么长……
  </p>
  <p class="graf graf--p">
   ⚠️联邦调查局的袭击发生在将近五年前（2014年8月），但是，起诉书直到今年3月才提出。差不多相隔了近五年了，现在 Hale 陷入了困境，等待司法部的下一步行动。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Hale 所暴露的是一项计划，该计划允许美国仅仅根据收集的情报而不是法庭起诉在世界各地实施暗杀
   </strong>
   。简曰：法外杀戮。
  </p>
  <p class="graf graf--p">
   政府一直在尝试无人机杀人，自从乔治·W·布什总统任期以来，每界政府都在暗杀领域不断持续升级。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    勇敢的举报人提供给我们的是政府从未希望我们拥有的信息
   </strong>
   。
  </p>
  <p class="graf graf--p">
   政府希望将这些事实永远保密 —— 主要是因为政府知道公众肯定不会全心全意地支持在世界各地杀人。
  </p>
  <p class="graf graf--p">
   Hale 不是罪犯。他是举报人，是说真话的人。但政府只针对举报人，政府是在以此来避免其他人继续说真话。
  </p>
