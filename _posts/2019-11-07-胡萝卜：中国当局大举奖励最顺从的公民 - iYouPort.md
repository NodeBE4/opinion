   胡萝卜：中国当局大举奖励最顺从的公民

       距离行刑日还有一个月

    </span>
   </li>

  <p>
   <img alt="" class="aligncenter size-full wp-image-6756 jetpack-lazy-image" data-lazy-sizes="(max-width: 634px) 100vw, 634px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头-3.jpg?resize=634%2C476&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头-3.jpg?w=634&amp;ssl=1 634w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头-3.jpg?resize=300%2C225&amp;ssl=1 300w" data-recalc-dims="1" height="476" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头-3.jpg?resize=634%2C476&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="634"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6756" data-recalc-dims="1" height="476" sizes="(max-width: 634px) 100vw, 634px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头-3.jpg?resize=634%2C476&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头-3.jpg?w=634&amp;ssl=1 634w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头-3.jpg?resize=300%2C225&amp;ssl=1 300w" width="634"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   中国已经提议给予社会信用评分高的公民一大堆优惠待遇，包括让他们优先获得搬到城市找工作的机会。
  </p>
  <p class="graf graf--p">
   新华社援引中央政府的话说，“可靠的”公民可以期待享受快速通道服务，同时可以更方便地为孩子选择幼儿园、寻找护理院和申请专利。
  </p>
  <p class="graf graf--p">
   这是胡萝卜。
  </p>
  <p class="graf graf--p">
   在胡萝卜之前，北
   <strong class="markup--strong markup--p-strong">
    京将超过1300万名“不值得信任”的人列入了黑名
   </strong>
   单，禁止他们乘坐火车和飞机以作为惩罚。
  </p>
  <p class="graf graf--p">
   中国的社会信用体系是北京主导的大型监视计划，由国家不断扩大的监控网络推动，
   <strong class="markup--strong markup--p-strong">
    该监控网络目前拥有2亿台人工智能摄像机
   </strong>
   。
  </p>
  <p class="graf graf--p">
   IYP 自2015年开始一直在提醒警惕，最初的分析详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%8f%88%e4%b8%80%e4%b8%aa%e8%a2%ab%e6%bc%a0%e8%a7%86%e7%9a%84%e6%b7%b1%e6%b8%8a/" href="https://www.iyouport.org/%e5%8f%88%e4%b8%80%e4%b8%aa%e8%a2%ab%e6%bc%a0%e8%a7%86%e7%9a%84%e6%b7%b1%e6%b8%8a/" rel="noopener noreferrer" target="_blank">
    又一个被漠视的深渊
   </a>
   》。但是，近五年了，中国社会没有任何抵抗。
  </p>
  <p class="graf graf--p">
   新华社说，该胡萝卜提案于7月份由中国国家发改委发布。
  </p>
  <p class="graf graf--p">
   中国发改委是社会信用评分体系的监督机构。
  </p>
  <p class="graf graf--p">
   这项全国大规模监视计划将于一个月正式完成并启动，根据每个人从财务记录到社交媒体活动的所有日常行为，对中国公民进行评级。
  </p>
  <p class="graf graf--p">
   中国不断扩大的监控网络为此提供了最强大的支持 —— 该监视网络目前拥有2亿个
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab/" href="https://www.iyouport.org/category/%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab/" rel="noopener noreferrer" target="_blank">
    面部识别
   </a>
   摄像头。
  </p>
  <p class="graf graf--p">
   该提案包括15项激励措施。
  </p>
  <p class="graf graf--p">
   根据该文件，具有较高社会信用评分的人会优先被允许在城市定居。
  </p>
  <p class="graf graf--p">
   搬到城市居住在西方人眼中有可能是很简单的事，不足以吸引人？并不是这样。因为中国有独特的户籍制度，将一个人的身份信息从家乡转移到新城市 —— 这个过程如果没有政府协助是无法做到的。
  </p>
  <p class="graf graf--p">
   那些没有正式“定居”城市的人面临着回到家乡处理一大堆繁琐的行政事务的麻烦。
  </p>
  <p class="graf graf--p">
   根据该提案，那些“值得信赖”的人如果想成为公务员，并且他们拥有与竞争对手相同的经验和资格，那么他们就可以更容易被聘用，而不是其他人。
  </p>
  <p class="graf graf--p">
   在申请抵押贷款或其他金融服务时，“信用评分”高的人也将获得“绿色通道”。
  </p>
  <p class="graf graf--p">
   中国发改委表示，在通过媒体发布该提案后，他们正在寻求社会的反馈。
  </p>
  <p class="graf graf--p">
   这是中国当局公布社会信用评分体系以来第一次发送胡萝卜。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6757 jetpack-lazy-image" data-lazy-sizes="(max-width: 634px) 100vw, 634px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-8.jpg?resize=634%2C383&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-8.jpg?w=634&amp;ssl=1 634w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-8.jpg?resize=300%2C181&amp;ssl=1 300w" data-recalc-dims="1" height="383" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-8.jpg?resize=634%2C383&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="634"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6757" data-recalc-dims="1" height="383" sizes="(max-width: 634px) 100vw, 634px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-8.jpg?resize=634%2C383&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-8.jpg?w=634&amp;ssl=1 634w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-8.jpg?resize=300%2C181&amp;ssl=1 300w" width="634"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   被社会信用评分体系列为“不值得信赖”的中国公民甚至面临着乘坐高速列车的限制。相反，他们将不得不使用慢速列车。
  </p>
  <p class="graf graf--p">
   此前，政府已宣布了一系列惩罚措施，包括限制使用高速列车、飞机以及禁止离开该国。
  </p>
  <p class="graf graf--p">
   根据国家发改委公布的数据，截至3月份，中国有1349万人被列为“不值得信赖”。
  </p>
  <p class="graf graf--p">
   当局已经阻止超过 2047 万次购买机票的尝试、和超过571万次购买高速火车票的尝试。
  </p>
  <p class="graf graf--p">
   今年早些时候，一位名叫 Michelle Ye Xuan 的中国女演员成为了第一个受到该制度惩罚的知名人士。
  </p>
  <p class="graf graf--p">
   这位39岁的女演员和前国际选美女王因拒绝遵守法院命令而被视为“不值得信赖”后被禁止出境。
  </p>
  <p class="graf graf--p">
   在美国长大的 Ye 在二月份即将在北京登机时才发现对她的旅行限制。
  </p>
  <p class="graf graf--p">
   这个社会信用评分得到了一个全国性的数据库的支持，该数据库跟踪和评估该国14亿人口的活动，曾经主要是在金融和法律领域。
  </p>
  <p class="graf graf--p">
   而现在，你不可能知道全部领域都在哪。更多详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%b7%b1%e5%a4%9c%e4%b8%8a%e7%bd%91%e9%83%bd%e4%bc%9a%e9%99%8d%e4%bd%8e%e4%bd%a0%e7%9a%84%e7%a4%be%e4%bc%9a%e4%bf%a1%e7%94%a8%e8%af%84%e5%88%86%ef%bc%9f%e6%97%a0%e4%ba%ba/" href="https://www.iyouport.org/%e6%b7%b1%e5%a4%9c%e4%b8%8a%e7%bd%91%e9%83%bd%e4%bc%9a%e9%99%8d%e4%bd%8e%e4%bd%a0%e7%9a%84%e7%a4%be%e4%bc%9a%e4%bf%a1%e7%94%a8%e8%af%84%e5%88%86%ef%bc%9f%e6%97%a0%e4%ba%ba/" rel="noopener noreferrer" target="_blank">
    “深夜上网”都会降低你的“社会信用评分”？无人能摆脱这种强制性监视
   </a>
   》。
  </p>
  <p class="graf graf--p">
   中国已经有了两亿台人工智能监视摄像机支持这个大规模的监控网络；
   <strong class="markup--strong markup--p-strong">
    两年内摄像机的数量将升至三倍。
   </strong>
  </p>
  <p class="graf graf--p">
   中国的监控网络被称为世界上最强大的面部识别系统之一，旨在在三秒内识别其14亿公民中的任何一个。
  </p>
  <p class="graf graf--p">
   这是政府侵犯公民隐私权和限制一切自由的基本方式。
  </p>
  <p class="graf graf--p">
   发改委称，截至2019年3月，约有359万违约者已经偿还了4.4亿元人民币的债务。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    仅在2018年一年内，就有近360万人被国家的社会信用监管机构所惩罚
   </strong>
   。
  </p>
  <p class="graf graf--p">
   一个月后，这个数字有可能增加到数十倍甚至……⚪️
  </p>
  <p>
   ⚠️⚠️没时间了！抵御社会信用评分，这里是
   <a href="https://twitter.com/DonEvansWm/status/1154422659091845120" rel="noopener noreferrer" target="_blank">
    您现在唯一能做的事
   </a>
   ，详见《
   <a href="https://www.iyouport.org/%e5%88%86%e8%a3%82%e4%ba%ba%e6%a0%bc%e7%9a%84%e9%ad%85%e5%8a%9b/" rel="noopener noreferrer" target="_blank">
    保护措施完整版
   </a>
   》。我们知道，并非所有人都能简单地做到这点，但如果能做到的人中有您，您就是希望。
  </p>
