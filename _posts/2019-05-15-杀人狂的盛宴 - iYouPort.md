   杀人狂的盛宴
  </h1>

 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      卖炸弹杀死老百姓已经不够了，现在他们要转让核技术给流氓政权，这是为了什么？怎样才能拦住这场危机？
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   众议院民主党人指责特朗普政府将可能违反美国法律的高度敏感的核技术卖给沙特。批评人士说，这笔交易可能危及国家安全，同时丰富了美国的亲密盟友 — 战争狂沙特政权。
  </p>
  <p class="graf graf--p">
   据称，沙特阿拉伯正在考虑到2030年建造多达16座核电站，但许多批评者担心该国可以利用这项技术来发展核武器，并在中东引发核军备竞赛。以下是公民权利组织 democracynow 对此做的评论家访谈。
  </p>
  <p class="graf graf--p">
   沙特王储穆罕默德·本·萨勒曼曾经明确表示，如果伊朗能获得核武器，沙特也一样要获得。
  </p>
  <p class="graf graf--p">
   众议院监督委员会发布了一份临时报告，此前已有多名举报人提出指控几位白宫高级顾问和特朗普盟友试图推动核技术转让，尽管有法律和道德警告也完全没能阻止。
   <strong class="markup--strong markup--p-strong">
    争议的核心是IP3公司，该公司旨在帮助美国公司在沙特阿拉伯和该地区的其他地方建造核电站，并将其称为“中东马歇尔计划”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    前国家安全顾问迈克尔弗林在进入白宫之前曾担任该公司子公司的顾问， IP3 由前里根官员 Bud McFarlane 共同创立，他承认参与了1988年的伊朗门事件
   </strong>
   。
  </p>
  <p class="graf graf--p">
   众议院的报告还提到了其他一些特朗普高级官员和盟友，据报道他们参与了拟议的沙特协议，其中包括特朗普的女婿兼顾问贾里德库什纳、能源部长里克佩里、和特朗普的亿万富翁朋友汤姆巴拉克，此人曾担任特朗普就职委员会主席。根据国会委员会的报告，白宫可能仍在讨论该计划。
  </p>
  <p class="graf graf--p">
   这是一个非常严重的问题。这就是为什么人们应该关心。沙特人一直在为也门的基地组织提供武器，而我们作为一个国家能做的最后一件事恐怕就是将核机密转移到沙特了，因为这很可能导致扩散并对我们自己的安全构成威胁。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    正如你所知道的，这里存在很多利益冲突。领导总统就职委员会以筹集1亿美元以上资金的汤姆巴拉克也在推动此项交易，并在该交易中有经济利益
   </strong>
   。因此，监督委员会将进行调查，以了解推动本届政府向沙特人出售核机密的经济利益是什么、以及违反了哪些法律，调查必须来自国会，根据“原子能法”进行调查。
  </p>
  <p class="graf graf--p">
   即使我们将核技术转让给印度等盟国时 — 当乔治布什这样做时，也需要多年的过程。它需要国会的协商。因为有些人可以从这些投资中获得数十亿美元的收益。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “中东马歇尔计划”就是在上述 IP3 中冒出来的，指的是，他们可以通过让美国公司向沙特阿拉伯出口核电技术来加强美国和沙特阿拉伯之间的经济和政治关系。汤姆巴拉克非常乐于接受这个想法，因为他正在推动对该地区进行类似的经济拓展。他与那里的领导人有很多联系。
  </p>
  <p class="graf graf--p">
   最近，政府一直在采取措施，通过正式程序，与沙特进行谈判，达成共享核电技术的正式协议。如果发生这种情况，国会就不必实际对该条约的审批了，生米熟饭。
  </p>
  <p class="graf graf--p">
   你会想到，与沙特阿拉伯相关的所有事 — — 谋杀记者 Jamal Khashoggi、国会决定切断对也门战争的军事援助。已经有很多两党人士对共享核电技术的想法持怀疑态度，并呼吁国会更多地了解这些谈判中发生的事，甚至取消这些谈判。而且你肯定也看到了升级，因为双方成员都说，“我们需要暂停，我们需要获得更多信息。“
  </p>
  <p class="graf graf--p">
   这是特朗普政府如何看待与沙特人关系的世界观。
   <strong class="markup--strong markup--p-strong">
    无论是出售武器还是销售核机密，一切都是关于金钱的。没有道德考虑。最近一周前，汤姆巴拉克就在那里为 Khashoggi 谋杀案辩护，并捍卫沙特政权谋杀 Khashoggi 的行为。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    为什么？只是经济利益而已。为了钱而向沙特出售这些致命的东西，他们不关心我们的安全，也不关心沙特政策的道德
   </strong>
   。
  </p>
  <p class="graf graf--p">
   现在沙特人想要核电。他们正在追求核电。如果美国不帮助他们获得核技术，他们是否会转向俄罗斯或中国？因此，这是一个非常真实的政策问题，安全专家正在努力解决这个问题。
  </p>
  <p class="graf graf--p">
   2018年1月，布鲁克菲尔德资产管理公司的子公司 Brookfield Business Partners 宣布计划以46亿美元收购 Westinghouse Electric。 Westinghouse Electric 是破产的核服务公司，是IP3拟议的在沙特阿拉伯建造核反应堆计划的一部分，并将受益于中东马歇尔计划。 2018年8月，布鲁克菲尔德资产管理公司购买了第五大道666号的合伙股份，这是 Jared Kushner 家族公司所拥有的一栋大楼。
  </p>
  <p class="graf graf--p">
   鉴于 Jared Kushner 很快就要前往该地区，这一点尤为有趣。众议院监督委员会的报告中讨论了关于为库什纳和总统之间的谈话准备材料的问题，显然贾里德库什纳与王储 MBS 建立了非常密切的关系，据称是犹如“in his pocket”。
  </p>
  <p class="graf graf--p">
   我只想说我们需要透明度。但总统和政府可以做的一件事就是与沙特政权保持一定距离，签署战争权力决议，这将阻止美国支持也门的沙特战争罪行。并且需要了解所有其中的利益关系，关于究竟是什么在推动美国和沙特的紧密关系。
  </p>
  <p class="graf graf--p">
   目前有将近1400万的也门人正在面临饥荒，因为沙特人一直在轰炸港口城市，食物和药品不被允许进入。这导致也门恶性通货膨胀。人们没有食物和药物。
  </p>
