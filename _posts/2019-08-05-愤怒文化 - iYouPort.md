  <p class="graf graf--p">
   丹麦存在主义者 SørenKierkegaard 曾经提出，人类经验的吸引力不在于安慰和自满，而在于斗争和自我发现。事实上，人类历史正是由灾难和集体发展的循环所决定的。虽然农作物可能会欠收、定居点可能被洪水吞没、疾病可能会蔓延，但人类依旧有能力重建并再次巩固住。
  </p>
  <p class="graf graf--p">
   科学和技术已经抑制了了人为的和自然灾害的刺痛。但是，
   <strong class="markup--strong markup--p-strong">
    这些进步也减少了关键的社会压力因素的影响，缩减了能将我们聚集在一起的闪点事件。其中一个后果就是愤怒文化。
   </strong>
  </p>
  <p class="graf graf--p">
   在没有自然灾难的情况下，人们创造了人为的灾难。人们认为，进化的心理适应性决定了*共渡难关的需要*。
  </p>
  <p class="graf graf--p">
   愤怒文化简直就是普通文化的祸害。
   <strong class="markup--strong markup--p-strong">
    通过打击不存在的危机，操作者可以通过这个过程轻松地实现团结一致
   </strong>
   。无论是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bbc.co.uk/news/world-us-canada-47326373" href="https://www.bbc.co.uk/news/world-us-canada-47326373" rel="noopener noreferrer" target="_blank">
    制造仇恨犯罪
   </a>
   的恶意行为者、
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theatlantic.com/ideas/archive/2019/01/julie-irwin-zimmerman-i-failed-covington-catholic-test/580897/" href="https://www.theatlantic.com/ideas/archive/2019/01/julie-irwin-zimmerman-i-failed-covington-catholic-test/580897/" rel="noopener noreferrer" target="_blank">
    夸大威胁的记者
   </a>
   ，还是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.thepostmillennial.com/progressive-professors-want-a-blacklist-of-quillette-writers/" href="https://www.thepostmillennial.com/progressive-professors-want-a-blacklist-of-quillette-writers/" rel="noopener noreferrer" target="_blank">
    创造黑名单
   </a>
   的学者，他们激起的群体愤怒都满足了“团结克服共同威胁”的强烈愿望。
  </p>
  <p class="graf graf--p">
   遭受灾难蹂躏的社区很少陷入混乱和沮丧。相反，它们变得越来越平等和相互依赖。
  </p>
  <p class="graf graf--p">
   考虑在第二次世界大战期间的英国人和德国人。
  </p>
  <p class="graf graf--p">
   在德国空军的轰炸下，英国政府预计平民士气将崩溃。但随着闪电战的进展，精神病院的入院率下降了。家庭情报报告发现，神经冲击案例仅占所有突袭伤亡人数的5％。一家伦敦医院报告平均每周只有两例“炸弹神经官能症”。
  </p>
  <p class="graf graf--p">
   然而，更引人注目的是社会团结的增加。虽然英国社会的特点是阶级差异，但这种差异在德国斯图卡的战争中消失了。
   <strong class="markup--strong markup--p-strong">
    在战争前从未互动过的人们此时建立了温暖的关系，在讨论他们的恐惧和愿望的同时分享资源
   </strong>
   。在家庭，企业，火车站和街道 — 无论人们聚集在哪里 — 都存在着一种共同体的氛围。特别是英国的酒吧充斥着谈话和歌曲，充斥着渴望释放宣泄的顾客。看来，每个人都有一个“战争故事”。
  </p>
  <p class="graf graf--p">
   但就像闪电战一样糟糕，与盟军轰炸机对德国城市所做的一切相比，
   <strong class="markup--strong markup--p-strong">
    它变得苍白无力
   </strong>
   。到1945年3月，柏林的住宅中心已经减少成了瓦砾。在德累斯顿，燃烧弹造成了火焰墙，吸收了如此多的氧气，以至于那些没有受到爆炸伤害的人最终死于窒息。
  </p>
  <p class="graf graf--p">
   德累斯顿在一夜之间失去的人数超过伦敦在整个战争期间的死亡人数。但值得注意的是，盟军的
   <a class="markup--anchor markup--p-anchor" data-href="http://udspace.udel.edu/handle/19716/1325" href="http://udspace.udel.edu/handle/19716/1325" rel="noopener noreferrer" target="_blank">
    报告显示
   </a>
   德国“士气”在被轰炸最多的城市中仍然是最高的。
  </p>
  <p class="graf graf--p">
   战争疲惫期间的团结和复原的情况并不少见。 1897年，社会学家埃米尔·迪尔凯姆（Emile Durkheim）就
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amazon.co.uk/Suicide-Emile-Durkheim/dp/1607966425" href="https://www.amazon.co.uk/Suicide-Emile-Durkheim/dp/1607966425" rel="noopener noreferrer" target="_blank">
    观察到
   </a>
   欧洲国家在战争和革命中的自杀率显著下降。在第二次石勒苏益格战争（1864年）和普鲁士战争（1866年）中，自杀率分别下降了16％和14％。
  </p>
  <p class="graf graf--p">
   爱尔兰心理学家H.A.进一步证实了这些发现。 1969年北爱尔兰骚乱期间，莱恩斯
   <a class="markup--anchor markup--p-anchor" data-href="https://www.ncbi.nlm.nih.gov/pmc/articles/PMC1787267/" href="https://www.ncbi.nlm.nih.gov/pmc/articles/PMC1787267/" rel="noopener noreferrer" target="_blank">
    报道
   </a>
   贝尔法斯特自杀事件减少了50％。
  </p>
  <p class="graf graf--p">
   奇怪的是，自然灾害的幸存者表现得与此类似。
  </p>
  <p class="graf graf--p">
   在对1952年白县龙卷风的一项研究中，国家舆论研究中心发现，53％的居民报告了个人性格方面的积极变化。事实上，Charles Fritz 在审查美国对自然灾害的社会反应时，没有发现灾难后持续出现恐慌的情况。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    是什么解释了灾难和团结之间的这种联系？
   </strong>
  </p>
  <p class="graf graf--p">
   根据 Fritz 的说法，灾难创造了一个“受害者社区”；个人统一的超越目标：生存。
   <strong class="markup--strong markup--p-strong">
    为克服灾难的危险和重新稳定社会生活而进行的斗争为日常生活中缺乏的人类活动提供了一种结构和目的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   而且，
   <strong class="markup--strong markup--p-strong">
    灾难实现了法律所不能实现的：平等。因为灾难在它们不分青红皂白地杀害的能力方面具有独特的平等主义，它们会削弱所有使我们与众不同的品质。财富、种族、政治派别，这些平日里人们最关注的东西在炸弹和飓风面前根本不存在。
   </strong>
  </p>
  <p class="graf graf--p">
   当人们聚集在一起打击存在的威胁时，他们会暂停种族、宗教和意识形态的差异。在这些条件下，我们可以使用悲剧作为*共同的参考框架*，自由地相互作用。
  </p>
  <p class="graf graf--p">
   灾难也具有神奇的原始性，它们重现了人类祖先进化的共同条件。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cep.ucsb.edu/codirectors/" href="https://www.cep.ucsb.edu/codirectors/" rel="noopener noreferrer" target="_blank">
    进化心理学家告诉我们
   </a>
   ，人类行为和认知是永恒进化过程的产物。我们的大脑和行为是由我们的狩猎采集者祖先所经历的环境和选择压力所塑造的。这些进化的程序中的每一个都存在，因为它们促进了一种增强早期人类生存和生殖健康的行为。
  </p>
  <p class="graf graf--p">
   例如，人类似乎在
   <a class="markup--anchor markup--p-anchor" data-href="https://onlinelibrary.wiley.com/doi/pdf/10.1111/j.1467-7687.2008.00753.x" href="https://onlinelibrary.wiley.com/doi/pdf/10.1111/j.1467-7687.2008.00753.x" rel="noopener noreferrer" target="_blank">
    进化上
   </a>
   已经准备好了害怕蛇和蜘蛛。我们的一些祖先看到了这些生物，并以恐惧和回避的方式回应。其他人则没有。那些幸存下来的人通过这个生存系统（见到蛇跑开）向后代遗传。
  </p>
  <p class="graf graf--p">
   我们的大脑发展为从环境中提取信息，然后用它来规范我们的行为。随着时间的推移，这些信息会产生专门的认知程序来指导我们的行为。
  </p>
  <p class="graf graf--p">
   这表明灾难与团结之间的关联决不是随机的。我们擅长克服危机，因为我们已经这样做了数千年。合作是关键。
  </p>
  <p class="graf graf--p">
   合作不是新发明的玩意，它是建立在数千年的进化试验和错误经验基础上的强大的生存机制。这是一种适应性行为。
  </p>
  <p class="graf graf--p">
   适应行为是改善个体及其亲属繁殖成功的策略。此外，它们也是对适应性问题的回应。灾难是合作演变为克服的一个主要的适应性问题。事实上，正是这种灾难与合作之间的相互作用决定了我们过去的进化。
  </p>
  <p class="graf graf--p">
   我们的早期祖先生活在约150人的小型亲属社区中。受到贪婪的掠食者和令人讨厌的疾病的折磨，人类通过分享食物和交易恩惠而幸免于难。
  </p>
  <p class="graf graf--p">
   考虑 Hill 和 Kaplan 的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.jstor.org/stable/2743137?seq=1#metadata_info_tab_contents" href="https://www.jstor.org/stable/2743137?seq=1#metadata_info_tab_contents" rel="noopener noreferrer" target="_blank">
    风险汇总假说
   </a>
   。作者发现巴拉圭东部的 Ache 避免了由于存在稀缺食物的风险而导致的饥饿。虽然蔬菜可以由个别家庭种植，但是在集体层面上共享稀有的营养密集的食物，如肉和蜂蜜。这确保了当供应量低时，部落内的所有人也都可以吃东西。
  </p>
  <p class="graf graf--p">
   研究人员还表示，我们与掠食者的互动塑造了人类合作的冲动。考古记录中充斥着致命的掠食者，他们既猎杀我们的祖先又与他们争夺食物。捕食风险和资源竞争的这种结合促进了早期人类的社会性。特别是，我们的祖先采用了资源防御模式，将肉类运送到拥有水和植物等固定资源的“家庭基地”。
  </p>
  <p class="graf graf--p">
   合作使我们与最亲近的进化亲属区别开来。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amazon.com/Sapiens-Humankind-Yuval-Noah-Harari/dp/0062316117/ref=sr_1_1?keywords=sapiens&amp;qid=1553344332&amp;s=gateway&amp;sr=8-1" href="https://www.amazon.com/Sapiens-Humankind-Yuval-Noah-Harari/dp/0062316117/ref=sr_1_1?keywords=sapiens&amp;qid=1553344332&amp;s=gateway&amp;sr=8-1" rel="noopener noreferrer" target="_blank">
    Sapiens
   </a>
   的作者 Yuval Harari 指出，一只黑猩猩会击败一个人类，但是1000个人类会击败1000只黑猩猩。这是因为人类的合作技能远远超过黑猩猩的合作技能。正如 Harari
   <a class="markup--anchor markup--p-anchor" data-href="https://ideas.ted.com/why-humans-run-the-world/" href="https://ideas.ted.com/why-humans-run-the-world/" rel="noopener noreferrer" target="_blank">
    所说
   </a>
   的，“在华尔街或洋基体育场放置10万只黑猩猩，你会感到混乱；如果在那里安置10万人，你将获得贸易网络和体育竞赛。“
  </p>
  <p class="graf graf--p">
   简而言之，
   <strong class="markup--strong markup--p-strong">
    合作改善了参与其中的人的成功
   </strong>
   。神经科学家 Matthew Lieberman 在他的著作
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amazon.co.uk/Social-Why-brains-wired-connect/dp/0199645043" href="https://www.amazon.co.uk/Social-Why-brains-wired-connect/dp/0199645043" rel="noopener noreferrer" target="_blank">
    “社会：为什么我们的大脑有连接”
   </a>
   中指出，前扣带回皮层和前脑岛 — 负责疼痛的大脑区域，可以激活社会疼痛和身体疼痛，感觉受到排斥的字面伤害与被踩脚或被灼伤的方式相同。
  </p>
  <p class="graf graf--p">
   它也负责快乐。参与奖励机制的大脑区域，腹侧纹状体和眶额皮层，以类似的方式响应社交愉悦和身体愉悦。研究发现，当我们被告知其他人喜欢我们时，就像吃了甜食或赢钱一样，大脑的同一区域会被激活。神经科学家 Jaak Panksepp
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amazon.com/Affective-Neuroscience-Foundations-Emotions-Science-ebook/dp/B00590X4HM" href="https://www.amazon.com/Affective-Neuroscience-Foundations-Emotions-Science-ebook/dp/B00590X4HM" rel="noopener noreferrer" target="_blank">
    表示
   </a>
   ，大脑的社会依恋系统选择了我们的痛苦和快乐系统，这促使我们保持社会联系并避免被孤立。
  </p>
  <p class="graf graf--p">
   进化打赌，劫持我们大脑的痛苦和快乐中心以回应社会认可或反对会增加我们的生存几率。显然，它赢了。
  </p>
  <p class="graf graf--p">
   考虑催产素，或者通常被称为“爱因子”。在怀孕期间和分娩后，催产素会淹没母亲的大脑，为孩子创造强大的依恋。事实上，研究人员提出，哺乳动物中所有社会性的生物学原型都存在于母婴互动中。
  </p>
  <p class="graf graf--p">
   关于催产素的流行误解是它使人们之间的关系更好。它确实如此，
   <strong class="markup--strong markup--p-strong">
    但仅限于我们团队中的那些人。催产素可以增加对我们认为是威胁的人的攻击性和敌意。
   </strong>
   正如 Lieberman 所说，“在非同伴中，催产素导致个人将所有外人视为可能的威胁，从而增强对他们的侵略。”并且对人类施用催产素“促进了对受欢迎的群体和陌生人的照顾，但它也促进了对不喜欢的成员的敌意“。
  </p>
  <p class="graf graf--p">
   然后就是社会大脑假说。我们的大脑是我们的祖先需要在复杂的社会关系网络中导航的结果。心理学家威廉·冯·希佩尔（William von Hippel）表示，“我们的情报并没有演变为解决抽象问题和处理环境的复杂方法。我们的智慧发展到了可以更有效地相互交流，并利用在一起工作时的技能和能力。“
  </p>
  <p class="graf graf--p">
   我们非常热衷社交、需要并渴望社交。通常，当外部威胁出现时，团结就会实现。
  </p>
  <p class="graf graf--p">
   我们现在生活的世界与祖先经常光顾的世界
   <a class="markup--anchor markup--p-anchor" data-href="https://pdfs.semanticscholar.org/09d4/7bbb8032f34d11d09b9c82edb4ab45d67065.pdf" href="https://pdfs.semanticscholar.org/09d4/7bbb8032f34d11d09b9c82edb4ab45d67065.pdf" rel="noopener noreferrer" target="_blank">
    截然不同
   </a>
   。然而，我们的认知计划是针对这种社会环境而形成的。技术进步使我们摆脱了过去进化的那种负担。
  </p>
  <p class="graf graf--p">
   考虑一下战争的频率不断缩小。在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amazon.co.uk/Enlightenment-Now-Science-Humanism-Progress/dp/0525427570" href="https://www.amazon.co.uk/Enlightenment-Now-Science-Humanism-Progress/dp/0525427570" rel="noopener noreferrer" target="_blank">
    <em class="markup--em markup--p-em">
     Enlightenment Now
    </em>
   </a>
   中，Steven Pinker 指出，“自1945年以来，任何一年的战争都不会超过三场，自1989年以来的大多数年份都没有战争，自美国领导的入侵伊拉克以来……。”2016年，随着哥伦比亚政府和哥伦比亚革命武装力量实现和平，西半球最后一次积极的政治武装冲突结束了。战争现在几乎全部集中在从尼日利亚到巴基斯坦的地区，该地区的人口不到世界人口的六分之一。
  </p>
  <p class="graf graf--p">
   此外，现代战争造成的死亡人数也比任何历史时期都要少。人际暴力的比率也在下降。犯罪学家 Manuel Eisner 记录了整个欧洲凶杀案的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.vrc.crim.cam.ac.uk/vrcresearch/paperdownload/manuel-eisner-historical-trends-in-violence.pdf" href="https://www.vrc.crim.cam.ac.uk/vrcresearch/paperdownload/manuel-eisner-historical-trends-in-violence.pdf" rel="noopener noreferrer" target="_blank">
    长期减少
   </a>
   情况。在历史上，人类生活得更加平静。
  </p>
  <p class="graf graf--p">
   自然灾害怎么样？虽然灾难性事件的发生频率没有下降，但这些事件的死亡人数却有所下降。 1900年至2018年间，全球自然灾害死亡率从每10万人76.8人降至每10万人0.14人。或者考虑
   <a class="markup--anchor markup--p-anchor" data-href="https://ourworldindata.org/grapher/fatality-rates-due-to-lightning-in-the-us?time=1900..2015" href="https://ourworldindata.org/grapher/fatality-rates-due-to-lightning-in-the-us?time=1900..2015" rel="noopener noreferrer" target="_blank">
    闪电死亡事故
   </a>
   ，在20世纪的第一个十年中，美国因闪电造成的年平均死亡率为百万分之4.5。在21世纪的前15年，它下降到百万分之0.12。
  </p>
  <p class="graf graf--p">
   这些发展是技术进步的幸运副产品。正如 Pinker 所解释的那样，“当地震发生时，会被坍塌的砖石压碎或死于火灾的人更少了。早期预警系统、预测模型和灾害管理战略的出现也减少了灾难的发生。
  </p>
  <p class="graf graf--p">
   然而就如法国谚语所说，
   <strong class="markup--strong markup--p-strong">
    事情变化得越多，它们就越保持不变。现代性降低了人为和自然灾害的杀伤力。但我们对团结的渴望仍然存在。人类被迫克服危机。这是一种进化本能。
   </strong>
  </p>
  <p class="graf graf--p">
   古代祖先的想法仍然留在我们的基因里。正如 Hector Garcia 在《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.penguinrandomhouse.com/books/605358/sex-power-and-partisanship-by-hector-a-garcia/9781633885141/" href="https://www.penguinrandomhouse.com/books/605358/sex-power-and-partisanship-by-hector-a-garcia/9781633885141/" rel="noopener noreferrer" target="_blank">
    性，权力和党派之争
   </a>
   》中所写的那样，“虽然自从我们作为狩猎采集者的年代以来被邻近部落屠杀的风险已经减弱，但我们的思想仍然会因为在部落内流血事件中挣扎的祖先环境而得到校准。”
  </p>
  <p class="graf graf--p">
   我们一起经历了艰辛困苦。
   <strong class="markup--strong markup--p-strong">
    联合早期人类的史前压力源不再存在。现在我们创造人造的压力源。这是今天激发人们的愤怒的一个基础。
   </strong>
  </p>
  <p class="graf graf--p">
   愤怒文化是平凡的灾难。任何事件，无论多么普通或缺乏想象力，都是道德整理和集体行动的机会。
  </p>
  <p class="graf graf--p">
   但是，令人不安的是，集体愤怒反映了我们克服挑战的愿望。SørenKierkegaard
   <a class="markup--anchor markup--p-anchor" data-href="https://press.princeton.edu/titles/4440.html" href="https://press.princeton.edu/titles/4440.html" rel="noopener noreferrer" target="_blank">
    指出
   </a>
   ，有一段时间里，
   <strong class="markup--strong markup--p-strong">
    你能为人们做的最好的事就是让事情变得困难。当生活的轻松变得如此之大以至于已经看不清时，人们将主动去寻找能够提供目标和实现的挑战。
   </strong>
  </p>
  <p class="graf graf--p">
   所有灾难都具有统一的潜力。事实上，
   <strong class="markup--strong markup--p-strong">
    愤怒文化是由对团体团结的无意识欲望驱动的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   鉴于这些平凡的事件没有任何可识别的后果（死亡、流离失所等）对错误没有任何有意义的权利。换句话说，它们并不重要。但重要的是团结。愤怒不是你做的事情，而是你所参与的事。
  </p>
  <p class="graf graf--p">
   愤怒文化似乎是由概念蠕变培养出来的。这个想法是，
   <strong class="markup--strong markup--p-strong">
    随着世界变得更加安全，我们对构成伤害的定义范围也会扩大
   </strong>
   。考虑暴力的概念。暴力曾经被定义为一种物理行为，但有些人已经将其扩展到了包括语言。
  </p>
  <p class="graf graf--p">
   概念蠕变似乎对我们来说很自然。在最近的
   <a class="markup--anchor markup--p-anchor" data-href="http://science.sciencemag.org/content/360/6396/1465" href="http://science.sciencemag.org/content/360/6396/1465" rel="noopener noreferrer" target="_blank">
    一项研究中
   </a>
   ，实验者向人们展示了一系列蓝色和紫色的点。然后要求他们判断每个点
   <strong class="markup--strong markup--p-strong">
    是否为蓝色
   </strong>
   。在早期试验中，一半的点是蓝色的，而另一半是紫色的。渐渐地，实验者开始操纵那些点，以便出现更多的点是紫色的。然而，参与者开始扩大了他们对“蓝色”的定义，并将许多紫色点计为蓝色。他们寻找蓝点并设法“找到”它们。
  </p>
  <p class="graf graf--p">
   但这不只是“点”的问题。在不
   <a class="markup--anchor markup--p-anchor" data-href="http://science.sciencemag.org/content/360/6396/1465" href="http://science.sciencemag.org/content/360/6396/1465" rel="noopener noreferrer" target="_blank">
    同版本的实验
   </a>
   中，参与者被邀请查看表情从中性到威胁。由于威胁面孔的出现次数较少，参与者开始将中性面孔描述为“具有威胁性”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当我们需要寻找信号时，通常会扩展对信号的定义，以使更多的观察结果“合适”。
   </strong>
  </p>
  <p class="graf graf--p">
   同样，我们不断增长的繁荣使我们扩大了对构成威胁的定义。随着逐渐习惯现代生活的奇迹，人们开始仔细审视每一个轻微的猥亵。正如威廉·冯·希佩尔（William von Hippel）在“社会飞跃”（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amazon.com/Social-Leap-Evolutionary-Science-Where/dp/0062740393/ref=sr_1_1?keywords=the+social+leap&amp;qid=1553348297&amp;s=gateway&amp;sr=8-1" href="https://www.amazon.com/Social-Leap-Evolutionary-Science-Where/dp/0062740393/ref=sr_1_1?keywords=the+social+leap&amp;qid=1553348297&amp;s=gateway&amp;sr=8-1" rel="noopener noreferrer" target="_blank">
    The Social Leap
   </a>
   ）中提出的那样，我们对不会杀死我们的食物和保护我们的装置的调整让小问题更加突出了。
  </p>
  <p class="graf graf--p">
   在这种情况下，平凡成为灾难性的。你可以在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.google.com/search?q=moana+dress+controversy&amp;source=lnms&amp;tbm=nws&amp;sa=X&amp;ved=0ahUKEwjluK_VspbhAhVxQRUIHT8_BpgQ_AUIECgD&amp;biw=1536&amp;bih=754" href="https://www.google.com/search?q=moana+dress+controversy&amp;source=lnms&amp;tbm=nws&amp;sa=X&amp;ved=0ahUKEwjluK_VspbhAhVxQRUIHT8_BpgQ_AUIECgD&amp;biw=1536&amp;bih=754" rel="noopener noreferrer" target="_blank">
    这里
   </a>
   、
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nationalreview.com/2019/03/the-culture-of-fake-outrage-comes-for-tucker-carlson/" href="https://www.nationalreview.com/2019/03/the-culture-of-fake-outrage-comes-for-tucker-carlson/" rel="noopener noreferrer" target="_blank">
    这里
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bbc.co.uk/news/entertainment-arts-46479017" href="https://www.bbc.co.uk/news/entertainment-arts-46479017" rel="noopener noreferrer" target="_blank">
    这里
   </a>
   找到一些有趣的例子。
  </p>
  <p class="graf graf--p">
   至少有两个原因可以解释这点。首先，人类不是为了舒适而建造的；并因压力和冲突而进化。其次，人类有一种联盟本能，成为优秀的团体成员是追求。显然，我们想要冲突、希望盟友帮助我们解决冲突。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    愤怒文化使我们能够满足这些冲动。通过扩大灾难的定义，我们创造了我们需要盟友解决的新问题；将问题归类、以及我们提出的解决方案，作为我们部落归属的指标
   </strong>
   。
  </p>
  <p class="graf graf--p">
   愤怒对我们团体中的人很感兴趣。这是一个集结的旗帜和一个失败的敌人；围绕不存在的危机建立的共同事业。只有捆绑在一起，我们才能对抗
   <a class="markup--anchor markup--p-anchor" data-href="https://www.npr.org/2018/12/05/673770902/baby-it-s-cold-outside-seen-as-sexist-frozen-out-by-radio-stations?t=1552940775624" href="https://www.npr.org/2018/12/05/673770902/baby-it-s-cold-outside-seen-as-sexist-frozen-out-by-radio-stations?t=1552940775624" rel="noopener noreferrer" target="_blank">
    risqué圣诞音乐
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://eu.usatoday.com/story/news/nation-now/2018/05/04/chinese-ok-utah-teens-controversial-cheongsam-prom-dress/580062002/" href="https://eu.usatoday.com/story/news/nation-now/2018/05/04/chinese-ok-utah-teens-controversial-cheongsam-prom-dress/580062002/" rel="noopener noreferrer" target="_blank">
    白人女孩穿旗袍
   </a>
   的威胁。愤怒是一种破坏性的药物，满足了我们对团结的最初愿望。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    真正的危机创造了真正的团结。而做作的危机会导致伪造的团结。通过制造人为危机，我们与我们的内部成员建立了一种暂时的统一感。通过围绕愤怒走到一起，人类撕裂了社会结构。
   </strong>
  </p>
  <p class="graf graf--p">
   要清楚，我们不是要求战争或地震。为了苦难而苦难是没有意义的。我们也没有谴责现代生活的奇迹。技术进步越多越好。
  </p>
  <p class="graf graf--p">
   人类无法与他们的痛苦分离。它赋予生命意义。弗朗西斯·福山（Francis Fukuyama）曾写道，
   <strong class="markup--strong markup--p-strong">
    “如果一个人不能为正义事业而斗争，那么他们就会反对正义事业。他们将为斗争而斗争。他们会挣扎，换句话说是出于某种无聊：因为他们无法想象生活在没有挣扎的世界。“
   </strong>
  </p>
  <p class="graf graf--p">
   只有忍受和克服困难，我们才能开始欣赏生命之美。我们可以更加深思熟虑地关注哪些威胁以及如何阻止它们，而不是让一时的愤怒劫持我们的注意力。我们渴望冲突。但我们可以更加警惕：我们正在如何满足这种渴望。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    W
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://quillette.com/2019/04/07/what-doesnt-kill-us-brings-us-together/" href="https://quillette.com/2019/04/07/what-doesnt-kill-us-brings-us-together/" rel="noopener noreferrer" target="_blank">
    hat Doesn’t Kill Us Brings Us Together.
   </a>
   Science and technology have softened the sting of manmade and natural disasters. But such advancements have reduced the impact of key social stressors. They have curtailed flashpoint events which bring us together. One consequence of this is outrage culture.
  </p>
