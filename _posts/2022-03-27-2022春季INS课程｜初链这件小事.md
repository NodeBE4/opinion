  <h2 class="has-text-align-center" style="font-size:25px">
   讲者简介
  </h2>
  <p style="font-size:15px">
   刘怿斯是Dimension.im公司的技术长，Dweb Shanghai发起人。本科和研究生就读于UIUC计算机科学系，主要的研究方向是机器学习与自然语言处理。他关心网络与人类生活之间相互的影响，现在正致力于以应用密码学为基础，帮助互联网用户重新获得个人数据的所有权。
  </p>
  <h2 class="has-text-align-center" style="font-size:25px">
   课程简述
  </h2>
  <p style="font-size:15px">
   在过去十多年间，区块链及相关技术与行业已经获得了长足的发展，市场也对此进行着充分的实验，以优化其体验与机制。从简单的记账工具到现在的百花齐放的生态，区块链到底在干嘛？它在多大程度上正在帮助我们解决现有问题？站在使用者的角度回答上述问题之后，我们还应该思考：区块链真的是我们的未来吗？希望在课程结束的时候，我们每个人心中都能得出一个自己的答案。
  </p>
  <p style="font-size:15px">
   本课程的主要目的是让非理工科背景的同学能够理解区块链与元宇宙基础知识，让大家能基于现有的开源工具与平台和基础的编程知识最大程度地参与到现有区块链与元宇宙项目中，思考新技术所带来的社会与伦理问题。本课程主要分为三个部分：1. 基础背景知识回顾与掌握 2. 区块链的原理与生态 3. 元宇宙的愿景与现实。

   </a>
  </p>

  <h3 class="has-text-align-center" id="基础工具和知识的回顾与掌握（2课时）" style="font-size:25px">
   基础工具和知识的回顾与掌握
  </h3>
  <p style="font-size:15px">
   #
   <strong>
    01
   </strong>
   <strong>
    工具的再探索
   </strong>
   <br/>
   工具对于生产力与生产效率的提升至关重要，本单元将会从课程的目标与话题出发，结合当下流行的生产力工具，为本课程所会涉及到的范围作准备，以帮助大家更好地参与到区块链世界中，为未来的创作打下基础。
  </p>
  <p style="font-size:15px">
   a. 课程介绍与愿景
   <br/>
   b. 搜索引擎
   <br/>
   c. 笔记整理
   <br/>
   d. 社交媒体
   <br/>
   e. 即时通讯
   <br/>
   f. 多人协作
  </p>
  <p style="font-size:15px">
   <strong>
    #02 研究理论与方法
   </strong>
   <br/>
   本单元将着重为大家讲述面对一个陌生的领域所应采取的学习与研究方法，包括但不仅限于如何利用现有工具、如何搜索领域内资料与文献、如何获取并处理数据等。本单元将会囊括一个现有例子作为案例分析，让大家可以以第一视角去体会与学习。
  </p>
  <p style="font-size:15px">
   a. 面对陌生领域应当采取的研究方法
   <br/>
   b. 工具的利用与协作
   <br/>
   c. 新的数据源与数据类型
   <br/>
   d. 案例分析
  </p>

  </p>

   <br/>
   c. 区块链发展进程
   <br/>
   d. 开源、社区与共识
   <br/>
   e. 区块链技术本身是否是决定性因素？

   <strong>
    #09 融合的技术
   </strong>
   <br/>
   将上述的这些技术融合在一起，我们会得到什么？我们真的需要它们吗？我们的生活真的需要这么多技术参与到我们生活的各个环节中吗？我们不应创造问题，而是解决问题，那么融合的技术是在创造还是解决问题呢？我们将花一节课的时间来共同探讨区块链技术相互之间的关系与可组合性，并尝试回答我们在课程之初提出的问题：区块链真的是我们的未来吗？
  </p>

    #11 元宇宙的未来
   </strong>
   <br/>
   元宇宙是要替代现实世界，还是与现实世界融合共生呢？我们为什么需要元宇宙，又或者，元宇宙为什么需要我们？我们如何才能成为元宇宙的公民，又或者，元宇宙需要有准入门槛吗？我们与元宇宙究竟有多远，在通往元宇宙的道路，我们又能做什么呢？
