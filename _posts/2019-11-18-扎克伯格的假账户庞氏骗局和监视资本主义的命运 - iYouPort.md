  <p class="graf graf--p">
   Facebook 现在的市值接近 6000 亿美元，这使其成为地球上最有价值的公司之一。这是一个真正的商业奇迹：这家2012年就已经开始失去用户的公司居然设法找到了“近乎无限和持续增长”的源泉……
  </p>
  <p class="graf graf--p">
   延伸：开源调查揭示《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%b7%b1%e5%b1%82%e6%94%bf%e6%b2%bb%ef%bc%9afacebook%e9%87%91%e5%ad%97%e5%a1%94%e7%9a%84%e4%ba%ba%e5%bd%a2%e8%82%8c%e7%90%86-%e4%bb%85%e4%bb%85%e4%be%9d%e9%9d%a0%e5%bc%80%e6%ba%90/" href="https://www.iyouport.org/%e6%b7%b1%e5%b1%82%e6%94%bf%e6%b2%bb%ef%bc%9afacebook%e9%87%91%e5%ad%97%e5%a1%94%e7%9a%84%e4%ba%ba%e5%bd%a2%e8%82%8c%e7%90%86-%e4%bb%85%e4%bb%85%e4%be%9d%e9%9d%a0%e5%bc%80%e6%ba%90/" rel="noopener noreferrer" target="_blank">
    深层政治：Facebook 金字塔的人形肌理 — — 扒开利维坦的内裤
   </a>
   》
  </p>
  <p class="graf graf--p">
   多么神奇啊，那个“天才”的商业秘密，将一个亏本的创业公司变成了全世界“最伟大的商业成功故事”之一。这是为什么？
  </p>
  <p class="graf graf--p">
   从 Bernie Madoff 身上就能看出原因之一：欺诈，以假帐户的形式。
  </p>
  <p class="graf graf--p">
   旧钱出来，新钱进去取而代之，这就是传统的庞氏骗局的运作方式。是 Madoff 坚持了几十年的努力，他成功地获得了纳斯达克主席的职位。
  </p>
  <p class="graf graf--p">
   扎克伯格的版本略有不同，但只是略有不同而已：老用户在感到厌倦、厌恶和不信任之后离开，新用户进来取代他们。
  </p>
  <p class="graf graf--p">
   扎克伯格的朋友 Sam Lessin 告诉我们，这个等式中的“新用户”部分已经在2012年就成为了一个问题。
  </p>
  <p class="graf graf--p">
   当年的10月26日，Lessin 写道，“我们已经没有人类资源了（从广告商的角度来看，已经耗尽了宝贵的人类资源）。”
  </p>
  <p class="graf graf--p">
   当时 Facebook 甚至还有一个可行的商业模式远未明朗，据 Frontline 称，由于该公司收入数据不佳，Sheryl Sandberg 感到恐慌。
  </p>
  <p class="graf graf--p">
   为了平衡并保持“增长”趋势，Facebook 所做的只是假装视而不见。
  </p>
  <p class="graf graf--p">
   在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.plainsite.org/dockets/3bvv82ier/california-northern-district-court/singer-v-facebook-inc/" href="https://www.plainsite.org/dockets/3bvv82ier/california-northern-district-court/singer-v-facebook-inc/" rel="noopener noreferrer" target="_blank">
    Singer v.Facebook
   </a>
   案中，这个在加利福尼亚州北部地区提起的诉讼称 Facebook 一直在告诉广告商它可以“覆盖”更多人，已经远远超过了每个主要大城市的实际人口 —— 原告引用前 Facebook 员工的内幕消息（完全可以理解，这种事只有隐藏很深的内部人士才能知道）Facebook 的“潜在影响力” 统计数据是“被人为制造出来的公关号”。
  </p>
  <p class="graf graf--p">
   而且，“那些负责确保准确性的人并没有在此做任何事”；“Facebook 的前运营承包商表示 Facebook
   <strong class="markup--strong markup--p-strong">
    并不关心
   </strong>
   阻止重复账户或假账户。”
  </p>
  <p class="graf graf--p">
   Facebook 在美国已
   <a class="markup--anchor markup--p-anchor" data-href="https://s21.q4cdn.com/399680738/files/doc_financials/2019/Q1/Q1-2019-Earnings-Presentation.pdf" href="https://s21.q4cdn.com/399680738/files/doc_financials/2019/Q1/Q1-2019-Earnings-Presentation.pdf" rel="noopener noreferrer" target="_blank">
    不再增长
   </a>
   ，2019年第一季度新增账户数为零，自2017年第一季度以来只有400万新账户。
  </p>
  <p class="graf graf--p">
   根据 Facebook 首席财务官大卫·韦纳（David Wehner）的说法，Facebook 在印度、印尼和菲律宾增长最快。
  </p>
  <p class="graf graf--p">
   Wehner 没有提到幻灯片第18页的细则，其中突出显示：菲律宾、印度尼西亚和越南是假冒账户中“最有意义的、百分比最高”的国家。
  </p>
  <p class="graf graf--p">
   换句话说，在全世界范围内，Facebook 被发现欺诈最多的地方“增长速度最快”。现在你明白了，Facebook 根本没有增长 —— 它正在萎缩。
  </p>
  <p class="graf graf--p">
   甚至印度、印尼和菲律宾也没有像以前那样增长； Instagram 上的许多“新”用户实际上都是来自核心平台的老用户。
  </p>
  <p class="graf graf--p">
   上一次扎克伯格表示 Facebook 的“增长鼎盛时期”是在2018年7月附近，当时股票暴跌，最终成为美国历史上任何一家公司股票单日跌幅最大的一天。
  </p>
  <p class="graf graf--p">
   在大约一个小时内，从每股220美元左右
   <a class="markup--anchor markup--p-anchor" data-href="https://www.marketwatch.com/story/facebook-stock-crushed-after-revenue-user-growth-miss-2018-07-25" href="https://www.marketwatch.com/story/facebook-stock-crushed-after-revenue-user-growth-miss-2018-07-25" rel="noopener noreferrer" target="_blank">
    暴跌20％
   </a>
   至165美元左右。
  </p>
  <p class="graf graf--p">
   不用说，一小时内市值损失约1200亿美元，这足以阻止扎克伯格避免重复的表演。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-7025 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-21.png?resize=1100%2C622&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-21.png?w=1206&amp;ssl=1 1206w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-21.png?resize=300%2C170&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-21.png?resize=768%2C434&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-21.png?resize=1024%2C579&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-21.png?resize=1100%2C622&amp;ssl=1 1100w" data-recalc-dims="1" height="622" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-21.png?resize=1100%2C622&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   由于勉强逃脱了华尔街的愤怒，扎克伯格知道他无法摆脱他已经折腾了多年的增长跑步机。唯一的解决方案：向投资者说谎，试图说服他们一切都很好。
  </p>
  <p class="graf graf--p">
   然而，
   <strong class="markup--strong markup--p-strong">
    扎克伯格的虚假账户与 Madoff 的虚假账户报表没有什么不同。
   </strong>
  </p>
  <p class="graf graf--p">
   谷歌趋势显示全球 “Facebook” 查询比2012年11月的峰值下降了80％。 （Instagram 甚至没有弥补任何损失）衡量 Facebook 移动应用程序使用情况的“移动指标”也明显下降。
  </p>
  <p class="graf graf--p">
   该公司自己关于虚假账户的披露存在明显问题，主要是其内部的不一致性 —— 一组数据A（以百分比衡量）向美国证券交易委员会披露，而另一组数据B以绝对数字形式显示在其所谓的“透明度门户网站”上。
  </p>
  <p class="graf graf--p">
   虽然他们“发现”问题以惊人的速度升级 —— Facebook 声称虚假账户为5％、重复账户为11％，2017年第二季度分别为1％和6％ —— 但他们什么也没做。
  </p>
  <p class="graf graf--p">
   截至2017年底，Facebook 决定停止每季度发布这些百分比，改为每年一次。眼不见，心不烦。
  </p>
  <p class="graf graf--p">
   有人可能会争辩说，美国证券交易委员会的披露受到“证券交易法”的严格规定，Facebook 居然那么大胆地向投资者撒谎？
  </p>
  <p class="graf graf--p">
   但他们真的这么做了。
  </p>
  <p class="graf graf--p">
   也许还有人认为透明度门户网站指标由一个学术团队审核 — 数据透明度咨询小组（DTAG），他们应该保证其有效性？
  </p>
  <p class="graf graf--p">
   但 DTAG 学者已经承认他们是由 Facebook 支付的。即使经过数月的努力，他们在4月份发布的最终
   <a class="markup--anchor markup--p-anchor" data-href="https://law.yale.edu/system/files/area/center/justice/document/dtag_report_5.22.2019.pdf" href="https://law.yale.edu/system/files/area/center/justice/document/dtag_report_5.22.2019.pdf" rel="noopener noreferrer" target="_blank">
    报告中
   </a>
   提到假账户只有三次。在关于Facebook的假账号问题上。DTAG 奇怪地表现出无话可说。
  </p>
  <p class="graf graf--p">
   也很难说真的“奇怪”，这种手段谷歌也用过《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%9d%83%e8%b0%8b%ef%bc%9a%e4%b8%ba%e6%b8%b8%e8%af%b4%e5%8d%8e%e7%9b%9b%e9%a1%bf%ef%bc%8c%e8%b0%b7%e6%ad%8c%e5%88%b6%e9%80%a0%e4%ba%86%e4%b8%80%e4%b8%aa%e5%86%a0%e5%86%95%e5%a0%82%e7%9a%87%e7%9a%84/" href="https://www.iyouport.org/%e6%9d%83%e8%b0%8b%ef%bc%9a%e4%b8%ba%e6%b8%b8%e8%af%b4%e5%8d%8e%e7%9b%9b%e9%a1%bf%ef%bc%8c%e8%b0%b7%e6%ad%8c%e5%88%b6%e9%80%a0%e4%ba%86%e4%b8%80%e4%b8%aa%e5%86%a0%e5%86%95%e5%a0%82%e7%9a%87%e7%9a%84/" rel="noopener noreferrer" target="_blank">
    权谋：为游说华盛顿，谷歌制造了一个冠冕堂皇的傀儡
   </a>
   》
  </p>
  <p class="graf graf--p">
   全面回顾 Facebook 的披露表明，在该公司的120亿总账户中，约有100亿是假的；其中多达10亿可能是活跃的（这是最保守的估算）。
  </p>
  <p class="graf graf--p">
   这可能是历史上最大的企业欺诈的关键。
  </p>
  <p class="graf graf--p">
   广告商在假设观看和点击广告的人是真实的人类并以此向 Facebook 付费。但事实并非如此。
  </p>
  <p class="graf graf--p">
   Facebook 绝对没有动力来解决这个问题 — 问题已经在法庭上，并且是它的前员工正在揭露。
  </p>
  <p class="graf graf--p">
   从扎克伯格的角度来看，这就是钞票雨。要让广告客户花钱，他所要做的就是让世界相信 Facebook 是巨大的，而且只会变得更大。
  </p>
  <p class="graf graf--p">
   没有任何媒体敢于提出这一可能令人尴尬的问题，更不用说国会了，而且很少有人理解 real-time pricing auctions、cookies、user disambiguation 的细枝末节。
  </p>
  <p class="graf graf--p">
   每个人都宁愿谈论该公司对所谓的“创新”的奉献精神、以及
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%9d%80%e6%ad%bb%e5%a4%a9%e7%a7%a4%e5%ba%a7%e5%9c%a8%e8%90%8c%e8%8a%bd%e7%8a%b6%e6%80%81/" href="https://www.iyouport.org/%e6%9d%80%e6%ad%bb%e5%a4%a9%e7%a7%a4%e5%ba%a7%e5%9c%a8%e8%90%8c%e8%8a%bd%e7%8a%b6%e6%80%81/" rel="noopener noreferrer" target="_blank">
    Libra 完全没必要的复杂
    <strong class="markup--strong markup--p-strong">
     伪
    </strong>
    加密货币系统
   </a>
   的可笑。
  </p>
  <p class="graf graf--p">
   扎克伯格很聪明，但他从来没有足够聪明到愿意倾听他人经验的程度。他最常用的缩略词是什么 — VR？ AI？ — 就像切尔诺贝利一样，由此产生的尘埃无处不在，无法清理，地上还有尸体。
  </p>
  <p class="graf graf--p">
   即使是他的联合创始人和前室友也不能完全支持他，尽管克里斯休斯仍然乖乖地将马克称为“善良的人”，“没有什么能比一个有才华的企业家的善良喧嚣更邪恶的事了”。
  </p>
  <p class="graf graf--p">
   这显然是错误的。扎克伯格不是什么善良的人。他所从事的唯一喧嚣就是普通意义上的欺诈。
  </p>
  <p class="graf graf--p">
   Facebook 是一个真正的产品，但像 Enron 一样；它也是一个骗局，现在是有史以来最大的企业丑闻。
  </p>
  <p class="graf graf--p">
   它不会公布有关2016年选举、假账户或任何重大事项的数据 —— 而且由于扎克伯格自己知道这是一个骗局，他不会同意在英国议会面前以可能要求他如实回答的方式作证任何实质性问题。
  </p>
  <p class="graf graf--p">
   而且，
   <strong class="markup--strong markup--p-strong">
    由于 Facebook 也是标普500指数的一部分，无数人都有动力维持现状。
   </strong>
  </p>
  <p class="graf graf--p">
   Facebook 已成为下届总统大选的竞选辩论问题之一。伊丽莎白沃伦说“应该拆解它”； Beto O’Rourke 希望“更强有力的监管”；Kamala Harris 宁愿谈只论隐私问题 ……
  </p>
  <p class="graf graf--p">
   延伸：为什么科技寡头非常乐于拥护政府监管？《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%8b%a5%e6%8a%a4%e7%9a%84%e8%83%8c%e5%90%8e/" href="https://www.iyouport.org/%e6%8b%a5%e6%8a%a4%e7%9a%84%e8%83%8c%e5%90%8e/" rel="noopener noreferrer" target="_blank">
    拥护的背后
   </a>
   》。
  </p>
  <p class="graf graf--p">
   其他人 — 甚至特朗普 — 普遍同意需要“做点什么”。
  </p>
  <p class="graf graf--p">
   然而，
   <strong class="markup--strong markup--p-strong">
    未说出口的中心问题仍然存在
   </strong>
   ：扎克伯格正在运行庞氏骗局，但华尔街、国会、联邦贸易委员会、智囊团及其同伙，都绕开了这点。
  </p>
  <p class="graf graf--p">
   在这一点上，任何了解寡头公司的人都不应该感到惊讶。扎克伯格不了解法治，也不了解新闻自由的作用，他有一种危险的信仰，认为自己是绝对正确的。
  </p>
  <p class="graf graf--p">
   只要有钱赚，他很愿意养活更多剑桥分析公司那样的“合作伙伴”。
  </p>
  <p class="graf graf--p">
   在哈佛大学法学院的一次演讲中，马特·斯托勒认为：Facebook、谷歌和亚马逊“生来就是垄断者”。
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/xM9GMGDsKUU" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   这可能是一个令人注目的说法，如果人们愿意忽视现实的话。
  </p>
  <p class="graf graf--p">
   但这不是这真的。
  </p>
  <p class="graf graf--p">
   首先，软件产品不是铁路，需要大量的物质资本和劳动力来建立；如果他决定这样做，扎克伯格可以花几周时间重建 Instagram 和 WhatsApp，政府真的没有办法阻止他。其次，至少当 Facebook 于2003年9月19日出现时它看起来毫无垄断的意味。
  </p>
  <p class="graf graf--p">
   垄断不是天生的，正相反是被长期以来不正当的奖励所滋养出来的。
  </p>
  <p class="graf graf--p">
   人们经常将监视资本主义与数字技术等同起来，或者认为是因果关系。其实不是一码事。
   <strong class="markup--strong markup--p-strong">
    监视资本主义是一种经济逻辑，它为了自己的目的而劫持了数字技术。
   </strong>
  </p>
  <p class="graf graf--p">
   监视资本主义经济从下方和上方侵蚀民主。
  </p>
  <p class="graf graf--p">
   在基层，该系统旨在压制个人意识、破坏人权机构、消除决策权、减少自主权、剥夺人们的反抗权。以极端的知识和权力集中化为形式（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a4%a7%e5%9e%8b%e4%ba%91%e4%be%9b%e5%ba%94%e5%95%86%e6%97%b6%e4%bb%a3%e7%9a%84%e5%ae%a1%e6%9f%a5%e5%88%b6%e5%ba%a6%ef%bc%9a%e9%9b%86%e4%b8%ad%e5%8c%96%e6%af%81%e6%8e%89%e4%ba%92/" href="https://www.iyouport.org/%e5%a4%a7%e5%9e%8b%e4%ba%91%e4%be%9b%e5%ba%94%e5%95%86%e6%97%b6%e4%bb%a3%e7%9a%84%e5%ae%a1%e6%9f%a5%e5%88%b6%e5%ba%a6%ef%bc%9a%e9%9b%86%e4%b8%ad%e5%8c%96%e6%af%81%e6%8e%89%e4%ba%92/" rel="noopener noreferrer" target="_blank">
    这里
   </a>
   ）。
  </p>
  <p class="graf graf--p">
   监视资本家了解我们的一切，但我们仍对它们知之甚少。
  </p>
  <p class="graf graf--p">
   在没有法律约束的情况下，监视资本主义快速蓬勃发展起来。
  </p>
  <p class="graf graf--p">
   就如 Shoshana Zuboff 所认为的那样，经常有人声称无法控制这种流氓资本主义，但事实是。目前为止还没有人试过真正去控制它。
  </p>
  <p class="graf graf--p">
   好消息是：
   <strong class="markup--strong markup--p-strong">
    人类社会过去曾经成功地压制了破坏性的资本主义形式
   </strong>
   ，推动新的法律将资本主义束缚在人们的真正需求范围内。民主结束了镀金时代。
  </p>
  <p class="graf graf--p">
   我们完全有理由相信我们可以再次获得成功。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    下一个伟大的监管愿景很可能是由反抗者构成的
   </strong>
   ：立法者、公民和技术专家，他们认识到只有民主才能通过法律和法规维护人民的利益。
  </p>
  <p class="graf graf--p">
   问题是，什么样的监管才有效？
   <strong class="markup--strong markup--p-strong">
    隐私法案和反托拉斯法律的现有方法是否可以作为答案？承认两者都很关键，但是，两者都不够
   </strong>
   。
  </p>
  <p class="graf graf--p">
   一个例子是隐私法要求“数据所有权”。
  </p>
  <p class="graf graf--p">
   这是一种误导性的概念，因为它
   <strong class="markup--strong markup--p-strong">
    合法地
   </strong>
   将单方面的人类经验 —— 你的脸，你的手机，你的冰箱，你的情感等等一切 —— 转化为数据。
  </p>
  <p class="graf graf--p">
   即使我们实现了控制向 Facebook 这样的公司提供的数据的“所有权”，也无法实现控制从中获取的预测的“所有权”。
  </p>
  <p class="graf graf--p">
   详见：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%95%b0%e6%8d%ae%e6%89%80%e6%9c%89%e6%9d%83%e4%b8%8d%e4%bb%85%e6%97%a0%e6%b3%95%e8%a7%a3%e5%86%b3%e7%8e%b0%e6%9c%89%e9%97%ae%e9%a2%98%ef%bc%8c%e5%ae%83%e8%bf%98%e5%88%b6%e9%80%a0/" href="https://www.iyouport.org/%e6%95%b0%e6%8d%ae%e6%89%80%e6%9c%89%e6%9d%83%e4%b8%8d%e4%bb%85%e6%97%a0%e6%b3%95%e8%a7%a3%e5%86%b3%e7%8e%b0%e6%9c%89%e9%97%ae%e9%a2%98%ef%bc%8c%e5%ae%83%e8%bf%98%e5%88%b6%e9%80%a0/" rel="noopener noreferrer" target="_blank">
    “数据所有权”不仅无法解决现有问题，它还制造了新的问题
   </a>
   》。
  </p>
  <p class="graf graf--p">
   我们永远不会拥有每秒钟产生的600万次预测。监视资本家知道这一点。这就是为什么监视资本主义寡头很高兴拥护”所有权”概念，并公开邀请隐私法规。
  </p>
  <p class="graf graf--p">
   详见：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%a5%e9%9a%90%e7%a7%81%e4%b8%ba%e7%9b%ae%e7%9a%84%e7%9a%84%e6%96%97%e4%ba%89%e5%a6%82%e4%bd%95%e8%ae%a9%e4%bd%a0%e5%8f%98%e6%88%90%e4%ba%86%e7%9b%91%e8%a7%86%e7%9a%84%e7%9b%9f%e5%8f%8b/" href="https://www.iyouport.org/%e4%bb%a5%e9%9a%90%e7%a7%81%e4%b8%ba%e7%9b%ae%e7%9a%84%e7%9a%84%e6%96%97%e4%ba%89%e5%a6%82%e4%bd%95%e8%ae%a9%e4%bd%a0%e5%8f%98%e6%88%90%e4%ba%86%e7%9b%91%e8%a7%86%e7%9a%84%e7%9b%9f%e5%8f%8b/" rel="noopener noreferrer" target="_blank">
    以隐私为目的的斗争如何让你变成了监视的盟友
   </a>
   》
  </p>
  <p class="graf graf--p">
   立法者应该怎么做？
  </p>
  <p class="graf graf--p">
   首先，中断和禁止监视资本主义的数据供应和收入流。
  </p>
  <p class="graf graf--p">
   这意味着，在前端，禁止秘密窃取私人经验；在后端，禁止数据交易。就如当年成功禁止贩卖奴隶那样。
  </p>
  <p class="graf graf--p">
   其次，过去十年的研究表明，当“用户”被告知监视资本主义的后台操作时，他们需要保护，他们需要替代方案。
  </p>
  <p class="graf graf--p">
   尤其需要法律和法规，旨在帮助那些有希望打破监视资本主义体系的公司。符合人们实际需求和市场民主规范的竞争者可能会吸引地球上每个人作为他们的客户。
  </p>
  <p class="graf graf--p">
   更多《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/google-%e4%ba%a7%e5%93%81%e7%9a%84%e6%9b%bf%e4%bb%a3%e5%93%81-%e5%ae%8c%e6%95%b4%e5%88%97%e8%a1%a8-%e5%85%b3%e6%b3%a8%e9%9a%90%e7%a7%81%e6%9d%83%e7%9a%84%e4%bd%a0%ef%bc%8c%e9%a9%ac%e4%b8%8a/" href="https://www.iyouport.org/google-%e4%ba%a7%e5%93%81%e7%9a%84%e6%9b%bf%e4%bb%a3%e5%93%81-%e5%ae%8c%e6%95%b4%e5%88%97%e8%a1%a8-%e5%85%b3%e6%b3%a8%e9%9a%90%e7%a7%81%e6%9d%83%e7%9a%84%e4%bd%a0%ef%bc%8c%e9%a9%ac%e4%b8%8a/" rel="noopener noreferrer" target="_blank">
    Google 产品的替代品完整列表
   </a>
   》。
  </p>
  <p class="graf graf--p">
   第三，立法者需要支持新形式的集体行动，正如近一个世纪以前，工人们为其组织、集体谈判和罢工的权利赢得的法律保护。
  </p>
  <p class="graf graf--p">
   监视资本家既富有又强大，但他们并非无懈可击。他们害怕法律。他们害怕坚持不同道路的公民。
  </p>
  <p class="graf graf--p">
   这两个团体在拯救民主的数字化未来的工作中应该联系在一起。⚪️
  </p>
