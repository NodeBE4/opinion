  <p class="graf graf--p graf-after--figure" id="9155">
   这辆装甲 SUV 停在 Aden 黑暗的街道上，前美国绿色贝雷帽挎着 AK-47 吮吸着一根棒棒糖，站在旁边。执行任务的另外两名战士是前美国海豹突击队队员。作为精英美国特种作战战士，他们多年来一直受到美国军方的专门训练，当然那些训练是为了保护美国。但是，现在他们正在为一位完全不同的雇主工作：一家由阿联酋雇佣的美国私人公司，这是一个位于波斯湾沙漠的小君主制国家。
  </p>
  <p class="graf graf--p graf-after--p" id="a275">
   2015年12月29日那天晚上，他们的工作是进行暗杀。
  </p>
  <p class="graf graf--p graf-after--p" id="2e01">
   两名参与者向
   <a class="markup--anchor markup--p-anchor" data-href="https://www.buzzfeednews.com/article/aramroston/mercenaries-assassination-us-yemen-uae-spear-golan-dahlan" href="https://www.buzzfeednews.com/article/aramroston/mercenaries-assassination-us-yemen-uae-spear-golan-dahlan" rel="nofollow noopener noreferrer" target="_blank">
    BuzzFeed News
   </a>
   描述了他们的武装攻击，并通过无人机监控镜头证实，令人震惊，这是该营利性暗杀企业的第一次行动。几个月来，在饱受战争蹂躏的也门，美国一些训练最为严格的士兵正在进行一项阴暗的雇佣军任务，以杀死杰出的神职人员和伊斯兰政治人物。
  </p>
  <p class="graf graf--p graf-after--p" id="4974">
   那天晚上他们的目标是：阿萨夫阿里梅奥 (Anssaf Ali Mayo)，伊斯兰政党 Al-Islah 的当地领导人。阿联酋认为 Al-Islah 是全球穆斯林兄弟会的也门分支，阿联酋称之为”恐怖组织”。但是许多专家则坚持认为，
   <strong class="markup--strong markup--p-strong">
    其成员获得了
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2011/oct/07/tawakkul-karman-profile" href="https://www.theguardian.com/world/2011/oct/07/tawakkul-karman-profile" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     诺贝尔和平奖
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    的 Al-Islah 并不是恐怖组织
   </strong>
   。他们说，
   <strong class="markup--strong markup--p-strong">
    这是一个合法的政党，不是通过暴力威胁阿联酋的，而是反对其在也门的野心
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="00f9">
   雇佣军的计划是将一枚炸弹安放在 Al-Islah 总部的门口，该总部位于 Aden 中部的一个足球场附近，这是一个关键的也门港口城市。探险队的一位领导人解释说，这次爆炸应该“杀死那个办公室里的每个人”。
  </p>
  <p class="graf graf--p graf-after--p" id="f1bf">
   当他们晚上9点57分到达时，一切都显得很安静。男人们从 SUV 中爬出来，准备好了枪。一个人向建筑物射击。但正当他即将到达大门时，团队的另一名成员开火，沿着灯光昏暗的街道向后射击，他们精心设计的计划陷入了困境。
   <a class="markup--anchor markup--p-anchor" data-href="https://vimeo.com/297059736" href="https://vimeo.com/297059736" rel="noopener nofollow noreferrer" target="_blank">
    在这里
   </a>
   看到视频：
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="350" src="//player.vimeo.com/video/297059736?title=0&amp;byline=0&amp;portrait=0&amp;color=8dc7dc" width="425">
   </iframe>
  </p>
  <div class="section-inner sectionLayout--insetColumn">
   <p class="graf graf--p graf-after--figure" id="0961">
    对 Mayo 的暗杀行动当时就被报道了，但
    <strong class="markup--strong markup--p-strong">
     直到现在还不知道该行动是由美国雇佣兵进行的
    </strong>
    — — 它是也门战争的一个支点，残酷的冲突导致儿童陷入严重饥饿、村庄遭到轰炸袭击、流行病霍乱席卷了苦难的平民。这次轰炸是一系列未解决的暗杀计划中的第一次齐射，
    <strong class="markup--strong markup--p-strong">
     这次暗杀造成了该组织的二十多位领导人死亡。
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="059d">
    雇佣兵们所在的公司是 Spear Operations Group，该公司在特拉华州注册成立，由居住在匹兹堡以外的富有魅力的匈牙利裔以色列安全承包商 Abraham Golan 创立。他领导了这支暗杀小组对 Mayo 的袭击。
   </p>
   <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="765e">
    “在也门有一个针对性的暗杀计划”，他告诉 BuzzFeed 新闻，“ 我正在运行它。我们做到了。它在联盟内被阿联酋批准。”
   </p>
   <p class="graf graf--p graf-after--p" id="0aa0">
    阿联酋和沙特阿拉伯领导了在也门九个国家的联盟，主要是
    <strong class="markup--strong markup--p-strong">
     对伊朗进行
    </strong>
    <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/world/un-probe-details-fallout-of-proxy-war-in-yemen-between-saudi-coalition-and-iran-/2018/01/11/3e3f9302-f644-11e7-9af7-a50bc3300042_story.html?utm_term=.c64bf6a1c93e" href="https://www.washingtonpost.com/world/un-probe-details-fallout-of-proxy-war-in-yemen-between-saudi-coalition-and-iran-/2018/01/11/3e3f9302-f644-11e7-9af7-a50bc3300042_story.html?utm_term=.c64bf6a1c93e" rel="noopener nofollow noreferrer" target="_blank">
     <strong class="markup--strong markup--p-strong">
      代理人战争
     </strong>
    </a>
    <strong class="markup--strong markup--p-strong">
     。
    </strong>
    <a class="markup--anchor markup--p-anchor" data-href="https://www.buzzfeednews.com/article/verabergengruen/khashoggi-congress-yemen-saudi-arabia-military-aid" href="https://www.buzzfeednews.com/article/verabergengruen/khashoggi-congress-yemen-saudi-arabia-military-aid" rel="noopener nofollow noreferrer" target="_blank">
     <strong class="markup--strong markup--p-strong">
      美国正在通过提供
     </strong>
    </a>
    <strong class="markup--strong markup--p-strong">
     武器、情报和其他支持来帮助沙特阿联酋
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="b3d9">
    阿联酋美国大使馆的新闻办公室、以及美国公共事务公司
    <a class="markup--anchor markup--p-anchor" data-href="https://www.odwyerpr.com/story/public/9954/2018-01-02/uae-extends-harbour-groups-10m-pact.html" href="https://www.odwyerpr.com/story/public/9954/2018-01-02/uae-extends-harbour-groups-10m-pact.html" rel="noopener nofollow noreferrer" target="_blank">
     HarbourGroup
    </a>
    都没有回复记者的多个电话和电子邮件。
   </p>
   <p class="graf graf--p graf-after--p" id="9fde">
    一个中东君主制国家雇佣美国人进行暗杀的消息曝光发生在世界专注于沙特阿拉伯 — — 这个与美国和阿联酋关系密切的专制政权据称谋杀了持不同政见的记者 Jamal Khashoggi 的热点时刻。（沙特驻美国大使馆没有回复评论请求。利雅得否认它杀死了 Khashoggi，尽管新闻报道显示它正在考虑将记者的死亡归咎于拙劣的审讯）
   </p>
   <p class="graf graf--p graf-after--p" id="3657">
    Golan 说，在他的公司长达几个月在也门期间，他的团队负责了一些高调的暗杀，尽管他拒绝具体说明是哪些。他认为美国需要一个类似于他所部署模式的暗杀计划。他说， “也许我是个怪物。也许我应该入狱。也许我是个坏人。但我是对的。”
   </p>
   <p class="graf graf--p graf-after--p" id="e4e4">
    Spear 行动小组的
    <strong class="markup--strong markup--p-strong">
     私人暗杀任务标志着改变全球战争方式的三个发展的汇合
    </strong>
    ：
   </p>
   <ul class="postList">
    <li class="graf graf--li graf-after--p" id="bf78">
     现代反恐战斗已经从传统的军事目标 — 例如摧毁机场、军事基地或营房 — 转移到了杀害特定个人，在很大程度上是为将战争重组的有组织暗杀。
    </li>
    <li class="graf graf--li graf-after--li" id="1499">
     战争越来越私有化，许多国家将大多数军事支持服务外包给私人承包商，使前线作战几乎成为美国和其他许多军队没有向营利性企业外包的唯一功能。
    </li>
    <li class="graf graf--li graf-after--li" id="c073">
     美国在阿富汗和伊拉克的长期战争严重依赖精锐的特种部队，产生了
     <strong class="markup--strong markup--li-strong">
      数万名训练有素的美国突击队员（杀人机器），他们可以根据国防合同要求私人部门的高额工资、或纯粹的雇佣军工作
     </strong>
     。
    </li>
   </ul>
   <p class="graf graf--p graf-after--li" id="4cfa">
    随着 Spear 行动小组在也门的任务，上述这些趋势融合成了一种新的和煽动性的业务：由经验丰富的美国战斗人员进行的
    <strong class="markup--strong markup--p-strong">
     军事化合同杀戮
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="8596">
    专家们表示，几乎不可思议的是，美国不会知道阿联酋 — 美国在几乎每个层面都训练过的军队 — 雇用了一支由美国退伍军人组成的公司，在一场密切监视的战争中进行暗杀计划。
   </p>
   <p class="graf graf--p graf-after--p" id="3b69">
    据熟悉该行动的三位消息人士透露，其中一名雇佣兵曾与中央情报局的“地面分支”合作，该机构相当于军方的特种部队。另一位是马里兰州陆军国民警卫队的特种部队中士。据知道他的四个人说，另一个人
    <strong class="markup--strong markup--p-strong">
     仍然在
    </strong>
    海军预备役中作为海豹突击队员并且有一个绝密的许可。消息人士告诉 BuzzFeed News，他是 SEAL Team 6 或 DEVGRU 的资深人士，这是“纽约时报”曾描述过的以杀害奥萨马·本·拉登而闻名的精英部队，作为一个“
    <strong class="markup--strong markup--p-strong">
     外
    </strong>
    <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2015/06/07/world/asia/the-secret-history-of-seal-team-6.html" href="https://www.nytimes.com/2015/06/07/world/asia/the-secret-history-of-seal-team-6.html" rel="noopener nofollow noreferrer" target="_blank">
     <strong class="markup--strong markup--p-strong">
      部监督极为有限的全球追捕机器
     </strong>
    </a>
    <strong class="markup--strong markup--p-strong">
    </strong>
    ”。
   </p>
   <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote graf-after--p" id="bae3">
    <p>
     <em class="markup--em markup--blockquote-em">
      “What vetting procedures are there to make sure the guy you just smoked is really a bad guy?”
     </em>
    </p>
   </blockquote>
   <p class="graf graf--p graf-after--blockquote" id="e324">
    美国中央情报局表示，它“没有”关于雇佣军暗杀计划的消息，海军的特种作战司令部拒绝发表评论。曾在阿联酋工作的前中央情报局官员最初告诉 BuzzFeed 新闻，美国人不可能被允许参加这样的行动。但是，经过检查后，他回电话说：“有些人基本上的确和你说的一样做了那些事”。他说，通过他所了解到的东西，他感到很惊讶：“你如何确保那个街边吸烟的人真的是一个坏人？“ 他说，
    <strong class="markup--strong markup--p-strong">
     雇佣兵 ”几乎就像一个谋杀小组
    </strong>
    ”。
   </p>
   <p class="graf graf--p graf-after--p" id="eef2">
    更令人惊讶的是，Spear 的雇佣兵行动是否违反美国法律都完全不清楚。一方面，美国法律规定在另一个国家“ 串谋杀害、绑架、伤害 ”某人是违法的；向外国提供军事服务的公司应该由国务院监管，国务院表示从未授权任何公司向另一个国家提供作战部队或雇佣军。
   </p>
   <p class="graf graf--p graf-after--p" id="5311">
    但是，另一方面，
    <a class="markup--anchor markup--p-anchor" data-href="https://www.buzzfeednews.com/article/aramroston/stephen-toumajan-general-us-uae-yemen-contractor" href="https://www.buzzfeednews.com/article/aramroston/stephen-toumajan-general-us-uae-yemen-contractor" rel="noopener nofollow noreferrer" target="_blank">
     <strong class="markup--strong markup--p-strong">
      美国并没有禁止雇佣军
     </strong>
    </a>
    。除了一些例外，在外国军队服役是完全合法的，无论是出于理想主义还是金钱。在没有法律后果的情况下，美国人曾在以色列国防军、法国外籍军团、甚至是在叙利亚与 ISIS 作战的民兵中服役。据三位消息人士称，Spear 行动小组安排阿联酋为参与执行任务的美国人提供军衔，这样就能为他们提供法律保障。
   </p>
   <p class="graf graf--p graf-after--p" id="79ef">
    尽管 Golan 游走在法律和政治的灰色地带，但他预示他的有针对性的暗杀品牌将成为一种精确的“反恐战略”，自称“减少平民伤亡”。但
    <strong class="markup--strong markup--p-strong">
     对 Mayo 的暗杀行动表明，这种新形式的战争带来了许多同样的老问题：突击队的计划出了问题，情报证明是有缺陷的，他们的袭击远非外科手术级的：他们附着在门上的爆炸物旨在杀死一个人而不是办公室里的每个人
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="a0e9">
    除了道德上必须反对之外，
    <strong class="markup--strong markup--p-strong">
     以营利为目标的暗杀事件也为现代战争增添了新的困境
    </strong>
    。私人雇佣兵在美国军方的指挥系统之外运作，所以如果他们犯错或犯下战争罪，就没有明确的制度让他们对此负责。如果雇佣军在街上杀死了一名平民，谁来调查？
   </p>
   <p class="graf graf--p graf-after--p" id="7ea0">
    暗杀 Mayo 的任务揭示了一个
    <strong class="markup--strong markup--p-strong">
     更为核心的问题：目标的选择
    </strong>
    。Golan 坚称，他只杀死了阿联酋政府确定的‘恐怖分子“，因为阿联酋是美国的盟友。但究竟谁是恐怖分子谁是政治家呢？什么是新的战争形式，什么是老式的谋杀？谁有权来决定谁死谁活？ — 不仅是在像阿联酋这样的秘密君主制的战争中，还有在像美国这样的民主国家的战争中？
   </p>
   <p class="graf graf--p graf-after--p" id="df5c">
    BuzzFeed News 将该公司袭击 Al-Islah 总部的内幕故事拼凑在一起，揭示了当今雇佣军战争的样子 — 以及它可能变成什么样。
   </p>
  </div>

    <p class="graf graf--p graf-after--figure" id="df68">
     这项将美国雇佣兵带到 Aden 街头的协议在阿布扎比的午餐时间里被敲定，这是阿联酋军事基地军官俱乐部的一家意大利餐馆。Golan 和一位棱角分明的专门从美国赶来的前美国海军海豹突击队员艾萨克·吉尔摩尔（Isaac Gilmore），正如 Gilmore 回忆的那样，开端不祥。
    </p>
    <p class="graf graf--p graf-after--p" id="401c">
     他们的东道主是 Mohammed Dahlan, 就是上图中间那位，此人是巴勒斯坦权力机构最可怕的前安防负责人。穿着一套精心剪裁的西装，他冷冷地看着他的雇佣兵客人，并告诉 Golan，在另一种情况下，他们会试图互相残杀。
    </p>
    <p class="graf graf--p graf-after--p" id="3084">
     事实上，他们不太可能配成一对。出生在匈牙利犹太家庭的 Golan 在以色列的安防业务方面保持着长期的联系，他说他在那里住了好几年。根据 2008 年他母亲的一篇文章，Golan 曾与前以色列政治家摩萨德的 Danny Yatom 在伦敦聚会，他的专长是“
     <a class="markup--anchor markup--p-anchor" data-href="https://www.motherjones.com/politics/2008/11/kurdistan-k-street-2/" href="https://www.motherjones.com/politics/2008/11/kurdistan-k-street-2/" rel="noopener nofollow noreferrer" target="_blank">
      为非洲的能源客户提供安全保障
     </a>
     。”根据三个消息来源，他的一份合同是保护钻探的船只。尼日利亚的海上油田曾遭到破坏和恐怖主义袭击。
    </p>
    <p class="graf graf--p graf-after--p" id="0d9f">
     Golan 留着胡子，抽着万宝路，浑身散发着热情。一位前中央情报局官员称 Golan 是一位优秀的推销员，并引用了小说家和哲学家安德烈·马尔罗（AndréMalraux）的话说：“一个人不是他所认为的那样，而是他试图隐藏的东西。”
    </p>
    <p class="graf graf--p graf-after--p" id="ce59">
     Golan 说，他在法国接受教育，加入了法国外籍军团，并在世界各地旅行，经常打仗或执行安防合同。他说，在贝尔格莱德，他了解臭名昭著的准军事斗士和匪徒 ŽeljkoRažnatović，更为人所知的是 Arkan，他于2001年被暗杀。“我非常尊重 Arkan，”他告诉 BuzzFeed News。
    </p>
    <p class="graf graf--p graf-after--p" id="85ba">
     BuzzFeed News 无法核实 Golan 的部分传记，包括他的军队服务经历，但 Gilmore 和另外一位与他一起在场的美国特种作战老兵说，很明显他有战斗经验。这位前中央情报局官员说，他被认为是
     <strong class="markup--strong markup--p-strong">
      能胜任的、无情的和精于算计的
     </strong>
     ；另一位前中央情报局官员说，他举止“容易夸张”，但“
     <strong class="markup--strong markup--p-strong">
      正是因为那些疯狂的计划，他就是你要雇佣的那种人
     </strong>
     。”
    </p>
    <p class="graf graf--p graf-after--p" id="292b">
     Dahlan 没有回应评论请求，他们在加沙的同一个难民营长大，在20世纪80年代的起义期间，他成为了一名重要的政治角色。在90年代，他被任命为巴勒斯坦权力机构在加沙的安全负责人，负责监督1995年和1996年对哈马斯的严厉镇压。他后来与乔治·W·布什总统会面，并与中央情报局建立了牢固的关系，多次与该机构的主任乔治·特尼特（George Tenet）会面。Dahlan 曾被吹捧为巴勒斯坦权力机构的可能领导人，但在 2007 年，他在巴勒斯坦权力机构的腐败指控和哈马斯与中央情报局和以色列的合作中失败。
    </p>
    <p class="graf graf--p graf-after--p" id="7f9e">
     一个没有国籍的人，他逃往阿联酋。据报道，他在那里重新成为了王储穆罕默德·本·扎耶德·阿勒纳哈扬（即 被称为阿布扎比真正统治者的 MBZ）的
     <a class="markup--anchor markup--p-anchor" data-href="https://www.jpost.com/Blogs/A-Mid-East-Journal/The-controversial-Mohammed-Dahlan-463624" href="https://www.jpost.com/Blogs/A-Mid-East-Journal/The-controversial-Mohammed-Dahlan-463624" rel="noopener nofollow noreferrer" target="_blank">
      重要顾问
     </a>
     。了解 Dahlan 的前中央情报局官员说：“阿联酋把他当作了自己的斗牛犬。”
    </p>
    <p class="graf graf--p graf-after--p" id="e481">
     回到故事的场景。现在，在官员俱乐部的午餐时间，斗牛犬正在挑战他的访客，要求这些来自美国的战士告诉他，他们有什么特别之处，为什么他们比阿联酋士兵更好？
    </p>
    <p class="graf graf--p graf-after--p" id="0ca7">
     Golan 用虚张声势的腔调回答道，他可以比阿联酋军队中的任何人更好地射击、训练、奔跑和战斗，Golan 说：随便挑你认为最出色的人，我肯定能打败他。任何人。
    </p>
    <p class="graf graf--p graf-after--p" id="fd11">
     巴勒斯坦人示意坐在附近的一位细心的年轻女助手，Dahlan 说，她就是我认为的最好的人。
    </p>
    <p class="graf graf--p graf-after--p" id="94cb">

    <p class="graf graf--p graf-after--figure" id="3fad">
     阿联酋拥有丰富的财富，但只有大约 100 万公民，依靠来自世界各地的农民工做各种事，从清洁厕所到教大学生。它的军队也没有什么不同，向急切的美国国防公司和前将军支付了巨额资金。自 2009 年以来，美国国防部已批准向阿联酋提供至少
     <a class="markup--anchor markup--p-anchor" data-href="https://www.ciponline.org/images/uploads/actions/Bill_Hartung_UAE_Arms_Report_92117.docx.pdf" href="https://www.ciponline.org/images/uploads/actions/Bill_Hartung_UAE_Arms_Report_92117.docx.pdf" rel="noopener nofollow noreferrer" target="_blank">

     </a>

    </p>
    <p class="graf graf--p graf-after--p" id="3e8b">
     退役的美国陆军将军 Stanley McChrystal 曾签约参加阿联酋军事公司的董事会。前海军海豹突击队队员和海军少将 Robert Harward 负责管理著名军火商洛克希德马丁的阿联酋分部。安全执行官 Erik Prince（现在牵涉到特别律师罗伯特·穆勒对俄罗斯选举干涉的调查）在那里做了一段时间的生意，帮助阿联酋雇佣哥伦比亚的雇佣兵。
    </p>
    <p class="graf graf--p graf-after--p" id="6d89">
     正如 BuzzFeed News 今年早些时候报道的那样，该国将外国人纳入其军队，并将一般的少将军衔给了美国中校 Stephen Toumajan，
     <a class="markup--anchor markup--p-anchor" data-href="https://www.buzzfeednews.com/article/aramroston/stephen-toumajan-general-us-uae-yemen-contractor" href="https://www.buzzfeednews.com/article/aramroston/stephen-toumajan-general-us-uae-yemen-contractor" rel="noopener nofollow noreferrer" target="_blank">
      让他担任
     </a>
     武装部队的一个分支的领导人。
    </p>
    <p class="graf graf--p graf-after--p" id="7948">
     阿联酋在使用国防承包商方面并不孤单; 事实上，
     <strong class="markup--strong markup--p-strong">
      正是美国帮助开创了全球军事私有化的举措
     </strong>
     。五角大楼支付公司执行许多传统功能，从喂养士兵到维护武器再到守卫车队。
    </p>
    <p class="graf graf--p graf-after--p" id="83c2">
     美国在战斗中划了一条线; 它不雇用雇佣兵进行攻击或直接参与战争，但这条线正在变得模糊。
     <a class="markup--anchor markup--p-anchor" data-href="https://www.fbo.gov/index?s=opportunity&amp;mode=form&amp;tab=core&amp;id=6247a35a5d9816a4b5e4f067b8758ecc&amp;_cview=0" href="https://www.fbo.gov/index?s=opportunity&amp;mode=form&amp;tab=core&amp;id=6247a35a5d9816a4b5e4f067b8758ecc&amp;_cview=0" rel="noopener nofollow noreferrer" target="_blank">
      私营公司提供
     </a>
     全副武装的安保细节，以保护战区外交官或外地情报人员。这些承包商可以像在利比亚班加西那样进行交火，当时有两名承包商在2012年因保护 CIA 而死。但是，正式来说，
     <strong class="markup--strong markup--p-strong">
      任务是保护，而不是战争！
     </strong>
    </p>
    <p class="graf graf--p graf-after--p" id="cf33">
     在美国以外，雇用雇佣兵进行战斗任务的情况很少见，尽管已经发生了。据报道，在尼日利亚，一支长期由南非雇佣军 Eeben Barlow 领导的攻击部队于 2015 年成功地对抗了伊斯兰激进组织 Boko Haram（博科圣地）。Barlow 又成立了一家名为 Executive Outcomes 的公司，在惨遭战争蹂躏的塞拉利昂镇压了血腥的 RUF 反叛力量在20世纪90年代。
    </p>
    <p class="graf graf--p graf-after--p" id="55eb">
     但是对于 Dahlan 来说，Golan 和 Gilmore 的意大利面餐会提供了一种
     <strong class="markup--strong markup--p-strong">
      特殊形式
     </strong>
     的雇佣兵服务。这不是提供安全细节，也不是传统的军事斗争或反叛乱战争。Golan 和 Gilmore 都承认，
     <strong class="markup--strong markup--p-strong">
      这是有针对性的杀戮
     </strong>
     。
    </p>
    <p class="graf graf--p graf-after--p" id="0cdb">
     Gilmore 说他不记得有人特意使用“暗杀”这个词。但是，从第一次会面中可以清楚地看出，这不是关于夺取或拘留 Al-Islah 的领导权，“目标是特定的，” Gilmore 说。Golan 说他被明确告知，帮助“扰乱和毁灭” Al-Islah，它被称之为“恐怖组织的政治分支”。
    </p>
    <p class="graf graf--p graf-after--p" id="755c">
     他和 Gilmore 承诺，他们可以迅速将具有合适技能的团队拉到一起。
    </p>
    <p class="graf graf--p graf-after--p" id="3d74">
     在午餐后的几个星期里，他们敲定了合同。Golan 和 Gilmore 告诉 BuzzFeed News，该团队
     <strong class="markup--strong markup--p-strong">
      每月将获得150万美元
     </strong>

    </p>
    <p class="graf graf--p graf-after--p" id="a132">
     Golan 和 Gilmore 还有另一个条件：他们希望被纳入阿联酋武装部队。他们希望他们的武器 — 以及他们的目标清单 — 来自穿制服的军官。这是“出于司法原因，” Golan 说，“因为一旦栽了，阿联酋的制服和徽章会标志着“雇佣兵和军人之间的区别”。
    </p>
    <p class="graf graf--p graf-after--p" id="b954">

      回到美国后，Golan 和 Gilmore 开始招募。Spear Operations Group 是一家小公司 — 没有像加尔达世界安保或 Constellis 这样的大块头 — 但它有大量的人才可供选择。

      <strong class="markup--strong markup--p-strong">
       所谓“反恐战争”的一个鲜为人知的后果，特别是美国在伊拉克和阿富汗战争的 17 年，是特种作战部队的数量自 9/11 以来增加了一倍以上
      </strong>
      ，从 33,000 增加到 70,000。这是由美国军方最精锐部队挑选、训练和战斗测试的高精尖士兵，如海军海豹突击队和陆军游骑兵队。一位不愿透露姓名的海豹突击队高级官员表示，
      <strong class="markup--strong markup--p-strong">
       一些特种作战预备役人员当雇佣兵的事几乎是公开的秘密
      </strong>
      ，“我知道有很多人做过这类事，”他说，如果士兵没有现役，他们就没有义务报告他们正在做什么。
     </p>
     <p class="graf graf--p graf-after--p" id="cb74">
      但特种作战退伍军人和预备役军人的选择并不像伊拉克战争初期那样。私人保安工作主要是在敌对环境中保护美国政府官员，缺乏实战的兴奋，也没有付出过去的代价。承包商说，虽然负责高端安保工作的精英退伍军人的起薪率通常为每天 700 美元或 800 美元，但现在这些费率已降至每天 500 美元左右。Golan 和 Gilmore 说，
      <strong class="markup--strong markup--p-strong">
       他们每月向美国战士提供 25,000 美元 — 每天约 830 美元 — 加上奖金，几乎是任何最慷慨的市场的总和
      </strong>
      。
     </p>
     <p class="graf graf--p graf-after--p" id="5b9c">
      尽管如此，也门的参战仍然进入了未知的领域，一些最优秀的士兵拒绝了。“也有钱解决不了的事”，Gilmore 说。
     </p>
     <p class="graf graf--p graf-after--p" id="af35">
      Gilmore 自己说他的记录也不完美。在海军时代他执行的实弹射击训练任务期间，他说他不小心击中了另一个海豹突击队员。Gilmore 说，这就是促使他在 2011 年离开海军的原因。他在加入 Spear 之前的最后一项重要工作是作为手工龙舌兰酒公司
      <a class="markup--anchor markup--p-anchor" data-href="http://www.marketwired.com/press-release/montalvo-spirits-appoints-isaac-gilmore-as-chief-business-development-officer-otcqb-tqla-1917210.htm" href="http://www.marketwired.com/press-release/montalvo-spirits-appoints-isaac-gilmore-as-chief-business-development-officer-otcqb-tqla-1917210.htm" rel="noopener nofollow noreferrer" target="_blank">
       的高管
      </a>
      。
     </p>
     <p class="graf graf--p graf-after--p" id="1b44">
      他说，他的军事生涯中的那种污点也促使他加入 Spear 的冒险：他是一个局外人，他不在储备中，而且没有养老金问题需要担心。
     </p>
     <p class="graf graf--p graf-after--p" id="0e1d">
      到 2015 年底，领导这项行动的 Golan 和 Gilmore 拼凑了一支由十几名男子组成的团队。其中三位是美国特种部队退伍军人，其余大部分都是前法国外籍退伍军人，
      <strong class="markup--strong markup--p-strong">
       他们更便宜
      </strong>
      ：每月只有大约1万美元，正如 Gilmore 记得的那样，这些人的薪水不到他和 Golan 为美国同行预算的一半。
     </p>
     <p class="graf graf--p graf-after--p" id="3b5c">
      这些人聚集在新泽西州泰特波罗机场附近的一家酒店。他们穿着各式各样的军装，有些穿着迷彩服，也有些穿着全黑。有些人留着胡须、肌肉丰满，有些人带着纹身，看着就很壮实。
     </p>
     <p class="graf graf--p graf-after--p" id="38c5">
      Gilmore 说，他们说服酒店的工作人员把外面挂着的美国国旗借给他们用一下。在一个临时的仪式上，他们将国旗折叠成一个小三角形并佩戴在身上。
     </p>
     <p class="graf graf--p graf-after--p" id="f49c">
      他们还包装了几个星期用的军事“即食餐”、防弹衣、通讯装备和医疗设备。肯定还有威士忌 — 三个 Basil Hayden，因为在也门不可能喝酒，更不用说别的好东西了。
     </p>
     <p class="graf graf--p graf-after--p" id="5b85">

      <p class="graf graf--p graf-after--p" id="bfaf">
       从历史上看，暗杀在美国战争和外交政策中起到了有限的作用。1945 年，中央情报局前任机构 OSS 的负责人、号称“狂野比尔”多的诺万提交了一份最终计划，在整个欧洲部署杀戮部队，以攻击纳粹领导人，如希特勒、希姆莱和戈林，以及党卫队官员，根据 Douglas Waller 的
       <a class="markup--anchor markup--p-anchor" data-href="https://www.amazon.com/Wild-Bill-Donovan-Spymaster-Espionage/dp/1416567445/ref=tmm_hrd_swatch_0?_encoding=UTF8&amp;qid=1539032910&amp;sr=1-1" href="https://www.amazon.com/Wild-Bill-Donovan-Spymaster-Espionage/dp/1416567445/ref=tmm_hrd_swatch_0?_encoding=UTF8&amp;qid=1539032910&amp;sr=1-1" rel="noopener nofollow noreferrer" target="_blank">

       </a>
       记载。但 OSS 负责人对“批量暗杀”项目感到不安并取消了它。
      </p>
      <p class="graf graf--p graf-after--p" id="84d0">
       <strong class="markup--strong markup--p-strong">
        在冷战期间，中央情报局在暗杀外国领导人的阴谋中发挥了重要作用
       </strong>
       ，如刚果民主共和国的帕特里斯卢蒙巴、多米尼加共和国的拉斐尔特鲁希略、和南越的 Ngo Dinh Diem。在越南战争的后期，美国启动了凤凰计划，其中中央情报局经常与美国军方联手“废止” — 或者干脆直说，暗杀 — 越共领导人。即便如此，有针对性的杀戮也并不是美国在越南的军事战略的核心支柱。在国会20世纪70年代揭露中央情报局的秘密活动之后，美国开始禁止暗杀外国领导人。
      </p>
      <p class="graf graf--p graf-after--p" id="d135">
       然后就是反恐战争了。
      </p>
      <p class="graf graf--p graf-after--p" id="689c">
       在乔治·W·布什总统的领导下，中央情报局和军方使用无人机杀死恐怖分子，中央情报局更加发展了秘密的暗杀能力。到了巴拉克•奥巴马（Barack Obama）时代停止了该机构的秘密暗杀计划，但大幅增加了对巴基斯坦、也门、阿富汗和索马里的无人机袭击的使用。很快，中央情报局和军方就开始使用飞机 — 远程视频监视器 — 通过仅基于目标协会和活动的 “signature strikes”
       <strong class="markup--strong markup--p-strong">
        来杀死美国自己甚至都不知道是谁的人
       </strong>
       。特朗普政府则进一步放松了无人机袭击的监管规则。
      </p>
      <p class="graf graf--p graf-after--p" id="a709">

       <strong class="markup--strong markup--p-strong">

       </strong>

       <a class="markup--anchor markup--p-anchor" data-href="https://worldview.stratfor.com/article/uaes-ulterior-motives-yemen" href="https://worldview.stratfor.com/article/uaes-ulterior-motives-yemen" rel="noopener nofollow noreferrer" target="_blank">

       。

       牛津大学也门专家 Elisabeth Kendall 指出，与基地组织或试图通过暴力夺取政权的其他恐怖组织
       <strong class="markup--strong markup--p-strong">

       </strong>
      </p>

       Golan 和 Gilmore 说，他们从阿联酋获得的首要目标就是 Al-Islah 在 Aden 的领导人 Mayo。Mayo 的头发简短，戴着金属丝眼镜，还有一缕山羊胡子。他在也门发表了反对美国无人机袭击的观点，并在
       <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/world/middle_east/in-yemen-us-airstrikes-breed-anger-and-sympathy-for-al-qaeda/2012/05/29/gJQAUmKI0U_story_2.html" href="https://www.washingtonpost.com/world/middle_east/in-yemen-us-airstrikes-breed-anger-and-sympathy-for-al-qaeda/2012/05/29/gJQAUmKI0U_story_2.html" rel="noopener nofollow noreferrer" target="_blank">

      </p>
      <p class="graf graf--p graf-after--p" id="9832">
       Golan 把自己的暗杀生意与以色列有针对性的杀戮计划做对比，该计划自以色列建国以来一直在进行，尽管有一些引人注目的错误和尴尬，但 Golan 声称自己已经做得很好了。他认为，有一些恐怖主义敌人如此危险和无情 — 并且如此难以逮捕 — 暗杀是最好的解决办法。

      </p>
      <p class="graf graf--p graf-after--p" id="549b">

       Gilmore 只呆了很短的时间。他说他在 2016 年4月离开了 Spear 暗杀小组。他和 Golan 拒绝透露原因，但 Gilmore 表示他希望他在也门更加积极。他说：“如果我能再做一遍，那么我们的风险就会减少。” “我们本可以做一些令人惊奇的事 — 虽然我们也可以做一些惊人的事，但最终都被关进了监狱。”
      </p>
      <p class="graf graf--p graf-after--p" id="e117">
       根据三位消息来源并通过照片证实，该团队的一名新成员于2016年初被聘用，是海豹突击队6队的老将 Daniel Corbett。Corbett 是一位出色的士兵，据那些了解他的人说，他曾在阿富汗和伊拉克进行过多次战斗之旅。他还在储备中，所以美军随时都可以部署他; 他领取了政府发放的工资; 并且他应该报告每月的演习。然而，他在也门签订了一份为外国军队工作的私人合同。目前还不清楚他自己是否参与暗杀任何人的任务。
      </p>
      <p class="graf graf--p graf-after--p" id="3fe2">
       在一个神秘的事件中，
       <a class="markup--anchor markup--p-anchor" data-href="https://www.buzzfeednews.com/article/aramroston/daniel-corbett-navy-seal-jail-serbia-mercenary-yemen" href="https://www.buzzfeednews.com/article/aramroston/daniel-corbett-navy-seal-jail-serbia-mercenary-yemen" rel="noopener nofollow noreferrer" target="_blank">
        Corbett 目前正在塞尔维亚的监狱里
       </a>
       ，在那里他因非法持有手枪罪名而被调查。自2018年2月以来，这位美国退伍军人一直被关押在那里。无法联系到 Corbett，他的律师没有回复寻求评论的电话。
      </p>
      <p class="graf graf--p graf-after--p" id="0369">
       当他们在也门工作时，雇佣兵们呆在小屋里，睡在狭窄的小床上。有些人携带着独特的武器来进行近距离战斗。根据照片，其中一个人在他的腰带上绑了两把刀，另一个带着一把小战斧。
      </p>
      <p class="graf graf--p graf-after--p" id="75dc">
       该团队开始开发 Gilmore 称之为 “esprit de corps”的东西。他们带着一个临时旗帜，上面有一个骷髅头骨和两把交叉的剑 — 一种黑色背景上的 Jolly Roger — 并将这个徽章涂在他们的军用车辆上和他们的生活区里。
      </p>
      <p class="graf graf--p graf-after--p" id="f51c">
       关于 Spear 雇佣军团队的许多事仍然不得而知，一些参与者明确表示他们不想琢磨失败的事。当被问及是否已被部署在也门任务中时，其中一位美国人回答说：
       <strong class="markup--strong markup--p-strong">
        “如果我是被部署的人，你知道，我无法讨论这个问题。
       </strong>
       ”在暗杀任务期间吮吸棒棒糖的前绿色贝雷帽给 BuzzFeed News 发了消息：“对你来说，一个重要的故事可能是一个人物角色的悲惨故事; 特别是如果他们是做正确的但不一定合法的事的好人。“
      </p>
      <p class="graf graf--p graf-after--p" id="7944">
       Gilmore 表示，“他本来希望暗杀行动不会受到关注。”但他决定与 BuzzFeed News 交谈，因为 “一旦出现这种情况，我就无法避开它了，所以，”我宁愿迎面接受。而且我不打算回避我做过的事。“
      </p>
      <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="e0a4">
       <strong class="markup--strong markup--p-strong">
        “暗杀仍然存在”，他说，“战争的未来有各种各样。”
       </strong>
      </p>
      <p class="graf graf--p graf-after--p" id="d8b9">
       Gilmore 已经不在雇佣军生意中了。从那以后，他发现自己处于另一个灰色地带的工作线上，尽管这个工作的危险性要小得多。他说他和一家加利福尼亚公司合作，计划为蒸发器生产大麻油。
      </p>
     </div>
