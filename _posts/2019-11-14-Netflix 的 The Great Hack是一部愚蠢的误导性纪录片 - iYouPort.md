    <span style="color: #00ccff;">
     <em>
      <strong>
       为了蹭热点而蹭热点是内容创作者能做的做糟糕的事。因为它只会将你的思考方式围绕在所谓的热点本身，从而忽略了造成悲剧的深远的背后问题。其结果是，你的作品不仅没有意义，而且还有可能具有严重的误导性
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-6991 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头-7.png?resize=1100%2C623&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头-7.png?w=1260&amp;ssl=1 1260w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头-7.png?resize=300%2C170&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头-7.png?resize=768%2C435&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头-7.png?resize=1024%2C580&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头-7.png?resize=1100%2C623&amp;ssl=1 1100w" data-recalc-dims="1" height="623" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头-7.png?resize=1100%2C623&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6991" data-recalc-dims="1" height="623" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头-7.png?resize=1100%2C623&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头-7.png?w=1260&amp;ssl=1 1260w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头-7.png?resize=300%2C170&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头-7.png?resize=768%2C435&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头-7.png?resize=1024%2C580&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/头-7.png?resize=1100%2C623&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   一部新的纪录片记录了剑桥分析公司的垮台。这是一家臭名昭著的政治咨询公司，利用数据入侵选民心理，以操纵政治。
  </p>
  <p class="graf graf--p">
   关于这种操纵是如何实现的，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" rel="noopener noreferrer" target="_blank">
    心理操纵的秘密：”行为经济学”+PR=？
   </a>
   》
  </p>
  <p class="graf graf--p">
   但是这部片子本身明显有问题。（然而其中文版却得到了一大堆好评，真的太令人心酸了）
  </p>
  <p class="graf graf--p">
   正如人们通常所理解的那样，“骇客”是指某人利用代码中的漏洞偷偷访问计算机系统或欺骗一个容易上当的用户泄露其凭据。
  </p>
  <p class="graf graf--p">
   要求计算机或社交网络的用户点击“我同意”按钮，然后收集他们的数据以操纵他们行为的家伙不是骇客。恰恰相反，
   <strong class="markup--strong markup--p-strong">
    正是互联网本身的商业模式
   </strong>
   。
  </p>
  <p class="graf graf--p">
   正如IYP类目“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e4%b8%8e%e6%93%8d%e7%ba%b5/" href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e4%b8%8e%e6%93%8d%e7%ba%b5/" rel="noopener noreferrer" target="_blank">
    监视与操纵
   </a>
   ”的说明中所强调的：监视是为了操纵，不只有放弃隐私那么简单。这个集中化的互联网前身就是一个大规模的监视项目，在这里看到具体分析《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e7%a9%b6%e7%ab%9f%e6%98%af%e4%bb%80%e4%b9%88%ef%bc%9f/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e7%a9%b6%e7%ab%9f%e6%98%af%e4%bb%80%e4%b9%88%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    互联网究竟是什么？
   </a>
   》
  </p>
  <p class="graf graf--p">
   剑桥分析公司是一家现已解散的政治咨询公司，利用的是 Facebook 本身的侵入式监视商业模式，它没有亲自入侵任何一台计算机。
  </p>
  <p class="graf graf--p">
   但是，该公司对人们的个人生活表现出了不同寻常的兴趣 —— 那些很乐意与他人分享自己私密信息的人。
  </p>
  <p class="graf graf--p">
   其实人们完全可以简单地通过该纪录片的名字看出其存在的问题。“The Great Hack”，居然是一部关于剑桥分析公司和“社交媒体黑暗面”的纪录片，这是彻底的跑题。
  </p>
  <p class="graf graf--p">
   这部电影主要关注三个角色。
  </p>
  <p class="graf graf--p">
   David Carroll（如上图）是一位美国教授，他向剑桥分析公司询问了他所掌握的信息，行使了1998年英国数据保护法案所载的权利。（信息专员办公室同意他对法律的解读，该权利适用于在英国处理数据的任何人）
  </p>
  <p class="graf graf--p">
   Carole Cadwalladr 是伦敦“观察家报”的记者：她对剑桥分析公司的报道在2018年引起了公众的注意，随后在英国和美国引发了多次调查，该公司最终消亡。
  </p>
  <p class="graf graf--p">
   Brittany Kaiser 是剑桥分析公司的前任商业发展总监，她缺乏道德指南针和阴暗的意图使她成为最有影响力的人物。她获得了最多的屏幕时间；制片人会把她和其他所有人的所有东西都当成面子拿来卖。
  </p>
  <p class="graf graf--p">
   就是这仨人。
  </p>
  <p class="graf graf--p">
   当 Carroll 说在线广告是一个价值数万亿美元的行业时，他并没有受到挑战（这是个没谱的数字）；
  </p>
  <p class="graf graf--p">
   当 Kaiser 声称“去年的数据在价值方面超过石油”时，电影制作人将这句话放在了他们的网站上（不存在这样的计量）；
  </p>
  <p class="graf graf--p">
   当 Cadwalladr 注意到根本性问题 即 “对人口进行大规模分析，然后确定对他们进行有目的的操纵的触发因素……感到对民主非常具有挑战性”时，并没有人指出：
   <strong class="markup--strong markup--p-strong">
    这就是互联网定位广告的意义所在
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这部电影的大部分内容都是由剑桥分析公司的销售平台组成的，该片导演似乎把这些东西视为了该公司的复杂和成功的“真相”。
  </p>
  <p class="graf graf--p">
   如果剑桥分析公司没有关闭，他们家老板完全可以使用这部电影 “The Great Hack” 作宣传片。
  </p>
  <p class="graf graf--p">
   至少对于这位评论家来说，目前还不清楚“The Great Hack”的重点是什么。
  </p>
  <p class="graf graf--p">
   如果要使用剑桥分析公司作为案例向更广泛的受众传达科技公司利用其用户数据的现实和不加思索地采用它的方式，那么这部片子
   <strong class="markup--strong markup--p-strong">
    只能算是打败了一匹病重垂死的老马
   </strong>
   。
  </p>
  <p class="graf graf--p">
   今天的公民已经非常熟悉社交网络上的错误信息和标题党；面部识别和人工智能现在引起最多的担忧。监视就是为了操纵。
  </p>
  <p class="graf graf--p">
   而这部片子几乎完全专注于剑桥分析公司，除了蹭热点想不到其他意义了。
  </p>
  <p class="graf graf--p">
   它可能会形成一个整洁的叙事弧线，但是，这意味着“The Great Hack”几乎无法解决更广泛的监视生态系统的本质性问题，或者说直接说：关于为什么今天的互联网是如此悲惨的地方。
  </p>
  <p class="graf graf--p">
   等扯多了就继续使用混淆和暗示的方法对俄罗斯人或民粹主义者喋喋不休。全部是毫无意义。
  </p>
  <p class="graf graf--p">
   关于 Whatsapp 在巴西的宣传活动可能是个问题，但它们与 Facebook 及其同类寡头的数据挖掘和算法操纵来说，根本不值一提。
  </p>
  <p class="graf graf--p">
   看看
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e8%b0%b7%e6%ad%8c/" href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e8%b0%b7%e6%ad%8c/" rel="noopener noreferrer" target="_blank">
    谷歌
   </a>
   是怎么做的吧，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" rel="noopener noreferrer" target="_blank">
    看不见的独裁
   </a>
   》。
  </p>
  <p class="graf graf--p">
   剑桥分析公司通过偷偷摸摸的方式获取了数千万美国人的 Facebook 数据，能够创建详细的个人资料，为候选人提供政治策略以获胜；更重要的是，剑桥分析公司
   <strong class="markup--strong markup--p-strong">
    只是在线操纵的一小部分
   </strong>
   ——
   <strong class="markup--strong markup--p-strong">
    通过可疑的“同意”来监视地球上的每个人，这个问题本身才是最值得讲述的故事
   </strong>
   。
  </p>
  <p class="graf graf--p">
   关于为什么剑桥分析公司只是一小部分，这里有一个案例《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a7%98%e5%af%86%e5%ae%a3%e4%bc%a0%e7%bd%91%e7%bb%9c/" href="https://www.iyouport.org/%e7%a7%98%e5%af%86%e5%ae%a3%e4%bc%a0%e7%bd%91%e7%bb%9c/" rel="noopener noreferrer" target="_blank">
    秘密宣传战网络
   </a>
   》；剑桥分析公司不仅仅是一家公司，而是政府国防机构的一部分，这就是为什么该公司可以迅速通过解散来避免被深究，并且，迅速组成新公司。在这里看到幕后黑手《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" rel="noopener noreferrer" target="_blank">
    心理操纵的秘密
   </a>
   》。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “The Great Hack”这部片子毫无意义，丝毫没有触及关键问题；不过也许挺适合 Facebook 时代的。
  </p>
  <p class="graf graf--p">
   就像在社交网络上嘈杂的争论那样，乏味的、只是看似复杂但智力不发达的，挤满了虚假的事实和夸大的统计数据，并且有几个吹牛人在自以为是和自我祝贺之间两头忙。
  </p>
  <p class="graf graf--p">
   就像其中一个论点那样，没有人全身而退。⚪️
  </p>
