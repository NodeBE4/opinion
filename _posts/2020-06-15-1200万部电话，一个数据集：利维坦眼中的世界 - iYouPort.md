  <p class="graf graf--p">
   每一天，在地球上的每一个角落，数十家寡头公司每时每刻都在记录着数千万人的活动，并将其信息存储在巨大的数据库中。这些公司几乎没有受到任何监督。
  </p>
  <p class="graf graf--p">
   时报隐私项目获得了一个这样的文件，
   <strong class="markup--strong markup--p-strong">
    是迄今为止最大的、最敏感的文件
   </strong>
   。
  </p>
  <p class="graf graf--p">
   跨越多个主要城市（包括华盛顿、纽约、旧金山和洛杉矶）超过1200万人的电话被收集了超过500亿个位置ping。
  </p>
  <p class="graf graf--p">
   该文件中的每条信息代表2016年和2017年几个月内单个智能手机的精确位置。
  </p>
  <p class="graf graf--p">
   数据是由匿名人士提供给 Times Opinion，他们要求匿名，因为他们无权共享这些数据，暴露身份可能会让他们受到严厉惩罚。消息来源说，他们对如何滥用这些数据感到震惊，并迫切希望向公众和立法者通报情况。
  </p>
  <p class="graf graf--p">
   在花了几个月的时间筛选数据、跟踪全国各地的人员流动、并与研究该领域的数十家数据公司、技术人员、律师和学者交谈之后，我们感到了同样的担忧。
  </p>
  <p class="graf graf--p">
   在数据文件覆盖的城市中，
   <strong class="markup--strong markup--p-strong">
    它跟踪几乎每个街区每个角落的所有人
   </strong>
   ，无论这些人是住在弗吉尼亚州亚历山大的移动房里的穷人还是住在曼哈顿豪华大厦中的有钱人。
  </p>
  <p class="graf graf--p">
   一次搜索发现，有十几个人造访了花花公子大厦，其中一些人在那里过夜。毫不费力就可以发现谁造访了约翰尼·德普（Johnny Depp）、泰格·伍兹（Tiger Woods）、和阿诺·施瓦辛格（Arnold Schwarzenegger）的地产，将设备的所有者简单地连接到具体地点。
  </p>
  <p class="graf graf--p">
   如果您居住在该数据集覆盖的城市之一，并使用了共享您的位置的应用程序 —— 从天气应用程序、到本地新闻应用程序、到优惠券保存程序的
   <strong class="markup--strong markup--p-strong">
    任何东西
   </strong>
   —— 您也在其中。
  </p>
  <p class="graf graf--p">
   如果您可以看到这个数据集的整个功能，那么您可能永远也不会再以相同的方式使用手机了。
  </p>
  <p class="graf graf--p">
   下面是纽约大中央总站的典型一天：
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="618" src="//player.vimeo.com/video/384335621?title=0&amp;byline=0&amp;portrait=0&amp;color=8dc7dc" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   请注意：数据并非来自电信或大型科技公司，也不来自政府监视部门。
  </p>
  <p class="graf graf--p">
   它起源于一个位置数据公司，该公司是
   <strong class="markup--strong markup--p-strong">
    数十个
   </strong>
   使用安装在手机应用程序上的软件
   <strong class="markup--strong markup--p-strong">
    悄悄收集实时的精确位置
   </strong>
   的监视公司之一。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    您可能从未听说过大多数公司，但是对于任何有权访问此数据的人来说，您的一切私生活都是完全透明的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   他们可以看到您每天造访的地方、您与谁会面或一起过夜的地方、您祈祷的地方，无论是去戒毒所、精神病医生的办公室、还是按摩院。
  </p>
  <p class="graf graf--p">
   很多媒体过去曾报道过智能手机跟踪。但是如此大的数据集是首次被曝光。
  </p>
  <p class="graf graf--p">
   即便如此，该文件仍只是位置跟踪行业每天收集和销售的所有东西中
   <strong class="markup--strong markup--p-strong">
    一小部分
   </strong>
   —— 监控在我们的数字生活中无处不在，现在似乎任何人都无法避免。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种永远在线的监视生活让整个世界都变成了极权体制
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在美国这种所谓的代议制民主国家中，如果政府试图强制要求每个12岁以上的人都携带一个追踪设备，每天24小时显示其位置，那么公民肯定会大为恼火。但是，自从苹果公司的 App Store 创建以来的十年中，人们已经主动同意了接受这种噩梦式的极权主义监视。
  </p>
  <p class="graf graf--p">
   如今，数以千万计的美国人，包括许多孩子，几乎白天的每时每刻都在自己的口袋里揣着一个间谍，而在晚上就把这个间谍放在床旁。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “对这些消费产品的诱惑是如此强大，以至于我们看不到存在另一种在不侵犯隐私的情况下获得该技术好处的可能性。但确实存在。” 堪萨斯大学监视研究中心创始人 William Staples 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “所有收集此位置信息的公司都像我所说的 Tiny Brothers 一样，使用各种数据海绵疯狂吸取数据，以进行每时每刻的监视。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    甚至，收集和出售所有这些信息完全合法
   </strong>
   。在美国 —— 就像在世界上大多数地方一样 —— 没有任何联邦法律可以限制这种疯狂的追踪，它已经成为一项庞大而利润丰厚的交易。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    你的一举一动的实时记录
   </strong>
  </h3>
  <p class="graf graf--p">
   收集有关您的一举一动的所有信息的公司基于以下三个主张来证明其业务合理：人们同意进行跟踪、数据是匿名的、数据是安全的。
   <strong class="markup--strong markup--p-strong">
    全都是谎言
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这个位置数据集包含数十亿个数据点，尽管没有诸如姓名或电子邮件地址之类的可识别信息；但是，将真实姓名与地图上显示的圆点联系起来是小孩子都能做到的事。
  </p>

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e8%a6%81%e4%bd%a0%e8%bf%9c%e7%a6%bb%e5%b9%bf%e5%91%8a%e7%89%8c-%e8%80%81%e5%a4%a7%e5%93%a5%e6%9d%a5%e4%ba%86/" href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e8%a6%81%e4%bd%a0%e8%bf%9c%e7%a6%bb%e5%b9%bf%e5%91%8a%e7%89%8c-%e8%80%81%e5%a4%a7%e5%93%a5%e6%9d%a5%e4%ba%86/" rel="noopener noreferrer" target="_blank">
     广告牌如何变得很危险
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   在没有您的精确位置的情况下，许多使用您的位置的应用程序（例如气象服务）都可以很好地运行，但是，
   <strong class="markup--strong markup--p-strong">
    收集您的位置将为分析、许可和将信息传递给第三方带来丰厚的利润。
   </strong>
  </p>
  <figure aria-describedby="caption-attachment-12178" class="wp-caption aligncenter" id="attachment_12178" style="width: 1180px">

  <p class="graf graf--p">
   对于一些人而言，他们暴露信息所面临的唯一真正风险将是尴尬或不便。但是，对于其他人，例如遭受虐待的幸存者、抗议者、记者、活动家 ……风险可能是巨大的。
  </p>
  <p class="graf graf--p">
   在一个案例中观察到 Microsoft 工程师的日常活动发生了变化。一个星期二下午，他访问了微软竞争对手亚马逊的西雅图主要校园。第二个月，他在亚马逊开始了新工作。只需几分钟时间就可以确定他是现任无人机交付服务 Amazon Prime Air 的经理 Ben Broili。
  </p>
  <p class="graf graf--p">
   Broili 说：“我不能感到惊讶。但是知道所有人都可以掌握我的工作和生活地点，这很奇怪。”
  </p>
  <figure aria-describedby="caption-attachment-12179" class="wp-caption aligncenter" id="attachment_12179" style="width: 2000px">

  <p class="graf graf--p">
   如果使用这种位置数据可以轻松地跟踪大公司员工，那么跟踪名人也同样容易。狗仔队的无人机监视方法简直弱爆了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    希望通过亲自与消息来源会面来逃避其他形式的监视的记者，可能需要重新考虑这种做法了
   </strong>
   。该数据集覆盖的每个主要新闻编辑室都包含数十个 ping。
  </p>
  <p class="graf graf--p">
   许多这类公司确实出售详细数据。买方通常是数据经纪人和广告公司。但其中一些与消费者广告无关，包括可以处理和分析大量信息的金融机构、地理空间分析公司和房地产投资公司。
  </p>
  <p class="graf graf--p">
   据一位同意匿名讲话的前位置数据公司员工称，⚠️
   <strong class="markup--strong markup--p-strong">
    他们可能会为一批数据支付超过100万美元
   </strong>
   。
  </p>
  <p class="graf graf--p">
   位置数据也会与移动广告ID一起收集和共享，移动广告ID大约30位长，允许广告客户和其他企业将应用之间的活动联系在一起。
  </p>
  <p class="graf graf--p">
   该ID还用于将位置信息与其他信息（例如您的姓名、家庭住址、电子邮件、电话号码、甚至是与您的Wi-Fi网络绑定的标识符）结合起来。
   <strong class="markup--strong markup--p-strong">
    也就是您完全裸奔。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这些数据几乎可以实时地进行交易，因此您的位置可以从智能手机转移到应用程序的服务器，并可以在几毫秒内导出到第三方。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    然后再转售、复制、盗用和滥用该数据。您永远无法找到它。
   </strong>
  </p>
  <p class="graf graf--p">
   位置数据远比消费者看到更多相关广告要严重得多。该信息为大企业提供了关键情报。
  </p>
  <p class="graf graf--p">
   例如，去年在洛杉矶
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2019/01/03/technology/weather-channel-app-lawsuit.html" href="https://www.nytimes.com/2019/01/03/technology/weather-channel-app-lawsuit.html" rel="noopener noreferrer" target="_blank">
    提起的诉讼
   </a>
   显示，Weather Channel 应用是美国最受欢迎的天气软件应用之一，诉讼称 Weather Compan y将这些数据用于有针对性的营销和对冲基金分析。而且，Foursquare 在2016年通过其数据调查
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/foursquare-direct/foursquare-predicts-chipotle-s-q1-sales-down-nearly-30-foot-traffic-reveals-the-start-of-a-mixed-78515b2389af" href="https://medium.com/foursquare-direct/foursquare-predicts-chipotle-s-q1-sales-down-nearly-30-foot-traffic-reveals-the-start-of-a-mixed-78515b2389af" rel="noopener noreferrer" target="_blank">
    预测
   </a>
   ，在一场大肠杆菌危机之后，Chipotle 的销售额将在未来几个月内下降30％，因此备受关注。其销售额最终下降了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnbc.com/2016/04/26/chipotle-reports-first-quarter-results.html" href="https://www.cnbc.com/2016/04/26/chipotle-reports-first-quarter-results.html" rel="noopener noreferrer" target="_blank">
    29.7％
   </a>
  </p>
  <p class="graf graf--p">
   对位置数据的大部分关注都集中在电信巨头上，这些巨头已经将位置数据卖给第三方多年了。去年，Motherboard 发现，
   <strong class="markup--strong markup--p-strong">
    一旦数据被出售，就可以共享这些数据，以帮助赏金猎人实时找到特定目标人的手机。
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    见《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%b8%89%e5%a4%a7%e8%bf%90%e8%90%a5%e5%95%86%e4%b8%80%e7%9b%b4%e5%9c%a8%e7%a7%81%e4%b8%8b%e5%87%ba%e5%94%ae%e7%94%a8%e6%88%b7%e6%9c%80%e5%85%b3%e9%94%ae%e7%9a%84%e9%9a%90%e7%a7%81%e6%95%b0%e6%8d%ae/" href="https://www.iyouport.org/%e4%b8%89%e5%a4%a7%e8%bf%90%e8%90%a5%e5%95%86%e4%b8%80%e7%9b%b4%e5%9c%a8%e7%a7%81%e4%b8%8b%e5%87%ba%e5%94%ae%e7%94%a8%e6%88%b7%e6%9c%80%e5%85%b3%e9%94%ae%e7%9a%84%e9%9a%90%e7%a7%81%e6%95%b0%e6%8d%ae/" rel="noopener noreferrer" target="_blank">
     三大运营商一直在私下出售用户最关键的隐私数据
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   然而，
   <strong class="markup--strong markup--p-strong">
    没有法律禁止他们这样做。
   </strong>
  </p>
  <p class="graf graf--p">
   位置数据通过软件开发工具包SDKs从您的手机传输。该工具包是小型程序，可用于在应用程序中构建功能。它们使应用程序开发人员可以轻松地仅包括位置跟踪功能，这是诸如天气应用程序之类的服务的有用组成部分。
  </p>
  <p class="graf graf--p">
   例如，Facebook、Google 和 Amazon 具有非常受欢迎的 SDKs，它们允许较小的应用程序连接到较大公司的广告平台。
  </p>
  <p class="graf graf--p">
   但是他们也可以在应用程序上收集位置数据，同时不向应用程序提供任何实际服务。定位公司可能会支付要包含的应用程序的费用 —— 收集可以货币化的有价值的数据。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果您有SDK经常收集位置数据，很有可能会在整个行业内转售”。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="618" src="//player.vimeo.com/video/384360774?title=0&amp;byline=0&amp;portrait=0&amp;color=8dc7dc" width="750">
   </iframe>
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    市场的 “圣杯”
   </strong>
  </h3>
  <p class="graf graf--p">
   如果此信息如此敏感，为什么要首先收集它？
  </p>
  <p class="graf graf--p">
   对于品牌而言，窥探某人的精确行动是 “理解客户历程” 的关键 —— 从查看广告到购买产品的过程的每个步骤。一位营销人员说，这是广告的圣杯，它是将所有人所有兴趣和在线活动与实际操作联系在一起的完整图景。
  </p>
  <p class="graf graf--p">
   其他团体也开始寻找使用它的方法。
  </p>
  <p class="graf graf--p">
   政治运动可以分析集会参与者的兴趣和身份信息，并使用这些信息来塑造他们的详细简介，
   <strong class="markup--strong markup--p-strong">
    以试图操纵特定的群体
   </strong>
   —— 世界各国政府都可以使用某种新工具来识别抗议者。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     这里是须知
    </strong>
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%8c%87%e5%8d%97%ef%bc%9a%e5%bd%93%e6%82%a8%e5%87%86%e5%a4%87%e5%8f%82%e5%8a%a0%e6%8a%97%e8%ae%ae%e6%b4%bb%e5%8a%a8/" href="https://www.iyouport.org/%e6%8c%87%e5%8d%97%ef%bc%9a%e5%bd%93%e6%82%a8%e5%87%86%e5%a4%87%e5%8f%82%e5%8a%a0%e6%8a%97%e8%ae%ae%e6%b4%bb%e5%8a%a8/" rel="noopener noreferrer" target="_blank">
     指南：当您准备参加抗议活动…
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   优惠券应用程序是否需要将第二位置数据出售给其他公司才能获利？这是否真的有理由让公司追踪数百万人并可能揭露我们每个人的私生活？
  </p>
  <p class="graf graf--p">
   数据公司表示，用户 “同意” 在他们共享位置时进行跟踪。但是，这些同意书很少能清楚说明如何打包和出售数据。如果公司对数据处理方式更加透明，还会有人会同意共享数据吗？
  </p>
  <p class="graf graf--p">
   那么在黑客入侵和泄漏使隐私成为首要问题之前收集的数据又如何呢？应该仍然使用它，还是永久删除它？
  </p>
  <p class="graf graf--p">
   如果今天存储的数据很容易被黑客入侵、泄露或被盗，那么值得冒这种风险继续收集吗？
  </p>
  <p class="graf graf--p">
   不能指望从我们的一举一动中获利的公司自愿限制自己的做法。
  </p>
  <p class="graf graf--p">
   可以肯定的是：我们生活在世界上最先进的监视系统中。该系统的建立是为了赚钱。技术公司曾经玩过的最邪恶诡计就是说服社会主动接受监视。⚪️
  </p>
