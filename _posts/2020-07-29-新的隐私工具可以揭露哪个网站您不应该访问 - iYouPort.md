       帮助您预知风险的防御措施

    </span>

  <p>
   <img alt="" class="aligncenter size-full wp-image-13019 jetpack-lazy-image" data-lazy-sizes="(max-width: 640px) 100vw, 640px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/07/0.jpg?resize=640%2C400&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/07/0.jpg?w=640&amp;ssl=1 640w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/07/0.jpg?resize=300%2C188&amp;ssl=1 300w" data-recalc-dims="1" height="400" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/07/0.jpg?resize=640%2C400&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="640"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13019" data-recalc-dims="1" height="400" sizes="(max-width: 640px) 100vw, 640px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/07/0.jpg?resize=640%2C400&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/07/0.jpg?w=640&amp;ssl=1 640w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/07/0.jpg?resize=300%2C188&amp;ssl=1 300w" width="640"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   尽管许多人都愿意提供自己的某些数据（例如电子邮件地址），但是对于敏感信息来说绝非如此。考虑到这一点，人们会仔细选择与哪些公司共享自己的数据，考虑数据是否会以负责任的方式处理或不安全地放置在某些第三方服务器中？
  </p>
  <p class="graf graf--p">
   然而，尽管人们可能会采取直接的预防措施，但有时却会被某些网站所欺骗。它们看起来似乎合法，声称拥有严格的隐私政策，但最终还是与第三方共享了您的数据。
  </p>
  <p class="graf graf--p">
   为了帮助您避免在这种情况下的损失，云安全公司 Instart 发布了一个名为
   <a class="markup--anchor markup--p-anchor" data-href="https://chrome.google.com/webstore/detail/privacy-alert-by-instart/ngigdbbgdaflcocdebbingdlkknddcam" href="https://chrome.google.com/webstore/detail/privacy-alert-by-instart/ngigdbbgdaflcocdebbingdlkknddcam" rel="noopener noreferrer" target="_blank">
    Privacy Alert
   </a>
   的新 Chrome 扩展程序。
  </p>
  <p class="graf graf--p">
   该插件使您可以检查每个站点的 “数据安全等级”，这基本上表明与相关站点共享数据是否是个好主意。
  </p>
  <p class="graf graf--p">
   该公司在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.instart.com/blog/protect-private-data-with-privacy-alert" href="https://www.instart.com/blog/protect-private-data-with-privacy-alert" rel="noopener noreferrer" target="_blank">
    博客
   </a>
   中说，Privacy Alert 可让您知道黑客是否可能窃取了您在Web表单中键入的信息或保存在Cookie中的信息。例如，如果您访问一个金融网站（如您的网上银行应用程序）并输入了用户名和密码，则该插件将显示第三方是否也可能查看这些登录凭据。
  </p>
  <p class="graf graf--p">
   例如，以下报告的屏幕截图揭示了如何通过Cookie与其他9个地方共享了您的数据，其中一个是 Google Analytics。另一方面，它使您知道敏感数据仍然是安全的。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-13020 jetpack-lazy-image" data-lazy-sizes="(max-width: 502px) 100vw, 502px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1.jpg?resize=502%2C319&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1.jpg?w=502&amp;ssl=1 502w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1.jpg?resize=300%2C191&amp;ssl=1 300w" data-recalc-dims="1" height="319" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1.jpg?resize=502%2C319&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="502"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13020" data-recalc-dims="1" height="319" sizes="(max-width: 502px) 100vw, 502px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1.jpg?resize=502%2C319&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1.jpg?w=502&amp;ssl=1 502w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1.jpg?resize=300%2C191&amp;ssl=1 300w" width="502"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   根据上面的示例，此隐私工具将大大减少用户受到威胁的机会。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果黑客已经用恶意代码感染了该网站，用户无需再等到公司发布新闻稿才能获知，他们可以在访问网站的那一刻直接看到警报。
   </strong>
  </p>
  <p class="graf graf--p">
   例如，去年底的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hackread.com/hackers-use-web-skimmers-to-steal-credit-card-data/" href="https://www.hackread.com/hackers-use-web-skimmers-to-steal-credit-card-data/" rel="noopener noreferrer" target="_blank">
    案例
   </a>
   显示攻击者如何克隆了澳大利亚联邦银行的付款页面。如果用户在付款之前已经验证了网站的安全状态，那么这种情况下就不会被欺骗。
  </p>
  <p class="graf graf--p">
   因此，建议用户尝试使用这个新的隐私工具，这不是因为它是新的，而是因为其本质上可以作为一种预防性安全措施。
  </p>
  <p class="graf graf--p">
   但是，将所有内容都留给插件是不明智的，建议您同时采用其他安全措施，例如通过使用 Apple Pay 等服务来最小化输入付款详细信息的过程以及验证您访问的网站的安全证书。⚪️
  </p>
  <p class="graf graf--p">

    New privacy tool exposes which website leaves your data unprotected
   </a>
  </p>
