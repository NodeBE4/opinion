  <p class="graf graf--p">
   这不是第一次介绍隐写术。您可以在这里回顾（1）部分的内容：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%9a%90%e5%86%99%e6%9c%af%e7%9a%84%e5%b0%8f%e5%bf%83%e6%9c%ba/" href="https://www.iyouport.org/%e9%9a%90%e5%86%99%e6%9c%af%e7%9a%84%e5%b0%8f%e5%bf%83%e6%9c%ba/" rel="noopener noreferrer" target="_blank">
    隐写术的小心机（1）
   </a>
   》。
  </p>
  <p class="graf graf--p">
   隐写术是隐藏在数据中的数据。它使您能够将文件、消息或链接嵌入到过滤器无法检测到的图像中。
  </p>
  <p class="graf graf--p">
   因此，您可以
   <strong class="markup--strong markup--p-strong">
    发送隐藏在图像或视频中的秘密消息
   </strong>
   。
  </p>
  <p class="graf graf--p">
   隐写术一词结合了希腊语 steganos（στεγᾰνός）（意为“被遮盖或隐藏”）和graphe（γραφή）（意为“书写”）。
  </p>
  <p class="graf graf--p">
   那么隐写术很容易吗？其实并不容易，它实际上非常复杂。但这篇文章将尽可能继续为您提供方便。
  </p>
  <p class="graf graf--p">
   在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E9%9A%90%E5%86%99%E6%9C%AF%E7%9A%84%E5%B0%8F%E5%BF%83%E6%9C%BA/" href="https://www.iyouport.org/%E9%9A%90%E5%86%99%E6%9C%AF%E7%9A%84%E5%B0%8F%E5%BF%83%E6%9C%BA/" rel="noopener noreferrer" target="_blank">
    上一篇文章中
   </a>
   我们演示的是 Steghide，这篇文章中将使用另一个名为 StegoSuite的隐写小工具，您可以
   <a class="markup--anchor markup--p-anchor" data-href="https://stegosuite.org/" href="https://stegosuite.org/" rel="noopener noreferrer" target="_blank">
    在此处下载
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    适合几乎所有平台
   </strong>
   。
  </p>
  <p class="graf graf--p">
   是的，在所有平台上都可以使用，但本教程将使用 Windows 环境，该应用程序将被沙箱化。
  </p>
  <p class="graf graf--p">
   一旦您安装了该应用程序，系统将要求您上传要用于附加机密消息的照片。
  </p>

  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9920 jetpack-lazy-image" data-lazy-sizes="(max-width: 661px) 100vw, 661px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-11.png?resize=661%2C408&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-11.png?w=661&amp;ssl=1 661w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-11.png?resize=300%2C185&amp;ssl=1 300w" data-recalc-dims="1" height="408" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-11.png?resize=661%2C408&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="661"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9920" data-recalc-dims="1" height="408" sizes="(max-width: 661px) 100vw, 661px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-11.png?resize=661%2C408&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-11.png?w=661&amp;ssl=1 661w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-11.png?resize=300%2C185&amp;ssl=1 300w" width="661"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   你完成了！
  </p>
  <p class="graf graf--p">
   机密信息是通过图片秘密发送的。
  </p>
  <p class="graf graf--p">
   ⚠️对于任何不信任互联网传输安全性的人来说、以及任何生活在警察国家的人来说，这个方法都是很有用的。
  </p>
  <p class="graf graf--p">
   ⚠️但是必须说，这是个比较常用的方法；于是，如果您已经是斯诺登那个级别的敏感人士 —— 也就是说您是重点目标、且您的对手是NSA那个级别的高明的技术专家，仅仅隐写是不够的。
  </p>
  <p class="graf graf--p">
   因为你的对手会通过所有可用的方法盯住你的一举一动、拦截一切来自你的信息，“仅仅”发送一张照片的行为在监视者眼中会被认为“很可疑”。
  </p>
  <p class="graf graf--p">
   于是
   <strong class="markup--strong markup--p-strong">
    您应该想办法让它看起来不可疑
   </strong>
   ，让它特别自然。以打消监视者探寻的好奇心。
  </p>
  <p class="graf graf--p">
   比如您可以这样，在邮件中写上：“我看中了一款打印机/车载音响（任何东西），如果你喜欢的话我会把它买下来送给你，很抱歉弄坏了你的***东西……”，等等，然后随附相关物品的照片。
  </p>
  <p class="graf graf--p">
   撒一个小谎很简单。
  </p>
  <p class="graf graf--p">
   这样一来，监视者拦截到的消息会显示为一封道歉信，也许就不会深入挖掘那张照片中有什么了。
  </p>
  <p class="graf graf--p">
   请注意，不要以附件的形式发送，因为太直观可以看到图像“好重”……超过一般图像的大小是很可疑的现象。
  </p>
  <p class="graf graf--p">
   没错，上传到社交媒体是个好主意，请附加一段再平凡不过的文字，比如感谢女朋友做了早餐等等…… 您一定知道该怎么做。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果您想使其更加安全一些，可以使用只有您和您的朋友知道的密码选项。这样一来即便监视者打开了机密消息，他们也很难拿到内容
   </strong>
   。
  </p>
  <p class="graf graf--p">
   就是这么简单！玩得开心。 ⚪️
  </p>
