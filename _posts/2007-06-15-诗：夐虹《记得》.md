   <h1>
    诗：夐虹《记得》
   </h1>
   <ul class="post-entry-meta">
    <li>
     By:
     <a href="http://www.ideobook.com/about-ideobook/" title="查看 爱德布克 的作者主页">
      爱德布克
     </a>
     . 2007-6-15.
     <a href="http://www.ideobook.com/372/post-views-count/" title="统计说明">
      11,779
     </a>
    </li>
   </ul>
  </header>
  <div class="post-entry-content">
   <p style="text-align: center;">
    你如果
    <br/>
    如果你对我说过
    <br/>
    一句一句
    <br/>
    真纯的话
    <br/>
    我早晨醒来
    <br/>
    我便记得它
   </p>
   <p style="text-align: center;">
    年少的岁月
    <br/>
    简单的事
    <br/>
    如果你说了
    <br/>
    一句一句
    <br/>
    浅浅深深
    <br/>
    云飞雪落的话
   </p>
   <p style="text-align: center;">
    *
   </p>
   <p style="text-align: center;">
    关切是问
    <br/>
    而有时
    <br/>
    关切
    <br/>
    是
    <br/>
    不问
   </p>
   <p style="text-align: center;">
    倘或一无消息
    <br/>
    如沉船后静静的
    <br/>
    海面，其实也是
    <br/>
    静静的记得
   </p>
   <p style="text-align: center;">
    *
   </p>
   <p style="text-align: center;">
    倘或在夏季之末
    <br/>
    秋季之初
    <br/>
    写过一两次
    <br/>
    隐晦的字
    <br/>
    影射那偶然
    <br/>
    像是偶然的
    <br/>
    落雨
    <br/>
    ——也是记得
   </p>

          <p>
           夐
           <br/>
           xiòng
          </p>
          <p>
           假借为“远”。辽远,距离遥远的 [distant]
           <br/>
           于嗟夐兮。——《韩诗·击鼓》
           <br/>
           夐入千乘之国。——《谷梁传·文公十四年》
           <br/>
           与为夐明。——《吕氏春秋·尺数》
           <br/>
           夐若千里,人罕造其门。——《旧五代史》
           <br/>
           又如:夐远辽远;夐隔(远隔);夐寥(辽远);夐然(辽远);夐阻(远隔)
           <br/>
           久远,长久 [remote]
           <br/>
           上哉夐昂?其书犹得而修也。——汉·班固《典引》
           <br/>
           又如:夐古(远古);夐然(久远)
           <br/>
           [影响、意义等] 深刻而长远的 [far-reaching]
           <br/>
           昔《南风》之辞,《卿云》之颂,厥义夐矣。——《梁书》
           <br/>
           又如:夐明(大智大彗);夐迁(远大)
           <br/>
           高超 [high]
           <br/>
           杜陵翁独为诗人冠冕者,吐辞不凡,夐出尘表。——宋·孙奕《履斋示儿编·诗说》
           <br/>
           高尚 [noble]
           <br/>
           介君祠庙在,风义夐难攀。——清·顾炎武《介休》
          </p>
