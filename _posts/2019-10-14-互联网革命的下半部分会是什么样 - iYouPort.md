  <p class="graf graf--p">
   2007年，史上第一次住在城市里的人比住在乡村的人更多。
  </p>
  <p class="graf graf--p">
   互联网的发展已经快速抵达中点。
  </p>
  <p class="graf graf--p">
   这个星球上有超过50％的人口现已上网，自从互联网在西方技术娴熟的人群中崛起后的仅仅四分之一个世纪，实现了普通人都可以使用。
  </p>
  <p class="graf graf--p">
   互联网革命的后半部分已经开始。它正在改变社会的运作方式 —— 并且也在创造
   <strong class="markup--strong markup--p-strong">
    一个新的商业难题
   </strong>
   。
  </p>
  <p class="graf graf--p">
   大多数新用户都在新兴市场；仅在过去三年内就有约7.26亿人上网。
  </p>
  <p class="graf graf--p">
   中国仍在快速增长。但对于整个互联网来说大部分的增长都来自较贫穷的地方，尤其是印度和非洲。
  </p>
  <p class="graf graf--p">
   富裕国家的公共语话不仅有为假新闻和 trolling 的焦虑，许多观察家也开始担心政治贬值，从印度选民的两极分化到缅甸罗兴亚少数民族遭受的迫害。这些都是互联网带来的。
  </p>
  <p class="graf graf--p">
   从积极的方面来看，慈善机构和援助工作者一直在无休止地谈论智能手机如何让农民更方便检查作物价格、让村民报名参加在线教育、并帮助医生提高疫苗接种率。
  </p>
  <p class="graf graf--p">
   不太讨喜的一点是，互联网革命的下半部分和上半部分的主要吸引力相同。
  </p>
  <p class="graf graf--p">
   社交和游戏，而不是工作和自我提升，才是吸引力；色情很受欢迎。
  </p>
  <p class="graf graf--p">
   消息应用程序可以帮助朋友间保持联系，让移民工人可以方便地对留在家里的孩子说晚安。
  </p>
  <p class="graf graf--p">
   人们在社交媒体上消遣他们的朋友和陌生人，在 YouTube 或 TikTok 上使用古怪的自制视频。
  </p>
  <p class="graf graf--p">
   便宜的数据计划和拇指驱动器将盗版电影带给数百万可能从未去过电影院的人。
  </p>
  <p class="graf graf--p">
   约会应用程序比农业建议更受欢迎；电子游戏比任何一种东西都更受欢迎。
  </p>
  <p class="graf graf--p">
   这些福利不太可能进入许多联合国发展报告。但它们可以促进人类幸福的存量。
  </p>
  <p class="graf graf--p">
   对于企业而言，互联网革命的下半部分提供了大量的客户。
  </p>
  <p class="graf graf--p">
   但是，这也让人头疼 ——
   <strong class="markup--strong markup--p-strong">
    这些新用户中的大多数都太穷了，不可能花很多钱。
   </strong>
  </p>
  <p class="graf graf--p">
   数十亿美元的风险投资资金已流入新兴市场（不包括中国）的互联网创业公司。
  </p>
  <p class="graf graf--p">
   硅谷巨头建立了庞大的用户群 ——
   <strong class="markup--strong markup--p-strong">
    超过15亿 Facebook 用户在发展中国家
   </strong>
   。
  </p>
  <p class="graf graf--p">
   谷歌拥有的视频网站 YouTube 越来越多地受到
   <strong class="markup--strong markup--p-strong">
    非西方用户
   </strong>
   的支配。
  </p>
  <p class="graf graf--p">
   沃尔玛去年斥资160亿美元收购了印度电子商务巨头 Flipkart。
  </p>
  <p class="graf graf--p">
   Jumia 是一家电子商务公司，在尼日利亚和其他13个非洲国家拥有400万客户，于4月份在纽约上市。
  </p>
  <p class="graf graf--p">
   尽管这些公司的估值很高，但他们仍在寻找可持续的商业模式。
  </p>
  <p class="graf graf--p">
   Reliance Jio 是一家印度公司，已投入370亿美元用于建设高速移动网络并收获了大部分贫困用户。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    亚洲的每个 Facebook 用户每年仅产生11美元的广告收入，而北美的用户则产生112美元
   </strong>
   。
  </p>
  <p class="graf graf--p">
   新兴市场（不包括中国）的所有互联网公司的总收入可能是每年1000亿美元。这个数字与康卡斯特差不多，康卡斯特是美国排名第31大的上市公司。
  </p>
  <p class="graf graf--p">
   尽管如此，这些公司对业务的影响将在两个方面变得更大。
  </p>
  <p class="graf graf--p">
   首先，它们将快速增长 — 尽管是否足够快以证明其估值合理仍有待观察。
  </p>
  <p class="graf graf--p">
   为了最大限度地提高自己的机会，许多人不仅提供单一服务（如搜索或视频），并且还在一个应用程序中提供一系列服务，以期
   <strong class="markup--strong markup--p-strong">
    从每个用户身上赚更多钱。
   </strong>
  </p>
  <p class="graf graf--p">
   这种方法是阿里巴巴和腾讯在中国率先推出的。印度尼西亚的 Go-Jek 紧跟其后，提供乘车、付款、药物处方和按摩服务的综合。
  </p>
  <p class="graf graf--p">
   Facebook 正在通过其聊天服务 WhatsApp 推动印度的数字支付系统。（不仅如此，Facebook 正在尝试消费微信，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/facebook%e3%80%81%e8%80%8c%e4%b8%8d%e6%98%af%e5%be%ae%e8%bd%af%ef%bc%8c%e6%98%af%e5%bc%80%e6%ba%90%e7%9a%84%e4%b8%bb%e8%a6%81%e5%a8%81%e8%83%81/" href="https://www.iyouport.org/facebook%e3%80%81%e8%80%8c%e4%b8%8d%e6%98%af%e5%be%ae%e8%bd%af%ef%bc%8c%e6%98%af%e5%bc%80%e6%ba%90%e7%9a%84%e4%b8%bb%e8%a6%81%e5%a8%81%e8%83%81/" rel="noopener noreferrer" target="_blank">
    见这里
   </a>
   ）
  </p>
  <p class="graf graf--p">
   其次，在新兴市场中成熟企业可能比富裕国家的现有企业更快地被打乱。他们拥有较少的基础设施，如仓库和零售网点。
  </p>
  <p class="graf graf--p">
   许多人，尤其是大城市以外的人，完全无法获得他们的服务。
  </p>
  <p class="graf graf--p">
   啤酒、洗发水和其他消费品公司很快会发现，随着营销变得数字化，新的反叛品牌会更快地获得牵引力。
  </p>
  <p class="graf graf--p">
   银行要被迫迅速适应数字支付否则会死亡。
  </p>
  <p class="graf graf--p">
   从这个角度来看，巨额资金濒临危险 — 中国境外新兴市场现有公司的总市值为8万亿美元。
  </p>
  <p class="graf graf--p">
   如果你认为互联网革命的前半部分是破坏性的，那就等到看第二幕吧。⚪️
  </p>
