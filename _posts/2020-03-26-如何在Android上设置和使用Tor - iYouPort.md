  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如何在Android上设置和使用Tor？
   </strong>
  </p>
  <p class="graf graf--p">
   在进入Tor所需的应用之前，必须首先确保您的 Android 设备已root。
  </p>
  <p class="graf graf--p">
   如果您熟悉 Windows 资源管理器，请将root视为设备的 C:\。
  </p>
  <p class="graf graf--p">
   Android 默认不允许您访问此文件，以防止用户意外删除重要文件。但是要通过设备访问Tor，您需要完成这一步。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、root 你的设备
   </strong>
  </p>
  <p class="graf graf--p">
   如何root设备将由您拥有的设备型号决定。由于存在很多种设备型号，因此没有一种通用的方法可以将它们全部root。
  </p>
  <p class="graf graf--p">
   最好的选择是对手机的型号以及如何root手机进行一些在线研究。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、越狱
   </strong>
  </p>
  <p class="graf graf--p">
   越狱是一个奇怪的词，听起来就像在做违法的事。其实不是这样。您要做的只是使您的设备能够从 Google Play 商店中找不到的来源安装应用。
  </p>
  <p class="graf graf--p">
   该过程很简单，但是取决于您的Android设备正在运行的操作系统版本。
  </p>
  <p class="graf graf--p">
   对于 Android Nougat 及更早版本：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    进入手机设置，点击以打开 Security
   </li>
   <li class="graf graf--li">
    切换到 “Unknown Sources” 选项，然后单击“确定”验证更改
   </li>
  </ul>
  <p class="graf graf--p">
   对于 Android Oreo 及更高版本：
  </p>
  <p class="graf graf--p">
   任何未通过 Google Play 商店下载的内容都会提示您确认。点击启用来自未知来源的安装就可以了。
  </p>

    <img alt="" class="aligncenter size-full wp-image-10502" data-recalc-dims="1" height="634" sizes="(max-width: 1084px) 100vw, 1084px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-12.png?resize=1084%2C634&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-12.png?w=1084&amp;ssl=1 1084w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-12.png?resize=300%2C175&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-12.png?resize=768%2C449&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-12.png?resize=1024%2C599&amp;ssl=1 1024w" width="1084"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   每当传输数据时，
   <a class="markup--anchor markup--p-anchor" data-href="https://f-droid.org/repository/browse/?fdid=org.ethack.orwall" href="https://f-droid.org/repository/browse/?fdid=org.ethack.orwall" rel="noopener noreferrer" target="_blank">
    OrWall
   </a>
   都会强制您的Android设备上所有应用都使用Tor网络。你知道，这样会更加安全。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    点击链接以下载APK文件并安装。
   </li>
   <li class="graf graf--li">
    安装完成后，运行安装程序，然后重新启动设备。
   </li>
   <li class="graf graf--li">
    备份设备后，您可以启动 OrWall 并手动确定哪些应用程序将通过Tor传输数据。
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    The OrBot App
   </strong>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10503 jetpack-lazy-image" data-lazy-sizes="(max-width: 1082px) 100vw, 1082px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-8.png?resize=1082%2C638&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-8.png?w=1082&amp;ssl=1 1082w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-8.png?resize=300%2C177&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-8.png?resize=768%2C453&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-8.png?resize=1024%2C604&amp;ssl=1 1024w" data-recalc-dims="1" height="638" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-8.png?resize=1082%2C638&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1082"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10503" data-recalc-dims="1" height="638" sizes="(max-width: 1082px) 100vw, 1082px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-8.png?resize=1082%2C638&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-8.png?w=1082&amp;ssl=1 1082w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-8.png?resize=300%2C177&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-8.png?resize=768%2C453&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-8.png?resize=1024%2C604&amp;ssl=1 1024w" width="1082"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://guardianproject.info/releases/Orbot-v14.1.0-PIE.apk" href="https://guardianproject.info/releases/Orbot-v14.1.0-PIE.apk" rel="noopener noreferrer" target="_blank">
    OrBot
   </a>
   是从您的设备到Tor网络的桥梁。该应用程序也可以在 Google Play 商店中找到，但提供的链接直接来自
   <a class="markup--anchor markup--p-anchor" data-href="https://guardianproject.info/" href="https://guardianproject.info/" rel="noopener noreferrer" target="_blank">
    开发者
   </a>
   的网站。
  </p>
  <p class="graf graf--p">
   下载并安装该应用程序后，请完成设置向导。仅在首次设置 OrBot 时才需要这样做。可以通过应用程序中的 “Power” 按钮完成所有后续激活。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    The OrFox App
   </strong>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://apkpure.com/orfox/info.guardianproject.orfox/download?from=details" href="https://apkpure.com/orfox/info.guardianproject.orfox/download?from=details" rel="noopener noreferrer" target="_blank">
    OrFox
   </a>
   是为 Orbot 创建的伙伴浏览器，也是由 Guardian Project 的杰出人士开发的。OrFox 是您在信息高速公路上行驶时要使用的车子。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10504 jetpack-lazy-image" data-lazy-sizes="(max-width: 1084px) 100vw, 1084px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-7.png?resize=1084%2C638&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-7.png?w=1084&amp;ssl=1 1084w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-7.png?resize=300%2C177&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-7.png?resize=768%2C452&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-7.png?resize=1024%2C603&amp;ssl=1 1024w" data-recalc-dims="1" height="638" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-7.png?resize=1084%2C638&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1084"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10504" data-recalc-dims="1" height="638" sizes="(max-width: 1084px) 100vw, 1084px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-7.png?resize=1084%2C638&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-7.png?w=1084&amp;ssl=1 1084w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-7.png?resize=300%2C177&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-7.png?resize=768%2C452&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-7.png?resize=1024%2C603&amp;ssl=1 1024w" width="1084"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   除了依赖 OrBot 之外，该应用没有其他必要的设置说明了。您只需启动它，然后等待 “success” 提示，让您知道已连接成功。
  </p>
  <p class="graf graf--p">
   OrFox 确实提供可以绕过大多数在线限制的好处。比如您的政府封锁了一些网站，或者您目前正在办公室和学校，那里禁止您访问某些网站，等等情况，您肯定知道什么时候需要它。
  </p>
  <p class="graf graf--p">
   OrWeb 是用于Android的Tor浏览器的基本版本，但随着 OrFox 的发布它就过时了。
  </p>
  <p class="graf graf--p">
   OrWeb 是OrBot下载和安装的默认浏览器，但它存在许多限制。请确保获得与OrBot配合使用的 OrFox，以免受到Tor浏览功能的限制。⚪️
  </p>
