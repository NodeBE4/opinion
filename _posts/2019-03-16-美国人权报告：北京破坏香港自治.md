title: 美国人权报告：北京破坏香港自治
---

<div id="entry">
<div class="at-above-post addthis_tool" data-url="http://zhanlve.org/?p=6911">
</div>
<p>
<img class="aligncenter" src="http://zhanlve.org/wp-content/uploads/2019/03/217DDE31-BF69-4933-9586-7D5D19852D5F_cx0_cy9_cw0_w1597_n_r1_st-300x169.jpg"/>
</p>
<p>
</p>
<p>
<span style="font-size: 12pt;">
   美国国务院发表的年度人权报告，详述2018年各个国家的人权状况，当中有关香港的篇章，也详述了香港过去一年损害人权的多件大事，批评北京政府损害香港自治及法治。而英国也表示香港的高度自治持续受压，考虑与香港再签订贸易协议时，加入人权条款。香港各界担心，香港的人权状况可能对经济带来实际影响。
  </span>
</p>
<p>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   报告以七大章节详述香港过去一年的人权狀况，包括在“尊重个人”丶“尊重公民权利”，以及“参政自由”等方面，并引用过去一年在香港发生的多个例子。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   在“尊重个人”方面，报告引用大陆富商肖建华和铜锣湾书店瑞典籍股东桂民海在香港“被失踪”的例子，指事件显示中国政府正在违反香港的法治和自治权。报告指多个消息来源，怀疑有中国政府特工在香港监控政界活跃人士丶非政府组织以及批评中国政府的学者。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   而在“尊重公民权利”方面，报告提到在中国政府不断警告港独主张“超越红线”下，港府决定取缔支持香港独立的香港民族党。而去年中国外交部驻港特派员公署曾向香港外国记者会施压，要求取消民族党召集人陈浩天的演讲，其後主持演讲的英国《金融时报》亚洲新闻主编马凯，更被港府禁止以旅客身份入境，港府更发稿公开谴责香港大学法律系副教授戴耀廷有关港独的言论。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   报告批评过去一年，香港特区政府和中国政府采取种种行动，限制人们表达政见的权利和学术自由，特别是有关支持香港独立的声音。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   报告也陈述了香港众志周庭被取消参选资格丶社民连梁国雄等人被夺去议员资格等个案，指现时的《基本法》限制了港人透过公平选举改变政府的权力。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   报告對香港人權狀況措辞严厉史无前例
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   香港时事评论员桑普认为，是次报告详细罗列了香港过去一年在人权上的负面个案，内容具体，而且措辞严厉，是史无前例的。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   桑普说：“我认为是亮起黄灯了，我不认为是亮起红灯，说有什麽进一步的改变， 但如果未来一两年香港的人权状况继续恶化下去，比方说要订立23条立法，或者有更多绑架的事情发生，我觉得美国真的有可能把香港独立关税区取消掉。”
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   香港人权情况或对经济带来实际影响
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   其实早前美国驻港总领事唐伟康已经公开警告，香港的政治和一国两制运行情况恶化，或令美国重新审视香港的独立关税区地位。而无独有偶，继美国之後，英国国会人权委员会周二(12日)发表的报告，也罕有地指出一国两制在香港虽然大致运作良好，但《中英联合声明》所保障的高度自治丶权利丶自由却持续受压，英国外交部更表明正计划利用脱欧时机，将来与香港再签订贸易和投资协议时，会加入人权条款。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   香港民主派立法会议员杨岳桥认为，除了美国和英国，很多国家在香港都有深厚的利益，如果这些国家都不信赖香港的一国两制，或会令香港进一步失去经贸优势。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   杨岳桥说：“我们的主要竞争对手新加坡已经差不多跟全世界有FTA(自由贸易协议)，香港现在是大落後，我们的(人权情况)形象给国外可能就是香港没有这个能力，或者人家没有兴趣跟我们达成自由贸易协订，那对香港的声誉也没有好处。”
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   对於美国国务院的人权报告，港府回应指，外国政府不应以任何形式干预香港的内部事务，又强调自归以来，特区一直按《基本法》规定实行港人治港丶高度自治，一国两制成功落实。而中国外交部则指，美国的报告充满意识形态偏见，对中国进行无端指责，表示坚决反对，又以《2018年美国的人权纪录》还击，以过万字记录美国“侵犯人权事项”，包括发生超过五万宗涉及枪械案件，又有163人在校园枪击案中伤亡。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
