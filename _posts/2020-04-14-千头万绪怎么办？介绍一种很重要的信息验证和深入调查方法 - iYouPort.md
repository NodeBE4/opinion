  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     什么是差距分析？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   差距分析实际上只是一种简单的方法，可以对您所拥有的信息进行排序，确定您需要对哪些东西进行处理，然后提出结构化的回应以推动调查。
  </p>
  <p class="graf graf--p">
   它是一个好仆人，但可能是一个糟糕的主人，所以，不要让它阻止你的脑洞。
  </p>
  <p class="graf graf--p">
   ⚠️绝对可以保证在任何类型的调查中都有可能出现的错误是：在一开始时做出了不必要的假设，然后在这些假设的基础上进行调查，然后就走进邪路了。用这种方式构建的案例最终将彻底崩溃。
  </p>
  <p class="graf graf--p">
   你需要避免这点。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    差距分析提出了四个简单的问题
   </strong>
   ：
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    我知道什么
   </li>
   <li class="graf graf--li">
    这意味着什么
   </li>
   <li class="graf graf--li">
    我还应该知道什么
   </li>
   <li class="graf graf--li">
    我如何找到它们
   </li>
  </ol>
  <p class="graf graf--p">
   就是这么简单。
  </p>
  <p class="graf graf--p">
   在第一个问题上花费时间是最重要的。请注意，
   <strong class="markup--strong markup--p-strong">
    只有第四个问题会导致询问“我使用什么工具？”
   </strong>
  </p>
  <p class="graf graf--p">
   💡在选择所需工具之前，必须先计划和选择一种方法 —— 这就是为什么说思考方式比工具更重要。
  </p>

   <img alt="" class="aligncenter size-full wp-image-10779 jetpack-lazy-image" data-lazy-sizes="(max-width: 927px) 100vw, 927px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/15-1.png?resize=927%2C348&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/15-1.png?w=927&amp;ssl=1 927w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/15-1.png?resize=300%2C113&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/15-1.png?resize=768%2C288&amp;ssl=1 768w" data-recalc-dims="1" height="348" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/15-1.png?resize=927%2C348&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="927"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10779" data-recalc-dims="1" height="348" sizes="(max-width: 927px) 100vw, 927px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/15-1.png?resize=927%2C348&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/15-1.png?w=927&amp;ssl=1 927w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/15-1.png?resize=300%2C113&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/15-1.png?resize=768%2C288&amp;ssl=1 768w" width="927"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   作为比较，以下是星期日的：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10780 jetpack-lazy-image" data-lazy-sizes="(max-width: 890px) 100vw, 890px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/16-2.png?resize=890%2C228&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/16-2.png?w=890&amp;ssl=1 890w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/16-2.png?resize=300%2C77&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/16-2.png?resize=768%2C197&amp;ssl=1 768w" data-recalc-dims="1" height="228" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/16-2.png?resize=890%2C228&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="890"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10780" data-recalc-dims="1" height="228" sizes="(max-width: 890px) 100vw, 890px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/16-2.png?resize=890%2C228&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/16-2.png?w=890&amp;ssl=1 890w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/16-2.png?resize=300%2C77&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/16-2.png?resize=768%2C197&amp;ssl=1 768w" width="890"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   因此你看到，周六多云，而周日则比较晴朗，湿度较低。所以10月5日星期六应该是正确的日期。
  </p>
  <p class="graf graf--p">
   MCantow 则通过使用历史性的
   <a class="markup--anchor markup--p-anchor" data-href="https://kachelmannwetter.com/de/gewitter/stadt-berlin/20191006-1530z.html" href="https://kachelmannwetter.com/de/gewitter/stadt-berlin/20191006-1530z.html" rel="noopener noreferrer" target="_blank">
    卫星图像
   </a>
   来
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/MCantow/status/1181660040584937473" href="https://twitter.com/MCantow/status/1181660040584937473" rel="noopener noreferrer" target="_blank">
    检查云量
   </a>
   ，从而提出了针对这一部分挑战的
   <strong class="markup--strong markup--p-strong">
    更简洁的解决方案
   </strong>
   。这是相关时间柏林的星期六：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10781 jetpack-lazy-image" data-lazy-sizes="(max-width: 754px) 100vw, 754px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/17-2.png?resize=754%2C614&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/17-2.png?w=754&amp;ssl=1 754w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/17-2.png?resize=300%2C244&amp;ssl=1 300w" data-recalc-dims="1" height="614" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/17-2.png?resize=754%2C614&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="754"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10781" data-recalc-dims="1" height="614" sizes="(max-width: 754px) 100vw, 754px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/17-2.png?resize=754%2C614&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/17-2.png?w=754&amp;ssl=1 754w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/17-2.png?resize=300%2C244&amp;ssl=1 300w" width="754"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   柏林以西有云，从这张照片中可以看出。将此与周日进行比较，那时几乎没有阴云：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10782 jetpack-lazy-image" data-lazy-sizes="(max-width: 758px) 100vw, 758px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/18-2.png?resize=758%2C569&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/18-2.png?w=758&amp;ssl=1 758w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/18-2.png?resize=300%2C225&amp;ssl=1 300w" data-recalc-dims="1" height="569" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/18-2.png?resize=758%2C569&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="758"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10782" data-recalc-dims="1" height="569" sizes="(max-width: 758px) 100vw, 758px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/18-2.png?resize=758%2C569&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/18-2.png?w=758&amp;ssl=1 758w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/18-2.png?resize=300%2C225&amp;ssl=1 300w" width="758"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   因此，可以确信星期六是最有可能的日期。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     最后
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   差距分析确实有效。当然这不是魔术，但是它确实可以帮助您以
   <strong class="markup--strong markup--p-strong">
    更加系统的方式
   </strong>
   评估您所拥有的信息。
  </p>
  <p class="graf graf--p">
   它还可以帮助您采用一种调查方法，
   <strong class="markup--strong markup--p-strong">
    将重点放在重要事项上
   </strong>
   ，而不必花费时间钻研无济于事的嘈杂问题。
  </p>
  <p class="graf graf--p">
   就是这样，您学会了吗？
  </p>
