   美国老大哥锦标赛上的中国”状元”
  </h1>

 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      一家中国监视技术公司获得了美国政府批准的老大哥锦标赛状元，这是哪家公司？
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   去年，中国警方在一个流行音乐会上逮捕了一名男子，因为警方在会场安装了一个面部识别系统，此人被算法标记为犯罪嫌疑人。该监视软件是由上海创业公司 Yitu Tech 开发的。但是请注意，该公司以美国政府批准的测试结果为荣。
  </p>
  <p class="graf graf--p">
   中国的 Yitu 是美国国家标准与技术研究院运行的测试项目的“最佳表现者”，该项目被认为对于快速发展的面部识别行业“至关重要”。超过60家公司参加了最新一轮的测试。这项排名由来自俄罗斯和中国的参赛者主导，这些政府都看好面部识别的监视能力并且相对不关心隐私。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “它被认为是行业标准，用户依赖NIST的商业基准进行决策，
   <strong class="markup--strong markup--p-strong">
    中国和国际客户都在询问这项技术
   </strong>
   “，Yitu 硅谷前哨的负责人 Shuang Wu 说。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Yitu 的技术被警察、地铁站、和 ATM 广泛使用
   </strong>
   。它目前在 NIST 的两个主要测试中排名第一，这项测试是挑战算法能力检测两张照片何时显示相同的脸部。该任务是检查护照或控制对建筑物和计算机系统的访问权限的核心。
  </p>
  <p class="graf graf--p">
   排名中接下来五个表现最好的公司都是俄罗斯人或中国人的公司。当美国国务院去年6月选择总部位于巴黎的 Idemia 提供的用于筛选护照申请的软件时
   <a class="markup--anchor markup--p-anchor" data-href="https://www.fbo.gov/index?s=opportunity&amp;mode=form&amp;id=175e4f707e6ae5ea8bd826b4b5129554&amp;tab=core&amp;_cview=0" href="https://www.fbo.gov/index?s=opportunity&amp;mode=form&amp;id=175e4f707e6ae5ea8bd826b4b5129554&amp;tab=core&amp;_cview=0" rel="noopener noreferrer" target="_blank">
    表示
   </a>
   ，已经选择了“最准确的非俄罗斯或中国软件”来管理其存档的3.6亿张面孔。
  </p>
  <p class="graf graf--p">
   在随后的一轮测试中，美国创业公司 Ever AI 排名第七，成为俄罗斯和中国之外表现最强监视能力的公司。Ever AI 首席执行官 Doug Aley 表示，“自从 NIST 业绩出现以来，客户流量一直非常稳定”，
   <strong class="markup--strong markup--p-strong">
    其中包括来自政府机构的新兴趣
   </strong>
   。
  </p>
  <p class="graf graf--p">
   NIST 是美国商务部的一个部门，其使命是通过推进计量科学来提升美国的竞争力。其面部识别供应商测试计划始于2000年，
   <strong class="markup--strong markup--p-strong">
    是在五角大楼的支持下
   </strong>
   ，在众多美国机构对使用该技术产生兴趣之后。
  </p>
  <p class="graf graf--p">
   从那以后，NIST 一直在追踪旨在仔细检查人类面貌的算法的稳步改进，并开发了新的测试方案以跟上发展。该机构现在在马里兰州盖瑟斯堡的一个地下计算机房中测试算法，使用数百万个
   <strong class="markup--strong markup--p-strong">
    来自政府机构的
   </strong>
   照片数据。
  </p>
  <p class="graf graf--p">
   另一项 NIST 测试模拟了警方调查员使用面部识别的方式，要求算法在人山人海中搜索特定的面部。在2010年，最好的软件可以在大约92％的时间内从160万个面部照片中准确找到某个特定的人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该测试中表现最好的老大哥是微软
   </strong>
   ，这是11月份 NIST 第一次测试结果。接下来的三位“最佳”都是俄罗斯人和中国人，其中 Yitu 是第四位，Ever AI 排在第五位。在美国国家标准与技术研究院最新报告中列出的60多个参与者中，有13个来自美国，12个来自中国，7个来自俄罗斯。
  </p>
  <p class="graf graf--p">
   对于俄罗斯和中国以外的公司而言，
   <strong class="markup--strong markup--p-strong">
    只要在 NIST 的排名上表现得很好，就能打开与美国政府签订合同的大门
   </strong>
   。NEC 联邦运营副总裁 Benji Hutchinson 说：“联邦机构不会在没有与NIST核实的情况下做出购买决定。” 该公司与政府、国土安全部和国防部门签订了面部识别合同，其技术正在接受测试，以检查美国几个机场的国际旅客身份。
  </p>
  <p class="graf graf--p">
   微软总裁 Brad Smith 在去年12月的一篇博客文章中吹捧了该公司在新的 NIST 结果中的成绩，该文章呼吁对该技术进行联邦法规管理，并强调了独立测试的重要性。但是，该公司拒绝回答关于其进入该计划的决定以及对政府面部识别合同的兴趣，
   <strong class="markup--strong markup--p-strong">
    在最近的证词中微软一直在为政府使用该技术进行辩护，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wired.com/story/microsoft-wants-rules-facial-recognition-just-not-these/" href="https://www.wired.com/story/microsoft-wants-rules-facial-recognition-just-not-these/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     反对
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    华盛顿州法案限制面部识别
   </strong>
   。
  </p>
  <p class="graf graf--p">
   IBM和亚马逊都向当地的执法机构出售面部识别技术，但都没有将其技术提交给NIST的测试。
   <a class="markup--anchor markup--p-anchor" data-href="https://aws.amazon.com/blogs/machine-learning/thoughts-on-recent-research-paper-and-associated-article-on-amazon-rekognition/" href="https://aws.amazon.com/blogs/machine-learning/thoughts-on-recent-research-paper-and-associated-article-on-amazon-rekognition/" rel="noopener noreferrer" target="_blank">
    亚马逊在1月份表示
   </a>
   ，它尊重NIST的测试，但其技术与亚马逊的云计算平台深度整合，无法发送到盖瑟斯堡。
  </p>
  <p class="graf graf--p">
   IBM 计算机视觉研究经理 John Smith 说，该公司正在与 NIST 合作，在决定是否参与之前，扩大其面部识别在不同人群中的表现。
  </p>
  <p class="graf graf--p">
   实验表明，亚马逊的技术在黑人的脸部识别上犯了更多错误，而且 IBM 和微软的面部分析软件对于皮肤较暗的女性来说不那么准确，科技公司和他们的批评者已经越来越担心面部识别中的人口偏差。亚马逊对调查结果提出异议，微软和 IBM 表示他们已经升级了系统。
  </p>
  <p class="graf graf--p">
   华盛顿大学的研究员 Os Keyes 说，这些研究结果表明，面部识别必须进行比通过实验室准确性测试更广泛的审查。
  </p>
  <p class="graf graf--p">
   Keyes 去年发表了一篇论文，批评 NIST 和其他人开发的性别识别软件，这种软件不考虑跨性别者，可能会给已经被边缘化的群体带来问题。2015年 NIST 关于性别识别软件测试的报告表明，该技术可用于女性浴室或更衣室的报警系统，以便在男性进入时发出警报。“
   <strong class="markup--strong markup--p-strong">
    NIST 需要雇用伦理学家或社会学家来评定技术，他们应该走出实验室看看这些技术的影响
   </strong>
   ，”Keyes 说。
  </p>
  <p class="graf graf--p">
   负责测试的 NIST 科学家之一 Patrick Grother 表示，他的团队正在扩大对面部识别技术人口统计差异的测试，并以自己的方式帮助解决技术中的潜在缺陷。
  </p>
  <p class="graf graf--p">
   尽管对种族和性别偏见的讨论有所增加，但需要做更多的工作来确定如何测试和测量它，Grother 说，并补充说 NIST 可以通过推进检测和跟踪它们的科学来帮助行业解决任何问题。 “我们试图将阳光和氧气带到市场。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    特朗普似乎希望 NIST 在维持人工智能发展方面发挥更积极的作用
   </strong>
   。他上个月签署的一项行政命令旨在鼓励人工智能在美国的发展，并指示该机构制定标准和工具，以鼓励“可靠，强大，值得信赖”的人工智能系统。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    W
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wired.com/story/china-earns-high-marks-us-test-facial-recognition/" href="https://www.wired.com/story/china-earns-high-marks-us-test-facial-recognition/" rel="noopener noreferrer" target="_blank">
    HY CHINESE COMPANIES PLUG A US TEST FOR FACIAL RECOGNITION
   </a>
   ? The rankings are dominated by entrants from Russia and China, where governments are bullish about facial recognition and relatively unconcerned about privacy.
  </p>
