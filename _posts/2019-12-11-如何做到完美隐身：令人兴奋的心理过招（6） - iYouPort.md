   到目前为止您已经完成了第1步，即 信息篡改；你已经梳理了所有外在信息，并做到了掩人耳目（您可以在上述链接中重新回顾那些方法）。
  </p>
  <p class="graf graf--p">
   现在，你要做的就是设置一系列虚假的线索，去误导追踪你的人，使其寻找你的过程困难重重。就像
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%81%9a%e5%88%b0%e5%ae%8c%e7%be%8e%e9%9a%90%e8%ba%ab%ef%bc%9a%e6%80%9d%e8%80%83%e6%96%b9%e5%bc%8f%e6%98%af%e6%88%98%e7%95%a5%e6%80%a7%e7%9a%84%ef%bc%883%ef%bc%89/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%81%9a%e5%88%b0%e5%ae%8c%e7%be%8e%e9%9a%90%e8%ba%ab%ef%bc%9a%e6%80%9d%e8%80%83%e6%96%b9%e5%bc%8f%e6%98%af%e6%88%98%e7%95%a5%e6%80%a7%e7%9a%84%ef%bc%883%ef%bc%89/" rel="noopener noreferrer" target="_blank">
    《最危险的猎物》中 Resford 的做法
   </a>
   。
  </p>
  <p class="graf graf--p">
   此时您应该有两大目标：
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    <strong class="markup--strong markup--blockquote-strong">
     扰乱追踪者的视线，让他们在错误的地方瞎忙；尽可能将您的个人资料变得极为复杂、极难获得，并且在展开解谜的过程中，让追踪者不断遭受挫折
    </strong>
    。
   </p>
  </blockquote>
  <p class="graf graf--p">
   人们在打点行李准备上路之时，一个最常见的、生死攸关的错误就是 —— 压根没想到应该让追踪者忙起来。
  </p>
  <p class="graf graf--p">
   如果只留下了一条线索，哪怕您费尽心机极力掩饰，也同样会让追踪者有机可乘。他们可以彻底调查这一线索。
  </p>
  <p class="graf graf--p">
   别便宜了他们。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    追踪者将是你见过的最富有行动力和想象力的人；只要线索正确，他们肯定会能挖出你。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    所以，你要做的就是，让线索不正确。
   </strong>
  </p>
  <p class="graf graf--p">
   这就是为什么杜撰消息非常重要 —— 追踪者如果掌握了主动权，他们毫无疑问会玩死你；而如果你能抢占主动权，你就可以玩死他们。
  </p>
  <p class="graf graf--p">
   你需要3个要素：鱼钩、鱼线、鱼饵。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、鱼钩
   </strong>
  </p>
  <p class="graf graf--p">
   鱼钩就是你故意布下的一条线索 —— 它是假的，其目的在于让追踪者发现。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    它不仅要逼真，而且要能够让追踪者在发现时兴奋不已
   </strong>
   。
  </p>
  <p class="graf graf--p">
   还记得《La casa de papel》中“教授”如何布局培训楼的情节吗？—— 遍地都是行动计划路线图、指纹、以及各种追踪线索，并且，让警方在费尽心机的情况下才发现了这一“宝藏”。
  </p>
  <p class="graf graf--p">
   然而一切都是假象，只为了绊住警方的脚，一切都是有意布下的局，只为了拖延时间让行动者顺利脱身。这就是鱼钩。
  </p>
  <p class="graf graf--p">
   （顺便说这部剧真的值得观看，尤其是，它其中使用的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
    社交工程
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e6%88%98%e5%92%8c%e4%bf%a1%e6%81%af%e6%88%98/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e6%88%98%e5%92%8c%e4%bf%a1%e6%81%af%e6%88%98/" rel="noopener noreferrer" target="_blank">
    心理战
   </a>
   战术，非常标准；那些方法IYP的专栏中都介绍过）
  </p>
  <p class="graf graf--p">
   在信息篡改中，鱼钩可以是，比如，故意地申请房贷、故意租房、或者办理信用卡，以此误导你的追踪者对你的征信状况进行调查。
  </p>
  <p class="graf graf--p">
   你可以做任何事 —— 请注意，
   <strong class="markup--strong markup--p-strong">
    最好能事先推测到你的追踪者最容易上什么当
   </strong>
   （或者说他们习惯于从哪里开始展开调查），然后就你可以事先在这些路径上挖坑了。
  </p>
  <p class="graf graf--p">
   比如，你明知道追踪者会监听你的电话，那就用这部电话给他们放烟雾弹 —— 这一做法也在《La casa de papel》中有充分呈现，一群行动者将警方植入Rio身体的监听器变成了操纵警方的便捷武器。
  </p>
  <p class="graf graf--p">
   还是那句话，知己知彼，很重要。
   <strong class="markup--strong markup--p-strong">
    你和你的对手谁先摸清了对方的思维方式，谁就能取得胜利。和下棋的道理一样
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、鱼线
   </strong>
  </p>
  <p class="graf graf--p">
   你留下的假线索越多、越乱，你的对手就会越多眩晕，他们越是专注，死得就越快。
  </p>
  <p class="graf graf--p">
   每一条线索上都让它布满疑点，疑点是最好的引擎，追踪者会格外好奇每一个无法解开的谜。他们会想要知道答案，他们就会去研究、去实地考证，这就会消耗他们大笔的金钱和精力。
  </p>
  <p class="graf graf--p">
   更快速地接近枯竭。
  </p>
  <p class="graf graf--p">
   不要让他们轻易得到这些线索。
  </p>
  <p class="graf graf--p">
   没有人会重视那些自己能够轻易得到的东西，就如你所知道的那样，这也体现在追踪者身上 —— 你需要做的就如下棋，在3步之外等着他。看着他踏遍所有你布置好的泥坑。
  </p>
  <p class="graf graf--p">
   就算你得罪的人是政府情报机构也一样。除非你是重大通缉犯，否则政府不会有无穷无尽的预算等着被消耗。
  </p>
  <p class="graf graf--p">
   时间是这场游戏中最值钱的东西。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、鱼饵
   </strong>
  </p>
  <p class="graf graf--p">
   当权者都喜欢全面掌控，肯定会第一时间追查银行卡和在线支付的使用；那么就在这里下鱼饵给他们。
  </p>
  <p class="graf graf--p">
   比如申请一张借记卡，然后交给那种满世界飞的出差狂人；然后，追踪者就会发现“你”正在全世界飞奔。

   </strong>

   </strong>
   。在IYP的 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e6%8a%80%e6%9c%af%e9%98%b2%e8%ba%ab-%e8%87%aa%e6%88%91%e4%bf%9d%e6%8a%a4%e6%96%b9%e6%b3%95/" href="https://www.iyouport.org/category/%e6%8a%80%e6%9c%af%e9%98%b2%e8%ba%ab-%e8%87%aa%e6%88%91%e4%bf%9d%e6%8a%a4%e6%96%b9%e6%b3%95/" rel="noopener noreferrer" target="_blank">
    技术防身
   </a>
   ” 以及“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e7%bb%99%e8%a1%8c%e5%8a%a8%e8%80%85%e7%9a%84%e6%8a%a4%e8%ba%ab%e7%ac%a6%e5%92%8c%e6%ad%a6%e5%99%a8/" href="https://www.iyouport.org/category/%e7%bb%99%e8%a1%8c%e5%8a%a8%e8%80%85%e7%9a%84%e6%8a%a4%e8%ba%ab%e7%ac%a6%e5%92%8c%e6%ad%a6%e5%99%a8/" rel="noopener noreferrer" target="_blank">
    给行动者的护身符和武器
   </a>
   ” 两个栏目中找到更多安防知识。
  </p>
  <p class="graf graf--p">
   如果你想买书学习，那就花现金买，千万不要使用任何在线支付渠道。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    6、其他工具
   </strong>
  </p>
  <p class="graf graf--p">
   除了上面这些之外你还需要些什么？这取决于你想在哪里生活、做什么、如何谋生。
  </p>
  <p class="graf graf--p">
   下面这些东西都能有所帮助（并非所有国家都能顺利找到这些东西，选择在你的国家相对可行的类似方法即可。
   <strong class="markup--strong markup--p-strong">
    这里的重点是思路而不是工具
   </strong>
   ）：
  </p>
  <p class="graf graf--p">
   -来电身份伪装卡/APP：它可以改变你的号码在对方手机上的显示方式。
  </p>
  <p class="graf graf--p">
   -开脱服务：类似 alibinetwork 那种，同类的还有很多。可提供由旅行社、酒店、研讨会和“雇主”提供的假的不在场证明。约会进展不顺利时他们还可以提供假的“救援电话”。
  </p>
  <p class="graf graf--p">
   -保守型语音信箱服务：允许你登陆和发送语音邮件，该邮件直接进入对方的邮箱，并没有电话提醒。
  </p>
  <p class="graf graf--p">
   -一次性电邮地址：曾经介绍过，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%a7%92%e8%89%b2%e6%89%ae%e6%bc%94%ef%bc%9a%e8%a1%8c%e5%8a%a8%e8%80%85%e5%88%9a%e9%9c%80video/" href="https://www.iyouport.org/%e8%a7%92%e8%89%b2%e6%89%ae%e6%bc%94%ef%bc%9a%e8%a1%8c%e5%8a%a8%e8%80%85%e5%88%9a%e9%9c%80video/" rel="noopener noreferrer" target="_blank">
    角色扮演
   </a>
   》、以及《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%BC%AA%E8%A3%85%E6%93%A6%E9%99%A4%e2%80%8a-%e2%80%8a-%E5%9C%A8%E7%BA%BF%E9%9A%90%E8%BA%AB%E7%9A%84%E9%87%8D%E8%A6%81%E6%AD%A5%E9%AA%A4%EF%BC%88%E5%B0%91%E6%95%B0%E6%B4%BE%E7%8E%A9/" href="https://www.iyouport.org/%E4%BC%AA%E8%A3%85%E6%93%A6%E9%99%A4%e2%80%8a-%e2%80%8a-%E5%9C%A8%E7%BA%BF%E9%9A%90%E8%BA%AB%E7%9A%84%E9%87%8D%E8%A6%81%E6%AD%A5%E9%AA%A4%EF%BC%88%E5%B0%91%E6%95%B0%E6%B4%BE%E7%8E%A9/" rel="noopener noreferrer" target="_blank">
    伪装+擦除
   </a>
   》
  </p>
  <p class="graf graf--p">
   -自由手机：没有身份信息、没有任何官僚主义的麻烦、支持比特币购买。但是小贵（当然 Purism 也一样），您需要权衡是钱重要还是命重要。在这里看到更多介绍：ptshamrock.com/auto/freedomphone.htm
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    7、记住紧跟时代
   </strong>
  </p>
  <p class="graf graf--p">
   科技每时每刻都在进步，政策也在变化；所以你必须紧跟这一进度，在出现新的跟踪监视技术/手段和安检政策时及时考虑添加新的策略。
  </p>
  <p class="graf graf--p">
   但要知道，
   <strong class="markup--strong markup--p-strong">
    最新的技术不会很快投入使用，更不会很快变得高效；人力监视在近几年内都将作为技术监视的必需搭配
   </strong>
   。
  </p>
  <p class="graf graf--p">
   于是，把重点发在搞定人心上（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
    社交工程
   </a>
   ）比不断遭受新技术惊吓，要实际得多。
  </p>
  <p class="graf graf--p">
   ⚠️尤其是，监视是一个持续不断的过程；随着时间的推移它对你的危险程度会不断加深 —— 因为它了解你更多了。
  </p>
  <p class="graf graf--p">
   ⚠️所以，不要指望在短时间内迅速实现全面隐藏自己。您应该在平日里就多加防备，使用我们提供的自我保护方法，熟悉这些安防技巧，这样才能在需要的时候快速实现一步到位。
  </p>
  <p class="graf graf--p">
   这就像攒钱的道理 —— 就算您现在什么都没打算买，也应该节俭以保证有足够的积蓄，这样才能应对不时之需。⚪️
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    —— 未完待续 ——
   </em>
  </p>
