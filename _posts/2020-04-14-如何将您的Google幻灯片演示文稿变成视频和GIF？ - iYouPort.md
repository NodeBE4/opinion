 <div class="entry-content">
  <ul>
   <li class="graf graf--p">
    <span style="color: #00ccff;">
     <em>
      <strong>
       中国有很多优秀的互联网技术玩家，只是很多技术的使用限于营销的目的。并不是说营销不对，而是可惜活动家社区在宣传推广自己的理念方面做的不够帅气。IYP介绍了很多有趣的玩法，帮助活动家社区在动员方面更具魅力
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-10761 jetpack-lazy-image" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/头.gif?resize=820%2C392&amp;is-pending-load=1#038;ssl=1" data-recalc-dims="1" height="392" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/头.gif?resize=820%2C392&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="820"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10761" data-recalc-dims="1" height="392" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/头.gif?resize=820%2C392&amp;ssl=1" width="820"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   使用 Microsoft Powerpoint，您可以轻松地将演示文稿转换为高分辨率视频文件，以上传到YouTube和其他视频网站。
  </p>
  <p class="graf graf--p">
   导出的视频文件还可以包含演示文稿中可能包含的所有语音旁白和背景音频。
  </p>
  <p class="graf graf--p">
   Google 幻灯片没有提供将演示文稿另存为视频的选项，但有一个新插件 ——
   <a class="markup--anchor markup--p-anchor" data-href="https://creatorstudio.dev/" href="https://creatorstudio.dev/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Creator Studio
    </strong>
   </a>
   ，可以为您的Google幻灯片带来新的导出功能。
  </p>
  <p class="graf graf--p">
   使用 Creator Studio，您还可以将演示文稿另存为动画GIF、图像序列、和高清视频（带有音频的哦）。
  </p>
  <p class="graf graf--p">
   下面是视频教程：
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/f_3hHIgnlYY" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   首先，请转到
   <a class="markup--anchor markup--p-anchor" data-href="https://creatorstudio.dev/" href="https://creatorstudio.dev/" rel="noopener noreferrer" target="_blank">
    creatorstudio.dev
   </a>
   并安装Google幻灯片附加组件。它需要某些权限，因为它必须读取图像以将其转换为电影。
  </p>
  <p class="graf graf--p">
   由于导出的文件会自动上传到您的Google云端硬盘，因此它还需要访问您在Google云端硬盘中创建文件的权限。
  </p>
  <p class="graf graf--p">
   安装附加组件后，打开Goog​​le幻灯片中至少包含2张以上幻灯片的任何文稿。转到附加组件菜单，选择 Creator Studio，它将打开一个侧边栏。
  </p>
  <p class="graf graf--p">
   指定输出文件的宽度（以像素为单位）。 Creator Studio 将自动计算高度以保持演示文稿的原始宽高比。
  </p>
  <p class="graf graf--p">
   接下来，指定时间间隔（以秒为单位），这是每张幻灯片在进入下一张幻灯片之前在视频中可见的持续时间。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10752 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-25.png?resize=1100%2C745&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-25.png?w=1512&amp;ssl=1 1512w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-25.png?resize=300%2C203&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-25.png?resize=1024%2C694&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-25.png?resize=768%2C520&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-25.png?resize=1100%2C745&amp;ssl=1 1100w" data-recalc-dims="1" height="745" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-25.png?resize=1100%2C745&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10752" data-recalc-dims="1" height="745" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-25.png?resize=1100%2C745&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-25.png?w=1512&amp;ssl=1 1512w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-25.png?resize=300%2C203&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-25.png?resize=1024%2C694&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-25.png?resize=768%2C520&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-25.png?resize=1100%2C745&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   最后，您有多种导出选项可供选择，而这些选项在Google幻灯片中本身是不可用的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    动图 ——
   </strong>
  </p>
  <p class="graf graf--p">
   可以创建无限循环的GIF图像，非常适合在加密通信和博客帖子中共享。您甚至可以通过将时间间隔设定得很短（例如0.2秒）来制作定格动画。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    图像序列 ——
   </strong>
  </p>
  <p class="graf graf--p">
   Creator Studio 将捕获演示文稿中每张幻灯片的屏幕截图，并将其另存为PNG文件到您的Google云端硬盘文件夹中。在内部，它使用 Google Slides API 生成这些缩略图。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    MP4视频 ——
   </strong>
  </p>
  <p class="graf graf--p">
   像GIF图像一样，Creator Studio 也可以生成MP4视频文件，您还可以调整播放速度。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    带音频的视频 ——
   </strong>
  </p>
  <p class="graf graf--p">
   您可以上传任何MP3或WAV格式的音频文件，该文件将与演示文稿一起播放。也就是说，您可以上传背景音乐甚至语音旁白。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-10757 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-2.png?resize=1100%2C759&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-2.png?w=1563&amp;ssl=1 1563w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-2.png?resize=300%2C207&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-2.png?resize=1024%2C706&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-2.png?resize=768%2C530&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-2.png?resize=1536%2C1059&amp;ssl=1 1536w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-2.png?resize=1100%2C759&amp;ssl=1 1100w" data-recalc-dims="1" height="759" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-2.png?resize=1100%2C759&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10757" data-recalc-dims="1" height="759" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-2.png?resize=1100%2C759&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-2.png?w=1563&amp;ssl=1 1563w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-2.png?resize=300%2C207&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-2.png?resize=1024%2C706&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-2.png?resize=768%2C530&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-2.png?resize=1536%2C1059&amp;ssl=1 1536w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/2-2.png?resize=1100%2C759&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   下面专门说一下做GIF的方法。因为它是
   <strong class="markup--strong markup--p-strong">
    活动家宣传工作中利器之一
   </strong>
   。
  </p>
  <p class="graf graf--p">
   介绍适用于Google幻灯片的全新 “
   <a class="markup--anchor markup--p-anchor" data-href="https://talltweets.com/" href="https://talltweets.com/" rel="noopener noreferrer" target="_blank">
    Tall Tweets
   </a>
   ” 应用。它将帮助您将演示文稿转换为高质量的动画GIF，并将其添加到电子邮件中、嵌入到网页中、或直接在 Twitter 上共享。
  </p>
  <p class="graf graf--p">
   下面是演示视频：
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/POdIuzaFVH8" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   首先，请打开 talltweets.com，使用您的Google帐户登录，然后从Google云端硬盘中选择任何现有的演讲文稿。
  </p>
  <p class="graf graf--p">
   TallTweets 应用程序可以将整个文稿转换为无限循环的GIF，也可以选择将演示文稿的各个幻灯片转换为PNG图像。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10759 jetpack-lazy-image" data-lazy-sizes="(max-width: 1072px) 100vw, 1072px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/3.png?resize=1072%2C438&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/3.png?w=1072&amp;ssl=1 1072w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/3.png?resize=300%2C123&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/3.png?resize=1024%2C418&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/3.png?resize=768%2C314&amp;ssl=1 768w" data-recalc-dims="1" height="438" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/3.png?resize=1072%2C438&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1072"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10759" data-recalc-dims="1" height="438" sizes="(max-width: 1072px) 100vw, 1072px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/3.png?resize=1072%2C438&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/3.png?w=1072&amp;ssl=1 1072w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/3.png?resize=300%2C123&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/3.png?resize=1024%2C418&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/3.png?resize=768%2C314&amp;ssl=1 768w" width="1072"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   您可以使用多个选项来自定义生成的GIF图像。您可以指定GIF图像的宽度（高度将自动计算以保持原始的宽高比），还可以通过更改幻灯片之间的时间间隔来加快或减慢播放速度。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在GIF中跳过或重新排序幻灯片
   </strong>
  </p>
  <p class="graf graf--p">
   GIF制作工具的一个独特功能是，它使您可以选择更改生成的GIF中的幻灯片图像出现顺序。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10760 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/4.jpg?resize=800%2C510&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/4.jpg?w=800&amp;ssl=1 800w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/4.jpg?resize=300%2C191&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/4.jpg?resize=768%2C490&amp;ssl=1 768w" data-recalc-dims="1" height="510" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/4.jpg?resize=800%2C510&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10760" data-recalc-dims="1" height="510" sizes="(max-width: 800px) 100vw, 800px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/4.jpg?resize=800%2C510&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/4.jpg?w=800&amp;ssl=1 800w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/4.jpg?resize=300%2C191&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/4.jpg?resize=768%2C490&amp;ssl=1 768w" width="800"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   假设您的演示文稿有6张幻灯片。如果将序列输入框留为空白，则会按顺序显示所有幻灯片在GIF中。但是，您可以跳过其中某些幻灯片、对其重新排序、甚至更改单个幻灯片显示的时间长度。
  </p>
  <p class="graf graf--p">
   比如 1,2,3,6 —— 就是跳过 #4和#5 播放；
  </p>
  <p class="graf graf--p">
   6,5,4,3 —— 将颠倒幻灯片的顺序；
  </p>
  <p class="graf graf--p">
   1,2,2,2,3,4 –将延长幻灯片＃2的持续时间。
  </p>
  <p class="graf graf--p">
   首次启动 Tall Tweets 时，将需要您授权对 Google 云端硬盘的访问权限，以便它可以选择 Google Presentation 来生成幻灯片。它会在您的浏览器中本地生成GIF，并且每个字节都不会离开您的Google帐户。
  </p>
  <p class="graf graf--p">
   您学会了吗？希望中国的活动家在宣传方面有更多创新，并因此吸引到更多的支持者，加油！⚪️
  </p>
