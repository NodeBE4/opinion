  <p class="graf graf--p">
   互联网正在对新闻关闭大门。
  </p>
  <p class="graf graf--p">
   在全球数字广告失败之后，出版商将希望寄托在订阅上。
  </p>
  <p class="graf graf--p">
   今年 WAN-IFRA 在巴黎举办的数字订阅会议上，我向西雅图时报总裁兼出版商 Alan Fisco 询问了他对数字新闻业务中“特朗普冲击 Trump Bump”的看法。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这是让我夜不能寐的事情之一”，他回答道。
  </p>
  <p class="graf graf--p">
   （特朗普养活了一众媒体，这不是新闻《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%8e%b0%e4%bb%a3%e6%96%b0%e9%97%bb%e4%b9%8b%e6%84%9a%e8%a0%a2-2%e2%80%8a-%e2%80%8a%e5%9b%9b%e7%a7%8d%e6%9c%80%e8%87%b4%e5%91%bd%e7%9a%84%e5%80%be%e5%90%91%e6%80%a7-%e4%b8%ad%e5%9b%bd/" href="https://www.iyouport.org/%e7%8e%b0%e4%bb%a3%e6%96%b0%e9%97%bb%e4%b9%8b%e6%84%9a%e8%a0%a2-2%e2%80%8a-%e2%80%8a%e5%9b%9b%e7%a7%8d%e6%9c%80%e8%87%b4%e5%91%bd%e7%9a%84%e5%80%be%e5%90%91%e6%80%a7-%e4%b8%ad%e5%9b%bd/" rel="noopener noreferrer" target="_blank">
    现代新闻之愚蠢
   </a>
   (2)》）
  </p>

  <p class="graf graf--p">
   自2016年大选以来，ST数字用户增长了2.3倍，而出版商采取了多个步骤来招募和挽留付费读者，投资于技术、数据分析和客户关系管理。
  </p>
  <p class="graf graf--p">
   因此，该报能够彻底改变其收入模式。
  </p>
  <p class="graf graf--p">
   2005年，广告收入与发行的比例为90%/10％。现在，广告占38％，发行占62％：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6330 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-1.png?resize=1100%2C681&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-1.png?w=1600&amp;ssl=1 1600w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-1.png?resize=300%2C186&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-1.png?resize=768%2C476&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-1.png?resize=1024%2C634&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-1.png?resize=1100%2C681&amp;ssl=1 1100w" data-recalc-dims="1" height="681" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-1.png?resize=1100%2C681&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   在最近转变为付费墙的过程中，太平洋西北地区媒体没有过分关注其商业模式的可持续性：它牢牢抓住了
   <strong class="markup--strong markup--p-strong">
    两个重要的观众驱动力 —— 本地新闻和体育。
   </strong>
  </p>
  <p class="graf graf--p">
   它也从未在新闻业务上偷工减料。它获得了10个普利策奖，最近还推出了一个调查基金，并且仍然是航空航天和科技新闻的绝佳来源。
  </p>
  <p class="graf graf--p">
   简而言之，“西雅图时报”保留了
   <strong class="markup--strong markup--p-strong">
    现代新闻经济学中的两个关键要素：独特性和价值主张。
   </strong>
  </p>
  <p class="graf graf--p">
   以下是成功订阅模型的独特之处：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6331 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-2.png?resize=1100%2C459&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-2.png?w=1372&amp;ssl=1 1372w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-2.png?resize=300%2C125&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-2.png?resize=768%2C320&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-2.png?resize=1024%2C427&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-2.png?resize=1100%2C459&amp;ssl=1 1100w" data-recalc-dims="1" height="459" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-2.png?resize=1100%2C459&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6331" data-recalc-dims="1" height="459" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-2.png?resize=1100%2C459&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-2.png?w=1372&amp;ssl=1 1372w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-2.png?resize=300%2C125&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-2.png?resize=768%2C320&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-2.png?resize=1024%2C427&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-2.png?resize=1100%2C459&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p>
   <strong class="markup--strong markup--li-strong">
    1、品牌信誉与亲和力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这些是读者考虑订阅的主要动机。
  </p>
  <p class="graf graf--p">
   不幸的是，行业趋势模糊了这些观念：
   <strong class="markup--strong markup--p-strong">
    近一半的用户从社交媒体平台获取新闻，而不是新闻网站，能够记住原始信息来源的人已经越来越少
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这是集中化平台造成的最严重的破坏
   </strong>
   ：Facebook、WhatsApp、谷歌新闻已经淡化了新闻品牌的概念 —— 以及作者身份的概念 —— 要知道这是新闻信誉的两大支柱。
  </p>
  <p class="graf graf--p">
   而 Apple News 更擅长在其界面中展示品牌，但其
   <a class="markup--anchor markup--p-anchor" data-href="https://mondaynote.com/apple-news-could-lead-to-a-massive-value-destruction-for-the-magazine-industry-752d297bffe" href="https://mondaynote.com/apple-news-could-lead-to-a-massive-value-destruction-for-the-magazine-industry-752d297bffe" rel="noopener noreferrer" target="_blank">
    破坏性的商业模式
   </a>
   —— 以及“Spotify for News”的
   <a class="markup--anchor markup--p-anchor" data-href="https://mondaynote.com/why-a-spotify-for-news-cant-fly-6f261793f29d" href="https://mondaynote.com/why-a-spotify-for-news-cant-fly-6f261793f29d" rel="noopener noreferrer" target="_blank">
    幻想
   </a>
   —— 正在扼杀新闻业的经济效益。
  </p>
  <p class="graf graf--p">
   在这里看到解释《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a4%a7%e7%a7%91%e6%8a%80%e5%af%a1%e5%a4%b4%e6%98%af%e5%90%a6%e7%9c%9f%e7%9a%84%e5%85%b3%e5%bf%83%e6%96%b0%e9%97%bb%e4%b8%9a%ef%bc%9f%e7%ac%91%e8%af%9d/" href="https://www.iyouport.org/%e5%a4%a7%e7%a7%91%e6%8a%80%e5%af%a1%e5%a4%b4%e6%98%af%e5%90%a6%e7%9c%9f%e7%9a%84%e5%85%b3%e5%bf%83%e6%96%b0%e9%97%bb%e4%b8%9a%ef%bc%9f%e7%ac%91%e8%af%9d/" rel="noopener noreferrer" target="_blank">
    科技寡头是否真的关心新闻业？笑话
   </a>
   》。
  </p>
  <p class="graf graf--p">
   恢复新闻品牌应该是该行业的第一步。
  </p>
  <p>
   <strong class="markup--strong markup--p-strong">
    2、独特性。
   </strong>
  </p>
  <p class="graf graf--p">
   这是将常规用户转换为付费用户的主要因素 — 但发布商仍然忽略了这一点。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    一家媒体必需能够说服读者，它是*唯一一个*在其报道中提供特定内容/角度/广度和深度的媒体
   </strong>
   ，比其他所有媒体都好，这样你才能要求人们付钱。
  </p>
  <p class="graf graf--p">
   这应该导致一些出版商重新考虑其产出的价值。
  </p>
  <p class="graf graf--p">
   目前，一家大型媒体机构，就如每天报道200个故事的全国性报纸，将承认
   <strong class="markup--strong markup--p-strong">
    只有
   </strong>
   15％到20％的报纸可以被贴上真正的增值标签，增值取决于新闻选择、强大的新闻报道、丰富的信息抓取等指标。
  </p>
  <p class="graf graf--p">
   剩下的就是所谓的
   <strong class="markup--strong markup--p-strong">
    “商品新闻”
   </strong>
   了，仅仅是为了说明需要穷举，这样的新闻已经被剥夺了特定的新闻价值和货币价值。
  </p>
  <p class="graf graf--p">
   由于担心内部的封地斗争，很少有新闻编辑部管理人员愿意重新审视这一原则。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    要求读者支付质量更好的新闻的额模糊主张不再是一个有吸引力的承诺。相反，揭示你最擅长的东西才是一个强有力的声明
   </strong>
   。
  </p>
  <p class="graf graf--p">
   延伸《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bc%95%e6%b5%81%e6%9a%b4%e6%94%bf%e5%92%8c%e7%94%9f%e5%ad%98%e5%9b%b0%e6%83%91/" href="https://www.iyouport.org/%e5%bc%95%e6%b5%81%e6%9a%b4%e6%94%bf%e5%92%8c%e7%94%9f%e5%ad%98%e5%9b%b0%e6%83%91/" rel="noopener noreferrer" target="_blank">
    引流暴政和生存困惑
   </a>
   》
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、明确的价值承诺。
   </strong>
  </p>
  <p class="graf graf--p">
   最成功的出版商的陈述清楚地表明了内容对读者的好处：“我们节省了时间”（经济学人）; “读者从我们提供的新闻中获取价值”（“金融时报”）。
  </p>
  <p class="graf graf--p">
   很少有出版商愿意在一个简短的概念中总结出自己的独特之处。
  </p>
  <p class="graf graf--p">
   一个读者可以选择每月10到15美元的价格，但可自由支配的收入一般只能应付两个订阅，这种情况下人们想要的不仅仅是一个模糊的承诺。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、对价格的感知价值
   </strong>
  </p>
  <p class="graf graf--p">
   简而言之，客户提出的问题是：我的钱能换来什么？
  </p>
  <p class="graf graf--p">
   对于一些人来说，由前行业首席财务官制作的关于航空公司经济的每周通讯、原始分析和/或聪明的策划，每个月价值30美元。
  </p>
  <p class="graf graf--p">
   而在其他情况下，对高质量新闻服务的订阅将失去价值，因为其内容往往会出现在其他地方。
  </p>
  <p class="graf graf--p">
   例如：
   <strong class="markup--strong markup--p-strong">
    来自付费媒体的优质报道很可能被无数免费的出版物转
   </strong>
   载。
  </p>
  <p class="graf graf--p">
   在信息快速衰退的时代，订阅的价值应该在于其他地方。
  </p>
  <p class="graf graf--p">
   例如，在早期访问原始和未过滤的信息；或者发布商界面提供的多种功能：细节、相关故事、多媒体元素、推荐引擎和个性化系统。
  </p>
  <p class="graf graf--p">
   延伸《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bf%98%e8%ae%b0%e5%81%87%e6%96%b0%e9%97%bb%e5%90%a7%ef%bc%8c%e6%a0%b9%e6%9c%ac%e6%80%a7%e9%97%ae%e9%a2%98%e5%9c%a8%e4%ba%8e%e7%9c%9f%e6%96%b0%e9%97%bb%e6%9c%ac%e8%ba%ab/" href="https://www.iyouport.org/%e5%bf%98%e8%ae%b0%e5%81%87%e6%96%b0%e9%97%bb%e5%90%a7%ef%bc%8c%e6%a0%b9%e6%9c%ac%e6%80%a7%e9%97%ae%e9%a2%98%e5%9c%a8%e4%ba%8e%e7%9c%9f%e6%96%b0%e9%97%bb%e6%9c%ac%e8%ba%ab/" rel="noopener noreferrer" target="_blank">
    忘记假新闻吧，根本性问题在于真新闻本身
   </a>
   》
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、客户关怀。
   </strong>
  </p>
  <p class="graf graf--p">
   这是留住客户以及新旧世界碰撞的最少考虑因素。
  </p>
  <p class="graf graf--p">
   在频谱的一端，你能看到像 Le Monde 这样的出版商要求挂号信取消订阅；或者如 The Information 那样的，当你在主要价格上涨时抗议不必要的订阅延期时你会收到他们严厉的回复““阅读细则，蠢货”。
  </p>
  <p class="graf graf--p">
   这种行为传达了一个微弱的信息：“为了实现我们的目标，并且因为我们不确定我们的产品能否让你留在这里，所以我们需要不惜一切代价锁定你”。
  </p>
  <p class="graf graf--p">
   作为客户，您可以咬紧牙关并在日历中添加提醒，以便在适当的时候取消。
  </p>
  <p class="graf graf--p">
   当你终于等到取消的机会时，你会得到一个自动回复，询问：“你会为100美元的折扣留下吗？”，然后你惊讶于该服务的真正价值。选择了退出。
  </p>
  <p class="graf graf--p">
   相比下，在向客户表达一些考虑因素时，科技寡头不会这么干。
  </p>
  <p class="graf graf--p">
   他们依靠正确的指标来做到这一点。当我对亚马逊说“我没有收到这个项目”时，我会得到退款，毫无疑问。
  </p>
  <p class="graf graf--p">
   为什么？因为，我是一个长期客户，具有显著的每用户平均收入（ARPU），更重要的是，高可靠性得分（我不滥用系统）。
  </p>
  <p class="graf graf--p">
   其他人选择让订阅门一直打开状态，就像 Netflix 那样，我可以在任何时候离开并按照自己的意愿随时可以回来。
  </p>
  <p class="graf graf--p">
   这种策略有几个优点： —— 客户享有情感、信任和自由； —— 在内部，与即将到来的订阅系统相关的固有脆弱性使每个人都处于优势地位：客户的概念渗透到公司的每一层。没有人满足于自己的成就。
  </p>
  <p class="graf graf--p">
   而新闻业不愿意考虑锁定客户模式之外的任何事。
  </p>
  <p class="graf graf--p">
   但事情似乎正在发生变化。根据今年在佩鲁贾新闻节上发表演讲的卫报执行官的说法，该报并不排除为其订户提供完全自由的选择。
  </p>
  <p class="graf graf--p">
   如果真的能做到，这有可能是一项业务革命。
  </p>
  <p class="graf graf--p">
   但是好像变革没那么简单。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    订阅领域正在由大型玩家接管，他们将不遗余力地建立最强大的准入门槛。
   </strong>
  </p>
  <p class="graf graf--p">
   订阅模型正在像杂草一样传播。只需看看您的手机屏幕就能知道：音乐、存储、应用程序、游戏、停车、健身……等等一大堆。
  </p>
  <p class="graf graf--p">
   我发现了一款名为 Bobby 的应用程序，致力于管理和跟踪您的订阅。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-6333 jetpack-lazy-image" data-lazy-sizes="(max-width: 892px) 100vw, 892px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/APP.png?resize=892%2C704&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/APP.png?w=892&amp;ssl=1 892w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/APP.png?resize=300%2C237&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/APP.png?resize=768%2C606&amp;ssl=1 768w" data-recalc-dims="1" height="704" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/APP.png?resize=892%2C704&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="892"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   对这种产品的需求量肯定很大。
  </p>
  <p class="graf graf--p">
   然而，有两件事让我感到震惊：
   <strong class="markup--strong markup--p-strong">
    在用于演示应用程序的屏幕截图中，没有一个是新闻产品；第二个是视觉效果中显示的数字：差不多每个月花费了数百美元用于各种付费服务……
   </strong>
  </p>
  <p class="graf graf--p">
   通过大规模切换到订阅模式，新闻部门正面临着一个可以在两个方面进行映射的惨淡竞争：
  </p>
  <p class="graf graf--p">
   个人或家庭可以分配给信息（一般、本地、专业）的可支配收入金额；以及可用于新闻消费的时间。
  </p>
  <p class="graf graf--p">
   前者不是新的。二十年前，你可以争辩说，订阅国家或地区报纸（加上一两本杂志）的支出是其他支出之外的可承受的必要性。没错，但是发生了两次变化。
  </p>
  <p class="graf graf--p">
   第一个是新闻收入模式的逆转。二十年前，25美分的报纸（算上通胀率也不到40美分）的基础是广告收入占比 80％，发行收入占比 20％。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    今天的比率倒转了：广告占30％，发行占70％
   </strong>
   。
  </p>
  <p class="graf graf--p">
   新闻媒体行业的生存、原创的生产能力、和经过验证的编辑，都需要以一种反复的方式伸入读者口袋。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    但问题是，许多玩家都想要做同样的情，他们一直在不遗余力地占领他们的市场份额
   </strong>
   。对于一个人来说，哪来这么多钱支付这些东西？
  </p>
  <p class="graf graf--p">
   延伸《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%9c%b0%e7%8b%b1%e8%88%ac%e7%9a%84%e4%bb%98%e8%b4%b9%e5%a2%99%ef%bc%9a%e5%85%8d%e8%b4%b9%e3%80%81%e5%bc%80%e6%94%be%e5%92%8c%e6%b0%91%e4%b8%bb%e5%8c%96%e7%9a%84-web-%e6%97%b6%e4%bb%a3%e5%b7%b2/" href="https://www.iyouport.org/%e5%9c%b0%e7%8b%b1%e8%88%ac%e7%9a%84%e4%bb%98%e8%b4%b9%e5%a2%99%ef%bc%9a%e5%85%8d%e8%b4%b9%e3%80%81%e5%bc%80%e6%94%be%e5%92%8c%e6%b0%91%e4%b8%bb%e5%8c%96%e7%9a%84-web-%e6%97%b6%e4%bb%a3%e5%b7%b2/" rel="noopener noreferrer" target="_blank">
    地狱般的付费墙：免费、开放和民主化的 Web 时代已成往事了吗？
   </a>
   》
  </p>
  <p class="graf graf--p">
   考虑在花费的时间方面，情况就很棘手了。与货币分配不同，时间是不能扩大的。
  </p>
  <p class="graf graf--p">
   顺便说，
   <strong class="markup--strong markup--p-strong">
    中国的手段之影响真的很糟糕 —— 学习强国APP 让用户“赚取利益”而不是支付
   </strong>
   ，它直接与工作绩效甚至继续工作的机会（威胁性）挂钩，严重侵占人们的精力和时间，其他优质内容将被严重排挤。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6332 jetpack-lazy-image" data-lazy-sizes="(max-width: 1016px) 100vw, 1016px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/图.png?resize=1016%2C938&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/图.png?w=1016&amp;ssl=1 1016w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/图.png?resize=300%2C277&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/图.png?resize=768%2C709&amp;ssl=1 768w" data-recalc-dims="1" height="938" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/图.png?resize=1016%2C938&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1016"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   Netflix 首席执行官 Reed Hastings 甚至自称：
   <strong class="markup--strong markup--p-strong">
    “你得到的节目或电影真的很难看，而你最终熬夜看，所以我们实际上是在与睡眠竞争。而我们赢了！”
   </strong>
  </p>
  <p class="graf graf--p">
   准确说，以流媒体视频点播（SVOD）为例。一年前，Deloitte 发表了一项引人注目的研究：
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    <em>
     目前，55％的美国家庭订购了至少一种视频流媒体服务，自2009年以来增长了450％；
    </em>
   </p>
  </blockquote>
  <blockquote class="graf graf--blockquote">
   <p>
    <em>
     平均而言，美国人每周观看38小时的视频内容（其中39％是流式播放），
     <strong class="markup--strong markup--blockquote-strong">
      几乎相当于一份全职工作；
     </strong>
    </em>
   </p>
  </blockquote>
  <blockquote class="graf graf--blockquote">
   <p>
    <em>
     在美国有超过200个 SVOD 选项，平均流媒体视频用户正在为三项服务付费，导致美国消费者整体
     <strong class="markup--strong markup--blockquote-strong">
      每月共同花费21亿美元
     </strong>
     购买 SVOD 服务；
    </em>
   </p>
  </blockquote>
  <blockquote class="graf graf--blockquote">
   <p>
    <em>
     高质量的原创内容似乎推动了流媒体的增长，近一半（48％）的美国消费者每天或每周都收看流媒体电视内容，同比增长11％。
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p">
   Netflix 的消费量为每天1小时36分，其电视剧剧集的一次性发行覆盖美国所有观众的70％；90％的千禧一代一次消费5集。
  </p>
  <p class="graf graf--p">
   考虑到游戏，情况更糟：亚马逊拥有的平台 Twitch 的用户每天花费1小时30分；三分之一的 Fortnite 玩家每周花费6到10个小时，38%的人每周超过10个小时；还有三分之二的人在游戏中以85美元的价格购买了各种额外内容……
  </p>
  <p class="graf graf--p">
   看到这些惊人的数字，销售订阅的思路需要一个大幅度的信念飞跃。
  </p>
  <p class="graf graf--p">
   以下是与媒体行业争夺金钱和时间的不同服务的摘要。
  </p>

  <p class="graf graf--p">
   最表面化的结论是，如果新闻媒体部门能
   <strong class="markup--strong markup--p-strong">
    专注于其核心产品价值，投资技术
   </strong>
   、停止像对待乘客一样对待客户，还是有希望解决问题的。
  </p>
  <p class="graf graf--p">
   但你必需知道，你提供的内容
   <strong class="markup--strong markup--p-strong">
    一定是要对读者具有高使用价值的、真实的、全面且充分的、具有透彻的历史反思能力和前瞻能力的洞见
   </strong>
   ，否则，不要期待任何人在上当一次后还会继续上当。
  </p>
  <p class="graf graf--p">
   订阅模式在一定程度上有好处，提高了人们的辨析意识，宣传、偏颇和不实用的内容将更有希望被过滤掉。当然这需要一定时间，需要在很多人发现了价值和发现没有价值的评判过程结束之后。⚪️
  </p>
  <p class="graf graf--p">
   重申，IYP 的立场依旧是替代媒体，而不是付费墙，让新闻业重回民主化的本质。在这里看到我们的解释《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ae%a9%e5%ae%83%e6%b0%91%e4%b8%bb%ef%bc%9a%e5%9c%a8%e7%9c%9f%e7%9b%b8%e4%ba%8b%e4%b8%9a%e4%b8%ad%e6%9e%84%e6%83%b3%e6%9c%aa%e6%9d%a5%e5%9b%be%e6%99%af%ef%bc%88%e4%b8%8a%ef%bc%89/" href="https://www.iyouport.org/%e8%ae%a9%e5%ae%83%e6%b0%91%e4%b8%bb%ef%bc%9a%e5%9c%a8%e7%9c%9f%e7%9b%b8%e4%ba%8b%e4%b8%9a%e4%b8%ad%e6%9e%84%e6%83%b3%e6%9c%aa%e6%9d%a5%e5%9b%be%e6%99%af%ef%bc%88%e4%b8%8a%ef%bc%89/" rel="noopener noreferrer" target="_blank">
    让它民主：在真相事业中构想未来图景（上）
   </a>
   》《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ae%a9%e5%ae%83%e6%b0%91%e4%b8%bb%ef%bc%9a%e6%9b%bf%e4%bb%a3%e5%aa%92%e4%bd%93%e4%bd%9c%e4%b8%ba%e6%8a%97%e4%ba%89%e4%b9%8b%e6%ba%90%ef%bc%88%e4%b8%8b%ef%bc%89/" href="https://www.iyouport.org/%e8%ae%a9%e5%ae%83%e6%b0%91%e4%b8%bb%ef%bc%9a%e6%9b%bf%e4%bb%a3%e5%aa%92%e4%bd%93%e4%bd%9c%e4%b8%ba%e6%8a%97%e4%ba%89%e4%b9%8b%e6%ba%90%ef%bc%88%e4%b8%8b%ef%bc%89/" rel="noopener noreferrer" target="_blank">
    让它民主：替代媒体作为抗争之源（下）
   </a>
   》
