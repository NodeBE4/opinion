  <p class="graf graf--p">
   使用在新疆控制维吾尔族人同样的方法，中国政府通过部署无人机、面部识别和雷达监控确保了一个几乎无法通行的边界，加强了对西藏的控制。
  </p>
  <p class="graf graf--p">
   在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%be%b5%e7%8a%af%e4%ba%ba%e6%9d%83%e6%af%94%e4%b8%80%e5%a0%b5%e5%a2%99%e6%9b%b4%e5%a5%bd%e5%90%97%ef%bc%9f/" href="https://www.iyouport.org/%e4%be%b5%e7%8a%af%e4%ba%ba%e6%9d%83%e6%af%94%e4%b8%80%e5%a0%b5%e5%a2%99%e6%9b%b4%e5%a5%bd%e5%90%97%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    建立真正有效的隔离墙的中美竞赛
   </a>
   中，中国提供了一个具有独特目标的模式：将被压迫的少数民族[以及他们受压迫的遭遇]隐藏在国内。
  </p>
  <p class="graf graf--p">
   几十年来，该地区令人生畏的地形使这项任务变得困难。在50年代被中国占领和吞并的西藏与南亚之间的边界是一个广阔的空间，绵延4,500多公里。边界大部分沿喜马拉雅山脉，超长的距离和偏远的地形使制造一堵物理墙变得不切实际。
  </p>
  <p class="graf graf--p">
   从1959年到2008年，每年有成千上万的藏人艰险跋涉穿越山脉。如今，估计有15万藏人居住在国外，印度、尼泊尔或包括美国和加拿大在内的西方国家。几乎所有人都可以将他们的自由追溯到危险的喜马拉雅旅程，而中国占领军很难有效地在那里巡逻。
  </p>
  <p class="graf graf--p">
   然而，近年来，来自西藏的难民流动
   <a class="markup--anchor markup--p-anchor" data-href="https://timesofindia.indiatimes.com/india/97-drop-in-tibetan-refugees-arrival-to-india-from-tibet/articleshow/67566650.cms" href="https://timesofindia.indiatimes.com/india/97-drop-in-tibetan-refugees-arrival-to-india-from-tibet/articleshow/67566650.cms" rel="noopener noreferrer" target="_blank">
    几乎完全停止
   </a>
   。
  </p>
  <p class="graf graf--p">
   2007年，约有3,000名藏人进入印度。到2017年，这一数字下降到只有80。
  </p>
  <p class="graf graf--p">
   与中国希望世界相信的理由完全不同，其原因并不是西藏的情况“有所改善” —— 实际上，几乎每个人权组织都认为西藏的镇压已经达到了几十年来的最高水平。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “今天的人权情况完全不同”，国际声援西藏运动德国办事处总监 Kai Muller 说，他强调中国当局加强了对藏人的迁徙、宗教信仰、甚至学习语言的权利的控制。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “从生活的方方面面来看，政府干预的范围都在扩大。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    逃离西藏的难民人数减少了97％，这是由于新技术使中国能够在其南部和西部边界建立了一个几乎无法逾越的数字化的和密不透风的边界墙
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    这些技术大致包括什么？在这里看到《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%b5%ae%e7%a9%ba%e5%99%a8%e3%80%81%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab%e3%80%81%e5%b0%84%e9%a2%91%e3%80%81%e6%97%a0%e4%ba%ba%e6%9c%ba-%e5%94%90%e5%ae%81%e8%a1%97%e8%81%9a%e9%9b%86/" href="https://www.iyouport.org/%e6%b5%ae%e7%a9%ba%e5%99%a8%e3%80%81%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab%e3%80%81%e5%b0%84%e9%a2%91%e3%80%81%e6%97%a0%e4%ba%ba%e6%9c%ba-%e5%94%90%e5%ae%81%e8%a1%97%e8%81%9a%e9%9b%86/" rel="noopener noreferrer" target="_blank">
     浮空器、生物识别、射频、无人机 ……唐宁街聚集大规模监视行业，其关键技术中国几乎全有
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   在西藏以及新疆 — 维吾尔人和其他突厥穆斯林的故乡，这已成为
   <a class="markup--anchor markup--p-anchor" data-href="https://codastory.com/authoritarian-tech/uyghur-women-fighting-china-surveillance/" href="https://codastory.com/authoritarian-tech/uyghur-women-fighting-china-surveillance/" rel="noopener noreferrer" target="_blank">
    广为报道的
    <strong class="markup--strong markup--p-strong">
     技术反乌托邦实践
    </strong>
   </a>
   ，无人机、监控摄像机、和互连的监控系统在边境巡逻，使士兵能够迅速监视并逮捕任何藏族、维吾尔族或哈萨克族人，只要他们被认为试图逃离。
  </p>
  <p class="graf graf--p">
   数字墙也从根本上阻止了绝大多数逃亡的尝试。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    从抗议到边境管制
   </strong>
  </p>
  <p class="graf graf--p">
   中国最早建造数字墙的迹象是在2008年3月，即 北京举办奥运会的五个月之前，当时在西藏各地
   <a class="markup--anchor markup--p-anchor" data-href="https://globalvoices.org/specialcoverage/2008-special-coverage/tibet-protests-2008/" href="https://globalvoices.org/specialcoverage/2008-special-coverage/tibet-protests-2008/" rel="noopener noreferrer" target="_blank">
    爆发了
   </a>
   广泛的反中抗议活动。
  </p>
  <p class="graf graf--p">
   抗议者的努力在全球成为头条新闻，随后，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2008/04/07/world/europe/07iht-07torch4.11736987.html" href="https://www.nytimes.com/2008/04/07/world/europe/07iht-07torch4.11736987.html" rel="noopener noreferrer" target="_blank">
    全球舆论抗议
   </a>
   藏族、维吾尔族和其他少数民族的人权受到压迫。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “那场起义本身是造成如今我们看到的中国当局发生的这种[维稳]偏执的原因，” 驻印度的西藏人权与民主促进中心主任 Tsering Tsomo 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他们[共产党政府]真的想要阻止任何人进出此地。”
  </p>
  <p class="graf graf--p">
   那一年，整个西藏自治区宣布了戒严，
   <strong class="markup--strong markup--p-strong">
    从那以后，除了国家的宣传喉舌可以旅行之外，当局还阻止外国记者和民间社会团体造访西藏
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这使得外界很难理解该地区的数字监视范围。外国记者甚至可以访问新疆，但无法访问西藏。
  </p>
  <p class="graf graf--p">
   有几条
   <a class="markup--anchor markup--p-anchor" data-href="http://theasiadialogue.com/2018/02/14/chinas-securitization-drive-in-tibet-and-xinjiang/" href="http://theasiadialogue.com/2018/02/14/chinas-securitization-drive-in-tibet-and-xinjiang/" rel="noopener noreferrer" target="_blank">
    关于西藏的数据
   </a>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    甚至包括新疆在内，没有任何地区的人均维稳支出超过西藏
   </strong>
   。
  </p>
  <p class="graf graf--p">
   西藏在2017年的人均维稳费用支出为 3,137 元人民币，比新疆的 2,417 元还高，远高于全中国平均水平的 763 元。
  </p>
  <p class="graf graf--p">
   西藏政策研究所研究员 Tenzin Tsultrim 说：“由此可见，整个西藏的维稳体系有多广泛，超过北京和新疆。”
  </p>
  <p class="graf graf--p">
   Tsultrim 还担心以强制性体检为幌子从藏人那里
   <strong class="markup--strong markup--p-strong">
    收集DNA样本
   </strong>
   的驱动力。“如果当局拥有人们的 DNA，追踪一个人会非常容易。这是监视的致命组合。”
  </p>

  <p class="graf graf--p">
   当局对普通藏人的控制瞄准他们的家。
  </p>
  <p class="graf graf--p">
   中国对2008年抗议活动的回应之一是大幅增加了西藏维持治安工作的规模和范围。这次改革是由党委书记陈全国领导的，在他成功地压制了西藏之后继续担任新疆共产党书记，在那里，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.albawaba.com/news/chen-quanguo-man-who-silenced-tibet-perfecting-police-state-xinjiang-china-1181388" href="https://www.albawaba.com/news/chen-quanguo-man-who-silenced-tibet-perfecting-police-state-xinjiang-china-1181388" rel="noopener noreferrer" target="_blank">
    他被认为
   </a>
   是
   <strong class="markup--strong markup--p-strong">
    高科技监视系统的总设计师
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在西藏，当局将他们称为“便利”派出所的小型警察局网络扩展到了城市地区的每一个角落，平均每200–300米一个，以快速应对任何所谓的威胁。
  </p>
  <p class="graf graf--p">
   拉萨等城市现在拥有完备的监视摄像机网络、和众多检查站，使当局可以跟踪所有进出的人员。
  </p>
  <p class="graf graf--p">
   最近，拥有面部识别技术的警察局和带有面部识别功能的监控摄像机已经普及到了农村，尤其是沿边境的村庄。
  </p>
  <p class="graf graf--p">
   Tsultrim 说：“仅在拉萨，就有600多个那种所谓的便利派出所”。他说，越来越多的维稳人员被派往农村地区。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “近年来，有超过21,000名维稳官员被派往[西藏]附近的农村。” 这是对至少已经有12,000名主要在城市地区工作的维稳人员的补充。
  </p>
  <p class="graf graf--p">
   实际上自2008年以来，藏人甚至难以在中国国内旅行。现在，藏人如果需要访问拉萨等城市就必须事先获得许可，据消息来源说，城市周围以及邻近地区和省份之间的边界存在许多检查站。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    监视和面部识别技术还可以识别藏人何时离开家乡，进一步阻碍了他们到达边境的能力。
   </strong>
  </p>

   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%ae%a9%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e5%a4%b1%e6%95%88%e7%9a%84%e6%80%9d%e8%80%83%e6%96%b9%e5%bc%8f/" href="https://www.iyouport.org/%e8%ae%a9%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e5%a4%b1%e6%95%88%e7%9a%84%e6%80%9d%e8%80%83%e6%96%b9%e5%bc%8f/" rel="noopener noreferrer" target="_blank">
     让面部识别失效的思考方式
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “就像当局在整个地区实施了一系列过滤器，过滤掉了任何他们不信任的人，并确保将那些人留在原地，” 驻香港的人权观察组织研究员 Maya Wang 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “从某种意义上说，边界是最后一道防线。但是，如果有了这种过滤器，那么首先就能实现最大程度上减少能够到达边界的人数。”
  </p>
  <p class="graf graf--p">
   最终能到达边境的藏人还面临着更多的挑战。
  </p>
  <p class="graf graf--p">
   政府报纸《中国日报》
   <a class="markup--anchor markup--p-anchor" data-href="http://www.chinadaily.com.cn/" href="http://www.chinadaily.com.cn/" rel="noopener noreferrer" target="_blank">
    报道了
   </a>
   中国制造的具有先进雷达、声学监测设备、和无人机组合的边境监控系统的部署情况。
  </p>
  <p class="graf graf--p">
   报道引述一位发言人说，
   <strong class="markup--strong markup--p-strong">
    任何试图越境的人都会被该监控系统检测到，该系统会自动通知士兵。
   </strong>
  </p>
  <p class="graf graf--p">
   中国还部署了GGJ-2无人机，可以在西藏和新疆边界高空运行长达20个小时，旨在监视全境、并在必要时在无法驻扎部队的地区进行进攻活动。
  </p>
  <p class="graf graf--p">
   过去就有过维稳官员对试图逃跑的藏人使用武力的情况。印度西藏政策研究所的研究员 Tenzin Dahla 认为，
   <strong class="markup--strong markup--p-strong">
    卫星也是日益自动化的数字墙的一部分
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%8d%ab%e6%98%9f%e5%b7%b2%e7%bb%8f%e5%bc%80%e5%a7%8b%e8%a7%82%e5%af%9f%e4%bd%a0%e7%9a%84%e6%af%8f%e4%b8%80%e4%b8%aa%e5%8a%a8%e4%bd%9c/" href="https://www.iyouport.org/%e5%8d%ab%e6%98%9f%e5%b7%b2%e7%bb%8f%e5%bc%80%e5%a7%8b%e8%a7%82%e5%af%9f%e4%bd%a0%e7%9a%84%e6%af%8f%e4%b8%80%e4%b8%aa%e5%8a%a8%e4%bd%9c/" rel="noopener noreferrer" target="_blank">
     卫星已经开始观察你的每一个动作
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   Dalha 说：“我最近从西藏境内的藏族人那里听说，他们正在边界地区附近使用
   <strong class="markup--strong markup--p-strong">
    卫星预警系统
   </strong>
   。
  </p>
  <p class="graf graf--p">
   他担心中国打算在西藏启动华为5G网络，这将使部署传感器和快速传输大量数据进行实时监控分析变得更加容易。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    延伸《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%b8%ad%e5%9b%bd5g%ef%bc%9a%e5%a5%bd%e8%8e%b1%e5%9d%9e%e5%a4%a7%e7%89%87%e7%9a%84%e6%bb%91%e7%a8%bd%e5%9c%ba%e6%99%af/" href="https://www.iyouport.org/%e4%b8%ad%e5%9b%bd5g%ef%bc%9a%e5%a5%bd%e8%8e%b1%e5%9d%9e%e5%a4%a7%e7%89%87%e7%9a%84%e6%bb%91%e7%a8%bd%e5%9c%ba%e6%99%af/" rel="noopener noreferrer" target="_blank">
     中国5G：好莱坞大片的滑稽场景
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/5g%e7%bd%91%e7%bb%9c%e7%9a%84%e5%8f%af%e6%80%95%e6%bd%9c%e5%8a%9b/" href="https://www.iyouport.org/5g%e7%bd%91%e7%bb%9c%e7%9a%84%e5%8f%af%e6%80%95%e6%bd%9c%e5%8a%9b/" rel="noopener noreferrer" target="_blank">

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%ba%e5%b7%a5%e6%99%ba%e8%83%bd%e6%96%b0%e5%86%b7%e6%88%98%e2%80%8a-%e2%80%8a-ai%e5%92%8c5g%e5%a6%82%e4%bd%95%e5%8f%98%e6%88%90%e4%b8%80%e5%9c%ba%e5%99%a9%e6%a2%a6/" href="https://www.iyouport.org/%e4%ba%ba%e5%b7%a5%e6%99%ba%e8%83%bd%e6%96%b0%e5%86%b7%e6%88%98%e2%80%8a-%e2%80%8a-ai%e5%92%8c5g%e5%a6%82%e4%bd%95%e5%8f%98%e6%88%90%e4%b8%80%e5%9c%ba%e5%99%a9%e6%a2%a6/" rel="noopener noreferrer" target="_blank">

   参与西藏便利数字监控系统的公司包括中国北方工业公司、深圳中集智能CIMC、在西藏大学设有实时跟踪中心的 Wiseweb 技术公司、还有 Intellifusion 和 IflyTek。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     iFlytek 最近也与美国麻省理工学院建立了多年的研究合作伙伴关系。这些“伙伴关系”起到了明确的破坏人权的作用
    </strong>
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%b8%80%e8%87%82%e4%b9%8b%e5%8e%89%ef%bc%9a%e7%be%8e%e5%9b%bd%e4%bc%81%e4%b8%9a%e6%ad%a3%e5%9c%a8%e5%b8%ae%e5%8a%a9%e5%bb%ba%e7%ab%8b%e4%b8%ad%e5%9b%bd%e7%9a%84%e5%a5%a5%e5%a8%81%e5%b0%94%e5%9b%bd/" href="https://www.iyouport.org/%e4%b8%80%e8%87%82%e4%b9%8b%e5%8e%89%ef%bc%9a%e7%be%8e%e5%9b%bd%e4%bc%81%e4%b8%9a%e6%ad%a3%e5%9c%a8%e5%b8%ae%e5%8a%a9%e5%bb%ba%e7%ab%8b%e4%b8%ad%e5%9b%bd%e7%9a%84%e5%a5%a5%e5%a8%81%e5%b0%94%e5%9b%bd/" rel="noopener noreferrer" target="_blank">

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%bb%b4%e6%97%8f%e5%a6%87%e5%a5%b3%e5%af%b9%e6%88%98%e4%b8%ad%e5%9b%bd%e7%9a%84%e7%9b%91%e8%a7%86%e5%9b%bd%e5%ae%b6videos/" href="https://www.iyouport.org/%e7%bb%b4%e6%97%8f%e5%a6%87%e5%a5%b3%e5%af%b9%e6%88%98%e4%b8%ad%e5%9b%bd%e7%9a%84%e7%9b%91%e8%a7%86%e5%9b%bd%e5%ae%b6videos/" rel="noopener noreferrer" target="_blank">
     维族妇女对战中国的监视国家[videos]
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    防火墙
   </strong>
  </p>
  <p class="graf graf--p">
   伴随着对藏人行动自由的严格限制，当局对数字信息流也进行了严格的控制。
  </p>
  <p class="graf graf--p">
   Citizen Lab 是加拿大多伦多大学的跨学科实验室，该实验室分析了在中国主要社交媒体和聊天平台微信上共享的内容，
   <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.ca/2017/01/tibetans-blocked-from-kalachakra-at-borders-and-on-wechat/" href="https://citizenlab.ca/2017/01/tibetans-blocked-from-kalachakra-at-borders-and-on-wechat/" rel="noopener noreferrer" target="_blank">
    研究发现
   </a>
   当局会定期过滤西藏境内和境外的某些关键字和词语。
  </p>
  <p class="graf graf--p">
   这些数字控制可能很广泛。
  </p>
  <p class="graf graf--p">
   去年三月，
   <a class="markup--anchor markup--p-anchor" data-href="https://highpeakspureearth.com/2018/guest-post-the-silence-of-the-state-the-jokhang-fire-and-the-response-of-the-chinese-government-by-francoise-robin/" href="https://highpeakspureearth.com/2018/guest-post-the-silence-of-the-state-the-jokhang-fire-and-the-response-of-the-chinese-government-by-francoise-robin/" rel="noopener noreferrer" target="_blank">
    拉萨大昭寺起火
   </a>
   ；大昭寺被认为是藏传佛教的圣地，人们很担心，不可替代的文化遗产遭到破坏。然而，在那之后的几个月中，没有任何关于火灾情况的进一步消息传出。
  </p>
  <p class="graf graf--p">
   中国审查员阻止了进一步发布视频、沉默了社交媒体，并禁止人们进入寺庙。
  </p>
  <p class="graf graf--p">
   同样，虽然很容易找到每月访问拉萨并拍照的成千上万中国游客发布的照片，这些照片经常发布到诸如微博或微信之类的中国社交网络上，但是，听取藏人的声音变得越来越困难。
  </p>
  <p class="graf graf--p">
   在仅仅十年的时间里，
   <strong class="markup--strong markup--p-strong">
    西藏在物理上和数字上都与世界隔绝了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这种数字种族隔离的最明显的迹象是沿着严格控制的边界。
  </p>
  <p class="graf graf--p">
   2017年末，尼泊尔和西藏之间的 Kerung 边境过境点因2015年地震造成的破坏而关闭了两年多后
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/travel/2017/sep/16/tibet-nepal-border-road-reopens-after-earthquake-kathmandu-lhasa" href="https://www.theguardian.com/travel/2017/sep/16/tibet-nepal-border-road-reopens-after-earthquake-kathmandu-lhasa" rel="noopener noreferrer" target="_blank">
    重新开放
   </a>
   。
  </p>
  <p class="graf graf--p">
   在边界保持开放的同时，藏人很少过境；相反，人流主要由进入尼泊尔的中国旅客和商人组成。
  </p>
  <p class="graf graf--p">
   Dalha 说：“ 2008年以前，有很多藏人从西藏逃到尼泊尔，但是，现在是中国人在尼泊尔蓬勃发展。”
  </p>
  <p class="graf graf--p">
   在最近对该地区的访问中，他指出，那里许多标语是中文的，而且有很多尼泊尔店主会说基本的中文。
  </p>
  <p class="graf graf--p">
   曾经是将西藏与南亚连接起来的过境点现在将中国与尼泊尔相连。
  </p>
  <p class="graf graf--p">
   印度达兰萨拉（Dharamshala）的局势已经变得严峻。
  </p>
  <p class="graf graf--p">
   1959年，印度贾瓦哈拉尔·尼赫鲁（Jawaharlal Nehru）政府向达赖喇嘛和流亡的西藏政府提供了避难所。曾经一度，包括儿童在内的难民不断涌入，使社区充满活力，并允许文化遗产（语言、宗教、歌舞）得到转移和保存，而这些在中国已经越来越多地被禁止。
  </p>
  <p class="graf graf--p">
   但是，今天，有500张床位的藏人接待中心
   <a class="markup--anchor markup--p-anchor" data-href="https://www.voanews.com/a/tibet-immigration/4463008.html" href="https://www.voanews.com/a/tibet-immigration/4463008.html" rel="noopener noreferrer" target="_blank">
    几乎空着
   </a>
   。每年学校能接收的孩子越来越少。
  </p>
  <p class="graf graf--p">
   随着越境藏人人数的减少和审查制度的日益强悍，两个社区之间的联系几乎被切断。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这样做的结果是，今天，大多数居住在西藏的藏人被置于与外界隔绝的茧中”，华盛顿特区国际声援西藏运动副主席 Bhuchung K.Tsering 说。⚪️
  </p>

    China’s Digital Wall Around Tibet
