  <p class="graf graf--p">
   当下的思想自由处于关键的十字路口。
  </p>
  <p class="graf graf--p">
   数字技术和心理学研究上的进步可以用来促进自由思想，它们可以保护内心世界，减少心理偏见，并创造新的思想空间；但是，事实上正相反，
   <strong class="markup--strong markup--p-strong">
    国家和寡头公司正在将这些进步锻造成限制我们思维的武器。
   </strong>
  </p>
  <p class="graf graf--p">
   失去思想自由将是失去某些人类特性的开端。
  </p>
  <p class="graf graf--p">
   人类和动物都有基本情感，但只有人类可以做退后一步，问自己：“我想生气吗？”、“我想成为那样的人吗？”、“我能变得更好吗？” ……
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    人类可以判断自己内心冒出的感觉和欲望是否与自己的目标、价值观和理想相一致。只有能做到这点，才能采取正确的行动
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是，有时也可以得出这样的结论：一些突然出现在你脑海中的想法是一种推动力，
   <a class="markup--anchor markup--p-anchor" data-href="https://link.springer.com/chapter/10.1007%2F978-1-4612-3950-5_6" href="https://link.springer.com/chapter/10.1007%2F978-1-4612-3950-5_6" rel="noopener noreferrer" target="_blank">
    但那不是你自己的力量
   </a>
   。比如你正坐下来准备开始工作，“点开 Facebook！” 在你脑海中闪烁。这个想法是来自您还是来自马克·扎克伯格？
  </p>
  <p class="graf graf--p">
   只有实现思想自由才能实现人格

    尊严
   </a>

    成为一个人
   </a>
   的一部分。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">

    </strong>
   </span>

  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.ohchr.org/en/professionalinterest/pages/ccpr.aspx" href="https://www.ohchr.org/en/professionalinterest/pages/ccpr.aspx" rel="noopener noreferrer" target="_blank">

   </a>

  </p>
  <p class="graf graf--p">
   但是，这项权利几乎被完全忽略了。
   <a class="markup--anchor markup--p-anchor" data-href="https://heinonline.org/HOL/LandingPage?handle=hein.journals/plr3&amp;div=12&amp;id=&amp;page=" href="https://heinonline.org/HOL/LandingPage?handle=hein.journals/plr3&amp;div=12&amp;id=&amp;page=" rel="noopener noreferrer" target="_blank">
    很少
   </a>
   在法庭上援引它。需要弄清楚我们希望这项权利意味着什么，以便我们可以用它来保护自己。
  </p>
  <p class="graf graf--p">
   我们应该使用它来捍卫精神隐私。否则，
   <strong class="markup--strong markup--p-strong">
    一致性的压力
   </strong>
   将阻碍我们自由发挥思想和寻求真理。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%bc%82%e8%ae%ae%e7%9a%84%e6%b6%88%e4%ba%a1%ef%bc%9a%e4%b8%ba%e4%bb%80%e4%b9%88%e7%bd%91%e7%bb%9c%e5%8f%98%e5%be%97%e5%90%8c%e8%b4%a8%e5%8c%96/" href="https://www.iyouport.org/%e5%bc%82%e8%ae%ae%e7%9a%84%e6%b6%88%e4%ba%a1%ef%bc%9a%e4%b8%ba%e4%bb%80%e4%b9%88%e7%bd%91%e7%bb%9c%e5%8f%98%e5%be%97%e5%90%8c%e8%b4%a8%e5%8c%96/" rel="noopener noreferrer" target="_blank">
     异议的消亡：为什么网络变得同质化
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   我们应该使用它来防止我们的思想被
   <strong class="markup--strong markup--p-strong">
    社交工程伎俩或威胁性惩罚
   </strong>
   所操纵。
  </p>
  <p class="graf graf--p">
   应该使用它来保护所有形式的思想。
  </p>
  <p class="graf graf--p">
   思想已经不仅发生在人的脑海中。有时人们还会通过写作或 Google 搜索来思考。如果我们将这些活动视为 “思想活动”，则它们应符合思想自由权下的绝对隐私权 ——
   <strong class="markup--strong markup--p-strong">
    你的搜索历史记录不应该为任何人所了解，你写下的每一个字都不应该接受任何审查监视
   </strong>
   。
  </p>
  <p class="graf graf--p">
   最后，我们应该利用这项权利，要求建立允许所有人自由思考的社会。这就是心理学可以提供帮助的地方。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-11051 jetpack-lazy-image" data-lazy-sizes="(max-width: 1050px) 100vw, 1050px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/操纵.jpeg?resize=1050%2C700&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/操纵.jpeg?w=1050&amp;ssl=1 1050w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/操纵.jpeg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/操纵.jpeg?resize=1024%2C683&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/操纵.jpeg?resize=768%2C512&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/操纵.jpeg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="700" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/操纵.jpeg?resize=1050%2C700&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1050"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11051" data-recalc-dims="1" height="700" sizes="(max-width: 1050px) 100vw, 1050px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/操纵.jpeg?resize=1050%2C700&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/操纵.jpeg?w=1050&amp;ssl=1 1050w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/操纵.jpeg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/操纵.jpeg?resize=1024%2C683&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/操纵.jpeg?resize=768%2C512&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/操纵.jpeg?resize=272%2C182&amp;ssl=1 272w" width="1050"/>
   </noscript>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     防止操纵
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   更好地理解自己的思想可以帮助保护我们免受他人的操纵。
  </p>
  <p class="graf graf--p">
   例如，心理学家丹尼尔·卡尼曼（Daniel Kahneman）
   <a class="markup--anchor markup--p-anchor" data-href="https://www.penguin.co.uk/books/563/56314/thinking--fast-and-slow/9780141033570.html" href="https://www.penguin.co.uk/books/563/56314/thinking--fast-and-slow/9780141033570.html" rel="noopener noreferrer" target="_blank">
    区分了
   </a>
   “经验法则” 和 “理性法则” 两种思维。
  </p>
  <p class="graf graf--p">
   经验法则思维涉及轻松而古老的思维过程，使人能够快速做出决定。这种速度的代价可能是出错。相反，理性法则思维是一个缓慢的、有意识地控制的过程，通常是基于语言的。它需要更长的时间，但是可以更准确。
  </p>
  <p class="graf graf--p">
   这表明在我们的思维中制造减速带可以帮助改善决策。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    毫不犹豫地点击宣传性内容的人是无法行使思想自由的。行为之前需要首先确定那是来自自己的欲望还是木偶大师的欲望
   </strong>
   。
  </p>
  <p class="graf graf--p">
   人们还必须将环境更改为
   <strong class="markup--strong markup--p-strong">
    支持自治的环境
   </strong>
   。
   <a class="markup--anchor markup--p-anchor" data-href="http://selfdeterminationtheory.org/SDT/documents/2008_OliverMarklandetal_ME.pdf" href="http://selfdeterminationtheory.org/SDT/documents/2008_OliverMarklandetal_ME.pdf" rel="noopener noreferrer" target="_blank">
    这样的环境
   </a>
   可以使每个人能够为自己的行动创造理由，最大限度地减少奖惩等外部控制，并鼓励选择、参与和共同决策。
  </p>
  <p class="graf graf--p">
   技术可以帮助创造这样的环境。但是执行此任务是谁的责任呢？
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     采取行动
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   必须帮助公民从小就掌握科学的思维方式。必须构建可以促进自由思考的社会。而政府有义务制止那些侵犯思想自由权的人，包括寡头公司。
  </p>
  <p class="graf graf--p">
   公司必须发挥作用。他们应将思想自由声明为一项政策承诺。他们应该对自己的活动如何损害思想自由进行尽职调查。他们必须解释他们是使用什么方法来操纵大众行为心理的。
  </p>
  <p class="graf graf--p">
   人民必须教育自己，必须促进和支持自由思想的价值观。必须谴责这种将人类行为作为提取资源以盈利的
   <a class="markup--anchor markup--p-anchor" data-href="https://t.me/iyouport/6559" href="https://t.me/iyouport/6559" rel="noopener noreferrer" target="_blank">
    监视资本主义体制
   </a>
   。
   <strong class="markup--strong markup--p-strong">
    必需能做到避免社会性变成人类最大的劣势之一
   </strong>
   。
  </p>
  <p class="graf graf--p">
   否则就不得不用脚和钱包投票以反对那些侵犯我们思想自由的人。
  </p>
  <p class="graf graf--p">
   所有这一切都是在假定所有人都想要思想自由的基础上。
  </p>
  <p class="graf graf--p">
   但是，是不是所有人都真的想要呢？实际上，仍然有许多人
   <a class="markup--anchor markup--p-anchor" data-href="https://science.sciencemag.org/content/345/6192/75" href="https://science.sciencemag.org/content/345/6192/75" rel="noopener noreferrer" target="_blank">
    宁愿被他人影响
   </a>
   ，也不愿安静地思考。
  </p>
  <p class="graf graf--p">
   你身边是否有许多人仍然希望政府和寡头公司代替自己的思考、提供预测和推动，以便他们可以简单地遵循？
  </p>
  <p class="graf graf--p">
   我们究竟需要多少思想自由？准备为此付出什么？需要首先回答这个问题。
  </p>
  <p class="graf graf--p">
   简而言之，我们仍然想成为人类吗？还是这种标志性能力（自由思考）所带来的痛苦、努力和责任已经变得太大，以至于无法承受？
  </p>
  <p class="graf graf--p">
   如果是后者，那一切将无从解决。⚪️
  </p>
  <p class="graf graf--p">
   作者是都柏林圣三一学院临床心理学和神经心理学副教授 Simon McCarthy-Jones
  </p>
