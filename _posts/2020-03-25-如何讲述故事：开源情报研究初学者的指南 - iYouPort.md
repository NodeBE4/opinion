  <p class="graf graf--p">
   许多公民开源调查人员都在依靠记者将调查报告转换为叙事的形式，然后将其传播给更广泛的受众。
  </p>
  <p class="graf graf--p">
   然而，今天的新闻业正处于危机之中。
  </p>
  <p class="graf graf--p">
   不论是美国还是中国，媒体都面临着自大萧条以来最糟糕的裁员 —— 而
   <strong class="markup--strong markup--p-strong">
    这正是公民调查人员填补传统媒体机构逐渐退位留下的空白的重要原因之一
   </strong>
   。
  </p>
  <p class="graf graf--p">
   尤其是在中国那样的新闻环境中，作为公民调查人员您恐怕难以与任何一家职业媒体合作有意义的调查报告。
  </p>
  <p class="graf graf--p">
   所以，公民调查人员必需能自己掌握讲述故事的技巧。
  </p>
  <p class="graf graf--p">
   这听起来似乎令人生畏。不过，不要害怕。通过学习和利用一套非常特殊的技能，您也可以亲自使您的研究覆盖更广泛的受众 —— 甚至可以改变世界（相信我，这个世界已经发生了奇怪的事）。
  </p>
  <p class="graf graf--p">
   那么如何做到这点？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    请注意，本文不仅对公民调查人员是有意义的，对NGO组织和活动家同样很有意义，因为社会是故事构成的，擅长讲述故事的人将获得更多的注目。
   </strong>
  </p>
  <p class="graf graf--p">
   有效的故事将能够把人们带入一个感知度更强烈的世界。讲故事是社会动员中激活人们热情的最有效方法之一。（这也是为什么专制国家努力阻止制独立电影和纪录片的发展，同时加强政权引导的叙事框架）
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     接受您的弱项是讲述成功故事的第一步
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   许多公民调查人员认为，有效的叙事是小说作家才需要培养的技能。 “我们处理事实，我们的工作与众不同！”
  </p>
  <p class="graf graf--p">
   好吧，您的作品的确“是”不同的，但是，类似的叙述原理也完全适用于它。
  </p>
  <p class="graf graf--p">
   因为，如果您不知道如何正确地呈现事实，就可能会混淆事实，您的调查工作就会因此被大打折扣。
  </p>
  <p class="graf graf--p">
   开源研究始于线程。随着各个线程的分解，使叙述尽可能清晰，信息丰富和可读性高是确保您的工作产生影响的唯一方法。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     相信自己
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   很多人的认识就像您需要MFA或新闻学硕士才能理解和创造叙事，无论是非虚构还是虚构。
  </p>
  <p class="graf graf--p">
   但事实并非如此，这与学历无关。您越早意识到这一点，您的工作就会越轻松。
  </p>
  <p class="graf graf--p">
   作为公民调查人员，您应该认识到讲故事是一项公共利益，您的工作正在满足公民知情权、弥补职业媒体不能提供的民主价值，这就是为什么您的工作很重要。
  </p>
  <p class="graf graf--p">
   相信自己，相信您的调查工作对社会意义，您可以很好地将调查报告理解为一种可以学习和利用的手工艺品。
  </p>
  <p class="graf graf--p">
   本文将一份曾经的两份开源情报调查为例《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%b7%b1%e5%b1%82%e6%94%bf%e6%b2%bb%ef%bc%9afacebook%e9%87%91%e5%ad%97%e5%a1%94%e7%9a%84%e4%ba%ba%e5%bd%a2%e8%82%8c%e7%90%86-%e4%bb%85%e4%bb%85%e4%be%9d%e9%9d%a0%e5%bc%80%e6%ba%90/" href="https://www.iyouport.org/%e6%b7%b1%e5%b1%82%e6%94%bf%e6%b2%bb%ef%bc%9afacebook%e9%87%91%e5%ad%97%e5%a1%94%e7%9a%84%e4%ba%ba%e5%bd%a2%e8%82%8c%e7%90%86-%e4%bb%85%e4%bb%85%e4%be%9d%e9%9d%a0%e5%bc%80%e6%ba%90/" rel="noopener noreferrer" target="_blank">
    深层政治：Facebook金字塔的人形肌理
   </a>
   》、以及《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e6%8f%ad%e7%a4%ba%e8%82%a2%e8%a7%a3%e5%8d%8e%e7%9b%9b%e9%a1%bf%e9%82%ae%e6%8a%a5%e4%b8%93%e6%a0%8f%e4%bd%9c%e5%ae%b6%e7%9a%84%e5%87%b6%e6%89%8b/" href="https://www.iyouport.org/%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e6%8f%ad%e7%a4%ba%e8%82%a2%e8%a7%a3%e5%8d%8e%e7%9b%9b%e9%a1%bf%e9%82%ae%e6%8a%a5%e4%b8%93%e6%a0%8f%e4%bd%9c%e5%ae%b6%e7%9a%84%e5%87%b6%e6%89%8b/" rel="noopener noreferrer" target="_blank">
    揭示肢解华盛顿邮报专栏作家的凶手
   </a>
   》，解释您所需要的步骤。
  </p>
  <p class="graf graf--p">
   但是我们必须说，
   <strong class="markup--strong markup--p-strong">
    对于讲述故事的技巧本身，另一份报告是更加具有参考价值的
   </strong>
   ：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" rel="noopener noreferrer" target="_blank">
    互联网霸主的时间线
   </a>
   》—— 此案例尤其对活动家和公民社会组织有价值；只因为它并非*开源*调查演示，而是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e9%80%8f%e6%98%8e%e5%ba%a6%e9%9d%a9%e5%91%bd/" href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e9%80%8f%e6%98%8e%e5%ba%a6%e9%9d%a9%e5%91%bd/" rel="noopener noreferrer" target="_blank">
    透明度革命
   </a>
   的第二步骤（第一步骤是揭露，）故本文没有引述它。
  </p>
  <p class="graf graf--p">
   <span style="color: #33cccc;">
    <strong class="markup--strong markup--p-strong">
     1、确定目标受众
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   最有可能对您的报告感兴趣的人是谁？您的报告中有哪些方面吸引了众多受众？您首先要确定这点。
  </p>
  <p class="graf graf--p">
   通过问自己这些问题并在头脑中积极“吸引”读者，可以使您的工作更轻松，并激发自己做得更好。
  </p>
  <p class="graf graf--p">
   许多开源情报研究人员在撰写调查报告时考虑的都是其他开源情报领域的同行。在某些情况下可能是完美的 —— 但并非全部。
  </p>
  <p class="graf graf--p">
   如果您认为您的报告是
   <strong class="markup--strong markup--p-strong">
    有社会公益价值
   </strong>
   的 —— 比如挖掘水军幕后的黑手、比如揭露政府的谎言，那么您应该考虑更广泛的受众的阅读角度。
  </p>
  <p class="graf graf--p">
   例如，《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%b7%b1%e5%b1%82%e6%94%bf%e6%b2%bb%ef%bc%9afacebook%e9%87%91%e5%ad%97%e5%a1%94%e7%9a%84%e4%ba%ba%e5%bd%a2%e8%82%8c%e7%90%86-%e4%bb%85%e4%bb%85%e4%be%9d%e9%9d%a0%e5%bc%80%e6%ba%90/" href="https://www.iyouport.org/%e6%b7%b1%e5%b1%82%e6%94%bf%e6%b2%bb%ef%bc%9afacebook%e9%87%91%e5%ad%97%e5%a1%94%e7%9a%84%e4%ba%ba%e5%bd%a2%e8%82%8c%e7%90%86-%e4%bb%85%e4%bb%85%e4%be%9d%e9%9d%a0%e5%bc%80%e6%ba%90/" rel="noopener noreferrer" target="_blank">
    深层政治
   </a>
   》报告，与以下方面直接相关：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    监视资本主义寡头的研究者
   </li>
   <li class="graf graf--li">
    反托拉斯立法者
   </li>
   <li class="graf graf--li">
    商业领域的观察者
   </li>
   <li class="graf graf--li">
    政治领域的观察者
   </li>
  </ul>
  <p class="graf graf--p">
   但除此之外，
   <strong class="markup--strong markup--p-strong">
    该报告还与所有反社交媒体监视的活动家、劳工运动支持者、以及在 deleteFacebook 运动中感觉困惑的所有社交媒体用户有关
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这就是为什么调查报告会首先向读者介绍一些相关背景知识 —— 简单说，您需要陈述您开启这份调查的背景。
  </p>
  <p class="graf graf--p">
   <span style="color: #33cccc;">
    <strong class="markup--strong markup--p-strong">
     2、计划叙事
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   您小学时就已经学到：一个故事应该分为开始、中间和结尾。就您的调查报告而言，通常会有一个引言，一个中间部分来展示您的数据分析和资料来源、以及您的结论。
  </p>
  <p class="graf graf--p">
   这听起来很容易，但是很多人会在此遇到一些陷阱，比如下面这样：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     简介不足
    </strong>
   </li>
  </ul>
  <p class="graf graf--p">
   当我说您的介绍不符合您的材料充分性时，可能意味着不同的意思。例如：
  </p>
  <p class="graf graf--p">
   *、导语中缺乏足够的背景信息；
  </p>
  <p class="graf graf--p">
   **、简介内容如此之多，以至于拖累了读者并淹没了细节；
  </p>
  <p class="graf graf--p">
   ***、遗漏了那些将决定人们为什么需要继续阅读的关键细节（尤其是对于试图分析特定事件的背景信息的报告来说，例如 枪击事件、暗杀事件、或特定激进团体等等） 在这方面《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e6%8f%ad%e7%a4%ba%e8%82%a2%e8%a7%a3%e5%8d%8e%e7%9b%9b%e9%a1%bf%e9%82%ae%e6%8a%a5%e4%b8%93%e6%a0%8f%e4%bd%9c%e5%ae%b6%e7%9a%84%e5%87%b6%e6%89%8b/" href="https://www.iyouport.org/%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e6%8f%ad%e7%a4%ba%e8%82%a2%e8%a7%a3%e5%8d%8e%e7%9b%9b%e9%a1%bf%e9%82%ae%e6%8a%a5%e4%b8%93%e6%a0%8f%e4%bd%9c%e5%ae%b6%e7%9a%84%e5%87%b6%e6%89%8b/" rel="noopener noreferrer" target="_blank">
    揭示肢解华盛顿邮报专栏作家的凶手
   </a>
   》报告是个好例子。
  </p>
  <p class="graf graf--p">
   以上是公民调查人员必须注意的基本面。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     中间过程杂乱无章
    </strong>
   </li>
  </ul>
  <p class="graf graf--p">
   就如作家们经常告诫的那样，您需要有 “kill your darlings” 的勇气；也就是说，
   <strong class="markup--strong markup--p-strong">
    摆脱那些您自己最关心的，但不一定适合本次写作的内容
   </strong>
   —— 简单说，收敛，这是一项非常重要的技能。
  </p>
  <p class="graf graf--p">
   比如关于
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%b7%b1%e5%b1%82%e6%94%bf%e6%b2%bb%ef%bc%9afacebook%e9%87%91%e5%ad%97%e5%a1%94%e7%9a%84%e4%ba%ba%e5%bd%a2%e8%82%8c%e7%90%86-%e4%bb%85%e4%bb%85%e4%be%9d%e9%9d%a0%e5%bc%80%e6%ba%90/" href="https://www.iyouport.org/%e6%b7%b1%e5%b1%82%e6%94%bf%e6%b2%bb%ef%bc%9afacebook%e9%87%91%e5%ad%97%e5%a1%94%e7%9a%84%e4%ba%ba%e5%bd%a2%e8%82%8c%e7%90%86-%e4%bb%85%e4%bb%85%e4%be%9d%e9%9d%a0%e5%bc%80%e6%ba%90/" rel="noopener noreferrer" target="_blank">
    Facebook 政治旋转门
   </a>
   的报告，您也许还知道很多关于 Facebook 如何利用专利实施霸权的情报线索、或者该公司为什么丑闻缠身却每一次都能全身而退的原因，没错这些都是优秀的调查线索，但是它们与本报告无关 —— 于是它们应该在此被舍弃。
  </p>
  <p class="graf graf--p">
   公民调查人员还必须学会
   <strong class="markup--strong markup--p-strong">
    对信息进行优先排序
   </strong>
   ，以及如何以最恰当的方式呈现信息。
  </p>
  <p class="graf graf--p">
   💡同时，您必须
   <strong class="markup--strong markup--p-strong">
    学习展示自己的作品
   </strong>
   。如果您从陈述事实转变为得出结论，你就必须告诉读者其中的原因 —— 结论是如何得出的；尤其是在结论不那么明显的情况下。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    陈述事实和分析事实不是一回事
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在 Facebook 政治旋转门报告中组织信息的方法很明显。首先通过确定哪些人员与本次分析密切相关、并细分它；
  </p>
  <p class="graf graf--p">
   然后，在对每个人的描述之后附有其背景和行为/参与的具体证据；
  </p>
  <p class="graf graf--p">
   像这样：
  </p>
  <p class="graf graf--p">
   识别关键人物&gt;&gt;出示证据；
  </p>
  <p class="graf graf--p">
   找出另一个关键人物&gt;&gt;出示证据；
  </p>
  <p class="graf graf--p">
   《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e6%8f%ad%e7%a4%ba%e8%82%a2%e8%a7%a3%e5%8d%8e%e7%9b%9b%e9%a1%bf%e9%82%ae%e6%8a%a5%e4%b8%93%e6%a0%8f%e4%bd%9c%e5%ae%b6%e7%9a%84%e5%87%b6%e6%89%8b/" href="https://www.iyouport.org/%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e6%8f%ad%e7%a4%ba%e8%82%a2%e8%a7%a3%e5%8d%8e%e7%9b%9b%e9%a1%bf%e9%82%ae%e6%8a%a5%e4%b8%93%e6%a0%8f%e4%bd%9c%e5%ae%b6%e7%9a%84%e5%87%b6%e6%89%8b/" rel="noopener noreferrer" target="_blank">
    揭示肢解华盛顿邮报专栏作家的凶手
   </a>
   》报告也一样，这里是一个目标人，围绕他关联并诠释各种线索。
  </p>
  <p class="graf graf--p">
   想象一下，如果没有
   <strong class="markup--strong markup--p-strong">
    对信息进行分组和区分优先级
   </strong>
   ，整体就会变得更加困难和混乱，对吧。
  </p>
  <p class="graf graf--p">
   💡简单说，您的调查陈述需要有层次。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     结论无法将调查过程联系在一起
    </strong>
   </li>
  </ul>
  <p class="graf graf--p">
   这是公民调查人员经常犯的错误。
  </p>
  <p class="graf graf--p">
   很多人认为结论是一种对 “太长没法看” 的解决方案；于是只需重述报告中说的内容，然后点击“保存”按钮即可。这是错误的。
  </p>
  <p class="graf graf--p">
   💡实际上，
   <strong class="markup--strong markup--p-strong">
    结论是很有价值的“修辞不动产”
   </strong>
   ，在这个地方您可以（如果合适的话）强调为什么您的发现很重要；或激发未来的研究人员进行更深入的研究；或呼吁更多事实；或情境化您的发现，等等。都是很好的选择。
  </p>
  <p class="graf graf--p">
   如果您想激发和引导读者的兴趣，那么您就不应该让读者感到您只是想说 “整个事情都结束了”。
  </p>
  <p class="graf graf--p">
   <span style="color: #33cccc;">
    <strong class="markup--strong markup--p-strong">
     3、设置正确的口吻
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   如果您正在从事一个关于用可怕的空袭杀死平民的调查项目，那您必须收起幽默感。
  </p>
  <p class="graf graf--p">
   另一方面，如果您正在调查可笑的互联网人物的阴暗交易，该怎么办？适当地讽刺可以帮助像这样的故事吸引更多的读者，并支持您主题的奇异性质。
  </p>
  <p class="graf graf--p">
   💡使用幽默来记录他们的行为，可以帮助您的读者从叙事中发现更多。
  </p>
  <p class="graf graf--p">
   <span style="color: #33cccc;">
    <strong class="markup--strong markup--p-strong">
     4、学习有效地组织和呈现信息
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   您真的需要 “kill your darlings” 吗？有时，您不必杀死它们，而是让它们为您更高效地工作。
  </p>
  <p class="graf graf--p">
   以下是一些可以用于此目的的常用工具。
  </p>
  <p class="graf graf--p">
   ——
   <strong class="markup--strong markup--p-strong">
    为那些非常喜欢做研究工作的人创建一个较长的完整报告PDF版本
   </strong>
   ，并将其链接发布到更简洁的版本旁边。
  </p>
  <p class="graf graf--p">
   例如对《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e6%8f%ad%e7%a4%ba%e8%82%a2%e8%a7%a3%e5%8d%8e%e7%9b%9b%e9%a1%bf%e9%82%ae%e6%8a%a5%e4%b8%93%e6%a0%8f%e4%bd%9c%e5%ae%b6%e7%9a%84%e5%87%b6%e6%89%8b/" href="https://www.iyouport.org/%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e6%8f%ad%e7%a4%ba%e8%82%a2%e8%a7%a3%e5%8d%8e%e7%9b%9b%e9%a1%bf%e9%82%ae%e6%8a%a5%e4%b8%93%e6%a0%8f%e4%bd%9c%e5%ae%b6%e7%9a%84%e5%87%b6%e6%89%8b/" rel="noopener noreferrer" target="_blank">
    肢解华盛顿邮报专栏作家的凶手
   </a>
   》的调查报告，网站上发布的不是完整版，而更详细的PDF版本的链接需要放在这篇文章的前面。
  </p>
  <p class="graf graf--p">
   请记住，您的许多报告可能会有重叠的受众。有些人会愿意读完整的东西。其他人则不会。还有一些人可能目前想要较短的版本，但是随着您的研究越来越多，他们将想要深入了解研究调查的细节。
  </p>
  <p class="graf graf--p">
   您需要周全考虑尽可能全面的受众面。
  </p>
  <p class="graf graf--p">
   ——
   <strong class="markup--strong markup--p-strong">
    尽可能使用表格，图表和视觉辅助工具
   </strong>
   。您是否正在对在特定城镇发现的军用车辆进行开源调查？是否有17名技术人员、2辆战车、3辆AAV？……将这些数字组织成图表，使您的读者更容易理解正在发生的事。
  </p>
  <p class="graf graf--p">
   当调查所涉人员的各种等级结构很复杂时，图表就更加重要了。
  </p>
  <p class="graf graf--p">
   《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%b7%b1%e5%b1%82%e6%94%bf%e6%b2%bb%ef%bc%9afacebook%e9%87%91%e5%ad%97%e5%a1%94%e7%9a%84%e4%ba%ba%e5%bd%a2%e8%82%8c%e7%90%86-%e4%bb%85%e4%bb%85%e4%be%9d%e9%9d%a0%e5%bc%80%e6%ba%90/" href="https://www.iyouport.org/%e6%b7%b1%e5%b1%82%e6%94%bf%e6%b2%bb%ef%bc%9afacebook%e9%87%91%e5%ad%97%e5%a1%94%e7%9a%84%e4%ba%ba%e5%bd%a2%e8%82%8c%e7%90%86-%e4%bb%85%e4%bb%85%e4%be%9d%e9%9d%a0%e5%bc%80%e6%ba%90/" rel="noopener noreferrer" target="_blank">
    深层政治
   </a>
   》报告是图表呈现的优秀案例 —— 让调查目标之间的关系变得非常清晰直观。
  </p>
  <p class="graf graf--p">
   ——
   <strong class="markup--strong markup--p-strong">
    标准化名称/标题/拼写
   </strong>
   。如果您要处理晦涩的主题，这一点尤其重要。
  </p>
  <p class="graf graf--p">
   比如您的正文中使用了多个 “Mark Zuckerberg/Zuckerberg”，在您的图表中就不应该出现 “Mark” 这样的简化写法，以避免给一无所知的读者带来混乱。
  </p>
  <p class="graf graf--p">
   ——
   <strong class="markup--strong markup--p-strong">
    重新陈述您要描述的个人的姓名和称谓/等级
   </strong>
   。
  </p>
  <p class="graf graf--p">
   您不能要求读者对一个陌生人的名字和身份特征一见如故，如果你的报告超过3页，就必修在第三页之后再次提及调查对象的全名和身份，否则有可能会给读者的理解带来混乱。
  </p>
  <p class="graf graf--p">
   ——
   <strong class="markup--strong markup--p-strong">
    遵守时间顺序规则
   </strong>
   。您会惊讶地发现及时在文本中来回走动是多么诱人。
  </p>
  <p class="graf graf--p">
   例如您要详细说明个人A在第1、2和3天所做的事，然后还必须谈论个人B在第1天所做的事，那么请确保您的日期和时间保持清楚，就像电影中将穿插回忆的部分显示为黑白那样。
  </p>
  <p class="graf graf--p">
   在这点上，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%83%b3%e7%9f%a5%e9%81%93%e8%80%81%e5%a4%a7%e5%93%a5%e6%98%af%e5%a6%82%e4%bd%95%e4%be%b5%e7%8a%af%e4%bd%a0%e7%9a%84%e5%90%97%ef%bc%9f%e6%8c%96%e6%8e%98%e6%9b%b4%e6%b7%b1%e5%b1%82%e7%9a%84%e5%86%85/" href="https://www.iyouport.org/%e6%83%b3%e7%9f%a5%e9%81%93%e8%80%81%e5%a4%a7%e5%93%a5%e6%98%af%e5%a6%82%e4%bd%95%e4%be%b5%e7%8a%af%e4%bd%a0%e7%9a%84%e5%90%97%ef%bc%9f%e6%8c%96%e6%8e%98%e6%9b%b4%e6%b7%b1%e5%b1%82%e7%9a%84%e5%86%85/" rel="noopener noreferrer" target="_blank">
    对 Hacking Team 泄漏邮件的开源情报调查
   </a>
   是一个很好的案例，它不仅在讲故事，还介绍了调查思路，有知识普及的价值。
  </p>
  <p class="graf graf--p">
   ——
   <strong class="markup--strong markup--p-strong">
    不要以为读者以同样的方式看待您的作品
   </strong>
   。您的读者没有走过您刚刚进行的调查的同一旅程。他们将需要被澄清，而您可能不需要。
  </p>
  <p class="graf graf--p">
   💡所以，请以您的读者的角度安排您的文章。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     大声朗读自己的作品
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   这是一个重要的习惯，尤其是在您刚刚接触写作时。
  </p>
  <p class="graf graf--p">
   大声朗读能帮助您更容易发现报告中的错漏。当特定句子或段落没有多大意义或难以改进时，您会更容易发现这点。
  </p>
  <p class="graf graf--p">
   这是您可以采取的简单而宝贵的步骤，可以改善您的写作。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     请记住，仅仅发布是不够的
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   您必须相信自己的工作很重要。您必须关心您的读者 —— 即使您是为非常有针对性的读者而写的！
  </p>
  <p class="graf graf--p">
   您必须以诚信对待工作。而且，
   <strong class="markup--strong markup--p-strong">
    您必须为自己设定一个高标
   </strong>
   准。
  </p>
  <p class="graf graf--p">
   学习讲故事是一个过程。您可能已经学习了很长时间，而且还在不断进步。请记住这里没有完美，只要您有意愿做得更好，下一次肯定会加分。
  </p>
  <p class="graf graf--p">
   本文只是基本指南。我们今后会视情况发布更加深入的指南，“情况”指的是，比如，中国社会出现了开源情报调查的专属社区，意味着人们对公民掌握真相的事业更加关注了。
  </p>
  <p class="graf graf--p">
   好运。⚪️
  </p>
