   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">
    通常
   </span>
   <span class="s2">
    ”
   </span>
   <span class="s1">
    都信任政府吗？）
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    相反，这些高层通常被视为强有力的领导或创新的源泉，可以根除较低级别的地方腐败，包括诈骗者和不道德的邻居。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    彭博社最近的一篇文章通过描述在一个中国城市试行的应用程序揭示了这一悖论，通过这个应用程序，用户可以向当局举报其他公民的
   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">
    可疑行为
   </span>
   <span class="s2">
    ”
   </span>
   <span class="s1">
    。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    彭博社说，
   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">
    在中国以外的地方这种东西会被描绘成一个令人毛骨悚然的数字圆形监狱，这个所谓的社会信用系统网络在中国却被视为产生该国严重缺乏的东西的一种手段：这种东西就是信任。为此，永久监视和失去隐私是一个很小的代价。
   </span>
   <span class="s2">
    ”
   </span>
   <span class="s1">
    （看起来彭博社和该报道真的没有调查过中国社会究竟有多少人对信用评分系统持反对意见）
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    “
   </span>
   <span class="s1">
    在商业方面，西方人倾向于相信他人，直到对方证明自己不值得信任，但在中国却恰恰相反，
   </span>
   <span class="s2">
    ” Brennan
   </span>
   <span class="s1">
    说。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    “
   </span>
   <span class="s1">
    它源于人际关系的深厚传统，朋友会将你介绍给你可以信任的人
   </span>
   <span class="s2">
    ”
   </span>
   <span class="s1">
    。他认为对于中国移动的
   </span>
   <span class="s2">
    5G
   </span>
   <span class="s1">
    广告来说，大多数中国观众都会
   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">
    看到它发挥力量去追捕坏人。与此同时，在西方社会，我们说国家有太多的控制权，面部识别被用于消极目的。这归结为对信任的不同看法。
   </span>
   <span class="s2">
    ”
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    尽管有这种深思熟虑，
   </span>
   <span class="s2">
    Brennan
   </span>
   <span class="s1">
    对做出一揽子陈述犹豫不决是正确的。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    看过中国移动广告的中国观众在中国微博上发表的细致评论并不缺乏。例如，一位评论者想知道，
   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">
    如果坏人使用这种面部识别技术瞄准你怎么办？
   </span>
   <span class="s2">
    ”
   </span>
   <span class="s1">
    ；另一位人士写道，
   </span>
   <span class="s2">
    “5G
   </span>
   <span class="s1">
    正在迎来一个可怕的、赤裸裸的时期，与广大公众的权利相比，个人权利已经变得毫无意义。
   </span>
   <span class="s2">
    ”
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    所有这些听起来都是愤世嫉俗的，另一位网友甚至更沮丧，写道：
   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">
    这项技术正在为我们创造一个肉眼看不见的笼子。
   </span>
   <span class="s2">
    ”
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    （最后这两段引用所暗示的问题是正确的：即
   </span>
   <span class="s1">
    <b>
     在中国最缺乏的并不是准确的认知，而是反抗精神
    </b>
    。但是这一暗示显然与前文中的陈述相悖。于是我们认为这一暗示有可能并非文章作者的主旨，它只是一个引用带来的副效应。非常可惜，这才是唯一正确的对中国社会的描述）
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    **
   </span>
   <span class="s1">
    我们翻译这篇文章的目的不仅仅是反监视，因为这篇文章中出现的问题是主流媒体最常见的问题，即
   </span>
   <span class="s1">
    <b>
     为了作者的预定论点去搜集
    </b>
   </span>
   <span class="s2">
    <b>
     “
    </b>
   </span>
   <span class="s1">
    <b>
     证据
    </b>
   </span>
   <span class="s2">
    <b>
     ”
    </b>
   </span>
   <span class="s1">
    <b>
     支持，仅选取支持性信息、忽略掉否定性信息
    </b>
    ，于是整体呈现片面的偏执。
   </span>
  </p>
  <p class="p3">
   <span class="s4">
    我们在《
    <a href="https://www.iyouport.org/%e4%bd%a0%e7%9f%a5%e9%81%93%e4%bb%80%e4%b9%88%e6%89%8d%e6%98%af%e7%9c%9f%e6%ad%a3%e7%9a%84%e8%b0%83%e6%9f%a5%e5%90%97%ef%bc%9f%e6%89%80%e8%b0%93%e7%9a%84%e8%b5%b7%e5%ba%95%e7%bb%8f%e5%b8%b8/" rel="noopener noreferrer" target="_blank">
     什么是调查、什么是证据
    </a>
    》
   </span>
   <span class="s4">
    的分析中详细介绍过如何判断。
   </span>
  </p>
  <p class="p4">
   <span class="s4">
    将中国社会描述为彻底的古怪、无知和愚蠢并无助于改变棘手的现实状况；
    <b>
     我们希望中国社会表现出积极的真实的反抗精神，而不是被挖苦性描述引发的自我实现预言效应所打败
    </b>
   </span>
   <span class="s2">
    **
   </span>
  </p>
