  <p>
   <img alt="" class="aligncenter size-full wp-image-13009 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/0-13.jpg?resize=1024%2C574&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/0-13.jpg?w=1024&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/0-13.jpg?resize=300%2C168&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/0-13.jpg?resize=768%2C431&amp;ssl=1 768w" data-recalc-dims="1" height="574" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/0-13.jpg?resize=1024%2C574&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>

  </p>
  <p class="graf graf--p">
   一个新的应用程序可让您查看周围的监视摄像头、蓝牙信标和智能扬声器，并帮助您选择退出数据收集。
  </p>
  <p class="graf graf--p">
   随着整个伦敦市的实时面部识别技术等监控水平的不断增强，美国也在亚特兰大的 Hartsfield-Jackson 机场中实施了该技术以及其他生物识别技术，而中国变成了什么样的反乌托邦全世界都知道。
  </p>
  <p class="graf graf--p">
   就如 Amazon Ring 和其他智能设备一样，需要有一个观察监视者的视角。
  </p>
  <p class="graf graf--p">
   老大哥在推动监控技术的使用 —— 如今不论您看向哪里，都是如此。从亚马逊通过其人脸识别技术帮助维稳部门抓人、到国土安全部（DHS）的抓移民运动、再到奥运会 …… 都在使用这种技术。
  </p>
  <p class="graf graf--p">
   甚至零售业都在推动这项技术的普及，作为一种防盗机制，FaceFirst 的面部识别软件在数千家商店中引入，以构建可疑行窃者的数据库。
  </p>
  <p class="graf graf--p">
   不论你走入哪个机场，都有那种东西对你进行扫描。美国也是如此，根据 Nextgov 的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.dhs.gov/sites/default/files/publications/19_0417_fy18-entry-and-exit-overstay-report.pdfhttps://www.dhs.gov/sites/default/files/publications/19_0417_fy18-entry-and-exit-overstay-report.pdf" href="https://www.dhs.gov/sites/default/files/publications/19_0417_fy18-entry-and-exit-overstay-report.pdfhttps://www.dhs.gov/sites/default/files/publications/19_0417_fy18-entry-and-exit-overstay-report.pdf" rel="noopener noreferrer" target="_blank">
    消息
   </a>
   ，CBP 预计将扩大该计划的覆盖范围，以在2021年之前覆盖超过97％的旅客。
  </p>
  <p class="graf graf--p">
   这款专为 iOS 和 Android 开发的应用程序可让您知道您周围正在使用哪些物联网（IoT）技术、以及它们正在收集哪些数据。
  </p>
  <p class="graf graf--p">
   由卡耐基梅隆大学的研究人员创建的该应用程序及其基础结构旨在为您提供对跟踪您的活动和信息的设备的更多控制。
  </p>
  <p class="graf graf--p">
   借助该应用程序，您将能够看到诸如带有面部识别技术的公共监视摄像头、跟踪您在商场中位置的蓝牙信标、以及邻居的智能门铃、或智能扬声器等间谍设备。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%9c%a8%e5%95%86%e5%ba%97%e9%87%8c%ef%bc%8c%e7%a7%98%e5%af%86%e7%9b%91%e8%a7%86%e5%a6%82%e4%bd%95%e8%b7%9f%e8%b8%aa%e4%bd%a0%e7%9a%84%e6%af%8f%e4%b8%80%e6%ad%a5%e8%a1%8c%e5%8a%a8%ef%bc%9f/https://www.iyouport.org/%e5%9c%a8%e5%95%86%e5%ba%97%e9%87%8c%ef%bc%8c%e7%a7%98%e5%af%86%e7%9b%91%e8%a7%86%e5%a6%82%e4%bd%95%e8%b7%9f%e8%b8%aa%e4%bd%a0%e7%9a%84%e6%af%8f%e4%b8%80%e6%ad%a5%e8%a1%8c%e5%8a%a8%ef%bc%9f/" href="https://www.iyouport.org/%e5%9c%a8%e5%95%86%e5%ba%97%e9%87%8c%ef%bc%8c%e7%a7%98%e5%af%86%e7%9b%91%e8%a7%86%e5%a6%82%e4%bd%95%e8%b7%9f%e8%b8%aa%e4%bd%a0%e7%9a%84%e6%af%8f%e4%b8%80%e6%ad%a5%e8%a1%8c%e5%8a%a8%ef%bc%9f/https://www.iyouport.org/%e5%9c%a8%e5%95%86%e5%ba%97%e9%87%8c%ef%bc%8c%e7%a7%98%e5%af%86%e7%9b%91%e8%a7%86%e5%a6%82%e4%bd%95%e8%b7%9f%e8%b8%aa%e4%bd%a0%e7%9a%84%e6%af%8f%e4%b8%80%e6%ad%a5%e8%a1%8c%e5%8a%a8%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     在商店里，秘密监视如何跟踪你的每一步行动？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   您可以查看这些设备收集的数据，并在可用时访问隐私选项，例如选择退出数据收集。
  </p>
  <p class="graf graf--p">
   Norman Sadeh 教授告诉CNET：“我们的研究表明，大多数人对当今物联网技术所收集的数据量知之甚少。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “研究还表明，很多人希望通过这些技术更好地了解其数据的收集和使用情况，尤其是在他们不希望被收集的情况下。另外，许多人希望可以自己选择退出某些收集。”
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    不要错过，
    <strong class="markup--strong markup--li-strong">
     物联网对维稳的作用
    </strong>
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e5%ae%b6%e5%86%b0%e7%ae%b1%e5%a6%82%e4%bd%95%e4%bd%9c%e4%b8%ba%e5%91%88%e5%a0%82%e8%af%81%e4%be%9b%ef%bc%9f/" href="https://www.iyouport.org/%e4%bd%a0%e5%ae%b6%e5%86%b0%e7%ae%b1%e5%a6%82%e4%bd%95%e4%bd%9c%e4%b8%ba%e5%91%88%e5%a0%82%e8%af%81%e4%be%9b%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     当你家冰箱作为呈堂证供
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   Sadeh 说，根据该报告，该应用程序已注册了30000多个IoT资源，“我们预计到下月底将拥有更多的资源 —— 其中一些是由单个用户注册的，而许多是由我们的团队输入的。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们期望更多地依靠个人用户来填充注册列表，但是我们知道，使这一点变得足够有价值是我们的基本责任。”
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/Q34eRGwyUQo" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   根据乔治敦法律隐私与技术中心的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.law.georgetown.edu/privacy-technology-center/" href="https://www.law.georgetown.edu/privacy-technology-center/" rel="noopener noreferrer" target="_blank">
    估计
   </a>
   ，美国超过一半的成年人的脸至少存在于一个用于面部识别的数据库中。
  </p>
  <p class="graf graf--p">
   2017年，国土安全部阐明了他们在国内间谍活动中的立场，甚至称：不希望自己的脸部被扫描的美国人“不应该旅行”。
  </p>
  <p class="graf graf--p">
   国土安全部在一份
   <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/3894574-CBP-assessment-on-facial-recognition-at-airports.html#document/p10/a362620" href="https://www.documentcloud.org/documents/3894574-CBP-assessment-on-facial-recognition-at-airports.html#document/p10/a362620" rel="noopener noreferrer" target="_blank">
    文件
   </a>
   中写道：“确保个人出国旅行时不受其生物特征信息收集的唯一方法是避免旅行。”
  </p>
  <p class="graf graf--p">
   这项技术的迅速发展引发了迫切需要的辩论，以延缓其推出速度。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    辩论炮药《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%af%b4%e8%b0%8e%e7%9a%84%e7%9c%bc%e7%9d%9b%ef%bc%9a%e6%a0%b9%e6%a4%8d%e4%ba%8e%e7%bb%b4%e5%a4%9a%e5%88%a9%e4%ba%9a%e6%97%b6%e4%bb%a3%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab/v" href="https://www.iyouport.org/%e8%af%b4%e8%b0%8e%e7%9a%84%e7%9c%bc%e7%9d%9b%ef%bc%9a%e6%a0%b9%e6%a4%8d%e4%ba%8e%e7%bb%b4%e5%a4%9a%e5%88%a9%e4%ba%9a%e6%97%b6%e4%bb%a3%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab/v" rel="noopener noreferrer" target="_blank">
     说谎的眼睛：根植于维多利亚时代面部识别
    </a>
    》
   </li>
   <li class="graf graf--li">
    辩论炮药《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%88%91%e4%bb%ac%e9%81%97%e8%90%bd%e7%9a%84%e9%aa%a8%e9%aa%b8/" href="https://www.iyouport.org/%e6%88%91%e4%bb%ac%e9%81%97%e8%90%bd%e7%9a%84%e9%aa%a8%e9%aa%b8/" rel="noopener noreferrer" target="_blank">
     我们遗落的骨骸
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   去年三月推出的参议院法案将迫使希望在消费者身上使用面部识别技术的公司首先获得他们的同意。
  </p>
  <p class="graf graf--p">
   众议院监督委员会主席 Elijah Cummings
   <a class="markup--anchor markup--p-anchor" data-href="https://www.youtube.com/watch?time_continue=8747&amp;v=jLmzEFkbNsg&amp;t=43m3s" href="https://www.youtube.com/watch?time_continue=8747&amp;v=jLmzEFkbNsg&amp;t=43m3s" rel="noopener noreferrer" target="_blank">
    说
   </a>
   ：“实际上没有任何控制措施……。无论您来自哪个行业，您都可能会参与此[监视]过程。”
  </p>
  <p class="graf graf--p">
   该委员会的最高共和党众议员 R-Ohio 还表示，“现在是时候了”，说明政府对监视技术的使用。
  </p>
  <p class="graf graf--p">
   ACLU 最近还起诉了包括联邦调查局和国土安全部在内的多家机构，以抵抗面部识别技术侵犯个人公民的隐私权。
  </p>
  <p class="graf graf--p">
   ACLU 和 “为未来而战” 以及许多其他公民社会团体都呼吁终结这种危险的技术，而且呼吁越来越强大。这场漫长的战斗已经取得了几项胜利。
  </p>
  <p class="graf graf--p">
   但是，仅仅停止面部识别是不够的。监视基础设施如果继续存在，就肯定会不断发展出同类的甚至更为侵入性的监视技术。必须彻底摧毁反乌托邦的基础结构，反抗才有希望。⚪️
  </p>
