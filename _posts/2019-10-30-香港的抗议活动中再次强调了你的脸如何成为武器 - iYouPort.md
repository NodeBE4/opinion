  <p class="graf graf--p">
   <strong>
    香港的抗议活动中发生的事再次证实了我们曾经强调过的内容非常重要。
   </strong>
  </p>
  <p class="graf graf--p">
   以下是纽约时报的报道。
  </p>
  <p class="graf graf--p">
   警察与科林·张(Colin Cheung)在一辆汽车里扭打起来。警察想要张的脸。
  </p>
  <p class="graf graf--p">
   警察抓住他的下巴，强迫他把头转向他的 iPhone。他们猛烈地拍打他的脸。
  </p>
  <p class="graf graf--p">
   警察大喊着，“睁开眼！”他们硬拉开他的眼皮，让他睁眼。
  </p>
  <p class="graf graf--p">
   这一切都失败了：因为，就在警察抓他的那一刻，张已经用快捷键关闭了手机的面部识别解锁功能。
  </p>
  <p class="graf graf--p">
   这就是我们强调的应急措施的实践 —— 具体操作方法见上面两个链接。
  </p>
  <p class="graf graf--p">
   香港的示威者和警察都将识别身份变成了一种武器。
  </p>
  <p class="graf graf--p">
   当局在网上追踪抗议活动的领导者，寻求破解他们的手机。许多抗议者现在都把脸遮起来，担心警察通过摄像头和其他任何可能的工具来抓人。
  </p>
  <p class="graf graf--p">
   随着暴力升级，警察不再佩戴有识别号码的徽章，一些抗议者开始在网上公布警察的身份。
  </p>
  <p class="graf graf--p">
   Telegram 上有一个用户快速增长的频道，寻找并发布有关警察及其家人的个人信息。
  </p>
  <p class="graf graf--p">
   这个名为“老豆揾仔”的频道已有5万多名订户。支持政府的对手们也有寻求以类似方式揭露抗议者身份的频道。
  </p>
  <p class="graf graf--p">
   因涉嫌“串谋及唆使谋杀”而被逮捕的张先生是“老豆揾仔”频道的订户，但他否认自己是该频道的创始人之一，那是警方的说法，他谴责了该频道上呼吁暴力的帖子。
  </p>
  <p class="graf graf--p">
   他认为，自己之所以成为警方的目标，是因为他开发了一个可以将图像与一组警官照片做比较以寻找匹配的工具。
  </p>
  <p class="graf graf--p">
   他后来放弃了这个项目。
  </p>
  <p class="graf graf--p">
   张已被保释，尚未被指控犯有不法行为。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “普通市民可以有工具可用，可以知道谁是警察，”张说。
  </p>
  <p class="graf graf--p">
   政府追踪危险罪犯与合法政治抗议者的能力，以及警察的打击目标进行反击的能力都在发生着重大变化，而香港正处于这一变化的最前沿。
  </p>
  <p class="graf graf--p">
   在边境另一边的中国内地，警方经常用从世界上侵入性最强的监控系统之一收集来的数字指纹以抓捕公民。
  </p>
  <p class="graf graf--p">
   人脸识别技术的出现、以及摄像机和其他跟踪工具的庞大网络的迅速扩大，已经极大程度上提高了这些能力。
  </p>
  <p class="graf graf--p">
   香港当局已对人脸识别的使用和其他生物特征数据的采集
   <a class="markup--anchor markup--p-anchor" data-href="https://www.info.gov.hk/gia/general/201906/05/P2019060500349.htm" href="https://www.info.gov.hk/gia/general/201906/05/P2019060500349.htm" rel="nofollow noopener noreferrer" target="_blank">
    做了概括性的严格隐私控制
   </a>
   ，但这些努力的程度尚不清楚。
  </p>
  <p class="graf graf--p">
   香港政府似乎也在使用其他技术手段来跟踪抗议者。六月，
   <a class="markup--anchor markup--p-anchor" data-href="https://cn.nytimes.com/china/20190614/hong-kong-telegram-protests/" href="https://cn.nytimes.com/china/20190614/hong-kong-telegram-protests/" rel="nofollow noopener noreferrer" target="_blank">
    一名22岁的男子被捕
   </a>
   ，因为他是 Telegram 上一个群组的管理员。
  </p>
  <p class="graf graf--p">
   抗议者做出了回应。
  </p>
  <p class="graf graf--p">
   七月，随着又一场示威演变为与警方的暴力冲突，一些参与者用激光笔照射警察的摄像头，并在中央政府驻香港联络办公室前用喷漆挡住了监控摄像机的镜头。
  </p>
  <p class="graf graf--p">
   防暴警察举着带有摄像头的杆子站在发射催泪瓦斯和橡皮子弹的最前排警察的后面。
  </p>
  <p class="graf graf--p">
   警察
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2019/06/24/world/asia/hong-kong-police-protest.html" href="https://www.nytimes.com/2019/06/24/world/asia/hong-kong-police-protest.html" rel="nofollow noopener noreferrer" target="_blank">
    摘下制服上的警号
   </a>
   后，抗议者的愤怒情绪更加强烈了。
  </p>
  <p class="graf graf--p">
   警察那样做大概是为了防止人们向政治领导人举报他们的暴力不端行为。
  </p>
  <p class="graf graf--p">
   在一些抗议者眼里，这种做法表明香港警察正在向中国大陆警察学习，那里缺乏公共问责制，警察常常不透露自己的身份。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “为什么我们被攻击的时候，警察可以全身而退？”理发师 Billy Tsui 问道。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他们如果有错，应该受法律的制裁。”他说，在暴力与和平之间他选择和平，但他也对那个揭露警察身份的 Telegram 群组表示了一些同情，认为那是对警察不当行为的一种检查。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “最初的目的只是为了识别那些警察是谁，”21岁的崔先生说。“如果他们隐藏自己的警号，不表明自己的身份，这就成了搞清他们身份的唯一方法。”
  </p>
  <p class="graf graf--p">
   香港警方代表曾表示，警察及其亲友的个人信息被
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2017/08/30/technology/doxxing-protests.html" href="https://www.iyouport.org/%e8%87%aa%e6%88%91%e4%ba%ba%e8%82%89/" rel="nofollow noopener noreferrer" target="_blank">
    “人肉搜索(doxxing)”
   </a>
   并在网上发布出来。
  </p>
  <p class="graf graf--p">
   警方7月3日说，他们以未经批准泄露个人信息及其他指控逮捕了八人。警方发言人说，在那些数据公布之后，警队成员已报告过800多起警员或家人受骚扰的事情。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “老豆揾仔”频道是发布警官私人信息的一个论坛，频道的名字是从一个 Facebook 群组的名字演化而来的，该群组是以帮助母亲找孩子的名义建立的，但最终成为亲政府团体收集抗议者照片的方式。
  </p>
  <p class="graf graf--p">
   老豆揾仔频道用的语言时而滑稽、时而幼稚、时而残暴，不断发布警察家庭成员的个人信息和照片，有时是来自社交媒体的亲密照片。
  </p>
  <p class="graf graf--p">
   虽然该频道播放过暴力呼吁，经常是以卡通式的方式，但没有证据表明其煽动过任何具体行动。
  </p>
  <p class="graf graf--p">
   其中一个帖子指导抗议者如何熟练地使用弹弓。另一个帖子解释如何使用气溶胶除臭剂制作喷枪。
  </p>
  <p class="graf graf--p">
   该频道最近做的一个调查向订户询问对付警察的最好办法。答案选项包括监狱、毒气室、活埋、断头台，用机关枪处决。活埋以得到大约三分之一的投票获胜。
  </p>
  <p class="graf graf--p">
   警方在该 Telegram 频道创建11天后逮捕了张先生，警方指控他是这个频道的管理员。
  </p>
  <p class="graf graf--p">
   他们还指控他发了一篇如何暗杀警察的指南。
  </p>
  <p class="graf graf--p">
   张先生否认了这些指控，《纽约时报》的检索也没有找到与警方描述相符的帖子。
  </p>
  <p class="graf graf--p">
   张先生现年29岁，个子瘦小，据他的描述，警方在7月18日中午前后在一家商场抓了他。四名便衣警察等着他把手机解锁后，向他扑来，试图从他手中夺走手机。
  </p>
  <p class="graf graf--p">
   在警察试图用他的面部解锁手机未果后，将其带到了警察局。
  </p>
  <p class="graf graf--p">
   他说，在那里他受到了粗暴对待和审讯。
  </p>
  <p class="graf graf--p">
   据他的说法，后来，警方来到他家，用一个装有黑客软件的U盘侵入了他的电脑。
  </p>
  <p class="graf graf--p">
   他说，他被拘留了十多个小时，他也不知道警方是如何确定他身份的。
  </p>
  <p class="graf graf--p">
   香港警方证实了这起调查，但拒绝对其做出进一步的评论。
  </p>
  <p class="graf graf--p">
   警方的动机可能是张先生制作的那个人脸识别工具。
  </p>
  <p class="graf graf--p">
   张先生说，他上个月在 Facebook 上发布的一段视频中展示了这个工具。
  </p>
  <p class="graf graf--p">
   曾在大学读计算机科学，但肄业的张先生使用谷歌的技术建立了一个算法，用网上发布的一小组照片来识别警察。
  </p>
  <p class="graf graf--p">
   他说，由于时间不够，他没有将这个项目继续下去。
  </p>
  <p class="graf graf--p">
   张说被捕加深了他的恐惧。
  </p>
  <p class="graf graf--p">
   他说，逮捕他的便衣警察在他们到警察局之前一直都没有表明身份。后来，一名穿着西装的调查人员敦促他解锁手机，以“向42楼”证明他的清白，张先生说，该说法似乎是指高级警官。
  </p>
  <p class="graf graf--p">
   他不认为警方最终得到了他手机里的内容，不过他们确实侵入了他的其他设备。
  </p>
  <p class="graf graf--p">
   警方最初也不允许他打电话。只是在他说，他有与叔叔一起打乒乓球的计划之后，他们才改变了态度，让他打了电话。
  </p>
  <p class="graf graf--p">
   他说，他没有给叔叔打电话，而是与一个朋友进行了联系。他补充说，“我讨厌体育运动。”
  </p>
  <p class="graf graf--p">
   张还说，他觉得，自从被逮捕以来，便衣警察一直在跟踪他。接受时报采访他迟到了一个小时，他称是因为他试图甩掉盯梢。
  </p>
  <p class="graf graf--p">
   他说，在自己黑色特斯拉的帮助下，他设法在香港北部新界的高速公路上超过了不知道是什么人开的车。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “警察正变得越来越好斗，”他说。“我不认为他们有权解锁我的iPhone或我的任何设备。他们开始变得疯狂了。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我觉得，香港变成与中国大陆一样的可能性更大了，”他说。
  </p>
  <p class="graf graf--p">
   注：香港的做法从对峙角度上是非常合理的 —— 我们不是支持暴力而是支持反击。生物识别信息成为反击的武器是恰当的；没有理由让公民永远被动。⚪️
  </p>
