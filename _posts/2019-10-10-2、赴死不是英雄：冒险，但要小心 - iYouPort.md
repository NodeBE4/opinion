    直接行动
   </a>
   是受压迫的人们在整个人类历史中用来重建力量的工具。
  </p>
  <p class="graf graf--p">
   当社区没有数十亿美元可用时，他们就会利用风险。
  </p>
  <p class="graf graf--p">
   他们把自己的身体、自由和安全放在了第一线。
  </p>
  <p class="graf graf--p">
   无疑，直接行动会带来一些固有的风险。
  </p>
  <p class="graf graf--p">
   因此，💡设计行动的目标应该是，以对参与者、社区、您自己和运动本身
   <strong class="markup--strong markup--p-strong">
    负责的方式，最小化风险
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️如果你让对抗的浪漫感在行动计划中掩盖了细致的关怀，就可能会让其他人受到伤害，受害者极可能会离开行动本身，以应对他们冒险行为的后果。
  </p>
  <p class="graf graf--p">
   💡一个好的行动计划者可以区分自己能够（和应该）控制的风险、和不能控制的风险，并向所有参与者阐明可能产生的后果。
  </p>
  <p class="graf graf--p">
   在策划阶段这是非常重要的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    彻底的行动计划必须建立在您对周围人的责任的基础上
   </strong>
   。
  </p>
  <p class="graf graf--p">
   即使您的计划很好，如果行动日到来并且情况不符合您的预期，也不要因害怕将其取消。
  </p>
  <p class="graf graf--p">
   更好推迟或另择日，而不是拉着你的团队进入完全没有准备的环境。
  </p>
  <p class="graf graf--p">
   Ruckus Society 的小册子概述了在规划行动时应该记住的一些关键注意事项。
  </p>
  <p class="graf graf--p">
   有些内容也许不是第一次讲述。重复是因为我们认为它非常重要。
  </p>
  <p class="graf graf--p">
   如果缺少某些东西，也并不意味着它不重要。也许只是我们没有想到，欢迎有经验的实践者前来补充。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    —— 重要基础 ——
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、要有战略意义地尽可能确保您的计划利于愿景和目标
   </strong>
   。
  </p>
  <p class="graf graf--p">
   不要以“软”、“硬”或“更激进”的标签方式来思考。
  </p>
  <p class="graf graf--p">
   相反，要考虑它是否适合目标、策略、情境、信息风险、升级程度等等。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、从最简单的开始。致力于构建它。
   </strong>
  </p>
  <p class="graf graf--p">
   不要一次性部署所有技术。
  </p>
  <p class="graf graf--p">
   如果合适，请留出可升级的空间。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、建立一个团队。
   </strong>
  </p>
  <p class="graf graf--p">
   这一切都始于一个良好的行动团队（亲和力的团体）和良好的非暴力/直接行动培训。
  </p>
  <p class="graf graf--p">
   所有角色都很重要。
  </p>
  <p class="graf graf--p">
   拥有好的支持对一个团队来说至关重要。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、了解你的极限
   </strong>
   。
  </p>
  <p class="graf graf--p">
   对您的能力和资源进行实际的评估。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、侦察，侦察，侦察
   </strong>
   。
  </p>
  <p class="graf graf--p">
   花更多的时间来了解你的位置，不论是物理层面的还是精神意识层面的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    6、了解你最有效的阻塞点
   </strong>
   —— 能使你成为最讨厌的阻挡者的地方。
  </p>
  <p class="graf graf--p">
   不能让你的对手在你周边正常工作/步行/开车 —— 要让他们无法忽视你。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    7、实践。
   </strong>
  </p>
  <p class="graf graf--p">
   一遍又一遍。通过练习熟悉它。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    8、得到帮助
   </strong>
   。
  </p>
  <p class="graf graf--p">
   利用他人的经验、培训或现场支持。每个人的第一次都要这样做。
  </p>
  <p class="graf graf--p">
   纸上谈兵是绝对不行的。
  </p>
  <p class="graf graf--p">
   您拥有的实战经验越多，您就越容易以适当的方式支持队友。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    9、不要仅仅*为*你的行动本身做计划，要尽可能计划*通过*行动能实现什么样的结果
   </strong>
   。
  </p>
  <p class="graf graf--p">
   把整个直接行动想象成一个“中间过程” —— 有大量的准备工作和大量的后续行动，法律、情感、政治……等等一切。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    10、制定 PlanB
   </strong>
   ……以及 C，D＆E，如果您认为可能用得到的话，全包括进去。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    11、制定媒体策略并执行
   </strong>
   。
  </p>
  <p class="graf graf--p">
   确保您的轰动效应，并且您的操作逻辑尽可能透明。
  </p>
  <p class="graf graf--p">
   不要让沟通成为事后的想法！
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    12、消除不必要的风险
   </strong>
   。
  </p>
  <p class="graf graf--p">
   让您的行动尽可能安全，以实现您的目标。
  </p>
  <p class="graf graf--p">
   这是一个负责的领导者所必要的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    13、不要忽视你的团队内部之间和警察/国家之间的权力动态
   </strong>
   。
  </p>
  <p class="graf graf--p">
   种族、阶级、性别认同（真实或感知）、年龄、身体能力、外表、移​​民身份、国籍等等；所有这些都会影响你与行动的关系。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    14、有效的穿着
   </strong>
   。
  </p>
  <p class="graf graf--p">
   确保您的外观有助于携带出您想要发出的声音，最好是无须解释的动机。
  </p>
  <p class="graf graf--p">
   并且穿着要舒适。
  </p>
  <p class="graf graf--p">
   如果您需要实施长时间的封锁占领，无法移动，请考虑穿着成人纸尿裤。这不是玩笑。在黄背心运动中，由于女性占据行动者较大比重，警方利用封锁禁止抗议者如厕以此来羞辱女性，她们不得不当街小便。
  </p>
  <p class="graf graf--p">
   此外，有可能需要携带必要的衣服或毯子，如果要过夜天气变冷，你可以支援周围的人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    15、保持供水
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果你不能携带足够的水，要确保支援部队可以给你供水。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    16、最后，要有创意
   </strong>
   。玩得开心。
  </p>
  <p class="graf graf--p">
