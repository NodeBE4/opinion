  <p class="graf graf--p">
   Private.sh 是一个新的私密搜索引擎，它使用加密技术来确保任何人（甚至用户自己）都无法跟踪搜索历史记录。
  </p>
  <p class="graf graf--p">
   Private.sh 与 GigaBlast 合作，这是一个免费的开源Web搜索引擎和目录，具有相同的隐私承诺。
  </p>
  <p class="graf graf--p">
   有了这种合作伙伴关系，第一个真正的私密搜索引擎才有可能实现，每个合作伙伴都是解决难题的关键部分，从而为搜索引擎提供了以加密方式保护安全的新标准。
  </p>
  <p class="graf graf--p">
   很有可能，您当前使用的搜索引擎不是私密的。
  </p>
  <p class="graf graf--p">
   作为监视资本主义的重要资源之一，搜索引擎的核心功能和商业模式就是跟踪搜索内容以及搜索对象。
  </p>
  <p class="graf graf--p">
   绝大多数搜索引擎都在使用个人身份信息（例如 您的IP地址和浏览器指纹）来创建或添加到您的个人资料中。
  </p>
  <p class="graf graf--p">
   除了记录您要搜索的内容之外，非私密的搜索引擎还记录您点击的结果、花费了多长时间、您是否感到幸运等等。
  </p>
  <p class="graf graf--p">
   使用非私密搜索引擎，不仅能够识别您（搜索者）、还要将搜索字词与用户个人资料相关联，同时针对您投放广告，这是监视资本主义业务模型的重要组成部分。
  </p>
  <p class="graf graf--p">
   使用 Private.sh，您的隐私将受到加密和匿名性设计的保护。
  </p>
  <p class="graf graf--p">
   当您在 Private.sh 中输入搜索字词时，该搜索字词会使用 GigaBlast 的公钥在客户端（您的计算机或设备上）进行加密，只有他们才能解密。实际上，这可确保 Private.sh 永远不会看到搜索词。
  </p>
  <p class="graf graf--p">
   搜索字词经过加密后，​会 通过Private. sh代理传 递给GigaBla st搜索引擎，因此 GigaBlast 无法看到您的IP地址、浏览器指纹或任何可能破坏您的隐私的内容。这意味着 Private.sh 或 GigaBlast 均无法存储您的搜索历史记录。
  </p>
  <p class="graf graf--p">
   最后，搜索结果由 GigaBlast 使用您的临时公钥加密，并通过 Private.sh 代理返回给您。然后将结果解密，使用仅存在于设备上的临时私钥在设备上本地呈现。对于每个搜索请求，都会更改此客户端密钥对。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    还在不断改进
   </strong>
  </h3>
  <p class="graf graf--p">
   另一个好处是，由于 Private.sh 和 GigaBlast 都无法在您身上建立用户个人资料，因此您每次都能获得无偏见的私密搜索结果。避免了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" rel="noopener noreferrer" target="_blank">
    被算法操纵。
   </a>
  </p>
  <p class="graf graf--p">
   您可能会注意到，这些结果并不那么 “准确” 或在算法上不适合您。而那些所谓的 “准确” 是建立在对你的一切心思充分窥探的基础上。这些东西积累起来您肯定会害怕的 —— 因为其目标是更有效地操纵您。
  </p>
  <p class="graf graf--p">
   Private.sh 称将不断努力改善您的搜索体验，在不影响您的安全性的基础上。
  </p>
