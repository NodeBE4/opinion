  <p class="graf graf--p">
   先讲一件真实的事。
  </p>
  <p class="graf graf--p">
   去年的某个时候，我们交流过的一位朋友告诉我们，TA在西方某国读书，但仅仅由于在 Facebook 上为香港的某个活动的页面点了赞，导致TA在中国的亲属被威胁，并要求TA回国。
  </p>
  <p class="graf graf--p">
   请注意，TA已经离开中国在海外读书近10年，TA 的社交媒体使用了别名（不是匿名而是别名）；TA 并没有发言，仅仅是点了赞。TA的 Facebook 个人资料中绝大多数都是“仅限朋友”的设置。
  </p>
  <p class="graf graf--p">
   此事也影响到了TA的学术研究，直到现在TA不敢回中国做某些调查。
  </p>
  <p class="graf graf--p">
   TA联系我们是由于TA想要知道自己是如何被中国当局盯住的。
  </p>
  <p class="graf graf--p">
   事实上很简单，TA没能做到完好地匿名 —— 社交媒体的个人简历（比如求学经历）、与中国社交媒体账户同步发布照片、人际关系在线清晰可见……
  </p>
  <p class="graf graf--p">

   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/facebook-%e6%ad%a3%e5%9c%a8%e4%bb%a4%e7%9c%9f%e7%9b%b8%e6%8c%96%e6%8e%98%e4%ba%8b%e4%b8%9a%e9%99%b7%e5%85%a5%e5%9b%b0%e5%a2%83/" href="https://www.iyouport.org/facebook-%e6%ad%a3%e5%9c%a8%e4%bb%a4%e7%9c%9f%e7%9b%b8%e6%8c%96%e6%8e%98%e4%ba%8b%e4%b8%9a%e9%99%b7%e5%85%a5%e5%9b%b0%e5%a2%83/" rel="noopener noreferrer" target="_blank">

  </p>

   </strong>

   </strong>
  </p>

   <strong class="markup--strong markup--p-strong">

   </strong>
   。
  </p>
  <p class="graf graf--p">
   正如我们最近所看到的那样，网站及其工具来来往往不断更新，这个领域并没有常态；即使您最喜欢的工具被淘汰了，了解它是如何运作的也能为您提供很大帮助。
  </p>

    好奇心是决胜的根基
   </a>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     在开始之前…
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   就像在“旧”图谱搜索中一样，ID号仍然很重要。您可以
   <strong class="markup--strong markup--p-strong">
    按照以下说明很容易地找到一个 Facebook ID 号
   </strong>
   ：
  </p>
  <p class="graf graf--p">
   1、在网络浏览器中打开感兴趣的页面或个人资料；
  </p>
  <p class="graf graf--p">
   2、右键单击页面上没有内容的某个地方（通常在浅灰色区域），然后选择“查看页面源代码”（或使用 Ctrl + U。这在大多数浏览器中都适用）；

  </p>

  <h2 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     组合查询
    </strong>
   </span>
  </h2>
  <p class="graf graf--p">
   以上的内容您掌握了吗？
  </p>
  <p class="graf graf--p">
   如果是，我们将进一步提高 —— 使用组合查询。
  </p>
  <p class="graf graf--p">
   例如，通过这种方式，您可以搜索居住在“加利福尼亚山景城”并为“ Google”工作的、名字里带有“John”的人，等等。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    要组合什么？
   </strong>
  </p>
  <p class="graf graf--p">
   请先回忆上述我们讨论过的类别：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    Search/top/
   </li>
   <li class="graf graf--li">
    Search/posts/
   </li>
   <li class="graf graf--li">
    Search/people/
   </li>
   <li class="graf graf--li">
    Search/photos/
   </li>
   <li class="graf graf--li">
    Search/videos/
   </li>
   <li class="graf graf--li">
    Search/pages/
   </li>
  </ul>
  <p>
   无法将“照片”中的查询与“人物”中的查询结合在一起。因此您必须使组合停留在一个类别中（例如，您使用个人资料在特定群组中查看过的帖子，等等）。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     如何开始组合
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   为了组合查询，我们必须将两行（或更多）JSON 行放在一起。
  </p>
  <p class="graf graf--p">
   最好用一个例子来解释：假设我们正在寻找居住在“加利福尼亚山景城” 为 “ Google” 工作的、名为“John”的人。
  </p>
  <p class="graf graf--p">
   首先构建基本URL。在该示例的情况下，它将如下所示：
  </p>
  <p class="graf graf--p">
   <span style="color: #ff0000;">
    facebook.com/
    <strong class="markup--strong markup--p-strong">
     search/people
    </strong>
    /?q=
    <strong class="markup--strong markup--p-strong">
     john
    </strong>
    &amp;epa=FILTERS&amp;filters=
   </span>
  </p>
  <p class="graf graf--p graf--startsWithSingleQuote">
   <strong class="markup--strong markup--p-strong">
    ‘search/people/’
   </strong>
   的意思是你想要找人；‘?q=
   <strong class="markup--strong markup--p-strong">
    john’
   </strong>
   是目的，查找名为 John 的人。
  </p>
  <p class="graf graf--p">
   因此，现在我们需要找到“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.facebook.com/places/Things-to-do-in-Mountain-View-California/108212625870265/" href="https://www.facebook.com/places/Things-to-do-in-Mountain-View-California/108212625870265/" rel="noopener noreferrer" target="_blank">
    加利福尼亚山景城
   </a>
   ”位置的ID和“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.facebook.com/104958162837/" href="https://www.facebook.com/104958162837/" rel="noopener noreferrer" target="_blank">
    Google
   </a>
   ”业务页面的ID。
  </p>
  <p class="graf graf--p">
   Mountain View, California = 108212625870265
   <br/>
   Google = 104958162837
  </p>
  <p class="graf graf--p">
   现在，使用JSON行来查找特定城市中的人，并插入 Mountain View 的ID：
  </p>
  <p class="graf graf--p">
   <span style="color: #ff0000;">
    {“city”:”{\”name\”:\”users_location\”,\”args\”:\”108212625870265\”}”}
   </span>
  </p>
  <p class="graf graf--p">
   其次，采用JSON行来查找为特定雇主工作的人，并插入 Google 的ID：
  </p>
  <p class="graf graf--p">
   <span style="color: #ff0000;">
    {“employer”:”{\”name\”:\”users_employer\”,\”args\”:\”104958162837\”}”}
   </span>
  </p>
  <p class="graf graf--p">
   现在需要将这两行结合起来。为
   <strong class="markup--strong markup--p-strong">
    了将它们组合在一起，需要删除一些花括号并添加一个逗号
   </strong>
   。
  </p>
  <p class="graf graf--p">
   下面：需要删除的花括号被突出显示，并且需要插入的逗号也被突出显示：
  </p>
  <p class="graf graf--p">
   <span style="color: #ff0000;">
    {“city”:”{\”name\”:\”users_location\”,\”args\”:\”108212625870265\”}”
    <strong class="markup--strong markup--p-strong">
     },{
    </strong>
    “employer”:”{\”name\”:\”users_employer\”,\”args\”:\”104958162837\”}”}
   </span>
  </p>
  <p class="graf graf--p">
   完整的正确JSON行将变成这样：
  </p>
  <p class="graf graf--p">
   <span style="color: #ff0000;">
    {“city”:”{\”name\”:\”users_location\”,\”args\”:\”108212625870265\”}”,”employer”:”{\”name\”:\”users_employer\”,\”args\”:\”104958162837\”}”
   </span>
  </p>
  <p class="graf graf--p">
   尝试组合这些查询时，
   <strong class="markup--strong markup--p-strong">
    对其进行测试很重要
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    通过以下方式可以非常轻松地完成测试
   </strong>
   ：
   <a class="markup--anchor markup--p-anchor" data-href="https://jsonformatter.curiousconcept.com/" href="https://jsonformatter.curiousconcept.com/" rel="noreferrer noopener" target="_blank">
    https://jsonformatter.curiousconcept.com/
   </a>
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    粘贴您创建的JSON行；
   </li>
   <li class="graf graf--li">
    选择JSON模板 ‘Compact’；
   </li>
   <li class="graf graf--li">
    点击 ‘Process’。
   </li>
  </ol>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9885 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-12.png?resize=1100%2C503&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-12.png?w=1316&amp;ssl=1 1316w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-12.png?resize=300%2C137&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-12.png?resize=768%2C351&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-12.png?resize=1024%2C468&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-12.png?resize=1100%2C503&amp;ssl=1 1100w" data-recalc-dims="1" height="503" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-12.png?resize=1100%2C503&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9885" data-recalc-dims="1" height="503" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-12.png?resize=1100%2C503&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-12.png?w=1316&amp;ssl=1 1316w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-12.png?resize=300%2C137&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-12.png?resize=768%2C351&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-12.png?resize=1024%2C468&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-12.png?resize=1100%2C503&amp;ssl=1 1100w" width="1100"/>

  </p>
  <p class="graf graf--p">
   如果您犯了一个错误，则会看到一个报错，并且，其中会包括您犯错的地方。真的很有帮助！
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9886 jetpack-lazy-image" data-lazy-sizes="(max-width: 1022px) 100vw, 1022px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-12.png?resize=1022%2C578&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-12.png?w=1022&amp;ssl=1 1022w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-12.png?resize=300%2C170&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-12.png?resize=768%2C434&amp;ssl=1 768w" data-recalc-dims="1" height="578" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-12.png?resize=1022%2C578&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1022"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9886" data-recalc-dims="1" height="578" sizes="(max-width: 1022px) 100vw, 1022px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-12.png?resize=1022%2C578&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-12.png?w=1022&amp;ssl=1 1022w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-12.png?resize=300%2C170&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-12.png?resize=768%2C434&amp;ssl=1 768w" width="1022"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   最常见的错误是JSON行中的引号不正确。必须是英文直引号。
  </p>
  <p class="graf graf--p">
   如果您在测试 JSON 行时遇到错误，请确保将 JSON 行复制并粘贴到 Microsoft Word 等文本程序中，选取文本并
   <strong class="markup--strong markup--p-strong">
    将字体更改为 “ Lucinda Console”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   现在，复制并粘贴 JSON 行，您的问题应该已解决。
  </p>
  <p class="graf graf--p">
   因此，既然已经拥有正确的 JSON 行，那么就可以回到
   <a class="markup--anchor markup--p-anchor" data-href="https://gchq.github.io/CyberChef/" href="https://gchq.github.io/CyberChef/" rel="noopener noreferrer" target="_blank">
    Cyber​​Chef
   </a>
   了。就像前文所演示的那样，我们将 JSON “bake”到 Base64。
  </p>
  <p class="graf graf--p">
   得到的结果如下：
  </p>
  <p class="graf graf--p">
   <span style="color: #ff0000;">
    eyJjaXR5Ijoie1wibmFtZVwiOlwidXNlcnNfbG9jYXRpb25cIixcImFyZ3NcIjpcIjEwODIxMjYyNTg3MDI2NVwifSIsImVtcGxveWVyIjoie1wibmFtZVwiOlwidXNlcnNfZW1wbG95ZXJcIixcImFyZ3NcIjpcIjEwNDk1ODE2MjgzN1wifSJ9
   </span>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9887 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/8-10.png?resize=1100%2C530&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/8-10.png?w=2556&amp;ssl=1 2556w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/8-10.png?resize=300%2C145&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/8-10.png?resize=768%2C370&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/8-10.png?resize=1024%2C494&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/8-10.png?resize=1100%2C530&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/8-10.png?w=2200&amp;ssl=1 2200w" data-recalc-dims="1" height="530" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/8-10.png?resize=1100%2C530&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     放到一起！
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   好的，现在已经创建了基本URL，并创建了包含两个查询的搜索组合。那么把它们放到一起：
  </p>
  <p class="graf graf--p">
   facebook.com/
   <strong class="markup--strong markup--p-strong">
    search/people
   </strong>
   /?q=
   <strong class="markup--strong markup--p-strong">
    john
   </strong>
   &amp;epa=FILTERS&amp;filters=
   <strong class="markup--strong markup--p-strong">
    eyJjaXR5Ijoie1wibmFtZVwiOlwidXNlcnNfbG9jYXRpb25cIixcImFyZ3NcIjpcIjEwODIxMjYyNTg3MDI2NVwifSIsImVtcGxveWVyIjoie1wibmFtZVwiOlwidXNlcnNfZW1wbG95ZXJcIixcImFyZ3NcIjpcIjEwNDk1ODE2MjgzN1wifSJ9

   <img alt="" class="aligncenter size-full wp-image-9888 jetpack-lazy-image" data-lazy-sizes="(max-width: 994px) 100vw, 994px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-10.png?resize=994%2C728&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-10.png?w=994&amp;ssl=1 994w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-10.png?resize=300%2C220&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-10.png?resize=768%2C562&amp;ssl=1 768w" data-recalc-dims="1" height="728" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-10.png?resize=994%2C728&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="994"/>

   如果您要专门搜索页面或城市，则在单击“选择城市”时使用页面ID号比在下拉菜单中搜索特定页面要有效得多。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     更多地尝试查询
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   您必须尝试不同的查询，以找出最适合您的查询。
  </p>
  <p class="graf graf--p">
   甚至尝试在查询中放入2条以上的JSON行。
  </p>
  <p class="graf graf--p">
   另外，不要忘记搜索栏！也有一些不错的过滤选项！
  </p>
  <p class="graf graf--p">
   而且，如果您在使用搜索栏时已注意到构建URL的方式，您会发现其中的 Base64 行 ——
   <strong class="markup--strong markup--p-strong">
    您总是可以尝试将 Base64 反向转换为 JSON（使用 Cyber​​Chef），以了解 Facebook 如何构建查询，以便您可以从中学习
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️再次强调：

    请注意本文中标注的诸多安全防御用例
