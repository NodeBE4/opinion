   有多么的容易，从对选民的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" rel="noopener noreferrer" target="_blank">
    心理操纵

   到对
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" rel="noopener noreferrer" target="_blank">
    民主本身
   </a>
   的操纵。大数据已经
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%9b%bd%e7%8e%8b%e4%bb%ac%e7%9a%84%e6%99%9a%e5%ae%b4-%e8%bf%99%e6%98%af%e4%b8%80%e5%9c%ba%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e6%94%bf%e5%8f%98/" href="https://www.iyouport.org/%e5%9b%bd%e7%8e%8b%e4%bb%ac%e7%9a%84%e6%99%9a%e5%ae%b4-%e8%bf%99%e6%98%af%e4%b8%80%e5%9c%ba%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e6%94%bf%e5%8f%98/" rel="noopener noreferrer" target="_blank">
    毁掉了民主的实质
   </a>
   。而这一切的根源在于，选举。
  </p>
  <p class="graf graf--p">
   在全球范围内，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%b0%91%e4%b8%bb%e6%ad%a3%e5%9c%a8%e6%ad%bb%e7%ba%bf%e4%b8%8a%e6%8c%a3%e6%89%8e%e5%90%97%ef%bc%9f%e6%80%8e%e4%b9%88%e5%8a%9e%ef%bc%9fis-democracy-in-its-death-throes-what-should-we-do/" href="https://www.iyouport.org/%e6%b0%91%e4%b8%bb%e6%ad%a3%e5%9c%a8%e6%ad%bb%e7%ba%bf%e4%b8%8a%e6%8c%a3%e6%89%8e%e5%90%97%ef%bc%9f%e6%80%8e%e4%b9%88%e5%8a%9e%ef%bc%9fis-democracy-in-its-death-throes-what-should-we-do/" rel="noopener noreferrer" target="_blank">
    许多民主国家的公民都在担心
   </a>
   他们的政府没有按照人民的意愿行事。
  </p>
  <p class="graf graf--p">
   当选民选出代表参加民主运动时，他们希望自己选出的是能够理解和回应选民需求的人。
  </p>
  <p class="graf graf--p">
   美国代表平均每人拥有超过
   <a class="markup--anchor markup--p-anchor" data-href="https://www.census.gov/prod/cen2010/briefs/c2010br-08.pdf" href="https://www.census.gov/prod/cen2010/briefs/c2010br-08.pdf" rel="noopener noreferrer" target="_blank">
    700,000名选民
   </a>
   ，即使带着最好的意图，这项任务也越来越难以捉摸。结果是，
   <a class="markup--anchor markup--p-anchor" data-href="https://news.gallup.com/poll/228281/satisfaction-government-remains-low.aspx" href="https://news.gallup.com/poll/228281/satisfaction-government-remains-low.aspx" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     只有不到40％
    </strong>
   </a>
   的美国人对他们的联邦政府感到满意。
  </p>
  <p class="graf graf--p">
   在
   <a class="markup--anchor markup--p-anchor" data-href="https://theconversation.com/what-is-italys-five-star-movement-69596" href="https://theconversation.com/what-is-italys-five-star-movement-69596" rel="noopener noreferrer" target="_blank">
    欧洲
   </a>
   、南美、中东和中国，大量的社会运动都在要求更好的政府，但是，都没能得到真正和持久的结果。
  </p>
  <p class="graf graf--p">
   在从事民主、公民权和种族问题的工作中，我从过去到现在一直在研究民主创新。在我的新书《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.rowmaninternational.com/book/the_crisis_of_liberal_democracy_and_the_path_ahead/3-156-9acd3359-4c63-40f6-822d-be21a4c2dacb" href="https://www.rowmaninternational.com/book/the_crisis_of_liberal_democracy_and_the_path_ahead/3-156-9acd3359-4c63-40f6-822d-be21a4c2dacb" rel="noopener noreferrer" target="_blank">
    自由民主的危机和前进之路
   </a>
   ：政治代表制和资本主义的替代品》中探讨了一个问题，即 麻烦可能实际上来自于民主选举本身。
  </p>
  <p class="graf graf--p">
   我的研究表明，另一种方法 —— 随机选择公民轮流执政 —— 有望振兴陷入困境的民主国家。
  </p>
  <p class="graf graf--p">

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     从头说起
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   民主始于自治，普通公民轮流参与公共事务。在古代雅典，民主要求许多小时的公共服务和积极参与。对所有40,000名成年男性公民开放的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.history.com/topics/ancient-greece/ancient-greece-democracy" href="https://www.history.com/topics/ancient-greece/ancient-greece-democracy" rel="noopener noreferrer" target="_blank">
    公众大会
   </a>
   ，每年开会40次，讨论法律。
  </p>
  <p class="graf graf--p">
   但是，即使在这样一个很小的社会中，也需要将一些权力下放给较小的群体。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.britannica.com/topic/Council-of-Five-Hundred-ancient-Greek-council" href="https://www.britannica.com/topic/Council-of-Five-Hundred-ancient-Greek-council" rel="noopener noreferrer" target="_blank">
    行政部门
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.britannica.com/topic/dicastery" href="https://www.britannica.com/topic/dicastery" rel="noopener noreferrer" target="_blank">
    法院
   </a>
   均由每天开会的500名成员组成。这些机构成员都是随机选出的公民。
  </p>
  <p class="graf graf--p">
   而最近的民主社会，特别是那些受到美国模式启发的民主社会，更青睐精英统治。
  </p>
  <p class="graf graf--p">
   詹姆斯·麦迪逊（James Madison）在
   <a class="markup--anchor markup--p-anchor" data-href="https://avalon.law.yale.edu/18th_century/fed63.asp" href="https://avalon.law.yale.edu/18th_century/fed63.asp" rel="noopener noreferrer" target="_blank">
    第63号联邦制文件
   </a>
   中主张将普通民众排除在政治权力之外，而支持当选代表，他认为这样做比较明智。
  </p>
  <p class="graf graf--p">
   麦迪逊和创始人亚历山大·汉密尔顿（Alexander Hamilton）担心 “暴民” 统治太多，以至于他们反对参议员和总统的直接选举。
  </p>
  <p class="graf graf--p">
   1913年，宪法
   <a class="markup--anchor markup--p-anchor" data-href="https://www.archives.gov/legislative/features/17th-amendment" href="https://www.archives.gov/legislative/features/17th-amendment" rel="noopener noreferrer" target="_blank">
    第17条
   </a>
   修正案改变了参议员的选举方式，但选举团依旧保留下来了。
  </p>
  <p class="graf graf--p">
   随着时间的流逝，美国人开始接受精英阶层的这一统治模式。他们退回到了自己的私人生活中，仅从事个人和专业业务，而
   <strong class="markup--strong markup--p-strong">
    把公共事务
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.gutenberg.org/files/815/815-h/815-h.htm" href="https://www.gutenberg.org/files/815/815-h/815-h.htm" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     留给了其他人
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    大量的学术研究已经记录了这种对政治的大范围漠视是如何导致了
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     对公众舆论的操纵
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    、以及经济精英和企业利益集团的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89/" href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     大规模权力滥用
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p">
   幸运的是，在2500年的民主经验中可以找到一些解决方案。
  </p>
  <figure aria-describedby="caption-attachment-10948" class="wp-caption aligncenter" id="attachment_10948" style="width: 1200px">
   <img alt="" class="size-full wp-image-10948 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-12.jpg?resize=1100%2C717&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-12.jpg?w=1200&amp;ssl=1 1200w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-12.jpg?resize=300%2C196&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-12.jpg?resize=1024%2C667&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-12.jpg?resize=768%2C500&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-12.jpg?resize=1100%2C717&amp;ssl=1 1100w" data-recalc-dims="1" height="717" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-12.jpg?resize=1100%2C717&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="size-full wp-image-10948" data-recalc-dims="1" height="717" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-12.jpg?resize=1100%2C717&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-12.jpg?w=1200&amp;ssl=1 1200w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-12.jpg?resize=300%2C196&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-12.jpg?resize=1024%2C667&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-12.jpg?resize=768%2C500&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-12.jpg?resize=1100%2C717&amp;ssl=1 1100w" width="1100"/>
   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-10948">
    A member of Congress makes a phone call – is a donor, a lobbyist or a constituent on the other end of the line? AP Photo/J. Scott Applewhite
   </figcaption>
  </figure>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     摆脱长期官员
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   长期当选的官员可以巩固知识、权力和对他人的影响力。
  </p>
  <p class="graf graf--p">
   国会议员
   <a class="markup--anchor markup--p-anchor" data-href="https://www.huffpost.com/entry/call-time-congressional-fundraising_n_2427291" href="https://www.huffpost.com/entry/call-time-congressional-fundraising_n_2427291" rel="noopener noreferrer" target="_blank">
    只花更多的时间
   </a>
   与捐助者和游说者在一起，以及花时间在连任和为政党筹集资金上，而不是与选民在一起。
  </p>
  <p class="graf graf--p">
   对连任的自私的关注使他们无法全力为公众服务。
  </p>
  <p class="graf graf--p">
   罗马共和国限制人们一生中多次担任公职。任期结束后，每个人都必须公开说明自己在任职期间的行为。这与从小型地方办事处到州立法机关再到国会及其他地方的美国标准政治道路相去甚远。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10949 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-10.jpg?resize=1100%2C700&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-10.jpg?w=1200&amp;ssl=1 1200w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-10.jpg?resize=300%2C191&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-10.jpg?resize=1024%2C652&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-10.jpg?resize=768%2C489&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-10.jpg?resize=1100%2C700&amp;ssl=1 1100w" data-recalc-dims="1" height="700" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-10.jpg?resize=1100%2C700&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10949" data-recalc-dims="1" height="700" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-10.jpg?resize=1100%2C700&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-10.jpg?w=1200&amp;ssl=1 1200w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-10.jpg?resize=300%2C191&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-10.jpg?resize=1024%2C652&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-10.jpg?resize=768%2C489&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-10.jpg?resize=1100%2C700&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     让所有人在本地参与
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   对于当地事务，公民可以直接参与当地决策。在佛蒙特州，3月的第一个星期二是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.timeanddate.com/holidays/us/town-meeting-day-vermont" href="https://www.timeanddate.com/holidays/us/town-meeting-day-vermont" rel="noopener noreferrer" target="_blank">
    城镇会议日
   </a>
   ，这是一个公共假期，在此期间，居民们聚集在市政厅讨论和商议他们希望解决的任何问题。
  </p>
  <p class="graf graf--p">
   在瑞士的一些州，城镇居民每年在
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Landsgemeinde" href="https://en.wikipedia.org/wiki/Landsgemeinde" rel="noopener noreferrer" target="_blank">
    Landsgemeinden
   </a>
   举行一次会议，选举公职人员并讨论预算。
   <strong class="markup--strong markup--p-strong">
    这是最古老的直接民主
   </strong>
   。
  </p>
  <p class="graf graf--p">
   30多年来，世界各地的社区都在通过“参与式预算”这一过程来
   <strong class="markup--strong markup--p-strong">
    让普通公民参与到如何使用公共资金的决策中来
   </strong>
   ，该过程涉及公开会议和邻里协会的参与。多达
   <a class="markup--anchor markup--p-anchor" data-href="https://www.participatorybudgeting.org/hope-for-democracy-a-new-book-reflects-on-30-years-of-participatory-budgeting/" href="https://www.participatorybudgeting.org/hope-for-democracy-a-new-book-reflects-on-30-years-of-participatory-budgeting/" rel="noopener noreferrer" target="_blank">
    7,000个城镇
   </a>
   通过这种方式分配了至少部分资金。
  </p>
  <p class="graf graf--p">
   纽约大学的治理实验室已将众包服务带到了一些城市，这些城市通过一种称为 “crowd-problem solving” 的过程来
   <strong class="markup--strong markup--p-strong">
    寻求一些最紧迫问题的创新解决方案
   </strong>
   。
  </p>
  <p class="graf graf--p">
   社区中的所有居民
   <a class="markup--anchor markup--p-anchor" data-href="https://www.thegovlab.org/static/files/publications/people-led.pdf" href="https://www.thegovlab.org/static/files/publications/people-led.pdf" rel="noopener noreferrer" target="_blank">
    都可以参与
   </a>
   、集思广益的想法，并选择可行的方法，而不必将问题留给少数官僚和专家。
  </p>
  <p class="graf graf--p">
   数字技术使大批人更容易了解和参与潜在的公共问题解决方案。例如，在波兰港口城市
   <a class="markup--anchor markup--p-anchor" data-href="https://apolitical.co/solution_article/citizens-tackle-public-problems/" href="https://apolitical.co/solution_article/citizens-tackle-public-problems/" rel="noopener noreferrer" target="_blank">
    格但斯克
   </a>
   ，市民能够帮助选择减少洪水造成的危害的方法。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     随机选择代表
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在较大的群体中，例如国家和国际治理，我认为值得回到雅典选拔代表的方法：通过随机选拔，而不是通过选举。
  </p>
  <p class="graf graf--p">
   与古代一样，
   <strong class="markup--strong markup--p-strong">
    这使普通民众可以
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.worldcat.org/title/citizen-legislature/oclc/11574116" href="https://www.worldcat.org/title/citizen-legislature/oclc/11574116" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     直接参与
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    政府活动，并削减了特殊利益 —— 游说者和财政捐助者的影响
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10950 jetpack-lazy-image" data-lazy-sizes="(max-width: 794px) 100vw, 794px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-20.png?resize=794%2C892&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-20.png?w=794&amp;ssl=1 794w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-20.png?resize=267%2C300&amp;ssl=1 267w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-20.png?resize=768%2C863&amp;ssl=1 768w" data-recalc-dims="1" height="892" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-20.png?resize=794%2C892&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="794"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10950" data-recalc-dims="1" height="892" sizes="(max-width: 794px) 100vw, 794px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-20.png?resize=794%2C892&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-20.png?w=794&amp;ssl=1 794w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-20.png?resize=267%2C300&amp;ssl=1 267w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-20.png?resize=768%2C863&amp;ssl=1 768w" width="794"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   斯坦福大学的政治学家詹姆斯·菲什金（James Fishkin）称这种想法为“
   <a class="markup--anchor markup--p-anchor" data-href="https://cdd.stanford.edu/" href="https://cdd.stanford.edu/" rel="noopener noreferrer" target="_blank">
    审议式投票
   </a>
   ”的一种变体，它涉及随机选择的公民，这些公民将获得专家信息和在讨论中的指导。
  </p>
  <p class="graf graf--p">
   在90年代，这种方法已经导致八家德克萨斯能源公司采用了美国最先进的
   <a class="markup--anchor markup--p-anchor" data-href="https://cdd.stanford.edu/1998/deliberative-polling-texas-electric-utilities/" href="https://cdd.stanford.edu/1998/deliberative-polling-texas-electric-utilities/" rel="noopener noreferrer" target="_blank">
    风能政策
   </a>
   。
  </p>
  <p class="graf graf--p">
   2016年，爱尔兰召集了99名随机选出的公民，并由一位国家最高法院法官担任主席。他们的任务是研究该国面临的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.citizensassembly.ie/en/" href="https://www.citizensassembly.ie/en/" rel="noopener noreferrer" target="_blank">
    主要问题
   </a>
   ，包括堕胎、人口老龄化和气候变化，并向国家报告。
  </p>
  <p class="graf graf--p">
   在考虑改革选举制度时，荷兰和蒙古以及加拿大的不列颠哥伦比亚省和安大略省
   <a class="markup--anchor markup--p-anchor" data-href="https://global.oup.com/academic/product/democracy-when-the-people-are-thinking-9780198820291?cc=us&amp;lang=en&amp;" href="https://global.oup.com/academic/product/democracy-when-the-people-are-thinking-9780198820291?cc=us&amp;lang=en&amp;" rel="noopener noreferrer" target="_blank">
    都是
    <strong class="markup--strong markup--p-strong">
     随机选择
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    公民来辩论这些问题的，而不是举行选举
   </strong>
   。
  </p>
  <p class="graf graf--p">
   所有这些使我得出一个结论，即 公众认为是最佳的政治决定不是由专业政治家做出的。
  </p>
  <p class="graf graf--p">
   相反，随机选择的普通公民 —— 并给予了时间、必要的信息和空间 —— 可以彼此倾听和辩论，更适合做出这些决定，同时获得有关政治的实践经验，并同时与广泛的政治疏远作斗争。
  </p>
  <p class="graf graf--p">
   此外，在必要时召集议员的随机选择，阻碍了专业政治人士的出现，并削弱了任何人竞选公职的必要性。个人财富和竞选捐款将是无关紧要的。
  </p>
  <p class="graf graf--p">
   媒体操纵也将毫无用处，因为没人会事先知道将要选谁，因此没人会宣传自己的功绩或攻击对手。
  </p>
  <p class="graf graf--p">
   每个公民都可以表达真实的声音、而没有特殊利益和虚假假信息的体制；对我来说，这听起来像是真正的民主。⚪️
  </p>
  <p class="graf graf--p">

    Voting could be the problem with democracy
   </a>
  </p>
