 </p>
 <p>
  <span style="font-size: 12pt;">
   2017年即将成为历史，在岁末年尾，博闻社和博讯新闻网联合评选出2017年国内十大新闻事件，并结合上述事件对国内政治、经济、文化和外交形势进行综合评述：
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   <strong>
    一、中共十九大召开
   </strong>
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   2017年10月18－24日，中共在京召开了十九大，选举了以习近平为核心的新一届政治局和常委会。除习近平、李克强外，新晋常委分别是栗战书、汪洋、王沪宁、赵乐际和韩正。据消息披露，王岐山预计将在2018年3月的人大会议上出任国家副主席。十九大习近平思想被写入党章，国家监察委员会将被设立，但未确定习近平的下一届接班人。十九大前，中共高层激烈权斗，曾被视为第六代领导人之一的孙政才被开除党籍，移送司法机关处理。在海外，具有国安背景的原国内房地产商人郭文贵利用媒体和自媒体爆料，以期搅局十九大，其矛头直指王岐山、孟建柱等。十九大的召开意味着习近平已掌握绝对权力，中国将进入习近平的新极权主义时代，李伟东先生五年前发文称习近平将建设一个红色帝国的预言应验。
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   <strong>
    二、刘晓波去世
   </strong>
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   诺贝尔和平奖得主、零八宪章主要起草人之一刘晓波于2017年6月在监狱中被确诊为肝癌晚期，7月13日病逝，7月15日火化后，被中共强制海葬。海外民运人士及国际组织曾多方呼吁中共释放刘晓波，让他赴海外就医。海外民主人士在多地为刘晓波举行了追思会和刘晓波精神遗产研讨会。11月7日，保外就医的异议人士杨天水在上海因癌症去世。目前，海外民运领袖之一的王炳章仍在狱中服刑，他已经被单独关押了15年。刘晓波的去世引发海内外民主人士对其倡导的“和平、理性和非暴力”抗争运动的思考，“暴力革命”和“非暴力运动”观点分歧明显。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   <strong>
    三、川普和习近平互访
   </strong>
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   4月6-7日，习近平对美国进行工作访问，在佛罗里达洲海湖庄园与川普总统进行了会谈；11月8-10日川普总统对中国进行国事访问，双方围绕经贸合作和朝鲜问题进行沟通。中方与美方达成了2500亿美元经贸合作意向。川普总统在访问期间未提及中国人权问题。白宫已明确表示不承认中国的市场经济国家地位；2018年美国国家安全战略报告称中国为竞争对手。2017年以来，朝鲜已经进行了多次弹道导弹试验和第六次核试验，其核打击能力已经可以覆盖美国。川普总统曾多次对金正恩政权发表措辞严厉的批评。中方为应对朝鲜局势，正在筹建五个难民营。韩国总统朴槿惠因决定部署美国萨德反导弹系统，遭到中方强烈反对，中韩外交关系严重受损。12月13-16日，韩国新任总统文在寅访华，但受冷遇。6月18日至8月28日期间，因中方在边界洞朗地区修路，印方军队越界进入中方领域，双方发生冲突，后印方撤出。2017年7月中俄签署了《中华人民共和国和俄罗斯联邦关于进一步深化全面战略协作伙伴关系的联合声明》核《中华人民共和国和俄罗斯联邦关于当前世界形势和重大国际问题的联合声明》，这两个重要声明向世人宣示了中俄盟友关系的进一步巩固和提升。2017年，美国希望从中国拿到更大的经济利益和借助中国遏制朝鲜核威胁；中国希望在经济合作的基础上，借助美国在国际社会上的退却，走近世界舞台中央。中美韩在朝鲜问题上的推诿和绥靖政策将会养痈遗患，而受危害最大的国家将是中国。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   <strong>
    四、新疆学校禁止使用维吾尔族语言
   </strong>
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   2017年，中共对新疆和西藏的种族歧视和迫害进一步升级。据报道，近年来新疆大学有2000余名大学生神秘失踪。据美联社报导说，在新疆库尔勒有三四个集中营，一共关押着数千名维吾尔学生。他们在里面学习“中文、法律、民族团结、去除极端思想、爱国主义”，并实施“五个一起”，即“一起住，一起操练，一起学习，一起吃饭，一起睡觉”，没有行动自由。2017年秋季开始，中共在新疆全面禁止学生在学校说维吾尔族语言，实行文化灭绝政策。2017年，中共完成对喇荣五明佛学院的数千僧舍的拆除和驱逐了近5000余名僧尼。2017年西藏僧尼共有5人自焚抗议，2009年至今已有150人自焚。中共的歧视性民族政策不仅维吾尔族和藏族人民强烈不满，就是汉人也很反感，可谓天怒人怨。中共的高压民族政策促使维吾尔族人民争取民族独立的呼声越来越高，通向民族和解的达赖喇嘛“中间道路”主张实现希望渺茫。
  </span>
 </p>
 <p>
 </p>
 <p>
  <strong>
   <span style="font-size: 12pt;">
    五、北京驱逐外来低端人口
   </span>
  </strong>
 </p>
 <p>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   11月18日晚，大兴区西红门镇新建二村发生重大火灾事故，造成多人伤亡。火灾发生后，20日起，北京市政府部署了为期40天的安全隐患大排查、大清理、大整治专项行动。在北京的气温已在零度以下的情况下，北京市政府在没有采取任何安置措施的情况下，使用 “暴力清退”的方法，强迫外来低端人口在数天内搬走，致使成千上万的外来人无家可归，流离失所。北京数百位公共知识分子发出公开信予以谴责，要求立即停止。此类蛮横不顾民生的做法还发生在华北地区，包括北京市的“煤改气”工程上。为改善空气质量，华北地区强“推煤改气”工程，不许老百姓继续使用煤炉取暖。但由于“煤改气”没有安装完成和天然气储备不足，加之天然气价格成倍上涨，部分老百姓只能挨冻。很多农村学校因不能使用煤炉取暖，只好让学生在户外阳光下上课，致使一些贫困学生被冻伤。11月23日，北京朝阳区管庄红黄蓝幼儿园新天地分园爆出一班级的幼儿遭到老师针扎、并被喂食成分不明的白色药片，成人对儿童性侵事件的丑闻。该事件被迅速传播到媒体上，引起了社会的广泛关注。后警方介入，称喂食药片和性侵事件系个别家长谣言，但称监控录像硬盘被损坏。2017年国内出现了众多群体性事件，如2017年北京多次出现数万复员转业军人请愿抗议事件；4月1日，四川省泸州市泸县太伏中学学生赵鑫坠楼死亡事件，从而引发居民与警方的冲突；深圳市善心汇文化传播有限公司涉庞氏骗局被查处，7月24日，善心汇约6万成员在北京大红门国际服装城和大红门国际会展中心聚集抗议。北京驱赶外来低端人口事件是十九大后中共极权主义政权全面流氓化的开始。众多群体性事件暴露出中共维稳机制已经难以为继，其统治已丧失公信力和合法性。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   <strong>
    六、微信群主责任连坐
   </strong>
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   2017年9月7日，网信办发布新规《互联网群组信息服务管理规定》，规定互联网群组的建立者、管理者和使用者均须认证真实身份；群组不得传播被禁止的资讯内容，群主对群组信息内容有管理责任，即“谁建群谁负责”、“谁管理谁负责”。新规将于10月8日实行。这一规定引发网民热议，网民称之为“连坐群主制”。8月，北京师范大学语言学副教授史杰鹏和山东工商学院政治系主任李默海均因在微博上发表政治言论被解聘。冯小刚导演的电影《芳华》计划于9月29日上映，但9月24日时电影官方宣布《芳华》将不会按原定日期上映。其原因疑为电影中有涉及越战老兵内容，担心引发群体事件干扰十九大召开。叶匡政批判中共打压言论自由的诗歌《我们都是木头人》在网络上广泛流传。2017年，中共对公众舆论的管控进一步升级，公民的言论自由进一步收窄。国内网民翻墙浏览境外网站难度增加，但体制内公共知识分子尚存一定发声空间。中共极权主义面临四大天敌。第一是互联网，第二是难以关闭的国门，第三是严重环境污染。第四是经过近四十年改革开放，已经深入人心的普世价值。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   <strong>
    七、肖建华香港被抓捕
   </strong>
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   2017年1月27日，明天系实际控制人肖建华被从香港四季酒店被带走，押回北京。肖建华为中共权贵们的白手套，掌握有巨额资金。肖建华被抓捕反映中共内斗已进入深水区。4月9日，原保监会主席项俊波因涉嫌严重违纪被双规，据悉项俊波曾对政泉系实际控制人郭文贵的盘古大观项目贷款32亿元。6月安邦保险董事长吴小晖6月被警方拘捕。吴小晖系邓小平的外孙女婿。从2015年戴相龙的女婿车峰被控制，到2017年肖建华、吴小晖、项俊波以及银监会主席助理杨家才相继入狱，一场金融界的政治清洗将会在2018年展开。
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   <strong>
    八、周强对司法独立亮剑
   </strong>
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   2017年1月14日，最高法院院长周强上周六在全国高级法院院长会议时表示：要坚决抵制西方"宪政民主"、"三权分立"、"司法独立"等错误思潮影响，敢于亮剑，坚决同否定中国共产党领导、诋毁中国特色社会主义法治道路和司法制度的错误言行作斗争，决不能落入西方错误 思想和司法独立的"陷阱"。该讲话立即引来海内外舆论的强烈反对浪潮。北大法学院教授贺卫方1月14日曾发表微博，表示这种说法"不可理喻"："无论怎样的国度，司法如果没有独立性，···最终的结果只能是冤屈遍地，引发造反。···把司法独立说成是什么西方观念，必欲除之而后快，是真正的祸国殃民的言行，完全是在开历史的倒车。" 2015年7月9日，中共对百余名维权律师、民间维权人士进行了大规模逮捕后，2017年继续对他们进行人身迫害。2017年4月维权律师李和平被判处有期徒刑3年，缓刑四年；5月8日谢阳律师当庭认罪，被取保后释放；4月21日夏霖律师被北京市高院二审以诈骗罪被判处10年有期徒刑。据报道，维权律师们在服刑期间遭遇酷刑和被强制喂灌不明精神类药物。海外民主人士将7月9日确定为中国人权律师节。2月17日山东聊城中级法院对因黑社会放贷人以生殖器侮辱其母亲而激愤伤人致死的被告人于欢一审判处无期徒刑，而引发社会舆论的声讨。6月23日，山东省高级法院终审改判于欢5年有期徒刑。3月17日，江西赣州农民明经国因反抗政府拆除其空心房而用镰铲袭击乡人大主席致死。11月16日赣州市中级法院开庭审理，但其辩护律师迟夙生却被法院以其没有会见被告人及阅卷的理由，拒绝其参加辩护。2017年，中国司法在中共统治下，其“刀把子”的功能更显突出，法治倒退明显。2018年中共对律师职业的政治管控将会进一步加强，刑事辩护业务将会更加艰难。
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   <strong>
    九、中英联合声明废止
   </strong>
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   2017年6月30日，中国外交部发言人陆慷在记者会上说：现在香港已经回归祖国怀抱20年，《中英联合声明》作为一个历史文件，不再具有任何现实意义，对中国中央政府对香港特区的管理也不具备任何约束力。英方对回归后的香港无主权、无治权，无监督权。英国政府随即对中方的说法提出异议。2016年7月，曾组织和参与“占中”的黄之锋、罗冠聪、周永康三人被香港东区裁判法院裁定犯“煽惑他人参与非法集会罪”和“参与非法集会罪”，被判处社会服务令及监禁缓刑。香港律政司认为刑罚过轻而上诉。8月香港高等法院裁定黄之锋等三人即时监禁6－8个月。12月8日，中国驻美公使李克新在华盛顿表示，他曾告诉美国国会的议员助手，大陆没有时间跟机会用《反分裂国家法》，如果美国把军舰派去台湾，就启动了《反分裂国家法》。美国军舰抵达高雄之日，就是我解放军武力统一台湾之时。李克新此番言论引起舆论哗然。2017年，香港基本法所确立的“一国两制”再遭中共破坏，中共对香港社会的渗透和控制在加速。中共对台湾的恐吓是一贯的，其底线就是“一个中国”原则，但中共以武力统一台湾的图谋难以得逞。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   <strong>
    十、设立雄安新区、推进“一带一路”战略
   </strong>
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   雄安新区是中国第19个国家级新区，也是首个由中共中央、国务院印发通知成立的国家级新区，位于河北省保定市东部，由雄县、容城县、安新县及其周边部分地区组成，于2017年4月1日正式成立。通知中将此新区定位为“千年大计、国家大事”，“是继深圳经济特区和上海浦东新区之后又一具有全国意义的新区”。新区主要任务是成为“北京非首都功能疏解集中承载地”。但雄安新区的设立明显违宪，存在剥夺了人民的知情权和参与权；剥夺了全国人大的审查和监督权；涉嫌侵犯国务院的工作职权等问题，决策论证的科学性不明。2017年中国继续推行一带一路的国际经济战略。5月14日，中国在京主办了“一带一路”国际合作高峰论坛。由于中国在“一带一路”的投资上发生了政策性改变，即央行行长周小川所言“一带一路”投融资合作不是中国单向的资金支持，需要各方共同出资、共担风险、共享收益，同时中方强调发挥本币在“一带一路”建设中的作用，即要中方的投入将是人民币而非美元，从而使参与国热情大减。据中国官方数据，2017年第三季度的中国GDP年均增速6.8%，也是中国经济连续第九个季度保持在6.7%—6.9%的运行区间内。纵观2017年中国经济形势，已经明显陷入长期衰退之中。何清涟女士认为，如果说，美国自2008年之后的经济走势是U字型，那么中国走的就是L型，在L下面那一横还将持续下滑。
  </span>
 </p>
