   您是个精明的购物者，非常清楚网络上潜伏的危险，对吗？

  <p class="graf graf--p">

   那么，为什么不通过使用一次性的信用卡号使安全更进一步呢？对此不熟悉吗？本文将带您浏览一下这方面哪些服务是可信的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    一次性信用卡如何工作？
   </strong>
  </p>
  <p class="graf graf--p">
   听起来像是在间谍电影中找到的东西，还有一次性电话和代号。不要怕，可以保证一次性信用卡是合法的，只要您从信誉良好的来源获得即可。
  </p>
  <p class="graf graf--p">
   一次性信用卡（也称为虚拟卡）采取您一次使用后就可以扔掉的临时号码。您可以通过某些金融机构和应用程序获得。
  </p>
  <p class="graf graf--p">
   目的是在发生数据泄露时掩盖您的真实信用卡详细信息。
   <strong class="markup--strong markup--p-strong">
    数据泄露基本无法避免，只能防备，以将损失降至最低。
   </strong>
  </p>
  <p class="graf graf--p">
   这些交易不会透露您的真实信用卡详细信息，从而提供了额外的保护屏障。
  </p>
  <p class="graf graf--p">
   这类似于我们之前讨论的
   <strong class="markup--strong markup--p-strong">
    虚拟借记卡服务
   </strong>
   。您可以在下面回顾：
  </p>

   <li class="graf graf--li">
    《

   ，您将有资格生成虚拟信用卡号。并非所有花旗卡都提供这些功能，因此您必须进行检查。
  </p>
  <p class="graf graf--p">
   您可以通过登录并检查您的卡权益来做到这一点。请注意：您使用虚拟卡进行的任何交易都会像其他购买交易一样显示在您的帐户中。
  </p>
  <p class="graf graf--p">
   此外，您可以为虚拟号码设置美元限额和有效期。每当您准备关闭虚拟帐户时，都可以使用 Citi 的工具（可以将其下载到PC上）进行关闭。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10547 jetpack-lazy-image" data-lazy-sizes="(max-width: 1022px) 100vw, 1022px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-8.png?resize=1022%2C640&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-8.png?w=1022&amp;ssl=1 1022w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-8.png?resize=300%2C188&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-8.png?resize=768%2C481&amp;ssl=1 768w" data-recalc-dims="1" height="640" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-8.png?resize=1022%2C640&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1022"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10547" data-recalc-dims="1" height="640" sizes="(max-width: 1022px) 100vw, 1022px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-8.png?resize=1022%2C640&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-8.png?w=1022&amp;ssl=1 1022w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-8.png?resize=300%2C188&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-8.png?resize=768%2C481&amp;ssl=1 768w" width="1022"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Capital One Eno
   </strong>
  </p>
  <p class="graf graf--p">
   已经获得 Capital 了？然后，您可以使用
   <a class="markup--anchor markup--p-anchor" data-href="https://chrome.google.com/webstore/detail/eno%C2%AE-from-capital-one%C2%AE/clmkdohmabikagpnhjmgacbclihgmdje?utm_source=lambdromeda" href="https://chrome.google.com/webstore/detail/eno%C2%AE-from-capital-one%C2%AE/clmkdohmabikagpnhjmgacbclihgmdje?utm_source=lambdromeda" rel="noopener noreferrer" target="_blank">
    Capital One的助手（Eno）
   </a>
   生成虚拟卡号。
  </p>
  <p class="graf graf--p">
   您可以通过下载浏览器扩展程序来访问它，该扩展程序在 Mozilla Firefox，Google Chrome 和 Microsoft Edge 上都可用。
  </p>
  <p class="graf graf--p">
   只需点击浏览器上的扩展程序按钮，即可生成虚拟卡。浏览器扩展程序还使您可以访问 Capital One 的网站。在这里，您可以查看、删除和锁定您的虚拟帐号。
  </p>
  <p class="graf graf--p">
   Eno 的优点是您可以即时生成虚拟号码，在结账时随手生成。快速而无缝 —— 正是消费者更喜欢的在线购物体验。
  </p>
  <p class="graf graf--p">
   就算您没有 Capital One 卡，也没关系，Eno 可用于其他 Capital 卡，例如 Venture Rewards 卡和 QuicksilverOne Rewards 卡。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    使用一次性虚拟信用卡更安全地购物
   </strong>
  </p>
  <p class="graf graf--p">
   这里只是当前的一次性虚拟卡提供商的简短列表。随着这种方式的普及，您完全可以期望这个市场会扩大。
  </p>
  <p class="graf graf--p">
   对于精明的消费者而言，这是个好消息，就如您，希望对自己的财务帐户的安全性有更多的控制。
  </p>
  <p class="graf graf--p">
   因此，如果您准备好开始保护自己的财务详细信息了，请尝试这些信誉良好的公司。你喜欢哪一个？⚪️
