  <p class="graf graf--p">
   全世界正在对造假的视频和照片的快速传播深感焦虑，而 Adobe 表示它同意这些担忧。
  </p>
  <p class="graf graf--p">
   现在，Adobe 与加州大学伯克利分校的科学家合作分享了
   <a class="markup--anchor markup--p-anchor" data-href="https://theblog.adobe.com/adobe-research-and-uc-berkeley-detecting-facial-manipulations-in-adobe-photoshop/" href="https://theblog.adobe.com/adobe-research-and-uc-berkeley-detecting-facial-manipulations-in-adobe-photoshop/" rel="noopener noreferrer" target="_blank">
    新的研究成果
   </a>
   ，这些科学家使用机器学习来自动检测面部图像何时被操纵了。
  </p>
  <p class="graf graf--p">
   报道说，这是该公司为此问题投入更多资源的最新迹象。
  </p>
  <p class="graf graf--p">
   去年，该公司的工程师创建了一个AI工具，可以检测通过拼接、克隆和删除对象而创建的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theverge.com/2018/6/22/17487764/adobe-photoshopped-fakes-edit-spotted-using-machine-learning-ai" href="https://www.theverge.com/2018/6/22/17487764/adobe-photoshopped-fakes-edit-spotted-using-machine-learning-ai" rel="noopener noreferrer" target="_blank">
    虚假媒体资源
   </a>
   。
  </p>
  <p class="graf graf--p">
   该公司表示，它没有立即计划将这项最新工作转变为商业产品，但一位发言人告诉 The Verge，这只是 Adobe 在更好地检测图像、视频、音频和文档操纵方面的众多努力之一。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “虽然我们为 Photoshop 和 Adobe 的其他创作工具对世界的影响感到自豪，但是，我们也认识到了这些技术的伦理含义”，该公司在博客中写道，“虚假内容已经是一个严重的且日益紧迫的问题。”
  </p>
  <p class="graf graf--p">
   该研究专门用于发现使用 Photoshop 的 Liquify 工具进行的编辑，该工具通常用于调整面部的形状和改变面部表情。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “该功能的效果可能很精致，这使得它成为一个有趣的测试案例，用于检测面部的激烈和微妙的变化，”Adobe 表示。
  </p>
  <p class="graf graf--p">
   为了创建软件，工程师在匹配面孔的数据库上训练神经网络，包含使用 Liquify 编辑之前和之后的图像。
  </p>
  <p class="graf graf--p">
   由此产生的算法非常有效。当被要求发现被编辑过的面孔样本时，人类志愿者只有在53％的时间内能得出正确答案，而算法在99％的时间内是正确的。
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/TUootD36Xm0" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   该工具甚至可以建议如何将照片恢复到其原始的未经编辑的外观，尽管这些结果通常比较错杂。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “虽然想要一键式恢复被编辑图像的魔术般的想法仍然远非现实，” 帮助开展这项工作的 Adobe 研究员 Richard Zhang 在一篇公司博客文章中说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “但我们生活在一个越来越难以信任我们自己所消费的数字信息的世界里，我期待能进一步探索这一研究领域。”
  </p>
  <p class="graf graf--p">
   研究人员表示，这项工作是第一次设计用于发现这些对面部图像的编辑，并且是创建可以识别复杂变化的工具的 “重要一步”，包括 “身体操纵和光度编辑，如皮肤抛光等”。
  </p>
  <p class="graf graf--p">
   虽然这些研究很有希望，但像这样的工具
   <strong class="markup--strong markup--p-strong">
    并不是
   </strong>
   阻止虚假信息操纵之有害影响的灵丹妙药。
  </p>
  <p class="graf graf--p">
   正如我们已经看到的那样，
   <strong class="markup--strong markup--p-strong">
    虚假信息的传播速度远高于真相，即使内容显然是虚假的、或者可以被迅速揭穿的，但它仍就被广泛分享并被社交媒体所接受。
   </strong>
  </p>
