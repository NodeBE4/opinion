  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在这个10年内，个人的数字双胞胎将进入消费市场。很快，您就能够使用自己的数字副本来为自己在现实世界中做出的决策提供信息

   数字双胞胎将带您进入 *预测* 的下一个前沿，使您能够在未来发生之前就知道未来是什么样。但是它们是如何工作的？
  </p>
  <p class="graf graf--p">
   您可能只是在想象一个超现实的化身，长得就像您一样、和您说一样的话，就像那些已经在游戏、社交媒体和色情行业中
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bbc.co.uk/news/av/technology-50188819/avatars-are-we-ready-for-our-digital-twins" href="https://www.bbc.co.uk/news/av/technology-50188819/avatars-are-we-ready-for-our-digital-twins" rel="noopener noreferrer" target="_blank">

   <strong class="markup--strong markup--p-strong">
    它们是代表真实对象、过程和系统的详细数据。
   </strong>
  </p>
  <p class="graf graf--p">
   您的个人数字双胞胎将成为您的精确虚拟模型，并与您手机上的通知集成，通过 Alexa 或甚至更身临其境的技术激活。
  </p>
  <p class="graf graf--p">
   每个人都在现实生活中不断产生大量的数据，于是这种数字副本将成为可能。
  </p>
  <p class="graf graf--p">
   从跟踪你的睡眠方式和饮食习惯、到你与谁在一起以及你如何花钱，您的一切在线生活正在各个领域中积累越来越详细的数字足迹。
  </p>
  <p class="graf graf--p">
   结合越来越准确的预测算法（该算法可以识别这些数据的模式和趋势以准确地模拟未来结果），这个数字双胞胎将成为探索未来的强大工具。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这将如何影响人们的生活？
   </strong>
  </h3>
  <p class="graf graf--p">
   您的数字双胞胎将充当您的预警系统，在您仍有机会进行更改时发出潜在漏洞的信号，并根据您当前面临的选择向您展示最佳状态。
  </p>
  <p class="graf graf--p">
   早期采用者将使用他们的数字双胞胎来可视化每一个决定可能导致的结果：“如果我学会编码会是什么样？”、或 “如果采用纯素饮食，我会更健康吗？”、”如果我每周开始步行去上班，会有所改善吗？”、或者，也许是一个更具说服力的问题：“如果我什么都没改变，会发生什么？”
  </p>
  <p class="graf graf--p">
   根据从您的个人数据中了解到的关于您的信息，该系统将生成有关每种情况的估计值。随着时间的推移，输入到系统中的个人数据越多，模拟的数字自我就会变得越准确。
  </p>
  <p class="graf graf--p">
   个人甚至可能开始将自己的数字双胞胎与其他人合并，以创建数字家庭和集体，从而模拟
   <strong class="markup--strong markup--p-strong">
    决策对较大人群的影响
   </strong>
   。以制定更明智的决策。
  </p>
  <p class="graf graf--p">
   一切都会变得更积极吗？不！
  </p>
  <p class="graf graf--p">
   ⚠️ 隐私权倡导者会警告寡头公司或流氓政府使用公民的数字双胞胎对人权构成威胁。而且，随着数字双胞胎进入主流，人们将需要提出关于自由意志和代理的更深层次的技术哲学问题。
  </p>
  <p class="graf graf--p">
   依赖这种东西是错误的。如果它出错了，所有糟糕的结果将由你来承受。就算它完全正确，对它的依赖也极有可能导致你本人成为废物。而一旦您不再明智，您的个人数据将出现更多错误，从而影响数字双胞胎的预测准确度。
  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   英国是无现金实践的全球领导者之一。现在英国大约有四分之三的零售交易都是通过刷卡支付的。
  </p>
  <p class="graf graf--p">
   但是，这将带来严重后果。钱包小了，付款卡的使用不断增加，而现金的使用却严重下降。无现金已经到了一个转折点，这将很快给所有人的生活带来根本性改变，远远超出了任何表面现象。
  </p>
  <p class="graf graf--p">
   在这个新10年，人们将开始看到无现金带来的后果：ATM沙漠化不断扩大。使人们难以掌握现金，
   <strong class="markup--strong markup--p-strong">
    你的一举一动都被追踪
   </strong>
   。
  </p>
  <p class="graf graf--p">
   也许还有一个新的现金用途：一种储蓄方式，而不是一种支出方式。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    告别现金
   </strong>
  </h3>
  <p class="graf graf--p">
   电子支付已不是什么新鲜事：自动柜员机自上世纪60年代就出现了，自90年代起用于信用卡和借记卡的销售点设备就出现了。
  </p>
  <p class="graf graf--p">
   因此，芯片和密码、非接触式支付、以及现在的移动支付的出现，还算不上彻底的革命。但是，它们从根本上加快了现金消失的步伐。
  </p>
  <p class="graf graf--p">
   我们现在正处于转折点。卡支付率从5％上升到25％，导致刷卡机和现金一样普及。但是，如果支付卡使用的比例达到95％，这个世界几乎可以全部消灭现金。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     现金为什么是敌人？因为隐私是敌人，隐私是敌人是因为它使政府控制变得更加困难
    </strong>
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%81%90%e6%80%96%e7%9a%84%e5%8f%8d%e7%8e%b0%e9%87%91%e4%b9%8b%e6%88%98/" href="https://www.iyouport.org/%e6%81%90%e6%80%96%e7%9a%84%e5%8f%8d%e7%8e%b0%e9%87%91%e4%b9%8b%e6%88%98/" rel="noopener noreferrer" target="_blank">
     恐怖的反现金之战
    </a>
    》
   </li>
  </ul>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    深刻的全球转变
   </strong>
  </h3>
  <p class="graf graf--p">
   目前在某些国家，现金仍然占主导地位：比如在意大利，仍有80％以上的付款以现金交易，而在德国，只有四分之一的零售交易是通过卡支付的。但是在英国，目前只有
   <a class="markup--anchor markup--p-anchor" data-href="https://www.accesstocash.org.uk/" href="https://www.accesstocash.org.uk/" rel="noopener noreferrer" target="_blank">
    十分之三
   </a>
   的交易是使用现金进行的，而十年前有十分之六。
  </p>
  <p class="graf graf--p">
   无现金经济的全球先驱是瑞典，该国许多银行现在已经不再处理纸币和硬币。零售支付主要通过借记卡和信用卡进行，就像在英国一样。甚至是个人之间或与小型交易者之间的支付，基本上也已经被瑞典银行联合运营的一款移动支付应用程序 Swish 所覆盖。
  </p>
  <p class="graf graf--p">
   这种现象不仅限于西方；在东方的印度，国家政府
   <a class="markup--anchor markup--p-anchor" data-href="https://timesofindia.indiatimes.com/city/goa/bjp-ruled-goa-to-go-cashless-from-october-this-year/articleshow/62711527.cms" href="https://timesofindia.indiatimes.com/city/goa/bjp-ruled-goa-to-go-cashless-from-october-this-year/articleshow/62711527.cms" rel="noopener noreferrer" target="_blank">
    积极促进
   </a>
   无现金交易，并已经取得了部分
   <a class="markup--anchor markup--p-anchor" data-href="https://www.business-standard.com/article/economy-policy/maharashtra-s-first-cashless-village-let-down-by-govt-banks-117100100546_1.html" href="https://www.business-standard.com/article/economy-policy/maharashtra-s-first-cashless-village-let-down-by-govt-banks-117100100546_1.html" rel="noopener noreferrer" target="_blank">
    成功
   </a>
   ，而在非洲部分地区，通过 M-Pesa 进行的移动交易已成为日常生活的一部分 ……
  </p>
  <p class="graf graf--p">
   这个转折点很快就会到来。在英国，卡交易额超过现金的第一年是2017年。而到2019年，卡支付次数已经翻了一番。 UK Finance 的预测
   <a class="markup--anchor markup--p-anchor" data-href="https://www.ukfinance.org.uk/sites/default/files/uploads/pdf/UK-Finance-UK-Payment-Markets-Report-2019-SUMMARY.pdf" href="https://www.ukfinance.org.uk/sites/default/files/uploads/pdf/UK-Finance-UK-Payment-Markets-Report-2019-SUMMARY.pdf" rel="noopener noreferrer" target="_blank">
    显示
   </a>
   ，到2023年，信用卡交易将与现金交易的比例为大约四比一，这意味着2020年正是这一突然而深刻的转变的半程。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这将如何影响人们的生活？
   </strong>
  </h3>
  <p class="graf graf--p">
   现金流失的巨大后果
   <strong class="markup--strong markup--p-strong">
    不仅
   </strong>
   在于从根本上改变了我们的支出方式。
  </p>
  <p class="graf graf--p">
   ⚠️希望保护私人行为隐私的人会发现自己被严重边缘化、举步维艰。您将被剥夺很多选择。（目前英国有超过一百万的成年人没有银行帐户）
  </p>
  <p class="graf graf--p">
   澳大利亚
   <a class="markup--anchor markup--p-anchor" data-href="https://www.dss.gov.au/families-and-children/programmes-services/welfare-conditionality/cashless-debit-card-overview" href="https://www.dss.gov.au/families-and-children/programmes-services/welfare-conditionality/cashless-debit-card-overview" rel="noopener noreferrer" target="_blank">
    已经开始尝试
   </a>
   通过电子支付卡支付一些福利，这严重限制了社会上一些最贫困的人获得现金的机会。这些功能类似于标准的 Visa 或 MasterCard，并通过相同的网络进行处理，但它们阻止用户将钱花在酗酒或赌博上。
  </p>
  <p class="graf graf--p">
   ⚠️简单说，政府不想让你把钱花在哪，他们就可以直接轻松地做到。
  </p>
  <p class="graf graf--p">
   即：
   <strong class="markup--strong markup--p-strong">
    政府、雇主或房东都可以使用电子支付系统及其收集的数据作为控制、研究、和操纵人们行为的工具
   </strong>
   。
  </p>
  <p class="graf graf--p">
   长期无现金状态将如何发展？要知道这并不是新的付款方式第一次取代旧的付款方式：很久以前法定货币就取代了贵金属，但是，黄金从未消失过，它不再是主要的价值交换方法，而是一种存储资金的方法。现金的未来也会是这样吗？
  </p>

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     3、21世纪工会的兴起
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   先关联一篇4月发布的分析，如果您错过了的话：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%8a%80%e6%9c%af%e5%a6%82%e4%bd%95%e5%a1%91%e9%80%a021%e4%b8%96%e7%ba%aa%e7%9a%84%e5%88%9b%e9%80%a0%e6%80%a7%e6%8a%97%e8%ae%ae%ef%bc%9f/" href="https://www.iyouport.org/%e6%8a%80%e6%9c%af%e5%a6%82%e4%bd%95%e5%a1%91%e9%80%a021%e4%b8%96%e7%ba%aa%e7%9a%84%e5%88%9b%e9%80%a0%e6%80%a7%e6%8a%97%e8%ae%ae%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     技术如何塑造21世纪的创造性抗议？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   已经有迹象表明，新的集体行动在数字工具的支持下，白领和蓝领工人再次积极地加入了激进主义团体和组织。2018年，世界各地的Google员工联合抗议该公司对女性员工的骚扰；优步司机一直在努力抗议以增加工资。
  </p>
  <p class="graf graf--p">
   在越来越不确定的劳动力市场中，激进主义的抗议活动正在增加。技术变革一直影响着劳动世界，人们的工作被重塑以使雇主能够充分利用新的创新。
  </p>
  <p class="graf graf--p">
   但是这种变化现在正以前所未有的高速度发生。在2012年至2017年之间，已经有
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cedefop.europa.eu/en/publications-and-resources/publications/9121" href="https://www.cedefop.europa.eu/en/publications-and-resources/publications/9121" rel="noopener noreferrer" target="_blank">
    近一半
   </a>
   的工人由于技术的改变而经历了工作方式的变化。在经历了一个世纪的低会员人数之后，工会大会（TUC）2019年
   <a class="markup--anchor markup--p-anchor" data-href="https://www.tuc.org.uk/blogs/trade-union-membership-rises-100000-single-year-challenges-remain" href="https://www.tuc.org.uk/blogs/trade-union-membership-rises-100000-single-year-challenges-remain" rel="noopener noreferrer" target="_blank">
    报告称
   </a>
   ，2017年至2018年期间，英国的工会会员人数增加了100000。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    技术的善与恶
   </strong>
  </h3>
  <p class="graf graf--p">
   当然，变化既可以是积极的，也可以是消极的。在某些地方，已经开发出新技术以减轻工作生活的压力，并赋予人们以适合自己的方式工作的灵活性。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2017/12/27/business/the-robots-are-coming-and-sweden-is-fine.html" href="https://www.nytimes.com/2017/12/27/business/the-robots-are-coming-and-sweden-is-fine.html" rel="noopener noreferrer" target="_blank">
    在瑞典
   </a>
   ，机器人被用于执行危险和肮脏的工作，例如采矿。在英国，在一个配送中心中，
   <a class="markup--anchor markup--p-anchor" data-href="https://fabians.org.uk/wp-content/uploads/2019/09/Commission-on-Workers-and-Technology-Briefing-Note-FINAL-COVER-min-1.pdf" href="https://fabians.org.uk/wp-content/uploads/2019/09/Commission-on-Workers-and-Technology-Briefing-Note-FINAL-COVER-min-1.pdf" rel="noopener noreferrer" target="_blank">
    已使用
   </a>
   小型自动驾驶汽车来缩短工人每天必须步行的距离。在全球范围内，在线平台可以为人们提供机会，使他们可以找到遥远异地的有偿工作。
  </p>
  <p class="graf graf--p">
   但在其他工作场所中，技术被用来通过跟踪监视工人和设定难以实现的目标来提高效率。这给工人带来了严重的压力，结果是，许多部门的工人都开始寻求工会可以提供的保护和支持。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e7%9a%84%e8%80%81%e6%9d%bf%e5%8f%af%e8%83%bd%e6%98%af%e6%af%94-facebook%e3%80%81%e8%b0%b7%e6%ad%8c%e6%9b%b4%e5%a4%a7%e7%9a%84%e8%80%81%e5%a4%a7%e5%93%a5/" href="https://www.iyouport.org/%e4%bd%a0%e7%9a%84%e8%80%81%e6%9d%bf%e5%8f%af%e8%83%bd%e6%98%af%e6%af%94-facebook%e3%80%81%e8%b0%b7%e6%ad%8c%e6%9b%b4%e5%a4%a7%e7%9a%84%e8%80%81%e5%a4%a7%e5%93%a5/" rel="noopener noreferrer" target="_blank">
     你的老板可能是比 Facebook、谷歌更大的老大哥
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   工会在此方面至关重要。迫切需要有效的工会，可以放大工人的声音来影响雇主的决定。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    转机潮
   </strong>
  </h3>
  <p class="graf graf--p">
   为21世纪做好准备的工会已经出现。
  </p>
  <p class="graf graf--p">
   欧洲许多人正在接受可以为工人谋福利的技术。在斯堪的纳维亚半岛，工会一直是在雇主、工人和技术之间建立相互支持关系的驱动力。
  </p>
  <p class="graf graf--p">
   瑞典的 TCO 工会
   <a class="markup--anchor markup--p-anchor" data-href="https://futurion.se/futurion-in-english/" href="https://futurion.se/futurion-in-english/" rel="noopener noreferrer" target="_blank">
    预计
   </a>
   数字化将导致劳动力市场的变化、丹麦的3F工会与 Hilfr（家庭清洁员的数字平台）达成协议，以便使用该站点的工人可以获得假期和病假等福利。
  </p>
  <p class="graf graf--p">
   在英国，工人一直在使用技术来重塑影响工作未来的力量。近年来，Organise、Earwig 和 Workerbird 等新的 “WorkerTech” 公司一直在使用在线工具来共享有关工作场所的信息，建立员工的声音并开展运动以改善条件。
  </p>
  <p class="graf graf--p">
   英国已建立的工会现在正在发挥领导作用，工会大会（TUC）发起了一项雄心勃勃的运动，利用自动化的生产力来推动每周四天的工作日。
  </p>
  <p class="graf graf--p">
   其他新兴的21世纪工会还包括
   <a class="markup--anchor markup--p-anchor" data-href="https://www.indycube.community/?option=com_content&amp;view=article&amp;id=28&amp;Itemid=248" href="https://www.indycube.community/?option=com_content&amp;view=article&amp;id=28&amp;Itemid=248" rel="noopener noreferrer" target="_blank">
    社区工会
   </a>
   ，这是一个较小的工会联盟，专门为解决英国自雇工人的需求而成立的。
   <a class="markup--anchor markup--p-anchor" data-href="https://iwgb.org.uk/page/about/about" href="https://iwgb.org.uk/page/about/about" rel="noopener noreferrer" target="_blank">
    英国独立工人联盟（IWGB）
   </a>
   是另一个相对较新的工会，代表低薪移民工人，例如清洁工和保安人员。
  </p>
  <p class="graf graf--p">
   2020年，随着自动化和不断变化的工作世界成为全球议事日程，工会的发展将变得更加快速。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这将如何影响人们的生活？
   </strong>
  </h3>
  <p class="graf graf--p">
   技术和自动化、结合服务类经济的增长和不稳定的工作，正在改变社会当前围绕工作而构建的方式。从事传统白领工作（例如会计工作）的人们也将很快开始看到其工作生活的巨大变化。
  </p>
  <p class="graf graf--p">
   这些变化将不可避免地导致混乱和失业，这意味着工会在确定可能失业的工人得到多大的技能再培训方面起着重要作用。
  </p>
  <p class="graf graf--p">
   此外，新的21世纪工会可以帮助塑造技术和自动化在工作场所中的应用方式，倡导赋予工人权力的选择，增加对人工能力的需求并提高生产力。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12270 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/4-scaled.jpeg?resize=1100%2C619&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/4-scaled.jpeg?w=2048&amp;ssl=1 2048w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/4-scaled.jpeg?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/4-scaled.jpeg?resize=1024%2C576&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/4-scaled.jpeg?resize=768%2C432&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/4-scaled.jpeg?resize=1536%2C864&amp;ssl=1 1536w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/4-scaled.jpeg?resize=1100%2C619&amp;ssl=1 1100w" data-recalc-dims="1" height="619" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/4-scaled.jpeg?resize=1100%2C619&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

    <img alt="" class="aligncenter size-full wp-image-12270" data-recalc-dims="1" height="619" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/4-scaled.jpeg?resize=1100%2C619&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/4-scaled.jpeg?w=2048&amp;ssl=1 2048w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/4-scaled.jpeg?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/4-scaled.jpeg?resize=1024%2C576&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/4-scaled.jpeg?resize=768%2C432&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/4-scaled.jpeg?resize=1536%2C864&amp;ssl=1 1536w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/4-scaled.jpeg?resize=1100%2C619&amp;ssl=1 1100w" width="1100"/>

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     4、如何保护自己免受数字暗杀
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   如今人们的一生都存储在互联网上。
  </p>
  <p class="graf graf--p">
   通过我们的 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    开源情报
   </a>
   ” 演示您足够可以发现，找到有关某人的最私密的个人信息可以非常容易 —— 尤其是如果他们没有采取积极措施来保护其数据时。
  </p>
  <p class="graf graf--p">
   依旧有大多数人人习惯性泄露自己已经用于安全保护的信息：例如 电话号码、宠物的名字、自己的生日、曾经居住过的街道等等。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e4%b8%8d%e5%ba%94%e8%af%a5%e5%9c%a8%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e4%b8%8a%e8%af%b4%e7%9a%84%e8%af%9d/" href="https://www.iyouport.org/%e4%bd%a0%e4%b8%8d%e5%ba%94%e8%af%a5%e5%9c%a8%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e4%b8%8a%e8%af%b4%e7%9a%84%e8%af%9d/" rel="noopener noreferrer" target="_blank">
     你不应该在社交媒体上说的话
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   不仅自满，而且许多人还没有采取任何行动来保护自己。几乎一半的英国人从未更改过他们的电子邮件密码。
  </p>
  <p class="graf graf--p">
   这种自满会给你带来很大的危机。犯罪分子可以利用您的个人信息偷走你的钱；他们还可以做一些可能更糟的事：
   <strong class="markup--strong markup--p-strong">
    数字化暗杀
   </strong>
   。
  </p>
  <p class="graf graf--p">
   他们可以接管你的全套身份，将真实的您本人从电子邮件、社交媒体、甚至财务帐户中踢出去，从而完全劫持并杀死您的数字生活。
  </p>
  <p class="graf graf--p">
   2020年，随着越来越多的人通过高调的泄漏丑闻和亲身经历攻击事件中意识到数字暗杀有多可怕，个人网络安全保险市场将开始迅速增长 —— 它将从少数知名人士和富人购买的产品发展到对普通人来说越来越负担得起的产品。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    数字刺客的崛起
   </strong>
  </h3>
  <p class="graf graf--p">
   网络犯罪影响着每个人。从国家政府到企业，再到个人，没有例外。骇客、trolls、和网军使用的工具越来越复杂、广泛可用、并且相对易于使用。
  </p>
  <p class="graf graf--p">
   结果就是，我们看到了越来越多的数字暗杀事件的发生，形式越来越多样化。
  </p>
  <p class="graf graf--p">
   数字刺客的动机也多种多样。对于某些人来说，他们只是想要钱；入侵受害者可以使他们有能力索取现金、或获得有价值的社交媒体资源。
  </p>
  <p class="graf graf--p">
   ⚠️ 而一些数字刺客是受薪专业人士，由政府间谍机构雇用来侵入人权组织和活动家，并公开其高度敏感的私人数据。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    更多详见 列表 -5 “
    <a class="markup--anchor markup--li-anchor" data-href="https://start.me/p/1kod2L/iyp-direct-action5" href="https://start.me/p/1kod2L/iyp-direct-action5" rel="noopener noreferrer" target="_blank">
     维稳面面观
    </a>
    ”
   </li>
  </ul>
  <p class="graf graf--p">
   尽管规模可能会有所不同，但是方法仍然很相似：即 发现并利用安全漏洞来侵入个人帐户、窃取其数据并劫持其身份，以试图在网上销毁这些致命数据。
  </p>
  <p class="graf graf--p">
   在2019年5月，《卫报》报道了一个特别残酷的数字暗杀案，一名年轻女子的社交媒体帐户被黑。她的裸照自拍被发送给了她的所有联系人，并且她自己被永久锁定在帐户之外，包括电子邮件。
  </p>
  <p class="graf graf--p">
   这次暗杀事件是完全彻底的 —— 受害人无法恢复任何帐户。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在繁荣的风口浪尖上
   </strong>
  </h3>
  <p class="graf graf--p">
   随着越来越多的人感受到数字暗杀的影响，他们将开始寻求保护。
  </p>
  <p class="graf graf--p">
   2019年1月，Swiss Re 保险公司发布了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.ft.com/content/dff25a64-1e65-11e9-b2f7-97e4dbd3580d" href="https://www.ft.com/content/dff25a64-1e65-11e9-b2f7-97e4dbd3580d" rel="noopener noreferrer" target="_blank">
    一项研究
   </a>
   ，显示一半的受访者会考虑购买个人网络保险。他们估计，到2025年，该产品的全球市场可能达到31亿美元。
  </p>
  <p class="graf graf--p">
   这些预测在过去曾经是错误的。 2000年，保守的预测是，2005年全球网络保险市场价值25亿美元，但是到了2008年，该市场的价值仅为5亿美元。
  </p>
  <p class="graf graf--p">
   但是，十年过去了，在线世界已经是一个截然不同的地方。人们对网络犯罪构成的威胁的性质和规模有了更好的了解。
  </p>
  <p class="graf graf--p">
   保险公司正在慢慢开始推出相应的产品。 2015年，一家为高净值个人和家庭服务的保险公司宣布了类似产品。2019年，国营农场保险服务
   <a class="markup--anchor markup--p-anchor" data-href="https://www.statefarm.com/insurance/identity-restoration" href="https://www.statefarm.com/insurance/identity-restoration" rel="noopener noreferrer" target="_blank">
    推出了
   </a>
   身份盗用保险，6月，哈特福德蒸汽锅炉监测和保险公司
   <a class="markup--anchor markup--p-anchor" data-href="https://www.insurancejournal.com/news/national/2016/06/14/416887.htm" href="https://www.insurancejournal.com/news/national/2016/06/14/416887.htm" rel="noopener noreferrer" target="_blank">
    也推出了
   </a>
   HSB 家庭网络保险，该产品以前仅适用于企业。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这将如何影响人们的生活？
   </strong>
  </h3>
  <p class="graf graf--p">
   尽管网络保险市场目前仅是少数偏执的百万富翁光顾的边缘行业，但是在新10年里，它将增长成为任何家庭保险计划的标准组成部分。
  </p>
  <p class="graf graf--p">
   但是，这种保险将保护您免受什么伤害，如果您面临数字攻击，它将为您做什么？
  </p>
  <p class="graf graf--p">
   从规模上看，如果您的个人笔记本电脑感染了恶意软件、并且您需要向公司付款以删除病毒，那么它保护您的费用每年可能为25英镑。
  </p>
  <p class="graf graf--p">
   在规模的另一端，对于那些知名度很高的人，其中可能包括资格标准和对消费者的背景调查。调查后保险公司的隐私权专家可能会发送一份您必须对自己的生活方式进行调整的列表，例如 经常更改您的电话号码和密码。
  </p>
  <p class="graf graf--p">
   为了获得更高的安全性，预计客户会牺牲一定程度的便利性来换取保护。
  </p>
  <p class="graf graf--p">
   您可能必须使用物理身份验证USB密钥登录帐户，或者被要求加密整个硬盘驱动器，甚至可能必须使用法拉第笼来防止移动设备受到无线黑客攻击。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    在我们的
    <a class="markup--anchor markup--li-anchor" data-href="https://start.me/p/xbYXdR/iyp-1" href="https://start.me/p/xbYXdR/iyp-1" rel="noopener noreferrer" target="_blank">
     列表-1
    </a>
    中看到
    <strong class="markup--strong markup--li-strong">
     自我保护的技巧汇总
    </strong>
    （不断更新中）
   </li>
  </ul>
  <p class="graf graf--p">
   ⚠️但是，保险产品可能产生相反的效果，反而导致人们在网上更加不小心。人们可能过度依赖网络保险套餐，而
   <strong class="markup--strong markup--p-strong">
    网络保险本身反过来可能会变得越来越具有侵入性。
   </strong>
  </p>
  <p class="graf graf--p">
   最终，我们可能将越来越多的私人数据和控制权移交给一家私人保险公司，虽然不是现在的政府和寡头了。却又有什么不同吗？
   <strong class="markup--strong markup--p-strong">
    《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%8a%8a%e4%bd%a0%e7%9a%84%e9%9a%90%e7%a7%81%e6%8e%a7%e5%88%b6%e6%9d%83%e4%ba%a4%e7%bb%99%e4%bb%bb%e4%bd%95%e4%ba%ba%e9%83%bd%e7%ad%89%e4%ba%8e%e5%89%a5%e5%a4%ba%e9%9a%90%e7%a7%81/" href="https://www.iyouport.org/%e6%8a%8a%e4%bd%a0%e7%9a%84%e9%9a%90%e7%a7%81%e6%8e%a7%e5%88%b6%e6%9d%83%e4%ba%a4%e7%bb%99%e4%bb%bb%e4%bd%95%e4%ba%ba%e9%83%bd%e7%ad%89%e4%ba%8e%e5%89%a5%e5%a4%ba%e9%9a%90%e7%a7%81/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     把你的隐私控制权交给任何人都等于剥夺隐私
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》。
   </strong>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     最后
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   这些预测也许是最有可能发生的事。它们各有利弊。以预测作为提醒大众警惕的一种方式，也许我们可以改变未来。⚪️
  </p>
