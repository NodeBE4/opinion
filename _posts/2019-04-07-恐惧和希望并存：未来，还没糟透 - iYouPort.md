  <p class="graf graf--p graf-after--figure" id="b60b">
   展望我们最狂野的内心或最深切的恐惧所定义的未来是一种与人类文化一样古老的本能。但随着人们正在全球范围内应对深刻的范式转变，它在近年来已经呈现出新的紧迫性。
  </p>
  <p class="graf graf--p graf-after--p" id="0a29">
   我们生活在历史上最具社会关联性和技术先进性的世界中，我们正在发现遥远的系外行星，设计模仿我们思考方式的算法，并熟练掌握生命的遗传语言。这其中哪些是噩梦？哪些是希望？
  </p>
  <p class="graf graf--p graf-after--p" id="429e">
   为了解释这些问题，Motherboard 向105名跨学科思想家询问了他们对未来的恐惧和希望。参与者包括（但不限于）19名空间科学家，19名生物学家和环境科学家，11名计算机科学家，8名医学科学家，5名律师，4名历史学家，一名音乐学家，一名古生物学家，一名宇航员和一位回应表情符号的数字艺术家。
  </p>
  <p class="graf graf--p graf-after--p" id="3399">
   大多数参与者都在美国（69人），但也收到了来自欧洲（20人），澳大利亚（5），加拿大（4），新加坡（2），南非（2），台湾（ 1），印度（1）和孟加拉国（1）的回复；包括 56名女性和49名男性。
  </p>
  <p class="graf graf--p graf-after--p" id="b491">
   对所有 105 名参与者都提出了这两个问题：你对未来最担心的是什么？是什么让你对未来充满希望？他们的答复如下所列。
  </p>
  <p class="graf graf--p graf-after--p" id="cabc">
   回应中出现了统一的想法。到目前为止，最常被提到的担忧是气候变化（29），其次是政治极端主义（21）的飙升；人工智能，特别是其偏见和不可预测性，是另一个共同关注的问题（10）；经常被提到的是虚假信息的扩散（8）和对科学和 STEM 教育的投入不足（8）。
  </p>
  <p class="graf graf--p graf-after--p" id="4a0f">
   WALL-E、终结者、疯狂麦克斯、Fury Road、Elysium 和 Black Mirror 可以作为未来恐惧的参考。三位受访者，两位天文学家和一位考古学家表示，他们并不担心未来。
  </p>
  <p class="graf graf--p graf-after--p" id="2254">
   大多数参与者将人类的聪明才智和集体行动作为希望和灵感的源泉。其中 26 人特别提到年轻一代让他们感觉乐观（没有压力）。Seven 表示，女性权利的提升给了她们希望，三人提到了#MeToo 运动。
  </p>
  <p class="graf graf--p graf-after--p" id="e24f">
   所有 105 位受访者都对人类的未来 — — 好的、坏的、丑的和非常奇怪的 — — 提供了宝贵的观点。希望这些意见能为您带来启发性。
  </p>
  <p class="graf graf--p graf-after--p" id="7b57">
   <strong class="markup--strong markup--p-strong">
    未来主义者和历史学家（10）
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="ac87">
   最恐惧的事：
   <strong class="markup--strong markup--p-strong">
    我们被意见打败，再也看不到共同的现实。学习批判性思维似乎已经不足解决如此严重的问题
   </strong>
   。虽然气候变化将导致海平面上升和天气稳定性增加，但新的研究表明恶性系统效应更为严重。我们放弃了我们的未来，很少有人关心。
  </p>
  <p class="graf graf--p graf-after--p" id="7566">
   希望：越来越重视在更广泛的社会中承认原住民文化的重要性。可以在同一个地点彼此采用不同的方式。
  </p>
  <p class="graf graf--p graf-after--p" id="3ae3">
   最恐惧的事：越来越多的
   <strong class="markup--strong markup--p-strong">
    错误观念
   </strong>
   认为，更多的科学和技术可以解决我们社会最大的问题，这些问题是社会和文化的核心。
  </p>
  <p class="graf graf--p graf-after--p" id="b06e">
   希望：人类，以及人类可以做到的最好的想法，具有极大的弹性。
  </p>
  <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote graf-after--p" id="eb12">
   <p>
    <em class="markup--em markup--blockquote-em">
     “Climate change, an apparently insane American president, and a set of British politicians that want to turn the UK into an island that emerges from the mist every 100 years”
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p graf-after--blockquote" id="17ea">
   最恐惧的事：《黑镜》那种技术深入的后果，令我担心的是
   <strong class="markup--strong markup--p-strong">
    生活模仿艺术和艺术模仿生活
   </strong>
   。即使我们有能力设想冲突以及道德和道德困境，我们仍然会对权力、资源和能源做出
   <strong class="markup--strong markup--p-strong">
    短视的决定
   </strong>
   ，这些决定是不公平的和不可持续的。当我们在没有同理心的情况下行动，没有自我意识的创新，没有洞察力或智慧的力量，最终我们会把自己变成自我的奴隶。
  </p>
  <p class="graf graf--p graf-after--p" id="4b08">
   希望：我们能够想象一个新的未来，概念化和探索新的真理。艺术家接受培训，以便将原创想法可视化，创造可以引导我们创造力的语话。是时候审视我们的经济、政治和宗教制度了，这些系统适用于所有人吗？如果不是，我们如何使用交叉实践来创建和谐、周到、充满同理心和联系的解决方案？
  </p>
  <p class="graf graf--p graf-after--p" id="71bd">
   最恐惧的事：科学家、艺术家和作家被焚烧，暴民欢呼。
  </p>
  <p class="graf graf--p graf-after--p" id="de96">
   希望
   <strong class="markup--strong markup--p-strong">
    ：
   </strong>
   全世界公民科学运动的兴起。
  </p>
  <p class="graf graf--p graf-after--p" id="71e6">
   最恐惧的事：在“西方世界”中显而易见的趋势，即走向
   <strong class="markup--strong markup--p-strong">
    衰老、偏执、无聊、政治正确
   </strong>
   的僵尸老年社会。
  </p>
  <p class="graf graf--p graf-after--p" id="8a06">
   希望：人类在地球上做了很多美好的事，并且可以继续在其他星球做更多奇妙的事，只要我们保持健康的无限、不敬和不合理的乐观主义储备。
  </p>
  <p class="graf graf--p graf-after--p" id="2252">
   最恐惧的事：在我们的一生中，我们会看到人工智能反映了我们人类的缺点。我们会发现自己被困在一个不可思议的山谷中 — — 生活几乎正常，但没有什么能让人感觉是完全正确的。我们将沮丧地诅咒制度化的阶级主义、仇外心理、性别歧视、同性恋恐惧症、和种族主义体现在我们制造的机器身上。
  </p>
  <p class="graf graf--p graf-after--p" id="591f">
   希望：未来还没有发生。我们现在正在共同创造未来，
   <strong class="markup--strong markup--p-strong">
    这意味着今天活着的每个人都有机会对人工智能的未来发挥有益的作用
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="35bd">
   最恐惧的事：
   <strong class="markup--strong markup--p-strong">
    傲慢的技术、知识分子、政治。我们不知道我们自己有多无知
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="1c87">
   希望：对我们政治时刻的正义的愤怒正在推动一股强大的激进主义和组织浪潮。包括科学家和技术专家在内的许多人都在倡导正义而不是权力。我想相信善意会赢。
  </p>
  <p class="graf graf--p graf-after--p" id="fb40">
   <strong class="markup--strong markup--p-strong">
    计算机科学家（11）
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="0b76">
   最恐惧的事：大规模的系统故障导致了对专家的普遍不信任，不恰当的责备，以及阴谋论的泛滥，这些阴谋论为其他神秘事件提供了简单但错误的解释。如果基于这些粗略的过度简化制定政策，我们将看到更多的灾难，系统故障和可怕的意外后果。
  </p>
  <p class="graf graf--p graf-after--p" id="bae7">
   希望：如果我们设计的系统具有灵活性而非最优性，我们就有能力缓解这些问题。我们应该选择并培养那些擅长将复杂概念传达给负担过重的决策者的
   <strong class="markup--strong markup--p-strong">
    跨学科思想家
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="b8fc">
   最恐惧的事：当我们开发模拟我们思考方式的算法表时，我们最终将[对后代]传递作为人类最有缺陷的判断方式。算法偏见现在是人工智能中的一个真正的问题，其破坏性就像瘟疫一样。在开发人员的头脑中持有的有意识和无意识的仇恨、蔑视甚至是偶然的陈规定型观念，现在很容易进入代码库。当我们将思维转移到机器上时，思维中的这些缺陷会无休止地延续下去并且越来越不会受到后代的挑战。
  </p>
  <p class="graf graf--p graf-after--p" id="4fb1">
   希望：看到像 Renee Teate，VincenteOrdóñezRomán 和 Ines Montani 这样的如此不可思议的多元化头脑，是明确的希望。技术可以使我们更紧密地联系在一起，增进我们的关系，并将我们与新老朋友联系起来。
   <strong class="markup--strong markup--p-strong">
    科技不会取代我们的关系; 它必须起到增强的作用
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="4387">
   恐惧：过度使用互联网和在线平台会对家庭生活产生负面影响。家庭成员更少花时间闲聊，看电视，一起吃饭和度假。
  </p>
  <p class="graf graf--p graf-after--p" id="82e4">
   希望：技术的进步及其解决现实问题的能力，从而提高人类生活质量。
  </p>
  <p class="graf graf--p graf-after--p" id="7ccf">
   恐惧：一切都变成了计算机，一切都被联网。
   <strong class="markup--strong markup--p-strong">
    我们想要防止这种情况已经无能为力。只要一个通电通电，它就会在互联网上
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="621a">
   希望：机器学习和人工智能的发展。我相信
   <strong class="markup--strong markup--p-strong">
    机器学习对于防守比进攻更有用。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="51b2">
   恐惧：人工智能算法缺乏透明度和问责制，而我们正越来越多地将其用于所有行业和政府服务。
  </p>
  <p class="graf graf--p graf-after--p" id="fa13">
   希望：亚洲、非洲和拉丁美洲各城市新兴中产阶级获得基本服务（如教育，医疗保健，流动性和安全性）的民主化。
  </p>
  <p class="graf graf--p graf-after--p" id="2812">
   恐惧：缺乏关于人工智能和无人系统的理性决策。关于人工智能的过度炒作导致美国各州忽视已知的公共安全风险，只希望获得企业投资。
  </p>
  <p class="graf graf--p graf-after--p" id="5cfd">
   希望：计算和机器人技术的可能性将导致有一天不再发生灾难 — 因为灾难可以预测和准备，响应和恢复是即时和无缝的。
  </p>
  <p class="graf graf--p graf-after--p" id="d338">
   恐惧：人们会相信软件为我们做出重大的生死决定（疾病诊断！战争！汽车刹车！），而没有适当的人为监督。
  </p>
  <p class="graf graf--p graf-after--p" id="f130">
   希望：我们在解决研究中的计算机科学问题方面取得了令人难以置信的进步。如果我们能够将这些想法的一小部分转化为现实世界的解决方案，我们将会处于良好的状态。
  </p>
  <p class="graf graf--p graf-after--p" id="4479">
   <strong class="markup--strong markup--p-strong">
    艺术家和作家（7）
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="06da">
   恐惧：人类的冷漠。
  </p>
  <p class="graf graf--p graf-after--p" id="f52f">
   希望：人类的聪明才智。
  </p>
  <p class="graf graf--p graf-after--p" id="d883">
   恐惧：物种灭绝，以每五分钟一个生物体的速度灭绝。其中大部分是由于人类的活动和过度发展。作为一名建筑师和城市设计师，我要对这种讽刺感负责。我们需要通过一切必要的手段来制止灭绝。
  </p>
  <p class="graf graf--p graf-after--p" id="b430">
   希望：人类是这个问题的罪魁祸首，因此人类有能力阻止它。
  </p>
  <p class="graf graf--p graf-after--p" id="daf8">
   恐惧：我们生活在一个极其充满激情的时代，我们面临着气候变化，不断加剧的社会不平等，冲突以及法西斯政权的崛起。所有这些都与当前的权力和统治结构密不可分。让我最担心的是放弃我们的想象力，我们的创造力，我们共同合作的人类能力，谈判，争论和集体讨论恐惧的能力。
   <strong class="markup--strong markup--p-strong">
    恐惧到处都是敌人。它破坏了信任，它扼杀了想象力。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="61c5">
   希望：
   <strong class="markup--strong markup--p-strong">
    如果我们不寻求希望，而是寻求意义呢？因此，无论事情是否有希望，我们都可以选择以在地球上成为人类意味着什么的伦理考虑为基础，包括共同努力
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="7e1e">
   <strong class="markup--strong markup--p-strong">
    律师和政治学家（8）
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="dc99">
   最恐惧的事：全球重心已经右转，特朗普政权及其盟友所造成的国家性和全球性破坏将使我们重新站稳脚的希望变得异常艰难。
  </p>
  <p class="graf graf--p graf-after--p" id="68c6">
   希望：青年运动。年轻人正在重塑激进主义和民主，寻找彻底的新方法来理解和解决长期存在的不公正现象。
  </p>
  <p class="graf graf--p graf-after--p" id="d803">
   最恐惧的事：随着气候变化并加速了流民的迁徙，将出现反对移民和不同民族之间频繁发生冲突的黑暗时代。
  </p>
  <p class="graf graf--p graf-after--p" id="9734">
   希望：从基本收入支付的试验到使可再生能源在市场上运作良好的方式，世界各地正在进行的所有实验都将发生集体进步的可能性。
  </p>
  <p class="graf graf--p graf-after--p" id="6f47">
   最恐惧的事：
   <strong class="markup--strong markup--p-strong">
    我们太在乎别人如何看待我们了。我们快乐的心情和情绪好坏的可能性将越来越多地与其他人对我们的看法联系在一起
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="b403">
   希望：社交媒体和通信技术将有助于分配机会和知识。
  </p>
  <p class="graf graf--p graf-after--p" id="4ada">
   最恐惧的事：合法化的自相残杀 — 我们将从根本上扩大生命采购和商业化实践的可能性，包括人体细胞和组织。最初，知情同意将用于促进将这些原材料转移到企业实体，这些企业开采、资本化和重新安排它们以获取利润；之后，为了有效转移，知情同意和隐私将被搁置。
  </p>
  <p class="graf graf--p graf-after--p" id="736c">
   希望：激进的创造力 — 下一代思想家正在超越纪律和其他界限，这些界限载有我们如何想象我们的未来以及我们将如何走向未来的道路。
  </p>
  <p class="graf graf--p graf-after--p" id="9c66">
   最恐惧的事：数字技术是对新闻业的一种生存威胁，正如我们在20世纪所知道的那样，当与独立媒体的巨大政治压力相结合时，在信息质量和信息平等方面，自由表达的许多基础设施可能指向了一个黑暗的地方。
  </p>
  <p class="graf graf--p graf-after--p" id="861d">
   希望：
   <strong class="markup--strong markup--p-strong">
    年轻人既不害怕也不敬畏数字技术，并且相比过去几代人都能更好地利用技术装备自己，充分利用技术，对集中化技术背后的巨头公司持怀疑态度，而不是自满或疯狂
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="2d18">
   <strong class="markup--strong markup--p-strong">
    人文社会科学家（15）
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="7625">
   最恐惧的事：我们是否会被人工智能僵化，我们已经在无意中通过编程以操纵人类自己？这些算法正在演变为比人类更好的心理学家，我们实际上并没有办法预测对人类大脑的这种大规模操纵将如何影响我们的未来。
  </p>
  <p class="graf graf--p graf-after--p" id="7d1a">
   希望：人类当务之急是创新，但有时创新也需要监管，以防止产生有害影响。许多行业现在已经将挑战和责任放在自己的手中，这有可能限制新技术的负面后果。
  </p>
  <p class="graf graf--p graf-after--p" id="4421">
   最恐惧的事：伪装成民主的专制正在消除人民的政治权利。公民的斗争应该在何处努力以及如何体现为更加平等的社会？
  </p>
  <p class="graf graf--p graf-after--p" id="0310">
   希望：平等和权利很少被发放，所以单靠希望是行不通的。当然，人类有能力设计其他形式的民主参与，以更公平地分配公共利益。
  </p>
  <p class="graf graf--p graf-after--p" id="4154">
   最恐惧的事：就像我们面对物种20万年来最大规模的生存威胁一样，我们越来越无法集体解决人类的问题。由于贪婪、恐惧和仇恨的原子化和容易操纵，我们对更大利益的想法失去了信心。换句话说，就在我们最需要天使的时候，天使似乎正在放弃我们。
  </p>
  <p class="graf graf--p graf-after--p" id="34d4">
   希望：尽管如此，生存是我们最根深蒂固的本能，其次是爱。贪婪和仇恨是偏误，最终需要付出太多努力才能实现可持续发展。
  </p>
  <p class="graf graf--p graf-after--p" id="5b1d">
   最恐惧的事：未来或多或少与现在相同。
  </p>
  <p class="graf graf--p graf-after--p" id="65f0">
   希望：现在我能想到的一切在未来都有可能是愚蠢的和错误的。
  </p>
  <p class="graf graf--p graf-after--p" id="90af">
   最恐惧的事：
   <strong class="markup--strong markup--p-strong">
    不断上升的民族主义以及世界各地的政治孤立和经济隔离
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="2179">
   希望：作为一名考古学家经常让自己后仰，深呼吸，放松。文明的结束经常被宣布，但很少发生。所以，老实说我对未来并不太担心。
  </p>
  <p class="graf graf--p graf-after--p" id="5b86">
   最恐惧的事：政治的两极分化一直在加剧，特别是经济和意识形态，增强了我们之间的分离。这使我们在迫切需要聚集在一起解决气候变化等紧急问题时，更难以相互交流和理解。
  </p>
  <p class="graf graf--p graf-after--p" id="560b">
   希望：女性正在崛起，尤其是与女性相关的品质，如包容和协作能力。男性正在接受他们内心存在的女性化的一面，以及父权制解体的明显证据（尽管有强烈反对）。
  </p>
  <p class="graf graf--p graf-after--p" id="2361">
   最恐惧的事：由于我们已经接近100亿人口，富人大量过度消费，穷人想要并追求更多消费，我们面临着气候变化，粮食不安全，水资源紧张，生物多样性丧失等问题。
  </p>
  <p class="graf graf--p graf-after--p" id="cba4">
   希望：我的学生和世界各地的年轻人正在为保护环境作出努力，他们的行动每天都激励着我。
  </p>
  <p class="graf graf--p graf-after--p" id="0839">
   最恐惧的事：许多事情似乎超出了我们的控制范围，学校里的枪支、不平等、虚假消息的泛滥、成瘾问题、分裂的言论、仇恨……我担心因为问题太大，我们会不会停止尝试。
  </p>
  <p class="graf graf--p graf-after--p" id="52bf">
   希望：年轻人。他们是承诺，参与，包容和理智（大多数情况下）。
  </p>
  <p class="graf graf--p graf-after--p" id="7245">
   最恐惧的事：建设未来在许多方面依赖于对过去的深刻理解。我们已经进入了一个由实时连续和短暂的信息流所决定的时代，我们将对文明的记忆转移到了数字和私人系统中，而这些系统不能保证长期的持久性。试着将它与大金字塔进行比较，这是一个古老国家的证词，已经存活了4500年！
  </p>
  <p class="graf graf--p graf-after--p" id="848f">
   希望：人类惊人的能力经常与悲观的专家和大师们的既定“真理”和预言相矛盾，并通过新的想法、发现和创新来推动我们的文明向前发展。
  </p>
  <p class="graf graf--p graf-after--p" id="b23c">
   最恐惧的事：我们与差异的持续斗争。
   <strong class="markup--strong markup--p-strong">
    我们对看起来与自己不同的人的愤怒、恐惧、焦虑和蔑视，正在削弱并继续破坏我们的民主、我们的进步，以及我们挖掘最佳思想和提供可以推动我们前进的机会的能力
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="d0fe">
   希望：像斯佩尔曼学院这样的高等学府的学生和校友。他们致力于改变这点。
  </p>
  <p class="graf graf--p graf-after--p" id="0842">
   <strong class="markup--strong markup--p-strong">
    生物学家和环境科学家（18）
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="49d2">
   最恐惧的事：考虑一下30年前的通信（大多数情况下是面对面的，少部分通过电话），20年前（面对面的交谈依旧很多，但是手机开始多起来了），而现在（绝大多数交流都通过某种类型的屏幕）。情愫在哪儿？
  </p>
  <p class="graf graf--p graf-after--p" id="e3de">
   希望：知识生产比以前快很多了，我希望新一代人的学习速度比我们历史上要快得多。
  </p>
  <p class="graf graf--p graf-after--p" id="545a">
   最恐惧的事：世界继续排斥科学和技术的多样性和包容性。百分之一的科学文献来自非洲大陆 — 为什么？非洲是我曾经历过的最具创新性的地方。我们必须为资源匮乏的国家的科学实验室提供持续的电力、互联网和运输服务。我担心我们做得不够快。
  </p>
  <p class="graf graf--p graf-after--p" id="2c8e">
   希望：非洲科学家通过使用最高科技和创新来解决问题。当科学成为一个多元化、包容性的地方时，饥饿将会结束 — 然后所有其他的大规模的挑战也将会下降。
  </p>
  <p class="graf graf--p graf-after--p" id="150b">
   最恐惧的事：我们的社会将不会注意到已经存在的许多警告，不记得我们在各种环境危机中学到的重要教训。有些人仍然相信更多的数据和事实会拯救我们，但如果气候变化（和心理学研究）能告诉我们什么的话，那就是，
   <strong class="markup--strong markup--p-strong">
    更多的事实并没有改变人们的思想/行为。技术也不会拯救我们，它首先是让我们陷入困境的原因
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="75fc">
   希望：希望是一个危险的词。当我们感到失去控制或无力再做任何事情时，我们就会这样做，我们“希望”别人会解决这个问题，或者科学家用“科学”帮助我们摆脱困境。
   <strong class="markup--strong markup--p-strong">
    我鼓励人们不要去希望，而是要做。做每一点事
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="800d">
   最恐惧的事：
   <span class="markup--quote markup--p-quote is-other" data-creator-ids="c8bc2c98d08c">
    <strong class="markup--strong markup--p-strong">
     观点和意见的泛滥超过了证据
    </strong>
    ，在任何方面。
   </span>
  </p>
  <p class="graf graf--p graf-after--p" id="9ae1">
   希望：我在科学、生活和彼此之间看到了人们的持续参与。
  </p>
  <p class="graf graf--p graf-after--p" id="387b">
   最恐惧的事：美国总统可以引导一大群人公开嘲笑一名声称受到性侵犯的女性，而且很少有人会因此愤怒。很多人喜欢卑鄙和欺凌。
  </p>
  <p class="graf graf--p graf-after--p" id="5a72">
   希望：也同时有很多人善良和聪明。
  </p>
  <p class="graf graf--p graf-after--p" id="5f2b">
   最恐惧的事：
   <strong class="markup--strong markup--p-strong">
    政策制定者将在决策中使用意识形态信念而不是基于证据的推理
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="6e05">
   希望：社交媒体促进的社会动员和公开传播以及信息共享。
  </p>
  <p class="graf graf--p graf-after--p" id="9f91">
   最恐惧的事：人类本能的自满情绪集中在眼前的利益上，而不是承认错误并对我们决策的长期影响负责。
  </p>
  <p class="graf graf--p graf-after--p" id="55c6">
   希望：一个日益联系的世界可能使社会创新能够帮助我们改善人类状况，而不是仍然专注于短期利益。
  </p>
  <p class="graf graf--p graf-after--p" id="0f09">
   最恐惧的事：作为一名在实验室工作的科学家，我坚信团队精神。所以我最担心的是，我们不会在科学原则和事实的基础上共同努力，解决我们的世界面临的一些最重要的问题。
  </p>
  <p class="graf graf--p graf-after--p" id="de12">
   希望：我们将继续推进基础科学以及技术的革命性发展，这将提高越来越多人的生活水平。
  </p>
  <p class="graf graf--p graf-after--p" id="22ab">
   最恐惧的事：网上虚假信息的泛滥，因为它可能导致直接的现实后果。
  </p>
  <p class="graf graf--p graf-after--p" id="189e">
   希望：新一代强大的生物技术（基因组编辑、癌症免疫疗法等）将在药物发现、临床治疗和人类健康方面取得重大突破。
  </p>
  <p class="graf graf--p graf-after--p" id="3e5e">
   最恐惧的事：
   <strong class="markup--strong markup--p-strong">
    缺乏参与和建设性的政治领导和政策语话；无限制的和暗箱操作的政治捐款导致政治进程的歪曲，导致一个国家缺乏领导力和前瞻性规划，以及腐败的风险
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="480e">
   希望：我们有很多聪明的年轻人，他们受过更好的教育，拥有比前几代更多的资源和机会。他们显然会创造一个更美好，更公平，更环保的社会。
  </p>
  <p class="graf graf--p graf-after--p" id="3823">
   <strong class="markup--strong markup--p-strong">
    太空科学家和航空航天工程师（19）
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="976a">
   最恐惧的事：科幻小说“WALL-E”中的所有内容都将成为现实。
  </p>
  <p class="graf graf--p graf-after--p" id="0b82">
   希望：孩子们现在知道刷牙可以防止蛀牙，还知道人类的活动也是导致气候变化的主要原因。
  </p>
  <p class="graf graf--p graf-after--p" id="0a59">
   最恐惧的事：人类的短视。我们的物种和整个星球的问题表现在数百年或数千年的时间尺度上。但个人、公司和政府只会考虑并计划下一季度、明年或最多，只看到我们自己的生命周期。
  </p>
  <p class="graf graf--p graf-after--p" id="ac31">
   希望：宇宙足够大，在其中的某个地方可能有真正的智慧生命。
  </p>
  <p class="graf graf--p graf-after--p" id="f442">
   最恐惧的事：不仅有我们的社会缺乏科学素养，甚至还有对它的赞美。
  </p>
  <p class="graf graf--p graf-after--p" id="b64f">
   希望：看到年轻一代的希望、灵感和动力。
  </p>
  <p class="graf graf--p graf-after--p" id="6af1">
   最恐惧的事：
   <strong class="markup--strong markup--p-strong">
    最重要的行动 — — 那些影响国家和全球未来的行动 — — 不能只留给那些能够当选的人。一个坏医生可以杀死很少的病人，但是一个坏的政客或一个坏的财政大臣可以消灭所有人。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="9daf">
   希望：我看不出希望应该来自哪里。
  </p>
  <p class="graf graf--p graf-after--p" id="52fb">
   最恐惧的事：
   <strong class="markup--strong markup--p-strong">
    偏见蔓延到整个社会，我们失去了质疑和争取真相的能力
   </strong>
   。我担心我们在科学研究中失去了太多人，因为他们厌倦了旨在保护白人教授的古老政策，并对学术界效率低下和冰川变化的速度不成比例感到沮丧。
  </p>
  <p class="graf graf--p graf-after--p" id="2aa3">
   希望：年轻人和网络的力量。他们质疑过时的政策，挑战陈规定型观念，并教导我们所有人要诚实。
  </p>
  <p class="graf graf--p graf-after--p" id="a66b">
   最恐惧的事：新自由主义资本主义似乎无情地渗透到了人类生存的方方面面。
  </p>
  <p class="graf graf--p graf-after--p" id="18ac">
   希望：青少年。
  </p>
  <p class="graf graf--p graf-after--p" id="6af2">
   ……
  </p>
  <p class="graf graf--p graf-after--p" id="53ce">
   您呢？您最恐惧的事是什么样的？希望又是什么？欢迎留言探讨。
  </p>
