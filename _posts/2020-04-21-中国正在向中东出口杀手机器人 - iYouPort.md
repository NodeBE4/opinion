  <p class="graf graf--p">
   国防部长马克·埃斯珀（Mark Esper）表示，中国正在向中东出口据其宣称具有杀伤力的无人机。这是美国国防部高级官员首次承认中国出售的是能够在几乎没有人为监督的情况下夺走生命的武装无人机。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “中国政府已经在向中东出口一些最先进的军用无人机，为下一代隐形无人机的出口做好准备”，Esper 在国家安全委员会人工智能会议上说。
  </p>
  <p class="graf graf--p">
   他还说，“此外，中国武器制造商正在出售标榜为具有完全自治能力的无人机，包括进行致命的有针对性的打击的能力。”
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    关于杀人者。很长一段时间，只有美国、英国和以色列拥有战斗无人机并用它们来执行全球战争 — — 但是，近年来这一领域发生了巨大变化。
    <br/>
    因为，中国来了。详见《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%E6%BD%98%E5%A4%9A%E6%8B%89%E7%9A%84%E8%BD%B0%E7%82%B8/" href="https://www.iyouport.org/%E6%BD%98%E5%A4%9A%E6%8B%89%E7%9A%84%E8%BD%B0%E7%82%B8/" rel="noopener noreferrer" target="_blank">
     潘多拉的轰炸
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">

   </noscript>

  <p class="graf graf--p">
   例如，中国公司紫燕（Ziyan）销售河豚A3，这实际上是一架配备机枪的直升无人机。
   <a class="markup--anchor markup--p-anchor" data-href="http://ziyanuav.com/blowfish2.html" href="http://ziyanuav.com/blowfish2.html" rel="noopener noreferrer" target="_blank">
    紫燕说
   </a>
   ：“它可以自主执行更复杂的战斗任务，包括定点定时检测、定距侦察、和有针对性的精确打击。”
  </p>
  <p class="graf graf--p">
   正如国防部联合人工智能中心战略与传播负责人 Greg Allen 在去年2月给 CNAS 的论文中
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnas.org/publications/reports/understanding-chinas-ai-strategy" href="https://www.cnas.org/publications/reports/understanding-chinas-ai-strategy" rel="noopener noreferrer" target="_blank">
    指出的那样
   </a>
   ，紫燕正在谈判将其河豚A2出售给巴基斯坦和沙特阿拉伯政府。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “尽管对人工智能军备竞赛表示担忧，但大多数中国领导人都认为，军事上增加对人工智能的使用是不可避免的，并且正在积极追求。中国已经出口了武装自主平台和监视AI，’’ Greg Allen 写道。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%ba%e5%b7%a5%e6%99%ba%e8%83%bd%e7%9a%84%e4%b8%ad%e5%9b%bd%e5%86%9b%e4%ba%8b/" href="https://www.iyouport.org/%e4%ba%ba%e5%b7%a5%e6%99%ba%e8%83%bd%e7%9a%84%e4%b8%ad%e5%9b%bd%e5%86%9b%e4%ba%8b/" rel="noopener noreferrer" target="_blank">

    》

   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%ba%e5%b7%a5%e6%99%ba%e8%83%bd%e6%96%b0%e5%86%b7%e6%88%98%e2%80%8a-%e2%80%8a-ai%e5%92%8c5g%e5%a6%82%e4%bd%95%e5%8f%98%e6%88%90%e4%b8%80%e5%9c%ba%e5%99%a9%e6%a2%a6/" href="https://www.iyouport.org/%e4%ba%ba%e5%b7%a5%e6%99%ba%e8%83%bd%e6%96%b0%e5%86%b7%e6%88%98%e2%80%8a-%e2%80%8a-ai%e5%92%8c5g%e5%a6%82%e4%bd%95%e5%8f%98%e6%88%90%e4%b8%80%e5%9c%ba%e5%99%a9%e6%a2%a6/" rel="noopener noreferrer" target="_blank">

    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/ai-%e5%86%9b%e5%a4%87%e7%ab%9e%e8%b5%9b/" href="https://www.iyouport.org/ai-%e5%86%9b%e5%a4%87%e7%ab%9e%e8%b5%9b/" rel="noopener noreferrer" target="_blank">
     AI 军备竞赛
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   去年，中国第三大国防公司NORINCO（北方工业集团公司）的高级主管 Zeng Yi 曾预测：“在未来的战场上，不会有人类的参与” —— 最早是在2025年。
  </p>
  <p class="graf graf--p">
   Esper 还表示，中国的监视软件和硬件网络可以帮助中国发展人工智能。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “所有迹象都表明，该国建立了一个二十一世纪的监视国家，旨在对言论进行审查，并以前所未有的规模否认基本人权。毫无疑问，该国已经使用监视手段来系统性镇压一百万以上的穆斯林维吾尔人”；
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “北京拥有迫使中国工业界和学术界支持其政府主导的努力所需的一切力量和工具。”
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     要记得《
    </strong>
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%b8%ad%e5%9b%bd%e7%9a%84%e5%8f%8d%e4%b9%8c%e6%89%98%e9%82%a6%e7%9b%91%e8%a7%86%e5%9b%bd%e5%ae%b6%e6%98%af%e7%be%8e%e5%9b%bd%e5%88%b6%e9%80%a0%e7%9a%84/" href="https://www.iyouport.org/%e4%b8%ad%e5%9b%bd%e7%9a%84%e5%8f%8d%e4%b9%8c%e6%89%98%e9%82%a6%e7%9b%91%e8%a7%86%e5%9b%bd%e5%ae%b6%e6%98%af%e7%be%8e%e5%9b%bd%e5%88%b6%e9%80%a0%e7%9a%84/" rel="noopener noreferrer" target="_blank">
     <strong class="markup--strong markup--li-strong">
      中国的反乌托邦监视国家是美国制造的
     </strong>
    </a>
    <strong class="markup--strong markup--li-strong">
     》
    </strong>
   </li>
  </ul>
  <p class="graf graf--p">
   美国国防创新委员会提出了一份针对美国军方的AI原则清单，该清单将人为可控列为关键原则，这与早在2012年的先前政策是一致的。Esper 称赞该清单。
  </p>
  <p class="graf graf--p">
   他说，“同样令人不安的是，外部公司或跨国公司不经意或默认提供给中国不道德地使用AI的技术能力或研究。”
  </p>
  <p class="graf graf--p">
   他应该直接说是美国。没有其他国家能做到这点。⚪️
