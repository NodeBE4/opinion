 <div class="entry-content">
  <ul>
   <li class="graf graf--p">
    <span style="color: #00ccff;">
     <em>
      <strong>
       前两天与端传媒的编辑邮件沟通，在结束时我写道：“祝福所有人都能平安渡过这一困境，获得真实的及时的资讯，尤其是，希望疫情过后，能够通过公民行动追查到切实的原因，和责任人（如果有的话）。” 以下是 Žižek 今天的文章，他潜在地提醒了我认为 “尤其” 的这部分。
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <figure aria-describedby="caption-attachment-12911" class="wp-caption aligncenter" id="attachment_12911" style="width: 768px">
   <img alt="" class="wp-image-12911 size-full jetpack-lazy-image" data-lazy-sizes="(max-width: 768px) 100vw, 768px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/0000-1.jpg?resize=768%2C432&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/0000-1.jpg?w=768&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/0000-1.jpg?resize=300%2C169&amp;ssl=1 300w" data-recalc-dims="1" height="432" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/0000-1.jpg?resize=768%2C432&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="768"/>
   <noscript>
    <img alt="" class="wp-image-12911 size-full" data-recalc-dims="1" height="432" sizes="(max-width: 768px) 100vw, 768px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/0000-1.jpg?resize=768%2C432&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/0000-1.jpg?w=768&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/0000-1.jpg?resize=300%2C169&amp;ssl=1 300w" width="768"/>
   </noscript>

   </figcaption>
  </figure>
  <p class="graf graf--p">
   就像社会抗议活动一样，疫情爆发不会突然出现然后消失。它们顽固地潜伏着，在最不经意之间爆发。我们应该接受这一点，并且有两种不同的方式可以做到。
  </p>
  <p class="graf graf--p">
   中国境外的人认为，隔离足以解决该病毒的传播问题，他们在 “隔离墙” 后面或多或少是安全的。但是，既然已经在20多个国家都报告发现了冠状病毒感染病例，这意味着急需一种新的方法。我们如何应对这种创伤性威胁？
  </p>
  <p class="graf graf--p">
   也许我们能从应对冠状病毒疫情的反应中学到一些东西，根据精神科医师兼作家 Elisabeth Kübler-Ross 在《论死亡和将死》（On Death and Dying）中提出的著名五阶段模式，关于人们在得知身患绝症时作何反应：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    拒绝 —— 只是拒绝接受这一事实，如 “这不可能发生，不可能发生在我身上；
   </li>
   <li class="graf graf--li">
    愤怒 —— 当我们无法再否认这一事实时便会爆发愤怒，如 “我怎么会这样？”
   </li>
   <li class="graf graf--li">
    讨价还价 —— 希望自己可以以某种方式推迟或减少这一事实，如 “让我活着看到我的孩子毕业吧”；
   </li>
   <li class="graf graf--li">
    抑郁 —— 虚无，如 “反正要死了何必自寻烦恼”；
   </li>
   <li class="graf graf--li">
    最终是接受 —— “我已经无法抗拒，应该为此做好准备”。
   </li>
  </ul>
  <p class="graf graf--p">
   Kübler-Ross 后来将这5个阶段模型
   <strong class="markup--strong markup--p-strong">
    应用于任何形式的灾难性个人损失
   </strong>
   （失业、亲人死亡、离婚、吸毒成瘾），并强调指出：这些阶段不一定按照同一顺序发生，
   <strong class="markup--strong markup--p-strong">
    也不是所有人都能经历完整的五个阶段
   </strong>
   。
  </p>
  <p class="graf graf--p">
   每当一个社会面临某种创伤事件时，总是可以辨别出这五个阶段。以应对生态灾难（气候危机）的威胁为例。
  </p>
  <p class="graf graf--p">
   首先，人们倾向于否认它 —— “这只是偏执狂的想法，真正发生的只是天气模式的惯常波动”。
  </p>
  <p class="graf graf--p">
   然后是愤怒 —— 对污染环境的寡头公司和对危险视而不见的政府的愤怒。
  </p>
  <p class="graf graf--p">
   再然后是讨价还价：“如果我们回收废物，可以争取到一些时间；另外，它也有很好的一面，我们现在可以在格陵兰岛种菜，船只将能够通过北部航线更快地将商品从中国运到美国，由于冻层融化，西伯利亚北部有了新的肥沃土地可用。”
  </p>
  <p class="graf graf--p">
   接着是抑郁 —— “太迟了，我们失败了”。
  </p>
  <p class="graf graf--p">
   最后是接受 —— “我们正面临严重威胁，将不得不改变我们的整个生活方式！”
  </p>
  <p class="graf graf--p">
   同样的模型同样适用于对我们的生活中
   <strong class="markup--strong markup--p-strong">
    日益增长的数字监控威胁
   </strong>
   。
  </p>
  <p class="graf graf--p">
   再一次，首先，人们倾向于否认它 —— 认为这是 “夸张的”，“更多左派的妄想症”，“没有什么机构可以控制我们的日常活动。”
  </p>
  <p class="graf graf--p">
   然后是愤怒 —— 愤怒于寡头公司和政府秘密间谍，他们 “比我们自己更了解我们”，并利用这些了解来
   <strong class="markup--strong markup--p-strong">
    控制和操纵
   </strong>
   我们。
  </p>
  <p class="graf graf--p">
   再然后是讨价还价 —— 劝导自己相信 “当局有权搜寻恐怖分子，而不会侵犯公民的隐私权”。
  </p>
  <p class="graf graf--p">
   再之后是抑郁 —— “太迟了，我们已经没有隐私人权了，个人自由的时代已经过去了”。
  </p>
  <p class="graf graf--p">
   最终就是接受 —— “数字控制是对我们自由的威胁，我们应该让公众意识到其所有方面，并参与到斗争中来！”
  </p>
  <figure aria-describedby="caption-attachment-12912" class="wp-caption aligncenter" id="attachment_12912" style="width: 1240px">
   <img alt="" class="size-full wp-image-12912 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/00-1.jpg?resize=1100%2C673&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/00-1.jpg?w=1240&amp;ssl=1 1240w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/00-1.jpg?resize=300%2C184&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/00-1.jpg?resize=1024%2C627&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/00-1.jpg?resize=768%2C470&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/00-1.jpg?resize=1100%2C673&amp;ssl=1 1100w" data-recalc-dims="1" height="673" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/00-1.jpg?resize=1100%2C673&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="size-full wp-image-12912" data-recalc-dims="1" height="673" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/00-1.jpg?resize=1100%2C673&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/00-1.jpg?w=1240&amp;ssl=1 1240w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/00-1.jpg?resize=300%2C184&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/00-1.jpg?resize=1024%2C627&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/00-1.jpg?resize=768%2C470&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/00-1.jpg?resize=1100%2C673&amp;ssl=1 1100w" width="1100"/>
   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-12912">
    A Chinese man wears a protective mask, February 12, 2020, Beijing, China © Getty Images / Kevin Frayer
   </figcaption>
  </figure>
  <p class="graf graf--p">
   甚至在政治领域也是如此，那些因特朗普当选而受到伤害的人也在重复同样的模式。
  </p>
  <p class="graf graf--p">
   首先是否认 —— “别担心，特朗普只是装腔作势，他掌权并不会有任何改变”；
  </p>
  <p class="graf graf--p">
   然后是愤怒 —— 把他推 上台的 “黑暗势力”、支持他的那些民粹主义者，对我们的道德实质构成威胁；
  </p>
  <p class="graf graf--p">
   再然后是讨价还价 —— “还没有失去一切，也许特朗普可以被遏制住，让我们容忍他的一些过分行为吧”；
  </p>
  <p class="graf graf--p">
   接着是抑郁 —— “我们正在走向法西斯主义，美国没有民主了”；
  </p>
  <p class="graf graf--p">
   最后是接受 —— ”美国已经出现了一个新政权，美式民主的美好旧时光一去不复返了，我们要直面危险冷静地计划如何才能战胜特朗普的民粹主义。”
  </p>
  <p class="graf graf--p">
   在中世纪时期，受鼠疫影响的城镇居民也以类似的方式做出反应。
  </p>
  <p class="graf graf--p">
   首先是否认，然后是愤怒 —— “我们罪恶的生活被惩罚了，或许甚至是让瘟疫发生的残酷的上帝”；
  </p>
  <p class="graf graf--p">
   接着讨价还价 —— “还不太糟，我们可以消避开那些被感染的病人”；
  </p>
  <p class="graf graf--p">
   然后是抑郁 —— 我们的生命结束了；
  </p>
  <p class="graf graf--p">
   接下来，有趣的是，纵欲 —— “既然我们的生命结束了，那就尽可能及时行乐吧。畅饮，性……”；
  </p>
  <p class="graf graf--p">
   最终，是接受 —— “我们还活着，那就尽可能表现得像正常生活在继续一样。”
  </p>
  <p class="graf graf--p">
   这也同样呈现在人们如何应对2019年底爆发的冠状病毒疫情中不是吗？
  </p>
  <p class="graf graf--p">
   首先是否认 —— 没有发生严重的事，不过是某些不负责的家伙散播恐慌；
  </p>
  <p class="graf graf--p">
   然后是愤怒 —— 通常是种族主义或反国家的形式：肮脏的中国人有罪，我们的国家不够高效……
  </p>
  <p class="graf graf--p">
   接着是讨价还价 —— 好吧有一些受害者，但没有非典那会儿严重，我们能够控制住损失；
  </p>
  <p class="graf graf--p">
   如果这行不通，抑郁就出现了 —— 别再自欺欺人了，我们都在劫难逃 ……
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    这里的 “接受” 会是什么样子的？
   </strong>
  </p>
  <p class="graf graf--p">
   古怪的事实在于，疫情显示出一种特性，与最新一轮的社会抗议存在很多共同点，比如那些在法国和香港发生的抗议：它们不是爆发式的速战速决的，而是一直呆在这里，坚持着，将永久的恐惧和脆弱注入人们的生活。
  </p>
  <p class="graf graf--p">
   我们应该接受的、应该与自己解释的是，存在着生命的下一层，即 病毒的不死，愚蠢的重复性地生存，它永远存在于我们身边，永远像阴影一样笼罩着我们，对我们的生存构成威胁，在我们最不期望的时候爆发出来。
  </p>
  <p class="graf graf--p">
   从更广泛的层面上讲，病毒流行提醒人们生命的最终偶然性和无意义：无论人类创造了多么宏伟的精神大厦，像病毒或小行星这类愚蠢的自然偶然事件就可以终结一切。更不用说生态学的教导了 —— 我们人类也可能在不知不觉中促成这一终结。
  </p>
  <p class="graf graf--p">
   但是，最终的 “接受” 可以采取两个方向。
  </p>
  <p class="graf graf--p">
   其中之一可能意味着只是疾病的正常化：好吧，人都会死，但生活会继续，甚至可能有一些良好的副作用。
  </p>
  <p class="graf graf--p">
   而另一种，
   <strong class="markup--strong markup--p-strong">
    “接受” 可以（并且应该）推动我们动员起来，抛弃恐慌和幻想，在集体团结中行动起来
   </strong>
   。⚪️
  </p>
