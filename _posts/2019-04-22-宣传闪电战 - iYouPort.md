   <li>
    <span style="color: #00ccff;">
     <strong>
      <em>
       IYP在中国年之前连续发布四篇关于新闻危害民主的分析，正因为中国正在发生变化，虽然目前仍处于初级阶段，但是，该模式一旦成熟，即便有了选票也不会有民主，为什么人们应该从现在开始警惕？
      </em>
     </strong>
    </span>
   </li>
  </ul>
  <div class="section-inner sectionLayout--insetColumn">
   <p class="graf graf--p graf-after--p" id="dbe8">
    由于担心年轻互联网用户的不满情绪日益增加，中国领导层已开始新一轮的宣传闪电战。
   </p>
   <p class="graf graf--p graf-after--p" id="aa76">
    观察人士说，北京方面严重担心网络媒体对中国年轻人产生“不良影响”，并试图确保他们更多地接触官方批准的叙述。
   </p>
   <p class="graf graf--p graf-after--p" id="1e1d">
    这一驱动力的核心是一支由“千禧一代的新媒体专家”组成的团队，他们避开传统宣传的行话风格，转而采用旨在与年轻人 — 精通网络的公民共鸣的故事。
   </p>
   <p class="graf graf--p graf-after--p" id="b5cd">
    习近平在1月22日的高级官员会议上表示，该执政党应该
    <strong class="markup--strong markup--p-strong">
     把公众异议的政治风险作为他们的首要任务，并为“最坏情况”做好准备
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="7878">
    随着经济开始放缓，中国领导层对公众持不同意见的危险非常敏感，最近几个月，军队退伍军人、教师和青年马克思主义学生都在各地区开展抗议以争取工人权利。
   </p>
   <p class="graf graf--p graf-after--p" id="5ac5">
    演讲后第四天，习近平率领中共中央政治局委员访问了人民日报的新媒体运作，这是该党的喉舌。他详细讲述了“
    <strong class="markup--strong markup--p-strong">
     在公共传播中促进综合媒体发展和扩大主流语话
    </strong>
    ”的重要性。
   </p>
   <p class="graf graf--p graf-after--p" id="73ca">
    ⚠️
    <strong class="markup--strong markup--p-strong">
     请注意，这就是 IYP 在中国年之前提醒警惕的“媒体融合”，它是被习近平加强的趋势，该趋势一旦实现，即便有了选票也没有民主
    </strong>
    — — 这就是为什么 IYP 当时连续发布了四篇文章分析这一问题。事实上这一苗头自去年开始已经呈现，当时我们立即发布了一个长篇独家报道，以分析中国社会面临的近期前景。详见以下部分：
   </p>
   <ul class="postList">
    <li class="graf graf--li graf-after--p" id="d937">
     《
     <a class="markup--anchor markup--li-anchor" data-href="https://medium.com/@iyouport/%E7%8B%AC%E5%AE%B6-%E5%8C%97%E4%BA%AC%E7%9A%84%E6%94%BF%E6%B2%BB%E5%AE%A3%E4%BC%A0%E6%A8%A1%E5%BC%8F%E6%AD%A3%E5%9C%A8%E6%94%B9%E5%8F%98-%E6%88%96%E5%B7%B2%E8%B8%A9%E4%B8%AD%E4%B8%AD%E5%9B%BD%E5%BC%82%E8%AE%AE%E7%9A%84%E5%BC%B1%E7%82%B9-d1bf5ddfd412" href="https://medium.com/@iyouport/%E7%8B%AC%E5%AE%B6-%E5%8C%97%E4%BA%AC%E7%9A%84%E6%94%BF%E6%B2%BB%E5%AE%A3%E4%BC%A0%E6%A8%A1%E5%BC%8F%E6%AD%A3%E5%9C%A8%E6%94%B9%E5%8F%98-%E6%88%96%E5%B7%B2%E8%B8%A9%E4%B8%AD%E4%B8%AD%E5%9B%BD%E5%BC%82%E8%AE%AE%E7%9A%84%E5%BC%B1%E7%82%B9-d1bf5ddfd412" rel="noopener noreferrer" target="_blank">

    </li>

   <p class="graf graf--p graf--startsWithDoubleQuote graf-after--li" id="bfac">
    <strong class="markup--strong markup--p-strong">
     “移动平台是首选，”习说，并希望他的宣传负责人能够“开发网站、微博、微信、电子报纸、IPTV 和其他形式的新媒体”
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="5c72">
    一些高级官员已经听取了福建省党委书记于伟国的来电，他在习近平发表评论后三天访问了当地人民日报编辑部。
   </p>
   <p class="graf graf--p graf-after--p" id="b638">
    中国的互联网一直受到严格的审查，但新的焦点是习近平要求确保“党的声音可以直接到达各种用户终端”，这促使
    <strong class="markup--strong markup--p-strong">
     宣​​传部长们追求来自全国各地的新媒体专家。他们被招募、为该党的官方喉舌提供平台，并在全国8亿网络用户中推广被称为“正能量”的玩意
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="9b0d">
    1月18日，公安部部长赵克智再次警告“颜色革命”的风险 — — 提到十年前席卷一些前苏联国家的抗议活动 — — 并要求所有执法部队“坚决捍卫政治安全”。
   </p>
   <p class="graf graf--p graf-after--p" id="8dec">
    经过40年的快速发展，中国的年轻一代现在能充分感觉到他们向上移动的机会较少，同时在工作和生活中面临着更大的压力。
   </p>
   <p class="graf graf--p graf-after--p" id="01ed">
    不断增长的不满情绪使该党的领导层感到震惊，引发了一场关于一些学者的所谓“代际不平等”的争论，因为年轻的中国人失去了在该国经济繁荣高峰期长大的那一代所享有的福利。
   </p>
   <p class="graf graf--p graf-after--p" id="a19a">
    其中一个结果是，一种被称为“丧”的懒散文化在80年代或90年代出生的人中变得流行。详见：《
    <a class="markup--anchor markup--p-anchor" data-href="https://pao-pao.net/article/839" href="https://pao-pao.net/article/839" rel="noopener nofollow noreferrer" target="_blank">
     百忧解时代（三）令人沉迷的广义化自杀
    </a>
    》
   </p>
   <p class="graf graf--p graf-after--p" id="6b44">
    中国UC大数据中心的一项研究表明，这种现象是由房价暴涨、社会流动性不足以及在现代中国寻找合作伙伴的困难等因素造成的异化感所驱动的。
   </p>
   <p class="graf graf--p graf-after--p" id="082c">
    社交媒体加剧了这些社会分歧，许多富裕的千禧一代喜欢在网上炫耀自己的财富。
   </p>
   <p class="graf graf--p graf-after--p" id="74ba">
    去年一个受欢迎的meme就叫“炫耀你的财富”挑战，在那里，人们张贴照片，显示自己躺在地上，被贵重的物品包围，好像他们已经被那些高级物品绊倒了一样。
   </p>
   <p class="graf graf--p graf-after--p" id="f809">
    不出所料，该党重新关注社交媒体，包括积极努力抵制这一趋势。
   </p>
   <p class="graf graf--p graf-after--p" id="9d1a">
    打击行动的一部分针对的是知名度高的社交媒体帐户，这些帐户被指责煽动社会问题或未能妥善监管其内容。
   </p>
   <p class="graf graf--p graf-after--p" id="f69b">
    反对“不负责任”的社交媒体账户“哄骗人们的焦虑”的另一个主要声音来自中央政法委的官方微信账号。这是该党努力控制社交媒体叙事的核心，使用长安剑的绰号，在11月披露其真实身份。
    <strong class="markup--strong markup--p-strong">
     这项新媒体运作计划由一个不到10人的小型中央团队负责，他们都出生于20世纪80年代或90年代
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="0d87">
    该核心团队的选拔过程以及在其下工作的省级行动都是非常严格的，包括筛查候选人的政治忠诚度。
   </p>
   <p class="graf graf--p graf-after--p" id="b983">
    <strong class="markup--strong markup--p-strong">
     他们还需要具备执法背景 — 无论是在警察、法院、司法部门还是检察机关的工作经历 — 而且都必须熟悉微信或微博等新媒体平台
    </strong>
    ，当然，他们还需要愿意在很长时间内工作于更常见在互联网初创企业而不是拥有政府部门的编制。
   </p>
   <p class="graf graf--p graf-after--p" id="dd16">
    入选者将被送到北京进行为期六个月的强化培训，然后回各地领导省级和市级新媒体团队。
   </p>
   <p class="graf graf--p graf-after--p" id="8ac2">
    一位消息人士称，这一过程帮助新媒体中心“让来自全国各地的优秀人才保持团队新鲜感”。
   </p>
   <p class="graf graf--p graf-after--p" id="3dfc">
    培训结束后，他们可以“带回在北京获得的经验，从而建立更强大的省级队伍”。
   </p>
   <p class="graf graf--p graf-after--p" id="5dbf">
    与传统的自上而下的宣传部门结构不同，新媒体团队有更多的编辑自由，“不一定遵循高层命令”。
    <strong class="markup--strong markup--p-strong">
     在决定推广哪些故事时，“首先考虑的是我们能够触及人心吗？
    </strong>
    ”这位知情人士说。
   </p>
   <blockquote class="graf graf--blockquote graf-after--p" id="5071">
    <p>
     <em class="markup--em markup--blockquote-em">
      只有当我们能夺取他们的注意力的时候，我们才有希望去征服他们的心和灵魂 — — 谷歌前执行主席 Eric Schmidt
     </em>
    </p>
   </blockquote>
   <p class="graf graf--p graf-after--blockquote" id="ec29">
    虽然传统的官方媒体是针对党员干部的，但新媒体业务必须覆盖更广泛的受众，因此他们
    <strong class="markup--strong markup--p-strong">
     “必须适应更易于读者阅读的格式，并在报道中使用较少的党派行话”
    </strong>
    。他还表示，到目前为止，他们已经取得了“一定的成功”。
   </p>
   <p class="graf graf--p graf-after--p" id="7985">
    <strong class="markup--strong markup--p-strong">
     官方新媒体平台在增加受众群体方面具有内在优势，因为所有省级和市级执法机构都要求其员工关注和“点赞”这些新媒体账户。所以看到那些拥有很多点赞的文章，请不要误会他们的影响力
    </strong>
    。更多详见《
    <a class="markup--anchor markup--p-anchor" data-href="https://iyouport.weebly.com/short-stories-enthusiast/3240723" href="https://iyouport.weebly.com/short-stories-enthusiast/3240723" rel="noopener nofollow noreferrer" target="_blank">
     互联网上究竟有多少东西是假的
    </a>
    ？》
   </p>
   <p class="graf graf--p graf-after--p" id="d1bb">
    中国没有公布究竟有多少网警的官方数据，但中国社会科学院研究员范鹏估计，这一数字在2014年约为160万。
   </p>
   <p class="graf graf--p graf-after--p" id="8c79">
    根据跟踪微信公众号的
    <a class="markup--anchor markup--p-anchor" data-href="http://leibei.com/" href="http://leibei.com/" rel="noreferrer noopener nofollow" target="_blank">
     Leibei.com
    </a>
    说，长安剑拥有约67万活跃用户，其微博帐户拥有超过600万粉丝。另一个官方媒体账户由人民日报负责的侠客岛 拥有超过100万活跃粉丝。
   </p>
   <p class="graf graf--p graf-after--p" id="af0d">
    一些分析师认为，由于官方媒体垄断了硬新闻报道，这些官方账户的增长将挤压其他新媒体播放器。这就是 IYP 此前强调的“主流”为什么必须警惕，更多详见《
    <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/kn-04182019-8248fec1da74" href="https://medium.com/@iyouport/kn-04182019-8248fec1da74" rel="noopener noreferrer" target="_blank">
     停止主流
    </a>
    》。
   </p>
   <p class="graf graf--p graf-after--p" id="819f">
    新加坡李光耀公共政策学院副教授 Alfred Wu 说：“在当前的气氛下，非官方新媒体播放器的空间将进一步被缩小。“所有政策，时事相关的报道都被官方渠道垄断，其他人只能写出星座等类似的软性内容。”
   </p>
   <p class="graf graf--p graf-after--p" id="e8cc">
    更多分析请详见《
    <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/media-3-ae3404d90746" href="https://medium.com/@iyouport/media-3-ae3404d90746" rel="noopener noreferrer" target="_blank">
     现代新闻之愚蠢 (3) — 深度剖析四种致命倾向性，及其政治代价
    </a>
    》的最后部分，解释了为什么这种危险的全球性趋势对中国社会来说是更加危险。
   </p>
