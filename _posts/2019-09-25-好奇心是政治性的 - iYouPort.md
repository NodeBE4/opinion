  <p class="graf graf--p">
   如题，好奇心是政治性的。
  </p>
  <p class="graf graf--p">
   好奇心的缺席、存在、培养和消除都是政治工具，具有几乎难以想象的力量。它们也是无处不在的政治后果的社会结果。
  </p>
  <p class="graf graf--p">
   左翼一直知道知识的轮廓是政治性的。近年来，对无知政治的兴趣也开始形成；agnotology（无知学）作为无知的哲学研究被命名，建立了政治、心理和公共记忆之间的联系，以描述无知的社会建构。
  </p>
  <p class="graf graf--p">
   很容易注意到，这种关于 agnotology 的基本见解实际上是长期存在的。
  </p>
  <p class="graf graf--p">
   Upton Sinclair 在1934年评论说，“当一个人赚钱是基于他不理解什么的时候，你就很难让他去理解这件事”，直接表达了一种无知的立场理论。
  </p>
  <p class="graf graf--p">
   不幸的是，认识论（研究我们如何知道我们所知道的哲学领域）和批评学都没有以任何值得注意的方式致力于好奇心，毕竟，这意味着既知道又无视。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    作为一种概念和现象的好奇心几乎完全被学者所忽视了，这是一个巨大的漏洞
   </strong>
   ，导致我对好奇心是政治性的这一完全明显的声明有一种“顿悟”的感觉。
  </p>
  <p class="graf graf--p">
   好奇心是一个笼统的术语；尽管它具有几乎普遍积极的现代内涵，但承认它并不一定就是十足的好事。
  </p>
  <p class="graf graf--p">
   拒绝好奇心可能是一种道德选择，例如缺乏对如何制造中子弹的好奇心，或者在维护隐私方面尊重对方的保密选择。
  </p>
  <p class="graf graf--p">
   好奇心也可以是一种放纵，仅仅是转移或分散注意力，甚至是对权力的渴望。想想19世纪帝国主义者和维多利亚时代探索者的那种不可思议的好奇心，寻求知识作为一种控制形式；或者各种采矿者和企业家的那种贪婪的好奇心。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    如果不想了解这个世界，也就永远不会想到如何可以改变它。如果不想了解其他思想、其他生物、和其他生存方式，我们就永远不会建立社区、不会团结、也就没有什么新世界
   </strong>
   。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4656 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/00-3.jpg?resize=1100%2C730&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/00-3.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/00-3.jpg?resize=300%2C199&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/00-3.jpg?resize=768%2C510&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/00-3.jpg?resize=1024%2C680&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/00-3.jpg?resize=1100%2C730&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/00-3.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="730" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/00-3.jpg?resize=1100%2C730&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>

  <p class="graf graf--p">
   虽然对不好奇和无知的支持可以在背景中具有重要的道德立场，比如军事研究或帝国主义者的勘探，但我们还必须倡导某些好奇心的政治性和道德价值 —— 好奇心，用米歇尔·福柯（Michel Foucault）的话说就是，“唤起人们对存在与可能存在的东西的关注”。
  </p>
  <p class="graf graf--p">
   我们生活在一个高度情绪化的全球时刻，人们在“不成功”时表现出畏惧、恐惧、愤怒、焦虑、异化、甚至感觉羞耻，而好奇心基本上需要建立在自我自力、智力开放和接受未知的潜在意愿之上。
  </p>
  <p class="graf graf--p">
   你读过 Joseph Heller 最经典的“第22条军规（catch-22）”吗？如今就是这样：为了建立和维护其他政治、社区和社会世界，人们需要对其他人、其他生物和其他生活方式产生深深的好奇心。但为了在社会中为好奇心腾出空间，就需要做出根本性的社会变革。
  </p>
  <p class="graf graf--p">
   与此同时，
   <strong class="markup--strong markup--p-strong">
    好奇心正被当权者积极压制、被文化攻击者扭曲为一种自卫形式，随处可见的是骚动的文化焦虑
   </strong>
   。
  </p>
  <p class="graf graf--p">
   好奇心的塑造始于出生时。人类天生就有一些珍贵的特征，好奇心就是其中之一。
  </p>
  <p class="graf graf--p">
   随着孩子们在我们的社会中成长，他们逐渐失去好奇心，变成了一个异常冷漠的成年人。
  </p>
  <p class="graf graf--p">
   尽管家庭动态和父母的风格会对孩子的好奇心产生明显的影响，有时是激励有时是抑制，但是，年轻人奇迹般地从闪闪发光的欣喜到无聊和闷闷不乐的巨大转变，在很大程度上可以归因于学校。
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    <em>
     据报道，今天的儿童早在一年级时就失去了对学校的兴趣。
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p">
   几个世纪以来，学校对自由范围的好奇心的踩踏一直是被抱怨的主题之一。
  </p>
  <p class="graf graf--p">
   但在过去，幼儿园和低年级的学生在精神和意念方面都是相对自由的形式，这种自由感使得学生们对后期死记硬背的教育模式产生了强烈的抵触。
  </p>
  <p class="graf graf--p">
   社会学家和心理学家过去常常认为四年级左右是孩子们失去好奇心的时候，自那时起，怨恨和倦怠超越了对新奇和探索的喜爱。
  </p>
  <p class="graf graf--p">
   甚至早在20世纪80年代中期，当托儿所优先考虑游戏教学时，Barbara Tizard 和 Martin Hughes 对学龄前儿童的研究就发现，孩子们提出的平均问题数量从幼儿在家时的每小时26个到学前班时期每小时2个。
  </p>
  <p class="graf graf--p">
   毫不奇怪，今天的孩子，早在幼儿园时期就受到了计划课程的影响，并且早在一年级时就被成绩报告打败了 —— 连新闻报道都承认，孩子们早在一年级就
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theatlantic.com/magazine/archive/2016/01/the-new-preschool-is-crushing-kids/419139/" href="https://www.theatlantic.com/magazine/archive/2016/01/the-new-preschool-is-crushing-kids/419139/" rel="noopener noreferrer" target="_blank">
    失去了对学校的兴趣
   </a>
   。
  </p>
  <p class="graf graf--p">
   虽然标准化考试、过度拥挤和资金不足等问题无疑对课堂探究的追求产生了特别的毒性影响，但 Paolo Freire 指出，好奇心麻醉发生在任何传统的教育机构中。
  </p>
  <p class="graf graf--p">
   在她的书“饥饿的心灵”中，Susan Engel 用了整整一章的内容具体描述了
   <strong class="markup--strong markup--p-strong">
    即使是最温暖、最善意的教师，在寻求留住学生的过程中也会消除孩子们的好奇心。因为教学双方都将学习变成了任务而不再是乐趣
   </strong>
   。
  </p>
  <p class="graf graf--p">
   好奇心究竟是一种教学的实际目的，还是其他动态的副产品？或者正如福柯所描述的那样，好奇心是一种危险的“关于重要和必要的传统等级制度的随意性”，被维持现状的教育监护人踩踏，他们急切地执行隐秘的服从式教育……
  </p>
  <p class="graf graf--p">
   或者，好奇心未能用资本主义术语来计算机会成本、对职业技能的顺利教育或爱国主义的灌输造成了障碍，而不是对无法解决的问题感到疑惑？
  </p>
  <p class="graf graf--p">
   我们可以进行辩论，但不可否认的是，那些被压抑的古怪的基本学校现实。
  </p>
  <p class="graf graf--p">
   学生们很快就会知道教师和学校对他们的问题不感兴趣。
  </p>
  <p class="graf graf--p">
   Herbert Kohl 的经典论文“
   <a class="markup--anchor markup--p-anchor" data-href="https://curriculumtoolkitbrendaboylan.files.wordpress.com/2016/05/i-wont-learn-from-you.pdf" href="https://curriculumtoolkitbrendaboylan.files.wordpress.com/2016/05/i-wont-learn-from-you.pdf" rel="noopener noreferrer" target="_blank">
    I Won’t Learn From You
   </a>
   ”所表达的只是对每个教师来说都应该知道的事 —— 即，好奇心可以作为不喜欢、拒绝或对抗的标志而被隐瞒，并且经常代表一种防守的尝试 —— 防御自责或担心失败，防御一种被傲慢和敌对内容贬低或攻击的文化。
  </p>
  <p class="graf graf--p">
   通过这种方式，不论是好奇心还是好奇心的缺席都是政治性的。
  </p>
  <p class="graf graf--p">
   事实上，一位优秀教师的标志之一就是能够通过创造微妙的情感联盟来躲过学生的抵触并引起好奇心，同时与学校保持一定的距离。
  </p>
  <p class="graf graf--p">
   学校里发生的事是大文化的一部分，而且往往仅是对它的反映。如果好奇心在教育环境中表现不佳，那么在更广阔的世界中会发生什么呢？
  </p>
  <p class="graf graf--p">
   首先，我们看到为了权力的目的而产生的共同的好奇心。 Justin E. H. Smith
   <a class="markup--anchor markup--p-anchor" data-href="http://www.jehsmith.com/1/2012/07/the-politics-of-curiosity.html" href="http://www.jehsmith.com/1/2012/07/the-politics-of-curiosity.html" rel="noopener noreferrer" target="_blank">
    写
   </a>
   道，
   <strong class="markup--strong markup--p-strong">
    在我们这个时代，“好奇心被国家利用了”
   </strong>
   ：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    因此，好奇心历史的下一章 — 后现代章节开始了。现在的状况是，国家越来越嫉妒个人的好奇心，而不是仅仅为了国家利益而扼杀它。每种能力都必须拥有许可证、以及每个兴趣和社团。
   </em>
  </p>
  <p class="graf graf--p">
   接下来，随着文化和教育领域成为“编制”的一部分，拒绝对它们感到好奇是抵抗性政治的一种形式。
  </p>
  <p class="graf graf--p">
   例如，当莎士比亚或实验室科学成为了精英的文化财产时，文化和知识领域就会产生好奇心的缺乏，否则就会变视为权力的同谋。
  </p>
  <p class="graf graf--p">
   在这里你可以观察到一种确认，即
   <strong class="markup--strong markup--p-strong">
    快乐或健康的好奇心需要一般的平等感
   </strong>
   。
  </p>
  <p class="graf graf--p">
   自卑感导致了探究能力的萎缩，也导致了对社会描述为“优越/优势”之领域的不满。
  </p>
  <p class="graf graf--p">
   尽管对好奇心的学术研究或调查显著缺乏，但我们仍可以利用稍微丰富的无知研究来洞察不正常性。
  </p>
  <p class="graf graf--p">
   由于好奇心是消除无知的欲望的个人和情感表达，好奇心是获得完全拥抱、有意义的知识的必要手段，于是对无知的哲学批评与对不正常性的考虑密切相关。
  </p>

   <img alt="" class="aligncenter size-full wp-image-4657 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/000-2.jpg?resize=1100%2C733&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/000-2.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/000-2.jpg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/000-2.jpg?resize=768%2C512&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/000-2.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/000-2.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/000-2.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="733" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/000-2.jpg?resize=1100%2C733&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4657" data-recalc-dims="1" height="733" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/000-2.jpg?resize=1100%2C733&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/000-2.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/000-2.jpg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/000-2.jpg?resize=768%2C512&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/000-2.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/000-2.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/000-2.jpg?resize=272%2C182&amp;ssl=1 272w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在许多分类中，Agnotologists 描述了三种无知形式：无知的本土状态、无知的可选择性和无知的积极建构。后两者都是通过抑制好奇心才能达成的。
  </p>
  <p class="graf graf--p">
   就像古生物学家谈论故意无知一样，或许现在是时候开始讨论故意的漠视了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当我们遇到故意的冷漠时，必须考虑它是否可能体现了阶级主义、种族主义、性别歧视或其他权力关系，就如那些充满怨恨的人拒绝了解他们所鄙视的人。
   </strong>
  </p>
  <p class="graf graf--p">
   隐瞒好奇心可能成为社会蔑视的标志，也是创造方便的无知以逃避责任的手段。然而，故意的漠视也可能体现了对阶级主义、种族主义或性别歧视所产生的知识的抵制。
  </p>
  <p class="graf graf--p">
   但是，拒绝好奇有一个令人不安的双重性边缘，在提供某种保护的同时也会产生功能障碍和毒性。虽然我们可能会鼓励学生的精神和情感脱离那个不受欢迎的教室，但这种脱离战略的成功也会渗透到生活的其余部分。
  </p>
  <p class="graf graf--p">
   孩子们不太可能在闷闷不乐的状态下度过课堂时间，就如成年人不太可能会在严重的情绪抵触中度过工作日。走出学校或工作场所的大门后，却无法进入健康快乐的可能性探索中。
  </p>
  <p class="graf graf--p">
   心灵和情感的习惯是通过不断的练习和重复来塑造的，并不是那么容易得到或拒绝。
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    <em>
     今天主宰我们心情的无力感和不稳定感直接取代并遏制了好奇心
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p">
   我们的好奇心不仅因故意的漠视而受到侵蚀。
  </p>
  <p class="graf graf--p">
   好奇心需要注意力、存在感和警觉性。一个有意义的活跃的公共领域需要对其他人的好奇心，以及对其他人的积极感知和认可。
  </p>
  <p class="graf graf--p">
   但是，如今我们生活在一个疯狂消磨注意力的世界里，一个真正无法参与进去的世界 —— 毕竟，参与需要耐心和等待。
  </p>
  <p class="graf graf--p">
   在某种意义上，作为冷漠的对立面，注意力和好奇心反过来又需要希望。
  </p>
  <p class="graf graf--p">
   好奇心意味着个人效能感和可能性，相信一个人的好奇心可以通过自己的行为来满足，以及对未来的感觉。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    今天主宰我们情绪的无力感和不连续感直接取代并抢占了好奇心，反而成为了当代资本主义的主导影响：焦虑
   </strong>
   。
  </p>
  <p class="graf graf--p">
   为了覆盖无法参与的焦虑，人们用无目的或无所事事的分心来取代注意力。
  </p>
  <p class="graf graf--p">
   几个月前，课堂教学开始让我感到沮丧，就在我顿悟的时候 ，学生们懒洋洋地滚动着手机，因为我们中的一些人讨论了气候变化和地球的生态状况。
  </p>
  <p class="graf graf--p">
   也许学生们只是想躲避这个恐怖的话题？但是当我们玩游戏时，他们也会有同样的反应，这种冷漠是教育机构暴力摧毁学生好奇心的结果吗？或者这是一种希望的完全失败？对课程政治的反动怨恨？
  </p>
  <p class="graf graf--p">
   如果我们想要摆脱强迫、抵抗、傲慢、愤怒、绝望和焦虑所产生的交际障碍，我们就必须能明白这种状况。
  </p>
  <p class="graf graf--p">
   要做的第一件事就是认识到道德上好的好奇心不仅仅是一种智力和学术上的关注，它也是一种情感和政治状态，迫切需要培养和温柔的爱护。⚪️
  </p>

  </p>
