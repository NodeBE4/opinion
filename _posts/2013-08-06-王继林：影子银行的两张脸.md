<div class="article">
 <div class="body-text">
  <p align="left">
  </p>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    影子银行是与传统商业银行相对应的概念。按
   </span>
   <span style="font-size: 9pt">
    2011
   </span>
   <span style="font-size: 9pt">
    年美国金融稳定委员会
   </span>
   <span style="font-size: 9pt">
    (FSB)
   </span>
   <span style="font-size: 9pt">
    在《影子银行：划定范围》中的定义，影子银行指"银行监管体系之外，可能引发系统性风险和监管套利等问题的信用中介体系"。由于各个国家金融发展阶段以及金融体制的不同，影子银行在不同国家对应着不同的金融机构，但在普遍意义上它们都具有两个明显的特点：一是业务不受或只受很少监管；二是能像传统商业银行一样为实体经济提供融资。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    2007
   </span>
   <span style="font-size: 9pt">
    年下半年美国爆发了席卷全球的次贷危机，随后住宅抵押贷款市场上的金融巨擎房利美和房地美面临破产风险，而具有
   </span>
   <span style="font-size: 9pt">
    150
   </span>
   <span style="font-size: 9pt">
    多年历史的投资银行雷曼兄弟更是宣布破产，影子银行在短时间内被世人所熟知，并且与次贷危机深深的绑架在一起，成为风险的代名词。目前国内影子银行系统也被看做是潜藏在经济背后的巨大风险，对于影子银行的清算与声讨也屡见报端，但一味的强调影子银行体系所积聚的风险，而不是以历史的、发展的眼光看待影子银行问题不免过于片面。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    首先，影子银行的存在与发展是国内多层次资本市场结构缺失以及金融体系垄断的结果。从某种程度上讲影子银行弥补了传统金融体系，使金融市场体现出一定的结构层次特征：不同风险水平的企业理论上都可以从金融市场获得资金支持，而唯一需要确定的只是企业应支付的利率水平。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    建立多层次资本市场一直是国内金融体制改革的目标之一，但目前以银行为核心的间接融资模式仍是企业融资的首选。银行贷款针对的企业客户主要为国企以及一些大型民营企业，中小企业的融资依然面临较大困难。虽然债券市场近几年发展较为迅速，但中小民营企业的融资仍然较为受限，去年私募债的推出也并未在根本上解决中小企业融资难的问题。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    金融市场最为本质的功能即是为实体经济提供融资，而融资不应只是针对资本实力雄厚、经营风险相对较低的大型企业，在一个健康运行的多层次资本市场中，中小企业的融资也应纳入金融市场范畴。影子银行显然弥补了社会融资与银行融资之间的缺口，成为了多层次资本市场中风险较高企业的融资市场。进一步的，风险并非一个万恶的概念，在融资过程透明、风险被充分认知的基础上，让金融机构和实体企业承担一定的风险也是经济健康发展的必要。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    其次，影子银行为实体经济的发展提供了大量的资金支持。在国内经济快速发展的背后，影子银行为经济的发展提供了不可忽视的资金支持。关于影子银行的具体规模并不存在官方权威的统计数据，瑞银中国首席经济学家汪涛在去年
   </span>
   <span style="font-size: 9pt">
    3
   </span>
   <span style="font-size: 9pt">
    季度曾估计当时广义影子银行与规模大约为
   </span>
   <span style="font-size: 9pt">
    24.4
   </span>
   <span style="font-size: 9pt">
    万亿，中金公司在今年
   </span>
   <span style="font-size: 9pt">
    4
   </span>
   <span style="font-size: 9pt">
    月份估计国内广义影子银行的规模大约为
   </span>
   <span style="font-size: 9pt">
    27
   </span>
   <span style="font-size: 9pt">
    万亿左右，从另一方面讲，影子银行向经济体中注入了体量巨大的流动性。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    如果没有影子银行为实体企业提供数以万亿计的资金融通，在银行垄断的前提下，实体经济的发展必将大打折扣，特别是对于投资驱动型的经济体而言，情况更是如此。正是考虑到影子银行在融资功能上的作用，从
   </span>
   <span style="font-size: 9pt">
    2010
   </span>
   <span style="font-size: 9pt">
    年底开始，中央政府在宏观经济调控中开始逐步淡化对于银行贷款总额度的考核，取而代之的是对"社会融资总规模"的重点考察。社会融资总规模不仅包含了银行贷款总额，还包括了委托贷款、未贴现的银行承兑汇票以及信托融资等融资方式
   </span>
   <span style="font-size: 9pt">
    (
   </span>
   <span style="font-size: 9pt">
    但并不包括民间融资、私募基金等
   </span>
   <span style="font-size: 9pt">
    )
   </span>
   <span style="font-size: 9pt">
    。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    再者，影子银行为评估社会真实利率水平提供了重要参照。在国内银行垄断体制下，实行的是利率的双轨制：传统商业银行体制下的利率与影子银行体系下利率并存。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    市场经济有效运行的核心即是价格机制，利率水平的扭曲必将导致资金的错配并造成寻租空间，比如民营企业从银行获得贷款除了要支付名义利率之外，可能还需要支付其他一些费用作为寻租费，这也是国内商业银行，特别是股份制银行中普遍存在的问题，也即银行名义利率低估了市场利率水平。比如温州民间融资综合利率指数显示，
   </span>
   <span style="font-size: 9pt">
    2013
   </span>
   <span style="font-size: 9pt">
    年
   </span>
   <span style="font-size: 9pt">
    7
   </span>
   <span style="font-size: 9pt">
    月
   </span>
   <span style="font-size: 9pt">
    26
   </span>
   <span style="font-size: 9pt">
    日
   </span>
   <span style="font-size: 9pt">
    温州民间融资综合利率指数为
   </span>
   <span style="font-size: 9pt">
    19.63%
   </span>
   <span style="font-size: 9pt">
    ，远高于银行一年期贷款基准利率
   </span>
   <span style="font-size: 9pt">
    6%
   </span>
   <span style="font-size: 9pt">
    的水平，而从
   </span>
   <span style="font-size: 9pt">
    2011
   </span>
   <span style="font-size: 9pt">
    年上半年至今，民间融资综合利率水平在绝大部分时间里均处于
   </span>
   <span style="font-size: 9pt">
    20%-26%
   </span>
   <span style="font-size: 9pt">
    的区间。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    从交易的供需层面看，民间借贷市场上资金供需双方对于利率水平的达成更加突出了竞争的作用，而传统商业银行贷款利率的决定更加突出垄断的力量，所以从某种程度上讲影子银行体系的利率水平相比银行贷款利率更为接近市场均衡利率水平。就像国内在市场经济初期经历的价格双轨制一样，利率双轨制也会随着利率市场化逐步消失，从而才能实现金融的真正市场化。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    由于影子银行在运行过程中缺乏透明性以及监管上存在的盲区，使其积累了大量的潜在风险，成为目前国内经济运行的不确定性因素。但在防范影子银行风险的同时，有必要正确的评价影子银行在国内垄断金融体制下为经济发展所作出的贡献。比起对影子银行潜在风险的担忧，国内金融垄断体制更应引起人们的关注和反思。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    目前利率市场化明显加快步伐，影子银行的生存与发展前景也倍受质疑，但从金融发展的历史角度看，影子银行不会消失，而是进一步创新以适应金融市场环境的改变。金融工具的"衍生化"可能会成为影子银行发展的方向，如何为影子银行的健康发展提供良好的制度安排以及如何把影子银行的运行纳入监管范畴才是真正该人们思考的问题，而不是一味的强调其本身的风险性，从而企图抑制影子银行的发展，因为他们已经，也应该属于整体经济中不可分割的一部分。
   </span>
  </div>
