  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     部分重现和遗忘
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   当你没时间复习考试相关的所有材料时，你可能已经注意到了这种效应。那些你没有复习到的问题将成为考试中最难回答的问题。
  </p>
  <p class="graf graf--p">
   👉即便一个人将大量关于某件事的信息记得滚瓜烂熟，如果你先问他该事件的某一个细节问题，然后再让他回想另一个特征细节，就会变得更加困难。
  </p>
  <p class="graf graf--p">
   同样，在庭审中，如果先问证人关于罪犯戴的眼镜的形状，那么随后证人就可能很难描述出罪犯穿的外套的颜色特征了。
  </p>
  <p class="graf graf--p">
   律师经常这么干 —— 使用这种方法在法庭上迷惑证人，如果证人对罪犯只有匆匆的一瞥，那就更是如此了。
  </p>
  <p class="graf graf--p">
   👉通过询问细枝末节的信息，并在 **恰当的时候** 打断，一个善于操纵他人的律师就能准确地打击证人的信息，并在证词中指出证人 “前后矛盾”。
  </p>
  <p class="graf graf--p">
   🌟如果你想让某人忘记你提出的某个问题，那么在下次与他们的会面中，你可以将所有 **其他问题** 重复一遍；这样一来对方就会把你希望他忘掉的那个问题抛在脑后了。或至少能在他记忆中变得模糊。
  </p>
  <p class="graf graf--p">
   对于调查人员和情报分析师来说，这点也是非常重要的。对材料的检查一定要细心：那些没有检查到的内容就会从你的记忆中消失。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     游戏 —— 火柴造型（等级3）
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   继续升级火柴游戏！
  </p>
  <p class="graf graf--p">
   你需要努力记住火柴的位置，将它们按照不同的几何形状进行分组。也许随机散落的火柴能让你想起某些事？那就努力回想那个故事，会很有用。
  </p>
  <p class="graf graf--p">
   如果你觉得自己做得不够好，也不要放弃；或者您还可以回到上一级的练习中重新再来。这不是线性的。
  </p>
  <p class="graf graf--p">
   记得要不断练习哦，越重复越熟练。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     Zeigarnik 效应
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   Zeigarnik 效应认为，未完成的或者被打断的任务，比起那些已经完成的任务更容易被记住。
  </p>
  <p class="graf graf--p">
   心理学家 Kurt Lewin 和 Blum Zeigarnik 在一家咖啡馆注意到，服务员并没有用笔记下他们点的东西，但随后送上来的东西分毫不差。
  </p>
  <p class="graf graf--p">
   然而，当他们询问服务员刚刚用完离开的顾客点了什么东西时，服务员却一点都想不起来了。
  </p>
  <p class="graf graf--p">
   👉随后的实验表明，与已完成的任务相比，被打断的任务被记住的概率可以高出一倍。这种 “未完成行动效应” 也许可以用保持激励性紧张的状态来解释，因为激励性紧张状态能激活记忆力 —— 任务完成后，激励就消失了，人们就会忘记与该任务相关的一切。
  </p>
  <p class="graf graf--p">
   Zeigarnik 效应在实践中非常有用。比如，如果你正在撰写一段长文，然后停在某一章节没有写完，那么下次再写时你就很容易进入写作姿态。未完成的行动可以帮你更快速地回忆起上次停笔的位置，以及接下来想写的内容。
  </p>
  <p class="graf graf--p">
   👉未完成行动效应也适用于与他人的互动。你可以就此操纵对方的注意力。
  </p>
  <p class="graf graf--p">
   🌟在交谈的中间部分是最为紧张的，这时候还没有得出结论，如果你就此中断对话，那么对话者受到的影响就会更大 —— 他们更有可能回顾这个问题，不断思考，也更有可能同意你的观点。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     催眠性迷睡
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   这来自KGB的研究：催眠性迷睡（ hypnotic trance）是一种将睡眠和意识结合起来的特殊心理状态。
  </p>
  <p class="graf graf--p">
   它伴随着更强烈的暗示性：在催眠状态下，人们可能很容易理所当然地相信那些在非催眠状态下他们永远也不会相信的事情。
  </p>
  <p class="graf graf--p">
   现代人对催眠的理解主要基于 19世纪法国精神病学家让·沙尔科（ Jean Charcot）和伊波利特·贝尔南（ Hippolyte Bernheim）的研究成果。
  </p>
  <p class="graf graf--p">
   俄罗斯科学家 V. M.贝特列夫（ V. M. Behterev）也研究催眠。
  </p>
  <p class="graf graf--p">
   然而，尽管长期以来，科学领域对这种现象非常感兴趣，但对催眠的机制却知之甚少。
  </p>
  <p class="graf graf--p">
   在表演期间对大规模观众使用催眠术的案例表明，催眠术只对一部分人有用，对另一部分人则无效。
  </p>
  <p class="graf graf--p">
   很可能不同的人受到催眠影响的程度不同。
  </p>
  <p class="graf graf--p">
   著名催眠师 Wolf Messing 一般会为自己的表演选择容易受他人影响的观众。在 Messing 的表演中，被催眠的人躺在两把相对的椅子上，只靠头部和脚支撑。他们可以承受放在自己肚子上几十千克的重物。
  </p>
  <p class="graf graf--p">
   他们不会感到痛苦，与平时相比他们会透露更多关于自己的信息，并为催眠师执行最奇怪的任务。
  </p>
  <p class="graf graf--p">
   许多催眠师声称他们可以随时催眠任何人，但事实并非如此。
  </p>
  <p class="graf graf--p">
   催眠术发展到今天的水平，还是需要被催眠的人帮助催眠师引发迷睡状态。
  </p>
  <p class="graf graf--p">
   包括军事科学家在内的科学家们正在研究新的催眠方法，这种方法可以向任何人施加影响，让其违背自己的意愿做事，但目前尚未被开发出来。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     调查人员的其他常见记忆法
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   遗忘是记忆力一个至关重要的功能，它可以保护大脑免受信息超载的影响。
  </p>
  <p class="graf graf--p">
   首先，每个人都会忘记那些很长时间都没有使用过的过剩信息。这些信息不可能用原始形式储存，那是不现实的，因此大脑会对其进行处理和泛化。
  </p>
  <p class="graf graf--p">
   有时候这种泛化以直觉的形式进行 —— 就是无法用逻辑来解释的那种感觉。通过这种方式，
   <strong class="markup--strong markup--p-strong">
    大脑基于我们已经遗忘的类似情况的经历，给我们提供了直觉的线索。
   </strong>
  </p>
  <p class="graf graf--p">
   其次，我们会倾向于忘记所有令人不快的信息；那些痛苦的回忆或违背个人价值观和信仰的行为，都会被无意识地从记忆中赶走，以便恢复心理上的舒适感。（这是说的普通人，而抑郁症患者在此恰恰相反）
  </p>
  <p class="graf graf--p">
   遗忘行为具有时间依赖性，从收到信息的那一刻起，经过的时间越长，遗忘的可能性就越大。
  </p>
  <p class="graf graf--p">
   👉对于追踪者来说，如果一个追踪项目长时间终止（以各种原因，比如资金短缺、其他案件继续出现没有时间等等），就可能会对该案件造成很大的影响 —— 这就是为什么我们说，
   <strong class="markup--strong markup--p-strong">
    时间，是侦查反侦察行动中最有价值的东西之一，只要你能抢到足够多的时间，就更有可能拖垮对手，就此结束游戏
   </strong>
   。

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     地点法
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   故事法并非是记住词语表的唯一方法。
  </p>
  <p class="graf graf--p">
   故事本身并不重要 —— 它只是将词语联系在一起的手段，将词语编码成图像、并置于带有情感色彩的情境中。
  </p>
  <p class="graf graf--p">
   👉成功的记忆需要使用关联性、意象和情感，而这些方法也是另一种方法 —— 地点法的基础。
  </p>
  <p class="graf graf--p">
   故事法将图像与故事情节联系起来，
   <strong class="markup--strong markup--p-strong">
    地点法则将它们放置在一个熟悉的空间里：一所房子、一个房间、或一条街上
   </strong>
   。
  </p>
  <p class="graf graf--p">
   地点法历史悠久。古罗马演说家 Marcus Tullius Cicero 曾经讲述过一个传说，提到地点法是由公元5～6世纪来自凯阿岛的希腊诗人 Simonides 发明的。
  </p>
  <p class="graf graf--p">
   Simonides 在出席一次盛宴时被一些不明人士叫走。诗人刚走出屋子，屋顶就倒塌了。其他所有客人因此丧命。人们无法辨认亲人的遗体，但 Simonides 记住了餐桌上每个人的座次，从而帮助人们找到了遗体。
  </p>
  <p class="graf graf--p">
   Cicero 在 “On the Orator” 中描述了这一事件，得出结论：要想记住一系列对象，就有必要构建这些对象的心理图像，并将他们放置在一个想象的地点中。
   <strong class="markup--strong markup--p-strong">
    空间的顺序也将记录记忆对象的顺序
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Cicero 本人在记忆自己的演讲稿时就使用了地点法。排练演讲时，他走遍自己的家，将每个房间与一个特定的主题或想法相关联。
  </p>
  <p class="graf graf--p">
   使用地点法的另一个例子是 Giulio Camillo 的 “Theatre of Memory”。16世纪文艺复兴期间这位哲学家和炼金术士试图建立一个全面的人类知识体系，将其置于一个特殊的建筑 —— 记忆剧场中。
  </p>
  <p class="graf graf--p">
   每个知识领域都在记忆剧场里占据一席之地，所有领域互相联系，共同形成一幅完整图景。
  </p>
  <p class="graf graf--p">
   Camillo 甚至着手为“剧场”建造了一座真实的木质建筑，但最终没有完成。他也未能详细阐述“剧场”的基本原理。然而，
   <strong class="markup--strong markup--p-strong">
    将知识及其各部分之间的联系进行可视化呈现
   </strong>
   的理念，对 Camillo 同时代的人们产生了重大影响。
  </p>
  <p class="graf graf--p">
   一直以来，记忆的艺术都与政府、国际关系、和情报机构有关。
  </p>
  <p class="graf graf--p">
   凯阿岛的 Simonides 写过一组关于希腊-波斯湾战争的诗歌，因此在希腊具有一定的政治影响力。此外他还是一名外交官，负责解决冲突并防止杀戮。Simonides 是罗马政治中的重要人物，其卓越的口才和记忆力令他备受瞩目，并当选成为执政官。
  </p>
  <p class="graf graf--p">
   亚里士多德、塞内加、奥古斯丁、大阿尔伯图斯、托马斯阿奎纳、Giordano Bruno、Wilhelm Leibniz、笛卡尔 —— 都为记忆理论和实践做出过贡献，当然还有许多其他人没有被提及。他们每个人在自己的时代都极具影响力，其中一些人还服务于情报部门。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     尝试一下
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   👉 为了掌握地点法，一开始最好选择一个小的、熟悉的地方：比如你的公寓。想象一下这个地方。在其中找到你能定位和安放各类物品的区域。
  </p>
  <p class="graf graf--p">
   比如，你的公寓内有一个门厅、一个客厅、一条走廊、一个厨房和一个浴室。**在你脑海中** 将每个房间以逆时针的方向巡视一遍 —— 如果你觉得比较难，那就亲自走一圈，仔细观察每一个房间。
  </p>
  <p class="graf graf--p">
   即使是在一个小公寓里也能找到二三十个放置物品的地方。显然，你的公寓和其中物品可能有所不同，但关键是你应该清楚地了解它们。
  </p>
  <p class="graf graf--p">
   还记得下面这个词汇表吗？—— 想象它是一个密码：
  </p>
  <p class="graf graf--p">
   烟花
  </p>
  <p class="graf graf--p">
   橙子
  </p>
  <p class="graf graf--p">
   汽车
  </p>
  <p class="graf graf--p">
   时钟
  </p>
  <p class="graf graf--p">
   沙发
  </p>
  <p class="graf graf--p">
   油画
  </p>
  <p class="graf graf--p">
   代数
  </p>
  <p class="graf graf--p">
   博士
  </p>
  <p class="graf graf--p">
   杂志
  </p>
  <p class="graf graf--p">
   员工
  </p>
  <p class="graf graf--p">
   摩天大楼
  </p>
  <p class="graf graf--p">
   陨石
  </p>
  <p class="graf graf--p">
   👉快速记住这个词语表的方法是什么？—— 把它们全部放进你的 “公寓” 里，安置在每一个恰当的地方，讲一个故事。试试看。
  </p>
  <p class="graf graf--p">
   也许画面不太合理，但它的确能帮你快速记忆这个词语表，其关键在于，将词语表中每一个词都 *放在* 你容易记忆的 *空间* 中。
  </p>
  <p class="graf graf--p">
   🌟请注意故事中所使用的助记法的两个主要原则：
  </p>
  <p class="graf graf--p">
   1、
   <strong class="markup--strong markup--p-strong">
    相关性
   </strong>
   —— 所有物品都与一个熟悉的地方相联系，有时物品之间还存有一定的联系。
  </p>
  <p class="graf graf--p">
   2、
   <strong class="markup--strong markup--p-strong">
    情绪
   </strong>
   —— 你害怕被烟火灼伤；你想要橙汁，但一想到酸味和果汁粘在杯子上的样子就会面露苦色；你为生锈的汽车感到心疼；你对那个总想找个沙发坐的胖子感觉惊讶；你对藏在油画后面的东西感觉好奇；同时也为博士担心，因为他必须携带一大堆厚重的杂志 ……
  </p>
  <p class="graf graf--p">
   就是这样。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     使用地点法
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   👉要想使用地点法，你必须对某个地点非常熟悉。
  </p>
  <p class="graf graf--p">
   你必需可以轻松地在这个地方穿梭，在不同房间之间神游，并能清楚地记得接下来的一切。
  </p>
  <p class="graf graf--p">
   具体来说，光记得门后有衣柜是不够的。你必须能想象关于这个衣柜的一切细节 —— 它的大小、形状、颜色、触摸时的感觉、柜门开合时的声音、以及柜内的气味。
  </p>
  <p class="graf graf--p">
   👉最好对地点进行选择，以便让它们有所不同。因为相似的或者并排放置的书柜可能会对你的记忆形成干扰，增加了混淆不同物品的风险。
  </p>
  <p class="graf graf--p">
   学习地点法时，最好选择真实存在的地点，特别是那些你可以实地探访的地方。
  </p>
  <p class="graf graf--p">
   🌟一旦你掌握了这个方法，就可以使用完全虚构的世界了。
  </p>
  <p class="graf graf--p">
   确保它们容易被记住，要彻底想清楚所有细节。经常在大脑里 “探访” 这个地方，这些虚构的地点会对你产生很大的帮助。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   选择三个位置来练习地点法。在心里把它们都 “逛” 一遍。想象每一个地点、每一件物品。
  </p>
  <p class="graf graf--p">
   你所在位置的所有东西都有自己的尺寸、形状、颜色、重量、质地和气味。使用这些东西：移动想象中的家具、敲一敲想象中的桌子、打开想象中的门、摸一下门把手。
  </p>
  <p class="graf graf--p">
   先使用实际存在的地点，熟悉之后就可以使用完全想象出来的地点了，你可以在其中 “安置” 任何东西。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff0000;">
    <strong class="markup--strong markup--p-strong">
     测试！ —— 快速回答
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    在克格勃科学顾问看来，在暗示性技巧改进领域中，军事心理学家研究的主要方向是什么？
   </span>
  </p>
  <p class="graf graf--p">
   （A）在催眠性迷睡中消除对已感知信息的怀疑
  </p>
  <p class="graf graf--p">
   （B）在催眠性迷睡中消除疼痛
  </p>
  <p class="graf graf--p">
   （C）在催眠性迷睡中增强人们的体力
  </p>
  <p class="graf graf--p">
   （D）在没有催眠者的帮助下催眠任何人的能力
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-11339 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-6.jpg?resize=1100%2C733&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-6.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-6.jpg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-6.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-6.jpg?resize=768%2C512&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-6.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-6.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="733" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-6.jpg?resize=1100%2C733&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     身体记忆
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   如果要记住的是一个简短的清单，你还可以使用自己的身体部位。
  </p>
  <p class="graf graf--p">
   👉假设你正在准备一次公开演讲，需要记住提纲。你可以将第一部分演讲内容的画面放在脚上，绑在鞋子上，第二部分内容和膝盖连接起来，第三部分贴在屁股上。
  </p>
  <p class="graf graf--p">
   不要笑这个方法很好用，因为你对自己的身体会很熟悉。你可以在身上放置不少于10件物品：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    脚
   </li>
   <li class="graf graf--li">
    膝盖
   </li>
   <li class="graf graf--li">
    臀部
   </li>
   <li class="graf graf--li">
    腰部
   </li>
   <li class="graf graf--li">
    腹部
   </li>
   <li class="graf graf--li">
    胸部
   </li>
   <li class="graf graf--li">
    肩膀
   </li>
   <li class="graf graf--li">
    脖子
   </li>
   <li class="graf graf--li">
    脑袋
   </li>
  </ul>
  <p class="graf graf--p">
   👉这种方法可以防止你在公开演讲时思路丢失。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习 —— 地点法记忆词语表（等级1）
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   通过这个简单的练习掌握地点法。记忆清单的时候，在心里将相关概念放在一个熟悉的空间里，尝试使用自己的身体作为助记空间。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习 —— 地点法记忆（等级2）
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   使用地点法掌握记忆词语的技巧。这一次，想象你的房间或公寓的空间。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     利弊
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   您已经熟悉了记忆清单的两种方法：故事法和地点法。它们有共同之处，最重要的共同点是，它们都满足助记法的两个基本原则：关联性和情感。
  </p>
  <p class="graf graf--p">
   但也有不同之处：在故事法中，关联是由一个对象到另一个对象那样构建的；而在地点法中，是从对象到其在虚构地点中的位置进行构建。
  </p>
  <p class="graf graf--p">
   👉同时使用这两种方法效果将会最好。也许你会选择其中一个，这完全取决于你，每种方法都有其优缺点。
  </p>
  <p class="graf graf--p">
   地点法的优势在于，在从已记住的清单中寻找某个对象时，你不需要把整个故事都回忆一遍。
  </p>
  <p class="graf graf--p">
   地点法的局限性在于，首先，你需要通过想象一个很大的空间来记住一个长长的清单；其次，你需要掌握很多熟悉的空间，以避免混淆不同的清单。但与故事法相比，人们依旧更喜欢它。
  </p>
  <p class="graf graf--p">
   当然，同一空间可以反复使用；一旦记住的清单不再有用，你可以在脑海中游览一下这个空间，并通过删除图像将其清空。然后再检查一下，确保没有任何东西留下了。下一次你就可以重新使用它了。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习 —— 地点法记忆（等级3）
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   尝试记住更长的清单，缩短记忆时间。每记住3～4个清单后切换方法。
  </p>
  <p class="graf graf--p">
   交替使用地点法和故事法记忆词语表。每次尝试每个词语只用3秒钟来记忆。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     游戏 —— 套话
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   找一整天空闲的时间，随手打车，随意逛整个城市。
  </p>
  <p class="graf graf--p">
   👉结束后您需要回答的问题是：出租车司机的姓名、出生地、住址、是否结婚、是否有孩子、孩子多大了、他们对工作的喜欢程度、对某个社会热点议题的观点，等等。越详细越好。
  </p>
  <p class="graf graf--p">
   不要用任何物理方法记录，只用脑记。
  </p>
  <p class="graf graf--p">
   这需要您首先撬开司机的话匣子。关键是找到对方感兴趣的话题，*让他*感兴趣而不是你，之后他就很可能侃侃而谈了。试试看，有些人一开始说就会停不下来的 ……
  </p>
  <p class="graf graf--p">
   下一部分内容中我们将进入专业级训练，请准备好。⚪️
  </p>
