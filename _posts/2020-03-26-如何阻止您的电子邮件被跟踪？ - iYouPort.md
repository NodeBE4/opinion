  <p class="graf graf--p">
   您的收件箱中挤满了所有令人讨厌的营销电子邮件，但它们不仅仅是为了推销产品。
  </p>
  <p class="graf graf--p">
   ⚠️它们还使用 MailChimp 之类的软件将跟踪器嵌入到邮件中，从而跟踪您是否已打开电子邮件、何时打开电子邮件、以及当时的位置。
  </p>
  <p class="graf graf--p">
   这种跟踪器是如何工作的？
  </p>
  <p class="graf graf--p">
   单个跟踪像素嵌入到电子邮件中通常（但不总是）隐藏在图像或链接里。目标人打开电子邮件后，跟踪代码会将信息发送回公司的服务器。
  </p>
  <p class="graf graf--p">
   已经进行了一些尝试来限制可以通过这种方式发送的信息量。例如，开发了诸如
   <a class="markup--anchor markup--p-anchor" data-href="https://uglyemail.com/" href="https://uglyemail.com/" rel="noopener noreferrer" target="_blank">
    Ugly Mail
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://chrome.google.com/webstore/detail/pixelblock/jmpmfcjnflbcoidlgapblgpgbilinlem" href="https://chrome.google.com/webstore/detail/pixelblock/jmpmfcjnflbcoidlgapblgpgbilinlem" rel="noopener noreferrer" target="_blank">
    PixelBlock
   </a>
   之类的扩展程序，以阻止 Chrome 和 Firefox 上的跟踪器。
  </p>
  <p class="graf graf--p">
   您还可以采取一个简单的基本步骤来避免跟踪器：阻止电子邮件自动加载图像，因为图像是大部分这些像素跟踪器隐藏的位置。
  </p>
  <p class="graf graf--p">
   您也许无法避免所有可能以这种方式隐藏在电子邮件中的跟踪器，但是这样做可以有效阻止其中大部分跟踪器。
  </p>
  <p class="graf graf--p">
   在主要的台式机和移动电子邮件应用程序中操作，方法如下：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、在GMAIL中禁用图像自动加载
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    点击右上角的齿轮图标以访问您的设置
   </li>
   <li class="graf graf--li">
    在“常规”标签中，向下滚动到“图片”
   </li>
   <li class="graf graf--li">
    选择“在显示外部图像之前询问”
   </li>
   <li class="graf graf--li">
    向下滚动到页面底部，然后单击“保存更改”
   </li>
  </ul>
  <p class="graf graf--p">
   请注意，这还将关闭 Gmail 的动态电子邮件功能，它是使电子邮件更具交互性的功能。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、在 MICROSOFT OUTLOOK（OFFICE 365）中禁用图像自动加载
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    点击“文件”&gt;“选项”
   </li>
   <li class="graf graf--li">
    在 “Outlook 选项” 窗口中，选择 “Trust Center”
   </li>
   <li class="graf graf--li">
    点击 “信任中心设置” 按钮
   </li>
   <li class="graf graf--li">
    选中标记为 “不要在标准HTML消息或RSS项目中自动下载图片” 和 “不要下载经过加密或签名的HTML电子邮件中的图片”的框。如果需要，可以选中第一个项目下面的复选框，从而对第一个项目进行一些例外处理。
   </li>
  </ul>

   <strong class="markup--strong markup--p-strong">
    3、在 MICROSOFT 邮件中禁用图像自动加载
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    点击左下角的齿轮图标
   </li>
   <li class="graf graf--li">
    从右侧的“设置”菜单中，选择 “Reading pane”
   </li>
   <li class="graf graf--li">
    在右侧菜单的底部，关闭 “自动下载S / MIME Mail 以外的外部图像和样式格式”、以及 “自动下载S / MIME 邮件的外部图像和样式格式”
   </li>
   <li class="graf graf--li">
    如果您使用了多个电子邮件帐户，请确保选中 “应用于所有帐户”
   </li>
  </ul>

    4、在苹果邮件中禁用图像自动加载

    点击您要使用的电子邮件帐户
   </li>
   <li class="graf graf--li">
    向下滚动并选择“图像”

    <img alt="" class="aligncenter size-full wp-image-10484" data-recalc-dims="1" height="996" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-6.png?resize=1100%2C996&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-6.png?w=1328&amp;ssl=1 1328w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-6.png?resize=300%2C272&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-6.png?resize=768%2C695&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-6.png?resize=1024%2C927&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-6.png?resize=1100%2C996&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   从2019年9月3日起，Google还将设置添加到了iOS版本的Gmail中。
   <strong class="markup--strong markup--p-strong">
    在iPhone客户端上禁用它的方法如下
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    打开iOS版Gmail，点击左上方的菜单，然后向下滚动至设置
   </li>
   <li class="graf graf--li">
    选择您使用的帐户，然后点击“图像”
   </li>
   <li class="graf graf--li">
    从 “始终显示外部图像” 切换为 “显示外部图像之前询问”
   </li>
  </ul>
  <p class="graf graf--p">
   请注意，目前看来，它仅适用于个人帐户，不适用于通过G Suite管理的企业帐户。
  </p>
  <p class="graf graf--p">
   6、
   <strong class="markup--strong markup--p-strong">
    在IPHONE上禁用图像自动加载
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    转到 “设置”&gt;“邮件”
   </li>
   <li class="graf graf--li">
    找到“消息”部分，然后关闭 “加载远程图像”
   </li>
  </ul>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10485 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-5.png?resize=1100%2C970&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-5.png?w=1352&amp;ssl=1 1352w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-5.png?resize=300%2C264&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-5.png?resize=768%2C677&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-5.png?resize=1024%2C903&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-5.png?resize=1100%2C970&amp;ssl=1 1100w" data-recalc-dims="1" height="970" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-5.png?resize=1100%2C970&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10485" data-recalc-dims="1" height="970" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-5.png?resize=1100%2C970&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-5.png?w=1352&amp;ssl=1 1352w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-5.png?resize=300%2C264&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-5.png?resize=768%2C677&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-5.png?resize=1024%2C903&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-5.png?resize=1100%2C970&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   另一种选择是
   <strong class="markup--strong markup--p-strong">
    使用更安全的电子邮件客户端
   </strong>
   ，例如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.thunderbird.net/en-US/" href="https://www.thunderbird.net/en-US/" rel="noopener noreferrer" target="_blank">
    Thunderbird
   </a>
   ，默认情况下会阻止远程图像加载。
  </p>
  <p class="graf graf--p">
   该应用程序允许您单独下载嵌入的内容，或者将自己信任的联系人列入白名单，以免在其图像中发送隐藏的跟踪器。
  </p>
