  <p class="graf graf--p">
   Doxing 是一种使用互联网上的来源追踪某人或收集有关个人信息的技术。其名称源自“Documents”或“Docx”。Doxing 方法完全基于监视者/攻击者/社交工程师识别有关其目标的有价值信息的能力，并将此信息用于实现他们的利益。它也是基于这样的想法：
  </p>
  <blockquote>
   <p class="graf graf--p">
    <em>
     <strong class="markup--strong markup--p-strong">
      “你对目标的了解越多，找到他或她的弱点就越容易”
     </strong>
     — —
     <a class="markup--anchor markup--p-anchor" data-href="http://www.urbandictionary.com/define.php?term=doxing" href="https://www.urbandictionary.com/define.php?term=doxing" rel="noopener noreferrer" target="_blank">
      Urban dictionary
     </a>
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p">
   然后就可以利用这些弱点以
   <strong class="markup--strong markup--p-strong">
    发起最有效的攻击
   </strong>
   。根据对手的不同，挖掘过程也大不相同。例如，分析信息可用于情报机构的秘密尝试；如果这些尝试都失败了并且你仍然热衷于批评政府，那么请期待政府支付的trolls（以及当前繁荣时期的例外主义）对你进行激进的妖魔化。
  </p>
  <p class="graf graf--p">
   Doxing 也是侦察的一部分，这种侦查被称为“人工情报 — ‘humint”，其重点在于目标人/群体的阴影面。这些技术可以或简单或复杂，并且可用于背景检查（渗透可能在这些技术中仍然不可见，但可能会检测到比较小的渗透）关于情报的分类和方法介绍详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E8%B0%B7%E6%AD%8C%E5%9C%B0%E5%9B%BE-%E6%98%AF%E6%AF%94%E8%A9%B9%E5%A7%86%E6%96%AF%E9%82%A6%E5%BE%B7%E6%9B%B4%E6%A3%92%E7%9A%84%E9%97%B4%E8%B0%8D-9df6eb4f6527" href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e5%9c%b0%e5%9b%be%e6%98%af%e6%af%94%e8%a9%b9%e5%a7%86%e6%96%af%e9%82%a6%e5%be%b7%e6%9b%b4%e6%a3%92%e7%9a%84%e9%97%b4%e8%b0%8d/" rel="noopener noreferrer" target="_blank">
    “谷歌地图”是比詹姆斯邦德更棒的间谍
   </a>
   》
  </p>
  <p class="graf graf--p">
   有不少专门为 d0xing 创建的工具。我更愿意使用尽可能少的自动化工具。因为我更愿意关注那些容易被忽视的细节 — —
   <strong class="markup--strong markup--p-strong">
    你越依赖自动的任何东西，你就越容易忽视细节。
   </strong>
  </p>
  <p class="graf graf--p">
   但是如果你想使用一个工具并且你不知道用什么，请做一些研究，（如果可能的话）跟踪制造商的资金，以评估代码的安全性。
  </p>
  <p class="graf graf--p">
   本指南收集 doxing 资源并作出简要介绍，如果您还有其他更好玩的资源，欢迎留言添加！
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    — — 在研究期间存储信息 — —

  <p class="graf graf--p">
   您的
   <strong class="markup--strong markup--p-strong">
    思考方式
   </strong>
   是这样的：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    收集你所知道的关于一个人的
    <strong class="markup--strong markup--li-strong">
     一切
    </strong>
    。在自我人肉中是关于您自己的一切；
   </li>
   <li class="graf graf--li">
    坚持信息收集的法律方法 — — 不要冒险入狱；
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     使用与平常不同的浏览器
    </strong>
    ，或者，删除浏览器历史记录，Cookie并清除缓存；
   </li>
   <li class="graf graf--li">
    首先使用工作表进行注释，或使用草稿纸绘制图表；
   </li>
   <li class="graf graf--li">
    为调查创建一个
    <strong class="markup--strong markup--li-strong">
     单独的文件夹
    </strong>
    ，并以您调查的主题命名；
   </li>
   <li class="graf graf--li">
    将整个网站的照片、头像、屏幕抓取、和 wgets 保存到您的文件夹中。有许多免费的浏览器扩展程序可以全屏抓取，例如 Firefox 的 Awesome Screenshot 或 Chrome 的 Screen Capture Extension。
   </li>
   <li class="graf graf--li">
    为您的研究创建一个
    <strong class="markup--strong markup--li-strong">
     工作文件
    </strong>
    。将每个 URL 和一段信息转储到其中。对于您添加的每条信息，请提供一个来源。定期组织该文件以将最重要信息放在顶部。制定一个 TL；
   </li>
   <li class="graf graf--li">
    为最重要的信息制作
    <strong class="markup--strong markup--li-strong">
     摘要文件
    </strong>
    ；
   </li>
   <li class="graf graf--li">
    Drive 或 Dropbox 对于与其他调查人员的
    <strong class="markup--strong markup--li-strong">
     协作
    </strong>
    非常有用 — — 调查记者、OSINT独立团队、真相爱好者、私人侦探……
   </li>
  </ul>
  <p class="graf graf--p">
   为什么要这样做？
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    如果你觉得某人可能是危险的或不值得信任的；
   </li>
   <li class="graf graf--li">
    如果一个人或多个人在网上骚扰您，并且您想进行风险评估，在此之前您需要了解您的对手。此外
    <a class="markup--anchor markup--li-anchor" data-href="https://www.mosaicmethod.com/" href="https://www.mosaicmethod.com/" rel="noopener noreferrer" target="_blank">
     MOSAIC 威胁评估系统
    </a>
    是一个详细的指南；
   </li>
   <li class="graf graf--li">
    如果你正在与一个人互动并怀疑他们可能正在进行恶作剧，您要证实这件事；
   </li>
   <li class="graf graf--li">
    如果您的判断是一个人对您所处的社区有危险，或者怀疑其为政府间谍，那么如果您愿意，您可以提供一系列支持性证据；
   </li>
   <li class="graf graf--li">
    还有就是本指南的主题：自我人肉，以确定您应该防备什么。
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    可能有用的信息
   </strong>
   ，如下：
  </p>
  <p class="graf graf--p">
   – 姓名：列出您认识的人使用过的所有姓名。
  </p>
  <p class="graf graf--p">
   – 屏幕名：昵称、社交媒体用户名等。这些信息可能会显示某些模式线索。
  </p>
  <p class="graf graf--p">
   – 社交媒体资料和网页：Twitter，FB，Foursquare，Flickr 等。
  </p>
  <p class="graf graf--p">
   – 电子邮件地址：列出它们；另外，搜索它们。
  </p>
  <p class="graf graf--p">
   – 域名：使用 whois。主域名可能会显示其名称或位置。
  </p>
  <p class="graf graf--p">
   – IP地址：可以搜索它们或通过它们关联活动。它们可能位于博客评论管理面板、服务器日志或电子邮件标头中。
  </p>
  <p class="graf graf--p">
   – 年龄/ DOB：出生日期很有用;查阅法庭记录是必要的。
  </p>
  <p class="graf graf--p">
   – 居住地：找到目标人居住的城市和地区。
  </p>
  <p class="graf graf--p">
   – 电话号码：收集当前和过去的号码。做反向查找。
  </p>
  <p class="graf graf--p">
   – 地址：工作地点，学校（注意 LinkedIn。如果您已登录，目标可能会看到您已查看其页面）
  </p>
  <p class="graf graf--p">
   – 互粉的“朋友“：查看目标人的 Facebook，Twitter 关系网，你能得出有用的结论吗？
  </p>
  <p class="graf graf--p">
   – 家庭成员：用于对身份进行三角测量式的定位。家谱网站可以在这里提供帮助。 Ancestry.com很好。在某些情况下，可以选择联系家庭成员，要求他们以富有同情心和非威胁性的方式帮助您的调查。
   <strong class="markup--strong markup--p-strong">
    一些便衣警察经常这么干
   </strong>
   。
  </p>
  <p class="graf graf--p">
   – 其他在线环聊，邮件列表，社区 ……所有你能找到的地方都要看。
  </p>
  <p class="graf graf--p">
   – 究竟是不是目标人：跟踪和区分不是您的目标的但是具有相似名字的人 — 有助于确保收集的信息属于正确的个人。
  </p>
  <p class="graf graf--p">
   – 法院记录：寻找法院和监狱数据库。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    以上需要提示的是：它应该同时被反过来看，也就是说为了防御，除了最后两条之外，其余的您都应该想办法避免被不可靠的人知道。尤其是这些信息都不应该出现在互联网上！
   </strong>
  </p>
  <p class="graf graf--p">

    — — 搜索提示 — —

   </strong>
   如果您有电子邮件地址，则可以安装此浏览器扩展程序以查找与其关联的其他社交帐户。安装后，创建一个包含相关地址的草稿电子邮件。将鼠标悬停在电子邮件地址上会弹出报告侧栏信息。
  </p>
  <p class="graf graf--p">
   Check references：如果您查到目标人在全国各地的任何组织中做志愿者工作，请联系该组织的某个人询问他们与该人的经历。这可以在不泄露您的怀疑或恐惧的情况下以中立的方式完成，并且也是一种合理的方式来检查一个人对您的组织志愿工作的参考。
  </p>
  <p class="graf graf--p">
   搜索姓名+逮捕，姓名+骚扰，姓名+监狱等等：这有时可以快速获得有关滥用和/或犯罪活动的更多信息。
  </p>

    http://archive.org
   </a>

   </strong>
  </p>
  <p class="graf graf--p">
   多重身份对于正义者和恶意者来说都有可能是必要的。但是，如果您现在的目的是调查，那么请对多种身份、sockpuppets 和恶作剧保持警惕。
  </p>
  <p class="graf graf--p">
   您可能想尝试研究自己的身份，或者与朋友一起看看那里有什么好玩的东西，并减少从各种来源暴露出的关于您的信息量。
  </p>
  <p class="graf graf--p">
   如果您拥有对自己的Web服务器的控制权，则可能更容易收集信息。这是拥有您自己网站的一个很好的理由。例如，如果有人私下向您发送消息，请为他们提供指向您的网站的链接。然后收集点击者的IP地址。
  </p>
  <p class="graf graf--p">
   专注于像 Twitter 这样的高使用率平台，以绘制目标人的社交图、寻找地理位置、和其他信息的线索。
  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   收集到这些信息之后，您可能会获得更多可帮助您做出决策的信息。
  </p>
  <p class="graf graf--p">
   你是否放心这个人不是一个直接的危险？如果是这样，那很好。你的调查取得了成功。
  </p>
  <p class="graf graf--p">
   或者，你需要采取进一步措施吗？您想在将来查看此人的活动或犯罪记录吗？或者，您想与他人分享调查结果吗？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当我们分享这些调查信息时，就是正在建立自己的力量。揭露是最强大的抵抗。当我们能够集体行动时，我们也处于更好的政治地位
   </strong>
   ，在复杂的调查中分享合作是一项极其有利的技能；在自我人肉以规避风险时，您所要做的只是想办法去除一切潜在的数字导火索。
  </p>
  <p class="graf graf--p">
   关于隐身，后面还有专门文章的更多介绍。⚪️
  </p>
