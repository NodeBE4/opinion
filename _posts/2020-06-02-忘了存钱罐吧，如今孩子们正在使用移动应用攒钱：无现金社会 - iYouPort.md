  </p>
  <p class="graf graf--p">
   如今的孩子们可能不会再将硬币放在存钱罐中了。随着向无现金社会的迈进，零用钱正朝着数字化方向发展。
  </p>
  <p class="graf graf--p">
   为了反映这一趋势，
   <strong class="markup--strong markup--p-strong">
    全球范围内
   </strong>
   涌现了一系列针对儿童的存钱移动应用程序：GoHenry，Osper 和 Gimi等。
  </p>
  <p class="graf graf--p">
   这些应用程序为孩子提供了简单的资金管理服务，通常需要每月由父母支付订阅费。
  </p>
  <p class="graf graf--p">
   父母可以向孩子的帐户添加资金、设置限额并监控交易，而孩子可以选择使用预付款卡（如借记卡）来存钱或消费。
  </p>
  <p class="graf graf--p">
   应用程序建议预付卡的最低年龄范围为6到9岁。
  </p>
  <p class="graf graf--p">
   即将加入零用钱市场的是数字银行 Revolut，正在推出面向7至18岁儿童的 Revolut Youth。
  </p>
  <p class="graf graf--p">
   已经是 Revolut 客户的父母将能够把他们的孩子作为二级用户添加到他们自己的帐户中，每个用户都有自己的个人借记卡。
  </p>
  <p class="graf graf--p">
   父母可以通过他们现有的应用程序监视孩子的帐户，而孩子可以下载自己的儿童友好版本。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    争辩
   </span>
  </h3>
  <p class="graf graf--p">
   这些应用程序背后的公司争辩说，在一个越来越无现金的社会中，它们可以成为一种教导幼儿有关金钱问题的有价值的方式。
  </p>
  <p class="graf graf--p">
   根据标准普尔（Standard＆Poor）的《全球金融素养调查》全球三分之二的成年人财务状况不佳，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.oecd.org/pisa/many-teenagers-struggle-to-understand-money-matters.htm" href="https://www.oecd.org/pisa/many-teenagers-struggle-to-understand-money-matters.htm" rel="noopener noreferrer" target="_blank">
    四分之一
   </a>
   的青少年无法就日常支出做出简单的决定。
  </p>
  <p class="graf graf--p">
   这些应用声称 “旨在克服这一问题”，他们称 “可以教孩子一些财务概念，例如预算，利率和收入”。
  </p>
  <p class="graf graf--p">
   例如，瑞典的Gimi应用程序 —— 在全球有120万用户 —— 拥有虚拟的储蓄罐，孩子们可以在其中存钱；父母可以在储蓄时支付子女的利息；并且有一项杂务功能，父母可以为完成家庭任务向孩子付钱。
  </p>
  <p class="graf graf--p">
   该帐户附有预付卡，该卡曾经仅在瑞典可用，但2020年已经在欧洲其他地方推出。
  </p>
  <p class="graf graf--p">
   Gimi 首席执行官 Philip Haglund 说，“现金才是教授金融知识的最好方法，因为它是如此有形且易于掌握。而现在，钱正在通过某种网络空间转移，这真的是抽象的，任何人都很难理解。”
  </p>
  <p class="graf graf--p">
   他说：“您不会仅仅因为拥有经济学学位就能擅长理财。这更多地是关于您六到十二岁时与父母之间关于的金钱的态度和关系。”
  </p>
  <p class="graf graf--p">
   但伦敦银行与金融学院金融能力董事总经理 Catherine Winter 警告说，尽管数字工具可以提供帮助，但仍需要一种更加结构化的金融教育方法。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     有利可图的一代人
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   金融网站 Money.co.uk 最近的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.money.co.uk/press/pocket-money-goes-digital-as-the-move-to-cashless-gathers-pace.htm" href="https://www.money.co.uk/press/pocket-money-goes-digital-as-the-move-to-cashless-gathers-pace.htm" rel="noopener noreferrer" target="_blank">
    一份报告
   </a>
   显示，数字银行的发展已经影响了父母如何掏出零用钱，英国三分之一的父母以数字方式支付零用钱。
  </p>
  <p class="graf graf--p">
   金融服务提供商
   <a class="markup--anchor markup--p-anchor" data-href="https://www.onefamily.com/our-story/media-centre/2019/teenagers-add-1-7-billion-to-uk-economy-each-year/" href="https://www.onefamily.com/our-story/media-centre/2019/teenagers-add-1-7-billion-to-uk-economy-each-year/" rel="noopener noreferrer" target="_blank">
    OneFamily
   </a>
   的《青少年金融报告》显示，总体而言，年龄在
   <strong class="markup--strong markup--p-strong">
    13–19岁之间的儿童每年估计为英国经济贡献17亿英镑
   </strong>
   （22亿美元）。
  </p>
  <p class="graf graf--p">
   Revolut Youth 的产品所有者 AurélienGuichard 说：“机会很大。” “我们为成年人提供了一个应用程序……因此我们将能够实现从 Revolut Youth 用户到普通用户的过渡。”
  </p>
  <p class="graf graf--p">
   根据英国竞争与市场管理局
   <a class="markup--anchor markup--p-anchor" data-href="https://www.gov.uk/government/news/bank-switching-to-be-overhauled" href="https://www.gov.uk/government/news/bank-switching-to-be-overhauled" rel="noopener noreferrer" target="_blank">
    的说法
   </a>
   ，这可以转化为终身客户，在任何一年中，只有3％的个人客户转而使用另一家银行。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     鼓励支出或债务？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   一个问题是，向幼儿介绍数字货币应用程序可能有助于鼓励不负责任的消费习惯。
  </p>
  <p class="graf graf--p">
   Winter 说：“如果孩子们在财务能力上没有很好的基础，那么就有可能将金钱应用视为另一种游戏。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “有一种风险，孩子可能不会了解货币的真正价值，并可能养成不良的货币习惯。”
  </p>
  <p class="graf graf--p">
   然而 Haglund 认为，父母可以监视孩子的消费习惯，因此孩子们可以免受这种情况的侵害。
  </p>
  <p class="graf graf--p">
   ……
  </p>
  <p class="graf graf--p">
   以上是CNN的报道。
   <strong class="markup--strong markup--p-strong">
    很奇怪这篇报道中为什么完全没有提起无现金社会的危险性？如果现在的孩子习惯于无现金生活，这个世界的未来谁最得利？
   </strong>
  </p>
  <p class="graf graf--p">
   肯定不是孩子，也不是父母 —— 而是监视者。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     现金为什么是敌人？因为隐私是敌人，隐私是敌人是因为它使政府控制变得更加困难
    </strong>
    。《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%81%90%e6%80%96%e7%9a%84%e5%8f%8d%e7%8e%b0%e9%87%91%e4%b9%8b%e6%88%98/" href="https://www.iyouport.org/%e6%81%90%e6%80%96%e7%9a%84%e5%8f%8d%e7%8e%b0%e9%87%91%e4%b9%8b%e6%88%98/" rel="noopener noreferrer" target="_blank">
     恐怖的反现金之战
    </a>
    》
   </li>
   <li class="graf graf--li">
    现金使用的越少，它作为后备力量的效果就越差。如今，一旦数字或电力连接下降，所有正确的地方都没有足够的现金来保持经济的长期运作《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%8e%b0%e9%87%91%e9%80%80%e5%9c%ba%e7%9a%84%e6%81%90%e6%85%8c/" href="https://www.iyouport.org/%e7%8e%b0%e9%87%91%e9%80%80%e5%9c%ba%e7%9a%84%e6%81%90%e6%85%8c/" rel="noopener noreferrer" target="_blank">
     现金退场的恐慌
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   我们认为这不是 “角度问题”，而是一种故意的忽略。希望您不要忘记是谁在鼓吹无现金，尤其是，他们的目的。
  </p>

   中国有没有这种情况？我们感觉很快会出现。如果您是父母，请教导您的孩子使用现金消费以保护隐私人权。⚪️
