 <div class="body-text">
  <p class="MsoNormal" style="text-indent:21.1pt;">
   <b>
    <span style="font-family:宋体;">
     原题：谁是世界经济思想家
    </span>
    <span>
    </span>
   </b>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span style="font-family:宋体;">
    由英国
   </span>
   <span>
    Prospect
   </span>
   <span style="font-family:宋体;">
    杂志发起的“
   </span>
   <span>
    2014
   </span>
   <span style="font-family:宋体;">
    世界思想家”评选活动，中国经济学家、天则经济研究所荣誉理事长、人文经济学会理事长茅于轼，成为十位获奖者之一。
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span style="font-family:宋体;">
    本次评选中排名前十名获奖名单的思想家依次为：
   </span>
   <span>
    1998
   </span>
   <span style="font-family:宋体;">
    年诺贝尔经济学奖得主阿玛蒂亚•森、印度央行行长拉古拉迈•拉詹、印度女作家阿兰达蒂•洛伊、中国经济学家茅于轼、天主教教皇方济各、世界银行首席经济学家考希克•巴苏、剑桥大学古典学教授玛丽•比尔德、
   </span>
   <span>
    2013
   </span>
   <span style="font-family:宋体;">
    年诺贝尔物理学奖得主彼得•希格斯、韩国经济学家张夏准、
   </span>
   <span>
    2002
   </span>
   <span style="font-family:宋体;">
    年诺贝尔经济学奖得主丹尼尔•卡内曼。
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span style="font-family:宋体;">
    由于印度媒体的强烈兴趣及其报道的影响，前三名都是印度的学者。剔除这个因素，茅于轼其实是这个榜单上的第一名，而包括天主教教皇以及诺奖获得者排名在茅于轼先生之后，也从另一个侧面证明了世界对中国问题的关注和了解。
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
    Prospect
   </span>
   <span style="font-family:宋体;">
    杂志介绍称，
   </span>
   <span>
    85
   </span>
   <span style="font-family:宋体;">
    岁的茅于轼是中国领先的个人权利、自由市场与政府改革倡导者，也是针对中国新毛左派的一个直言不讳的批评者。他是
   </span>
   <span>
    18
   </span>
   <span style="font-family:宋体;">
    本非虚构类著作的作者，包括《生活中的经济学》在内，这些著作旨在向中国人介绍和解释市场经济的功能与运作。鉴于其工作在全球范围产生的影响，他在
   </span>
   <span>
    2012
   </span>
   <span style="font-family:宋体;">
    年被美国加图研究所授予米尔顿•弗里德曼促进自由奖。
   </span>
   <span>
    Prospect
   </span>
   <span style="font-family:宋体;">
    杂志介绍，茅于轼先生是创办于
   </span>
   <span>
    2012
   </span>
   <span style="font-family:宋体;">
    年的民间学术机构“人文经济学会”的理事长。
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span style="font-family:宋体;">
    关于茅于轼先生入选
   </span>
   <span>
    2014
   </span>
   <span style="font-family:宋体;">
    世界思想家，天则经济研究所学术委员会主席张曙光教授、理事长姚中秋教授、所长盛洪教授、副所长冯兴元教授以及理事赵农教授围绕以下三个问题进行了回应。
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span style="font-family:宋体;">
    第一，
   </span>
   <span style="font-family:宋体;">
    英国杂志通过网络在全球范围内进行思想家的评选活动，其意义是什么？
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21.1pt;">
   <b>
    <span style="font-family:宋体;">
     张曙光：
    </span>
   </b>
   <span style="font-family:宋体;">
    可以看到，虽然参加投票的力量主要来自民间，但是入选的思想家当中有阿玛蒂亚森以及卡内曼这样的理论经济学者。普通人虽没有什么名气，但是他们关心整个思想界、整个社会的问题，这从一个侧面表明民间是推动社会进步的重要力量。
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21.1pt;">
   <b>
    <span style="font-family:宋体;">
     盛洪：
    </span>
   </b>
   <span style="font-family:宋体;">
    英国杂志的此次评选不是精英和专家的投票，而它正好反映了大众对思想家、思想界的关心，这也反过来反映了思想的力量如何影响了大众。
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21.1pt;">
   <b>
    <span style="font-family:宋体;">
     冯兴元：
    </span>
   </b>
   <span style="font-family:宋体;">
    全球追求自由的精英、大众都能参加投票，这是一个开放社会的特点。虚拟的网络让世界变得扁平，用低的成本就能保证公众的自由参与权。这就是这个评选的意义。
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21.1pt;">
   <b>
    <span style="font-family:宋体;">
     赵农：
    </span>
   </b>
   <span>
    Prospect
   </span>
   <span style="font-family:宋体;">
    杂志的评选运用的是海选的方式，先评出
   </span>
   <span>
    50
   </span>
   <span style="font-family:宋体;">
    名候选人，再通过海选评出
   </span>
   <span>
    10
   </span>
   <span style="font-family:宋体;">
    名入选人。茅于轼先生获得的弗里德曼自由奖则代表了另外一种方式，是由精英或者固定的评委来评选。比之后一种方式，海选的方式有更大的参与面，代表了思想界能够影响的深度和广度，因此有特别的意义。
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span style="font-family:宋体;">
    第二，
   </span>
   <span style="font-family:宋体;">
    茅于轼
   </span>
   <span style="font-family:宋体;">
    先生入选此评选，对中国的影响和意义是什么？
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21.1pt;">
   <b>
    <span style="font-family:宋体;">
     张曙光：
    </span>
   </b>
   <span style="font-family:宋体;">
    我想应该对中国的自由主义传播有很大的作用。尽管茅于轼先生在国内常常遇到各种批评甚至谩骂，但是此次入选表明全世界公认茅于轼先生的理念。反对茅于轼先生的人也需要通过这个事情反思自我。
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21.1pt;">
   <b>
    <span style="font-family:宋体;">
     姚中秋：
    </span>
   </b>
   <span style="font-family:宋体;">
    茅老入选“世界思想家”，标志着中国越来越重要了，中国思想正在引起世界关注。过去三十年来，中国经济有快速增长，社会有快速演进，如何理解这样的增长和演进？世界相当关注。中国未来的前景如何？世界更为关注。世界关注中国人对中国故事的思考。我想，未来会有越来越多的中国思想人物引起世界关注。
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21.1pt;">
   <b>
    <span style="font-family:宋体;">
     盛洪：
    </span>
   </b>
   <span style="font-family:宋体;">
    思想家不仅仅指那些藏在书斋里的思想家，思想家的理念会影响长远的历史，但是需要在当代就产生影响。思想家不仅能够独立的自我思考，而且可以有效表达，在他生活的那段历史里就产生了作用。这就是说，思想家的思想和理念需要能够得以传播，茅于轼先生入选此次评选就表现出了思想传播的作用。这一点对中国的学者会有一定的启发。
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21.1pt;">
   <b>
    <span style="font-family:宋体;">
     冯兴元：
    </span>
   </b>
   <span style="font-family:宋体;">
    当然会有一定的影响。能称得上思想家的人凤毛麟角，茅于轼先生更是中国传统中的儒雅君子，他靠自己的思考得出了思想上的火花与结晶。茅于轼先生的思考看上去只是星星之火，但是又不仅仅是星星之火。思想的作用就是指路明灯，无论左派右派，都会通过这个评选激发出自己的思考。
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21.1pt;">
   <b>
    <span style="font-family:宋体;">
     赵农：
    </span>
   </b>
   <span style="font-family:宋体;">
    作为中国最富盛名的学者，茅老此次入选也标志着他的思想得到世界范围内的认可。一方面是由于中国问题在全球范围都是焦点，更是由于茅老思想创造和传播的贡献得到世界的认同，这必然对中国的学界和社会大众产生一定影响。
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span style="font-family:宋体;">
    第三，
   </span>
   <span style="font-family:宋体;">
    茅于轼
   </span>
   <span style="font-family:宋体;">
    先生本人给中国思想界带来哪些影响，有哪些贡献？
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21.1pt;">
   <b>
    <span style="font-family:宋体;">
     张曙光：
    </span>
   </b>
   <span style="font-family:宋体;">
    茅于轼
   </span>
   <span style="font-family:宋体;">
    先生是中国自由、宪政思想的一面旗帜，无论在学界还是在民间都是如此，而这些思想又恰恰是当代中国最需要的。尽管有人批评或者不认同，茅于轼先生在思想界的地位都是崇高的。
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21.1pt;">
   <b>
    <span style="font-family:宋体;">
     姚中秋：
    </span>
   </b>
   <span style="font-family:宋体;">
    茅老中国思想界贡献的最大财富，是陈寅恪先生说的“独立之思想，自由之精神”。茅老一直独立而严肃地思考，不媚权，不媚钱，不媚俗。惟其如此，茅老才赢得了世人的尊敬。现在很多学者的学问也许很好，知识也许很丰富，但做不到这一点，终究难成大器。
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21.1pt;">
   <b>
    <span style="font-family:宋体;">
     盛洪：
    </span>
   </b>
   <span style="font-family:宋体;">
    从学者的角度来评价，茅于轼先生的主要贡献是推导出择优分配原理。第一是由于择优分配理论是他独立思考得出的，在理论成就上达到了与西方新古典理论相似的高度。第二是由于择优分配理论是在封闭的、计划经济的条件下得出的，并且与西方主流经济学的结论相近，这也证明了市场的逻辑是普适的。除此之外，更重要的是，茅于轼先生做了大量通俗的普及工作。茅老和天则经济研究所坚持普及的经济自由主义原则，正是中国不重蹈计划经济覆辙的根本保证。看似这些原则没有那么重要，实际上却会对中国社会产生深远的影响。
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21.1pt;">
   <b>
    <span style="font-family:宋体;">
     冯兴元：
    </span>
   </b>
   <span style="font-family:宋体;">
    茅于轼
   </span>
   <span style="font-family:宋体;">
    先生所言所为最能体现“独立之思想、自由之精神”这两句话。他的择优分配原理不仅是数学分析，而且是哲学分析，自由的交换可以使得每个人的尊严得到保证。茅老能够做到对事情进行深入浅出的阐释，是由于他对生活的思考以及对伦理的提炼有系统的考量，这恰恰是思想家需要具备的能力。茅老入选思想家评选，是实至名归。
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21pt;">
   <span>
   </span>
  </p>
  <p class="MsoNormal" style="text-indent:21.1pt;">
   <b>
    <span style="font-family:宋体;">
     赵农：
    </span>
   </b>
   <span style="font-family:宋体;">
    茅于轼
   </span>
   <span style="font-family:宋体;">
    先生弘扬普世价值，传播市场经济理念，宣传宪政民主，是当下中国思想界的一面旗帜。茅老对自由的哲学理解，以及平等对自由的保证，或许甚至连他本人也还未意识到这些思想的重要性。从学术上来讲，茅老从另外一个途径推导出了最优配置原理，也非常了不起。茅老注重从基本逻辑出发，演绎评价一些政策及其后果，虽然在当时有诸多批评，但是事后却往往证明了茅老才是对的。茅老不仅崇尚科学和文明，他那种超乎农耕文明，超乎国人大众乃至民族历史的价值观，已经是近乎于宗教般的高度。本人也通过茅老思想传授班等等的活动，为传播茅老思想做了一些有益的工作，并且为此感到了深深的欣慰。
   </span>
  </p>
