  <p class="graf graf--p graf--startsWithDoubleQuote">
   “机器人”（冒充真实人类的虚假社交媒体帐户）在 Twitter 等平台上占有重要地位。它们的数量有百万计；最大的僵尸机器人网络
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bbc.com/news/technology-38724082" href="https://www.bbc.com/news/technology-38724082" rel="noopener noreferrer" target="_blank">
    有几十万
   </a>
   虚假账户彼此关联。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这些“机器人”会严重扭曲民主辩论，尤其是当它们集体出动的时候
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    它们可用于操纵热门话题或主题标签趋势；它们还可用于放大或攻击特定的消息或文章；并同时用来骚扰其他用户
   </strong>
   。
  </p>
  <p class="graf graf--p">
   同时，相对而言，许多机器人网络很容易被发现，而无需使用专门的软件或商业分析工具。
  </p>
  <p class="graf graf--p">
   本文列出了许多线索，这些线索对于揭露虚假帐户来说都很有用。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第一原则
   </strong>
  </p>
  <p class="graf graf--p">
   Twitter 机器人只是一个由软件运行的帐户，类似于自动驾驶飞机。由于可以打开和关闭自动驾驶功能，因此，
   <strong class="markup--strong markup--p-strong">
    这些虚假帐户在不同时间的行为是不同的，有时可以像自动程序一样，有时也可以像人类用户一样
   </strong>
   。
  </p>
  <p class="graf graf--p">
   因此，以下提示应被视为 *判断给定时间段内机器人行为* 的指示，而不是该帐户是否“是”机器人的非黑即白定义。
  </p>
  <p class="graf graf--p">
   并非所有的机器人都是恶意的或政治攻击性的。自动帐户也可以发布例如诗歌、摄影或新闻（新闻媒体经常使用机器人账户推送内容），而不会产生任何扭曲的效果。
  </p>
  <p class="graf graf--p">
   因此，
   <strong class="markup--strong markup--p-strong">
    我们的重点目标应该是伪装成人类并故意放大政治类信息的机器人
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️在所有情况下，重要的是要注意，不能依靠任何单一因素来识别类似机器人的行为。重要的是多种因素的组合。根据我们的经验，最重要的三个因素可以称为“三个A”：即 activity, anonymity, &amp; amplification。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、活动
   </strong>
  </p>
  <p class="graf graf--p">
   帐户是否自动执行的最明显指标是其活动。
  </p>
  <p class="graf graf--p">
   可以通过查看其个人资料页面并用帖子总数除以其活跃天数来计算得出。
  </p>
  <p class="graf graf--p">
   要找到确切的创建日期，请将鼠标悬停在“ Joined…”条目上。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9594 jetpack-lazy-image" data-lazy-sizes="(max-width: 921px) 100vw, 921px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-5.png?resize=921%2C525&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-5.png?w=921&amp;ssl=1 921w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-5.png?resize=300%2C171&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-5.png?resize=768%2C438&amp;ssl=1 768w" data-recalc-dims="1" height="525" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-5.png?resize=921%2C525&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="921"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9594" data-recalc-dims="1" height="525" sizes="(max-width: 921px) 100vw, 921px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-5.png?resize=921%2C525&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-5.png?w=921&amp;ssl=1 921w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-5.png?resize=300%2C171&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-5.png?resize=768%2C438&amp;ssl=1 768w" width="921"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   可疑活动的基准各不相同。牛津互联网学院的
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/oiioxford" href="https://twitter.com/oiioxford" rel="noopener noreferrer" target="_blank">
    研究人员认为
   </a>
   ，平均每天发布 50多个帖子是可疑的；这是一个广泛认可的和应用的基准，但可能偏低。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://twitter.com/DFRLab" href="https://twitter.com/DFRLab" rel="noopener noreferrer" target="_blank" title="Twitter profile for @DFRLab">
    DFRLab
   </a>
   将每天发送72条推文（连续10个小时，每十分钟发送一次）视为可疑，将每天发布144条以上推文视为高度可疑。
  </p>
  <p class="graf graf--p">
   例如，账户 @ sunneversets100 于2016年11月14日创建，这是克里姆林宫信息战的放大器。
  </p>
  <p class="graf graf--p">
   到2017年8月28日，该账户共创立了288天。在那个时期，它发布了203,197条推文（同样，
   <strong class="markup--strong markup--p-strong">
    确切的数字可以通过将鼠标悬停在“ Tweets”条目上找到
   </strong>
   ）。
  </p>
  <p class="graf graf--p">
   这意味着每天平均有705个帖子，或者说每天平均连续9个小时在线每分钟1个帖子，持续9个月。这不是人类的行为模式。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    仅仅
   </strong>
   发帖数量大
   <strong class="markup--strong markup--p-strong">
    并不是
   </strong>
   判断标准，在热门事件（尤其是选举期间）发帖量大的真实人类账户有很多。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、匿名性
   </strong>
  </p>
  <p class="graf graf--p">
   第二个指标是帐户显示的匿名程度。通常，提供的个人信息越少、成为机器人的可能性就越大。
  </p>
  <p class="graf graf--p">
   例如，@ Sunneversets100 将佛罗伦萨大教堂的照片作为头像，曲线图作为背景，以及匿名的ID和账户名。
  </p>
  <p class="graf graf--p">
   唯一的特点是简介里贴的位于美国的政治行动委员会的链接；这远远不足以提供身份证明。
  </p>
  <p class="graf graf--p">
   另一个示例是
   <a class="markup--anchor markup--p-anchor" data-href="http://twitter.com/BlackManTrump" href="https://twitter.com/BlackManTrump" rel="noopener noreferrer" target="_blank" title="Twitter profile for @BlackManTrump">
    @BlackManTrump
   </a>
   ，这是另一个活跃帐户，该帐户在2016年8月28日至2016年12月19日之间发布了89,944条推文（请参阅
   <a class="markup--anchor markup--p-anchor" data-href="https://t.co/T6crZSxTBz" href="https://t.co/T6crZSxTBz" rel="noopener noreferrer" target="_blank">
    此处的存档
   </a>
   ），平均每天发789条帖子。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9595 jetpack-lazy-image" data-lazy-sizes="(max-width: 1061px) 100vw, 1061px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-5.png?resize=1061%2C708&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-5.png?w=1061&amp;ssl=1 1061w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-5.png?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-5.png?resize=768%2C512&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-5.png?resize=1024%2C683&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-5.png?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="708" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-5.png?resize=1061%2C708&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1061"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   此帐户完全不提供任何个人信息。简介只是一般性的政治声明，位置标注“美国”。因此没有迹象表明背后隐藏着什么人。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    请注意：我们始终会强烈建议您不要在社交媒体上采用任何真实身份信息，不要在简介、头像、帖子中留下任何真实身份信息
   </strong>
   。这是为了安全。这样做并不会令您被误会为机器人，请一直记得，任何单一指标都不构成判断条件。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、放大效果
   </strong>
  </p>
  <p class="graf graf--p">
   这也许是最重要的指标。因为机器人的主要作用之一是通过转发、点赞或引用其他用户的帖子来增强它们想要传达的信息。
  </p>
  <p class="graf graf--p">
   因此，典型机器人的时间轴将包括一系列的转发和新闻标题的完整引用，很少或没有原创帖子。
  </p>
  <p class="graf graf--p">
   判断这点的最有效方法是对大量帖子进行机器扫描。但通过点击帐户的“推文和回复”栏并向下滚动最新的200条帖子，就可以进行更简单的识别。
  </p>
  <p class="graf graf--p">
   数字200在很大程度上是任意的，旨在提供合理且易于管理的大样本；拥有更多时间的研究人员完全可以查看更多内容。
  </p>
  <p class="graf graf--p">
   例如，截至8月28日，机器人 @Sunneversets100 的最后200条推文中有195条是转推，其中许多都是来自克里姆林宫官方媒体的网点：
  </p>

  <p class="graf graf--p">
   @BlackManTrump 直到11月14日的大部分帖子都显示出更高的复杂程度，不再有“RT @ ”这类非人的东西出现 。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9597 jetpack-lazy-image" data-lazy-sizes="(max-width: 600px) 100vw, 600px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-2.png?resize=600%2C660&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-2.png?w=600&amp;ssl=1 600w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-2.png?resize=273%2C300&amp;ssl=1 273w" data-recalc-dims="1" height="660" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-2.png?resize=600%2C660&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="600"/>

  <p class="graf graf--p">
   因此，@ BlackManTrump和 @Sunneversets 都显示出清晰的类似机器人的行为，结合了很高的活动性、匿名性和针对性放大的迹象。

  </p>
  <p class="graf graf--p">
   一个很好的例子是机器人的奇怪网络，这些机器人从通常用于赌博的帐户 @ every1bets 转推了一个政治性的帖子。
  </p>
  <p class="graf graf--p">
   如下面的列表所示，转推者具有各种身份：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9628 jetpack-lazy-image" data-lazy-sizes="(max-width: 512px) 100vw, 512px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/32.png?resize=512%2C468&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/32.png?w=512&amp;ssl=1 512w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/32.png?resize=300%2C274&amp;ssl=1 300w" data-recalc-dims="1" height="468" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/32.png?resize=512%2C468&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="512"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9628" data-recalc-dims="1" height="468" sizes="(max-width: 512px) 100vw, 512px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/32.png?resize=512%2C468&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/32.png?w=512&amp;ssl=1 512w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/32.png?resize=300%2C274&amp;ssl=1 300w" width="512"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   但是他们都倾向于发布高比例的广告内容。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9629 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/33.png?resize=1100%2C353&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/33.png?w=1378&amp;ssl=1 1378w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/33.png?resize=300%2C96&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/33.png?resize=768%2C246&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/33.png?resize=1024%2C328&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/33.png?resize=1100%2C353&amp;ssl=1 1100w" data-recalc-dims="1" height="353" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/33.png?resize=1100%2C353&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

    <img alt="" class="aligncenter size-full wp-image-9629" data-recalc-dims="1" height="353" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/33.png?resize=1100%2C353&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/33.png?w=1378&amp;ssl=1 1378w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/33.png?resize=300%2C96&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/33.png?resize=768%2C246&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/33.png?resize=1024%2C328&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/33.png?resize=1100%2C353&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   以如此方式转发的帐户，尤其是，如果它们以多种语言显示时，很可能是商业僵尸网络的成员，被雇用给想要放大或发布其帖子的客户 。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    11、自动化软件
   </strong>
  </p>
  <p class="graf graf--p">
   潜在自动化的另一个线索是使用URL缩短器。
  </p>
  <p class="graf graf--p">
   这些主要用于跟踪特定链接上的流量，但是，使用它们的频率可以指示自动化的程度。
  </p>
  <p class="graf graf--p">
   例如，一个最近公开的假账户“ Angee Dixson”使用了德国超模 Lorena Rae 的照片，分享了许多右翼政治帖子。每个帖子都标有缩短网址 ift.tt：
  </p>
  <p class="graf graf--p">
   请注意：开源情报社区中很多人也使用缩短链接工具，主要是为了节省字数空间 —— 再一次，仅仅使用缩短链接也不是决定性因素。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9630 jetpack-lazy-image" data-lazy-sizes="(max-width: 499px) 100vw, 499px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/34.png?resize=499%2C649&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/34.png?w=499&amp;ssl=1 499w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/34.png?resize=231%2C300&amp;ssl=1 231w" data-recalc-dims="1" height="649" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/34.png?resize=499%2C649&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="499"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9630" data-recalc-dims="1" height="649" sizes="(max-width: 499px) 100vw, 499px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/34.png?resize=499%2C649&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/34.png?w=499&amp;ssl=1 499w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/34.png?resize=231%2C300&amp;ssl=1 231w" width="499"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   这是由一家名为 ifttt.com 的公司生产的一种软件，它使用户可以根据多种条件来自动执行其帖子 — 例如，转发带有给定标签的任何帖子。
  </p>
  <p class="graf graf--p">
   因此，充满了 ift.tt 缩短链接的时间线很可能是机器人执行的。
  </p>
  <p class="graf graf--p">
   如果其他短链接在整个时间线中重复出现，这种情况也可以显示为机器人。
  </p>
  <p class="graf graf--p">
   有些机器人会在网站上发布一长串 ow.ly，这也表明可能是自动化的。Twitter 自己的 TweetDeck 工具允许用户嵌入各种短链，例如 bit.ly 或 tinyurl.com。
  </p>
  <p class="graf graf--p">
   顺便说 TweetDeck 是优秀的开源情报调查工具，您可以在这里看到它在调查方面的技巧介绍《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%8e%e6%8e%a8%e7%89%b9%e4%b8%ad%e6%8c%96%e6%8e%98%e7%9c%9f%e7%9b%b8%e4%b8%8d%e9%9c%80%e8%a6%81%e5%a4%aa%e5%a4%8d%e6%9d%82%e7%9a%84%e5%b7%a5%e5%85%b7%ef%bc%9a%e4%b8%80%e4%b8%aa%e5%b8%b8%e7%94%a8/" href="https://www.iyouport.org/%e4%bb%8e%e6%8e%a8%e7%89%b9%e4%b8%ad%e6%8c%96%e6%8e%98%e7%9c%9f%e7%9b%b8%e4%b8%8d%e9%9c%80%e8%a6%81%e5%a4%aa%e5%a4%8d%e6%9d%82%e7%9a%84%e5%b7%a5%e5%85%b7%ef%bc%9a%e4%b8%80%e4%b8%aa%e5%b8%b8%e7%94%a8/" rel="noopener noreferrer" target="_blank">
    从推特中挖掘真相不需要太复杂的工具：一个常用工具的全面指南
   </a>
   》。
  </p>
  <p class="graf graf--p">
   再次强调，使用缩短链接是很多人的在线生活习惯之一，于是不可以仅仅根据短链接的使用判断是否为机器人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    12、转推和点赞
   </strong>
  </p>
  <p class="graf graf--p">
   可以通过比较特定帖子的转发和点赞来收集僵尸网络正在运行的最终指示。
  </p>
  <p class="graf graf--p">
   有些机器人被编程为同时转发和点赞相同的推文。在这种情况下，转发和点赞的次数将几乎相同，并且执行转发和点赞的帐户系列也可能完全匹配，就如下面这样：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9631 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/35.png?resize=1100%2C480&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/35.png?w=1374&amp;ssl=1 1374w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/35.png?resize=300%2C131&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/35.png?resize=768%2C335&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/35.png?resize=1024%2C447&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/35.png?resize=1100%2C480&amp;ssl=1 1100w" data-recalc-dims="1" height="480" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/35.png?resize=1100%2C480&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

    <img alt="" class="aligncenter size-full wp-image-9631" data-recalc-dims="1" height="480" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/35.png?resize=1100%2C480&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/35.png?w=1374&amp;ssl=1 1374w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/35.png?resize=300%2C131&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/35.png?resize=768%2C335&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/35.png?resize=1024%2C447&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/35.png?resize=1100%2C480&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在此示例中，转发和点赞的次数几乎相同 — 差异小于0.1％。
  </p>
  <p class="graf graf--p">
   完全相同的帐户列表以相同的顺序和时间转发并点赞该推文。
  </p>
  <p class="graf graf--p">
   在13,000个用户样本中，这不太可能是巧合。它表明存在一个协调行动的网络，所有这些网络都经过编程以点赞和转发相同的内容。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    结论
   </strong>
  </p>
  <p class="graf graf--p">
   机器人是Twitter生活中不可分割的一部分。许多机器人是完全合法的。那些不合法的往往具有共同的关键特征。
  </p>
  <p class="graf graf--p">
   如上所述，最常见的指标是活动模式、匿名性和针对性放大效果，即机器人识别指标的“三个A”。但也存在其他标准。
  </p>
  <p class="graf graf--p">
   比如使用被盗的图像、字母数字组合的不规则ID和不匹配的用户名可能会显示账户的伪造；大量的商业帖子或异常多语种的帖子也是如此。
  </p>
  <p class="graf graf--p">
   然而，最重要的是意识。
  </p>
  <p class="graf graf--p">
   能够识别机器人的用户不太可能被机器人所操纵；他们甚至可以举报僵尸网络并关闭它们。本文的目的就是帮助更多人识别鉴定机器人。希望对您有所帮助。⚪️
  </p>
