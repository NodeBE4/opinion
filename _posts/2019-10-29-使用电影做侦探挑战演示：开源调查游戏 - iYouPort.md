  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    结论
   </strong>
  </p>
  <p class="graf graf--p">
   本演示是希望继续提供一些思考方式，帮助您简化漫长的信息验证挑战。
  </p>
  <p class="graf graf--p">
   也许很多时候你知道有一大堆有用的工具可以帮你实现一个目的，但是最终发现它们可能都不够好。
  </p>
  <p class="graf graf--p">
   因为需要
   <strong class="markup--strong markup--p-strong">
    巧妙的结合
   </strong>
   才能让工具发挥各自最佳的作用。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    结合的路线图就是调查人员的思考方式：它来自长期的演练实践，您可以亲自练习，也可以通过增加阅读他人的演示来获得。
   </strong>
  </p>
  <p class="graf graf--p">
   💡对你最有帮助的不是培训，而是耐心和坚持，培养分析性思维，当然还需要有很好的知识。
  </p>
  <p class="graf graf--p">
   让自己日常习惯于考察，可以快速培养分析性思维；您不一定一开始就能做的很好，也许最初会留下一大堆未解的难题，但是您会发现自己的思考方式越来越趋向于对真相的渴求。
  </p>
  <p class="graf graf--p">
   这本身就是一大胜利。您的技术能力会从此变得更好。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    请不要将 OSINT 视为任何新鲜事物，事实上每个人的日常中都会下意识地进行很多 OSINT 调查
   </strong>
   ，只是人们没有命名它罢了。最常见的就如社交网络上人们习惯性地询问“这是真的吗？/要不要搜一下？”
  </p>
  <p class="graf graf--p">
   当然，可以作为证据的 OSINT 专业调查报告对于初学者来说可能有点难度（我们进行了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    大量的演示
   </a>
   ，您可以参考）但它们的基本思考框架都是一样的。
  </p>
  <p class="graf graf--p">
   掌握这一框架，并增加更多创意，您将很容易获得满意的结论。
  </p>
  <p class="graf graf--p">
   最后，
   <strong class="markup--strong markup--p-strong">
    重要的是《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a5%bd%e5%a5%87%e5%bf%83%e4%bd%9c%e4%b8%ba%e9%a9%b1%e5%8a%a8%e5%8a%9b/" href="https://www.iyouport.org/%e5%a5%bd%e5%a5%87%e5%bf%83%e4%bd%9c%e4%b8%ba%e9%a9%b1%e5%8a%a8%e5%8a%9b/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     好奇心作为驱动力
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》；虚无主义是一切的杀手，不止真相事业。
