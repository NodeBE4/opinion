  <p class="graf graf--p">
   您还可以专门搜图，也可以按社区/评论来过滤。
  </p>
  <p class="graf graf--p">
   地址：
   <a class="markup--anchor markup--p-anchor" data-href="https://cse.google.com/cse?cx=016621447308871563343:0p9cd3f8p-k" href="https://cse.google.com/cse?cx=016621447308871563343:0p9cd3f8p-k" rel="nofollow noopener noreferrer" target="_blank">
    https://cse.google.com/cse?cx=016621447308871563343:0p9cd3f8p-k
   </a>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、跨平台追踪目标人
   </strong>
  </h3>
  <p class="graf graf--p">
   NameCheckup 是一个跨平台的用户名搜索工具，测试效果不错：namecheckup.com
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    6、在线制作假身份
   </strong>
  </h3>
  <p class="graf graf--p">
   我们曾经介绍过一种，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bc%aa%e8%a3%85%e6%93%a6%e9%99%a4%e2%80%8a-%e2%80%8a-%e5%9c%a8%e7%ba%bf%e9%9a%90%e8%ba%ab%e7%9a%84%e9%87%8d%e8%a6%81%e6%ad%a5%e9%aa%a4%ef%bc%88%e5%b0%91%e6%95%b0%e6%b4%be%e7%8e%a9/" href="https://www.iyouport.org/%e4%bc%aa%e8%a3%85%e6%93%a6%e9%99%a4%e2%80%8a-%e2%80%8a-%e5%9c%a8%e7%ba%bf%e9%9a%90%e8%ba%ab%e7%9a%84%e9%87%8d%e8%a6%81%e6%ad%a5%e9%aa%a4%ef%bc%88%e5%b0%91%e6%95%b0%e6%b4%be%e7%8e%a9/" rel="noopener noreferrer" target="_blank">
    伪装+擦除 — — 在线隐身的重要步骤
   </a>
   》。
  </p>
  <p class="graf graf--p">
   这里还有两种。
  </p>
  <p class="graf graf--p">
   第一个是：
   <a class="markup--anchor markup--p-anchor" data-href="https://names.igopaygo.com/" href="https://names.igopaygo.com/" rel="nofollow noopener noreferrer" target="_blank">
    names.igopaygo.com
   </a>
   可以生成假身份、完整联系方式、信用卡、北美地址、邮件地址、电话号码转换等等。
  </p>
  <p class="graf graf--p">
   第二个是来自中国的假身份生成器（注意：是不安全链接）：
   <a class="markup--anchor markup--p-anchor" data-href="http://www.myfakeinfo.com/index.php" href="http://www.myfakeinfo.com/index.php" rel="nofollow noopener noreferrer" target="_blank">
    http://www.myfakeinfo.com/index.php
   </a>
   生成假身份、姓名、地址、电话号码、信用卡号、社会保险号、身份证号等，包括中国的身份证号码。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    7、社交工程工具包 (SET)
   </strong>
  </h3>
  <p class="graf graf--p">
   Social-Engineer Toolkit 是一个为社交工程设计的开源渗透测试框架。SET具有许多自定义攻击向量，可让您快速进行可信的攻击。
  </p>
  <p class="graf graf--p">
   支持：
  </p>

   <li class="graf graf--li">
    Linux
   </li>
   <li class="graf graf--li">
    Mac OS X
   </li>
  </ul>
  <p class="graf graf--p">
   警告：本工具仅用于测试目的，并且只能在经过严格同意的情况下使用。请勿将其用于非法目的。
