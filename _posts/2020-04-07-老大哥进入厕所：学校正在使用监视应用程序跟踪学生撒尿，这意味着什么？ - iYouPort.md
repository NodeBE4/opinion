  <p class="graf graf--p">
   毫无疑问，我们正生活在越来越深入的奥威尔时代。
  </p>
  <p class="graf graf--p">
   当人们发现另一个政府机构以另一种方式监视公民时，大多数人都不会感到惊讶。在你到过的所有地方（甚至在自己的家中）监视都已成为常态。
  </p>
  <p class="graf graf--p">
   即便从事间谍活动的不是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e6%94%bf%e5%ba%9c%e7%9a%84%e9%97%b4%e8%b0%8d%e8%a1%8c%e4%b8%ba/" href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e6%94%bf%e5%ba%9c%e7%9a%84%e9%97%b4%e8%b0%8d%e8%a1%8c%e4%b8%ba/" rel="noopener noreferrer" target="_blank">
    政府机构
   </a>
   ，也肯定就是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89/" href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89/" rel="noopener noreferrer" target="_blank">
    寡头技术公司
   </a>
   。几乎每天都能听到来自 Google、Facebook 和 Amazon 的新型侵犯隐私的行为。
  </p>
  <p class="graf graf--p">
   最近的宣布是，寡头技术公司和政府正在组建
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theorganicprepper.com/harpa-social-credit-big-tech-govt-gun-control/" href="https://www.theorganicprepper.com/harpa-social-credit-big-tech-govt-gun-control/" rel="noopener noreferrer" target="_blank">
    一个名为HARPA的邪恶反乌托邦联盟
   </a>
   ，该联盟将利用包括 Google、Amazon 和 Apple 在内的公司收集有关表现出可能导致“暴力行为” 情绪的任何用户的数据。
  </p>
  <p class="graf graf--p">
   如果您认为避免使用社交媒体平台就可以让您免受寡头技术公司的窥探？那就真的错了。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     不幸的是，儿童也不能幸免
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   学校曾经是孩子们学习的地方，如今已逐渐成为大政府的灌输中心。使这种痛苦变得明显的一种方法是公立学校实施的监视措施。
  </p>
  <p class="graf graf--p">
   我读到了一个令人脊背发冷对的标题：
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://futurism.com/the-byte/schools-app-track-students-e-hallpass" href="https://futurism.com/the-byte/schools-app-track-students-e-hallpass" rel="noopener noreferrer" target="_blank">
    SCHOOLS ARE USING AN APP TO TRACK STUDENTS WHILE THEY PEE
   </a>
  </p>
  <p class="graf graf--p">
   还记得放风通行证吗？就是教师给学生的纸条，以表明他们有权离开课堂几分钟。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在美国各地的数百所学校中，曾经那种纸质的放风通行证已被特别具有侵入性的应用程序所取代。
   </strong>
  </p>
  <p class="graf graf--p">
   该应用程序称为电子通行证，
   <a class="markup--anchor markup--p-anchor" data-href="https://futurism.com/the-byte/schools-app-track-students-e-hallpass" href="https://futurism.com/the-byte/schools-app-track-students-e-hallpass" rel="noopener noreferrer" target="_blank">
    其工作方式如下
   </a>
   ：
  </p>
  <p class="graf graf--p">
   在使用 e-Hallpass 的学校中，学生通过该应用程序提交离开教室的请求。该监视系统会记录任何“危险信号”，例如同一位学生的频繁上厕所请求。然后，教师选择是否批准请求。
  </p>
  <p class="graf graf--p">
   如果学生被授予许可就可以暂时离开；如果学生上厕所花费的时间太长，该应用会自动通知管理员以对其进行检查。
  </p>
  <p class="graf graf--p">
   在收集本文的信息时，我访问了 e-hallpass 网站以了解有关该系统的更多信息。由于种种原因，本文不分享指向该站点的链接。
  </p>
  <p class="graf graf--p">
   如果确实要造访该网站，请务必查看 e-hallpass 系统的视频。如果您有自由意识并关心隐私，您会知道这有多么令人不安。
  </p>
  <p class="graf graf--p">
   有了这种电子通行证，不仅您的孩子的老师会一直监视学生的一举一动，而且整个学校的工作人员也是如此。
  </p>
  <p class="graf graf--p">
   如果学生是由于健康问题而需要经常去洗手间怎么办？这类健康问题可能会使孩子感到尴尬，继而还要被跟踪和询问他们为什么在洗手间呆了10分钟。
  </p>
  <p class="graf graf--p">
   学生不必在个人设备上安装 e-hallpass 应用程序，但是如果愿意，他们可以使用自己的设备。
  </p>
  <p class="graf graf--p">
   由于明显的原因，
   <strong class="markup--strong markup--p-strong">
    这引起了更多的隐私问题 —— 应用程序经常因访问不应访问的数据而臭名昭著
   </strong>
   。想象一下一个孩子在自己的设备上使用该应用程序，并且该设备与家庭计算机同步，这种监视的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theorganicprepper.com/red-flag-gun-laws-meme/" href="https://www.theorganicprepper.com/red-flag-gun-laws-meme/" rel="noopener noreferrer" target="_blank">
    延伸
   </a>
   正在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theorganicprepper.com/gun-owner-social-media-search-history/" href="https://www.theorganicprepper.com/gun-owner-social-media-search-history/" rel="noopener noreferrer" target="_blank">
    发生
   </a>
   。
  </p>
  <p class="graf graf--p">
   e-hallpass 在其隐私政策中表示，学生数据不会与第三方公司共享以进行营销或广告宣传，家长可以请求删除。
  </p>
  <p class="graf graf--p">
   但是，父母呢？他们对收集哪些关于孩子的数据以及如何使用这些数据有发言权吗？
  </p>
  <p class="graf graf--p">
   正如《邮报》
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/technology/2019/10/29/school-apps-track-students-classroom-bathroom-parents-are-struggling-keep-up/?arc404=true" href="https://www.washingtonpost.com/technology/2019/10/29/school-apps-track-students-classroom-bathroom-parents-are-struggling-keep-up/?arc404=true" rel="noopener noreferrer" target="_blank">
    解释的那样
   </a>
   ：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    家庭通常已经在校外采取了各种预防措施，比如在家里执行屏幕时间规则，并限制家人在社交媒体上发布自己孩子的照片。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    但是，控制在学校发生的监视变得更加困难
   </strong>
   ，部分原因是不需要学区告知家长学生使用的每一种软件。这些应用程序以及部署它们的学校对于如何使用、共享和存储数据具有不同的规则。
  </p>
  <p class="graf graf--p">
   全国各地的学校都在使用数百种大小不一的应用程序，从跟踪家庭作业到改变行为，无所不能。
  </p>
  <p class="graf graf--p">
   因此，学校甚至不必告诉父母他们使用的是哪种类型的监视跟踪应用程序和软件，这些监视技术收集了哪些其他数据也没人知道。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     学生们对这种监视的感觉如何？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   高中生 Christian Chase 对华盛顿邮报
   <a class="markup--anchor markup--p-anchor" data-href="https://www.change.org/p/heritage-high-school-abolish-e-hallpass-at-heritage-high-school" href="https://www.change.org/p/heritage-high-school-abolish-e-hallpass-at-heritage-high-school" rel="noopener noreferrer" target="_blank">
    说
   </a>
   ：“我只知道这是对我们隐私的侵犯，我认为这不是必须采取的措施。我会了解这是否瞄准特定人群甚至仅仅针对下层阶级。”
  </p>
  <p class="graf graf--p">
   Chase 在 Change.org 上发表了一份请愿书。他描述了该监视系统如何工作以及为什么必须反对该监视系统：
  </p>
  <p class="graf graf--p">
   新系统要求学生在移动设备或笔记本电脑上输入老师给的密码，否则不能去学校的任何地方，包括上厕所、喝水或去看辅导员。在通行证有效期间，
   <strong class="markup--strong markup--p-strong">
    它在整个学校范围内任何地方跟踪你
   </strong>
   ，管理员可以检查谁在走廊上聚会。这是对学生隐私的严重侵犯，完全没有必要。
  </p>
  <p class="graf graf--p">
   去年，对伊利诺伊州高中的学生进行了关于电子通行证的调查。大多数人对该系统不满意：有70％的学生认为电子通行证可能会干扰教室内的学习过程。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     家长对该监视应用程序一无所知
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   一些父母对使用该应用程序表示严重担忧。
  </p>
  <p class="graf graf--p">
   主题是社交媒体法律、生活和技术的 Shear 网站其隐私律师 Brad Shear 的两个孩子在马里兰州的蒙哥马利县学区读书。
  </p>
  <p class="graf graf--p">
   他告诉《邮报》，如果学区决定在学校测试电子通行证，他和其他父母会联合反对，他说：“这是卫生间里的老大哥”。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     一些政治家也表达了疑问
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   最近，一些立法者给50多家教育技术公司和数据收集者写了封信，要求他们解释他们收集什么样的信息以及如何使用它们。
  </p>
  <p class="graf graf--p">
   发送信件的公司列表包括：
  </p>
  <p class="graf graf--p">
   Google, Facebook, Smart Sparrow, DreamBox Learning, ScootPad, ST Math, Curriculum Associates i-Ready, Realizeit, Macmillan, McGraw-Hill, Snapwiz, Kaplan, Wiley Education Services, the College Board, ACT, Pearson, Student Opportunity Center, Cognitive ToyBox, AdmitHub, Upswing, Formative, Flocabulary, BrightBytes, Hapara, Intellus Learning, Civitas Learning, Education Elements, NoRedInk, StraighterLine, Turnitin, Cengage, VitalSource, RedShelf, Barnes &amp; Noble Education, Canvas Instructure, Blackboard, Sakai, Moodle, D2L Brightspace, Edmodo, Quizlet, Schoology Accurate Leads, American Student Marketing, AmeriList, ASL Marketing, Caldwell List Co., Complete Mailing Lists, DataMasters, DMDatabases, Dunhill International List Co., Exact Data, InfoUSA, Lake B2B, NRCCUA and Scholarships.com.
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     您可以采取什么措施来保持孩子的数据安全？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   1、研究现有的《家庭教育权利和隐私法案》（FERPA），《学生权利保护修正案》（PPRA），《儿童在线隐私保护法案》（COPPA），以及适用于 EdTech 服务的地方法律。
  </p>
  <p class="graf graf--p">
   2、加入本地社区的讨论，与社区成员联合。
  </p>
  <p class="graf graf--p">
   3、对上级联盟和信息共享权利组织进行研究，以寻求支撑。
  </p>
  <p class="graf graf--p">
   4、研究与学校有关的网络漏洞。
  </p>
  <p class="graf graf--p">
   5、考虑使用身份盗用监控，以检查有关孩子身份的任何欺诈性使用。
  </p>
  <p class="graf graf--p">
   6、定期对孩子的信息进行互联网搜索，以帮助确定其信息的暴露和传播程度。
  </p>
  <p class="graf graf--p">
   由纽约的 Leonie Haimson 和科罗拉多州的 Rachael Stickland 于2014年7月成立的学生隐私家长联盟提供了免费的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.studentprivacymatters.org/toolkit/" href="https://www.studentprivacymatters.org/toolkit/" rel="noopener noreferrer" target="_blank">
    学生隐私家长工具包
   </a>
   ，您可以下载该工具包。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     儿童是否已适应扩大化的警察国家状态？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   孩子们正在被编程为无原则接受任何 “权威人物” 的要求 —— 包括监视他们上厕所。
  </p>
  <p class="graf graf--p">
   这些所谓的 “权威人物” 教导孩子们必须以安全为名牺牲隐私（该电子通行证宣传视频一再提及“安全”），并使孩子们认为24/7监视是 “正常和必要的”。
  </p>
  <p class="graf graf--p">
   也许最令人不安的是，被定罪的年龄越来越低。
  </p>
  <p class="graf graf--p">
   John W. Whitehead 写道：
  </p>
  <p class="graf graf--p">
   学校已经拥有了威权主义特征。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    从孩子进入98,000所公立学校之一到毕业的所有时间里，他们都在受到严格的饮食限制；
   </li>
   <li class="graf graf--li">
    严厉的零容忍政策将幼稚的行为定为犯罪；
   </li>
   <li class="graf graf--li">
    将言论定为犯罪；
   </li>
   <li class="graf graf--li">
    标准化测试强调死记硬背的答案；
   </li>
   <li class="graf graf--li">
    政治正确导致年轻人自我审查并审查周围人；
   </li>
   <li class="graf graf--li">
    以及广泛的生物识别和各种其他监视系统，迫使年轻人去适应没有思想、言论或行动自由的世界。
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    政府开展了以利润为导向的监视扩展运动，许多学校都已将自己转变为准监狱，并配备了监控摄像头、金属探测器、警察巡逻、零容忍政策、封锁、缉毒犬、脱衣搜身和射手训练

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     牺牲自由以换取所谓的“安全”是危险的
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   某些人可能依旧相信所有这些监视都是为了某种程度上所谓的安全。就如当权者灌输的那样：“如果你没有做错任何事，那就不用担心。”
  </p>
  <p class="graf graf--p">
   在2016年的一次采访中，Edward Snowden 对此表示：
  </p>
  <p class="graf graf--p">
   隐私不是要隐藏东西。隐私是要保护东西。保护的就是你。是隐私让你之所以成为你。隐私是自我的权利。
  </p>
  <p class="graf graf--p">
   隐私使您能够按照自己的意愿与世界分享自己的身份。让世界了解您要成为的人，并保护自己不确定的部分，因为您仍在努力。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    没有隐私的人就会失去做自己的能力。隐私是所有其他权利的源泉
   </strong>
   。
  </p>

     同一枚硬币的两面 — — 隐私权和言论自由权

  <p class="graf graf--p">
   如果您无法拥有一个安静的空间，一个内在的空间，那么言论自由就不再有意义。
  </p>
  <p class="graf graf--p">
   没有隐私，您将一无所有。
  </p>
  <p class="graf graf--p">
   如果你认为自己可以不关心隐私是因为你 “没什么可隐藏的”，就如同你不关心言论自由，是因为你无话可说。(
   <a class="markup--anchor markup--p-anchor" data-href="https://www.businessinsider.com/edward-snowden-privacy-argument-2016-9" href="https://www.businessinsider.com/edward-snowden-privacy-argument-2016-9" rel="noopener noreferrer" target="_blank">
    source
   </a>
   )
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    权利的运作方式就是，政府必须证明侵犯你的权利的 “正当性”。
   </strong>
  </p>
  <p class="graf graf--p">
   您可能会认为监视孩子上厕所 “没什么大不了的”，但请考虑一下大局。
  </p>
  <p class="graf graf--p">
   您如何说服同胞政府为他工作，而他一生的大部分时间都被监禁在一个机构中，该机构教导年轻人要学习做服从和顺从的人，不要反驳，不要质疑，也不要挑战权威？(
   <a class="markup--anchor markup--p-anchor" data-href="https://www.rutherford.org/publications_resources/john_whiteheads_commentary/say_no_to_hardening_the_schools_with_zero_tolerance_policies_and_gun_toting_cops" href="https://www.rutherford.org/publications_resources/john_whiteheads_commentary/say_no_to_hardening_the_schools_with_zero_tolerance_policies_and_gun_toting_cops" rel="noopener noreferrer" target="_blank">
