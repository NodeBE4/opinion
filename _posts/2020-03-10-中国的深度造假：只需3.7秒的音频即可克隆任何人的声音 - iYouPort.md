  <p class="graf graf--p">
   只需3.7秒的音频，中国科技巨头百度开发的新 “AI算法” 就可以克隆出相当可信的假声音。
  </p>
  <p class="graf graf--p">
   就像机器学习软件的迅速发展使假视频的创建变得非常大众化那样，
   <strong class="markup--strong markup--p-strong">
    这项研究也表明了为什么越来越难以相信互联网上发布的任何媒体信息 —— 不仅是图片、视频，现在还包括声音，都有可能是假的
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%bd%93%e6%81%b6%e6%84%8f%e8%80%85%e6%8e%8c%e6%8f%a1%e6%8a%80%e6%9c%af%ef%bc%9a%e4%bf%a1%e6%81%af%e6%88%98%e7%9a%84%e6%9c%aa%e6%9d%a5%e7%a9%b6%e7%ab%9f%e6%9c%89%e5%a4%9a%e5%8f%af%e6%80%95%ef%bc%9f/" href="https://www.iyouport.org/%e5%bd%93%e6%81%b6%e6%84%8f%e8%80%85%e6%8e%8c%e6%8f%a1%e6%8a%80%e6%9c%af%ef%bc%9a%e4%bf%a1%e6%81%af%e6%88%98%e7%9a%84%e6%9c%aa%e6%9d%a5%e7%a9%b6%e7%ab%9f%e6%9c%89%e5%a4%9a%e5%8f%af%e6%80%95%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     当恶意者掌握技术：信息战的未来究竟有多可怕
    </a>
    ？》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%b7%b1%e5%ba%a6%e9%80%a0%e5%81%87%e7%9c%9f%e7%9a%84%e6%98%af%e4%b8%80%e5%9c%ba%e7%9c%9f%e7%9b%b8%e5%b7%b2%e6%ad%bb%e7%9a%84%e5%8d%b1%e6%9c%ba%e5%90%97%ef%bc%9f%e6%88%91%e4%bb%ac%e5%a6%82%e4%bd%95/" href="https://www.iyouport.org/%e6%b7%b1%e5%ba%a6%e9%80%a0%e5%81%87%e7%9c%9f%e7%9a%84%e6%98%af%e4%b8%80%e5%9c%ba%e7%9c%9f%e7%9b%b8%e5%b7%b2%e6%ad%bb%e7%9a%84%e5%8d%b1%e6%9c%ba%e5%90%97%ef%bc%9f%e6%88%91%e4%bb%ac%e5%a6%82%e4%bd%95/" rel="noopener noreferrer" target="_blank">
     深度造假真的是一场真相已死的危机吗？我们如何化解这场危机？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   这家科技巨头的研究人员展示了他们在 “Deep Voice” 项目中的最新进展，该系统是用于克隆语音的。
  </p>
  <p class="graf graf--p">
   一年前，该技术还需要大约30分钟的音频才能创建新的假音频剪辑。现在，仅需几秒钟的音频材料输入，它就可以产生更好的结果。
  </p>
  <p class="graf graf--p">
   当然，获得的训练样本越多，输出的效果就越好：单一源输入的结果听起来仍然有点嘈杂，但并不比低质量的真实音频文件差很多。
  </p>
  <p class="graf graf--p">
   您可以
   <a class="markup--anchor markup--p-anchor" data-href="https://audiodemos.github.io/" href="https://audiodemos.github.io/" rel="noopener noreferrer" target="_blank">
    在此处收听
   </a>
   声音样本和 AI 生成的结果。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该系统可以将女性的声音转换为男性，将英国人的口音转换为美国口音，这表明AI可以学习模仿不同的说话风格，将文本语音转换提升到了新的水平
   </strong>
   。
  </p>
  <p class="graf graf--p">
   研究人员在有关该研究的
   <a class="markup--anchor markup--p-anchor" data-href="http://research.baidu.com/neural-voice-cloning-samples/" href="http://research.baidu.com/neural-voice-cloning-samples/" rel="noopener noreferrer" target="_blank">
    博客文章
   </a>
   中写道：“语音克隆有望在人机界面的个性化方向上发挥重要作用。”（百度链接，谨慎打开）
  </p>
  <p class="graf graf--p">
   Deep Voice 的这一迭代标志着近年来AI生成的语音模仿的又一发展。
  </p>
  <p class="graf graf--p">
   Adobe 在2016年
   <a class="markup--anchor markup--p-anchor" data-href="https://motherboard.vice.com/en_us/article/jpgkxp/after-20-minutes-of-listening-new-adobe-tool-can-make-you-say-anything" href="https://motherboard.vice.com/en_us/article/jpgkxp/after-20-minutes-of-listening-new-adobe-tool-can-make-you-say-anything" rel="noopener noreferrer" target="_blank">
    已经展示
   </a>
   了其 VoCo 软件，该软件可以在听了20分钟的声音后从文本生成语音。总部位于蒙特利尔的AI初创公司 Lyrebird
   <a class="markup--anchor markup--p-anchor" data-href="https://techcrunch.com/2017/04/25/lyrebird-is-a-voice-mimic-for-the-fake-news-era/" href="https://techcrunch.com/2017/04/25/lyrebird-is-a-voice-mimic-for-the-fake-news-era/" rel="noopener noreferrer" target="_blank">
    也声称
   </a>
   它仅用一分钟的音频就能进行文本到语音的转换。
  </p>
  <p class="graf graf--p">
   这些技术代表了AI进步的一种飞跃，研究人员和理论家对深度造假的担忧一直是人们关注的重点。
  </p>
  <p class="graf graf--p">
   如果只需要几秒钟的声音和一个人脸的数据集，就可以轻松地伪造出整个采访、新闻发布会或新闻片段 …… 任何新闻都不会再可信了，是吗？⚪️
  </p>
  <p class="graf graf--p graf--startsWithSingleQuote">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.vice.com/en_us/article/3k7mgn/baidu-deep-voice-software-can-clone-anyones-voice-with-just-37-seconds-of-audio" href="https://www.vice.com/en_us/article/3k7mgn/baidu-deep-voice-software-can-clone-anyones-voice-with-just-37-seconds-of-audio" rel="noopener noreferrer" target="_blank">
    ‘Deep Voice’ Software Can Clone Anyone’s Voice With Just 3.7 Seconds of Audio
   </a>
  </p>
