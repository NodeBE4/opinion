  <p class="graf graf--p">
   随着面部识别技术变得越来越普遍，人们开始反击 —— 这场反击是由时装设计师引领的。
  </p>
  <p class="graf graf--p">
   我们生活在大规模监视的时代，在这种监视中，所有人在公开场合所做的几乎每件事都被监视和记录着。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab%e5%8f%8a%e5%af%b9%e6%8a%97%e6%96%b9%e6%b3%95/" href="https://www.iyouport.org/category/%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab%e5%8f%8a%e5%af%b9%e6%8a%97%e6%96%b9%e6%b3%95/" rel="noopener noreferrer" target="_blank">
    面部识别技术
   </a>
   在一定程度上受到执法机构越来越多的使用的推动，近年来迅速普及，并且这一趋势预计将持续到未来很多年。
  </p>
  <p class="graf graf--p">
   根据 MarketsandMarkets 最近发布的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.marketsandmarkets.com/Market-Reports/facial-recognition-market-995.html?gclid=EAIaIQobChMItv7QoZjG5QIViKQYCh1Mww3VEAAYASAAEgLbQ_D_BwE" href="https://www.marketsandmarkets.com/Market-Reports/facial-recognition-market-995.html?gclid=EAIaIQobChMItv7QoZjG5QIViKQYCh1Mww3VEAAYASAAEgLbQ_D_BwE" rel="noopener noreferrer" target="_blank">
    一份报告
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    预计全球面部识别市场将从2019年的32亿美元增长到2024年的70亿美元。
   </strong>
  </p>
  <p class="graf graf--p">
   这些监视相机在街道、酒吧、饭店、购物中心、机场、工作场所、甚至人们的家里。难道我们对此无能为力吗？
  </p>

  <p class="graf graf--p">
   许多权利组织呼吁禁止该技术，但政府的行动格外缓慢。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    而来自世界各地的艺术家、设计师和研究人员并没有等待政府去听从这些呼吁，而是决定通过设计服装和配饰来实际帮助人们解决问题，这些服装和配饰可以愚弄面部识别算法。
   </strong>
  </p>
  <p class="graf graf--p">
   在 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab/" href="https://www.iyouport.org/category/%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab/" rel="noopener noreferrer" target="_blank">
    生物识别和对抗方法
   </a>
   ” 栏目中我们已经介绍过很多反监视直接行动的创意，以下是一些汇总。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     1、一条新的服装生产线旨在混淆监控摄像机
    </strong>
   </span>
  </h3>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “对抗时尚” 是由计算机安全专业人士和业余时装设计师 Kate Rose 设计的一套新服装系列。
  </p>
  <p class="graf graf--p">
   它是与众不同的。
  </p>
  <p class="graf graf--p">
   “对抗时尚” 系列中的物品首次在拉斯维加斯的 DefCon 网络安全会议上亮相，它采取了非常不寻常的设计，其特征是各种车牌图像。
  </p>
  <p class="graf graf--p">
   但是，这种非正统的设计的主要目的不是要发表任何形式的时尚声明，而是要使自动车牌读取器感到困惑。近年来，这种自动车牌读取器已在全球变得越来越普遍。
  </p>
  <p class="graf graf--p">
   自动车牌阅读器通常安装在电线杆、路灯、高速公路立交桥或移动拖车上，它们使用联网的监控摄像头和图像识别技术，通过将其抓取的车牌号与特定的GPS位置信息及日期和时间相关联，以跟踪人们的行迹。
  </p>
  <p class="graf graf--p">
   但是，该技术远非完美，往往会出现准确性问题，经常会错误地从广告牌和栅栏中抓取数字。
  </p>
  <p class="graf graf--p">
   这就激发了 Rose 尝试设计能够模仿车牌花样的衣服，并借此欺骗车牌阅读器，使其认为那是一辆汽车。
  </p>
  <p class="graf graf--p">
   对抗时尚系列包括衬衫、帽衫、夹克、短裙和连衣裙，价格从25美元到55美元不等。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://adversarialfashion.com/" href="https://adversarialfashion.com/" rel="noopener noreferrer" target="_blank">
    该网站
   </a>
   还包含各种DIY资源，例如API和图像编辑工具，可让任何人设计自己的反监视服装。
  </p>
  <p class="graf graf--p">
   Rose 希望她的创作能够提高人们对面部识别技术侵入性的认识，甚至可以激发人们对此进行反击。
  </p>
  <p class="graf graf--p">
   Rose 表示：“其中一部分只是教人们如何使用该技术，并提高人们对部分监视状态的理解。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “但这确实有效。即使只是一个人穿着其中一件衣服，并且导致这些监视相机摄取了10到50个完全是垃圾的假车牌，也是非常有意义的事。对机器来说所有一切信息都是必须存储和分析的，所有这些都要花钱。”
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     2、Hyperface 项目使佩戴者无法被面部识别软件读取
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   Hyperface 项目是总部位于柏林的艺术家兼技术专家 Adam Harvey 与国际互助工作室 Hyphen-Labs 合作的产物，采用了与对抗时尚类似的方法。
  </p>
  <p class="graf graf--p">
   除了车牌之外，它还涉及采取看起来具有面部特征（例如眼睛、嘴巴和鼻子）的图案遮盖衣物或纺织品，以便诱使面部识别软件将它们标记为人脸。
  </p>
  <p class="graf graf--p">
   这样做的目的是使计算机将注意力从佩戴者的实际面部上移开，然后将注意力集中在这些错误的 “面部” 上，从而可导致系统中出现数千次错误命中，使其无法分辨出哪些面部是真实的。
  </p>
  <p class="graf graf--p">
   一旦印在一件衣服或纺织品上，这些图案就可以被穿着、或被用来覆盖一个区域。
  </p>
  <p class="graf graf--p">
   这些图案针对特定算法，包括 OpenCV 的默认正面人脸轮廓，基于 HoG / SVM 的边缘检测器和卷积神经网络。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/D0J4yj0gz-Y" width="750">
   </iframe>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     3、Phantom 眼镜阻止面部识别技术
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   由设计师 Scott Urban 创建的 Phantom 眼镜是隐私眼镜的新系列，旨在阻止某些形式的面部识别技术。
  </p>
  <p class="graf graf--p">
   更具体地说，这种眼镜旨在通过红外光反射来欺骗3D点阵面部映射系统，例如 Apple 的 Face ID 中使用的那种。这样一来，它们就可以防止IR摄像机捕获您的面部清晰图像。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “将隐私等同于罪犯的时代已经过去，” Urban 说。“现在，一切都将变成了公司/政府使用您的脸来了解有关您的所有信息，并将所有数据收集到简洁的个人资料中。我和许多其他人只是不想生活在布满监视系统的世界里。”
  </p>
  <p class="graf graf--p">
   除了智能手机中的面部识别技术外，越来越受欢迎的红外监视摄像头也依赖于3D点阵映射。通常安装在酒吧、饭店、停车场和街道拐角处，它们使用 IR LED 灯照亮黑暗的地方，然后使用特殊的摄像头捕获最终的图像。
  </p>
  <p class="graf graf--p">
   它被认为是最准确的面部识别技术之一，并且很快也将成为最常见的一种。但是，由于有了 Phantom 眼镜，您就不必再为此担心了。
  </p>
  <p class="graf graf--p">
   这些眼镜将使您无论白天或黑夜都可以保护自己的隐私。
   <strong class="markup--strong markup--p-strong">
    “我只是提供了退出功能，因为这些监视系统不提供退出功能，它们无处不在，不受管制”
   </strong>
   ，Urban 说。
  </p>
  <p class="graf graf--p">
   Phantom 眼镜看起来就像一副普通的眼镜，尽管镜框略宽。反射的红外光束会使您的面部数据失真，从而使3D红外测绘技术混乱，以此保护您的隐私。
  </p>
  <p class="graf graf--p">
   这款眼镜的售价为142美元，预计将于2020年开始发货。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/giK6mEc667k" width="750">
   </iframe>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     4、Incognitio 是一款极简主义的面具，可帮助佩戴者避免面部识别
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   波兰设计师 Ewa Nowak 开发了一种名为 Incognitio 的极简金属面罩，该面罩可以使面部识别算法无法识别佩戴者的脸。
  </p>
  <p class="graf graf--p">
   这款面部装饰是用黄铜制成的，具有三个突出的部分：一个细长的多边形，延伸到佩戴者的额头上，两个圆圈定位在佩戴者眼睛的下方。
  </p>
  <p class="graf graf--p">
   这些零件通过一束金属丝连接在一起，形状适合面部轮廓，像眼镜一样可以固定在耳朵后面的头部。
  </p>

   <img alt="" class="aligncenter size-full wp-image-11275 jetpack-lazy-image" data-lazy-sizes="(max-width: 750px) 100vw, 750px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.jpg?resize=750%2C1121&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.jpg?w=750&amp;ssl=1 750w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.jpg?resize=201%2C300&amp;ssl=1 201w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.jpg?resize=685%2C1024&amp;ssl=1 685w" data-recalc-dims="1" height="1121" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.jpg?resize=750%2C1121&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="750"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11275" data-recalc-dims="1" height="1121" sizes="(max-width: 750px) 100vw, 750px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.jpg?resize=750%2C1121&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.jpg?w=750&amp;ssl=1 750w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.jpg?resize=201%2C300&amp;ssl=1 201w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.jpg?resize=685%2C1024&amp;ssl=1 685w" width="750"/>
   </noscript>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “该项目涉及反监视社会和在公共场所保护自己的形象这一主题。旨在保护佩戴者免受安装在公共场所的现代相机中使用的面部识别算法的影响”，Nowak 解释说。
  </p>
  <p class="graf graf--p">
   这款装饰面罩通过遮盖脸部的关键部位从而阻止了算法识别佩戴者。
  </p>
  <p class="graf graf--p">
   Nowak 上传了几张佩戴该饰品的模特的图像，对 Facebook 的 DeepFace 算法进行了测试，该算法无法识别这些图像。
  </p>
  <p class="graf graf--p">
   虽然吸引了很多兴趣，但 Incognitio 面罩目前仅是一个概念，尚不清楚是否以及何时可以购买。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     时尚在反击大众监控中的作用
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   面部识别技术几乎遍及我们社会的各个方面，几乎不可能逃脱其笼罩。
  </p>
  <p class="graf graf--p">
   尽管众多隐私和民权组织多次呼吁禁止面部识别技术，但因为监视技术太赚钱了，其市场规模仍在不断增长。
  </p>
  <p class="graf graf--p">
   作为响应，许多艺术家和时装设计师创造了旨在帮助人们避免被面部识别算法检测到的衣服和配饰。
  </p>
  <p class="graf graf--p">
   此处列出的大多数示例都是为利用大规模监视系统中的已知缺陷而设计的。但是，随着技术的不断进步，这些缺陷必将最终得到解决，到那时将使这些产品失去效用。
  </p>
  <p class="graf graf--p">
   他们的创作者非常清楚这一点，他们指出设计并不是要解决这个问题，而是要教育人们大规模监视技术的工作原理以及如何与之对抗。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “用于识别任何类型图像的监视系统每天都在变得更精进。计算机视觉领域正在不断进步。我知道我的衬衫可能会在几年后失效。我看过其他反监视艺术家的旧项目，那些项目已经不再起作用了，但我觉得重要的是，
   <strong class="markup--strong markup--p-strong">
    尽我所能，即使我只是一个人在反抗，也要竭尽全力
   </strong>
   ”，Rose 说。
  </p>
  <p class="graf graf--p">
   这就是直接行动。
  </p>
  <p class="graf graf--p">
   近年来的反监视时尚已经变得越来越流行。尽管其中一些设计已被证明是相当有效的，但重要的是，
   <strong class="markup--strong markup--p-strong">
    它们不应被视为解决此问题的最终方案，而应被视为促使人们就此进行对话的一种有效方式。
   </strong>
  </p>
  <p class="graf graf--p">
   尽管存在广泛的担忧，但面部识别技术不太可能消失。如果使用得当，它可以成为有益的力量，但是政府需要找到一种方法来解决与其使用相关的隐私和安全问题。在此之前，反监视时尚可能是保护您的最佳选择。⚪️
  </p>
