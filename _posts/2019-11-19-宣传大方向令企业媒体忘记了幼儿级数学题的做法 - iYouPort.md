  <p class="graf graf--p">
   不幸的是，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.fppolitics.com/msnbc-misreports-data-shortchanges-bernie-sanders/" href="https://www.fppolitics.com/msnbc-misreports-data-shortchanges-bernie-sanders/" rel="noopener noreferrer" target="_blank">
    她忘了说

   她引用的研究只收集了捐赠200美元或以上的捐赠者，200以下的全抹掉了。还赞扬 Gillibrand “在针对女性捐赠者方面做得最好”。

   Maddow 引用解释指出 Rachel Maddow 的结果是偏颇的：
  </p>

   <p>
    <em>
     桑德斯从小额捐赠者那里获得的资金最多……达到74％…小额捐款的性别格局并不存在。
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p">
   事实上，根据桑德斯的传播总监的说法，在第一季度为桑德斯竞选做出贡献的525,000人中有46％是女性。
  </p>
  <p class="graf graf--p">
   她发推文说，“几乎可以肯定，为我们的活动捐赠的女性比给其他任何人捐赠的女性都多。”
  </p>
  <p class="graf graf--p">
   Rachel Maddow 说：“你是学者，还主持一个晚间新闻节目，每年赚700万美元，却没有透露这项研究只关注那些富有的家伙……”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    6、23分钟= 5分钟
   </strong>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    结果：桑德斯被从突出反对种族主义和性别歧视变成了“根本不提”这些问题。
   </em>
  </p>
  <p class="graf graf--p">
   3月份，MSNBC 的 Alex Whit 主持了一个小组讨论桑德斯3月2日的竞选启动演讲。
  </p>
  <p class="graf graf--p">
   该小组成员和 MSNBC 政治分析家 Zerlina Maxwell 说：“我计时了。他[Bernie Sanders]直到演讲23分钟才提及种族或性别问题。”
  </p>
  <p class="graf graf--p">
   正如桑德斯的代理人、记者、组织者、活动家和推特上的人们所指出的那样，桑德斯在演讲五分钟时就提到了种族和性别问题，当时他说：“我们政府的基本原则是不要种族主义、性别歧视、仇外心理、同性恋恐惧症和宗教偏见”。
  </p>

   视频：
  </p>

   之后前希拉里克林顿的职员 Maxwell 确实删除了一条推文，上面写道：“在桑德斯在演讲中直到23分钟时才提到种族和性别。”
  </p>
  <p class="graf graf--p">
   桑德斯谈到了刑事司法背景下的种族和性别问题，任何观看演讲的人都会知道这点。但报道的准确性似乎并不是那么重要了，由于你无法解释的原因。公民 —— 包括 MSNBC 声称代言的公民 —— 本应该得到更好的待遇，而不是被低端蒙骗。⚪️
