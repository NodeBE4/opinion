  <p class="graf graf--p">
   对所有所有传媒说，现在是时候采用不同的社交媒体方式了 — — 2019 年可能也应该会成为出版商（新闻媒体机构）投入资源并专注于最终可以做出什么样的改变的一年。
  </p>
  <p class="graf graf--p">
   2018 年一定程度上已经成为社交媒体大肆失宠的一年。首先是 Facebook 算法的改变，它从人们的 NewsFeed 中擦除了很多媒体页面。然后是剑桥分析公司的丑闻、以及随之而来的且备受鼓励的#deletefacebook 运动。更何况 Facebook 被指控在视频统计上撒谎。
  </p>
  <p class="graf graf--p">
   我们非常支持#deletefacebook 运动，但是必须承认，社交媒体已经垄断了绝大多数人的新闻来源，几乎所有媒体一直以来都在严重依赖社交媒体作为宣传管道，而不是传统的卖报纸那么直接。于是社交媒体巨头的审查和 Feeds 定位算法，将对媒体的生存产生致命的影响。
  </p>
  <p class="graf graf--p">
   并且，更有：
  </p>
  <p class="graf graf--p">
   WhatsApp 如何专门用来传播假消息？ 还记得 Snapchat 重新设计崩溃和随之而来的危机？ Instagram 的骚扰问题如此严重怎么办？机器人充满了 Twitter 的水军，然后呢？……
  </p>
  <p class="graf graf--p">
   在经历了如此可怕的一年之后，社交媒体如何继续成为出版商数字战略的中心？
  </p>
  <p class="graf graf--p">
   我们现在可能需要重新思考这件事了。曾经关于社交网络终结的预测从未被证明是正确的。平台极有可能将继续存在至少在足够长的时间内，包括所有问题和明显的弊端。新闻界等不了这么久。追求跨越它们已经几乎是不可能的，除非我们决定让数百万人“落后” — 因为付费墙的新兴而被锁定在媒体网站之外。
  </p>
  <p class="graf graf--p">
   在2018年，马克扎克伯格一直在密切关注他的公司的座右铭，即所谓的“快速行动并打破局面”，因为他宣布 Facebook 的 NewsFeed 将显示更多来自“朋友”的内容，而不是来自出版商。通过相应地改变算法，扎克伯格已经打破了长期以来的错觉，即 Facebook “可以”满足主流媒体的增长战略。
  </p>
  <p class="graf graf--p">
   但是重点在于 Facebook 完全不能满足媒体，因为媒体需要多元化，你的“朋友”提供的都是严重单一化的内容，因为谁是“朋友”的算法被 Facebook 掌握，它高度依赖于是否与你同质。不止 Facebook，单一平台永远都是不够的 — — 即使每月有27亿活跃用户也不行。
  </p>
  <p class="graf graf--p">
   因此，在这个阶段，2019 年是我们应该开始
   <strong class="markup--strong markup--p-strong">
    应对社交媒体问题并通过建立社区和加强信任 — — 从多个平台中获益的一年
   </strong>
   。明年，我们将以更新的方式和不同的意识迎接转变。我们已经见证了一些例子。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “经济学人”的读者与编辑 Adam Smith 于4月创建了一个 Facebook 小组，以促进有关市场、技术和民主的角色对话，为“经济学人开放未来会议”活动做准备。
  </p>
  <p class="graf graf--p">
   想象一下，数百人被要求就性别平等、民粹主义、气候变化等争议性最大的主题分享他们自己的观点 ……听起来像是完美的风暴吧，如果发生在中国，基本上只能是风暴。不过这在里正相反，由于简单的规则和谨慎的审核，“开放未来”成为了一个积极的对话环境。
  </p>
  <p class="graf graf--p">
   为什么？因为批判性思考的教育和传统足够深入人心。这就是中国的教育体制缺乏的。
  </p>
  <p class="graf graf--p">
   并且，最重要的就是，
   <strong class="markup--strong markup--p-strong">
    无论是点击率还是后期覆盖率都不是根本问题
   </strong>
   — 人们的想法才是，这有助于创建一个积极主动的和参与的社区，即使在今天，开放未来会议活动已经结束了几个月后仍在被讨论。
  </p>

  <p class="graf graf--p">
   经济学人不是唯一的。另一个有趣的实验是 ExposedABC 小组。澳大利亚广播公司创建了它，以建立一个业余侦探社区，就是 IYP 主张的开源调查信息挖掘，他们的目标是深入了解澳大利亚
   <a class="markup--anchor markup--p-anchor" data-href="https://www.abc.net.au/tv/programs/exposed-the-case-of-keli-lane/" href="https://www.abc.net.au/tv/programs/exposed-the-case-of-keli-lane/" rel="noopener noreferrer" target="_blank">
    Keli Lane 的疑案
   </a>
   。阴谋论、错误信息和一些仇恨言论的空间很大吗？ 逆转它！美国广播公司新闻报道的 Caro Meldrum-Hanna 使用了拥有超过 30,000 名成员的 Facebook 小组来揭穿并挑战围绕这个故事出现的每一种神话。
  </p>
  <p class="graf graf--p">
   去年四月，The Telegraph 推出了自己的 Facebook 小组，以讨论围绕自由市场政策的新想法。目标是
   <strong class="markup--strong markup--p-strong">
    与年轻观众互动，以便“为年轻人制作一系列文章和评论”
   </strong>
   。负责该组织的记者 Daniel Capurro 和 Helena Horton 发布了促成对话和成员贡献的文章，最终发表在“电讯报”网站上。同年，“每日电讯报”还任命 Robin Hough 为社区负责人，领导一个致力于社区参与项目的新记者团队。该团队特别关注评论、问答和 Facebook 小组（他们推出的最后两个小组是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.facebook.com/groups/TelegraphFamily/?ref=pages_profile_groups_tab&amp;source_id=143666524748" href="https://www.facebook.com/groups/TelegraphFamily/?ref=pages_profile_groups_tab&amp;source_id=143666524748" rel="noopener noreferrer" target="_blank">
    Telegraph Family
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.facebook.com/groups/telegraphcarsgroup/?ref=pages_profile_groups_tab&amp;source_id=143666524748" href="https://www.facebook.com/groups/telegraphcarsgroup/?ref=pages_profile_groups_tab&amp;source_id=143666524748" rel="noopener noreferrer" target="_blank">
    Telegraph Motoring Club
   </a>
   ）。
  </p>
  <p class="graf graf--p">
   但是内容不仅仅是在 Facebook 上的。
  </p>
  <p class="graf graf--p">
   虽然 Snapchat 已经步入了迄今为止最糟糕的一年，但 PinkNews 发现了在 Discover 上茁壮成长的肥沃土壤。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在短短三个月的时间里，他们就达到了吸引高达 3500 万名25岁以下的参与者
   </strong>
   — — 是的，在Z世代，你值得拥有 — — 更重要的是，他们开始尝试新的形式和内容，帮助他们与年轻读者建立关系，与绝大多数出版商合作。即使从长远来看它不适用于 Snapchat，但是，它们今天运行的测试将成为构建未来新战略的良好材料。
  </p>
  <p class="graf graf--p">
   Gen Zers 是由 Vogue International 社交策略和故事讲述负责人 Hannah Ray 领导的另一个令人兴奋的项目中心。这一次，使用的平台是 Instagram，@ vogue 主持新兴艺术家、设计师和表演者的贡献，以
   <strong class="markup--strong markup--p-strong">
    吸引年轻观众
   </strong>
   。通过发布他们的视频、图像和文字，该帐户以他们的第一人称叙述为特色，并专注于@ vogue 社区最关心的故事和主题。
  </p>
  <p class="graf graf--p">
   并且，我们曾经推荐过的，在 Twitter 上，BBC 展示了一个看起来简单的功能 — 使用 thread 能够实现新的故事叙述模式。英国广播公司“非洲之眼”的开源调查“杀戮剖析”首次作为一个10分钟的
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/BBCAfrica/status/1044186344153583616" href="https://twitter.com/BBCAfrica/status/1044186344153583616" rel="noopener noreferrer" target="_blank">
    在线视频发布
   </a>
   ，演示
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wix.com/dashboard/6fc0666c-c125-4701-93b5-303f8db9aa33/blog/5c22338b47a661001a54c012/edit/search/posts?query=%23OSINT" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    #OSINT
   </a>
   真相挖掘思路，格外引人注目。
  </p>
  <p class="graf graf--p">
   您也许会认为讨论组和鼓励媒体忽视粉丝量经济，这些方法有点简单？但是 OSINT 绝对不那么简单，它可能需要大量的投入。BBC 等主流媒体有丰厚的资金和人力资源，他们还可以雇佣专业的法医专家、谍报部门的退役分析师、以及技术高超的数据分析团队，而作为独立媒体，我们只能全凭热情。所幸我们的热情是充分的。
  </p>
  <p class="graf graf--p">
   2018年是 OSINT 的100周年纪念。OSINT 从最初的情报机构的
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E8%B0%B7%E6%AD%8C%E5%9C%B0%E5%9B%BE-%E6%98%AF%E6%AF%94%E8%A9%B9%E5%A7%86%E6%96%AF%E9%82%A6%E5%BE%B7%E6%9B%B4%E6%A3%92%E7%9A%84%E9%97%B4%E8%B0%8D-9df6eb4f6527" href="https://www.iyouport.org/%E8%B0%B7%E6%AD%8C%E5%9C%B0%E5%9B%BE%E6%98%AF%E6%AF%94%E8%A9%B9%E5%A7%86%E6%96%AF%E9%82%A6%E5%BE%B7%E6%9B%B4%E6%A3%92%E7%9A%84%E9%97%B4%E8%B0%8D/" rel="noopener noreferrer" target="_blank">
    政策擦边球
   </a>
   ，到后来的公民社会深度调查爱好者的宠物，再到现在，不仅是捍卫公民知情权的有效工具、也是抵御社交媒体假消息的有力武器，并且已经开始被主流媒体所热爱 — — 成为了
   <strong class="markup--strong markup--p-strong">
    社交媒体时代重新思考新闻业的途径
   </strong>
   。
  </p>
  <p class="graf graf--p">
   就像主流媒体对透明度革命的模仿那样（在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://steemit.com/wikileaks/@iyouport/wikileaks" href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e5%aa%92%e4%bd%93%e6%ad%a3%e5%9c%a8%e9%80%90%e6%b8%90%e5%8f%98%e6%88%90-wikileaks-%e5%aa%92%e4%bd%93%e6%94%b9%e9%9d%a9%e6%88%90%e6%95%88%e5%b7%b2%e5%88%9d%e6%ad%a5%e6%98%be%e7%8e%b0/" rel="noopener noreferrer" target="_blank">
    美国媒体正在逐渐变成 Wikileaks 媒体改革成效已初步显现
   </a>
   》）
   <strong class="markup--strong markup--p-strong">
    我们必须鼓励并推荐这种做法
   </strong>
   ，因为就如开源革命一样，如果能争取到技术巨头的支持，将事半功倍。但是，同时也应该对他们保持警惕，就如那些专利巨魔一样，主流媒体虽然开通了匿名举报平台（基本层面不是很信任他们的匿名技术，
   <a class="markup--anchor markup--p-anchor" data-href="https://pao-pao.net/article/1020" href="https://www.iyouport.org/%E4%B8%8E%E4%B8%AD%E5%9B%BD%E4%BD%95%E5%85%B6%E7%9B%B8%E4%BC%BC%EF%BC%8C%E5%B7%B4%E5%B0%94%E5%B9%B2%E8%A7%A3%E5%AF%86%E5%96%9C%E5%BF%A7%E5%8F%82%E5%8D%8A/" rel="noopener noreferrer" target="_blank">
    在这里的链接
   </a>
   看到为什么）但无疑，他们的选题将
   <strong class="markup--strong markup--p-strong">
    依旧
   </strong>
   严重受到资方和资方代表的权贵的政治立场的左右。他们也许因为拥有充分的资源可以实现比独立媒体更加优质的调查报告，但是很难指望他们能代表信息验证最基本的原则 — — 即 无任何政治意识形态色彩。（在这里看到演示《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E7%9C%9F%E5%81%87%E6%B6%88%E6%81%AF%E9%89%B4%E5%88%AB%E6%A1%88%E4%BE%8B%E6%BC%94%E7%A4%BA-%E6%98%AF%E5%90%A6%E4%B8%BA%E7%9C%9F%E5%BA%94%E8%AF%A5%E4%B8%8E%E4%BD%A0%E6%88%91%E7%9A%84%E4%B8%AA%E4%BA%BA%E7%AB%8B%E5%9C%BA%E6%AF%AB%E6%97%A0%E5%85%B3%E7%B3%BB-e2cecbdb49e0" href="https://www.iyouport.org/%e7%9c%9f%e5%81%87%e6%b6%88%e6%81%af%e9%89%b4%e5%88%ab%e6%a1%88%e4%be%8b%e6%bc%94%e7%a4%ba%ef%bc%9a%e6%98%af%e5%90%a6%e4%b8%ba%e7%9c%9f%e5%ba%94%e8%af%a5%e4%b8%8e%e4%bd%a0%e6%88%91%e7%9a%84%e4%b8%aa/" rel="noopener noreferrer" target="_blank">
    真假消息鉴别案例演示：是否为真应该与你我的个人立场毫无关系
   </a>
   》）
  </p>
  <p class="graf graf--p">
   OSINT 和绝大多数技术工具一样，它完全可以用在正义和邪恶两种方面，其邪恶用法就如俄罗斯的 trolls 和美国极右翼
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E7%96%AF%E7%8B%82%E8%83%9C%E8%BF%87%E8%99%9A%E6%97%A0-%E5%BC%80%E6%BA%90%E8%B0%83%E6%9F%A5%E7%BD%91%E7%AB%99%E5%88%9B%E5%A7%8B%E4%BA%BA%E8%B0%88-osint-%E6%8A%80%E6%9C%AF%E6%8D%8D%E5%8D%AB%E5%85%AC%E6%B0%91%E6%9D%83-b98bd3a8ff48" href="https://medium.com/@iyouport/%E7%96%AF%E7%8B%82%E8%83%9C%E8%BF%87%E8%99%9A%E6%97%A0-%E5%BC%80%E6%BA%90%E8%B0%83%E6%9F%A5%E7%BD%91%E7%AB%99%E5%88%9B%E5%A7%8B%E4%BA%BA%E8%B0%88-osint-%E6%8A%80%E6%9C%AF%E6%8D%8D%E5%8D%AB%E5%85%AC%E6%B0%91%E6%9D%83-b98bd3a8ff48" rel="noopener noreferrer" target="_blank">
    4chan 所做的那样
   </a>
   。
   <strong class="markup--strong markup--p-strong">
    如何让技术为正义服务？只能是调动更多追求正义的人们加入到对技术的使用中来
   </strong>
   。尤其是独立媒体 — — 公民知情权的守夜人。
  </p>
  <p class="graf graf--p">
   我不会骗你 — 上述这些实验本身也许并不能挽救新闻业整体。但它们是尚未公开的社交网络潜力的好例子。
  </p>
  <p class="graf graf--p">
   挑战就在那里，等待人们加入进来。希望其中有你！
  </p>

    OSINT
   </a>
   ”
