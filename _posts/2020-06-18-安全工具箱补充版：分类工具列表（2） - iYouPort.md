   <a class="markup--anchor markup--p-anchor" data-href="https://www.mozilla.org/en-US/thunderbird/" href="https://www.mozilla.org/en-US/thunderbird/" rel="noopener noreferrer" target="_blank">
    Thunderbird
   </a>
   ：跨平台的开源电子邮件应用，使用 Enigmail 插件，你可以实现邮件加密。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    注
   </strong>
   ：默认情况下 Thunderbird 不会加密你的电子邮件。Security In A Box 提供了
   <a class="markup--anchor markup--p-anchor" data-href="https://securityinabox.org/en/guide/thunderbird/linux/" href="https://securityinabox.org/en/guide/thunderbird/linux/" rel="noopener noreferrer" target="_blank">
    一份指南
   </a>
   帮助您使用 Thunderbird、GNU 隐私卫士（GNU Privacy Guard, GPG）和 Enigmail 插件来加密自己的电子邮件。
  </p>
  <p class="graf graf--p">
   经过设置，Thunderbird 可以在 TorBirdy 插件下连接 Tor 网络。
  </p>
  <p class="graf graf--p">

   。VirtualBox 可以安装在 Windows, Linux, macOS 和 Solaris 系统中。这意味着如果你因为一些原因不得不使用 Windows 或 macOS 之类的不安全系统，你可以使用虚拟机来提高安全性。
  </p>
  <p class="graf graf--p">
   <span style="color: #99cc00;">
    <strong class="markup--strong markup--p-strong">
     17、密码管理
    </strong>
   </span>
   （替代 1Password、LastPass、Roboform 等）
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.kunzisoft.com/KeepassDX/" href="https://www.kunzisoft.com/KeepassDX/" rel="noopener noreferrer" target="_blank">
    KeePass DX
   </a>
   ：安卓的。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://keepassxc.org/" href="https://keepassxc.org/" rel="noopener noreferrer" target="_blank">
    KeePassXC
   </a>
   ：为那些极其重视个人数据安全管理的人设计的，可以在一个数据库里保存用户名、密码、网址、附件和注释等各种内容。
  </p>
  <p class="graf graf--p">
   <span style="color: #99cc00;">
    <strong class="markup--strong markup--p-strong">
     18、生产力
    </strong>
   </span>
   （替代 Microsoft Outlook、GroupWise、谷歌文档、Evernote、GroupWise、iWork、Doodle 等）
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://ethercalc.net/" href="https://ethercalc.net/" rel="noopener noreferrer" target="_blank">
    EtherCalc
   </a>
   ：共享编辑的电子表格服务器，也可以自行架设。EtherCalc 的在线服务器会使用 HTTPS 技术加密您传输的数据，但随便什么人只要猜到了你的 URL 地址就可以浏览那些内容。如果您在意私密性，建议下载源码，自己搭建一个 EtherCalc 平台。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://riseup.net/en" href="https://riseup.net/en" rel="noopener noreferrer" target="_blank">
    Riseup
   </a>
   ：安全私密的通信和协作平台。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    注：
   </strong>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://prism-break.org/zh-CN/all/#email-accounts" href="https://prism-break.org/zh-CN/all/#email-accounts" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Riseup
    </strong>
   </a>
   同样也提供电子邮件、XMPP 和聊天服务，都可以通过
   <a class="markup--anchor markup--p-anchor" data-href="https://prism-break.org/zh-CN/all/#mesh-networks" href="https://prism-break.org/zh-CN/all/#mesh-networks" rel="noopener noreferrer" target="_blank">
    Tor 暗网服务
   </a>
   访问。相应的地址列表
   <a class="markup--anchor markup--p-anchor" data-href="https://www.riseup.net/en/tor#riseups-tor-hidden-services" href="https://www.riseup.net/en/tor#riseups-tor-hidden-services" rel="noopener noreferrer" target="_blank">
    在这里
   </a>
   。
  </p>
  <p class="graf graf--p">
   <span style="color: #99cc00;">
    <strong class="markup--strong markup--p-strong">
     19、社交网络
    </strong>
   </span>
   （替代 Twitter、Facebook、linkedin 等东西）
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://diasporafoundation.org/" href="https://diasporafoundation.org/" rel="noopener noreferrer" target="_blank">
    diaspora*
   </a>
   ：由社区维护的分布式社交网络。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://friendica.com/" href="http://friendica.com/" rel="noopener noreferrer" target="_blank">
    friendica
   </a>
   ：分布式联合社交网络。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://movim.eu/" href="https://movim.eu/" rel="noopener noreferrer" target="_blank">
    Movim
   </a>
   ：私密的、去中心化的社交网络服务器。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://pump.io/" href="http://pump.io/" rel="noopener noreferrer" target="_blank">
    pump.io
   </a>
   ：自行架设的社交信息流服务器。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://retroshare.net/" href="http://retroshare.net/" rel="noopener noreferrer" target="_blank">
    RetroShare
   </a>
   ：安全的 P2P 通讯平台，提供即时通讯，论坛，网络电话，文件共享等功能。
  </p>
  <p class="graf graf--p">
   <span style="color: #99cc00;">
    <strong class="markup--strong markup--p-strong">
     20、视频与音频通话
    </strong>
   </span>
   （替代 Skype、Facebook、Discord、Ventrilo、TeamSpeak等）
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://jami.net/" href="https://jami.net/" rel="noopener noreferrer" target="_blank">
    Jami
   </a>
   ：去中心化的音频/视频通话应用程序。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://www.mumble.info/" href="https://www.mumble.info/" rel="noopener noreferrer" target="_blank">
    Mumble
   </a>
   ：加密的低延迟的语音聊天。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://signal.org/" href="https://signal.org/" rel="noopener noreferrer" target="_blank">
    Signal
   </a>
  </p>
  <p class="graf graf--p">
   <span style="color: #99cc00;">
    <strong class="markup--strong markup--p-strong">
     21、浏览器插件
    </strong>
   </span>
   （替代 Ghostery）
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://decentraleyes.org/" href="https://decentraleyes.org/" rel="noopener noreferrer" target="_blank">
    Decentraleyes
   </a>
   ：保护你免受集中式的内容分发网络（CDN）的跟踪。它可以拦截到达诸如 Google Hosted Libraries 的请求，并在本地提供相应的文件。内容拦截规则会定期补充。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    注：
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    不使用提供免费服务的内容分发网络以保护您的隐私。
   </li>
   <li class="graf graf--li">
    辅助正则屏蔽器，例如 uBlock Origin，Privacy Badger 等。
   </li>
   <li class="graf graf--li">
    即装即用；使用前绝对不需要任何配置。
   </li>
  </ul>
  <p class="graf graf--p">
   Decentraleyes 不是什么银弹，但它的确能防止许多网站的跟踪。 最后，您还可以让 Decentraleyes 屏蔽任何缺失的 CDN 资源。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.eff.org/https-everywhere" href="https://www.eff.org/https-everywhere" rel="noopener noreferrer" target="_blank">
    HTTPS Everywhere
   </a>
   ：强制使用 HTTPS，为您与数千个网站间建立加密通信。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://www.mailvelope.com/" href="https://www.mailvelope.com/" rel="noopener noreferrer" target="_blank">
    Mailvelope
   </a>
   ：OpenPGP 邮件加密，支持多数 Email 服务器。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://noscript.net/" href="https://noscript.net/" rel="noopener noreferrer" target="_blank">
    NoScript
   </a>
   ：只启用你信任的网站上的 JavaScript、Java 和 Flash 内容。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    注：
   </strong>
  </p>
  <p class="graf graf--p">
   安装
   <strong class="markup--strong markup--p-strong">
    NoScript
   </strong>
   并启用 ‘全局禁用脚本’ 以提高您的浏览器的安全性。这主要可以避免 JavaScript 的零日漏洞攻击。这是一个激进的做法，因为很多网站会因此无法使用，因为它们非常依赖 JavaScript。
  </p>
  <p class="graf graf--p">
   NoScript 提供了一个白名单功能，您可以在信任的网站上选择性地启用 JavaScript，但如果你在
   <strong class="markup--strong markup--p-strong">
    Tor Browser Bundle
   </strong>
   （Tor 浏览器）上也这么做，这会被认为是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.torproject.org/docs/faq.html.en#TBBJavaScriptEnabled" href="https://www.torproject.org/docs/faq.html.en#TBBJavaScriptEnabled" rel="noopener noreferrer" target="_blank">
    对匿名性非常有害
   </a>
   的。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.eff.org/privacybadger" href="https://www.eff.org/privacybadger" rel="noopener noreferrer" target="_blank">
    Privacy Badger
   </a>
   ：可能不如传统的基于黑名单的 uBlock 或 Disconnect 等解决方案更有效。因为广告主可以通过宣称不对你进行跟踪而获得解封。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://github.com/gorhill/uBlock" href="https://github.com/gorhill/uBlock" rel="noopener noreferrer" target="_blank">
    uBlock Origin
   </a>
   ：Firefox 和 Chromium 浏览器中的一个有效的拦截插件。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    注：
   </strong>
  </p>
  <p class="graf graf--p">
   不建议您在 Tor 浏览器上安装自己的扩展程序，因为它们有可能会绕过 Tor 连接网络，或者以其他方式损害匿名性和隐私。
  </p>
  <p class="graf graf--p">
   如果您使用的是基于火狐的浏览器，你可以通过安装
   <strong class="markup--strong markup--p-strong">
    uBlock Origin
   </strong>
   、
   <strong class="markup--strong markup--p-strong">
    Request Policy
   </strong>
   和
   <strong class="markup--strong markup--p-strong">
    HTTPS Everywhere
   </strong>
   等扩展程序来保护隐私。
  </p>
  <p class="graf graf--p">
   比
   <strong class="markup--strong markup--p-strong">
    Request Policy
   </strong>
   更强大的替代品是
   <a class="markup--anchor markup--p-anchor" data-href="https://github.com/gorhill/uMatrix" href="https://github.com/gorhill/uMatrix" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     uMatrix
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p">
   安装
   <strong class="markup--strong markup--p-strong">
    NoScript
   </strong>
   扩展可以全局屏蔽或允许运行网页脚本，可以帮助您免受 JavaScript 零日漏洞的威胁。
  </p>
  <p class="graf graf--p">
   为什么不推荐 Adblock Plus ？因为 Adblock Plus 会默认显示“可接受的广告”，这与广告过滤插件的初衷相违背。建议您关闭这一选项，或者迁移到
   <strong class="markup--strong markup--p-strong">
    uBlock Origin
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <span style="color: #99cc00;">
    <strong class="markup--strong markup--p-strong">
     22、浏览器
    </strong>
   </span>
   （替代 chrome、IE、Safari、Opera 等）
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://mozilla.org/firefox" href="https://mozilla.org/firefox" rel="noopener noreferrer" target="_blank">
    Firefox
   </a>
   ：开源的 Web 浏览器，扩展丰富。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    注：
   </strong>
  </p>
  <p class="graf graf--p">
   默认情况下火狐并不会对第三方隐藏你的 IP 地址，如果你想要隐藏 IP，建议使用 Tor Browser Bundle。
  </p>
  <p class="graf graf--p">
   火狐默认情况下使用 Google 作为搜索引擎，建议您改成 DuckDuckGo 或者 Startpage 以提升匿名性。
  </p>
  <p class="graf graf--p">
   建议在工具-选项-安全（或 偏好设置-隐私）中取消
   <strong class="markup--strong markup--p-strong">
    选中 ”Block reported attack sites” 和 ”Block reported web forgeries” 选项
   </strong>
   ，以避免火狐向 Google 发送网址的哈希值。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://mike.tig.as/onionbrowser/" href="https://mike.tig.as/onionbrowser/" rel="noopener noreferrer" target="_blank">
    Onion Browser
   </a>
   ：IOS 上的开源浏览器，基于 Tor。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.torproject.org/projects/torbrowser.html.en" href="https://www.torproject.org/projects/torbrowser.html.en" rel="noopener noreferrer" target="_blank">
    Tor Browser
   </a>
   ：通过Tor网络，以加密、匿名的方式浏览。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    注：
   </strong>
  </p>
  <p class="graf graf--p">
   操作安全！《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%bf%e7%94%a8-tor-%e4%bf%9d%e6%8a%a4%e8%87%aa%e5%b7%b1%e6%97%b6%e5%8d%83%e4%b8%87%e4%b8%8d%e8%a6%81%e5%81%9a%e8%bf%99%e4%b9%9d%e4%bb%b6%e4%ba%8b%ef%bc%81/" href="https://www.iyouport.org/%e4%bd%bf%e7%94%a8-tor-%e4%bf%9d%e6%8a%a4%e8%87%aa%e5%b7%b1%e6%97%b6%e5%8d%83%e4%b8%87%e4%b8%8d%e8%a6%81%e5%81%9a%e8%bf%99%e4%b9%9d%e4%bb%b6%e4%ba%8b%ef%bc%81/" rel="noopener noreferrer" target="_blank">
    使用 Tor 保护自己时千万不要做这九件事
   </a>
   》
  </p>
  <p class="graf graf--p">
   <span style="color: #99cc00;">
    <strong>
     23、世界地图
    </strong>
   </span>
   （替代 谷歌、Bing、雅虎 等）
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://www.openstreetmap.org/" href="https://www.openstreetmap.org/" rel="noopener noreferrer" target="_blank">
    OpenStreetMap
   </a>
   ：自由的世界性的互动地图。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://osmand.net/" href="https://osmand.net/" rel="noopener noreferrer" target="_blank">
    OsmAnd
   </a>
   ：使用 OpenStreetMap 数据的开源导航系统。

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    最后
   </strong>
  </h3>
  <p class="graf graf--p">
   再次重申：仅仅使用技术工具可能无法确保万无一失，操纵安全始终是第一位的。摆脱专有软件并不容易，但是，开发者社区一直在为此努力。
  </p>
  <p class="graf graf--p">
   本文希望能帮助更多人了解开源自由软件的可行替代品，如果您觉得有用，欢迎推荐给您的朋友 —— 安全是需要协作才能完成的《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%a6%81%e6%83%b3%e4%bf%9d%e6%8a%a4%e6%af%8f%e4%b8%aa%e4%ba%ba%e7%9a%84%e4%ba%ba%e6%9d%83%e5%92%8c%e5%ae%89%e5%85%a8%ef%bc%8c%e6%88%91%e4%bb%ac%e5%b0%b1%e9%9c%80%e8%a6%81%e8%81%94%e5%90%88/" href="https://www.iyouport.org/%e8%a6%81%e6%83%b3%e4%bf%9d%e6%8a%a4%e6%af%8f%e4%b8%aa%e4%ba%ba%e7%9a%84%e4%ba%ba%e6%9d%83%e5%92%8c%e5%ae%89%e5%85%a8%ef%bc%8c%e6%88%91%e4%bb%ac%e5%b0%b1%e9%9c%80%e8%a6%81%e8%81%94%e5%90%88/" rel="noopener noreferrer" target="_blank">
    要想保护每个人的人权和安全，我们就需要联合
