  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     什么是开源情报？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   开源情报是指可以合法地从有关个人或组织的免费公共来源中收集的任何信息。
  </p>
  <p class="graf graf--p">
   开源情报不是新鲜事物，它已经有一百年的历史 —— 2018年是它的百年纪念。您可以
   <strong class="markup--strong markup--p-strong">
    在这里下载上图中提到的那本书
   </strong>
   ：
   <a class="markup--anchor markup--p-anchor" data-href="https://t.me/iyouport/6648" href="https://t.me/iyouport/6648" rel="nofollow noopener noreferrer" target="_blank">
    https://t.me/iyouport/6648
   </a>
  </p>
  <p class="graf graf--p">
   👉实际上，这往往意味着可以在互联网上找到的所有信息都属于开源情报。
  </p>
  <p class="graf graf--p">
   从技术上讲，任何公共信息都属于开源情报类别，无论是在公共图书馆中的书籍或报告、报纸上的文章还是新闻稿中的声明。
  </p>
  <p class="graf graf--p">
   👉开源情报还包括可以在不同类型的媒体中找到的信息。尽管人们通常认为它是基于文本的，但是图像、视频、网络研讨会、公开演讲和会议中的信息都属于开源情报。
  </p>
  <p class="graf graf--p">
   简单说，只要你能挖出来（而不是偷），它就属于你。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     开源情报干什么用？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   👉通过收集有关特定目标的公开信息源，攻击者（或友好的渗透测试人员）可以精准地描述潜在的受害者，以便更好地了解其特征、并缩小搜索范围，以寻找可能的漏洞。
   <strong class="markup--strong markup--p-strong">
    以此来制定可能最有效的攻击计划
   </strong>
   。
  </p>
  <p class="graf graf--p">
   攻击者无需主动与目标交战，就可以利用产生的情报来构建威胁模型和制定攻击计划。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    像军事攻击一样，有针对性的网络攻击同样始于侦察，而数字侦察的第一阶段是被动地获取情报而不会警告目标
   </strong>
   。
  </p>
  <p class="graf graf--p">
   👉将开源情报收集运用到您自己或您的组织上，是了解您为潜在的攻击者提供了哪些信息的好方法。也就是我们介绍过的 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%87%aa%e6%88%91%e4%ba%ba%e8%82%89/" href="https://www.iyouport.org/%e8%87%aa%e6%88%91%e4%ba%ba%e8%82%89/" rel="noopener noreferrer" target="_blank">
    自我人肉
   </a>
   ”。
  </p>
  <p class="graf graf--p">
   一旦您知道可以从公开来源收集到什么样的情报，就可以使用它来帮助您自己或您的团队制定更好的防御策略。
  </p>
  <p class="graf graf--p">
   您的公共信息暴露了哪些漏洞？攻击者可以从中了解到什么，他们可以利用其进行哪些
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
    社会工程攻击
   </a>
   或网络钓鱼攻击？
  </p>
  <p class="graf graf--p">
   帮自己回答这些问题。
  </p>

  <p class="graf graf--p">
   Twint 是用Python编写的 Twitter 抓取工具，可轻松
   <strong class="markup--strong markup--p-strong">
    匿名收集和寻找
   </strong>
   Twitter 上的信息，而无需注册 Twitter 服务本身、也不必使用API​​密钥，就像使用 Recon-ng 这样的工具一样。
  </p>
  <p class="graf graf--p">
   使用 Twint，完全不需要身份验证或API。只需安装该工具并开始搜寻即可。您可以按用户、地理位置和时间范围等进行搜索。这只是 Twint 的一些选项，也有许多其他选项可用。请参见上面的链接。

   那么，如何使用 Twint 帮助您跟上开源情报的发展呢？好吧，这很容易，并且是 Twint 在行动中的一个很好的例子。
  </p>
  <p class="graf graf--p">
   由于 Twint 允许您指定 -since，以便仅从特定日期开始提取推文，因此您可以将其与 Twint 的搜索词结合使用，比如每天抓取标有#OSINT的新推文。
  </p>
  <p class="graf graf--p">
   您可以使用保存为 SQLite 格式的 Twint 的 -database选项，自动执行该脚本并将结果输入数据库中，以方便您查看。
  </p>
