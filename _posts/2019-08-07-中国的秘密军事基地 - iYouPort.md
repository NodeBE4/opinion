  <p class="graf graf--p">
   随着北京加强与哈拉雷的军事合作，中国正准备在津巴布韦驻扎精锐特种部队，Spotlight Zimbabwe 曾报道，担心这个亚洲强国将在该国建立一个秘密的地下军事基地。
  </p>
  <p class="graf graf--p">
   新报告发布一年后，Spotlight Zimbabwe 现在透露，中国在该国安装下一代地对空导弹（SAM），与其在南中国海安置的导弹相同。
  </p>
  <p class="graf graf--p">
   中国新的军事基地将在全国范围内保护其大型钻石矿和金矿，其中一些 SAM 发射器已经位于该国。
  </p>
  <p class="graf graf--p">
   根据前领导人总统罗伯特穆加贝政府前部长的说法，中国自2014年以来一直计划派遣他们的特种部队到津巴布韦国民军（ZNA）“向他们提供技术援助和支持”。然而，几年前穆加贝在指责中国腐败和掠夺马兰吉矿产区的自然资源之后取消了这项计划。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     他们（中国）一直渴望在这个国家建立永久的军事存在，以保护他们在这里的巨大经济利益，但穆加贝一直在抵制这些提议
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    ，“这位前内阁部长说。
   </em>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “尽管表面上的论证是围绕向我们的武装部队提供技术援助和支持，但后来很明显，姆南加格瓦与中国有自己的协议和安排。
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     这激怒了穆加贝，也是在同一时期，姆南加瓦首次作为副总统前往中国，举行高层会议
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    ，而他的老板尚未完全了解情况。这一事件增加了穆加贝对 Mnangagwa 的政治不信任，他怀疑他正在向习近平展示自己，作为一个“最佳政治角色”确保中国对津巴布韦的投资。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    其余的都是历史。此后，Mnangagwa 还邀请中国回到马兰吉的钻石矿，他们的特种部队已获得了副总统 Rtd General，Constatino Chiwenga 的绿灯，现在有充分的理由相信，穆加贝2017年11月的下台，可能是因为中国将他的权力视为对其经济投资的威胁，特别是在剥夺了他们的钻石开采权之后
   </em>
   。
  </p>
  <p class="graf graf--p">
   津巴布韦在过去几年中已经看到数十亿美元的中国投资，主要是在采矿、农业和电信等关键经济领域。这些投资是“一带一路”倡议（BRI）的一部分，这是一项在跨大陆范围内改善区域经济的雄心勃勃的努力，
   <strong class="markup--strong markup--p-strong">
    它旨在加强中国与其他65个国家之间的基础设施，贸易和投资，这些国家共占全球国内生产总值（GDP）的30％，世界人口的62％和已知全球能源储备的75％。
   </strong>
  </p>
  <p class="graf graf--p">
   Emmerson Mnangagwa 总统上个月表示，BRI 是旧丝绸之路的重大改进。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “在过去，有丝绸之路，并且更大程度上没有包含整个大陆。津巴布韦的幸运只有在 800到1000年前，当我们进口瓷器时，Munhumutapa 王国和中国之间有贸易关系。
   </em>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “但今天的 BRI 倡议已经让所有人都参与其中，以便我们能够相互交流，从而使我们的经济能够互相帮助实现现代化和机械化。我们正在相互联系并互惠互利。
   </em>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “如果你看一下目前的 FOCAC 会议就能知道，我们将要解决10个问题，这些问题实际上是津巴布韦等发展中国家的主要问题。
   </em>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “运输问题、我们各国的基础设施发展问题……我们相信，在非洲中东部地区正在与中国进行对话的中非合作论坛可以帮助中国与津巴布韦和非洲加强这种关系。当这种情况发生时它会创造中国和非洲市场的整合，因此我们很高兴我们成为这一全球愿景的一部分，“他说。
   </em>
  </p>
  <p class="graf graf--p">
   中国已表示将继续投资非洲，通过非洲联盟2063年议程、2030年联合国可持续发展议程、以及非洲各国的发展战略。
  </p>
  <p class="graf graf--p">
   然而，
   <strong class="markup--strong markup--p-strong">
    在津巴布韦的情况下，中国的 BRI 投资之后是永久的军事存在
   </strong>
   。据称，现在将建造一个特种部队的秘密军事基地，这一举动肯定会激怒华盛顿。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    C
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.zerohedge.com/news/2019-04-08/chinas-special-forces-station-zimbabwe-build-secret-underground-base-protect" href="https://www.zerohedge.com/news/2019-04-08/chinas-special-forces-station-zimbabwe-build-secret-underground-base-protect" rel="noopener noreferrer" target="_blank">
    hina’s Special Forces To Station In Zimbabwe, Build Secret Underground Base To Protect Natural Resource Claims.
   </a>
   China’s new military base is set to protect its large diamond claims and gold mines across the country, where some of its SAM launchers are already located.
  </p>
