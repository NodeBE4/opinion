<div class="article">
 <div class="body-text">
  <p align="left">
  </p>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    20
   </span>
   <span style="font-size: 9pt">
    年前的茅老师那个时候还是年富力强，曙光老师
   </span>
   <span style="font-size: 9pt">
    20
   </span>
   <span style="font-size: 9pt">
    年前也是正当年，盛洪基本上是同龄人，可以说是小伙子。
   </span>
   <span style="font-size: 9pt">
    20
   </span>
   <span style="font-size: 9pt">
    年过去是非常不容易的。我个人想过去的
   </span>
   <span style="font-size: 9pt">
    20
   </span>
   <span style="font-size: 9pt">
    年是三个真实的人，用真实的方法来研究真实的中国问题，这是我对天则所
   </span>
   <span style="font-size: 9pt">
    20
   </span>
   <span style="font-size: 9pt">
    年走过的路的基本评价，真实的人，大家讲了许多关于茅老师、曙光老师、盛洪的故事。真实的方法，就是天则所一直坚持采用案例研究和调查研究。真实的问题，也就是始终关注中国的经济、社会和政治的过去、现在和将来。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    在这里，我只想讲一件事。就是我与天则所经历过的一个真实的事。在
   </span>
   <span style="font-size: 9pt">
    1995
   </span>
   <span style="font-size: 9pt">
    年，我和曙光、万中心合作承担了天则所的一个科研项目，题目是"中国贸易保护的代价"。我们用计量和实政方法研究了一个问题，以
   </span>
   <span style="font-size: 9pt">
    1994
   </span>
   <span style="font-size: 9pt">
    年为基期，中国贸易保护程度究竟是很高呢，还是中等的，还是较低呢。当时人们都假定说，
   </span>
   <span style="font-size: 9pt">
    1994
   </span>
   <span style="font-size: 9pt">
    年中国还是从计划经济向市场经济转型的攻坚阶段，人们都假定中国的贸易管制是非常严格的。那么，中国贸易保护壁垒的墙究竟有多高呢，比如说有
   </span>
   <span style="font-size: 9pt">
    2.2
   </span>
   <span style="font-size: 9pt">
    米
   </span>
   <span style="font-size: 9pt">
    高，这意味着世界上最强壮的狼也很难跨越
   </span>
   <span style="font-size: 9pt">
    2.2
   </span>
   <span style="font-size: 9pt">
    米
   </span>
   <span style="font-size: 9pt">
    高的墙进来吃中国的肥羊。中美关于这个墙究竟降到多高是合适的，谈了
   </span>
   <span style="font-size: 9pt">
    15
   </span>
   <span style="font-size: 9pt">
    年。比如说，中国认为要保护本国的企业，这个墙最好能在
   </span>
   <span style="font-size: 9pt">
    1.8
   </span>
   <span style="font-size: 9pt">
    米
   </span>
   <span style="font-size: 9pt">
    以上，只有美国最强壮的狼才能够翻过
   </span>
   <span style="font-size: 9pt">
    1.8
   </span>
   <span style="font-size: 9pt">
    米
   </span>
   <span style="font-size: 9pt">
    的墙进来吃中国的肥羊。美国说不行，
   </span>
   <span style="font-size: 9pt">
    1.8
   </span>
   <span style="font-size: 9pt">
    的墙太高了，贸易开放以后从中国得到的好处太少了，最好变成
   </span>
   <span style="font-size: 9pt">
    1.5
   </span>
   <span style="font-size: 9pt">
    或者
   </span>
   <span style="font-size: 9pt">
    1.2
   </span>
   <span style="font-size: 9pt">
    ，让中等的狼也能跳进来分享中国开放大餐。当时我们的研究经过美国最好的计量专家验证，我们发现这个墙有多高呢，这个墙是皇帝的新装，很矮，连小狼崽都可以迈过。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    当年，中国申请加入
   </span>
   <span style="font-size: 9pt">
    WTO
   </span>
   <span style="font-size: 9pt">
    的时候，我们有三句话，一是说加入
   </span>
   <span style="font-size: 9pt">
    WTO
   </span>
   <span style="font-size: 9pt">
    的目的是为了积极参与全球化，融入世界。二是中国加入
   </span>
   <span style="font-size: 9pt">
    WTO
   </span>
   <span style="font-size: 9pt">
    ，是为了在体制机制上与国际通行规则接轨。三是"狼来了"，门要开了，外面的饿狼要吃家里的肥羊。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    第一个问题是，入世以后实际上发生什么情况呢，我们期待的狼冲击没有发生，为什么？原因很简单，也就是在非洲大草原的动物世界案例告诉我们，只有在狼吃饱肥羊以后，他才会对肥羊无动于衷，也就是说，狼早就在圈里了。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    第二个问题是，既然开放的效益，早在
   </span>
   <span style="font-size: 9pt">
    1994
   </span>
   <span style="font-size: 9pt">
    年之前就释放了，为什么加入
   </span>
   <span style="font-size: 9pt">
    WTO
   </span>
   <span style="font-size: 9pt">
    ，中国取得了巨大的进步。关键是从
   </span>
   <span style="font-size: 9pt">
    1986
   </span>
   <span style="font-size: 9pt">
    年到
   </span>
   <span style="font-size: 9pt">
    2001
   </span>
   <span style="font-size: 9pt">
    年的改革使中国的体制机制，中国的战略，中国的结构发生巨大的变化。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    第三个问题是，真正的狼冲击会发生在什么时候，是过去的十年，还是未来的十年。现在看来，真正的狼冲击将发生在未来的十年。所以，中国要想应对未来十年的狼冲击，我们必须要加大体制的改革。这是在
   </span>
   <span style="font-size: 9pt">
    1995
   </span>
   <span style="font-size: 9pt">
    年，我们和天则所合作研究的真实故事。谢谢大家！
   </span>
