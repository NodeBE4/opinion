  <p class="p1">
   <span class="s1">
    这个世界与
   </span>
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    的关系已经从迷恋变为了被动攻击。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    人们对
   </span>
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    的情绪爱恨交织，经常能看到他们在
   </span>
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    上分享批评
   </span>
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    的帖子；以及愤怒驱动的注销但少卿便会回归。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    事实上，
   </span>
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    已经做到了让人们很容易错误地接受它存在的严重问题，不论是数据管理、用户权利、隐私人权、透明度、还是认知控制。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    五月底，
   </span>
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    的联合创始人克里斯休斯呼吁联邦贸易委员会与
   </span>
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    散伙。休斯指出，
   </span>
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    是一个事实上的垄断企业。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    他绝不是第一个说出实话的人。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    经济学人是另一个先行者，值得注意的是，它还指出了
    <b>
     垄断的真实本质
    </b>
    ，而不仅仅针对
   </span>
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    ：即
   </span>
   <span class="s1">
    <b>
     如果数据是新石油，
    </b>
   </span>
   <span class="s2">
    <b>
     BigTech
    </b>
   </span>
   <span class="s1">
    <b>
     拥有的则是石油钻井平台
    </b>
    。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    等数据驱动的产品首先获取数据，并用它来增强产品，以收集更多的数据。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    然而，正如
   </span>
   <span class="s2">
    ZDNet
   </span>
   <span class="s1">
    的
   </span>
   <span class="s2">
    Larry Dignan
   </span>
   <span class="s1">
    所指出的那样，联邦贸易委员会是否会将
   </span>
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    视为垄断是值得怀疑的。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    就算这种情况发生也需要一段时间；就算发生，散伙是否会起作用也是值得怀疑的
   </span>
   <span class="s2">
    ——
   </span>
   <span class="s1">
    正如互联网先驱
   </span>
   <span class="s2">
    Jaron Lanier
   </span>
   <span class="s1">
    所指出的那样，基于服务换取数据的商业模式中单独的打破总是会导致不对称。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    也就是说，就算
   </span>
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    本身被打破，基于相同模型的
   </span>
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    克隆体会接连出现，其结果可能没有太大区别。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    在演讲中
   </span>
   <span class="s2">
    Lanier
   </span>
   <span class="s1">
    反思了谷歌和
   </span>
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    等公司在数字文化基础上制造的
   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">
    全球性悲剧、令人震惊的荒谬错误
   </span>
   <span class="s2">
    ” ——
   </span>
   <span class="s1">
    以及我们如何改变这一窘境。他说：
   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">
    如果两个人希望沟通我们无法建立一个社会，唯一可能的方式，它是由希望操纵它们的第三人资助的。
   </span>
   <span class="s2">
    ”
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    在这里看到
   </span>
   <span class="s4">
    视频
   </span>
   <span class="s3">
    ：
    <a href="https://www.ted.com/talks/jaron_lanier_how_we_need_to_remake_the_internet?utm_campaign=tedspread&amp;utm_medium=referral&amp;utm_source=tedcomshare">
     <span class="s5">
      https://www.ted.com/talks/jaron_lanier_how_we_need_to_remake_the_internet?utm_campaign=tedspread&amp;utm_medium=referral&amp;utm_source=tedcomshare
     </span>
    </a>
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    要解决这个问题，需要结合使用技术、数据和社交工程学。
   </span>
  </p>
  <p class="p4">
   <span class="s3">
    <b>
     数据文化：支付数据的价格
    </b>
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    关于人们对数据收集的态度的调查是有说服力的。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    受访者希望获得更多隐私，但是不愿意为此付费。他们中的很大一部分人甚至愿意提供更多数据以换取便捷的免费服务。这已经足够说明了目前世界上的数据主权和意识的状况。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    尽管在个人层面上进行分析应该是令人担忧的，但这本身就是最重要的。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    <b>
     你的数据是为
    </b>
   </span>
   <span class="s2">
    <b>
     Facebook
    </b>
   </span>
   <span class="s1">
    <b>
     提供动力的石油。随着每一个滚动、点击、点赞、阅读，你一直不断地为它的垄断提供能源，你已经用你的手指投票支持它了
    </b>
    。虽然看起来是一种交易，但是，
    <b>
     即使您不使用
    </b>
   </span>
   <span class="s2">
    <b>
     Facebook
    </b>
   </span>
   <span class="s1">
    <b>
     、甚至没有拥有帐户，
    </b>
   </span>
   <span class="s2">
    <b>
     Facebook
    </b>
   </span>
   <span class="s1">
    <b>
     也同样在尽力获取您的数据
    </b>
    。
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    并且《
   </span>
   <a href="https://www.iyouport.org/facebook-%e7%bb%99%e4%ba%86%e5%85%a8%e7%90%83%e5%bd%93%e6%9d%83%e8%80%85%e4%b8%80%e4%b8%aa%e7%bb%88%e6%9e%81%e6%9a%b4%e6%94%bf%e8%93%9d%e5%9b%be/" rel="noopener noreferrer" target="_blank">
    <span class="s1">
     Facebook
    </span>
   </a>
   <span class="s3">
    <a href="https://www.iyouport.org/facebook-%e7%bb%99%e4%ba%86%e5%85%a8%e7%90%83%e5%bd%93%e6%9d%83%e8%80%85%e4%b8%80%e4%b8%aa%e7%bb%88%e6%9e%81%e6%9a%b4%e6%94%bf%e8%93%9d%e5%9b%be/" rel="noopener noreferrer" target="_blank">
     给了全球当权者一个终极暴政蓝图
    </a>
    》
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    通过其实践积累的数据量意味着它在数据财富竞争和
   </span>
   <span class="s2">
    AI
   </span>
   <span class="s1">
    竞争中占据了极大的优势。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    这个世界正在越来越多地通过机器学习算法来运行，数据训练这些算法的重要性是至关重要的。由于研究人员需要数据来使算法发挥作用，
   </span>
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    的数据库也吸引了大量人才。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    然而，这给我们带来了一个非常现实的问题。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    正如休斯指出的那样，在剑桥分析公司丑闻曝光后，对
   </span>
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    的强烈抗议已经愈演愈烈。
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    关联《
    <a href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" rel="noopener noreferrer" target="_blank">
     心理操纵的秘密：
    </a>
   </span>
   <a href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" rel="noopener noreferrer" target="_blank">
    <span class="s1">
     ”
    </span>
    <span class="s3">
     行为经济学
    </span>
    <span class="s1">
     ”+PR=
    </span>
   </a>
   <span class="s3">
    <a href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" rel="noopener noreferrer" target="_blank">
     ？
    </a>
    》
   </span>
  </p>
  <p class="p4">
   <span class="s3">
    <b>
     但是，
    </b>
   </span>
   <span class="s2">
    <b>
     #deletefacebook
    </b>
   </span>
   <span class="s3">
    <b>
     并未导致太多变化，因为缺乏可行的替代方案，意味着人们基本上只有两种选择：完全放弃社交网络，或继续使用
    </b>
   </span>
   <span class="s2">
    <b>
     Facebook
    </b>
   </span>
   <span class="s3">
    。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    但除了
   </span>
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    之外真的别无选择吗？
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    要回答这个问题，必须首先在功能方面分解
   </span>
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    确实汇集了强大的功能，它们都捆绑在一起，通过其后天的生态系统进一步扩展
   </span>
   <span class="s2">
    —— WhatsApp
   </span>
   <span class="s1">
    和
   </span>
   <span class="s2">
    Instagram
   </span>
   <span class="s1">
    。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    尽管很少有替代品能够以
   </span>
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    这样的无缝方式提供所有这些选择，但确实存在替代方案。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    WhatsApp
   </span>
   <span class="s1">
    曾经是其中之一，可惜它现在被
   </span>
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    吞并了；其他消息应用程序也有联系人管理和对群组和渠道的支持，其中一些，如
   </span>
   <span class="s2">
    Telegram
   </span>
   <span class="s1">
    、
   </span>
   <span class="s2">
    Mastodon
   </span>
   <span class="s1">
    或
   </span>
   <span class="s2">
    Diaspora
   </span>
   <span class="s1">
    ，也是开源的，它们使用户可以选择更多地控制他们的数据。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    但是，这些应用主要被已经脱离社交网络的人或边缘群体的人使用。那么，为什么它们没有赶上
   </span>
   <span class="s2">
    Facebook
   </span>
   <span class="s1">
    ？
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    碎片化是一个原因。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    如果您的某些联系人使用
   </span>
   <span class="s2">
    A
   </span>
   <span class="s1">
    应用程序，而其他人使用
   </span>
   <span class="s2">
    B
   </span>
   <span class="s1">
    应用程序，那么如何通过它们进行通信呢？
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    但这已经不再是一个问题
   </span>
   <span class="s1">
    ——
   </span>
   <span class="s3">
    如果您从
   </span>
   <span class="s1">
    Fediverse
   </span>
   <span class="s3">
    、
   </span>
   <span class="s1">
    Federation
   </span>
   <span class="s3">
    、或
   </span>
   <span class="s1">
    Activity Web
   </span>
   <span class="s3">
    中发布，就可以做到跨平台沟通。
   </span>
  </p>
  <p class="p3">

    Linked Data
   </span>
   <span class="s1">
    的技术。今天，开放数据集可能正在增长，但这些数据集的可用方式是不同的。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    让
   </span>
   <span class="s2">
    Solid
   </span>
   <span class="s1">
    生产就绪并能够在其上构建应用程序，在很大程度上取决于对开发者的吸引。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    吸引开发者是一门艺术。也需要有很多钱。我们不知道
   </span>
   <span class="s2">
    Solid
   </span>
   <span class="s1">
    是否具备所有这些条件，它是一个很好的起点，但并不一定能保证成功。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    不过只要想想将
   </span>
   <span class="s2">
    Fediverse
   </span>
   <span class="s1">
    和
   </span>
   <span class="s2">
    Solid
   </span>
   <span class="s1">
    放在一起就可以开辟的可能性。这是社交网络变革的关键，赶走中间的独裁者，实现数据主权和一个全新的创新生态系统。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    祈祷
   </span>
   <span class="s2">
    Solid
   </span>
   <span class="s1">
    很快就会变得坚实吧。
   </span>
  </p>
