    帮助改善战略以赢得成功

  </p>
  <p class="graf graf--p">
   每个环都标记了数字，从0到6 —— 其中0表示不存在影响或不存在给定属性，6表示普遍存在的影响或存在给定的属性。

  <p class="graf graf--p">
   该战略网可用于映射反抗运动及其对手的属性。然后可以比较两者之间的漏洞和弹性的战略差距。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    终结暴政的清单
   </strong>
  </p>
  <p class="graf graf--p">
   👌战略战术的选择对公民抵抗运动是最具有明显影响力的因素，活动家应该从中分析出那些具有共性的能力、技巧和选项 ——
   <strong class="markup--strong markup--p-strong">
    因为这些因素可以在各类抵抗运动中得到检验，是抵抗运动各种成功因素的共同点
   </strong>
   。
  </p>
  <p class="graf graf--p">
   所有因素中有3个是最关键的，另外3个是最具影响力的。也就是说，要想获得一场成功的运动，您就应该保证在您的战略中至少这6个因素处于高值。
  </p>
  <p class="graf graf--p">
   下面逐一分析一下。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、联合/联盟
   </strong>
  </p>
  <p class="graf graf--p">
   👉专制政府最擅长的是分化、打散，以瓦解反对派。因此，反对派就必须在联合和联盟的能力方面做得更好、更具有创造性。
  </p>
  <p class="graf graf--p">
   想要发展并维持联合局面需要多方面的努力，其中需要优先考虑的就是：提出一个为各方所共同接受的具有包容性的抵抗运动愿景。
  </p>
  <p class="graf graf--p">
   为此，运动领导人需要深入地了解民众的文化价值观，倾听其疾苦、人生志向等等，这样才能有效动员民众。
  </p>
  <p class="graf graf--p">
   👌愿景的有效性在于引起人们的共鸣，并能
   <strong class="markup--strong markup--p-strong">
    与个人经历和普通人的感情联系起来
   </strong>
   ，这样才能动员更多人一起投入公民抵抗运动。
  </p>
  <p class="graf graf--p">
   开创联盟局面的另一关键是领导人有能力保持运动组织结构的合理性。
  </p>
  <p class="graf graf--p">
   人们 *自愿* 参加抵抗运动，领导人不具有任何权威发布命令或实施任何形式的控制。这意味着任何抵抗运动的决策和执行必须使运动参与者感受到其合理性。
  </p>
  <p class="graf graf--p">
   每一个成功的公民抵抗运动都有不同的方式来做到这一点 — — 有些是逐级实施，有些是集中管理，有些兼而有之。不管运动的具体领导人是谁，或组织结构有何不同，领导人都要通过联合才能使之顺利运转。
  </p>
  <p class="graf graf--p">
   👉任何一位被推举的名义领袖或自然产生的全国性领袖，都要紧紧依靠各地区性领导人来建立联盟、进行谈判、实现不同群体的利益。这要求领导人有能力联盟来自不同地区、代表不同族群、具有不同规模（本地或全国性）的组织和群体。
  </p>
  <p class="graf graf--p">
   在80年代反对南非种族隔离运动中，致力于联盟的各方努力非常突出。当时全国各地数百个公民组织如雨后春笋，他们纷纷要求当地政府提供清洁用水和服务，同时，他们都有一个共同愿景，那就是结束种族隔离、实现民族和解。
  </p>
  <p class="graf graf--p">
   这些公民组织后来组成了 “团结民主联盟”，它分布各地的基层领导人展开了有效的本地抵抗（例如 抵制消费）战术，同时与更大规模的全国性组织行动相呼应、展开联动。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、精心计划战略、战术
   </strong>
  </p>
  <p class="graf graf--p">
   开展有效的公民抵抗运动的复杂性超出了人们的想象。
  </p>
  <p class="graf graf--p">
   通常人们能想到公民抵抗的形式就是上街，而事实上，这只是数百个抵抗战术之一。
  </p>
  <p class="graf graf--p">
   👉要展开最有效的抵抗运动就必须知道如何选择这些战术，在何时何地如何实施这些战术，由谁执行，目标是什么， 在和其它战术配合时如何安排先后顺序。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    要对这些问题胸有成竹， 就必须认清敌我双方存在优势、劣势、机会和威胁，同时评估冲突所在的环境，哪些是中立和旁观者（包括国际社会），因为这些因素对公民抵抗运动也会产生影响。
   </strong>
  </p>
  <p class="graf graf--p">
   👉掌握这些情况之后，运动组织者要提出一个有效的短期和中长期目标，以及对应的行动计划。
  </p>
  <p class="graf graf--p">
   当然这些计划会随着运动发展而变化（就像所有计划一样）。但是，制定计划、实施计划的过程和培养运动参与者的观念比计划本身至关重要。
  </p>
  <p class="graf graf--p">
   在极少数情况下，正是在制定计划的过程中清晰地看到抵抗运动的条件和能力交汇点。
  </p>
  <p class="graf graf--p">
   制定行动计划的基础就是对客观条件的评估，也就是分析抵抗运动所处的环境优劣。
   <strong class="markup--strong markup--p-strong">
    只有这样，抵抗运动才能最大程度的利用有利条件，努力克服、转化和绕开那些不利条件，这需要有做出战略选择的能力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   从波兰1980年代的团结工会运动中可以看到实施行动计划的战略。
  </p>
  <p class="graf graf--p">
   波兰工人清楚自己的优势和能力，因此提出了一个响亮、而且在政治上较为实际的要求：成立独立工会（而不是结束共产制度这种在当时不可能实现的口号）。
  </p>
  <p class="graf graf--p">
   遂即组织劳工运动，在复杂多样的工人群体中开展动员工作，以此应对当时政府面临的履步维艰的经济形势及其缺乏合法性的问题。
  </p>
  <p class="graf graf--p">
   团结工会选择了一个有效的罢工战术就是占领他们的工厂（
   <strong class="markup--strong markup--p-strong">
    而不是在工厂外示威抗议，从而降低了被镇压的威胁 — — 此前他们尝试过街头抗议并有失败的教训
   </strong>
   ）。
  </p>
  <p class="graf graf--p">
   工人们还用印发传单、小册子等其它手段使行动成效更加显著。其实印刷传单等手段在此前十几年里已经成为不同城市之间的工人进行沟通的有效途径。
  </p>
  <p class="graf graf--p">
   由于选择了切合实际的战略、目标和战术（即占领工厂和使用印刷品），扬长避短，才能更有效的在复杂的斗争环境中克服不利因素，稳扎稳打，最终在1989年实现了波兰民主化的目标。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、纪律
   </strong>
  </p>
  <p class="graf graf--p">
   维持非暴力纪律 — — 即使面临挑衅，领导者也要有能力保持非暴力 — — 是
   <strong class="markup--strong markup--p-strong">
    保持公民抵抗运动持久生命力的核心
   </strong>
   所在。
  </p>
  <p class="graf graf--p">
   只有保持非暴力，运动才能最大程度地动员广大民众参与，
   <strong class="markup--strong markup--p-strong">
    增加对手实施镇压的成本，使镇压带来更多反抗
   </strong>
   ，才更有可能导致暴政赖以生存的重要机构内部的反叛。
  </p>
  <p class="graf graf--p">
   很多研究表明，正是上述这些困素决定了非暴力运动成功率高于暴力运动的成功率。
  </p>
  <p class="graf graf--p">
   保持非暴力纪律要求抵抗运动相信公民抵抗是有效指导和推进抵抗运动的途径。有关公民抵抗运动的运作机制和历史记录证明了这一点。
  </p>
  <p class="graf graf--p">
   有效的战略可以为抵抗运动奠定胜利的基础，时间越长，效果越好，这正是公民抵抗运动的成功机制。同时需要建立一种文化和一套规范。
  </p>
  <p class="graf graf--p">
   例如上个世纪末塞尔维亚推翻了独裁者米洛舍维奇的抵抗组织“欧特珀”对新成员开展系统性培训，帮助他们理解公民抵抗的机制以及保持非暴力的重要性。
  </p>
  <p class="graf graf--p">
   这些工作并不要求运动就其非暴力的道德是非进行辩论，但参与者必须深入领会非暴力纪律带来的实际效果，同时要汲取一些案例带来的教训（如叙利亚发生的导致悲剧性结果的案例） — — 因为对非暴力公民抵抗缺乏耐心或信心转而付诸暴力，从而带来灾难性的结果。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、扩大民众参与
   </strong>
  </p>
  <p class="graf graf--p">
   公民抵抗运动若能极大地调动普通民众参与是一个抵抗运动取得成功的晴雨表（尽管也有人提出异议）。
  </p>
  <p class="graf graf--p">
   因为
   <strong class="markup--strong markup--p-strong">
    越多普通民众拒不服从独裁政府，就越能削弱独裁政府的统治，增加暴政试图恢复其控制的代价
   </strong>
   。 有具体数据表明，民众参与程度越高，镇压造成的反叛越多，即使在暴政的支持者中也是这样。
  </p>
  <p class="graf graf--p">
   2011年发生在埃及和突尼斯的革命赢得各界群众的极大支持，城乡各地男女老幼，教徒和世俗阶层，高收入和中低收入阶层的民众纷纷加入革命行列，再次证明了民众高度参与的重要性。
  </p>
  <p class="graf graf--p">
   与此相反，1989年发生在中国的学生运动和2009年开始的绿色运动虽然动员了数百万民众上街，各大媒体头条竞相报道，👉
   <strong class="markup--strong markup--p-strong">
    但是，两者均未能取得预计的目标，其中部分原因就是这两次运动后来都没有更广泛的民众参与，只局限于最初发起的那部分人群。
   </strong>
  </p>
  <p class="graf graf--p">
   具有广泛联合力量的愿景有助于形成内在合力，并能鼓舞更多民众参加运动。
  </p>
  <p class="graf graf--p">
   👉制定运动战略有助于坚定信心。设计各种战术时要考虑到参加者各自不同的需求，包括抵御风险的能力、参与运动的时间，为运动做出牺牲的决心。
  </p>
  <p class="graf graf--p">
   👌有效的战略可以为各个群体（无论贫富、年龄和阶层的差异）提供参与运动所需要的选择，使人们可以用不同的方式支持运动。
  </p>
  <p class="graf graf--p">
   暴力武装抵抗只有身强力壮的人才能参与，而非暴力运动可以吸引社会各阶层的参与 —— 于是胜算更大。
   <strong class="markup--strong markup--p-strong">
    非暴力是一种获胜技巧，而非什么道德信仰
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、削弱镇压的能力、增进反叛的能力
   </strong>
  </p>
  <p class="graf graf--p">
   暴政最有效的工具就是镇压。要进行有效抵抗就必须学会如何削弱镇压的效果，增加当权者镇压的成本。
  </p>
  <p class="graf graf--p">
   有一个具体方法就是精确评估风险并选择有效战术，因为各种战术带来的镇压风险不同。
  </p>
  <p class="graf graf--p">
   比如大规模群众示威之类的集中行动招致镇压的可能性就更大，而消费抵制、居家罢工、装病罢课， 或在公共场合张贴小型匿名传单等等，政府镇压的难度就会更大 —— 是难度更大，而非不可能。
  </p>
  <p class="graf graf--p">
   因为是否参与了这类战术比较难以判断（例如凭直观看不出某人是否抵制了某类消费），这会削弱镇压能力，甚至使镇压无能为力（例如居家罢工就需要警察到每个参与罢工的工人家里去调查）。
  </p>
  <p class="graf graf--p">
   1983年智利独裁者皮诺切特的政治反对派就必须设法找到避免镇压的办法。经历长达10年的大规模逮捕、杀戮、酷刑和强迫失踪，人们对暴政心怀恐惧。
  </p>
  <p class="graf graf--p">
   当年4月，铜矿工人号召在圣地亚哥郊外进行罢工，而皮诺切特则威胁要派军队在罢工开始前包围矿山。
  </p>
  <p class="graf graf--p">
   面对这种形势，工会领袖决定停止罢工， 取而代之的是在全国举行一天的怠工、慢步和慢驶行动，继而在当晚8时全国敲响锅碗瓢勺进行抗议。
  </p>
  <p class="graf graf--p">
   这次抗议得到了史无前例的大规模民众的参与，由此开始例行每月一次的此类抗议活动。
  </p>
  <p class="graf graf--p">
   这是在对付皮诺切特制造恐怖、分化民众、实施独裁统治中取得的第一个突破。
  </p>
  <p class="graf graf--p">
   参加这类行动带来的风险是可以接受的，并且不至于招来严重的镇压 —— 于是它更容易联盟更多人。
  </p>
  <p class="graf graf--p">
   这一战术的结果是，皮诺切特的安全部队对如此大规模的分散行动感到束手无策。
  </p>
  <p class="graf graf--p">
   其它削弱镇压效果、增加镇压成本，进而让镇压行动招致更多抵抗的方法包括：利用民众的不满情绪提出非政治性的要求。
  </p>
  <p class="graf graf--p">
   👉建立明确的领导人继任制度（一人被捕另一人替补），加强运动参与者与国内外各团体的沟通，使暴政镇压导致更多反抗。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    6、策反
   </strong>
  </p>
  <p class="graf graf--p">
   随着公民抵抗运动的发展，统治集体内部积极和消极的支持者出现分化和反叛。
  </p>
  <p class="graf graf--p">
   例如，政府内部的改革派和强硬派争夺控制权的争斗表面化，而经济利益则迫使国家政权不得不满足抵抗运动的要求，以换取经济活动的恢复。
  </p>
  <p class="graf graf--p">
   政府公职人员，包括士兵、警察、公务员已经开始改变态度，进而支持抵抗运动，因为抵抗运动提出的要求能够说服他们，他们也讨厌独裁者，或者他们的亲朋好友都参加了公民抵抗运动。
  </p>
  <p class="graf graf--p">
   支持政府的精英也开始怀疑政权的可持续性，他们采取中立立场，以免站错队，因为公民抵抗运动胜利在望。
  </p>
  <p class="graf graf--p">
   2004年，乌克兰“橙色革命”期间，异见人士说服前退役军官作牵线人与安全部队沟通。
  </p>
  <p class="graf graf--p">
   同样情景也发生在米洛舍维奇统治下的塞尔维亚。塞尔维亚的安全部队并没有公开反叛，只是不积极执行当局的命令。
  </p>
  <p class="graf graf--p">
   在塞尔维亚抵抗运动的高潮时，军人拒绝执行向贝尔格莱德大街上的示威群众空投燃烧弹的命令，理由是天气不好，在直升机上无法辨认示威群众，其实当天万里晴空。这名军人后来回忆说，当天自己的家人可能就在示威群众中，所以他无法执行命令。
  </p>
  <p class="graf graf--p">
   反叛者无论是被说服的还是出于一己私利，常常会受到公民抵抗运动的长期影响。
  </p>
  <p class="graf graf--p">
   这个趋势和前两个一样，都源自抵抗运动联合、精心计划、和坚持非暴力原则。
  </p>
  <p class="graf graf--p">
   研究数据表明，广大民众的参与程度越高，产生反叛的机会越大；同时，联合协作、精心计划和坚持非暴力更能够增加抵抗运动参与的广度和深度。
  </p>
