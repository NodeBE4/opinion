      </strong>
     </em>

  <p class="graf graf--p">
   Google 以21亿美元收购了健康追踪设备公司 Fitbit，这引发了人们对该技术巨头如何使用用户数据的担忧。
  </p>
  <p class="graf graf--p">
   谷歌似乎预料到了这种担忧，他们在公告中宣称，“隐私和安全对公司至关重要”。Google承诺对其收集的数据保持透明，“绝不将个人信息出售给任何人”。谷歌还称为用户提供 “查看、转移或删除其数据” 的选择。
  </p>
  <p class="graf graf--p">
   有超过
   <strong class="markup--strong markup--p-strong">
    2700万活跃用户
   </strong>
   的 Fitbit 回应了该消息。该公司在新闻稿中说：“自第一天起，严格的隐私和安全准则就已经成为 Fitbit DNA 的一部分，而且不会改变。”
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    数据隐私专家警告说，当前的法律和法规并不能使 Google 和其他寡头公司信守承诺。
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e9%9a%90%e7%a7%81%e6%b3%95%e6%a1%88%e7%ae%a1%e4%b8%8d%e4%bd%8f-facebook-%e5%92%8c-google%ef%bc%8c%e6%80%8e%e4%b9%88%e5%8a%9e/" href="https://www.iyouport.org/%e9%9a%90%e7%a7%81%e6%b3%95%e6%a1%88%e7%ae%a1%e4%b8%8d%e4%bd%8f-facebook-%e5%92%8c-google%ef%bc%8c%e6%80%8e%e4%b9%88%e5%8a%9e/" rel="noopener noreferrer" target="_blank">
     隐私法案管不住 Facebook 和 Google
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bb%a5%e9%9a%90%e7%a7%81%e4%b8%ba%e7%9b%ae%e7%9a%84%e7%9a%84%e6%96%97%e4%ba%89%e5%a6%82%e4%bd%95%e8%ae%a9%e4%bd%a0%e5%8f%98%e6%88%90%e4%ba%86%e7%9b%91%e8%a7%86%e7%9a%84%e7%9b%9f%e5%8f%8b/" href="https://www.iyouport.org/%e4%bb%a5%e9%9a%90%e7%a7%81%e4%b8%ba%e7%9b%ae%e7%9a%84%e7%9a%84%e6%96%97%e4%ba%89%e5%a6%82%e4%bd%95%e8%ae%a9%e4%bd%a0%e5%8f%98%e6%88%90%e4%ba%86%e7%9b%91%e8%a7%86%e7%9a%84%e7%9b%9f%e5%8f%8b/" rel="noopener noreferrer" target="_blank">
     以隐私为目的的斗争如何让你变成了监视的盟友
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    寡头公司如何将健康数据用于获利
   </strong>
  </p>
  <p class="graf graf--p">
   位于旧金山的 Fitbit 公司生产可穿戴设备，跟踪用户走的步数、心率和睡眠质量、以及其他个人健康指标。
  </p>
  <p class="graf graf--p">
   通过收购 Fitbit，Google 成功扩大了其已经收集的大量个人数据。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e7%9a%84%e7%9c%9f%e9%9d%a2%e7%9b%ae%e2%80%8a-%e2%80%8a-theyre-always-watching/" href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e7%9a%84%e7%9c%9f%e9%9d%a2%e7%9b%ae%e2%80%8a-%e2%80%8a-theyre-always-watching/" rel="noopener noreferrer" target="_blank">
     谷歌的真面目 — — They’re always watching
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   在过去的15年中，Google 购买了200多家公司，包括 YouTube、Waze 和 Nest Labs（一家制造智能家居设备的公司）。
  </p>
  <p class="graf graf--p">
   这些收购活动 —— 除了其内部产品（例如 Google Maps 和 Google Search）以及在美国使用的超过124,000台 Android设备 —— 为Google提供了大量可从中获利的个人数据。
  </p>
  <p class="graf graf--p">
   想一想在Google上搜索一双鞋子之后，您可能会在各种网站上收到关于鞋子的针对性广告。因为它一直在跟踪你。
  </p>
  <p class="graf graf--p">
   尽管谷歌表示不会将 Fitbit 数据用于广告，但隐私权倡导者警告，谷歌及其第三方合作伙伴
   <strong class="markup--strong markup--p-strong">
    还有很多其他可以利用数据的方式
   </strong>
   。
  </p>
  <p class="graf graf--p">
   非营利性倡导组织“患者隐私权”的创始人 Deborah Peel 博士说：
   <strong class="markup--strong markup--p-strong">
    “健康数据是最值钱的东西。”
   </strong>
  </p>
  <p class="graf graf--p">
   制药公司可以使用这些数据来决定他们应该在哪些疾病研究中投入资源以开发有利润的新药。购买数据的第三方企业还可以推断基于健康的行为并相应地销售其产品。
  </p>
  <p class="graf graf--p">
   Fitbit 的隐私权政策禁止共享任何可识别的信息，他们声称防止诸如保险公司拒绝患者健康保险或向患者收取更高的保险费之类的事。
  </p>
  <p class="graf graf--p">
   但是研究表明，
   <strong class="markup--strong markup--p-strong">
    即使是匿名信息也可以轻松地被“重新识别”。
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%8d%b3%e4%bd%bf%e6%95%b0%e6%8d%ae%e6%98%af%e5%8c%bf%e5%90%8d%e7%9a%84%ef%bc%8c%e6%82%a8%e7%9a%84%e8%ba%ab%e4%bb%bd%e4%bb%8d%e7%84%b6%e5%8f%af%e4%bb%a5%e8%a2%ab%e8%af%86%e5%88%ab/" href="https://www.iyouport.org/%e5%8d%b3%e4%bd%bf%e6%95%b0%e6%8d%ae%e6%98%af%e5%8c%bf%e5%90%8d%e7%9a%84%ef%bc%8c%e6%82%a8%e7%9a%84%e8%ba%ab%e4%bb%bd%e4%bb%8d%e7%84%b6%e5%8f%af%e4%bb%a5%e8%a2%ab%e8%af%86%e5%88%ab/" rel="noopener noreferrer" target="_blank">
     即使数据是匿名的，您的身份仍然可以被识别
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   重新识别的威胁已导致隐私权倡导者质疑创建健康应用程序的公司的动机，
   <strong class="markup--strong markup--p-strong">
    这些公司尚未被证明可以改善健康状况。那么他们为了什么而创立？
   </strong>
  </p>
  <p class="graf graf--p">
   苹果自己的健康应用程序在2014年推出时在 Apple 和 Fitbit 之间引起了裂痕，它还通过 iPhone 和 Apple Watch 收集了有关用户健康的大量数据，但该公司表示该数据是加密的，不用于商业目的。
  </p>
  <p class="graf graf--p">
   但是，在iPhone上使用 Fitbit 之类的应用程序的人仍需遵守这些公司的隐私权政策（而不是Apple的隐私权政策），该政策允许数据共享。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Fitbit 用户如何改变隐私政策
   </strong>
  </p>
  <p class="graf graf--p">
   联邦法规禁止Google获取 Fitbit 数据并以健康跟踪公司的隐私权政策未概述的方式使用它们。如果 Google 编辑了隐私权政策，则需要要求用户 “点击允许” 新的更改。
  </p>
  <p class="graf graf--p">
   当然，已经购买了 Fitbit 但又不喜欢新的隐私权政策的消费者的钱就白花了。（Fitbit手表零售价为199美元）
  </p>
  <p class="graf graf--p">
   消费者报告资深政策顾问 Dena Mendelsohn 表示：“在某个时候，科技公司似乎在说，
   <strong class="markup--strong markup--p-strong">
    ‘如果您对此有疑问，那就不要使用它。
   </strong>
   ”
  </p>
  <p class="graf graf--p">
   隐私权专家和业务观察者表示，Google不太可能大刀阔斧地改变 Fitbit 的隐私权政策，因为它不想失去其核心客户群。如果政策确实发生变化，Google仍可以允许其客户在“同意”的同时选择退出某些数据收集和共享做法。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    法律怎么总是跟不上技术
   </strong>
  </p>
  <p class="graf graf--p">
   消费者报告、患者隐私权和其他权利团体都表示，在数据收集和共享方面，科技公司拥有太多的回旋余地。他们主张制定更严格的联邦隐私法。
  </p>
  <p class="graf graf--p">
   Mendelsohn 表示：“如果没有基准法律，消费者就必须依赖其所参与的任何公司的政策惯例。”
  </p>
  <p class="graf graf--p">
   非营利组织“未来隐私论坛”（Future of Privacy Forum）的政策负责人 John Verdi 说，联邦法律对通过应用程序收集的健康数据的限制仍然少于对医院使用的医疗设备的限制。
  </p>
  <p class="graf graf--p">
   如果公司违反自己的隐私政策或以有害方式使用数据，则联邦贸易委员会可以对其处以罚款。美国联邦贸易委员会（FTC）今年早些时候因滥用用户的个人信息而对 Facebook 处以创纪录的50亿美元罚款 —— 尽管与该公司因滥用该服务而可能赚到的钱相比，
   <strong class="markup--strong markup--p-strong">
    该罚款的数字简直太毛毛雨了。
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/50%e4%ba%bf%e7%be%8e%e5%85%83%e7%9a%84%e6%83%a9%e7%bd%9a%e5%8f%aa%e8%83%bd%e4%bb%a4-facebook-%e6%ac%a3%e5%96%9c%e8%8b%a5%e7%8b%82/" href="https://www.iyouport.org/50%e4%ba%bf%e7%be%8e%e5%85%83%e7%9a%84%e6%83%a9%e7%bd%9a%e5%8f%aa%e8%83%bd%e4%bb%a4-facebook-%e6%ac%a3%e5%96%9c%e8%8b%a5%e7%8b%82/" rel="noopener noreferrer" target="_blank">
     50亿美元的惩罚只能令 Facebook 欣喜若狂
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   Google 也正在被调查潜在的违反反托拉斯法和可能侵犯消费者隐私的行为。
  </p>
  <p class="graf graf--p">
   许多立法者对 Google 收购 Fitbit 作出了回应，呼吁对现有隐私法规进行更严格的执行。但是，
   <strong class="markup--strong markup--p-strong">
    迄今为止，立法者之间尚未就如何最好地应对迅速变化的技术前景达成共识。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “立法本质上是一刀切的”，戴维斯·赖特·特赖恩（Davis Wright Tremaine）律师事务所的合伙人、联邦贸易委员会的前隐私事务官员 Helen Goff Foster 说，“但是，如何提出在发布之日尚未过时的正确法规？在隐私方面，我们还没有弄清楚。” ⚪️
  </p>
