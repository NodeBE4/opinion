  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    举报人：Google 正在开发 “上帝一般” 的人工智能
   </strong>
  </h3>
  <p class="graf graf--p">
   谷歌前高级软件开发商和现在的举报人 Zach Vorhies 出席了 SiriusXM 的晚间新闻节目，讨论谷歌与共产中国以及该公司的人工智能项目 DeepMind 的关系，该项目被称为 “AI Manhattan Project”。
  </p>
  <p class="graf graf--p">
   主持人：过去，谷歌因与中国共产党政府开展业务而遇到麻烦。我想问一下您在谷歌公司任职期间正在进行哪些工作？您能给我们提供任何见解吗？
  </p>
  <p class="graf graf--p">
   Zach Vorhies：Google 收购了一家名为 DeepMind 的AI公司，让我用通俗易懂的语言来表达，即
   <strong class="markup--strong markup--p-strong">
    DeepMind 正在创建一个类似于上帝的AI系统，该系统能够吸收整个互联网上所有可用公共信息，并使之形成情报
   </strong>
   。
  </p>
  <p class="graf graf--p">
   可以将其比做 Amazon Alexa，但要高级得多。我相信该AI系统已移出英国，并进入了中国，并被称为 AI Manhattan Project。
  </p>
  <p class="graf graf--p">
   Vorhies ：“著名企业家 Peter Thiel 指责 Google 叛逆。
   <strong class="markup--strong markup--p-strong">
    这是由美国公司制造并放置在中国的世界上最复杂、最先进的AI项目
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   Vorhies：当人们想到监视时，通常都会想到国家监视。这是对表面网络上可用的所有信息以及从深层网络中可获取的所有信息的全面监视，并利用 Google 对扫描内容的广泛深入分析。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    将所有数据融合在一起，并据此形成决策过程 —— 诱导、欺骗、操纵、抓捕人们。本质上，这就是 Google DeepMind 项目的目的
   </strong>
   。
  </p>
