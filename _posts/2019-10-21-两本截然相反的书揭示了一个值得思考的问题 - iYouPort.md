  <p class="graf graf--p">
   这里有两本书，形成了鲜明的对比。这一对比提醒了一件事 —— 在人们的恐惧和愤怒中被忽略的事 —— 关于，责骂和羞辱真的有用吗？如果“解决方案”只是加强管制，其结果是什么？
  </p>
  <p class="graf graf--p">
   在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E5%A6%82%E6%9E%9C%E5%85%A8%E7%BD%91%E5%B0%81%E9%94%81%E5%91%A8%E5%B0%8F%E5%B9%B3%EF%BC%9F%E4%B8%80%E4%B8%AA%E8%88%86%E8%AE%BA%E7%83%AD%E7%82%B9%E6%88%96%E8%AE%B8%E8%83%BD%E5%B8%AE%E5%8A%A9%E4%B8%AD/" href="https://www.iyouport.org/%E5%A6%82%E6%9E%9C%E5%85%A8%E7%BD%91%E5%B0%81%E9%94%81%E5%91%A8%E5%B0%8F%E5%B9%B3%EF%BC%9F%E4%B8%80%E4%B8%AA%E8%88%86%E8%AE%BA%E7%83%AD%E7%82%B9%E6%88%96%E8%AE%B8%E8%83%BD%E5%B8%AE%E5%8A%A9%E4%B8%AD/" rel="noopener noreferrer" target="_blank">
    Alex Jones 被全网封杀
   </a>
   的时候一位记者提出过这样的问题，“既然 Jones 以阴谋论家著称，这说明大多数人不信任他所说的；那么将他称之为‘危害’的原因是什么？并且，封杀真的能解决阴谋论的流行吗？”
  </p>
  <p class="graf graf--p">
   这正是这本书希望解释的问题……
  </p>
  <p>

  </p>
  <p class="graf graf--p">
   六年前，“meme culture”这个短语只能让人想起比如嘲讽脸和悲伤蛙，“感觉很糟糕的人”。
  </p>
  <p class="graf graf--p">
   这些角色是
   <strong class="markup--strong markup--p-strong">
    新兴数字民俗的面孔
   </strong>
   。
  </p>
  <p class="graf graf--p">
   自那时以来，已经有了许多变化。
  </p>
  <p class="graf graf--p">
   那只青蛙被命名为佩佩，并且在2015年，被美国未来的总统放进了推文。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5856 jetpack-lazy-image" data-lazy-sizes="(max-width: 611px) 100vw, 611px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-8.png?resize=611%2C860&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-8.png?w=611&amp;ssl=1 611w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-8.png?resize=213%2C300&amp;ssl=1 213w" data-recalc-dims="1" height="860" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-8.png?resize=611%2C860&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="611"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5856" data-recalc-dims="1" height="860" sizes="(max-width: 611px) 100vw, 611px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-8.png?resize=611%2C860&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-8.png?w=611&amp;ssl=1 611w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-8.png?resize=213%2C300&amp;ssl=1 213w" width="611"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   此后南方贫困法律中心花了不到一年的时间将 Pepe 这只青蛙
   <a class="markup--anchor markup--p-anchor" data-href="https://www.splcenter.org/hatewatch/2016/09/28/pepe-joins-echoes-new-hate-symbols" href="https://www.splcenter.org/hatewatch/2016/09/28/pepe-joins-echoes-new-hate-symbols" rel="noopener noreferrer" target="_blank">
    归类为
   </a>
   仇恨的象征，被白人至上主义者用作吉祥物的东西。
  </p>
  <p class="graf graf--p">
   与此同时，希拉里克林顿的竞选活动发表了
   <a class="markup--anchor markup--p-anchor" data-href="https://qz.com/780663/hillary-clintons-website-now-has-an-explainer-about-pepe-the-frog-a-white-supremacist-symbol/" href="https://qz.com/780663/hillary-clintons-website-now-has-an-explainer-about-pepe-the-frog-a-white-supremacist-symbol/" rel="noopener noreferrer" target="_blank">
    一个郑重其事的解释
   </a>
   ：“那只卡通青蛙比你想象的还要险恶！”
  </p>
  <p class="graf graf--p">
   在2000年创造了佩佩仅仅是作为一个玩偶形象的艺术家马特·弗里（Matt Furie）被迫采取行动。
  </p>
  <p class="graf graf--p">
   当新纳粹分子于2017年8月在弗吉尼亚州的夏洛茨维尔加入游行时，佩佩被正式杀死了，被埋葬在最后的一页漫画中。
  </p>
  <p class="graf graf--p">
   Pepe 跌宕的兴衰故事正是过去十年来公众对互联网的看法。
  </p>
  <p class="graf graf--p">
   互联网曾经被称为文化平台，尽管如今充满了trolls和犯罪、吸引了从民主被腐蚀的自由派道德恐慌到关于 meme 叛乱力量的右翼崛起的几乎一切。
  </p>
  <p class="graf graf--p">
   在特朗普就职的前一天，一位共和党选民
   <a class="markup--anchor markup--p-anchor" data-href="https://www.thisamericanlife.org/608/transcript" href="https://www.thisamericanlife.org/608/transcript" rel="noopener noreferrer" target="_blank">
    说道
   </a>
   ：”我们做到了。我们让他当上了总统。”
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5857 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/特朗普.png?resize=1100%2C732&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/特朗普.png?w=1106&amp;ssl=1 1106w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/特朗普.png?resize=300%2C200&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/特朗普.png?resize=768%2C511&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/特朗普.png?resize=1024%2C681&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/特朗普.png?resize=1100%2C732&amp;ssl=1 1100w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/特朗普.png?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="732" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/特朗普.png?resize=1100%2C732&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   哈佛大学伯克曼克莱因互联网与社会研究中心的研究员 An Xiao Mina 的新书《Memes to Movements》试图描绘一种互联网的政治经济。
  </p>
  <p class="graf graf--p">
   事实证明，专注于 meme 是一种特别清晰的论述方式。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    对于在线政治宣传、虚假新闻、和回声壁，人们只是一片恐慌。这些担忧只能使普通互联网用户成为无能为力的受害者
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是，Mina 建议我们应该停止将用户视为棋子。
  </p>
  <p class="graf graf--p">
   以假新闻为例。
  </p>
  <p class="graf graf--p">
   在2016年的一个普遍幻想中，俄罗斯特工、clickbait 网站、和 Facebook 的邪恶商业模式被认为将假新闻推向了无助的未受过训练的人群，并由此损害了美国的民主。
  </p>
  <p class="graf graf--p">
   Mina 拒绝这种屈尊俯就的态度。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “将假新闻视为一系列谎言，这的确很有吸引力”。但是，这会让那些分享假消息的人只是被视为愚蠢。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当人们分享假新闻时，他们不只是在宣称自己信任什么立场是什么；一个虚假的新闻帖子，就像个 meme，还有助于传达政治忠诚、引起对问题的关注、以及“表现出政治认同”。
   </strong>
  </p>
  <p class="graf graf--p">
   Mina 认为，
   <strong class="markup--strong markup--p-strong">
    事实检查无助于解决人们决定分享谎言的根本性原因。
   </strong>
  </p>
  <p class="graf graf--p">
   这点的确是关键。而且大多数人不愿意承认。
  </p>
  <p class="graf graf--p">
   Memes to Movements 这本书中重复出现的焦点就是 meme，而不是在线新闻或 Twitter 上的辩论。这是作者的聪明之举。
  </p>
  <p class="graf graf--p">
   您可以对 Tweet 或文章进行实际检查，但任何关于meme“准确性”的讨论都不是重要的。相反，我们应该问一下：meme对共享它的人做了什么。
  </p>
  <p class="graf graf--p">
   Mina 对“meme”的定义比通常的应用范围更广泛，
   <strong class="markup--strong markup--p-strong">
    这有助于思考在线和离线的各种媒体。
   </strong>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5858 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/书.png?resize=1100%2C617&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/书.png?w=1306&amp;ssl=1 1306w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/书.png?resize=300%2C168&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/书.png?resize=768%2C430&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/书.png?resize=1024%2C574&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/书.png?resize=1100%2C617&amp;ssl=1 1100w" data-recalc-dims="1" height="617" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/书.png?resize=1100%2C617&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5858" data-recalc-dims="1" height="617" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/书.png?resize=1100%2C617&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/书.png?w=1306&amp;ssl=1 1306w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/书.png?resize=300%2C168&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/书.png?resize=768%2C430&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/书.png?resize=1024%2C574&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/书.png?resize=1100%2C617&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    meme 的流行如此之快是因为人们乐于传播它们，而且这样做的理由经常反映出线下的政治现实
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Mina 对互联网文化基层移情式的关切是这本书最大的优势。
  </p>
  <p class="graf graf--p">
   在大量案例研究中，该书展示了 meme 之所以引起人们关注的原因 —— 建立政治叙事和创建数字社区。Mina 也花了一些时间证明
   <strong class="markup--strong markup--p-strong">
    幽默和严肃性是密切相关的，在线文化渗透到物质世界，meme 可以挑战权力也可以加强权力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在 Pepe 这只青蛙都能成为一个严肃的政治问题的世界里，这一切都不令人惊讶。
  </p>
  <p class="graf graf--p">
   这本书的偶尔天真可能是对过去两年来关于互联网政治潜力的悲观情绪的有益纠正。
  </p>
  <p class="graf graf--p">
   虽然 Mina 展示了 meme 如何能够为强大的利益服务，但她从
   <strong class="markup--strong markup--p-strong">
    不会对民间 memes 挑战等级制度并赋予无助者以力量的能力感觉兴奋
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这种谨慎的乐观态度与最近出版的另一本关于互联网的书《LikeWar：社交媒体的武器化》形成鲜明对比。这本书的作者是 P. W. Singer 和 Emerson T. Brooking。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5859 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/另一本书.png?resize=1100%2C745&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/另一本书.png?w=1114&amp;ssl=1 1114w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/另一本书.png?resize=300%2C203&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/另一本书.png?resize=768%2C520&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/另一本书.png?resize=1024%2C693&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/另一本书.png?resize=1100%2C745&amp;ssl=1 1100w" data-recalc-dims="1" height="745" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/另一本书.png?resize=1100%2C745&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5859" data-recalc-dims="1" height="745" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/另一本书.png?resize=1100%2C745&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/另一本书.png?w=1114&amp;ssl=1 1114w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/另一本书.png?resize=300%2C203&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/另一本书.png?resize=768%2C520&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/另一本书.png?resize=1024%2C693&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/另一本书.png?resize=1100%2C745&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   这本书的作者决心让人们相信，他们称之为“有毒沼泽”的互联网是一个险恶的战场，黑暗的政治力量在这个战场上掠夺这个民主社会的脆弱性。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “互联网现在只是一个信息本身被武器化的战场”，书中写道。
  </p>
  <p class="graf graf--p">
   以下是作者称其已成为“武器化”的不完整列表：“病毒式营销”、“社交媒体提供的个性化”、“一本正经的谎言”、“智能手机”、“meme”和“我们自己的大脑”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    并不是说这些东西不能成为武器。肯定能。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    但是，把一切都当成武器的渴望听起来就像是一只战鼓、它自己的宣传形式。
   </strong>
  </p>
  <p class="graf graf--p">
   当然，一旦你确定了对社会的威胁，下一步就是提供补救措施。
  </p>
  <p class="graf graf--p">
   《Memes to Movements》以充满希望的音符结束，将 memes 称为“我们时代的强大森林”；《LikeWar》的作者则提出了一系列令人不寒而栗的建议。
  </p>
  <p class="graf graf--p">
   他们希望美国通过建立一个控制中心来打击错误信息的“病毒式爆发”，从而对其本国公民进行宣传：
   <strong class="markup--strong markup--p-strong">
    分享虚假信息的人应该“被看作是教唆犯、是企图伤害整个社会的敌人”。
   </strong>
  </p>
  <p class="graf graf--p">
   LikeWar 所做的正是 Memes to Movements 试图避免的，即 将每种新形式的数字争议视为一种威胁性的新奇事物。
  </p>
  <p class="graf graf--p">
   Mina 合理地将 meme 战视为人类历史和权力斗争数千年的产物。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    互联网可能是改变游戏规则的东西，但这个游戏本身一直存在
   </strong>
   。⚪️
  </p>
  <p class="graf graf--p">
   **Alex Jones 之所以流行并非源于“阴谋论在技术协助下的极速膨胀”（虽然互联网传播力的助推效果的确存在），而是，他说出了人们的意见和立场 —— 这些意见和立场存在已久，才会在虚假故事中被激发。封杀 Alex Jones 即 阻止激发，并不能改变那些意见和立场的存在；它所暴露的社会政治问题才是真正需要想办法解决的东西。**
  </p>

  </p>
