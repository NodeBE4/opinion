  <p class="graf graf--p">
   2005年，我被中国的秘密警察从上海一直跟踪到杭州。我的“罪行”是什么？只是因为我准备会面几位中国作家。
  </p>
  <p class="graf graf--p">
   当时我在中国为 PEN International 提供有关迎合文学作家的报告。作家关心的问题是什么？他们参与了哪些活动？等等。
  </p>
  <p class="graf graf--p">
   中国间谍一路尾随着我们的汽车，一直在我们后面移动；这是一次可怕的经历，虽然我和 PEN 的同伴最终没有被捕，我们在被监视和追捕的情况下完成了工作。
  </p>
  <p class="graf graf--p">
   但是，另一方面，
   <strong class="markup--strong markup--p-strong">
    我们前一天晚上在中国上海一家餐馆会面的中国作家却遭到了拘留和讯问。当局阻止他们与我们会面
   </strong>
   。
  </p>
  <p class="graf graf--p">
   我们只能希望我们更多地了解这些作家并在工作中支持他们的努力，而不会给他们带来任何真正的伤害。
  </p>
  <p class="graf graf--p">
   这段经历让我对他们甚至同意与我们见面的勇气表示持久的钦佩。
  </p>
  <p class="graf graf--p">
   但那是近15年前的事了。
  </p>
  <p class="graf graf--p">
   如果我们今天再次回到中国做类似的报道，谁会知道我们是否正在被监视呢？
  </p>
  <p class="graf graf--p">
   在很短的时间内，
   <strong class="markup--strong markup--p-strong">
    中国的监视能力已经变得非常复杂，现在已经完全超越了曾经仅仅针对持不同政见者的瞄准，中国已经建立一个监视整个人口一举一动的大规模间谍系统
   </strong>
   。
  </p>
  <p class="graf graf--p">
   事实上，你可以说，曾经承诺成为“解放力量”的技术现在可以轻易地被用来扼杀异议、巩固威权主义、羞辱和起诉奥威尔式政府认为不合时宜的一切。
  </p>
  <p class="graf graf--p">
   详见分析《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%B8%BA%E4%BB%80%E4%B9%88%E6%8A%80%E6%9C%AF%E6%9C%89%E5%88%A9%E4%BA%8E%E6%9A%B4%E6%94%BF%EF%BC%9F%e2%80%8a-%e2%80%8a-%E4%B8%80%E5%9C%BA%E4%BD%A0%E5%BF%85%E9%A1%BB%E5%8F%82%E4%B8%8E/" href="https://www.iyouport.org/%E4%B8%BA%E4%BB%80%E4%B9%88%E6%8A%80%E6%9C%AF%E6%9C%89%E5%88%A9%E4%BA%8E%E6%9A%B4%E6%94%BF%EF%BC%9F%e2%80%8a-%e2%80%8a-%E4%B8%80%E5%9C%BA%E4%BD%A0%E5%BF%85%E9%A1%BB%E5%8F%82%E4%B8%8E/" rel="noopener noreferrer" target="_blank">
    为什么技术有利于暴政？ — — 一场你必须参与的政治斗争
   </a>
   》
  </p>
  <p class="graf graf--p">
   自1989年结束天安门广场民主抗议活动的大屠杀发生以来，数字技术给中国政府带来了全新的和更隐蔽的沉默、压迫和消失持不同政见者的模式，并扼杀了任何对那段历史的谈论。
  </p>
  <p class="graf graf--p">
   这包括在线审查甚至仅仅是对6月4日的提及，以及不断变化的关键词和短语的黑名单，根据情况，被视为“威胁”的还包括“女权主义”、“1984”、“我不同意”……维吾尔族或西藏人的权利，或台湾独立。
  </p>
  <p class="graf graf--p">
   Twitter 等社交媒体平台一直在被中国禁止，并且
   <strong class="markup--strong markup--p-strong">
    许多已经找到解决方法来绕过审查的人最近一直被拘留
   </strong>
   。
  </p>
  <p class="graf graf--p">
   根据国际特赦组织的报告，中国是“世界上被监禁记者和网络持不同政见者人数最多的国家之一”，这当然与其拥有“世界上最先进的网络控制和监视系统”直接相关，正如美国有线电视新闻网报道的那样。
  </p>
  <p class="graf graf--p">
   虽然我们曾经希望互联网能够为我们提供言论自由、跨国界自由交流的能力，甚至可以成为反对意见蓬勃发展的渠道，但是，我们现在看到的情况恰恰相反。
  </p>
  <p class="graf graf--p">
   更糟糕的是，中国模式现在正在出口：中国正在“向其他国家出口技术反乌托邦模式……
  </p>
  <p class="graf graf--p">
   关于这点的最全面中文版报告详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E6%95%B0%E5%AD%97%E5%A8%81%E6%9D%83%E4%B8%BB%E4%B9%89%E7%9A%84%E5%85%B4%E8%B5%B7-%E7%81%BE%E5%B9%B4%E4%B9%8B%E7%95%94/" href="https://www.iyouport.org/%E6%95%B0%E5%AD%97%E5%A8%81%E6%9D%83%E4%B8%BB%E4%B9%89%E7%9A%84%E5%85%B4%E8%B5%B7-%E7%81%BE%E5%B9%B4%E4%B9%8B%E7%95%94/" rel="noopener noreferrer" target="_blank">
    数字威权主义的兴起: 灾年之畔
   </a>
   》，以及《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%9b%be%e8%a7%a3%e4%b8%ad%e5%9b%bd%e7%a7%91%e6%8a%80%e5%b7%a8%e5%a4%b4-%e5%8d%8a%e5%a3%81%e9%ab%98%e5%a0%a1%e5%a5%87%e4%ba%ba/" href="https://www.iyouport.org/%e5%9b%be%e8%a7%a3%e4%b8%ad%e5%9b%bd%e7%a7%91%e6%8a%80%e5%b7%a8%e5%a4%b4-%e5%8d%8a%e5%a3%81%e9%ab%98%e5%a0%a1%e5%a5%87%e4%ba%ba/" rel="noopener noreferrer" target="_blank">
    图解中国科技巨头 — 半壁“高堡奇人”
   </a>
   》。
  </p>
  <p class="graf graf--p">
   中国国内监控设备的规模非同寻常。
  </p>
  <p class="graf graf--p">
   该国正在建立一个“社会信用”体系，被称为“老大哥”、“黑镜”，这是每个反乌托邦未来主题的科幻作家都曾梦想的东西，该系统将于明年投入全面运营。
  </p>
  <p class="graf graf--p">
   该社会信用体系将使政府和其他人能够轻松获取人们行为的详细信息，对其进行评级并使其公开。人们会因政治理由而被拒绝就业、甚至被逮捕。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    不论是西方还是中国，利用互联网追踪个人的行为都已经变得非常猖獗，正在促进压迫的爆发，为威权主义铺平了道路
   </strong>
   。
  </p>
  <p class="graf graf--p">
   我们西方人正在开始理解自己被社交媒体公司监控和操纵的绝对程度，通过社交媒体公司跟踪我们的数据并通过将其出售给政党、零售商甚至外国政府，实现货币化，以此
   <strong class="markup--strong markup--p-strong">
    操纵民主
   </strong>
   。
  </p>
  <p class="graf graf--p">
   对此的详细分析参见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" rel="noopener noreferrer" target="_blank">
    看不见的独裁
   </a>
   》；《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E5%9B%BD%E7%8E%8B%E4%BB%AC%E7%9A%84%E6%99%9A%E5%AE%B4-%E8%BF%99%E6%98%AF%E4%B8%80%E5%9C%BA%E7%9C%8B%E4%B8%8D%E8%A7%81%E7%9A%84%E6%94%BF%E5%8F%98/" href="https://www.iyouport.org/%E5%9B%BD%E7%8E%8B%E4%BB%AC%E7%9A%84%E6%99%9A%E5%AE%B4-%E8%BF%99%E6%98%AF%E4%B8%80%E5%9C%BA%E7%9C%8B%E4%B8%8D%E8%A7%81%E7%9A%84%E6%94%BF%E5%8F%98/" rel="noopener noreferrer" target="_blank">
    国王们的晚宴
   </a>
   》。
  </p>
  <p class="graf graf--p">
   剑桥分析公司的案例远不是唯一的，主流媒体将注意力集中在 Facebook和剑桥分析公司身上是一种根本的选择性失明，
   <strong class="markup--strong markup--p-strong">
    事实上存在一众同类的操纵民主的公司、他们做着同样的事，包括剑桥分析公司在内他们都是情报机构的附属，正因此它们不会因被调查而终止这种行为。在这里看到详细介绍《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a7%98%e5%af%86%e5%ae%a3%e4%bc%a0%e7%bd%91%e7%bb%9c/" href="https://www.iyouport.org/%e7%a7%98%e5%af%86%e5%ae%a3%e4%bc%a0%e7%bd%91%e7%bb%9c/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     秘密宣传战网络
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》
   </strong>
  </p>
  <p class="graf graf--p">
   今天奥威尔式中国正在发生的事应该对西方提出警告，我们如此轻率地认为理所当然的自由已经受到了严重的欺骗和操纵。
  </p>
  <p class="graf graf--p">
   这种控制背后的威权主义冲动已经渗透到了美国的政治体系中，并且没有被警惕，也没有人愿意反击。我们所有人都可能受到中国那种规模的监视。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    注
   </strong>
   ：这篇文章指的是对特朗普政府的定位监视，从2016年的选举之前就开始了，数月来一直是美国舆论界的热点话题。
  </p>
  <p class="graf graf--p">
   但是，如果你以棱镜计划为角度检查这篇文章的思考方式会发现它是错误的 —— 事实上是中国在模仿美国的监视而是反过来。不同的只是中国大张旗鼓地做，美国秘密地做。
  </p>
  <p class="graf graf--p">
   我们在“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/?s=%E7%9B%91%E8%A7%86%E4%B9%8B%E6%81%B6" href="https://www.iyouport.org/?s=%E7%9B%91%E8%A7%86%E4%B9%8B%E6%81%B6" rel="noopener noreferrer" target="_blank">
    监视之恶
   </a>
   ”系列文章中详细介绍过这些历史并给出了分析。
  </p>
  <p class="graf graf--p">
   不过，这篇文章所描述的中国问题是完全真实的，希望能借此激励中国社会的反抗（我们依旧天真地认为依旧有很多中国公民是因不了解反乌托邦实情而没能加入反抗的）。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/commentisfree/2019/may/07/chinese-style-surveillance-exported-west" href="https://www.theguardian.com/commentisfree/2019/may/07/chinese-style-surveillance-exported-west" rel="noopener noreferrer" target="_blank">
    Is Chinese-style surveillance coming to the west?
   </a>
   The Chinese model is now being exported — and the results could be terrifying
  </p>
