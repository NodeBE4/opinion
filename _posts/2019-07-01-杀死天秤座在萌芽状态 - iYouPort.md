    <span style="color: #00ccff;">
     <em>
      <strong>
       “天秤座”不是真正的加密货币，它只是寡头公司拥有的另一个数据库；部分中国知识分子对“天秤座”的崇拜令人恐惧
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-6339 jetpack-lazy-image" data-lazy-sizes="(max-width: 1058px) 100vw, 1058px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-2.png?resize=1058%2C714&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-2.png?w=1058&amp;ssl=1 1058w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-2.png?resize=300%2C202&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-2.png?resize=768%2C518&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-2.png?resize=1024%2C691&amp;ssl=1 1024w" data-recalc-dims="1" height="714" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-2.png?resize=1058%2C714&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1058"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6339" data-recalc-dims="1" height="714" sizes="(max-width: 1058px) 100vw, 1058px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-2.png?resize=1058%2C714&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-2.png?w=1058&amp;ssl=1 1058w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-2.png?resize=300%2C202&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-2.png?resize=768%2C518&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-2.png?resize=1024%2C691&amp;ssl=1 1024w" width="1058"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Libra.org 于6月中旬发布的
   <a class="markup--anchor markup--p-anchor" data-href="https://libra.org/en-US/white-paper/" href="https://libra.org/en-US/white-paper/" rel="noopener noreferrer" target="_blank">
    白皮书
   </a>
   为单一的全球性货币设计了一个巧妙的系统，由一个广泛的、从头开始构建的基础设施支持，这将允许几乎无摩擦的交易。
  </p>
  <p class="graf graf--p">
   这份白皮书读起来就像许多其他白皮书一样，充满了像“区块链”、“分散化”、“公链”这样的词语。
  </p>
  <p class="graf graf--p">
   但是，天秤座并不是真正的分散化或无需许可的。
   <strong class="markup--strong markup--p-strong">
    从根本上它不是区块链，它只是一个数据库。
   </strong>
  </p>
  <p class="graf graf--p">
   基本上只是公关。
  </p>
  <p class="graf graf--p">
   Libra 的核心论点非常巧妙：“为17亿被剥夺了进入任何银行系统的人提供廉价资本和交易系统”。
  </p>
  <p class="graf graf--p">
   他们被称为“无银行账户”人士。
  </p>
  <p class="graf graf--p">
   这份白皮书充满了关于帮助“无银行账户”人士获得融资的
   <strong class="markup--strong markup--p-strong">
    典型噪音
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在这方面，天秤座实际上可以成功。如今世界上几乎所有人口都拥有智能手机，智能手机是世界上许多最贫困人口首次获得互联网接入的方式，并能够在其附近进行支付。
  </p>
  <p class="graf graf--p">
   天秤座无疑会让他们轻松做到这一点。
  </p>
  <p class="graf graf--p">
   阅读白皮书，你会
   <strong class="markup--strong markup--p-strong">
    误认为
   </strong>
   这是 Facebook 的唯一目标。
  </p>
  <p class="graf graf--p">
   然而，这实际上是关于创建一个储备金和支付系统，利用 Facebook 和 WhatsApp 拥有的庞大用户群。您将能够通过 WhatsApp 或 Facebook Messenger 汇款。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这也很容易解释为什么这份白皮书是在亚洲时区的白天发布的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   它清楚地表明了 Facebook 认为其市场在哪里。
  </p>
  <p class="graf graf--p">
   很简单，WhatsApp 在印度拥有巨大的用户群，据信它拥有多达3亿用户。如果你能让他们发送金钱和信息给你，其潜力清晰可见。
  </p>
  <p class="graf graf--p">
   比特币，尽管取得了成功，却未能做到的一件事就是：
   <strong class="markup--strong markup--p-strong">
    让人们用比特币来思考。
   </strong>
  </p>
  <p class="graf graf--p">
   您将始终看到美元、英镑或等值的欧元。人们用美元、欧元或英镑来思考。
  </p>
  <p class="graf graf--p">
   让买家和卖家用“天秤座”来思考也会很难。并且会产生手续费和结售汇的烟幕。
  </p>
  <p class="graf graf--p">
   总之，你能在这个项目中找出一万个漏洞。其他人也能，这非常容易。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    但是必须警告，这也许只是一个开始。BigTech 不会停留于此。
   </strong>
  </p>
  <p class="graf graf--p">
   这些寡头以区块链为幌子进行加密货币炒作只是为了提升他们的围墙花园并开辟新的捞钱途径；在不久的将来，Facebook、苹果和亚马逊都将努力让自己成为银行。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这是超级集中化
   </strong>
   。请记得，它只是寡头公司拥有的另一个数据库。您的匿名已经消失了，您的所有权受到质疑。
  </p>
  <p class="graf graf--p">
   1984正在敲门。无人问津。
  </p>
  <p class="graf graf--p">
   人们应该开始了解这意味着什么。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://mondaynote.com/nipping-libra-in-the-bud-595cea72c6a4" href="https://mondaynote.com/nipping-libra-in-the-bud-595cea72c6a4" rel="noopener noreferrer" target="_blank">
    Frederic Filloux 认为
   </a>
   “天秤座”根本不会发生，不应该发生。你觉得呢？以下是他的文章。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6340 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-2.png?resize=1100%2C580&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-2.png?w=1264&amp;ssl=1 1264w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-2.png?resize=300%2C158&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-2.png?resize=768%2C405&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-2.png?resize=1024%2C540&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-2.png?resize=1100%2C580&amp;ssl=1 1100w" data-recalc-dims="1" height="580" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-2.png?resize=1100%2C580&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

    <img alt="" class="aligncenter size-full wp-image-6340" data-recalc-dims="1" height="580" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-2.png?resize=1100%2C580&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-2.png?w=1264&amp;ssl=1 1264w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-2.png?resize=300%2C158&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-2.png?resize=768%2C405&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-2.png?resize=1024%2C540&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-2.png?resize=1100%2C580&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Libra 很聪明。它应该是一个全球金融体系，虽然它还不是一个银行。
  </p>
  <p class="graf graf--p">
   因此，它成功逃脱了这些机构的相关规定。
  </p>
  <p class="graf graf--p">
   同样，它本身并不是一种新货币，它的创造并不影响全球货币供应；它是一组由一系列强大资产支持的未指明货币 — Libra Reserve — 由政府证券和银行存款构成。
  </p>
  <p class="graf graf--p">
   以上所有都应该确保稳定性。
  </p>
  <p class="graf graf--p">
   然而真相是：没有人知道天秤座会有多稳定；甚至 Facebook 在其系统介绍中也承认，天秤座确实会出现波动。
  </p>
  <p class="graf graf--p">
   简而言之，天秤座是
   <strong class="markup--strong markup--p-strong">
    一个典型的 Facebook 概念
   </strong>
   ：易于掌握，看起来对许多人有益，尤其是所谓的最需要的人。
  </p>
  <p class="graf graf--p">
   它的规模会很大。如果西方存款人只将其银行储蓄的十分之一用于天秤座，其储备基金的价值将
   <strong class="markup--strong markup--p-strong">
    达到2万亿美元
   </strong>
   。
  </p>
  <p class="graf graf--p">
   顺便说一句，这将使天秤座成为债券市场上最大的单一参与者。
  </p>
  <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote">
   <p>
    “Quid lucrum istic Facebook est”
   </p>
  </blockquote>
  <p class="graf graf--p">
   Facebook 在其中捞到了什么？
  </p>
  <p class="graf graf--p">
   有两部分：
  </p>
  <p class="graf graf--p">
   1、它已经赚了天秤座的钱。每个早期的支持者都支付了1000万美元。
  </p>
  <p class="graf graf--p">
   对于最初的
   <a class="markup--anchor markup--p-anchor" data-href="https://libra.org/en-US/partners/" href="https://libra.org/en-US/partners/" rel="noopener noreferrer" target="_blank">
    27个合作伙伴
   </a>
   ，已经转化为2.7亿美元。
  </p>
  <p class="graf graf--p">
   到2020年推出时，天秤座预计将招募100个会员。
  </p>
  <p class="graf graf--p">
   假设他们支付相同的价格，一次的费用就将是惊人的
   <strong class="markup--strong markup--p-strong">
    十亿美元
   </strong>
   。
  </p>
  <p class="graf graf--p">
   对于一个主要是以损失厌恶为动机的客户来说这也许不坏；但是，在未来的角度上，这是在用微不足道的小利换来庞大的噩梦。
  </p>
  <p class="graf graf--p">
   2、在全球交易系统几乎无摩擦的承诺中，Facebook 的关键词是“nearly”。
  </p>
  <p class="graf graf--p">
   这可不是 nearly。在拥有24亿人口的专属客户群中，每用户平均收入 ARPU 的微小变化会对公司的收入产生巨大影响。
  </p>
  <p class="graf graf--p">
   目前，Facebook 的每用户平均收入（ARPU）和收入分配如下所示：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6341 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-3.png?resize=1100%2C356&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-3.png?w=1352&amp;ssl=1 1352w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-3.png?resize=300%2C97&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-3.png?resize=768%2C249&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-3.png?resize=1024%2C332&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-3.png?resize=1100%2C356&amp;ssl=1 1100w" data-recalc-dims="1" height="356" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-3.png?resize=1100%2C356&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   显然对于西方，ARPU 的向上移动具有挑战性；天秤座的主要目标是最后两条线：亚太地区和 ROW。
  </p>
  <p class="graf graf--p">
   在这两个问题上，如果天秤座能够只按每年每用户一美元移动一针，它就将转化为
   <strong class="markup--strong markup--p-strong">
    14亿美元的额外收入
   </strong>
   ，很可能完全是盈利，因为运营结构的成本大部分会被企业会员支付的费用所抵消。
  </p>
  <p class="graf graf--p">
   这仅仅是最低垂的果子。
  </p>
  <p class="graf graf--p">
   我们不是在谈论西方市场或难以捉摸的所谓“无银行账户”。
  </p>
  <p class="graf graf--p">
   前者很难评估，因为西方人比菲律宾、巴西或印度（WhatsApp 最大的市场）的人
   <strong class="markup--strong markup--p-strong">
    更加怀疑 Facebook
   </strong>
   。
  </p>
  <p class="graf graf--p">
   无论如何，我们可能会谈到 Facebook 的盈利和亏损（2018年该公司的净收入为22亿美元）获得额外的数十亿美元可能会出现在2021财年……
  </p>
  <p class="graf graf--p">
   但是，无论数学看起来多么吸引人，都无可否认，这只是纸上的大饼。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    防线 ——
   </strong>
  </p>
  <p class="graf graf--p">
   在天秤座的发展中，Facebook 面临三条阻力：监管机构、金融机构和公众信任。
  </p>
  <p class="graf graf--p">
   1、监管机构在西方比其他地方更强大。
  </p>
  <p class="graf graf--p">
   美联储、欧洲中央银行和其他国家已经不打算让天秤座顺利流通。
  </p>
  <p class="graf graf--p">
   作为金融体系稳定的监护人，他们并不热衷于看到这样一个庞大的玩家在没有任何适当监管的情况下出现。
  </p>
  <p class="graf graf--p">
   至于欧盟监管机构，他们对此事一直保持沉默。
  </p>
  <p class="graf graf--p">
   负责维护欧盟公平竞争的著名反科技寡头垄断专员玛格丽特维斯塔特（Margrethe Vestager）也什么都没说出来（相比 Facebook 或亚马逊，她更沉迷于针对谷歌）。
  </p>
  <p class="graf graf--p">
   即使 Vestager 目前正在忙于竞选欧盟委员会主席，但是，忽视这样一个
   <strong class="markup--strong markup--p-strong">
    太过典型的滥用权力的案例
   </strong>
   ，对她的身份来说也是非常奇怪的。
  </p>
  <p class="graf graf--p">
   2、传统的金融机构有很多损失，但不会轻易放弃。
  </p>
  <p class="graf graf--p">
   零售银行业是一个很容易受到严重破坏的行业。它提供繁重而平庸的服务，并且显然难以发展。
  </p>
  <p class="graf graf--p">
   他们的运营成本很高，而且他们的表现指标很糟糕。
  </p>
  <p class="graf graf--p">
   更具体地说，天秤座的崛起通过吸走大量的 Facebook 用户很容易损害银行的资产负债表。
  </p>
  <p class="graf graf--p">
   仅在美国加拿大和欧洲市场，天秤座就将有超过6亿的潜在客户，比摩根大通等网上银行
   <strong class="markup--strong markup--p-strong">
    大12倍
   </strong>
   。
  </p>
  <p class="graf graf--p">
   此外，金融行业中没有人希望看到如此规模的新人进入证券市场。
  </p>
  <p class="graf graf--p">
   除了信用卡之外，天秤座的第一批支持者中显然没有金融部门，这表明它将尽浑身解数为保持现状而斗争。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Facebook：“相信我。把你的数据给我。把你的钱给我”
   </strong>
  </p>
  <p class="graf graf--p">
   3、天秤座部署的最大障碍仍然是人们对 Facebook 越来越多的不信任，特别是在西方世界。
  </p>
  <p class="graf graf--p">
   最后，问题归结为：我们是否想将自己的钱交给以下这样的公司：
  </p>
  <p class="graf graf--p">
   -一个监视资本主义寡头？
  </p>
  <p class="graf graf--p">
   -在社交网络和一个名为 Calibra 的瑞士非营利组织管理的电子钱包之间会有一道隔离墙吗？如你所知就像 Instagram 或 WhatsApp 与 Facebook 之间的关系？
  </p>
  <p class="graf graf--p">
   -一家一直在使用强大的监视工具监视我们所有人的情绪、健康问题、政治背景和性取向的公司？
  </p>
  <p class="graf graf--p">
   -一个永远不会更改其商业规则并无需考虑其客户或合作伙伴的公司？
  </p>
  <p class="graf graf--p">
   -一个内部施行独裁统治、没有任何异议的对抗、对中国领导者表达个人崇拜的、新兴超级监视社会的宏伟建筑师？
  </p>
  <p class="graf graf--p">
   -一个拥有无上权力的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%ba%e7%b1%bb%e6%95%b0%e6%8d%ae%e5%ba%93%e5%92%8c%e7%ae%97%e6%b3%95%e5%8a%b3%e5%b7%a5%ef%bc%9afacebook-%e7%9a%84%e7%ae%97%e6%b3%95%e5%b7%a5%e5%8e%82-%e7%ac%ac2%e9%83%a8/" href="https://www.iyouport.org/%e4%ba%ba%e7%b1%bb%e6%95%b0%e6%8d%ae%e5%ba%93%e5%92%8c%e7%ae%97%e6%b3%95%e5%8a%b3%e5%b7%a5%ef%bc%9afacebook-%e7%9a%84%e7%ae%97%e6%b3%95%e5%b7%a5%e5%8e%82-%e7%ac%ac2%e9%83%a8/" rel="noopener noreferrer" target="_blank">
    数字极权
   </a>
   ？
  </p>
  <p class="graf graf--p">
   ……
  </p>
  <p class="graf graf--p">
   我怀疑任何人会在此点头。
  </p>
  <p class="graf graf--p">
   天秤座根本不应该发生。但这将是一场艰苦的战斗。⚪️
  </p>
