  <p class="graf graf--p">
   十年前，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wired.com/story/right-to-repair-tenants-on-our-own-devices/wired.com/tag/amazon" href="https://www.wired.com/story/right-to-repair-tenants-on-our-own-devices/wired.com/tag/amazon" rel="noopener noreferrer" target="_blank">
    亚马逊
   </a>
   突然从其美国客户的 Kindles 中删除了 George Orwell《1984》的副本。
  </p>
  <p class="graf graf--p">
   就如这部反乌托邦小说中描述的“记忆洞”，亚马逊此举迅速激发了愤怒和嘲笑。
  </p>
  <p class="graf graf--p">
   但是，事后看来，亚马逊的行为已经在十年前暴露了一个惊人的预兆，这种转变自那时起已经变得更加明显了：即 我们只是一个租户社会。
  </p>
  <p class="graf graf--p">
   在这种特殊情况下，亚马逊称这些书籍是被一家实际上没有权利的供应商添加到 Kindle 商店的。
  </p>
  <p class="graf graf--p">
   亚马逊的发言人称：“当权利持有人通知我们时，我们从我们的系统和客户的设备中删除了非法副本，并赔偿了客户”。
  </p>
  <p class="graf graf--p">
   但是，该公司无法改变的一件事就是，
   <strong class="markup--strong markup--p-strong">
    此举展示了其绝对的权力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   即使是最大的传统零售商也很难想象会直接进入人们的房子并收回他们已经卖给人们的东西。而这些寡头就能做到。
  </p>
  <p class="graf graf--p">
   今天，
   <strong class="markup--strong markup--p-strong">
    我们可能认为自己“拥有”的东西是因为我们付了钱并将它们带回家的，但是，只要它们运行软件或具有数字连接，卖方就会继续控制产品
   </strong>
   。
  </p>
  <p class="graf graf--p">
   正如我们在上个月这篇文章中所分析的，详见
   <strong class="markup--strong markup--p-strong">
    《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%9c%b0%e4%b8%bb2-0%ef%bc%9a%e7%a7%91%e6%8a%80%e7%9a%84%e6%96%b0%e9%a3%9f%e5%88%a9%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89/" href="https://www.iyouport.org/%e5%9c%b0%e4%b8%bb2-0%ef%bc%9a%e7%a7%91%e6%8a%80%e7%9a%84%e6%96%b0%e9%a3%9f%e5%88%a9%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     地主2.0：科技的新食利资本主义
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》。
   </strong>
  </p>
  <p class="graf graf--p">
   我们只是“我们自己”的物品的租户，还是在真正的所有者的恩典之下。
  </p>
  <p class="graf graf--p">
   当然，所谓的“智能”联网机器确实有很多好处。
  </p>
  <p class="graf graf--p">
   现代化的洗衣机可以感知水位和湿度，并可以调节旋转的时间，让你的衣服在适当的干燥程度下完成洗涤；汽车更省油了，因为其配备的计算机可以优化其操作的许多方面，从燃油喷射到制动 …… 所有这些对环境和钱包都有好处。
  </p>
  <p class="graf graf--p">
   但是，这不是全部事实。
  </p>
  <p class="graf graf--p">
   大公司正在使用连接性和嵌入式的智能系统来增加他们的利润并尽可能多地控制原本可以侥幸逃脱的控制。
  </p>
  <p class="graf graf--p">
   也许最令人震惊的例子当属 John Deere 拖拉机了 —— 那些标志性的、明亮的绿色巨人，在大片土地上轰鸣，收割小麦、玉米和大豆。
  </p>
  <p class="graf graf--p">
   几代人一直在农庄里自己修理拖拉机。
  </p>
  <p class="graf graf--p">
   但是现在，在推动建造更加自动化、布满传感器的农业设备方面，John Deere 已经在其拖拉机上安装了严格的软件锁，迫使客户必需通过该公司的维修店才能获得修理。
  </p>
  <p class="graf graf--p">
   农民们抱怨说，即使是简单的维修，也要被收取过高的费用。
  </p>
  <p class="graf graf--p">
   在收获季节，他们失去了前往维修店的关键时刻。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    绝望的农民们已经开始在互联网论坛上闲逛，寻找可以绕过 John Deere 锁的软件，这些锁阻止了他们维修自己购买的机器
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这种感受你一定不陌生。
  </p>
  <p class="graf graf--p">
   苹果公司也发起了一场焦土运动，反对那些大胆修理产品或更换电池的人。
  </p>
  <p class="graf graf--p">
   2017年时人们发现该公司正在用旧电池秘密控制iPhone，从而降低其性能。
  </p>
  <p class="graf graf--p">
   苹果公司表示，这样做只是为了防止老旧手机彻底崩溃。这一举动的技术角度是有道理的；但是，这是秘密操作的事实并没有道理。
  </p>
  <p class="graf graf--p">
   并且很难忽视这种方式可能给该公司带来的巨大利益。
  </p>
  <p class="graf graf--p">
   实际上，手机速度越来越慢的用户不得不去购买新设备，这让苹果公司再次增加了已经很大的利润空间。
  </p>
  <p class="graf graf--p">
   丑闻曝光后，尴尬的苹果公司提供了廉价的替换电池。事情很快就变得很清楚了。
  </p>
  <p class="graf graf--p">
   新 iPhone 销量下滑；蒂姆库克告诉股东，增加维修是这一趋势的“因素”。
  </p>
  <p class="graf graf--p">
   据报道，最近苹果公司与亚马逊达成协议，从亚马逊市场上删除了 “未经授权” 的苹果产品翻新商 —— 那些转售维修过的机器的人。
  </p>
  <p class="graf graf--p">
   作为回报，该公司将让亚马逊销售新的苹果产品：
   <strong class="markup--strong markup--p-strong">
    这是两大巨头的双赢，但是消费者的全输。
   </strong>
  </p>
  <p class="graf graf--p">
   苹果还迫使回收商粉碎旧的 iPhone 和 Macbook，而不是重复使用他们的零件和材料。这不仅对消费者而且对环境都是坏事。
  </p>
  <p class="graf graf--p">
   而且这不仅仅是对价格和利润率的争夺。
  </p>
  <p class="graf graf--p">
   想想看，当您对汽车、电话或其他物体采取了公司总部不喜欢的行动时，会发生什么？
  </p>
  <p class="graf graf--p">
   所有联网设备都可以简单地按命令进行操纵。例如，当表面上的所有者在付款期限三天内没能及时支付时，汽车就被固定了。
  </p>
  <p class="graf graf--p">
   所谓 “未经授权维修”的 John Deere 拖拉机同样被这样变砖了。
  </p>
  <p class="graf graf--p">
   这种情况从什么时候开始的，在家用电器成为间谍监视我们一举一动之前的多久？如果医生不想让你喝第七杯咖啡，咖啡机还会在第六杯之后正常工作吗？
  </p>
  <p class="graf graf--p">
   确实，复杂小工具的维修有时可能需要由持证专业人士进行。但是公司完全可以扩大能够完成工作的专业人员规模，而不是更加
   <strong class="markup--strong markup--p-strong">
    保密地和独家地控制
   </strong>
   。
  </p>
  <p class="graf graf--p">
   对于运行软件的设备来说，连接也是必要的；需要修复错误并更新软件。
  </p>
  <p class="graf graf--p">
   但是，没有任何理由认为这是一个无限制的许可，可以允许公司随意阻止设备或删除其内容。
  </p>
  <p class="graf graf--p">
   今年3月，美国参议员和民主党总统候选人伊丽莎白沃伦呼吁国会通过一项“国家维修权法律”，“允许农民自己修理设备，而无需寻求授权代理人。”
  </p>
  <p class="graf graf--p">
   但这不仅仅是农民遭遇的问题。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这是我们所有人遭遇的问题。
   </strong>
  </p>
  <p class="graf graf--p">
   我们作为数字租户的权利远远少于我们作为房地产租户的权利，后者的驱逐至少受到正当程序的约束。
  </p>
  <p class="graf graf--p">
   如果我们购买了这件东西，那这东西就是我们的。我们不应该让所有权落在1984的“memory hole”里。⚪️
  </p>

    WE ARE TENANTS ON OUR OWN DEVICES
