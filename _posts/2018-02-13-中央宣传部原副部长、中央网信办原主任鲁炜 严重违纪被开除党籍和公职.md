   <h1>
    中央宣传部原副部长、中央网信办原主任鲁炜 严重违纪被开除党籍和公职
   </h1>
   <ul class="post-entry-meta">
    <li>
     By:
     <a href="http://www.ideobook.com/about-ideobook/" title="查看 爱德布克 的作者主页">
      爱德布克
     </a>
     . 2018-2-13.
     <a href="http://www.ideobook.com/372/post-views-count/" title="统计说明">
      7,316
     </a>
    </li>
   </ul>
  </header>
  <div class="post-entry-content">
   <p>
    来源：
    <a href="http://www.ccdi.gov.cn/toutiao/201802/t20180213_164223.html">
     中央纪委监察部网站
    </a>
    发布时间：2018-02-13 18:25
   </p>
   <p style="text-align: center;">
    日前，经中共中央批准，中共中央纪委对中央宣传部原副部长、中央网信办原主任鲁炜严重违纪问题立案审查。
   </p>
   <p style="text-align: center;">
    经查，鲁炜
    <br/>
    严重违反政治纪律和政治规矩，
    <br/>
    阳奉阴违、
    <br/>
    欺骗中央，
    <br/>
    目无规矩、
    <br/>
    肆意妄为，
    <br/>
    妄议中央，
    <br/>
    干扰中央巡视，
    <br/>
    野心膨胀，
    <br/>
    公器私用，
    <br/>
    不择手段为个人造势，
    <br/>
    品行恶劣、
    <br/>
    匿名诬告他人，
    <br/>
    拉帮结派、
    <br/>
    搞“小圈子”；
    <br/>
    严重违反中央八项规定精神和群众纪律，
    <br/>
    频繁出入私人会所，
    <br/>
    大搞特权，
    <br/>
    作风粗暴、
    <br/>
    专横跋扈；
    <br/>
    违反组织纪律，
    <br/>
    组织谈话函询时不如实说明问题；
    <br/>
    违反廉洁纪律，
    <br/>
    以权谋私，
    <br/>
    收钱敛财；
    <br/>
    违反工作纪律，
    <br/>
    对中央关于网信工作的战略部署搞选择性执行；
    <br/>
    以权谋色、
    <br/>
    毫无廉耻。
    <br/>
    利用职务上的便利为他人谋取利益
    <br/>
    并收受巨额财物涉嫌受贿犯罪。
   </p>
   <p style="text-align: center;">
    鲁炜身为党的高级干部，
    <br/>
    理想信念缺失，
    <br/>
    毫无党性原则，
    <br/>
    对党中央极端不忠诚，
    <br/>
    “四个意识”个个皆无，
    <br/>
    “六大纪律”项项违反，
    <br/>
    是典型的“两面人”，
    <br/>
    是党的十八大后不收敛、
    <br/>
    不知止，
    <br/>
    问题严重集中，
    <br/>
    群众反映强烈，
    <br/>
    政治问题与经济问题相互交织的典型，
    <br/>
    性质十分恶劣、
    <br/>
    情节特别严重。
    <br/>
    依据《中国共产党纪律处分条例》等有关规定，经中央纪委常委会会议研究并报中共中央批准，决定给予鲁炜开除党籍、开除公职处分；收缴其违纪所得；将其涉嫌犯罪问题、线索及所涉款物移送有关国家机关依法处理。
    <br/>
    <span id="more-2712">
    </span>
   </p>
   <p style="text-align: center;">
    鲁炜简历
   </p>
   <p>
    鲁炜，男，汉族，1960年1月生，安徽巢湖人，中共党员。1978年10月参加工作，研究生学历，硕士学位，高级记者。2000年起为享受国务院特殊津贴专家。中共十八大代表。
   </p>
   <p>
    1978.07——1978.10 安徽巢县庙岗乡鲁集大队回乡知青；
    <br/>
    1978.10——1984.09 安徽巢县小学、中学教师，桂林风动工具厂机械工人、宣传干事(1982.09-1985.08在广西广播电视大学语文专业学习)；
    <br/>
    1984.09——1991.04 桂林市中级人民法院书记员、干事，广西法制报记者、总编辑助理；
    <br/>
    1991.04——1994.06 新华社桂林记者站站长、桂林支社社长；
    <br/>
    1994.06——1997.11 新华社广西分社副社长、党组成员；
    <br/>
    1997.11——2001.10 新华社广西分社社长、党组书记（1997.09-2001.06在广西大学研究生班学习）；
    <br/>
    2001.10——2001.11 新华社副秘书长，兼任总经理室总经理；
    <br/>
    2001.11——2004.05 新华社党组成员、副秘书长、秘书长，兼总经理室总经理、中国经济信息社社长、新闻信息中心主任；
    <br/>
    2004.05——2011.03 新华社副社长、党组成员，先后兼任秘书长、经济信息编辑部主任、中经社控股董事长兼总经理、金融信息交易所理事长、中国金融信息大厦有限公司董事长、金融信息交易所理事长、国家金融信息中心有限责任公司董事长兼总经理（06.09-09.01在中央党校研究生班学习，06.09-06.11在中央党校省部级进修班学习，08.11-11.07在中国人民大学攻读硕士学位）；
    <br/>
    2011.03——2013.04 中共北京市委常委、宣传部部长，北京市副市长；
    <br/>
    2013.04——2014.05 国家互联网信息办公室主任兼中央外宣办、国务院新闻办副主任；
    <br/>
    2014.05 中宣部副部长、中央网络安全和信息化领导小组办公室主任、国家互联网信息办公室主任；
    <br/>
    2016.06——，中共中央宣传部副部长。（简历摘自中国共产党新闻网）
   </p>
   <hr/>
   <p>
    附：旧闻一则——
   </p>
   <p style="text-align: center;">
    <strong>
     鲁炜当选北京市副市长 蔡赴朝辞任
    </strong>
   </p>
   <p>
    2011年03月31日 北京晚报 廖雁
   </p>
   <p>
    （北京）市人大常委会第二十四次会议上午召开
   </p>
   <p>
    今天上午，市人大常委会第二十四次会议以投票表决的方式任命鲁炜为北京市人民政府副市长。
   </p>
   <p>
    市人大常委会上午审议表决通过了《北京市人民代表大会常务委员会关于接受蔡赴朝辞去北京市副市长职务请求的决定》。会议指出，蔡赴朝在担任北京市副市长期间，认真贯彻落实科学发展观，坚持依法行政，认真履行职责，积极协助市长做好政府工作，勇于改革创新，不断开拓进取，推动北京市经济和社会事业的发展，在负责文化、文物、精神文明等方面的工作中，做了大量卓有成效的工作，取得了明显的成绩。常委会组成人员对他的辛勤工作和作出的贡献表示感谢。
   </p>
   <p>
    郭金龙市长在就提请任免的副市长人选名单作说明时说，
   </p>
   <p style="text-align: center;">
    <span style="color: #0000ff;">
     鲁炜，
     <br/>
     男，1960年1月生，安徽巢湖人，
     <br/>
     2011年3月任北京市委常委、宣传部长，市政府党组成员。
     <br/>
     他政治上坚定，
     <br/>
     认真学习邓小平理论和“三个代表”重要思想，
     <br/>
     深入贯彻落实科学发展观，
     <br/>
     有较强的政治敏锐性和鉴别力，
     <br/>
     认真贯彻党的新闻宣传工作路线方针政策，
     <br/>
     在政治上、思想上、行动上自觉与党中央保持一致。
     <br/>
     他熟悉新闻宣传工作，
     <br/>
     理论功底比较深厚，
     <br/>
     组织领导能力强。
     <br/>
     较长时间在基层工作，
     <br/>
     经历和阅历比较丰富。
     <br/>
     注意学习，
     <br/>
     善于思考，
     <br/>
     注重理论研究。
     <br/>
     思路开阔，
     <br/>
     开拓创新意识强，
     <br/>
     工作敢抓敢管，
     <br/>
     办事果断、
     <br/>
     有魄力。
     <br/>
     他事业心责任感强，
     <br/>
     作风正派，
     <br/>
     处事公道，
     <br/>
     为人谦和，
     <br/>
     团结同志，
     <br/>
     能够听取不同意见。
     <br/>
     要求自己严格，注意廉洁自律。
    </span>
   </p>
   <p>
    鲁炜在任命后作了表态发言。
   </p>
