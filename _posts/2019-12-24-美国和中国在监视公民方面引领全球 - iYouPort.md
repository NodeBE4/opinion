  <p class="graf graf--p">
   我们从内部得到的消息是这样的：中国的人脸识别技术的运算一般分为两种方案，
  </p>
  <p class="graf graf--p">
   一种是集中化的，监视摄像头只负责采集图像，后端传输到数据中心，进行大规模运算。
  </p>
  <p class="graf graf--p">
   另外一种是为了减少中央服务器的负担，将一部分识别任务交给摄像头本身。
  </p>
  <p class="graf graf--p">
   这就要求摄像头内嵌一个操作系统。
  </p>
  <p class="graf graf--p">
   而处理识别部分的硬件设备，已经脱离了嵌入式设备的范畴，可以理解为每一个摄像头内置一个运算力并不弱的微型 PC。摄像头仅仅抓取人脸，但不去验证人脸身份，抓取后通过网络传输到后端服务器。
  </p>
  <p class="graf graf--p">
   这两种技术无法通过外观辨别。
  </p>
  <p class="graf graf--p">
   关于部署情况：目前即便是北上广这样的大城市，带有运算功能的摄像头也是凤毛麟角，大概主要原因是造价太高；而且，要想将所有传统摄像头更换，是一个大工程。
  </p>
  <p class="graf graf--p">
   目前仅仅在一些关键位置部署，比如是敏感地区的地铁、路口。你能猜到的那些地方。
  </p>
  <p class="graf graf--p">
   另外，关于传统摄像头的 **后端运算问题**。答案是，现在当局对传统摄像头的后端运算非常不满意，主要是运算压力太大，基本上实用价值大打折扣。
  </p>
  <p class="graf graf--p">
   目前它们做的是“两手抓”，在逐步替换旧摄像头（这是个漫长的过程）的同时，抓紧增加后端的算力和算法。
  </p>
  <p class="graf graf--p">
   通过消息人士的语气推测，**目前可能存在1～2年的“窗口期”，过了这个时间，人脸识别检测的应用价值就有一个突发的增长。**
  </p>
  <p class="graf graf--p">
   请注意，距离中国的社会信用评分系统全国范围内正式实践，还剩最后2个月。遍布全国各地每一个角落的监视器，将极大程度上辅助社会信用评分的奖惩。
  </p>
  <p class="graf graf--p">
   我们介绍过一些防御人脸识别的思考方式、以及目前已经成功实验的防御技术；今后还会继续介绍。根据 Comparitech 的最新调查数据，中国和美国是全球两大 最热衷于监控其公民 的国家，不同的也许是，美国社会正在努力抵抗，希望中国社会也能跟上来。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9470 jetpack-lazy-image" data-lazy-sizes="(max-width: 898px) 100vw, 898px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/加-2.png?resize=898%2C1032&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/加-2.png?w=898&amp;ssl=1 898w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/加-2.png?resize=261%2C300&amp;ssl=1 261w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/加-2.png?resize=768%2C883&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/加-2.png?resize=891%2C1024&amp;ssl=1 891w" data-recalc-dims="1" height="1032" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/加-2.png?resize=898%2C1032&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="898"/>

  <p class="graf graf--p">
   这已经不再是猜测，美国执法部门一直在向公众撒谎关于扩大CCTV摄像机监控的范围。
  </p>
  <p class="graf graf--p">
   Comparitech 最近发布的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.comparitech.com/vpn-privacy/the-worlds-most-surveilled-cities/" href="https://www.comparitech.com/vpn-privacy/the-worlds-most-surveilled-cities/" rel="noopener noreferrer" target="_blank">
    一份报告
   </a>
   证实了多年来一直在警告人们的事：美国执法部门已成为监视其公民的全球领导者，之一。
  </p>
  <p class="graf graf--p">
   根据 Comparitech 的一项新分析，中国的城市也正处于世界上最严重的监控之下。但是，随着对全球120个城市的公共 CCTV 监视摄像机数量的观察，居住在美国、英国、阿联酋、澳大利亚和印度等城市的一些居民也会发现自己被大量监视之眼所包围。
  </p>
  <p class="graf graf--p">
   根据 Comparitech 的数据，伦敦和亚特兰大是除中国之外仅有的监视力度排名前十的城市。
  </p>
  <p class="graf graf--p">
   Comparitech 衡量了一个城市每千人的CCTV监视摄像机数量，美国两个城市位居前20名。
  </p>
  <p class="graf graf--p">
   亚特兰大拥有 7,800 台监视摄像机，可容纳501,178人，每千人拥有 15.56 台监视摄像机；芝加哥拥有 35,000 台监视摄像机，可容纳2,679,044人，每千人拥有13.06台监视摄像机。
  </p>

   但

  </p>
  <p class="graf graf--p">
   美国正在变成与中国一样的监视国家，尽管主流媒体和执法机构一直在努力避免你看到这点。
  </p>
  <p class="graf graf--p">
   如果这是间谍奥运会，中国和美国将以各自13枚金牌并列领先。

  <p class="graf graf--p">
   令人惊讶的是，新奥尔良市居然没有被列入这个名单。特别是在2017年， Bryan Lagarde 宣布该市为 “
   <a class="markup--anchor markup--p-anchor" data-href="https://massprivatei.blogspot.com/2017/02/mardi-gras-and-bourborn-street-have.html" href="https://massprivatei.blogspot.com/2017/02/mardi-gras-and-bourborn-street-have.html" rel="noopener noreferrer" target="_blank">
    全美最大的高清网络犯罪摄像头系统
   </a>
   ’’ 之后。
  </p>
  <p class="graf graf--p">
   更令人不安的是，就算把香港也包括在内，中国在前120名间谍城市中只占13个。这个数字与美国并驾齐驱，中美两国共同获得了全球间谍监视冠军的称号。
  </p>
  <p class="graf graf--p">
   巴西将以9个 *最间谍城市* 获得银牌，而英国以3个最间谍城市获得铜牌。
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   将公共CCTV监视摄像机的数量与
   <a class="markup--anchor markup--p-anchor" data-href="https://www.numbeo.com/crime/rankings_current.jsp" href="https://www.numbeo.com/crime/rankings_current.jsp" rel="noopener noreferrer" target="_blank">
    Numbeo报告的犯罪和安全指数
   </a>
   进行了比较。对于两个指数，相关性都很弱（r = 0.168，r = -0.168，n = 120）。结论是，监视摄像机的数量和犯罪率之间几乎没有什么关系。
  </p>
  <p class="graf graf--p">
   从广义上讲，更多的摄像头并不一定会使人们感到更安全。
  </p>
  <p class="graf graf--p">
   对中国闭路电视摄像机的数量的估计会有所不同，但据报道，到2020年，中国使用的摄像机数量将从2亿增加到6.26亿。
  </p>
  <p class="graf graf--p">
   根据该国目前的14亿人口计算，这意味着每两个人就要使用一部摄像头。尽管此预测看似庞大，但可能只是实际数字的一小部分。
  </p>
  <p class="graf graf--p">
   中国的深圳市计划在未来几年内安装16,680,000台监控摄像机，目前的数字是 1,929,600台。
  </p>
  <p class="graf graf--p">
   如果整个中国将闭路电视摄像机的数量都增加同样的幅度，则意味着总共有22.9亿台监控摄像机，平均每人接近两台。
  </p>
  <p class="graf graf--p">
   中国还是面部识别监控技术的热心采用者。结合国家的
   <strong class="markup--strong markup--p-strong">
    社会信用评分体系
   </strong>
   ，通常用于限制个人的行动自由。
  </p>

  <p class="graf graf--p">
   我们专注于世界上人口最多的城市，但忽略了找不到足够数据的城市。在可能的情况下，还尝试找到正在使用的私人CCTV监视摄像机的数量。但是，由于并非每个城市都可以做到这一点，因此并未在总数中包括私人监视摄像机。
  </p>
  <p class="graf graf--p">
   对于中国，我们使用了这些来源，但也根据该区域显示的增长率创建了估算值。
  </p>
  <p class="graf graf--p">
   到2020年，预计中国将拥有6.26亿个CCTV摄像机。 213％的增长已应用于未来预测。

   同样，如果当前CCTV摄像机数据的来源早于2019年，这里将采用同比20％的增长。这是中国公安部指示的增长率。
  </p>
