  </p>
  <p class="graf graf--p">
   曾经没有互联网的时候，如果您想出版自己的书，那么需要先打字写稿，将副本投递给各个出版商，然后祈祷他们愿意阅读。如果他们真的读了，那么您将得到一本书仅5％的报酬（这是说如果您很幸运的话）。
  </p>
  <p class="graf graf--p">
   但是随着互联网的到来，尤其是 Kindle时代的到来，就不需要祈祷了。 Kindle允许任何人出版自己的书，并切断传统的出版商网络。
  </p>
  <p class="graf graf--p">
   当然，还有许多其他电子书平台，例如 Apple Books、Kobo、Nook、Tolino、Google Play 等等。喜欢书的人能列出一大堆。
  </p>
  <p class="graf graf--p">
   但是很多有经验的人会告诉你，要想制作那样的书有可能比较贵。
  </p>
  <p class="graf graf--p">
   于是本文的问题是：可以在几乎没有预算的情况下出书吗❓
  </p>
  <p class="graf graf--p">
   下面将介绍这件事可以怎么做。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     创建电子书需要啥
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   要制作可上传到各种平台的电子书，您需要以下这些东西：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    您的手稿（显然）
   </li>
   <li class="graf graf--li">
    一种将手稿转换为电子书格式的方法
   </li>
   <li class="graf graf--li">
    这本书的封面
   </li>
  </ul>
  <p class="graf graf--p">
   没了。
  </p>
  <p class="graf graf--p">
   接下来我们逐一介绍这些。应该首先指出的是，廉价生产的不良电子书是没人会买的。因此，尽管您可以在这里和那里削减成本，但不要在此过程中牺牲质量。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、手稿
   </strong>
  </p>
  <p class="graf graf--p">
   打字的方式完全取决于您自己。我知道有些人崇拜 Microsoft Office，但这显然不是免费的。对于预算有限的作家，您可以使用：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <a class="markup--anchor markup--li-anchor" data-href="https://www.libreoffice.org/" href="https://www.libreoffice.org/" rel="noreferrer noopener" target="_blank">
     ibreOffice
    </a>
   </li>
   <li class="graf graf--li">
    <a class="markup--anchor markup--li-anchor" data-href="https://docs.google.com/" href="https://docs.google.com/" rel="noreferrer noopener" target="_blank">
     Google Docs
    </a>
   </li>
   <li class="graf graf--li">
    Pages (仅限 MacOS)
   </li>
  </ul>
  <p class="graf graf--p">
   行距应为1.15或1.5，不要看中字体。我更喜欢 Arial 或 Garamond。
  </p>
  <p class="graf graf--p">
   密切注意您的书的长度，因为 Amazon Kindle 对书的大小施加了严格的限制。如果超过100,000个单词应减少到大约80,000（或考虑将其变成两本书）。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、将原稿转换为电子书格式
   </strong>
  </p>
  <p class="graf graf--p">
   您已经彻底检查并编辑了稿件，对吗？那就开始形成格式吧。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10815 jetpack-lazy-image" data-lazy-sizes="(max-width: 1090px) 100vw, 1090px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-26.png?resize=1090%2C724&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-26.png?w=1090&amp;ssl=1 1090w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-26.png?resize=300%2C199&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-26.png?resize=1024%2C680&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-26.png?resize=768%2C510&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-26.png?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="724" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-26.png?resize=1090%2C724&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1090"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   这里使用了价格小贵的仅限 MacOS 的名为
   <a class="markup--anchor markup--p-anchor" data-href="https://www.vellum.pub/" href="https://www.vellum.pub/" rel="noopener noreferrer" target="_blank">
    Vellum
   </a>
   的软件应用程序，之所以购买该软件是因为我将其视为长期投资（包括税收减免）。但是，如果您没有预算，也没关系，还有其他选择。
  </p>
  <p class="graf graf--p">
   首先，将文件转换为PDF文件。所有的文字处理平台（包括 Google 文档）都具有“导出为 PDF”按钮。
  </p>
  <p class="graf graf--p">
   但是 Amazon 和 Draft2Digital 实际上更喜欢 Word DOCx 文件。然后，他们将为您转换DOCx文件。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10816 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-18.png?resize=1100%2C823&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-18.png?w=1272&amp;ssl=1 1272w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-18.png?resize=300%2C225&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-18.png?resize=1024%2C766&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-18.png?resize=768%2C575&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-18.png?resize=1100%2C823&amp;ssl=1 1100w" data-recalc-dims="1" height="823" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-18.png?resize=1100%2C823&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10816" data-recalc-dims="1" height="823" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-18.png?resize=1100%2C823&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-18.png?w=1272&amp;ssl=1 1272w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-18.png?resize=300%2C225&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-18.png?resize=1024%2C766&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-18.png?resize=768%2C575&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-18.png?resize=1100%2C823&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   另一种选择是使用免费的
   <a class="markup--anchor markup--p-anchor" data-href="https://calibre-ebook.com/" href="https://calibre-ebook.com/" rel="noopener noreferrer" target="_blank">
    Calibre
   </a>
   并将文件转换为必要的 eBook 格式。 Kindle 使用 .MOBI 格式，其他则要求 .EPUB。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10817 jetpack-lazy-image" data-lazy-sizes="(max-width: 368px) 100vw, 368px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-19.png?resize=368%2C734&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-19.png?w=368&amp;ssl=1 368w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-19.png?resize=150%2C300&amp;ssl=1 150w" data-recalc-dims="1" height="734" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-19.png?resize=368%2C734&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="368"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10817" data-recalc-dims="1" height="734" sizes="(max-width: 368px) 100vw, 368px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-19.png?resize=368%2C734&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-19.png?w=368&amp;ssl=1 368w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-19.png?resize=150%2C300&amp;ssl=1 150w" width="368"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、书的封面
   </strong>
  </p>
  <p class="graf graf--p">
   他们总是说“不要凭封面判断书”，但实际上每个人都这么干，你也是。您是否会掏钱买下它在一定程度上取决于封面有多吸引您。
  </p>
  <p class="graf graf--p">
   所以，别让一个糟糕的封面伤害到你的优秀作品。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10818 jetpack-lazy-image" data-lazy-sizes="(max-width: 848px) 100vw, 848px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-14.png?resize=848%2C864&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-14.png?w=848&amp;ssl=1 848w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-14.png?resize=294%2C300&amp;ssl=1 294w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-14.png?resize=768%2C782&amp;ssl=1 768w" data-recalc-dims="1" height="864" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-14.png?resize=848%2C864&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="848"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10818" data-recalc-dims="1" height="864" sizes="(max-width: 848px) 100vw, 848px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-14.png?resize=848%2C864&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-14.png?w=848&amp;ssl=1 848w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-14.png?resize=294%2C300&amp;ssl=1 294w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-14.png?resize=768%2C782&amp;ssl=1 768w" width="848"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   您将需要一张可以合法用于商业目的的图像（为安全起见，从
   <a class="markup--anchor markup--p-anchor" data-href="https://www.istockphoto.com/" href="https://www.istockphoto.com/" rel="noopener noreferrer" target="_blank">
    iStockPhoto
   </a>
   购买一个图像只需9美元，比侵犯版权的诉讼便宜多了）。
  </p>
  <p class="graf graf--p">
   这里是制作书皮的选项：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     Adobe Photoshop
    </strong>
    —— 仅适用于真正知道自己在做什么的人。免费的替代品是
    <a class="markup--anchor markup--li-anchor" data-href="https://www.gimp.org/" href="https://www.gimp.org/" rel="noopener noreferrer" target="_blank">
     <strong class="markup--strong markup--li-strong">
      GIMP
     </strong>
    </a>
    。
   </li>
   <li class="graf graf--li">
    亚马逊
    <strong class="markup--strong markup--li-strong">
     Kindle Cover Creator
    </strong>
    —— 这是一个“封面创建者”，它制作了相当基本的封面。这应该被视为您的最后选择。
   </li>
   <li class="graf graf--li">
    <a class="markup--anchor markup--li-anchor" data-href="https://www.canva.com/" href="https://www.canva.com/" rel="noopener noreferrer" target="_blank">
     <strong class="markup--strong markup--li-strong">
      Canva
     </strong>
    </a>
    —— Canva 内部有制作书皮的模板。模板是免费的，但是如果您使用Canva的剪贴画，则可能会要求您支付几美元。
   </li>
   <li class="graf graf--li">
    <a class="markup--anchor markup--li-anchor" data-href="https://www.fiverr.com/" href="https://www.fiverr.com/" rel="noopener noreferrer" target="_blank">
     <strong class="markup--strong markup--li-strong">
      Fiverr
     </strong>
    </a>
    —— 如果图形设计不是您的专长，这个可能会是最佳选择。您可以雇用别人做，大约40至50美元。
   </li>
  </ul>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     上传互联网
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   如果上述步骤都完成了，那么说明您已经准备就绪了。现在是时候让您的 Precious 在线。
  </p>
  <p class="graf graf--p">
   除 Amazon Kindle 外，还有许多其他电子书平台。但是，为什么不使用聚合服务而是分别上传到每个平台呢？
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://draft2digital.com/" href="https://draft2digital.com/" rel="noopener noreferrer" target="_blank">
    Draft2Digital
   </a>
   （将获得10％的销售额作为回报）会将您的图书上传到多个电子书平台，例如 Apple、Nook、Kobo 和 OverDrive（将电子书放入图书馆）。非常节省时间的选择。
  </p>
  <p class="graf graf--p">
   对于亚马逊，请在
   <a class="markup--anchor markup--p-anchor" data-href="https://kdp.amazon.com/" href="https://kdp.amazon.com/" rel="noopener noreferrer" target="_blank">
    Kindle Direct Publishing（KDP）
   </a>
   上注册一个帐户，然后按照步骤进行操作就行了（Amazon 会对其进行详细说明）。
  </p>

   <img alt="" class="aligncenter size-full wp-image-10819 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-12.png?resize=1100%2C275&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-12.png?w=1602&amp;ssl=1 1602w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-12.png?resize=300%2C75&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-12.png?resize=1024%2C256&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-12.png?resize=768%2C192&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-12.png?resize=1536%2C384&amp;ssl=1 1536w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-12.png?resize=1100%2C275&amp;ssl=1 1100w" data-recalc-dims="1" height="275" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-12.png?resize=1100%2C275&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     广告与促销
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在线营销和促销本身就是一个巨大的话题。因此，尝试在一篇文章中覆盖所有内容是不可能的。
  </p>
  <p class="graf graf--p">
   而且由于大多数广告和促销活动都涉及金钱，因此我们需要研究的是，
   <strong class="markup--strong markup--p-strong">
    不花钱可以做到的事。
   </strong>
  </p>
  <p class="graf graf--p">
   A：
   <strong class="markup--strong markup--p-strong">
    免费赠送第一本书
   </strong>
   —— 假设您计划了不止一本书，为什么不免费赠送第一本书来建立粉丝群呢？
  </p>
  <p class="graf graf--p">
   设置您的网站，然后在其中添加
   <a class="markup--anchor markup--p-anchor" data-href="https://www.mailerlite.com/" href="https://www.mailerlite.com/" rel="noopener noreferrer" target="_blank">
    Mailerlite
   </a>
   的邮件列表表单。当某人选择了该书时，系统会自动向他们发送一封包含下载链接的电子邮件。
  </p>
  <p class="graf graf--p">
   B：
   <strong class="markup--strong markup--p-strong">
    将免费副本发送给审阅者和影响力大的人
   </strong>
   —— 首先是确定谁会喜欢您的书，并希望此人将其推荐给他人。然后通过电子邮件向他们发送免费的电子书副本。
  </p>
  <p class="graf graf--p">
   C：
   <strong class="markup--strong markup--p-strong">
    社交媒体
   </strong>
   —— 无需任何费用即可建立社交媒体帐户。 Twitter、Facebook 和 Instagram 。您可以举办免费宣传活动，创建主题标签活动等。
  </p>
  <p class="graf graf--p">
   D：
   <strong class="markup--strong markup--p-strong">
    在 Canva 上制作促销图
   </strong>
   —— 在 Canva 上制作海报、社交媒体分享图像、电子邮件分享图像等。
  </p>
  <p class="graf graf--p">
   E：
   <strong class="markup--strong markup--p-strong">
    制作 YouTube 视频
   </strong>
   —— 如果有专门知识，请设置自己的 YouTube 频道，并在自己的电影风格预告片中宣传您的图书。
  </p>
  <p class="graf graf--p">
   在营销和促销方面，您的想象力是唯一的限制。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     最后
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   以上只是出版您自己的电子书的最简易指南，但希望它能使您对所涉及的内容和操作的简易程度有所了解。
  </p>
