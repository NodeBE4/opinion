   </noscript>
  </p>
  <p class="graf graf--p">
   上面那张图看起来像围绕着一颗遥远的恒星的行星，美丽而宁静的东西？然而，事实上它是非常险恶的东西。它们是间谍洞，用于监视整个东欧共产党政权监狱中被拘留者的小洞。
  </p>
  <p class="graf graf--p">
   摄影师 Valentyn Odnoviun 在访问克格勃在立陶宛维尔纽斯的前总部时首先注意到了这些洞。在这个庄严的新古典主义建筑里有一个恐怖的地下室监狱，19个牢房、折磨室和一个处决室，在1944年到60年代初期，有超过1000人死在这里。
  </p>
  <p class="graf graf--p">
   虽然是一个小小的细节，但门上的这些间谍洞突出了监狱中的日常生活。“监狱长通过这些洞监视里面的人，这是一个镜头，”Odnoviun 说，“正是这些洞剥夺了囚犯的人性，使他们成为了纯粹的被监视对象。”
  </p>
  <p class="graf graf--p">
   像维尔纽斯这样的监狱是整个监视系统的一部分，使苏联及其傀儡国家能够统治东欧近半个世纪。秘密警察 — 苏维埃的克格勃、东德的斯塔西以及类似国家机构 — 以及他们庞大的合作者和告密者网络（仅在东德就就有超过28万人）对公民进行广泛的监视，深入每个人的家庭 — — 你的邻居和朋友都是间谍。那些被认为对当权者构成威胁的人在重金属门后面被审问、被折磨，被监禁。
  </p>
  <p class="graf graf--p">
   Odnoviun 记录了立陶宛、爱沙尼亚、拉脱维亚、波兰、乌克兰和德国八所前监狱的间谍洞，他的作品题为“监视”。
  </p>
  <p class="graf graf--p">
   这些监狱的范围从小型的看似普通的房子也许只能关押几个人、到壮观的石头堡垒，有瞭望塔和成千上万的囚犯。其中一些监狱跨越多个政权：在苏联于1944年夺回控制权之前，波兰警察和纳粹德国的盖世太保使用了乌克兰西部的 Lontsky 街监狱。
  </p>
  <p class="graf graf--p">
   虽然 Odnoviun 自己的家庭也受到这些政权的影响，但他试图保持自己的情绪，并且“公正地工作，就像一名独立调查员”。他用一个微距镜头安装在数码单反相机上，然后使用15到30秒的曝光来捕捉图像。
  </p>
  <p class="graf graf--p">
   你在图片中看不到多少 — 那里大部分是划痕、锈迹和类似血液的斑点。这个洞本身成为了重点：整个监视设备的象征，它支撑了独裁统治，并将日常生活变成了悲惨的间谍游戏
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我认为间谍洞两边的人 — — 不论是囚犯还是监视者 — — 都在以自己的方式成为整个监视系统的囚犯，”Odnoviun 说。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    现在这个监狱系统只是变得更大了，它装进了全球的人，而在洞外监视的是极少数技术寡头、邪恶的当权者和他们的情报机构。你想出狱吗？你应该知道该怎么做
   </strong>
   。
  </p>
