  <p class="graf graf--p">
   现在所有人都知道监视资本主义是什么东西了。他们用你的一切行为习惯和内心赚钱。而在这个互联网上的某些角落里，信息会更加值钱。
  </p>
  <p class="graf graf--p">
   现在，一份新的报告发现，即使在心理健康网站上 —— 那种您渴求抒发内心最痛苦的阴影的地方，您的隐私人权也是岌岌可危的。
  </p>
  <ul>
   <li>
    我们提醒过，
    <a href="https://www.iyouport.org/%e5%b0%8f%e5%bf%83%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e4%b8%8a%e7%9a%84%e6%89%80%e6%9c%89%e5%b0%8f%e6%b5%8b%e8%af%95%e5%92%8c%e9%97%ae%e7%ad%94%e6%b8%b8%e6%88%8f/" rel="noopener noreferrer" target="_blank">
     不要使用社交媒体上的任何小测试和问答游戏
    </a>
   </li>
  </ul>
  <p class="graf graf--p">
   国际隐私组织发布了一份报告《
   <a class="markup--anchor markup--p-anchor" data-href="https://privacyinternational.org/sites/default/files/2019-09/Your%20mental%20health%20for%20sale%20-%20Privacy%20International.pdfhttps://privacyinternational.org/sites/default/files/2019-09/Your%20mental%20health%20for%20sale%20-%20Privacy%20International.pdf" href="https://privacyinternational.org/sites/default/files/2019-09/Your%20mental%20health%20for%20sale%20-%20Privacy%20International.pdfhttps://privacyinternational.org/sites/default/files/2019-09/Your%20mental%20health%20for%20sale%20-%20Privacy%20International.pdf" rel="noopener noreferrer" target="_blank">
    您的心理健康正在被出售
   </a>
   》，该报告探讨了心理健康网站如何处理用户数据的问题。结果挺可怕的。
  </p>
  <p class="graf graf--p">
   隐私国际组织查看了Google法国、谷歌德国和谷歌英国的136个心理健康网页。通过英法德语分别搜索抑郁症，并根据网络分析服务 SameWeb 列出了访问量最大的网站。
  </p>
  <p class="graf graf--p">
   使用开源软件 webxray 来识别第三方 HTTP 请求和 cookie。然后对网站进行了分析。
  </p>
  <p class="graf graf--p">
   分析发现，97.78％的网页具有第三方跟踪，其中可能包括 cookie，JavaScript 或外部服务器上托管的图像。这些第三方元素的主要目的是追踪定位广告。
  </p>
  <p class="graf graf--p">
   Webxray 的分析发现，有 76.04％ 的网页具有用于营销目的跟踪器 —— 法国的80.49％的网页、德国的61.36％的网页、和英国的86.27％的网页。
  </p>
  <p class="graf graf--p">
   第三方跟踪器中还包括来自Google、Facebook 和 Amazon 的广告服务，
   <strong class="markup--strong markup--p-strong">
    其中以 Google 跟踪器最为流行
   </strong>
   ，其次是 Facebook 和 Amazon。
  </p>
  <p class="graf graf--p">
   报告还更深入地研究了这些网站的子集（这三个国家的Google搜索结果排名前三位的 “抑郁测试” 网页），结果表明这些跟踪器以更具体、更糟糕的方式诱骗了所有人最私密的数据。
  </p>
  <p class="graf graf--p">
   隐私国际组织发现，一些抑郁症测试网站存储了用户的回复结果，并与第三方共享了这些数据以及人们的测试结果。
  </p>
  <p class="graf graf--p">
   调查还发现，两个抑郁症测试网站都使用了 Hotjar，这是一种在线反馈工具，可以
   <strong class="markup--strong markup--p-strong">
    记录某人在网页上键入和点击的所有内容
   </strong>
   —— 不难想象他们会如何利用这种数据。
  </p>
  <p class="graf graf--p">
   该报告还提出了有关知情同意的问题，尤其是考虑到欧盟 GDPR 以及其声称为更好地保护消费者的数字隐私所做的努力。
  </p>
  <p class="graf graf--p">
   这些网页中不仅有许多跟踪用户敏感信息的跟踪器，而且这样做是在不符合知情同意的新法律标准的情况下进行的。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    更不用说同意根本不起作用《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%95%b0%e6%8d%ae%e6%89%80%e6%9c%89%e6%9d%83%e4%b8%8d%e4%bb%85%e6%97%a0%e6%b3%95%e8%a7%a3%e5%86%b3%e7%8e%b0%e6%9c%89%e9%97%ae%e9%a2%98%ef%bc%8c%e5%ae%83%e8%bf%98%e5%88%b6%e9%80%a0/" href="https://www.iyouport.org/%e6%95%b0%e6%8d%ae%e6%89%80%e6%9c%89%e6%9d%83%e4%b8%8d%e4%bb%85%e6%97%a0%e6%b3%95%e8%a7%a3%e5%86%b3%e7%8e%b0%e6%9c%89%e9%97%ae%e9%a2%98%ef%bc%8c%e5%ae%83%e8%bf%98%e5%88%b6%e9%80%a0/" rel="noopener noreferrer" target="_blank">
     “数据所有权”不仅无法解决现有问题，它还制造了新的问题
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   隐私国际发现，在分析的9个抑郁症测试网站中，有3个未包含Cookie横幅以表明该网站具有第三方跟踪器。并且调查还发现，在确实有横幅的6个网站中，有4个网站没有采用任何明确的方式允许用户退出跟踪。
  </p>
  <p class="graf graf--p">
   报告指出：“许多心理健康网站并未认真对待访问者的隐私。这项研究还表明，一些心理健康网站将访问者的个人数据视为商品，却没有履行欧洲数据保护和隐私法规定的义务。”
  </p>
  <p class="graf graf--p">
   虽然现在您知道了，阴暗的数据经纪人已经将黑手伸到了互联网更脆弱和更敏感的部分，但这确实不足为奇，只是依然需要沮丧地提醒您，您不应该指望任何网站满足你的利益，无论其性质如何。
  </p>
  <p class="graf graf--p">
   国际隐私组织在其报告中建议用户在填写在线心理健康测试以确定它是否安全之前，应在其浏览器中阻止第三方 Cookie，使用广告拦截插件和反跟踪扩展程序，并进行一些尽职调查。
  </p>
  <p class="graf graf--p">
   简单说，
   <strong class="markup--strong markup--p-strong">
    您只能也必须自己保护自己
   </strong>
   。在我们的
   <a class="markup--anchor markup--p-anchor" data-href="https://start.me/p/xbYXdR/iyp-1" href="https://start.me/p/xbYXdR/iyp-1" rel="noopener noreferrer" target="_blank">
    列表-1 中的“技术防身”
   </a>
   栏目中看到具体保护措施；如果您是敏感人士，请参见
   <a class="markup--anchor markup--p-anchor" data-href="https://start.me/p/1kod2L/iyp-direct-action5" href="https://start.me/p/1kod2L/iyp-direct-action5" rel="noopener noreferrer" target="_blank">
    列表-5 中的 “护盾” 栏目
   </a>
   。⚪️
  </p>
