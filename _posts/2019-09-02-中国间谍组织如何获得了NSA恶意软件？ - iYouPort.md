      </strong>
     </em>
    </span>
   </li>

  <p>
   <img alt="" class="aligncenter size-full wp-image-4144 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/hackers.png?resize=1100%2C617&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/hackers.png?w=1258&amp;ssl=1 1258w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/hackers.png?resize=300%2C168&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/hackers.png?resize=768%2C431&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/hackers.png?resize=1024%2C575&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/hackers.png?resize=1100%2C617&amp;ssl=1 1100w" data-recalc-dims="1" height="617" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/hackers.png?resize=1100%2C617&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4144" data-recalc-dims="1" height="617" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/hackers.png?resize=1100%2C617&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/hackers.png?w=1258&amp;ssl=1 1258w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/hackers.png?resize=300%2C168&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/hackers.png?resize=768%2C431&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/hackers.png?resize=1024%2C575&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/hackers.png?resize=1100%2C617&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   据美国网络安全公司赛门铁克称，中国网络间谍组织使用 NSA 的恶意软件已超过一年之久，此前 Shadow Brokers 在网上泄露了同样的漏洞，将其暴露给全世界。
  </p>
  <p class="graf graf--p">
   美国当局于2017年底起诉了三名黑客，美国声称这三人是一家名为 Boyusec 的网络安全公司的幕后黑手，该公司是中国国家安全部的前线，并且攻击了 Moody’s Analytics、西门子和 Trimble 等西方公司。
  </p>
  <p class="graf graf--p">
   该团体被认为是中国政府支持的 APT（高级持续性威胁）中的先进集合，可以自己定制黑客工具和挖掘零日漏洞。
  </p>
  <p class="graf graf--p">
   然而，在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.symantec.com/blogs/threat-intelligence/buckeye-windows-zero-day-exploit" href="https://www.symantec.com/blogs/threat-intelligence/buckeye-windows-zero-day-exploit" rel="noopener noreferrer" target="_blank">
    一份报告中
   </a>
   ，赛门铁克表示，它发现
   <strong class="markup--strong markup--p-strong">
    有证据表明该组织在相同的恶意软件被广泛使用之前很久就已经在使用NSA开发的恶意软件了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   根据赛门铁克发布的图片，Buckeye 组织自2016年3月起开始使用 DoublePulsar 后门版本，超过13个月后，一群被称为 Shadow Brokers 的神秘黑客才于2017年4月在线泄露NSA黑客工具，作为更大规模缓存的一部分。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4145 jetpack-lazy-image" data-lazy-sizes="(max-width: 970px) 100vw, 970px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/赛门铁克.png?resize=970%2C1164&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/赛门铁克.png?w=970&amp;ssl=1 970w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/赛门铁克.png?resize=250%2C300&amp;ssl=1 250w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/赛门铁克.png?resize=768%2C922&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/赛门铁克.png?resize=853%2C1024&amp;ssl=1 853w" data-recalc-dims="1" height="1164" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/赛门铁克.png?resize=970%2C1164&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="970"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4145" data-recalc-dims="1" height="1164" sizes="(max-width: 970px) 100vw, 970px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/赛门铁克.png?resize=970%2C1164&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/赛门铁克.png?w=970&amp;ssl=1 970w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/赛门铁克.png?resize=250%2C300&amp;ssl=1 250w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/赛门铁克.png?resize=768%2C922&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/赛门铁克.png?resize=853%2C1024&amp;ssl=1 853w" width="970"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   这家美国安全厂商表示，它没有看到该团队使用其他与NSA相关的恶意软件，例如 FuzzBunch 框架 —— 这是NSA网络操作员用于在受感染主机上部署 DoublePulsar 后门的常用工具。
  </p>
  <p class="graf graf--p">
   相反，中国黑客团队使用了自己的工具 Bemstour。
  </p>
  <p class="graf graf--p">
   但也有一个转折点。赛门铁克的研究人员表示，Buckeye 使用的
   <strong class="markup--strong markup--p-strong">
    DoublePulsar 版本与 Shadow Brokers 泄露的版本不同，暗示了不同的起源。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “它似乎包含针对较新版 Windows（Windows 8.1 和 Windows Server 2012 R2）的代码，表明它是该恶意软件的新版本，”赛门铁克表示。 “它还包括一层额外的混淆。”
  </p>
  <p class="graf graf--p">
   至于中国黑客使用这个版本 DoublePulsar 的方法显示，似乎他们从未真正理解该恶意软件的全部功能。
  </p>
  <p class="graf graf--p">
   赛门铁克表示，Buckeye 小组“通常使用[DoublePulsar]来执行创建新用户帐户的 shell 命令”，但没有意识到 DoublePulsar 拥有的更多先进秘密功能，这将使攻击者能够执行更多其他隐藏的操作。
  </p>
  <p class="graf graf--p">
   该小组仅在一部分攻击中使用了 DoublePulsar，这表明他们并不那么信任它，相比他们自己的工具。
  </p>
  <p class="graf graf--p">
   赛门铁克报告称，这一版本的 DoublePulsar 袭击了比利时、卢森堡、越南、香港、和菲律宾的组织 —— 通常是为了窃取信息。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4146 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/攻击目录.png?resize=1100%2C614&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/攻击目录.png?w=1616&amp;ssl=1 1616w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/攻击目录.png?resize=300%2C167&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/攻击目录.png?resize=768%2C429&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/攻击目录.png?resize=1024%2C572&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/攻击目录.png?resize=1100%2C614&amp;ssl=1 1100w" data-recalc-dims="1" height="614" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/攻击目录.png?resize=1100%2C614&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   <img alt="" class="aligncenter size-full wp-image-4147 jetpack-lazy-image" data-lazy-sizes="(max-width: 970px) 100vw, 970px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/第三.png?resize=970%2C1164&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/第三.png?w=970&amp;ssl=1 970w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/第三.png?resize=250%2C300&amp;ssl=1 250w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/第三.png?resize=768%2C922&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/第三.png?resize=853%2C1024&amp;ssl=1 853w" data-recalc-dims="1" height="1164" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/第三.png?resize=970%2C1164&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="970"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   在其他泄露的 NSA 工具（例如 EternalBlue 漏洞）被用于世界上一些最大的网络攻击之后，例如 WannaCry 和 NotPetya 勒索软件，Buckeye 组织在2017年中期停止使用 DoublePulsar 后门版本。
  </p>
  <p class="graf graf--p">
   这可能是因为到那时为止，大多数网络安全供应商都能够检测到 DoublePulsar 感染，并且使用他们的 DoublePulsar 版本效率已经下降。
  </p>
  <p class="graf graf--p">
   但最大的谜团仍然是中国黑客团队如何得到 DoublePulsar 后门的？
  </p>
  <p class="graf graf--p">
   赛门铁克和绝大多数信息安全界都支持的理论是，
   <strong class="markup--strong markup--p-strong">
    Buckeye 团队发现了NSA在中国系统上部署的后门，然后他们拿去用了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这就解释了为什么中国黑客使用的是不同版本的 DoublePulsar，与一年后被 Shadow Appkers 泄露的版本不同，很可能来自另一个来源。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.zdnet.com/article/chinese-hackers-were-using-nsa-malware-a-year-before-shadow-brokers-leak/" href="https://www.zdnet.com/article/chinese-hackers-were-using-nsa-malware-a-year-before-shadow-brokers-leak/" rel="noopener noreferrer" target="_blank">
    Chinese hackers were using NSA malware a year before Shadow Brokers leak.
   </a>
   Hacker group used a unique version of the DoublePulsar backdoor, not the one released by the Shadow Brokers.
  </p>
