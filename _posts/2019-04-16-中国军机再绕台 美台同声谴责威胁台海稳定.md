title: 中国军机再绕台 美台同声谴责威胁台海稳定
---

<div id="entry">
<div class="at-above-post addthis_tool" data-url="http://zhanlve.org/?p=7056">
</div>
<p>
<img alt="" class="aligncenter wp-image-7059 size-medium" height="169" sizes="(max-width: 300px) 100vw, 300px" src="http://zhanlve.org/wp-content/uploads/2019/04/EC6DA5E4-9AD5-4B60-AADA-8670E9878EB1_w1023_r1_s-300x169.jpg" srcset="http://zhanlve.org/wp-content/uploads/2019/04/EC6DA5E4-9AD5-4B60-AADA-8670E9878EB1_w1023_r1_s-300x169.jpg 300w, http://zhanlve.org/wp-content/uploads/2019/04/EC6DA5E4-9AD5-4B60-AADA-8670E9878EB1_w1023_r1_s-768x432.jpg 768w, http://zhanlve.org/wp-content/uploads/2019/04/EC6DA5E4-9AD5-4B60-AADA-8670E9878EB1_w1023_r1_s.jpg 1023w" width="300"/>
</p>
<p>
</p>
<p>
<span style="font-size: 12pt;">
   继3月底中国军机飞越台海中线后，星期一(4月15日)，多架解放军军机再度飞经台湾附近海域进行绕岛演训，台湾总统蔡英文批评北京频繁军事活动已成“台海及区域稳定现状最大威胁”，美国在台协会主席莫健也说，“任何意图通过威胁与恫吓来影响台湾的作为都会被美国认为是造成区域不稳并威胁台海稳定”。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   台湾国防部说，包括空警-500、轰-6K、运-8、苏-30及歼-11等各型飞机多架，星期一中午从中国南部陆续出海，飞经台湾南部巴士海峡朝东南航向西太平洋进行远海长航训练，除了轰-6K继续向北经由宫古水道返回驻地外，其余军机都循原航线返回驻地，中共“借由军事行动及舆论战遂行其‘文攻武吓’，企图改变台海现状之事实，此举无助于台海和平，更影响区域稳定与安定”。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   声明说，对于中国军机活动，台湾军方都有军机、军舰联合严密监侦掌握及应处，确保台湾空域及海疆安全。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   中国军机的绕岛演训的时间点，正是美台双方欢庆《台湾关系法》40周年之际。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   星期一傍晚，美国在台协会(AIT)在台北内湖新馆举行“《台湾关系法》&amp; AIT@40”庆祝酒会，包括台湾总统蔡英文、美国在台协会主席莫健(James Moriarty)、台北办事处处长郦英杰(Brent Christensen)，以及一个由前美国国会众议院议长瑞安(Paul Ryan)带领的庆贺团都参加了这个酒会。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   瑞安在酒会上说，“我们无法找到一个比台湾更好的朋友，台湾是一个民主成功的故事，一个可依赖的伙伴，也是世界上一股良善的力量，我们希望全世界都能更像台湾。”
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   莫健在受访时针对解放军绕岛演训的问题表示，“任何通过威胁或胁迫来影响台湾的企图都会被美国认为是造成区域不稳与威胁台海稳定”。他说，“将战机和轰炸机飞到台湾周边，猜测大概与我们今天下午在这里做的事有关，这一点都没有帮助。”
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   莫健说，“这么做损害稳定、损害两岸关系，也损害任何中国想要赢得台湾民心的企图。”
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   蔡英文总统也在其脸书上发文说，中国军机飞经台湾及邻近国家周边，再次进行所谓的“远海长航”，全程都受到台湾海、空军的严密监控，“共军动态无所遁形”，并表示北京“频繁的军事活动，已经是台海及区域稳定现状的最大威胁”，台湾会与区域伙伴密切合作，信守对印太地区和平、安全及稳定的共同承诺。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   针对解放军在台海周边的绕岛演训，解放军《军报》微博15日快讯报道张春晖空军大校就中国海空兵力绕岛巡航发表谈话说，近日中国解放军东部战区出动包括舰艇、轰炸机和侦察机在内的海空兵力绕岛巡航，并进行必要的训练演练，“检验了多军种一体化联合作战能力”，此次行动是年度计划内的正常安排，“完全是主权国家正当合法权利，有利于维护台海和平稳定。”
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   不过，对于解放军称绕岛巡航是“合法权利”并“有利于维护台海和平稳定”，台湾陆委会批评这种说法“令人不齿”。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   陆委会15日发表声明，对解放军“在台海周边空域绕行的挑衅作为”表达严正抗议及强烈谴责。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   声明说，中共军机、军舰借由远海长训频繁出没台海周边区域，对台湾进行武力威慑，年初“习五条”发布后更步步进逼，3月底甚至跨越台海中线，“严重破坏台海现状与区域安全，引发国际社会关注与警告”，“今日竟还大言不惭地声称‘绕岛巡航’是合法权利、有利台海和平，令人不齿。”
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   在中国正不断加大对台湾的军事、外交施压时，美国也在采取应对作为，协助台湾稳定与拉美国家的外交关系以抗衡中国夺取台湾邦交国的努力。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   美国国务卿蓬佩奥上周末在与台湾有外交关系的巴拉圭访问时，与马里奥·阿卜多·贝尼特斯(Mario Abdo Benitez，台湾称阿布铎)总统举行会谈。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   美国国务院发言人摩根·奥特加斯(Morgan Ortagus)4月13日发布的声明说，蓬佩奥与阿卜多重申两国在共享民主价值下将持续深化彼此伙伴关系的承诺，两人也“讨论了巴拉圭持续强化与台湾关系的努力，以及维持台海现状的重要性，包括对台湾的外交承认。”
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
