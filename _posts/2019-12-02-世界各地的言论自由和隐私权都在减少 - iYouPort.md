  <p>
   <img alt="" class="aligncenter size-full wp-image-8606 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/言论自由.png?resize=1100%2C658&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/言论自由.png?w=1400&amp;ssl=1 1400w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/言论自由.png?resize=300%2C179&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/言论自由.png?resize=768%2C459&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/言论自由.png?resize=1024%2C612&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/言论自由.png?resize=1100%2C658&amp;ssl=1 1100w" data-recalc-dims="1" height="658" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/言论自由.png?resize=1100%2C658&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  </p>
  <p class="graf graf--p">
   研究人员发现，世界上近一半的人口生活在言论自由和隐私权受到严重侵蚀的国家。
  </p>
  <p class="graf graf--p">
   198个国家在大规模监视和数据隐私等问题上的排名显示，
   <strong class="markup--strong markup--p-strong">
    “强人”政权正通过技术、镇压抗议和监禁记者等方式，试图扼杀不同意见，并巩固政治权力，公民正在越来越多地遭受监视
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这份由全球风险分析公司 Verisk Maplecroft 最新发布的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.maplecroft.com/insights/analysis/strongmen-regimes-lead-charge-against-freedom-of-speech-and-privacy" href="https://www.maplecroft.com/insights/analysis/strongmen-regimes-lead-charge-against-freedom-of-speech-and-privacy" rel="noopener noreferrer" target="_blank">
    报告
   </a>
   发现，约有33.8亿人（约占全球人口的46％）生活在隐私权和表达自由权均处于“极端风险”的国家。
  </p>
  <p class="graf graf--p">
   在198个国家中，有58个国家被归类为公民隐私权方面的“极端风险”，比2017年的水平高出9％。这些国家在言论自由指数中也被列为“极端风险”。
  </p>
  <p class="graf graf--p">
   从2017年的52个国家增加到2019年的58个，增长了11％。
  </p>
  <p class="graf graf--p">
   然而，仅有28个国家被确定为民主治理指数方面的“极端风险”，该指数确定并衡量专制程度。
  </p>
  <p class="graf graf--p">
   报告称，这些数字代表了全球权利和自由受到侵蚀的“令人担忧”的趋势，这种趋势“超越了最核心的专制国家”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这告诉我们，现在有越来越多的国家将控制公共叙述和消除公众异议置于公民的基本权利之上。”
  </p>
  <p class="graf graf--p">
   报告重点介绍了柬埔寨，去年举行的全国大选使洪森能担任该国首相34年，他通过罚款和监禁两年的手段来打击所谓的“假新闻”，扼杀言论自由。（顺便说现在中国也这样做了，见下图）
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8607 jetpack-lazy-image" data-lazy-sizes="(max-width: 956px) 100vw, 956px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/中间图.png?resize=956%2C1188&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/中间图.png?w=956&amp;ssl=1 956w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/中间图.png?resize=241%2C300&amp;ssl=1 241w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/中间图.png?resize=768%2C954&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/中间图.png?resize=824%2C1024&amp;ssl=1 824w" data-recalc-dims="1" height="1188" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/中间图.png?resize=956%2C1188&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="956"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8607" data-recalc-dims="1" height="1188" sizes="(max-width: 956px) 100vw, 956px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/中间图.png?resize=956%2C1188&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/中间图.png?w=956&amp;ssl=1 956w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/中间图.png?resize=241%2C300&amp;ssl=1 241w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/中间图.png?resize=768%2C954&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/中间图.png?resize=824%2C1024&amp;ssl=1 824w" width="956"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   这不是新鲜事，多年前就已开始“流行”，在这里看到最完整报告《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%95%b0%e5%ad%97%e5%a8%81%e6%9d%83%e4%b8%bb%e4%b9%89%e7%9a%84%e5%85%b4%e8%b5%b7-%e7%81%be%e5%b9%b4%e4%b9%8b%e7%95%94/" href="https://www.iyouport.org/%e6%95%b0%e5%ad%97%e5%a8%81%e6%9d%83%e4%b8%bb%e4%b9%89%e7%9a%84%e5%85%b4%e8%b5%b7-%e7%81%be%e5%b9%b4%e4%b9%8b%e7%95%94/" rel="noopener noreferrer" target="_blank">
    数字威权主义的兴起: 灾年之畔
   </a>
   》。
  </p>
  <p class="graf graf--p">
   洪森还通过扩展网络监控来扼杀在线反对言论，并命令所有互联网流量通过国有数据管理中心传输。
  </p>
  <p class="graf graf--p">
   报告称，中国的言论自由和隐私权的总得分最低。作为全球五分之一的互联网用户的家园，这个国家已经面临着最严峻的互联网限制。
  </p>
  <p class="graf graf--p">
   然而，该研究警告说 2020 年即将推出有争议的社会信用评分系统的影响 —— 一个大众监视和人工智能计划，根据公民的社会、政治和经济行为给予评分 —— 在中国国内和国际都会产生影响。
  </p>
  <p class="graf graf--p">
   在这里看到部分介绍：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    越来越多的国家已经实现了这种反乌托邦，中国只是其中之一《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%a4%be%e4%bc%9a%e4%bf%a1%e7%94%a8%e7%9b%91%e8%a7%86%ef%bc%9a%e8%8b%b1%e5%9b%bd%e3%80%81%e4%b8%ad%e5%9b%bd%e3%80%81%e7%be%8e%e5%9b%bd%e5%92%8c%e6%97%a5%e6%9c%ac%e3%80%90viedo%e3%80%91/" href="https://www.iyouport.org/%e7%a4%be%e4%bc%9a%e4%bf%a1%e7%94%a8%e7%9b%91%e8%a7%86%ef%bc%9a%e8%8b%b1%e5%9b%bd%e3%80%81%e4%b8%ad%e5%9b%bd%e3%80%81%e7%be%8e%e5%9b%bd%e5%92%8c%e6%97%a5%e6%9c%ac%e3%80%90viedo%e3%80%91/" rel="noopener noreferrer" target="_blank">
     “社会信用”监视：英国、中国、美国和日本
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%83%a1%e8%90%9d%e5%8d%9c%ef%bc%9a%e4%b8%ad%e5%9b%bd%e5%bd%93%e5%b1%80%e5%a4%a7%e4%b8%be%e5%a5%96%e5%8a%b1%e6%9c%80%e9%a1%ba%e4%bb%8e%e7%9a%84%e5%85%ac%e6%b0%91/" href="https://www.iyouport.org/%e8%83%a1%e8%90%9d%e5%8d%9c%ef%bc%9a%e4%b8%ad%e5%9b%bd%e5%bd%93%e5%b1%80%e5%a4%a7%e4%b8%be%e5%a5%96%e5%8a%b1%e6%9c%80%e9%a1%ba%e4%bb%8e%e7%9a%84%e5%85%ac%e6%b0%91/" rel="noopener noreferrer" target="_blank">
     胡萝卜比大棒更危险
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   据保护记者委员会称，在土耳其，去年有更多的记者被判入狱，而不是其他任何国家。
  </p>
  <p class="graf graf--p">
   土耳其政府摧毁了与2016年军事政变失败相关的30多万本书；去年，Pen基金会的一份报告发现，有200家媒体和出版机构被关闭、80名作家受到调查和起诉、超过5,800名学者被118所公立大学解雇。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8608 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图.png?resize=1100%2C713&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图.png?w=1240&amp;ssl=1 1240w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图.png?resize=300%2C195&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图.png?resize=768%2C498&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图.png?resize=1024%2C664&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图.png?resize=1100%2C713&amp;ssl=1 1100w" data-recalc-dims="1" height="713" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图.png?resize=1100%2C713&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8608" data-recalc-dims="1" height="713" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图.png?resize=1100%2C713&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图.png?w=1240&amp;ssl=1 1240w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图.png?resize=300%2C195&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图.png?resize=768%2C498&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图.png?resize=1024%2C664&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图.png?resize=1100%2C713&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在隐私权方面，柬埔寨、吉布提、肯尼亚、索马里、坦桑尼亚和赞比亚，都陷入了“极端风险”。
  </p>
  <p class="graf graf--p">
   被列为“言论自由极端风险”的国家包括吉尔吉斯斯坦、摩洛哥、莫桑比克、巴布亚新几内亚、卡塔尔、乌干达和委内瑞拉。
  </p>
  <p class="graf graf--p">
   Verisk Maplecroft 的人权分析师 Sofia Nazalya 表示，特朗普对新闻媒体越来越独裁的立场有可能在其他国家鼓励类似的行为，在那些国家，言论自由和隐私权已成为人们关注的主题。
  </p>
  <p class="graf graf--p">
   Nazalya 说：“特朗普和他的反媒体立场所发生的事与我们认为民主国家的领导人应如何采取行动完全矛盾，这确实为其他国家提供了一个危险的先例。”
  </p>
  <p class="graf graf--p">
   该报告还强调了中国新疆地区的侵犯人权行为，估计有100万维吾尔族人被关押在拘留中心。
  </p>
  <p class="graf graf--p">
   该报告警告跨国公司有污染的服装供应链。中国是世界上最大的棉花生产国，其中四分之三生产在新疆，该行业与强迫劳动有关。
  </p>
  <p class="graf graf--p">
   Nazalya表示，寻求清理供应链的公司可能会发现这个过程非常困难。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “对该地区进行标准尽职调查可能会产生问题，因为中国政府很可能会阻止供应链追溯”。⚪️
  </p>
