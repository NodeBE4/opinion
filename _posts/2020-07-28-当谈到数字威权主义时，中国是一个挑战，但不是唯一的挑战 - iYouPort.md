  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    什么是数字威权主义？
   </strong>
  </h3>
  <p class="graf graf--p">
   准确定义数字威权主义至关重要。一个
   <a class="markup--anchor markup--p-anchor" data-href="https://www.brookings.edu/wp-content/uploads/2019/08/FP_20190827_digital_authoritarianism_polyakova_meserole.pdf" href="https://www.brookings.edu/wp-content/uploads/2019/08/FP_20190827_digital_authoritarianism_polyakova_meserole.pdf" rel="noopener noreferrer" target="_blank">
    常见的定义
   </a>
   中会包括 “专制政权使用数字信息技术监视、压制和操纵国内外人口。”
  </p>
  <p class="graf graf--p">
   这是一个明智的出发点，但是，通过技术和战术对数字威权主义进行分类可以提供更多的见解。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    数字压制包括六种技术：监视、审查、社交操纵和骚扰、网络攻击、互联网关闭、以及针对在线用户的有针对性的迫害
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这六种技术并不是互斥的。例如，由政府间谍植入用户计算机的侵入性间谍软件，它既是监视的形式，也是网络攻击的形式。
  </p>
  <p class="graf graf--p">
   但是每种技术都提供了一组特定的目标，并从一组独特的工具中汲取力量以实现其功能。
  </p>
  <p class="graf graf--p">
   数字监视工具在促进政府对公民的控制和减轻政治对手的挑战方面发挥着越来越重要的作用。
  </p>
  <p class="graf graf--p">
   各国都在部署由人工智能增强的复杂监视工具。领导人通过采用颠覆性的虚假信息策略来巩固自己的统治，
   <strong class="markup--strong markup--p-strong">
    这种策略会导致虚无、玩世不恭和政治上的脱离。
   </strong>
  </p>
  <p class="graf graf--p">
   许多政府正在采用复杂的审查策略，并辅以广泛的所谓 “反假新闻” 法规 —— 即
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%a8%80%e8%ae%ba%e5%ae%a1%e6%9f%a5%e5%85%ad%e9%aa%91%e5%a3%ab%e4%b9%8b%e4%b8%80%ef%bc%9a%e5%81%87%e6%96%b0%e9%97%bb%ef%bc%8c%e6%ad%a3%e5%9c%a8%e4%be%b5%e8%9a%80%e4%b8%9c%e5%8d%97/" href="https://www.iyouport.org/%e8%a8%80%e8%ae%ba%e5%ae%a1%e6%9f%a5%e5%85%ad%e9%aa%91%e5%a3%ab%e4%b9%8b%e4%b8%80%ef%bc%9a%e5%81%87%e6%96%b0%e9%97%bb%ef%bc%8c%e6%ad%a3%e5%9c%a8%e4%be%b5%e8%9a%80%e4%b8%9c%e5%8d%97/" rel="noopener noreferrer" target="_blank">
    “六骑士” 之一
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    这些法规赋予当局不受约束的自由裁量权，以迫害政治反对者和公民活动家
   </strong>
   。
  </p>
  <p class="graf graf--p">
   当所有其他方法都失败时，数十个国家（尤其是在非洲和南亚国家）愿意直接断开互联网连接，并使其人口陷入长期的数字隔离。

   强调，数字威权主义的 “中国模式” 正在成为非民主政权的标准做法。
  </p>
  <p class="graf graf--p">
   2018年互联网自由度报告指出，中国公司正在向全世界的压制性政权提供面部识别技术和先进的分析工具，并且中国正在促进数字威权主义 “作为政府通过技术控制公民的一种方式。”
  </p>
  <p class="graf graf--p">
   一些人甚至
   <a class="markup--anchor markup--p-anchor" data-href="https://www.foreignaffairs.com/articles/world/2018-07-10/how-artificial-intelligence-will-reshape-global-order" href="https://www.foreignaffairs.com/articles/world/2018-07-10/how-artificial-intelligence-will-reshape-global-order" rel="noopener noreferrer" target="_blank">
    预测
   </a>
   ，随着志趣相投的国家 “购买或仿效中国制度”，这将为21世纪 “自由民主与数字专制主义” 之间的史诗般的斗争奠定基础。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    中国与数字威权主义
   </strong>
  </h3>
  <p class="graf graf--p">
   中国并不是大多数技术的主要推动者，它是模仿者。相反，中国只是在监视技术的扩散中真正发挥了巨大的影响。
  </p>
  <p class="graf graf--p">
   例如，对于互联网关闭和对在线用户的迫害，中国没有明显的全球性作用。这些行动由国家政府独立采取。实际上，当涉及到互联网关闭时，中国当局采取了
   <a class="markup--anchor markup--p-anchor" data-href="https://press.princeton.edu/books/hardcover/9780691178868/censored" href="https://press.princeton.edu/books/hardcover/9780691178868/censored" rel="noopener noreferrer" target="_blank">
    更为复杂的审查战术
   </a>
   。长时间关闭互联网的概念太过陈旧，无法满足中国的目的。
  </p>
  <p class="graf graf--p">
   当涉及到在线社交操纵和虚假信息战时，中国是游戏的新手。正如牛津大学研究人员 Philip Howard 和 Samantha Bradshaw
   <a class="markup--anchor markup--p-anchor" data-href="https://comprop.oii.ox.ac.uk/wp-content/uploads/sites/93/2019/09/CyberTroop-Report19.pdf" href="https://comprop.oii.ox.ac.uk/wp-content/uploads/sites/93/2019/09/CyberTroop-Report19.pdf" rel="noopener noreferrer" target="_blank">
    报告的那样
   </a>
   ，“我们发现中国很少使用社交媒体来操纵其他国家的舆论，直到最近才在这方面突出出来。”
  </p>
  <p class="graf graf--p">
   尽管中国人在这一领域的活动已经开始增加，但他们的大部分努力集中在操纵国内言论或针对特定目标开展外国影响力行动方面（例如 香港抗议、台湾大选）；很少有中国当局为独裁政府配备高科技虚假信息宣传能力的例子。
  </p>
  <p class="graf graf--p">
   同样，在提供技术以促进国家发起的网络攻击方面（操纵性软件、数据、计算机系统或网络，以降低运营能力或收集信息的目的），中国只是众多参与者之一。
  </p>
  <p class="graf graf--p">
   当然，这并不是说北京不黑客。仅仅是说与自由民主国家相比，中国并不是黑客技术的 “主要” 出口国。
  </p>
  <p class="graf graf--p">
   隐私国际组织2016年关于全球私人监控行业的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.privacyinternational.org/sites/default/files/2017-12/global_surveillance_0.pdf" href="https://www.privacyinternational.org/sites/default/files/2017-12/global_surveillance_0.pdf" rel="noopener noreferrer" target="_blank">
    详尽报告
   </a>
   记录了528家公司兜售商业入侵技术（例如恶意代码、手机恶意软件等），
   <strong class="markup--strong markup--p-strong">
    其中超过87％的公司位于经合组织成员国，而75％的公司总部位于北约国家
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在审查制度方面，中国的确为独裁政权提供了诱人的模式。
  </p>
  <p class="graf graf--p">
   中国已采取措施，通过对外国官员进行大规模培训，提供某些审查技术来推广其社会压制做法，并要求在中国开展业务的国际公司无论在何处经营都必须遵守中国政府对内容的规定。在下面看到报告。

   <a class="markup--anchor markup--p-anchor" data-href="https://www.hrw.org/report/2019/10/24/speak-out-dangerous/criminalization-peaceful-expression-thailand" href="https://www.hrw.org/report/2019/10/24/speak-out-dangerous/criminalization-peaceful-expression-thailand" rel="noopener noreferrer" target="_blank">
    武器化
   </a>
   ，以对抗异议并压制言论自由。
  </p>
  <p class="graf graf--p">
   尽管技术在帮助当局抑制言论自由方面发挥着重要作用，不幸的是用于强制执行内容限制的最主要工具本质上是合法的。也就是说，没有法律能约束它。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    中国在监控技术推广中的作用
   </strong>
  </h3>
  <p class="graf graf--p">
   在数字威权主义方面，中国在监视领域发挥了大部分影响力。华为等中国公司正在建立先进的 “安全/维稳城市” 平台，为压制性政府提供面部识别和智能视频监控系统，并提供先进的计算机视觉分析功能。
  </p>

  </p>
  <p class="graf graf--p">
   在调查的176个国家中，至少有75个国家的政府都部署了AI监视功能。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    他们以三种形式进行操作 —— 所谓的安全/维稳的城市平台、面部识别监视系统和智能警务
   </strong>

  </p>
  <p class="graf graf--p">
   但是，中国公司并非唯一的供应商。位于美国、法国、英国、德国、西班牙和以色列的公司，也都是维稳监视的主要供应商。

  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   同时，沙特、阿联酋、或哈萨克斯坦等被认为是中国压制技术客户国的国家，他们均从多个国家进口监视技术。
  </p>
  <p class="graf graf--p">
   沙特是一个典型的例子。
   <strong class="markup--strong markup--p-strong">
    中国寡头
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://e.huawei.com/en/publications/global/ict_insights/201708310903/manufacturing/201712061133" href="https://e.huawei.com/en/publications/global/ict_insights/201708310903/manufacturing/201712061133" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     华为
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    正在与该国政府合作建设维稳城市，美国寡头
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.datacenterdynamics.com/news/google-cloud-continues-to-grow-is-coming-to-saudi-arabia/" href="https://www.datacenterdynamics.com/news/google-cloud-continues-to-grow-is-coming-to-saudi-arabia/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     谷歌
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    和
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://news.microsoft.com/en-xm/2018/05/08/microsoft-introduces-azure-stack-in-saudi-arabia-with-sahara-net-and-lenovo/" href="https://news.microsoft.com/en-xm/2018/05/08/microsoft-introduces-azure-stack-in-saudi-arabia-with-sahara-net-and-lenovo/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     微软
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    在该国运营云计算服务器，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/business/2017/jun/15/bae-mass-surveillance-technology-repressive-regimes" href="https://www.theguardian.com/business/2017/jun/15/bae-mass-surveillance-technology-repressive-regimes" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     英国军火制造商 BAE
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    为该国提供了大规模监控系统，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.forbes.com/sites/thomasbrewster/2018/12/04/manhole-covers-that-spy-meet-the-westerners-helping-saudis-build-surveillance-cities/#2bcbd245eb13" href="https://www.forbes.com/sites/thomasbrewster/2018/12/04/manhole-covers-that-spy-meet-the-westerners-helping-saudis-build-surveillance-cities/#2bcbd245eb13" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     NEC
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    则销售面部识别监视摄像头，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bloomberg.com/news/articles/2017-10-26/saudis-are-talking-to-amazon-alibaba-over-new-city-prince-says" href="https://www.bloomberg.com/news/articles/2017-10-26/saudis-are-talking-to-amazon-alibaba-over-new-city-prince-says" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     亚马逊和阿里巴巴
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    正准备在该国 *合作* 开展一项重要的所谓智能城市项目。
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%b8%96%e7%95%8c%e5%9f%8e%e5%b8%82%e6%ad%a3%e5%9c%a8%e6%88%90%e4%b8%ba%e6%96%b0%e5%8a%a0%e5%9d%a1/" href="https://www.iyouport.org/%e4%b8%96%e7%95%8c%e5%9f%8e%e5%b8%82%e6%ad%a3%e5%9c%a8%e6%88%90%e4%b8%ba%e6%96%b0%e5%8a%a0%e5%9d%a1/" rel="noopener noreferrer" target="_blank">
     世界城市正在成为新加坡
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   大多数政府，特别是那些有资源的政府，故意避免依赖单一的技术供应商。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    怎么办
   </strong>
  </h3>
  <p class="graf graf--p">
   总的来说，只有有限的迹象表明中国正在推行某种宏大战略，以系统性地扩散数字威权主义工具。相反，中国的努力会因国家/地区、当地情况和自身利益的不同而异。
  </p>
  <p class="graf graf--p">
   正如康奈尔大学学者 Jessica Weiss
   <a class="markup--anchor markup--p-anchor" data-href="https://docs.house.gov/meetings/IG/IG00/20190516/109462/HHRG-116-IG00-Wstate-ChenWeissJ-20190516.pdf" href="https://docs.house.gov/meetings/IG/IG00/20190516/109462/HHRG-116-IG00-Wstate-ChenWeissJ-20190516.pdf" rel="noopener noreferrer" target="_blank">
    所说
   </a>
   ，“数字威权主义的传播与旨在重塑中国形象的努力并非同一回事。尽管这些监视系统可以帮助政府监视和控制其人民，但是如何使用它们取决于当地政治。”
  </p>
  <p class="graf graf--p">
   这仍然是政策和研究界争论不休的问题。像 Elizabeth Economy 这样的专家则
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cfr.org/blog/yes-virginia-china-exporting-its-model" href="https://www.cfr.org/blog/yes-virginia-china-exporting-its-model" rel="noopener noreferrer" target="_blank">
    断言
   </a>
   中国正在寻求输出 “各种威权资本主义”。
  </p>
  <p class="graf graf--p">
   意图对于识别地缘政治动机和未来行动来说至关重要。如果中国正在采取一项综合战略，以将先进技术推广到非民主政权、对抗自由联盟，这需要制定具体的政策以应对。
  </p>
  <p class="graf graf--p">
   另一方面，如果中国的行动没有机会主义那么系统化（看起来像是机会主义的），那么这将大大改变民主国家对中国入侵的反应。
  </p>
  <p class="graf graf--p">
   美国决策者应该继续认真对待中国独裁技术带来的威胁。重要目标应该是向中国政府施加压力，要求其停止向世界上最专制的政权提供危险工具 —— 并迫使中国停止在国内尤其是新疆和西藏地区部署这种技术。
  </p>
  <p class="graf graf--p">
   但是美国和其他自由民主国家在扩散这些反乌托邦工具时也必须考虑自己的责任。提出一个负责任使用的框架，对故意向压制性政权出售有害产品的公司处以罚款，这将有可能产生有意义的变化。
  </p>
  <h3 class="graf graf--p">
   结论
  </h3>
  <p class="graf graf--p">
   数字压制是一个广义术语，涵盖了一系列技术和策略。中国在特定领域积极销售技术 —— 先进的监视技术。
  </p>
  <p class="graf graf--p">
   但是在其他领域，无论是提供渗透性间谍软件还是兜售复杂的虚假信息宣传战功能，它都不是在单独行动。
  </p>
  <p class="graf graf--p">
   政策制定者应该对中国的行动保持警惕。同时，他们应该注意到，中国并不是唯一一个致力于传播数字威权主义技术的国家。
  </p>
