  <div class="section-inner sectionLayout--insetColumn">
   <p class="graf graf--p graf-after--p" id="1a13">
    世界上最富有的城市之一正在向残疾人和无家可归者发动战争。
   </p>
   <p class="graf graf--p graf-after--p" id="5bd0">
    2018年2月，在旧金山的 Mission 区张贴了一张未签名的传单，警告无家可归者：“如果你今晚天黑之后还在这里，驱逐就会到来。今晚太阳下山后，如果你在距离这个公园100码的范围内，我们会砸你、烧你、打你、赶走你。“
   </p>
   <p class="graf graf--p graf-after--p" id="ee2b">
    传单最终被与当地俱乐部老板
    <a class="markup--anchor markup--p-anchor" data-href="http://www.sfexaminer.com/ex-employee-comes-forward-id-club-owner-threats-burn-homeless-encampment/" href="http://www.sfexaminer.com/ex-employee-comes-forward-id-club-owner-threats-burn-homeless-encampment/" rel="noopener nofollow noreferrer" target="_blank">
     Jason Perkins
    </a>
    联系在一起，此人与旧金山政府关系密切，他们经常
    <a class="markup--anchor markup--p-anchor" data-href="http://www.sfexaminer.com/major-mission-district-sweeps-city-placed-just-6-homeless-shelter/" href="http://www.sfexaminer.com/major-mission-district-sweeps-city-placed-just-6-homeless-shelter/" rel="noopener nofollow noreferrer" target="_blank">
     拆毁无家可归者的帐篷
    </a>
    、建立
    <a class="markup--anchor markup--p-anchor" data-href="https://www.alternet.org/civil-liberties/5-ways-we-design-our-cities-make-them-inhospitable-human-life-photos" href="https://www.alternet.org/civil-liberties/5-ways-we-design-our-cities-make-them-inhospitable-human-life-photos" rel="noopener nofollow noreferrer" target="_blank">
     “敌对”建筑
    </a>
    ，以阻止穷人在公共场所居住，并且对那些不能迅速离开的人
    <a class="markup--anchor markup--p-anchor" data-href="http://america.aljazeera.com/articles/2014/2/18/san-francisco-tensionbetweenhomelessandsecondtechboomers.html" href="http://america.aljazeera.com/articles/2014/2/18/san-francisco-tensionbetweenhomelessandsecondtechboomers.html" rel="noopener nofollow noreferrer" target="_blank">
     使用水管冲击
    </a>
    。（著名美剧“Shameless”中包含了类似的场景，虽然高度美化了）
   </p>
   <p class="graf graf--p graf-after--p" id="aaa7">
    这座城市针对无家可归者的最新武器是一项法律，让人想起疯人院的时代。在旧金山和全国其他主要城市地区，政府正在通过一项名为“监管”的计划，终止残疾人和无家可归者做出最基本决定的能力。这个过程消除了他们的自决权，并经常将他们关在监狱般的设施中。
   </p>
   <p class="graf graf--p graf-after--p" id="d373">
    苏珊·米兹纳（Susan Mizner）是一名旧金山律师，曾于2012年帮助创建了 ACLU 的残疾人权利计划，并曾担任旧金山政府残疾人办公室的负责人。在加利福尼亚州民事和残疾人权利团体联盟10月召开的一次名为“志愿服务第一”的会议上，Mizner 指出，
    <strong class="markup--strong markup--p-strong">
     “监狱是除了死刑以外最大的剥夺公民权利的东西”
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="2f0d">
    Imade Borha 在旧金山心理健康协会工作，这是一个将心理健康服务与同伴导师联系在一起的非营利组织。Borha 是纽约的一名研究生，当时她开始经历慢性抑郁症的发作，导致进入精神科病房两次。从哥伦比亚大学毕业并获得创意写作学位并在马里兰州一家报纸工作两年后，Borha 开始担任心理健康组织的沟通角色。她描述了威尔康奈尔医疗中心的“混乱”。她抗议那些强制性的”康复治疗“，她告诉任何愿意倾听的人：”这让我变得更糟。我需要处在一个更安全的环境中。“Borha 在那里受到了约束、受到注射威胁，并被关进隔离室隔离。
   </p>
   <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="5f0f">
    “我们不会称它为’单独监禁’; 我们把它称为“隔离室”，但是单独监禁是同样的做法……这里只是关押，没有任何治疗方法。”
   </p>
   <p class="graf graf--p graf-after--p" id="9823">
    对残疾人或无家可归者的监禁并非新鲜事物。
    <strong class="markup--strong markup--p-strong">
     臭名昭著的
    </strong>
    <a class="markup--anchor markup--p-anchor" data-href="https://thebaffler.com/latest/after-the-ugly-laws-taylor" href="https://thebaffler.com/latest/after-the-ugly-laws-taylor" rel="noopener nofollow noreferrer" target="_blank">
     <strong class="markup--strong markup--p-strong">
      “ugly laws”
     </strong>
    </a>
    <strong class="markup--strong markup--p-strong">
     使得监禁在公共场所的有明显残疾的人是合法的，该法律在美国已经存在了一个多世纪
    </strong>
    。1867年，旧金山成为第一个制定此类法律的城市。在2018年11月30日，无家可归者联盟主任 Jennifer Friedenbach 在论坛上再次呼吁保护权。
   </p>
   <p class="graf graf--p graf-after--p" id="e0c9">
    但是，某些州正在扩大类似法律，限制残疾人并剥夺他们的权利。在加利福尼亚州，无家可归者和他们的拥护者正在动员反对旧金山立法者 Scott Wiener 的州法律。于2018年9月通过的 SB 1045 扩大了现行的监管法律，允许三个拥有高无家可归人口的地区 — 洛杉矶、圣地亚哥和旧金山
    <strong class="markup--strong markup--p-strong">
     — 无限期地限制
    </strong>
    残疾人和/或去年被执法部门拘留过的吸毒者。根据倡导团体的说法，该法律可以作为剥夺无家可归者、尤其是残疾人和/或吸毒者的公民权利的国家原型。
   </p>
   <p class="graf graf--p graf-after--p" id="2544">
    SF 审查员报告说，
    <strong class="markup--strong markup--p-strong">
     在2018年4月该城区最大的一次针对无家可归人员的大扫荡之后，126名无家可归者的帐篷被强行拆除 — — 被扔进垃圾箱并被运往垃圾填埋场。在失去了他们唯一的家园之后，只有六人被安置在避难所
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="28bc">
    2016年，当五名抗议者在当地警察局外绝食抗议一系列警察杀人事件时，Teitel 被罢工者的反暴力信息所感动，并与其他加州大学旧金山分校的学生一起非正式地提供了对周围的服务。为罢工者提供了超过两周的保健支持。罢工结束后，他们组成了“
    <a class="markup--anchor markup--p-anchor" data-href="https://www.donoharmcoalition.org/" href="https://www.donoharmcoalition.org/" rel="noopener nofollow noreferrer" target="_blank">
     Do No Harm Coalition
    </a>
    ”，将医疗福祉原则与社会正义问题融为一体。
   </p>
   <p class="graf graf--p graf-after--p" id="6c31">
    根据 Teitel 的说法，到目前为止，在加利福尼亚州，要保护一个人“真的很难”，然而这应该是应有的方式。“你正在剥夺他们所有的医疗权，……并把权力交给国家官僚” ……在某些情况下
    <strong class="markup--strong markup--p-strong">
     只有极少数人可能会受到家庭成员的照顾，但是很多时候亲戚可能也不会考虑到无家可归者的最佳利益：例如，许多最终流落街头的人都是因家庭暴力而在那里的，LGBTQ 青年离家出走，很大程度上是因为他们被家人逼出来的或遭到了明确的虐待
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="98b0">
    Teitel 表示，SB 1045 只是一个开始。警察往往会抓捕最多的人 — — 如黑人，Brown，穷人和残疾人 — — 他们将面临更多的警察恐怖。根据2014年的数据，旧金山警察杀害的人中有60％存在心理健康问题。在全国范围内，这个数字
    <a class="markup--anchor markup--p-anchor" data-href="http://rudermanfoundation.org/media-missing-the-story-half-of-all-recent-high-profile-police-related-killings-are-people-with-disabilities/" href="http://rudermanfoundation.org/media-missing-the-story-half-of-all-recent-high-profile-police-related-killings-are-people-with-disabilities/" rel="noopener nofollow noreferrer" target="_blank">
     约为50％
    </a>
    。在许多情况下，警察的“反应”就是杀，就如被杀害的
    <a class="markup--anchor markup--p-anchor" data-href="https://truthout.org/articles/angelenos-take-to-the-streets-for-ezell-ford-omar-abrego-and-ferguson-solidarity/" href="https://truthout.org/articles/angelenos-take-to-the-streets-for-ezell-ford-omar-abrego-and-ferguson-solidarity/" rel="noopener nofollow noreferrer" target="_blank">
     Ezell Ford
    </a>
    ，
    <a class="markup--anchor markup--p-anchor" data-href="http://www.dailycal.org/2014/05/07/leaked-documents-shed-new-light-kayla-moore-death/" href="http://www.dailycal.org/2014/05/07/leaked-documents-shed-new-light-kayla-moore-death/" rel="noopener nofollow noreferrer" target="_blank">
     Kayla Moore
    </a>
    ，
    <a class="markup--anchor markup--p-anchor" data-href="https://splinternews.com/cops-shouldnt-be-above-the-americans-with-disabilities-1793846671" href="https://splinternews.com/cops-shouldnt-be-above-the-americans-with-disabilities-1793846671" rel="noopener nofollow noreferrer" target="_blank">
     Teresa Sheehan
    </a>
    和无数其他人的案例。
   </p>
   <p class="graf graf--p graf-after--p" id="40df">
    志愿服务第一联盟成员组织者 Raia Small 表示，
    <strong class="markup--strong markup--p-strong">
     SB 1045 强调“将无家可归的人赶出街头”，很显然，“立法是为了赢得上流社会的支持者”，那些上流不喜欢在他们的街区看到无家可归的人。
    </strong>
    “
   </p>
   <p class="graf graf--p graf-after--p" id="63bd">
    The No New SF Jail Coalition 组织的成员最近要求该市将注意力转移到创建低收入人群的合作住房上。该市最新的住房报告显示，绝大多数新住宅楼只能为最富有的人提供。
    <a class="markup--anchor markup--p-anchor" data-href="https://www.vox.com/policy-and-politics/2017/4/20/15343720/housing-income-limits" href="https://www.vox.com/policy-and-politics/2017/4/20/15343720/housing-income-limits" rel="noopener nofollow noreferrer" target="_blank">
     每年118,000美元的四人家庭会被视为“低收入”
    </a>
    ，并有资格获得住房补贴。然而，
    <strong class="markup--strong markup--p-strong">
     获得这份补助的可能性是
    </strong>
    <a class="markup--anchor markup--p-anchor" data-href="http://www.sfexaminer.com/80-apply-every-low-income-rental-sf/" href="http://www.sfexaminer.com/80-apply-every-low-income-rental-sf/" rel="noopener nofollow noreferrer" target="_blank">
     <strong class="markup--strong markup--p-strong">
      1/82
     </strong>
    </a>
    <strong class="markup--strong markup--p-strong">
     。
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="408c">
    截至2019年1月8日，
    <strong class="markup--strong markup--p-strong">
     该市要求“低收入”租房者每月至少赚取2,702美元，才能申请其一间有补贴的一居室公寓
    </strong>
    。住房部门允许开发商以456,000美元的价格出售这些单人卧室（还不包括每月480美元的房主协会费用）。
   </p>
   <p class="graf graf--p graf-after--p" id="d147">
    1992年，大约 5,000 名无家可归者加入了集体诉讼，最终导致了迈阿密的
    <a class="markup--anchor markup--p-anchor" data-href="https://www.miamiherald.com/news/local/community/miami-dade/article212189034.html" href="https://www.miamiherald.com/news/local/community/miami-dade/article212189034.html" rel="noopener nofollow noreferrer" target="_blank">
     Pottinger 协议
    </a>
    。联邦地方法院认定，城市及其警察部门对无家可归者的待遇相当于“
    <a class="markup--anchor markup--p-anchor" data-href="http://osaka.law.miami.edu/~schnably/pottinger/pottinger.html" href="http://osaka.law.miami.edu/~schnably/pottinger/pottinger.html" rel="noopener nofollow noreferrer" target="_blank">
     残忍和不寻常的惩罚
    </a>
    ”。法院法官引用的两起事件特别令人震惊：两次，警察给无家可归的人戴上手铐、并放火焚烧他们的财产，请注意，他们的身份证、药品、衣服、甚至圣经，都变成了一堆灰烬。
   </p>
   <p class="graf graf--p graf-after--p" id="b85b">
    然而，
    <strong class="markup--strong markup--p-strong">
     Pottinger 协议
    </strong>
    <a class="markup--anchor markup--p-anchor" data-href="https://www.miaminewtimes.com/news/video-shows-miami-green-shirts-destroying-homeless-property-violating-pottinger-agreement-10292474" href="https://www.miaminewtimes.com/news/video-shows-miami-green-shirts-destroying-homeless-property-violating-pottinger-agreement-10292474" rel="noopener nofollow noreferrer" target="_blank">
     <strong class="markup--strong markup--p-strong">
      经常被忽视
     </strong>
    </a>
    <strong class="markup--strong markup--p-strong">
     ; 美国公民自由联盟的律师说，在2018年4月至10月期间，共有72起警方违反 Pottinger 规定的投诉。现在，该市正试图以高档化的名义推翻 Pottinger
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="74fa">
    根据
    <a class="markup--anchor markup--p-anchor" data-href="https://www.ncbi.nlm.nih.gov/pmc/articles/PMC4679127/" href="https://www.ncbi.nlm.nih.gov/pmc/articles/PMC4679127/" rel="noopener nofollow noreferrer" target="_blank">
     国立卫生研究院的数据
    </a>
    ，住房不安全是导致精神疾病和吸毒的主要压力因素，导致最恶性的周期：被诊断为精神心理疾病的人容易受到无家可归和药物滥用的影响，缺乏住房和医疗保健意味着更多的人在街头受苦。
   </p>
   <p>
    <span class="graf-dropCap">
     O
    </span>
    <a class="markup--anchor markup--p-anchor" data-href="https://truthout.org/articles/cities-across-us-are-stripping-homeless-people-of-their-autonomy/" href="https://truthout.org/articles/cities-across-us-are-stripping-homeless-people-of-their-autonomy/" rel="noopener nofollow noreferrer" target="_blank">
     ne of the world’s richest cities is waging war on disabled and homeless people
    </a>
    . The process takes away their self-determination and often leaves them locked up in jail-like facilities, allegedly for their own protection.
   </p>
