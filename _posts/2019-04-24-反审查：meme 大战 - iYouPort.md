 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <strong>
      <em>
       GFW只是阻止访问，并没有阻止投资，正相反它鼓励后者，中国的互联网巨头一直在通过全球扩散的投资以实现反转GFW的监视和审查，这次他们的目标是谁？
      </em>
     </strong>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   在有报道称中国科技巨头腾讯参与 Reddit 的3亿美元融资之后，Reddit 上发出了警告，用户担心该公司可能会代表北京开始审查内容。
  </p>
  <p class="graf graf--p">
   用户以最好的方式回应他们：用 meme 嘲笑中国政府的审查。
  </p>
  <p class="graf graf--p">
   据彭博社报道，尽管 Reddit 在中国被禁，但腾讯控股还是投了一轮融资，该网站估值为30亿美元。并且，以典型的 Reddit 方式，该网站的用户通过发布在中国被禁止的图像作出回应，例如小熊维尼（由于用户嘲笑其与习近平的相似性而被禁止）以及来自天安门广场的著名“坦克人”的图像。在一小群 Reddit 用户的共同努力下将这些 meme 推向了 Reddit 头版。
  </p>
  <p class="graf graf--p">
   Reddit 首席执行官史蒂夫赫夫曼告诉 BBG，Reddit 对视频游戏社区的关注是吸引两家公司合作的一部分。
  </p>
  <p class="graf graf--p">
   一位 Reddit 用户打趣道，“我认为在我们新的光荣霸主决定不能再发布这些内容之前，贴一张天安门广场的”Tank Man“照片会很不错。”
  </p>
  <p class="graf graf--p">
   但是，如果 Reddit 用户决定大规模放弃该网站，他们究竟会去哪里呢？
  </p>
  <p>
   <img alt="" class="aligncenter wp-image-446 size-large jetpack-lazy-image" data-lazy-sizes="(max-width: 861px) 100vw, 861px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-25.png?resize=861%2C1024&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-25.png?resize=861%2C1024&amp;ssl=1 861w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-25.png?resize=252%2C300&amp;ssl=1 252w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-25.png?resize=768%2C913&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-25.png?w=898&amp;ssl=1 898w" data-recalc-dims="1" height="1024" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-25.png?resize=861%2C1024&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="861"/>
   <noscript>
    <img alt="" class="aligncenter wp-image-446 size-large" data-recalc-dims="1" height="1024" sizes="(max-width: 861px) 100vw, 861px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-25.png?resize=861%2C1024&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-25.png?resize=861%2C1024&amp;ssl=1 861w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-25.png?resize=252%2C300&amp;ssl=1 252w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-25.png?resize=768%2C913&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-25.png?w=898&amp;ssl=1 898w" width="861"/>
   </noscript>
  </p>
  <p>
  </p>
  <p class="graf graf--p">
   中国科技巨头腾讯是第四轮融资的新成员，投资1.5亿美元以加入该公司的先前投资者，其中包括 Sequoia，Fidelity，Andreessen Horowitz，Quiet Capital，VY 和说唱歌手 Snoop Dogg。通过此次融资，该公司表示计划扩大国际市场并扩大其广告平台 — — 以竞争 Facebook 和谷歌主导的市场。
  </p>
  <p class="graf graf--p">
   Reddit 首席执行官史蒂夫·霍夫曼（Steve Huffman）表示，“对我们来说非常重要的事情之一就是我们现在可以向广告商保证，您将在 Reddit 上获得积极的体验，甚至可能获得新的体验，一种与客户联系的新方式，” 他表示，Reddit 对广告商承诺 “免于滥用和其他类型的行为不端”。
  </p>
  <p class="graf graf--p">
   自 Reddit 成立于2005年以来，该平台已经多次引起了对滥用、骚扰和盗版的批评和担忧。但霍夫曼表示，该公司已投资于更好的技术，以捕捉滥用，和更便于用户报告的工具。（简称审查）
  </p>
  <p class="graf graf--p">
   他说，这些工具正在发挥作用。 Reddit 已经吸引了拥有3.3亿月活跃用户，其中一半年龄在18到24岁之间，对广告商来说这是一个有价值的、难以接触的人群。虽然该公司不会透露其收入，但它去年
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnbc.com/2018/06/29/how-reddit-plans-to-make-money-through-advertising.html" href="https://www.cnbc.com/2018/06/29/how-reddit-plans-to-make-money-through-advertising.html" rel="noopener noreferrer" target="_blank">
    赚到了超过1亿美元
   </a>
   。
  </p>
  <p class="graf graf--p">
   腾讯的投资值得注意。 因为 Reddit 不仅在中国被封锁，而且该平台也被中国贬低。但考虑到腾讯也是一些 Reddit 重视的视频游戏公司的投资者，这项投资可能很有意义。现在，通过这笔额外融资，霍夫曼表示，他希望能够获得Facebook和谷歌在线广告领域的巨大市场份额。
  </p>
  <p>
   <span class="graf-dropCap">
    R
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.zerohedge.com/news/2019-02-11/reddit-users-wage-meme-war-against-beijing-after-tencent-investment" href="https://www.zerohedge.com/news/2019-02-11/reddit-users-wage-meme-war-against-beijing-after-tencent-investment" rel="noopener noreferrer" target="_blank">
    eddit have responded with alarm following reports that Chinese tech giant Tencent participated in a $300 million funding round for the website
   </a>
   , with users fearing that the company might push Reddit to start censoring content on behalf of Beijing.
  </p>
