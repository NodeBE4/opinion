  <p class="graf graf--p">
   一般来说，关于已知和未知大约有以下四种情况：
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    已经确认知道；
   </li>
   <li class="graf graf--li">
    知道自己还不知道；
   </li>
   <li class="graf graf--li">
    不知道自己不知道；
   </li>
   <li class="graf graf--li">
    不知道自己已经知道。
   </li>
  </ol>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这其中真正能够使你受到伤害并且给你造成无法估量的损失的，是那些你自以为知晓、实际上并不知道的事；以及那些你甚至都不知道与你即将做出的决定高度相关的事。
   </strong>
  </p>
  <p class="graf graf--p">
   因为构想恰当的科学问题要求我们考虑到已知和未知的事。恰当的科学假说需要经得起检验 —— 至少在理论上有可供我们采取的现成步骤，来检验事实，从而确定假说真实与否。
  </p>
  <p class="graf graf--p">
   其现实意义在于，在实验之前需要提前考虑具有替代性的解释，并以此来设计实验，进而排除替代性解释。
  </p>
  <p class="graf graf--p">
   这在包括情报分析在内的大多数分析中都非常常用。
  </p>
  <p class="graf graf--p">
   比如你计划在两组人之间试验新药品的功效，那么试验的条件必须一致。
  </p>
  <p class="graf graf--p">
   如果A组被安排在海景房里。而B组安排在遍地老鼠屎的地下室里，那么你就会得到一种混合因素，你无法得出“药效差异完全因为药物本身”的结论。
  </p>
  <p class="graf graf--p">
   *腐臭的地下室*属于一种
   <strong class="markup--strong markup--p-strong">
    已知的已知
   </strong>
   情况；药品A是否比药品B更好，属于一种
   <strong class="markup--strong markup--p-strong">
    已知的未知
   </strong>
   情况（也是做此试验的原因）。这里
   <strong class="markup--strong markup--p-strong">
    未知的未知
   </strong>
   情况则会是其他一些潜在的混合因素。
  </p>
  <p class="graf graf--p">
   被试的身体基础状况是其中的因素之一；家庭遗传史也有可能起到作用；被试在当天服用药物的时间不同也有可能导致结果差异。
  </p>
  <p class="graf graf--p">
   一旦你确定了某种混合因素，情况的种类就会干净利索地从
   <strong class="markup--strong markup--p-strong">
    未知的未知
   </strong>
   转向
   <strong class="markup--strong markup--p-strong">
    已知的未知
   </strong>
   。然后，就可以采用修正试验或增加研究次数的方法来找到原因。
  </p>
  <p class="graf graf--p">
   设计出优秀的实验 —— 或评估已完成实验的技巧 —— 可以归结为人们做出可替代解释的能力。
  </p>
  <p class="graf graf--p">
   几乎可以说：
   <strong class="markup--strong markup--p-strong">
    发现未知的未知是分析研究的首要工作
   </strong>
   。
  </p>
  <p class="graf graf--p">
   真正的的科学家绝不会自始至终死守个人偏爱的理论；
   <strong class="markup--strong markup--p-strong">
    真正的科学家应该知道，只有当事情未能按照自己预测发展时，才是真正学习的时机
   </strong>
   。
  </p>
  <p class="graf graf--p">
   活动家对行动的设计、观察家对事态的分析、调查人员对情报的解码，等等，都应该遵循科学的思考方式，才有可能得出有价值的结论。
  </p>
  <p class="graf graf--p">
   下面是一些概述：
  </p>
  <p class="graf graf--p">
   1、对于一些已知的事，比如地球到太阳的距离，如果不去查阅资料或许很多人不能得出答案。但你很确定知道答案是已知的。这就是“
   <strong class="markup--strong markup--p-strong">
    已知的已知
   </strong>
   ”。
  </p>
  <p class="graf graf--p">
   2、对于一些我们不知道的事，比如神经放电如何产生愉悦的感受，我们知道自己不知道这个问题的答案。这就是“
   <strong class="markup--strong markup--p-strong">
    已知的未知
   </strong>
   ”。
  </p>
  <p class="graf graf--p">
   3、对于一些我们知道的事，可能并不知道自己知道，或者忘了自己知道它们。比如你曾祖母娘家姓什么？上幼儿园的时候谁谁在你旁边的床上？
  </p>
  <p class="graf graf--p">
   如果有正确的回忆方法，那么尽管你现在没有意识到，你也肯定会发现自己其实知道这些问题的答案。这就是“
   <strong class="markup--strong markup--p-strong">
    未知的已知
   </strong>
   ”。
  </p>
  <p class="graf graf--p">
   4、还有一种，对于我们不知道的事，我们甚至没有意识到自己不知道。
  </p>
  <p class="graf graf--p">
   假如你购买了一套房产，你可能会聘请各种检查人员以对房屋的各方面质量进行检测。
  </p>
  <p class="graf graf--p">
   假如你从未听说过氡，而你的房产代理人只关心完成交易而不关心你和家人的身体健康。那么你绝不会想到对房屋里的氡含量进行检测。
  </p>
  <p class="graf graf--p">
   但是许多房子的确含有大量的致癌物质 —— 氡。这种情况就叫做“
   <strong class="markup--strong markup--p-strong">
    未知的未知
   </strong>
   ”（当然，您阅读到这里时就不是“未知”了）。
  </p>
  <p class="graf graf--p">
   请注意，
   <strong class="markup--strong markup--p-strong">
    你对一种未知情况知晓与否，要取决于你的专业知识和经验。
   </strong>
  </p>
  <p class="graf graf--p">
   检查员只负责汇报那些可见的东西 —— 他可能知道你的房子里存在一些隐秘的致命性的因素，但他却无法接近那些因素所在的位置。
  </p>
  <p class="graf graf--p">
   如果这种损害确实存在，那么对检查员来说其性质和程度虽然属于未知事物，但他知道它们有可能就在那里（即
   <strong class="markup--strong markup--p-strong">
    已知的未知
   </strong>
   ）。
  </p>
  <p class="graf graf--p">
   如果你盲目地接受他的报告，并认为该报告已经完成，那么你就不知道还可能存在隐蔽的损害因素（即
   <strong class="markup--strong markup--p-strong">
    未知的未知
   </strong>
   ）。
  </p>
  <p class="graf graf--p">
   刷流量的媒体、抢热点的大V、伪知识分子，和任何信任他们的人之间基本都属于这类关系。
  </p>

   </strong>

  <p class="graf graf--p">
   有些事情之所以会造成糟糕的和致命的后果，主要原因就在于，人们相信了那些不真实的说辞。
  </p>
  <p class="graf graf--p">
   思考很重要。有些时候好的思考方法可以保命 —— 比如在反抗运动的对峙中。
  </p>
  <p class="graf graf--p">
   这就是本文希望说的事。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    贝叶斯思维
   </strong>
  </p>
  <p class="graf graf--p">
   在贝叶斯概率中，根据新近得到的数据或先验概率，你就可以把自己对某事物的认识进行修正或更新 —— 假定你表现出了某种症状，计算你患有某种疾病的概率；或假定选民的居住地点，计算他们会投票给谁的概率，等等。
  </p>
  <p class="graf graf--p">
   更多：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%8a%95%e7%a5%a8%e5%bf%83%e7%90%86%e5%ad%a6%ef%bc%9a%e8%a1%8c%e4%b8%ba%e6%88%90%e5%9b%a0%e5%88%86%e6%9e%90-%e5%b8%ae%e4%bd%a0%e6%8a%b5%e5%88%b6%e5%ae%a3%e4%bc%a0%e7%9a%84%e8%bf%b7%e6%83%91/" href="https://www.iyouport.org/%e6%8a%95%e7%a5%a8%e5%bf%83%e7%90%86%e5%ad%a6%ef%bc%9a%e8%a1%8c%e4%b8%ba%e6%88%90%e5%9b%a0%e5%88%86%e6%9e%90-%e5%b8%ae%e4%bd%a0%e6%8a%b5%e5%88%b6%e5%ae%a3%e4%bc%a0%e7%9a%84%e8%bf%b7%e6%83%91/" rel="noopener noreferrer" target="_blank">

   </a>
   》
  </p>
  <p class="graf graf--p">
   在计算贝叶斯概率时，首先给假定条件指定一个带有主观性的概率 —— 称为
   <strong class="markup--strong markup--p-strong">
    事前概率
   </strong>
   ，然后根据所收集的数据对这个概率进行修正 —— 所得出的即为
   <strong class="markup--strong markup--p-strong">
    事后概率
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在检验假设之前，如果已经有理由认为它是真实的，那么用不了太多的证据就能将其证实；相反，我们认为假设不太可能成立，就需要更多的证据加以证实。
  </p>
  <p class="graf graf--p">
   从贝叶斯算法的角度来说，
   <strong class="markup--strong markup--p-strong">
    可能性不大的表述与可能性较大的表述相比，前者需要更强有力的证据。
   </strong>
  </p>
  <p class="graf graf--p">
   更多：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e7%9f%a5%e9%81%93%e4%bb%80%e4%b9%88%e6%89%8d%e6%98%af%e7%9c%9f%e6%ad%a3%e7%9a%84%e8%b0%83%e6%9f%a5%e5%90%97%ef%bc%9f%e6%89%80%e8%b0%93%e7%9a%84%e8%b5%b7%e5%ba%95%e7%bb%8f%e5%b8%b8/" href="https://www.iyouport.org/%e4%bd%a0%e7%9f%a5%e9%81%93%e4%bb%80%e4%b9%88%e6%89%8d%e6%98%af%e7%9c%9f%e6%ad%a3%e7%9a%84%e8%b0%83%e6%9f%a5%e5%90%97%ef%bc%9f%e6%89%80%e8%b0%93%e7%9a%84%e8%b5%b7%e5%ba%95%e7%bb%8f%e5%b8%b8/" rel="noopener noreferrer" target="_blank">
    关于什么是证据，什么是调查
   </a>
   》
  </p>
  <p class="graf graf--p">
   在使用贝叶斯概率时，上面那个四重表就是方法之一。
  </p>
  <p class="graf graf--p">
   有些证据与我们的已知情况相一致；但相比下，科学家
   <strong class="markup--strong markup--p-strong">
    应该为那些与标准理论或模型相悖的证据设置更高的检验门槛
   </strong>
   。
  </p>
  <p class="graf graf--p">
   贝叶斯概率法并不是处理不可能事件时可以采用的唯一方法。在探索希格斯玻色子是否存在时，物理学家们设定的门槛要比一般门槛的严格度高出五千倍 —— 并不是说希格斯玻色子存在的可能性很小，而是犯错的成本太高。
  </p>
  <p class="graf graf--p">
   司法科学界的例子也非常能说明贝叶斯法则的应用。法国物理学家 Edmond Locard 指出：凡接触必定留下痕迹。
  </p>
  <p class="graf graf--p">
   Locard 认为，嫌疑人要么留下了证据，要么带走了证据，比如他的举止、身体、或衣服上，表明他曾经去过什么地方或做过什么事。
  </p>
  <p class="graf graf--p">
   下面举一个例子（演示）。
  </p>
  <p class="graf graf--p">
   假设有一名嫌疑人在比赛前进入马厩给马下药，那么他必定会在现场留下一些痕迹 —— 脚印、皮肤碎屑、头发、衣服碎片等等。同理，他也会带走马厩的泥土、马的毛发、地毯纤维，以及其他东西。
  </p>
  <p class="graf graf--p">
   现在假设有人在事发第二天被捕。警方从他的衣服、双手、指甲中取得了证据样本，并发现这些样本和证据现场的样本具有相似之处。
  </p>
  <p class="graf graf--p">
   检察官要评估这些证据的作证力度。因为嫌疑人有罪，所以两种样本具有相似性；或者嫌疑人无罪，但与犯罪分子接触过，这种接触会留下痕迹；再或者嫌疑人在事发时身处另一处马厩，于是导致两种样本有相似性。
  </p>
  <p class="graf graf--p">
   客观概率，比如犯罪嫌疑人的DNA与犯罪现场采集的DNA相匹配的概率；和那些个人的主观看法，比如证人的可信度，或者警方罪案调查科负责管理DNA样本的警官的诚信度。使用贝叶斯法则可以将它们结合起来。
  </p>
  <p class="graf graf--p">
   嫌疑人有前科吗？或者他是否对赛马一无所知？是否具有充分的不在场证明？……这些因素都可以提供很好的主观概率。
  </p>
  <p class="graf graf--p">
   美国司法系统有这样一种假设：在被证明有罪之前，嫌疑人一概无辜。
  </p>
  <p class="graf graf--p">
   如果直接搬运这条假设，那么犯罪嫌疑人有罪的事前概率就为零，并且任何证据 —— 不管有多糟糕，都不会产生一个大于零的事后概率，因为你的计算结果总要乘以零。
  </p>
  <p class="graf graf--p">
   建立嫌疑人无罪的事前概率还有一种更为合理的方法，即 把统计群组中的每个人的作案概率皆视为均等。这样一来，如果嫌疑人属于一个拥有十万人口的城市，并且调查人员有理由相信作案者是这个城市的公民，那么嫌疑人有罪的事前概率就是十万分之一。
  </p>
  <p class="graf graf--p">
   当然，证据肯定能够缩小统计群组的范围。比如确切知道现场没有强行闯入马厩的痕迹，那么嫌疑人肯定是50名能自由出入马厩的人员之一。
  </p>
  <p class="graf graf--p">
   这样，事前假设是嫌疑人有罪的概率是0.02。现在假设嫌疑人和马发生过撕扯，现场发现了人的血迹。检测发现，嫌疑人血样的DNA与取自犯罪现场血样的DNA匹配的概率为0.85。
  </p>
  <p class="graf graf--p">
   按照这些信息创建四重表。
  </p>
  <p class="graf graf--p">
   首先填写下方的方格：嫌疑人有罪的概率为0.02；无罪的概率为0.98；DNA匹配概率为0.85，于是左下方的数字必定为0.15。（上下方两个方格的概率之和必为1）
  </p>
  <p class="graf graf--p">
   血液相匹配的概率0.85，也可能意味着其他事：其他人在现场留下血迹的概率为0.15，而此人并不是嫌疑人。
  </p>
  <p class="graf graf--p">
   再看那个0.15，用 49*0.15 得到7.35，把这个数字填到右上方格中；再用49减去它，从而得到右下格的数字。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-7197 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2.jpeg?resize=1100%2C394&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2.jpeg?w=1317&amp;ssl=1 1317w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2.jpeg?resize=300%2C108&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2.jpeg?resize=768%2C275&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2.jpeg?resize=1024%2C367&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2.jpeg?resize=1100%2C394&amp;ssl=1 1100w" data-recalc-dims="1" height="394" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2.jpeg?resize=1100%2C394&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-7197" data-recalc-dims="1" height="394" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2.jpeg?resize=1100%2C394&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2.jpeg?w=1317&amp;ssl=1 1317w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2.jpeg?resize=300%2C108&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2.jpeg?resize=768%2C275&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2.jpeg?resize=1024%2C367&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2.jpeg?resize=1100%2C394&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   这就是法官和陪审团需要评估的信息：
  </p>
  <p class="graf graf--p">
   P（嫌疑人有罪 ∣血液相匹配） = 0.85/ 8.2 = 0.10
  </p>
  <p class="graf graf--p">
   P（嫌疑人无罪 ∣血液相匹配） = 7.35/ 8.2 = 0.90
  </p>
  <p class="graf graf--p">
   上述证据说明，嫌疑人无罪的概率大约是其有罪概率的9倍。
  </p>
  <p class="graf graf--p">
   计算的起点是他有罪的概率为0.02；所以新的信息实际上使他有罪的概率增加了5倍。尽管如此，他无罪的概率仍然很大。
  </p>
  <p class="graf graf--p">
   然而，现在假设有出现了
   <strong class="markup--strong markup--p-strong">
    新证据
   </strong>
   —— 在嫌疑人的外衣上发现了马的毛发，而且毛发属于被下药的马的概率为0.95（属于其他马的概率仅为0.05）。
  </p>
  <p class="graf graf--p">
   现在就可以把贝叶斯概率串联起来了，重新填写一个表格。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-7198 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3.jpeg?resize=1100%2C406&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3.jpeg?w=1302&amp;ssl=1 1302w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3.jpeg?resize=300%2C111&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3.jpeg?resize=768%2C284&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3.jpeg?resize=1024%2C378&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3.jpeg?resize=1100%2C406&amp;ssl=1 1100w" data-recalc-dims="1" height="406" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3.jpeg?resize=1100%2C406&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-7198" data-recalc-dims="1" height="406" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3.jpeg?resize=1100%2C406&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3.jpeg?w=1302&amp;ssl=1 1302w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3.jpeg?resize=300%2C111&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3.jpeg?resize=768%2C284&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3.jpeg?resize=1024%2C378&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3.jpeg?resize=1100%2C406&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   于是，现在通过计算可以看到：
  </p>
  <p class="graf graf--p">
   P（嫌疑人有罪 ∣证据） = 0.68 P（证据 ∣嫌疑人有罪） = 0.95
  </p>
  <p class="graf graf--p">
   P（嫌疑人无罪 ∣证据） = 0.32 P（证据 ∣嫌疑人无罪） = 0.05
  </p>
  <p class="graf graf--p">
   新的证据表明，在假定新证据真实的条件下，嫌疑人有罪的概率大约是其无罪概率的两倍。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    上述演示看起来很简单，但是现实中很多法官和律师并不懂得如何采用这样的方法来组织证据；P（嫌疑人有罪 ∣证据）=P（证据 ∣嫌疑人有罪）的错误非常普遍，它甚至有一个名字叫“检察官谬误”
   </strong>
   （prosecutor’s fallacy）。
  </p>
  <p class="graf graf--p">
   但现在你已经明白了这个方法是多么有用。下次再涉及冤狱时你的博客文章可以比法官更有说服力。
  </p>
  <p class="graf graf--p">
   实际上贝叶斯算法可以通过数学计算完成，而无需那个四重表。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    最后
   </strong>
  </p>
  <p class="graf graf--p">
   在《1984》中，大洋国的政府宣传机构负责篡改历史记录和其他文件，以表达政府的宣传议程。必要时真理部还会提出如“2+2=5”。
  </p>
  <p class="graf graf--p">
   这种情况直到今天依然存在。看一个最近的例子《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%ae%a3%e4%bc%a0%e5%a4%a7%e6%96%b9%e5%90%91%e4%bb%a4%e4%bc%81%e4%b8%9a%e5%aa%92%e4%bd%93%e5%bf%98%e8%ae%b0%e4%ba%86%e5%b9%bc%e5%84%bf%e7%ba%a7%e6%95%b0%e5%ad%a6%e9%a2%98%e7%9a%84%e5%81%9a%e6%b3%95/" href="https://www.iyouport.org/%e5%ae%a3%e4%bc%a0%e5%a4%a7%e6%96%b9%e5%90%91%e4%bb%a4%e4%bc%81%e4%b8%9a%e5%aa%92%e4%bd%93%e5%bf%98%e8%ae%b0%e4%ba%86%e5%b9%bc%e5%84%bf%e7%ba%a7%e6%95%b0%e5%ad%a6%e9%a2%98%e7%9a%84%e5%81%9a%e6%b3%95/" rel="noopener noreferrer" target="_blank">
    宣传大方向令企业媒体忘记了幼儿级数学题的做法
   </a>
   》。
  </p>
  <p class="graf graf--p">
   不幸的是，在互联网时代，人们想要弄清一个报道是真知识还是反知识，已经成为极其困难的事。
  </p>
  <p class="graf graf--p">
   正如本系列6篇文章所演示的，不论是专家还是媒体，都不可完全信任，政客更不用说了，他们满嘴都是谎言。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在反科学的偏见已经进入公共语话和网络世界的今天，只有最严谨的批判性思考才能保住真相
   </strong>
   。
  </p>
  <p class="graf graf--p">
   批判性思考是一种积极的、持续不断的过程，并用新的信息去更新自己的知识。就如上述演示中希望呈现的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    花费时间在评估表述上并不意味着很好地利用了时间，它还应当被看作隐藏在我们所有人讨价还价过程中的一部分
   </strong>
   。
  </p>
  <p class="graf graf--p">
   从前，收集信息并加以分析所需要的时间从数个小时到数周不等，而现在，只需几秒钟便能完成。以前，人们要去图书馆和距离很远的档案馆，或者在厚厚的书本中查阅所需的信息，这些都要花费很长的时间。今天在这方面已经节省了大量时间。
  </p>
  <p class="graf graf--p">
   我们都需要
   <strong class="markup--strong markup--p-strong">
    把那种隐性的讨价还价转化为显性
   </strong>
   。
  </p>
  <p class="graf graf--p">
   隐性的讨价还价指的就是把自己在获取信息方面节省出来的时间用于进行恰当的信息核实工作。
  </p>
  <p class="graf graf--p">
   正如你很难再相信那些曾经向你撒过谎的人一样，如果你自己的一半知识都属于反知识，那么你也很难再相信自己。
  </p>
  <p class="graf graf--p">
   实际情况是，现在反知识在整个互联网上俯拾皆是。
  </p>
  <p class="graf graf--p">
   与大量不具备确定性的事情相比，如果我们能够知道差不多数量的具有确定性的事，便会获益良多。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    反知识和错误信息的成本可能会非常高昂，甚至可以改变历史走向
   </strong>
   。就如我们曾经在《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ad%a6%e6%83%95%e5%be%ae%e5%a6%99%e7%9a%84%e5%8a%9d%e5%af%bc%e6%9c%af%ef%bc%9a%e4%bc%aa%e7%9c%9f%e7%9b%b8-%e6%96%b0%e9%97%bb%e6%9c%ba%e6%9e%84%e5%92%8c%e8%af%bb%e8%80%85%e7%9a%86/" href="https://www.iyouport.org/%e8%ad%a6%e6%83%95%e5%be%ae%e5%a6%99%e7%9a%84%e5%8a%9d%e5%af%bc%e6%9c%af%ef%bc%9a%e4%bc%aa%e7%9c%9f%e7%9b%b8-%e6%96%b0%e9%97%bb%e6%9c%ba%e6%9e%84%e5%92%8c%e8%af%bb%e8%80%85%e7%9a%86/" rel="noopener noreferrer" target="_blank">
    伪真相
   </a>
   》中分析的那样。
  </p>
  <p class="graf graf--p">
   本系列提供的简单知识完全可以帮助您走在撒谎者和轻信者的前面。⚪️
