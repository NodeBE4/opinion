  <h1 class="entry-title">
   鄙视带来的优越感
  </h1>
 </header>
 <div class="entry-meta">
  <span class="byline">
   <a href="https://www.iyouport.org/author/don-evans/" rel="author" title="由McCaffrey发布">

    互联网时代/媒介效应
   </a>
  </span>

 </div>
 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <strong>
      <em>
       我们在社交网络上很难看到赞誉，在任何语种中都是如此；观察中国社交网络的这几年来我们发现，很多政治评论家对”谁的方法不行”梳理得头头是道，但从来没听说过他们赞赏”谁的方法最有希望”，难道没有任何希望吗？这肯定是不合理的 … 这种生态来自哪里？它造成了什么样的危机？
      </em>
     </strong>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   如今这个时代你会更容易发现，很多人倾向于努力批评别人而不是表达赞赏、钦佩或合作的欲望。其实这不一定意味着被批评者真的很差，而是评论者在努力表现他们自己的智商优越感。
  </p>
  <p class="graf graf--p">
   由于已经有很多证据可以支持这一结论，它已经不言而喻了：即 否定意见能令人看起来更聪明。
  </p>
  <p class="graf graf--p">
   在20世纪80年代，哈佛大学研究员特雷莎·阿马比尔（Teresa Amabile）从“纽约时报 ”的书评部分中找出了两篇文学评论：一篇是正面赞赏的，一篇是否定的，她把这两篇文章向55名学生展示。学生们普遍得出的结论是：表达批评意见的作者比表达赞美意见的作者更加聪明、看起来更有说服力。
  </p>
  <p class="graf graf--p">
   然而，事实上，这两篇文章是同一位评论家写的，而且是同一评论的改编版本。约翰·斯图亚特·穆勒（John Stuart Mill）写道，
   <strong class="markup--strong markup--p-strong">
    “并不是那个在其他人绝望的时候满怀希望的人，而是那个在其他人都满怀希望时表达绝望的人，被一大群人视为圣人。
   </strong>
   ”
  </p>
  <p class="graf graf--p">
   我们不得不为这种状况心冷。鉴于当下弥漫的虚无主义情绪 — 它正在迅速令所有人变得虚弱和悲情 — 面对如此的环境，除了客观审视现实并寻找出路之外，别无他法。如果越来越多的人深陷虚无主义，我们拥有的可联合基数就会迅速下降，直至积极人士彻底变成被边缘的小众，甚至变成“那些愚蠢的理想主义废物”。
  </p>
  <p class="graf graf--p">
   中文 Twitter 是很容易观察到这种氛围的地方，看看这多年来那些转发量最高的和最著名的政治评论者都是什么样的口吻，就能明白了。事实上社交媒体本身正是最容易观察到这一现象的平台，不论在任何语种群体内，它都是突出的。
  </p>
  <p class="graf graf--p">
   要想扭转这一局面可能非常困难。在某种程度上，这是因为责备实际上比赞美更为罕见。在过去的50年里，跨文化研究已经证明了一种被称为
   <strong class="markup--strong markup--p-strong">
    语言积极性偏见
   </strong>
   的现象：人类的言语中充满了“伟大”、“可爱”和“令人惊讶”之类的词汇，而像“可怕”、“丑陋”这样的词汇并不那么频繁出现。这大概是因为人们主要使用语言作为一种将自身和周围社区更紧密地联系在一起的手段，从而提高了创造社区感的词语使用频率。批评和贬低的词汇由于它们不是常态因而很容易被突出，这反过来便向读者或听众发出了信号，表明他们将自己与群体区别开来。
  </p>
  <p class="graf graf--p">
   因此，作为一种可选择性的否定不仅会让你看起来更具洞察力，而且能享有充分表现力的机会：这些批评的词汇为表达者自己提供了自我满足感。
  </p>
  <p class="graf graf--p">
   Yelp 甚至因此为个人提供了一个平台专门吐槽用，无论是创造性地还是仅仅自以为是的。你可以理解为中国的“大众点评”，只是更突出地专注于刺激激烈的反馈。Greyhound buses 倒是的确引发了一些创造性的批评。2012年，一位名叫 Sonia B.的用户键入了以下内容：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    Greyhound, Greyhound
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    You’re not that fast,
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    If you were in a race you’d probably finish last.
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    I use you sometimes when I’m going far,
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    Even though your service is kind of sub-par.
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    But when I consider gas prices these days,
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    You really often are the cheapest way.
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    (Especially the advance webfare)
   </em>
   .
  </p>
  <p class="graf graf--p">
   我敢说这样的诗歌几乎不会被任何正规媒体发表。这正是激励 Yelp 评论家的一个主要因素：不一定表达对产品和服务的热情观点，而是
   <strong class="markup--strong markup--p-strong">
    表达那种最能令自己涨粉的观点
   </strong>
   。在2014年 Fast Company 的采访中，Yelp 的副总裁说，“如果你在 Yelp 上写下很棒的东西，你就会知道很多人都能读到它们。你会发出自己的声音。Yelp 就是你的扩音器。“
  </p>
  <p class="graf graf--p">
   其实每个社交媒体平台都是“扩音器”，它们都在
   <strong class="markup--strong markup--p-strong">
    史无前例地激发所有人的自恋情绪和表现欲
   </strong>
   。
  </p>
  <p class="graf graf--p">
   更多的扩音器意味着更多的表现机会。但是，无疑，被放大的悲观虚无情绪表达会对当下的反抗形式造成严重的影响 — — 因为吐槽是无效的，即便吐槽对象满身都是梗，也是一样，
   <strong class="markup--strong markup--p-strong">
    吐槽永远不是解决方案，想办法扳倒邪恶才是
   </strong>
   。
  </p>
  <p>
   <img alt="" class="aligncenter wp-image-668 size-large jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-9.png?resize=1024%2C667&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-9.png?resize=1024%2C667&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-9.png?resize=300%2C195&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-9.png?resize=768%2C500&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-9.png?w=1067&amp;ssl=1 1067w" data-recalc-dims="1" height="667" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-9.png?resize=1024%2C667&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>
   <noscript>
    <img alt="" class="aligncenter wp-image-668 size-large" data-recalc-dims="1" height="667" sizes="(max-width: 1024px) 100vw, 1024px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-9.png?resize=1024%2C667&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-9.png?resize=1024%2C667&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-9.png?resize=300%2C195&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-9.png?resize=768%2C500&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-9.png?w=1067&amp;ssl=1 1067w" width="1024"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   然而，社交媒体的表现欲刺激让绝大多数人的力气都被浪费在吐槽上了。
  </p>
  <p class="graf graf--p">
   发泄的自由感觉赋予了某种权力，它的确令人兴奋，但是当在社交媒体这种基于声誉的劳动力市场上释放时，它就变成了一种强大的气场，以至于观察人士会
   <strong class="markup--strong markup--p-strong">
    误认为
   </strong>
   整个社会都没有改变的智慧、甚至没有意图。事实上很可能并非如此，反抗的智慧和意图只是被虚无的氛围所掩盖了。而当局对批判者的抓捕和压制更加促进了人们对吐槽之能力的误会。
  </p>
  <p class="graf graf--p">
   如果说 Yelp 提供的是一个准文学或艺术自我表达的机会，那么 Uber 或 Ziosk（一个鼓励人们给连锁餐厅的服务打分的应用程序）就是在要求用户主要通过数字表达自己，有时伴随着现成的评论可以从列表中选择。用户并不是必须填写评价，正相反，是公司提示或要求用户反馈，好像一个人的满意或不满是公司最关心的问题。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    评级并不只是衡量情绪反应：它们在创造情绪反应
   </strong>
   。这被称为单纯测量效应，这是一种社会学的怪癖，当被要求提出意见时超个人主义自我就会被鼓动起来。其实中国网络用户在很多年前就清楚地阐述过这一效应，他们将翻阅微博讽刺为“批奏折”，这是非常准确的形容。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    效果足以改变一个人的行为
   </strong>
   ，这是心理学阐述的基本理论之一。在一项研究中，参与者只是被问及他们是否打算投票，就可以让更多人加入投票。随着用户习惯于被问及他们对服务或他人/群体的看法，他们就会被激发出更强烈的意见。最少最明显的是，市场交易本身正在逐渐消失淡化，而消费产品或服务的情感体验更加突出了。
  </p>
  <p class="graf graf--p">
   Yelp 是第三方平台，而 Ziosk 和 Uber 则直接控制工人是否能保住工作。然而，将一个人的感受转化为评论的过程基本相同 — 如果没有文本元素，这些感受会更快地转化为硬数据。
  </p>
  <p class="graf graf--p">
   只要用心体察，几乎每个人都能了解到自己在不同时间的不同情绪状态下对同一件事的看法会大相径庭。比如你看到一个人在社交网络上讽刺“知识分子都是最愚蠢和无能的人”，但你不会看到这位评论者刚刚和一位知识分子发生了争执，以及很可能是源于彼此间的思考方式不同而产生的误会。正因为鄙视的评价非常容易吸引目光，于是你几乎在所有时候都能看到这类以偏概全的点评，相反，当人们对某些事物满意的时候，为了保持公开场合的批判者智商优越感，他们就不会告诉你。
  </p>
  <p class="graf graf--p">
   暂时的个人好恶不仅会被社交网络的扩音器放大为生态，并且，频繁的表述还很容易收窄你自己的视野。当你强调“知识分子一无是处”的时候很难记得这种感觉只是来自于你自己曾经遇到的不超过5个伪知识分子的弊端，而人们都有言行合一的下意识，于是这一表述将明确阻断你与知识分子群体的广泛合作。如果你是一位活动家，那么你将因此失去一大群结盟力量。
  </p>
  <p class="graf graf--p">
   就如教学评估以学生评价教授的方式调动了种族主义、性别歧视和其他形式的歧视，让顾客有权选择工人的平台也提供了类似的机会，不仅放纵了偏见，而且
   <strong class="markup--strong markup--p-strong">
    让他们的世界变成一个封闭社区 — — 只能体验到绝对舒适的无摩擦的狭窄隧道
   </strong>
   。
  </p>
  <p class="graf graf--p">
   您一定熟悉中国的小圈子形式的社会交往，这些圈子最初有可能较大，但后期会变得越来越小，圈内人无情地淘汰所有不一致的意见（那些意见被称为“傻逼”）使其成员无限接近同化。IYP 解释过为什么中国式的圈子不是社区，因为社区不会轻易排斥任何人。更多详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/kn-03192019-5ba092bf5c40" href="https://medium.com/@iyouport/kn-03192019-5ba092bf5c40" rel="noopener noreferrer" target="_blank">
    异议的消亡：为什么网络变得同质化
   </a>
   》《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/kn-03192019-5ba092bf5c40" href="https://medium.com/@iyouport/kn-03192019-5ba092bf5c40" rel="noopener noreferrer" target="_blank">
    无用的觉醒
   </a>
   》《
   <a class="markup--anchor markup--p-anchor" data-href="https://pao-pao.net/article/773" href="https://www.iyouport.org/%E7%A4%BE%E4%BA%A4%E5%AA%92%E4%BD%93%E7%9A%84%E9%9A%A7%E9%81%93%E7%94%9F%E6%80%81%E5%92%8C%E5%BF%83%E7%90%86%E6%93%8D%E6%8E%A7/" rel="noopener noreferrer" target="_blank">
    社交媒体的隧道生态和心理操控
   </a>
   》
  </p>
  <p class="graf graf--p">
   并且，这事关民主体制的延续。
  </p>
  <p class="graf graf--p">
   你不喜欢互联网上的嘈杂？认为很多人都是愚蠢的？好吧，但是必须知道，
   <strong class="markup--strong markup--p-strong">
    那些引起你不快的低效元素正是民主和社区的根基。有效的民主文化必须包含与某种混乱和尴尬的不期而遇。社区也一样
   </strong>
   。人们必须能够接触到一些他们本心并不想接触的东西，必须能听到并不愿意听到的话和观点，然而这个时代，你需要花点力气、在意识的推动下去主动寻找，去努力，而且稍不留神便会滑入个人中心主义的黑洞。更多详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://pao-pao.net/article/833" href="https://iyouport.weebly.com/iyp_blog/7277728" rel="noopener noreferrer" target="_blank">
    从媒体卖乖到政治品牌化（二）消费者经济时代脆弱的民主
   </a>
   》
  </p>
  <p class="graf graf--p">
   当民主派人士公开互相抨击彼此的时候，他们是无从联合的，不仅如此，这一外观还会给围观的民众一种“你们这个群体整体都不行”的错误印象，要知道，民主派人士最有望动员的新生力量正在这些围观者群体中。
  </p>
  <p class="graf graf--p">
   当民主派人士在内部互相批驳中消耗了大部分精力时，他们的敌人正在竭尽全力吸纳新的联合力量以壮大自身。⚪️
  </p>
