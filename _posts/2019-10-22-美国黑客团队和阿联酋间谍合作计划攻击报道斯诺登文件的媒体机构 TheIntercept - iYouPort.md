    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" rel="noopener noreferrer" target="_blank">

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    DarkMatter 将前美国国家安全局黑客和其他美国情报人员和退伍军人、与阿联酋分析人员一起，专门攻击国内外政治异议人士的电子设备，其中攻击目标还包括美国公民
   </strong>
   。
  </p>
  <p class="graf graf--p">
   代号为 Project Raven 的攻击行动始于巴尔的摩，也就是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E6%AD%BB%E4%BA%8E%E8%87%AA%E5%B7%B1%E5%88%B6%E9%80%A0%E7%9A%84%E6%AD%A6%E5%99%A8/" href="https://www.iyouport.org/%E6%AD%BB%E4%BA%8E%E8%87%AA%E5%B7%B1%E5%88%B6%E9%80%A0%E7%9A%84%E6%AD%A6%E5%99%A8/" rel="noopener noreferrer" target="_blank">
    美国国安局的后院
   </a>
   ——
   <strong class="markup--strong markup--p-strong">
    该项行动专注于攻击持不同政见者和批评阿联酋政府的任何人。
   </strong>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2016/10/24/darkmatter-united-arab-emirates-spies-for-hire/" href="https://theintercept.com/2016/10/24/darkmatter-united-arab-emirates-spies-for-hire/" rel="noopener noreferrer" target="_blank">
    2016 年 Intercept 的文章
   </a>
   透露，总部位于马里兰州的技术公司 Cyber​​Point 组建了一支美国人的团队，以达成一项合同，目标是磨练阿联酋最新崭露头角的骇客攻击和监视能力，让一些新兵感觉不安。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    就是这个 Cyber​​Point 团队，其中很多人后来被 DarkMatter 挖走了
   </strong>
   。DarkMatter 是一家与阿联酋政府关系密切的公司，
   <strong class="markup--strong markup--p-strong">
    国家电子安全局 NESA
   </strong>
   ，总部只有两层楼，该部门的地位相当于美国国家安全局（前身国家电子安全局，后来成为信号情报局）。
  </p>
  <p class="graf graf--p">
   McLaughlin 的一位消息人士称这一事件是阿联酋政府的“恶意收购”。
  </p>
  <p class="graf graf--p">
   McLaughlin 随后
   <a class="markup--anchor markup--p-anchor" data-href="https://foreignpolicy.com/2017/12/21/deep-pockets-deep-cover-the-uae-is-paying-ex-cia-officers-to-build-a-spy-empire-in-the-gulf/" href="https://foreignpolicy.com/2017/12/21/deep-pockets-deep-cover-the-uae-is-paying-ex-cia-officers-to-build-a-spy-empire-in-the-gulf/" rel="noopener noreferrer" target="_blank">
    为“外交政策”杂志撰写的文章
   </a>
   详细介绍了 DarkMatter 的美国间谍在构建阿联酋情报机构方面的重要性。
  </p>
  <p class="graf graf--p">
   NESA 将继续成为 Project Raven 的主要“客户”，负责传递团队和组织的目标和攻击计划。
  </p>
  <p class="graf graf--p">
   2016 年的报告揭示了 DarkMatter 与阿联酋政府之间的联系，目标是美国媒体 Intercept。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “当[McLaughlin 的第一篇]文章出现时，其中提到了 DarkMatter，所以我们不得不对老虎团做出回应”，该消息来源说，使用的是响应团队的行话，“任何时候 NESA 或 DarkMatter 想要攻击任何媒体，我们都会被引入以开发攻击目标列表。”
  </p>
  <p class="graf graf--p">
   阿联酋领事馆没有回复评论请求。
  </p>
  <p class="graf graf--p">
   熟悉此事的第二个人证实了针对 TheIntercept 的攻击讨论，
   <strong class="markup--strong markup--p-strong">
    并称参与会谈的人中包括最大牌的美国籍 DarkMatter 执行官马克·拜尔（Marc Baier）
   </strong>
   。这个人没有说明讨论是否导致了决定。
  </p>
  <p class="graf graf--p">
   第二个消息来源指出，DarkMatter 首席财务官 Samer Khalife 将一些美国人目标从 DarkMatter 转移到一家新公司 Connection Systems。
  </p>
  <p class="graf graf--p">
   而后者这家新公司成立的目的是创造出 “DarkMatter 不再代表阿联酋政府进行监视和网络操作”的
   <strong class="markup--strong markup--p-strong">
    假象
   </strong>
   —— 但 Khalife 把他的兄弟安插在新公司里作为该公司的名义老板。
  </p>
  <p class="graf graf--p">
   LinkedIn 显示，
   <strong class="markup--strong markup--p-strong">
    Connection Systems 目前雇佣了多名前 DarkMatter 员工
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Khalife 和 Baier 都没有回复评论请求。
  </p>
  <p class="graf graf--p">
   目前尚不清楚是否曾对“TheIntercept”进行过攻击。
  </p>
  <p class="graf graf--p">
   但是，攻击定位于 2016 年发生，因此如果有恶意邮件存在的话可能被垃圾邮件过滤器阻挡或被丢弃了。
  </p>
  <p class="graf graf--p">
   第三个熟悉 Project Raven 的人说，他们不知道有任何针对 Intercept 或其员工的企图，而且在没有吸引美国同行的注意力的情况下 DarkMatter 的常驻 NESA 黑客不太可能尝试这种协调攻击。
  </p>
  <p class="graf graf--p">
   不过，该消息来源指出，
   <strong class="markup--strong markup--p-strong">
    阿联酋国民对美国出版物进行秘密攻击在技术上是可行的。
   </strong>
  </p>
  <p class="graf graf--p">
   Intercept 联系的其他前 Project Raven 成员拒绝发表评论，一些人引用了联邦调查局的调查。联邦调查局也拒绝发表评论。
  </p>
  <p class="graf graf--p">
   DarkMatter 营销主管 Priscilla Dunn 在给 The Intercept 的电子邮件中否认报道。但他没有明确否认那次讨论的发生，并拒绝评论 Project Raven 的存在或该公司与 NESA 的合作，他的回复的东西基本是广告台词，就不翻译了。
  </p>

  <p class="graf graf--p">
   消息人士说，该讨论在 Project Raven 阿布扎比总部的一个会议室里进行，当时他们在场。
  </p>
  <p class="graf graf--p">
   房间的其他人包括两名 NESA 分析师 Al Anood Al Kaabi 和 Fatema Mohammed Al Shehhi、还有该团队的美国运营总监 Ryan Adams，此人是美国空军退伍军人。此人也没有回复评论请求。
  </p>
  <p class="graf graf--p">
   根据消息来源和 Cole 的说法，⚠️各个 Project Raven 分析师针对新闻文章制定的
   <strong class="markup--strong markup--p-strong">
    攻击目标列表不仅包括文章作者，还包括与该作者有关的其他人，包括可能贡献信息的任何人、编辑，甚至分享文章的人
   </strong>
   。
  </p>
  <p class="graf graf--p">
   消息来源还说，
   <strong class="markup--strong markup--p-strong">
    有时不仅扫描一篇文章的作者，还有“该作者的男朋友、女朋友、或兄弟…
   </strong>
   …从原始目标中延伸出两到三个选项。
  </p>
  <p class="graf graf--p">
   编制实际目标列表将取决于各个 Project Raven 分析师。
  </p>
  <p class="graf graf--p">
   在设定攻击目标列表之后，
   <strong class="markup--strong markup--p-strong">
    下一步就是涉及由 NESA 分析师执行的文本、社交媒体、和电子邮件的鱼叉式网络钓鱼攻击，被称为 “试图平息问题或重新牵引公众注意力”以远离负面文章
   </strong>
   。
  </p>
  <p class="graf graf--p">
   网络钓鱼邮件旨在诱骗收件人采取某些行动，而在 DarkMatter 的案例中，通常涉及尝试让攻击目标以 PDF、RTF、和 Microsoft Word 等常见格式打开恶意文档。
  </p>
  <p class="graf graf--p">
   和所有钓鱼攻击一样，
   <strong class="markup--strong markup--p-strong">
    此类攻击是在事先侦察了目标对象的数字弱点后针对性进行的。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “包括目标的技术足迹、任何社交媒体账户、任何偏好、移动设备、电子邮件、社交媒体ID等等，就是重要的东西”，消息来源说。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    这就是为什么保护性措施的第一步应该是《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a4%a7%e6%b8%85%e6%b4%97%ef%bc%9a%e5%8f%8d%e4%be%a6%e5%af%9f%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" href="https://www.iyouport.org/%e5%a4%a7%e6%b8%85%e6%b4%97%ef%bc%9a%e5%8f%8d%e4%be%a6%e5%af%9f%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" rel="noopener noreferrer" target="_blank">
     大清洗
    </a>
    》在这里看到具体步骤
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    一旦目标的在线生活被映射出来，“你就可以将这些信息与[计算机]漏洞利用”配对，这些漏洞来自 DarkMatter 的“内部漏洞利用储存库”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️如果客户“真的想干的更狠点”，Project Raven 会从第三方购买漏洞，包括所谓的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E9%9B%B6%E6%97%A5%E6%BC%8F%E6%B4%9E%E5%92%8C-cia-%E9%BB%91%E5%AE%A2%E5%B7%A5%E5%85%B7/" href="https://www.iyouport.org/%E9%9B%B6%E6%97%A5%E6%BC%8F%E6%B4%9E%E5%92%8C-cia-%E9%BB%91%E5%AE%A2%E5%B7%A5%E5%85%B7/" rel="noopener noreferrer" target="_blank">
    零日漏洞
   </a>
   。
  </p>
  <p class="graf graf--p">
   这种攻击特别危险，难以防范。一旦中招此类漏洞可能会授予对目标计算机的持久访问权限。
  </p>
  <p class="graf graf--p">
   根据 Cole 的说法，
   <strong class="markup--strong markup--p-strong">
    攻击记者通常是达到目的的手段而不是目标本身
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果来自 The Intercept 的人正在与他们的消息来源进行对话，跟踪 Intercept 的目标就可以找到消息来源。Project Raven 骇客经常通过初始消息冒充潜在的消息来源或支持者，以建立与攻击目标的融洽关系，只为稍后将陷阱埋在里面。”
  </p>
  <p class="graf graf--p">
   准确说他们的终极目标是透明度革命中的爆料人。
  </p>
  <p class="graf graf--p">
   ⚠️“一旦你获得了信任，你就可以将[恶意]文件注入进去 ……如果你以安全的方式建立通信，那么可能会有人会开启漏洞利用。”
  </p>
  <p class="graf graf--p">
   如果目标前往阿联酋，Project Raven 将利用他们的“主场优势”，⚠️
   <strong class="markup--strong markup--p-strong">
    有时甚至会直接闯入目标的酒店房间，消息来源解释说：“如果目标在国内，我们只需要去和服务提供商打个招呼就行了，‘嗨我需要访问…’。或者直接入侵目标在酒店的无线接入”。
   </strong>
  </p>
  <p class="graf graf--p">
   该消息来源说，⚠️Project Raven 要求阿联酋国家安全部门的间谍
   <strong class="markup--strong markup--p-strong">
    “假扮成维修工，在你住的酒店里，悄悄将你的笔记本电脑充电器换成看起来很相似的充电器”（恶意女仆攻击），以便入侵目标的设备。
   </strong>
  </p>
  <p class="graf graf--p">
   Project Raven 针对美国或其他国家新闻记者的做法在路透社的报道中已经披露，该报告提到在阿布扎比的 DarkMatter 总部“攻击目标列表中有三个……美国名字”。这些名字尚未确定。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “2012年，DarkMatter 的主要目标之一是 Rori Donaghy ……一位英国记者和人权活动家，撰写了批评该国人权记录的文章。”
  </p>
  <figure aria-describedby="caption-attachment-5918" class="wp-caption aligncenter" id="attachment_5918" style="width: 1210px">
   <img alt="" class="size-full wp-image-5918 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/图2.png?resize=1100%2C700&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/图2.png?w=1210&amp;ssl=1 1210w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/图2.png?resize=300%2C191&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/图2.png?resize=768%2C489&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/图2.png?resize=1024%2C652&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/图2.png?resize=1100%2C700&amp;ssl=1 1100w" data-recalc-dims="1" height="700" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/图2.png?resize=1100%2C700&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </figcaption>
  </figure>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    美国人
   </strong>
  </p>
  <p class="graf graf--p">
   消息来源说，虽然他们个人拒绝帮助瞄准美国个人和组织，但是这项工作最终以这种或那种方式完成了。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “不幸的是，当你已经生成一个攻击目标列表并且你正在和一个老虎团合作时，我随时可以说我自己没有攻击列表上某个人，但是团队里其他人做了”。
  </p>
  <p class="graf graf--p">
   消息来源说，普遍
   <strong class="markup--strong markup--p-strong">
    的感觉是，“这个就他妈的是客户现在要的人，我们需要这样做。客户不关心公民身份是什么。”
   </strong>
  </p>
  <p class="graf graf--p">
   从事针对非美国目标的类似行动的 Cole 表示，他并没有参与或直接了解针对美国人或美国计算机系统攻击的行为，但是他回忆起当时有关美国同事对这些努力的警告。
  </p>
  <p class="graf graf--p">
   Cole 还表示，他担心美国人会成为美国援助黑客的目标，尽管上述人士保证，“有关美国公民的任何信息被吸入该集团的骇客入侵数据库是偶然的”。
  </p>
  <p class="graf graf--p">
   Cole 说，他现在认为这种收集是故意的，并且 Raven 项目的领导层“错误地声称”美国政府已经知道任何对美国人的所谓偶然监视，并且这些数据经常被从 DarkMatter 计算机中清除。
  </p>
  <p class="graf graf--p">
   他说，“我怀疑了一段时间，我正在关注这件事。我的理解是[被收集的美国人数据]被传达回美国情报机构。这就是我们当时所认为的，但我们后来发现这是一个谬误。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “总有不同的理由和借口，” Cole 说，就如承诺清除美国人的数据却未能实现。 “我被告知的事和我观察到的事完全相反。”
   </strong>
  </p>
  <p class="graf graf--p">
   Cole 认为，同样令人担忧的是美国人在 Project Raven 中的角色转变。
  </p>
  <p class="graf graf--p">
   他们表面上出现在 DarkMatter 的阿布扎比​​别墅中进行培而。在实践中，出于法律原因，这意味着他们会根据指导方针给出非常具体的指示，他们可以做“除了键盘工作的一切”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    他们 “提供步骤和说明”，用于在不亲自执行任何步骤的情况下攻击 NESA 想要的任何目标。全球范围内。
   </strong>
  </p>
  <p class="graf graf--p">
   根据熟悉 DarkMatter 关于针对 Intercept 的讨论的第二个消息来源说，即使当美国公民不是明确的目标时，合法性问题也一直是对 DarkMatter 的长期担忧;
  </p>
  <p class="graf graf--p">
   入侵居住在美国的非美国公民的 Facebook 或 Gmail 帐户仍然意味着可能破坏在美国的服务器。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    路透社的报道还显示，美国国家安全局虽然没有直接参与 DarkMatter 的日常运营，但是，“已批准并定期听取 Raven 的活动简报。”
   </strong>
  </p>
  <p class="graf graf--p">
   针对记者的攻击只是美国人被 Project Raven 攻击或监视的众多方式之一，Cole 和消息来源说。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    连线电视杂志、国际特赦组织、和人权观察组织，也很可能成为攻击目标 —— 在 NESA 等阿联酋客户的要求下
   </strong>
   。他们告诉 Intercept，但无法回想起细节。⚪️
  </p>
