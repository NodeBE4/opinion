  <p class="p3">
   <span class="s1">
    Guoxinhua
   </span>
   <span class="s3">
    想要教电脑识别回声。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    他和他的同事已经制造了一台大约相当于笔记本电脑尺寸的设备，它发出的声音频率比短笛能支持的最尖锐音高
   </span>
   <span class="s2">
    10
   </span>
   <span class="s1">
    倍。人耳无法听到这样的声音。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    郭的团队将该设备对准一个人并发射这种声音，同时使用其数百个嵌入式麦克风来监听回声。然后，他的团队采用人工智能技术，尝试从单独的反射声中破译人的行为。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    该技术仍处于起步阶段，但他们已经取得了一些明显的初步成果。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    位于中国的武汉理工大学，郭的团队已经对四名不同的大学生进行了麦克风阵列测试，发现
    <b>
     他们可以确定这个人是坐着、还是站着，走路还是摔倒，并且完全准确
    </b>
   </span>
   <span class="s2">
    <b>
    </b>
    ——
   </span>
   <span class="s1">
    他们在论文中这样写道。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    虽然他们仍然需要测试该技术是否适用于更多人，并且它是否可以识别更广泛的行为，但是目前为止的示范成功暗示了一种可以用于监视人类行为的新技术正在出现。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    郭的设备属于一种被称为
    <b>
     人类活动识别
    </b>
    的技术，其中计算机分析信号以弄清楚人们在做什么。这种技术本身不奇怪，就如智能手表上的计步器将加速度计和陀螺仪数据转换为佩戴者的步数。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    现在，像郭这样的研究人员正在设计能够识别更复杂的人类行为的系统，借助更复杂的人工智能技术。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    不止他一家在研究
    <b>
     这种用于监视人类行为的东西
    </b>
    。其中有些人使用声音数据，比如郭；其他人正在开发更好的图像识别算法。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    慕尼黑路德维希马克西米利安大学的计算机科学家
   </span>
   <span class="s2">
    Albrecht Schmidt
   </span>
   <span class="s1">
    说，一些研究人员甚至已经证明了他们可以通过分析环境
   </span>
   <span class="s2">
    Wi-Fi
   </span>
   <span class="s1">
    信号来识别简单的人体姿势。
   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">
    当人体通过这些信号时，他们就会改变这些信号，例如，
   </span>
   <span class="s2">
    Wi-Fi
   </span>
   <span class="s1">
    信号的波动可以揭示人在拍手、打电话或下蹲
   </span>
   <span class="s2">
    ……
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    图：声音监视
   </span>
  </p>
  <p class="p4">
   <span class="s3">
    郭的团队设计了一种算法，
    <b>
     将声音信号与特定的人体姿势联系起来
    </b>
    。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    在设备捕获回声之后，算法首先去除一些环境噪声，然后分析数据的模式。例如，反射声中的频率混合可以提供关于房间内发生了什么的提示。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    该算法利用这些差异来确定人的姿势。郭说，如果你使用更多的麦克风来获取回声，这种算法可以更准确地工作。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    各种姿势之间的音调差异会更加明显。郭的当前阵列使用了
   </span>
   <span class="s2">
    256
   </span>
   <span class="s1">
    个麦克风，体积庞大，可能因为太大而无法大规模生产，因此他们现在正试图在不影响精度的情况下减少麦克风的数量。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    但公司尚未将这些新的行为检测技术发展为商业产品。
   </span>
   <span class="s2">
    Schmidt
   </span>
   <span class="s1">
    说，目前还不清楚它们将被用于干什么。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    郭对此有一些想法。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    他认为一种可能性是将其纳入未来类似
   </span>
   <span class="s2">
    Amazon Echo
   </span>
   <span class="s1">
    的设备中
   </span>
   <span class="s4">
    ⚠️
   </span>
   <span class="s1">
    ，这样他们就可以通过听来监视家里的人的一举一动。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    郭还认为它可以用作图像或视频识别软件的替代品。他说，
   </span>
   <span class="s4">
    ⚠️
   </span>
   <span class="s1">
    声音可以识别相机无法识别的环境中的物体，例如在黑暗的房间或烟雾缭绕的地方。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    郭还认为，基于声音的监控可以比视频监控更好地
   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">
    保护个人隐私
   </span>
   <span class="s2">
    ”
   </span>
   <span class="s1">
    ，这可能使人们更愿意在家中接受这种技术。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    但这是胡扯。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    位于纽约的研究机构
   </span>
   <span class="s2">
    Data
   </span>
   <span class="s1">
    ＆
   </span>
   <span class="s2">
    Society
   </span>
   <span class="s1">
    的技术伦理学家
   </span>
   <span class="s2">
    Jake Metcalf
   </span>
   <span class="s1">
    认为，声音监控可以很容易地与视频监控结合，以创建一种
    <b>
     更高级的监控形式
    </b>
    。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    这是因为尽管研究人员可能会考虑到如何开发该技术，但他们无法控制人们最终如何使用该技术。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    Metcalf
   </span>
   <span class="s1">
    说，健全的监控可能会最终挽救某人的年迈父母的生命，但也可能被国家利用来寻找并迫害正在祈祷的穆斯林，他指的是中国正在对少数族裔维吾尔族人进行的监视。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    黑客也可以重新调整它来监视他人的性生活。
   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">
    这些技术只是在抢着被重新利用，这就是他们的全部观点。它们只是想让自己看起来有价值。
   </span>
   <span class="s2">
    ”
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    不过郭的技术还没有完全实现
   </span>
   <span class="s2">
    ——
   </span>
   <span class="s1">
    他的团队必须先进一步开发这种设备，然后才能将其部署到任何产品中。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    他们需要缩减硬件，他们计划运行更多测试以确保算法适用于更多人和更多场景。例如，
   </span>
   <span class="s4">
    ⚠️
   </span>
   <span class="s1">
    <b>
     他们将从更多不同的人口中收集更多的行
    </b>
   </span>
   <span class="s2">
    <b>
     ​​
    </b>
   </span>
   <span class="s1">
    <b>
     为数据
    </b>
    ，郭说。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    随着这种新的监控手段变得越来越聪明，也许有人会弄清楚如何使用它。
   </span>
  </p>
