  <h3 data-key="421">
   <span data-key="422">
    <span class="prism-token token title important">
     导语：表白和呼吁
    </span>
   </span>
  </h3>
  <div data-key="431">
   <span data-key="430">
    同时作为操场与工厂的互联网，将是未来教育的基础设施：三代人在其中共同学习，构成两种以上文化背景的全球班级，同时学习科学、设计和艺术，工作和研究不分。每一个人都对大数据做贡献式、可另选和
    <strong>
     <span class="prism-token token bold">
      习语化编目（idiotic categorization）
     </span>
    </strong>
    ，加入一个个生产性解释共同体，来推动新司法，用人人手上的解释权，来对冲权力对我们的计算和控制。
   </span>
  </div>
  <div data-key="433">
   <span data-key="432">
   </span>
  </div>
  <div data-key="435">
   <span data-key="434">
    这同时也是“我”与“我们”的共同个人化，由此形成一种新型社会。
   </span>
  </div>
  <div data-key="437">
   <span data-key="436">
   </span>
  </div>
  <div data-key="439">
   <span data-key="438">
    这同时也帮助我们造就一个可以脱自动化的自动社会，一个可被批判的社会，并找到一个现有的未来之外的未来。
   </span>
  </div>
  <div data-key="441">
   <span data-key="440">
   </span>
  </div>
  <div data-key="443">
   <span data-key="442">
    算法统治下，原来的学习-劳动组织正在瓦解。但这也给了每一个人做业余爱好者的机会：既是科学家，也是哲学家，也是艺术家，并且还是设计师。大学里的研究人员与社会中的业余爱好者打通、联合，相互贡献地做open science。这将是我们的主导的教-研格式。
   </span>
  </div>
  <div data-key="445">
   <span data-key="444">
   </span>
  </div>
  <div data-key="447">
   <span data-key="446">
    这也是今天的人文科学数码化的大方向。让人人去获得他们的新知识，还是重建数据库，这一选择已很清楚！我们的下一代绝不会搞错。
   </span>
  </div>
  <div data-key="449">
   <span data-key="448">
   </span>
  </div>
  <div data-key="451">
   <span data-key="450">
    而且不光需要一种以数码研究打底的新文科，我们更需要一种
    <span class="prism-token token bold punctuation">
     **
    </span>
    <span class="prism-token token bold">
     新的智性
    </span>
    <span class="prism-token token bold punctuation">
     **
    </span>
    ，一种能够将数码第三存留重新概念化，并加以重新改造的新的哲学和新的器官术。
   </span>
  </div>
  <div data-key="453">
   <span data-key="452">
   </span>
  </div>
  <div data-key="455">
   <span data-key="454">
    新知识将来自对我们的病和苦难的新的治疗方式，来自新的药罐，正是后者构成了新的做、活和思的方式和理性，使我们从生存走向汇存。
   </span>
  </div>
  <div data-key="457">
   <span data-key="456">
   </span>
  </div>
  <div data-key="459">
   <span data-key="458">
    每一个人带着自己的新知识来参与集体论争，而得到一个集体认识：这就是古希腊人说的
    <em>
     <strong>
      <span class="prism-token token bold">
       aletheia
      </span>
     </strong>
    </em>
    <span class="prism-token token italic">
     。这就是个人的心理-集体跨个人化。这个作为共同论争结果的
    </span>
    <em>
     <strong>
      <span class="prism-token token bold">
       aletheia
      </span>
     </strong>
    </em>
    <span class="prism-token token italic">
     ，会引领我们去找到新的交往规则，而不仅仅成为神谕和猜知。只有人人都成为能被
     <em>
      <strong>
       <span class="prism-token token bold">
        aletheia
       </span>
      </strong>
     </em>
    </span>
    影响的此在时，我们才能形成一个可被批判的社会，也就是说，社会才不会自动化，算法-人工智能才不会架空人类的行动能力。
   </span>
  </div>
  <div data-key="461">
   <span data-key="460">
   </span>
  </div>
  <div data-key="463">
   <span data-key="462">
    为了响应这样一种未来，同济大学人文学院和中国美术学院跨媒体艺术学院为响应数码时代的编目式、贡献式发明的召唤，特向大家推出这个“共知”开源出版项目。
   </span>
  </div>
  <div data-key="465">
   <span data-key="464">
   </span>
  </div>
  <div data-key="467">
   <span data-key="466">
    它作为编目式发明，也许做得还很幼稚，但我们的决心是不可逆的：只有重新发明我们所处的学术机构和编辑机器，在我们自己手里促进知识型的彻底转变，我们自己才会有明天！这首先是自救！
   </span>
  </div>
  <div data-key="469">
   <span data-key="468">
   </span>
  </div>
  <div data-key="471">
   <span data-key="470">
    所有的编目式发明，都是我们的智性的量子一跳，既是器官术革命，也是认识论革命。只有在一次次的破天荒的编目式发明下，我们才能找到自己的新的大草原，成为膜翅目昆虫，成为黄蜂，成为蜜蜂，成为食虫动物，成为植食性动物，哪怕脱自然，也自身也携带着自然，能成全各种其它的自然。我们发明，才能存在。
   </span>
  </div>
  <div data-key="473">
   <span data-key="472">
   </span>
  </div>
  <div data-key="475">
   <span data-key="474">
    只有人人展开这种编目式发明，人民的心智才能凌驾于市场。人人都拿出
    <strong>
     <span class="prism-token token bold">
      发明式慷慨
     </span>
    </strong>
    ，我们才能将社会带进另一个未来。我们深信能将这一接力棒送到下一个单位或个人手里！
   </span>
  </div>
  <div data-key="477">
   <span data-key="476">
   </span>
  </div>
  <div data-key="479">
   <strong>
    <span class="prism-token token bold">
     同济大学人文学院
    </span>
   </strong>
  </div>

   <strong>
    <span class="prism-token token bold">
     中国美术学院跨媒体艺术学院

  <div data-key="483">

  <div data-key="483">
   <hr/>
   <blockquote>
    <p>
     是技术导致了人文！
    </p>
   </blockquote>
   <p>
    –舍赫，《拇指姑娘》
   </p>
   <blockquote>
    <p>
     我进了好几个名校，以便更好地自学。
    </p>
   </blockquote>
   <p>
    –舍赫，《关于科学、文化和时间的对话》
   </p>
   <blockquote>
    <p>
     教育是一个关怀的系统，不光由父母和学校构成，更形成一个社会系统。必须使我们的社会中存在一个关怀系统，去帮助二十岁以前的年青人。这种关怀在英语里叫care,在德语里，海德格尔称它是Sorge\(而不是Besorgen\)，在法语里，是soin。
    </p>
   </blockquote>
   <p>
    –斯蒂格勒，《超物质经济和心理权力》
   </p>
   <p>
    两年前，读过舍赫（Michel Serres）老师的《拇指姑娘》后，我如坐针毡。书中，八十五岁的老先生表达出对90后和00后的深重期望，将我们的文明的赌注押到了这两代人的技术生命的进化上，并宣告他从此将成为90后。夹在中间的那一代的我，真叫难堪！遭遇今年这个毕业季，我突然想到，何不将舍赫老先生、今天的新毕业生（他们都是同济大学的应届毕业生）和我这样的中间一代对于自己所处的时代的各自的感怀拉到一起，做个集体表白，形成一个三明治文本，以开源的方式，作为毕业礼物献给今年的全体毕业生，顺便将这位老先生的伟大期望，托付给大家，共同勉励，也可了却我心中一直的忐忑！你现在看到的，就是这样一个开源出版项目。
   </p>
   <p>
    感谢中国美术学院跨媒学院和同济大学人文学院几任领导层对我的社会项目的一贯的实力支持。也很感动于网络社会研究所的同事们对这个开源出版项目的物质支援。特别感谢刘益红和偏飞设计事务所的贡献-发明式编辑设计，使我们的说了很久、很久的开源出版，终于有了一个看得见的模板。希望它成为众多后来者的有用的垫脚石，不枉诸位的用心之如此良苦！
   </p>
   <p>
    很有幸与同济大学人文学院的王晓芬、钱如意、姜卓含、韩悦音、黄龑、张诗瑶、林云帆七位同学在毕业之际参与这一开源项目！开源的结果也包括：
    <b>
     我原来想要献给他们的这个毕业礼物，转而献给了你们2017届的全体毕业生！
    </b>
    十分荣幸啊！祝大家和你们的家长各自找到自己的平滑面，都能像小鸟那样啁啾，为每一个早晨兴奋，快乐地向自己发布任务！
   </p>
   <p>
    <b>
     陆兴华
     <br/>
     2017年6月30日
