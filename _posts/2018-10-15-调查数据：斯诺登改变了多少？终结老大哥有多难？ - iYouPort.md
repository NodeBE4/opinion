     <em>
      <strong>
       最可怕的并不是死亡。而是即便你放弃优越的生活、甚至舍弃生命安全去选择捍卫正义，结果却是因不被理解，而难以达成改变现实的目的。曾经有中国的“勇敢者”说出这样的话，“我不怕危险，怕的是社会的麻木让我不值得冒险”。但这是真的吗？
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5437 jetpack-lazy-image" data-lazy-sizes="(max-width: 640px) 100vw, 640px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-247.jpeg?resize=640%2C320&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-247.jpeg?w=640&amp;ssl=1 640w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-247.jpeg?resize=300%2C150&amp;ssl=1 300w" data-recalc-dims="1" height="320" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-247.jpeg?resize=640%2C320&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="640"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5437" data-recalc-dims="1" height="320" sizes="(max-width: 640px) 100vw, 640px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-247.jpeg?resize=640%2C320&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-247.jpeg?w=640&amp;ssl=1 640w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-247.jpeg?resize=300%2C150&amp;ssl=1 300w" width="640"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="709d">
   Glenn 说过很多事，其中最令我感觉印象深刻的是他对 Snowden 的采访中的一句话。当时 Snowden 坚持要公开自己的身份，以防止当局以妖魔化知情人的手段拉低真相的影响力。抹黑检举者是最常用的手段，比如把他说成“多变和幼稚”，甚至怀疑他的精神心理状况，就像你知道的那样，从而让当权者抢走更多解释权，以继续愚弄公众。这就是为什么 Snowden 要坚持实名曝光自己，也就是相当于将所有枪口拉向自己。
  </p>
  <p class="graf graf--p graf-after--p" id="b0ba">
   这是在送死。Glenn 问 Snowden，你最担心的是什么？如果不是生命安危的话。Snowden 说，最担心的是公众的反应，担心公众的不理解和冷漠。如果自己放弃了一切为了揭示真相也无法让公众觉醒以捍卫人权、推动变革，那么包括生命在内的所有冒险将变得毫无价值。
  </p>
  <p class="graf graf--p graf-after--p" id="08ac">
   五年很快就过去了。
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/snowden-%E5%90%AF%E7%A4%BA%E4%BA%94%E5%91%A8%E5%B9%B4-%E6%88%91%E4%BB%AC%E7%A6%BB%E8%87%AA%E7%94%B1%E8%BF%98%E6%9C%89%E5%A4%9A%E8%BF%9C-9e7d8d6d03f8" href="https://medium.com/@iyouport/snowden-%E5%90%AF%E7%A4%BA%E4%BA%94%E5%91%A8%E5%B9%B4-%E6%88%91%E4%BB%AC%E7%A6%BB%E8%87%AA%E7%94%B1%E8%BF%98%E6%9C%89%E5%A4%9A%E8%BF%9C-9e7d8d6d03f8" rel="noopener noreferrer" target="_blank">
    当我们再一次回顾这五年中真相对现实究竟带来了哪些促进作用
   </a>
   的时候，依旧抱着某种一言难尽的心情，这恰恰是 Snowden 曾经担心的东西。但是改变还是鲜明的，调查显示已经有更多人开始在乎隐私问题，以及对监视资本主义有了更多的了解。
  </p>
  <p class="graf graf--p graf-after--p" id="98e9">
   虽然这还远远不够，大规模收集仍在继续，情报机构仍然大言不惭地宣称“我们想要更多”，监视技术的能力还在飞速前进，更多的当权者开始效仿美国的棱镜计划对其公民实施侵入性监视，更多的资本注入，监视技术公司疯狂地竞争老大哥的地位。
  </p>
  <p class="graf graf--p graf-after--p" id="6db9">
   但与此同时，公民社会也在快速跟上进度，通过不断的揭露、问责和动员社会参与，以及保护性技术的开发和普及，推动社会变革，侵犯和反侵犯的斗争一直在持续。这是 Snowden 和所有支持者能看到希望的所在。
  </p>
  <p class="graf graf--p graf-after--p" id="09be">
   然而中国呢？如果中国人中诞生了这样的勇气，其结果会是什么？当作为“民主派前辈”的人们模仿当权者的措辞斥责 Snowden“危害国家安全”时，你很难继续说服自己这其中存在希望。
  </p>
  <p class="graf graf--p graf-after--p" id="d135">
   但不可否认，中国的改变已经出现，有更多的人开始认识到隐私权的重要性，
   <strong class="markup--strong markup--p-strong">
    尤其是年轻人
   </strong>
   ，熟悉技术、擅长利用技术以捍卫自己的权利，他们将为变革带来瞩目的动力 — — 在美国也明显如此。虽然目前还很难带动老一代“民主派”人士加入到技术性反抗的队伍中来。
  </p>
  <p class="graf graf--p graf-after--p" id="64d8">
   我们推荐中国社会以美国社会为榜样，学习美国社会积极人士的反抗精神和清晰的认知能力，学习公民社会的专业性。这需要一个过程，美国也是经历了明显的过渡才走到今天的，毕竟当权者对“现实”的垄断是非常强大的。其前提是我们应该抱有希望，一直努力下去。战胜大众的认知失调在任何国家都不是一件容易的事。
  </p>
  <p class="graf graf--p graf-after--p" id="5e26">
   我们一直在支持打破意识形态隔膜，基于人权本身的反抗，这将是联合的促进力。
  </p>
  <p class="graf graf--p graf-after--p" id="aa7e">
   来看看美国的民调，可以清楚的看到这一改变的发生，以及它的缓慢。这是来自皮尤调查中心的数据：
  </p>
  <p class="graf graf--p graf-after--p" id="04ec">
   1、美国人在斯诺登披露后的立即调查显示，美国社会对泄密的影响的认识存在明显分歧，当时甚至大多数人表示“政府应起诉泄密者”。根据皮尤研究中心在斯诺登泄密后几天里进行的一项调查显示，大约一半的美国人（49％）表示，发布的机密信息符合公众利益，而 44％的人则表示这“损害了公众利益”。
  </p>
  <p class="graf graf--p graf-after--p" id="11b6">
   <strong class="markup--strong markup--p-strong">
    调查显示年龄小于30岁的成年人比年长的美国人更有可能认为泄密符合公众利益（60％），而且这些观点中没有任何党派意识形态特征
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="c45d">
   与此同时，54％ 的公众表示政府应该对泄密者提起刑事诉讼，共和党人和民主党人（各占 59％）比独立人士（48％）更高。结果斯诺登于2013年6月（两个月后）被指控“从事间谍活动”。没有任何国家愿意接受他的避难申请，他
   <a class="markup--anchor markup--p-anchor" data-href="https://www.refinery29.uk/wikileaks-sarah-harrison" href="https://www.refinery29.uk/wikileaks-sarah-harrison" rel="noopener nofollow noreferrer" target="_blank">
    在机场被困了一个月
   </a>
   ，最终逃难到俄罗斯。
  </p>
  <p class="graf graf--p graf-after--p" id="84bc">
   2、美国人在接下来的几个月里
   <strong class="markup--strong markup--p-strong">
    对政府监视计划本身的反对程度有所提高
   </strong>
   ，甚至在此之后 — 美国总统巴拉克•奥巴马（Barack Obama）概述了国家安全局数据收集的“变化”之后，不赞成政府收集电话和互联网数据的美国人的比例
   <strong class="markup--strong markup--p-strong">
    从首次调查后的 47％ 增加到次年1月的 53％
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="a69d">
   皮尤调查中心的其他研究也表明，大多数成年人（56％）并不认为法院对电话和互联网数据的收集提供了足够的限制。这是正确的。事实证明即便有立法限制也无法解决大规模监视的问题。因为国安部门经常会拉拢其他机构，以便能对其所作所为大开绿灯 — — 就如在美国众所周知的，国会的情报委员会已经被国安部门拿下，力挺监控。
  </p>
  <p class="graf graf--p graf-after--p" id="4ae0">
   但立法改变至少能推动这样的原则，即 不加区别的大规模监视在隐私受到宪法保护的民主国家没有立足之地。
  </p>
  <p class="graf graf--p graf-after--p" id="ae10">
   此外调查显示，70％ 的美国人认为政府正在将监视数据用于所谓的反恐
   <strong class="markup--strong markup--p-strong">
    以外的目的
   </strong>
   。大约27％ 的人表示，他们认为政府窃听了实际内容的电话通话、阅读了他们私人邮件。（2017 年调查中也出现了近似的数据）
  </p>
  <p class="graf graf--p graf-after--p" id="2a8b">
   3、
   <strong class="markup--strong markup--p-strong">
    政府监控的披露也促使一些美国人改变了他们使用技术的方式
   </strong>
   。在皮尤调查中心 2014 年底和 2015 年初的一项调查中，87％ 的美国人表示他们至少听说过有关政府监控计划的事。在听过这些消息的人中，有 25％ 的人表示，自斯诺登揭露以来，他们已经“大量”或“稍微”改变了他们的技术使用模式。
  </p>
  <p class="graf graf--p graf-after--p" id="d302">
   在另一个问题上，34％ 知道政府监控计划的人表示，他们至少采取了进一步措施来隐藏或屏蔽政府的窥视，例如改变在社交媒体上的隐私设置。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5438 jetpack-lazy-image" data-lazy-sizes="(max-width: 310px) 100vw, 310px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-235.png?resize=310%2C472&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-235.png?w=310&amp;ssl=1 310w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-235.png?resize=197%2C300&amp;ssl=1 197w" data-recalc-dims="1" height="472" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-235.png?resize=310%2C472&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="310"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5438" data-recalc-dims="1" height="472" sizes="(max-width: 310px) 100vw, 310px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-235.png?resize=310%2C472&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-235.png?w=310&amp;ssl=1 310w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-235.png?resize=197%2C300&amp;ssl=1 197w" width="310"/>
   </noscript>
  </p>
  <p>
   4、根据 2014～15 年的调查，美国人普遍认为政府可以监控某些人，但肯定不是全体美国公民。大约八成的成年人（82％）表示，政府可以监控“恐怖分子嫌疑人”的通讯，同等，多数人表示，监控美国领导人和外国领导人的沟通是可以接受的（各占 60％）。同时，57％ 的美国人表示政府监控美国公民的通信是不可接受的。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5439 jetpack-lazy-image" data-lazy-sizes="(max-width: 311px) 100vw, 311px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-236.png?resize=311%2C399&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-236.png?w=311&amp;ssl=1 311w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-236.png?resize=234%2C300&amp;ssl=1 234w" data-recalc-dims="1" height="399" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-236.png?resize=311%2C399&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="311"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5439" data-recalc-dims="1" height="399" sizes="(max-width: 311px) 100vw, 311px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-236.png?resize=311%2C399&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-236.png?w=311&amp;ssl=1 311w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-236.png?resize=234%2C300&amp;ssl=1 234w" width="311"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="018f">
   5、大约一半的美国人（52％）表示担心 2014 年和 2015 年的监控计划，但他们对监控自己数据的担忧更多。大约四分之一的人表示，他们对政府监控他们在搜索引擎、电子邮件和手机上的活动的行为有些担心或非常担心。大约三分之一的受访者表示对监控他们在社交媒体和移动应用上的活动表示同样的担忧。
  </p>
  <p class="graf graf--p graf-after--p" id="85fe">
   6、绝大大多数美国人（93％）表示，谁可以得到并控制他们的相关信息，是很重要的，根据2015 年的报告。与此同时，一个同样大的部分群体（90％）表示，控制收集有关他们的什么样的信息是非常重要的。
  </p>
  <p class="graf graf--p graf-after--p" id="a758">
   然而，很少有美国人认为他们自己对日常生活中的信息收集有很多控制权。只有 9％ 的美国人表示他们对收集的有关他们的信息有很多控制权。在之前的调查中，
   <strong class="markup--strong markup--p-strong">
    91％
   </strong>
   的人同意消费者已经失去了对公司数据收集和使用个人信息的控制权的说法。
  </p>
  <p class="graf graf--p graf-after--p" id="c5b9">
   7、大约 49％ 的人在 2016 年表示，他们对联邦政府保护数据的能力没有信心。大约三分之一的美国人（28％）对政府保护个人记录的能力完全没有信心，而 21％ 的人不太相信。只有 12％ 的美国人对政府保护数据的能力“非常有信心”（49％ 的人表示至少有一点信心）。
  </p>
  <p class="graf graf--p graf-after--p" id="9966">
   美国人对其他机构（如手机制造商和信用卡公司）在保护他们的数据方面相比下更有信心。大约七分之一的手机用户非常有信心（27％），有些（43％）的用户对手机制造商可以保证其个人信息安全有一点信心。同样，大约三分之二的在线成年人非常（20％）或有些（46％）确信电子邮件提供商会保证他们的信息安全。
  </p>

   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5440" data-recalc-dims="1" height="470" sizes="(max-width: 424px) 100vw, 424px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-237.png?resize=424%2C470&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-237.png?w=424&amp;ssl=1 424w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-237.png?resize=271%2C300&amp;ssl=1 271w" width="424"/>
   </noscript>
   <img alt="" class="aligncenter size-full wp-image-5441 jetpack-lazy-image" data-lazy-sizes="(max-width: 310px) 100vw, 310px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/00-3.png?resize=310%2C430&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/00-3.png?w=310&amp;ssl=1 310w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/00-3.png?resize=216%2C300&amp;ssl=1 216w" data-recalc-dims="1" height="430" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/00-3.png?resize=310%2C430&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="310"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5441" data-recalc-dims="1" height="430" sizes="(max-width: 310px) 100vw, 310px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/00-3.png?resize=310%2C430&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/00-3.png?w=310&amp;ssl=1 310w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/00-3.png?resize=216%2C300&amp;ssl=1 216w" width="310"/>
   </noscript>
  </p>
  <p>
   8、大约有一半的美国人（49％）表示他们的个人数据的安全性与前五年（不知道棱镜计划之前）相比更低，根据 2016 年的调查。
   <strong class="markup--strong markup--p-strong">
    50 岁及以上的美国人特别担心他们的数据安全性
   </strong>
   ：58％ 的美国老年人表示他们的数据不如五年前安全。
   <strong class="markup--strong markup--p-strong">
    年轻的成年人不太担心他们的数据不那么安全（技术能力较强，可以更好的保护自己的隐私）
   </strong>
   ; 不过仍然有 41％ 的 18 至 49 岁的人认为他们的个人信息不如五年前那么安全。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5442 jetpack-lazy-image" data-lazy-sizes="(max-width: 310px) 100vw, 310px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-238.png?resize=310%2C430&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-238.png?w=310&amp;ssl=1 310w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-238.png?resize=216%2C300&amp;ssl=1 216w" data-recalc-dims="1" height="430" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-238.png?resize=310%2C430&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="310"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5442" data-recalc-dims="1" height="430" sizes="(max-width: 310px) 100vw, 310px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-238.png?resize=310%2C430&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-238.png?w=310&amp;ssl=1 310w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-238.png?resize=216%2C300&amp;ssl=1 216w" width="310"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="3d8e">
   Snowden 的揭露所产生的国际影响要大于其国内影响，因为
   <strong class="markup--strong markup--p-strong">
    “棱镜计划”是监视全球的，而不仅仅限于美国
   </strong>
   。此举令全世界的目光都集中在了无处不在的政府监控和对恶行“机密”化处理的普遍性上；Snowden 的揭露引发了全球范围内关于数字时代个人隐私问题的深刻讨论，对互联网霸权统治提出了挑战；改变了国与国之间的关系，极大程度上影响了人们对新闻工作者应该与政府权力之间如何较力的更为明确的认识。
  </p>
  <p class="graf graf--p graf-after--p" id="d827">
   <strong class="markup--strong markup--p-strong">
    联合国大会通过不记名投票对德国和巴西所提出的决议予以支持，确认在线隐私属于基本人权。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="f22e">
   所有这一切也并未说明这场硬仗已经以胜利告终。国安部门有着异乎寻常的势力，他们手中的权力甚至会超过人民选举出的最高官员；他们还有着大量的极富影响力的忠实簇拥，愿意以一切代价捍卫国安部门的利益。这其中包括联邦法官 — — 斯诺登泄密后一位联邦法官利用人们对“911”的记忆宣称“国安局监控项目在宪法中应另当别论”。
  </p>
  <p class="graf graf--p graf-after--p" id="b43d">
   还有些美国高管甚至认为不仅 Snowden 应该被指控，包括普利策奖得主 Glenn Greenwald 在内的协助 Snowden 曝光文件的几位新闻记者“也应当被起诉，被扔进监狱”。
  </p>
  <p class="graf graf--p graf-after--p" id="77a1">
   奥巴马政府对说出真相的人的打击力度超过了美国以往各届政府的总和，希望以制造紧张气氛的方式，吓倒任何想说真话的人。随着监视技术的快速进化，政府当局对“机密”的保护能力将比以前更加强大，曝光不再容易；加之对正义的揭露者的惩罚不断出现、并被宣传，寒蝉效应已经铺开。这也是为什么说 Snowden 的启示应该发挥更大的效用，形成真正的彻底的变革，不仅是在美国，而是在全球，
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E4%BA%92%E8%81%94%E7%BD%91%E7%A9%B6%E7%AB%9F%E6%98%AF%E4%BB%80%E4%B9%88-%E6%9D%A5%E8%87%AA-%E7%9B%91%E8%A7%86%E8%B0%B7-%E6%83%8A%E4%BA%BA%E7%9A%84%E5%90%AF%E7%A4%BA-cdd6b5b6bd9d" href="https://medium.com/@iyouport/%E4%BA%92%E8%81%94%E7%BD%91%E7%A9%B6%E7%AB%9F%E6%98%AF%E4%BB%80%E4%B9%88-%E6%9D%A5%E8%87%AA-%E7%9B%91%E8%A7%86%E8%B0%B7-%E6%83%8A%E4%BA%BA%E7%9A%84%E5%90%AF%E7%A4%BA-cdd6b5b6bd9d" rel="noopener noreferrer" target="_blank">
    重新认识这个互联网
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="e9e3">
   决定我们希望生存于什么样的世界的人应该是人类整体，而不是极少数暗中行事的精英人物。推进人们理性思考和决策的能力，这才是透明度革命、活动家和时政工作者的真实意图所在。我们相信变革终将能成为现实，因为我们不会放弃努力。⚪️
  </p>
