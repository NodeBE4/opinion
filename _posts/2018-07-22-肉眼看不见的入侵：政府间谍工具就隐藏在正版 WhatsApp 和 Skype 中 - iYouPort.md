      </strong>

  <p>
   <img alt="" class="aligncenter size-full wp-image-5059 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-176.jpeg?resize=1100%2C413&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-176.jpeg?w=1600&amp;ssl=1 1600w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-176.jpeg?resize=300%2C113&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-176.jpeg?resize=768%2C288&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-176.jpeg?resize=1024%2C384&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-176.jpeg?resize=1100%2C413&amp;ssl=1 1100w" data-recalc-dims="1" height="413" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-176.jpeg?resize=1100%2C413&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5059" data-recalc-dims="1" height="413" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-176.jpeg?resize=1100%2C413&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-176.jpeg?w=1600&amp;ssl=1 1600w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-176.jpeg?resize=300%2C113&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-176.jpeg?resize=768%2C288&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-176.jpeg?resize=1024%2C384&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-176.jpeg?resize=1100%2C413&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="134b">
   网络安全研究人员警告，包括 WhatsApp、Skype 和 VLC Player 在内的
   <strong class="markup--strong markup--p-strong">
    流行软件的合法下载，据称在互联网服务提供商（ISP）层面就被骇客攻击，以传播称为“FinFisher”的高级监控软件
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="1afa">
   FinFisher 已经出售给
   <strong class="markup--strong markup--p-strong">
    全球
   </strong>
   政府和情报机构，可用于窥探网络摄像头、按键、麦克风和网页浏览。以前由
   <a class="markup--anchor markup--p-anchor" data-href="https://wikileaks.org/spyfiles/" href="https://wikileaks.org/spyfiles/" rel="noopener nofollow noreferrer" target="_blank">
    Wikileaks 公布的文件
   </a>
   表明，一个名为“FinFly ISP”的工具可能与此案有关。
  </p>
  <p class="graf graf--p graf-after--p" id="4884">
   数字监控工具由一家名为 Gamma Group 的国际公司兜售，过去曾出售给包括巴林、埃及和阿联酋在内的
   <strong class="markup--strong markup--p-strong">
    专制政权
   </strong>
   （有没有中国不知道）。去年3月，该公司参加了
   <strong class="markup--strong markup--p-strong">
    由英国内政部赞助
   </strong>
   的安全会议。
  </p>
  <p class="graf graf--p graf-after--p" id="4d7c">
   来自网络安全公司 Eset 的专家称，已经在七个国家发现了新的 FinFisher 变种，其中两个被 ISP 中的“中间人”攻击（MitM）作为目标 — — 间谍软件打包在真正软件的下载中。
  </p>
  <p class="graf graf--p graf-after--p" id="f2c9">
   该公司称已被发现的软件包括 WhatsApp，Skype，Avast，VLC Player 和 WinRAR，并补充说“
   <strong class="markup--strong markup--p-strong">
    实际上任何应用都可能以这种方式被滥用
   </strong>
   。”
  </p>
  <p class="graf graf--p graf-after--p" id="4ec4">
   研究发现，当被监视目标正在下载软件时，他们将在不知不觉中被重定向到感染了 FinFisher 的版本。
  </p>
  <p class="graf graf--p graf-after--p" id="41bc">
   下载后，该软件将正常安装 — 但 Eset 发现，它也将隐蔽地与监控工具捆绑在一起。
  </p>
  <p class="graf graf--p graf-after--p" id="88cf">
   隐形感染过程被描述为“肉眼看不见的入侵”。
  </p>
  <p class="graf graf--p graf-after--p" id="c6e8">
   Avast 的发言人说：“攻击者将始终关注最突出的目标。使用恶意软件包装在合法应用程序的官方安装程序中并不是一个新概念，我们对本报告中提到的 PC 应用程序并不感到惊讶。然而最新消息显示，这似乎发生在更高层次上，我们不知道
   <strong class="markup--strong markup--p-strong">
    ISP 是否与恶意软件分销商合作了，或 ISP 的基础设施是否已被劫持
   </strong>
   ……
  </p>
  <p class="graf graf--p graf-after--p" id="96a8">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.welivesecurity.com/2017/09/21/new-finfisher-surveillance-campaigns/" href="https://www.welivesecurity.com/2017/09/21/new-finfisher-surveillance-campaigns/" rel="noopener nofollow noreferrer" target="_blank">
    最新版本的 FinFisher
   </a>
   被发现了
   <strong class="markup--strong markup--p-strong">
    新的定制代码，使其无法被杀毒软件发现
   </strong>
   ，Eset 称之为“战术改进”。它补充说，
   <strong class="markup--strong markup--p-strong">
    一些技巧旨在破坏端到端（E2E）加密软件和已知的隐私工具
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="91d1">
   “看起来他们决定使用最有效的感染方法，从技术角度来看实际上并不难实现，”Eset 的恶意软件研究员 Filip Kafka 告诉 IBTimes UK 说，“由于我们看到的感染比过去监测到的程度更高了，这证明
   <strong class="markup--strong markup--p-strong">
    如今 FinFisher 正在被更广泛地用于监视受影响国家的公民
   </strong>
   。”
  </p>
  <p class="graf graf--p graf-after--p" id="6ddc">
   打破加密已成为世界各国政府的主要兴趣，其中许多政府一直在致力于进行大量通信收集。政治家们经常
   <strong class="markup--strong markup--p-strong">
    在没有证据的情况下
   </strong>
   争辩说，加密通讯“妨碍反恐”。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5060 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-177.jpeg?resize=1100%2C453&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-177.jpeg?w=1920&amp;ssl=1 1920w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-177.jpeg?resize=300%2C123&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-177.jpeg?resize=768%2C316&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-177.jpeg?resize=1024%2C421&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-177.jpeg?resize=1100%2C453&amp;ssl=1 1100w" data-recalc-dims="1" height="453" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-177.jpeg?resize=1100%2C453&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5060" data-recalc-dims="1" height="453" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-177.jpeg?resize=1100%2C453&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-177.jpeg?w=1920&amp;ssl=1 1920w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-177.jpeg?resize=300%2C123&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-177.jpeg?resize=768%2C316&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-177.jpeg?resize=1024%2C421&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-177.jpeg?resize=1100%2C453&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="7590">
   Wikileaks 的一篇关于 FinFly ISP 的
   <a class="markup--anchor markup--p-anchor" data-href="https://wikileaks.org/spyfiles4/documents/FinFly-ISP-Catalog.pdf" href="https://wikileaks.org/spyfiles4/documents/FinFly-ISP-Catalog.pdf" rel="noopener nofollow noreferrer" target="_blank">
    文件
   </a>
   显示了其在 ISP 级别进行监控的能力。该软件的宣传册上写着：“FinFly ISP 能够修补目标即时下载的文件或为流行软件发送虚假的软件更新。”
  </p>
  <p class="graf graf--p graf-after--p" id="24e7">
   它还补充说，它“可以安装在互联网服务提供商的网络上”，并列出了一个用例，显示该软件之前曾由一个未命名的
   <strong class="markup--strong markup--p-strong">
    情报机构部署
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="854e">
   Eset 发现，
   <strong class="markup--strong markup--p-strong">
    其中一个国家内的所有受影响目标都使用相同的 ISP
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="bc24">
   <strong class="markup--strong markup--p-strong">
    “史无前例的“
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="f47a">
   “在泄露的文件中提到的 ISP 级 MitM 攻击技术的部署此前从未被揭示过，”研究人员在他们的分析中说。“这些 FinFisher 的活动代表了一个复杂而隐蔽已久的监控项目，其方法和覆盖范围的组合是史无前例的。不知道它已经被隐藏了多久”。
  </p>
  <p class="graf graf--p graf-after--p" id="1313">
   目前尚不清楚谁是这一骇客攻击活动背后的主使者，但
   <strong class="markup--strong markup--p-strong">
    FinFisher 这种间谍软件几乎完全适合政府、警察或情报机构的使用
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="5b51">
   这不是第一次在欧洲设有办事处的公司与可疑的商业行为挂钩。2013 年，科技公司 Mozilla 在其软件被冒充成 Firefox 浏览器版本后，发送了一封终止声明。它在一篇
   <a class="markup--anchor markup--p-anchor" data-href="https://blog.mozilla.org/blog/2013/04/30/protecting-our-brand-from-a-global-spyware-provider/" href="https://blog.mozilla.org/blog/2013/04/30/protecting-our-brand-from-a-global-spyware-provider/" rel="noopener nofollow noreferrer" target="_blank">
    博客文章
   </a>
   中抱怨说：“我们绝不能使用我们的名字来伪装一家软件公司以隐藏其在线监控工具，这些工具可能 、并且在某些情况下实际上已经被 Gamma 集团用来侵犯用户的人权和在线隐私”。
  </p>
  <p class="graf graf--p graf-after--p" id="6893">
   同年，无国界记者组织在一份年度报告中称 Gamma 集团为“
   <a class="markup--anchor markup--p-anchor" data-href="http://surveillance.rsf.org/en/" href="http://surveillance.rsf.org/en/" rel="noopener nofollow noreferrer" target="_blank">
    互联网的企业敌人
   </a>
   ”之一。令人毛骨悚然的侵入性间谍软件也可以通过更传统的方式传播 — — 例如恶意电子邮件附件。
  </p>
  <p class="graf graf--p graf-after--p" id="e012">
   早在 2011 年，英国子公司 Gamma International 就出售了一种伪装成苹果 iTunes 媒体播放器更新的恶意软件木马。
  </p>
  <p class="graf graf--p graf-after--p" id="bf59">
   在被修补之前，这个漏洞已经被利用了大约三年！这是安全记者 Brian Krebs 发现的。⚪️
  </p>
