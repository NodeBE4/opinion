 <p>

   <img alt="" class="aligncenter wp-image-6698 size-medium" height="199" sizes="(max-width: 300px) 100vw, 300px" src="http://zhanlve.org/wp-content/uploads/2019/02/201902070631341-300x199.png" srcset="http://zhanlve.org/wp-content/uploads/2019/02/201902070631341-300x199.png 300w, http://zhanlve.org/wp-content/uploads/2019/02/201902070631341.png 622w" width="300"/>
  </span>
 </p>
 <p>
 </p>
 <p>
  <span style="font-size: 12pt;">
   中国和全球的汉人在过猪年春节，大吃大喝、大小红包，以及一切繁文缛节，是中华传统文化发挥之最。在汉人们兴高采烈过节的时候，超过一百万被关在集中营，与汉人不同种族、不同习俗的维吾尔人，他们在做什么？
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   既然是再教育的洗脑集中营，在还没有"毕业"的时候，自然不会放他们回家，过维吾尔人自己的生活，也减少"春运"的压力。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   有没有可能集中营的管教人员自己放假过节，让那些维吾尔人留在营里过自己的日子？依照共产党的脾性，也不会这样便宜他们。没人管教，谁知道他们会出什么乱子？
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   因此应该是让那些维吾尔人与汉人一起过节，与汉人"同乐"。以"善心"的名义把他们融入在中华传统文化之中，彻底改造维吾尔人的文化习俗，还可以美其名曰"丰富多彩"。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   然而今年是猪年，维吾尔人信仰的伊斯兰教禁吃猪肉。然而在集中营里过猪年，铺天盖地的猪只图样当然不可避免，如果共产党不强迫你吃猪肉已经是大恩大德，岂会给你不看猪的自由？哪管是金猪还是蠢猪。孔夫子的"非礼勿视"在这里也敌不过习近平思想了。从而彻底改造他们信仰。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   如果有朝一日，有人强迫汉人欣赏蟑螂的图片，或强迫汉人吃蟑螂，说那是可以致富或长命百岁的金蟑螂，那些汉人会怎么想？会不会像周永康的大媳妇那样为当年周永康担任政法委书记时迫害人权的行为作出道歉？因为这是剃人头者人亦剃之。前国家主席刘少奇被毛泽东整死以前也傻呼呼的拿着《中华人民共和国宪法》要同毛泽东说理。未来习近平的下场可能同罗马尼亚的齐奥塞斯库一样，连法庭都不必上，更加环保。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   新疆的维吾尔人有900万人，关在集中营里的超过100万人，照片里所见，他们都是青壮年。可以想见，现在留守在住家的基本上就是老弱妇幼。纳粹在80年前的所为，今日在中国重见，其美化集中营的程度更是棋高一着，连一些西方人士也被骗，因为他们已经忘记当年所谓的"犹太乐园"是怎么一回事。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   由湖南文艺出版社在2016年出版的美国作家、普利兹文学奖获得者赫尔曼．沃克的文学巨著《战争与回忆（1941-1945）》对"犹太乐园"有非常详细的描述，而之前出版的姐妹篇《战争风云（1939-1941）》都是传世的二次大战史诗性巨著。后者于文革后期内部出版，让人们从希特勒的崛起看到毛泽东，现在全套的出版，让人看到习近平的习特勒如何进行种族灭绝。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   所幸自由世界领袖的美国没有忘记这一点，国会、国务院与白宫不断谴责中国政府的所为。今年1月底，美国国会13名议员不分党派，赶在今年度诺贝尔和平奖提名截止前，联名推荐维吾尔族经济学者伊力哈木．土赫提角逐。他在5年前的农历新年前被捕，判处无期徒刑。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   更重要的是，再早一些，国会两党提出名为《维吾尔族人权政策法》，内容包括指派联邦调查局（FBI）负责调查中国国安人员过去两年来是否曾透过其遍及全球的势力，恐吓要将居住在美国的维吾尔人遣送回中国接受所谓的"再教育"；该法案也再度主张应该动用"全球马格尼茨基人权问责法"，对参与镇压新疆少数民族的中国官员进行金融与差旅制裁。因此在印度尼西亚也出现要求驱逐中国外交使节的呼声，从而有望形成更为强大的国际力量。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   1979年，美国国会通过了《台湾关系法》，1992年通过了《香港政策法》，今年可能再通过《维吾尔族人权政策法》。美国将成为被中国共产党霸凌的弱小民族的救星。世界非常需要强大的国际警察来压制无法无天的国际强盗，地球才能维持和平与安全。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
