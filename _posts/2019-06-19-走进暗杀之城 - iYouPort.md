  <p class="graf graf--p">
   JAYPEE LAROSA 站在菲律宾棉兰老岛达沃市的一家网吧前，当时三名身着深色夹克的男子骑在摩托车上向他开火。那个夏天的傍晚，20岁的 Larosa 被杀了。
  </p>
  <p class="graf graf--p">
   Larosa 于2008年7月17日被谋杀，是在达沃市进行的数百起法外杀戮事件之一，现在这座城市已有160万人，而现任菲律宾总统的杜特尔特曾担任该市市长。在全国各地发起臭名昭著的血腥“毒品战争”之前的几年，杜特尔特在地方一级就主持了类似的杀戮。根据人权观察组织
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hrw.org/sites/default/files/reports/philippines0409webwcover_0.pdf" href="https://www.hrw.org/sites/default/files/reports/philippines0409webwcover_0.pdf" rel="noopener noreferrer" target="_blank">
    2009年的一项调查
   </a>
   ，在担任市长期间，敢死队经常暗杀街头儿童、毒贩和小罪犯; 在某些情况下，研究人员发现了政府官员和警察共谋或直接参与的证据。
  </p>
  <p class="graf graf--p">
   杜特尔特一直否认与这次杀戮运动有任何联系，但有时，他对暴力的支持几乎没有任何隐瞒。据人权组织和
   <a class="markup--anchor markup--p-anchor" data-href="https://newsinfo.inquirer.net/735457/davao-drug-suspect-dead-after-duterte-ultimatum" href="https://newsinfo.inquirer.net/735457/davao-drug-suspect-dead-after-duterte-ultimatum" rel="noopener noreferrer" target="_blank">
    当地报纸称
   </a>
   ，作为市长，杜特尔特将公开宣布“罪犯”的名称或地点，其中一些人此后就会被杀害。虽然报道没有指责杜特尔特亲自参与，但菲律宾监察员办公室
   <a class="markup--anchor markup--p-anchor" data-href="https://www.rappler.com/newsbreak/in-depth/137446-davao-death-squad-rodrigo-duterte-chr-resolution" href="https://www.rappler.com/newsbreak/in-depth/137446-davao-death-squad-rodrigo-duterte-chr-resolution" rel="noopener noreferrer" target="_blank">
    在2012年部分承认
   </a>
   了警方在容忍杀人事件中所起的作用，调查发现21名达沃市警官渎职。
  </p>
  <p class="graf graf--p">
   但
   <strong class="markup--strong markup--p-strong">
    这种潜在的侵犯人权行为的共谋并未阻止 IBM 向达沃市的执法部门提供监控技术
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www-03.ibm.com/press/us/en/pressrelease/38152.wss" href="https://www-03.ibm.com/press/us/en/pressrelease/38152.wss" rel="noopener noreferrer" target="_blank">
    2012年6月27日
   </a>
   ，即灾难性的人权观察报告发布三年后，IBM 发布了一份简短的新闻稿，宣布与达沃达成协议，升级其警察指挥中心，以“进一步加强该市的公共安全行动”。
  </p>
  <p class="graf graf--p">
   IBM 的安装称为“智能运营中心”，
   <a class="markup--anchor markup--p-anchor" data-href="https://www-03.ibm.com/press/us/en/pressrelease/38152.wss" href="https://www-03.ibm.com/press/us/en/pressrelease/38152.wss" rel="noopener noreferrer" target="_blank">
    承诺
   </a>
   通过尖端的视频分析，多通道通信技术和支持 GPS 的巡逻车，增强当局实时监控居民的能力。不到两个月后，菲律宾人权委员会
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hrw.org/news/2012/08/16/philippines-prosecute-officials-death-squad-killings" href="https://www.hrw.org/news/2012/08/16/philippines-prosecute-officials-death-squad-killings" rel="noopener noreferrer" target="_blank">
    发布了一项决议
   </a>
   ，谴责达沃当局在杀戮方面造成“有罪不罚的氛围”，并建议国家调查局对当地可能妨碍司法公正的行为进行独立调查。（杜特尔特
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hrw.org/news/2018/01/18/philippines-dutertes-drug-war-claims-12000-lives" href="https://www.hrw.org/news/2018/01/18/philippines-dutertes-drug-war-claims-12000-lives" rel="noopener noreferrer" target="_blank">
    最近谴责
   </a>
   该委员会，质疑其动机，并建议废除该委员会）
  </p>
  <p class="graf graf--p">
   2012年的 IBM 交易由 Rodrigo Duterte 的女儿 Sara Duterte 签署，她当时是达沃市的名义市长；在 Sara Duterte 的统治下，
   <a class="markup--anchor markup--p-anchor" data-href="http://www.cbcpnews.com/cbcpnews/?p=76531" href="http://www.cbcpnews.com/cbcpnews/?p=76531" rel="noopener noreferrer" target="_blank">
    杀戮仍在继续
   </a>
   。根据当地新闻报道，该系统已于2013年6月部署，Sara Duterte 告诉 Durian Post ，“警察指挥中心现在已经注入了 IBM 的 IOC 技术”，允许警察“从响应重大事件转向预测和预防性监视”。
  </p>
  <p class="graf graf--p">
   虽然 The Intercept 和 Type 的调查未能在 IBM 关于该计划的公开公司文件中找到对达沃敢死队的任何提及，但2014年公司对该装置的概述清楚地表明 IBM 知道“毒品战”、预测性警务和犯罪抑制是达沃市安全部队的“优先领域”。
  </p>
  <p class="graf graf--p">
   从2013年到2016年底，当一名达沃安全官员估计 IBM 计划正在停止使用时，与人权委员会密切合作的菲律宾
   <strong class="markup--strong markup--p-strong">
    人权活动家声称已经至少
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://www.cbcpnews.com/cbcpnews/?p=76531" href="http://www.cbcpnews.com/cbcpnews/?p=76531" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     记录了
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    达沃敢死队进行的213次法外处决
   </strong>
   。
  </p>
  <p class="graf graf--p">
   达沃市官员没有回应有关 IBM 视频监控系统的问题、或其在运行期间在法外杀戮行动中的潜在作用。但去年在达沃市接受采访的三名警察和城市安全官员表示，
   <strong class="markup--strong markup--p-strong">
    该计划加强了警方的视频监控能力
   </strong>
   ，他们认为这些能力在达沃对所谓的毒品战中被证明
   <strong class="markup--strong markup--p-strong">
    是有用的
   </strong>
   。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hrw.org/report/2009/04/06/you-can-die-any-time/death-squad-killings-mindanao" href="https://www.hrw.org/report/2009/04/06/you-can-die-any-time/death-squad-killings-mindanao" rel="noopener noreferrer" target="_blank">
    人权报告
   </a>
   已经表明，这场战争往往针对的是低级别的吸毒者和小贩，而不是主要的贩运者。
  </p>
  <p class="graf graf--p">
   总部位于达沃的人权组织前发言人 Amado Picardal 称，IBM 的工作“很不道德”，因为在与达沃市达成协议之前的几年里，有些杀人事件已经被证实与杜特尔特的警察有关。
  </p>
  <p class="graf graf--p">
   IBM 拒绝回应有关其在达沃市的人权记录的询问。IBM发言人爱德华·巴比尼（Edward Barbini）简要地指出，该公司“不再向达沃的智能运营中心提供技术”，但他拒绝透露 IBM 是否在此之后为该技术提供服务，并且 IBM 的公开文件提到该计划继续进行。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “菲律宾达沃市的150万公民将成为亚洲第一个受益于智能运营中心的地区公民，”2013年4月3日，IBM 披露，“新的预测性警务系统将监控关键风险指标，以便各机构能够在情况升级之前采取快速行动。”
  </p>
  <p class="graf graf--p">
   自 IBM 计划逐步取消以来，菲律宾警方对尖端监控基础设施的兴趣几乎没有减弱。
   <strong class="markup--strong markup--p-strong">
    国家当局正在寻求在一个名为“安全菲律宾”的项目以在全国范围内部署实时面部识别，并考虑了来自各种国际供应商的技术，包括中国的华为
   </strong>
   。
  </p>
  <p class="graf graf--p">
   去年12月，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.philstar.com/headlines/2018/12/13/1876639/dilg-chinese-firm-install-p20-billion-cctv-network" href="https://www.philstar.com/headlines/2018/12/13/1876639/dilg-chinese-firm-install-p20-billion-cctv-network" rel="noopener noreferrer" target="_blank">
    当地一家报纸报道
   </a>
   菲律宾已经获得了 200 亿比索的贷款，来自与中国公司的合作，该合作在达沃市和马尼拉地铁上安装数千台监控摄像机，据报道，该装置包括一个国家指挥中心和面部识别及车辆识别软件。在菲律宾电视台1月份的一次
   <a class="markup--anchor markup--p-anchor" data-href="https://news.abs-cbn.com/news/01/31/19/dilg-safe-philippines-can-help-thwart-another-jolo-attack" href="https://news.abs-cbn.com/news/01/31/19/dilg-safe-philippines-can-help-thwart-another-jolo-attack" rel="noopener noreferrer" target="_blank">
    采访中
   </a>
   ，内政和地方政府副部长 Epimaco Densing III 表示，该项目的目标是发现嫌疑人的面孔并在犯罪发生前“预防犯罪”。
  </p>
  <p class="graf graf--p">
   菲律宾活动家担心这种能力可能会助长侵犯人权行为。在过去三年中，该国部分地区一直处于临时戒严状态，杜特尔特的“毒品战”至少造成了5000人死亡这只是官方数字，人权组织称可能多达27,000人死亡（警察和人权组织的估计
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2018/dec/19/dutertes-philippines-drug-war-death-toll-rises-above-5000" href="https://www.theguardian.com/world/2018/dec/19/dutertes-philippines-drug-war-death-toll-rises-above-5000" rel="noopener noreferrer" target="_blank">
    差异很大
   </a>
   ）。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    被杀害的人包括反杜特尔特的活动家、民选官员和直言不讳的天主教神父
   </strong>
   。目前，杜特尔特正在开展修改宪法的行动，此举可以为行政部门
   <a class="markup--anchor markup--p-anchor" data-href="https://www.scmp.com/news/asia/southeast-asia/article/2154392/draft-constitution-proposes-giving-philippine-president" href="https://www.scmp.com/news/asia/southeast-asia/article/2154392/draft-constitution-proposes-giving-philippine-president" rel="noopener noreferrer" target="_blank">
    提供更大的权力
   </a>
   ，以进一步压制政治对手。
  </p>
  <p class="graf graf--p">
   2012年6月，市长 Sara Duterte
   <a class="markup--anchor markup--p-anchor" data-href="https://www.prnewswire.com/news-releases/city-of-davao-and-ibm-collaborate-to-build-a-smarter-city-160480725.html" href="https://www.prnewswire.com/news-releases/city-of-davao-and-ibm-collaborate-to-build-a-smarter-city-160480725.html" rel="noopener noreferrer" target="_blank">
    宣布了
   </a>
   一项价值超过300万美元的合约，IBM 将改善其实时监控的能力。该公告承诺通过改进的通信和监控技术“扩大”达沃的公共安全和安全指挥中心（PSSCC）。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    前 IBM 销售负责人萨亚吉·辛德（Sayaji Shinde）表示，他是支持指挥中心交易团队的一员，他回忆说他的团队渴望与杜特尔特政府合作
   </strong>
   。“Dutertes 会把注意力集中在公共部门的安全上，”Shinde说，“而且我认为，对于我们来说，这是我们花时间和建议他们的驱动因素之一，因为我们看到他们真的很想确保城市变得更加安全。”
  </p>
  <p class="graf graf--p">
   在该项目的初始阶段，IBM 将 Davao 的警用摄像机映射到地理信息系统，允许运营商快速访问感兴趣的任何位置附近的摄像机信号，Shinde 说。
  </p>
  <p class="graf graf--p">
   根据 Shinde 的说法，这项推出还采用了多通道通信系统，允许警察、交通和国防人员相互通信。它还包括视频分析技术，可以根据物理属性自动标记摄像机上捕获的对象，如汽车和人。
  </p>
  <p class="graf graf--p">
   根据
   <a class="markup--anchor markup--p-anchor" data-href="https://www.scribd.com/document/251701696/IBM-Smarter-Cities-Overview-Cities-must-become-smarter" href="https://www.scribd.com/document/251701696/IBM-Smarter-Cities-Overview-Cities-must-become-smarter" rel="noopener noreferrer" target="_blank">
    2014年11月 IBM
   </a>
   向亚洲开发银行的演示文稿，标签包括对象的大小、速度、颜色、轨迹和方向，允许指挥中心操作员通过相机镜头梳理搜索嫌疑人的描述。（IBM 已经使用秘密访问纽约警察局的相机镜头改进了这些类型的监视功能，如
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2018/09/06/nypd-surveillance-camera-skin-tone-search/" href="https://theintercept.com/2018/09/06/nypd-surveillance-camera-skin-tone-search/" rel="noopener noreferrer" target="_blank">
    9月份的调查报告
   </a>
   所揭示的）
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这可能是有史以来第一次在亚洲进行的视频分析监控，”Shinde 说，并指出该系统可以用于在犯罪前后跟踪嫌疑人的汽车。他指出，该软件“非常用户友好”，因此指挥中心的达沃安全官员可以很容易胜任该计划的对象搜索功能。
  </p>

   。根据权利组织的前发言人 Picardal 的说法，一个综合的国家实时面部识别技术系统将确保实现杜特尔特的“完全专制/独裁统治和压制异议的计划”。
  </p>
  <p class="graf graf--p">
   自从掌权以来，杜特尔特政府一直试图关闭或减轻关键的新闻报道，包括1月份的在线新闻网站拉普勒，
   <strong class="markup--strong markup--p-strong">
    有众多政治活动人士被暗杀
   </strong>
   。与此同时，臭名昭著的“毒品战”已造成数千人死亡。 “这不仅仅是造成数千人死亡，而且导致一个杀戮组织的生成，很容易扩大化。毒品战正在预兆着更大规模杀戮的未来。“
  </p>
