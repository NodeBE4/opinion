  <p class="graf graf--p">
   Ondřej Geršl 在波希米亚南部城市 Ceske Budejovice 长大，一开始做的是调酒师工作，该镇以其拥有数百年历史的啤酒厂而闻名，这是常见职业选择。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    经过长时间的倾注和听取顾客们对政治制度的批评，Gersl 的野心很快发生了根本性的转变
   </strong>
   。
  </p>
  <p class="graf graf--p">
   他为自己这样的新闻迷创建了一个网站 —— 如今，他主持着该国最流行的也是最具争议的另类新闻帝国之一。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “如今的大部分媒体都在向公众隐瞒重要事实 …… 而我们带来了答案。”
   </strong>
   Gersl 的新闻业务叫 AC24，它的 Facebook “关于” 部分是这样写的。
  </p>
  <p class="graf graf--p">
   Ondřej 说，他成功进入媒体世界是出于良好的商业意识，他致力于通过重新包装其他地方报道的新闻内容并关注 *轰动性话题* 来降低成本。
  </p>
  <p class="graf graf--p">
   到2019年1月，AC24 的访问量已经
   <a class="markup--anchor markup--p-anchor" data-href="http://www.netmonitor.cz/online-data-ola" href="http://www.netmonitor.cz/online-data-ola" rel="noopener noreferrer" target="_blank">
    超过100万
   </a>
   ，在一个仅拥有1000万人口的国家中这个数字令人印象深刻。
  </p>
  <p class="graf graf--p">
   在 Facebook 上，AC24 拥有超过 100000 个关注者。在过去的两年中，Gersl 的利润也在不断增长。
  </p>
  <p class="graf graf--p">
   但是，他的批评者却有更险恶的看法。他们将 AC24 的受欢迎程度视为当今混乱且细分的媒体环境中的一个警示性故事，
   <strong class="markup--strong markup--p-strong">
    它说明了曾经是次边缘网站宣传的另类政治观点如何演变成了侵略性的虚假信息渠道
   </strong>
   。
  </p>
  <p class="graf graf--p">
   捷克非政府智囊机构 European Values 在2016年4月进行的一项研究中对 AC24 进行了评估：
   <strong class="markup--strong markup--p-strong">
    该网站 “采用了一些操纵技术，从故意的错误信息到将阴谋论作为重大新闻发表。”
   </strong>
  </p>
  <p class="graf graf--p">
   捷克和斯洛伐克网站的公共数据库
   <a class="markup--anchor markup--p-anchor" data-href="https://www.konspiratori.sk/en/" href="https://www.konspiratori.sk/en/" rel="noopener noreferrer" target="_blank">
    Kospiratori.sk
   </a>
   <strong class="markup--strong markup--p-strong">
    慌称
   </strong>
   AC24 为 “克里姆林宫喉舌”。2月13日，该网站上发表的10篇文章中有3篇源于俄罗斯拥有的卫星通讯社的捷克文版本。
  </p>
  <p class="graf graf--p">
   随着欧盟选举的到来，AC24 可以在一个国家中发挥重要影响力。民粹主义现任总统 Milos Zeman 呼吁就捷克共和国加入欧盟和北约组织举行全民公决，这两个机构都经常被 AC24 新闻报道所批评。
  </p>
  <p class="graf graf--p">
   Gersl 展现自己是一个重塑社会的使者，同时始终在赚钱。
  </p>
  <p class="graf graf--p">
   他在 Skype 接受采访时说，
   <strong class="markup--strong markup--p-strong">
    他与任何政治运动或政党没有任何联系。他也不拥护特定的意识形态。他说，他正在利用一个市场来获取捷克人渴望消费的各种新闻。
   </strong>
  </p>
  <p class="graf graf--p">
   他还承认，他实际上并未检查 AC24 上发布的文章的真实性。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “看起来，我们的网站不是最真实或最值得信赖的网站。我们也并不希望我们的读者仅依赖一种信息来源。他们应该能够比较来自更多来源的信息”，Gersl 说。
  </p>
  <p class="graf graf--p">
   Gersl 于2011年左右开始成为现代媒体大亨，当时捷克人和其他欧洲人被阿拉伯世界的革命和占领华尔街的基层运动所吞没。
  </p>
  <p class="graf graf--p">
   他建立了一个网站，讨论那些媒体的问题以及其他他认为主流媒体没有报道的社会运动。他说，AC24 的早期版本吸引了“顽固的政治迷”，而且收看的观众并不多。
  </p>
  <p class="graf graf--p">
   Gersl 随后雇用了另外两个人，并创办了一家名为《Consciousness》的杂志，这是他打算发表长篇论文和新闻的平台。该杂志的座右铭是 “唤醒人们”。
  </p>
  <p class="graf graf--p">
   捷克事实检查网站 Manipulatori.cz 的主编 Jan Cemper 自其成立以来一直关注 AC24。他在一次占领集会上遇见了 Gersl，并密切注视着网站进行的彻底变革。
  </p>
  <p class="graf graf--p">
   Cemper 说：“当时他们提供了有关示威者要求的信息。他们正在帮助试图改进体制的所有学生运动放大声音。”
  </p>
  <p class="graf graf--p">
   随着俄罗斯入侵乌克兰，AC24 的内容于2014年开始发生变化。该网站上的文章描述了由于乌克兰法西斯主义加剧而导致的冲突，这与克里姆林宫官员的一个普遍观点相呼应。
  </p>
  <p class="graf graf--p">
   Cemper 说，该网站上的文章语气已转变为对西方政府机构和价值观的完全敌意，“他们开始攻击西方民主的基本原则。”
  </p>
  <p class="graf graf--p">
   AC24 还采用了美国和其他市场常见的出版趋势。他缩短了故事的长度，着眼于容易包装的重大新闻故事，这些故事通常伴随着视频。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11888 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-9.jpg?resize=1024%2C576&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-9.jpg?w=1024&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-9.jpg?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-9.jpg?resize=768%2C432&amp;ssl=1 768w" data-recalc-dims="1" height="576" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-9.jpg?resize=1024%2C576&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11888" data-recalc-dims="1" height="576" sizes="(max-width: 1024px) 100vw, 1024px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-9.jpg?resize=1024%2C576&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-9.jpg?w=1024&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-9.jpg?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-9.jpg?resize=768%2C432&amp;ssl=1 768w" width="1024"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Gersl 说，他的在线业务比杂志更赚钱。他于2016年5月出版了上一期的《Consciousness》。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们以前读过很多很厚的书，但是现在没人对这种东西感兴趣了。我决定将我的钱投入其他方面。从商业角度来讲，这将带来更多附加值”，Gersl 说。
  </p>
  <p class="graf graf--p">
   这些天来，典型的 AC24 新闻报道很少超过五段，并且特别关注欧洲右翼政客将其视为从移民到反穆的各种重大威胁。
  </p>
  <p class="graf graf--p">
   他们采取耸人听闻的红色标题，例如 “美国正在瓦解，但没人关注”，或 “比尔·盖茨和全球精英没有为他们自己的孩子接种疫苗，有充分的理由”，与无数的广告横幅并排出现在网站上。
  </p>
  <p class="graf graf--p">
   同时，Gersl 说他的利润在增长。 2015年，AC24 获利 100,000 捷克克朗（4,000美元）；而 2017年，这一数字增加到1,100,000捷克克朗（44,000美元）。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%81%87%e6%96%b0%e9%97%bb%e4%ba%a7%e4%b8%9a%e8%a7%a6%e7%9b%ae%e6%83%8a%e5%bf%83-%e4%ba%92%e8%81%94%e7%bd%91%e4%ba%ba%e7%9c%9f%e7%9a%84%e5%a6%82%e6%ad%a4%e5%a5%bd%e9%aa%97%ef%bc%9f/" href="https://www.iyouport.org/%e5%81%87%e6%96%b0%e9%97%bb%e4%ba%a7%e4%b8%9a%e8%a7%a6%e7%9b%ae%e6%83%8a%e5%bf%83-%e4%ba%92%e8%81%94%e7%bd%91%e4%ba%ba%e7%9c%9f%e7%9a%84%e5%a6%82%e6%ad%a4%e5%a5%bd%e9%aa%97%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     假新闻产业触目惊心 互联网人真的如此好骗？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   Gersl 驳斥了像森珀（Cemper）这样的批评家，后者批评他在传播假新闻中的作用。他说，
   <strong class="markup--strong markup--p-strong">
    只要能赚钱，他就会继续经营 AC24
   </strong>
   。
  </p>
  <p class="graf graf--p">
   他说：“这只是为了适应媒体行业的变化。有些人想要这类信息，因此我们将其提供给他们。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    编者按：
   </strong>
   这是一件有趣的事。我们记得几年前，VICE制作的纪录片其中有一集讲述特朗普如何上台的背后故事，但是，片中把原因几乎完全归因于 “假新闻”。
  </p>
  <p class="graf graf--p">
   这是一个主流论调，不论是特朗普上台还是英国脱欧，都被完全归因于虚假消息的盛行。
   <strong class="markup--strong markup--p-strong">
    不排除虚假消息的影响力，但是，为什么没有人反问：“那些虚假消息如何赢得如此多人心的？而不是真实消息？”
   </strong>
  </p>
  <p class="graf graf--p">
   很简单，因为他们说出了大众的内心所想。
  </p>
  <p class="graf graf--p">
   换句话说，大众有此想法在先，才会被虚假信息所触动 —— 人们认为这些虚假信息恰好 “证实” 了自己的想法；尤其是，大众对那些
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%ba%bd%e7%ba%a6%e6%97%b6%e6%8a%a5%e6%89%bf%e8%ae%a4%ef%bc%8c%e5%9c%a8%e7%a8%bf%e4%bb%b6%e5%8f%91%e5%b8%83%e4%b9%8b%e5%89%8d%e5%ae%83%e4%bc%9a%e4%b8%bb%e5%8a%a8%e6%8e%a5%e5%8f%97%e6%94%bf/" href="https://www.iyouport.org/%e7%ba%bd%e7%ba%a6%e6%97%b6%e6%8a%a5%e6%89%bf%e8%ae%a4%ef%bc%8c%e5%9c%a8%e7%a8%bf%e4%bb%b6%e5%8f%91%e5%b8%83%e4%b9%8b%e5%89%8d%e5%ae%83%e4%bc%9a%e4%b8%bb%e5%8a%a8%e6%8e%a5%e5%8f%97%e6%94%bf/" rel="noopener noreferrer" target="_blank">
    屈服于审查的主流媒体
   </a>
   的普遍厌弃。
  </p>
  <p class="graf graf--p">
   于是我们认为，将广泛的问题完全归因于虚假信息是不负责任的，特别是，这一盲目的归因还促使寡头技术平台（谷歌、Facebook、和中国的那些）
   <strong class="markup--strong markup--p-strong">
    以 “扫除虚假信息/谣言” 为借口，压制不同意见 —— 信息末日启示录的 “
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%8f%8d%e5%ae%a1%e6%9f%a5%ef%bc%9a%e5%87%bb%e9%80%80%e4%b8%83%e9%aa%91%e5%a3%ab%e4%b9%8b%e4%b8%80/" href="https://www.iyouport.org/%e5%8f%8d%e5%ae%a1%e6%9f%a5%ef%bc%9a%e5%87%bb%e9%80%80%e4%b8%83%e9%aa%91%e5%a3%ab%e4%b9%8b%e4%b8%80/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     七骑士
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ”。
   </strong>
  </p>
  <p class="graf graf--p">
   IYP 非常讨厌虚假消息，但更对其背后的成因感兴趣。每一种新事物的出现都有其背景，就如，如果不是当权者如此擅长掩盖真相，AC24 这类平台也难以赚到这么多的金钱和知名度。
  </p>
  <p class="graf graf--p">
   解决根源问题才能解决所有问题，希望抵抗假新闻的人能意识到这点。⚪️
  </p>
