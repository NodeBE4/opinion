   变态的雇佣关系
  </h1>
 </header>

 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      给人打工真的是充满艰险 ……你的老板不止是”关心你的健康问题”，他们想要追踪你的一举一动，这甚至是合法的，为什么？
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   如果您的工作场所为您提供健身追踪器，你的老板可以亲自监控所有这些数据。这意味着你的老板会知道你经常走动的频率以及有关你生活方式的一切其他信息，甚至你是否有患心脏病的风险等。
  </p>
  <p class="graf graf--p">
   很多雇主会配发健身追踪器给员工，以鼓励员工采取更积极的生活方式，比如接受采访的一位男士，他在增加每日步数后居然惊讶地接到了老板打来的祝贺电话。如果员工同意佩戴该设备，他们甚至可以获得现金奖励或降低医疗保险费。
  </p>
  <p class="graf graf--p">
   Fitbit Health Solutions 高级副总裁 Adam Pellegrini 告诉邮报说，“通过该系统，我们实际上可以看到谁没有达到他们的目标，谁没有遵守该行动计划。”
  </p>
  <p class="graf graf--p">
   但是，作为交换，员工们正在交出他们的完整个人生活，甚至没有意识到这点。鉴于健身追踪器众所周知是不准确的，这意味着你的老板可能会根据虚假信息对你进行评估，这直接影响到你的升迁问题，甚至是否能保住工作。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “雇主了解员工的生活越多，特别是在工作场所以外，他们对你的生活的控制或影响就越大，”电子前沿基金会高级职员律师 Lee Tien 告诉邮报。 “很可能会影响到你是否被保留，晋升，降级 — 甚至被解雇。”
  </p>
  <p class="graf graf--p">
   更多详见：《
   <a class="markup--anchor markup--p-anchor" data-href="https://steemit.com/surveillance/@iyouport/276ycw" href="https://steemit.com/surveillance/@iyouport/276ycw" rel="noopener noreferrer" target="_blank">
    你正在为一个贴身老大哥努力工作：地狱的新常态
   </a>
   》
  </p>
  <p class="graf graf--p">
   跟踪器通过公司的保险提供商免费或通过少量费用提供给员工。 只有自愿选择参加该计划的员工才能获得。 设备跟踪他们的健身活动，然后将这些数据发送到他们老板手机上的应用程序。
  </p>
  <p class="graf graf--p">
   像 Fitbit 这样的设备制造商甚至添加了一个呼叫服务来提醒员工他们的健康目标。
  </p>
  <p class="graf graf--p">
   Fitbit Health Solutions 高级副总裁亚当佩莱格里尼告诉邮报说，持续的行为改变确实是焦点，“通过该系统，我们实际上可以看到谁没有达到他们的目标，谁没有遵守该行动计划。”
  </p>
  <p class="graf graf--p">
   大约20％的提供健康保险的雇主都表示他们在2018年从可穿戴设备中收集数据，这比上一年年同期的14％有显著增加。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    请注意，这些数据通常不仅与老板共享，还与设备制造商、健康保险公司和其他第三方共享。因为使用是自愿的，所以这些数据不受健康保险隐私与责任法案（HIPAA）的保护，该法案保护健康记录免于被公开披露
   </strong>
   。
  </p>
  <p class="graf graf--p">
   因此，员工在志愿参加公司提供的健身追踪器之前应仔细考虑清楚。
  </p>
  <p class="graf graf--p">
   Tien 告诉邮报说，’Fitbit 或 Apple Watch 应用程序……可能会提供你甚至不知道或者没有准备让其他人知道的关于你的信息的线索。购买这些设备的个人和消费者可能还不明白这是一个严重的潜在后果。”
  </p>
