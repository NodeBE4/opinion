  <p class="graf graf--p">
   在开源调查期间，经常会需要查找特定日期特定位置所拍摄的图片；也或许是一系列日期。
  </p>
  <p class="graf graf--p">
   本文想要讨论的是：如何使用 Google 和 YouTube、Facebook 和 Instagram 在特定时间点从特定位置查找视觉证据。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Facebook（以及一个新的功能较少的二代图谱搜索）
   </strong>
  </p>
  <p class="graf graf--p">
   很多调查者仍然能记得过去的好日子，你可以轻松地在 URL 栏中输入类似这样的搜索查询来查找 Facebook 上隐藏的内容：
   <a class="markup--anchor markup--p-anchor" data-href="https://www.facebook.com/search/39875583837/photos-in" href="https://www.facebook.com/search/39875583837/photos-in" rel="nofollow noopener noreferrer" target="_blank">
    https://www.facebook.com/search/39875583837/photos-in
   </a>
  </p>
  <p class="graf graf--p">
   在这里，您告诉 Facebook 在纽约时代广场搜索图片，这就是所谓的图谱搜索最简单的方式。
  </p>
  <p class="graf graf--p">
   但是，正如您在单击该链接时会注意到的那样，它不再起作用了。因为《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/facebook-%e6%ad%a3%e5%9c%a8%e4%bb%a4%e7%9c%9f%e7%9b%b8%e6%8c%96%e6%8e%98%e4%ba%8b%e4%b8%9a%e9%99%b7%e5%85%a5%e5%9b%b0%e5%a2%83/" href="https://www.iyouport.org/facebook-%e6%ad%a3%e5%9c%a8%e4%bb%a4%e7%9c%9f%e7%9b%b8%e6%8c%96%e6%8e%98%e4%ba%8b%e4%b8%9a%e9%99%b7%e5%85%a5%e5%9b%b0%e5%a2%83/" rel="noopener noreferrer" target="_blank">
    Facebook 正在令真相挖掘事业陷入困境
   </a>
   》。
  </p>
  <p class="graf graf--p">
   自2019年6月以来，Facebook 对这些搜索的执行方式进行了重大改变。
  </p>
  <p class="graf graf--p">
   幸运的是，Henk van Ess、Dan Nemec 和 “sowdust” 在更新现有工具方面做出了一些努力，并告知记者、调查人员和其他民间真相爱好者如何使用重新构建的图谱搜索来执行这些查询。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在 Facebook 上搜索视频和图片
   </strong>
  </p>
  <p class="graf graf--p">
   现在实践一下，看看如何找到在时代广场拍摄的并于2018年1月1日上传到 Facebook 的视频。
  </p>
  <p class="graf graf--p">
   打开 Facebook 帐户并输入“时代广场”，然后告诉它想要寻找视频，然后选择日期。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6994 jetpack-lazy-image" data-lazy-sizes="(max-width: 797px) 100vw, 797px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-21.png?resize=797%2C681&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-21.png?w=797&amp;ssl=1 797w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-21.png?resize=300%2C256&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-21.png?resize=768%2C656&amp;ssl=1 768w" data-recalc-dims="1" height="681" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-21.png?resize=797%2C681&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="797"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6994" data-recalc-dims="1" height="681" sizes="(max-width: 797px) 100vw, 797px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-21.png?resize=797%2C681&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-21.png?w=797&amp;ssl=1 797w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-21.png?resize=300%2C256&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-21.png?resize=768%2C656&amp;ssl=1 768w" width="797"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   马上就遇到了第一个问题。你只能选择一个年份和一个月份，而不是确切的日期（例如2018年1月1日）。
  </p>
  <p class="graf graf--p">
   这不是一个大问题，因为在视频下面有日期，我们可以快速浏览它们并转到1月1日上传的内容上。
  </p>
  <p>
   <img alt="" class="size-full wp-image-6995 aligncenter jetpack-lazy-image" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-19.png?resize=222%2C235&amp;is-pending-load=1#038;ssl=1" data-recalc-dims="1" height="235" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-19.png?resize=222%2C235&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="222"/>
   <noscript>
    <img alt="" class="size-full wp-image-6995 aligncenter" data-recalc-dims="1" height="235" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-19.png?resize=222%2C235&amp;ssl=1" width="222"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   然而，这显然很烦人，可能浪费你很多时间。特别是对于那些每天都有数千人参观的热门地点来说，视频和照片绝对是海量的。
  </p>
  <p class="graf graf--p">
   此前 sowdust（
   <a class="markup--anchor markup--p-anchor" data-href="https://github.com/sowdust/?source=post_page---------------------------" href="https://github.com/sowdust/?source=post_page---------------------------" rel="noopener noreferrer" target="_blank">
    github
   </a>
   ）找到了一个方法可以简化操作，现在我们来深入了解一下。
  </p>
  <p class="graf graf--p">
   这里将首先关注从时代广场寻找视频。完整且有效的 URL 是这样的：
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://graph.tips/facebook.html" href="https://graph.tips/facebook.html" rel="nofollow noopener noreferrer" target="_blank">
    https://graph.tips/facebook.html
   </a>
   and
   <a class="markup--anchor markup--p-anchor" data-href="https://sowdust.github.io/fb-search/" href="https://sowdust.github.io/fb-search/" rel="nofollow noopener noreferrer" target="_blank">
    https://sowdust.github.io/fb-search/
   </a>
  </p>
  <p class="graf graf--p">
   这基本上会将您引入同一页面。
  </p>
  <p class="graf graf--p">
   打开页面后，您需要执行以下操作：
  </p>
  <ol>
   <li>
    从第一个下拉菜单中选择“视频”。
   </li>
  </ol>
  <p class="graf graf--p">
   <img alt="" class="size-full wp-image-6996 aligncenter jetpack-lazy-image" data-lazy-sizes="(max-width: 699px) 100vw, 699px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-18.png?resize=699%2C417&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-18.png?w=699&amp;ssl=1 699w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-18.png?resize=300%2C179&amp;ssl=1 300w" data-recalc-dims="1" height="417" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-18.png?resize=699%2C417&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="699"/>
   <noscript>
    <img alt="" class="size-full wp-image-6996 aligncenter" data-recalc-dims="1" height="417" sizes="(max-width: 699px) 100vw, 699px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-18.png?resize=699%2C417&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-18.png?w=699&amp;ssl=1 699w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-18.png?resize=300%2C179&amp;ssl=1 300w" width="699"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   2. 现在添加要搜索的日期，然后选择添加日期过滤器。
  </p>
  <p class="graf graf--p">
   <img alt="" class="size-full wp-image-6997 aligncenter jetpack-lazy-image" data-lazy-sizes="(max-width: 933px) 100vw, 933px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/4-12.png?resize=933%2C407&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/4-12.png?w=933&amp;ssl=1 933w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/4-12.png?resize=300%2C131&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/4-12.png?resize=768%2C335&amp;ssl=1 768w" data-recalc-dims="1" height="407" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/4-12.png?resize=933%2C407&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="933"/>
   <noscript>
    <img alt="" class="size-full wp-image-6997 aligncenter" data-recalc-dims="1" height="407" sizes="(max-width: 933px) 100vw, 933px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/4-12.png?resize=933%2C407&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/4-12.png?w=933&amp;ssl=1 933w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/4-12.png?resize=300%2C131&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/4-12.png?resize=768%2C335&amp;ssl=1 768w" width="933"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   3. 添加您要查找的关键字，在本例中为“时代广场”。
  </p>
  <p class="graf graf--p">
   4. 在新窗口中打开URL。
  </p>
  <p class="graf graf--p">
   <img alt="" class="size-full wp-image-6998 aligncenter jetpack-lazy-image" data-lazy-sizes="(max-width: 933px) 100vw, 933px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/5-11.png?resize=933%2C712&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/5-11.png?w=933&amp;ssl=1 933w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/5-11.png?resize=300%2C229&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/5-11.png?resize=768%2C586&amp;ssl=1 768w" data-recalc-dims="1" height="712" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/5-11.png?resize=933%2C712&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="933"/>
   <noscript>
    <img alt="" class="size-full wp-image-6998 aligncenter" data-recalc-dims="1" height="712" sizes="(max-width: 933px) 100vw, 933px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/5-11.png?resize=933%2C712&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/5-11.png?w=933&amp;ssl=1 933w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/5-11.png?resize=300%2C229&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/5-11.png?resize=768%2C586&amp;ssl=1 768w" width="933"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   您还可以添加过滤器 Live，以仅搜索上传到实时广播的视频。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在 Facebook 上按上传日期搜索照片
   </strong>
  </p>
  <p class="graf graf--p">
   如上，您可以为照片做同样的事。唯一的区别是这次在下拉菜单中选择“照片”而不是“视频”。
  </p>
  <p class="graf graf--p">
   打开新URL时，您将看到9个图像分为三行，每行三个图像。
  </p>
  <p>
   <img alt="" class="size-full wp-image-6999 aligncenter jetpack-lazy-image" data-lazy-sizes="(max-width: 933px) 100vw, 933px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/6-9.png?resize=933%2C451&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/6-9.png?w=933&amp;ssl=1 933w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/6-9.png?resize=300%2C145&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/6-9.png?resize=768%2C371&amp;ssl=1 768w" data-recalc-dims="1" height="451" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/6-9.png?resize=933%2C451&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="933"/>

    <img alt="" class="size-full wp-image-6999 aligncenter" data-recalc-dims="1" height="451" sizes="(max-width: 933px) 100vw, 933px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/6-9.png?resize=933%2C451&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/6-9.png?w=933&amp;ssl=1 933w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/6-9.png?resize=300%2C145&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/6-9.png?resize=768%2C371&amp;ssl=1 768w" width="933"/>

  </p>
  <p class="graf graf--p">
   您可以单击它们并验证它们是否来自您指定的日期。
  </p>
  <p class="graf graf--p">
   您可能会获得在不同日期、一天内不同时间标记的图像，这是因为拍摄/上传此照片的位置与当下位置之间有时差。
  </p>
  <p class="graf graf--p">
   现在尝试点击“查看全部”以查看该地点和那个时间的其他图像。
