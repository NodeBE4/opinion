  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    您需要在这个过程中尽可能保持冷静 —— 很多尝试隐身的人由于紧迫的情况而处于慌乱之中，也因此很多平日里熟知的技巧都想不起来了
   </strong>
   。这是极大的悲剧。
  </p>
  <p class="graf graf--p">
   不管做任何事，心理素质都是需要强调的硬项；⚠️恶意行为者（包括追踪者）最擅长挑战目标人的心理自助能力，以刺激其慌乱，从而导致其疏忽、并自动做出愚蠢的决策。
  </p>
  <p class="graf graf--p">
   抵制方案就是加强自己的冷静。情绪管理 ，这在平时需要一些练习。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在信息篡改之前，你必须认真思考信息可能隐藏的每一个地方
   </strong>
   —— 这就是我们在《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a4%a7%e6%b8%85%e6%b4%97%ef%bc%9a%e5%8f%8d%e4%be%a6%e5%af%9f%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" href="https://www.iyouport.org/%e5%a4%a7%e6%b8%85%e6%b4%97%ef%bc%9a%e5%8f%8d%e4%be%a6%e5%af%9f%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" rel="noopener noreferrer" target="_blank">
    大清洗
   </a>
   》中强调过的思路；当然，大清洗是针对数字世界的技巧和工具，但对于现实中的隐身来说，思路是一样的。
  </p>
  <p class="graf graf--p">
   你需要仔细寻找，
   <strong class="markup--strong markup--p-strong">
    把自己想像成追踪者，对自己的一切进行侦查
   </strong>
   。列出一个清单，然后逐一处理。
  </p>
  <p class="graf graf--p">
   乍看下似乎是一个庞大的工作，但是这里面也可以有技巧，以逻辑简化工作量。于是本文首先将涉及的内容就是引导您检视自己在现实中的痕迹。
  </p>
  <p class="graf graf--p">
   重点先关于以下这些部分 —— 这是
   <strong class="markup--strong markup--p-strong">
    大多数人最容易留下痕迹的部分
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、钱包和口袋
   </strong>
  </p>
  <p class="graf graf--p">
   拿出你的钱包看一下里面装了哪些卡。
  </p>
  <p class="graf graf--p">
   每一张卡（任何卡）上有大量的个人信息，而且信息量叹为观止。
  </p>
  <p class="graf graf--p">
   除了基本的联系方式信息和交易记录之外，部分卡上可能还有奖励计划。
  </p>
  <p class="graf graf--p">
   如果你有常旅客里程数、优惠出租车服务、或者连锁酒店积分等等，可以想像，一旦追踪者找到这些无疑是找到了一座金矿；尤其是当你用里程数换了机票飞往了新生活时…… 你可能就没有新生活了。
  </p>
  <p class="graf graf--p">
   此外，你有百货店的奖励卡吗？常客卡呢？这些商店是否有你的真实姓名和联系方式？
  </p>
  <p class="graf graf--p">
   如果你的身上还有一部智能手机或iPod，苹果/安卓账户会揭示多少关于你的个人信息？你是否还有其他在线商店的账户？订餐？网购？网约车？共享自行车？……
  </p>
  <p class="graf graf--p">
   一切都是导火索，都需要清除和篡改。最好是从平日就注意这些细节。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、钥匙链或者其他免费小商品
   </strong>
  </p>
  <p class="graf graf--p">
   继续思考。药店掌握多少关于你的信息？杂货店呢？健身房呢？或者公交卡？你是否因为被动加入过某个公司的群发邮件列表？或者因为参加过其他促销活动而获得了一个免费的钥匙链或者其他很垃圾的小商品？
  </p>
  <p class="graf graf--p">
   继续问自己：谁有备用钥匙？你是否曾经把钥匙交给生命中任何一个重要的人、朋友、或者保洁阿姨？
  </p>
  <p class="graf graf--p">
   如果已经与此人断绝了关系，你是否要回了钥匙？你能保证此人没有私下另配一把新的钥匙吗？
  </p>
  <p class="graf graf--p">
   立即换锁才是完美选择。我们不会建议您对亲密关系也保持怀疑，但是要知道，人是会变的（这也是人类最大的神秘魅力之一），并且你不可能预测到变化的准确时间。于是，留一手会有好处。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、客厅
   </strong>
  </p>
  <p class="graf graf--p">
   到客厅去看看。如果你有电视、音响、空调、以及其他设备，那么一定会有保修单。
  </p>
  <p class="graf graf--p">
   保修单上都印了哪些内容？生产商手里是否有你的现住址和手机号码？
  </p>
  <p class="graf graf--p">
   如果某些电器是别人送的，那么生产商手里掌握的究竟是送你东西的人的信息，还是你的信息？
  </p>
  <p class="graf graf--p">
   无论是哪种情况，
   <strong class="markup--strong markup--p-strong">
    保修单对于追踪者来说都是巨大的优势
   </strong>
   ；哪怕找不到你本人的联系方式，也会能找到至少一位你的老朋友或家人的联系方式，于是就可以通过
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
    社交工程战术
   </a>
   从他们那里套出关于你的信息。
  </p>
  <p class="graf graf--p">
   继续。沙发、长椅、柜子等等这些家具，是否都有保修单？如果追踪者冒充你的身份找到了那些商家，他们能从商家那里套出哪些信息？
  </p>
  <p class="graf graf--p">
   所有这一切东西都是如何支付费用的？是信用卡还是通过商店的账户？
  </p>
  <p class="graf graf--p">
   任何一个聪明的追踪者只要嗅到你的任何一丝气息，就一定会努力利用它。
  </p>
  <p class="graf graf--p">
   他们会找到当初你购物时的家具店、地毯店或者是品牌经销处；他会与这些商店联系，谎称他是你，就说单人沙发或者任何一件你买过的东西的保修单找不到了。⚠️不出5分钟，他就会能拿到地址和电话号码。
  </p>
  <p class="graf graf--p">
   而且这种利用保修单和签帐卡的社交工程方法甚至可以实现
   <strong class="markup--strong markup--p-strong">
    跨国追踪
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、卫生间
   </strong>
  </p>
  <p class="graf graf--p">
   卫生间可能是个危险的地方。个人药箱里充斥着大量私人信息；药品标签说明你的健康状况，甚至还会列出给你看病的医生和药房的名字。
  </p>
  <p class="graf graf--p">
   药房有你的医疗保险记录，而医疗保险公司会把你的所有开药情况详细记录下来。
  </p>
  <p class="graf graf--p">
   尤其是，如果你是一个政客或公众人物，你肯定不想让任何人知道你在使用赞诺安或百忧解（镇静剂和抗抑郁药物），伟哥呢，也一样吧？
  </p>
  <p class="graf graf--p">
   但是想要加害你的人会想方设法查出这些信息。它们可以非常致命。
  </p>
  <p class="graf graf--p">
   对那些想要借用你的洗手间的人保持警惕。最好是不要把上述物品放在触手可及的地方。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、车库
   </strong>
  </p>
  <p class="graf graf--p">
   车库一样是追踪者的天堂。
  </p>
  <p class="graf graf--p">
   人们会放置很多机器和工具在车库里，还有早已被遗忘的垃圾中，都能找到关于你的线索。
  </p>
  <p class="graf graf--p">
   不论是割草机、电动钻头还是什么工具，上面一般都会有序列号，而且序列号是贴在保修单和购物单上的。
  </p>
  <p class="graf graf--p">
   保修单又附在购物单之后，
   <strong class="markup--strong markup--p-strong">
    上面一般会有你的姓名、地址、电话号码，以及你所提供的一切信息。满满的，都是情报
   </strong>
   。
  </p>
  <p class="graf graf--p">
   移除所有这些东西上的序列号，在扔掉和卖掉这些东西之前。
  </p>
  <p class="graf graf--p">
   大多数人在整理“垃圾”的时候只是想着“我还用得到/我不会用得到”的问题，而不是：“追踪者能用它们来做些什么”。
  </p>
  <p class="graf graf--p">
   ⚠️这种思考方式肯定不是被支持的，它很危险。尤其是当你正在准备逃难的情况下。
  </p>
  <p class="graf graf--p">
   我们会建议您在日常整理垃圾的过程中也能保持*安全的思考方式* ——
   <strong class="markup--strong markup--p-strong">
    多想想自己扔掉或卖掉的东西对于恶意行为者来说意味着什么
   </strong>
   。
  </p>
  <p class="graf graf--p">
   不要简单地把废旧物品塞进垃圾袋，最可靠的方法是，烧掉它们。
  </p>

  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    6、卧室
   </strong>
  </p>
  <p class="graf graf--p">
   人人都知道不雅视频是什么，所以卧室必须是私密空间。
  </p>
  <p class="graf graf--p">
   平日就不要在卧室里放置任何带有摄像头和麦克风的设备，比如手机、电脑、家用摄像头、所谓的智能某某某等，如果习惯于在床上看手机/电脑，用黑胶布贴住摄像头。
  </p>
  <p class="graf graf--p">
   ⚠️任何可以用来威胁和勒索你的东西，都是你最大的弱点。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    7、厨房
   </strong>
  </p>
  <p class="graf graf--p">
   和客厅一样，厨房里也会有一大堆保修单之类的东西：冰箱、微波炉、洗碗机、炉灶……甚至食品也存在能指向你的线索。
  </p>
  <p class="graf graf--p">
   食物是从哪买的？你是否参与过团购、聚划算之类的活动？几乎所有商店都有“认同卡”，只要扫一扫就可以享受当周的优惠价格 —— 这些商店把你的地址和电话都存档了，有些甚至还会有你的支票账户信息。
  </p>
  <p class="graf graf--p">
   当然，他们都知道你买了什么、在哪买的、什么时候买的…… 你为什么要把这些真实的信息提供给那些人？
  </p>
  <p class="graf graf--p">
   你喜欢美食吗？订阅过美食杂志吗？是否在特色美食公司邮购过食品？聪明的追踪者就会利用这些信息找到你的下落。
  </p>
  <p class="graf graf--p">
   ⚠️如果追踪你的人知道你喜欢吃什么 —— 你在社交网络上晒的美食照片中哪种食品的出镜率最高 —— 追踪者就可以瞄准提供此种食品的所有公司，一一给这些公司打电话，声称他就是你“这个老客户”，要求订购，并声称送货地址已变更。
  </p>
  <p class="graf graf--p">
   很简单就能套出你被登记过的旧地址甚至私密联系方式。
  </p>
  <p class="graf graf--p">
   ⚠️我可以肯定的说，绝大多数人并不会以为自己在社交媒体上晒美食这件事有多大的危险性，当然，很多时候它不危险，但是，如果有人暗中针对你，这些美食就是你的枷锁。

  </p>
  <p class="graf graf--p">
   最好平日就注意这点：
   <strong class="markup--strong markup--p-strong">
    不要让任何人尤其是不怎么可靠的人了解你
   </strong>
   。尤其是不要把真实的自己放在互联网那种地方上。
  </p>

   <strong class="markup--strong markup--p-strong">
    8、书房
   </strong>
  </p>
  <p class="graf graf--p">
   电子设备、以及放置电子设备的地方，对于追踪者来说很容易形成突破口。
  </p>
  <p class="graf graf--p">
   你的IP地址以及硬盘上没能清理干净的信息，会告诉侵入者或骇客能找到你或者盗用那你的身份所需要的一切。
  </p>
  <p class="graf graf--p">
   如果追踪者通过上述任何一种方法找到了你的旧址，闯入了你的房子，他们就可以拿到你的电子设备，以及你家里未受保护的无线网络传输信息，用不了多久就能找到你的下落 —— 你要去什么地方、到了那里之后计划做什么。

  </p>
  <p class="graf graf--p">
   如果你想要知道自己是否彻底完成了清扫追踪线索的工作，你可以自己聘请一位私家侦探来追踪你。以此检验你的隐身质量。这就像是网络安全措施中的渗透测试。
  </p>
  <p class="graf graf--p">
   但是不要使用任何在线支付的方法和任何银行卡支付，用现金；最好能选一个离你比较远的私人侦探。
  </p>
  <p class="graf graf--p">
   把名字给他们，附加一些基本的信息，比如旧地址、电话号码等，看他们能挖出多少东西。那些人是只认识钱的，他们一般不关心是谁在聘请他们以及什么目的。
  </p>
  <p class="graf graf--p">
   一旦他们挖出任何蛛丝马迹，你应该立即补上漏洞。到此才能称为一切就绪。
  </p>
  <p class="graf graf--p">
   下面应该开始第二步工作了：信息杜撰。
  </p>
  <p class="graf graf--p">
   重复强调这句话：⚠️
   <strong class="markup--strong markup--p-strong">
    人们之所以会陷入无法扭转的危机，是因为一个广泛的心理弱点 —— 总是认为最糟糕的事不会发生在自己身上。这是大脑的防御机制；但它也非常危险
   </strong>
   。
  </p>
  <p class="graf graf--p">
   没有人能预测危机，所以聪明的人应该知道如何在平日里保护自己。就像你会下意识地离家前锁门、将干燥的纸张远离火源那样。⚪️
  </p>
  <p class="graf graf--p">
   <em>
    —— 未完待续 ——
   </em>
