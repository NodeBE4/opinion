  <p class="graf graf--p">
   印度有成千上万的人使用端对端加密的 WhatsApp 通话方式来避免政府的监视 —— 这种监视无处不在且不受监管，人们不得不假设当局已经找到了破解 WhatsApp 的方法。
  </p>
  <p class="graf graf--p">
   在去年10月，这一假设
   <a class="markup--anchor markup--p-anchor" data-href="https://scroll.in/latest/942218/nagpur-lawyer-notified-by-whatsapp-of-surveillance-says-bhima-koregaon-accused-were-also-targetted" href="https://scroll.in/latest/942218/nagpur-lawyer-notified-by-whatsapp-of-surveillance-says-bhima-koregaon-accused-were-also-targetted" rel="noopener noreferrer" target="_blank">
    得到了证实
   </a>
   。
   <strong class="markup--strong markup--p-strong">
    甚至被证实了
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://scroll.in/latest/942218/nagpur-lawyer-notified-by-whatsapp-of-surveillance-says-bhima-koregaon-accused-were-also-targetted" href="https://scroll.in/latest/942218/nagpur-lawyer-notified-by-whatsapp-of-surveillance-says-bhima-koregaon-accused-were-also-targetted" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     更多
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p">
   事实证明，
   <strong class="markup--strong markup--p-strong">
    以色列最臭名昭著的NSO集团出售的全球最先进的间谍软件Pegasus 可以使用 WhatsApp（以及许多其他消息传递服务）帮助政府骇客入侵目标人的电话 —— 这种东西只卖给政府
   </strong>
   。
  </p>
  <p class="graf graf--p">
   它只需要给目标人拨打一个电话，不论是否被接听，都能成功入侵。即 黑市上售价最高的 “无交互攻击”。
  </p>
  <p class="graf graf--p">
   这种恶意的音频或视频通话（通常来自国际号码）不仅会破解通讯本身，还能入侵包括密码、电子邮件、文本消息、日历、照片、联系人列表 ……在内的
   <strong class="markup--strong markup--p-strong">
    整个手机
   </strong>
   。甚至可以
   <strong class="markup--strong markup--p-strong">
    打开用户设备的摄像头和麦克风进行直接监视监听
   </strong>
   。
  </p>
  <p class="graf graf--p">
   而受害人完全不会发现。
  </p>
  <p class="graf graf--p">
   我们曾经上传过
   <strong class="markup--strong markup--p-strong">
    Pegasus 间谍软件的文档
   </strong>
   ，您可以在这里看到：
   <a class="markup--anchor markup--p-anchor" data-href="https://t.me/iyouport/6379" href="https://t.me/iyouport/6379" rel="nofollow noopener noreferrer" target="_blank">
    https://t.me/iyouport/6379
   </a>
  </p>
  <p class="graf graf--p">
   这种间谍软件 — — 以色列 NSO 生产的 Pegasus — — ⚠️ 已经感染了45个国家地区，被用来监视和迫害异议人士，包括香港。
  </p>
  <p class="graf graf--p">
   请注意，**并不确定** 中国大陆当局是否在本地使用了这种间谍软件，因为无法对中国做DNS缓存探测。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/report%ef%bc%9a%e4%bb%a5%e8%89%b2%e5%88%97-nso-%e7%94%9f%e4%ba%a7%e7%9a%84%e9%97%b4%e8%b0%8d%e8%bd%af%e4%bb%b6-pegasus-%e5%9c%a845%e4%b8%aa%e5%9b%bd%e5%ae%b6%e8%a2%ab%e7%94%a8%e4%ba%8e%e7%9b%91/" href="https://www.iyouport.org/report%ef%bc%9a%e4%bb%a5%e8%89%b2%e5%88%97-nso-%e7%94%9f%e4%ba%a7%e7%9a%84%e9%97%b4%e8%b0%8d%e8%bd%af%e4%bb%b6-pegasus-%e5%9c%a845%e4%b8%aa%e5%9b%bd%e5%ae%b6%e8%a2%ab%e7%94%a8%e4%ba%8e%e7%9b%91/" rel="noopener noreferrer" target="_blank">
    在这里
   </a>
   看到详细报告。
  </p>
  <p class="graf graf--p">
   沙特被肢解的异议人士
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%9d%80%e6%ad%bb%e5%8d%8e%e7%9b%9b%e9%a1%bf%e9%82%ae%e6%8a%a5%e8%ae%b0%e8%80%85-jamal-khashoggi-%e7%9a%84%e6%9d%80%e6%89%8b%e5%9c%a8%e7%be%8e%e5%9b%bd%e6%8e%a5%e5%8f%97%e7%9a%84%e5%9f%b9%e8%ae%ad/" href="https://www.iyouport.org/%e6%9d%80%e6%ad%bb%e5%8d%8e%e7%9b%9b%e9%a1%bf%e9%82%ae%e6%8a%a5%e8%ae%b0%e8%80%85-jamal-khashoggi-%e7%9a%84%e6%9d%80%e6%89%8b%e5%9c%a8%e7%be%8e%e5%9b%bd%e6%8e%a5%e5%8f%97%e7%9a%84%e5%9f%b9%e8%ae%ad/" rel="noopener noreferrer" target="_blank">
    Jamal Khashoggi
   </a>
   就是被这个东西杀死的。
  </p>
  <p class="graf graf--p">
   并且，
   <strong class="markup--strong markup--p-strong">
    以色列间谍软件公司出售这种监视技术和知识，在130个国家针对记者、异议人士和LGBT人群
   </strong>
   ，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%a5%e8%89%b2%e5%88%97%e9%97%b4%e8%b0%8d%e8%bd%af%e4%bb%b6%e5%85%ac%e5%8f%b8%e5%87%ba%e5%94%ae%e7%9b%91%e8%a7%86%e6%8a%80%e6%9c%af%e5%92%8c%e7%9f%a5%e8%af%86%e5%9c%a8130%e4%b8%aa%e5%9b%bd/" href="https://www.iyouport.org/%e4%bb%a5%e8%89%b2%e5%88%97%e9%97%b4%e8%b0%8d%e8%bd%af%e4%bb%b6%e5%85%ac%e5%8f%b8%e5%87%ba%e5%94%ae%e7%9b%91%e8%a7%86%e6%8a%80%e6%9c%af%e5%92%8c%e7%9f%a5%e8%af%86%e5%9c%a8130%e4%b8%aa%e5%9b%bd/" rel="noopener noreferrer" target="_blank">
    在这里
   </a>
   看到详细报告。
  </p>
  <p class="graf graf--p">
   scroll 获得了有关此次 WhatsApp 间谍活动关键针对目标的信息。
  </p>
  <p class="graf graf--p">
   该间谍软件所针对的
   <strong class="markup--strong markup--p-strong">
    总计约1,400 名 WhatsApp 用户
   </strong>
   。
  </p>
  <p class="graf graf--p">
   该间谍软件并不便宜 —— 2016年的价格表显示，前10个攻击目标的成本超过100万美元。这意味着它不会像常规病毒那样大规模传播，而是必须有高度针对性的使用。
  </p>
  <p class="graf graf--p">
   多伦多大学网络安全小组“公民实验室”与 WhatsApp 合作，一直在向某些人发出警告，告知他们已成为攻击目标。
  </p>
  <p class="graf graf--p">
   在印度，至少包括17个人，
   <strong class="markup--strong markup--p-strong">
    主要是人权活动家、律师、学者和新闻记者
   </strong>
   ，他们向 Scroll 确认他们都已经感染这种间谍软件。
  </p>
  <p class="graf graf--p">
   揭露提出了一些问题。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    政府能否认部署了此间谍软件吗？
   </strong>
  </p>
  <p class="graf graf--p">
   印度信息与科技、法律与司法联盟部长 Ravi Shankar Prasad 表示，政府对安全漏洞 “表示关注”，印度已要求 Facebook 拥有的消息服务 “进行解释”。
  </p>
  <p class="graf graf--p">
   他的发言像所有当权者那样官僚和扯蛋，坚持认为国家机构出于对所谓的 “国家利益” 的明确理由，有一套完善的协议以拦截印度国内的公民通话。然后他利用这个机会在国会上大肆宣传，提醒公众有关上届政府操作的国家监视的指控。
  </p>
  <p class="graf graf--p">
   这还不够。尽管 NSO 拒绝阐明该间谍工具的细节，但该公司明确声称
   <strong class="markup--strong markup--p-strong">
    仅将其间谍软件零售给政府客户。
   </strong>
  </p>
  <p class="graf graf--p">
   该间谍软件的成本也高得令人望而却步，以至于只有少数几个实体能够部署该软件，而政府则名列嫌疑人榜首。
  </p>
  <p class="graf graf--p">
   此外，
   <strong class="markup--strong markup--p-strong">
    那些被定位为目标的人的名册清楚地表明了在人权领域工作的共同点 —— 这些人都专注于揭露政府当局的阴暗面
   </strong>
   。
  </p>
  <p class="graf graf--p">
   显然政府在针对这些人方面有明显的动机。
  </p>
  <p class="graf graf--p">
   从过去到现在，印度政府在监视本国公民方面充满了糟糕的声誉。考虑到这一点，重要的是政府难以明确否认它没有使用该间谍软件来侵犯公民所享有的基本隐私权。
  </p>
  <p class="graf graf--p">
   这不仅对联邦政府及其许多部门（包括情报局和其他机构）适用，而且对许多州政府也适用，这些州政府也可能使用此间谍工具。
  </p>
  <p class="graf graf--p">
   联邦政府和州政府能否诚实地告诉公众，他们究竟有没有使用这种间谍软件？他们可以用什么样的自辨以令公众信任？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    政府在此做了什么？
   </strong>
  </p>
  <p class="graf graf--p">
   即便假设政府没有干这种事，或者至少不是联邦政府的行动，仅仅要求 WhatsApp 进行解释就足够了吗？
  </p>
  <p class="graf graf--p">
   在这种情况下，执政的巴拉迪亚·贾纳塔党（BJP）的成员已经在
   <a class="markup--anchor markup--p-anchor" data-href="https://mobile.twitter.com/vinitgoenka/status/1189876152887730176" href="https://mobile.twitter.com/vinitgoenka/status/1189876152887730176" rel="noopener noreferrer" target="_blank">
    试图质疑那些被攻击的人
   </a>
   —— 而不是那些进行攻击的人，仅仅是因为到目前为止出现的受害者名字都对他们的政治持批评态度。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10307 jetpack-lazy-image" data-lazy-sizes="(max-width: 998px) 100vw, 998px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-2.png?resize=998%2C738&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-2.png?w=998&amp;ssl=1 998w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-2.png?resize=300%2C222&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-2.png?resize=768%2C568&amp;ssl=1 768w" data-recalc-dims="1" height="738" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-2.png?resize=998%2C738&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="998"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10307" data-recalc-dims="1" height="738" sizes="(max-width: 998px) 100vw, 998px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-2.png?resize=998%2C738&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-2.png?w=998&amp;ssl=1 998w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-2.png?resize=300%2C222&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-2.png?resize=768%2C568&amp;ssl=1 768w" width="998"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   这是不合情理的。无论个人政治立场如何，公民的基本权利都在这里受到了严重威胁。
  </p>
  <p class="graf graf--p">
   如果一家使用非法以色列软件的美国公司正在窃取印度公民的个人信息，那么它将在多个层面上对印度构成威胁，无论是政府机构还是个人。
  </p>
  <p class="graf graf--p">
   过去，政府除了对数据泄露表达了一点口头担忧之外，没有采取任何行动，例如，他们没有对
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" rel="noopener noreferrer" target="_blank">
    剑桥分析公司
   </a>
   Facebook 数据转储的影响进行调查。这次会有所不同吗？
  </p>
  <p class="graf graf--p">
   WhatsApp 骇客的消息不是唯一的的，与此同时，有关在暗网上出售大量印度信用卡和借记卡详细信息、以及印度核工厂计算机上出现的恶意软件的详细信息，纷纷浮出水面。印度政府会更加重视数据保护吗？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    国会将如何处理？
   </strong>
  </p>
  <p class="graf graf--p">
   几年前，印度人以及印度最高法院被告知，国会将通过一项明确旨在保护我们数据的法律。
  </p>
  <p class="graf graf--p">
   尽管《个人数据保护法案》草案已经存在一年多了，但没有任何迹象表明政府打算何时将其提交国会。
  </p>
  <p class="graf graf--p">
   此外，该草案回避了所有监视问题，认为应由另一项立法来处理。
  </p>
  <p class="graf graf--p">
   因此，当务之急是，国会必须在即将召开的会议上，在旨在保护印度公民的个人数据不受所有实体（私人或政府）侵害的法律上，取得进展，同时还应制定一项法律规范印度国家对本国公民的监视行为。
  </p>
  <p class="graf graf--p">
   对于广大公众来说，要求立法者提供这种保护至关重要。
  </p>
  <p class="graf graf--p">
   网络安全、监视和数据保护必须脱离传闻的范围，要彻底曝光才能解决问题。⚪️
  </p>
