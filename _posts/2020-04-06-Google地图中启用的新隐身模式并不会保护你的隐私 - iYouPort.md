  <p class="graf graf--p">
   也许是为了回应人们对数字行为跟踪（尤其是位置跟踪）日益增长的担忧，Google引入了大量所谓的 “隐私保护措施”。
  </p>
  <p class="graf graf--p">
   其中包括：一种新的 Google 隐身模式，该模式号称可防止 Google 保存某些类型的数据、并从其 Maps 应用中删除个人化的设置。
  </p>
  <p class="graf graf--p">
   但是，请不要以为它会使您的动作无法被追踪。
  </p>
  <p class="graf graf--p">
   该公司在博客中写道；它基于2008年首次推出的 Chrome 浏览器和去年早些时候推出的 YouTube 应用的隐身模式。
  </p>
  <p class="graf graf--p">
   但是，与这些前辈一样，
   <strong class="markup--strong markup--p-strong">
    重要的不是了解隐身模式的 “好处”，而且要了解其局限性。
   </strong>
  </p>
  <p class="graf graf--p">
   启用该新功能将阻止特定设备生成的数据显示在您的Google位置历史记录中。如果您要通过Google地图与某人共享位置，则该设备会在激活时暂停有关该设备位置的更新。地图隐身模式还会停止与应用程序有关的通知和所谓的个性化。
  </p>
  <p class="graf graf--p">
   但是，它并不能在所有设备上提供保护。
  </p>
  <p class="graf graf--p">
   当您打开“地图隐身模式”时，会出现一条警告：“在地图中打开隐身模式
   <strong class="markup--strong markup--p-strong">
    不会
   </strong>
   影响互联网提供商、其他应用程序、语音搜索、和其他Google服务使用或保存您的活动的方式。”
  </p>
  <p class="graf graf--p">
   这意味着“地图隐身模式”不会采取任何措施来屏蔽或匿名化您向外部世界展示的网络访问量 —— 仅仅是减少了您在自己的Google帐户中浏览地图的记录，并提供了一种暂停与其他Google地图用户共享位置信息的方法。
  </p>
  <p class="graf graf--p">
   同样，Chrome 的所谓隐身模式也无法阻止网站、您的雇主或互联网服务提供商看到您的浏览历史。
  </p>
  <p class="graf graf--p">
   Google Maps产品经理 Marlo McGriff 称：“我们希望在如何管理 Google Maps 数据方面给人们更多选择和控制权，并且我们知道有时您可能不希望自己的活动与真实身份联系在一起，例如，假设您与其他人共享设备 —— 您可能不希望他们知道您最近搜索了附近的订婚戒指商店。”
  </p>
  <p class="graf graf--p">
   现在您知道它是干什么的了？是的， —— 仅仅在您偶尔在去洗手间或从冰箱里拿啤酒时用一下它。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果您需要的是匿名化浏览或破坏互联网服务提供商的洞察力，请考虑使用Tor或VPN。
   </strong>
  </p>

  <p class="graf graf--p">
   要在地图中使用隐身模式，只需在屏幕的右上角（搜索栏旁边）点击您的帐户照片，然后点击打开隐身模式。
  </p>
  <p class="graf graf--p">
   您还可以退出 Google Maps，这可以完成很多相同的事，并且无需完全在Android和IOS上注销 Google 帐户。
  </p>
  <p class="graf graf--p">
   但是，当您执行此操作时，该应用程序仍会记录并显示诸如最近的搜索之类的内容。
  </p>
  <p class="graf graf--p">
   而且，当您未登录时，您将无法执行诸如与其他用户共享位置信息之类的操作。
  </p>
  <p class="graf graf--p">
   Google 的隐私权政策规定，当您未登录 Google 帐户时，该公司仍会在其服务中收集和存储有关您的浏览和行为的信息，但会将这些数据与唯一标识符而非完整帐户相关联。Google 称，隐身地图模式中的数据存储有分配给唯一标识符。该ID不与用户帐户绑定，并且在每个新会话中都会重置。
  </p>
  <p class="graf graf--p">
   随着时间的推移，所谓的隐私调整的确会增加，看起来似乎在为用户数据接口提供 “透明性和控制力”。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    对于诸如位置跟踪和智能助手交互之类的问题 —— 它们就是监视资本主义的生存之本，并无法解决
   </strong>
   。
  </p>
  <p class="graf graf--p">
   要警惕这点：
