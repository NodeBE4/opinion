  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    直接通过 Bluetooth、Wifi Direct 或 NFC 共享文件
   </strong>
  </h3>
  <p class="graf graf--p">
   您无需互联网就可以通过蓝牙、Wifi Direct 或近场通信（NFC）将手机与附近的其他设备连接起来（NFC 有时在较旧的设备上被称为 Android Beam）。
  </p>
  <p class="graf graf--p">
   蓝牙和 Wifi Direct 都是无线技术，可以 “配对” 两个设备，而中间可以没有其他路由器或接入点。
  </p>
  <p class="graf graf--p">
   与蓝牙相比，WiFi Direct 可以提供更广泛的范围和更快的数据传输，但消耗的电力更多。
  </p>
  <p class="graf graf--p">
   同时，NFC 的范围比蓝牙或 WiFi Direct 短得多（〜4cm），传输速度也慢得多，但连接速度更快，耗电量也更少，因此在双手握有两个设备的情况下进行小型传输时非常有用。
  </p>

  <p class="graf graf--p">
   ⚠️ 重要提示：这些服务的主要缺点是它们不安全。
  </p>
  <p class="graf graf--p">
   蓝牙和 wifi 信标/扫描仪可用于跟踪您的位置或探测设备以获取信息。渗透者可能会尝试与您的设备配对，向您发送不需要的文件，甚至在设备易受攻击的状态下控制您的设备。
  </p>
  <p class="graf graf--p">
   这就是为什么我们要强调：
   <strong class="markup--strong markup--p-strong">
    为了更安全，请在不使用这些服务时将其关闭，
   </strong>
   将应用程序权限限制为您所需的对象，并学习良好的操作安全性，例如 及时运行更新和使用强密码等。
  </p>

   <strong class="markup--strong markup--p-strong">
    使用无线驱动器或通过无线局域网（WLAN）共享文件
   </strong>
  </h3>
  <p class="graf graf--p">
   无线硬盘驱动器或闪存驱动器可用于一次性在团队中或多个人之间共享文件。
  </p>
  <p class="graf graf--p">
   wifi 驱动通常会附带说明和/或用于将手机连接到驱动器的应用程序，并且相对易于使用。为了安全起见，请记住在驱动器上设置密码。

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    通过点对点聊天进行交流
   </strong>
  </h3>
  <p class="graf graf--p">
   激进主义者网络可以帮您意识到有两个新的点对点消息传递应用程序是：
   <a class="markup--anchor markup--p-anchor" data-href="https://briarproject.org/" href="https://briarproject.org/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Briar
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    和
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://www.bfy.buzz/" href="http://www.bfy.buzz/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Bridgefy
    </strong>
   </a>
   。感兴趣的话您可以试试。
  </p>
  <p class="graf graf--p">
   Briar 是一个开源的端到端的加密消息传递应用程序，它不依赖中央服务器，而是在用户设备之间同步消息（因此内容保存在每个用户自己的设备上）。
  </p>
  <p class="graf graf--p">
   即使没有互联网只使用蓝牙或WiFi它也可以同步（当有互联网时，该应用会通过 Tor 网络同步）。
  </p>
  <p class="graf graf--p">
   Briar 还设有私人群组、公共论坛和博客。离线使用时，您的范围受到蓝牙或WiFi范围的限制（最大〜100米）。
  </p>
  <p class="graf graf--p">
   同时，Bridgefy 是使用蓝牙发送消息的端到端加密消息应用程序（使用“广播”功能时除外）。
  </p>
  <p class="graf graf--p">
   与 Briar 不同，在这里，消息可以通过沿其他 Bridgefy 用户的网状网络跳动而传播更长的距离（只有目标收件人才能阅读消息）。
  </p>
  <p class="graf graf--p">
   Bridgefy 缺乏 Briar 所拥有的私人群组、论坛和博客功能，但是它确实具有广播模式，通过该模式，您可以向固定范围内的7位 Bridgefy 用户发送消息，这些用户完全不需要成为您的联系人（广播消息不一定是加密的）。
  </p>

  </p>
  <p class="graf graf--p">
   短信不是端对端加密的，这意味着文本消息（及其元数据）可以被政府和移动运营商读取，也可以被黑客拦截。
  </p>

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e9%aa%8c%e8%af%81%e6%82%a8%e7%9a%84-vpn-%e8%bf%9e%e6%8e%a5%e6%98%af%e5%90%a6%e5%ae%89%e5%85%a8%ef%bc%9f/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e9%aa%8c%e8%af%81%e6%82%a8%e7%9a%84-vpn-%e8%bf%9e%e6%8e%a5%e6%98%af%e5%90%a6%e5%ae%89%e5%85%a8%ef%bc%9f/" rel="noopener noreferrer" target="_blank">

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e6%a3%80%e6%9f%a5%e6%82%a8%e7%9a%84vpn%e8%bf%9e%e6%8e%a5%e6%98%af%e5%90%a6%e5%ae%9e%e9%99%85%e5%8a%a0%e5%af%86/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e6%a3%80%e6%9f%a5%e6%82%a8%e7%9a%84vpn%e8%bf%9e%e6%8e%a5%e6%98%af%e5%90%a6%e5%ae%9e%e9%99%85%e5%8a%a0%e5%af%86/" rel="noopener noreferrer" target="_blank">

   请注意VPN提供商所在的国家/地区，以及根据其管辖范围可能要遵循的法律程序。
  </p>
  <p class="graf graf--p">
   此外，必须知道，政府批准的VPN实际上可以对数据进行全面监视和检查。切勿使用！
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    DNS服务器
   </strong>
  </p>
  <p class="graf graf--p">
   DNS（“域名系统”）服务器用来将用户在浏览器中键入的域名或URL转换为互联网用来标识网页的数字IP地址。ISP可以修改其控制的DNS服务器以阻止某些查询，或者返回不正确的页面以表明该网站不存在。
  </p>
  <p class="graf graf--p">
   2014年，土耳其 Recep TayyipErdoğan 试图使用这种技术在土耳其大选期间
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theverge.com/2014/3/21/5532522/turkey-twitter-ban-google-dns-workaround" href="https://www.theverge.com/2014/3/21/5532522/turkey-twitter-ban-google-dns-workaround" rel="noopener noreferrer" target="_blank">
    封锁 Twitter
   </a>
   。激进主义者迅速
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2014/mar/21/turkey-twitter-users-flout-ban-erdogan" href="https://www.theguardian.com/world/2014/mar/21/turkey-twitter-users-flout-ban-erdogan" rel="noopener noreferrer" target="_blank">
    绕过
   </a>
   了该禁令，他们分享了有关如何使用VPN和更改DNS服务器的逐步提示。
  </p>

  <p class="graf graf--p">
   您可以在手机的网络或wifi设置中更改默认的DNS服务器。
  </p>
  <p class="graf graf--p">
   除了默认的DNS服务器，您还可以使用其他DNS服务器（例如
   <a class="markup--anchor markup--p-anchor" data-href="https://developers.google.com/speed/public-dns/docs/using" href="https://developers.google.com/speed/public-dns/docs/using" rel="noopener noreferrer" target="_blank">
