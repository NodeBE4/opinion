   <h1>
    黄灿然：一生就是这样在泪水中
   </h1>
   <ul class="post-entry-meta">
    <li>
     By:
     <a href="http://www.ideobook.com/huangcanran/" title="查看 黄灿然 的作者主页">
      黄灿然
     </a>
     . 2008-9-12.
     <a href="http://www.ideobook.com/372/post-views-count/" title="统计说明">
      5,021
     </a>
    </li>
   </ul>
  </header>
  <div class="post-entry-content">
   <p>
    一生就是这样在泪水中默默吞忍。
    <br/>
    从黑暗中来，到白云中去，
    <br/>
    从根茎里来却不能回泥土里去，
    <br/>
    一生就是这样在时光中注满怨恨。
   </p>
   <p>
    一生就是这样在时光中戕害自身。
    <br/>
    在烟雾中思考，在思考中沉睡，
    <br/>
    在处心积虑中使灵魂伤痕累累——
    <br/>
    一生就是这样在火光中寻找灰烬。
   </p>
   <p>
    就是这样，用牙齿、用刺，
    <br/>
    用一个工具挖掘一生的问题；
    <br/>
    用回忆消愁，用前途截断退路，
    <br/>
    用春天的枝叶遮住眼中的耻辱。
   </p>
   <p>
    就是这样，把命运比作淤血，
    <br/>
    把挫折当成病，把悲哀的债务还清；
    <br/>
    就是这样发闷、发呆、发热，
    <br/>
    发出痛哭的叹息并在痛苦中酝酿绝症。
   </p>
   <p>
    一生就是这样在痛苦中模拟欢乐。
    <br/>
    做砖、做瓦、做牛、做马，
    <br/>
    做那被制度阻隔的团圆梦，
    <br/>
    一生就是这样在诺言中迁徙漂泊。
   </p>
   <p>
    一生就是这样在守望中舔起伤口。
    <br/>
    对人冷漠，对己残酷，
    <br/>
    对世界视若无睹，对花草不屑一顾，
    <br/>
    一生就是这样在反省中拒绝悔悟。
   </p>
   <p>
    就是这样，吃惊，然后镇静，
    <br/>
    蠢蠢欲动然后打消念头，
    <br/>
    猛地想起什么，又沮丧地被它逃走，
    <br/>
    就是这样困顿、疑惑、脑筋僵硬。
   </p>
   <p>
    就是这样建设、摧毁、不得安宁。
    <br/>
    在挖掘中被淘汰，在吞忍中被戕害，
    <br/>
    在碌碌无为中被迫离开——
    <br/>
    一生就是这样在迁徙漂泊中饱尝悲哀。
   </p>
   <p>
    一生就是这样在爱与被爱中不能尽情地爱。
    <br/>
    回忆一夜千金的温馨，把脑筋拧了又拧，
    <br/>
    回忆稻田、麦浪、飞蛾，想一生是多么失败，
    <br/>
    一生就是这样在饱尝挫折中积郁成病。
   </p>
   <p>
    人就是这样，在泪水中结束一生。
   </p>
