  </p>
  <p class="graf graf--p">
   智囊团卡托研究所（Cato Institute）呼吁国会调查联邦调查局是否在监视国内政治团体。因为该研究所是被监视目标之一。
  </p>
  <p class="graf graf--p">
   Cato 的研究员 Patrick Eddington 已提交了200多份《信息自由法案》（FOIA）请求，要求公开FBI的档案文件，涉及
   <strong class="markup--strong markup--p-strong">
    该间谍机构对政治倡导团体、公民自由组织、智囊团和各个政治领域出版物的秘密监视
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是请求失败了。
  </p>
  <p class="graf graf--p">
   联邦调查局表示，目前为止它 “既不能确认也不能否认” 是否已经在这些团体上收集了各种情报。
  </p>
  <p class="graf graf--p">
   这些被政府间谍瞄准的组织包括移
   <strong class="markup--strong markup--p-strong">
    民权利组织
   </strong>
   “Kids in Need of Defense”、
   <strong class="markup--strong markup--p-strong">
    变性人法律中心
   </strong>
   、前众议员 Ron Paul 的 “
   <strong class="markup--strong markup--p-strong">
    自由运动
   </strong>
   ” 组织、
   <strong class="markup--strong markup--p-strong">
    基层第四修正案倡导组织
   </strong>
   Restore the Fourth、卡托研究所、以及 Reason 基金会。
  </p>
  <p class="graf graf--p">
   您可以
   <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/6603016-OIP-Search-Adequacy-Affirmation-Glomar-Reason.html" href="https://www.documentcloud.org/documents/6603016-OIP-Search-Adequacy-Affirmation-Glomar-Reason.html" rel="noopener noreferrer" target="_blank">
    在此处
   </a>
   看到司法部的答复，该判决支持联邦调查局拒绝确认或否认情报记录的存在。
  </p>
  <p class="graf graf--p">
   卡托研究所在发布的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cato.org/news-releases/2020/1/7/cato-calls-congress-investigate-possible-fbi-surveillance-domestic-policy" href="https://www.cato.org/news-releases/2020/1/7/cato-calls-congress-investigate-possible-fbi-surveillance-domestic-policy" rel="noopener noreferrer" target="_blank">
    新闻稿
   </a>
   中说，这些回应 “表明国会有必要对联邦调查局的国内监视做法展开积极调查。”
  </p>
  <p class="graf graf--p">
   烂大街的 “无法确认也无法否认” 回应是一种常用托辞。该术语起源于1975年基于 FOIA 的一项滚石新闻记者针对 CIA 的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wnycstudios.org/podcasts/radiolab/articles/confirm-nor-deny" href="https://www.wnycstudios.org/podcasts/radiolab/articles/confirm-nor-deny" rel="noopener noreferrer" target="_blank">
    诉讼
   </a>
   。
  </p>
  <p class="graf graf--p">
   当时一位联邦法官裁定，如果中情局本身认为公开文档会 “损害国家安全”，则中央情报局可以拒绝承认此类记录的存在。此后，这种说法就传播到了其他联邦机构，甚至流传至州和地方政府机构，
   <a class="markup--anchor markup--p-anchor" data-href="https://reason.com/2018/03/29/new-york-high-court-rules-nypd-can-refus/" href="https://reason.com/2018/03/29/new-york-high-court-rules-nypd-can-refus/" rel="noopener noreferrer" target="_blank">
    例如
   </a>
   NYPD。
  </p>
  <p class="graf graf--p">
   这种回应是有意设计的，
   <strong class="markup--strong markup--p-strong">
    压制了公民监督政府活动的任何权利
   </strong>
   。Eddington 收到的回应可能毫无意义。
  </p>
  <p class="graf graf--p">
   然而，Eddington 说，尽管这种回应并不能证明联邦调查局已经收集了这些公民权利团体的情报，但是他们明显的针对性至少增加了上述团体遭遇违宪监视或信息搜集的可能性。
  </p>
  <p class="graf graf--p">
   联邦调查局拒绝对此事发表评论。
  </p>
  <p class="graf graf--p">
   联邦调查局
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2019/10/22/terrorism-fbi-political-dissent/" href="https://theintercept.com/2019/10/22/terrorism-fbi-political-dissent/" rel="noopener noreferrer" target="_blank">
    长期以来一直在监视持不同政见的政治团体
   </a>
   ，从20世纪初的社会主义者和劳工组织者、到民权运动领袖、9/11后的环保主义者和和平主义者、再到 “Black Lives Matter” 抗议者，都遭到监视。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%9c%a8%e6%b2%a1%e6%9c%89cctv%e7%9a%84%e6%97%b6%e4%bb%a3%e8%ad%a6%e5%af%9f%e5%90%8c%e6%a0%b7%e5%85%a8%e9%9d%a2%e7%9b%91%e6%8e%a7%e6%8a%97%e8%ae%ae%e6%b4%bb%e5%8a%a8%ef%bc%8c%e4%bb%96%e4%bb%ac/" href="https://www.iyouport.org/%e5%9c%a8%e6%b2%a1%e6%9c%89cctv%e7%9a%84%e6%97%b6%e4%bb%a3%e8%ad%a6%e5%af%9f%e5%90%8c%e6%a0%b7%e5%85%a8%e9%9d%a2%e7%9b%91%e6%8e%a7%e6%8a%97%e8%ae%ae%e6%b4%bb%e5%8a%a8%ef%bc%8c%e4%bb%96%e4%bb%ac/" rel="noopener noreferrer" target="_blank">
     在没有CCTV的时代警察同样全面监控抗议活动，他们是怎么做的
    </a>
    ？》
   </li>
  </ul>
  <p class="graf graf--p">
   2013年，卫报
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2013/nov/06/fbi-monitored-anti-war-website-in-error-documents" href="https://www.theguardian.com/world/2013/nov/06/fbi-monitored-anti-war-website-in-error-documents" rel="noopener noreferrer" target="_blank">
    报道
   </a>
   说，联邦调查局（FBI）对反战组织 Antiwar.com 的联合创始人进行了秘密监视并创建了 “威胁评估” 长达六年之久。去年九月，第九巡回上诉法院
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theamericanconservative.com/articles/court-fbi-must-destroy-memos-calling-antiwar-com-a-threat/" href="https://www.theamericanconservative.com/articles/court-fbi-must-destroy-memos-calling-antiwar-com-a-threat/" rel="noopener noreferrer" target="_blank">
    裁定
   </a>
   联邦调查局必须清除现场监视记录。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们是 *后爱德华·斯诺登* 的组织，因此我们特别需要关注受到FBI监视的可能性，” Restore the Fourth 组织的负责人 Alex Marthews 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们认为这种监视是非常不适当的，因为我们知道我们的活动是完全和平的和符合宪法的。”
  </p>
  <p class="graf graf--p">
   Eddington 说，任何这样的监视都是在危害言论自由。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “联邦调查局
   <strong class="markup--strong markup--p-strong">
    一贯持续地
   </strong>
   搜集有关正在行使其第一修正案权利的新闻机构或国内社会团体的数据，在没有任何犯罪依据的情况下，应该明确禁止这种行为。” ⚪️
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://reason.com/2020/01/08/is-the-fbi-snooping-on-political-groups-and-ideological-publications/" href="https://reason.com/2020/01/08/is-the-fbi-snooping-on-political-groups-and-ideological-publications/" rel="noopener noreferrer" target="_blank">
    Is the FBI Snooping on Political Groups and Ideological Publications?
   </a>
  </p>
