  <p>
   ​应投稿者要求，第五届网络社会年会“实践智慧之网”青年学者论坛征稿截稿日期延迟至2020年10月10日。征稿启事详见：
   <a href="http://mp.weixin.qq.com/s?__biz=MzI0MzQzNTczMA==&amp;mid=2247486358&amp;idx=1&amp;sn=93ee83f746a6cb5ab88f4db50ccccd42&amp;chksm=e96c5a2bde1bd33d7b72cccaba00343278ffc82ebee35988d1d4f7376ef78919ee1dda7a4540&amp;scene=21#wechat_redirect" rel="noreferrer noopener" target="_blank">
    第五届网络社会年会“实践智慧之网”全球青年学者论坛征稿
   </a>
   。
   <br/>
   <strong>
    欢迎各位继续来稿！
   </strong>
  </p>
  <hr class="wp-block-separator"/>
  <figure class="wp-block-image">
   <img alt="" src="https://mmbiz.qpic.cn/mmbiz_png/Jw8hS31CiclurbZV4TckeP6ZBqGciczUTwvXdPVe8l9ztD84tdVXGl8MYXdwW4qNickiakFbsOzRSf6IaG7JuTxkMg/640?wx_fmt=png"/>

  <p>
   新冠疫情的全球扩散看似令资本主义喘息暂止，实则令监控资本主义顺二昂起。今天，在由病毒爆发带来的集中化数位政权拥有绝对正当权利的线上世界里，线下的人们正面临更为严密的监控和个人资讯的集中化收集。
  </p>
  <p>
   假设实践（praxis）产生的实践智慧（phronesis）可以使人之行动揭蔽（aletheuein），以达操心的环视（solicitous circumspection）的效果。那么，为了践行亚里士多德所谓的“善用技术（techne）”，我们需要在实践智慧的指导下筹划和选择，以利用那些对揭露真理有重大缺陷，无法去蔽的，处在自身之外的技术。本次年会意在将（有所行动的）思想当作制作技术的始点，以便我们开启针对全球技术扩散与集中并行的讨论；去创造有分枝的未来，以谋求在病毒与治理紧急状态下的新生活。为此，中国美院跨媒体艺术学院网络社会研究所将邀请来自世界各地的学者报告全球最新研究心得，并对中国网络社会进行提问。
   <br/>
  </p>

   过往四届年会，
   <a href="http://mp.weixin.qq.com/s?__biz=MzI0MzQzNTczMA==&amp;mid=2247483807&amp;idx=1&amp;sn=a44b086645b6a386a24bf89ce6c749e2&amp;chksm=e96c5022de1bd934de454e3bfa5d954c12190fa55148b60fc26c8e7b637f560e44741c22e995&amp;scene=21#wechat_redirect" rel="noreferrer noopener" target="_blank">
    “网络化的力量”（2016）
   </a>
   ，
   <a href="http://mp.weixin.qq.com/s?__biz=MzI0MzQzNTczMA==&amp;mid=2247484604&amp;idx=1&amp;sn=5c3996bdb8c26a474008cc98676e18b9&amp;chksm=e96c5501de1bdc17dc03435c4e1b4b19dc4e2f6531df55d15d8b3e9e61bc8bc805f24a192242&amp;scene=21#wechat_redirect" rel="noreferrer noopener" target="_blank">
    “与列斐弗尔同行：算法时代下的都市与日常生活批判”（2017）

   ，
   <a href="http://mp.weixin.qq.com/s?__biz=MzI0MzQzNTczMA==&amp;mid=2247484979&amp;idx=1&amp;sn=3e54bdc3d5007479cc9e1c05dd42bec7&amp;chksm=e96c578ede1bde9873989a3e5aaf65e0fbc9c72b74337aa5570092bf519866a82ead4cd39d26&amp;scene=21#wechat_redirect" rel="noreferrer noopener" target="_blank">
    “智慧都市网络”（2018）
   </a>
   和
   <a href="http://mp.weixin.qq.com/s?__biz=MzI0MzQzNTczMA==&amp;mid=2247485676&amp;idx=1&amp;sn=28e5ec20c8f01426da87fa4db5fd5f55&amp;chksm=e96c5951de1bd047b11ebbd00f99d0aaa38f0d61779a697a899361335b14a551e9af204d9472&amp;scene=21#wechat_redirect" rel="noreferrer noopener" target="_blank">
    “网民21：超越个人账户”（2019）
   </a>
   中，我们连结了各国专家学者与行动者，开展了当前网络社会的理论批判与行动网络。第五届网络社会年会将以
   <strong>
    “实践智慧之网”（The Web of Phronesis)
   </strong>
   为题，于
   <strong>
    2020年10月20日
   </strong>
   及
   <strong>
    11月20-22日
   </strong>
   在中国美术学院召开。本届年会
   <strong>
    青年学者论坛
   </strong>
   将在
   <strong>
    2020年11月20-22日
   </strong>
   举办。本届年会现向全世界青年学者与行动者就以下
   <strong>
    五个主题
   </strong>
   公开征稿，维持“青年圆桌论坛”传统，以研究、实践、行动、交流为主。会后将集结主题演讲与论坛发表文章出书，由中国美术学院出版。

     <strong>
      主题五：加密主义 Crypto-ist
     </strong>
    </h4>
    <p class="has-large-font-size">
     网络
     <strong>
      隐私
     </strong>
     与
     <strong>
      安全
     </strong>
     本来是基本人权，现在，这一人权正被那些云端计算能力大幅提升的垄断社交媒体剥夺。对网民来说，拿回丧失的权利将是一场社会和经济的双料革命。在过去十年里，为这一革命做准备的技术已有大幅进展。它们包括公钥加密、点对点加密传输，零知识证明以及各种用于互动、认证和验证的软件协议等。无论如何，我们将都是加密主义者，差别在于人们选择主宰自己权利还是还是被主宰。
