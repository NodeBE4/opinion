 <div class="entry-content">
  <p class="graf graf--p">
   中国最大的科技巨头正在为美国最大的科技寡头提供支持。
  </p>
  <p class="graf graf--p">
   据与会者介绍，Qingteng Club 是一家隶属于社交媒体和游戏巨头腾讯控股有限公司的组织，于上周在加利福尼亚举办的一场私人活动中接待了来自中国和美国在线视频行业的一大堆高管和名人。
  </p>
  <p class="graf graf--p">
   这项名为东西方论坛的活动在 VidCon 的 Anaheim 酒店举行，这是一个为在线影响力人士的粉丝举办的大会。
  </p>
  <p class="graf graf--p">
   全能中文应用微信的所有者
   <strong class="markup--strong markup--p-strong">
    腾讯正试图鼓励更多美国社交媒体明星在世界第二大经济体 —— 中国 —— 开展业务
   </strong>
   。
  </p>
  <p class="graf graf--p">
   该活动的开幕式名为“腾讯如何帮助您在中国实现影响力业务”。
  </p>
  <p class="graf graf--p">
   腾讯在利用中国新兴市场方面优于 YouTube：谷歌拥有的视频服务在该国受阻。
  </p>
  <p class="graf graf--p">
   此时恰逢高堡奇人（中美）竞争最激烈的时段。
  </p>
  <p class="graf graf--p">
   中国的“抖音”受欢迎程度已经动摇了中国的科技产业，电子商务巨头阿里巴巴集团控股有限公司、搜索引领者百度和腾讯等寡头公司也被迫捍卫自己的地盘。
  </p>
  <p class="graf graf--p">
   腾讯的微信消息服务被十多亿人使用，之前已经阻止了与抖音的链接。而由百度控制的 Netflix 风格的流媒体服务爱奇艺也正在开发一款竞争性的应用。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “东西方论坛是一项独家活动，将东西方的技术和娱乐行业的领导者聚集在一起，相互了解，建立潜在的合作关系，” 该峰会的声明中是这样写的。
  </p>
  <p class="graf graf--p">
   来自腾讯百度等中国寡头的高管们都在今年的 VidCon 上发表了演讲，这是为 YouTube 明显与粉丝们见面而举办的活动。
  </p>
  <p class="graf graf--p">
   该会议已经扩展到了其他集中化在线平台，如 亚马逊公司的 Twitch、Facebook、Instagram 和 Twitter。
  </p>
  <p class="graf graf--p">
   东西方论坛还将汇集腾讯的高管、中国创业公司 RED 和 Bilibili Inc. 的创始人，以及社交媒体网红 Jordi 和 Azzy。此外，它还有望吸引法恩兄弟的媒体高管，该公司在 YouTube 上运营着一些最受欢迎的频道。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “在 VidCon 有一个非常庞大的且非常豪华的中国高管代表团”，香港媒体执行官兼制片人 Jasper Donat 说，“近年来很难让很多中国人进入美国，但这种会议就可以做到”。⚪️
  </p>
  <p>
   注：彭博社这篇报道写得很差劲，完全没有说明白为什么中国寡头要如此卖力地为美国寡头进入中国开辟路径 —— 简单说：与
   <a href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e7%a7%91%e6%8a%80%e5%b7%a8%e5%a4%b4%e5%a6%82%e4%bd%95%e5%b8%ae%e5%8a%a9%e4%b8%ad%e5%9b%bd%e5%bb%ba%e7%ab%8b%e5%a4%a7%e8%a7%84%e6%a8%a1%e7%9b%91%e8%a7%86%e7%b3%bb%e7%bb%9f/" rel="noopener noreferrer" target="_blank">
    中国监视技术在美国的鼎力协助下的腾飞
   </a>
   之间的关系究竟是什么。也许彭博社在努力避免承认这点？
  </p>
