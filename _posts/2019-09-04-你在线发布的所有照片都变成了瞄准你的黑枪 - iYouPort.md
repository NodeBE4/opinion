  <p class="graf graf--p">
   ⚠️如果您的面部可以在网上找到，那么现在它可能已经是面部识别数据库的一部分了。
  </p>
  <p class="graf graf--p">
   我们曾经详细介绍过在线上传照片的危害，详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%BD%A0%E6%9B%BE%E7%BB%8F%E5%9C%A8%E7%A4%BE%E4%BA%A4%E7%BD%91%E7%BB%9C%E4%B8%8A%E5%8F%91%E5%B8%83%E8%BF%87%E8%87%AA%E5%B7%B1%E7%9A%84%E7%85%A7%E7%89%87%EF%BC%9F%E4%BD%A0%E5%8D%B1%E9%99%A9%E4%BA%86/" href="https://www.iyouport.org/%E4%BD%A0%E6%9B%BE%E7%BB%8F%E5%9C%A8%E7%A4%BE%E4%BA%A4%E7%BD%91%E7%BB%9C%E4%B8%8A%E5%8F%91%E5%B8%83%E8%BF%87%E8%87%AA%E5%B7%B1%E7%9A%84%E7%85%A7%E7%89%87%EF%BC%9F%E4%BD%A0%E5%8D%B1%E9%99%A9%E4%BA%86/" rel="noopener noreferrer" target="_blank">
    你曾经在社交网络上发布过自己的照片？你危险了！
   </a>
   》
  </p>
  <p class="graf graf--p">
   执法部门使用的则是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.techdirt.com/articles/20140819/11390628257/fbis-criminal-database-is-filling-up-with-non-criminals-no-one-law-enforcement-seems-to-care.shtml" href="https://www.techdirt.com/articles/20140819/11390628257/fbis-criminal-database-is-filling-up-with-non-criminals-no-one-law-enforcement-seems-to-care.shtml" rel="noopener noreferrer" target="_blank">
    数以百万计
   </a>
   非犯罪者的面部数据，或者说不管你是不是遵纪守法，都会被收集以监视，只要你在线上传过自己的照片。
  </p>
  <p class="graf graf--p">
   虽然有些面孔不在那些数据库中，但是！
   <strong class="markup--strong markup--p-strong">
    它们被用来训练面部识别AI算法，着眼于将该算法卖给执法部门和其他政府机构。
   </strong>
  </p>
  <p class="graf graf--p">
   那些鼓励你晒照片应用程序很可能只是个老鼠夹子。
  </p>
  <p class="graf graf--p">
   已经发现了另一家照片存储公司使用用户的照片来调试面部识别监控软件……所有这一切都没有得到那些被当成了技术工厂饲料的人的同意。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “留下回忆”：这是照片存储应用程序网站上的口号，伴随着草书的徽标和一张名为“周末与爷爷在一起”的专辑示例。
  </p>
  <p class="graf graf--p">
   关于 Ever 品牌的所有信息都是温暖而模糊的，所谓的在线分享您的“最佳时刻”，同时释放手机上的空间……
  </p>
  <p class="graf graf--p">
   然而这一切都是骗局。
  </p>
  <p class="graf graf--p">
   在 Ever 的网站或应用程序中你无法找到对用户的警告 —— 他们没有提醒人们分享的照片被用于训练公司的面部识别系统，这项技术将被出售给私营公司、执法部门和军方。（NBC 新闻在四月份与公司联系后，该警告已被添加到隐私政策中。而在此前完全没有）
  </p>
  <p class="graf graf--p">
   Ever 的首席执行官 Doug Aley 告诉 NBC 新闻，Ever AI “不会与面部识别客户分享照片或任何有关用户的识别性信息”。
  </p>
  <p class="graf graf--p">
   但他就是不告诉你监视算法就是这些用户“帮助”创建的 —— 当然是在不知不觉中。
  </p>
  <p class="graf graf--p">
   很多人宁愿不参与创建监控技术。
  </p>
  <p class="graf graf--p">
   大多数人从未在为执法部门、情报机构、和为美国制作产品的公司寻找工作。但在没有得到任何通知的情况下，他们认为自己与家人和朋友的亲密照片已被用于使大规模监视更容易、更普遍，还可能“更好”。
  </p>
  <p class="graf graf--p">
   Ever 只是最新被发现的一个案例而已。在此之前，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnn.com/2019/04/19/tech/ai-facial-recognition/index.html" href="https://www.cnn.com/2019/04/19/tech/ai-facial-recognition/index.html" rel="noopener noreferrer" target="_blank">
    Flickr 照片已经被 IBM
   </a>
   编制的面部识别技术所使用。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    ⚠️这张可爱的家庭合影照片于2013年上传到照片共享网站 Flickr，而且不仅可爱；它具有不同位置角度的一堆不同面孔，对于训练面部识别系统来说非常有用，面部识别系统使用人工智能识别照片和视频中的人物。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    IBM利用一百万张图片来创建一个新项目，旨在帮助研究人员提升面部识别技术中的准确性，或称为
   </em>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.ibm.com/blogs/research/2019/01/diversity-in-faces/" href="https://www.ibm.com/blogs/research/2019/01/diversity-in-faces/" rel="noopener noreferrer" target="_blank">
    <em class="markup--em markup--p-em">
     面部多样性
    </em>
   </a>
   。
  </p>
  <p class="graf graf--p">
   IBM 甚至还为未经许可将人们的照片用于其数据集而“道歉”。它表示欢迎用户随时选择退出，但是，⚠️并没有给用户提供任何工具来查明他们的照片是否已被使用。
  </p>
  <p class="graf graf--p">
   除了将您的 Flickr ID 交给 IBM 之外，也没有任何方法可以快速删除找到的照片。呵。
  </p>
  <p class="graf graf--p">
   任何互联网用户都会发现使用其他人的照片来编译数据集是多么容易。
  </p>
  <p class="graf graf--p">
   Tinder 用户有许多动机可以将他们的数据上传到该约会应用。但是，为可训练卷积神经网络的可下载数据集提供面部生物识别数据，可能并不是这些用户在注册时能想到的事。
  </p>
  <p class="graf graf--p">
   然而，
   <strong class="markup--strong markup--p-strong">
    谷歌
   </strong>
   最近收购的机器学习和数据科学竞赛平台 Kaggle 已经将用户数据上传了一个面部数据集，它说这是通过利用 Tinder 的 API 从该约会应用程序中搜集的40,000张个人资料照片创建的 —— 男女各20,000个。
  </p>
  <p class="graf graf--p">
   该数据集名为
   <a class="markup--anchor markup--p-anchor" data-href="https://www.kaggle.com/scolianni/people-of-tinder" href="https://www.kaggle.com/scolianni/people-of-tinder" rel="noopener noreferrer" target="_blank">
    People of Tinder
   </a>
   ，由六个可下载的 zip 文件组成，其中四个包含大约10,000张资料照片，两个文件包含每个性别约500张图像的样本集。
  </p>
  <p class="graf graf--p">
   Tinder 的反应是将此称为“违反其服务条款”。但是，这种决心并没有消除损害，也没有阻止其他人做同样的事。
  </p>
  <p class="graf graf--p">
   正如 TechCrunch 所说，Tinder 的用户并不满意他们的照片被他们不知道是谁的人拿去进行研究 —— 其中一些照片从未在其他平台上传过。
  </p>
  <p class="graf graf--p">
   这么做并非非法，但是，透明度是关键，为什么在用户不知情的情况下偷偷摸摸地做？
  </p>
  <p class="graf graf--p">
   在 Ever 的情况下更糟糕的是，
   <strong class="markup--strong markup--p-strong">
    该应用程序公司本身就是开发面部识别软件的一方，这本应该让用户质疑该应用程序开发人员的意图。然而却没有
   </strong>
   ，那些毫无戒心的用户只是看到了一个可以玩和晒的地方……
  </p>
  <p class="graf graf--p">
   中国的面部识别算法多次被美国“钦佩”，与中国用户如此爱晒有多少关系？
  </p>
