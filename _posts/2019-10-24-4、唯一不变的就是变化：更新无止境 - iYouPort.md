  <p class="graf graf--p">
   人们聚集在一起讨论，最初的目的很有可能是获得一个可以增进效力的共识；但很多讨论的结果反而让事情变得更糟了。这是为什么？
  </p>
  <p class="graf graf--p">
   原因也许有很多，但其中最主要的原因大概是，每个人都不想改变自己，而是希望改变他人。
  </p>
  <p class="graf graf--p">
   很容易理解，资深的活动家都会有自己的“成功经验”；“我曾经做到了”这一句式非常具有说服力。然而，它仅仅能说服，却并不意味着正确。
  </p>
  <p class="graf graf--p">
   并且，还有可能更危险。曾经成功的活动家拒绝进一步的优化式改变、似乎下一次改变是在“否定其做出的成绩”。
  </p>
  <p class="graf graf--p">
   其结果是，“资深人士”变成了团体中前进的障碍。新生力量被所谓的资历竞争边缘化；积极人士内部（而不是整个社会）以更鲜明的代际撕裂。
  </p>
  <p class="graf graf--p">
   这是一系列糟糕的现象，而且并不罕见。
  </p>
  <p class="graf graf--p">
   其实原因很简单，只是因为人们忽略了一个常识。
  </p>
  <p class="graf graf--p">
   生活在暴政中的积极人士寻求正义，生活在压迫中的活动家寻求真相，生活在警察国家的人们寻求自由……
  </p>
  <p class="graf graf--p">
   人们寻求变革是因为对现状感到遗憾 —— 但也因为是认识到了需要持续的改进。
  </p>
  <p class="graf graf--p">
   然而，当改变真正到来时、与人们已经习惯的日常发生矛盾时，许多人会一下子陷入慌乱。
  </p>
  <p class="graf graf--p">
   虽然改变一直是人们所期待的和为之努力的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    事实上变化应该始于自我。
   </strong>
  </p>
  <p class="graf graf--p">
   我们所处的社会、政治、环境和经济不仅必须改变，我们自己也必须成为自己想要在这个世界上看到的改变。
  </p>
  <p class="graf graf--p">
   ⚠️为了有效，反抗者必须始终审查、评估和调整自己的方法和模式。
  </p>
  <p class="graf graf--p">
   必须愿意承认并改变自己过时的心态并放弃所有常数。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果真的想要改变世界，就必须认识到：变化才是唯一不变的。
   </strong>
  </p>
  <p class="graf graf--p">
   从变化是唯一不变的前提出发，你很快会能意识到，💡并没有唯一“正确”的方式来参与集体行动，行动需要不断采取许多*正确的方式*。
  </p>
  <p class="graf graf--p">
   为了实现变革，行动必须保持持续发展的状态。
  </p>
  <p class="graf graf--p">
   💡反抗运动的整体是一个不断反思的过程，也是不断学习和提升的旅程，就像一个圆圈一样，没有开始也没有结束。
  </p>
  <p class="graf graf--p">
   例如，当首次启动运动或抗议活动时，它可以从一组五人的特定决策过程开始。
  </p>
  <p class="graf graf--p">
   随着时间的推移，这个团队对自己的运作模式感到满意；但是，⚠️当你的团队成长为50名组织者的时候，你的情况将会大不相同，你需要接受这样一个事实，即：你的决策过程需要改变。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这是适用于组织动员过程中的所有关键节点，从招募到战术再到组织等等。
   </strong>
  </p>
  <p class="graf graf--p">
   💡拥抱*变革是唯一不变*的原则有助于活动家对他人的反馈和想法更加开放。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-6112 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-13.jpg?resize=1100%2C825&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-13.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-13.jpg?resize=300%2C225&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-13.jpg?resize=768%2C576&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-13.jpg?resize=1024%2C768&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-13.jpg?resize=1100%2C825&amp;ssl=1 1100w" data-recalc-dims="1" height="825" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-13.jpg?resize=1100%2C825&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   仅仅因为一个新的观点与你自己的观点不一致，或者它否定了大多数人深信不疑的观点，这
   <strong class="markup--strong markup--p-strong">
    并不意味着
   </strong>
   它应该被抛弃。
  </p>
  <p class="graf graf--p">
   相反，
   <strong class="markup--strong markup--p-strong">
    正是在这个时刻，该团体必须暂停并重新思考自己的心态
   </strong>
   。
  </p>
  <p class="graf graf--p">
   一个好的活动家需要能放开自我、谦虚、欣赏他人的洞察力和观点；有意识地寻找更新的和更有效的行动方法。
  </p>
  <p class="graf graf--p">
   同样，仅仅因为一个策略有效，也并不意味着我们应该一遍又一遍地使用它。详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/2%e3%80%81%e6%9c%89%e7%ad%96%e7%95%a5%e5%9c%b0%e5%8d%87%e7%ba%a7%ef%bc%9a%e4%b8%8d%e8%a6%81%e7%88%b1%e4%b8%8a%e6%88%98%e6%9c%af/" href="https://www.iyouport.org/2%e3%80%81%e6%9c%89%e7%ad%96%e7%95%a5%e5%9c%b0%e5%8d%87%e7%ba%a7%ef%bc%9a%e4%b8%8d%e8%a6%81%e7%88%b1%e4%b8%8a%e6%88%98%e6%9c%af/" rel="noopener noreferrer" target="_blank">
    不要爱上你的战术
   </a>
   》。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    活动家应该对实践和失败的可能性感到满意
   </strong>
   ；只有这样，才能做到丰富知识并获得提升。
  </p>
  <p class="graf graf--p">
   失败意味着并没有停止尝试，这本身就值得欣慰。失败才是创新业务的核心。
  </p>
  <p class="graf graf--p">
   以失败为契机，与可能提供新见解或让您访问不同网络或受众的新人联系并讨论挑战。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    由于资源有限和环境的不完善，人们在大部分时间里都会处于失败状态。没关系。如果你能事先接受这点，它就会给你诚实的自我反省的自由；它允许组织者为所有人提供机会。
   </strong>
  </p>
  <p class="graf graf--p">
   组织是一个不断学习、不断调整和迭代的过程；一个持续的实践循环和更新的过程。
  </p>
  <p class="graf graf--p">
   最后，要了解更广泛的背景以及您当地的情况。详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/1%e3%80%81%e4%ba%86%e8%a7%a3%e4%bd%a0%e7%9a%84%e5%9c%b0%e5%bd%a2%e6%98%af%e7%ad%96%e5%88%92%e8%a1%8c%e5%8a%a8%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" href="https://www.iyouport.org/1%e3%80%81%e4%ba%86%e8%a7%a3%e4%bd%a0%e7%9a%84%e5%9c%b0%e5%bd%a2%e6%98%af%e7%ad%96%e5%88%92%e8%a1%8c%e5%8a%a8%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" rel="noopener noreferrer" target="_blank">
    了解你的地形是策划行动的第一步
   </a>
   》。
  </p>
  <p class="graf graf--p">
   当今世界的社会政治动态是充满活力和复杂的，需要积极人士始终保持关注，分析正在发生的事，并了解动态变化如何影响你的运动和行动。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    潜在风险 ——
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️在应用这一原理时，组织者必须意识到三种风险：多数人的暴政、集体主义思维、和解释学循环。
  </p>
  <p class="graf graf--p">
   集体主义思维方式和多数人暴政会令少数派声音被迅速边缘化。
  </p>
  <p class="graf graf--p">
   然而，正是这些不同的声音往往能提供最有洞察力的观点、或倾向于避免集体灾难的深思熟虑。
  </p>
  <p class="graf graf--p">
   陷入解释学循环的群体倾向于沉迷无休止的理论讨论和辩论会。
  </p>
  <p class="graf graf--p">
   ⚠️这在中国的活动家群体中非常普遍。最终的结果大家都知道：积极人士四分五裂，没有任何人获得提升，也没有整体进展。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果所讨论的改进、想法和可能性在实践中没有得到应用，那么讨论本身只不过是纸上谈兵。
   </strong>
  </p>
  <p class="graf graf--p">
   有效的激进活动需要遵循一个循环。
  </p>
  <p class="graf graf--p">
   💡从变革发生的理论开始；然后根据我们的理论采取行动；然后退后一步，思考行动是如何进行的，以便重新塑造理论。
  </p>
  <p class="graf graf--p">
   基本上，实践意味着“学习”。
  </p>
  <p class="graf graf--p">
   这看似简单，但是很少有活动家能真正做到这样。
  </p>

   Praxis 要求我们成为自己的经验和背景的学生。
  </p>
  <p class="graf graf--p">
   这不仅仅是关于机智和反思；它还涉及建立促进战略、听取汇报和修订习惯的特定行为和群体规范。
  </p>
  <p class="graf graf--p">
   这是关于您的团队的组织风格、组织结构和领导力动态。
  </p>
  <p class="graf graf--p">
   我们举个例子。
  </p>
  <p class="graf graf--p">
   假设这里是一所大学的学生会。如果我们的团队缺乏实践，活动家可能会这样说：“让我们欢迎激进的思想家X在我们的校园发言！”我们认为这个活动将是“好的”。
  </p>
  <p class="graf graf--p">
   然后活动开启。有很多人参加，但之后很快我们的团队内部对此感觉不够完美。于是我们决定继续前进，并举办另一场活动……
  </p>
  <p class="graf graf--p">
   这种情况听起来很常见吧？⚠️看起来好像很活跃，但是没有成功的可能性。
  </p>
  <p class="graf graf--p">
   因为缺乏方向性。没有实际的理论，也没有反思的基础。
  </p>
  <p class="graf graf--p">
   相反，您应该从一个理论开始。
  </p>
  <p class="graf graf--p">
   开始小组会议时这样说：“将激进的思想家X带到我们的校园将有助于我们的活动。他可以谈论为什么激进主义是强大的，激进主义为什么将会吸引那些尚未参与我们的活动的新观众。我们要在最火爆的咖啡店和餐馆张贴传单。将有三百人参加，五十人将报到，其中五位代表将出席我们的下一次会议……”。
  </p>
  <p class="graf graf--p">
   现在这是一个真正的理论了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    它有一个明确的逻辑，一个如何执行行动的过程，以及您期望的具体可衡量的结果。
   </strong>
  </p>
  <p class="graf graf--p">
   继续。
  </p>
  <p class="graf graf--p">
   不理想的事发生了。只有一百人参加，其中大多数人已经与您的组织合作，所以这意味着只有少数人在列表上签名，没有新人参加您的下一次会议。
  </p>
  <p class="graf graf--p">
   别急，你现在有了反思的真实基础。
  </p>
  <p class="graf graf--p">
   你可以汇报整个过程，
   <strong class="markup--strong markup--p-strong">
    而不是主观地谈论
   </strong>
   你自己认为它是好事还是坏事；你可以谈谈为什么它不符合你的成功指标，以及下次应该做些什么。
  </p>
  <p class="graf graf--p">
   💡这些反思将塑造您的组织下一个更加成功的活动方式。
  </p>
  <p class="graf graf--p">
   💡组织者应该让实践循环始终在头脑中旋转。保持婴儿般的好奇心，从周围发生的事中获得学习。
  </p>
  <p class="graf graf--p">
   但是，⚠️必需知道，想要在你的小组中建立实践文化的关键是：你的整个团队都可以这样学习，而不仅仅是几个组织者。
  </p>
  <p class="graf graf--p">
   当您与团队一起制定计划和目标、然后进行真正的汇报后，所有人都可以从中获得学习。
  </p>
  <p class="graf graf--p">
   如果没能做到这点，您的组织会走向沉闷和荒芜，将被更有智慧和谦逊的新生组织所取代。⚪️
