  </p>
  <p class="graf graf--p">
   <em>
    此举表明，无论算法何时起作用，面部识别监控都是危险的
   </em>
  </p>
  <p class="graf graf--p">
   如果您乘坐的是华盛顿特区地铁，而一个穿着白色连身衣的人用绑在头上的手机扫描您的脸，您可能会感到震惊。
  </p>
  <p class="graf graf--p">
   数字权利激进主义者正在使用亚马逊销售的面部扫描技术 Rekognition 来扫描国会大厅外和城市最繁忙的地铁站内成千上万名DC居民的面部，以展示面部识别监控的有害后果。
  </p>
  <p class="graf graf--p">
   全天，数字权利组织 “为未来而战” 将扫描到的脸部数据与记者、游说者和国会议员的数据库进行交叉检查。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们将看到准确度有多高”，为未来而战的主管 Evan Greer 对主板说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们试图提出的观点是，无论以哪种方式进行，面部识别技术都是危险的。当出于所谓的执法目的误认人员时会更加危险，准确识别同样危险。这意味着任何人，无论是政府、私人公司还是你令人毛骨悚然的前任，都可以使用它来瞄准你。”
  </p>
  <p class="graf graf--p">
   几个小时内扫描行动已经处理了近14,000张面孔，确定了一名国会议员、七名记者、和25名游说者。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12464 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18-scaled.jpeg?resize=1100%2C1467&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18-scaled.jpeg?w=1536&amp;ssl=1 1536w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18-scaled.jpeg?resize=225%2C300&amp;ssl=1 225w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18-scaled.jpeg?resize=768%2C1024&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18-scaled.jpeg?resize=1152%2C1536&amp;ssl=1 1152w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18-scaled.jpeg?resize=1100%2C1467&amp;ssl=1 1100w" data-recalc-dims="1" height="1467" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18-scaled.jpeg?resize=1100%2C1467&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12464" data-recalc-dims="1" height="1467" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18-scaled.jpeg?resize=1100%2C1467&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18-scaled.jpeg?w=1536&amp;ssl=1 1536w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18-scaled.jpeg?resize=225%2C300&amp;ssl=1 225w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18-scaled.jpeg?resize=768%2C1024&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18-scaled.jpeg?resize=1152%2C1536&amp;ssl=1 1152w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18-scaled.jpeg?resize=1100%2C1467&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   一些数字权利活动人士虽然承认今天这场行动是出于善意的，但他们批评并质疑这种抗议形式的广泛影响。Maccomb 社区学院的数字化政策专家 Chris Gilliard 发
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/hypervisible/status/1195009444171722752" href="https://twitter.com/hypervisible/status/1195009444171722752" rel="noopener noreferrer" target="_blank">
    推文
   </a>
   说：“对成千上万不同意这样做的人进行面部监控，其中许多人是黑人、棕色人，也许没有证件，这真的好吗？”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这是通过突出重点来试图做好事的行动，但它的作用与批评一样。这实际上并不能提高抵御能力。只会让人们担心这个问题，却不知道该怎么办”，伦敦维多利亚和阿尔伯特博物馆数字设计策展人 ​​Natalie Kane 说。
  </p>
  <p class="graf graf--p">
   但 “为未来而战” 组织的 Greer 说，让人们关注正是这场行动的重点 —— 促使他们考虑了如何在最容易受到监视的社区中开展这场行动。
  </p>
  <p class="graf graf--p">
   她说，“这些行动地点是精心选择的，是受监视的重灾区，而不是其他地点我们的团队在被监视方面有充分的个人经验，并感到反击的紧迫性和重要性”。
  </p>
  <p class="graf graf--p">
   特别是，他们希望向政客，记者和说客展示这种技术如何直接影响他们自己 — 不仅仅是老百姓，而不是以抽象的方式向从事政治工作的专业人员介绍这种技术。
  </p>
  <p class="graf graf--p">
   Greer 说：“这是在直观地体现危险，并呼吁立法者禁止这种技术。”
  </p>
  <p class="graf graf--p">
   这场行动是在人们对面部识别技术的道德规范日益关注的时期开展的，该技术已经被广泛用于警察维稳和监视有色人种、移民和其他弱势人群。
  </p>
  <p class="graf graf--p">
   美国公民自由联盟（ACLU）表示，亚马逊的面部识别软件对社区构成了严重威胁，其中包括有色人种和移民 —— 尤其是其惊人的错误率，该软件误判了加利福尼亚州的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnet.com/news/amazons-rekognition-falsely-matched-lawmakers-to-criminals-again-aclu-says/" href="https://www.cnet.com/news/amazons-rekognition-falsely-matched-lawmakers-to-criminals-again-aclu-says/" rel="noopener noreferrer" target="_blank">
    26名议员
   </a>
   。
  </p>
  <p class="graf graf--p">
   到目前为止，奥兰多警察局以及俄勒冈州的警长部门已经部署了 Amazon 的 Rekognition。2018年，亚马逊与移民与海关执法局（ICE）
   <a class="markup--anchor markup--p-anchor" data-href="https://www.thedailybeast.com/amazon-pushes-ice-to-buy-its-face-recognition-surveillance-tech" href="https://www.thedailybeast.com/amazon-pushes-ice-to-buy-its-face-recognition-surveillance-tech" rel="noopener noreferrer" target="_blank">
    亲密接触
   </a>
   ，以推销其 Rekognition 监视系统，从而招致其自身工程师的强烈反对。
  </p>
  <p class="graf graf--p">
   同时，参议员 Chris Coons（特拉华州）和 Mike Lee（犹他州）
   <a class="markup--anchor markup--p-anchor" data-href="https://www.coons.senate.gov/imo/media/doc/FRTWA%20One-Pager%20FinalFinal.pdf" href="https://www.coons.senate.gov/imo/media/doc/FRTWA%20One-Pager%20FinalFinal.pdf" rel="noopener noreferrer" target="_blank">
    提出
   </a>
   了一项立法，要求执法人员必须获得逮捕令才能使用面部识别技术进行公共监视。
  </p>
  <p class="graf graf--p">
   Lee 参议员在新闻稿中说：“面部识别技术可以成为执法人员的有力工具。” “但是，它的强大功能也使它容易受到滥用。这就是为什么美国公民应受到保护，防止面部识别滥用。”
  </p>
  <p class="graf graf--p">
   领导这场反抗行动的组织 “为未来而战”（Fight for the Future）反对新立法，认为：除了完全禁止执法部门使用监视技术外，其他措施不足以成功。
  </p>
  <p class="graf graf--p">
   该组织说：“它存在巨大的漏洞，可以在没有适当的司法监督的情况下授权将面部识别用于各种虐待目的 …… 很高兴看到国会想到解决这个问题，但是，这项法案远远不够。”
  </p>
  <p class="graf graf--p">
   立法者已经在包括旧金山，伯克利，奥克兰和马萨诸塞州索默维尔在内的多个城市彻底禁止了这项技术。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%a6%81%e6%ad%a2%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e5%8f%af%e4%bb%a5%e5%81%9a%e5%88%b0%e4%bb%80%e4%b9%88%e3%80%81%e4%b8%8d%e8%83%bd%e5%81%9a%e5%88%b0%e4%bb%80%e4%b9%88/" href="https://www.iyouport.org/%e7%a6%81%e6%ad%a2%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e5%8f%af%e4%bb%a5%e5%81%9a%e5%88%b0%e4%bb%80%e4%b9%88%e3%80%81%e4%b8%8d%e8%83%bd%e5%81%9a%e5%88%b0%e4%bb%80%e4%b9%88/" rel="noopener noreferrer" target="_blank">
     禁止面部识别可以做到什么、不能做到什么
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   在采取行动之后，“为未来而战” 组织将发布一个工具，该工具将允许在这场行动地点居住的人们检查自己的脸部是否被扫描过。这些照片和数据将在两周后销毁。⚪️
  </p>
