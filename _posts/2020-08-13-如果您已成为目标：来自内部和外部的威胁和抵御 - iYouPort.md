  <p class="graf graf--p">
   社交工程学有非常多的用法，其结果完全取决于目标。其中就包括当权者和间谍机构对公民施加的操纵和控制。
  </p>
  <p class="graf graf--p">
   👉 “如果您已成为目标”，您就必须随时留意这些迹象，以准确判断当权者的真实目的。否则，您可能会越陷越深，很快失去反击的能力。
  </p>
  <p class="graf graf--p">
   而另一方面，在出现下面部分迹象时，您可能更加难以动员人们反抗。
  </p>
  <p class="graf graf--p">
   那么这些威胁迹象都包括什么？它分为外部和内部两部分。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     —— 外部威胁 ——
    </strong>
   </span>
  </h3>
  <h3 class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     1、稀缺的力量
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   稀缺性指的是，人们被告知，他们最想要的东西非常有限和稀罕，要想获得它，就必须遵循某种态度和行为。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13394 jetpack-lazy-image" data-lazy-sizes="(max-width: 634px) 100vw, 634px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/1-2.png?resize=634%2C1284&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/1-2.png?w=634&amp;ssl=1 634w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/1-2.png?resize=148%2C300&amp;ssl=1 148w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/1-2.png?resize=506%2C1024&amp;ssl=1 506w" data-recalc-dims="1" height="1284" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/1-2.png?resize=634%2C1284&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="634"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13394" data-recalc-dims="1" height="1284" sizes="(max-width: 634px) 100vw, 634px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/1-2.png?resize=634%2C1284&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/1-2.png?w=634&amp;ssl=1 634w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/1-2.png?resize=148%2C300&amp;ssl=1 148w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/1-2.png?resize=506%2C1024&amp;ssl=1 506w" width="634"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   但是，他们往往不会明确指出所谓的 “正当态度和行为” 究竟是什么，而是向表现出让当权者满意的行为的人展示奖赏。
  </p>
  <p class="graf graf--p">
   有些时候心理捷径是有益的，可以使人们能够更有效地应对生活的复杂性。但是，
   <strong class="markup--strong markup--p-strong">
    稀缺性的恐吓却具有严重的误导性。
   </strong>
  </p>
  <p class="graf graf--p">
   而且，
   <strong class="markup--strong markup--p-strong">
    这个作用是双向促进的
   </strong>
   。当有人得到了别人没有得到的东西～～ 比如特殊权限、食物、水，等等，就会使他们感到自己 “特殊”，“独特”，“崇高”，并将这些人置于捍卫暴政的位置上。他们害怕失去这份特权。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    害怕失去是一种依恋。依恋导致懦弱和奴性。
   </strong>
  </p>
  <p class="graf graf--p">
   一个例子是南非政府，他们夺走了人们生命所必需的东西，并使它 “稀缺” 且仅供支持政府的人可用 —— 这是一种恶意的但非常有效的操纵策略。
  </p>
  <p class="graf graf--p">
   更糟糕的是，
   <strong class="markup--strong markup--p-strong">
    稀缺性创造了使稀缺性永存的心态
   </strong>
   。
  </p>
  <p class="graf graf--p">
   穷人保持贫穷，孤独者保持孤独，忙碌者保持忙碌 …… 稀缺性创造了一种使稀缺性永存的心态。
  </p>
  <p class="graf graf--p">
   如果所有这些看起来都很惨淡，那么另一种观点就出现了：穷人之所以贫穷，是因为他们缺乏技能；孤独者之所以孤独，因为它们令人讨厌；忙碌的人之所以忙碌是因为他们缺乏组织生活的能力 ……
  </p>
  <p class="graf graf--p">
   用这种替代的观点，稀缺变成了深层的个人问题的结果，很难改变。而相反，稀缺性心态是因果关系的结果，更容易接受补救措施。
  </p>
  <p class="graf graf--p">
   👉稀缺性本身就是环境条件的结果，而不是个人特质，这种条件通常可以加以管理从而改变它。但为什么人们却忘记了如何去改变？
  </p>
  <p class="graf graf--p">
   稀缺性削弱人的勇气，使决策过程变得复杂，因为通常只有短暂的机会窗口可以选择稀缺的东西。所以你的全部注意力都会被吸收在那个窗口上。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这对所有人来说都是如此，不论智商如何
   </strong>
   ，当看到眼前的供应开始减少时，人们就必定会感到特别有必要采取行动。经历过疫情抢购的人们应该很容易理解这点。
  </p>
  <p class="graf graf--p">
   每种情况和稀缺性都是不同的，经常并不是厕纸和大米这么明显，
   <strong class="markup--strong markup--p-strong">
    当权者甚至可以人为制造出稀缺 —— 塑造公众的价值观
   </strong>
   。比如中国的户籍制度，强行将某些城市定义为高档并限制迁居。
  </p>
  <p class="graf graf--p">
   不论如何，请随时记得，当你感知到稀缺的紧迫感时，试图去寻找它的成因，并想办法抵抗不公正的稀缺。而非跟随欲望。越多的人认同稀缺的 “合理性”，它就越容易加速深化，将所有人都拉入深渊。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     2、惧怕权威
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   许多人在一个被视为权威的人物面前会感到焦虑，不是他们对那个人感到焦虑，而是对恐吓他们的人所代表的地位和权力。
  </p>
  <p class="graf graf--p">
   于是，攻击者很擅长扮演着执法人员或公司高级官员之类的权威人物角色，以从受害者那里提取敏感信息。
  </p>

  <p class="graf graf--p">
   👉 恐惧和服从并不是攻击者 “植入” 的，而是来自人们根深蒂固的对权威的下意识反应。攻击者所做的只是 **启发** 了你内心的冲动。

   学习这些知识可以帮助您理解操纵者的手段，并随时控制自己的行为。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     3、提供帮助的渴望
    </strong>
   </span>
  </h3>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “帮助” 这个词可能会存在歧义，因为
   <strong class="markup--strong markup--p-strong">
    这里所指的帮助并无关善良和爱心，准确说是在追求一种自我炫耀的满足感
   </strong>
   ，“你看我知道这件事/我能做到***”。
  </p>
  <p class="graf graf--p">
   攻击者的做法就是刺激目标人的这种满足感的欲望。
  </p>
  <p class="graf graf--p">
   越是不怎么懂的人越希望彰显出自己懂一些东西，越是权力低微的人越渴望表现出自己的权力。攻击者很了解这点，这就是为什么他们以看门人、清洁人员为目标，以获得进入重要设施的权限，或者套出敏感的致命信息。
  </p>
  <p class="graf graf--p">
   这在互联网上也一样。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当你显得自己 “无所不知” 时，任何人都不会告诉你他们知道什么。相反，如果你让自己看起来像个白痴 …… 几乎所有人都会努力 “纠正” 你的错误、试图教导你什么才是真实的信息。
   </strong>
  </p>
  <p class="graf graf--p">
   👉 攻击者要做的就是给目标人一种 “你比我智高一筹” 的优越感，在这种感觉的熏陶下，目标人就会非常愿意吐露更多重要的敏感信息。
  </p>
  <p class="graf graf--p">
   这种手段屡试不爽。
   <strong class="markup--strong markup--p-strong">
    尤其是通过社交媒体这种激发表现欲的媒介 ——
   </strong>
   你只需要稍微装傻，就可以轻松获得很多人隐秘的知识和重要信息。
  </p>
  <p class="graf graf--p">
   👌总之：请随时留意您说出的每一句话，使用 *对抗性思考方式* —— 即：考虑如果恶意者听到/看到了你所说的，他们可以用此信息来做什么。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     4、习以为常
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   我们所有人都遇到过某些工作，这些工作仅要求我们执行一组指定的活动，仅此而已。
  </p>
  <p class="graf graf--p">
   这会使每天重复执行相同任务的人感到无聊，并且随着时间的流逝，这种无聊感会促使你在满足目标的同时以最小的努力来完成任务。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这就导致了一种很随意的态度，并且容易受到攻击
   </strong>
   。这些攻击者将此类人作为目标，知道他们可以轻松获得攻击者所需的信息。
  </p>
  <p class="graf graf--p">
   👉习以为常感会冲淡人的态势感知力，甚至其中一些人开始向 “身边人” 倾诉自己的无聊，以获得心理层面的呼应。但是他们忘了，对他们自己来说 “习以为常” 的东西，对外人来说经常是稀奇和新鲜的。听者会传播这些信息，从而被攻击者捕获。
  </p>
  <p class="graf graf--p">
   👌 如果您是组织中的负责人，请尽可能安排您的队友在不同时间段内做不同的工作，并同时制定严格合理的安全标准，以避免恶意者借助无聊情绪来渗透你的组织。
  </p>
  <p class="graf graf--p">
   如果您是工作者，请随时提醒自己保持警觉，主动转移一下自己的侧重，做一些其他的事；

  </p>
  <p class="graf graf--p">
   斯诺登揭露的所有NSA黑客行动中，其中一项特别以其精巧和隐蔽性而引人注目。自2005年以来，美国国家安全局（NSA）及其合作间谍机构英国的 GCHQ 广泛使用了各种这类旁观者攻击骇客技术，以入侵高价值的、难以到达的系统，并植入恶意软件。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    全部攻击手段分析《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" rel="noopener noreferrer" target="_blank">
     互联网霸主的时间线
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   检测的重点是识别在浏览器尝试访问网页时发送到受害者的浏览器客户端的数据包中的异常。具体方法
   <a class="markup--anchor markup--p-anchor" data-href="https://blog.fox-it.com/2015/04/20/deep-dive-into-quantum-insert/" href="https://blog.fox-it.com/2015/04/20/deep-dive-into-quantum-insert/" rel="noopener noreferrer" target="_blank">
    见这里
   </a>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     5、钓鱼
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   网络钓鱼一般是指创建和使用虚假的网站和电子邮件地址的行为，这些网站和电子邮件的外观看起来非常像是来自知名的合法企业、金融机构和政府机构，以欺骗接收者泄露其真实个人信息。
  </p>
  <p class="graf graf--p">
   将您的电子邮件客户端设置为接收纯文本电子邮件，而不是HTML格式。
  </p>
  <p class="graf graf--p">
   仔细检查电子邮件中的任何网址。虚假地址可能只是一个字母不同或缺失。
  </p>
  <p class="graf graf--p">
   更多方法参见下面：
  </p>

  <h3 class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     6、网站、在线论坛和社交媒体上的开源信息
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   有关组织结构（正式和非正式）、电子邮件地址、电话号码的大量信息都可以在网站、论坛和社交媒体上公开获得。
   <strong class="markup--strong markup--p-strong">
    攻击者可以使用此信息来完善其方法，并瞄准最有可能成功的目标人、制定最有可能成功的策划
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    开源情报
   </a>
   的诸多演示已经足够证明这点：关于社交媒体有多可怕。根本性的也是最简单的解决方案就是，不要使用任何社交媒体和在线论坛。
  </p>
  <p class="graf graf--p">
   但对于大多数组织和个人来说，这不现实。于是最好的做法是
   <strong class="markup--strong markup--p-strong">
    分身，尤其是在不同的社交媒体上采取不同的身份，避免将个人信息集中在一处
   </strong>

  <h3 class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     7、逆向社交工程
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   逆向社交工程学攻击是指这样一种攻击，其中攻击者说服目标他或她有问题、或将来可能遇到特定问题，并且攻击者已准备好帮助解决该问题。
  </p>
  <p class="graf graf--p">
   逆向社交工程涉及三个部分：
  </p>
  <p class="graf graf--p">
   ▶️ 破坏活动：攻击者获得对系统的简单访问权后，破坏了系统或使系统看上去受到破坏。当用户看到系统处于损坏状态时，他就会开始寻求帮助以解决问题。
  </p>
  <p class="graf graf--p">
   ▶️营销：为了确保用户准确地寻求攻击者的帮助，攻击者会给自己打广告自称为唯一可以解决问题的人。
  </p>
  <p class="graf graf--p">
   ▶️支持：在此步骤中，攻击者获得了目标的信任并获得了对敏感信息的访问权限。
  </p>
  <p class="graf graf--p">
   当您的系统出现故障时，恰逢某人在适当的时间出现以帮助您解决问题，那也许是好运，但也请考虑其故障的真正成因，也许它是某种 “虚假困境”。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     最后
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   基本上就是这样。这些威胁有可能并非单独出现，而是结合在一起、尤其是结合在一套逻辑完好的攻击策划中。
  </p>
  <p class="graf graf--p">
   但它们的出现都具有一定程度的迹象，您需要做的就是保持批判性思考和警觉的意识。尤其是当攻击针对群体操作时，周围人的想法和行为很可能对您产生影响。希望每一位阅读过本文的朋友都能成为组织中金丝雀。⚪️
  </p>
