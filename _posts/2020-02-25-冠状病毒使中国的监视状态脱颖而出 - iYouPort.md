      </strong>
     </em>

  <p class="graf graf--p">
   当一名杭州男子出差后返回家中时，当地警方与他取得了联系。警方跟踪了他的车牌，尽管距离疫情爆发中心很远，但是那里已经发生了一系列冠状病毒病例。警方要求这名男子在室内隔离两个星期。
  </p>
  <p class="graf graf--p">
   大约12天后，他感到很无聊，于是提前出门了。而这一次，不仅警察来找他，他的老板也来找他了。
  </p>
  <p class="graf graf--p">
   他被带有面部识别技术的摄像头发现在杭州西湖附近出现，当局已找到他的公司并作出了警告。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12973 jetpack-lazy-image" data-lazy-sizes="(max-width: 620px) 100vw, 620px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-14.png?resize=620%2C1470&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-14.png?w=620&amp;ssl=1 620w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-14.png?resize=127%2C300&amp;ssl=1 127w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-14.png?resize=432%2C1024&amp;ssl=1 432w" data-recalc-dims="1" height="1470" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-14.png?resize=620%2C1470&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="620"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12973" data-recalc-dims="1" height="1470" sizes="(max-width: 620px) 100vw, 620px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-14.png?resize=620%2C1470&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-14.png?w=620&amp;ssl=1 620w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-14.png?resize=127%2C300&amp;ssl=1 127w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-14.png?resize=432%2C1024&amp;ssl=1 432w" width="620"/>
   </noscript>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <strong class="markup--strong markup--p-strong">
    我对大规模监视网络的能力和效率感到震惊。他们基本上可以随时随地使用AI技术和大数据来追踪我们的一举一动
   </strong>
   ” ，这位男子说，他不愿透露姓名，因为担心会受到打击。
  </p>
  <p class="graf graf--p">
   中国人早就意识到，自己一直在被世界上最先进的电子监视系统所跟踪。冠状病毒的紧急情况使其中的某些技术脱颖而出，为当局提供了全面的高科技社会控制方法的 “理由”。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12974 jetpack-lazy-image" data-lazy-sizes="(max-width: 777px) 100vw, 777px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/2-3-scaled.jpeg?resize=777%2C2048&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/2-3-scaled.jpeg?w=777&amp;ssl=1 777w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/2-3-scaled.jpeg?resize=114%2C300&amp;ssl=1 114w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/2-3-scaled.jpeg?resize=388%2C1024&amp;ssl=1 388w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/2-3-scaled.jpeg?resize=768%2C2025&amp;ssl=1 768w" data-recalc-dims="1" height="2048" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/2-3-scaled.jpeg?resize=777%2C2048&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="777"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12974" data-recalc-dims="1" height="2048" sizes="(max-width: 777px) 100vw, 777px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/2-3-scaled.jpeg?resize=777%2C2048&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/2-3-scaled.jpeg?w=777&amp;ssl=1 777w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/2-3-scaled.jpeg?resize=114%2C300&amp;ssl=1 114w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/2-3-scaled.jpeg?resize=388%2C1024&amp;ssl=1 388w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/2-3-scaled.jpeg?resize=768%2C2025&amp;ssl=1 768w" width="777"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   人工智能和监视摄像头公司吹嘘他们的系统可以扫描街道，甚至找到发低烧的人，👉即使戴着口罩也能认出他们的脸，并向当局报告。
  </p>
  <p class="graf graf--p">
   如果冠状病毒患者登上了火车，铁路的 “实名制” 系统就可以提供附近人员的名单。
  </p>
  <p class="graf graf--p">
   手机应用程序可以告诉用户他们是否乘坐过载有已知冠状病毒携带者的飞机或火车，而地图则可以向用户显示被感染患者居住的建筑物的位置。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    尽管社交媒体上有一些匿名的抱怨，但是目前，中国公民似乎正在接受这种额外的入侵，甚至支持这种入侵，以此作为应对突发卫生事件的一种手段。
   </strong>
  </p>

   <li class="graf graf--li">

  <p class="graf graf--p">
   DLA Piper 律师事务所香港合伙人 Carolyn Bigg 表示：“在这种情况下，即使没有明确告知，个人也可能认为这是合理的。”
  </p>
  <p class="graf graf--p">
   电信公司长期以来一直在秘密地跟踪其用户的活动。中国移动最近甚至对此服务进行了推广，向北京居民发送短信，告诉他们电信公司可以让他们检查自己过去30天的所有去向。
  </p>
  <p class="graf graf--p">
   它没有解释为什么用户可能需要这样做，但是如果当局或他们的雇主对他们的出行提出质疑，这可能会很有用。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “在大数据和互联网时代，每个人的一举一动都可以被清晰地看到。因此，我们现在与SARS时代有所不同，” 流行病学家李岚娟上周在接受中国国家广播电视台央视采访时说，将本次疫情与2003年造成800人死亡的病毒进行了比较。
  </p>
  <p class="graf graf--p">
   她说：“有了这样的新技术，我们应该充分利用它们来寻找感染源并控制感染源。”
  </p>
  <p class="graf graf--p">
   工信部最近向中国的AI公司和研究机构发出了通知，呼吁它们帮助抗击疫情。这些公司对此做出了一系列回应，
   <strong class="markup--strong markup--p-strong">
    自夸其[监视]技术的功能
   </strong>
   。
  </p>
  <p class="graf graf--p">
   面部识别技术公司 Megvii 周二表示，它已经开发出一种新方法来发现和识别发烧患者。⚠️其新的 “人工智能温度测量系统” 可通过热成像仪检测体温，并利用身体和面部数据识别个人身份，目前已在北京地区进行了测试。
  </p>
  <p class="graf graf--p">
   另一家领先的 AI 公司 SenseTime 表示，它已经建立了一个类似的系统用于建筑物入口，⚠️可以识别戴着口罩的人，从而克服了早期技术的缺点。
  </p>

    </a>

   </li>

  <p class="graf graf--p">
   监控摄像机公司浙江大华表示，它可以使用红外摄像机检测发烧，准确度在0.3摄氏度以内。
  </p>
  <p class="graf graf--p">
   中国铁道科学研究院的朱建生在接受国家新闻社的采访时，解释了技术如何帮助当局找到在火车上可能暴露于确诊或疑似的冠状病毒病例的人。
  </p>
  <p class="graf graf--p">
   他说：“我们将检索有关乘客的相关信息，包括列车号、车厢号以及与该人接近的乘客的信息，例如 在该人前后三排座位上的人。”
  </p>
  <p class="graf graf--p">
   他说：“我们将提取信息，然后将其提供给相关的防疫部门。” ⚪️
  </p>
