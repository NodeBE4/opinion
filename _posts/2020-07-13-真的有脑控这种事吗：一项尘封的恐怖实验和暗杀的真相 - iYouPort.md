  <p class="graf graf--p">
   1953年11月一个寒冷的凌晨，曼哈顿第七大道的某扇窗子忽然碎了，几秒之后，一个人掉落在人行道上。Statler 酒店的门房 Jimmy 一时惊呆，之后他转身跑进酒店大堂，大喊道：“有人跳楼了！有人跳楼了！”
  </p>
  <p class="graf graf--p">
   夜班经理在黑暗之中环视这座巨大的酒店。不多时，他发现一个房间的窗户洞开，窗帘在风中飞舞。这间房间就是1018A，入住登记卡上有两个名字 — — Frank Olson 和 Robert Lashbrook 。
  </p>
  <p class="graf graf--p">
   警察持枪闯入了1018A房间，但房内空无一人、窗户大开。
  </p>
  <p class="graf graf--p">
   他们将浴室门推开，发现 Lashbrook 坐在马桶上，双手抱头。他说，他之前一直在睡觉，然后 “被某个声响吵醒了”。
  </p>
  <p class="graf graf--p">
   一位警员问道：“从窗口跳下去的那个男人叫什么？”
  </p>
  <p class="graf graf--p">
   他回答：“Olson,。Frank Olson。”
  </p>
  <p class="graf graf--p">
   酒店经理之后回忆道：“我从业多年，从未遇过这种事 — —
   <strong class="markup--strong markup--p-strong">
    有人半夜醒来，只穿着内衣在黑暗的房间里穿行，避开两张床，冲向百叶和窗帘紧闭的窗户。
   </strong>
   ”
  </p>
  <figure aria-describedby="caption-attachment-12579" class="wp-caption aligncenter" id="attachment_12579" style="width: 600px">
   <img alt="" class="size-full wp-image-12579 jetpack-lazy-image" data-lazy-sizes="(max-width: 600px) 100vw, 600px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-11.jpg?resize=600%2C776&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-11.jpg?w=600&amp;ssl=1 600w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-11.jpg?resize=232%2C300&amp;ssl=1 232w" data-recalc-dims="1" height="776" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-11.jpg?resize=600%2C776&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="600"/>

  <p class="graf graf--p">
   经理离开房间，回到大堂，出于直觉询问接线员，1018A房间最近是否有呼出电话。
  </p>
  <p class="graf graf--p">
   她给出了肯定的回答，并透露还曾偷听到谈话的内容  — — 这种做法在那个时代十分常见，因为酒店的电话都是经由总机转接过去的。
  </p>
  <p class="graf graf--p">
   她说，房间里有人曾拨打了一个去长岛的电话，该号码在电话录上显示归属于著名医生 Harold Abramson。他还有并不那么为人所知的身份 — — LSD 专家、及中情局医学合作者之一（LSD 就是 D-麦角酸二乙胺，一种强烈的半人工致幻剂，能造成使用者4～12小时的感官、记忆和自我意识的变化，可作化学武器使用：
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Lysergic_acid_diethylamide" href="https://en.wikipedia.org/wiki/Lysergic_acid_diethylamide" rel="noopener noreferrer" target="_blank">
    来自维基百科
   </a>
   ）。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12580 jetpack-lazy-image" data-lazy-sizes="(max-width: 624px) 100vw, 624px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-23.png?resize=624%2C986&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-23.png?w=624&amp;ssl=1 624w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-23.png?resize=190%2C300&amp;ssl=1 190w" data-recalc-dims="1" height="986" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-23.png?resize=624%2C986&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="624"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   打电话的人说：“他走了。” Abramson 回复：“好吧，那太糟糕了。”
  </p>
  <p class="graf graf--p">
   对于亲临现场的第一批警员来说，他们见过了太多的人间悲剧，这不过是其中的一起而已  — — 沮丧不安的心智失常的人结束了自己的生命。
  </p>
  <p class="graf graf--p">
   但是，他们不知道的是，
   <strong class="markup--strong markup--p-strong">
    这个死者和另一个生者都曾是帮助指导美国政府最高机密项目的科学家。
   </strong>
  </p>
  <p class="graf graf--p">
   第二天一大早，与 Olson 比较亲近的一位同事前往马里兰州，将这一噩耗告知死者家属。
  </p>
  <p class="graf graf--p">
   他跟 Alice Olson 及三个孩子说，Frank 从酒店窗户 “坠落或跳楼” 身亡。他们自然震惊无比，但除了接受这一事实外也别无选择。
  </p>
  <p class="graf graf--p">
   家属将不被允许看望遗体，妻子 Alice 对此也并未提出异议。尸体被放入一个密闭的棺材里埋葬。至此，这桩案子似乎已经划上了句号。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    然而，数十年之后，在令人震惊的披露之下，Olson
   </strong>
   <strong class="markup--strong markup--p-strong">
    之死似乎又有了截然不同的说法
   </strong>
   。
  </p>
  <p class="graf graf--p">
   首先，CIA承认，Olson 死前不久，其同事曾邀他参加聚会，并在他毫不知情的情况下让其服下了 LSD。
  </p>
  <p class="graf graf--p">
   后又证实，Olson 曾谈及离开CIA，并告诉过妻子他犯下了 “一个可怕的错误”。
  </p>
  <p class="graf graf--p">
   对立的证言也逐渐浮现 — — Olson 对工作深感不安并意图辞职，导致其同事认为他是个风险因素。所有这一切，将他引至1018A房间。
  </p>

  <p class="graf graf--p">
   二战时期，Frank Olson 作为第一批科学家，被派往位于马里兰州弗雷德里克市德特里克堡的美国秘密生化实验室工作。
  </p>
  <p class="graf graf--p">
   在那儿，Olson 开始与一些同事并肩工作，正是这些人伴随了他的整个秘密职场生涯。
  </p>
  <p class="graf graf--p">
   其中一位就是 Harold Abramson，
   <strong class="markup--strong markup--p-strong">
    其他人还包括被带到美国参与秘密任务的前纳粹科学家们。
   </strong>
  </p>
  <p class="graf graf--p">
   有一段时间，他们曾研究气溶剂对的技术 — — 就是将细菌或有毒物质以雾状投放到敌方，并让自己对此种攻击做出防护。
  </p>
  <p class="graf graf--p">
   之后，Olson 见到了曾经在欧洲试验 “吐真剂” 的美国情报官员。
  </p>

  <p class="graf graf--p">
   Olson 于1944年退伍，但仍以平民身份留在德特里克堡，并继续进行空气生物学研究。
  </p>
  <p class="graf graf--p">
   他曾数次造访与几乎世隔绝的犹他州杜格威试验场，
   <strong class="markup--strong markup--p-strong">
    这个地点曾用来测试 “活性生物病原体、军需品、和气雾剂产品”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   他与别人合著了一份长达220页的学术研究论文，名为 “实验性空气传播感染”，描述了
   <strong class="markup--strong markup--p-strong">
    “强感染性病原体空气团” 实验
   </strong>
   。
  </p>

  <p class="graf graf--p">
   1949年他前往加勒比海地区参与 “治理行动”，该行动测试动物对有毒气团的脆弱度。
  </p>
  <p class="graf graf--p">
   第二年，他参与了 “海上喷雾行动”，在旧金山附近释放炭疽样的灰尘。
  </p>
  <p class="graf graf--p">
   他还定期前往泰瑞港口 — — 这是棕榈岛上的一个
   <strong class="markup--strong markup--p-strong">
    秘密军事基地
   </strong>
   ，位于长岛东端，用于测试有毒物质。此处位置偏远，可避免有毒物质进入美国大陆。
  </p>
  <p class="graf graf--p">
   就在那个时候，高级军队和 CIA 官员开始深深忌惮苏联在微生物武器军备上的进展。
  </p>
  <p class="graf graf--p">
   这种担忧导致了 “特别行动部门” 的成立。关于其工作内容的传闻在各机构和实验室流传。
  </p>
  <p class="graf graf--p">
   Olson 是在晚上和同事 John Schwab 玩时了解到该机构的。但他所不知道的是，Schwab 就曾经是该机构的首任长官。他邀请 Olson 加入，并立即得到了肯定答复。
  </p>
  <p class="graf graf--p">
   不到一年，Olson 就作为 Schwab 的继任者成为了该机构的负责人。
  </p>
  <p class="graf graf--p">
   他的工作描述模糊不详，却又具诱惑：收集 “部门需要的” 数据，“尤其是医药生物方面的”，协调 “与其他相似或相关机构” 的工作 — — 这是指 CIA。
  </p>
  <p class="graf graf--p">
   研究显示，Olson 的专长是 “生物细菌在空气中的传播”。
  </p>
  <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote">
   <p>
    “Olson 博士发明了一系列便携手持的致命气溶胶，被仿造成剃须膏和防虫剂。它们含有葡萄球菌肠毒素，这是一种有严重危害的食物毒素；更致命的是，委内瑞拉马脑脊髓炎病毒；最致命的是炭疽 …… 他正在研制的其他生物武器还包括瞬间释放致命毒气的打火机、接触皮肤就能致命的口红、可以让哮喘患者诱发肺炎的小型喷雾剂 ……”
   </p>
  </blockquote>

  <p class="graf graf--p">
   1953年初，Olson 因病辞去了特别行动部门负责人的职务（那时他已经加入了CIA），但他仍待在该部门。
  </p>
  <p class="graf graf--p">
   依据官方说法，特别行动部门隶属于军队，但实际上是隐藏在军事基地里的一个 CIA 研究点。
  </p>
  <p class="graf graf--p">
   在这里，Olson 认识了 Sidney Gottlieb 和他的副手 Robert Lashbrook。这两位科学家将要主导一个代号为 MK-Ultra 的CIA绝密计划。
  </p>
  <p class="graf graf--p">
   Gottlieb 是CIA的毒药制作负责人。20多年来，他见证了成百上千人在医学实验和 “特殊审问” 项目中备受折磨，许多人因此精神失常。
  </p>
  <p class="graf graf--p">
   这个过程中，
   <strong class="markup--strong markup--p-strong">
    CIA有种执念 — — 一定存在能控制心智的方法，一旦找到就能控制全世界。
   </strong>
  </p>
  <p class="graf graf--p">
   MK-Ultra 就是这样一个尝试控制心智的绝密项目。
  </p>
  <p class="graf graf--p">
   其常规做法是将 LSD 用于 “可消耗” 的实验对象身上。
  </p>
  <p class="graf graf--p">
   Gottlieb 想要弄清楚人体能承受的LSD最大剂量。
   <strong class="markup--strong markup--p-strong">
    他想知道是否存在一个临界点，在大剂量之下，人类心智和意识会被冲洗干净，留下一片空白，可以将新的意识冲动甚至全新的人格植入其中？
   </strong>
  </p>
  <p class="graf graf--p">
   在德特里克堡的实验室里，Olson 主持的实验包括给动物放毒气或下毒。这让他心神不安。他的儿子 Eric 后来回忆说：“他每天早上去工作，然后看到成排的死猴子。这让他很痛苦。他不是适合干这个的人。”
  </p>
  <p class="graf graf--p">
   Olson 也见证过人类在实验中被折磨。尽管他自己不是施害者，但
   <strong class="markup--strong markup--p-strong">
    他在多个国家里都观察并监控了此种折磨过程。
   </strong>
  </p>
  <p class="graf graf--p">
   一项研究称，“在德国的CIA安全屋里，Olson 频繁见证了那些惨无人道的可怕审讯。因疑似间谍或泄露机密而被拘押的
   <strong class="markup--strong markup--p-strong">
    人类 ‘实验品’
   </strong>
   ，在各种实验中被审讯致死。手段包括药物、催眠、酷刑 ……
   <strong class="markup--strong markup--p-strong">
    实验目的是掌握洗脑技术和抹除记忆
   </strong>
   。”
  </p>

  <p class="graf graf--p">
   1953年感恩节前夕，Olson 接到了邀请，于11月18日星期三前往马里兰州西部的深溪湖畔小屋参加聚会。
  </p>
  <p class="graf graf--p">
   这次聚会是 Gottlieb 每几个月就组织一次的聚会。准确说这是两大组织的碰头：执行 MK-Ultra 的、技术服务部门的4位CIA科学家，和化学部队特别行动部门的5名军方科学家。
  </p>
  <p class="graf graf--p">
   事实上这群人因工作联系紧密已联结成一个小团体。
  </p>
  <p class="graf graf--p">
   在探寻重大机密面前，他们齐心协力。这种聚会对他们来说意义重大，因为可以在一个放松的环境中讨论项目、交流想法。
  </p>
  <p class="graf graf--p">
   这次聚会的第一天相安无事。星期四晚上，所有人一起用了晚餐，饭后喝了酒。
  </p>
  <p class="graf graf--p">
   Lashbrook 作为 Gottlieb 的助手，拿出一瓶法国君度，为所有人倒酒。包括 Olson 在内的几个人都喝得十分尽兴。
  </p>
  <p class="graf graf--p">
   20分钟后，Gottlieb 问 “是否有人感到异样”。
   <strong class="markup--strong markup--p-strong">
    有几个人说确实如此。随后他告诉：他们酒里被偷偷掺入了LSD。
   </strong>
  </p>
  <p class="graf graf--p">
   即便在扭曲的状态下，这些人也都很明白自己喝进去的是什么。Olson 的变化尤其强烈。
  </p>
  <p class="graf graf--p">
   据 Olson 的儿子说，他变得 “暴怒不已，很难将事实和虚幻区分开来。”
  </p>
  <p class="graf graf--p">
   不过很快，他们所有人就被带到了一个虚妄世界。
  </p>
  <p class="graf graf--p">
   Gottlieb 后来汇报说，他们 “吵吵嚷嚷、大声笑闹，无法继续正常的聚会，也难以进行理智的对话。”
  </p>
  <p class="graf graf--p">
   第二天早上，他们的状况才稍好一点。聚会不欢而散。Olson 返回弗雷德里克。那时候他已经变得不一样了。
  </p>
  <p class="graf graf--p">
   第二天清晨，也就是11月23日，Olson 很早就来到了德特里克堡。他的上司 Vincent Ruwet 很快也到了，同样状态不佳。
  </p>
  <p class="graf graf--p">
   在不知情的情况下 “被服用” LSD已经超过4天了。Ruwet 事后称这是 “亲身体验过或想象中最恐怖的经历”。
  </p>
  <p class="graf graf--p">
   Olson 开始大量倾吐怀疑和恐惧。Ruwet 后来回忆说：“他似乎焦虑不安，问我是该解雇他，还是他自己辞职。”
  </p>
  <p class="graf graf--p">
   他努力安抚 Olson，向他保证他的工作很出色，且公认如此。慢慢地，Olson 被说服了，认为辞职过于极端。
  </p>
  <p class="graf graf--p">
   这时，MK-Ultra 项目已进行了7个月。
   <strong class="markup--strong markup--p-strong">
    这是政府的最高机密之一
   </strong>
   ，就像 Olson 加入特别行动部门时被告知的那样 —— 安保措施 “紧之又紧”。
  </p>
  <p class="graf graf--p">
   差不多只有20几个人知道该部门的真正目的。参与深溪湖聚会的9个人中，有些被偷偷下了LSD。现在其中一人似乎已经失控。
  </p>
  <p class="graf graf--p">
   对于相信 MK-Ultra 项目的成败可能决定美国和全人类命运的人来说，这并非小事。
  </p>
  <p class="graf graf--p">
   Olson 在德特里克堡待了十年，对特殊行动部门的机密知道得八九不离十。他多次前往德国，并从海德伯格和柏林带回了一些照片，而美军在这些地方仍继续运行
   <strong class="markup--strong markup--p-strong">
    秘密审讯点
   </strong>
   。
  </p>
  <p class="graf graf--p">
   1951年8月16日，在法国的圣埃斯普里特桥村庄，全村200多名居民变得歇斯底里、神志失常，并有数人死亡。
  </p>
  <p class="graf graf--p">
   Olson 和其他几位特别行动部门的科学家当场见证了这一切。
  </p>
  <p class="graf graf--p">
   原因后被定为
   <strong class="markup--strong markup--p-strong">
    麦角菌中毒
   </strong>
   ，而麦角菌是LSD的成分来源。
  </p>
  <p class="graf graf--p">
   也许最具威胁性的是，如果美国军方在朝鲜战争中确实使用了生化武器（有间接但无确凿证据），那 Olson 肯定知情。他可能会透露自己所见或所做的任何事的前景令那些秘密间谍恐惧。
  </p>
  <p class="graf graf--p">
   Olson 的朋友和同事 Norman Cournoyer 后来提及：“他极为坦诚，敢于吐露自己的想法。他毫不在意，任何时候都敢于直言 …… 我敢肯定，这就是让他们畏惧的地方。”
  </p>
  <p class="graf graf--p">
   Olson 的疑虑逐渐加深。
  </p>
  <p class="graf graf--p">
   1953年春天，他造访了位于英国威尔特郡波登当的微生物研究中心，
   <strong class="markup--strong markup--p-strong">
    该中心为最高机密，政府的科学家在此研究沙林毒气和其他神经毒气的影响
   </strong>
   。
  </p>
  <p class="graf graf--p">
   5月6日，一位20岁的士兵主动想要作为沙林毒气的实验对象。一开始他口吐白沫，之后痉挛不已，于1小时后死亡。
  </p>
  <p class="graf graf--p">
   之后，Olson 和帮助主导该项研究的精神病学家 William Sargant 谈起他的不适之处。
  </p>
  <p class="graf graf--p">
   一月后，Olson 返回德国。根据后来的旅行记录推测，在该段行程中，他还 “到访了德国斯图加特附近的一个CIA安全屋，在那里
   <strong class="markup--strong markup--p-strong">
    他目睹了人们痛苦地死于他制造的武器”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在斯堪的纳维亚地区和巴黎停留后，他返回英国并再次拜访 Sargant。
  </p>
  <p class="graf graf--p">
   在这次会面后，Sargant 立即撰写了报告，提到 Olson “对德国CIA安全屋的所见所闻深感不安”，并 “显露出了公开这些景象的征兆”。
  </p>
  <p class="graf graf--p">
   他将这份报告呈给他的上司，并知晓这份报告会由他们上报至CIA。Sargant 后来说：“有些共同利益是需要保护的。”
  </p>
  <p class="graf graf--p">
   服用 LSD 的5天后，Olson 仍然有些不在状态。他在特别行动部门的上司 Ruwet 打电话给 Gottlieb 报告此事。
  </p>
  <p class="graf graf--p">
   Gottlieb 请他安排和 Olson 的谈话。
  </p>
  <p class="graf graf--p">
   Gottlieb 后来证实，谈话中 Olson 似乎 “在某些思维方面深感困惑”。
  </p>
  <p class="graf graf--p">
   他立马做出了决定 — — Olson 必须被送往纽约，由和 MK-Ultra 项目联系最紧密的医生 Harold Abramson 博士接手。
  </p>
  <p class="graf graf--p">
   Alice Olson 被告知，选择 Abramson 的原因是她丈夫 “不得不去看医生，而这位医生同样经过了安全调查，因而可以畅所欲言”。
  </p>
  <p class="graf graf--p">
   这在一定程度上是真实的。但是，Abramson 并不是精神病学家，而是 MK-Ultra 项目的同谋。
  </p>
  <p class="graf graf--p">
   Gottlieb 知道 Abramson 首先对 MK-Ultra 很忠诚，或按他的话来说，是对美国安全的忠诚。这让他成为了探究 Olson 内心想法的理想人选。
  </p>
  <p class="graf graf--p">
   Olson 告诉 Abramson，自从深溪湖的那次聚会之后，他就无法再好好工作了，不能集中精力，经常忘了如何拼写，甚至夜不能寐。
  </p>
  <p class="graf graf--p">
   Abramson 试图安抚 Olson，这让 Olson 后来似乎放松了下来。
  </p>

  <p class="graf graf--p">
   距离 Olson 被下毒 LSD 已经过去一周了。他计划回家过感恩节。
  </p>
  <p class="graf graf--p">
   见过 Abramson 后的第二天，他在 Lashbrook 和 Ruwet 的陪同下登上了去华盛顿的飞机。
  </p>
  <p class="graf graf--p">
   一位 MK-Ultra 项目的同事前去接机。Ruwet 和 Olson 坐上了开往弗雷德里克的汽车。
  </p>
  <p class="graf graf--p">
   他们刚启程，Olson 就改变主要要求停车，并对 Ruwet 说他感觉 “没脸见妻子和家人”，因为他自己 “一团糟”。
  </p>
  <p class="graf graf--p">
   Ruwet 问：“你想让我怎么做？”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “让我走吧。我自己走。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “不行。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “那么，就把我交给警察吧。反正他们也正在找我。”
  </p>
  <p class="graf graf--p">
   Ruwet 建议 Olson 返回纽约，在 Abramson 那里继续另一阶段的治疗。Olson 同意了，于是他们乘坐出租车前往长岛，Abramson 周末都住在那里。
  </p>
  <p class="graf graf--p">
   Abramson 和 Olson 待了一小时，又和 Lashbrook 待了20分钟。
  </p>
  <p class="graf graf--p">
   第二天早上，Abramson、Lashbrook 和 Olson 驱车返回曼哈顿。
  </p>
  <p class="graf graf--p">
   Abramson 在58号大街的办公室里给 Olson 做治疗时说服他主动前往马里兰州疗养所修养。Olson 和 Lashbrook 离开后，去往斯塔特勒酒店登记入住，房间号是1018A。
  </p>
  <p class="graf graf--p">
   在酒店用晚餐时，Olson 告诉 Lashbrook 他非常期待去疗养院。他还仔细思考了将要读的书。
  </p>
  <p class="graf graf--p">
   Lashbrook 后来说他 “几乎恢复到了实验前我认识的那个人”。
  </p>
  <p class="graf graf--p">
   两人返回房间。Olson 在水池里洗了袜子，看了会电视就睡觉了。
  </p>
  <p class="graf graf--p">
   凌晨2点25分，他跳楼了。
  </p>

  <p class="graf graf--p">
   每个特勤局都需要那种专门清理烂摊子的官员。在20世纪50年代的 CIA，这些官员为 Sheffield Edwards 工作。在 Frank Olson 死后的数小时到数天里，他主持的洗地行动堪称高效。
  </p>
  <p class="graf graf--p">
   Sheffield Edwards 在CIA以冷静自信而闻名，他宣布了清理行动的进行方式。
  </p>
  <p class="graf graf--p">
   首先，
   <strong class="markup--strong markup--p-strong">
    说服纽约警方不进行调查，并配合他们误导媒体
   </strong>
   。
  </p>
  <p class="graf graf--p">
   其次，为 Lashbrook 编造一个虚假的职业生涯 — — 一个 “传奇”。作为唯一的证人，Lashbrook 将受到调查人员的讯问，无论如何都不能暴露他为CIA工作的实情，更不用说参与 MK-Ultra 项目了。
  </p>
  <p class="graf graf--p">
   第三，必须通知 Olson 的家人，做好安抚工作，让其保持合作。
  </p>
  <p class="graf graf--p">
   当 Olson 的妻子在马里兰的家中得知丈夫的死讯时，Lashbrook 正在纽约的1018A房间欢迎CIA派来的人。
  </p>
  <p class="graf graf--p">
   来的人只有一位。在内部报告中，他被称为 “特工 James McC”。后来，他的身份被确认，是
   <strong class="markup--strong markup--p-strong">
    水门事件的窃贼之一 James McCord
   </strong>
   。
  </p>
  <p class="graf graf--p">
   McCord 以前是FBI的特工，专门从事反情报活动。
   <strong class="markup--strong markup--p-strong">
    混淆警方调查方向是他的专长之一
   </strong>
   。
  </p>

  <p class="graf graf--p">
   11月28日凌晨，McCord 接到 Edwards 的电话后就立即采取了行动。
  </p>
  <p class="graf graf--p">
   他乘坐首班飞机去了纽约，大约早上8点到达斯塔特勒酒店。
  </p>
  <p class="graf graf--p">
   他花了一个小时的时间询问 Lashbrook，然后在上午9点半左右建议他按照警方要求去贝尔维尤医院的太平间辨认 Olson 的尸体。
  </p>
  <p class="graf graf--p">
   Lashbrook 不在的时候，McCord 仔细搜查了1018A号房间和附近房间。
  </p>
  <p class="graf graf--p">
   中午过后不久，Lashbrook 回到了酒店，McCord 正在那里等着。
  </p>
  <p class="graf graf--p">
   在接下来的几个小时里，Lashbrook 打了一大堆的电话。其中一个是打给 Gottlieb 的。
  </p>
  <p class="graf graf--p">
   挂断电话后，他告诉 McCord，Gottlieb 让他去 Abramson 的办公室拿一份报告，并由他亲自带回华盛顿。
  </p>
  <p class="graf graf--p">
   Lashbrook 乘午夜的火车将此报告送到华盛顿。纽约的CIA安全官员负责处理其他细节。
  </p>
  <p class="graf graf--p">
   负责调查的警探得出结论，Olson 死于 “跳楼或坠落导致的” 多处骨折。这成了官方说辞。
  </p>
  <p class="graf graf--p">
   尽管掩盖事实的行动十分成功，但 Olson 的死对CIA来说几乎是一场灾难。它几乎威胁到了 MK-Ultra 项目的存在。
  </p>
  <p class="graf graf--p">
   对于 Gottlieb 和他在CIA的上司来说，这本应该是一个反思的时刻。鉴于这起死亡事件，他们本应考虑终止精神活性药物的进一步实验，至少不再用于不知情的受试者。
   <strong class="markup--strong markup--p-strong">
    然而，他们继续进行，仿佛 Olson
   </strong>
   <strong class="markup--strong markup--p-strong">
    之死从未发生过
   </strong>
   。
  </p>

  <p class="graf graf--p">
   1975年6月12日《华盛顿邮报》报道了一名军方科学家被CIA下毒LSD，反应过激，以至于从纽约一家酒店的窗户跳楼。
  </p>
  <p class="graf graf--p">
   这个与耸人听闻的毒药、死亡和CIA相关的故事，激起了人们的好奇。
  </p>
  <p class="graf graf--p">
   在接下来的几天里，记者们不断向CIA询问更多细节。Olson 一家在自家后院召开了新闻发布会。Olson 的妻子读了一份声明，说他们决定 “可能会在两周内向CIA提起诉讼，并要求几百万美元的赔偿”。
  </p>
  <p class="graf graf--p">
   她坚称丈夫在生命的最后几天里 “并没有丧失理智或生病”，但 “非常忧郁”，而且 “说过他将辞职”。
  </p>
  <p class="graf graf--p">
   她说：“1953年以来，我们一直努力把 Frank Olson 的死理解为莫名其妙的 ‘自杀’，而死亡的真相被隐瞒了22年。”
  </p>
  <p class="graf graf--p">
   除了宣布起诉CIA外，Olson 的家人还要求纽约警察局展开新的调查。
  </p>
  <p class="graf graf--p">
   曼哈顿地区检察官 Robert Morgenthau 立即回复，承诺他的办公室将开始 “调查此案的某些方面”。
  </p>
  <p class="graf graf--p">
   Olson 家人宣布起诉CIA后

    白宫就拉响了警报
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果允许提起诉讼，相当于给这家人以及纽约的凶杀案侦探提供了工具，一种可以用来强制披露深层机密的工具。
  </p>
  <p class="graf graf--p">
   福特总统的幕僚长 Donald Rumsfeld 和他的副手 Dick Cheney意识到了这种危险。Cheney 在一份备忘录中警告 Rumsfeld，诉讼 “可能会迫使CIA披露高度机密的国家安全信息”。
  </p>
  <p class="graf graf--p">
   为了避免这场灾难，他建议福特公开 “表示遗憾”，并 “表示愿意亲见 Olson 夫人和孩子”。
  </p>

   1975年7月21日，他们在总统办公室会面。这是一个绝无仅有的历史时刻：美国总统唯一一次召见惨死的CIA官员的家人，并代表美国政府道歉。
  </p>
  <p class="graf graf--p">
   后来，他们在弗吉尼亚州兰利的CIA总部会见了CIA局长 William Colby。他为一件 “本不该发生” 的 “可怕事件” 道歉。
  </p>
  <p class="graf graf--p">
   Colby 说：“在那个年代，我们有些人失控了。他们太过分了。当时的监督管理存在问题。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    白宫律师愿意向 Olson 家人支付75万美元封口费，以换取他们放弃法律诉讼。
   </strong>
  </p>
  <p class="graf graf--p">
   一番犹豫过后，他们接受了。
  </p>
  <p class="graf graf--p">
   国会通过了一项特别法案以批准这笔款项。如果 Frank Olson 在坟墓里保持沉默，这个案子就可以结案了。
  </p>

  <p class="graf graf--p">
   在 Olson 的葬礼上，Gottlieb 告诉悲伤的亲属，如果他们想问 “发生了什么”，他很乐意回答。
  </p>
  <p class="graf graf--p">
   20多年后，也就是1984年底，家属决定接受这一提议，并打电话约定见面时间。当 Alice、Eric 和 Nils Olson 出现在 Gottlieb 家门口时，他的第一反应是如释重负。
  </p>
  <p class="graf graf--p">
   Gottlieb 说：“我很高兴你们没带武器。我昨晚做了个梦，梦见你们都走到这扇门前，朝我开了枪。”
  </p>
  <p class="graf graf--p">
   Eric 大吃一惊。后来，
   <strong class="markup--strong markup--p-strong">
    他惊奇于 Gottlieb 操纵人心的手段
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们还没进门，就已经开始向我们道歉，并让我们安心。这种扭转立场的方式聪明而圆滑”。
  </p>
  <p class="graf graf--p">
   他首先告诉了这家人1953年11月19日在深溪湖发生的事。
  </p>
  <p class="graf graf--p">
   他说，给 Olson 等人服用LSD是实验的一部分，以观察 “如果一个科学家被捕并被下药，会发生什么？他会泄露秘密研究和信息吗？”
  </p>
  <p class="graf graf--p">
   然后他开始谈及 Olson。“你父亲和我很像，”他告诉 Eric。“我们都是出于爱国之心。但我们都做得有点过头了，做了一些可能不应该做的事儿。”
  </p>
  <p class="graf graf--p">
   这是 Gottlieb 最坦白的一次。
  </p>
  <p class="graf graf--p">
   他不愿透露 MK-Ultra 项目在哪些方面 “过头”，也不愿透露他和 Olson 做了哪些 “可能不应该做” 的事。他也不会对 Olson 之死前后矛盾的说法提出质疑。当 Eric 追问他时，他反应激烈。
  </p>
  <p class="graf graf--p">
   当这家人起身准备离开时，Gottlieb 把 Eric 拉到一边：“显然，你父亲的自杀让你很困扰。你有没有想过加入为父母自杀的人开设的心理治疗小组？”
  </p>
  <p class="graf graf--p">
   Eric 没有采纳这个建议，但却给他留下了深刻印象。
  </p>
  <p class="graf graf--p">
   多年来，他一直对父亲的死感到困惑和沮丧。直到与 Gottlieb 见面之后，他才决心将探索真相置于生活的中心。
  </p>
  <p class="graf graf--p">
   他说：“我当时没有信心去怀疑这一切，
   <strong class="markup--strong markup--p-strong">
    导致我忽视了他的伎俩
   </strong>
   ，但是，当他提出治疗小组的建议时 — — 那一刻我意识到他管得太宽了。在那一刻，我明白了 Gottlieb 多么想平息我的情绪。也就是在那一刻，我决定要证明他在谋杀我父亲的过程中扮演了什么角色。”
  </p>
  <p class="graf graf--p">
   Eric Olson 又等了10年，直到他母亲去世后他才采取下一步行动：安排挖掘他父亲的遗体。
  </p>
  <p class="graf graf--p">
   1994年6月2日，在马里兰州弗雷德里克的林登山公墓，几名记者站在他旁边，一台挖掘机正在翻土。
  </p>
  <p class="graf graf--p">
   乔治·华盛顿大学法学院的法医病理学家 James Starrs 花了一个月的时间检查 Olson 的尸体。完成后，他召开了新闻发布会。
  </p>
  <p class="graf graf--p">
   他报告说，体内毒素的检验没有发现任何问题。然而，
   <strong class="markup--strong markup--p-strong">
    伤口形状很奇怪 —— 受害者的头或脖子上没有发现玻璃碎片，这与他从窗户跳下的前提不大一致。
   </strong>
   最令人费解的是，据说 Olson 是仰面着地的，但他左眼上方的头骨却严重受损。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我敢说，这个血肿是唯一的证据，证明 Olson 博士在从1018A房间的窗户摔出去之前，可能被人或器械猛击了头部。”
  </p>
  <p class="graf graf--p">
   James Starrs 总结道。后来，他更加强调：“我认为，Frank Olson 是被蓄意、恶意地扔出窗外的。”
  </p>
  <p class="graf graf--p">
   除了进行尸检，Starrs 还采访了与案件有关的其他人，其中一个就是 Gottlieb。两人于周日上午在弗吉尼亚州 Gottlieb 的家中会面。
  </p>
  <p class="graf graf--p">
   Starrs 后来写道，这是：“我进行的所有采访中最令人困惑的一次”。
  </p>
  <p class="graf graf--p">
   Starrs 写道：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “我壮着胆子问他，他怎么能如此鲁莽和傲慢地用 LSD 进行 ‘深溪小屋’ 实验，伤害这么多手下人的生命。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    他毫不客气地说，‘教授，你就是不明白。这个国家的安全掌握在我的手中。’他没有再说什么，也不必再说了。我目瞪口呆，也没有反驳。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    手段-目的论的寓意非常清楚。拿不知情的实验受害者的生命冒险，只是为了更大层面上的利益，被认为是保护国家安全的必要手段。”
   </em>
  </p>
  <figure aria-describedby="caption-attachment-12590" class="wp-caption aligncenter" id="attachment_12590" style="width: 2048px">
   <img alt="" class="size-full wp-image-12590 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/12.jpeg?resize=1100%2C734&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/12.jpeg?w=2048&amp;ssl=1 2048w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/12.jpeg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/12.jpeg?resize=1024%2C683&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/12.jpeg?resize=768%2C512&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/12.jpeg?resize=1536%2C1025&amp;ssl=1 1536w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/12.jpeg?resize=1100%2C734&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/12.jpeg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="734" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/12.jpeg?resize=1100%2C734&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   <figcaption class="wp-caption-text" id="caption-attachment-12590">
    Errol Morris 的新电影 “Wormwood” 融合了纪录片和艺术化，讲述了 Frank Olson 的死亡。Peter Sarsgaard 饰演 Olson
   </figcaption>
  </figure>
  <p class="graf graf--p">
   Olson 的家人在1975年接受75万美元封口费时已签字放弃了获得法律救济的权利，所以他们不能起诉CIA。
  </p>
  <p class="graf graf--p">
   尽管 Starrs 的报告和其他发现让 Eric 加深了本就已经很强烈的怀疑 — — 他父亲的死存在幕后黑手，但他无法证明。
  </p>
  <p class="graf graf--p">
   意识到这一痛苦的事实后，他和弟弟决定重新安葬父亲的遗体。
  </p>
  <p class="graf graf--p">
   2002年8月8日，在重新安葬的前一天，他把记者叫到家中，宣布他对父亲的遭遇有了新的结论。
  </p>
  <p class="graf graf--p">
   他宣称：“1953年11月28日 Frank Olson 的死亡是谋杀，而非自杀。
   <strong class="markup--strong markup--p-strong">
    这不是1975年呈现给世人的那样单纯的LSD药物实验， 而是关于生化战的事件
   </strong>
   。Frank Olson 并非是由于在实验中经历了 “糟糕体验” 而自杀。
   <strong class="markup--strong markup--p-strong">
    他之所以被害，是有人担心他会泄露20世纪50年代早期CIA的绝密审讯项目，以及美国在朝鲜战争中使用生化武器的信息
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   2017年，曾调查过 Olson 一案并一直对此感兴趣的纽约退休助理检察官 Stephen Saracco 第一次来到 Olson 身前居住的酒店房间。Saracco 环顾房间，提出了一个问题：Olson 是如何做到这一点的？
  </p>
  <p class="graf graf--p">
   Saracco 总结道：“如果这是自杀，那将很难达成。这起案件存在他杀的动机，因为他知道冷战最高层、最黑暗的秘密。如果政府认为一个为CIA和军队工作的科学家是所谓的安全隐患，他们会杀了他吗？肯定有人会说：‘必然会的。’”
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/us-news/2019/sep/06/from-mind-control-to-murder-how-a-deadly-fall-revealed-the-cias-darkest-secrets" href="https://www.theguardian.com/us-news/2019/sep/06/from-mind-control-to-murder-how-a-deadly-fall-revealed-the-cias-darkest-secrets" rel="noopener noreferrer" target="_blank">
    From mind control to murder? How a deadly fall revealed the CIA’s darkest secrets
   </a>
  </p>
