  </p>
  <p class="graf graf--p">
   这是
   <strong>
    导读部分
   </strong>
   ：
  </p>
  <ul>
   <li class="graf graf--p graf--startsWithDoubleQuote">
    “问题不在于夺取权力，而在于行使权力。” ——《
    <a href="https://www.iyouport.org/1%E3%80%81-%E5%85%B1%E8%AF%86%E5%8F%AA%E6%98%AF%E6%89%8B%E6%AE%B5%E8%80%8C%E4%B8%8D%E6%98%AF%E7%9B%AE%E7%9A%84/" rel="noopener noreferrer" target="_blank">
     共识只是手段而不是目的
    </a>
    》；
   </li>
   <li>
    “展示运动的广度、多样性和力量；你不需要把队友集中在同一个大目标上” ——《
    <a href="https://www.iyouport.org/2%E3%80%81%E5%88%86%E5%B8%83%E5%BC%8F%E8%A1%8C%E5%8A%A8%E7%9A%84%E8%A1%97%E5%A4%B4%E6%B4%BE%E5%AF%B9/" rel="noopener noreferrer" target="_blank">
     分布式行动的街头派对
    </a>

   </li>
   <li>
    “关于框架有一个基本的事实：如果你接受了别人的框架，你就输了” —— 《
    <a href="https://www.iyouport.org/3%E3%80%81%E6%A1%86%E6%9E%B6%E6%98%AF%E4%B8%80%E7%A7%8D%E6%9D%83%E5%8A%9B/" rel="noopener noreferrer" target="_blank">
     框架是一种权力
    </a>
    》；
   </li>
   <li>
    “你永远不可能仅仅通过对抗现有的现实来改变这一切。要改变某些东西，你必须建立一个新的模型，使现有模型过时” —— 《
    <a href="https://www.iyouport.org/4%E3%80%81%E5%88%AB%E6%8A%8A%E6%B3%A8%E6%84%8F%E5%8A%9B%E9%9B%86%E4%B8%AD%E5%9C%A8%E6%8A%B5%E6%8A%97%E4%B8%8A%EF%BC%9A%E9%A2%84%E8%AE%BE%E6%80%A7%E5%B9%B2%E9%A2%84%E6%88%98%E6%9C%AF/" rel="noopener noreferrer" target="_blank">
     预设性干预战术
    </a>
    》
   </li>
  </ul>

   </noscript>
  </p>
  <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote">
   <p>
    <em>
     “问题不在于夺取权力，而在于行使权力。”
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “经过投票大家一致同意”，这句话听起来无比美好是吧，事实上并不一定。
  </p>
  <p class="graf graf--p">
   我们得知威权国家的公民社会组织和反对派团体内部在积极地施行民主实践，这真是太好了；只有通过不断实践才能掌握民主议事的要领。
  </p>
  <p class="graf graf--p">
   不过，如果实践中屡战屡败，有可能严重消耗集体热情，甚至会有人因此怀疑民主本身，那就太糟糕了。
  </p>
  <p class="graf graf--p">
   <strong>
    如果你觉得自己在这个社会里是边缘人士，缺乏发表不同意见的机会，于是你加入了反对派，以抵制社会不公；然而你发现自己在反对派组织中也处于边缘地位，你被忽视，经常只能“参见主流意见”，那么将说明你加入的组织在运作方面出了点问题
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这种时候你怎么办？离开他们组建自己的小圈子？也许曾经有很多人这样做了，这就是为什么你看到一个大社区不断地分化、变小、再变小……直到重新回到一盘散沙。
  </p>
  <p class="graf graf--p">
   共识决策是反抗者组织所声称追求的，罗伯特议事规则被奉为圣经。但是，有些时候它也会出问题；尤其是对于那些缺乏民主经验的社会中的反对派组织而言。
  </p>
  <p class="graf graf--p">
   <strong>
    共识决策的两个基本价值观是
   </strong>
   ：赋予每个人充分参与决策的权利，尊重和容纳不同意见的能力。
  </p>
  <p class="graf graf--p">
   ⚠️这些价值观比形式本身更重要，活动家应根据需要努力协调以维护这些价值观。
  </p>
  <p class="graf graf--p">
   共识决策是一种平等主义和包容性的方法，在集体成员积极参与和同意的基础上达成协议，共同作出决定。
  </p>
  <p class="graf graf--p">
   共识决策的重点在于决策本身的基本流程和价值观，而不是形式。共识这个词源于拉丁语“consentire”，意思是“共同的体验或感受”。
  </p>
  <p class="graf graf--p">
   共识根植于世界各地实行的直接民主的分散化模式 —— 从印度的村庄 panchayats 到土著 Haudenosaunee 联邦（又名 Iroquois），从贵格会到无政府主义者轮辐会议。
  </p>
  <p class="graf graf--p">
   顺便说，中国的社区团体不论是地域性的还是立场性的，都可以作为直接民主的实验田。关键在于掌握协商要领。
  </p>
  <p class="graf graf--p">
   一方面，共识是提案经过辩论然后进行投票，这种方法如今大多数人都能理解；另一方面，共识是对我们根据直接民主原则组织自己的权力的一种预设的承诺：横向、参与、包容、合作和非强制性。但要做到并不容易。
  </p>
  <p class="graf graf--p">
   然而，⚠️滥用共识是对与之相关的结构和形式的教条式依附，它有时可能会变得与它想要取代的体制一样的霸道和令人疏远。
  </p>
  <p class="graf graf--p">
   或许在中国的异议人士会有相关经验？你是否发现反对派群体和他们所反对的人在某些方面“如此相似”？
  </p>
  <p class="graf graf--p">
   在这肯定是很不好的，令人失望。
  </p>
  <p class="graf graf--p">
   如果发生这种情况，参与者的反应不应该是“这就是共识的工作方式！”相反，我们的共同责任是深入研究可能造成这些负面反应的动态。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    共识存在五个常见问题，可能会造成挫败感
   </strong>
   。
  </p>
  <p class="graf graf--p">
   首先，共识通常通过创建那些达成一致的阵营而不是那些阻挠的阵营来重现多数主义统治。
  </p>
  <p class="graf graf--p">
   与普遍看法相反，共识并不一定意味着一致同意。
  </p>
  <p class="graf graf--p">
   如果误解这点，会导致人们错误地认为异议是一种分心或障碍，并增加了使意见同质化的压力。
  </p>
  <p class="graf graf--p">
   你一定熟悉一群人相处一段时间后异议就消失了，那种令人困惑的局面。
  </p>
  <p class="graf graf--p">
   第二，一些声音可以支配讨论，这个问题往往会使种族，阶级，性别和教育水平的权力不平衡永久化。甚至小圈子内也存在这种情况。
  </p>
  <p class="graf graf--p">
   第三，通常存在一种错误的假设，即 沉默意味着同意，最终可能会扼杀更广泛的讨论和对替代提案的考虑。
  </p>
  <p class="graf graf--p">
   第四，协调人有一种不幸的倾向，即通过基于他们自己的偏见来指导对话，从而行使隐蔽形式的权力。
  </p>
  <p class="graf graf--p">
   共识的第五个问题更具基础性和结构性。
   <strong class="markup--strong markup--p-strong">
    具有讽刺意味的是，表面上看起来的一致就可以隐藏系统性不平等的令人不安的真相
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果是这样的，那可不是个好的民主，哪怕赋予投票权。
  </p>
  <p class="graf graf--p">
   几乎天生就是如此，共识过程可能导致人们无法积极研究特权和压迫的问题如何塑造我们的空间。
  </p>
  <p class="graf graf--p">
   为了解决这些问题，许多社区和集体开始尝试使用修改后的共识形式 —— 例如，
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    优先考虑并从代表性不足的边缘群体以及直接受到决策影响的人那里获得意见；
   </li>
   <li class="graf graf--li">
    促进小型突破小组，以确保更多的参与；
   </li>
   <li class="graf graf--li">
    鼓励更多的辩论和讨论，而不仅仅是要求将团体细分（即 你熟知的圈子越来越小）；
   </li>
   <li class="graf graf--li">
    积极纳入反压迫原则，以防止处于不利地位的人被进一步边缘化。
   </li>
  </ul>
  <p class="graf graf--p">
   共识可以是美好的和变革性的，但是，只有当结构和过程满足参与其中的人的需求和愿望时，它才能做到。
  </p>
  <p class="graf graf--p">
   <strong>
    否则，它可能是像传统的权威决策系统一样的枷锁
   </strong>
   。
  </p>
  <p class="graf graf--p">
   请记住，共识是达到目的的手段，而不是目的本身。⚪️
  </p>
