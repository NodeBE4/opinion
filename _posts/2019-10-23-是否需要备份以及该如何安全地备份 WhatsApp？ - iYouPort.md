  <p class="graf graf--p">
   消息传递平台 WhatsApp 在全球拥有15亿用户，⚠️并且越来越多地被用作组织，动员和分享重要人权内容的工具。
  </p>
  <p class="graf graf--p">
   本教程希望能帮助您决定是否应该使用 WhatsApp 的内置备份选项，如果是，该如何最安全地备份 WhatsApp。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    备份 WhatsApp 与从 WhatsApp 导出内容是不同的。
   </strong>
  </p>
  <p class="graf graf--p">
   备份用于恢复 WhatsApp 帐户，例如，如果您要更换手机。备份以加密的 WhatsApp 数据库格式存储，于是它无法在 WhatsApp 之外打开或读取。
  </p>
  <p class="graf graf--p">
   导出的内容无法像备份一样恢复到您的 WhatsApp 帐户，但如果您想将重要内容保存在 WhatsApp 之外，则非常有用。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    为什么要备份 WhatsApp？
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    如果您需要从手机中删除 WhatsApp，稍后重新安装。
   </li>
  </ul>
  <p class="graf graf--p">
   ⚠️如果您正在面临被搜查手机的状况（不论是在机场还是在街头）；或者您担心自己的物理和数字安全性，则可能需要在手机上先卸载 WhatsApp。
  </p>
  <p class="graf graf--p">
   通过在卸载之前备份 WhatsApp，您可以稍后从此备份重新安装和恢复您的内容。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    如果您的手机丢失、被盗、损坏、或您需要更换它。
   </li>
  </ul>
  <p class="graf graf--p">
   WhatsApp 消息和媒体存储在您的手机上，而不是存储在 WhatsApp 的服务器上（WhatsApp 在30天后删除来自其服务器的消息）。
  </p>
  <p class="graf graf--p">
   但是手机可能会丢失、被盗或被损坏。💡将 WhatsApp 备份到您的计算机上应该是确保您可以在设备出现问题时恢复 WhatsApp 的唯一方法。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    如果您要还原为备份副本。
   </li>
  </ul>
  <p class="graf graf--p">
   在某些情况下，您可能希望将 WhatsApp 还原为备份副本；例如，如果您不小心删除了一些重要内容。
  </p>
  <p class="graf graf--p">
   在大多数情况下，例如使用 Google 云端硬盘或 iCloud 备份时，您只能恢复到最新的备份。
  </p>
  <p class="graf graf--p">
   但是，Android 设备会存储最近7天的备份，Android 用户可以从较旧的本地备份进行恢复。
  </p>
  <p class="graf graf--p">
   iPhone 用户只能通过整个设备的 iTunes 备份恢复到较旧（非最新）的备份。
  </p>
  <p class="graf graf--p">
   请务必注意，从备份还原时，将丢失备份日期之后创建的所有新内容。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    为什么不想备份 WhatsApp？
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    WhatsApp 备份不如 WhatsApp 聊天安全。
   </li>
  </ul>
  <p class="graf graf--p">
   WhatsApp 提供内置选项来备份本地（Android）和谷歌 Drive（Android）或 iCloud（iPhone）。
  </p>
  <p class="graf graf--p">
   Android 用户可以在 Google 云端硬盘上免费备份所有 WhatsApp 数据，不受文件大小限制（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%82%a8%e5%bf%85%e9%9c%80%e4%bd%bf%e7%94%a8-whatsapp%ef%bc%9a%e8%bf%99%e9%87%8c%e6%98%af%e5%87%8f%e5%b0%91%e4%bc%a4%e5%ae%b3%e7%9a%84%e5%8a%9e%e6%b3%95/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%82%a8%e5%bf%85%e9%9c%80%e4%bd%bf%e7%94%a8-whatsapp%ef%bc%9a%e8%bf%99%e9%87%8c%e6%98%af%e5%87%8f%e5%b0%91%e4%bc%a4%e5%ae%b3%e7%9a%84%e5%8a%9e%e6%b3%95/" rel="noopener noreferrer" target="_blank">
    不推荐这样做
   </a>
   ）。
  </p>
  <p class="graf graf--p">
   这些备份由 WhatsApp 加密，如果上传，也可以通过谷歌或 Apple 的服务器端加密，但它们不受 WhatsApp 的端到端加密保护。
  </p>
  <p class="graf graf--p">
   当您进行备份时，WhatsApp 会使用您的电话号码生成密钥对其进行加密。
  </p>
  <p class="graf graf--p">
   当备份上传到 Google 云端硬盘或 iCloud 时，Google 或 Apple 也会使用自己的服务器端对其进行加密。
  </p>
  <p class="graf graf--p">
   ⚠️请注意：这意味着这些公司或拥有您的 Google 帐户或 iCloud 授权的任何人都可以解密备份中的服务器端加密，并且可以向执法部门提供备份。
  </p>
  <p class="graf graf--p">
   ⚠️同时，用于解密的密钥存储在您的手机上，或者可以由使用带有您的电话号码的 SIM 卡的任何人重新生成，因为它旨在使用户能够轻松地将备份恢复到新手机上。当然也因此不安全。
  </p>
  <p class="graf graf--p">
   同样重要的是要注意，WhatsApp 备份加密只加密文本对话，而不是媒体内容（视频和图像）。
  </p>
  <p class="graf graf--p">
   WhatsApp 不会在手机上加密媒体附件，也不会将其上传到 Google 云端硬盘和 iCloud。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    谷歌和苹果可能会向执法部门提供备份访问权限。
   </li>
  </ul>
  <p class="graf graf--p">
   ⚠️当您将内容备份到所谓的云端时，您就彻底无法控制它了。如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%82%a8%e5%bf%85%e9%9c%80%e4%bd%bf%e7%94%a8-whatsapp%ef%bc%9a%e8%bf%99%e9%87%8c%e6%98%af%e5%87%8f%e5%b0%91%e4%bc%a4%e5%ae%b3%e7%9a%84%e5%8a%9e%e6%b3%95/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%82%a8%e5%bf%85%e9%9c%80%e4%bd%bf%e7%94%a8-whatsapp%ef%bc%9a%e8%bf%99%e9%87%8c%e6%98%af%e5%87%8f%e5%b0%91%e4%bc%a4%e5%ae%b3%e7%9a%84%e5%8a%9e%e6%b3%95/" rel="noopener noreferrer" target="_blank">
    前文所述
   </a>
   。
  </p>
  <p class="graf graf--p">
   所以建议使用替代方案。
  </p>
  <p class="graf graf--p">
   Android 用户可以选择仅使用本地（加密）备份，而 iPhone 用户可以使用 iTunes（带加密选项）将整个手机备份到自己的计算机上。
  </p>
  <p class="graf graf--p">
   导出 WhatsApp 内容是备份以保存重要文本和媒体信息的另一种方法，尽管您无法像将常规备份那样将导出的内容简单的恢复到 WhatsApp 帐户。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    请注意，所有这些选项都有其自身的安全风险
   </strong>
   ，例如，如果您的设备丢失、被盗或被警察没收，或者您的云服务或设备遭到黑客入侵或破坏 ……
  </p>
  <p class="graf graf--p">
   请按照我们
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%82%a8%e5%bf%85%e9%9c%80%e4%bd%bf%e7%94%a8-whatsapp%ef%bc%9a%e8%bf%99%e9%87%8c%e6%98%af%e5%87%8f%e5%b0%91%e4%bc%a4%e5%ae%b3%e7%9a%84%e5%8a%9e%e6%b3%95/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%82%a8%e5%bf%85%e9%9c%80%e4%bd%bf%e7%94%a8-whatsapp%ef%bc%9a%e8%bf%99%e9%87%8c%e6%98%af%e5%87%8f%e5%b0%91%e4%bc%a4%e5%ae%b3%e7%9a%84%e5%8a%9e%e6%b3%95/" rel="noopener noreferrer" target="_blank">
    上一篇文章中
   </a>
   的说明操作。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    云备份使内容更容易受到攻击和窃取
   </li>
  </ul>
  <p class="graf graf--p">
   由于云存储可能引入许多漏洞，因此在云上存储数据可能会非常不安全。
  </p>
  <p class="graf graf--p">
   个人帐户可能会被弱密码、糟糕的密码管理、和网络钓鱼攻击所破坏（使用双因素身份验证，这有助于保护用户。在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e4%b8%8d%e6%9a%b4%e9%9c%b2%e6%89%8b%e6%9c%ba%e5%8f%b7%e3%80%81%e4%b8%8d%e8%8a%b1%e9%92%b1%ef%bc%8c%e5%ae%9e%e7%8e%b0%e5%8f%8c%e5%9b%a0%e7%b4%a0%e8%ba%ab%e4%bb%bd%e9%aa%8c%e8%af%81/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e4%b8%8d%e6%9a%b4%e9%9c%b2%e6%89%8b%e6%9c%ba%e5%8f%b7%e3%80%81%e4%b8%8d%e8%8a%b1%e9%92%b1%ef%bc%8c%e5%ae%9e%e7%8e%b0%e5%8f%8c%e5%9b%a0%e7%b4%a0%e8%ba%ab%e4%bb%bd%e9%aa%8c%e8%af%81/" rel="noopener noreferrer" target="_blank">
    不使用手机号码实现双因素身份验证的方法
   </a>
   》）。
  </p>
  <p class="graf graf--p">
   云存储提供商也可能具有较差的安全实践，例如对其数据库的保护较弱，使他们及其用户容易受到破坏、骇客攻击和泄密。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如何备份 WhatsApp？
   </strong>
  </p>
  <p class="graf graf--p">
   如下，选择适合您的内容。
  </p>
  <p class="graf graf--p">
   <strong>
    1、如果想在 Android 操作系统上备份 WhatsApp
   </strong>
  </p>
  <p class="graf graf--p">
   <strong>
    1–1 想做一个本地备份
   </strong>
  </p>
  <p class="graf graf--p">
   你不需要做任何事情。 WhatsApp 每天凌晨2:00自动为您的设备创建本地备份。
  </p>
  <p class="graf graf--p">
   无法关闭此功能。
  </p>
  <p class="graf graf--p">
   WhatsApp 将最近7天的备份存储在您的设备上。
  </p>
  <p class="graf graf--p">
   如果要在24小时备份周期之间的某个时间点手动备份，请转到 WhatsApp &gt;菜单&gt;设置&gt;聊天&gt;聊天备份&gt;备份。
  </p>
  <p class="graf graf--p">

   <img alt="" class="aligncenter size-full wp-image-6024 jetpack-lazy-image" data-lazy-sizes="(max-width: 734px) 100vw, 734px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/21.png?resize=734%2C128&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/21.png?w=734&amp;ssl=1 734w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/21.png?resize=300%2C52&amp;ssl=1 300w" data-recalc-dims="1" height="128" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/21.png?resize=734%2C128&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="734"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6024" data-recalc-dims="1" height="128" sizes="(max-width: 734px) 100vw, 734px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/21.png?resize=734%2C128&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/21.png?w=734&amp;ssl=1 734w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/21.png?resize=300%2C52&amp;ssl=1 300w" width="734"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   2. 在 iTunes 中的“备份”下，选择“此计算机”。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6025 jetpack-lazy-image" data-lazy-sizes="(max-width: 1089px) 100vw, 1089px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/22.png?resize=1089%2C337&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/22.png?w=1089&amp;ssl=1 1089w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/22.png?resize=300%2C93&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/22.png?resize=768%2C238&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/22.png?resize=1024%2C317&amp;ssl=1 1024w" data-recalc-dims="1" height="337" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/22.png?resize=1089%2C337&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1089"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6025" data-recalc-dims="1" height="337" sizes="(max-width: 1089px) 100vw, 1089px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/22.png?resize=1089%2C337&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/22.png?w=1089&amp;ssl=1 1089w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/22.png?resize=300%2C93&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/22.png?resize=768%2C238&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/22.png?resize=1024%2C317&amp;ssl=1 1024w" width="1089"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   3. 您可以选择
   <strong class="markup--strong markup--p-strong">
    “加密 iPhone 备份”
   </strong>
   来保护您的数据。加密备份包括来自 iPhone 的密码和健康数据等信息，这些信息不会包含在未加密的备份中。
  </p>
  <p class="graf graf--p">
   请注意，如果丢失密码，则无法恢复加密备份。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6026 jetpack-lazy-image" data-lazy-sizes="(max-width: 512px) 100vw, 512px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/23.png?resize=512%2C320&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/23.png?w=512&amp;ssl=1 512w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/23.png?resize=300%2C188&amp;ssl=1 300w" data-recalc-dims="1" height="320" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/23.png?resize=512%2C320&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="512"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6026" data-recalc-dims="1" height="320" sizes="(max-width: 512px) 100vw, 512px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/23.png?resize=512%2C320&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/23.png?w=512&amp;ssl=1 512w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/23.png?resize=300%2C188&amp;ssl=1 300w" width="512"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   4. 使用“立即备份”启动备份：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6027 jetpack-lazy-image" data-lazy-sizes="(max-width: 1089px) 100vw, 1089px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/24.png?resize=1089%2C337&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/24.png?w=1089&amp;ssl=1 1089w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/24.png?resize=300%2C93&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/24.png?resize=768%2C238&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/24.png?resize=1024%2C317&amp;ssl=1 1024w" data-recalc-dims="1" height="337" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/24.png?resize=1089%2C337&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1089"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6027" data-recalc-dims="1" height="337" sizes="(max-width: 1089px) 100vw, 1089px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/24.png?resize=1089%2C337&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/24.png?w=1089&amp;ssl=1 1089w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/24.png?resize=300%2C93&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/24.png?resize=768%2C238&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/24.png?resize=1024%2C317&amp;ssl=1 1024w" width="1089"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   iTunes 备份存储在计算机上的 (hidden) ~/Library/Application Support/MobileSync/Backup/ 文件夹中。
  </p>
  <p class="graf graf--p">
   您还可以导航到 iTunes&gt;首选项&gt;设备中的备份位置，右键单击所选备份，然后选择“在 Finder 中显示”。
  </p>
  <p class="graf graf--p">
   但是，要确保 iTunes 可以正确还原备份，请不要移动或更改备份文件。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6028 jetpack-lazy-image" data-lazy-sizes="(max-width: 512px) 100vw, 512px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/25.png?resize=512%2C199&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/25.png?w=512&amp;ssl=1 512w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/25.png?resize=300%2C117&amp;ssl=1 300w" data-recalc-dims="1" height="199" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/25.png?resize=512%2C199&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="512"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6028" data-recalc-dims="1" height="199" sizes="(max-width: 512px) 100vw, 512px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/25.png?resize=512%2C199&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/25.png?w=512&amp;ssl=1 512w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/25.png?resize=300%2C117&amp;ssl=1 300w" width="512"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong>
    2–4 如果想从 iTunes 备份恢复
   </strong>
  </p>
  <p class="graf graf--p">
   请注意，WhatsApp 不支持此方法，并且无法保证100％恢复聊天文本和媒体内容。
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    将手机连接到电脑。解锁手机并允许计算机/ iTunes 访问它。系统将提示您关闭手机上的“查找我的 iPhone”功能。要暂时关闭此功能，请转到设置&gt; iCloud&gt;查找我的 iPhone。关闭查找我的 iPhone并输入您的 Apple ID 密码。在 iTunes 中，选择设备图标：
   </li>
  </ol>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6029 jetpack-lazy-image" data-lazy-sizes="(max-width: 734px) 100vw, 734px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/26.png?resize=734%2C128&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/26.png?w=734&amp;ssl=1 734w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/26.png?resize=300%2C52&amp;ssl=1 300w" data-recalc-dims="1" height="128" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/26.png?resize=734%2C128&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="734"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6029" data-recalc-dims="1" height="128" sizes="(max-width: 734px) 100vw, 734px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/26.png?resize=734%2C128&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/26.png?w=734&amp;ssl=1 734w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/26.png?resize=300%2C52&amp;ssl=1 300w" width="734"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   2. 在 iTunes 中的“备份”下，选择“还原备份”，然后选择要从中还原的备份：

   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6030" data-recalc-dims="1" height="936" sizes="(max-width: 1024px) 100vw, 1024px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/27.png?resize=1024%2C936&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/27.png?w=1024&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/27.png?resize=300%2C274&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/27.png?resize=768%2C702&amp;ssl=1 768w" width="1024"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   3. 重新打开手机上的“查找我的 iPhone”功能。
  </p>
  <p class="graf graf--p">
   <strong>
    2–5 如果你想删除 iTunes 备份
   </strong>
  </p>
  <p class="graf graf--p">
   在 iTunes 中，转到 iTunes&gt;首选项&gt;设备。右键单击要删除的备份，然后选择“删除”：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6031 jetpack-lazy-image" data-lazy-sizes="(max-width: 577px) 100vw, 577px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/28.png?resize=577%2C477&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/28.png?w=577&amp;ssl=1 577w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/28.png?resize=300%2C248&amp;ssl=1 300w" data-recalc-dims="1" height="477" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/28.png?resize=577%2C477&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="577"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6031" data-recalc-dims="1" height="477" sizes="(max-width: 577px) 100vw, 577px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/28.png?resize=577%2C477&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/28.png?w=577&amp;ssl=1 577w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/28.png?resize=300%2C248&amp;ssl=1 300w" width="577"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   请注意，备份文件夹会在 Time Machine 中备份（如果启用了此功能），因此，从计算机中删除备份后，仍可能存在备份的现有副本。如果您正在做的是《
