   </noscript>
  </p>
  <p class="graf graf--p">
   通过向我们出售硬件但保留软件和数据的所有权，科技公司正在将用户视为数字租户。
  </p>
  <p class="graf graf--p">
   当蒂姆库克和他的同事们在3月25日为苹果公司的活动登台表演时，他们宣布了
   <strong class="markup--strong markup--p-strong">
    一个重要的战略支点
   </strong>
   。
  </p>
  <p class="graf graf--p">
   该公司知道其 iPhone、iPad 和 Mac 的市场
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bloomberg.com/news/articles/2019-01-03/apple-s-cook-faces-toughest-test-yet-navigating-iphone-slowdown" href="https://www.bloomberg.com/news/articles/2019-01-03/apple-s-cook-faces-toughest-test-yet-navigating-iphone-slowdown" rel="noopener noreferrer" target="_blank">
    停滞不前
   </a>
   ，这一趋势不太可能很快改变。
  </p>
  <p class="graf graf--p">
   这意味着 Apple 需要向其客户出售其他产品。
  </p>
  <p class="graf graf--p">
   苹果公司在宣布推出一系列需要订阅的新服务后宣称：“苹果公司作为一家服务公司的重塑是真实的：
  </p>
  <p class="graf graf--p">
   Apple TV +用于播放电影和电视节目；Apple News + 用于汇总新闻出版物；Apple Arcade 用于玩游戏；Apple Card 用于支付所有费用……
  </p>
  <p class="graf graf--p">
   凭借 Apple 庞大的内置用户群，金融分析师估计该公司可能在短短几年内就能达到1亿用户，随着时间的推移
   <strong class="markup--strong markup--p-strong">
    创造出“每年70亿至100亿美元的收入流”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   苹果公司所做的不仅仅是应对竞争压力 —— 而是，它正在跟随
   <strong class="markup--strong markup--p-strong">
    技术如何被用来改变财产所有权和利润积累之概念的转变
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Facebook、Uber 和 Netflix 等等都在构建平台并提供服务，努力将自己插入社交关系、经济交易和个人消费。
  </p>
  <p class="graf graf--p">
   它们深度渗透到人们的日常活动中，并无止境地收集有关我们行为和兴趣的宝贵数据。
  </p>
  <p class="graf graf--p">
   而且，至关重要的是，他们收取访问权费用 —— 而不是所有权，这种模式已经越来越过时了。
  </p>
  <p class="graf graf--p">
   然而，所有这些公司正在做的事 —— 包括苹果公司 ——
   <strong class="markup--strong markup--p-strong">
    正在重振长期与地主和封建主义联系在一起的旧形式资本主义幽灵。
   </strong>
  </p>
  <p class="graf graf--p">
   无论我们称之为“平台资本主义”、“监视资本主义”，还是仅仅“下一代资本主义”，都没什么不同，这种资本运作模式采用的是调解和封闭来实现对其主体的提取和控制。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “Rentier”是指资产所有者向其他人收取访问该资产的费用的关系，就像房东向租户收取房租一样。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    很难低估
   </strong>
   这种租赁模式现在应用于消费者世界的程度。
  </p>
  <p class="graf graf--p">
   近年来，大量企业将自己描述为 “Uber for X” 或 “X as a service” —— “即服务”句式，例如 WeWork
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/commentisfree/2017/dec/03/digital-hippies-integrate-life-and-work-wework-data-firms" href="https://www.theguardian.com/commentisfree/2017/dec/03/digital-hippies-integrate-life-and-work-wework-data-firms" rel="noopener noreferrer" target="_blank">
    提供
   </a>
   “空间即服务”、或亚马逊的 Mechanical Turk
   <a class="markup--anchor markup--p-anchor" data-href="https://reallifemag.com/potemkin-ai/" href="https://reallifemag.com/potemkin-ai/" rel="noopener noreferrer" target="_blank">
    提供
   </a>
   “人类即服务”（人类变成投喂机器的流水线工人）。
  </p>
  <p class="graf graf--p">
   风险资本家和企业家正在寻找机会来获取价值 —— 美元和数据 —— 通过控制资产然后向用户收取访问这些资产的费用，无论是办公空间、音乐还是游戏。
  </p>
  <p class="graf graf--p">
   除了消费者服务之外，许多政府、企业、大学和其他组织都从平台租用核心服务，例如软件和存储。
  </p>
  <p class="graf graf--p">
   这些软件即服务业务现在
   <strong class="markup--strong markup--p-strong">
    发生在私人平台的生态系统中，为这些平台提供持续的收入来源，同时巩固了其在经济和社会中的关键地位。
   </strong>
  </p>
  <p class="graf graf--p">
   虽然像 Uber 和 Airbnb 这样的例子相当明显，但通过广泛应用的 X-as-a-service 模式，平台也能够以包含日常事物的方式扩展租赁关系。
  </p>
  <p class="graf graf--p">
   这里的关键技术是软件许可证，它允许新的租用者声称对嵌入的软件拥有所有权，并且从我们日常生活中使用的越来越多的物理实体中获取数据。
  </p>
  <p class="graf graf--p">
   由于物联网概念的崛起，许多曾经普普通通的日用品，如咖啡机和牙刷，现在都配备了软件、传感器和网络连接。所谓的“智能”版本正在成为产品设计和销售的默认方式。
  </p>
  <p class="graf graf--p">
   这些软件成为了物品功能的一部分，传感器收集有关物品使用方式的数据，Wi-Fi 将物品连接到公司的平台，以便下载和上传数据。
  </p>
  <p class="graf graf--p">
   而且，关键的是，
   <strong class="markup--strong markup--p-strong">
    当您购买智能设备时，您拥有的只是物理对象。
   </strong>
  </p>
  <p class="graf graf--p">
   数字软件是“许可的”，这只是“租用”的另一个委婉说辞，
   <strong class="markup--strong markup--p-strong">
    我们通过使用该东西生成的恒定数据流构成了我们向公司支付的“租金”的一部分。
   </strong>
  </p>
  <p class="graf graf--p">
   通过将曾经普通的物体集成到物联网中，这些巨头公司能够制定一种微圈地形式，在这种形式下，
   <strong class="markup--strong markup--p-strong">
    它们保留对物理实体的数字部分的所有权 —— 以及访问、控制和关闭软件的权利 —— 即使你“购买”了它。
   </strong>
  </p>
  <p class="graf graf--p">
   无论是流媒体内容还是许可软件，我们都在因将控制私有财产的特权转交给公司看门人而付出代价。
  </p>
  <p class="graf graf--p">
   数字控制限制你如何使用智能猫砂盒是一回事，但是花30,000美元购买的一辆车甚至花10万美元买的拖拉机似乎就是另一回事了 ——
   <strong class="markup--strong markup--p-strong">
    您拥有的只是一大块金属和橡胶，而实际操作车辆所需的软件你只能租用。
   </strong>
  </p>
  <p class="graf graf--p">
   Twitter 上充满了冰箱、门铃和割草机的
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/internetofshit?lang=en" href="https://twitter.com/internetofshit?lang=en" rel="noopener noreferrer" target="_blank">
    物联网例子
   </a>
   ，这些例子都需要互联网连接和常规软件更新才能发挥作用，而这些关键东西总是被
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/wp-admin/post.php?post=3635&amp;action=edit" href="https://www.iyouport.org/wp-admin/post.php?post=3635&amp;action=edit" rel="noopener noreferrer" target="_blank">
    互联网霸主
   </a>
   掌握，只能等待进一步的指示。
  </p>
  <p class="graf graf--p">
   正如 Gizmodo 的 Kashmir Hill
   <a class="markup--anchor markup--p-anchor" data-href="https://www.gizmodo.com.au/2018/02/the-house-that-spied-on-me/" href="https://www.gizmodo.com.au/2018/02/the-house-that-spied-on-me/" rel="noopener noreferrer" target="_blank">
    报道的那样
   </a>
   ，她连接的智能家居中的所有设备都与它们的制造商进行日常联系，不断向他们发送数据并ping更新。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “你购买的智能设备并不属于你；你与制造它的公司分享监护权。”
   </strong>
  </p>
  <p class="graf graf--p">
   作为一种封闭技术，软件许可广泛且秘密地成功实现了从“用户”到“所有者”的大规模权利转移。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这就好像你买了一套房子，但你不知道，以前的房东继续拥有厨房；你没有发现，直到他们收回你的厨房，只因为你试图在没有经过他们许可的情况下进行修理或改造。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    别忘了，他们一直在监视你如何使用厨房
   </strong>
   。
  </p>
  <p class="graf graf--p">
   考虑一下智能家居的兴起，这种类比看起来并不那么牵强。
  </p>
  <p class="graf graf--p">
   平台公司认为自己是服务提供商，而不是数字地主从他们的网络空间或他们的软件飞地中收取租金。但是，无论你叫它什么，也无法改变租借关系的实质。
  </p>
  <p class="graf graf--p">
   人们应该将这些新发展置于索取财产、控制权、捕获价值和促进不平等的长期历史中去思考。
  </p>
  <p class="graf graf--p">
   地理学家阿利斯泰尔·弗雷泽（Alistair Fraser）创造了“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.tandfonline.com/doi/abs/10.1080/03066150.2017.1415887" href="https://www.tandfonline.com/doi/abs/10.1080/03066150.2017.1415887" rel="noopener noreferrer" target="_blank">
    数据掠夺
   </a>
   ”这一短语，以展示这些
   <strong class="markup--strong markup--p-strong">
    当代的圈地动态
   </strong>
   是如何跟随人类历史上更为暴力的过程的脚步 —— 就如“土地掠夺”。
  </p>
  <p class="graf graf--p">
   我们现在看到的
   <strong class="markup--strong markup--p-strong">
    食利资本主义的版本正在适应数字时代，但它仍然保持着提取和控制的相同基本特征。
   </strong>
  </p>
  <p class="graf graf--p">
   这些租赁者不是通过建筑围栏和收租以获得利润，而是通过使用数字平台和给物理设备安装软件以获取价值。
  </p>
  <p class="graf graf--p">
   通过平台业务积累巨额财富的公司告诉你，所有权是一种老式的想法，同时大规模部署了诸如“分享”和“便利”这类诱人的流行语。
  </p>
  <p class="graf graf--p">
   但是，他们避免提到访问权的暂时性和掌控权的缺乏 —— 让你同意租用你每天使用的东西。
  </p>
  <p class="graf graf--p">
   没有人会希望自己无法掌控自己的全部生活，仅仅作为一个租客。然而，当人们默许或不抵制“提取即服务”的扩展时，这种情况就会发生。⚪️
  </p>
