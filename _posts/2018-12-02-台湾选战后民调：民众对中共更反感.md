 <p>
  <span style="font-size: 10pt;">
   台湾民众对中国共产党好感度下降
  </span>
 </p>
 <p>
 </p>
 <p>
  <span style="font-size: 12pt;">
   台湾最新民调显示，总统蔡英文执政不信任度创下就任两年以来新高。台湾九合一选举民进党惨败，有舆论认为，这是“亲中”立场的胜利，但是，同一项民调也显示，台湾民众对中国共产党的好感度不增反减。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   台湾九合一选举过后的第一份民调出炉。美丽岛电子报11月的国政民调显示，蔡英文的信任度掉了两个百分点为24.7%，创下就任新低，不信任度将近六成则创下新高纪录。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   由蔡英文领导的民进党在选举中面临空前挫败，台湾民众民进党的好感度也掉了4个百分点，降到2成4；但是，获得15席县市长选举大胜的国民党却也没讨到便宜，好感度比上个月下跌4.8个百分点。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   这次选举中主张“九二共识”的国民党高雄市长当选人韩国瑜在台湾掀起一股“韩”流，外溢效应也拉抬其他国民党候选人，因此有不少外媒解读，这是“亲中”立场的胜利。但是，在这次调查中却显示，台湾民众对中国共产党的好感度，从上个月的1成8，滑落近3个百分点只剩1成5。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   执行这次民调的戴立安解析，这次选举结果不乏认为中国获胜而美国挫败的分析，甚至揣测台湾民众接纳中国大陆对台的政治主张，这些说法显然错误，且未充分理解台湾民意。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   戴立安：“民众选国民党是希望在内政的处理上，或是解决经济的问题上能够有比较好的表现，或是比现在民进党有好的表现。并不等于民众支持国民党在两岸关系上、或两岸政策上往‘亲中’的方向走。”
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   尽管数据显示，台湾民众对于中国共产党的好感不增反减，但是，戴立安认为这场选举后，中国大陆借由释放农渔产品、增加大陆观光客的“红利”帮助蓝营执政县市首长推动经济发展，与民进党执政县市区隔，进而以地方包围中央，影响2020总统大选的意图明显。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   “经济一中”发威 蓝绿抢分食
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   日前大陆国台办发言人马晓光明确表示，“九二共识”是两岸关系和平发展的政治基础。在对两岸关系、两岸城市交流的性质有正确认知基础上，欢迎包含高雄在内的台湾更多县市参与两岸城市交流合作。他也明确指出，已有中国大陆的旅游业者组团赴高雄旅游，是一个好的开始。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   中国大陆释放的“利多”不仅是蓝营15个县市长想要雨露均沾，就连民进党执政的最大县桃园县长郑文灿都想来争食大陆观光客赴台的这块大饼，甚至释出与韩国瑜就任市长后的高雄市合作的意愿。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   郑文灿：“南进北出、北进南出都是一个好方案，桃园跟高雄可以合作，让观光客可以多元了解、体验台湾。”
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   就连邻近的屏东县民进党籍县长潘孟安也担心被磁吸边缘化，潘孟安也提出，将主动拜访韩国瑜表达区域治理能持续的期盼。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   曾经担任民进党法律顾问的庄胜荣律师，在一场座谈中指出，中国大陆重新夺回对台湾的掌控权，要松要紧完全操控在中国大陆之手。他认为，当务之急是台湾联合其它民主国家推动中国民主化。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   庄胜荣：“台湾人跟世界其他人一样追求自由、民主，也要money(钱)，这是世界共同的道理。在台湾很简单，我们最大的优势就是八个字“自由、民主、人权、法治”，这是对方没有的，我们用这(八字)来催化他。”
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   选后中国大陆借由“经济一中”手段，转化成“政治一中”企图相当明显，2020总统大选考验台湾人民的抉择。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
