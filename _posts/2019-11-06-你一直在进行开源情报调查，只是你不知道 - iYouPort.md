   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%ab%98%e7%ba%a7%e8%bf%90%e7%ae%97%e7%ac%a6%e8%be%85%e5%8a%a9%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%ef%bc%9a%e5%b7%a7%e7%94%a8%e6%90%9c%e7%b4%a2%e5%bc%95%e6%93%8e%e6%8c%96%e6%8e%98%e6%83%85%e6%8a%a5/" href="https://www.iyouport.org/%e9%ab%98%e7%ba%a7%e8%bf%90%e7%ae%97%e7%ac%a6%e8%be%85%e5%8a%a9%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%ef%bc%9a%e5%b7%a7%e7%94%a8%e6%90%9c%e7%b4%a2%e5%bc%95%e6%93%8e%e6%8c%96%e6%8e%98%e6%83%85%e6%8a%a5/" rel="noopener noreferrer" target="_blank">
    高级运算符辅助开源调查：巧用搜索引擎挖掘情报
   </a>
   》。
  </p>
  <p class="graf graf--p">
   对于不熟悉此问题的人，可以做一个最简短的解释：如果您在引号之间放置任何文字，例如，“我今天吃了一个金枪鱼三明治”，您可以找到完全匹配的内容。
  </p>
  <p class="graf graf--p">
   正如你将看到的那样，这只会返回完全匹配的结果，它会排除像“今天我吃了金枪鱼三明治”这样的短语，因为它不完全一样。
  </p>
  <p class="graf graf--p">
   但是当您搜索时发现返回的结果太多（太泛泛）时，这种方法就比较好用了。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6621 jetpack-lazy-image" data-lazy-sizes="(max-width: 683px) 100vw, 683px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/8-2.png?resize=683%2C203&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/8-2.png?w=683&amp;ssl=1 683w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/8-2.png?resize=300%2C89&amp;ssl=1 300w" data-recalc-dims="1" height="203" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/8-2.png?resize=683%2C203&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="683"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6621" data-recalc-dims="1" height="203" sizes="(max-width: 683px) 100vw, 683px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/8-2.png?resize=683%2C203&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/8-2.png?w=683&amp;ssl=1 683w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/8-2.png?resize=300%2C89&amp;ssl=1 300w" width="683"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   如您所见，上面的查询会产生太多结果。因此，尝试使用引号：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6622 jetpack-lazy-image" data-lazy-sizes="(max-width: 687px) 100vw, 687px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/9-2.png?resize=687%2C201&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/9-2.png?w=687&amp;ssl=1 687w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/9-2.png?resize=300%2C88&amp;ssl=1 300w" data-recalc-dims="1" height="201" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/9-2.png?resize=687%2C201&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="687"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6622" data-recalc-dims="1" height="201" sizes="(max-width: 687px) 100vw, 687px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/9-2.png?resize=687%2C201&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/9-2.png?w=687&amp;ssl=1 687w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/9-2.png?resize=300%2C88&amp;ssl=1 300w" width="687"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   你看，这样结果就好多了，只有198个。
  </p>
  <p class="graf graf--p">
   这里的问题在于句子太短而且不完整，所以你仍会得到一些无用的结果。
  </p>
  <p class="graf graf--p">
   因此，我们来尝试一种不同的方法。例如，搜索伊斯坦布尔高架桥历史。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6623 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/10-1.png?resize=1100%2C533&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/10-1.png?w=1333&amp;ssl=1 1333w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/10-1.png?resize=300%2C145&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/10-1.png?resize=768%2C372&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/10-1.png?resize=1024%2C496&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/10-1.png?resize=1100%2C533&amp;ssl=1 1100w" data-recalc-dims="1" height="533" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/10-1.png?resize=1100%2C533&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6623" data-recalc-dims="1" height="533" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/10-1.png?resize=1100%2C533&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/10-1.png?w=1333&amp;ssl=1 1333w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/10-1.png?resize=300%2C145&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/10-1.png?resize=768%2C372&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/10-1.png?resize=1024%2C496&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/10-1.png?resize=1100%2C533&amp;ssl=1 1100w" width="1100"/>

  <p class="graf graf--p">
   就是上面这样。依旧太多。
  </p>
  <p class="graf graf--p">
   那么添加 -aqueduct，减号运算符将过滤掉包含单词 aqueduct 的所有结果。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6624 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/11-3.png?resize=1100%2C510&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/11-3.png?w=1333&amp;ssl=1 1333w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/11-3.png?resize=300%2C139&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/11-3.png?resize=768%2C356&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/11-3.png?resize=1024%2C475&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/11-3.png?resize=1100%2C510&amp;ssl=1 1100w" data-recalc-dims="1" height="510" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/11-3.png?resize=1100%2C510&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   现在有一个看起来很像是这座桥的结构；其他一些返回了正确位置查询：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    istanbul arch bridge
   </li>
   <li class="graf graf--li">
    istanbul arch -university -mosque -aqueduct
   </li>
   <li class="graf graf--li">
    istanbul historical bridge -galata -bosphorus
   </li>
  </ul>
  <p class="graf graf--p">
   还有很多。这里只包括了英语的，但你也可以尝试不同的语言。
  </p>
  <p class="graf graf--p">
   现在，您可以使用 Yandex 进行反向图像搜索了，以便更好地了解这座桥。

    “Cumhuriyeti ve demokrasiyi seviyoruz.”（我们热爱民主和共和）
   </em>
  </p>
  <p class="graf graf--p">
   你还能查到，这行字在2011年已经被擦除了。

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    最后
   </strong>
  </p>
  <p class="graf graf--p">
   工具和反向图像搜索很有趣，但不得不说，Google搜索运算符非常重要。
  </p>
  <p class="graf graf--p">
   如果您还没有掌握它，现在就可以做到，在这里《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%ab%98%e7%ba%a7%e8%bf%90%e7%ae%97%e7%ac%a6%e8%be%85%e5%8a%a9%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%ef%bc%9a%e5%b7%a7%e7%94%a8%e6%90%9c%e7%b4%a2%e5%bc%95%e6%93%8e%e6%8c%96%e6%8e%98%e6%83%85%e6%8a%a5/" href="https://www.iyouport.org/%e9%ab%98%e7%ba%a7%e8%bf%90%e7%ae%97%e7%ac%a6%e8%be%85%e5%8a%a9%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%ef%bc%9a%e5%b7%a7%e7%94%a8%e6%90%9c%e7%b4%a2%e5%bc%95%e6%93%8e%e6%8c%96%e6%8e%98%e6%83%85%e6%8a%a5/" rel="noopener noreferrer" target="_blank">
    高级运算符辅助开源调查：巧用搜索引擎挖掘情报
   </a>
   》。
  </p>
  <p class="graf graf--p">
   您也许不需要掌握 Google dorking 的深度，但对于最基本的搜索运算符，了解它们是非常有好处的。
  </p>
  <p class="graf graf--p">
   尤其是在尝试地理定位时，反向图像搜索和运算符的组合相互补充。
  </p>
  <p class="graf graf--p">
   对于刚接触这项工作的人来说，您可以
