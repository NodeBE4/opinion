   举办全球新闻自由峰会的同时争取继续对媒体进行间谍活动的权力
  </h1>

 <div class="entry-content">
  <p class="graf graf--p">
   7月份，中国大张旗鼓地宣传其已经获得了“世界新闻教育大会”的主办权。
  </p>
  <p class="graf graf--p">
   表面上看这的确很讽刺：在新闻自由度垫底的国家之一举办这样的全球性会议，该国如何自圆其说？
  </p>
  <p class="graf graf--p">
   然而不幸的是，主办方国家的确可以同时做到主持全球媒体自由峰会和针对媒体进行间谍活动。
  </p>
  <p class="graf graf--p">
   英国就是个典型的例子 —— 该国是本届全球媒体自由峰会的主办方。
  </p>
  <figure aria-describedby="caption-attachment-6704" class="wp-caption aligncenter" id="attachment_6704" style="width: 1026px">
   <img alt="" class="wp-image-6704 size-full jetpack-lazy-image" data-lazy-sizes="(max-width: 1026px) 100vw, 1026px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/上.png?resize=1026%2C1112&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/上.png?w=1026&amp;ssl=1 1026w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/上.png?resize=277%2C300&amp;ssl=1 277w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/上.png?resize=768%2C832&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/上.png?resize=945%2C1024&amp;ssl=1 945w" data-recalc-dims="1" height="1112" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/上.png?resize=1026%2C1112&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1026"/>

   </figcaption>
  </figure>
  <p class="graf graf--p">
   外交大臣 Jeremy Hunt 7月10日在伦敦主持全球媒体自由会议。而与此同时，在距离此地 400 英里外 Strasbourg，英国政府的律师正在争取继续监视媒体的权力。
  </p>
  <p class="graf graf--p">
   公民组织调查新闻局向欧洲人权法院提起了诉讼，反对英国情报机构对新闻和公众的大规模监听、以及这种监视所带来的对媒体自由的严重冲击。
  </p>
  <p class="graf graf--p">
   该案件正在欧洲人权法院（ECHR）被听取，而与此同时，英国外交大臣正在主持一个全球性会议，来自世界各地的1,000名媒体和民间社会代表参会，会议声称旨在争取保护新闻自由和记者的信息来源的安全。
  </p>
  <p class="graf graf--p">
   当 Hunt 在伦敦发表演讲时，英国政府则在 Strasbourg 捍卫其大规模监控制度的合理性 —— 包括监视新闻传播 —— 声称监视是情报部门“保护”公众的工具包的重要组成部分。James Eadie 爵士领导这一辩论。
  </p>
  <p class="graf graf--p">
   Edward Snowden 在六年前就已经透露，包括 GCHQ 在内的情报机构如何存储和监控大量的数字通信。此后公民权利组织一直在努力反抗。
  </p>
  <p class="graf graf--p">
   这种监视可以覆盖大多数新闻传播，包括记者与信息来源之间的通信，因此是对新闻自由的严重侵犯。
  </p>
  <p class="graf graf--p">
   如果没有足够的保护，记者就无法与信息来源自由交流，信息来源也无法匿名。
  </p>
  <p class="graf graf--p">
   在提交给欧洲人权法院的案件中，公民组织调查新闻局认为政府的拦截是非法的，因为它无法保护“欧洲人权公约”第10条所详述的言论自由权所要求的新闻保密。
  </p>
  <p class="graf graf--p">
   去年，法院的裁定支持调查新闻局的观点，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.thebureauinvestigates.com/stories/2018-09-13/bureau-wins-case-to-defend-press-freedom-at-the-european-court-of-human-rights" href="https://www.thebureauinvestigates.com/stories/2018-09-13/bureau-wins-case-to-defend-press-freedom-at-the-european-court-of-human-rights" rel="noopener noreferrer" target="_blank">
    裁定
   </a>
   政府的监控计划是非法的。这是一个伟大的胜利。
  </p>
  <p class="graf graf--p">
   法官在其裁决中指出，“言论自由是民主社会的重要基础之一，并且向新闻界提供的保障是重中之重。”
  </p>
  <p class="graf graf--p">
   欧洲人权法院还指出，GCHQ 的大规模数据收集监视也侵犯了普通公民的基本人权。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6705 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/中.png?resize=1100%2C949&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/中.png?w=1268&amp;ssl=1 1268w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/中.png?resize=300%2C259&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/中.png?resize=768%2C663&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/中.png?resize=1024%2C883&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/中.png?resize=1100%2C949&amp;ssl=1 1100w" data-recalc-dims="1" height="949" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/中.png?resize=1100%2C949&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6705" data-recalc-dims="1" height="949" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/中.png?resize=1100%2C949&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/中.png?w=1268&amp;ssl=1 1268w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/中.png?resize=300%2C259&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/中.png?resize=768%2C663&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/中.png?resize=1024%2C883&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/中.png?resize=1100%2C949&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   根据现行制度，在收集记者的通信之前无需获得法官的事先授权。新闻调查局认为，应该有明确的规则和问责制来管理政府对他们收集到的任何信息的使用。
  </p>
  <p class="graf graf--p">
   这一点尤其重要，因为政权对透明度革命施加了严重的寒蝉效应，举报人是为公众利益服务的重要揭露的关键来源。
  </p>
  <p class="graf graf--p">
   调查新闻局主编 Rachel Oldroyd 说：“这可能是保护新闻传播和保护我们的消息来源在过去20年中最重要的案例。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “作为记者，我们必须能够自由地与消息来源交谈，而不必担心受到监视。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “英国政府对我们所有通信的大规模收集和存储系统严重威胁着我们这一专业的基石，并严重侵犯了新闻自由。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “具有讽刺意味的是，在欧洲最高法院对英国政府监视行为提起诉讼的那一天，同一政府正在举办一场关于新闻自由的全球峰会。”
  </p>
  <p class="graf graf--p">
   该案件不仅对在英国工作的记者有重要影响，而且对于未来几年在整个欧洲的新闻业都有重要意义。
  </p>
  <p class="graf graf--p">
   最终，这个案例是关于能否确保对斯诺登所揭示的政府监听水平采取措施，并确保在技术发展和大规模监视的时代这些措施足够抵御监视。
  </p>
  <p class="graf graf--p">
   代表新闻调查局的 Leigh Day 律师事务所律师 Rosa Curling 说：“新闻权是新闻自由的一个关键方面。民主社会运作的基础是记者与其信息来源进行私密和自由交流的能力。绝不允许政府窥探这些通讯。” ⚪️
  </p>
