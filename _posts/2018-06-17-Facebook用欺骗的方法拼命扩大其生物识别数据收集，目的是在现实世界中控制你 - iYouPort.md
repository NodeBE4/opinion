   Facebook用欺骗的方法拼命扩大其生物识别数据收集，目的是在现实世界中控制你

 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      该技术可用于在不知情或未经同意的情况下远程识别人员，它就是实现大规模监视系统的基础设施
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p graf-after--figure" id="158c">
   当 Facebook 今年在欧盟推出面部识别工具时，它使用的是撒谎的方式，称“该技术在帮助人们保护他们的在线身份“。“脸部识别技术可以让我们帮助保护你免受陌生人使用你的照片冒充你的风险，”Facebook 这样告诉其在欧洲的用户。
  </p>
  <p class="graf graf--p graf-after--p" id="bdda">
   显然社交网络正在采取一个危险的举动。六年前，在那里的监管机构提出有关其面部识别同意系统所存在的问题之后，Facebook 已经停止了它欧洲的技术。然而现在，Facebook 正在卷土重来，重新引入该服务，作为其在欧洲更新其用户许可流程的一部分。
  </p>
  <p class="graf graf--p graf-after--p" id="f993">
   然而，在美国和欧洲的数据挖掘实践受到严格审查之际，Facebook 对该技术的积极推动已经导致了巨大的声誉风险。已经有十几个隐私组织和消费者团体，以及至少一些官员，认为该公司使用面部识别侵犯了人们的隐私权，因为他们没有获得适当的用户同意。
  </p>
  <p class="graf graf--p graf-after--p" id="4cbc">
   这些投诉加剧了硅谷巨头在处理用户个人信息方面遭受的批评。目前，一些美国政府机构正在调查 Facebook 对政治咨询公司 Cambridge Analytica 收集用户数据的回应。
  </p>
  <p class="graf graf--p graf-after--p" id="3b4c">
   Facebook 对面部识别的推动也使该公司被置于了关于如何处理强大技术这一更广泛和激烈辩论的中心。该技术可用于在不知情或未经同意的情况下远程识别人员。
   <strong class="markup--strong markup--p-strong">
    公民自由专家警告说，它就是实现大规模监视系统的基础设施
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="798a">
   <em class="markup--em markup--p-em">
    延伸阅读：《
   </em>
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E5%A6%82%E4%BD%95%E5%AF%B9%E6%8A%97%E9%9D%A2%E9%83%A8%E8%AF%86%E5%88%AB%E6%8A%80%E6%9C%AF-d3a6238ebc71" href="https://medium.com/@iyouport/%E5%A6%82%E4%BD%95%E5%AF%B9%E6%8A%97%E9%9D%A2%E9%83%A8%E8%AF%86%E5%88%AB%E6%8A%80%E6%9C%AF-d3a6238ebc71" rel="noopener noreferrer" target="_blank">
    <em class="markup--em markup--p-em">
     如何对抗面部识别技术？包括被 Facebook 强加的面部识别
    </em>
   </a>
   <em class="markup--em markup--p-em">
    》
   </em>
  </p>
  <p class="graf graf--p graf-after--p" id="5045">
   面部识别的工作原理是在照片或视频中扫描未命名人物的面部，然后将他们的面部图案的代码与指定人员的数据库中的代码进行匹配。 虽然 Facebook 表示，用户可控制面部识别过程，但显然这又是撒谎。
  </p>
  <p class="graf graf--p graf-after--p" id="45ce">
   批评人士表示，人们实际上并不能控制这项技术 — 因为即使面部识别设置被关闭，Facebook 也会在照片中扫描他们的脸部。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="cb7d">
   “该公司试图通过让 Facebook 看起来像好人的方式来解释他们的做法，但他们完全回避了他们正在扫描每张照片的事实，”数字版权组织电子前哨基金会的高级律师 Jennifer Lynch 说。
  </p>
  <p class="graf graf--p graf-after--p" id="15fc">
   在欧盟，GDPR 这项新的数据保护法现在要求公司在收集面部数据等敏感信息之前获得明确的的“自由”的同意。一些批评者，包括最初提出新法律的前政府官员，一致认为 Facebook 以不正当的方式诱导用户的意见，只为了合法化他们的面部识别。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="cb5b">
   “Facebook 在某种程度上威胁用户，如果不接受面部识别，用户将面临危险。这是在操纵同意，完全违反欧洲法律”，欧洲委员会前司法委员 Viviane Reding 说，他现在是欧洲议会议员。
  </p>
  <p class="graf graf--p graf-after--p" id="fe23">
   <em class="markup--em markup--p-em">
    延伸阅读：《
   </em>
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E4%B8%BA%E4%BB%80%E4%B9%88%E8%AF%B4%E6%AC%A7%E6%B4%B2%E6%96%B0%E6%B3%95%E8%A7%84%E5%8F%8D%E8%80%8C%E4%BC%9A%E5%8A%A0%E5%BC%BA-facebook-%E5%92%8Cgoogle-%E7%9A%84%E9%9C%B8%E6%9D%83-795e628592d5" href="https://medium.com/@iyouport/%E4%B8%BA%E4%BB%80%E4%B9%88%E8%AF%B4%E6%AC%A7%E6%B4%B2%E6%96%B0%E6%B3%95%E8%A7%84%E5%8F%8D%E8%80%8C%E4%BC%9A%E5%8A%A0%E5%BC%BA-facebook-%E5%92%8Cgoogle-%E7%9A%84%E9%9C%B8%E6%9D%83-795e628592d5" rel="noopener noreferrer" target="_blank">
    <em class="markup--em markup--p-em">
     为什么说欧洲新法规反而会加强 Facebook 和 Google 的霸权？
    </em>
   </a>
   <em class="markup--em markup--p-em">
    》
   </em>
  </p>
  <p class="graf graf--p graf-after--p" id="8f34">
   欧洲监管机构也对 Facebook 的面部识别实践表示了担忧。在 Facebook 的国际总部爱尔兰，数据保护委员会的一位发言人表示，监管机构“已就此技术向 Facebook 提出了一些具体问题。正在评估 Facebook 的回应“。
  </p>
  <p class="graf graf--p graf-after--p" id="8496">
   在美国，Facebook 正在打击伊利诺斯州居民提起的诉讼，诉讼指该公司的面部识别行为违反了州隐私法。这是一起集体诉讼案件，损害赔偿金可能高达数十亿美元。 然而 5月，一家上诉法院批准了 Facebook 的要求延迟审判并审查集体认证令。
  </p>
  <p class="graf graf--p graf-after--p" id="17ff">
   另外，隐私和消费者团体在4月份向联邦贸易委员会提出申诉，称 Facebook 增加了面部识别服务，而未经人们事先同意，就默认开启。这些团体认为，Facebook 违反了 2011 年的同意法令，该法令禁止欺骗隐私的做法。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-1005 jetpack-lazy-image" data-lazy-sizes="(max-width: 1000px) 100vw, 1000px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-1.jpeg?resize=1000%2C2048&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-1.jpeg?w=1000&amp;ssl=1 1000w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-1.jpeg?resize=146%2C300&amp;ssl=1 146w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-1.jpeg?resize=768%2C1573&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-1.jpeg?resize=500%2C1024&amp;ssl=1 500w" data-recalc-dims="1" height="2048" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-1.jpeg?resize=1000%2C2048&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1000"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-1005" data-recalc-dims="1" height="2048" sizes="(max-width: 1000px) 100vw, 1000px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-1.jpeg?resize=1000%2C2048&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-1.jpeg?w=1000&amp;ssl=1 1000w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-1.jpeg?resize=146%2C300&amp;ssl=1 146w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-1.jpeg?resize=768%2C1573&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-1.jpeg?resize=500%2C1024&amp;ssl=1 500w" width="1000"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="e296">
   投诉说：“Facebook 经常做出虚假陈述，诱使消费者采用更广泛和更普遍的面部识别技术。”
  </p>
  <p class="graf graf--p graf-after--p" id="b0de">
   更要命的是，Facebook 不是唯一拥抱面部识别技术的科技巨头。在过去几年中，亚马逊、苹果、Facebook、谷歌和微软都提交了面部识别专利申请。注意，还是可怕的专利！
  </p>
  <p class="graf graf--p graf-after--p" id="a022">
   5月，公民自由团体批评亚马逊向警察部门推销名为 Rekognition 的面部技术。
  </p>
  <p class="graf graf--p graf-after--p" id="ee62">
   批评人士指出，Facebook 在面部识别技术方面处于领先地位，部分原因是由于该公司使用了欺骗的方法，将该技术作为了用户的默认开启选项。 2010 年，该公司推出了照片标签功能，该功能使用面部匹配软件来建议用户给照片中的人添加名字。
  </p>
  <p class="graf graf--p graf-after--p" id="4012">
   隐私专家指出，Facebook 既没有获得用户对该技术的选择性同意，也没有明确告知他们该公司可以从扫描他们的照片中受益。
  </p>
  <p class="graf graf--p graf-after--p" id="a75e">
   面部识别公司 Kairos 的首席执行官 Brian Brackeen 表示，“当 Facebook 的标签建议问你’这是吉尔吗？’时，你往往没有意识到自己正在帮助改善 Facebook 的人脸识别算法。”
  </p>
  <p class="graf graf--p graf-after--p" id="613c">
   ⚠️ 这个庞大的面部数据库使得 Facebook 能够迅速开发出世界上最强大的商业面部识别引擎之一。 2014年，Facebook 的研究人员表示，他们已经在迄今为止最大的面部数据集上训练了识别软件，这是一个多达 400 万张面部图像的数据集 — — 全部来自于被该公司欺骗了的用户。
  </p>
  <p class="graf graf--p graf-after--p" id="f422">
   <strong class="markup--strong markup--p-strong">
    然而这只是一个开始。Facebook 的面部识别专利申请的目标是，利用该技术在现实世界中跟踪他们的用户
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="b7c0">
   去年11月发布的一项专利申请描述了一种系统，
   <strong class="markup--strong markup--p-strong">
    该系统可以检测商店内的消费者并将这些购物者的面孔与他们的社交网络数据相匹配，然后分析他们的朋友的特征和其他细节，使用这些信息来确定每个购物者的“信誉等级”
   </strong>
   。据文件称，被认为“值得信赖”的消费者可能有资格获得特殊待遇，例如在锁定的陈列柜中获取商品。
  </p>
  <p class="graf graf--p graf-after--p" id="b28e">
   另一份 Facebook 的专利文件描述了收银台附近的摄像头如何捕捉购物者的面孔，将其与社交网络配置文件相匹配，然后将购买确认的信息发送到他们的手机上。
  </p>
  <p class="graf graf--p graf-after--p" id="84d0">
   隐私组织 EPIC 称专利文件显示 Facebook 正在如何从用户的脸上赚钱：“Facebook 的专利申请证明了公司的主要商业目的，
   <strong class="markup--strong markup--p-strong">
    即扩大其生物识别数据收集，以及它在不久的将来设想的面部识别技术的普遍使用
   </strong>
   ”。
  </p>
