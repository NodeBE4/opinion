  <p class="graf graf--p">
   有关以色列监视技术公司被强权雇用以迫使持不同政见者保存沉默的最新报道涉及沙特。如今臭名昭著的 Pegasus 间谍软件的发明者NSO 被沙特雇用来入侵包括 Jamal Khashoggi 在内的著名批评家的手机。
  </p>
  <p class="graf graf--p">
   沙特依靠一家以色列科技公司来监视它认为有威胁的任何东西，这不足为奇。如今，整个中东国家都依赖NSO集团等以色列间谍公司来确保其自身权力的稳固，这反映了
   <strong class="markup--strong markup--p-strong">
    以色列与这些专制国家之间不断发展的维稳伙伴关系
   </strong>
   。
  </p>
  <p class="graf graf--p">
   全世界的政权都很喜欢雇用的另一家以色列公司是 Black Cube，其服务和产品包括通过挖掘目标人过去曾经发生过的阴暗面（软肋）而对目标人施加威胁以令其保持沉默。
  </p>
  <p class="graf graf--p">
   随着以色列公司监视技术的发展，
   <strong class="markup--strong markup--p-strong">
    它对于那些希望监视所有异议和反对派日常活动的政府非常有用
   </strong>
   ：记者、激进主义者、博客作者、艺术家 —— 他们成为攻击的主要目标，他们的手机被入侵，他们的所有私密谈话被记录，他们的所有动作每时每刻都在被观察。
  </p>

   <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.ca/2018/09/hide-and-seek-tracking-nso-groups-pegasus-spyware-to-operations-in-45-countries/" href="https://citizenlab.ca/2018/09/hide-and-seek-tracking-nso-groups-pegasus-spyware-to-operations-in-45-countries/" rel="noopener noreferrer" target="_blank">

   ，以色列监视技术公司 NSO Group 已被沙特聘请，以在针对 Jamal Khashoggi 于10月被谋杀的计划之前追踪和监视他。
  </p>
  <p class="graf graf--p">
   这一重磅炸弹报告发出后，一些沙特持不同政见者发现他们自己的手机也被NSO集团的 Pegasus 间谍软件所感染。
  </p>
  <p class="graf graf--p">
   位于伦敦的著名的沙特激进主义者 Yahya Assiri 和 YouTube 喜剧讽刺作家 Ghanem Almasarir 都成为该间谍软件的攻击目标；居住在蒙特利尔的沙特异议人士 Omar Abdulaziz 告诉 Vice，他也遭到了黑客攻击。
  </p>
  <ul class="postList">
   <li class="graf graf--li">

     列表- 4 “以色列和NSO”

  <p class="graf graf--p">
   Pegasus 允许监视者仅知道手机的SIM卡号码就可以实现入侵，从而为黑客提供了手机上的各种私密信息，包括文本、通话、WhatsApp消息、照片、电子邮件、联系人和其他数据。它还使黑客能够利用麦克风窃听电话、并通过摄像头随时监视。
  </p>
  <p class="graf graf--p">
   根据 Haaretz 对 NSO 集团与沙特的交易的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.timesofisrael.com/israeli-hacking-firm-nso-group-offered-saudis-cellphone-spy-tools-report/" href="https://www.timesofisrael.com/israeli-hacking-firm-nso-group-offered-saudis-cellphone-spy-tools-report/" rel="noopener noreferrer" target="_blank">
    调查
   </a>
   ，沙特情报部门前负责人 Turki al-Faisal 的同伙 Abdullah al-Malihi 和本萨勒曼的亲信 Nasser al-Qahtani 与NSO代表会面，并同意
   <strong class="markup--strong markup--p-strong">
    以高达2.08亿美元的价格印进 Pegasus 间谍软件。
   </strong>
  </p>
  <p class="graf graf--p">
   沙特并不是唯一的客户。 Citizen Lab 的报告查找了 Pegasus 间谍软件使用的证据，结果发现，
   <strong class="markup--strong markup--p-strong">
    在几十个国家中
   </strong>
   ，都有间接证据表明国家政府或实体购买了这种用于监视目的的间谍软件。
  </p>
