  </p>
  <p class="graf graf--p">
   去年底出现了创纪录的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.scmp.com/tech/start-ups/article/3038255/shenzhen-tech-fair-window-how-chinese-firms-are-reducing-reliance-us" href="https://www.scmp.com/tech/start-ups/article/3038255/shenzhen-tech-fair-window-how-chinese-firms-are-reducing-reliance-us" rel="noopener noreferrer" target="_blank">
    44个
   </a>
   代表团参加在深圳举行的中国高科技博览会，这些代表团来自俄罗斯，伊朗，匈牙利，奥地利，瑞典等国，表面上看似乎在弥补缺席的美国公司。
  </p>
  <p class="graf graf--p">
   展会上唯一的美国公司来自休斯敦，该公司负责销售羊毛产品。
  </p>
  <p class="graf graf--p">
   在美国政府将中国公司列入黑名单后，两国之间的贸易紧张局势日益加剧，因此美国公司现在在与外国技术公司进行业务交易时持谨慎态度。
  </p>
  <p class="graf graf--p">
   美国政府禁止购买中国面部识别和生物识别监视公司 Megvii、华为、海康威视和 SenseTime 等公司的产品。然而
   <strong class="markup--strong markup--p-strong">
    欧洲公司正在考虑对中国半导体、5G和AI技术进行投资。
   </strong>
  </p>
  <p class="graf graf--p">
   华为与中国云提供商金蝶合作推出了云服务。去年，两家公司签署了合作伙伴关系，随后金蝶从亚马逊的AWS转移到华为的云，称 “为减少对美国供应商的依赖”。
  </p>
  <p class="graf graf--p">
   参加深圳科技博览会的嘉宾都经过了面部识别筛查。
  </p>
  <p class="graf graf--p">
   Phoenix AI 首席执行官兼创始人 Laurent Renard 表示
   <strong class="markup--strong markup--p-strong">
    ，比利时实际上禁止使用面部识别技术，但这促使了本地公司开发针对中国市场的面部识别解决方案。
   </strong>
  </p>
  <p class="graf graf--p">
   如此可见，一国阻止面部识别技术可能是无用的，甚至该国的开发者会更加主动地为其他没有禁止该技术的国家提供资源。
  </p>
  <p class="graf graf--p">
   这很严重。这意味着我们
   <strong class="markup--strong markup--p-strong">
    需要更多明确的法律对出口监视技术的限制
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《报告：
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%E5%9C%A8%E4%B8%80%E5%85%9A%E4%B8%93%E5%88%B6%E7%9A%84%E5%9B%BD%E5%AE%B6%E9%87%8C%EF%BC%8C%E5%87%A0%E4%B9%8E%E6%B2%A1%E6%9C%89%E6%9C%BA%E4%BC%9A%E5%AF%B9%E9%9D%A2%E9%83%A8%E8%AF%86%E5%88%AB%E7%B3%BB/" href="https://www.iyouport.org/%E5%9C%A8%E4%B8%80%E5%85%9A%E4%B8%93%E5%88%B6%E7%9A%84%E5%9B%BD%E5%AE%B6%E9%87%8C%EF%BC%8C%E5%87%A0%E4%B9%8E%E6%B2%A1%E6%9C%89%E6%9C%BA%E4%BC%9A%E5%AF%B9%E9%9D%A2%E9%83%A8%E8%AF%86%E5%88%AB%E7%B3%BB/" rel="noopener noreferrer" target="_blank">
     是谁在提供先进的监控技术协助压迫性政府对人权的侵犯？
    </a>
    》
   </li>
   <li class="graf graf--li">
    曾经一直以来《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%b8%ad%e5%9b%bd%e7%9a%84%e5%8f%8d%e4%b9%8c%e6%89%98%e9%82%a6%e7%9b%91%e8%a7%86%e5%9b%bd%e5%ae%b6%e6%98%af%e7%be%8e%e5%9b%bd%e5%88%b6%e9%80%a0%e7%9a%84/" href="https://www.iyouport.org/%e4%b8%ad%e5%9b%bd%e7%9a%84%e5%8f%8d%e4%b9%8c%e6%89%98%e9%82%a6%e7%9b%91%e8%a7%86%e5%9b%bd%e5%ae%b6%e6%98%af%e7%be%8e%e5%9b%bd%e5%88%b6%e9%80%a0%e7%9a%84/" rel="noopener noreferrer" target="_blank">
     中国的反乌托邦监视国家是美国制造的
    </a>
    》，此后可能会变成欧洲制造？
    <strong class="markup--strong markup--li-strong">
     不能让这种事继续发生
    </strong>
    。
   </li>
   <li class="graf graf--li">
    在IYP的专栏 《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/category/%e4%bd%9c%e6%81%b6%e8%80%85%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%b8%ae%e4%b8%ad%e5%9b%bd/" href="https://www.iyouport.org/category/%e4%bd%9c%e6%81%b6%e8%80%85%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%b8%ae%e4%b8%ad%e5%9b%bd/" rel="noopener noreferrer" target="_blank">
     作恶者：谁在帮中国
    </a>
    》看到更多揭露。
   </li>
  </ul>
  <p class="graf graf--p">
   在这次会议期间，
   <strong class="markup--strong markup--p-strong">
    欧洲代表推动加强中欧合作，特别是在数字支付和电子商务等领域。
   </strong>
  </p>
  <p class="graf graf--p">
   参会的意大利前经济发展部副部长是意大利参与中国“一带一路”基础设施计划的最高倡导者；来自医疗设备制造商的许多欧洲代表认为，监管和文化障碍使他们无法在该国扩展业务（他们就跑去中国 “开展业务” 了）。
  </p>
  <p class="graf graf--p">
   根据 France24 的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.france24.com/en/20191124-watch-this-china-surveillance-tech-seeks-to-go-global" href="https://www.france24.com/en/20191124-watch-this-china-surveillance-tech-seeks-to-go-global" rel="noopener noreferrer" target="_blank">
    报道
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    中国公司 —— 特别是一些专注于开发监视系统的公司，在巴黎举行的国土安全贸易盛会 Milipol 上亮相。在1,100家公司中，有89家来自中国
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是，它没有展示武器和弹药，而是侧重于面部识别技术、头盔、防弹背心和战术服装。
  </p>
  <p class="graf graf--p">
   Hytera Communications Corporation
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hytera.com/en/" href="https://www.hytera.com/en/" rel="noopener noreferrer" target="_blank">
    展示了
   </a>
   一种通信设备，该设备将用身体摄像机收集的图像发送到智能手机，这是对警察来说理想的解决方案。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     关于身体相机，这是美国最先引领的监视方案
    </strong>
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%ba%ab%e4%bd%93%e7%9b%b8%e6%9c%ba%e9%80%82%e5%be%97%e5%85%b6%e5%8f%8d%e5%90%97%ef%bc%9f%e4%b8%ad%e5%9b%bd%e6%ad%a3%e5%9c%a8%e5%bc%80%e5%a7%8b%e6%b5%81%e8%a1%8c%e8%bf%99%e7%a7%8d%e4%b8%9c%e8%a5%bf/" href="https://www.iyouport.org/%e8%ba%ab%e4%bd%93%e7%9b%b8%e6%9c%ba%e9%80%82%e5%be%97%e5%85%b6%e5%8f%8d%e5%90%97%ef%bc%9f%e4%b8%ad%e5%9b%bd%e6%ad%a3%e5%9c%a8%e5%bc%80%e5%a7%8b%e6%b5%81%e8%a1%8c%e8%bf%99%e7%a7%8d%e4%b8%9c%e8%a5%bf/" rel="noopener noreferrer" target="_blank">
     身体相机适得其反吗？中国正在开始流行这种东西
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   公司代表 Sylvain Shuang 说：“每个警官都可以在智能手机上直接看到战术状况。” 他补充说，
   <strong class="markup--strong markup--p-strong">
    这些图像可以与监控摄像头的图像匹配，并且它们集成了面部识别技术。
   </strong>
  </p>
  <p class="graf graf--p">
   尽管该公司无法获得与美国的合同，但其40％的业务是在中国境外的 —— 英国、加拿大和西班牙。
  </p>
  <p class="graf graf--p">
   视频监控公司海康威视（Hikvision）在美国也遇到类似的情况。该公司2018年与尼斯市合作为其配备了视频监控系统，但去年在Milipol却没有展位。
  </p>
  <p class="graf graf--p">
   参加此次活动的另一家中国公司是 Norinco，他们展示了其面部识别技术。
  </p>
  <p class="graf graf--p">
   一位公司代表说：“在中国，面部识别不是一项新技术，在大多数公共场所都有此监视系统”。他进一步补充说，由于隐私问题，该技术在欧洲很难销售。
  </p>
  <p class="graf graf--p">
   这份报道提醒的重点问题是：监视技术依旧是财源，这导致了在本国无法销售的监视工具更多倾向于销往隐私权法律不完善的专制国家。专制国家的反监视活动家可能需要更多的努力才能阻止这一事态的发展。⚪️
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.biometricupdate.com/201911/european-interest-in-facial-recognition-and-other-chinese-tech-balances-u-s-absence-at-shenzhen-event" href="https://www.biometricupdate.com/201911/european-interest-in-facial-recognition-and-other-chinese-tech-balances-u-s-absence-at-shenzhen-event" rel="noopener noreferrer" target="_blank">
    European interest in facial recognition and other Chinese tech balances U.S. absence at Shenzhen event
   </a>
  </p>
