  <p class="graf graf--p">
   2018年初，德国杜伊斯堡市与中国电信巨头华为合作，将杜伊斯堡重塑为所谓的 “智能城市”；这一理念被华为倡导为治理的未来。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “双方都同意通过实施新的信息和通信技术（ICT）在几年内使杜伊斯堡成为西欧的创新和数字化样板城市，”有关此事的新闻稿说。
  </p>
  <p class="graf graf--p">
   杜伊斯堡和华为于去年6月6日同意开始实施 “Rhine Cloud”，这为将来的发展奠定了基础。
  </p>
  <p class="graf graf--p">
   Rhine Cloud 是统一的 “智能城市” 和公共服务云平台的另一种名称，它将原本分散的城市维护和治理形式统一到一个单一的数据库中。也就是高度集中化。
  </p>
  <p class="graf graf--p">

   </a>

  </p>
  <p class="graf graf--p">
   华为西欧区总裁 Vincent Pang 说：“华为致力于与市政府合作，使现代城市变得更聪明、更易于居住。
  </p>
  <p class="graf graf--p">
   但是，如果杜伊斯堡的新系统随着时间的流逝而发展成为现在的可销售产品，则必须借助某个地方进行发展。
  </p>

   </strong>
   。

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    完善 “智能城市”
   </strong>
  </h3>
  <p class="graf graf--p">
   为了优化和组织中国在新疆地区的监视国家，部分程度上它依赖于华为。
  </p>
  <p class="graf graf--p">
   去年5月初，华为开始在新疆首府乌鲁木齐市建立 “智能安全产业” 创新实验室。
  </p>
  <p class="graf graf--p">
   该实验室的目的是成为一个情报中心，在该中心中，可以为所谓的 “智能城市” 组织和分析从监视工作中收集的所有数据。
  </p>
  <p class="graf graf--p">
   中国少数民族问题专家 Adrian Zenz 指出，华为 “智能城市” 的部分产品已开始出口到德国杜伊斯堡。
  </p>
  <p class="graf graf--p">
   而且不止杜伊斯堡。根据华为的网站，Rhine Cloud 等智能城市技术
   <strong class="markup--strong markup--p-strong">
    已经出口到120多个城市！
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    调查报告《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%9b%be%e8%a7%a3%e4%b8%ad%e5%9b%bd%e7%a7%91%e6%8a%80%e5%b7%a8%e5%a4%b4-%e5%8d%8a%e5%a3%81%e9%ab%98%e5%a0%a1%e5%a5%87%e4%ba%ba/" href="https://www.iyouport.org/%e5%9b%be%e8%a7%a3%e4%b8%ad%e5%9b%bd%e7%a7%91%e6%8a%80%e5%b7%a8%e5%a4%b4-%e5%8d%8a%e5%a3%81%e9%ab%98%e5%a0%a1%e5%a5%87%e4%ba%ba/" rel="noopener noreferrer" target="_blank">
     图解中国科技巨头 — 半壁“高堡奇人”
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   这使人权专家感到担忧。 “今天在新疆开发和部署的监视技术将很快在全球销售和推广，” 大赦国际的 William Nee 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “智能城市” 概念中最具争议的元素是哪种数据被认为与有效治理相关。
  </p>
  <p class="graf graf--p">
   数据库中包含城市维护的常规功能，例如 路灯触发器，该功能已成为自动化系统的一部分。但是还包括
   <strong class="markup--strong markup--p-strong">
    维稳功能，为政府提供了一套新的先进工具，可以对他们认为不可信、可疑或激进的任何人进行监视
   </strong>
   。
  </p>
  <p class="graf graf--p">
   尽管国内法律可以减少潜在的侵犯人权行为或隐私问题，但是，新疆的 “智能城市” 技术试验场几乎没有限制，
   <strong class="markup--strong markup--p-strong">
    这使华为等公司可以自由地开发最大程度的侵入性监视技术，而不会产生任何影响
   </strong>
   。
  </p>
  <p class="graf graf--p">
   根据 Zenz 的说法，这给中国公司提供了相对于欧洲或美国竞争对手公司的优势地位，后者不得不在公民反抗和隐私监督机构的严格约束下操作和测试其技术。而中国公司根本不会受到这种挑战。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12300 jetpack-lazy-image" data-lazy-sizes="(max-width: 487px) 100vw, 487px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-3.jpg?resize=487%2C960&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-3.jpg?w=487&amp;ssl=1 487w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-3.jpg?resize=152%2C300&amp;ssl=1 152w" data-recalc-dims="1" height="960" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-3.jpg?resize=487%2C960&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="487"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12300" data-recalc-dims="1" height="960" sizes="(max-width: 487px) 100vw, 487px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-3.jpg?resize=487%2C960&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-3.jpg?w=487&amp;ssl=1 487w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-3.jpg?resize=152%2C300&amp;ssl=1 152w" width="487"/>
   </noscript>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “智能城市” 被用来对付本国公民的最明显例子之一就是配备了先进的面部识别技术的闭路电视摄像机的武器化。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%ad%a6%e5%99%a8%e5%8c%96%e5%9f%8e%e5%b8%82/" href="https://www.iyouport.org/%e6%ad%a6%e5%99%a8%e5%8c%96%e5%9f%8e%e5%b8%82/" rel="noopener noreferrer" target="_blank">
     武器化城市
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   在新疆，数百万个放置在战略要地的闭路电视摄像机不断跟踪和识别居民，确定他们的日常路线，并为国家政权提供了一种恒定的方式来监视所有人的活动和行为。
  </p>
  <p class="graf graf--p">
   被捕获的可疑活动（例如祈祷）可能会被标记，并由中国情报人员进行检查。被怀疑的人可能会在“再教育”营中被长期拘留。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    特别推荐这本书《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bb%8a%e5%a4%a9%e7%9a%84%e9%9b%86%e4%b8%ad%e8%90%a5%e6%9c%89%e5%be%88%e5%a4%9a%e5%90%8d%e5%ad%97%ef%bc%8c%e4%bd%86%e5%ae%83%e4%bb%ac%e4%be%9d%e6%97%a7%e6%98%af%e9%9b%86%e4%b8%ad%e8%90%a5/" href="https://www.iyouport.org/%e4%bb%8a%e5%a4%a9%e7%9a%84%e9%9b%86%e4%b8%ad%e8%90%a5%e6%9c%89%e5%be%88%e5%a4%9a%e5%90%8d%e5%ad%97%ef%bc%8c%e4%bd%86%e5%ae%83%e4%bb%ac%e4%be%9d%e6%97%a7%e6%98%af%e9%9b%86%e4%b8%ad%e8%90%a5/" rel="noopener noreferrer" target="_blank">
     今天的集中营有很多名字，但它们依旧是集中营
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   如今，这种技术正在进入欧洲城市，这可能使侵犯人权的行为变得更加容易 —— 由大数据驱动。
  </p>
  <figure aria-describedby="caption-attachment-12303" class="wp-caption aligncenter" id="attachment_12303" style="width: 655px">
   <img alt="" class="size-full wp-image-12303 jetpack-lazy-image" data-lazy-sizes="(max-width: 655px) 100vw, 655px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/5.png?resize=655%2C430&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/5.png?w=655&amp;ssl=1 655w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/5.png?resize=300%2C197&amp;ssl=1 300w" data-recalc-dims="1" height="430" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/5.png?resize=655%2C430&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="655"/>
   <noscript>
    <img alt="" class="size-full wp-image-12303" data-recalc-dims="1" height="430" sizes="(max-width: 655px) 100vw, 655px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/5.png?resize=655%2C430&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/5.png?w=655&amp;ssl=1 655w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/5.png?resize=300%2C197&amp;ssl=1 300w" width="655"/>
   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-12303">
    (Rami Khoury/Al Bawaba)
   </figcaption>
  </figure>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    新的圆形监狱
   </strong>
  </h3>
  <p class="graf graf--p">
   圆形监狱的概念比 “大数据” 和所谓的 “智能城市” 的年代早了数百年。尽管如此，这是一个非常相关的类比。
  </p>
  <p class="graf graf--p">
   在这里的人们只能假定自己一直在受到监视，他们只能通过自我审查令监视者满意。圆形监狱让所有人主动为监视者服务。
  </p>
  <p class="graf graf--p">
   Simon Zhen 等法律分析师认为，正在出口的现代监视国家就是新的圆形监狱。“并不是说政府实际上一直在关注并控制着你。但是你已经无法做任何事。如果你组织一场大规模的反抗，就很容易陷入困境。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “再比如，没有什么可以阻止您在流行的社交媒体网站上发布有关天安门事件或法轮功的文章，但是一旦这样做，将有真实而直接的后果，因为政府立刻就能找到你。”
  </p>
  <p class="graf graf--p">
   华为总裁称：“这个智能城市的神经系统包括一个 “大脑” [控制中心]和 “周围神经” [网络和传感器]，收集有关城市状况的实时信息，传输数据，使 “大脑” 能够分析并做出明智的决定，提供反馈命令，并最终执行行动。这在物理世界和数字世界之间建立了无缝链接。利用云计算、物联网和人工智能等领先的新型 ICT 技术，华为致力于打造强大的神经系统，为智能城市提供动力……”
  </p>
  <p class="graf graf--p">
   随处可见的控制中心以及将城市的各个部分连接到中央数据系统的外围神经，都具有很大的吸引力，因为它是可销售的。
  </p>
  <p class="graf graf--p">
   尽管杜伊斯堡本身的运营成本可能会下降，但其潜在成本仍然存在并且还会增加：其 “智能城市” 使用的技术已经对中国境内数百万人实施了侵犯人权的监视，并且是全球更广泛监视的一部分。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://start.me/p/aL8RrM/iyp-4" href="https://start.me/p/aL8RrM/iyp-4" rel="noopener noreferrer" target="_blank">
    这些新的监控技术企业
   </a>
   正在激励监控技术的发展，并且不鼓励各国政府调查其来源或哪些权利可能受到侵犯，只因为开发这种东西能让他们收益颇丰。⚪️
  </p>
