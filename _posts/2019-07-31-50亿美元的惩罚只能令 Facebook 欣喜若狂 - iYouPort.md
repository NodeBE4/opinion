  <p>
   <img alt="" class="aligncenter size-full wp-image-7090 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-58.png?resize=1100%2C641&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-58.png?w=1208&amp;ssl=1 1208w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-58.png?resize=300%2C175&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-58.png?resize=768%2C448&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-58.png?resize=1024%2C597&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-58.png?resize=740%2C430&amp;ssl=1 740w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-58.png?resize=1100%2C641&amp;ssl=1 1100w" data-recalc-dims="1" height="641" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-58.png?resize=1100%2C641&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   你可能会认为被迫向联邦政府掏出50亿美元以“补偿”各种不当行为对任何人来说都是一个非常糟糕的事？但是，对于 Facebook 来说，截然不同。
  </p>
  <p class="graf graf--p">
   周三，该公司
   <strong class="markup--strong markup--p-strong">
    高兴地宣布
   </strong>
   了其获得的这个联邦贸易委员会闪亮的弹脑门儿细节，
   <strong class="markup--strong markup--p-strong">
    欣然同意
   </strong>
   了创纪录的罚款和一些名义上的所谓新隐私报告，但是，
   <strong class="markup--strong markup--p-strong">
    没有
   </strong>
   对其经营方式进行任何有意义的改变。
  </p>
  <p class="graf graf--p">
   在一篇回应 FTC 的博客文章中，该公司预示着一个新时代，一个实施“全面的新框架”来保护用户隐私和 Facebook 开展业务的“根本转变”。
  </p>
  <p class="graf graf--p">
   Facebook 总法律顾问 Colin Stretch 说，“这将标志着一种更加明显的隐私转向，其规模与我们过去所做的任何规模都不同。”
  </p>
  <p class="graf graf--p">
   尽管广泛的批评谴责不断揭露这个世界上最大的社交网络的无耻内幕，但美国联邦贸易委员会却将50亿美元的罚款称为“破纪录”的努力、还说“大大削弱了扎克伯格的权力”，同时“为 Facebook 的隐私增加了前所未有的透明度”……
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这都不是真的。
   </strong>
  </p>
  <p class="graf graf--p">
   罚款是因 Facebook 违反了2012年与FTC的先前协议条款。
  </p>
  <p class="graf graf--p">
   这些违规行为包括剑桥分析公司丑闻，以及 Facebook 的其他邪恶做法，包括使用面部识别技术收集用户的生物识别数据、以及假装双因素身份验证以骗取的用户电话号码进行广告宣传，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e-facebook/" href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e-facebook/" rel="noopener noreferrer" target="_blank">
    等等
   </a>
   。
  </p>
  <p class="graf graf--p">
   在2012年和解协议中，Facebook 同意“[给予]消费者明确和突出的通知”，并在共享用户数据时获得“明确同意”。
  </p>
  <p class="graf graf--p">
   好吧你能看到该协议本身就是虚伪的：目前已经众所周知，“知情同意”不能保护数字安全（也许当年还有很多人不了解这点）。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    延伸《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e8%af%b4%e6%ac%a7%e6%b4%b2%e6%96%b0%e6%b3%95%e8%a7%84%e5%8f%8d%e8%80%8c%e4%bc%9a%e5%8a%a0%e5%bc%ba-facebook-%e5%92%8cgoogle-%e7%9a%84%e9%9c%b8%e6%9d%83%ef%bc%9f/" href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e8%af%b4%e6%ac%a7%e6%b4%b2%e6%96%b0%e6%b3%95%e8%a7%84%e5%8f%8d%e8%80%8c%e4%bc%9a%e5%8a%a0%e5%bc%ba-facebook-%e5%92%8cgoogle-%e7%9a%84%e9%9c%b8%e6%9d%83%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     为什么说欧洲新法规反而会加强 Facebook 和Google 的霸权？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   至少当年，该协议是为 Facebook 制定的，“以确保它在未来兑现其承诺。”
  </p>
  <p class="graf graf--p">
   并不是 FTC 的每个人都同意这个所谓的和解协议的条款。美国联邦贸易委员会委员的 Rebecca Kelly Slaughter 和 Rohit Chopra 都表示反对，并指出，
   <strong class="markup--strong markup--p-strong">
    该交易的具体细节实际上是允许 Facebook 基本上自行执行任何新要求
   </strong>
   。
  </p>
  <p class="graf graf--p">
   就如 Chopra 在推文中所说的那样：50亿美元听起来似乎很多，但是其中大部分对 Facebook 来说是值得庆祝的而非惩罚。
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    剑桥分析公司的模式和针对用户的战术
    <strong class="markup--strong markup--blockquote-strong">
     只是对 Facebook 自身实践的小规模反映。
    </strong>
   </p>
  </blockquote>
  <blockquote class="graf graf--blockquote">
   <p>
    值得注意的是，FTC 正在让剑桥分析公司对“个人不当行为”负责。而不是让 Facebook 负责。
   </p>
  </blockquote>
  <blockquote class="graf graf--blockquote">
   <p>
    这就是底线：
    <strong class="markup--strong markup--blockquote-strong">
     Facebook 的公然违规行为是他们大规模监控和操纵业务模式的直接结果，而罚款保护了这一模式的延续
    </strong>
    。和解并不能解决根本性问题。
   </p>
  </blockquote>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “马克扎克伯格、Sheryl Sandberg 和其他 Facebook 高管因其在违规行为中的作用反而获得了全面豁免。
  </p>
  <p class="graf graf--p">
   这是错误的、并且开创了一个可怕的先例，Chopra 在一条推文中写道。 “法律没有给予他们特殊的豁免。”
  </p>
  <p class="graf graf--p">
   法律的确没给他们这份豁免，但掌握法律解释权的政权有可能给了。
   <strong class="markup--strong markup--p-strong">
    分析 Facebook 的政治旋转门《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%b7%b1%e5%b1%82%e6%94%bf%e6%b2%bb%ef%bc%9afacebook%e9%87%91%e5%ad%97%e5%a1%94%e7%9a%84%e4%ba%ba%e5%bd%a2%e8%82%8c%e7%90%86-%e4%bb%85%e4%bb%85%e4%be%9d%e9%9d%a0%e5%bc%80%e6%ba%90/" href="https://www.iyouport.org/%e6%b7%b1%e5%b1%82%e6%94%bf%e6%b2%bb%ef%bc%9afacebook%e9%87%91%e5%ad%97%e5%a1%94%e7%9a%84%e4%ba%ba%e5%bd%a2%e8%82%8c%e7%90%86-%e4%bb%85%e4%bb%85%e4%be%9d%e9%9d%a0%e5%bc%80%e6%ba%90/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     深层政治：Facebook金字塔的人形肌理
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Chopra 和其他人提请注意该和解协议宽泛的措辞。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “50亿美元只是 Facebook 年度收入的一小部分，而联邦贸易委员会已经让 Facebook 和其高管豁免了我们所知道的一切违法行为，甚至是潜在的犯罪行为，”参议员 Ron Wyden 说，并称其为
   <strong class="markup--strong markup--p-strong">
    “甜心交易”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Slaughter 表示，虽然 Facebook 的50亿美元罚款是史无前例的，但它仍然无法触及该公司自身前所未有的规模和权力。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “尽管这一解决方案具有历史意义，但如果要我支持它，我必须确信其合并后的条款将有效地阻止 Facebook 未来继续实施违法行为，”Slaughter 写道，她说，“现在我不相信这种情况。”
  </p>
  <p class="graf graf--p">
   这笔50亿美元的罚款仅仅是 Facebook 财务路线图的预期内 —— 而且它早已直接写入计划了。
   <strong class="markup--strong markup--p-strong">
    在4月份的季度财报电话会议上，Facebook 表示将放弃30亿美元以应对预期的罚款
   </strong>
   。
  </p>
  <p class="graf graf--p">
   周三，Facebook 发布了自己的信息图，概述了其松散的新隐私要求，其中许多都是废话。该公司还提供了马克扎克伯格关于该主题的新视频剪辑。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们对今天宣布的具有里程碑意义的惩罚和行为补救感到非常自豪”，支持和解的联邦贸易委员会委员在一份公开声明中写道。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    对于 Facebook 来说，这笔交易完全是一种解脱
   </strong>
   。惩罚听起来很严厉，对于任何按正常标准规模的企业来说都是如此；但对于 Facebook 来说完全是另一码事。
  </p>
  <p class="graf graf--p">
   随着立法者正排队等候该公司降级，而 Facebook 正在采取一种怯懦的表面姿态，实际上它蠢蠢欲动准备进入即将到来的战斗，以及之后的战斗。
  </p>
  <p class="graf graf--p">
   FTC 的所谓大交易甚至不是 Facebook 当天唯一的一场战斗。该公司自己的博客中写道：它还同意向证券交易委员会拨款
   <a class="markup--anchor markup--p-anchor" data-href="https://www.sec.gov/news/press-release/2019-140" href="https://www.sec.gov/news/press-release/2019-140" rel="noopener noreferrer" target="_blank">
    1亿多美元
   </a>
   ，因为它没有向投资者披露隐私失误。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    罚款真的不能解决问题，尤其是对 Facebook 这种已经成为全球最大经济体的经济支柱的寡头企业来说，推翻它需要付出再一次政变的努力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   更多《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%9b%bd%e7%8e%8b%e4%bb%ac%e7%9a%84%e6%99%9a%e5%ae%b4-%e8%bf%99%e6%98%af%e4%b8%80%e5%9c%ba%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e6%94%bf%e5%8f%98/" href="https://www.iyouport.org/%e5%9b%bd%e7%8e%8b%e4%bb%ac%e7%9a%84%e6%99%9a%e5%ae%b4-%e8%bf%99%e6%98%af%e4%b8%80%e5%9c%ba%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e6%94%bf%e5%8f%98/" rel="noopener noreferrer" target="_blank">
    国王们的晚宴 — — 这是一场看不见的政变
