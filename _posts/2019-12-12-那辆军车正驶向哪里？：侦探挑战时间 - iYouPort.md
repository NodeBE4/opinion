  <p class="graf graf--p">
   你经常会在网络上看到其他人发布一些军车的照片或视频，并同时声称某某军正在进入某某地点。但它是真的吗？定位技术是您获得判断的唯一方法。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    今天的挑战是练习军车识别和探索军事单位组织和结构的好机会。
   </strong>
  </p>
  <p class="graf graf--p">
   开源情报分析是一个非常广泛和多样化的学科，其中就包括能够识别军用车辆和那些使用它们的人的能力，这会成为调查的一个重要组成部分。

   本文的其余部分将介绍在找到正确的位置之前，如何识别车辆和军事部门。

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    坦克和装甲战车
   </strong>
  </p>
  <p class="graf graf--p">
   有多种方法可以开始这一难题，但本文将从车牌开始。
  </p>
  <p class="graf graf--p">
   在黄色背景上的黑色6位字母数字可能会属于多个不同的国家，但其中两个最有可能的选项是荷兰和丹麦。
  </p>

   是这类事物的绝佳资源）
  </p>
  <p class="graf graf--p">
   所以，也许如果能找到荷兰和丹麦军队使用的车辆图像，就可以确认车辆类型，然后从那里开始，以确定使用这种车辆的军事部门。
  </p>
  <p class="graf graf--p">
   但如何识别车辆？
  </p>
  <p class="graf graf--p">
   对于大多数人来说，它看起来像坦克 —— 但这在严格意义上不够正确。
  </p>
  <p class="graf graf--p">
   即使您不是军用爱好者，此照片中显示的一些细节也有助于识别一般车型。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    它不是坦克
   </strong>
   。坦克通常要大得多，枪支也更大，装甲更重，并且没有能力在内部运送更多的士兵。
  </p>

   搜索电气化铁路照片，以确定丹麦使用的是哪种类型：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9041 jetpack-lazy-image" data-lazy-sizes="(max-width: 393px) 100vw, 393px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/16.png?resize=393%2C352&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/16.png?w=393&amp;ssl=1 393w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/16.png?resize=300%2C269&amp;ssl=1 300w" data-recalc-dims="1" height="352" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/16.png?resize=393%2C352&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="393"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9041" data-recalc-dims="1" height="352" sizes="(max-width: 393px) 100vw, 393px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/16.png?resize=393%2C352&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/16.png?w=393&amp;ssl=1 393w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/16.png?resize=300%2C269&amp;ssl=1 300w" width="393"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   3）
   <strong class="markup--strong markup--p-strong">
    蓝色框
   </strong>
   —— 符号显示以-drup结尾的地方的出口，以及以-ov或-on结尾的第二个地方。标志后面还有其他东西是灰色的，也许是道路或桥梁。
  </p>
  <p class="graf graf--p">
   所以，把所有信息放在一起，
   <strong class="markup--strong markup--p-strong">
    已经可以确定以下内容
   </strong>
   ：
  </p>
  <p class="graf graf--p">
   1、该位置位于高速公路上，很可能是西 — 东/东 — 西走向；
  </p>
  <p class="graf graf--p">
   2、高速公路与电气化铁路平行；
  </p>
  <p class="graf graf--p">
   3、铁路必须位于高速公路的北侧，而不是南侧；
  </p>
  <p class="graf graf--p">
   4、该位置位于高速公路出口旁边，地名以-drup / -derup结尾，另一个以-ov / -on结尾；
  </p>
  <p class="graf graf--p">
   5、它必须非常靠近 Holstebro 或 Slagelse 的一个军营。
  </p>
  <p class="graf graf--p">
   因此，通过严格应用这些标准，可以非常快速地缩小搜索范围。
  </p>
  <p class="graf graf--p">
   查看下面的丹麦地图，其中显示了所有主要的高速公路：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9042 jetpack-lazy-image" data-lazy-sizes="(max-width: 350px) 100vw, 350px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/17.jpg?resize=350%2C418&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/17.jpg?w=350&amp;ssl=1 350w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/17.jpg?resize=251%2C300&amp;ssl=1 251w" data-recalc-dims="1" height="418" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/17.jpg?resize=350%2C418&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="350"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9042" data-recalc-dims="1" height="418" sizes="(max-width: 350px) 100vw, 350px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/17.jpg?resize=350%2C418&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/17.jpg?w=350&amp;ssl=1 350w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/17.jpg?resize=251%2C300&amp;ssl=1 251w" width="350"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   紧记上述5个标准，要把它们全部应用进去，很快就会知道正确的位置。
  </p>
  <p class="graf graf--p">
   看图，E39，E45，E47 和 E55 可以排除，因为它们是南北方向而不是西东。
  </p>
  <p class="graf graf--p">
   E20是东西方向的；E20 也离 Holstebro 不远，这意味着 Slagelse 是两个选项中最有可能的地方。
  </p>
  <p class="graf graf--p">
   出于这个原因可以决定不把重点放在埃斯比约和科灵之间的E20西段，因为距离任何一个军事基地都很远。
  </p>
  <p class="graf graf--p">
   先从哥本哈根到欧登塞开始。
   <strong class="markup--strong markup--p-strong">
    目的是：看高速公路的哪一段在北侧有一条平行于它的电气化铁路，然后专注于那条路
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这种方法的优点是可以以卫星模式使用谷歌地图。这个阶段只是在高速公路旁寻找铁路，所以不需要街道细节。
  </p>
  <p class="graf graf--p">
   你当然可以在街景模式下看 E20，但这需要很长时间。而使用卫星图的高度也意味着可以在很短的时间内覆盖大面积：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9043 jetpack-lazy-image" data-lazy-sizes="(max-width: 637px) 100vw, 637px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/18.png?resize=637%2C437&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/18.png?w=637&amp;ssl=1 637w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/18.png?resize=300%2C206&amp;ssl=1 300w" data-recalc-dims="1" height="437" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/18.png?resize=637%2C437&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="637"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9043" data-recalc-dims="1" height="437" sizes="(max-width: 637px) 100vw, 637px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/18.png?resize=637%2C437&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/18.png?w=637&amp;ssl=1 637w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/18.png?resize=300%2C206&amp;ssl=1 300w" width="637"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   经过大约十分钟的检查后发现，E20 的唯一一部分，从东西方向延伸的铁路沿线是
   <a class="markup--anchor markup--p-anchor" data-href="https://goo.gl/maps/DaMw4nyFWnrEjcW66" href="https://goo.gl/maps/DaMw4nyFWnrEjcW66" rel="noopener noreferrer" target="_blank">
    Olby
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://goo.gl/maps/j5BQCLGCqJsdrr7N9" href="https://goo.gl/maps/j5BQCLGCqJsdrr7N9" rel="noopener noreferrer" target="_blank">
    Fjaellebro
   </a>
   之间的区域（
   <a class="markup--anchor markup--p-anchor" data-href="https://goo.gl/maps/A4bddjVXATDnaqU66" href="https://goo.gl/maps/A4bddjVXATDnaqU66" rel="noopener noreferrer" target="_blank">
    Svenstrup
   </a>
   附近有另一个小的区域，但是对街景图的快速查看显示它不匹配）。
  </p>
  <p class="graf graf--p">
   Olby-Fjaellebro 区段距离公路仅约20公里，所以它只是一个很小的区域。
  </p>
  <p class="graf graf--p">
   目标位置必须靠近出口处；出口必须是以-drup结尾的地名和以-or / -ov结尾的第二个名字。
  </p>
  <p class="graf graf--p">
   开始一个非常快速的街景视图查找只关注出口点，在这里看到：33号出口，Vemmedrup / Bjaerskov.
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9044 jetpack-lazy-image" data-lazy-sizes="(max-width: 649px) 100vw, 649px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/19.png?resize=649%2C499&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/19.png?w=649&amp;ssl=1 649w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/19.png?resize=300%2C231&amp;ssl=1 300w" data-recalc-dims="1" height="499" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/19.png?resize=649%2C499&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="649"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9044" data-recalc-dims="1" height="499" sizes="(max-width: 649px) 100vw, 649px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/19.png?resize=649%2C499&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/19.png?w=649&amp;ssl=1 649w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/19.png?resize=300%2C231&amp;ssl=1 300w" width="649"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   这是唯一可能的位置，上述几个标准都符合；但是有一个问题 — 它看起来不像照片上的样子！
  </p>
  <p class="graf graf--p">
   在原始照片中，高速公路旁边有一个堤坝和一些灌木丛，但这里没有堤坝，也没有灌木丛。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    正确的地方，错误的观察点
   </strong>
  </p>
  <p class="graf graf--p">
   在调查过程中也许你经常会遇到类似的状况：从逻辑上讲，这必须是正确的位置，但是街景图像不匹配。
  </p>
  <p class="graf graf--p">
   看起来很矛盾吗？事实上并不存在矛盾。
  </p>
  <p class="graf graf--p">
   现在你要做的是使用街景工具重新对照照片，因为你确定
   <strong class="markup--strong markup--p-strong">
    分析逻辑是对的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在原始图片中，观察点距离路标更远，所以这里需要对 Streetview 进行一些调整，然后就会找到
   <a class="markup--anchor markup--p-anchor" data-href="https://www.google.co.uk/maps/place/55%C2%B028'25.0%22N+12%C2%B004'50.0%22E/@55.4734715,12.079755,389m/data=!3m1!1e3!4m6!3m5!1s0x0:0x0!7e2!8m2!3d55.4736193!4d12.0805668" href="https://www.google.co.uk/maps/place/55%C2%B028%2725.0%22N+12%C2%B004%2750.0%22E/@55.4734715,12.079755,389m/data=!3m1!1e3!4m6!3m5!1s0x0:0x0!7e2!8m2!3d55.4736193!4d12.0805668" rel="noopener noreferrer" target="_blank">
    这个位置
   </a>
   ：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9045 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/20.png?resize=1100%2C618&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/20.png?w=1489&amp;ssl=1 1489w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/20.png?resize=300%2C168&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/20.png?resize=768%2C431&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/20.png?resize=1024%2C575&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/20.png?resize=1100%2C618&amp;ssl=1 1100w" data-recalc-dims="1" height="618" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/20.png?resize=1100%2C618&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9045" data-recalc-dims="1" height="618" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/20.png?resize=1100%2C618&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/20.png?w=1489&amp;ssl=1 1489w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/20.png?resize=300%2C168&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/20.png?resize=768%2C431&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/20.png?resize=1024%2C575&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/20.png?resize=1100%2C618&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   它看起来几乎是正确的；将街景视图与原始视图进行比较时，该标志似乎也是正确的：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9046 jetpack-lazy-image" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/21.png?resize=147%2C134&amp;is-pending-load=1#038;ssl=1" data-recalc-dims="1" height="134" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/21.png?resize=147%2C134&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="147"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9046" data-recalc-dims="1" height="134" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/21.png?resize=147%2C134&amp;ssl=1" width="147"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   但是原图中没有防撞栏。
  </p>
  <p class="graf graf--p">
   在尝试尽可能多地调整几分钟之后终于看懂了，街景图位于高速公路的1号车道（车道内），而原始照片的观察点在2号车道（车道外）拍摄，这就解释了为什么街景图与原始照片之间略有差异 —— 是角度问题。
  </p>
  <p class="graf graf--p">
   鉴于原始图像中卡车的位置，也解释了为什么防撞栏不太明显（只是被挡住了）。
  </p>
  <p class="graf graf--p">
   所以答案是：Vemmedrup 附近 E20 高速公路的33号出口位置，于是军车 CV90 可以前往的最近的军营很可能是 Slagelse 的 Antvorskov Barracks。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这个游戏演示了一系列思考方式，包括在失败的情况下如何继续推进
   </strong>
   。下一次再有人上传某种军车照片或视频并宣称什么军队进入了哪里，您应该知道该如何检验这类消息了。⚪️
  </p>
