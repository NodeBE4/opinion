  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    什么是 Maltego？以及为什么将它用于开源情报调查？
   </strong>
  </p>
  <p class="graf graf--p">
   Maltego 是一种高能的数据挖掘工具，可挖掘
   <strong class="markup--strong markup--p-strong">
    各种
   </strong>
   开源数据资源，并使用这些数据创建用于分析的直观的可视化图表。
  </p>
  <p class="graf graf--p">
   通过这些图表，您可以轻松地在名称、电子邮件组织结构、域名、文档等信息之间建立联系。
   <strong class="markup--strong markup--p-strong">
    可以清楚地看到调查对象私密的全景
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Maltego 使用 Java，因此它可以在 Windows、Mac 和 Linux 上运行，并且可以在 Buscador 或 Kali 等许多 OSINT Linux 发行版中使用。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    我应该下载哪个 Maltego 版本？
   </strong>
  </p>
  <p class="graf graf--p">
   有几种版本的 Maltego 可用：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    Maltego XL —— 适用于大量数据的高级版本；
   </li>
   <li class="graf graf--li">
    Maltego Classic —— 付费版本，包括所有APIs（transforms）；
   </li>
   <li class="graf graf--li">
    Maltego CE —— 免费版本，限制 API（transforms）；
   </li>
   <li class="graf graf--li">
    Casefile —— 包含大部分与商业版本相同的功能，但不可用于商业用途，在返回结果时也有相应的数量限制，不支持导出结果图表
   </li>
  </ul>
  <p class="graf graf--p">
   Maltego Classic、Maltego XL 和 Maltego CE 之间的主要区别在于可以从单个转换返回的实体数量、以及可以在单个图形上的最大实体数量。
  </p>
  <p class="graf graf--p">
   出于本指南的目的，这里将使用的是 Maltego CE，这是一个有限转换的免费版本。
  </p>
  <p class="graf graf--p">
   Maltego 就预装在 Buscador Linux 发行版中，这通常是开源情报调查员的最爱。
  </p>

   <strong class="markup--strong markup--p-strong">
    安装 Maltego
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Buscador
   </strong>
   ：如果你有通过 Buscador 的 Maltego，它最初将作为 Casefile 版本出现。你需要去
   <a class="markup--anchor markup--p-anchor" data-href="https://www.paterva.com/community/community.php" href="https://www.paterva.com/community/community.php" rel="noopener noreferrer" target="_blank">

  <p class="graf graf--p">
   创建帐户后，您将收到一个密钥，将您的 Casefile 版本转换为CE。
