  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      <strong>
       读心术已经彻底实现了。新的技术能够在你的想法进入意识之前就预测到它，还可以将你脑子里出现的东西转化为可理解的和可识别的语音。这对隐私意味着什么？对那些被执法部门审讯的人意味着什么？
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-2852 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-46.png?resize=1100%2C730&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-46.png?w=1350&amp;ssl=1 1350w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-46.png?resize=300%2C199&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-46.png?resize=768%2C510&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-46.png?resize=1024%2C680&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-46.png?resize=1100%2C730&amp;ssl=1 1100w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-46.png?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="730" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-46.png?resize=1100%2C730&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-2852" data-recalc-dims="1" height="730" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-46.png?resize=1100%2C730&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-46.png?w=1350&amp;ssl=1 1350w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-46.png?resize=300%2C199&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-46.png?resize=768%2C510&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-46.png?resize=1024%2C680&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-46.png?resize=1100%2C730&amp;ssl=1 1100w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-46.png?resize=272%2C182&amp;ssl=1 272w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   你究竟有没有自由意志？这个问题在学界已经争议了很多年。根据一项新的研究显示，也许人类真的没有自由。该研究显示，我们对个人选择的控制可能比我们想象的要少很多。在我们意识到它之前，无意识的大脑活动似乎已经决定了我们的选择。
  </p>
  <p class="graf graf--p">
   澳大利亚新南威尔士大学心理学院未来思想实验室的研究人员能够预测参与者在有意识地宣布他们的决定之前所做出的基本选择。他们的研究结果发表在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nature.com/articles/s41598-019-39813-y" href="https://www.nature.com/articles/s41598-019-39813-y" rel="noopener noreferrer" target="_blank">
    “科学报告”杂志
   </a>
   上。
  </p>
  <p class="graf graf--p">
   研究人员要求14名参与者在两种视觉模式之间自由选择 — — 一种是红色水平条纹，另一种是绿色垂直条纹，在功能性磁共振成像仪（fMRI）观察之前有意识地想象选定的图案。
  </p>
  <p class="graf graf--p">
   他们最多有20秒的时间来选择模式。一旦他们做出决定，就按下一个按钮，并用10秒钟的时间尽可能地将图案可视化。接下来，他们被问到“你想到什么？”和“它有多生动？”参与者按下按钮回答这些问题。
  </p>
  <p class="graf graf--p">
   科学家们能够在人们的思想变得有意识之前就预测到人们会选择哪种模式。
  </p>
  <p class="graf graf--p">
   以下是
   <a class="markup--anchor markup--p-anchor" data-href="https://newsroom.unsw.edu.au/news/science-tech/our-brains-reveal-our-choices-we%E2%80%99re-even-aware-them-study" href="https://newsroom.unsw.edu.au/news/science-tech/our-brains-reveal-our-choices-we%E2%80%99re-even-aware-them-study" rel="noopener noreferrer" target="_blank">
    新南威尔士大学新闻稿
   </a>
   对结果的解释：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    研究人员不仅可以预测参与者会选择哪种模式，还可以预测参与者对其可视化的评价程度。在机器学习的帮助下，
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     研究人员成功地在思想变得有意识之前的平均11秒就能预测到了参与者的意志选择
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    揭示未来选择信息的大脑区域是大脑的执行区域 — — 帮助我们进行有意识的决策 — — 以及视觉和皮质下结构，这表明了一个扩展的思想诞生区域网络
   </em>
   。
  </p>
  <p class="graf graf--p">
   Joel Pearson 教授说，我们可能会根据之前的大脑活动进行思考，然后
   <strong class="markup--strong markup--p-strong">
    在没有意识到的情况下过往的经验就会影响我们的最终决定
   </strong>
   ：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “我们相信，当我们面对两种或两种以上思考的选择时，已经存在无意识的思维痕迹，有点像无意识的幻觉。当决定考虑什么时，大脑的执行区域会选择更强大的思维轨迹。换句话说，
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     如果任何预先存在的大脑活动与您的一个选择相匹配，那么您的大脑将更有最可能选择该选项，因为它会被预先存在的大脑活动所提升
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    例如，这可以解释为什么一遍又一遍地思考某些东西会导致对它的更多思考，因为它发生在一个积极的反馈循环中。
   </em>
   “
  </p>
  <p class="graf graf--p">
   未来思想的主观力量还取决于早期视觉皮层中的活动，早期视觉皮层是大脑中从外部世界接收视觉信息的区域。研究人员解释说，这表明感知区域（被认为随机变化）的当前活动状态会对我们对事物的思考程度产生影响。
  </p>
  <p class="graf graf--p">
   虽然这些研究结果可能看起来令人震惊，但这项研究并不是第一次表明思想可以在意识到之前被预测的研究。
  </p>
  <p class="graf graf--p">
   Quartz 报告称，这项研究建立在之前的研究基础之上：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    正如研究人员指出的那样，类似的技术已经能够在人们的思考进入意识到之前的7到10秒之间预测到该决策，并且在人们有意识之前最多可以提前4秒预测到。总之，这些研究表明了对大脑的理解使我们对自由意志的概念复杂化了。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    神经科学家早就知道大脑在你有意识采取行动之前就会作出准备，在意识和非意识之间只有几毫秒。这几毫秒可以让我们有机会有意识地拒绝无意识的冲动，似乎形成了自由意志的基础
   </em>
   。
  </p>
  <p class="graf graf--p">
   研究人员表示，他们的发现可能对精神障碍有影响，这些精神障碍涉及使用心理图像的思维入侵，如创伤后应激障碍。他们告诫不要假设所有选择都是通过预先存在的大脑活动来预先确定的。“我们的结果不能保证所有选择都是在非自愿图像出现之前，但它表明这种机制的确存在，它可能会影响我们的日常选择，” Pearson 教授说。
  </p>
  <p class="graf graf--p">
   这种研究可以使患有某些疾病的人受益 — — 但代价是什么？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果它落入错误的人手中，那么这种技术肯定会有被滥用和被操纵的可能性。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    想想看，这对隐私意味着什么？对那些被执法部门审讯的人意味着什么？
   </strong>
  </p>
  <p class="graf graf--p">
   这一清单还可以继续下去。
  </p>
  <p class="graf graf--p">
   这还只是侵入性技术日益严重的问题的一部分。
  </p>
  <p class="graf graf--p">
   就在上个月，
   <strong class="markup--strong markup--p-strong">
    哥伦比亚大学的神经工程师团队报告说
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theorganicprepper.com/mind-reading-technology/" href="https://www.theorganicprepper.com/mind-reading-technology/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     他们开发了一个系统
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，可以将人们的想法转化为可理解的和可识别的语音
   </strong>
   。这是真正的读心术了。
  </p>
  <p class="graf graf--p">
   如果您担心所有这些研究会走多远，那么您并不孤单。正如我在“科学事实”中所说的：
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theorganicprepper.com/mind-reading-technology/" href="https://www.theorganicprepper.com/mind-reading-technology/" rel="noopener noreferrer" target="_blank">
    读心技术现在已成为现实
   </a>
   ：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    随着技术进步的快速发展，我们不得不怀疑……我们离技术奇点到底有多接近
   </em>
   ？
  </p>
  <p class="graf graf--p">
   就此而言 — 事实上 Facebook 已经真正想到了直接从用户的大脑读取信息的方法。是的，你没看错：
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wired.com/story/zuckerberg-wants-facebook-to-build-mind-reading-machine/" href="https://www.wired.com/story/zuckerberg-wants-facebook-to-build-mind-reading-machine/" rel="noopener noreferrer" target="_blank">
    Facebook 想要读懂用户的想法
   </a>
   。在2月份与哈佛大学法学院教授 Jonathan Zittrain 的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.youtube.com/watch?v=WGchhsKhG-A" href="https://www.youtube.com/watch?v=WGchhsKhG-A" rel="noopener noreferrer" target="_blank">
    访谈中
   </a>
   ，首席执行官马克扎克伯格已经提到了社交媒体巨头正在研究人们的大脑 — — （这东西看起来类似于浴帽）。
  </p>
  <p class="graf graf--p">
   作为回应，Zittrain 说，“第五修正案的影响是惊人的。”
  </p>
  <p class="graf graf--p">
   尽管多年来的媒体报道，公众对侵犯隐私权的行为感到愤怒，以及数百万用户的流失，但 Facebook 仍然决心要感染我们的生活，无论我们是否愿意。如果您认为自己不受科技巨头令人毛骨悚然的追踪者策略的影响，请再想一想：
   <strong class="markup--strong markup--p-strong">
    即使您已经停用了所有社交媒体账户 — — 或者从未有过任何社交媒体账户 — — 您的隐私也无法保证
   </strong>
   。
  </p>
  <p class="graf graf--p">
   技术进步如此之快，难以跟上。
  </p>
  <p class="graf graf--p">
   几乎每天都有关于新的和潜在侵入性发展的报道正在宣布。科学家和科技公司声称这项研究是为了“社会的利益”，即使在警告我们潜在的危险时也是如此地胡扯。
  </p>
  <p class="graf graf--p">
   是的，反乌托邦的噩梦正在逼近。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    It
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.zerohedge.com/news/2019-03-19/reality-mind-reading-these-neuroscientists-can-predict-your-choices-11-seconds-you" href="https://www.zerohedge.com/news/2019-03-19/reality-mind-reading-these-neuroscientists-can-predict-your-choices-11-seconds-you" rel="noopener noreferrer" target="_blank">
    sure seems like a dystopian nightmare is approaching
   </a>
   . The Reality Of Mind-Reading: Neuroscientists Can Predict Your Choices 11 Seconds Before You Make Them
  </p>
