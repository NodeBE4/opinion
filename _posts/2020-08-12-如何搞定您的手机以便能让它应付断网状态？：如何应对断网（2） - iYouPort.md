  </p>
  <p class="graf graf--p">
   互联网关闭通常与加强信息控制以及限制表达和集会自由相吻合。这在任何国家都是如此。
   <strong class="markup--strong markup--p-strong">

    不同的其他手机
   </strong>
   进行记录。

   您也可以确保在恢复出厂设置之前对设备的内容进行加密。
  </p>
  <p class="graf graf--p">
   无论如何，加密是当前大多数电话上的默认设置，但如果没有，则在重置之前先进入“设置”&gt;“安全性”&gt;“加密电话”。这样，当您将手机恢复出厂设置时，加密密钥将丢失，并且所有未擦除的数据将无法读取。
  </p>

   </noscript>
  </p>
  <p class="graf graf--p">
   4、使用文件管理器应用程序创建隐藏文件夹（以 “.” 开头的文件夹）。
  </p>
  <p class="graf graf--p">
   您可以手动将文件移动到隐藏文件夹，或者如果您使用像
   <a class="markup--anchor markup--p-anchor" data-href="http://opencamera.org.uk/" href="http://opencamera.org.uk/" rel="noopener noreferrer" target="_blank">
    Open Camera
   </a>
   之类的应用程序，则可以指定存储录制和拍摄的媒体的位置。
  </p>
  <p class="graf graf--p">
   确保在 “设置” 中关闭 “显示隐藏文件” 选项，以使隐藏文件不可见。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13380 jetpack-lazy-image" data-lazy-sizes="(max-width: 768px) 100vw, 768px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/04/6.png?resize=768%2C592&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/04/6.png?w=768&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/04/6.png?resize=300%2C231&amp;ssl=1 300w" data-recalc-dims="1" height="592" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/04/6.png?resize=768%2C592&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="768"/>

  </p>

   </noscript>
  </p>
  <p class="graf graf--p">
   5、某些专门的应用程序 —— 例如
   <a class="markup--anchor markup--p-anchor" data-href="https://hzontal.org/tella/" href="https://hzontal.org/tella/" rel="noopener noreferrer" target="_blank">
    Tella
   </a>
   或
   <a class="markup--anchor markup--p-anchor" data-href="https://play.google.com/store/apps/details?id=com.camera.easy&amp;hl=en" href="https://play.google.com/store/apps/details?id=com.camera.easy&amp;hl=en" rel="noopener noreferrer" target="_blank">
    Eyewitness to Atrocities
   </a>
   —— 将文档存储在单独的加密相册中，这些相册的内容只能在该应用程序中访问，这可能会使搜查您的手机的人不容易找到。
  </p>
  <p class="graf graf--p">
   访问这些安全图库中的文档需要单独的密码，因此即使您的手机未锁定，这一部分仍会保持加密状态。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13382 jetpack-lazy-image" data-lazy-sizes="(max-width: 606px) 100vw, 606px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/8.png?resize=606%2C408&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/8.png?w=606&amp;ssl=1 606w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/8.png?resize=300%2C202&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/8.png?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="408" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/04/8.png?resize=606%2C408&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="606"/>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    有关隐藏内容的重要说明
   </strong>
  </h3>
  <p class="graf graf--p">
   ⚠️需要注意的是，以上这些技术可能足以甩掉那些快速搜查您的手机的人，就像在地铁出口处随机拦住任何人搜查的行为，但是，可能无法阻止专心致志的警察取证。在下面看到：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%95%b0%e5%ad%97%e5%8f%96%e8%af%81%e6%ad%a3%e5%9c%a8%e6%b3%9b%e6%bb%a5%ef%bc%9a%e6%94%bf%e5%ba%9c%e5%8f%af%e4%bb%a5%e4%bb%8e%e4%bd%a0%e7%9a%84%e6%89%8b%e6%9c%ba%e4%b8%ad%e6%8f%90%e5%8f%96%e5%93%aa/" href="https://www.iyouport.org/%e6%95%b0%e5%ad%97%e5%8f%96%e8%af%81%e6%ad%a3%e5%9c%a8%e6%b3%9b%e6%bb%a5%ef%bc%9a%e6%94%bf%e5%ba%9c%e5%8f%af%e4%bb%a5%e4%bb%8e%e4%bd%a0%e7%9a%84%e6%89%8b%e6%9c%ba%e4%b8%ad%e6%8f%90%e5%8f%96%e5%93%aa/" rel="noopener noreferrer" target="_blank">
     数字取证正在泛滥：政府可以从你的手机中提取哪些数据？
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%ae%a9%e6%94%bf%e5%ba%9c%e6%9c%ba%e6%9e%84%e4%bb%8e%e6%82%a8%e4%bd%bf%e7%94%a8%e7%9a%84%e5%ba%94%e7%94%a8%e7%a8%8b%e5%ba%8f%e4%b8%ad%e6%8f%90%e5%8f%96%e5%a4%a7%e9%87%8f%e6%95%b0%e6%8d%ae%e7%9a%84/" href="https://www.iyouport.org/%e8%ae%a9%e6%94%bf%e5%ba%9c%e6%9c%ba%e6%9e%84%e4%bb%8e%e6%82%a8%e4%bd%bf%e7%94%a8%e7%9a%84%e5%ba%94%e7%94%a8%e7%a8%8b%e5%ba%8f%e4%b8%ad%e6%8f%90%e5%8f%96%e5%a4%a7%e9%87%8f%e6%95%b0%e6%8d%ae%e7%9a%84/" rel="noopener noreferrer" target="_blank">

    </a>

  <p class="graf graf--p">
   还请记住，某些国家/地区可能有法律限制、将使用上述这类安全应用程序定为犯罪。
  </p>
  <p class="graf graf--p">
   使用这些安全应用程序来防止当局访问您的数据，可能会被当权者视为 “破坏证据或妨碍调查”，并可能构成犯罪。
  </p>
  <p class="graf graf--p">
   👉如果您对您所在国家/地区的法律不很清楚，请查看
   <a class="markup--anchor markup--p-anchor" data-href="https://www.gp-digital.org/world-map-of-encryption/" href="https://www.gp-digital.org/world-map-of-encryption/" rel="noopener noreferrer" target="_blank">
    此处的地图
   </a>
   （很全面，从2017年开始）是一个很好的起点。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    设置离线共享
   </strong>
  </h3>
  <p class="graf graf--p">
   在捕获到当权者的罪证和反抗者的重要行动记录后却没有互联网的情况下，出于安全原因，或为了释放空间或与他人共享，您可能仍希望将文档从手机上转移出去。
  </p>
  <p class="graf graf--p">
   定期从手机上卸载文档也将有助于最大程度地减少泄露的信息，比如，您的设备可能会被查抄或被强行解锁。
  </p>
  <p class="graf graf--p">
   即使您无法连接到互联网，也仍然可以在本地连接启用 wifi 或蓝牙的设备，例如另一部电话或 wifi USB 驱动器。
  </p>
  <p class="graf graf--p">
   如果您的手机支持，也可以插入 USB On-The-Go（OTG）以将文档卸载到OTG驱动或其他设备。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在遇到危机之前进行练习
   </strong>
  </h3>
  <p class="graf graf--p">
   如果您可以上网，请立即设置手机。在日常情况下（不存在安全问题的情况下）开始练习使用这些应用程序，以使您能熟悉和舒适地使用它们。
  </p>
  <p class="graf graf--p">
   👉将良好的基本手机安全操作作为默认习惯性做法。只有这样，当您陷入危机中有很多其他事需要担心时，才不至于因慌乱而出错。⚪️
  </p>
