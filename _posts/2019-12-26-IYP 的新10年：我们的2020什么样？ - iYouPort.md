  <p class="graf graf--p">
   Hi，所有亲爱的每一位，深深感谢您再次持续一年的关注！由衷地希望我们提供的内容能对您有所帮助。
  </p>
  <p class="graf graf--p">
   IYP（
   <a class="markup--anchor markup--p-anchor" data-href="http://iyouport.org" href="http://iyouport.org" rel="noopener noreferrer" target="_blank">
    iyouport
   </a>
   ）已经工作5年，这其间经历了很多事；包括最初的 .com 网站的消失、多次转移平台、改革、以及联盟。
  </p>
  <p class="graf graf--p">
   我们不打算回忆，也不打算总结2019；IYP 只希望往前走，不抱怨、不扯皮、不自怜、也不自赏。
  </p>
  <p class="graf graf--p">
   我们知道在过去的数年中自己还没能做到什么，于是我们希望在新的10年中能弥补这一缺憾。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     IYP 的 2020
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     1、我们将调整发布数量。
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   陆续有读者反馈说，我们提供的内容 “太多了”，短时间内难以消化。这的确是个问题。
  </p>
  <p class="graf graf--p">
   由于 IYP 的内容大多是实用性的，您需要
   <strong class="markup--strong markup--p-strong">
    亲手实践
   </strong>
   它们才有希望掌握并获得创意灵感；为了给大家充分的时间，我们将调整为每日4篇文章，同时关注反馈。
  </p>
  <p class="graf graf--p">
   发布时间不变，依旧是：周二～周五，北欧时间 (GMT+2) 每晚 18:00 后、中国时间 (GMT+8)每日零点后，您将在首页 “The New” 看到最新的发布。
  </p>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     2、（适当）增加互动。
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   IYP是不盈利的，也没有任何资金，从来都是如此 —— 我们不受地主和资本家的束缚。
  </p>
  <p class="graf graf--p">
   但也因此，我们的人手极少，所有人都是义务劳动。于是，在提供丰富内容的同时协调更广泛的互动对我们来说是有难度的。
  </p>
  <p class="graf graf--p">
   但是，我们真的希望和您沟通；如果您生活在中国，您的经验将对IYP和我们的所有读者都具有非常高的价值。
  </p>
  <p class="graf graf--p">
   在2019年我们已经开始接受来稿 —— 在此感谢所有来稿的朋友 —— 2020年IYP将继续接受来稿，不过我们希望简单说明一下规则。
  </p>
  <p class="graf graf--p">
   如下：
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     A：内幕。
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   如果您希望提供内幕消息，首先请保持足够的匿名性，这是为了您的基本安全利益；与此同时，请提供
   <strong class="markup--strong markup--p-strong">
    具体文件
   </strong>
   ，文件需要能说明您希望说明的信息，而不仅仅是 “我听说/我知道”。
  </p>
  <p class="graf graf--p">
   我们会通过多种方法验证文件的真实性。
  </p>
  <p class="graf graf--p">
   如上所述，我们的人手有限，暂时没能实现基于网站的安全投递渠道（嗯，这不仅需要人，也需要钱，租地堡并不便宜）；于是，我们
   <strong class="markup--strong markup--p-strong">
    不会主动
   </strong>
   建议您提供内幕资料，如果您确认自己安全，我们会很欢迎您的知识。
  </p>
  <p class="graf graf--p">
   对于安全投递渠道，有多家媒体平台提供。但并非所有平台的相关技术都是可靠的。不可靠的技术已经导致很多举报人被捕。
  </p>
  <p class="graf graf--p">
   但是 Wikileaks 的渠道已经过10几年的检验，它是可靠的；此外 intercept 也可以考虑，如果您深信中美之间 “没有” 情报共享的话。
  </p>
  <p class="graf graf--p">
   我们在 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e9%80%8f%e6%98%8e%e5%ba%a6%e9%9d%a9%e5%91%bd/" href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e9%80%8f%e6%98%8e%e5%ba%a6%e9%9d%a9%e5%91%bd/" rel="noopener noreferrer" target="_blank">
    透明度革命
   </a>
   ” 专栏中介绍的技巧和经验希望能给您以帮助。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     B：实用性。
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   目前我们对来稿的期待着眼于
   <strong class="markup--strong markup--p-strong">
    实用性
   </strong>
   —— 也就是说，我们希望知道 “怎么办”，而不仅仅是 “发生了什么”。
  </p>
  <p class="graf graf--p">
   举个例子。一些中文媒体曾经报道过黄貂鱼在中国的布设；IYP 随即提供了《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ba%94%e4%bb%98%e6%94%bf%e5%ba%9c%e4%bd%bf%e7%94%a8%e9%bb%84%e8%b2%82%e9%b1%bc%e5%af%b9%e5%85%ac%e6%b0%91%e8%bf%9b%e8%a1%8c%e7%9a%84%e5%a4%a7%e8%a7%84%e6%a8%a1%e9%97%b4%e8%b0%8d/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ba%94%e4%bb%98%e6%94%bf%e5%ba%9c%e4%bd%bf%e7%94%a8%e9%bb%84%e8%b2%82%e9%b1%bc%e5%af%b9%e5%85%ac%e6%b0%91%e8%bf%9b%e8%a1%8c%e7%9a%84%e5%a4%a7%e8%a7%84%e6%a8%a1%e9%97%b4%e8%b0%8d/" rel="noopener noreferrer" target="_blank">
    如何应付政府使用黄貂鱼对公民进行的大规模间谍行为？#反监视技术
   </a>
   》。
  </p>
  <p class="graf graf--p">
   如果您在中国生活，并拥有
   <strong class="markup--strong markup--p-strong">
    识别/阻断
   </strong>
   黄貂鱼等各种攻击的相关经验，我们会非常期待能把您的经验分享给更多人。这就是实用性。
  </p>
  <p class="graf graf--p">
   再比如，我们的频道分享了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ba%94%e4%bb%98%e6%94%bf%e5%ba%9c%e4%bd%bf%e7%94%a8%e9%bb%84%e8%b2%82%e9%b1%bc%e5%af%b9%e5%85%ac%e6%b0%91%e8%bf%9b%e8%a1%8c%e7%9a%84%e5%a4%a7%e8%a7%84%e6%a8%a1%e9%97%b4%e8%b0%8d/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ba%94%e4%bb%98%e6%94%bf%e5%ba%9c%e4%bd%bf%e7%94%a8%e9%bb%84%e8%b2%82%e9%b1%bc%e5%af%b9%e5%85%ac%e6%b0%91%e8%bf%9b%e8%a1%8c%e7%9a%84%e5%a4%a7%e8%a7%84%e6%a8%a1%e9%97%b4%e8%b0%8d/" rel="noopener noreferrer" target="_blank">
    直接行动模版资料
   </a>
   ，如果您在阅读后开始在本地社区
   <strong class="markup--strong markup--p-strong">
    实践
   </strong>
   ，您从中发现了什么？有没有遇到麻烦？是否从实践中找到了更多的灵感？以及哪些方法更适合中国的社区？我们认为这些知识非常有价值。
  </p>
  <p class="graf graf--p">
   以上都是实用性。
  </p>
  <p class="graf graf--p">
   事实上
   <strong class="markup--strong markup--p-strong">
    还有一种
   </strong>
   实用性。曾经有读者发现，Tor 的3次跳跃中的2次都在同一个国家。这是个问题。如果您能将您的发现写成报告，我们可以发表它，并且帮助您与Tor的技术人士
   <strong class="markup--strong markup--p-strong">
    沟通
   </strong>
   ，探讨解决方案。
  </p>
  <p class="graf graf--p">
   同样，如果您的测试发现中国的GFW采取了新的扫描封锁方式，以至于曾经可以使用的反封锁工具失败，您可以记录下您的调查过程，我们将以此为线索与国际上的各种反封锁技术团队探讨更好的绕避方法。
  </p>
  <p class="graf graf--p">
   这只是一些例子，还可以有更多。
  </p>
  <p class="graf graf--p">
   如上，这是我们希望的来稿。IYP 专注于解决问题，而不是仅仅希望听闻发生了什么。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     C：您的创造力。
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   网络封锁和监控是一种工具，它在全球大部分国家被用来镇压、甚至预测群体事件，以提前维稳。
  </p>
  <p class="graf graf--p">
   原因在于：这个互联网不是民主的，公民无法掌握它的基础设施。老大哥就内嵌在这个互联网中。
  </p>
  <p class="graf graf--p">
   还记得 IYP 的口头禅吗？“别依赖你无法掌握的东西”。
  </p>
  <p class="graf graf--p">
   它还有下半句：我们需要构建自己能掌握的东西。
  </p>
  <p class="graf graf--p">
   这就是为什么
   <strong class="markup--strong markup--p-strong">
    Mesh Networking
   </strong>
   在近年来一直都是全球数字权利社区的热门话题。
  </p>
  <p class="graf graf--p">
   如果您是动手爱好者，非常欢迎您实践 “自己的互联网” 创意，并写下您的构想；如果您的测试取得了进展，我们甚至可能为您提供相关的技术协助、与全球优秀开发者面对面探讨的机会 …… 甚至更多。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     D
    </strong>
    ：
    <strong class="markup--strong markup--p-strong">
     翻译志愿者
    </strong>
    。
   </span>
  </p>
  <p class="graf graf--p">
   我们的
   <a class="markup--anchor markup--p-anchor" data-href="https://t.me/iyouport" href="https://t.me/iyouport" rel="noopener noreferrer" target="_blank">
    telegram 频道
   </a>
   提供了很多书籍和资料的下载。它们都是英文的。已经有多位读者留言表示期待中文版。
  </p>
  <p class="graf graf--p">
   IYP 再次受限于人手过少，但是，如果任何读者希望翻译我们提供的书籍和资料，我们都会格外感激您！
  </p>
  <p class="graf graf--p">
   您可能不需要翻译全部，可以翻译其中最精彩的一两个章节，我们会将您的劳动成果与更大范围的读者群体共享。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     E：多媒体形式
    </strong>
    。
   </span>
  </p>
  <p class="graf graf--p">
   想用 IYP 赚钱吗？
  </p>
  <p class="graf graf--p">
   IYP 正式邀请读者将我们提供的知识制作成多媒体形式，
   <strong class="markup--strong markup--p-strong">
    以此获得的一切收益（不论是广告收入还是打赏）完全归您所有！IYP分文不取
   </strong>
   。
  </p>
  <p class="graf graf--p">
   👉只有一点：请在您的作品中加入 IYP 的 logo 和域名 “iyouport.org”，并在简介中写明您使用的是哪一篇文章。
  </p>
  <p class="graf graf--p">
   有灵感吗？我提示一下：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    比如，实践我们提供的文字
    <a class="markup--anchor markup--li-anchor" data-href="https://start.me/p/xbYXdR/iyp-1" href="https://start.me/p/xbYXdR/iyp-1" rel="noopener noreferrer" target="_blank">
     教程
    </a>
    ，发表您的屏幕录制（配音可选）；
   </li>
   <li class="graf graf--li">
    再比如，将我们介绍的
    <a class="markup--anchor markup--li-anchor" data-href="https://start.me/p/1kod2L/iyp-direct-action-5" href="https://start.me/p/1kod2L/iyp-direct-action-5" rel="noopener noreferrer" target="_blank">
     反侦察
    </a>
    知识做成短片，演示社交工程攻击和防御；
   </li>
   <li class="graf graf--li">
    还有，使用我们提供的
    <a class="markup--anchor markup--li-anchor" data-href="https://start.me/p/1kod2L/iyp-direct-action-5" href="https://start.me/p/1kod2L/iyp-direct-action-5" rel="noopener noreferrer" target="_blank">
     开源情报
    </a>
    工具，揭露网络造假、挖掘信息战、曝光水军、追踪诈骗，等等；
   </li>
   <li class="graf graf--li">
    甚至可以，和您的团队合作尝试亲测 “
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/?s=%E5%AE%8C%E7%BE%8E%E9%9A%90%E8%BA%AB" href="https://www.iyouport.org/?s=%E5%AE%8C%E7%BE%8E%E9%9A%90%E8%BA%AB" rel="noopener noreferrer" target="_blank">
     完美隐身
    </a>
    ”，拍摄系列剧……
   </li>
  </ul>
  <p class="graf graf--p">
   📢再一次！您的作品获得的任何收入均归您一人所有！只需要添加 IYP 的 Logo 和域名 “iyouport.org”，并在简介中写明您使用的是哪一篇文章！
  </p>
  <p class="graf graf--p">
   A、B、C、D、E，上述就是 IYP 认为最佳的互动方式。如果这其中有适合您的部分，您已经收到了我的飞吻 😘
