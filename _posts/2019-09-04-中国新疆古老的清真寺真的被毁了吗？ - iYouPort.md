  <p class="graf graf--p">
   Twitter 上的活动人士正在声称一条消息：中国一直在摧毁新疆的历史清真寺。新疆是大量维吾尔族人的家园，维吾尔族是中国主要的穆斯林少数民族。
  </p>
  <p class="graf graf--p">
   这些消息是真的吗？
  </p>
  <p class="graf graf--p">
   我们来验证一下。
  </p>
  <p class="graf graf--p">
   通过使用开源调查技术和卫星图像，可以找到这些清真寺、并检查这些声明的真实性。当涉嫌的破坏行为发生时，用这些方法也可以缩小范围找到准确的位置。
  </p>
  <p class="graf graf--p">
   现在开始。
  </p>

    1、Keriya Aitika 清真寺

   虽然使用谷歌地球历史图像已经可以显示门楼肯定已被摧毁，但你仍然可以使用

    <img alt="" class="aligncenter size-full wp-image-4291 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/变.png?resize=1100%2C802&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/变.png?w=1210&amp;ssl=1 1210w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/变.png?resize=300%2C219&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/变.png?resize=768%2C560&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/变.png?resize=1024%2C746&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/变.png?resize=1100%2C802&amp;ssl=1 1100w" data-recalc-dims="1" height="802" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/变.png?resize=1100%2C802&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
    <noscript>
     <img alt="" class="aligncenter size-full wp-image-4291" data-recalc-dims="1" height="802" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/变.png?resize=1100%2C802&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/变.png?w=1210&amp;ssl=1 1210w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/变.png?resize=300%2C219&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/变.png?resize=768%2C560&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/变.png?resize=1024%2C746&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/变.png?resize=1100%2C802&amp;ssl=1 1100w" width="1100"/>
    </noscript>
   </p>
   <p class="graf graf--p">
    <p class="graf graf--p">
     <strong class="markup--strong markup--p-strong">
      2、Kargilik 清真寺
     </strong>
    </p>
    <p class="graf graf--p">
     关于这个主题的原始帖子
     <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/shawnwzhang/status/1113148584310927360" href="https://twitter.com/shawnwzhang/status/1113148584310927360" rel="noopener noreferrer" target="_blank">
      thread
     </a>
     中第二个用户询问了 Kargilik 清真寺以及它是否被夷为平地。
    </p>
    <p class="graf graf--p">
     <img alt="" class="aligncenter size-full wp-image-4284 jetpack-lazy-image" data-lazy-sizes="(max-width: 516px) 100vw, 516px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/7.jpg?resize=516%2C540&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/7.jpg?w=516&amp;ssl=1 516w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/7.jpg?resize=287%2C300&amp;ssl=1 287w" data-recalc-dims="1" height="540" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/7.jpg?resize=516%2C540&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="516"/>
     <noscript>
      <img alt="" class="aligncenter size-full wp-image-4284" data-recalc-dims="1" height="540" sizes="(max-width: 516px) 100vw, 516px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/7.jpg?resize=516%2C540&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/7.jpg?w=516&amp;ssl=1 516w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/7.jpg?resize=287%2C300&amp;ssl=1 287w" width="516"/>
     </noscript>
    </p>
    <p class="graf graf--p">
     找到这座清真寺的确切位置有点困难，但不是难太多。只需在 Google 中键入“Kargilik Mosque”，就会发现第一个结果是协作制图服务
     <a class="markup--anchor markup--p-anchor" data-href="http://wikimapia.org/21695457/Kargilik-mosque" href="http://wikimapia.org/21695457/Kargilik-mosque" rel="noopener noreferrer" target="_blank">
      Wikimapia
     </a>
     中的一个位置。
    </p>
    <p class="graf graf--p">
     通过比较这个位置的卫星图像和找到的关于这个清真寺的其他图像，可以确认这个位置是正确的。
    </p>
    <p class="graf graf--p">
     <img alt="" class="aligncenter size-full wp-image-4285 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/8.png?resize=1100%2C412&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/8.png?w=1234&amp;ssl=1 1234w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/8.png?resize=300%2C112&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/8.png?resize=768%2C288&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/8.png?resize=1024%2C383&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/8.png?resize=1100%2C412&amp;ssl=1 1100w" data-recalc-dims="1" height="412" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/8.png?resize=1100%2C412&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
     <noscript>
      <img alt="" class="aligncenter size-full wp-image-4285" data-recalc-dims="1" height="412" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/8.png?resize=1100%2C412&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/8.png?w=1234&amp;ssl=1 1234w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/8.png?resize=300%2C112&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/8.png?resize=768%2C288&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/8.png?resize=1024%2C383&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/8.png?resize=1100%2C412&amp;ssl=1 1100w" width="1100"/>

    <p class="graf graf--p">
     查看 Google 地球上公开提供的图像，可以发现这座清真寺没有明显的变化。
    </p>
    <p class="graf graf--p">
     这是因为，在撰写本文时，Google 地球上最新的可用图像是从2018年9月1日开始的，那时该清真寺仍然存在。
    </p>
    <p class="graf graf--p">
     但是，使用 Planet Labs 图像就可以看到，在2018年末，这个位置发生了重大变化。
    </p>

     <p class="graf graf--p">
      <strong class="markup--strong markup--p-strong">
       结论
      </strong>
     </p>
     <p class="graf graf--p">
      通过使用各种工具，包括开源和协作制图应用，可以确定这两个清真寺在2018年期间经历了重大的结构变化。Keriya 清真寺的门楼和 Kargilik 清真寺的大部分遭到破坏。
     </p>
     <p class="graf graf--p">
      此前背景消息显示，新疆的穆斯林维吾尔族少数民族人口遭到有系统的镇压和监禁，而对文化和宗教十分重要的伊斯兰建筑物的破坏可能是这种持续镇压的一部分。
     </p>
     <p class="graf graf--p">
      商业制图服务和卫星图像服务将成为跟踪此类事件的重要工具，特别是对于难以进入的地区。
     </p>
