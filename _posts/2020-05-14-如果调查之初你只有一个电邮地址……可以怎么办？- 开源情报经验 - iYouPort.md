  <p class="graf graf--p">
   在调查开始的时候往往是你拥有的东西最少的时候。想象一下，在您的调查中有一个电子邮件地址：您知道该如何处理吗？
  </p>
  <p class="graf graf--p">
   有太多可用的开源情报工具，可能有些让人不知所措。这就需要经验了。例如，您使用哪些工具来验证电子邮件地址？以及如何检查电子邮件地址是否与社交媒体上的帐户相关联？—— 最好能找到，
   <strong class="markup--strong markup--p-strong">
    因为社交媒体是情报金矿，从哪里，追踪将最有可能直升式起飞
   </strong>
   。
  </p>
  <p class="graf graf--p">
   本文将 *概述* 调查电子邮件地址的步骤，希望能为您提供一些帮助。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     开源情报检查清单：电子邮件地址
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   以下是一些清单，每个清单都关注可能会提供给您的可用信息。例如，这可以是电子邮件地址、电话号码或数码照片。
  </p>
  <p class="graf graf--p">
   清单将指出您可以使用当前信息采取的调查步骤。下面将教您如何执行清单的所有调查步骤。以下是一些示例 —— 从最简单的开始。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11328 jetpack-lazy-image" data-lazy-sizes="(max-width: 616px) 100vw, 616px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-9.png?resize=616%2C871&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-9.png?w=616&amp;ssl=1 616w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-9.png?resize=212%2C300&amp;ssl=1 212w" data-recalc-dims="1" height="871" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-9.png?resize=616%2C871&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="616"/>

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     用搜索引擎
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   使用搜索引擎，您可以检查目标电子邮件地址是否存在于所谓的 “表层网络” 上。例如，当在Google上搜索某电子邮件时将向您显示包含该电子邮件地址的网站的结果。
  </p>
  <p class="graf graf--p">
   👉不同的搜索引擎的结果将有所不同，请检查多个搜索引擎，并使您选择的搜索引擎与目标相匹配（目标是来自哪里的？在当地哪个搜索引擎最热门，那就选它）。
  </p>
  <p class="graf graf--p">
   👉您还可以试试元搜索引擎和Carrot2之类的集群引擎。在下面看到工具包：
  </p>

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     搜索深网
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   当使用普通搜索引擎找不到该电子邮件地址时，并不一定意味着该电子邮件地址未在互联网上被使用。这是因为搜索引擎仅显示来自冰山一角的信息。您应该继续探索深网。
  </p>
  <p class="graf graf--p">
   请注意，网站管理员可以通过操纵搜索引擎的索引编制过程以不对其网站进行索引。可以通过在其网站上指定 robots.txt 文件来执行此操作。
  </p>
  <p class="graf graf--p">
   搜索引擎被禁止索引的所有信息都位于深网中。社交媒体通常被部分索引，但有时根本不被索引。
  </p>
  <p class="graf graf--p">
   那么如何搜索深网？搜索深网的两种方法是调查数据库和调查部分被索引或根本没有被索引的社交媒体。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     搜索社交媒体
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   如何执行此操作取决于您要查看的社交媒体平台是什么。
  </p>
  <p class="graf graf--p">
   在 Facebook 上，您可以使用搜索栏检查是否提到了目标电子邮件地址；在 Twitter 上，您也可以直接填写目标电子邮件地址。
  </p>
  <p class="graf graf--p">
   通过在搜索栏中输入目标电子邮件地址，您可以查找提到目标电子邮件地址的故事（在 Facebook上）和推文（在 Twitter 上）。
  </p>
  <p class="graf graf--p">
   ⚠️所以您应该小心 —— 再一次，必须严格区分可公开和私人。比如，如果有读者想要联系IYP，在推特上询问我们的邮箱，我们只会提供公开邮箱（您可以在网站上直接看到的邮箱），这是为避免被挖掘。
  </p>
  <p class="graf graf--p">
   👉请始终花几分钟时间思考您的目标。目标的年龄、出生地和其他详细信息，可能与您的调查重点非常相关。
  </p>
  <p class="graf graf--p">
   例如，Vkontakte，Nk.pl，Xing 或 Qzon e等社交媒体平台可能是您进行调查的地方；但请确保不要忘记如 Skype、YouTube 和 Spotify 这类平台。每个社交媒体平台都有其有效搜索信息的方式。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     搜索数据库
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   当使用搜索引擎进行搜索不会产生任何结果时，您可能希望查看一下（商用）数据库，例如 Pipl、Yasni 和 ThatsThem。
  </p>
  <p class="graf graf--p">
   👉这类数据库如何收集数据并不总是完全透明的，但是现有的数据可能很重要。请确保验证所有结果，因为信息可能已过时，甚至不正确。
  </p>
  <p class="graf graf--p">
   ⚠️如果您是防御者，请通过这些方法搜索自己，一旦检查到有重要信息包含其中，请迅速更改您的使用习惯。
  </p>
  <p class="graf graf--p">
   提示：使用 API 或付费许可时，某些数据库和服务可为您提供更好的结果。API的使用有时是免费的。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     搜索被黑的电子邮件地址
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   常规搜索引擎有时也可以向您显示被黑的帐户详细信息，例如电子邮件地址和密码。但是，有更好的方法来搜索被黑的电子邮件地址。
  </p>
  <p class="graf graf--p">
   👉可以使用的选择有
   <a class="markup--anchor markup--p-anchor" data-href="https://haveibeenpwned.com/" href="https://haveibeenpwned.com/" rel="noopener noreferrer" target="_blank">
    HaveIBeenPwned
   </a>
   ，
   <a class="markup--anchor markup--p-anchor" data-href="https://dehashed.com/" href="https://dehashed.com/" rel="noopener noreferrer" target="_blank">
    Dehashed
   </a>
   （付费）和
   <a class="markup--anchor markup--p-anchor" data-href="https://pastebin.com/" href="https://pastebin.com/" rel="noopener noreferrer" target="_blank">
    Pastebin
   </a>
   。这些网站将向您显示目标电子邮件地址是否因哪个违规行为而受到破坏。
  </p>
  <p class="graf graf--p">
   👉这使您可以洞悉目标可能使用的帐户和服务，使用的密码、以及是否有其他与该密码关联的电子邮件帐户。
  </p>
  <p class="graf graf--p">
   ⚠️作为防御者，请定期使用上述方法搜索自己，一旦发现泄漏，迅速更改凭据和相关绑定。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     电子邮件地址是否与社交媒体上的帐户相关联？

  </p>
  <p class="graf graf--p">
   👉使用电子邮件地址，可能会发现社交媒体上是否有与该电子邮件地址相关联的帐户。检查方法取决于您要调查的社交媒体平台。
