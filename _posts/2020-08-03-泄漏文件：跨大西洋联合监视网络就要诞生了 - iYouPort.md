  <p class="graf graf--p">
   西班牙的一名警探正试图解决犯罪问题，但她手边只有一张嫌疑犯的脸部图像，是被附近的监视摄像头捕获的。
  </p>
  <p class="graf graf--p">
   长期以来，欧洲警察一直可以访问欧盟27个国家的指纹和DNA数据库 —— 在某些情况下还包括美国的，但是不久之后，这些警察也许还可以搜索遍布整个欧洲和美国的面部数据库网络。
  </p>
  <p class="graf graf--p">
   根据泄露的欧盟内部文件，欧盟可能很快会建立一个国家警察面部识别数据库网络。由奥地利领导的10个欧盟成员国的国家警察部队起草的一份报告呼吁将此引入欧盟立法，以便在每个成员国都
   <strong class="markup--strong markup--p-strong">
    引入和互连此类监视数据库
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这份报告是从一位关注该监视网络发展的欧洲官员那里获得的，该报告于2019年11月在欧盟和各国官员中分发。
  </p>
  <p class="graf graf--p">
   而且这个新的面部识别监视网络可能会连接到美国的类似监视数据库，从而创造出隐私研究人员所说的：跨大西洋生物识别数据的大规模整合方法。
  </p>
  <p class="graf graf--p">
   该报告是关于扩展 Prüm 系统的讨论的一部分 ——
   <a class="markup--anchor markup--p-anchor" data-href="https://www.europarl.europa.eu/RegData/etudes/STUD/2018/604971/IPOL_STU(2018)604971_EN.pdf" href="https://www.europarl.europa.eu/RegData/etudes/STUD/2018/604971/IPOL_STU%282018%29604971_EN.pdf" rel="noopener noreferrer" target="_blank">
    Prüm 监视系统
   </a>
   是欧盟范围内的一项计划，
   <strong class="markup--strong markup--p-strong">
    旨在连接DNA、指纹、和车辆注册数据库，以进行更为广泛的监视搜索。
   </strong>
  </p>
  <p class="graf graf--p">
   美国与任何参与免签计划的国家/地区之间都存在类似的监视系统，其中包括大多数欧盟国家；双边协议允许美国和欧洲的机构访问彼此的指纹和DNA数据库。
  </p>
  <p class="graf graf--p">
   尽管尚未提出遵循该报告建议的新立法，但筹备工作仍在进行中。
  </p>
  <p class="graf graf--p">
   欧盟委员会去年11月向欧洲议会提供的信息显示，咨询公司德勤（Deloitte）将花费近700,000欧元（约合75万美元）以研究 Prüm 监视系统的可能变化，其中一部分专注于面部识别技术。
  </p>
  <p class="graf graf--p">
   欧盟委员会还分别向爱沙尼亚取证研究所牵头的一个公共机构支付了500,000欧元，以 “描绘出所有欧盟成员国在刑事调查中面部识别使用的现状”，旨在 “朝着可能交换面部数据的方向发展”，根据发送给布鲁塞尔国家代表的项目介绍中所说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这在国家层面和欧洲层面都是令人担忧的，
   <strong class="markup--strong markup--p-strong">
    特别是在一些欧盟国家转向更专制的政府的情况下
   </strong>
   ，” 国际隐私组织倡导总监 Edin Omanovic 说。
  </p>
  <p class="graf graf--p">
   Omanovic 担心泛欧洲生物识别数据库被用于 “出于政治动机的监视”，而不仅仅是标准的警察工作。
  </p>
  <p class="graf graf--p">
   大规模、不正当或非法监视的可能性是面部识别技术的许多批评之一。另一个是众所周知的错误率，特别是对于有色人种而言。
  </p>
  <p class="graf graf--p">
   Omanovic 说：“没有使面部识别技术合法的透明度和法律保障，就不应该使用它。”
  </p>
  <p class="graf graf--p">
   近年来，欧盟已采取重大步骤来连接大量移民管理和监视的数据库。去年4月通过的新立法建立了一个超级数据库，
   <strong class="markup--strong markup--p-strong">
    该数据库将存储多达3亿个 *非欧盟国家公民* 的指纹、面部图像、和其他个人数据，并合并了来自五个不同系统的数据。
   </strong>
  </p>
  <p class="graf graf--p">
   根据十个警察部队的报告，德勤咨询公司提议对警察的面部图像进行同样的处理，但是，这个想法遭到了执法人员的一致反对 ——
   <strong class="markup--strong markup--p-strong">
    警察有特权，监视只针对老百姓。
   </strong>
  </p>
  <p class="graf graf--p">
   尽管如此，该报告仍建议连接所有欧盟成员国的面部数据库，这似乎具有相同的实际效果。
  </p>
  <p class="graf graf--p">
   在另一份欧盟内部警察报告中（该报告来自 Prüm 工作组），警察指出 “相互连接的国家登记册网络可以看作是虚拟的全欧洲登记册。”
  </p>
  <p class="graf graf--p">
   对于警察而言，互连的面部识别数据库的优势显而易见。
  </p>
  <p class="graf graf--p">
   这份由奥地利领导的报告将这项监视技术视为识别未知嫌疑人的 “高度合适” 的生物统计工具，并建议应 “尽快” 创建和连接所有数据库。
  </p>
  <p class="graf graf--p">
   隐私专家指出，在数据保护控制措施严重不足的情况下，创建任何此类大规模监视系统都是迈向更大程度地监视的第一步。
  </p>
  <p class="graf graf--p">
   美国公民自由联盟高级立法顾问 Neema Singh Guliani 表示，欧洲整合警用面部识别数据的举动与美国的类似做法非常相似。许多美国执法机构在所谓的 “融合中心” 工作，他们能够广泛地共享数据。
  </p>
  <p class="graf graf--p">
   Guliani 说，如果您与 FBI 或国土安全部达成了信息共享协议，那么 “很可能也就此达成了与美国其他级别的执法部门的信息共享。”
  </p>
  <p class="graf graf--p">
   她补充说：“这提出了许多问题。警方如何在正当程序和《第一修正案》方面解释如何使用面部识别和收集图像；鉴于现有的信息共享关系，美国很可能希望访问这些信息。”
  </p>
  <p class="graf graf--p">
   早在
   <strong class="markup--strong markup--p-strong">
    2004年
   </strong>
   ，美国驻布鲁塞尔大使馆就呼吁与欧盟建立关系，以允许 “扩大交流并
   <strong class="markup--strong markup--p-strong">
    共享包括个人数据在内的所有形式的数据
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   近年来，朝着这个目标的努力已经明显加强。根据政府问责办公室的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.gao.gov/assets/680/676948.pdf" href="https://www.gao.gov/assets/680/676948.pdf" rel="noopener noreferrer" target="_blank">
    报告
   </a>
   ，2015年，国土安全部开始要求实施免签证计划国家/地区的数据共享协议。这包括联邦调查局向其他国家提供协助，以建立必要的计算机网络。
  </p>
  <p class="graf graf--p">
   奥地利刑事情报局高级官员 Reinhard Schmid 解释说，奥地利就是一个例子，该国已经于2017年10月开始根据FBI的刑事调查指纹数据库检查指纹。
  </p>
  <p class="graf graf--p">
   自那时以来，已经对大约 12000 个人的生物识别数据进行了交叉检查，共进行了150次监视。Schmid 说：“在这些被追踪的人中，大约有20人正在接受调查；在56起案件中，目标人试图使用虚假身份。
  </p>
  <p class="graf graf--p">
   Guliani 说：“他们的逻辑是，‘当我有一个罪案需要调查，并且想要在数据库中查找某人的照片时，为什么不应该拥有这个？”
  </p>
  <p class="graf graf--p">
   但是，她补充说，隐私隐患是巨大的。 “
   <strong class="markup--strong markup--p-strong">
    一旦获得访问权限，您最终就可以识别几乎任何地方的任何人
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   10个警察部队的报告呼吁欧盟警察情报和情报共享机构 Europol 在与 *非欧盟国家* 交换面部识别和其他生物识别数据方面发挥作用。
  </p>
  <p class="graf graf--p">
   这与欧洲各国政府本身的建议相呼应：
   <a class="markup--anchor markup--p-anchor" data-href="https://www.consilium.europa.eu/register/en/content/out?&amp;typ=ENTRY&amp;i=ADV&amp;DOC_ID=ST-11227-2018-INIT" href="https://www.consilium.europa.eu/register/en/content/out?&amp;typ=ENTRY&amp;i=ADV&amp;DOC_ID=ST-11227-2018-INIT" rel="noopener noreferrer" target="_blank">
    2018年7月
   </a>
   的一项
   <a class="markup--anchor markup--p-anchor" data-href="https://www.consilium.europa.eu/register/en/content/out?&amp;typ=ENTRY&amp;i=ADV&amp;DOC_ID=ST-11227-2018-INIT" href="https://www.consilium.europa.eu/register/en/content/out?&amp;typ=ENTRY&amp;i=ADV&amp;DOC_ID=ST-11227-2018-INIT" rel="noopener noreferrer" target="_blank">
    宣言呼吁
   </a>
   欧盟委员会考虑 “扩大 Prüm 监视网络的范围’’，并要求 Europol 率先与第三国共享数据。
  </p>
  <p class="graf graf--p">
   联邦调查局和欧洲刑警组织没有回答有关欧盟与美国之间数据共享协议的问题。欧盟委员会发言人承认将面部识别数据添加到了 Prüm 监视网络中的前景，但是拒绝透露更多细节。⚪️
  </p>
