   <strong>
    3D阿凡达扫描过程
   </strong>

  <div>
   今日的民主社會，隨著公民成長歷程與價值的多元化，如何透過溝通達成共感，成為一門日益重要的學問。對於政府官員及行政部門來說，更是如此。
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    在上個世紀，身為國家的領導者、政府官員與公務人員，有沒有足夠的「判斷力」與「專業能力」，是最重要的評價因素。但在今天，「建立共識的能力」，或許也同樣重要。這個能力的關鍵，不在於面對問題時「知道該怎麼做一定最好」，而是理解大家認為哪些問題是重要的，並且在眾人提出的事實、感受中，找出讓社會能夠相互理解、願意接受的解決方法。
   </span>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    與過去閱讀會議簡報，一起坐著開會、進行討論與決策的模式相比，ＶＲ情境有許多特性，能讓參與者獲得更直接的資訊交流與情感連結，我想分成六個面向來討論。

  <h3 data-usually-unique-id="155701955664537">
  </h3>
  <h4>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    1. 空間維度上的突破
   </span>
  </h4>
  <div>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    首先，傳統的開會方式，雖然是面對面溝通，但能夠運用的題材大多僅限於文字，照片與影像，都是二維樣態。然而，討論的主題卻往往是多面向的。這中間的差距，很容易讓彼此的理解出現落差。
   </span>
  </div>
  <div>
  </div>
  <div>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    舉例來說，如果今天的討論主題是一座社區公園，就目前一般的做法，討論的基礎可能是只有建築師看得懂的平面設計圖，也可能是幾張模擬示意簡報。除非參與者具有很強的空間感，否則難以比較各種方案的優劣。
   </span>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    因此，致力於「參與式規劃」的朋友們開發了許多方法，例如模型組合、現場放樣、集體創作、電子決策劇場，都是重要的傾聽方式。運用ＶＲ輔助這些流程的進行，不僅可以大幅節省成本、增加參與感，甚至連年紀很小的參與者，都可以提出符合使用需求的建議，也能讓討論者對主題的理解「具體化」，聚焦在共同的想像上。
   </span>
  </div>
  <div>
   <h4 data-usually-unique-id="106077398469319">
    <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
     2. 時間尺度上的突破
    </span>
   </h4>
  </div>
  <div>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    其次，傳統討論雖然可以錄音或錄影，但呈現的方式，還是只有單一的聲音檔，或是二維的影像重播。ＶＲ場域則可以用三維的方式，一次次存取不同的視角。
   </span>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    也就是說，ＶＲ能夠讓無法在「第一時間」到場的參與者，透過日後的「再進入」，產生身歷其境的感受、延長討論過程的感染力，讓更多的人進入相同的情境，進而有效形成共感。
   </span>
  </div>

     3. 解除距離的限制
    </span>
   </h4>
  </div>
  <div>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    ＶＲ可以為參與者省去前往實際地點的交通成本。虛擬空間的特性，不只能避免實體空間太小、太大等
   </span>
   <span class="author-d-iz88z86z86za0dz67zz78zz78zz74zz68zjz80zz71z9iz90za3mz84zrc9z86zz78z0z88zz78zz66zvz73zykvvz89z6z90zz78zmz80zdp9tz81zz79z">
    運
   </span>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    用上的障礙，更可以同時兼顧縱觀全球的視角，以及體驗在地的文化脈絡。
   </span>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    上述效果，只是ＶＲ在空間重組上的短期效益。長遠來看，ＶＲ的運用也會改變實體空間的配置。當越來越多重要的政府會議運用ＶＲ進行，每次實際討論的主題區域，就可以
   </span>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    <b>
     動態形成當下的「政治中心」
    </b>
   </span>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    ，而不是以往固定的中心、邊陲角色。
   </span>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    目前這方面的資通訊技術仍然處於萌芽階段，改變的過程也會需要數十年的持續對話、調適，但仍是一個值得觀察的後續發展。
   </span>
  </div>
  <div>
  </div>
  <div>
   <h4 data-usually-unique-id="183111442458868">
    <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
     4. 平等的聆聽空間
    </span>
   </h4>
  </div>
  <div>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    在ＶＲ場域裡，
   </span>
   <span class="thread-841642401712025 attrcomment attrcommentfirst thread-841642401712025-first author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    參與者都是以虛擬、但基本上一致的方式呈現
   </span>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    ，因此在彼此互動時，就可以減少因為性別、膚色、年齡等因素所造成的權力不對等。如此一來，引導者更能將參與者的注意力，集中在聆聽不同意見的「多元性」，而不是放大某些意見的「代表性」。
   </span>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    簡單來說，就是在保留數十萬年來面對面溝通、累積人際感受的前提下，
   </span>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    <b>
     解消「以貌取人」與「以人廢言」帶來的問題
    </b>
   </span>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    。

   <h4 data-usually-unique-id="982914400070810">
    <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
     5. 推廣產業發展
    </span>
   </h4>
  </div>
  <div>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    ＶＲ雖然是虛擬場域，但也有很實際的一面：在智慧型手機之後，ＶＲ被廣泛視為現階段應用科技突破的重點之一。
   </span>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    台灣做為ＶＲ設備的創製基地，以及數位內容的創造者，政府官員及相關部門，應該善用民間的貢獻。
   </span>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    這原本就是我近年來的研究主題，在十月上任成為數位政委之後，更是責無旁貸。
   </span>
  </div>
  <div>
  </div>
  <div>
   <h4 data-usually-unique-id="057520601798304">
    <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
     6. 嶄新的全球互動模式
    </span>
   </h4>
  </div>
  <div>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    最後，ＶＲ提供的「綜觀效應」，讓我們能跨越國界，從社會整體，乃至於全球的視角來看每個政策、每次行動的影響，這不但建立人與人互動的新場域，也同時創造出嶄新的協作模式。
   </span>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    傳統的國際場合，有嚴謹的組成與互動規則，但ＶＲ卻能提供更多開放與平等交流的機會，為我們與
   </span>
   <span class="author-d-4z65zz66zl57z75zyiz66zfr2fz87zwz89znujelz78zc1xz71zz77zz80zz68zhiz90z7z70zhz76zz77zhz73zz78z1z90zz84z5fgut">
    <b>
     世界各地的互動
