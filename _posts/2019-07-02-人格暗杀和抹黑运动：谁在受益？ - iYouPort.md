  <p class="graf graf--p">
   人格暗杀（CA）是一种蓄意的和持续的努力，目的是破坏个人的声誉或信誉，也经常选择性地应用于社会团体和机构。
  </p>
  <p class="graf graf--p">
   人格暗杀的操作者采用开放的和隐蔽的方法来实现目标，例如提出错误的指控、栽赃和煽动谣言，以及操纵信息。即便此后指控被证明是错误的也没什么关系，它借用了公众执迷于负面消息的宁信其有，并且煽动效果已经达到。这就是为什么说这手活非常黑。
  </p>
  <p class="graf graf--p">
   人格 暗杀是通过人格攻击实现的，可以采取多种形式，例如口头侮辱、演讲、流行小册子、竞选广告、漫画和互联网meme。
  </p>
  <p class="graf graf--p">
   人格攻击可以导致个人被其专业社区或其社会或文化环境的其他成员所拒绝，断送其发展前途。试想，
   <strong class="markup--strong markup--p-strong">
    如果一位知名的而且行动力卓越的活动家被当局声称其曾“秘密服务于政府（即为政府线人）”？
   </strong>
   不仅该活动家本人会被断送，而且很可能会因此捣毁一个组织和一场运动。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    CA 攻击如果不能有效防御可能会完全毁灭一个人，因为伤害可以持续一生。对于一些历史人物来说，这种损害已经持续了几个世纪
   </strong>
   。
  </p>
  <p class="graf graf--p">
   CA 可能涉及夸大、误导性的半真半假、或操纵事实以呈现目标人物的不真实画面。可以成为宣传战中的诉诸人身（Ad hominem）的一种形式。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “人格暗杀”这个概念从1930年左右开始流行，作为学术研究的主题；六十年后，Icks 和 Shiraev 通过解决和比较各种针对历史人物的人格暗杀事件，使这一术语重新焕发了活力并恢复了学术兴趣。[Martijn Icks &amp; Eric Shiraev (eds.), Character Assassination throughout the Ages (2014). New York: Palgrave and Macmillan]
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    CA 的基本原理 ——
   </strong>
  </p>
  <p class="graf graf--p">
   根据定义，人格暗杀是
   <strong class="markup--strong markup--p-strong">
    故意而为
   </strong>
   的：是为了破坏个人在他人眼中的声誉而发起的，于是不包括不知情的人无意中协助的传播；如果一个人的声誉被意外损坏，例如通过一种轻率的评论，那就不构成人格暗杀；由于人格攻击与声誉有关，因此它们也是
   <strong class="markup--strong markup--p-strong">
    公共性质
   </strong>
   的定义。私下侮辱某人并不会导致人格暗杀。
  </p>
  <p class="graf graf--p">
   每个人格暗杀行动总是涉及
   <strong class="markup--strong markup--p-strong">
    五个方面/支柱
   </strong>
   ：攻击者、目标、信息中介或媒体、观众和背景。最后一类“背景”可以指攻击发生的政治系统、文化环境、技术水平、或影响和确定人格暗杀的任何其他因素。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    许多人格攻击发生在政治领域，例如在竞选活动中针对竞选对手、或在政治竞争中针对社会活动家及异议人士
   </strong>
   。然而，其他领域的知名人物也很可能成为人格攻击的目标，如 宗教领袖、科学家、运动员、电影明星。
  </p>
  <p class="graf graf--p">
   之所以称为“暗杀”有双重含义，其一是它造成的真实后果，即 如杀戮一般可以彻底毁掉一个人/意识形态和主张或一场革命；其二它发生于真正的暗杀不可行或不足以摧毁目标的时候，比如杀死一名调查记者，反而会激励其勇敢的同行继续遇害者的工作，并同时强调了该记者所涉及的调查之重要性。相反，如果采取人格暗杀，即 栽赃该调查记者与敌国间谍私通，很可能就此摧毁其调查价值，导致无人敢继续这项工作。也就是说，人格暗杀有可能比见血的杀戮效果更高。
  </p>
  <p class="graf graf--p">
   另个角度看，人格暗杀不会很轻易使用，其攻击目标对攻击者来说往往都具有足够高的价值。
  </p>
  <p class="graf graf--p">
   此外，人格暗杀似乎是一种近乎普遍的、跨文化的现象，可以在大多数国家和不同历史时期找到。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    对 CA 的研究 ——
   </strong>
  </p>
  <p class="graf graf--p">
   Icks 和 Shiraev（2014）讨论了几种政治科学模型，以从攻击者的角度解释CA的原因。他们认为，攻击者的动机往往是基于在心理上摧毁目标、或减少他/她的公众支持、和/或在政治竞争中取得成功的机会。
  </p>
  <p class="graf graf--p">
   例如，在选举期间，
   <strong class="markup--strong markup--p-strong">
    这种攻击通常被用来影响犹豫不决的选民、给坚定的选民制造不确定性，或防止支持者背叛
   </strong>
   。因此，这些攻击成为操纵选民迈向攻击者理想之行动的有效手段。它们还有助于将原本有利的支持者重新定位到“犹豫不决”或“放弃”的行列。
  </p>
  <p class="graf graf--p">
   从根本上说，对一个人的形象、声誉或品牌的攻击是否成功取决于相关受众的看法。
  </p>
  <p class="graf graf--p">
   例如，动机推理领域的研究表明，消费者会更多选择“他们认为可信”的信息，更愿意接受与自身现有的态度、期望或行为最为一致的内容，例如候选人的投票记录。而“混合”处理模型表明选民使用两种类型的信息来构建他们的候选人印象或回应 CA：即 更新的和诉诸人身的。
   <strong class="markup--strong markup--p-strong">
    焦虑、或不确定性和紧张的情绪状态，都可以影响人的态度转变。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当组织和领导人发现自己陷入危机时，他们就特别容易受到针对他们的合法性或社会责任的攻击
   </strong>
   。然后，他们的声誉会在舆论法庭中进行评判，舆论关注的是公开定位的原则，包括道德、社会和政治价值观，或文化、宗教信仰。
  </p>
  <p class="graf graf--p">
   情境危机传播理论（SCCT）在组织和政治背景下特别适用。该理论认为，声誉威胁的程度取决于公众*是否认为*组织引发了危机、组织的危机历史以及组织与公众之间的关系声誉（如选民、利益相关者等）。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    表征人格攻击 ——
   </strong>
  </p>
  <p class="graf graf--p">
   可以根据各种标准对人格攻击进行分类。当我们根据
   <strong class="markup--strong markup--p-strong">
    层次结构
   </strong>
   考虑它们时，可以将其区分为水平和垂直攻击。
  </p>
  <p class="graf graf--p">
   前者指的是攻击者和目标拥有大致相同的权力和资源（例如一名总统候选人攻击另一名）的情况，后者是指权力和资源存在巨大差异的情况（例如独裁者攻击反对派，反之亦然）。
  </p>
  <p class="graf graf--p">
   如果考虑
   <strong class="markup--strong markup--p-strong">
    时机
   </strong>
   的话，可以根据目标在被攻击时是否还活着来区分实时攻击和死后攻击。死后攻击是为诋毁死者的政治价值和意识形态，以削弱其追随者的斗志。
  </p>
  <p class="graf graf--p">
   在政治竞选期间经常会发生实时的/瞬时的攻击。这种快速人格攻击通常是机会主义的。另一方面，人格中毒的缓慢操作是基于长效的愿景。例如自20世纪60年代以来，著名的俄罗斯作家和持不同政见者亚历山大索尔仁尼琴就被指控为叛徒、纳粹合作者、监狱小飞贼和有偿的外国情报人员等污名。
  </p>
  <p class="graf graf--p">
   根据 Thomas 的说法，人格暗杀是一种故意的尝试，通常是由自恋者和/或其代理人，以某种方式影响某人的写照或声誉，从而导致他人对目标人产生极端消极或排斥性的看法。它通常涉及故意夸大或操纵事实，传播谣言和蓄意的错误信息，以呈现目标人物的不真实情况，以及毫无根据和过度的批评。[Thomas, D (2010), Narcissism: Behind the Mask]
  </p>
  <p class="graf graf--p">
   《Snakes in Suits: When Psychopaths Go to Work》这本书的作者描述了一个在典型的工作场所如何攀登并维持权力的五阶段模型。在第四阶段（对抗）中，使用的就是人格暗杀技术来维持他们的议程。[Baibak, P;
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Robert_D._Hare" href="https://en.wikipedia.org/wiki/Robert_D._Hare" rel="noopener noreferrer" target="_blank" title="Robert D. Hare">
    Hare, R. D
   </a>
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Snakes_in_Suits:_When_Psychopaths_Go_to_Work" href="https://en.wikipedia.org/wiki/Snakes_in_Suits:_When_Psychopaths_Go_to_Work" rel="noopener noreferrer" target="_blank" title="Snakes in Suits: When Psychopaths Go to Work">
    Snakes in Suits: When Psychopaths Go to Work
   </a>
   (2007)]
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在政治方面 ——
   </strong>
  </p>
  <p class="graf graf--p">
   在政治方面，CA通常是政治“诽谤运动”的一部分，涉及故意的、有预谋的努力，以破坏个人或团体的声誉和信誉。此类活动的目的是阻止或削弱目标的支持基础。另一个目的是迫使目标在时间、能量和资源方面做出反应。
  </p>
  <p class="graf graf--p">
   CA也是一种消极竞选形式，而且是最常用的形式，被委婉地称之为
   <strong class="markup--strong markup--p-strong">
    “反对派研究“
   </strong>
   ，全部工作就是收集尽可能多的可用于诋毁性操作的信息的做法。抹黑运动是使用虚假或扭曲的信息进行宣传（后面详细介绍）；丑闻可以被用来以虚假或夸大的方式将一个人与一个主要负面事件联系起来。
  </p>
  <p class="graf graf--p">
   通常包括以无法辨认的谣言和歪曲、半真半假、甚至是彻头彻尾的谎言形式的广告攻击；诽谤运动通常由八卦杂志和网站传播。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    即使抹黑运动背后的事实已被证明缺乏适当的基础，这种策略通常也依旧是有效的，因为无论事实如何，目标的声誉已经受到了损害，极难修复
   </strong>
   。
  </p>
  <p class="graf graf--p">
   而且，抹黑也可以非常有效地将目标的注意力从重点信息中转移开。被抹黑的目标必须专注于解决纠正错误信息的问题，而不是专注于他们对原始问题的回应。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在极权主义政权中 ——
   </strong>
  </p>
  <p class="graf graf--p">
   个人驱动的人格暗杀的效果不等于国家驱动的。
   <strong class="markup--strong markup--p-strong">
    由大规模政治宣传和文化机制促成的国家支持的声誉破坏运动可能产生更深远的影响
   </strong>
   。
  </p>
  <p class="graf graf--p">
   一个政府支持或直接鼓励旨在摧毁其对手的尊严和声誉的运动、一个社会默许犯罪（甚至是大屠杀罪犯）完全不受惩罚的最早迹象之一就是人格暗杀。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    动员破坏对手声誉是动员暴力以歼灭对方的前奏；宣传战的舆论攻击都会发生在实质性的身体攻击之前
   </strong>
   。
  </p>
  <p class="graf graf--p">
   研究机构 ——
  </p>
  <p class="graf graf--p">
   国际人格暗杀研究学会（ISSCA）专门研究如何在历史和当代时期执行人格攻击和人格暗杀。
  </p>
  <p class="graf graf--p">
   2011年7月，来自9个国家的学者聚集在德国海德堡大学，讨论“历史和当下的抹黑运动和诽谤艺术”。他们组建了一个小组来研究各个时代的人格暗杀事件。该小组包括历史学家、政治学家和政治心理学家。[
   <a class="markup--anchor markup--p-anchor" data-href="https://characterattack.files.wordpress.com/2012/01/2011-schedule.pdf" href="https://characterattack.files.wordpress.com/2012/01/2011-schedule.pdf" rel="nofollow noopener noreferrer" target="_blank">
    “Character Assassination: The Art of Defamation Throughout the Ages. International Colloquium Heidelberg July 21st — July 23rd, 2011”
   </a>
   (PDF).]
  </p>
  <p class="graf graf--p">
   人格暗杀和声誉政治研究实验室（CARP）成立于2016年，与ISSCA合作，包括心理学、历史、传播和公共关系学科的学者。CARP 团队将工作重点放在三个方面：研究历史和当代人格暗杀的案例；为学术和公众提供有关人格暗杀原因、影响和预防的教育；风险评估，以确定关注其声誉的公众人物的脆弱性和缓解策略。
  </p>
  <p class="graf graf--p">
   Mason CARP 网站提供有关实验室及其活动的资料。 CARP实验室还发布了一个博客，并与全球信息项目联系起来，该项目是世界公开秘密、不成文规则和隐藏实践的领先在线资源。

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    —— 抹黑运动 ——
   </strong>
  </p>
  <p class="graf graf--p">
   抹黑是抹黑，抹黑运动是一场运动。这场运动可以非常可怕，因为它
   <strong class="markup--strong markup--p-strong">
    动员社会公信力高的组织和团体共同陈述一个虚假的“事实”，尤其是主流媒体的参与，其结果将有可能等同于伪真相
   </strong>
   。（在这里看到我们曾经对伪真相的分析《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ad%a6%e6%83%95%e5%be%ae%e5%a6%99%e7%9a%84%e5%8a%9d%e5%af%bc%e6%9c%af%ef%bc%9a%e4%bc%aa%e7%9c%9f%e7%9b%b8-%e6%96%b0%e9%97%bb%e6%9c%ba%e6%9e%84%e5%92%8c%e8%af%bb%e8%80%85%e7%9a%86/" href="https://www.iyouport.org/%e8%ad%a6%e6%83%95%e5%be%ae%e5%a6%99%e7%9a%84%e5%8a%9d%e5%af%bc%e6%9c%af%ef%bc%9a%e4%bc%aa%e7%9c%9f%e7%9b%b8-%e6%96%b0%e9%97%bb%e6%9c%ba%e6%9e%84%e5%92%8c%e8%af%bb%e8%80%85%e7%9a%86/" rel="noopener noreferrer" target="_blank">
    警惕微妙的劝导术：伪真相 — — 新闻机构和读者皆需谨慎
   </a>
   》）
  </p>
  <p class="graf graf--p">
   抹黑运动的共同目标是公职人员、政治家、政治候选人、公民活动家和前配偶。该术语也适用于其他环境，如工作场所。
  </p>
  <p class="graf graf--p">
   抹黑运动一词在1936年左右开始流行。[
   <a class="markup--anchor markup--p-anchor" data-href="https://books.google.com/ngrams/graph?content=Smear+campaign&amp;case_insensitive=on&amp;year_start=1900&amp;year_end=2008&amp;corpus=15&amp;smoothing=3&amp;share=&amp;direct_url=t4;,Smear+campaign;,c0;,s0;;smear+campaign;,c0;;Smear+Campaign;,c0" href="https://books.google.com/ngrams/graph?content=Smear+campaign&amp;case_insensitive=on&amp;year_start=1900&amp;year_end=2008&amp;corpus=15&amp;smoothing=3&amp;share=&amp;direct_url=t4;,Smear+campaign;,c0;,s0;;smear+campaign;,c0;;Smear+Campaign;,c0" rel="nofollow noopener noreferrer" target="_blank">
    “Google Ngram Viewer”
   </a>
   。
   <em class="markup--em markup--p-em">
    books.google.com
   </em>
   ]
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    定义 ——
   </strong>
  </p>
  <p class="graf graf--p">
   抹黑运动是故意的、有预谋的努力，目的是破坏个人或团体的声誉、信誉和品格。与消极竞选活动一样，大多数情况下都是针对政治家、政治候选人和其他公众人物进行的诽谤。但是，私人或团体也可能成为公司、机构、法律系统和其他正式团体中的诽谤运动的目标。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    抹黑策略与正常的讨论或辩论完全不同
   </strong>
   ，因为抹黑不涉及有关的问题或论点。抹黑是一种简单的企图恶化一个群体或个人的努力，目的是破坏他们的可信度。它利用的是公众对奇闻/花边新闻的巨大兴趣。
  </p>
  <p class="graf graf--p">
   即使抹黑运动背后的事实被证明缺乏适当的基础，这种策略通常也是很有效的，因为目标的声誉在事实已知之前就已经被损害了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    抹黑也可以有效地将公众的注意力从有关问题转移到特定的个人或群体身上。被抹黑的目标通常必须集中于纠正错误信息而不是原始问题
   </strong>
   。
  </p>
  <p class="graf graf--p">
   总体过程就是这样的：你虚构了一些东西，然后你想办法让主流媒体陈述你虚构的东西，由于虚构的刺激性有很大的流量吸引力，其他主流媒体和小媒体会纷纷引述这一“热点”，你的虚构会被迅速推高，然后你就会看到，几乎所有人都相信这是真的。
  </p>
  <p class="graf graf--p">
   许多人认为抹黑策略是低级的、虚伪的话语形式；然而它们非常常见，甚至非常有效。
  </p>
  <p class="graf graf--p">
   在许多国家，法律承认声誉和信誉的价值。构陷（虚假和破坏性出版物）和诽谤（虚假和破坏性的口头陈述）通常会受到法律惩罚，并可能导致监禁或损害赔偿。但是，如果抹黑运动是由当权者操作的，他们就有办法令其操纵不被证明为诽谤。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    抹黑运动案例 ——
   </strong>
  </p>
  <p class="graf graf--p">
   1、JohnC.Frémont — 1856年美国总统大选候选人
  </p>
  <p class="graf graf--p">
   在1856年的总统大选期间，JohnC.Frémont 成为诽谤运动的目标，攻击者声称他是天主教徒。该运动旨在破坏对那些怀疑天主教的人对 Frémont 的支持。
  </p>
  <p class="graf graf--p">
   2、通用汽车公司对阵 Ralph Nader
  </p>
  <p class="graf graf--p">
   20世纪60年代，拉尔夫·纳德（Ralph Nader）成为了诽谤运动的受害者，当时他正在为汽车安全而竞选。为了抹黑 Nader 并将公众的注意力从他的竞选活动中转移出来，通用汽车公司聘请了私人调查员搜寻他过去的破坏性或尴尬的事件。
  </p>
  <p class="graf graf--p">
   1966年3月初，多家媒体报道称，通用汽车曾试图诋毁 Nader，
   <strong class="markup--strong markup--p-strong">
    聘请私人侦探窃听他的手机并调查他的过去，甚至雇佣妓女以诱骗他。
   </strong>
  </p>
  <p class="graf graf--p">
   3、Gary Hart  — 1988年美国总统候选人
  </p>
  <p class="graf graf--p">
   Gary Hart 是1988年美国总统竞选期间抹黑运动的目标。这次是与黄色新闻相关的竞选策略，黄色新闻是这样一种新闻报道，
   <strong class="markup--strong markup--p-strong">
    它只提供很少的研究，相反，它使用引人注目的头条、丑闻和耸人听闻来吸引公众注意力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   例如，在 Gary Hart 1988年总统竞选期间，“纽约邮报”在头版贴了一串超大的黑色字母：“GARY: I’M NO WOMANIZER”[William Safire. (May 3, 1987).

    “On Language; Vamping Till Ready”
   </a>

   <em class="markup--em markup--p-em">

  </p>

  </p>

   <strong class="markup--strong markup--p-strong">
    —— 为什么更多人格暗杀刺客不会被起诉？——
   </strong>
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    支持或反对一场人格攻击不应该取决于你我对被攻击对象的印象如何。遗憾的是，大多数人难以做到这点。
   </p>
  </blockquote>
  <p class="graf graf--p">
   对人格暗杀的广泛研究发现的第一个悲催的事实就是，这种“传播策略”在各个时代的持久力。
  </p>
  <p class="graf graf--p">
   正如
   <a class="markup--anchor markup--p-anchor" data-href="https://carplab.wordpress.com/2019/04/12/character-assassination-the-historical-perspective-part-2/" href="https://carplab.wordpress.com/2019/04/12/character-assassination-the-historical-perspective-part-2/" rel="noopener noreferrer" target="_blank">
    Martijn Icks 的作品
   </a>
   所表明的那样，人格攻击甚至可以追溯到古代雅典的民主先辈们。我们还看到了针对美国历史上以及整个西欧历史上每位总统的人格暗杀企图。事实上，在搜索过的所有历史时期和背景中，都找到了使用人格攻击的例子。
  </p>
  <p class="graf graf--p">
   民主政府似乎有一些东西可以使人格暗杀成为一种特别有用的策略（当然，众所周知专制政权也使用人格暗杀的策略，经常让目标几乎无法清除）。
  </p>
  <p class="graf graf--p">
   的确，当选举代表为国会的利益服务时，我们这些普通人的时间和注意力都很有限。
   <strong class="markup--strong markup--p-strong">
    选民可能会使用诸如真诚度和讨喜之类的捷径来做出决策。然而这两个人格元素都是人格攻击的常见目标，特别是以虚伪的指责形式
   </strong>
   。事实上，“真诚度” 已经
   <a class="markup--anchor markup--p-anchor" data-href="https://www.vox.com/2019/3/25/18263768/2020-democrats-presidential-primary-authentic-beto-harris" href="https://www.vox.com/2019/3/25/18263768/2020-democrats-presidential-primary-authentic-beto-harris" rel="noopener noreferrer" target="_blank">
    成为 2020年大选的基石
   </a>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    至少在政治上，一个值得注意的关于人格暗杀的事实是，在遭受袭击之后，公众人物很少能够通过起诉要求赔偿。
   </strong>
   简而言之，新闻自由的民主标志及其中的宪法保护使得成功起诉一个人格暗杀案件变得非常困难。
  </p>
  <p class="graf graf--p">
   4月，来自加利福尼亚州的共和党代表德文·努涅斯（Devin Nunes）起诉弗雷斯诺蜂（Fresno Bee）进行人格攻击时，人格暗杀与法律之间的关系就在新闻中表现出来。
  </p>
  <p class="graf graf--p">
   Nunes 反对一个描述旧金山湾邮轮的报道，该邮轮以毒品和卖淫为特色，由他部分拥有的一家酒厂的投资者参与。该报道指出，Nunes 是否知道该事件是“不清楚”的。Nunes 说他不了解该游轮，并且媒体在文章中加入他的名字是出于政治动机的原因“摧毁他的声誉”的计划。
  </p>
  <p class="graf graf--p">
   Nunes 不太可能获胜，因为宪法要求“保护新闻自由”，除非公职人员提起的诽谤诉讼能证明媒体采取“实际恶意”和“鲁莽无视真相”的行为。这是非常常见的。其中最引人注目的是1964年纽约时报诉沙利文案。根据最高法院的解释，该法律为公职人员和公众人物制​​定了单独的标准，表明保护私人声誉的优先级高于第一修正案所涉及的公共声誉。
  </p>
  <p class="graf graf--p">
   想象一下，如果媒体被权势绑架而针对揭露真相的人或政治反对派发起人格攻击，不论如何被攻击者都很难申诉。
  </p>
  <p class="graf graf--p">
   简单说，只要举起“新闻自由”的大旗，被伤害的公众人物最多只能抱怨攻击者，而无法通过法律渠道进行申诉。第一修正案总是需要在竞争性利益（言论自由和保护声誉）之间进行微妙的平衡，私人和公众人物之间的区别是法院决定平衡这些利益的一个重要方式。
  </p>
  <p class="graf graf--p">
   每个人都同意民主取决于能否批评我们的领导人，并且很明显，人格暗杀通常在此是受保护的部分。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    —— 2019年 ——
   </strong>
  </p>
  <p class="graf graf--p">
   这里以美国为例。事实上在各个国家都一样，因为如今几乎所有国家都在互联网上。
  </p>
  <p class="graf graf--p">
   2019年，公众对政治竞争者的支持将
   <strong class="markup--strong markup--p-strong">
    主要由党派认同决定
   </strong>
   ，而不是被认为具有品格优势，如 能力或可信度。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    美国社会日益扩大的两极分化培养了
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amazon.com/Political-Tribes-Group-Instinct-Nations/dp/0399562850/ref=sr_1_2?ie=UTF8&amp;qid=1547404990&amp;sr=8-2&amp;keywords=tribalism" href="https://www.amazon.com/Political-Tribes-Group-Instinct-Nations/dp/0399562850/ref=sr_1_2?ie=UTF8&amp;qid=1547404990&amp;sr=8-2&amp;keywords=tribalism" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     政治和文化的*部落主义*
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，部落主义表现为一种催产素效应：支持对不同意见的不容忍、赞美党派和对政党议程的忠诚
   </strong>
   。在2018年最高法院提名人 Brett Kavanaugh 的听证会上，候选人在其指定职位上的价值辩论完全被他的支持者和反对者之间的意识形态斗争所取代。这只是数不胜数的同类案例之一。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    偏见和不文明将使政治家、个体企业家和媒体受益
   </strong>
   ：
  </p>
  <p class="graf graf--p">
   尽管“人格”的相关性地位在下降，但是并没有证据表明人格攻击的数量会减少。
   <strong class="markup--strong markup--p-strong">
    在尖锐的意识形态分裂的推动下，政治派别将继续将人格暗杀作为社会说服的极端手段
   </strong>
   。
   <strong class="markup--strong markup--p-strong">
    通过收集攻击性材料并随后勒索公众人物，个体企业家也将从持续的政治丑闻中受益
   </strong>
   。
  </p>
  <p class="graf graf--p">
   然而，像特朗普这样的知名人士的公众利益将保持高位，因此
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wilsoncenter.org/blog-post/trump-bump-and-trump-filter-bubble-how-the-medias-obsession-trump-may-encourage-the" href="https://www.wilsoncenter.org/blog-post/trump-bump-and-trump-filter-bubble-how-the-medias-obsession-trump-may-encourage-the" rel="noopener noreferrer" target="_blank">
    主流媒体
   </a>
   通过吸引人的负面报道以吸引流量
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wilsoncenter.org/blog-post/trump-bump-and-trump-filter-bubble-how-the-medias-obsession-trump-may-encourage-the" href="https://www.wilsoncenter.org/blog-post/trump-bump-and-trump-filter-bubble-how-the-medias-obsession-trump-may-encourage-the" rel="noopener noreferrer" target="_blank">
    不断将其货币化
   </a>
   （我们分析过特朗普如何在事实上“帮助”了媒体的盈利）；同样，由于经济上的迫切需要，互联网公司也对公民语话和频繁的CA活动的不断增长做出了更多的*贡献* ——越刺激越火。
  </p>
  <p class="graf graf--p">
   根据 Zeynep Tufekci 最近的研究，社交媒体算法直接导致了政治极端主义的兴起。她在2017年出版的书
   <strong class="markup--strong markup--p-strong">
    《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.twitterandteargas.org/" href="https://www.twitterandteargas.org/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Twitter 和催泪瓦斯
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》证明了只要你能不断告诉人们刺激性的内容，人们就会花更多的时间留在网站上。因此，YouTube 的算法实际上是为了激励用户而设计的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   关于 Google 的不透明算法如何操纵选举，详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E7%9C%8B%E4%B8%8D%E8%A7%81%E7%9A%84%E7%8B%AC%E8%A3%81/" href="https://www.iyouport.org/%E7%9C%8B%E4%B8%8D%E8%A7%81%E7%9A%84%E7%8B%AC%E8%A3%81/" rel="noopener noreferrer" target="_blank">
    看不见的独裁
   </a>
   》
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    技术公司将加大力量实施在线审查
   </strong>
   ：
  </p>
  <p class="graf graf--p">
   围绕所谓的“虚假新闻”的歇斯底里使公司更有借口以极权主义的方式应用新算法进行审查。2018年10月，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/technology/2018/oct/16/facebook-political-activism-pages-inauthentic-behavior-censorship" href="https://www.theguardian.com/technology/2018/oct/16/facebook-political-activism-pages-inauthentic-behavior-censorship" rel="noopener noreferrer" target="_blank">
    Facebook 宣布它已经
   </a>
   删除了800多个政治主页和账户。
  </p>
  <p class="graf graf--p">
   监视资本主义崛起，今天的平台会大量监控其用户。所谓的“分散监控”，公司通过
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%E5%85%B3%E4%BA%8E%E6%99%BA%E8%83%BD%E9%97%B4%E8%B0%8D%E5%AE%B6%E5%B1%85-%E5%9F%8E%E5%B8%82/" href="https://www.iyouport.org/category/%E5%85%B3%E4%BA%8E%E6%99%BA%E8%83%BD%E9%97%B4%E8%B0%8D%E5%AE%B6%E5%B1%85-%E5%9F%8E%E5%B8%82/" rel="noopener noreferrer" target="_blank">
    智能家居设备
   </a>
   秘密收集有关公民私生活的信息。最重要的是，互联网公司和情报机构现在可以访问大量的个人数据，
   <strong class="markup--strong markup--p-strong">
    包括泄露可能
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://carplab.wordpress.com/2018/02/14/big-data/" href="https://carplab.wordpress.com/2018/02/14/big-data/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     用于勒索下一代政治和商业领袖的信息
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   趋势 ——
  </p>
  <p class="graf graf--p">
   治理术是由法国哲学家米歇尔·福柯（Michel Foucault）在20世纪70年代末制定的一个术语。它强调通过受治理者的自愿参与来管理人们的行为。
   <strong class="markup--strong markup--p-strong">
    众包意见可以用作象征性暴力的手段，并对社会地位和行使个人代理的能力产生严重影响。
   </strong>
  </p>
  <p class="graf graf--p">
   根据 J. Walter Thompson Intelligence 的Future 100
   <a class="markup--anchor markup--p-anchor" data-href="https://www.jwtintelligence.com/trend-reports/the-future-100-2019/" href="https://www.jwtintelligence.com/trend-reports/the-future-100-2019/" rel="noopener noreferrer" target="_blank">
    趋势报告
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    在线影响从购物到道德制裁的所有内容，包括他们所代表的个人和团体
   </strong>
   。例如，优步正在加强这种审查文化，根据他们的评级惩罚或奖励司机和骑手。
  </p>
  <p class="graf graf--p">
   在中国，当局采用了福柯式的全景扫描实践，通过对其公民进行“社会评分”来加强更严格的控制。因此，新的“社会信用系统”根据金融交易和日常行为计算每个公民的分数。公民可能会因分数而受到惩罚，被禁止上学、乘坐公共交通工具，或因低分而被拒绝晋升。
  </p>
  <p class="graf graf--p">
   在我看来，
   <strong class="markup--strong markup--p-strong">
    与声誉损害相关的风险将在2019年变得更加突出，因为个人和团体将面临更少的选择和过时的能力难以管理他们的声誉
   </strong>
   。
  </p>
  <p class="graf graf--p">
   除了像特朗普这样的民粹主义领导人之外，公众人物将非常谨慎地调整他们在公共领域提出意见和真正价值观的方式。战略性模糊和印象管理的成熟对策将受到高度需求，以及，公共关系专家和诽谤律师的服务。
  </p>
  <p class="graf graf--p">
   如果您和您的组织正在尝试挑战主流政治，请记住本文中提供的知识。好运。⚪️
  </p>
