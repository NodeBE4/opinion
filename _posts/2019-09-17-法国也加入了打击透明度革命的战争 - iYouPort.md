  <h1 class="entry-title">
   法国也加入了打击透明度革命的战争
  </h1>
 </header>
 <div class="entry-meta">
  <span class="byline">
   <a href="https://www.iyouport.org/author/don-evans/" rel="author" title="由McCaffrey发布">

    关于透明度革命
   </a>
   ,
   <a href="https://www.iyouport.org/category/%e5%aa%92%e4%bd%93%e5%92%8c%e6%96%b0%e9%97%bb%e8%87%aa%e7%94%b1/" rel="category tag">

 <div class="entry-content">
  <ul>
   <li class="graf graf--p">
    <span style="color: #00ccff;">
     <em>
      <strong>
       与职业记者合作的透明度革命似乎有点问题了。
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   举报人可以通过技术加以保护，可以做到始终隐藏身份；但是在职记者都是在明处的，事实显示当权者真的可以完全无视宪法和法律对新闻自由的承诺。
  </p>
  <p class="graf graf--p">
   已经不仅美国了，现在就连曾经的新闻自由偶像 — 法国 — 也在试图杀死透明度革命。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E6%9C%80%E4%BD%B3%E5%BA%87%E6%8A%A4%E6%89%80%E5%92%8C%E8%B7%A8%E5%9B%BD%E7%BB%B4%E7%A8%B3-%E9%80%8F%E6%98%8E%E5%BA%A6%E9%9D%A9%E5%91%BD%E7%9A%84%E6%9C%AA%E6%9D%A5/" href="https://www.iyouport.org/%E6%9C%80%E4%BD%B3%E5%BA%87%E6%8A%A4%E6%89%80%E5%92%8C%E8%B7%A8%E5%9B%BD%E7%BB%B4%E7%A8%B3-%E9%80%8F%E6%98%8E%E5%BA%A6%E9%9D%A9%E5%91%BD%E7%9A%84%E6%9C%AA%E6%9D%A5/" rel="noopener noreferrer" target="_blank">
    John Young
   </a>
   当年不与任何职业媒体合作（当然不仅是担心记者的安全，他压根就不信任他们，关于原因可见这篇KN《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%ae%88%e9%97%a8%e4%ba%ba%e4%bd%9c%e4%b8%ba%e8%ae%a4%e7%9f%a5%e6%9d%80%e6%89%8b/" href="https://www.iyouport.org/%e5%ae%88%e9%97%a8%e4%ba%ba%e4%bd%9c%e4%b8%ba%e8%ae%a4%e7%9f%a5%e6%9d%80%e6%89%8b/" rel="noopener noreferrer" target="_blank">
    守门人作为认知杀手
   </a>
   》）。
  </p>
  <p class="graf graf--p">
   Young 自建网站，只是纯粹地将文件扔出来，
   <strong class="markup--strong markup--p-strong">
    让证据自己说话
   </strong>
   —— 就如我们在《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e7%9f%a5%e9%81%93%e4%bb%80%e4%b9%88%e6%89%8d%e6%98%af%e7%9c%9f%e6%ad%a3%e7%9a%84%e8%b0%83%e6%9f%a5%e5%90%97%ef%bc%9f%e6%89%80%e8%b0%93%e7%9a%84%e8%b5%b7%e5%ba%95%e7%bb%8f%e5%b8%b8/" href="https://www.iyouport.org/%e4%bd%a0%e7%9f%a5%e9%81%93%e4%bb%80%e4%b9%88%e6%89%8d%e6%98%af%e7%9c%9f%e6%ad%a3%e7%9a%84%e8%b0%83%e6%9f%a5%e5%90%97%ef%bc%9f%e6%89%80%e8%b0%93%e7%9a%84%e8%b5%b7%e5%ba%95%e7%bb%8f%e5%b8%b8/" rel="noopener noreferrer" target="_blank">
    什么是调查
   </a>
   》中解释的那样。他一个人承担了一切风险；Young 被很多人视为激进派，主要是因为他可以完全无视现实中的路障。
  </p>
  <p class="graf graf--p">
   这一路障就是：主流媒体占据了90%的认知市场；如果你希望具有公共利益信息迅速被大众所接收 —— 也就是说抢在当局开始打压之前实现它的价值，那么你只能与主流媒体合作……
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4374 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/法国.jpg?resize=1100%2C550&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/法国.jpg?w=1440&amp;ssl=1 1440w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/法国.jpg?resize=300%2C150&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/法国.jpg?resize=768%2C384&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/法国.jpg?resize=1024%2C512&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/法国.jpg?resize=1100%2C550&amp;ssl=1 1100w" data-recalc-dims="1" height="550" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/法国.jpg?resize=1100%2C550&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4374" data-recalc-dims="1" height="550" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/法国.jpg?resize=1100%2C550&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/法国.jpg?w=1440&amp;ssl=1 1440w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/法国.jpg?resize=300%2C150&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/法国.jpg?resize=768%2C384&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/法国.jpg?resize=1024%2C512&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/法国.jpg?resize=1100%2C550&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   法国新闻通讯社在处理秘密文件方面遇到了前所未有的困境，这些文件详细说明了该国参与也门暴行的惊人内幕。
  </p>
  <p class="graf graf--p">
   结果是，
   <strong class="markup--strong markup--p-strong">
    法国广播电台的记者和总部位于巴黎的调查新闻机构 Disclose 的联合创始人被叫到对内安全总局（DGSI）的办公室“问话”。就是传唤
   </strong>
   。
  </p>
  <p class="graf graf--p">
   DGSI 的任务是打击恐怖主义、间谍活动和其他国内威胁，其功能与美国联邦调查局类似。
  </p>
  <p class="graf graf--p">
   这两家新闻机构于4月份发布了那份报道 —— 连同发表的还有美国媒体 The Intercept 、Mediapart、ARTE Info、和 Konbini News —— 报道揭示了大量证据，证明法国、英国和美国的军事设备销往沙特和阿联酋，并随后被用于在也门杀人。
  </p>
  <p class="graf graf--p">
   报道提供的证据十分惊人，
   <strong class="markup--strong markup--p-strong">
    在这里看到中文版《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%a1%80%e6%97%8f%e7%9a%84%e9%80%bb%e8%be%91/" href="https://www.iyouport.org/%e8%a1%80%e6%97%8f%e7%9a%84%e9%80%bb%e8%be%91/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     血族的逻辑
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这些报道 —— 基于法国军事情报局的秘密文件 —— 强调了法国政府的高级官员向公众谎报法国武器在战争中的作用。
  </p>
  <p class="graf graf--p">
   ⚠️这些机密文件证明了英美法参与破坏性冲突的程度，这场冲突造成了 17,900 多名平民死亡或受伤，并导致大约85,000名儿童陷入饥荒的折磨。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    法国政府一直在保密这些消息不希望将该文件公之于众，当该报告成为世界各地的头条新闻时真的激怒了那些官员
   </strong>
   。
  </p>
  <p class="graf graf--p">
   文章发布后不久，Disclose 的联合创始人 Geoffrey Livolsi 和 Mathias Destal 以及法国广播电台记者 Benoît Collombat 立刻就被 DGSI 总部传唤了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    记者们在这个地下四层的房间里被讯问了一小时，被问及他们的工作、他们的信息来源、以及他们在 Facebook 和 Twitter 上的帖子
   </strong>
   。
  </p>
  <p class="graf graf--p">
   记者们拒绝回答问题，理由是他们有权保持沉默，并且他们提出了关于他们的新闻和他们相信发布的文件符合公众利益的声明。
  </p>
  <p class="graf graf--p">
   自从
   <a class="markup--anchor markup--p-anchor" data-href="https://www.legifrance.gouv.fr/affichTexte.do?cidTexte=LEGITEXT000006070722" href="https://www.legifrance.gouv.fr/affichTexte.do?cidTexte=LEGITEXT000006070722" rel="noopener noreferrer" target="_blank">
    1881年的新闻法
   </a>
   颁布以来，新闻自由在法国已经得到了130多年的强烈保护，这使新闻记者有权保护其来源的机密性。
  </p>
  <p class="graf graf--p">
   法律还规定了某些“新闻犯罪”，记者可能会被指控的情况 —— 例如诽谤 —— 并通过法庭实施惩罚，方法包括罚款、和极端情况下的监禁，法律概述了如何处理这些罪行的程序。
  </p>
  <p class="graf graf--p">
   但是，国家安全问题并没有作为“新闻犯罪”被列入新闻法，而 DGSI 似乎抓住了这个漏洞，⚠️指责披露消息的调查性新闻机构 Disclose 和法国电视台记者们从持有机密文件的那一刻起就“危害国家安全”了。
  </p>
  <p class="graf graf--p">
   ⚠️根据
   <a class="markup--anchor markup--p-anchor" data-href="https://www.legifrance.gouv.fr/affichCode.do;jsessionid=08BF2E2434C084FBB485119C45542B53.tplgfr25s_2?idSectionTA=LEGISCTA000006165357&amp;cidTexte=LEGITEXT000006070719&amp;dateTexte=20190515" href="https://www.legifrance.gouv.fr/affichCode.do;jsessionid=08BF2E2434C084FBB485119C45542B53.tplgfr25s_2?idSectionTA=LEGISCTA000006165357&amp;cidTexte=LEGITEXT000006070719&amp;dateTexte=20190515" rel="noopener noreferrer" target="_blank">
    2009年法国颁布
   </a>
   的禁止“攻击国防机密”的法律，
   <strong class="markup--strong markup--p-strong">
    如果某人未经授权处理任何机密文件，即构成犯罪。这项法律对记者没有例外，也没有公益辩护。显然就是针对透明度革命而来的。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他们想拿我们杀鸡，因为这是法国第一次出现像这样的泄漏，” Disclose 的联合创始人 Livolsi 说，他指的是该文件的敏感性，该文件是法国军方分析人员去年9月在 Emmanuel Macron 的高级别简报中编写的。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “他们想吓唬记者和消息来源，让他们不敢再触碰任何国家机密。”
   </strong>
  </p>
  <p class="graf graf--p">
   在最糟糕的情况下，记者可能要面临长达五年的监禁和75,000欧元（约83,900美元）的罚款。
  </p>
  <p class="graf graf--p">
   该案的下一阶段仍不清楚。有可能会移交给法官，法官可以进行进一步的调查，并可能决定将案件提交审判。
  </p>
  <p class="graf graf--p">
   Disclose 的律师和董事会成员 Virginie Marquet 周二在 DGSI 的一次听证会上代表 Destal 出席。她希望记者不会面临牢狱之灾。
  </p>
  <p class="graf graf--p">
   但她指出，
   <strong class="markup--strong markup--p-strong">
    政府似乎正在推动严厉的惩罚
   </strong>
   。
  </p>
  <p class="graf graf--p">
   上周，武装部队部长 Florence Parly
   <a class="markup--anchor markup--p-anchor" data-href="https://rmc.bfmtv.com/mediaplayer/video/florence-parly-j-ai-eu-le-rapport-de-la-situation-securitaire-au-yemen-entre-les-mains-1159658.html?fbclid=IwAR34H-MYCkiKLPCGSBKvcJQqkPvSnzAm9xutQvHIfLqe8Qdjgn83ZD-Qg_g" href="https://rmc.bfmtv.com/mediaplayer/video/florence-parly-j-ai-eu-le-rapport-de-la-situation-securitaire-au-yemen-entre-les-mains-1159658.html?fbclid=IwAR34H-MYCkiKLPCGSBKvcJQqkPvSnzAm9xutQvHIfLqe8Qdjgn83ZD-Qg_g" rel="noopener noreferrer" target="_blank">
    在公开声明中暗示
   </a>
   ，披露这些文件违反了“我们国家的所有规则和法律”，并补充道：⚠️“只要你披露机密文件，你就会受到处罚。”
  </p>
  <p class="graf graf--p">
   无论结果如何，DGSI对案件的处理已经发出了这一信号 —— 要遏制透明度革命。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “有一种令人不寒而栗的效果，”Marquet 说。 “这是对每个记者的警告：不要涉及这类主题，不要调查政府的秘密。”
  </p>
  <p class="graf graf--p">
   无国界记者组织法律办公室负责人 Paul Coppin 告诉 The Intercept 他无法预测案件的结果，因为这是前所未有的。
  </p>
  <p class="graf graf--p">
   他补充说，
   <strong class="markup--strong markup--p-strong">
    记者可能因处理机密文件而被处罚 —— 无论这些文件的公共利益有多高 —— 特别是考虑到，⚠️政府可以轻松地将任何信息归类为机密
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这是非常有问题的，”Coppin 说。 “这揭示了记者在行使职责时本应从中受益的程序保障被严重削弱。法国本应该有一个更强大的框架来保护记者的工作。”
  </p>
  <p class="graf graf--p">
   监督 DGSI 的法国内政部没有回应评论请求。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2019/05/17/france-takes-unprecedented-action-against-reporters-who-published-secret-government-document/" href="https://theintercept.com/2019/05/17/france-takes-unprecedented-action-against-reporters-who-published-secret-government-document/" rel="noopener noreferrer" target="_blank">
    “They want to scare journalists and their sources away from revealing state secrets.”
   </a>
   FRANCE TAKES UNPRECEDENTED ACTION AGAINST REPORTERS WHO PUBLISHED SECRET GOVERNMENT DOCUMENT
  </p>
