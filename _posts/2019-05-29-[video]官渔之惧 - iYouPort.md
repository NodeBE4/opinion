   [video]官渔之惧
  </h1>

    关于政府的间谍行为
   </a>
  </span>
  <span class="published-on">

 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      每当人们说起网络钓鱼的时候总会想到攻击者、骇客，但政府亲自实施的网络钓鱼以骗取反对派的用户名和密码这种事，也是存在的，它发生在哪里？
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   据安全研究人员称，为陷入困境的委内瑞拉政府工作的骇客试图通过网络钓鱼诱骗活动人士泄漏密码，涉及 Gmail，Facebook，Twitter 等平台。
  </p>
  <p class="graf graf--p">
   志愿者的工作主要围绕在网站
   <a class="markup--anchor markup--p-anchor" data-href="http://voluntariosxvenezuela.com/" href="http://voluntariosxvenezuela.com/" rel="noreferrer noopener" target="_blank">
    voluntariosxvenezuela.com
   </a>
   上进行。一周后，在2月11日，有人注册了一个几乎完全相同的域名，voluntariosvenezuela [.] com。根据安全公司卡巴斯基实验室以及推特上的委内瑞拉用户
   <a class="markup--anchor markup--p-anchor" data-href="https://securelist.com/dns-manipulation-in-venezuela/89592/" href="https://securelist.com/dns-manipulation-in-venezuela/89592/" rel="noopener noreferrer" target="_blank">
    的说法
   </a>
   ，在委内瑞拉试图访问原始官方 VoluntariosxVenezuela 网站的用户被重定向到了新的钓鱼网站。
  </p>
  <p class="graf graf--p">
   在这里看到视频：
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="350" src="//player.vimeo.com/video/321730462?title=0&amp;byline=0&amp;portrait=0&amp;color=8dc7dc" width="425">
   </iframe>
  </p>
  <p class="graf graf--p">
   在研究虚假网站时，研究人员发现网络钓鱼网站托管在同一个IP地址上，根据安全公司 CrowdStrike 和独立研究人员
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2019/01/23/world/americas/venezuela-protests-guaido-maduro.html" href="https://www.nytimes.com/2019/01/23/world/americas/venezuela-protests-guaido-maduro.html" rel="noopener noreferrer" target="_blank">
    的说法
   </a>
   ，有证据表明，第二个显然是虚假和恶意的网站背后的人正在为马杜罗政府工作。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这显然是委内瑞拉政府的工作，试图找出反对他们的人，以便可以制止反对者，”负责分析袭击的公司 CrowdStrike 的情报副总裁 Adam Meyers 说。
  </p>
  <p class="graf graf--p">
   根据 PassiveTotal 和其他互联网监控服务收集的公共数据，虚假网站
   <a class="markup--anchor markup--p-anchor" data-href="https://checkphish.ai/ip/159.65.65.194" href="https://checkphish.ai/ip/159.65.65.194" rel="noopener noreferrer" target="_blank">
    的IP地址
   </a>
   托管了几个旨在骗取用户名和密码的域名，攻击目标包括 Gmail，Facebook，Instagram，Microsoft Live，Linkedin 和 Apple 的 iCloud 以及其他网站。
  </p>
  <p class="graf graf--p">
   根据主板查看的 WHOIS 记录，钓鱼网站均在委内瑞拉国家顶级域名.ve注册，该域名由委内瑞拉政府电信管理局的 CONATEL 控制。
  </p>
  <p>
   <img alt="" class="aligncenter wp-image-1387 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-148.png?resize=800%2C589&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-148.png?w=1067&amp;ssl=1 1067w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-148.png?resize=300%2C221&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-148.png?resize=768%2C566&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-148.png?resize=1024%2C754&amp;ssl=1 1024w" data-recalc-dims="1" height="589" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-148.png?resize=800%2C589&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>
   <noscript>
    <img alt="" class="aligncenter wp-image-1387" data-recalc-dims="1" height="589" sizes="(max-width: 800px) 100vw, 800px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-148.png?resize=800%2C589&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-148.png?w=1067&amp;ssl=1 1067w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-148.png?resize=300%2C221&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-148.png?resize=768%2C566&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-148.png?resize=1024%2C754&amp;ssl=1 1024w" width="800"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   追踪袭击事件的当地黑客行动者何塞 — 路易斯·里瓦斯（Jose-Luis Rivas）
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/ghostbar" href="https://twitter.com/ghostbar" rel="noopener noreferrer" target="_blank">
    表示
   </a>
   ，假冒网站上有一个表格可以登记个人数据，如姓名，电子邮件，手机和位置。“这就是他们追求的目标。确定谁是反对者，对其任意拘留”。
  </p>
  <p class="graf graf--p">
   Facebook，Apple，微软和 Twitter 都没有回复评论请求。承载服务器和攻击中使用的IP地址的公司 DigitalOcean 表示，在确定“这违反了我们的服务条款”后它已被关闭。谷歌表示，它将警告用户有关网络钓鱼域名的信息。
  </p>
  <p class="graf graf--p">
   Meyers 表示，他们没有足够的数据来了解网络钓鱼活动的效果。但是，鉴于 Guaido 最近表示已有近10万人签署帮助提供援助，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.afp.com/en/news/15/venezuela-opposition-announces-new-aid-hub-prepares-protest-doc-1db8lk2" href="https://www.afp.com/en/news/15/venezuela-opposition-announces-new-aid-hub-prepares-protest-doc-1db8lk2" rel="noopener noreferrer" target="_blank">
    可能已经
   </a>
   有至少数百人成为了攻击目标。
  </p>
  <p class="graf graf--p">
   Rivas 说，委内瑞拉的 VoluntariosxVenezuela 访客被国家控制的 ISP CANTV 重定向到虚假网站，后者正在操纵域名系统或 DNS。委内瑞拉的其他技术专家和互联网用户也
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/phenobarbital/status/1095527511738540033" href="https://twitter.com/phenobarbital/status/1095527511738540033" rel="noopener noreferrer" target="_blank">
    报告了
   </a>
   相同的调查结果。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    V
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://motherboard.vice.com/en_us/article/d3mdxm/venezuela-government-hack-activists-phishing" href="https://motherboard.vice.com/en_us/article/d3mdxm/venezuela-government-hack-activists-phishing" rel="noopener noreferrer" target="_blank">
    enezuela’s Government Appears To Be Trying to Hack Activists With Phishing Pages
   </a>
   . Security researchers and activists exposed a brazen attempt to steal Venezuelan usernames and passwords for popular email and social media websites such as Gmail, Facebook, Microsoft Live, and Twitter.
  </p>
