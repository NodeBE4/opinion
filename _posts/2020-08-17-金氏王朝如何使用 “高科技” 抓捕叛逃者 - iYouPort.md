   </li>

  <p>
   <img alt="" class="aligncenter size-full wp-image-13419 jetpack-lazy-image" data-lazy-sizes="(max-width: 992px) 100vw, 992px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/08/0.jpg?resize=992%2C661&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/08/0.jpg?w=992&amp;ssl=1 992w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/08/0.jpg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/08/0.jpg?resize=768%2C512&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/08/0.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="661" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/08/0.jpg?resize=992%2C661&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="992"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   去年12月，一个身份不明的黑客偷走了997名朝鲜难民的个人信息，震惊了韩国的难民社区。
  </p>
  <p class="graf graf--p">
   根据统一部的说法，难民的姓名、生日和住址是从哈那中心的一台个人计算机上
   <a class="markup--anchor markup--p-anchor" data-href="https://www.reuters.com/article/us-northkorea-southkorea/mystery-hacker-steals-data-on-1000-north-korean-defectors-in-south-idUSKCN1OR0CS" href="https://www.reuters.com/article/us-northkorea-southkorea/mystery-hacker-steals-data-on-1000-north-korean-defectors-in-south-idUSKCN1OR0CS" rel="noopener noreferrer" target="_blank">
    窃取的
   </a>
   ，该中心是由该部经营的一个机构，朝鲜难民到达位于韩国庆尚北道的该中心后可以得到帮助。
  </p>
  <p class="graf graf--p">
   对于朝鲜难民来说，拥有大量个人私密信息的机构其计算机系统中的漏洞问题引起了严重的关注，因为，如果朝鲜难民的这些信息被朝鲜政府拿走，他们和家人可能会被胁迫重返朝鲜。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们中的许多人在得知这一消息后就无法日常生活了。与我们家庭成员安全有关的任何事对我们来说都是至关重要的，” 朝鲜难民和民主化委员会主席 许光日 Heo Kwang-il 说。
  </p>
  <p class="graf graf--p">
   另一位朝鲜难民 Lee Han-byul 说，有些难民甚至晚上不敢开灯。
  </p>
  <p class="graf graf--p">
   李说：“自从哈那中心发生数据泄漏事件以来，许多难民就变得格外谨慎了，再难以与公共机构和政府机构共享任何个人信息。”
  </p>
  <p class="graf graf--p">
   众所周知，韩国拥有世界上最强大的信息技术基础设施之一，而朝鲜难民群体极易受到攻击，哈那中心无法理解针对它的网络威胁的严重性。
  </p>
  <p class="graf graf--p">
   卫生部证实，
   <strong class="markup--strong markup--p-strong">
    哈那中心在处理朝鲜难民的个人信息时违反了使用隔离网络的规定，导致通过电子邮件发送的恶意代码感染了员工的个人计算机。
   </strong>
  </p>
  <p class="graf graf--p">
   尽管该中心在泄漏事件发生后宣布调整网络隔离，但专家表示，网络隔离并不是万能的补救措施。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “网络隔离并不能解决所有网络攻击问题。
   <strong class="markup--strong markup--p-strong">
    需要为处理难民信息的所有员工提供更多的预防措施
   </strong>
   。” 韩国亚洲大学网络安全教授 Kwak Jin 说。
  </p>
  <p class="graf graf--p">
   朝鲜网络技术能力的增长速度加剧了难民的恐惧。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “通常人们只是关注朝鲜核能力的增长，尽管核武器非常严重，但这种专注会削弱对朝鲜重要网络技术能力的关注，” 澳大利亚战略政策研究所国际网络政策中心研究员 Elise Thomas。
  </p>
  <p class="graf graf--p">
   朝鲜不断增强的网络技术能力意味着坐在屋子里就可以控制和操纵整个社会。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    朝鲜在国外进攻性地使用其网络力量，而在国内通过建立数字隔离墙来 “防御” 朝鲜人民从国外访问信息
   </strong>
   ，从而将人们限制在其国家控制的宣传网站上，这些网站称赞金正恩的一切举动，同时谴责韩国和美国政府。
  </p>
  <p class="graf graf--p">
   人们已经敏锐地意识到了朝鲜的网络技术能力以及过去案件中暴露的信息的后果，因此，有家人被带回朝鲜的朝鲜难民一直充满焦虑。
  </p>
  <p class="graf graf--p">
   2006年，一名韩国哨兵在韩国江原道高城的船上发现了一群朝鲜难民。一旦朝鲜政府了解了难民的身份，就会胁迫其留在朝鲜的家庭成员让他们遭受惩罚，难民们要求韩国调查人员不要向公众透露他们的信息。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    韩国江原道省警察厅向韩国新闻媒体发布了一份报告，其中就包含了难民身份的详细信息，以此向公众公开了难民的致命隐私。
   </strong>
  </p>
  <p class="graf graf--p">
   在联系了他们在朝鲜的信息源后，难民得知了
   <a class="markup--anchor markup--p-anchor" data-href="http://www.donga.com/news/article/all/20110518/37331652/1" href="http://www.donga.com/news/article/all/20110518/37331652/1" rel="noopener noreferrer" target="_blank">
    灾难性的消息
   </a>
   ：他们的直系亲属中共有22名成员在朝鲜失踪了。他们的下落仍然未知。
  </p>
  <p class="graf graf--p">
   难民和北朝鲜民主化委员会主席 Heo 说，当新闻再次没有保护难民时，难民社区被“激怒”是很正常的。
  </p>
  <p class="graf graf--p">
   随着时间的流逝，针对韩国政府和公共机构的网络威胁越来越严重。专家说，他们的目标之一是统一部。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “最近，朝鲜及其与韩国和美国的关系有关的问题在全球引起了很多关注。对统一部的网络攻击的次数增加也与此有关”，Kwak 说。
  </p>
  <p class="graf graf--p">
   根据统一部在2019年有关黑客和网络攻击尝试的
   <a class="markup--anchor markup--p-anchor" data-href="http://theminjoo.kr/inspectionDetail.do?nt_id=16&amp;bd_seq=154589" href="http://theminjoo.kr/inspectionDetail.do?nt_id=16&amp;bd_seq=154589" rel="noopener noreferrer" target="_blank">
    报告
   </a>
   ，在过去的几年中，该部遭受的网络攻击数量已大大增加。从2017年到2018年，攻击次数几乎翻了一番，达到 630 次。
  </p>
  <p class="graf graf--p">
   攻击哈那中心的黑客的身份仍然未知。韩国警方仍在调查此案。许多朝鲜难民，包括许光日，都认为网络攻击是由朝鲜政府进行的。
  </p>
  <p class="graf graf--p">
   Heo 说：“朝鲜拥有世界上最强大的网络军队之一。他们的黑客不仅在朝鲜境内活动，而且在不同国家/地区活动。”
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    不要错过这份报告，很详细《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e3%80%81%e4%b8%ad%e5%9b%bd%e5%92%8c%e4%bf%84%e7%bd%97%e6%96%af%ef%bc%9a%e6%9a%97%e7%bd%91%e6%9c%80%e5%bc%ba%e5%a4%a7%e7%9a%84%e6%b0%91%e6%97%8f%e5%9b%bd%e5%ae%b6%e9%aa%87%e5%ae%a2/" href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e3%80%81%e4%b8%ad%e5%9b%bd%e5%92%8c%e4%bf%84%e7%bd%97%e6%96%af%ef%bc%9a%e6%9a%97%e7%bd%91%e6%9c%80%e5%bc%ba%e5%a4%a7%e7%9a%84%e6%b0%91%e6%97%8f%e5%9b%bd%e5%ae%b6%e9%aa%87%e5%ae%a2/" rel="noopener noreferrer" target="_blank">
     美国、中国和俄罗斯：暗网最强大的民族国家骇客
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   据称，朝鲜在过去几年中曾多次发动网络攻击。
  </p>
  <p class="graf graf--p">
   2018年，美国财政部指控朝鲜政府资助的黑客组织 Lazarus Group 主导实施了 Wannacry 勒索软件攻击，该攻击影响了150个国家的医院、银行和其他公司，造成的损失从数亿美元到数十亿美元不等。
  </p>
  <p class="graf graf--p">
   朝鲜否认在网络攻击中发挥任何作用，但财政部指出了 Lazarus 以及其他两个朝鲜的黑客组织。FBI还指控了一名朝鲜计算机程序员 Park Jin-hyok，该人据称曾在 Lazarus 从事 WannaCry 攻击行动以及参与了2014年对 Sony Pictures 的黑客攻击。
  </p>
  <p class="graf graf--p">
   随着朝鲜的网络技术能力日益成为世界上最危险的力量之一，包括美国在内的其他国家也正在采取行动，加强其国防系统。
  </p>
  <p class="graf graf--p">
   作为受朝鲜网络威胁打击最严重的国家，韩国的网络防御系统在很大程度上能阻止朝鲜黑客的攻击。但是许光日认为，生活在韩国的许多北朝鲜难民有不同的感觉。
  </p>
  <p class="graf graf--p">
   李说：“哈那中心在处理朝鲜难民的个人信息时不够谨慎。统一部也不应忽视对难民的保护。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “韩语里有句老话：一头牛被盗后要修好谷仓的门。对我们许多人来说，被告知在信息被盗后联系接待处似乎毫无意义，这并不像我们向韩国政府索要什么东西一样比如钱。
   <strong class="markup--strong markup--p-strong">
    我们不是在要求其他任何东西，而是，要把我们视为国家的公民。让我们成为与其他韩国人处于同一地位的人
   </strong>
   。” ⚪️
  </p>
