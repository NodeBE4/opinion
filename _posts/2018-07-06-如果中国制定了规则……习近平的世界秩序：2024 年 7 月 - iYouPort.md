  <p class="graf graf--p graf-after--h3" id="82ac">
   这个故事所暗示的主题依旧是人们所熟悉的对“美国缺位”的愤怒和哀叹，然而它使用了一个非常巧妙的叙述方式，并由此“不小心”带入了更多显而易见却一直被很多人所回避的问题。经济学人没有深入谈论的那些事，我们非常关心。
  </p>
  <p class="graf graf--p graf-after--p" id="644e">
   当帝国确立，Edward Snowden 和 Julian Assange 等诸多说出真话的人所面对的危险和协迫，正是这个故事中描述的中国软件工程师 Fu Xuedong 的遭遇。我们明白作为英国媒体，经济学人采取如下切入角度是“安全的”，但也因此尤其令我们感到焦虑，因为它恰恰证明了帝国的势力已经造成了触手可及的灾难。
  </p>
  <p class="graf graf--p graf-after--p" id="c85c">
   我们需要一个民主的世界，用大多数人的道德共识谴责和修正那些违背这一规则的人。
   <strong class="markup--strong markup--p-strong">
    规则不该有哪个单一的国家决定
   </strong>
   ，中国不能，美国也不能，然而中美的霸主之争已经从互联网世界进入了现实政治 —
   <strong class="markup--strong markup--p-strong">
    高堡奇人的启示
   </strong>
   。我们认为经济学人文章已经暗示了这点，这就是为什么我们要翻译它，尽管它没能充分明确我们期待的结论。
  </p>
  <p class="graf graf--p graf-after--p" id="1e60">
   注：故事中暗示的“要求企业在电子产品中添加后门”是美国的做法，“没有赢家”在暗示美国及其盟友采取地缘政治立场诠释斯诺登是错误的思路。故事中对中国私设仲裁法庭、英国脱欧后抛弃欧洲并屈从北京、被单方面操控的国际刑警组织等讽刺，都非常鲜明。而大使馆断网正是目前 Julian 在厄瓜多尔伦敦使馆的遭遇……
  </p>

  <p class="graf graf--p graf-after--figure" id="acdf">
   新闻报道称他为“中国的爱德华斯诺登”；他的全球粉丝称他为“傅兄弟” — 这个标签现在正不断出现在 T恤和政治 meme 中。据说这两个标签都“委屈了“这位在加拿大接受过教育的软件工程师 Fu Xuedong，他对中国网络间谍行为的指控在 2024 年的夏天震惊了全球。
  </p>
  <p class="graf graf--p graf-after--p" id="a877">
   在特朗普第二任期的最后一年，傅先生因为指控中国的情报部门与美国的科技公司合作而引发了一片混乱，这一史无前例的监视计划已经将美国、欧洲和亚洲的数百万辆汽车变成了远程的间谍设备，让北京可以实时跟踪车辆、通过面部识别技术识别乘客，甚至窃听他们。
  </p>
  <p class="graf graf--p graf-after--p" id="f11e">
   中国否认这些指控，如果得以确认，这将构成历史上规模最大的间谍活动。然而，由于傅先生被指称为“幻想家”和“史无前例的骗子”，其反应出的愤怒令人不安。
  </p>
  <p class="graf graf--p graf-after--p" id="485c">
   他是深圳一家创新实验室的网络安全专家，他已经出逃五个星期 — 他过去四年来曾在伊斯坦布尔的美国领事馆躲藏，外交官和政治家们对他的命运争论不休，到目前为止，傅先生的传奇
   <strong class="markup--strong markup--p-strong">
    是一个没有赢家，但有一大堆输家的故事
   </strong>
   — 包括一些世界上最大的公司和政府，这些公司和政府在中国第一个愤怒的暗示中就屈服了。
  </p>
  <p class="graf graf--p graf-after--p" id="bd2d">
   但
   <strong class="markup--strong markup--p-strong">
    最重要的输家可能是一个抽象的原则：即基于国际规则的秩序概念
   </strong>
   ，其中可预测的、透明的法律原则甚至可以约束最强大的国家。Fu Xuedong 的考验 — 一个看上去因突然被恶名而感到震惊的、柔弱的、言语温和的人 — 生动地揭示了中国对法治的承诺在多大程度上是有条件的。正如一位资深外交官所讽刺的那样：“中国想要的是真正模糊的规则，以及全部解释权。”
  </p>
  <p class="graf graf--p graf-after--p" id="fd6a">
   被傅先生的举报所损伤的第一批是美国、欧洲和日本的汽车制造商，股价暴跌。Fu 发布了一段 YouTube 视频，描述了他如何偶然发现一个似乎已建立在数百万辆先进的半自动驾驶汽车系统上的
   <strong class="markup--strong markup--p-strong">
    秘密后门
   </strong>
   。它允许收集并通过加密通道发送这些数据，并在另一个方向上还携带诸如交通警报、导航建议和车辆软件更新等信息。
  </p>
  <p class="graf graf--p graf-after--p" id="326a">
   今天近四分之三的高科技汽车使用中国设计的 5G 移动芯片进行这些数据传输，此前外国汽车制造商向中国施加压力，采用其技术被作为交易的一部分，以便更多地进入中国汽车市场。
  </p>
  <p class="graf graf--p graf-after--p" id="49b3">
   网络间谍丑闻也损害了不止一个外国政府的声誉。从中国逃到土耳其后，出生在香港的傅先生携带英国签发的旅行证件，最初在英国领事馆寻求庇护，逃避了据称埋伏在伊斯坦布尔机场的中国特工。但在候车室待了几个小时后，傅先生被英国外交官要求离开，他们引用了加拿大和中国的双重国籍这一线索，并敦促他向这些国家领事馆寻求帮助。
  </p>
  <p class="graf graf--p graf-after--p" id="f8e0">
   英国政府否认佛罗里达州参议员 Marco Rubio 对共和党人提出的强烈反对意见，这一决定是与伦敦成为中国企业法律和金融中心的野心相关的“狡猾的投降”。具体而言，Rubio 先生指责伦敦在
   <strong class="markup--strong markup--p-strong">
    英国退欧后失去了纽约和法兰克福的地位
   </strong>
   ，为全球基础设施中心提供新的标准制定论坛和风险投资中心是完全无望的。
  </p>
  <p class="graf graf--p graf-after--p" id="6659">
   GIC 是一个位于北京的信息交换中心，用于与过去被称为“一带一路”的项目相关联，这是中国雄心勃勃的项目，旨在通过新的铁路、港口、道路和数据线连接欧洲、非洲和中东。
  </p>
  <p class="graf graf--p graf-after--p" id="bad7">
   除了实体联系外，习近平还通过 GIC 促进了包括中国标准和治理规范在内的有形债券。至关重要的是，它决定哪些计划有资格获得数十亿美元的中国贷款和赠款，并选择外国公司作为合作伙伴使用
   <strong class="markup--strong markup--p-strong">
    共产党规划者设计的不透明规则
   </strong>
   。伦敦希望自己成为 GIC 首选的全球资本市场门户，为英国银行家、律师和顾问带来利益。
  </p>
  <p class="graf graf--p graf-after--p" id="253d">
   傅先生甚至没机会去加拿大领事馆，外交官通过电话通知他，他们需要更多时间来研究全球警察机构国际刑警组织应中国当局要求发出的紧急
   <strong class="markup--strong markup--p-strong">
    “红色通缉令”
   </strong>
   。该通缉令没有法律约束力，要求所有 192 个国际刑警组织成员国以“涉嫌从事间谍活动、盗窃和破坏国家安全”为由抓捕傅先生。加拿大反对派政客指责总理 Doug Ford 牺牲了傅先生，以争取一项有争议的协议，将加拿大的北极水域开放给中国的油轮和其他航运。
  </p>
  <p class="graf graf--p graf-after--p" id="dc53">
   在美国领事馆获得避难后，傅先生最初通过 Skype 接受了一系列媒体采访。但第二天，
   <strong class="markup--strong markup--p-strong">
    领事馆的互联网接入就受到了严格限制
   </strong>
   。美国外交官表示，根据此前与中国签订的秘密双边安全协议条款，土耳其在领事馆周围建立了中国版的防火墙，表面上是为了遏制维吾尔族的伊斯兰武装分子，维吾尔族是一个讲突厥语的少数民族，来自中国动荡不安的西部地区新疆。
  </p>
  <p class="graf graf--p graf-after--p" id="457f">
   所有这一切都让美国成为傅先生的超级大国保护者。白宫新闻秘书 Laura Ingraham 一再敦促土耳其允许这位工程师不受干扰地离开该国，称中国对他提起刑事指控是“假新闻”。在国会，众议院和参议院情报委员会已经发出传票，要求傅先生尽快在华盛顿特区提供证据。Rubio 先生已提出亲自从伊斯坦布尔接回傅先生。但令人遗憾的是，对于软件工程师来说，特朗普的美国是一个分散注意力的超级大国，它
   <strong class="markup--strong markup--p-strong">
    已经破坏了以规则为基础的秩序
   </strong>
   。因此，它对中国无视长期规则和规范的批评现在已经变得空洞。
  </p>
  <p class="graf graf--p graf-after--p" id="8996">
   特朗普先生的顾问们在他上任时仍然存在分歧。经济民族主义者希望他利用所谓的301条款来惩罚中国，迫使汽车制造商安装“
   <strong class="markup--strong markup--p-strong">
    通过同一个后门
   </strong>
   窃取美国机密信息和工作”的技术，正如特朗普的国家经济安全顾问 Lou Dobbs 所说的那样。一个不同寻常的联盟，包括福特和通用汽车的老板、欧盟以及日本、德国、英国和法国的政府，希望特朗普将中国带到世界贸易组织中。但美国通过与朋友和敌人进行
   <strong class="markup--strong markup--p-strong">
    贸易斗争
   </strong>
   ，以及拒绝任命法官到其上诉机构，为中国建立自己的竞争机制开辟了道路，嘲弄了世界贸易组织。
  </p>
  <blockquote class="graf graf--blockquote graf-after--p" id="600f">
   <p>
    <em class="markup--em markup--blockquote-em">
     America’s disdain for the rules-based order means its criticism of China now rings hollow
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p graf-after--blockquote" id="9eb8">
   最初为审理与“一带一路”倡议相关的纠纷而设立的一个不起眼的中国本土仲裁小组于 2022 年更名为“全球基础设施法庭”。在第一批案例中，这个奇怪的贸易法庭提供了一个中国主导的商业规则如何运作的瞥见。与WTO不同，它没有区分具有国家指导型经济和市场经济的国家。其法官声称对支持国家发展的补贴采取良性观点。虽然他们谈论了一个关于知识产权保护的问题，但他们一直认为是主权政府而不是个人企业应该在专利纠纷中拥有最终决定权。
  </p>
  <p class="graf graf--p graf-after--p" id="152a">
   在总统于 2018 年退出伊朗核协议后，特朗普政府与其盟国进行的另一场斗争
   <strong class="markup--strong markup--p-strong">
    阻碍了西方试图对这种中国人的招摇所采取的联合挑战
   </strong>
   。当美国财政部官员试图惩罚使用美元购买和出售伊朗货物（尤其是石油）的公司和银行时，中国创建了一个替代系统，用于支付中国人民币、欧元和俄罗斯卢布的国际支付，面向构成“一带一路”倡议支柱的欧亚国家。
  </p>
  <p class="graf graf--p graf-after--p" id="25e7">
   这一新系统受到中国技术和加密标准的推动，削弱了美元在国际业务中的主导地位，并构成了对比利时 SWIFT 财团长期享有的跨境银行业务的控制。俄罗斯利用这一新系统创建了不受美元制裁影响的银行，客户称其中包括伊朗、叙利亚、苏丹和朝鲜政府。
  </p>
  <p class="graf graf--p graf-after--p" id="f254">
   由于财政部官员发现实施金融制裁更加困难，美国海军指挥官正在努力收紧对中国的套索。从理论上讲，中国与其亚洲邻国在南中国海有争议的岛屿上仍然存在争执。但中国扩大和加强这些珊瑚礁的努力与其外交胜利交相辉映了。它迫使邻国宣布南海大部分地区禁止涉及外部势力的军事演习，在与这些有争议的水域接壤的国家之间使用冠冕堂皇的“行为准则”。
  </p>
  <p class="graf graf--p graf-after--p" id="ccad">
   像新加坡这样的一些国家拒绝被规则绑定。其他国家，如菲律宾和越南，正在
   <strong class="markup--strong markup--p-strong">
    获得由中国控制的石油和天然气的份额，并且立场摇摆不定
   </strong>
   。对于其亚洲邻国而言，藐视中国并帮助美国治理公海的政治和经济的成本正在上升。
  </p>
  <p class="graf graf--p graf-after--p" id="a803">
   总而言之，如果傅先生感到茫然无知，完全不足为奇。他告诉记者，在得出结论认为自动驾驶汽车被中国间谍变成全球监视网络后，他决定冒着职业生涯危机甚至是自由的风险揭发出来。
  </p>
  <p class="graf graf--p graf-after--p" id="09ab">
   在这个传奇故事中，
   <strong class="markup--strong markup--p-strong">
    很少有人能够清楚地表明是非
   </strong>
   。中国对新世界秩序的愿景出现的时间早于人们的预期，因为以特朗普的美国为首的西方国家已经如此仓促地撤退了。中国很少面临直接挑战，相反中国对全球治理的各个方面进行了测试、探索和并引入了一堆含糊不清。已被确立的权势并没有被证明无法抗拒。傅先生“不合时宜”地陈述了他的原则，那些
   <strong class="markup--strong markup--p-strong">
    已经被该原则的缔造者所放弃的东西
   </strong>
   ，America first（最后这个短语有双重含义，就如你所知道的那样，于是不翻译最好，以避免浪费其中一个暗示）
