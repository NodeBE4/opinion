  <p class="graf graf--p">
   历史上一个最重要教训就是，崩溃的金融体系会引发专制政治…..
  </p>
  <p class="graf graf--p">
   今天的世界正在遵循这个脚本，因为不断上升的债务导致几乎每个举行自由选举的国家都发生了政治变化，而法西斯主义再次被那些在正常时期居住在政治中心的人们认真对待。
  </p>
  <p class="graf graf--p">
   但这次有一个很大的不同：社会控制技术正处于史无前例的先进状态。过去的政府在试图遏制不同意见时，仅仅能限于愚蠢的工具政策，如宵禁、电话窃听和断线。而今天的老大哥可以做得更多，并且
   <strong class="markup--strong markup--p-strong">
    在许多情况下把“即将到来的金融/政治紧急事件”作为借口，将奥威尔的谚语变成公民脖子上的枷锁
   </strong>
   。
  </p>
  <p class="graf graf--p">
   华尔街日报最近一篇题为“Autocrat 的新工具包”的文章中有一些例子：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    中国当局现在正在使用大数据工具来检测该国新疆地区穆斯林的“行为“ — 然后确定每个所谓的离经叛道，以将其变为针对性监视目标；
   </li>
   <li class="graf graf--li">
    埃及政府计划今年晚些时候从开罗搬迁到一个尚未命名的新首都，正如该项目的发言人所说的那样，“到处都有摄像机和传感器”、“一个控制整个城市的指挥中心”。
   </li>
   <li class="graf graf--li">
    莫斯科已经安装了约5000台带有面部识别功能的摄像机，它可以将护照数据库、警察档案和社交媒体上的照片，与当局感兴趣的面孔相匹配。
   </li>
  </ul>
  <p class="graf graf--p">
   虽然这些事听起来很可怕，但与即将到来的状况相比的话，它依旧是粗糙的。那么即将到来的是什么？如下这些算是*至少*级别上的：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “微观定位” — — 使政府能够建立公民的人格评估，并为目标的心理弱点量身定制宣传
   </strong>
   。据报道，俄罗斯互联网研究机构在2016年美国总统竞选期间从 Facebook 收集数据，为个人选民制作特定信息。而私营公司正在开发此类人工智能技术，可以为整个人群自动化这种定制 — — 简称定向洗脑。由于每个人的心理弱点不尽相同，于是首先研究出目标人最薄弱的缓解，针对性的制作洗脑模版，将产生极高的效果。你可以理解为大规模社交工程。
  </p>
  <p class="graf graf--p">
   这并不奇怪，因为这正是集中化互联网的最基本可利用“价值”，详见IYP早年的分析《
   <a class="markup--anchor markup--p-anchor" data-href="https://iyouport.weebly.com/iyp_blog/9719369" href="https://www.iyouport.org/%e5%bd%93-%e4%b8%8d%e4%bd%9c%e6%81%b6-%e6%88%90%e4%b8%ba%e7%a9%ba%e8%b0%88%ef%bc%8c%e5%ae%9a%e5%88%b6%e4%ba%ba%e5%b0%86%e7%bb%88%e7%bb%93%e6%b0%91%e4%b8%bb/" rel="noopener noreferrer" target="_blank">
    当 “不作恶” 成为空谈，“定制人”将终结民主
   </a>
   》
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    机器人 — — 仿效人类博主的算法 — — 将很快与人类无法区分
   </strong>
   ，也就是说，能够谴责反政权活动家、攻击竞争对手、并以栩栩如生的方式放大支持国家政权的信息。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    深度造假 — — 现在可以创建几乎与真实无法区分的假图像、声音和视频
   </strong>
   。大众无法知道他们所看到和听到的是真实的还是政府故意制造的假货。IYP 对此有过详细介绍，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/informationwar-115c07f0f8b0" href="https://medium.com/@iyouport/informationwar-115c07f0f8b0" rel="noopener noreferrer" target="_blank">
    当恶意者掌握技术：信息战的未来究竟有多可怕？
   </a>
   》
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    AI剖析 — — 人工智能正在学习从我们的社交媒体帖子中提取“态度、情感和意图”，让政府能够在异议进入行动之前瞄准异议
   </strong>
   。事实上不需要 AI 也能做到这种提前维稳式的监控，IYP 做过一个开源调查演示，详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E6%83%85%E6%84%9F%E5%88%86%E6%9E%90-%E5%A6%82%E4%BD%95%E5%9C%A8%E7%A4%BE%E4%BA%A4%E7%BD%91%E7%BB%9C%E4%B8%8A%E6%8A%93%E5%8F%96%E5%B9%B6%E5%88%86%E6%9E%90%E5%A4%A7%E4%BC%97%E5%AF%B9%E6%9F%90%E4%BA%9B%E8%AF%9D%E9%A2%98%E7%9A%84%E6%83%85%E7%BB%AA-c86a0f2360ae" href="https://medium.com/@iyouport/%E6%83%85%E6%84%9F%E5%88%86%E6%9E%90-%E5%A6%82%E4%BD%95%E5%9C%A8%E7%A4%BE%E4%BA%A4%E7%BD%91%E7%BB%9C%E4%B8%8A%E6%8A%93%E5%8F%96%E5%B9%B6%E5%88%86%E6%9E%90%E5%A4%A7%E4%BC%97%E5%AF%B9%E6%9F%90%E4%BA%9B%E8%AF%9D%E9%A2%98%E7%9A%84%E6%83%85%E7%BB%AA-c86a0f2360ae" rel="noopener noreferrer" target="_blank">
    情感分析：如何在社交网络上抓取并分析大众对某些话题的情绪？
   </a>
   》以及案例《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E5%AF%B9%E7%89%B9%E6%9C%97%E6%99%AE%E6%8E%A8%E6%96%87%E7%9A%84%E6%83%85%E6%84%9F%E5%88%86%E6%9E%90-python-d6a26d40154c" href="https://medium.com/@iyouport/%E5%AF%B9%E7%89%B9%E6%9C%97%E6%99%AE%E6%8E%A8%E6%96%87%E7%9A%84%E6%83%85%E6%84%9F%E5%88%86%E6%9E%90-python-d6a26d40154c" rel="noopener noreferrer" target="_blank">
    对特朗普推文的情感分析：Python🐍
   </a>
   》
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    超级“智能“卡
   </strong>
   — — 委内瑞拉推出了自己的“carnet de la patria”（祖国卡），这是一种基于智能芯片的识别标准，公民需要此卡才能获得医疗保健和补贴食品等政府服务。人权观察报告称，该卡也可以捕获投票历史记录。没错，
   <strong class="markup--strong markup--p-strong">
    它就是个监视器。那么中国的各种“一卡通”呢，您应该知道它是什么了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    超人类面部识别
   </strong>
   — — 人类天然擅长识别熟悉的面孔，而明天的计算机网络会在此做得更深入。中国警察已经部署了面部识别眼镜，并在非洲和欧洲销售这种技术。这种眼镜不仅可以用来帮助识别像小偷和闯红灯，也能
   <strong class="markup--strong markup--p-strong">
    用来追捕人权活动家和民主抗议者
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    所谓的智能城市
   </strong>
   — — 将物联网与大数据分析相结合，您可以将整个城市的摄像机覆盖每英寸的土地，并要求使用公共服务（如公共汽车）的人需要提交生物识别 ID。一些中国餐馆已经有了“微笑支付”系统来解释面部表情。IYP 对此的分析包括《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E6%99%BA%E8%83%BD%E5%9F%8E%E5%B8%82-%E7%A9%B6%E7%AB%9F%E6%98%AF%E4%B8%AA%E4%BB%80%E4%B9%88%E9%AC%BC-81dbf603e2c2" href="https://medium.com/@iyouport/%E6%99%BA%E8%83%BD%E5%9F%8E%E5%B8%82-%E7%A9%B6%E7%AB%9F%E6%98%AF%E4%B8%AA%E4%BB%80%E4%B9%88%E9%AC%BC-81dbf603e2c2" rel="noopener noreferrer" target="_blank">
    “智能城市”究竟是个什么鬼？！
   </a>
   》更多详见分类知识列表2“
   <a class="markup--anchor markup--p-anchor" data-href="https://docs.google.com/document/d/1tmU2VmsdHRo79NniIM85yWJrS-A_GIhku_0yXZOg8Fk/edit?usp=sharing" href="https://docs.google.com/document/d/1tmU2VmsdHRo79NniIM85yWJrS-A_GIhku_0yXZOg8Fk/edit?usp=sharing" rel="noopener noreferrer" target="_blank">
    智能/间谍城市
   </a>
   ”部分。
  </p>
  <p class="graf graf--p">
   互联网分散了数据，但新的技术进步可以将其力量集中在极少数人手中。预计到2020年将有超过300亿台设备连接到互联网，每个设备都能产生新数据，那些能够控制、处理和利用信息热潮的设备将具有极大的优势。一个渴望稳定的政权可能会感到自己急需变成老大哥。
  </p>
  <p class="graf graf--p">
   为下一次金融危机做准备需要比过去更多的思考。
   <strong class="markup--strong markup--p-strong">
    当恐慌的政府能够随时识别、跟踪和借记金融账户时，您如何保护自己的财富？即使您看似私密的对话和社交媒体互动正在流入自动分析算法，以便为随谁抓捕你做好准备，您如何保持思想、联想和行动的自由？
   </strong>
  </p>
  <p class="graf graf--p">
   显然，作为私人财富储存和匿名交易的手段，金币和银币在这个世界上已经变得越来越更有价值。地理多样化（即 在不同国家储存财富）减少了被任何一个政府完全没收的可能性。寻找逃避新兴监控状态的方法至关重要，需要开动脑筋。
  </p>
