   一家旨在帮助唐纳德·特朗普（Donald Trump）赢得2020年总统大选的手机追踪公司最近告诉投资者，
   <strong class="markup--strong markup--p-strong">

   尽管 Phunware 尚未向特朗普的竞选工作组公开其提供的服务的详细信息，但是，其网站和新闻报道均充分记录了该公司拥有的令人印象深刻的且具有高度侵入性的监视技术能力。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “除了识别出令人垂涎的选民之外，这些移动数据还可以用于以极高的准确性确定广告定位参数，” Phunware 在自己的网站上写道，
   <strong class="markup--strong markup--p-strong">

   </strong>
   。

  <p class="graf graf--p">
   华尔街日报于2019年10月的报道称，Phunware 帮助了一名民主党政治策略家瞄准了目标选民，
   <strong class="markup--strong markup--p-strong">
    这些公民的电话被追踪到了2017年的反特朗普游行
   </strong>
   Women’s March。

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该公司的隐私权政策授予公司极大的权力可将任何个人数据用于其认为必要的任何目的，无论需要追溯多长时间
   </strong>
   。

  <p class="graf graf--p">

   </a>

   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2020/03/27/ventilators-federal-coronavirus-contracts-timeline/" href="https://theintercept.com/2020/03/27/ventilators-federal-coronavirus-contracts-timeline/" rel="noopener noreferrer" target="_blank">
    拙劣
   </a>
   回应的同时，Phunware 希望 *利用* 实施社会隔离等大流行措施的必要性。
  </p>
  <p class="graf graf--p">
   3月30日，Phunware 联合创始人兼首席执行官 Alan Knitowski 致电投资者，他解释说，该公司 “正在积极寻求新的商机 …… 包括远程医疗、在总统大选年内的虚拟集会以进行政治宣传、远程工作优化和社会疏离政策的执行，为地方、州和联邦一级的智能城市和政府组织提供服务。”
  </p>
  <p class="graf graf--p">
   Phunware 首席运营官 Randall Crowder 在电话会议上详细阐述了该公司的 “移动数字门户”，它可以实现从动态的社会疏离政策实施到与电子健康记录（EHR）、远程保健、远程医疗和设施管理平台的无缝集成。 ”
  </p>
  <p class="graf graf--p">
   Crowder 表示，该公司还处于有利条件，可以
   <strong class="markup--strong markup--p-strong">
    从本地的封锁制度中获利
   </strong>
   ：“我们先进的移动校园体验技术将帮助雇主更好地远程管理员工 ……”。
  </p>
  <p class="graf graf--p">
   在 Phunware 投资者致电后两天发出的一次采访中，首席执行官 Knitowski 解释说，该公司可以利用
   <strong class="markup--strong markup--p-strong">
    地理围栏技术
   </strong>
   —— 用于瞄准潜在选民的相同方法 —— 来 “实施社会疏离政策”。
  </p>
  <p class="graf graf--p">
   地理围栏允许应用程序 —— 以及兜售它们的公司 —— 在地图上绘制虚拟边界，并通过结合使用蜂窝塔和 GPS ping 来跟踪手机的位置，从而监视进出的所有人员。
  </p>
  <p class="graf graf--p">
   现在，
   <strong class="markup--strong markup--p-strong">
    地理围栏技术已经在公共卫生领域中找到了新的粉丝，现在在印度和香港等地，围绕被隔离患者的病房都绘制了这种虚拟围栏。如果有人想要突破隔离并越过地理围栏，当局将收到警报
   </strong>
   。

  <p class="graf graf--p">
   尽管智能手机本身就是贴身间谍这件事如今应该尽人皆知了，但仍不清楚这些位置跟踪记录是否足够精确以用于医疗目的：路透社
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/reuters/2020/03/30/world/middleeast/30reuters-health-coronavirus-israel-tracking.html" href="https://www.nytimes.com/reuters/2020/03/30/world/middleeast/30reuters-health-coronavirus-israel-tracking.html" rel="noopener noreferrer" target="_blank">
    上月报道
   </a>
   ，以色列国防部长 Naftali Bennett 说：
   <strong class="markup--strong markup--p-strong">
    “对手机进行跟踪和地理定位的数据在寻找最可能的[冠状病毒]携带者方面不再有效。”
   </strong>
  </p>
  <p class="graf graf--p">
   确实，尽管 “社会疏离” 准则要求人与人之间应保持6英尺的距离，以防止冠状病毒传播，但是，智能手机的GPS功能只能精确到16英尺左右 —— 如果目标人在建筑物内或建筑物周围，效果会更糟。
  </p>
  <p class="graf graf--p">
   通过查找附近的已知Wi-Fi热点可以提高跟踪的准确性，Phunware 在2019年的一项专利中
   <a class="markup--anchor markup--p-anchor" data-href="https://www.businesswire.com/news/home/20190411005204/en/Phunware-Announces-Award-Patent-Mobile-Device-Localization" href="https://www.businesswire.com/news/home/20190411005204/en/Phunware-Announces-Award-Patent-Mobile-Device-Localization" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     声称
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    已开发了一种算法，该算法可以使用通常不准确的手机信号塔提供高精度的跟踪
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是，根据隐私研究人员 Wolfie Christl 的说法，我们有理由对整个行业对位置准确性的声称表示怀疑：这些公司以前从未真正关心过精度。
  </p>
  <p class="graf graf--p">
   Christl 解释说：“在很多情况下，在用户不知情的情况下从智能手机应用程序中收集的位置数据不仅存在缺陷和偏见，甚至是欺诈性的。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “声称拥有每个人的运动数据的移动数据经纪人，大多是在推销他们自己的能力。他们使用不可靠的数据源并寻求捷径。”
  </p>
  <p class="graf graf--p">
   正如美国公民自由联盟的 Jennifer Granick 通过电子邮件解释的那样，广告技术公司也从来不必应付高额赌注或真正的问责制，使从侵入式营销到流行病学的转向充满了麻烦。
  </p>
  <p class="graf graf--p">
   Granick说：“您不能出于任何原因而将信息重新利用。例如，传统上，广告公司不必担心准确性。就算他们搞砸了也不过是导致有人看到了自己根本不感兴趣的广告而已。但是，当我们谈论公共卫生问题、潜在的执法和就业问题时，这些公司的错误可能是灾难性的。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    从收集用于政治广告的数据、到加强监控的 Phunware 的工作重心转移，反映了广告技术行业 —— 监视资本主义 —— 的广泛趋势。
   </strong>
  </p>
  <p class="graf graf--p">
   上个月华尔街日报
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wsj.com/articles/to-track-virus-governments-weigh-surveillance-tools-that-push-privacy-limits-11584479841" href="https://www.wsj.com/articles/to-track-virus-governments-weigh-surveillance-tools-that-push-privacy-limits-11584479841" rel="noopener noreferrer" target="_blank">
    报道
   </a>
   ，白宫冠状病毒特别工作组已经与谷歌，Facebook 和 亚马逊等科技巨头以及许多初创公司会面，以开发针对冠状病毒大流行的技术跟踪解决方案。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果我们要利用商业技术挽救生命，我们如何制定政策框架才能使我们不像韩国、中国或以色列那样？” Camber Systems 公司首席执行官 Ian Allen 告诉华尔街日报。坎伯系统公司是一家总部位于华盛顿特区的新兴公司，致力于开发位置跟踪服务。
  </p>
  <p class="graf graf--p">
   总部位于纽约的数据分析公司 Verint Systems 同样宣布，该公司将出售新的解决方案，以使用地理定位技术 “检测和警告非法聚集的人群”。
  </p>
  <p class="graf graf--p">
   Verint 网站上发布的产品指南显示了警察机构如何使用监视技术来加强对公民的行动限制。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    备忘录中提到， “没有足够的警力在每个城市的每个街角部署”；但是，通过移动电话进行地理位置定位就可用于对违反隔离措施的人员 “产生相关警报并进行流行病学调查”。
   </strong>
  </p>
  <p class="graf graf--p">
   疾病预防控制中心是 Verint 现有的客户之一。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们的某些执法机构负责 Covid-19 的执法，保护公共安全以强制执行这些检疫非常重要”， Verint 首席执行官 Dan Bodner 在3月31日与投资者的电话会议上说。
  </p>
  <p class="graf graf--p">
   他补充说，该公司的 “面部识别” 产品可以帮助 “确保人们在家里得到基本的隔离，并符合要求。” ⚪️
  </p>
  <p class="graf graf--p">
   注：IYP将在未来制作一份全球性的综合报告，关于冠状病毒流行如何导致了反乌托邦噩梦的大规模兴起。这不是美国、中国、以色列、俄罗斯或任何一个国家的独特性，而是全球的，于是它必须被详细分析和思考。
  </p>
