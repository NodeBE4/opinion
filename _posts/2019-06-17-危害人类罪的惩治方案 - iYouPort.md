  <p class="graf graf--p">
   根据国际人民法庭最近的一项裁决，菲律宾总统罗德里戈·杜特尔特及其政府在美国总统唐纳德·特朗普及其政府的协助和怂恿下犯下了战争罪和危害人类罪。以下文章来自此案的证人之一。
  </p>
  <p class="graf graf--p">
   该仲裁庭于2018年9月18日和19日在比利时布鲁塞尔举行，于3月8日就这些罪行作出了
   <a class="markup--anchor markup--p-anchor" data-href="https://humanrightsphilippines.net/events/international-peoples-tribunal-2018/" href="https://humanrightsphilippines.net/events/international-peoples-tribunal-2018/" rel="noopener noreferrer" target="_blank">
    长达84页的决定
   </a>
   。仲裁庭的召集人包括国际民主律师协会、欧洲民主律师协会、霍尔丹社会主义律师协会，IBON国际和菲律宾国际人权联盟。来自埃及，法国，意大利，马来西亚，荷兰和美国的八名陪审员组成的小组听取了31名证人的证词。
  </p>
  <p class="graf graf--p">
   这些陪审员命令被告作出赔偿、恢复原状，并可能对其罪行进行起诉和制裁。虽然仲裁庭无权执行这些措施，但其事实和法律结论的调查结果可用于支持国际刑事法院（ICC）目前待决的杜特尔特政权之罪行的初步审查。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “法庭终于作出了历史性的全面决定，”菲律宾全国人民律师联盟（NUPL）主席埃德雷奥拉利亚（Edre Olalia）在一封电子邮件中告诉 Truthout。“它在陈述事实和证据方面是广泛的，精辟阐述了被告的行为和不作为之间的关系及其责任“。
  </p>
  <p class="graf graf--p">
   Olalia 补充说，这一决定“明确地传达了一个信息：
   <strong class="markup--strong markup--p-strong">
    一个不断受到专制政府和镇压政府以及剥削实体侵害的人们将能够在那些愿意给予他们战斗机会的人面前尽可能地寻求正义。” “这一决定与菲律宾人民的反抗精神和坚持不懈的努力密切相关
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   这种暴政、暴行和腐败的大部分都得到了美国的暗示或明确的认可。菲律宾和美国政府之间的邪恶联盟是长期存在的。在过去的18年里，在布什总统，奥巴马总统和特朗普总统的领导下，美国继续向菲律宾政府提供援助，使菲律宾政府能够对自己的人民犯下战争罪和危害人类罪，并剥夺人民的合法自主权。
  </p>
  <p class="graf graf--p">
   在911袭击事件发生后，布什宣布菲律宾成为反恐战争的第二战线，称其为“持久自由行动 — 菲律宾人”。菲律宾政府利用布什的竞选活动，提升其反对穆斯林以及针对异议个人和组织的恶性反叛乱计划。
  </p>
  <p class="graf graf--p">
   菲律宾政府将特定的人和群体称为“恐怖分子”，这使他们成为该政权的目标。政府还进行“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.philstar.com/headlines/2019/01/10/1883928/state-media-slammed-validating-red-tagging-journos" href="https://www.philstar.com/headlines/2019/01/10/1883928/state-media-slammed-validating-red-tagging-journos" rel="noopener noreferrer" target="_blank">
    红色标记
   </a>
   ” — 政治诽谤。
   <strong class="markup--strong markup--p-strong">
    这些标签可能导致骚扰、攻击、拘留、酷刑甚至谋杀。目标通常是人权活动家和倡导者、政治反对者、社区组织者或争取民族解放的团体
   </strong>
   。
  </p>
  <p class="graf graf--p">
   事实上，全国人民律师联合会秘书长 Benjamin Ramos 律师于
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hrw.org/news/2018/11/07/another-rights-lawyer-murdered-philippines" href="https://www.hrw.org/news/2018/11/07/another-rights-lawyer-murdered-philippines" rel="noopener noreferrer" target="_blank">
    2018年11月6日
   </a>
   在仲裁程序的两个月后被暗杀。“Ramos 是内格罗斯的一位领先的人权律师，他热情地倡导真正的土地改革和农民权利，“NUPL 在一份声明中说。
   <strong class="markup--strong markup--p-strong">
    Ramos 是杜特尔特政权杀害的第34位律师。自那以后又有两人被杀。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    仲裁庭认定被告罗德里戈杜特尔特及其政权、唐纳德特朗普及其政府犯有严重和系统侵犯公民、政治、经济、社会和文化权利的罪行; 以及人民对民族自决和发展的权利
   </strong>
   。
  </p>
  <p class="graf graf--p">
   杜特尔特根据指挥责任原则对其政府的罪行负责。如果指挥官知道或者应该知道下属会犯下谋杀罪、并且他们没有采取任何措施阻止，他们对其下属犯下的谋杀和其他罪行负有刑事责任。
  </p>
  <p class="graf graf--p">
   特朗普政府的责任是基于其作为帮凶的角色。 “国际刑事法院罗马规约”包括
   <a class="markup--anchor markup--p-anchor" data-href="https://iccforum.com/forum/permalink/110/13388" href="https://iccforum.com/forum/permalink/110/13388" rel="noopener noreferrer" target="_blank">
    协助和教唆战争罪
   </a>
   的责任。如果个人在犯罪或企图犯罪时“帮助、教唆或以其他方式协助”，则可以在国际刑事法院判定犯有战争罪。
   <strong class="markup--strong markup--p-strong">
    这包括“提供其佣金的手段。”美国政府在2017年和2018年为杜特尔特政权提供了
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.equaltimes.org/the-international-people-s?lang=en#.XIlBLlNKhbU" href="https://www.equaltimes.org/the-international-people-s?lang=en#.XIlBLlNKhbU" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     1.75亿美元
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    的外国军事融资，2019年提供了
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.rappler.com/move-ph/187953-trump-ph-war-on-drugs-karapatan" href="https://www.rappler.com/move-ph/187953-trump-ph-war-on-drugs-karapatan" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     1.11亿美元
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p">
   法庭认定杜特尔特政权对“大规模谋杀、严重侵犯正当程序权利、有增无减的杀戮、袭击、以恐怖主义标记人权维护者和政治反对者，扼杀言论自由、以及对人权的完全蔑视“负责。
  </p>
  <p class="graf graf--p">
   杜特尔特正在进行一场无情的“毒品战”，这场战争的形式是针对可疑吸毒者的暴力战争。毒品战争的大多数受害者都是来自贫民窟的穷人。一份警方备忘录命令将疑似吸毒者杀死。截至2018年7月31日，政府承认至少有
   <a class="markup--anchor markup--p-anchor" data-href="https://newsinfo.inquirer.net/1036460/drug-war-death-toll-now-at-4854-govt?utm_expid=.XqNwTug2W6nwDVUSgFJXed.1" href="https://newsinfo.inquirer.net/1036460/drug-war-death-toll-now-at-4854-govt?utm_expid=.XqNwTug2W6nwDVUSgFJXed.1" rel="noopener noreferrer" target="_blank">
    4,410名
   </a>
   涉嫌吸毒的人被杀。独立消息来源称这一数字为
   <a class="markup--anchor markup--p-anchor" data-href="http://inthesetimes.com/article/21500/peoples-tribunal-philippines-rodrigo-duterte-drug-war-executions-repression" href="http://inthesetimes.com/article/21500/peoples-tribunal-philippines-rodrigo-duterte-drug-war-executions-repression" rel="noopener noreferrer" target="_blank">
    23,000人
   </a>
   。警察声称他们采取了自卫行动。
  </p>
  <p class="graf graf--p">
   但是，全国人民律师联合会主席、法庭检察官 Neri Colmenares 认为，“包括目击证人在内的直接证据、CCTV和其他目击者都表明，警察本身杀害了那些没有反抗能力的受害者。当受害者跪下并恳求饶命时，他们依旧在实施杀戮。“
  </p>
  <p class="graf graf--p">
   Colmenares 注意到这些杀戮的厚颜无耻，他说：“就是在光天化日之下，在公共场所，在许多目击者面前杀戮……”。
   <strong class="markup--strong markup--p-strong">
    菲律宾的官员有一种不受惩罚的文化。进行非法杀戮的警察不会被绳之以法。他们会
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hrw.org/news/2018/06/07/philippine-police-promotions-affront-drug-war-victims" href="https://www.hrw.org/news/2018/06/07/philippine-police-promotions-affront-drug-war-victims" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     晋升到更高的职位
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    许多律师害怕为毒品犯罪嫌疑人辩护，因为担心他们自己可能会被杀害。自杜特尔特于2016年7月1日就职以来，该政权非法杀害了10名检察官，21名律师，3名法官和13名记者
   </strong>
   。
  </p>
  <p class="graf graf--p">
   法庭写道：“对于人权维护者以及批评当前不民主和反人民政策和制度的菲律宾民间社会的法外杀人事件也愈演愈烈。”“截至2018年6月，
   <strong class="markup--strong markup--p-strong">
    进步运动组织的169名领导人成为法外杀人（EJKs）的受害者，另有509名政治犯被非法监禁，受到捏造的刑事指控和栽赃的证据
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   杜特尔特毫无歉意。在2018年9月27日，他
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2018/09/27/world/asia/rodrigo-duterte-philippines-drug-war.html" href="https://www.nytimes.com/2018/09/27/world/asia/rodrigo-duterte-philippines-drug-war.html" rel="noopener noreferrer" target="_blank">
    公开承认
   </a>
   ，“我唯一的罪过是法外杀人。”
  </p>
  <p class="graf graf--p">
   国际刑事法院首席检察官 Fatou Bensouda 在2016年10月的一份声明中
   <a class="markup--anchor markup--p-anchor" data-href="https://www.icc-cpi.int/pages/item.aspx?name=161013-otp-stat-php" href="https://www.icc-cpi.int/pages/item.aspx?name=161013-otp-stat-php" rel="noopener noreferrer" target="_blank">
    写道
   </a>
   ，就菲律宾的情况，法外杀人可能属于国际刑事法院的管辖范围，“如果是根据国家政策实施的这种攻击，作为对平民人口进行广泛或系统攻击的一部分。“这就是危害人类罪的定义。
  </p>
  <p class="graf graf--p">
   证人在法庭上作证说，嫌犯和囚犯遭受身心折磨。棉兰老岛的矿工 Janry Mensis 通过视频作证。他描述了他和他的兄弟是如何被逮捕，拘留和折磨的。他们被捆绑并被扣留九天。他们的嘴巴被塞入包装带，然后士兵试图勒死他们。当他的兄弟们假装失去知觉时，他们被扔进了一个装有木头和石油的坑里再点燃。士兵离开后他们逃出了那个坑。他们每个人都遭受了三度烧伤和酷刑造成的其他伤害。
  </p>
  <p class="graf graf--p">
   杜特尔特于2017年5月23日在棉兰老岛宣布戒严，是为了回应一个据称受ISIS启发的群体入侵城市（伊斯兰国也被称为 Daesh）。
   <strong class="markup--strong markup--p-strong">
    他的政府利用戒严法进行非法逮捕和拘留，强迫失踪，强迫流离失所和任意剥夺财产，破坏清真寺和学校，以及任意拒绝向陷入交火的平民提供人道主义援助的人
   </strong>
   。
  </p>
  <p class="graf graf--p">
   根据“罗马规约”和“日内瓦公约”，谋杀，酷刑和残忍待遇构成战争罪。
  </p>
  <p class="graf graf--p">
   作为针对任何平民的广泛或系统攻击的一部分而发生的谋杀或酷刑，构成“罗马规约”规定的危害人类罪。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    菲律宾和美国政府并不是审判庭中唯一接受审判的实体。其他被告包括国际货币基金组织（IMF），世界银行，世界贸易组织（WTO）以及在菲律宾开展业务的跨国公司和外国银行
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “杜特尔特的经济政策导致剥夺真正的政府服务，因为他们将公共资金转移到被告世界银行、国际货币基金组织、世界贸易组织和跨国公司要求的腐败和大额项目，”该法庭写道。
  </p>
  <p class="graf graf--p">
   仲裁庭认定，杜特尔特“实行了反民主和排他性的经济和治理，因为他通过系统地侵犯基本人权，如在采矿业中所体现的那样，在菲律宾开展业务的被告行为人和跨国实体强加或影响了新自由主义政策。 “此外，仲裁庭的结论是，”这更加严重地侵犯了人民的社会、经济和文化权利。“
  </p>
  <p class="graf graf--p">
   证人证实“剥夺了数百万菲律宾人生计的剥削制度的影响，摧毁了边缘化穷人的棚屋，抢走了农民的土地，并通过永久的合同化和劳工出口使工人陷入永久的贫困“，Colmenares 总结道。
  </p>
  <p class="graf graf--p">
   仲裁庭的结论是，杜特尔特政权“一直未能提供工作的基本权利; 这包括生活工资和正规就业; 达到适当的生活水平; 以及“健康、住房和教育。”法庭还指责政权强加“主要针对穷人的新税收; 并迫使贫困家庭流离失所，在他们的土地上建造旅游项目。“
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “农民被剥夺了他们耕种的土地多年并遭到袭击; 工人被剥削，他们的罢工被暴力驱逐; 城市贫民在维护自己的权利时仍然无家可归并受到威胁; 教育已商业化，绝大多数人无法接受教育，“该法庭指出。此外，“根据劳务输出政策，每天都有数千人被迫迁移，包括护士; 生计权受到限制; 被困的海外工人被忽视和抛弃。“
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “杜特尔特基本上表明了他对美国在亚太地区的帝国主义目标的忠诚，”该法庭得出结论。他的政府“也推翻了人民阻止美军基地的胜利。”
  </p>
  <p class="graf graf--p">
   仲裁庭解释了美国在菲律宾的军事基地如何促进杜特尔特的反叛乱计划：“美国存在以及美国军队的永久和扩大基地正在进一步鼓励被告杜特尔特政府实施反叛乱计划 Oplan Kapayapaan ，该计划是在2009年美国反叛乱指南之后制定并由被告美国政府资助。“
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    美国政府对杜特尔特政府的援助包括提供“情报，资金，方向，培训和武器，以促进和追求其在该地区的经济和地缘政治利益”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   仲裁庭采用的我的证词如下：“美国对菲律宾政府的军事援助有助于其对本国人民犯下战争罪和危害人类罪。与菲律宾领导人一样，美国政治和军事领导人可能需要在国际刑事法院承担战争罪和危害人类罪的助手和教唆者的责任。
  </p>
  <p class="graf graf--p">
   菲律宾人民有自决权，其中包括发展权。正如“发展权利宣言”所述，“凭借”自决，人民“有权自由决定其政治地位，追求经济，社会和文化发展。”人民“对其财富和资源拥有完全主权的不可剥夺的权利。“
  </p>
  <p class="graf graf--p">
   证人记录了对土著人民和少数民族的广泛和有系统的攻击，以及使用白磷气体和强迫失踪手段 ，这些都构成危害人类罪。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “菲律宾和美国的政治和军事领导人不会因犯罪而逍遥法外。为菲律宾人民伸张正义不仅仅是为菲律宾人民的。全世界的人都有责任将罪犯绳之以法，“法庭写道，“菲律宾人民继续为民族解放和自决而进行勇敢的斗争。对菲律宾和美国官员的罪行申诉法律责任将有助于阻止他们犯下更多罪行。“
  </p>
  <p class="graf graf--p">
   即使国际刑事法院最终没有调查和起诉菲律宾政府军事和警察官员犯下的战争罪和危害人类罪，其他国家也可以根据既定的普遍管辖权原则将罪犯绳之以法。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当嫌疑人的祖国不能或不愿起诉时，任何国家都可以审判外国国民的战争罪和危害人类罪，而杜特尔特则不愿起诉那些对菲律宾人民犯下滔天罪行的人
   </strong>
   。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
