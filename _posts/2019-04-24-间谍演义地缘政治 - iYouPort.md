   间谍演义地缘政治

 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <strong>
      <em>
       谍战片真的没什么惊奇了，摩萨德行动的现实比电影更神奇，他们是怎么做的？
      </em>
     </strong>
    </span>
   </li>
  </ul>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “每日邮报”在一份新报道中讲述了一个令人震惊的间谍揭露。一名伊朗核科学家被偷运出伊朗并被带入英国，然后再前往美国 — 美国此前曾帮助以色列情报部门计划在伊朗国内暗杀另一名核科学家。
  </p>
  <p class="graf graf--p">
   据报道，这名叛逃的伊朗科学家被确认为47岁的男性“核技术员”，他在一艘充气筏上穿过英吉利海峡，躲藏在一大群移民中。报告说，“军情六处的间谍通过使用海峡移民危机作为掩护，将这位叛逃的伊朗核科学家偷运到英国。”
  </p>
  <p class="graf graf--p">
   至关重要的是，这名未具名的叛逃者涉嫌参与2012年暗杀德黑兰的 Mostafa Ahmadi Roshan，当时他被认为是德黑兰的核心专家。在“每日邮报”称之为去年10月开始的“萃取”行动之前，这位叛逃的科学家似乎一直在为摩萨德和其他外国情报机构工作。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “每日邮报”援引消息人士透露，2012年“Sunday Times”
   <a class="markup--anchor markup--p-anchor" data-href="https://www.businessinsider.com/sunday-times-mossad-assassinated-iranian-scientist-2012-1" href="https://www.businessinsider.com/sunday-times-mossad-assassinated-iranian-scientist-2012-1" rel="noopener noreferrer" target="_blank">
    确认摩萨德
   </a>
   是汽车爆炸事件的幕后黑手，该事件杀死了 Mostafa Roshan，据报道，这名47岁的叛逃科学家已经提供了信息。
  </p>
  <p class="graf graf--p">
   根据2012年1月
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/commentisfree/2012/jan/16/iran-scientists-state-sponsored-murder" href="https://www.theguardian.com/commentisfree/2012/jan/16/iran-scientists-state-sponsored-murder" rel="noopener noreferrer" target="_blank">
    “卫报”对暗杀事件的描述
   </a>
   ：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    1月11日上午，伊朗纳坦兹铀浓缩设施副主任穆斯塔法·艾哈迈迪·罗森（Mostafa Ahmadi Roshan）被安装在汽车门上的磁力炸弹炸毁了他的汽车，当年他32岁，已婚有一个小儿子。他没有武装，事发地点附近没有任何战场。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    自2010年以来，其他三名伊朗核科学家都在类似的情况下被杀，其中包括一名35岁的电子专家 Darioush Rezaeinejad 去年七月在德黑兰他女儿的托儿所外面被枪杀
   </em>
   。
  </p>
  <p class="graf graf--p">
   最近在2018年制定了一项计划，目标就是这位重量级情报资产和叛逃者，其中涉及摩萨德帮助他逃离伊朗到土耳其，之后军情六处利用移民危机作为他进入英国的“掩护”。
  </p>
  <p class="graf graf--p">
   他的失踪立即引发了伊斯兰革命卫队的广泛搜查。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “每日邮报”引用匿名情报来源报告描述：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    即使进入了法国，如何让他进入英国的问题仍然存在。我们不能简单地让他进来。虽然不寻常，但最终确定，将他渗透到一群准备乘船穿越海峡寻求避难的移民中
   </em>
   。
  </p>
  <p class="graf graf--p">
   显然这样做是有效的，报告说：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    这名47岁的男子在充气船上隐藏在其他伊朗难民中间穿越海峡。据报道，这名科学家通过陆路前往加来附近的法国海岸，以色列特工莫萨德将其从其祖国走私到邻近的土耳其
   </em>
   。
  </p>
  <p class="graf graf--p">
   这种奇怪行动的理由是军情六处想要采访叛逃者，但鉴于英国仍然是2015年伊朗核协议的一部分，于是这件事在政治上是敏感的。
  </p>
  <p class="graf graf--p">
   据报道军情六处在最终继续前往美国之前采访了这名男子。 “每日邮报”
   <a class="markup--anchor markup--p-anchor" data-href="https://www.dailymail.co.uk/news/article-6688053/British-spies-MI6-smuggled-defecting-Iranian-nuclear-scientist-UK-dinghy.html" href="https://www.dailymail.co.uk/news/article-6688053/British-spies-MI6-smuggled-defecting-Iranian-nuclear-scientist-UK-dinghy.html" rel="noopener noreferrer" target="_blank">
    称
   </a>
   ，“军情六处在他飞往美国之前就伊朗的核计划采访了这位核技术人员”。
  </p>
  <p class="graf graf--p">
   与此同时，以色列的媒体机构已经开始传播这个故事，这份报道尚未得到任何高层政府的确认，他们也不太可能承认这种涉及以色列摩萨德、军情六处或中央情报局的秘密行动。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “耶路撒冷邮报”
   <a class="markup--anchor markup--p-anchor" data-href="https://www.jpost.com/Israel-News/Mossad-MI6-smuggled-Iranian-nuclear-scientist-to-the-UK-Report-580237" href="https://www.jpost.com/Israel-News/Mossad-MI6-smuggled-Iranian-nuclear-scientist-to-the-UK-Report-580237" rel="noopener noreferrer" target="_blank">
    充分记录了
   </a>
   针对伊朗官员的这种奇异和高风险的情报和暗杀行动：“据说
   <strong class="markup--strong markup--p-strong">
    以色列发动了一系列秘密行动，通过消除那些被认为有价值的科学家来阻止他们参与伊朗的核计划
   </strong>
   。 “但报告还说，“以色列从未证实或否认过这种说法。”
  </p>
  <p class="graf graf--p">
   如果中央情报局目前正在与现在居住在美国土地上的伊朗高级核科学家叛逃者交谈，那么这是否可能成为下一个
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2011/feb/15/curveball-iraqi-fantasist-cia-saddam" href="https://www.theguardian.com/world/2011/feb/15/curveball-iraqi-fantasist-cia-saddam" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     ”Curveball”
    </strong>
   </a>
   型的线人，为政府提供所需的一切，以便为伊朗的政权更迭提供理由？
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    M
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.zerohedge.com/news/2019-02-11/mossad-mi6-smuggled-defected-iranian-nuclear-scientist-migrant-caravan-uk" href="https://www.zerohedge.com/news/2019-02-11/mossad-mi6-smuggled-defected-iranian-nuclear-scientist-migrant-caravan-uk" rel="noopener noreferrer" target="_blank">
    ossad, MI6 Smuggled Defected Iranian Nuclear Scientist In Migrant Caravan To UK
   </a>
   . The 47-year-old man is thought to have hidden among other Iranians on an inflatable boat when making the crossing.
  </p>
