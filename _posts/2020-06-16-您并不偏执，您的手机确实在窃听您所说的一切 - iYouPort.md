  <p class="graf graf--p">
   场景像惊悚片一样：你拿出手机，然后看到 AirPods 的广告。
  </p>
  <p class="graf graf--p">
   等一下。我不只是和朋友谈论了有关 AirPods 的话题吗？难道我的电话在偷听我说的话？
  </p>
  <p class="graf graf--p">
   是的，可能正是如此。
   <strong class="markup--strong markup--p-strong">
    使用默认设置时，您所说的一切都可能通过设备的内置麦克风录制
   </strong>
   。尽管没有具体证据，但许多人认为自己的手机会定期收集语音数据。
  </p>
  <p class="graf graf--p">
   而且，
   <strong class="markup--strong markup--p-strong">
    您的手机并不是偷窥和窃听您的唯一设备
   </strong>
   。如果您不保护智能电视，骇客可以把你家的电视变成监视器。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     这种事CIA至少6年前就做过
    </strong>
    ，见《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e9%9b%b6%e6%97%a5%e6%bc%8f%e6%b4%9e%e5%92%8c-cia-%e9%bb%91%e5%ae%a2%e5%b7%a5%e5%85%b7/" href="https://www.iyouport.org/%e9%9b%b6%e6%97%a5%e6%bc%8f%e6%b4%9e%e5%92%8c-cia-%e9%bb%91%e5%ae%a2%e5%b7%a5%e5%85%b7/" rel="noopener noreferrer" target="_blank">
     零日漏洞和 CIA 黑客工具
    </a>
    》
   </li>
   <li class="graf graf--li">
    而且您现在根本买不到普通的无法被入侵的电视，因为《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%94%b5%e8%a7%86%e6%9c%ba%e7%9a%84%e9%98%b4%e8%b0%8b/" href="https://www.iyouport.org/%e7%94%b5%e8%a7%86%e6%9c%ba%e7%9a%84%e9%98%b4%e8%b0%8b/" rel="noopener noreferrer" target="_blank">
     电视机的阴谋
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   ⚠️是的，这完全是合法的，但这并不意味着您必须喜欢它。
  </p>
  <p class="graf graf--p">
   许多人都为看到他们仅仅是说过的事而不是在浏览器上搜索的东西被莫名其妙地针对而感到惊讶。幸运的是，有一些方法可以阻止您的设备窃听您。
  </p>
  <p class="graf graf--p">
   智能手机配备了大量监控设备：多个麦克风和摄像头旨在吸收音频和视频。尽管这些工具对于创建媒体可能有用，但它们
   <strong class="markup--strong markup--p-strong">
    绝对是监视者的金矿
   </strong>
   。
  </p>
  <p class="graf graf--p">
   早在
   <strong class="markup--strong markup--p-strong">
    2018年
   </strong>
   中，一位Vice的记者就
   <a class="markup--anchor markup--p-anchor" data-href="https://www.vice.com/en_nz/article/wjbzzy/your-phone-is-listening-and-its-not-paranoia" href="https://www.vice.com/en_nz/article/wjbzzy/your-phone-is-listening-and-its-not-paranoia" rel="noopener noreferrer" target="_blank">
    进行了一次实验
   </a>
   ，以了解智能手机收听用户对话的接近程度。为了测试手机，该记者连续五天每天两次讲预选的话。同时，他监视了自己的 Facebook feed，看是否发生了任何变化。
  </p>
  <p class="graf graf--p">
   ⚠️果然，这些变化似乎在一夜之间就出现了。他的测试用语之一涉及 “回到大学”，第二天早晨，记者就看到了暑期课程的广告。
  </p>
  <p class="graf graf--p">
   然后，他将自己的测试用语改为 “便宜的衬衫”，并很快在 Facebook 提要上看到了低成本服装的广告。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     非常推荐这篇文章
    </strong>
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e4%bd%a0%e4%b8%8d%e5%81%8f%e6%89%a7%ef%bc%8c%e9%82%a3%e5%b0%b1%e6%98%af%e7%96%af%e4%ba%86/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e4%bd%a0%e4%b8%8d%e5%81%8f%e6%89%a7%ef%bc%8c%e9%82%a3%e5%b0%b1%e6%98%af%e7%96%af%e4%ba%86/" rel="noopener noreferrer" target="_blank">
     “如果你不偏执，那就是疯了”
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   该报告引发了关于社交媒体平台的监控效果的研究热潮。
  </p>
  <p class="graf graf--p">
   尽管并非每项研究都提供了明确的答案，但由于多个应用程序和社交媒体平台的用户协议中的暗示，人们对此问题达成了普遍共识。
  </p>
  <p class="graf graf--p">
   这些用户协议明确声明：录制的音频可用于目标广告目的。有趣的是，这种做法并不违法。此举使科技公司可以进一步扩大监视行为，以鼓励人们购买那些根本不需要的东西。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    怎么做才能避免被监视？
   </strong>
  </p>
  <p class="graf graf--p">
   您可以使用多种方法来减轻智能手机的间谍能力。也就是说，您可能无法使用某些方便的功能，例如 唤醒词和语音助手，因此
   <strong class="markup--strong markup--p-strong">
    您必须确定这些功能是否值得牺牲您的人权、甚至人身安全。
   </strong>
  </p>
  <p class="graf graf--p">
   最大的漏洞来自大多数语音助手的 “始终在线” 能力。要拾取 “Hey Siri” 之类的唤醒字，麦克风需要始终保持打开状态 —— ⚠️这意味着您的手机始终在监听。
  </p>
  <p class="graf graf--p">
   也就是说您需要关闭手机上的 “始终打开” 的麦克风功能。方法如下：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、停用 “嘿 Siri”
   </strong>
  </p>
  <p class="graf graf--p">
   苹果因录制 Siri 用户的声音已经备受抨击。
  </p>
  <p class="graf graf--p">
   要关闭 “Siri”，请导航至iOS设备的 “设置”，然后导航至 Siri＆Search。然后，将 “Hey Siri” 切换为 “关”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、禁用 “OK Google”
   </strong>
  </p>
  <p class="graf graf--p">
   每次您使用 “OK Google” 或其他语音控制功能时，它们都会记录您的查询，并将其保存到您的Google帐户中。
  </p>
  <p class="graf graf--p">
   如果您在 Android 上使用 Google Assistant，请打开 Goog​​le Assistant 设置。说 “ OK Google” 或按住电话的主按钮，然后点击右上角的文件图标，然后点击三个点菜单。选择 More，然后选择设置。
  </p>
  <p class="graf graf--p">
   此时，您需要向下滚动到设备类别并选择您的手机，然后在顶部附近的 Google 助手部分中，点击蓝色的滑块按钮将其更改为灰色，从而关闭 Google 助手。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、禁用所有应用程序的麦克风访问
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️对于某些应用程序 —— 如 Facebook —— 仅禁用手机上的麦克风功能是不够的。该社交网站是规则的例外，除非您进行更改，否则它将继续监听。
  </p>
  <p class="graf graf--p">
   您可以通过以下方法关闭 Facebook 的音频录制，而且相同的步骤适用于所有使用麦克风的应用：
  </p>
  <p class="graf graf--p">
   iPhone ——
  </p>
  <p class="graf graf--p">
   进入设置，将麦克风开关滑动到左侧，使其从绿色变为白色，就是关了。或者，您可以转到设置&gt;&gt;隐私&gt;&gt;麦克风，然后找到 Facebook 并进行相同的操作。请注意，在这里您也可以为其他应用打开和关闭麦克风。
  </p>
  <p class="graf graf--p">
   Android ——
  </p>
  <p class="graf graf--p">
   打开设置，然后选择应用程序管理器。查找 Facebook，然后点击 “应用程序权限”，然后关闭麦克风。
  </p>
  <p class="graf graf--p">
   关闭 Facebook 的麦克风访问权限会影响和禁用某些功能，例如实时视频。什么时候使用这些功能再打开一下下，用完马上关闭它。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    最后
   </strong>
  </p>
  <p class="graf graf--p">
   如果您 “必须” 使用智能手机，那么请注意应该注意的事项，不要让它伤害您和您的家人。⚪️
  </p>
