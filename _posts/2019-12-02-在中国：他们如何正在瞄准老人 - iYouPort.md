  <p class="graf graf--p">
   半年前我们介绍过《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E8%80%81%E9%BE%84%E5%8C%96%E4%BA%92%E8%81%94%E7%BD%91/" href="https://www.iyouport.org/%E8%80%81%E9%BE%84%E5%8C%96%E4%BA%92%E8%81%94%E7%BD%91/" rel="noopener noreferrer" target="_blank">
    老龄化互联网
   </a>
   》的问题，提醒：现在的互联网正在发生变化，从年轻人的天下转变为老年人的天下，这意味着什么？如果恶意行为者通过你的缺乏网络素养的祖母将带有间谍软件的链接转发给你，这意味着什么？
  </p>
  <p class="graf graf--p">
   中国正在加强对此的利用。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8600 jetpack-lazy-image" data-lazy-sizes="(max-width: 796px) 100vw, 796px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-41.png?resize=796%2C1244&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-41.png?w=796&amp;ssl=1 796w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-41.png?resize=192%2C300&amp;ssl=1 192w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-41.png?resize=768%2C1200&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-41.png?resize=655%2C1024&amp;ssl=1 655w" data-recalc-dims="1" height="1244" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-41.png?resize=796%2C1244&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="796"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8600" data-recalc-dims="1" height="1244" sizes="(max-width: 796px) 100vw, 796px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-41.png?resize=796%2C1244&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-41.png?w=796&amp;ssl=1 796w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-41.png?resize=192%2C300&amp;ssl=1 192w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-41.png?resize=768%2C1200&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-41.png?resize=655%2C1024&amp;ssl=1 655w" width="796"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   晚餐后不久，上海市中心的襄阳公园变成了一个舞厅。
  </p>
  <p class="graf graf--p">
   当老人们在梧桐树下翩翩起舞时，扬声器播放出古老的流行歌曲。
  </p>
  <p class="graf graf--p">
   整体上是一幅老年人怀旧的场景 —— 直到你遇到施女士和周先生，他们是一对70多岁的老夫妇，他们对华尔兹的热情和对智能手机的热情基本一样。
  </p>
  <p class="graf graf--p">
   老夫妇都喜欢 WeChat，一个中国的消息应用程序。 “我可以不吃饭，但不能没有智能手机，”施女士承认说。
  </p>
  <p class="graf graf--p">
   即便如此，这对老夫妻仍然是不寻常的。
  </p>
  <p class="graf graf--p">
   中国50岁以上的人中只有不到三分之一的人表示在2016年拥有了智能手机，这是皮尤研究中心获得相关数据的最近一年，这个数字只占美国市场份额的一半。
  </p>
  <p class="graf graf--p">
   中国社会科学院和腾讯公司的一项调查发现，只有17％的人经常使用手机购物；近一半的人从未这样做过。
  </p>
  <p class="graf graf--p">
   科技公司正努力从网上吸引更多像施女士和周先生这样的老人，并预计在2020年，让中国老年人在花费于消费品上的7万亿元人民币（1万亿美元）中占更大份额。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    对于科技公司而言，中国2.5亿多老龄人口（即18％的人口）的社会脱节是一个“机会”
   </strong>
   —— 与年轻人不同，退休人员会争先恐后地争夺成千上万的应用程序。
  </p>
  <p class="graf graf--p">
   一旦学会了上网，这些老年人就开始挥霍。 2017年，一家大型电子商务公司“京东”发现，
   <strong class="markup--strong markup--p-strong">
    这些老年人的花费是普通用户的2.3倍
   </strong>
   。
  </p>
  <p class="graf graf--p">
   老龄人口在由中国寡头阿里巴巴控制的在线现金管理服务“支付宝”中的平均存款为7000元，而所有年龄段人口的平均为4000元。
  </p>
  <p class="graf graf--p">
   当商店拒绝现金时，熟练于在线支付的人可能会比典型的老年人做得更好。
  </p>
  <p class="graf graf--p">
   但是，创业公司看到了丰富的选择。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “I Have A Partner”，这是一个约会应用程序，去年推出了大胆的字体和语音消息，专门服务于打字很慢的人。
  </p>
  <p class="graf graf--p">
   糖豆广场舞（“Jelly Bean Square Dance”）开始发布舞蹈视频（还是带滤镜的，可以消除皱纹……），他们渴望成为专门服务于老人的一站式商店。该应用程序自2015年推出以来下载量
   <strong class="markup--strong markup--p-strong">
    已经超过2亿
   </strong>
   !
  </p>
  <p class="graf graf--p">
   当然，寡头公司也非常希望迅速锁定老年人。
  </p>
  <p class="graf graf--p">
   60岁以上的人中有五分之四的人的移动数据都在微信上，而18–35岁的人中这个数字为7％。
  </p>
  <p class="graf graf--p">
   2017年，腾讯制作了一段视频，讲述了老一辈人对技术的困惑，鼓励孩子们教会他们的父母如何借助 WeChat “帮助”功能学习使用。
  </p>
  <p class="graf graf--p">
   55岁以上的人现在是微信用户增长最快的群体。去年，阿里巴巴在线商店淘宝专门为老年顾客推出了“pay-for-me”选项，供家庭成员帮助代付款。
  </p>
  <p class="graf graf--p">
   该网站每天播放超过1,000个针对老年人的直播节目。
  </p>
  <p class="graf graf--p">
   去年阿里巴巴收购的送餐服务 Ele.me 现在正在为老年人提供膳食和药品交付服务，并提供一次性帮助，例如换灯泡。
  </p>
  <p class="graf graf--p">
   预计到2050年，60岁以上人口的比例将翻一番，达到总数的三分之一。
  </p>
  <p class="graf graf--p">
   请注意，我们翻译经济学人这篇文章想要提醒的并不是您家老人花了多少钱、或者跳了多少广场舞，而是，
   <strong class="markup--strong markup--p-strong">
    他们对互联网的浅薄理解很可能让他们成为最危险的弱点
   </strong>
   。如最上那张图所示。⚪️
  </p>
