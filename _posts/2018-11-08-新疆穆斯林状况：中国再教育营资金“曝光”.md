title: 新疆穆斯林状况：中国再教育营资金“曝光”
---

<div id="entry">
<div class="at-above-post addthis_tool" data-url="http://zhanlve.org/?p=6250">
</div>
<p>
<span style="font-size: 12pt;">
<img alt="中国被指大规模拘禁新疆维吾尔族人以及其他穆斯林。" src="https://boxun.com/news/images/2018/11/201811071930china3.jpg"/>
</span>
<br/>
</p>
<p>
<span style="font-size: 12pt;">
   中国被指大规模拘禁新疆维吾尔族人以及其他穆斯林。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   一份最新的报告指，中国在2017年大幅提升在西部新疆自治区的安保支出，而据称那里有数以十万计穆斯林被拘禁。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   总部位于美国的詹姆斯敦基金会（Jamestown Foundation）声称，在2016至2017年间升幅达213%的费用支出，“几乎解释了全部与安保相关的设施建设”。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   另有卫星数据显示，新的安保设施在2017年显著增加。中国方面称，这些是培训中心。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   不过，德国学者郑国恩（Adrian Zenz）在报告中核查的数据显示，在新疆，职业培训的支出在2017年实际上下降了7%。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   而与此同时，与安保相关的设施建设，则上升了近200亿元人民币（29亿美元）。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   郑国恩表示，与新疆的费用数额“所反映的支出模式相一致的，是高安全规格的政治再教育设施和项目建设，它们的目的是以最简单的程序对数以十万计的（维吾尔人）进行监禁。”
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   郑国恩此前曾指，根据地方政府的招标文件，在新疆至少有几十万甚至上百万的维吾尔族及其他穆斯林少数族裔可能被拘禁。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   今年较早前，联合国人权部门也引用了这些估算数据。中国则否认这些设施是拘留营。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
<img alt="中国被指大规模拘禁新疆维吾尔族人以及其他穆斯林。" src="https://boxun.com/news/images/2018/11/201811071930china2.jpg"/>
</span>
</p>
<p>
<span style="font-size: 12pt;">
   该地区的高层官员表示，“职业再教育和培训项目”帮助人们“反思自己的错，并清楚看到恐怖主义和宗教极端主义的实质与危害”。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   据地区官员称，营内会教授中国历史、语言和文化的课程。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   过去曾被拘禁过但现在身居海外的维吾尔人曾向BBC表示，他们在教育营内必须会唱共产党的歌曲，还要准确地背出法律条文，否则就会被打。其中一名男子说，2015年他因为被警察发现他的手机里有一张披面纱的女人照片而被拘禁。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   郑国恩是德国欧洲文化与神学学院（European School of Culture and Theology）的中国民族政策专家。他还发现，新疆再教育营的建设机构与过去中国已废除的劳改体系监督机构是同一个。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   另一方面，周二（11月6日）在日内瓦，联合国人权理事会审议了中国的人权纪录。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   在就英国、法国、德国和美国对于疑似大规模监禁的担忧作出回应时，中国代表乐玉成表示，中国“不接受一些国家，出于政治动机，作出充满偏见的指控”。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   乐玉成坚称，在人权问题上，中国必须自主“选择自己的道路”。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   美国方面已经表示，会考虑针对中国的政策对其实施制裁。人权观察组织（Human Rights Watch）指，中国的政策侵犯了言论、宗教和个人自由。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   维吾尔族是主要生活在新疆的突厥语穆斯林少数民族，占新疆居民的45%。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   他们在文化和种族上认为自己更接近于中亚诸国，而他们的语言也与土耳其语相似。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   最近数十年，大量中国汉族人移居新疆，维吾尔族人感觉自己的文化和生活受到了威胁。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   在行政上，新疆与其南边的西藏一样，属于中国的自治区。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   出处 ：BBC
