  <p class="graf graf--p">
   卢旺达提出了世界上第一个全国范围的DNA数据库，该项目将涉及从该国1200万公民中收集DNA样本。
  </p>
  <p class="graf graf--p">
   该计划引起了人权人士的严重担忧，他们认为该数据库可能被政府滥用并违反国际人权法。
  </p>
  <p class="graf graf--p">
   卢旺达司法部长兼司法部长 Johnston Busingye 宣布了该数据库的计划。
  </p>
  <p class="graf graf--p">
   他在该国首都基加利说，该项目将“有助于打击强奸和谋杀等罪行”。
  </p>
  <p class="graf graf--p">
   他补充说：“我想向您保证，最终目标是拥有所有必要的设备和技术知识，以提供关于谁应对犯罪负责的准确信息”。
  </p>
  <p class="graf graf--p">
   据卢旺达日报“新时代报”报道，官员们仍然需要为该项目确定预算，并通过议会推动立法，以允许建立数据库。
  </p>
  <p class="graf graf--p">
   人权组织对新计划持谨慎态度，因为他们认为政府可能会滥用DNA数据，这可能会揭示广泛的亲密医学和遗传细节。
  </p>
  <p class="graf graf--p">
   慈善机构 Privacy International 的全球项目负责人 Alexandrine Pirlot de Corbion 表示：“这种数据库在未来可能会被误用，这是一种固有的风险。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <strong class="markup--strong markup--p-strong">
    在世界各地，我们已经看到大量数据被滥用于压制的情况 — — 允许当局识别和描述政府可能想要找到的任何社会群体
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   根据英国法律改革组织 Justice 的说法，从法律角度来看，保留这些数据本身也存在问题，并且根据其实施情况，可能会违反“国际人权法案”的规定。
  </p>
  <p class="graf graf--p">
   在1994年种族灭绝事件发生后，卢旺达仍然遭受紧张局势，当时该国胡图族政治精英指挥大屠杀该国图西族人口后，
   <strong class="markup--strong markup--p-strong">
    在短短一百天内就杀死了80万卢旺达人
   </strong>
   。
  </p>
  <p class="graf graf--p">
   据估计，该国图西人口的70％在种族灭绝中丧生。
  </p>
  <p class="graf graf--p">
   虽然官员希望卢旺达成为第一个拥有全面 DNA 数据库的国家，该数据库记录了该国所有公民 — — 其他几个国家也已经开始实施大规模的 DNA 数据库。
  </p>
  <p class="graf graf--p">
   有人担心中国政府正在使用 DNA 测试来压迫该国主要的穆斯林维吾尔少数民族。
  </p>
  <p class="graf graf--p">
   人权组织表示，中国已经在新疆建立了一个庞大的维吾尔族 DNA 数据库，未经近3600万人的同意收集了样本，假扮为免费体检计划的一部分。
  </p>
  <p class="graf graf--p">
   2014年，中国研究人员发
   <a class="markup--anchor markup--p-anchor" data-href="https://www.independent.co.uk/news/world/asia/china-expands-dna-testing-xinjiang-muslims-security-crackdown-human-rights-watch-a7739791.html" href="https://www.independent.co.uk/news/world/asia/china-expands-dna-testing-xinjiang-muslims-security-crackdown-human-rights-watch-a7739791.html" rel="noopener noreferrer" target="_blank">
    表了一篇论文
   </a>
   ，描述了如何使用 DNA 测试来区分维吾尔族与其他种族群体。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.independent.co.uk/news/world/asia/china-muslims-re-education-camps-amnesty-uighur-religion-human-rights-watch-a8678156.html" href="https://www.independent.co.uk/news/world/asia/china-muslims-re-education-camps-amnesty-uighur-religion-human-rights-watch-a8678156.html" rel="noopener noreferrer" target="_blank">
    研究人员认为
   </a>
   ，在中国所谓的“再教育”营中，有超过一百万的维吾尔人被系统性地拘留。
  </p>
  <p class="graf graf--p">
   中国新疆政府否认它利用其医疗计划建立 DNA 数据库，并否认维吾尔族人在没有正当程序的情况下被拘留。
  </p>
  <p class="graf graf--p">
   2015年，科威特通过了世界上第一部法律，要求所有公民和访客提交他们的 DNA 样本，但它在生效之前就已被撤销。
  </p>
  <p class="graf graf--p">
   它的目的被称为“识别恐怖分子”，但在2017年，该国的宪法法院裁定该法律违反了科威特对个人自由的宪法保障。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    R
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.independent.co.uk/news/world/africa/rwanda-dna-database-human-rights-privacy-a8832051.html" href="https://www.independent.co.uk/news/world/africa/rwanda-dna-database-human-rights-privacy-a8832051.html" rel="noopener noreferrer" target="_blank">
    wanda sparks human rights concerns in proposing world’s first nationwide DNA database
   </a>
   . The database will require officials to take DNA samples from all of Rwanda’s 12 million citizens
  </p>
