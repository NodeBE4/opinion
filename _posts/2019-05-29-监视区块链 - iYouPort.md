   监视区块链

 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      当权者对区块链的监视需求与日俱增，现在他们毫不掩饰这点，明目张胆地雇佣全球最臭名昭著的间谍软件制造者来实现监视能力。谁又上了头条？
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   2月份，当加密货币行业巨头 Coinbase 在
   <a class="markup--anchor markup--p-anchor" data-href="https://blog.coinbase.com/welcoming-neutrino-to-coinbase-b3f56171850d" href="https://blog.coinbase.com/welcoming-neutrino-to-coinbase-b3f56171850d" rel="noopener noreferrer" target="_blank">
    宣布收购
   </a>
   一家名为 Neutrino 的小型创业公司时引发了公众愤怒。
  </p>
  <p class="graf graf--p">
   通常情况下，这样的收购可能会看起来比较普通，但是 Neutrino 可不是一般的创业公司。该公司由 Hacking Team 的三名前雇员创建，
   <strong class="markup--strong markup--p-strong">
    该团队是一家头牌臭名昭著的意大利监控技术供应商，曾多次向暴政销售间谍软件，其中包括埃塞俄比亚、沙特阿拉伯和苏丹等专制国家。在
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://docs.google.com/document/d/1tmU2VmsdHRo79NniIM85yWJrS-A_GIhku_0yXZOg8Fk/edit?usp=sharing" href="https://docs.google.com/document/d/1tmU2VmsdHRo79NniIM85yWJrS-A_GIhku_0yXZOg8Fk/edit?usp=sharing" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     IYP 分类知识列表2
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    中看到 Hacking Team 的更多详细资料
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这个 Neutrino 为执法和金融机构开发技术，专门调查和跟踪区块链上的交易。Coinbase 是世界上买卖加密货币的最大平台之一，因此在这里能追踪到大多数交易。对区块链的监视近年来是间谍机构的重点目标，在上述列表中找到
   <a class="markup--anchor markup--p-anchor" data-href="https://docs.google.com/document/d/1tmU2VmsdHRo79NniIM85yWJrS-A_GIhku_0yXZOg8Fk/edit?usp=sharing" href="https://docs.google.com/document/d/1tmU2VmsdHRo79NniIM85yWJrS-A_GIhku_0yXZOg8Fk/edit?usp=sharing" rel="noopener noreferrer" target="_blank">
    “监视区块链和反现金之战”
   </a>
   栏目。
  </p>
  <p class="graf graf--p">
   该公司声称能够监控和跟踪比特币 — 比特币是相对容易追踪的 — 但该公司也可以追踪隐私导向的币种如 Monero。2017年，该公司得出结论，破坏性勒索软件 WannaCry 背后的朝鲜黑客兑现了他们的比特币，并将
   <a class="markup--anchor markup--p-anchor" data-href="https://motherboard.vice.com/en_us/article/qvky75/hackers-behind-wannacry-cashed-out-bitcoin-while-no-one-was-watching" href="https://motherboard.vice.com/en_us/article/qvky75/hackers-behind-wannacry-cashed-out-bitcoin-while-no-one-was-watching" rel="noopener noreferrer" target="_blank">
    其转变成了 Monero
   </a>
   。
  </p>
  <p class="graf graf--p">
   Marco Valleri，黑客社区中被称为“NaGa”，以及被称为“ALoR”的 Alberto Ornaghi，分别是Neutrino 的首席研究官和首席技术官。在21世纪初，Valleri 和 Ornaghi 开发了 Ettercap，这是作为 Hacking Team
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theverge.com/2013/9/13/4723610/meet-hacking-team-the-company-that-helps-police-hack-into-computers" href="https://www.theverge.com/2013/9/13/4723610/meet-hacking-team-the-company-that-helps-police-hack-into-computers" rel="noopener noreferrer" target="_blank">
    技术基础
   </a>
   的软件。 Neutrino 首席执行官 Giancarlo Russo 曾经是 Hacking Team 的首席运营官。
  </p>
  <p class="graf graf--p">
   Coinbase 决定与曾经为政府开发间谍软件的人合作，某些区块链行业参与者非常不爽。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “当我说让更多的信息安全人员参与’加密’空间会很棒时，我并不是说美国最大的交易所应该收购一个由前 Hacking Team 成员运营的间谍工具公司，但结果却是这样，”区块链创业公司 Clovyr 的首席执行官 Amber Baldet
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/AmberBaldet/status/1099716851574235138" href="https://twitter.com/AmberBaldet/status/1099716851574235138" rel="noopener noreferrer" target="_blank">
    在推特上说
   </a>
   。
  </p>
  <p class="graf graf--p">
   Baldet 在网上聊天中告诉 Motherboard，“考虑到 Coinbase 开设的账户数量，他们如何选择实施合规工具、他们与执法部门的合作关系，将对很多人产生影响。”
  </p>
  <p class="graf graf--p">
   加拿大公司 Bull Bitcoin 的联合创始人弗朗西斯·普利奥特（Francis Pouliot）更加直言不讳，他在一条推文中称，
   <strong class="markup--strong markup--p-strong">
    Hacking Team 是一个“亲政府的骇客雇佣军公司，为沙特阿拉伯提供强大的监视工具/咨询服务以追踪/消除持不同政见者”…… “快跑！
   </strong>
   ”
  </p>
  <p class="graf graf--p">
   Coinbase 发言人在给主板的一份声明中表示，该公司“不会宽恕，也不会捍卫 Hacking Team 的行为”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们知道 Neutrino 的联合创始人之前曾在 Hacking Team 工作，我们在安全、技术和招聘方面进行了审核，”发言人说。
  </p>
  <p class="graf graf--p">
   但发言人解释说，Neutrino 的技术对 Coinbase 来说“太重要了”。区块链取证分析（就是监视）是一个新兴的行业，Coinbase 决定在内部实现该功能。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “越来越多的第三方区块链分析公司正在向他们所服务的加密货币公司请求客户数据。对于 Coinbase 来说，重要的是将内部功能用于完全控制和保护客户的数据，而 Neutrino 的技术是我们在实现这一目标的空间中遇到的最好的技术，“发言人通过电子邮件说。
  </p>
  <p class="graf graf--p">
   Neutrino 的联合创始人没有立即回应评论请求。
  </p>
  <p class="graf graf--p">
   开发和销售像 Hacking Team 这样的间谍软件本身并不违法，尽管它可能非常危险。
   <strong class="markup--strong markup--p-strong">
    摩洛哥政府一直在使用 Hacking Team 软件
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://slate.com/technology/2012/08/moroccan-website-mamfakinch-targeted-by-government-grade-spyware-from-hacking-team.html" href="https://slate.com/technology/2012/08/moroccan-website-mamfakinch-targeted-by-government-grade-spyware-from-hacking-team.html" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     监视一个名为 Mamfakinch 的公民记者网站
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。该网站的一位创始人表示，独立新闻网站
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amnesty.org/en/latest/research/2016/12/how-a-hacking-campaign-helped-shut-down-an-award-winning-news-site/" href="https://www.amnesty.org/en/latest/research/2016/12/how-a-hacking-campaign-helped-shut-down-an-award-winning-news-site/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     随后被关闭
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，因为贡献者害怕他们成为政府监控的目标
   </strong>
   。
  </p>
  <p class="graf graf--p">
   有一个蓬勃发展的行业致力于为政府机构提供工具来攻击和监视计算机和手机。目前，这个行业几乎不受监管，这为全球政府针对人权活动家和持不同政见者的监视软件的扩散敞开了大门。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    F
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://motherboard.vice.com/en_us/article/a3bn5e/coinbase-neutrino-former-hacking-team-members" href="https://motherboard.vice.com/en_us/article/a3bn5e/coinbase-neutrino-former-hacking-team-members" rel="noopener noreferrer" target="_blank">
    ormer Hacking Team Members Are Now Spying on the Blockchain for Coinbase
   </a>
   . Coinbase said it’s aware of the Neutrino founders’ controversial past, but their technology is the best the major exchange has encountered.
  </p>
