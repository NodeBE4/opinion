  <p class="graf graf--p graf-after--figure" id="b51a">
   在 Facebook 中的任何人有意愿、甚至可以说是有能力控制 Facbook 吗？这是上周纽约时报在对社交媒体巨头的调查中
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2018/11/14/technology/facebook-data-russia-election-racism.html" href="https://www.nytimes.com/2018/11/14/technology/facebook-data-russia-election-racism.html" rel="noopener nofollow noreferrer" target="_blank">
    涉及的基本问题
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="df5a">
   越来越明显的迹象表明，这些巨头公司生存和增长的前提不仅仅是依赖于类似剑桥分析公司丑闻那种侵犯性数据交易的共谋罪行，还依赖于对其平台上那些传播暴力行为用户的低效的打地鼠式处理方式 — — 正是这种行为助长了缅甸政府军对罗兴亚人口的大屠杀。这个问题貌似在公司的改革尝试中并没有得到解决，而该公司领导层却把眼光放到了一种进攻性防御的方式上，他们一边游说法律制定者，一边向公众辩解来逃避和模糊他们自己的责任。
  </p>
  <p class="graf graf--p graf-after--p" id="fbf8">
   Facebook 所体现的问题究竟是因为互联网乌托邦的诺言陷入了歧途？还是是因为互联网及产品本身的结构所导致的必然结果？
  </p>
  <p class="graf graf--p graf-after--p" id="ba71">
   艺术家兼技术人员 James Bridle 的一本新书认为：后者才是原因。
  </p>
  <p class="graf graf--p graf-after--p" id="57d1">
   《新黑暗时代：技术与未来的终结（New Dark Age: Technology and the End of the Future）》这本书中认为，
   <strong class="markup--strong markup--p-strong">
    正是互联网的核心
   </strong>
   — — 高速计算和全球化网络的结合 — — 对日益增长的侵入性数据挖掘、‘假新闻’和大规模监控行为，
   <strong class="markup--strong markup--p-strong">
    负有很大责任
   </strong>
   。
   <span style="color: #ff6600;">
    <strong>
     您可以在这里下载这本书
    </strong>
   </span>
   ：https://t.me/iyouport/6696
  </p>
  <p class="graf graf--p graf-after--p" id="0e99">
   <strong class="markup--strong markup--p-strong">
    换句话说，互联网并非是被邪恶污染了，正相反，它正在以最初设计时的原则和目的正常运行
   </strong>
   。
   <strong class="markup--strong markup--p-strong">
    而这种’正常运行’不仅在模糊化我们对世界的认知，而且还使世界越发充满危险
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li graf-after--p" id="2567">
    这不是第一次揭示同样的被隐瞒的背景真相，另见：《
    <a class="markup--anchor markup--li-anchor" data-href="https://medium.com/@iyouport/%E4%BA%92%E8%81%94%E7%BD%91%E7%A9%B6%E7%AB%9F%E6%98%AF%E4%BB%80%E4%B9%88-%E6%9D%A5%E8%87%AA-%E7%9B%91%E8%A7%86%E8%B0%B7-%E6%83%8A%E4%BA%BA%E7%9A%84%E5%90%AF%E7%A4%BA-cdd6b5b6bd9d" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e7%a9%b6%e7%ab%9f%e6%98%af%e4%bb%80%e4%b9%88%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     互联网究竟是什么？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p graf-after--li" id="2d59">
   一个日常的小例子一定程度上可以证实 Bridle 提到的人们的一种最常见观点：“感谢互联网，我在任何时间和地点都能收到天气预报。虽然结果可能有误，然后因为这些错误预报我可能已经变成了落汤鸡。但是我还是对在外出前能得到力所能及的‘最佳信息’而心怀感激。最坏的情况不过就是淋成了落汤鸡而已“。
  </p>
  <p class="graf graf--p graf-after--p" id="8e47">
   Bridle 的新书指出，技术不仅在模糊化我们对世界的认知，而且还使世界越发充满危险。
  </p>
  <p class="graf graf--p graf-after--p" id="4cc3">
   Bridle 的新书以深远的方式展示了上文中对天气预报抱有的观点是多么的错误。天气预报，正如所有的预测型计算一样，本质上
   <strong class="markup--strong markup--p-strong">
    假设
   </strong>
   未来和过去是相似的，它整合过去时间点上的数据来对未来进行预演。但是由于全球气候变化，这种天气预报的计算方式在长远来看变得越来越不准确，未来和过去越来越不同。
  </p>
  <p class="graf graf--p graf-after--p" id="d5ce">
   不仅如此，对天气预报的计算自身 — — 如今，大规模天气预测计算是一个高耗能的事业 — — 它所需要的碳排放量，正在削弱其自身的准确性。正如 Bridle 所写到的，“计算即是因气候变化的受害者，也是对气候变化的加害者“。每当我刷新天气预报时，我的行为都对气候变化产生了影响，从而使预报更不准确。而气候变化已经使天气越来越难以预测，而且正对世界造成各种破坏。
  </p>
  <p class="graf graf--p graf-after--p" id="e1ab">
   短期来说，我可能避免了成为落汤鸡，但长远来看，我所住的城市以后大部分可能会被水淹没。如今，
   <strong class="markup--strong markup--p-strong">
    我们所知的互联网就是以这种方式把我们带向了 Bridle 所称的 “新黑暗时代”：它用内化的方式假设世界是什么、和应该是怎样的，但是掩盖了互联网本身正在构建一个更多变的未来的事实。因为未来的变化与过去的模式之间差异越来越大，我们会发现，我们对于世界的了解不是越来越多，而恰恰相反，是越来越少
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="3d6f">
   为什么会这样？Bridle 提到，原因之一就是由于构成互联网的结构本身一开始就不是为了产生无私的知识的。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4559 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-163.jpeg?resize=1024%2C1024&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-163.jpeg?w=1024&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-163.jpeg?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-163.jpeg?resize=300%2C300&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-163.jpeg?resize=768%2C768&amp;ssl=1 768w" data-recalc-dims="1" height="1024" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-163.jpeg?resize=1024%2C1024&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4559" data-recalc-dims="1" height="1024" sizes="(max-width: 1024px) 100vw, 1024px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-163.jpeg?resize=1024%2C1024&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-163.jpeg?w=1024&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-163.jpeg?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-163.jpeg?resize=300%2C300&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-163.jpeg?resize=768%2C768&amp;ssl=1 768w" width="1024"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="9d11">
   例如，气象学计算本身最初是美英军事联盟发展的事业，目的专门是为了在冷战中西方可以获得战略性的核优势和空中优势的霸权。 这项事业的中心构架后来就成为了组成国防高等计划研究署（DARPA ）的核心，DARPA 这个美国政府机构正是我们现在所知的互联网的先驱 — — ARPANET — — 的缔造者。
  </p>
  <p class="graf graf--p graf-after--p" id="bdeb">
   寄希望于这些技术（比如 ARPANET ）的延伸（Internet）可以和
   <strong class="markup--strong markup--p-strong">
    最初的设计目的（为了资本的霸权和公司利润最大化）
   </strong>
   分离转而为公共事业服务，这个想法十分天真。现实是，我们使用的这些技术让世界和我们所希望的（技术为公共事业服务）的情况越发遥远。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="628f">
   “New Dark Age” 的每个章节都从不同视野揭示了这些最初的设计目的已经如此深层次地充满了我们的社交关系，以及它们让真正的知识变得多么的模糊。
  </p>
  <p class="graf graf--p graf-after--p" id="d23d">
   在‘气候（Climate）’ 章节里讨论了我们联网的高科技社会是怎样加速了气候变化。且在这个过程中，实际上还破坏了真实知识的类比来源（比如种子库和冻土层中的考古学遗骸）。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="1507">
   “复杂性 (Complexity)” 章节认为计算技术在高级金融领域的应用歪曲了经济信息并且加剧了不公。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="cb6f">
   “阴谋 (Conspiracy)” 章节显示了互联网的算法排序如何使人们的无力感和对边缘群体的偏见常态化。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="01cc">
   “并发（Concurrency）”章节关注了 YouTube 对最大化产生吸睛效益的追求 (Google 的基本要求)， 教唆了算法生成的对儿童身心健康不利而且是自动播放的视频的流传。 (Bridle 去年在 Medium 上写了个案例，里面提到如果父母给小孩子找了一个封面看似无害的 “Peppa Pig” 视频，结果那个视频可能显示的是“Peppa Pig”遭到各种酷刑和摧残的故事)。
  </p>
  <p class="graf graf--p graf-after--p" id="cf8d">
   这些问题的核心 Bridle 称之为计算型思维 (computational thinking)。
   <strong class="markup--strong markup--p-strong">
    计算型思维认为所有关于过去的信息都可以而且也应该被全部收集，然后在整合分析以形成对未来的决策。在 “共谋(Complicity)这一章节中,” Bridle 认为正是这种思维方式同样造就了政府部门的大规模监控行为：政府情报部门以*收集一切*而闻名于世，他们的大规模监控行为从没有让公共领域变得更安全，也没有对减少暴力做出任何帮助
   </strong>
   。(由于这个事实，2013 年总统委员会宣布大规模监控对于预防袭击来讲并不是必需的)。
  </p>
  <p class="graf graf--p graf-after--p" id="ae44">
   问题是， Bridle 认为，这种监控行为方式本质上即是溯及既往的又是充满报复性，它假定仅仅只要曝光就够了，只要把问题放置到光线之下，问题就会被揭发甚至自己就解决了。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="22f8">
   <strong class="markup--strong markup--p-strong">
    “我们变得相信于把一个事物公之于众和对它进行思考是同一件事。同样，对待事件只要存在有关部门去管就够了”，Bridle 警告说。不，事实上完全相反
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="6902">
   这种自欺欺人不仅局限于不透明的政府，记者同样有罪。Bridle 举的主要例子是 2013 年 Edward Snowden 揭露 NSA 的事件 (正是这个事件启发了 Intercept 的成立)。
  </p>
  <p class="graf graf--p graf-after--p" id="a20b">
   虽然最初对美国和其盟友的大规模监视行为的精准披露造成了很大反响，但真正的改革需求却被最大程度的无视，然后公众的视野很快转向了其他地方。
  </p>
  <p class="graf graf--p graf-after--p" id="d5d3">
   <strong class="markup--strong markup--p-strong">
    这里的问题是，Bridle 讲到，人们期盼只要单纯的曝光就会导向理解和行动。与监控和计算行为一样，记者也是深陷于这种妄想型的偏执逻辑：因为这种逻辑沉迷于过去的完美镜像，而放弃了对现在和未来的真正理解
   </strong>
   。
  </p>
  <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote graf-after--p" id="5f7e">
   <p>
    <em class="markup--em markup--blockquote-em">
     “We have become convinced that throwing light upon the subject is the same thing as thinking it, and thus having agency over it,” Bridle warns.
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--blockquote" id="0ca2">
   “新黑暗时代”一书中提出的对这种问题的应对方式，更多是一种新的思考方式而不是如何去行动， “我们需要的不是新的技术”，Bridle 在第一页就写到， “而是新的隐喻”，和错误的认为可以看到和理解世界上每一个事实的计算型思维相反。
  </p>
  <p class="graf graf--p graf-after--p" id="bd87">
   Bridle 提出“云雾型思维 (cloudy thinking)”，一种接受不可预知的事物、并从另一种角度看待问题的实践。Bridle 实际上希望“新黑暗时代 (New Dark Age) ” 这个标题被理解为一个机遇，而非挽歌。
  </p>
  <p class="graf graf--p graf-after--p" id="6d9b">
   要拥抱这个机会，他论述到，我们要放弃对完美知识的自负追求，而应该对我们赖以生活和工作的互联网正在主动地模糊化我们对真实世界的认知和加剧不公正这件事上，保持一种负责任的怀疑主义态度。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="8b5e">
   “我并不是在反对技术”，Bridle 坚持说， “这么做是在反对我们自己”。技术即是重塑世界的工具，也是我们去了解世界的一种隐喻。
  </p>
  <p class="graf graf--p graf-after--p" id="3ea7">
   好吧：在需要重新认识我们自身和技术的关系这点上 Bridle 是对的。我们不能期盼技术会带来完美的信息和社会共识。
   <strong class="markup--strong markup--p-strong">
    我们要对来自可信来源的时效性信息采取行动，而不是等着技术去整合分析和验证每个事实
   </strong>
   。
   <strong class="markup--strong markup--p-strong">
    如果只是等待，一切都会变的太晚
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="3adc">
   但是如果不去面对互联网如今已经变成了什么，而是仅仅接受 Bridle 的建议看起来是不够的。虽然在我们的生活如今和互联网密不可分这点上 Bridle 是对的，但他好像屈服在一种宿命论的而且天真的乐观主义里了。在宿命上他认为我们只能适应如今这种互联网，乐观的认为只要换个角度去看待这种互联网，我们就能避免它会带来的毁灭性社会灾难。
  </p>
  <blockquote class="graf graf--blockquote graf-after--p" id="ff38">
   <p>
    <em class="markup--em markup--blockquote-em">
     互联网的基础是资本，这些资本来自于政府和大公司的投资，他们当然会期盼投资带来回报。
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p graf-after--blockquote" id="168a">
   问题是我们所知的互联网实际上、甚至本质上并不是一个虚拟的东西：它的实现是由充满世界的数据线海洋、数字集权的摩天大楼、以及相当于一个城市的超大规模数据中心构成的。它的基础是资本，这些资本来自于政府和公司的投资，他们当然会期盼投资带来回报。
  </p>
  <p class="graf graf--p graf-after--p" id="ecb1">
   Bridle 和作家 Ingrid Burrington 对构成互联网基础设施的本质的揭露对我们是有很大价值的。“新黑暗时代”中同样指出了巨大的物质利润和回报驱动着这些资本投入。
   <strong class="markup--strong markup--p-strong">
    “帝国几乎放弃了对实际领土的诉求，但他们把目光转向了基础设施，以互联网的形式维持他们权力”，
   </strong>
   Bridle 写到。
  </p>
  <p class="graf graf--p graf-after--p" id="4496">
   不幸的是，Bridle 并没有从这个洞察结论继续对它的逻辑性结果进行追问。权力不是靠思想就可以获得的 — — 它只能从行动中获得。如果互联网的基础设施是由那以剥削大众为目标的帝国控制的，我们还怎么能对这种互联网可以促进人类发展和改善人权抱有幻想呢？
  </p>
  <p class="graf graf--p graf-after--p" id="d15b">
   呼吁互联网社会化和公益化，要求公众控制数据，这些看起来也许不现实，但至少是在直面这个问题。令人失望的是，这样一本对人类文明所面对的灾难性现状认识清晰的书却没有更深入的提及解决方案。
  </p>
  <p class="graf graf--p graf-after--p" id="5c9c">
   也许比起说互联网的末日，说成世界末日更容易想像些吧。但 Bridle 以后的作品不得不面对这个他现在拒绝置评的问题：要拯救互联网和这个星球，是不是人们要从当权者手中夺回那些基础设施，即公众实际控制每一个数据中心甚至每一块电路板，这样互联网才终于可以服务于大众和社会事业？然而，假如做不到这样，是不是完全摧毁这个互联网才是符合人类利益的？⚪️
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="2db7">
   “We have become convinced that throwing light upon the subject is the same thing as thinking it, and thus having agency over it,”
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2018/11/24/james-bridle-new-dark-age-review/" href="https://theintercept.com/2018/11/24/james-bridle-new-dark-age-review/" rel="noopener nofollow noreferrer" target="_blank">
    IS IT EASIER TO IMAGINE THE END OF THE WORLD THAN THE END OF THE INTERNET?
   </a>
  </p>
