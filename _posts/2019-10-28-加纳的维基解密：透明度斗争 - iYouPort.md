  <p class="graf graf--p">
   六月份听到过一个有趣的事。G20前夕，中国国社组织“揭露美国弊端”的系列话题，其中一个话题是关于信息自由法案的。
  </p>
  <p class="graf graf--p">
   可以说这次打得有点准。
   <strong class="markup--strong markup--p-strong">
    信息自由法案基本是个摆设，不仅在美国，在英国和其他很多国家都是如此 —— 不仅信息请求严重耗时，并且，最终被揭露的信息很有可能是被编辑过的，而不是真相
   </strong>
   。
  </p>
  <p class="graf graf--p">
   我们介绍过反抗者如何利用技术能力抵制这种恶意掩盖真相的行为，详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%ab%98%e6%98%8e%e7%9a%84%e5%8f%8d%e6%8a%97/" href="https://www.iyouport.org/%e9%ab%98%e6%98%8e%e7%9a%84%e5%8f%8d%e6%8a%97/" rel="noopener noreferrer" target="_blank">
    高明的反抗
   </a>
   》。
  </p>
  <p class="graf graf--p">
   回过头来看，正因为信息自由法案基本是个摆设，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/?s=%E9%80%8F%E6%98%8E%E5%BA%A6%E9%9D%A9%E5%91%BD" href="https://www.iyouport.org/?s=%E9%80%8F%E6%98%8E%E5%BA%A6%E9%9D%A9%E5%91%BD" rel="noopener noreferrer" target="_blank">
    透明度革命
   </a>
   才应该具有强大且持久的生命力。就如耕耘。
  </p>
  <p class="graf graf--p">
   任何政权都不喜欢真相。因为权力建立在谎言之上。
  </p>
  <p class="graf graf--p">
   当权者会想尽各种办法掩人耳目，阻止公民知情。不论是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/facebook-%e6%ad%a3%e5%9c%a8%e4%bb%a4%e7%9c%9f%e7%9b%b8%e6%8c%96%e6%8e%98%e4%ba%8b%e4%b8%9a%e9%99%b7%e5%85%a5%e5%9b%b0%e5%a2%83/" href="https://www.iyouport.org/facebook-%e6%ad%a3%e5%9c%a8%e4%bb%a4%e7%9c%9f%e7%9b%b8%e6%8c%96%e6%8e%98%e4%ba%8b%e4%b8%9a%e9%99%b7%e5%85%a5%e5%9b%b0%e5%a2%83/" rel="noopener noreferrer" target="_blank">
    今年徒增的
   </a>
   科技寡头封杀OSINT技术工具、还是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E8%B0%B7%E6%AD%8C%E5%9C%B0%E5%9B%BE%E6%98%AF%E6%AF%94%E8%A9%B9%E5%A7%86%E6%96%AF%E9%82%A6%E5%BE%B7%E6%9B%B4%E6%A3%92%E7%9A%84%E9%97%B4%E8%B0%8D/" href="https://www.iyouport.org/%E8%B0%B7%E6%AD%8C%E5%9C%B0%E5%9B%BE%E6%98%AF%E6%AF%94%E8%A9%B9%E5%A7%86%E6%96%AF%E9%82%A6%E5%BE%B7%E6%9B%B4%E6%A3%92%E7%9A%84%E9%97%B4%E8%B0%8D/" rel="noopener noreferrer" target="_blank">
    去年开始的
   </a>
   对公民掌握真相事业的歪曲和打压。
  </p>
  <p class="graf graf--p">
   但是中国国社的稿件在最终编辑的过程中中被删除了一段话。
  </p>
  <p class="graf graf--p">
   被删除的这段话的大概意思是：“中国应该效仿信息自由法案的实施，相信中国能比美国做得更好……”。您肯定已经猜到了，因为，
   <a href="https://twitter.com/iyouport_news/status/1177108199788298240%EF%BC%89" rel="noopener noreferrer" target="_blank">
    9月份已经出现了举世震惊的暴力污染
   </a>
   （在这个链接里看到 thread）。
  </p>
  <p class="graf graf--p">
   显然，那句话不符合该议题的整体目标 ——
   <strong>
    目标仅仅是抨击美国的弊端，并没有对信息自由的支持
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是以美国为首的国家正在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%8e%e4%ba%ba%e6%a0%bc%e6%9a%97%e6%9d%80%e5%88%b0%e5%af%b9%e7%9c%9f%e7%9b%b8%e7%9a%84%e8%b0%8b%e6%9d%80-%e2%8f%b3/" href="https://www.iyouport.org/%e4%bb%8e%e4%ba%ba%e6%a0%bc%e6%9a%97%e6%9d%80%e5%88%b0%e5%af%b9%e7%9c%9f%e7%9b%b8%e7%9a%84%e8%b0%8b%e6%9d%80-%e2%8f%b3/" rel="noopener noreferrer" target="_blank">
    对透明度革命发起猛烈攻击
   </a>
   ，五眼国家澳大利亚甚至翻出了多年前的旧报道，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%9a%8f%e7%9d%80%e5%af%b9%e6%96%b0%e9%97%bb%e4%b8%9a%e7%9a%84%e6%88%98%e4%ba%89%e5%9c%a8%e5%85%a8%e7%90%83%e5%8d%87%e7%ba%a7%ef%bc%8c%e8%ad%a6%e5%af%9f%e5%b1%95%e5%bc%80%e7%aa%81%e7%84%b6%e8%a2%ad/" href="https://www.iyouport.org/%e9%9a%8f%e7%9d%80%e5%af%b9%e6%96%b0%e9%97%bb%e4%b8%9a%e7%9a%84%e6%88%98%e4%ba%89%e5%9c%a8%e5%85%a8%e7%90%83%e5%8d%87%e7%ba%a7%ef%bc%8c%e8%ad%a6%e5%af%9f%e5%b1%95%e5%bc%80%e7%aa%81%e7%84%b6%e8%a2%ad/" rel="noopener noreferrer" target="_blank">
    对记者抄家
   </a>
   。
  </p>
  <p class="graf graf--p">
   在这种情况下，信息自由法案“有”比没有更好。法律存在，就可以不断通过抗议修补其漏洞，作为 planB 是有必要的。
  </p>
  <p class="graf graf--p">
   但信息自由法案不能替代透明度革命。因为当权者掌握对法律的控制权。
  </p>
  <p class="graf graf--p">
   加纳是一个很好的案例。这里的透明度革命抗争从来没有停止（虽然很不专业，安全性非常低）。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    透明度革命的生命力不仅需要民主热情和勇气，还需要高明的技术能力
   </strong>
   —— 因为，如果说真话的人不断被捕，只会迅速挫败革命士气，捍卫真相的工作将难以延续。
  </p>

  <p class="graf graf--p">
   经过二十年的拖延，加纳新信息权法案于四月获得通过。理论上它应该为国家机构带来透明度，授权调查记者了解真相；但是，它可能会产生相反的效果。
  </p>
  <p class="graf graf--p">
   在政府拖延的二十年中，记者和民间社会团体无数次示威，现在加纳终于通过了期待已久的信息权法案。这部法律本应旨在确保反腐败斗争的透明度。
  </p>
  <p class="graf graf--p">
   新的立法由 Nana Akufo-Addo 总统签署成为法律。
  </p>
  <p class="graf graf--p">
   但是，记者和活动人士担心，
   <strong class="markup--strong markup--p-strong">
    该法律可用作政府限制记者在 Facebook 和 Twitter 等流行在线平台上分享曝光信息的工具
   </strong>
   。
  </p>
  <p class="graf graf--p">
   社交媒体在允许记者和公民（通常是匿名的）发布泄露的政府文件方面发挥了重要的作用，使调查新闻事业可以蓬勃发展。
  </p>
  <p class="graf graf--p">
   加纳是非洲排名第七位的腐败程度较低的国家，但是在打击制度化腐败方面仍有很长的路要走。
  </p>
  <p class="graf graf--p">
   在加纳大学最近进行的一项调查中，88％的受访者表示他们坚信该国政府部门存在腐败现象。
  </p>
  <p class="graf graf--p">
   如上所述，
   <strong class="markup--strong markup--p-strong">
    该法律被拖延的这二十年恰恰是加纳对新闻记者的攻击数量快速增多的二十年。
   </strong>
  </p>
  <p class="graf graf--p">
   从2005年到2014年，
   <a class="markup--anchor markup--p-anchor" data-href="http://accra.sites.unicnetwork.org/2014/11/05/ghana-records-138-incidents-of-violation-against-journalists-from-2005-till-date/" href="http://accra.sites.unicnetwork.org/2014/11/05/ghana-records-138-incidents-of-violation-against-journalists-from-2005-till-date/" rel="noopener noreferrer" target="_blank">
    共发生了138起针对记者的暴力事件
   </a>
   ，其中大多数攻击者是警察、军队和政党分支机构的成员。
  </p>
  <p class="graf graf--p">
   今年1月，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2019/jan/17/ghanian-undercover-journalist-shot-dead-in-accra" href="https://www.theguardian.com/world/2019/jan/17/ghanian-undercover-journalist-shot-dead-in-accra" rel="noopener noreferrer" target="_blank">
    一名揭露加纳足球协会（GFA）腐败内幕的记者
   </a>
   在一次驾车射击中被杀死。
  </p>
  <p class="graf graf--p">
   受害者 Ahmed Hussein-Suale 是调查记者团队的成员，正是他们拍摄到了前 GFA 老板 Kwesi Nyantakyi 接受贿赂的内幕。
  </p>
  <p class="graf graf--p">
   Nyantakyi 被国际足联撤职并被罚款50万美元。
  </p>

  <p class="graf graf--p">
   活动人士和记者曾希望信息权法案可以帮助揭露更多的政治腐败。
  </p>
  <p class="graf graf--p">
   2018年，加纳选举委员会主席因涉嫌为新办公大楼花费了约 795,000 美元而
   <a class="markup--anchor markup--p-anchor" data-href="https://punchng.com/ghana-gets-new-electoral-commission-chair/" href="https://punchng.com/ghana-gets-new-electoral-commission-chair/" rel="noopener noreferrer" target="_blank">
    被下台
   </a>
   。理论上新法律应该能让更多腐败官员面临后果，或至少是接受公众监督。
  </p>
  <p class="graf graf--p">
   预计在未来五年内实施的 RTI 法律
   <a class="markup--anchor markup--p-anchor" data-href="https://www.pulse.com.gh/bi/finance/here-is-how-much-it-will-cost-to-implement-the-right-to-information-in-ghana/l2wjbgx" href="https://www.pulse.com.gh/bi/finance/here-is-how-much-it-will-cost-to-implement-the-right-to-information-in-ghana/l2wjbgx" rel="noopener noreferrer" target="_blank">
    将耗资1.45亿美元
   </a>
   ，将要求各部委等公共机构在12个月内公布其活动详情，并任命信息RTI官员。
  </p>
  <p class="graf graf--p">
   地区办事处和总部的工作人员将处理记者和公民的请求。公共机构被要求提供的信息本身是免费的，但加纳人必须自己支付复印费用或用以转储的闪存驱动器或光盘的费用。
  </p>
  <p class="graf graf--p">
   该法案
   <strong class="markup--strong markup--p-strong">
    不允许访问
   </strong>
   有关预算、合同关系、财务信息和计划受益人的信息。
  </p>
  <p class="graf graf--p">
   要获取信息，记者必须发送手写的访问信息请求，然后等待在14天内收到回复。（看起来比美国和英国更快。后者
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%a7%a3%e5%af%86%e6%96%87%e4%bb%b6%ef%bc%9a%e7%a7%91%e6%8a%80%e4%ba%a7%e4%b8%9a%e5%b7%a8%e5%a4%b4%e4%b8%ba%e4%bb%80%e4%b9%88%e8%83%bd%e6%88%90%e4%b8%ba%e5%b7%a8%e5%a4%b4%ef%bc%9f%e7%a7%81%e6%9c%89/" href="https://www.iyouport.org/%e8%a7%a3%e5%af%86%e6%96%87%e4%bb%b6%ef%bc%9a%e7%a7%91%e6%8a%80%e4%ba%a7%e4%b8%9a%e5%b7%a8%e5%a4%b4%e4%b8%ba%e4%bb%80%e4%b9%88%e8%83%bd%e6%88%90%e4%b8%ba%e5%b7%a8%e5%a4%b4%ef%bc%9f%e7%a7%81%e6%9c%89/" rel="noopener noreferrer" target="_blank">
    经常需要等到数月甚至数年
   </a>
   ）
  </p>
  <p class="graf graf--p">
   一旦获得批准，记者将能够以书面或电子形式接收信息。
  </p>
  <p class="graf graf--p">
   但是目前，记者在从政府获取信息方面面临许多障碍。随着RTI法案，记者希望法律能帮助他们战胜闭口不言的官员。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “作为一名记者，议会通过 RTI 法案对我来说是很好的第一步，但是，
   <strong class="markup--strong markup--p-strong">
    需要制定适当的措施和政策才能有效实施该法律
   </strong>
   ，”加纳商业新闻负责人 Emmanuel Dogbevi 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “将信息权法的执行情况从司法部转移到新闻部并不是一个好的决定，因为该部是政府的喉舌。几乎无法信任与政府有关的机构能提供真实的信息”。
   </strong>
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    二十年的延误
   </strong>
  </p>
  <p class="graf graf--p">
   RTI 法案于1999年由加纳经济事务研究所首次起草，并经过三次选举都没有能通过。
  </p>
  <p class="graf graf--p">
   议会在2003年、2005年和2007年再次审查了这项法案，但是，据活动人士称，由于政客们担心受到媒体的严格监督，该法律从未经过投票。
  </p>
  <p class="graf graf--p">
   现任政府未能履行其在2017年1月就职100天内通过该法案的
   <a class="markup--anchor markup--p-anchor" data-href="http://citifmonline.com/2017/02/02/well-pass-rti-bill-this-year-bawumia/" href="http://citifmonline.com/2017/02/02/well-pass-rti-bill-this-year-bawumia/" rel="noopener noreferrer" target="_blank">
    初步承诺
   </a>
   ，此后才通过了RTI法案。
  </p>
  <p class="graf graf--p">
   去年11月，沮丧的记者、律师和其他利益相关者穿着红色衣服在阿克拉街头开启了游行示威；抗议活动在 Twitter 上以标签 #RTIRedFriday 呈现。
  </p>
  <p class="graf graf--p">
   大约30名示威者举着标语牌，上面写着“现在就通过 RTI 法案！”、“真的什么都没有隐藏吗？ #PassTheRTIBillNow”。
  </p>
  <p class="graf graf--p">
   抗议活动由三个组织领导：信息权媒体联盟、加纳信息权联盟、和占领加纳。
  </p>
  <p class="graf graf--p">
   RTI 法案的通过也恰逢许多加纳人要求提高其国家与美国在军事方面之关系的透明度。
  </p>
  <p class="graf graf--p">
   2018年3月，
   <strong class="markup--strong markup--p-strong">
    成千上万的加纳人
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2018/03/28/world/africa/ghana-us-military-deal-protests.html" href="https://www.nytimes.com/2018/03/28/world/africa/ghana-us-military-deal-protests.html" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     反对2000亿美元的交易
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，这笔交易将为加纳军队提供设备和培训，并为美国和加纳的联合军演铺平道路
   </strong>
   。
  </p>
  <p class="graf graf--p">
   加纳政府表示，该协议是美国与西非国家之间现有的双边和多边军事关系的延伸。
  </p>
  <p class="graf graf--p">
   拖延 RTI 法案通过的一个重要原因可以归结为对草案中一系列漏洞的
   <a class="markup--anchor markup--p-anchor" data-href="https://allafrica.com/stories/201808060350.html" href="https://allafrica.com/stories/201808060350.html" rel="noopener noreferrer" target="_blank">
    分歧
   </a>
   。
  </p>
  <p class="graf graf--p">
   第一版 RTI 法案阻止记者获取总统办公室、副总统和内阁所掌握的信息。
  </p>
  <p class="graf graf--p">
   仅总统办公室就有200多名员工。
  </p>
  <p class="graf graf--p">
   媒体活动家、民间社会团体、和智囊团都在使议会中的 RTI 委员会放松规则，这些规则将使获取信息变得更加困难。
  </p>
  <p class="graf graf--p">
   他们还推动 RTI 委员会同意取消了获取信息支付费用的要求。
  </p>
  <p class="graf graf--p">
   但是，漏洞仍然存在。
  </p>
  <p class="graf graf--p">
   通过的法案表明，如果信息对个人、团体或金融机构或任何其他机构造成“不应有的损失”，则不能披露信息。极为模糊的措辞。
  </p>
  <p class="graf graf--p">
   该法律还可以防止披露与政治家纳税申报相关的信息，并且，如果被认为“损害所提及的任何其他个人的声誉”，则可以免除披露个人相关的信息。
  </p>
  <p class="graf graf--p">
   很明显，该法律非常无用。
  </p>
  <p class="graf graf--p">
   更令人担忧的是，
   <strong class="markup--strong markup--p-strong">
    该法案还将总统和副总统作为“私人”保护起来了
   </strong>
   ，只允许政府部门内的其他办公室接受披露请求。
  </p>
  <p class="graf graf--p">
   媒体组织指出，这些限制削弱了该法案为众所周知存在的秘密政府交易带来透明度的能力。
  </p>
  <p class="graf graf--p">
   例如，前政府拒绝披露一项有争议的商业合资企业的细节，金额约为732,000美元，其中一家公交品牌营销公司被指控对纳税人资金管理不善。
  </p>
  <p class="graf graf--p">
   政府最初拒绝公布有关该交易的信息，以避免暴露公职人员的秘密。加纳高等法院采取干预措施，命令政府披露其信息。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    加纳的维基解密
   </strong>
  </p>
  <p class="graf graf--p">
   在没有 RTI 法案的情况下，社交媒体在帮助公民责令官员承担责任方面发挥了重要作用。
  </p>
  <p class="graf graf--p">
   Facebook 和 Twitter 已成为一种“加纳的维基解密”，让记者和公民分发和分析被泄露的政府机密文件。虽然，这一渠道非常不安全。
  </p>
  <p class="graf graf--p">
   现在，记者们担心政府可以使用 RTI 法案来打击任何未经政府许可共享机密文件的人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该法律规定，通过包括社交媒体在内的任何平台披露此类信息的个人将受到制裁。处罚可包括超过40,000美元的罚款，1至3年的监禁，或两者兼而有之。
   </strong>
  </p>
  <p class="graf graf--p">
   加纳新闻部长最近暗示，政府关注的只是新闻机构和记者如何使用舆论和社交媒体。
  </p>
  <p class="graf graf--p">
   这位部长指出：“在一个虚假新闻兴盛的时代，社交媒体不应该成为对 RTI 法律的威胁。相反，它应该被用来纠正信息”。
  </p>
  <p class="graf graf--p">
   Dogbevi 说：“我在社交媒体上因为想要让政治家负起责任而受到攻击。”
  </p>
  <p class="graf graf--p">
   政府还认为，任何有关国家安全的信息，如国家武装部队的准备情况、或涉嫌恐怖袭击的个人信息……等等，都可能危及加纳的“稳定” —— 以维稳大局为由抵制透明度革命。事实上是为稳固执政党的权力。
  </p>
  <p class="graf graf--p">
   2016年4月，社交媒体上泄露的安全备忘录显示政府原本知道
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bbc.co.uk/news/world-africa-36052843" href="https://www.bbc.co.uk/news/world-africa-36052843" rel="noopener noreferrer" target="_blank">
    恐怖袭击即将发生
   </a>
   却未采取措施。
  </p>
  <p class="graf graf--p">
   尽管存在一堆漏洞，记者和活动家仍然希望该法律能够迅速实施。
  </p>
  <p class="graf graf--p">
   该法案第13条规定，当信息请求有可能破坏政府机构工作的情况下，信息免于被披露。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “对该条款的修订仍然不足以解决其负面影响”，Finder Newspaper 的编辑 Elvis Darko 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “目前的第13条草案会严重（如果不是完全的话）削弱信息权利。我们提出了弥补缺陷的建议：议会提出第13（1）（c）条明确规定在审议程序结束后此类信息不再豁免。如果没有这一明确规定，第13条可以很容易地被解释为使审议过程持续进行。”
  </p>
  <p class="graf graf--p">
   非洲采用信息权立法的国家数量正稳步增加。
  </p>
  <p class="graf graf--p">
   2011年，非洲大陆
   <a class="markup--anchor markup--p-anchor" data-href="https://africacheck.org/factsheets/factsheet-freedom-of-information-laws-on-the-african-continent/" href="https://africacheck.org/factsheets/factsheet-freedom-of-information-laws-on-the-african-continent/" rel="noopener noreferrer" target="_blank">
    只有五个
   </a>
   国家通过了 RTI 法案，但截至2017年，
   <a class="markup--anchor markup--p-anchor" data-href="http://www.africafex.org/access-to-information/22-african-countries-that-have-passed-access-to-information-laws" href="http://www.africafex.org/access-to-information/22-african-countries-that-have-passed-access-to-information-laws" rel="noopener noreferrer" target="_blank">
    约有22个非洲国家
   </a>
   采用了类似的法律，马拉维是最新通过该法律的国家。
  </p>
  <p class="graf graf--p">
   加纳每年因腐败
   <a class="markup--anchor markup--p-anchor" data-href="https://www.myjoyonline.com/politics/2018/October-3rd/ghana-losing-gh135bn-to-corruption-every-year-chraj.php" href="https://www.myjoyonline.com/politics/2018/October-3rd/ghana-losing-gh135bn-to-corruption-every-year-chraj.php" rel="noopener noreferrer" target="_blank">
    损失约279.3亿美元
   </a>
   。许多损失可以追溯到欺诈性的政府采购流程和盗窃行为。
  </p>
  <p class="graf graf--p">
   2018年，加纳遭遇了一系列腐败丑闻，其中包括1.79亿美元的通信部交易，引发了争议。
  </p>
  <p class="graf graf--p">
   现有的 RTI 法案将强制要求合同的透明度，并证明招标公司的专业知识。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “总统现在需要同意RTI法律并签署实施，”Darko 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们要求草案中包括建立执行该法律的时间表，但是议会拒绝了该意见书。如果有明确的时间表，将可以加速在2020年1月之前实施。”⚪️
