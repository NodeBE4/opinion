  <p class="graf graf--p">
   很多社运组织和公民社会团体使用 Facebook Messenger 做社区联络，也包括一些独立媒体、还有各种营销商，结合主页的维护以促进凝聚力并拉拢支持者。如果您的人手较少，无法实时亲自维护怎么办？
  </p>
  <p class="graf graf--p">
   本文帮您创建一个机器人，您可以根据自己的需求进行设置，以便满足您的推广目的。
  </p>
  <p class="graf graf--p">
   请注意您可以创建匿名账户哦，请在这里回顾基本方法《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%bb%ba%e7%ab%8b%e4%b8%80%e4%b8%aa%e5%8c%bf%e5%90%8d-facebook-%e8%b4%a6%e6%88%b7%e4%bb%a5%e4%bf%9d%e6%8a%a4%e4%b8%aa%e4%ba%ba%e6%95%b0%e6%8d%ae%e7%9a%84%e7%a7%81%e5%af%86%e6%80%a7/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%bb%ba%e7%ab%8b%e4%b8%80%e4%b8%aa%e5%8c%bf%e5%90%8d-facebook-%e8%b4%a6%e6%88%b7%e4%bb%a5%e4%bf%9d%e6%8a%a4%e4%b8%aa%e4%ba%ba%e6%95%b0%e6%8d%ae%e7%9a%84%e7%a7%81%e5%af%86%e6%80%a7/" rel="noopener noreferrer" target="_blank">
    如何建立一个匿名社交媒体账户以保护个人数据的私密性？
   </a>
   》。
  </p>
  <p class="graf graf--p">
   Facebook 聊天机器人具有很多优点：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    全天候不休息
   </li>
   <li class="graf graf--li">
    有问必答
   </li>
   <li class="graf graf--li">
    即时回答不拖延
   </li>
   <li class="graf graf--li">
    将繁琐的任务自动化，为您减压
   </li>
  </ul>
  <p class="graf graf--p">
   已经在 SAP Conversational AI 平台上为用户创建了逐步的集成过程，因此您仅需几分钟即可向 Facebook 粉丝显示您的聊天机器人。现在开始吧。
  </p>
  <p class="graf graf--p">
   <strong>
    第1步：准备好聊天机器人
   </strong>
  </p>
  <p class="graf graf--p">
   首先，您需要一个聊天机器人（似乎合法，对吧）。请注意，您的聊天机器人在 Facebook 上在线后，便可以对其进行修改，并且您对其所做的任何更改都会显示在 Messenger 聊天中。
  </p>
  <p class="graf graf--p">
   在这里创建您的帐户（完全免费）：
   <a class="markup--anchor markup--p-anchor" data-href="https://cai.tools.sap/signup" href="https://cai.tools.sap/signup" rel="noopener noreferrer" target="_blank">
    https://cai.tools.sap/signup
   </a>
   还可以在这里阅读创建机器人的教程：
   <a class="markup--anchor markup--p-anchor" data-href="https://www.freecodecamp.org/news/how-to-build-your-first-chatbot-with-the-sap-conversational-ai-9a1a2bd44e3c/" href="https://www.freecodecamp.org/news/how-to-build-your-first-chatbot-with-the-sap-conversational-ai-9a1a2bd44e3c/" rel="nofollow noopener noreferrer" target="_blank">
    https://www.freecodecamp.org/news/how-to-build-your-first-chatbot-with-the-sap-conversational-ai-9a1a2bd44e3c/
   </a>
  </p>
  <p class="graf graf--p">
   注册后选择一种机器人，比如，“讲笑话的聊天机器人”（或者您想构建的任何东西），之后再次回到这里。
  </p>
  <p class="graf graf--p">
   <strong>
    第2步：准备好您的 Facebook 页面
   </strong>
  </p>
  <p class="graf graf--p">
   您的聊天机器人仅可用于在 Facebook 页面上集成（不适用于您的个人账户）。这意味着您必须创建一个主页，或者存在一个要使用的页面。
  </p>
  <p class="graf graf--p">
   在 Facebook 主页上建立聊天机器人后，私人信息就会自动连接到您的页面。因此，如果您决定删除聊天机器人，则将立即恢复为传统的人与人对话（这意味着用户输入消息之前，除非您手动回答，否则什么也不会发生）。
  </p>
  <p class="graf graf--p">
   <strong>
    第3步：建立 Messenger Facebook app
   </strong>
  </p>
  <p class="graf graf--p">
   创建应用程序将有助于使 SAP Conversational AI 与您的 Facebook 页面之间建立连接。没有该应用程序您将无法在 Facebook 主页上发布聊天机器人。
  </p>
  <p class="graf graf--p">
   在顶部菜单中选择 “My Apps”，然后单击 “Add New App”。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10149 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-27.png?resize=800%2C363&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-27.png?w=800&amp;ssl=1 800w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-27.png?resize=300%2C136&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-27.png?resize=768%2C348&amp;ssl=1 768w" data-recalc-dims="1" height="363" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-27.png?resize=800%2C363&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10149" data-recalc-dims="1" height="363" sizes="(max-width: 800px) 100vw, 800px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-27.png?resize=800%2C363&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-27.png?w=800&amp;ssl=1 800w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-27.png?resize=300%2C136&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-27.png?resize=768%2C348&amp;ssl=1 768w" width="800"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   创建后，您必须添加一个 Messenger “产品”。
  </p>
  <p class="graf graf--p">
   转到App的仪表板，然后在 Messenger 框中单击“设置”。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10150 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-25.png?resize=800%2C498&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-25.png?w=800&amp;ssl=1 800w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-25.png?resize=300%2C187&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-25.png?resize=768%2C478&amp;ssl=1 768w" data-recalc-dims="1" height="498" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-25.png?resize=800%2C498&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10150" data-recalc-dims="1" height="498" sizes="(max-width: 800px) 100vw, 800px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-25.png?resize=800%2C498&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-25.png?w=800&amp;ssl=1 800w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-25.png?resize=300%2C187&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-25.png?resize=768%2C478&amp;ssl=1 768w" width="800"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在左侧菜单中，您将在 [PRODUCTS] 下看到 Messenger。
  </p>
  <p class="graf graf--p">
   <strong>
    第4步：获取页面令牌和应用程序密码
   </strong>
  </p>
  <p class="graf graf--p">
   现在已经创建了 Messenger app，需要将其链接到您的 Facebook 页面（默认情况下，Facebook app 是一个独立的实体）。
  </p>
  <p class="graf graf--p">
   有了这种连接，您将获得一个令牌，该令牌基本上是一个唯一的代码，上面写着 “OK, this is the code of the Messenger app of the page X”。
  </p>
  <p class="graf graf--p">
   在左侧菜单中，单击产品 Messenger 下方的设置。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10151 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-21.png?resize=800%2C221&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-21.png?w=800&amp;ssl=1 800w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-21.png?resize=300%2C83&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-21.png?resize=768%2C212&amp;ssl=1 768w" data-recalc-dims="1" height="221" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-21.png?resize=800%2C221&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10151" data-recalc-dims="1" height="221" sizes="(max-width: 800px) 100vw, 800px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-21.png?resize=800%2C221&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-21.png?w=800&amp;ssl=1 800w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-21.png?resize=300%2C83&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-21.png?resize=768%2C212&amp;ssl=1 768w" width="800"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   选择您希望聊天机器人出现的页面。
  </p>
  <p class="graf graf--p">
   您可能需要允许该应用与您的 Facebook 页面进行交互。单击蓝色的“编辑权限”按钮，选择页面，然后选中不同的框。
  </p>
  <p class="graf graf--p">
   授予权限后，将生成令牌。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10152 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-19.png?resize=800%2C240&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-19.png?w=800&amp;ssl=1 800w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-19.png?resize=300%2C90&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-19.png?resize=768%2C230&amp;ssl=1 768w" data-recalc-dims="1" height="240" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-19.png?resize=800%2C240&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10152" data-recalc-dims="1" height="240" sizes="(max-width: 800px) 100vw, 800px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-19.png?resize=800%2C240&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-19.png?w=800&amp;ssl=1 800w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-19.png?resize=300%2C90&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-19.png?resize=768%2C230&amp;ssl=1 768w" width="800"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   返回到 SAP Conversational AI 聊天机器人中的 “Connect” 选项，选择 Messenger，然后将在上面步骤中得到的令牌粘贴到 “
   <em class="markup--em markup--p-em">
    Page token
   </em>
   ” 字段中。
  </p>
  <p class="graf graf--p">
   很棒！现在已经完成超过一半了。现在，获取 “app secret”，就像您的其他应用程序的密码一样。
  </p>
  <p class="graf graf--p">
   在左侧菜单中，转到“设置”&gt;“基本”。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10153 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-18.png?resize=800%2C136&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-18.png?w=800&amp;ssl=1 800w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-18.png?resize=300%2C51&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-18.png?resize=768%2C131&amp;ssl=1 768w" data-recalc-dims="1" height="136" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-18.png?resize=800%2C136&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10153" data-recalc-dims="1" height="136" sizes="(max-width: 800px) 100vw, 800px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-18.png?resize=800%2C136&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-18.png?w=800&amp;ssl=1 800w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-18.png?resize=300%2C51&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-18.png?resize=768%2C131&amp;ssl=1 768w" width="800"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   为了保护隐私，应用程序密码是隐藏的。点击 show，然后将其复制并粘贴到聊天机器人的 “Connect” 选项上的 “
   <em class="markup--em markup--p-em">
    App secret
   </em>
   ” 字段（类似于您对页面令牌所做的操作）。
  </p>
  <p class="graf graf--p">
   单击 SAP Conversational AI 下的 Update channel。
  </p>
  <p class="graf graf--p">
   <strong>
    第5步：将 SAP Conversational AI 连接到您的app
   </strong>
  </p>
  <p class="graf graf--p">
   在 Products &gt; Messenger&gt;设置页面上，转到 Webhooks 部分，然后单击 Subscribe To Events。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10154 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-19.png?resize=800%2C180&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-19.png?w=800&amp;ssl=1 800w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-19.png?resize=300%2C68&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-19.png?resize=768%2C173&amp;ssl=1 768w" data-recalc-dims="1" height="180" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-19.png?resize=800%2C180&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10154" data-recalc-dims="1" height="180" sizes="(max-width: 800px) 100vw, 800px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-19.png?resize=800%2C180&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-19.png?w=800&amp;ssl=1 800w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-19.png?resize=300%2C68&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-19.png?resize=768%2C173&amp;ssl=1 768w" width="800"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在弹出窗口中，输入在上述第4步中将找到的 “Callback URL” 和“Verify token”的值。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10155 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-17.png?resize=800%2C136&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-17.png?w=800&amp;ssl=1 800w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-17.png?resize=300%2C51&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-17.png?resize=768%2C131&amp;ssl=1 768w" data-recalc-dims="1" height="136" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-17.png?resize=800%2C136&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10155" data-recalc-dims="1" height="136" sizes="(max-width: 800px) 100vw, 800px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-17.png?resize=800%2C136&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-17.png?w=800&amp;ssl=1 800w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-17.png?resize=300%2C51&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-17.png?resize=768%2C131&amp;ssl=1 768w" width="800"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   还要选择下面显示的这些复选框：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10156 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/8-14.png?resize=800%2C241&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/8-14.png?w=800&amp;ssl=1 800w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/8-14.png?resize=300%2C90&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/8-14.png?resize=768%2C231&amp;ssl=1 768w" data-recalc-dims="1" height="241" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/8-14.png?resize=800%2C241&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10156" data-recalc-dims="1" height="241" sizes="(max-width: 800px) 100vw, 800px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/8-14.png?resize=800%2C241&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/8-14.png?w=800&amp;ssl=1 800w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/8-14.png?resize=300%2C90&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/8-14.png?resize=768%2C231&amp;ssl=1 768w" width="800"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   页面重新加载后，在列表中选择您的主页，以便它可以访问您的 Webhook。
  </p>
  <p class="graf graf--p">
   <strong>
    第6步：测试并发布聊天机器人
   </strong>
  </p>
  <p class="graf graf--p">
   现在，您可以以管理员身份测试您的机器人了（也可以使用左侧菜单中的“角色”&gt;“测试用户”来授予某些测试角色）。
  </p>
  <p class="graf graf--p">
   您必须更改状态，才能公开访问您的机器人，因此请花一些时间进行测试，并确保一切正常，然后再将其发布给公众。
  </p>
  <p class="graf graf--p">
   如果将切换开关设置为ON（在右上角），则会重定向到设置并提示您提供一些其他信息，然后再发布您的机器人。直到您对机器人满意为止。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10157 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-15.png?resize=800%2C161&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-15.png?w=800&amp;ssl=1 800w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-15.png?resize=300%2C60&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-15.png?resize=768%2C155&amp;ssl=1 768w" data-recalc-dims="1" height="161" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-15.png?resize=800%2C161&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10157" data-recalc-dims="1" height="161" sizes="(max-width: 800px) 100vw, 800px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-15.png?resize=800%2C161&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-15.png?w=800&amp;ssl=1 800w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-15.png?resize=300%2C60&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/9-15.png?resize=768%2C155&amp;ssl=1 768w" width="800"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   最后一步：Facebook 将要审查和测试您的 Messenger 聊天机器人。他们在文档中对这一步的说明如下：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em>
    “When you are ready to release your bot to the public, you must submit it to our team for review and approval. This review process allows us to ensure your Messenger bot abides by our policies and functions as expected before it is made available to everyone on Messenger.” — Facebook Documentation
   </em>
  </p>
  <p class="graf graf--p">
   在左侧菜单中，转到产品&gt; Messenger&gt;设置，然后单击 pages_messaging 中的“添加到提交”。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10158 jetpack-lazy-image" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/10-9.png?resize=300%2C68&amp;is-pending-load=1#038;ssl=1" data-recalc-dims="1" height="68" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/10-9.png?resize=300%2C68&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="300"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10158" data-recalc-dims="1" height="68" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/10-9.png?resize=300%2C68&amp;ssl=1" width="300"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   审核一般情况下会很快。
  </p>
  <p class="graf graf--p">
   就是这样，完成了！希望您的机器人是社交高手。⚪️
  </p>
