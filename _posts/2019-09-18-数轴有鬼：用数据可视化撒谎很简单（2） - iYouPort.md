  <p class="graf graf--p">
   继续上周的话题，如果您还没有读过（1）
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ae%a9%e6%95%b0%e6%8d%ae%e8%af%b4%e8%af%9d%e5%8f%aa%e6%98%af%e4%b8%aa%e5%94%ac%e4%ba%ba%e7%9a%84%e5%b9%8c%e5%ad%90%ef%bc%9a%e8%af%86%e7%a0%b4%e8%b0%8e%e8%a8%80%ef%bc%881%ef%bc%89/" href="https://www.iyouport.org/%e8%ae%a9%e6%95%b0%e6%8d%ae%e8%af%b4%e8%af%9d%e5%8f%aa%e6%98%af%e4%b8%aa%e5%94%ac%e4%ba%ba%e7%9a%84%e5%b9%8c%e5%ad%90%ef%bc%9a%e8%af%86%e7%a0%b4%e8%b0%8e%e8%a8%80%ef%bc%881%ef%bc%89/" rel="noopener noreferrer" target="_blank">
    在这里看到
   </a>
   。
  </p>
  <p class="graf graf--p">
   虽然人类的大脑进化至今仍然不能处理数量巨大的、以数字表示的数据文本；但是
   <strong class="markup--strong markup--p-strong">
    人的眼睛却可以很容易捕捉形象化数据所含有的模式。这就是可视化新闻为什么火了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   通过制作图表来展示每个单一的数值是一种呈现数据的常用方式。
  </p>
  <p class="graf graf--p">
   虽然这种方式呈现出来的数据最准确，但是，
   <strong class="markup--strong markup--p-strong">
    其可解释性却最小
   </strong>
   。
  </p>
  <p class="graf graf--p">
   不过，对于大多数人来说，在这样的数据中识别模式和趋势依旧是比较困难的，或者说几无可能。于是，⚠️
   <strong class="markup--strong markup--p-strong">
    在此做诈者会很容易得手
   </strong>
   。
  </p>
  <p class="graf graf--p">
   要想解决这个问题，或许需要依靠一些辅助。
  </p>
  <p class="graf graf--p">
   图表分为两大类：一种是使用形象化的方式来代表每一个数据点（如散点图）；另一种是采用数据缩减的方式，借以把数据汇总考察，比如只分析数据的中位数。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    使用图表操纵、扭曲、和错误地呈现数据的方法有很多种。你很难免被骗
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是依旧有几个迹象能提醒你：小心欺诈。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    I 数轴未做标记
   </strong>
  </p>
  <p class="graf graf--p">
   用统计图表说谎的最基本方法就是在数轴上不做任何标记。
  </p>
  <p class="graf graf--p">
   如果不标记数轴，那么你就可以随意绘制图表了！
  </p>
  <p class="graf graf--p">
   来看一个例子。一个学术大会上，一名学生向听众展示了下面这张图（中文内容原图中全没有）：
  </p>

   这是什么意思？
  </p>
  <p class="graf graf--p">
   可以在报告的文本中了解到，这个大会的主题是关于精神分裂症患者的大脑激活。
  </p>
  <p class="graf graf--p">
   那 HCs 又是什么？
  </p>
  <p class="graf graf--p">
   不得不再次看报告，其中说它代表的是在与精神分裂症对比。或许是“健康管理”？
  </p>
  <p class="graf graf--p">
   现在，HCs 和 SZs 两者看起来确实有不同之处，但是……图中 y 轴标有数字，但是却没有任何单位。
  </p>
  <p class="graf graf--p">
   那么这个图代表什么呢？考试分数？大脑活动水平？激活脑区的数量？是吃掉的包子数量、还是最近六周内看电影的数量？……

  <p class="graf graf--p">
   这张图的状况和前面的例子一样，只不过这次问题是处在X轴上。
  </p>
  <p class="graf graf--p">
   虽然X轴标记了数字，但是没有说明数字代表什么。
  </p>
  <p class="graf graf--p">
   你可能会猜到是自然年度或财务年度，那些线的起伏说明是按月份来取的数据 —— 请注意这是瞎猜。
  </p>
  <p class="graf graf--p">
   图中也没有Y轴，所以你不知道它衡量的东西是什么，究竟是销售单位还是货币单位；也不知道每条水平直线的意义是什么。
  </p>
  <p class="graf graf--p">
   它可以描述销售额从5分钱/年增长到5块钱/年，也可以描述5000万增长到5亿。好吧，这张图还至少附带了一句话：“这又是一个收获很大的年份”。只能认同后者这种猜测了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    II 截短纵轴
   </strong>
  </p>
  <p class="graf graf--p">
   一张绘制精良的图能够让你清晰地看出一个连续体相关的终止点。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果你记录的是数量的实际或预计变化情况，这一点是尤其重要的。因为你的目的是让读者从图中得出正确的结论
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果你的图表示的内容是犯罪率、死亡率、出生率、收入或其他任何可以取值为0的数量，那么0就是图中的最低点。
  </p>
  <p class="graf graf--p">
   ⚠️但如果你的目的是制造一种使人感到恐慌和愤慨的效果，那么就要在接近图最低点的地方开始画Y轴 —— 这样做能够使你想要凸显的差异得到强化，
   <strong class="markup--strong markup--p-strong">
    因为人的眼睛很容易捕捉差异大小，同时差异的实际大小却被弱化了。
   </strong>
  </p>
  <p class="graf graf--p">

   没人希望多缴5倍的税吧？数字恐惧症的人或者看得太仓促的人，也许没有时间分析下它的问题。也就无法发现，实际上是税率从35%上涨到39.6%。

    <img alt="" class="aligncenter size-full wp-image-4451" data-recalc-dims="1" height="582" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/17.jpeg?resize=1100%2C582&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/17.jpeg?w=1194&amp;ssl=1 1194w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/17.jpeg?resize=300%2C159&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/17.jpeg?resize=768%2C407&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/17.jpeg?resize=1024%2C542&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/17.jpeg?resize=1100%2C582&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   你要看到，堕胎服务略有增长，而癌症筛查服务数量呈现下降趋势。
  </p>
  <p class="graf graf--p">
   对于那个凶险图来说，其实第一眼就应该怀疑。因为图中的线条特别平滑，在数据图中很罕见这种状况。
  </p>
  <p class="graf graf--p">
   绘图者很有可能选取了两个特别的年份，并在比较之后两点一线连起来了。或许他是为了有意强调差异；也或许是中间年份数据波动比较大。不管怎么样，一条完美直线的情况都是极不可能的。
  </p>
  <p class="graf graf--p">
   类似这样的图并非总能达成人们期望的那种意义。
  </p>
  <p class="graf graf--p">
   计划生育协会的任务是尽可能多提供一些堕胎服务？除了这样的解释之外难道就没有对上述数据的其他解释了？
  </p>
  <p class="graf graf--p">
   看第一张图。2006年提供了 2007371次癌症筛查服务，同时还进行了 289750 次的堕胎服务；提供的癌症筛查服务的数量几乎是堕胎服务的的 7倍。到 2013年两种服务数量的差距缩小了，但癌症筛查服务的数量仍然几乎是堕胎服务数量的 3倍。
  </p>
  <p class="graf graf--p">
   计划生育协会主席 Cecile Richards 曾经对数量差距缩小的情况做了解释。因为一些抗癌服务项目在就医指南方面出现了变化，比如巴氏图片检验，就减少了需要做癌症筛查的人数。其他方面诸如社会对堕胎的态度、社会人口的年龄、卫生保健替代品的增加，都会影响这两项服务的数量。
  </p>
  <p class="graf graf--p">
   因此图中的数据
   <strong class="markup--strong markup--p-strong">
    不能证明
   </strong>
   计划生育协会支持堕胎。即使它支持，这些数据也无法作为证据。
  </p>
  <p class="graf graf--p">
   就是这么简单。💡防止被操纵式宣传蒙骗的办法就是，想一分钟。
  </p>
  <p class="graf graf--p">
   尤其是在互联网上，能够做到想一分钟的人极为稀罕。想想看，加载速度超过3秒你就会血压上升，对吧？所以用这种动过手脚的图示来进行宣传操纵的伎俩，在互联网上经常能得手……
  </p>
