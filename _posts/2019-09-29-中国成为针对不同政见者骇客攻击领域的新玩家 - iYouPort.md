   <img alt="" class="aligncenter size-full wp-image-9452 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/骇客中国.png?resize=1100%2C620&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/骇客中国.png?w=1200&amp;ssl=1 1200w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/骇客中国.png?resize=300%2C169&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/骇客中国.png?resize=768%2C433&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/骇客中国.png?resize=1024%2C577&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/骇客中国.png?resize=1100%2C620&amp;ssl=1 1100w" data-recalc-dims="1" height="620" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/骇客中国.png?resize=1100%2C620&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   根据一份新的报告，骇客试图闯入多名藏人的 iPhone 和 Android 设备，其中一些受害者在达赖喇嘛办公室、西藏中央政府和藏族议会工作。
  </p>
  <p class="graf graf--p">
   骇客使用为 iOS 和 Android 操作系统开发的漏洞利用程序和间谍软件，并试图通过 WhatsApp 发送精心制作的网络钓鱼信息以传播这些间谍软件，
   <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.ca/2019/09/poison-carp-tibetan-groups-targeted-with-1-click-mobile-exploits" href="https://citizenlab.ca/2019/09/poison-carp-tibetan-groups-targeted-with-1-click-mobile-exploits" rel="noopener noreferrer" target="_blank">
    根据报告
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    这些恶意行为者假装成记者、非政府组织工作人员、以及藏族人权组织的志愿者
   </strong>
   。
  </p>
  <p class="graf graf--p">
   该报告由数字权利组织公民实验室（Citizen Lab）发布。
  </p>
  <p class="graf graf--p">
   研究人员得出的结论是，此次攻击活动是由针对居住在中国的维吾尔族人口的同一个骇客组织发起的。
  </p>
  <p class="graf graf--p">
   8月底，谷歌详细描述了
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/DonEvansWm/status/1170658847528083458" href="https://twitter.com/DonEvansWm/status/1170658847528083458" rel="noopener noreferrer" target="_blank">
    一场针对穆斯林少数族裔的令人震惊的攻击运动
   </a>
   ，攻击者利用的是当时的0day。
  </p>
  <figure aria-describedby="caption-attachment-9450" class="wp-caption aligncenter" id="attachment_9450" style="width: 1190px">
   <img alt="" class="size-full wp-image-9450 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-15.png?resize=1100%2C603&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-15.png?w=1190&amp;ssl=1 1190w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-15.png?resize=300%2C164&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-15.png?resize=768%2C421&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-15.png?resize=1024%2C561&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-15.png?resize=1100%2C603&amp;ssl=1 1100w" data-recalc-dims="1" height="603" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-15.png?resize=1100%2C603&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="size-full wp-image-9450" data-recalc-dims="1" height="603" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-15.png?resize=1100%2C603&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-15.png?w=1190&amp;ssl=1 1190w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-15.png?resize=300%2C164&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-15.png?resize=768%2C421&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-15.png?resize=1024%2C561&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-15.png?resize=1100%2C603&amp;ssl=1 1100w" width="1100"/>

  <p class="graf graf--p graf--startsWithDoubleQuote">
   “一般而言，这种对公民社会的针对性攻击的一个问题是：谁的技术正在帮助恶意行为者闯入持不同政见者的手机？
   <strong class="markup--strong markup--p-strong">
    长期以来，NSO 集团和他们臭名昭著的 Pegasus 似乎是该领域唯一的玩家，但是，现在我们看到了一个新出现的、与中国有联系的玩家，正在崭露头角
   </strong>
   ”。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e4%bb%a5%e8%89%b2%e5%88%97%e5%92%8cnso/" href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e4%bb%a5%e8%89%b2%e5%88%97%e5%92%8cnso/" rel="noopener noreferrer" target="_blank">
     关于以色列和NSO
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/report%ef%bc%9a%e4%bb%a5%e8%89%b2%e5%88%97-nso-%e7%94%9f%e4%ba%a7%e7%9a%84%e9%97%b4%e8%b0%8d%e8%bd%af%e4%bb%b6-pegasus-%e5%9c%a845%e4%b8%aa%e5%9b%bd%e5%ae%b6%e8%a2%ab%e7%94%a8%e4%ba%8e%e7%9b%91/" href="https://www.iyouport.org/report%ef%bc%9a%e4%bb%a5%e8%89%b2%e5%88%97-nso-%e7%94%9f%e4%ba%a7%e7%9a%84%e9%97%b4%e8%b0%8d%e8%bd%af%e4%bb%b6-pegasus-%e5%9c%a845%e4%b8%aa%e5%9b%bd%e5%ae%b6%e8%a2%ab%e7%94%a8%e4%ba%8e%e7%9b%91/" rel="noopener noreferrer" target="_blank">
     以色列 NSO 生产的间谍软件 Pegasus 在45个国家被用于监视和攻击异议人士及活动家 — ⚠️ 亚洲有香港
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   调查这次攻击事件的首席研究员 Bill Marczak 在一次采访中说，“他们[中国玩家]显然有点毛糙：他们
   <strong class="markup--strong markup--p-strong">
    使用稍微过时的漏洞利用来捕获那些更新不够及时的人，而且他们不一定会追踪电话上的“一切”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   公民实验室看到了15次攻击尝试；尽管有些受害者点击了恶意链接，但没有人真正被入侵，因为他们的手机更新都比较及时。
  </p>
  <p class="graf graf--p">
   Citizen Lab 追踪的攻击活动似乎与
   <a class="markup--anchor markup--p-anchor" data-href="https://www.vice.com/en_us/article/59nyqb/how-google-changed-the-secretive-market-for-the-most-dangerous-hacks-in-the-world" href="https://www.vice.com/en_us/article/59nyqb/how-google-changed-the-secretive-market-for-the-most-dangerous-hacks-in-the-world" rel="noopener noreferrer" target="_blank">
    Google Project Zero
   </a>
   所揭示的攻击活动重叠，后者揭示了多个iOS零日的使用；以及
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cyberscoop.com/china-uyghur-hacking-china-android-iphone/" href="https://www.cyberscoop.com/china-uyghur-hacking-china-android-iphone/" rel="noopener noreferrer" target="_blank">
    安全公司 Volexity 的详细描述
   </a>
   ，确定了多个用于传播 Android 恶意软件的网站。
  </p>
  <p class="graf graf--p">
   公民实验室说，“我们确定这三个攻击活动可能是由同一操作者或具有共享资源的紧密合作的团队进行的”；
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这些事件表明，对移动设备的利用需求不断增长。从对手的角度来看，推动这一需求的因素是显而易见的。移动设备是大多数人的在线生活，民间社会组织利用它来组织动员，智能手机内部的数据展示了所有这一切的全景图”。
  </p>
  <p class="graf graf--p">
   Google 最初拒绝评论，然后发送了以下声明：“我们与 Citizen Lab 合作进行了这项研究，并赞赏他们为提高所有平台的安全性所做的努力。如报告中所述，这些问题已得到修补，不再对使用最新软件的用户构成风险。”
  </p>
  <p class="graf graf--p">
   苹果公司发言人发表了以下声明：“客户的数据安全性是苹果公司的重中之重，我们非常重视与 Citizen Lab 等安全研究人员的合作。该报告中详细介绍的iOS问题已由Apple的安全团队发现并修复。我们始终鼓励客户下载最新版本的iOS，以获取最佳和最新的安全性增强。”
  </p>
  <p class="graf graf--p">
   这不是结束，大概只是一个开始。
  </p>
  <p class="graf graf--p">
   ⚠️中国作为零日攻击领域的新玩家，不一定会在一夜之间赶超以色列的 NSO，但是无疑会不断进化；与此同时请记得，NSO 的间谍软件销售并不会顾及地缘政治关系 ——
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/report%ef%bc%9a%e4%bb%a5%e8%89%b2%e5%88%97-nso-%e7%94%9f%e4%ba%a7%e7%9a%84%e9%97%b4%e8%b0%8d%e8%bd%af%e4%bb%b6-pegasus-%e5%9c%a845%e4%b8%aa%e5%9b%bd%e5%ae%b6%e8%a2%ab%e7%94%a8%e4%ba%8e%e7%9b%91/" href="https://www.iyouport.org/report%ef%bc%9a%e4%bb%a5%e8%89%b2%e5%88%97-nso-%e7%94%9f%e4%ba%a7%e7%9a%84%e9%97%b4%e8%b0%8d%e8%bd%af%e4%bb%b6-pegasus-%e5%9c%a845%e4%b8%aa%e5%9b%bd%e5%ae%b6%e8%a2%ab%e7%94%a8%e4%ba%8e%e7%9b%91/" rel="noopener noreferrer" target="_blank">
    他们一直在把产品卖给出得起钱的任何人
   </a>
   。
  </p>
  <p class="graf graf--p">
   限于本案例（或者说限于当前短时间内），中国敏感人士应该注意至少以下三点：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    在互联网上联系你的陌生人，如果对方声称自己是NGO、记者或权利组织志愿者，在确认对方的真实身份之前不要相信不要点击他们传递的任何信息 —— 加密通信群组同样鱼龙混杂，切勿忽视作为人性骇客的
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
     社交工程攻击
    </a>
    ；
   </li>
   <li class="graf graf--li">
    及时更新你的系统和软件，不要嫌麻烦；
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     让操作安全进入习惯，保持警惕，至少做到这篇文章中提醒的注意事项
    </strong>
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%a7%bb%e9%99%a4%e4%bd%a0%e7%9a%84%e7%bb%b5%e7%be%8a%e5%a2%99%ef%bc%9a%e7%bb%99%e6%99%ae%e9%80%9a%e4%ba%ba%e7%9a%845%e4%b8%aa%e9%98%b2%e5%be%a1%e8%af%be%e7%a8%8b/" href="https://www.iyouport.org/%e7%a7%bb%e9%99%a4%e4%bd%a0%e7%9a%84%e7%bb%b5%e7%be%8a%e5%a2%99%ef%bc%9a%e7%bb%99%e6%99%ae%e9%80%9a%e4%ba%ba%e7%9a%845%e4%b8%aa%e9%98%b2%e5%be%a1%e8%af%be%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
     移除你的绵羊墙：给普通人的5个防御课程
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   关于操作安全，IYP 此后还将介绍更多详细知识。⚠️只要你还在使用数字设备，就必须小心；并无所谓你的民族身份。⚪️
  </p>
