  <span style="font-size: 10pt;">
   台湾统派人士在台北101大楼前展示五星旗
  </span>
 </p>
 <p>
 </p>
 <p>
  <span style="font-size: 12pt;">
   台湾独派人士提出禁止公开展示中国五星旗的公投提案，此举是否违反宪法保障的言论自由？台湾学者提出了不同看法。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   台湾中央选举委员会星期四(5号)举行一场有关立法禁止公开展示中国五星旗公投的听证会，并邀请专家学者发表看法。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   心生恐惧
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   提出这项公投案的独派人士刘曜华表示，五星旗在台湾任意飘扬，让人感觉被冒犯，甚至心生恐惧，影响了正常生活。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   他说：“让人不自主的会产生中共政权可以公开在台湾土地上冒犯台湾人的生存权，所以我认为这种视觉污染，如果我们不正视，不加以对抗，而且持续在过往民主前辈牺牲性命争取而得的言论自由框架下，给与过度的保护，我个人是很担心。”
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   李曜华还说，尊重相关团体的思想和言论自由，但是言论自由不能被放任，需要法律的规范，尤其五星旗是敌对国的代表。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   台湾统派人士近年来在一些热闹景点，比如台北101大楼前或是西门町等地挥舞或悬挂五星旗，甚至也在一些抗议场合当中高举五星旗。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   台湾民众今年初在国家发展委员会公共政策网络参与平台上提案，建议增设刑法条文禁止中国五星旗在台湾公开悬挂、展示、陈列，并获得5千人附议成案。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   不过，权责机关台湾法务部则回应，如果悬挂五星旗就施以刑罚，不符合宪法保障人民言论自由意旨，提案不予采纳。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   言论自由
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   出席听证会的中央研究院法律学研究所副研究员苏彦图表示，这项公投提案确实抵触了目前宪法保障的言论自由。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   他说：“所谓五星旗代表的是特定政治立场和意见的象征，所以它的公开展示和悬挂，基本上我们也会当做是一种政治言论。”
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   苏彦图还指出，台湾民主化之后，基本的宪法原则是，除非叛乱的言论已经涉及强暴胁迫，并对于国家安全及民主宪政秩序构成明显而立即的危害，否则政府不得予以限制。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   应有界限
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   中正大学法律系教授李仁淼则认为，言论自由仍然要有一定的界限，五星旗代表了某种政治意图，2005年中国通过的反分裂国家法，就是要消灭中华民国的存在，这样的“言论自由”需要被保护吗？
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   成功大学法律学系副教授陈怡凯表示，宪法是以国际法为基础，然后发展出各式各样的国民基本权利，依据国际法，国民对国家有忠诚及认同的义务，如果把认同五星旗看作是言论自由，就代表台湾人民可以认同中华人民共和国是他的国家。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   陈怡凯教授还说，台湾的统派团体，不断的吸收成员，让原本认同台湾的人，改而认同中华人民共和国，在国际法上，这是严重的国族变更，当然也不应该受到宪法的保障。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   台湾中央选举委员会表示，听证会举行30天之后，将会就是否同意禁止公开悬挂五星旗的公投做出决定。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
