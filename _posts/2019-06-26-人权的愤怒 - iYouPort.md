   <li>
    <span style="color: #00ccff;">
     <em>
      <strong>
       英国正在向数十个国家出售价值7500万英镑的间谍设备，这些技术被用来打击记者，博主，工会活动家，抗议者和其他任何质疑政府政策或权威的异议人士。为什么还在不要脸地狡辩？
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-2972 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-66.png?resize=1100%2C723&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-66.png?w=1174&amp;ssl=1 1174w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-66.png?resize=300%2C197&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-66.png?resize=768%2C505&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-66.png?resize=1024%2C673&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-66.png?resize=1100%2C723&amp;ssl=1 1100w" data-recalc-dims="1" height="723" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-66.png?resize=1100%2C723&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-2972" data-recalc-dims="1" height="723" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-66.png?resize=1100%2C723&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-66.png?w=1174&amp;ssl=1 1174w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-66.png?resize=300%2C197&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-66.png?resize=768%2C505&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-66.png?resize=1024%2C673&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-66.png?resize=1100%2C723&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   英国已经批准向数十个国家出售价值7500万英镑的电信骇客设备，其中包括一些对持不同政见者发动残酷镇压的国家。
  </p>
  <p class="graf graf--p">
   这一揭露令人权组织感到震惊，这些组织质疑英国制造的技术是否被独裁者用来瞄准他们的对手。
  </p>
  <p class="graf graf--p">
   根据对电信拦截设备出口许可证的分析，自2015年首次单独整理数据以来，
   <strong class="markup--strong markup--p-strong">
    英国已获得了7500万英镑的间谍软件和监控技术的出口许可。这些监视镇压技术被出售给多个拥有可疑人权记录的国家，包括巴林，沙特阿拉伯，菲律宾和土耳其
   </strong>
   。
  </p>
  <p class="graf graf--p">
   *
   <em class="markup--em markup--p-em">
    在 IYP 类目中
   </em>
   <a class="markup--anchor markup--p-anchor" data-href="https://docs.google.com/document/d/1zHDRsb7BtR05ILg3F3rhAxBAcD9PJpbBwpGXPaAPP9w/edit?usp=sharing" href="https://www.iyouport.org/category/%E7%9B%91%E8%A7%86%E4%B8%8E%E6%93%8D%E7%BA%B5/" rel="noopener noreferrer" target="_blank">
    <em class="markup--em markup--p-em">
     “监视与操纵”部分
    </em>
   </a>
   <em class="markup--em markup--p-em">
    看到全球出口间谍技术的更多详细报告；在“政府的间谍行为”部分看到当权者对这些技术的使用报告。
   </em>
  </p>
  <p class="graf graf--p">
   一些出口许可证被确认是用于窃听电话交谈的 IMSI 捕手，而另一些用于监视软件。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这不仅仅是可用于压迫和权力滥用的常规武器，它也是监听和监视的设备，”反对武器贸易运动的 Andrew Smith 说，“将这种设备出售给那些可以用它来抵制积极人士和活动家的政权是完全鲁莽的和不负责任的。”
  </p>
  <p class="graf graf--p">
   Smith 质疑政府没有做过任何检查以确保该技术不被用于促进侵犯人权。“这些销售是一种政策的表现，政府一贯优先考虑武器出口而不是人权。一旦这种设备离开英国海岸，就无法充分调节它的使用方式和使用对象。“
  </p>
  <p class="graf graf--p">
   负责竞选组织 Global Justice Now 的律师事务所 Leigh Day 致函国际贸易部，要求澄清出口的监控设备如何在国外使用。通
   <strong class="markup--strong markup--p-strong">
    过信息自由请求获取信息的尝试几乎没有产生效果
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “坦率地说，就国际贸易部而言，信息自由法案已彻底崩溃，”Global Justice Now 主任 Nick Dearden 说。“他们是在尊重透明度和信息自由方面表现最差的政府部门。这使我们完全无法控制他们的行为和活动。“
  </p>
  <p class="graf graf--p">
   Leigh Day 指出，IMSI 捕手向菲律宾的出口是在其总统罗德里戈·杜特尔特承认该国至少已经窃听了两名政客，其中一人随后被杀，之后进行交易的。
  </p>
  <p class="graf graf--p">
   在杜特尔特政权实施的更广泛的侵犯人权行为的背景下，对这种监视设备的滥用情况令人担忧，这些技术帮助了国家支持的“达沃敢死队”的法外杀戮、死刑、威胁、恐吓和对批评者的监禁，记者和任何收集侵犯人权证据的人，以及包括非政府组织和联合国人士都在政府的黑名单上。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    * 关于敢死队的暗杀如何被中国和美国的监视技术所协助的报告，在这里看到《
   </em>
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/newsletters-06202019-8d2b6c303dc1" href="https://www.iyouport.org/%e8%b5%b0%e8%bf%9b%e6%9a%97%e6%9d%80%e4%b9%8b%e5%9f%8e/" rel="noopener noreferrer" target="_blank">
    <em class="markup--em markup--p-em">
     走进暗杀之城
    </em>
   </a>
   <em class="markup--em markup--p-em">
    》
   </em>
  </p>
  <p class="graf graf--p">
   报告还强调了这样一种担忧，即
   <strong class="markup--strong markup--p-strong">
    能够远程安装在通信设备上以提取数据和控制功能的入侵软件可能已被出口到埃及，埃及已开始打击记者，博主，工会活动家，抗议者和其他任何质疑政府政策或权威的异议人士
   </strong>
   。
  </p>
  <p class="graf graf--p">
   能够对公民通信进行大规模监视的专业监视技术还被许可出口到沙特阿拉伯，该国的间谍已经残忍地谋杀了美国记者 Jamal Khashoggi。
  </p>
  <p class="graf graf--p">
   国际商务大臣利亚姆•福克斯（Liam Fox）在回应 Leigh Day 的信中表示，未经认真考虑，不会签发出口许可。我们自然也会更广泛地考虑进口国对国际人权法的态度，以及设备的使用之间是否存在直接或因果关系，这可能超出了你提出的压制异议的具体问题和那些言论自由权的问题，“他说。
  </p>
  <p class="graf graf--p">
   但是，反对国家监视的隐私国际组织负责人的 Edin Omanovich 表示，英国的许可证制度需要进行紧急改革。“如果一些许可证被批准给我们明确知道的滥用权力侵犯人权的国家，那么该系统就必须被打破，”他说。
  </p>
