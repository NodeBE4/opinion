  <p class="graf graf--p">
   端到端加密消息传递是所谓的执法的一个主要障碍 —— 全球各国政府一直在破解加密的尝试中歪招迭出。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这堆政府面临的关键挑战是：这些服务是由技术公司提供的，而这些公司大多位于美国
   </strong>
   ，这在很大程度上令这些应用无法被其他地方的立法者触及。
  </p>
  <p class="graf graf--p">
   消息传递服务是“over-the-top”，这意味着内容或服务建构在网络基础服务之上，而不需要电信运营商额外的支持。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    所有这些都意味着，像大多数互联网技术一样，这里的权力经纪人是美国政府。
   </strong>
  </p>
  <p class="graf graf--p">
   这就是为什么 Politico 报道“特朗普政府高级官员会面讨论是否寻求
   <strong class="markup--strong markup--p-strong">
    立法禁止
   </strong>
   科技公司使用无法破解的加密形式”，这件事值得重点警惕。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “技术正在快速发展，隐私需要跟上这一变化”，超级安全消息平台 Wickr 的首席执行官 Joel Wallenstrom 说。
  </p>
  <p class="graf graf--p">
   Politico 引用了一些未透露姓名的消息来源报道称：“加密挑战，政府称之为‘going dark’，是国安委会议的焦点，其中包括来自几个主要机构的副手官员。”
  </p>
  <p class="graf graf--p">
   该讨论的重点是消息应用的“隐私和安全功能”，“使当局调查恐怖主义、贩毒和儿童色情内容受挫”。
  </p>
  <p class="graf graf--p">
   包括美国在内的各国政府面临的挑战是，在正在进行的有关隐私、数据安全和信息完整性的辩论中，消息传递应用的隐私已经成为一个中心主题。
  </p>
  <p class="graf graf--p">
   世界各地的人们正在从公共社交媒体发布转向了封闭的通信平台群组，而消息传递应用的竞相出现也是这方面的主要推动力。
  </p>
  <p class="graf graf--p">
   甚至 Facebook 也将消息安全和隐私置于其新战略的中心。请注意，隐私不是目的地，只是一个开始，却已经被寡头公司利用为挡箭牌了，详细《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%a5%e9%9a%90%e7%a7%81%e4%b8%ba%e7%9b%ae%e7%9a%84%e7%9a%84%e6%96%97%e4%ba%89%e5%a6%82%e4%bd%95%e8%ae%a9%e4%bd%a0%e5%8f%98%e6%88%90%e4%ba%86%e7%9b%91%e8%a7%86%e7%9a%84%e7%9b%9f%e5%8f%8b/" href="https://www.iyouport.org/%e4%bb%a5%e9%9a%90%e7%a7%81%e4%b8%ba%e7%9b%ae%e7%9a%84%e7%9a%84%e6%96%97%e4%ba%89%e5%a6%82%e4%bd%95%e8%ae%a9%e4%bd%a0%e5%8f%98%e6%88%90%e4%ba%86%e7%9b%91%e8%a7%86%e7%9a%84%e7%9b%9f%e5%8f%8b/" rel="noopener noreferrer" target="_blank">
    以隐私为目的的斗争如何让你变成了监视的盟友
   </a>
   》。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们希望有真正富有成效的对话和解决问题的方法，”Wallenstrom 告诉我，在NSC新闻爆发之前。
  </p>
  <p class="graf graf--p">
   在他看来，需要避免政府方面的“划红线”和“顽固不化”，中国、朝鲜和伊朗“就技术而言，不是我们想要效仿的模版”。
  </p>
  <p class="graf graf--p">
   微信在中国的例子特别明显，当局在相当公开的基础上监控信息流量，并随时可以对任何被认为不当的行为进行制裁。
  </p>
  <p class="graf graf--p">
   天安门广场30周年的时候，据报道，微信用户发现“与该活动相关的关键词或图片几乎被即时删除，他们的照片和海报有时会立即被封锁。在那段时间里人们甚至无法更换头像。
  </p>
  <p class="graf graf--p">
   最近在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%a6%99%e6%b8%af%ef%bc%9a%e5%b9%b4%e8%bd%bb%e7%9a%84%e6%8a%97%e8%ae%ae%e8%80%85%e6%93%85%e9%95%bf%e6%8a%b5%e5%be%a1%e6%95%b0%e5%ad%97%e7%9b%91%e6%8e%a7%ef%bc%88%e9%99%84%e5%ae%89%e5%85%a8%e9%a1%bb/" href="https://www.iyouport.org/%e9%a6%99%e6%b8%af%ef%bc%9a%e5%b9%b4%e8%bd%bb%e7%9a%84%e6%8a%97%e8%ae%ae%e8%80%85%e6%93%85%e9%95%bf%e6%8a%b5%e5%be%a1%e6%95%b0%e5%ad%97%e7%9b%91%e6%8e%a7%ef%bc%88%e9%99%84%e5%ae%89%e5%85%a8%e9%a1%bb/" rel="noopener noreferrer" target="_blank">
    香港举行的公众抗议活动
   </a>
   中也有许多类似的故事。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们希望这种[端到端加密]技术能够存在，”Wallenstrom 说，“它不可阻挡……这种技术的提供者希望解决这些问题并找到前进的明智之路，不要变成二元悖论，不能说我们不会与[政府]合作来完成这些辩论。”
  </p>
  <p class="graf graf--p">
   Politico 报道，在这个问题上，美国政府内部没有一统一的观点，甚至是“行政部门内部也没有”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “商务部和国务院不同意，指出强制加密’后门’的经济、安全和外交后果。”
  </p>
  <p class="graf graf--p">
   国土安全部内部甚至存在完全矛盾的利益冲突。“网络安全和基础设施部门知道加密敏感数据的重要性，特别是在关键基础设施运营中；但 ICE 和特勤局认为加密是障碍。”
  </p>
  <p class="graf graf--p">
   Wallenstrom 引用了2015年的圣贝纳迪诺恐怖袭击事件，该事件是司法部对抗苹果公司以获取其中一名攻击者的 iPhone。
  </p>
  <p class="graf graf--p">
   Politico 做了同样的事。
  </p>
  <p class="graf graf--p">
   对于政府而言，恐怖主义、贩卖儿童和严重有组织犯罪的因素在处理公众辩论隐私方面成为一个王牌，最终总是：“yes, but”句式。
  </p>
  <p class="graf graf--p">
   早些时候，一个由技术公司，隐私专家和人权组织组成的联盟发布了对英国间谍机构 GCHQ 的讨论文件的公开回应，该文件暗示了一个以 ghost 协议终结加密的想法，允许政府随时监听。
  </p>
  <p class="graf graf--p">
   来自苹果，微软，谷歌和 WhatsApp 等公司的回应是直言不讳的：“这会破坏认证过程……引入潜在的无意识漏洞，增加通信系统可能被滥用或误用的风险……对话是否受到加密保护已经无关紧要。通讯本身不再安全。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “幽灵协议的想法已经被证明是不可持续的，”Wallenstrom 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “决定谁可以使用这种[拦截]技术，这意味着我们的目标仅仅是确定谁是好人，谁是坏人”。
  </p>
  <p class="graf graf--p">
   他还指出，删除隐私保护会暴露内容，因此平台本身可以“窥探用户数据”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果美国确实开始对禁止加密的立法进行正式辩论，那么反弹会很快升级。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “管理”社交媒体上共享的内容是一回事，终结用户隐私就是另一回事了。
  </p>
  <p class="graf graf--p">
   但当权者执迷于执法和所谓的国家安全。从四骑士到六骑士，这些陈词滥调从未被过时。
  </p>
  <p class="graf graf--p">
   真的没有简单的答案。
  </p>
  <p class="graf graf--p">
   马克扎克伯格在3月写道，“我相信沟通的未来将转向私人加密服务，人们可以放心，他们对彼此的言论保持安全，信息和内容不会永远存在……”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “没错，但是……”。
  </p>
  <p class="graf graf--p">
   **警告：当权者提出的任何看起来所谓折中的方案都不能答应。这道防线一旦一处破漏，就会迅速全面倒塌。
  </p>
  <p class="graf graf--p">
   永远记着权势是需要被防备的东西，不论他们看起来有多善意。
  </p>
  <p class="graf graf--p">
   IYP 的立场在“为自由而战”系列中有详细讲述。例如《

    卧室里的坦克 — — 被军事化的互联网人，以及可以怎么办
   </a>
   》；《
