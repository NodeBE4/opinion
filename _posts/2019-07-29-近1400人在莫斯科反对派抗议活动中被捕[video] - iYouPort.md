  <p class="graf graf--p">
   周一在莫斯科举行的抗议活动中已经有近1,400人被捕。反对派领导人阿列克谢·纳瓦尔尼（Alexei Navalny）已经被判入狱30天。
  </p>
  <p class="graf graf--p">
   自2011年以来一直在记录警方逮捕行为的 OVD-Info 报道，周六抗议活动中的被拘留人数已经达到了1,373人。
  </p>
  <p class="graf graf--p">
   OVD-Info 和一位法律援助组织律师周日表示，绝大多数人很快被释放，但仍有150人被拘留。
  </p>
  <p class="graf graf--p">
   国家新闻机构 Tass 和 RIA-Novosti 引用警方的话说，有1,074人被捕。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这是莫斯科多年来最大规模的抗议活动
   </strong>
   。上述大规模逮捕发生在抗议持续一周后，当时约有22,000人参加抗议，为推翻当局阻止反对派候选人参加9月份选举的决定。
  </p>
  <p class="graf graf--p">
   莫斯科市长 Sergei Sobyanin 称未经授权的抗议是“安全威胁”，并补充说“将根据相关法律确保秩序”。
  </p>
  <p class="graf graf--p">
   在普京不受欢迎的养老金改革和连续五年实际收入下降的情况下，抗议运动也在试图冲击俄罗斯严格控制的政治体制。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-7080 jetpack-lazy-image" data-lazy-sizes="(max-width: 1060px) 100vw, 1060px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-20.png?resize=1060%2C670&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-20.png?w=1060&amp;ssl=1 1060w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-20.png?resize=300%2C190&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-20.png?resize=768%2C485&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-20.png?resize=1024%2C647&amp;ssl=1 1024w" data-recalc-dims="1" height="670" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-20.png?resize=1060%2C670&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1060"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-7080" data-recalc-dims="1" height="670" sizes="(max-width: 1060px) 100vw, 1060px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-20.png?resize=1060%2C670&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-20.png?w=1060&amp;ssl=1 1060w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-20.png?resize=300%2C190&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-20.png?resize=768%2C485&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-20.png?resize=1024%2C647&amp;ssl=1 1024w" width="1060"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Navalny 因召集示威而被判入狱30天。其他主要反对派人士和潜在候选人也被逮捕，他们所有人的家在事件发生前的几小时和几天内被搜查。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “老实说，我很害怕，”42岁的IT工作人员 Alexei Sprizhitsky 在示威时对媒体说。
  </p>
  <p class="graf graf--p">
   在这里看到视频：
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/adXa324J0WA" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   莫斯科市中心的安全措施非常紧张，警察关闭了市政厅外的区域，抗议者计划聚集在该区域，警方的封锁迫使参与者走到小街上。
  </p>
  <p class="graf graf--p">
   当警察封锁该区域时，人群高呼：“这是我们的城市！”，“羞耻！”，“我们要自由选举”。
  </p>
  <p class="graf graf--p">
   政治家和被取消参选资格的候选人 Dmitry Gudkov 在游行前不久被捕，并在晚上获释。早些时候他曾说过这个国家的未来受到了威胁。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果我们现在失败，选举将不再作为政治工具存在，”他说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们所谈论的是公民参与今天的俄罗斯政治还是否合法，我们谈论的是我们要继续居住的国家。”
  </p>
  <p class="graf graf--p">
   虽然亲克里姆林宫的候选人得到了国家的支持，但独立候选人表示他们已经成功跳过无数障碍。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-7081 jetpack-lazy-image" data-lazy-sizes="(max-width: 1060px) 100vw, 1060px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-21.png?resize=1060%2C670&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-21.png?w=1060&amp;ssl=1 1060w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-21.png?resize=300%2C190&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-21.png?resize=768%2C485&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-21.png?resize=1024%2C647&amp;ssl=1 1024w" data-recalc-dims="1" height="670" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-21.png?resize=1060%2C670&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1060"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-7081" data-recalc-dims="1" height="670" sizes="(max-width: 1060px) 100vw, 1060px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-21.png?resize=1060%2C670&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-21.png?w=1060&amp;ssl=1 1060w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-21.png?resize=300%2C190&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-21.png?resize=768%2C485&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-21.png?resize=1024%2C647&amp;ssl=1 1024w" width="1060"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在活动家和普通的莫斯科公民上周走上街头后，包括在地方选举委员会大楼外，官方调查人员表示，他们正在发起“阻碍选举官员工作”的刑事调查。
  </p>
  <p class="graf graf--p">
   如果被定罪，组织者可能会被判五年徒刑。
  </p>
  <p class="graf graf--p">
   Navalny 盟友 Ivan Zhdanov 说他在示威发起前不久被捕；被禁止的候选人 Ilya Yashin 在星期六早上在被警方突袭了他的家后被拘留。
  </p>
  <p class="graf graf--p">
   本周发起绝食的潜在候选人 Lyubov Sobol 在示威中被逮捕，之后被释放并被罚款 30,000 卢布（475美元）。
  </p>
  <p class="graf graf--p">
   68岁的示威者 Elena Rastovka 告诉媒体：“我一生都害怕，但已经足够了。如果我们一直呆在家里，一切都不会改变”。
  </p>
  <p class="graf graf--p">
   路透社采访的目击者说，一些被拘留的人似乎是偶然出现在示威地点的普通路人。
  </p>
  <p class="graf graf--p">
   其中一名被拘留者，45岁的 Alexander Latyshev 说，他是从附近的弗拉基米尔地区来到莫斯科的，与一名助手讨论业务，并被警方随机拘留。
  </p>
  <p class="graf graf--p">
   虽然莫斯科市议会议员的选举不是全国性的，但反对派活动家认为这是一个有希望在俄罗斯首都获得立足点的机会，克里姆林宫支持的候选人在这里比在其他地方更不受欢迎。
  </p>
  <p class="graf graf--p">
   一些人说，当局的强烈反应使当地问题成为了一场重大的抗议活动。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “谁会想到参加像莫斯科议会选举这样奇怪而无聊的事情会变得如此重要？”在集会之前30岁的插画艺术家 Viktoria Popova 说。
  </p>
  <p class="graf graf--p">
   这场集会在 Facebook 召集的时候获得了 11,000人的点赞。
  </p>
  <p class="graf graf--p">
   警方要求准备报道这场抗议运动的所有媒体主动通知当局；并呼吁普通市民绕过抗议地点。⚪️
  </p>
