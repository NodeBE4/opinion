  <p class="graf graf--p">
   Wikileaks 创始人 Julian Assange 被特朗普政府司法部
   <a class="markup--anchor markup--p-anchor" data-href="https://web.archive.org/web/20190524001203/https://www.theguardian.com/media/2019/may/23/wikileaks-founder-julian-assange-with-violating-the-espionage-act-in-18-count-indictment" href="https://web.archive.org/web/20190524001203/https://www.theguardian.com/media/2019/may/23/wikileaks-founder-julian-assange-with-violating-the-espionage-act-in-18-count-indictment" rel="noopener noreferrer" target="_blank">
    指控17项罪名
   </a>
   ，最高刑期为175年。正如 Assange 和他的辩护人一直以来所警告那样，这一警告持续了近十年。有多少人在听？
  </p>
  <p class="graf graf--p">
   这份起诉书就如上个月 Assange 的被捕一样完全是欺诈性的，因为它起诉 Assange 的所谓“罪行”与传统的新闻实践完全无法区分。
  </p>
  <p class="graf graf--p">
   这些指控基于与奥巴马政府当年相同的理由 ——
   <strong class="markup--strong markup--p-strong">
    正如调查记者 Glenn Greenwald 去年所指出的那样 —— 当年 奥巴马的司法部不得不放弃了起诉 Assange，理由是要付出摧毁新闻自由的代价
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在这里看到原文中文版《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/dnc-%E9%92%88%E5%AF%B9-wikileaks-%E7%9A%84%E8%AF%89%E8%AE%BC%E5%AF%B9%E6%96%B0%E9%97%BB%E8%87%AA%E7%94%B1%E6%9E%84%E6%88%90%E4%B8%A5%E9%87%8D%E5%A8%81%E8%83%81-%F0%9F%86%93%E2%8F%B3/" href="https://www.iyouport.org/dnc-%E9%92%88%E5%AF%B9-wikileaks-%E7%9A%84%E8%AF%89%E8%AE%BC%E5%AF%B9%E6%96%B0%E9%97%BB%E8%87%AA%E7%94%B1%E6%9E%84%E6%88%90%E4%B8%A5%E9%87%8D%E5%A8%81%E8%83%81-%F0%9F%86%93%E2%8F%B3/" rel="noopener noreferrer" target="_blank">
    DNC 针对 Wikileaks 的诉讼对新闻自由构成严重威胁
   </a>
   》。
  </p>
  <p class="graf graf--p">
   德雷克塞尔大学托马斯·克莱恩法学院副教授 Hanna Bloch-Wehba 表示，
   <a class="markup--anchor markup--p-anchor" data-href="https://threadreaderapp.com/thread/1131668294052696064.html" href="https://threadreaderapp.com/thread/1131668294052696064.html" rel="noopener noreferrer" target="_blank">
    这份起诉书是“第一修正案的噩梦
   </a>
   ，就如空难发生前的垂死呼救”。
  </p>
  <p class="graf graf--p">
   Bloch-Wehba 说，起诉书的“责任理论在很大程度上依赖于 Assange 与 Manning 的关系，以及他倾向于鼓励 Manning 继续为维基解密提供材料”，
   <strong class="markup--strong markup--p-strong">
    事实上这就是任何记者在培养与其消息来源的关系时所做的
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “纽约时报”关于 Assange 起诉书的报告中有一个版本已经被修改过，但是对 Slate 的引述保留在那里了，他说，“官员们不会接受有关 Assange 先生被指控的重罪行为与普通调查性新闻没什么不同的问题。
   <strong class="markup--strong markup--p-strong">
    别忘了，纽约时报和许多其他主流新闻机构都在没有得到政府授权的情况下，从 Wikileaks 获得了完全相同的档案文件”。
   </strong>
  </p>
  <p class="graf graf--p">
   新闻自由组织一直在以鲜明和明确的语言谴责这些将新闻工作称之为间谍的指控。
  </p>
  <p class="graf graf--p">
   新闻自由基金会执行主任特雷弗·蒂姆（Trevor Timm）在一份声明中说：“简而言之，对 Julian Assange 和 Wikileaks 的这些前所未有的指控是对21世纪第一修正案的最重大和最可怕的威胁。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “特朗普政府正在明确将新闻定为刑事犯罪，如果这起诉讼得以实施，纽约时报、华盛顿邮报和其他地方的数十名记者都将面临被指控的危险。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “新闻界发布政府希望保密的事实的能力对公众的知情权和基本权利都至关重要。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “司法部的这一决定是特朗普政府新闻战争的一次大规模和前所未有的升级，可以毫不夸张地说是第一修正案本身存在风险。任何关心新闻自由的人都应该立即全心全意地谴责这些指控。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “根据所谓的间谍法对 Julian Assange 发布的机密信息的起诉是对第一修正案的严重攻击、以及对所有发布政府希望保密的信息的记者的威胁，”保护记者委员会执行主任
   <a class="markup--anchor markup--p-anchor" data-href="https://cpj.org/2019/05/julian-assange-charged-espionage-wikileaks.php" href="https://cpj.org/2019/05/julian-assange-charged-espionage-wikileaks.php" rel="noopener noreferrer" target="_blank">
    Joel Simon 发表声明
   </a>
   说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这起诉讼危及美国和世界各地的新闻自由。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “在我国历史上，政府首次根据”间谍法“对发布真实信息的出版商提起刑事指控，”
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.aclu.org/press-releases/aclu-comment-julian-assange-indictment" href="https://www.aclu.org/press-releases/aclu-comment-julian-assange-indictment" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     ACLU 发表声明
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    说。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这是对第一修正案的直接攻击。这些指控是特朗普政府对新闻攻击的特别升级，
   <strong class="markup--strong markup--p-strong">
    它建立了一个危险的先例，可以用来瞄准打击任何希望通过说真话让政府负责的新闻机构
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “对于揭露其他国家秘密的美国记者来说，对 Assange 的指控同样危险。如果美国可以起诉外国出版商违反我们的保密法，那么就没有什么能阻止中国或俄罗斯这样做了。”
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   一些主流媒体的记者似乎已经醒过来了，虽然为时已晚。就在此前不久他们还在努力为官方站台联手攻击 Wikileaks。
  </p>
  <p class="graf graf--p">
   在这里看到：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E2%9A%A1%EF%B8%8F%E8%87%B4%E5%8A%9B%E4%BA%8E%E5%B8%AE%E5%8A%A9%E5%BD%93%E6%9D%83%E8%80%85%E6%8A%B9%E9%BB%91%E6%9B%9D%E5%85%89%E6%9D%83%E5%8A%9B%E9%98%B4%E6%9A%97%E9%9D%A2%E7%9A%84%E4%BA%BA%EF%BC%9A/" href="https://www.iyouport.org/%E2%9A%A1%EF%B8%8F%E8%87%B4%E5%8A%9B%E4%BA%8E%E5%B8%AE%E5%8A%A9%E5%BD%93%E6%9D%83%E8%80%85%E6%8A%B9%E9%BB%91%E6%9B%9D%E5%85%89%E6%9D%83%E5%8A%9B%E9%98%B4%E6%9A%97%E9%9D%A2%E7%9A%84%E4%BA%BA%EF%BC%9A/" rel="noopener noreferrer" target="_blank">
    致力于帮助当权者抹黑曝光权力阴暗面的人：新闻业正在破坏自己的声誉
   </a>
   》、《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E2%9D%97%EF%B8%8F%E4%BC%AA%E5%8A%A3%E6%8A%A5%E9%81%93%E5%92%8C%E6%9C%89%E7%9B%AE%E7%9A%84%E7%9A%84%E6%94%BF%E6%B2%BB%E5%AE%A3%E4%BC%A0%E5%A6%82%E4%BD%95%E6%89%AD%E6%9B%B2%E4%BA%86%E7%8E%B0%E5%AE%9E/" href="https://www.iyouport.org/%E2%9D%97%EF%B8%8F%E4%BC%AA%E5%8A%A3%E6%8A%A5%E9%81%93%E5%92%8C%E6%9C%89%E7%9B%AE%E7%9A%84%E7%9A%84%E6%94%BF%E6%B2%BB%E5%AE%A3%E4%BC%A0%E5%A6%82%E4%BD%95%E6%89%AD%E6%9B%B2%E4%BA%86%E7%8E%B0%E5%AE%9E/" rel="noopener noreferrer" target="_blank">
    伪劣报道和有目的的政治宣传如何扭曲了现实政治
   </a>
   》。
  </p>
  <p class="graf graf--p">
   MSNBC 的 Chris Hayes 在
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/chrislhayes/status/1131660186429599744" href="https://twitter.com/chrislhayes/status/1131660186429599744" rel="noopener noreferrer" target="_blank">

  </p>
  <p class="graf graf--p">
   MSNBC 的 Ari Melber 在
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/AriMelber/status/1131657854904164356" href="https://twitter.com/AriMelber/status/1131657854904164356" rel="noopener noreferrer" target="_blank">

  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “无论一个人对 Julian Assange 的观点如何，这都是一个法律事实。新的特朗普 DOJ 起诉书将大多数顶级报纸的日常工作 —— 收集和发布机密材料 —— 变成了间谍行为……
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    事实上这不是什么突发事件，Wikileaks 的工作人员和他们的支持者多年来一直在警告这一点，只是因为偏执狂的阴谋论家们拒绝听取
   </strong>
   ，他们坚称 Assange 只是在厄瓜多尔驻伦敦大使馆里避免所谓的强奸指控。
  </p>
  <p class="graf graf--p">
   Wikileaks 的推特账号中有很多推文表明，美国在2010年一直试图根据“间谍法案”起诉 Assange，十年来他们一直在警告这件事，但是没有人听过。
  </p>
  <p class="graf graf--p">
   多年来，Assange 一直在警告的这一情况即将到来。他一直在表示，他从一开始就非常明确地表示很愿意参与瑞典的调查，而且由于害怕美国的引渡和政治起诉而只能向厄瓜多尔寻求庇护，厄瓜多尔
   <a class="markup--anchor markup--p-anchor" data-href="http://www.ecuador.org/blog/?p=2120" href="http://www.ecuador.org/blog/?p=2120" rel="noopener noreferrer" target="_blank">
    明确表示
   </a>
   这就是给予 Assange 庇护的理由 —— 人权。
  </p>
  <p class="graf graf--p">
   他一直都是对的。历史证明了他是对的，他的批评者是错的。
  </p>
  <p class="graf graf--p">
   我们也已经看到 Assange 在他的警告中证明了他的起诉对新闻自由的意义。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    一场大大升级的对新闻的战争正在实施，最近几天有新的发展 —— 旧金山的记者因接收举报人递送的警方机密文件而被 FBI 抄家
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.sfchronicle.com/crime/article/Adachi-leak-SF-cops-to-return-property-to-13867179.php" href="https://www.sfchronicle.com/crime/article/Adachi-leak-SF-cops-to-return-property-to-13867179.php" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     被指控“阴谋”
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    、著名的法国记者
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/world/europe/prominent-french-journalist-summoned-by-police-for-reporting-on-corruption-in-macrons-government/2019/05/22/a20b2fb0-7c92-11e9-8ede-f4abf521ef17_story.html?utm_term=.df9029c8a6b7" href="https://www.washingtonpost.com/world/europe/prominent-french-journalist-summoned-by-police-for-reporting-on-corruption-in-macrons-government/2019/05/22/a20b2fb0-7c92-11e9-8ede-f4abf521ef17_story.html?utm_term=.df9029c8a6b7" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     被警方传唤
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，只因报道了马克龙政府的腐败现象
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4430 jetpack-lazy-image" data-lazy-sizes="(max-width: 920px) 100vw, 920px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3-5.png?resize=920%2C832&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3-5.png?w=920&amp;ssl=1 920w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3-5.png?resize=300%2C271&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3-5.png?resize=768%2C695&amp;ssl=1 768w" data-recalc-dims="1" height="832" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3-5.png?resize=920%2C832&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="920"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4430" data-recalc-dims="1" height="832" sizes="(max-width: 920px) 100vw, 920px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3-5.png?resize=920%2C832&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3-5.png?w=920&amp;ssl=1 920w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3-5.png?resize=300%2C271&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3-5.png?resize=768%2C695&amp;ssl=1 768w" width="920"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   所有这一切当然都引出了一个问题：他还有什么其他的权利？任何有一点点智力诚实的人就算曾经对 Assange 有疑问的人，现在也必须开始问自己这个问题了。
  </p>
  <p class="graf graf--p">
   如果您想要完整的故事，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/the-persecution-of-julian-assange-must-end-or-our-future-will-end-in-tragedy-all-of-us/" href="https://www.iyouport.org/the-persecution-of-julian-assange-must-end-or-our-future-will-end-in-tragedy-all-of-us/" rel="noopener noreferrer" target="_blank">
    在这里回顾这
    <strong class="markup--strong markup--p-strong">
     场惊人的联手迫害计划
    </strong>
   </a>
   ，你能在这个故事中找到人性中最肮脏的几乎一切。
  </p>
