  <p class="graf graf--p graf-after--figure" id="e29f">
   当美国特种部队于 2001 年进入阿富汗时，Facebook 并不存在，iPhone 尚未被发明，而“A.I.”经常被称为 NBA 明星。十七年后，美国特种作战部队继续在阿富汗农村骑马，但信息技术已经飞速发展了。最近机器人技术和人工智能（AI）的突破已经引起了大众丰富的想象，并引发了即将发生的“人工智能革命”的话题。
  </p>
  <p class="graf graf--p graf-after--p" id="fd49">
   然而令人惊讶的是，很少有人谈到 21 世纪美国外交政策占主导地位的小型战争和叛乱。
  </p>
  <p class="graf graf--p graf-after--p" id="db82">
   最近有两本新书新版，提供了有关大数据和人工智能时代将如何影响美国现代冲突的建议。其中一本是《
   <em class="markup--em markup--p-em">
    Small Wars, Big Data: The Information Revolution in Modern Conflict
   </em>
   》作者 Eli Berman, Joseph Felter, 和 Jacob Shapiro，该书对反叛的未来几乎没有提供任何思考，但就
   <strong class="markup--strong markup--p-strong">
    信息技术形成反叛的方式
   </strong>
   提出了令人信服的理论；另一本是《
   <em class="markup--em markup--p-em">
    Army of None: Autonomous Weapons and the Future of War
   </em>
   》，作者 Paul Scharre，虽然没有提供反叛乱的战略，但充分关注了
   <strong class="markup--strong markup--p-strong">
    人工智能将如何重塑武装冲突
   </strong>
   。
   <span style="color: #ff6600;">
    在这里下载这本书
   </span>
   ：https://t.me/iyouport/6502
  </p>
  <p class="graf graf--p graf-after--p" id="1f0d">
   总之，这些书开始勾勒出
   <strong class="markup--strong markup--p-strong">
    人工智能和大数据如何改变反叛动态
   </strong>
   的愿景。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="85f5">
   “Small Wars, Big Data“这本书的核心见解是，
   <strong class="markup--strong markup--p-strong">
    反叛最终是对信息而不是领土或意识形态的竞争
   </strong>
   。由于反叛者很容易与当地人口融为一体，政权的部队无法打败反叛者，除非当地居民协助确定反叛者的身份和地点。
  </p>
  <p class="graf graf--p graf-after--p" id="9564">
   因此，政权面临的挑战是说服当地平民提供这些信息，而反叛者面临的挑战是说服他们不要提供信息给政权。在这本书中描述的几乎所有叛乱中发生的事 — — 一方面是建造学校和医院，另一方面是不分青红皂白地屠杀平民 — — 都可以理解为企图哄骗或恐吓平民泄露或隐瞒他们所知道的东西。
  </p>
  <p class="graf graf--p graf-after--p" id="5161">
   这本书绝不是第一个提供这一论点的。但就其整理的经验证据的广度和深度而言，它是独一无二的。从21世纪初 Stathis Kalyvas 的开创性研究开始，Lisa Hultman 和 Laia Balcells 等政治学家就反叛和内战的“微观基础”编写了一系列非凡的实证研究。作为这些文献本身的主要贡献者，该书的作者在调查其研究结果方面做了令人钦佩的工作。
  </p>
  <p class="graf graf--p graf-after--p" id="3103">
   特别值得注意的是第四章，它全面概述了关于信息技术发展对叛乱之影响的辩论。
  </p>
  <p class="graf graf--p graf-after--p" id="a6b7">
   正如作者所指出的那样，
   <strong class="markup--strong markup--p-strong">
    一方面，信息技术可以通过让各国更容易收集有关反叛者的情报来减少反叛者的攻击；另一方面，信息技术也可以通过使反叛者更好地相互协调来增加攻击能力
   </strong>
   。通过比较手机网络和伊拉克叛军的行为，作者认为，手机 — 通过低成本、匿名的方式提供有关反叛者的信息，对线人来说风险很小 — 确实可以减少和破坏反叛活动。
  </p>
  <p class="graf graf--p graf-after--p" id="206a">
   阿富汗的塔利班和尼日利亚的博科圣地都在反复针对手机信号塔，尽管他们的通信能力有所改善，足以表明即使是反叛者本身也会担心
   <strong class="markup--strong markup--p-strong">
    信息技术已经使权力平衡倾向于国家
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="3e66">
   该书对 Jacqueline Hazelton 去年夏天在一篇
   <a class="markup--anchor markup--p-anchor" data-href="https://www.mitpressjournals.org/doi/abs/10.1162/ISEC_a_00283?af=R" href="https://www.mitpressjournals.org/doi/abs/10.1162/ISEC_a_00283?af=R" rel="noopener nofollow noreferrer" target="_blank">
    有争议的文章
   </a>
   中讨论过的那种“蛮力”策略不以为然。如果能更深入地讨论
   <strong class="markup--strong markup--p-strong">
    为什么利用信息技术在打击反叛方面比大规模监禁、大规模重新安置甚至大规模杀戮等严厉政策更有效
   </strong>
   ，那么本书可以变得更好。
  </p>
  <p class="graf graf--p graf-after--p" id="5deb">
   另一个批评是，这本书的标题，暗示了一个它永远无法解决的话题：它主要是一本关于信息在叛乱中的作用的书，而不是信息技术本身。这个书名非常适合营销，但有点用词不当。
  </p>
  <p class="graf graf--p graf-after--p" id="a59b">
   另一本书“Army of None”，Scharre 花了将近十年的时间来构建关于 D.C.和五角大楼自主武器的早期辩论，这本书简明扼要地阐述了人工智能和自主武器为武装冲突的未来所提出的主要问题。
  </p>
  <p class="graf graf--p graf-after--p" id="d9fc">
   虽然这本书是为支持者写的，但即使是知识渊博的学者也会发现，人工智能注入武器系统必定将带入技术、道德和战略等诸多问题。
  </p>
  <p class="graf graf--p graf-after--p" id="2b5b">
   Scharre 关于即将到来的大数据和人工智能无处不在的论点对反叛的未来产生了深远的影响。粗略看两种未来是可能的：
   <strong class="markup--strong markup--p-strong">
    人工智能和自主武器都是分布式和商品化的，因此反叛者可以负担得起几乎与任何特定政权相同的武器系统
   </strong>
   。想想伊斯兰国在伊拉克和叙利亚部署的现代商用无人机，而不是低端情报和传统的检测。
  </p>
  <p class="graf graf--p graf-after--p" id="5e78">
   由于许多尖端的 AI 项目都是开源的并且是公开的，因此很有可能临时建立起致命的自主武器系统，这些系统几乎与最先进的系统一样好，但成本相比下很低。在这种情况下，权力平衡可能会略微转向反叛者; 如果袭击者拥有致命的无人机群……你完全可以想象
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnn.com/2013/09/18/world/asia/mumbai-terror-attacks/index.html" href="https://www.cnn.com/2013/09/18/world/asia/mumbai-terror-attacks/index.html" rel="noopener nofollow noreferrer" target="_blank">
    孟买那种袭击
   </a>
   的结果可能就是大屠杀。
  </p>
  <p class="graf graf--p graf-after--p" id="07eb">
   然而，
   <strong class="markup--strong markup--p-strong">
    如果人工智能的未来有利于规模和集中化，那么人工智能可能会给政权力量带来上风。全球范围内能够获得大国产品和基础设施的政权，如美国或中国，可能会迅速消解反叛者网络。
   </strong>
   甚至可以想象一种“没有战争”的情景，由于政权掌握了压倒性的信息优势……
  </p>
  <p class="graf graf--p graf-after--p" id="0b40">
   回想一下
   <em class="markup--em markup--p-em">
    Small Wars, Big Data
   </em>
   这本书的中心点是，反叛活动主要是关于反叛者身份和地点的信息，即使是相对简单的技术 — 基于文本的提示 — 似乎能让各国更容易获得这类信息。那么使用配备面部识别功能的实时监控系统替换 tipline 时会发生什么？
  </p>
  <p class="graf graf--p graf-after--p" id="6d60">
   随着普遍监视的成本下降，可能会看到
   <strong class="markup--strong markup--p-strong">
    新的反叛乱策略的出现
   </strong>
   ：虽然诱骗的方法试图从民众那里获得信息，强制的方法也可能逼迫出信息透露，而
   <strong class="markup--strong markup--p-strong">
    “老大哥”的做法可以绕过平民
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="c364">
   <strong class="markup--strong markup--p-strong">
    如果你有足够的传感器、摄像头和处理能力来识别和跟踪每个人，那么就几乎不需要线人了。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="701b">
   尽管这种想法看起来非常牵强，但早期版本已经开始推行了。
  </p>
  <p class="graf graf--p graf-after--p" id="e3e8">
   在美国，由 Peter Thiel 资助的最新创业公司 Anduril Industries 正在快速建造一个
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wired.com/story/palmer-luckey-anduril-border-wall/" href="https://www.wired.com/story/palmer-luckey-anduril-border-wall/" rel="noopener nofollow noreferrer" target="_blank">
    “电子”墙
   </a>
   ，该墙已经证明在检测和监控德克萨斯州未经授权的过境点方面非​​常有效；
  </p>
  <p class="graf graf--p graf-after--p" id="b19b">
   与此同时，在中国，当局通过广泛部署面部识别系统和大规模监视技术以压制新疆地区。中国当局的能力只会增长：北京最近宣布对人工智能创业公司 SenseTime 进行了大规模的投资，该公司的下一代产品旨在
   <strong class="markup--strong markup--p-strong">
    同时识别 100,000 个实时摄像头输入中的物体和个人
   </strong>
   。
  </p>

  <p class="graf graf--p graf-after--p" id="10c7">
   人工智能和信息技术究竟是会赋予国家还是反叛者以权力，仍然不明朗。最有可能的是，它会同时赋予两者 — — 人工智能和信息技术在失败国或弱国增加了反叛，同时，其他地方也就是强国，发生反叛的空间很可能被压制。
  </p>
  <p class="graf graf--p graf-after--p" id="bf72">
   似乎可以肯定的是，信息革命很快就会彻底改变反叛。
  </p>
