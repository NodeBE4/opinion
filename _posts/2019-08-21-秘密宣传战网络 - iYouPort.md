 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      这是一个秘密宣传战网络。他们汇集了一大群高级军事和情报人员、记者和学者，制作和传播有利于本国及其盟国地缘政治和经济目标的宣传战信息。他们是如何被曝光的？
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   此前我们已经介绍过很多关于信息战和秘密宣传的案例及其操作方法。宣传真的不是你能看得到的表面现象那么简单，它的目的就是让你意识不到，让你下意识地感觉被灌输的东西“出自自己的本心”。这样的结果是你不仅不会主动去揭穿谎言，而且会拼命维护谎言，就如维护你“自己”那样。本文也是信息战的一个案例，虽然它没那么复杂，但是掩护得很好，它利用合法的慈善组织作为宣传战武器，直到其被揭穿。
  </p>
  <p class="graf graf--p">
   Institute for Statecraft 及其分支机构
   <a class="markup--anchor markup--p-anchor" data-href="https://politicsandinsights.org/2019/01/10/defending-disinformation-against-democracy-the-integrity-initiative/" href="https://politicsandinsights.org/2019/01/10/defending-disinformation-against-democracy-the-integrity-initiative/" rel="noopener noreferrer" target="_blank">
    “诚信倡议”
   </a>
   构成了一个与英国间谍机构和更广泛的其他机构密切相关的秘密宣传战网络。
   <strong class="markup--strong markup--p-strong">
    他们汇集了一大群高级军事和情报人员、记者和学者，制作和传播有利于英国及其盟国地缘政治和经济目标的宣传
   </strong>
   。
  </p>
  <p class="graf graf--p">
   去年11月，匿名者组织
   <a class="markup--anchor markup--p-anchor" data-href="https://kittysjones.wordpress.com/2018/12/09/authoritarian-uk-government-is-funding-military-grade-psyops-to-smear-and-calumniate-hms-opposition/" href="https://kittysjones.wordpress.com/2018/12/09/authoritarian-uk-government-is-funding-military-grade-psyops-to-smear-and-calumniate-hms-opposition/" rel="noopener noreferrer" target="_blank">
    发布了一系列文件
   </a>
   ，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cyberguerrilla.org/blog/operation-integrity-initiative-british-informational-war-against-all-part-4/" href="https://www.cyberguerrilla.org/blog/operation-integrity-initiative-british-informational-war-against-all-part-4/" rel="noopener noreferrer" target="_blank">
    文件中
   </a>
   详细描述了
   <strong class="markup--strong markup--p-strong">
    一个由诸多政治家、记者、军事人员、科学家、和学术界人士组成的宣传战网络，明确称之为“战斗”，这是由北约附属的英国资助的 Statecraft 研究所于2015年创建的。
   </strong>
  </p>
  <p class="graf graf--p">
   所谓的“诚信倡议“由军事情报和通信专家负责。
  </p>
  <p class="graf graf--p">
   这个高度机密的、政府资助的“网络的网络”已经发现自己受到严格审查，他们将反对派领导人 Jeremy Corbyn 称为“克里姆林宫的傀儡” — 表面上伪装成反对“俄罗斯虚假信息”的一部分。
  </p>
  <p class="graf graf--p">
   被曝光的文件显示，
   <strong class="markup--strong markup--p-strong">
    信息战部门在整个欧洲发展了一大批亲权势的记者和“关键影响力人士”的秘密“集群”，他们利用社交媒体进行活动
   </strong>
   ，以“反击虚假信息”为幌子伪装自己。并且，该倡议在两年内从
   <strong class="markup--strong markup--p-strong">
    外交部获得了超过220万英镑的资助
   </strong>
   。
  </p>
  <p class="graf graf--p">
   事实上该倡议只是
   <a class="markup--anchor markup--p-anchor" data-href="https://politicsandinsights.org/2019/01/10/defending-disinformation-against-democracy-the-integrity-initiative/" href="https://politicsandinsights.org/2019/01/10/defending-disinformation-against-democracy-the-integrity-initiative/" rel="noopener noreferrer" target="_blank">
    捍卫了反民主的虚假信息
   </a>
   。
  </p>
  <p class="graf graf--p">
   泄密事件表明，该组织在去年3月 Sergei Skripal 和他的女儿 Yulia 在索尔兹伯里神秘中毒后，在塑造媒体叙述方面发挥了核心作用。值得注意的是，该组织早在2015年就提出的许多严厉的措施都是在 Skripal 事件之后迅速得到实施的 — — 尽管白厅拒绝用证据支持其责备框架。
  </p>
  <p class="graf graf--p">
   该倡议服务于
   <strong class="markup--strong markup--p-strong">
    高度政治性的“战略通信”角色
   </strong>
   是毋庸置疑的。
  </p>
  <p class="graf graf--p">
   所谓的“诚信倡议“自我宣布的“慈善机构”，由英国外交部、英国陆军和国防部资助，已被星期日邮报描述为右翼信息部队。但更清楚的描述应该是：一个右翼在编的秘密行动宣传单位。
  </p>
  <p class="graf graf--p">
   在该组织现已向 Jeremy Corbyn 道歉后，工党再次呼吁对政府资助的该行动进行调查，并且最表面的证据是它违反了慈善法。
  </p>
  <p class="graf graf--p">
   星期日邮报曝光的推文发布后，Integrity Initiative 及其支持者一开始否认 Corbyn 受到了不公平的攻击。
  </p>
  <p class="graf graf--p">
   然而，几个月后又忽然出现了道歉，IFS 的创始人 Chris Donnelly 显然承认这些活动违反了外交部规则和苏格兰慈善法。该团体的注册地址是法夫行政区。
  </p>
  <figure aria-describedby="caption-attachment-2273" class="wp-caption aligncenter" id="attachment_2273" style="width: 1067px">

  <p class="graf graf--p">
   工党领袖 Neil Findlay 说：“该组织已经道歉是正确和恰当的，但是，这里还有更多严肃的问题需要得到解答。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “这是一家在苏格兰注册的并由苏格兰慈善监管机构监管的组织，由英国政府资助。它应该永远不会对工党和工党运动进行政治攻击。
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     这种明显的政治攻击不应该来自任何慈善机构。我们需要知道为什么外交部一直在资助该组织。
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    这是不被允许的。我们需要对该组织的行动及其与保守党政府的联系进行全面调查。
   </em>
   “
  </p>
  <p class="graf graf--p">
   Findlay 此前曾表示：
   <em class="markup--em markup--p-em">
    “所谓诚信倡议的叙事已经变得更加隐晦和模糊 — — 现在我们看到它暴露出
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     他们受到了伊拉克灾难期间流传的一些最糟糕的假新闻背后组织的支持
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    。
   </em>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “英国议会和苏格兰的慈善监管机构 OSCR 现在必须认真考虑这个所谓的慈善机构的活动和资金来源，
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     这些慈善机构似乎只不过是一个宣传战的战线
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    。
   </em>
   ”
  </p>
  <p class="graf graf--p">
   所谓的“诚信倡议”已经获得了200万英镑的公共资金，已经成为苏格兰慈善监管机构（OSCR）调查的主题。
  </p>
  <p class="graf graf--p">
   然而，“纽约时报”发表的引述显示，Donnelly 似乎承认 OSCR 的规则已被打破。
  </p>
  <p class="graf graf--p">
   他说：
   <em class="markup--em markup--p-em">
    “我们推出了26,000条推文。大约400条提到了一些政党或政治家，但我们不应该派遣他们，因为外交部不允许我们发表任何政党相关的评论，苏格兰慈善法也不允许。
   </em>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “这是一个错误，我们给 Jeremy Corbyn 写了道歉信。我是两位保守党国防部长的特别顾问，也是工党 John Reid 和 George Robertson 的特别顾问，所以我们应该是非政治的。
   </em>
   “
  </p>
  <p class="graf graf--p">
   事实证明，
   <strong class="markup--strong markup--p-strong">
    这个慈善机构与臭名昭著的战略通信大师 John Rendon 有联系，他们的 Rendon 集团在90年代时就被中央情报局聘用，负责宣传“伊拉克拥有大规模杀伤性武器”的虚假故事。
   </strong>
  </p>
  <p class="graf graf--p">
   匿名者曝光的文件显示，
   <strong class="markup--strong markup--p-strong">
    自称为“信息战战士”和“感知管理人”的 Rendon 也是一个价值45,000英镑的研讨会主讲人，为那个所谓的“诚信倡议”组织教育“宣传战核心团队和集群”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这个 Donnelly 是军事情报部门的名誉上校。该部门领导层的另一名成员 Dan Lafayeedney 于1978年担任 SAS 士兵，负责指导 Stephen Dalziel 从事军事情报工作。
  </p>
  <p class="graf graf--p">
   工党已经要求进行独立调查。外交部长 Alan Duncan 起初承诺进行全面调查，但是后来又改嘴了，试图将这起丑闻视为“俄罗斯的虚假信息”。所谓的“诚信倡议“官员辩称，其他推文“也一直批评其他政党的政客”。然而，数据显示，
   <strong class="markup--strong markup--p-strong">
    其批评的重点目标是 HM 的反对党领袖
   </strong>
   。
  </p>
  <p class="graf graf--p">
   工党的 Chris Williamson 此前曾表示：
   <em class="markup--em markup--p-em">
    “宣传战中最令人担忧的一个方面就是将这个国家推向战争基础的这一看似隐蔽的努力。
   </em>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “像 John Rendon 这样的宣传战大师参与其中的事实是非常令人担忧的，因为这正是他专门研究的那种东西。我们将要求在议会进行更多辩论，并要求外交部提供更多答案，以便找出这里发生了什么。
   </em>
   ”
  </p>

  <p class="graf graf--p">
   到目前为止，还没有看到任何人将这个宣传战工具“诚信倡议”的曝光与剑桥分析公司丑闻联系起来。我们此前发布的长篇报告已经揭露（详见《
   <a href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" rel="noopener noreferrer" target="_blank">
    心理操纵的秘密
   </a>
   》），
   <strong class="markup--strong markup--p-strong">
    政府雇用了多个神秘的公司，他们在竞选活动中使用与剑桥分析公司相同的方法，花费了数十万英镑的公民资金，因此保守党可以利用针对性洗脑的黑暗广告和精心制作的宣传战来瞄准目标选民
   </strong>
   。
  </p>
  <p class="graf graf--p">
   一旦保守党在2010年随便宣布他们的“行为改变”议程，并设立了’Nudge Unit’，剑桥分析公司/ SCL 这类的丑闻就不可避免了。人们怎么可能期望一个越来越专制的政府会抵制利用所谓的“行为科学”技术来操纵公民的观念、认知、行为、选择以及最终投票决策的诱惑？
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “监视资本主义 ”是哈佛大学学者 Shoshanna Zuboff 在2015年创造的术语，用来描述这种大规模监视和修改人类行为以获取利润的经济体制 — — 更多详见 IYP 类目“
   <a href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89/" rel="noopener noreferrer" target="_blank">
    监视资本主义
   </a>
   ”。它涉及刮取数亿人的私密生活、选择、和行为的大数据集的预测分析，允许识别相关性和模式，推断和分析个体信息，以及预测目标个体或群体的未来行为和决策。
   <strong class="markup--strong markup--p-strong">
    然后，这些分析结论会通过极具针对性的和“动态”定向的宣传战来影响目标的行为。
   </strong>
  </p>
  <p class="graf graf--p">
   整个过程通过实验方法进行了改进 — — 在不同的人口统计数据上测试宣传的各种变化，以确定最有效的方法。每次我们登录时，我们都可能成为不知情的且因此不同意的小白鼠，
   <strong class="markup--strong markup--p-strong">
    操纵者的目的是确定如何最有效地从我们身上赚到钱或说服我们信任某些东西以利于他们的利益
   </strong>
   。
  </p>
  <p class="graf graf--p">
   隐秘宣传战和剑桥分析公司等秘密机构的共同点是隐蔽地使用强大的行为修正战术：psyops。
  </p>
  <p class="graf graf--p">
   我们的个人数据被用于构建所谓的“说服模版”，使用一组基于概率的估计 — — 关于特定影响策略对个人的有效性，这也是基于调查目标人过去对这些策略的反应所集成的大数据分析。其中一些公司也正在试验生物识别技术。
  </p>
  <p class="graf graf--p">
   很多人一直认为是“其他国家”试图干预英国的选举。事实上
   <strong class="markup--strong markup--p-strong">
    并不是这么简单的
   </strong>
   。使用大数据分析和心理操纵来定位社交媒体上的任何目标人以灌输政治内容的行为，已经产生了深远的政治影响，但它仍然在很大程度上没有得到重视。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    通过高度借助监视资本主义的设备和机制，政治竞选已经从公共过程转变成了私人的、个性化的微观监控操纵策略的网络。这个过程导致政府将公民作为达到其自身目的的手段 — — 只是维持权力，维护现状。
   </strong>
  </p>
  <p class="graf graf--p">
   在斯诺登于2013年揭露的
   <a class="markup--anchor markup--p-anchor" data-href="https://kittysjones.wordpress.com/2015/06/26/controversial-gchq-unit-engaged-in-domestic-law-enforcement-online-propaganda-psychology-research-glenn-greenwald-and-andrew-fishman/" href="https://kittysjones.wordpress.com/2015/06/26/controversial-gchq-unit-engaged-in-domestic-law-enforcement-online-propaganda-psychology-research-glenn-greenwald-and-andrew-fishman/" rel="noopener noreferrer" target="_blank">
    英国间谍头子 GCHQ 和美国间谍头子 NSA 的秘密活动
   </a>
   后，立即成为了被世界各地争议的头条新闻。GCHQ 的目标是汇总[大规模收集]网络上
   <a class="markup--anchor markup--p-anchor" data-href="https://arstechnica.com/information-technology/2015/09/gchq-tried-to-track-web-visits-of-every-visible-user-on-internet/" href="https://arstechnica.com/information-technology/2015/09/gchq-tried-to-track-web-visits-of-every-visible-user-on-internet/" rel="noopener noreferrer" target="_blank">
    每个用户的私人习惯
   </a>
   。 这项臭名昭著的调查权力法案，俗称
   <a class="markup--anchor markup--p-anchor" data-href="https://opendemocracy.net/digitaliberties/jane-duncan/britain-s-investigatory-powers-bill-gift-to-securocrats-everywhere" href="https://opendemocracy.net/digitaliberties/jane-duncan/britain-s-investigatory-powers-bill-gift-to-securocrats-everywhere" rel="noopener noreferrer" target="_blank">
    “监听器章程
   </a>
   ”，允许维稳部门和情报机构被法律授权获取来自技术公司的任何个人数据，并且，文件显示，英国政府寻求与美国达成协议，这些协议将给予英国情报机构更充分的权力以访问这些数据库。
  </p>
  <p class="graf graf--p">
   监控公司与国家安全和情报机构之间的数据共享已经建立。在美国，巨头科技公司已被迫将有关其用户的数据移交给 NSA。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2014/sep/11/yahoo-nsa-lawsuit-documents-fine-user-data-refusal" href="https://www.theguardian.com/world/2014/sep/11/yahoo-nsa-lawsuit-documents-fine-user-data-refusal" rel="noopener noreferrer" target="_blank">
    当雅虎拒绝时，他们每天都
   </a>
   被罚款25万美元，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2014/sep/11/yahoo-nsa-lawsuit-documents-fine-user-data-refusal" href="https://www.theguardian.com/world/2014/sep/11/yahoo-nsa-lawsuit-documents-fine-user-data-refusal" rel="noopener noreferrer" target="_blank">
    并且每周罚款加倍
   </a>
   ，他们的违规行为继续存在，面对金融危机的前景，他们默许了。
  </p>
  <p class="graf graf--p">
   显然，
   <strong class="markup--strong markup--p-strong">
    监视实践改变了公民与国家之间的关系，改变了权力平衡并扭曲了民主
   </strong>
   。
  </p>
  <p class="graf graf--p">
   私人公司或政府把公民变成巴甫洛夫的狗是不对的。这种针对性的心理说服策略严重破坏了人类自尊，自尊是人格独立性和民主的核心。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    F
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://politicsandinsights.org/2019/04/09/further-calls-for-inquiry-into-psyops-initiative-following-an-apology-for-smearing-jeremy-corbyn/" href="https://politicsandinsights.org/2019/04/09/further-calls-for-inquiry-into-psyops-initiative-following-an-apology-for-smearing-jeremy-corbyn/" rel="noopener noreferrer" target="_blank">
    urther calls for inquiry into psyops initiative following an apology for smearing Jeremy Corbyn
   </a>
   . “The involvement of someone like John Rendon is extremely concerning as this seems to be exactly the sort of thing that he specialises in.
  </p>
