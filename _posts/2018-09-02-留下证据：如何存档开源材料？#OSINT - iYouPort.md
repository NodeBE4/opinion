      </strong>

  <p class="graf graf--p graf-after--figure" id="f9aa">
   不论您是做在线调研，还是职业媒体，都存在一个证据收集的问题。您需要不停地保存您发现的材料。但如何在您发布调查之后用户删除了帖子、或者被审查者移除了内容，比如 YouTube 上，关于叙利亚战争罪行的视频
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2017/08/22/world/middleeast/syria-youtube-videos-isis.html" href="https://www.nytimes.com/2017/08/22/world/middleeast/syria-youtube-videos-isis.html" rel="noopener nofollow noreferrer" target="_blank">
    就会被谷歌移除
   </a>
   ，根据谷歌的所谓规矩，但是您依旧需要证实您的调查和报道中索引材料的真实性。
  </p>
  <p class="graf graf--p graf-after--p" id="f77d">
   存档您使用调查的所有数字证据的主要原因有两个：如果原始来源被删除了，您依然可以保留它，并向您的受众证明材料确实存在；屏幕截图可以很容易被伪造，那些用截图做证据的都不是专业人士，想做一个无可辩驳的专业人士您就必须找到一种方法来保留材料，以表明您没有机会修改内容，也就是它的可证实性。
  </p>
  <p class="graf graf--p graf-after--p" id="0c17">
   <strong class="markup--strong markup--p-strong">
    第三方存档平台
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="1357">
   对于大多数内容，包括社交媒体帖子，新闻报道和其他网页，通常有两种服务可用：
   <a class="markup--anchor markup--p-anchor" data-href="http://archive.today/" href="https://archive.today/" rel="noreferrer noopener nofollow" target="_blank">
    Archive.today
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="http://archive.org/" href="https://archive.org/" rel="noreferrer noopener nofollow" target="_blank">
    Archive.org
   </a>
   。这两个站点将网页保存在自己的服务器上，任何拥有 URL 的人都可以访问它们。更有利的是，这两个站点都会随着时间的推移保存页面的快照，因此您可以观察每次存档时的
   <strong class="markup--strong markup--p-strong">
    更改痕迹
   </strong>
   ，例如在编辑新闻文章之前和之后。
  </p>
  <p class="graf graf--p graf-after--p" id="366d">
   我们通常建议在两个站点上保存材料，以便最大化存档内容的数量。我们将总结每个网站的工作方式，以及它们在多个最受欢迎的社交网站上捕获网页的效果。一般来说，
   <a class="markup--anchor markup--p-anchor" data-href="http://archive.today/" href="https://archive.today/" rel="noreferrer noopener nofollow" target="_blank">
    Archive.today
   </a>
   在保存社交网络页面方面更加通用，因为它们通过为这些网站创建帐户以保存页面，同时存档。
  </p>
  <p class="graf graf--p graf-after--p" id="55f4">
   <a class="markup--anchor markup--p-anchor" data-href="http://archive.today/" href="https://archive.today/" rel="noreferrer noopener nofollow" target="_blank">
    Archive.today
   </a>
  </p>
  <p class="graf graf--p graf-after--p" id="eab2">
   在两个主要档案网站中，
   <a class="markup--anchor markup--p-anchor" data-href="http://archive.is/" href="https://archive.fo/" rel="noreferrer noopener nofollow" target="_blank">
    Archive.is
   </a>
   是功能最多、更友好的网站。然而，它几乎与
   <a class="markup--anchor markup--p-anchor" data-href="http://archive.org/" href="https://archive.org/" rel="noreferrer noopener nofollow" target="_blank">
    Archive.org
   </a>
   差不多，应该被视为不太稳定。此外，由于极端主义内容有时会通过
   <a class="markup--anchor markup--p-anchor" data-href="http://archive.today/" href="https://archive.today/" rel="noreferrer noopener nofollow" target="_blank">
    Archive.today
   </a>
   链接共享，因此该网站在许多国家/地区已被禁止。该站点有备用 URL（
   <a class="markup--anchor markup--p-anchor" data-href="http://archive.is/" href="https://archive.fo/" rel="noreferrer noopener nofollow" target="_blank">
    Archive.is
   </a>
   ，
   <a class="markup--anchor markup--p-anchor" data-href="http://archive.li/" href="https://archive.li/" rel="noreferrer noopener nofollow" target="_blank">
    Archive.li
   </a>
   ，
   <a class="markup--anchor markup--p-anchor" data-href="http://archive.ch/" href="http://archive.ch/" rel="noreferrer noopener nofollow" target="_blank">
    Archive.ch
   </a>
   …），
   <strong class="markup--strong markup--p-strong">
    可以让您绕过一些（但不是全部）国家的审查，例如俄罗斯、中国和芬兰。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="ea36">
   <a class="markup--anchor markup--p-anchor" data-href="http://archive.today/" href="https://archive.today/" rel="noreferrer noopener nofollow" target="_blank">
    Archive.today
   </a>
   上保存的页面完全来自用户提交的请求，而不是像
   <a class="markup--anchor markup--p-anchor" data-href="http://archive.org/" href="https://archive.org/" rel="noreferrer noopener nofollow" target="_blank">
    Archive.org
   </a>
   那样的自动检索。要在此站点上保存页面，只需在红色框中输入要保存的 URL 即可。
  </p>
  <p class="graf graf--p graf-after--p" id="7b20">
   您还可以通过将书签保存到浏览器的书签栏来存档页面，创建一键式路径以保存当前所在页面的快照。要执行此操作，请使用以下 URL 在书签（或收藏夹）栏中保存新页面：javascript:void(open(‘
   <a class="markup--anchor markup--p-anchor" data-href="https://archive.today/?run=1&amp;url=%E2%80%99+encodeURIComponent(document.location)" href="https://archive.today/?run=1&amp;url=%E2%80%99+encodeURIComponent%28document.location%29" rel="noreferrer noopener nofollow" target="_blank">
    https://archive.today/?run=1&amp;url=’+encodeURIComponent(document.location)
   </a>
   ))
  </p>
  <p class="graf graf--p graf-after--p" id="b13d">
   只需单击新创建的书签，即可在浏览器的选项卡中保存您打开的任何页面。
  </p>
  <p class="graf graf--p graf-after--p" id="703d">
   或者，您可以单击
   <a class="markup--anchor markup--p-anchor" data-href="http://archive.today/" href="https://archive.today/" rel="noreferrer noopener nofollow" target="_blank">
    Archive.today
   </a>
   首页上的按钮并将其拖动到书签栏，从而无需手动创建书签。
  </p>

     <strong class="markup--strong markup--p-strong">
      保存照片和视频
     </strong>
    </p>
    <p class="graf graf--p graf-after--p" id="971a">
     很明显
     <a class="markup--anchor markup--p-anchor" data-href="http://archive.org/" href="https://archive.org/" rel="noreferrer noopener nofollow" target="_blank">
      Archive.org
     </a>
     和
     <a class="markup--anchor markup--p-anchor" data-href="http://archive.today/" href="https://archive.today/" rel="noreferrer noopener nofollow" target="_blank">
      Archive.today
     </a>
     都无法保存来自 Instagram 和 YouTube 的照片和视频，并且在从 Facebook，VK 和其他网站保存照片方面存在问题。对于这些网站，创建一个托管媒体的第三方“中立”平台要困难得多。相反，
     <strong class="markup--strong markup--p-strong">
      我们需要单独下载这些材料，然后提供补充材料（例如显示元数据的屏幕截图，材料的镜像版本等），以证明图像和视频是真实的
     </strong>
     。
    </p>
    <p class="graf graf--p graf-after--p" id="f7bd">
     YouTube
    </p>
    <p class="graf graf--p graf-after--p" id="d69d">
     有很多网站可以从 YouTube 中提取视频，例如 KeepVid，Y2Mate 和其他网站。只要您有足够的硬盘或云存储空间来存储来自 YouTube 的视频并不困难。
     <strong class="markup--strong markup--p-strong">
      请务必截取元数据并将页面保存在Archive.today上，以便保留标题、上传日期和说明，即使视频未保存在页面上也是如此
     </strong>
     。
    </p>

     Instagram
    </p>

    <img alt="" class="aligncenter size-full wp-image-4863" data-recalc-dims="1" height="970" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-89.jpeg?resize=1100%2C970&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-89.jpeg?w=1619&amp;ssl=1 1619w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-89.jpeg?resize=300%2C264&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-89.jpeg?resize=768%2C677&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-89.jpeg?resize=1024%2C903&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-89.jpeg?resize=1100%2C970&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="a0e1">
   从 Odnoklassniki 获取视频的网站数量少于其他社交网络，但还是可能的，例如
   <a class="markup--anchor markup--p-anchor" data-href="http://video-download.co/" href="http://video-download.co/" rel="noreferrer noopener nofollow" target="_blank">
    Video-Download.co
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="41f3">
   <strong class="markup--strong markup--p-strong">
    其他归档解决方案
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="4a15">
   通常，您不能使用之前讨论过的服务来下载网页或视频，因为它们落后于隐私设置（限制对
   <a class="markup--anchor markup--p-anchor" data-href="http://archive.today/" href="https://archive.today/" rel="noreferrer noopener nofollow" target="_blank">
    Archive.today
   </a>
   这样的网站的访问），或者他们使用像 KeepVid 这样的网站无法吸引的模糊视频播放平台。本指南前面提到的所有解决方案都是免费的; 但是，还有一些其他服务需要一些付款或试用期限制。
  </p>
  <p class="graf graf--p graf-after--p" id="2ae5">
   有些软件解决方案可以从大多数视频网站中获取材料，即使他们不使用 YouTube 或其他流行平台。尽管需要付费才能完全使用，但 Apowersoft 的视频下载捕获功能对于几乎所有嵌入式视频、包括直播视频，效果都非常出色。该软件能够检测您的浏览器中正在播放的视频，然后（通常）能够从其原始源下载它。如果您有想要下载的特定视频但无法找到任何其他解决方案，则可能需要尝试免费试用此软件。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4864 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-90.jpeg?resize=1100%2C684&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-90.jpeg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-90.jpeg?resize=300%2C187&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-90.jpeg?resize=768%2C478&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-90.jpeg?resize=1024%2C637&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-90.jpeg?resize=1100%2C684&amp;ssl=1 1100w" data-recalc-dims="1" height="684" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-90.jpeg?resize=1100%2C684&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4864" data-recalc-dims="1" height="684" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-90.jpeg?resize=1100%2C684&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-90.jpeg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-90.jpeg?resize=300%2C187&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-90.jpeg?resize=768%2C478&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-90.jpeg?resize=1024%2C637&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-90.jpeg?resize=1100%2C684&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p>
   对于躲在隐私设置背后的网页，很难找到任何创建该网站的受信任的第三方存档副本的解决方案。将页面保存为 HTML 格式是非常混乱的。另一种解决方案可能是将页面保存为 PDF 文件，方法是将其打印为 PDF（文件 →打印 →打印到 PDF），或者使用 Adobe Create 将网页转换为 PDF。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4865 jetpack-lazy-image" data-lazy-sizes="(max-width: 469px) 100vw, 469px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-91.jpeg?resize=469%2C608&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-91.jpeg?w=469&amp;ssl=1 469w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-91.jpeg?resize=231%2C300&amp;ssl=1 231w" data-recalc-dims="1" height="608" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-91.jpeg?resize=469%2C608&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="469"/>

   <p class="graf graf--p graf-after--figure" id="3e05">
    也就是说，仍然可以在 PDF 中修改这些页面的内容。目前，显示隐私锁定页面内容的最可信但仍然不完美的方式是，在访问页面时记录您的屏幕（请参阅
    <a class="markup--anchor markup--p-anchor" data-href="https://www.techradar.com/how-to/how-to-record-your-pcs-screen" href="https://www.techradar.com/how-to/how-to-record-your-pcs-screen" rel="noopener nofollow noreferrer" target="_blank">
