 <div class="entry-content">
  <ul>
   <li class="graf graf--p">
    <span style="color: #00ccff;">
     <em>
      <strong>
       曾经很多人在询问：究竟有没有说真话的媒体？当然有。优秀的独立媒体和替代媒体是美国的荣耀。但是，人们可能看不到它们，而且越来越难以看到了。为什么？
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   依赖主流搜索引擎和主流社交媒体为认知基础设施的人们可能正在面临被更深层操纵的前景。当然，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" rel="noopener noreferrer" target="_blank">
    操纵从来存在
   </a>
   ，因为可见度算法一直都是黑箱；但这一问题现在似乎正在变得更加严重，尤其是，在“恢复主流媒体公信力”的大旗下，一场针对异议的大清洗正在开启……
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5873 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/算法暴政.png?resize=1100%2C612&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/算法暴政.png?w=1258&amp;ssl=1 1258w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/算法暴政.png?resize=300%2C167&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/算法暴政.png?resize=768%2C427&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/算法暴政.png?resize=1024%2C570&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/算法暴政.png?resize=1100%2C612&amp;ssl=1 1100w" data-recalc-dims="1" height="612" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/算法暴政.png?resize=1100%2C612&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5873" data-recalc-dims="1" height="612" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/算法暴政.png?resize=1100%2C612&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/算法暴政.png?w=1258&amp;ssl=1 1258w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/算法暴政.png?resize=300%2C167&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/算法暴政.png?resize=768%2C427&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/算法暴政.png?resize=1024%2C570&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/算法暴政.png?resize=1100%2C612&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   由一群著名新保守派人士支持的新闻评级系统 Newsguard 失败之后，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.mintpressnews.com/newsguard-european-union/254453/" href="https://www.mintpressnews.com/newsguard-european-union/254453/" rel="noopener noreferrer" target="_blank">
    为了吸引科技寡头
   </a>
   ，另一个组织悄悄介入了，指导谷歌、Facebook、微软等科技巨头的新闻算法。
  </p>
  <p class="graf graf--p">
   虽然与 Newsguard 不同，但这个被称为“Trust Project”的组织也有类似的目标，即
   <strong class="markup--strong markup--p-strong">
    恢复企业媒体的“公信力”，相对于独立的替代方案，该项目以明显不透明的方式将“信任指标”应用于社交媒体新闻算法
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “Trust Project”的资金 —— 主要来自谷歌等大型科技公司；与政府密切相关的技术寡头，如皮埃尔奥米迪亚；正如 Newsguard 的关键投资者奈特基金会（Knight Foundation）所说，不懈地推广“传统”主流媒体，限制其他媒体的成功。
  </p>
  <p class="graf graf--p">
   特别重要的是，
   <strong class="markup--strong markup--p-strong">
    Trust Project 的“信任指标”已被用于控制由谷歌和必应等顶级搜索引擎以及像 Facebook 这样的大型社交媒体网络的算法
   </strong>
   —— 关于推动哪些内容、及压制哪些内容，也就是说，大多数人的认知来源。
  </p>
  <p class="graf graf--p">
   尽管这些“信任指标”（其中八个目前正在使用）的描述是公开的，但主要技术寡头和社交媒体巨头使用它们的方式
   <strong class="markup--strong markup--p-strong">
    完全不透明
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Trust Project 的目标是增加公众对传统媒体渠道的信任，以牺牲独立的替代新闻来源为代价。
  </p>
  <p class="graf graf--p">
   即使其在建立媒体中促进“信任”的努力失败，其隐藏在参与新闻网站内的嵌入式代码也允许这些网点压过其独立媒体的竞争对手，使这种“信任”问题在很大程度上令人生疑，因为
   <strong class="markup--strong markup--p-strong">
    它使网络媒体格局同质化，不成比例地有利于主流媒体。
   </strong>
  </p>
  <p class="graf graf--p">
   Trust Project 的主管 Sally Lehrman
   <a class="markup--anchor markup--p-anchor" data-href="https://www.scu.edu/ethics/the-democracy-project/what-people-really-want-from-news-organizations/" href="https://www.scu.edu/ethics/the-democracy-project/what-people-really-want-from-news-organizations/" rel="noopener noreferrer" target="_blank">
    明确表示
   </a>
   ，在她看来，主流媒体缺乏公信力及其读者人数不断减少的状况是由于无意义的“竞争”进一步破坏了主流媒体作为引擎的角色和目的。”
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5874 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-45.jpg?resize=1100%2C618&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-45.jpg?w=1400&amp;ssl=1 1400w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-45.jpg?resize=300%2C168&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-45.jpg?resize=768%2C431&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-45.jpg?resize=1024%2C575&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-45.jpg?resize=1100%2C618&amp;ssl=1 1100w" data-recalc-dims="1" height="618" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-45.jpg?resize=1100%2C618&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5874" data-recalc-dims="1" height="618" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-45.jpg?resize=1100%2C618&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-45.jpg?w=1400&amp;ssl=1 1400w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-45.jpg?resize=300%2C168&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-45.jpg?resize=768%2C431&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-45.jpg?resize=1024%2C575&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-45.jpg?resize=1100%2C618&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <h3 class="graf graf--p">
   <span style="font-family: impact, sans-serif;">
    <strong class="markup--strong markup--p-strong">
     了解 Trust Project
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   Trust Project
   <a class="markup--anchor markup--p-anchor" data-href="https://thetrustproject.org/" href="https://thetrustproject.org/" rel="noopener noreferrer" target="_blank">
    将自己描述为
   </a>
   “一个由顶级新闻公司组成的联盟”，参与制定“透明度标准，帮助您轻松评估新闻的质量和可信度”。
  </p>
  <p class="graf graf--p">
   它通过创建所谓的“信任指标”来实现这一目标，该项目的网站将其描述为“满足人们需求的数字标准”。
  </p>
  <p class="graf graf--p">
   然而，Trust Project 似乎远非满足“人们的需求”，而是旨在操纵搜索引擎和社交媒体新闻算法，使该项目的媒体合作伙伴受益，而不是为了公众的利益。
  </p>
  <p class="graf graf--p">
   该项目的起源
   <a class="markup--anchor markup--p-anchor" data-href="https://thetrustproject.org/faq/#behind_idea" href="https://thetrustproject.org/faq/#behind_idea" rel="noopener noreferrer" target="_blank">
    可以追溯到
   </a>
   由圣克拉拉大学 Markkula 应用伦理中心主办的2012年“圆桌会议”，
   <strong class="markup--strong markup--p-strong">
    该中心由前苹果公司首席执行官 Mike Markkula 资助
   </strong>
   。
  </p>
  <p class="graf graf--p">
   该圆桌会议由 Sally Lehrman 创建，他
   <a class="markup--anchor markup--p-anchor" data-href="https://www.niemanlab.org/2017/11/the-trust-project-brings-news-orgs-and-tech-giants-together-to-tag-and-surface-high-quality-news/" href="https://www.niemanlab.org/2017/11/the-trust-project-brings-news-orgs-and-tech-giants-together-to-tag-and-surface-high-quality-news/" rel="noopener noreferrer" target="_blank">
    明确表示
   </a>
   ，Trust Project 仅对“遵守传统标准的新闻机构”开放。
  </p>
  <p class="graf graf--p">
   挑战权势的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ae%a9%e5%ae%83%e6%b0%91%e4%b8%bb%ef%bc%9a%e6%9b%bf%e4%bb%a3%e5%aa%92%e4%bd%93%e4%bd%9c%e4%b8%ba%e6%8a%97%e4%ba%89%e4%b9%8b%e6%ba%90%ef%bc%88%e4%b8%8b%ef%bc%89/" href="https://www.iyouport.org/%e8%ae%a9%e5%ae%83%e6%b0%91%e4%b8%bb%ef%bc%9a%e6%9b%bf%e4%bb%a3%e5%aa%92%e4%bd%93%e4%bd%9c%e4%b8%ba%e6%8a%97%e4%ba%89%e4%b9%8b%e6%ba%90%ef%bc%88%e4%b8%8b%ef%bc%89/" rel="noopener noreferrer" target="_blank">
    替代媒体
   </a>
   和独立媒体是不行的。
  </p>
  <p class="graf graf--p">
   推动 Trust Project 创建的具体想法诞生于2014年圆桌会议上，当时 Lehrman “在 Twitter 上询问机器学习专家、谷歌新闻负责人 Richard Gingras，算法是否可以用于支持道德，他们说是的。Gingras 同意合作。”
  </p>
  <p class="graf graf--p">
   换句话说，Trust Project 背后的想法
   <strong class="markup--strong markup--p-strong">
    从一开始就是依靠谷歌和 Twitter 等主要科技公司操弄搜索引擎和社交媒体的不透明算法
   </strong>
   。
  </p>
  <figure aria-describedby="caption-attachment-5875" class="wp-caption aligncenter" id="attachment_5875" style="width: 1400px">
   <img alt="" class="wp-image-5875 size-full jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-8.jpg?resize=1100%2C556&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-8.jpg?w=1400&amp;ssl=1 1400w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-8.jpg?resize=300%2C152&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-8.jpg?resize=768%2C388&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-8.jpg?resize=1024%2C517&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-8.jpg?resize=1100%2C556&amp;ssl=1 1100w" data-recalc-dims="1" height="556" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-8.jpg?resize=1100%2C556&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

    Sally Lehrman discusses the Trust Project at 2018 WordCamp For Publishers
   </figcaption>
  </figure>
  <p class="graf graf--p">
   正如 Trust Project
   <a class="markup--anchor markup--p-anchor" data-href="https://thetrustproject.org/tech-development-and-ux-working-groups/" href="https://thetrustproject.org/tech-development-and-ux-working-groups/" rel="noopener noreferrer" target="_blank">
    本身所指出
   </a>
   的那样，改变算法的方法是与 Gingras 等技术巨头和“80家新闻媒体和机构的行业顶级编辑”共同开发的，
   <strong class="markup--strong markup--p-strong">
    所有这些都是企业媒体。没有独立媒体。
   </strong>
  </p>
  <p class="graf graf--p">
   值得注意的是，Trust Project 的媒体合作伙伴 —— 即
   <a class="markup--anchor markup--p-anchor" data-href="https://thetrustproject.org/tech-development-and-ux-working-groups/" href="https://thetrustproject.org/tech-development-and-ux-working-groups/" rel="noopener noreferrer" target="_blank">
    参与为新闻算法创建这些新“标准”
   </a>
   的人，
   <strong class="markup--strong markup--p-strong">
    基本都是富有的寡头拥有的主要出版物
   </strong>
   ：华盛顿邮报，由世界首富 Jeff Bezos 拥有；经济学人，由富有的罗斯柴尔德家族拥有；还有加拿大最富有的家族汤姆逊（Thomsons）拥有的环球邮报（Globe and Mail）和 汤森路透（Thomson Reuters）。
  </p>
  <p class="graf graf--p">
   其他合作伙伴还包括，纽约时报、Mic、 Hearst 电视台、BBC 和 USA Today。
  </p>
  <p class="graf graf--p">
   其他主要媒体在 Markkula 中心的
   <a class="markup--anchor markup--p-anchor" data-href="https://web.archive.org/web/20190506003600/https://www.scu.edu/ethics/focus-areas/journalism-ethics/programs/the-trust-project/news-leadership-council/" href="https://web.archive.org/web/20190506003600/https://www.scu.edu/ethics/focus-areas/journalism-ethics/programs/the-trust-project/news-leadership-council/" rel="noopener noreferrer" target="_blank">
    新闻领导评议会
   </a>
   中有代表，包括“金融时报”、“Gizmodo Media”、和“华尔街日报”。
  </p>
  <p class="graf graf--p">
   该评议会 —— 其中还包括 Gingras 和 Facebook 的产品管理总监 Andrew Anker —— 被称为“指导我们的 Trust Project 信任指标。”
  </p>
  <p class="graf graf--p">
   这些“信任指标”是 Trust Project 活动的核心，揭示了谷歌、Twitter 和 Facebook 改变其算法以支持具有良好“信任指标”得分的网点的关键机制之一。
  </p>
  <p class="graf graf--p">
   从表面上看，Trust Project 旨在使新闻出版物“更加透明”，以此作为增强公信力的手段。尽管共开发了37种，但目前只有8种正在使用中。
  </p>
  <p>

  <h3 class="graf graf--p">
   <span style="font-family: impact, sans-serif;">
    <strong class="markup--strong markup--p-strong">
     信任指标操纵技术寡头的算法
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   Trust Project 的真正力量是以一种公众不可见的形式出现的。
  </p>
  <p class="graf graf--p">
   这些信任指标虽然偶尔显示在合作伙伴网站上，但也与嵌入在 Facebook、Google、Bing 和 Twitter 使用的参与网站和文章的 HTML 代码中的“
   <a class="markup--anchor markup--p-anchor" data-href="https://thetrustproject.org/news18/" href="https://thetrustproject.org/news18/" rel="noopener noreferrer" target="_blank">
    机器可读信号
   </a>
   ”相结合。
  </p>
  <p class="graf graf--p">
   正如 Lehrman 在2017年的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.scu.edu/ethics/the-democracy-project/what-people-really-want-from-news-organizations/" href="https://www.scu.edu/ethics/the-democracy-project/what-people-really-want-from-news-organizations/" rel="noopener noreferrer" target="_blank">
    一篇文章中
   </a>
   指出的那样，Trust Project 当时“已经与这四家科技寡头公司合作，所有这些公司都表示他们希望利用我们的指标来优先处理报道良好的新闻。”
  </p>
  <p class="graf graf--p">
   谷歌新闻的 Gingras
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@richardgingras/news-then-news-now-journalism-in-a-digital-age-5fefd9ccd574" href="https://medium.com/@richardgingras/news-then-news-now-journalism-in-a-digital-age-5fefd9ccd574" rel="noopener noreferrer" target="_blank">
    还指出
   </a>
   ，谷歌使用信任指标作为
   <strong class="markup--strong markup--p-strong">
    “帮助搜索引擎更好地理解和排名结果的线索
   </strong>
   …… 以帮助模拟媒体生活的无数算法系统。”
  </p>
  <p class="graf graf--p">
   去年 Trust Project 的新闻稿进一步强调了嵌入式“指标”改变社交媒体和搜索引擎算法的重要性：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “虽然每个指标在项目新闻合作伙伴的页面上对用户可见，但它也嵌入在文章和站点代码中供机器阅读 —— 标准化技术语言，提供有关新闻网站对透明度承诺的相关信息”……
   </em>
  </p>
  <p class="graf graf--p">
   尽管声称要增加公众对“新闻网站对透明度的承诺”的了解，但谷歌和 Facebook 等主要科技寡头使用这些指标的方式并不透明。很难知道这些指标是如何使用的。
  </p>
  <p class="graf graf--p">
   例如，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cbsnews.com/news/fake-news-not-if-it-has-publishers-trust-indicators/" href="https://www.cbsnews.com/news/fake-news-not-if-it-has-publishers-trust-indicators/" rel="noopener noreferrer" target="_blank">
    CBS 新闻援引
   </a>
   Craig Newmark —— Craigslist 的创始人，他提供了 Trust Project 的种子资金 —— 建议“谷歌的搜索算法可以通过使用项目的信任指标将可信来源在搜索结果中排在其他网站前面”。
  </p>
  <p class="graf graf--p">
   请注意，
   <strong class="markup--strong markup--p-strong">
    这些信任指标的机器可读版本仅供参与机构使用，这些机构就是企业媒体。
   </strong>
  </p>
  <p class="graf graf--p">
   尽管正在开发 WordPress 和 Drupal 插件，以便向小型发布商提供搜索引擎和社交媒体的嵌入式信号，但是，它
   <a class="markup--anchor markup--p-anchor" data-href="https://www.niemanlab.org/2017/11/the-trust-project-brings-news-orgs-and-tech-giants-together-to-tag-and-surface-high-quality-news/" href="https://www.niemanlab.org/2017/11/the-trust-project-brings-news-orgs-and-tech-giants-together-to-tag-and-surface-high-quality-news/" rel="noopener noreferrer" target="_blank">
    只能提供给
   </a>
   “合格的发布者”，也就是说依旧被 Trust Project 和它的同事们所掌握。
  </p>
  <p class="graf graf--p">
   Richard Gingras 在2017年的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.niemanlab.org/2017/11/the-trust-project-brings-news-orgs-and-tech-giants-together-to-tag-and-surface-high-quality-news/" href="https://www.niemanlab.org/2017/11/the-trust-project-brings-news-orgs-and-tech-giants-together-to-tag-and-surface-high-quality-news/" rel="noopener noreferrer" target="_blank">
    一份声明
   </a>
   中指出，“这些指标可以帮助我们的算法更好地理解权威新闻 —— 并帮助我们更好地向消费者展示权威。”
  </p>
  <p class="graf graf--p">
   因此，非常清楚的是，这些仅包含在“合格”和“权威”新闻网站中的指标将用于倾斜搜索引擎和社交媒体新闻算法，以
   <strong class="markup--strong markup--p-strong">
    偏重体制内新闻网站
   </strong>
   。
  </p>
  <p class="graf graf--p">
   最重要的是，这些嵌入式和独家指标允许某些新闻媒体避免近期算法变化的破坏性影响，包括 MintPress 在内的许多新闻网站的流量近年来一直在急剧下降。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    通过使独立媒体竞争者挨饿，而实现在线新闻环境的同质化，只有能与 Trust Project 合作的企业媒体才有出路
   </strong>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="font-family: impact, sans-serif;">
    <strong class="markup--strong markup--p-strong">
     Trust Project 背后的技术亿万富翁
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   鉴于 Trust Project 对在线新闻领域的影响越来越丰富，发现
   <strong class="markup--strong markup--p-strong">
    它的资金来自于富人和政府、以及在线控制新闻和信息流动的明显受益者
   </strong>
   ，这并不奇怪。
  </p>
  <p class="graf graf--p">
   根据其网站，该项目目前获得了 Craig Newmark 慈善事业、谷歌、Facebook、eBay创始人皮埃尔奥米迪亚的民主基金、奈特基金会和马库拉基金会的
   <a class="markup--anchor markup--p-anchor" data-href="https://thetrustproject.org/faq/#funding" href="https://thetrustproject.org/faq/#funding" rel="noopener noreferrer" target="_blank">
    资助
   </a>
   。
  </p>
  <p class="graf graf--p">
   该项目的网站还指出谷歌是“早期的财政支持者”，最初由 Craigslist 的创始人Craig Newmark资助。如前所述，
   <strong class="markup--strong markup--p-strong">
    Trust Project 的联合创始人是现任 Google 新闻副总裁 Richard Gingras。
   </strong>
  </p>
  <p class="graf graf--p">
   Trust Project 的网站描述了 Gingras 目前在该组织中扮演“强大的传播者”的角色；Newmark 目前在该项目中的角色被描述为“资助者和有价值的连接者”。

  <p class="graf graf--p">

  <p class="graf graf--p">
   Omidyar 的民主基金会为 Kristol 提供财政支持，并支持 Kristol 的 ASD，这是德国马歇尔基金会智囊团的一个项目，以其神秘的 Hamilton68 而闻名。（在这里看到 Hamilton68 是什么东西《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E2%9D%97%EF%B8%8F%E4%BC%AA%E5%8A%A3%E6%8A%A5%E9%81%93%E5%92%8C%E6%9C%89%E7%9B%AE%E7%9A%84%E7%9A%84%E6%94%BF%E6%B2%BB%E5%AE%A3%E4%BC%A0%E5%A6%82%E4%BD%95%E6%89%AD%E6%9B%B2%E4%BA%86%E7%8E%B0%E5%AE%9E/" href="https://www.iyouport.org/%E2%9D%97%EF%B8%8F%E4%BC%AA%E5%8A%A3%E6%8A%A5%E9%81%93%E5%92%8C%E6%9C%89%E7%9B%AE%E7%9A%84%E7%9A%84%E6%94%BF%E6%B2%BB%E5%AE%A3%E4%BC%A0%E5%A6%82%E4%BD%95%E6%89%AD%E6%9B%B2%E4%BA%86%E7%8E%B0%E5%AE%9E/" rel="noopener noreferrer" target="_blank">
    伪劣报道和有目的的政治宣传如何扭曲了现实政治
   </a>
   》）
  </p>
  <p class="graf graf--p">
   并且，Kristol 的新出版物 The Bulwark 的联合创始人兼编辑
   <a class="markup--anchor markup--p-anchor" data-href="https://thebulwark.com/author/charlessykes/" href="https://thebulwark.com/author/charlessykes/" rel="noopener noreferrer" target="_blank">
    Charles Sykes
   </a>
   是该基金的国家咨询委员会成员。
  </p>
  <p class="graf graf--p">
   在德国马歇尔基金会工作的 Kristol 的助手 Jamie Fly
   <a class="markup--anchor markup--p-anchor" data-href="https://www.mintpressnews.com/facebook-censorship-of-alternative-media-just-the-beginning-says-top-neocon-insider/250967/" href="https://www.mintpressnews.com/facebook-censorship-of-alternative-media-just-the-beginning-says-top-neocon-insider/250967/" rel="noopener noreferrer" target="_blank">
    去年10月表示
   </a>
   ，他们对以批评帝国和警察暴力而闻名的独立媒体的清洗只是“刚刚开始”。
  </p>
  <p class="graf graf--p">
   Omidyar 也是有争议的反诽谤联盟（ADL）努力成为硅谷科技巨头审查网点关键驱动因素的主要支持者。
  </p>
  <p class="graf graf--p">
   ADL 最初成立的目的是“阻止对犹太人民的诽谤”，但批评人士说，多年来它已经开始将以色列政府的批评者称为“反犹”。
  </p>
  <p class="graf graf--p">
   鉴于 Trust Project 与 ADL 共享一个关键资助者（Pierre Omidyar）和多个外部技术合作伙伴这一事实，谷歌和 Facebook 等主要科技公司如何在其算法中使用信任指标、以及 ADL 对这些相同算法的影响之间是否存在重叠，仍有待观察。
  </p>
  <p class="graf graf--p">
   然而，很清楚的是，由于为 Trust Project 提供种子基金并继续为其提供资金的 Craig Newmark 也是 ADL 的
   <a class="markup--anchor markup--p-anchor" data-href="https://morristowngreen.com/2019/04/08/craigslist-founder-giving-away-fortune-with-jewish-values-honed-in-morristown-as-moral-compass/" href="https://morristowngreen.com/2019/04/08/craigslist-founder-giving-away-fortune-with-jewish-values-honed-in-morristown-as-moral-compass/" rel="noopener noreferrer" target="_blank">
    主要捐助者和顾问
   </a>
   ，因此存在无可否认的重叠。
  </p>
  <p class="graf graf--p">
   2017年，Newmark 向 ADL 的事件响应中心捐赠了10万美元，并且是该集团技术顾问委员会的成员。
  </p>
  <h3 class="graf graf--p">
   <span style="font-family: impact, sans-serif;">
    <strong class="markup--strong markup--p-strong">
     外包审查
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   当然，Trust Project 中最有趣和令人不安的捐赠者是谷歌和 Facebook，他们都使用他们资助的项目作为“第三方”来证明他们操纵新闻源和搜索引擎算法的合理性。
  </p>
  <p class="graf graf--p">
   Facebook 在改变算法方面的其他合作伙伴包括大西洋理事会 —— 由美国政府、北约和武器制造商等资助 —— 而且 Facebook 也直接与外国政府合作，
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2016/09/12/facebook-is-collaborating-with-the-israeli-government-to-determine-what-should-be-censored/" href="https://theintercept.com/2016/09/12/facebook-is-collaborating-with-the-israeli-government-to-determine-what-should-be-censored/" rel="noopener noreferrer" target="_blank">
    如以色列政府
   </a>
   ，以压制有关政府希望从社交媒体平台中删除的真实信息。
  </p>
  <p class="graf graf--p">
   Trust Project 特别突出了“私人”审查制度、技术寡头审查制度、和政府审查制度之间的模糊性。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%E8%B0%B7%E6%AD%8C%E7%9A%84%E7%9C%9F%E9%9D%A2%E7%9B%AE%e2%80%8a-%e2%80%8a-theyre-always-watching/" href="https://www.iyouport.org/%E8%B0%B7%E6%AD%8C%E7%9A%84%E7%9C%9F%E9%9D%A2%E7%9B%AE%e2%80%8a-%e2%80%8a-theyre-always-watching/" rel="noopener noreferrer" target="_blank">
     谷歌的政治背景
    </a>
    》、《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%E6%B7%B1%E5%B1%82%E6%94%BF%E6%B2%BB%EF%BC%9Afacebook%E9%87%91%E5%AD%97%E5%A1%94%E7%9A%84%E4%BA%BA%E5%BD%A2%E8%82%8C%E7%90%86-%E4%BB%85%E4%BB%85%E4%BE%9D%E9%9D%A0%E5%BC%80%E6%BA%90/" href="https://www.iyouport.org/%E6%B7%B1%E5%B1%82%E6%94%BF%E6%B2%BB%EF%BC%9Afacebook%E9%87%91%E5%AD%97%E5%A1%94%E7%9A%84%E4%BA%BA%E5%BD%A2%E8%82%8C%E7%90%86-%E4%BB%85%E4%BB%85%E4%BE%9D%E9%9D%A0%E5%BC%80%E6%BA%90/" rel="noopener noreferrer" target="_blank">
     Facebook 的政治背景
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Pierre Omidyar 也一样。他与军事工业综合体和情报部门
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.mintpressnews.com/ebay-founder-pierre-omidyar-is-funding-a-global-media-information-war/255199/" href="https://www.mintpressnews.com/ebay-founder-pierre-omidyar-is-funding-a-global-media-information-war/255199/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     关系密切
    </strong>
   </a>
   。Bezos 的公司亚马逊则是全球最大的间谍机构的主要
   <a class="markup--anchor markup--p-anchor" data-href="https://dailycaller.com/2018/08/20/amazon-government-contracting/" href="https://dailycaller.com/2018/08/20/amazon-government-contracting/" rel="noopener noreferrer" target="_blank">
    承包商
   </a>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种与科技寡头、主流媒体和政府不断增长的关系表明，这些实体的利益越来越相似和互补，其中包括对独立记者和新闻媒体的审查
   </strong>
   ，这些新闻媒体一直在试图挑战主流权力和叙述。
  </p>
  <p class="graf graf--p">
   Trust Project 的创建是一种外包审查独立新闻网站的方式，同时试图挽救主流媒体破碎的声誉，令其重新主导叙事
  </p>
  <p class="graf graf--p">
   Trust Project 在参与网站中开发的隐藏算法改变代码表明，其真正的目标不是提高公信力，而是为硅谷审查独立媒体渠道提供独特的外观，这些独立媒体对权力说真话。⚪️
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.mintpressnews.com/the-trust-project-big-media-and-silicon-valleys-weaponized-algorithms-silence-dissent/259030/" href="https://www.mintpressnews.com/the-trust-project-big-media-and-silicon-valleys-weaponized-algorithms-silence-dissent/259030/" rel="noopener noreferrer" target="_blank">
    The Trust Project: Big Media and Silicon Valley’s Weaponized Algorithms Silence Dissent

  </p>
