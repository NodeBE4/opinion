  <h3 class="graf graf--h3">
   <strong>
    — — 互联网霸主 — —
   </strong>
  </h3>
  <p class="graf graf--p">
   从20世纪70年代开始，“信息自由流通”的论调就一直是美国外交政策的核心原则。在非殖民化和冷战时代，这一学说声称是一道“闪亮的灯塔”，照亮了全世界从帝国主义和国家镇压中解放出来的道路。
  </p>
  <p class="graf graf--p">
   今天，它继续以一种吸引人的普世人权语言描绘出根深蒂固的经济和战略利益。“互联网自由”、“连接自由”、“网络自由” — — 希拉里克林顿和谷歌高管在 WCIT 筹备期间共同发布的条款 — — 至今都是“free flow”规则的基本表述版本。
  </p>
  <p class="graf graf--p">
   但是，就像以前一样，“互联网自由”一直是个红鲱鱼（详见《
   <a href="https://www.iyouport.org/%E5%AE%A3%E4%BC%A0%E6%88%98%E7%9A%84%E8%AF%A1%E8%BF%B9/" rel="noopener noreferrer" target="_blank">
    宣传战的诡迹
   </a>
   》） — —
   <strong class="markup--strong markup--p-strong">
    指的是 以修辞或文学的手法转移议题焦点以牵制大众注意力的手段。
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://mondediplo.com/2013/02/15internet" href="https://mondediplo.com/2013/02/15internet" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     它指导我们将基本的人权委托给一对强大的和自私的社会行动者：公司和国家
    </strong>
   </a>
   。
  </p>
  <h3 class="graf graf--h3">
   <strong>
    — — 监视国家的养成，1898–2020 — —
   </strong>
  </h3>
  <p class="graf graf--p">
   监视国家的形成其实跨度很长，1898年开始：如今，它是一个无所不在的现实，但它悠久的历史却鲜为人知，其未来更是很少能被掌握。
  </p>
  <p class="graf graf--p">
   斯诺登泄露的文件显示，在9/11事件后的状态下，国家安全局（NSA）能够建立一个庞大的监视系统，可以秘密监视几乎每个美国人的私人通信。虽然使用的技术是最先进的；但事实证明，这种大规模间谍行为
   <strong class="markup--strong markup--p-strong">
    并不是什么新鲜事
   </strong>
   。
  </p>
  <p class="graf graf--p">
   一个多世纪以来，美国战争所谓的
   <strong class="markup--strong markup--p-strong">
    “监视反击”战略
   </strong>
   一直在确保更大规模的无所不在的全方位监视系统的建立。它的未来（虽然不是我们的）看起来确实很光明。[
   <a class="markup--anchor markup--p-anchor" data-href="http://www.tomdispatch.com/blog/175724/alfred_mccoy_surveillance_blowback" href="http://www.tomdispatch.com/blog/175724/alfred_mccoy_surveillance_blowback" rel="noopener noreferrer" target="_blank">
    Surveillance Blowback: The Making of the U.S. Surveillance State, 1898–2020
   </a>
   ]

  <p class="graf graf--p">
   John Young — — 这是透明度革命的激进派前辈之一，在他1999年10月发布的 NSA FOIA 中展示了八份与 tempest 相关的文件。文档中的打印效果很差，显示是手动打印的。最后一份文件显示，美国政府保留了
   <strong class="markup--strong markup--p-strong">
    一份其认为美国有能力和动力对其发动袭击的国家名单

   <a class="markup--anchor markup--p-anchor" data-href="http://cryptome.org/nsa-foia-app2.htm" href="http://cryptome.org/nsa-foia-app2.htm" rel="noopener noreferrer" target="_blank">

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    秘密之身 — —
   </strong>
  </p>
  <p class="graf graf--p">
   2001年4月25日：一名前国家安全局开创性工作的作者提供了一本关于该神秘机构掩盖、窃听和秘密任务的新书。
  </p>
  <p class="graf graf--p">
   1982年，詹姆斯·班福德（James Bamford）出版了“益智宫”（The Puzzle Palace），这是他对国家安全局的第一次曝光。他对国家安全局的第二次曝光被称为“秘密之身”。
  </p>
  <p class="graf graf--p">
   二十年的过程，在情报业务上有了很大的不同。在这20年里，里根的军事集结来去匆匆、苏联垮台、冷战结束，出现了一批新的军事敌人。
  </p>
  <p class="graf graf--p">
   电子通信通过传真、手机、互联网等新媒介实现了爆炸式增长。密码学从阴影中走出来，成为网络世界的基本技术。而且计算能力增加了一万倍。
  </p>
  <p class="graf graf--p">
   同样在这20年中，国家安全局逐渐向外界敞开大门。但
   <strong class="markup--strong markup--p-strong">
    它的任务 — — 窃听所有美国感兴趣的外国通信 — — 始终保持不变
   </strong>
   。这个过去曾称自己为“无此机构”和“从不说什么”的神秘机构，开始出现在公众面前，与媒体交谈、并开始宣传自己。詹姆斯·班福德（James Bamford）对国家安全局内部的了解恐怕比其他任何人都多。[
   <a class="markup--anchor markup--p-anchor" data-href="http://www.salon.com/2001/04/25/nsa_3/" href="http://www.salon.com/2001/04/25/nsa_3/" rel="noopener noreferrer" target="_blank">
    Salon: “Body of Secrets” by James Bamford]
   </a>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Optical tempest — —
   </strong>
  </p>
  <p class="graf graf--p">
   2002年：发现了一种以前未知的泄漏形式。在某些条件下，数据通信设备上的 LED 状态指示器显示为携带与设备处理的信息显著相关的调制光信号。不需要物理访问，攻击者就可以获取通过设备的所有数据，包括数据加密系统中的明文。
  </p>
  <p class="graf graf--p">
   实验表明，
   <strong class="markup--strong markup--p-strong">
    可以在相当远的实际条件下拦截数据
   </strong>
   。发现许多不同类型的设备（包括调制解调器和 Internet 协议路由器）都很容易受到攻击。被称为“Optical tempest”。[
   <a class="markup--anchor markup--p-anchor" data-href="http://applied-math.org/optical_tempest.pdf" href="http://applied-math.org/optical_tempest.pdf" rel="noopener noreferrer" target="_blank">
    Information Leakage from Optical Emanations
   </a>
   ]

    监视及其近亲兄弟 — — 即 丑闻和肮脏的信息，一直是华盛顿寻求全球统治的关键武器
   </strong>
   。毫不奇怪，在9/11后的两党行政权力行动中，乔治•W•布什和巴拉克•奥巴马（Barack Obama）已经主持了建立国家安全局的步骤，秘密迈向数字全景监控，旨在监控全球每一位美国人和其他国家领导人的沟通。[
   <a class="markup--anchor markup--p-anchor" data-href="http://www.juancole.com/2014/01/surveillance-leverage-security.html" href="http://www.juancole.com/2014/01/surveillance-leverage-security.html" rel="noopener noreferrer" target="_blank">
    NSA Surveillance is about Control &amp; Leverage, not Security
   </a>
   ]
  </p>
  <h3 class="graf graf--h3">
   <strong>
    — — FBI &amp; PRISM — —
   </strong>
  </h3>
  <p class="graf graf--p">
   2015年1月：国家安全局并不是唯一涉及大规模监视的 PRISM 计划的政府组织。
  </p>
  <p class="graf graf--p">
   一份解密的报告显示，FBI 也逐渐加大了对搜索记录、聊天、和电子邮件收集计划的参与度。该报告有许多被修改之处，由司法部发布，这是时报提起的信息自由法案诉讼的结果。它详细说明了联邦调查局对 PRISM 日益增长的兴趣，已成为该计划的积极参与者。该计划显然扩大了该局作为监视机构的初始角色，该角色本应该旨在确保美国人的账户不被攫取，现在它被反过来了。[
   <a class="markup--anchor markup--p-anchor" data-href="http://pando.com/2015/01/12/the-fbi-took-advantage-of-the-nsas-prism-program/" href="https://pando.com/2015/01/12/the-fbi-took-advantage-of-the-nsas-prism-program/" rel="noopener noreferrer" target="_blank">
    The FBI took advantage of the NSA’s PRISM program
   </a>
   ]
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    黑客根据 NSA 的监视工具包创建了间谍插件 — —
   </strong>
  </p>
  <p class="graf graf--p">
   NSA 泄漏的文件还被发现了一个隐蔽的 USB 棒，可用于监视离线设备。但该设备的成本高达惊人的20,000美元（13,200英镑）。然后，就有黑客团队使用便宜的电路板和其他部件创建了自己版本的监控套件，总价仅为20美元（13英镑）。[
   <a class="markup--anchor markup--p-anchor" data-href="http://www.dailymail.co.uk/sciencetech/article-2920419/When-USBs-attack-Hackers-create-covert-spy-plug-inspired-NSA-s-Cottonmouth-surveillance-kit.html" href="https://www.dailymail.co.uk/sciencetech/article-2920419/When-USBs-attack-Hackers-create-covert-spy-plug-inspired-NSA-s-Cottonmouth-surveillance-kit.html" rel="noopener noreferrer" target="_blank">
    When USBs attack: Hackers create spy plug inspired by the NSA’s surveillance kit — and it costs just £13 to make
   </a>
   ]
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    DEA 操作曝光 — —
   </strong>
  </p>
  <p class="graf graf--p">
   2015年4月8日：人权组起诉DEA大量收集电话记录。在美国政府监管的法律挑战中开辟了另一个阵线。
  </p>
  <p class="graf graf--p">
   人权观察的律师在洛杉矶的美国地方法院提起诉讼。该诉讼要求法官宣布 DEA 计划非法，并禁止DEA再次批量收集通话记录。该计划于2013年9月结束，然而
   <strong class="markup--strong markup--p-strong">
    至此，监视计划已经运行了15年
   </strong>
   。[
   <a class="markup--anchor markup--p-anchor" data-href="http://www.reuters.com/article/2015/04/08/us-usa-dea-lawsuit-idUSKBN0MZ1F720150408" href="https://www.reuters.com/article/2015/04/08/us-usa-dea-lawsuit-idUSKBN0MZ1F720150408" rel="noopener noreferrer" target="_blank">
    Reuters: Rights group sues DEA over bulk collection of phone records
   </a>
   ]
  </p>
  <p class="graf graf--p">
   2015年4月9日：DEA Global Surveillance Dragnet Exposed; 数据收集可能继续。“今日美国”报道的庞大计划与臭名昭著的“Project Crisscross”有一些相同的标志：它始于20世纪90年代初，表面上旨在收集有关贩毒的情报和全球目标国家情报，重点关注中美洲和南美洲。
  </p>
  <p class="graf graf--p">
   这也让人联想到所谓的“半球项目”，即 2013年9月“纽约时报”发布的DEA行动，其历史可追溯到1987年，并且几乎每天都收集大量的国际通话记录。[
   <a class="markup--anchor markup--p-anchor" data-href="https://firstlook.org/theintercept/2015/04/08/dea-surveillance-phone-records-crisscross-nsa/" href="https://firstlook.org/theintercept/2015/04/08/dea-surveillance-phone-records-crisscross-nsa/" rel="noopener noreferrer" target="_blank">
    DEA Global Surveillance Dragnet Exposed; Access to Data Likely Continues
   </a>
   ]
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这是政治！ — —
   </strong>
  </p>
  <p class="graf graf--p">
   2015年6月4日：一群两党华盛顿立法者向五角大楼官员征求意见，关于他们
   <strong class="markup--strong markup--p-strong">
    能用些什么方法来“摧毁”前国家安全局承包商爱德华·斯诺登“在媒体和公众舆论中的可信度”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这是根据VICE通过信息自由法（FOIA）诉讼获得的解密政府文件中暴露的。2013年12月和2014年2月，国会情报局（DIA）的高级官员在众议院和参议院监督委员会关于DIA对所谓的国家安全损害的评估的机密简报中提出了信息请求。[
   <a class="markup--anchor markup--p-anchor" data-href="https://news.vice.com/article/exclusive-inside-washingtons-quest-to-bring-down-edward-snowden" href="https://news.vice.com/article/exclusive-inside-washingtons-quest-to-bring-down-edward-snowden" rel="noopener noreferrer" target="_blank">
    Inside Washington’s Quest to Bring Down Edward Snowden
   </a>
   ]
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    间谍行为的扩大化 —
   </strong>
   —
  </p>
  <p class="graf graf--p">
   2015年6月4日：根据国家安全局的机密文件，奥巴马政府在没有公开通知或辩论的情况下扩大了国家安全局对美国国际互联网流量的无证监控。
  </p>
  <p class="graf graf--p">
   文件显示，2012年年中，司法部律师写了两份秘密备忘录，允许间谍机构在没有逮捕令的情况下在美国境内大规模监视搜查。[
   <a class="markup--anchor markup--p-anchor" data-href="https://www.propublica.org/article/new-snowden-documents-reveal-secret-memos-expanding-spying" href="https://www.propublica.org/article/new-snowden-documents-reveal-secret-memos-expanding-spying" rel="noopener noreferrer" target="_blank">
    ProPublica: New Snowden Documents Reveal Secret Memos Expanding Spyin
   </a>
   ]
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    Edward Snowden: 随着每个法庭的胜利，随着法律的每一次变化，我们可以证明事实比恐惧更有说服力。作为一个社会，我们将重新发现权利的价值不在于它隐藏的东西，而在于它所保护的东西。
   </p>
  </blockquote>
  <p class="graf graf--p">
   感谢所有冒着生命危险说出真相的人。透明度革命已经有近40年历史，只要透明度革命还在继续，民主就不会死亡。更多监视操纵的报告详见 IYP 类目
   <a class="markup--anchor markup--p-anchor" data-href="https://docs.google.com/document/d/1zHDRsb7BtR05ILg3F3rhAxBAcD9PJpbBwpGXPaAPP9w/edit?usp=sharing" href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e4%b8%8e%e6%93%8d%e7%ba%b5/" rel="noopener noreferrer" target="_blank">
    “监视和操纵”、“技术暴政”
   </a>
   、“
   <a class="markup--anchor markup--p-anchor" data-href="https://docs.google.com/document/d/1tmU2VmsdHRo79NniIM85yWJrS-A_GIhku_0yXZOg8Fk/edit?usp=sharing" href="https://www.iyouport.org/category/%e5%86%9b%e4%ba%8b%e5%b7%a5%e4%b8%9a%e7%bb%bc%e5%90%88%e4%bd%93/" rel="noopener noreferrer" target="_blank">
    军事工业综合体
   </a>
   ”等部分。⚪️
