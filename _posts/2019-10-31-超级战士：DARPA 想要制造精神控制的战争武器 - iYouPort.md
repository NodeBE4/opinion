  <p class="graf graf--p">
   去年，美国国防高级研究计划局（DARPA）
   <a class="markup--anchor markup--p-anchor" data-href="https://www.darpa.mil/news-events/2018-03-16" href="https://www.darpa.mil/news-events/2018-03-16" rel="noopener noreferrer" target="_blank">
    宣布
   </a>
   ，正在寻求“无创手段”“无需手术即可实现高水平的大脑系统通信”。
  </p>
  <p class="graf graf--p">
   该机构称，这些技术将“允许与特定神经元或神经元组的精确、高质量的连接”。
  </p>
  <p class="graf graf--p">
   该机构希望制造精神控制类的战争武器，正如他们（含糊地）在最近的两份新闻稿中解释过的那样。
  </p>
  <p class="graf graf--p">
   为实现这一目标，DARPA最近成立了一个名为
   <a class="markup--anchor markup--p-anchor" data-href="https://www.darpa.mil/news-events/2018-03-16" href="https://www.darpa.mil/news-events/2018-03-16" rel="noopener noreferrer" target="_blank">
    Next-Generation Nonsurgical Neurotechnology
   </a>
   （N3）的计划。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    DARPA 希望士兵能够用自己的意识控制机器
   </strong>
  </p>
  <p class="graf graf--p">
   根据2019年5月20日发布的新闻稿，该机构已向六家公司提供资金支持N3：
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.darpa.mil/news-events/2019-05-20" href="https://www.darpa.mil/news-events/2019-05-20" rel="noopener noreferrer" target="_blank">
    新闻稿说
   </a>
   ：巴特尔纪念研究所、卡内基梅隆大学、约翰霍普金斯大学应用物理实验室、帕洛阿尔托研究中心（PARC）、莱斯大学、和 Teledyne Scientific，领导多学科团队开发高分辨率，双向脑机接口……
  </p>
  <p class="graf graf--p">
   ……这些可穿戴接口最终可以实现多样化的应用，例如控制活动网络防御系统和大群的无人机，或者在执行复杂任务期间与计算机系统合作以进行多任务处理。
  </p>
  <p class="graf graf--p">
   去年宣布这项为期四年的计划时，DARPA在一份新闻稿中首先提到了脑机界面的所有积极因素：
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.darpa.mil/news-events/2018-03-16" href="https://www.darpa.mil/news-events/2018-03-16" rel="noopener noreferrer" target="_blank">
    他们称
   </a>
   ：大脑系统通信的最新研究采用了侵入性技术，可以与特定神经元或神经元组进行精确的、高质量的连接，以帮助患者控制假肢等活动。
  </p>
  <p class="graf graf--p">
   听起来很高大上吗？再继续阅读您就不会这么感觉了：
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.darpa.mil/news-events/2018-03-16" href="https://www.darpa.mil/news-events/2018-03-16" rel="noopener noreferrer" target="_blank">
    他们说
   </a>
   ：“但是，这些技术并不适合身体健全的人。 于是 DARPA 现在寻求在其新计划 N3 中实现高水平的大脑系统通信而无需手术。”
  </p>
  <p class="graf graf--p">
   如果您还没有发现令人不寒而栗，请继续阅读：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “DARPA 创建了N3，以寻求通向安全、便携式神经接口系统的途径，
   <strong class="markup--strong markup--p-strong">
    该系统能够同时对大脑中的多个点进行读取和写入
   </strong>
   ”，这是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.darpa.mil/staff/dr-al-emondi" href="https://www.darpa.mil/staff/dr-al-emondi" rel="noopener noreferrer" target="_blank">
    Al Emondi 博士
   </a>
   说的，他是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.darpa.mil/about-us/offices/bto" href="https://www.darpa.mil/about-us/offices/bto" rel="noopener noreferrer" target="_blank">
    DARPA 生物技术办公室（BTO）
   </a>
   的项目经理。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.darpa.mil/news-events/2018-03-16" href="https://www.darpa.mil/news-events/2018-03-16" rel="noopener noreferrer" target="_blank">
    他还说
   </a>
   ，“高分辨率、非手术神经技术一直难以捉摸，但由于生物医学工程、神经科学、合成生物学、和纳米技术的最新进展，我们现在相信这个目标是可以实现的。”
  </p>
  <p class="graf graf--p">
   他们希望这项技术能够在未来几年内做好准备。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    DARPA 希望建立与植入电极的性能相匹配但不进行任何手术的脑机接口
   </strong>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://spectrum.ieee.org/the-human-os/biomedical/bionics/darpa-funds-ambitious-neurotech-program" href="https://spectrum.ieee.org/the-human-os/biomedical/bionics/darpa-funds-ambitious-neurotech-program" rel="noopener noreferrer" target="_blank">

   <strong class="markup--strong markup--p-strong">
    士兵可以在没有触摸键盘的情况下，用意识命令控制中心；用意识直观地操作无人机；甚至用知觉入侵安全网络。
   </strong>
  </p>
  <p class="graf graf--p">
   根据
   <a class="markup--anchor markup--p-anchor" data-href="https://spectrum.ieee.org/the-human-os/biomedical/bionics/darpa-funds-ambitious-neurotech-program" href="https://spectrum.ieee.org/the-human-os/biomedical/bionics/darpa-funds-ambitious-neurotech-program" rel="noopener noreferrer" target="_blank">

   <a class="markup--anchor markup--p-anchor" data-href="https://engineering.cmu.edu/news-events/news/2019/05/20-darpa.html" href="https://engineering.cmu.edu/news-events/news/2019/05/20-darpa.html" rel="noopener noreferrer" target="_blank">

   </a>
   <strong class="markup--strong markup--p-strong">

   ”
  </p>
  <p class="graf graf--p">
   该报告还提供了关于该计划的一些额外细节：
  </p>
  <p class="graf graf--p">
   具体来说，该计划正在寻找能够在50毫秒往返中读取和写入脑细胞的技术，并且能够以1立方毫米的分辨率（包含数千个神经元的空间）与大脑中的至少16个位置进行交互……
  </p>
  <p class="graf graf--p">
   …… Emondi 说，为期四年的N3计划将分为三个阶段。在目前的第1阶段，团队有一年的时间来展示通过头骨读取（记录）和写入（刺激）脑组织的能力。
  </p>
  <p class="graf graf--p">
   …… 成功的团队将进入第2阶段。在随后的18个月中，这些团队将开发工作设备
   <strong class="markup--strong markup--p-strong">
    并对活体动物进行测试
   </strong>
   。任何成功的团队都将进入
   <strong class="markup--strong markup--p-strong">
    第3阶段，也就是在人体上进行测试
   </strong>
   。

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种脑机接口技术真正的用途是什么？
   </strong>
  </p>
  <p class="graf graf--p">
   DARPA 关于 N3 的新闻稿相当模糊，该机构的内部运作很隐秘。
  </p>
  <p class="graf graf--p">
   2018年11月，大西洋月刊公布了一份
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theatlantic.com/magazine/archive/2018/11/the-pentagon-wants-to-weaponize-the-brain-what-could-go-wrong/570841/" href="https://www.theatlantic.com/magazine/archive/2018/11/the-pentagon-wants-to-weaponize-the-brain-what-could-go-wrong/570841/" rel="noopener noreferrer" target="_blank">
    关于 DARPA 的令人瞩目的报告
   </a>
   。
  </p>
  <p class="graf graf--p">
   以下是一些令人不安的摘录，强烈建议您完整阅读该报告。
  </p>
  <p class="graf graf--p">
   DARPA 几十年来一直梦想着将人类和机器融合在一起。几年前，当精神控制武器的前景成为该机构的公关责任时，官员们就打开了独特的聪明才智……
  </p>
  <p class="graf graf--p">
   ……他们重新阐述了神经科学研究的既定目的，
   <strong class="markup--strong markup--p-strong">
    表面上
   </strong>
   关注的是愈合创伤和治疗疾病的狭隘目标。
  </p>
  <p class="graf graf--p">
   ……机构官员称，这项工作不是武器或战争。这是关于治疗和保健。
  </p>
  <p class="graf graf--p">
   ……但是，即使这种说法完全属实，这种变化也会产生广泛的伦理、社会和形而上学的争议。几十年后，神经科技可能会导致社会混乱，使智能手机和互联网看起来像历史池塘中的温柔涟漪。
  </p>
  <p class="graf graf--p">
   … 然而，
   <strong class="markup--strong markup--p-strong">
    关于健康目标的声称不完全是真的
   </strong>
   。该机构一直有更大的使命，在2015年的一次演讲中，一位项目经理 —— 硅谷招聘人员 —— 描述了这一使命：“让人们摆脱健康身体的局限”。
  </p>
  <p class="graf graf--p">
   ……
   <strong class="markup--strong markup--p-strong">
    “我们的使命是让人类成为一种不同于普通人类的东西，超越我们出生时的能力，超越我们能够有机地获得的能力”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   …… 对普通人类的改造成为该机构的优先事项：“没有生理或认知限制的士兵将是未来生存和运营优势的关键”，Michael Goldblatt 说，他在1999年加入 DARPA。
  </p>
  <p class="graf graf--p">
   …… 正如 Annie Jacobsen 在她2015年出版的“The Pentagon’s Brain”一书中所描述的那样，这个时代经常被描绘成疯狂科学家的领地，创造那种可以7/24工作、长达一周不睡觉的士兵。
  </p>
  <p class="graf graf--p">
   …… Jacobsen 指出，darpa 的神经技术研究，包括上肢假肢和脑机接口，并不像他们声称的那样：这些假肢的主要目标很可能是给机器人的，而不是人类，更强大的胳膊和手。”……
  </p>
  <p class="graf graf--p">
   DARPA 几十年来一直试图用机器人技术创造超级战士，正如 Daisy Luther 在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theorganicprepper.com/robot-apocalypse-artificial-intelligence/" href="https://www.theorganicprepper.com/robot-apocalypse-artificial-intelligence/" rel="noopener noreferrer" target="_blank">
    机器人启示录
   </a>
