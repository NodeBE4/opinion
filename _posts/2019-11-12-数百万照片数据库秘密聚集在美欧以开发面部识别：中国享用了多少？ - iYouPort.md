       这是一个持续了十多年的大型监视证据，并且惊人地被很多国家的监视项目分享。但为什么，直到现在才报道？而且报道并不完整……
      </strong>

   <img alt="" class="aligncenter size-full wp-image-6854 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-11.jpg?resize=1100%2C550&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-11.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-11.jpg?resize=300%2C150&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-11.jpg?resize=768%2C384&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-11.jpg?resize=1024%2C512&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-11.jpg?resize=1100%2C550&amp;ssl=1 1100w" data-recalc-dims="1" height="550" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-11.jpg?resize=1100%2C550&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p graf--startsWithDoubleQuote">
   “纽约时报” 发布的一份报告显示，有数百万张面部照片来自隐藏的监视摄像头，一直被储存在用于人工智能（AI）研究和开发目的的数据库中，而且已经共享了十多年。
  </p>
  <p class="graf graf--p">
   ⚠️由名牌大学和科技公司
   <strong class="markup--strong markup--p-strong">
    秘密创建的该照片数据集一直以来
   </strong>
   被广泛用于面部识别和生物识别技术的研发，这些技术现在被全世界的警察和国家情报机构普遍使用。
  </p>
  <p class="graf graf--p">
   ⚠️这个大型数字脸部和“自拍”照片数据库
   <strong class="markup--strong markup--p-strong">
    来自网站、社交媒体、照片共享应用程序和在线约会平台
   </strong>
   ，并没有经过授权，也包括被
   <strong class="markup--strong markup--p-strong">
    公共场所的监视相机拍摄到的图像
   </strong>
   —— 该数据集已经被国家机构、软件工程师和研究人员使用人工智能算法和图像模式分析，以寻求创造更前沿的面部识别技术。
  </p>
  <p>
   延伸：《
   <a href="https://www.iyouport.org/%e4%bd%a0%e6%9b%be%e7%bb%8f%e5%9c%a8%e7%a4%be%e4%ba%a4%e7%bd%91%e7%bb%9c%e4%b8%8a%e5%8f%91%e5%b8%83%e8%bf%87%e8%87%aa%e5%b7%b1%e7%9a%84%e7%85%a7%e7%89%87%ef%bc%9f%e4%bd%a0%e5%8d%b1%e9%99%a9%e4%ba%86/" rel="noopener noreferrer" target="_blank">
    你曾经在社交网络上发布过自己的照片？你危险了！
   </a>
   》
  </p>
  <p class="graf graf--p">
   该报告很大程度上基于 Adam Harvey 和 Jules LaPlace 发表在 MegaPixels.cc 网站上的信息 —— 其中至少有30个面部图像数据集可以追溯到至少2007年。
  </p>
  <p class="graf graf--p">
   Megapixels “精确定位了
   <strong class="markup--strong markup--p-strong">
    由微软、斯坦福大学和其他公司建造的这些存储库
   </strong>
   ，其中一个存储了超过1000万个图像，而另一个存储超过200万个。”
  </p>
  <p class="graf graf--p">
   ⚠️“
   <strong class="markup--strong markup--p-strong">
    不仅包括公司和大学，还有澳大利亚、中国、印度、新加坡和瑞士的研究人员、政府和私营企业，广泛地分享了这些数据，用于培训人工智能
   </strong>
   ……”。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-6857 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/4-6.jpg?resize=1100%2C619&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/4-6.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/4-6.jpg?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/4-6.jpg?resize=768%2C432&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/4-6.jpg?resize=1024%2C576&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/4-6.jpg?resize=1100%2C619&amp;ssl=1 1100w" data-recalc-dims="1" height="619" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/4-6.jpg?resize=1100%2C619&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   关于这些数据集的使用者，纽约时报
   <strong class="markup--strong markup--p-strong">
    显然没有提到
   </strong>
   其中一个重量级的单位：美国政府和军事机构，他们一直在通过与私营公司和大学研究机构合作以进行测试和开发。
  </p>
  <p class="graf graf--p">
   例如，2014年斯坦福大学和德国马克斯普朗克信息学研究所共同发起了一个名为 Brainwash 的项目，在旧金山市中心的 Brainwash 咖啡馆中部署了一个隐藏的网络摄像头。
  </p>
  <p class="graf graf--p">
   斯坦福大学以其与美国军事情报界的密切联系而闻名。
  </p>
  <p class="graf graf--p">
   谷歌就是在斯坦福大学开发的，由国防高级研究计划局（DARPA）和其他国家情报机构在20世纪90年代初提供的资金。
  </p>
  <p class="graf graf--p">
   在这里看到具体介绍
   <strong class="markup--strong markup--p-strong">
    《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e7%9a%84%e7%9c%9f%e9%9d%a2%e7%9b%ae%e2%80%8a-%e2%80%8a-theyre-always-watching/" href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e7%9a%84%e7%9c%9f%e9%9d%a2%e7%9b%ae%e2%80%8a-%e2%80%8a-theyre-always-watching/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     谷歌的真面目 — — They’re always watching
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》
   </strong>
   。
  </p>
  <p class="graf graf--p">
   纽约时报显然也没有提及，马克斯普朗克研究所与德国政府有着长期的直接联系。
  </p>
  <p class="graf graf--p">
   在为期三天的时间内，未经 BrainwashCafé 咖啡店同意，该监视项目共捕获了11,917个视频流，每个视频流100秒。
  </p>
  <p class="graf graf--p">
   根据 MegaPixels 的说法，“
   <strong class="markup--strong markup--p-strong">
    没有任何普通的咖啡馆客户会想到他们的形象会最终出现在用于监控研究和开发的数据集中
   </strong>
   ，但这正是旧金山 Brainwash Cafe 的客户身上所发生的事。”
  </p>
  <p class="graf graf--p">
   MegaPixels 还表示，这些视频是来自 AngelCam 并被在线发布的，AngelCam 是一种网络流媒体服务，用于家庭监视目的，每月只需6美元。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    Brainwash 数据库随后被用于中国、瑞士、荷兰、美国、印度和加拿大的 AI 研究目的
   </strong>
   。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-6855 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-13.png?resize=1100%2C613&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-13.png?w=1200&amp;ssl=1 1200w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-13.png?resize=300%2C167&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-13.png?resize=768%2C428&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-13.png?resize=1024%2C571&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-13.png?resize=1100%2C613&amp;ssl=1 1100w" data-recalc-dims="1" height="613" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-13.png?resize=1100%2C613&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>

  <p class="graf graf--p">
   在另一个案例中，纽约时报说，杜克大学的研究人员于2014年在校园内使用八台摄像机启动了一个名为 Duke MTMC 的项目，构建面部图像数据库。
  </p>
  <p class="graf graf--p">
   这些摄像机的下面贴有标志，上面有一个电话号码和电子邮件地址，供想要退出该研究的人使用。
  </p>
  <p class="graf graf--p">
   该监视项目在14个小时内收集了大约2,700个人的200万个同步视频帧，其中大多数被拍到的人是学生。
  </p>
  <p class="graf graf--p">
   纽约时报
   <strong class="markup--strong markup--p-strong">
    再次隐藏了
   </strong>
   有关美国政府使用杜克 MTMC 数据集的重要细节。
  </p>
  <p class="graf graf--p">
   不过 MegaPixels 报道⚠️
   <strong class="markup--strong markup--p-strong">
    中国政府也使用了杜克大学的照片 —— 仅在2018年就有超过90个研究项目 —— 用于监视目的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Harvey 和 LaPlace 还解释说，该
   <strong class="markup--strong markup--p-strong">
    数据集的最初创建“部分得到了美国陆军研究实验室的支持”，并用于“监视目的自动分析人群和社交聚会”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   此外，MegaPixels 报告称，“
   <strong class="markup--strong markup--p-strong">
    来自美国和欧洲的案例显示出与中国相似的趋势
   </strong>
   ，包括美国国土安全部、IARPA、IBM、为ICE提供监视技术的 Microsoft、以及与英国国防部合作的 Vision Semantics。
  </p>
  <p class="graf graf--p">
   时报还回顾了2016年创建的名为 MS Celeb 的微软数据集，该数据集包含1000万张图像，这些图像来自“表面上是名人数据库”的网站。
  </p>
  <p class="graf graf--p">
   但是，这可不仅仅是“名人”，许多其他人的名字和图片都包含在数据库中。
  </p>
  <p class="graf graf--p">
   纽约时报还没有提到，⚠️MegaPixels 在 MS Celeb 数据库中发布了24个名单，其中包括作者、记者、电影制作人、博主和数字权利活动家。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    尤其还包括杰里米·斯卡希尔（Jeremy Scahill），他是
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e9%bb%91%e5%ae%a2%e5%9b%a2%e9%98%9f%e5%92%8c%e9%98%bf%e8%81%94%e9%85%8b%e9%97%b4%e8%b0%8d%e5%90%88%e4%bd%9c%e8%ae%a1%e5%88%92%e6%94%bb%e5%87%bb%e6%8a%a5%e9%81%93%e6%96%af%e8%af%ba/" href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e9%bb%91%e5%ae%a2%e5%9b%a2%e9%98%9f%e5%92%8c%e9%98%bf%e8%81%94%e9%85%8b%e9%97%b4%e8%b0%8d%e5%90%88%e4%bd%9c%e8%ae%a1%e5%88%92%e6%94%bb%e5%87%bb%e6%8a%a5%e9%81%93%e6%96%af%e8%af%ba/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Intercept
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    的记者和编辑，曾广泛撰写有关战争罪的文章，并为维基解密编辑朱利安·阿桑奇（
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/?s=Julian+Assange" href="https://www.iyouport.org/?s=Julian+Assange" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Julian Assange
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ）辩护，反对美国的监禁和引渡
   </strong>
   。
  </p>
  <p class="graf graf--p">
   MS Celeb 数据集包含200张 Scahill 的面部照片。
  </p>
  <p class="graf graf--p">
   延伸：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e9%bb%91%e5%ae%a2%e5%9b%a2%e9%98%9f%e5%92%8c%e9%98%bf%e8%81%94%e9%85%8b%e9%97%b4%e8%b0%8d%e5%90%88%e4%bd%9c%e8%ae%a1%e5%88%92%e6%94%bb%e5%87%bb%e6%8a%a5%e9%81%93%e6%96%af%e8%af%ba/" href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e9%bb%91%e5%ae%a2%e5%9b%a2%e9%98%9f%e5%92%8c%e9%98%bf%e8%81%94%e9%85%8b%e9%97%b4%e8%b0%8d%e5%90%88%e4%bd%9c%e8%ae%a1%e5%88%92%e6%94%bb%e5%87%bb%e6%8a%a5%e9%81%93%e6%96%af%e8%af%ba/" rel="noopener noreferrer" target="_blank">
    美国黑客团队和阿联酋间谍合作计划攻击报道斯诺登文件的媒体机构 TheIntercept
   </a>
   》。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-6856 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-7.jpg?resize=1100%2C529&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-7.jpg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-7.jpg?resize=300%2C144&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-7.jpg?resize=768%2C370&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-7.jpg?resize=1024%2C493&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-7.jpg?resize=1100%2C529&amp;ssl=1 1100w" data-recalc-dims="1" height="529" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-7.jpg?resize=1100%2C529&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6856" data-recalc-dims="1" height="529" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-7.jpg?resize=1100%2C529&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-7.jpg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-7.jpg?resize=300%2C144&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-7.jpg?resize=768%2C370&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-7.jpg?resize=1024%2C493&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-7.jpg?resize=1100%2C529&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   MS Celeb 数据集的目标是针对100万人，并包括另外 900,000 个没有附加图像的名称。其中 10万人的数据集已被十几个国家访问。
  </p>
  <p class="graf graf--p">
   MegaPixels 网站显示，⚠️
   <strong class="markup--strong markup--p-strong">
    2018年在全球发生的124个研究项目中使用了 MS Celeb 数据集，其中大部分位于中国（47）和美国（42）
   </strong>
   。
  </p>
  <p class="graf graf--p">
   MegaPixels 网站上还有两个图像数据库
   <strong class="markup--strong markup--p-strong">
    未被
   </strong>
   “纽约时报”报道，其中一个来自牛津大学，另一个来自科罗拉多大学。
  </p>
  <p class="graf graf--p">
   牛津中心数据集包含自2007年起从英国牛津 Cornmarket 街和市场街拐角处安装的监控摄像头拍摄到的2,200人的视频。
  </p>
  <p class="graf graf--p">
   该监视项目由牛津大学在一项名为 Project HERMES 的欧盟人工智能计划的支持下进行。
  </p>
  <p class="graf graf--p">
   MegaPixels 报告说，
   <strong class="markup--strong markup--p-strong">
    该图像数据集已被广泛分享，提供给来自世界各地的80个研究项目使用
   </strong>
   。
  </p>
  <p class="graf graf--p">
   根据 MegaPixels 的说法，最终的数据集来自科罗拉多大学科罗拉多斯普林斯分校，其中1,700名学生和其他行人“
   <strong class="markup--strong markup--p-strong">
    在他们不知情的情况下被使用远程高分辨率监控摄像机拍摄
   </strong>
   ”。
  </p>
  <p class="graf graf--p">
   这些照片是在2012–2013学年春季学期在科罗拉多校区西草坪上拍摄的，并且是在学生们在课间休息时。
  </p>
  <p class="graf graf--p">
   MegaPixels 报道，毫无约束的脸部数据集 “为研究人员提供了真实的监控图像，以帮助构建
   <strong class="markup--strong markup--p-strong">
    面向国防、情报和商业合作伙伴的人脸识别系统
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   总的来说，MegaPixels 在30个公开可用的人脸识别和面部分析数据集中找到了2400万张“非合作的，非自愿被拍摄的照片”，这些照片是在没有任何明确同意的情况下被收集的，这是一种研究人员称之为“在野”的面部图像。
  </p>
  <p class="graf graf--p">
   每张图片至少包含一张脸，许多照片包含多张脸。在所有2400万张图片中，大约有100万个独特的身份。“
  </p>
  <p class="graf graf--p">
   软件公司 Clarifai 收集了一个面部数据库，其中包含来自约会网站 OKCupid 的图片。
  </p>
  <p class="graf graf--p">
   Clarifai 的首席执行官 Matthew Zeiler 告诉“纽约时报”，他有权访问 OKCupid 图片，因为
   <strong class="markup--strong markup--p-strong">
    “一些约会网站的创始人投资了他的公司。”
   </strong>
  </p>
  <p class="graf graf--p">
   Zeiler 还表示，⚠️
   <strong class="markup--strong markup--p-strong">
    他与一家大型未透露名字的社交媒体公司签署了一项协议，“将其图像完全用于训练人脸识别模型”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   猜猜看，这家“大型”社交媒体公司，是谁？
  </p>
  <p class="graf graf--p">
   Clarifai 使用 OKCupid 照片开发面部识别软件，可以识别分析面部的年龄、性别和种族。
  </p>
  <p class="graf graf--p">
   当被问及意图时，Zeiler 说：⚠️
   <strong class="markup--strong markup--p-strong">
    “如果情况合适，Clarifai 会将其面部识别技术出售给外国政府、军事行动和警察部门。”
   </strong>
  </p>
  <p class="graf graf--p">
   欧洲和美国大学以及硅谷科技公司十多年来一直参与收集“非合作、非自愿照片”，用于研究目的，此揭露再次表明，实施面部识别和生物识别技术的国家监视工作进展顺利。
  </p>
  <p class="graf graf--p">
   这些组织秘密创建和分享用于人工智能开发的面部图像之内幕也暴露了
   <strong class="markup--strong markup--p-strong">
    学术界和企业界明显参与了攻击基本民主权利的交易
   </strong>
   。
  </p>
  <p class="graf graf--p">
   尽管 Adam Harvey 和 Jules LaPlace 在 MegaPixels 上发布的信息得到了 Mozilla 开源社区的支持，但自2017年11月开始提供以来，包括纽约时报在内的企业媒体一直没有报道，直到现在。
  </p>
  <p class="graf graf--p">
   现在报道这件事是因为美国公众对地方、州和联邦警察机构对整个社会进行面部识别和生物识别监视的反对意识和愤怒日益增加。简单说，目前报道它能获得更多点击量。⚪️
  </p>
  <p>

   </a>
  </p>
