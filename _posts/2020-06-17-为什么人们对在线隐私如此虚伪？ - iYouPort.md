  <p class="graf graf--p">
   社会心理学家在几十年前已经知道，态度和行为之间的关系是复杂的，即使不是脆弱的话。例如，
   <strong class="markup--strong markup--p-strong">
    尽管您可能会被无处不在的网上跟踪所困扰，但它可能并没有改变您的在线行为。
   </strong>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12222 jetpack-lazy-image" data-lazy-sizes="(max-width: 794px) 100vw, 794px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-3.png?resize=794%2C790&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-3.png?w=794&amp;ssl=1 794w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-3.png?resize=300%2C298&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-3.png?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-3.png?resize=768%2C764&amp;ssl=1 768w" data-recalc-dims="1" height="790" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-3.png?resize=794%2C790&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="794"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12222" data-recalc-dims="1" height="790" sizes="(max-width: 794px) 100vw, 794px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-3.png?resize=794%2C790&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-3.png?w=794&amp;ssl=1 794w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-3.png?resize=300%2C298&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-3.png?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-3.png?resize=768%2C764&amp;ssl=1 768w" width="794"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   同样，当 Edward Snowden 曝光国家安全局（NSA）惊人的间谍全球内幕后，
   <strong class="markup--strong markup--p-strong">
    公众普遍的愤怒和不信任却并没有减少他们对互联网的使用。
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    为什么人们没有主动保护自己《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/6%e5%b9%b4%ef%bc%8c%e6%94%b9%e5%8f%98%e4%ba%86%e4%bb%80%e4%b9%88%ef%bc%9f/" href="https://www.iyouport.org/6%e5%b9%b4%ef%bc%8c%e6%94%b9%e5%8f%98%e4%ba%86%e4%bb%80%e4%b9%88%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     6年，改变了什么
    </a>
    ？》
   </li>
  </ul>
  <p class="graf graf--p">
   实际上，斯诺登的曝光甚至都没有提高社交媒体上更高安全性设置的采用率。
  </p>
  <p class="graf graf--p">
   换句话说，即使人们说他们关注在线隐私，他们的关注也可能远远不足以推动数字节制。尽管越来越多的人正在使用VPN、广告拦截器和跟踪拦截器来收回一点丢失的隐私，但这些人在整体上仍然占极少数。
  </p>
  <p class="graf graf--p">
   由于您正在阅读本文，因此我们只能假设您至少有点担心自己的在线隐私。但是，真正采取措施的可能性很小，例如 控制他人发送给您的电子邮件、从浏览器中删除Cookie、定期检查计算机中是否存在间谍软件、以及删除浏览器历史记录等等。
  </p>
  <p class="graf graf--p">
   最近的
   <a class="markup--anchor markup--p-anchor" data-href="http://www.tandfonline.com/doi/abs/10.1080/00224545.2014.914881?journalCode=vsoc20" href="http://www.tandfonline.com/doi/abs/10.1080/00224545.2014.914881?journalCode=vsoc20" rel="noopener noreferrer" target="_blank">
    研究表明
   </a>
   ，即使消费者不相信自己的社交媒体数据是安全的，他们也没有计划采取保护性措施或选择退出。
  </p>
  <p class="graf graf--p">
   但是，意图通常是后续行为的较弱预测指标。
  </p>
  <p class="graf graf--p">
   例如，大多数人说他们宁可减少薪水以减少工作量、或做更多有趣的工作，但实际上很少有人真的这样做了。因此，最大的问题不是人们计划为隐私保护做些什么，而是他们实际上在做什么。
  </p>
  <p class="graf graf--p">
   为了解决这个问题，最近对166项研究进行了综合性分析，涉及包括来自34个国家的75,269名参与者，探索了所谓的 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%9a%90%e7%a7%81%e6%82%96%e8%ae%ba/" href="https://www.iyouport.org/%e9%9a%90%e7%a7%81%e6%82%96%e8%ae%ba/" rel="noopener noreferrer" target="_blank">
    隐私悖论
   </a>
   ”，即：人们对隐私的关注很少会转化为保护性行为的令人困惑的事实。
  </p>
  <p class="graf graf--p">
   与以前的研究相反，元分析的结果表明，对隐私更加关注和了解更多的人倾向于使用较少的在线服务、采取更强的安全性设置 、并披露较少的个人信息。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    当涉及社交媒体使用时，确实存在明显的隐私悖论，因为即使是表达担忧的个人也表现得很粗心，
   </strong>
   进行不适当的自我披露，大量公开其数字足迹，并允许广泛使用一系列外部应用程序以访问其数据。
  </p>
  <p class="graf graf--p">
   据
   <a class="markup--anchor markup--p-anchor" data-href="http://conferences.sigcomm.org/imc/2011/docs/p61.pdf" href="http://conferences.sigcomm.org/imc/2011/docs/p61.pdf" rel="noopener noreferrer" target="_blank">
    估计
   </a>
   ，根据（相当不安全的）默认设置共享了将近40％的 Facebook 内容，并且隐私设置仅在37％的时间内符合用户的期望。
  </p>
  <p class="graf graf--p">
   因此，
   <strong class="markup--strong markup--p-strong">
    可以说似乎没有多少隐私问题会使社交媒体用户更加谨慎。
   </strong>
  </p>
  <p class="graf graf--p">
   隐私悖论的一种可能解释是第三人效应，这表明，即使人们意识到使用社交媒体的潜在风险，他们仍会以某种方式认为这些风险并不适用于自己 —— 仅适用于他人。
  </p>
  <p class="graf graf--p">
   这种效应已在从饮酒、吸烟、到无保护性行为等各种与风险相关的活动中被发现。
  </p>
  <p class="graf graf--p">
   并不是人们不了解与这些活动相关的风险，也不是说人们一开始就愿意承担这些风险，而是幻想这些风险仅适用于其他人而不适用于自己。
  </p>
  <p class="graf graf--p">
   隐私悖论的另一个更明显的解释是简单的风险回报评估。
  </p>
  <p class="graf graf--p">
   我们大多数人确实希望在上网时感到安全和受到保护，但是使用免费网站和披露个人信息所带来的 “好处” 被认为
   <a class="markup--anchor markup--p-anchor" data-href="http://www.sunypress.edu/p-3659-boundaries-of-privacy.aspx" href="http://www.sunypress.edu/p-3659-boundaries-of-privacy.aspx" rel="noopener noreferrer" target="_blank">
    远大于
   </a>
   风险。
  </p>
  <p class="graf graf--p">
   正如科学研究所表明的那样，大多数人都使用社交网络来满足基本的心理需求，例如相处、构建和展示其价值观和身份、以及获得娱乐的需求。
  </p>
  <p class="graf graf--p">
   如果让消费者选择为他们使用的免费应用程序和在线服务付费，以换取保留所有个人数据的权利，那么他们可能会拒绝，而宁愿使用其个人数据 — 人权 来“支付”。
  </p>
  <p class="graf graf--p">
   由于人们对隐私的关注似乎并未转化为保护隐私的行为，可以很容易地预见未来，我们在网上所做的一切都将成为我们公众声誉的一部分。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    你的数字足迹已经可以用来推断你
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://www.pnas.org/content/110/15/5802.short" href="http://www.pnas.org/content/110/15/5802.short" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     内心最深处的
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    东西
   </strong>
   。 2013年对58,000名 Facebook 用户（自愿参加）的一项研究显示，大数据能够
   <strong class="markup--strong markup--p-strong">
    可靠地预测
   </strong>
   性取向、性别、种族、年龄、宗教、政治观点、智力水平、饮酒和吸烟状况、吸毒状况、以及是否与父母分居。
  </p>
  <p class="graf graf--p">
   研究人员还能够在某种程度上预测性格特征，例如 性格外向还是内向、是否尽责、是否开放、情绪是否稳定、是否和蔼可亲。
  </p>
  <p class="graf graf--p">
   想想看，Uber评分可以用来推断你的喜好或情绪智力、Spotify 和 Netflix 偏好可以推断出你的好奇心和开放性、亚马逊或淘宝浏览历史被用来推断你的冲动和尽责性 ……
  </p>
  <p class="graf graf--p">
   甚至你在 Twitter 上说的每一句话、在 Facebook 上点赞的每一个东西、倾向于访问的网站、以及
   <a class="markup--anchor markup--p-anchor" data-href="https://www.researchgate.net/profile/Laura_Fernandez_Gallardo/publication/304776271_Speech_Likability_and_Personality-Based_Social_Relations_A_Round-Robin_Analysis_over_Communication_Channels/links/577a348308ae213761c9a99d/Speech-Likability-and-Personality-Based-Social-Relations-A-Round-Robin-Analysis-over-Communication-Channels.pdf" href="https://www.researchgate.net/profile/Laura_Fernandez_Gallardo/publication/304776271_Speech_Likability_and_Personality-Based_Social_Relations_A_Round-Robin_Analysis_over_Communication_Channels/links/577a348308ae213761c9a99d/Speech-Likability-and-Personality-Based-Social-Relations-A-Round-Robin-Analysis-over-Communication-Channels.pdf" rel="noopener noreferrer" target="_blank">
    你的声音
   </a>
   ，都可以转变成相当详细的心理特征，而交易这些数据的潜力绝不局限于此营销世界 ——
   <a class="markup--anchor markup--p-anchor" data-href="https://start.me/p/RMQrDD/iyp-2" href="https://start.me/p/RMQrDD/iyp-2" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     监视的目的是操纵
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p">
   毋庸置疑，从黑客入侵，歧视到奥威尔式监视状态，数字分析的泛滥有许多潜在的危险（和道德问题）。公司可以知道的和应该知道的有很大的不同。
  </p>
  <p class="graf graf--p">
   正如最近一本有关在线说服心理学的书中所显示的那样，尽管目标数据可能会对购买意愿产生积极影响，但是它可能会带来严重的隐性成本，尤其是那些更具侵入性的做法。
  </p>
  <p class="graf graf--p">
   这种代价被称为心理反抗，是指我们在感知到对自己的自由和自治的威胁时所经历的那种厌恶情绪状态。供应商和应用程序将需要限制他们对消费者数据的处理方式，这意味着抵制过度使用数据的诱惑。
  </p>
  <p class="graf graf--p">
   小说家 Gabriel García Márquez 也许可以预见到今天这一隐私辩论的核心问题：“所有人都拥有三种生活：公共生活、私人生活和秘密生活。”
  </p>
  <p class="graf graf--p">
   我们拥有自己的公共生活，就是我们愿意在各种社会环境中与他人共享的生活。我们也有自己的私生活，并且很不情愿放弃这些私生活，希望它不会对世界或那些不应该看到的人充分展现。
  </p>
  <p class="graf graf--p">
   最后，还有我们的秘密生活，目前只能在离线状态下找到。实际上，您可能会感到奇怪：
   <strong class="markup--strong markup--p-strong">
    真的存在 “秘密” 生活吗？因为没有离线状态了。
   </strong>
   ⚪️
  </p>
  <p class="graf graf--p">

    Why We’re So Hypocritical About Online Privacy
   </a>
  </p>
