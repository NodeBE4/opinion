  <p class="graf graf--p">
   我想你已经学会了装瘸以应对步态识别、学会了装哑以绕开声音识别、学会了戴面具以避免面部识别、学会了带墨镜以防止虹膜识别 …… 但是你的手在哪儿？现在急于关注它了，因为另一种生物识别监视技术正在流行起来，即 手指静脉识别。
  </p>
  <p class="graf graf--p">
   你可以使用“slu”来购买一罐苏打水的日子已经一去不复返了，因为今天的自动售货机会
   <strong class="markup--strong markup--p-strong">
    在你购买物品之前就知道你是谁、关于你的一切包括你的犯罪记录
   </strong>
   。
  </p>
  <p class="graf graf--p">
   美国绿色自动售货机（
   <a class="markup--anchor markup--p-anchor" data-href="https://americangreen.com/" href="https://americangreen.com/" rel="noopener noreferrer" target="_blank">
    AGM
   </a>
   ）使用手指静脉读取器识别每个客户的身份。
  </p>
  <p class="graf graf--p">
   简介上说：“绿色智能系统是一个年龄验证系统，可以提供商品推销和销售年龄限制产品的机会。绿色技术界面是独一无二的，可以做一些非常酷的事，如静脉验证生物识别技术。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    客户必须首先向 AGM 注册其个人生物识别和身份信息，否则不被允许购买任何东西
   </strong>
   。
  </p>
  <p class="graf graf--p">
   是什么让这个报道如此令人不安？ — — 是这些巨头公司急于将生物识别扫描仪放在你生活中的各个角落。
  </p>
  <p class="graf graf--p">
   想象一下，在不久的将来，您的新车内置了
   <a class="markup--anchor markup--p-anchor" data-href="https://massprivatei.blogspot.com/2017/12/new-cars-to-be-equipped-with.html?token=APq4FmBXDpZCCD_Tk5XAyAcLy4ke-qgrOwJY00AnCjN_oiLfNkhOWW4dw6A5u5uEzptpPd1c3ZAgXTgGPK0gWcLIzKevnY-7KdNJC8R0i6XwVUq83Mfyda_sPLoVGxsLT833HncbZK8r&amp;postId=3842598436400995981&amp;type=POST" href="https://massprivatei.blogspot.com/2017/12/new-cars-to-be-equipped-with.html?token=APq4FmBXDpZCCD_Tk5XAyAcLy4ke-qgrOwJY00AnCjN_oiLfNkhOWW4dw6A5u5uEzptpPd1c3ZAgXTgGPK0gWcLIzKevnY-7KdNJC8R0i6XwVUq83Mfyda_sPLoVGxsLT833HncbZK8r&amp;postId=3842598436400995981&amp;type=POST" rel="noopener noreferrer" target="_blank">
    TSA 生物识别扫描仪
   </a>
   ，在您进入体育场之前监视者就可以识别您和您的家人。
  </p>
  <p class="graf graf--p">
   但是如果你不开车怎么办？没关系，TSA 也找到了可以识别你的方法。
  </p>
  <p class="graf graf--p">
   想象一下，您和您的家人决定不开车进入体育场，而是走到入口处。不用担心 CLEAR
   <a class="markup--anchor markup--p-anchor" data-href="https://massprivatei.blogspot.com/2019/03/mlb-going-full-blown-tsa-as-teams.html" href="https://massprivatei.blogspot.com/2019/03/mlb-going-full-blown-tsa-as-teams.html" rel="noopener noreferrer" target="_blank">
    也已经安装了
   </a>
   TSA 生物识别扫描仪。
  </p>
  <p class="graf graf--p">
   对购买年龄限制产品的人进行间谍活动是一件大事，AGM 和 CLEAR 正在全力以赴地获取您的个人信息。
  </p>
  <p class="graf graf--p">
   Fast Company 的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.fastcompany.com/40443614/only-in-america-youll-soon-be-able-to-buy-alcohol-weed-and-maybe-guns-from-a-vending-machine" href="https://www.fastcompany.com/40443614/only-in-america-youll-soon-be-able-to-buy-alcohol-weed-and-maybe-guns-from-a-vending-machine" rel="noopener noreferrer" target="_blank">
    一篇文章
   </a>
   警告称，生物识别自动售货机旨在使“特权”受益。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “吸引人的问题与成人使用有关，”AGM 顾问 Stephen Shearin 说，“但是，现实情况是，这是一个系统，它确切地说明了你的身份。”在校园环境中，某些产品如苏打水或糖果的出售可能会受到限制，尤其是对那些想要保持一贯好成绩的人来说。就像中国的信用评分的逻辑。
  </p>
  <p class="graf graf--p">
   去年，“今日美国
   <a class="markup--anchor markup--p-anchor" data-href="https://www.usatoday.com/story/money/business/2017/05/11/vending-machines-sell-booze-pot/101514340/" href="https://www.usatoday.com/story/money/business/2017/05/11/vending-machines-sell-booze-pot/101514340/" rel="noopener noreferrer" target="_blank">
    ”警告说
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    药店已经开始使用 AGM 的自动售货机识别患者。
   </strong>
  </p>
  <p class="graf graf--p">
   而且
   <strong class="markup--strong markup--p-strong">
    这种识别购物者的监视资本主义竞争已经变得非常糟糕
   </strong>
   ，像 Walgreens、雀巢、Coors 这样的零售商都想要使用 Iris 跟踪相机！
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当您意识到您的个人信息可能与全球数据库绑定时，使用生物识别技术创建社会“老大哥”的影响已经变得更加令人不安
   </strong>
   。第一个视频：
   <a class="markup--anchor markup--p-anchor" data-href="https://youtu.be/K_bt0GfG584" href="https://youtu.be/K_bt0GfG584" rel="noreferrer noopener" target="_blank">
    https://youtu.be/K_bt0GfG584
   </a>
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="314" src="//www.youtube.com/embed/K_bt0GfG584" width="560">
   </iframe>
  </p>
  <p class="graf graf--p">
   AGM 使用日立的非侵入式
   <a class="markup--anchor markup--p-anchor" data-href="http://www.m2sys.com/finger-vein-reader.htm" href="http://www.m2sys.com/finger-vein-reader.htm" rel="noopener noreferrer" target="_blank">
    M2-FingerVein
   </a>
   手指静脉识别装置，但
   <strong class="markup--strong markup--p-strong">
    他们不希望公众知道的是，它旨在创建一个全球性的生物识别数据库，用于执法强化
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Hitachi 的 M2SYS 下拉菜单名为“Industries”，它显示它是
   <a class="markup--anchor markup--p-anchor" data-href="http://www.m2sys.com/jail_management_software/" href="http://www.m2sys.com/jail_management_software/" rel="noopener noreferrer" target="_blank">
    专为执法部门设计的
   </a>
   ，可以识别囚犯等等。
  </p>
  <p class="graf graf--p">
   国际刑警组织、联邦调查局和苏格兰场的手指静脉阅读器 “
   <a class="markup--anchor markup--p-anchor" data-href="http://www.m2sys.com/elaw-enforcement-solutions-suite/" href="http://www.m2sys.com/elaw-enforcement-solutions-suite/" rel="noopener noreferrer" target="_blank">
    旨在满足政府执法机构的识别和数据管理要求
   </a>
   ”（要了解有关 M2SYS ePolice 的更多信息，请
   <a class="markup--anchor markup--p-anchor" data-href="http://www.m2sys.com/epolice-law-enforcement-operational-management-biometric-system/" href="http://www.m2sys.com/epolice-law-enforcement-operational-management-biometric-system/" rel="noopener noreferrer" target="_blank">
    单击此处
   </a>
   。）第二个视频：
   <a class="markup--anchor markup--p-anchor" data-href="https://youtu.be/p3mBA7tUQMg" href="https://youtu.be/p3mBA7tUQMg" rel="noreferrer noopener" target="_blank">
    https://youtu.be/p3mBA7tUQMg
   </a>
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="314" src="//www.youtube.com/embed/p3mBA7tUQMg" width="560">
   </iframe>
  </p>
  <p class="graf graf--p">
   日立的“
   <a class="markup--anchor markup--p-anchor" data-href="http://www.m2sys.com/biometric-fingerprint-software-industry-overview/" href="http://www.m2sys.com/biometric-fingerprint-software-industry-overview/" rel="noopener noreferrer" target="_blank">
    行业概览
   </a>
   ”揭示了手指静脉生物识别技术背后的真正含义。生物特征识别将用于跟踪我们所做的一切，无论是购买产品、上班、上医院、去银行、还是旅行。
  </p>
  <p class="graf graf--p">
   使用生物识别技术跟踪你所做的一切的老大哥之存在是令人恐惧的，所以，现在开始，在你决定把自己的手指送给巨头公司之前，最好考虑一下。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    F
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.blacklistednews.com/article/71913/finger-vein-vending-machines-and-a-global-biometric-police.html" href="https://www.blacklistednews.com/article/71913/finger-vein-vending-machines-and-a-global-biometric-police.html" rel="noopener noreferrer" target="_blank">
    INGER VEIN VENDING MACHINES AND A GLOBAL BIOMETRIC POLICE DATABASE
   </a>
   . The Big Brother implications of using biometrics to track everything we do is horrifying, so think about that before you give a corporation your finger vein.
  </p>
