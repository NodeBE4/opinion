  <p class="graf graf--p">
   他们一直在控制内容、控制你有机会认识什么样的人、有可能得到什么样的知识，他们一直在把我们每个人的私人生活转变为他们自己的巨额利润，把全球人变成了数字庄园里的奴隶。现在，正是他们，需要买断互联网本身。这意味着什么？
  </p>
  <p class="graf graf--p">
   谷歌从其云平台赚取了数十亿美元。现在，该公司正在利用这数十亿美元购买互联网本身 — — 或至少是构成互联网骨干网的海底光缆。
  </p>
  <p class="graf graf--p">
   2月，该公司宣布打算继续开发
   <a class="markup--anchor markup--p-anchor" data-href="https://t.sidekickopen10.com/s2t/c/5/f18dQhb0S7lC8dDMPbW2n0x6l2B9nMJN7t5XZsQsW6YW2m2NbT3QZmLFN3J30yY19JW9f7zJflM03?t=https%3A%2F%2Fwww.prnewswire.com%2Fnews-releases%2Fgoogle-selects-equinix-for-curie-subsea-cable-landing-station-300791414.html&amp;si=5807407437185024&amp;pi=ebb27c33-ff41-493c-8d13-f7da2f996c55" href="https://t.sidekickopen10.com/s2t/c/5/f18dQhb0S7lC8dDMPbW2n0x6l2B9nMJN7t5XZsQsW6YW2m2NbT3QZmLFN3J30yY19JW9f7zJflM03?t=https%3A%2F%2Fwww.prnewswire.com%2Fnews-releases%2Fgoogle-selects-equinix-for-curie-subsea-cable-landing-station-300791414.html&amp;si=5807407437185024&amp;pi=ebb27c33-ff41-493c-8d13-f7da2f996c55" rel="noopener noreferrer" target="_blank">
    Curie cable
   </a>
   ，这是一条从加利福尼亚到智利的新海底线路。这将是由一家大型非电信公司建造的第一条私人洲际电缆。在过去几年里谷歌已经完全资助了其中的一些洲际电缆。
  </p>
  <p class="graf graf--p">
   不是只有谷歌在买断互联网。从历史上看，海底光缆一直由私营公司集团（大多数是电信供应商）所拥有，但2016年大规模海底光缆业开始蓬勃发展，而这一次，买家都是内容提供商。像
   <a class="markup--anchor markup--p-anchor" data-href="https://venturebeat.com/2018/09/06/facebook-is-building-a-1553-mile-subsea-cable-to-boost-internet-speeds-in-argentina/" href="https://venturebeat.com/2018/09/06/facebook-is-building-a-1553-mile-subsea-cable-to-boost-internet-speeds-in-argentina/" rel="noopener noreferrer" target="_blank">
    Facebook
   </a>
   、
   <a class="markup--anchor markup--p-anchor" data-href="https://venturebeat.com/2017/09/22/microsoft-and-facebook-backed-marea-trans-atlantic-cable-is-now-complete/" href="https://venturebeat.com/2017/09/22/microsoft-and-facebook-backed-marea-trans-atlantic-cable-is-now-complete/" rel="noopener noreferrer" target="_blank">
    微软
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://t.sidekickopen10.com/s2t/c/5/f18dQhb0S7lC8dDMPbW2n0x6l2B9nMJN7t5XZsQsW6YW2m2NbT3QZmLFN3J30yY19JW9f7zJflM03?t=https%3A%2F%2Fwww.ntt.com%2Fen%2Fabout-us%2Fpress-releases%2Fnews%2Farticle%2F2017%2F1030.html&amp;si=5807407437185024&amp;pi=ebb27c33-ff41-493c-8d13-f7da2f996c55" href="https://t.sidekickopen10.com/s2t/c/5/f18dQhb0S7lC8dDMPbW2n0x6l2B9nMJN7t5XZsQsW6YW2m2NbT3QZmLFN3J30yY19JW9f7zJflM03?t=https%3A%2F%2Fwww.ntt.com%2Fen%2Fabout-us%2Fpress-releases%2Fnews%2Farticle%2F2017%2F1030.html&amp;si=5807407437185024&amp;pi=ebb27c33-ff41-493c-8d13-f7da2f996c55" rel="noopener noreferrer" target="_blank">
    亚马逊
   </a>
   这样的公司似乎都在和谷歌争抢着对互联网支配地位的渴望。
  </p>
  <p class="graf graf--p">
   我一直在观察这种趋势的发展，最近这些巨头的动作肯定是令人担忧的。
  </p>
  <p class="graf graf--p">
   科技寡头们对互联网骨干网的所有权将产生深远的影响这并不陌生。消费者一直在放弃控制权 — — 放弃隐私人权。
  </p>
  <p class="graf graf--p">
   我们正在进入互联网成熟的下一个阶段：只有那些那些耀眼的巨头才能在媒体上真正获胜。
  </p>
  <p class="graf graf--p">
   消费者很快就需要确定他们究竟可以在这些巨头公司身上建立多少信心以推测明天的互联网是什么样。而这些监视资本家正在窃取我们的私人生活并将其转化为不断增长的利润。这事你的确需要仔细权衡。每个人都需要。
  </p>
  <p class="graf graf--p">
   这些不起眼的电缆横跨全球的海底，传输95–99％的国际数据。总而言之，目前使用的海底电缆长度超过
   <a class="markup--anchor markup--p-anchor" data-href="https://t.sidekickopen10.com/s2t/c/5/f18dQhb0S7lC8dDMPbW2n0x6l2B9nMJN7t5XZsQsW6YW2m2NbT3QZmLFN3J30yY19JW9f7zJflM03?t=https%3A%2F%2Fwww.nec.com%2Fen%2Fglobal%2Fabout%2Fmitatv%2F02%2F&amp;si=5807407437185024&amp;pi=ebb27c33-ff41-493c-8d13-f7da2f996c55" href="https://t.sidekickopen10.com/s2t/c/5/f18dQhb0S7lC8dDMPbW2n0x6l2B9nMJN7t5XZsQsW6YW2m2NbT3QZmLFN3J30yY19JW9f7zJflM03?t=https%3A%2F%2Fwww.nec.com%2Fen%2Fglobal%2Fabout%2Fmitatv%2F02%2F&amp;si=5807407437185024&amp;pi=ebb27c33-ff41-493c-8d13-f7da2f996c55" rel="noopener noreferrer" target="_blank">
    700,000英里
   </a>
   。
  </p>
  <p class="graf graf--p">
   虽然过去的电缆制造商利用电缆所有权来销售带宽，但现在，内容提供商正在有目的地建造私人电缆。
  </p>
  <p class="graf graf--p">
   互联网通常被描述为“云”。实际上，它是一系列潮湿易碎的管子，Google 即将拥有其惊人的数量。这些数字不言自明; 当
   <a class="markup--anchor markup--p-anchor" data-href="https://t.sidekickopen10.com/s2t/c/5/f18dQhb0S7lC8dDMPbW2n0x6l2B9nMJN7t5XZsQsW6YW2m2NbT3QZmLFN3J30yY19JW9f7zJflM03?t=https%3A%2F%2Fwww.submarinenetworks.com%2Fen%2Fsystems%2Fbrazil-us%2Fcurie&amp;si=5807407437185024&amp;pi=ebb27c33-ff41-493c-8d13-f7da2f996c55" href="https://t.sidekickopen10.com/s2t/c/5/f18dQhb0S7lC8dDMPbW2n0x6l2B9nMJN7t5XZsQsW6YW2m2NbT3QZmLFN3J30yY19JW9f7zJflM03?t=https%3A%2F%2Fwww.submarinenetworks.com%2Fen%2Fsystems%2Fbrazil-us%2Fcurie&amp;si=5807407437185024&amp;pi=ebb27c33-ff41-493c-8d13-f7da2f996c55" rel="noopener noreferrer" target="_blank">
    Curie cable
   </a>
   在今年晚些时候完成时，谷歌将在国际上拥有10,433英里的海底电缆。
  </p>
  <p class="graf graf--p">
   当 Facebook、微软和亚马逊这些财团都加入进来时，总数将达到 63,605 英里。
  </p>
  <p class="graf graf--p">
   当您观察过去十年中发生的流量增长时，就会发现这种海底电缆的所有权问题非常有意义。
  </p>
  <p class="graf graf--p">
   在大西洋和太平洋地区，内容提供商占2017年总需求的一半以上。在过去10年中，内容提供商数据使用率从不到8％飙升至接近40％。
  </p>
  <p class="graf graf--p">
   值得注意的是，非洲和中东的统计数据明显较低，这表明发达国家对视频内容和云应用程序的渴望是这一趋势的更大推动因素。这得到了各国之间国际带宽总体使用数据的支持。 2017年，印度仅使用了 4,977 Mbps 的国际带宽；而同年，美国使用了惊人的 4,960,388 Mbps。
  </p>
  <p class="graf graf--p">
   就像网络中立的消除一样，互联网基础设施私有化只会降低价格。但我们现在面临的问题是道德问题：我们想要私人互联网吗？或者说是否希望保留迄今为止所拥有的“狂野西部”式的网络？
  </p>
  <p class="graf graf--p">
   不幸的是，问题并不像在“好”和“坏”之间画一条线那么简单。边缘网络和零费率等实践对 Netflix 和 AT＆T 等公司的商业模式至关重要 — 它们在技术上也不违反规则，最终可以为消费者提供更好的服务。
  </p>
  <p class="graf graf--p">
   但是，是时候问这个问题了：如果那些因审查和监视而闻名的内容服务也控制着互联网骨干网络，互联网将会是个什么样子。私有化基础设施可能在短期内为消费者带来无尽的好处，但是，我们难道就不用考虑成本吗？
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    G
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://venturebeat.com/2019/04/06/google-and-other-tech-giants-are-quietly-buying-up-the-most-important-part-of-the-internet/" href="https://venturebeat.com/2019/04/06/google-and-other-tech-giants-are-quietly-buying-up-the-most-important-part-of-the-internet/" rel="noopener noreferrer" target="_blank">
    oogle and other tech giants are quietly buying up the most important part of the internet
   </a>
   . Google isn’t alone. Corporations like Facebook, Microsoft, and Amazon all seem to share Google’s aspirations for bottom-of-the-ocean dominance.
  </p>
