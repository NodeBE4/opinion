  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    查找电子邮件地址
   </strong>
  </h3>
  <p class="graf graf--p">
   找到某个人或某个机构的电邮地址是调查的重点之一。在很多情况下，直接询问对方可能并不是很方便  — — 不光低效，还可能暴露你，让被调查的对象对你的行动有所察觉。
  </p>
  <p class="graf graf--p">
   但您可以借助
   <a class="markup--anchor markup--p-anchor" data-href="https://hunter.io/" href="https://hunter.io/" rel="noopener noreferrer" target="_blank">
    Hunter.io
   </a>
   ，它可以悄无声息地帮你找到被调查对象的电邮地址。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://hunter.io/" href="https://hunter.io/" rel="noopener noreferrer" target="_blank">
    Hunter.io
   </a>
   使用起来很简单，只需要在搜索框输入你要检索的域名，它就可以找到在互联网公开信息中与此域名关联的邮箱。
  </p>
  <p class="graf graf--p">
   比如，搜索 “microsoft.com”，就会出现近两万个相关联的邮箱地址，以及显示这些邮箱地址的网页链接。
  </p>
  <p class="graf graf--p">
   除了查找电邮地址外，
   <a class="markup--anchor markup--p-anchor" data-href="https://hunter.io/" href="https://hunter.io/" rel="noopener noreferrer" target="_blank">
    Hunter.io
   </a>
   还可以
   <a class="markup--anchor markup--p-anchor" data-href="https://hunter.io/email-verifier" href="https://hunter.io/email-verifier" rel="noopener noreferrer" target="_blank">
    批量验证
   </a>
   邮箱地址是否可用，同时它还有一个
   <a class="markup--anchor markup--p-anchor" data-href="https://chrome.google.com/webstore/detail/hunter-find-email-address/hgmhmanijnjhaffoampdlllchpolkdnj" href="https://chrome.google.com/webstore/detail/hunter-find-email-address/hgmhmanijnjhaffoampdlllchpolkdnj" rel="noopener noreferrer" target="_blank">
    Chrome 扩展
   </a>
   ，在你访问网页的时候方便查询到与此网页关联的电邮地址。
  </p>
  <p class="graf graf--p">
   如果您的查询请求少于50次/月，那么不用付费就可以使用
   <a class="markup--anchor markup--p-anchor" data-href="https://hunter.io/" href="https://hunter.io/" rel="noopener noreferrer" target="_blank">
    Hunter.io
   </a>
   。
  </p>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    查找用户名
   </strong>
  </h3>
  <p class="graf graf--p">
   另一个在线调查的小技巧就是找到信息之间的联系。
  </p>
  <p class="graf graf--p">
   比如说，您可能知道一个用户在某个网站上的用户名，然后通过搜索用户名，就可以找到这个用户注册的其他互联网服务。
  </p>
  <p class="graf graf--p">
   即使您不知道准确的用户名，也可以通过尝试不同组合的搜索关键词来确定用户名。

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    同时检索多个数据参数
   </strong>
  </h3>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   这类工具中最受欢迎的是
