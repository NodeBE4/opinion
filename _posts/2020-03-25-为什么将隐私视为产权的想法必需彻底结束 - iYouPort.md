     <em>
      <strong>
       最近，一位在中国经商的读者对我说：“微信是我们这些人的生命线，如果离开它就相当于切断了收入来源”。他是通过signal对我说的这句话，因为他知道，联系我最安全的方式是什么 …… 这让我想起立法界最近正在努力推动的奇怪的产权模型。我们需要对隐私的概念有一个更加科学和理性的思考。
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>

  </p>
  <p class="graf graf--p">
   这一讨论实际上已经进行一段时间了，因为我们大多数人并不真正了解隐私，因此我们在监管隐私方面确实很糟糕。
  </p>
  <p class="graf graf--p">
   人们倾向于将隐私视为“事物”。但它并不是。它应该是一组权衡，可以根据涉及的人、习惯性、以及权衡的条件而改变。
  </p>
  <p class="graf graf--p">
   我们多次使用的示例是离开家购买食品杂货。这样做需要放弃一些隐私，因为有人可以看到你。他们甚至可能看到您购物车中的东西。但是对于大多数人来说，这种权衡是值得的。
  </p>
  <p class="graf graf--p">
   这里的隐私“损失”很小。被他人看到您购买了西兰花也没什么大不了的。
   <strong class="markup--strong markup--p-strong">
    但是，对于某些人而言，权衡可能会大不相同
   </strong>
   。
  </p>
  <p class="graf graf--p">
   例如，如果您是电影明星，那么进入杂货店可能会带来巨大负担，并会影响您的隐私。狗仔队可能会跟踪你、其他客户可能会打扰你，您购买的商品可能会被无聊的新闻头条分析，被嘲笑、甚至更糟 ……
  </p>
  <p class="graf graf--p">
   其他因素也会起作用，例如您要购买的是什么？西兰花可能没什么大不了的，但是，
   <strong class="markup--strong markup--p-strong">
    如果买的是其他东西，有可能会因为被分析被联想而导致非常危险的结果
   </strong>
   —— 虽然你只是日常使用。比如：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%b9%b0%e4%b8%80%e6%8a%8a%e9%94%a4%e5%ad%90%e5%b0%b1%e4%bc%9a%e8%83%bd%e8%ae%a9%e4%bd%a0%e5%8f%98%e6%88%90%e6%81%90%e6%80%96%e5%88%86%e5%ad%90/" href="https://www.iyouport.org/%e4%b9%b0%e4%b8%80%e6%8a%8a%e9%94%a4%e5%ad%90%e5%b0%b1%e4%bc%9a%e8%83%bd%e8%ae%a9%e4%bd%a0%e5%8f%98%e6%88%90%e6%81%90%e6%80%96%e5%88%86%e5%ad%90/" rel="noopener noreferrer" target="_blank">
     买一把锤子就会能让你变成“恐怖分子”
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   以上只是非常简单的道理，但是它在很多情况下都适用。
   <strong class="markup--strong markup--p-strong">
    我们做出的许多决定都涉及有关隐私的基本权衡。
   </strong>
  </p>
  <p class="graf graf--p">
   我们所有人都隐含地进行着各种计算，涉及相当简单的成本效益分析。我们从x中获得的价值是否大于潜在的侵犯隐私损失？
  </p>
  <p class="graf graf--p">
   而且，当然，“成本”计算通常会变得更加困难，其中一些不透明的概率在起作用。
  </p>
  <p class="graf graf--p">
   例如，某人进入药店购买避孕套可能会略有失去隐私权 —— 但如果此人这样做是有外遇，那么“代价”可能是被此人的伴侣发现。
  </p>
  <p class="graf graf--p">
   问题在于，如果你将隐私仅视为必须“保护”的“事物”，往往无法考虑到各种折衷的细微差别。
  </p>
  <p class="graf graf--p">
   它不能解释这样一个事实，即 在相同的情况下，不同的人可能会有不同的权衡、折衷不同的方面，并且可能对可接受和不可接受的权衡有着完全不同的信念。
  </p>
  <p class="graf graf--p">
   在此基础上，我们今天遇到的有关“隐私”的真正问题是，我们常常对权衡方程的两面都不了解。
  </p>
  <p class="graf graf--p">
   一些人只是对互联网公司疯狂吸取所有人的数据而感到担忧或不安，但尚不完全清楚自己损失的成本是多少。
  </p>
  <p class="graf graf--p">
   没有足够的透明度，也没有足够的能力来准确了解实际风险，因此，我们经常使权衡决策变得有些盲目。
  </p>
  <p class="graf graf--p">
   很多人似乎认为让 Facebook 或微信吸收所有数据是与自己的家人和朋友保持联系的一项 “有价值的交易”。而另一些人对此仍一无所知。但是部分问题是，成本充其量是无定形的，而对许多人来说却有可见的实惠。
  </p>
  <p class="graf graf--p">
   尽管如此，对于收集哪些数据以及如何使用数据缺乏透明度，再加上最终用户完全没有控制权，给某些人造成紧张感是完全合理的。
  </p>
  <p class="graf graf--p">
   问题是成本可能会很高。我们可能真的没有任何办法对此做任何事。这就是大多数人担心社交媒体对隐私影响的主要视点。
  </p>
  <p class="graf graf--p">
   鉴于如今人们对“调节”隐私非常感兴趣，人们用来理解隐私的模型可能会产生很大的影响。
  </p>
  <p class="graf graf--p">
   使用错误的模型将导致非常糟糕的决定。
  </p>
  <p class="graf graf--p">
   不幸的是，最糟糕的想法之一却是最流行的：即 将“隐私”转变为准知识产权的想法。也就是将其设置为好像带有价格的“财产”一样。可悲的是，这种模型在法规方面有很多支持者。
  </p>
  <p class="graf graf--p">
   《纽约时报》最近发表了一篇由 Sarah Jeong 撰写的意见，解释了为什么将隐私设置为产权是一个
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2019/07/05/opinion/health-data-property-privacy.html" href="https://www.nytimes.com/2019/07/05/opinion/health-data-property-privacy.html" rel="noopener noreferrer" target="_blank">
    糟糕的主意
   </a>
   。
  </p>
  <p class="graf graf--p">
   这篇评论文章发表在 Caming Kerry 和 John Morris 的一篇类似（甚至更详尽）的文章之后的大约一个星期，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.brookings.edu/blog/techtank/2019/06/26/why-data-ownership-is-the-wrong-approach-to-protecting-privacy/" href="https://www.brookings.edu/blog/techtank/2019/06/26/why-data-ownership-is-the-wrong-approach-to-protecting-privacy/" rel="noopener noreferrer" target="_blank">
    那篇文章
   </a>
   同样地解释了为什么
   <strong class="markup--strong markup--p-strong">
    数据所有权是保护隐私的“错误方法”
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%95%b0%e6%8d%ae%e6%89%80%e6%9c%89%e6%9d%83%e4%b8%8d%e4%bb%85%e6%97%a0%e6%b3%95%e8%a7%a3%e5%86%b3%e7%8e%b0%e6%9c%89%e9%97%ae%e9%a2%98%ef%bc%8c%e5%ae%83%e8%bf%98%e5%88%b6%e9%80%a0/" href="https://www.iyouport.org/%e6%95%b0%e6%8d%ae%e6%89%80%e6%9c%89%e6%9d%83%e4%b8%8d%e4%bb%85%e6%97%a0%e6%b3%95%e8%a7%a3%e5%86%b3%e7%8e%b0%e6%9c%89%e9%97%ae%e9%a2%98%ef%bc%8c%e5%ae%83%e8%bf%98%e5%88%b6%e9%80%a0/" rel="noopener noreferrer" target="_blank">
     “数据所有权”不仅无法解决现有问题，它还制造了新的问题
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bb%a5%e9%9a%90%e7%a7%81%e4%b8%ba%e7%9b%ae%e7%9a%84%e7%9a%84%e6%96%97%e4%ba%89%e5%a6%82%e4%bd%95%e8%ae%a9%e4%bd%a0%e5%8f%98%e6%88%90%e4%ba%86%e7%9b%91%e8%a7%86%e7%9a%84%e7%9b%9f%e5%8f%8b/" href="https://www.iyouport.org/%e4%bb%a5%e9%9a%90%e7%a7%81%e4%b8%ba%e7%9b%ae%e7%9a%84%e7%9a%84%e6%96%97%e4%ba%89%e5%a6%82%e4%bd%95%e8%ae%a9%e4%bd%a0%e5%8f%98%e6%88%90%e4%ba%86%e7%9b%91%e8%a7%86%e7%9a%84%e7%9b%9f%e5%8f%8b/" rel="noopener noreferrer" target="_blank">
     以隐私为目的的斗争如何让你变成了监视的盟友
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   一些政策制定者正在认真考虑这种想法。参议员 John Kennedy（R-LA）提出了一项三页的法案，即 “2019年拥有自己的数据法案”，该法案宣布“每个人都对其在互联网上生成的数据拥有专有财产权”，要求社交媒体公司获得使用此数据的许可。
  </p>
  <p class="graf graf--p">
   参议员 Mark Warner（D-VA）和 Josh Hawley（Josh Hawley）（R-MO）正在提交立法，要求 Facebook、Google和其他大型数据收集者披露他们收集的个人数据的价值，尽管该法案不要求付款。
  </p>
  <p class="graf graf--p">
   在加利福尼亚州，州长 Gavin Newsome 希望推行“数据红利”，旨在 “分享由[人民]数据创造的财富”。
  </p>
  <p class="graf graf--p">
   但是正如 Kerry / Morris 所说，以这种方式处理私人数据会遇到各种各样的问题 —— 包括与第一修正案的冲突：
  </p>
  <p class="graf graf--p">
   问题是，这不是您的数据；当然也不是监视巨头的数据。
   <strong class="markup--strong markup--p-strong">
    像对待财产一样对待数据，即使人们选择“出售”，也无法认识到各种个人信息的价值或个人对其私密信息的持久兴趣。数据不是商品，它是信息。
   </strong>
  </p>
  <p class="graf graf--p">
   任何信息权制度，无论是专利、版权、其他知识产权还是隐私权，都会对第一修正案所反映的信息自由流通表现出明显的冲突。
  </p>
  <p class="graf graf--p">
   对我们的个人信息的需求正是因为它在众多用途中对他人和社会都具有价值。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    将个人信息视为要获得许可或出售的财产，可能会导致人们以极少的价值折衷自己的隐私权，同时给信息的自由流通带来巨大的阻力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   加强隐私的更好方法是确保在个人信息流向所需用途时尊重个人隐私利益，而不是将个人数据贬低为商品。
  </p>
  <p class="graf graf--p">
   同样，Jeong 的著作着重强调了将隐私视为财产权的几个问题：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    产权是可转让的 —— 一旦您出售房屋，产权就消失了。但是最基本的人权是不可剥夺的，通常是因为一旦权利被剥夺，这些权利就变得毫无意义
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果可以出售，生命和自由的意义何在？
   </strong>
  </p>
  <p class="graf graf--p">
   Jeong 还提出了一个重要的观点：您的“私有”数据实际上也可能是其他人的私有数据，这带来了一些更严重的麻烦：
  </p>
  <p class="graf graf--p">
   您的位置数据可以透露您的配偶的下落；您的健康记录会泄露有关您的孩子的信息；如果您出售自己的DNA隐私，您的父母有权获得分成吗？
  </p>
  <p class="graf graf--p">
   您是否需要获得配偶的批准才能出售位置信息？您是否需要未出生的后代的许可才能出售您的遗传信息？……
  </p>
  <p class="graf graf--p">
   Kerry / Morris 的文章还强调了通常来说对于某些信息的共享、不计价格，对整个社会的有用性。
  </p>
  <p class="graf graf--p">
   对所有数据定价，会给基本的交易带来不必要的摩擦；而且，将一切都转换为商品/产权模型将带来相当可观的成本。
   <strong class="markup--strong markup--p-strong">
    其中一些人完全 “买得起”，而其他许多人实际上对保护“隐私”无能为力。
   </strong>
  </p>
  <p class="graf graf--p">
   当前的 “知情同意”模型失败了，因为用户实际上无法理解他们的数据将被如何使用或随之而来的隐私风险，尤其是在当今互联世界中在线交流不断发展的情况下。
  </p>
  <p class="graf graf--p">
   结果是人们直接点击隐私权声明的 “同意”，直达他们想要的信息或服务。
  </p>
  <p class="graf graf--p">
   就像 Je​​ong 所指出的那样，
   <strong class="markup--strong markup--p-strong">
    这样的系统可能只是鼓励人们 —— 尤其是那些最脆弱的人们 —— 有效地“出售”自己的人权
   </strong>
   。
  </p>
  <p class="graf graf--p">
   美国公民自由联盟称该法案为特洛伊木马。
  </p>
  <p class="graf graf--p">
   电子前沿基金会表示，“该法律会通过特别鼓励弱势低收入人群将更多的个人信息注入一个利用和歧视他们的行业，来激励人们放弃其基本的隐私权，加剧不平等。”
  </p>
  <p class="graf graf--p">
   正如 Kerry / Morris 指出的那样：几乎任何个人的私人数据的“价值”都肯定是非常低的：
  </p>
  <p class="graf graf--p">
   英国《金融时报》
   <a class="markup--anchor markup--p-anchor" data-href="https://ig.ft.com/how-much-is-your-personal-data-worth/" href="https://ig.ft.com/how-much-is-your-personal-data-worth/" rel="noopener noreferrer" target="_blank">
    有一个计算器
   </a>
   。默认值为0.007美元，但作为一个经常旅行的富裕人士，Kerry 数据的价值估计为1.78美元。如果价格由服务提供商确定，那么最终的系统很可能会与当前的 “知情同意” 结果非常相似。
  </p>
  <p class="graf graf--p">
   如果定价是由消费者或通过谈判确定的，则服务-用户交互的复杂性将更大。这种新的复杂性可能会减慢用户对所需信息和服务的访问，并将“点击疲劳”转变为“协商疲劳”。
  </p>
  <p class="graf graf--p">
   当然，政府可以设定价格，但是有人认为他们会以任何有意义的方式设定价格吗？
  </p>
  <p class="graf graf--p">
   甚至这还没有涉及多方数据的问题 —— 例如电子商务交易：
  </p>
  <p class="graf graf--p">
   在信息作为财产的制度下，购买者和零售商是否都拥有有关交易信息的产权？在这样的财产制度下，零售商难道不能简单地将购买者必须授予零售商许​​可的条件作为销售条件，以将信息用于指定用途？
  </p>
  <p class="graf graf--p">
   难道这不是仅仅导致另一种形式的专制暴政吗，即 想要方便在线购买的购买者被迫将权利转让给零售商。目前尚不清楚信息作为财产制度是否会真正改善当前的隐私状态。
  </p>
  <p class="graf graf--p">
   有很多合理的理由（以及一些不太合理的理由）来关注当今的隐私状态。但是，如果“解决方案”是在某种产权制度下将您的数据转变为商品，我们将制造更多的问题，而不是解决问题。
  </p>
  <p class="graf graf--p">
   事实上我们之前走过这条愚蠢的路，这一愚蠢的想法是根据版权法将可以被无限复制的“内容”视为“财产”。
  </p>
  <p class="graf graf--p">
   这样就产生了一个巨大的混乱局面，尤其是对于言论自由来说，这种想法就是可以以某种形式拥有或限制表达。
  </p>
  <p class="graf graf--p">
   确实，过去几十年中有关互联网的许多权利辩论实际上都代表着人们正在进行的努力，即 人们正在努力认识到版权法规定的“内容即财产”制度如何与互联网的“用于通信和共享内容的全球网络”概念并存。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e8%af%b4%e7%9f%a5%e8%af%86%e4%ba%a7%e6%9d%83%e5%8f%aa%e6%98%af%e4%b8%80%e7%a7%8d%e6%84%9a%e8%a0%a2%e7%9a%84%e5%a7%94%e5%a9%89%e8%af%b4%e8%be%9e/" href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e8%af%b4%e7%9f%a5%e8%af%86%e4%ba%a7%e6%9d%83%e5%8f%aa%e6%98%af%e4%b8%80%e7%a7%8d%e6%84%9a%e8%a0%a2%e7%9a%84%e5%a7%94%e5%a9%89%e8%af%b4%e8%be%9e/" rel="noopener noreferrer" target="_blank">
     为什么说“知识产权”只是一种愚蠢的委婉说辞
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   我们不会通过在“隐私”周围添加组成“财产”的另一类别对任何人有所帮助（除了一些律师之外）。而且，最糟糕的是，对于大多数人而言，实际上无法改善隐私效果。⚪️
  </p>
