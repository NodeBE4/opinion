       2014年（6年前），我的一位朋友在辩论中就这样怒吼：“民主都快死了！你们变革能变成什么？我们需要先抓住重点！” 当时没有人能听懂他说的是什么意思。直到最近两年，一些人开始醒悟。当然，他所指的远非这篇文章主题所强调的内容，这里只是 **一个侧面**……

  <p class="graf graf--p">
   人们不是不希望保护自己，而是感觉那些 *真正* 能起到保护作用的方法 “根本不现实”。比如，
   <strong class="markup--strong markup--p-strong">
    最基本的保护性措施就是：扔掉你的智能手机
   </strong>
   。
  </p>
  <p class="graf graf--p">
   真话说出来就是这样。而漂亮的假话可以说一大堆。
  </p>
  <p class="graf graf--p">
   但是人们依旧想要知道 “如何更改一些设置以缓解灾难”。没错，方法的确有，但是，可能无法解决根本性问题。
  </p>
  <p class="graf graf--p">
   就如 deleteFacebook 运动中很多人寻求的依旧是如何调整隐私设置，而不是如何彻底删除它。
  </p>
  <p class="graf graf--p">
   我们已经被绑架了，还患上了斯德哥尔摩综合征，甚至不愿逃离。结果是什么？毁掉民主的就是每个人口袋里的东西。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="659" src="//player.vimeo.com/video/381288444?title=0&amp;byline=0&amp;portrait=0&amp;color=8dc7dc" width="800">
   </iframe>
  </p>
  <p class="graf graf--p">
   夜幕降临时，香港的街道几乎到处都是灯光，成千上万的手机被人们举起在头顶摇曳。每一个闪烁都是出席的标志，也是对自由的恳求。
  </p>
  <p class="graf graf--p">
   一些抗议者戴着面具以阻止被政府的面部识别监视摄像头网络捕获。
  </p>
  <p class="graf graf--p">
   但是，除了明亮的灯光外，每部手机还在黑暗中发出另一个信标 —— 人眼看不见的信标。
  </p>
  <p class="graf graf--p">
   有时每分钟多次捕获和收集此信号，不是通过无人机、而是通过智能手机应用程序。
   <strong class="markup--strong markup--p-strong">
    在抗议者关掉相机灯、回到家中并脱下口罩之后很久，这些信号一直在广播。
   </strong>
  </p>
  <p class="graf graf--p">
   在美国乃至世界各地，
   <strong class="markup--strong markup--p-strong">
    几乎所有国家都会追踪任何将智能手机带到公开示威活动中的示威者，并将目标人在行动中的身影记录在商业数据集中。同时，各政党一直在收集和购买电话位置数据以进行针对性的选民说服
   </strong>
   。
  </p>

  <p class="graf graf--p">
   哥伦比亚大学新闻学教授，60年代著名的激进主义者 “学生争取民主社会” 组织主席 Todd Gitlin 说：“毫无疑问，这是险恶的。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “这将有效地压制那些宪法声称保护的自由表达。如果人们知道自己会被跟踪，那么肯定会让他们退却。”
   </strong>
  </p>
  <p class="graf graf--p">
   Times Opinion 获得了来自超过1200万美国人的电话中的超过
   <strong class="markup--strong markup--p-strong">
    500亿
   </strong>
   个位置的大量数据，这有助于说明这种全面监控给公民建立和参与健康民主的权利带来的巨大风险。
  </p>
  <p class="graf graf--p">
   ⚠️无需任何特殊培训，在几分钟之内，只需进行少量的Google搜索，就可以在各个海岸的大型和小型公共示威活动中找出并识别出任何人。
  </p>
  <p class="graf graf--p">
   ⚠️ 通过跟踪特定的设备，我们跟踪了从2017年 Women’s March 大型游行中返回家的所有示威者；并且在2017年就职日 Black Bloc 抗议活动中识别出了任何参与的个人。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="659" src="//player.vimeo.com/video/381292422?title=0&amp;byline=0&amp;portrait=0&amp;color=8dc7dc" width="800">
   </iframe>
  </p>
  <p class="graf graf--p">
   示威者的匿名性长期以来一直是一个有争议的问题。
   <strong class="markup--strong markup--p-strong">
    各国政府普遍不喜欢这个主意
   </strong>
   ，他们声称担心蒙面抗议者更有可能煽动骚乱。
  </p>
  <p class="graf graf--p">
   👉 包括纽约州和佐治亚州在内的多个州都制定了禁止在公共示威活动中戴口罩的法律。
  </p>
  <p class="graf graf--p">
   👉 包括加拿大和西班牙在内的国家/地区都制定了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2017/04/26/us/protests-masks-laws.html" href="https://www.nytimes.com/2017/04/26/us/protests-masks-laws.html" rel="noopener noreferrer" target="_blank">
    限制或禁止
   </a>
   在所谓的非法聚会中戴口罩的规则。
  </p>
  <p class="graf graf--p">
   但是，你必须知道，
   <strong class="markup--strong markup--p-strong">
    在智能手机时代 —— 无论是否蒙面 —— 已经没有人能够迷失在茫茫人海中了。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️请注意以下噩梦：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    政府使用位置数据识别重大抗议活动中的政治敌人；
   </li>
   <li class="graf graf--li">
    检察官或警察使用位置信息以威吓刑事被告进行认罪交易；
   </li>
   <li class="graf graf--li">
    广告定位技术公司的流氓员工与出于政治动机的团体共享原始数据；
   </li>
   <li class="graf graf--li">
    大型捐赠者购买了监视定位公司，以帮助增强其政党的政治定位能力，并利用这些信息来压制抗议者；
   </li>
   <li class="graf graf--li">
    白人至上主义者组织入侵了一家小型企业的不安全服务器，并了解了潜在攻击目标的家乡地址 ……
   </li>

  <p class="graf graf--p">
   香港中文大学的活动家，研究员兼教授 Lokman Tsui 告诉我们，出售此数据的第三方是一个问题，因为 “购买此信息的标准非常宽松，公司没有道德审查委员会。我所在的大学
   <strong class="markup--strong markup--p-strong">
    就可以购买数据，而且很容易获得。他们可以购买的数据种类让我大为惊讶
   </strong>
   ，“哦，哇，你居然能买到那种东西？”令人毛骨悚然的数据。”
  </p>
  <p class="graf graf--p">
   ⚠️ 这些数据甚至都不需要泄漏或转手 —— 它的存在本身就已经给民主参与带来令人生畏的影响。
  </p>

  <p class="graf graf--p">
   话语已经在更专业的抗议者圈子中流传开来 —— 将手机留在家中、将其切换为飞行模式、或仅将其关闭电源（关闭电源没用，除非能把电池抠出来）。
  </p>
  <p class="graf graf--p">
   许多反法西斯抗议者参加的集会都以掩面为基础，以保护自己的身份免受仇恨团体、警察和新闻界的袭击。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “ 我们告诉人们不要将您的手机带到抗议活动中，或者如果您这样做了，请至少保持GPS处于关闭状态……”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “然而，要知道，您自己越安全，组织起来的能力就越差，”
   </strong>
   反法西斯抗议研究员说。
  </p>
  <p class="graf graf--p">
   活动家 Joshua Wong 曾协助推动香港2014年雨伞运动，现在是香港持续抗议活动的关键人物。他说，无所不在的追踪从根本上改变了香港的民主抗议活动。
  </p>
  <p class="graf graf--p">
   他说：
   <strong class="markup--strong markup--p-strong">
    “过去，人们觉得，如果自己不是激进主义者或知名人物，就不会受到监视，因为他们没有什么可隐藏的。但是，他们最近才意识到，随着抗议活动越来越多，监视对所有人构成了严重的威胁，即使他们不引人注目，政府也将其作为攻击目标。”
   </strong>
  </p>
  <p class="graf graf--p">
   即使是公开身份参加抗议活动的人，也会对自己的运动被归类到可购买、可出售、可合并、可被黑客入侵的数据库中，而感到不安。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我个人很高兴向特朗普提出抗议，并让人们知道这一点。但是有很多人说，
   <strong class="markup--strong markup--p-strong">
    国家间谍可以仅凭行程模式就可以确定你的身份。老实说，这很可怕
   </strong>
   ”，住在马里兰州 的 Eric Hensal 说，他出现在2016年华盛顿特朗普酒店外示威游行被监控的数据集中。
  </p>

    现在你只能使用技术来保护自己，因此只有那些有技术能力的人才能做到这一点
   </strong>
   。但是，隐私并不仅仅是服务于有钱人或极客的。隐私权应该是普世的基本人权。”
  </p>
  <p class="graf graf--p">
   即使在非专制国家，不受限制的移动监视的未来也无疑会迫使持不同政见者做出艰难的决定。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “对于反法西斯研究人员和抗议者来说，他们会承担完全暴露的危险。恶劣的状况持续升温。它迫使政府和其他组织之间变得越来越好战。”
  </p>
  <p class="graf graf--p">
   全球激进主义者的未来可能越来越像香港。
  </p>
  <p class="graf graf--p">
   去年的无领导抗议运动是通过技术实现的。留言板 LIHKG 和诸如 Telegram 之类的加密聊天应用程序使这种组织得以继续进行。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    运动本身也被同样的技术所破坏
   </strong>
   。抗议者使用的一种实时位置跟踪应用程序，用于识别警察的位置，已被苹果的 App Store 删除，这表明政府在资源上将具有竞争优势。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11768 jetpack-lazy-image" data-lazy-sizes="(max-width: 1028px) 100vw, 1028px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-7.jpeg?resize=1028%2C2048&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-7.jpeg?w=1028&amp;ssl=1 1028w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-7.jpeg?resize=151%2C300&amp;ssl=1 151w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-7.jpeg?resize=514%2C1024&amp;ssl=1 514w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-7.jpeg?resize=768%2C1530&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-7.jpeg?resize=771%2C1536&amp;ssl=1 771w" data-recalc-dims="1" height="2048" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-7.jpeg?resize=1028%2C2048&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1028"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11768" data-recalc-dims="1" height="2048" sizes="(max-width: 1028px) 100vw, 1028px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-7.jpeg?resize=1028%2C2048&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-7.jpeg?w=1028&amp;ssl=1 1028w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-7.jpeg?resize=151%2C300&amp;ssl=1 151w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-7.jpeg?resize=514%2C1024&amp;ssl=1 514w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-7.jpeg?resize=768%2C1530&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-7.jpeg?resize=771%2C1536&amp;ssl=1 771w" width="1028"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   尽管抗议者戴着口罩，用激光挡住政府摄像机，甚至拆掉可疑的监控路灯柱，但是，
   <strong class="markup--strong markup--p-strong">
    每个人口袋里的间谍却在悄悄破坏他们的一切努力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   与我们其他人所有人一样，手机就是你的贴身间谍。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%96%af%e8%af%ba%e7%99%bb%ef%bc%9a%e6%99%ba%e8%83%bd%e6%89%8b%e6%9c%ba%e6%9c%ac%e8%ba%ab%e5%b0%b1%e6%98%af%e4%b8%8d%e5%ae%89%e5%85%a8%e7%9a%84%e4%b8%9c%e8%a5%bf%ef%bc%88video%ef%bc%89/" href="https://www.iyouport.org/%e6%96%af%e8%af%ba%e7%99%bb%ef%bc%9a%e6%99%ba%e8%83%bd%e6%89%8b%e6%9c%ba%e6%9c%ac%e8%ba%ab%e5%b0%b1%e6%98%af%e4%b8%8d%e5%ae%89%e5%85%a8%e7%9a%84%e4%b8%9c%e8%a5%bf%ef%bc%88video%ef%bc%89/" rel="noopener noreferrer" target="_blank">
     斯诺登：智能手机本身就是不安全的东西
    </a>
    》
   </li>
   <li>
    非常推荐这个访谈！《
    <a href="https://www.iyouport.org/%e4%bd%a0%e7%9a%84%e6%89%8b%e6%9c%ba%e5%b0%b1%e6%98%af%e7%a7%98%e5%af%86%e7%9b%91%e8%a7%86%e8%ae%be%e5%a4%87-snowden%e8%ae%bf%e8%b0%88%e8%a7%86%e9%a2%91/" rel="noopener noreferrer" target="_blank">
     你的手机就是秘密监视设备 — Snowden访谈视频
    </a>
    》
   </li>

  <p class="graf graf--p">
   在香港等地，成千上万的手机照亮了天空，这是和平反对权威的表现。但是，只有当这些闪亮的手机最终全部消失的时候，它们希望代表的鼓舞人心的图像和民主精神才能发挥作用。⚪️
  </p>
