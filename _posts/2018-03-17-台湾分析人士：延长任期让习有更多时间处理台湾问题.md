title: 台湾分析人士：延长任期让习有更多时间处理台湾问题
---

<div id="entry">
<div class="at-above-post addthis_tool" data-url="http://zhanlve.org/?p=4766">
</div>
<p>
<img alt="台湾示威者在台北抗议中国领导人习近平与当时的台湾总统马英九在新加坡举行的60多年来两岸领导人的首次会晤。（2015年11月7日）" class="aligncenter" src="https://boxun.com/news/images/2018/03/201803142312taiwan1.jpg"/>
</p>
<p>
</p>
<p>
<span style="font-size: 12pt;">
   台北 — 有专家认为，中国国家主席习近平任职终身制会给中国更多的空间，寻求统一台湾的新途径。台湾的绝大多数人更喜欢自治。
  </span>
</p>
<p>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   中国人大星期天（3月11日）通过了对宪法部分内容的修改提案，国家主席副主席可以无限期连任，删除了国家主席副主席任期不得超过两届，每届五年的限制。这意味着现年64岁的习近平可以合法地终身留在国家主席的位子上。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   分析人士说，习近平去年10月再次承诺寻求统一台湾，现在他将有更多的时间尝试强硬和温和的手段来实现他的目标。习近平说过，台湾统一的问题不能留给下一代中共领导人。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   台湾国立政治大学国际事务学院副院长黄奎博说：“习近平有可能不会急于压台湾谈判，只要习没有遇到外部影响，他有更多的时间处理台湾问题。”
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   台湾和中国大陆1949年实行分治，当时蒋介石的国民党败给中共，被迫退守台湾。1980年代台湾实行民主制度，2015年以来政府历次民调都显示，绝大多数受访民众反对跟当今专制独裁的中国统一。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   中国一直声称对台湾拥有主权，并且发誓在必要情况下武统台湾。习近平去年十月说过，他将依据中共1997年接管香港时的一国两制模式，寻求和平统一台湾.
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   台湾执政的民进党立委李俊俋指出，如果习近平长期担任国家最高职务，他对台湾的态度可能会强硬，但是不会有实质性改变。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   在这种情况下，李俊俋认为，中共一方面对台湾领导人施加军事和外交压力，另一方面以“温和”手段来诱惑民众。本月初中国宣布31条惠台措施，鼓励台湾人到对岸就业、学习和投资。
  </span>
</p>
<p>
</p>
<p>
<span style="font-size: 12pt;">
   中华二十一世纪智库协会CEO雷倩认为，习近平任期延长可能会令他的政府放慢施压台湾的步伐。雷倩说，如果习2023年任期届满，距离他最终解决台湾问题的时间就非常紧迫，中国可能会向台湾施加更大的压力。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   台湾学者李宪荣认为，台湾政府和民众会努力适应习近平长期留任。他说，不仅是蔡英文政府，包括我认识的所有人都不知道该如何应对这个变化。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   台湾陆委会日前回应中国废除任期限制时表示，将观察北京接下来的动作。
  </span>
</p>
<p>
</p>

   出处:美国之音
