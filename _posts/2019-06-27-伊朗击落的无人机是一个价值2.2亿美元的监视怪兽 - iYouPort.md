  <p class="graf graf--p">
   周四早上，伊朗在波斯湾和阿曼湾之间的霍尔木兹海峡击落了一架美军无人机。
  </p>
  <p class="graf graf--p">
   伊朗将这架无人机确定为
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wired.com/2011/06/it-was-march-madness-for-global-hawk-spy-drone/" href="https://www.wired.com/2011/06/it-was-march-madness-for-global-hawk-spy-drone/" rel="noopener noreferrer" target="_blank">
    RQ-4A“全球鹰”
   </a>
   ——
   <strong class="markup--strong markup--p-strong">
    这是一架价值2.2亿美元的无人机，可作为天空中的大型监视平台。
   </strong>
  </p>
  <p class="graf graf--p">
   这次袭击标志着美国和伊朗之间的紧张局势有所升级，特别是因为被击落的无人机的高价值和技术敏感性。
  </p>
  <p class="graf graf--p">
   伊朗伊斯兰革命卫队周四表示，诺斯洛普·格鲁门公司制造的全球鹰 —— 可追溯到2001年的数十亿美元计划的一部分 —— 进入伊朗领空并在伊朗水域坠毁；美国中央司令部
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/CENTCOM/status/1141703098152493056" href="https://twitter.com/CENTCOM/status/1141703098152493056" rel="noopener noreferrer" target="_blank">
    确认了
   </a>
   袭击的时间和一般位置，但是坚称无人机当时在国际空域飞行。
  </p>
  <p class="graf graf--p">
   此事件发生在上周的另一种情况之后，美国指责伊朗袭击了阿曼湾的两艘油罐车。
  </p>
  <p class="graf graf--p">
   美国
   <a class="markup--anchor markup--p-anchor" data-href="https://www.latimes.com/politics/la-na-pol-iran-drone-attack-20190616-story.html" href="https://www.latimes.com/politics/la-na-pol-iran-drone-attack-20190616-story.html" rel="noopener noreferrer" target="_blank">
    还说
   </a>
   ，伊朗曾试图击落一架不同的无人机 — 一架MQ-9收割者无人机 — 但失败了。
  </p>
  <p class="graf graf--p">
   两周前，五角大楼
   <a class="markup--anchor markup--p-anchor" data-href="https://www.latimes.com/politics/la-na-pol-iran-drone-attack-20190616-story.html" href="https://www.latimes.com/politics/la-na-pol-iran-drone-attack-20190616-story.html" rel="noopener noreferrer" target="_blank">
    还将
   </a>
   伊朗与在也门的一架死神无人机袭击事件联系起来。
  </p>
  <p class="graf graf--p">
   然而，星期四的这次袭击针对的是一架巨大的且昂贵得多的监视无人机，可能代表了更明确的升级。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这里有很多事发生，我们可能只看到了其中的一部分，”战略与国际研究中心导弹防御项目主任 Thomas Karako 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <strong class="markup--strong markup--p-strong">
    这是一种更昂贵的、更高海拔的、更强大的远程情报监视侦察机
   </strong>
   。如果这样的飞机在国际空域被击落，很可能会引发某种程度的报复。”
  </p>
  <p class="graf graf--p">
   全球鹰是自2001年开始运作的大型监视平台，翼展超过130英尺，最大起飞重量超过16吨，相当于大约7个可卡因的集装箱。
  </p>
  <p class="graf graf--p">
   它们的航程超过12,000海里，可以在惊人的60,000英尺高空飞行，并且可以在高空停留34小时。
  </p>
  <p class="graf graf--p">
   它们的价值在于它们能将覆盖范围、有利位置和持久性与强大的监视传感器结合起来，能够非常详细地监视地面或海上的活动。
  </p>
  <p class="graf graf--p">
   这大概就是为什么美国会花费2.2亿美元来制造和装备这种监视平台。
  </p>
  <p class="graf graf--p">
   全球鹰通常在其传感器库中包括红外和热成像、雷达和电光成像。
  </p>
  <p class="graf graf--p">
   巨大的体积和重量使得这种无人机能够利用巨大的长焦相机镜头等设备来获取目标的详细视图。
  </p>
  <p class="graf graf--p">
   但正如欧洲外交关系委员会无人机研究员 Ulrike Franke 所指出的那样，美国军方为不同的任务提供了不同的无人机装备，于是无法确定这个特定的全球鹰携带的究竟是什么装备。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <strong class="markup--strong markup--p-strong">
    这种飞机总会有一些超级秘密的间谍技术
   </strong>
   ，我们不知道”，Franke 说。
  </p>
  <p class="graf graf--p">
   但 Franke 表示，这种特殊的全球鹰作为一种典型的监视平台，更可能是因地缘政治原因而不是技术侦察的具体目标而被击落的。
  </p>
  <p class="graf graf--p">
   目前还不清楚这些无人机的关键部分是否可以恢复，或者是否已经在攻击中被摧毁了。
  </p>
  <p class="graf graf--p">
   伊朗在2011年12月以令人难忘的方式夺取了美国 RQ-170 哨兵无人机，后来声称已经对飞机的硬件和软件进行了逆向工程以复制其技术。
  </p>
  <p class="graf graf--p">
   哨兵无人机被认为使用了隐形技术进行空中侦察。
  </p>
  <p class="graf graf--p">
   去年，以色列官员说他们截获了一架
   <a class="markup--anchor markup--p-anchor" data-href="https://www.defensenews.com/global/mideast-africa/2018/02/12/israel-air-force-says-seized-iranian-drone-is-a-knockoff-of-us-sentinel/" href="https://www.defensenews.com/global/mideast-africa/2018/02/12/israel-air-force-says-seized-iranian-drone-is-a-knockoff-of-us-sentinel/" rel="noopener noreferrer" target="_blank">
    似乎是哨兵“副本”
   </a>
   的伊朗无人机。
  </p>
  <p class="graf graf--p">
   至于全球鹰是否飞越伊朗领空，研究人员表示，明确的证据将要求美国公布有关无人机飞行路径的细节。
  </p>
  <p class="graf graf--p">
   Karako 说：“他们是否想要发布更多细节是一个政策决定。但到目前为止，美国中央司令部坚持认为飞机在国际空域。”
  </p>
  <p class="graf graf--p">
   目前还不知道无人机在被击落时飞行的高度如何，但如果它当时在高海拔地区，那意味着是一个有点困难的目标。
  </p>
  <p class="graf graf--p">
   不过 Franke 仍然强调，高空拦截是在伊朗众所周知的能力范围内的。
  </p>
  <p class="graf graf--p">
   他表示，“作为卖点通常认为全球鹰飞得如此之高应该不会被击落。但击落这样的飞机并非不可思议，虽然相对困难。它显示了政治方面的决心”。
  </p>
  <p class="graf graf--p">
   作为回应，特朗普最初在周四发推文称，“伊朗犯了一个很大的错误！”
  </p>
  <p class="graf graf--p">
   然而，在后来的评论中，
   <strong class="markup--strong markup--p-strong">
    特朗普似乎采取了一种不那么激进的语气
   </strong>
   ，说：“我觉得很难相信这是故意的。”
  </p>
  <p class="graf graf--p">
   不过分析人士表示，尽管伊朗拥有拦截无人机的技术并不令人震惊，但要做到这点需要雷达制导的地对空导弹系统 —— 看似俄罗斯向伊朗提供的 SA-6或 SA-17 SAM。
  </p>
  <p class="graf graf--p">
   更灵活的寻热制导肩扛式地空导弹系统无法在高空击中目标。换句话说，这只能是故意的。⚪️
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wired.com/story/iran-global-hawk-drone-surveillance/" href="https://www.wired.com/story/iran-global-hawk-drone-surveillance/" rel="noopener noreferrer" target="_blank">
    “There could always be super secret spy tech onboard that we don’t know about.”
   </a>
  </p>
