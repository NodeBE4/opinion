 <div class="body-text">
  <p align="left">
  </p>
  <div style="text-indent: 18pt">
   <strong>
    <span style="font-size: 9pt">
     导语：
    </span>
   </strong>
   <span style="font-size: 9pt">
    一方面是储蓄利率低，另一方面是投资渠道少，"中国大妈"可以做出的选择并不多。可以说金融体制若不市场化改革，中国大妈们的积蓄保值增值的愿望就难以实现。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    国际黄金市场至今仍跌跌不休，"中国大妈"从被媒体爆出之后不断受到各种评判和解读，"中国大妈"为何用辛苦积攒的钱押注黄金？或许可以从这一现象背后发现中国金融制度的问题。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    中国的储蓄率向来很高，高储蓄率与中国社会保障体系不完善以及国内投资渠道太少都有直接关系，后者对于储蓄率高企有着最为本质的影响。同时，人为压低的利率与高储蓄率形成合力，将很多投资者推向了黄金市场。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    中国的金融体系以国有银行为主，企业的融资也是以银行贷款为主，直接融资的比例偏低。政府实行低利率的政策来拉动投资，中国能实现较高的经济增长水平，某种程度上也得益于此。那么值得讨论的是，这个体制的受益者与受损者分别是谁。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    中国金融制度的受益者一个是银行本身，另一个则是国有企业，他们有两个共同点：国有和垄断。
   </span>
   <span style="font-size: 9pt">
    2012
   </span>
   <span style="font-size: 9pt">
    年，
   </span>
   <span style="font-size: 9pt">
    2400
   </span>
   <span style="font-size: 9pt">
    多家上市公司的总利润构成中，
   </span>
   <span style="font-size: 9pt">
    16
   </span>
   <span style="font-size: 9pt">
    家上市银行的净利润占比竟高达
   </span>
   <span style="font-size: 9pt">
    53.57%
   </span>
   <span style="font-size: 9pt">
    ，让人不禁惊呼银行正在"剥削"国内实业。另一方面，国有企业不仅从银行得到贷款相对容易，其支付利息水平也比其他性质的企业要低，天则经济研究所在《国有企业的性质、表现与改革》一文中曾估算在
   </span>
   <span style="font-size: 9pt">
    2001
   </span>
   <span style="font-size: 9pt">
    年到
   </span>
   <span style="font-size: 9pt">
    2008
   </span>
   <span style="font-size: 9pt">
    年的
   </span>
   <span style="font-size: 9pt">
    8
   </span>
   <span style="font-size: 9pt">
    年间，国有企业少支付利息竟达
   </span>
   <span style="font-size: 9pt">
    2.85
   </span>
   <span style="font-size: 9pt">
    万亿之巨。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    这种制度安排下的受损者是民营企业和储蓄者。储蓄者把钱交给银行本是想得到关于资金时间成本的补偿，但事与愿违，国内货币供应量近几年的大幅增加所造成的通胀水平远比国家统计局公布的
   </span>
   <span style="font-size: 9pt">
    CPI
   </span>
   <span style="font-size: 9pt">
    要高。耶鲁大学经济学教授陈志武曾对中美两国的通胀水平做过对比，
   </span>
   <span style="font-size: 9pt">
    2000
   </span>
   <span style="font-size: 9pt">
    年到
   </span>
   <span style="font-size: 9pt">
    2012
   </span>
   <span style="font-size: 9pt">
    年间，中国统计局公布的
   </span>
   <span style="font-size: 9pt">
    CPI
   </span>
   <span style="font-size: 9pt">
    累积上涨
   </span>
   <span style="font-size: 9pt">
    31%
   </span>
   <span style="font-size: 9pt">
    ，同期间美国累积上涨
   </span>
   <span style="font-size: 9pt">
    36%
   </span>
   <span style="font-size: 9pt">
    ，按统计数据，人民币的购买力相对来说理应更强，但现实却完全相反。人为压低的名义利率水平与高企的实际通胀水平，共同决定了一个事实：储蓄者在补贴银行以及银行背后的国有企业。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    民营企业作为金融制度的另一受害者，想从银行得到贷款要比国有企业困难得多，这也给民间借贷与地下钱庄的发展打开了一扇门，但政府并没有给民间借贷合法身份，使得民间借贷不能正常发展。近几年屡有涉及大额资金民间借贷案发生，究其根本就是没有打破金融垄断，没有给民间借贷正常发展的法律地位与良好环境。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    储蓄的低利率也开始导致"金融脱媒"现象的发生，激励着人们把钱从低利率的银行中抽脱出来，想别的办法增值，甚至去放高利贷。储蓄者在银行的垄断体制下不能获得所被占用资金的补偿，他们当然会选择把钱投资到其他领域中，"中国大妈"只是其中的一个具有代表性的群体。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    把钱投资到银行存款外的其他领域也并非那么容易。按投资方向的不同，投资大体可分为金融投资与实业投资。国内适合个人投资者投资的领域实在太少，最具代表性的是股市，但股市制度的缺陷以及其"十年零涨幅"的表现让人望而却步。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    实业投资环境也并不比股市更能吸引人。在国有经济主导的前提下，民营企业很难与国有企业公平竞争。由中国企业联合会、中国企业家协会发布的《
   </span>
   <span style="font-size: 9pt">
    2011
   </span>
   <span style="font-size: 9pt">
    年中国
   </span>
   <span style="font-size: 9pt">
    500
   </span>
   <span style="font-size: 9pt">
    强企业发展报告》称，中国
   </span>
   <span style="font-size: 9pt">
    500
   </span>
   <span style="font-size: 9pt">
    强企业中，民营企业的利润仅占总额的
   </span>
   <span style="font-size: 9pt">
    18.12%
   </span>
   <span style="font-size: 9pt">
    ，而在占比高达
   </span>
   <span style="font-size: 9pt">
    81.88%
   </span>
   <span style="font-size: 9pt">
    的国有企业中，最为突出的当属三大石油公司以及五大国有银行，其利润额在全国
   </span>
   <span style="font-size: 9pt">
    500
   </span>
   <span style="font-size: 9pt">
    强中占比分别达到
   </span>
   <span style="font-size: 9pt">
    10.71%
   </span>
   <span style="font-size: 9pt">
    、
   </span>
   <span style="font-size: 9pt">
    26.13%
   </span>
   <span style="font-size: 9pt">
    。虽然国内曾经针对民营企业投资受限问题出台过新、老"
   </span>
   <span style="font-size: 9pt">
    36
   </span>
   <span style="font-size: 9pt">
    条"，但最后都不了了之。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    在一般投资领域受抑制的投资环境下，房地产市场与艺术品市场是两个例外。房地产自不必言，门槛相对高的艺术品市场也受到热捧甚至炒作，从当今电视里众多的鉴宝节目中即可看出，除古玩、赌石之外，一些把玩用的核桃、蜜蜡等都吸引了大批投资者。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    这段时间阿里巴巴推出的余额宝是一个不错的尝试，意味着人们又多了一个投资渠道。但
   </span>
   <span style="font-size: 9pt">
    6
   </span>
   <span style="font-size: 9pt">
    月
   </span>
   <span style="font-size: 9pt">
    21
   </span>
   <span style="font-size: 9pt">
    日
   </span>
   <span style="font-size: 9pt">
    证监会称余额宝违规，其生存前景也不太明朗。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    一方面是储蓄利率低，另一方面是投资渠道少，"中国大妈"可以做出的选择并不多。可以说金融体制若不市场化改革，中国大妈们的积蓄保值增值的愿望就难以实现。
   </span>
  </div>
