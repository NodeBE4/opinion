   独特的算法测量正在彻底改变识别过程。但生物识别技术远非完美 —— 它们的便利性和看似无懈可击的高端背后需要你付出巨大的代价 —— 最重要的是，隐私人权。
  </p>
  <p class="graf graf--p">
   这就是这一集的主要内容。
  </p>
  <p class="graf graf--p">
   视频：

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、“同意吧”
   </strong>
  </p>
  <p class="graf graf--p">
   我们所有人如何成为了互联网上最大谎言的同谋？
  </p>
  <p class="graf graf--p">
   关于这一问题，见我们曾经的中文版分析（这一集的内容没有我们曾经介绍的内容更深刻，但足够指明问题）：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bb%a5%e9%9a%90%e7%a7%81%e4%b8%ba%e7%9b%ae%e7%9a%84%e7%9a%84%e6%96%97%e4%ba%89%e5%a6%82%e4%bd%95%e8%ae%a9%e4%bd%a0%e5%8f%98%e6%88%90%e4%ba%86%e7%9b%91%e8%a7%86%e7%9a%84%e7%9b%9f%e5%8f%8b/" href="https://www.iyouport.org/%e4%bb%a5%e9%9a%90%e7%a7%81%e4%b8%ba%e7%9b%ae%e7%9a%84%e7%9a%84%e6%96%97%e4%ba%89%e5%a6%82%e4%bd%95%e8%ae%a9%e4%bd%a0%e5%8f%98%e6%88%90%e4%ba%86%e7%9b%91%e8%a7%86%e7%9a%84%e7%9b%9f%e5%8f%8b/" rel="noopener noreferrer" target="_blank">
     以隐私为目的的斗争如何让你变成了监视的盟友
    </a>

   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/6%e5%b9%b4%ef%bc%8c%e6%94%b9%e5%8f%98%e4%ba%86%e4%bb%80%e4%b9%88%ef%bc%9f/" href="https://www.iyouport.org/6%e5%b9%b4%ef%bc%8c%e6%94%b9%e5%8f%98%e4%ba%86%e4%bb%80%e4%b9%88%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     为什么说我们所有人是反乌托邦的同谋
    </a>
    》
   </li>

  <p class="graf graf--p">
   屏幕后面有一大群神秘的人类在工作。他们被称为行为架构师、有说服力的设计师或用户体验专家，他们拥有的权力是巨大的。
  </p>
  <p class="graf graf--p">
   你是否有种冲动想要不断刷新推文？—— 这是被设计出的心理操纵效果；
  </p>
  <p class="graf graf--p">
   你是否几乎不会详细阅读就轻松点击了“我同意”条款？—— 这也是被设计出来的操纵；
  </p>
  <p class="graf graf--p">
   在Tinder上向右或向左滑动？嗯，这也是被设计的……
  </p>
  <p class="graf graf--p">
   我们所有人已经被绑架在一个别人精心设计的网络世界中，大多数人甚至都没有仔细想过这件事。这一切都是有预谋的。
  </p>
  <p class="graf graf--p">
   这就是本集的内容。

  </p>
  <p class="graf graf--p">
   这就是我们所有人生活的世界，已经无法逃离。我们需要原地战斗。
  </p>

   战斗能成功的前提是让更多人了解它，了解我们的对手是谁，他们是怎么对付我们的，才能找到足以获胜的解决方案。⚪️
