     <em>
      <strong>
       在监视资本家的眼里，你身上的一切都应该拿来卖钱。包括你的虚荣心。
      </strong>
     </em>
    </span>
   </li>

  <p class="graf graf--p">
   如今真的应该非常小心那些用晒照片来吸引注册的应用；因为所有这些照片都会被拿去训练算法，以形成更加危险的面部识别监视技术。
  </p>
  <p class="graf graf--p">
   Ever 大概是其中做得最“绝”的一个 —— 它的开发目的就是为钓鱼。但要知道，这种做法整体上并不特殊。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6245 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/图-3.png?resize=1100%2C879&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/图-3.png?w=1246&amp;ssl=1 1246w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/图-3.png?resize=300%2C240&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/图-3.png?resize=768%2C614&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/图-3.png?resize=1024%2C819&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/图-3.png?resize=1100%2C879&amp;ssl=1 1100w" data-recalc-dims="1" height="879" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/图-3.png?resize=1100%2C879&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  </p>
  <p class="graf graf--p">
   云存储应用程序 Ever 是一个以危险的方式开发面部识别技术的例子，并且可能冒着放大歧视的风险。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.everalbum.com/#main" href="https://www.everalbum.com/#main" rel="noopener noreferrer" target="_blank">
    Ever 是一个云存储应用程序
   </a>
   ，自称“帮助您捕捉和重新发现生活的记忆”，包括上传和存储个人照片；但是该应用程序并没有告知用户
   <strong class="markup--strong markup--p-strong">
    它在使用人们上传的数百万张照片来训练其
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab/" href="https://www.iyouport.org/category/%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     面部识别软件
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，⚠️然后将识别技术出售给执法部门、私营公司、甚至军方
   </strong>
   。
  </p>
  <p class="graf graf--p">
   需要严重关切的是，Ever 正在以一种侵犯人们隐私权的方式使用其平台，它创造了一种固有的歧视性、容易出错的、法律上不可靠的技术。
  </p>
  <p class="graf graf--p">
   Ever 创造技术的整个过程都是不透明和不负责任的。
  </p>
  <p class="graf graf--p">
   虽然亚马逊和微软等许多巨头公司
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e6%9b%be%e7%bb%8f%e5%9c%a8%e7%a4%be%e4%ba%a4%e7%bd%91%e7%bb%9c%e4%b8%8a%e5%8f%91%e5%b8%83%e8%bf%87%e8%87%aa%e5%b7%b1%e7%9a%84%e7%85%a7%e7%89%87%ef%bc%9f%e4%bd%a0%e5%8d%b1%e9%99%a9%e4%ba%86/" href="https://www.iyouport.org/%e4%bd%a0%e6%9b%be%e7%bb%8f%e5%9c%a8%e7%a4%be%e4%ba%a4%e7%bd%91%e7%bb%9c%e4%b8%8a%e5%8f%91%e5%b8%83%e8%bf%87%e8%87%aa%e5%b7%b1%e7%9a%84%e7%85%a7%e7%89%87%ef%bc%9f%e4%bd%a0%e5%8d%b1%e9%99%a9%e4%ba%86/" rel="noopener noreferrer" target="_blank">
    都在使用可公开获得的数据集
   </a>
   （包括在没有用户知情或许可的情况下获得的照片）培训其面部识别技术。
  </p>
  <p class="graf graf--p">
   但 Ever 本身就是一个坑，它创建技术并诱导用户上传照片的目的就是利用。
  </p>
  <p class="graf graf--p">
   只是在媒体追问后 Ever 才
   <a class="markup--anchor markup--p-anchor" data-href="https://www.everalbum.com/privacy-policy" href="https://www.everalbum.com/privacy-policy" rel="noopener noreferrer" target="_blank">
    更新了政策
   </a>
   ，写出了它“使用这些照片训练面部识别技术”，人们上传的内容“可用于帮助改进和培训我们的产品和这些技术”。
  </p>
  <p class="graf graf--p">
   尽管隐私政策做出了修改，但是，Ever 使用人们的照片的具体方式依旧无法预测，最终可用于什么样的监视软件或产生哪些歧视性结果，都不清楚。
  </p>
  <p class="graf graf--p">
   并且，
   <strong class="markup--strong markup--p-strong">
    没有注册过该应用程序的人 —— 用户的朋友和家人，也被利用了，他们并不知道自己的脸被用于了什么。
   </strong>
  </p>
  <p class="graf graf--p">
   使用人们的照片并将识别技术出售给外部公司并不主要为了保护客户的利益或改善他们的云存储体验。Ever 的做法
   <strong class="markup--strong markup--p-strong">
    从根本上侵犯
   </strong>
   了使用其服务的人的隐私权以及在上传到 Ever 的照片中出现的其他人的隐私权。
  </p>
  <p class="graf graf--p">
   正在开发的技术明显缺乏透明度和问责制。Ever 使用的照片数据集无法代表更广泛的人口。因此，该技术可能具有更高的错误率。
  </p>
  <p class="graf graf--p">
   因为
   <strong class="markup--strong markup--p-strong">
    Ever 的数据集来自人们存储的照片，所以独立的外部团队无法检查数据库、也无法发现此类问题并要求更改
   </strong>
   。
  </p>
  <p class="graf graf--p">
   相比之下，在亚马逊的 Rekognition 的情况下，研究人员揭露了软件中的错误和偏见，这些错误和偏见推动了亚马逊停止将其出售给执法部门的要求。
  </p>
  <p class="graf graf--p">
   此外，目前还不清楚 Ever 在其面部识别技术中是如何标记人们面部特征的，该技术依赖于机器学习创建人物面部的详细生物特征图并用于匹配照片。
  </p>
  <p class="graf graf--p">
   错误标记的特征会导致歧视性结果并加剧歧视程度。
  </p>
  <p class="graf graf--p">
   Ever 作为一家利用用户发财的公司是一个典型的例子，造成这种生意出现的部分原因是由于缺乏限制面部识别技术的开发和使用的法律。
  </p>
  <p class="graf graf--p">
   在世界各地，这种技术正在被推广并被用于广泛的目的，通常都是秘密操作的，具有高度歧视性的影响。
  </p>
  <p class="graf graf--p">
   旧金山正朝着正确的方向迈出一步，并正在作为司法管辖区如何应对此类问题的蓝图。但是，旧金山的法令不适用于私营公司。警方可以与私营公司合作绕避法律限制。
  </p>
  <p class="graf graf--p">
   具体分析详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a6%81%e6%ad%a2%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e5%8f%af%e4%bb%a5%e5%81%9a%e5%88%b0%e4%bb%80%e4%b9%88%e3%80%81%e4%b8%8d%e8%83%bd%e5%81%9a%e5%88%b0%e4%bb%80%e4%b9%88/" href="https://www.iyouport.org/%e7%a6%81%e6%ad%a2%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e5%8f%af%e4%bb%a5%e5%81%9a%e5%88%b0%e4%bb%80%e4%b9%88%e3%80%81%e4%b8%8d%e8%83%bd%e5%81%9a%e5%88%b0%e4%bb%80%e4%b9%88/" rel="noopener noreferrer" target="_blank">
    禁止面部识别可以做到什么、不能做到什么
   </a>
   》；《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a6%81%e6%ad%a2%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%ef%bc%8c%e7%9c%9f%e7%9a%84%e5%8f%af%e4%bb%a5%e5%90%97%ef%bc%9f/" href="https://www.iyouport.org/%e7%a6%81%e6%ad%a2%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%ef%bc%8c%e7%9c%9f%e7%9a%84%e5%8f%af%e4%bb%a5%e5%90%97%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    禁止面部识别，真的可以吗？
   </a>
   》。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    每年的6月20日是英国的监控摄像机日。人权倡导者可以利用这样的日子推广隐私保护意识。
   </strong>
  </p>
  <p class="graf graf--p">
   比如制作参与式活动，看看谁知道所有本地摄像机的位置都在哪？您当地的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bbc.co.uk/news/technology-23665490" href="https://www.bbc.co.uk/news/technology-23665490" rel="noopener noreferrer" target="_blank">
    垃圾箱
   </a>
   上有没有监视器？
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%b7%af%e7%81%af%e9%97%b4%e8%b0%8d/" href="https://www.iyouport.org/%e8%b7%af%e7%81%af%e9%97%b4%e8%b0%8d/" rel="noopener noreferrer" target="_blank">
    灯柱
   </a>
   或
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e8%a6%81%e4%bd%a0%e8%bf%9c%e7%a6%bb%e5%b9%bf%e5%91%8a%e7%89%8c-%e8%80%81%e5%a4%a7%e5%93%a5%e6%9d%a5%e4%ba%86/" href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e8%a6%81%e4%bd%a0%e8%bf%9c%e7%a6%bb%e5%b9%bf%e5%91%8a%e7%89%8c-%e8%80%81%e5%a4%a7%e5%93%a5%e6%9d%a5%e4%ba%86/" rel="noopener noreferrer" target="_blank">
    广告牌
   </a>
   上都有什么样的监视器？
  </p>
  <p class="graf graf--p">
   你当地的社区有
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/uk-news/2018/may/15/uk-police-use-of-facial-recognition-technology-failure" href="https://www.theguardian.com/uk-news/2018/may/15/uk-police-use-of-facial-recognition-technology-failure" rel="noopener noreferrer" target="_blank">
    面部识别面包车
   </a>
   /公交车/地铁吗？警察的直升机正盘旋在你家房顶
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/uk-news/2017/jul/18/south-yorkshire-police-helicopter-crew-filmed-people-naked-sheffield-crown-court-told" href="https://www.theguardian.com/uk-news/2017/jul/18/south-yorkshire-police-helicopter-crew-filmed-people-naked-sheffield-crown-court-told" rel="noopener noreferrer" target="_blank">
    拍摄你晒日光浴
   </a>
   的画面吗？
  </p>
  <p class="graf graf--p">
   监视摄像机可能是肉眼看不到的，例如那些据报被用来监视 Boots、Asda 和 Tesco 购物者的情况；更不用说人们不可能知道哪里启用了面部识别技术；在超市结账排队时可能会有不止一只监视器在
   <a class="markup--anchor markup--p-anchor" data-href="https://newsroom.ibm.com/2019-01-15-IBM-Showcases-New-AI-Innovations-at-NRF-2019-to-Help-Retail-Industry-Accelerate-Customer-Experience" href="https://newsroom.ibm.com/2019-01-15-IBM-Showcases-New-AI-Innovations-at-NRF-2019-to-Help-Retail-Industry-Accelerate-Customer-Experience" rel="noopener noreferrer" target="_blank">
    分析你的脸和情绪状态
   </a>
   。
  </p>
  <p class="graf graf--p">
   美国公民自由联盟指出，
   <strong class="markup--strong markup--p-strong">
    “视频分析”技术可以被政府用于监控政治竞争对手，企业可以根据人们的习惯不成比例地剥夺消费者的利益，警监视公民的一举一动并实施所谓的预测犯罪。
   </strong>
  </p>
  <p class="graf graf--p">
   更多详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%9c%aa%e6%9d%a5%e6%98%af%e4%b8%80%e9%a1%b9%e6%96%b9%e7%a8%8b%e5%bc%8f-%e5%b0%91%e6%95%b0%e6%b4%be%e6%8a%a5%e5%91%8a%e7%9a%84%e5%a8%81%e8%83%81%e4%b8%8d%e6%98%af%e8%bf%99%e4%b8%a4/" href="https://www.iyouport.org/%e6%9c%aa%e6%9d%a5%e6%98%af%e4%b8%80%e9%a1%b9%e6%96%b9%e7%a8%8b%e5%bc%8f-%e5%b0%91%e6%95%b0%e6%b4%be%e6%8a%a5%e5%91%8a%e7%9a%84%e5%a8%81%e8%83%81%e4%b8%8d%e6%98%af%e8%bf%99%e4%b8%a4/" rel="noopener noreferrer" target="_blank">
    “未来是一项方程式” — 少数派报告的威胁不是这两年才有的新事物
   </a>
   》。
  </p>
  <p class="graf graf--p">
   人们有权匿名表达意见，有权参加公共集会，参加和平抗议或参与有争议的对话。但是在监视状态下这一切都不再可能。
  </p>
  <p class="graf graf--p">
   监视远不止隐私和数据的问题。
  </p>
  <p class="graf graf--p">
   要重新抢回公民权，必须有更多人站起来抵制监视技术、要求透明度和问责制。
  </p>
  <p class="graf graf--p">
   在你的社区开展活动，让更多人
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/?s=%E7%9B%91%E8%A7%86%E4%B9%8B%E6%81%B6" href="https://www.iyouport.org/?s=%E7%9B%91%E8%A7%86%E4%B9%8B%E6%81%B6" rel="noopener noreferrer" target="_blank">

   </a>
   。在为时已晚之前。⚪️
