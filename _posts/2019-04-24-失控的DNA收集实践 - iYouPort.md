 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <strong>
      <em>
       来，张开嘴巴！纽约警察局将不遗余力地收集并永久存储所有人的 DNA  ——不论你是有罪或无辜。究竟发生了什么？DNA监视如何构陷无辜的人变成罪犯？
      </em>
     </strong>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   根据
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nydailynews.com/opinion/ny-oped-the-nypds-new-dna-dragnet-20190206-story.html" href="https://www.nydailynews.com/opinion/ny-oped-the-nypds-new-dna-dragnet-20190206-story.html" rel="noopener noreferrer" target="_blank">
    纽约每日新闻
   </a>
   报道，纽约警察局将不遗余力地收集并永久存储所有人的 DNA — — 不论你是有罪或无辜。
  </p>
  <p class="graf graf--p">
   无论是挨家挨户、用卷烟贿赂他人、还是随意的路边拦截，纽约警察局一直在积极编制一个奥威尔式的DNA数据库，该数据库截至2017年已有了64,000多个档案。现在是2019年。
  </p>
  <p class="graf graf--p">
   以警方调查霍华德海滩慢跑女被杀案为例。在他们确定嫌疑人之前，纽约警察局已经收集了来自纽约东部地区一大群男性的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.thedailybeast.com/he-confessed-to-karina-vetranos-murder-but-did-he-do-it" href="https://www.thedailybeast.com/he-confessed-to-karina-vetranos-murder-but-did-he-do-it" rel="noopener noreferrer" target="_blank">
    500多张DNA图谱
   </a>
   。你可以想象警察穿着制服敲门，手里拿着试纸，要求居民张开嘴巴交出DNA 证明他们并没有杀死附近公园里的慢跑者。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    并且事情变得更糟了。
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     对于那些已被明确排除在此案件之外的人，该市犯罪实验室的首席医学检查官办公室也将这些DNA数据
    </em>
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://gothamist.com/2019/01/10/dna_nypd_judge.php" href="http://gothamist.com/2019/01/10/dna_nypd_judge.php" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     <em class="markup--em markup--p-em">
      永久地保存
     </em>
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     在他们的数据库中
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    ，并定期将其与所有城市的犯罪情况进行比较
   </em>
   。
  </p>
  <p class="graf graf--p">
   同时，首席医学检查官办公室已经明确表示增加数据收集是合理的 — 而不是制定自己的规则来管理他们如何使用某人的DNA。
  </p>
  <p class="graf graf--p">
   简而言之 — 任何向纽约警察局提供DNA的人都会定期与每一次强奸、谋杀和任何其他暴力犯罪的DNA样本进行比较。
  </p>
  <p class="graf graf--p">
   但是等等，你可能会说 — 如果你没有做错什么，为什么要担心？
  </p>
  <p class="graf graf--p">
   你应该非常担心。因为，正如“每日新闻”所指出的那样，“DNA具有无可比拟的错误控制能力。
   <strong class="markup--strong markup--p-strong">
    基因测试现在可以揭示基于少数细胞的特征，我们每天都会丢失数十万个细胞，也就是说，你的DNA 可以出现在任何你从未到过的地方
   </strong>
   。
  </p>
  <p class="graf graf--p">
   例如，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.themarshallproject.org/2018/04/19/framed-for-murder-by-his-own-dna" href="https://www.themarshallproject.org/2018/04/19/framed-for-murder-by-his-own-dna" rel="noopener noreferrer" target="_blank">
    Lukis Anderson 的 DNA
   </a>
   是在一名被谋杀的亿万富翁的指甲下被发现的，他并没有杀人。但是安德森在2013年的赎罪前面临死刑。
  </p>
  <p class="graf graf--p">
   原因是这样的。在 van Oorschot 的一项实验中，坐在一起聊天喝饮品20分钟后，将拭子放在被试者的手、椅子、桌子、水壶和果汁杯上，都能测试到遗传物质；每个参与者至少触摸过一次水壶，几乎在每种情况下都会在手柄上找到每个人的DNA；虽然参与者之间从未相互接触过，但三分之一的人手掌上带有另一个人的DNA；三分之一的人眼镜上带有其他人的DNA，他们没有互相接触过的前提下。唯一的解释是：
   <strong class="markup--strong markup--p-strong">
    参与者不知不觉中沾染到了他人的DNA
   </strong>
   ，也许是他们那天早上亲吻的情人、与他们共用公交车的陌生人、或是他们下午买拿铁的咖啡师。
  </p>
  <p class="graf graf--p">
   看一个视频吧：
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="314" src="//www.youtube.com/embed/Ou3j3CGcPPo" width="560">
   </iframe>
  </p>
  <p class="graf graf--p">
   根据警方合同，普通官员不给实验室准确的目标DNA，这意味着实验室不能轻易地排除可能的犯罪现场污染。这意味着敲门收集DNA的警员要求人们自愿做出他们不该做的事。
  </p>
  <p class="graf graf--p">
   基本隐私是另一个真正的担忧。我们每天都看到我们的个人信息一旦被放在互联网上，就永远无法重新获得保护。过去几年来的发展预示着 DNA 的使用令人难以置信的扩展。你现在就可以想象一下30年后的可能性。
  </p>
  <p class="graf graf--p">
   你认为私人 DNA 测试公司是安全的吗 — 如 FamilyTreeDNA，不断增长的市场的先驱之一，但是它也一样，就如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.buzzfeednews.com/article/salvadorhernandez/family-tree-dna-fbi-investigative-genealogy-privacy" href="https://www.buzzfeednews.com/article/salvadorhernandez/family-tree-dna-fbi-investigative-genealogy-privacy" rel="noopener noreferrer" target="_blank">
    BuzzFeed 的报告
   </a>
   所证实的那样，该公司已经悄然授予联邦调查局访问其近200万遗传资料的巨大资源。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    T
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.zerohedge.com/news/2019-02-12/inside-nypds-out-control-dna-collection-practices" href="https://www.zerohedge.com/news/2019-02-12/inside-nypds-out-control-dna-collection-practices" rel="noopener noreferrer" target="_blank">
    he NYPD is going to great lengths to collect, and permanently store DNA from everyone they can — guilty or innocent, according to the NY Daily News
   </a>
   .
  </p>
