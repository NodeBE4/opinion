      <strong>
       “你是受贿人，所以你无权争辩” —— 他们这样告诉你。不，这不是人们选择接受的贿赂，而是强加给每个人的。基于这一问题的辩论将是能否扭转反乌托邦大跃进的关键
      </strong>
     </em>
    </span>
   </li>

  <p>
   <img alt="" class="aligncenter size-full wp-image-6415 jetpack-lazy-image" data-lazy-sizes="(max-width: 1012px) 100vw, 1012px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/便利.jpg?resize=1012%2C550&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/便利.jpg?w=1012&amp;ssl=1 1012w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/便利.jpg?resize=300%2C163&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/便利.jpg?resize=768%2C417&amp;ssl=1 768w" data-recalc-dims="1" height="550" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/便利.jpg?resize=1012%2C550&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1012"/>
   <noscript>

   </noscript>
  </p>
  <p class="graf graf--p">
   李彦宏被泼了一瓶矿泉水。那是7月初发生的“热点事件”，我们当时没有评论它，主要因为在我们看来这瓶水是一项未完成的事业，或者说，它是不够的。
  </p>
  <p class="graf graf--p">
   百度的可恶不仅在于它和其他监视资本主义寡头一样的本性，并且，李彦宏还公开声称“中国人愿意用便利交换隐私”；这和扎克伯格厚颜无耻地称其用户“像傻逼一样地信任我”，基本没有区别。
  </p>
  <p class="graf graf--p">
   但是，以便利交换隐私的辩词并非李彦宏发明的，监视资本主义寡头的发言人经常会以各种版本声称这点；甚至，有些最初的辩论者已经受到这一错误逻辑的影响而开始退缩。
  </p>
  <p class="graf graf--p">
   如果这一邪恶逻辑继续存在，将更加快速地变本加厉。
  </p>
  <p class="graf graf--p">
   Facebook 的天秤座就是一个新的典型。
  </p>
  <p class="graf graf--p">
   延伸《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%9d%80%e6%ad%bb%e5%a4%a9%e7%a7%a4%e5%ba%a7%e5%9c%a8%e8%90%8c%e8%8a%bd%e7%8a%b6%e6%80%81/" href="https://www.iyouport.org/%e6%9d%80%e6%ad%bb%e5%a4%a9%e7%a7%a4%e5%ba%a7%e5%9c%a8%e8%90%8c%e8%8a%bd%e7%8a%b6%e6%80%81/" rel="noopener noreferrer" target="_blank">
    杀死天秤座在萌芽状态
   </a>
   》。
  </p>
  <p class="graf graf--p">
   如果您还记得我们在今年的周年纪念文章中所陈述的观点，关于为什么监视全球的惊人内幕曝光后六年而没有得到任何解决？为什么说我们所有人都已经成为反乌托邦世界的帮凶？详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/6%e5%b9%b4%ef%bc%8c%e6%94%b9%e5%8f%98%e4%ba%86%e4%bb%80%e4%b9%88%ef%bc%9f/" href="https://www.iyouport.org/6%e5%b9%b4%ef%bc%8c%e6%94%b9%e5%8f%98%e4%ba%86%e4%bb%80%e4%b9%88%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    6年，改变了什么？
   </a>
   》
  </p>
  <p class="graf graf--p">
   是否继续屈服于便利的暴政，是决定未来的关键。
  </p>
  <p class="graf graf--p">
   当 Facebook 宣布该公司的全球货币天秤座时，它表达了希望“为人们和企业提供便捷的服务，例如只需按一下按钮即可支付账单，通过扫描代码购买一杯咖啡或乘坐当地公共交通工具而无需携带现金或地铁通行证”。
  </p>
  <p class="graf graf--p">
   当 JetBlue 希望使用面部识别软件办理登机手续服务而受到压力时，它在
   <a class="markup--anchor markup--p-anchor" data-href="https://mediaroom.jetblue.com/investor-relations/press-releases/2018/11-15-2018-184045420" href="https://mediaroom.jetblue.com/investor-relations/press-releases/2018/11-15-2018-184045420" rel="noopener noreferrer" target="_blank">
    新闻稿
   </a>
   中解释说：“登机接触点是一个需要创新的领域，我们认为生物识别将改变航空旅行的未来，因为我们希望整个机场的旅程更加无缝”。
  </p>
  <p class="graf graf--p">
   如果您担心亚马逊的家庭助理 Echo 如何使用它收集的有关您的数据、或者关于亚马逊员工正在窃听您的对话，那么
   <strong class="markup--strong markup--p-strong">
    该公司绝对会确保您了解 Alexa 将如何“让生活更轻松”，以便能说服您接受监视
   </strong>
   。
  </p>
  <p class="graf graf--p">
   几乎在所有例子中都是如此：人们将行动中的便利价值视为技术的基本原理：所谓的舒适和“无摩擦”，看起来生活“更顺利”。
  </p>
  <p class="graf graf--p">
   这种可取性似乎是不言而喻的。为什么我们不能希望事情变得更容易、更有效、更舒适，对吧？你会说被步行者踩出来的路往往都是捷径。
  </p>
  <p class="graf graf--p">
   但是 ，
   <strong class="markup--strong markup--p-strong">
    如果这条“捷径”必须穿越蟒蛇的领地呢？是否要舍命图“便利”？
   </strong>
  </p>
  <p class="graf graf--p">
   也许答案不是很简单 。
  </p>
  <p class="graf graf--p">
   以牺牲其他价值甚至更大价值为代价来优先考虑便利，不仅仅是自然的或下意识的 —— 它还很有历史。
  </p>
  <p class="graf graf--p">
   就如在 One / Zero 最近的一篇文章“便利的暴政”中，Colin Horgan 所写道的：“科技公司目前创造的便利性和我们大多数人所接受的便利性，是我们将在哪里结束的关键 —— 生活在一个我们都选择了的世界里，但似乎没有人真的想要。”
  </p>
  <p class="graf graf--p">
   这个不受欢迎的世界包括被剥夺的隐私人权、被实时监视的定位广告、算法暴政的操纵和各种注意力精神剥夺。
  </p>
  <p class="graf graf--p">
   这些是我们为触手可及的天气应用程序、乘车服务、搜索引擎、以及点播娱乐的小愿望，而付出的
   <strong class="markup--strong markup--p-strong">
    巨大代价
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Horgan 表示，
   <strong class="markup--strong markup--p-strong">
    既然便利性是“我们个人触手可及”的价值，它最终会“超过隐私、民主或平等……等一切更为抽象的概念”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   然而
   <strong class="markup--strong markup--p-strong">
    “便利性”只是一个魔鬼的披风
   </strong>
   ，不仅在于个人层面上的 —— 它正在掠夺我们的人权和自主权，而且在于人与人关系层面上的：“我”的便利通常是由别人付出的代价。
  </p>
  <p class="graf graf--p">
   这些隐藏的成本往往只会加剧现有的不平等和不公正。
  </p>
  <p class="graf graf--p">
   例如，面部识别的部署可能会使一些人通过安全检查点更方便，但同时，也会使那些更有可能被这种软件误识别的人和被当权者瞄准的异议人士的生活变得更加危险。
  </p>
  <p class="graf graf--p">
   隐私、民主和平等这些抽象概念只能由那些认为它们是理所当然的人去体验。
  </p>
  <p class="graf graf--p">
   而且，
   <strong class="markup--strong markup--p-strong">
    随着这些抽象理想的失败，更加巨大的不便正在以新的或未曾预料的形式覆盖所有人。
   </strong>
  </p>
  <p class="graf graf--p">
   最终，便利性停止促进其他任何目标，其本身成为一个全面控制的终极目的，不再有价值。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-6416 jetpack-lazy-image" data-lazy-sizes="(max-width: 936px) 100vw, 936px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-7.png?resize=936%2C716&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-7.png?w=936&amp;ssl=1 936w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-7.png?resize=300%2C229&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-7.png?resize=768%2C587&amp;ssl=1 768w" data-recalc-dims="1" height="716" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-7.png?resize=936%2C716&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="936"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6416" data-recalc-dims="1" height="716" sizes="(max-width: 936px) 100vw, 936px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-7.png?resize=936%2C716&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-7.png?w=936&amp;ssl=1 936w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-7.png?resize=300%2C229&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-7.png?resize=768%2C587&amp;ssl=1 768w" width="936"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   社会学家 Thomas F. Tierney 在1993年发布的著作《便利的价值：技术文化的谱系》是在互联网全面发展之前的真知灼见，
   <strong class="markup--strong markup--p-strong">
    如今这本书仍然是对“便利性”如何塑造我们的技术环境的有益探索
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这本书将便利性的概念不仅仅局限于自然和固有的易用性和舒适性。
  </p>
  <p class="graf graf--p">
   并且，它指出了关键：
   <strong class="markup--strong markup--p-strong">
    便利性并不是“由个人自由选择的，而是现代性技术秩序在各个方面所要求的”。
   </strong>
  </p>
  <p class="graf graf--p">
   也就是说，
   <strong class="markup--strong markup--p-strong">
    “方便”，从根本上来说，是强加给我们的。
   </strong>
  </p>
  <p class="graf graf--p">
   接下来的问题是我们如何接受这一价值作为我们自己的价值，甚至以牺牲其他更为实质性的理想做为代价。
  </p>
  <p class="graf graf--p">
   现代性提出，通过“消费各种技术设备”可以克服这些障碍。Tierney 指出，当技术带来便利时，这就是他们的承诺：克服身体某些障碍
   <strong class="markup--strong markup--p-strong">
    而不是
   </strong>
   仔细地满足其需求。
  </p>
  <p class="graf graf--p">
   他将便利性视为一种禁欲主义，而不是自我放纵或自私。
  </p>
  <p class="graf graf--p">
   也许违反直觉，但是，
   <strong class="markup--strong markup--p-strong">
    这一解释有助于将一系列奇特的硅谷风潮置于背景之中
   </strong>
   ：杰克多尔西的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.rollingstone.com/culture/culture-features/jack-dorsey-diet-intermittent-fasting-820185/" href="https://www.rollingstone.com/culture/culture-features/jack-dorsey-diet-intermittent-fasting-820185/" rel="noopener noreferrer" target="_blank">
    间歇性斋戒
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/jack/status/1071575088695140353?lang=en" href="https://twitter.com/jack/status/1071575088695140353?lang=en" rel="noopener noreferrer" target="_blank">
    冥想静修
   </a>
   ；伊隆马斯克的睡眠剥夺; Ray Kurzweil 的饮食和药丸
   <a class="markup--anchor markup--p-anchor" data-href="https://www.businessinsider.com/ray-kurzweils-immortality-diet-2015-4" href="https://www.businessinsider.com/ray-kurzweils-immortality-diet-2015-4" rel="noopener noreferrer" target="_blank">
    驱动的
   </a>
   奇点。
  </p>
  <p class="graf graf--p">
   代餐公司
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Soylent_(meal_replacement)" href="https://en.wikipedia.org/wiki/Soylent_%28meal_replacement%29" rel="noopener noreferrer" target="_blank">
    Soylent
   </a>
   的商业逻辑特别体现了便利性、禁欲主义和自我掌控之间的这种联系。
  </p>
  <p class="graf graf--p">
   身体对营养的需求不是必须遵守的规则，更不用说享受了。它只是尽可能被消除的东西。
   <strong class="markup--strong markup--p-strong">
    这种态度最终导致了硅谷的后人类主义 —— 后者将身体本身设置为完全不方便，并想象甚至努力实现了一个人类完全放弃自身的未来
   </strong>
   。
  </p>
  <p class="graf graf--p">
   毕竟，死亡本身才是最终限制。
  </p>
  <p class="graf graf--p">
   将死亡视为一个需要解决的问题，Tierney 引用了天文学家 Robert Jastrow 1981年发表的未来学作品《魔法织机：宇宙中的心灵》。
  </p>
  <p class="graf graf--p">
   当时的这本书写道：“最终，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%b6%85%e7%ba%a7%e6%88%98%e5%a3%ab%ef%bc%9adarpa-%e6%83%b3%e8%a6%81%e5%88%b6%e9%80%a0%e7%b2%be%e7%a5%9e%e6%8e%a7%e5%88%b6%e7%9a%84%e6%88%98%e4%ba%89%e6%ad%a6%e5%99%a8/" href="https://www.iyouport.org/%e8%b6%85%e7%ba%a7%e6%88%98%e5%a3%ab%ef%bc%9adarpa-%e6%83%b3%e8%a6%81%e5%88%b6%e9%80%a0%e7%b2%be%e7%a5%9e%e6%8e%a7%e5%88%b6%e7%9a%84%e6%88%98%e4%ba%89%e6%ad%a6%e5%99%a8/" rel="noopener noreferrer" target="_blank">
    人类大脑被安置在计算机中
   </a>
   ，已经摆脱了道德肉体的弱点。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “大脑被连接到相机、仪器和引擎控制，大脑看到、感觉和响应刺激。它完全 被控制了”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    将我们对技术的使用定位于所谓的便利性，反映了禁欲主义甚至虚无主义假设的成功。
   </strong>
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-6417 jetpack-lazy-image" data-lazy-sizes="(max-width: 864px) 100vw, 864px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.png?resize=864%2C700&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.png?w=864&amp;ssl=1 864w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.png?resize=300%2C243&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.png?resize=768%2C622&amp;ssl=1 768w" data-recalc-dims="1" height="700" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.png?resize=864%2C700&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="864"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6417" data-recalc-dims="1" height="700" sizes="(max-width: 864px) 100vw, 864px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.png?resize=864%2C700&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.png?w=864&amp;ssl=1 864w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.png?resize=300%2C243&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.png?resize=768%2C622&amp;ssl=1 768w" width="864"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   便利性是一种自相矛盾的愿望：
   <strong class="markup--strong markup--p-strong">
    “需要尽快完成一件事并尽快获得优势是一种永远无法满足的需求，”
   </strong>
   Tierney 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “每一次进步都会带来新的障碍，并需要一种更精致或新的技术形式。”
  </p>
  <p class="graf graf--p">
   因为它主要涉及手段和旨在对目标保持中立，所以便利性非常适合自由秩序，但其前提应该是支持个人选择自己目标的权利。
  </p>
  <p class="graf graf--p">
   因此，便利性不是关于你去哪里或你在做什么，而是让旅程更加便捷和舒适。广告宣传的便利性很大程度上是为制造一种错觉。
  </p>
  <p class="graf graf--p">
   我们很少有机会询问便利性是否适合我们追求的目标，或者是否能实现其休闲的承诺。唯一的重点是继续前进，更快，更有效。
  </p>
  <p class="graf graf--p">
   任何方向的恒定运动都不是“终点”。
   <strong class="markup--strong markup--p-strong">
    如果你不知道目标是什么、为了什么、代价是什么，那么节省时间就是徒劳的。
   </strong>
  </p>
  <p class="graf graf--p">
   Tierney 引用了 Ruth Schwartz Cowan 1983年关于节省时间的家庭技术的经典研究 “More Work for Mother”，该研究证明了20世纪早期的技术如洗衣机、吸尘器和洗碗机的采用，
   <strong class="markup--strong markup--p-strong">
    很少
   </strong>
   能直接转化为女性的休闲时间。
  </p>
  <p class="graf graf--p">
   这些节省时间的技术重新配置了家庭中的角色，并以这样的方式重新构建了期望，即：
   <strong class="markup--strong markup--p-strong">
    家庭主妇通常最终承担了更多而不是更少的工作。
   </strong>
  </p>
  <p class="graf graf--p">
   Cowan 的分析专门针对特定时间和人群，但每当新技术进入社会或制度生态系统时，这些模式都会重演。
  </p>
  <p class="graf graf--p">
   一些人的收益变成了他人的损失、角色被重新配置、新任务被不断生成、合作结构可能被取代 ……
  </p>
  <p class="graf graf--p">
   换句话说，实现便利的承诺往往是复杂的，甚至表面上的受益者本身可能也会失去更多的收益。
  </p>
  <p class="graf graf--p">
   Neil Postman 在其1992年出版的《预约技术》一书中指出，技术社会中的“赢家”不会告诉“失败者”“他们的生活将更有效地进行”，也就是说更便利。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他们拒绝说清效率的成本”。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-6418 jetpack-lazy-image" data-lazy-sizes="(max-width: 1010px) 100vw, 1010px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-5.png?resize=1010%2C712&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-5.png?w=1010&amp;ssl=1 1010w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-5.png?resize=300%2C211&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-5.png?resize=768%2C541&amp;ssl=1 768w" data-recalc-dims="1" height="712" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-5.png?resize=1010%2C712&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1010"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6418" data-recalc-dims="1" height="712" sizes="(max-width: 1010px) 100vw, 1010px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-5.png?resize=1010%2C712&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-5.png?w=1010&amp;ssl=1 1010w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-5.png?resize=300%2C211&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-5.png?resize=768%2C541&amp;ssl=1 768w" width="1010"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在90年代那本书里，Postman 有先见之明地问道：“计算机技术在多大程度上对大众有利？ … 人
   <strong class="markup--strong markup--p-strong">
    们的私人事务更容易被强大的机构所接管；人们更容易被操纵和控制；接受更多以权势的价值观为标准的检验；活生生的人经常仅仅被简化为一个数字对象……
   </strong>
   ”。
  </p>
  <p class="graf graf--p">
   因为便利性是针对效率的，所以不会让你更清闲。
  </p>
  <p class="graf graf--p">
   相反，
   <strong class="markup--strong markup--p-strong">
    便利性往往只会加剧对生产力的需求，使我们对更多的产出负责。
   </strong>
  </p>
  <p class="graf graf--p">
   因此，
   <strong class="markup--strong markup--p-strong">
    所谓的便利性技术所提供的需求严格来说并不是消费者的需求。
   </strong>
  </p>
  <p class="graf graf--p">
   从生产者的角度来看，消费者认为的方便是“技术进步的领导者，科学家和技术人员” —— 一种主宰自然的愿望。
  </p>
  <p class="graf graf--p">
   但这显然只是极少数人的特权，而非大众的权力。
  </p>
  <p class="graf graf--p">
   曾经被称为解放工具的数字技术现在看起来只是压迫性的工具。那些曾经期待技术产生民主的人，如今已经清楚地发现，这只是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.jstor.org/stable/3101118?origin=crossref&amp;seq=1#page_scan_tab_contents" href="https://www.jstor.org/stable/3101118?origin=crossref&amp;seq=1#page_scan_tab_contents" rel="noopener noreferrer" target="_blank">
    Lewis Mumford 称之为“专制主义工艺”
   </a>
   的高度集权和控制系统。
  </p>
  <p class="graf graf--p">
   Mumford 在1964年的作品中就精准地描述了当下的世界：
   <strong class="markup--strong markup--p-strong">
    “我们只能在被批准的红线内讨价还价，…… 他们采取的是宏伟的贿赂形式；只有在他们的框架内你才能要求一些最基本的东西……”。
   </strong>
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    “你无法要求系统不提供的东西。你能得到的只是和其他人同样的加工和制造、均质化和均衡化的，以系统认为正确的数量配发。是他们想要给你的，而不是你想要获得的”。
   </strong>
  </p>
  <p class="graf graf--p">
   就如 Horgan 当年所担忧的，所谓的便利性烟雾不仅仅是取代了隐私、平等和民主；并且，已经摧毁它们。
  </p>
  <p class="graf graf--p">
   现在这一担忧几乎成为了现实。
  </p>
  <p class="graf graf--p">
   过去二十年来，领先科技寡头积累的巨大财富和权力已经足够证实了这一点。
  </p>
  <p class="graf graf--p">
   他们正是通过提供“便利性”来实现这点的。
  </p>
  <p class="graf graf--p">
   我们呢？我们究竟有没有得到“便利性”的优越一面？没人回答。也许没人愿意承认接受了这种狡诈的贿赂，并因此付出了一切。
  </p>
  <p class="graf graf--p">
   Mumford 是对的，这就是一种贿赂，无论它看起来多么壮观。他们授予一点点便利性，就把所有人关入了该系统的牢笼。
  </p>
  <p class="graf graf--p">
   这种狡诈的贿赂从开始到现在几乎没有任何失败。
  </p>
  <p class="graf graf--p">
   当然，简单地拒绝这种便利性并不足以使一切变得顺利。需要更高明的办法，要让这种贿赂彻底失败。⚪️
  </p>
