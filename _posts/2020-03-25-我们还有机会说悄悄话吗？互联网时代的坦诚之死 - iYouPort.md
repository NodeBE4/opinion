   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/us-news/2019/jul/07/donald-trump-inept-and-dysfunctional-uk-ambassador-to-us-says" href="https://www.theguardian.com/us-news/2019/jul/07/donald-trump-inept-and-dysfunctional-uk-ambassador-to-us-says" rel="noopener noreferrer" target="_blank">
    泄露给了
   </a>
   《星期日邮报》。
  </p>
  <p class="graf graf--p">
   Darroch 的罪名绝非哪种 “不正确的行为”。全世界似乎唯一不同意这位大使的评估的人是特朗普本人 —— 特朗普在随后的一系列失常的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/us-news/2019/jul/09/donald-trump-calls-theresa-may-foolish-as-diplomatic-row-escalates" href="https://www.theguardian.com/us-news/2019/jul/09/donald-trump-calls-theresa-may-foolish-as-diplomatic-row-escalates" rel="noopener noreferrer" target="_blank">
    推文中
   </a>
   称这位公务员“古怪”和“非常愚蠢”。
  </p>
  <p class="graf graf--p">
   但是不可否认的是，这些备忘录并不是外交行为。
   <strong class="markup--strong markup--p-strong">
    驻美国大使的一项重要技能是平衡两个相互竞争的要求：诚实地向上级报告总统无能的事实，同时，与无能的总统保持友好。
   </strong>
  </p>
  <p class="graf graf--p">
   Darroch 绝不是第一个因私人想法被曝光而面临麻烦的人。
  </p>

  <p class="graf graf--p">
   越来越清楚的是，实际上只有两种方法可以避免这种情况。
  </p>
  <p class="graf graf--p">
   首先是要避免进入公众视野。
  </p>
  <p class="graf graf--p">
   但这越来越难：您可能会忽然被某个名人转推、虽然您的 Facebook 帖子已设置仅限好友但却在一个私人群组中被分享，或者只是被拍摄到在公共场合做了一些奇怪虽然无害的事 ……
  </p>
  <p class="graf graf--p">
   你必须承认 “public” 是一个滑动的指标。
  </p>
  <p class="graf graf--p">
   原本发送给只有4人的WhatsApp群组的消息被重新发送到有20人的群组中，或者更大的上百人的群组，您仍然可以觉得它是 “未公开” 的，但是，如果您在被朋友出卖方面毫无经验，那么它可能毁了你的人生。
  </p>
  <p class="graf graf--p">
   因此，第二点是：确保您不论是公开还是私下，仅仅谈论关于任何问题的大众化的、礼貌的、和被认可的意见。其余一概不要说。
  </p>
  <p class="graf graf--p">
   如果您必须做到直言不讳，请尝试确保在面对面的对话中，最好是没有任何笔记和录音的情况下进行。就像奥巴马用的那种衬有锡纸的帐篷。
  </p>
  <p class="graf graf--p">
   即使这样，您仍可能会遇到对话者令人尴尬的回忆录。
  </p>
  <p class="graf graf--p">
   因此，也许最好保持沉默……？？？
  </p>
  <p class="graf graf--p">
   但是，坦白地说，直言不讳的死亡绝对是一场灾难。
  </p>
  <p class="graf graf--p">
   您只能选择那些经过仔细衡量的字词，你必须假定一切都会在某一天被公开甚至被用来断章取义地羞辱你。即使是关上门说悄悄话，面对一个无能的总统所产生的愤怒也必须被压制下去。
  </p>

  <p class="graf graf--p">
   稍停片刻，盘点一下近几年来你被浪费掉的生活，其中有多少东西已经被记录、被存档、并可搜索。熟悉 “开源情报” 调查的每个人都不会怀疑这点。
  </p>
  <p class="graf graf--p">
   您可能已经很少打电话了，而是通过 WhatsApp、Facebook 和 Twitter 发送更多的消息。
   <strong class="markup--strong markup--p-strong">
    现在几乎可以肯定的是，您的一举一动、每一个想法都有记录，要么藏在 iPhone 里，要么存储在 Google 和腾讯的服务器上
   </strong>
   。
  </p>
  <p class="graf graf--p">
   并且可以肯定的是，如果您是一位早期互联网使用者，那么您可能已经在十年前就开始在 Facebook 上发布并标记了个人照片，而今天的机器学习算法利用的就是你曾经发布过的这些内容 —— 用来识别你。
  </p>

    <img alt="" class="aligncenter size-full wp-image-10458" data-recalc-dims="1" height="1230" sizes="(max-width: 800px) 100vw, 800px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-9.png?resize=800%2C1230&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-9.png?w=800&amp;ssl=1 800w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-9.png?resize=195%2C300&amp;ssl=1 195w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-9.png?resize=768%2C1181&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-9.png?resize=666%2C1024&amp;ssl=1 666w" width="800"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   那是因为我们的辩论中缺少这句话：他们侵犯了某种形式的隐私。企业家和激进主义者 MaciejCegłowski 称其为“环境隐私”：“这种理解使我们彼此之间的日常互动具有价值，它不应受到监控，而且我们日常生活中的小细节也应被遗忘。我们在家里、在工作中、在教堂、学校或业余时间里所做的一切都不应该被永久记录。绝大多数谈话不应该被存储。”
  </p>
  <p class="graf graf--p">
   坦诚的死亡是我们日常生活中广泛缺乏环境隐私的一种征兆。
  </p>
  <p class="graf graf--p">
   监视资本主义状态不仅阻止所有人公开的言论，甚至私下的言论也不再安全。这种状态不允许你通过互联网传输任何可能危害自己就业前景的信息。
   <strong class="markup--strong markup--p-strong">
    你必须审核自己所有言话，以确保它在所呈现的上下文之外没有任何其他的引申含义。
   </strong>
  </p>
  <p class="graf graf--p">
   Cegłowski 指出，没有外部机构来强制进行这种自我审查，至少在西方没有，这是人们自主进行的：在中国和沙特阿拉伯那种国家，同样的能力被用作社会控制的工具。
  </p>
  <p class="graf graf--p">
   在西方，
   <strong class="markup--strong markup--p-strong">
    监视资本家正在使用对你的数据的收集和分析以操纵你的行为。全面监视的基础架构在全球都是相同的，并且各处都在大规模部署
   </strong>
   。
  </p>
  <p class="graf graf--p">
   另一个框架使问题变得清晰了。问题不在于数据泄露：而是“数据排放”，即 作为我们数字生活的副产品而产生的信息。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果它仅仅被用来压制坦诚、削弱我们表达自己的自由、并为威权国家提供令人恐惧的能力，那么这种数字排放的风险就已经是非常严重的。
   </strong>
  </p>
  <p class="graf graf--p">
   但问题更有，竞争环境是不公平的。有两组人浮出水面，周围的环境隐私的消失完全不会伤及他们。
  </p>
  <p class="graf graf--p">
   第一种是天使：那些内心深处的思想已经可以安全表达的人，那种永远不需要在背后说话的人，因为他们想要说的话可以自由表达出来，而不会有任何无所事事的评论来咬他们，他们说的每个字都经过深思熟虑，没有把柄。
  </p>
  <p class="graf graf--p">
   这样的人几乎不存在。
  </p>
  <p class="graf graf--p">
   第二类则是摒弃了一致性的观念、将粗鲁性提升为一种美德的人，他们破坏了共享现实的整个概念。
  </p>
  <p class="graf graf--p">
   如果您以一贯的诚实为基础建立了声誉，那么一次虚伪就可能毁于一旦。但是，如果您因撒谎被报纸开除后当选了国会议员 …… 你几乎可以不用写任何东西了；你还会被贴在公交车身上，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/politics/2013/mar/24/boris-johnson-interview-eddie-mair" href="https://www.theguardian.com/politics/2013/mar/24/boris-johnson-interview-eddie-mair" rel="noopener noreferrer" target="_blank">
    就像 Boris Johnson 那样
   </a>
   。
  </p>
  <p class="graf graf--p">
   同样，在背后将总统称之为无能也可能有失大使风范；但是，如果像 Nigel Farage 那样养成习惯，面对谈判对手时
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2010/feb/25/nigel-farage-herman-van-rompuy-damp-rag" href="https://www.theguardian.com/world/2010/feb/25/nigel-farage-herman-van-rompuy-damp-rag" rel="noopener noreferrer" target="_blank">
    直接说
   </a>
   “你有令人失望的所有魅力和低级银行业务员的外表” …… 那这封邮件被泄漏又怎么可能会伤害到您呢？
  </p>
  <figure aria-describedby="caption-attachment-10459" class="wp-caption aligncenter" id="attachment_10459" style="width: 1128px">
   <img alt="" class="size-full wp-image-10459 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-5.jpg?resize=1100%2C660&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-5.jpg?w=1128&amp;ssl=1 1128w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-5.jpg?resize=300%2C180&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-5.jpg?resize=768%2C461&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-5.jpg?resize=1024%2C615&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-5.jpg?resize=1100%2C660&amp;ssl=1 1100w" data-recalc-dims="1" height="660" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-5.jpg?resize=1100%2C660&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="size-full wp-image-10459" data-recalc-dims="1" height="660" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-5.jpg?resize=1100%2C660&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-5.jpg?w=1128&amp;ssl=1 1128w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-5.jpg?resize=300%2C180&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-5.jpg?resize=768%2C461&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-5.jpg?resize=1024%2C615&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-5.jpg?resize=1100%2C660&amp;ssl=1 1100w" width="1100"/>
   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-10459">
    Kim Darroch shakes hands with Donald Trump, 2018. Photograph: @KimDarroch/Twitter
   </figcaption>
  </figure>
  <p class="graf graf--p">
   威胁不仅仅是对政治的。我们越是去适应缺乏“环境隐私”的环境，世界就越会更加分裂：
   <strong class="markup--strong markup--p-strong">
    我们中最厚颜无耻的人会升至顶峰，他们的糟糕表现就根植于他们的声誉中，他们可以毫无顾忌地不断重新诠释这点
   </strong>
   ，而其余的所有人不得不远离公众领域，不断寻找着那种不会保存记录的新空间。
  </p>
  <p class="graf graf--p">
   但是，就如同您不可能仅仅通过改变自己的生活习惯来应对气候危机一样，您也无法自己赢得 “环境隐私”。
  </p>
  <p class="graf graf--p">
   风险是社会的风险，需要在社会层面上加以应对。您可以尝试，当然。退出 Facebook 可能等同于将您的汽车换成自行车；删除所有推文，就像坐火车去度假而放弃飞机……
  </p>
  <p class="graf graf--p">
   这样做会有所帮助，如果每个人都这样做，就会带来很多帮助。
  </p>
  <p class="graf graf--p">
   但是，大多数人都没有这样做。仅仅极少数人的个人选择可能没有帮助。
  </p>
  <p class="graf graf--p">
   将塑料袋换成帆布袋可能需要数年才能看到回报。从公开的 Twitter 转移到私密的 Facebook 群组却只能给人一种隐私的错觉。它仍在 Facebook 上，仍然受到监视、存档和算法的操纵。
  </p>
  <p class="graf graf--p">
   Cegłowski 总结说：“大规模监视的基础设施太复杂，技术寡头垄断太强大，以至于谈论所谓的知情同意已经没有任何意义。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我相信Google的 Sundar Pichai 和Facebook 的扎克伯格对隐私的个人承诺是真诚的，就像我确信美孚石油和壳牌公司的首席执行官们也不希望自己的孩子生活在失控的全球变暖世界里。但是，他们的核心业务活动与他们自称的价值观完全不相符。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们关于隐私的讨论需要扩展，以解决有关自动化作用的基本问题：我们的一举一动都进入了公司数据库的后果是什么？我们的孩子从小就被迫适应各种机器学习算法以塑造其行为，这意味着什么？”
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%bd%93-%e4%b8%8d%e4%bd%9c%e6%81%b6-%e6%88%90%e4%b8%ba%e7%a9%ba%e8%b0%88%ef%bc%8c%e5%ae%9a%e5%88%b6%e4%ba%ba%e5%b0%86%e7%bb%88%e7%bb%93%e6%b0%91%e4%b8%bb/" href="https://www.iyouport.org/%e5%bd%93-%e4%b8%8d%e4%bd%9c%e6%81%b6-%e6%88%90%e4%b8%ba%e7%a9%ba%e8%b0%88%ef%bc%8c%e5%ae%9a%e5%88%b6%e4%ba%ba%e5%b0%86%e7%bb%88%e7%bb%93%e6%b0%91%e4%b8%bb/" rel="noopener noreferrer" target="_blank">
     定制人如何终结了民主
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   但个人行动依旧很重要。它可以作为支持更广泛斗争的信号。更多人开始加入斗争就会给寡头公司带来压力；反抗的意识在个人心理层面也是很有帮助的。
  </p>
  <p class="graf graf--p">
   下载一个诸如 Jumbo 之类的隐私应用程序，并使用它来锁定您的 Facebook，清理您的Google历史记录并清除您的推文。从 WhatsApp 切换到 Signal，从 Instagram 切换到 Snapchat，读过的邮件随手清除。把 Alexa 从您的房子里扔出去 …… 为坦诚而战。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    不要让《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e5%ae%b6%e5%86%b0%e7%ae%b1%e5%a6%82%e4%bd%95%e4%bd%9c%e4%b8%ba%e5%91%88%e5%a0%82%e8%af%81%e4%be%9b%ef%bc%9f/" href="https://www.iyouport.org/%e4%bd%a0%e5%ae%b6%e5%86%b0%e7%ae%b1%e5%a6%82%e4%bd%95%e4%bd%9c%e4%b8%ba%e5%91%88%e5%a0%82%e8%af%81%e4%be%9b%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     你家冰箱成为呈堂证供
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   但是，如果您确实认为您的电子邮件可能有一天会被大声朗读出来，那么就遵循 Nuzzi 的建议吧。在错误的地方坦诚是一个错误，这样的错误您只会犯一次。⚪️
  </p>
