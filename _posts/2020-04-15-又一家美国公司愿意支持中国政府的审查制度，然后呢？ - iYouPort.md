      <strong>
       中国的互联网用户已经很难讨论甚至最普通的话题。现在，在另一家甘心为奴的美国公司的帮助下，情况只会变得更糟
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-10788 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-20.jpg?resize=1100%2C550&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-20.jpg?w=1440&amp;ssl=1 1440w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-20.jpg?resize=300%2C150&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-20.jpg?resize=1024%2C512&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-20.jpg?resize=768%2C384&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-20.jpg?resize=1100%2C550&amp;ssl=1 1100w" data-recalc-dims="1" height="550" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-20.jpg?resize=1100%2C550&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  </p>
  <p class="graf graf--p">
   愿意支持中国政府的审查制度、阻止可能冒犯中国独裁者的任何搜索结果 —— 这样的案例并不少，现在是，著名的美国公司、在线图片和照片的提供者 Shutterstock，为此又添了一个案例。
  </p>
  <p class="graf graf--p">
   这家公开上市的公司凭借其庞大的图像数据库，每年创造了6.39亿美元的业务，这些图像目录几乎描述了博客作者或广告商可以想象的任何事物。
  </p>
  <p class="graf graf--p">
   该公司现在在150多个国家/地区开展业务。
  </p>
  <p class="graf graf--p">
   但是在中国，Shutterstock 的产品完全不同。
  </p>
  <p class="graf graf--p">
   在9月初，Shutterstock 的工程师被赋予了新的目标：
   <strong class="markup--strong markup--p-strong">
    创建搜索黑名单，该黑名单将从查询结果图像中删除与中国政府禁止的关键字相关的任何图像。
   </strong>
  </p>
  <p class="graf graf--p">
   Intercept 得知，该黑名单已经于去年10月生效。
  </p>
  <p class="graf graf--p">
   任何拥有中国大陆IP地址的人都会发现，在 Shutterstock 中搜索 “习主席”、“毛主席”、“台湾国旗”、“独裁者”、“黄雨伞”、甚至“中国国旗”，将完全没有结果。
  </p>
  <p class="graf graf--p">
   黑名单还禁止使用这些术语的变体，包括 “雨伞运动”（目前去年席卷香
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%a6%99%e6%b8%af%e7%9a%84%e6%8a%97%e8%ae%ae%e6%b4%bb%e5%8a%a8%e4%b8%ad%e5%86%8d%e6%ac%a1%e5%bc%ba%e8%b0%83%e4%ba%86%e4%bd%a0%e7%9a%84%e8%84%b8%e5%a6%82%e4%bd%95%e6%88%90%e4%b8%ba%e6%ad%a6%e5%99%a8/" href="https://www.iyouport.org/%e9%a6%99%e6%b8%af%e7%9a%84%e6%8a%97%e8%ae%ae%e6%b4%bb%e5%8a%a8%e4%b8%ad%e5%86%8d%e6%ac%a1%e5%bc%ba%e8%b0%83%e4%ba%86%e4%bd%a0%e7%9a%84%e8%84%b8%e5%a6%82%e4%bd%95%e6%88%90%e4%b8%ba%e6%ad%a6%e5%99%a8/" rel="noopener noreferrer" target="_blank">
    港的大规模民主抗议活动
   </a>
   的前身）。
  </p>
  <p class="graf graf--p">
   Shutterstock 做出无声帮助中国审查制度的决定，正是在舆论对美国公司与习近平的专制政权之间的关系进行严格审查之际。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/DonEvansWm/status/1183713834613252096" href="https://twitter.com/DonEvansWm/status/1183713834613252096" rel="noopener noreferrer" target="_blank">
    苹果
   </a>
   、
   <a class="markup--anchor markup--p-anchor" data-href="https://variety.com/2019/gaming/news/blizzard-hearthstone-overwatch-china-hong-kong-protests-1203369240/" href="https://variety.com/2019/gaming/news/blizzard-hearthstone-overwatch-china-hong-kong-protests-1203369240/" rel="noopener noreferrer" target="_blank">
    暴雪
   </a>
   娱乐公司、
   <a class="markup--anchor markup--p-anchor" data-href="https://nymag.com/intelligencer/2019/10/daryl-morey-nba-hong-kong-protests-china-rockets.html" href="https://nymag.com/intelligencer/2019/10/daryl-morey-nba-hong-kong-protests-china-rockets.html" rel="noopener noreferrer" target="_blank">
    NBA
   </a>
   、和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e6%ad%a3%e5%9c%a8%e5%af%b9%e5%85%b6%e4%b8%ad%e5%9b%bd%e5%ae%a1%e6%9f%a5%e7%89%88%e6%90%9c%e7%b4%a2%e9%a1%b9%e7%9b%ae%e8%bf%9b%e8%a1%8c%e7%a7%98%e5%af%86%e7%9a%84%e7%bb%a9%e6%95%88/" href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e6%ad%a3%e5%9c%a8%e5%af%b9%e5%85%b6%e4%b8%ad%e5%9b%bd%e5%ae%a1%e6%9f%a5%e7%89%88%e6%90%9c%e7%b4%a2%e9%a1%b9%e7%9b%ae%e8%bf%9b%e8%a1%8c%e7%a7%98%e5%af%86%e7%9a%84%e7%bb%a9%e6%95%88/" rel="noopener noreferrer" target="_blank">
    Google
   </a>
   等家喻户晓的公司因放任中国共产党的政策指令均受到了严厉批评。
  </p>
  <p class="graf graf--p">
   决定遵循审查制度证明 Shutterstock 的价值已经发生了鲜明的转变 —— Shutterstock 将自身定位为创造力的推动者。
  </p>
  <p class="graf graf--p">
   该照片公司与中国的关系至少可以追溯到2014年，当时该公司与 ZCool（一家服务于视觉艺术家的中国社交网络和作品集网站）达成了经销协议。
  </p>
  <p class="graf graf--p">
   去年，Shutterstock 宣布对ZCool进行1500万美元的投资，并指出由于建立了合作伙伴关系，
   <strong class="markup--strong markup--p-strong">
    “ Shutterstock 的内容现在为中国的大型技术平台提供了动力，例如 腾讯社交广告”，这是中国技术寡头腾讯的在线广告子公司。
   </strong>
  </p>
  <p class="graf graf--p">
   Shutterstock 的审查功能似乎在公司内部立即引起强烈的争议，促使180多名 Shutterstock 工人签署了一份抵制搜索黑名单的请愿书，并指控
   <strong class="markup--strong markup--p-strong">
    该公司以其自身价值进行交易只为进入利润丰厚的中国市场。
   </strong>
  </p>
  <p class="graf graf--p">
   中国的互联网用户已经很难讨论甚至最普通的话题。现在，在另一家甘心为奴的美国公司的帮助下，情况只会变得更糟。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “是的，我们是一个富有创意的照片和视频市场，但我们还是社论新闻中心，” Shutterstock 的一位员工告诉 The Intercept。“想了解香港抗议活动的故事吗？它们不存在！想了解台湾吗？它从来不存在！习近平不是独裁者，因为他自己这么说的 …… 这是如此黑暗的世界。”
  </p>
  <p class="graf graf--p">
   提供给 The Intercept 的请愿文本可以在下面阅读，完整版
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2019/11/06/shutterstock-china-censorship-tech/#petition" href="https://theintercept.com/2019/11/06/shutterstock-china-censorship-tech/#petition" rel="noopener noreferrer" target="_blank">
    在这里
   </a>
   。
  </p>
  <p class="graf graf--p">
   Shutterstock 的创始人兼首席执行官 Jon Oringer 几天后回复了请愿书；那些希望改变这点的人会感到失望。
  </p>
  <p class="graf graf--p">
   Oringer 认为，Shutterstock 与中国政府在审查制度上的妥协是 “有道理的”，因为拒绝在中国开展业务而不是帮助中国政府扩大其信息控制计划，“才是真正破坏企业利益的举动”：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们是否仍将大多数内容提供给了中国的13亿公民，还是我们剥夺了他们完全访问内容的能力？我们最终相信，与我们的品牌承诺相一致，对于讲故事的人来说，能够更有效地利用我们的收藏集来创造性地、有影响力地讲述他们的故事，这是更有价值的。”
  </p>
  <p class="graf graf--p">
   Oringer 甚至称，具有定制审查功能的 Shutterstock “具有更大的力量”，并且“将比替代方案更好地为中国人民服务” …… 真是可怕的歪曲。
  </p>
  <p class="graf graf--p">
   你可以
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2019/11/06/shutterstock-china-censorship-tech/#response" href="https://theintercept.com/2019/11/06/shutterstock-china-censorship-tech/#response" rel="noopener noreferrer" target="_blank">
    在这里
   </a>
   看到 Oringer 在全公司范围内的回复。小心会吐。
  </p>
  <p class="graf graf--p">
   在搜索词黑名单的实施之后，一些员工担心公司对审查制度的使用会不断增加。
  </p>
  <p class="graf graf--p">
   Shutterstock 的工作人员告诉 The Intercept：“如果中国要求在审查黑名单中再增加多少个搜索词，我们将采取什么行动？或者，如果另一个国家也向我们提出了类似要求呢？…… 我们已经被毁了。”
  </p>
  <p class="graf graf--p">
   在发给 The Intercept 的电子邮件中，Shutterstock 发言人确认了审查功能当前处于活动状态，尽管他们无法确认自禁用关键字列表首次生效以来是否已更改或扩展。
  </p>
  <p class="graf graf--p">
   与 The Intercept 交谈的 Shutterstock 工作人员补充说，对于公司的许多人来说，该问题是个人的，他们拒绝 “遵守中国政府的要求符合中国客户的最大利益” 的狡辩说法。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “我们有许多员工都是在压迫性政权下长大的，有些人因为担心签证丢失而无法大声疾呼，我们让他们都失望了”。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “我们每天上班，而所有工作都是在使这个世界变得更糟。为了什么？就为了能够在中国出售几张白色背景上切成薄片的水果的照片。”
   </strong>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong>
     Shutterstock 员工的请愿：
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   我们是 Shutterstock 的签约雇员，我们呼吁该公司拒绝中国政府的要求，即 禁止针对中国网站用户的政治敏感主题的搜索结果。
  </p>
  <p class="graf graf--p">
   虽然合规将使公司受益于在中国的持续运营，但我们认为，任何审查制度都将树立有害的先例，并对我们的公司、中国乃至世界，产生有害影响。
  </p>
  <p class="graf graf--p">
   通过遵守这些规定，我们将使不公正现象成为现实，包括歧视香港人民、镇压中国的政治异议、以及破坏台湾人民的主权。
  </p>
  <p class="graf graf--p">
   建立搜索过滤器的第一步为将来出现更多类型的歧视敞开了大门。
  </p>
  <p class="graf graf--p">
   我们为 Shutterstock 在诸如网络中立、移民政策、和反犹主义等重要话题上保持立场的历史感到自豪。而现在，通过满足中国政府的要求，我们将传达这样一个信息，即：我们对价值观和底线的承诺只是第二要务。那不是我们。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    我们反对任何内容过滤审查的原则不仅是面向中国，我们从根本上反对技术帮助强国压迫脆弱人群，无论他们身在何处
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Shutterstock 自己的员工来自世界各地 —— 许多人亲身经历过政府的压迫 —— 在接受这个项目时，我们让他们失望了！
  </p>
  <p class="graf graf--p">
   因此，我们要求 Shutterstock 终止该项目。 Shutterstock 有能力成为变革的领导者。
  </p>
  <p class="graf graf--p">
   作为员工和股东，我们应该知道我们正在建设什么，我们应该在这些重大决策中拥有发言权。”
  </p>
