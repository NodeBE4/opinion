   </span>
   <span class="s1">
    、通过对活动家中的积极分子进行调查，然后向公众发布其负面的个人信息，为目标群体创建负面的公众形象
   </span>
   <span class="s3">
    ——
   </span>
   <span class="s1">
    也就是
    <b>
     《
     <a href="https://www.iyouport.org/%e4%ba%ba%e6%a0%bc%e6%9a%97%e6%9d%80%e5%92%8c%e6%8a%b9%e9%bb%91%e8%bf%90%e5%8a%a8%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%8f%97%e7%9b%8a%ef%bc%9f/%20" rel="noopener noreferrer" target="_blank">
      人格暗杀
     </a>
     》
    </b>
    。
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    2
   </span>
   <span class="s1">
    、通过让特工人员激化种族矛盾，或发送匿名信制造民间组织的内部矛盾，从而瓦解反抗组织的内部
   </span>
   <span class="s3">
    ——
   </span>
   <span class="s1">
    也就是
    <b>
     挑拨离间、制造混乱和不信任
    </b>
    。《
    <a href="https://www.iyouport.org/%e4%bb%96%e4%bb%ac%e4%b8%ba%e4%bb%80%e4%b9%88%e5%a6%82%e6%ad%a4%e7%8c%96%e7%8b%82%ef%bc%9f%e5%9b%a0%e4%b8%ba%e4%bd%a0%e4%bb%ac%e4%ba%92%e4%b8%8d%e4%bf%a1%e4%bb%bb/" rel="noopener noreferrer" target="_blank">
     他们为什么如何猖狂，因为你们互不信任
    </a>
    》
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    3
   </span>
   <span class="s1">
    、通过散布其他团体偷钱的谣言，在团体之间制造分歧
   </span>
   <span class="s3">
    ——
   </span>
   <span class="s1">
    这是为了
    <b>
     拆散联盟的可能性
    </b>
    。当民间组织之间能够联合起来的时候，才是当权者最恐惧的时候。
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    4
   </span>
   <span class="s1">
    、通过向非营利组织施压，限制其获得公共资源的机会，切断对非营利组织的资金或物质支持
   </span>
   <span class="s3">
    ——
   </span>
   <span class="s1">
    简称，
    <b>
     饿死异议
    </b>
    。《
    <a href="https://www.iyouport.org/%e5%aa%92%e4%bd%93%e5%b7%a8%e5%a4%b4%e5%92%8c%e7%a1%85%e8%b0%b7%e7%9a%84%e6%ad%a6%e5%99%a8%e5%8c%96%e7%ae%97%e6%b3%95%e6%b2%89%e9%bb%98%e5%bc%82%e8%ae%ae/" rel="noopener noreferrer" target="_blank">
     媒体巨头和硅谷的武器化算法沉默异议
    </a>
    》
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    5
   </span>
   <span class="s1">
    、让渗透进入的特工在策划抗议和抗议活动期间
    <b>
     宣传针对警察的暴力，以此来限制组织抗议的能力
    </b>
    。
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    6
   </span>
   <span class="s1">
    、通过人格暗杀、伪造的逮捕（也就是没有法律依据的的逮捕监禁，以恐吓为目的）、渗透监视等方式，
    <b>
     限制个人参加集体活动的能力
    </b>
    。
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    COINTELPRO
   </span>
   <span class="s1">
    的文件中披露了许多
   </span>
   <span class="s3">
    FBI
   </span>
   <span class="s1">
    意图阻止反越战的群众抗议的案例。
   </span>
  </p>
  <p class="p3">
   <span class="s1">
    为了完成
   </span>
   <span class="s3">
    “
   </span>
   <span class="s1">
    任务
   </span>
   <span class="s3">
    ”
   </span>
   <span class="s1">
    ，使用了许多
    <b>
     肮脏的手段
    </b>
    。
   </span>
  </p>
  <p class="p1">
   <span class="s5">
    这些
    <a href="https://www.amazon.com/Cointelpro-FBIs-Secret-Political-Freedom/dp/0873488776" rel="noopener noreferrer" target="_blank">
     文件
    </a>
   </span>
   <span class="s2">
    <a href="https://www.amazon.com/Cointelpro-FBIs-Secret-Political-Freedom/dp/0873488776" rel="noopener noreferrer" target="_blank">
     指出
    </a>
    ：
   </span>
   <span class="s1">
    <b>
     “
    </b>
   </span>
   <span class="s2">
    <b>
     这些手段包括促进反战势力之间的分裂让他们无法联合，鼓励对活动家进行迫害，并由此来推动暴力对抗，以取代大规模的和平示威。
    </b>
   </span>
   <span class="s1">
    <b>
     ”
    </b>
   </span>
  </p>
  <p class="p3">
   <span class="s1">
    渗透收集情报、故意破坏和瓦解社会运动的行为，一直都是很常见的。
   </span>
  </p>
  <p class="p3">
   <span class="s1">
    在历史上，当贫富不均的程度已经达到如此惊人时、最富有的
   </span>
   <span class="s3">
    400
   </span>
   <span class="s1">
    人所拥有的财富和最底层的
   </span>
   <span class="s3">
    15.4
   </span>
   <span class="s1">
    万人的财富相同时、当失业率和法拍屋率居高不下时，当几千万人买不起医保、学生上不起大学时，当权势担心人民会起义时
   </span>
   <span class="s3">
    ……
   </span>
   <span class="s1">
    这一切间谍行动就会发生。
   </span>
  </p>
  <p class="p3">
   <span class="s1">
    近年来发生的所有事件，特别是
   </span>
   <span class="s3">
    “
   </span>
   <span class="s1">
    占领
   </span>
   <span class="s3">
    “
   </span>
   <span class="s1">
    事件，显示出这场起义已经开始了。当权者会利用现有的一切工具来阻止
   </span>
   <span class="s3">
    “
   </span>
   <span class="s1">
    占领
   </span>
   <span class="s3">
    “
   </span>
   <span class="s1">
    ，包括渗透破坏、分裂和误导。
   </span>
  </p>
  <p class="p3">
   <span class="s1">
    本系列下一部分的内容将描述渗透者的常见行为，以及其他社会运动如何试图将渗透的影响降到最低。
   </span>
  </p>
  <p class="p3">
   <span class="s1">
    然后将研究
   </span>
   <span class="s3">
    “
   </span>
   <span class="s1">
    占领
   </span>
   <span class="s3">
    “
   </span>
   <span class="s1">
    的基本结构，并分析其在渗透的背景下的优缺点。
   </span>
  </p>
  <p class="p3">
   <span class="s1">
    希望这一系列的研究能够在社会运动内部引起更广泛的讨论，以便努力探讨如何保护公民反抗运动不受干扰和分裂。
   </span>
  </p>
  <p class="p1">
   <em>
    <span class="s1">
     ——
    </span>
    <span class="s2">
     未完待续
    </span>
    <span class="s1">

   </em>
