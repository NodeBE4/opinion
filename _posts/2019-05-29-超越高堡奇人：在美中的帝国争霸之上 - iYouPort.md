   <li>
    <span style="color: #00ccff;">
     <em>
      <strong>
       这是帝国争霸战。人们从一开始就应该意识到这点。
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   我们一直在强调“高堡奇人” —— 美国和中国的关系是帝国争霸，其中没有人民的利益，正相反是以人民的利益为代价。
  </p>
  <p class="graf graf--p">
   IYP 的分类知识列表中将“高堡奇人”标记为*最重要的地缘政治* —— 因为它是本世纪中心级的帝国争霸战。
  </p>
  <p class="graf graf--p">
   帝国是一种不应该存在的东西。如果一直处于高堡奇人的世界，不论支持任何一方都是99%的损失。唯一的胜利者是那1%。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4481 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国2.jpg?resize=1100%2C736&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国2.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国2.jpg?resize=300%2C201&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国2.jpg?resize=768%2C514&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国2.jpg?resize=1024%2C685&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国2.jpg?resize=1100%2C736&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国2.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="736" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国2.jpg?resize=1100%2C736&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4481" data-recalc-dims="1" height="736" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国2.jpg?resize=1100%2C736&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国2.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国2.jpg?resize=300%2C201&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国2.jpg?resize=768%2C514&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国2.jpg?resize=1024%2C685&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国2.jpg?resize=1100%2C736&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国2.jpg?resize=272%2C182&amp;ssl=1 272w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   中美贸易谈判的崩溃给全球资本主义体系带来了冲击波。
  </p>
  <p class="graf graf--p">
   此前几乎每个人都预计会达成一项协议，包括特朗普自己，他曾预测这将是“史诗般的”。
  </p>
  <p class="graf graf--p">
   当这一切都失败后，他将失败归咎于中国，并将低强度的贸易战升级为全面爆发。
  </p>
  <p class="graf graf--p">
   随着挑战的逐渐减弱，北京宣布对价值600亿美元美国商品的关税税率从10％提高到25％。国家电视台的一
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2019/05/14/business/us-china-tariffs.html" href="https://www.nytimes.com/2019/05/14/business/us-china-tariffs.html" rel="noopener noreferrer" target="_blank">
    位发言人警告说
   </a>
   ：“如果美国想谈谈，我们的大门是敞开的。如果美国想要战斗，我们将与他们一起战斗到最后。”
  </p>
  <p class="graf graf--p">
   贸易战的威胁吓坏了全世界的投资者。
  </p>
  <p class="graf graf--p">
   在世界各地的
   <a class="markup--anchor markup--p-anchor" data-href="http://time.com/5588518/trade-war-china-stocks-drop/?xid=tcoshare" href="https://time.com/5588518/trade-war-china-stocks-drop/?xid=tcoshare" rel="noopener noreferrer" target="_blank">
    股票市场大规模抛售
   </a>
   中，他们从账面上抹去了一万亿美元。
  </p>
  <p class="graf graf--p">
   像
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2019/05/13/us/politics/us-china-trade-tariffs.html" href="https://www.nytimes.com/2019/05/13/us/politics/us-china-trade-tariffs.html" rel="noopener noreferrer" target="_blank">
    苹果和波音
   </a>
   这样严重依赖中国并且首次受到新关税影响的美国跨国公司遭受了特别严厉的打击。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这两个帝国
   </strong>
   本月晚些时候可能在北京举行会谈，或者6月底在日本大阪举行的G20峰会上达成什么协议。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    但是，无论他们是否达成任何协议，两国之间日益加剧的冲突都不会减弱；这是21世纪帝国争霸战的中心
   </strong>
   。
  </p>
  <p class="graf graf--p">
   特朗普提出的是中国永远不会满足的要求。特朗普的定义是对的，中国是一个国家资本主义大国，目标是推动其作为帝国的崛起，并挑战美国的全球霸权。
  </p>
  <p class="graf graf--p">
   中国认为美国的要求是源于华盛顿企图捍卫其全球统治地位、侵犯中国的国家主权，并阻止其重新成为世界强国。
  </p>
  <p class="graf graf--p">
   这种帝国对抗正是美国自冷战结束以来所寻求避免的。
  </p>
  <p class="graf graf--p">
   它采取了一种强有力的新自由主义世界秩序战略，将其他国家纳入其中，承诺实现“双赢”全球化，在此过程中选择任何潜在的同行竞争对手，并通过制裁对所谓的无赖国家实施政权更迭。
  </p>
  <p class="graf graf--p">
   作为这个项目的一部分，美国采取了一项针对中国的政策，将参与和遏制结合到许多人所称的“遏制- 接触战略 congagement”中。
  </p>
  <p class="graf graf--p">
   它提供给北京以胡萝卜，就如加入世界贸易组织，条件是接受新自由主义规则。但也同时通过整个亚太地区的海军巡逻和军事基地挥舞着军事力量。
  </p>
  <p class="graf graf--p">
   华盛顿与北京达成协议，允许美国跨国公司将生产分包给中国的企业，这些企业利用该国廉价劳动力生产苹果 iPhone 等产品。
  </p>
  <p class="graf graf--p">
   在此过程中，北京从经济疲软转变为世界第二大经济体。
  </p>
  <p class="graf graf--p">
   在中国上升的同时，美国相对下降。由于将无数人的生命和资金投入到阿富汗和伊拉克的徒劳战争中，经济受到大萧条的破坏，并陷入政治两极分化和华盛顿的瘫痪之中。
  </p>
  <p class="graf graf--p">
   当然，美国仍然是世界上唯一最大经济体，拥有最强大的军事力量，并因此依旧是具有最大地缘政治影响力的超级大国。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4482 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国.jpg?resize=1100%2C748&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国.jpg?w=2048&amp;ssl=1 2048w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国.jpg?resize=300%2C204&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国.jpg?resize=768%2C522&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国.jpg?resize=1024%2C696&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国.jpg?resize=1100%2C748&amp;ssl=1 1100w" data-recalc-dims="1" height="748" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国.jpg?resize=1100%2C748&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4482" data-recalc-dims="1" height="748" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国.jpg?resize=1100%2C748&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国.jpg?w=2048&amp;ssl=1 2048w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国.jpg?resize=300%2C204&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国.jpg?resize=768%2C522&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国.jpg?resize=1024%2C696&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国.jpg?resize=1100%2C748&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    但华盛顿的相对衰落为中国打开了通向新帝国的大门。
   </strong>
  </p>
  <p class="graf graf--p">
   习近平就是这样做的：他放弃了他的前任“和平崛起”的大战略，并采用了他称之为“中华民族复兴大梦”的新战略，旨在迫使世界其他国家承认北京是一个强大的力量。
  </p>
  <p class="graf graf--p">
   中国推出了价值1万亿美元的“一带一路”项目，旨在发展交通基础设施，将中国与欧洲、亚洲和非洲经济融为一体。
  </p>
  <p class="graf graf--p">
   他还推出了“中国制造2025”项目，该项目将国家资金投入到高科技公司，使其成为能够与美国和欧洲竞争对手竞争的冠军。
  </p>
  <p class="graf graf--p">
   中国对其军队进行了现代化改造，特别是其海军，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.businessinsider.com/china-is-building-a-powerful-navy-to-take-on-the-us-in-the-pacific-2019-4" href="https://www.businessinsider.com/china-is-building-a-powerful-navy-to-take-on-the-us-in-the-pacific-2019-4" rel="noopener noreferrer" target="_blank">
    专门用于对抗美国
   </a>
   。它在整个南海和东海的岛屿上建立了军事基地，以控制以前受美国监管的战略航道，声称其拥有渔业、海底的石油和天然气。
  </p>
  <p class="graf graf--p">
   中国还在非洲之角的吉布提建立了第一个海外军事基地，毗邻通往苏伊士运河的主要航道。
  </p>
  <p class="graf graf--p">
   五角大楼警告说，这只是一个开始；它声称，中国将通过“一带一路”建设项目建设一个完整的基地网络，以与美国竞争。
   <strong class="markup--strong markup--p-strong">
    迄今为止，美国几乎
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.thenation.com/article/the-us-has-military-bases-in-172-countries-all-of-them-must-close/" href="https://www.thenation.com/article/the-us-has-military-bases-in-172-countries-all-of-them-must-close/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     垄断了所有这些军事前哨
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   因此，华盛顿的“遏制- 接触战略”政策适得其反，无意中帮助了中国成为同行竞争对手。
  </p>
  <p class="graf graf--p">
   特朗普夺取总统职位的承诺是，他可以“让美国再次伟大”，并以新的经济民族主义战略打败中国。
  </p>
  <p class="graf graf--p">
   特朗普的“put America first” 绕过多边机构，以交易方式对待盟友，甚至对加拿大和欧洲征收关税。
  </p>
  <p class="graf graf--p">
   在他的国家安全战略中，特朗普淡化了所谓的反恐战争，专注于对抗强大的竞争对手，特别是中国和俄罗斯。
  </p>
  <p class="graf graf--p">
   特朗普国防部警告说，美国军工集团对中国供应链的依赖是对国家安全的威胁。
  </p>
  <p class="graf graf--p">
   为了解决这个问题，国防部主张联邦政府制定一项产业政策来重建其国内制造业基地，并规定任何离岸军事生产都要从中国转移到盟国。
  </p>
  <p class="graf graf--p">
   特朗普还特别指出中国的高科技产业是对国家安全的威胁，特别是华为，后者正在全球领导5G网络的建设。
  </p>
  <p class="graf graf--p">
   将注意力集中在华为身上是错误的，
   <strong class="markup--strong markup--p-strong">
    华为只是帝国争霸的道具之一
   </strong>
   ，并无关“安全”问题 —— 如果你读过斯诺登文件并拥有基本价值观的话就能明白：所有后门都危及安全，而中美争的并不是*有没有后门*，而是*谁来掌握后门*。
  </p>
  <p class="graf graf--p">
   特朗普的经济民族主义给美国资本带来了两难境地。其跨国部门仍然致力于美国国家监督自由贸易全球化新自由主义秩序的旧战略。
  </p>
  <p class="graf graf--p">
   但他们对中国的国家资本主义感到沮丧，特别是其盗窃的知识产权估计每年使美国公司损失高达6,000亿美元。
  </p>
  <p class="graf graf--p">
   这就是为什么美国商会调查的大多数公司支持特朗普的关税政策，迫使中国加入新的贸易协议。
  </p>
  <p class="graf graf--p">
   帝国官僚机构中的政治阶层、国家管理者、以及资本主义政党也在特朗普后面排起了长队。共和党人，以前是自由贸易的热心倡导者，现在已经陷入困境。
  </p>
  <p class="graf graf--p">
   民主党从其企业领导层到其持续不断的持不同政见者，要么保持沉默 —— 因为害怕失去被特朗普保护主义诱惑的选民，要么放弃了权利 —— 要求他在谈判中采取更加强硬的立场。
  </p>
  <p class="graf graf--p">
   新自由主义前副总统乔拜登除外。他淡化了中国构成的威胁，向人们保证美国至高无上的地位，并挑战特朗普的对抗态度，特别是他的关税。他夸口说：“中国要吃我们的午餐？来吧，伙计……他们不是坏人，他们不是我们的竞争对手。“
  </p>
  <p class="graf graf--p">
   关联：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%B9%94%E6%B2%BB%E6%8B%9C%E7%99%BB%E7%9A%84%E5%84%BF%E5%AD%90%E6%8A%95%E8%B5%84%E4%B8%AD%E5%9B%BD%E8%80%81%E5%A4%A7%E5%93%A5%E7%9B%91%E8%A7%86%E6%8A%80%E6%9C%AF/" href="https://www.iyouport.org/%E4%B9%94%E6%B2%BB%E6%8B%9C%E7%99%BB%E7%9A%84%E5%84%BF%E5%AD%90%E6%8A%95%E8%B5%84%E4%B8%AD%E5%9B%BD%E8%80%81%E5%A4%A7%E5%93%A5%E7%9B%91%E8%A7%86%E6%8A%80%E6%9C%AF/" rel="noopener noreferrer" target="_blank">
    乔治拜登的儿子投资中国’老大哥’监视技术
   </a>
   》
  </p>
  <p class="graf graf--p">
   然而，大多数民主党人采取了自由民族主义立场。参议院少数党领袖 Chuck Schumer
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnbc.com/2019/05/06/schumer-urges-trump-to-hang-tough-after-china-trade-tariff-threat.html" href="https://www.cnbc.com/2019/05/06/schumer-urges-trump-to-hang-tough-after-china-trade-tariff-threat.html" rel="noopener noreferrer" target="_blank">
    在推特上写道
   </a>
   ：“总统先生，要对中国坚持不懈。不要退缩。实力是赢得中国的唯一途径。”
  </p>
  <p class="graf graf--p">
   作为亲资本主义进步人士的参议员伊丽莎白沃伦宣称，“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.motherjones.com/politics/2018/11/elizabeth-warren-presidential-campaign-2020-foreign-policy-nafta-free-trade-trump/" href="https://www.motherjones.com/politics/2018/11/elizabeth-warren-presidential-campaign-2020-foreign-policy-nafta-free-trade-trump/" rel="noopener noreferrer" target="_blank">
    中国武器化其经济
   </a>
   ”，同时她承诺通过保护主义立法和重新谈判的贸易协议挽救美国的就业机会。
  </p>
  <p class="graf graf--p">
   自称为社会主义支持者的参议员伯尼·桑德斯（Bernie Sanders）当然不会被排除在外，他因为对中国的软弱而抨击拜登，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnbc.com/2019/05/02/sanders-slams-bidens-china-trade-stance-in-2020-democratic-primary.html" href="https://www.cnbc.com/2019/05/02/sanders-slams-bidens-china-trade-stance-in-2020-democratic-primary.html" rel="noopener noreferrer" target="_blank">
    并承诺
   </a>
   他将更好地领导美国与中国的斗争。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4483 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国3.jpg?resize=1100%2C803&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国3.jpg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国3.jpg?resize=300%2C219&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国3.jpg?resize=768%2C560&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国3.jpg?resize=1024%2C747&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国3.jpg?resize=1100%2C803&amp;ssl=1 1100w" data-recalc-dims="1" height="803" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国3.jpg?resize=1100%2C803&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4483" data-recalc-dims="1" height="803" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国3.jpg?resize=1100%2C803&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国3.jpg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国3.jpg?resize=300%2C219&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国3.jpg?resize=768%2C560&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国3.jpg?resize=1024%2C747&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/帝国3.jpg?resize=1100%2C803&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   像特朗普内阁中的彼得·纳瓦罗（Peter Navarro）和像史蒂夫·班农（Steve Bannon）这样的强硬民族主义者正在推动美国资本与中国脱钩。
  </p>
  <p class="graf graf--p">
   &lt;顺便透露一个内幕消息，几个月前，中国国社内部组织“学习”史蒂夫·班农的讲话 —— 请注意真的是史蒂夫·班农，他的演讲全文中文版，在国社内部会议中被全文朗读。
   <strong>
    没错，目标是民族主义，而非其他
   </strong>
   &gt;
  </p>
  <p class="graf graf--p">
   像白宫经济顾问拉里·库德洛（Larry Kudlow）这样的其他人有意通过强制达成协议来开放中国以进一步投资，但很难相信北京会同意这些条款。
  </p>
  <p class="graf graf--p">
   与此同时，脱钩开始发生。跨国公司担心贸易战，也对中国的政策感到沮丧，不愿意支付中国劳动力成本的上涨，开始将生产转移到像越南这样的低工资国家，以及像墨西哥这样的靠近家乡的国家。这是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2019/04/05/business/china-trade-trump-jobs-decoupling.html" href="https://www.nytimes.com/2019/04/05/business/china-trade-trump-jobs-decoupling.html" rel="noopener noreferrer" target="_blank">
    纽约时报说的
   </a>
   。
  </p>
  <p class="graf graf--p">
   中国也可能开始重新调整经济，避免依赖美国而独立发展自身能力和与其他国家的关系。许多中国公司都在谈论“中美共同体”的终结。
  </p>
  <p class="graf graf--p">
   中国民族主义者，尤其是军队中的民族主义者，渴望推动这种分离。 “经济学人”指出，“习近平看到了对中国气概的考验”。去年9月习宣称，“保护主义使得从国外获取重要技术变得困难。中国必须走’自力更生的道路’。“
  </p>
  <p class="graf graf--p">
   这可能会在国家资助的资本主义发展道路上翻倍，就像中国制造2025一样，它已经引发了美国国家和华盛顿的骚动。它还将尝试通过BRI寻找替代投资场所、其他所需商品的来源、以及其产品的其他市场，尤其是欧盟。
  </p>
  <p class="graf graf--p">
   特朗普只是对伊朗实施制裁，禁止购买石油，中国是该国原油出口的最大购买者。作为回应，北京要求特朗普收回政权更迭的威胁，参与多边会谈以维护核协议，并放弃对石油的制裁。
  </p>
  <p class="graf graf--p">
   与此同时，美国和中国经济的相互依存、贸易或地缘政治冲突不断升级的威胁可能会对世界经济造成损害，可能会促使特朗普和习近平退缩。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    他们完全有理由担心这一点。尽管两位领导人都吹嘘自己的经济，但两者都很脆弱。
   </strong>
  </p>
  <p class="graf graf--p">
   中国的经济增长继续放缓，其加速的努力已经在整个体系中累积了大量的债务和大规模的产能过剩；无论特朗普的助推主义如何，美国经济都面临着投资不足、生产力低下、公共和私人债务庞大、以及劳动力参与率过低等潜在问题。
  </p>
  <p class="graf graf--p">
   两国都不希望贸易战和地缘政治冲突使他们陷入衰退。
  </p>
  <p class="graf graf--p">
   与此同时，美国和中国都无法被视为在谈判桌上屈服的人，尤其是双方民族主义阵营一直不断为更加激烈的对抗而努力。
  </p>
  <p class="graf graf--p">
   这种压力只会在经济放缓或更大的地缘政治对抗的情况下增长，特朗普和习近平
   <strong class="markup--strong markup--p-strong">
    作为极端民族主义者
   </strong>
   将毫不犹豫地将他们的国家团结在他们身后。
  </p>
  <p class="graf graf--p">
   然而，即使在这种情况下，他们也可能会试图避免任何争议失控导致冒着军事对抗的风险，比如核战争的风险，双方都希望避免这种对抗。
  </p>
  <p class="graf graf--p">
   因此，美中的竞争可能会继续以地缘政治和地缘经济竞争的方式表达。
  </p>
  <p class="graf graf--p">
   新的社会主义运动必须对美国和中国采取明确而有原则的反帝国主义立场。
  </p>
  <p class="graf graf--p">
   必须反对特朗普的民族主义和保护主义；也应该反对中国国家。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    北京声称自己是社会主义者是可笑的；中国只是一个利用其工人阶级的资本主义独裁政权。
   </strong>
  </p>
  <p class="graf graf--p">
   中国与美国竞争的是帝国主义的势力，争夺世界市场的主导地位。
  </p>
  <p class="graf graf--p">
   新社会主义运动不应在中国和美国这两个帝国之间徘徊，而是应建立在高科技工作者的先例之上，他们已经组织了对谷歌的国际罢工运动，并组织了对中国
   <a class="markup--anchor markup--p-anchor" data-href="https://www.jacobinmag.com/2019/05/tech-workers-chinese-solidarity-microsoft-github" href="https://www.jacobinmag.com/2019/05/tech-workers-chinese-solidarity-microsoft-github" rel="noopener noreferrer" target="_blank">
    反996工人维权
   </a>
   的支持。
  </p>
  <p class="graf graf--p">
   在全球化的资本主义经济中，这种工人阶级的团结是结束帝国间争霸并以国际社会主义取而代之的唯一途径。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://truthout.org/articles/deal-or-no-deal-the-rivalry-between-the-us-and-china-will-intensify/" href="https://truthout.org/articles/deal-or-no-deal-the-rivalry-between-the-us-and-china-will-intensify/" rel="noopener noreferrer" target="_blank">
    The new socialist movement must adopt a clear and principled anti-imperialist position against both the U.S. and China
   </a>
   .
  </p>
