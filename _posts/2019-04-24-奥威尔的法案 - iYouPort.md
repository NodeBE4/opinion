   奥威尔的法案
  </h1>

 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <strong>
      <em>
       想买枪吗？必须首先交出你的社交媒体密码，包括 Twitter，Facebook，Snapchat 和 Instagram 等帐户；还必须允许警方在 Google，Yahoo 和 Bing 上查看一年的搜索历史… 这是怎么回事？
      </em>
     </strong>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   去年年底，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theorganicprepper.com/this-anti-gun-bill-would-require-the-social-media-history-and-internet-search-history-of-would-be-buyers/" href="https://www.theorganicprepper.com/this-anti-gun-bill-would-require-the-social-media-history-and-internet-search-history-of-would-be-buyers/" rel="noopener noreferrer" target="_blank">
    奥威尔枪支法案
   </a>
   在纽约州最先出台。如果签署成为法律，任何想要购买枪支的人都必须交出他们的社交媒体历史记录和他们的互联网搜索历史记录至少一年。
  </p>
  <p class="graf graf--p">
   布鲁克林区负责人 Eric Adams 说：“要对社交媒体资料进行为期三年的审查，可以很容易地了解一个不适合持有和拥有枪支的人。”他曾与纽约州参议员 Kevin Parker 提出此立法。
  </p>
  <p class="graf graf--p">
   在购买枪支之前，
   <strong class="markup--strong markup--p-strong">
    申请人必须交出他们的社交媒体密码包括 Twitter，Facebook，Snapchat 和 Instagram 等帐户；他们还必须允许警方在 Google，Yahoo 和 Bing 上查看一年的搜索历史
   </strong>
   。
  </p>
  <p class="graf graf--p">
   该法律还要求任何续签手枪许可证的人也要受到这种侵犯隐私的搜查。
  </p>
  <p class="graf graf--p">
   Daisy Luther 写道：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    请记住，这些事永远不会停留在一个州！
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    看看在中期选举中最近从红色变为蓝色的所有州。如果你认为它不会发生在你所处的位置，你就彻底错了。请记住，我既不是民主党人也不是共和党人
   </em>
   。
  </p>
  <p class="graf graf--p">
   事实证明，Daisy 的预测是正确的。现在，在伊利诺伊州，想要拥有自己的枪，你也必须拥有一张特殊的卡，获得该卡的要求事实上是更具侵入性的：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    与此同时，在伊利诺伊州众议院，州议员 Daniel Didech，D-Buffalo Grove 已提交了 HB 888，它要求那些申请国家颁发的枪支持有证的任何人交出他们的社交媒体账户，这是对合法枪支所有者的强制要求。州警察​​将使用这些信息来确定社交媒体帐户是否有任何“信息会使此人无法获得或必须撤销”FOID卡
   </em>
   。
  </p>
  <p class="graf graf--p">
   该 FOID 卡还需要您的照片、身高、体重、地址、生日、头发颜色和眼睛颜色 — — 这是政府颁发的身份证才需要的基本信息。
  </p>
  <p class="graf graf--p">
   要想获得此卡，你必须回答一份调查问卷，询问你是否曾经被判犯有重罪，是否沉迷于麻醉品，是否曾在精神病院接受过治疗，或者是“智力障碍”。其他问题询问具体的罪行，你是不是非法的外国人，是否……
  </p>
  <p class="graf graf--p">
   除了新法案的明显问题之外，还有一些值得关注的问题：社交媒体账户上发现的信息究竟会被用来“取消”人们取得卡的资格，还是导致 FOID 卡的撤销？
  </p>
  <p class="graf graf--p">
   这个看似重要的细节没有在任何地方被明确出来（可以
   <a class="markup--anchor markup--p-anchor" data-href="http://ilga.gov/legislation/fulltext.asp?DocName=&amp;SessionId=108&amp;GA=101&amp;DocTypeId=HB&amp;DocNum=888&amp;GAID=15&amp;LegID=115206&amp;SpecSess=&amp;Session=" href="http://ilga.gov/legislation/fulltext.asp?DocName=&amp;SessionId=108&amp;GA=101&amp;DocTypeId=HB&amp;DocNum=888&amp;GAID=15&amp;LegID=115206&amp;SpecSess=&amp;Session=" rel="noopener noreferrer" target="_blank">
    在这里阅读
   </a>
   ）。
  </p>
  <p class="graf graf--p">
   并且，其判断是否仅仅取决于被指派来评估社交媒体账户的个别警察的想法和感受？
  </p>
  <p class="graf graf--p">
   到底有没有具体、客观的指导方针？
  </p>
  <p class="graf graf--p">
   一些公民社会团体已经明确表示反对该法案，包括枪支权利团体和ACLU。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    当人们看到这个法案时，每个拥有 Facebook 帐户或电子邮件帐户或推特帐户的人都会被激怒或应该被激怒。ACLU 伊利诺伊州的 Rebecca Glenberg 表示，该法案“没有说明这些信息的收集将如何保留，以及可能会使用多长时间、用途是什么。”；“一个人的政治信仰，一个人的宗教信仰，不应该参与此人获取 FOID 卡有任何相关性。
   </em>
   ”
  </p>
  <p class="graf graf--p">
   就在几天前，我们报道了一项新的研究，该研究发现，那些已经停用所有社交媒体账户的人也并不能保证隐私，因为所有认识你的人都会在完全无意识的情况下出卖你 — — 通过他们就可以预测你的行为。如果技术人员说有一天你可能会这样做，你不必真正犯任何罪，就可以直接被捕了。
  </p>
  <p class="graf graf--p">
   控制狂的政客们正在开始要求监视你的朋友、和朋友的朋友 等等，仅仅强调你不能和不注重隐私的人交往也许已经不够了。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <a href="https://www.zerohedge.com/news/2019-02-08/another-state-wants-every-gun-owner-hand-over-their-social-media-accounts-and" rel="noopener noreferrer" target="_blank">
    <span class="graf-dropCap">
     A
    </span>
    nother State Wants Every Gun Owner To Hand Over Their Social Media Accounts And Internet Search History; Unconstitutional gun law ideas seem to spread from one state to another like some kind of insidious virus
   </a>
   .
  </p>
