  <p class="graf graf--p">
   *结论*就是分析人员对事态的根本性判断；而且，⚠️一旦得出结论，将高度影响分析人员对材料的理解。这点非常重要。
  </p>
  <p class="graf graf--p">
   认知心理学认为，知觉形成的速度是很快的，而且不容易改变。一旦你对某一事物有了*知觉*，就会形成某种看法，也就是某种认知；即便此后知觉发生了变化，你也会继续以原有的认知去理解它。
  </p>
  <p class="graf graf--p">
   在情报分析过程中，通常都是从迹象十分模糊的初始阶段开始接触情况的；随着接收到更多的补充性信息，情况会逐渐变得明朗，但是，最初的认知会严重影响人们对客观情况的认识。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    出现证伪的信息并不会改变人们最初的看法，许多零散情报中的积累信息也经常不会被作为一个完整的信息来研究。
   </strong>
   这是固有的思维缺陷。
  </p>
  <p class="graf graf--p">
   ⚠️这就是为什么你看到辟谣是无效的；也是为什么“人格暗杀”那种拙劣的造假却屡试不爽。在这里看到详细解读《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%ba%e6%a0%bc%e6%9a%97%e6%9d%80%e5%92%8c%e6%8a%b9%e9%bb%91%e8%bf%90%e5%8a%a8%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%8f%97%e7%9b%8a%ef%bc%9f/" href="https://www.iyouport.org/%e4%ba%ba%e6%a0%bc%e6%9a%97%e6%9d%80%e5%92%8c%e6%8a%b9%e9%bb%91%e8%bf%90%e5%8a%a8%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%8f%97%e7%9b%8a%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    人格暗杀和抹黑运动：谁在受益？
   </a>
   》。
  </p>
  <p class="graf graf--p">
   并且，就如你在互联网上遭遇到的信息消防栓轰炸那样，情报分析人员也会面对同样的事 —— 紧迫的时间和大量信息的压力；在任何新事态出现后，决策者就会马上要求分析师进行解读，而此时的分析人员并没有能完全了解情报资源。
  </p>
  <p class="graf graf--p">
   互联网上被情绪高度驱使的肤浅的认知判断就是这种情况的结果，情报分析人员也经常会采取先入之见。
  </p>
  <p class="graf graf--p">
   认知心理学认为，即然最初的判断很难改变，不如强调对一件事的认知没有完成。新信息正在不断涌现，在这种情况下分析人员
   <strong class="markup--strong markup--p-strong">
    很有必要明确区分*提出观点阶段*和*评估阶段*，尽可能减缓做出结论
   </strong>
   。
  </p>
  <p class="graf graf--p">
   并且，应该定期从头开始对关键问题进行彻底的分析，以求避免增量分析中存在的问题。
  </p>
  <p class="graf graf--p">
   💡这些建议对我们所有通过互联网了解局势并期待改变现实的人来说，同样是非常必要的。
  </p>
  <p class="graf graf--p">
   这是抵制欺骗的关键。
  </p>
  <h4 class="graf graf--h4">
   <strong class="markup--strong markup--h4-strong">
    3、证伪假设
   </strong>
  </h4>
  <p class="graf graf--p">
   如你所知，最常被使用的分析方法是*证实方法* —— 当出现一个新的情况时，分析人员就要动用各种手段对其进行核实。就像你正在网上判断假新闻时所做的那样。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    ⚠️但是，这样做有严重缺陷，尤其是很容易遭到战略性欺骗攻击
   </strong>
   —— 就如精心策划的任何假消息一样，它们都带有你天然觉得正确无误的特点，这样你才能上当。
  </p>
  <p class="graf graf--p">
   如果所有信息收集手段都指向同一个目标，那么分析者会理所当然地认为“新情况得到了证实”，结论由此而生。
  </p>
  <p class="graf graf--p">
   而那些较少被佐证的材料则被视为“假消息”或不想干的消息，从而错过了识别欺骗战略的大好时机。
  </p>
  <p class="graf graf--p">
   证实的基础是归纳法。
  </p>
  <p class="graf graf--p">
   经验主义哲学家认为，科学是“经验的科学”，科学知识来自对经验事实的归纳；因此，归纳法是*发现*的方法。
  </p>
  <p class="graf graf--p">
   然而，
   <strong class="markup--strong markup--p-strong">
    这种归纳方法在逻辑上是有严重缺陷的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   大卫休谟证明了归纳法永远不能被证明；卡尔波普尔认为：从逻辑观点上看，无论从多少单称陈述中都不能推论出全称的陈述，因为用这种方法得出的结论总是有可能错误的 —— 不论我们看到多少只白天鹅，都不能证明天下所有天鹅都是白的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在对手的战略性欺骗广泛存在的情报分析中，归纳法的缺陷非常明显
   </strong>
   。
  </p>
  <p class="graf graf--p">
   对手方的情报系统往往会大规模释放虚假信号，从而将其真实的意图掩盖起来。
  </p>
  <p class="graf graf--p">
   为了使假情报看起来确有其事，对手会制造很多周详的欺骗计划，利用各种手段，向被骗者提高假消息。
   <strong class="markup--strong markup--p-strong">
    政治宣传也是这样对大众下手的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️这种情况下，如果你一味地利用佐证法去衡量信息的真伪，就恰好落入了对手的圈套。
  </p>
  <p class="graf graf--p">
   认知心理学构建的“科学发现”逻辑倚重证伪法。波普尔认为，知识增长模型以“试错法”为基本方法，通过各种尝试，排除错误的可能性，逐渐接近正确。
  </p>
  <p class="graf graf--p">
   💡一个全称陈述不能被证实，但是可以被证伪。因此，提出假设，然后证伪，再提出假设，再加以证伪 …… 这样循环下去就可以一步步接近真相。
  </p>
  <p class="graf graf--p">
   这一方法符合科学研究的逻辑。
  </p>
  <p class="graf graf--p">
   逻辑实证主义认为，科学始于中立的观察，观察者不应该带有成见，“观察 — 假设 — 验证” 构成了科学发现的逻辑。
  </p>
  <p class="graf graf--p">
   但是，英国科学哲学家汉森的研究表明，
   <strong class="markup--strong markup--p-strong">
    中立的观察语言是不存在的，因为观察本身就渗透着理论；观察是受理论偏见支配的，人们在不同的理论偏见的引导下会从相同的的图片中看出完全不同的东西
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这就是所谓的“格式塔转换”。
  </p>
  <p class="graf graf--p">
   例如，就日落这一天文现象而言，相信地心说的人看到的是太阳在绕地球转动；而相信日心说的人则会看到完全相反的结果。
  </p>
  <p class="graf graf--p">
   无疑，观察总是渗透着理论的，人的认识总会带有局限性，总伴随着错误、偏见。就如波普尔所言：
   <strong class="markup--strong markup--p-strong">
    理论至少是一些基本的理论或期望，总是首先出现的，它们总是先于观察。
   </strong>
  </p>
  <p class="graf graf--p">
   于是作为分析人员和追求真相的任何人，都应该被建议不断提出假设，不断地进行证伪，直到得出符合实际的结论。
  </p>
  <p class="graf graf--p">
   正是这种与我们大多数人所习惯的传统思维相反的方法，才能真的寻求到真相。
  </p>

  <h4 class="graf graf--h4">
   <strong class="markup--strong markup--h4-strong">
    —— 提高你的反欺骗能力 ——
   </strong>
  </h4>
  <p class="graf graf--p">
   情报界非常注重分析人员的能力，没有高超的能力就没有高质量的情报。
  </p>
  <p class="graf graf--p">
   二战中美国和英国的战略情报分析之所以如此成功，正是得益于高素质的分析团队。
  </p>
  <p class="graf graf--p">
   大量的专家学者加入了情报机构，成为了分析的主力。他们掌握的科学研究方法和专业素养使分析工作的质量得到了保障。
  </p>
  <p class="graf graf--p">
   ⚠️如今在遍地是欺骗 的互联网上，任何追求免于被操纵之基本权利的人们都应该学习一些反欺骗的技巧。
  </p>
  <h4 class="graf graf--h4">
   <strong class="markup--strong markup--h4-strong">
    1、改善思维质量
   </strong>
  </h4>
  <p class="graf graf--p">
   传统的思维模式受实证主义和行为主义方法论的影响，使用基本的逻辑方法如演绎法和归纳法为分析工具，强调对观点的证实。
  </p>
  <p class="graf graf--p">
   但是这种方法完全忽视了情报分析的艺术性；
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    他们只是在强调客观，忽视了信息的模糊性、欺骗性和不完整性。
   </li>
   <li class="graf graf--li">
    他们也不太了解思维过程，不了解思维中的认知偏见；
   </li>
   <li class="graf graf--li">
    他们只是在接受那些与自己的期望一致的资料，而排斥那些与自己的期望不一致的所有东西；
   </li>
   <li class="graf graf--li">
    他们单纯地认为情报空白是搜集能力不足引起的，却没有意识到这可能是对手拒止策略引起的；
   </li>
   <li class="graf graf--li">
    他们认为自己的判断是客观的，却没有意识到在判断过程中充满了各种主观臆断；
   </li>
   <li class="graf graf--li">
    他们没有受过反欺骗方面的训练，面对对手的狡猾缺乏准备。
   </li>
  </ul>
  <p class="graf graf--p">
   为打破传统思维的局限，西方情报界引进了自然科学、哲学和心理学的研究成果，推出了很多主张，其中最重要的就是：批判性思考模式。
  </p>
  <p class="graf graf--p">
   这种思维方式建立在认知心理学基础之上，是进行科学研究必备的思维方式。
  </p>
  <p class="graf graf--p">
   IYP在多年前曾专注于解读过批判性思考的步骤问题，但是在互联网环境下，不仅有信息轰炸和被剥夺注意力的眼球争夺战，还有，回音壁的隧道生态和算法专制操弄的真相危机，于是批判性思考很可能在此被严重抑制了。这就是为什么回归“慢新闻”的支持率近年来一直上升。在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bc%95%e6%b5%81%e6%9a%b4%e6%94%bf%e5%92%8c%e7%94%9f%e5%ad%98%e5%9b%b0%e6%83%91/" href="https://www.iyouport.org/%e5%bc%95%e6%b5%81%e6%9a%b4%e6%94%bf%e5%92%8c%e7%94%9f%e5%ad%98%e5%9b%b0%e6%83%91/" rel="noopener noreferrer" target="_blank">
    引流暴政和生存困惑
   </a>
   》。
  </p>
  <p class="graf graf--p">
   人们的心智结构不能适应直面的变化，往往会因为我们的“安分”而形成一种与单调的常规相结合的习惯。也就是说，
   <strong class="markup--strong markup--p-strong">
    新变旧、复杂变单纯、把一切归结于类似的范式和习惯。
   </strong>
  </p>
  <p class="graf graf--p">
   显然，它经常是错误的、视野狭隘的、和容易被骗的。
  </p>
  <p class="graf graf--p">
   而要改变这种范式和习惯，💡就需要从技能和情感两方面入手来培养批判性思考。
  </p>
  <p class="graf graf--p">
   1990年，美国哲学学会采纳了德尔菲法得出的专家一致意见：
   <strong class="markup--strong markup--p-strong">
    批判性思考包括认知技能和情感意向。
   </strong>
  </p>
  <p class="graf graf--p">
   Richard Paul 指出，
   <strong class="markup--strong markup--p-strong">
    具有批判性思考的人应该具备以下品质
   </strong>
   ，
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    对个人中心主义和社群中心主义有深刻的理解，
   </li>
   <li class="graf graf--li">
    具有谦逊的精神，不会轻易下结论，
   </li>
   <li class="graf graf--li">
    勇于挑战既有标准，坚持不设双标，
   </li>
   <li class="graf graf--li">
    求知欲强，不断探求形形色色的情绪背后存在的观点和林林总总的观点所反映的情绪。
   </li>
  </ul>
  <p class="graf graf--p">
   即
   <strong class="markup--strong markup--p-strong">
    追求真相的精神 truth-seeking，思想开放 open-mindedness，分析性 analyticity，系统性 systematicity, 自信 self-confidence，好奇心 inquisitiveness，成熟的判断力 maturity of judgment
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这是一个优秀的分析人员必须具备的核心技能。
  </p>
  <p class="graf graf--p">
   这样的思维品质对防止被欺骗来说十分重要。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4174 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpg?resize=1100%2C733&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpg?resize=768%2C512&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="733" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/3.jpg?resize=1100%2C733&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    I 批判精神
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️批判精神是一种面对权威不盲从的精神，是一种对真理和真相执着追求的精神。
  </p>
  <p class="graf graf--p">
   基本特征是：认识到通过科学方法获得的知识可能存在错误，任何已有的科学结论都可能被推翻。
  </p>
  <p class="graf graf--p">
   任何科学都必然包含错误，需要被不断检验。这恰恰是科学的优点和力量所在；不断的检验就是科学的进步。
  </p>
  <p class="graf graf--p">
   这对情报分析来说十分重要。因为情报分析都是从假设开始的；在分析过程中，自身的预期、群体的共识、既往经验，都会对假设的提出产生影响。
  </p>
  <p class="graf graf--p">
   因此，
   <strong class="markup--strong markup--p-strong">
    这一假设并不是客观产物，而是思维活动的结果。不幸的是，很多时候这样的假设被当成了结论，尤其是那种来自所谓的权威人士的判断
   </strong>
   ，往往被作为理所当然的指标，波及到很多其他分析工作中。
  </p>
  <p class="graf graf--p">
   ⚠️这就是被骗的开端。权威崇拜会害死人的。
  </p>
  <p class="graf graf--p">
   苏德战争前，斯大林的阴谋论成为了情报机构情报分析的指南针，所谓的最高指示，任何不符合斯大林判断的情报都会被认为是假的。
  </p>
  <p class="graf graf--p">
   所有情报都被情报部长戈里克夫分为了“来源可靠”和“来源可疑”两部分。对那些与斯大林的看法相左的信息，根本不会向上级传送，直接扔掉。然后就是你知道的事情了。
  </p>
  <p class="graf graf--p">
   第四次中东战争中也发生过这种情况，以色列南部军区司令部情报处中校递交了一份报告，认为“从一切可以观察到的迹象来看，战争是不可避免的”。然而，这样的判断却被他的上级扣留了，理由是：“与情报部的主流观点不符”。
  </p>
  <p class="graf graf--p">
   多荒唐是吧。
   <strong class="markup--strong markup--p-strong">
    在集体组织社区和社会中经常会发生这种等级制的压迫，人们被动地被服从于权威，而不敢发表独立见解，哪怕见解非常重要；甚至即便发表了见解也会被压制而起不到作用。这是对批判性思考的最大阻碍。
   </strong>
  </p>
  <p class="graf graf--p">
   IYP在社运技巧的系列中提到过，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%8d%e8%a6%81%e8%ae%a9%e4%bd%a0%e7%9a%84%e5%af%b9%e6%89%8b%e5%ae%9a%e4%b9%89%e4%bd%a0%ef%bc%88%e4%ba%8c%ef%bc%89/" href="https://www.iyouport.org/%e4%b8%8d%e8%a6%81%e8%ae%a9%e4%bd%a0%e7%9a%84%e5%af%b9%e6%89%8b%e5%ae%9a%e4%b9%89%e4%bd%a0%ef%bc%88%e4%ba%8c%ef%bc%89/" rel="noopener noreferrer" target="_blank">
    社区组织中坚持“无结构原则”时出现的问题
   </a>
   。不存在真正的无结构，问题在于其领导者是否能尊重批判性思考、是否能把握平等交流的宗旨。
  </p>
  <p class="graf graf--p">
   社会不彰时社区必须发达，⚠️但社区就如家庭，它经常会变成社会的缩影。于是社区领导者必须有意识地避免这种影响，尤其是威权国家的社区，要想获得民主实践，对此更需要强调。
  </p>
  <p class="graf graf--p">
   情报界也有鲜明的等级制，但是分析人员必须知道，情报分析的质量和一个人的职务高度没有任何关系；即便是曾经得到了验证的假设，也需要再次进行验证。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    主动思考是批判性思维的重要特征
   </strong>
   。
  </p>
  <p class="graf graf--p">
   一个有判断力的思维主体绝不能被动地接受来自环境的刺激，对别人的观点不加思考地照单全收。应该亲自主动去分析，这样才能使主体的觉知本身也成为思考的结果。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    批判性思考的一个重点方面是：对自己的质疑
   </strong>
   。需要询问自己提出的问题中哪些是先入之见，支持自己判断的证据中哪些只是个人推论，等等。
  </p>
  <p class="graf graf--p">
   于是，💡如果团队中每个人都能秉承批判性思考 — 包括领导者 — 就不会出现权威依赖的糟糕结果。
  </p>
  <h4 class="graf graf--h4">
   <strong class="markup--strong markup--h4-strong">
    II 探寻真相的品质
   </strong>
  </h4>
  <p class="graf graf--p">
   探寻真相的品质是批判性思考的基础。只有具备这样的品质才能接受他人的质疑、或者对一些流行的“理所应当”表示怀疑。
  </p>
  <p class="graf graf--p">
   情报分析就是对真相的探求。
  </p>
  <p class="graf graf--p">
   在分析一个问题时，分析人员必须能做到自我质问：目前的态势有哪些异常？在表面正常的情况下是否隐藏着奇怪的暗流？目前自己掌握的情况真的全面吗？有没有空白？这种空白是如何产生的？是对手的拒止措施的结果吗？
   <strong class="markup--strong markup--p-strong">
    我所看到的东西是否是对手希望我看到的？
   </strong>
   有没有可能发生一些“不可能发生的事”？…… 等等。
  </p>
  <p class="graf graf--p">
   ⚠️这也是防止被宣传操纵的重要步骤。
  </p>
  <h4 class="graf graf--h4">
   <strong class="markup--strong markup--h4-strong">
    III 开放的胸怀
   </strong>
  </h4>
  <p class="graf graf--p">
   这对于思想者来说非常重要。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这个世界不存在终极真理，任何判断都需要接受质疑，分析者要有平等探讨的雅量
   </strong>
   。
  </p>
  <p class="graf graf--p">
   情报分析中这点格外重要。分析是一种主观的产物，任何判断只是你个人的看法，必须接受他人和物证（即新信息）的验证。
  </p>
  <p class="graf graf--p">
   批判性思维要明确认识到认知偏误的存在，同时认识到思维模式在其中所起的作用不可避免，于是为了克服先入之见，就需要从不同视角去批判性地考察自己的结论，不预设前提，不能偏听偏信，不要让自己的思维被单一解释所主宰。
  </p>

  <h4 class="graf graf--h4">
   <strong class="markup--strong markup--h4-strong">
    —— 发展创造性思维能力 ——
   </strong>
  </h4>
  <p class="graf graf--p">
   创造性思维是对应习常性思维而言的。
  </p>
  <p class="graf graf--p">
   创造性思维不能是简单地重复已有的思维过程和思维模式，而是在新的事实面前、在新的思维环境中，运用一种新的思维方法。
  </p>
  <p class="graf graf--p">
   它是思维的一种能力品质，
   <strong class="markup--strong markup--p-strong">
    其重要标志在于
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    善于在真正有问题的地方发现问题，并分析从而找出核心所在；
   </li>
   <li class="graf graf--li">
    善于建立独特的假说；
   </li>
   <li class="graf graf--li">
    善于安排巧妙的实验或观察来验证、修正或推翻假说；
   </li>
   <li class="graf graf--li">
    善于在实验和观察中发现相同事物的不同特点、以及不同事物的相同点；
   </li>
   <li class="graf graf--li">
    善于抓住偶然出现的异常，并追踪研究；
   </li>
   <li class="graf graf--li">
    善于从新实验中概括出新的理论见解，提出新的假说。
   </li>
  </ul>
  <p class="graf graf--p">
   创造性思维是一种联想性思维。它善于由此及彼产生连贯的思索，从一类事物联想到另一类，从一个思路到多个思路，从正向到逆向，引起一系列连锁反应，体现出思维的
   <strong class="markup--strong markup--p-strong">
    灵活性、变通性和流畅性
   </strong>
   。
  </p>
  <p class="graf graf--p">
   💡包括OSINT开源情报分析在内的一切分析都是如此，随时要记住，你的对手的主要工作就是避免被你抓到把柄，于是他们已经在千方百计地隐藏和欺骗。于是想要揭露他们，
   <strong class="markup--strong markup--p-strong">
    你的思维方式就必须比他们更灵活
   </strong>
   。
  </p>
  <p class="graf graf--p">
   还记得我们
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E7%BD%91%E7%BB%9C%E6%B0%B4%E5%86%9B%E5%A6%82%E4%BD%95%E8%AF%95%E5%9B%BE%E5%BD%B1%E5%93%8D%E9%80%89%E4%B8%BE%EF%BC%9F-%E4%B8%80%E4%B8%AA%E8%B0%83%E6%9F%A5%E6%A1%88%E4%BE%8B%EF%BC%8C%E8%A7%82%E5%AF%9F/" href="https://www.iyouport.org/%E7%BD%91%E7%BB%9C%E6%B0%B4%E5%86%9B%E5%A6%82%E4%BD%95%E8%AF%95%E5%9B%BE%E5%BD%B1%E5%93%8D%E9%80%89%E4%B8%BE%EF%BC%9F-%E4%B8%80%E4%B8%AA%E8%B0%83%E6%9F%A5%E6%A1%88%E4%BE%8B%EF%BC%8C%E8%A7%82%E5%AF%9F/" rel="noopener noreferrer" target="_blank">
    在挖网络水军的演示
   </a>
   中对头像照片的挖掘吗？对手非常精心地设计了一个真人照片，但是其背景是合成的。这是一个重要线索。
  </p>
  <p class="graf graf--p">
   相反，如果你的经验中不存在如此精细的欺骗案例，就很可能会错过这一重要线索。
  </p>
  <p class="graf graf--p">
   创造性思维对分析具有重要意义。它可以打破认知的禁锢，避免先入之见，从而*看到*新情况的出现，及时免于欺骗的侵害。
  </p>
  <p class="graf graf--p">
   911后不论是国安顾问还是国防部官员，都承认，他们真的从来没想到过飞机撞大楼的袭击方式。
  </p>
  <p class="graf graf--p">
   显然，⚠️发展严谨的逻辑思维能力的同时，情报人员必须能培养创造性思维，以确定各种非传统的威胁可能性，识别那些打破先例和趋势的突变性事件。
  </p>
  <p class="graf graf--p">
   创造性思维只能后天培养；通过心理上的“自我调节”有意识地培养自己的创造性思维。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    I 想象力
   </strong>
  </p>
  <p class="graf graf--p">
   想象力是人们运用储存在大脑中的信息进行综合分析、推断和设想的思维能力。
  </p>
  <p class="graf graf--p">
   在思维过程中，如果没有想象力的参与就会发生困难。
  </p>
  <p class="graf graf--p">
   创造性的思维灵感来自直觉和顿悟；往往需要长期的准备和积累，才会有看似短时间的突破。
  </p>
  <p class="graf graf--p">
   直觉属于创造性思维的范畴，它可以产生和形成于任何科学、艺术、技术产品的思想和构思。
  </p>
  <p class="graf graf--p">
   法国科学家 Poincaré 说过，
   <strong class="markup--strong markup--p-strong">
    逻辑是证明的工具，直觉是发现的工具，逻辑可以告诉我们哪一条路可以指引抵达目的地。为此，必须从远处瞭望目标，而瞭望的本领就是直觉。
   </strong>
  </p>
  <p class="graf graf--p">
   直觉思维在分析工作中非常重要。情报分析不仅是一门科学，也是一门艺术。
   <strong class="markup--strong markup--p-strong">
    你提出的假设越周详，失误的概率就越低
   </strong>
   。
  </p>
  <p class="graf graf--p">
   💡你的社区很可能是由共同理想或共同遭遇为纽带形成的，于是，请认识到你的周围人很可能具有不同的教育背景、不同的思维方式，看问题的视角完全不同，在这种情况下作为领导者/组织者，你应该采取一些机制化的方法，把不同年龄、不同文化背景的成员组成不同的研究小组，各自发展创造性思维，再将意见综合探讨。
  </p>
  <p class="graf graf--p">
   头脑风暴不仅是吸引人的游戏，也能做到突破传统思维的约束，得出惊喜的提升。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    II 培养发散思维
   </strong>
  </p>
  <p class="graf graf--p">
   发散思维可不是走神儿，而是
   <strong class="markup--strong markup--p-strong">
    从一个目标出发，沿着各种不同的途径去思考、探求各种问题的思维方式
   </strong>
   。发散性思维和创造性思维的主要特点。
  </p>
  <p class="graf graf--p">
   发散性思维的重要特点是逆向思维 thinking backwards —— 即 与一般思维程序相反的、或从事物的反面思考问题的方法。
  </p>
  <p class="graf graf--p">
   人的常规思维都是正向的，公式性的，逆向思维恰恰相反，但绝不是主张人们在思考时背离常规，而是训练一种小概率的思维模式，⚠️欺骗往往就利用人们不擅长逆向的特点实施的。
  </p>
  <p class="graf graf--p">
   分析人员首先假设未来确实会发生某些意料之外的事，然后将自己置身于未来，回顾过去，解释为什么会发生这样的事？在出现这种状况的半年或一年前发生了什么，才造成今天的可能性？依照这一思路一直推演到真实的当下，就可以获得一个全新的视角，找到一条避免未来噩梦的路径。
  </p>

  <h4 class="graf graf--h4">
   <strong class="markup--strong markup--h4-strong">
    —— 发展竞争性分析 ——
   </strong>
  </h4>
  <p class="graf graf--p">
   所谓的主流往往是过时的、笨拙的代名词，这可不是狂妄。
  </p>
  <p class="graf graf--p">
   ⚠️正因为大众都有信赖和依附主流观点的倾向，欺骗才得以实施的。还记得我们
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/?p=3136" href="https://www.iyouport.org/?p=3136" rel="noopener noreferrer" target="_blank">
    在操纵式宣传的分析中提到的“前期宣传”概念
   </a>
   吗？如果你直接宣传，很可能没人听你的，但是，如果你能够首先让你的主旨变成一种流行亚文化，人们就会自动顺应你的宣传目的。
  </p>
  <p class="graf graf--p">
   ⚠️这就是盲目追求主流带来的危害。有人在利用这种效应，而且多数情况下都不是善意的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    战略性欺骗也是如此。所有欺骗举措都围绕着强化欺骗对象内心预期而展开，因此，如果你能做到挑战在自己头脑中固有的思维模式，力争从多个角度分析情况，就能成为阻止被骗的关键武器
   </strong>
   。
  </p>
  <p class="graf graf--p">
   渗透测试的智慧就来自这里。比如我们介绍过的《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%87%aa%e6%88%91%e4%ba%ba%e8%82%89/" href="https://www.iyouport.org/%e8%87%aa%e6%88%91%e4%ba%ba%e8%82%89/" rel="noopener noreferrer" target="_blank">
    自我人肉
   </a>
   》思考方式（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%83%85%e6%8a%a5%e5%92%8c%e5%8f%8d%e6%83%85%e6%8a%a5%e5%9f%ba%e6%9c%ac%e5%b7%a5%e5%85%b7%e6%89%8b%e5%86%8c%ef%bc%9a%e8%87%aa%e6%88%91%e4%ba%ba%e8%82%89%ef%bc%882/" href="https://www.iyouport.org/%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%83%85%e6%8a%a5%e5%92%8c%e5%8f%8d%e6%83%85%e6%8a%a5%e5%9f%ba%e6%9c%ac%e5%b7%a5%e5%85%b7%e6%89%8b%e5%86%8c%ef%bc%9a%e8%87%aa%e6%88%91%e4%ba%ba%e8%82%89%ef%bc%882/" rel="noopener noreferrer" target="_blank">
    这里还有
   </a>
   自我人肉的工具和详细方法）。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    让你变成自己的对手，从对手的角度上对自己发起“攻击”，便很容易找到自己的破绽
   </strong>
   。
  </p>
  <p class="graf graf--p">
   竞争性分析有很多机制化的做法，比如和专业的渗透测试团队合作，魔鬼代言人游戏等等。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    I 魔鬼代言人
   </strong>
  </p>
  <p class="graf graf--p">
   也就是故意唱反调，这个词来自罗马天主教会。
  </p>
  <p class="graf graf--p">
   教会将那些信仰坚定 、品德高尚的人选为圣贤，为了确定其确实配得起圣贤的称号，教廷需要对候选人进行调查，主要就是挖掘候选人的任何污点。
  </p>
  <p class="graf graf--p">
   执行调查的人员就叫 Devil’s advocate。
  </p>
  <p class="graf graf--p">
   后来这个词延伸到现实生活中，指的就是在讨论中故意唱反调的人；
   <strong class="markup--strong markup--p-strong">
    提出互相矛盾的假设，鼓励人们进行思想交锋，从而使所有可能的判断都得到平等的对待。最终从中选出正确的那个
   </strong>
   。
  </p>
  <p class="graf graf--p">
   以色列摩萨德是最早使用这种方法的。在第四次中东战争爆发前，军事情报部长泽拉迷信以色列的军事优势，提出了两个信条：埃及在取得空中优势之前不会发动战争；没有埃及的配合叙利亚就没有能力发动战争 。
  </p>
  <p class="graf graf--p">
   显然由于泽拉的职位之高，他的论断被当成了金科玉律 ，凡是违背这两个假设的任何消息都被认为“不实”。
  </p>
  <p class="graf graf--p">
   南部军区的中尉两次呈交报告表明埃及的演习是为了掩饰它的战争准备，但报告都被情报部长驳回了。
  </p>
  <p class="graf graf--p">
   那次失败之后以色列认识到必须设立“魔鬼代言人”，以挑战主导地位的假设，才能防止情报失误。
  </p>
  <p class="graf graf--p">
   美国情报界也采取了这种方法，中情局的培训材料中就介绍了这种方法，即 找一位逻辑思维强大的人来唱反调，以破除思维定势。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    II 对抗分析法
   </strong>
  </p>
  <p class="graf graf--p">
   AB队的“对抗性分析”是美国的首创，在实践中有多次被使用。
  </p>
  <p class="graf graf--p">
   当年很多人认为美国低估了苏联的威胁性，于是1974年芝加哥大学教授 Albert Wohlstetter 提出要对中情局关于苏联的《国家分析评估》进行竞争性分析。
  </p>
  <p class="graf graf--p">
   中情局负责“A队”，政府其他机构和学术界组成“B队”，分析结果显示，美国的确是大大低估了苏联的战略力量，误以为在战略武器的使用方面苏联和美国“一样”。
  </p>
  <p class="graf graf--p">
   ⚠️这与组织辩论不同，辩论的双方很可能以自身的“胜利”为目标；而对抗分析则需要持不同意见的双方彼此参照对方的意见进行批判性思考。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    III 竞争性思维假设
   </strong>
  </p>
  <p class="graf graf--p">
   竞争性思维假设有8个步骤：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    召集持不同看法的分析人员集体讨论，确定所有可能存在的假设；
   </li>
   <li class="graf graf--li">
    列出支持和反对每种假设的重要证据和论据；
   </li>
   <li class="graf graf--li">
    完善上述表格，重新考虑以前的假设，删除无诊断价值的证据和论据；
   </li>
   <li class="graf graf--li">
    对每种假设的可能性做出临时结论，继续
    <strong class="markup--strong markup--li-strong">
     证伪
    </strong>
    每一种假设，而非证实；
   </li>
   <li class="graf graf--li">
    分析初步结论对于少数证据的依赖程度；
   </li>
   <li class="graf graf--li">
    上报结论，阐述
    <strong class="markup--strong markup--li-strong">
     所有
    </strong>
    假设的可能性，而不是可能性最大的假设；
   </li>
   <li class="graf graf--li">
    找出表明事态正在朝着意想不到方向发展的标志性事件，以供日后观测。
   </li>
  </ul>
  <p class="graf graf--p">
   虽然看起来步骤繁琐，但是有四个关键点：
   <strong class="markup--strong markup--p-strong">
    1、详细周全的假设；2、制作矩阵图，核对每种假设与证据的关系；3、对假设进行证伪而非证实；4、保持证伪工作的连续性
   </strong>
   。
  </p>
  <p class="graf graf--p">
   竞争性假设综合了上述所有方法，更容易发现对手潜在的欺骗行为，是克服先入之见的有力武器。
  </p>
  <p class="graf graf--p">
   大量的案例研究、不同的方法探讨，对于了解情报分析过程中的认知陷阱和克服思维误区来说，非常有效。
  </p>
  <p class="graf graf--p">
   总之，对欺骗性案例的研究越多，认识就能越深刻；在挖掘真相的行动中，越多的练习就能做到越熟练。实际操作起来并没有上述所描述得那么费时，
   <strong class="markup--strong markup--p-strong">
    当你能熟练使用反情报方法时就会觉得它如职业思维那般自如自然
   </strong>
   。⚪️
  </p>
