  <p class="graf graf--p">
   大多数人都将注意力集中在寡头公司随时随地的监控危机上，而很少有人会想到，自己每一天都在将大量的敏感私人数据传送给雇主。
  </p>
  <p class="graf graf--p">
   但事实上，雇主比以往任何时候都更加密切地监视着工人。
  </p>
  <p class="graf graf--p">
   研究和咨询公司 Gartner 在2018年对239家大公司进行的调查发现，超过一半的公司（三年前为30％）使用了“某种非传统监控技术”，
   <strong class="markup--strong markup--p-strong">
    其中包括诸如分析电子邮件和
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%80%e4%b9%88%e6%98%af%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%83%85%e6%8a%a5%ef%bc%9f%e4%bd%a0%e7%9a%84%e6%8e%a8%e7%89%b9%e5%a6%82%e4%bd%95%e6%88%90%e4%b8%ba%e6%8a%93%e6%8d%95%e4%bd%a0%e7%9a%84/" href="https://www.iyouport.org/%e4%bb%80%e4%b9%88%e6%98%af%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%83%85%e6%8a%a5%ef%bc%9f%e4%bd%a0%e7%9a%84%e6%8e%a8%e7%89%b9%e5%a6%82%e4%bd%95%e6%88%90%e4%b8%ba%e6%8a%93%e6%8d%95%e4%bd%a0%e7%9a%84/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     社交网络
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    的策略，甚至包括对
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab%e5%8f%8a%e5%af%b9%e6%8a%97%e6%96%b9%e6%b3%95/" href="https://www.iyouport.org/category/%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab%e5%8f%8a%e5%af%b9%e6%8a%97%e6%96%b9%e6%b3%95/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     生物特征
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    数据的收集
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e6%ad%a3%e5%9c%a8%e4%b8%ba%e4%b8%80%e4%b8%aa%e8%b4%b4%e8%ba%ab%e8%80%81%e5%a4%a7%e5%93%a5%e5%8a%aa%e5%8a%9b%e5%b7%a5%e4%bd%9c%ef%bc%9a%e5%9c%b0%e7%8b%b1%e7%9a%84%e6%96%b0%e5%b8%b8%e6%80%81/" href="https://www.iyouport.org/%e4%bd%a0%e6%ad%a3%e5%9c%a8%e4%b8%ba%e4%b8%80%e4%b8%aa%e8%b4%b4%e8%ba%ab%e8%80%81%e5%a4%a7%e5%93%a5%e5%8a%aa%e5%8a%9b%e5%b7%a5%e4%bd%9c%ef%bc%9a%e5%9c%b0%e7%8b%b1%e7%9a%84%e6%96%b0%e5%b8%b8%e6%80%81/" rel="noopener noreferrer" target="_blank">
     你正在为一个贴身老大哥努力工作：地狱的新常态
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   随着此类监控的普及，员工接受监控的意愿不仅没下降还随之提高了，尤其是在雇主坦白的情况下：Gartner发现，2018年有十分之三的员工（2015年为十分之一的员工）对雇主监控自己的电子邮件之行为“感到满意”。在了解了跟踪原因之后，超过一半的人甚至 “感到安心”。
  </p>
  <p class="graf graf--p">
   真可怕……
  </p>
  <p class="graf graf--p">
   劳工律师 Paula Brantner 表示：
   <strong class="markup--strong markup--p-strong">
    “越来越难以使人们对这些工作场所的入侵给予足够的关注，以至于根本没有任何回击，更不用说有效阻止雇主这样做了。”
   </strong>
  </p>
  <p class="graf graf--p">
   尽管我们拥有基本的隐私权，但我们实际上已经“放弃了”，驻 Raleigh 的人力资源顾问 Laurie Ruettimann 说：“人们说，‘你知道吗，我们会给你一些数据，以换取薪水’……
  </p>
  <p class="graf graf--p">
   康奈尔大学工业和劳资关系学院的助理教授 Ifeoma Ajunwa 说，雇主监视员工的主要方法有三种：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    包括通过公司分发的手机的 GPS 或员工 ID 徽章进行位置跟踪，通过电子邮件进行通讯监视，监视 Slack 消息，甚至击键记录，以及跟踪健康数据（包括睡眠模式）
   </strong>
   。
  </p>
  <p class="graf graf--p">
   许多蓝领员工也越来越多地发现自己
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wsj.com/articles/memo-to-workers-the-boss-is-watching-1382485923" href="https://www.wsj.com/articles/memo-to-workers-the-boss-is-watching-1382485923" rel="noopener noreferrer" target="_blank">
    受到监视
   </a>
   。例如，据报道，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theverge.com/2019/4/25/18516004/amazon-warehouse-fulfillment-centers-productivity-firing-terminations" href="https://www.theverge.com/2019/4/25/18516004/amazon-warehouse-fulfillment-centers-productivity-firing-terminations" rel="noopener noreferrer" target="_blank">
    亚马逊可跟踪每位员工的工作效率
   </a>
   。
  </p>
  <p class="graf graf--p">
   总部位于威斯康星州的科技公司 Three Square Market
   <a class="markup--anchor markup--p-anchor" data-href="https://www.marketwatch.com/story/5-ways-your-employer-is-tracking-you-2017-07-25-11882132" href="https://www.marketwatch.com/story/5-ways-your-employer-is-tracking-you-2017-07-25-11882132" rel="noopener noreferrer" target="_blank">
    在2017年
   </a>
   爆发了
   <strong class="markup--strong markup--p-strong">
    将微芯片植入员工身体
   </strong>
   的新闻，声称为员工提供了一种更无缝的方式使用门禁并在休息室进行购买（该公司坚持认为这种方式“不是跟踪”）。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    防止工作场所监视的法律保护
   </strong>
  </p>
  <p class="graf graf--p">
   不同地区对工作场所监视的保护措施各不相同，Ajunwa 和她在
   <a class="markup--anchor markup--p-anchor" data-href="https://papers.ssrn.com/sol3/papers.cfm?abstract_id=2746211" href="https://papers.ssrn.com/sol3/papers.cfm?abstract_id=2746211" rel="noopener noreferrer" target="_blank">
    2017年的论文
   </a>
   的合著者认为，“我们不能依靠每个州的立法机构来（零星地）完成保护工人的工作。”
  </p>
  <p class="graf graf--p">
   Ajunwa 说：“
   <strong class="markup--strong markup--p-strong">
    没有
   </strong>
   针对工作场所监视的联邦法律 —— 通常，雇
   <strong class="markup--strong markup--p-strong">
    主有完整的权力可以在工作场所或工作设备上安装任何监视系统
   </strong>
   。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “一个警告是，这种监视在骚扰员工或实施就业歧视方面已经是非法的。”
  </p>
  <p class="graf graf--p">
   更重要的是，EFF 技术员 Seth Schoen 说，员工很难在公司提供的设备上检测到多种类型的监视 —— “如果雇主希望监视行为保密，那么就基本上难以检测到。”
  </p>
  <p class="graf graf--p">
   Schoen 在一封电子邮件中说：
   <strong class="markup--strong markup--p-strong">
    “雇主几乎可以无限制地篡改硬件和软件，以安装监视工具，并且员工无法期望找到这些监视工具。”
   </strong>
  </p>
  <p class="graf graf--p">
   他补充说：“我相信技术精湛的雇主或IT部门可以在用户无法找到的设备上轻松设置监视功能，特别是如果员工没有管理员权限的话。”
  </p>
  <p class="graf graf--p">
   （Schoen 说，员工可以检查的一件事是 “雇主是否正在使用自己的证书颁发机构来拦截其HTTPS加密通信。”）
  </p>
  <p class="graf graf--p">
   不论如何，您依旧可以采取一些步骤来保护自己免受公司老大哥的监视，并最大程度地降低自己的风险。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    专家建议
   </strong>
   ：
  </p>
  <p class="graf graf--p">
   1、如有任何疑问，请假设您正在受到监视。
  </p>
  <p class="graf graf--p">
   “一般来说，如果您担心工作场所的隐私状况，我会假设您在公司电子邮件、公司Wi-Fi、公司笔记本电脑、或公司空间中所做的任何事，都可以被跟踪和监视”，耶鲁大学法学院信息社会计划的研究员 Tiffany Li 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这可能包括任何东西，从记录办公室情况的摄像头到跟踪公司笔记本电脑上员工活动的软件。因此，即使您看不到别人在监视着你，保持专业水平也很重要。”
  </p>
  <p class="graf graf--p">
   监视公司 Slack 去年
   <a class="markup--anchor markup--p-anchor" data-href="https://nypost.com/2018/03/22/slack-will-allow-employers-to-read-your-private-messages/" href="https://nypost.com/2018/03/22/slack-will-allow-employers-to-read-your-private-messages/" rel="noopener noreferrer" target="_blank">
    更新了其隐私政策
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    允许雇主订购其高级计划，以在员工完全不知情的情况下下载员工的私人通信消息
   </strong>
   。
  </p>
  <p class="graf graf--p">
   尽管大多数公司老板可能都知道员工白天会休息一下，以仔细阅读新闻或浏览 Zappos，但 Ruettimann 警告说，“作为员工您应该了解雇主对这种与工作完全无关的活动的了解程度”。
  </p>
  <p class="graf graf--p">
   2、将工作业务与个人业务完全分开。
  </p>
  <p class="graf graf--p">
   Ruettimann 建议维护与工作设备完全分开的个人设备，并
   <strong class="markup--strong markup--p-strong">
    确保不要将个人电子设备连接到公司的无线网络上。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “通常，
   <strong class="markup--strong markup--p-strong">
    与您公司的 Wi-Fi 相连的任何设备都可以根据您的雇主的条款和条件进行扫描和监视。即使您是承包商或顾问，在设备方面，您也应该像雇员一样警惕这种监视
   </strong>
   。
  </p>
  <p class="graf graf--p">
   使用电子邮件也是如此。人们经常犯的一个错误是将公司电子邮件视为个人电子邮件。这就相当于被全面监视。
  </p>
  <p class="graf graf--p">
   3、了解您向雇主提供了哪些信息。
  </p>
  <p class="graf graf--p">
   Schoen说，员工应依靠法律或工会合同，要求披露监视​​信息，并要求人力资源主动披露监视信息。
  </p>
  <p class="graf graf--p">
   如果您担心与您的人力资源或IT部门联系会有麻烦，您可以要求查看与公司的员工隐私政策有关的文档。
  </p>
  <p class="graf graf--p">
   您还需要问一些特定的问题：
   <strong class="markup--strong markup--p-strong">
    公司计算机或电话上是否有跟踪软件？您的电子邮件被记录了吗？你的雇主正在收集或存储哪些员工数据？谁有权访问员工数据？你需要知道这些问题的答案。
   </strong>
  </p>
  <p class="graf graf--p">
   4、请仔细阅读文件和合同。
  </p>
  <p class="graf graf--p">
   很多（监控）必须得到员工的同意，但这通常呈现在您的入职合同或雇用合同中，因此您需要仔细查看。
  </p>
  <p class="graf graf--p">
   阅读合同、入职文书和任何公司培训或文件，因为它们有时会包含关于雇主在工作场所追踪的具体规定。
  </p>
  <p class="graf graf--p">
   5、认真考虑工作中的健康状况监视。
  </p>
  <p class="graf graf--p">
   工作场所健康计划还可以监控您运动的频率或是否去健身房，其中一些甚至可以决定您购买的健康保险是否能享受折扣。
  </p>
  <p class="graf graf--p">
   但是人们应该意识到放弃隐私的隐患，您将自己全部私生活透露给雇主的结果仅仅是换取一点点折扣的可能性，这值得吗？
  </p>
  <p class="graf graf--p">
   此外，请谨慎使用笔记本电脑共享任何个人健康相关的信息 ——
   <strong class="markup--strong markup--p-strong">
    不要使用公司网络搜索有关生育、心理健康、疾病等相关的任何信息
   </strong>
   ，这些信息可能会无意中落入雇主的手中。当必须裁员时，你就会陷入危机。
  </p>
  <p class="graf graf--p">
   6、意识到作为同事的诉讼的一部分，您的通讯可能会受到审查。
  </p>
  <p class="graf graf--p">
   通常，如果公司被其一名雇员起诉，那么您的某些信息可能最终会作为诉讼的证据。通常，在此过程中，雇主会要求提供某些文件，这些文件可以作为诉讼中的证据 —— 因此
   <strong class="markup--strong markup--p-strong">
    您应注意，您的通信历史记录中是否有任何东西在未来有可能成为呈堂证供。
   </strong>
  </p>
  <p class="graf graf--p">
   7、与其放弃，不如尝试从内部改变。
  </p>
  <p class="graf graf--p">
   Ruettimann 说：“我一直在鼓励那些因这一切感到愤怒的人得到晋升并进入权力位置。置之不理是无法修复任何东西的，只有了解它并坚持修复才有希望彻底改变这一状况。” ⚪️
  </p>
