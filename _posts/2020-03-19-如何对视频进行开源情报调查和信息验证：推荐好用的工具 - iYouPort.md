  <p class="graf graf--p">
   许多互联网调查和信息验证都涉及视频证据。
   <strong class="markup--strong markup--p-strong">
    能够捕获和分析视频内容是一项重要技能，尤其是在您的任务涉及识别人员或地点时
   </strong>
   。
  </p>
  <p class="graf graf--p">
   对于
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/10%e4%b8%aa%e7%ae%80%e5%8d%95%e6%8a%80%e5%b7%a7-%e5%bf%ab%e9%80%9f%e9%aa%8c%e8%af%81%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e7%83%ad%e9%97%a8%e8%a7%86%e9%a2%91%e7%9a%84%e7%9c%9f%e4%bc%aa/" href="https://www.iyouport.org/10%e4%b8%aa%e7%ae%80%e5%8d%95%e6%8a%80%e5%b7%a7-%e5%bf%ab%e9%80%9f%e9%aa%8c%e8%af%81%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e7%83%ad%e9%97%a8%e8%a7%86%e9%a2%91%e7%9a%84%e7%9c%9f%e4%bc%aa/" rel="noopener noreferrer" target="_blank">
    视频信息的验证
   </a>
   ，我们经常会说 “取一帧”
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%90%9c%e5%9b%be%e5%8a%9f%e8%83%bd%e5%93%aa%e5%ae%b6%e5%bc%ba%ef%bc%9f%e5%9b%be%e5%83%8f%e9%aa%8c%e8%af%81%e6%96%b9%e6%b3%95%e7%9a%84%e8%83%bd%e5%8a%9b%e5%af%b9%e6%af%94/" href="https://www.iyouport.org/%e6%90%9c%e5%9b%be%e5%8a%9f%e8%83%bd%e5%93%aa%e5%ae%b6%e5%bc%ba%ef%bc%9f%e5%9b%be%e5%83%8f%e9%aa%8c%e8%af%81%e6%96%b9%e6%b3%95%e7%9a%84%e8%83%bd%e5%8a%9b%e5%af%b9%e6%af%94/" rel="noopener noreferrer" target="_blank">
    进行反向图像搜索
   </a>
   ，但是人们往往很难手动抓取到关键帧。
  </p>
  <p class="graf graf--p">
   您可以随时暂停视频以尝试查看感兴趣的细节，但是这可能会很慢且效率很低，尤其是当重要的细节仅持续一秒钟时。
  </p>
  <p class="graf graf--p">
   本文将推荐一个工具，帮助您更简单地抓取到视频内容中的关键情报，以及对视频内容进行真伪验证。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://ffmpeg.org/" href="https://ffmpeg.org/" rel="noopener noreferrer" target="_blank">
    FFMPEG
   </a>
   是一个简单但功能强大的工具，可让您从视频中提取每个帧并将其转换为PNG或JPG格式的静止图像。
  </p>
  <p class="graf graf--p">
   它还允许您从视频文件中获取音频，以便您可以仔细收听和分析它。无需不断暂停即可找到所需的详细信息。nixintel进行了一个很好的总结。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     安装和设定
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   FFMPEG 是在 Windows、Mac OS 和 Linux 上运行的
   <strong class="markup--strong markup--p-strong">
    免费
   </strong>
   命令行工具。每个平台上的安装和设置都不同，但之后的用法都是相同的。
  </p>
  <p class="graf graf--p">
   下面逐一介绍。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     Windows ——
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   建议您遵循
   <a class="markup--anchor markup--p-anchor" data-href="http://blog.gregzaal.com/how-to-install-ffmpeg-on-windows/" href="http://blog.gregzaal.com/how-to-install-ffmpeg-on-windows/" rel="noopener noreferrer" target="_blank">
    这里的指南
   </a>
   了解如何为 Windows 安装 FFMPEG，而不仅仅是重复别人写的东西。
  </p>
  <p class="graf graf--p">
   只需记住从下载页面选择正确的版本即可。
  </p>
  <p class="graf graf--p">
   除非您运行的是非常非常古老的 Windows 计算机，否则任何64位版本的都可以。另外，这里有一个很棒的视频指南：
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/qjtmgCb8NcE" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     Mac OS ——
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   FFMPEG 可以直接从终端安装并运行。如果您尚未安装 Homebrew，则需要先按照
   <a class="markup--anchor markup--p-anchor" data-href="https://brew.sh/" href="https://brew.sh/" rel="noopener noreferrer" target="_blank">
    这里的指南
   </a>
   进行操作。
  </p>
  <p class="graf graf--p">
   安装完成后，打开一个终端并输入以下内容：
  </p>
  <pre class="graf graf--pre"><code>brew install ffmpeg</code></pre>
  <p class="graf graf--p">
   就行了。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     Linux ——
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   FFMPEG 可能已经安装在您的 Linux 版本中了。要检查是否存在，请打开一个终端并输入：
  </p>
  <pre class="graf graf--pre"><code>which ffmpeg</code></pre>
  <p class="graf graf--p">
   如果已安装 FFMPEG，则会显示文件路径 (usually /usr/bin/ffmpeg)。如果未安装，则不会返回任何消息。
  </p>
  <p class="graf graf--p">
   要将其安装在任何基于 Debian 的发行版中，例如 Ubuntu、Kali 或 Mint，只需在终端中输入以下内容：
  </p>
  <pre class="graf graf--pre"><code>sudo apt install ffmpeg</code></pre>
  <p class="graf graf--p">
   运行安装过程，您就完成了。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     用法
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   FFMPEG 完全在命令行中运行。您输入的命令采用以下格式：
  </p>
  <pre class="graf graf--pre"><code>ffmpeg &lt;global options&gt; &lt;input file options&gt; -i  &lt;input file&gt; &lt;output file options&gt; &lt;output file&gt;.</code></pre>
  <p class="graf graf--p">
   不要害怕，其实没那么复杂。
   <strong class="markup--strong markup--p-strong">
    FFMPEG 可以完成各种有用的任务
   </strong>
   ；转换视频格式、调整视频大小、添加音频、删除音频、添加字幕、拆分和合并视频，以及其他许多操作 —— 但本指南将侧重于主题。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    本文的目的是开源情报调查和信息验证
   </strong>
   。
  </p>
  <p class="graf graf--p">
   同样重要的是要记住，与大多数其他命令行工具一样，FFMPEG 假定您已经在目标视频文件所在的文件夹中。您可以在
   <strong class="markup--strong markup--p-strong">
    -i
   </strong>
   后指定文件的路径，也可以使用
   <strong class="markup--strong markup--p-strong">
    cd
   </strong>
   移至您要进入的目录。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     1）从视频中抓取帧
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   这是用于调查和分析视频的非常有用的技术。
  </p>
  <p class="graf graf--p">
   从视频捕获静止图像允许使用反向图像搜索对其进行真伪验证，并允许对视频进行逐帧评估。它比您尝试在适当的时间点暂停视频并希望可以找到想要的东西要有效得多。
  </p>
  <p class="graf graf--p">
   要捕获视频中的所有帧，请使用以下命令：
  </p>
  <pre class="graf graf--pre"><code>ffmpeg -i myvideo.mp4 img%06d.png -hide_banner</code></pre>
  <p class="graf graf--p">
   以下是命令中每个部分的作用，明白意思就容易记住了：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     ffmpeg
    </strong>
    — 告诉计算机运行 ffmpeg
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     -i
    </strong>
    告诉 ffmpeg 输入文件的名称是什么，即 您要处理的视频
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     myvideo.mp4
    </strong>
    — 视频文件
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     img%06d.png
    </strong>
    — 静止图像的命名协议。这种语法意味着每个帧都将带有 img 标签。 ％06d表示 img 后将跟随六位数字；因此，第一个仍然是 img000001.png，然后是 img000002.png，依此类推。％04d 只能是四位数，依此类推。 .png 告诉 ffmpeg 您想要的静止图像的格式。如果您更喜欢 jpeg，请使用 .jpg。
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     -hide_banner
    </strong>
    抑制大量不必要的文本输出
   </li>
  </ul>
  <p class="graf graf--p">
   一旦运行完成，您就会获得一个编号的 PNG 文件集合，视频中的每一帧都有一个图像。
  </p>
  <p class="graf graf--p">
   这对于简短的视频来说很好，但是您很快就会得到并不需要的成千上万个几乎完全相同的帧。
  </p>
  <p class="graf graf--p">
   例如，当我使用这种技术从视频中获取静止图像时，即使原始镜头只有14秒长，我最终还是获得了400多个静止图像！
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10375 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-7.png?resize=1100%2C582&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-7.png?w=1400&amp;ssl=1 1400w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-7.png?resize=300%2C159&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-7.png?resize=768%2C406&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-7.png?resize=1024%2C542&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-7.png?resize=1100%2C582&amp;ssl=1 1100w" data-recalc-dims="1" height="582" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-7.png?resize=1100%2C582&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10375" data-recalc-dims="1" height="582" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-7.png?resize=1100%2C582&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-7.png?w=1400&amp;ssl=1 1400w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-7.png?resize=300%2C159&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-7.png?resize=768%2C406&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-7.png?resize=1024%2C542&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-7.png?resize=1100%2C582&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   不要着急，
   <strong class="markup--strong markup--p-strong">
    FFMPEG 允许您指定每秒要捕获多少帧
   </strong>
   。缺省情况下，它每秒捕获25帧，但是可以使用
   <strong class="markup--strong markup--p-strong">
    -r
   </strong>
   指定每秒应捕获多少帧：
  </p>
  <pre class="graf graf--pre"><code>ffmpeg -i myvideo.mp4 -r 1 img%06d.png -hide_banner</code></pre>
  <p class="graf graf--p">
   因此，在此示例中，
   <strong class="markup--strong markup--p-strong">
    -r
   </strong>
   的值为1。这意味着，视频的每一秒只捕获一帧。对于长度超过几秒钟的文件，这样更易于管理。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     2）从视频中获取音频
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   FFMPEG 还允许您从视频中提取声音，以便可以将其与视频本身分开进行处理。
  </p>
  <p class="graf graf--p">
   语法如下：
  </p>
  <pre class="graf graf--pre"><code>ffmpeg -i myvideo.mp4 -vn soundtrack.mp3</code></pre>
  <p class="graf graf--p">
   此命令告诉 ffmpeg 接收输入文件
   <strong class="markup--strong markup--p-strong">
    myvideo.mp4
   </strong>
   ，忽略所有视频内容（
   <strong class="markup--strong markup--p-strong">
    -vn
   </strong>
   ），然后将声音输出到名为
   <strong class="markup--strong markup--p-strong">
    soundtrack.mp3
   </strong>
   的新文件中。
  </p>
  <p class="graf graf--p">
   这将使您既保留原始视频文件，又保留了包含声音的单独mp3文件。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     将其与 YouTube-DL 整合在一起
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   如果您忘记了
   <strong class="markup--strong markup--p-strong">
    关于 YouTube-DL 的使用方法
   </strong>
   ，在这里回顾：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%bf%ab%e9%80%9f%e4%bb%8e%e5%a4%a7%e9%83%a8%e5%88%86%e7%a4%be%e4%ba%a4%e7%bd%91%e7%bb%9c%e4%b8%8a%e6%8a%93%e5%8f%96%e8%a7%86%e9%a2%91%ef%bc%9f%e6%b7%b1%e5%ba%a6%e8%b0%83%e6%9f%a5/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%bf%ab%e9%80%9f%e4%bb%8e%e5%a4%a7%e9%83%a8%e5%88%86%e7%a4%be%e4%ba%a4%e7%bd%91%e7%bb%9c%e4%b8%8a%e6%8a%93%e5%8f%96%e8%a7%86%e9%a2%91%ef%bc%9f%e6%b7%b1%e5%ba%a6%e8%b0%83%e6%9f%a5/" rel="noopener noreferrer" target="_blank">
    如何快速从大部分社交网络上抓取视频？深度调查的必备基础工具
   </a>
   》。
  </p>
  <p class="graf graf--p">
   使用 YouTube-dl 捕获视频，然后使用 FFMPEG 将所有帧转换为图像文件、并提取音频，以便分别收听分析。
  </p>
  <p class="graf graf--p">
   YouTube-dl 是很棒的互联网视频采集器，它与 Twitter 内容非常兼容。如果您以前从未使用过它，建议阅读上述指南。
  </p>
  <p class="graf graf--p">
   下面是一个案例，结合使用这两种工具将一个14秒视频转换成更易于使用的方式 —— 这个视频是
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/nixintel/status/1188358090481119232" href="https://twitter.com/nixintel/status/1188358090481119232" rel="noopener noreferrer" target="_blank">
    一个侦探挑战游戏的测试题目
   </a>
   ，它的问题是：该视频中的火车是开往哪里的。
  </p>
  <p class="graf graf--p">
   以下是调查步骤。
  </p>
  <p class="graf graf--p">
   1、使用 YouTube-DL 抓取视频，将其另存为 train_footage.mp4
  </p>
  <pre class="graf graf--pre"><code>$ youtube-dl https://twitter.com/dondude/status/1187776360544120832 train_footage.mp4</code></pre>
  <p class="graf graf--p">
   2、抓取视频的每一帧并将其另存为PNG文件：
  </p>
  <pre class="graf graf--pre"><code>$ ffmpeg -i train_footage.mp4 img%06d.png -hide_banner</code></pre>
  <p class="graf graf--p">
   3、翻录视频中的音频，并将其另存为单独的 mp3，称为 train_audio：
  </p>
  <pre class="graf graf--pre"><code>$ ffmpeg -i train_footage.mp4 -vn train_audio.mp3</code></pre>
  <p class="graf graf--p">
   事实证明，音频对于本案例来说没有帮助，但是，
   <strong class="markup--strong markup--p-strong">
    从其中一个视频帧中发现了在原始镜头中错过的一个重要细节
   </strong>
   ：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10376 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-3.png?resize=1100%2C619&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-3.png?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-3.png?resize=300%2C169&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-3.png?resize=768%2C432&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-3.png?resize=1024%2C576&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-3.png?resize=1100%2C619&amp;ssl=1 1100w" data-recalc-dims="1" height="619" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-3.png?resize=1100%2C619&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10376" data-recalc-dims="1" height="619" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-3.png?resize=1100%2C619&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-3.png?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-3.png?resize=300%2C169&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-3.png?resize=768%2C432&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-3.png?resize=1024%2C576&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-3.png?resize=1100%2C619&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   当查看上面的静止图像时，建筑物后部的卫星天线清晰可见：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10377 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-4.png?resize=1100%2C902&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-4.png?w=1132&amp;ssl=1 1132w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-4.png?resize=300%2C246&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-4.png?resize=768%2C630&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-4.png?resize=1024%2C839&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-4.png?resize=1100%2C902&amp;ssl=1 1100w" data-recalc-dims="1" height="902" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-4.png?resize=1100%2C902&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10377" data-recalc-dims="1" height="902" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-4.png?resize=1100%2C902&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-4.png?w=1132&amp;ssl=1 1132w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-4.png?resize=300%2C246&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-4.png?resize=768%2C630&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-4.png?resize=1024%2C839&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-4.png?resize=1100%2C902&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   由于卫星天线通常
   <a class="markup--anchor markup--p-anchor" data-href="https://www.quora.com/What-is-the-reason-that-satellite-dishes-face-south" href="https://www.quora.com/What-is-the-reason-that-satellite-dishes-face-south" rel="noopener noreferrer" target="_blank">
    指向赤道
   </a>
   ，因此可以确定照片的方位。天线朝南并正对着相机，因此拍摄者必须是面朝北的。
  </p>
  <p class="graf graf--p">
   这意味着火车正在向西行驶，因此火车刚离开的轨道和车站必须是东西方向。
  </p>
  <p class="graf graf--p">
   结合其他一些细节，可以使用上述信息来缩小位置范围。
  </p>
  <p class="graf graf--p">
   视频是开动的火车上拍摄的，如果您没有看到那个天线，就会错过一个重要的验证细节。这就是准确提取帧的重要性。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     其他任务
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   ffmpeg 的另一个简单任务就是调整视频大小，也就是说如果看不清的话可以改大一点，您需要做的就是在
   <strong class="markup--strong markup--p-strong">
    -s
   </strong>
   后指定新的分辨率：
  </p>
  <pre class="graf graf--pre">ffmpeg -i video_input.mov -s 1024x576 video_output.mp4</pre>
  <p class="graf graf--p">
   另外，您可能要指定
   <strong class="markup--strong markup--p-strong">
    -c：a
   </strong>
   以确保输出文件的音频编解码器是正确的：
  </p>
  <pre class="graf graf--pre">ffmpeg -i video_input.h264 -s 640x480 -c：a video_output.mov</pre>
  <p class="graf graf--p">
   您还可以使用
   <strong class="markup--strong markup--p-strong">
    -aspect
   </strong>
   更改长宽比：
  </p>
  <pre class="graf graf--pre">ffmpeg -i video_input.mp4-方面4：3 video_output.mp4</pre>
  <p class="graf graf--p">
   还有
   <strong class="markup--strong markup--p-strong">
    将音频转换为视频
   </strong>
   的方法。
  </p>
  <p class="graf graf--p">
   使用单张照片（例如专辑封面）给它添加音频。
  </p>
  <p class="graf graf--p">
   当您要将音频文件上传到除视频/图像之外不允许任何其他内容的网站时（例如 YouTube 和 Facebook），此功能非常有用。
  </p>
  <p class="graf graf--p">
   这是一个例子：
  </p>
  <pre class="graf graf--pre">ffmpeg -loop 1 -i image.jpg -i audio.wav -c:v libx264 -c:a aac -strict experimental -b:a 192k -shortest output.mp4</pre>
  <p class="graf graf--p">
   只需要改变编解码器（
   <strong class="markup--strong markup--p-strong">
    -c:v
   </strong>
   指定视频编解码器，
   <strong class="markup--strong markup--p-strong">
    -c:a
   </strong>
   指定音频编解码器）和你的文件的名称。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    给视频添加字幕
   </strong>
   ：
  </p>
  <p class="graf graf--p">
   添加字幕的方法是这样的，输入以下内容：
  </p>
  <pre class="graf graf--pre">ffmpeg -i video.mp4 -i subtitles.srt -c:v copy -c:a copy -preset veryfast -c:s mov_text -map 0 -map 1 output.mp4</pre>
  <p class="graf graf--p">
   当然，您可以指定所需的任何编解码器（以及与音频/视频相关的任何其他附加选项）。
  </p>
  <p class="graf graf--p">
   更多详细功能的指南可以
   <a class="markup--anchor markup--p-anchor" data-href="https://itsfoss.com/ffmpeg/" href="https://itsfoss.com/ffmpeg/" rel="noopener noreferrer" target="_blank">
    参见这里
   </a>
   。⚪️
  </p>
