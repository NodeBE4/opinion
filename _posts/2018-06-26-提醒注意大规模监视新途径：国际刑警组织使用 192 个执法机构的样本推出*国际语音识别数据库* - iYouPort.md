   SiiP’s database will include samples from YouTube, Facebook, publicly recorded conversations, and other sources where individuals might not realize that their voices are being turned into biometric voice prints.

  <p class="graf graf--p graf-after--p" id="5102">
   <strong class="markup--strong markup--p-strong">
    编者按：
   </strong>
   当权者一直在千方百计地打破匿名。不论是线上加密技术、还是线下躲避监控摄像头的面纱，每一种寻求隐私权的努力都面对着大规模监控肆无忌惮的挑战。这篇报道所揭示的技术说明，这一挑战正在上升到前所未有的严重程度。
  </p>
  <p class="graf graf--p graf-after--p" id="22be">
   两年前一位采访敏感人士的独立记者告诉我，他采取了面部打码以保护受访对象的隐私。我提醒他：
   <strong class="markup--strong markup--p-strong">
    声纹也应该被编辑
   </strong>
   ，并且要彻底销毁原始录音。利用声纹识别一个人的技术早已不是科幻电影中才有的东西。The Intercept 的报道证实了我当年的担忧。
  </p>
  <p class="graf graf--p graf-after--p" id="22b1">
   昨天我们发布了一则消息：一名英国的税务人员在一个有争议的身份识别计划中暗中收录了数百万人的独特“声纹”，隐私活动人士警告这可能是非法的。政府已承认其在未经许可的情况下存储录音，现在正面临调查。隐私保护组织 Big Brother Watch 揭示，⚠️ 自去年年初以来已经有
   <strong class="markup--strong markup--p-strong">
    510 万名拨打热线电话的用户不得不说出’我的声音是我的密码’这句话。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="f805">
   这就是被纳入语音识别数据库的声音数据。每个人的声音都被记录下来并存储起来，用作高科技“安检”。在回电时声音被作为密码来解锁账户。⚠️ 该计划的危险在于人们
   <strong class="markup--strong markup--p-strong">
    无法选择退出
   </strong>
   ，声纹被强制收集并储存，没有经过每个人的同意。报道指出：万一数据库泄漏会导致个人银行账户被入侵。
  </p>
  <p class="graf graf--p graf-after--p" id="c02d">
   现在看起来银行账户不是唯一面临危险的东西，英国也不是唯一采取这种方式侵犯隐私权的国家。国际刑警组织建立的语音生物识别数据库囊括了全球
   <strong class="markup--strong markup--p-strong">
    192
   </strong>
   个执法机构的数据，这些数据是被那些政府利用各种方式 — 包括强制和诱骗 — 收集到的。并且你我不可能知道当权者会如何利用这些数据。
  </p>
  <p class="graf graf--p graf-after--p" id="fc00">
   我曾经提醒一位正在学习德语的朋友：
   <strong class="markup--strong markup--p-strong">
    不用使用学习软件提供的按键朗读功能
   </strong>
   ，你不知道你的语音数据会被什么人拿到，以及被用来做什么。这篇报道中所指出的，中国当局利用移动电话网络收集声纹数据的做法，则更加值得警惕。
  </p>
  <p class="graf graf--p graf-after--p" id="c556">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    此事提醒人们注意：涉及重要话题时请不要使用语音通讯。
   </strong>
  </p>

  <p class="graf graf--p graf-after--figure" id="76ba">
   这是一项为期四年的耗资 1000万 欧元的项目，最近即将完成。
  </p>
  <p class="graf graf--p graf-after--p" id="f7fb">
   该项目名为 “Speaker Identification Integrated Project” 他们称之为 SiiP，标志着
   <strong class="markup--strong markup--p-strong">
    语音生物识别技术在执法领域国际扩展
   </strong>
   方面的一项重大发展 — 并且在隐私方面引发高度警惕。
  </p>
  <p class="graf graf--p graf-after--p" id="3e5e">
   这种语音识别技术的工作原理是获取已知语音的样本，捕捉其独特性，然后将这些特征转换成称为声纹或语音模型的算法模板。
  </p>
  <p class="graf graf--p graf-after--p" id="e99f">
   通过在
   <strong class="markup--strong markup--p-strong">
    全球的音频数据库
   </strong>
   中收集到足够的声纹和样本，国际刑警组织的 SiiP 系统将能够上传未知的声音，并且不论是什么语言，都将这一声音数据与可能的备选列表进行匹配，以识别身份。
  </p>
  <p class="graf graf--p graf-after--p" id="9baf">
   SiiP 的数据库允许上传和下载来自全球 192 个执法机构的样本。
  </p>
  <p class="graf graf--p graf-after--p" id="eaf3">
   SiiP 将加入国际刑警组织现有的指纹和人脸识别数据库，其关键“优势”在于能促进快速识别过程 — 比如绑架者拨打电话 — 这种没有其他识别码可以利用的时候。该平台还具备按性别、年龄、语言和口音过滤声音样本的功能。当录音来自类似的声学环境时，准确率可能非常高。
  </p>
  <p class="graf graf--p graf-after--p" id="407f">
   <strong class="markup--strong markup--p-strong">
    这项语音识别技术可以在每一次开口讲话的同时进行识别，并标记身份，基本可以称之为
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.forbes.com/sites/realspin/2016/10/06/voice-recognition-every-single-day-every-word-you-say/#30bc3f48786d" href="https://www.forbes.com/sites/realspin/2016/10/06/voice-recognition-every-single-day-every-word-you-say/#30bc3f48786d" rel="nofollow noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     匿名终结者
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="bf04">
   国际刑警组织的
   <a class="markup--anchor markup--p-anchor" data-href="https://vimeo.com/219362794" href="https://vimeo.com/219362794" rel="nofollow noopener noreferrer" target="_blank">
    宣传视频
   </a>
   称，使用 SiiP 的部门可以上传拦截到的电话，也可以在社交媒体上进行搜索。
   <strong class="markup--strong markup--p-strong">
    SiiP 的数据库将包括来自 YouTube、Facebook、公开录制的对话、网络协议通话技术录音以及其他个人可能完全没有意识到他们的声音正在变成生物识别的样本。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="d286">
   “由于各种原因，人们经常会选择在线上传资料，但我怀疑这项技术是让警察和武器公司将人们独特的生物识别信息登记到秘密数据库中，供世界各地的警方使用”，Privacy International 监视专家 Edin Omanovic 解释说。
  </p>
  <p class="graf graf--p graf-after--p" id="816d">
   人权观察研究员 Cynthia Wong 警告说，广泛的授权可能会导致隐私数据收集的范围不断扩大。“在很多情况下，人们可能会同意为了一个目的将自己的声音录制下来，但肯定会反对将其用于其他用途，包括使用我们的声音来构建和训练大量的语音生物识别数据库和识别系统这种做法”，她说。“或者我们可能根本不同意自己的声音被录制 — 也许我们的声音被秘密录制了、或无意中被出现在录音背景中，但现在已被放置在国际刑警组织的数据库里”。
  </p>
  <p class="graf graf--p graf-after--p" id="013e">
   国际刑警组织的系统似乎代表了迄今为止最大的国际声音收集努力。但正如最近国际刑警调查报告所指出的，
   <strong class="markup--strong markup--p-strong">
    许多执法机构已经悄然使用了语音识别系统。在接受调查的 69 个国家的 91 个部门中，超过半数已经运行了 SiiP 程序。在 2010 年，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://speechpro-usa.com/media/news/2010-06-03" href="https://speechpro-usa.com/media/news/2010-06-03" rel="nofollow noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     墨西哥宣布
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    该国已经创建了世界上第一个全国性的自动语音识别系统，使用俄罗斯的语音技术中心。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="2746">
   The Intercept 一月份
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2018/01/19/voice-recognition-technology-nsa/" href="https://theintercept.com/2018/01/19/voice-recognition-technology-nsa/" rel="nofollow noopener noreferrer" target="_blank">
    报道
   </a>
   显示，国家安全局至少在 2004 年以来一直使用语音识别系统监控政治家、机构雇员、毒贩、间谍或者恐怖分子。国土安全部的新 HART 数据库 — 除了 DNA、疤痕和纹身之外，已经储存了全球人口生物识别数据中的一小部分 — 包括语音数据。
  </p>
  <p class="graf graf--p graf-after--p" id="cdae">
   <strong class="markup--strong markup--p-strong">
    人权观察去年发布了一份
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hrw.org/news/2017/10/22/china-voice-biometric-collection-threatens-privacy" href="https://www.hrw.org/news/2017/10/22/china-voice-biometric-collection-threatens-privacy" rel="nofollow noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     重磅报告
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，揭露了中国最先进的系统，该系统似乎与移动电话网络相结合，并实时识别已知目标人的声音
   </strong>
   。Wong 对此表示，中国的综合监控系统很可能为其他国家的效仿提供先例。
  </p>
  <p class="graf graf--p graf-after--p" id="2611">
   目前还不清楚能有哪些法律可以阻止创建这种国际生物识别数据库，
   <strong class="markup--strong markup--p-strong">
    这些生物识别数据库可以无限期地保留数据。可以监督情报机构的力量很少。
   </strong>
   Wong 说：“刑警组织如何确保其他机构提交的声音样本是合法截获的？根据我们的调查工作，世界各地的许多国家基本不具备对监视行为的有效监督”。
  </p>
  <p class="graf graf--p graf-after--p" id="d270">
   Wong，Omanovic 和其他人也关注那个创建可疑声音“黑名单”的计划。不知道将由谁来审核这个名单，国家行为者将某个人加入这一黑名单所依照的是什么标准，以及如果有人
   <a class="markup--anchor markup--p-anchor" data-href="https://www.scientificamerican.com/article/voice-analysis-should-be-used-with-caution-in-court/" href="https://www.scientificamerican.com/article/voice-analysis-should-be-used-with-caution-in-court/" rel="nofollow noopener noreferrer" target="_blank">
    被错误地加入
   </a>
   了黑名单会发生什么，Omanovic 说。Wong 想知道，在某些政府致力于打压持不同政见者或定罪记者的国家里，这类技术的使用将采用何种保障措施。
  </p>
  <p class="graf graf--p graf-after--p" id="4349">
   国际刑警组织关于研究生物识别伦理学的报告已经承认了这种收集可能具有
   <strong class="markup--strong markup--p-strong">
    巨大的寒蝉效应
   </strong>
   。但是该报告说：“对从未实际查看的音频数据进行处理本身不算入侵，虽然它可能导致未来的入侵。特别是在调查背景下，其目标是为将来的审查确定嫌疑人”。该报告也承认了：“即使没有人查看，这些技术的应用本身也会产生类似真实入侵的后果：这类技术很容易导致人们因担心自己的信息被暴露，从而变成循规蹈矩、变成无原则认同权力的木偶。”
  </p>
  <p class="graf graf--p graf-after--p" id="a3cb">
   <strong class="markup--strong markup--p-strong">
    Verint 是一家跨国生物识别技术公司，一直领导着该项目的开发
   </strong>
   。该公司没有回应评论请求。
  </p>
  <p class="graf graf--p graf-after--p graf--trailing" id="a65c">
   正如国际隐私组织所记录的，
   <strong class="markup--strong markup--p-strong">
    该公司已被证实“在全球范围内销售并安装侵入式和大规模监视系统，包括将这类技术卖给威权政府
   </strong>
   。” Omanovic 说，“此事足见道德意识之欠缺已经到了足够危险的程度。欧盟正在进行没有任何公开辩论的证券化。而最终，军火行业工作者才是终极受益者。”
  </p>
