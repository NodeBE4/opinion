  <p class="graf graf--p">
   我们曾经介绍过《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e6%89%be%e5%88%b0%e6%97%85%e5%ba%97%e6%88%bf%e9%97%b4%e9%87%8c%e9%9a%90%e8%97%8f%e7%9a%84%e6%91%84%e5%83%8f%e5%a4%b4%ef%bc%9f/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e6%89%be%e5%88%b0%e6%97%85%e5%ba%97%e6%88%bf%e9%97%b4%e9%87%8c%e9%9a%90%e8%97%8f%e7%9a%84%e6%91%84%e5%83%8f%e5%a4%b4%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    如何找到旅店房间里隐藏的摄像头？
   </a>
   》；本文是一个专注于将手机作为主要工具的查找摄像头方法。
  </p>
  <p class="graf graf--p">
   因为有可用的应用程序可以帮您做到这点。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    您正在被监视
   </strong>
  </h3>
  <p class="graf graf--p">
   有人正在偷窥你。在后斯诺登时代，这几乎是无可辩驳的。
  </p>
  <p class="graf graf--p">
   多年来，随着人们对大街上的各种监视摄像机等玩意越来越习以为常，它的发展已经足够令人惊讶。
  </p>
  <p class="graf graf--p">
   但是，就在屋里的那种呢？办公室、卧室、甚至洗手间和更衣室中隐藏的监视摄像机……
  </p>
  <p class="graf graf--p">
   它们几乎是最直接的危险。尤其是，制作这种东西非常简单，就算不买专业制造的摄像头，个人也不难鼓捣出一个同等能力的暗器来。Raspberry Pi 就可以做到，甚至旧的智能手机或平板电脑都可以做到。
  </p>
  <p class="graf graf--p">
   智能手机可以检测到隐藏的摄像头。
  </p>
  <p class="graf graf--p">
   通常，使用两种常用方法来实现此目的。
  </p>
  <p class="graf graf--p">
   一种是通过使用智能手机硬件检测电磁场。通过安装单个应用程序，您可以将手机移动到怀疑摄像头隐藏的区域，如果检测到强信号，则可以确定墙壁或物体内藏有摄像头。
  </p>
  <p class="graf graf--p">
   另一种方式是检测镜头反射的光。虽然这种方法不太可靠，但仍然值得拥有这样一个应用程序，即使它只是发现了落在地毯上的小物件。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    使用 Android 或 iOS 查找隐藏的摄像头
   </strong>
  </h3>
  <p class="graf graf--p">
   对于 iPhone，$ 4.99 的隐藏摄像机检测器是最佳选择，是啊它不免费。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Download
   </strong>
   :
   <a class="markup--anchor markup--p-anchor" data-href="https://itunes.apple.com/us/app/hidden-camera-detector/id532882360?mt=8" href="https://itunes.apple.com/us/app/hidden-camera-detector/id532882360?mt=8" rel="noopener noreferrer" target="_blank">
    Hidden Camera Detector
   </a>
   for iOS ($4.99)
  </p>
  <p class="graf graf--p">
   Android 用户也可以考虑使用 Hidden Camera Detector。还可以使用 Glint Finder，效果一样，都是免费的。
  </p>
  <p class="graf graf--p">
   当然还有其他选项可用。例如，使用无线网络的低成本设备很可能会出现在您家附近的Wi-Fi设备列表中。
  </p>
  <p class="graf graf--p">
   如果您有预算，而且被监视的后果非常严重，那么还可以考虑使用价格为130美元的RF检测硬件，它能够读取传统无线摄像头广播的信号。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Download
   </strong>
   :
   <a class="markup--anchor markup--p-anchor" data-href="https://play.google.com/store/apps/details?id=hiddencamdetector.futureapps.com.hiddencamdetector" href="https://play.google.com/store/apps/details?id=hiddencamdetector.futureapps.com.hiddencamdetector" rel="noopener noreferrer" target="_blank">
    Hidden Camera Detector
   </a>
   for Android (Free)
   <br/>
   <strong class="markup--strong markup--p-strong">
    Download
   </strong>
   :
   <a class="markup--anchor markup--p-anchor" data-href="https://play.google.com/store/apps/details?id=com.workshop512.glintfinder" href="https://play.google.com/store/apps/details?id=com.workshop512.glintfinder" rel="noopener noreferrer" target="_blank">
    Glint Finder
   </a>
   for Android (Free)
   <br/>
   <strong class="markup--strong markup--p-strong">
    Download
   </strong>
   : Hidden Camera Detector and Spy Cam Finder for Android (Free)
  </p>
  <p class="graf graf--p">
   无论您选择哪种应用，都可以检测到摄像头和扬声器，甚至可以检测隐藏的计算机。
  </p>
  <p class="graf graf--p">
   它们通常以相同的方式工作：显示与摄像机或其他监视设备的距离，从而使您对其位置有所了解。
  </p>
  <p class="graf graf--p">
   比如使用 Hidden Camera Detector，当手机靠近摄像头时应用程序屏幕上会发红光。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12525 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-21.png?resize=1100%2C916&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-21.png?w=1338&amp;ssl=1 1338w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-21.png?resize=300%2C250&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-21.png?resize=1024%2C853&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-21.png?resize=768%2C639&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-21.png?resize=1100%2C916&amp;ssl=1 1100w" data-recalc-dims="1" height="916" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-21.png?resize=1100%2C916&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12525" data-recalc-dims="1" height="916" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-21.png?resize=1100%2C916&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-21.png?w=1338&amp;ssl=1 1338w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-21.png?resize=300%2C250&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-21.png?resize=1024%2C853&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-21.png?resize=768%2C639&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-21.png?resize=1100%2C916&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   如果监测到其他硬件它也会发光，那么究竟是不是摄像头，请注意中间那个数字，如果检测到的是摄像头，那么这个数字应该超过100。
  </p>
  <p class="graf graf--p">
   Hidden Camera Detector 还具有红外模式。
  </p>
  <p class="graf graf--p">
   请注意，在使用这些应用程序时，首先应确保您了解室内使用了什么技术。电视、计算机、所谓的智能助手（例如 Amazon Echo）和其他硬件可能会形成干扰。
  </p>
  <p class="graf graf--p">
   操作方面需要找到技巧，手机的位置很重要。进行一点练习就能帮您找到合适的角度。
  </p>
  <p class="graf graf--p">
   如果仍然遇到问题，试试把手机保护套取下来。某些外壳的材料可能会阻挡来自其他设备的信号，并破坏手机自身的辐射场。短暂地卸下保护套可以为您带来更快更准确的检测结果。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    找到隐藏的摄像头时该怎么办
   </strong>
  </h3>
  <p class="graf graf--p">
   首先请注意，一旦存在隐藏的摄像头，您寻找它的过程就可能已经被监视者发现了。
  </p>
  <p class="graf graf--p">
   找到后您可以用黑胶布或喷漆之类的东西遮盖它。但您很可能很难 “利用” 它了（利用指的是，故意诱导监视者让他们重视你提供的假消息），因为他们很可能已经看到您在检测了。
  </p>
  <p class="graf graf--p">
   不论摄像头是谁藏在那里的，它本质上都是一种侵犯性的非法的间谍行为。您应该能找到最恰当的方法以维护自己的利益。⚪️
  </p>
