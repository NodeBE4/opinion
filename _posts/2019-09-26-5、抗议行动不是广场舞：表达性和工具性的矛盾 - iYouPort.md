    “少数激进派行动者在敌人的领土上英勇就义，就像孤独的游击战士。也许能让某些人感到光荣，但是无疑，这是一次自杀任务”《

   </li>

    <a href="https://www.iyouport.org/5%E3%80%81%E6%8A%97%E8%AE%AE%E8%A1%8C%E5%8A%A8%E4%B8%8D%E6%98%AF%E5%B9%BF%E5%9C%BA%E8%88%9E%EF%BC%9A%E8%A1%A8%E8%BE%BE%E6%80%A7%E5%92%8C%E5%B7%A5%E5%85%B7%E6%80%A7%E7%9A%84%E7%9F%9B%E7%9B%BE-2/" rel="noopener noreferrer" target="_blank">

   </li>

  <blockquote>
   <p>
    <em>
     <span class="s1">
      “
     </span>
     <span class="s2">
      如果真正的激进派发现长发会给沟通和组织活动设置心理障碍，他们就会减掉头发
     </span>
     <span class="s1">
      ” —— —Saul Alinsky
     </span>
    </em>
   </p>
  </blockquote>
  <p class="p1">
   <span class="s2">
    你看过广场舞吧，一些人在播放着自己的音乐、表演着自己创造的舞姿，他们好像很兴奋，而且旁若无人。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    他们给你的感受是什么？尤其是，当你逛公园是想要一份宁静的时候、或者晚间在自家院子里纳凉的轻松时间，你耳边却是不间断的嘈杂音乐声？
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    <b>
     对广场舞的描述经常伴随着
    </b>
   </span>
   <span class="s1">
    <b>
     “
    </b>
   </span>
   <span class="s2">
    <b>
     烦人
    </b>
   </span>
   <span class="s1">
    <b>
     ”
    </b>
   </span>
   <span class="s2">
    <b>
     这样的形容，而不是他们似乎想要声明的
    </b>
   </span>
   <span class="s1">
    <b>
     “
    </b>
   </span>
   <span class="s2">
    <b>
     健身
    </b>
   </span>
   <span class="s1">
    <b>
     ”
    </b>
   </span>
   <span class="s2">
    <b>
     概念
    </b>
    。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    也许因为它是被少数人定制并要求多数人模仿的？
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    不一定。关于
   </span>
   <span class="s1">
    “
   </span>
   <span class="s2">
    烦人
   </span>
   <span class="s1">
    ”
   </span>
   <span class="s2">
    还有另一种类似的形式，非洲手鼓聚会。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    这种手鼓聚会是现场演绎的，没有定制。你与朋友甚至陌生人一起在公园里坐下来，一起制作节奏，直观地交流，每个人都可以添加自己的节奏，最终形成宏大和美丽的声音。听起来很棒是吧。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    不过它依旧被描述为烦人。
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    路人经过这里并没有期待听到一片激烈的鼓声。
    <b>
     虽然表演者看起来非常投入，他们拼命地敲打着羊皮鼓面，他们被自己的表达欲望所笼罩着，就如广场舞一样，旁若无人
    </b>
    。
   </span>
  </p>
  <p class="p1">
   <span class="s4">
    ⚠️
   </span>
   <span class="s2">
    成为表演者的一部分是一回事，从外面体验它，则是另一回事。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    这是一个非常严肃的思考。
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    <b>
     很多时候，街头抗议运动被搞得就像一个非洲手鼓聚会
    </b>
    。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    从外面看，它可能是痛苦的、缺乏想象力的、自我中心主义的，而且很烦人。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    然而，对于参与创建行动的人来说，体验可以是有益的和变革性的
   </span>
   <span class="s1">
    ——
   </span>
   <span class="s2">
    即使其他人都感到困惑或烦恼。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    <b>
     如果这种情况发生而你没有察觉
    </b>
   </span>
   <span class="s1">
    <b>
     ——
    </b>
   </span>
   <span class="s2">
    <b>
     就如跳广场舞的大妈，那将意味着你已经很容易成为政治认同悖论的牺牲品
    </b>
    。（《自杀不是明智之举：警惕政治认同悖论》）
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    根源问题在哪儿？
   </span>
  </p>
  <p class="p1">
   <span class="s4">
    ⚠️
   </span>
   <span class="s2">
    因为你沉浸于表演
   </span>
   <span class="s1">
    /
   </span>
   <span class="s2">
    表达
   </span>
   <span class="s1">
    /
   </span>
   <span class="s2">
    表现，而不是作为目标的
   </span>
   <span class="s1">
    *
   </span>
   <span class="s2">
    改变
   </span>
   <span class="s1">
    *
   </span>
   <span class="s2">
    。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    有时，积极人士会直接采取行动而不会过多地考虑别人如何接受它，或者行动将达到什么样的目标。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    许多人参与行动是因为行动对他们自己有意义，或者仅仅因为做正确的事感觉很好。我们称之为
   </span>
   <span class="s1">
    *
   </span>
   <span class="s2">
    行动的表达部分
   </span>
   <span class="s1">
    *
   </span>
   <span class="s2">
    。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    <b>
     表达性行动来自内心和直觉
    </b>
   </span>
   <span class="s1">
    <b>
     ——
    </b>
   </span>
   <span class="s2">
    <b>
     无论头脑中是否计算出具体结果
    </b>
    。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    游行期间
   </span>
   <span class="s1">
    “
   </span>
   <span class="s2">
    走街
   </span>
   <span class="s1">
    ”
   </span>
   <span class="s2">
    就是一个很好的例子。当然，在街上游行的感觉的确是不错的。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    你和你的同伴勇敢地违反了警察的命令，一起走进了行车道。你几乎可以闻到空气中飘逸的团队凝聚力。令人陶醉。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    而就更广泛的社会运动目标而言，这些通常是无关紧要的。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    不过，你听说过游行是
   </span>
   <span class="s1">
    “
   </span>
   <span class="s2">
    糟糕的
   </span>
   <span class="s1">
    ”
   </span>
   <span class="s2">
    仅仅是因为人们留在了人行道上吗？当有人这么说时，
   </span>
   <span class="s4">
    ⚠️
   </span>
   <span class="s2">
    可能是因为
    <b>
     他们的目标主要是表达；影响社会变革是次要的
    </b>
    。
   </span>
  </p>
  <p class="p1">
   <span class="s4">
    💡
   </span>
   <span class="s2">
    而大多数训练有素的组织者都会在另一个层面上思考：无论参与者的自我表达价值如何，我们都会问：
   </span>
   <span class="s1">
    “
   </span>
   <span class="s2">
    这个行动实际上对我们的问题、原因、行动或竞选活动有什么影响？
   </span>
   <span class="s1">
    ”
   </span>
   <span class="s2">
    我们称之为
   </span>
   <span class="s1">
    *
   </span>
   <span class="s2">
    行动的工具价值
   </span>
   <span class="s1">
    *
   </span>
   <span class="s2">
    。
   </span>
  </p>
  <p>

   </noscript>
  </p>
  <p class="p1">
   <span class="s2">
    这两个方面其实都很重要，虽然精心设计的行动可以同时实现两者，但是，
   </span>
   <span class="s4">
    ⚠️
   </span>
   <span class="s2">
    表
    <b>
     达性和工具性经常是相互对立的
    </b>
    。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    许多强硬的组织者专注于有形的影响，而忘记了行动的自我表达维度在肯定价值观和建立群体身份方面发挥着关键作用。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    另一方面，许多团体可以
    <b>
     在没有赢得任何东西的情况下
    </b>
    进行一连串的表达性行动。
   </span>
  </p>
  <p class="p1">
   <span class="s4">
    ⚠️
   </span>
   <span class="s2">
    这里的危险很明显：那些没有根据工具性目标评估其战术成功度的团体，可能会变得自恋和自我指责。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    <b>
     这就是你经常能在中国看到的
    </b>
    。异议人士和活动家至少花费一半精力去批评周边人
   </span>
   <span class="s1">
    “
   </span>
   <span class="s2">
    为什么这么傻
   </span>
   <span class="s1">
    /
   </span>
   <span class="s2">
    为什么不懂我
   </span>
   <span class="s1">
    /
   </span>
   <span class="s2">
    为什么你们还有脸岁月静好？
   </span>
   <span class="s1">
    ”……
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    他们的思路陷入了无关紧要的境地，因为他们
    <b>
     没有
    </b>
    将行动调整到可影响团队之外的任何群体的层面上。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    虽然工具性行动通常集中在
   </span>
   <span class="s1">
    “
   </span>
   <span class="s2">
    外部
   </span>
   <span class="s1">
    ”
   </span>
   <span class="s2">
    结果上，比如说，你可以对行动定位的目标（政客、财团）施加一些可衡量的压力，但是，它们也可以有一个
   </span>
   <span class="s1">
    “
   </span>
   <span class="s2">
    内部
   </span>
   <span class="s1">
    ”
   </span>
   <span class="s2">
    的焦点。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    考虑一种大众教学，它旨在建立您的组织的能力、或提高参与者的技能、或改变您的运动思维。在这里，
   </span>
   <span class="s4">
    💡
   </span>
   <span class="s2">
    行动的表达价值挖去可以直接转化为工具性结果。
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    因此，
    <b>
     表达性和工具性不会是相互排斥的，而是我们需要注意的动态
    </b>
    。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    工具性行为可以进一步细分为
   </span>
   <span class="s1">
    “
   </span>
   <span class="s2">
    交际
   </span>
   <span class="s1">
    ”
   </span>
   <span class="s2">
    和
   </span>
   <span class="s1">
    “
   </span>
   <span class="s2">
    实际
   </span>
   <span class="s1">
    ”
   </span>
   <span class="s2">
    ；交际或象征性行为旨在影响观点、表达观点或促进公共语话；而实际行为则旨在对目标产生实际影响。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    这是衡量工具性结果的两种不同方式。
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    <b>
     虽然自我表达是社会变革过程的必要组成部分，但仅仅它肯定是不够
    </b>
    的。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    通过我们的自我表达仪式，我们肯定了自己的价值观和愿景，并建立了一种群体身份和凝聚力，没有这种身份和凝聚力，我们就会变得过于软弱无序，无法改变世界。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    也就是说，表达价值与参与社会和实现系统性变革并不相同。
   </span>
  </p>
  <p class="p1">
   <span class="s4">
    ⚠️
   </span>
   <span class="s2">
    如果真的想要改变这个世界，就必须知道
   </span>
   <span class="s1">
    ——
   </span>
   <span class="s2">
    并且必需巧妙地平衡
   </span>
   <span class="s1">
    ——
   </span>
   <span class="s2">
    工具性目标与我们对自我表达的渴望之间的区别。
   </span>
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5184 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-21.jpg?resize=1100%2C733&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-21.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-21.jpg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-21.jpg?resize=768%2C512&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-21.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-21.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-21.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="733" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-21.jpg?resize=1100%2C733&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5184" data-recalc-dims="1" height="733" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-21.jpg?resize=1100%2C733&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-21.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-21.jpg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-21.jpg?resize=768%2C512&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-21.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-21.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-21.jpg?resize=272%2C182&amp;ssl=1 272w" width="1100"/>
   </noscript>
  </p>
  <p class="p1">
   <span class="s2">
    请注意，对于任何给定的策略来说
   </span>
   <span class="s1">
    ——
   </span>
   <span class="s2">
    比如
   </span>
   <span class="s2">
    封锁
   </span>
   <span class="s1">
    ——
   </span>
   <span class="s2">
    都可以是交际的和实际的；其中一个并不比另一个更好。
    <b>
     诀窍是要理解差异
    </b>
    。
   </span>
   <span class="s4">
    ⚠️
   </span>
   <span class="s2">
    你不应该指望一个主要是象征性的行为能产生实际性的结果，反之亦然。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    一种策略在某种程度上肯定是具体的，它旨在实现一个特定的、可量化的目标。例如，反战组织者可能会试图封锁一个港口以防止一批武器通过。它有一个特定的目标，港口和使用它的公司的实际成本，以及评估成功的方法：阻止武器运输，或者阻止失败。
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    <b>
     当一种策略旨在传达政治立场、价值观或世界观时，它就是交际性的
    </b>
    ；应对不公正的大规模游行可以属于这一类。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    交际性策略可以用来激励我们的基础、建立联盟网络、寻求动摇公众舆论、或吓唬目标，但是，它往往没有具体的、可衡量的、可实现的、现实的、有时限的目标。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    成功需要更具质量。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    要取得成功，具体的策略必须迫使目标做出回应。而交际性策略可能有目标，但也可以完全没有目标。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    虽然某些行为既可以是交际性的，也可以是实际性的，但理解这些差异非常重要。
   </span>
  </p>
  <p class="p1">
   <span class="s4">
    ⚠️
   </span>
   <span class="s2">
    如果你参与的是交际性的行动却期望得到具体实际的结果，那只能感到气馁。
   </span>
  </p>
  <p class="p1">
   <span class="s4">
    💡
   </span>
   <span class="s2">
    最好从一开始就清楚它是什么样的行动，这样每个人都能知道该如何衡量价值
   </span>
   <span class="s1">
    ——
   </span>
   <span class="s2">
    并有助于行动的影响。
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    同样重要的是要记住，
   </span>
   <span class="s1">
    <b>
     “
    </b>
   </span>
   <span class="s3">
    <b>
     实际性
    </b>
   </span>
   <span class="s1">
    <b>
     ”
    </b>
   </span>
   <span class="s3">
    <b>
     和
    </b>
   </span>
   <span class="s1">
    <b>
     “
    </b>
   </span>
   <span class="s3">
    <b>
     交际性
    </b>
   </span>
   <span class="s1">
    <b>
     ”
    </b>
   </span>
   <span class="s3">
    <b>
     是衡量一个行动的工具结果的方法，而不是它的表现维度。
    </b>
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    您的行动的表达部分侧重于参与者的自我表达，而行动的工具性结果则关注您的行动更直接的目标导向影响（交际和实际都如此）。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    举个例子以分析这两者。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    想象一下，一个旨在封锁高盛银行办公室入口的经济正义组织。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    在行动计划会议上，由于对行动是否具有交际性或实际性缺乏明确分析，因此，讨论没有成效。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    有些人希望通过简单的人体封锁来拒绝武器的使用，而其他人则希望通过使用链条和其他
   </span>
   <span class="s1">
    “
   </span>
   <span class="s2">
    硬装备
   </span>
   <span class="s1">
    ”
   </span>
   <span class="s2">
    来提高赌注。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    使用装备具有保持能力的好处（警察更难以移除你），但是，它会带来更大的风险并且更难以部署。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    很明显，该小组
    <b>
     没有时间去仔细调研
    </b>
    以策划如何封锁住每一个出口。因此，如果该行动被认为实际性的（试图关闭高盛），那么它将失败，因为它无法实现现实的工具性结果。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    然而，如果它是交际性的
   </span>
   <span class="s1">
    ——
   </span>
   <span class="s2">
    一种放大信息的象征性行为
   </span>
   <span class="s1">
    ——
   </span>
   <span class="s2">
    这样定位它可能就会是成功的。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    通过为参与者提供可宣泄的、变革性的体验来建立参与者的决心、联系和承诺，交际性行动可能具有强大的表达性结果。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    当参与者同意进行交际性行动时，就不再需要武装以获得持久力：在此延长时间没有战术优势。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    相反，该组织决定采用人体封锁，这在媒体上的发挥会更好（
    <b>
     媒体效应是表达性行为取得成功的一个重要指标
    </b>
    ）。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    如果积极人士们没有评估他们行动的目的并理解这一目标（特别是他们的目标是交际性的而不是实际性的程度），他们能做出的战略选择就会非常少。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    现在您明白了，为什么香港的六四纪念游行会在活动家内部产生矛盾？其中有些人认为已经是鸡肋，因而停止了参与。正是因为没能清楚地阐述其定位：它是表达性的，而非实际性。如果您的期待是实际的变革，那将是一个具有伤害性的误会。
   </span>
   <span class="s4">
    ⚪️
   </span>
  </p>
