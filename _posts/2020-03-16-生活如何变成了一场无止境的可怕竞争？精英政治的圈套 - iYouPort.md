      <strong>
       精英政治奖励成就高于一切，它使所有人都陷入悲惨 —— 包括富人。究竟有没有出路？
      </strong>
     </em>
    </span>
   </li>

  <p>
   <img alt="" class="aligncenter size-full wp-image-10211 jetpack-lazy-image" data-lazy-sizes="(max-width: 720px) 100vw, 720px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/头-12.jpg?resize=720%2C405&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/头-12.jpg?w=720&amp;ssl=1 720w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/头-12.jpg?resize=300%2C169&amp;ssl=1 300w" data-recalc-dims="1" height="405" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/头-12.jpg?resize=720%2C405&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="720"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10211" data-recalc-dims="1" height="405" sizes="(max-width: 720px) 100vw, 720px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/头-12.jpg?resize=720%2C405&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/头-12.jpg?w=720&amp;ssl=1 720w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/头-12.jpg?resize=300%2C169&amp;ssl=1 300w" width="720"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   1987年夏天，我从德克萨斯州奥斯丁市的一所公立高中毕业，不久后离开家前往东北方向的耶鲁大学报道。
  </p>
  <p class="graf graf--p">
   接下来，我花了将近15年时间辗转多所大学念书 — — 先是伦敦经济学院、接着是牛津大学、哈佛大学，最后是耶鲁法学院，一路走来攒了一大摞学位证书。
  </p>
  <p class="graf graf--p">
   今天我在耶鲁法学院教书，这里的学生和我年轻时一模一样，相似度令人头皮发麻：
   <strong class="markup--strong markup--p-strong">
    他们无一例外都是精英家长和一流大学的产物。
   </strong>
  </p>
  <p class="graf graf--p">
   我传授给他们的东西是一种社会优势，而这份优势正是多年前我的老师们授予我的。
  </p>
  <p class="graf graf--p">
   我的老师、我的学生和我自己，都应该把自身的经济优越与社会地位优势归功于精英制度。
  </p>
  <p class="graf graf--p">
   在20多年前，当我刚刚开始写关于经济不平等问题的文章时，
   <strong class="markup--strong markup--p-strong">
    社会精英制度似乎更像是一种解决方案，而非问题根源。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    早期提倡施行精英制度的学者，认为这将有助于提高社会阶层的流动性。
   </strong>
  </p>
  <p class="graf graf--p">
   比如在上世纪60年代，时任耶鲁大学校长的 Kingman Brewster 就
   <a class="markup--anchor markup--p-anchor" data-href="http://archives.yalealumnimagazine.com/issues/99_12/admissions.html" href="http://archives.yalealumnimagazine.com/issues/99_12/admissions.html" rel="noopener noreferrer" target="_blank">
    率先采用了
    <strong class="markup--strong markup--p-strong">
     精英准入原则
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，此举的初衷是杜绝精英阶层的世袭可能性。
   </strong>
  </p>
  <p class="graf graf--p">
   在当时，耶鲁大学的校友们都信心满满，认定自己的儿子一生下来就有了入学耶路的权利，但新的入学政策看重的是学生的成就而非血统，这就让那些真正有潜力的学生有了入学机会。
  </p>
  <p class="graf graf--p">
   在相当长的一段时期内，社会精英制度把那些有天赋又肯吃苦的新人引流到各个圈层，替代了那些曾经心高气傲的圈内人士。
  </p>
  <p class="graf graf--p">
   时至今日，精英阶层仍然宣称自己是用才华和汗水换来的进身之阶，这些手段对任何人都是公开、公平的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    但事实上，今天的精英阶层已经把任何不符合狭义精英标准的人排除在圈层之外了。
   </strong>
  </p>
  <p class="graf graf--p">
   哈佛大学、普林斯顿大学、斯坦福大学和耶鲁大学都出现了同样的招生现象，
   <strong class="markup--strong markup--p-strong">
    从那些家庭收入水平位列金字塔顶端1%的少数家庭中招收更多的学生，而且人数
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://www.equality-of-opportunity.org/papers/coll_mrc_paper.pdf" href="http://www.equality-of-opportunity.org/papers/coll_mrc_paper.pdf" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     远远超过了从收入处于金字塔底部60%
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    的大多数家庭中招收的学生人数。
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="http://www.equality-of-opportunity.org/papers/coll_mrc_paper.pdf" href="http://www.equality-of-opportunity.org/papers/coll_mrc_paper.pdf" rel="noopener noreferrer" target="_blank">
     Mobility Report Cards: The Role of Colleges in Intergenerational Mobility
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10212 jetpack-lazy-image" data-lazy-sizes="(max-width: 658px) 100vw, 658px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-8.jpeg?resize=658%2C998&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-8.jpeg?w=658&amp;ssl=1 658w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-8.jpeg?resize=198%2C300&amp;ssl=1 198w" data-recalc-dims="1" height="998" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-8.jpeg?resize=658%2C998&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="658"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10212" data-recalc-dims="1" height="998" sizes="(max-width: 658px) 100vw, 658px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-8.jpeg?resize=658%2C998&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-8.jpeg?w=658&amp;ssl=1 658w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-8.jpeg?resize=198%2C300&amp;ssl=1 198w" width="658"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    遗产优先权、任人唯亲、营私舞弊等等贪腐手段继续为那些有钱人家的入学申请者大开方便之门。
   </strong>
  </p>
  <p class="graf graf--p">
   之所以权利再度向财富倾斜，其主要原因仍然与精英制度有关
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/news/wonk/wp/2014/03/05/these-four-charts-show-how-the-sat-favors-the-rich-educated-families/?utm_term=.e3fb1373d3c0" href="https://www.washingtonpost.com/news/wonk/wp/2014/03/05/these-four-charts-show-how-the-sat-favors-the-rich-educated-families/?utm_term=.e3fb1373d3c0" rel="noopener noreferrer" target="_blank">
    根据美国大学理事会（The College Board）的统计
   </a>
   ，如果对比父母年收入超过20万美元的家庭以及年收入只有4万~6万美元家庭，富有家庭子女的学术能力评估测试（SAT）平均分数，比贫穷家庭子女的平均分数高250分。
  </p>
  <p class="graf graf--p">
   对于那些年收入排名在倒数三分之一区间的家庭，每200名考生只有1人的SAT分数可以达到耶鲁大学录取分数的中间值。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    与此同时，那些顶级银行、律师事务所以及那些高薪企业几乎只招收精英大学的毕业生。
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://secure-media.collegeboard.org/digitalServices/pdf/research/2013/TotalGroup-2013.pdf" href="https://secure-media.collegeboard.org/digitalServices/pdf/research/2013/TotalGroup-2013.pdf" rel="noopener noreferrer" target="_blank">
     Total Group Mean SAT Scores
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   根据另一项研究，
   <strong class="markup--strong markup--p-strong">
    吃苦耐劳的圈外人已经享受不到真正的机会了：
   </strong>
   如果把家庭收入水平等分为5个等级，
   <strong class="markup--strong markup--p-strong">
    最贫穷的第五级阶层中，每100个孩子之中只有1人能最终获得成功，跻身到收入水平前5%的金字塔顶端。
   </strong>
  </p>
  <p class="graf graf--p">
   相比较而言，对于那些收入等级在中间的第三级家庭，每50个孩子之中几乎就有1人能攀登到收入金字塔顶峰。
  </p>
  <p class="graf graf--p">
   另外，
   <a class="markup--anchor markup--p-anchor" data-href="http://www.equality-of-opportunity.org/papers/abs_mobility_paper.pdf" href="http://www.equality-of-opportunity.org/papers/abs_mobility_paper.pdf" rel="noopener noreferrer" target="_blank">
    绝对经济流动性也在下降
   </a>
   ，自上世纪中叶至今，中产阶级家庭子女在收入能力方面超过父母的可能性下降了一半还多，在这一点上，中产阶级家庭的下滑比贫困家庭更严重。
  </p>
  <p class="graf graf--p">
   对这些现象，精英制度的回应是将其定义为一种失败，或者某种无法达标的结果，其本质就是在经济损失的现象上涂抹了一层道德羞辱。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="http://www.equality-of-opportunity.org/papers/abs_mobility_paper.pdf" href="http://www.equality-of-opportunity.org/papers/abs_mobility_paper.pdf" rel="noopener noreferrer" target="_blank">
     THE FADING AMERICAN DREAM:
     <br/>
     TRENDS IN ABSOLUTE INCOME MOBILITY SINCE 1940
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   经济不平等往往引发愤怒情绪，而大众习惯于把矛头对准精英制度建立起来的社会体系。
  </p>
  <p class="graf graf--p">
   皮尤研究中心通过
   <a class="markup--anchor markup--p-anchor" data-href="https://www.people-press.org/2017/07/10/sharp-partisan-divisions-in-views-of-national-institutions/" href="https://www.people-press.org/2017/07/10/sharp-partisan-divisions-in-views-of-national-institutions/" rel="noopener noreferrer" target="_blank">
    调查发现
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    将近五分之三的共和党人认为如今的大学和专科学院对美国有害。
   </strong>
  </p>
  <p class="graf graf--p">
   今年年初，美国多家大学被爆出招生丑闻，自此，普遍存在的强烈公愤进一步恶化，转变为一种深刻的憎恨情绪。
  </p>
  <p class="graf graf--p">
   公众当然有理由感到愤怒，但如今这种愤恨也是极度扭曲的。
  </p>
  <p class="graf graf--p">
   公众当然可以对任人唯亲感到愤怒，也可以对其他丑陋的精英阶层特权感到愤怒，然而正是这种愤怒无形中抬高了精英理想体系的价值。
  </p>
  <p class="graf graf--p">
   当然，精英制度本身才是更大的问题，因为它正在瓦解美国梦 — — 在精英制度创造的竞争中，就算所有人都守规矩，最后的赢家仍然是富人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    但是，精英阶层到底赢得了什么呢？如今，精英制度的受益者也在为自身的需求付出惨痛代价。
   </strong>
  </p>
  <p class="graf graf--p">
   正如那些设法跻身上流社会的局外人都必须高强度地工作，富人们为了在圈层之内栖身不得不通过极端昂贵的教育支出获得相应回报，精英制度无情地让富人落入这个圈套，正如它无情地把其他人排挤到精英阶层以外。
  </p>
  <p class="graf graf--p">
   不是说让你为了富人的遭遇掉眼泪，但富人正在遭受精英制度带来的伤害，这不仅是个事实，而且也很能说明问题。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    我们总是习惯性地认为缩减不平等就意味着削减富人的利益，但如今的情形并非如此 — — 由于造成了不平等结果的精英制度并没有真正服务于任何人，
   </strong>
   因此，逃离精英体系的陷阱将最终让所有人受益。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10213 jetpack-lazy-image" data-lazy-sizes="(max-width: 672px) 100vw, 672px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-7.jpg?resize=672%2C871&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-7.jpg?w=672&amp;ssl=1 672w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-7.jpg?resize=231%2C300&amp;ssl=1 231w" data-recalc-dims="1" height="871" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-7.jpg?resize=672%2C871&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="672"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10213" data-recalc-dims="1" height="871" sizes="(max-width: 672px) 100vw, 672px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-7.jpg?resize=672%2C871&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-7.jpg?w=672&amp;ssl=1 672w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-7.jpg?resize=231%2C300&amp;ssl=1 231w" width="672"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   对于精英阶层来说，他们早在幼儿阶段就开始面对精英制度带来的压力。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    父母并不会让孩子在幼年时接受以实验、游戏为主的教育，而是过早地让孩子接受那些重复训练、积累人脉的教育，这些都是进入精英大学并获得精英工作的必要条件。
   </strong>
  </p>
  <p class="graf graf--p">
   尽管父母有时也很不情愿，但终究还是感到自己别无选择。
  </p>
  <p class="graf graf--p">
   在纽约、波士顿、旧金山等城市，
   <strong class="markup--strong markup--p-strong">
    那些收入颇丰的父母普遍要替孩子向10家幼儿园投递入学申请，还要准备应付一连串的作文考试、评估测试还有面试 — — 别忘了，所有这些测试都是为了对年仅4岁的儿童进行评估。
   </strong>
  </p>
  <p class="graf graf--p">
   当然，申请入学精英小学、中学和高中时还会一遍遍重复这一切。
  </p>
  <p class="graf graf--p">
   即使进入这些学府气氛也并不轻松，贵族子弟曾经在这里享受着特权自我陶醉，如今，精英阶层的后代不得不谨慎计算着自己的未来。
  </p>
  <p class="graf graf--p">
   这里的孩子盘算着计划和日程，这里的舞台上演着精心编制的才艺表演，
   <strong class="markup--strong markup--p-strong">
    这里的生活重复着一个熟悉的节奏，总有野心、偶尔期待、很快焦虑。
   </strong>
  </p>
  <p class="graf graf--p">
   精英学校也鼓励学生们顺应这样的节奏，比如，在美国东北部的一所精英小学里，老师提出了一种名叫“每日问题”的任务，学生们必须在放学回家之前解决这个问题，然而老师却没有个这项任务单独安排时间。
  </p>
  <p class="graf graf--p">
   而这项练习的意义，就是训练这些五年级的小学生学会牺牲休息，挤出更多时间处理工作事项。
  </p>
  <p class="graf graf--p">
   这类硬性要求的确付出了代价，今天美国的精英中学、精英高中通常要求学生在晚上花费3到5个小时写作业，对此，美国疾病控制和预防中心（CDC）的流行病学家曾经提出过警告，认为课业负担将导致睡眠不足。
  </p>
  <p class="graf graf--p">
   更可怕的是，在全美范围内，富人阶层的孩子们比贫穷的同龄人有着更高的吸毒、酗酒概率；
   <strong class="markup--strong markup--p-strong">
    他们患上抑郁症、焦虑症的概率高达同龄人的3倍多
   </strong>
   。
  </p>
  <p class="graf graf--p">
   2015年，圣路易斯大学医学院的儿科医生、斯图亚特·斯莱文教授对硅谷欧文顿高中的2100名学生进行了不记名调查，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2016/01/03/opinion/sunday/is-the-drive-for-success-making-our-children-sick.html" href="https://www.nytimes.com/2016/01/03/opinion/sunday/is-the-drive-for-success-making-our-children-sick.html" rel="noopener noreferrer" target="_blank">
    结果令人震惊
   </a>
   ：
  </p>
  <p class="graf graf--p">
   54%的学生表现出中度到重度的抑郁症症状，不仅如此，
   <strong class="markup--strong markup--p-strong">
    还有高达80%的学生表现出中度到重度的焦虑症症状。
   </strong>
  </p>
  <p class="graf graf--p">
   尽管如此，这些学生们仍然有不少理由继续逼迫自己沿着这条路走下去。因为在几十年前，这所高中毕业生的精英大学入学率曾经是30%，如今已经降至10%甚至更低。
  </p>
  <p class="graf graf--p">
   某些特定大学的入学率骤降趋势尤其明显：就在1995年，该高中毕业生申请芝加哥大学的入学率曾经高达71%，到了2019年，这个数字连6%都不到
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   步入职场后竞争只会愈演愈烈，因为在职场中只有不断竞争才有可能抓住一个超越其他精英的机会。
  </p>
  <p class="graf graf--p">
   但问题是，
   <strong class="markup--strong markup--p-strong">
    当一个人的财富和地位只来自其人脉关系，那么此人在选择工作和事业的时候就不可能从自身的兴趣和热情出发。
   </strong>
  </p>
  <p class="graf graf--p">
   不仅如此，如果此人还希望自己的收入足够支持孩子接受精英教育，并通过孩子的教育进一步巩固自己在精英阶层的地位，
   <strong class="markup--strong markup--p-strong">
    此人必须把工作视为一种人脉资源的变现渠道
   </strong>
   ，而且只能在一个窄小的范围内选择金融、管理、法律和医疗相关工作，并且把自己的全部精力投入到这份工作之中。
  </p>
  <p class="graf graf--p">
   于是一件奇怪的事情发生了：
   <strong class="markup--strong markup--p-strong">
    尽管这些当代贵族本人都自认为生活在一个条件优渥的安逸阶层，但如今的精英们正顶着前所未有的强度拼命工作。
   </strong>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10214 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-8.jpg?resize=1100%2C746&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-8.jpg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-8.jpg?resize=300%2C203&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-8.jpg?resize=768%2C521&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-8.jpg?resize=1024%2C694&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-8.jpg?resize=1100%2C746&amp;ssl=1 1100w" data-recalc-dims="1" height="746" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-8.jpg?resize=1100%2C746&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10214" data-recalc-dims="1" height="746" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-8.jpg?resize=1100%2C746&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-8.jpg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-8.jpg?resize=300%2C203&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-8.jpg?resize=768%2C521&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-8.jpg?resize=1024%2C694&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-8.jpg?resize=1100%2C746&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   1962年，当时的精英律师们收入可能只有如今同行的三分之一，但是美国律师协会骄傲地宣布，美国的普通律师“平均每年的收费工作时长为1300小时。”
  </p>
  <p class="graf graf--p">
   相比之下，到了2000年，一家大型律师事务所曾经同样信心满满地宣布，
   <strong class="markup--strong markup--p-strong">
    公司旗下的律师每年的带薪工作时长高达2400小时
   </strong>
   。
  </p>
  <p class="graf graf--p">
   不过这一次，他们还加上了不少委婉的定语修饰：“如果时间管理得当的话”、“这并不是不合理的”、“投入这样的时间精力才有望找到合伙人开展自己的事业”。
  </p>
  <p class="graf graf--p">
   然而，由于律师的工作性质，并不是投入的所有工时都可以算作计费工时，因此所谓的每年2400小时的带薪工作也就意味着公司可以轻易要求员工从早8点工作到晚8点，每周工作6天，全年无休假、无病假、无公休。
  </p>
  <p class="graf graf--p">
   在金融行业情况也是如此，比如“银行家工时”（Bankers’ Hours）这个说法，最早出现于十九世纪，指的是银行的营业时间，也就是一天中的上午10点至下午3点，直到二十世纪中叶银行的营业时间都没有任何变化，因此，在上世纪这个词就变成一个戏称，指代任何轻松的工作。
  </p>
  <p class="graf graf--p">
   如今，这个词仍然广为流传，但其含义变得更讽刺了，如今这个词还有另一种说法，是“银行家的9至5工时”（Bankers’ 9-to-5），指的是从上午9点开始工作，直到第二天的凌晨5点下班。
  </p>
  <p class="graf graf--p">
   再说说那些精英管理层。在上世纪50年代，他们曾经被称为“组织的化身”（Organization Men），意思是企业中被等级制度保护着的高层人士，其资历的重要性远远高于绩效。
  </p>
  <p class="graf graf--p">
   如今，一个人在企业中爬得越高，雇主也就越是要求此人吃苦耐劳。比如亚马逊公司提出的“领导原则”，要求经理们保持“坚持不懈的高标准”，并且时刻做到“交付成果”。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2015/08/16/technology/inside-amazon-wrestling-big-ideas-in-a-bruising-workplace.html" href="https://www.nytimes.com/2015/08/16/technology/inside-amazon-wrestling-big-ideas-in-a-bruising-workplace.html" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     该公司告诫那些管理层的经
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    理们：当你们在工作中“撞到了一面墙”时，唯一的解决方案就是“爬到墙上去”。
   </strong>
  </p>
  <p class="graf graf--p">
   在一项2005年的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hup.harvard.edu/catalog.php?isbn=9780674018396" href="https://www.hup.harvard.edu/catalog.php?isbn=9780674018396" rel="noopener noreferrer" target="_blank">
    调查中
   </a>
   ，那些每周工作60小时以上的美国人表示，他们希望平均每周可以减少25小时的工时。
  </p>
  <p class="graf graf--p">
   他们认为工作让自己陷入了一种“时间饥荒”，是工作阻碍了他们与配偶和孩子建立牢固的亲密关系，另外，
   <strong class="markup--strong markup--p-strong">
    工作并不利于他们维系家庭，甚至让他们的性生活不尽如人意。
   </strong>
  </p>
  <p class="graf graf--p">
   在最近一次哈佛商学院的调查中，一位高管回答问题时自豪地表示：“我每晚和孩子共处的10分钟非常有意义，与其把这10分钟花在工作上，花在孩子身上能发挥出100万倍的价值。”
   <strong class="markup--strong markup--p-strong">
    等等……每天竟然只有10分钟！
   </strong>
  </p>
  <p class="graf graf--p">
   如何优雅地承受如此巨大的工时压力？或者，至少做到面不改色地迎接漫长的工时？这样的问题已经成为了精英阶层的成功标准。
  </p>
  <p class="graf graf--p">
   社会学家Arlie Russell Hochschild 在她的著作《时间绑定》（
   <a class="markup--anchor markup--p-anchor" data-href="https://us.macmillan.com/books/9780805066432" href="https://us.macmillan.com/books/9780805066432" rel="noopener noreferrer" target="_blank">
    <em class="markup--em markup--p-em">
     The Time Bind
    </em>
   </a>
   ）中有一段面对某大型企业高管的采访，根据这位高管的观察，那些既有野心又能够证明自身能力和奉献精神的经理们，最终还是会面临一次“终极淘汰”：“有些人会把自己燃尽，因为他们一直在工作所以变得越来越奇怪……
   <strong class="markup--strong markup--p-strong">
    然而那些最顶部的人非常聪明，他们既能保持疯狂地工作，也能保护自己不会燃烧殆尽，他们仍然能保持良好的精神状态，仍然能维系家庭生活 — — 只有这样的人最终赢了比赛。”
   </strong>
  </p>
  <p class="graf graf--p">
   当一个人只能从自身的人脉资源中获得收入和社会地位，反过来说，也恰恰是被囚禁于那个关系网，直到把自己耗尽 — — 对于那些精英学生，越是害怕失败，就越是渴求传统意义上的成功标志，哪怕他们很清楚那只是个符号，也开玩笑嘲笑那些东西是孩子才喜欢的“五角星”、“闪亮的小装饰”；而精英体系中的员工将越来越清晰地认识到，无论是真正的热情还是生命的意义感，想要通过工作找到这些东西都无限接近于不可能。
  </p>
  <p class="graf graf--p">
   就这样，
   <strong class="markup--strong markup--p-strong">
    精英体系把一代人拖入陷阱之中，既害怕从社会阶梯上跌落，又被虚假的野心隐隐怂恿，就好象一头永远饥饿的野兽，从未找到真正的食物，也从不清楚什么才是真正的食物。
   </strong>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10215 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-4.jpg?resize=1100%2C733&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-4.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-4.jpg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-4.jpg?resize=768%2C512&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-4.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-4.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-4.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="733" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-4.jpg?resize=1100%2C733&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   精英阶层不应该也无权祈求同情，那些被排除在阶层特权与利益高楼之外的人不会对他们产生任何同情。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    当前精英体系统治之下富人同样是受压迫的一方，如果忽视这一点必将是个重大失误。
   </strong>
  </p>
  <p class="graf graf--p">
   主导着当今社会的富人并不轻松，而是正在拼尽全力。
  </p>
  <p class="graf graf--p">
   我们熟悉过去那个贵族时代的社会不平等，也成功打破了当时的社会不平等，但是我们所熟悉的那些情形并不适用于今天的这个经济体系，如今这个不平等的体系恰恰建立在奖励努力与技能的基础之上。
  </p>
  <p class="graf graf--p">
   其结果就是，
   <strong class="markup--strong markup--p-strong">
    一个每周工作上百个小时的银行家只能换来人们的信任，相信自己不是个赚取民脂民膏的投机倒把主义者。
   </strong>
  </p>
  <p class="graf graf--p">
   如此说来，如今富人付出的努力实际上也没有得到相应的回报。
  </p>
  <p class="graf graf--p">
   富人会比你想象得更容易接受这个论点，因为精英阶层自身所处的那个精英陷阱也正在收紧袋口，越来越多的富人开始反对精英体系本身，相信你已经能听到祈求工作、生活重获平衡的呼声越来越急迫了。
  </p>
  <p class="graf graf--p">
   大约有三分之二的精英阶层员工表示，如果升迁意味着新的工作要求他们付出更多时间，那么他们会直接拒绝升迁。
  </p>
  <p class="graf graf--p">
   当 Larry Kramer 在斯坦福法学院担任院长时，曾警告毕业生，那些顶尖律师事务所的律师们似乎陷入了一个无休止的悖论：想获得更高工资就需要年轻的律师付出更多时间以增加计费工时；但是要投入更多的工时，又需要得到更高、更合理的工资回报。
  </p>
  <p class="graf graf--p">
   Kramer 感叹道：
   <strong class="markup--strong markup--p-strong">
    这个体系到底是在为谁服务？有任何一方真的愿意生活在这样的体系之中吗？
   </strong>
  </p>
  <p class="graf graf--p">
   想要摆脱这个精英体系的陷阱并不容易，精英阶层自然会抵制任何有损自身权利、威胁其利益的政策。
  </p>
  <p class="graf graf--p">
   但是需要认清现实：
   <strong class="markup--strong markup--p-strong">
    单纯从自己的人脉资源中获得财富是不可能的，更重要的是开发自己的潜力，提升内心的生活质量而非物质生活。
   </strong>
  </p>
  <p class="graf graf--p">
   那些期望从目前精英体系中分一杯羹就抽身脱离体系的人，实际上也是在自欺欺人。
  </p>
  <p class="graf graf--p">
   想让精英阶层放弃对自身社会地位执念，唯一的方法是让更广泛的人群能获得社会中良好的教育机会、良好的就业机会，这样一来，攀登某种社会地位的金字塔就变得没那么重要了。
  </p>
  <p class="graf graf--p">
   然而真正的问题在于，如何实现这样的社会呢？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    首先，教育必须变得更开放、更包容，而如今的教育主要关注富裕家庭中那些接受奢侈训练的孩子，并以他们作为自身的利益。
   </strong>
  </p>
  <p class="graf graf--p">
   如果私立学校和大学无法做到从那些收入分配末位三分之二的家庭中至少招收一半的学生，那么就不应该继续享有免税特权。
  </p>
  <p class="graf graf--p">
   同时，应该通过公共补贴的方式鼓励学校达到这样的要求，扩大招生。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    与此同时，还应该推行一项平行政策，支持那些不包含复杂培训的行业，也支持那些没有高等学历的员工。
   </strong>
  </p>
  <p class="graf graf--p">
   比如，卫生保健体系应该进一步强调公共卫生、预防保健的重要性，以及其他由执业护士主导的卫生事业，而不是一味强调那些需要专业医生以及高科技治疗的卫生领域。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    另外，法律系统应该增设“法律技术人员”，因为很多日常事务并不需要考取法学博士才能经手管理，比如房地产交易、简单的遗嘱，甚至是无争议的协议离婚。
   </strong>
  </p>
  <p class="graf graf--p">
   在金融领域，限制外来金融项目、支持本地小型银行的政策可以帮助那些拥有中等技术的工人获得工作机会。而企业的管理者应该把控制权下放到主管级以下，提高公司其他员工的决定权。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    解决精英体系带来的不平等问题，其障碍不来自于技术，而是政策。
   </strong>
  </p>
  <p class="graf graf--p">
   如今的情况正在引发更多的不满，更普遍的悲观，也让人不断逼近绝望的边缘。
  </p>
  <p class="graf graf--p">
   政治学家 Jeffrey A. Winters 在他的著作《寡头政治》（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cambridge.org/core/books/oligarchy/5CC556B4483F7F3FDE1CADF928C04671" href="https://www.cambridge.org/core/books/oligarchy/5CC556B4483F7F3FDE1CADF928C04671" rel="noopener noreferrer" target="_blank">
    <em class="markup--em markup--p-em">
     Oligarchy
    </em>
   </a>
   ）中研究了从古典时代（Classical Antiquity）到20世纪人类的多个文明、社会，并记录下一旦财富被集中在狭隘的精英阶层中社会将发生什么后果。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    几乎在每一个案例中，这种不平等所带来的割裂都会引发社会的崩溃，其形式可能是军事上的失败（比如罗马帝国），也可能是引发革命（比如法兰西王国和俄罗斯帝国）。
   </strong>
  </p>
  <p class="graf graf--p">
   尽管如此，我们还是有理由对未来心怀希望的。因为历史上的确有一些积极的案例，社会和文明可以从财富集中的不平等状态中复苏，比如上世纪20年代和30年代的美国，政府以罗斯福新政应对大萧条，也带动了一批中产阶级家庭成长，最终成为上世纪中叶美国的核心支柱。
  </p>
  <p class="graf graf--p">
   重要的是，在这一过程中起到主引擎作用的，并非政府的重新分配政策。罗斯福政府建立起来的普遍繁荣，其基础来自于一个劳动力经济市场，正是这个市场促进了经济平等，遏制了等级制度 — — 比如，通过《美国军人权利法案》（GI Bill）大幅度增加受教育机会，另外，促使中等技能的中产阶级工人在生产中扮演核心角色。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    面对今天的局面，我们仍然可以将这个法案进行更新迭代并投入施行，重新增加受教育机会，重新加强中产阶级劳动者的作用，这两方面的政策可以相互支持。
   </strong>
  </p>
  <p class="graf graf--p">
   与此同时，只要收入和社会地位的下降仍在可接受的范围内，精英阶层也终于能够再次享受到应得的休息。
  </p>
  <p class="graf graf--p">
   不仅如此，中产阶级也可以重新获得相应的收入水平和社会地位，并再次支撑起美国人的生活。
  </p>
  <p class="graf graf--p">
   重建民主的经济秩序势必困难重重，但是经济民主将给每一个人带来益处，值得我们为之努力。
  </p>
  <p class="graf graf--p">
   放任现状愈演愈烈只能导致悲剧与崩溃，这样的未来让我们别无选择，只能尝试改变。
  </p>
  <p class="graf graf--p">

   </em>
   <a class="markup--anchor markup--p-anchor" data-href="http://www.amazon.com/exec/obidos/ISBN=0735221995/theatla05-20/" href="https://www.amazon.com/exec/obidos/ISBN=0735221995/theatla05-20/" rel="noopener noreferrer" target="_blank">

   </a>

  </p>
