   <h1>
    倪湛舸：玫瑰是玫瑰是玫瑰是玫瑰
   </h1>
   <ul class="post-entry-meta">
    <li>
     By:
     <a href="http://www.ideobook.com/about-ideobook/" title="查看 爱德布克 的作者主页">
      爱德布克
     </a>
     . 2005-7-13.
     <a href="http://www.ideobook.com/372/post-views-count/" title="统计说明">
      16,912
     </a>
    </li>
   </ul>
  </header>
  <div class="post-entry-content">
   <blockquote>
    <p>
     A rose is a rose is a rose is a rose.
     <br/>
     You can love a name and if you love a name then saying that name any number of times only makes you love it more, more violently more persistently more tormentedly.
    </p>
    <p>
     —Gertrude Stein
    </p>
   </blockquote>
   <blockquote>
    <p>
     白马非马，玫瑰是玫瑰
     <br/>
     难道只是诡辩？我们注定与身外的世界对立
     <br/>
     为建筑藏匿之巢而滥伐词语的丛林
     <br/>
     风雨于四十九年后停息，幸存者无力直腰
     <br/>
     却有箭矢从亡祖处射来，遍地挺立起玫瑰
     <br/>
     如同火柴和磷擦肩而过，又一个名字灼痛手指：
     <br/>
     “玫瑰”。它与现实平行，却无需空气、阳光、或水
     <br/>
     有这样的情人存在：早夭被打制成秘密容器
     <br/>
     她最后的呼吸永远近在咫尺，我们却从不曾到达
    </p>
    <p>
     玫瑰是玫瑰——是玫瑰
     <br/>
     盗火是罪，命名权却随意分发
     <br/>
     爱上你的时候，十指滴血落地生花
     <br/>
     疼痛的水面上浮现这样的波纹：玫瑰是玫瑰
     <br/>
     紧握穿透掌心的钉，水上行走的人终要离去，
     <br/>
     而我披盔戴甲守卫空坟，青苔爬满肺腑
     <br/>
     石像怎会有温度，最生动的姿势也不能记录挣扎
     <br/>
     曾经的战役有精兵突进，旌旗共血肉一色
     <br/>
     ——对，就是玫瑰绽放的瞬间
     <br/>
     我们为一个名字战死，仿佛遍地出壳的蜗牛
    </p>
    <p>
     玫瑰是玫瑰——是——玫瑰是玫瑰
     <br/>
     总是热衷同语反复，镜中的舌尖天旋地转
     <br/>
     舔平那无底陷阱上的草皮和血迹
     <br/>
     夏天失血过度，沉重的尘衣把空腹花瓶
     <br/>
     当作死于难产的女人而宠爱。窗帘静止
     <br/>
     画架上油彩龟裂，未完成的面容沉入混沌
     <br/>
     却在夜深时被呼嚎声惊起，不知往何处去
     <br/>
     世界尽头原来只是长宽搭建的木框
     <br/>
     烛光从另一维度入侵，成为危险的异端
    </p>
    <p>
     玫瑰！是玫瑰！是玫瑰！是玫瑰！
     <br/>
     炉火正旺，孤单的铁匠听见窗外的声响
     <br/>
     那些天流星接二连三地砸穿屋顶
     <br/>
     积水的鞋子里一下倒出不长眼睛的鱼
     <br/>
     我砸我打我敲我造，火里涌动着老虎绵羊和蠕虫
     <br/>
     窗外唱歌的孩子等着带它们回家
     <br/>
     玫瑰是玫瑰是玫瑰是玫瑰，不是锤子，更不是我
     <br/>
     我双手托起火里绽放的世界，独自打着寒战
     <br/>
     无数人死于那场地震，头骨被臆想中的玫瑰刺穿
    </p>
   </blockquote>
