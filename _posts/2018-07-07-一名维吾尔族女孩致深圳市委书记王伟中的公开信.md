   诺大的中国为何没有维吾尔族人的立锥之地？
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   尊敬的王伟中书记：
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   您好！我是一名新疆维吾尔族的女孩。前不久，深圳一家知名企业聘请我负责公司整体的项目推广工作，担任营销总监一职。于是我辞去了原新疆央企的工作，来到了我心向往之的改革开放之地深圳。来深圳这段时间，我的工作很顺利，但有一件事让我久久不能释怀，那就是住房问题。几乎每个房东都愿意将房子租给我住，因为就我一个人住，房租也从不讨价还价。但是，我没想到的事发生了，没有过几天，房东就要与我解除租约，要我搬家，说是社区不让维吾尔族人居住。我感到很不理解，于是到社区和派出所咨询。他们的意见是新疆维吾尔族人可以住，但房东必须每周带着我去派出所报告。房东都不愿接受这一条件，回绝了我，因为嫌麻烦，说多一事不如少一事，不如租给别人。我于是又与社区和派出所商量，但他们表示没有任何让步的可能。我告诉他们，我是一名中共党员，又是一名企业高管，无神论者，并表示自愿每周主动到社区和派出所报到。但我的请求被回绝了，最终我只得从所租的房子里搬了出来，在同事家里暂住，但这给同事带来了不必要的麻烦，我的心里很不是滋味，觉得对不起人家。再说，长此以往，我也需要一个自己的私人空间。王书记，我不明白为什么深圳要这样对待维吾尔族人？如果深圳都不能容下一个维吾尔女孩，诺大的中国哪里又有维吾尔族人的立锥之地？如果我都不能在内地生存下去，其他维吾尔人是否只能永远呆在新疆？
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>

   此致，
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   敬礼！
