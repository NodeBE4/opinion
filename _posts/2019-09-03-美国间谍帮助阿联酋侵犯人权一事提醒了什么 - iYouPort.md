 <div class="entry-content">
  <p class="graf graf--p">
   美国前情报人员帮助阿联酋政权监视记者和政府批评者的报道应该激发西方政府领导人的一些认真的反省，他们的行为正在以“打击犯罪和恐怖主义”的借口破坏公民的网络安全。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://uk.reuters.com/article/uk-usa-spying-raven-specialreport/special-report-inside-the-uaes-secret-hacking-team-of-u-s-mercenaries-idUKKCN1PO1A6" href="https://uk.reuters.com/article/uk-usa-spying-raven-specialreport/special-report-inside-the-uaes-secret-hacking-team-of-u-s-mercenaries-idUKKCN1PO1A6" rel="noopener noreferrer" target="_blank">
    路透社报道
   </a>
   ，包括前国家安全局（NSA）分析员在内的前美国情报人员，一直在为阿联酋的秘密团队工作，以帮助该威权政府监视其敌人和对手。目标不仅包括竞争对手国家的政府官员（如卡塔尔人），还包括批评政府的人权活动家，被针对的目标甚至包括美国公民。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我正在为一个针对美国人的外国情报机构工作，”前 NSA 分析师 Lori Stroud 告诉路透社。 “我是正式的间谍。”Stroud 于2014年离开了美国国家安全局 — 部分原因是她推荐了爱德华·斯诺登（Edward Snowden）担任国家安全局的承包商。此后她开始为阿联酋工作。
  </p>
  <p class="graf graf--p">
   她参加了一项名为 Project Raven 的计划，她最初认为这是一项旨在保护阿联酋免受黑客和威胁攻击的防御性反恐行动。然而不久，她很快就明白了。这是一个击败阿联酋政权的敌人的侵略性计划。
  </p>
  <p class="graf graf--p">
   Project Raven 项目
   <a class="markup--anchor markup--p-anchor" data-href="https://www.reuters.com/article/us-usa-spying-karma-exclusive/exclusive-uae-used-cyber-super-weapon-to-spy-on-iphones-of-foes-idUSKCN1PO1AN" href="https://www.reuters.com/article/us-usa-spying-karma-exclusive/exclusive-uae-used-cyber-super-weapon-to-spy-on-iphones-of-foes-idUSKCN1PO1AN" rel="noopener noreferrer" target="_blank">
    利用 iPhone 的安全漏洞
   </a>
   ，允许间谍们在用户不知道、甚至无需做任何事的情况下安装恶意软件。这个名为 Karma 的工具允许黑客收集照片和位置信息，并收集保存的密码。
  </p>
  <p class="graf graf--p">
   阿联酋试图侵入持不同政见者和活动家的手机，这不足为奇。由于阿联酋一
   <a class="markup--anchor markup--p-anchor" data-href="https://reason.com/blog/2016/08/26/government-sponsored-hacking-of-activist" href="https://reason.com/blog/2016/08/26/government-sponsored-hacking-of-activist" rel="noopener noreferrer" target="_blank">
    直在钟情于破坏该国人权倡导者电话的恶意软件工具
   </a>
   ，苹果实际上在2016年发布了对 iPhone 的紧急更新。
  </p>
  <p class="graf graf--p">
   路透社的报告为我们提供了另一个理由，即 选民应该以打击犯罪和恐怖主义的名义拒绝政府要求破坏网络安全的行动。这里指的是真正的犯罪行为，而不是当局的借口。
  </p>
  <p class="graf graf--p">
   具体而言，我指的是政府官员和执法领导人不断坚持认为电话和在线通信平台和应用程序应该有某种后门，或可以让政府绕过加密的机制。在美国、英国和澳大利亚，都有政治领导人和执法及情报部门的负责人，他们坚持认为科技公司必须帮助他们实现对加密通信进行监视。
  </p>
  <p class="graf graf--p">
   隐私专家和技术专家习惯性地警告这些领导者，你不能以这样的方式危害网络安全，因为任何可用于绕过加密的技术都不可避免地会落入坏人之手并被用来对抗好人。这些坏人可能是想要搞身份盗窃或诈骗的罪犯，更可能就是危险的政府本身，如阿联酋希望在美国国家安全局的帮助下惩罚人权活动家。政府是最大的犯罪分子。
  </p>
  <p class="graf graf--p">
   急需提醒下澳大利亚议会，他们的反加密立法实际上可能会杀人。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    B
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://reason.com/blog/2019/01/31/bombshell-report-about-americans-spying" href="https://reason.com/blog/2019/01/31/bombshell-report-about-americans-spying" rel="noopener noreferrer" target="_blank">
    ombshell Report About Americans Helping UAE Hack Phones Is a Warning Against Compromising Encryption
   </a>
   . Hacking tools end up in the hands of some dangerous people. So, apparently, do our government hackers.
  </p>
