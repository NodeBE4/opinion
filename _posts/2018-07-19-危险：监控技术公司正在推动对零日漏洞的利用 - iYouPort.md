   <p class="graf graf--p graf-after--figure" id="ee50">
    销售当今最具侵入性的监控系统的私人监控技术公司，通过购买被广泛使用的软件中的安全漏洞，并将其与自己的入侵产品捆绑在一起，为其客户提供对目标计算机和电话设备的前所未有的入侵能力。
   </p>
   <p class="graf graf--p graf-after--p" id="57d7">
    随着 Wikileaks 对情报机构间谍行为的曝光，一段时间以来，很多人都知道了，
    <strong class="markup--strong markup--p-strong">
     政府通常会付出相当大的代价从安全研究人员那里购买被称为零日攻击的漏洞，用于监视和间谍活动
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="8b4c">
    虽然重点是政府直接购买这些漏洞，但需要强调的是，
    <strong class="markup--strong markup--p-strong">
     私人监视技术公司在漏洞销售市场中的作用同等重要
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="45cf">
    <strong class="markup--strong markup--p-strong">
     <em class="markup--em markup--p-em">

    </strong>
    <a class="markup--anchor markup--p-anchor" data-href="https://iyouport.wixsite.com/iyouport/blog/%E9%9B%B6%E6%97%A5%E6%BC%8F%E6%B4%9E%E5%92%8C-cia-%E9%BB%91%E5%AE%A2%E5%B7%A5%E5%85%B7" href="https://www.iyouport.org/%E9%9B%B6%E6%97%A5%E6%BC%8F%E6%B4%9E%E5%92%8C-cia-%E9%BB%91%E5%AE%A2%E5%B7%A5%E5%85%B7/" rel="nofollow noopener noreferrer" target="_blank">

   <p class="graf graf--p graf-after--p" id="b783">
    最近几个月来，随着对特定监视技术出口的新控制在表面上达成一致，关于销售和使用零日攻击的辩论也越来越多。但是，尽管声称已经出台了新的出口管制措施来“规范”这类销售，但零日漏洞问题
    <strong class="markup--strong markup--p-strong">
     仍然无法控制
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="bdc2">
    软件 bug 的出售早已构成了一个利润丰厚的市场，通常可以让计算机研究人员获得声望和金钱。当识别出这些漏洞时，传统的做法是将其出售给软件公司（为了及时修补漏洞），但事实并不是如此，
    <strong class="markup--strong markup--p-strong">
     很多人会见漏洞卖给出价最高的人，而资金丰厚的政府和情报机构显然是第一受益人
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="410e">
    在 Gamma Group 宣传其
    <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/804667-787_gamma-group_brochure_finfly-exploit-portal.html" href="https://www.documentcloud.org/documents/804667-787_gamma-group_brochure_finfly-exploit-portal.html" rel="nofollow noopener noreferrer" target="_blank">
     FinFly 漏洞利用的门户
    </a>
    网站宣传册中你能看到，该公司向各国政府“提供大型的0日和1日漏洞利用库，
    <strong class="markup--strong markup--p-strong">
     可以访问各种流行的软件，如 Microsoft Office，Internet Explorer，Adobe Acrobat Reader 等等”，以更好地使用他们自己的监控技术。
    </strong>
   </p>
   <p>
    <img alt="" class="aligncenter size-full wp-image-2649 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-17.jpeg?resize=1100%2C620&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-17.jpeg?w=1920&amp;ssl=1 1920w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-17.jpeg?resize=300%2C169&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-17.jpeg?resize=768%2C433&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-17.jpeg?resize=1024%2C578&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-17.jpeg?resize=1100%2C620&amp;ssl=1 1100w" data-recalc-dims="1" height="620" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-17.jpeg?resize=1100%2C620&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   <p class="graf graf--p graf-after--figure" id="30df">
    通过使用 FinFly 漏洞利用门户，政府可以将复杂的入侵技术（如 FinSpy）作用到目标计算机上。利用门户通过提供更多的部署选择，
    <strong class="markup--strong markup--p-strong">
     为压迫性政府提供了更多的监视社会的权力。
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="ec72">
    更令人惊讶的是，FinFly Exploit Portal 还为用户提供了
    <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/804667-787_gamma-group_brochure_finfly-exploit-portal.html#document/p1/a12594" href="https://www.documentcloud.org/documents/804667-787_gamma-group_brochure_finfly-exploit-portal.html#document/p1/a12594" rel="nofollow noopener noreferrer" target="_blank">
     <strong class="markup--strong markup--p-strong">
      四种可行的攻击方式
     </strong>
    </a>
    ，可用于世界上所有最常见的软件产品，例如 Microsoft 的 Internet Explorer 和 Adobe 的程序等等。
   </p>
   <p class="graf graf--p graf-after--p" id="e1c9">
    Gamma 销售的漏洞利用了互联网浏览器、文件阅读器和其他应用程序中的漏洞来提供其产品 Finspy。
    <strong class="markup--strong markup--p-strong">
     利用用户对微软和苹果等提供商的信任，通过该漏洞提供的恶意软件可以把自己伪装为 iTunes 的更新出现，还可能是您可能感兴趣的 .pdf 文件，甚至是网页！
    </strong>
   </p>

    <img alt="" class="aligncenter size-full wp-image-2650 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-18.jpeg?resize=1100%2C660&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-18.jpeg?w=1920&amp;ssl=1 1920w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-18.jpeg?resize=300%2C180&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-18.jpeg?resize=768%2C461&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-18.jpeg?resize=1024%2C614&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-18.jpeg?resize=1100%2C660&amp;ssl=1 1100w" data-recalc-dims="1" height="660" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-18.jpeg?resize=1100%2C660&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   <div class="section-inner sectionLayout--insetColumn">
    <p class="graf graf--p graf-after--figure" id="1618">
     与 Gamma 类似，意大利监控公司 Hacking Team 也提供零日攻击。他们的“
     <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/810561-841-hacking-team-product-description-remote.html" href="https://www.documentcloud.org/documents/810561-841-hacking-team-product-description-remote.html" rel="nofollow noopener noreferrer" target="_blank">
      远程控制系统
     </a>
     ”显示出很多与 Gamma 产品系列相同的特性，包括对设备的入侵和信息的收集。
    </p>
    <p class="graf graf--p graf-after--p" id="f6cb">
     根据 Omega Research 提供的监控行业指数中的文件，骇客团队似乎提供了一个
     <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/810561-841-hacking-team-product-description-remote.html#document/p2/a134927" href="https://www.documentcloud.org/documents/810561-841-hacking-team-product-description-remote.html#document/p2/a134927" rel="nofollow noopener noreferrer" target="_blank">
      <strong class="markup--strong markup--p-strong">
       零日漏洞利用库
      </strong>
     </a>
     ，以帮助将遥控系统安装到个人设备上。
    </p>
    <p class="graf graf--p graf-after--p" id="f50a">
     该骇客团队以前曾参与过
     <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.org/2012/10/backdoors-are-forever-hacking-team-and-the-targeting-of-dissent/" href="https://citizenlab.org/2012/10/backdoors-are-forever-hacking-team-and-the-targeting-of-dissent/" rel="nofollow noopener noreferrer" target="_blank">
      针对摩洛哥活动家
     </a>
     和埃塞俄比亚调查
     <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.org/2014/02/hacking-team-targeting-ethiopian-journalists/" href="https://citizenlab.org/2014/02/hacking-team-targeting-ethiopian-journalists/" rel="nofollow noopener noreferrer" target="_blank">
      记者
     </a>
     的间谍活动，并已在阿塞拜疆、埃及、埃塞俄比亚、哈萨克斯坦、马来西亚、尼日利亚、阿曼、沙特阿拉伯、苏丹、土耳其和乌兹别克斯坦等地
     <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.org/2014/02/mapping-hacking-teams-untraceable-spyware/" href="https://citizenlab.org/2014/02/mapping-hacking-teams-untraceable-spyware/" rel="nofollow noopener noreferrer" target="_blank">
      使用过
     </a>
     这种间谍技术。
    </p>
    <p class="graf graf--p graf-after--p" id="69d6">
     监控公司不是唯一销售零日攻击的公司。 VUPEN security 是一家总部位于法国的公司，同样销售漏洞利用程序，
     <strong class="markup--strong markup--p-strong">
      这种程序使监控公司能够破解设备的安全功能，并
     </strong>
     <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/409356-279-vupen-thread-exploits.html#document/p1/a134929" href="https://www.documentcloud.org/documents/409356-279-vupen-thread-exploits.html#document/p1/a134929" rel="nofollow noopener noreferrer" target="_blank">
      <strong class="markup--strong markup--p-strong">
       提供
      </strong>
     </a>
     <strong class="markup--strong markup--p-strong">
      对“VUPEN 安全研究人员在内部发现的未公开漏洞”的访问权限。
     </strong>
    </p>
    <p class="graf graf--p graf-after--p" id="b4f0">
     VUPEN 的工作能力与 Gamma 的漏洞利用门户类似：支付资金是为了获得访问提供漏洞的安全门户。
    </p>
    <p class="graf graf--p graf-after--p" id="2e52">
     VUPEN 的数据库运行一个订阅服务，就像 LexisNexis 和 JSTOR 背后的付费墙一样。除了新闻和学术文章外，VUPEN 还为世界上一些最大的科技公司产品
     <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/804676-1266_vupen_product-description_vulnerability.html#document/p2/a125956" href="https://www.documentcloud.org/documents/804676-1266_vupen_product-description_vulnerability.html#document/p2/a125956" rel="nofollow noopener noreferrer" target="_blank">
      提供漏洞
     </a>
     。
    </p>
    <p class="graf graf--p graf-after--p" id="8754">
     营销手册第三页显示，
     <strong class="markup--strong markup--p-strong">
      VUPEN 允许访问其“威胁防护计划”，以便大型企业了解其系统中发现的任何漏洞。而在第二页，VUPEN 宣传其“执法机构的漏洞”，其中相同的漏洞被出售给执法和情报机构，用于抓人。两种不同的方法，完全是同一个目标：为 VUPEN 赚钱。
     </strong>
    </p>
    <p class="graf graf--p graf-after--p" id="e4b6">
     那么，VUPEN 究竟销售给谁呢？我们最近看到，
     <strong class="markup--strong markup--p-strong">
      美国已经购买了对 VUPEN 漏洞利用库的访问权限。
     </strong>
    </p>
    <p class="graf graf--p graf-after--p" id="2583">
     成为独家 VUPEN 俱乐部会员有一些要求：
    </p>
    <ul class="postList">
     <li class="graf graf--li graf-after--p" id="cd43">
      客户必须是情报机构或执法机构;
     </li>
     <li class="graf graf--li graf-after--li" id="4d49">
      必须签署保密协议;
     </li>
     <li class="graf graf--li graf-after--li" id="dfac">
      必须是北约、澳新或东盟的成员。
     </li>
    </ul>
    <p class="graf graf--p graf-after--li" id="8c0f">
     然而，该政策似乎没什么实际用途，因为
     <strong class="markup--strong markup--p-strong">
      公民实验室已经发现，VUPEN 漏洞被用来
     </strong>
     <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.org/2012/10/backdoors-are-forever-hacking-team-and-the-targeting-of-dissent/" href="https://citizenlab.org/2012/10/backdoors-are-forever-hacking-team-and-the-targeting-of-dissent/" rel="nofollow noopener noreferrer" target="_blank">
      <strong class="markup--strong markup--p-strong">
       瞄准阿联酋的一位著名博主
      </strong>
     </a>
     <strong class="markup--strong markup--p-strong">
      ，至少仅此一例就完全不符合 VUPEN 自己的购买者要求。
     </strong>
    </p>
    <p class="graf graf--p graf-after--p" id="51bd">
     VUPEN
     <a class="markup--anchor markup--p-anchor" data-href="http://www.issworldtraining.com/iss_mea/Track4.html" href="http://www.issworldtraining.com/iss_mea/Track4.html" rel="nofollow noopener noreferrer" target="_blank">
      上个月在迪拜
     </a>
     参加了 ISS World Middle East，这是一个全球监控公司的贸易展，让他们有机会与来自中东的客户聚集在一起。 VUPEN 首席执行官还举办了一次关于“IT 入侵（计算机和移动设备）零日漏洞”的培训研讨会，教授执法机构如何使用他们的产品，以便他们可以入侵任何计算机和移动设备！

   <p class="graf graf--p graf-after--figure" id="0b5c">
    对于安全研究人员而言，漏洞利用是非常有价值的。它们具有完全合法的目的，应该鼓励与其发展有关的研究，特别是对脆弱性的研究。
   </p>
   <p class="graf graf--p graf-after--p" id="b271">
    然而，
    <strong class="markup--strong markup--p-strong">
     滥用的结果非常危险，而且滥用几乎无法避免
    </strong>
    。目前对漏洞出口的限制根本就是绣花枕头，无法解决根本性问题。了解当前的开发市场是关键。了解开发人员是谁、客户是谁、交易方式、以及使用方式，至关重要。
   </p>
