  <p class="graf graf--p">
   特朗普的国务院在其年度人权报告中的表述非常坦率。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “[我国]政府的政策就是与其他政府接触，无论他们的人权记录如何，只要这样做能进一步有益于美国的利益，”国务卿迈克庞培在报告的序言中写道。该文件于周三发布，正式名为“2018年国家人权报告”。
  </p>
  <p class="graf graf--p">
   这一真小人的坦诚引起了国际特赦组织的注意，该组织称：“令人惊讶的是，这种话居然出现在美国政府发布的官方人权报告中，这一报告长期以来一直被世界各国的领导人视为一项重要的可信性评估，关于世界各国的人权记录。“
  </p>
  <p class="graf graf--p">
   该组织在一份声明中说：“这一声明向包括沙特王储穆罕默德·本·萨勒曼、和埃及总统 Sisi 在内的世界各国领导人发出的信号表明：
   <strong class="markup--strong markup--p-strong">
    美国不关心人权，甚至会原谅或忘记最恶劣的侵犯人权行为。”
   </strong>
  </p>
  <p class="graf graf--p">
   Sisi 被称为“现代埃及历史上最具压制性的独裁者”，并被指责以“践踏人权和公民权利”为标志，推动他的国家“进一步走向专制主义”，包括“在全国范围内不断打击各种形式的异议”。
  </p>
  <p class="graf graf--p">
   沙特阿拉伯 — 一个重要的美国盟友 — 因其恶劣的人权记录而受到三十个国家的指责。王国的王储穆罕默德·本·萨勒曼（Mohammad bin Salman，有时被称为 MbS）是“一位铁腕的领导者，尽管进行了表面上的改革，但他的国家人民则陷入了更多的镇压，并且导致也门三年的战争充满了战争罪行” — 根据人权观察的 Philippe Bolopion 的说法。
  </p>
  <p class="graf graf--p">
   MbS 的暴政针对妇女权利活动家，他也与记者 Jamal Kahsoggi 的可怕杀戮有关。然而，正如 CNN 报道的那样，
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    报告指出沙特阿拉伯杀害了华盛顿邮报的记者 Jamal Khashoggi，但是并没有提及是穆罕默德·本·萨勒曼王储下令的
   </em>
   。
  </p>
  <p class="graf graf--p">
   国务院报告的其他方面也引起了批评。
  </p>
  <p class="graf graf--p">
   与去年一样，该部门没有解决生殖权利问题。 “这个政府正在一次性删除妇女的权利，这是一项糟糕的政策，”生殖权利中心在一条推文中回应道。
  </p>
  <p class="graf graf--p">
   该报告还忽视了国际观点，并提到了西岸、加沙、以及首先将戈兰高地称为“以色列控制的”而不是以色列占领的。
  </p>
  <p class="graf graf--p">
   联合国将这三个地区均描述为以色列占领的地区。
  </p>
