  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    卢旺达种族灭绝事件25周年，新电影展示了正义与治愈之旅】
   </strong>
   随着全世界纪念卢旺达种族灭绝25周年，即将出版的
   <a class="markup--anchor markup--p-anchor" data-href="https://thegroundtruthproject.org/humanity-lost-humanity-found-25-years-after-rwandas-genocide/?utm_source=The+GroundTruth+Project+mailing+list&amp;utm_campaign=299c0a0876-EMAIL_CAMPAIGN_2019_04_05_04_12&amp;utm_medium=email&amp;utm_term=0_9afb6f4318-299c0a0876-124402945&amp;mc_cid=299c0a0876&amp;mc_eid=6eb01ac223#directors-statement" href="https://thegroundtruthproject.org/humanity-lost-humanity-found-25-years-after-rwandas-genocide/?utm_source=The+GroundTruth+Project+mailing+list&amp;utm_campaign=299c0a0876-EMAIL_CAMPAIGN_2019_04_05_04_12&amp;utm_medium=email&amp;utm_term=0_9afb6f4318-299c0a0876-124402945&amp;mc_cid=299c0a0876&amp;mc_eid=6eb01ac223#directors-statement" rel="noopener noreferrer" target="_blank">
    新纪录片
   </a>
   揭示了幸存者和肇事者走向正义和治愈的漫长旅程。
  </p>
  <p class="graf graf--p">
   1994年4月7日，卢旺达占主导地位的胡图族成员发起了为期100天的杀戮狂潮，针对内地东非国家少数民族图西族人。估计有80万人或更多人被屠杀 — — 95,000名儿童没有父母。国际社会在围观。
  </p>
  <p class="graf graf--p">
   这场大屠杀使 Gadi Habumugisha，Bizimana Jean 和 Mussa Uwitonze 成为了孤儿。他们现在是
   <a class="markup--anchor markup--p-anchor" data-href="https://thegroundtruthproject.org/about/" href="https://thegroundtruthproject.org/about/" rel="noopener noreferrer" target="_blank">
    The GroundTruth 项目
   </a>
   的专业摄影师和研究员，该项目来自位于波士顿的非营利媒体组织，名为 Camera Kids。
  </p>
  <p class="graf graf--p">
   在这里看到预告片：
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="576" src="//player.vimeo.com/video/328732815?title=0&amp;byline=0&amp;portrait=0&amp;color=8dc7dc" width="700">
   </iframe>
  </p>
  <p class="graf graf--p">
   种族灭绝结束后，Gadi，Bizimana 和 Mussa 由美国人道主义者 Rosamond Carr 收留在卢旺达穆图拉的 Imbabazi 孤儿院长大。在那里，他们收到了来自
   <a class="markup--anchor markup--p-anchor" data-href="https://www.camerakids.photos/" href="https://www.camerakids.photos/" rel="noopener noreferrer" target="_blank">
    Through the Eyes of Children
   </a>
   的一次性富士相机，这是一个非营利组织，于2000年开始举办摄影工作坊，教导弱势儿童如何捕捉画面，并与世界分享他们的故事。
  </p>
  <p class="graf graf--p">
   参加非营利组织第一次研讨会的19名卢旺达孤儿被称为“相机儿童”。今天，纪录片中的三名男子已经20多岁了，他们都是“Through the Eyes of Children”项目的协调员。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “当我们还是孩子的时候，[Carr]曾经告诉我们，我们应该与他人分享我们所拥有的东西，”Bizimana
   <a class="markup--anchor markup--p-anchor" data-href="http://time.com/5564906/rwanda-genocide-perpetrators-documentary-photography/" href="https://time.com/5564906/rwanda-genocide-perpetrators-documentary-photography/" rel="noopener noreferrer" target="_blank">
    告诉
   </a>
   时代周刊，“这是她给我们的传承。我们给其他孩子摄影是在做我们答应过她的事。”
  </p>
  <p class="graf graf--p">
   Mussa 通过镜头讲故事，电影中解释说，“这可能就像一种药。可以治愈某人。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “当我学习摄影时，那是我能表达自己的时候，我能从中意识到我很重要”，Gadi 说。
  </p>
  <p class="graf graf--p">
   GroundTruth 项目的
   <a class="markup--anchor markup--p-anchor" data-href="https://thegroundtruthproject.org/humanity-lost-humanity-found-25-years-after-rwandas-genocide/?utm_source=The+GroundTruth+Project+mailing+list&amp;utm_campaign=299c0a0876-EMAIL_CAMPAIGN_2019_04_05_04_12&amp;utm_medium=email&amp;utm_term=0_9afb6f4318-299c0a0876-124402945&amp;mc_cid=299c0a0876&amp;mc_eid=6eb01ac223#directors-statement" href="https://thegroundtruthproject.org/humanity-lost-humanity-found-25-years-after-rwandas-genocide/?utm_source=The+GroundTruth+Project+mailing+list&amp;utm_campaign=299c0a0876-EMAIL_CAMPAIGN_2019_04_05_04_12&amp;utm_medium=email&amp;utm_term=0_9afb6f4318-299c0a0876-124402945&amp;mc_cid=299c0a0876&amp;mc_eid=6eb01ac223#directors-statement" rel="noopener noreferrer" target="_blank">
    电影
   </a>
   将于2020年推出，他们通过采访和拍摄那些幸存下来的人，以及那些对大规模杀戮负有责任的人及其家人，来探索夺走他们父母生命的种族灭绝事件。
  </p>
  <p class="graf graf--p">
   PBS NewsHour 已经播出了该电影的片段。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我遇到了这对夫妇，”Gadi 说， “一个是犯罪者的儿子，另一个是幸存者。他们曾经互相残杀，现在互相原谅，彼此团聚，一起生活，现在结婚，过去正在远离。”
  </p>
  <p class="graf graf--p">
   在这里看到片段：
   <a class="markup--anchor markup--p-anchor" data-href="https://www.pbs.org/newshour/show/how-young-rwandan-genocide-survivors-are-documenting-25-years-of-healing" href="https://www.pbs.org/newshour/show/how-young-rwandan-genocide-survivors-are-documenting-25-years-of-healing" rel="noreferrer noopener" target="_blank">
    https://www.pbs.org/newshour/show/how-young-rwandan-genocide-survivors-are-documenting-25-years-of-healing
   </a>
  </p>
  <p class="graf graf--p">
   在 The GroundTruth 项目发表的一份导演声明中，美国电影制片人 Beth Murphy 强调了全球社会可以从卢旺达种族灭绝事件中获得的关于仇恨与和平的教训 — 希望在 Camera Kids。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果没有与当今世界正在发生的事相提并论，就不可能真正研究这段历史，”Murphy 写道。 “随着世界各地极右翼民粹主义的兴起，美国和所有西方国家的白人民族主义者正在使用与卢旺达的种族灭绝肇事者在1994年大屠杀之前的几个月和几年中使用过的同样令人恐惧的语言。这令人不寒而栗。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “当卢旺达从100天的黑暗中走出来时，宽恕和和解计划不仅得到了鼓励，而且得到了强制执行，”她指出。“人们曾经被教导如何憎恨，现在他们被教导如何实现和平。种族灭绝后二十五年，该国找到了平衡正义与治愈、复仇和宽恕的方法，并为世界带来了一个希望的模式。”
  </p>
