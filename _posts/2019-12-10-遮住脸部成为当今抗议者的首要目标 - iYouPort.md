   这是CNN的文章，我们翻译了它；这篇文章足够显示主流媒体只为蹭热点而不打算帮助解决问题的写作方式。
  </p>
  <p class="graf graf--p">
   如果您经常以主流媒体为信息来源，不仅难以找到解决方案，并且很容易抑制对相关问题的思考（它会缩窄你的眼界）。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    用技术对战技术
   </strong>
  </p>
  <p class="graf graf--p">
   美国设计师佩德罗·奥利维拉（Pedro Oliveira）2013年在巴西本土的抗议活动中目睹了警方的暴行后，开始研究世界各地的政府当局如何部署技术来对付示威者。
  </p>
  <p class="graf graf--p">
   在一些国家，使用的手段是断网；在其他国家，则是广泛的审查。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    附《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e7%94%a8%e6%8c%87%e6%a0%87%e9%a2%84%e6%b5%8b%e4%ba%92%e8%81%94%e7%bd%91%e4%b8%ad%e6%96%ad%ef%bc%9f%e4%b8%80%e6%97%a6%e6%96%ad%e7%bd%91%e8%bf%98%e5%8f%af%e4%bb%a5%e6%80%8e%e4%b9%88/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e7%94%a8%e6%8c%87%e6%a0%87%e9%a2%84%e6%b5%8b%e4%ba%92%e8%81%94%e7%bd%91%e4%b8%ad%e6%96%ad%ef%bc%9f%e4%b8%80%e6%97%a6%e6%96%ad%e7%bd%91%e8%bf%98%e5%8f%af%e4%bb%a5%e6%80%8e%e4%b9%88/" rel="noopener noreferrer" target="_blank">

  <p class="graf graf--p">
   为了提高人们抵制当局对抗议者使用的手段的认识，Oliveira和其他设计师创作了一个抗议工具包，作为他们的艺术和设计项目
   <a class="markup--anchor markup--p-anchor" data-href="http://www.backslash.cc/" href="http://www.backslash.cc/" rel="noopener noreferrer" target="_blank">
    Backslash
   </a>
   的一部分。
  </p>
  <p class="graf graf--p">
   该工具包包括一个
   <a class="markup--anchor markup--p-anchor" data-href="http://www.backslash.cc/" href="http://www.backslash.cc/" rel="noopener noreferrer" target="_blank">
    “智能”头巾
   </a>
   ，用于隐藏佩戴者的身份，同时通过计算机生成的图案在抗议者之间传递消息，该图案只能由自定义应用程序读取。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8954 jetpack-lazy-image" data-lazy-sizes="(max-width: 727px) 100vw, 727px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-16.jpg?resize=727%2C485&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-16.jpg?w=727&amp;ssl=1 727w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-16.jpg?resize=300%2C200&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-16.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="485" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-16.jpg?resize=727%2C485&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="727"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8954" data-recalc-dims="1" height="485" sizes="(max-width: 727px) 100vw, 727px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-16.jpg?resize=727%2C485&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-16.jpg?w=727&amp;ssl=1 727w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-16.jpg?resize=300%2C200&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-16.jpg?resize=272%2C182&amp;ssl=1 272w" width="727"/>

  <p class="graf graf--p">
   该套件还具有一个可穿戴设备，可以及时提醒示威者注意警察的存在，以及一个模板，可以创建涂鸦“标签” —— 只有应用程序可读 —— 以便在一个地区受到监视时通知抗议者。
  </p>
  <p class="graf graf--p">
   但这些工具包不会出售。相反，Oliveira 希望开始就他们认为当局与示威者之间日益增长的权力失衡状态展开对话。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们并不认为这些工具应该作为一种解决方案提供给抗议者。我们只是认为重要的是研究和谈论警察的超军事化和所有这些技术被用来对付抗议者的问题，” Oliveira 说。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8955 jetpack-lazy-image" data-lazy-sizes="(max-width: 727px) 100vw, 727px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-16.jpg?resize=727%2C485&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-16.jpg?w=727&amp;ssl=1 727w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-16.jpg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-16.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="485" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-16.jpg?resize=727%2C485&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="727"/>

  <p class="graf graf--p">
   面部识别技术的进步是示威者和隐私权活动家特别关注的问题。
  </p>
  <p class="graf graf--p">
   这项技术大部分仍处于开发阶段，其工作原理是捕捉人脸图像，并将生物识别信息（如面部特征之间的距离）与现有的照片ID数据库相匹配。
  </p>
  <p class="graf graf--p">
   在创作项目
   <a class="markup--anchor markup--p-anchor" data-href="https://cvdazzle.com/" href="https://cvdazzle.com/" rel="noopener noreferrer" target="_blank">
    CV Dazzle
   </a>
   中，艺术家 Adam Harvey 提出了一些致力于帮助人们躲避监视技术的创造性方法，包括面部修饰和精致的发型。
  </p>
  <p class="graf graf--p">
   在他的网站上，他声称应用色调渐变的化妆可以创造虚假的轮廓，模糊人们的面部特征 —— 被称为“anti-face”，正如他所说的那样 —— 使面部识别系统更难收集到可识别一个人的数据。
  </p>
  <p class="graf graf--p">
   他还建议不对称地使用化妆，以便抵制面部识别算法。
  </p>
  <p class="graf graf--p">
   另外，伦敦的
   <a class="markup--anchor markup--p-anchor" data-href="http://www.zachblas.info/" href="http://www.zachblas.info/" rel="noopener noreferrer" target="_blank">
    Zach Blas
   </a>
   在他的项目“面部武器化套件”中使用了来自多个面孔的生物识别数据，发明了“集体面具”。
  </p>
  <p class="graf graf--p">
   其中一个面具解决了面部识别软件对肤色的内在偏见，这一直是活动家批评的重点。由于面部识别算法不成比例地过多使用了白人男性的面孔数据，女性和有色人种被错误识别的可能性会明显增加。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8956 jetpack-lazy-image" data-lazy-sizes="(max-width: 727px) 100vw, 727px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/4-5.jpg?resize=727%2C485&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/4-5.jpg?w=727&amp;ssl=1 727w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/4-5.jpg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/4-5.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="485" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/4-5.jpg?resize=727%2C485&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="727"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8956" data-recalc-dims="1" height="485" sizes="(max-width: 727px) 100vw, 727px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/4-5.jpg?resize=727%2C485&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/4-5.jpg?w=727&amp;ssl=1 727w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/4-5.jpg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/4-5.jpg?resize=272%2C182&amp;ssl=1 272w" width="727"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   由此产生的伪装在机器看来就像是无定形的多彩的斑点，Blas 说这种伪装可以欺骗面部识别技术 —— 在某种程度上，佩戴者甚至不被认定为人类。
  </p>
  <p class="graf graf--p">
   但
   <strong class="markup--strong markup--p-strong">
    这些不寻常的策略可能会产生自己的问题。就如我们曾经在《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ae%a9%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e5%a4%b1%e6%95%88%e7%9a%84%e6%80%9d%e8%80%83%e6%96%b9%e5%bc%8f/" href="https://www.iyouport.org/%e8%ae%a9%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e5%a4%b1%e6%95%88%e7%9a%84%e6%80%9d%e8%80%83%e6%96%b9%e5%bc%8f/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     让面部识别失效的思考方式
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》中分析过的，它让你太过显眼了，在人群中格外出众肯定不是一个隐身者想要的效果。
   </strong>
  </p>
  <p class="graf graf--p">
   这些隐藏面部的方法很可能会将其他人被“武器化”成为监视的一部分，这意味着，有人会向当局报告任何与众不同的东西 —— 如不寻常的化妆或类似斑点的面具。
  </p>
  <p class="graf graf--p">
   就如 Selvaggio 所说的，“这些反监视项目没能真正考虑到监视在人类方面的效应”。
  </p>
  <p class="graf graf--p">
   Selvaggio 提出了另一种解决方案（依旧不好）：是他自己脸部的假面具。就像 Blas 的“集体”面具一样，它们可以隐藏佩戴者的真实身份。
  </p>
  <p class="graf graf--p">
   但与 Blas 的伪装不同，这类面具将被监视系统作为人脸进行分析，愚弄当局认为一个人同时在多个地方。
  </p>
  <p class="graf graf--p">
   真正有效的方法和隐身应有的思考方式可以在上述链接集中找到。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    香港的简单解决方案
   </strong>
  </p>
  <p class="graf graf--p">
   在香港街头，抗议者选择了现成的解决方案。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “人们倾向于在抗议活动中进行大量尝试，” Gavin Grindon 说，他曾在伦敦维多利亚和阿尔伯特博物馆举办2014年抗议活动展览。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这通常是简单的解决方案 — 快速解决方案 — 这是优雅而有效的解决方案，”Grindon 说。
  </p>
  <p class="graf graf--p">
   许多人选择配备护目镜或防毒面具来掩盖自己的身份。“我们需要隐藏自己，这样就不会担心被监视相机捕捉到”，一位不愿透露姓名的22岁志愿者助手说道。
  </p>
  <p class="graf graf--p">
   雨伞也被广泛用于抵制监视摄像机，并防止催泪瓦斯，以及为墙壁上的绘画标语提供遮盖。最近几周，抗议者策略性地使用激光笔来遮挡监视摄像头并分散警察的注意力。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8957 jetpack-lazy-image" data-lazy-sizes="(max-width: 727px) 100vw, 727px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5-6.jpg?resize=727%2C500&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5-6.jpg?w=727&amp;ssl=1 727w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5-6.jpg?resize=300%2C206&amp;ssl=1 300w" data-recalc-dims="1" height="500" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5-6.jpg?resize=727%2C500&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="727"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8957" data-recalc-dims="1" height="500" sizes="(max-width: 727px) 100vw, 727px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5-6.jpg?resize=727%2C500&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5-6.jpg?w=727&amp;ssl=1 727w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5-6.jpg?resize=300%2C206&amp;ssl=1 300w" width="727"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   这位22岁的志愿者表示，这些想法是通过在线讨论形成的。
  </p>
  <p class="graf graf--p">
   抗议者彼此提供建议，例如穿着相同的服装和在鞋子上贴上标签，以便彼此认识；一些人在护目镜上附有反光带。
  </p>
  <p class="graf graf--p">
   使用反光材料逃避监察不只是在香港被探讨。2016年，美国艺术家斯科特城市（Scott Urban）设立了一个 Kickstarter 页面，为他的反监视太阳镜 Reflectacles 提供众筹基金。
  </p>
  <p class="graf graf--p">
   眼镜由反射红外光的材料制成，这意味着框架在监控录像中显示为白光闪烁。Urban 表示，由于最近的抗议活动，他的网站因来自香港的点击率飙升而变得火爆。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我不是在试图兜售一种产品，”Urban 在电话采访中说， “我只是想告诉别人，当你的面孔被用来识别你的身份时，你将在任何公共场所不断被追踪。”
  </p>

   视频：

  <p>
   <iframe allowfullscreen="allowfullscreen" height="618" src="//player.vimeo.com/video/357012380?title=0&amp;byline=0&amp;portrait=0&amp;color=8dc7dc" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    更广泛的担忧
   </strong>
  </p>
  <p class="graf graf--p">
   那么艺术家和设计师提出的想法能跟上不断改进的技术吗？抗议者能否领先于技术发展？
  </p>
  <p class="graf graf--p">
   英国索伦特大学研究隐私和数字文化的 Garfield Benjamin 表示，虽然概念在提出时可能有效，但由于技术发展如此迅速，它们很快就会过时。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这是一场军备竞赛”。
  </p>
  <p class="graf graf--p">
   面部识别软件的技术已经变得越来越复杂。
  </p>
  <p class="graf graf--p">
   根据卡内基梅隆大学电子与计算机工程和计算机科学教授 Lujo Bauer 的说法，现在有些系统已经可以使用机器学习来模仿人类相互认识的方式。
  </p>
  <p class="graf graf--p">
   其他软件使用红外光来创建人脸的3D模型 — 这是iPhone X上使用的技术。这些示例比简单地测量面部特征之间的距离更先进。
  </p>
  <p class="graf graf--p">
   密歇根州立大学计算机科学教授 Anil Jain 表示，目前，世界各地大多使用的是二维面部识别技术，这种技术无法识别深度。政府可能已经可以访问公民照片的大型数据库 — 例如驾驶执照照片或面部照片 — 他们可以将这些数据与面部识别数据进行比较，以便识别个人。
  </p>
  <p class="graf graf--p">
   据人权观察组织称，在中国大陆，面部识别技术已经产生了寒蝉效应；北京建立了一个大规模监视系统，密切监视维吾尔族和其他少数民族在新疆西部地区的活动。
  </p>
  <p class="graf graf--p">
   在美国和英国，当局也已开始使用面部识别来捕捉嫌疑人，甚至当抓不到嫌疑人时，《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%bb%99%e4%bd%a0%e6%95%b4%e5%ae%b9%ef%bc%8c%e4%bb%a5%e4%be%bf%e7%ac%a6%e5%90%88%e5%ab%8c%e7%96%91%e4%ba%ba%e9%9d%a2%e9%83%a8%e7%89%b9%e5%be%81/" href="https://www.iyouport.org/%e7%bb%99%e4%bd%a0%e6%95%b4%e5%ae%b9%ef%bc%8c%e4%bb%a5%e4%be%bf%e7%ac%a6%e5%90%88%e5%ab%8c%e7%96%91%e4%ba%ba%e9%9d%a2%e9%83%a8%e7%89%b9%e5%be%81/" rel="noopener noreferrer" target="_blank">
    给你’整容’，以便符合嫌疑人面部特征
   </a>
   》。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们已经走到这一步了（在美国），这些技术在全国范围内推广，没有透明度，没有指导方针，没有法规，公众对实际技术如何运作知之甚少，”市议员 Ben Ewen 说。
  </p>
  <p class="graf graf--p">
   使用面部识别软件的监视依赖于大规模铺设的基础设施；如果没有庞大的摄像头网络捕捉公民的图像，该技术就无法实施。
  </p>
  <p class="graf graf--p">
   根据 Comparitech 的最新研究，
   <strong class="markup--strong markup--p-strong">
    在全球范围内，这种视频监控正在“崛起”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   消费者技术网站发布的一份报告发现，世界上监视最多的10个城市中有8个在中国。伦敦和亚特兰大跃进了前十名，新加坡、悉尼和德里排名前20。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8958 jetpack-lazy-image" data-lazy-sizes="(max-width: 727px) 100vw, 727px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-3.jpg?resize=727%2C462&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-3.jpg?w=727&amp;ssl=1 727w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-3.jpg?resize=300%2C191&amp;ssl=1 300w" data-recalc-dims="1" height="462" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-3.jpg?resize=727%2C462&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="727"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8958" data-recalc-dims="1" height="462" sizes="(max-width: 727px) 100vw, 727px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-3.jpg?resize=727%2C462&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-3.jpg?w=727&amp;ssl=1 727w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-3.jpg?resize=300%2C191&amp;ssl=1 300w" width="727"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在香港，监视措施令抗议者感到担忧。示威者试图强行拆除一些智能灯柱，并由此与防暴警察发生了冲突。
  </p>
  <p class="graf graf--p">
   其他抗议者担心该市的闭路电视摄像机，尽管香港政府表示该市的闭路电视摄像机“没有配备”自动面部识别（AFR）技术。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8959 jetpack-lazy-image" data-lazy-sizes="(max-width: 727px) 100vw, 727px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-1.jpg?resize=727%2C484&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-1.jpg?w=727&amp;ssl=1 727w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-1.jpg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-1.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="484" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-1.jpg?resize=727%2C484&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="727"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8959" data-recalc-dims="1" height="484" sizes="(max-width: 727px) 100vw, 727px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-1.jpg?resize=727%2C484&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-1.jpg?w=727&amp;ssl=1 727w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-1.jpg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-1.jpg?resize=272%2C182&amp;ssl=1 272w" width="727"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   但香港抗议者不仅关注被警方逮捕的短期问题。许多人担心未来，因为允许某些自由和自治的 “一国两制” 政策将在2047年到期。
  </p>
  <p class="graf graf--p">
   一名20岁的学生抗议者在炎热的抗议期间在阴凉处休息的时候也戴着面罩，即使周围并没有警察。⚪️
  </p>
  <p class="graf graf--p">

    Why protests are becoming increasingly faceless
