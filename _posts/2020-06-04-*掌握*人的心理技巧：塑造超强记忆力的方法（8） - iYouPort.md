  <p class="graf graf--p">
   人脸识别并不那么可怕，可怕的是，您的脸和您的**一切**相关联的结果：您的风格、您的爱好、您的习惯、您的心理弱点、您的政治偏好、您的信仰 …… 后面这一大堆才是关键。换句话说，
   <strong class="markup--strong markup--p-strong">
    如果您能保住后面这一堆东西尽可能对陌生人不可见
   </strong>
   ，那么，他们拿走您的脸、您的指纹、虹膜、步态等等一切，也很难直接对您构成威胁。
  </p>
  <p class="graf graf--p">
   有几种情况也许你可能已经遇到过：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    一场激烈的突发事件后，目击证人不论如何都想不起肇事者长什么样，无法协助破案；
   </li>
   <li class="graf graf--li">
    一个陌生人走过来借打火机，然后你发现自己的手机不见了，这才意识到刚才那个人是小偷，却怎么都想不起他长什么样子；
   </li>
   <li class="graf graf--li">
    你组织一个比较大的团队一起工作，却总是难以为每个人合理地安排任务，因为你记不住他们各自擅长什么、有哪些心理特点和习惯；
   </li>
   <li class="graf graf--li">
    你遭遇了一个可怕的政府间谍的滋扰，却难以准确地描绘此人，以至于无法帮助你的队友防范攻击；
   </li>
   <li class="graf graf--li">
    ……
   </li>
  </ul>
  <p class="graf graf--p">
   还有很多。
  </p>
  <p class="graf graf--p">
   👉对人的记忆和对事的记忆同样重要 —— 尤其是在行动中，因为您需要与大量的人打交道，“脸盲” 不仅仅是个笑话，甚至会有生命危险。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    不论是团队组织还是个人防御，快速准确地记忆人的能力都是必备的基础。当然您还需要一些心理学技巧，也就是
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     社交工程
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p">
   这里是 “超强记忆里” 第8集，主题就是帮助您记忆人，以及掌握人的心理技巧。
  </p>
  <p class="graf graf--p">
   👉记忆人不仅是记住面孔和名字，还包括他们的风格、他们喜欢什么、他们的偏好习惯、弱点、过去的经历等等 —— 所有这一切都需要在您的大脑记忆库里
   <strong class="markup--strong markup--p-strong">
    编写易于索引提取的文档
   </strong>
   。
  </p>
  <p class="graf graf--p">
   对于 *快速识别和利用* 来说，只记住名字可能没有用，名字只是一个标签，面部可以化妆掩饰，而一个人的潜在风格个性很难长时间地掩饰。如果您的大脑里有上述文档，就很容易能识破伪装。
  </p>
  <p class="graf graf--p">
   人类是社会性动物，我们每个人都有一定程度的识别人脸的遗传能力，但
   <strong class="markup--strong markup--p-strong">
    记住对方的偏好习惯、特征、和弱点，可能需要付出一些努力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   👉记住这些信息的最好方法是将它们与对方的外表特征联系起来。
  </p>
  <p class="graf graf--p">
   仔细观察一个人。注意他/她不同寻常的特征。可以是任意的细节，
   <strong class="markup--strong markup--p-strong">
    尤其是难以掩盖的细节
   </strong>
   。比如，前额是很高还是很低，是否有独特的鼻子或耳朵的形状、眼间距是否很宽或很窄、下巴上有浅凹吗、哪个部位有痣或者疤痕？
  </p>
  <p class="graf graf--p">
   然后编写一个将其名字与这些特征相联系的故事。为了更好地记住这个故事和这个人，你需要确立你的情感态度：你喜欢他/她的主要特征吗？为什么？
  </p>
  <p class="graf graf--p">
   第一次见到某人时，看他的脸，仔细倾听他说话。将他的外表与不寻常的特征进行形象化描述。
  </p>

   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习
    </strong>

   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     语言描述
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   人物肖像描述法是19世纪时法国犯罪学家 Alphonse Bertillon 发明的，它为侦查学带来了彻底的改变。
  </p>
  <p class="graf graf--p">
   这些技术有助于识别一个特定的人，找到他，并且确认其在犯罪现场的在场证明。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    👉学习如何创作语言肖像是非常有用的能力。它有助于确认您在分析人物外表时应该注意的要点。这样就能帮助您更好地记住某个人
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在行动中
   </strong>
   尽可能不要保存和传播目标人的照片，这样会有很大风险，但是你可以轻松地使用语言来描述出对方的肖像 —— 准确的语言肖像描述可以帮助您和队友
   <strong class="markup--strong markup--p-strong">
    避免落入圈套
   </strong>
   ，比如对方假扮本应与你会面的人的身份联络你……
  </p>
  <p class="graf graf--p">
   现在语言描述系统基于
   <strong class="markup--strong markup--p-strong">
    四种类型的特征
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    解剖学特征：描述人体 —— 性别、年龄、身高等等；
   </li>
   <li class="graf graf--li">
    动态特征：描述动态 —— 姿势、面部表情等；
   </li>
   <li class="graf graf--li">
    区别性特征：个体特征 —— 疤痕、身体部位残缺、跛行等；
   </li>
   <li class="graf graf--li">
    附加特征：服装品味、配饰特点等等。
   </li>
  </ul>
  <p class="graf graf--p">
   语言描述不仅仅是针对面部和外表的任意描述，而是针对每个特征的一组有限的参数。
  </p>
  <p class="graf graf--p">
   比如，一个人额头的轮廓可以是斜的、直的、或者外凸的；因此，
   <strong class="markup--strong markup--p-strong">
    与其说一个人是被描述出来的，不如说此人是被 **归类** 出来的。这种方法从根本上简化了对他人语言描述的理解。
   </strong>
  </p>

  <p class="graf graf--p">
   构建肖像时使用上文中的语言系统，然后和你的朋友互换角色进行练习。
  </p>
  <p class="graf graf--p">
   👉当您可以对自己的描述的准确度满意的时候，开始在记忆文档中加入目标人的出生日期和职业、性格特点等信息。
   <strong class="markup--strong markup--p-strong">
    尽可能记住你能记住的一切，因为接下来的训练内容将需要您回忆这些信息。
   </strong>
  </p>
  <p class="graf graf--p">
   生日问候是与您感兴趣的人保持联系的好借口。
  </p>
  <p class="graf graf--p">
   在生活中努力使用这套方法吧，你会感激自己的记忆力的，因为你永远不知道自己会在何时、何地可能需要利用这样的联系。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff0000;">
    <strong class="markup--strong markup--p-strong">
     测试！快速回答：
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    埃里克·芬克是在哪里去世的？
   </span>
  </p>
  <p class="graf graf--p">
   A : 纳粹德国空军医院
  </p>
  <p class="graf graf--p">
   B：达豪集中营
  </p>
  <p class="graf graf--p">
   C：柏林
  </p>
  <p class="graf graf--p">
   D：诺伊施塔特

  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习 —— 书桌游戏（等级3）
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   记住一堆杂乱物品的具体位置，可以训练你的记忆力和注意力。培养自己的观察技能，并组织你的思维。
  </p>
  <p class="graf graf--p">
   试着在现实生活中不断做这个练习，比如 记住书架上的书籍摆放的位置，书桌上物品的排列位置，以及停车场里不同汽车停放的具体位置。等等。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     建立联系
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   游击队的反情报工作依赖于团队的敏锐。对员应该能熟练地以不同方式与完全不同的人沟通。活动家也一样，他们需要与更多人交谈、不断需要获得关注和支持 —— 这样才能扩展你的行动能力。
  </p>
  <p class="graf graf--p">
   通常情况下，第一次会面会有很大压力，因为你不知道对方的真心所想、对你和你的组织如何认识，甚至，对方是否敌方的间谍。随时都有危险。
  </p>
  <p class="graf graf--p">
   👉这个时候最重要的事之一就是树立信心，缓解内心的焦虑和紧张。要做到这一点，你就需要尽可能进行不具威胁性的对话，在彼此都能感觉安全的地方进行交流。
  </p>
  <p class="graf graf--p">
   对方的兴趣爱好是一个非常好的话题。每个人都有自己会喋喋不休一直谈论的主题，
   <strong class="markup--strong markup--p-strong">
    你要做到就是迅速找到对方的兴趣点，这是走入对方内心的路
   </strong>
   。
  </p>
  <p class="graf graf--p">
   爱好可以是普通的，比如邮票、国际象棋、摄影、足球、占星学等等；也可以是罕见的，比如手绘艺术、种植稀有植物、探险经历等。
  </p>
  <p class="graf graf--p">
   👉如果你发现了对方的兴趣点，你要做的是尽可能少插话，耐心地认真地倾听；你的微笑的专注是给对方最好的鼓励。
  </p>
  <p class="graf graf--p">
   如果你还未能了解对方的兴趣爱好，那就在进行对话之前试着猜一下对对方来说有可能比较重要的事。可以尝试从他的职业入手；比如，你可以和工程师谈汽车、和教师谈 “如今的年轻人”等等。
  </p>
  <p class="graf graf--p">
   ⚠️请记住，人们最喜欢谈论的是他们自己，而不是你所关心的东西 —— 你能够走入对方内心的途径就在这里。彼此都滔滔不绝的对话是基本无用的，你必须能做到仔细倾听，保持对话的平稳延续，这样才能尽快与对方建立信任，之后对方才会对你说的话感兴趣。
  </p>
  <p class="graf graf--p">
   还记得我们在 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a4%a7%e4%bf%9d%e5%81%a5%e5%ba%94%e8%af%a5%e6%88%90%e4%b8%ba%e4%b9%a0%e6%83%af%ef%bc%9a%e5%8f%98%e5%be%97%e9%9a%be%e4%bb%a5%e8%a2%ab%e8%bf%bd%e8%b8%aa%e7%9a%84%e7%ae%80%e5%8d%95%e6%96%b9%e6%b3%95/" href="https://www.iyouport.org/%e5%a4%a7%e4%bf%9d%e5%81%a5%e5%ba%94%e8%af%a5%e6%88%90%e4%b8%ba%e4%b9%a0%e6%83%af%ef%bc%9a%e5%8f%98%e5%be%97%e9%9a%be%e4%bb%a5%e8%a2%ab%e8%bf%bd%e8%b8%aa%e7%9a%84%e7%ae%80%e5%8d%95%e6%96%b9%e6%b3%95/" rel="noopener noreferrer" target="_blank">
    变得难以被追踪
   </a>
   ” 系列中强调过的吗：
   <strong class="markup--strong markup--p-strong">
    “您越是安静，听到的声音就越多。”
   </strong>
  </p>
  <p class="graf graf--p">
   在试图与对方建立信任感的时候，不要害怕讨论陌生的话题。坦白地承认自己的无知，多问多听，这样反而能激发对方的自豪感，从而让对方更喜欢你。
  </p>
  <p class="graf graf--p">
   👉要发自内心地对对方所谈论的内容感兴趣。如果你的提问不够真诚，对方会第一时间意识到（识别虚情假意这是每个人都有的天赋），这样你会很尴尬，补救它会非常难。
  </p>
  <p class="graf graf--p">
   安全的对话可以缓解焦虑，营造轻松亲密的氛围，从中，你可以迅速对交谈对象有一个
   <strong class="markup--strong markup--p-strong">
    深透的了解
   </strong>
   。
  </p>
  <p class="graf graf--p">
   👉记住，你要做的就是引导对方充分表现出他们自己的骄傲感、他们的优势、他们比你更强的地方，让对方在你面前处于这种心理环境下 ——
   <strong class="markup--strong markup--p-strong">
    他们就会滔滔不绝，他们的弱点就会由此而暴露。
   </strong>
  </p>
  <p class="graf graf--p">
   总之，先走进对方心里，再提出你的想法。这个顺序不可变动，否则无效。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   生活中有很多流于形式的和表面化的交流，比如出租车司机、店员、理发师、保安等等，他们说的话几分钟后你就会忘掉，当然他们自己也不会记得你。
  </p>
  <p class="graf graf--p">
   👉但是，你完全可以利用这样闲散轻松的对话机会，培养自己与他人建立信任感的能力。这种能力在几乎任何时候都是极大的优势。
  </p>
  <p class="graf graf--p">
   试着与他人攀谈，尝试猜测他们的爱好。比如，与司机攀谈，询问他们的经历；与理发师攀谈，询问他们是如何学会剪头发的；再比如，称赞小店里的某个商品，店主会迅速对你感兴趣。
  </p>
  <p class="graf graf--p">
   就算你没有从谈话中获得任何想要的东西，你也没有任何损失；但是，你肯定能获得宝贵的经验，甚至可以学到很多有趣的东西。
  </p>
  <p class="graf graf--p">
   ⚠️ 骄傲，这是绝大多数人都扛不住的心理弱点。只要你能让对方自我感觉良好，把你自己变成一个谦逊的渴望新知的小学生，这样你就能很容易得到你想要的东西。
  </p>
  <p class="graf graf--p">
   稍过一段时间，你就可以自如地引入你自己的主题了。⚪️
  </p>
  <p class="graf graf--p">

    —— 未完待续 ——

  </p>
