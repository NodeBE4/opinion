  <p class="p1">
   <span class="s1">
    据
    <a href="http://www.nationaldefensemagazine.org/articles/2019/5/17/news-from-itec-armys-google-earth-on-steroids-to-include-inside-of-buildings" rel="noopener noreferrer" target="_blank">
     国防杂志报道
    </a>
    ，
   </span>
   <span class="s2">
    5
   </span>
   <span class="s1">
    月
   </span>
   <span class="s2">
    15
   </span>
   <span class="s1">
    日在瑞典斯德哥尔摩举行的
   </span>
   <span class="s2">
    IEEE
   </span>
   <span class="s1">
    交通电气化展览会（
   </span>
   <span class="s2">
    ITEC
   </span>
   <span class="s1">
    ）
   </span>
   <span class="s2">
    2019
   </span>
   <span class="s1">
    年会议上公布了一项预计将为美国作战人员改造训练和模拟练习的新地图技术。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    南加州大学创新技术研究所地理空间科学与技术主任
   </span>
   <span class="s2">
    Jason Knowles
   </span>
   <span class="s1">
    在陆军附属研究中心发表演讲，他谈到了新的地形捕获和重建软件，可以重现复杂的环境，包括为模拟演习和战争规划的城市。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    南加州大学创新技术研究所地理空间科学与技术主任
   </span>
   <span class="s2">
    Jason Knowles
   </span>
   <span class="s1">
    在陆军附属研究中心发表演讲，他谈到了新的地形捕获和重建软件，可以重现复杂的环境，包括为模拟演习和战争规划使用的城市。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    该研究所是一个致力于绘图软件的项目
   </span>
   <span class="s1">
    （称为
   </span>
   <span class="s2">
    One World Terrain
   </span>
   <span class="s1">
    （
   </span>
   <span class="s2">
    OWT
   </span>
   <span class="s1">
    ））的跨职能团队的一部分。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    Knowles
   </span>
   <span class="s1">
    将新软件描述为
   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">
    超级版谷歌地球
   </span>
   <span class="s2">
    ”
   </span>
   <span class="s1">
    。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    在
   </span>
   <span class="s2">
    ITEC
   </span>
   <span class="s1">
    期间的一次简报中，
   </span>
   <span class="s2">
    Knowles
   </span>
   <span class="s1">
    向观众展示了一个敌人基地的图片，该基地被使用商业软件和小型无人机在大约一小时内捕获，并进行了数字重建。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    他说：
   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">
    我们能够把
   </span>
   <span class="s2">
    UAS
   </span>
   <span class="s1">
    扔掉了，在一小时内捕获它，把它放在笔记本电脑上，轻松处理、推出。
   </span>
   <span class="s2">
    ”
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    Knowles
   </span>
   <span class="s1">
    说，
    <b>
     该软件可以
    </b>
   </span>
   <span class="s2">
    <b>
     “
    </b>
   </span>
   <span class="s1">
    <b>
     逐层剥离建筑物的外部，看清建筑物内部是什么
    </b>
    。这对操作人员来说显然非常有用
   </span>
   <span class="s2">
    ”
   </span>
   <span class="s1">
    。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    他说该软件与
   </span>
   <span class="s2">
    GPS
   </span>
   <span class="s1">
    数据相关联，因此战争规划者可以用它组织未来的实际操作。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    快速的
   </span>
   <span class="s2">
    3D
   </span>
   <span class="s1">
    地形捕获和数字重建系统由来自卫星和飞机的航空图像支持。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    他补充说，为了获得更高的分辨率，侦察小组可以部署小型便捷无人机来收集更清晰的图像。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    该软件使用机器学习和人工智能
   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">
    使模型变得更聪明，因此它不仅仅是在
   </span>
   <span class="s2">
    [
   </span>
   <span class="s1">
    识别对象
   </span>
   <span class="s2">
    ]
   </span>
   <span class="s1">
    图片，
   </span>
   <span class="s2">
    ”
   </span>
   <span class="s1">
    他说。
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    例如，
    <b>
     它可以告诉部队敌方基地的围墙是否需要用车辆或弹药进行破坏
    </b>
    。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    除了远程精确导弹、下一代战斗车、未来升降库、空中导弹防御、定向能武器、下一代战斗步枪、和士兵杀伤力之外，新的测绘软件是陆军最重要的现代化优先事项之一。
   </span>
  </p>
