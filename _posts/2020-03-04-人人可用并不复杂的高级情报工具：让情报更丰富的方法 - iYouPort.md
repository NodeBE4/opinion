  <p class="graf graf--p">
   许多执行开源情报调查的人员使用非常基本的方法来提取信息。通常，它们仅限于具有图形用户界面（GUI）的工具。
  </p>
  <p class="graf graf--p">
   当然，使用图形用户界面的许多工具（例如 TweetBeaver）都非常强大。
  </p>
  <p class="graf graf--p">
   这篇文章肯定不是要抹黑使用GUI的任何工具，其目的是向依赖GUI的工具的用户介绍能产生更多
   <strong class="markup--strong markup--p-strong">
    可自定义结果
   </strong>
   的工具。
  </p>
  <p class="graf graf--p">
   下面即将介绍的是2种开源情报挖掘工具，都很简单；但它们可以利用中级计算机知识来产生高级调查结果。

  <p class="graf graf--p">
   Photon 自称为 “专为侦查设计的极快的爬虫”。
  </p>
  <p class="graf graf--p">
   也就是说，“侦查”一词表明这是用于渗透测试的工具。
  </p>
  <p class="graf graf--p">
   当然，渗透测试不是唯一的用例。除了渗透测试之外，Photon 还有许多其他
   <strong class="markup--strong markup--p-strong">
    情报挖掘方面的用途
   </strong>
   。
  </p>
  <p class="graf graf--p">
   以下是它可以做的：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、它可以在几秒钟内从网站上提取所有内部和外部链接
   </strong>
  </p>
  <p class="graf graf--p">
   如果您想知道一个网站是否链接到另一个网站或引用了某些资料，它可以在几秒钟内帮您找出来。
  </p>
  <p class="graf graf--p">
   如果您想知道网站上有多少个坏链（查找是否对网站进行了主动管理），请通过坏链检查器运行，同样可以在几秒钟内找到。
  </p>
  <p class="graf graf--p">
   如果您想监视一段时间内的变化，请立即运行 Photon，然后在三个月内再次运行它，以衡量差异。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、它可以在几秒钟内提取所有电子邮件、社交媒体帐户和亚马逊存储桶
   </strong>
  </p>
  <p class="graf graf--p">
   这是不言自明的。如果您要进行调查追踪，那就运行 Photon 吧。
  </p>
  <p class="graf graf--p">
   通过简单的查询就可以找到所有联系点。尤其是，⚠️如果您要进

    社交工程
   </a>
   攻击前的审计，几秒钟内就可以提取所有目标相关的信息。
  </p>

   </strong>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9990 jetpack-lazy-image" data-lazy-sizes="(max-width: 887px) 100vw, 887px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-17.png?resize=887%2C390&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-17.png?w=887&amp;ssl=1 887w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-17.png?resize=300%2C132&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-17.png?resize=768%2C338&amp;ssl=1 768w" data-recalc-dims="1" height="390" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-17.png?resize=887%2C390&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="887"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9990" data-recalc-dims="1" height="390" sizes="(max-width: 887px) 100vw, 887px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-17.png?resize=887%2C390&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-17.png?w=887&amp;ssl=1 887w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-17.png?resize=300%2C132&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-17.png?resize=768%2C338&amp;ssl=1 768w" width="887"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   GasMask 自称为 “一站式开源情报信息收集工具”。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.twelvesec.com/" href="https://www.twelvesec.com/" rel="noopener noreferrer" target="_blank">
    twelvesec
   </a>
   制作，功能非常强大。
  </p>
  <p class="graf graf--p">
   同样，GasMask 的目标受众是渗透测试人员，如果您就是，它对您来说会是完美的。当然，它的功能可以有很多其他用例，包括个人防御。
  </p>
  <p class="graf graf--p">
   如果您已经学到了使用Photon的方法，那么此过程实际上是相同的。
  </p>
  <p class="graf graf--p">
   那么 GasMask 可以做什么？这里是清单。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、多索引搜索
   </strong>
  </p>
  <p class="graf graf--p">
   您可以同时通过以下这些索引运行域名调查：
  </p>
  <pre class="graf graf--pre"><code>basic,whois,dns,revdns,vhosts,google,bing,yahoo,ask,dogpile,yandex,censys,linkedin,twitter,googleplus,youtube,reddit,github,instagram,crt,pgp,netcraft,virustotal,dnsdump</code></pre>
  <p class="graf graf--p">
   您甚至可以使用 Shodan API 密钥与 Shodan 进行交互。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、除了渗透测试之外还能怎么用？
   </strong>
  </p>
  <p class="graf graf--p">
   如果您正在进行调查，并发现目标对象有一个博客，那么想查看它是否已在网上其他任何地方（包括社交媒体）被编入索引，请将其插入GasMask。
  </p>
  <p class="graf graf--p">
   如果您发现一个托管目标信息的网站，则可以使用 GasMask 在网上其他地方进行跟踪。
  </p>
  <p class="graf graf--p">
   如果您想跟踪
   <a class="markup--anchor markup--p-anchor" data-href="https://storage.googleapis.com/qurium/iyouport.org/category-%e5%bf%83%e7%90%86%e6%88%98%e5%92%8c%e4%bf%a1%e6%81%af%e6%88%98-1.html" href="https://storage.googleapis.com/qurium/iyouport.org/category-%e5%bf%83%e7%90%86%e6%88%98%e5%92%8c%e4%bf%a1%e6%81%af%e6%88%98-1.html" rel="noopener noreferrer" target="_blank">
    宣传战
   </a>
   的轨迹，则可以通过 GasMask 运行一个目标假消息站点的追踪。
  </p>
  <p class="graf graf--p">
   如果您想寻找抢注或误植域名的用户，请通过 GasMask 运行域跟踪。
  </p>
  <p class="graf graf--p">
   这里是一个小演练的视频（背景音乐很糟糕，你可以静音看）：
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/dVEP5rHZEAw" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   如果您想了解更多信息，请访问他们的GitHub页面：
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://github.com/twelvesec/gasmask" href="https://github.com/twelvesec/gasmask" rel="noopener noreferrer" target="_blank">
    GasMask
   </a>
  </p>
  <p class="graf graf--p">
   您可以使用这些方法检验自己的安全漏洞，也可以组建安全防御小组，或者成为开源情报调查小组的一员。
  </p>
  <p class="graf graf--p">
   总之还是那句话：
   <a class="markup--anchor markup--p-anchor" data-href="https://storage.googleapis.com/qurium/iyouport.org/category-osint-1.html" href="https://storage.googleapis.com/qurium/iyouport.org/category-osint-1.html" rel="noopener noreferrer" target="_blank">
    相关工具
   </a>
   攻防皆可用，必须用在正义的目的上。⚪️
  </p>
