  <p class="graf graf--p">
   中兴通讯（000063.SZ）于3月份与 Jujuy 签订了
   <strong class="markup--strong markup--p-strong">
    价值近3000万美元
   </strong>
   的监控合同，提供包括监控摄像头、监控中心、应急服务和电信基础设施。三年前中兴通讯首次在该国推出了销售宣传。
  </p>
  <p class="graf graf--p">
   这笔交易是中国在拉丁美洲取得技术进步的又一个例子。
  </p>
  <p class="graf graf--p">
   布宜诺斯艾利斯的一位中国官员告诉路透社，Jujuy 项目可以通过
   <strong class="markup--strong markup--p-strong">
    鼓励其他城市采用类似的监视技术
   </strong>
   来帮助中国扩大其在拉美的科技足迹。
  </p>
  <p class="graf graf--p">
   延伸：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%ad%e5%9b%bd%e8%80%81%e5%a4%a7%e5%93%a5%e7%9b%af%e4%b8%8a%e6%8b%89%e7%be%8e%e4%ba%ba/" href="https://www.iyouport.org/%e4%b8%ad%e5%9b%bd%e8%80%81%e5%a4%a7%e5%93%a5%e7%9b%af%e4%b8%8a%e6%8b%89%e7%be%8e%e4%ba%ba/" rel="noopener noreferrer" target="_blank">
    中国老大哥盯上拉美人
   </a>
   》
  </p>
  <p class="graf graf--p">
   美国国务院西半球事务局发言人在一份声明中说，华盛顿对中兴通讯在胡胡伊的项目表示“担忧”。
  </p>
  <p class="graf graf--p">
   美国称其对盟国发出了警告，但是几乎没有提供任何证据，阿根廷和中国官员在对路透社的采访和声明中淡化了对胡胡伊项目的担忧。
  </p>
  <p class="graf graf--p">
   在阿根廷占有一席之地的中兴通讯拒绝分享其在拉丁美洲监控技术项目的公司数据，也没有对此事进行评论。
  </p>
  <p class="graf graf--p">
   5月份在首府圣萨尔瓦多德胡胡伊举行的一次活动中，中兴通讯在阿根廷的总经理 Dennis Wang 与他的安全部长和警察官员 Gerardo Morales 一起做了演讲，并介绍了该技术如何帮助降低中国的犯罪率。
  </p>
  <p class="graf graf--p">
   胡胡伊早已与中国关系密切。一家中国公司在该省大量投资锂矿开采，中国公司还为一个巨大的太阳能发电厂提供融资和技术，这是南美洲最大的太阳能发电场。
  </p>
  <p class="graf graf--p">
   现在，胡胡伊可以“像中国一样安全”，当地政府在有关此事件的公告中这样写道。
  </p>
  <p class="graf graf--p">
   中国较低的犯罪率部分是由于其庞大且有争议的监视器、摄像机、面部识别软件、公民数据库和由执政的共产党控制的高定罪率的法院。
  </p>
  <p class="graf graf--p">
   人权监督机构表示，中国滥用这些权力压制其本国公民的异议和抗议。
  </p>
  <p class="graf graf--p">
   中国在新疆西北部地区建立了广泛的视频监控系统，并安装了智能手机监控技术，在这里，有超过一百万的维吾尔族和其他穆斯林被关押在集中营。
  </p>
  <p class="graf graf--p">
   根据包括新疆监控基础设施在内的采购文件显示，
   <strong class="markup--strong markup--p-strong">
    中兴通讯及其子公司积极参与建设中国的内部监控项目。
   </strong>
  </p>
  <p class="graf graf--p">
   布宜诺斯艾利斯的中国官员认为美国的担忧是对中国企业成功的“焦虑”，尤其是中兴通讯和华为技术有限公司[HWT.UL]等公司正在争夺5G电信技术的主导权。
  </p>
  <p class="graf graf--p">
   这位不愿透露姓名的官员表示，拉丁美洲国家对提高安全性的需求日益增长，
   <strong class="markup--strong markup--p-strong">
    为中国创造了以具有竞争力的价格销售监控技术的机会。
   </strong>
  </p>
  <p class="graf graf--p">
   阿根廷对这笔交易没有任何担忧，中国被视为一个主要的贸易伙伴和金融家 —— 被视为一个重要的盟友，而不是一个威胁。
  </p>
  <p class="graf graf--p">
   胡胡伊官员有理由对经济实惠的中国科技保持开放。该省是阿根廷最贫穷的省份之一，犯罪率略高于全国平均水平。
  </p>
  <p class="graf graf--p">
   安全部长 Ekel Meyer 在圣萨尔瓦多德胡胡伊的一次采访中说，居民“接受了”安全摄像头的监视，以“换取”更安全的街道。
  </p>
  <p class="graf graf--p">
   他希望通过将监视系统扩展到包括
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab/" href="https://www.iyouport.org/category/%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab/" rel="noopener noreferrer" target="_blank">
    面部识别技术
   </a>
   来进一步发展，这是中国作为其国内监控网络的一部分而推出的重要工具。
  </p>
  <p class="graf graf--p">
   政府表示，中兴通讯已在当地公共城市公园内建立了一个小型试验区，该项目第一阶段的出货于9月到达。
  </p>
  <p class="graf graf--p">
   一位了解 Jujuy 项目的消息人士表示，政府官员没有对该技术的漏洞进行任何形式的评估。
  </p>
  <p class="graf graf--p">
   联合国数据隐私问题特别报告员约瑟夫·卡纳塔西（Joseph Cannataci）在最近访问阿根廷期间告诉路透社，该国需要更好地审查其购买的安全设备和软件。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我不在乎你从谁那里购买它……你不应该认为你所购买的供应商是完全值得信任的。你应该进行隐私影响评估”，他说，指的是联合国推荐的测试，以确保人们的隐私权不受侵犯。
  </p>
  <p class="graf graf--p">
   在委内瑞拉，中兴通讯还帮助总统尼古拉斯·马杜罗（Nicolas Maduro）政府推出了名为“祖国卡”的智能ID卡。
  </p>
  <p class="graf graf--p">
   持卡人的数据会传输到中兴通讯提供的服务器上，并且此卡越来越多地被政府与补贴食品、健康服务和其他社会计划联系起来。
  </p>
  <p class="graf graf--p">
   在厄瓜多尔，华为和国有的中国电子进出口公司于2011年为该国政府建立了4000多台摄像机的监控系统。
  </p>

   <li class="graf graf--li">

   在乌拉圭，政府于2月份宣布中国捐赠了2,100台华为监视摄像机。安装在四个城市。
  </p>
