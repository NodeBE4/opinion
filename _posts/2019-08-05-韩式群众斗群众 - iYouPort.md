  <p class="graf graf--p">
   朝鲜政府已宣布将为非法活动举报提供奖励，以便更密切地监视其公民。而不是激励人们互相窃听，许多人对这个想法表示不满。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “省警察局于3月24日召开会议，宣布他们将开始向那些举报非法的电话用户提供奖励，包括指责国家或政府的人、走私者和吸毒者的人，” 来自平壤的消息人士说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “想要获得奖励的人可以提交举报，然后[政府就可以]消除那些充满不满情绪的人，”消息人士说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “通过每个城市、县和地区的警察局接收群众的举报材料，”消息人士称，“奖励可以从50,000到300,000朝鲜元（约6美元到7美元）不等。”
  </p>
  <p class="graf graf--p">
   消息来源表示，这个想法过去已经尝试过，但并不是很受欢迎。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “它运作得不好。国家甚至不能保证人民的基本生活条件，人民不能在不违法的情况下自谋生活，那么为什么有人想举报其他人呢？“消息人士说。
  </p>
  <p class="graf graf--p">
   消息人士称，这些奖励是政府努力阻止对美国和联合国制裁带来的当前经济状况的不满的一部分，这种制裁使每个人的生活变得困难。
  </p>
  <p class="graf graf--p">
   制裁旨在剥夺该政权可用于支持其核计划和导弹计划的资源。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “政府意识到[金正恩]如何处理人民对[经济状况]的不满，因此中央计划利用奖励制度来消除那些公开对国家表达不满的人，”消息人士说。
  </p>
  <p class="graf graf--p">
   来自咸镜北道的第二个消息来源也表示，奖励制度是政府采取措施阻止不同意见。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “自从最近在越南与美国举行的首脑会议以及议会选举[选民只能选出一名候选人]失败以来，人们真的感到失望，并公开表达自己的想法，”消息人士说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “[这就是为什么]他们推出了奖励举报的制度，以维持社会秩序，并在太阳日之前让抱怨者保持沉默，这是朝​​鲜前领导人金日成诞辰的全国性假日“。
  </p>
  <p class="graf graf--p">
   该消息人士称，有些人认为该奖励制度的实施表明了体制内部的摩擦。“一些居民认为存在安全风险或严重的内部情况，当有重要的国家事务时，他们通常就会开始发放这些奖励。这只能意味着焦虑增加了”。
  </p>
  <p class="graf graf--p">
   朝鲜经常利用其人民相互监视以保持一致。当地政府将每个公民分配给一个 inminban（邻里监视单位），其中成员负责监视对方是否有任何政治不服从或犯罪行为的迹象。这在中国也一样，更多详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://iyouport.weebly.com/short-stories-enthusiast/2102523" href="https://www.iyouport.org/%E4%B8%AD%E5%9B%BD%EF%BC%9A%E4%B8%BA%E4%BB%80%E4%B9%88%E8%A6%81%E9%87%8D%E5%90%AF%E7%AC%A8%E6%8B%99%E4%BD%8E%E6%95%88%E7%9A%84%E6%96%AF%E5%A1%94%E8%A5%BF%E5%BC%8F%E7%BE%A4%E4%BC%97%E6%96%97%E7%BE%A4/" rel="noopener noreferrer" target="_blank">
    中国：为什么要重启笨拙低效的斯塔西式群众斗群众？
   </a>
   》
  </p>
  <p class="graf graf--p">
   最重要的是，每个公民必须每周召开一次名为“saenghwal chonghwa”的会议，这是一项团体活动，每个公民都必须承认个人缺点，然后听取同行的批评 — — 毛泽东的批评与自我批评。
  </p>
  <p class="graf graf--p">
   3月中旬的一份RFA报告描述了在与美国峰会失败后，saenghwal chonghwa 如何变得更加严格和侵略性。
  </p>
