  <p class="graf graf--p graf-after--blockquote" id="072b">
   如今所有人应该都还能记得，特朗普上台一开始便召集了所有硅谷巨头的会议，当时这一视频还被中国的社交媒体下载并流传。事实上它是有传统的。而且，并不简单。
  </p>
  <p class="graf graf--p graf-after--p" id="75f2">
   2011年2月，白宫召集14为硅谷巨头参加奥巴马特地为他们举行的晚宴，他们无一缺席。当晚，奥巴马笑称，如果他能再次当选总统，就会任命当时的谷歌公司CEO埃里克施密特（Eric Emerson Schmidt）出任商务部部长。美国媒体将这场聚会称之为
   <a class="markup--anchor markup--p-anchor" data-href="https://www.sfgate.com/bayarea/article/Obama-dines-with-tech-stars-of-Silicon-Valley-2529458.php" href="https://www.sfgate.com/bayarea/article/Obama-dines-with-tech-stars-of-Silicon-Valley-2529458.php" rel="noopener nofollow noreferrer" target="_blank">
    “国王们的晚宴”
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="2ec4">
   当时政府部门还在
   <a class="markup--anchor markup--p-anchor" data-href="http://flickr.fr/" href="http://flickr.fr/" rel="noreferrer noopener nofollow" target="_blank">
    Flickr.Fr
   </a>
   上发布了两张照片 — —
   <strong class="markup--strong markup--p-strong">
    这就是大数据权力的一个重要转折点
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="7a45">
   由信息学家、同时也是预测分析专家 Rayid Ghani（当年仅24岁的埃森哲技术实验室带头人）指定了计划，在硅谷巨头的鼎力协助下，奥巴马在2012年再次赢得了大选。
  </p>
  <p class="graf graf--p graf-after--p" id="fe67">
   其间，50多名信息学专家花了好几个月，在被称为“洞穴”的战场总部秘密大厅闭门不出。他们的工作是处理网络上收集的几十亿甚至上百亿条数据，从中找出有可能支持奥巴马的选民。对于尚未决定投票给谁的公民，科学家们通过大数据把他们
   <strong class="markup--strong markup--p-strong">
    筛选出来并隔离
   </strong>
   ，再用尽全部努力凭借最恰当的演讲对他们进行
   <strong class="markup--strong markup--p-strong">
    针对性游说
   </strong>
   ，使他们转向支持奥巴马。
  </p>
  <p class="graf graf--p graf-after--p" id="7271">
   采用最优化的量身定制的门对门助选，显然得到了与众不同的效果。奥巴马的第一次当选也是在大数据公司的帮助下成功的。
  </p>
  <p class="graf graf--p graf-after--p" id="d1b6">
   这种“竞选”方法同样也启发了奥朗德的团队。在当时的法国大选期间，奥朗德效仿奥巴马的网络战场，建立了某种类似的“法式洞穴”。
  </p>

  </p>
  <p class="graf graf--p graf-after--p" id="0b02">

   <strong class="markup--strong markup--p-strong">
    根据搜索引擎数据的导向性，投票者中37%改变了最初的投票意愿
   </strong>
   。

   </strong>
   ，正因为，如今所有人都在互联网上，网民对网上获得的信息量往往深信不疑，而同时对报纸上和电视上的内容信任度越来越低 — — 人们互相告诫着：“消息是真是假你搜一下就知道了”。但是几乎没人能想到，你有可能搜不到切实的真相。因为算法被不透明的黑手控制着。由此，人们已经得出结论，
   <strong class="markup--strong markup--p-strong">

   </strong>

  </p>
  <p class="graf graf--p graf-after--p" id="306f">

  <p class="graf graf--p graf-after--p" id="3e98">

  </p>

   </strong>
   。

  </p>
