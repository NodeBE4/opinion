      </strong>
     </span>

  <p class="graf graf--p">
   上个月，新加坡航空公司使用他们的应用程序
   <a class="markup--anchor markup--p-anchor" data-href="https://www.fastcompany.com/90303599/some-iphone-apps-are-secretly-recording-your-screen" href="https://www.fastcompany.com/90303599/some-iphone-apps-are-secretly-recording-your-screen" rel="noopener noreferrer" target="_blank">
    偷偷记录
   </a>
   了消费者的手机屏幕。然后，旅行者发现该航空公司还在每把椅子背面的机载娱乐系统中
   <a class="markup--anchor markup--p-anchor" data-href="https://www.fastcompany.com/90310098/someone-found-cameras-in-singapore-airlines-in-flight-entertainment-system" href="https://www.fastcompany.com/90310098/someone-found-cameras-in-singapore-airlines-in-flight-entertainment-system" rel="noopener noreferrer" target="_blank">
    嵌入了监视摄像头
   </a>
   。记者随后发现，座椅靠背监控的蔓延并不仅限于新加坡：
   <a class="markup--anchor markup--p-anchor" data-href="https://www.buzzfeednews.com/article/nicolenguyen/united-delta-airlines-seat-back-screens-cameras?bftwnews&amp;utm_term=4ldqpgc#4ldqpgc" href="https://www.buzzfeednews.com/article/nicolenguyen/united-delta-airlines-seat-back-screens-cameras?bftwnews&amp;utm_term=4ldqpgc#4ldqpgc" rel="noopener noreferrer" target="_blank">
    包括美联航和台达
   </a>
   在内的其他航空公司也都在其座位上安装了摄像头。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    越来越多的不只是消费者技术在暗中监视我们：而是我们身边的普通物体
   </strong>
   。飞机娱乐系统只是许多事情中的一个而已 — — 部分归功于更便宜、更小的硬件 — — 正在内置摄像头和麦克风。
  </p>
  <p class="graf graf--p">
   虽然航空公司声称他们“从未使用过摄像头”，但总是可以打开这些摄像头。在一个隐私规则无限宽松的世界中，公司们倾向于尽可能深度地侵犯您的隐私，这就是监控经济本身的大方向。
   <strong class="markup--strong markup--p-strong">
    监视资本主义正在迅速从数字领域转向物理世界，目标是收集更多关于您的信息，以便他们可以通过这些信息获利
   </strong>
   。
  </p>
  <p class="graf graf--p">
   我们中的许多人将看似无关紧要的物体，如 联网的恒温器和扬声器带入自己家厨房和卧室，却没有意识到他们可能会持续监视着我们（和我们的邻居），并向公司发送报告。
  </p>
  <p class="graf graf--p">
   但是，在公共场合进行的越来越多的监视会使我们典型的、即使已经破裂的数据收集关系变得更加复杂化，公司和政府要求我们“同意“。
  </p>
  <p class="graf graf--p">
   正如英国数字研究员 Lilian Edwards
   <a class="markup--anchor markup--p-anchor" data-href="https://zenodo.org/record/34501/files/CREATe-Working-Paper-2015-11.pdf" href="https://zenodo.org/record/34501/files/CREATe-Working-Paper-2015-11.pdf" rel="noopener noreferrer" target="_blank">
    在2015年的一篇论文
   </a>
   中所写的那样，“虽然消费者在签订合同之前至少理论上有机会阅读 Nest 恒温器的隐私政策，但是，
   <strong class="markup--strong markup--p-strong">
    在现实中，他们在任何时候都没有这样的机会。他们的数据是被通过他们上班必行的智能道路或不得不乘坐的智能电车收集的，或当他步行路过智能垃圾箱时
   </strong>
   。“
  </p>
  <p class="graf graf--p">
   以下是一个随机对象的样本，关于在公共场合一直在密切监视你的东西都是些什么：
  </p>
  <p class="graf graf--p">
   道路标志、广告牌和街道 — —
  </p>
  <p class="graf graf--p">
   你觉得自己在独自一人行走，可能不会有人看到你的一举一动，但是，肯定有人在看 — — 特别是政府。
  </p>
  <p class="graf graf--p">
   除了街道交叉点和全国各地建筑物侧面的闭路电视摄像机之外，还有更多的监视设备就隐藏在各种其他物体中，这些物体看起来就是城市的日常景观，完全没有任何异样。
  </p>
  <p class="graf graf--p">
   你知道那些能告诉你驾驶速度有多快的数字路标吗？正如
   <a class="markup--anchor markup--p-anchor" data-href="https://qz.com/1400791/that-road-sign-telling-you-how-fast-youre-driving-may-be-part-of-a-us-government-surveillance-network/" href="https://qz.com/1400791/that-road-sign-telling-you-how-fast-youre-driving-may-be-part-of-a-us-government-surveillance-network/" rel="noopener noreferrer" target="_blank">
    Quartz 报道
   </a>
   的那样，其中一些是嵌入了车牌阅读器的，作为十年前所谓的缉毒管理计划的一部分。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    DEA 和 ICE 还秘密地将摄像机嵌入到普通的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://qz.com/1458475/the-dea-and-ice-are-hiding-surveillance-cameras-in-streetlights/" href="https://qz.com/1458475/the-dea-and-ice-are-hiding-surveillance-cameras-in-streetlights/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     街灯
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    和那些大号的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.fbo.gov/index?s=opportunity&amp;mode=form&amp;id=a0459e93674685daf7c08cfa3db0ed2e&amp;tab=core&amp;tabmode=list&amp;=" href="https://www.fbo.gov/index?s=opportunity&amp;mode=form&amp;id=a0459e93674685daf7c08cfa3db0ed2e&amp;tab=core&amp;tabmode=list&amp;=" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     橙色交通桶
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    中。即使是路边的景色也不会安全。 Ars Technica 发现，亚利桑那州的一个城镇
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://arstechnica.com/tech-policy/2015/05/arizona-town-mounts-dozens-of-new-license-plate-readers-in-fake-cactuses/" href="https://arstechnica.com/tech-policy/2015/05/arizona-town-mounts-dozens-of-new-license-plate-readers-in-fake-cactuses/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     在假仙人掌内安装
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    了数十个车牌阅读器！
   </strong>
  </p>
  <p class="graf graf--p">
   警察巡逻车，路灯甚至私人车辆都可以看到更明显的车牌阅读器，执法者和他们的代理人使用这些车辆数据编制数十亿的车辆记录。隐私权倡导者警告说，
   <strong class="markup--strong markup--p-strong">
    该技术还可用于制作非犯罪嫌疑人的详细肖像，包括他们参加政治示威活动的详情
   </strong>
   。
  </p>
  <p class="graf graf--p">
   人脸识别软件悄然与高清街道级闭路电视摄像机结合时，已经引发了类似的对大规模监控的警报，关于当权者在公共场合跟踪人们的能力。种族司法和民权团体特别担心这种监视的影响，认为面部识别会加剧现有的偏见，并对弱势群体社区造成不成比例的影响。
  </p>
  <p class="graf graf--p">
   更隐蔽的监视跟踪形式涉及使用隐藏的 Stingrays 或 ISMI 捕手，旨在拦截所有附近的移动电话流量，以便跟踪他们想要的任何人。
  </p>
  <p class="graf graf--p">
   虽然这些设备在几个州已被裁定为违宪，但是，它们可以被设计为无法检测到：它们可以像手机一样小，隐藏为手机
   <a class="markup--anchor markup--p-anchor" data-href="https://motherboard.vice.com/en_us/article/jp5azg/fbi-admits-it-uses-fake-cell-phone-towers-to-track-you" href="https://motherboard.vice.com/en_us/article/jp5azg/fbi-admits-it-uses-fake-cell-phone-towers-to-track-you" rel="noopener noreferrer" target="_blank">
    信号塔的一部分
   </a>
   ，或者就放在
   <a class="markup--anchor markup--p-anchor" data-href="https://motherboard.vice.com/en_us/article/nz798m/harris-imsi-catcher-picture-phone-tracking-device-in-the-wild" href="https://motherboard.vice.com/en_us/article/nz798m/harris-imsi-catcher-picture-phone-tracking-device-in-the-wild" rel="noopener noreferrer" target="_blank">
    警车内
   </a>
   。它们的使用也越来越多地超越了执法部门：据报道，许多神秘实体正在使用这些技术以拦截政府官员的电话，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.fastcompany.com/90256599/china-and-russia-tapped-trump-phone-may-be-largest-white-house-breach-ever-says-former-official" href="https://www.fastcompany.com/90256599/china-and-russia-tapped-trump-phone-may-be-largest-white-house-breach-ever-says-former-official" rel="noopener noreferrer" target="_blank">
    包括特朗普
   </a>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果你在美国的某些州（并不是全美），检测到 ISMI 捕手，是可以起诉违宪的，检测方法在这里《
   </strong>
   <a href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ba%94%e4%bb%98%e6%94%bf%e5%ba%9c%e4%bd%bf%e7%94%a8%e9%bb%84%e8%b2%82%e9%b1%bc%e5%af%b9%e5%85%ac%e6%b0%91%e8%bf%9b%e8%a1%8c%e7%9a%84%e5%a4%a7%e8%a7%84%e6%a8%a1%e9%97%b4%e8%b0%8d/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     如何应付政府使用黄貂鱼对公民进行的大规模间谍行为
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》；
   </strong>
   如果在中国，建议不要携带智能手机出门。
  </p>
  <p class="graf graf--p">
   你需要警惕的不仅仅是执法部门或政府。当您在街上行走或路过广告牌时，广告商也会跟踪您。
  </p>
  <p class="graf graf--p">
   现实世界中的广告变得越来越技术化了，集成了面部识别，位置数据，人工智能以及其他通常与您的手机相关的强大工具。
   <strong class="markup--strong markup--p-strong">
    在街头你没有选择退出的机会
   </strong>
   。广告牌变成了巨型监视器？详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://iyouport.weebly.com/short-stories-enthusiast/3029226" href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e8%a6%81%e4%bd%a0%e8%bf%9c%e7%a6%bb%e5%b9%bf%e5%91%8a%e7%89%8c-%e8%80%81%e5%a4%a7%e5%93%a5%e6%9d%a5%e4%ba%86/" rel="noopener noreferrer" target="_blank">
    远离广告牌!!!
   </a>
   》
  </p>
  <p class="graf graf--p">
   一家大型广告牌公司 Clear Channel Outdoor America 提供跟踪人们旅行模式的服务，以便根据其位置识别看到某个广告牌的人的年龄和性别。该公司基本没有提供隐私保证。现实生活中的大多数广告都是在按照数字广告的方式跟踪你，这只是时间问题。
  </p>
  <p class="graf graf--p">
   现代药店大多是低技术含量的地方，货架上摆满了各种各样的产品，人们开始配药。但是，Walgreens 正试图通过像科技公司在线跟踪客户一样在现实中跟踪你。怎么做的？
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.fastcompany.com/90302382/its-not-just-google-or-facebook-the-freezer-aisle-is-ad-targeting-you-too" href="https://www.fastcompany.com/90302382/its-not-just-google-or-facebook-the-freezer-aisle-is-ad-targeting-you-too" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     通过在通道中安装数字屏幕覆盖的门、配备传感器
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，可以检测您的性别、您的一般年龄范围、您正在查看的产品、您站在某个柜台前的时间，甚至您对特定产品的情绪反应
   </strong>
   。
  </p>
  <p class="graf graf--p">
   然后，制作门的公司 Cooler Screens 可以使用这些信息为您提供有针对性的广告，这些广告会在您走过冰柜门前的时候时即时出现。
  </p>
  <p class="graf graf--p">
   例如，一个男人可能会看到一个 Coke Zero 广告，而一个女人可能会看到健怡可乐的广告。据称，冰箱也可以为您提供独特的优惠。
  </p>
  <p class="graf graf--p">
   Cooler Screens 首席执行官兼联合创始人 Arsen Avakian 告诉我，“你路过啤酒冰柜的门，[门]可能会注意到你正在购买六件装 Miller Coors。现在是下午4点，也就是说接近晚餐时间。于是那扇门可能为您提供这样的广告：如果您购买六包 Miller Coors 就可以特价购买 DiGiorno 比萨饼。“
  </p>
  <p class="graf graf--p">
   该公司表示，除非您选择加入，否则它不会存储数据或将信息与客户资料相关联。
   <strong class="markup--strong markup--p-strong">
    但这是一个更大的问题，门的设计并没有表明他们正在分析你的一举一动。选择退出的唯一方法是：从一开始就不要进入这家商店
   </strong>
   。
  </p>
  <p class="graf graf--p">
   传感器和摄像机无处不在，从家到街道，再到药房，并且它们在娱乐场所也是非常普遍的。体育场馆通常会
   <a class="markup--anchor markup--p-anchor" data-href="https://www.fastcompany.com/1686697/ny-jets-command-center-offers-real-time-analytics-sales-stats-freaks" href="https://www.fastcompany.com/1686697/ny-jets-command-center-offers-real-time-analytics-sales-stats-freaks" rel="noopener noreferrer" target="_blank">
    追踪球迷的行为
   </a>
   ，但过去一年来的一些音乐会也将这种监控提升到了全新的水平。
  </p>
  <p class="graf graf--p">
   在中国，警方在音乐会前的安检中使用了面部识别技术，
   <a class="markup--anchor markup--p-anchor" data-href="https://qz.com/1285912/chinas-facial-recognition-cameras-keep-catching-fugitives-at-pop-star-jacky-cheungs-concerts/" href="https://qz.com/1285912/chinas-facial-recognition-cameras-keep-catching-fugitives-at-pop-star-jacky-cheungs-concerts/" rel="noopener noreferrer" target="_blank">
    逮捕了
   </a>
   前来观看张学友演唱会的逃犯。为了解决粉丝“太热情”的问题，Taylor Swift 的安全团队最近在她的巡回演出中
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/technology/2019/feb/15/how-taylor-swift-showed-us-the-scary-future-of-facial-recognition" href="https://www.theguardian.com/technology/2019/feb/15/how-taylor-swift-showed-us-the-scary-future-of-facial-recognition" rel="noopener noreferrer" target="_blank">
    安装了配备具有面部识别技术
   </a>
   的监视摄像头。该技术来自一家名为 ISM Connect 的公司，该公司表示任何与确定的一组人员不匹配的面孔都会对 Swift “造成威胁”。
  </p>
  <p class="graf graf--p">
   某些司法管辖区正试图要求企业在部署人脸识别时发布明确的警告。去年，纽约市议员提出了一项法案，该法案要求在使用面部识别时
   <a class="markup--anchor markup--p-anchor" data-href="https://www.fastcompany.com/90281673/nyc-face-recognition-law-ritchie-torres" href="https://www.fastcompany.com/90281673/nyc-face-recognition-law-ritchie-torres" rel="noopener noreferrer" target="_blank">
    发布通知
   </a>
   。伊利诺伊州、德克萨斯州和华盛顿州已禁止在未经知情同意的情况下收集生物识别数据。
  </p>
  <p class="graf graf--p">
   人们会意识到自己在物理世界中放弃的隐私人权就像在线时一样容易吗？当冰柜和自拍亭以及门铃、飞机上娱乐系统和广告牌，都在跟踪你时，究竟该如何选择“退出”？
  </p>
  <p class="graf graf--p">
   顺便说，
   <strong class="markup--strong markup--p-strong">
    从今年初开始，中国官方媒体就被下发了一个大型主题：5G，其中一个分支题目就是：让所有常规物品全部*在线* — everything is watching everybody
   </strong>
   。
  </p>
  <p class="graf graf--p">
   欢迎来到2019年。⚪️
  </p>

    ven cacti have eyes: The random things surveilling our everyday lives
   </a>
   . Your every move online is tracked. In the era of smart homes and smart cities, everyday objects are starting to keep tabs on you too. When freezers and selfie booths and doorbells and in-flight entertainment systems and billboards are all tracking you, how exactly do opt out?
