  <p class="graf graf--p graf--startsWithDoubleQuote">
   “历史表明，政府经常会寻求精细、彻底、敏锐、详尽地规范所有人的生活。
  </p>
  <p class="graf graf--p">
   刑法已经变得越来越厚，涵盖了如此多的以前完全无辜的行为，如今几乎任何人都可以因某件事而被捕。
  </p>
  <p class="graf graf--p">
   如果国家可以使用这些法律并不是为了他们的预期目的、而是为了让那些发表不受欢迎的想法的人保持沉默，那么第一修正案的自由就不再有意义，我们将回到曾经的暴政年代……
  </p>
  <p class="graf graf--p">
   在没有被逮捕风险的情况下发言的自由是“我们区分自由国家的主要特征之一。” —— — Justice Neil Gorsuch, dissenting,
   <a class="markup--anchor markup--p-anchor" data-href="https://casetext.com/case/nieves-v-bartlett-1" href="https://casetext.com/case/nieves-v-bartlett-1" rel="noopener noreferrer" target="_blank">
    Nieves v. Bartlett
   </a>
   (2019)
  </p>
  <p class="graf graf--p">
   第一修正案所保护的 —— 并且也是一个健康的宪法共和国所需求的 —— 是那些经常行使权利说实话的公民。
  </p>
  <p class="graf graf--p">
   而警察国家的建筑师想要的是顺从、服从、合作、温顺的人民，不反驳、不挑战政府的权威、不反对政府的不端行为，不要走出界限……
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    对于那些拒绝温顺地接受警察国家暴政的人来说，危险是非常真实的。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    我们生活在一个“人民”受到军事化、武器化的警察摆布的时代，警察几乎绝对有权决定谁是威胁、什么构成抵抗，就是这些原本被制定为“服务于人民和保护人民”的警察，现在正在严厉地惩治他们的服务对象
   </strong>
   。
  </p>
  <p class="graf graf--p">
   那些在与警察接触时寻求行使其第一修正案权利的人越来越多地发现根本没有言论自由这样的事。
  </p>
  <p class="graf graf--p">
   这是一个痛苦的教训。
   <strong class="markup--strong markup--p-strong">
    在任何时候公民表达对政府的不满或挑战、甚至只是质疑权力的权威时，都会有人被逮捕，被指控越来越多的“藐视”罪名。
   </strong>
  </p>
  <p class="graf graf--p">
   当警察发出命令时，仅仅因为你敢于质疑、挑战或犹豫不决，就会使你被指控抵制逮捕或行为不检。你因行使言论自由权而受到谴责。
  </p>
  <p class="graf graf--p">
   事实上，被指控或被逮捕现在是你遇到警察时最好的选项了，这些警察被允许在他们的言论是法律并且没有任何形式的分歧甚至被质疑空间的情况下进行操作。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    最糟糕的情况还包括被警方探查、被殴打，被抓捕、被电击枪、被搜查、被扣押、被粗暴对待、甚至被射击或杀害。
   </strong>
  </p>
  <p class="graf graf--p">
   任何穿着政府制服（士兵、警察、监狱看守）的人你都必须毫无疑问地服从，这种心态是威权主义者走向极权主义的一个标志。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    理由如下：
   </strong>
  </p>
  <p class="graf graf--p">
   完全按照我说的做，我们会相处得很好。不要质疑我或以任何方式反驳我；
  </p>
  <p class="graf graf--p">
   你无权反对我可能会说或要求你做的任何事，就算我的要求不明确或相互矛盾，你也无权澄清；
  </p>
  <p class="graf graf--p">
   无论我的命令多么武断、不合理、歧视或公然种族主义，你也必须毫不犹豫地在任何情况下都服从我；
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    除了立即完美的束手就擒之外的一切行为都将被标记为抵制逮捕，你会见识我的暴力反应有多敏捷；
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种反应可能会导致严重的伤害甚至死亡。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    而且我不负任何责任。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    你只有两个选择：遵守、或者死亡
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ……
  </p>
  <p class="graf graf--p">
   事实上，正如洛杉矶警察局局长 Sunil Dutta 所建议的那样：
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://www.washingtonpost.com/posteverything/wp/2014/08/19/im-a-cop-if-you-dont-want-to-get-hurt-dont-challenge-me/" href="https://www.washingtonpost.com/posteverything/wp/2014/08/19/im-a-cop-if-you-dont-want-to-get-hurt-dont-challenge-me/" rel="noopener noreferrer" target="_blank">
    如果你不想被枪击、被殴打、被胡椒喷雾、被大棒击打、或被扔到地上，那就按我说的做
   </a>
   。
  </p>
  <p class="graf graf--p">
   不要和我争辩、不要指责我、不要对我申明你的权利、不要说我是种族主义猪，不要威胁说你会起诉我。
  </p>
  <p class="graf graf--p">
   这些都没用。
  </p>
  <p class="graf graf--p">
   ……
  </p>
  <p class="graf graf--p">
   这绝不是一个理解、更不用说尊重言论自由的人的态度。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5765 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-5.jpg?resize=1100%2C733&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-5.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-5.jpg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-5.jpg?resize=768%2C512&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-5.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-5.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-5.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="733" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-5.jpg?resize=1100%2C733&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   再说一遍，
   <strong class="markup--strong markup--p-strong">
    当政府用枪杆子语言说话时，公民就不会有任何言论自由。
   </strong>
  </p>
  <p class="graf graf--p">
   这种枪杆子语言是什么？
  </p>
  <p class="graf graf--p">
   军事化的警察。防暴队。迷彩装备。黑色制服。装甲车。大规模逮捕。
  </p>
  <p class="graf graf--p">
   胡椒喷雾。催泪弹。警棍。任意搜查。监控摄像机。
   <a class="markup--anchor markup--p-anchor" data-href="http://www.wnyc.org/story/militarization-american-police-departments/" href="https://www.wnyc.org/story/militarization-american-police-departments/" rel="noopener noreferrer" target="_blank">
    防弹背心
   </a>
   。
  </p>
  <p class="graf graf--p">
   无人机。
   <a class="markup--anchor markup--p-anchor" data-href="http://www.slate.com/articles/news_and_politics/politics/2014/08/police_in_ferguson_military_weapons_threaten_protesters.html" href="http://www.slate.com/articles/news_and_politics/politics/2014/08/police_in_ferguson_military_weapons_threaten_protesters.html" rel="noopener noreferrer" target="_blank">
    致命武器
   </a>
   。橡皮子弹。水炮。眩晕手榴弹。
  </p>
  <p class="graf graf--p">
   逮捕记者。
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Kettling" href="https://en.wikipedia.org/wiki/Kettling" rel="noopener noreferrer" target="_blank">
    人群控制战术
   </a>
   。恐吓战术。残酷。所谓的蔑视警察指控……
  </p>
  <p class="graf graf--p">
   这不是自由的语言。这甚至不是法律和秩序的语言。
  </p>
  <p class="graf graf--p">
   不幸的是，这就是联邦政府、州政府和地方各级政府如今回应那些选择行使第一修正案权利的人的方式。
  </p>
  <p class="graf graf--p">
   事实上，
   <strong class="markup--strong markup--p-strong">
    就在最近，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://casetext.com/case/nieves-v-bartlett-1" href="https://casetext.com/case/nieves-v-bartlett-1" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     最高法院发布了一项裁决，保护警方免受因伪造“藐视警察”指控以遏制合法的第一修正案活动所产生的后果
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   合法的第一修正案活动包括 如拍摄警察的行动、询问警察问题、拒绝与警方交谈等等，现在这些行为都被称为“藐视警察”，你可能会因此遭受上述暴力。而且警察不负责。
  </p>
  <p class="graf graf--p">
   在 Nieves v.Bartlett 案中，法院以6–3裁定驳回了阿拉斯加居民 Russell Bartlett 的案件，他在一次户外节日活动中因“行为不检”而被捕，因为他拒绝被警察审讯拒绝逮捕。
  </p>
  <p class="graf graf--p">
   在露营派对上，Bartlett 只是行使了他的第一修正案的权利，以避免与一名正在监视未成年人饮酒事件的州警官交谈。
  </p>
  <p class="graf graf--p">
   而他被绊倒，被摔在地上，并被威胁如果他反抗就让他以“藐视警察”罪名被捕。
  </p>
  <p class="graf graf--p">
   然而 Bartlett 的案子却被驳回了。Bartlett 起诉，声称他是因为质疑部队的权威而被捕的。
  </p>
  <p class="graf graf--p">
   虽然法院承认人们有权免于对合法的第一修正案活动进行的报复性逮捕，但它裁定，如果警察有可能的逮捕理由，被告不能起诉违反言论自由，除非被告能够证明其他人没有因同样的行为而被捕。
  </p>
  <p class="graf graf--p">
   目前在最高法院审理的另一起案件，Ogle诉德克萨斯州，
   <strong class="markup--strong markup--p-strong">
    这位德克萨斯州男子
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.rutherford.org/publications_resources/on_the_front_lines/under_texas_law_banning_annoying_or_offensive_electronic_messages_man_faces_a_year_in_prison_for_criticizing_police_about_pissing_on_constitution" href="https://www.rutherford.org/publications_resources/on_the_front_lines/under_texas_law_banning_annoying_or_offensive_electronic_messages_man_faces_a_year_in_prison_for_criticizing_police_about_pissing_on_constitution" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     面临长达一年的监禁，和4000美元的罚款
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ；只因为他向警方发送电子邮件批评他们没有回应他的援助请求。
   </strong>
  </p>
  <p class="graf graf--p">
   此案中的德克萨斯州的法律规定，发送“烦人”、“警告”或“骚扰”的电子信息是犯罪行为。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该法律是如此地宽泛，以至于它完全可以被用来惩罚在网上发布对餐馆的负面评论的人、甚至 Facebook 的苛刻评论帖子。
   </strong>
  </p>
  <p class="graf graf--p">
   在另一起案件中，一
   <a class="markup--anchor markup--p-anchor" data-href="https://www.rutherford.org/publications_resources/on_the_front_lines/rapper_charged_with_terrorism_for_lyrics_criticizing_police_supreme_court_fails_to_protect_controversial_artistic_political_expression" href="https://www.rutherford.org/publications_resources/on_the_front_lines/rapper_charged_with_terrorism_for_lyrics_criticizing_police_supreme_court_fails_to_protect_controversial_artistic_political_expression" rel="noopener noreferrer" target="_blank">
    名说唱歌手在 Facebook 和 YouTube 上发布了一首批评警察的歌曲后，被指控“制造恐怖威胁”
   </a>
   。
  </p>
  <p class="graf graf--p">
   当警察发现了网上这首名为“F ** k the Police”的歌曲时，警方就一直在
   <strong class="markup--strong markup--p-strong">
    积极全面地监视说唱歌手 Jamal Knox（a.k.a.“Mayhem Mal”）的社交媒体
   </strong>
   ，并指控 Knox 和他的说唱乐队伙伴多项“恐怖主义威胁”。
  </p>
  <p class="graf graf--p">
   多项”恐怖主义威胁”，就因为一首歌。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5766 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-3.jpg?resize=1100%2C721&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-3.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-3.jpg?resize=300%2C197&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-3.jpg?resize=768%2C503&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-3.jpg?resize=1024%2C671&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-3.jpg?resize=1100%2C721&amp;ssl=1 1100w" data-recalc-dims="1" height="721" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-3.jpg?resize=1100%2C721&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5766" data-recalc-dims="1" height="721" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-3.jpg?resize=1100%2C721&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-3.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-3.jpg?resize=300%2C197&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-3.jpg?resize=768%2C503&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-3.jpg?resize=1024%2C671&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-3.jpg?resize=1100%2C721&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   如果我们不再有权告诉人口普查工作人员别碰我们的财产、如果我们不再有权在警察敢于闯入我们的家之前要求他们展示搜查令、如果我们不再有权通过在公共场合或我们的T恤上或在立法机构门前表达我们的反对意见来抗议不公正的法律、如果我们不再有权站在最高法院面前，戴着抗议标志或与当选代表联系以申明我们的观点 ……那时我们就没有言论自由。
  </p>
  <p class="graf graf--p">
   我们所拥有的仅仅剩下受监管、控制、审查的言论，这已经是另一个世界。
  </p>
  <p class="graf graf--p">
   请记住，第一修正案中所载的不言而喻的自由是公民可以对政府代理人提出质、自由思考、和公开辩论问题而不会视为罪犯的权利。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    所谓的“抗议法”只有一个目的：阻止异议，并提醒民众抵制警察国家的暴政是徒劳的。
   </strong>
   （关于“抗议法”的介绍：“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.pbs.org/newshour/show/gop-lawmakers-try-rein-mass-protests-new-state-laws" href="https://www.pbs.org/newshour/show/gop-lawmakers-try-rein-mass-protests-new-state-laws" rel="noopener noreferrer" target="_blank">
    共和党立法者试图通过新的法律来控制大规模抗议活动
   </a>
   ”）
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “蔑视警察”指控已经被警察、检察官、法院和立法机关武器化，已成为另一种惩罚那些拒绝被捂嘴的人的手段。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    路易斯安那州语言艺术老师 Deyshia Hargrave
    <a class="markup--anchor markup--li-anchor" data-href="https://www.theatlantic.com/politics/archive/2018/01/can-government-officials-have-you-arrested-for-speaking-to-them/550517/" href="https://www.theatlantic.com/politics/archive/2018/01/can-government-officials-have-you-arrested-for-speaking-to-them/550517/" rel="noopener noreferrer" target="_blank">
     因在学校会议的公众意见征询期间发表异议讲话
    </a>
    而
    <a class="markup--anchor markup--li-anchor" data-href="https://www.theatlantic.com/politics/archive/2018/01/can-government-officials-have-you-arrested-for-speaking-to-them/550517/" href="https://www.theatlantic.com/politics/archive/2018/01/can-government-officials-have-you-arrested-for-speaking-to-them/550517/" rel="noopener noreferrer" target="_blank">
     被逮捕
    </a>
    。
   </li>
   <li class="graf graf--li">
    Fane Lozman
    <a class="markup--anchor markup--li-anchor" data-href="https://www.theatlantic.com/politics/archive/2018/01/can-government-officials-have-you-arrested-for-speaking-to-them/550517/" href="https://www.theatlantic.com/politics/archive/2018/01/can-government-officials-have-you-arrested-for-speaking-to-them/550517/" rel="noopener noreferrer" target="_blank">
     因
    </a>
    在佛罗里达州棕榈滩
    <a class="markup--anchor markup--li-anchor" data-href="https://www.theatlantic.com/politics/archive/2018/01/can-government-officials-have-you-arrested-for-speaking-to-them/550517/" href="https://www.theatlantic.com/politics/archive/2018/01/can-government-officials-have-you-arrested-for-speaking-to-them/550517/" rel="noopener noreferrer" target="_blank">
     举行的市议会会议期间发言暗指政府腐败
    </a>
    而
    <a class="markup--anchor markup--li-anchor" data-href="https://www.theatlantic.com/politics/archive/2018/01/can-government-officials-have-you-arrested-for-speaking-to-them/550517/" href="https://www.theatlantic.com/politics/archive/2018/01/can-government-officials-have-you-arrested-for-speaking-to-them/550517/" rel="noopener noreferrer" target="_blank">
     被捕
    </a>
    。
   </li>
   <li class="graf graf--li">

    <a class="markup--anchor markup--li-anchor" data-href="https://www.nytimes.com/2017/09/06/business/media/tom-price-journalist-arrest.html" href="https://www.nytimes.com/2017/09/06/business/media/tom-price-journalist-arrest.html" rel="noopener noreferrer" target="_blank">
     因
    </a>
    在西弗吉尼亚州议会大厦门前向
    <a class="markup--anchor markup--li-anchor" data-href="https://www.nytimes.com/2017/09/06/business/media/tom-price-journalist-arrest.html" href="https://www.nytimes.com/2017/09/06/business/media/tom-price-journalist-arrest.html" rel="noopener noreferrer" target="_blank">
     卫生和公众服务部秘书 Tom Pric e“积极”提问
    </a>
    而
    <a class="markup--anchor markup--li-anchor" data-href="https://www.nytimes.com/2017/09/06/business/media/tom-price-journalist-arrest.html" href="https://www.nytimes.com/2017/09/06/business/media/tom-price-journalist-arrest.html" rel="noopener noreferrer" target="_blank">

    </a>
    。
   </li>
   <li class="graf graf--li">
    大学教授 Ersula Ore 因
    <a class="markup--anchor markup--li-anchor" data-href="https://www.vice.com/en_us/article/av4nn5/you-should-be-able-to-talk-back-to-the-police-without-getting-assaulted-420" href="https://www.vice.com/en_us/article/av4nn5/you-should-be-able-to-talk-back-to-the-police-without-getting-assaulted-420" rel="noopener noreferrer" target="_blank">
     抨击警察而被捕，因为她反对校园警察所表现出的“不尊重态度”
    </a>
    ，警察在路上拦截她并要求她出示身份证。
   </li>
   <li class="graf graf--li">
    费城律师 Rebecca Musarra
    <a class="markup--anchor markup--li-anchor" data-href="https://abovethelaw.com/2017/08/lawyer-arrested-for-not-talking-to-police-wins-settlement-but-the-cops-are-still-on-the-beat/" href="https://abovethelaw.com/2017/08/lawyer-arrested-for-not-talking-to-police-wins-settlement-but-the-cops-are-still-on-the-beat/" rel="noopener noreferrer" target="_blank">

    </a>
    而
    <a class="markup--anchor markup--li-anchor" data-href="https://abovethelaw.com/2017/08/lawyer-arrested-for-not-talking-to-police-wins-settlement-but-the-cops-are-still-on-the-beat/" href="https://abovethelaw.com/2017/08/lawyer-arrested-for-not-talking-to-police-wins-settlement-but-the-cops-are-still-on-the-beat/" rel="noopener noreferrer" target="_blank">
     被捕，
    </a>
    因为她并拒绝回答警察例行停车检查时提出的问题。
   </li>
   <li class="graf graf--li">
    ……
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    像这样的案件已经成为今天的警察国家生活两极化的典型特征：你可能在纸面上依然拥有截然不同的受保护权利，但是，如果你敢于行使这些权利，你就会面临罚款、被逮捕、被伤害、甚至死亡的风险。
   </strong>
  </p>
  <p class="graf graf--p">
   这是今天行使自由权利的不幸代价。
  </p>
  <p class="graf graf--p">
   然而，这些并非新发展。
  </p>
  <p class="graf graf--p">
   事实上，大约50年前，Lewis Colten
   <a class="markup--anchor markup--p-anchor" data-href="https://scholar.google.com/scholar_case?case=7926620308068158831" href="https://scholar.google.com/scholar_case?case=7926620308068158831" rel="noopener nofollow noreferrer" target="_blank">
    在肯塔基州列克星敦外被捕，只因为在交通阻塞期间对警察提出质疑并向他的朋友提供建议
   </a>
   。
  </p>
  <p class="graf graf--p">
   Colten 是20名左右的大学生之一，他们当时前往蓝草机场
   <strong class="markup--strong markup--p-strong">
    向时任第一夫人帕特尼克松示威
   </strong>
   。在离开机场后，警察拦截了 Colten 车队中的一辆车，因为它装有一个过期的非本州车牌。同时 Colten 和其他车手也被拉到了路边。
  </p>
  <p class="graf graf--p">
   由于担心发生警察暴力事件，Colten 离开了他的车并站在附近，警察给了他的朋友 Mendez 一张罚单，并安排将他的车拖走。警方多次要求 Colten 离开。一名州警官宣称：“这不关你的事 ……回到你的车里，继续前进并清理道路。“
  </p>
  <p class="graf graf--p">
   Colten 坚持要为他的朋友和他朋友车里的乘客做出交通安排，但没有用，他自己反而因“违反肯塔基州的行为不检”法而被捕。Colten 随后质疑他的被捕是违反了他的第一修正案言论自由的权利，并将案件一直提交给了站在警察一边的最高法院。
  </p>
  <p class="graf graf--p">
   尽管法院承认 Colten 本身没有擅自侵犯或违反任何交通规则，但更多表示 Colten “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.law.cornell.edu/supremecourt/text/407/104" href="https://www.law.cornell.edu/supremecourt/text/407/104" rel="noopener nofollow noreferrer" target="_blank">
    没有宪法权利来观察交通罚单的开具
   </a>
   、没资格与当时发罚单的警察交谈。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    最高法院的底线是：保护警察免受阻碍比保护言论更为重要，而政府认为，言论“没有社会价值”。
   </strong>
   虽然对于一个倾向于与警察齐头并进的司法机构而言，这一裁决本身并不令人惊讶，但是 William O. Douglas 大法官的异议强烈提醒政府存在为人民服务而不是相反的方式。
  </p>
  <p class="graf graf--p">
   Douglas 大法官强调 Colten 的发言是安静的，不是喧闹，也没有“语言暴力”，并且没有涉及正常含义中的愚蠢或不检行为，Douglas 大法官只是通过与政府代表交谈提出了这些问题。在这种情况下，警察认为 Colten 是“令人不安的并且使警察烦恼”。
  </p>
  <p class="graf graf--p">
   在为言论自由的辩护中，Douglas 大法官指出：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     我们美国人何时被期望屈服于权威，并对那些代表我们的人敬畏地说话？
    </em>
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     宪法理论写的是我们人民才是主人，国家和联邦官员只是我们的代理人。
    </em>
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     拥有最终决定权的人民可以轻声或愤怒地说话，我们可以寻求挑战和滋扰，因为我们不需要保持温顺和安静
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    Colten 的情况也许认为是不合适的，但在宪法层面，言论不一定是镇静的; 它可以是破坏性的
   </em>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果政府可以控制言论，它可以控制思想；如果它可以控制思想，它就可以控制公民的精神。
   </strong>
  </p>
  <p class="graf graf--p">
   在警察国家，你要么是羊群，与政府的指令一脉相承，要么你是贱民、嫌犯、罪犯、麻烦制造者、恐怖分子、激进分子……
  </p>
  <p class="graf graf--p">
   通过扼杀公民、关闭他们可以表达思想的阀门，政府正在创造一种暴力变得不可避免的气氛。
  </p>
  <p class="graf graf--p">
   正如约翰·肯尼迪在1962年3月所警告的那样，“那些不可能实现和平革命的人将使暴力革命不可避免。”⚪️
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    <a href="https://www.counterpunch.org/2019/06/11/how-the-police-state-muzzles-our-right-to-speak-truth-to-power/" rel="noopener noreferrer" target="_blank">
     作者 John W. Whitehead
    </a>
    是卢瑟福学会的总裁，这是一个非盈利组织，总部设在美国弗吉尼亚州的夏洛茨维尔，致力于捍卫公民自由和人权。
   </em>
  </p>
