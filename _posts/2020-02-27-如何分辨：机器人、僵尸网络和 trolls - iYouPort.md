  </p>
  <h3 class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     什么是 bot？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   机器人是由算法（而不是真实的人）运行的自动社交媒体帐户。换句话说，机器人被设计为无需人工干预即可发布帖子。
  </p>
  <p class="graf graf--p">
   关于识别，参见：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%8f%91%e7%8e%b0%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e4%b8%8a%e7%9a%84%e6%9c%ba%e5%99%a8%e4%ba%ba%e8%b4%a6%e6%88%b7%ef%bc%9f-12%e7%a7%8d%e6%96%b9%e6%b3%95/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%8f%91%e7%8e%b0%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e4%b8%8a%e7%9a%84%e6%9c%ba%e5%99%a8%e4%ba%ba%e8%b4%a6%e6%88%b7%ef%bc%9f-12%e7%a7%8d%e6%96%b9%e6%b3%95/" rel="noopener noreferrer" target="_blank">
    如何发现社交媒体上的机器人账户？- 12种方法
   </a>
   》
  </p>
  <p class="graf graf--p">
   如果某个帐户撰写了各个帖子，并且发表了评论、回复，或与其他用户的帖子进行了互动，则该帐户将无法归类为机器人。
  </p>
  <p class="graf graf--p">
   机器人主要出现在 Twitter 和其他允许用户创建多个帐户的社交网络上。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     有没有办法判断一个账户 *不是* 机器人？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   检查帐户是否不是机器人的最简单方法是查看他们自己编写的推文：使用Twitter 搜索栏中的简单搜索功能；
  </p>
  <pre class="graf graf--p">From: handle</pre>
  <p class="graf graf--p">
   如果搜索返回的推文是真实的（即未从其他用户那里复制），则该帐户极不可能是机器人。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     Trolls 和机器人之间有什么区别？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   Trolls 是通过在在线社区或社交网络发布煽动性或不合主题的帖子而故意引发在线冲突或冒犯其他用户的家伙，它们的目的是激发分歧、撕裂对话。
  </p>
  <p class="graf graf--p">
   它们的方法是激发他人的情绪反应并破坏讨论。
  </p>
  <p class="graf graf--p">
   Trolls 与机器人不同，因为 trolls 是真实用户，而机器人是自动的。两种类型的帐户是互斥的。
  </p>
  <p class="graf graf--p">
   但是，Trolling 作为一项活动并不仅仅局限于 trolls 。 DFRLab 已观察到 trolls 也使用机器人来放大其某些消息。
  </p>
  <p class="graf graf--p">
   例如，早在2017年8月，在关于夏洛茨维尔抗议的文章之后，被机器人放大的 trolls 账户将 DFRLab 作为了目标。在这方面，机器人可以并且已经被用于 trolling 的目的。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     什么是僵尸网络？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   僵尸网络是由同一个人或团体管理的机器人帐户组成的网络。
  </p>
  <p class="graf graf--p">
   管理僵尸网络的人在部署之前需要原始的人工输入，被称为僵尸牧民。
  </p>
  <p class="graf graf--p">
   ⚠️僵尸网络以网络形式协调运作，是因为它们旨在制造社交媒体参与度，从而使僵尸网络所部署的主题看起来比“实际”用户更加活跃。
  </p>
  <p class="graf graf--p">
   在社交媒体平台上羊群效应是非常明显的 —— 参与会带来更多的参与，因此，
   <strong class="markup--strong markup--p-strong">
    成功的僵尸网络会将其部署的主题强行推到更多真实用户的面前
   </strong>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     僵尸网络做什么？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   僵尸网络的目标是使主题标签、用户或关键字看起来很热门或流行 —— 事实上完全不是如此。⚠️由于它们的存在，所谓的热门话题、热门标签，实际上都不一定可信。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    真实的人类们特别喜欢追热点，其结果就是，被这些僵尸引领着，远离了原本最重要的东西
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这些僵尸们的目标是操弄社交媒体算法来影响趋势部分，于是它们会使毫无戒心的用户接触到被僵尸们故意放大的对话。
  </p>
  <p class="graf graf--p">
   僵尸网络很少针对人类用户，⚠️如果他们这样做了，那么基本是在发送垃圾邮件、或对目标人或群体进行普遍的骚扰 —— *而不是* 积极尝试改变目标的意见或政治观点。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     如何识别僵尸网络？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   鉴于 Twitter 的机器人清除功能和增强的检测方法，僵尸牧民已经变得更加谨慎了，他们一直在努力使单个机器人更难以被发现。
  </p>
  <p class="graf graf--p">
   识别单个僵尸程序的一种替代方法是分析大型僵尸网络的模式，以确认其单个帐户是机器人。
  </p>
  <p class="graf graf--p">
   DFRLab 确定了
   <strong class="markup--strong markup--p-strong">
    六个可帮助识别僵尸网络的指标
   </strong>
   。如果您遇到怀疑是僵尸网络一部分的帐户网络，请注意以下几点。
  </p>
  <p class="graf graf--p">
   ⚠️在分析僵尸网络时，重要的是要记住，没有任何一个单独的指标足以得出可疑帐户是僵尸网络的一部分的结论。至少三个僵尸网络指示同时存在才能支持此类声明。
  </p>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     1、言语方式
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   通过算法运行的机器人被编程为使用相同的语言模式。
  </p>
  <p class="graf graf--p">
   如果您遇到使用完全相同的语言模式的多个帐户，例如 使用标题作为推文的文本发布新闻文章，则这些帐户可能使用的是相同的算法运行。
  </p>
  <p class="graf graf--p">
   在马来西亚大选前，DFRLab 发现了22,000个机器人，所有机器人都使用完全相同的语言模式。
  </p>
  <p class="graf graf--p">
   其中每个机器人都使用两个针对反对派联盟的标签，并在13至16个真实用户之间进行标记，以促进他们参与对话。
  </p>

  <p class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     2、相同的帖子
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   另一个僵尸网络指标是相同的发布内容。
  </p>
  <p class="graf graf--p">
   由于大多数机器人都是非常简单的计算机程序，因此它们无法产生真实的内容。结果就是，大多数机器人会发布相同的帖子。
  </p>
  <p class="graf graf--p">
   DFRLab 对一个推特活动的分析表明，该活动敦促取消在荷兰的一场伊斯兰憎恶卡通比赛，结果发现数十个帐户发布了相同的推文。
  </p>

  <p class="graf graf--p">
   尽管这些账户都是新注册的，没有过往的帖子可参考以判断，但是它们的团体协作性的行为表明，它们很可能是同一僵尸网络的一部分。
  </p>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     3、处理模式
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   识别大型僵尸网络的另一种方法是查看可疑帐户的 handle 模式。机器人创建者在命名其机器人时经常使用相同的 handle 模式。
  </p>
  <p class="graf graf--p">
   例如，在2018年1月，DFRLab 遇到了一个可能的僵尸网络，其中每个僵尸程序的 handle 末尾都有一串八位数的数字。
  </p>

   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     4、创建日期和时间
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   属于同一僵尸网络的机器人倾向于呈现出相似的创建日期。
  </p>
  <p class="graf graf--p">
   如果您一下子遇到在同一天内或同一周内创建的数十个帐户，则表明这些帐户可能是同一僵尸网络的一部分。
  </p>

   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     5、相同的 Twitter 活动
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   另一个僵尸网络的判断标准是相同的活动。
  </p>
  <p class="graf graf--p">
   如果多个帐户在 Twitter 上执行完全相同的任务、或以完全相同的方式参与，则它们很可能是同一僵尸网络的一部分。
  </p>
  <p class="graf graf--p">
   例如，早在2017年8月，DFRLab 的僵尸网络研究就跟踪到了三个看似未关联的帐户 —— NATO发言人 Oana Lungescu，可疑的僵尸牧民（@belyjchelovek）、以及一个带有猫咪头像的帐户（@ gagarinprosti）。
  </p>

   </strong>
   —— 绝不仅仅是巧合，因此可以作为强有力的僵尸网络指示器。
  </p>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     6、位置
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   最后一个指标（在政治僵尸网络中特别常见）是许多可疑帐户共享的位置。
  </p>
  <p class="graf graf--p">
   政治僵尸牧民倾向于使用他们正在推动的竞选候选人或政党所处的位置来尝试推动其在特定选区的内容趋势。
  </p>
  <p class="graf graf--p">
   例如，在墨西哥大选之前，一个在普埃布拉州推动两名PRI候选人的僵尸网络，使用的都是普埃布拉作为其所在地。
  </p>

   这样做可能是为了确保来自普埃布拉州的真实 Twitter 用户能够看到被该僵尸程序自动放大的推文和帖子。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     所有的机器人都是政治机器人吗？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   不，大多数机器人都是商业机器人帐户，这意味着它们由团体和个人经营，这些团体和个人扩大了他们为推广而支付的任何内容。
  </p>
  <p class="graf graf--p">
   当然可以雇用商业机器人来宣传政治内容。
  </p>
  <p class="graf graf--p">
   另一方面，创建政治机器人的唯一目的是扩大特定政党、候选人、利益集团、或观点的政治内容。
  </p>
  <p class="graf graf--p">
   DFRLab 在墨西哥大选前发现了多个政治僵尸网络，它们协调行动在普埃布拉州推广PRI政党候选人。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     所有俄罗斯机器人都隶属于俄罗斯吗？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   不，许多具有俄语/西里尔语ID或用户名的僵尸网络都是由希望在网上谋生的企业家俄罗斯人经营的。
  </p>
  <p class="graf graf--p">
   有许多公司和个人在俄罗斯互联网上公开销售 Twitter、Facebook 和 YouTube 的僵尸账户，以及参与、转推和点赞的服务。
  </p>
  <p class="graf graf--p">
   这点和中国一样。
  </p>

   尽管他们的服务非常便宜（

    1,000名追随者3美元
   </a>
   ），但是拥有1000个机器人的僵尸牧民每天可以服务10个客户，每天赚取33美元以上。
  </p>
  <p class="graf graf--p">
   这意味着他们每个月可以赚900美元，这是俄罗斯平均工资的两倍。
  </p>
  <figure aria-describedby="caption-attachment-9875" class="wp-caption aligncenter" id="attachment_9875" style="width: 721px">

  <p class="graf graf--p">
   例如，DFRLab 观察到，俄罗斯商业僵尸网络扩大了全球的政治内容。
  </p>
  <p class="graf graf--p">
   例如，在墨西哥大选前发现了一个僵尸网络，放大了墨西哥的绿党。但是，这些机器人不是政治性的，它们放大了各种不同的帐户，从日本的旅游吉祥物到保险公司的首席执行官。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     结论
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   机器人、僵尸网络和trolls很容易区分，并且可以使用正确的方法和工具进行识别。
  </p>
  <p class="graf graf--p">
   但是，要记住的一件事是，除非经过仔细的证明，否则不能将帐户视为机器人或 troll。⚪️
