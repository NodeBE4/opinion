  <p class="graf graf--p">
   伊斯兰国（IS）首次声称它在印度建立了一个“省”，此前克什米尔地区的武装分子和安全部队发生冲突，导致一名据称与该组织有关系的武装分子
   <a class="markup--anchor markup--p-anchor" data-href="https://www.thehindu.com/news/national/islamic-state-jammu-and-kashmir-chiefs-associate-shot-dead-in-kashmir/article27097509.ece" href="https://www.thehindu.com/news/national/islamic-state-jammu-and-kashmir-chiefs-associate-shot-dead-in-kashmir/article27097509.ece" rel="noopener noreferrer" target="_blank">
    死亡
   </a>
   。
  </p>
  <p class="graf graf--p">
   IS 的 Amaq 通讯社在一份声明中宣布了这个名为“Wilayah of Hind”的新省，该声明还声称 IS 在克什米尔 Shopian 区的 Amshipora 镇对印度军队士兵发动攻击并造成伤亡。
  </p>
  <p class="graf graf--p">
   IS 的声明与警方的声明相对应，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.thehindu.com/news/national/islamic-state-jammu-and-kashmir-chiefs-associate-shot-dead-in-kashmir/article27097509.ece" href="https://www.thehindu.com/news/national/islamic-state-jammu-and-kashmir-chiefs-associate-shot-dead-in-kashmir/article27097509.ece" rel="noopener noreferrer" target="_blank">
    警方称
   </a>
   一名名叫 Ishfaq Ahmad Sofi 的武装分子在 Shopian 的一次冲突中丧生。
  </p>
  <p class="graf graf--p">
   IS 建立新省的声明似乎是为了加强其地位，该组织于4月从伊拉克和叙利亚的自称“哈里发”中
   <a class="markup--anchor markup--p-anchor" data-href="https://www.thehindu.com/news/international/is-caliphate-on-the-brink-of-collapse/article26295758.ece" href="https://www.thehindu.com/news/international/is-caliphate-on-the-brink-of-collapse/article26295758.ece" rel="noopener noreferrer" target="_blank">
    被驱逐
   </a>
   ，在那里该组织一度控制了数千英里的领土。
  </p>
  <p class="graf graf--p">
   IS 已经加强了肇事逃逸和自杀式袭击，包括表示对
   <a class="markup--anchor markup--p-anchor" data-href="https://www.thehindu.com/news/international/two-islamic-state-men-linked-to-sri-lanka-easter-blasts/article26936390.ece" href="https://www.thehindu.com/news/international/two-islamic-state-men-linked-to-sri-lanka-easter-blasts/article26936390.ece" rel="noopener noreferrer" target="_blank">
    斯里兰卡大型爆炸事件
   </a>
   负责，此次事件造成了至少253人死亡。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “在一个没有类似实际治理的地区建立一个’省’是荒谬的，但它不应该被注销，”跟踪伊斯兰国的 SITE 主管 Rita Katz 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “世界可能会注意到这些发展，但对于在脆弱地区的圣战分子来说，这些都是为帮助重建 IS 的’哈里发’地图奠定基础的重要手势。”
  </p>
  <p class="graf graf--p">
   此前一名军方官员和 Sofi 接受一份同情 IS 的斯利那加杂志的采访，Sofi 已经参与了克什米尔的几个激进组织，然后宣誓效忠伊斯兰国。
  </p>
  <p class="graf graf--p">
   警方和军方消息人士称，Sofi 被怀疑对该地区的安全部队进行了数次手榴弹袭击。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这是一次干净的行动，在交火期间没有发生任何附带损害，”警方发言人在声明中表示。这位军方官员说，有可能 Sofi 是与伊斯兰国有关的唯一一个留在克什米尔的武装分子。
  </p>
  <p class="graf graf--p">
   分离主义者几十年来一直在克什米尔与印度统治进行武装冲突。
  </p>
  <p class="graf graf--p">
   这些团体中的大多数希望克什米尔独立或加入巴基斯坦。他们没有像伊斯兰国一样寻求在穆斯林世界建立一个帝国。
  </p>
  <p class="graf graf--p">
   核大国印度和巴基斯坦在克什米尔问题上打了两场战争，在巴基斯坦武装组织于 Pulwama
   <a class="markup--anchor markup--p-anchor" data-href="https://www.thehindu.com/news/national/ied-blast-in-kashmirs-pulwama-many-crpf-jawans-killed/article26268289.ece" href="https://www.thehindu.com/news/national/ied-blast-in-kashmirs-pulwama-many-crpf-jawans-killed/article26268289.ece" rel="noopener noreferrer" target="_blank">
    杀害了至少40名
   </a>
   准军事警察的自杀式袭击事件发生后，今年早些时候又发生了第三次战争。
  </p>
