  <p class="graf graf--p">
   中国正在努力“消除贫困” …… 就如你所知道的那样，当权者鼓励普通公民 “献爱心”，但是，这真的是消除贫困的有效方法吗？不。恰恰相反，这是拒绝消除贫困的手段，因为，你首先要知道真正的贫困来自哪里。
  </p>
  <p class="graf graf--p">
   《纽约时报》于去年1月在其头版发表社论，题为 “废除亿万富翁’’。
  </p>
  <p class="graf graf--p">
   该社论提出了一个严重的问题：如果亿万富翁不是经济成功的标志，而是经济失败的标志，那会如何？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    亿万富翁的蓬勃发展以及极端财富的急剧增加将以何种方式产生危害？
   </strong>
  </p>
  <p class="graf graf--p">
   要回答这个问题，首先需要了解亿万富翁财富的来源，并了解一旦获得财富后这些钱会如何被使用。
  </p>
  <p class="graf graf--p">
   这两个问题的答案都正确地质疑了超级富豪在社会中的价值。
  </p>
  <p class="graf graf--p">
   亿万富翁的财富中大约有三分之一来自继承。很难证明继承财富的经济效用，相反，有充分的理由证明继承财富破坏了社会流动性和经济进步。
  </p>
  <p class="graf graf--p">
   相反，
   <strong class="markup--strong markup--p-strong">
    它创造了一个新的贵族，他们仅仅因为父母有钱而变得富有
   </strong>
   ，这很难被视为一件好事。
  </p>
  <p class="graf graf--p">
   无论是继承还是以其他方式获得担保，极端财富都会发挥自己的作用。超级富豪有钱花在最好的投资建议上，
   <strong class="markup--strong markup--p-strong">
    亿万富翁的财富自2009年以来
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www-cdn.oxfam.org/s3fs-public/file_attachments/bp-economy-for-99-percent-160117-en.pdf" href="https://www-cdn.oxfam.org/s3fs-public/file_attachments/bp-economy-for-99-percent-160117-en.pdf" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     平均每年增长11％
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，远高于普通储户可获得的利率
   </strong>
   。
  </p>
  <p class="graf graf--p">
   比尔·盖茨（Bill Gates）在2019年的身家接近1000亿美元，几乎是他辞去微软负责人时身家的两倍。尽管他对捐出自己的钱做出了很好听的承诺。
  </p>
  <p class="graf graf--p">
   正如托马斯·皮凯蒂（Thomas Piketty）在他的《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amazon.com/exec/obidos/ASIN/0674979850/counterpunchmaga" href="https://www.amazon.com/exec/obidos/ASIN/0674979850/counterpunchmaga" rel="noopener noreferrer" target="_blank">
    21世纪的资本
   </a>
   》一书中所阐述的：“无论最初财富不平等的理由有多么合理的，在社会效用方面，它都会超出任何合理的理由。”
  </p>
  <p class="graf graf--p">
   乐施会的迪迪埃·雅各布斯（Didier Jacobs）几年前
   <a class="markup--anchor markup--p-anchor" data-href="https://www-cdn.oxfam.org/s3fs-public/file_attachments/dp-extreme-wealth-is-not-merited-241115-en.pdf" href="https://www-cdn.oxfam.org/s3fs-public/file_attachments/dp-extreme-wealth-is-not-merited-241115-en.pdf" rel="noopener noreferrer" target="_blank">
    曾计算过
   </a>
   ，另一位亿万富翁的
   <strong class="markup--strong markup--p-strong">
    财富来自与政府和垄断的裙带关系
   </strong>
   。
  </p>
  <p class="graf graf--p">
   经济学人已经开发出一种对裙带资本主义的
   <a class="markup--anchor markup--p-anchor" data-href="http://infographics.economist.com/2016/Cronyism_index/" href="http://infographics.economist.com/2016/Cronyism_index/" rel="noopener noreferrer" target="_blank">
    衡量方法
   </a>
   ，其发现也与此类似。很明显，
   <strong class="markup--strong markup--p-strong">
    腐败和与政府的裙带关系是亿万富翁获得财富的很大一部分原因
   </strong>
   。
  </p>
  <p class="graf graf--p">
   正如尼克·沙克森（Nick Shaxson）在其新书《金融诅咒》（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amazon.com/exec/obidos/ASIN/1847924212/counterpunchmaga" href="https://www.amazon.com/exec/obidos/ASIN/1847924212/counterpunchmaga" rel="noopener noreferrer" target="_blank">
    Finance Curse
   </a>
   ）中所详述的那样，
   <strong class="markup--strong markup--p-strong">
    全球经济的几乎所有领域现在都存在强大的资本垄断
   </strong>
   。
  </p>
  <p class="graf graf--p">
   无论是食品、药品、媒体、金融、还是技术，都是仅仅由极少数几个大型公司把控的。
  </p>
  <p class="graf graf--p">
   显而易见，数十年来的并购潮是寡头企业消灭竞争对手的最基本方式，从而导致了这种情况。
  </p>
  <p class="graf graf--p">
   从历史上看，尤其是在20世纪初期，在美国，垄断势力被正确地视为对经济和社会的严重威胁，并采取了措施来打破垄断。
  </p>
  <p class="graf graf--p">
   富兰克林·罗斯福总统曾有句著名的话
   <a class="markup--anchor markup--p-anchor" data-href="http://legacy.buzzflash.com/commentary/fdr-government-by-organized-money-is-just-as-dangerous-as-government-by-organized-mob" href="http://legacy.buzzflash.com/commentary/fdr-government-by-organized-money-is-just-as-dangerous-as-government-by-organized-mob" rel="noopener noreferrer" target="_blank">
    说
   </a>
   ：“有组织的金钱就如有组织的暴民一样危险”。
  </p>
  <p class="graf graf--p">
   但是，近几十年来，新自由主义经济学对垄断力量的看法更为温和了，现在几乎没有采取任何行动拆除它们。
  </p>
  <p class="graf graf--p">
   我认为这是新自由主义和古典自由主义经济学之间的主要区别。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9924 jetpack-lazy-image" data-lazy-sizes="(max-width: 478px) 100vw, 478px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/中.png?resize=478%2C1064&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/中.png?w=478&amp;ssl=1 478w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/中.png?resize=135%2C300&amp;ssl=1 135w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/中.png?resize=460%2C1024&amp;ssl=1 460w" data-recalc-dims="1" height="1064" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/中.png?resize=478%2C1064&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="478"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9924" data-recalc-dims="1" height="1064" sizes="(max-width: 478px) 100vw, 478px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/中.png?resize=478%2C1064&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/中.png?w=478&amp;ssl=1 478w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/中.png?resize=135%2C300&amp;ssl=1 135w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/中.png?resize=460%2C1024&amp;ssl=1 460w" width="478"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这些垄断者对每个消费者都征收隐性的垄断税，因为这使这些寡头公司及其富裕的股东能够从市场中获取过多的利润，从而直接以牺牲普通市民为代价，促进了极端财富的增长
   </strong>
   。
  </p>
  <p class="graf graf--p">
   公司的行动、包括走向垄断的行动，仅仅受到为股东不断增加的回报的驱使，这些股东主要是非常富有的人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    乐施会发布的有关“G7 七宗罪”的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.oxfam.org/en/research/g7-deadly-sins-inequality-crisis" href="https://www.oxfam.org/en/research/g7-deadly-sins-inequality-crisis" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     论文显示了
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    股东回报率如何显著增加，而实际工资却几乎没有任何增加
   </strong>
   。
  </p>
  <p class="graf graf--p">
   公司权力和公司行为的背后是超富豪股东的权力
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/commentisfree/2019/jul/26/trump-johnson-nationalists-billionaire-oligarchs" href="https://www.theguardian.com/commentisfree/2019/jul/26/trump-johnson-nationalists-billionaire-oligarchs" rel="noopener noreferrer" target="_blank">
    越来越大
   </a>
   。
  </p>
  <p class="graf graf--p">
   一旦亿万富翁的财富积累起来，其使用方式也会使人们对拥有亿万财富的富翁的有用性产生怀疑。
  </p>
  <p class="graf graf--p">
   正如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e8%bf%8e%e6%8e%a5%e6%9b%b4%e5%a4%9a%e7%9a%84%e5%b7%b4%e6%8b%bf%e9%a9%ac%e6%96%87%e4%bb%b6%ef%bc%9f/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e8%bf%8e%e6%8e%a5%e6%9b%b4%e5%a4%9a%e7%9a%84%e5%b7%b4%e6%8b%bf%e9%a9%ac%e6%96%87%e4%bb%b6%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    巴拿马文件
   </a>
   和其他消息所揭示的那样，超级富豪利用他们的财富支付尽可能少的税，积极利用了秘密的避税天堂全球网络。
  </p>
  <p class="graf graf--p">
   一项基于这一泄漏信息进行的开创性
   <a class="markup--anchor markup--p-anchor" data-href="https://gabriel-zucman.eu/files/AJZ2017.pdf" href="https://gabriel-zucman.eu/files/AJZ2017.pdf" rel="noopener noreferrer" target="_blank">
    研究表明
   </a>
   ，超级富豪实际所付的税款比他们应该缴付的少了30％，这使政府损失了数十亿美元的税收收入，而这些钱原本可以用于学校或医院的公益性建设。
  </p>
  <p class="graf graf--p">
   超级富豪得到了信托与房地产从业者协会（STEP）的支持，
   <strong class="markup--strong markup--p-strong">
    该组织是一个秘密组织，由20,000多名财富管理人员组成，该组织积极向政府施压，要求他们降低最富有者的税收。
   </strong>
  </p>
  <p class="graf graf--p">
   数十亿美元不仅用于确保降低税收；
   <strong class="markup--strong markup--p-strong">
    它们也可以用来获得有罪不罚的特权 —— 收买政客或收买顺从的媒体。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    使用“黑钱”来影响选举和公共政策是世界范围内都日益严重的问题
   </strong>
   。
  </p>
  <p class="graf graf--p">
   乐施会的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.oxfam.org/en/pressroom/pressreleases/2018-11-16/influence-elites-governments-latin-america-contributes-increase" href="https://www.oxfam.org/en/pressroom/pressreleases/2018-11-16/influence-elites-governments-latin-america-contributes-increase" rel="noopener noreferrer" target="_blank">
    另一项最新研究
   </a>
   表明，拉丁美洲非常富有的人掌握了许多政治手段。
   <strong class="markup--strong markup--p-strong">
    当今许多新型的民族主义、种族主义领导人
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/commentisfree/2019/jul/26/trump-johnson-nationalists-billionaire-oligarchs" href="https://www.theguardian.com/commentisfree/2019/jul/26/trump-johnson-nationalists-billionaire-oligarchs" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     都有大量的财政支持
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   超级富豪的这种积极的政治影响力通过构建强化的反馈循环而直接加剧了更大的不平等 —— 在这种循环中，游戏的获胜者获得了更多的资源，可以在下一次赢得更大的胜利。
  </p>
  <p class="graf graf--p">
   出于所有这些原因，我认为有必要提出一个有力的理由，正如一位美国评论员最近所说的那样：
   <strong class="markup--strong markup--p-strong">
    “每个亿万富翁都是政策失败的标志”
   </strong>
   ；尤其是，如果我们要消除贫困并建立更公平的社会，我们就需要结束极端财富。⚪️
  </p>
