  </p>
  <p class="graf graf--p">

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e6%99%ba%e8%83%bd%e9%97%b4%e8%b0%8d%e5%ae%b6%e5%b1%85-%e5%9f%8e%e5%b8%82/" href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e6%99%ba%e8%83%bd%e9%97%b4%e8%b0%8d%e5%ae%b6%e5%b1%85-%e5%9f%8e%e5%b8%82/" rel="noopener noreferrer" target="_blank">
     关于智能[间谍]家居/城市
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    购买之前需要考虑的因素包括数据应用、安全设计和安全硬件、以及制造过程中的透明度和可见性
   </strong>
   —— 而不是制造商宣传的有多大的内存、几个摄像头、拍照效果多牛逼 ，他们只是想让你掏钱并转移你的注意力远离最重要的事。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     采取对抗性心态避免受害
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在现代社会中，几乎经常是温柔而正义的人不断成为狡猾的掠食者、骗子和罪犯所设置的陷阱的受害者。
  </p>
  <p class="graf graf--p">
   并不是说要你放弃温柔正义，而是说，
   <strong class="markup--strong markup--p-strong">
    你需要将自己的精神思想和身体变成坚不可摧的堡垒
   </strong>
   。
  </p>
  <p class="graf graf--p">
   走这条路永远不会太迟，但是无疑，你将需要迎接各种考验。
  </p>
  <p class="graf graf--p">
   必须知道每个人都有缺陷，也就是弱点，这是坏人的主要目标。就像一只烫手的烧水壶，你要把它拿起来、控制住它，就需要抓住壶的把手 ——
   <strong class="markup--strong markup--p-strong">
    目标人固有的弱点对于操纵者来说就是那个把手
   </strong>
   。
  </p>
  <p class="graf graf--p">
   相反，如果你想要防御各种侵害，就需要让自己的“把手”不可见。中文里“小辫子”一词在此格外形象。
  </p>
  <p class="graf graf--p">
   就像窃贼那样，网络掠食者会充分搜集并分析您的一切在线行为，并观察可公开获得的关于你的每个细节，以确定他们是否可以利用您的任何漏洞。
  </p>
  <p class="graf graf--p">
   由于公开的已经太多了，对于您隐藏的或不公开的内容，他们完全可以推断出大概。
   <strong class="markup--strong markup--p-strong">
    这就是为什么说每个人都必须时刻意识到自己放在互联网上的都是些什么 —— 这就是防御直觉。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    我们在 “
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     社交工程
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ” 栏目中、开 “
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     开源情报调查
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ” 栏目中所演示的诸多追踪、胁迫、操纵和身份盗取的操作，都是为提醒您的防御直觉
   </strong>
   —— 只有在您深刻理解恶意行为者的操作模式时，才能最有效地防御。
  </p>
  <p class="graf graf--p">
   您必须做到隐藏那个烧水壶的把手 —— 您的小辫子，也就是我们强调的 “大清洗”，尽可能去除一切容易被恶意者利用的东西。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a4%a7%e6%b8%85%e6%b4%97%ef%bc%9a%e5%8f%8d%e4%be%a6%e5%af%9f%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" href="https://www.iyouport.org/%e5%a4%a7%e6%b8%85%e6%b4%97%ef%bc%9a%e5%8f%8d%e4%be%a6%e5%af%9f%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" rel="noopener noreferrer" target="_blank">
     如何躲避监视追踪？大清洗：反侦察的第一步
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   似乎依旧有很多人不甚了解 **究竟什么东西是危险的** —— 了解这点是防御的关键。我们来举个例子：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    一个人在社交网络发布了一张照片，显示为某个当日举行的会议/活动的现场，并随附 “今天天气真好，是和家人一起出行的好日子”。
   </em>
  </p>
  <p class="graf graf--p">
   大多数人会认为这样的信息 “不敏感”；但是人们
   <strong class="markup--strong markup--p-strong">
    没能通过恶意行为者的视角来观察它
   </strong>
   。
  </p>
  <p class="graf graf--p">
   你没干过坏事我知道，正因此你可能感觉很难使用坏人的思考方式来检查自己的漏洞，但只有这样做才能真正有效地防御。
  </p>
  <p class="graf graf--p">
   这需要练习。
  </p>
  <p class="graf graf--p">
   现在我们就来试试看，
   <strong class="markup--strong markup--p-strong">
    通过上面这条内容你能知道什么？
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    此目标人正在全家外出中，所以TA家中很可能没有人；
   </li>
   <li class="graf graf--li">
    迅速查找目标人的其他社交媒体平台账户，以搜寻此人可能的住址；
   </li>
   <li class="graf graf--li">
    通过简单的搜索立刻能知道，目标人提及的会议/活动的召开和结束时间；
   </li>
   <li class="graf graf--li">
    通过该活动的性质你能知道目标人的偏好；
   </li>
   <li class="graf graf--li">
    通过
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%ae%a9%e9%98%b4%e5%bd%b1%e8%af%b4%e8%af%9d%ef%bc%9a%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e5%b0%8f%e6%b8%b8%e6%88%8f/" href="https://www.iyouport.org/%e8%ae%a9%e9%98%b4%e5%bd%b1%e8%af%b4%e8%af%9d%ef%bc%9a%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e5%b0%8f%e6%b8%b8%e6%88%8f/" rel="noopener noreferrer" target="_blank">
     照片中阴影
    </a>
    的角度你能知道拍摄的大致时间，以对比发布时间判断同步程度；如果拍摄时间和发布时间之间间隔多小时，这将说明更多细节；
   </li>
   <li class="graf graf--li">
    仔细看照片，其中有
    <strong class="markup--strong markup--li-strong">
     哪些特征
    </strong>
    ：有电子设备吗？能看出品牌型号吗？有徽标/胸牌吗？能否判断其他参与者的身份？胸牌是门禁卡吗？这意味着安全设施方面的信息；其中多数人穿的是什么样的衣服？这可以判断
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a5%bd%e7%8e%a9%e7%9a%84%e9%97%b4%e8%b0%8d%e6%8c%91%e6%88%98/" href="https://www.iyouport.org/%e5%a5%bd%e7%8e%a9%e7%9a%84%e9%97%b4%e8%b0%8d%e6%8c%91%e6%88%98/" rel="noopener noreferrer" target="_blank">
     其阶层和品味偏好
    </a>
    ……
    <strong class="markup--strong markup--li-strong">
     所有这一切都是实施社交工程攻击和黑客入侵的关键信息。
    </strong>
   </li>
   <li class="graf graf--li">
    还可以有更多。
   </li>
  </ul>
  <p class="graf graf--p">
   也许作为你，和家人一起出行参加活动是一件乐事，但是，对于攻击者来说，您的愉快假日已经变成了明确的攻击入口。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    类似的状况几乎铺天盖地，这就是为什么我们要强调 —— 不要通过互联网讲述关于你自己和家人朋友的任何信息。是 **任何**。
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    最基本的《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e4%b8%8d%e5%ba%94%e8%af%a5%e5%9c%a8%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e4%b8%8a%e8%af%b4%e7%9a%84%e8%af%9d/" href="https://www.iyouport.org/%e4%bd%a0%e4%b8%8d%e5%ba%94%e8%af%a5%e5%9c%a8%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e4%b8%8a%e8%af%b4%e7%9a%84%e8%af%9d/" rel="noopener noreferrer" target="_blank">
     你不应该在社交媒体上说的话
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    您需要防备的不只是警察和政府间谍
   </strong>
   ，恶意行为者在社会上到处都是，他们能干出任何你可能想都想不到的事。因为他们整天都在琢磨这些东西。
  </p>
  <p class="graf graf--p">
   只要恶意者能发现您的PII，例如 您的全名、卡号、出生日期、手机号和家庭住址，那么他们就能窃取您的身份，并以您的名义开放信用额度，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/sim%e5%8a%ab%e6%8c%81%ef%bc%9a%e5%a6%82%e4%bd%95%e5%ae%9e%e6%96%bd%e4%bf%a1%e7%94%a8%e5%8d%a1%e8%af%88%e9%aa%97%e6%94%bb%e5%87%bb%ef%bc%9f/" href="https://www.iyouport.org/sim%e5%8a%ab%e6%8c%81%ef%bc%9a%e5%a6%82%e4%bd%95%e5%ae%9e%e6%96%bd%e4%bf%a1%e7%94%a8%e5%8d%a1%e8%af%88%e9%aa%97%e6%94%bb%e5%87%bb%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    劫持你的SIM
   </a>
   ，不要等钱没了时再表示惊讶。
  </p>
  <p class="graf graf--p">
   人类最大的弱点就是总会下意识地认为最糟糕的事不会发生在自己身上，这是一种心理防御机制，是你的大脑在试图把问题简化从而避免紧张焦虑的自然能力。
  </p>
  <p class="graf graf--p">
   但是，它可能很有危害性，
   <strong class="markup--strong markup--p-strong">
    正是它导致人们在看到他人的教训时将自己放在旁观者的角度上，错过了吸取教训的机会
   </strong>
   。
   <strong class="markup--strong markup--p-strong">
    也正因此，一模一样的恶行手段会不断重演，而且每次都能有效。
   </strong>
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    <em>
     学校和真实生活之间的一个重要区别在于：学校先教你知识然后考试，生活则是先考试再教你知识。你不需要亲自被考糊了才能学到东西，要知道一直不断有人被考糊，他们的教训应该成为更多人的知识。
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p">
   并不是说要你草木皆兵地生活，防御直觉指的是：您需要了解当前每个领域有可能发生哪些恶行，并随时为最糟糕的状况做好准备。
  </p>
  <p class="graf graf--p">
   你准备得越好，最糟糕的状况就越不容易发生。
  </p>
  <p class="graf graf--p">
   在这里看到有关防御直觉的更多介绍：
  </p>

   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%9e%84%e5%bb%ba%e6%8a%80%e6%9c%af%e7%9b%b4%e8%a7%89%ef%bc%9a%e7%94%9f%e5%ad%98%e5%9c%a8%e6%95%b0%e5%ad%97%e4%b8%96%e7%95%8c%e5%bf%85%e5%a4%87%e7%9a%84%e6%9c%ac%e8%83%bd/" href="https://www.iyouport.org/%e6%9e%84%e5%bb%ba%e6%8a%80%e6%9c%af%e7%9b%b4%e8%a7%89%ef%bc%9a%e7%94%9f%e5%ad%98%e5%9c%a8%e6%95%b0%e5%ad%97%e4%b8%96%e7%95%8c%e5%bf%85%e5%a4%87%e7%9a%84%e6%9c%ac%e8%83%bd/" rel="noopener noreferrer" target="_blank">
     构建技术直觉：生存在数字世界必备的本能
