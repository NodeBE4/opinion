  <p class="graf graf--p">
   英国正开始加入中国签署的万亿美元“一带一路”倡议，这背离了华盛顿和布鲁塞尔对北京采取的更具对抗性的路线。
  </p>
  <p class="graf graf--p">
   英国脱欧计划陷入僵局之后，伦敦表现出明显的脆弱感，这使得英国开始重新评估其全球贸易关系。
  </p>
  <p class="graf graf--p">
   英国财政大臣 Philip Hammond 在北京的“一带一路”论坛上
   <a class="markup--anchor markup--p-anchor" data-href="https://uk.reuters.com/article/uk-china-silkroad-britain/britain-committed-to-helping-china-realise-belt-and-road-idUKKCN1S20O5" href="https://uk.reuters.com/article/uk-china-silkroad-britain/britain-committed-to-helping-china-realise-belt-and-road-idUKKCN1S20O5" rel="noopener noreferrer" target="_blank">
    表示
   </a>
   ，英国致力于帮助中国实现其雄心勃勃但备受争议的基础设施投资项目。
  </p>
  <p class="graf graf--p">
   Hammond 访问这个备受瞩目的论坛，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.gov.uk/government/news/economic-talks-to-further-strengthen-uk-china-relations" href="https://www.gov.uk/government/news/economic-talks-to-further-strengthen-uk-china-relations" rel="noopener noreferrer" target="_blank">
    强调了
   </a>
   他所谓的“中英关系的黄金时代”，这是两国在前任总理大卫·卡梅伦（David Cameron）执政期间开始的更紧密的经济关系。
  </p>
  <p class="graf graf--p">
   尽管英国与意大利不同，目前尚未签署正式的谅解备忘录以参加“一带一路”计划，但 Hammond
   <a class="markup--anchor markup--p-anchor" data-href="https://www.ft.com/content/9f054218-66af-11e9-a79d-04f350474d62" href="https://www.ft.com/content/9f054218-66af-11e9-a79d-04f350474d62" rel="noopener noreferrer" target="_blank">
    表示
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    他将游说英国公司参与一带一路项目，尤其是金融和工程项目。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    而且，这一计划已经在逐步推进了
   </strong>
   ：总部位于伦敦的渣打银行刚刚与中国进出口银行签署了一项促进“一带一路”项目的
   <a class="markup--anchor markup--p-anchor" data-href="http://www.chinadaily.com.cn/a/201904/26/WS5cc1d980a3104842260b867e.html" href="http://www.chinadaily.com.cn/a/201904/26/WS5cc1d980a3104842260b867e.html" rel="noopener noreferrer" target="_blank">
    协议
   </a>
   。
  </p>
  <p class="graf graf--p">
   英国的献媚对中国来说是个好消息，中国一直在试图恢复该外交政策倡议失去的那些光彩，这是习近平的一个主要优先事项。
  </p>
  <p class="graf graf--p">
   自“一带一路”于2013年底宣布以来，中国的此项计划被许多批评家称之为“债务陷阱外交”，尤其是最近，
   <a class="markup--anchor markup--p-anchor" data-href="https://foreignpolicy.com/2018/10/29/sri-lankas-political-shake-up-is-a-win-for-china/" href="https://foreignpolicy.com/2018/10/29/sri-lankas-political-shake-up-is-a-win-for-china/" rel="noopener noreferrer" target="_blank">
    批评越来越多，指
   </a>
   中国利用金融力量扩大北京在整个亚洲、非洲和欧洲部分地区的政治影响力。
  </p>
  <p class="graf graf--p">
   接收中国贷款的国家，如斯里兰卡、马来西亚和缅甸，已开始对中国投资带来的繁重条款感到愤怒。美国官员一再谴责北京利用“债务链”绑架较小的国家以扩大其影响范围。
  </p>
  <p class="graf graf--p">
   然后习近平
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wsj.com/articles/chinas-xi-vows-new-direction-for-belt-and-road-after-criticism-11556249652?shareToken=st2962926313f048099a7f200bbe81c19f" href="https://www.wsj.com/articles/chinas-xi-vows-new-direction-for-belt-and-road-after-criticism-11556249652?shareToken=st2962926313f048099a7f200bbe81c19f" rel="noopener noreferrer" target="_blank">
    承诺
   </a>
   ，“一带一路”项目要“更加透明”，并控制鲁莽贷款。
  </p>
  <p class="graf graf--p">
   但是，英国的动向也是在中国和西方之间越来越多地怀疑包括网络间谍、经济竞争以及控制南中国海的酝酿争端等问题之大背景上出现的。
  </p>
  <p class="graf graf--p">
   美国现在称中国为“战略竞争对手”，而欧盟今年早些时候将中国
   <a class="markup--anchor markup--p-anchor" data-href="https://ec.europa.eu/commission/sites/beta-political/files/communication-eu-china-a-strategic-outlook.pdf" href="https://ec.europa.eu/commission/sites/beta-political/files/communication-eu-china-a-strategic-outlook.pdf" rel="noopener noreferrer" target="_blank">
    称为
   </a>
   “系统性竞争对手”。
  </p>
  <p class="graf graf--p">
   作为其反击中国影响力的活动的一部分，美国向包括英国在内的盟友施加压力，将华为公司制造的设备排除在先进的移动电话网络之外。
  </p>
  <p class="graf graf--p">
   但是，就在 Hammond 前往北京的前几天，英国宣布同意允许华为参与其新网络的“非核心”部分。
  </p>
  <p class="graf graf--p">
   尽管中国在“一带一路”论坛上的魅力令人反感，但与西方的紧张关系仍然很明显。中国上周五猛烈抨击法国在台湾海峡航行的一艘战舰。
  </p>
  <p class="graf graf--p">
   中国对附近水域的敏感性也影响了与英国的关系：Hammond 计划今年早些时候访问北京是在英国官员谈到向南海派遣航空母舰这一计划被打破之后，突显了英国政府对中国的不同看法。
  </p>
  <p class="graf graf--p">
   财政部和贸易官员渴望加深与中国的经济联系，特别是在英国脱欧后不久开始寻求新的贸易伙伴的背景下。英国贸易和出口促进部部长 Rona Fairhead 以及英国“一带一路”特使 Douglas Flint 也出席了在北京举行的论坛。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://foreignpolicy.com/2019/04/26/china-gets-a-british-bedfellow-belt-and-road-hammond-xi/" href="https://foreignpolicy.com/2019/04/26/china-gets-a-british-bedfellow-belt-and-road-hammond-xi/" rel="noopener noreferrer" target="_blank">
    China Gets a British Bedfellow
   </a>
   . Left vulnerable by Brexit, the U.K. looks eager to sign onto Beijing’s giant Belt and Road program.
  </p>
