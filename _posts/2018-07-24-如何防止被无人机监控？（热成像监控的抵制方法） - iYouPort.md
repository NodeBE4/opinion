  <div class="section-inner sectionLayout--insetColumn">
   <p class="graf graf--p graf-after--figure" id="a385">
    最早推出于六年前的 Stealth Wear 又回来了。据说新的产品将于今年的某个时候发布。本文简单介绍这种防护产品，期待很快能再次进入销售。（我们不卖产品，与厂家没有任何关系。本文是对反监控立场的支持）
   </p>
   <p class="graf graf--p graf-after--p" id="d500">
    Stealth Wear 在监控崛起的时代里致力于用时尚解决隐私问题。
   </p>
   <p class="graf graf--p graf-after--p" id="6856">
    通过
    <a class="markup--anchor markup--p-anchor" data-href="https://ahprojects.com/projects/cv-dazzle" href="https://ahprojects.com/projects/cv-dazzle" rel="noopener nofollow noreferrer" target="_blank">
     CV Dazzle
    </a>
    的先前工作，隐藏面部检测，Stealth Wear 继续探索隐私的美学和时尚的潜力，挑战专制主义的监控技术。
   </p>
   <p class="graf graf--p graf-after--p" id="d791">
    Stealth Wear 的想法在无人机战争的背景下出现，目标是切断人体与无人机的视角。物品采用镀银织物制成，可反射热辐射，使佩戴者能够避免来自头顶的热监视。
   </p>
   <p class="graf graf--p graf-after--p" id="f60c">
    Stealth Wear 于 2012 年秋季进行了研究和开发，并于 2013 年1月17日首次在伦敦坦克杂志总部展出。今年将再次回归。以下是具体测试结果。

     <p class="graf graf--p graf-after--p" id="2643">
      <strong class="markup--strong markup--p-strong">
       这是什么材料用于阻挡热成像监视？
      </strong>
     </p>
     <p class="graf graf--p graf-after--p" id="f867">
      这些服装采用镀银合成纤维制成，具有高度的柔韧性、耐磨性和热反射性。镀金属纤维反射和扩散由身体发射的热辐射，减少了使用长波红外相机（LWIR）（例如 FLIR 相机）观察下的佩戴者的热特征。
     </p>
     <p class="graf graf--p graf-after--p" id="6f7d">
      <strong class="markup--strong markup--p-strong">
       卖原料吗？
      </strong>
     </p>
     <p class="graf graf--p graf-after--p" id="acf7">
      抱歉不卖。理想材料是镀银的，具有最高的热反射率。薄材料比较厚的织物更有效地传递热量。根据您的项目预算平衡织物的成本和耐用性，压碎的银涂层会散热并减少眩光，镀金属织物对附近的热辐射具有高反射性。
     </p>
     <p class="graf graf--p graf-after--p" id="b3d6">
      <strong class="markup--strong markup--p-strong">
       有更便宜的替代品吗？
      </strong>
     </p>
     <p class="graf graf--p graf-after--p" id="7070">
      仅使用镀铝聚酯薄膜毯也可以实现成本低得多但
      <strong class="markup--strong markup--p-strong">
       仅功能性
      </strong>
      的解决方案，聚碳酸酯薄膜也具有热反射性。但是，在成本和耐用性之间存在一个权衡。太空毯是低成本的，但不是很耐磨。镀银织物更耐磨，但价格贵一些。
     </p>
