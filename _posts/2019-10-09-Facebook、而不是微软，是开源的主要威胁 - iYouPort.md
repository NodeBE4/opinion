  </p>
  <p class="graf graf--p">
   在未来，Facebook 将不会再做简单的社交媒体网站。
  </p>
  <p class="graf graf--p">
   Facebook 目前受到很多审查和压力。它被指控帮助外国恶意行为者通过广告和虚假账户来操纵选举 —— 例如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.newyorker.com/magazine/2018/10/01/how-russia-helped-to-swing-the-election-for-trump" href="https://www.newyorker.com/magazine/2018/10/01/how-russia-helped-to-swing-the-election-for-trump" rel="noopener noreferrer" target="_blank">
    在美国
   </a>
   传播谎言 —— 以及在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nbcnews.com/tech/tech-news/streamed-facebook-spread-youtube-new-zealand-shooting-video-circulates-online-n983726" href="https://www.nbcnews.com/tech/tech-news/streamed-facebook-spread-youtube-new-zealand-shooting-video-circulates-online-n983726" rel="noopener noreferrer" target="_blank">
    新西兰
   </a>
   和其他地方充当恐怖主义的传播渠道。
  </p>
  <p class="graf graf--p">
   越来越多的人开始发誓要打破 Facebook；或者至少做到控制 Facebook。
  </p>
  <p class="graf graf--p">
   更多《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%89%93%e7%a0%b4-facebook-%e7%a9%b6%e7%ab%9f%e9%9c%80%e8%a6%81%e4%bb%80%e4%b9%88%ef%bc%9f/" href="https://www.iyouport.org/%e6%89%93%e7%a0%b4-facebook-%e7%a9%b6%e7%ab%9f%e9%9c%80%e8%a6%81%e4%bb%80%e4%b9%88%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    打破 Facebook 究竟需要什么？
   </a>
   》、《如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/video%e5%a6%82%e4%bd%95%e6%89%8d%e8%83%bd%e6%8c%91%e6%88%98-facebook-%e5%b9%b6%e5%8f%96%e5%be%97%e8%83%9c%e5%88%a9%ef%bc%9f-distributed-social-network/" href="https://www.iyouport.org/video%e5%a6%82%e4%bd%95%e6%89%8d%e8%83%bd%e6%8c%91%e6%88%98-facebook-%e5%b9%b6%e5%8f%96%e5%be%97%e8%83%9c%e5%88%a9%ef%bc%9f-distributed-social-network/" rel="noopener noreferrer" target="_blank">
    何才能挑战 Facebook 并取得胜利？
   </a>
   》。
  </p>
  <p class="graf graf--p">
   为了阻止这些反对意见、并限制最近事件对 Facebook 声誉造成的影响，马克扎克伯格一直在发表一些哲学帖子，试图击败其中主要的批评者。
  </p>
  <p class="graf graf--p">
   在他最近的一篇文章中，他呼吁在四个方面对网络世界进行新的监管：有害内容、选举完整性、隐私、和数据可移植性。
  </p>
  <p class="graf graf--p">
   但是
   <strong class="markup--strong markup--p-strong">
    这种对监管的拥护本身就是骗局
   </strong>
   ，在这里看到具体解释《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%8b%a5%e6%8a%a4%e7%9a%84%e8%83%8c%e5%90%8e/" href="https://www.iyouport.org/%e6%8b%a5%e6%8a%a4%e7%9a%84%e8%83%8c%e5%90%8e/" rel="noopener noreferrer" target="_blank">
    拥护的背后
   </a>
   》。
  </p>
  <p class="graf graf--p">
   对数据可移植性的呼吁提到了 Facebook 对
   <a class="markup--anchor markup--p-anchor" data-href="https://datatransferproject.dev/" href="https://datatransferproject.dev/" rel="noopener noreferrer" target="_blank">
    数据传输项目
   </a>
   的支持。这显然是企图驳回有关 Facebook 垄断和封闭的指责。
  </p>
  <p class="graf graf--p">
   Facebook 确实使用并支持大量的开源程序，所以在这点上，这是一个公平的宣称。
  </p>
  <p class="graf graf--p">
   扎克伯格从2019年3月初开始发布的帖子篇幅要长得多，帖子概述了 Facebook 如何运用他所谓的“以隐私为重点的社交网络愿景”。
  </p>
  <p class="graf graf--p">
   当然，更好地保护隐私绝对受欢迎。但是，如果你认为扎克伯格的帖子就是你想的这样，那就太天真了。
  </p>
  <p class="graf graf--p">
   再一次，这些帖子只是为了试图阻止越来越多的批评 —— 在这种情况下，Facebook 是破坏了数据保护。
  </p>
  <p class="graf graf--p">
   这里的关键是下面这句话：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    我相信通信的未来将越来越多地转向私人加密服务，人们可以确信他们彼此之间的通话保持安全，他们的信息和内容不会永远存在
   </em>
   。
  </p>
  <p class="graf graf--p">
   虽然对于 Facebook 用户来说这听起来像是个好消息，但对于 Facebook 本身来说，这也是一个很大的优势。
  </p>
  <p class="graf graf--p">
   由于将采用端到端加密，该公司将无法看到人们在私人聊天中分享的内容。
  </p>
  <p class="graf graf--p">
   既然如此，就不能要求对这些内容进行监管了。
  </p>
  <p class="graf graf--p">
   如果 Facebook 能够实现这种“转变”，就将能减少公司的政治压力 —— 即 要求检查其用户的行为。
  </p>
  <p class="graf graf--p">
   这种方法的一个大问题是，无法知道谁发送了什么，因此当局和其他观察者将失去关于什么样的虚假信息正在传播的宝贵见解。
  </p>
  <p class="graf graf--p">
   不仅如此，
   <strong class="markup--strong markup--p-strong">
    这种大肆宣扬隐私和加密的做法还有另一个关键驱动因素
   </strong>
   。
  </p>
  <p class="graf graf--p">
   正如扎克伯格写道的：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    我们计划以我们开发 WhatsApp 的方式构建它：专注于最基本和私有的用例 —— 即 消息 —— 使其尽可能安全，然后构建更多方式让人们进行交互，包括通话、视频聊天、群组、故事、企业、支付、商业，并最终成为许多其他类型的私人服务的平台。
   </em>
  </p>
  <p class="graf graf--p">
   请注意，扎克伯格需要“尽可能安全地”提供服务，而不是为了保护用户的隐私，从而产生足够的信心，让人们愿意相信 Facebook 负责日常金融活动。
  </p>
  <p class="graf graf--p">
   也就是说，他希望将 Facebook 从社交媒体网站转变为运行各种应用程序的平台。
  </p>
  <p class="graf graf--p">
   Facebook 的用户市场已经接近枯竭，因此它需要找到新的方式来产生利润。
  </p>
  <p class="graf graf--p">
   虽然是一个大胆的愿景，但它几乎不是原创的 —— 因为这正是中国互联网巨头腾讯对微信所做的事。
  </p>
  <p class="graf graf--p">
   最初，微信只是人们与朋友群组和同事交流信息的一种方式。 到2017年，腾讯开始在其平台上运行“小程序”。就如维基百科解释的：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    企业主可以在微信系统中创建迷你应用程序，使用 JavaScript 和专有API实现。用户可以在微信应用中安装这些。 2018年1月，微信宣布了58万个小程序的纪录。通过一个迷你程序，消费者可以在超市柜台使用手机扫描QR码，并通过用户的微信手机钱包支付账单。微信游戏受到了极大的欢迎，其“Jump Jump”游戏在不到3天的时间内吸引了4亿玩家，并在推出后的两周内获得了1亿活跃用户，截至2018年1月。
   </em>
  </p>
  <p class="graf graf--p">
   如今，微信每月活跃用户超过10亿，它实际上是中国社会的操作系统。
  </p>
  <p class="graf graf--p">
   随着 Facebook 宣称转向“以隐私为重点的社交网络愿景”，扎克伯格显然希望将 Facebook 变成微信那样的操作系统。
  </p>
  <p class="graf graf--p">
   对于开源来说，这是一个巨大的问题。
  </p>
  <p class="graf graf--p">
   虽然 Linux 是当今使用的数十亿 Android 手机的基础，但是在其上运行自由软件的迹象很少。
  </p>
  <p class="graf graf--p">
   随着人们开始安装 Facebook 的迷你程序在 Android 上运行，一切依赖于 Linux 的事实变得更加无关紧要。
  </p>
  <p class="graf graf--p">
   无论新的迷你程序是否开源，Facebook 的平台肯定不会开源，无论扎克伯格多么喜欢自由软件。
  </p>
  <p class="graf graf--p">
   Facebook 将成为新的 Windows，区别在于在GNU / Linux而不是Windows中交换是很简单的；但想在 Facebook 的新平台做同样的事可不容易。
  </p>
  <p class="graf graf--p">
   社区无疑会提出一个比 Facebook 更好的 Facebook —— 毕竟，开源世界有一些非常棒的程序员，他们喜欢挑战。
  </p>
  <p class="graf graf--p">
   它可能是一个内置隐私和安全性的分布式系统。但技术细节在这里确实无关紧要。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    困难的部分不是制作更好的 Facebook，而是说服人们使用它
   </strong>
   。网络效应导致打破 Facebook 对社交媒体的控制会难以置信的困难。
  </p>
  <p class="graf graf--p">
   一旦扎克伯格建立了 Facebook 作为一个完整的平台，并且人们在他们清醒的时候经常和本能性地使用它，那将更加困难。那么开源将扮演什么角色？
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.linuxjournal.com/content/facebook-not-microsoft-main-threat-open-source" href="https://www.linuxjournal.com/content/facebook-not-microsoft-main-threat-open-source" rel="noopener noreferrer" target="_blank">
    Facebook, Not Microsoft, Is the Main Threat to Open Source
   </a>
  </p>
