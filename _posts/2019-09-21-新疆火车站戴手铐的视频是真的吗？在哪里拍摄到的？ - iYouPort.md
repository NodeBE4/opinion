  <p class="graf graf--p">
   4天前，有一段视频发布，并迅速成为热门话题 —— 很多家媒体报道这件事。
  </p>
  <p class="graf graf--p">
   视频中显示了大约 300～400 名被拘留者在新疆的火车站被戴着手铐、蒙住眼睛押送。
  </p>
  <p class="graf graf--p">
   该视频被上传到了 YouTube，如下。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/gGYoeJ5U7cQ" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   这段视频是真的吗？在哪里拍摄的？什么时间拍摄的？
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@Nrg8000" href="https://medium.com/@Nrg8000" rel="noopener noreferrer" target="_blank">
    Nathan Ruser
   </a>
   做了一个开源调查演示，以挖掘上述问题的答案。
  </p>
  <p class="graf graf--p">
   首先，看一下这个视频。
  </p>
  <p class="graf graf--p">
   它显示了一个大疆无人机的屏幕录像，该无人机捕获了数百名被押送到火车站的被拘留者（大概是维吾尔族人），这些人似乎刚刮过胡子，被蒙住眼睛、戴着手铐。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9291 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-6.jpg?resize=1100%2C815&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-6.jpg?w=1200&amp;ssl=1 1200w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-6.jpg?resize=300%2C222&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-6.jpg?resize=768%2C569&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-6.jpg?resize=1024%2C759&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-6.jpg?resize=1100%2C815&amp;ssl=1 1100w" data-recalc-dims="1" height="815" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-6.jpg?resize=1100%2C815&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  </p>
  <p class="graf graf--p">
   该视频的可验证信息很少。说明中只写着：”这些视频是在中国拍摄的”。
  </p>
  <p class="graf graf--p">
   此外，YouTube 的发布者是一个
   <strong>
    新账户，该视频是该账户发布的唯一视频
   </strong>
   。也就是说，没有上传者的相关信息可以参考。
  </p>
  <p class="graf graf--p">
   但是我们可以看一下视频本身，第一步观察视频中
   <strong>
    可供我们定位的内容
   </strong>
   ：建筑物、蜂窝塔、蓝色建筑群、停车场、树木、火车轨道等等。
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   最大的线索是左下角的方位图，上面写着：“巴音郭楞蒙古自治州”。这是新疆的一个自治州，首府是库尔勒市。指针显示为西北方向。
  </p>
  <p class="graf graf--p">
   该无人机的 FPV 摄像头几乎直指火车轨道，如果测量一下你会看到、它会显示大约282°的方向。
  </p>
  <p class="graf graf--p">
   这就缩小了可能的火车轨道范围。
  </p>

  <p class="graf graf--p">
   更大的线索是位于左下角的10公里比例尺（在我的屏幕截图上）显示为6.48cm。从Bayingol标签的大致中心到该位置的距离是8.05厘米（在〜300°的方位）(8.05/6.48)*10 = 12.42 km 与标签之间的方位角度为300°。
  </p>
  <p class="graf graf--p">
   将其输入到 Google Earth 可以助您一臂之力。
  </p>
  <p class="graf graf--p">
   您可以看到建筑物、树木、铁轨、停车场 —— 这里是2019年6月17日的卫星图像，这点非常重要，后面会提到。

  </p>
