  <p class="graf graf--p graf-after--figure" id="b84a">
   今天发表的一份新报告非常惊人，它揭示了法国政府和几家法国公司如何通过向 Sisi 政权提供军事和监视设备，从而参与了埃及过去五年的血腥镇压。
  </p>
  <p class="graf graf--p graf-after--p" id="f937">
   <strong class="markup--strong markup--p-strong">
    通过为埃及安全部门和执法机构提供强大的数字工具，这些公司帮助埃及当局建立了一个奥威尔式的大规模监控社会架构，用于消除一切形式的异议和公民行为
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="964c">
   自 2013 年7月由 Sisi 政权策划的军事政变以来，埃及社会一直处于无情的镇压阵痛中。其安全部门的镇压记录是基本是
   <strong class="markup--strong markup--p-strong">
    毁灭性
   </strong>
   的：通过军事手段驱散的和平示威活动（2013 年8月14日，人们在开罗的 Rabaa Al Adawiya 静坐，暴力镇压造成
   <strong class="markup--strong markup--p-strong">
    1000多人死亡
   </strong>
   ）;自 2013 年以来至少有
   <strong class="markup--strong markup--p-strong">
    6 万名政治犯
   </strong>
   被监禁;
   <strong class="markup--strong markup--p-strong">
    数以千计的法外处决和强迫失踪
   </strong>
   （包括 2013 年7月至 2016 年6月期间安全部门实施的
   <strong class="markup--strong markup--p-strong">
    2,811 起强迫失踪
   </strong>
   案件）; 加之系统性地使用
   <strong class="markup--strong markup--p-strong">
    酷刑
   </strong>
   ; 并且
   <strong class="markup--strong markup--p-strong">
    死刑
   </strong>
   明显增加。
  </p>
  <p class="graf graf--p graf-after--p" id="31a1">
   尽管欧盟外交事务委员会于 2013 年 8月21 日宣布“成员国已决定暂停向埃及发送任何可用于国内镇压的设备的出口许可证”，但
   <strong class="markup--strong markup--p-strong">
    至少有八家法国公司 — 受到一个又一个政府的鼓励 — 他们还是从这种镇压中获利，甚至获得了创纪录的利润
   </strong>
   。 2010 年至 2016 年间，法国向埃及运送的武器从 3960 万欧元增加到 13 亿欧元！
  </p>
  <p class="graf graf--p graf-after--p" id="a9c0">
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     延伸阅读：民主国家及其公司长期以来如何通过向专制国家出售大规模监视设备，造成严重的人权危机 — — 《
    </em>
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://steemit.com/bigdata/@iyouport/5ufmfe" href="https://steemit.com/bigdata/@iyouport/5ufmfe" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     <em class="markup--em markup--p-em">
      监视之恶（二）公私监控伙伴关系
     </em>
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     》
    </em>
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="21cb">
   “虽然欧洲理事会宣布停止出口军事设备和监视设备，以谴责埃及的独裁统治，但是法国依旧获得了市场份额，并实现了创纪录的出口！”FIDH 总裁 Dimitris Christopoulos 表示。
  </p>
  <p class="graf graf--p graf-after--p" id="aee9">
   <strong class="markup--strong markup--p-strong">
    一些公司向造成数百名平民死亡的军队出售常规武器
   </strong>
   ，包括在西奈：米斯特拉尔战舰（DCNS）; Fremm 护卫舰（DCNS）; 炮舰（Gowind）; 阵风战斗机; 装甲车（Arquus）; 云母空对空导弹，和 SCALP 巡航导弹（MBDA）; 以及两枚 ASM 空对地导弹（SAGEM）。
  </p>
  <p class="graf graf--p graf-after--p" id="1d45">
   更值得一提的式，
   <strong class="markup--strong markup--p-strong">
    一些公司已经向埃及出售了用于大规模监控社会的技术设施（AMESYS / NEXA / AM系统）; 大规模拦截技术（SUNERIS / ERCOM）; 个人数据收集技术（IDEMIA）; 和人群控制技术（赛峰无人机，AIRBUS / THALES 卫星和 Arquus 以前叫 RTD）轻型装甲车很适应城市环境）。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="0d95">
   通过这种销售，这些外国公司都参与了埃及政府广泛的监视和人群控制架构的建设，
   <strong class="markup--strong markup--p-strong">
    旨在防止所有异议和社会运动，并导致成千上万的反对者和活动家被捕。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="ce23">
   “今天法国正在通过建立奥威尔式的监视和控制系统，致力于粉碎那群在 2011 年发起著名的埃及革命的活动家们。抗议活动目前在埃及已被扼杀“，CIHRS 主任 Bahey Eldin Hassan 说。
  </p>
