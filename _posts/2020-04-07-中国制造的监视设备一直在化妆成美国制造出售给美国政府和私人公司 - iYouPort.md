   中国制造的监视设备一直在化妆成”美国制造”出售给美国政府和私人公司

     <em>
      <strong>
       这篇是官媒的报道，显然没有说清楚一件重要的事：该公司是如何战胜一大群真正的美国制造公司而直接打入美国政府机构和军事分支机构内部的？而且持续了13年之久？美国政府和军方的采购依照的标准是什么？
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-10533 jetpack-lazy-image" data-lazy-sizes="(max-width: 931px) 100vw, 931px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-14.jpg?resize=931%2C524&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-14.jpg?w=931&amp;ssl=1 931w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-14.jpg?resize=300%2C169&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-14.jpg?resize=768%2C432&amp;ssl=1 768w" data-recalc-dims="1" height="524" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-14.jpg?resize=931%2C524&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="931"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10533" data-recalc-dims="1" height="524" sizes="(max-width: 931px) 100vw, 931px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-14.jpg?resize=931%2C524&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-14.jpg?w=931&amp;ssl=1 931w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-14.jpg?resize=300%2C169&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-14.jpg?resize=768%2C432&amp;ssl=1 768w" width="931"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   纽约一家公司及其七名现任和前任雇员被指控非法进口中国制造的监视设备，然后将其出售给美国政府机构、军事分支机构和私人客户，并谎称那是美国制造的。
  </p>
  <p class="graf graf--p">
   根据当局的说法，
   <strong class="markup--strong markup--p-strong">
    Aventura Technologies 出售的中国监视设备最终出现在美国海军舰船和军事基地上
   </strong>
   ，并且在许多情况下都具有网络安全漏洞，
   <strong class="markup--strong markup--p-strong">
    自2010年以来，该公司获得了超过2000万美元的联邦合同
   </strong>
   。
  </p>
  <p class="graf graf--p">
   官方说，被指控的是 Aventura 的整个高级管理团队，包括其事实上的所有者和实际的首席执行官 Jack Cabasso，他拥有广泛的犯罪记录。
  </p>
  <p class="graf graf--p">
   指控还涉及他的妻子 Frances Cabasso，她在公司中没有实际执行职务。
  </p>
  <p class="graf graf--p">
   所有七名被告均被控以非法进口和串谋实施邮件和电汇欺诈罪。Cabassos 还被指控洗钱阴谋，被指控通过空壳公司骗取 Aventura 的非法利润。
  </p>
  <p class="graf graf--p">
   总部位于长岛 Commack 的 Aventura 办公室被联邦当局突袭。特工们在该公司查封了所有文件盒和其他证据。
  </p>
  <p class="graf graf--p">
   特工还缴获了300万美元和据称是 Aventura 拥有的豪华游艇，该游艇被 Capabassos 用于私人巡游并出租给其他人。
  </p>
  <p class="graf graf--p">
   该公司的网站将自己描述为安全硬件和软件制造商。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “由于向美国政府作出了虚假陈述，Aventura 获取了数千万美元，
   <strong class="markup--strong markup--p-strong">
    他们出售的是中国制造的监视系统，这些监视系统最终被部署在美国陆军和空军基地、能源设施、海军设施、甚至美国航母上
   </strong>
   。” Richard Donaghue 律师在纽约布鲁克林举行的新闻发布会上说。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-10534 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/中.jpg?resize=1100%2C825&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/中.jpg?w=1200&amp;ssl=1 1200w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/中.jpg?resize=300%2C225&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/中.jpg?resize=768%2C576&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/中.jpg?resize=1024%2C768&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/中.jpg?resize=1100%2C825&amp;ssl=1 1100w" data-recalc-dims="1" height="825" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/中.jpg?resize=1100%2C825&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10534" data-recalc-dims="1" height="825" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/中.jpg?resize=1100%2C825&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/中.jpg?w=1200&amp;ssl=1 1200w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/中.jpg?resize=300%2C225&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/中.jpg?resize=768%2C576&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/中.jpg?resize=1024%2C768&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/中.jpg?resize=1100%2C825&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Donaghuse 说，Aventura 将从中国进口的设备说成是在 Commack 生产的，并在设备上贴上“美国制造”的标签，作为该计划的一部分。
  </p>
  <p class="graf graf--p">
   据有关部门称，自2010年以来，该公司从中国接收了大约 1000 批 “复杂的” 电子设备。
  </p>
  <p class="graf graf--p">
   检察官还声称，Aventura 高管与中国供应商“合谋”对产品进行“修改”，以掩盖其在中国制造的事实。
  </p>
  <p class="graf graf--p">
   Donaghue 说，这包括修改电路板上的标记和软件代码中的某些“身份标识”。
  </p>
  <p class="graf graf--p">
   Donaghue 说，
   <strong class="markup--strong markup--p-strong">
    据称该公司从事设备的进口和转售已超过13年
   </strong>
   。他说，某些系统具有 “已知的漏洞”。
  </p>
  <p class="graf graf--p">
   Jack Cabasso 没有获得保释。检察官希望将他一直拘留直到审判。
  </p>
  <p class="graf graf--p">
   他的妻子以一百万美元的保释金获释。
  </p>
  <p class="graf graf--p">
   检察官在一份法庭文件中说，Jack Cabasso “策划”了“国际欺诈阴谋”，并使用洗钱技术（包括空壳公司和与第三方受益人的房地产交易）“赚了大钱”。
  </p>
  <p class="graf graf--p">
   该文件说，Cabasso “漫长的犯罪历史” 包括 “涉及欺骗和干预司法程序的众多罪名。”
  </p>
  <p class="graf graf--p">
   1992年，Cabasso 因涉嫌串谋影响陪审员而被定罪，他在另一起刑事案件中被指控电汇欺诈计划。他还于1982年被判犯有大型盗窃罪，以及1985年的盗窃罪企图。
  </p>
  <p class="graf graf--p">
   检察官的备案文件指出：“在2000年3月，Cabasso 因参与一项股票欺诈计划而被定罪，据报他从16,000名投资者手里窃取了1.76亿美元。”
  </p>
  <p class="graf graf--p">
   Cabasso 在2012年因涉嫌欺诈性使用商业伙伴信用卡来支付Google广告活动（该广告活动将流量吸引到 Aventura 网站）的计划而被捕，最终无罪释放。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    进口廉价设备并重新标记为 “美国制造” 的情况比较普遍。
   </strong>
  </p>
  <p class="graf graf--p">
   但是，在 Aventura 案中的指控说明了尽管美国试图限制进口某些中国技术，但中国设备仍最终出现在供应链中，甚至在敏感的政府设施中也被使用。
  </p>
  <p class="graf graf--p">
   美国在其禁止的供应商名单中又增加了一些中国公司，原名单中已经包括该国一些最大的设备制造商比如华为和中兴通讯。新名单包括监视行业巨头海康威视和大华科技，以及在数字取证、信息安全和面部识别领域的重要中国公司。
  </p>
  <p class="graf graf--p">
   海康威视对 CNBC 称，该公司交付给 Aventura 的所有产品均未包含任何标签来表明原产地为“美国制造”。⚪️
  </p>
