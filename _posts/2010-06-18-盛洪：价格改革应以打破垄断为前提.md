  <div>
   <b>
    <font size="2">
     理财一周报：为什么选择在当前涨价？涨价幅度超过预期？
    </font>
   </b>
  </div>
  <div>
   <b>
   </b>
  </div>
  <div>
   <font size="2">
    <b>
     盛洪：
    </b>
    天然气是一种稀缺资源，价格应该反映其稀缺性。但是涨价不应该是企业的涨价，而是企业应该向政府支付资源的租金的提高。
   </font>
  </div>
  <div>
   <font size="2">
    资源租金和资源税的概念不太一样。租金是由资源的稀缺性决定的，税是企业向政府交纳一笔钱，因为政府给企业提供了公共服务。在我国经常会把资源税和资源租金混为一谈，概念上是不清楚的。
   </font>
  </div>
  <div>
   <font size="2">
    不能将天然气涨价理解成天然气改革，涨价可能是石油天然气垄断集团政策游说的结果。
   </font>
  </div>
  <div>
  </div>
  <div>
   <b>
    <font size="2">
     理财一周报：那怎样才是真正意义的改革？
    </font>
   </b>
  </div>
  <div>
  </div>
  <div>
   <b>
    <span style="font-size: 9pt">
     盛洪
    </span>
   </b>
   <span style="font-size: 9pt">
    ：
   </span>
   <span style="font-size: 9pt">
    天然气应该是市场定价的，市场化才应该是改革的方向。天然气的定价制度改革应该是市场化定价改革，这在许多国家都有现成的案例。
   </span>
  </div>
  <div style="text-indent: 22.5pt">
   <span style="font-size: 9pt">
    比如美国，天然气是市场竞价的，因为有多家生产天然气公司。但输气管道是自然垄断的，输气管道的租金就由政府管制，输气管道每输一立方米的气，就按管制的水平收取租金。天然气价格形成是市场竞争的结果。这是第一点。
   </span>
  </div>
  <div style="text-indent: 22.5pt">
   <span style="font-size: 9pt">
    第二点，输气管道企业要和上下游分离，既不能做气田的开发，也不能做天然气的零售。这两端应该是竞争的，这样就能知道他们是怎样进行天然气的市场竞价。两端都是竞争的，中间是受政府管制的管道租金，等于气的运输费政府是管制了，那么其他地方都是放开的，让多家企业去竞价。所以美国的天然气价格是浮动的，老百姓也都接受。
   </span>
  </div>
  <div style="text-indent: 22.5pt">
   <span style="font-size: 9pt">
    所以中国的提价不能叫改革，甚至是反改革的。
   </span>
  </div>
  <div>
  </div>
  <div>
   <b>
    <font size="2">
     理财一周报：之前也有过两次的天然气价格调整，而这一次是取消了"双轨制"，有何意义？
    </font>
   </b>
  </div>
  <div>
   <b>
   </b>
  </div>
  <div>
   <b>
    <span style="font-size: 9pt">
     盛洪：
    </span>
   </b>
   <span style="font-size: 9pt">
    这个"双轨制"不是原来我们讲的那个"双轨制"：一轨是计划轨，另一轨是市场轨。现在"双轨制"，只是两个管制价格而已，不过其中一个可以由垄断企业浮动一定幅度。所以谈不上"双轨制"。这个并轨有什么意义，原来这两个轨都不是市场定的价，现在的并轨无非是从低的价格往高的价格上去并，那怎么叫"双轨制"？把两种管制价格并轨为一种管制价格，给垄断企业定价空间，说是"市场化"，说是"改革"，这是瞒天过海，偷换概念。
   </span>
  </div>
  <div>
  </div>
  <div>
   <b>
    <span style="font-size: 9pt">
     理财一周报：价格改革，只是天然气改革的第一步？接下去要做的是什么？放开对民间资本的限制吗？
    </span>
   </b>
  </div>
  <div>
  </div>
  <div>
   <b>
    <span style="font-size: 9pt">
     盛洪：
    </span>
   </b>
   <span style="font-size: 9pt">
    不是因为开采天然气风险大，所以民间资本不进去，而是行政部门不让他们进去。你要是放开，民间资本就会进去。
   </span>
   <span style="font-size: 9pt">
    3 8
   </span>
   <span style="font-size: 9pt">
    号文件（
   </span>
   <span style="font-size: 9pt">
    《关于清理整顿小炼油厂和规范原油成品油流通秩序的意见》
   </span>
   <span style="font-size: 9pt">
    ）把几乎所有方面的垄断权交给了中石油、中石化。
   </span>
  </div>
  <div>
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    前几年还有陕北油田的问题，陕北油田是硬从民营企业那里抢回来的，所以不存在民营企业不愿意进去的问题。真正的改革首先是要让大量的企业进入到石油天然气的开采领域，而不是由中石油、中石化这两大家垄断。
   </span>
  </div>
  <div>
  </div>
  <div>
   <b>
    <span style="font-size: 9pt">
     理财一周报：提高国产陆上天然气出厂价格对居民的影响？
    </span>
   </b>
  </div>
  <div>
   <b>
   </b>
  </div>
  <div>
   <b>
    <span style="font-size: 9pt">
     盛洪：
    </span>
   </b>
   <span style="font-size: 9pt">
    影响应该是比较小的，因为水、气涨价带来的费用实际上并不大，只占一般居民的收入的很小比率，如果比率是
   </span>
   <span style="font-size: 9pt">
    1%
   </span>
   <span style="font-size: 9pt">
    ，那确实是影响不大。但是对贫困人口是一个问题，所以政府应该对这些人口进行补贴。
   </span>
  </div>
  <div>
  </div>
  <div>
   <b>
    <span style="font-size: 9pt">
     理财一周报：能源改革似乎给人的印象是价格改革就等同于上涨，对此您是怎么看的呢？
    </span>
   </b>
  </div>
  <div>
   <b>
   </b>
  </div>
  <div>
   <b>
    <span style="font-size: 9pt">
     盛洪：
    </span>
   </b>
   <span style="font-size: 9pt">
    第一是大家对垄断企业的不信任。老百姓不知道涨价部分到底给谁，天然气涨价了，钱都变成中石化、中石油的利润了。它们又不交利润。所以老百姓对这种情况比较愤怒。
   </span>
  </div>
  <div style="text-indent: 22.5pt">
   <span style="font-size: 9pt">
    第二点，征收资源税和环境税是合理的，但这确实影响了居民的收入。所以我认为在增税的同时还要等量的减税。
   </span>
  </div>
  <div style="text-indent: 22.5pt">
   <span style="font-size: 9pt">
    资源税和环境税所要解决的问题是一个局部性的问题，国家通过经济手段让居民去节约资源这是对的。但不应因此降低居民和企业的收入水平。平衡的做法是可以降低其它税种的税率，如企业所得税、个人所得税。
   </span>
  </div>
  <div>
  </div>
  <div>
   <b>
    <span style="font-size: 9pt">
     理财一周报：但现在的情况是加税容易，减税难？
    </span>
   </b>
  </div>
  <div>
  </div>
  <div>
   <b>
    <span style="font-size: 9pt">
     盛洪
    </span>
   </b>
   <span style="font-size: 9pt">
    ：减税比较难出台就是政府的问题，他只是想从老百姓手里多拿些钱，不想减少居民的负担，而实际上政府应该是有长远眼光的，政府不是为政府自身而存在的，而是为社会而存在的。如果不为老百姓服务，多拿的钱是没有意义的。
   </span>
  </div>
  <div>
  </div>
  <div>
   <b>
    <span style="font-size: 9pt">
     理财一周报：研究报告显示，天然气涨价和资源税征收对冲后，两大石油公司仍是最大受益者，从这点上看，资源垄断性企业似乎是最大的赢家？
    </span>
   </b>
  </div>
  <div>
   <b>
   </b>
  </div>
  <div>
   <b>
    <span style="font-size: 9pt">
     盛洪：
    </span>
   </b>
   <span style="font-size: 9pt">
    现在中石油、中石化每吨石油直接给国家
   </span>
   <span style="font-size: 9pt">
    24
   </span>
   <span style="font-size: 9pt">
    元
   </span>
   <span style="font-size: 9pt">
    -30
   </span>
   <span style="font-size: 9pt">
    元，大概叫作资源使用费。但这远远低于资源租金。如果以资源租金算，怎么也得
   </span>
   <span style="font-size: 9pt">
    300
   </span>
   <span style="font-size: 9pt">
    元至
   </span>
   <span style="font-size: 9pt">
    400
   </span>
   <span style="font-size: 9pt">
    元每吨。其实中石油、中石化根本就不赚钱，只是他们少交资源租金，少交了地租，甚至少纳了税，这剩下的钱就变成了所谓"利润"。
   </span>
  </div>
  <div>
   <span style="font-size: 9pt">
   </span>
   <span style="font-size: 9pt">
    中石油、中石化认为自己的利益是刚性的，政府征收资源税，他们就游说政府涨价。为什么中石油、中石化的利润就一定是刚性的呢？如果征收资源租金，他们根本无利润可言，他们盈利是因为他们少交了很多本该上缴的钱。把这部分钱当成自己的合法的收益了。
   </span>
  </div>
  <div>
  </div>
  <div>
   <b>
    <span style="font-size: 9pt">
     理财一周报：在进行资源改革时，不能有效地引进竞争机制的障碍在哪里？
    </span>
   </b>
  </div>
  <div>
   <b>
   </b>
  </div>
  <div>
   <b>
    <span style="font-size: 9pt">
     盛洪：
    </span>
   </b>
   <span style="font-size: 9pt">
    一是
   </span>
   <span style="font-size: 9pt">
    特殊的垄断利益集团守住垄断利益不退，二是政府没有下定决心去打破垄断。不过这些垄断企业太有实力了，既有政治实力，又有经济实力。这是我们国家政治结构的问题。
   </span>
  </div>
  <div>
  </div>
  <div>
   <b>
    <span style="font-size: 9pt">
     理财一周报：如果是政治结构问题，那么当前和今后的一系列改革都可能是治标不治本？
    </span>
   </b>
  </div>
  <div>
  </div>
  <div>
   <b>
    <span style="font-size: 9pt">
     盛洪
    </span>
   </b>
   <span style="font-size: 9pt">
    ：是的。解决问题的根本办法就是要打破垄断，就是要废掉石油领域的
   </span>
   <span style="font-size: 9pt">
    38
   </span>
   <span style="font-size: 9pt">

   </span>
   <span style="font-size: 9pt">
    36
   </span>
   <span style="font-size: 9pt">
    条"，但是如果不废止一些相关的所谓"文件"，就无法真正实行。
   </span>
   <span style="font-size: 9pt">
   </span>
  </div>
  <div style="text-indent: 13.5pt">
   <span style="font-size: 9pt">
    其实最重要的还是要有多家企业竞争，多家企业生产石油、天然气，这样就有市场，有市场才有市场价格，这就理顺了，不然永远理不顺。
   </span>
