  <p class="graf graf--p">
   从购物到银行再到登机，基于面部识别的经济正在日本扎根，消费者被置于无现金的生活状态。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    现金为什么是敌人？因为隐私是敌人，隐私是敌人是因为它使政府控制变得更加困难《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%81%90%e6%80%96%e7%9a%84%e5%8f%8d%e7%8e%b0%e9%87%91%e4%b9%8b%e6%88%98/" href="https://www.iyouport.org/%e6%81%90%e6%80%96%e7%9a%84%e5%8f%8d%e7%8e%b0%e9%87%91%e4%b9%8b%e6%88%98/" rel="noopener noreferrer" target="_blank">
     恐怖的反现金之战
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   当游客降落在和歌山县西部的南纪白滨机场时，他们看到的不是白川市闻名的熊猫，而是电视屏幕上显示的 “欢迎先生/女士 ——
   <strong class="markup--strong markup--p-strong">
    访客的真实姓名
   </strong>
   ’’。
  </p>
  <p class="graf graf--p">
   访客可以在商店里
   <strong class="markup--strong markup--p-strong">
    刷脸
   </strong>
   购买口香糖，也可以
   <strong class="markup--strong markup--p-strong">
    不带钥匙
   </strong>
   进入酒店房间。他们可以前往世界主题公园，在这里客人可以预先注册以
   <strong class="markup--strong markup--p-strong">
    通过面部识别
   </strong>
   进入，观看熊猫和海豚。
  </p>
  <p class="graf graf--p">
   这些是NEC在白滨町进行的面部识别实验的一部分。
  </p>
  <p class="graf graf--p">
   这家日本电子制造商押注，所谓的便利性与对数据安全性的信心结合起来将 “成为面部识别技术市场上的制胜法宝”。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     《
    </strong>
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%be%bf%e5%88%a9%e7%9a%84%e6%9a%b4%e6%94%bf/" href="https://www.iyouport.org/%e4%be%bf%e5%88%a9%e7%9a%84%e6%9a%b4%e6%94%bf/" rel="noopener noreferrer" target="_blank">
     <strong class="markup--strong markup--li-strong">
      便利的暴政
     </strong>
    </a>
    <strong class="markup--strong markup--li-strong">
     》
    </strong>
   </li>
  </ul>
  <p class="graf graf--p">
   据印度研究公司 Mordor Intelligence 称，随着技术扩展到自动驾驶、金融和医疗，全球面部识别市场将从2018年到2024年翻一番，
   <strong class="markup--strong markup--p-strong">
    达到91亿美元
   </strong>
   。
  </p>
  <p class="graf graf--p">
   毫无疑问，中国公司在该领域占据主导地位，因为它们在规模和准确性上均名列前茅。但是，随着华盛顿和北京在贸易和技术上争执不休，日本企业正在看到 “机遇”。
  </p>
  <figure aria-describedby="caption-attachment-12185" class="wp-caption aligncenter" id="attachment_12185" style="width: 770px">
   <img alt="" class="size-full wp-image-12185 jetpack-lazy-image" data-lazy-sizes="(max-width: 770px) 100vw, 770px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/图.jpg?resize=770%2C433&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/图.jpg?w=770&amp;ssl=1 770w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/图.jpg?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/图.jpg?resize=768%2C432&amp;ssl=1 768w" data-recalc-dims="1" height="433" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/图.jpg?resize=770%2C433&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="770"/>

    <img alt="" class="size-full wp-image-12185" data-recalc-dims="1" height="433" sizes="(max-width: 770px) 100vw, 770px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/图.jpg?resize=770%2C433&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/图.jpg?w=770&amp;ssl=1 770w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/图.jpg?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/图.jpg?resize=768%2C432&amp;ssl=1 768w" width="770"/>

   <figcaption class="wp-caption-text" id="caption-attachment-12185">
    Visitors can enter theme park Adventure World in western Japan’s Wakayama Prefecture with their faces. (Photo by Kosuke Shimizu)
   </figcaption>
  </figure>
  <p class="graf graf--p">
   NEC 目前世界领先的面部识别技术是基于半个世纪前该公司构建的技术。
   <strong class="markup--strong markup--p-strong">
    1989年该公司就开始研究面部识别了
   </strong>
   ，早期的错误识别率约为30％。
  </p>
  <p class="graf graf--p">
   而现在该公司采用人工智能并开发了原始算法，稳步提高了其技术的准确性和速度。
  </p>
  <p class="graf graf--p">
   现在，NEC的错误识别率已降至0.5％，而其系统每秒可进行2.3亿次惊人的识别操作。速度和容量远远超过其他公司的能力，这就是NEC的技术在美国国家标准技术研究院（NIST）基准测试中排名第一的原因。
  </p>
  <p class="graf graf--p">
   然而，尽管NEC拥有技术优势 —— 包括应对衰老和遮盖脸部的识别能力 —— 但中国的监视相机和面部识别系统供应商仍在全球市场上抢占了很大的份额。
  </p>
  <p class="graf graf--p">
   执政的共产党对监视每一个公民的巨大需求帮助催生了蓬勃发展的监视部门。估计有2亿台监视摄像机散布在中国各地。消费者很少有理由抵制面部识别系统。
  </p>
  <p class="graf graf--p">
   杭州海康威视数字技术有限公司是一家获利的中国企业。它已成为世界上最大的监控摄像头制造商，部分原因是其提供的产品价格低于发达经济体供应商收取的价格的一半。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    列表-4 “
    <a class="markup--anchor markup--li-anchor" data-href="https://start.me/p/aL8RrM/iyp-4" href="https://start.me/p/aL8RrM/iyp-4" rel="noopener noreferrer" target="_blank">
     作恶者：谁在帮中国
    </a>
    ”
   </li>
  </ul>
  <p class="graf graf--p">
   在NIST的2018年排名前五名中仅落后于NEC的三位的中国参与者是：易图科技、深圳先进技术研究院、和SenseTime。
  </p>
  <p class="graf graf--p">
   与2013年相比，这是令人震惊的一步，当时前五名竞争对手中完全没有中国公司。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12186 jetpack-lazy-image" data-lazy-sizes="(max-width: 769px) 100vw, 769px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/图2-10.png?resize=769%2C487&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/图2-10.png?w=769&amp;ssl=1 769w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/图2-10.png?resize=300%2C190&amp;ssl=1 300w" data-recalc-dims="1" height="487" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/图2-10.png?resize=769%2C487&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="769"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12186" data-recalc-dims="1" height="487" sizes="(max-width: 769px) 100vw, 769px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/图2-10.png?resize=769%2C487&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/图2-10.png?w=769&amp;ssl=1 769w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/图2-10.png?resize=300%2C190&amp;ssl=1 300w" width="769"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   但是，随着华盛顿和北京就贸易和技术争执不休，新的壁垒正在上升。海康威视的增长速度已经放缓，2019年上半年销售额同比增长10％ —— 是2017年全年增长率的三分之一。
  </p>
  <p class="graf graf--p">
   去年10月，特朗普政府禁止了28个中国高科技设备公司。海康威视、Yitu 和 SenseTime 均在列表中。该禁令限制了对软件、半导体和其他重要组件的采购。
  </p>
  <p class="graf graf--p">
   对于 NEC 之类的日本公司来说，美国的决定以及对中国监控行业的普遍怀疑给日本监视技术提供了一个机会窗口。
  </p>
  <p class="graf graf--p">
   NEC与日本国立信息与通信技术研究所共同开发了一种基于量子密码技术的面部识别系统，这种形式的加密技术被认为几乎无法破解。该公司希望通过强调 “降低泄漏风险” 来推销该监视系统。（也就是竭尽全力维持监视的存在状态）
  </p>
  <p class="graf graf--p">
   中国企业以低价打入了新兴市场，这意味着日本和其他竞争对手也需要提供负担得起的选择来提高竞争力。
  </p>
  <p class="graf graf--p">
   松下在去年11月推出了一种基于云的面部认证系统，采用即付即用模式。该公司对每张身份验证收取1日元（不足1美分）—— 它愿意以低利润率来推广该服务。
  </p>
  <p class="graf graf--p">
   但是，隐私问题可能会以无法预料的方式发挥作用。微软总裁 Brad Smith 去年曾表示：“可以说，面部识别精灵刚刚从瓶子里冒出来。”
  </p>
  <p class="graf graf--p">
   他警告说
   <strong class="markup--strong markup--p-strong">
    这些系统可能会被用来歧视少数群体、侵犯人们的隐私、并导致全球范围内的大规模监视
   </strong>
   。他说：“我们相信，各国政府必须开始采用新法律来规范这项技术。”
  </p>
  <p class="graf graf--p">
   旧金山市今年禁止警察和其他机构使用面部识别软件。欧洲联盟也正在认真考虑类似的举动。
  </p>
  <p class="graf graf--p">
   在微软的推荐之后，NEC 发表了一份声明，总裁兼首席执行官 Takashi Niino 表示，他“欢迎这次辩论”。⚪️
  </p>
