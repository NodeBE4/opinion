   自2013年以来，由国家资助的中国黑客一直在瞄准东南亚

      <strong>
       您以为自己下载的是VPN，但事实上那是一个间谍软件 …… 这是来自中国的攻击者使用的手段
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-9754 jetpack-lazy-image" data-lazy-sizes="(max-width: 796px) 100vw, 796px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/中国.jpg?resize=796%2C417&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/中国.jpg?w=796&amp;ssl=1 796w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/中国.jpg?resize=300%2C157&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/中国.jpg?resize=768%2C402&amp;ssl=1 768w" data-recalc-dims="1" height="417" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/中国.jpg?resize=796%2C417&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="796"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9754" data-recalc-dims="1" height="417" sizes="(max-width: 796px) 100vw, 796px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/中国.jpg?resize=796%2C417&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/中国.jpg?w=796&amp;ssl=1 796w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/中国.jpg?resize=300%2C157&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/中国.jpg?resize=768%2C402&amp;ssl=1 768w" width="796"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   研究人员透露，一些未公开的来自中国的攻击者自2013年以来在东南亚地区至少开展了六次不同的网络间谍活动。
  </p>
  <p class="graf graf--p">
   Palo Alto Networks 威胁情报小组 Unit 42
   <a class="markup--anchor markup--p-anchor" data-href="https://unit42.paloaltonetworks.com/pkplug_chinese_cyber_espionage_group_attacking_asia/" href="https://unit42.paloaltonetworks.com/pkplug_chinese_cyber_espionage_group_attacking_asia/" rel="noopener noreferrer" target="_blank">
    披露的调查结果
   </a>
   将这些攻击与一个名为 PKPLUG 的组织联系在一起，该组织的名字来自于其在ZIP文件中传送
   <a class="markup--anchor markup--p-anchor" data-href="https://logrhythm.com/blog/deep-dive-into-plugx-malware/" href="https://logrhythm.com/blog/deep-dive-into-plugx-malware/" rel="noopener noreferrer" target="_blank">
    PlugX 恶意软件
   </a>
   ，并带有签名
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/List_of_file_signatures" href="https://en.wikipedia.org/wiki/List_of_file_signatures" rel="noopener noreferrer" target="_blank">
    “PK”
   </a>
   。
  </p>
  <p class="graf graf--p">
   Unit 42 表示，其归因含糊不清是因为“我们目前的可见度无法让我们高度确定这是一个小组的工作，还是使用相同工具、执行相同任务的多个小组的协同工作”。
  </p>
  <p class="graf graf--p">
   PKPLUG 被发现可以在受害者的系统（包括移动设备）上安装后门特洛伊木马植入程序，以跟踪和收集信息，尽管其最终动机尚不清楚。
  </p>
  <p class="graf graf--p">
   该攻击组织的主要目标包括缅甸、台湾、越南、印度尼西亚，以及蒙古、西藏和新疆，后者均以与中国的有争议的关系而闻名。
  </p>
  <p class="graf graf--p">
   特别是新疆，是该国维吾尔族穆斯林少数民族的家园。近年来，维吾尔族穆斯林一直受到迫害和加强的监视。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e5%9b%be%e8%a7%a3%e6%96%b0%e7%96%86%e5%86%8d%e6%95%99%e8%82%b2%e8%90%a5/" href="https://www.iyouport.org/%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e5%9b%be%e8%a7%a3%e6%96%b0%e7%96%86%e5%86%8d%e6%95%99%e8%82%b2%e8%90%a5/" rel="noopener noreferrer" target="_blank">
     开源调查图解新疆”再教育”营
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   Unit 42 的 Alex Hinchliffe 说：“这个攻击小组（或多个小组）
   <strong class="markup--strong markup--p-strong">
    拥有悠久的历史，并且创建了一系列定制工具，这意味着它们具有持久性和丰富的资源
   </strong>
   。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “例如，自定义 Android 恶意软件的创建和使用[…]可能表明其目标需要基于大众所普遍使用的操作系统、或需要的功能。这个小组很有耐心。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    PKPLUG 操作方式
   </strong>
  </p>
  <p class="graf graf--p">
   Unit 42 的报告结合了自己的研究成果和 Blue Coat Labs 及 Arbor Networks 等其他网络安全公司发布的研究成果，汇总了 PKPLUG 基于恶意软件的策略、技术和程序（TTP）的时间表。
  </p>
  <p class="graf graf--p">
   据说最早的 PKPLUG 攻击是在2013年11月发生的，当时他们被发现使用 PlugX 恶意软件针对蒙古人。
  </p>
  <p class="graf graf--p">
   三年后，他们再次被发现使用
   <a class="markup--anchor markup--p-anchor" data-href="https://www.trendmicro.com/vinfo/us/threat-encyclopedia/malware/poisonivy" href="https://www.trendmicro.com/vinfo/us/threat-encyclopedia/malware/poisonivy" rel="nofollow noopener noreferrer" target="_blank">
    Poison Ivy
   </a>
   恶意软件针对缅甸和其他亚洲国家的人。
  </p>
  <p class="graf graf--p">
   同年，Unit 42 报告了通过鱼叉式网络钓鱼电子邮件的攻击，其中电子邮件用于下载托管在 Google 云端硬盘上的ZIP文件，该文件加载了
   <a class="markup--anchor markup--p-anchor" data-href="https://unit42.paloaltonetworks.com/unit-42-attack-delivers-9002-trojan-through-google-drive/" href="https://unit42.paloaltonetworks.com/unit-42-attack-delivers-9002-trojan-through-google-drive/" rel="noopener noreferrer" target="_blank">
    木马程序
   </a>
   以破坏受害者的设备。
  </p>
  <p class="graf graf--p">
   在2018年初，该公司发现了一个名为
   <a class="markup--anchor markup--p-anchor" data-href="https://unit42.paloaltonetworks.com/unit42-henbox-chickens-come-home-roost/" href="https://unit42.paloaltonetworks.com/unit42-henbox-chickens-come-home-roost/" rel="noopener noreferrer" target="_blank">
    HenBox
   </a>
   的新恶意软件家族，⚠️
   <strong class="markup--strong markup--p-strong">
    它伪装成合法的 Android 应用程序，主要针对维吾尔族人并针对性危害小米设备，目的是捕获打入中国的外来电话、并访问手机的麦克风和摄像头
   </strong>
   。
  </p>
  <p class="graf graf--p">
   值得注意的是 PKPLUG 邪恶的另一招，以诱使毫无戒心的用户：
   <strong class="markup--strong markup--p-strong">
    攻击者使间谍软件在第三方 Android 应用商店上可用，⚠️甚至冒充名为 DroidVPN 的VPN应用来诱使用户下载
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9755 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.jpeg?resize=1024%2C527&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.jpeg?w=1024&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.jpeg?resize=300%2C154&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.jpeg?resize=768%2C395&amp;ssl=1 768w" data-recalc-dims="1" height="527" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.jpeg?resize=1024%2C527&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9755" data-recalc-dims="1" height="527" sizes="(max-width: 1024px) 100vw, 1024px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.jpeg?resize=1024%2C527&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.jpeg?w=1024&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.jpeg?resize=300%2C154&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.jpeg?resize=768%2C395&amp;ssl=1 768w" width="1024"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   VPN 应用程序是中国公民绕过防火墙的最常见方式，而且，它们被用作攻击媒介的事实表明，⚠️
   <strong class="markup--strong markup--p-strong">
    网络犯罪分子越来越倾向于
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://thenextweb.com/google/2019/09/20/vpn-apps-with-millions-of-installs-caught-serving-disruptive-ads-to-android-users/" href="https://thenextweb.com/google/2019/09/20/vpn-apps-with-millions-of-installs-caught-serving-disruptive-ads-to-android-users/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     依靠用户对与安全相关的应用程序的信任
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    来安装恶意软件
   </strong>
   。
  </p>
  <p class="graf graf--p">
   快进到2019年2月，Unit 42 发现了该攻击组织对 Windows 已知后门程序
   <a class="markup--anchor markup--p-anchor" data-href="https://unit42.paloaltonetworks.com/farseer-previously-unknown-malware-family-bolsters-the-chinese-armoury/" href="https://unit42.paloaltonetworks.com/farseer-previously-unknown-malware-family-bolsters-the-chinese-armoury/" rel="noopener noreferrer" target="_blank">
    Farseer
   </a>
   的使用，该后门程序利用侧载库来安装恶意有效载荷 — 一种将其不同活动连接在一起的常用技术。
  </p>
  <p class="graf graf--p">
   该研究指出，PKPLUG 对域名和IP地址的重用也有助于加强不同网络攻势之间的重叠。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    端点安全性的需求
   </strong>
  </p>
  <p class="graf graf--p">
   去年10月，Check Point Research
   <a class="markup--anchor markup--p-anchor" data-href="https://thenextweb.com/security/2019/10/02/chinese-hacking-group-targets-southeast-asian-governments-with-data-stealing-malware/" href="https://thenextweb.com/security/2019/10/02/chinese-hacking-group-targets-southeast-asian-governments-with-data-stealing-malware/" rel="noopener noreferrer" target="_blank">
    披露了
   </a>
   由一个名为 Rancor 的中国骇客组织针对东南亚进行的类似数据窃取活动，此后 PKPLUG 精心策划的网络间谍攻击的细节才被曝光。
  </p>
  <p class="graf graf--p">
   当被问及这些国家赞助的恶意行为者是否可以协同工作以大规模开展有针对性的攻击行动时，Hinchliffe 表示：“我们已经看到像俄罗斯这样的民族国家可以雇用各种团体来发动攻击，尽管在这种情况下有可能，但我们目前没有足够的数据来验证这种假设。”
  </p>
  <p class="graf graf--p">
   除了避免从第三方 Android 商店下载应用程序外，Hinchliffe 还建议企业保护其网络端点，并通过限制设备对网络的访问来评估其 BYOD（“自带设备”的缩写）策略。
  </p>
  <p class="graf graf--p">
   尽管这项最新研究可以一窥 PKPLUG 的活动，但远非定论。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “也许还需要更多的案例研究。对 PKPLUG 等威胁性组织的了解越多，防御的方向就会越完整”，Hinchliffe 说。⚪️
  </p>
  <p class="graf graf--p">

  </p>
