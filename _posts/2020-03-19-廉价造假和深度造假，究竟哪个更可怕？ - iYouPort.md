 <div class="entry-content">
  <p class="graf graf--p">
   在地缘政治信息战中，廉价造假比深度造假更好用；但对于网络攻击来说，深度造假是值得警惕的问题
  </p>
  <p class="graf graf--p">
   在关于深度造假的诸多讨论中，最有趣的一个现象就是：
   <strong class="markup--strong markup--p-strong">
    政府党派背景资助的调查报告全部倾向于夸大深度造假的能力，并强行将讨论重点放置于地缘政治威胁的角度上
   </strong>
   。
  </p>
  <p class="graf graf--p">
   事实上这是一种偏见 —— 有利于地缘政治斗争/民族主义的偏见。因为深度造假技术最大的危害并非是信息战。
  </p>
  <p class="graf graf--p">
   以下是
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@thegrugq/cheap-fakes-beat-deep-fakes-b1ac91e44837" href="https://medium.com/@thegrugq/cheap-fakes-beat-deep-fakes-b1ac91e44837" rel="noopener noreferrer" target="_blank">
    thaddeus t. grugq 的文章
   </a>
   ，他说出了一些关键的问题。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10359 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-6.jpg?resize=1100%2C825&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-6.jpg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-6.jpg?resize=300%2C225&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-6.jpg?resize=768%2C576&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-6.jpg?resize=1024%2C768&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-6.jpg?resize=1100%2C825&amp;ssl=1 1100w" data-recalc-dims="1" height="825" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-6.jpg?resize=1100%2C825&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10359" data-recalc-dims="1" height="825" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-6.jpg?resize=1100%2C825&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-6.jpg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-6.jpg?resize=300%2C225&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-6.jpg?resize=768%2C576&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-6.jpg?resize=1024%2C768&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/头-6.jpg?resize=1100%2C825&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   我一直对深度造假持怀疑态度。它们能有什么用？对于信息战中虚假信息所使用的深度造假的实用性，我从未感到兴奋。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%b7%b1%e5%ba%a6%e9%80%a0%e5%81%87%e7%9c%9f%e7%9a%84%e6%98%af%e4%b8%80%e5%9c%ba%e7%9c%9f%e7%9b%b8%e5%b7%b2%e6%ad%bb%e7%9a%84%e5%8d%b1%e6%9c%ba%e5%90%97%ef%bc%9f%e6%88%91%e4%bb%ac%e5%a6%82%e4%bd%95/" href="https://www.iyouport.org/%e6%b7%b1%e5%ba%a6%e9%80%a0%e5%81%87%e7%9c%9f%e7%9a%84%e6%98%af%e4%b8%80%e5%9c%ba%e7%9c%9f%e7%9b%b8%e5%b7%b2%e6%ad%bb%e7%9a%84%e5%8d%b1%e6%9c%ba%e5%90%97%ef%bc%9f%e6%88%91%e4%bb%ac%e5%a6%82%e4%bd%95/" rel="noopener noreferrer" target="_blank">
     深度造假真的是一场真相已死的危机吗？我们如何化解这场危机？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   信息战不需要深度造假的东西也能实现，廉价的假货已经绰绰有余了。
  </p>
  <p class="graf graf--p">
   最后，有人发现了深度造假作为攻击性网络工具的用途，因此，让我们深入研究一下这些伪造品吧。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    关于深度造假的炒作
   </strong>
  </p>
  <p class="graf graf--p">
   通常，关于深度造假的威胁是作为
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e6%88%98%e5%92%8c%e4%bf%a1%e6%81%af%e6%88%98/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e6%88%98%e5%92%8c%e4%bf%a1%e6%81%af%e6%88%98/" rel="noopener noreferrer" target="_blank">
    信息战
   </a>
   挑战提出的，认为该技术将使虚假信息的应对变得更加困难。
  </p>
  <p class="graf graf--p">
   这一观点无处不在。这是最近的一个例子：
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theverge.com/2019/6/13/18677847/deep-fakes-regulation-facebook-adam-schiff-congress-artificial-intelligence" href="https://www.theverge.com/2019/6/13/18677847/deep-fakes-regulation-facebook-adam-schiff-congress-artificial-intelligence" rel="noopener noreferrer" target="_blank">
    国会正努力监管深度造假
   </a>
   。
  </p>
  <p class="graf graf--p">
   但事实上，深度造假对信息操作的威胁很小。
  </p>
  <p class="graf graf--p">
   因为一般而言，人们并不是为了改变自己的想法而进行修辞的，而是为了与他人建立联盟。
   <strong class="markup--strong markup--p-strong">
    在寻求提供证据以支持自己的既有观点时，人们只是在寻找那些可以抹黑反对者观点的同时捍卫自己观点的示例。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    无法使深度造假在此方面比现有的廉价造假技术更有
   </strong>
   效，例如，对视听媒体的操纵、语境错误和去语境化的视听媒体等。
  </p>
  <p class="graf graf--p">
   深度造假对信息战构成威胁是因为它们很容易掩盖虚假的谎言。
  </p>
  <p class="graf graf--p">
   信息战的基本规则之一是，你永远不要说谎（除非有必要）。深度造假可以作为人造内容被检测到，从而揭示了谎言。这就能区分信息的来源和其他论点。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    对于信息战来说，使用伪造品是高风险的主张
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     完全真实的但片面的信息才是信息战决胜的关键
    </strong>
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%93%8d%e7%ba%b5%e5%bc%8f%e5%ae%a3%e4%bc%a0%ef%bc%9a%e4%ba%94%e4%b8%aa%e8%bf%87%e6%bb%a4%e5%99%a8%e5%92%8c%e8%af%86%e5%88%ab%e7%ba%bf%e7%b4%a2/" href="https://www.iyouport.org/%e6%93%8d%e7%ba%b5%e5%bc%8f%e5%ae%a3%e4%bc%a0%ef%bc%9a%e4%ba%94%e4%b8%aa%e8%bf%87%e6%bb%a4%e5%99%a8%e5%92%8c%e8%af%86%e5%88%ab%e7%ba%bf%e7%b4%a2/" rel="noopener noreferrer" target="_blank">
     操纵式宣传：五个过滤器和识别线索
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    对深度造假的怀疑
   </strong>
  </p>
  <p class="graf graf--p">
   我一直对 Deepfake 虚假信息的实用性表示怀疑。它们根本是不必要的。它们没有提供任何信息战操纵者无法通过廉价造假就能做到的事。
  </p>
  <p class="graf graf--p">
   有关造假、去背景化、伪造背景化的研究和书籍已经非常多；例如《21世纪的宣传与信息战》非常推荐阅读，您可以在这里下载电子版：
   <a class="markup--anchor markup--p-anchor" data-href="https://t.me/iyouport/6399" href="https://t.me/iyouport/6399" rel="nofollow noopener noreferrer" target="_blank">
    https://t.me/iyouport/6399
   </a>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    深度造假真正的威胁
   </strong>
  </p>
  <p class="graf graf--p">
   1、
   <strong class="markup--strong markup--p-strong">
    个人更容易受到深度造假的影响
   </strong>
   ，因为他们更容易受到针对性的攻击。例如人格暗杀运动的目标人。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%ba%e6%a0%bc%e6%9a%97%e6%9d%80%e5%92%8c%e6%8a%b9%e9%bb%91%e8%bf%90%e5%8a%a8%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%8f%97%e7%9b%8a%ef%bc%9f/" href="https://www.iyouport.org/%e4%ba%ba%e6%a0%bc%e6%9a%97%e6%9d%80%e5%92%8c%e6%8a%b9%e9%bb%91%e8%bf%90%e5%8a%a8%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%8f%97%e7%9b%8a%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     人格暗杀和抹黑运动：谁在受益？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   2、攻击者
   <strong class="markup--strong markup--p-strong">
    容易利用不良的安全程序和流程
   </strong>
   。
  </p>
  <p class="graf graf--p">
   安全不仅仅是技术访问控制问题，它还是遵循的流程以解决人为因素的问题。如果没有所有这些措施并保存一致，那么真正的安全状态就很薄弱。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    内部威胁
   </strong>
  </p>
  <p class="graf graf--p">
   高风险的人员和团体必须具有减轻内部威胁的安全程序。例如，访问控制过程应该可以有效禁止任何一个人随意转移金钱。
  </p>
  <p class="graf graf--p">
   如果没有这样的程序，他们就很容易受到内部威胁的利用。
  </p>
  <p class="graf graf--p">
   高风险个人经常求助于技术解决方案以减轻诸如身份盗用之类的威胁（例如使用多因素身份验证）。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    他们最需要做的应该是执行有效的安全程序以防止被招募为不知情的代理人 —— 作为社交工程攻击的突破口
   </strong>
   （例如，对接近他们的人进行审查）。这个主题太丰富了，无法在本文中完全探讨。
  </p>
  <p class="graf graf--p">
   内部威胁以多种形式出现，以下是部分列表：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    由外部恶意威胁制造的
    <strong class="markup--strong markup--li-strong">
     社交工程攻击的突破口
    </strong>
    。电子邮件是用于实施这类攻击的常用媒介，有时是各种形式的网络钓鱼，而有时只是一种通信渠道。
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     冒充他人
    </strong>
    通常是通过电子邮件进行的，例如 许多针对政治运动或其他公司的骗局。
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     恶意代理
    </strong>
    ，例如 心怀不满的员工或由威胁参与者雇佣的员工，甚至是恶意威胁参与者通过就业申请获得的合法访问权限。例子不胜枚举。
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     人为错误
    </strong>
    。人类思维的固有弱点是最难避免的东西。
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     更多详细攻击模式见
    </strong>
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
     <strong class="markup--strong markup--li-strong">
      “社交工程” 栏目
     </strong>
    </a>
    。
   </li>
  </ul>
  <p class="graf graf--p">
   公司受到内部威胁的方式和可能的手段有很多，但是大多数讨论都集中在&lt;故意的&gt;恶意威胁参与者身上。但是，关于身份盗用和不知情状况下被利用的讨论却很少，这才是深度造假更容易发挥作用的地方。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    利用访问控制抵挡内部威胁
   </strong>
  </p>
  <p class="graf graf--p">
   深度造假可以绕开某些常用的防御监测，例如通过语音身份验证进行确认。
  </p>
  <p class="graf graf--p">
   如果唯一能防止将大量资金从公司账户转移到任意账户的方法是“确认某位高管通过电话发出的声音”，那么该公司就很容易受到内部人利用深度造假实施的威胁。
  </p>
  <p class="graf graf--p">
   ⚠️这是Deepfake具有强大效用的一种攻击方式，它利用了错误的访问控制，例如 语音识别授权。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    个人和公司遭受基于深度造假攻击的风险要大得多 —— 而不是国际地缘政治游戏
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这是需要解决的问题。从安全角度来看，解决方案通常是绕道，也就是实施可减轻深度造假能力的防御过程。
  </p>
  <p class="graf graf--p">
   对于面临迫害的个人，我没有答案。互联网迫害是一个尚未解决的问题。⚪️
  </p>
