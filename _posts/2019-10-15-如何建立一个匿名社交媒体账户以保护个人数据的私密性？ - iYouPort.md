   本指南介绍如何制作匿名的 Facebook 账户，但是要知道，⚠️这些方法也同样适合于 Twitter和其他寡头控制的社交媒体平台。

  <p>
   马上开始。
  </p>
  <p class="graf graf--p">
   步骤是这样的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第一步：制作一个 Burner Email 或一次性电话号码
   </strong>
  </p>
  <p class="graf graf--p">
   首先，您需要使用
   <strong class="markup--strong markup--p-strong">
    *仅用于*新*
   </strong>
   个人资料的电子邮件帐户或电话号码进行注册。
  </p>
  <p class="graf graf--p">
   ⚠️如果您使用已经预先存在的电子邮件帐户或电话号码，您的邮件、搜索历史记录、关联的应用程序等等一切，都是 Facebook 及其广告客户的免费大餐。
  </p>
  <p class="graf graf--p">
   最好的做法是什么？当然是白手起家。
  </p>
  <p class="graf graf--p">
   对于 Burner Email，您可以通过几种方式进行操作。
  </p>
  <p class="graf graf--p">
   首先，您可以注册一个新的电子邮件帐户，并仅用于创建您的匿名 Facebook 个人资料。
  </p>
  <p class="graf graf--p">
   您可以选择最适合您的电子邮件服务 —— ⚠️重点是
   <strong class="markup--strong markup--p-strong">
    确保它仅用于您的新Facebook帐户
   </strong>
   。

   💡更安全的方法是利用 Temp Email 应用程序。

    <img alt="" class="aligncenter size-full wp-image-5715" data-recalc-dims="1" height="1044" sizes="(max-width: 1070px) 100vw, 1070px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/6-3.png?resize=1070%2C1044&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/6-3.png?w=1070&amp;ssl=1 1070w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/6-3.png?resize=300%2C293&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/6-3.png?resize=768%2C749&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/6-3.png?resize=1024%2C999&amp;ssl=1 1024w" width="1070"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第五步：删除和禁用位置历史数据
   </strong>
  </p>
  <p class="graf graf--p">
   Facebook 有一项称为“位置历史记录”的功能，它是随着您使用该应用程序而制作的一组位置记录。
  </p>
  <p class="graf graf--p">
   但是，由于您的位置可用于识别您的身份，因此为了匿名，您应该将其关闭。（在设置里面，按照步骤做）
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第六步：创建一个新的 Facebook 帐户
   </strong>
  </p>
  <p class="graf graf--p">
   一旦您考虑了上述步骤，无论是在iOS还是Android上，现在是时候创建新的 Facebook 个人资料了。
  </p>
  <p class="graf graf--p">
   前往Facebook.com，或打开 Metal 以开始使用。
  </p>
  <p class="graf graf--p">
   点击“创建新帐户”，然后输入假名字。
  </p>
  <p class="graf graf--p">
   接下来，输入您的一次性电话号码，或通过点击下一步的“使用电子邮件注册”输入您的一次性电子邮件。完成后，点按“下一步”。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5716 jetpack-lazy-image" data-lazy-sizes="(max-width: 1068px) 100vw, 1068px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-2.png?resize=1068%2C602&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-2.png?w=1068&amp;ssl=1 1068w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-2.png?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-2.png?resize=768%2C433&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-2.png?resize=1024%2C577&amp;ssl=1 1024w" data-recalc-dims="1" height="602" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-2.png?resize=1068%2C602&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1068"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5716" data-recalc-dims="1" height="602" sizes="(max-width: 1068px) 100vw, 1068px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-2.png?resize=1068%2C602&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-2.png?w=1068&amp;ssl=1 1068w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-2.png?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-2.png?resize=768%2C433&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/7-2.png?resize=1024%2C577&amp;ssl=1 1024w" width="1068"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   接下来，输入密码，然后点击“注册”。
  </p>
  <p class="graf graf--p">
   如果出现提示，您可以选择是否一键登录 Facebook，而不是使用您的姓名和密码登录。
  </p>
  <p class="graf graf--p">
   如果您提供了电话号码，Facebook 将向您发送短信验证代码，如果您输入了电子邮件，则会向您发送验证邮件。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5717 jetpack-lazy-image" data-lazy-sizes="(max-width: 1070px) 100vw, 1070px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/8-2.png?resize=1070%2C596&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/8-2.png?w=1070&amp;ssl=1 1070w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/8-2.png?resize=300%2C167&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/8-2.png?resize=768%2C428&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/8-2.png?resize=1024%2C570&amp;ssl=1 1024w" data-recalc-dims="1" height="596" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/8-2.png?resize=1070%2C596&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1070"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5717" data-recalc-dims="1" height="596" sizes="(max-width: 1070px) 100vw, 1070px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/8-2.png?resize=1070%2C596&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/8-2.png?w=1070&amp;ssl=1 1070w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/8-2.png?resize=300%2C167&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/8-2.png?resize=768%2C428&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/8-2.png?resize=1024%2C570&amp;ssl=1 1024w" width="1070"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   欢迎来到您的新的、匿名的以隐私为中心的 Facebook 帐户。
  </p>
  <p class="graf graf--p">
   现在，您无需交出个人信息即可查看群组、活动等。
  </p>
  <p class="graf graf--p">
   配置完这些设置后……
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第七步：更改您的 Facebook 隐私设置
   </strong>
  </p>
  <p class="graf graf--p">
   仅仅因为您在上述步骤中采取了预防措施，并不能保护您免受 Facebook 上的某些隐私侵权行为的侵害。
  </p>
  <p class="graf graf--p">
   默认情况下，您的帐户的隐私设置并不完美，您使用帐户的时间越长保密性越低。
  </p>
  <p class="graf graf--p">
   请更改设置。
  </p>
  <p class="graf graf--p">
   虽然 Facebook 仍然有很多方法可以在事后滥用您的隐私，但更改这些设置可以为您提供一些装备。在今天的世界里，你只需要拿走你能得到的东西。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    把帖子设置为仅限朋友；把简介设置为仅限自己；
   </li>
   <li class="graf graf--li">
    好友列表设置为私密；
   </li>
   <li class="graf graf--li graf--startsWithDoubleQuote">
    “时间线”设置下选择“仅限朋友”或“仅限自己”才能发帖；
   </li>
   <li class="graf graf--li graf--startsWithDoubleQuote">
    “位置”设置为“从不”
   </li>
   <li class="graf graf--li">
    点击“帐户设置”下的“面部识别”选择“不”；
   </li>
   <li class="graf graf--li">
    在“帐户设置”中的“广告”下，您会找到“您的信息”等设置，其中会显示广告客户可以访问您的个人资料的哪些部分，在此禁用所有选项。
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第八步：阻止隐藏权限（Android — 需要Root）
   </strong>
  </p>
  <p class="graf graf--p">
   虽然 Android 确实允许用户管理应用程序可访问的权限，但 Facebook 可以访问得更多，并且默认情况下您无法对这些侵犯进行任何阻止操作。
  </p>
  <p class="graf graf--p">
   例如，即使关闭位置权限，Facebook 仍然可以使用您的网络确定您的大致位置，这是位置权限未涵盖的内容。
  </p>
  <p class="graf graf--p">
   这就是您需要 App Ops 的原因。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     Play Store Link:

     <strong class="markup--strong markup--li-strong">
      App Ops
     </strong>
    </a>
    <strong class="markup--strong markup--li-strong">
     (free)
    </strong>
   </li>
  </ul>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5718 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9.jpg?resize=1100%2C344&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9.jpg?w=1312&amp;ssl=1 1312w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9.jpg?resize=300%2C94&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9.jpg?resize=768%2C240&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9.jpg?resize=1024%2C320&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9.jpg?resize=1100%2C344&amp;ssl=1 1100w" data-recalc-dims="1" height="344" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9.jpg?resize=1100%2C344&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5718" data-recalc-dims="1" height="344" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9.jpg?resize=1100%2C344&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9.jpg?w=1312&amp;ssl=1 1312w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9.jpg?resize=300%2C94&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9.jpg?resize=768%2C240&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9.jpg?resize=1024%2C320&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/9.jpg?resize=1100%2C344&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   最初是 Android 4.3 Jelly Bean 中的隐藏功能，App Ops 允许您调整各个应用程序的大量权限，包括默认情况下 Android 没有控件的应用程序。
  </p>
  <p class="graf graf--p">
   此应用程序还将在最后授予权限时通知您。
  </p>
  <p class="graf graf--p">
   对于大多数权限，您要么允许应用程序访问、要么拒绝它、要么仅在使用时允许，从而节省电池电量并保护您的隐私。
  </p>
  <p class="graf graf--p">
   但对于Facebook，应该禁止一切。
  </p>
  <p class="graf graf--p">
   使用 App Ops 的最简单方法是 root 设备。但是，也可以在没有 root 的情况下使用，这需要ADB命令。
  </p>
  <p class="graf graf--p">
   安装完成后，打开 App Ops 并选择“Facebook”。然后逐行选择每个条目并拒绝其访问，阻止应用程序了解您的身份。⚪️
  </p>
