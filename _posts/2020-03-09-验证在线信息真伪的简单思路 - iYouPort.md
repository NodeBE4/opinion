  <p class="graf graf--p">
   我们介绍过很多信息验证的方法和技术工具，本文将侧重于思考方式 ——
   <strong class="markup--strong markup--p-strong">
    思考方式是最重要的东西，因为你可能拥有一大堆工具，却不知道该怎么使用，是思考方式在指导快速而有意义的实践
   </strong>
   。
  </p>
  <p class="graf graf--p">
   信息验证主要是关于重复性、持久性、以及使用具有一点创造力的数字调查工具。
  </p>
  <p class="graf graf--p">
   就如我们曾经介绍过的那样；现在有很多验证技巧和工具。
   <strong class="markup--strong markup--p-strong">
    实际上，最困难的部分可能是如何记住您可以使用的所有资源。
   </strong>
  </p>
  <p class="graf graf--p">
   这就是思考方式的重要性所在 —— 精简的验证向导指南。这里推荐一本书，它包括基本概念、清单、以及提示和技术。在文末看到。
  </p>
  <p class="graf graf--p">
   最重要的是，它将为您介绍验证的五个支柱，并有望作为如何解决每个支柱的快速参考。
  </p>
  <p class="graf graf--p">
   互联网时代信息传播是如此之快 —— 而且假消息的传播永远都比真相更快，以至于每个人（不仅是技术记者和社交媒体编辑）都必须了解基本的真伪验证技能。
  </p>
  <p class="graf graf--p">
   新闻编辑室还需要保护自己，避免被骗和无意间向更多听众介绍虚假信息。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    许多虚假信息战操纵者会将其精心制作的假消息被新闻媒体报道视为最终目标，并利用在线空间操纵内容，以期扩大受众。
   </strong>
  </p>
  <p class="graf graf--p">
   而且视觉效果是传播虚假信息的一种特别有效的方法，因为人们更倾向于相信自己看到的东西。与文本虚假信息相比，被恶意编造的图像也更难检测。
  </p>
  <p class="graf graf--p">
   验证技术一直在不断变化，科技平台会修改其隐私设置、删除研究工具并构建新工具；
   <strong class="markup--strong markup--p-strong">
    就如调查技术和安全防御技术一样，信息验证工具也会不断变化，以适应新的局面
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在进行在线验证之前，请问自己
   <strong class="markup--strong markup--p-strong">
    第一个最基本的问题
   </strong>
   是：我正在查看的内容是否与实际发生的事件有关？
  </p>
  <p class="graf graf--p">
   在某些情况下，例如突发新闻事件，这个问题可能正是您要通过验证确定的事。但是在某些情况下不是。
  </p>
  <p class="graf graf--p">
   想象一下，您发现了一个视频，讲述的是某个机场排长队、乘客表达不满。
   <strong class="markup--strong markup--p-strong">
    在确认视频的拍摄者、日期和时间、真正的位置之前，您的第一个问题应该是：机场是否真的有相关问题报告？
   </strong>
  </p>
  <p class="graf graf--p">
   这就是帮助您更快速达成目标的思考方式。
  </p>
  <p class="graf graf--p">
   另一个例子是2016年美国大选之前臭名昭著的标题：“教皇赞同唐纳德·特朗普”。您可以对网站、创建者、文章的发布时间、以及传播的范围等线索进行所有方式的验证，但是，文章的基本主张应该是您首先要检查的内容。
  </p>

  <p class="graf graf--p">
   幸运的是，它很容易分解。因为无论您是在观看目击者视频、被操纵的照片、伪造的账号、还是一个meme，您必须对它们进行的基本检查都是相同的，如下：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     出处
    </strong>
    ：您正在查看的是原始账户、文章或内容吗？
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     来源
    </strong>
    ：谁创建了该帐户或文章、或捕获了原始内容？
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     日期
    </strong>
    ：它是何时创建的？
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     位置
    </strong>
    ：这些内容是在哪里捕获/创建的？真实的位置而不是发布者声称的；
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     动机
    </strong>
    ：这些人为什么要创建这样的内容、网站或账户？有可能谁受益？
   </li>
  </ul>
  <p class="graf graf--p">
   您对每个支柱了解得越多，您的验证就越强大。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     确定性
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   验证很难是万无一失的。它更像是寻找线索并收集确凿的证据。
  </p>
  <p class="graf graf--p">
   想象一个老式的侦探调查小组，把可见的一些信息都固定在墙板上：位置、名称、明确的内容主题。
  </p>
  <p class="graf graf--p">
   线索之间的线条呈现出相关性的网络。如今也是如此：记者和调查人员一直使用相同的旧方法来弄清事实，但只是把这些方法应用于互联网了。
  </p>
  <p class="graf graf--p">
   因此，
   <strong class="markup--strong markup--p-strong">
    您需要做的就是尽可能地收集证据；但要意识到，您将不会总是能得到非常明确的答案。
   </strong>
  </p>
  <p class="graf graf--p">
   还要认识到：不确定性会笼罩整个过程，这意味着我们要
   <strong class="markup--strong markup--p-strong">
    更加清楚自己所知道的、和自己所不知道的东西
   </strong>
   ，尤其是，如果这些信息将为您的调查报告提供帮助的话。要非常小心。
  </p>
  <p class="graf graf--p">
   以下是为验证视觉媒体而建立的主题，可帮助您评估验证的严密性以及漏洞的位置。
  </p>

  <p class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     1、了解自己的弱点
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   验证通常需要一些时间；并且，某些时候，它可能会导致您走上毫无结果的迷途。学会弄清楚什么时候应该放弃 —— 无法验证真伪时放弃报道。
  </p>
  <p class="graf graf--p">
   我们是人类，很容易对某段内容的真假作出自然反应。但是也正因此，必须时刻保持怀疑态度。
   <strong class="markup--strong markup--p-strong">
    记住老编辑的格言：即使您妈妈说她爱您，也请检查核对。
   </strong>
  </p>
