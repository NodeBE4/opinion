  <p class="graf graf--p">
   从IYP将“高堡奇人”作为地缘政治主要标签的那一刻起，这个问题就非常值得警惕了。
  </p>
  <p class="graf graf--p">
   那是在所谓的贸易战开始
   <strong class="markup--strong markup--p-strong">
    之前很久
   </strong>
   。最初它是以中美瓜分全球互联网的数字殖民生态为根本的 —— 全世界的人只有两个选择，要么硅谷、要么BAT；虽然如今依旧如此，但是欧洲的技术公司正在崭露头角、反数字殖民已经成为国际权利组织的主题之一。
  </p>
  <p class="graf graf--p">
   但无疑，高堡奇人不会停留在互联网上。
  </p>
  <p class="graf graf--p">
   是时候认识到，这是一个新局面的开始。高堡奇人，虽然至少目前为止我们还不叫它新冷战。
  </p>
  <p class="graf graf--p">
   很多观察者已经开始使用 #newcoldwar 的标签，但我们依旧认为不合适，它与上一次冷战有着根本性区别，不仅由于这一次并非意识形态对峙。（这里才是新冷战，一份保密报告曝光，主题是《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e6%91%a7%e6%af%81%e4%bf%84%e7%bd%97%e6%96%af%ef%bc%9f/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e6%91%a7%e6%af%81%e4%bf%84%e7%bd%97%e6%96%af%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    如何摧毁俄罗斯
   </a>
   ？》）
  </p>
  <p class="graf graf--p">
   就如美国观察家多次指出的，美国对中国和俄罗斯有着完全不同的思考方式。
  </p>
  <p class="graf graf--p">
   ⚠️请注意，美国的技术产品的确
   <strong class="markup--strong markup--p-strong">
    曾经有可能
   </strong>
   比中国和俄罗斯的产品更安全，但
   <strong class="markup--strong markup--p-strong">
    绝非
   </strong>
   是技术问题；而是，透明度革命支持者在努力揭露它，不论是 Wikileaks 曾经曝光的震惊世界的 Vault 7，还是 Snowden 揭露的全球性间谍阴谋。
  </p>
  <p class="graf graf--p">
   透明度革命需要来自本地内部知情者勇敢的曝光内幕；如果中国有这样的勇士，中国的技术产品也将提高安全性。
  </p>
  <p class="graf graf--p">
   但随着
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%BB%8E%E4%BA%BA%E6%A0%BC%E6%9A%97%E6%9D%80%E5%88%B0%E5%AF%B9%E7%9C%9F%E7%9B%B8%E7%9A%84%E8%B0%8B%E6%9D%80-%E2%8F%B3/" href="https://www.iyouport.org/%E4%BB%8E%E4%BA%BA%E6%A0%BC%E6%9A%97%E6%9D%80%E5%88%B0%E5%AF%B9%E7%9C%9F%E7%9B%B8%E7%9A%84%E8%B0%8B%E6%9D%80-%E2%8F%B3/" rel="noopener noreferrer" target="_blank">
    美国对透明度革命的全力扼杀
   </a>
   ，真相有可能会越来越少，美国的产品可能很快将与中国和俄罗斯的一样阴暗。
  </p>
  <p class="graf graf--p">
   美国评论说的没错，美国将他们的产品卖给了中国，而中国人试图利用他们所获得的东西赚取了比美国人想象的更多的钱。不仅是“Money”而且是“Wealth”。
   <strong class="markup--strong markup--p-strong">
    中国和美国都在采取拖延战术，玩一场让全世界成为输家的游戏，而他们自己是唯一的赢家
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这就是为什么要呼吁人们离开高堡奇人的世界。留在这场对峙中只能成为任意帝国的炮灰。
  </p>
  <p class="graf graf--p">
   对此我们曾经谈到过解决方案，在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%b6%85%e8%b6%8a%e9%ab%98%e5%a0%a1%e5%a5%87%e4%ba%ba%ef%bc%9a%e5%9c%a8%e7%be%8e%e4%b8%ad%e7%9a%84%e5%b8%9d%e5%9b%bd%e4%ba%89%e9%9c%b8%e4%b9%8b%e4%b8%8a/" href="https://www.iyouport.org/%e8%b6%85%e8%b6%8a%e9%ab%98%e5%a0%a1%e5%a5%87%e4%ba%ba%ef%bc%9a%e5%9c%a8%e7%be%8e%e4%b8%ad%e7%9a%84%e5%b8%9d%e5%9b%bd%e4%ba%89%e9%9c%b8%e4%b9%8b%e4%b8%8a/" rel="noopener noreferrer" target="_blank">
    超越高堡奇人：在美中的帝国争霸之上
   </a>
   》，虽然很遗憾理解者仍不够多。
  </p>
  <p class="graf graf--p">
   以下是荷兰作家 Ivo Vegter 的文章，很高兴欧洲的观察者能清楚地看到这一问题的根本。可为什么中国和美国的观察者却在努力回避？
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4636 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-9.png?resize=1100%2C621&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-9.png?w=1250&amp;ssl=1 1250w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-9.png?resize=300%2C169&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-9.png?resize=768%2C434&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-9.png?resize=1024%2C578&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-9.png?resize=1100%2C621&amp;ssl=1 1100w" data-recalc-dims="1" height="621" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-9.png?resize=1100%2C621&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   美国已将中国网络设备和智能手机制造商华为列入黑名单，以满足其对本土公司的要求：向政府监控机器提供数据。
   <strong class="markup--strong markup--p-strong">
    大规模监视和威权主义正沿着冷战的沟壑撕裂互联网和技术市场
   </strong>
   。
  </p>
  <p class="graf graf--p">
   美国称华为威胁国家安全。华为是全球最大的电信设备制造商之一，也是仅次于三星的第二大智能手机制造商。凭借180,000名员工和70亿美元的利润，收入超过900亿美元，它在2017财富500强排行榜上
   <a class="markup--anchor markup--p-anchor" data-href="http://fortune.com/global500/huawei-investment-holding/" href="http://fortune.com/global500/huawei-investment-holding/" rel="noopener noreferrer" target="_blank">
    排名第72位
   </a>
   。
  </p>
  <p class="graf graf--p">
   美国的决定已经有效地禁止了美国公司安装华为网络设备，或将组件出口到华为。美国官员还告诉盟国，如果他们使用华为或其他中国技术来建立他们的下一代网络，美国将停止与他们分享情报。
  </p>
  <p class="graf graf--p">
   黑名单可能对华为产生重大影响；华为将损失掉主要客户，并削减与谷歌和微软等科技巨头的联系。例如，它无法在智能手机上安装 Android 操作系统或 Google 应用生态系统（尽管
   <strong class="markup--strong markup--p-strong">
    有很简单的方法可以解决这些限制
   </strong>
   ）。
  </p>
  <p class="graf graf--p">
   此举是在美国和中国未能在贸易战中达成停战协议的几天后出现的。特朗普要求中国在一个月内达成协议，否则所有商品面临更严格的关税。
  </p>
  <p class="graf graf--p">
   这是美中之间长期贸易战中最新的一场战争，其中美国既是侵略者，也是最大的输家。这是因为特朗普似乎相信
   <strong class="markup--strong markup--p-strong">
    在18世纪晚期已经过时
   </strong>
   的重商主义，如今仍是个好主意。
  </p>
  <p class="graf graf--p">
   重商主义是基于贸易是零和游戏的概念，也就是说，如果你的贸易伙伴获利，那就意味着你输了。它旨在限制进口，促进出口，并保护强大的国内利益免受外国竞争。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    重商主义曾经通过建立殖民帝国和使常备军队和海军与其他经济大国保持战争来实施
   </strong>
   。
  </p>
  <p class="graf graf--p">
   由于这些方法如今已经不再受欢迎了，今天的保护主义措施主要限于出口补贴、进口关税以及偶尔的直接贸易禁令。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这些政策适用于激发经济民粹主义和民族主义言论的选民的兴奋，但它实际上损害了保护其工业的国家消费者的利益
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://taxfoundation.org/trade-china-tariffs-economic-impact/#threatened" href="https://taxfoundation.org/trade-china-tariffs-economic-impact/#threatened" rel="noopener noreferrer" target="_blank">
    税务专家已经警告
   </a>
   ，特朗普的关税将降低GDP、提高价格、降低工资、并减少美国的就业率。通过加强美元，进口变得更便宜、出口更加昂贵，这直接抵消了对减少进口并增加出口的希望。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.forbes.com/sites/francescoppola/2018/09/18/president-trumps-tariffs-will-hurt-america-more-than-china/#33456d6e6276" href="https://www.forbes.com/sites/francescoppola/2018/09/18/president-trumps-tariffs-will-hurt-america-more-than-china/#33456d6e6276" rel="noopener noreferrer" target="_blank">
    关税主要伤害的是自己的人
   </a>
   。
  </p>
  <p class="graf graf--p">
   我说特朗普似乎喜欢重商主义，因为他很难确定这个主题。一方面，他一直在说“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nbcnews.com/politics/donald-trump/trump-defends-tariff-plan-says-trade-wars-are-good-markets-n852581" href="https://www.nbcnews.com/politics/donald-trump/trump-defends-tariff-plan-says-trade-wars-are-good-markets-n852581" rel="noopener noreferrer" target="_blank">
    贸易战是好的
   </a>
   ”；而另一方面，他告诉 G7 领导人 —— 就在他对欧洲、加拿大和墨西哥征收关税之后 —— 他说
   <a class="markup--anchor markup--p-anchor" data-href="https://www.businessinsider.co.za/trump-suggests-dropping-all-tariffs-trade-barriers-at-g7-summit-2018-6" href="https://www.businessinsider.co.za/trump-suggests-dropping-all-tariffs-trade-barriers-at-g7-summit-2018-6" rel="noopener noreferrer" target="_blank">
    世界应该放弃所有贸易壁垒
   </a>
   、关税和补贴，采取自由贸易。
  </p>
  <p class="graf graf--p">
   华为设备可能确实存在安全风险。今年3月，英国监督委员会发布了
   <a class="markup--anchor markup--p-anchor" data-href="https://assets.publishing.service.gov.uk/government/uploads/system/uploads/attachment_data/file/790270/HCSEC_OversightBoardReport-2019.pdf" href="https://assets.publishing.service.gov.uk/government/uploads/system/uploads/attachment_data/file/790270/HCSEC_OversightBoardReport-2019.pdf" rel="noopener noreferrer" target="_blank">
    一份年度报告
   </a>
   ，该报告确定了华为产品中存在的安全问题，并且在之前的报告中发现的问题没有取得进展。它的结论是，很难管理在英国网络中使用华为产品所带来的风险。
  </p>
  <p class="graf graf--p">
   今年四月，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bloomberg.com/news/articles/2019-04-30/vodafone-found-hidden-backdoors-in-huawei-equipment" href="https://www.bloomberg.com/news/articles/2019-04-30/vodafone-found-hidden-backdoors-in-huawei-equipment" rel="noopener noreferrer" target="_blank">
    彭博社报道说
   </a>
   ，沃达丰近十年前发现在几个欧洲国家的网络设备上存在潜在的后门，但它认为这些问题已得到解决。该公司公开反对针对华为的诉讼，理由是价格和延误是主要问题。
  </p>
  <p class="graf graf--p">
   荷兰情报部门也声称已经在一个主要的国家电信网络中找到了一个后门，它归因于华为，并表示后门可能允许该公司访问客户数据并可能将其传递给中国政府。
  </p>

   从其与最严重的暴君和反对自由世界的独裁者站在一起的记录来判断，我不相信南非国家的政府会选择相比下较小的邪恶者。
  </p>
  <p class="graf graf--p">
   同样不难想象南非政府要求在国内电信中使用的设备在当地生产的未来。它可以找到一大堆理由：对外国间谍的恐惧、强大国内监视和审查的能力、刺激当地科技产业发展，以及为受青睐的供应商提供政治赞助。
  </p>
  <p class="graf graf--p">
   所谓的“国家安全”是一个强有力的借口，但它不会使重商主义或专制政府对普通公民的危害更小。⚪️
  </p>
