  <p class="graf graf--p">
   当年中国当局起诉浦志强律师时据称打印了他多年来发表的大量微博。这一细节曾经震惊了很多人。
  </p>
  <p class="graf graf--p">
   如今人们已经熟悉社交媒体情报的概念，应该很清楚任何留在网上的东西都有可能转变成瞄准你的黑枪。
  </p>
  <p class="graf graf--p">
   早晚（不是早晚，有可能是很快）被起诉的人可能会被打印出一大堆推文。
  </p>
  <p class="graf graf--p">
   这就是为什么应该提早预防。
  </p>
  <p class="graf graf--p">
   如果你希望清理一下那些旧的东西、没有那么高的保留价值的东西，这里有一个工具可以做到：
   <a class="markup--anchor markup--p-anchor" data-href="https://github.com/micahflee/semiphemeral" href="https://github.com/micahflee/semiphemeral" rel="noopener noreferrer" target="_blank">
    semiphemeral
   </a>
   。
  </p>
  <p class="graf graf--p">
   并不是仅仅因为没价值才清理，⚠️因为这些旧的东西有可能暴露你很多私密信息。
  </p>
  <p class="graf graf--p">
   毕竟我们所有人都是在依赖和信任监视资本主义很多年后才醒过来的，邪恶的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%BB%80%E4%B9%88%E6%98%AF%E7%A4%BE%E4%BA%A4%E5%AA%92%E4%BD%93%E6%83%85%E6%8A%A5%EF%BC%9F%E4%BD%A0%E7%9A%84%E6%8E%A8%E7%89%B9%E5%A6%82%E4%BD%95%E6%88%90%E4%B8%BA%E6%8A%93%E6%8D%95%E4%BD%A0%E7%9A%84/" href="https://www.iyouport.org/%E4%BB%80%E4%B9%88%E6%98%AF%E7%A4%BE%E4%BA%A4%E5%AA%92%E4%BD%93%E6%83%85%E6%8A%A5%EF%BC%9F%E4%BD%A0%E7%9A%84%E6%8E%A8%E7%89%B9%E5%A6%82%E4%BD%95%E6%88%90%E4%B8%BA%E6%8A%93%E6%8D%95%E4%BD%A0%E7%9A%84/" rel="noopener noreferrer" target="_blank">
    社交媒体情报
   </a>
   也是在大众真正依赖科技巨头之后才诞生并流行的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在人们醒悟过来之前肯定留下了很多可被追踪的重要信息。
   </strong>
  </p>
  <p class="graf graf--p">
   并且，trolls 也经常使用类似OSINT的做法挖掘他们的攻击目标，针对很久以前完全不相关的信息对你发起攻势。
  </p>
  <p class="graf graf--p">
   所以您有必要清理一下旧的内容 —— 如果您已经在近期学到并开始执行
   <strong class="markup--strong markup--p-strong">
    社交媒体隐私操作基本规则
   </strong>
   了。
  </p>
  <p class="graf graf--p">
   基本规则在这里：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%BD%A0%E4%B8%8D%E5%BA%94%E8%AF%A5%E5%9C%A8%E7%A4%BE%E4%BA%A4%E5%AA%92%E4%BD%93%E4%B8%8A%E8%AF%B4%E7%9A%84%E8%AF%9D/" href="https://www.iyouport.org/%E4%BD%A0%E4%B8%8D%E5%BA%94%E8%AF%A5%E5%9C%A8%E7%A4%BE%E4%BA%A4%E5%AA%92%E4%BD%93%E4%B8%8A%E8%AF%B4%E7%9A%84%E8%AF%9D/" rel="noopener noreferrer" target="_blank">
    你不应该在社交媒体上说的话
   </a>
   》。
  </p>
  <p class="graf graf--p">
   以下方法可以作为我们曾经介绍过的
   <strong class="markup--strong markup--p-strong">
    《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a4%a7%e6%b8%85%e6%b4%97%ef%bc%9a%e5%8f%8d%e4%be%a6%e5%af%9f%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" href="https://www.iyouport.org/%e5%a4%a7%e6%b8%85%e6%b4%97%ef%bc%9a%e5%8f%8d%e4%be%a6%e5%af%9f%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     大清洗
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》基本步骤
   </strong>
   的一部分。
  </p>
  <p class="graf graf--p">
   有很多应用程序可以让你的推文“定时消失”，你可以在一切正常的情况下发推文，但是，超过一个月或更早的东西会被自动删除。
  </p>
  <p class="graf graf--p">
   你还可以在 GitHub 上找到很多可以实现这种效果的简单脚本。
  </p>
  <p class="graf graf--p">
   但是，这些工具不一定适合所有人。
  </p>
  <p class="graf graf--p">
   因为它们会导致超过时间的内容全部删除；而您的旧推文中肯定有一些值得保留的东西，尤其是您引以为傲的东西。比如，如果您是一位政治经济学家，在2006年预测到未来会有一位亿万富翁当选总统，当特朗普在2016年走入白宫时，你的预测就很值得拿出来了。
  </p>
  <p class="graf graf--p">
   类似情况还有很多；您也许想要保存一些东西。
  </p>
  <p class="graf graf--p">
   所以您应该有一个标准，设定一个特定阈值，超过者删除。
  </p>
  <p class="graf graf--p">
   如果您记忆中比较精彩的推文是以 thread 形式呈现的，那么您的筛查标准应该加上它。当然，您肯定知道，标签也是个可靠的方法（如果您长期喜欢使用标签的话）。
  </p>
  <p class="graf graf--p">
   这就是 Semiphemeral。源代码：
   <a class="markup--anchor markup--p-anchor" data-href="https://github.com/micahflee/semiphemeral" href="https://github.com/micahflee/semiphemeral" rel="nofollow noopener noreferrer" target="_blank">
    https://github.com/micahflee/semiphemeral
   </a>
  </p>
  <p class="graf graf--p">
   它个是命令行工具，而不是服务。
  </p>
  <p class="graf graf--p">
   要使用它，您需要使用您的 Twitter 帐户生成 Twitter API 凭证 —
   <strong class="markup--strong markup--p-strong">
    这意味着您将直接与 Twitter API 连接，而不是授予某些第三方访问您的 Twitter 帐户的权限。
   </strong>
  </p>
  <p class="graf graf--p">
   它可以很好地在你的笔记本电脑上运行，但如果你想让它自动删除旧的推文，可能希望安排它在某个服务器上的日常 cron 作业上运行。
  </p>
  <p class="graf graf--p">
   你可以用pip安装：
  </p>
  <pre class="graf graf--pre"><code>$ pip3 install semiphemeral</code></pre>
  <p class="graf graf--p">
   这里是如何使用：
  </p>
  <pre class="graf graf--pre"><code class="markup--code markup--pre-code">$ semiphemeral 
Usage: semiphemeral [OPTIONS] COMMAND [ARGS]...</code></pre>
  <pre class="graf graf--pre"><code>  Automatically delete your old tweets, except for the ones you want to keep</code></pre>
  <pre class="graf graf--pre"><code class="markup--code markup--pre-code">Options:
  --help  Show this message and exit.</code></pre>
  <pre class="graf graf--pre"><code class="markup--code markup--pre-code">Commands:
  configure  Start the web server to configure semiphemeral
  delete     Delete tweets that aren't automatically or manually excluded
  fetch      Download all tweets
  stats      Show stats about tweets in the database</code></pre>
  <p class="graf graf--p">
   运行 semiphemeral configure 时，将启动本地Web应用程序（由 flask 提供支持），您可以在Web浏览器中加载该应用程序。
  </p>
  <p class="graf graf--p">
   在这里，您可以使用 Twitter API 凭据以及您想要使用的设置来配置它 —— 当推文被发布后多长时间自动删除、类似阈值应该是什么等等。
  </p>
  <p class="graf graf--p">
   以下是设置页面的外观：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5893 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/设置页面.png?resize=1100%2C1063&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/设置页面.png?w=1486&amp;ssl=1 1486w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/设置页面.png?resize=300%2C290&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/设置页面.png?resize=768%2C742&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/设置页面.png?resize=1024%2C990&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/设置页面.png?resize=1100%2C1063&amp;ssl=1 1100w" data-recalc-dims="1" height="1063" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/设置页面.png?resize=1100%2C1063&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  </p>
  <p class="graf graf--p">
   此Web应用程序还有一个界面，可让您查看所有删除的推文，过滤推文文本快速查找特定词组，并手动排除选中的任何推文避免被删除。
  </p>
  <p class="graf graf--p">
   配置您的 API 凭证后，您必须通过运行 semiphemeral fetch 从 Twitter 下载您的整个时间轴。根据您的推文数量多少，所需要的时间会不同。
  </p>
  <p class="graf graf--p">
   Semiphemeral 会在达到 Twitter API 的速率限制时自动暂停，有时一次最多15分钟。
  </p>
  <p class="graf graf--p">
   它可以在 sqlite 数据库中保存您的推文的整个历史记录，包括评论。数据库代码由
   <a class="markup--anchor markup--p-anchor" data-href="https://www.sqlalchemy.org/" href="https://www.sqlalchemy.org/" rel="noopener noreferrer" target="_blank">
    SQLAlchemy
   </a>
   提供支持。
  </p>
  <p class="graf graf--p">
   注：从数据结构的角度来看，threads 实际上是树；如果是单独的推文就是树的根。如果我回复某人但他们的帐户已经被冻结或者他们删除了我回复的推文，我的推文就是孤立的，并且没有简单的方法可以确定其帖子的根实际是什么。
  </p>
  <p class="graf graf--p">
   最后，在您确定已经配置了所需的设置并筛选了所有旧推文、手动移除了特定推文之后，您可能需要下载 Twitter 存档以便安全保存。
  </p>
  <p class="graf graf--p">
   然后，运行 semiphemeral delete 删除所有旧推文，但要保留的推文除外。
  </p>
  <p class="graf graf--p">
   第一次运行也可能需要比较长时间。您可以把 semiphemeral delete 放在日常 cron 作业中，以继续删除旧的推文。
  </p>
  <p class="graf graf--p">
   设置存储在〜/ .semiphemeral / settings.json 中。所有推文（包括例外和已删除的推文）都存储在 sqlite 数据库〜/ .semiphemeral / tweets.db 中。
  </p>
  <p class="graf graf--p">
   好啦，现在干净了。
  </p>
  <p class="graf graf--p">
   ⚠️最后提一句：如果您是活动家，使用 Twitter 是为了动员和宣传您的运动理念，非常建议您设置真正独立的且匿名的账户，与可能追查到您的身份的主账户完全隔离《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%88%86%e8%a3%82%e4%ba%ba%e6%a0%bc%e7%9a%84%e9%ad%85%e5%8a%9b/" href="https://www.iyouport.org/%e5%88%86%e8%a3%82%e4%ba%ba%e6%a0%bc%e7%9a%84%e9%ad%85%e5%8a%9b/" rel="noopener noreferrer" target="_blank">
    在这里看到操作方法
   </a>
   》；在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%BD%BF%E7%94%A8%E7%A4%BE%E4%BA%A4%E5%AA%92%E4%BD%93%E6%97%B6%E4%BD%A0%E5%BF%85%E9%A1%BB%E6%B3%A8%E6%84%8F%E4%BB%80%E4%B9%88%EF%BC%9F%e2%80%8a/" href="https://www.iyouport.org/%E4%BD%BF%E7%94%A8%E7%A4%BE%E4%BA%A4%E5%AA%92%E4%BD%93%E6%97%B6%E4%BD%A0%E5%BF%85%E9%A1%BB%E6%B3%A8%E6%84%8F%E4%BB%80%E4%B9%88%EF%BC%9F%e2%80%8a/" rel="noopener noreferrer" target="_blank">
    运行这一匿名账户的方法
   </a>
   》。而不是简单的删除就推文就可以了。
  </p>
  <p class="graf graf--p">
   日常请注意保持您的隐私操作习惯；随时记得*哪些话不能在互联网上说*。玩得开心。⚪️
  </p>
