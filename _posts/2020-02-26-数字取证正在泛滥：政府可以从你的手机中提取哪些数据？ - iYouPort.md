    <span style="color: #00ccff;">
     <em>
      <strong>
       如今的人们都知道数字取证是什么东西了 —— 因为已经有太多人因此被抓进监狱、被判刑。但是，数字证据真的靠谱吗？任何人可以信任它作为剥夺一个人后半生的绝对依据吗？本文将展示它有什么问题，并警告自我保护
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-9801 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/头-6.png?resize=1100%2C783&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/头-6.png?w=1256&amp;ssl=1 1256w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/头-6.png?resize=300%2C214&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/头-6.png?resize=768%2C547&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/头-6.png?resize=1024%2C729&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/头-6.png?resize=1100%2C783&amp;ssl=1 1100w" data-recalc-dims="1" height="783" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/头-6.png?resize=1100%2C783&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9801" data-recalc-dims="1" height="783" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/头-6.png?resize=1100%2C783&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/头-6.png?w=1256&amp;ssl=1 1256w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/头-6.png?resize=300%2C214&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/头-6.png?resize=768%2C547&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/头-6.png?resize=1024%2C729&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/头-6.png?resize=1100%2C783&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   正义的实现取决于社会对证据准确性和完整性的信任。
  </p>
  <p class="graf graf--p">
   但是，数字取证领域很难做到这点。
  </p>
  <p class="graf graf--p">
   本文以英国为例 —— 仅仅是为例，⚠️数字取证正在全球泛滥，它们在各个国家所表现出的问题是几乎完全一样的。
  </p>
  <p class="graf graf--p">
   英国法医科学监管机构 Gillian Tulley 将英格兰和威尔士的法医学交付描述为完全不充分的、且已经“在危机之间陷入困境”。
  </p>
  <p class="graf graf--p">
   他们这么说是有道理的。该报告发出了与数字证据有关的严厉警告，并且，实话说，
   <strong class="markup--strong markup--p-strong">
    目前缺乏应对“数字法证证据迅速增长”的“任何可行战略”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   并且，使用手机提取 [以获取数字证据] 可以说是警察使用的最具侵入性的技术形式 —— ⚠️不仅涉及手机用户本人的数据，还涉及曾经与该用户通信的许多其他人的数据。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9802 jetpack-lazy-image" data-lazy-sizes="(max-width: 916px) 100vw, 916px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-4.jpeg?resize=916%2C2590&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-4.jpeg?w=916&amp;ssl=1 916w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-4.jpeg?resize=106%2C300&amp;ssl=1 106w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-4.jpeg?resize=768%2C2172&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-4.jpeg?resize=362%2C1024&amp;ssl=1 362w" data-recalc-dims="1" height="2590" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-4.jpeg?resize=916%2C2590&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="916"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9802" data-recalc-dims="1" height="2590" sizes="(max-width: 916px) 100vw, 916px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-4.jpeg?resize=916%2C2590&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-4.jpeg?w=916&amp;ssl=1 916w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-4.jpeg?resize=106%2C300&amp;ssl=1 106w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-4.jpeg?resize=768%2C2172&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-4.jpeg?resize=362%2C1024&amp;ssl=1 362w" width="916"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   隐私国际组织（Privacy International）2018年的报告
   <a class="markup--anchor markup--p-anchor" data-href="https://privacyinternational.org/report/1699/digital-stop-and-search-how-uk-police-can-secretly-download-everything-your-mobile" href="https://privacyinternational.org/report/1699/digital-stop-and-search-how-uk-police-can-secretly-download-everything-your-mobile" rel="noopener noreferrer" target="_blank">
    Digital Stop and Search
   </a>
   中指出，⚠️至少从2012年开始，一线警官就一直在使用手机提取工具来获取所有类型的有关犯罪受害者、证人、和嫌疑人的个人数据。
  </p>
  <p class="graf graf--p">
   尽管已经有多年的经验，警方可以熟练使用强大的技术来提取、分析和呈现手机中包含的所有形式的数据，但是，这不仅会严重侵犯个人隐私，而且还包含
   <strong class="markup--strong markup--p-strong">
    诸多的安全隐患
   </strong>
   。
  </p>
  <p class="graf graf--p">
   更有，⚠️这种提取没有任何有意义的规范、缺乏明确其实际具有合法性的依据、迄今未向公众做任何咨询 —— 违背民主程序。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     信任？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   如果一个民主国家的公民想要“同意“将自己大量的高度私密的数据委托给警察官员，那么，
   <strong class="markup--strong markup--p-strong">
    不仅提取的合法性应该清晰易懂，而且，还应该确认数据保护措施具有良好的安全性，并且，还要看到具备清晰可见的专门针对移动电话和数据萃取的详细安全策略和程序
   </strong>
   。
  </p>
  <p class="graf graf--p">
   作为安全风险分析的一部分，
   <strong class="markup--strong markup--p-strong">
    公民还需要知道当局在数字提取中使用的是哪些技术，并且需要对这些技术有所了解
   </strong>
   。
  </p>
  <p class="graf graf--p">
   然而，尽管每年对信息安全的威胁都在加剧，而且，显而易见的是，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.ncsc.gov.uk/collection/protecting-bulk-personal-data" href="https://www.ncsc.gov.uk/collection/protecting-bulk-personal-data" rel="noopener noreferrer" target="_blank">
    国家网络安全中心
   </a>
   一直在强调： “大量数据存储是各种攻击者的诱人目标”，向委员会提供的证据和调查结果表明，关于提取技术的部署以及已有的安全控制措施，存在大量问题。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9803 jetpack-lazy-image" data-lazy-sizes="(max-width: 862px) 100vw, 862px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-11.png?resize=862%2C892&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-11.png?w=862&amp;ssl=1 862w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-11.png?resize=290%2C300&amp;ssl=1 290w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-11.png?resize=768%2C795&amp;ssl=1 768w" data-recalc-dims="1" height="892" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-11.png?resize=862%2C892&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="862"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9803" data-recalc-dims="1" height="892" sizes="(max-width: 862px) 100vw, 862px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-11.png?resize=862%2C892&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-11.png?w=862&amp;ssl=1 862w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-11.png?resize=290%2C300&amp;ssl=1 290w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-11.png?resize=768%2C795&amp;ssl=1 768w" width="862"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   数据泄露是由于实施不力或完全没有安全控制而发生的。
  </p>
  <p class="graf graf--p">
   在英国，警察部门对于因自己的过错而
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/uk-news/2017/may/04/greater-manchester-police-fined-victim-interviews-lost-in-post" href="https://www.theguardian.com/uk-news/2017/may/04/greater-manchester-police-fined-victim-interviews-lost-in-post" rel="noopener noreferrer" target="_blank">
    令人震惊的数据泄露事件
   </a>
   并不陌生，他们并已收到了信息专员的多项罚款。
  </p>
  <p class="graf graf--p">
   格洛斯特郡警方因泄露了虐待儿童受害者的身份而在2018年
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bbc.co.uk/news/uk-england-gloucestershire-44486535" href="https://www.bbc.co.uk/news/uk-england-gloucestershire-44486535" rel="noopener noreferrer" target="_blank">
    被罚款80,000英镑
   </a>
   。至于安全控制，北约克郡警察因
   <a class="markup--anchor markup--p-anchor" data-href="https://www.northyorkshire-pfcc.gov.uk/content/uploads/2016/10/7ae-Mobile-Phone-Examination.pdf" href="https://www.northyorkshire-pfcc.gov.uk/content/uploads/2016/10/7ae-Mobile-Phone-Examination.pdf" rel="noopener noreferrer" target="_blank">
    未能做到数据加密
   </a>
   而受到严厉批评。
  </p>
  <p class="graf graf--p">
   简单说，民主国家的警察也不能直接信任；民主程序在数字取证领域目前最多只能做到事后批评。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9804 jetpack-lazy-image" data-lazy-sizes="(max-width: 918px) 100vw, 918px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-10.png?resize=918%2C882&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-10.png?w=918&amp;ssl=1 918w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-10.png?resize=300%2C288&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-10.png?resize=768%2C738&amp;ssl=1 768w" data-recalc-dims="1" height="882" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-10.png?resize=918%2C882&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="918"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9804" data-recalc-dims="1" height="882" sizes="(max-width: 918px) 100vw, 918px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-10.png?resize=918%2C882&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-10.png?w=918&amp;ssl=1 918w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-10.png?resize=300%2C288&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-10.png?resize=768%2C738&amp;ssl=1 768w" width="918"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   尽管数据的数量和保留是基本的，但仍然存在更广泛的安全问题。正如 Bruce Schneier 所述：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “安全是复杂且具有挑战性的。简单的答案无济于事，因为简单的答案总是错误的。您需要的是一种新的思维方式。”
   </strong>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     证据的准确性：警察知道自己在做什么吗？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   使用手机提取技术的警务人员通常很少接受专业的取证培训，或者
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/law/2018/may/15/police-mishandling-digital-evidence-forensic-experts-warn" href="https://www.theguardian.com/law/2018/may/15/police-mishandling-digital-evidence-forensic-experts-warn" rel="noopener noreferrer" target="_blank">
    根本没有任何培训
   </a>
   ；并且，他们的工作越来越依赖于他们完全不了解其功能的设备，尤其是在预算削减和数据量迅速增加的情况下。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9805 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-10.png?resize=1100%2C791&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-10.png?w=1226&amp;ssl=1 1226w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-10.png?resize=300%2C216&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-10.png?resize=768%2C553&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-10.png?resize=1024%2C737&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-10.png?resize=1100%2C791&amp;ssl=1 1100w" data-recalc-dims="1" height="791" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-10.png?resize=1100%2C791&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9805" data-recalc-dims="1" height="791" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-10.png?resize=1100%2C791&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-10.png?w=1226&amp;ssl=1 1226w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-10.png?resize=300%2C216&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-10.png?resize=768%2C553&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-10.png?resize=1024%2C737&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-10.png?resize=1100%2C791&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   取证专家强调，警方严重依赖于那些易于使用的工具 —— 这些工具通过“按一些按钮” 就能从电话中获取数据，
   <strong class="markup--strong markup--p-strong">
    这纯粹是愚弄数字法证学，并且可能破坏真正有意义的检查。
   </strong>
  </p>
  <p class="graf graf--p">
   Discovery Forensics 的常务董事兼高级取证调查员 Jan Collie 博士
   <a class="markup--anchor markup--p-anchor" data-href="http://data.parliament.uk/writtenevidence/committeeevidence.svc/evidencedocument/science-and-technology-committee-lords/forensic-science/oral/93059.html" href="http://data.parliament.uk/writtenevidence/committeeevidence.svc/evidencedocument/science-and-technology-committee-lords/forensic-science/oral/93059.html" rel="noopener noreferrer" target="_blank">
    向委员会强调了
   </a>
   使用 MPE 技术的人员缺乏取证技能的情况：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “我在现场看到的是，常规警务人员正试图成为数字取证分析员，因为他们被赋予了这些无所不包的魔术工具，可以执行所有操作，而常规警务人员，尽管他可能会尽职，但是他们不是数字取证分析师 ……
   </em>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “正是这些毫无经验的人最终对数据意味着什么做出结论，因为他们被迫这样做；
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     他们没有资源也没有培训，他们无法从这些结果中做出正确的推断。要知道这些结论会被摆在法庭面前，作为决定一个人生死存亡的关键
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    。”
   </em>
  </p>
  <p class="graf graf--p">
   严重欺诈办公室（Serious Fraud Office，SFO）向委员会争辩说，随着数字证据在刑事审判中变得越来越普遍，有必要制定更多的法规或遵守商定的标准和规范的机制。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     证据的完整性：技术可以做什么？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   约克大学的数字取证科学家安格斯·马歇尔（Angus Marshall）和计算机科学家理查德·佩奇（Richard Page）都指出，“我们倾向于依赖第三方工具，我们相信这些工具是根据良好的工程实践生产的；如果考虑使用手机提取，则检查将从内容未知的手机开始，因此，整个取证过程的输入是未知的。
  </p>
  <p class="graf graf--p">
   这影响到是否有可能利用所谓的证据来确定哪里出了问题”。
  </p>
  <p class="graf graf--p">
   以下是他们的声明：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <strong class="markup--strong markup--p-strong">
    一个工具完全有可能错误地处理输入并产生一些看起来仍然与正确操作一致的东西
   </strong>
   。在没有客观验证证据的情况下，对工具是否能产生任何结果的正确性方面或其他方面的评估，完全取决于操作员的经验。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “还应该记住，
   <strong class="markup--strong markup--p-strong">
    就用户而言，硬件和软件的更新可能不会对系统行为产生明显影响，但是，可能会大大改变内部处理和存储数据的方式
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   在美国，美国国家标准技术研究院有一个网络取证项目，旨在向公众提供取证工具的测试报告。
  </p>
  <p class="graf graf--p">
   根据 MobileEdit，他们的测试表明，在所有竞争工具中，单个数据类型之间的结果存在显著差异。
  </p>
  <p class="graf graf--p">
   每个工具都能够展示出相对于其他工具的某些优势，但是，并没有任何一个工具能够在所有测试类别中都表现出优越性。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9806 jetpack-lazy-image" data-lazy-sizes="(max-width: 887px) 100vw, 887px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-9.png?resize=887%2C1036&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-9.png?w=887&amp;ssl=1 887w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-9.png?resize=257%2C300&amp;ssl=1 257w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-9.png?resize=768%2C897&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-9.png?resize=877%2C1024&amp;ssl=1 877w" data-recalc-dims="1" height="1036" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-9.png?resize=887%2C1036&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="887"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9806" data-recalc-dims="1" height="1036" sizes="(max-width: 887px) 100vw, 887px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-9.png?resize=887%2C1036&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-9.png?w=887&amp;ssl=1 887w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-9.png?resize=257%2C300&amp;ssl=1 257w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-9.png?resize=768%2C897&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-9.png?resize=877%2C1024&amp;ssl=1 877w" width="887"/>
   </noscript>

  <p class="graf graf--p">
   在英国，没有对执法机构购买的设备进行任何测试。正如取证监管者 Gillian Tully 博士所说：“
   <strong class="markup--strong markup--p-strong">
    到目前为止，所有那些[手机提取工具]尚未全部经过正确的测试……”；“而且不允许对数字取证工具进行逆向工程，因为无论如何这都是非法的
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   为什么会缺乏测试？为什么逆向工程非法？很简单，主要理由就是所谓的商业机密。
  </p>
  <p class="graf graf--p">
   正如安格斯·马歇尔（Angus Marshall）向英国专责委员会发表的评论所指出的那样：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “私有技术提供商不愿透露有关他们自己的开发和测试方法的任何信息，这意味着许多数字取证方法的正确性基础非常薄弱，或根本不存在。”
   </strong>
  </p>
  <p class="graf graf--p">
   一家提取公司 Magnet Forensics 直接举例说明了由于缺乏对手机提取工具功能的了解而导致的问题，以至于它们可以做非法的事，但警察没有意识到这一点。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     风险处理缺失：远程访问和网络安全
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   没有被触及的领域之一是：⚠️提取技术公司能够远程访问警务人员购买的产品。
  </p>
  <p class="graf graf--p">
   此访问权限不仅用于产品更新，而且还用于分析。
  </p>
  <p class="graf graf--p">
   Cellebrite 是一家数字取证公司，其 UFED 设备在包括英国在内的
   <strong class="markup--strong markup--p-strong">
    世界各地的警察部队中都很受欢迎
   </strong>
   ，该公司在其隐私权政策中指出：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “当您使用 UFED Cloud Analyzer 之类的产品时，如果得到您明确的同意，我们将收集有关 Cellebrite 产品的使用信息，例如您使用该软件执行的提取次数、提取的数据源类型、使用产品时遇到的任何错误、收集的事件数量、使用Web搜寻器的频率、以及使用的视图 …… 以上所有数据均称为分析型信息。”
  </p>
  <p class="graf graf--p">
   ⚠️Cellebrite 和其他同类公司都可以远程访问位于警察局内部的设备，从而访问与受害者、证人和嫌疑人的电话相连的工具。而且这个领域是一个黑箱。
  </p>
  <p class="graf graf--p">
   此外，
   <strong class="markup--strong markup--p-strong">
    由于成千上万部电话被直接插入到这些取证工具中，因此，该工具本身很容易受到网络攻击
   </strong>
   —— 例如，它们可能被劫持以发送恶意更新，或者可以提供故意感染了恶意软件的电话进行提取，从而有可能实现入侵警察部门的基础设施。
  </p>
  <p class="graf graf--p">
   对于这种攻击形式，可能不是显而易见的或最简单的方法，但是，坚定的对手可以尝试所有必要的手段。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     执法部门可以从手机中提取哪些类型的数据？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   移动设备是一个庞大的敏感数据存储库，可以提供有关其所有者以及用户与之交互的许多其他人的大量信息。
  </p>
  <p class="graf graf--p">
   Cellebrite、MSAB 和 Oxygen Forensics 等公司向执法部门出售软件和硬件。将手机连接到这些数字提取工具之一后，设备将提取、分析并显示手机中包含的数据。
  </p>
  <p class="graf graf--p">
   这些工具可以提取哪些数据、以及使用哪种方法，将取决于操作系统、安全功能、和电话型号。
  </p>

   隐私国际组织使用 Cellebrite 的 UFED 通过 Physical [ADB Rooted] 提取功能提取了 Android 手机（HTC Desire）以做测试。

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     下一步会走向哪里？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   随着所有人越来越多的在线生活（手机变得越来越“万能”），个人数据已经迅速变得更有价值。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这些具有显著价值的数据正是警察想要收集、访问和开采的东西，并且也是犯罪分子想要窃取数据的原因
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️搜查电话要比搜查房屋更具侵入性，不仅是信息的数量和细节，还有，历史性内容的回溯、以及所有与您相关的所有亲密关系 —— 你甚至无法放在家里的东西全在手机里。
  </p>
  <p class="graf graf--p">
   任何当权者都不应无拘无束地获得某人的全部生活，使用手机提取功能需要最严格的保护。
  </p>
  <p class="graf graf--p">
   但是，即使合法获得提取权限，也没有任何数据可以做到完全安全。
  </p>
  <p class="graf graf--p">
   ⚠️一旦有人存储了数据，由于事故、粗心大意、内部威胁、或敌对的入侵 ……数据就很容易受到破坏。
  </p>
  <p class="graf graf--p">
   处理数据的不良做法，包括对获取数据方式的不了解，都会破坏对严重犯罪的起诉，并导致曝光包含从未被起诉的无辜者的详细信息。
  </p>
  <p class="graf graf--p">
   在手机提取的背景下，迫切需要严格的监管以及详细实用的技术标准来确保对这些强大技术部署的公民监督。
  </p>
  <p class="graf graf--p">
   在非民主国家，人们应该学会保护自己；开发者应该致力于那些能够保护无辜者免遭数字提取工具侵害的功能。
  </p>
