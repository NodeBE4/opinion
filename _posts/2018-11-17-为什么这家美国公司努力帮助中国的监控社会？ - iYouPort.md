  <p class="graf graf--p graf-after--figure" id="9560">
   在距离拉斯维加斯大道一英里的一个不起眼的办公大楼里，两个女人在一个没有窗户的房间里劳作。这里每个房间都有各种尺寸、款式和色调的比基尼。这是一家在线泳装店 Bikini.com的库房。而在大厅里，同事们正在努力打入中国炙手可热的人工智能监控技术市场。
  </p>
  <p class="graf graf--p graf-after--p" id="4c11">
   Bikini.com由 Remark Holdings 拥有，
   <strong class="markup--strong markup--p-strong">
    Remark Holdings 是一家小型上市公司，由好莱坞制片人 Brett Ratner 担任董事会成员
   </strong>
   ，并与 Dr. Mehmet Oz 建立了金融联系。Remark 的领导者正试图将无利可图、负债累累的网站运营商转变为亚洲（尤其是中国）的企业 AI 技术提供商。Remark 的业务和股价正在挣扎，但其独特的人工智能项目却取得了一些进展。
  </p>
  <p class="graf graf--p graf-after--p" id="2168">
   今年3月，Remark 表示已经与一家泰国企业集团达成了协议，该企业在泰国经营着超过 10,000 家 7-Eleven 连锁店，以及许多其他公司。根据为 CP 集团董事长 Soopakij Chearavanont 提供建议的执行官 Athikom Asvanund 表示，Charoen Pokphand 集团希望能够识别客户在忠诚度计划以及防止商店扒手方面的表现。Chearavanont 在一份声明中表示，
   <strong class="markup--strong markup--p-strong">
    Remark 的技术还可以帮助中国保险巨头平安保险，其任务包括处理索赔
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="b6f0">
   <strong class="markup--strong markup--p-strong">
    中国是 Remark 人工智能项目的重点
   </strong>
   。该公司在中国拥有多家子公司，中国的工程师便宜，而且每周工作六天。在中国第五大城市杭州，
   <strong class="markup--strong markup--p-strong">
    他们的一家子公司建造了供警察使用的技术，该技术分析监控视频
   </strong>
   ，以识别在被禁止的街道上行驶的摩托车。
  </p>
  <p class="graf graf--p graf-after--p" id="bc34">
   Remark 还帮助中国超市集团
   <strong class="markup--strong markup--p-strong">
    通过面部识别软件识别常客
   </strong>
   。
   <strong class="markup--strong markup--p-strong">
    Remark 首席执行官 Shing Tao，一位纽约本地人
   </strong>
   表示，他的公司正在利用这项技术为工人提供时钟，用于采取两班倒的中国建筑工地，并且他还正在寻求将该技术的使用出售给中国警察，帮他们“追踪逃犯”。
  </p>
  <p class="graf graf--p graf-after--p" id="e5fd">
   <strong class="markup--strong markup--p-strong">
    Remark 正试图利用中国基于维稳和商业目的的强烈监视需求
   </strong>
   。去年发布的一项国家战略宣布，该国将于 2020 年在人工智能方面超过美国。中国快速采用面部识别技术意味着你的脸现在可以从 ATM 提取现金、或者被当地警察在上万人的音乐会中精确瞄准。Tao 表示，
   <strong class="markup--strong markup--p-strong">
    中国对技术的开放性和对隐私的宽松态度使其成为 Remark 的合理目标
   </strong>
   ：“中国是低垂的果实”，他说，“中国人真的理解并接受人工智能，他们已经准备好了。”
  </p>
  <p class="graf graf--p graf-after--p" id="5e50">
   与处于人工智能研究前沿的美国大型科技公司相比，Remark 可以说微不足道; 它的 AI 推动力量很小。Remark 去年报告的收入为 7000 万美元，主要来自其旅游业务; 这比谷歌平均一天的收入还少。Remark 的首席技术官 Jason Wei 表示，他在中国的开发人员非常依赖 Google 和学术界发布的开源工具，并且
   <strong class="markup--strong markup--p-strong">
    通过从中国和美国的社交媒体网站（包括 Facebook）中删除的照片来磨练他们的面部识别软件
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="c708">
   这并不是世界级人工智能技术的完美配方。Remark 的人工智能产品并没有收到 Tao 告诉投资者的收入数字，而且该公司的股价今年下跌了 70％。美国的制裁和中国蓬勃发展的国内人工智能提供商生态系统也给 Remark 在该国建立业务的努力带来了挑战。当被问及 Remark 时，领先的面部识别创业公司 Yitu Technology 的首席创新官称从未听说过该公司或其中国品牌。
  </p>
  <ul class="postList">
   <li class="graf graf--li graf-after--p" id="69cc">
    延伸阅读：关于中国的监视技术公司 Yitu《

  <p class="graf graf--p graf-after--li" id="ff3a">
   尽管如此，小型网站运营商正在构建人工智能产品的概念表明，在开源软件的帮助下，图像识别等领域的进步有多快。
  </p>
  <p class="graf graf--p graf-after--p" id="a5e8">
   四年前，对于一家没有技术创新的小公司来说，创造一个能够识别摩托车等物体的工具是不可信的。Palo Alto 视频识别创业公司 Matroid 的创始人 Reza Zadeh 说，现在，即便是没有机器学习经验的开发人员也可以在四个月内拼凑出一个能够相当好地检测出一些常见物体的系统。这样的软件对于客户来说并不容易使用，也难以与 Matroid 等更复杂的产品竞争，但它可能足以赢得交易。“这就是你能看到的缺乏经验的初创公司试图兜售的东西，” Zadeh 说，
  </p>
  <p class="graf graf--p graf-after--p" id="3948">
   像 Remark 这样的小型企业是可能会更自由地采用人工智能技术的地方。
   <strong class="markup--strong markup--p-strong">
    谷歌和微软都在中国设有人工智能研究实验室
   </strong>
   ，但他们由于与美国情报机构的密切联系而无法参与竞标给中国警察部门的面部识别技术合同。当被问及 Tao 是否对与中国当局的合作感觉不妥时，他说，他只是想帮助客户“更有效地运营”，无论他们是零售商还是当地的执法官员。“我们没有违反规定”，他补充道。
  </p>
  <p class="graf graf--p graf-after--p" id="954d">
   <strong class="markup--strong markup--p-strong">
    事情怎么样了？
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="f93d">
   Remark 并非始于科技公司。它创建于 2006 年，作为 Discovery Communications 的一个名为 How Stuff Works International（或 HSWI）的衍生产品，用于推广新兴市场流行的解释网站版本。当这项业务被证明不如预期那般有利可图时，HSWI 于 2009 年共同创办了一个名为 ShareCare 的在线健康网站，包括 WebMD 创始人，Oz 博士和 Oprah Winfrey 的制作公司在内的合作伙伴。Remark 仍然拥有5％。2011 年，HSWI 更名为 Remark Media，并开始进行更广泛的扩张，收购
   <a class="markup--anchor markup--p-anchor" data-href="http://bikini.com/" href="http://bikini.com/" rel="noreferrer noopener nofollow" target="_blank">
    Bikini.com
   </a>
   以及后来提供酒店和演出预订的
   <a class="markup--anchor markup--p-anchor" data-href="http://vegas.com/" href="http://vegas.com/" rel="noreferrer noopener nofollow" target="_blank">
    Vegas.com
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="a89e">
   这些收购增加了 Remark 的收入，但 Tao 担心媒体和电子商务受到成长型科技公司的威胁。在 Remark 董事会工作了一段时间后，他于 2012 年被任命为首席执行官。此前，他曾在台湾科技公司的交易中担任投资者，并担任花花公子企业董事会成员。“很明显，我们需要拥有某种类型的技术，某种类型的优势，需要没有人可以与我们竞争，” Tao 先生回忆说。
  </p>
  <p class="graf graf--p graf-after--p" id="fb0c">
   Tao 转向现任首席技术官的 Wei，以评估具有最小技术专长的公司可能开发出何种技术。Wei 先生被引进来改善 Bikini.com的出版系统，但此前他曾在东芝工作，并创立了一家虚拟世界创业公司。
  </p>
  <p class="graf graf--p graf-after--p" id="2237">
   Wei 为 Tao 的狗制作了一款可穿戴式摄像机、以及一种防止开车发短信的装置。两者似乎都没有什么前景，工程成本也在增加。Wei 估计他的探险活动在两个月内就烧掉了超过10万美元。“我们决定，好吧，也许我们应该尝试去中国，因为在中国的成都，这笔钱可能会维持一年，”他说。
  </p>
  <p class="graf graf--p graf-after--p" id="8b38">
   在成都，Wei 组织了软件工程师队伍，为营销人员建立了社交媒体分析服务。
   <strong class="markup--strong markup--p-strong">
    该软件避开了通过社交网络实施抓取的障碍，并收集了来自新浪微博和 Facebook 等社交媒体网络的公共更新和照片。目标是识别属于同一个人的帐户，并按内容主题和地理位置对其进行索引
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="f954">
   该项目使得 Remark 在硅谷启动了 AI 技术。Wei 研究了一种被称为深度学习的技术能够使计算机更好地分类图像。他借用谷歌等公司和领先学者的想法和开源代码。他描述了牛津大学教授 Andrew Zisserman，他也与 Alphabet 合作，作为他的团队的“英雄”，发布可以识别照片中物体和面孔的即用型深度学习模型。
  </p>
  <p class="graf graf--p graf-after--p" id="a7ba">
   Remark 以 KanKan 品牌推出其社交媒体分析服务，能以普通话观看和探索。据 Tao 说，其客户包括阿斯顿马丁在中国的广告代理商。他还将公司名称改为 Remark Holdings，并开始向亚洲的公司和公共机构推销其图像处理软件。“我们建立了强大的数据基础，在此期间，人工智能的成本一直在下降，” Tao 说。
  </p>
  <p class="graf graf--p graf-after--p" id="81a8">
   <strong class="markup--strong markup--p-strong">
    跟踪面孔
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="01f2">
   在 Remark 位于拉斯维加斯的办事处，Wei 热切地将视频会议系统的馈送连接到他们团队的软件。在一种模式中，该软件
   <strong class="markup--strong markup--p-strong">
    在房间内的每个面孔周围绘制框，随着人的移动而移动
   </strong>
   ；在另一个模式中，它追踪人的肢体运动，并
   <strong class="markup--strong markup--p-strong">
    能够正确地说出 WIRED 记者戴着眼镜，年龄在30到40岁之间
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="90ca">
   Wei 说他的团队的另一部分软件
   <strong class="markup--strong markup--p-strong">
    可以确定几十种面部表情
   </strong>
   ，尽管许多面部表情的精确度很低。“微笑、哭泣、愤怒、紧张等基本问题，对于零售商店来说，想要确定他们的客户是否满意、或者员工是否正在做正确的工作，是非常有用的，”他说。
  </p>

   <li class="graf graf--li graf-after--p" id="a8ef">

  <p class="graf graf--p graf-after--li" id="0344">
   Wei 还在一家繁忙的餐厅厨房里安装了这种监视，装在六名忙碌的工人的头顶上方。监视器跟踪两名不戴口罩的工人的脸。他说，该软件的开发是为了竞标中国上海的一个试点项目 — — 与来自中国国有媒体的报告中所描述的类似 — — 当局在餐厅厨房安装摄像头，自动提醒市检查员哪里违反了“食品安全规定”。
  </p>
  <p class="graf graf--p graf-after--p" id="1b49">
   在附近的一间会议室里，Wei 展示了这个黑色金属外壳、其中包括图形芯片的来自制造商 Nvidia 的计算板，其强大的芯片已成为主要人工智能项目的核心，还有臭名昭著的海康威视的监视摄像头，海康威视是中国政府部分拥有的领先视频监控公司。
  </p>

   <li class="graf graf--li graf-after--li" id="d1c7">
    <strong class="markup--strong markup--li-strong">
     美国精英为什么要维护中国的老大哥“海康威视”的利益？《
    </strong>
    <a class="markup--anchor markup--li-anchor" data-href="https://medium.com/@iyouport/%E5%8F%AF%E6%80%9C%E6%AD%A3%E4%B9%89%E6%B2%A1%E6%9C%89%E9%92%B1-d3bec8596c6c" href="https://www.iyouport.org/%e5%8f%af%e6%80%9c%e6%ad%a3%e4%b9%89%e6%b2%a1%e6%9c%89%e9%92%b1/" rel="noopener noreferrer" target="_blank">
     <strong class="markup--strong markup--li-strong">
      可怜正义没有钱
     </strong>
    </a>
    <strong class="markup--strong markup--li-strong">
     》
    </strong>
   </li>
  </ul>
  <p class="graf graf--p graf-after--li" id="6601">
   Wei 说，Remark 将其软件安装在计算机上，然后连接到摄像机。他描述了对 Remark 面部识别技术的测试，其中它正确地发现了在拉斯维加斯大道上与一大群人混在一起的工作人员。
  </p>
  <p class="graf graf--p graf-after--p" id="1f8b">
   但是所有这项技术都没有像 Tao 所预测的那样迅速提升 Remark 的财务状况。3月份，他告诉投资者，KanKan 将在 2018 年贡献 5000 万美元的收入。根据公司提交的文件显示，今年前六个月，Remark 的技术业务收入仅为 460 万美元。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="d7da">
   “这是一个不可能的故事，”总部位于中国北京的金融研究公司 J Capital Research 的联合创始人 Anne Stevenson-Yang 说，她今年发表了关于 Remark AI 梦想的两篇尖锐的研究报告。她在报告中描述了未能确认 Remark 声称的与中资银行签订的合同。她的结论是：该公司的技术极小且极度夸大其词。“只是为了股票，这就是全部，”她补充道。J Capital 的 Remark 报告网页指出，该公司的客户可能正在抛售 Remark 的股票。
  </p>
  <p class="graf graf--p graf-after--p" id="badf">
   Tao 说，Remark 的 AI 收入没有达到他的预期，因为合同的完成时间比预期的要长，中国政府对银行的审查越来越多。他去年告诉投资者，中国上海餐馆系统的目标是 2000 家餐厅[安装监视系统]，价值“七位数”，但现在已经被市政官员推迟了。
  </p>
  <p class="graf graf--p graf-after--p" id="6960">
   Tao 向 WIRED 展示了 Remark 与超市运营商北京华联之间的谅解备忘录，
   <strong class="markup--strong markup--p-strong">
    该备忘录旨在通过面部识别技术跟踪购物者
   </strong>
   。其他合同显示，Remark 的子公司承诺
   <strong class="markup--strong markup--p-strong">
    向在上海市静安区工作的上海公司和杭州警方使用的摩托车检测系统提供监视技术
   </strong>
   。Tao 说，
   <strong class="markup--strong markup--p-strong">
    监视器涵盖十几个十字路口
   </strong>
   ，并自动标记违规车辆的图像。
  </p>
  <p class="graf graf--p graf-after--p" id="b3fc">
   他抨击 J Capital 的报道，称他的商业伙伴不应该向爱管闲事的分析师透露所有内容，称报告误导投资者的指责没有意义。“我们没有理由把我们的核心业务置于风险之中啊？” Tao 说。
  </p>
  <p class="graf graf--p graf-after--p" id="60b6">
   <strong class="markup--strong markup--p-strong">
    上周，在一年一度的乌镇“世界互联网大会”上，面部识别是一个热门话题
   </strong>
   。中国政府利用这一事件来展示其数字创新的图景 — — 在精心定义的政治语话范围内。
  </p>
  <p class="graf graf--p graf-after--p" id="87b8">
   去年，苹果首席执行官蒂姆库克和谷歌首席执行官桑达皮采都在中国乌镇做出了主题演讲。今年，在特朗普所谓的贸易战的阴影下，出席乌镇的美国高级首席执行官是高通公司的 Steve Mollenkopf。
  </p>
  <p class="graf graf--p graf-after--p" id="fa3d">
   当然，Remark 也出席了，在展览会上摆了一个摊位。周四，该公司在推特上发布了一张 Tao 的照片，显示在乌镇的舞台上展示 ​​KanKan 的 AI 平台。Tao 说：“人工智能正在改变世界，中国正在引领世界采用这项技术”。
  </p>
