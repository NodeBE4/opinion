  <p class="graf graf--p graf-after--figure" id="a2a4">
   互联网中断如今的出现频率更高了。根据国际权利组织 Access Now 发起的在线活动
   <span class="markup--quote markup--p-quote is-other" data-creator-ids="1669e893873f">
    #KeepItOn
   </span>
   的调查，2016 年就记录到了全球范围内令人印象深刻的 56 次断网，该活动将断网定义为：
   <strong class="markup--strong markup--p-strong">
    “ 故意破坏互联网或电子通信，使其无法访问、或使其在特定人群或位置内无法使用，通常是为了控制信息的流动。“
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="3388">
   有大量的
   <a class="markup--anchor markup--p-anchor" data-href="https://scholar.google.nl/scholar?start=0&amp;q=internet+shutdown&amp;hl=en&amp;as_sdt=0,5" href="https://scholar.google.nl/scholar?start=0&amp;q=internet+shutdown&amp;hl=en&amp;as_sdt=0,5" rel="noopener nofollow noreferrer" target="_blank">
    学术文献和新闻报道
   </a>
   提供了互联网封锁和个人网站审查封锁的资讯，以及它们与现实世界政治事件的联系。该领域的专业人士和活动家们已经进行了大量的研究，并
   <a class="markup--anchor markup--p-anchor" data-href="https://internetfreedomfestival.org/wiki/index.php/Analyzing_the_Causes_%26_Impacts_of_Internet_Shutdowns" href="https://internetfreedomfestival.org/wiki/index.php/Analyzing_the_Causes_%26_Impacts_of_Internet_Shutdowns" rel="noopener nofollow noreferrer" target="_blank">
    继续讨论了触发互联网关闭的因素
   </a>
   ，以及对实现其预期目的的有效性。
  </p>
  <p class="graf graf--p graf-after--p" id="c1b7">
   在世界各国政府决定关闭互联网服务的各种理由中，一个非常普遍的特征是，所谓的或暗示的旨在防止物理骚乱蔓延的意图（可能是出于政治、宗教、经济或种族原因而发生的大型群体事件）。
  </p>
  <p class="graf graf--p graf-after--p" id="9ec5">
   因此，
   <strong class="markup--strong markup--p-strong">
    产生这项研究的动机在于我们是否能够在发生之前对其识别、分析和监测潜在的全面或选择性关闭的指标
   </strong>
   。本文希望成为一个分析性实验，开始填补这一空白，并提出一些关于潜在指标的想法，以便在公开场合进行监测。
  </p>
  <p class="graf graf--p graf-after--p" id="0441">
   <strong class="markup--strong markup--p-strong">
    测试分析模型
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="5b9f">
   对一个国家的互联网流量采取严厉镇压措施的决定取决于其政府。虽然人类的不可预测性是一个需要考虑的因素，特别是对于反复无常的当权者来说，但我们仍可以尝试制定一个模型，通过一些关键因素来评估这种决策出现的可能性：
  </p>
  <p class="graf graf--p graf-after--p" id="52e9">
   理由：
  </p>
  <ul class="postList">
   <li class="graf graf--li graf-after--p" id="499f">
    在线动员成功地引发、促进或全面指导了群体事件;
   </li>
   <li class="graf graf--li graf-after--li" id="5a08">
    反对党和领导人成功地利用社交论坛传播他们的信息，或压过了主流媒体的宣传地位;
   </li>
   <li class="graf graf--li graf-after--li" id="be0b">
    该国公民的侨民社区 — — 包括其外国媒体、非政府组织和其他实体 — — 有效地分发宣传语言，进一步推动了大规模群体事件;
   </li>
  </ul>
  <p class="graf graf--p graf-after--li" id="89c4">
   能力：
  </p>
  <ul class="postList">
   <li class="graf graf--li graf-after--p" id="8c3c">
    在互联网基础设施很简单、国有化、发育不良、和/或被当局易于控制的情况下;
   </li>
   <li class="graf graf--li graf-after--li" id="a670">
    互联网过滤和监控技术在国内早已家喻户晓的情况下;
   </li>
   <li class="graf graf--li graf-after--li" id="4104">
    当局已经被怀疑或已被证明了在线监视活动的部署。
   </li>
  </ul>
  <p class="graf graf--p graf-after--li" id="3b64">
   并且，作为一个消极因素：
  </p>
  <p class="graf graf--p graf-after--p" id="859b">
   <strong class="markup--strong markup--p-strong">
    威慑力量
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li graf-after--p" id="ef00">
    该国的在线社区规模庞大，
    <strong class="markup--strong markup--li-strong">
     国际联系
    </strong>
    广泛;
   </li>
   <li class="graf graf--li graf--startsWithDoubleQuote graf-after--li" id="94ef">
    “
    <strong class="markup--strong markup--li-strong">
     数字经济
    </strong>
    ” 涵盖了该国经济的很大一部分和/或已被作为战略性国家资产;
   </li>
   <li class="graf graf--li graf-after--li" id="16f0">
    由于地缘政治背景，该国对
    <strong class="markup--strong markup--li-strong">
     国际压力
    </strong>
    很敏感。
   </li>
  </ul>
  <p class="graf graf--p graf-after--li" id="f0ce">
   <strong class="markup--strong markup--p-strong">
    — — 案例研究：巴拉圭 — —
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="9f43">
   巴拉圭这个贫穷的小国，去年正在经历一场（新的）宪法危机。总统奥拉西奥·卡特斯（Horacio Cartes）是一个统治该国权力节点并于 2013 年当选的小精英领袖，他试图强制实施一项宪法修正案，允许他在 2018 年再次当选。这被抗议者认为是经过将近三十年的民主后，这个国家已经出现了专制统治的明显转变。
  </p>
  <p class="graf graf--p graf-after--p" id="64e6">
   事实上，阿拉贝多·斯特罗斯纳将军 (Alfredo Stroessner) 在巴拉圭统治了 35 年，其独裁统治于 1989 年结束。从那时起，该国的民主一直摇摇欲坠，并受到腐败和政治动荡的困扰。
  </p>
  <p class="graf graf--p graf-after--p" id="1bca">
   2017 年3月31日，抗议者大批涌入国会，并点燃火把扔进建筑物。警方的反应最终导致了反对党 PLRA 总部的风暴、以及一个年轻的党员和活动家 Rodrigo Quintana 的死亡。
  </p>
  <p class="graf graf--p graf-after--p" id="d2a5">
   骚乱持续到 2017 年 4月5 日。
  </p>
  <p class="graf graf--p graf-after--p" id="6dce">
   <strong class="markup--strong markup--p-strong">
    研究方法
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="564a">
   即使针对特定网站，互联网封锁也不一定是基于战略思想的理性决策。正相反
   <strong class="markup--strong markup--p-strong">
    此举经常是专制政府或军政府对他们统治的社会失去控制的下意识反应。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="99de">
   例如，泰国军政府于 2014 年5月28日短暂封锁了整个国家的 Facebook，以抑制 5月22日政变后抗议内容的传播，并随后向社交媒体公司施压以求“合作”; 印度国家当局经常断网，以限制所谓“谣言”的传播 — 基本上是煽动抗议和骚乱的消息 — 就像去年在奥里萨邦的状况那样。
  </p>
  <p class="graf graf--p graf-after--p" id="4124">
   考虑到这一点，要尽可能评估政府采取这种极端措施的可能性，就需要了解相关在线社区的情况。为了做到这点，在深入研究分析模型的三个类别之前，查看舆情上下文的关键部分可能很重要：即本地在线社区的动态。
  </p>
  <p class="graf graf--p graf-after--p" id="3172">
   <strong class="markup--strong markup--p-strong">
    &gt; 人们如何访问互联网？
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="f748">
   这是了解政府在限制或破坏互联网服务访问方面的杠杆作用和激励的关键问题。我们需要查看当前在线社区的行为：
  </p>
  <ul class="postList">
   <li class="graf graf--li graf-after--p" id="a51b">
    固话宽带连接是最常用的访问网络的方式，还是移动服务占主导地位？
   </li>
   <li class="graf graf--li graf-after--li" id="2d37">
    宽带服务是最受欢迎的社交方式，还是电话通信（包括短信）更常见？在互联网基础设施严重不发达的国家，尤其是在撒哈拉以南的非洲，这种情况是很普遍的。
   </li>
  </ul>
  <p class="graf graf--p graf-after--li" id="87ec">
   在巴拉圭的背景下：
  </p>
  <ul class="postList">
   <li class="graf graf--li graf-after--p" id="b805">
    固话连接很少。在广受欢迎的移动互联网服务中，有两家供应商主导市场；
   </li>
   <li class="graf graf--li graf-after--li" id="0f09">
    虽然电话服务仍然是一种重要的通信手段，但社交媒体的使用是很普遍的，在40岁以下的人口中占有很大的比例（约 70％），联合国儿童基金会估计 2014 年巴拉圭平均每个月在社交网络上创建大约
    <a class="markup--anchor markup--li-anchor" data-href="https://www.unicef.org/paraguay/spanish/TORs_LaVozDeLosJovenes.pdf" href="https://www.unicef.org/paraguay/spanish/TORs_LaVozDeLosJovenes.pdf" rel="noopener nofollow noreferrer" target="_blank">
     30 000 个新用户
    </a>
    。
   </li>
  </ul>
  <p class="graf graf--p graf-after--li" id="ed37">
   <strong class="markup--strong markup--p-strong">
    &gt; 该国的主要通信平台是什么？
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="1c79">
   社交媒体的使用因国家/地区而异。同样，即时消息传递应用在过去十年中发展迅速，并且根据所讨论社区的不同，存在很多重要差异。
  </p>
  <p class="graf graf--p graf-after--p" id="d034">
   经常被引用的一个消息来源是
   <a class="markup--anchor markup--p-anchor" data-href="http://vincos.it/world-map-of-social-networks/" href="http://vincos.it/world-map-of-social-networks/" rel="noopener nofollow noreferrer" target="_blank">
    Vincenzo Cosenza
   </a>
   的社交网络世界地图。至2017年1月（动乱发生时）的显示是这样的：
  </p>
  <ul class="postList">
   <li class="graf graf--li graf-after--p" id="f035">
    巴拉圭（以及拉丁美洲其他地区）的顶级热门社交媒体平台是 Facebook ;
   </li>
   <li class="graf graf--li graf-after--li" id="c9ca">
    亚军是 Instagram ;
   </li>
   <li class="graf graf--li graf-after--li" id="0e63">
    可用的公共数据显示，Twitter 只是一个遥远的次要社交平台的存在。2017年3月，只有1.5％的社交媒体使用通过 Twitter 发生 — 而不是 Facebook 的 97％。其他消息来源也证实了这一点，包括
    <a class="markup--anchor markup--li-anchor" data-href="https://www.latamclick.com/estadisticas-redes-sociales-2016-paraguay/" href="https://www.latamclick.com/estadisticas-redes-sociales-2016-paraguay/" rel="noopener nofollow noreferrer" target="_blank">
     本地产生的统计数据
    </a>
    ，估计截至 2016 年巴拉圭的 Twitter 用户为 400.000。
   </li>
  </ul>
  <p>

   <p class="graf graf--p graf-after--p" id="88b1">
    至于
    <strong class="markup--strong markup--p-strong">
     监视
    </strong>
    ，巴拉圭在五月份辞职的总统奥拉西奥卡特斯的领导下，已经在试图实施有争议的措施。例如，一项旨在迫使互联网服务供应商长期保留数据的法案，被称为“
    <a class="markup--anchor markup--p-anchor" data-href="https://www.eff.org/deeplinks/2014/12/pyrawebs-paraguayans-rise-against-mandatory-data-retention" href="https://www.eff.org/deeplinks/2014/12/pyrawebs-paraguayans-rise-against-mandatory-data-retention" rel="noopener nofollow noreferrer" target="_blank">
     Pyrawebs
    </a>
    ”，最终在全球活动家和当地互联网社区虚拟起义后，于2015年中期被众议院和参议院否决。
   </p>
   <p class="graf graf--p graf-after--p" id="d2de">
    大约在同一时间，巴拉圭政府进入了加拿大公民实验室
    <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.org/2015/10/mapping-finfishers-continuing-proliferation/" href="https://citizenlab.org/2015/10/mapping-finfishers-continuing-proliferation/" rel="noopener nofollow noreferrer" target="_blank">
     报告的引用
    </a>
    ，
    <strong class="markup--strong markup--p-strong">
     他们可能是 FinFisher 间谍软件技术的使用者，该间谍软件不仅被用于执法调查，而且还被用于政治间谍活动，针对反对派和异议
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="c0f3">
    近年来，在线言论审查也是一个严重的问题。例如，在 2012 年，两个市场主导的移动运营商（Tigo 和 Personal）几乎整天都在阻止对本地网站的访问，主要针对的是回复
    <a class="markup--anchor markup--p-anchor" data-href="https://advox.globalvoices.org/2012/09/27/paraguay-isps-block-news-website-without-warrant/" href="https://advox.globalvoices.org/2012/09/27/paraguay-isps-block-news-website-without-warrant/" rel="noopener nofollow noreferrer" target="_blank">
     政治类内容
    </a>
    评论的用户，而且封锁没有法庭命令。
   </p>
   <p class="graf graf--p graf-after--p" id="91f3">
    互联网关闭是最终极的激烈措施，在巴拉圭仍然是史无前例的。
   </p>
   <p class="graf graf--p graf-after--p" id="427b">
    总体而言，在较高的层面上，我们了解到了一个不成熟的互联网基础设施，在一个小国，其政府已经显示出至少致力于关于监视和控制的显著迹象，以及有针对性的审查技术的存在。
   </p>
   <p class="graf graf--p graf-after--p" id="1763">
    如果政治动荡进一步加剧，并有影响政治现状的能力，可以认为全面的互联网阻滞是可能的。
   </p>
   <p class="graf graf--p graf-after--p" id="cee2">

     国际压力
    </strong>
    （特别是来自西方国家的压力- 来自该国的侨民社区也可以推动地方政府就互联网限制发表意见）可能是一个重要的威慑因素。他们与西方的联系、对当地社区的影响以及国际压力，是一个非常微妙和流动性的领域，例如 Steven Levitsky 和 Lucan A. Way 已经
    <a class="markup--anchor markup--p-anchor" data-href="https://books.google.nl/books/about/Competitive_Authoritarianism.html?id=NZDI05p1PDgC&amp;redir_esc=y" href="https://books.google.nl/books/about/Competitive_Authoritarianism.html?id=NZDI05p1PDgC&amp;redir_esc=y" rel="noopener nofollow noreferrer" target="_blank">
     做过的详细研究
    </a>
    ，与西方国家关系的地缘政治因素在其中有明显作用。
   </p>
   <p class="graf graf--p graf-after--p" id="146e">
    由于各种原因（其中包括文化、经济和其他原因）巴拉圭可以被视为一个与西方有着温和并现存联系的国家。喀麦隆这样的国家就是个相反的例子 — 自 2017 年1月以来，由于政治原因，互联网在整个地区被封锁 — 这可能需要分别进一步分析，关于具体国家与发达民主国家之间存在的联系、及其权力（或意志）可影响当地政府行动的能力。
   </p>
   <p class="graf graf--p graf-after--p" id="a8be">
    巴拉圭当局更有可能采取行动，选择性地阻止政治话语活跃的社交媒体（或即时通讯）平台。这可能会持续更长的时间，并且对抗议者来说，会带来进一步令人沮丧的风险，重要的救援渠道被封堵，最终导致他们面对增大的政府压力。
   </p>
   <p class="graf graf--p graf-after--p" id="62fa">

     最后回顾
    </strong>
   </p>

   </p>
   <p class="graf graf--p graf-after--p" id="a20a">
    鉴于社会持久的对政府的负面情绪，以及抗议者表现出快速聚集的可能性，政府关闭互联网的动机仍然极有可能出现。
   </p>
   <p class="graf graf--p graf-after--p" id="8e7c">
    在研究哪些因素可能影响结果的过程中，我们概述了一个初步的研究和监测模型，可以作为分析和预测的初始模板，希望有助于进一步研究，

    <p class="graf graf--p graf-after--figure" id="28eb">
     即便互联网已无法访问，但每个人的电脑实际上还是互相连接的。换句话说你们仍处于同一个巨型局域网之中，甚至仍有公网 IP。所以理论上通过某种方式，你或许能再度利用那些被荒废的网线和电缆。这里有
     <strong class="markup--strong markup--p-strong">
      三个方法
     </strong>
     ，你只需要一个 U 盘：
    </p>
    <p class="graf graf--p graf-after--p" id="5b7e">
     1. 无服务器、永远在线的网站 — —
     <strong class="markup--strong markup--p-strong">
      ZeroNet
     </strong>
     网址：
     <a class="markup--anchor markup--p-anchor" data-href="https://zeronet.io/" href="https://zeronet.io/" rel="noreferrer noopener nofollow" target="_blank">
      https://zeronet.io/
     </a>
    </p>
    <p class="graf graf--p graf-after--p" id="e16b">
     ZeroNet 是一个免费、开源、使用 Bitcoin 加密技术和 BT 技术的全平台分布式网络工具，其原理就像BT下载：当通过种子(torrent）下载电影和音乐时，这些资源的来源并非某个机房中的服务器，而是其他用户的电脑硬盘，只要在网络中还存在任何一台电脑『做种』，资源就能够被持续下载，同时提供资源的设备量也会迅速增长，因此 BT 下载无法被单点封禁。
    </p>
    <p class="graf graf--p graf-after--p" id="4a45">
     ZeroNet 的工作原理与 BT 大致相同。在你通过 ZeroNet 搭建一个网站后，处于同一个网络中的用户即可通过 ZeroNet 将你的网络论坛下载到他的硬盘里，同时进行浏览。当第三个用户访问这个站点时，你和第二个用户会同时向他上传网站。如果用户基数足够大，那么即便你的电脑关机甚至硬盘损坏，只要别人的电脑上仍保存有网站的文件，这个网站就能永远『存活』下去。
    </p>
    <p class="graf graf--p graf-after--p" id="e14d">
     ZeroNet 提供论坛、博客、社交网站、加密邮件、即时聊天等，几乎覆盖了大多数的互联网服务。在断网时期，网民实际上有能力以极低成本来为彼此搭建这些基础服务。不仅如此，ZeroNet 还可以通过 Tor 网络隐藏身份。
    </p>
    <p class="graf graf--p graf-after--p" id="7607">
     2. 断网后照样聊天 — —
     <strong class="markup--strong markup--p-strong">
      Tox
     </strong>
     网址：
     <a class="markup--anchor markup--p-anchor" data-href="https://tox.chat/" href="https://tox.chat/" rel="noreferrer noopener nofollow" target="_blank">
      https://tox.chat/
     </a>
    </p>
    <p class="graf graf--p graf-after--p" id="dc01">
     Tox 是一个开源免费的分布式加密通讯协议，诞生于斯诺登曝光之后，对用户的通讯内容进行端到端加密，中间不经过任何中央服务器，还可以用于局域网内通讯。
    </p>
    <p class="graf graf--p graf-after--p" id="f565">
     Tox 和 ZeroNet 一样，也使用了 BT 技术，将用户点对点的连接起来。在你使用 Tox 的时候，会生成一个唯一的用户名，只要你的朋友知道你的用户名，即可开始聊天。
    </p>
    <p class="graf graf--p graf-after--p" id="d43b">
     目前 Tox 已经支持全平台，并有很多版本的客户端，界面简单易用不需要复杂配置。
    </p>
    <p class="graf graf--p graf-after--p" id="d59e">
     3. 传递资料 — —
     <strong class="markup--strong markup--p-strong">
      Resilio Sync
     </strong>
    </p>
    <p class="graf graf--p graf-after--p" id="b213">
     Resilio Sync 是一个分布式同步工具，支持全平台（但不开源）。当你把一个文件夹加入 Resilio Sync 中时，可以生成一串密钥，如果在另一台设备的 Resilio Sync 中输入该密钥，就能将你的文件夹远程同步到这台设备中。之后加入的所有同步节点，都可以在下载资源的同时向其他用户上传资源。
    </p>
    <p class="graf graf--p graf-after--p" id="f367">
     由于文件存储在本地硬盘而非服务器上、对传输内容进行了加密、又以 BT 基础为依托，Resilio Sync 实际上是一个
     <strong class="markup--strong markup--p-strong">
      高安全度、无存储空间限制、无下载流量限制、无审查、无法被封杀的超级网盘
     </strong>
     。资源拥有者甚至可以通过设置权限来控制资源的分发和更改。例如你和你的朋友希望分发你们的音乐专辑，那么你可以分配给他读写权限，然后将只读权限通过各种方法公开出去，此时你们二人可以远程协作共同编辑专辑，而其他网友则可自由下载，当文件被修改的时候，所有同步节点都会被更改。
    </p>
    <p class="graf graf--p graf-after--p" id="f4ac">
     不论如何，准备一个 PlanB 不至于措手不及。⚪️
    </p>
