  <p class="graf graf--p">
   从手机跟踪到空中无人机监视、从事无巨细的健康记录到GPS脚腕手环 ……
   <strong class="markup--strong markup--p-strong">
    这些监视国家的手段正在全球范围内流行
   </strong>
   。
  </p>
  <p class="graf graf--p">
   当然，所有这些都是以 “与COVID-19的传播作斗争” 的借口完成的 ——
   <strong class="markup--strong markup--p-strong">
    每一次危机都被用来作为入侵公民自由的借口。
   </strong>
  </p>
  <p class="graf graf--p">
   但是，在病毒表现最糟糕并继续蔓延很久之后，人类很可能会被这些侵犯我们隐私人权的行为所困，除非我们现在开始努力挽回。
  </p>
  <p class="graf graf--p">
   在一定范围内，监视的原理很容易理解。与大流行作斗争的公共卫生当局想要知道谁在传播该病毒，他们可能感染了哪些人，以及可能携带病毒的人的活动。
  </p>
  <p class="graf graf--p">
   COVID-19爆发始于中国，该国利用其已经深深嵌入的社会控制系统来迫使人们安装手机监视应用程序，该应用程序据称根据人们接触传染病蔓延的风险为他们每个人分配标示码。
  </p>
  <p class="graf graf--p">
   允许旅行或进入公共场所的权限就取决于该码 ——
   <strong class="markup--strong markup--p-strong">
    但很显然，该软件还跟踪每个人的下落并与当局共享公民手机的数据。
   </strong>
  </p>
  <p class="graf graf--p">
   朝鲜的疫情状况并不像中国那样，但是该国仍然在跟踪人们的手机和信用卡使用情况。官员们还使用监视摄像机监视有可能被感染者的一切活动。
  </p>
  <p class="graf graf--p">
   西班牙当局效仿中国的战术，转向空中无人机来侦查未经授权的人员集会 —— 然后，无人机上的扬声器命令违规者返回家中。
  </p>

   <a class="markup--anchor markup--p-anchor" data-href="https://www.lexology.com/library/detail.aspx?g=fdc484ee-7914-476d-b1d9-900bf16bf37a" href="https://www.lexology.com/library/detail.aspx?g=fdc484ee-7914-476d-b1d9-900bf16bf37a" rel="noopener noreferrer" target="_blank">

   <a class="markup--anchor markup--p-anchor" data-href="https://reason.com/2020/04/08/covid-19-patients-and-their-families-are-being-put-on-police-enforced-lockdown/" href="https://reason.com/2020/04/08/covid-19-patients-and-their-families-are-being-put-on-police-enforced-lockdown/" rel="noopener noreferrer" target="_blank">
    GPS脚环监视器
   </a>
   安装在COVID-19患者和那些可能暴露在外的人身上，以免他们在该国散步或触摸路边的杂物。
  </p>
  <p class="graf graf--p">
   在大多数情况下，老大哥式战术都已作为临时措施出售，旨在应对COVID-19大流行带来的非常现实的危险。当权者声称监视是 “为了防疫”。
  </p>
  <p class="graf graf--p">
   但是，疫情带给这些当权者以极大的反乌托邦权力，公民能否信任当权者的话声称疫情过后他们将放弃这种至高无上的权力？
  </p>
  <p class="graf graf--p">
   电子前沿基金会（EFF）的高级职员 Adam Schwartz
   <a class="markup--anchor markup--p-anchor" data-href="https://www.eff.org/deeplinks/2020/04/how-eff-evaluates-government-demands-new-surveillance-powers" href="https://www.eff.org/deeplinks/2020/04/how-eff-evaluates-government-demands-new-surveillance-powers" rel="noopener noreferrer" target="_blank">
    警告
   </a>
   说：
   <strong class="markup--strong markup--p-strong">
    政府对新型高科技监视能力的垂涎实在是太熟悉了。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “即使在危机之中，公众也必须仔细评估政府的这种要求，因为监视会侵犯人权，阻止言论自由，并给弱势群体带来不成比例的负担。”
  </p>
  <p class="graf graf--p">
   Schwartz 警告：“而且，新的监视能力往往会持续存在。”
  </p>
  <p class="graf graf--p">
   EFF 并不是唯一担心这件事的人。
  </p>
  <p class="graf graf--p">
   安全专家 Bruce Schneier
   <a class="markup--anchor markup--p-anchor" data-href="https://www.schneier.com/blog/archives/2020/03/privacy_vs_surv.html" href="https://www.schneier.com/blog/archives/2020/03/privacy_vs_surv.html" rel="noopener noreferrer" target="_blank">
    写道
   </a>
   ：“我认为 COVID-19 的后果将比9/11恐怖袭击的后果更为严重：不仅在监视方面，而且在我们社会的各种方面。”
  </p>
  <p class="graf graf--p">
   尽管许多正常情况下无法接受的事 “特殊情况下”是可以做的，但是，我们需要确保一旦当前的大流行结束，我们就必须迅速收回政府的这种权力。”
  </p>
  <p class="graf graf--p">
   我们是否有能力收回政府的权力，是这里监视权的关键。正如 Schwartz 所提醒的那样，各国政府倾向于扩大应对危机的能力，但随着危险的消退，政府只会放弃部分新权力。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “在每次重大危机之后，政府的规模虽然比危机期间要小，但是，仍要比危机前更大”，Robert Higgs 在1987年的 Crisis and Leviathan 一书中就这样写道。他创造了“棘轮效应”一词来描述这一现象。

   </a>
   ，以获取有关食源性疾病的信息，以瞄准并关闭可疑的餐馆。技术性监视项目的 Albert Fox Cahn 和 John Veiszlemlein
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nbcnews.com/think/opinion/covid-19-tracking-data-surveillance-risks-are-more-dangerous-their-ncna1164281" href="https://www.nbcnews.com/think/opinion/covid-19-tracking-data-surveillance-risks-are-more-dangerous-their-ncna1164281" rel="noopener noreferrer" target="_blank">
    说
   </a>
   ，我们应该期望最终将这种技术应用于大流行。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    人们会为了对健康的期待而允许老大哥行为，但不幸的是，我们的自由和隐私将在此过程中缓慢且令人窒息地死亡。也就是说，除非我们开始积极反对监测我们私人活动的努力。
   </strong>
  </p>
  <p class="graf graf--p">
   中国是地球上最严酷的监视国家之一，它为寻求逃避审查的人激发了无限的创新能力。人们学会了在手机上安装反间谍软件，或更简单的方法是携带两种设备，一个不敏感的等着被审查，另一个藏起来。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e7%9a%84%e6%89%8b%e6%9c%ba%e5%a6%82%e4%bd%95%e8%83%8c%e5%8f%9b%e4%ba%86%e6%b0%91%e4%b8%bb%ef%bc%9f/" href="https://www.iyouport.org/%e4%bd%a0%e7%9a%84%e6%89%8b%e6%9c%ba%e5%a6%82%e4%bd%95%e8%83%8c%e5%8f%9b%e4%ba%86%e6%b0%91%e4%b8%bb%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    手机本身是一个贴身的高级间谍
   </a>
   ，越来越多的人意识到在哪些敏感的重要的情况下不要携带自己的手机。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    你很难完全退出，但是请记得，监视者所能做到的跟踪是对你的手机的定位，而不是你的人，如果你不把手机带着身上，他们就找不到你了。
   </strong>
  </p>
  <p class="graf graf--p">
   您完全可以避免在任何社交媒体上发布有关您的私人生活的任何细节，包括您的健康状况。
  </p>
  <p class="graf graf--p">
   至于无人机监视，击落他们，调整各种不同的射程。一个案例 《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%8a%97%e8%ae%ae%e8%80%85%e4%bd%bf%e7%94%a8%e6%95%b0%e7%99%be%e4%b8%aa%e6%bf%80%e5%85%89%e7%ac%94%e5%87%bb%e8%90%bd%e8%ad%a6%e5%af%9f%e6%97%a0%e4%ba%ba%e6%9c%ba/" href="https://www.iyouport.org/%e6%8a%97%e8%ae%ae%e8%80%85%e4%bd%bf%e7%94%a8%e6%95%b0%e7%99%be%e4%b8%aa%e6%bf%80%e5%85%89%e7%ac%94%e5%87%bb%e8%90%bd%e8%ad%a6%e5%af%9f%e6%97%a0%e4%ba%ba%e6%9c%ba/" rel="noopener noreferrer" target="_blank">
    抗议者使用数百个激光笔”击落”警察无人机
   </a>
   》。
  </p>
  <p class="graf graf--p">
   但最好的办法是，在危机逐渐解除后，政府官员可以能够放弃这些监视措施。
   <strong class="markup--strong markup--p-strong">
    公民的反抗、揭露和监督必须能跟上节奏
   </strong>
   。⚪️
  </p>
  <p class="graf graf--p">

    The Surveillance State Thrives During the Pandemic

  </p>
