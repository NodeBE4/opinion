    <strong class="markup--strong markup--li-strong">
     在全球范围内监视所有电话和互联网通信的项目
    </strong>
    ；
   </li>
   <li class="graf graf--li">
    您的手机现已成为秘密监视设备，可以跟踪您的一举一动，几乎所有通信和在线活动；
   </li>
   <li class="graf graf--li">
    即使您的手机已关闭GPS，只要您已连接到Wi-Fi，由于Wi-Fi调制解调器中的全局唯一标识符，您的确切位置仍会被识别和记录；
   </li>
   <li class="graf graf--li">
    我们需要的是能够自己决定哪些应用和程序可以进行通信以及何时进行通信，只有这样才能重新获得控制个人隐私的能力。
   </li>

  </p>

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     斯诺登是谁？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   斯诺登（Snowden）于2013年夏天成为全球家喻户晓的名字，当时他作为泄露美国国家安全局（NSA）机密文件的来源，这些文件证明存在了旨在监视全球的系统。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    在这里看到
    <strong class="markup--strong markup--li-strong">
     完整版报告的中文解读
    </strong>
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" rel="noopener noreferrer" target="_blank">
     互联网霸主的时间线 — — 他们将互联网视为间谍、破坏和战争的主场
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   斯诺登（Snowden）是高级安全技术专家，曾是美国中央情报局（US Central Intelligence Agency）的前雇员，他曾在咨询公司和国防承包商 Booz Allen Hamilton 担任技术专家。
  </p>
  <p class="graf graf--p">
   以此身份，他被签约到美国国家安全局在夏威夷的一家工厂工作。在NSA任职的四年中，他了解了这些秘密的监视程序。
  </p>
  <p class="graf graf--p">
   斯诺登最终泄露了一大堆文件，
   <strong class="markup--strong markup--p-strong">
    这些文件显示了公众受到监视的极端程度 —— 而且目前为止这种监视一直在持续
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%96%af%e8%af%ba%e7%99%bb%e6%9b%9d%e5%85%896%e5%b9%b4%e5%90%8e%ef%bc%8c%e5%a4%a7%e8%a7%84%e6%a8%a1%e6%8b%a6%e6%88%aa%e7%9a%84%e7%8e%b0%e7%8a%b6/" href="https://www.iyouport.org/%e6%96%af%e8%af%ba%e7%99%bb%e6%9b%9d%e5%85%896%e5%b9%b4%e5%90%8e%ef%bc%8c%e5%a4%a7%e8%a7%84%e6%a8%a1%e6%8b%a6%e6%88%aa%e7%9a%84%e7%8e%b0%e7%8a%b6/" rel="noopener noreferrer" target="_blank">
     斯诺登曝光6年后，大规模拦截的现状
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   斯诺登在向《华盛顿邮报》泄露的第一份文件所附的便条中说，他披露信息的唯一动机是 “向公众通报政府以他们的名义所作的一切如何对他们不利。”
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     透明度革命
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   虽然有些人（尤其是情报界的人）将他视为叛徒，但公民普遍将他视为爱国者和英雄。毫无疑问，他是现代历史上最重要的举报人之一。
  </p>
  <p class="graf graf--p">
   斯诺登牺牲了自己的一切只为使大众意识到正在发生的非法的侵犯性的严重监视行为。根据《卫报》 2013年的报道：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “在此之前他过着 ‘非常舒适的生活’，其中包括大约200,000美元的薪水，与女友合住的夏威夷的房子、稳定的职业、以及他所爱的家庭。
   </em>
  </p>
  <p class="graf graf--p graf--startsWithSingleQuote">
   <em class="markup--em markup--p-em">
    ‘我愿意牺牲所有这一切，因为我不能凭良心让美国政府使用他们秘密建造的这种大型监视机器破坏全世界人们的隐私权、互联网自由和基本自由。”
   </em>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e9%97%b4%e8%b0%8d%e3%80%81%e5%92%8c%e5%8c%bf%e5%90%8d%e7%9c%9f%e7%9b%b8/" href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e9%97%b4%e8%b0%8d%e3%80%81%e5%92%8c%e5%8c%bf%e5%90%8d%e7%9c%9f%e7%9b%b8/" rel="noopener noreferrer" target="_blank">

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     全球监视合法吗？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在接受罗根（Rogan）采访时，斯诺登（Snowden）回答了一个普遍的误解，即：他发现的全球技术监视计划没什么大不了的，因为，“这不是间谍机构应该做的吗？”

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e6%8e%a7%e5%88%b6%e7%9a%84%e4%b8%96%e7%95%8c%e5%92%8c%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89-2/" href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e6%8e%a7%e5%88%b6%e7%9a%84%e4%b8%96%e7%95%8c%e5%92%8c%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89-2/" rel="noopener noreferrer" target="_blank">
     谷歌控制的世界和监视资本主义
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   要深入了解 Google 如何审查政治信息，请参阅 “
   <a class="markup--anchor markup--p-anchor" data-href="https://articles.mercola.com/sites/articles/archive/2019/11/02/google-whistleblower-veritas.aspx" href="https://articles.mercola.com/sites/articles/archive/2019/11/02/google-whistleblower-veritas.aspx" rel="noopener noreferrer" target="_blank">
    Google举报人 Zach Vorhies 的演讲
   </a>
   ”。在许多方面，
   <strong class="markup--strong markup--p-strong">
    Vorhies 就是 Google 的斯诺登，他揭开了监视资本主义巨头的神秘面纱，以揭示Google操纵用户并从幕后影响政治的多种方式
   </strong>
   。
  </p>
  <p class="graf graf--p">
   即使目前应对监视问题的解决方案依旧很少，但如果我们仍然不知道监视的工作方式甚至不知道监视的发生，我们将永远找不到任何答案。

   IYP 分析过关于谷歌数据收集的问题《

    到底有多少跟踪器在盗窃你的隐私？谷歌以其惊人的跟踪量位居榜首
   </a>
   》；答案的一部分很简单，就是抵制Google，避免使用 Google 产品。
  </p>
  <p class="graf graf--p">
   这里有一些建议：

   <li class="graf graf--li">

     Google 产品的替代品 — 完整列表 关注隐私权的你，马上行动起来～
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e4%bd%a0%e6%93%85%e9%95%bf%e6%90%9c%e7%b4%a2%ef%bc%8c%e4%bd%a0%e8%83%bd%e6%89%be%e5%88%b0%e4%b8%80%e5%88%87%ef%bc%9a%e6%b2%a1%e6%9c%89%e8%b0%b7%e6%ad%8c%e7%9a%84%e5%bc%ba%e5%a4%a7/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e4%bd%a0%e6%93%85%e9%95%bf%e6%90%9c%e7%b4%a2%ef%bc%8c%e4%bd%a0%e8%83%bd%e6%89%be%e5%88%b0%e4%b8%80%e5%88%87%ef%bc%9a%e6%b2%a1%e6%9c%89%e8%b0%b7%e6%ad%8c%e7%9a%84%e5%bc%ba%e5%a4%a7/" rel="noopener noreferrer" target="_blank">
