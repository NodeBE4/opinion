  <p class="graf graf--p">
   👉于是这里的关键在于，你必须牢牢记住你的虚构身份有关的 **一切**，而不仅仅是ID；需要在任何试探和诱导面前都不会露馅。
  </p>
  <p class="graf graf--p">
   有些情况下需要多个掩护身份同时存在，而且在面对极大压力的情况下很容易混淆这些身份。一旦 …… 那就完败了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    反情报工具有一套成熟的机制，目的就是为识破掩护身份
   </strong>
   。比如，
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    要求被怀疑对象以倒序重读他从现在到过去的经历；
   </li>
   <li class="graf graf--li">
    重新描述问题，对它进行扩展，或者相反，使其更加具体；
   </li>
   <li class="graf graf--li">
    询问细节，然后将对方的回答与其他来源的信息进行对比，等等
   </li>
  </ul>
  <p class="graf graf--p">
   如果存在任何一点矛盾和不一致的地方，或者一点问题都没有，那么这个人就值得警惕了。（一点问题都没有也不行？是的，那会显得太假。）
  </p>

     练习
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在你的头脑中回顾最近6个月或一年中发生的一切。记住这段时间内的所有事，细节越多越好，并将记忆编织成一个完整的故事。
  </p>
  <p class="graf graf--p">
   使用尽可能所有可用的素材：家庭档案、信件、账单、本地新闻等等各种资料。询问你的家人、朋友和同事，以检查自己的记忆是否有误。

     利用气味记忆
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   对于所有生物来说。气味都是重要的信息来源。
  </p>
  <p class="graf graf--p">
   嗅觉是一种警告系统，提醒你注意危险信号，指明了食物或其他个体接近你的程度。
  </p>
  <p class="graf graf--p">
   对于某些物种而言，嗅觉是一种交流手段。相比下人类的嗅觉不如某些动物，但依然
   <strong class="markup--strong markup--p-strong">
    会对记忆有很大的影响
   </strong>
   。
  </p>
  <p class="graf graf--p">
   人类的嗅觉记忆远强于视觉和听觉记忆。也许这是因为接收来自嗅觉信号的大脑区域与负责形成长期记忆的海马体之间有强大的神经联系。
  </p>
  <p class="graf graf--p">
   气味可以唤醒长期遗忘的记忆及与之相关的感受。比如，你能永远记住家的气息，而且一闻到就能立即识别出来。即使分别多年，男人一闻到女人的香水味就能激起内心的情绪。
  </p>
  <p class="graf graf--p">
   参与治疗失忆症的心理学家知道，在某些情况下，重要的熟悉气味可以帮助病人恢复对事件的记忆。催眠师也使用这种方法将被试的注意力集中在目标情境中。
  </p>
  <p class="graf graf--p">
   你也可以对记忆和嗅觉之间的关系加以利用。如上，如果你希望记住掩饰身份相关的一切，不如使用特定的气味标记它。
  </p>
  <p class="graf graf--p">
   这种气味标签只有你知道，于是很难被对手所利用。
  </p>
  <p class="graf graf--p">
   再比如，如果你能记住某次聚餐中食物的气味，就可以帮助你回忆这场聚会中的对话。
  </p>

   <span style="color: #00ccff;">

  </p>

  <h3 class="graf graf--p">
   <span style="color: #99cc00;">
    <strong class="markup--strong markup--p-strong">
     前瞻性记忆
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   前瞻性记忆是记住意图或计划行动的能力。与记忆过去真实发生的东西的回顾性记忆不同，
   <strong class="markup--strong markup--p-strong">
    前瞻性记忆处理的是未来 —— 那些准备去做的、尚未发生的事。
   </strong>
  </p>
  <p class="graf graf--p">
   记住未来这一行为的发起是截然不同的，因为没有相关经验或者过往记忆可以回想。
  </p>
  <p class="graf graf--p">
   工程、航空和军事心理学家都深入研究过前瞻性记忆。大多数事故和灾难都是由于存在前瞻性记忆缺陷的人造成的：操作人员忘了某件事，并因此导致了一连串的恶性后果。
  </p>
  <p class="graf graf--p">
   在航空领域，有很多规则可以弥补前瞻性记忆缺陷。这些规则系统化地将关键性动作组织起来，以便将对个人注意力和记忆力的依赖降到最低。
  </p>
  <p class="graf graf--p">
   比如，在飞行控制阶段，所有操作都必须遵循打印好的任务清单上的条款，并且每一步至少需要两名机组成员参与：其中一人负责执行，然后向另一人汇报自己的行为，另一人负责检查操作效果。
  </p>
  <p class="graf graf--p">
   所有操作都有必须被牢牢记住的、习惯性的和毫无疑问的执行程序。然而即便采取了这些措施，出错也不能完全避免。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    👉对于几乎任何一种行动来说，前瞻性记忆都是非常重要的，否则代价有可能很高
   </strong>
   。如果您是团队的组织者，一定要注意这点 —— 在行动之前请您的合作者/队友复述步骤，正序和倒序复述都要进行。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #99cc00;">
    <strong class="markup--strong markup--p-strong">
     前瞻性记忆训练
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   要确定你的前瞻性记忆有多好，请回答下面的问题：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    你曾经忘记过生日、假期、周年纪念日吗？
   </li>
   <li class="graf graf--li">
    你有没有烧开水后忘了泡茶的情况？
   </li>
   <li class="graf graf--li">
    你是否会忘记一些小任务，以致于其他人需要不断重复给你？
   </li>
   <li class="graf graf--li">
    你有没有走到冰箱前，却不知道自己为什么要走到这里，不知道刚才想要拿什么东西？
   </li>
   <li class="graf graf--li">
    在谈话过程中如果被打断，在继续之前你是否会问 “我刚才说到哪了？”
   </li>
   <li class="graf graf--li">
    你是否容易忘记与他人的约定，尤其是在你很忙的情况下？
   </li>
  </ul>
  <p class="graf graf--p">
   如果有任何一个问题的答案为 “是”，那么您就有必要训练前瞻性记忆了。
  </p>
  <p class="graf graf--p">
   如前所述，
   <strong class="markup--strong markup--p-strong">
    前瞻性记忆由两部分组成：对意图的记忆，以及在正确的时间回想意图
   </strong>
   。训练前瞻性记忆的方法有很多种，您此前练习的记忆词汇表已经向前迈出了一步。
  </p>
  <p class="graf graf--p">
   但是，由于缺乏外部提醒，大多数错误都发生在前瞻性记忆的第二部分。
  </p>
  <p class="graf graf--p">
   心理学家 Stephen LaBerge 提出过一种训练方法，您可以试试。那就是，设定 “目标” —— 那种一天内会可能发生好几次的事件。比如，“目标” 可以是 “我看到了一个数字7”、“有人在清扫大街”、“穿红色衣服的女人过马路”。
  </p>
  <p class="graf graf--p">
   于是您的目标就是，在 白天尽可能注意数字7、街道清洁员、穿红衣服的女人。仔细想想你有多少次命中了目标？请在日记中标记出这些事。
  </p>
  <p class="graf graf--p">
   每周做一次命中数据的分析。可以先用一个目标训练几天。然后再换另一个目标。也可以采取增量的形式，从一个目标开始，然后是两个、三个 ……
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   用目标事件进行训练，可以培养为了回应外部事件而记忆意图的能力。
  </p>
  <p class="graf graf--p">
   但如果想要拥有好的前瞻性记忆，仅仅这样还不够：你必须能够在正确的时间回想起自己的意图。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    行动者必须有很强的时间感
   </strong>
   。首先，时间感能帮助您在没有外部提示的情况下在合适的时间记起所需的内容；其次，您能够更准确地完成任务，并清楚地意识到你的工作需要花多长时间。
  </p>
  <p class="graf graf--p">
   👉现代人一直在依靠手机和手表来感知时间，真正的时间感就会逐渐退化。为了恢复这种感觉，您应该有意识地调用自己的直觉。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   注意秒表或者带表针的手表上的时间。不看手表，试着估算一分钟的长度。
  </p>
  <p class="graf graf--p">
   然后估算5分钟、十分钟、一小时。
  </p>
  <p class="graf graf--p">
   不要数秒，专心做其他事。
  </p>
  <p class="graf graf--p">
   如果你的生物钟很慢或者很快，可以检查一下，然后再重新尝试。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   不戴手表也不看手机，过一整天。这样能帮您觉察到自己是如何支配时间的。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   开始一项任务的时候试着预测一下它需要花费多长时间完成。工作时不看手表。任务完成后将实际消耗的时间与你的预测时间相比较，看看它们之间差多少？是高估了还是低估了？
  </p>
  <p class="graf graf--p">
   从写电子邮件、打扫房间等小的任务开始；达到一定程度后就可以准备应付大任务了。

  <h3 class="graf graf--p">
   <span style="color: #99cc00;">
    <strong class="markup--strong markup--p-strong">
     如何提高前瞻性记忆
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   前瞻性记忆可以储存关于意图的信息，并及时就这些信息给你提醒。
  </p>
  <p class="graf graf--p">
   要提醒自己某件事，
   <strong class="markup--strong markup--p-strong">
    最简单的方法就是改变你所处的环境
   </strong>
   。让它变得与众不同、显而易见。比如，你可以将训练器材放在牙刷附近，以免忘了晨练。
  </p>
  <p class="graf graf--p">
   还有一种特别的方法，可以让你记住必须携带的东西，比如把车钥匙放进冰箱里。在一个
   <strong class="markup--strong markup--p-strong">
    不寻常的地方
   </strong>
   找到钥匙之后，你不会忘记带上答应给同事的书。
  </p>
  <p class="graf graf--p">
   无论是在紧密相关的时刻，还是无关紧要的时刻，你都可以不断提醒自己某项重要任务。这种做法可以用一个很好的例子来说明，那就是在手上画一个十字。一看到自己的手就会提醒你需要完成什么。
  </p>
  <p class="graf graf--p">
   口袋型记事本和时间表都可以支持前瞻性记忆。待办清单不仅是一个任务列表，
   <strong class="markup--strong markup--p-strong">
    也是一种行为策略
   </strong>
   。
  </p>
  <p class="graf graf--p">
   记住意图与一种习惯相关，这种习惯就是在开始工作前和每项任务结束时都检查一下自己的时间表。因此，在接受外部任务时也会支持前瞻性记忆的内部驱动力。
  </p>
  <p class="graf graf--p">
   另一种策略是定期 “扫描”自己的意图。如果你每次开车经过商店都能想起买需要的东西，那你就不会忘了这件事了。
  </p>
  <p class="graf graf--p">
   事件和意图可以直接联系起来，或者恰恰相反，可以用最奇特的方式联系起来。比如，每个月的第一天检查上个月的账单；好天气会让人产生度假的想法。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff0000;">
    <strong class="markup--strong markup--p-strong">
     测试！快速回答 ——
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    Stephen LaBerge 提出的训练法中第2个示例的 “目标” 是什么？
   </span>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    红衣服女人
   </li>
   <li class="graf graf--li">
    数字9
   </li>
   <li class="graf graf--li">
    扫大街的人
   </li>
   <li class="graf graf--li">
    数字7
   </li>
   <li class="graf graf--li">
    白色西装的男人
   </li>
  </ul>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   制定规则，以扫描自己想要执行的任务。从一两个开始；选择一些事件，与你要扫描的意图
   <strong class="markup--strong markup--p-strong">
    产生回应关系
   </strong>
   。
  </p>
  <p class="graf graf--p">
   👉遵守这些规则，让它成为一种习惯。根据你的目标和需求，逐渐增加规则的数量。
  </p>
  <p class="graf graf--p">
   前文中讲述的助记法也可以用于前瞻性记忆。想象一个事件或一个场合，以便回想一个计划好的任务。用这个事件和任务来想象一个生动的情感画面。
  </p>
  <p class="graf graf--p">
   比如你准备购买ABC先生的演讲集，那么可以创造一个相应的事件 “路过书店门前”。想象一下，你路过最喜欢的书店，不巧碰到了拥堵，ABC先生本人制造了这场拥堵事件，他在马路中间签名售书导致了拥堵。
  </p>
  <p class="graf graf--p">
   最后，不忘事的最有效方法就是
   <strong class="markup--strong markup--p-strong">
    遵守规律和仪式
   </strong>
   。
  </p>
  <p class="graf graf--p">
   👉成熟稳重的组织和团队都需要确立很多规则，比如保密的艺术，它就遵循一系列规则：在与重要人员见面前、见面时和见面后分别应该做什么，怎样谈论你的工作、谈论哪些内容、如何回答棘手问题、如何摆脱跟踪监视 ……
  </p>
  <p class="graf graf--p">
   任何一个环节发生错误都可能是致命的。这有点像航空作业，唯一的区别是，
   <strong class="markup--strong markup--p-strong">
    行动者团队不储存规则清单的文件，因为任何一种储存都可能不安全，成员必须用脑子来记住
   </strong>
   。
  </p>
  <p class="graf graf--p">
   前文提到，前瞻性记忆之所以发生错误，主要是因为在重要环节出现了分心的情况。为了减少这类错误，请不要在分心后切换另一项任务。这时你需要暂停一下，在记忆中仔细想想目前做到哪一步了，然后创建一个新计划 —— 也就是说，将第一项任务延后，先做第二项，然后再回去继续第一项。
  </p>
  <p class="graf graf--p">
   由于任务没有完成，因此记忆会给它一个标记，提醒你继续去完成。想想
   <strong class="markup--strong markup--p-strong">
    泽伊加尔尼克效应
   </strong>
   ？
  </p>
  <p class="graf graf--p">
   有时前瞻性记忆会犯很奇怪的错误，比如，忘了自己是不是锁了门、是不是喂了猫、是不是拔了电源插头 …… 这是因为这些动作是习惯性的，大脑不会特意去记忆它们。如果您也有反复查看是否锁了门的毛病，请设置一个 *目标事件*：比如 离开家、或下了车。将一个事件与某个行动（锁门）联系起来。这样就可以帮助您记住行动本身。⚪️
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    —— 未完待续 ——
   </em>
  </p>
