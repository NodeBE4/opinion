  </p>
  <p class="graf graf--p">
   正如 Thomas Powers 所说的：“保守秘密的第一条规则就是，不要在纸面上留下任何东西：记录可能丢失或被盗，或者只是流入了错误的人手里；如果你真的想保守秘密，从一开始就不要把它写下来。”
  </p>
  <p class="graf graf--p">
   但是，对于纸来说，你可以烧掉；而对于计算机来说，很多时候你无法真正清除它。
  </p>
  <p class="graf graf--p">
   如今越来越多的人不再使用纸了。
  </p>
  <p class="graf graf--p">
   每家公司都在努力保护他们的数据，也在努力培训员工如何充分做到这一点。他们还会使用很多工具来防止可能的数据泄漏。
  </p>
  <p class="graf graf--p">
   但这是防不胜防的。
  </p>
  <p class="graf graf--p">
   这些信息有很多方法可以在互联网上泄露。
  </p>
  <p class="graf graf--p">
   有时机密信息会被故意泄露，有时会因人为错误而泄露，有时因技术错误而泄露。
  </p>
  <p class="graf graf--p">
   但是，正如您所知道的，“在网上发布的所有内容都会永远存在，即使您自己觉得已经删除了。”
  </p>
  <p class="graf graf--p">
   因此，将技术和人为错误的可能性结合起来时，结果就是机密信息可以轻松出现在 Google 上。
  </p>
  <p class="graf graf--p">
   我们曾经介绍过这个玩法，详见《高
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E9%AB%98%E7%BA%A7%E8%BF%90%E7%AE%97%E7%AC%A6%E8%BE%85%E5%8A%A9%E5%BC%80%E6%BA%90%E8%B0%83%E6%9F%A5%EF%BC%9A%E5%B7%A7%E7%94%A8%E6%90%9C%E7%B4%A2%E5%BC%95%E6%93%8E%E6%8C%96%E6%8E%98%E6%83%85%E6%8A%A5/" href="https://www.iyouport.org/%E9%AB%98%E7%BA%A7%E8%BF%90%E7%AE%97%E7%AC%A6%E8%BE%85%E5%8A%A9%E5%BC%80%E6%BA%90%E8%B0%83%E6%9F%A5%EF%BC%9A%E5%B7%A7%E7%94%A8%E6%90%9C%E7%B4%A2%E5%BC%95%E6%93%8E%E6%8C%96%E6%8E%98%E6%83%85%E6%8A%A5/" rel="noopener noreferrer" target="_blank">
    级运算符辅助开源调查：巧用搜索引擎挖掘情报 #OSINT
   </a>
   》
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    机密信息
   </strong>
  </p>
  <p class="graf graf--p">
   俗话说的没错，只要你搜得对，就没有找不到的东西。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5826 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/间谍.png?resize=1100%2C751&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/间谍.png?w=1104&amp;ssl=1 1104w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/间谍.png?resize=300%2C205&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/间谍.png?resize=768%2C525&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/间谍.png?resize=1024%2C699&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/间谍.png?resize=1100%2C751&amp;ssl=1 1100w" data-recalc-dims="1" height="751" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/间谍.png?resize=1100%2C751&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5826" data-recalc-dims="1" height="751" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/间谍.png?resize=1100%2C751&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/间谍.png?w=1104&amp;ssl=1 1104w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/间谍.png?resize=300%2C205&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/间谍.png?resize=768%2C525&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/间谍.png?resize=1024%2C699&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/间谍.png?resize=1100%2C751&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   你需要做的第一件事就是定位与搜索最相关的关键字。
  </p>
  <p class="graf graf--p">
   在本文的案例中您可以定位的关键字有很多，比如：机密；仅限内部使用；不分发；不得公开发行；classified; document is private, 等等。
  </p>
  <p class="graf graf--p">
   你想要找哪个国家的机密信息，就使用其官方语言搜索。
  </p>
  <p class="graf graf--p">
   在搜索机密数据时创建字符串非常简单，只需为该搜索生成一个最佳关键字列表。
  </p>
  <p class="graf graf--p">
   比如这样
   <strong class="markup--strong markup--p-strong">
    (“confidential” OR “internal use only” OR “not for distribution” OR “not for public distribution” OR “classified”)
   </strong>
  </p>
  <p class="graf graf--p">
   您可以添加当前年份结合更多运算符，如 intitle :，这应该在页面标题中找到2019的所有页面。
  </p>
  <p class="graf graf--p">
   比如这样
   <strong class="markup--strong markup--p-strong">
    (“confidential” OR “internal use only” OR “not for distribution” OR “not for public distribution” OR “classified”) intitle:2019
   </strong>
  </p>
  <p class="graf graf--p">
   如果你希望定位更多年份，可以添加更多 intitle：。
  </p>
  <p class="graf graf--p">
   这样
   <strong class="markup--strong markup--p-strong">
    (“confidential” OR “internal use only” OR “not for distribution” OR “not for public distribution” OR “classified”) (intitle:2018 OR intitle:2019)
   </strong>
  </p>
  <p class="graf graf--p">
   还可以将 inurl: 与 intitle: 结合使用。
  </p>
  <p class="graf graf--p">
   这样
   <strong class="markup--strong markup--p-strong">
    (“confidential” OR “internal use only” OR “not for distribution” OR “not for public distribution” OR “classified”) (intitle:2019 OR inurl:2019)
   </strong>
  </p>
  <p class="graf graf--p">
   还可以尝试定位文件类型。这就是 filetype: 派上用场的地方。
  </p>
  <p class="graf graf--p">
   这样
   <strong class="markup--strong markup--p-strong">
    filetype:pdf (“confidential” OR “internal use only” OR “not for distribution” OR “not for public distribution” OR “classified”)
   </strong>
  </p>
  <p class="graf graf--p">
   使用更为通用的关键字如“机密”时，将获得包含该关键字的大量结果。无疑，这里面会有很多不相关的结果，这就是为什么不推荐使用它。
  </p>
  <p class="graf graf--p">
   例如
   <strong class="markup--strong markup--p-strong">
    filetype:pdf (“internal use only” OR “not for distribution” OR “not for public distribution” OR “classified”)
   </strong>
  </p>
  <p class="graf graf--p">
   如果您想使用一些高级字符串，只需添加更多运算符或关键字。
  </p>
  <p class="graf graf--p">
   这样
   <strong class="markup--strong markup--p-strong">
    filetype:pdf (“internal use only” OR “not for distribution” OR “not for public distribution” OR “classified”) (sourcing AND recruitment)
   </strong>
  </p>
  <p class="graf graf--p">
   您还可以使用 site: 运算符来定位特定站点。
  </p>
  <p class="graf graf--p">
   这样
   <strong class="markup--strong markup--p-strong">
    site:com (“internal use only” OR “not for distribution” OR “not for public distribution” OR “classified”)
   </strong>
  </p>
  <p class="graf graf--p">
   这只定位.com 域名。例如，如果您想定位来自德国的域名，只需将 site：com 替换为 site：de。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    结论
   </strong>
  </p>
  <p class="graf graf--p">
   保守秘密的最好方法是假装没有秘密。
  </p>
  <p class="graf graf--p">
   不要添加以下任何相关的关键字：如 机密、限内部使用、不分发、不得用于公开发行等等一切。
  </p>
  <p class="graf graf--p">
   如果您必需在自己的文档或演示文稿中使用这些单词或短语，可以使用一个简单的技巧来保护这些文档。
  </p>
  <p class="graf graf--p">
   将文本“仅限内部使用”和类似的警告替换为该文本的图像。
  </p>
  <p class="graf graf--p">
   搜索引擎机器人无法对文件中包含的图像运行OCR（光学字符识别）。
  </p>
  <p class="graf graf--p">
   而且因为他们无法阅读照片中的文字，搜索意外泄露机密信息的人不会在网上找到你的文件。
  </p>
  <p class="graf graf--p">
   反过来看，如果您希望揭露的对手了解上述策略，或者压根儿非常落后数字化程度很低，那么您只能期待
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%E5%85%B3%E4%BA%8E%E9%80%8F%E6%98%8E%E5%BA%A6%E9%9D%A9%E5%91%BD/" href="https://www.iyouport.org/category/%E5%85%B3%E4%BA%8E%E9%80%8F%E6%98%8E%E5%BA%A6%E9%9D%A9%E5%91%BD/" rel="noopener noreferrer" target="_blank">
    透明度革命
   </a>
   后继有人了。⚪️
  </p>
