  </p>
  <p class="graf graf--p">
   一名独立记者发誓要保护他的消息来源 —— 旧金山警方突击搜查他的家和办公室，同时将他戴上手铐关押了数小时，作为所谓的刑事调查的一部分。
  </p>
  <p class="graf graf--p">
   Bryan Carmody 告诉“洛杉矶时报”，警方周五敲门，⚠️并没收了数十件他的个人物品，包括笔记本电脑、手机、台式电脑、移动硬盘和相机。
  </p>
  <p class="graf graf--p">
   请注意，被警方没收的电子设备即便此后归还也需要格外警惕，因为很可能有危险；如果您舍不得丢弃设备，请找到这篇文章中介绍的方法获得帮助《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%88%91%e7%9a%84%e8%ae%be%e5%a4%87%e5%a5%bd%e5%83%8f%e8%a2%ab%e9%bb%91%e4%ba%86%e6%80%8e%e4%b9%88%e5%8a%9e%ef%bc%9f%e2%80%8a-%e2%80%8a-%e7%bb%99%e6%95%8f%e6%84%9f%e4%ba%ba%e5%a3%ab/" href="https://www.iyouport.org/%e6%88%91%e7%9a%84%e8%ae%be%e5%a4%87%e5%a5%bd%e5%83%8f%e8%a2%ab%e9%bb%91%e4%ba%86%e6%80%8e%e4%b9%88%e5%8a%9e%ef%bc%9f%e2%80%8a-%e2%80%8a-%e7%bb%99%e6%95%8f%e6%84%9f%e4%ba%ba%e5%a3%ab/" rel="noopener noreferrer" target="_blank">
    我的设备好像被黑了怎么办？ — — 给敏感人士和小白朋友的数字急救包
   </a>
   》
  </p>
  <p class="graf graf--p">
   一名法官签署了搜查令，上面写着警方以调查“被盗或被盗”的财产为由闯入他的家。
  </p>
  <p class="graf graf--p">
   当局说，这次突袭是为了调查“泄密者” —— 关于是谁泄露了一份关于2月22日旧金山公共辩护人 Jeff Adachi 去世的机密警方报告。
  </p>
  <p class="graf graf--p">
   Carmody 说，警方调查人员几个星期前曾要求他说出向他提供信息的人是谁。这位记者说他礼貌地拒绝了。
  </p>
  <p class="graf graf--p">
   时报报道，当他被抓捕时，警察获得了第二张手令，即 可以搜查他的新闻编辑室。警方在那里查获了一个拇指驱动器、CD、并在保险柜内找到了被泄露的死亡报告。（他们甚至打开了保险柜！）
  </p>
  <p class="graf graf--p">
   49岁的 Carmody 说，他没有与任何人分享他的消息来源的名字，并且
   <strong class="markup--strong markup--p-strong">
    该文件上的任何标记都不能追溯到提供该文件的人的真实身份
   </strong>
   —— 做的非常漂亮。
  </p>
  <p class="graf graf--p">
   对于 Carmody 和他的律师来说，警方的袭击严重侵犯了专业记者的工作。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这是为了恐吓，”他的律师 Thomas Burke 说。 “这基本上是扫荡新闻编辑室。”
  </p>
  <p class="graf graf--p">
   与此同时，这一事件为 Adachi 的意外死亡之真相提供了新的转折 ——
   <strong class="markup--strong markup--p-strong">
    死者 Adachi 留下了支持民权的遗产。
   </strong>
  </p>
  <p class="graf graf--p">
   初步报道说，这位59岁的当选公设辩护人“在旅行中突发心脏病而死亡”。
  </p>
  <p class="graf graf--p">
   但是，Carmody 说，在试图找出 Adachi 死亡的确切信息时，开始出现了很难以确认的色情细节。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “泄密发生在所有地方，”他回忆说。他最终获得​​了一份事故报告，详细说明了 Adachi 死前的最后时刻。
  </p>
  <p class="graf graf--p">
   旧金山纪事报还获得了该报告的副本，但未获得 Carmody 持有的副本。
  </p>
  <p class="graf graf--p">
   据旧金山 KGO-TV 报道，这份文件详细说明，在 Adachi 去世前不久，他在与一位名叫“卡特琳娜”的女士共进晚餐，这名女士不是他的妻子，然后 Adachi 回到他安排在周末使用的公寓。
  </p>
  <p class="graf graf--p">
   就是这名女士打电话给911寻求紧急医疗救助的，而 Adachi 被送往医院时已经去世了。
  </p>
  <p class="graf graf--p">
   旧金山纪事报报道，当晚晚些时候，警察前往公寓，发现了“酒精、大麻注入的胶囊和注射器，据信已被医护人员使用过”。公寓的照片由 KTVU-TV 和其他新闻媒体在线传播。
  </p>
  <p class="graf graf--p">
   Carmody 告诉“纪事报”，他将 Adachi 的新闻给了三家电视台。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    市政府官员批评警方没能阻止机密报告的细节最终成为头条新闻
   </strong>
   。
  </p>
  <p class="graf graf--p">
   然后就是警方对该报告的泄漏进行了内部调查，导致周五袭击了 Carmody 的家。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “旧金山市的领导人要求对这一泄漏进行彻底的调查，这一行动是调查潜在的妨碍司法案件以及非法分发机密警察材料的过程中的一部分”，警方发言人 David Stevenson 在致纽约时报的一份声明中表示。
  </p>
  <p class="graf graf--p">
   警方用来突袭 Carmody 住所的宣誓书
   <strong class="markup--strong markup--p-strong">
    是在密封状态下提交的
   </strong>
   ，因此不清楚调查人员是怎么对法官讲述的理由。
  </p>
  <p class="graf graf--p">
   Carmody 坚持要保护信息来源的身份。而且他发誓他从来没有为泄密信息付钱。 他告诉“纽约时报”，“没有任何支付，甚至连一杯咖啡都没有。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    注：本文提醒几件事 —— 关于透明度革命的须知
   </strong>
   ，事实上我们已经在很久前的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/?s=%E9%80%8F%E6%98%8E%E5%BA%A6%E9%9D%A9%E5%91%BD" href="https://www.iyouport.org/?s=%E9%80%8F%E6%98%8E%E5%BA%A6%E9%9D%A9%E5%91%BD" rel="noopener noreferrer" target="_blank">
    同名系列文章
   </a>
   和“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/?s=%E4%B8%BA%E8%87%AA%E7%94%B1%E8%80%8C%E6%88%98" href="https://www.iyouport.org/?s=%E4%B8%BA%E8%87%AA%E7%94%B1%E8%80%8C%E6%88%98" rel="noopener noreferrer" target="_blank">
    为自由而战
   </a>
   ”系列中有过详细解释。
  </p>
  <p class="graf graf--p">
   1、
   <strong class="markup--strong markup--p-strong">
    迅速有效地分发
   </strong>
   。美国史上最著名的举报人 — 五角大楼文件提供者 Daniel Ellsberg 最早
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E9%80%8F%E6%98%8E%E5%BA%A6%E9%9D%A9%E5%91%BD%EF%BC%9A%E4%BD%A0%E5%BA%94%E8%AF%A5%E7%9F%A5%E9%81%93%E7%9A%84%E6%8A%80%E6%9C%AF%E3%80%81%E7%BB%8F%E9%AA%8C%E5%92%8C%E6%95%99%E8%AE%AD/" href="https://www.iyouport.org/%E9%80%8F%E6%98%8E%E5%BA%A6%E9%9D%A9%E5%91%BD%EF%BC%9A%E4%BD%A0%E5%BA%94%E8%AF%A5%E7%9F%A5%E9%81%93%E7%9A%84%E6%8A%80%E6%9C%AF%E3%80%81%E7%BB%8F%E9%AA%8C%E5%92%8C%E6%95%99%E8%AE%AD/" rel="noopener noreferrer" target="_blank">
    采取了这一措施
   </a>
   。
  </p>
  <p class="graf graf--p">
   Ellsberg 很聪明，他预料到了政府会封锁新闻，他提前准备了很多副本，迅速分发给各大媒体组织。但是政府不断围堵媒体，第一个就压制了华盛顿邮报的刊发。华盛顿邮报怂了。
  </p>
  <p class="graf graf--p">
   而 Ellsberg 没有停止，他继续将副本发给波士顿环球报、洛杉矶时报、基督教科学箴言报、圣路易斯邮报……等等媒体，这种无止境的打地鼠游戏直接令白宫陷入了混乱，最终当局放弃了对出版的阻止。
  </p>
  <p class="graf graf--p">
   这也是为什么 Wikileaks 当年要和大型媒体合作（虽然那些大型媒体在赚够了钱之后就站到了当局一边抹黑 Wikileaks），因为这样才能有利于信息被迅速传播 —— 抢在被当局封堵之前。想想看，如何是一家生存艰难的独立媒体，即便报道了这一重要消息，就如本文中 Carmody 的状况，只要警方突袭大扫荡，一切都会结束。
  </p>
  <p class="graf graf--p">
   2、
   <strong class="markup--strong markup--p-strong">
    不论发生任何事，切勿透露泄密者的任何信息
   </strong>
   。这点非常关键，而不仅仅是一两个人的人身安全问题、以及记者的天职问题，更有，万一举报人被你当局抓住，将产生严重的寒蝉效应，甚至断送透明度革命的前景。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    更直接的是断送了记者本身的信誉
   </strong>
   —— 如果你曾经将你的信息来源“送入”监狱，今后就没有任何举报人会选择与你合作了。
  </p>
  <p class="graf graf--p">
   Wikileaks 是最可靠的渠道，不仅因为他们具备比洋葱更安全的投递渠道，并且，他们从来没有泄漏过任何举报人的任何信息。即便是在遭受了史上最大规模的人格暗杀之后。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%ba%e6%a0%bc%e6%9a%97%e6%9d%80%e5%92%8c%e6%8a%b9%e9%bb%91%e8%bf%90%e5%8a%a8%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%8f%97%e7%9b%8a%ef%bc%9f/" href="https://www.iyouport.org/%e4%ba%ba%e6%a0%bc%e6%9a%97%e6%9d%80%e5%92%8c%e6%8a%b9%e9%bb%91%e8%bf%90%e5%8a%a8%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%8f%97%e7%9b%8a%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    在这里看到人格暗杀
   </a>
   有多可怕。
  </p>
  <p class="graf graf--p">
   记者始终在明，举报人必须在暗；于是记者应该有一套保护自己的有效方法，这取决于您对风险的准确评估。
  </p>
  <p class="graf graf--p">
   3、
   <strong class="markup--strong markup--p-strong">
    被警方没收的电子设备不可直接使用
   </strong>
   。如上所述（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%88%91%e7%9a%84%e8%ae%be%e5%a4%87%e5%a5%bd%e5%83%8f%e8%a2%ab%e9%bb%91%e4%ba%86%e6%80%8e%e4%b9%88%e5%8a%9e%ef%bc%9f%e2%80%8a-%e2%80%8a-%e7%bb%99%e6%95%8f%e6%84%9f%e4%ba%ba%e5%a3%ab/" href="https://www.iyouport.org/%e6%88%91%e7%9a%84%e8%ae%be%e5%a4%87%e5%a5%bd%e5%83%8f%e8%a2%ab%e9%bb%91%e4%ba%86%e6%80%8e%e4%b9%88%e5%8a%9e%ef%bc%9f%e2%80%8a-%e2%80%8a-%e7%bb%99%e6%95%8f%e6%84%9f%e4%ba%ba%e5%a3%ab/" rel="noopener noreferrer" target="_blank">
    文章
   </a>
   ），这种情况也有解决方案，并不需要您自己掌握很多技术技能，而是需要您知道该与谁合作以减轻或消除危险。
  </p>
