   这是一种美国版的“信用评分”。
  </div>
  <div>
   <p class="graf graf--p graf-after--p" id="a5dc">
    中国的信用评分什么样众所周知了，这篇报道描述了美国版的，看起来同样离谱，重要的是评判标准离谱。中国版的信用评分决定你能不能买火车票、以及你的孩子能读什么样的学校；美国版的信用评分决定了谁住在哪里、以及他们可以拥有什么样的汽车，甚至影响就业机会。
   </p>
   <p class="graf graf--p graf-after--p" id="c8dc">
    美国版版的“公民得分”是由私人团体编制的，他们可以获得信息，人们在放弃隐私时就已经别无选择了。
   </p>
   <p class="graf graf--p graf-after--p" id="d151">
    美国公民自由联盟目前正在起诉波士顿警察局，就是因为这个极为特别的“公民得分”。
    <strong class="markup--strong markup--p-strong">
     BPD 掌握着一个黑名单叫“帮派数据库”，他们可以随便把任何人的名字添加进去，或者抹掉。公民积极做的事 — 甚至他们被迫做的事- 都可以令他们被放入那个被严格监视的列表中
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="67e4">
    和中国的信用评分一样，美国的这个黑名单也是积分制的。如果一个人被评估为六分或更多，就会被指定为“与帮派相关”; 如果被评估为十分或更多分，则被指定为“帮派成员”。
   </p>
   <p class="graf graf--p graf-after--p" id="4be4">
    这个积分系统可以指定任何为“帮派成员”，而不需要判断此人是否从事过暴力或犯罪活动。
   </p>
   <p class="graf graf--p graf-after--p" id="e429">
    如果这个人是与帮派有关的暴力行为的受害者，他甚至会被评估为8分！是的你没听错，
    <strong class="markup--strong markup--p-strong">
     如果你被打了，你就会成为被告。如果你被没长眼的流弹集中了，你就会神奇地升级为黑帮成员
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="13ee">
    还有更离谱的呢。仅仅是照片也能定罪，如果你与被指控的帮派成员或同伙是同学、同事、邻居或家庭成员，你也会被认为存在相关性。
   </p>
   <p class="graf graf--p graf-after--p" id="b763">
    如果警方报告称此人 “与帮派成员一起”吃饭、交流或以其他方式联系“; 包括做手势，如果被认为是一个帮派标志的话你就麻烦了; 并且穿着警方认为与帮派有关的衣服，也算，关键字是“警方认为是”。
   </p>
   <p class="graf graf--p graf-after--p" id="b397">
    如果你在早上起来时穿“错了”衣服，就足以让你的生活完全颠倒。
    <strong class="markup--strong markup--p-strong">
     该诉讼指出，BPD 已经指定像芝加哥公牛队的帽子和耐克运动鞋这样的衣着都算“帮派服装”
    </strong>
    。官员经常在公园、学校和娱乐中心监视年轻人 — 他们肯定会看到人们穿着 NBA 品牌的衣服和耐克鞋。而且，和穿着这样的衣服的人再在一起，你也会被列为“同伙”。
   </p>
   <p class="graf graf--p graf-after--p" id="f09b">
    更要命的是，ICE 可以访问 BPD 的数据库，并根据这个极其错误的信息做出驱逐决定。简单说就是，哪天你穿“错了”衣服，会导致你被驱逐出境。
   </p>
   <p class="graf graf--p graf-after--p" id="4d1c">
    下面是在纽约发生的事。
   </p>
   <p class="graf graf--p graf-after--p" id="d470">
    — –
   </p>
   <p class="graf graf--p graf-after--p" id="cefa">
    那天，Keith Shenery 在 Harlem 公共住宅区的院子里和朋友们玩耍，这时警方注意到他从裤子里掏出了个小袋子。警察上前询问时，Shenery 对他们说‘就是一些大麻’。之后警察从他身上搜出了一小袋大麻和一个他祖母给他的礼物，一把折叠刀。
   </p>
   <p class="graf graf--p graf-after--p" id="2b40">
    Shenery, 21岁, 因为非法持有大麻和“非法持有武器”而被逮捕（这可是重罪）。
   </p>
   <p class="graf graf--p graf-after--p" id="1ab7">
    这是一个很不同寻常的严重起诉。公诉人要求他支付一笔 $10,000 的巨额罚款. 他们声称 Shenery 是一个“在册“的帮派成员。
   </p>
   <p class="graf graf--p graf-after--p" id="c3f7">
    注：非法持有武器(重罪) — — 即
    <a class="markup--anchor markup--p-anchor" data-href="https://www.legalmatch.com/law-library/article/felon-in-possession-of-a-weapon-laws.html" href="https://www.legalmatch.com/law-library/article/felon-in-possession-of-a-weapon-laws.html" rel="noopener nofollow noreferrer" target="_blank">
     Felon in Possession of a Weapon
    </a>
    ，一般只适用于被判过重罪的人持有枪支等危险武器的情况。
   </p>
   <p class="graf graf--p graf-after--p" id="7200">
    Shenery 从青年时期开始算，只有三次
    <strong class="markup--strong markup--p-strong">
     非暴力
    </strong>
    轻罪的记录。这次的“罪行”本来也应是那种只要自己承认，当晚就可以被释放的轻罪。Shenery 不明白为什么检方认定他是“帮派成员”而且强烈反对他的上诉。他的案子在法庭纠纷将近两年了，公诉人不停的和法官说 “Shenery 是帮派成员”。但是根据法院的文档，检方并没有提供任何证据来证明 Shenery 是属于 Harlem’s Cash Money Boys 这个他们认定的“帮派”，这个帮派是 1990 年发迹于 Lenox 地区的暴力贩毒团体。
   </p>
   <p class="graf graf--p graf-after--p" id="0193">
    自 2017 年4月21日他被逮捕这一年多的时间里，Shenery 发现，检方之所以认定他是“帮派成员”和他们这样起诉的根据，看来是因为 Shenery 被列入了一个纽约警署的数据库，这个数据库里记录了 42,000 多个纽约市民的资料，这些人全被纽约警署认为是“帮派成员”。
   </p>
   <p class="graf graf--p graf-after--p" id="4b15">
    正如 Intercept
    <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2018/06/11/new-york-gang-database-expanded-by-70-percent-under-mayor-bill-de-blasio/" href="https://theintercept.com/2018/06/11/new-york-gang-database-expanded-by-70-percent-under-mayor-bill-de-blasio/" rel="noopener nofollow noreferrer" target="_blank">
     曾经报道过的
    </a>
    , 虽然近年来和帮派有关的犯罪降到了历史上的最低点，纽约警署的“帮派数据库”却大规模的扩大了。
   </p>
   <p class="graf graf--p graf-after--p" id="e44c">
    这数据库里的秘密名单只对公诉人开放，而对名单里的人却是保密的。通常这些人只有当被警方逮捕后、遭到莫名其妙的严重起诉、或者高额罚金时，才意识到自己被警方标记在这个名单里了。这个数据库曾经因其专横、标准过于宽泛、和充满歧视性，而受到广泛的批评，根本没有一个明确的程序来发现和质疑一个人和帮派的关系。
    <strong class="markup--strong markup--p-strong">
     数据库里如 Shenery 这样的年轻非裔和拉美裔人群占据了压倒性的大部分
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="ca20">
    很多纽约当地组织要求 NYPD 对这个数据库更加透明化，去年其中一个组织 Legal Aid Society 建立了一个
    <a class="markup--anchor markup--p-anchor" data-href="https://legalaidfoil.backspace.com/" href="https://legalaidfoil.backspace.com/" rel="noopener nofollow noreferrer" target="_blank">
     网站
    </a>
    ，来帮助纽约市民申请索取公开记录的要求，以了解他们是不是在这个数据库的名单里。目前有 300 多人提交了申请，但全被纽约警方拒绝了。
   </p>
   <p class="graf graf--p graf-after--p" id="0a58">
    因为公诉人在法庭上认定他是帮派成员这件事，使 Shener y认识到了这个数据库个存在。
   </p>
   <p class="graf graf--p graf-after--p" id="acfe">
    去年7月 Shenery 提交了一个基于信息自由法案 (Freedom of Information Law) 的申请，希望了解是警方以什么理由把他标记到了这个数据库的名单里。但是，申请不到一天就被 NYPD 拒绝了。在得到 NYPD 回复说警方找到了关于他的相应记录但是拒绝提供给他后，11月份 Shenery 开始了上诉和控告。
   </p>
   <p class="graf graf--p graf-after--p" id="43fc">
    Shenery 拒绝本文的采访。对帮派数据库和 Shenery 的指控 NYPD 的发言人没有回复 The Intercept 的一系列问题，但是发来一封邮件说“对判断某个人是否属于一个已知的犯罪团体，坚持着一个全国最严厉的判断标准“。
   </p>
   <p class="graf graf--p graf-after--p" id="3808">
    一个曼哈顿区检察院办公室的发言人因 Shenery 的“犯罪案件”还在进行中而拒绝发表评论。但是提供了法庭档案，其中包括检方以 Shenery 的过往记录为由，要求罚款的申请。理由还包括曾经缺席法庭审判和一个犯罪公正机构 （Criminal Justice Agency）的建议。 Criminal Justice Agency 是一个独立的城市机构，评估一个嫌疑人是否应被释放。
   </p>
   <p class="graf graf--p graf-after--p" id="a053">
    发言人还说，检察院没有直接连入 NYPD 帮派数据库的权限并且写到，“我们的公诉人不会是引用 NYPD 的帮派数据库名单为理由，此外任何对被告是曼哈顿匪帮的指控都是基于我们办公室的独立分析，其中包括我们独立收集的情报。检方声称被告是“在册“帮派成员的依据是来自社区成员的信息和其他司法部门，包括我们办公室在内的独立收集的情报……“。
   </p>
   <p class="graf graf--p graf-after--p" id="8b55">
    律师辩论说没有提供证据就指控某人是帮派成员，将是对被告人司法公正权益的严重冲击。
   </p>
   <p class="graf graf--p graf-after--p" id="40b5">
    来自 Legal Aid Community Justice Unit 代表 Shenery 对抗 NYPD 的主管律师 Anthony Posada 说， “在法庭上仅仅因为一个标签就定罪，我们认为人们是在
    <strong class="markup--strong markup--p-strong">
     被犯罪。
    </strong>
    在法庭中如果没有确凿证据和所有合理怀疑被排除前，一个人应该假定是无辜的。而
    <strong class="markup--strong markup--p-strong">
     现在发生的却是区检查院依据帮派数据库给人贴标签，正在作出带有偏见的结论
    </strong>
    “。
   </p>
   <h4 class="graf graf--h4 graf-after--p" id="7c5d">
    当黑人吸大麻时
   </h4>
   <p class="graf graf--p graf-after--h4" id="9c52">
    虽然纽约警方对他们如何使用所谓的“犯罪集团数据库“透露的很少，因为这个数据库只有他们内部了解，但是众所周知的是，他们会和公诉人和其他司法部门分享被怀疑和帮派有关的人的信息。同样一个清晰的事实是，一但被定义为帮派成员，即使只是依据莫须有的证据并且这些所谓证据不对被告和其律师公开，将会对被告人在法庭上的命运产生极大的不利影响。虽然和帮派有关本身并不构成犯罪，但是公诉人通常用这个理由来作为他们的支持点。
   </p>
   <p class="graf graf--p graf-after--p" id="9547">
    这就是发生在 Shenery 身上的事。
   </p>
   <p class="graf graf--p graf-after--p" id="1821">
    今年早些时候，曼哈顿区检察院发表
    <a class="markup--anchor markup--p-anchor" data-href="https://www.manhattanda.org/tomorrow-d-a-vance-ends-prosecution-of-marijuana-possession-and-smoking-cases/" href="https://www.manhattanda.org/tomorrow-d-a-vance-ends-prosecution-of-marijuana-possession-and-smoking-cases/" rel="noopener nofollow noreferrer" target="_blank">
     公告
    </a>
    说，不在对持有少量大麻的行为进行公诉，而且呼吁立法者规范化和合法化大麻的使用并且引证到“执法中存在的不可容忍的种族差异缺少道德辩护“。
   </p>
   <p class="graf graf--p graf-after--p" id="a81c">
    Shenery 是在政策改变前被逮捕的，但是在一个使用大麻对大部分人来说已经是合法的城市里，
    <strong class="markup--strong markup--p-strong">
     他的形象符合那些最容易受到起诉的纽约居民 — — 年轻、黑人、来自低收入社区
    </strong>
    。在检察院发表公告之前，黑人纽约居民因少量持有大麻而被逮捕的数量是白人纽约居民的 8 倍。在曼哈顿则是
    <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2018/05/13/nyregion/marijuana-arrests-nyc-race.html" href="https://www.nytimes.com/2018/05/13/nyregion/marijuana-arrests-nyc-race.html" rel="noopener nofollow noreferrer" target="_blank">
     15倍
    </a>
    。
   </p>
   <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="18aa">
    “重力刀”，就是公诉者声称在 Shenery 身上找到的折叠刀，在纽约市也是有
    <a class="markup--anchor markup--p-anchor" data-href="https://www.thedailybeast.com/new-york-just-saved-a-law-used-to-jail-thousands-of-minorities" href="https://www.thedailybeast.com/new-york-just-saved-a-law-used-to-jail-thousands-of-minorities" rel="noopener nofollow noreferrer" target="_blank">
     争议
    </a>
    的，
    <strong class="markup--strong markup--p-strong">
     批评者认为折叠刀的非法化导致了上千个工人阶级被逮捕，其中大部分人是有色人种
    </strong>
    。对重力刀的禁止法案最初只是希望针对危险的弹簧刀，后来甚至对最常见的口袋刀也适用，比如手动折叠刀。您一定知道中国对刀具的管制吧？嗯，在纽约也是如此。
   </p>
   <p class="graf graf--p graf-after--p" id="6ceb">
    检察院告诉 The Intercept 说，如果当事人在6个月内没有被再逮捕，对于工作需要携带这种道具的人 DA 不给予起诉，或者给一个破坏公共秩序的警告。但是律师说，工人因为刀具被
    <a class="markup--anchor markup--p-anchor" data-href="https://theappeal.org/manhattan-d-a-s-office-still-prosecuting-people-for-the-knives-they-carry-for-work/" href="https://theappeal.org/manhattan-d-a-s-office-still-prosecuting-people-for-the-knives-they-carry-for-work/" rel="noopener nofollow noreferrer" target="_blank">
     逮捕并被起诉的事件还是在发生
    </a>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="386b">
    虽然 Shenery 的案子突出了纽约市警方与少数族裔之间的棘手问题，但要不是被定性为帮会成员，Shenery 也不太会受到重罪起诉和过分的巨额罚款。“对纽约市的几乎任何一个人，这行为顶多是以轻罪被逮捕“， Jane White 来自 Legal Aid 的 Shenery 代理律师如是说。“他们对大部分被告都不这么干，只有对那些被他们盯上的人才这么做，比如针对那些他们希望得到更多信息的人，或者那些他们认为是所谓的“重点人员”的人。
   </p>
   <p class="graf graf--p graf-after--p" id="6386">
    对 Shenery 提出的索取数据库记录的申请， NYPD 声称没有在不暴露对帮会成员进行“非常规”调查方法的情况下来公开关于 Shenery 的记录的方法。但 Legal Aid 的律师回击到 NYPD 决定谁是帮会成员的那些所谓方法早就被公共讨论了，并且
    <strong class="markup--strong markup--p-strong">
     它们充满了严重的问题
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="1462">
    6月在城市议会的听证中， NYPD 警长 Dermot Shea 证言，如果个人“承认”属于帮派或者有“两个独立的可信来源”证明这个人属于帮派，那这个人就会被记录到数据库里。在缺少鉴别条件时，NYPD 可能也会将个人加入这个列表，如果这个人符合一个
    <strong class="markup--strong markup--p-strong">
     宽泛标准列表
    </strong>
    中的至少两个条件，比如出现在“已知”的帮会聚集地点、社交媒体内容、疤痕、文身、使用帮会“符号”和“颜色”…… 一份 The Intercept 在6月
    <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/4499937-NYPD-Gang-Presentations.html#document/p40by" href="https://www.documentcloud.org/documents/4499937-NYPD-Gang-Presentations.html#document/p40by" rel="noopener nofollow noreferrer" target="_blank">
     发表
    </a>
    的文档中列出了 NYPD 认为的所谓帮派颜色：黑、金、黄、红、紫、绿、蓝、白、棕、卡其色、灰、橙色和柠檬绿。（没法穿衣服了已经）
   </p>

    被列入名单不需要有于犯罪有关的证据，但 Shea 声称当局“引入了防错机制”来确保进入名单的人有“证据支持”。
   </p>

   <p class="graf graf--p graf-after--p" id="97ba">

   </p>
   <p class="graf graf--p graf-after--p" id="141d">
    因为 NYPD 不答应 Shenery 的 FOIL(Freedom of Information Law) 申请，他和他的律师不知道是以什么标准将他列入名单的。“检察院 (DA) 和纽约警属 (NYPD) 之间公开地分享这些信息，可就是不向我们公开，这十分的荒诞”，Shenery 的律师 White 说。DA 发言人告诉 The Intercept 说，“对于向犯罪被告公开合法的相关信息的规定，办公厅一贯合法和专业地履行甚至超额履行应尽的义务，包括在 Shenery 的这个案件里也一样”。
   </p>
   <p class="graf graf--p graf-after--p" id="21a4">

   <h4 class="graf graf--h4 graf--startsWithDoubleQuote graf-after--p" id="0244">
    “精准执法“
   </h4>
   <p class="graf graf--p graf-after--h4" id="8a37">
    检方和警方常常辩解说他们的执法手段如
    <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2018/06/07/rico-gang-prosecution-nyc/" href="https://theintercept.com/2018/06/07/rico-gang-prosecution-nyc/" rel="noopener nofollow noreferrer" target="_blank">
     外科手术般精确
    </a>
    — 即便是这些手段常导致的是
    <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2016/07/12/in-new-york-gang-sweeps-prosecutors-use-conspiracy-laws-to-score-easy-convictions/" href="https://theintercept.com/2016/07/12/in-new-york-gang-sweeps-prosecutors-use-conspiracy-laws-to-score-easy-convictions/" rel="noopener nofollow noreferrer" target="_blank">
     大规模搜查和起诉
    </a>
    。在执法部门的字典里，对帮派的执法和公诉是 “情报驱动的” 和“主动积极的”， 但实际的意思是“以对帮派的执法为理由，一个人即使没有、甚至没有意图参与任何严重犯罪或者帮派犯罪的情况下，也可以受到执法部门的搜查。
   </p>
   <p class="graf graf--p graf-after--p" id="ff07">
    The Intercept 得到了一系列来自曼哈顿检察院的培训用文件，这些文件里展示了帮派标签是如何标记的，对于一个因微小违规和警察偶尔打过交道的人，一个未经证实的指控可以诱发一系列的后果并强化指控。在一个犯罪战略单位 (Crime Strategies Unit) 的 PPT 和一份关于
    <a class="markup--anchor markup--p-anchor" data-href="https://www.courtinnovation.org/sites/default/files/documents/IDPM_Research_Report_FINAL.PDF" href="https://www.courtinnovation.org/sites/default/files/documents/IDPM_Research_Report_FINAL.PDF" rel="noopener nofollow noreferrer" target="_blank">
     创新调查的报告里
    </a>
    ,检方展示了在司法过程中，当逮捕令在警方和检方分享时，一个人同时会也被标记在一系列执法部门名单里,其中包括帮派数据库。在标题为“案件强化 (case enhancement)”的幻灯片里，还列出了
    <strong class="markup--strong markup--p-strong">
     诸如嫌疑人的社交媒体或昵称的信息。
    </strong>
    检察院发言人称,“案件强化(case enhancement)”可以引导潜在的调查步骤，发掘更多信息和帮助量刑。
   </p>
   <p class="graf graf--p graf-after--p" id="b4bd">
    Shenery 的案件就是这种执法是如何执行的一本教科书。
   </p>
   <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="f7f3">

    <strong class="markup--strong markup--p-strong">
     精准执法，
    </strong>
    他们加重了指控，以一把手动折叠刀附加上了非法持有武器(重罪)这个罪名”。
   </p>
   <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="b3cc">
    “
    <strong class="markup--strong markup--p-strong">
     [帮派标签] 在法庭上被武器化了，区检察院可以用它来给案件施加更多压力
    </strong>
    ”，他补充道。

    <a class="markup--anchor markup--p-anchor" data-href="https://ccrjustice.org/home/press-center/press-releases/civil-rights-groups-sue-nypd-over-failure-disclose-information-gang" href="https://ccrjustice.org/home/press-center/press-releases/civil-rights-groups-sue-nypd-over-failure-disclose-information-gang" rel="noopener nofollow noreferrer" target="_blank">

   <p class="graf graf--p graf-after--p" id="d182">

     NYPD 的关于帮派的执法手段是一种暗箱操作的截查 (stop and frisk ) 行为，这种行为在过去几年中把上千个纽约市的年轻有色人种置于警方监控、骚扰甚至更糟糕的情况下
    </strong>

    <a class="markup--anchor markup--p-anchor" data-href="https://www.naacpldf.org/files/about-us/NYDA%20FOIL%20Request.pdf" href="https://www.naacpldf.org/files/about-us/NYDA%20FOIL%20Request.pdf" rel="noopener nofollow noreferrer" target="_blank">

    <a class="markup--anchor markup--p-anchor" data-href="https://web.archive.org/web/20181021040545/https:/www.naacpldf.org/files/case_issue/NYC%20DOE%20Gang%20Policing%20FOIL%20-%20FINAL.pdf" href="https://web.archive.org/web/20181021040545/https:/www.naacpldf.org/files/case_issue/NYC%20DOE%20Gang%20Policing%20FOIL%20-%20FINAL.pdf" rel="noopener nofollow noreferrer" target="_blank">

    </a>

   <p class="graf graf--p graf-after--p" id="44ab">
    今年早些时候，这个帮派数据库在纽约市听证会受到了高度的关注调查。在听证期间 NYPD 对 The Intercept 的关于这个数据库
    <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2018/06/14/nypd-gang-database-city-council-dermot-shea/" href="https://theintercept.com/2018/06/14/nypd-gang-database-city-council-dermot-shea/" rel="noopener nofollow noreferrer" target="_blank">

    <a class="markup--anchor markup--p-anchor" data-href="https://www.buzzfeednews.com/article/kendalltaggart/nypd-inspector-general-discipline-false-statements" href="https://www.buzzfeednews.com/article/kendalltaggart/nypd-inspector-general-discipline-false-statements" rel="noopener nofollow noreferrer" target="_blank">

    </a>

   <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="04d7">

    <strong class="markup--strong markup--p-strong">
     公众除了起诉，甚至没有别的办法可以阻止他们
    </strong>
    ”。
   </p>
   <p class="graf graf--p graf-after--p" id="95ef">
    Trujillo 提到这个帮派数据库所建立的充满歧视性且不受约束的执法方式不是新鲜事，但 NYPD 对依赖数据驱动，积极主动的执法技术日益增长的强调是史无前例的。
   </p>
   <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="c47b">
    “人们直接的感到警察会怎么用这个帮派标签：一个警察可以直接给你贴上帮派标签，然后公诉方可以在法庭上对你做他们想要的任何事。从人们的经验和历代记忆里知道对有色人种的执法和公诉是什么样的”，Trujillo 说.，“我认为人们不知道的是，警察把这种执法制度化到了什么程度、以及警方被允许应用的是什么样的技术。人们没有意识到有多少基础设施，多少投资投入，到了这种将来的执法方式里。你问100个纽约人，有99个不知道什么叫
    <strong class="markup--strong markup--p-strong">
     预防性执法
    </strong>
    ” — — 就是少数派报告中的那种东西。
   </p>
   <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="d0e7">
    “这个不一定和公众的无知有关，而更多的是 NYPD 的不透明和暗中允许 NYPD 这种执法的实行有关”。 Trujillo 补充道“我们现在刚知道事情的一角，就很令人担忧了。
    <strong class="markup--strong markup--p-strong">
     你能想像到这种事的全景吗？警方在暗地里还在做些什么？
    </strong>
    ”
   </p>
   <p class="graf graf--p graf-after--p" id="fdd9">
    尽管关于这个数据库和它的用法还是秘密，但人们已经感受到了它所造成的冲击。
   </p>
   <p class="graf graf--p graf-after--p" id="b384">
    两年来的案件纠纷让 Shenery 身心疲惫，一下子发生了那么多事，他甚至只想干脆接受重罪的判决，让这一切快点结束。最后他还是被判处有罪，不过达成了一个“协议”：如果他在之后的一年中遵守一系列条件，重罪会被降级为轻罪。不过这也意味着他不得不常常要回到法院，他的下个听证在星期四。
   </p>
   <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="0d42">
    “当听到这些时，Shenery 十分难以接受，这个基于错误观点对他起诉的案子时间拖的如此漫长，而且结果又是这样”， White 说， “Shenery 认为这不公平，而且他们也没有理由和正当基础对他做出这些事。但他同时感觉这个斗争也是徒劳的，反正每当警察看见他时都会认定了他是帮派成员……
   </p>
   <p class="graf graf--p graf-after--p" id="8c53">
    如何才能制止这种荒诞的噩梦？⚪️
   </p>
