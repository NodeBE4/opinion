   <p class="graf graf--p graf-after--figure" id="90a2">
    您的国家的政府是否履行了保护公民权利和政治权利的义务？联合国拥有一批具有这一使命的独立专家，旨在加强民间社会更多参与到自由表达倡导的全球空间中来。
    <strong class="markup--strong markup--p-strong">
     您可以通过多种方式和他们合作，将其纳入您的维权工作中
    </strong>
    。以下是 IFEX 的5分钟指南，您将学会应该如何寻求帮助。
   </p>
   <p class="graf graf--p graf-after--p" id="1059">
    人权委员会是干什么的？
   </p>
   <p class="graf graf--p graf-after--p" id="6680">
    这完全是关于“公民权利和政治权利国际公约”（
    <a class="markup--anchor markup--p-anchor" data-href="https://www.ohchr.org/EN/ProfessionalInterest/Pages/CCPR.aspx" href="https://www.ohchr.org/EN/ProfessionalInterest/Pages/CCPR.aspx" rel="noopener nofollow noreferrer" target="_blank">
     ICCPR
    </a>
    ）。
   </p>
   <p class="graf graf--p graf-after--p" id="232b">
    ICCPR 是一项于 1973 年生效的条约，人权委员会由18名独立专家组成，负责监督其实施情况。虽然候选人是由国家提名的，并且由 ICCPR 的签署国选举产生，但他们以个人身份提供服务，而不是作为其政府的代表。他们的任期是四年。
   </p>
   <p class="graf graf--p graf-after--p" id="4a1a">
    只有作为“公民权利和政治权利国际公约”签署国的国家承担履行条约所承认的权利的法律义务，并且只有它们受到人权事务委员会的监督。您可以在
    <a class="markup--anchor markup--p-anchor" data-href="http://indicators.ohchr.org/" href="http://indicators.ohchr.org/" rel="noopener nofollow noreferrer" target="_blank">
     此处
    </a>
    查看完整列表。请记住，ICCPR的规定
    <strong class="markup--strong markup--p-strong">
     具有法律约束力
    </strong>
    。
   </p>
  </div>
  <div class="section-inner sectionLayout--outsetColumn">

   “公民权利和政治权利国际公约”第19条和第20条扩大了 1948 年“世界人权宣言”第19条所载的言论自由权的定义，并包括有关可接受的限制的具体措辞。

  <div>
  </div>
  <div>
   作为“公民权利和政治权利国际公约”签署国的国家应定期向人权事务委员会提交报告，通常每四至六年提交一次。第一份报告应在“公民权利和政治权利国际公约”对该国生效一年后提交。它应该包括国家如何遵守整个 ICCPR 的概述。随后的报告较短，往往侧重于回应人权事务委员会专家的意见。
  </div>
  <div>
   <p class="graf graf--p graf-after--p" id="69d8">
    <strong class="markup--strong markup--p-strong">
     人权事务委员会工作的A-B-C-D
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="ffc3">
    该小组每年举行三次会议，为期三周，通常在3月，7月和10月在日内瓦举行。
   </p>
   <p class="graf graf--p graf-after--p" id="b0cc">
    它有四个主要功能，
    <strong class="markup--strong markup--p-strong">
     民间社会组织可以参与 — 或至少从中受益
    </strong>
    — 大部分都是如此。
   </p>
   <ul class="postList">
    <li class="graf graf--li graf-after--p" id="93b4">
     A.它监测一个国家政治和公民权利的进展情况，并向各国提出建议：报告周期
    </li>
    <li class="graf graf--li graf-after--li" id="beb6">
     B.它针对被指侵犯“公民权利和政治权利国际公约”的个人投诉采取行动
    </li>
    <li class="graf graf--li graf-after--li" id="c060">
     C.就“公民权利和政治权利国际公约”的具体条款发表专家意见：一般性意见
    </li>
    <li class="graf graf--li graf-after--li" id="9301">
     D.审议国家间的投诉
    </li>
   </ul>
   <p class="graf graf--p graf-after--li" id="c330">
    <strong class="markup--strong markup--p-strong">
     以下是非政府组织在这四个职能中的作用：
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="5ecc">
    A — 签署国同意的过程有几个阶段，每个阶段都有自己的机会。
   </p>
   <p>
    <img alt="" class="aligncenter size-full wp-image-2427 jetpack-lazy-image" data-lazy-sizes="(max-width: 611px) 100vw, 611px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-4.png?resize=611%2C450&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-4.png?w=611&amp;ssl=1 611w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-4.png?resize=300%2C221&amp;ssl=1 300w" data-recalc-dims="1" height="450" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-4.png?resize=611%2C450&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="611"/>
    <noscript>
     <img alt="" class="aligncenter size-full wp-image-2427" data-recalc-dims="1" height="450" sizes="(max-width: 611px) 100vw, 611px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-4.png?resize=611%2C450&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-4.png?w=611&amp;ssl=1 611w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/07/0-4.png?resize=300%2C221&amp;ssl=1 300w" width="611"/>
    </noscript>
   </p>
   <p class="graf graf--p graf-after--figure" id="7ff6">
    第1步 — 通过问题清单（LOI）或报告前问题清单（LOIPR）。在其三届年会的每一届会议期间，人权事务委员会在即将召开的会议之前（通常提前一年）通过每个国家的议题清单。
   </p>
   <p class="graf graf--p graf-after--p" id="09da">
    •
    <strong class="markup--strong markup--p-strong">
     非政府组织可以通过提前向会议提交问题清单来参与这一过程
    </strong>
    。之后，非政府组织可以就人权事务委员会决定的问题清单提供意见。
   </p>
   <p class="graf graf--p graf-after--p" id="897a">
    第2步 — 各国报告和非政府组织的“替代报告”。
    <strong class="markup--strong markup--p-strong">
     这是非政府组织参与的最重要步骤
    </strong>
    。在三届年会期间，人权事务委员会审查各国关于其遵守“公民权利和政治权利国际公约”的报告，一般侧重于先前审查和“公约”清单所要求的信息。鼓励各国与包括民间社会团体在内的其他行动者合作编写报告 — 但这种情况可能并非总是如此。在某些情况下，一个国家可能根本没有提供任何报告。
   </p>
   <p class="graf graf--p graf-after--p" id="bfc2">
    • 非政府组织、联合国实体和国家人权机构可以提供自己的报告，称为“替代报告”。非政府组织 —
    <strong class="markup--strong markup--p-strong">
     无论是个人还是作为一个群体
    </strong>
    — 都可以就一个国家的公民权利和政治权利提交书面文稿，或关于公约的具体条款，
    <strong class="markup--strong markup--p-strong">
     例如言论自由权
    </strong>
    。提交
    <strong class="markup--strong markup--p-strong">
     截止日期
    </strong>
    在
    <a class="markup--anchor markup--p-anchor" data-href="https://tbinternet.ohchr.org/_layouts/TreatyBodyExternal/MasterCalendar.aspx" href="https://tbinternet.ohchr.org/_layouts/TreatyBodyExternal/MasterCalendar.aspx" rel="noopener nofollow noreferrer" target="_blank">
     联合国人权事务高级专员网站
    </a>
    上公布，通常在会议开始前约一个月审查国家。非政府组织的书面报告不应超过10,000字，
    <strong class="markup--strong markup--p-strong">
     应以英文、法文或西班牙文撰写
    </strong>
    。英语是最推荐的语言，因为这些文件没有翻译。
   </p>
   <p class="graf graf--p graf-after--p" id="77eb">
    第3步 — 国家审查。在其三届年会的每一届会议期间，人权事务委员会的18位专家根据他们收到的所有报告，评估五至七个国家遵守“公民权利和政治权利国际公约”的情况。每个国家/地区的审核持续一天。国家代表为专家的提问提供答案。在某些情况下，会议期间会提供答案;在其他情况下，国家在正式会议之后以书面形式提交答复。
   </p>
   <p class="graf graf--p graf-after--p" id="a017">
    • 非政府组织可以参加并观察审核过程。非政府组织在本届会议期间不能作口头发言。
   </p>
   <p class="graf graf--p graf-after--p" id="7d5c">
    • 不过，
    <strong class="markup--strong markup--p-strong">
     非政府组织可以在正式会议上向人权事务委员会的专家提供口头陈述
    </strong>
    。这些通常安排在会议开始前的一天，并且非常有效。建议您准备一份引人注目的陈述，讲述一个故事，最好由受到侵权行为直接影响的人提供，并且需要包括具体建议以及在审核期间向专家提出的问题。如需注册，请联系

    </a>
    和
    <a class="markup--anchor markup--p-anchor" data-href="mailto:ccpr@ohchr.org" href="/cdn-cgi/l/email-protection#e88b8b989aa887808b809ac6879a8f" rel="noreferrer noopener" target="_blank">
     <span class="__cf_email__" data-cfemail="dcbfbfacae9cb3b4bfb4aef2b3aebb">
      [email protected]
     </span>
    </a>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="bfcd">
    • 非政府组织还可以相互协调（在许多情况下，与设在日内瓦的
    <a class="markup--anchor markup--p-anchor" data-href="http://ccprcentre.org/" href="http://ccprcentre.org/" rel="noopener nofollow noreferrer" target="_blank">
     公民权利和政治权利中心
    </a>
    合作）与专家组织就特定国家举行非正式会议（无需口译）。这通常发生在国家审查的前一天或上午。
   </p>
   <p class="graf graf--p graf-after--p" id="5b85">
    • 非政府组织也可以尝试在预定会议之外采取非正式途径，您可以在
    <a class="markup--anchor markup--p-anchor" data-href="https://www.ohchr.org/EN/HRBodies/CCPR/Pages/Membership.aspx" href="https://www.ohchr.org/EN/HRBodies/CCPR/Pages/Membership.aspx" rel="noopener nofollow noreferrer" target="_blank">
     这里
    </a>
    了解专家。
   </p>
   <p class="graf graf--p graf-after--p" id="06b3">
    第4步 — 报告：结论性意见。在为期三周的国家审查会议之后，人权事务委员会发布一份名为“结论性意见”的报告。这通常在会议结束后一周左右发布。报告中提出每个国家在实施“公民权利和政治权利国际公约”方面取得的进展的意见，并向它们提出具体建议。
   </p>
   <p class="graf graf--p graf-after--p" id="b0d0">
    • 然后，非政府组织可以利用这份报告来提高认识、倡导、监督并向各国政府施加压力，使其尊重公民权利和政治权利。
   </p>
   <p class="graf graf--p graf-after--p" id="b94f">
    • 参与此过程的非政府组织可根据其关注的问题和建议是否包含在报告中来评估其自身的成功。
   </p>
   <p class="graf graf--p graf-after--p" id="3a11">
    第5步 — 跟进。在人权事务委员会审查国家一年后，要求国家提交一份关于结论性意见中所列建议的报告，这些建议被认为是紧急的。
   </p>
   <p class="graf graf--p graf-after--p" id="b016">
    • 允许
    <strong class="markup--strong markup--p-strong">
     非政府组织提供关于国家在这些紧急建议方面取得的进展的报告
    </strong>
    ，因此人权事务委员会可以对已经采取的行动有另一个渠道的了解。否则，他们可以使用国家报告来监督他们对人权事务委员会提出的建议的承诺。
   </p>
   <p class="graf graf--p graf-after--p" id="8eec">
    您可以在
    <a class="markup--anchor markup--p-anchor" data-href="https://tbinternet.ohchr.org/_layouts/TreatyBodyExternal/SessionsList.aspx?Treaty=CCPR" href="https://tbinternet.ohchr.org/_layouts/TreatyBodyExternal/SessionsList.aspx?Treaty=CCPR" rel="noopener nofollow noreferrer" target="_blank">
     此处
    </a>
    查看要审核的国家/地区列表和提交截止日期。
   </p>
   <p class="graf graf--p graf-after--p" id="ce06">
    B — 请注意，只有当有关国家也是“
    <a class="markup--anchor markup--p-anchor" data-href="https://www.ohchr.org/Documents/HRBodies/CCPR/OHCHR_Map_ICCPR-OP1.pdf" href="https://www.ohchr.org/Documents/HRBodies/CCPR/OHCHR_Map_ICCPR-OP1.pdf" rel="noopener nofollow noreferrer" target="_blank">
     第一任择议定书
    </a>
    ”的缔约国时，才有可能提出个人申诉。
   </p>
   <p class="graf graf--p graf-after--p" id="fc63">
    非政府组织可以支持作为“公民权利和政治权利国际公约”第一任择议定书缔约国的个人直接提交来文，如果他或她在条约下的权利受到国家侵犯的话。以下需注意：
   </p>
   <p class="graf graf--p graf-after--p" id="88ff">
    • 必须由权利受到侵犯或经个人书面同意的个人提交。沟通不能是匿名的。
   </p>
   <p class="graf graf--p graf-after--p" id="87fd">
    • 来文必须表明
    <strong class="markup--strong markup--p-strong">
     国内补救办法已经用尽
    </strong>
    （或证明所有措施都已经尝试过）
   </p>
   <p class="graf graf--p graf-after--p" id="011a">
    • 另一项国际调查或解决程序不能考虑“同一事项”（同一作者，同一事实和同一实质权利）。 （例外情况是已将此事项提交人权理事会投诉程序，特别报告员或人权理事会工作组）。
   </p>
   <p class="graf graf--p graf-after--p" id="03e2">
    • 必须发送至以下
    <strong class="markup--strong markup--p-strong">
     地址
    </strong>
    :
   </p>
   <p class="graf graf--p graf-after--p" id="4e72">
    Petitions and Inquiries Section
   </p>
   <p class="graf graf--p graf-after--p" id="60f3">
    Office of the High Commissioner for Human Rights
   </p>
   <p class="graf graf--p graf-after--p" id="ff39">
    United Nations Office at Geneva
   </p>
   <p class="graf graf--p graf-after--p" id="0a47">
    1211 Geneva 10, Switzerland
   </p>
   <p class="graf graf--p graf-after--p" id="f52c">
    <a class="markup--anchor markup--p-anchor" data-href="mailto:petitions@ohchr.org" href="/cdn-cgi/l/email-protection#057560716c716c6a6b76456a6d666d772b6a7762" rel="noreferrer noopener" target="_blank">
     <span class="__cf_email__" data-cfemail="6a1a0f1e031e030504192a05020902184405180d">
      [email protected]
     </span>
    </a>
   </p>
   <p class="graf graf--p graf-after--p" id="63fd">
    • 个人可以使用特殊格式提交投诉，可以在
    <a class="markup--anchor markup--p-anchor" data-href="https://www.ifex.org/international/2018/07/04/complaintformopiccpr_cat_cerd.doc" href="https://www.ifex.org/international/2018/07/04/complaintformopiccpr_cat_cerd.doc" rel="noopener nofollow noreferrer" target="_blank">
     此处
    </a>
    下载。您也可以在
    <a class="markup--anchor markup--p-anchor" data-href="https://www.ohchr.org/EN/HRBodies/TBPetitions/Pages/IndividualCommunications.aspx" href="https://www.ohchr.org/EN/HRBodies/TBPetitions/Pages/IndividualCommunications.aspx" rel="noopener nofollow noreferrer" target="_blank">
     此处
    </a>
    查看指南。在
    <a class="markup--anchor markup--p-anchor" data-href="http://juris.ohchr.org/en/Home/Index/" href="http://juris.ohchr.org/en/Home/Index/" rel="noopener nofollow noreferrer" target="_blank">
     这里
    </a>
    看到人权事务委员会的判例。
   </p>
   <p class="graf graf--p graf-after--p" id="6aa5">
    一旦被接受，人权事务委员会将在其中一届会议期间审议该申诉。他们的审议工作不公开，但其最终决定（称为“意见”）和后续行动是公开的，具有约束力。在此过程中，可允许当事方向人权事务委员会作出口头陈述（
    <a class="markup--anchor markup--p-anchor" data-href="https://www.ohchr.org/Documents/HRBodies/CCPR/GuidelinesOralHearings.pdf" href="https://www.ohchr.org/Documents/HRBodies/CCPR/GuidelinesOralHearings.pdf" rel="noopener nofollow noreferrer" target="_blank">
     此处
    </a>
    为准则）。
   </p>
   <p class="graf graf--p graf-after--p" id="507c">
    C — 人权事务委员会的一个重要作用是帮助解释“公民权利和政治权利国际公约”的条款。所谓的“一般性意见”为其实施提供了指导。
   </p>
   <p class="graf graf--p graf-after--p" id="8051">
    2011年，CCPR 发表了关于言论自由权的
    <a class="markup--anchor markup--p-anchor" data-href="http://www2.ohchr.org/english/bodies/hrc/docs/gc34.pdf" href="http://www2.ohchr.org/english/bodies/hrc/docs/gc34.pdf" rel="noopener nofollow noreferrer" target="_blank">
     第34号一般性意见
    </a>
    。这种解释对于支持您的国家的 NGO 宣传工作非常有用！
   </p>
   <p class="graf graf--p graf-after--p" id="a0df">
    本文加强了对 ICCPR 第19条的解释; 它澄清了对言论自由权的合法限制的界限，着眼于各国有义务不仅保护权利，而且应积极防止任何侵犯权利行为并促进权利的享受，包括媒体自由。注意，它还
    <strong class="markup--strong markup--p-strong">
     考虑了互联网的影响，并且不仅承认传统记者的权利，而且还承认博主、和“从事自我出版形式的其他人”的权利
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="e6fb">
    • 非政府组织也可以参与这一进程。在拟定一般性意见时，呼吁民间社会分享投入。
   </p>
   <p class="graf graf--p graf-after--p" id="d191">
    D — 缔约国可提交来文，指称另一缔约国未履行“公民权利和政治权利国际公约”规定的义务。但是，到目前为止，尚未使用此工具。在这个过程中，非政府组织没有任何作用。
   </p>
   <p class="graf graf--p graf-after--p" id="a2a0">
    <strong class="markup--strong markup--p-strong">
     最后一句话：战略提示和有用资源
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="1409">
    IFEX 与人权委员会合作的六大秘诀
   </p>
   <p class="graf graf--p graf-after--p" id="6256">
    1、
    <strong class="markup--strong markup--p-strong">
     时间就是一切
    </strong>
    ！您可以利用与人权委员会有关的活动，例如出版问题清单（LOI），审查会议，结论性意见的传播和后续报告，通过组织活动，国家和地方一级的宣传行动和宣传战略
   </p>
   <p class="graf graf--p graf-after--p" id="f3bf">
    。审查会议特别擅长吸引媒体关注。
   </p>
   <p class="graf graf--p graf-after--p" id="0541">
    2.
    <strong class="markup--strong markup--p-strong">
     联合的力量
    </strong>
    。考虑与您所在国家的其他非政府组织合作。这可以增强你的声音 — 不仅在日内瓦，而且在你自己的国家，加强合作。
   </p>
   <p class="graf graf--p graf-after--p" id="ca7e">
    3.
    <strong class="markup--strong markup--p-strong">
     了解你需要的东西
    </strong>
    。在编写替代报告时，请审查国家以前的报告，结论性意见和后续建议，以及以前的问题清单。您可以在
    <a class="markup--anchor markup--p-anchor" data-href="https://tbinternet.ohchr.org/_layouts/treatybodyexternal/TBSearch.aspx?Lang=en" href="https://tbinternet.ohchr.org/_layouts/treatybodyexternal/TBSearch.aspx?Lang=en" rel="noopener nofollow noreferrer" target="_blank">
     此数据库
    </a>
    中找到所有这些信息。
   </p>
   <p class="graf graf--p graf-after--p" id="9128">
    4.
    <strong class="markup--strong markup--p-strong">
     会话只是一个开始
    </strong>
    。使用结论性观察作为宣传和提高认识的起点。例如：密切监测国家采取的措施，以解决其标志性问题; 与国家就这些问题进行对话; 围绕他们组织会议，研讨会和专题讨论会。
   </p>
   <p class="graf graf--p graf-after--p" id="4ae4">
    5.
    <strong class="markup--strong markup--p-strong">
     超越人权委员会
    </strong>
    。您的长期战略可以包括
    <a class="markup--anchor markup--p-anchor" data-href="https://www.ohchr.org/EN/HRBodies/Pages/HumanRightsBodies.aspx" href="https://www.ohchr.org/EN/HRBodies/Pages/HumanRightsBodies.aspx" rel="noopener nofollow noreferrer" target="_blank">
     监督人权遵守情况的其他一些联合国机制
    </a>
    ，例如普遍定期审议（UPR）和特别程序以及区域机制。一旦你学会了如何与一个人交往，你就可以更容易地与他人交往。此外，为一个机构收集的信息可以提供给其他机构。
   </p>
   <p class="graf graf--p graf-after--p" id="c0e5">
    6. 不要忘记在报告中加入
    <strong class="markup--strong markup--p-strong">
     性别层面
    </strong>
    比较。您所在国家的政治和公民权利受到侵犯的事实是否会以同样的方式影响女性或 LGBTQI 个体？所有人的解决方案和建议是否相同？
   </p>
