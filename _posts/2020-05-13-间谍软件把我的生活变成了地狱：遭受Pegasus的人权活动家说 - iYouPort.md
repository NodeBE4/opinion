  <p class="graf graf--p">
   NSO的间谍软件
   <strong class="markup--strong markup--p-strong">
    并没有
   </strong>
   在去年的大规模揭露和全球性的谴责之下偃旗息鼓，调查人员继续发现它被用于危害人权。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a4%a7%e8%a7%84%e6%a8%a1%e7%bd%91%e7%bb%9c%e9%92%93%e9%b1%bc%e3%80%81%e7%9b%91%e8%a7%86%e5%92%8c%e5%9c%a8%e7%ba%bf%e6%94%bb%e5%87%bb%e6%88%90%e4%b8%ba%e6%b4%bb%e5%8a%a8%e5%ae%b6%e4%b8%8d%e5%be%97-2/" href="https://www.iyouport.org/%e5%a4%a7%e8%a7%84%e6%a8%a1%e7%bd%91%e7%bb%9c%e9%92%93%e9%b1%bc%e3%80%81%e7%9b%91%e8%a7%86%e5%92%8c%e5%9c%a8%e7%ba%bf%e6%94%bb%e5%87%bb%e6%88%90%e4%b8%ba%e6%b4%bb%e5%8a%a8%e5%ae%b6%e4%b8%8d%e5%be%97-2/" rel="noopener noreferrer" target="_blank">
     大规模网络钓鱼、监视和在线攻击成为活动家不得不面对的危险
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   国际特赦组织（Amnesty International）的一份新报告显示，为政府工作的黑客利用臭名昭著的以色列监视供应商 NSO 集团制造的恶意软件瞄准了两名摩洛哥人权活动家。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    更多参见专栏 “
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e4%bb%a5%e8%89%b2%e5%88%97%e5%92%8cnso/" href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e4%bb%a5%e8%89%b2%e5%88%97%e5%92%8cnso/" rel="noopener noreferrer" target="_blank">
     关于以色列和NSO
    </a>
    ”
   </li>
  </ul>
  <p class="graf graf--p">
   在报告中，大赦国际的研究人员详细介绍了对历史学家和记者 Maati Monjib 、及代表摩洛哥一群抗议者的人权律师 Abdessadak El Bouchattaoui 发起的一系列攻击。
  </p>
  <p class="graf graf--p">
   自2017年以来，两人收到了一系列的文本消息，其中包含曾经被用于攻击国际特赦组织成员以及数字权利组织 Citizen Lab 的基础设施的恶意链接。
  </p>
  <p class="graf graf--p">
   只要目标人点击了这些链接，就可以自动在目标手机上静默安装 NSO 的 Pegasus 间谍软件。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/report%ef%bc%9a%e4%bb%a5%e8%89%b2%e5%88%97-nso-%e7%94%9f%e4%ba%a7%e7%9a%84%e9%97%b4%e8%b0%8d%e8%bd%af%e4%bb%b6-pegasus-%e5%9c%a845%e4%b8%aa%e5%9b%bd%e5%ae%b6%e8%a2%ab%e7%94%a8%e4%ba%8e%e7%9b%91/" href="https://www.iyouport.org/report%ef%bc%9a%e4%bb%a5%e8%89%b2%e5%88%97-nso-%e7%94%9f%e4%ba%a7%e7%9a%84%e9%97%b4%e8%b0%8d%e8%bd%af%e4%bb%b6-pegasus-%e5%9c%a845%e4%b8%aa%e5%9b%bd%e5%ae%b6%e8%a2%ab%e7%94%a8%e4%ba%8e%e7%9b%91/" rel="noopener noreferrer" target="_blank">
     Report：以色列 NSO 生产的间谍软件 Pegasus 在45个国家被用于监视和攻击异议人士及活动家 — ⚠️ 亚洲有香港
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   Pegasus 是 NSO 集团生产的监视软件。在过去的几年中，安全研究人员发现了许多证据，证明墨西哥、沙特阿拉伯和阿联酋等国家都在使用 Pegasus 监视记者、人权活动家和持不同政见者。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11266 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-4.png?resize=1100%2C759&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-4.png?w=1278&amp;ssl=1 1278w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-4.png?resize=300%2C207&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-4.png?resize=1024%2C707&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-4.png?resize=768%2C530&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-4.png?resize=1100%2C759&amp;ssl=1 1100w" data-recalc-dims="1" height="759" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-4.png?resize=1100%2C759&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11266" data-recalc-dims="1" height="759" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-4.png?resize=1100%2C759&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-4.png?w=1278&amp;ssl=1 1278w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-4.png?resize=300%2C207&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-4.png?resize=1024%2C707&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-4.png?resize=768%2C530&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-4.png?resize=1100%2C759&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   而这次发现的攻击是政府如何使用先进的监视技术来骚扰和恐吓异议人士和人权活动家的又一个例子。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “通过侵入性数字监视使说出摩洛哥人权记录真相的和平评论家和激进抗议者遭受到骚扰或恐吓，这是对他们隐私权和表达自由权的严重侵犯”，国际特赦组织副主任在新闻稿中说。
  </p>
  <p class="graf graf--p">
   Monjib 告诉 Motherboard，在过去的几年中，“物理监视和电子监视使我陷入了地狱般的生活。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他们希望我感到不安全，他们的确在这方面取得了成功，而且一次又一次地。我一生中的每一分钟都承受着压力。但我不会投降。” 他在网上聊天中说， “他们的确很强大，但我想向他们展示我的决心。”
  </p>
  <p class="graf graf--p">
   NSO集团发言人继续狡辩，声称这是滥用，他们“会调查”。但是这种间谍技术只出售给政府，NSO “期待” 政府如何使用它呢？
  </p>
  <p class="graf graf--p">
   NSO 集团是向全球政府出售间谍软件和黑客工具的监视巨头公司之一。该公司表示，它只出售给执法部门和情报机构。
  </p>
  <p class="graf graf--p">
   国际特赦组织研究人员还发现了一种更不寻常且隐秘的黑客类型的证据，称为“中间人攻击”或网络注入攻击。控制互联网基础结构的攻击者操纵并拦截Web流量，以将对合法网站的访问重定向到恶意网站，从而使恶意软件感染目标。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11267 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-3.png?resize=1100%2C724&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-3.png?w=1282&amp;ssl=1 1282w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-3.png?resize=300%2C198&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-3.png?resize=1024%2C674&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-3.png?resize=768%2C506&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-3.png?resize=1100%2C724&amp;ssl=1 1100w" data-recalc-dims="1" height="724" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-3.png?resize=1100%2C724&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11267" data-recalc-dims="1" height="724" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-3.png?resize=1100%2C724&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-3.png?w=1282&amp;ssl=1 1282w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-3.png?resize=300%2C198&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-3.png?resize=1024%2C674&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-3.png?resize=768%2C506&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-3.png?resize=1100%2C724&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   去年早些时候，Monjib 尝试在其 iPhone 的 Safari 浏览器中输入 Yahoo.fr 来访问法语版的雅虎网站。
  </p>
  <p class="graf graf--p">
   据大赦国际的研究人员称，
   <strong class="markup--strong markup--p-strong">
    浏览器并没有将他定向到官方网站，而是转到了另一个可能在手机上安装恶意软件的网站，他们相信至少有一次 Monjib 的手机被成功入侵了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   研究人员能够通过检查 Monjib 的浏览历史来发现这一点。
  </p>
  <p class="graf graf--p">
   调查不确定这些中间人攻击是否也使用了 NSO 集团的技术，但一些间接证据指明了这一方向。
  </p>
  <p class="graf graf--p">
   首先，网络注入攻击是在恶意SMS攻击之后出现的。其次，根据作为 Hacking Team 泄露的产品手册，其中显示NSO向客户提供了这种功能。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/category/hackingteam/" href="https://www.iyouport.org/category/hackingteam/" rel="noopener noreferrer" target="_blank">
     关于 Hacking Team
    </a>
   </li>
  </ul>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “很重要的一点是，要记录针对民间社会使用的攻击能力类型，以便帮助处于危险境地的个人正确地制定战略措施来保护自己，” 调查这些攻击的研究人员说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这种类型的攻击很难识别和预防，摩洛哥的人权捍卫者及其支持网络应被告知可能面临的挑战。” ⚪️
  </p>
