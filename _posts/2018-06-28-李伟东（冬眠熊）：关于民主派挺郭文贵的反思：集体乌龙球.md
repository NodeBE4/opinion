<p>
<span style="font-size: 12pt;">
   【郭（文贵）妖伪爆事件反思之一：反垬（土共的缩写）的未必是尊崇民主自由的，很可能是跟垬一样坏的流氓文痞人渣，他们可能混迹于民运，但跟民主运动毫无关系】以袁白衣（指袁红兵）和复读郭（指郭宝胜）及昭明（澳洲的王德利）为代表的几十个疯狂挺郭妖的所谓海外民运人士，其实与民主运动毫无关系，他们既不要民主也不要自由，他们只要流氓专政及以反垬为商标的生意。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   补充：
  </span>
<br/>
<span style="font-size: 12pt;">
   海外数年生活经验告诉我：反垬反专制并非寻找同道的唯一标准。中国的社会制度会制造大批的失意者，少部分人会来到海外。海外的自由言论空间可以让这些人充分宣泄对中共的愤怒，但他们内心可能只希望换一个明君或能抬升他在体制内的地位。他们不理解也不要民主，哪怕喊革命也不是要民主而是自己做民王。
  </span>
</p>
<p>
</p>
<p>
<span style="font-size: 12pt;">
   【郭妖伪爆事件反思之二：民运陷入郭陷阱的原因】自郭爆红始，海外六大民运组织（海外联席会、两个民主党、民联、民阵、公民力量）中有五个的主要领袖（除民联）挺郭，还新生出以挺郭始反郭终的唐（伯桥）袁（红兵）两个新组织。他们之所以挺郭是因为与郭的行为模式同构：热衷宫廷内斗期待成为圆桌一方、不择手段、被金主意志操纵。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   补充：
  </span>
<br/>
<span style="font-size: 12pt;">
   郭骗现在已是一具挺着双臂跳来跳去的僵尸，每天在屏幕上扮演搞笑的小丑而已。那么郭骗落幕是不是我们批的结果，惭愧，不是～我们只提醒了若干本来就比较聪明的人，大量郭粉并不愿反思～是郭自己把自己玩残了。郭骗完蛋，共党是不是得意了？也不是，因为聪明人都会从被党国培育出的郭骗身上看到垬的邪恶。

<br/>
<span style="font-size: 12pt;">

<span style="font-size: 12pt;">

<p>
<span style="font-size: 12pt;">
   【郭妖伪爆事件反思之三：民运任由郭骗搅局的原因】表因是若干组织人士被收买；看到郭突然蹿红很艳羡要搭便车等。深因是民运基本理论过时～民运始自帮共党改革并要求平反的运动而非革命取代运动，因此热衷宫廷内斗，期待成为圆桌一方，特别看重和参与揭秘，而非新纲领和战略构建，因而特别容易被骗子利用。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   补充：
  </span>
<br/>
<span style="font-size: 12pt;">
   看到陈立群副主席关于民主党不挺不砸郭的声明，我抱歉把贵党列入挺郭系列，也看到立群、闯创对自由派也批民运的不满，很理解。但也请民运能经得起爱护性帮忙性的反思批评。我尊重很多反垬老战士如胡平、立群等，但这次很多人和组织卷入挺郭闹剧，让人痛心耻笑，不反思不道歉，文过饰非，是不会重新赢回尊重的。

<span style="font-size: 12pt;">

<span style="font-size: 12pt;">
   【郭妖伪爆事件反思之五：民运固有优势被郭糟损殆尽，郭败落同时也是旧民运的挽歌。希望能通过深刻反思，置于死地而后生】我曾说民运只有主义、理想、道义优势，绑到郭骗战车上就都随之死掉了。无耻至死的典型就是袁白衣（郭袁不过是一枚伪币的正反面）。【郭骗是郭大骗的通行证，郭粉是郭粉们的墓志铭】。

<span style="font-size: 12pt;">

<br/>
<span style="font-size: 12pt;">
   【郭壕落幕总结之四】短命的“造谣革命”或何清琏归纳的“郭氏推特革命及海外文革”。郭粉吹嘘的“爆料革命”（真好笑：揭短能带来革命？），其实是一场造谣起哄闹剧。谣言能否对体制构成革命动因？短期信息不对称情况下有可能，但在开放的信息世界将迅速被揭穿，成为笑柄。谣言可能风行一时但对真革命没用。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   【郭壕落幕总结之五】群众永远是极权的B面。群众是什么，中文本意就是“乌合之众”。群众的五大特质：被领袖煽起聚合，忠于盲从领袖，领袖指哪打哪，主动攻击领袖的敌人，领袖死鸟兽散。文革就是领袖与群众的超级历史模版，模仿之无任何进步意义只能酿成灾难，此次郭式文革亦然。民主需要公民而非群众。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   补充解释：群众、人民，不同于大众和公民。民主当然需要基础大众和全体公民的支持和参与，自由民主人士和反对党也当然需要动员公众和寻求普遍支持。但群众和人民这两个概念是垬宣传语言中的特定历史名词，人民是分阶级的，群众是为领袖而存在的。“朝阳群众”是依附于极权告密监控体系的。别当盲从群众。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   【郭壕落幕总结之六】七十亿人的世界，媒体是将其聚合、普及规则、树立道德良心公平正义、信息开放和促成公民参与监督的重要公器和第四权力。媒体如果沦落将造成重大灾难。此次由媒体掀起郭旋风，但明镜、美国之音、自由亚洲、纽约时报至今都没搞清他们推出的人到底叫什么和身份，真是媒体的莫大耻辱。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   【郭壕落幕总结之七】海外民运在此次郭事件中整体失算、失态、失尊和失败。海外民运多年来理论陈旧、政治纲领缺失、目标模糊、沉湎于宫廷内斗的猜谜和参与、用阴谋论猜度对手和内斗、奉行机会主义和厚黑哲学，这些是他们迅速整体追捧郭骗子、蹭他热度的原因。但郭七条让民运整体尴尬。该好好反思一下了
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   【郭壕落幕总结之八】革命与道德。滕彪已有分析，我都同意。补：郭人性之坏，是一直未走出文革宿命之中国利用了市场机制之后凝结出的第一坏种。跟随他的四大坏人（不点名了）尤以昭明王德力为最，他谩骂胁迫所有人支持郭，蹭郭的热度和钱财自肥，这种精致利己和践踏人性的后生是未来中国最该警惕的物种
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   【郭壕落幕总结之九】“上错车的民主派”（借章语）。此次郭事件，国内外有大批民主人士搭郭的便车，海外名人中只有民运理论家胡平守住了底线，独立异议人士如何清琏夏业良滕彪北风坚持了批评（匿名还有很多），国内只有章立凡沈良庆坚持批评立场。原因在于支持者仍陷于传统抗争策略缺少应对新极权的方略
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   【郭壕落幕总结之十】郭浩云是不是有一手好牌，最后打遭了？非也。他用威胁爆料、清君侧、郭七条投诚勾兑和分化打击利用民运及海外媒体这四条策略出手，下了一盘很大的棋，成为呼风唤雨的一时枭雄。但不能成功的根本原因是他那木桶有两根致命的短板：假料和缺德。大家误把他忽悠出来的“势”当成了“牌”。
  </span>
</p>
