   不论是所谓的深度造假，还是其他技术高明的甚至仅仅是拙劣的信息战谎言，都能骗倒一大群人。
  </p>
  <p class="graf graf--p">
   不准确的信息 —— 或者干脆是故意捏造的虚假信息 —— 如今已成为互联网的常态。人们以各种目的发布虚假的信息，而不仅仅是信息战中的受雇人员，比如，只是为强调自己的立场、或者只是追求吸睛度和粉丝量而故意青睐那些刺激性的虚假信息。
  </p>
  <p class="graf graf--p">
   这是一种生态。真相永远赶不上假消息的刺激和吸引力。在这种情况下，甚至美国总统
   <a class="markup--anchor markup--p-anchor" data-href="https://www.worldcat.org/title/deciding-whats-true-the-rise-of-political-fact-checking-in-american-journalism/oclc/941139313&amp;referer=brief_results" href="https://www.worldcat.org/title/deciding-whats-true-the-rise-of-political-fact-checking-in-american-journalism/oclc/941139313&amp;referer=brief_results" rel="noopener noreferrer" target="_blank">
    也需要
   </a>
   不断接受事实核查。华盛顿邮报
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/graphics/politics/trump-claims-database/" href="https://www.washingtonpost.com/graphics/politics/trump-claims-database/" rel="noopener noreferrer" target="_blank">
    统计
   </a>
   ，迄今为止，他每天平均有15次虚假或误导性的公开声明。
  </p>
  <p class="graf graf--p">
   再一次，这绝非哪个民族国家的特征。一些
   <a class="markup--anchor markup--p-anchor" data-href="https://www.ideasworthteachingawards.com/2019-course-winners/market-manipulations" href="https://www.ideasworthteachingawards.com/2019-course-winners/market-manipulations" rel="noopener noreferrer" target="_blank">
    研究表明
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    世界各地的人们一直对虚幻的东西怀有甜蜜的爱慕之心，人们倾向于被那些 “太符合理想” 以至于难以置信的信息所吸引。
   </strong>
  </p>
  <p class="graf graf--p">
   认知科学家已经确定了许多避免人们受骗的常见方法。但是骗子，尤其擅长社交工程学的骗子，他们以吸引目标人士的偏见、固有信念和突出的愿景的方式来讲述虚假的故事。他们使用的是利用人类弱点的策略。

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    令人不快的现实
   </strong>
  </h3>
  <p class="graf graf--p">
   通常，“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.ft.com/content/7802f662-a7b2-11e9-984c-fac8325aaa04" href="https://www.ft.com/content/7802f662-a7b2-11e9-984c-fac8325aaa04" rel="noopener noreferrer" target="_blank">
    情绪脆弱
   </a>
   ” 的人不愿意接受令人不快的现实。
  </p>
  <p class="graf graf--p">
   英国作家 Arthur Conan Doyle，创造了夏洛克·福尔摩斯的那位，他的经典台词是：你一旦去除了不可能性，最后留下来的，哪怕多么难以置信，都是事实真相。
  </p>
  <p class="graf graf--p">
   然而，在经历了家庭悲剧和一次世界大战中的死亡恐怖之后，Doyle 在1916年公开
   <a class="markup--anchor markup--p-anchor" data-href="https://www.worldcat.org/title/conan-doyle-and-the-mysterious-world-of-light-1887-1920/oclc/1052838293&amp;referer=brief_results" href="https://www.worldcat.org/title/conan-doyle-and-the-mysterious-world-of-light-1887-1920/oclc/1052838293&amp;referer=brief_results" rel="noopener noreferrer" target="_blank">
    宣布
   </a>
   ，他赞成唯心论的信仰，包括死者的灵魂可以与活人交流。
  </p>
  <p class="graf graf--p">
   1922年，Doyle 在纽约市的家中拜访了魔术师 Harry Houdini，后者向他展示了一个巧妙的魔术，包括自动在悬挂的石板上书写。而 Houdini 无法说服 Doyle
   <a class="markup--anchor markup--p-anchor" data-href="https://web.archive.org/web/20170412193049/http://www.csicop.org/si/show/houdinirsquos_impossible_demonstration" href="https://web.archive.org/web/20170412193049/http://www.csicop.org/si/show/houdinirsquos_impossible_demonstration" rel="noopener noreferrer" target="_blank">
    这不是
   </a>
   超自然现象。
  </p>
  <p class="graf graf--p">
   为什么？因为 Doyle 已经深信超自然的存在，他不再接受现实说法。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    你的需求
   </strong>
  </h3>
  <p class="graf graf--p">
   有时人们垂涎其他同龄人已经取得的成就，以致于他们会忽视表面上的东西并欺骗自己和他人，以争取更好的机会和更好的生活。
  </p>
  <p class="graf graf--p">
   1822年，苏格兰骗子 Gregor MacGregor 说服了寻求轻松赚钱和美好生活的同胞购买债券、土地和特殊权限，加满了两艘船，驶向田园诗般的国家
   <a class="markup--anchor markup--p-anchor" data-href="https://www.worldcat.org/title/land-that-never-was-sir-gregor-macgregor-and-the-most-audacious-fraud-in-history/oclc/229019939&amp;referer=brief_results" href="https://www.worldcat.org/title/land-that-never-was-sir-gregor-macgregor-and-the-most-audacious-fraud-in-history/oclc/229019939&amp;referer=brief_results" rel="noopener noreferrer" target="_blank">
    Poyais
   </a>
   。
  </p>
  <p class="graf graf--p">
   MacGregor 为 Poyais 的土地定价，以使那些苏格兰商人和不熟练的工人也能够负担得起，他们听说过南美的有前途的投资，但缺乏利用这些投资的手段。 他们被告知 Poyais 拥有独特的旗帜、自己的货币和在伦敦的外交机构。
  </p>
  <p class="graf graf--p">
   唯一的问题是，Poyais 并不存在。参加这次航行的大多数人死于洪都拉斯的蚊子海岸。
  </p>
  <p class="graf graf--p">
   少数幸存者后来表示，他们之所以被吸引，是因为
   <strong class="markup--strong markup--p-strong">
    他们拒绝接受 Poyais 实际上并不存在的事实
   </strong>
   ，并辩称是 MacGregor “被骗了”。
  </p>
  <p class="graf graf--p">
   有趣吧，几乎被折腾到死掉的人却反过来为骗子辩护。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    贪婪者都是瞎子
   </strong>
  </h3>
  <p class="graf graf--p">
   贪婪会阻止人们看到他们做出的决定违背了常识。
  </p>
  <p class="graf graf--p">
   1925年，骗子 Victor Lustig 利用了法国政府的公开抱怨，即 翻修一座正在腐烂的埃菲尔铁塔要比拆除它花费更多。
  </p>
  <p class="graf graf--p">
   他聚集了一大群废铁交易商，说服他们将塔拆了卖给他们其中一个人。然后他故伎重演再次 “卖掉” 铁塔。 Lustig 在历史上被誉为 “出售埃菲尔铁塔的人”。
  </p>
  <p class="graf graf--p">
   这个故事听起来可笑吗？一点都不。如今使用这种骗术的人依旧存在。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    利用无知
   </strong>
  </h3>
  <p class="graf graf--p">
   骗子可以在你对某些事无知和不熟悉当地习俗的状态下找到机会。
  </p>
  <p class="graf graf--p">
   就如，骗子 George C. Parker 将布鲁克林大桥卖了四次，都是卖给了那些新移民，他们不知道这座桥是不能卖出去的。
  </p>
  <p class="graf graf--p">
   这家伙还出售了格兰特陵墓、大都会艺术博物馆和自由女神像。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    苦难滋生的绝望
   </strong>
  </h3>
  <p class="graf graf--p">
   绝望的人不会再怀疑。他们认为，当替代方案过于悲惨时，人们就会相信承诺是真实的。
  </p>
  <p class="graf graf--p">
   John D. Rockefeller 的父亲威廉是重婚者，也是兜售假药的人，走江湖的巫医，这种药物对生病的人来说毫无帮助。
  </p>
  <p class="graf graf--p">
   据说这位 “医生” Rockefeller 曾为他的儿子、标准石油信托基金的创建者，进行过辅导。
  </p>
  <p class="graf graf--p">
   在这种情况下，“替代方案” 是死亡。这一点也不奇怪。当执业医师表示这种病无药可用时，人们就会相信任何东西能 “治疗它”，包括中国乡村一些人依旧信奉的香灰。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    有时只是信任
   </strong>
  </h3>
  <p class="graf graf--p">
   人们相信一个故事是因为他们相信讲故事的人。
  </p>
  <p class="graf graf--p">
   他们不知道如何刨根问底，也不想这么做，或者认为没有必要这样做。
  </p>
  <p class="graf graf--p">
   骗子 Bernie Madoff 早在1980年代中期就开始在美国富有的犹太退休人员及其慈善组织和欧洲的贵族家庭成员中寻找庞氏骗局的投资者。
  </p>
  <p class="graf graf--p">
   而那些受害人只是信任了团体中为 Madoff 及其投资担保的其他人。
  </p>
  <p class="graf graf--p">
   您需要时刻提醒自己觉察这点：您是否已经信任另一个人超过了信任自己的判断力？
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    追究很难，或费用高昂
   </strong>
  </h3>
  <p class="graf graf--p">
   1912年，在英国东萨塞克斯郡的皮尔当（Piltdown）发现了一块头骨，一些骨头和其他遗物。
  </p>
  <p class="graf graf--p">
   这些骨骸被当时的考古学专家宣称为某种前所未见的早期人类遗骸化石，而拉丁学名 Eoanthropus dawsoni 即被赋予该样本代表的古人类。
  </p>
  <p class="graf graf--p">
   结果是，花费了40多年的时间才得以确认 Piltdown Man 是一个骗局，花了100多年的时间才确定谁伪造了它。
  </p>
  <p class="graf graf--p">
   这类难以反驳的东西是行骗的最有利途径 —— 考虑持续搜寻大脚怪或尼斯湖怪兽的行动。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    人们总是渴望梦想成真
   </strong>
  </h3>
  <p class="graf graf--p">
   有时，尽管有内在的怀疑，但是人们依旧会非常希望将不可能的事情变成现实，以梦想带动世界。
  </p>
  <p class="graf graf--p">
   例如，如果外星飞船
   <a class="markup--anchor markup--p-anchor" data-href="http://content.time.com/time/specials/packages/article/0,28804,1860871_1860876_1861006,00.html" href="http://content.time.com/time/specials/packages/article/0,28804,1860871_1860876_1861006,00.html" rel="noopener noreferrer" target="_blank">
    确实坠毁
   </a>
   在某地，那将意味着星际旅行是可能的。在90年代的外星人谣言爆发的热门期间，这类骗术几乎铺天盖地。
  </p>
  <p class="graf graf--p">
   换句话说，如果骗子能知道你最想要什么，那么你就肯定危险了…… 您一定听说过 “不挨饿的减肥大法”、“愉快的戒烟之旅” 等等？
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    重复 —— 社交媒体的标志性欺骗 —— *创造*信念
   </strong>
  </h3>
  <p class="graf graf--p">
   一遍又一遍地重复听到一个虚假的主张，就足以使人们对此产生信心。
  </p>
  <p class="graf graf--p">
   最简单的如，常见的广告和公关策略，就是通过增加 “印象次数” 来使其高度可见，从而使人们信任它的。
  </p>
  <p class="graf graf--p">
   这也是戈培尔的拿手好戏之一。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    集体出动的欺骗运动
   </strong>
  </h3>
  <p class="graf graf--p">
   单单重复可能还不够。当人们尝试评估某件事是否正确时，他们通常会寻找客观的理由作为其信仰的基础，例如，找到关于该事件的两个相似的独立判断。它被称为
   <a class="markup--anchor markup--p-anchor" data-href="https://ssrn.com/abstract=1278110" href="https://ssrn.com/abstract=1278110" rel="noopener noreferrer" target="_blank">
    Rule of Two
   </a>
   。
  </p>
  <p class="graf graf--p">
   在社交媒体上，用户经常会重复看到由不同的朋友或联系人发布的说法。相同的信息似乎不仅来自世界各地，而且显然来自独立的来源。
  </p>
  <p class="graf graf--p">
   但是，事实上通常只有一个来源，尽管轻松的在线共享使它看起来不止于一个来源。
  </p>
  <p class="graf graf--p">
   这就是为什么如此多的观察员担心社交媒体在政治中扮演的角色的原因 ——
   <strong class="markup--strong markup--p-strong">
    它可以轻而易举地使人们相信错误的说法是真实的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这也是当权者操纵的人格暗杀运动的关键支柱，他们买通一群人同时撒谎，其中包括公信力很高的企业媒体。更多解释详见 《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%ba%e6%a0%bc%e6%9a%97%e6%9d%80%e5%92%8c%e6%8a%b9%e9%bb%91%e8%bf%90%e5%8a%a8%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%8f%97%e7%9b%8a%ef%bc%9f/" href="https://www.iyouport.org/%e4%ba%ba%e6%a0%bc%e6%9a%97%e6%9d%80%e5%92%8c%e6%8a%b9%e9%bb%91%e8%bf%90%e5%8a%a8%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%8f%97%e7%9b%8a%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    人格暗杀和抹黑运动：谁在受益？
   </a>
   》。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/Xs0K4ApWl4g" width="750">
   </iframe>
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    人们倾向于相信别人似乎相信的东西
   </strong>
  </h3>
  <p class="graf graf--p">
   人们有内在的意愿去屈服于显然是所谓的专家或合法机构的自信主张。
  </p>
  <p class="graf graf--p">
   在著名的 Stanley Milgram 的实验中，普通百姓遵守了 “科学家” 的指示，对受试者施加令人痛苦的电击 “治疗”。电击是真的，但科学家是假的，于是并不存在什么治疗。
  </p>
  <p class="graf graf--p">
   一个充满激情和说服力的骗子，经常伪装成各种专家 —— 例如，艺术品经销商或奇迹疗法研究人员 —— 利用这种心理弱点使人们相信虚假的说法。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    见《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%ad%a6%e6%83%95%e5%be%ae%e5%a6%99%e7%9a%84%e5%8a%9d%e5%af%bc%e6%9c%af%ef%bc%9a%e4%b8%80%e4%b8%aa%e8%83%bd%e5%8a%9b%e6%8e%92%e4%bc%97%e8%ae%ae%e5%8d%9a%e5%8f%96%e8%ae%a4%e5%90%8c%e7%9a%84/" href="https://www.iyouport.org/%e8%ad%a6%e6%83%95%e5%be%ae%e5%a6%99%e7%9a%84%e5%8a%9d%e5%af%bc%e6%9c%af%ef%bc%9a%e4%b8%80%e4%b8%aa%e8%83%bd%e5%8a%9b%e6%8e%92%e4%bc%97%e8%ae%ae%e5%8d%9a%e5%8f%96%e8%ae%a4%e5%90%8c%e7%9a%84/" rel="noopener noreferrer" target="_blank">
     警惕微妙的劝导术：一个能力排众议博取认同的“骗子”长啥样
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13361 jetpack-lazy-image" data-lazy-sizes="(max-width: 804px) 100vw, 804px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-16.png?resize=804%2C998&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-16.png?w=804&amp;ssl=1 804w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-16.png?resize=242%2C300&amp;ssl=1 242w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-16.png?resize=768%2C953&amp;ssl=1 768w" data-recalc-dims="1" height="998" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-16.png?resize=804%2C998&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="804"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13361" data-recalc-dims="1" height="998" sizes="(max-width: 804px) 100vw, 804px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-16.png?resize=804%2C998&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-16.png?w=804&amp;ssl=1 804w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-16.png?resize=242%2C300&amp;ssl=1 242w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-16.png?resize=768%2C953&amp;ssl=1 768w" width="804"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Robert Cialdini 引入的一种相关机制被称为 “社会证明”，即：看到别人去做您想做的事时就可以激发您也行动起来。刚经历过冠状病毒的人们肯定熟悉抢购，就是这个原理。
  </p>
  <p class="graf graf--p">
   他人的行为似乎就像是正确操作的 “证据”。这就是骗子经常使用 “先令” 的原因，骗子的帮手向受害者确认骗子的计划是合法的。
  </p>
  <p class="graf graf--p">
   以上这些说起来似乎很容易，但处在那样的环境中的人往往难以躲避欺骗，就算他们懂得一些骗术的道理。这就是为什么骗子很可怕，而且传统骗术也能经久不衰。
  </p>
  <p class="graf graf--p">
   追求事实核查是必须的。但是，如果每项事物都通过事实核查，那么包括政府在内的社会将无法正常运转。
  </p>
  <p class="graf graf--p">
   然而骗子们却在商业、政治和日常经验中年复一年地成长。
  </p>
  <p class="graf graf--p">
   互联网构成了一个另类事实的世界，与现实脱钩。宣传、操纵、误导、信息战 …… 各种骗术在这里盛行。除了帮助更多人了解这些骗术在外，也别无他法。不论如何，审查绝不能成为解决方案。
  </p>
