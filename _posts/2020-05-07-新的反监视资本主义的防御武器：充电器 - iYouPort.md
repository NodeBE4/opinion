  <p class="graf graf--p">
   奥地利林茨的艺术家兼编码员马丁·纳达尔（Martin Nadal）发明了
   <a class="markup--anchor markup--p-anchor" data-href="http://martinnadal.eu/fango/" href="http://martinnadal.eu/fango/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     FANGo
    </strong>
   </a>
   ，这是一种“反监视资本主义的防御武器”，被伪装成手机充电器。
  </p>
  <p class="graf graf--p">
   Nadal 在介绍该设备的
   <a class="markup--anchor markup--p-anchor" data-href="http://martinnadal.eu/fango/" href="http://martinnadal.eu/fango/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     页面上
    </strong>
   </a>
   解释说，充电器的内部隐藏着一个微控制器，它通过访问操作系统的 “调试模式” 来控制 Android 智能手机。
  </p>
  <p class="graf graf--p">
   然后，该设备进行查询并与 Google、Amazon、YouTube 和其他站点上的页面进行交互，以 “欺骗其数据捕获过程”。
  </p>
  <p class="graf graf--p">
   它的工作原理类似于假冒的 Apple
   <a class="markup--anchor markup--p-anchor" data-href="https://www.vice.com/en_us/article/evj4qw/these-iphone-lightning-cables-will-hack-your-computer" href="https://www.vice.com/en_us/article/evj4qw/these-iphone-lightning-cables-will-hack-your-computer" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     雷电电缆
    </strong>
   </a>
   （现已批量生产），一旦连接，它就会劫持您的设备。
  </p>
  <p class="graf graf--p">
   阻止广告商或数据经纪人进行跟踪尝试的工具并不是什么新工具 —— 比如
   <a class="markup--anchor markup--p-anchor" data-href="https://adnauseam.io/" href="https://adnauseam.io/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     AdNauseam
    </strong>
   </a>
   是一个可以点击所有广告的插件，而
   <a class="markup--anchor markup--p-anchor" data-href="https://trackmenot.io/" href="https://trackmenot.io/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     TrackmeNot
    </strong>
   </a>
   可以在不同的搜索引擎上进行随机搜索。但是，此类项目仅专注于台式机和Web浏览器。
  </p>
  <p class="graf graf--p">
   Nadal 在一封电子邮件中对母板说：“如今的人们主要通过移动设备与互联网进行交互。移动设备上还会使用很多应用程序，所有这些应用程序都不可能使用这些插件来阻碍监控，使用户很无奈。”
  </p>
  <p class="graf graf--p">
   这就是为什么创造了 FANGo。
  </p>
  <p class="graf graf--p">
   该设备的名称是 Facebook、Amazon、Netflix 和 Google 的首字母缩写，它们代表着世界上最赚钱的寡头公司 —— 也是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89/" href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     资本主义
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    的引擎
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    <strong>
     您可以在这里下载这本书
    </strong>
   </span>
   《监视资本主义时代》：
   <a class="markup--anchor markup--p-anchor" data-href="https://t.me/iyouport/6559" href="https://t.me/iyouport/6559" rel="nofollow noopener noreferrer" target="_blank">
    https://t.me/iyouport/6559
   </a>
  </p>
  <p class="graf graf--p">
   Nadal 说，“让我印象最深刻的是，监视资本主义以人类的经验为基础，将其转化为对我们行为的预测。这种方法特别邪恶，正因此创建 ‘反监视资本主义武器 ’的想法引起了我的兴趣。”
  </p>
  <p class="graf graf--p">
   因此，FANGo 的任务很简单：
   <strong class="markup--strong markup--p-strong">
    破坏监视机制，在提取的数据中添加噪声，并破坏数据的质量，以使其无法可靠地被使用或出售
   </strong>
   。
  </p>
  <p class="graf graf--p">
   您还可以通过其他方式污染监视资本家收集的数据，无论是在字面上的用无意义的方式填充 Facebook 帖子，还是浏览互联网同时留下
   <a class="markup--anchor markup--p-anchor" data-href="https://trackthis.link/" href="https://trackthis.link/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     虚假的数据足迹
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p">
   Nadal 正在使用地理欺骗等技术来添加可能破坏监视资本家搜集的新功能。他说：[这就可以做到]“虽然我的手机正在家里安静地充电，但数据经纪人会认为我正在城市或世界的另一部分散步或用餐。”
  </p>
  <p class="graf graf--p">
   首先必须说，FANGo 服务于移动设备的想法是非常重要的，在移动上网普及程度如此之高的今天，对移动设备的保护却少得可怜，这是不正常的。
  </p>
  <p class="graf graf--p">
   不过必须提及额外的担忧。随着虚拟世界用户量趋近饱和，监视资本主义正在全力瞄准现实世界中的物品 ——
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e6%99%ba%e8%83%bd%e9%97%b4%e8%b0%8d%e5%ae%b6%e5%b1%85-%e5%9f%8e%e5%b8%82/" href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e6%99%ba%e8%83%bd%e9%97%b4%e8%b0%8d%e5%ae%b6%e5%b1%85-%e5%9f%8e%e5%b8%82/" rel="noopener noreferrer" target="_blank">
    让你家中一切通电的东西都联网
   </a>
   ，以便更大程度上汲取数据。而且这件事已经很危险，见：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e5%ae%b6%e5%86%b0%e7%ae%b1%e5%a6%82%e4%bd%95%e4%bd%9c%e4%b8%ba%e5%91%88%e5%a0%82%e8%af%81%e4%be%9b%ef%bc%9f/" href="https://www.iyouport.org/%e4%bd%a0%e5%ae%b6%e5%86%b0%e7%ae%b1%e5%a6%82%e4%bd%95%e4%bd%9c%e4%b8%ba%e5%91%88%e5%a0%82%e8%af%81%e4%be%9b%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    你家冰箱如何作为呈堂证供
   </a>
   ？》
  </p>
  <p class="graf graf--p">
   并且，⚠️政府间谍已经开始行动了，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%a5%e8%89%b2%e5%88%97%e5%88%9b%e4%b8%9a%e5%85%ac%e5%8f%b8%e7%ad%b9%e9%9b%86%e4%ba%861250%e4%b8%87%e7%be%8e%e5%85%83%ef%bc%8c%e8%ae%a9%e6%94%bf%e5%ba%9c%e5%8f%af%e4%bb%a5%e7%a0%b4%e8%a7%a3/" href="https://www.iyouport.org/%e4%bb%a5%e8%89%b2%e5%88%97%e5%88%9b%e4%b8%9a%e5%85%ac%e5%8f%b8%e7%ad%b9%e9%9b%86%e4%ba%861250%e4%b8%87%e7%be%8e%e5%85%83%ef%bc%8c%e8%ae%a9%e6%94%bf%e5%ba%9c%e5%8f%af%e4%bb%a5%e7%a0%b4%e8%a7%a3/" rel="noopener noreferrer" target="_blank">
    以色列创业公司筹集了1250万美元，让政府可以破解物联网
   </a>
   》；甚至这件事可能任何人都可以做，方法：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%bf%ab%e9%80%9f%e6%89%be%e5%88%b0%e4%bb%bb%e4%bd%95%e5%9b%bd%e5%ae%b6%e9%9a%90%e7%a7%98%e7%9a%84%e5%85%b3%e9%94%ae%e5%9f%ba%e7%a1%80%e8%ae%be%e6%96%bd/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%bf%ab%e9%80%9f%e6%89%be%e5%88%b0%e4%bb%bb%e4%bd%95%e5%9b%bd%e5%ae%b6%e9%9a%90%e7%a7%98%e7%9a%84%e5%85%b3%e9%94%ae%e5%9f%ba%e7%a1%80%e8%ae%be%e6%96%bd/" rel="noopener noreferrer" target="_blank">
     如何快速找到任何国家隐秘的关键联网基础设施
    </a>
    》
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     《
    </strong>
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%90%9c%e8%bf%9c%e7%a8%8b%e6%a1%8c%e9%9d%a2%e3%80%81%e6%91%84%e5%83%8f%e5%a4%b4%e3%80%81%e6%89%93%e5%8d%b0%e6%9c%ba%e3%80%81%e5%ae%b6%e7%94%a8%e8%ae%be%e5%a4%87-%e7%ad%89%e4%b8%80%e5%88%87/" href="https://www.iyouport.org/%e6%90%9c%e8%bf%9c%e7%a8%8b%e6%a1%8c%e9%9d%a2%e3%80%81%e6%91%84%e5%83%8f%e5%a4%b4%e3%80%81%e6%89%93%e5%8d%b0%e6%9c%ba%e3%80%81%e5%ae%b6%e7%94%a8%e8%ae%be%e5%a4%87-%e7%ad%89%e4%b8%80%e5%88%87/" rel="noopener noreferrer" target="_blank">
     <strong class="markup--strong markup--li-strong">
      搜远程桌面、摄像头、打印机、家用设备 … 等一切的超级Shodan搜索：恐怖指南

    </a>
    <strong class="markup--strong markup--li-strong">

   </li>
  </ul>

   总之，反监视资本主义的斗争需要不断持续，直到能全面拆除这一体制。⚪️
