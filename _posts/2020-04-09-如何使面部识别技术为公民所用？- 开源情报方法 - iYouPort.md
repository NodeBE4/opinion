  <p class="graf graf--p">
   开源情报是一个术语，用于指代对一切公开来源数据进行的收集、评估和应用。
  </p>
  <p class="graf graf--p">
   引用中央情报局的话那就是：“
   <strong class="markup--strong markup--p-strong">
    信息不一定必须机密才有价值
   </strong>
   。无论是在浏览的博客中、在观看的广播中、还是在阅读的专业期刊中，都有无穷无尽的信息可以帮助我们了解世界……”。
  </p>
  <p class="graf graf--p">
   听着有点吓人吗？
   <strong class="markup--strong markup--p-strong">
    你的一举一动都是情报
   </strong>
   ？是的，它就是如此 —— 在有目的的人眼中。这就是为什么说你应该小心，要非常小心。
  </p>
  <p class="graf graf--p">
   在互联网上找到此类信息的最常见形式之一并不是纯文本，而是以照片形式存储的。
  </p>
  <p class="graf graf--p">
   传统上，这些图像必须由观看者手动过滤以确定和评估其内容，但是由于有了面部识别软件，整个过程可以完全自动化。
  </p>

   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab%e5%8f%8a%e5%af%b9%e6%8a%97%e6%96%b9%e6%b3%95/" href="https://www.iyouport.org/category/%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab%e5%8f%8a%e5%af%b9%e6%8a%97%e6%96%b9%e6%b3%95/" rel="noopener noreferrer" target="_blank">

   </a>

  </p>
  <pre class="graf graf--p">sudo apt-get update &amp;&amp; apt-get upgrade</pre>
  <p class="graf graf--p">
   虽然在某些发行版中默认包含了 Pip，但也可以通过运行以下命令来安装它。此命令将获取软件包的最新版本并进行安装。

  <pre class="graf graf--p">sudo apt-get install python-pip</pre>

     第二步：组织数据进行分析
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   用 Python 实现的用于面部识别的库意味着可以相对轻松地将其实现到其他应用程序中。
  </p>
  <p class="graf graf--p">
   也就是说
   <strong class="markup--strong markup--p-strong">
    可以更简单地将面部识别功能添加到网络监视摄像头实时流、网络爬虫、或其他以数字方式显示包含面部的图像或视频的格式
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是，这需要编写额外的原始代码。
  </p>

   相反，本教程将使用随附的命令行工具，并将其与其他系统实用程序结合起来，以执行类似的任务，而无需编写新的脚本。
  </p>
  <p class="graf graf--p">
   随附的命令行工具旨在处理两批文件。一个文件夹包含一组可识别的面孔，另一文件夹包含用户希望进行识别的一组图像。
  </p>

   Wget 可以从命令行进行操作，并且可以用于下载任何内容，从单个页面或图像到整个网站。如下所示的命令将跟随给定域中的每个链接，并下载该页面上所有可用的内容。
  </p>
  <p class="graf graf--p">
   可以随时按 Ctrl + C 退出此命令，否则它将一直运行到给定请求完成为止。
  </p>
  <pre class="graf graf--p">wget -p -r -E -e robots=off — convert-links -U mozilla — level 1 whitehouse.gov</pre>

  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     -p
    </strong>
    指定该工具还将下载其他内容，例如图像、样式表和脚本。
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     -E
    </strong>
    要求每个下载的文件都以其完整文件扩展保存，这将使您更容易过滤图像。
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     -r
    </strong>
    用于递归，因此也可以下载链接和其他目录。
   </li>
   <li class="graf graf--li">
    在字符串的后面，递归级别指定为
    <strong class="markup--strong markup--li-strong">
     -level1
    </strong>
    。这可以确保索引页面中的链接被跟踪一次。1可以改为其他想要的数字，也可以删除该标志以尝试无限递归。
   </li>
   <li class="graf graf--li">
    如果希望确保不从其他网站下载任何文件，则可以使用
    <strong class="markup--strong markup--li-strong">
     -domain
    </strong>
    ，后接该命令应限制到的域。
   </li>
   <li class="graf graf--li">
    递归请求可能会被阻止或在某些服务器上引起问题，因此您可能需要通过使用
    <strong class="markup--strong markup--li-strong">
     -w
    </strong>
    在请求之间创建延迟来限制递归或限制请求速度。
   </li>
   <li class="graf graf--li">
    使用
    <strong class="markup--strong markup--li-strong">
     -e robots=off
    </strong>
    可以避免将请求作为不受欢迎的Web爬网程序而被拒绝或阻止。
   </li>
   <li class="graf graf--li">
    出于相同的原因，
    <strong class="markup--strong markup--li-strong">
     -U mozilla
    </strong>
    用于指定应使用 Mozilla用户代理，以便该工具看起来像是Web浏览器。
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     -convert-links
    </strong>
    为了重新格式化链接，以便可以更轻松地离线使用该页面。虽然这对于筛选从站点下载的图像来说不是必需的，但它可以更便于追溯以确定图像在网页上的位置。
   </li>
  </ul>
  <p class="graf graf--p">
   一旦让程序运行了所需的时间，就可以在新目录中复制所有图像文件，以便更轻松地使用它们。首先，通过运行以下命令创建一个新目录。
  </p>
  <pre class="graf graf--p">mkdir unknown_faces</pre>
  <p class="graf graf--p">
   接下来，通过运行以下两个命令将所有从 Wget 请求中检索到的图像文件复制到此新目录中，第一个用于JPG文件，第二个用于PNG文件。
   <strong class="markup--strong markup--p-strong">
    urlfolder
   </strong>
   应替换为Wget创建的文件夹的名称，通常对应于初始命令中使用的URL或域。
  </p>
  <pre class="graf graf--p">cp urlfolder**/*.jpg unknown_faces/
cp urlfolder**/*.png unknown_faces/</pre>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    cp
   </strong>
   命令将复制所有匹配的文件，后跟目录参数，在本例中为 whitehouse.gov。此目录后的**（双星号）指定递归，对于任何JPG或PNG图像，后跟一个*（通配符）。最后一个参数指定应将文件复制到
   <strong class="markup--strong markup--p-strong">
    unknown_faces
   </strong>
   目录。

   <span style="color: #ff6600;">

     第五步：分析面部识别匹配

  </h3>
  <p class="graf graf--p">

    grep
   </strong>
   和
   <strong class="markup--strong markup--p-strong">
    results.txt

   文本文件来查看使用给定面孔进行的每次匹配。下面的命令将返回文本文件中用 Donald Trump 标识的每一行。
  </p>
  <pre class="graf graf--p">grep “Donald Trump” results.txt</pre>

  <p class="graf graf--p">
   还可以结合使用其他系统实用程序，通过运行以下命令来确定程序每个结果的频率。
  </p>
  <pre class="graf graf--p">cat results.txt | sed ‘s/.*,//’ | sort | uniq -c</pre>
  <p class="graf graf--p">
   首先，运行
   <strong class="markup--strong markup--p-strong">
    cat results.txt
   </strong>
   会将 results.txt 的内容作为标准输入读取。这被发送到
   <strong class="markup--strong markup--p-strong">
    sed
   </strong>
   ，在此删除每行之前（包括逗号）的所有内容。然后结果列表发送到
   <strong class="markup--strong markup--p-strong">
    sort
   </strong>
   。最后它用作
   <strong class="markup--strong markup--p-strong">
    uniq
   </strong>
   的输入，
   <strong class="markup--strong markup--p-strong">
    -c
   </strong>
   参数的结果是计算每行的出现次数。
  </p>

   </noscript>
  </p>
  <p class="graf graf--p">
   这将产生一个列表，其中每个结果均以其相应的频率列出，从而允许
   <strong class="markup--strong markup--p-strong">
    对较大的数据集进行简单分析
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果希望查看由面部识别工具识别的特定图像，则可以使用类似的技术。可以使用下面的命令来生成一个新的文本文件，是文件名的列表，由程序用给定的结果标记并保存在先前的输出文件中。
  </p>
  <pre class="graf graf--p">grep “Donald Trump” results.txt | sed ‘s/,.*$//’ &gt; identified.txt</pre>
  <p class="graf graf--p">
   在这种情况下，将使用 result.txt 文本文件中的
   <strong class="markup--strong markup--p-strong">
    grep
   </strong>
   搜索所有标识为 Donald Trump 的图像。 这些结果作为标准输入发送到
   <strong class="markup--strong markup--p-strong">
    sed
   </strong>
   ，在其中包括逗号在内的所有内容均从每一行中删除。最后，重定向运算符
   <strong class="markup--strong markup--p-strong">
    &gt;
   </strong>
   将输出发送到名为
   <strong class="markup--strong markup--p-strong">
    Identification.txt
   </strong>
   的文本文件。
  </p>
  <p class="graf graf--p">
   如果运行诸如
   <strong class="markup--strong markup--p-strong">
    head Identification.txt
   </strong>
   之类的命令，将看到该文件仅包含文件名。可以将此文件名列表用作简短脚本的输入，以将这些文件中的每个文件复制到新文件夹中。
  </p>
  <p class="graf graf--p">
   可以通过运行以下命令来使用 Nano 创建一个新的脚本文件。
  </p>
  <p class="graf graf--p">

  <pre class="graf graf--p">#!/bin/bash
mkdir identified
while read line; do

   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     #!/bin/bash
    </strong>
    声明该脚本用于Bash。
   </li>
   <li class="graf graf--li">
    第二行创建一个新目录，命名为
    <strong class="markup--strong markup--li-strong">
     identified
    </strong>
    ，所有肯定标识的图像都将移至该目录。
   </li>
   <li class="graf graf--li">
    第三行与最后一行结合使用，读取文本文件
    <strong class="markup--strong markup--li-strong">
     identified.txt
    </strong>
    作为输入，以便在 while 语句中单独修改文件的每一行。
   </li>
   <li class="graf graf--li">
    第四行创建一个名为
    <strong class="markup--strong markup--li-strong">
     filename

    <strong class="markup--strong markup--li-strong">
     sed
    </strong>
    删除每行中斜杠之前的所有字符。
   </li>
   <li class="graf graf--li">

    <strong class="markup--strong markup--li-strong">
     identified
    </strong>
    目录并分配其原始文件名。
   </li>

   <strong class="markup--strong markup--p-strong">
    identified.sh

  <pre class="graf graf--p">chmod +x identified.sh</pre>
  <p class="graf graf--p">

   <strong class="markup--strong markup--p-strong">
    ./identified.sh

   并按 Enter / return 键即可运行脚本本身。
