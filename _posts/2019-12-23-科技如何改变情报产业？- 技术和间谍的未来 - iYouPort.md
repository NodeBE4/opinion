  <p class="graf graf--p">
   国家情报局前局长丹·科茨（Dan Coats）在2018年举行的关于情报机构未来挑战的会议上称，他认为美国情报界的转型必须是一场革命，而不仅仅是一种进化。
  </p>
  <p class="graf graf--p">
   他称，情报社区必须具有创新性和灵活性，
   <strong class="markup--strong markup--p-strong">
    能够在可能出现的任何地方迅速采用创新技术
   </strong>
   。
  </p>
  <p class="graf graf--p">
   情报界现在正处于十字路口上：人工智能、大数据、机器人技术、物联网和区块链等技术的不断扩散改变了游戏规则。
  </p>
  <p class="graf graf--p">
   这些技术的普及（其中大多数是民用技术）很容易造成数据泄露，并给情报机构造成后门威胁。
  </p>
  <p class="graf graf--p">
   此外，由于这些技术的价格适中且无处不在，因此很容易被用于恶意目的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    近年来的技术突破已导致情报组织面临极大挑战
   </strong>
   。这些组织传统的分层、分隔、产业结构现在正在发生变化，主要围绕新技术与传统情报工作的集成以及人类在情报过程中的角色的重新定义。
  </p>
  <p class="graf graf--p">
   在这里看到
   <strong class="markup--strong markup--p-strong">
    情报的分类、历史演化，以及它们各自代表什么，尤其是对开源情报的发展介绍
   </strong>
   《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e5%9c%b0%e5%9b%be%e6%98%af%e6%af%94%e8%a9%b9%e5%a7%86%e6%96%af%e9%82%a6%e5%be%b7%e6%9b%b4%e6%a3%92%e7%9a%84%e9%97%b4%e8%b0%8d/" href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e5%9c%b0%e5%9b%be%e6%98%af%e6%af%94%e8%a9%b9%e5%a7%86%e6%96%af%e9%82%a6%e5%be%b7%e6%9b%b4%e6%a3%92%e7%9a%84%e9%97%b4%e8%b0%8d/" rel="noopener noreferrer" target="_blank">
    “谷歌地图”是比詹姆斯邦德更棒的间谍
   </a>
   》。
  </p>
  <p class="graf graf--p">
   以
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    开源情报
   </a>
   （OSINT）为例。
  </p>
  <p class="graf graf--p">
   传统上，这种信息被认为“不如”机密信息。结果是与其他类型的技术和资源相比，对 OSINT 技术的投资大大降低了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    然而，现在这种情况正在逆转；情报机构现在已经意识到，与其他（更具挑战性）的信息类型相比，OSINT 易于获取，并且可能更有价值
   </strong>
   。
  </p>
  <p class="graf graf--p">
   不过这种理解只是昙花一现，因为情报机构对OSINT的使用仍然涉及繁琐的过程，包括未机密和已机密的IT环境的缓慢而复杂的集成（体现出民间独立开源情报调查的优势？至少没有官僚机构那么多复杂的繁琐流程）。
  </p>
  <p class="graf graf--p">
   因此，毫不奇怪，情报主管（例如，国务院情报部门负责人或被任命为国家侦察办公室主任的提名人）最近辩称，情报社区最大的挑战之一是如何将 OSINT 快速有效地整合到运营中。
  </p>
  <p class="graf graf--p">
   确实，技术创新一直是情报行业的中心。
  </p>
  <p class="graf graf--p">
   但是，当涉及到处理、分析、解释和对情报采取行动时，人类的能力（尽管有其所有局限性）始终被毫无疑问地认为是优越的 —— 就算是开源情报，也需要大量的人力工作。
   <strong class="markup--strong markup--p-strong">
    机器无法有人类那种慎密的思考
   </strong>
   。
  </p>
  <p class="graf graf--p">
   显然，数据和数据源的激增需要更好的优先级排序和分析系统。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    谁应该具有至高无上的地位呢？人还是机器？
   </strong>
  </p>
  <figure aria-describedby="caption-attachment-9380" class="wp-caption aligncenter" id="attachment_9380" style="width: 2048px">

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    ⚠️大数据为间谍行业而来
   </strong>
  </p>
  <p class="graf graf--p">
   辩论汹涌。资深情报人士声称，人类的判断无可替代。
  </p>
  <p class="graf graf--p">
   他们认为，人工智能将永远无法理解战略决策中的所有考虑因素，并且，人工智能无法评估人类行为解释中的抽象问题。
  </p>
  <p class="graf graf--p">
   机器可以收集数据并可能识别模式，但是它们在像人类一样解释现实方面将永远不会成功。
  </p>
  <p class="graf graf--p">
   对此，见简要分析《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%81%9c%e6%ad%a2%e5%b0%86%e9%82%a3%e4%ba%9b%e4%b8%9c%e8%a5%bf%e7%a7%b0%e4%b8%ba%e4%ba%ba%e5%b7%a5%e6%99%ba%e8%83%bd/" href="https://www.iyouport.org/%e5%81%9c%e6%ad%a2%e5%b0%86%e9%82%a3%e4%ba%9b%e4%b8%9c%e8%a5%bf%e7%a7%b0%e4%b8%ba%e4%ba%ba%e5%b7%a5%e6%99%ba%e8%83%bd/" rel="noopener noreferrer" target="_blank">
    停止将那些东西称为”人工智能”
   </a>
   》。
  </p>
  <p class="graf graf--p">
   其他人也警告说，在生死攸关的情况下，例如 决定是否开战时，依靠机器会产生伦理问题。
  </p>
  <p class="graf graf--p">
   相比之下，技术优越主义者则声称，人类的优势已经在技术面前屈服，虽然是人类优势定义了上个世纪的情报活动。
  </p>
  <p class="graf graf--p">
   尽管人类仍然很重要，但他们的角色不再是排他性的，甚至可能不是最重要的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    一般的情报人员如何应对现代世界产生的无休止的庞大信息量？
   </strong>
  </p>
  <p class="graf graf--p">
   从1995年到2016年，覆盖低优先级国家/地区的美国情报研究人员的平均阅读量，每天从20,000字增加到200,000字。
  </p>
  <p class="graf graf--p">
   这仅仅是开始。根据预测，到2025年人类将产生的数字数据量将是今天产生的数字量的
   <strong class="markup--strong markup--p-strong">
    十倍
   </strong>
   。
  </p>
  <p class="graf graf--p">
   有人认为，这个信息量只能由计算机来处理和分析。
  </p>
  <p class="graf graf--p">
   当然，
   <strong class="markup--strong markup--p-strong">
    最热衷于将机器集成到情报工作中的倡导者并没有完全消除人类的参与。即使是最怀疑的人也不会怀疑将人工智能集成到情报活动中的必要性
   </strong>
   。
  </p>
  <p class="graf graf--p">
   辩论集中在谁将帮助谁的问题上：机器帮助人类、还是人类帮助机器。
  </p>
  <p class="graf graf--p">
   多数内部人士认为，将情报界带入21世纪的关键在于打破组织间和组织内的隔离墙，包括 国家安全机构内的服务；在公共部门、私营部门和学术界之间；以及不同国家的情报部门之间。
  </p>
  <p class="graf graf--p">
   因此，毫不奇怪，推动技术创新是当前情报革命的一部分。
  </p>
  <p class="graf graf--p">
   间谍部门已经认识到私营部门和学术界是技术创新的主要驱动力。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    私人情报和国家情报
   </strong>
  </p>
  <p class="graf graf--p">
   在美国，这些机构与安全界之间存在动态合作，包括
   <strong class="markup--strong markup--p-strong">
    由政府和私人公司共同拥有的风险投资基金
   </strong>
   。
  </p>
  <p class="graf graf--p">
   以In-Q-Tel为例，这是成立于20年前的风险投资基金，用于挑选和投资开发可为美国国家安全服务的创新技术的公司，从而将美国情报界置于技术发展的最前沿。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    该基金是一家独立公司，不隶属于任何政府机构，但是，它与中央情报局保持持续协调，美国政府是其主要投资者。
   </strong>
  </p>
  <p class="graf graf--p">
   数据集成和知识管理提供商 Palantir 是最成功的尝试，尽管颇有争议，但如今已发展成为数十亿美元的大公司。
  </p>
  <p class="graf graf--p">
   但是还有很多其他初创公司和更成熟的公司，范围从
   <strong class="markup--strong markup--p-strong">
    复杂的化学检测
   </strong>
   （例如 908devices）、
   <strong class="markup--strong markup--p-strong">
    自动语言翻译
   </strong>
   （例如 Lilt）、和
   <strong class="markup--strong markup--p-strong">
    数字图像
   </strong>
   （例如 mmersive Wisdom），到
   <strong class="markup--strong markup--p-strong">
    传感器技术
   </strong>
   （例如 Echodyne）、
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a4%be%e4%bc%9a%e4%bf%a1%e7%94%a8%e7%9b%91%e8%a7%86%ef%bc%9a%e8%8b%b1%e5%9b%bd%e3%80%81%e4%b8%ad%e5%9b%bd%e3%80%81%e7%be%8e%e5%9b%bd%e5%92%8c%e6%97%a5%e6%9c%ac%e3%80%90viedo%e3%80%91/" href="https://www.iyouport.org/%e7%a4%be%e4%bc%9a%e4%bf%a1%e7%94%a8%e7%9b%91%e8%a7%86%ef%bc%9a%e8%8b%b1%e5%9b%bd%e3%80%81%e4%b8%ad%e5%9b%bd%e3%80%81%e7%be%8e%e5%9b%bd%e5%92%8c%e6%97%a5%e6%9c%ac%e3%80%90viedo%e3%80%91/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     预测分析
    </strong>
   </a>
   （例如 Tamr） 、和
   <strong class="markup--strong markup--p-strong">
    网络安全
   </strong>
   （例如 Interset）。
  </p>
  <p class="graf graf--p">
   实际上，无论大小，这类公司已经在情报工作中占了很大一部分。 Hexagon、Nice、Splunk、Cisco 和 NEC 等公司为情报和执法机构提供了一整套平台和服务，包括各种分析解决方案，例如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.openpr.com/wiki/video-content" href="https://www.openpr.com/wiki/video-content" rel="noopener noreferrer" target="_blank">
    视频分析
   </a>
   、身份分析和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%80%e4%b9%88%e6%98%af%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%83%85%e6%8a%a5%ef%bc%9f%e4%bd%a0%e7%9a%84%e6%8e%a8%e7%89%b9%e5%a6%82%e4%bd%95%e6%88%90%e4%b8%ba%e6%8a%93%e6%8d%95%e4%bd%a0%e7%9a%84/" href="https://www.iyouport.org/%e4%bb%80%e4%b9%88%e6%98%af%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%83%85%e6%8a%a5%ef%bc%9f%e4%bd%a0%e7%9a%84%e6%8e%a8%e7%89%b9%e5%a6%82%e4%bd%95%e6%88%90%e4%b8%ba%e6%8a%93%e6%8d%95%e4%bd%a0%e7%9a%84/" rel="noopener noreferrer" target="_blank">
    社交媒体分析
   </a>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这些平台通过使用实时数据流分析和机器学习，帮助间谍机构从收集到的历史数据中获取见解并做出预测
   </strong>
   。一站式的情报商店，如果您愿意的话。
  </p>
  <p class="graf graf--p">
   政府与非政府机构合作的另一个例子是情报高级研究计划活动（IARPA），这是一个向国家情报局局长（DNI）汇报的非营利组织。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    IARPA 成立于2006年，致力于资助与美国情报界有关的高级研究，重点是广泛的技术和社会科学领域的学术机构和私营部门之间的合作
   </strong>
   。
  </p>
  <p class="graf graf--p">
   该基金的年度运营预算相对较小，约为30亿美元，因此优先考虑满足情报界具体需求的多年发展项目。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9382 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-7.jpg?resize=1100%2C619&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-7.jpg?w=1400&amp;ssl=1 1400w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-7.jpg?resize=300%2C169&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-7.jpg?resize=768%2C432&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-7.jpg?resize=1024%2C576&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-7.jpg?resize=1100%2C619&amp;ssl=1 1100w" data-recalc-dims="1" height="619" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-7.jpg?resize=1100%2C619&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9382" data-recalc-dims="1" height="619" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-7.jpg?resize=1100%2C619&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-7.jpg?w=1400&amp;ssl=1 1400w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-7.jpg?resize=300%2C169&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-7.jpg?resize=768%2C432&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-7.jpg?resize=1024%2C576&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-7.jpg?resize=1100%2C619&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    挑战情报行业的政府霸权
   </strong>
  </p>
  <p class="graf graf--p">
   这些都是令人兴奋的机会。但是，未来对于情报机构来说将面临一些挑战：
  </p>
  <p class="graf graf--p">
   首先，情报界在收集、处理和分发数据方面失去了首要地位。直到最近，这些组织存在的理由还是首先要获得有关敌人的信息，然后才能说出敌人可以掩盖这些信息。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    今天有很多信息可供使用，并且有
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%f0%9f%92%8e-%e8%b6%85%e7%ba%a7%e6%83%85%e6%8a%a5%e6%94%b6%e9%9b%86%e5%b7%a5%e5%85%b7%e5%ba%93%ef%bc%9a%e5%bc%80%e6%ba%90%e9%aa%8c%e8%af%81%e5%92%8c%e8%b0%83%e6%9f%a5%e5%b7%a5%e5%85%b7%e5%8f%8a/" href="https://www.iyouport.org/%f0%9f%92%8e-%e8%b6%85%e7%ba%a7%e6%83%85%e6%8a%a5%e6%94%b6%e9%9b%86%e5%b7%a5%e5%85%b7%e5%ba%93%ef%bc%9a%e5%bc%80%e6%ba%90%e9%aa%8c%e8%af%81%e5%92%8c%e8%b0%83%e6%9f%a5%e5%b7%a5%e5%85%b7%e5%8f%8a/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     大量现成的工具
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    （其中很多是免费的工具）允许包括个人在内的所有各方收集、处理和分析大量数据
   </strong>
   。
  </p>
  <p class="graf graf--p">
   只需看一下IBM的 i2 Analyst Notebook，它可以为分析师提供数千美元的
   <strong class="markup--strong markup--p-strong">
    多维可视化分析功能
   </strong>
   ，以便分析师可以快速发现数据中隐藏的关系和模式；
  </p>
  <p class="graf graf--p">
   请注意，直到最近，这种能力只属于政府组织。
  </p>
  <p class="graf graf--p">
   情报组织面临的第二个挑战在于，信息本身的性质及其多种不同格式，以及通常是独立且缺乏标准化的收集和处理系统。
  </p>
  <p class="graf graf--p">
   结果是，很难将所有可用信息合并到一个情报产品中。
  </p>
  <p class="graf graf--p">
   因此，情报组织正在开发强调合作与权力下放的概念和结构。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    私人情报市场提供了多种用于合并信息的工具；从简单的现成解决方案到支持复杂组织过程的复杂工具。
   </strong>
  </p>
  <p class="graf graf--p">
   某些工具可以购买和快速实施，例如数据和知识共享与管理平台，而其他工具则由组织自己开发以满足其特定需求。
  </p>
  <p class="graf graf--p">
   第三个挑战与情报优先原则的改变有关。
  </p>
  <p class="graf graf--p">
   在过去，有关给定目标的信息的收集需要做出特定的决定，并为此目的分配专用资源，这通常是以将资源分配给不同目标为代价的。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    在当下这个信息量无限、几乎无限制地访问信息、先进的数据存储功能和操纵数据的时代
   </strong>
   ，情报机构现在可以大规模收集和存储信息，而无需立即进行处理 —— 可以根据需要进行处理。
  </p>
  <p class="graf graf--p">
   这种发展带来了其他挑战，包括：需要在需要时快速查明相关信息；快速处理信息；识别模式，并从大量数据中得出结论；还要使所生产的知识可供情报消费者使用。
  </p>
  <p class="graf graf--p">
   因此，毫不奇怪，情报领域中的大多数技术进步都能应对这些挑战，通常将实时技术与诸如具有人工智能的大数据、先进的信息存储功能、和先进的信息可视化呈现等技术结合在一起。
  </p>
  <p class="graf graf--p">
   最后，情报机构是根据工业时代高峰期提出的概念来建立和运行的，该概念倡导线性和周期性装配线的原理。
  </p>
  <p class="graf graf--p">
   情报周期的线性模型 —— 收集、处理、研究、分配和反馈 —— 变得不那么重要了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在这个新时代，各种情报职能之间以及情报组织及其生态系统之间的界限越来越模糊
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9383 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-6.jpg?resize=1100%2C619&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-6.jpg?w=1400&amp;ssl=1 1400w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-6.jpg?resize=300%2C169&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-6.jpg?resize=768%2C432&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-6.jpg?resize=1024%2C576&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-6.jpg?resize=1100%2C619&amp;ssl=1 1100w" data-recalc-dims="1" height="619" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-6.jpg?resize=1100%2C619&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9383" data-recalc-dims="1" height="619" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-6.jpg?resize=1100%2C619&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-6.jpg?w=1400&amp;ssl=1 1400w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-6.jpg?resize=300%2C169&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-6.jpg?resize=768%2C432&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-6.jpg?resize=1024%2C576&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-6.jpg?resize=1100%2C619&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    情报新世界
   </strong>
  </p>
  <p class="graf graf--p">
   因此，需要新的情报工作顺序，情报组织目前正在重新定义过程中。
  </p>
  <p class="graf graf--p">
   传统分区 —— 例如在收集和研究之间；内部安全组织和积极情报；公共和私营部门 —— 都已过时。
  </p>
  <p class="graf graf--p">
   这已经不是进行结构改革的另一种尝试了：存在认识论破裂的感觉，需要重新定义学科、情报组织与环境之间的关系（从决策者到普通大众）以及开发新的结构和构想。
  </p>
  <p class="graf graf--p">
   当然，还有更广泛的担忧。立法者需要创建一个法律框架，以一种能够将这些技术的预测方面考虑在内的方式，准确地合并基于数据的评估，同时仍要保护尊重这些概念的民族国家的公民的隐私权和安全权利。
  </p>
  <p class="graf graf--p">
   尽管人们认识到它们周围正在发生深刻的变化，但
   <strong class="markup--strong markup--p-strong">
    今天的情报机构仍然本着冷战构想的精神建立和运作。
   </strong>
  </p>
  <p class="graf graf--p">
   从某种意义上说，情报组织没有内部化现今特征的复杂性，这种复杂性要求*放弃*对情报机构的（内外）二分化理解，以及对情报企业和政府机构的理解；这些东西在如今的去中心化、网络化和繁荣的时代已经是过时的概念。
  </p>
  <p class="graf graf--p">
   尽管有些人怀疑情报机构是否具有改变和适应未来挑战的能力，但毫无疑问，他们必须在这个由速度和相关性决定谁胜出的时代继续存在。⚪️
  </p>

    How tech is transforming the intelligence industry
