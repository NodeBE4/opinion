  <p class="graf graf--p">
   芬兰数据保护主管表示，他将调查诺基亚所有者HMD，因为有人声称其手机已向中国国有服务器发送数据。
  </p>
  <p class="graf graf--p">
   该调查是在挪威公共广播公司 NRK 发布的一份报告之后启动的，该报告称诺基亚7 Plus手机可能已将用户的敏感信息发送到中国的服务器数月。
  </p>
  <p class="graf graf--p">
   NRK 在接到一位名叫 Henrik Austad 的用户提供的信息后进行了调查，他监视着诺基亚7的流量 — 发现它在开机时向中国服务器“
   <a class="markup--anchor markup--p-anchor" data-href="http://vnet.cn/" href="http://vnet.cn/" rel="noreferrer noopener" target="_blank">
    vnet.cn
   </a>
   ”
   <strong class="markup--strong markup--p-strong">
    发送了未加密的信息
   </strong>
   。 据报道，这些数据包括他的位置、SIM 卡号和手机的序列号。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    该信息包括电话和附加电话号码的可识别数据，以及与之相关的基站，这使得数据接收者以及在此过程中能够访问流量的任何人都能够跟踪手机的实时动作。- NRK
   </em>
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-2962 jetpack-lazy-image" data-lazy-sizes="(max-width: 1067px) 100vw, 1067px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-63.png?resize=1067%2C366&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-63.png?w=1067&amp;ssl=1 1067w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-63.png?resize=300%2C103&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-63.png?resize=768%2C263&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-63.png?resize=1024%2C351&amp;ssl=1 1024w" data-recalc-dims="1" height="366" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-63.png?resize=1067%2C366&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1067"/>

    <img alt="" class="aligncenter size-full wp-image-2962" data-recalc-dims="1" height="366" sizes="(max-width: 1067px) 100vw, 1067px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-63.png?resize=1067%2C366&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-63.png?w=1067&amp;ssl=1 1067w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-63.png?resize=300%2C103&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-63.png?resize=768%2C263&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-63.png?resize=1024%2C351&amp;ssl=1 1024w" width="1067"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   当 NRK 试图追踪信息发送到的 URL 的所有权信息时，他们找到了以下的内容：
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-2963 jetpack-lazy-image" data-lazy-sizes="(max-width: 900px) 100vw, 900px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-64.png?resize=900%2C358&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-64.png?w=900&amp;ssl=1 900w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-64.png?resize=300%2C119&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-64.png?resize=768%2C305&amp;ssl=1 768w" data-recalc-dims="1" height="358" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-64.png?resize=900%2C358&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="900"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-2963" data-recalc-dims="1" height="358" sizes="(max-width: 900px) 100vw, 900px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-64.png?resize=900%2C358&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-64.png?w=900&amp;ssl=1 900w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-64.png?resize=300%2C119&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-64.png?resize=768%2C305&amp;ssl=1 768w" width="900"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    在登记册中，“中国互联网网络信息中心”（CNNIC）被列为联络点。该机构负责顶级域名 .cn。当我们联系CNNIC时，我们被告知国有电信公司中国电信是该域名的所有者。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    通过中国消息来源，我们试图找到谁处理了服务器的答案，但中国电信的客户服务经理表示无法找到相关信息。然而，快速谷歌搜索带来了进一步的发展：
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    在开发人员的共享和协作平台 Github 上有一个开放的配置文件，
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     美国电子制造商高通公司
    </em>
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://raw.githubusercontent.com/bcyj/android_tools_leeco_msm8996/f242858a8cbb2c9ad9b5b2e53d75380ae775727a/qrdplus/ChinaTelecom/apps/AutoRegistration/src/com/qualcomm/qti/autoregistration/RegistrationTask.java" href="https://raw.githubusercontent.com/bcyj/android_tools_leeco_msm8996/f242858a8cbb2c9ad9b5b2e53d75380ae775727a/qrdplus/ChinaTelecom/apps/AutoRegistration/src/com/qualcomm/qti/autoregistration/RegistrationTask.java" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     <em class="markup--em markup--p-em">
      提供代码
     </em>
    </strong>
   </a>
   <em class="markup--em markup--p-em">
    。此代码包含的注册过程与诺基亚7 Plus 使用的方法非常相似。该代码收集了大量有关手机的数据，并以相同的格式将信息发送到我们从提示中收到的同一台服务器
   </em>
   。 — NRK
  </p>
  <p class="graf graf--p">
   据路透社报道，芬兰的数据保护监察官 Reijo Aarnio 将调查涉及“个人信息以及是否有法律依据”的任何违规行为。
  </p>
  <p class="graf graf--p">
   芬兰创业公司 HMD Global 于2016年与微软签署了诺基亚品牌的十年牌照，据报道，NRK承认，一批诺基亚7手机已向中国发送数据。该公司表示已修复大多数客户已安装的1月份软件更新中的“错误”。 HMD 声称这些手机没有发送任何可以识别其所有者的个人数据。诺基亚7本身是2017年10月推出的中国专供手机。第二代版本诺基亚7.1在一年后在美国发布。
  </p>
  <p class="graf graf--p">
   针对欧盟去年实施的更严格的隐私法，Aarnio 告诉 NRK，他的第一反应是“这至少违反了GDPR立法”。今年早些时候谷歌已经违反了法国的指导方针，因为其涉嫌不透明的数据“同意”政策而被罚款5000万欧元（约合5700万美元）。 — Engadget
  </p>
  <p class="graf graf--p">
   诺基亚7Plus于2018年第一季推出，立即取得了成功，
   <strong class="markup--strong markup--p-strong">
    在中国前五分钟就销售了25万部手机
   </strong>
   。
  </p>
  <p class="graf graf--p">
   中招的人中有您吗？
