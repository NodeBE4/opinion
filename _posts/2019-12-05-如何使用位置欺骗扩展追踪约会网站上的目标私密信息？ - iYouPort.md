  <p class="graf graf--p">
   约会网站允许您在非常亲密的环境中看清楚一个人，于是对于追踪调查来说，它是个极为强大的渠道。
  </p>
  <p class="graf graf--p">
   这些配置文件中包含的信息通常无法在其他地方找到，从而可以独特地了解一个人的私生活。
  </p>
  <p class="graf graf--p">
   如果您知道有人在那里闲逛，您可以在Chrome浏览器中使用GPS欺骗扩展程序，以挖掘对方的 Tinder 个人资料。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    从开源情报（OSINT）研究员、社交工程师或黑客进行侦察的角度来看，约会网站是可搜索数据库的独特集合
   </strong>
   。
  </p>
  <p class="graf graf--p">
   通过约会网站研究目标可以揭示对黑客和社交工程有用的多种信息。
  </p>
  <p class="graf graf--p">
   视频：
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/b7bpXlRHgqY" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   人们经常会使用相同的昵称，于是您通常可以通过查找常用的别名来跟踪互联网上的某个人。
  </p>
  <p class="graf graf--p">
   由于许多用户还链接他们的 Instagram 个人资料，如果他们倾向于重复使用相同的昵称，那么拦截昵称可能是调查某个人整个在线存在的关键。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Google 图片搜索功能强大
   </strong>
  </p>
  <p class="graf graf--p">
   您可以用来深入了解一个人生活的另一个因素是 Tinder 上的个人资料图片。
  </p>
  <p class="graf graf--p">
   约会应用程序允许发布很多照片，因此提供了许多机会使用反转图像搜索每张个人资料照片以查找此人的其他帐户，即使他们有不同的昵称。
  </p>
  <p class="graf graf--p">
   最后，当涉及到个人描述时，人们往往不会非常有创意。
  </p>
  <p class="graf graf--p">
   通常他们只是复制粘贴同样的内容，于是你可以很容易地根据文本中存在的相同的错误或措辞特征，在其他网站上找到它们。
  </p>
  <p class="graf graf--p">
   大多数人在打字时都会犯错，当他们重复部分文本时会将这些错误与其他帐户联系起来，并且会在常规 Google 搜索中找到同样独特的错误。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8789 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-23.jpg?resize=1100%2C589&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-23.jpg?w=1456&amp;ssl=1 1456w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-23.jpg?resize=300%2C161&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-23.jpg?resize=768%2C411&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-23.jpg?resize=1024%2C548&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-23.jpg?resize=1100%2C589&amp;ssl=1 1100w" data-recalc-dims="1" height="589" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-23.jpg?resize=1100%2C589&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    约会简介比你想象的更公开
   </strong>
  </p>
  <p class="graf graf--p">
   虽然 Tinder 长期以来一直作为移动应用程序提供，但大多数用户并不知道该网站现在允许从电脑访问。
  </p>
  <p class="graf graf--p">
   于是你可以使用此权限在 Google Chrome 中快速欺骗自己的位置，从而使调查人员能够伪装成在其所期望的用户所在位置附近，并开始对个人资料进行过滤。
  </p>
  <p class="graf graf--p">
   由于您可以从任何地方轻松完成此任务，因此
   <strong class="markup--strong markup--p-strong">
    开始此类攻击所需的唯一信息
   </strong>
   就是了解目标的外观以及他们居住、工作或闲逛的一般概况。
  </p>
  <p class="graf graf--p">
   反过来看，如果你是目标，你被了解得越多也就越危险。
  </p>
  <p class="graf graf--p">
   Tinder 允许您在Web浏览器中进行一些基本的调查，即使初学者也不难开始。
  </p>
  <p class="graf graf--p">
   要开始使用，您需要下载并安装 Google Chrome 浏览器，以及名为
   <a class="markup--anchor markup--p-anchor" data-href="https://chrome.google.com/webstore/detail/manual-geolocation/jpiefjlgcjmciajdcinaejedejjfjgki?hl=en" href="https://chrome.google.com/webstore/detail/manual-geolocation/jpiefjlgcjmciajdcinaejedejjfjgki?hl=en" rel="noopener noreferrer" target="_blank">
    Manual Geolocation
   </a>
   的扩展程序。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第一步：安装位置欺骗扩展
   </strong>
  </p>
  <p class="graf graf--p">
   让你的浏览器学会撒谎。这很重要，因为它使您能够搜索特定区域的一个或多个人。
  </p>
  <p class="graf graf--p">
   如果您的目标只是识别隶属公司A的员工，那么将自己的位置放到公司A总部，就可以直接从 Tinder 上找到您想要找的人。
  </p>
  <p class="graf graf--p">
   为此，需要从
   <a class="markup--anchor markup--p-anchor" data-href="https://chrome.google.com/webstore/detail/manual-geolocation/jpiefjlgcjmciajdcinaejedejjfjgki?hl=en" href="https://chrome.google.com/webstore/detail/manual-geolocation/jpiefjlgcjmciajdcinaejedejjfjgki?hl=en" rel="noopener noreferrer" target="_blank">
    Chrome 网上应用商店
   </a>
   下载 Manual Geolocation 扩展程序。
  </p>
  <p class="graf graf--p">
   添加该扩展程序并接受所需的任何权限，您会在附加组件栏的右上角看到一个图标。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8790 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-12.jpg?resize=1100%2C696&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-12.jpg?w=1100&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-12.jpg?resize=300%2C190&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-12.jpg?resize=768%2C486&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-12.jpg?resize=1024%2C648&amp;ssl=1 1024w" data-recalc-dims="1" height="696" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-12.jpg?resize=1100%2C696&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8790" data-recalc-dims="1" height="696" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-12.jpg?resize=1100%2C696&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-12.jpg?w=1100&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-12.jpg?resize=300%2C190&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-12.jpg?resize=768%2C486&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-12.jpg?resize=1024%2C648&amp;ssl=1 1024w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第二步：伪装你的位置到目标的位置
   </strong>
  </p>
  <p class="graf graf--p">
   点击 Manual Geolocation 图标将打开一个地图，允许您自己选择浏览器将向其访问的任何网站报告的位置，在本例中为 Tinder。
  </p>
  <p class="graf graf--p">
   双击地图上的位置，选择目标所在的位置。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8791 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-13.jpg?resize=1100%2C583&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-13.jpg?w=1193&amp;ssl=1 1193w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-13.jpg?resize=300%2C159&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-13.jpg?resize=768%2C407&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-13.jpg?resize=1024%2C542&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-13.jpg?resize=1100%2C583&amp;ssl=1 1100w" data-recalc-dims="1" height="583" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-13.jpg?resize=1100%2C583&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   您可以使用鼠标移动调整以优化您的位置。最后，单击窗口右上角的开/关以打开扩展。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第三步：从欺骗位置注册一个Tinder帐户
   </strong>
  </p>
  <p class="graf graf--p">
   现在，不论您访问任何网站，都会将此虚构位置信息作为你的真实位置读取，而 Tinder 将认为您实际上位于那个指定的位置。
  </p>
  <p class="graf graf--p">
   导航到 Tinder 并等到下面的页面加载。

   </li>

  </p>

   对于一些人来说，在现实中和在数字世界中的形象有一些差别，但约会网站中找到的数据往往能衔接现实和网络。
  </p>
  <p class="graf graf--p">
   希望有更多中国行动者参与到开源情报调查工作中来。不仅仅是因为开源情报市场已经数十亿美元、相关工作都是高薪的；更有，中国真的需要挖掘真相的民间独立组织。⚪️
  </p>
