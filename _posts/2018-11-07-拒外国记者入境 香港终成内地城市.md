 </p>
 <p>
 </p>
 <p>
  <span style="font-size: 12pt;">
   香港政府十月份拒绝延续英国金融时报亚洲版编辑马凯的媒体记者签证，引起了全球媒体的关注。事实上，作为中国特别行政区、实行所谓一国两制的香港，近年来多次拒绝所谓外国知名人物进入香港。有外媒认为，在回归二十一年之后，香港终于变得更像是中国大陆的一个城市。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   曾被誉为东方之珠的香港，在只有一千平方公里面积的土地上创出了经济奇迹。由于大批制造企业搬到珠江三角洲，经济专家认为，香港如要维持其经济表现，必须维持三个中心的地位：国际金融中心、国际物流和商贸服务中心，以及国际信息中心。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   国际上几乎所有的大型媒体，都在香港设有办事处，以利用香港信息自由的特点，报道中国大陆发生的事件。以前，这些常常报道中国所谓“负面新闻”的记者，从未惹上任何麻烦。但金融时报驻香港的亚洲版编辑马凯被拒签证之后，所有外媒记者都开始担心，香港是否还是一个 “自由港”。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   今年早些时候，作为香港外国记者俱乐部副主席的马凯，拒绝了中国政府和香港特区政府的干预，坚持邀请支持独立的香港民族党的陈浩天前往演讲。因此，他的签证被拒，显然是香港政府的报复行动。虽然香港特首陈方安生否认此举是中国政府授意，但她既未交代拒绝外国记者签证的理由，也拒绝澄清是否有大陆官员与她谈过此事。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   纽约时报周日报道说，来自美国、英国和欧盟的外交官要求香港政府对此作出解释。跨国企业则担心香港作为区域基地的未来。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   事实上，这并不是香港首次因为政治原因拒人于门外。2017年，因为民进党选举获胜，台湾陆委会的两名官员被拒入境。今年早些时候，英国下议院一名经常批评中国的议员，被拒绝入境香港。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   被马凯邀请讲话的陈浩天，是香港一个可能只有数十人的小型政治团体—香港民族党—的负责人，该党主张香港独立建国，因此受到香港以及中国大陆政府的全力打压。在香港，陈浩天和民族党并不受香港市民的支持，然而经此事件，陈浩天和香港民族党登上了全球各大媒体的版面，成了香港另一个标志性人物。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   缺乏完善民主体制的香港之所以成功，与其自由港的特性有关。左派和右派、中国和西洋的各式人物在香港对抗、合作，达成社会运行的默契，全都建立在法制和自由的基础上。不少人认为，当法制被侵蚀，自由遭到剥夺，香港只会是一个普通的中国沿海城市而已。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   1997年中国接管香港主权时，美国新闻周刊以“香港之死”为封面故事，对香港的未来表达了极为负面的看法。过去二十年，香港在亚洲的地位虽然日渐降低，但仍然维持了其自由港的经济和政治作用。纽约时报说，马凯被拒签证一事，使得许多人重新想起这一预言。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
