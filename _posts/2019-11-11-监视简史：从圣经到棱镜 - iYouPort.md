      <strong>
       监视如人类文明本身一样古老
      </strong>
     </em>
    </span>
   </li>

  <p>
   <img alt="" class="aligncenter size-full wp-image-6842 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-40.png?resize=1100%2C834&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-40.png?w=1450&amp;ssl=1 1450w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-40.png?resize=300%2C228&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-40.png?resize=768%2C583&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-40.png?resize=1024%2C777&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-40.png?resize=1100%2C834&amp;ssl=1 1100w" data-recalc-dims="1" height="834" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/0-40.png?resize=1100%2C834&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  </p>
  <p class="graf graf--p">
   一说到监视人们就会想到棱镜、人脸识别、DNA跟踪、NSO的间谍软件……似乎一切都基于高科技。
  </p>
  <p class="graf graf--p">
   但是监视绝非现代社会所独有的。正相反，它有着深远的历史。
  </p>
  <p class="graf graf--p">
   正如 Keith Laidler 在他的著作《监视无限：我们如何成为地球上最受关注的人》中提出的那样，“
   <strong class="markup--strong markup--p-strong">
    间谍和监视行为基本与人类文明本身一样古老
   </strong>
   。城市国家和帝国的崛起意味着每个当权者都需要知道他们的敌人的性情和能力，以及他们自己人口的忠诚度和普遍情绪。“
  </p>
  <p class="graf graf--p">
   在圣经中就有很多可能的监视例子。
  </p>
  <p class="graf graf--p">
   例如撒母耳记中大卫偷看拔示巴洗澡的故事；其中监视被用于大卫自己的个人收获和快乐，而不是更大的利益。
  </p>
  <p class="graf graf--p">
   Kieith Laidler 指出的另一个例子可以在“数字之书”中找到其中，“间谍需要收集的信息的细节”主要是关于土地、土地上有多少人、城镇的布局、土壤的质量和树木的存在。
  </p>
  <p class="graf graf--p">
   与 David 和 Bathsheba 的故事不同，这里的监视通过收集有关人民和土地的信息，间谍将能够确定其敌人的优势和劣势。
  </p>
  <p class="graf graf--p">
   这种监视的使用满足了历史和现代监视的最基本特征之一。
  </p>
  <p class="graf graf--p">
   深刻了解“敌人”，以掌握最准确的进攻机会，这就是监视的主要目的。
  </p>
  <p class="graf graf--p">
   换句话说，监视被瞄准谁，谁就是监视者的敌人。
  </p>
  <p class="graf graf--p">
   在过去的几年里，乔治城大学英美研究副教授 Brian Hochman 一直在研究电子监控 —— 包括可能进行窃听的技术发展、以及150多年来使其成为社会生活一部分的文化和政治现实。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “美国人已经接受了这样一个痛苦的事实：没有电子窃听就没有电子通讯” ，Hochman 说，2017–2018 年他在国家人文科学公共学者基金会工作，他目前正在撰写一本关于监视主题的书。
  </p>
  <p class="graf graf--p">
   以下是 Hochman 讲述的关于窃听的早期阶段，以及对“数据监控”的未来的预测。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    我们需要多久才能找到窃听的起源？
   </strong>
  </p>
  <p class="graf graf--p">
   窃听早在电话出现之前就已经开始了。
  </p>
  <p class="graf graf--p">
   最早的禁止窃听法规于1862年在加利福尼亚州写成，就在太平洋电报公司到达西海岸之后，第一个被定罪的人是在1864年一位名为 D.C.Williams 的股票经纪人。
  </p>
  <p class="graf graf--p">
   他的计划非常巧妙：他窃听了公司的电报线路，并将他听到的信息卖给了股票交易员。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    谁在偷听？
   </strong>
  </p>
  <p class="graf graf--p">
   直到20世纪20年代，窃听最常被私人侦探和公司使用。
  </p>
  <p class="graf graf--p">
   直到禁酒时期（从1920年至1933年期间）窃听已经成为一种被使用极为普遍的“执法工具”。
  </p>
  <p class="graf graf--p">
   但即使在1928年最高法院裁决对政府窃听的合宪性进行了狭隘的肯定之后，其合法性及其道德问题仍然是激烈争论的焦点。
  </p>
  <p class="graf graf--p">
   然后，20世纪30年代的揭露表明，窃听是企业管理层根除工会活动的一种普遍且恶毒的工具。
  </p>
  <p class="graf graf--p">
   例如，美国参议院的 La Follette 公民自由委员会发现了公司暗中实施的各种恶意窃听行为。
  </p>
  <p class="graf graf--p">
   聘请私人侦探监视工会是这一时期最经典的肮脏伎俩之一。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    大众何时开始关注窃听问题？
   </strong>
  </p>
  <p class="graf graf--p">
   直到在20世纪20年代，普通美国人才开始注意到窃听，直到20世纪50年代窃听才被视为国家问题。
  </p>
  <p class="graf graf--p">
   即便如此，当时人们关心的主要是涉及私人窃听问题。
  </p>
  <p class="graf graf--p">
   雇佣私人侦探进行窃听在某些地方非常普遍，其中最著名的地方是在纽约。
  </p>
  <p class="graf graf--p">
   例如，根据阴暗的“一方同意法” —— 也就是，只要一方同意“听取”，即不成为“窃听” —— 雇用一名电子监视专家（被称为“私人耳朵”）就是合法的，可以监听你家的电话，看看你的妻子是否正在与另一名男子闲聊。
  </p>
  <p class="graf graf--p">
   毋庸置疑，美国公众担心这群非官方监视者有能力也有专业知识可以进入迅速扩大的电话网络。
  </p>
  <p class="graf graf--p">
   关于“官方”窃听的反馈意见好坏参半。
  </p>
  <p class="graf graf--p">
   到1965年，美国的规范政治立场是，为“国家安全”目的的窃听是一种必要的罪恶。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    今天，情况正好相反。大多数人担心政府窃听
   </strong>
  </p>
  <p class="graf graf--p">
   这开始于水门事件，当时公众看到了行政部门滥用窃听的惊人程度，并且还有斯诺登对国家安全局全球窃听的披露。
  </p>
  <p class="graf graf--p">
   但重要的是要认识到，今天为刑事调查而进行的窃听监视数量几乎是“国家安全”窃听的两倍。
  </p>
  <p class="graf graf--p">
   由于刑事调查中的窃听不成比例地针对非裔和拉丁裔美国人，因此这不仅仅是一个公民自由问题：更是民权问题。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-6843 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/窃听.png?resize=1100%2C598&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/窃听.png?w=1302&amp;ssl=1 1302w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/窃听.png?resize=300%2C163&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/窃听.png?resize=768%2C418&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/窃听.png?resize=1024%2C557&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/窃听.png?resize=1100%2C598&amp;ssl=1 1100w" data-recalc-dims="1" height="598" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/窃听.png?resize=1100%2C598&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6843" data-recalc-dims="1" height="598" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/窃听.png?resize=1100%2C598&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/窃听.png?w=1302&amp;ssl=1 1302w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/窃听.png?resize=300%2C163&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/窃听.png?resize=768%2C418&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/窃听.png?resize=1024%2C557&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/窃听.png?resize=1100%2C598&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    150多年的窃听历史对于今天来说意味着什么？
   </strong>
  </p>
  <p class="graf graf--p">
   如今，电子监控存在着截然不同的东西：它在大规模的程度上运作。
  </p>
  <p class="graf graf--p">
   直到20世纪80年代，窃听和电子监视都是高度个性化的。专注于私人之间的通信。
  </p>
  <p class="graf graf--p">
   现在，特别是“dataveillance”的兴起，这是从20世纪60年代、70年代甚至20世纪80年代的角度来看，几乎无法想象的监视规模。
  </p>
  <p class="graf graf--p">
   Dataveillance 是对元数据的跟踪。美国国家安全局确实窃听人们的谈话，这正是传统意义上被认为是“窃听”的行为，只是现在它用数据形式表现。
  </p>
  <p class="graf graf--p">
   重要的不一定是你在电话里说了什么，而是你打电话给谁、打电话的时间、手机的位置、金融交易等等元数据 。
  </p>
  <p class="graf graf--p">
   他们对一百万个不同的数据点进行三角测量，他们可以非常清楚地了解发生了什么。
  </p>
  <p class="graf graf--p">
   但是，即使是考虑最早期的窃听，其中一个领域也是持续至今的，即：电信行业在监视国家的崛起中的参与程度以及监控数据在电信基础设施与执法基础设施之间流动的程度。
  </p>
  <p class="graf graf--p">
   20世纪20年代执法部门为执行酒精战争而采取的最简单的方法就是通过贝尔系统中央交换机窃听电话。如今也一样。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    然而，人们却愿意让公司窃听他们
   </strong>
  </p>
  <p class="graf graf--p">
   那些智能音箱？它们基本上都是窃听器。
  </p>
  <p class="graf graf--p">
   他们一直在听。
  </p>
  <p class="graf graf--p">
   这是一种新型的企业监控：如果让他们听你说的话，他们就可以在你想要的时候给你你想要的东西。人们喜欢这样。
  </p>
  <p class="graf graf--p">
   但这些数据还会被用来做什么？鬼才相信它是全心全意为人们服务的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    下一步是什么？
   </strong>
  </p>
  <p class="graf graf--p">
   历史学家不是在做预测，但可以肯定地说，电子监视和数据监控很可能会继续扩大规模。
  </p>
  <p class="graf graf--p">
   监视将更加全球化，更加即时实时。
  </p>
  <p class="graf graf--p">
   如果按照曾经以来的样子持续下去，甚至可以更确定地说，公众对这些问题的关注将会消退。
  </p>
  <p class="graf graf--p">
   这是关于美国窃听历史如此引人注目的事情之一：它从来就不是秘密，但每隔10到15年就会发生一次重大公共丑闻。
  </p>
  <p class="graf graf--p">
   于是，人们会有一些短暂的愤怒时刻，然后就是漫长的自满，就像现在这样 —— 这就是使监视能够以它的方式持续存在的原因之一。
  </p>
  <p class="graf graf--p">
   延伸《

    为什么大规模监视曝光六年后一切都没能改变？
   </a>
   》
  </p>
  <p class="graf graf--p">
   公民的抵抗是结束噩梦的唯一办法。⚪️
