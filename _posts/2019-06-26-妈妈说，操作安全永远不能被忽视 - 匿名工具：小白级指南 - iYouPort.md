  <p class="graf graf--p">
   我们都知道政府和他们领导的警察国家有多么强的跟踪监视能力。很多时候匿名是绝对必要的。如果您无法以正确的方式使用匿名工具，会造成很多危险。
  </p>
  <p class="graf graf--p">
   几乎所有的逮捕、曝光、诱捕或任何去匿名化结果都是在小而技术性的东西背后完成的，用户忘记了开启或使用保护性措施，或做了一些无意中将你的真实生活角色与你的在线活动联系起来的事。
  </p>
  <p class="graf graf--p">
   在本指南中，我们将引导您完成安装和设置
   <a class="markup--anchor markup--p-anchor" data-href="https://www.whonix.org/" href="https://www.whonix.org/" rel="noopener noreferrer" target="_blank">
    匿名工具 Whonix
   </a>
   ，该操作系统旨在保证您的安全，从技术层面做到匿名访问暗网。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在我们开始之前……
   </strong>
  </p>
  <p class="graf graf--p">
   你应该拥有至少 2Gb 的 RAM，最近几年内几乎任何机器都会有这个。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    请记住，工具不是解药！工具是为用户提供保持匿名的能力。解药是可以拯救一个人，但匿名工具不能拯救用户。它们只能授权用户有权拯救自己。所以，根源性问题还在于你
   </strong>
   。
  </p>
  <p class="graf graf--p">
   要知道该如何保证自己的安全，首先就要知道自己面临的威胁都是什么。
  </p>
  <p class="graf graf--p">
   您的威胁建模将根据您的身份和所需的操作安全级别而有所不同。Whonix 只是一个工具，如果使用得当，它将帮助您保持安全，并拥有在暗网上保持匿名的显著优势。但即使你完美地使用了 Whonix，如果你的具体动作出现幼儿园级别的错误揭露了你的现实身份 — 比如从暗网市场购买物品，比特币连接到信用卡 — 那么 Whonix 也无法拯救你。
  </p>
  <p class="graf graf--p">
   Whonix 团队在整个安装过程中非常明确地阐述了这一点，这些警告字段在本教程中不会突出显示。但是，
   <strong class="markup--strong markup--p-strong">
    您应该能时刻记得这些警告，注意安全。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第 0 步：首先的首先 Virtual Box
   </strong>
  </p>
  <p class="graf graf--p">
   您需要
   <a class="markup--anchor markup--p-anchor" data-href="https://www.virtualbox.org/wiki/Downloads" href="https://www.virtualbox.org/wiki/Downloads" rel="noopener noreferrer" target="_blank">
    安装 Virtual Box
   </a>
   才能使用 Whonix。这是由于 Whonix 的独特方式。
  </p>
  <p class="graf graf--p">
   Whonix 作为两个同时运行的虚拟机运行。最初可能感觉有点奇怪，但了解 Whonix 工作原理的基础知识会有助于整个过程。
  </p>
  <p class="graf graf--p">
   有一个 “Gateway” Whonix 虚拟机和一个 “Workstation” Whonix 虚拟机，用作 Tor 网络的门户，还充当防火墙。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “Workstation”虚拟机是所有操作发生的地方，它受 Gateway 保护与可怕的互联网相隔离：它连接到 Gateway 虚拟机而不是直接连接到互联网……我知道，它好像有点复杂。但我们的读者中应该有 Whonix 的天才。总之您只需要知道：
   <strong class="markup--strong markup--p-strong">
    通过永远不将 Workstation 直接连接到互联网，它增加了另一层保护 — 超出了我们介绍过的类似其他工具（如 Tails）
   </strong>
   。
  </p>
  <p class="graf graf--p">
   好啦简短概述完成，继续安装 Virtual Box。
  </p>
  <p class="graf graf--p">
   由于 Whonix 在 Virtual Box 中运行，因此本教程适用于所有操作系统。我今天尝试在 OSX 上安装 Whonix，因为它是一个流行的操作系统，是的我无法忍受 Windows。安装 Virtual Box 后，我们就可以开始了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第1步：下载 Whonix
   </strong>
  </p>
  <p class="graf graf--p">
   导航
   <a class="markup--anchor markup--p-anchor" data-href="https://www.whonix.org/wiki/VirtualBox" href="https://www.whonix.org/wiki/VirtualBox" rel="noopener noreferrer" target="_blank">
    到此页面
   </a>
   。点击第一个标题下的“expand”。
  </p>
  <p class="graf graf--p">
   您需要总共下载四个文件。下载两个.ova 文件，总共刚刚超过 4Gb。这些文件是两个虚拟机的基础：网关和工作站。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-2996 jetpack-lazy-image" data-lazy-sizes="(max-width: 873px) 100vw, 873px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-71.png?resize=873%2C327&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-71.png?w=873&amp;ssl=1 873w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-71.png?resize=300%2C112&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-71.png?resize=768%2C288&amp;ssl=1 768w" data-recalc-dims="1" height="327" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-71.png?resize=873%2C327&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="873"/>

    <img alt="" class="aligncenter size-full wp-image-2996" data-recalc-dims="1" height="327" sizes="(max-width: 873px) 100vw, 873px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-71.png?resize=873%2C327&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-71.png?w=873&amp;ssl=1 873w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-71.png?resize=300%2C112&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-71.png?resize=768%2C288&amp;ssl=1 768w" width="873"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   另外两个要下载的重要文件是.ova 的 sha512 哈希文件。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    验证从互联网上下载的所有文件非常重要，尤其是当它是匿名工具时！
   </strong>
   我们将引导您完成使用这两个哈希文件进行验证的过程。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第 2 步：验证下载
   </strong>
  </p>
  <p class="graf graf--p">
   验证下载可以通过几种不同的方式完成，具体取决于操作系统。如果您使用的是 Windows，请
   <a class="markup--anchor markup--p-anchor" data-href="https://www.microsoft.com/en-us/download/details.aspx?id=11533" href="https://www.microsoft.com/en-us/download/details.aspx?id=11533" rel="noopener noreferrer" target="_blank">
    前往此处获取指南
   </a>
   。如果您使用的是 Linux 发行版，那么您很可能已经知道该怎么做了。如果您使用的是 OSX，请继续阅读。
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    首先打开终端。您可以通过按 Command 和 Space 来执行此操作，然后开始键入“terminal”并按 Enter 键。
   </li>
   <li class="graf graf--li">
    类型 ‘ls Downloads’ 并按 Enter 键。这会将您的终端 shell 带到 Downloads 文件夹。如果您在其他任何地方下载了四个文件，则需要将“下载”替换为文件路径（/ this / format）。
   </li>
   <li class="graf graf--li">
    接下来，我们需要获取’.ova’文件的哈希值。输入 ‘shasum -a 512 Whonix-Gateway-13.0.0.1.4.ova’ 并按 Enter 键。Shasum 是任务。这让我们可以指定 512 的哈希率（以匹配 Whonix 提供的）。您只记住动作就可以了。
   </li>
   <li class="graf graf--li">
    接下来，选择输出，右键单击并选择“复制”。
   </li>
  </ol>
  <p>
   <img alt="" class="aligncenter size-full wp-image-2997 jetpack-lazy-image" data-lazy-sizes="(max-width: 612px) 100vw, 612px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-72.png?resize=612%2C561&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-72.png?w=612&amp;ssl=1 612w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-72.png?resize=300%2C275&amp;ssl=1 300w" data-recalc-dims="1" height="561" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-72.png?resize=612%2C561&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="612"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-2997" data-recalc-dims="1" height="561" sizes="(max-width: 612px) 100vw, 612px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-72.png?resize=612%2C561&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-72.png?w=612&amp;ssl=1 612w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-72.png?resize=300%2C275&amp;ssl=1 300w" width="612"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   有几种方法可以使用哈希进行实际验证。但是，对于本文来说，复制文件并打开下载的哈希是最简单的：
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    打开 ‘Finder’ 并前往 ‘Downloads’ 文件夹。
   </li>
   <li class="graf graf--li">
    双击 “Whonix-Gateway-13.0.0.1.4.sha512sum” 文件。
   </li>
   <li class="graf graf--li">
    使用您选择的程序打开; 转到列表底部并选择“TextEdit”。
   </li>
   <li class="graf graf--li">
    接下来，点击“Command＆F”打开“查找”。然后点击’Command V’粘贴我们从终端会话中获得的哈希。它应该与文本文件中的哈希完全匹配。
   </li>
  </ol>
  <p class="graf graf--p">
   对于“Workstation”文件，重复此步骤的全部内容。完成后，您将验证自己确实已经下载了未经破坏且准确的 Whonix .ova 文件。现在可以随意删除两个’.sha512sum’文件。
  </p>

   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第 3 步：创建虚拟机
   </strong>
  </p>
  <p class="graf graf--p">
   现在我们有了两个经过验证的.ova文件，是时候用它们来创建两个可用的虚拟机了。打开 Virtual Box 并在“文件”下单击’Import Appliance.’。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-2999 jetpack-lazy-image" data-lazy-sizes="(max-width: 723px) 100vw, 723px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-74.png?resize=723%2C448&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-74.png?w=723&amp;ssl=1 723w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-74.png?resize=300%2C186&amp;ssl=1 300w" data-recalc-dims="1" height="448" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-74.png?resize=723%2C448&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="723"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   单击文件路径字段右侧的小文件夹图标，然后转到‘下载’。单击 Gateway .ova 文件并单击继续。
  </p>

   <img alt="" class="aligncenter size-full wp-image-3009 jetpack-lazy-image" data-lazy-sizes="(max-width: 1075px) 100vw, 1075px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-84.png?resize=1075%2C769&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-84.png?w=1075&amp;ssl=1 1075w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-84.png?resize=300%2C215&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-84.png?resize=768%2C549&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-84.png?resize=1024%2C733&amp;ssl=1 1024w" data-recalc-dims="1" height="769" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-84.png?resize=1075%2C769&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1075"/>

    <img alt="" class="aligncenter size-full wp-image-3009" data-recalc-dims="1" height="769" sizes="(max-width: 1075px) 100vw, 1075px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-84.png?resize=1075%2C769&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-84.png?w=1075&amp;ssl=1 1075w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-84.png?resize=300%2C215&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-84.png?resize=768%2C549&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-84.png?resize=1024%2C733&amp;ssl=1 1024w" width="1075"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Tor 浏览器尚未实际安装。这可能是为了保持 Whonix 下载的最小尺寸。确认您要下载浏览器，然后选择最稳定的版本。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-3010 jetpack-lazy-image" data-lazy-sizes="(max-width: 1080px) 100vw, 1080px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-85.png?resize=1080%2C774&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-85.png?w=1080&amp;ssl=1 1080w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-85.png?resize=300%2C215&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-85.png?resize=768%2C550&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-85.png?resize=1024%2C734&amp;ssl=1 1024w" data-recalc-dims="1" height="774" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-85.png?resize=1080%2C774&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1080"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-3010" data-recalc-dims="1" height="774" sizes="(max-width: 1080px) 100vw, 1080px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-85.png?resize=1080%2C774&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-85.png?w=1080&amp;ssl=1 1080w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-85.png?resize=300%2C215&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-85.png?resize=768%2C550&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-85.png?resize=1024%2C734&amp;ssl=1 1024w" width="1080"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   它将使用 Tor 网络，这可能需要比你习惯的时间更长一些，所以，去弄杯咖啡吧。
  </p>
  <p class="graf graf--p">
   下载完成后，它将运行浏览器。点击第一个链接’检查 IP’。这会将您重定向到 Tor Checker，这是您应该在 Tor 浏览器的每次启动时运行的，只是为了确认您已连接到网络。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-3011 jetpack-lazy-image" data-lazy-sizes="(max-width: 1096px) 100vw, 1096px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-86.png?resize=1096%2C783&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-86.png?w=1096&amp;ssl=1 1096w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-86.png?resize=300%2C214&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-86.png?resize=768%2C549&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-86.png?resize=1024%2C732&amp;ssl=1 1024w" data-recalc-dims="1" height="783" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-86.png?resize=1096%2C783&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1096"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-3011" data-recalc-dims="1" height="783" sizes="(max-width: 1096px) 100vw, 1096px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-86.png?resize=1096%2C783&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-86.png?w=1096&amp;ssl=1 1096w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-86.png?resize=300%2C214&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-86.png?resize=768%2C549&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-86.png?resize=1024%2C732&amp;ssl=1 1024w" width="1096"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   已经完成了！你现在可以使用 Whonix 安全地匿名访问暗网了。通过将 Gateway 虚拟机用作 shell 层，您可以在 Workstation 中获得额外的保护。
  </p>
  <p class="graf graf--p">
   Whonix 通过隔离来掩护 Workstation 虚拟机。Workstation 位于 Gateway 后面，理论上任何恶意连接尝试都只能抵达 Gateway，那里不包含任何您的信息或工作文件。你一直都在暗门后面。
  </p>
