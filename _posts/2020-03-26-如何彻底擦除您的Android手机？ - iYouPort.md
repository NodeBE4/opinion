  <p class="graf graf--p">
   不论您是想要以旧换新获得折扣、在eBay上出售旧手机、赠予朋友、还是将其扔进垃圾桶，您都需要首先彻底擦除所有数据，将其重置为出厂状态。
  </p>
  <p class="graf graf--p">
   幸运的是，这很容易做到。
  </p>
  <p class="graf graf--p">
   开始之前，请注意，本次说明以运行 Android 9 的 Pixel 3 XL 为例，但大多数当前的 Android 手机的擦洗过程大致是相同的。
  </p>
  <p class="graf graf--p">
   不用说，
   <strong class="markup--strong markup--p-strong">
    首先要确保所有数据都已传输到新手机、已备份、或同时完成这两者。因为旧手机重置后将无法恢复
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “重置”屏幕为您提供三个选择：

   </noscript>
  </p>
  <p class="graf graf--p">
   1、前往 “设置”
  </p>
  <p class="graf graf--p">
   2、选择 系统&gt;高级&gt;重置选项
  </p>
  <p class="graf graf--p">
   3、您将看到上面那三个选择。
  </p>
  <p class="graf graf--p">
   如果您在连接时遇到问题，“Reset Wi-Fi, mobile &amp; Bluetooth” 有时会很有用；
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “Reset app preferences” 为还原所有应用程序和通知默认设置等；
  </p>
  <p class="graf graf--p">
   这些都不会删除手机上的数据。
  </p>
  <p class="graf graf--p">
   这里您应该使用的是第三个选项，“清除所有数据（恢复出厂设置）”。
  </p>
  <p class="graf graf--p">
   您会看到一个警告屏幕，提醒您继续操作将删除的数据以及当前登录的所有帐户。选择 “Reset phone”。
  </p>
  <p class="graf graf--p">
   4、如果您有PIN或其他安全设置，系统会要求您输入。
  </p>
  <p class="graf graf--p">
   5、您将看到另一个屏幕，询问您是否确定要执行此操作。你确定吗？然后点击“清除所有内容”按钮。
  </p>
  <p class="graf graf--p">
   就是这么简单。该过程通常只需要两到三分钟。
  </p>
  <p class="graf graf--p">
   您的手机将重新启动，并开始要求回答常见的开放性问题，例如语言、帐户、网络等。看到这些问题后您就能知道刚才做对了。
  </p>
  <p class="graf graf--p">
   有几件事要注意。
  </p>
  <p class="graf graf--p">
   首先，如果您拥有三星手机，那么在重置之前，系统会要求您输入三星帐户密码（前提是您已创建了一个三星帐户密码并将其添加到手机中）。
  </p>
  <p class="graf graf--p">
   如果您的 Android 手机使用的是早期版本（例如，版本8或更早），那么重置可能会在重启后要求您登录。如果您已经将电话送给了其他人，这可能会很尴尬。
  </p>
  <p class="graf graf--p">
   因此，如果您的手机比较旧，最好先关闭屏幕锁定（转到安全&gt;屏幕锁定，并选择“None”）；删除您的 Google 帐户，您可以在“设置”中找到（设置&gt;帐户&gt; Google），然后再开始重置。
  </p>
  <p class="graf graf--p">
   ⚠️有证据表明，即使在重置手机后，某些数据仍可能保留在手机上。要格外小心！您可以
   <strong class="markup--strong markup--p-strong">
    在重置手机之前对其进行加密
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    转到设置&gt;安全和位置&gt;高级&gt;加密和凭据，然后选择“加密手机”
   </li>
   <li class="graf graf--li">
    返回设置并开始重置过程
   </li>
  </ul>
  <p class="graf graf--p">
   当然，如果您不打算将手机交给任何人只想废弃它，那么就砸烂它吧。砸越得越烂越好。⚪️
  </p>
