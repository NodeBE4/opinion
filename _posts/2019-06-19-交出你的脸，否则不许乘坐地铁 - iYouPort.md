  <p class="graf graf--p">
   只有交出生物识别数据否则不可乘坐地铁的日子可能已经不远了。在中国的科技之都深圳，一家当地的地铁运营商正在测试由5G网络驱动的面部识别地铁“检票”。
  </p>
  <p class="graf graf--p">
   到目前为止，试验仅限于这一个站点，还不清楚这种面部识别对于双胞胎或其他长相相似的人是如何工作的。进入车站的人可以在屏幕上扫描他们的面孔，然后他们的票价会自动从关联账户中扣除。
   <strong class="markup--strong markup--p-strong">
    人们需要预先登记自己的面部数据，并将付款方式与地铁帐户相关联
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这意味着所有人每次进入地铁的行程都将被追踪到他们脸部的像素。目前尚不清楚这是否比已经完成的跟踪更多（主要是不知道中国究竟跟踪了多少）。中国许多主要城市拥有广泛的监控摄像系统，可记录公民的面孔、年龄、性别、以及他们在该地区居住的时间。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    面部识别技术的算法是在深圳地铁和手机制造商华为监督的实验室中设计的
   </strong>
   。深圳地铁尚未给出面部识别能够跟踪其所有车站和地铁线路的时间表。
  </p>
  <p class="graf graf--p">
   使用面部识别进行支付并不新鲜，尽管在地铁上使用它够新鲜。在中国的肯德基商店，人们可以
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theverge.com/2017/9/4/16251304/kfc-china-alipay-ant-financial-smile-to-pay" href="https://www.theverge.com/2017/9/4/16251304/kfc-china-alipay-ant-financial-smile-to-pay" rel="noopener noreferrer" target="_blank">
    扫描自己的脸以购买炸鸡
   </a>
   ，这事自2017年以来一直存在。
  </p>
  <p class="graf graf--p">
   在移动支付领域，中国领先于美国，因为该国的人完全不重注隐私，有近一半的人口在2018年使用手机进行支付。
  </p>
  <p class="graf graf--p">
   通过微信支付或宝支付的方法在中国非常受欢迎，中国央行去年不得不警告商店不要拒绝现金否则会面临未指明的处罚。不过，当我去年在中国时，一家商店表示不愿接受现金，因为无法找零。
  </p>

    Chinese subway is experimenting with facial recognition to pay for fares
   </a>
   . Fare would be automatically deducted from a linked payment method
  </p>
