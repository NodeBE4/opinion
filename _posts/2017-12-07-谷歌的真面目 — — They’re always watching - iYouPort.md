  <p class="graf graf--p graf-after--figure" id="3b0f">
   二十年前，美国情报界与硅谷密切合作，努力追踪网络公民 — — 谷歌就是这个大规模间谍活动的核心。谷歌雄心勃勃创作的诸多研究都是由情报界建立的研究小组资助和协调的，目的是：寻找在线跟踪个人和群体的方法。他们成功了，这就是为什么今天你能生活在这个风声鹤唳的反乌托邦世界里。
  </p>
  <p class="graf graf--p graf-after--p" id="5867">
   情报界希望该国领先的计算机科学家能够将非机密信息和用户数据与所谓的互联网相结合，并创建营利性商业企业，最重要的是满足情报界的需求。
   <strong class="markup--strong markup--p-strong">
    他们希望从一开始就指导超算革命，以便了解数百万人在这个数字信息网络中所做的事。这种合作使今天的全球公众所遭受的大规模监视状态成为可能
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="2362">

   <strong class="markup--strong markup--p-strong">
    这不仅仅是谷歌的起源故事：它本身就是大规模监视国家的起源故事，最重要的是资助它的政府资金
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="0606">
   <strong class="markup--strong markup--p-strong">
    — — 背景：情报界和硅谷 — —
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="60bc">
   在20世纪90年代中期，美国的情报界开始意识到他们有了一个新机会。超算社区刚刚开始从大学环境迁移到私营部门，由一个名为硅谷的地方投资领导。
  </p>
  <p class="graf graf--p graf-after--p" id="ade2">
   一场数字革命正在开启：一场即将改变数据收集世界以及我们如何理解大量信息的革命。情报界希望
   <strong class="markup--strong markup--p-strong">
    在一开始就塑造
   </strong>
   硅谷的超算工作，因此
   <strong class="markup--strong markup--p-strong">
    这些研发对于军事和维稳目的都非常重要
   </strong>
   。这个能够存储数TB信息的超算网络是否能够准确地感知并分析人类留下的数字轨迹？
  </p>
  <p class="graf graf--p graf-after--p" id="5522">
   情报界对这个问题的答案非常感兴趣。
  </p>
  <p class="graf graf--p graf-after--p" id="33fb">
   情报收集可能是他们的整个世界，但中央情报局（CIA）和国家安全局（NSA）已经意识到他们的未来可能会在政府之外形成深刻的影响。当时在克林顿政府内部的军事和情报预算处于紧张之中，而私营部门拥有大量资源可供支配。如果情报界希望进行史无前例的大规模监视，那么就需要政府与新兴超算公司之间的紧密合作。
  </p>
  <p class="graf graf--p graf-after--p" id="cb6f">
   为此，他们开始接触正在创造这场超算革命的美国大学的科学家们，这些科学家正在开发一种方法来做那些国安局和中情局工作站的人都无法做到的事：收集大量数据，并对其进行情报感知。
  </p>
  <p class="graf graf--p graf-after--p" id="402a">
   <strong class="markup--strong markup--p-strong">
    — — 丰富的政府科研经费历史 — —
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="c760">
   美国最好的科学家和情报界之间已经有长久的合作历史，从原子弹和卫星技术的创造到将人类送上月球的努力。
  </p>

   整个硅谷都不例外。到20世纪90年代中期，情报界正在为学术界最有希望的超算工作提供资金，指导创造大量信息以对私营部门和情报界都有用的努力。
  </p>
  <p class="graf graf--p graf-after--p" id="c811">
   他们通过一个由大型军事和情报承包商为中央情报局和国家安全局管理的非机密、高度划分的计划为这些计算机科学家提供资金。该计划被称为大规模数字数据系统（MDDS）项目。
  </p>
  <p class="graf graf--p graf-after--p" id="c035">
   <strong class="markup--strong markup--p-strong">
    — — 大规模数字数据系统（MDDS）项目 — —
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="882f">
   MDDS被介绍给斯坦福大学、加州理工学院、麻省理工学院、卡内基梅隆大学、哈佛大学等数十名计算机科学家，并在白皮书中介绍了 CIA，NSA，DARPA 和其他机构希望实现的目标。
  </p>
  <p class="graf graf--p graf-after--p" id="1c6f">
   该研究主要由 NSF 等机构提供资金和管理，如果能够实现情报界所希望的，那么该体系结构将在私营部门扩大规模。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="89e5">
   “不仅活动变得更加复杂，而且不断变化的需求要求IC [情报社区]处理不同类型以及更大量的数据，”情报界在1993年的
   <a class="markup--anchor markup--p-anchor" data-href="https://groups.google.com/forum/#!topic/mail.cypherpunks/4CDiW59hS88" href="https://groups.google.com/forum/#!topic/mail.cypherpunks/4CDiW59hS88" rel="nofollow noopener noreferrer" target="_blank">
    MDDS 白皮书
   </a>
   中说。“因此，IC 正在积极推动有效管理海量数据库的研究，并确保将 IC 要求纳入或适应商业产品。由于挑战并非任何一个机构所独有，社区管理人员（CMS）已经委托大规模数字数据系统[MDDS]工作组来满足其需求，并确定和评估可能的解决方案。“
  </p>
  <p class="graf graf--p graf-after--p" id="3837">
   <strong class="markup--strong markup--p-strong">
    在接下来的几年里，该计划的目标是提供数十亿美元的赠款，以推进这一研究概念
   </strong>
   。这些拨款主要通过美国国家科学基金会进行，以便最有希望的、最成功的努力可以作为知识产权获取，并成为吸引硅谷投资公司的基础。
  </p>
  <p class="graf graf--p graf-after--p" id="25b7">
   这种公私合作“创新
   <a class="markup--anchor markup--p-anchor" data-href="https://www.sbir.gov/success-story/qualcomm-inducted-sbir-hall-fame" href="https://www.sbir.gov/success-story/qualcomm-inducted-sbir-hall-fame" rel="nofollow noopener noreferrer" target="_blank">
    体系
   </a>
   ”帮助推出了
   <a class="markup--anchor markup--p-anchor" data-href="https://nsf.gov/news/news_summ.jsp?cntn_id=242100&amp;org=NSF&amp;from=news" href="https://nsf.gov/news/news_summ.jsp?cntn_id=242100&amp;org=NSF&amp;from=news" rel="nofollow noopener noreferrer" target="_blank">
    高通、赛门铁克
   </a>
   、Netscape 等强大的科技公司，并
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nsf.gov/about/history/sensational60.pdf" href="https://www.nsf.gov/about/history/sensational60.pdf" rel="nofollow noopener noreferrer" target="_blank">
    资助了
   </a>
   多普勒雷达和光纤等领域的关键研究，这些研究是当今 AccuWeather，Verizon 和AT＆T等大公司的核心。
  </p>
  <p class="graf graf--p graf-after--p" id="2bd3">
   今天，NSF 为大学计算机科学研究
   <a class="markup--anchor markup--p-anchor" data-href="https://www.asm.org/index.php/statements-and-testimony/item/6300-nsf-fy2018" href="https://www.asm.org/index.php/statements-and-testimony/item/6300-nsf-fy2018" rel="nofollow noopener noreferrer" target="_blank">
    提供了近90％
   </a>
   的联邦资金。
  </p>
  <ul class="postList">
   <li class="graf graf--li graf-after--p" id="6159">
    更多详见：《
    <a class="markup--anchor markup--li-anchor" data-href="https://medium.com/@iyouport/%E5%85%B3%E4%BA%8E-%E4%BA%94%E7%9C%BC%E8%81%94%E7%9B%9F-%E7%9A%84%E6%9C%80%E6%96%B0%E6%8A%AB%E9%9C%B2%E6%96%87%E4%BB%B6-%E7%A7%91%E6%8A%80%E4%BA%A7%E4%B8%9A%E5%B7%A8%E5%A4%B4%E4%B8%BA%E4%BB%80%E4%B9%88%E8%83%BD%E6%88%90%E4%B8%BA%E5%B7%A8%E5%A4%B4-de3df79d6df9" href="https://medium.com/@iyouport/%E5%85%B3%E4%BA%8E-%E4%BA%94%E7%9C%BC%E8%81%94%E7%9B%9F-%E7%9A%84%E6%9C%80%E6%96%B0%E6%8A%AB%E9%9C%B2%E6%96%87%E4%BB%B6-%E7%A7%91%E6%8A%80%E4%BA%A7%E4%B8%9A%E5%B7%A8%E5%A4%B4%E4%B8%BA%E4%BB%80%E4%B9%88%E8%83%BD%E6%88%90%E4%B8%BA%E5%B7%A8%E5%A4%B4-de3df79d6df9" rel="noopener noreferrer" target="_blank">
     最新披露文件：科技产业巨头为什么能成为巨头？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p graf-after--li" id="5c6b">
   <strong class="markup--strong markup--p-strong">
    — — CIA和NSA的最终目标 — —
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="5cd6">
   中央情报局和国家安全局的研究部门希望学术界最好的计算机科学家们能够识别出他们所谓的“鸟羽”：就像大雁一起以超大的V字形飞翔，或成群的麻雀般的聚集，使突发的运动能将人们融合在一起，他们预测志同道合的人群会在网上步调一致地移动。
  </p>
  <p class="graf graf--p graf-after--p" id="fb63">
   情报界为科学家们提供了他们的第一次非机密简报“鸟羽”，并在1995年春天的圣何塞费尔蒙特酒店举办了“关于大规模数字系统情报社区倡议的会议”。
  </p>
  <p class="graf graf--p graf-after--p" id="94a7">
   他们的研究目标是
   <strong class="markup--strong markup--p-strong">
    跟踪迅速扩展的全球信息网络中的数字指纹
   </strong>
   ，该网络当时被称为万维网。
  </p>
  <p class="graf graf--p graf-after--p" id="cc62">
   是否可以监视整个数字信息世界，以便跟踪和分类人类在这样的网络中发出的请求？人们的查询搜索是否可以按重要性排列和排名？能否在这个信息海洋中识别出“鸟羽”，以便能够以有组织的规模跟踪社区和群体吗？
  </p>
  <p class="graf graf--p graf-after--p" id="aa65">
   通过与新兴的商业数据公司合作，他们的目的是跟踪互联网上志同道合的人群，并从他们留下的数字指纹中将他们识别出来，就像法医科学家使用指纹污迹识别罪犯一样。
  </p>
  <p class="graf graf--p graf-after--p" id="418a">
   <strong class="markup--strong markup--p-strong">
    — — 谢尔盖布林和拉里佩奇 — —
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="b005">
   1995年，斯坦福大学的计算机科学研究团队获得了最早和最有前途的 MDDS 资助之一，与NSF和DARPA资助有着长达十年的合作历史。
  </p>
  <p class="graf graf--p graf-after--p" id="4719">
   此授权的
   <a class="markup--anchor markup--p-anchor" data-href="http://db.cs.pitt.edu/idm/reports/1998/9631952.html" href="http://db.cs.pitt.edu/idm/reports/1998/9631952.html" rel="nofollow noopener noreferrer" target="_blank">
    主要目标是
   </a>
   “使用’查询人群’方法描述的非常复杂的查询优化。”第二个拨款 — — 与 Google 的起源最密切相关的 DARPA-NSF 拨款 — — 是以互联网为骨干建立大规模数字文库的协调努力的一部分。
  </p>
  <p class="graf graf--p graf-after--p" id="c1f5">
   <strong class="markup--strong markup--p-strong">
    这两项拨款都资助了两名研究生的研究，他们在网页排名方面取得了快速的进展，并且跟踪（并理解）用户的查询：他们就是未来的谷歌联合创始人谢尔盖布林和拉里佩奇
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="be07">
   Brin 和 Page 在这些拨款下的研究成为了 Google 的核心：人们使用搜索功能在一个非常大的数据集中准确找到他们想要的东西。然而，情报界在他们的研究中看到了一个稍微不同的好处：网络能够如此有效地组织，以便能够精准地识别和跟踪每一个用户吗？
  </p>
  <p class="graf graf--p graf-after--p" id="69a0">
   这就解释了为什么情报界发现布林和佩奇的研究工作如此吸引人；在此之前，中央情报局主要利用该领域的人力情报工作来识别可能构成威胁的人员和团体。提供虚拟网络跟踪人们的能力（与现场工作相结合）将改变一切。
  </p>
  <p class="graf graf--p graf-after--p" id="0775">
   这就是短短几年内成为谷歌的开端。负责领导该计划的
   <strong class="markup--strong markup--p-strong">
    两名情报部门管理人员在他的研究进展中
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.utdallas.edu/~bxt043000/Motivational-Articles/Big_Data-Have_we_seen_it_before.pdf" href="https://www.utdallas.edu/~bxt043000/Motivational-Articles/Big_Data-Have_we_seen_it_before.pdf" rel="nofollow noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     定期与布林会面
    </strong>
   </a>
   ，并且在他和佩奇离开谷歌之前，他还是
   <a class="markup--anchor markup--p-anchor" data-href="http://db.cs.pitt.edu/idm/reports/1998/9631952.html" href="http://db.cs.pitt.edu/idm/reports/1998/9631952.html" rel="nofollow noopener noreferrer" target="_blank">
    其他几篇研究论文的作者
   </a>
   ，这些论文都是由这项 MDDS 资助产生的。
  </p>
  <p class="graf graf--p graf-after--p" id="8a0a">
   这笔赠款使 Brin 和 Page 能够完成他们的工作，并为他们在网页排名和跟踪用户查询方面的突破做出了贡献。谷歌尚未成立时他只是一名斯坦福研究员，利用国家安全局和中央情报局通过 MDDS 计划提供的资助。
  </p>
  <p class="graf graf--p graf-after--p" id="e267">
   <strong class="markup--strong markup--p-strong">
    — — 离开谷歌的故事 — —
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="d05c">
   MDDS 资助的主要研究者特别将 Google
   <a class="markup--anchor markup--p-anchor" data-href="http://db.cs.pitt.edu/idm/reports/2000/9631952.html" href="http://db.cs.pitt.edu/idm/reports/2000/9631952.html" rel="nofollow noopener noreferrer" target="_blank">
    命名为
   </a>
   他们的研究的直接结果：“它的核心技术，使它能够比其他搜索引擎更准确地找到页面，这部分资金起到了作用。”在包括 Brin 一些关键工作的已发表的
   <a class="markup--anchor markup--p-anchor" data-href="http://citeseerx.ist.psu.edu/viewdoc/download?doi=10.1.1.36.2806&amp;rep=rep1&amp;type=pdf" href="http://citeseerx.ist.psu.edu/viewdoc/download?doi=10.1.1.36.2806&amp;rep=rep1&amp;type=pdf" rel="nofollow noopener noreferrer" target="_blank">
    研究论文中
   </a>
   ，作者还参考了由 MDDS 计划创建的 NSF 资助。
  </p>
  <p class="graf graf--p graf-after--p" id="687a">
   相反，每个关于 Google 创始的故事中只提到一个联邦资助：
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nsf.gov/awardsearch/showAward?AWD_ID=9411306" href="https://www.nsf.gov/awardsearch/showAward?AWD_ID=9411306" rel="nofollow noopener noreferrer" target="_blank">
    NSF / DARPA “数字文库”
   </a>
   资助，旨在让斯坦福大学的研究人员能够搜索当时存储在大学服务器上的整个万维网。但这不是全部事实。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="3e0c">
   “谷歌算法的开发是在各种计算机上进行的，主要由斯坦福大学 NSF-DARPA-NASA 资助的数字文库项目提供，”斯坦福大学的
   <a class="markup--anchor markup--p-anchor" data-href="http://infolab.stanford.edu/pub/voy/museum/pictures/display/0-4-Google.htm" href="http://infolab.stanford.edu/pub/voy/museum/pictures/display/0-4-Google.htm" rel="nofollow noopener noreferrer" target="_blank">
    Infolab 说
   </a>
   ，NSF 同样仅在其自己的 Google 历史记录中引用数字文库资助，而不是 MDDS 资助。
  </p>
  <p class="graf graf--p graf-after--p" id="e508">
   在著
   <a class="markup--anchor markup--p-anchor" data-href="http://citeseerx.ist.psu.edu/viewdoc/summary?doi=10.1.1.109.4049" href="http://citeseerx.ist.psu.edu/viewdoc/summary?doi=10.1.1.109.4049" rel="nofollow noopener noreferrer" target="_blank">
    名的研究论文
   </a>
   “大型超文本网络搜索引擎剖析”中描述了 Google 的创建，Brin 和 Page 感谢 NSF 和 DARPA 向斯坦福提供的数字文库资助。
   <strong class="markup--strong markup--p-strong">
    但是，来自情报界 MDDS 计划的资助 — 专门为谷歌建立的突破而设计 — 这一历史记录已经逐渐被消失
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="60cc">
   谷歌过去曾表示，它“不是由中央情报局资助或创建的”。例如，当2006年流传的谷歌多年来从情报界获得资金时，该公司
   <a class="markup--anchor markup--p-anchor" data-href="http://battellemedia.com/archives/2006/11/google_on_cia_untrue.php" href="http://battellemedia.com/archives/2006/11/google_on_cia_untrue.php" rel="nofollow noopener noreferrer" target="_blank">
    告诉连线杂志创始人 John Battelle
   </a>
   ，“与谷歌有关的陈述完全是不真实的。”如果人们一直使用谷歌“查找真相”，就无法找到这些真正的真相。
  </p>
  <p class="graf graf--p graf-after--p" id="cb0d">
   中央情报局是否直接为布林和佩奇的工作提供资金，从而创建谷歌？似乎没有。但是，布林和佩奇是否正确地研究了美国国家安全局、中央情报局和情报界希望得到的帮助？绝对是。
  </p>
  <p class="graf graf--p graf-after--p" id="3b4e">
   要理解这一重要性，你必须考虑到情报界正在努力实现的目标，因为它为学术界最好的计算机科学家提供了资助：
   <strong class="markup--strong markup--p-strong">
    美国中央情报局和国家安全局资助了一个非机密的、分隔的计划，该计划从一开始就是这样设计的，以促进看起来几乎与谷歌完全一样的东西的发展
   </strong>
   。 Brin 通过跟踪用户查询并将其与所进行的许多搜索联系起来进行页面排名的突破性研究 — 基本上就是识别“鸟羽” — 也就是情报界 MDDS 计划的主要目标。
  </p>
  <p class="graf graf--p graf-after--p" id="2849">
   是谷歌，成功达成了间谍们最疯狂的梦想。
  </p>
  <p class="graf graf--p graf-after--p" id="c34d">
   <strong class="markup--strong markup--p-strong">
    — — 情报界在硅谷的持久遗产 — —
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="ba07">
   近年来，数字隐私对情报社区与商业技术巨头之间交叉的关注日益增加。但是大多数人仍然不了解情报界在多大程度上依赖世界上最大的科技公司进行间谍工作。
  </p>
  <p class="graf graf--p graf-after--p" id="6622">
   民间自由权利倡导团体多年来一直在关注人们的隐私问题，特别是因为这一问题现在涉及臭名昭著的“爱国者法案”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="ba55">
   “9/11之后匆忙通过的所谓爱国者法案就是监视法律的许多变化中的第一个，这使得政府更容易通过扩大监视电话和电子邮件通信的权力来广泛监视普通的美国公民，收集银行和信用报告记录，并跟踪互联网上无辜美国人的活动，“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.aclu.org/issues/national-security/privacy-and-surveillance/surveillance-under-patriot-act" href="https://www.aclu.org/issues/national-security/privacy-and-surveillance/surveillance-under-patriot-act" rel="nofollow noopener noreferrer" target="_blank">
    ACLU 说
   </a>
   。 “虽然大多数美国人认为这是为了抓捕恐怖分子，但
   <strong class="markup--strong markup--p-strong">
    所谓的”爱国者法案“实际上将普通公民全部变为了嫌疑人
   </strong>
   。”
  </p>
  <p class="graf graf--p graf-after--p" id="da52">
   当被问到时，最大的技术和通信公司 — 从 Verizon 和 AT＆T 到谷歌、Facebook 和微软 — 都辩称他们从未故意并主动将他们庞大的数据库提供给联邦安全和执法机构：他们说他们只是回应根据所谓“爱国者法案”条款提交的传票或请求 — — 就像他们帮助中国等专制国家监视其公民一样的辩称：这是依照当地法律。
  </p>
  <p class="graf graf--p graf-after--p" id="2dd0">
   但是，即使粗略地浏览最近的公开记录，也会发现，有一系列不断提出的请求完全会破坏这一隐私承诺背后的意图。根据公司向公众提供的数据请求记录，在2016年至2017年的报告期内，地方、州和联邦政府机构寻求数据时向 Verizon 提出了超过
   <a class="markup--anchor markup--p-anchor" data-href="http://transparency.verizon.com/us-data" href="http://transparency.verizon.com/us-data" rel="nofollow noopener noreferrer" target="_blank">
    260,000份传票
   </a>
   、法院命令和其他法律请求，向 AT＆T 提出了超过
   <a class="markup--anchor markup--p-anchor" data-href="http://about.att.com/content/csr/home/frequently-requested-info/governance/transparencyreport.html" href="https://about.att.com/content/csr/home/frequently-requested-info/governance/transparencyreport.html" rel="nofollow noopener noreferrer" target="_blank">
    250,000份此类请求
   </a>
   ，向 Google 提出近
   <a class="markup--anchor markup--p-anchor" data-href="http://www.google.com/transparencyreport/userdatarequests/US/" href="https://www.google.com/transparencyreport/userdatarequests/US/" rel="nofollow noopener noreferrer" target="_blank">
    24,000份请求
   </a>
   、搜查令或法院命令。直接的国家安全或反恐要求只是这一整体要求中的极小一部分，但爱国者法案的法律程序现在变得如此常规，以至于每个公司都有一组员工，他们的工作只是专注于处理这些请求 — — 也就是专门负责破坏用户隐私。
  </p>
  <p class="graf graf--p graf-after--p graf--trailing" id="a6e6">
   通过这种方式，情报界与大型商业科技公司之间的合作取得了巨大的成功。当这些间谍机构需要识别和跟踪人员和群体时，他们知道应该找谁 — — 并经常这样做。这是一开始就确立了的目标。如今它的“成功”超乎曾经所有人的想象。
