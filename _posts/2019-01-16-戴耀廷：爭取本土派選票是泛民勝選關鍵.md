 </p>
 <p>
 </p>
 <p>
  <span style="font-size: 12pt;">
   香港大學法律副教授戴耀廷日前接受記者採訪時表示，去年的兩次立法會補選中泛民均失利的主因是支持本土派的選票流失，削弱了非建制陣營的總體支持率優勢，對今年的區議會選舉和明年的立法會選舉是嚴峻的挑戰。泛民應積極研究如何才能重新獲得本土票的支持。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   戴耀廷說，雖然非建制派的總體支持率佔55%-58%，但當中約四分之一是那些不出來投票的本土派支持者，在補選時導致泛民不夠建制派的票數。建制派實際上可以控制的鐵票和穩定支持率接近百分之四十，加上一些游動過來的中間票，大概可以達到42%，泛民的鐵票不如建制派多，如果本土派支持者的票能夠爭取回來，泛民的票數就多過建制，而且可以贏很多。去年兩次補選泛民輸了，其實不是建制派的票多了，而是泛民的票少了。除了本土派的票要爭取，兩頭游動的中間選民票也是要爭取的，如果泛民團結，選戰的勢頭好，中間選民是可以更多地靠過來的。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   泛民目前面臨的挑戰是如何說服本土派和中間選民投票給泛民候選人。戴耀廷分析，有些本土意識的年輕人可能恨泛民多過共產黨，這些人只是極少數。有些人則覺得泛民多年的努力是徒勞的，泛民就需要給他們信心，讓他們認為泛民做的事情並非無用，是可以改變現狀的。要說服這些年輕人，只要他們出來投票，非建制派在2020的立法會選舉將能夠取得多少席，如果民主派在地區直選中拿到22-23票，加上功能組別，席位可以達到35，在立法會席位中佔一半，整個遊戲規則就會完全改變。有些中間選民因為看到民主派內部有矛盾，便對投票意興闌珊。還有一些潛在支持者要看民主派的勢頭如何，所謂“西瓜靠大邊”，如果泛民做出選戰的強勢，他們就會出來投票。面對非建制派支持者中的這些思想和行為，就需要泛民處理好內部的問題，給外部積極投票的信心。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   戴耀廷指的泛民內部矛盾是去年第二次立法會補選時馮檢基與李卓人名單的衝突。他說，如果馮檢基不出來參選，並且號召他的支持者投票給李卓人，那麼李卓人有可能以微弱票數勝出。馮檢基的部分“關係票”會投向李卓人，一些看重民主派團結的中間選民的票也會相應增加。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   戴耀廷認為，今年底的區議會選舉是明年立法會選舉的前哨戰，對非建制陣營反擊建制派很重要。在區議會選舉中，可以用交叉支持方式解決泛民與本土派的矛盾。例如採用建立地區聯盟方式，在一些地區，本土派和泛民候選人即使不在一個選區，但是可以交叉支持投票，即泛民陣營動員支持者投票給鄰近選區的本土派候選人，而本土派陣營也動員支持者投票給另一區的泛民候選人，達到一種選票交換的效應。選舉聯盟如果順利成立和運作，便會促進泛民和本土派的選戰合作，進而擴大至各個抗爭領域。同時，可以給那些中間派選民一個信心，堅信非建制派一起努力就可以邁向議會的控制權。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
 </p>
 <p>
  <span style="font-size: 12pt;">
   戴耀廷表示，泛民把今年的區議會選舉作為一個反攻的機會，如果反擊失敗，可能會導致2020立法會選舉失敗。如果今年的區議會選舉，泛民能夠拿到多於現在的120席，達到150-160席，就可以稱之為勝利了，然後再部署立法會選舉，實行再一次的機會。
  </span>
  <br/>
  <span style="font-size: 12pt;">
  </span>
  <br/>
  <span style="font-size: 12pt;">
   對於最近爭議很大的“國歌法”，戴耀廷認為此法完全沒有必要，把“國歌法”內的行為刑事化，有違香港講求自由的價值觀。美國也有“國歌法”，但是沒有刑責。香港政府推“國歌法”，目的是加速中港融合，把法律的功能大陸化，意識形態大陸化。
  </span>
 </p>
