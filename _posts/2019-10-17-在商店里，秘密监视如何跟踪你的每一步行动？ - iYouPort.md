  <p class="graf graf--p">
   你正在你喜欢的超市里购物。
  </p>
  <p class="graf graf--p">
   当你接近乳品过道时，你的手机忽然响起来了，是一条通知：“您最喜欢的酸奶现在可享受10％的优惠！点击此处兑换优惠券。“
  </p>
  <p class="graf graf--p">
   你上一次来这里购物原本想要买酸奶，而最终没买。但是，你的手机是怎么知道这件事的？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    因为您的智能手机一直在跟踪您。
   </strong>
  </p>
  <p class="graf graf--p">
   杂货店获得了您的位置数据，并向一群有影响力的营销人员付费使用这些信息来制定针对您的广告。
  </p>
  <p class="graf graf--p">
   ⚠️最近的报告只是指出了监视资本主义公司如何使用从手机信号塔、环境Wi-Fi和GPS收集数据。但是，位置数据行业还有一个更加精确的、不引人注目的工具：蓝牙信标。
  </p>
  <p class="graf graf--p">
   这些信标是隐藏在整个杂货店内的小型的、不引人注目的电子设备；手机上与他们通信的应用程序一直在将您的一举一动告诉公司：不仅是您进入了建筑物，而且还包括您在低脂肪 Chobanis 面前徘徊了两分钟。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5796 jetpack-lazy-image" data-lazy-sizes="(max-width: 978px) 100vw, 978px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-252.png?resize=978%2C508&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-252.png?w=978&amp;ssl=1 978w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-252.png?resize=300%2C156&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-252.png?resize=768%2C399&amp;ssl=1 768w" data-recalc-dims="1" height="508" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-252.png?resize=978%2C508&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="978"/>

   </noscript>

  <p class="graf graf--p">
   大多数定位服务使用手机信号塔和GPS，但这些技术都有局限性。
  </p>
  <p class="graf graf--p">
   手机信号塔覆盖范围广，但定位精度低：比如监视者会认为你在 Walgreens，但你实际上在隔壁麦当劳。
  </p>
  <p class="graf graf--p">
   相比之下，GPS可以精确到半径约5米（16英尺），不过它在室内不能很好地工作。
  </p>
  <p class="graf graf--p">
   但是，如上图所示，
   <strong class="markup--strong markup--p-strong">
    蓝牙信标可以精准地跟踪您的位置。这种东西还非常节能，并且在室内也能很好用。
   </strong>
  </p>
  <p class="graf graf--p">
   这使得那些想要在商店内进行精确跟踪的公司很喜欢这东西。
  </p>
  <p class="graf graf--p">
   它的工作原理是这样的：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5797 jetpack-lazy-image" data-lazy-sizes="(max-width: 1080px) 100vw, 1080px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/零售店监视长图.jpeg?resize=1080%2C1910&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/零售店监视长图.jpeg?w=1080&amp;ssl=1 1080w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/零售店监视长图.jpeg?resize=170%2C300&amp;ssl=1 170w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/零售店监视长图.jpeg?resize=768%2C1358&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/零售店监视长图.jpeg?resize=579%2C1024&amp;ssl=1 579w" data-recalc-dims="1" height="1910" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/零售店监视长图.jpeg?resize=1080%2C1910&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1080"/>

   </noscript>

  <p class="graf graf--p">
   绝大多数人都不知道自己正在被信标监视，但是 “beacosystem” 每天都会追踪数百万人。
  </p>
  <p class="graf graf--p">
   信标被放置在机场、商场、地铁、公共汽车、出租车、体育竞技场、健身房、酒店、医院、音乐节、电影院和博物馆 …… 甚至广告牌上。（广告牌上可不只有信标，还有人脸识别监视，详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%B8%BA%E4%BB%80%E4%B9%88%E8%A6%81%E4%BD%A0%E8%BF%9C%E7%A6%BB%E5%B9%BF%E5%91%8A%E7%89%8C-%E8%80%81%E5%A4%A7%E5%93%A5%E6%9D%A5%E4%BA%86/" href="https://www.iyouport.org/%E4%B8%BA%E4%BB%80%E4%B9%88%E8%A6%81%E4%BD%A0%E8%BF%9C%E7%A6%BB%E5%B9%BF%E5%91%8A%E7%89%8C-%E8%80%81%E5%A4%A7%E5%93%A5%E6%9D%A5%E4%BA%86/" rel="noopener noreferrer" target="_blank">
    为什么要你远离广告牌
   </a>
   》）
  </p>
  <p class="graf graf--p">
   为了跟踪您并触发将消息传送到您的手机的操作，这些公司需要您在手机上安装可识别商店中信标的应用程序。
  </p>
  <p class="graf graf--p">
   使用蓝牙信标的零售商（如 Target 和 Walmart 等）通常会在他们自己的应用程序中构建跟踪。
  </p>
  <p class="graf graf--p">
   ⚠️但零售商希望确保能够跟踪大多数客户 —— 而不仅仅是下载他们自己的特定应用的客户。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    正因为需求量如此之大，第三方位置营销公司的隐藏行业已经激增并蓬勃发展。
   </strong>
  </p>
  <p class="graf graf--p">
   这些公司采用他们的信标跟踪代码并将其捆绑到开发人员可以使用的工具包中。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    许多流行应用程序的制造商 —— 例如新闻或天气更新的应用程序 —— 将这些工具包插入到他们自己的应用程序中。
   </strong>
  </p>
  <p class="graf graf--p">
   它们可能由信标公司支付或从中获得其他好处，例如，获得有关其用户的详细报告。
  </p>
  <p class="graf graf--p">
   位置数据公司经常收集各种应用程序提供的数据。
  </p>
  <p class="graf graf--p">
   例如，一家名为 Pulsate 的位置公司
   <a class="markup--anchor markup--p-anchor" data-href="https://pulsate.readme.io/docs/passing-user-information-to-pulsate" href="https://pulsate.readme.io/docs/passing-user-information-to-pulsate" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     鼓励应用开发者向他们传递客户的电子邮件地址和名字
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   像 Reveal Mobile 这样的公司
   <strong class="markup--strong markup--p-strong">
    从
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.businesswire.com/news/home/20160324005723/en/Reveal-Mobile-Unacast-Partner-Solve-Bluetooth-Beacons%E2%80%99" href="https://www.businesswire.com/news/home/20160324005723/en/Reveal-Mobile-Unacast-Partner-Solve-Bluetooth-Beacons%E2%80%99" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     数百个
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    常用应用程序中的软件开发工具包中收集数据。
   </strong>
  </p>
  <p class="graf graf--p">
   在美国，另一家公司 inMarket 覆盖了38％的千禧代妈妈和大约
   <a class="markup--anchor markup--p-anchor" data-href="https://youtu.be/eS8CxObMNoI?t=375" href="https://youtu.be/eS8CxObMNoI?t=375" rel="noopener noreferrer" target="_blank">
    四分之一
   </a>
   的智能手机，
   <strong class="markup--strong markup--p-strong">
    每月跟踪
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://go.inmarket.com/blog/how-have-amazons-price-reductions-impacted-whole-foods-foot-traffic" href="https://go.inmarket.com/blog/how-have-amazons-price-reductions-impacted-whole-foods-foot-traffic" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     5000万人
    </strong>
   </a>
   。这场游戏中的其他玩家都有类似的影响力。
  </p>
  <p class="graf graf--p">
   位置数据公司还有其他令人不安的伎俩。
  </p>
  <p class="graf graf--p">
   例如，⚠️inMarket 开发了
   <strong class="markup--strong markup--p-strong">
    “
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://geomarketing.com/beacon-marketing-platform-inmarket-introduces-predictive-targeting-program" href="https://geomarketing.com/beacon-marketing-platform-inmarket-introduces-predictive-targeting-program" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     心态定位
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ”技术
   </strong>
   ，可以预测一个人何时最容易轻信广告。然后就在那个时候对你下手。
  </p>
  <p class="graf graf--p">
   这些技术基于通过数百万人类行为观察计算出的统计概率。这就是监视资本主义的逻辑：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%E8%B0%B7%E6%AD%8C%E6%8E%A7%E5%88%B6%E7%9A%84%E4%B8%96%E7%95%8C%E5%92%8C%E7%9B%91%E8%A7%86%E8%B5%84%E6%9C%AC%E4%B8%BB%E4%B9%89-2/" href="https://www.iyouport.org/%E8%B0%B7%E6%AD%8C%E6%8E%A7%E5%88%B6%E7%9A%84%E4%B8%96%E7%95%8C%E5%92%8C%E7%9B%91%E8%A7%86%E8%B5%84%E6%9C%AC%E4%B8%BB%E4%B9%89-2/" rel="noopener noreferrer" target="_blank">

    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/70%e5%b9%b4%e5%90%8e%ef%bc%8c%e8%bf%98%e6%98%af1984/" href="https://www.iyouport.org/70%e5%b9%b4%e5%90%8e%ef%bc%8c%e8%bf%98%e6%98%af1984/" rel="noopener noreferrer" target="_blank">
     70年后，还是1984
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   像 Hellman’s、Heineken 和 Hillshire Farms 这样的品牌已经使用这些技术来推动产品活动。
  </p>
  <p class="graf graf--p">
   位置营销旨在了解
   <a class="markup--anchor markup--p-anchor" data-href="https://streetfightmag.com/2017/02/09/swirl-launches-direct-facebook-integration-for-online-to-offline-attribution/" href="https://streetfightmag.com/2017/02/09/swirl-launches-direct-facebook-integration-for-online-to-offline-attribution/" rel="noopener noreferrer" target="_blank">
    “在线到线下归因”
   </a>
   。例如，如果星巴克咖啡广告给您发了电子邮件，营销人员想知道您是否真的去过那里并购买了咖啡。
  </p>
  <p class="graf graf--p">
   这些公司如何“了解”？很简单，那就是
   <strong class="markup--strong markup--p-strong">
    持续监控您的在线和离线的一切行为习惯
   </strong>
   。
  </p>
