 <div class="entry-content">
  <p>
   作者是海军陆战队退伍军人和亚利桑那州的前国会候选人。他有近 80,000 名粉丝，现在被推特封锁了，他说他并不在乎，因为他开启了新帐户。于是这篇文章并不是关于个人的，而是关于我们正在拥有的是一个什么样的国家，以及我们想要什么样的国家。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-9670 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/12/推特.jpg?resize=1024%2C768&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/12/推特.jpg?w=1024&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/12/推特.jpg?resize=300%2C225&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/12/推特.jpg?resize=768%2C576&amp;ssl=1 768w" data-recalc-dims="1" height="768" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/12/推特.jpg?resize=1024%2C768&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9670" data-recalc-dims="1" height="768" sizes="(max-width: 1024px) 100vw, 1024px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/12/推特.jpg?resize=1024%2C768&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/12/推特.jpg?w=1024&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/12/推特.jpg?resize=300%2C225&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/12/推特.jpg?resize=768%2C576&amp;ssl=1 768w" width="1024"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   我一直在尽己所能，不要抱怨自己走过的人生路线，但我希望被人们理解成一个有远见的人，
   <a class="markup--anchor markup--p-anchor" data-href="https://thefederalist.com/2018/08/23/the-left-wont-stop-at-alex-jones-we-are-sliding-down-a-slippery-slope/" href="https://thefederalist.com/2018/08/23/the-left-wont-stop-at-alex-jones-we-are-sliding-down-a-slippery-slope/" rel="noopener noreferrer" target="_blank">
    能遇见未来的人
   </a>
   。
  </p>
  <p class="graf graf--p">
   但是，正如您所知，Twitter昨天禁止了我的帐户 —— 不是暂停，而是永久封锁。
  </p>
  <p class="graf graf--p">
   我有将近80,000名追随者，这些可怜的人现在漫无目的地在社交媒体上徘徊，以寻找他们再找不到的内容。
  </p>
  <p class="graf graf--p">
   现在，我不太在乎，因为我将要开设一个新帐户，并且它会比我的上一个帐户更好（如果可能的话）。
  </p>
  <p class="graf graf--p">
   这不是关于我个人的。这是关于我们拥有一个什么样的国家，以及我们想要成为什么样的国家。
  </p>
  <p class="graf graf--p">
   我们已经成为一群敏感的失败者，只纠结于一言一语；我们仅仅关心事物“让我们如何感觉”。如今的人们只是想着如何把自己的才能和思想放到市场上等着有人愿意去买。
  </p>
  <p class="graf graf--p">
   在美国，希望言论自由受到审查的人已经取代了言论自由的精神。这在社交媒体世界中最为明显。
  </p>
  <p class="graf graf--p">
   正如我之前
   <a class="markup--anchor markup--p-anchor" data-href="https://thefederalist.com/2018/08/23/the-left-wont-stop-at-alex-jones-we-are-sliding-down-a-slippery-slope/" href="https://thefederalist.com/2018/08/23/the-left-wont-stop-at-alex-jones-we-are-sliding-down-a-slippery-slope/" rel="noopener noreferrer" target="_blank">
    所说过的
   </a>
   ，社交媒体不是一件小事。不再是三个书呆子挤在宿舍里幻想着一个女人承认自己存在的日子。
  </p>
  <p class="graf graf--p">
   社交媒体已经超越了电话。这是与他人建立联系和交流的基本手段：25亿人使用 Facebook 和 Twitter。
  </p>
  <p class="graf graf--p">
   那不是消失的边缘事物。现在，它已成为人类相互交流的方式。
  </p>
  <p class="graf graf--p">
   它完全由知道自己拥有多大权力的硅谷寡头经营着。他们正在使用这种至高无上的权力。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    这种至高无上的权力是什么？《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%9b%bd%e7%8e%8b%e4%bb%ac%e7%9a%84%e6%99%9a%e5%ae%b4-%e8%bf%99%e6%98%af%e4%b8%80%e5%9c%ba%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e6%94%bf%e5%8f%98/" href="https://www.iyouport.org/%e5%9b%bd%e7%8e%8b%e4%bb%ac%e7%9a%84%e6%99%9a%e5%ae%b4-%e8%bf%99%e6%98%af%e4%b8%80%e5%9c%ba%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e6%94%bf%e5%8f%98/" rel="noopener noreferrer" target="_blank">
     国王们的晚宴
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   但是权力是一件有趣的事。权力，无论当时看起来多么凶恶，总是有限的。
  </p>
  <p class="graf graf--p">
   它不会永远持续下去。如果历史能告诉我们点什么的话，那就是，消音永远是只一个临时“解决方案”。
  </p>
  <p class="graf graf--p">
   审查制度是一件可怕的事，并且，它有一个致命的缺陷：它行不通。声音已经爆发出来了，它们就不能被掩盖。
  </p>
  <p class="graf graf--p">
   从长远来看 Twitter 禁止我进入他们的平台只会伤害他们自己。他们将继续边缘化自己，而我将继续成长。
  </p>
  <p class="graf graf--p">
   我喜欢谈论政治，也喜欢谈论人。我需要Twitter来发声。我的广播节目很成功。我很高兴有一群朋友可以分享和交流彼此的想法。
  </p>
  <p class="graf graf--p">
   但是更大的问题是：它会在哪里结束？我们如何容纳异议人士？压制我们不喜欢的声音可能会导致更好的社会吗？被沉默的声音最终会怎样？我们是否能通过强迫遵守一个与许多美国人的核心价值观背道而驰的学说来帮自己一个忙？……
  </p>
  <p class="graf graf--p">
   我们将找出这些问题的答案。我们将解决这些问题。
  </p>
  <p class="graf graf--p">
   我们将学会拒绝，学会向不同意见的人表达自己的意见。Twitter不会，但是美国比Twitter更好。一向如此。⚪️
  </p>
