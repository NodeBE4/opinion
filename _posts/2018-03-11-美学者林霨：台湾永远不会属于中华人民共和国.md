title: 美学者林霨：台湾永远不会属于中华人民共和国
---

<div id="entry">
<div class="at-above-post addthis_tool" data-url="http://zhanlve.org/?p=4747">
</div>
<p>
<img class="aligncenter" src="https://gdb.voanews.com/89E6D4AC-E376-4F51-AA61-644D4C70B471_w1023_r1_s.jpg"/>
</p>
<p>
</p>
<p>
<span style="font-size: 12pt;">
   北京全国两会期间，中国领导人的涉台讲话及人事布局不仅引起台湾各界关心，关注台海情势的人也在注意。就在中国政府强调要推进“祖国和平统一进程”之际，一位美国学者说，“台湾永远不会成为中华人民共和国的一部分”。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   宾州大学国家关系及中国历史学教授林霨(Arthur Waldron)指出，要谈论台海两岸问题，必须从一个他认为是“最简单的事实”开始，那就是“台湾永远不会成为中华人民共和国的一部分”，因为早在1979年蒋经国全面掌权，北京当局无法让他接受关于统一的谈判时，中共就已错失可以实现这个目标的良机。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   林霨在全球台湾研究中心(GTI)最新一期的《全球台湾简报》(Global Taiwan Brief)发表的一篇关于2018年台海两岸关系的文章中指出，当中华人民共和国代表团在联合国取代中华民国席位时，“没有发表任何官方声明说台湾是中国一部分”，“这个事实一直困扰着(haunted)周恩来”，因为他发现“没有任何国际性文件明确提到这一点”。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   林蔚说，台湾被排除在国际组织外唯一的原因就是中国的强力干预，以及美国承认台湾是一个国家的事实，不过尽管如此，“美国从来没有同意过中国对台湾拥有主权。”
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   他说，如今东亚局势已经发生巨变，中国的军事扩张导致美国对尼克松总统以来的政策做“最深切的重新思考”，川普政府如今已“不再随中国起舞”(dance to China’s tune)，许多亚洲国家也开始合作对中国进行威慑。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   林霨指出，日本去年底在距离台湾花莲150英里的石垣岛建设军事基地，意味着日本“理解到中国一旦控制台湾将对日本的生存构成威胁”，因此即使美国“撕毁40年来对台湾安全的承诺”而不协防台湾，“日本会这么做”，因为“侵略台湾就等于侵略日本”。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   《中新社》报道，中国国台办主任张志军星期二在参加福建代表团审议政府工作报告时，解释了政府报告中涉台部分的多层意思，其中一层强调需要坚守的“红线”，是“绝不容忍任何‘台独’分裂图谋和行径”。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   另一层则提到对台工作任务，他说，过去很多都只是讲两岸关系和平发展，但这次还提到“推进祖国和平统一进程”。他认为“这个意义是重要的”。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   张志军还指出，国台办上周推出的31条措施，“为台湾同胞，特别是年轻人”，到中国学习、创业、就业、生活提供更多便利，促进两岸经济社会融合发展。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   不过，台湾中央研究院社会研究所副研究员吴介民认为，这31条着重在吸引台湾青年到中国就业创业的“惠台”措施，是中国对台湾“用商业模式做统战”的“中国因素进化版”，由于台湾在寻求主体性过程中的这种“中国因素”，两岸融合并非易事。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   吴介民星期三在台湾研究基金一场论坛中提到，他自2011年到2017年的6次关于统独立场的电话调查发现，赞成台湾独立的年轻人有增加的趋势，“台独年轻化”的现象确实存在，虽然2年来支持台湾独立的年轻人比例下降，但下降的部分并没有转为支持统一或是对中国的认同，而是改为支持维持现状，这表示维持现状的中间立场已成为台湾人民处理认同问题的“调节器”和“避风港”，“统派的市场、中国人认同的市场，在台湾已完全失去吸引力”。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   吴介民说，面对中国崛起，台湾现在已经对习近平所宣示的“中华民族伟大复兴”的“中国梦”出现一种“社会抵抗”，因为习近平正在寻求“中华帝国的重建”，一旦他的任期不受限制即可“称帝”，而台湾人民要的是自己的主体性，不要成为这个帝国的一部分。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   中研院台湾历史研究所副所长张隆志在论坛上表示，台湾现在在如何面对中国的课题上有高度危机感，值得思考的是过去20多年来台湾的历史教育改革除了强调以台湾为主之外，是否也同时提供一个“新的认识中国的管道、中国观、中国的知识”？
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
   张隆志说，台湾是一个多元、宽容的民主社会，在坚持这种价值观和生活方式的同时，台湾必须在重新建构主体性的过程中，从公共政策及治理的层面上，思考如何建立一个更宽容及理性的对话氛围。
  </span>
</p>
