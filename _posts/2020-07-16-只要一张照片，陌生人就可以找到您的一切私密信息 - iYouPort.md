      <strong>
       一个鲜为人知的初创企业正在帮助警察将陌生人的照片与他们的在线图像进行匹配，它已经彻底终结了公开场合的匿名性 …… 这篇报道中出现了很多有趣的细节，仔细看
      </strong>
     </em>

  <p class="graf graf--p">
   直到最近，Hoan Ton-That 的 iPhone 游戏可以让人们将特朗普独特的黄色头发放在自己的照片上，仍然不那么起眼。
  </p>
  <p class="graf graf--p">
   然后，Ton-That 先生（这位澳大利亚技术员和曾经的模特）做了一件大事：他发明了一种工具，
   <strong class="markup--strong markup--p-strong">
    可以终结街头任何人的匿名性，并将该工具提供给了数百个执法机构
   </strong>
   ，从佛罗里达州的当地警察到联邦调查局、还有国土安全部。
  </p>
  <p class="graf graf--p">
   他的小公司 Clearview AI 设计了一款突破性的面部识别应用程序。该应用程序可以允许您上传任何想要的人的照片，然后你就可以看到该人的所有公开照片、以及指向这些照片出现在哪里的链接。
  </p>
  <p class="graf graf--p">
   该系统的骨干是一个数据库，其中包含 Clearview 声称从 Facebook、YouTube、Venmo 和数百万其他网站中抓取的30亿张图像，
   <strong class="markup--strong markup--p-strong">
    它远远超出了美国政府或硅谷巨头建造的任何数据库
   </strong>
   。
  </p>
  <p class="graf graf--p">
   联邦和州执法人员说，尽管他们对 Clearview 的工作原理以及背后的人知之甚少，但他们还是使用该公司创建的应用程序来帮助抓人。
  </p>
  <p class="graf graf--p">
   到目前为止，由于其对隐私的彻底剥夺，可以轻易根据任何人面部识别每个人的技术一直是禁忌。有能力发布此类工具的技术公司还没有这样做。 包括旧金山在内的一些大城市已经禁止警察使用面部识别技术。
  </p>
  <p class="graf graf--p">
   但该公司表示，在没有经过公众审查的情况下，
   <strong class="markup--strong markup--p-strong">
    去年有600多家执法机构已经开始使用 Clearview。
   </strong>
  </p>
  <p class="graf graf--p">
   纽约时报分析了该应用程序代码，其中包括将其与增强现实眼镜搭配使用的编程语言。也就是说用户将有可能识别出他们看到的每个人。
  </p>
  <p class="graf graf--p">
   从参加抗议活动的所有人、到地铁上的陌生人，该应用程序能识别几乎一切 —— ⚠️
   <strong class="markup--strong markup--p-strong">
    不仅揭示目标人的姓名，而且还揭示他们的住所、他们的一举一动、和认识的其他人
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这种监视工具不仅被卖给了警察，Clearview 还将其许可给了多家公司。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这种监视工具被武器化的可能性无穷无尽，” 圣塔克拉拉大学高科技法律研究所联合主任 Eric Goldman 说。
  </p>
  <p class="graf graf--p">
   Clearview 一直保密，避免对其推动技术的争论。当记者11月份开始调查该公司时，其网站上只有一个空白页面，显示不存在曼哈顿的营业地点。
  </p>
  <p class="graf graf--p">
   该公司在 LinkedIn 上列出的一名员工，名为 “ John Good ”的销售经理，实际上就是 Ton-That 本人，他用了一个假名。一个月以来，与该公司有关联的人都没有回复记者的电子邮件或电话。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在公司躲避记者的同时，它也在监视记者
   </strong>
   。应记者的要求，许多警察已通过 Clearview 应用程序搜索了这位记者的照片。但他们很快接到该公司代表打来的电话，询问他们是否在与媒体对话。
   <strong class="markup--strong markup--p-strong">
    这表明 Clearview 有能力也有意愿监视警察都搜索了什么人。
   </strong>
  </p>
  <p class="graf graf--p">
   面部识别技术一直存在争议，人们对这种老大哥格外紧张。它倾向于为某些群体提供错误的匹配，例如有色人种；而且，一些警察使用的面部识别产品（包括 Clearview 的产品）尚未得到独立专家的审核。
  </p>
  <p class="graf graf--p">
   Clearview 的应用程序带来了额外的风险，因为
   <strong class="markup--strong markup--p-strong">
    执法机构将敏感照片上传到该公司的服务器上，这些公司的数据保护能力未经测试
   </strong>
   。
  </p>
  <p class="graf graf--p">
   该公司最终开始回答了记者的问题，称其较早的沉默是 “在隐身模式下进行早期启动的典型表现”。
  </p>
  <p class="graf graf--p">
   Ton-That 先生承认设计了用于增强现实眼镜的原型，但表示该公司没有计划发布它。他说，记者的照片敲响了警钟，因为该应用程序 “标记了可能的异常搜索行为”，以防止用户进行其认为 “不适当的搜索”。
  </p>
  <p class="graf graf--p">
   除 Ton-That 外，Clearview 的
   <strong class="markup--strong markup--p-strong">
    另一位联合创始人是 Richard Schwartz （他曾经是前纽约市市长 Rudolph W. Giuliani 的助手），并得到了 Facebook 和 Palantir 背后的风险投资家 Peter Thiel 的财政支持
   </strong>
   。
  </p>
  <p class="graf graf--p">
   另一个早期投资者是一家名为 Kirenaga Partners 的小型公司。它的创始人 David Scalzo 消除了对 Clearview 使人脸可搜索的担忧，称它是 “解决犯罪的宝贵工具”。
  </p>
  <p class="graf graf--p">
   Scalzo 说：“我得出的结论是，由于信息不断增加，因此永远不会有隐私。” [？？？？？]“法律必须确定什么是合法的，但您不能禁止监视技术。当然，这可能会导致反乌托邦的未来或其他事物，但您不能禁止它。” 真能扯蛋。
  </p>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    沉迷于AI
   </strong>
  </h3>
  <p class="graf graf--p">
   31岁的 Ton-That 距离硅谷很远。在他的祖国澳大利亚，他是在越南的皇室祖先的故事中长大的。2007年，他辍学，搬到了旧金山。他的目标是尽早进入他期望的充满活力的社交媒体应用程序市场。但是他的早期冒险从未获得真正的吸引力。
  </p>
  <p class="graf graf--p">
   2009年，Ton-That 创建了一个网站，使人们可以与即时通讯工具中的所有联系人共享视频链接。在被冠以 “
   <a class="markup--anchor markup--p-anchor" data-href="https://bits.blogs.nytimes.com/2009/02/24/viddyho-phishing-scam-hits-gmail/" href="https://bits.blogs.nytimes.com/2009/02/24/viddyho-phishing-scam-hits-gmail/" rel="noopener noreferrer" target="_blank">
    网络钓鱼骗局
   </a>
   ” 后该网站被关闭。
  </p>
  <p class="graf graf--p">
   2015年，他开始鼓捣
   <a class="markup--anchor markup--p-anchor" data-href="https://www.148apps.com/app/1040750174/" href="https://www.148apps.com/app/1040750174/" rel="noopener noreferrer" target="_blank">
    特朗普的头发
   </a>
   ，在照片共享应用程序中让用户可以把特朗普的头发添加到自己的照片上。两个创业都失败了。
  </p>
  <p class="graf graf--p">
   Ton-That 沮丧不已，于2016年移居纽约。他说，他考虑过从事模特职业，但是经过一番拍摄之后，他又回过头去重新尝试找出科技领域的下一件大事。
  </p>
  <p class="graf graf--p">
   他开始阅读有关人工智能、图像识别和机器学习的学术论文。
  </p>
  <p class="graf graf--p">
   Schwartz 和 Ton-That 于2016年在曼哈顿研究所（一个保守派智囊团）举行的读书会上相识。
  </p>
  <p class="graf graf--p">
   Schwartz 现年61岁，在1990年代为前市长 Giuliani 工作时
   <strong class="markup--strong markup--p-strong">
    积累了令人印象深刻的人脉资源
   </strong>
   ，并在2000年代初担任《纽约每日新闻》的社论版编辑。
  </p>
  <p class="graf graf--p">
   两人很快决定一起从事面部识别业务：Ton-That 将创建该应用程序，而 Schwartz 负责使用他广泛的人脉资源来激发市场的兴趣。
  </p>
  <p class="graf graf--p">
   警察部门使用面部识别工具已有
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2020/01/12/technology/facial-recognition-police.html" href="https://www.nytimes.com/2020/01/12/technology/facial-recognition-police.html" rel="noopener noreferrer" target="_blank">
    近20年的历史
   </a>
   了，但是从历史上看，他们只能搜索政府数据库里提供的图像，例如证件大头照和驾照。
  </p>
  <p class="graf graf--p">
   近年来，面部识别算法的准确性有所提高，亚马逊等公司提供的产品可以为任何图像数据库创建面部识别程序。
  </p>
  <p class="graf graf--p">
   Ton-That 想超越这一范围。他从2016年开始招募了一些工程师。一个人帮助设计了一个程序，
   <strong class="markup--strong markup--p-strong">
    该程序可以自动收集整个互联网上的人脸图像
   </strong>
   ，例如 就业网站、新闻网站、教育网站、以及社交网络，包括 Facebook，YouTube，Twitter，Instagram 甚至 Venmo。这些公司的代表表示，他们的政策禁止这种抓取行为，
   <a class="markup--anchor markup--p-anchor" data-href="https://developer.twitter.com/en/developer-terms/more-on-restricted-use-cases" href="https://developer.twitter.com/en/developer-terms/more-on-restricted-use-cases" rel="noopener noreferrer" target="_blank">
    Twitter 说
   </a>
   ，它明确禁止使用其数据进行面部识别。
  </p>
  <p class="graf graf--p">
   他雇用的另一名工程师负责完善源自学术论文的面部识别算法。
  </p>
  <p class="graf graf--p">
   结果是：一个系统诞生了，使用的是 Ton-That 所说的 “最先进的神经网络”，可以根据面部几何形状将所有图像转换为数学公式或向量，例如人的眼间距。Clearview 创建了一个庞大的目录，将具有相似矢量的所有照片聚集在一起。
  </p>
  <p class="graf graf--p">
   当用户将一张头像的照片上传到 Clearview 的系统中时，它就会将照片中的面部转换为向量，然后显示它附近存储的所有抓取到的其他照片 —— 以及指向这些图片来源网站的链接。
  </p>
  <p class="graf graf--p">
   Schwartz 支付了服务器成本和基本费用，但不包括操作。每个人都在家工作。Ton-That 说：“我靠信用卡欠债生活。此外，我是比特币信徒，所以我存了一些比特币。”
  </p>

  <h3 class="graf graf--p">

    警察最喜欢这种东西
   </strong>

  <p class="graf graf--p">
   到2017年底，该公司拥有了一个强大的面部识别工具，就是 Smartcheckr。但是 Schwartz 和 Ton-Ton 不确定他们会把它卖给谁。
  </p>
  <p class="graf graf--p">
   也许可以用来装在家里审查保姆的工作，或作为监视摄像机的附加功能；或者用于建筑物大厅的保安人员或帮助酒店按客人的名字打招呼？ Ton-That 说：“我们想到了每个主意。”
  </p>
  <p class="graf graf--p">
   据称，2017年底，最怪异的推销活动之一是 Paul Nehlen，
   <strong class="markup--strong markup--p-strong">
    他是一名反犹主义者且自称为 “pro-white” 的共和党人
   </strong>
   ，在威斯康星州竞选国会议员，将 “非常规数据库” 用于所谓的 “极端异议研究”。提供给 Nehlen 的文件后来在网上发布。 Ton-That 说，该公司从未提供过此类服务。
  </p>
  <p class="graf graf--p">
   该公司很快将其名称更改为 Clearview AI，并开始向执法部门推广。
  </p>
  <p class="graf graf--p">
   那是该公司从外部投资者 Thiel 先生和 Kirenaga Partners 获得第一轮资金的时候。
  </p>
  <p class="graf graf--p">
   Thiel 尤其以秘密资助绿巨人霍根的诉讼而闻名，该诉讼使颇受欢迎的网站 Gawker 破产。Thiel 和 Ton-That 都是 Gawker 的负面文章的主题。
  </p>
  <p class="graf graf--p">
   Thiel 的发言人 Jeremiah Hall 说：“ 2017年，Peter 给了一位才华横溢的年轻创始人20万美元，两年后又转换为 Clearview AI 的股权。” “那是 Peter 唯一的贡献；他没有参与公司。”
  </p>
  <p class="graf graf--p">
   根据追踪初创企业投资的网站
   <a class="markup--anchor markup--p-anchor" data-href="https://pitchbook.com/profiles/company/232177-96" href="https://pitchbook.com/profiles/company/232177-96" rel="noopener noreferrer" target="_blank">
    Pitchbook 的数据
   </a>

  </p>
  <p class="graf graf--p">
   2月，印第安纳州警察开始对 Clearview 进行试验。他们在使用该应用程序后的20分钟内就解决了一个案例。两名男子在公园打架，其中一个人对另一个人开枪。一名旁观者用手机记录了这一罪行，因此警察可以通过 Clearview 的应用程序对持枪者的脸进行了搜索。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">

   </strong>

   </strong>

   <a class="markup--anchor markup--p-anchor" data-href="https://www.nist.gov/programs-projects/face-recognition-vendor-test-frvt-ongoing" href="https://www.nist.gov/programs-projects/face-recognition-vendor-test-frvt-ongoing" rel="noopener noreferrer" target="_blank">

  </p>
  <p class="graf graf--p">
   但是现任和前任执法人员均表示该应用程序有效。“对我们来说，测试就是要确认是不是奏效，” 前印第安纳州警察局局长 Cohen 说。

   <strong class="markup--strong markup--p-strong">
    Clearview 投资者 Thiel 先生就是 Facebook 董事会的成员
   </strong>
   。 Nancarrow 拒绝评论 Thiel 先生的个人投资。
  </p>
  <p class="graf graf--p">
   一些执法人员说，他们不知道上传的照片已发送到并存储在 Clearview 的服务器上。Clearview 尝试通过常见问题解答（F.A.Q.）提供给潜在客户的
   <a class="markup--anchor markup--p-anchor" data-href="https://int.nyt.com/data/documenthelper/6690-clearview-faq/c8b081a0bcca12e7903a/optimized/full.pdf#page=1" href="https://int.nyt.com/data/documenthelper/6690-clearview-faq/c8b081a0bcca12e7903a/optimized/full.pdf#page=1" rel="noopener noreferrer" target="_blank">
    文件
   </a>
   中写道其客户支持员工不会查看警察上传的照片。
  </p>
  <figure aria-describedby="caption-attachment-12664" class="wp-caption aligncenter" id="attachment_12664" style="width: 1730px">
   <img alt="" class="size-full wp-image-12664 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5-1.png?resize=1100%2C688&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5-1.png?w=1730&amp;ssl=1 1730w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5-1.png?resize=300%2C188&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5-1.png?resize=1024%2C640&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5-1.png?resize=768%2C480&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5-1.png?resize=1536%2C961&amp;ssl=1 1536w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5-1.png?resize=1100%2C688&amp;ssl=1 1100w" data-recalc-dims="1" height="688" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5-1.png?resize=1100%2C688&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

    <img alt="" class="size-full wp-image-12664" data-recalc-dims="1" height="688" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5-1.png?resize=1100%2C688&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5-1.png?w=1730&amp;ssl=1 1730w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5-1.png?resize=300%2C188&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5-1.png?resize=1024%2C640&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5-1.png?resize=768%2C480&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5-1.png?resize=1536%2C961&amp;ssl=1 1536w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5-1.png?resize=1100%2C688&amp;ssl=1 1100w" width="1100"/>
   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-12664">
    Clearview’s marketing materials, obtained through a public-records request in Atlanta.
   </figcaption>
  </figure>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Clearview 还聘请了乔治·W·布什总统领导的美国总检察长 Paul D. Clement，以缓解对该应用程序合法性的担忧
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在8月 Clearview 向包括亚特兰大警察局和佛罗里达州 Pinellas 县警长办公室在内的潜在客户提供的备忘录中，Clement 说，执法机构 “在使用 Clearview 时
   <strong class="markup--strong markup--p-strong">
    不违反
   </strong>
   联邦宪法或相关的生物识别和隐私法，也能达到预期的目的。”
  </p>
  <p class="graf graf--p">
   Clement 现在是 Kirkland＆Ellis 的合伙人，他写道，只要不是获得逮捕令逮捕他们的唯一依据，当局就
   <strong class="markup--strong markup--p-strong">
    不必
   </strong>
   告诉被告他们已通过 Clearview 被识别。Clement 没有回应多次置评请求。
  </p>
  <p class="graf graf--p">
   该备忘录似乎很有效。亚特兰大警察和皮涅拉斯县警长办公室很快开始使用 Clearview。
  </p>
  <p class="graf graf--p">
   由于警察上传了他们试图识别的人的照片，因此 Clearview 拥有越来越多的个人脸部数据，这些数据库引起了执法部门的关注。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该公司还可以操纵警方看到的结果
   </strong>
   。
  </p>
  <p class="graf graf--p">
   该公司意识到记者要警官通过该应用程序搜索记者的照片后，这位记者的脸被 Clearview 的系统标记，有一段时间没有显示匹配。当被问到这个问题时，Ton-That 笑了起来，称其为 “软件 bug”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他们正在做的事令人毛骨悚然，但是这类公司将会变得更多。斯坦福大学法学院的隐私教授 Al Gidari 说， “由于没有非常严格的联邦隐私法，我们全完蛋了。”
  </p>
  <p class="graf graf--p">
   Ton-That 说，他的公司仅使用公开可用的图片。他说，如果您在 Facebook 中更改了隐私设置，以使搜索引擎无法链接到您的个人资料，那么您的 Facebook 照片将不会包含在数据库中。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e6%9b%be%e7%bb%8f%e5%9c%a8%e7%a4%be%e4%ba%a4%e7%bd%91%e7%bb%9c%e4%b8%8a%e5%8f%91%e5%b8%83%e8%bf%87%e8%87%aa%e5%b7%b1%e7%9a%84%e7%85%a7%e7%89%87%ef%bc%9f%e4%bd%a0%e5%8d%b1%e9%99%a9%e4%ba%86/" href="https://www.iyouport.org/%e4%bd%a0%e6%9b%be%e7%bb%8f%e5%9c%a8%e7%a4%be%e4%ba%a4%e7%bd%91%e7%bb%9c%e4%b8%8a%e5%8f%91%e5%b8%83%e8%bf%87%e8%87%aa%e5%b7%b1%e7%9a%84%e7%85%a7%e7%89%87%ef%bc%9f%e4%bd%a0%e5%8d%b1%e9%99%a9%e4%ba%86/" rel="noopener noreferrer" target="_blank">
     你曾经在社交网络上发布过自己的照片？你危险了！
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    如果您的个人资料已经被抓取，那就为时已晚了。该公司将保留所有已刮取的图像，即使以后将其删除或隐藏也没有用了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   波士顿东北大学法律与计算机科学教授 Woodrow Hartzog 将 Clearview 视为美国应禁止面部识别的最新证据。
  </p>
  <p class="graf graf--p">
   他说，“我们依靠行业的努力来实现自警，不采用如此冒险的技术，但是现在这些大坝正在破裂，因为该技术的利润太高了。阻止它的唯一方法就是禁止它。”
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    每个人都知道你名字的世界
   </strong>
  </h3>
  <p class="graf graf--p">
   Ton-That 最近在 Clearview 公司位于曼哈顿的办公室接受采访时，亲自演示了该应用程序。他自拍并上传了照片。该应用程序提取了他的23张照片。
  </p>
  <p class="graf graf--p">
   然后，Ton-That 使用该应用拍了这位记者的照片。 所谓的 “软件bug” 已得到修复，⚠️
   <strong class="markup--strong markup--p-strong">
    现在记者的照片返回了许多结果，甚至可追溯到十年前，其中包括记者自己都从未见过的自己的照片
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当记者用手遮住鼻子和脸颊时，该应用程序仍然返回了7个正确的匹配项。
   </strong>
  </p>
  <p class="graf graf--p">
   警察和 Clearview 的投资者预测，其应用最终将向公众开放。
  </p>
  <p class="graf graf--p">
   Ton-That 说他不愿意向公众公开。他说：“总会有一群滥用它的坏人。”
  </p>
  <p class="graf graf--p">
   即使 Clearview 并未公开发布其应用，但由于这种禁忌已经被打破，因此模仿公司还将继续出现。
   <strong class="markup--strong markup--p-strong">
    通过脸孔搜索某人可能会变得像谷歌搜索名字一样容易。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️陌生人将能够收听敏感的对话，为任何目标人拍照并了解个人的秘密。沿着街道行走的人将立即被识别出真实身份 —— 只需单击几下即可获得他或她的家庭住址。这将预示着公开场合匿名的彻底终结。⚪️
  </p>

    The Secretive Company That Might End Privacy as We Know It
