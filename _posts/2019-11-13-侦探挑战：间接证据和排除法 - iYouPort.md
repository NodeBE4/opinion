  </p>
  <p class="graf graf--p">
   Sector035 完成了这个挑战。他使用两个参数过滤数据，以便先将可能的位置数量减少到更小，并最终找到了正确答案。

  <p class="graf graf--p">
   这就是排除法的思考方式。
  </p>
  <p class="graf graf--p">
   通过假设该位置可能位于累西腓 Recife（这是基于挑战游戏发起人

  </p>
  <p class="graf graf--p">
   思路就是这么简单。
  </p>
  <p class="graf graf--p">
   但是，即便有了明确的思路，这仍然是一项相对耗时的工作，但从整个巴西开始并取消所有不属于累西腓的地点，以及所有不是四车道的道路，就可以更快找到确切的位置。
  </p>
  <p class="graf graf--p">

   这意味着现在你可以立即排除所有其他位置了。
  </p>

  </p>
  <p class="graf graf--p">
   因此，很简单，仅将一个参数（列车编号）应用于检索就可以消除其他所有不正确的位置，可能的答案范围缩小到只有一个。
  </p>
  <p class="graf graf--p">
   正确的位置是
   <a class="markup--anchor markup--p-anchor" data-href="https://goo.gl/maps/hpBHT8ZG31jYNqrE9" href="https://goo.gl/maps/hpBHT8ZG31jYNqrE9" rel="noopener noreferrer" target="_blank">
    罗斯托克 — 瓦尔内明德
   </a>
   站。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    那么在罗斯托克的那艘船叫什么名字？
   </strong>
  </p>
  <p class="graf graf--p">
   现在是回答第二个问题的时候了。
  </p>
  <p class="graf graf--p">
   有几种方法可以解决这个问题。一种方法可能是尝试识别照片中的船只然后跟随它们的运动轨迹。
  </p>
  <p class="graf graf--p">
   例如第二张照片中显示的这艘船明显属于Aida邮轮公司。
  </p>
  <p class="graf graf--p">
   理论上您可以查看他们的船队，查看最近出现在罗斯托克的船及具体时间，以推测答案。
  </p>
  <p class="graf graf--p">
   在某些情况下这是适当的技术，但是，显然，这种做法也会非常耗时，并且在这种情况下还有更快的方法。

   照片拍摄的时间可能性跨度也很小。 Lars 没有指定日期，但有必要解决这个问题，虽然不在原始提问范围内。
  </p>
  <p class="graf graf--p">
   毕竟，如果我们甚至不知道他什么时候拍摄的，又怎么可能知道 Lars 看到了哪些船？排除法可以再次在这里派上用场。
  </p>
  <p class="graf graf--p">
   💡即 通过应用一些参数，可以消除所有不正确的日期并找到最终正确的日期 —— 缩小可能性，然后让它变得更小。
  </p>
  <p class="graf graf--p">
   罗斯托克港公布了游轮过去和未来的行程清单，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.rostock-port.de/en/cruise-shipping/port-calls/2019-port-calls.html" href="https://www.rostock-port.de/en/cruise-shipping/port-calls/2019-port-calls.html" rel="noopener noreferrer" target="_blank">
    在这里
   </a>
   。Lars 则透露，他在访问此处的当天看到了四艘游轮。
  </p>
  <p class="graf graf--p">
   这意味着可以通过
   <strong class="markup--strong markup--p-strong">
    将少于四艘游轮同时出现的所有日期取消
   </strong>
   ，以排除不正确的日期。
  </p>
  <p class="graf graf--p">
   通过上述清单，很明显，7月12日星期五是今年迄今为止唯一的一个“合格”日期，当时四艘或更多的游轮来到罗斯托克，所以此日必须是正确的（假设 Lars 是今年访问的此地）。

  </p>

   </a>

  </p>

   好了就是这么简单！
  </p>
