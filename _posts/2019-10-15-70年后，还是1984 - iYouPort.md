  <p class="graf graf--p">
   1947年10月，埃里克·布莱尔（Eric Blair）以笔名乔治·奥威尔（George Orwell）写信给 Secker &amp; Warburg 出版社的共同所有人。
  </p>
  <p class="graf graf--p">
   在那封信中，奥威尔说，他正处于小说草稿的“最后阶段”，称其为“最可怕的混乱”。
  </p>
  <p class="graf graf--p">
   为了完成这部小说，奥威尔在苏格兰的汝拉岛上隐居。他在第二年完成了这本书，将他最“可怕的混乱”变成了“1984”，这是20世纪最重要的小说之一。
  </p>
  <p class="graf graf--p">
   这部小说于1949年出版，今年年满70岁。
  </p>
  <p class="graf graf--p">
   周年纪念为所有人提供了一个机会：反思小说的意义、和最有价值但经常被忽视的教训。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    BigOther
   </strong>
  </p>
  <p class="graf graf--p">
   19年来，私营公司实行了前所未有的经济逻辑，监视资本主义劫持了互联网和数字技术。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%E4%BA%92%E8%81%94%E7%BD%91%E7%A9%B6%E7%AB%9F%E6%98%AF%E4%BB%80%E4%B9%88%EF%BC%9F/" href="https://www.iyouport.org/%E4%BA%92%E8%81%94%E7%BD%91%E7%A9%B6%E7%AB%9F%E6%98%AF%E4%BB%80%E4%B9%88%EF%BC%9F/" rel="noopener noreferrer" target="_blank">
     互联网究竟是什么
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   从2000年开始被谷歌发明以来，监视资本主义经济学秘密地声称原本私有的人类经验是翻译成行为数据的免费原材料。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%E8%B0%B7%E6%AD%8C%E6%8E%A7%E5%88%B6%E7%9A%84%E4%B8%96%E7%95%8C%E5%92%8C%E7%9B%91%E8%A7%86%E8%B5%84%E6%9C%AC%E4%B8%BB%E4%B9%89-2/" href="https://www.iyouport.org/%E8%B0%B7%E6%AD%8C%E6%8E%A7%E5%88%B6%E7%9A%84%E4%B8%96%E7%95%8C%E5%92%8C%E7%9B%91%E8%A7%86%E8%B5%84%E6%9C%AC%E4%B8%BB%E4%B9%89-2/" rel="noopener noreferrer" target="_blank">
     谷歌控制的世界和监视资本主义
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这不仅是监视，还意味着诱导和操纵，不仅可以塑造商业行为，更会塑造政治行为 —— 依旧一人一票，但是，操纵者在控制人们应该如何投票
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" rel="noopener noreferrer" target="_blank">
     心理操纵的秘密：”行为经济学”+PR=？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   当监视资本主义蓬勃发展时，民主就会沉睡。
  </p>
  <p class="graf graf--p">
   因此，监视资本家现在拥有21世纪独特的无上权力，这与近一个世纪前的极权主义一样前所未有。
  </p>
  <p class="graf graf--p">
   老大哥并不是一个人，而是一系列监视资本主义寡头，Zuboff 称其为“BigOther”。它是一种更为危险的东西。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Big Other 了解所有事，而其自身的行动仍然隐藏，消除了人们抵抗的权利；它破坏了人类的基本自治和自决的能力，没有自治和自决民主就无法生存。
   </strong>
  </p>
  <p class="graf graf--p">
   Big Other 了解我们的一切，而我们几乎对它一无所知。
  </p>
  <p class="graf graf--p">
   甚至这种权力的不平衡并非违法，因为至今仍没有法律来控制它，但它从根本上说绝对是反民主的。
  </p>
  <p class="graf graf--p">
   监视资本家声称他们的方法是数字技术“不可避免的后果”。
   <strong class="markup--strong markup--p-strong">
    这是完全的B
   </strong>
   S。很容易想象没有监视资本主义的数字化未来，但是，没有数字技术就无法想象监视资本主义的存在。
  </p>
  <p class="graf graf--p">
   奥威尔鄙视“在当下的征服者之前鞠躬的本能”。他坚持认为，勇气要求我们坚持自己的道德观，甚至反对那些看似无敌的势力。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5742 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2.png?resize=1100%2C622&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2.png?w=1248&amp;ssl=1 1248w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2.png?resize=300%2C170&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2.png?resize=768%2C434&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2.png?resize=1024%2C579&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2.png?resize=1100%2C622&amp;ssl=1 1100w" data-recalc-dims="1" height="622" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2.png?resize=1100%2C622&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    语言的伟大和陷阱同在
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “1984”强调的主要教训并不完全是“持续监视是噩梦”、或者“威权政府是危险的”。这些都只是对真实的陈述，但不是最重要的信息。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “1984”的核心是关于语言的；
   <strong class="markup--strong markup--p-strong">
    政府如何使用语言来驯服人心和混淆概念，以及公民如何抵制压迫。
   </strong>
  </p>
  <p class="graf graf--p">
   奥威尔是英语语言的大师，他的遗产继续存在于他创造的一些词汇中。
  </p>
  <p class="graf graf--p">
   即使那些没有读过“1984”的人也能知道并理解这些词汇。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “1984”使用了特有的词汇讨论监视、警察状态和威权主义，其中包括诸如“老大哥”、“思想警察”等术语，也包括 “Unperson”和“Doublethink”等等，仅举几例。
  </p>
  <p class="graf graf--p">
   奥威尔描述的威权政府不仅仅是严厉惩罚异议 ——
   <strong class="markup--strong markup--p-strong">
    它甚至试图不让任何异议存在。
   </strong>
  </p>
  <p class="graf graf--p">
   当内党成员奥布莱恩折磨“1984”主角 Winston Smith 时，他举起手伸出四根手指，问 Smith 他看到了多少根手指。
  </p>
  <p class="graf graf--p">
   当 Smith 回答 “四个！四！我还能说什么？四！” 他遭受了难以忍受的痛苦。
  </p>
  <p class="graf graf--p">
   在 Smith 最终声称看到五根手指后，奥布莱恩强调说“五”是不够的； “‘不，Winston，这没用。你在说谎。你仍然认为是四。”
  </p>
  <p class="graf graf--p">
   这正是威权主义的特点。
  </p>
  <p class="graf graf--p">
   就连奥威尔自己的名字也变成了一个形容词“奥威尔式”，它在现代政治修辞中被广泛使用。
  </p>
  <p class="graf graf--p">
   通常人们把自己的敌人都称为奥威尔式的行为，这也是奥威尔的才能的证明，每个人似乎都认为“1984”是关于他们的政治对手的，而不是自己。
  </p>
  <p class="graf graf--p">
   政治左派在白宫和刑事司法系统中看到了许多奥威尔式的倾向；政治右派为大学校园和社交媒体公司的“思想警察”感到悲伤，将用户变成了“Unpersons”。
  </p>
  <p class="graf graf--p">
   这些都是事实。
  </p>
  <p class="graf graf--p">
   但是，政客们在没有奥威尔式的情况下依旧会撒谎，而封锁社交媒体账户的私营寡头公司就像是一个谋杀某人并从历史中彻底抹除他们的威权国家。
  </p>
  <p class="graf graf--p">
   同样，被广泛感知到的学术一致性（政治正确）可能会令人窒息，但它很难与警察国家强制执行的一致性相提并论。
  </p>
  <p class="graf graf--p">
   然而，
   <strong class="markup--strong markup--p-strong">
    当政府官员使用诸如“强化审讯”、“替代事实”、“附带损害”、或“极端分子”等词语时，他们自己明白，他们所描述的实际上是“折磨”、“谎言”、“无辜的平民死亡” “、和”持不同政见者
   </strong>
   。（《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ad%a6%e5%af%9f%e5%b0%86%e6%b4%bb%e5%8a%a8%e5%ae%b6%e7%a7%b0%e4%b8%ba%e5%9b%bd%e5%86%85%e6%9e%81%e7%ab%af%e5%88%86%e5%ad%90%e7%9a%84%e4%ba%94%e4%b8%aa%e8%8d%92%e8%b0%ac%e7%90%86/" href="https://www.iyouport.org/%e8%ad%a6%e5%af%9f%e5%b0%86%e6%b4%bb%e5%8a%a8%e5%ae%b6%e7%a7%b0%e4%b8%ba%e5%9b%bd%e5%86%85%e6%9e%81%e7%ab%af%e5%88%86%e5%ad%90%e7%9a%84%e4%ba%94%e4%b8%aa%e8%8d%92%e8%b0%ac%e7%90%86/" rel="noopener noreferrer" target="_blank">
    警察将活动家称为“国内极端分子”的五个荒谬理由
   </a>
   》）
  </p>
  <p class="graf graf--p">
   更加邪恶和野蛮的语言歪曲在全球泛滥。
  </p>
  <p class="graf graf--p">
   根据活动人士和研究人员的报道，中国政府已经在“再教育”营中关押了约100万人；报告显示，该营并不是关于教育的学校；而是残酷的政治灌输场所，囚犯们被迫背诵共产党的宣传并放弃伊斯兰教。
  </p>
  <p class="graf graf--p">
   朝鲜这个最接近体现“1984”的国家，以令人沮丧的成功程度阻止了公民为自己思考的能力。
  </p>
  <p class="graf graf--p">
   在回忆录中，脱北者 Yeonmi Park 描述了韩语词汇原本的丰富性，并指出：
   <strong class="markup--strong markup--p-strong">
    “当你有更多的词语来描述世界时，你会增加思考复杂思想的能力。”
   </strong>
  </p>
  <p class="graf graf--p">
   当 Park 阅读奥威尔的经典寓言小说“动物农场”时，她觉得好像奥威尔知道她来自哪里，这并不奇怪。
  </p>
  <p class="graf graf--p">
   奥威尔不是先知，但他确定了任何成功的威权政府的必要特征。
  </p>
  <p class="graf graf--p">
   奥威尔试图提醒人们，即使在纳粹失败后，仍然存在威胁民主的极权主义危险。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    奥威尔敦促“不断的批评”，他警告任何对极权主义“免疫”的想法都不应该被视为理所当然：“如果不能不间断地抵制，极权主义可以在任何地方取得胜利。”
   </strong>
  </p>
  <p class="graf graf--p">
   为了有效地控制你，它不仅可以威胁死亡、监禁或酷刑；禁止书籍和宗教是不够的。
  </p>
  <p class="graf graf--p">
   只要国家政权不能主宰你的意识，它就会不断被推翻。
  </p>
  <p class="graf graf--p">
   在70年后这个政治不诚实猖獗的时代里，应该记住1984年最重要的一课：国家真的可以占据你的思想。⚪️
  </p>
