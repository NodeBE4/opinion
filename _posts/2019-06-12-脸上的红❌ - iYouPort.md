     <em>
      新披露的文件详细介绍了政府如何建立一个活动家、记者和社交媒体名人的秘密监视数据库。这种违宪行为是怎么来的？
     </em>

   </li>

    <img alt="" class="aligncenter size-full wp-image-2356" data-recalc-dims="1" height="617" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-12.png?resize=1100%2C617&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-12.png?w=1386&amp;ssl=1 1386w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-12.png?resize=300%2C168&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-12.png?resize=768%2C431&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-12.png?resize=1024%2C575&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-12.png?resize=1100%2C617&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在 NBC7 公布的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nbcsandiego.com/investigations/Source-Leaked-Documents-Show-the-US-Government-Tracking-Journalists-and-Advocates-Through-a-Secret-Database-506783231.html" href="https://www.nbcsandiego.com/investigations/Source-Leaked-Documents-Show-the-US-Government-Tracking-Journalists-and-Advocates-Through-a-Secret-Database-506783231.html" rel="noopener noreferrer" target="_blank">
    泄露文件
   </a>
   后不到24小时，国会正在做出反应。这些文件详细介绍了美国政府如何建立一个活动家、记者和社交媒体名人的秘密监视数据库。
  </p>
  <p class="graf graf--p">
   美国国土安全委员会的一封措辞强硬的信件称这种做法“令人不安”，有关此类计划的合法性的争论已经开始。舆论要求对这一秘密监视作出解释。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “记者是受第一修正案保护的，通常要对记者进行监视，你必须有一些非常具体的理由认为记者行为危害了国家安全”，加州大学圣地亚哥分校的 Dan Hallin 教授说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “现在还不完全清楚监视的动机是什么，但作为一名记者，并没有什么能引起你从事犯罪活动的合理怀疑“，法律分析师 Dan Eaton 说。
  </p>
  <p class="graf graf--p">
   NBC7与前美国检察官进行了交谈，他表示，从律师或记者那里获取信息是一条“明确的路线”。他也非常坚定地认为，批准必须来自高层 — 可能是副检察长或司法部长本人。
  </p>
  <p class="graf graf--p">
   根据美国海关和边境保护局的统计，秘密数据库的核心是2018年底圣伊西德罗入境口岸的冲突。来自中美洲的大约5000名移民通过墨西哥向北进入美国南部边境。这个故事成为国际头条新闻。
  </p>
  <p class="graf graf--p">
   在随后的几个月中，报道大篷车的记者以及向大篷车成员提供援助的记者们表示，他们确信自己已经成为边境官员进行秘密监视和审查的重点的目标。
  </p>
  <p class="graf graf--p">
   国家安全局内部官员在不愿透露姓名的情况下向 NBC 7 提供了泄露文件，并证实了记者们的怀疑。
  </p>
  <p class="graf graf--p">
   在监视名单上被列出的人员包括10名记者，其中7名是美国公民，还有美国律师，和48名来自美国和其他国家的人，被标记为组织者、煽动者或角色“未知”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “海关和边境保护局对集会的管控方面有很大的自由裁量权，”Eaton 说。“这里的问题在于这种秘密监视是否会遏制记者的工作能力。”
  </p>
  <p class="graf graf--p">
   一名摄影记者说，当她试图越过边境时，被墨西哥当局拘留了13个小时。最终，她被拒绝入境并被遣返回美国。因为她在黑名单上。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这让我想起了20世纪60年代发生的各种事，”Hallin 说。
   <strong class="markup--strong markup--p-strong">
    “这似乎是早期时代噩梦的回归，当时最著名的间谍行为是联邦调查局对各种活动人士以及记者进行的监视
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   Hallin 将这一监视行为比作名为 COINTELPRO 的 J. Edgar Hoover 的间谍计划。该计划被用来收集有关民权活动家的所有信息，其中包括 Martin Luther King Jr. 博士。
  </p>
  <p class="graf graf--p">
   Hallin 还说，这让人想起1971年理查德尼克松总统的“敌人名单”。该名单包括一些被认为与政府对抗的记者的私密信息。该名单在参议院听证会期间公开，国会禁止这种审查 — 这一点肯定会出现在反对当前计划的争论中。
  </p>
  <p class="graf graf--p">
   消息人士称，这些文件或屏幕截图显示了海关和边境保护局（CBP）移民与海关执法局（ICE），美国边境巡逻队，国土安全调查局，以及联邦调查局（FBI）圣地亚哥圣地亚哥部门代理商一起分享这些监视资源。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    被监视目标不仅有大批记者和律师，还包括来自 Border Angels 和 Pueblo Sin Fronteras 等权利组织的倡导者
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nbcsandiego.com/multimedia/PHOTOS-Leaked-Documents-to-NBC-7-Investigates-506782041.html" href="https://www.nbcsandiego.com/multimedia/PHOTOS-Leaked-Documents-to-NBC-7-Investigates-506782041.html" rel="noopener noreferrer" target="_blank">
    在此处
   </a>
   查看这一文档。这些文件的标题是“移民大篷车 FY-2019，疑似组织者，协调员，煽动者和媒体”，日期为2019年1月9日。
  </p>
  <p class="graf graf--p">
   对于每个被监视的人来说，
   <strong class="markup--strong markup--p-strong">
    这些文件通常显示的是他们的护照照片，但在某些情况下，相关信息来自被监视者的社交媒体帐户，以及他们的个人信息
   </strong>
   。这些信息包括该人的出生日期、他们的“出生国”、以及他们所谓的与移民大篷车相关的角色。该信息还包括官方是否对该人的护照发出警报。
  </p>
  <p class="graf graf--p">
   有些人的照片上带有一个彩色的“X”，表明他们是被逮捕的、被面谈的、或者被官员撤销了签证或 SENTRI 通行证的。国土安全部的消息来源告诉NBC 7，监视者还为每个被列出的人创建了档案。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们是一个刑事调查机构，不是情报机构，我们无法为被监视者制作档案，这是边境搜查部门制作的档案。他们在滥用权力“，国土安全部消息人士告诉NBC 7。
  </p>
  <p class="graf graf--p">
   与 NBC 7共享的一份档案是关于墨西哥蒂华纳移民和难民法律中心 Al Otro Lado 的律师 Nicole Ramos 的。该档案包括 Ramos 的详细个人详细信息，包括她驾驶的汽车的具体细节，她母亲的名字，以及她的工作和旅行历史。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    G
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nbcsandiego.com/news/local/Governments-Surveillance-of-Journalists-Reminds-Media-Expert-of-60s-506860451.html" href="https://www.nbcsandiego.com/news/local/Governments-Surveillance-of-Journalists-Reminds-Media-Expert-of-60s-506860451.html" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     overnment’s Surveillance of Journalists Reminds Media Expert of
    </strong>
   </a>
   ’60s. “It reminded me of the kinds of things that happened in the 1960s,” said Hallin. “It seemed like a throwback to an earlier era when the FBI would conduct surveillance on activists of various kinds as well as journalists.”
  </p>
