  <p class="graf graf--p">
   正如你所知道的，动机这个东西在几乎任何情况下都是一种促进力，关键在于，你选对了动机。
  </p>
  <p class="graf graf--p">
   一个人的动机和需求对记忆力来说也有很大的影响。
  </p>
  <p class="graf graf--p">
   比如，您可能会发现，试图记住铁路线上所有站点是件 “困难的事”；但是对于售票员来说，他们会记忆得非常深刻，因为他们需要这些记忆来进行工作。
  </p>
  <p class="graf graf--p">
   换句话说，在某一种情况下，你会不自觉地记住东西。
  </p>
  <p class="graf graf--p">
   记住日常需要的信息会更容易。大脑会将它所收到的信息分为&lt;重要&gt;和&lt;其他&gt;两大类，以此来节省自身能量。由此一来，就有了
   <strong class="markup--strong markup--p-strong">
    以下建议
   </strong>
   ：
  </p>
  <p class="graf graf--p">
   👉在需要记住任何事情之前先在心里明确一下关于为什么要记住它，以及记住它之后会得到什么样的回报。同样的，如果你希望别人记住你问他们的事，那就要给对方创建一个
   <strong class="markup--strong markup--p-strong">
    记忆触发器
   </strong>
   —— 吸引对方的兴趣。
  </p>
  <p class="graf graf--p">
   这需要一点
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
    社交工程
   </a>
   ，也就是你对对方的了解：关于他们最感兴趣什么、什么东西最容易吸引他们的注意。正如中文词汇所描述的 “说到心坎里”，我觉得这个短语特别形象。
  </p>
  <p class="graf graf--p">
   需求和动机经常会对记忆开某种残酷的玩笑，就如弗洛伊德所描述的那样：人类对于情感慰籍的渴望往往会导致忘记可能令人不快的信息，这种现象被称为“压抑”。
  </p>
  <p class="graf graf--p">
   如果你总是忘记与某个特定的人会面，而在与其他人会面时这种情况很少出现，于是你需要琢磨一下，也许与那个人交流或与此人相关的联想会引发你情绪上的某种不适感。
  </p>
  <p class="graf graf--p">
   很难想象有人会忘记向一个非常慷慨的朋友打电话借钱，但债务人经常会忘记打电话给债权人商谈还钱的日期。他们可能是真的忘了，他们的大脑强制他们回避 “不愉快” 的东西。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #ff00ff;">
    <strong class="markup--strong markup--p-strong">
     遗忘曲线
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   19世纪末，德国心理学家 Hermann Ebbinghaus 通过研究总结出了记忆的遗忘曲线，解释了人们忘记所学信息的过程。
  </p>
  <p class="graf graf--p">
   Ebbinghaus 要求受试者记住三个字母组成的音节，这些音节并没有实际意义。如果不动用想象力，只靠死记硬背，一个小时后只能记得44%的信息；而一周后只能记得不到25%。
  </p>
  <p class="graf graf--p">
   幸运的是，通过有意识地记忆，信息被遗忘的速度会变慢。
  </p>
  <p class="graf graf--p">
   大部分信息在学习后的几个小时内就被忘光了，这种情况下怎么办呢？进一步的实验表明，复习可以降低遗忘率；信息被重复的次数越多，记忆效果就会越好。
  </p>
  <p class="graf graf--p">
   确定遗忘曲线的实验得出了
   <strong class="markup--strong markup--p-strong">
    具有应用价值的结论
   </strong>
   。首先，试图一次性记住所有信息是不现实的，最好的方法是分阶段记忆信息，并留出时间进行重复。
  </p>
  <p class="graf graf--p">
   如果你有一整天的时间用来记忆，最佳重复率是这样的：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    第一次：学习后的15～20分钟；
   </li>
   <li class="graf graf--li">
    第二次：6～8小时后；
   </li>
   <li class="graf graf--li">
    第三次：24小时后。
   </li>
  </ul>
  <p class="graf graf--p">
   最好主动地重复这些信息，与其重读和重听，不如尝试从记忆中把信息绘制或写下来，并在必要时与源文件对照检查。
  </p>
  <p class="graf graf--p">
   如果您有更多的时间用于记忆，请重复信息如下：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    第一次：初学的当天；
   </li>
   <li class="graf graf--li">
    第二次：初学后的第四天；
   </li>
   <li class="graf graf--li">
    第三次：初学后的第七天。
   </li>
  </ul>
  <p class="graf graf--p">
   如果信息很多，
   <strong class="markup--strong markup--p-strong">
    最好以不同的详细程度进行重复
   </strong>
   ：第一次重复全部，第二次只重复关键点，第三次再重复全部，但以
   <strong class="markup--strong markup--p-strong">
    不同的分组和不同的顺序
   </strong>
   进行重复。
  </p>
  <p class="graf graf--p">
   👉处理的程度越深，信息就越容易被回想起来。
  </p>

   <span style="color: #ff00ff;">
    <strong class="markup--strong markup--p-strong">
     辨别干扰信息
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   干扰信息的本质是相似的记忆被混杂在一起了。
  </p>
  <p class="graf graf--p">
   两段相似的记忆会互相影响，它们越是相似，准确记忆它们就越难。这就是为什么要强调：
   <strong class="markup--strong markup--p-strong">
    您的多个分身之间尽可能不要有任何相似性，不仅是为避免被追踪者关联起来，而且还有，你不能把自己绕进去
   </strong>
   。
  </p>
  <p class="graf graf--p">
   举个最简单的例子，你使用某张银行卡已经多年，已经完美地记住了它的密码；但是这张卡到期了，银行发给你一张新卡。最初，每次你操作时都会自动想起旧密码，以至于你必须有意识地去回忆新密码。但是一段时间后这种习惯会改变：新密码被自动唤起，而旧密码需要努力去想。
  </p>
  <p class="graf graf--p">
   与这一情况相关的类似记忆都会互相干扰。
  </p>
  <p class="graf graf--p">
   👉为了减少干扰的影响，你可以有意识地将类似信息的记忆任务分隔开来。
  </p>
  <p class="graf graf--p">
   比如在准备考试时，尽可能把彼此截然不同的要点放在一起进行记忆。这一规则在很多情况下都适用。
  </p>
  <p class="graf graf--p">
   如果你需要编辑一个文档、撰写一篇评论并绘制一张图表，那么最好用绘制图表这个任务将编辑文本和撰写评论分隔开。
  </p>
  <p class="graf graf--p">
   👉相反，
   <strong class="markup--strong markup--p-strong">
    如果你想要某人忘记某件事，你就可以对他灌输大量相关主题的信息
   </strong>
   。你可以假装征求他的意见，以详细讨论这个话题，并且让他密切参与。信息越多越好，他越容易迷糊在里面。
  </p>
  <p class="graf graf--p">
   这就是有意地制造干扰，最终他将无法记清楚你试图让他忘记的信息；反之，如果没有混淆，他会记得更清楚。至少，
   <strong class="markup--strong markup--p-strong">
    混淆能让他感到困惑，并失去信心。
   </strong>
  </p>
  <p class="graf graf--p">
   👉如果有人试图在对话中记住某些事，而你想要阻止他记住，可以尝试提示他 —— 错误的但相似的建议会妨碍记忆的进程，从而形成干扰。
  </p>
  <p class="graf graf--p">
   🌟这是律师经常使用的社交工程技巧，在法庭审理中有意识地使证人混淆自己的记忆，然后律师趁机抓住把柄推翻证词。
  </p>
  <p class="graf graf--p">
   在日常生活中，当你的朋友或家人试图帮助你记住某些事的时候，你也会经历上面那种情况，亲友的帮忙通常只会让事情变得更加困难。
  </p>

  </p>
  <p class="graf graf--p">
   比如，一名中国人来到英国，他就会犯迷糊，因为英国的交通规则是靠左行驶，而中国是靠右。“右手”技能不仅没有任何作用，而且还是干扰因素，你必须克服它。
  </p>
  <p class="graf graf--p">
   ⚠️在紧张的情况下，当有意识的控制被削弱时，旧的习惯会再次出现。这就很容易让角色扮演和伪装露出马脚。
  </p>
  <p class="graf graf--p">
   还记得我们
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E8%AE%A9%E4%BD%A0%E7%9C%8B%E8%A7%81-%E6%AC%BA%E9%AA%97%E7%9A%84%E7%A7%AF%E6%9E%81%E6%96%B9%E6%B3%95%E4%B9%8B%E4%BA%8C%EF%BC%885%EF%BC%89/" href="https://www.iyouport.org/%E8%AE%A9%E4%BD%A0%E7%9C%8B%E8%A7%81-%E6%AC%BA%E9%AA%97%E7%9A%84%E7%A7%AF%E6%9E%81%E6%96%B9%E6%B3%95%E4%B9%8B%E4%BA%8C%EF%BC%885%EF%BC%89/" rel="noopener noreferrer" target="_blank">
    讲过的故事
   </a>
   吗？那是美国战略情报局的外勤特工讲述的，当时被派往法国的间谍要学会将香烟一直吸到烟蒂处。有一次，一名特工扔掉了吸了一半的香烟，结果正因此被盖世太保识破身份。因为在香烟紧缺的法国，节俭的普通人是绝对不会这样做的。
  </p>
  <p class="graf graf--p">
   情报史上因这种无意识的行为暴露身份的案例比比皆是。
  </p>
  <p class="graf graf--p">
   👉克格勃曾经发布了一份备忘录，其中列出了可用于
   <strong class="markup--strong markup--p-strong">
    识别外国间谍的特征清单
   </strong>
   。其中包括，比如：把冰块放在酒精饮料里、小恩小惠也要酬谢、晚餐不吃面包，等等。这些行为对于西欧和美国人来说完全正常，但是苏联人可不会这么做。
  </p>
  <p class="graf graf--p">
   🌟想要识别某人的身份吗？让他暴露旧习惯就行了……
  </p>
  <p class="graf graf--p">
   如何做到让他暴露？你可以提一个特别敏感的话题，或者让他对言语和行为 **迅速** 作出反应，以此来迷惑他。
   <strong class="markup--strong markup--p-strong">
    速度越快越好，此人如果没有经过专业级的训练，就很容易在这种情况下被分心，从而暴露自己
   </strong>
   。
  </p>
  <p class="graf graf--p">
   此外，疲劳和酒精中毒，都会削弱有意识的控制能力，从而容易暴露一个人真实的面目。👉有意识地制造这类环境就变成了识破身份的手段。
  </p>
