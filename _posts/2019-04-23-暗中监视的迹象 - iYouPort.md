 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <strong>
      <em>
       你知道政府正在如何监视你吗？你不需要是什么特殊的人物，就如我一样普通，也同样”享受”着这一系列监视。您应该知道怎样识别它们的存在。
      </em>
     </strong>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   技术正在飞速发展，政府的权力也是一样。如今几乎每个人都知道政府正在监视自己，但其中仍有很多人把这想得太简单了，就如公民社会组织 ACLU 所说：
   <strong class="markup--strong markup--p-strong">
    政府比你想象的要狡猾得多
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如何判断自己正在被监视？本文列举一些值得注意的奇怪迹象、以及危险度比较高的情况，一旦您发现所处的环境相符，请尽快撤离，或想办法缓解。
  </p>
  <p class="graf graf--p">
   1. 如果你有一部“智能”电视
  </p>
  <p class="graf graf--p">
   几年前我不知道为什么会有人想要买这种东西回家，它是一个活间谍。可惜如今你已经几乎买不到普通电视了，因为市场生产非“智能”产品不赚钱。而且不止电视机，有可能很快，你将会买不到普通的门锁、买不到普通的吸尘器、甚至普通的灯泡……
  </p>
  <p class="graf graf--p">
   你所有的所谓“智能”技术都很酷 — 但它们不会给你的隐私带来任何好处。Wikileaks 文件中曾经最著名的之一就是揭示了中央情报局可以闯入三星智能电视，在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://iyouport.wixsite.com/iyouport/blog/%E9%9B%B6%E6%97%A5%E6%BC%8F%E6%B4%9E%E5%92%8C-cia-%E9%BB%91%E5%AE%A2%E5%B7%A5%E5%85%B7" href="https://iyouport.wixsite.com/iyouport/blog/%E9%9B%B6%E6%97%A5%E6%BC%8F%E6%B4%9E%E5%92%8C-cia-%E9%BB%91%E5%AE%A2%E5%B7%A5%E5%85%B7" rel="noopener noreferrer" target="_blank">
    零日漏洞和 CIA 黑客工具
   </a>
   》
  </p>
  <p class="graf graf--p">
   受到这种工具感染的电视机被隐藏录制音频，一旦电视机完全切换回来，将通过互联网传送数据到 CIA 的计算机服务器。
  </p>
  <p class="graf graf--p">
   截至2016年，中情局已经建立了 24 个 Android 的“零日”武库。其中一些据说是由中央情报局自己开发的，其他一些据称是从英国的 GCHQ 机构以及 NSA 和未命名的第三方获得的。
  </p>
  <p class="graf graf--p">
   中央情报局中还有一个专门单位其工作就是瞄准 iPhone 和 iPad，允许该机构看到目标对象的地理位置、激活他们的设备的摄像头和麦克风，并阅读文字通信。该部门还利用 GCHQ，NSA 和 FBI 获得的进一步 iOS “零日漏洞”。
  </p>
  <p class="graf graf--p">
   所以，如果你家有这类“智能设备”，请在家里时小心说话。
  </p>
  <p class="graf graf--p">
   2. 您尚未更新设备
  </p>
  <p class="graf graf--p">
   不论如何，你必须依照最基本的网络卫生。防止安全漏洞的最佳方法之一就是：保持设备更新。过时的软件更容易受到损害，监视者很乐意首先盯着那些懒人（可能因为0-day太贵了）
  </p>
  <p class="graf graf--p">
   一个例子
   <a class="markup--anchor markup--p-anchor" data-href="http://money.cnn.com/2015/05/27/technology/iphone-text-message-hack/?iid=EL" href="https://money.cnn.com/2015/05/27/technology/iphone-text-message-hack/?iid=EL" rel="noopener noreferrer" target="_blank">
    发生在2015年
   </a>
   ，当时发现95％的安卓机可以通过一条短信被黑入。 Apple 遇到了类似的问题，他们迅速更新了。更新可能看起来很麻烦，但是当您收到更新警报时请尽快完成。
  </p>
  <p class="graf graf--p">
   3. 你在机场被标记了
  </p>
  <p class="graf graf--p">
   您可能会在机场被标记一次或两次，但请注意这是否经常发生。 HowStuffWorks 解释说，如果你在机场多次被拦下，这就是一个标志，
   <strong class="markup--strong markup--p-strong">
    你的名字无论如何都在政府观察名单上
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://mentalfloss.com/article/68073/8-ways-you-can-end-no-fly-list" href="https://mentalfloss.com/article/68073/8-ways-you-can-end-no-fly-list" rel="noopener noreferrer" target="_blank">
    Mental Floss 指出
   </a>
   ，现在
   <strong class="markup--strong markup--p-strong">
    已经有一些人由于有争议的社交媒体帖子而无法登机
   </strong>
   。虽然人们不能因“不可靠或不可信”的信息被列入黑名单，但是，社交媒体帖子肯定不会被监视者忽视。
  </p>
  <p class="graf graf--p">
   4. 你正在分享你的cookies
  </p>
  <p class="graf graf--p">
   存储在浏览器中的微小数据都可跟踪您一切的在线活动。这些 cookie 可以保留从您的密码到您想在亚马逊上购买的所有内容。Cookies 被放入一个巨大的数据库，公司可以在这里找出你的离线身份、并出售这些信息。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.huffingtonpost.com/2014/04/03/government-surveillance_n_5084623.html" href="https://www.huffingtonpost.com/2014/04/03/government-surveillance_n_5084623.html" rel="noopener noreferrer" target="_blank">
    赫芬顿邮报解释
   </a>
   说：“有两个主要群体可以访问这些数据：政府（包括国家安全局和当地警察局）、以及私营部门（即：广告业）。”
  </p>
  <p class="graf graf--p">
   5. 你打开了可疑的电子邮件
  </p>
  <p class="graf graf--p">
   我们都收到了可疑的电子邮件。问题是你打开了吗？只需单击它，即可将发件人引入到您的设备上。我们在社交工程攻击演示中介绍过这种跟踪。
  </p>
  <p class="graf graf--p">
   无论是政府老大哥还是垃圾邮件发送者，都会威胁您的个人隐私，您首先要做的是删除那种电子邮件而不要打开它。无论如何都不要回复那种电子邮件。如果攻击者确认您的帐户处于有效状态，就会引发进一步的间谍活动。
  </p>
  <p class="graf graf--p">
   6. 你的名字和政府监视黑名单上的某个人姓名相同
  </p>
  <p class="graf graf--p">
   即使你没有做任何让自己进入政府监视黑名单的事，也可能那个名单上有与你同名的人。如果这样的话你可能就麻烦了。不用想着改名字，因为你根本不知道那个名单上都有什么名字。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://people.howstuffworks.com/government-watch-list2.htm" href="https://people.howstuffworks.com/government-watch-list2.htm" rel="noopener noreferrer" target="_blank">
    HowStuffWorks
   </a>
   在2004年的报告中写道，一位名叫 John Lewis 的人在政府监视名单上。这是个特别普通的名字。在接下来的几年里，由于名字相同，每一个叫 John Lewis 的人登机都会遇阻。
  </p>
  <p class="graf graf--p">
   检查您的姓名是否在财政部公布的特别指定国民和被封锁人员名单上，以确保没有不受欢迎的未来意外（包括查水表）。不过如果你住在中国的话，很可能查不到类似的名单。
  </p>
  <p class="graf graf--p">
   7. 你在自己的车子上找到了一个跟踪器
  </p>
  <p class="graf graf--p">
   政府可以密切关注的不仅仅是您的技术工具。很多情况下跟踪器通常隐藏在汽车的底盘中。它很小，以至于很多时候难以发觉，它们也可以安装在轮舱或排气系统中，因此有人可以随时跟踪您的汽车位置。
  </p>
  <p class="graf graf--p">
   如果您觉得自己被跟踪了，或者行程比较敏感，
   <strong class="markup--strong markup--p-strong">
    在上路之前请仔细检查您的车子
   </strong>
   ，特别是在较小的缝隙中。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    你要提防的不只有警察和间谍，因为如今任何人都可以在线购买到这种微型跟踪器
   </strong>
   。
  </p>
  <p class="graf graf--p">
   8. 您的数据使用率高于您实际使用的数据
  </p>
  <p class="graf graf--p">
   您是专用Wi-Fi 用户，但您的数据正在逐渐增加。这是为什么？
   <a class="markup--anchor markup--p-anchor" data-href="https://www.usatoday.com/story/news/nation/2013/12/08/cellphone-data-spying-nsa-police/3902809/" href="https://www.usatoday.com/story/news/nation/2013/12/08/cellphone-data-spying-nsa-police/3902809/" rel="noopener noreferrer" target="_blank">
    “今日美国”报道
   </a>
   ，这是因为国家安全局可能正在侦察您的行踪。而且当地警方也开始使用相同的做法。
  </p>
  <p class="graf graf--p">
   因为数据量上升可能来自间谍软件，它可以追踪您的日常生活的一切。攻击者也可以使用此技术窃取您的个人信息。
  </p>
  <p class="graf graf--p">
   9. 一辆没有任何标志的面包车停在你家门外
  </p>
  <p class="graf graf--p">
   不，这可不是间谍电影里的俗气道具。政府确实在使用无标志的车辆进行监视。
  </p>
  <p class="graf graf--p">
   不是每辆停在你家外面的面包车都值得关注，但一些关键指标可能意味着它是一辆监视车。
  </p>
  <p class="graf graf--p">
   寻找那些带有色窗户的无标志货车，这些车辆停放数天，没有任何人进入或离开车子的迹象。
  </p>
  <p class="graf graf--p">
   10. 摄像头上的指示灯亮起 — 即使您没有使用它
  </p>
  <p class="graf graf--p">
   指示灯亮起时意味着您的网络摄像头已开启。如果您的网络摄像头指示灯即使在未使用时也会亮起，请不要认为它只是故障。可能是 FBI 正在通过你的相机看着你。
  </p>
  <p class="graf graf--p">
   这里有一些更可怕的东西：
   <a class="markup--anchor markup--p-anchor" data-href="http://www.dailymail.co.uk/news/article-2520707/FBI-spy-webcam-triggering-indicator-light.html" href="https://www.dailymail.co.uk/news/article-2520707/FBI-spy-webcam-triggering-indicator-light.html" rel="noopener noreferrer" target="_blank">
    每日邮报
   </a>
   在线报道，
   <strong class="markup--strong markup--p-strong">
    政府有技术可以触发您的网络摄像头，甚至不会让灯亮起来。也就是说，你可能会在没有任何意识的情况下被窥探
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果您担心这点，请不使用网络摄像头。或者用一块不透明胶带盖住它。覆盖镜头可以防止任何潜在的间谍看到任何东西，但请记住你的麦克风可能仍然会被打开！
  </p>
  <p class="graf graf--p">
   11. 在通话期间，您听到了奇怪的声音
  </p>
  <p class="graf graf--p">
   是的，这意味着
   <a class="markup--anchor markup--p-anchor" data-href="https://it.toolbox.com/blogs/voipdesk/listen-up-17-signs-that-you-are-being-wiretapped-072312" href="https://it.toolbox.com/blogs/voipdesk/listen-up-17-signs-that-you-are-being-wiretapped-072312" rel="noopener noreferrer" target="_blank">
    窃听发生了
   </a>
   。如果您在手机上听到噪音，例如点击、弹出、刮擦或其他奇怪的声响，那么有人可能正在监听您的对话。
  </p>
  <p class="graf graf--p">
   另外，请注意手机在不使用时产生的噪音，这是另一个标志，意味着有人正在搞乱你的线路。
  </p>
  <p class="graf graf--p">
   12. 你家里似乎有些不对劲
  </p>
  <p class="graf graf--p">
   你应该最熟悉自己家里的布局应该是什么样的了，尤其是一些小的细节。你真的应该牢记这些小细节。因为，
   <a class="markup--anchor markup--p-anchor" data-href="http://www.businessinsider.com/yes-im-getting-a-bit-afraid-of-my-government-2012-11" href="http://www.businessinsider.com/yes-im-getting-a-bit-afraid-of-my-government-2012-11" rel="noopener noreferrer" target="_blank">
    Business Insider 指出
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    警方可以在没有告诉您的情况下、在没有任何保证的情况下，在私人财产上安装隐藏的监控摄像头
   </strong>
   。这并不意味着政府会选择偷偷潜入任何人的家，但是，如果他们认为你对社会构成威胁，你可能会发现你的整个家都被窃听了。
  </p>
  <p class="graf graf--p">
   13. 周围的人
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.npr.org/templates/story/story.php?storyId=92207687" href="https://www.npr.org/templates/story/story.php?storyId=92207687" rel="noopener noreferrer" target="_blank">
    NPR 采访了
   </a>
   联邦调查局特工，发现他们是融入人群的专家。这些特工表示，他们已准备好了为所有类型的活动搭配的衣物，能够非常好地融入他们正在监视的人群中 — —
   <strong class="markup--strong markup--p-strong">
    参加大型群体事件时尤其需要注意这点。
   </strong>
  </p>
  <p class="graf graf--p">
   如果你是单一目标，当你被跟踪时，通常会有整个团队围绕着你，尽管你可能不会注意到。现在您应该知道：如果您认为自己被跟踪了，那么
   <strong class="markup--strong markup--p-strong">
    记住可疑的面孔而不是衣服是很重要的
   </strong>
   ，因为衣服和头发随时可以换。
  </p>
  <p class="graf graf--p">
   14. 他们如何跟踪你的电子邮件
  </p>
  <p class="graf graf--p">
   你是否曾经在电子邮件中讨论过某个产品或服务，然后在事后看到了一堆相关的广告？很简单，你的邮件被监视了。
  </p>
  <p class="graf graf--p">
   你不应该对此感到惊讶，因为
   <strong class="markup--strong markup--p-strong">
    雅虎和谷歌都有权扫描你的电子邮件
   </strong>
   ，为你提供更具针对性的广告。我们一直强调不要使用Gmail，这是最主要的原因之一。
  </p>
  <p class="graf graf--p">
   并且，您的电子邮件服务商可能
   <a class="markup--anchor markup--p-anchor" data-href="https://www.huffingtonpost.com/2014/04/03/government-surveillance_n_5084623.html" href="https://www.huffingtonpost.com/2014/04/03/government-surveillance_n_5084623.html" rel="noopener noreferrer" target="_blank">
    不是唯一一个
   </a>
   阅读这些内容的人。 FBI 还可以随时查看您保存或共享的任何电子邮件、图片、文件和社交媒体帖子。
  </p>
  <p class="graf graf--p">
   15. 他们如何跟踪你的浏览
  </p>
  <p class="graf graf--p">
   你可能已经允许网站在没有意识到的情况下使用 cookie 来跟踪你，它会记住您的密码和您查看的项目，所以在某种程度上，它们可以会让人感觉很方便。但是，你必须禁用它们。这些cookie 中的信息会出售给你都不知道是谁的人以了解关于您的更多信息。
  </p>
  <p class="graf graf--p">
   政府也可以访问这些信息。虽然广告业和私营部门可以比政府收集更多关于你的信息 — 而且他们也完全会从中获得巨额利润。
  </p>
  <p class="graf graf--p">
   16. 你在开车的时候他们如何监视你
  </p>
  <p class="graf graf--p">
   即使您在没有带手机的情况下离开家，他们也有办法密切监视您。
  </p>
  <p class="graf graf--p">
   ACLU 指出警车和道路标志都可以使用自动车牌阅读器作为监视。扫描您的车牌号码、日期、时间和地点。然后，这些信息会被汇集到当局的大型数据库中。
  </p>
  <p class="graf graf--p">
   有些信息会被保存多年 — 或永远保存。
   <strong class="markup--strong markup--p-strong">
    如果你出行的路线是一个比较敏感的地点…不要使用私人任何车辆
   </strong>
   。
  </p>
  <p class="graf graf--p">
   17. 你注册会员卡和信用卡时他们如何监视你
  </p>
  <p class="graf graf--p">
   每个人都喜欢优惠价，这就是会员卡如此受欢迎的原因。你可能已经注册了几个 — 而且它们可能为你省钱。但是，您可能没有意识到某些会员卡、信用卡和类似东西很容易监视你。
  </p>
  <p class="graf graf--p">
   使用这些信用卡和会员卡可以存储任何关于您购买行为的数据，并且会出售这些数据以及您在线活动的统计数据，以便向广告客户提供有关您的更多信息。当然，法律还可以要求访问您的信用卡历史记录。
  </p>
  <p class="graf graf--p">
   18. 当你走在大街上他们如何监视你
  </p>
  <p class="graf graf--p">
   国安局花了数百万美元购买高科技摄像机，为了在你走在街上时监控你。2012年，ACLU 报告仅在芝加哥就有1万部公共和私人拥有的摄像机。是的，在您当地的城市可能会有很多。
  </p>
  <p class="graf graf--p">
   并且请记住，
   <strong class="markup--strong markup--p-strong">
    如果你居住的楼层比较低的话，这些玩意可以通过窗口看到你家里。所以，请记得随时拉上窗帘
   </strong>
   。
  </p>
  <p class="graf graf--p">
   更多可见：《
