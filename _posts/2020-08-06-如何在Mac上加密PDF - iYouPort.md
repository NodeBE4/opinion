   <li class="graf graf--p">
    <span style="color: #00ccff;">
     <em>
      <strong>
       对于那些有可能被其他人查看设备的人来说，它可以是安全链上的一个小环节
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-13214 jetpack-lazy-image" data-lazy-sizes="(max-width: 750px) 100vw, 750px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/0-13.jpeg?resize=750%2C1000&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/0-13.jpeg?w=750&amp;ssl=1 750w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/0-13.jpeg?resize=225%2C300&amp;ssl=1 225w" data-recalc-dims="1" height="1000" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/0-13.jpeg?resize=750%2C1000&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="750"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13214" data-recalc-dims="1" height="1000" sizes="(max-width: 750px) 100vw, 750px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/0-13.jpeg?resize=750%2C1000&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/0-13.jpeg?w=750&amp;ssl=1 750w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/0-13.jpeg?resize=225%2C300&amp;ssl=1 225w" width="750"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   如果要在Mac上加密PDF，最简单的方法是使用
   <a class="markup--anchor markup--p-anchor" data-href="https://www.howtogeek.com/201638/use-your-macs-preview-app-to-crop-resize-rotate-and-edit-images/" href="https://www.howtogeek.com/201638/use-your-macs-preview-app-to-crop-resize-rotate-and-edit-images/" rel="noopener noreferrer" target="_blank">
    Preview app
   </a>
   。您可以将密码添加到最敏感的PDF文档中，以确保其安全。
  </p>
  <p class="graf graf--p">
   Preview 是 MacOS 的核心应用程序，因此您可以在任何 Mac 计算机上找到它。无需其他软件，您只需要一个PDF文档和一个Mac即可加密。
  </p>
  <p class="graf graf--p">
   如果您想使用其他软件，那么 Microsoft Office 也可以对PDF进行密码保护，但是 Preview 提供了使用Mac核心应用程序最快的方法。
  </p>
  <p class="graf graf--p">
   要将密码添加到 macOS 上的PDF文档中，请从启动台打开 “Preview” 应用，您可以从Mac显示屏底部的Dock中看到。
  </p>
  <p>
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-13218 jetpack-lazy-image" data-lazy-sizes="(max-width: 486px) 100vw, 486px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/11-1.png?resize=486%2C433&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/11-1.png?w=486&amp;ssl=1 486w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/11-1.png?resize=300%2C267&amp;ssl=1 300w" data-recalc-dims="1" height="433" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/11-1.png?resize=486%2C433&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="486"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13218" data-recalc-dims="1" height="433" sizes="(max-width: 486px) 100vw, 486px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/11-1.png?resize=486%2C433&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/11-1.png?w=486&amp;ssl=1 486w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/11-1.png?resize=300%2C267&amp;ssl=1 300w" width="486"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   或者，您也可以从 Finder 的 Applications 文件夹中启动 Preview —— 您还将在Dock上找到它的图标。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-13219 jetpack-lazy-image" data-lazy-sizes="(max-width: 349px) 100vw, 349px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/22.png?resize=349%2C138&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/22.png?w=349&amp;ssl=1 349w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/22.png?resize=300%2C119&amp;ssl=1 300w" data-recalc-dims="1" height="138" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/22.png?resize=349%2C138&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="349"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13219" data-recalc-dims="1" height="138" sizes="(max-width: 349px) 100vw, 349px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/22.png?resize=349%2C138&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/22.png?w=349&amp;ssl=1 349w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/22.png?resize=300%2C119&amp;ssl=1 300w" width="349"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <p class="graf graf--p">
    打开 “Preview” 后，从菜单栏中单击 “文件”&gt;“打开” 以打开要使用密码加密的PDF文档。
   </p>
   <p class="graf graf--p">
    <img alt="" class="aligncenter size-full wp-image-13220 jetpack-lazy-image" data-lazy-sizes="(max-width: 421px) 100vw, 421px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/33.png?resize=421%2C247&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/33.png?w=421&amp;ssl=1 421w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/33.png?resize=300%2C176&amp;ssl=1 300w" data-recalc-dims="1" height="247" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/33.png?resize=421%2C247&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="421"/>
    <noscript>
     <img alt="" class="aligncenter size-full wp-image-13220" data-recalc-dims="1" height="247" sizes="(max-width: 421px) 100vw, 421px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/33.png?resize=421%2C247&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/33.png?w=421&amp;ssl=1 421w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/33.png?resize=300%2C176&amp;ssl=1 300w" width="421"/>
    </noscript>
   </p>
   <p class="graf graf--p">
    要开始加密在预览中打开的PDF文档，请从菜单栏中单击文件&gt;导出。
   </p>
   <p class="graf graf--p">
    <img alt="" class="aligncenter size-full wp-image-13215 jetpack-lazy-image" data-lazy-sizes="(max-width: 473px) 100vw, 473px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-8.png?resize=473%2C462&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-8.png?w=473&amp;ssl=1 473w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-8.png?resize=300%2C293&amp;ssl=1 300w" data-recalc-dims="1" height="462" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-8.png?resize=473%2C462&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="473"/>
    <noscript>
     <img alt="" class="aligncenter size-full wp-image-13215" data-recalc-dims="1" height="462" sizes="(max-width: 473px) 100vw, 473px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-8.png?resize=473%2C462&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-8.png?w=473&amp;ssl=1 473w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-8.png?resize=300%2C293&amp;ssl=1 300w" width="473"/>
    </noscript>
   </p>
   <p class="graf graf--p">
    这将显示各种选项以不同的格式导出PDF，包括添加密码的选项。
   </p>
   <p class="graf graf--p">
    如果您不想覆盖现有文档，则需要在 “导出为” 框中为新文档命名。在 “位置” 下拉菜单中确认保存位置。
   </p>
   <p class="graf graf--p">
    要添加密码，请单击选项菜单中的“加密”复选框，然后在下面的“密码”和“验证”框中键入安全密码。
   </p>
   <p class="graf graf--p">
    确认设置后，按 “保存” 按钮保存加密的PDF文档。
   </p>
   <p class="graf graf--p">
    <img alt="" class="aligncenter size-full wp-image-13216 jetpack-lazy-image" data-lazy-sizes="(max-width: 451px) 100vw, 451px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-9.png?resize=451%2C362&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-9.png?w=451&amp;ssl=1 451w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-9.png?resize=300%2C241&amp;ssl=1 300w" data-recalc-dims="1" height="362" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-9.png?resize=451%2C362&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="451"/>
    <noscript>
     <img alt="" class="aligncenter size-full wp-image-13216" data-recalc-dims="1" height="362" sizes="(max-width: 451px) 100vw, 451px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-9.png?resize=451%2C362&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-9.png?w=451&amp;ssl=1 451w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/2-9.png?resize=300%2C241&amp;ssl=1 300w" width="451"/>
    </noscript>
   </p>
   <p class="graf graf--p">
    您的PDF文档将使用您选择的选项导出。加密文件后，在 Preview 中加载文档时将需要您输入密码。
   </p>
   <p class="graf graf--p">
    <img alt="" class="aligncenter size-full wp-image-13217 jetpack-lazy-image" data-lazy-sizes="(max-width: 311px) 100vw, 311px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-9.png?resize=311%2C219&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-9.png?w=311&amp;ssl=1 311w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-9.png?resize=300%2C211&amp;ssl=1 300w" data-recalc-dims="1" height="219" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-9.png?resize=311%2C219&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="311"/>
    <noscript>
     <img alt="" class="aligncenter size-full wp-image-13217" data-recalc-dims="1" height="219" sizes="(max-width: 311px) 100vw, 311px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-9.png?resize=311%2C219&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-9.png?w=311&amp;ssl=1 311w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-9.png?resize=300%2C211&amp;ssl=1 300w" width="311"/>
    </noscript>
   </p>
   <p class="graf graf--p">
    如果密码正确，这将解密文件并打开文档。您可以继续查看文件，直到将其关闭为止。关闭后，您需要重新输入密码才能再次查看文件。
   </p>
   <p class="graf graf--p">
    就是这么简单。对于那些有可能被他人查看设备的人来说，这也许可以作为一步缓解措施。绝不是全部措施！更多方法可以参见
