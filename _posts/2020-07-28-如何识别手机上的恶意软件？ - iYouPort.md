  <p class="graf graf--p">
   对于任何智能手机用户来说，了解如何检查自己的手机是否感染了病毒，是重要的一步。
  </p>
  <p class="graf graf--p">
   有几种迹象可能表明设备上存在病毒，如果您识别出了这些迹象，立刻采取适当的措施至关重要，例如 使用可靠的防病毒软件。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如何确定我的手机是否感染了病毒？
   </strong>
  </p>
  <p class="graf graf--p">
   智能手机用户必须意识到随时可能出现的诸多威胁，包括攻击者现在执行的最新DNS劫持。
  </p>
  <p class="graf graf--p">
   感染后的手机难以检查病毒会导致多种问题，包括 勒索软件、密码盗取、窃取机密数据等等。此外，还有一些恶意软件也可能导致智能手机的物理功能出现问题。
  </p>
  <p class="graf graf--p">
   以下7个迹象表明您的智能手机已被恶意软件感染：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、数据使用量突然增加
   </strong>
  </p>
  <p class="graf graf--p">
   在查看手机是否感染病毒时，最早出现的迹象之一就是数据使用量会突然增加。
  </p>
  <p class="graf graf--p">
   即使智能手机上的某些应用程序和游戏可能会占用大量数据，但在没有明显原因的情况下使用的数据量突然增加时，这可能表明手机已感染病毒；尤其是 ADB.miner 之类的恶意软件以在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hackread.com/monero-mining-malware-infect-android-smart-tv-phones/" href="https://www.hackread.com/monero-mining-malware-infect-android-smart-tv-phones/" rel="noopener noreferrer" target="_blank">
    Android 手机和智能电视
   </a>
   上挖掘 Monero 币而闻名。
  </p>
  <p class="graf graf--p">
   iOS 和 Android 都在 “设置” 部分提供了一个功能，使智能手机用户可以大致了解自己的设备上正在使用多少数据。
  </p>
  <p class="graf graf--p">
   Android 设备也将能够提供一个图表，显示指定时间段内的数据使用情况 —— 这样就很容易检测到突然增加的情况。
  </p>
  <p class="graf graf--p">
   对于 Android：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    打开手机的“设置”应用。
   </li>
   <li class="graf graf--li">
    点击 Network &amp; internet &gt; Data usage.
   </li>
   <li class="graf graf--li">
    在 “Mobile” 下，您会看到您使用了多少数据。
   </li>
   <li class="graf graf--li">
    要查看图表和详细信息，请点击 Mobile data usage；要选择时间，请点击向下箭头；要查看每个应用使用了多少数据，拉到图表下方。
   </li>
  </ul>
  <p class="graf graf--p">
   对于 iPhone：
  </p>
  <p class="graf graf--p">
   要打开或关闭蜂窝数据，请转到 “设置”，然后点击 “蜂窝数据” 或 “移动数据”。如果您使用的是iPad，则可能会看到 Settings &gt; Cellular Data；如果您使用的是双SIM卡 iPhone，则需要将其中一个号码设置为主要数据。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、耗电量增加，明显比平时更费电
   </strong>
  </p>
  <p class="graf graf--p">
   这是另一个感染迹象。尽管有时这可能表明电池已损坏，但仍有多种病毒可能会导致智能手机电池快速耗尽。
  </p>
  <p class="graf graf--p">
   所以依旧要建议您在花钱更换电池之前，应该首先对手机进行排除感染的扫描。2019年2月，研究人员揭露了一项名为
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hackread.com/android-ad-fraud-campaign-drains-battery-eats-data/" href="https://www.hackread.com/android-ad-fraud-campaign-drains-battery-eats-data/" rel="noopener noreferrer" target="_blank">
    DrainerBot
   </a>
   的 Android 主要广告欺诈骗局，其特征就是费电和吞噬设备数据。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12996 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-15.png?resize=1100%2C739&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-15.png?w=1349&amp;ssl=1 1349w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-15.png?resize=300%2C201&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-15.png?resize=1024%2C688&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-15.png?resize=768%2C516&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-15.png?resize=1100%2C739&amp;ssl=1 1100w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-15.png?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="739" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-15.png?resize=1100%2C739&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12996" data-recalc-dims="1" height="739" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-15.png?resize=1100%2C739&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-15.png?w=1349&amp;ssl=1 1349w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-15.png?resize=300%2C201&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-15.png?resize=1024%2C688&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-15.png?resize=768%2C516&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-15.png?resize=1100%2C739&amp;ssl=1 1100w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-15.png?resize=272%2C182&amp;ssl=1 272w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、智能手机的性能突然下降
   </strong>
  </p>
  <p class="graf graf--p">
   市场上一些较新的智能手机倾向于配备功能强大的处理器和足够的RAM，以轻松处理应用程序和游戏中使用的最新技术。
  </p>
  <p class="graf graf--p">
   当装有这种技术的智能手机 *突然* 开始出现性能降低的情况时，也可能表明该设备已感染病毒。
  </p>
  <p class="graf graf--p">
   这也许不大容易被注意到。有时只有进入需要强大CPU和GPU的游戏时，性能下降才会引起注意。
  </p>
  <p class="graf graf--p">
   而在其他情况下，病毒也可能导致移动设备的整体性能下降，即使执行简单的任务（如拨打电话或阅读短信）也是如此
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、智能手机开始过热
   </strong>
  </p>
  <p class="graf graf--p">
   除了发现智能手机的性能不足外，过热可能是另一个迹象表明它已被
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hackread.com/hiddenminer-android-monero-mining-malware-cause-device-failure/" href="https://www.hackread.com/hiddenminer-android-monero-mining-malware-cause-device-failure/" rel="noopener noreferrer" target="_blank">
    HiddenMiner
   </a>
   之类的恶意软件感染。
  </p>
  <p class="graf graf--p">
   在大多数智能手机中这都是一个罕见的问题。因此，当出现过热问题时，病毒扫描可以帮助检查手机上的恶意软件。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、陌生的应用程序开始显示在设备上
   </strong>
  </p>
  <p class="graf graf--p">
   感染智能手机的病毒有时可能会导致您不熟悉的应用程序被自动安装。
  </p>
  <p class="graf graf--p">
   因此，如果有一些应用程序在用户不知情的情况下出现在手机上，就可能是该设备已感染病毒的又一个迹象。
  </p>
  <p class="graf graf--p">
   当该应用程序看起来很可疑时，不仅应将其删除，而且，还应彻底扫描设备本身以查找可能的病毒的残留痕迹。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    6、Wi-Fi和移动数据自动开启
   </strong>
  </p>
  <p class="graf graf--p">
   您应该时刻警惕移动数据和Wi-Fi开关。由于病毒通常会通过互联网发送和接收数据，在用户禁用移动数据或Wi-Fi连接后感染也可能会导致数据被打开。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    7、弹窗广告
   </strong>
  </p>
  <p class="graf graf--p">
   另一个烦人但很明显的迹象是弹出广告的突然出现。即使在用户离开应用程序并留在手机首页后，这些内容仍然会显示出来。
  </p>
  <p class="graf graf--p">
   这类弹出广告有时会显示不适当的内容，或将用户引导至智能手机应用程序商店中。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    结论
   </strong>
  </p>
  <p class="graf graf--p">
   恶意软件会同时影响台式计算机和移动设备。当智能手机受到影响时，问题可能更严重，因为那上面通常都有银行卡和信用卡的详细信息。
  </p>
  <p class="graf graf--p">
   如果您怀疑自己可能受到感染，就需要了解如何确定病毒的存在；这可以帮助您采取措施删除病毒以保护您的敏感数据。⚪️
  </p>
