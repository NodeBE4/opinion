     <em>
      三个典型案例的核查结果都能违背很多人的既有认知，关键在于人们是否能承认这些结论，将意味着是否能行使真正的公民权

  <div class="section-inner sectionLayout--insetColumn">
   <p class="graf graf--p graf-after--figure" id="6c30">
    上一篇文章中我们强调了：是否承认真相或虚假应该与您的立场如何完全无关。这几乎是一句废话，但是它的确已经成为假消息侵蚀民主的主要原因。假消息本身并不是绝对致命的，真正致命的因素是缺乏判断力的受众，甚至在核查结论推出后依然不承认或无视。很难想象吗？并不然，以下三个案例将能够清楚地揭示这一问题。
   </p>
   <p class="graf graf--p graf-after--p" id="81a1">
    <strong class="markup--strong markup--p-strong">
     1、高度发达和富有的美国真的有如此悲惨的公民冻饿在街头吗？
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="7dbd">
    上月底，一张极其骇人的照片在社交网络流传。照片显示了一名严重营养不良的无家可归者，他说自己是一名退伍军人，瘦得皮包骨，就躺在洛杉矶的一条街道上。他说自己叫 Kent。
   </p>
   <p class="graf graf--p graf-after--p" id="ca86">
    这些照片显然让很多观众感到震惊，不仅包括无条件崇拜美国的中国观众，一些美国观众也不敢承认，他们认为这些照片可能只是艺术作品或万圣节装饰，而不是真人。不幸的是，经过第三方独立核查机构的验证，它就是真实的。
   </p>
   <p class="graf graf--p graf-after--p" id="77a9">
    这些照片首先由 Facebook 用户 Nick Spano 于 2018 年9月22日在线发布。最初的 Facebook 帖子包含了一些长篇的文章，主要讲述的是 Spano 关于美国如何对待无家可归者（特别是无家可归的退伍军人）的观点。一些关于个人细节的描述显示，此人是名叫 Kent 的 68 岁男子。这名男子告诉 Spano，他是一名老兵，当 Spano 给了他一些水，然后叫来医护人员将他送到医院时，他说自己正在“找个能死的地方”。
   </p>
   <p>

   <p class="graf graf--p graf-after--figure" id="5816">
    以下是对照片发布者 Nick Spano 的帖子的部分翻译。不得不翻译，因为他说的是真话（也是很多美国公民的共识）：
   </p>
   <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="e7b4">
    “这里是美国：Kent，这位 68 岁的退伍老兵正在洛杉矶市区的丛林中死去。他脸上一侧的垒球大小的肿瘤现在变成了一个巨大的洞，干血覆盖着他的手和脸，苍蝇就停在他的头上等着他死掉，然后将卵产在他撕裂的伤口上。他试图说话，但吐字模糊我根本听不懂。这是他妈的美国！
   </p>
   <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="53a0">
    “我只是想找个能死的地方，”他说。这就是他对我说的第一句话，当时他就躺在我家外面的灌木丛里。像许多人一样，我最初的直觉反应是，你可别死在我的后院里！但我知道，无家可归是我们破碎的和被浪费的体系之症状，基本缺乏体面，对彼此的同情。我们像垃圾一样对待别人。我们想要丢弃我们不想要的社会部分，但是就像塑料一样，它们永远不会消失。我们所做的只是让问题变得更糟，而不是意识到我们自己的习惯正是导致问题的原因。我们对待他人的方式就是问题所在……
   </p>
   <p class="graf graf--p graf-after--p" id="656b">
    所以，我带了点水给 Kent，并坐下来与他谈了一段时间。他把帽衫拉到一边，向我展示他脸上的肿瘤。他说他去了弗吉尼亚州接受医疗护理，但他说他觉​​得医生只是在等他死，所以他就离开了。我问是否可以给他一些帮助，但他拒绝了。“我只是想找个能死的地方，”他说。
   </p>
   <p class="graf graf--p graf-after--p" id="d0fb">
    伙计们，看看我们自己有多糟糕！让我们对自己诚实点儿吧。在围观 Instagram 自拍、展览 Facebook 帖子，以及对哪个党派更美国化的激烈政治咆哮之间，我们已经失去了灵魂。我们忘记了真正的财富来自彼此的帮助。
    <strong class="markup--strong markup--p-strong">
     这不是“Me too”时代，它只是“Me”时代，是人类历史上最富有、最浪费的消费文化时代，我们购买他们卖给我们的一切，甚至是他们的借口……
    </strong>
    我们无法解决那些糟糕的问题？！因为问题太大了！太贵了！不。
   </p>
   <p class="graf graf--p graf-after--p" id="3666">
    我们在叙利亚投下的每枚炸弹花费了1亿美元，我们已经投下了数十枚。伊拉克战争花费了超过2.5万亿美元的美国纳税人血汗钱，这场战争就是一个谎言！我们有自己的国家紧急情况，但我们没有足够的钱来解决它。你能想象吗，如果我们只把在其中一枚炸弹上投入的资金节省下来，并将其投入到美国社区的社会服务中会发生什么; 让无家可归者进入庇护所; 协助医疗服务; 给弗林特干净的水？！？！（弗林特的水危机问题是多年的热点）
   </p>
   <p class="graf graf--p graf-after--p" id="a65f">
    不要说我们无法解决问题，因为那是胡说八道。我们可以！！！我们真是被洗脑洗残了，连我也自满地认为我们实际上没有任何力量可以改变这些现状。但是，当我们停止忽视问题时，改变就会开始，要意识到我们就是能解决问题的人。即使是最小的行为也会产生很大的影响。
   </p>
   <p class="graf graf--p graf-after--p" id="44bd">
    我在等待护理人员到达的时候，拍了这些“Kent”的照片，然后把它张贴在这里供所有人浏览，这件事没有在我身上被忽视埋没。也许我的行为对某些人来说可能显得虚伪，但这不会改变我对社区中“Kent”这样的人的承诺。我将继续以尊严和尊重的态度对待这些人，无论如何我都能为他们服务。
   </p>
   <p class="graf graf--p graf-after--p" id="7e7d">
    如果这个帖子至少得到你们中一个人的支持，下次当你看到有人在街头走投无路时，请停下来与他们交谈吧; 把手放在他们的肩上; 称他们为兄弟或姐妹; 通过一种简单的同情来恢复他们心中已经微小的尊严感，这必然是值得的。不是每个人都能以同样的方式提供帮助，有时人们会不想要帮助，但是，请让我们彼此保持人性。“
   </p>
   <p class="graf graf--p graf-after--p" id="2b72">
    ……
   </p>
   <p class="graf graf--p graf-after--p" id="26a2">
    Spano 几天后更新了他的帖子，但他没有太多额外的信息可以报告，他写道：“不幸的是，目前没什么进展可以分享。我终于找到了 Kent 被带走和接受治疗的医院，但由于患者隐私法（HIPPA）以及我不知道 Kent 全名的事实，院方现在不会给我任何信息。我不知道 Kent 是否已经过世，或者在接受照料。我已向主管请求分享信息给我，我也明确表示有很多人想表达他们的支持，所以我希望他们能帮助我们实现这一目标。”
   </p>
   <p class="graf graf--p graf-after--p" id="a21a">
    第三方事实核查机构
    <a class="markup--anchor markup--p-anchor" data-href="https://www.snopes.com/fact-check/homeless-veteran-dying-bush/" href="https://www.snopes.com/fact-check/homeless-veteran-dying-bush/" rel="noopener nofollow noreferrer" target="_blank">
     snopes 证实了这条信息的真实性
    </a>

    验证结果显示，这段视频与华盛顿特区、特朗普总统、或性侵嫌疑人 Brett Kavanaugh 没有任何关系。它实际上是在法国巴黎的 Boulevard Peripherique 拍摄到的，并且自 2016 年以来一直在线：
    <a class="markup--anchor markup--p-anchor" data-href="https://youtu.be/luQBhadCfuA" href="https://youtu.be/luQBhadCfuA" rel="noopener nofollow noreferrer" target="_blank">
     视频在这里
    </a>
    (如果下面的演示无法观看的话可以点击这个链接。新的版权法案非常烦人)
   </p>

    <p class="graf graf--p graf-after--figure" id="c672">
     虽然不确定这段视频的真实拍摄时间，但它清楚地显示了地理位置是巴黎的 Boulevard Peripherique。
     <a class="markup--anchor markup--p-anchor" data-href="https://www.gettyimages.com/detail/news-photo/bikers-demonstrate-on-the-boulevard-peripherique-in-paris-news-photo/521876214" href="https://www.gettyimages.com/detail/news-photo/bikers-demonstrate-on-the-boulevard-peripherique-in-paris-news-photo/521876214" rel="noopener nofollow noreferrer" target="_blank">

    <p>
     <em class="markup--em markup--blockquote-em">
      **我们曾经多次强调视频和图像被技术性修改的错误信息制造方法，比如
     </em>
     <a class="markup--anchor markup--blockquote-anchor" data-href="https://medium.com/@iyouport/%E6%B7%B1%E5%9C%B3%E9%81%AD%E9%81%87%E5%8F%B0%E9%A3%8E%E7%9A%84-dragonair-%E6%B3%A2%E9%9F%B3-737-%E7%9A%84%E8%A7%86%E9%A2%91%E6%98%AF%E5%81%87%E7%9A%84-%E8%BF%99%E9%87%8C%E6%98%AF%E4%B8%BA%E4%BB%80%E4%B9%88-44fa902c10f7" href="https://medium.com/@iyouport/%E6%B7%B1%E5%9C%B3%E9%81%AD%E9%81%87%E5%8F%B0%E9%A3%8E%E7%9A%84-dragonair-%E6%B3%A2%E9%9F%B3-737-%E7%9A%84%E8%A7%86%E9%A2%91%E6%98%AF%E5%81%87%E7%9A%84-%E8%BF%99%E9%87%8C%E6%98%AF%E4%B8%BA%E4%BB%80%E4%B9%88-44fa902c10f7" rel="noopener noreferrer" target="_blank">
      <em class="markup--em markup--blockquote-em">
       这篇文章
      </em>
     </a>
     <em class="markup--em markup--blockquote-em">
      ，还有这篇，这是政治博弈游戏中越来越常见的玩法。但日常情况下，更为多见的是将真实的视频和图片安插在虚假的描述之下，以制造符合发布者政治立场的内容。想必至少一半的中国读者见过这类操作的至少一个案例。揭穿这类谎言的方法相比下很简单，就是我们介绍过的反向图片搜索，目的是找到该视频最初在互联网上出现的事件及其描述。**
     </em>
    </p>
   </blockquote>
   <p class="graf graf--p graf-after--blockquote" id="185b">
    以上是三个比较典型的案例，关于：
    <span class="markup--quote markup--p-quote is-other" data-creator-ids="anon">
     <strong class="markup--strong markup--p-strong">
      即便违背你的认知的信息也有可能是真的；即便是主流媒体发布的信息也有可能是假的；即便视频本身没有被篡改过，该消息也有可能不实。
     </strong>
    </span>
   </p>
   <p class="graf graf--p graf-after--p" id="974c">
    三个案例都比较简单。但要知道，如今造假的方法已经多种多样，尤其是在技术的配合之下，分辨起来恐怕不会很轻松。但只要我们有追求真相的勇气，大家联合起来各自发挥自己的能力优势，我们的民主就能得到保障。非常期待中国的积极人士能参与到事实核查的工作中来，这将是一种绝佳的公民社会经验积累。
