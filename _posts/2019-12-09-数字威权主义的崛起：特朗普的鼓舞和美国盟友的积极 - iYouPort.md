  </p>
  <p class="graf graf--p">
   这显然是个相对狭隘的角度。

  <p class="graf graf--p">
   尽管有明确的相反证据，特朗普总统依旧声称，“没有人能直接指责王储穆罕默德·本·萨勒曼”谋杀了著名的沙特记者 Jamal Khashoggi。
  </p>
  <p class="graf graf--p">
   记者指出了中央情报局和联合国提供的调查结果，即 萨勒曼可能会直接授权法外处决，但是特朗普对这个调查结果提出了质疑，并
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnn.com/2019/06/29/politics/trump-khashoggi-bin-salman/index.html" href="https://www.cnn.com/2019/06/29/politics/trump-khashoggi-bin-salman/index.html" rel="noopener noreferrer" target="_blank">
    重申了
   </a>
   他与王储的关系强度：“我与穆罕默德相处融洽。”
  </p>
  <p class="graf graf--p">
   特朗普
   <strong class="markup--strong markup--p-strong">
    绝对不是第一个
   </strong>
   公然无视国际侵犯人权行为的美国总统，特别是当侵犯人权罪行是由美国盟友犯下的时候。
  </p>
  <p class="graf graf--p">
   然而，这种动态似乎在特朗普的领导下降至了新低 —— 特朗普不仅给予了专制领导人以绝对支持的态度、并且，继续提供重要的军事资源，以实际行动支持中东和北非地区的非民主政权。
  </p>
  <p class="graf graf--p">
   斯坦福大学研究员拉里·戴蒙德（Larry Diamond）研究当前
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" rel="noopener noreferrer" target="_blank">
    全球民主倒退
   </a>
   的问题，他
   <a class="markup--anchor markup--p-anchor" data-href="https://www.foreignaffairs.com/articles/2019-06-11/democracy-demotion" href="https://www.foreignaffairs.com/articles/2019-06-11/democracy-demotion" rel="noopener noreferrer" target="_blank">
    认为
   </a>
   ，“只有美国再次承担起推动民主的斗争，民主的衰落才会逆转”。
  </p>
  <p class="graf graf--p">
   然而，作为
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" rel="noopener noreferrer" target="_blank">
    全球监视资本主义专制
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%95%b0%e5%ad%97%e6%ae%96%e6%b0%91%e4%b8%bb%e4%b9%89%e5%a8%81%e8%83%81%e5%85%a8%e7%90%83%e5%8d%97%e6%96%b9%e5%9b%bd%e5%ae%b6/" href="https://www.iyouport.org/%e6%95%b0%e5%ad%97%e6%ae%96%e6%b0%91%e4%b8%bb%e4%b9%89%e5%a8%81%e8%83%81%e5%85%a8%e7%90%83%e5%8d%97%e6%96%b9%e5%9b%bd%e5%ae%b6/" rel="noopener noreferrer" target="_blank">
    数字殖民
   </a>
   的发源地，美国需要思考的东西不只有“再次承担”这么简单。
  </p>
  <p class="graf graf--p">
   一个重要但经常被忽视的方面是，高科技监视和审查技术的扩散如何帮助支持了专制政权，通过在线监控其整个社会、控制所有可用信息、
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%b5%81%e4%ba%a1%e4%b9%9f%e6%b2%a1%e6%9c%89%e5%ae%89%e5%85%a8/" href="https://www.iyouport.org/%e6%b5%81%e4%ba%a1%e4%b9%9f%e6%b2%a1%e6%9c%89%e5%ae%89%e5%85%a8/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     在全球范围内
    </strong>
    跟踪和扼制持不同政见者
   </a>
   。
  </p>
  <p class="graf graf--p">
   联合国公布的关于谋杀 Khashoggi 的长达100页的报告中指出：“沙特当局通过使用
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/report%ef%bc%9a%e4%bb%a5%e8%89%b2%e5%88%97-nso-%e7%94%9f%e4%ba%a7%e7%9a%84%e9%97%b4%e8%b0%8d%e8%bd%af%e4%bb%b6-pegasus-%e5%9c%a845%e4%b8%aa%e5%9b%bd%e5%ae%b6%e8%a2%ab%e7%94%a8%e4%ba%8e%e7%9b%91/" href="https://www.iyouport.org/report%ef%bc%9a%e4%bb%a5%e8%89%b2%e5%88%97-nso-%e7%94%9f%e4%ba%a7%e7%9a%84%e9%97%b4%e8%b0%8d%e8%bd%af%e4%bb%b6-pegasus-%e5%9c%a845%e4%b8%aa%e5%9b%bd%e5%ae%b6%e8%a2%ab%e7%94%a8%e4%ba%8e%e7%9b%91/" rel="noopener noreferrer" target="_blank">
    以色列间谍软件 Pegasus
   </a>
   感染了 Abdulaziz 先生的电话，获得了 Khashoggi 与[沙特籍流亡异议人士] Abdulaziz 先生的私密通信”，这就是 Khashoggi 被谋杀的原因。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a1%85%e8%b0%b7%e5%85%85%e6%96%a5%e7%9d%80%e4%b8%ad%e5%9b%bd%e5%92%8c%e6%b2%99%e7%89%b9%e7%9a%84%e8%b5%84%e9%87%91%ef%bc%8c%e6%9c%89%e8%b0%81%e5%85%b3%e5%bf%83%ef%bc%9f/" href="https://www.iyouport.org/%e7%a1%85%e8%b0%b7%e5%85%85%e6%96%a5%e7%9d%80%e4%b8%ad%e5%9b%bd%e5%92%8c%e6%b2%99%e7%89%b9%e7%9a%84%e8%b5%84%e9%87%91%ef%bc%8c%e6%9c%89%e8%b0%81%e5%85%b3%e5%bf%83%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    沙特
   </a>
   否认这些指控。
  </p>
  <p class="graf graf--p">
   间谍软件来自
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e4%bb%a5%e8%89%b2%e5%88%97%e5%92%8cnso/" href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e4%bb%a5%e8%89%b2%e5%88%97%e5%92%8cnso/" rel="noopener noreferrer" target="_blank">
    以色列臭名昭著的NSO集团
   </a>
   ，这是一家监视技术公司，已被发现
   <strong class="markup--strong markup--p-strong">
    向全球专制政权
   </strong>
   出售侵入式监控技术。该集团在今年早些时候由
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e5%85%ac%e5%85%b3%e5%85%ac%e5%8f%b8%e4%b8%ba%e4%bb%a5%e8%89%b2%e5%88%97%e9%97%b4%e8%b0%8d%e8%bd%af%e4%bb%b6%e5%b7%a8%e5%a4%b4%e6%9c%8d%e5%8a%a1/" href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e5%85%ac%e5%85%b3%e5%85%ac%e5%8f%b8%e4%b8%ba%e4%bb%a5%e8%89%b2%e5%88%97%e9%97%b4%e8%b0%8d%e8%bd%af%e4%bb%b6%e5%b7%a8%e5%a4%b4%e6%9c%8d%e5%8a%a1/" rel="noopener noreferrer" target="_blank">
    美国私募股权公司 Francisco Partners 拥有
   </a>
   。
  </p>
  <p class="graf graf--p">
   使用以色列制造的监视技术杀人，只是近年来在沙特阿拉伯发生的一系列数字权利侵犯行为中的一个而已。
  </p>
  <p class="graf graf--p">
   自由之家的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%95%b0%e5%ad%97%e5%a8%81%e6%9d%83%e4%b8%bb%e4%b9%89%e7%9a%84%e5%85%b4%e8%b5%b7-%e7%81%be%e5%b9%b4%e4%b9%8b%e7%95%94/" href="https://www.iyouport.org/%e6%95%b0%e5%ad%97%e5%a8%81%e6%9d%83%e4%b8%bb%e4%b9%89%e7%9a%84%e5%85%b4%e8%b5%b7-%e7%81%be%e5%b9%b4%e4%b9%8b%e7%95%94/" rel="noopener noreferrer" target="_blank">
    报告指出
   </a>
   ：“沙特的互联网自由度在2018年因各种形式的针对政治、社会和宗教异议的不容忍升级而下降。”
  </p>
  <p class="graf graf--p">
   根据卡塔尔西北大学的一份报告，在沙特现在有54％的人
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e9%aa%8c%e8%af%81%e6%82%a8%e7%9a%84-vpn-%e8%bf%9e%e6%8e%a5%e6%98%af%e5%90%a6%e5%ae%89%e5%85%a8%ef%bc%9f/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e9%aa%8c%e8%af%81%e6%82%a8%e7%9a%84-vpn-%e8%bf%9e%e6%8e%a5%e6%98%af%e5%90%a6%e5%ae%89%e5%85%a8%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    使用VPN
   </a>
   或类似服务，企图将他们的互联网流量隐藏在政府监控之外以规避审查。
  </p>
  <p class="graf graf--p">
   同样，在埃及 ——
   <strong class="markup--strong markup--p-strong">
    美国的维稳援助每年仍为13亿美元的国家
   </strong>
   —— 该国政权对所有潜在的异议空间均进行了暴力镇压，包括在线空间。
  </p>
  <p class="graf graf--p">
   据人权观察组织称，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/ibm%e5%92%8c%e8%b0%b7%e6%ad%8c%e5%a6%82%e4%bd%95%e9%80%9a%e8%bf%87%e4%b8%ad%e5%9b%bd%e5%8d%8f%e5%8a%a9%e7%8b%ac%e8%a3%81%e6%94%bf%e6%9d%83%e7%9a%84%e9%97%b4%e8%b0%8d%e8%a1%8c%e4%b8%ba%ef%bc%9f/" href="https://www.iyouport.org/ibm%e5%92%8c%e8%b0%b7%e6%ad%8c%e5%a6%82%e4%bd%95%e9%80%9a%e8%bf%87%e4%b8%ad%e5%9b%bd%e5%8d%8f%e5%8a%a9%e7%8b%ac%e8%a3%81%e6%94%bf%e6%9d%83%e7%9a%84%e9%97%b4%e8%b0%8d%e8%a1%8c%e4%b8%ba%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    埃及独裁总统
   </a>
   阿卜杜勒·法塔赫·西西被特朗普称为“一个伟大的人物” ——
   <strong class="markup--strong markup--p-strong">
    这位伟大人物监禁了大约6万名政治犯
   </strong>
   。
  </p>
  <p class="graf graf--p">
   人们因在线分享信息而被捕，有人仅仅因为把米老鼠的耳朵P在了总统照片上而
   <a class="markup--anchor markup--p-anchor" data-href="https://www.journalofdemocracy.org/articles/egyptian-youths-digital-dissent/" href="https://www.journalofdemocracy.org/articles/egyptian-youths-digital-dissent/" rel="noopener noreferrer" target="_blank">
    被捕
   </a>
   。
  </p>
  <p class="graf graf--p">
   随着各国政府越来越希望通过抑制在线言论自由来维持其统治，民主自由受到了整体性的破坏。
  </p>
  <p class="graf graf--p">
   然而，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%ae%a1%e6%9f%a5%e5%92%8c%e8%87%aa%e6%88%91%e5%ae%a1%e6%9f%a5/" href="https://www.iyouport.org/category/%e5%ae%a1%e6%9f%a5%e5%92%8c%e8%87%aa%e6%88%91%e5%ae%a1%e6%9f%a5/" rel="noopener noreferrer" target="_blank">
    审查制度
   </a>
   不仅仅源于对抑制国内异议的需求。区域内冲突也越来越多地影响了当权者对内容的控制。
  </p>
  <p class="graf graf--p">
   伯克曼克莱因研究中心的 Helmi Norman 发现：“国家审查机构阻止了很多有关冲突的观点和报道，只因这些观点和报道与政权自己的叙述相矛盾”。
  </p>
  <p class="graf graf--p">
   防止其他观点的传播可能只会拖延和加剧该地区的冲突，因为公民无法看到各种各样的观点。
  </p>
  <p class="graf graf--p">
   特朗普不仅在这个问题上保持沉默，而且，还有越来越多的证据表明，美国公司直接与专制政权技术制造商合作。
  </p>
  <p class="graf graf--p">
   正如 The Intercept 最近的一份调查报告显示，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/ibm%e5%92%8c%e8%b0%b7%e6%ad%8c%e5%a6%82%e4%bd%95%e9%80%9a%e8%bf%87%e4%b8%ad%e5%9b%bd%e5%8d%8f%e5%8a%a9%e7%8b%ac%e8%a3%81%e6%94%bf%e6%9d%83%e7%9a%84%e9%97%b4%e8%b0%8d%e8%a1%8c%e4%b8%ba%ef%bc%9f/" href="https://www.iyouport.org/ibm%e5%92%8c%e8%b0%b7%e6%ad%8c%e5%a6%82%e4%bd%95%e9%80%9a%e8%bf%87%e4%b8%ad%e5%9b%bd%e5%8d%8f%e5%8a%a9%e7%8b%ac%e8%a3%81%e6%94%bf%e6%9d%83%e7%9a%84%e9%97%b4%e8%b0%8d%e8%a1%8c%e4%b8%ba%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    IBM 和微芯片制造商 Xilinx 已经与 Semptian 合作
   </a>
   —— 这是一家以创建监控技术而闻名的中国公司。
  </p>
  <p class="graf graf--p">
   随着
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e4%b8%8e%e6%93%8d%e7%ba%b5/" href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e4%b8%8e%e6%93%8d%e7%ba%b5/" rel="noopener noreferrer" target="_blank">
    监视技术出售
   </a>
   的不透明现象日益紧张，联合国言论自由问题特别报告员 David Kaye 向联合国提出了全球暂停
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%8f%af%e8%a7%86%e5%8c%96%e6%95%b0%e6%8d%ae%ef%bc%9a%e5%85%a8%e7%90%83%e7%9b%91%e6%8e%a7%e8%a1%8c%e4%b8%9a%e2%80%8a-%e2%80%8a-%e8%b0%81%e6%98%af%e5%a4%a7%e5%93%a5%ef%bc%9f%e8%b0%81/" href="https://www.iyouport.org/%e5%8f%af%e8%a7%86%e5%8c%96%e6%95%b0%e6%8d%ae%ef%bc%9a%e5%85%a8%e7%90%83%e7%9b%91%e6%8e%a7%e8%a1%8c%e4%b8%9a%e2%80%8a-%e2%80%8a-%e8%b0%81%e6%98%af%e5%a4%a7%e5%93%a5%ef%bc%9f%e8%b0%81/" rel="noopener noreferrer" target="_blank">
    销售间谍软件
   </a>
   的建议。
  </p>
  <p class="graf graf--p">
   他写道：“对特定个人 —— 通常是记者、活动家、反对派人士、批评者和行使言论自由权的其他人 —— 的监视，已被证明会导致了严重的任意拘留，有时甚至遭受酷刑，甚至可能导致法外处罚”。
  </p>
  <p class="graf graf--p">
   除了暂停销售之外，世界各国政府也必须开始公开谴责对这种压迫性技术的使用。“那些对其公民下黑手的政权并不仅仅认为特朗普政府只是容忍他们的破坏性冲动，而是肯定了他们的行为。”
  </p>
  <p class="graf graf--p">
   为了全球民主和国际人权，政府必须考虑到数字权利的滥用以及盟友如何使用、分享和出售这些监视技术。如果不能实现这一目标，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e6%8a%80%e6%9c%af%e5%85%ac%e5%8f%b8%e5%a6%82%e4%bd%95%e5%b8%ae%e5%8a%a9%e5%9f%83%e5%b0%94%e5%a4%9a%e5%ae%89%e7%9a%84%e6%9a%b4%e6%94%bf%e5%af%b9%e5%9c%9f%e8%80%b3%e5%85%b6%e5%85%ac/" href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e6%8a%80%e6%9c%af%e5%85%ac%e5%8f%b8%e5%a6%82%e4%bd%95%e5%b8%ae%e5%8a%a9%e5%9f%83%e5%b0%94%e5%a4%9a%e5%ae%89%e7%9a%84%e6%9a%b4%e6%94%bf%e5%af%b9%e5%9c%9f%e8%80%b3%e5%85%b6%e5%85%ac/" rel="noopener noreferrer" target="_blank">
    严重的侵犯人权
   </a>
   行为可能只会继续蓬勃发展。
  </p>
  <p class="graf graf--p">
   Jamal Khashoggi 生前说过的最后一些话是这样的：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “曾经有一段时间，记者们相信互联网将从与印刷媒体相关的审查和控制中解放信息。但这些依赖信息控制的政府已经成功阻止了互联网。”
  </p>
  <p class="graf graf--p">
   要知道，这个集中化的互联网
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%88%91%e4%bb%ac%e4%b8%8d%e8%83%bd%e8%a2%ab%e4%ba%92%e8%81%94%e7%bd%91%e6%91%a7%e6%af%81%e2%80%8a-%e2%80%8a-%f0%9f%93%a2%e6%96%b0%e7%9a%84%e5%85%a8%e7%90%83/" href="https://www.iyouport.org/%e6%88%91%e4%bb%ac%e4%b8%8d%e8%83%bd%e8%a2%ab%e4%ba%92%e8%81%94%e7%bd%91%e6%91%a7%e6%af%81%e2%80%8a-%e2%80%8a-%f0%9f%93%a2%e6%96%b0%e7%9a%84%e5%85%a8%e7%90%83/" rel="noopener noreferrer" target="_blank">
    并不存在解放力量
   </a>
   ，因为基础设施并不是民主的，监视和审查就
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%88%9b%e5%bb%ba%e5%b1%9e%e4%ba%8e%e6%88%91%e4%bb%ac%e8%87%aa%e5%b7%b1%e7%9a%84%e5%9b%bd%e5%9c%9f%ef%bc%9b%e6%8e%a8%e8%8d%90%e9%9d%9e%e6%9a%b4%e5%8a%9b%e7%9b%b4%e6%8e%a5%e8%a1%8c%e5%8a%a8%e7%9a%84/" href="https://www.iyouport.org/%e5%88%9b%e5%bb%ba%e5%b1%9e%e4%ba%8e%e6%88%91%e4%bb%ac%e8%87%aa%e5%b7%b1%e7%9a%84%e5%9b%bd%e5%9c%9f%ef%bc%9b%e6%8e%a8%e8%8d%90%e9%9d%9e%e6%9a%b4%e5%8a%9b%e7%9b%b4%e6%8e%a5%e8%a1%8c%e5%8a%a8%e7%9a%84/" rel="noopener noreferrer" target="_blank">
    内嵌在这个互联网里
   </a>
   ；曾经人们一直被欺骗着。现在人们明白了这个道理，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/web-%e6%b0%91%e4%b8%bb%e5%8c%96%e7%9a%84%e4%b8%80%e5%b0%8f%e6%ad%a5%ef%bc%9a%e5%bc%80%e6%ba%90%e5%8e%bb%e4%b8%ad%e5%bf%83%e5%8c%96%e5%b9%b3%e5%8f%b0-solid-%e6%8e%a8%e5%87%ba/" href="https://www.iyouport.org/web-%e6%b0%91%e4%b8%bb%e5%8c%96%e7%9a%84%e4%b8%80%e5%b0%8f%e6%ad%a5%ef%bc%9a%e5%bc%80%e6%ba%90%e5%8e%bb%e4%b8%ad%e5%bf%83%e5%8c%96%e5%b9%b3%e5%8f%b0-solid-%e6%8e%a8%e5%87%ba/" rel="noopener noreferrer" target="_blank">
    变革的时间才真的到来
   </a>
   。
  </p>
  <p class="graf graf--p">
   数字威权主义在全球崛起，并没有凸显在哪个地区，这足够提醒人们关于为什么
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e6%8a%80%e6%9c%af%e6%9c%89%e5%88%a9%e4%ba%8e%e6%9a%b4%e6%94%bf%ef%bc%9f%e2%80%8a-%e2%80%8a-%e4%b8%80%e5%9c%ba%e4%bd%a0%e5%bf%85%e9%a1%bb%e5%8f%82%e4%b8%8e/" href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e6%8a%80%e6%9c%af%e6%9c%89%e5%88%a9%e4%ba%8e%e6%9a%b4%e6%94%bf%ef%bc%9f%e2%80%8a-%e2%80%8a-%e4%b8%80%e5%9c%ba%e4%bd%a0%e5%bf%85%e9%a1%bb%e5%8f%82%e4%b8%8e/" rel="noopener noreferrer" target="_blank">
    技术有利于暴政
   </a>
   的思考。
  </p>
  <p class="graf graf--p">
   期待对人性之疯狂的主动抑制并不现实，民主能做的最好的事就是，让疯狂难以实现。这就是技术能做到的事，而非政治。⚪️
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://fpif.org/digital-authoritarianism-is-rising-in-the-middle-east/" href="https://fpif.org/digital-authoritarianism-is-rising-in-the-middle-east/" rel="noopener noreferrer" target="_blank">
    Emboldened by Donald Trump, U.S. allies are leading the charge toward greater authoritarianism in the region.
   </a>
  </p>
