     <span style="color: #00ccff;">
      <em>
       从个人情感到政治、商业，如今生活的每一方面，以自我为中心的文化塑造了新的常规和预期，社会性正高速下降。人们无法做出长期承诺，对所有不能直接或立即与自己发生联系的人和观点越来越没兴趣，甚至无法容忍它们的存在。同理心弱化了，“HELP”没人理的调侃漫画在网络上热传，人们越来越难以相信人类之间存在共同点，歧视问题再次高涨触目惊心…这一切是如何造成的？
      </em>
     </span>
    </strong>
   </li>

  <p>
   <span style="color: #515151;">
    日前，一位网友在一面石墙上用马克笔写上了两个字：nobody cares。几分钟内就在Tumblr获得了近70万的点赞，足见寂寞无助之感是有多大众。现在是2017年，自我中心主义前所未有的至高点。
   </span>
  </p>
  <p>
   当被问及“理想是什么”时，一名四五岁的孩子仰着天真稚气的脸坚定的说：“赚大钱，做王健林”。
  </p>
  <p>
   魔兽世界最热的时候我采访过一些玩家，在问及游戏的魅力时他们说：“在游戏中，我可以做任何我想做的事，去任何我想去的地方，我是英雄、是超人和上帝”。那是一副骄傲和自嘲混杂在一起的表情。
  </p>
  <p>
   以下这些头疼病定有一款适合你：
   <br/>
   ​
   <br/>
   <em>
    网络声援高调、高热，具体动员却少有响应；
   </em>
   <br/>
   <em>
    知识分子和学者们猛追热点、只求点击量，放弃理想主义追求，从公众社会中隐退；
    <br/>
    极端主义高涨，政治前所未有的针锋相对；
    <br/>
    民粹政客纷纷崛起、
    <a href="https://web.archive.org/web/20190224234830/https://pao-pao.net/article/782">
     地摊伪媒体膘肥体壮
    </a>
    ，真相是什么鬼？让你开心才是唯一目的；
    <br/>
    痴迷电子游戏的人越来越多，网吧猝死事件频频登陆新闻；
    <br/>
    没人愿意讲理，一切都可用“关你屁事和关我屁事”来解释；
    <br/>
    男朋友劈腿让你措不及防、女朋友剁手俩富二代绑一块都不一定养得起；
    <br/>
    亲戚朋友不再走动，逢年过节抱着手机就够了，身边一片冷清；
    <br/>
    不发红包没人会搭理你，甭管你说的多正确、立场多正义……
    <br/>
    这个社会究竟是怎么了？
   </em>
  </p>
