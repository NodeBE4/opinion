  <p class="graf graf--p">
   根据主板获取的内部文档、屏幕截图和录音的缓存，证明了为微软工作的承包商一直在通过该应用程序的翻译服务窃听Skype用户的私人对话。
  </p>
  <p class="graf graf--p">
   尽管 Skype 的网站表示该公司可能会分析用户想要翻译的电话音频，以“改善聊天平台的服务”，但并未说明这些分析是由人类完成的。
  </p>
  <p class="graf graf--p">
   这好像不奇怪，还记得亚马逊的“机器学习”实际上是一大群人在背后监听吗？见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%9b%86%e4%bd%93%e7%9b%91%e5%90%ac/" href="https://www.iyouport.org/%e9%9b%86%e4%bd%93%e7%9b%91%e5%90%ac/" rel="noopener noreferrer" target="_blank">
    集体监听
   </a>
   》。
  </p>
  <p class="graf graf--p">
   主板获得的 Skype 音频包括人们与亲人之间的私密交谈，还有对个人问题的谈论，如减肥，以及其他问题。
  </p>
  <p class="graf graf--p">
   Motherboard 获得的其他文件显示，微软承包商也在监听用户与公司语音助手 Cortana 通话的语音命令。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “事实上，我甚至可以与你分享其中一些内容，这表明了在保护用户数据方面有多么松懈”，向主板提供文件缓存的微软承包商说。
  </p>
  <p class="graf graf--p">
   主板授权消息源匿名，以便可以更坦率地谈论微软的内部实践，也因为该消息人与该公司签订了保密协议。
  </p>
  <p class="graf graf--p">
   主板获得的音频片段通常很短，持续5到10秒。消息人士说，其他段落可能会更长一些。
  </p>
  <p class="graf graf--p">
   2015年，Skype推出了翻译服务，让用户可以在电话和视频通话期间获得接近实时的音频翻译。
  </p>
  <p class="graf graf--p">
   在该功能发布之前，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wired.com/2014/12/skype-used-ai-build-amazing-new-language-translator/" href="https://www.wired.com/2014/12/skype-used-ai-build-amazing-new-language-translator/" rel="noopener noreferrer" target="_blank">
    WIRED发表了一篇
   </a>
   题为“Skype 如何使用AI构建其惊人的新语言翻译器”的文章。
  </p>
  <p class="graf graf--p">
   该产品确实使用了人工智能，并且在主板自己的测试中的翻译效果令人印象深刻。
  </p>
  <p class="graf graf--p">
   但是，与许多其他人工智能或机器学习项目一样，事实证明，一些工作是由人类推动的而不是机器，人类来完成AI应该完成的相同任务，以便改进算法本身。
  </p>
  <p class="graf graf--p">
   承包商屏幕的附带截图显示，主板获得的部分音频被指定为来自 Skype 的 Android 应用程序的翻译功能。
  </p>
  <p class="graf graf--p">
   Skype 翻译的常见
   <a class="markup--anchor markup--p-anchor" data-href="https://web.archive.org/web/20190805192712/https://support.skype.com/en/faq/FA34583/skype-translator-privacy-faq" href="https://web.archive.org/web/20190805192712/https://support.skype.com/en/faq/FA34583/skype-translator-privacy-faq" rel="noopener noreferrer" target="_blank">
    问题解答说
   </a>
   ，当人们使用该服务时，“Skype 会收集并使用您的对话来帮助改进 Microsoft 产品和服务。
  </p>
  <p class="graf graf--p">
   该部分
   <strong class="markup--strong markup--p-strong">
    并未说明
   </strong>
   是人类在监听 Skype 各种应用程序的翻译功能所捕获的音频。微软的
   <a class="markup--anchor markup--p-anchor" data-href="https://web.archive.org/web/20190805202347/https://privacy.microsoft.com/en-gb/privacystatement" href="https://web.archive.org/web/20190805202347/https://privacy.microsoft.com/en-gb/privacystatement" rel="noopener noreferrer" target="_blank">
    隐私政策
   </a>
   里
   <strong class="markup--strong markup--p-strong">
    也没有
   </strong>
   明确这一点。
  </p>
  <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote">
   <p>
    <strong class="markup--strong markup--blockquote-strong">
     “我听过的一些东西显然可以被描述为手机性爱。”
    </strong>
   </p>
  </blockquote>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “人们使用 Skype 打电话给他们的恋人、面试工作、或与家人联系。公司如果必须记录人们的谈话，应该100％透明，以及这些录音的使用方式”，权利组织 Privacy International 的数据开发项目负责人 Frederike Kaltheuner 在网上聊天时表示。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果您的声音样本要进行人工审核（无论出于何种原因），系统应该询问用户是否对此感到满意，或者至少让用户有权选择退出，”她补充说。
  </p>
  <p class="graf graf--p">
   视频：
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/UnEbQtp_r3A" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   来自Privacy Matters 的活动家 Pat Walshe 表示，“Skype 翻译的营销措辞很模糊”，指的是使用人工智能而不是人类在听。这整个领域都需要进行监管审查。
  </p>
  <p class="graf graf--p">
   在审查了 Skype 翻译常见问题解答之后，他补充道，“我看过这东西，并且不相信这相当于透明和公平的处理。”
  </p>
  <p class="graf graf--p">
   微软发言人在一封电子邮件声明中告诉 Motherboard，“微软收集语音数据，以提供和改进语音服务，如搜索、语音命令、听写或翻译服务 ……微软在收集和使用他们的语音数据之前获得了客户的许可。”
  </p>
  <p class="graf graf--p">
   根据截图和其他文件，当微软提交承包商并录制一段音频时，他们还会获得由 Skype 翻译系统生成的一系列近似翻译。然后，承包商需要选择最准确的翻译或提供自己的翻译。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “我听过的一些东西显然可以被描述为手机性爱。我听到人们在 Cortana 命令中输入的完整地址，或者要求 Cortana 提供色情类内容查询的搜索回复
   </strong>
   。虽然我不确切知道人们如何看待这些信息，但对我来说这很奇怪，它并没有在更加可控的环境中处理，”承包商说。
  </p>
  <p class="graf graf--p">
   微软表示，音频数据仅通过安全的在线门户网站提供给承包商，并且该公司删除了用户或设备识别号码等识别性信息。
  </p>
  <p class="graf graf--p">
   可以肯定的是，
   <strong class="markup--strong markup--p-strong">
    在家工作的承包商正在监听和处理 Skype 和 Cortana 上的音频
   </strong>
   。主板发现微软承包商的在线职位列表表明员工可以在家工作。
  </p>
  <p class="graf graf--p">
   承包商消息人说，“虽然我们无法访问用户可识别性信息，但如果微软用户意识到穿着睡衣坐在家里的随机人员可能会和自己的朋友在网上开玩笑关于他们刚听到的东西，那些用户会喜欢那样的事吗”。⚪️
  </p>
