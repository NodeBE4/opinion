  <p class="graf graf--p">
   互联网上有许多网站保存了开源情报工具列表，包括我们的
   <a class="markup--anchor markup--p-anchor" data-href="https://start.me/p/1kod2L/iyp-direct-action5" href="https://start.me/p/1kod2L/iyp-direct-action5" rel="noopener noreferrer" target="_blank">
    列表-5 “开源情报”
   </a>
   栏目。
  </p>
  <p class="graf graf--p">
   尽管这些列表都非常有用，但是由于选择过多，有时您可能会迷失在里面。
  </p>
  <p class="graf graf--p">
   还是那句话，重要的是思考方式，工具是在思考方式科学确立的前提下提供的辅助。想想看，在您的需求中哪种工具才能提供最佳效果？某种工具是如何工作的？
  </p>
  <p class="graf graf--p">
   此外，许多工具仅适用于特定区域，而不是万能的，
   <strong class="markup--strong markup--p-strong">
    针对您每一次不同的需求，您需要选择不同的工具，或者将它们以最佳方式组合起来
   </strong>
   。
  </p>
  <p class="graf graf--p">
   例如，一些用于定位人的工具主要在北美或澳大利亚等地区有最佳表现；而同一种工具在欧洲和亚洲可能就不太合适。
  </p>
  <p class="graf graf--p">
   因此，是时候创建自己的开源情报流程图了。
  </p>
  <p>
   aware-online 介绍了这件事该怎么做。《
   <a href="https://www.aware-online.com/en/create-your-own-osint-flowcharts/" rel="noopener noreferrer" target="_blank">
    Create your own OSINT flowcharts
   </a>
   》
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    什么是开源情报流程图？
   </strong>
  </h3>
  <p class="graf graf--p">
   使用流程图，您可以示意性地表示和可视化开源情报调查的进程。流程图是一种方便的逐步计划，可用于构建您的研究。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    不要错过
    <strong class="markup--strong markup--li-strong">
     思考方式技巧的介绍
    </strong>
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/osint-%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e5%ba%94%e8%af%a5%e6%98%af%e4%b8%80%e7%a7%8d%e5%bf%83%e6%99%ba%ef%bc%9a%e5%be%97%e5%88%b0%e5%ae%83%e4%bd%a0%e9%9c%80%e8%a6%81%e5%87%a0%e4%b8%aa%e6%ad%a5/" href="https://www.iyouport.org/osint-%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e5%ba%94%e8%af%a5%e6%98%af%e4%b8%80%e7%a7%8d%e5%bf%83%e6%99%ba%ef%bc%9a%e5%be%97%e5%88%b0%e5%ae%83%e4%bd%a0%e9%9c%80%e8%a6%81%e5%87%a0%e4%b8%aa%e6%ad%a5/" rel="noopener noreferrer" target="_blank">
     开源调查应该是一种心智：得到它你需要几个步骤
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   使用流程图，您可以从一个起点开始，然后继续推进深化。
  </p>
  <p class="graf graf--p">
   👉 起点可以是例如研究需要回答的问题或某些能获取到的线索信息，例如 电话号码、电子邮件地址或用户名等。
  </p>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    开源情报流程图如何工作？
   </strong>
  </h3>
  <p class="graf graf--p">
   流程图以某些信息（例如电子邮件地址）开头。为了使自己清楚可以使用电子邮件地址执行哪些步骤，将这些步骤放在流程图中会很有用。
  </p>
  <p class="graf graf--p">
   例如，在上方这个流程图中包括了 “电子邮件&gt;验证” 和 “电子邮件&gt;域” 步骤。这意味着您在调查中可以验证电子邮件地址是否存在/或处于活动状态，并且可以检查是否有与正在调查的电子邮件地址相关联的域名。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    开源情报流程图不是已经存在很长时间了吗？
   </strong>
  </h3>
  <p class="graf graf--p">
   没错。有用的流程图可在 osintframework.com 和 inteltechniques.com 上找到。而且可能还有许多其他网站 —— 您可以使用《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a7%8d%e5%ad%90%e3%80%81%e6%9a%97%e7%bd%91%e3%80%81%e5%8c%ba%e5%9d%97%e9%93%be%e3%80%81%e6%8a%96%e9%9f%b3%e3%80%81telegram%e3%80%81%e6%95%b0%e6%8d%ae%e9%9b%86-%ef%bc%9a%e6%96%b0/" href="https://www.iyouport.org/%e7%a7%8d%e5%ad%90%e3%80%81%e6%9a%97%e7%bd%91%e3%80%81%e5%8c%ba%e5%9d%97%e9%93%be%e3%80%81%e6%8a%96%e9%9f%b3%e3%80%81telegram%e3%80%81%e6%95%b0%e6%8d%ae%e9%9b%86-%ef%bc%9a%e6%96%b0/" rel="noopener noreferrer" target="_blank">
    新生情报工具（12）
   </a>
   》中介绍的 “搜脑图” 工具，然后使用关键字 “OSINT” 找到更多有趣的流程图。
  </p>
  <p class="graf graf--p">
   尽管这些流程图非常有用，但它们并不总是适用于您本人需求的情况。对于欧洲人/亚洲人，有些热门流程图中的某些部分可能效果不佳。
  </p>
  <p class="graf graf--p">
   另外，其他人制作的流程图有时很难理解，因为我们不完全了解它们的含义。这就是为什么您很有必要自己制作适合自己的流程图。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    并不是
   </strong>
   说您能搜到的流程图完全没用哦，对于初学者来说，开源情报社区热门的流程图、尤其是介绍基本思考方式的图，能帮助您理解调查的思路。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    创建自己的开源情报流程图
   </strong>
  </h3>
  <p class="graf graf--p">
   有多种用于创建开源情报流程图的程序。我们更喜欢
   <strong class="markup--strong markup--p-strong">
    XMind ZEN
   </strong>
   ，可以
   <a class="markup--anchor markup--p-anchor" data-href="https://www.xmind.net/zen/" href="https://www.xmind.net/zen/" rel="noopener noreferrer" target="_blank">
    在这里免费下载
   </a>
   。
  </p>
  <p class="graf graf--p">
   选择该软件的原因是它易于使用、图形界面清晰、还有易于导出的能力。
  </p>
  <p class="graf graf--p">
   当然，还有许多其他程序也可以做到这一点。
  </p>
  <p class="graf graf--p">
   您是否已完成下载和安装程序？然后，您就可以开始创建一个新流程图，并为它选择一个主题。在本案例中选择的是 “经典”。

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    进一步完成开源情报流程图
   </strong>
  </h3>
  <p class="graf graf--p">
   上面的流程图当然太简单了，无法实际使用于复杂的调查工作。因此，我们将进一步扩展现有主题，并添加新主题。
  </p>
  <p class="graf graf--p">
   通过单击一个主题，然后单击顶部菜单中的 “
   <em class="markup--em markup--p-em">
    Topic
   </em>
   ” 来开始添加新主题。您可以通过使用鼠标按住主题来拖动主题的位置：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12489 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-8.png?resize=1100%2C474&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-8.png?w=1288&amp;ssl=1 1288w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-8.png?resize=300%2C129&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-8.png?resize=1024%2C441&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-8.png?resize=768%2C331&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-8.png?resize=1100%2C474&amp;ssl=1 1100w" data-recalc-dims="1" height="474" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-8.png?resize=1100%2C474&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12489" data-recalc-dims="1" height="474" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-8.png?resize=1100%2C474&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-8.png?w=1288&amp;ssl=1 1288w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-8.png?resize=300%2C129&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-8.png?resize=1024%2C441&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-8.png?resize=768%2C331&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-8.png?resize=1100%2C474&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   通过上面的流程图，您可以知道，使用电子邮件地址可以采取比第一个最简单的图所显示的更多的步骤。
  </p>
  <p class="graf graf--p">
   例如，现在可以检查目标电子邮件地址是否曾经发生过数据泄露，以及电子邮件地址是否链接到了某个社交媒体帐户。
  </p>
  <p class="graf graf--p">
   但是，应该如何检查呢？这就是需要开始
   <strong class="markup--strong markup--p-strong">
    创建子主题
   </strong>
   的原因：
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-12490 jetpack-lazy-image" data-lazy-sizes="(max-width: 1067px) 100vw, 1067px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-8.png?resize=1067%2C587&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-8.png?w=1067&amp;ssl=1 1067w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-8.png?resize=300%2C165&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-8.png?resize=1024%2C563&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-8.png?resize=768%2C423&amp;ssl=1 768w" data-recalc-dims="1" height="587" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-8.png?resize=1067%2C587&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1067"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12490" data-recalc-dims="1" height="587" sizes="(max-width: 1067px) 100vw, 1067px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-8.png?resize=1067%2C587&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-8.png?w=1067&amp;ssl=1 1067w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-8.png?resize=300%2C165&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-8.png?resize=1024%2C563&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-8.png?resize=768%2C423&amp;ssl=1 768w" width="1067"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   例如，在上面的流程图中显示，您可以采取步骤来检查电子邮件地址是否链接到域，并且可以通过诸如
   <a class="markup--anchor markup--p-anchor" data-href="http://www.sidn.nl" href="https://www.sidn.nl" rel="nofollow noopener noreferrer" target="_blank">
    www.sidn.nl
   </a>
   、
   <a class="markup--anchor markup--p-anchor" data-href="http://www.domainbigdata.com" href="http://www.domainbigdata.com" rel="nofollow noopener noreferrer" target="_blank">
    www.domainbigdata.com
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="http://www.viewdns.info" href="http://www.viewdns.info" rel="nofollow noopener noreferrer" target="_blank">
    www.viewdns.info
   </a>
   之类的网站来执行此操作。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    添加注释
   </strong>
  </h3>
  <p class="graf graf--p">
   您还可以通过单击一个主题，然后单击 “
   <em class="markup--em markup--p-em">
    Notes
   </em>
   ” 来添加注释。例如，在注释中，您可以描述特定步骤的含义。
  </p>
  <p class="graf graf--p">
   例如，在 “电子邮件地址
   <a class="__cf_email__" data-cfemail="664b26" href="/cdn-cgi/l/email-protection">
    [email protected]
   </a>
   gmail.com”步骤中，放置了一个注释，表示可以删除一部分电子邮件地址 (“
   <a class="__cf_email__" data-cfemail="f09a9f989ede949f95b0979d91999cde939f9d" href="/cdn-cgi/l/email-protection">
    [email protected]
   </a>
   ”) (“@ gmail.com”) 保留用户名（“ john.doe”）。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12491 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-5.png?resize=1100%2C538&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-5.png?w=1111&amp;ssl=1 1111w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-5.png?resize=300%2C147&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-5.png?resize=1024%2C500&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-5.png?resize=768%2C375&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-5.png?resize=1100%2C538&amp;ssl=1 1100w" data-recalc-dims="1" height="538" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-5.png?resize=1100%2C538&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12491" data-recalc-dims="1" height="538" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-5.png?resize=1100%2C538&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-5.png?w=1111&amp;ssl=1 1111w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-5.png?resize=300%2C147&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-5.png?resize=1024%2C500&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-5.png?resize=768%2C375&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-5.png?resize=1100%2C538&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    导出开源情报流程图
   </strong>
  </h3>
  <p class="graf graf--p">
   您是否已完成个人开源情报流程图的创建？然后，您可以在所选扩展中导出流程图。
  </p>
  <p class="graf graf--p">
   例如，您可以通过 “
   <em class="markup--em markup--p-em">
    Share
   </em>
   ” 选项将流程图另存为PDF格式或Png格式。
  </p>
  <p class="graf graf--p">
   您是否希望以后能够调整该文件？那就通过 “文件&gt;另存为” 将文件另存为 “XMIND工作簿”。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    您自己的开源情报流程图的优势
   </strong>
  </h3>
  <p class="graf graf--p">
   您是否要构建开源情报研究而不依赖其他工具和列表？您可以根据需要制作流程图，并且可以准确描述所有内容，从而了解自己在做什么。
  </p>
  <p class="graf graf--p">
   一个非常大的附加优点是流程图的基本步骤几乎不会改变，您只需要调整最终步骤（工具/网站）。
  </p>
  <p class="graf graf--p">
   也就是说，例如，检查电子邮件地址是否链接到域的基本步骤不会很快改变，但是这种检查方式可能会改变。通过在流程图中包括多种选择，您将很少会遇到麻烦。
  </p>
  <p class="graf graf--p">
   和任何一种学习一样， 能从书面上教授的都是基本方法，学习取决于举一反N的能力 —— 通过您的具体实践就能发现最适合自己的方法，而后者可能很难通过培训获得。
  </p>
  <p class="graf graf--p">
   勤动手是掌握技能的硬道理！不要错过我们的 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    开源情报
   </a>
   ” 栏目，其中通过实例演示介绍的小技巧都是有用的。⚪️
  </p>
