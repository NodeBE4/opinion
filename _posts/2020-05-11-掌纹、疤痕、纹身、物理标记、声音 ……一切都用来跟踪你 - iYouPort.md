  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     国土安全部拥有一个称为HART的新识别系统
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   全称叫 Homeland Advanced Recognition Technology，该系统是国土安全部正在采用的令人震惊的新的身份监视系统。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    请注意：美国国土安全部正在淘汰基于面部识别的旧系统。它已被HART（基于云的系统）所取代，⚠️该系统可存储数亿人口的身份信息
   </strong>
   。
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    新的基于云的平台称为 Homeland Advanced Recognition Technology System（HART），有望为该部门的生物识别业务带来更多的处理能力、新的分析功能并提高准确性。⚠️它还将使该机构能够
    <strong class="markup--strong markup--blockquote-strong">
     超越如今使用的三种生物识别数据类型（面部、虹膜和指纹），通过各种其他特征（例如掌纹、疤痕、纹身、物理标记、和声音）来识别人
    </strong>
    。（
    <a class="markup--anchor markup--blockquote-anchor" data-href="https://www.nextgov.com/it-modernization/2019/10/legacy-systems-held-dhs-biometrics-programs-back-not-anymore/160347/" href="https://www.nextgov.com/it-modernization/2019/10/legacy-systems-held-dhs-biometrics-programs-back-not-anymore/160347/" rel="noopener noreferrer" target="_blank">
     source
    </a>
    ）
   </p>
  </blockquote>
  <p class="graf graf--p">
   顺便说一句，HART 的云托管工作完全是由亚马逊完成的，你知道 ——
   <strong class="markup--strong markup--p-strong">
    “

   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%9a%e9%a9%ac%e9%80%8a%e5%b8%8c%e6%9c%9b%e5%87%ba%e5%94%ae%e7%9b%91%e6%8e%a7%e5%8d%b3%e6%9c%8d%e5%8a%a1/" href="https://www.iyouport.org/%e4%ba%9a%e9%a9%ac%e9%80%8a%e5%b8%8c%e6%9c%9b%e5%87%ba%e5%94%ae%e7%9b%91%e6%8e%a7%e5%8d%b3%e6%9c%8d%e5%8a%a1/" rel="noopener noreferrer" target="_blank">

   <a class="markup--anchor markup--p-anchor" data-href="https://www.nextgov.com/it-modernization/2017/11/amazon-web-services-announces-secret-cloud-region-cia/142662/" href="https://www.nextgov.com/it-modernization/2017/11/amazon-web-services-announces-secret-cloud-region-cia/142662/" rel="noopener noreferrer" target="_blank">

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     有关 HART 的更多信息
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   随着 HART 的确立，那句古老的谚语已经变得越来越真实：“你可以奔跑，但你无法躲藏” ……
  </p>
  <p class="graf graf--p">
   国土安全部（DHS）对新系统可以带给美国人更多的监视而感到 “高兴”。
  </p>
  <p class="graf graf--p">
   根据国土安全部生物识别身份办公室运营主管 Patrick Nemeth 的说法，通过使该机构摆脱传统系统的限制，HART 还可以让官员们扩大与他们共享生物识别数据和分析功能的外部合作伙伴的网络。
  </p>
  <p class="graf graf--p">
   Nemeth 在接受 Nextgov 采访时表示：“当我们进入 HART 时，我们会变得更好、更快、更强大。’’ “我们将免除现在存在的许多容量问题……然后从那里开始，我们将能够添加更多[功能]。”（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nextgov.com/it-modernization/2019/10/legacy-systems-held-dhs-biometrics-programs-back-not-anymore/160347/" href="https://www.nextgov.com/it-modernization/2019/10/legacy-systems-held-dhs-biometrics-programs-back-not-anymore/160347/" rel="noopener noreferrer" target="_blank">
    source
   </a>
   ）
  </p>
  <p class="graf graf--p">
   国土安全部希望通过其新的和“改进的”监视系统摆脱旧系统的限制。 HART 将使用多个生物特征数据来提高识别准确性。
  </p>
  <p class="graf graf--p">
   如今，当通过旧的 IDENT 监视系统庞大的数据库扫描人的脸部、指纹或虹膜时，系统不会返回任何结果。相反，它汇集了数十个具有不同置信度的潜在候选人的列表，然后，人类分析人员必须仔细检查才能进行最终匹配。
  </p>
  <p class="graf graf--p">
   旧的系统一次只能处理一个模态，因此如果假设使用两个不同的数据点来识别目标人，则他们需要评估两个候选列表以找到一个匹配项。
  </p>
  <p class="graf graf--p">
   例如，如果系统将同一个人识别为最有可能匹配目标指纹和面部的人，这不是问题，但是由于生物识别技术仍是一门不完善的科学，因此结果很少那么清晰。
  </p>
  <p class="graf graf--p">
   但是，新的 HART 监视平台可以在一个查询中包含多个数据点，这意味着它将基于所有可用信息对潜在匹配项进行排名。
  </p>
  <p class="graf graf--p">
   Nemeth 说，这不仅使监视者更容易分析潜在的匹配，而且还将帮助克服经常困扰生物特征扫描的数据质量问题。
  </p>
  <p class="graf graf--p">
   例如，如果面部图像原始，但指纹模糊，则该监视系统将赋予更高质量的数据点以更大的权重。
  </p>
  <p class="graf graf--p">
   未来几年还将添加掌纹、疤痕、纹身和其他形式，该系统将能够将其整合到其匹配过程中。（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nextgov.com/it-modernization/2019/10/legacy-systems-held-dhs-biometrics-programs-back-not-anymore/160347/" href="https://www.nextgov.com/it-modernization/2019/10/legacy-systems-held-dhs-biometrics-programs-back-not-anymore/160347/" rel="noopener noreferrer" target="_blank">
    source

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%8c%bf%e5%90%8d%e7%bb%88%e7%bb%93%e8%80%85%ef%bc%9a%e4%b8%ba%e4%bb%80%e4%b9%88%e8%af%b4%e5%a3%b0%e9%9f%b3%e8%af%86%e5%88%ab%e6%98%af%e6%89%80%e6%9c%89%e9%9a%90%e7%a7%81%e5%85%a5%e4%be%b5%e6%89%8b/" href="https://www.iyouport.org/%e5%8c%bf%e5%90%8d%e7%bb%88%e7%bb%93%e8%80%85%ef%bc%9a%e4%b8%ba%e4%bb%80%e4%b9%88%e8%af%b4%e5%a3%b0%e9%9f%b3%e8%af%86%e5%88%ab%e6%98%af%e6%89%80%e6%9c%89%e9%9a%90%e7%a7%81%e5%85%a5%e4%be%b5%e6%89%8b/" rel="noopener noreferrer" target="_blank">
     匿名终结者：为什么说声音识别是所有隐私入侵手段中最为可怕的一种？
    </a>
    》
   </li>
  </ul>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     HART 监视系统也将使用DNA
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   还记得前一段时间最具争议的 DNA 站点与面部识别软件合作的报道吗？好吧，HART 现在正在进一步扩大这个邪恶的联盟。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e5%85%ac%e5%8f%b8%e5%a6%82%e4%bd%95%e5%b8%ae%e5%8a%a9%e4%b8%ad%e5%9b%bd%e5%80%9f%e5%8a%a9-dna-%e6%95%b0%e6%8d%ae%e8%bf%bd%e8%b8%aa%e5%85%ac%e6%b0%91/" href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e5%85%ac%e5%8f%b8%e5%a6%82%e4%bd%95%e5%b8%ae%e5%8a%a9%e4%b8%ad%e5%9b%bd%e5%80%9f%e5%8a%a9-dna-%e6%95%b0%e6%8d%ae%e8%bf%bd%e8%b8%aa%e5%85%ac%e6%b0%91/" rel="noopener noreferrer" target="_blank">
     <strong class="markup--strong markup--li-strong">
      美国公司如何帮助中国借助 DNA 数据追踪公民
     </strong>
    </a>
    <strong class="markup--strong markup--li-strong">
     》
    </strong>
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a4%b1%e6%8e%a7%e7%9a%84dna%e6%94%b6%e9%9b%86%e5%ae%9e%e8%b7%b5/" href="https://www.iyouport.org/%e5%a4%b1%e6%8e%a7%e7%9a%84dna%e6%94%b6%e9%9b%86%e5%ae%9e%e8%b7%b5/" rel="noopener noreferrer" target="_blank">
     失控的DNA收集实践
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%bd%93%e6%94%bf%e5%ba%9c%e6%83%b3%e8%a6%81%e4%bd%a0%e7%9a%84-dna/" href="https://www.iyouport.org/%e5%bd%93%e6%94%bf%e5%ba%9c%e6%83%b3%e8%a6%81%e4%bd%a0%e7%9a%84-dna/" rel="noopener noreferrer" target="_blank">
     当政府想要你的 DNA
    </a>
    》
   </li>
  </ul>
  <blockquote class="graf graf--blockquote">
   <p>
    第二阶段征集也将DNA匹配列为 HART 监视系统的潜在应用。尽管该部门目前尚未分析DNA，但官员
    <a class="markup--anchor markup--blockquote-anchor" data-href="https://www.nytimes.com/2019/10/02/us/dna-testing-immigrants.html" href="https://www.nytimes.com/2019/10/02/us/dna-testing-immigrants.html" rel="noopener noreferrer" target="_blank">
     已经宣布
    </a>
    ，⚠️他们将开始将从成千上万被拘留移民中收集的 DNA 添加到 FBI 的犯罪数据库中。Nemeth 在采访中说，该机构仍在研究存储和共享此类敏感数据的法律含义。他说，目前还不清楚DNA信息是否将存储在 HART 系统或单独的数据库中。（
    <a class="markup--anchor markup--blockquote-anchor" data-href="https://www.nextgov.com/it-modernization/2019/10/legacy-systems-held-dhs-biometrics-programs-back-not-anymore/160347/" href="https://www.nextgov.com/it-modernization/2019/10/legacy-systems-held-dhs-biometrics-programs-back-not-anymore/160347/" rel="noopener noreferrer" target="_blank">
     source
    </a>
    ）
   </p>
  </blockquote>
  <p class="graf graf--p">
   好漂亮的辩词。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     DHS运行的监控没有任何类型的监管和规定
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   目前，尚无对收集和使用此类数据的政府机构的监管或监督。公民自由活动家和一些立法者对此表示震惊，他们担心隐私问题和歧视问题。但是，这并没有拖慢DHS的速度。
  </p>
  <p class="graf graf--p">
   批评者对政府错综复杂的信息共享协议提出了特别的质疑，该协议允许数据传播到
   <strong class="markup--strong markup--p-strong">
    远远超出收集权限
   </strong>
   的范围。
  </p>
  <p class="graf graf--p">
   国土安全部目前
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nextgov.com/emerging-tech/2019/08/dhs-collecting-biometrics-thousands-refugees-who-will-never-enter-us/159310/" href="https://www.nextgov.com/emerging-tech/2019/08/dhs-collecting-biometrics-thousands-refugees-who-will-never-enter-us/159310/" rel="noopener noreferrer" target="_blank">
    与众多团体共享
   </a>
   其生物识别数据和功能，包括但不限于司法、国防和其他国家部门。
  </p>
  <p class="graf graf--p">
   Nemeth 表示，在未来的几年中，HART承诺将加强这些伙伴关系，并使其他伙伴 “蓬勃发展”。如今，该部门限制了其他机构对旧的 IDENT 的访问权限，以确保它们不会消耗过多的有限计算能力，而新监视系统 HART 将消除这些限制。（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nextgov.com/it-modernization/2019/10/legacy-systems-held-dhs-biometrics-programs-back-not-anymore/160347/" href="https://www.nextgov.com/it-modernization/2019/10/legacy-systems-held-dhs-biometrics-programs-back-not-anymore/160347/" rel="noopener noreferrer" target="_blank">
    source
   </a>
   ）
  </p>
  <p class="graf graf--p">
   民主与技术中心的自由安全与技术项目的政策顾问玛娜·阿扎尔米（Mana Azarmi）就是反对者之一。
  </p>
  <p class="graf graf--p">
   Azarmi 在与 Nextgov 的一次对话中说，一个人可能会向单个机构提供信息，认为该信息将被用于特定目的，但是根据机构间信息共享的方式，人们可能会发现自己遭受到不可预见的负面后果。
  </p>
  <p class="graf graf--p">
   她说：“政府在分享信息方面有很多余地。” “在这个数据收集令人难以置信的时代，我认为我们需要重新考虑一些现行规则，尤其是在9/11之后蓬勃发展的一些做法。”（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nextgov.com/it-modernization/2019/10/legacy-systems-held-dhs-biometrics-programs-back-not-anymore/160347/" href="https://www.nextgov.com/it-modernization/2019/10/legacy-systems-held-dhs-biometrics-programs-back-not-anymore/160347/" rel="noopener noreferrer" target="_blank">
    source
   </a>
   ）
  </p>
  <p class="graf graf--p">
   你怎么看？
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     许多人自愿提供生物识别数据
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   许多人提供生物统计数据时完全没有仔细考虑这件事。
  </p>
  <p class="graf graf--p">
   他们很高兴地自拍，然后将照片发送到各种社交网络；不仅提供自己的DNA，而且还与许多从未允许其 DNA 进入数据库的亲戚进行在线匹配。
  </p>
  <p class="graf graf--p">
   然后是手机。如果您的手机较新，则很有可能要求您设置指纹登录、面部识别甚至语音识别。
   <strong class="markup--strong markup--p-strong">
    你知道这些生物识别数据最终会落到谁的手里吗？
   </strong>
  </p>
  <p class="graf graf--p">
   除此之外，您的设备正在跟踪您遍历各种各样看似无害的应用程序的每个位置，它们所有都在跟踪你。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     您不能选择退出
    </strong>
   </span>
  </h3>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theorganicprepper.com/the-great-american-dragnet-over-200-million-people-are-in-the-facial-recognition-database/" href="https://www.theorganicprepper.com/the-great-american-dragnet-over-200-million-people-are-in-the-facial-recognition-database/" rel="noopener noreferrer" target="_blank">
    “The Great American Dragnet”
   </a>
   是2013年的文章，那时候面部识别还听起来像某种玩笑。驾驶执照是创建大规模监视数据库的第一个尝试，但即使在2013年，也远远超过了这个数目。
  </p>
  <p class="graf graf--p">
   存在另一个甚至更大的数据库。美国国务院拥有一个包含2.3亿可搜索图像的数据库。持有护照或移民签证的任何人都可能会发现自己被迫放入了这个数据库中。
  </p>
  <p class="graf graf--p">
   以下是谁拥有照片数据库的细分：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    国务院拥有约1500万张护照或签证持有人的照片
   </li>
   <li class="graf graf--li">
    联邦调查局大约有1500万张被捕或被定罪的人的照片
   </li>
   <li class="graf graf--li">
    国防部大约有600万张照片，主要是伊拉克人和阿富汗人的照片
   </li>
   <li class="graf graf--li">
    各个警察机构和州至少拥有2.1亿张驾驶执照照片
   </li>
  </ul>
  <p class="graf graf--p">
   这种侵犯隐私行为只是监视状态的另一个微小的侧面，考虑到 Edward Snowden 曝光的间谍全球的内幕，以上简直不足为奇。
  </p>
  <p class="graf graf--p">
   我们正在向政府提交自己的身份，他们可以随意识别我们，而无需任何可能的原因。（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theorganicprepper.com/the-great-american-dragnet-over-200-million-people-are-in-the-facial-recognition-database/" href="https://www.theorganicprepper.com/the-great-american-dragnet-over-200-million-people-are-in-the-facial-recognition-database/" rel="noopener noreferrer" target="_blank">
    source
   </a>
   ）
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     东德曾经是东德，而如今，是全球
    </strong>
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%ba%b3%e7%b2%b9%e5%92%8c%e4%bd%a0/" href="https://www.iyouport.org/%e7%ba%b3%e7%b2%b9%e5%92%8c%e4%bd%a0/" rel="noopener noreferrer" target="_blank">
     纳粹和你
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种情况不是某个国家的特例，而是全球性的
   </strong>
   ，在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%93%81%e5%b9%95%e4%b9%8b%e4%b8%8b%ef%bc%9a%e5%85%a8%e7%90%83%e5%8f%8d%e6%8a%97%e8%80%85%e5%a6%82%e4%bd%95%e5%af%b9%e6%88%98%e7%9b%91%e8%a7%86%e8%80%85%ef%bc%9f/" href="https://www.iyouport.org/%e9%93%81%e5%b9%95%e4%b9%8b%e4%b8%8b%ef%bc%9a%e5%85%a8%e7%90%83%e5%8f%8d%e6%8a%97%e8%80%85%e5%a6%82%e4%bd%95%e5%af%b9%e6%88%98%e7%9b%91%e8%a7%86%e8%80%85%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    铁幕之下：全球反抗者如何对战监视者？
   </a>
   》。
  </p>
  <p class="graf graf--p">
   甚至有些人似乎都不介意自己的身份已被政府标记并归档。即便介意也别无选择，如果您想开车或在国外旅行或拥有任何一种政府颁发的身份证（无论是否喜欢），您都在数据库中。
  </p>
  <p class="graf graf--p">
   使用此监视技术的当局声称，其目的是通过帮助防止身份欺诈和识别罪犯来使社会 “更安全”。但是，我们为了所谓的 “安全” 放弃了什么自由？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    我们正在放弃拥有最基本的隐私形式的自由 —— 能够在不被监视和识别的情况下开展日常生活的基本人权
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e7%9c%9f%e7%9a%84%e6%83%b3%e8%a6%81%e6%80%9d%e6%83%b3%e8%87%aa%e7%94%b1%e5%90%97%ef%bc%9f%e5%a6%82%e6%9e%9c%e6%98%af%ef%bc%8c%e5%8f%af%e4%bb%a5%e6%80%8e%e4%b9%88%e5%8a%9e%ef%bc%9f/" href="https://www.iyouport.org/%e4%bd%a0%e7%9c%9f%e7%9a%84%e6%83%b3%e8%a6%81%e6%80%9d%e6%83%b3%e8%87%aa%e7%94%b1%e5%90%97%ef%bc%9f%e5%a6%82%e6%9e%9c%e6%98%af%ef%bc%8c%e5%8f%af%e4%bb%a5%e6%80%8e%e4%b9%88%e5%8a%9e%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     你真的想要思想自由吗？如果是，可以怎么办？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   一旦确定您的身份，监视系统就可以连接到已编译的各种其他个人信息：您的地址、您的行车记录和犯罪记录、以及可能已经整齐地归档到您所在社区的所有其他信息。
  </p>
  <p class="graf graf--p">
   如果您已经无法在没有监视的情况下离开家，您哪里还有任何真正的自由？你只是被囚困在一个大笼子里的牲畜。
  </p>
  <p class="graf graf--p">
   还有多少人尚未真正意识到中国的社会信用少数派报告监视计划、审查制度将变得多么疯狂？
  </p>
  <p class="graf graf--p">
   现在，它就在这里，并且仍在疯狂飞奔。⚪️
  </p>
