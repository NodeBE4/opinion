  </p>
  <p class="graf graf--p">
   👉这其中传播学就是最重要的武器。
  </p>
  <p class="graf graf--p">
   在社会科学领域内，早期的传播学并不是个起眼的学科，直到上世纪50年代，它摇身一变不仅成为了一个独立的学科，并且能授予博士学位；再后来，传播学开始承担着培养新闻记者、公关嘴炮、和广告营销等专业技能人才的任务 — —
   <strong class="markup--strong markup--p-strong">
    这些人才被称为 “意识形态工人”
   </strong>
   。
  </p>

   是什么导致了这一变化？谁在推动它？目的是什么？…… 👉这些问题的答案将能够帮助您理解和抵御今天蔓延全球的心理战、洗脑和操纵。
  </p>
  <p class="graf graf--p">
   心理战是一种帝国的工具，而不是解决冲突的工具；其首要功能是压制和扭曲人民大众中非官方的传播活动，其中当然包括挑战帝国权威的异议。

  <p class="graf graf--p">

    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">

   </strong>

    <strong class="markup--strong markup--p-strong">

    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    的角度进行分析，包括渗透和识别渗透的方法
   </strong>

   <a class="markup--anchor markup--p-anchor" data-href="http://https//start.me/p/1kod2L/iyp-direct-action5" href="http://https//start.me/p/1kod2L/iyp-direct-action5" rel="nofollow noopener noreferrer" target="_blank">
    列表-5 直接行动
   </a>
   。

   </strong>
   在中国社会促进行动力，也没有关系，至少我们应该努力保证它们带来的经验和教训被中文界所了解；更基本的需要保证，
