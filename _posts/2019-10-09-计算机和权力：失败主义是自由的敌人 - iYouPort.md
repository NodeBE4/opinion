   计算机和权力：失败主义是自由的敌人

       相信自己做不到才是真正做不到的根本性原因，而不是被描述的任何障碍
      </strong>
     </em>

  <p>

  </p>
  <p class="graf graf--p">
   当情势对你不利时，需要很大的勇气和强烈的道德信念才能撑住。
  </p>
  <p class="graf graf--p">
   Richard M. Stallman 博士作为计算领域的先驱，他可以看到如何通过控制计算来实现控制用户，并且他开始了解放用户的事业。
  </p>
  <p class="graf graf--p">
   在某些方面，Stallman 的生活和方法可以与甘地相媲美。
  </p>
  <p class="graf graf--p">
   甘地努力让印度人摆脱殖民主义者的攻击，Stallman 的使命是确保软件用户不受软件提供商的控制，这些提供商目前只包含一些巨头企业 —— 他们是互联网上的殖民者。甘地的斗争是为了印度次大陆人民的政治自由，Stallman 反对的是
   <strong class="markup--strong markup--p-strong">
    寡头企业针对世界各地的计算机用户的数字殖民
   </strong>
   。
  </p>
  <p class="graf graf--p">
   甘地的生活具有强烈的价值观和信念，这是我们无法企及的；甘地的坚定信念是在印度的自由斗争中扮演催化剂。 Stallman 博士在计算领域的努力也是如此。
  </p>
  <p class="graf graf--p">
   Richard Stallman 和他的作品曾在20世纪90年代中期引起了公众的注意。这是互联网在全球范围内取得进展的时候。
  </p>
  <p class="graf graf--p">
   为了利用互联网的可能性，类UNIX系统是必要的。1983年，Richard Stallman 推出GNU，作为一个操作系统，GNU可以将一起工作的所有用户联系起来，同时每个用户都可以自由地控制自己的计算机；在老练的互联网用户中非常受欢迎，他传达的自由信念也是如此。
  </p>
  <p class="graf graf--p">
   自由软件开始在全世界范围内赢得利益，虽然更多只是在技术娴熟的用户和活动家中。
  </p>
  <p class="graf graf--p">
   在那些日子里，操作主要发生在桌面或服务器机房中。我们有软件发送和接收邮件，就存储在我们自己的机器中，软件用于制作文档和演示文稿，在自己的机器上运行。
  </p>
  <p class="graf graf--p">
   在2000年初，操作自由看起来好像是可以实现的。从那里开始，流行的操作变成了 Stallman 所谓的“斯大林的梦想”。
  </p>
  <p class="graf graf--p">
   虽然如今我们每天使用的机器（无论是笔记本电脑还是手机）的计算能力已经大幅提升，但今
   <strong class="markup--strong markup--p-strong">
    天我们的大部分计算都发生在少数大型企业的机器上
   </strong>
   。
  </p>
  <p class="graf graf--p">
   我们的电子邮件和办公文档位于Google服务器上。我们与朋友的交流发生在 Facebook 上。我们不再能在自己的机器中存储电子邮件和文档。
  </p>
  <p class="graf graf--p">
   人们心甘情愿地放弃对计算的控制。当然，事情变得更方便了。但是，如果我们不为这种便利付费，那谁来支付呢？
  </p>
  <p class="graf graf--p">
   这种便利会带来多大的代价？这些是人们一度没有提出的重要问题。
  </p>
  <p class="graf graf--p">
   为了方便使用移动技术，我们与一些我们甚至都不知道是谁的人 —— 通常是政府和大公司 —— 分享我们最重要的私密信息。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" rel="noopener noreferrer" target="_blank">
    Edward Snowden 揭露真相
   </a>
   时，世界遭遇了第一次震惊。民族国家的主要领导人受到监视的消息在全世界引起了巨大震动。
  </p>
  <p class="graf graf--p">
   而对于那些谈论操作自由的人来说，这并不奇怪，
   <strong class="markup--strong markup--p-strong">
    只是一个经验验证
   </strong>
   。
  </p>
  <p class="graf graf--p">
   当技术变得非常强大时，除非通过法律和政治手段进行检查，否则它无疑将被用于邪恶的目的。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%E4%B8%BA%E4%BB%80%E4%B9%88%E6%8A%80%E6%9C%AF%E6%9C%89%E5%88%A9%E4%BA%8E%E6%9A%B4%E6%94%BF%EF%BC%9F%e2%80%8a-%e2%80%8a-%E4%B8%80%E5%9C%BA%E4%BD%A0%E5%BF%85%E9%A1%BB%E5%8F%82%E4%B8%8E/" href="https://www.iyouport.org/%E4%B8%BA%E4%BB%80%E4%B9%88%E6%8A%80%E6%9C%AF%E6%9C%89%E5%88%A9%E4%BA%8E%E6%9A%B4%E6%94%BF%EF%BC%9F%e2%80%8a-%e2%80%8a-%E4%B8%80%E5%9C%BA%E4%BD%A0%E5%BF%85%E9%A1%BB%E5%8F%82%E4%B8%8E/" rel="noopener noreferrer" target="_blank">
     为什么技术有利于暴政？ — — 一场你必须参与的政治斗争
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    印度的政治领导层没有对他们受到监视的揭露性信息做出强烈反应。因为该国政府本身就缺乏这样做的道德权威
   </strong>
   。
  </p>
  <p class="graf graf--p">
   事实上，印度拥有自己的监控机器来控制全社会，并且该监控系统一直在不断扩大规模。
  </p>
  <p class="graf graf--p">
   斯诺登的启示谈到了一些间接的操纵。随后就出现了有关于英国脱欧和美国大选受到外部干涉的报道；表明了民主国家在这些新技术面前的脆弱性以及这些寡头企业所掌握的庞大权力。
  </p>
  <p class="graf graf--p">
   很明显，民主是人们对自己行使权力的制度，它容易受到巨头公司和其他政治利益的影响。
  </p>
  <p class="graf graf--p">
   当 Ambani 谈到让印度人民控制印度的数据时，不是为了保护你和我，而是为了保护 Reliance 的利益。因为对他而言，“数据就是新的石油”。
  </p>
  <p class="graf graf--p">
   当他试图通过援引甘地和殖民斗争来煽动民族主义的热情时，我们必须带着一点点审慎；用 Reliance 取代 Google 并不是一个解决方案；它可能会让事情变得更糟。
  </p>
  <p class="graf graf--p">
   有人可能会想，“我怎么能阻止这一切？我太弱小了无法发挥作用吗？”
  </p>
  <p class="graf graf--p">
   Stallman 提醒我们不要陷入这种虚无逻辑的陷阱。这种无助的想法才是导致失败的原因。（更多：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%80%e4%b9%88%e6%98%af%e9%ab%98%e5%b3%b0%e5%86%b7%e6%bc%a0%ef%bc%9f%e4%bb%8e%e7%99%bd%e5%ae%ab%e5%88%b0%e5%be%ae%e4%bf%a1/" href="https://www.iyouport.org/%e4%bb%80%e4%b9%88%e6%98%af%e9%ab%98%e5%b3%b0%e5%86%b7%e6%bc%a0%ef%bc%9f%e4%bb%8e%e7%99%bd%e5%ae%ab%e5%88%b0%e5%be%ae%e4%bf%a1/" rel="noopener noreferrer" target="_blank">
    什么是高峰冷漠？从白宫到微信
   </a>
   》）
  </p>
  <p class="graf graf--p">
   像 Stallman 一样生活，即 不使用现代技术，可能不方便，但并非不可能。他的实践证明了这一点。
  </p>
  <p class="graf graf--p">
   Stallman 对新技术的批判性态度与一个不加批判地接受每项技术的社会相对应。
  </p>
  <p class="graf graf--p">
   确实，技术帮助人类取得了很多成就；Stallman 本人也对其发展做出了巨大贡献。但是，我们究竟想要成为什么样的社会？
  </p>
  <p class="graf graf--p">
   这个问题对我们来说更为重要。对技术进行批判性分析并对其进行民主控制是必要的。技术是政治性的。将技术视为非政治本身就是政治议程的一部分。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    所有先知都会面临同样的挑战：即 大多数人都不会理解他们的警告，直到为时已晚并且毁灭已经接近
   </strong>
   。
  </p>
  <p class="graf graf--p">
   他们就像煤矿里的金丝雀。最终，识别信号还是因无视信号而消亡则是由更大的社会来决定的。
  </p>
  <p class="graf graf--p">
   如果群聚效应能够识别出利害关系 —— 在此是民主和人权 —— 则可以进行及时更正。
  </p>
  <p class="graf graf--p">
   **目前全球只有两个国家掌握数字殖民的权力，即 中国和美国 —— 高堡奇人。印度是高堡奇人的争夺重点，欧洲也是；现在欧洲正在努力打破高堡奇人，希望印度也能赶上来。最重要的是，相信走出高堡奇人的世界是可能的**
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.deccanchronicle.com/discourse/290119/computers-and-authority-defeatism-enemy-of-freedom.html" href="https://www.deccanchronicle.com/discourse/290119/computers-and-authority-defeatism-enemy-of-freedom.html" rel="noopener noreferrer" target="_blank">
    Fighting a battle when the odds are against you requires a lot of courage and a strong moral conviction
   </a>
   .
  </p>
