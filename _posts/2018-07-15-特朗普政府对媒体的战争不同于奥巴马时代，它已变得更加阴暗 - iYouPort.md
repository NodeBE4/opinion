  <p class="graf graf--p graf-after--figure" id="485b">
   上个月，James Wolfe 因为他在参议院情报特别委员会工作期间与四名记者的接触一事向 FBI 说了谎，此事一度是公共舆论的焦点议题，
   <strong class="markup--strong markup--p-strong">
    关于新闻自由被侵蚀的恐惧正在蔓延
   </strong>
   。但他的起诉书以及媒体对此的报道，却只关注了其中一位记者：阿里·沃特金斯（Ali Watkins），起诉书显示她曾经是 Wolfe 多年的女友。
  </p>
  <p class="graf graf--p graf-after--p" id="5f1b">
   <em class="markup--em markup--p-em">
    延伸阅读：我们针对这个案子的报道侧重于情报机构偷窥记者电子邮件和手机记录，应该怎么办《
   </em>
   <a class="markup--anchor markup--p-anchor" data-href="https://steemit.com/crypto/@iyouport/2tkj4i" href="https://steemit.com/crypto/@iyouport/2tkj4i" rel="noopener nofollow noreferrer" target="_blank">
    <em class="markup--em markup--p-em">
     反侦察：情报机构再次监听记者通讯，媒体该如何保证爆料人的安全？
    </em>
   </a>
   <em class="markup--em markup--p-em">
    》
   </em>
  </p>
  <p class="graf graf--p graf-after--p" id="a4be">
   虽然这份长达 11 页的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/4498572-6-7-18-US-v-Wolfe-Indictment.html" href="https://www.documentcloud.org/documents/4498572-6-7-18-US-v-Wolfe-Indictment.html" rel="noopener nofollow noreferrer" target="_blank">
    起诉书
   </a>
   中没有提供关于其他记者的信息，但关于 Watkins 的信息却有很多，她 26 岁，被描述为“记者＃2”。起诉书指出，她的职业生涯始于华盛顿特区，是“新闻服务实习生”（McClatchy Newspapers），并陆续为“几个涉及国家安全的新闻机构”工作（赫芬顿邮报，BuzzFeed 新闻和 Politico） 。
  </p>
  <p class="graf graf--p graf-after--p" id="c623">
   起诉书称，她与 James Wolfe 的关系始于 2013 年 12 月，一直持续到 2017 年 12 月。在此期间， Watkins 发表了数十篇关于情报委员会的文章，那段时间里 Wolfe 正担任参议院情报特别委员会安全主任。
  </p>
  <p class="graf graf--p graf-after--p" id="dcdc">
   这份起诉书简直就是 TMZ 风格的笔触（译者注：
   <strong class="markup--strong markup--p-strong">
    TMZ 是时代华纳旗下的一个八卦网站
   </strong>
   ）。它指出，Watkins 和 Wolfe “经常在各个地方会面。” 在哪里？在参议院办公大楼的楼梯间、餐馆和公寓里。他们曾经在一天内交换了 82 个文本，总共交换了成千上万封短信、电子邮件和电话 — —
   <strong class="markup--strong markup--p-strong">
    说明全部被监听和监视了
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="84e4">
   司法部知道所有这一切，部分是因为司法部秘密拦截了记者 Watkins 多年来的电子邮件和电话记录 — 这是对记者生活的极端性入侵。
  </p>
  <p class="graf graf--p graf-after--p" id="066f">
   <strong class="markup--strong markup--p-strong">
    奇怪的是，这份措辞粗暴的起诉书并没有暗示 Wolfe 向记者 Watkins 透露了一些比他据称向其他记者透露的更为敏感的东西
   </strong>
   。那么为什么 Watkins 在起诉书中获得了最大的关注而不是其他记者？尽管从法律角度来看，她似乎并不是最重要的记者。
  </p>
  <p class="graf graf--p graf-after--p" id="a861">
   据称 Wolfe 与 Watkins 讨论过的信息
   <strong class="markup--strong markup--p-strong">
    都不算机密信息
   </strong>
   — 尽管联邦调查局询问 Wolfe 是否向未具名的“记者＃1”（另一位记者）提供了机密信息。但起诉书没有提供关于那位记者的专业或个人信息，也没有任何关于他们身份的线索。
  </p>
  <p class="graf graf--p graf-after--p" id="7541">
   一众媒体在很大程度上并没有质疑司法部为什么要把焦点放在 Watkins 身上，而是积极地揣测这个案件可能是“纸牌屋”的一集 — — 华盛顿特区一位年轻的女记者通过和有权力的老人上床以获取敏感信息。这一讨论框架对 Watkins 极为有害，指责 Watkins 违反新闻主义 — — 起诉书在暗示这就是记者实际开展工作的方式。
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/julie_kelly2/status/1005445110518243329" href="https://twitter.com/julie_kelly2/status/1005445110518243329" rel="noopener nofollow noreferrer" target="_blank">
    Twitter 上
   </a>
   的特朗普支持者并不羞于提出
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/PoliticalShort/status/1015620224190701568" href="https://twitter.com/PoliticalShort/status/1015620224190701568" rel="noopener nofollow noreferrer" target="_blank">
    这点
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="2dc8">
   值得注意的是：
   <strong class="markup--strong markup--p-strong">
    记者 Watkins 案似乎揭示了特朗普政府在对新闻业的战争中的一种新策略。除了特朗普经常将记者描述为“人民的敌人”和“最不诚实的人”之外，他的司法部也正在使用起诉书指控接受所谓机密信息的记者，这些信息本身在法律上无法被起诉，但他们开始借八卦故事以抹黑记者的地位和信誉 — — 那些发表批评或搅扰白宫的报道的记者（特别是关于俄罗斯干涉美国总统选举的报道）
   </strong>
   。
  </p>
  <blockquote class="graf graf--blockquote graf-after--p" id="b9f3">
   <p>
    The public have an insatiable curiosity to know everything, except what is worth knowing
   </p>
  </blockquote>
  <p class="graf graf--p graf-after--blockquote" id="c031">
   这是对奥巴马政府打压记者的战争的巧妙提升。在其最重要的泄密起诉之一，巴拉克奥巴马的司法部试图
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2018/01/03/my-life-as-a-new-york-times-reporter-in-the-shadow-of-the-war-on-terror/" href="https://theintercept.com/2018/01/03/my-life-as-a-new-york-times-reporter-in-the-shadow-of-the-war-on-terror/" rel="noopener nofollow noreferrer" target="_blank">
    强迫当时的纽约时报记者 James Risen 透露消息来源的身份
   </a>
   。现在为 The Intercept 工作的 Risen 冒着入狱的风险也拒绝暴露信息源，这场斗争使他成为受到
   <a class="markup--anchor markup--p-anchor" data-href="https://www.vanityfair.com/news/2015/03/james-risen-anonymous-source-government-battle" href="https://www.vanityfair.com/news/2015/03/james-risen-anonymous-source-government-battle" rel="noopener nofollow noreferrer" target="_blank">
    广泛称赞
   </a>
   的新闻自由辩护人之一。
  </p>
  <p class="graf graf--p graf-after--p" id="e1d8">
   <em class="markup--em markup--p-em">
    延伸阅读：保护信息源的匿名安全是每个新闻人的重要职责之一，这点上 Wikileaks 是全球媒体中最安全的爆料渠道《
   </em>
   <a class="markup--anchor markup--p-anchor" data-href="https://steemit.com/wikileaks/@iyouport/dnc-wikileaks" href="https://steemit.com/wikileaks/@iyouport/dnc-wikileaks" rel="noopener nofollow noreferrer" target="_blank">
    <em class="markup--em markup--p-em">
     DNC 针对 Wikileaks 的诉讼对新闻自由构成严重威胁
    </em>
   </a>
   <em class="markup--em markup--p-em">
    》
   </em>
  </p>
  <p class="graf graf--p graf-after--p" id="002d">
   请注意，对记者 Risen 发起的舆论攻势也包括对其工作可靠性的负面暗示，但不同的是，奥巴马政府主要集中在让他说出前中央情报局官员 Jeffrey Sterling 是否是他揭露白宫对伊朗进行拙劣的秘密行动的消息来源。
  </p>
  <p class="graf graf--p graf-after--p" id="e426">
   <strong class="markup--strong markup--p-strong">
    而特朗普政府正在放弃对消息来源的追踪，转而专注于利用其泄​​密指控向公众扩散一种信号，寄希望于这些信息能有效地损害给政权“添麻烦”的记者的声誉
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="1505">
   明尼苏达大学媒体伦理与法律教授 Jane Kirtley 对 Wolfe 起诉书中关于记者 Watkins 的大量信息感到震惊。 “由于她的个人行为问题，[检察官]能够播下她的动机不光彩的种子，也许正是为了打击她的事业。从法律角度上来看，记者的私人生活与案子完全不相关。 …… [检察官]故意将记者描绘为不是寻求真相并向公众报道的人。
   <strong class="markup--strong markup--p-strong">
    他们想把她描述为缺乏道德的人
   </strong>
   。”
  </p>

  <p class="graf graf--p graf-after--figure" id="004b">
   如果起诉书的目的是破坏一般记者的声誉，那么可以说它已经取得了令人钦佩的成功，有大量新闻机构中招儿，包括纽约时报，去年年底该报聘请了 Watkins 担任职业作家。为了回应起诉书，“泰晤士报”发表了多篇关于 Watkins 与 Wolfe 的关系的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2018/06/24/business/media/james-wolfe-ali-watkins-leaks-reporter.html" href="https://www.nytimes.com/2018/06/24/business/media/james-wolfe-ali-watkins-leaks-reporter.html" rel="noopener nofollow noreferrer" target="_blank">
    报道
   </a>
   ，其中包括一篇题为“记者和安全助理之间的关系如何令华盛顿媒体感到震惊”的​​三篇文章，3,300 字。这个故事比“泰晤士报”在入侵伊拉克之前对萨达姆·侯赛因持有大规模杀伤性武器的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2004/05/26/world/from-the-editors-the-times-and-iraq.html" href="https://www.nytimes.com/2004/05/26/world/from-the-editors-the-times-and-iraq.html" rel="noopener nofollow noreferrer" target="_blank">
    著名错误报道
   </a>
   的调查时间长三倍。华盛顿邮报也发表了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/blogs/erik-wemple/wp/2018/07/03/new-york-times-ali-watkins-made-some-poor-judgments/?utm_term=.8c66962f86aa" href="https://www.washingtonpost.com/blogs/erik-wemple/wp/2018/07/03/new-york-times-ali-watkins-made-some-poor-judgments/?utm_term=.8c66962f86aa" rel="noopener nofollow noreferrer" target="_blank">
    几篇
   </a>
   关于此案的文章，也是主要集中在 Watkins 身上，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.vanityfair.com/news/2018/06/sex-press-freedom-and-the-complicated-case-of-a-times-reporter" href="https://www.vanityfair.com/news/2018/06/sex-press-freedom-and-the-complicated-case-of-a-times-reporter" rel="noopener nofollow noreferrer" target="_blank">
    许多其他
   </a>
   出版物均是如此。
  </p>
  <p class="graf graf--p graf-after--p" id="8fdd">
   虽然这些文章中的大多数都有指出，案件涉及一个至关重要的却未被充分认识到的问题 — — 政府秘密偷窥记者的通讯记录 — — 侵犯隐私达到了惊人的程度。
  </p>
  <p class="graf graf--p graf-after--p" id="3019">
   但是，
   <strong class="markup--strong markup--p-strong">
    特朗普政府成功地让上述主流媒体把注意力放在了起诉书中的贬义小玩意儿上，而忽略了根本性问题
   </strong>
   。事实上，很难看出将关于 Watkins 与 Wolfe 的关系的许多细节包含在内究竟有什么意义，除了
   <strong class="markup--strong markup--p-strong">
    将公众注意力转移到她的性生活上，并远离政府为获取有关她的信息所做出的侵犯性举动
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="bbe8">
   正如“华盛顿邮报”的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/news/the-fix/wp/2018/06/08/how-a-reporters-romance-with-her-source-muddies-the-fbis-seizure-of-her-records/?utm_term=.15c2d9b7ade9" href="https://www.washingtonpost.com/news/the-fix/wp/2018/06/08/how-a-reporters-romance-with-her-source-muddies-the-fbis-seizure-of-her-records/?utm_term=.15c2d9b7ade9" rel="noopener nofollow noreferrer" target="_blank">
    精准论述
   </a>
   所言：这种浪漫故事“引发了对新闻伦理问题的一系列扭曲认识，并且支持了特朗普将记者定性为华盛顿沼泽生物的歪论，暗示他们可以为挖料儿做任何事情。”
  </p>
  <p class="graf graf--p graf-after--p" id="51b8">
   这不是特朗普政府第一次利用泄密指控起诉并歪曲记者工作。在特朗普时代的第一次经典级泄密起诉中，前国家安全局承包商 Reality Winner 根据“间谍法”被起诉，因为她将一份机密文件交给了记者（Winner 已经认罪）。在对 Winner 的起诉书中携带的搜查令，包括有关她据称将消息泄露给新闻机构的方式的详细信息，该组织已向 NSA 发送了该文件的副本以供核实。逮捕令称副本显示原始文件已折叠，这表明泄密者已将其从工作计算机上打印出来了。
  </p>
  <p class="graf graf--p graf-after--p" id="d39d">
   包含这些信息似乎没有任何法律用途。但为什么会这样？似乎最可能的是，它意在暗示新闻组织无意中提供了导致政府针对 Winner 的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2017/06/06/business/media/intercept-reality-winner-russia-trump-leak.html" href="https://www.nytimes.com/2017/06/06/business/media/intercept-reality-winner-russia-trump-leak.html" rel="noopener nofollow noreferrer" target="_blank">
    关键线索
   </a>
   。据称，“新闻组织”被广泛报道为 The Intercept，在 Winner 被捕期间，The Intercept
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2017/07/11/first-look-to-support-defense-of-reality-winner-in-espionage-act-prosecution/" href="https://theintercept.com/2017/07/11/first-look-to-support-defense-of-reality-winner-in-espionage-act-prosecution/" rel="noopener nofollow noreferrer" target="_blank">
    发布
   </a>
   了一份关于俄罗斯黑客攻击美国选举系统的绝密文件。虽然 The Intercept 承认其文件处理实践并不像应有的那样谨慎，但政府有很多方法可以追踪到 Winner。
  </p>
  <p class="graf graf--p graf-after--p" id="16ef">
   <strong class="markup--strong markup--p-strong">
    特别是自 2013 年爱德华·斯诺登泄露大量国家安全局文件以来，政府机构加强了对公务员的监控，包括监控他们的计算机、打印机和工作空间，跟踪某些文件的打印和查看历史，甚至丢出诱饵文件，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wired.com/2011/11/darpa-trap-wikileaks/" href="https://www.wired.com/2011/11/darpa-trap-wikileaks/" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     引诱
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    潜在的泄漏者进入他们的系统
   </strong>
   。几乎可以肯定， Winner 发送给 The Intercept 的国安局文件被盯住了。
  </p>
  <p class="graf graf--p graf-after--p" id="7e8b">
   特朗普对新闻业发起战争的利害关系远远超出了对记者声誉的抹黑，或其信息来源的命运。
   <strong class="markup--strong markup--p-strong">
    这些攻击是为了减少向公众传播重要信息的整体流量 — 他们向潜在的消息来源发出信号，恐吓他们将被起诉，他们向调查记者发出信号，恐吓他们将被抹黑
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="4648">
   事实上，在关于他与特朗普的一次会谈的备忘录中，前联邦调查局局长 James Comey
   <a class="markup--anchor markup--p-anchor" data-href="https://money.cnn.com/2018/04/20/media/james-comey-trump-jail-journalists/index.html" href="https://money.cnn.com/2018/04/20/media/james-comey-trump-jail-journalists/index.html" rel="noopener nofollow noreferrer" target="_blank">
    写道
   </a>
   ，他提醒总统“信息价值头等重要”要让泄密者和记者停止发布敏感信息 — — 这是公然地破坏新闻自由。
  </p>
  <p class="graf graf--p graf-after--p" id="c927">
   特朗普对记者的战争（以及科米对这一战争的鼓励）的问题在于，泄漏通常是公众需要了解的属于公共领域事物的唯一方式 — 例如，五角大楼文件揭示了越南战争背后的真相。特朗普否认或淡化了俄罗斯在2016年大选中的角色，他的政府一直在关注媒体对政府所作所为的披露。
  </p>
  <p class="graf graf--p graf-after--p" id="54fc">
   The Intercept 发布的泄露文件清楚地表明，美国国家安全局有证据证明俄罗斯试图破解美国的投票系统，参议院的一份
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2018/05/09/russian-hacking-us-election-senate-reality-winner/" href="https://theintercept.com/2018/05/09/russian-hacking-us-election-senate-reality-winner/" rel="noopener nofollow noreferrer" target="_blank">
    报告
   </a>
   和选举官员都已经
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2018/06/20/state-election-russia-hacking-voting-system/" href="https://theintercept.com/2018/06/20/state-election-russia-hacking-voting-system/" rel="noopener nofollow noreferrer" target="_blank">
    承认
   </a>
   ，媒体报道在警告这一威胁方面发挥了关键作用，特别顾问 Robert Mueller 的最新俄罗斯起诉书，其中
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/petersterne/status/1017852728117219328?s=19" href="https://twitter.com/petersterne/status/1017852728117219328?s=19" rel="noopener nofollow noreferrer" target="_blank">
    部分就依赖于
   </a>
   Winner 泄露的文件中的信息。
  </p>
  <p class="graf graf--p graf-after--p" id="a0f0">
   在执政的最后一年，奥巴马政府似乎退出了对记者进行打击的一些最糟糕的部分。奥巴马在位期间的最后一次对泄密起诉是针对已退休的 James Cartwright 将军，他承认向纽约时报记者 David Sanger 和新闻周刊记者 Daniel Klaidman 提供了关于伊朗问题虚假陈述的机密信息。奥巴马在宣判之前赦免了 Cartwright）
  </p>
  <p class="graf graf--p graf-after--p" id="b34f">
   在这个案件中，司法部提交的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.justice.gov/opa/file/903496/download" href="https://www.justice.gov/opa/file/903496/download" rel="noopener nofollow noreferrer" target="_blank">
    两页信息
   </a>
   里提到了 Sanger 和 Klaidman 一次，没有提及他们与 Cartwright 的接触 — 他们经常交谈的时间、地点、是否一起出去喝啤酒，记者是否留下了任何电话或电子邮件帮助政府弄清楚 Cartwright 已将消息泄露给了他们。也没有企图贬低记者 Sanger 和 Klaidman。
  </p>
  <p class="graf graf--p graf-after--p" id="900a">
   <strong class="markup--strong markup--p-strong">
    Wolfe 的起诉书表明贬低运动时代又回来了
   </strong>
   。记者 Watkins 的长期后果尚不清楚。本月早些时候，“泰晤士报”宣布她已从华盛顿特区重新分配到纽约的新职位，在那里她将得到导师的“指导”（被回炉了）。她为此写了一个
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2018/07/03/business/media/ali-watkins-times-reporter-memo.html" href="https://www.nytimes.com/2018/07/03/business/media/ali-watkins-times-reporter-memo.html" rel="noopener nofollow noreferrer" target="_blank">
    950 字的故事
   </a>
   ，故事中显示该报的执行编辑 Dean Baquet 已经向工作人员提供了一份关于 Watkins 的备忘录。该报还发表了
   <a class="markup--anchor markup--p-anchor" data-href="https://int.nyt.com/data/documenthelper/65-dean-baquet-ali-watkins-memo/8c561cd6318a885ae85e/optimized/full.pdf#page=1" href="https://int.nyt.com/data/documenthelper/65-dean-baquet-ali-watkins-memo/8c561cd6318a885ae85e/optimized/full.pdf#page=1" rel="noopener nofollow noreferrer" target="_blank">
    575 字的备忘录
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="2957">
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2018/07/15/ali-watkins-new-york-times-leak-indictment/" href="https://theintercept.com/2018/07/15/ali-watkins-new-york-times-leak-indictment/" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     TRUMP FINDS A NEW WEAPON FOR HIS WAR ON JOURNALISM — LEAK INDICTMENTS AIMED AT SMEARING REPORTERS
    </strong>
   </a>
   by Peter Maass July 15 2018, 1:00 p.m.
  </p>
