  <p class="graf graf--p">
   虚拟借记卡服务可让您在购买时隐藏真实身份信息。
  </p>
  <p class="graf graf--p">
   例如，您可以插入虚拟的借记卡卡号，而不必使用常规的卡号从网上购买比较私密的商品，这样万一该卡发生了任何奇怪的事，它也不会影响您的真实借记卡。
  </p>
  <p class="graf graf--p">
   您可以期望
   <strong class="markup--strong markup--p-strong">
    最好的虚拟卡服务
   </strong>
   提供一些简洁的功能，包括 限制其在某些时间和地点的使用，使用未完全实名的银行账户进行购买，只需几秒钟即可创建新卡，无需激活，并在一段时间后自动过期。
  </p>
  <p class="graf graf--p">
   并没有很多值得选择的虚拟卡服务；它们中的大多数实际上都连接到了现有的银行，但是本文列出的是一些最好的
   <strong class="markup--strong markup--p-strong">
    独立（无银行隶属关系）虚拟支付服务
   </strong>
   ，其中一个是完全免费的（另外两个可免费试用）。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     为什么要使用虚拟借记卡？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   使用虚拟借记卡的最常见原因是为了防止未经授权的收费并隐藏您的真实身份。但是在几种其他情况下您可能也会需要它。
  </p>
  <p class="graf graf--p">
   如果您遇到以下这类情况之一，则值得使用虚拟支付卡：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    对于不信任的网站
   </li>
   <li class="graf graf--li">
    手边没有实体卡可以用
   </li>
   <li class="graf graf--li">
    您正在将您的卡号提供给朋友代表您购买东西或转账
   </li>
   <li class="graf graf--li">
    想要隐藏商家名称，使其不会显示在银行对帐单上
   </li>
   <li class="graf graf--li">
    由于您的实体卡号无效或过期，因此被拒绝购买
   </li>
   <li class="graf graf--li">
    想要限制您在特定网站上的支出
   </li>
  </ul>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">

   <a class="markup--anchor markup--p-anchor" data-href="https://privacy.com/" href="https://privacy.com/" rel="noreferrer noopener" target="_blank">
    Privacy
   </a>
   正如它的名字，就是为隐私而来的。它具有一些独特的功能：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    它是完全免费的（绝对不收取任何费用）
   </li>
   <li class="graf graf--li">
    可以创建一次性借记卡或可重复使用的借记卡
   </li>
   <li class="graf graf--li">
    将卡锁定到一个商户，使其只能在此处使用
   </li>
   <li class="graf graf--li">
    设置最高支出限额（每次、总计或每月/每年）
   </li>
   <li class="graf graf--li">
    规定每张卡从哪个银行提取资金
   </li>
   <li class="graf graf--li">
    随时暂停和恢复卡
   </li>
   <li class="graf graf--li">
    取消卡以永久禁用卡号
   </li>
   <li class="graf graf--li">

   <li class="graf graf--li">

   </li>
   <li class="graf graf--li">
    在线使用或使用
    <a class="markup--anchor markup--li-anchor" data-href="https://play.google.com/store/apps/details?id=com.privacy.pay&amp;hl=en_US" href="https://play.google.com/store/apps/details?id=com.privacy.pay&amp;hl=en_US" rel="noopener noreferrer" target="_blank">
     Android

    或
    <a class="markup--anchor markup--li-anchor" data-href="https://itunes.apple.com/us/app/privacy-com/id1040298266?mt=8" href="https://itunes.apple.com/us/app/privacy-com/id1040298266?mt=8" rel="noopener noreferrer" target="_blank">
     iOS

    应用

  <p class="graf graf--p">
   Privacy 虚拟卡有很多非常好的用途。一种是如果您在陌生或不安全的地方购物，并且担心商户会丢失/出售您的卡号，只需制作一张商户锁定卡并将支出限额设置为所需的确切金额即可。
  </p>
  <p class="graf graf--p">
   获取需要付款信息的试用版是另一个完美的用例。将支出限额设置为1块钱或低于第一笔付款金额的某个限额，然后使用您的卡进行注册即可，无需担心会被收取费用。或者，制作一张卡，然后将其详细信息输入到您要试用的项目，然后注销您的卡。
  </p>
  <p class="graf graf--p">
   您所有虚拟卡上的所有交易都将显示在您的帐户中，就好像这是一张大银行对帐单一样，这有助于保持您对帐户使用方式的了解。
  </p>
  <p class="graf graf--p">
   您还可以查看今天和过去30天内在 Privacy 上花费了多少。
  </p>
  <p class="graf graf--p">
   Privacy 支持
   <strong class="markup--strong markup--p-strong">
    双因素身份验证
   </strong>
   ，这使恶意者即使拥有密码也很难访问您的卡。
  </p>
  <p class="graf graf--p">
   Privacy另一个便捷的安全功能是
   <strong class="markup--strong markup--p-strong">
    “私密付款”选项
   </strong>
   ，该选项将在您的银行对帐单上显示 Privacy 的交易，或者您选择的以下任何一种选项（而不是您实际购买的商家）：Privacy.com，H＆H Hardware，Smileys Corner商店 或 NSA礼品店。
  </p>
  <p class="graf graf--p">
   您还可以通过 Privacy 获得现金返还；使用 Privacy 卡进行的每笔购买可返利1％。但是，您需要一个
   <a class="markup--anchor markup--p-anchor" data-href="https://privacy.com/cashback" href="https://privacy.com/cashback" rel="noopener noreferrer" target="_blank">
    Cashback Key
   </a>
   才能使其正常工作。
  </p>

  </p>
  <p class="graf graf--p">
   使用该虚拟卡非常简单。当需要输入卡信息时，您可以选择制作一张新的化名卡。Blur 会询问购买金额，然后立即为您提供卡号以匿名进行购买。

    Skrill
   </a>
   不仅使您可以在线使用虚拟卡、还可以向朋友汇款、购买加密货币、在线购物甚至玩扑克。
  </p>
  <p class="graf graf--p">
   一旦您的 Skrill 帐户中有钱，就可以制作虚拟卡，以在支持万事达卡的任何地方在线使用。
  </p>
  <p class="graf graf--p">
   只有第一张虚拟卡是免费的，但是它可以让您对网站隐藏个人真实信息，以保护您的隐私并保护资金。
  </p>
  <p class="graf graf--p">
   Skrill 上的虚拟卡可以使用以下任何一种货币：USD、EUR、GBP或PLN。
  </p>
  <p class="graf graf--p">
   您可以随时取消该卡，甚至可以选择该卡应处于激活状态的时间长短，以使其自动过期，对于拥有该卡号的任何人来说都将变得无效。
  </p>
