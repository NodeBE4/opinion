  <p class="graf graf--p">
   中国目前正在测试据称能够检测人们情绪的监视系统。
  </p>
  <p class="graf graf--p">
   虽然并不令人感到意外，而且其基础技术并不像广告宣传中所描述的那样运作良好，但是，中国政府已经开始在中国西部推广这种技术。
  </p>
  <p class="graf graf--p">
   中国的智能城市理念显然还包括安装在公共场所的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.techspot.com/news/62760-after-trying-predict-age-microsoft-has-introduced-new.html" href="https://www.techspot.com/news/62760-after-trying-predict-age-microsoft-has-introduced-new.html" rel="noopener noreferrer" target="_blank">
    “情感识别” 硬件
   </a>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%99%ba%e8%83%bd%e5%9f%8e%e5%b8%82%e7%a9%b6%e7%ab%9f%e6%98%af%e4%b8%aa%e4%bb%80%e4%b9%88%e9%ac%bc%ef%bc%9f%ef%bc%81/" href="https://www.iyouport.org/%e6%99%ba%e8%83%bd%e5%9f%8e%e5%b8%82%e7%a9%b6%e7%ab%9f%e6%98%af%e4%b8%aa%e4%bb%80%e4%b9%88%e9%ac%bc%ef%bc%9f%ef%bc%81/" rel="noopener noreferrer" target="_blank">
     所谓的 “智能城市” 剥夺了哪些人权？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   最近在中国深圳举行的一个监视技术博览会上，该国希望在机场、地铁和购物中心安装各种监视系统，声称 “帮助识别罪犯”。
  </p>
  <p class="graf graf--p">
   参与的公司包括百度和华为，后者已接近从美国的贸易禁令中获得缓和。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%8a%80%e6%9c%af%e8%b6%85%e7%ba%a7%e7%a2%97ces%e6%ad%a3%e5%9c%a8%e5%b8%ae%e5%8a%a9%e4%b8%ad%e5%9b%bd%e7%9b%91%e8%a7%86%e5%85%ac%e5%8f%b8%e6%b8%85%e7%90%86%e5%bd%a2%e8%b1%a1/" href="https://www.iyouport.org/%e6%8a%80%e6%9c%af%e8%b6%85%e7%ba%a7%e7%a2%97ces%e6%ad%a3%e5%9c%a8%e5%b8%ae%e5%8a%a9%e4%b8%ad%e5%9b%bd%e7%9b%91%e8%a7%86%e5%85%ac%e5%8f%b8%e6%b8%85%e7%90%86%e5%bd%a2%e8%b1%a1/" rel="noopener noreferrer" target="_blank">
     技术超级碗CES正在帮助中国监视公司清理形象
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   所有这些都是在该地区寻求更高级监视的广泛努力的一部分，因为中国希望最终能够使用机器学习来 “预测犯罪”。
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   就像海康威视、Sensetime 和 iFlytek 这样的公司已经抓住了这个机会。
  </p>
  <p class="graf graf--p">
   据英国《金融时报》报道，这种情绪识别技术目前正在新疆部署，该地区是在大型拘留营中关押了超过一百万穆斯林的地方。
  </p>
  <p class="graf graf--p">
   新疆似乎是中国当局测试其监视系统（尤其是面部扫描技术）最喜欢针对的地点。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e5%9b%be%e8%a7%a3%e6%96%b0%e7%96%86%e5%86%8d%e6%95%99%e8%82%b2%e8%90%a5/" href="https://www.iyouport.org/%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e5%9b%be%e8%a7%a3%e6%96%b0%e7%96%86%e5%86%8d%e6%95%99%e8%82%b2%e8%90%a5/" rel="noopener noreferrer" target="_blank">

    </a>

  <p class="graf graf--p">
   来自新疆的警务专家 Li Xiaoyu 称，“通过视频录像、情绪识别，可以通过分析犯罪嫌疑人的心理状态来快速识别犯罪嫌疑人，以防止包括恐怖主义和走私在内的非法行为。”
  </p>
  <p class="graf graf--p">
   但是，你知道，当局会视什么为 “非法” 。
  </p>
  <p class="graf graf--p">
   安全专家对技术准确检测人类情绪的能力表示怀疑。
  </p>
  <p class="graf graf--p">
   ACLU 高级政策分析师 Jay Stanley 说：“关于情感识别的科学是很虚假的”，并且一些研究表明，将情感状态与面部表情正确地联系起来是困难的，即使不是不可能的话。
  </p>
  <p class="graf graf--p">
   好消息是，特别是对中国公民而言，新的监视技术尚未准备好进行大规模生产，被称为 “噱头”。
  </p>
  <p class="graf graf--p">
   像微软、谷歌和亚马逊这样的公司都在开发自己的面部识别系统。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%9a%e9%a9%ac%e9%80%8a%e5%b8%ae%e5%8a%a9%e8%ad%a6%e6%96%b9%e6%9e%84%e5%bb%ba%e4%ba%86%e4%b8%80%e4%b8%aa%e5%b9%bf%e6%b3%9b%e7%9a%84%e7%9b%91%e6%8e%a7%e7%bd%91%e7%bb%9c/" href="https://www.iyouport.org/%e4%ba%9a%e9%a9%ac%e9%80%8a%e5%b8%ae%e5%8a%a9%e8%ad%a6%e6%96%b9%e6%9e%84%e5%bb%ba%e4%ba%86%e4%b8%80%e4%b8%aa%e5%b9%bf%e6%b3%9b%e7%9a%84%e7%9b%91%e6%8e%a7%e7%bd%91%e7%bb%9c/" rel="noopener noreferrer" target="_blank">

    </a>
    》
   </li>

   这不是中国首次采取有争议的行动来加强对公民的监视。
  </p>
  <p class="graf graf--p">
   该国还一直在测试步态识别功能，甚至还建立了社会信用监视体系和一个可绘制“债务人”地图的应用程序，因此那些人可以被其他人公开羞辱。
  </p>
  <p class="graf graf--p">
   当今全球的监视水平都在极速上升，这是急需警惕的问题。在下面看到详细报告：
  </p>
