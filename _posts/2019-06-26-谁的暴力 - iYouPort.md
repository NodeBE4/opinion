      <em>
       国家政权和主流媒体联手栽赃抗议者，将受害者说成是罪犯、将罪犯说成受害者。这是一种值得警惕的秘密行动，它是什么样的？
      </em>

    </span>
   </li>

  <p class="graf graf--p">
   暴力是统治阶级用来诋毁来自下方的运动并为其镇压辩护的一种壮观的武器。暴力是强有力的政治工具，可以治理群众，并保持群众与统治阶层的地位关系不变。
  </p>
  <p class="graf graf--p">
   然而，为了做到这一点，暴力武器在第二种意义上也是壮观的：
   <strong class="markup--strong markup--p-strong">
    它创造了一个精心策划的混战，旨在隐藏统治阶级本身的暴力，同时将抵抗行为转化为犯罪
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这就是大众媒体正在对黄背心所做的事，它们称“黄背心发动了过多的暴力事件，现在需要以最强烈的措辞予以压制。”
  </p>
  <p class="graf graf--p">
   香榭丽舍大街委员会主席 Jean-NoëlReinhardt 被许多主要新闻媒体的麦克风所包围，他在接受采访时
   <a class="markup--anchor markup--p-anchor" data-href="https://www.youtube.com/watch?v=WEg5untzHoI" href="https://www.youtube.com/watch?v=WEg5untzHoI" rel="noopener noreferrer" target="_blank">
    宣称
   </a>
   ，该运动“不再是黄背心”，而是黑背心，只是“表达仇恨和摧毁意志”。并宣称由于其对商业和旅游活动的“影响”，以及对香榭丽舍大街全球象征的“诽谤”，这种情况不能继续下去；他的声明无形地融入了总理 ÉdouardPhilippe
   <a class="markup--anchor markup--p-anchor" data-href="https://www.francetvinfo.fr/economie/transports/gilets-jaunes/video-nouveau-prefet-de-police-interdictions-de-manifester-les-cinq-annonces-d-edouard-philippe-apres-les-violences-sur-les-champs-elysees_3239177.html" href="https://www.francetvinfo.fr/economie/transports/gilets-jaunes/video-nouveau-prefet-de-police-interdictions-de-manifester-les-cinq-annonces-d-edouard-philippe-apres-les-violences-sur-les-champs-elysees_3239177.html" rel="noopener noreferrer" target="_blank">
    的声明
   </a>
   中，这家伙称：将采取新措施禁止某些地方的抗议活动，并
   <strong class="markup--strong markup--p-strong">
    允许更严厉的警察镇压
   </strong>
   。
  </p>
  <p class="graf graf--p">
   值得注意的是，Unsa 警察工会的 Philippe Capon 公开表示，警察在星期六接到的命令是不介入，因为有一个明确的选择是“让一些事变得更糟”。时机不可能更好了，因为政府在插手。黄背心一直没有撤退，并且在整个冬天和对他们发动的极端形式的镇压性暴力中幸存了下来。
  </p>
  <p class="graf graf--p">
   因此，目前的暴力场景有两个目的。
   <strong class="markup--strong markup--p-strong">
    首先，它掩盖了资本主义和富豪寡头政治的结构性暴力，这正是当前起义的主要根源
   </strong>
   。群众的生活条件越来越不可接受，传统的政党政治和工会制度功能失调。这种强烈的“暴力”正是为了分散资本主义统治下的常规破坏性，而不是认真对待这种国家暴力的无处不在和日常化，即 资本主义不平等的暴力。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当权者指责焚烧银行的“暴力”，而不是建立银行的“暴力”，或更普遍的是银行系统在确保全球统治阶级霸权的日常角色中的暴力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   由国家和大众媒体精心策划的暴力奇观，其目的是将暴力符号强行附加到黄背心运动中，以便同时将其定罪并为当权者自己的残酷镇压做辩护。
  </p>
  <p class="graf graf--p">
   在许多案件中，
   <strong class="markup--strong markup--p-strong">
    警察已经被通过录像记录下破坏财产并将其归咎于抗议者的证据
   </strong>
   ，并且许多警官携带锤子的画面已经被拍照和录像，大概就是为此目的。防暴警察中
   <a class="markup--anchor markup--p-anchor" data-href="https://lemediapourtous.fr/un-crs-temoigne-on-nous-donne-lordre-de-laisser-casser-pour-que-le-mouvement-devienne-impopulaire/?fbclid=IwAR0htiNQDBVnrbDhlSSu0Pxiq0qcRN0ognUfCys692MofByq1THniUo2zk8" href="https://lemediapourtous.fr/un-crs-temoigne-on-nous-donne-lordre-de-laisser-casser-pour-que-le-mouvement-devienne-impopulaire/?fbclid=IwAR0htiNQDBVnrbDhlSSu0Pxiq0qcRN0ognUfCys692MofByq1THniUo2zk8" rel="noopener noreferrer" target="_blank">
    至少有一名成员公开反对
   </a>
   了针对非暴力抗议者的暴力行为，内政部长对此表示支持，并反对在抗议活动中煽动暴力的努力。
  </p>
  <p class="graf graf--p">
   法国的精英圈子在宣传活动方面并没有取得完全的成功，因为即使是像联合国、欧洲理事会、欧洲议会、和大赦国际这样的自由派机构，他们也一定程度上试图让国家暴力变得无形，或者“至少是有道理的”。
  </p>
  <p class="graf graf--p">
   欧洲委员会人权事务专员 DunjaMijatović 于2月26日编写了
   <a class="markup--anchor markup--p-anchor" data-href="https://rm.coe.int/commdh-2019-8-memorandum-france-en/1680932f57" href="https://rm.coe.int/commdh-2019-8-memorandum-france-en/1680932f57" rel="noopener noreferrer" target="_blank">
    一份备忘录
   </a>
   ，总结了一些暴力事件，同时也批评了
   <strong class="markup--strong markup--p-strong">
    国家和主流媒体保存的统计数据缺乏准确性和严谨性
   </strong>
   ：“根据内政部12 122 LBD的数据，在黄背心运动开始至2019年2月4日之间，发射或投掷了1 428枚即时催泪瓦斯手榴弹和4 942枚手持式手榴弹。” 根据报告中引用的一名独立记者的计算，“38人上肢受伤，包括5人失去了手，52人下肢伤，189人头部受伤，其中20人失明“。在其中医务人员和记者也经常遭到袭击，并且发生了无数次野蛮的袭击、并有
   <a class="markup--anchor markup--p-anchor" data-href="https://www.rtl.fr/actu/debats-societe/gilets-jaunes-216-personnes-incarcerees-un-record-pour-un-mouvement-social-7796112085" href="https://www.rtl.fr/actu/debats-societe/gilets-jaunes-216-personnes-incarcerees-un-record-pour-un-mouvement-social-7796112085" rel="noopener noreferrer" target="_blank">
    数量创纪录的抗议者被捕
   </a>
   。
  </p>
  <p class="graf graf--p">
   尽管如此，国家的重要部门、大众媒体和专家们都在竭尽全力掩盖对非暴力抗议者、医务人员、记者和旁观者的系统性伤害。
  </p>
  <p class="graf graf--p">
   马克龙（Emmanuel Macron）通过断然宣称我们今天不能在法国谈论“镇压”或“警察暴力”，从而提炼出关于国家的自由主义意识形态的本质，因为“这些词语在法治下是不可接受的”[dansunétatdedroit ]“。因此，严格来说，不存在“国家暴力”这样的事，因为国家反对暴力，而暴力只能来自“外面的野蛮行为和无政府主义势力”。
  </p>
  <p class="graf graf--p">
   在这里，我们看到壮观的
   <strong class="markup--strong markup--p-strong">
    双重暴力运动
   </strong>
   。一方面，国家努力通过资本主义统治来掩盖其壮观的剥削，并对任何抵抗进行同样壮观的镇压。另一方面，它试图在抗议活动中煽动或制造引人注目的“暴力”，以便同时诋毁抗议者、并利用这一景观作为国家自身增加的剥削和镇压的掩护。这就是法国目前正在发生的壮观暴力的两个主要方面。
  </p>
  <p class="graf graf--p">
   必须确定这种策略是什么，并找到新的战略来对抗其极为恶劣的影响。否则，我们就要冒着被 Malcolm X 在1964年12月13日的演讲中如此有先见之明地指出的意识形态倒置的风险，他在演讲中解释说，
   <strong class="markup--strong markup--p-strong">
    新闻界在其“形象制作角色”中是如此的强大，以至于它可以使罪犯看起来像受害者，让受害者看起来像罪犯
   </strong>
   。“
  </p>
  <p class="graf graf--p">
   这一案例是尤其值得警惕的。它是一种非常狡猾的秘密行动，追求正义的人们需要格外警惕 — — 我们将在明天的知识点栏目中详细介绍关于什么的秘密行动、世上经典的秘密行动案例，并提供揭露秘密行动之方法的相关资料。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    S
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.counterpunch.org/2019/03/22/spectacular-violence-as-a-weapon-of-war-against-the-yellow-vests/" href="https://www.counterpunch.org/2019/03/22/spectacular-violence-as-a-weapon-of-war-against-the-yellow-vests/" rel="noopener noreferrer" target="_blank">
    pectacular Violence as a Weapon of War Against the Yellow Vests
   </a>
   , the press is so powerful in its “image-making role” that “it can make a criminal look like he’s the victim and make the victim look like he’s a criminal.”
  </p>
