   </noscript>
  </p>
  <p class="graf graf--p">
   早上9点过后不久，老人社区护理中心开始热闹起来了。一个男人在唱民歌，一个百岁老人坐在一盘象棋旁，等待对手。
  </p>
  <p class="graf graf--p">
   虚拟现实机器可以让老人体验杂货店购物和乘坐地铁等场景，它被闲置在角落里。相比下晨练是更具吸引力的活动，几十个人一起拉伸关节……
  </p>
  <p class="graf graf--p">
   这里是中国快速老龄化的前沿，这一趋势已经开始限制该国的经济潜力。
  </p>
  <p class="graf graf--p">
   自该护理中心于半年前在上海市中心长宁开业以来，已经迎来了超过12,000名老人。这座城市于2014年设立了这些养老中心，结合了卫生所、临时医疗设施和老人的住所。
  </p>
  <p class="graf graf--p">
   这些养老设施计划到2022年增加到400个。“我们等不及了”，社区组织者彭艳丽[音]说：“我们必须尽一切努力来构建这些[养老院]。”
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10247 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-1.png?resize=1100%2C1198&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-1.png?w=1200&amp;ssl=1 1200w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-1.png?resize=275%2C300&amp;ssl=1 275w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-1.png?resize=768%2C836&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-1.png?resize=940%2C1024&amp;ssl=1 940w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-1.png?resize=1100%2C1198&amp;ssl=1 1100w" data-recalc-dims="1" height="1198" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-1.png?resize=1100%2C1198&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10247" data-recalc-dims="1" height="1198" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-1.png?resize=1100%2C1198&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-1.png?w=1200&amp;ssl=1 1200w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-1.png?resize=275%2C300&amp;ssl=1 275w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-1.png?resize=768%2C836&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-1.png?resize=940%2C1024&amp;ssl=1 940w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-1.png?resize=1100%2C1198&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   老龄化对中国的压力越来越大。今年将是一个不祥的里程碑。
  </p>
  <p class="graf graf--p">
   根据联合国预测，到2020年，中国公民的年龄中位数将超过美国人。然而，中国仍然贫穷得多，其中位收入仅为美国的四分之一。
  </p>
  <p class="graf graf--p">
   人们一直在广泛讨论的担心 ——
   <strong class="markup--strong markup--p-strong">
    中国会在变得富裕之前就老掉了
   </strong>
   这个问题 —— 已经不再是理论上的可能性，而是正在成为的现实。
  </p>
  <p class="graf graf--p">
   根据联合国的预测，在未来25年内，中国65岁以上人口所占的百分比将从12％增至25％，翻一番以上。
  </p>
  <p class="graf graf--p">
   相比之下，美国要达到同样的水平需要一个世纪，欧洲需要60多年。
  </p>
  <p class="graf graf--p">
   中国的步伐与日本的步伐相似，但比韩国的步伐稍慢，这两个国家也是迅速老龄化的代表，但它们的人均财富大约是中国的三倍。
  </p>
  <p class="graf graf--p">
   1960年出生的中国人的预期寿命为44岁，比同年出生的加纳人的预期寿命短。当前出生的中国婴儿的预期寿命为76岁，仅次于美国。但这也是中国臭名昭著的人口控制策略的结果。
  </p>
  <p class="graf graf--p">
   1973年，当政府开始限制生育之前，中国妇女平均每人生育4.6个孩子。今天，她们平均只有1.6个孩子，一些学者说，即使这个估计也是过高了。
  </p>
  <p class="graf graf--p">
   随着中国变得更加富裕，生育率必然会下降，但一胎化政策造成的生育率下降之幅度更大。
  </p>
  <p class="graf graf--p">
   尽管该国在2016年开始实行了二胎政策，可能很快就会完全取消限制，
   <strong class="markup--strong markup--p-strong">
    但这一放松可能为时已晚。
   </strong>
  </p>
  <p class="graf graf--p">
   从2012年就开始减少的劳动力人口，将在未来几十年内继续下降。到本世纪中叶，劳动力人口将比现在缩小近五分之一。
  </p>
  <p class="graf graf--p">
   对经济的影响主要从两方面体验到。最明显的是需要照顾所有的老人。2014年，退休人员的养老金支出已经超过了在职人员的工资。根据中国社会科学院的数据，到2035年，国家养老基金可能会用光所有钱。
  </p>
  <p class="graf graf--p">
   财政部正在采取一些小的步骤来加强这一体系：9月，他们将其在四家大型国有金融机构中的10％的股权转给了养老基金。但是仍然不够，还需要更多。
  </p>
  <p class="graf graf--p">
   政府在养老金和医疗保健方面的支出约为GDP的十分之一，仅是较富裕的老龄化国家通常水平的一半，而这些国家随着国民年龄的增长，将不得不增加支出。
  </p>
  <p class="graf graf--p">
   第二个影响是增长。一些中国经济学家，尤其是北京大学的林毅夫认为，老龄化并不一定会使中国的经济放慢脚步，这在一定程度上要归功于技术的进步。
  </p>
  <p class="graf graf--p">
   但迄今为止，由中国社会科学院蔡昉领导的另一个阵营赢得了这场争论。
  </p>
  <p class="graf graf--p">
   不断萎缩的劳动力正在推高工资，并且随着公司在技术上投入更多资金以替代工人，从而降低了资本投资的回报。
  </p>
  <p class="graf graf--p">
   蔡昉计算得出的结果是，中国的潜在增长率已降至约6.2％，几乎与今天的水平完全一样。劳动力短缺不仅打击了公司，而且打击了所有城市。
  </p>
  <p class="graf graf--p">
   从北部的西安到南部的深圳，直辖市使大学毕业生的搬迁更加容易，希望借此吸引技术娴熟的年轻工人。
  </p>
  <p class="graf graf--p">
   从理论上讲，中国可以通过增加劳动力的参与度和生产率来缓解老龄化带来的不利影响，也就是说，可以让更多的人上班，从他们身上得到更多。但是都不容易。
  </p>
  <p class="graf graf--p">
   中国的退休年龄其实非常低（在许多工作中，男性为60岁，女性为50岁），但政府担心会出现反弹，因此拒绝提高退休年龄。
  </p>
  <p class="graf graf--p">
   另方面习近平领导下恢复国家主导的增长似乎正在损害生产率。正如经济学家乔治·马格努斯（George Magnus）在《警告：为什么习近平的中国处于危险之中》一书中所写到的，人口问题不是命运，中国有时间改变路线；“但是，坏消息是可用的时间正在迅速流逝。”
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10248 jetpack-lazy-image" data-lazy-sizes="(max-width: 317px) 100vw, 317px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2.jpg?resize=317%2C475&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2.jpg?w=317&amp;ssl=1 317w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2.jpg?resize=200%2C300&amp;ssl=1 200w" data-recalc-dims="1" height="475" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2.jpg?resize=317%2C475&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="317"/>

  </p>

  <p class="graf graf--p">
   一个好消息是，中国正在创造性地考虑如何照顾日益增长的养老金领取者。
  </p>
  <p class="graf graf--p">
   传统上，期望孩子们照顾年迈的父母，这有助于解释为什么对养老院的公共投资很少。但是现在大多数家庭只有一个孩子，而那个孩子还需要工作。
  </p>
  <p class="graf graf--p">
   苏州是上海附近的一个富裕城市，向人们展示了中国如何利用其规模优势。2007年，企业家 Lu Zhong 创立了居家乐养老服务中心（Jujiale），将其称为“虚拟退休之家”，按需向私人住宅派遣佣工。
  </p>
  <p class="graf graf--p">
   现在，它有1,800名员工，为130,000名退休人员服务。Lu 说，为了跟上需求，它每年需要增长约15％。
  </p>
  <p class="graf graf--p">
   但这只是一大片乌云周边的一丝银线而已。
  </p>
  <p class="graf graf--p">
   去年10月1日，北京庆祝中华人民共和国成立70周年。习近平发誓，到2049年的百年纪念日时，中国将发展到足以让全世界看到的实力。
  </p>
  <p class="graf graf--p">
   但正如著名经济学家任泽平在最近的一份报告中指出的那样，到2050年，中国的年龄中位数将接近50岁，而美国为42岁，印度为38岁。他写道，这提出了一个问题：“我们可以依靠这种人口结构实现民族伟大复兴吗？” ⚪️
  </p>
