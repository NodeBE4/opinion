  <p class="graf graf--p">
   我们分析过《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%96%e4%bb%ac%e6%98%af%e5%a6%82%e4%bd%95%e6%8a%93%e6%8d%95%e8%af%b4%e7%9c%9f%e8%af%9d%e7%9a%84%e4%ba%ba%ef%bc%9f%e2%9a%a0%ef%b8%8f%e7%81%be%e9%9a%be%e5%92%8c%e6%95%99%e8%ae%ad/" href="https://www.iyouport.org/%e4%bb%96%e4%bb%ac%e6%98%af%e5%a6%82%e4%bd%95%e6%8a%93%e6%8d%95%e8%af%b4%e7%9c%9f%e8%af%9d%e7%9a%84%e4%ba%ba%ef%bc%9f%e2%9a%a0%ef%b8%8f%e7%81%be%e9%9a%be%e5%92%8c%e6%95%99%e8%ae%ad/" rel="noopener noreferrer" target="_blank">
    当局抓人的诸多手段
   </a>
   》 ——
   <strong class="markup--strong markup--p-strong">
    任何政府间谍都可以使用这些手段，因为这些手段所基于的数字技术是全球共享的
   </strong>
   。不同的仅仅是有些当权者拥有额外的便利。这其中最大便利的拥有者是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e9%ab%98%e5%a0%a1%e5%a5%87%e4%ba%ba/" href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e9%ab%98%e5%a0%a1%e5%a5%87%e4%ba%ba/" rel="noopener noreferrer" target="_blank">
    美国和中国
   </a>
   。
  </p>
  <p class="graf graf--p">
   所有这些都是安全防御失败的教训。血淋淋的教训 —— 已经有人因说出真相而面临10年的监禁。
  </p>
  <p class="graf graf--p">
   判刑之所以会发生是因为当局拿到了“证据”。反之，只要行动者不让对手轻易拿到证据，就是保证安全的基本方法。
  </p>
  <p class="graf graf--p">
   上面那篇文章很重要。
  </p>
  <p class="graf graf--p">
   但本文即将解释的是另一种很基本的威胁 —— 每个人都有可能面对的危险。尤其是在中国，因为
   <strong class="markup--strong markup--p-strong">
    据称中国当局甚至可以“随意检查手机”，甚至没收你的电子设备。
   </strong>
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-9446 jetpack-lazy-image" data-lazy-sizes="(max-width: 614px) 100vw, 614px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/查手机.png?resize=614%2C626&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/查手机.png?w=614&amp;ssl=1 614w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/查手机.png?resize=294%2C300&amp;ssl=1 294w" data-recalc-dims="1" height="626" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/查手机.png?resize=614%2C626&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="614"/>

  <p class="graf graf--p">
   甚至，⚠️
   <strong class="markup--strong markup--p-strong">
    根本不需要没收你的设备 —— 不需要物理接触你的设备 —— 也能实现同样目的的攻击
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%96%af%e8%af%ba%e7%99%bb%ef%bc%9a%e6%99%ba%e8%83%bd%e6%89%8b%e6%9c%ba%e6%9c%ac%e8%ba%ab%e5%b0%b1%e6%98%af%e4%b8%8d%e5%ae%89%e5%85%a8%e7%9a%84%e4%b8%9c%e8%a5%bf%ef%bc%88video%ef%bc%89/" href="https://www.iyouport.org/%e6%96%af%e8%af%ba%e7%99%bb%ef%bc%9a%e6%99%ba%e8%83%bd%e6%89%8b%e6%9c%ba%e6%9c%ac%e8%ba%ab%e5%b0%b1%e6%98%af%e4%b8%8d%e5%ae%89%e5%85%a8%e7%9a%84%e4%b8%9c%e8%a5%bf%ef%bc%88video%ef%bc%89/" rel="noopener noreferrer" target="_blank">
    手机是非常不安全的东西
   </a>
   ，现在您知道了，不仅仅是因为智能手机本身就是监视器；并且，手机号码本身和其作为身份验证渠道的使用，令其增加了很多不同的危险性。
  </p>
  <p class="graf graf--p">
   不幸的是，如今这个世界上的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.independent.co.uk/life-style/gadgets-and-tech/news/there-are-officially-more-mobile-devices-than-people-in-the-world-9780518.html" href="https://www.independent.co.uk/life-style/gadgets-and-tech/news/there-are-officially-more-mobile-devices-than-people-in-the-world-9780518.html" rel="noopener noreferrer" target="_blank">
    智能手机比活人还多
   </a>
   。
  </p>
  <p class="graf graf--p">
   这些被随身携带的监视器中包含大量的个人信息、私人文件和敏感数据。
  </p>
  <p class="graf graf--p">
   如今的人们出门完全可以不带钱包（他们乐于使用在线支付），甚至不带钥匙（指纹锁的流行），但是很难不带手机，无法不令这个监视器随时随地的跟着你。想想看，这意味着什么？…… 太多危险了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    新闻故事会误导你
   </strong>
   。就如苹果与FBI案等新闻有助于传播这样的*
   <strong class="markup--strong markup--p-strong">
    误解*
   </strong>
   ：似乎移动设备被PIN或生物识别技术锁定、加密和保护，那么它就是“安全的”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这是十足的误解
   </strong>
   。
  </p>
  <p class="graf graf--p">
   事实是，⚠️包括iOS和Android在内的主要操作系统都在帮助您、甚至鼓励您通过某些功能和不安全的设置来降低锁定设备的安全性。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    再次强调：上述对安全的误解似乎很深远
   </strong>
   ：在香港的抗议活动中，也有英文媒体赞赏过活动家“迅速锁定屏幕”（禁用人脸识别功能）的安全措施，在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%a6%99%e6%b8%af%e7%9a%84%e6%8a%97%e8%ae%ae%e6%b4%bb%e5%8a%a8%e4%b8%ad%e5%86%8d%e6%ac%a1%e5%bc%ba%e8%b0%83%e4%ba%86%e4%bd%a0%e7%9a%84%e8%84%b8%e5%a6%82%e4%bd%95%e6%88%90%e4%b8%ba%e6%ad%a6%e5%99%a8/" href="https://www.iyouport.org/%e9%a6%99%e6%b8%af%e7%9a%84%e6%8a%97%e8%ae%ae%e6%b4%bb%e5%8a%a8%e4%b8%ad%e5%86%8d%e6%ac%a1%e5%bc%ba%e8%b0%83%e4%ba%86%e4%bd%a0%e7%9a%84%e8%84%b8%e5%a6%82%e4%bd%95%e6%88%90%e4%b8%ba%e6%ad%a6%e5%99%a8/" rel="noopener noreferrer" target="_blank">
    香港的抗议活动中再次强调了你的脸如何成为武器
   </a>
   》。
  </p>
  <p class="graf graf--p">
   ⚠️但是，必须知道，仅仅如此锁屏并不能真正保证安全。
  </p>
  <p class="graf graf--p">
   并不是说锁屏不对，绝不是，而是说，与此同时您还应该与其他安全措施协调使用，才能真正起到保护作用（本文后面将有具体解释）。
  </p>
  <p class="graf graf--p">
   移动设备上的所谓私人助理非常受欢迎。 Siri、OK Google、和 Cortana 这些 只是其中的几个；它们可以执行多项任务，包括通话、发送电子邮件、和阅读SMS等诸多**敏感动作**。
  </p>
  <p class="graf graf--p">
   ⚠️你有没想过它们的风险都包括什么？你是否可以信任它们？
  </p>
  <p class="graf graf--p">
   本文将带您研究锁定状态下移动设备的安全性 —— 您可以从中发现问题，并理解到如何保护自己。
  </p>
  <p class="graf graf--p">
   警惕媒体对指纹/面部解锁合宪性问题的炒作，因为这一议题本身有可能在安全角度上转移注意力。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    短信是不安全的（众所周知）
   </strong>
  </p>
  <p class="graf graf--p">
   多年来，安全社区一直在谈论SMS的不安全性以及为什么不应该将SMS用作出于安全目的的任何安全通道。
  </p>
  <p class="graf graf--p">
   但是似乎没有多少人在听。
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   在开发该工具时，我开始研究诸如 Paypal 和 Facebook 之类的流行服务的 API，以了解密码重置流程，并实施该密码重置流程，以便该工具可以自动执行此操作。
  </p>
  <p class="graf graf--p">
   很快就意识到这种方法存在一些问题。
  </p>
  <p class="graf graf--p">
   这需要时间，这很乏味，我发现我经常遇到验证码的挑战，而不是在工具重置密码时遇到挑战。另外，我的最终目标是编写一个可以轻松扩展以支持更多在线服务的工具，该工具允许通过SMS重置密码。
  </p>
  <p class="graf graf--p">
   这就是为什么我想到使用
   <a class="markup--anchor markup--p-anchor" data-href="https://www.seleniumhq.org/" href="https://www.seleniumhq.org/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Selenium
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p">
   Selenium 是一个库，可让您以编程方式在UI级别与浏览器进行交互。开发人员可以使用它来测试图形用户界面。
  </p>
  <p class="graf graf--p">
   它基本上模拟了人类的单击按钮、填写表格并通常在站点中导航。
  </p>
  <p class="graf graf--p">
   事实证明，Selenium 是该问题的完美解决方案：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     简便
    </strong>
    ：Selenium 的API非常简单明了，就像CSS那样；
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     速度快
    </strong>
    ：
    <a class="markup--anchor markup--li-anchor" data-href="https://www.seleniumhq.org/projects/ide/" href="https://www.seleniumhq.org/projects/ide/" rel="noopener noreferrer" target="_blank">
     有些浏览器扩展程序
    </a>
    可以记录您与网站的互动并自动生成 Selenium 码；
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     方便
    </strong>
    ：我发现使用 Selenium 来自动执行密码重置过程时，对验证码的挑战并不多，因为网站无法说出这不是人工点击。
   </li>
  </ul>
  <p class="graf graf--p">
