  </p>
  <p class="graf graf--p">
   这就是为什么我们说，开源情报不是实现变革的终极解决方案，而是其中一个重要的步骤 —— 关于信息民主化的步骤。
  </p>
  <p class="graf graf--p">
   随着全面数字化的进程接近完成，开源情报的能力将更加明显。

   <strong class="markup--strong markup--p-strong">
    谷歌与公共机构（特别是欧洲政府机构）之间的旋转门。
   </strong>

  <p class="graf graf--p">
   谷歌自己承认，该公司 “与许多其他公司一样”，与政府机构和政府官员建立了密切的关系。
  </p>
  <p class="graf graf--p">
   谷歌透露，2015年它在游说欧盟方面花费了超过400万欧元 —— 远远超过了2012年（也就是仅三年前）在游说方面花费的100万欧元。
  </p>
  <p class="graf graf--p">
   但是，谷歌与公共机构和政府官员之间的关系更为微妙：在过去十年中，已经确定至少有80人在Google和欧洲政府之间转移了工作。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    正是这种“旋转门”构成了这项调查的基础
   </strong>
   。
  </p>
  <p class="graf graf--p">
   首先提出几个问题：哪些人从Google跑到政府去工作，反之亦然？他们确切是从哪里转移的？何时转移的？最重要的是，可以使用公开信息源找到这些问题中的哪些答案？
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     查询 Google 的旋转门
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   谷歌的
   <a class="markup--anchor markup--p-anchor" data-href="http://googletransparencyproject.org/articles/googles-european-revolving-door" href="http://googletransparencyproject.org/articles/googles-european-revolving-door" rel="noopener noreferrer" target="_blank">
    透明度报告
   </a>
   项目记录了政府组织与谷歌之间的人员往来，即 所谓的 “旋转门’’，从而跟踪了人员从政府到谷歌的就业过渡，反之亦然。
  </p>
  <p class="graf graf--p">
   在2016年6月的报告显示，过去十年来至少有80个人在 Google 和欧洲政府之间转移了工作。
  </p>
  <p class="graf graf--p">
   当然，
   <strong class="markup--strong markup--p-strong">
    这不仅是欧洲的现象，而且 Google 透明度报告还发布了有关在美国发生的政治旋转门数据
   </strong>
   。
  </p>
  <p class="graf graf--p">
   将 Google 透明度报告发布的数据集用作起点，来
   <strong class="markup--strong markup--p-strong">
    扩展
   </strong>
   收集的证据。再将此数据集与从 LinkedIn 和 Twitter 等网络平台收集的信息、搜索引擎搜索请求、以及从在线新闻站点收集的数据结合在一起。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.transparency.org/glossary/term/revolving_door" href="https://www.transparency.org/glossary/term/revolving_door" rel="noopener noreferrer" target="_blank">
    Transparency International
   </a>
   将 “旋转门” 定义为 “个人在公职和私营部门或志愿部门中以同一职位转移的现象。如果监管不当，可能会遭到滥用”。
  </p>
  <p class="graf graf--p">
   这里的滥用主要是指利益冲突。如谷歌透明度报告
   <a class="markup--anchor markup--p-anchor" data-href="http://googletransparencyproject.org/about-us" href="http://googletransparencyproject.org/about-us" rel="noopener noreferrer" target="_blank">
    所述
   </a>
   ：“谷歌一直是政府、美国企业（和欧洲）社会中透明度和开放性的最积极倡导者之一，但并没有使其自身达到相同的透明度水平。
   <strong class="markup--strong markup--p-strong">
    尽管要求别人开放，但该公司对其自身的运营和与政府的往来却非常不透明
   </strong>
   。从与当选官员和任命官员的关系到政治游说和公共政策运作，美国人（和欧洲人）对 Google 如何从政府那里获得想要的东西知之甚少。”
  </p>

     LinkedIn调查
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   由于 Google Transparency Project 已经从 LinkedIn 上
   <strong class="markup--strong markup--p-strong">

   信息，因此需要专注于数据集中未包含的数据，例如名单上的人就读的大学以及他们添加到LinkedIn个人资料中的主要技能和专业知识。
  </p>
  <p class="graf graf--p">
   从 LinkedIn 收集信息被证明很困难，因为需要一定数量的同意才能访问该信息。但
   <strong class="markup--strong markup--p-strong">
    确实搜集到的 LinkedIn 数据质量相对较好
   </strong>
   ，因为它与欧盟官方 *游说者数据库* 中的记录基本吻合。
  </p>
  <p class="graf graf--p">
   通过将注意力集中在英国得以在这一探索上更多的发展，因为 LinkedIn 倾向于在英美语境中有更广泛的使用。此外，在技术公司工作的人往往在 LinkedIn 等平台上有很好的代表性。
  </p>
  <p class="graf graf--p">
   在此数据集中确实找到了一些可以通过对报纸文章进行定性研究验证为有政治影响力的人。但是，
   <strong class="markup--strong markup--p-strong">
    鉴于他们的职位，他们中的许多人并不像人们期望的那样活跃在网络平台上 —— 本调查发现许多高层员工很难被追查
   </strong>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     连接关系
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   这里想要绘制的是个人以前曾工作或目前正在工作的组织。希望能提供一些洞察力，了解到那些通过旋转门的人通过以前的工作场所彼此认识，以及那些使用旋转门的人是否在特定的机构或公司中被过分突出。
  </p>
  <p class="graf graf--p">
   通过绘制这些数据，可以在可视化结果中清楚地看到英国的统治地位。工党、保守党、唐宁街10号和下议院显然是主要的共享工作场所。
  </p>
  <p class="graf graf--p">
   该数据集中包含的 Google 员工工作的两个最常见的地方是在欧洲议会或欧盟委员会。共同工作场所列表中提到的其他公司是 Uber 和 Cisco Systems。

     Twitter
    </strong>
   </span>

  <p class="graf graf--p">

   </strong>

   </strong>

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     影响力网络
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   最重要的问题是：Google 对政府的影响力如何在网上关系中体现出来？这些在线联系在多大程度上可以推断出与现实世界的联系。
  </p>
  <p class="graf graf--p">
   通过查看社交网站以尝试确定可用于判断离线关系的在线世界中的指标。
  </p>
  <p class="graf graf--p">
   采用多种方法来识别连接，而不是采用适合所有技术的单一方法。由于数据集很小，于是限制了分析的范围，这使得采用这些技术更容易管理；但的确发现存在许多限制和约束。
  </p>
  <p class="graf graf--p">
   具有真正权力和影响力的人难以通过在线追踪和收集信息，因为他们的在线个人资料比那些短暂任职的Google官员和政府官员更受限制。
  </p>
  <p class="graf graf--p">
   为了找到有关这些人的信息，我们发现
   <strong class="markup--strong markup--p-strong">
    媒体档案库比在社交网络平台上搜索更有用
   </strong>
   —— 查找讨论其技能、兴趣和影响力的媒体提及。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     最后
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   👉这项简短的调查着重于信息收集的方法以及潜在的分析结果。
  </p>
  <p class="graf graf--p">
   类似于
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%83%b3%e7%9f%a5%e9%81%93%e8%80%81%e5%a4%a7%e5%93%a5%e6%98%af%e5%a6%82%e4%bd%95%e4%be%b5%e7%8a%af%e4%bd%a0%e7%9a%84%e5%90%97%ef%bc%9f%e6%8c%96%e6%8e%98%e6%9b%b4%e6%b7%b1%e5%b1%82%e7%9a%84%e5%86%85/" href="https://www.iyouport.org/%e6%83%b3%e7%9f%a5%e9%81%93%e8%80%81%e5%a4%a7%e5%93%a5%e6%98%af%e5%a6%82%e4%bd%95%e4%be%b5%e7%8a%af%e4%bd%a0%e7%9a%84%e5%90%97%ef%bc%9f%e6%8c%96%e6%8e%98%e6%9b%b4%e6%b7%b1%e5%b1%82%e7%9a%84%e5%86%85/" rel="noopener noreferrer" target="_blank">
    对 Hacking Team 的调查
   </a>
   ，主题不是数据集中最重要的元素，而是对它们的部署的分析，然后可以将其复制到许多主题和数据集上。
  </p>
  <p class="graf graf--p">
   这项调查的重点是查看可以在网上找到哪些信息，这些信息是否可以扩展或提供与已经获得的数据不同的叙述。
  </p>
  <p class="graf graf--p">
   尚未讨论使用 LinkedIn 作为数据源的伦理问题（可以单独分析）。这是一个社交网站，对于那些将其用于预定目的的人来说它应该是私有的，但实际上，包含在其中的所有数据对于所有想要查找的人来说都是公开的。⚪️
  </p>
