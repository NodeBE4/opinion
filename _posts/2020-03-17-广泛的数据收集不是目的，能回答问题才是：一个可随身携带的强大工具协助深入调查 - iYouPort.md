   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     你需要些什么
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   试用 Buscador 很容易。您需要从 IntelTechniques 网站
   <a class="markup--anchor markup--p-anchor" data-href="https://inteltechniques.com/buscador/" href="https://inteltechniques.com/buscador/" rel="noopener noreferrer" target="_blank">
    下载最新版本的 Buscador
   </a>
   。
  </p>
  <p class="graf graf--p">

   <a class="markup--anchor markup--p-anchor" data-href="https://www.virtualbox.org/wiki/Downloads" href="https://www.virtualbox.org/wiki/Downloads" rel="noopener noreferrer" target="_blank">
    下载 VirtualBox
   </a>

   <a class="markup--anchor markup--p-anchor" data-href="https://www.virtualbox.org/wiki/Downloads" href="https://www.virtualbox.org/wiki/Downloads" rel="noopener noreferrer" target="_blank">

   </a>
   才能运行该虚拟机。
   <strong class="markup--strong markup--p-strong">
    确保完全安装这两者
   </strong>

   对于电子邮件抓取，Buscador 随附了
   <strong class="markup--strong markup--p-strong">
    TheHarvester
   </strong>
   ，可让您搜索感兴趣的域上的所有电子邮件地址。
  </p>
  <p class="graf graf--p">
   还有强大的跨平台开源情报工具
   <strong class="markup--strong markup--p-strong">
    Spiderfoot
   </strong>
   。 Spiderfoot 会自动收集有关目标的信息，并将结果转换为易于理解的报告。在下面看到使用方法：

    Recon-ng
   </strong>

  </p>
  <p class="graf graf--p">
   有许多有用的自定义脚本可用于从目标社交媒体帐户以及其他人们共享自身信息的地方下载视频和图像。这些工具使您只需单击几下就可以获取、处理和分析大量数据。
  </p>
  <p class="graf graf--p">
   我们将在未来逐步介绍这些工具。如果您想了解有关 Mike Bazzell 的开源情报技术的更多信息，可以查看他的书。（我们已经上传到 telegram 频道，您可以在这里下载）
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     Buscador 是非常强大的
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   设置 Buscador 之后，建议您查看本网“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    开源调查
   </a>
   ”栏目中演示和介绍的诸多指南，以开始进行首次调查。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    最重要的是练习使用这些工具回答问题，而不仅仅是寻求搜集信息。
   </strong>
  </p>
  <p class="graf graf--p">
   否则，您很可能迷失在工具可以返回的信息海洋中。
  </p>
  <p class="graf graf--p">
   就像读书一样，想要学到东西，您需要带着问题去读书 —— 带着您的目的 —— 在开源调查中，目的是指导全程的根本。
  </p>
  <p class="graf graf--p">
   如果不能清楚地回答指导研究者对情况的理解的目标问题，那么所有这些数据几乎没有任何价值。
  </p>
  <p class="graf graf--p">
   正因此，这个专门的VM的目标正是将研究人员需要的所有工具和数据整合在一起，以决定他们要提出哪些问题来为其提供支持。
  </p>
  <p class="graf graf--p">
   如果您需要一个功能强大且方便的系统来进行Google搜索范围之外的专业级调查，Buscador 就是您所需的答案。
