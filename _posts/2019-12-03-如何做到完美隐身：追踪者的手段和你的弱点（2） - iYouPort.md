  <p class="graf graf--p">
   如果您还没有读过第一部分，在这里看到：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%81%9a%e5%88%b0%e5%ae%8c%e7%be%8e%e9%9a%90%e8%ba%ab%ef%bc%9a%e7%9b%b4%e9%9d%a2%e4%bd%a0%e7%9a%84%e5%af%b9%e6%89%8b%e2%80%8a-%e2%80%8a-6%e5%a4%a7%e5%8e%9f%e5%88%99%ef%bc%881/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%81%9a%e5%88%b0%e5%ae%8c%e7%be%8e%e9%9a%90%e8%ba%ab%ef%bc%9a%e7%9b%b4%e9%9d%a2%e4%bd%a0%e7%9a%84%e5%af%b9%e6%89%8b%e2%80%8a-%e2%80%8a-6%e5%a4%a7%e5%8e%9f%e5%88%99%ef%bc%881/" rel="noopener noreferrer" target="_blank">
    如何做到完美隐身：直面你的对手 — 6大原则（1）
   </a>
   》
  </p>
  <p class="graf graf--p">
   也许你很幸运，可以负担得起水电以及生活中的一切便利；还可能更幸运，买得起电脑，可以上网，在 Facebook 和 Twitter 跟一群人扯蛋；还有便捷的在线金融服务、电话公司、电力公司、宽带公司、以及各大公司提供的优越在线客户服务……
  </p>
  <p class="graf graf--p">
   各家公司争先恐后地想要标榜自己有多么地用户友好。全天候的热线电话、无微不至的关怀。你会感叹：世界真美好啊。
  </p>
  <p class="graf graf--p">
   抱歉，忘了说了：⚠️以上是追踪者的感叹。
  </p>
  <p class="graf graf--p">
   这里是“完美隐私”的第二部分内容，本部分的主题就是：⚠️
   <strong class="markup--strong markup--p-strong">
    客户友好型公司不是你的好友，而是追踪者的好友。
   </strong>
  </p>
  <p class="graf graf--p">
   联邦调查局第一任局长 Edgar Hoover 在历史上已经树立了一个坚固的反乌托邦标志 —— Hoover 先生因骚扰侵犯政治异议人士和活动家、搜集政治领袖的秘密档案、以及采取各种非法手段收集证据，而闻名于世界。
  </p>
  <p class="graf graf--p">
   自从那个年代开始，侵犯个人隐私的罪魁祸首就是群众斗群众 ——
   <strong class="markup--strong markup--p-strong">
    互联网时代将群众斗群众自动化了，并且笼络了跨地区、甚至跨国的群众，任何人都能卖了你
   </strong>
   。
  </p>
  <p class="graf graf--p">
   所以，如果你想要隐身，首先要做到的就是：
  </p>
  <p class="graf graf--p">
   👉1、对于任何一家想让你的生活变得更加便利的公司，都应该敬而远之。
  </p>
  <p class="graf graf--p">
   现如今每一家大公司基本都是如此，网站则更甚。
  </p>
  <p class="graf graf--p">
   当一个中年人想要找到小学同学可以易如反掌的时候，隐身这件事可就不那么简单了。
  </p>
  <p class="graf graf--p">
   当然，前提是你要把电子邮件、地址、电话号码、个人照片、家庭照片……统统上传到网站，供全世界的人观看。
   <strong class="markup--strong markup--p-strong">
    对这些东西感兴趣的人中99%都是社交工程师
   </strong>
   ，如何足够幸运，剩下的1%是你希望联系的人。
  </p>
  <p class="graf graf--p">
   在线电话薄、博客、互动式网站、病毒式营销广告……也都是如此。尤其应该关注的还包括：大型国有水电企业、电话公司、互联网公司，等等，专业的追踪者经常通过这些地方获取重要信息：随便找个借口就可以撬开那些接线员的嘴。
  </p>
  <p class="graf graf--p">
   客户越是容易接入在线人工服务，追踪者就越容易对其造成真正的伤害。
  </p>
  <p class="graf graf--p">
   ⚠️如果任何一家公司了解你的真身，你都应该非常小心。
  </p>
  <p class="graf graf--p">
   我们介绍过如何将真实身份隐匿在单独的角色中，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%88%86%e8%a3%82%e4%ba%ba%e6%a0%bc%e7%9a%84%e9%ad%85%e5%8a%9b/" href="https://www.iyouport.org/%e5%88%86%e8%a3%82%e4%ba%ba%e6%a0%bc%e7%9a%84%e9%ad%85%e5%8a%9b/" rel="noopener noreferrer" target="_blank">
    具体做法在这里
   </a>
   。
  </p>
  <p class="graf graf--p">
   如果您觉得这个方法不适合您，同时您还希望隐身，那么就要
   <strong class="markup--strong markup--p-strong">
    避免使用任何上述这些服务
   </strong>
   ，彻底远离互联网。
  </p>
  <p class="graf graf--p">
   👉2、社交网络、搜索引擎、病毒式媒体，都是个人隐私的最大敌人。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%81%9a%e5%88%b0%e5%ae%8c%e7%be%8e%e9%9a%90%e8%ba%ab%ef%bc%9a%e7%9b%b4%e9%9d%a2%e4%bd%a0%e7%9a%84%e5%af%b9%e6%89%8b%e2%80%8a-%e2%80%8a-6%e5%a4%a7%e5%8e%9f%e5%88%99%ef%bc%881/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%81%9a%e5%88%b0%e5%ae%8c%e7%be%8e%e9%9a%90%e8%ba%ab%ef%bc%9a%e7%9b%b4%e9%9d%a2%e4%bd%a0%e7%9a%84%e5%af%b9%e6%89%8b%e2%80%8a-%e2%80%8a-6%e5%a4%a7%e5%8e%9f%e5%88%99%ef%bc%881/" rel="noopener noreferrer" target="_blank">
    在上篇文章中
   </a>
   我们提到过：追踪是一种非常古老的行当；如今想要在互联网上找到一个人的信息，你完全不必成为一个黑客（你甚至可以一点技术都不懂），照样能搞定。IYP 的开源情报栏目中很多实现靠的都是思考方式，而不是技术。
  </p>
  <p class="graf graf--p">
   当然，有技术会更好，至少您会使用其他人开发的技术，可以帮助追踪工作提速。自动化是个好东西。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    一旦嗅到你的气息，追踪者要做的第一件事就是想方设法打入你的社交关系网
   </strong>
   。
  </p>
  <p class="graf graf--p">
   你经常去的公司是哪里、经常出入的场所都包括什么、经常和谁混在一起……这些都是非常重要的目标。
  </p>
  <p class="graf graf--p">
   接下来社交工程师就会开始行动了，打电话、发邮件、搭讪，直到有人上当，吐露了最重要的信息 —— ⚠️而社交媒体网站将所有这一切都变成了一站式服务。
  </p>
  <p class="graf graf--p">
   只要你有 Facebook，即便进行了高度严格的隐私设置，你的个人主页依旧是一座金矿。
  </p>
  <p class="graf graf--p">
   除非你的隐私设置为最高级别，否则任何陌生人都会能看到你的好友清单，哪怕他们还没有添加你“好友”。
  </p>
  <p class="graf graf--p">
   一旦添加成功就更厉害了。社交工程师的袜子木偶经常这样做，在这里看到一个简易版防御措施《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e8%af%86%e5%88%ab-facebook-%e4%b8%8a%e7%9a%84%e5%81%87%e6%9c%8b%e5%8f%8b%e8%af%b7%e6%b1%82/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e8%af%86%e5%88%ab-facebook-%e4%b8%8a%e7%9a%84%e5%81%87%e6%9c%8b%e5%8f%8b%e8%af%b7%e6%b1%82/" rel="noopener noreferrer" target="_blank">
    如何识别 Facebook 假朋友
   </a>
   》。
  </p>
  <p class="graf graf--p">
   你可能会说，“陌生人只会看到部分好友啊”，没错，但是，只需要点击刷新，就可以积少成多。因为每次刷新后好友信息都会有所不同。这是最傻瓜的做法，但很有效。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Facebook 是间谍的金矿，但绝对不是唯一一个金矿；所有社交网络、媒体、博客、购物网站、支付平台……都是
   </strong>
   。
  </p>
  <p class="graf graf--p">
   所有社交网站都喜欢爆料，不论你选择了什么样的东西挂在网上。优秀的追踪者都可以用来对付你。
  </p>
  <p class="graf graf--p">
   他可以冒充一个想和你取得联系的“老熟人”，添加你的朋友和家人为“好友”。
  </p>
  <p class="graf graf--p">
   ⚠️任何遭遇过国家情报部门间谍活动的人（不论是在中国还是美国还是欧洲）都应该很熟悉这种手段，这是间谍们的套路之一。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-8688 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-10.jpg?resize=1100%2C825&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-10.jpg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-10.jpg?resize=300%2C225&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-10.jpg?resize=768%2C576&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-10.jpg?resize=1024%2C768&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-10.jpg?resize=1100%2C825&amp;ssl=1 1100w" data-recalc-dims="1" height="825" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-10.jpg?resize=1100%2C825&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   👉3、你永远不会知道互联网上的所谓“好友”都是些什么东西。
  </p>
  <p class="graf graf--p">
   避免的方法很简单：如果你打算同意添加某人为好友，哪怕这个人是你非常熟悉的老朋友，也要记得，给对方打个电话或者写上三言两语，以对他们表示感谢 —— 目的是借此确认对方是不是真的。
  </p>
  <p class="graf graf--p">
   一个人多年杳无音信，忽然之间跑到网上要加你“好友”？这种需要格外小心。
  </p>
  <p class="graf graf--p">
   一位社交工程师讲过一个故事。那还是社交媒体火爆之前的事。
  </p>
  <p class="graf graf--p">
   某次社交工程师接到一个生意，需要寻找一个人A，A是一名卡车司机，据称其谎报受伤，申请残疾保险。
  </p>
  <p class="graf graf--p">
   社交工程师对A进行了广泛的监控，但是一无所获，没有发现关于他的任何欺诈行为。
  </p>
  <p class="graf graf--p">
   A看起来是个失业人士，而且非常隐蔽，住在一个破烂不堪的拖车公园里。谷歌搜索他的名字以及多个变体，还加上出生年月，最终才在一个校友网上看到了一则寻人启事，其中A说要想要寻找失散多年的中学女友。
  </p>
  <p class="graf graf--p">
   然后社交工程师的机会就来了。
  </p>
  <p class="graf graf--p">
   他们使用寻人启事的联系方式与A取得了联系，声称自己就是那位失散的女友，还诉说自己婚姻的不幸，很高兴A还记得她……等等。
  </p>
  <p class="graf graf--p">
   很快就收到了回信。A写了好几页，满满都是爱意。
  </p>
  <p class="graf graf--p">
   一系列邮件往来后，A产生了一些怀疑，他要求打电话聊一下。社交工程师婉拒了，称“她”丈夫有家暴倾向。A做出了让步。
  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">

  </p>

  </p>
  <p class="graf graf--p">
   这就是社交工程师用来换钱的东西了 —— 那位寻人的客户在该日期到来时，题着一部相机坐上了A指出的那辆巴士。
  </p>
  <p class="graf graf--p">
   最终，A因谎称残疾骗保，犯有欺诈罪，要在监狱里待一段时间了。
  </p>
  <p class="graf graf--p">
   ⚠️很多人都和A一样，使用社交网络只是出于某种兴趣爱好，不论是公仔、猫咪、国标、汽车、名酒、还是任何话题；如果你也有某种兴趣爱好，那么你应该进入真实的世界，在社区俱乐部里或旅行中满足自己，而不是跑到社交网络上。

  </p>
  <p class="graf graf--p">

   <strong class="markup--strong markup--p-strong">
    普通监视方法失败
   </strong>
   。
  </p>
  <p class="graf graf--p">
   没关系，社交工程师开始在网上挖掘关于她的东西。很快就在一个麦当娜粉丝签名簿里找到了她的名字。由此还找到了“我的空间”网中她的网页。
  </p>
  <p class="graf graf--p">
   于是社交工程师给她写了一封信，说她赢得了一项大奖，有机会让她和其他获奖者一起竞争参与麦当娜录像的机会。
  </p>
  <p class="graf graf--p">
   社交工程师给她发了一个网址还有一个电话号码；两者都含有比赛信息。网站是在一家海外私人企业注册的，通过预付费卡支付的费用，电话号码则是一个预付手机号。于是她
   <strong class="markup--strong markup--p-strong">
    无法追溯到来源
   </strong>
   （哪怕她知道该怎么做）。
  </p>
