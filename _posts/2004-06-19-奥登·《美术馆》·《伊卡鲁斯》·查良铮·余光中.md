  <div class="post-entry-content">
   <p>
    “Landscape with the Fall of Icarus” by Pieter Bruegel
   </p>
   <blockquote>
    <p>
     <strong>
      Musee des Beaux Arts
     </strong>
    </p>
    <p>
     by
     <a href="/273/">
      W. H. Auden
     </a>
    </p>
    <p>
     About suffering they were never wrong,
     <br/>
     The Old Masters; how well, they understood
     <br/>
     Its human position; how it takes place
     <br/>
     While someone else is eating or opening a window or just walking dully along;
     <br/>
     How, when the aged are reverently, passionately waiting
     <br/>
     For the miraculous birth, there always must be
     <br/>
     Children who did not specially want it to happen, skating
     <br/>
     On a pond at the edge of the wood:
     <br/>
     They never forgot
     <br/>
     That even the dreadful martyrdom must run its course
     <br/>
     Anyhow in a corner, some untidy spot
     <br/>
     Where the dogs go on with their doggy life and the torturer’s horse
     <br/>
     Scratches its innocent behind on a tree.
    </p>
    <p>
     In Breughel’s
     <em>
      Icarus
     </em>
     , for instance: how everything turns away
     <br/>
     Quite leisurely from the disaster; the ploughman may
     <br/>
     Have heard the splash, the forsaken cry,
     <br/>
     But for him it was not an important failure; the sun shone
     <br/>
     As it had to on the white legs disappearing into the green
     <br/>
     Water; and the expensive delicate ship that must have seen
     <br/>
     Something amazing, a boy falling out of the sky,
     <br/>
     Had somewhere to get to and sailed calmly on.
    </p>
   </blockquote>
   <p>
    <iframe allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen="" frameborder="0" height="315" src="https://www.youtube.com/embed/zEJyZepZIr8" width="560">
    </iframe>
   </p>
   <blockquote>
    <p>
     <strong>
      美术馆
     </strong>
    </p>
    <p>
     查良铮 译
    </p>
    <p>
     关于苦难他们总是很清楚的，
     <br/>
     这些古典画家：他们多么深知它在
     <br/>
     人心中的地位，深知痛苦会产生，
     <br/>
     当别人在吃，在开窗，或正作着无聊的散步的时候；
     <br/>
     深知当老年人热烈地、虔敬地等候
     <br/>
     神异的降生时，总会有些孩子
     <br/>
     并不特别想要他出现，而却在
     <br/>
     树林边沿的池塘上溜着冰。
     <br/>
     他们从不忘记：
     <br/>
     即使悲惨的殉道也终归会完结
     <br/>
     在一个角落，乱糟糟的地方，
     <br/>
     在那里狗继续过着狗的生涯，而迫害者的马
     <br/>
     把无知的臀部在树上摩擦。
    </p>
    <p>
     在勃鲁盖尔的《伊卡鲁斯》里，比如说；
     <br/>
     一切是多么安闲地从那桩灾难转过脸：
     <br/>
     农夫或许听到了堕水的声音和那绝望的呼喊，
     <br/>
     但对于他，那不是了不得的失败；
     <br/>
     太阳依旧照着白腿落进绿波里；
     <br/>
     那华贵而精巧的船必曾看见
     <br/>
     一件怪事，从天上掉下一个男孩，
     <br/>
     但它有某地要去，仍静静地航行。
    </p>
   </blockquote>
   <blockquote>
    <p>
     <strong>
      美术馆
     </strong>
    </p>
    <p>
     余光中 译
    </p>
    <p>
     说到苦难，他们从未看错，
     <br/>
     古代那些大师：他们深切体认
     <br/>
     苦难在人世的地位；当苦难降临，
     <br/>
     别人总是在进食或开窗或仅仅默然走过；
     <br/>
     当长者正虔诚地、热烈地等，
     <br/>
     等奇迹降临，总有孩子们
     <br/>
     不特别期待它发生，正巧
     <br/>
     在林边的池塘上溜冰：
     <br/>
     大师们从不忘记
     <br/>
     即使可怖的殉道也必须在一隅
     <br/>
     独自进行，在杂乱的一隅
     <br/>
     一任狗照常过狗的日子，酷吏的马匹
     <br/>
     向一颗树干摩擦无辜的后臀。
    </p>
    <p>
     例如布鲁果的《伊卡瑞斯》，众人
     <br/>
     都悠然不顾那劫难，那农夫可能
     <br/>
     听见了水波溅洒，呼救无望，
     <br/>
     但是不当它是惨重的牺牲；阳光灿照，
     <br/>
     不会不照见白净的双腿没入碧湛
     <br/>
     的海波；那豪华优雅的海舟必然看见
     <br/>
     一幕奇景，一童子自天而降，
     <br/>
     却有路要赶，仍安详地向前航行。
    </p>
   </blockquote>

           蒋经国死了以后,余光中写了一首诗拍马屁,从此为他赢得了“马屁诗人”的“美誉”。
           <br/>
           下面就是这首诗：
          </p>
          <p>
           余光中
           <br/>
           《送别
           <br/>
           悲哀的半旗，壮烈的半旗，为你而降，
           <br/>
           悲哀的黑纱，沉重的黑纱，为你而戴，
           <br/>
           悲哀的菊花，纯洁的菊花，为你而开，
           <br/>
           悲哀的灵堂，肃静的灵堂，为你而拜，
           <br/>
           悲哀的行列，依依的行列，为你而排，
           <br/>
           悲哀的泪水，感激的泪水，为你而流，
           <br/>
           悲哀的背影，劳累的背影，不再回头，
           <br/>
           悲哀的柩车，告别的柩车，慢慢地走，
           <br/>
           亲爱的朋友，辛苦的领袖，慢慢地走。
          </p>
