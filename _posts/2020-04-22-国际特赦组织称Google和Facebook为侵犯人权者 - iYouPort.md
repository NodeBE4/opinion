   大赦国际说，Facebook和谷歌实施的 “无处不在的监视” 对人权构成威胁，两家公司对此表示争议。
  </p>
  <p class="graf graf--p">
   在英国，以记录酷刑和种族清洗恶行而闻名的该倡导组织发布了一份报告，让这两个依赖于收集个人数据维生的监视资本主义巨头为其商业模式承担责任。
  </p>
  <p class="graf graf--p">
   大赦国际秘书长 Kumi Naidoo 在一份声明中说：“谷歌和 Facebook 统治着我们的现代生活 —— 通过收集和货币化数十亿人的个人数据来积累对数字世界至高无上的权力。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他们[这些寡头公司]对我们数字生活的阴险控制破坏了隐私权的本质，并且
   <strong class="markup--strong markup--p-strong">
    是当今时代人权面临的决定性挑战之
   </strong>
   一。”

   该组织的报告题为：“
   <strong class="markup--strong markup--p-strong">
    监视巨人：Google 和 Facebook 的商业模式如何威胁人权
   </strong>
   ”，探索了监视资本主义公司如何开展业务、如何影响隐私并影响社会，其市场力量如何限制民主问责制、以及可以做些什么来改变这一状况。

  <p class="graf graf--p">
   报告说：“谷歌和 Facebook 两家公司基于监视的商业模式迫使人们进行浮士德式的交易，人们只能依顺于这个侵犯人权的系统才能享受在线生活。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “首先，对隐私权的侵犯达到了前所未有的规模，其后一系列的连锁效应，对从表达和见解到思想和不受歧视的权利，均构成了严重威胁。”
  </p>
  <p class="graf graf--p">
   然而，言论自由在不受监督的情况下可能会造成问题，正如 Facebook 因未能在缅甸禁止仇恨言论其结果所证明的那样。
  </p>
  <p class="graf graf--p">
   该报告承认，其他大公司也参与了“监视资本主义”，尤其是亚马逊和微软，以及各种数据经纪人和电信公司。
  </p>
  <p class="graf graf--p">
   但是，Facebook 和 Google 由于强大的垄断优势而在特定市场上独树一帜。
  </p>
  <p class="graf graf--p">
   根据报告中引用的第三方统计数据，不包括中国（中国是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e9%ab%98%e5%a0%a1%e5%a5%87%e4%ba%ba/" href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e9%ab%98%e5%a0%a1%e5%a5%87%e4%ba%ba/" rel="noopener noreferrer" target="_blank">
    高堡奇人
   </a>
   的另一部分），Facebook 拥有全球约70％的社交媒体用户和75％的移动消息用户（由于 WhatsApp）；Google 垄断了全球90％的互联网搜索，还拥有 YouTube — 第二大搜索服务和领先的视频平台，并通过 Chrome 统治了浏览器市场，还掌握着最大的移动操作系统 Android（每月活跃设备约25亿）。
  </p>
  <p class="graf graf--p">
   两家公司之间的收入占在线广告收入的60％，占数字广告市场增长的90％。
  </p>

   了。
