  <p class="graf graf--p">
   教师罢工的浪潮始于西弗吉尼亚州，最初分布在几个红州，然后进入蓝州，在洛杉矶、丹佛和奥克兰蔓延。
  </p>
  <p class="graf graf--p">
   领导教师激进运动的 Gillian Russom 在洛杉矶担任高中历史教师已有18年。在大部分时间里，她一直是洛杉矶联合教师工会（UTLA）的积极分子，目前在工会的董事会任职。她也是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.haymarketbooks.org/books/393-education-and-capitalism" href="https://www.haymarketbooks.org/books/393-education-and-capitalism" rel="noopener noreferrer" target="_blank">
    Haymarket Books
   </a>
   出版的“教育与资本主义”一书的撰稿人。
  </p>
  <p class="graf graf--p">
   以下是她与 Truthout 的访谈，讨论教师工会应如何参与选举政治。
   <strong class="markup--strong markup--p-strong">
    其中体现的一些线索可能是值得中国的劳工运动借鉴的经验。
   </strong>
  </p>
  <p class="graf graf--p">
   Ashley Smith: 教师的罢工震动了美国各州，到目前为止罢工的关键经验教训是什么？如何将这些推广到其他公共部门，特别是工会仍然非常薄弱的​​私营部门？
  </p>
  <p class="graf graf--p">
   Gillian Russom: 首先，
   <strong class="markup--strong markup--p-strong">
    不仅仅是教师才会引人注目。罢工远远超出教育职业的范围
   </strong>
   ，远远超出了
   <a class="markup--anchor markup--p-anchor" data-href="https://truthout.org/articles/photo-essay-marriott-workers-celebrate-victory-on-the-picket-line/" href="https://truthout.org/articles/photo-essay-marriott-workers-celebrate-victory-on-the-picket-line/" rel="noopener noreferrer" target="_blank">
    酒店工作人员
   </a>
   、高科技工作者、护士、甚至工业工人……例如，宾
   <a class="markup--anchor markup--p-anchor" data-href="https://socialistworker.org/2019/03/06/fighting-for-the-jobs-that-erie-deserves" href="https://socialistworker.org/2019/03/06/fighting-for-the-jobs-that-erie-deserves" rel="noopener noreferrer" target="_blank">
    夕法尼亚州伊利的 Wabtec 工厂
   </a>
   。
  </p>
  <p class="graf graf--p">
   我们还看到了墨西哥境内的罢工，教师一直在引领这场斗争，现在我们看到了沃尔玛和
   <a class="markup--anchor markup--p-anchor" data-href="https://truthout.org/articles/strike-wave-wins-raises-for-mexican-factory-workers/" href="https://truthout.org/articles/strike-wave-wins-raises-for-mexican-factory-workers/" rel="noopener noreferrer" target="_blank">
    加工厂的罢工
   </a>
   。
  </p>
  <p class="graf graf--p">
   我认为人们正在学习的最明显的教训就是
   <strong class="markup--strong markup--p-strong">
    如何用行动吸引和动员周边人
   </strong>
   。
   <strong class="markup--strong markup--p-strong">
    在一两个星期的时间内，你就将获得几年来和几十年来你尝试任何和平方法都没能获得的改革。人们是亲眼看到这一点的，这就是为什么罢工潮一直在蔓延。
   </strong>
  </p>
  <p class="graf graf--p">
   另一个重要的经验教训是
   <strong class="markup--strong markup--p-strong">
    积极的、踊跃的、和充满启发性的普通职工参与是获胜的关键
   </strong>
   。例如，在西弗吉尼亚州，通过 Facebook 连接的普通职工在一起热情地讨论和辩论，然后纷纷参与进来。
  </p>
  <p class="graf graf--p">
   在像洛杉矶这样的地方，通过
   <strong class="markup--strong markup--p-strong">
    新的组织策略
   </strong>
   ，例如在工作现场建立行动小组，有意识地调动了普通人的参与。如果没能这样不断调动普通人的加入和行动，最好的结果不过是获得一场轻微的罢工，稍微见到一点成功就解散了。
   <strong class="markup--strong markup--p-strong">
    这样就很容易因被当权者的敷衍而迅速叫停。
   </strong>
  </p>
  <p class="graf graf--p">
   你不仅可以自己罢工并赢得胜利，而且在今天的政治环境中，还会得到大众的支持。
   <strong class="markup--strong markup--p-strong">
    人们渴望看到有人可以站出来采取反对富人的立场，因为阶级不平等已经失控。
   </strong>
  </p>
  <p class="graf graf--p">
   所有这些都适用于私营部门。许多私营部门与公众密切合作，提供人们所依赖的服务。人们已经厌倦了大公司的运作方式，并在工人罢工时会与工人们站在一起。来自
   <a class="markup--anchor markup--p-anchor" data-href="https://truthout.org/articles/striking-hotel-workers-slow-season-no-excuse-for-losing-health-care/" href="https://truthout.org/articles/striking-hotel-workers-slow-season-no-excuse-for-losing-health-care/" rel="noopener noreferrer" target="_blank">
    芝加哥酒店工作人员
   </a>
   的大力支持就是一个典型例子。
   <strong class="markup--strong markup--p-strong">
    提高共同的良好需求为目标将这种支持转变为积极参与
   </strong>
   。这不仅仅是指我们代表社区行事，而是我们为所有人想要的一切而奋斗、共同努力。
  </p>
  <p class="graf graf--p">
   这将进一步扩大你的力量。是的，他们说谢谢你，但他们也说，
   <strong class="markup--strong markup--p-strong">
    “这也是我自己的斗争”。这种团结是强大的，可能是不可阻挡的。
   </strong>
  </p>
  <p class="graf graf--p">
   罢工还表明了在合同谈判中
   <strong class="markup--strong markup--p-strong">
    跳出旧有思维框架
   </strong>
   的重要性。即使你认为自己无法赢得胜利，也应该真正思考最大的问题并提出要求，因为这会让人们考虑争取
   <strong class="markup--strong markup--p-strong">
    更大的
   </strong>
   社会和经济正义。
   <strong class="markup--strong markup--p-strong">
    别局限自己在眼下的小问题上，当你停工时，你永远不知道你能真正获得什么。
   </strong>
  </p>
  <p class="graf graf--p">
   对我们来说，另一种思维方式是
   <strong class="markup--strong markup--p-strong">
    克服公共部门面临的经典困境
   </strong>
   ；通常，无论你与谁讨价还价，都会被告知“缺乏资金”。因此，他们可以声称他们没有钱来满足你的要求。这种时候你要清醒。对于我们的案例，我们面临一个拥有20亿美元盈余的地区。但在其他情况下，他们没有盈余，而是赤字。
  </p>
  <p class="graf graf--p">
   在这种情况下，你可能会认为你必须缩小自己的要求，因为据说“钱不在那里”。我们必须拒绝这种做法。那里有钱。所有这些都是由减税和紧缩造成的危机。
  </p>
  <p class="graf graf--p">
   所以，你必须提出要求，可能需要我们的
   <strong class="markup--strong markup--p-strong">
    直接雇主以外的人与他们谈判
   </strong>
   。这就是我们在洛杉矶所做的。最初我不知道这是否会起作用，以及我们能够在多大程度上迫使国家采取行动。
  </p>
  <p class="graf graf--p">
   但我们让加利福尼亚州州长签署了关于
   <a class="markup--anchor markup--p-anchor" data-href="https://www.gov.ca.gov/2019/03/05/charter-school-transparency-legislation/" href="https://www.gov.ca.gov/2019/03/05/charter-school-transparency-legislation/" rel="noopener noreferrer" target="_blank">
    特许学校透明度的法律
   </a>
   ；这在我们的罢工之前从未发生过。
  </p>
  <p class="graf graf--p">
   我们的学校董事会要求州政府以5比1的票数限制特许学校。是一个以私有化为主的学校董事会。洛杉矶市长是一位非常主流的政治家，而他现在表示，他将帮助支持“
   <a class="markup--anchor markup--p-anchor" data-href="https://schoolsandcommunitiesfirst.org/about/" href="https://schoolsandcommunitiesfirst.org/about/" rel="noopener noreferrer" target="_blank">
    学校和社区第一法案
   </a>
   ”，该法案将最终以正确的财产价值向公司征税。
  </p>
  <p class="graf graf--p">
   这一切都不是在罢工之前发生的。因此，我们
   <strong class="markup--strong markup--p-strong">
    在更大的层面上提出了这些问题
   </strong>
   ，即使我们知道
   <strong class="markup--strong markup--p-strong">
    这些问题是我们的直接雇主所无法解决的
   </strong>
   ，以便强迫他们谈论并希望强制采取某些行动也不行。这就是现在正在发生的事。
  </p>
  <p class="graf graf--p">
   Ashley Smith: 这次罢工浪潮最重要的因素之一是普通教职员工的主导作用。为什么这对工会运动的经验学习如此重要？
  </p>
  <p class="graf graf--p">
   Gillian Russom: 如果你想要获胜，那么就不如
   <strong class="markup--strong markup--p-strong">
    调动普通人积极地参与整个斗争，从塑造需求到组织罢工，以及与更广泛的社区建立团结
   </strong>
   。许多工会都在谈论普通人的权力和参与，但仅限于结构方面。
  </p>
  <p class="graf graf--p">
   但是什么驱使这些结构的？它必须是您
   <strong class="markup--strong markup--p-strong">
    在工作现场进行的对话
   </strong>
   。如果活动家们没有谈论什么，我们又为什么需要与活动家的比例为10：1？
  </p>
  <p class="graf graf--p">
   因此，使我们能够显著地涉及普通人的方法是我们
   <strong class="markup--strong markup--p-strong">
    基于大局的对话
   </strong>
   。我们决定与我们的成员讨论紧缩和私有化带来的整个问题，而不是眼下的诸多小问题。
  </p>
  <p class="graf graf--p">
   我们有一个摘要表，上面有谈话要点，与我们的成员讨论公共教育面临的存在危机、尤其是造成这一危机背后的人 — — 那些亿万富翁。因此
   <strong class="markup--strong markup--p-strong">
    要激活普通人的斗志，您需要的不仅仅是结构；你有一个具有激活力量的真正目的。
   </strong>
  </p>
  <p class="graf graf--p">
   最后一个重要的教训是，我们在工会运动中应该保持信心，如果我们将我们的问题提取出要点应对来自亿万富翁阶层的攻击，我们就将赢得支持和团结。我们可以建立更广泛的工人阶级反击。
  </p>
  <p class="graf graf--p">
   Ashley Smith: 现在在工会运动中有很多关于如何在选举舞台上推进其议程的讨论，特别是在民主党内部。关于工会如何应对选举政治，您有何看法？
  </p>
  <p class="graf graf--p">
   Gillian Russom: 罢工浪潮极大地改变了全国各地，特别是民主党内部公共教育的讨论，甚至在支持紧缩和私有化的一些人中也是如此。许多民主党人现在都将自己视为我们运动的盟友。
  </p>
  <p class="graf graf--p">
   但事实是……新自由主义的教育议程一直是两党的，民主党人已经超越了共和党人所做的。想想奥巴马的联邦政策，通过开放更多的特许学校并将教师评估与标准化测试联系起来，迫使学区争夺一小笔钱。
  </p>
  <p class="graf graf--p">
   记住，当党派的整个部分正在改变他们关于教育的观点时，民主党在这方面的作用至关重要。我们显然知道原因：那就是 我们罢工的受欢迎程度以及我们如何改变对话。这应该使我们坚定自己的立场并继续罢工浪潮，而不是简单地集结到这个或那个现在改变立场的政治家。
  </p>
  <p class="graf graf--p">
   在我的工会中，每次学校董事会选举时，总是一个经典的克制，因为我们有一个七人的学校董事会，为了得到我们想要的东西就必须依赖于其中四个民主党人。在这种情况下有些人甚至说“如果我们不全力参加学校董事会选举，我们在罢工期间赢得的一切都将处于危险之中”。
  </p>
  <p class="graf graf--p">
   幸运的是，我们工会中的许多人拒绝这种思维方式。我们说：“记得我们输掉了最后一次校董会选举，但他们现在投票支持我们的议程，
   <strong class="markup--strong markup--p-strong">
    因为我们进行了罢工
   </strong>
   。他们甚至要求在加利福尼亚的特许学校上限！正是我们来自工作场所的力量改变了对话和政策。“
  </p>
  <p class="graf graf--p">
   这并不意味着我们应该忽视选举舞台。因为我们已经改变了对话，我们可以换一种方式对政治家们说，如果你不再需要紧缩和私有化，就相当于正在为公立学校提供资金，这正是我们想要你做的事。这正是我们在加利福尼亚所做的。
  </p>
  <p class="graf graf--p">
   我们还说，如果你现在与教师们站在一起，及时限制特许学校并对其进行规范，并且需要在开设任何新的特许学校之前提供社区影响报告。大多数社区已经有足够的公立学校。开设特许学校的唯一影响将是失去教师和解散公立学校。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    通过报告揭露这些事实，我们可以为更大规模的反对运动建立支持
   </strong>
   。我们可以为萨克拉门托的法案提供支持，以暂停特许学校。我们应该利用罢工积累的势头要求所有候选人都采纳我们的要求。
  </p>
  <p class="graf graf--p">
   Ashley Smith: 这对即将举行的总统选举意味着什么？
  </p>
  <p class="graf graf--p">
   Gillian Russom: 它使我们有机会提出一个国家计划，即我们在全国范围内进行的罢工在联邦一级的要求和需求。政府制定了所有这些可怕的政策，如 Race to the Top，将一切都缩小到州和地方层面。但许多州的政客声称资金并不是他们的问题，因为它主要是一个国家问题。
  </p>
  <p class="graf graf--p">
   我们需要改变那次谈话。这有几种方法可以做到这一点。首先，Title I 是一个联邦计划，根据低收入学生人数为学校提供资金。但是现在，政治家们正在通过大规模的“资金不足”借口来违反这一承诺。我们应该要求所有总统候选人改变这一点并为 Title I 提供全额资金。
  </p>
  <p class="graf graf--p">
   其次，我们需要鼓励特殊教育经费。联邦政府应该将这笔资金的 40％ 捐给地区，但他们几十年来一直没有这样做（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/education/2019/03/28/so-what-is-devos-trump-record-special-education-funding-anyway/?utm_term=.e19d66f77bca" href="https://www.washingtonpost.com/education/2019/03/28/so-what-is-devos-trump-record-special-education-funding-anyway/?utm_term=.e19d66f77bca" rel="noopener noreferrer" target="_blank">
    他们目前仅提供14.7％
   </a>
   ）。因此，学区因为根据“残障人士教育法”在法律上有义务提供特殊教育，所以学区已经弥补了差异，并且这些都来自当地的学校预算。这就是为什么当地学校预算匮乏的原因。
  </p>
  <p class="graf graf--p">
   因此，我们可以要求联邦政府履行为特殊教育提供资金的义务。这次选举是一个开端。我们应该推动我们的工会。
  </p>
  <p class="graf graf--p">
   现在不是到处考虑代言或者我们更喜欢哪个人的时候；现在是时候在地方、州和联邦层面制定公共教育议程，并要求政治家签署这一议程。我们或许能够为2020年之前的教育提出“新政”。
  </p>
  <p class="graf graf--p">
   Ashley Smith: 新一代社会主义者在这场罢工浪潮中发挥着关键作用。社会主义者应该如何组织这个新的令人兴奋的局面？
  </p>
  <p class="graf graf--p">
   Gillian Russom: 如果你在任何这些罢工期间都在纠察线上，你会感受到我们
   <strong class="markup--strong markup--p-strong">
    作为工会会员所拥有的不可思议的力量
   </strong>
   。这就是让这些罢工如此惊人的原因。我认为我们作为社会主义者的角色必须是
   <strong class="markup--strong markup--p-strong">
    推动这种为共同利益而议价的战略
   </strong>
   ，但我们需要超越这一目标，
   <strong class="markup--strong markup--p-strong">
    为共同利益创造工会，为共同利益创造一整套工人运动。
   </strong>
  </p>
  <p class="graf graf--p">
   我们必须将这种精神带入我们为整个工人阶级运动制定战略的方式。我们必须成为我们的领导者，帮助扭转已经持续数十年的片面阶级战争。所有这一切的关键是将阶级需求与压迫，特别是种族主义的具体问题结合起来。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    我们国家有这么多钱，集中在富人和他们的公司手中
   </strong>
   。想想硅谷和所有公司，如 Apple，Wells Fargo，Chevron 等。然后再想想在洛杉矶中南部和东洛杉矶上学的学生。
   <strong class="markup--strong markup--p-strong">
    这种不平等是一种犯罪。
   </strong>
  </p>
  <p class="graf graf--p">
   与此同时，我们工会的社会主义者认为，UTLA 应该推动市政投票措施，以阻止监狱资金，并为执法制定更多的问责制。这被称为改革洛杉矶监狱，将从监狱中剥离资金，并将其转入就业资金和心理健康计划。洛杉矶县监狱系统是全国最大的监狱机构。这只是我们如何在社会主义观点基础上领导运动的一个例子，
   <strong class="markup--strong markup--p-strong">
    这种观点将阶级不平等、种族主义和所有其他压迫的斗争结合在一起进行共同斗争。
   </strong>
  </p>
  <p class="graf graf--p">
   **当“正规流程”失败时普通人还有一个重量级的武器可以让政策制定者改变主意，那就是运动。
   <strong class="markup--strong markup--p-strong">
    这篇访谈希望中国读者能尤其注意到
   </strong>
   ：1、是什么激发了运动的蔓延？或者说，如何用行动作品吸引和动员尚未行动起来的人。2、如何避免被当权者的敷衍迅速灭掉运动的气焰？3、从塑造需求到组织罢工，以及与更广泛的社区建立团结，这其中的关键在于什么？
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    If
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://truthout.org/articles/teachers-are-the-underdogs-in-a-decades-long-class-war/" href="https://truthout.org/articles/teachers-are-the-underdogs-in-a-decades-long-class-war/" rel="noopener noreferrer" target="_blank">
    you want to win, the rank and file must be actively involved in the whole struggle, from shaping demands to organizing strikes and building solidarity.
   </a>
   Striking works. In a period of a week or two, you win reforms that you have not gotten in years and decades. — Teachers Are the Underdogs in a Decades-Long Class War
  </p>
