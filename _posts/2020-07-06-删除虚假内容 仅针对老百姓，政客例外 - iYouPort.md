    <img alt="" class="aligncenter size-full wp-image-12446" data-recalc-dims="1" height="584" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-6.png?resize=1100%2C584&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-6.png?w=1390&amp;ssl=1 1390w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-6.png?resize=300%2C159&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-6.png?resize=1024%2C544&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-6.png?resize=768%2C408&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/0-6.png?resize=1100%2C584&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Instagram 正在给政客们自由传播虚假消息的特权，与其母公司 Facebook 一样。
  </p>
  <p class="graf graf--p">
   Instagram 从5月开始在美国扩展其有限的事实检查测试，现在将与45个第三方组织合作以评估其应用程序中照片和视频内容的真实性。被评级为 “假” 的材料将从 “浏览” 和 “标签” 页面中隐藏，并显示警告，直到用户再次点击以查看帖子。
  </p>
  <p class="graf graf--p">
   这比 Facebook 早期尝试在内容旁边的链接上附加警告的举措迈出了更大的一步，但是仍然允许用户可以立即使用该虚假信息。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12447 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18.png?resize=1100%2C726&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18.png?w=2048&amp;ssl=1 2048w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18.png?resize=300%2C198&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18.png?resize=1024%2C676&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18.png?resize=768%2C507&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18.png?resize=1536%2C1014&amp;ssl=1 1536w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18.png?resize=1100%2C726&amp;ssl=1 1100w" data-recalc-dims="1" height="726" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18.png?resize=1100%2C726&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12447" data-recalc-dims="1" height="726" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18.png?resize=1100%2C726&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18.png?w=2048&amp;ssl=1 2048w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18.png?resize=300%2C198&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18.png?resize=1024%2C676&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18.png?resize=768%2C507&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18.png?resize=1536%2C1014&amp;ssl=1 1536w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-18.png?resize=1100%2C726&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Instagram 将使用图像匹配技术来查找虚假内容的其他副本，并应用相同的标签，在 Facebook 和 Instagram 内容中均执行此操作。
  </p>
  <p class="graf graf--p">
   这可能成为 Facebook 的
   <strong class="markup--strong markup--p-strong">
    说辞
   </strong>
   ，因为该公司一直在试图阻止监管机构解散该公司并剥离 Instagram 的意向。
  </p>
  <p class="graf graf--p">
   Instagram 试图全面加强其所谓的安全实践。它开始提醒用户，如果即将在照片或视频上发布的标题令人反感或被视为欺凌行为，用户有机会在发布文本之前对其进行编辑。
  </p>
  <p class="graf graf--p">
   Instagram 还开始询问新用户的年龄，以确保他们的年龄在13岁或以上。它曾经在此冒充无知，以躲避《儿童在线隐私保护法》的罚款。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12448 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-19.png?resize=1100%2C745&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-19.png?w=1536&amp;ssl=1 1536w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-19.png?resize=300%2C203&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-19.png?resize=1024%2C693&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-19.png?resize=768%2C520&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-19.png?resize=1100%2C745&amp;ssl=1 1100w" data-recalc-dims="1" height="745" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-19.png?resize=1100%2C745&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12448" data-recalc-dims="1" height="745" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-19.png?resize=1100%2C745&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-19.png?w=1536&amp;ssl=1 1536w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-19.png?resize=300%2C203&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-19.png?resize=1024%2C693&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-19.png?resize=768%2C520&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-19.png?resize=1100%2C745&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    其中有人拥有不受事实检查的特权 —— 那就是政治人物群体
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这些政客在 Instagram 上的原始内容（包括政治广告）都不会被进行事实检查，即使这些内容明显地虚假。
  </p>
  <p class="graf graf--p">
   这与 Facebook 的政策相吻合，Facebook 的政策遭到包括 TechCrunch 在内的批评家的强烈反对，后者表示，
   <strong class="markup--strong markup--p-strong">
    这项政策可以让候选人随意抹黑竞争对手、煽动两极分化、并通过谎言筹集资金
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Instagram 首席执行官 Adam Mosseri 坚持
   <a class="markup--anchor markup--p-anchor" data-href="https://techcrunch.com/2019/10/17/zuckerberg-speech/" href="https://techcrunch.com/2019/10/17/zuckerberg-speech/" rel="noopener noreferrer" target="_blank">
    认为
   </a>
   ，禁止政治广告 “可能会伤害需要提升自己的挑战者候选人，并且很难在政治广告和有问题的广告之间划清界限”。
  </p>
  <p class="graf graf--p">
   幸运的是，Instagram 在这方面的危险性较小，因为帖子无法直接链接到政客可以筹集资金的网站。但是，
   <strong class="markup--strong markup--p-strong">
    经过验证的用户就可以将链接附加到故事
   </strong>
   ，并且每个人都可以在个人资料中拥有一个链接。
  </p>
  <p class="graf graf--p">
   ⚠️ 这意味着，
   <strong class="markup--strong markup--p-strong">
    政客仍然可以在该应用程序上故意将虚假信息武器化
   </strong>
   ，以牺牲真相为代价来进一步
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e6%88%98%e5%92%8c%e4%bf%a1%e6%81%af%e6%88%98/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e6%88%98%e5%92%8c%e4%bf%a1%e6%81%af%e6%88%98/" rel="noopener noreferrer" target="_blank">
    深化信息战
   </a>
   ，却使大众认为可以相信自己在 Instagram 上看到的内容。⚪️
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://techcrunch.com/2019/12/16/instagram-fact-checking/" href="https://techcrunch.com/2019/12/16/instagram-fact-checking/" rel="noopener noreferrer" target="_blank">
    Instagram hides false content behind warnings, except for politicians
   </a>
  </p>
