  <p class="graf graf--p">
   这不是我们第一次强调这点：很多反乌托邦技术都是西方提供的，并且西方领导人并不真的关心其造成的人权危害。
  </p>
  <p class="graf graf--p">
   Coda 的这篇报道证据充分，可以再次解释这一问题。
  </p>
  <p class="graf graf--p">
   2012年8月，中国在新疆省偏远的石油城镇克拉玛依
   <a class="markup--anchor markup--p-anchor" data-href="http://www.chinadaily.com.cn/business/2012-11/14/content_15935692.htm" href="http://www.chinadaily.com.cn/business/2012-11/14/content_15935692.htm" rel="noopener noreferrer" target="_blank">
    启动
   </a>
   了首个重要的“智能城市”项目。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “信息技术不仅是关于技术的。它应该与我们城市生活的方方面面融为一体，让人们的生活更加便利，“克拉玛依市市长陈新发说，“对于克拉玛依来说，这不是未来，而是现在正在发生的事。”
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    在这里看到全球多个国家对所谓的智能城市的广泛追求，中国是之一；还有技术分析《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%E6%99%BA%E8%83%BD%E5%9F%8E%E5%B8%82%E7%A9%B6%E7%AB%9F%E6%98%AF%E4%B8%AA%E4%BB%80%E4%B9%88%E9%AC%BC%EF%BC%9F%EF%BC%81/" href="https://www.iyouport.org/%E6%99%BA%E8%83%BD%E5%9F%8E%E5%B8%82%E7%A9%B6%E7%AB%9F%E6%98%AF%E4%B8%AA%E4%BB%80%E4%B9%88%E9%AC%BC%EF%BC%9F%EF%BC%81/" rel="noopener noreferrer" target="_blank">
     “智能城市”究竟是个什么鬼？！
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   陈说，克拉玛依这个拥有40万居民的城市的公交车站均配备了显示旅行信息的电子屏幕；
  </p>
  <p class="graf graf--p">
   在家中，老年居民可以随时按下紧急按钮以呼叫紧急服务或亲属；
  </p>
  <p class="graf graf--p">
   如果社会保障体系记录了失业人口的增加，
   <strong class="markup--strong markup--p-strong">
    官员们马上就会知道
   </strong>
   —— 这一切都归功于克拉玛依的所谓智能城市网络 ……
  </p>
  <p class="graf graf--p">
   克拉玛依是智能城市趋势的早期采用者；
   <strong class="markup--strong markup--p-strong">
    如今这种反乌托邦形式正在重塑世界大部分地区
   </strong>
   ，中国仍然是该领域的领导者之一。
  </p>
  <p class="graf graf--p">
   根据英国金融服务公司德勤（Deloitte）的数据，目前巴西、沙特阿拉伯和阿联酋等国家的智能城市项目已经
   <strong class="markup--strong markup--p-strong">
    超过1,000个
   </strong>
   。
  </p>
  <p class="graf graf--p">
   全球近一半的智能城市在中国，使这个国家成为世界上最大的智能城市集中地。
  </p>
  <p class="graf graf--p">
   从事智能城市行业的那些家伙将该项目作为城市规划的最前沿。
  </p>
  <p class="graf graf--p">
   Deloitte China 去年发布了一份名为”超级智能城市：更高质量的快乐社会”的
   <a class="markup--anchor markup--p-anchor" data-href="https://www2.deloitte.com/content/dam/Deloitte/cn/Documents/public-sector/deloitte-cn-ps-super-smart-city-en-180629.pdf" href="https://www2.deloitte.com/content/dam/Deloitte/cn/Documents/public-sector/deloitte-cn-ps-super-smart-city-en-180629.pdf" rel="noopener noreferrer" target="_blank">
    报告
   </a>
   ，其中说，庆祝中国建立”超级智能城市”的动力，这些城市整合了医疗保健、交通、教育和公共安全等服务的数据。
  </p>
  <p class="graf graf--p">
   报告还称，“无人机、可穿戴监视设备、面部识别、和预测犯罪系统有助于执法部门打击犯罪并维护公共安全” ……“
   <strong class="markup--strong markup--p-strong">
    各机构通过利用包括社交媒体和众包数据在内的所有数据流来预防犯罪
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   据专家和分析师称，中国智能城市被 Deloitte 称为虚拟乌托邦，是任何一个国家有史以来能够建造的最大的、也是最具侵入性的监控架构。
  </p>
  <p class="graf graf--p">
   像克拉玛依这样早期的智能城市项目目前已经发展成为新疆的计算机化警察国家，这里是有史以来最全面的监控建设的所在地。
  </p>
  <p class="graf graf--p">
   克拉玛依的版本实际上遍布新疆，可以追踪约250万居民，针对该省的维吾尔穆斯林少数民族。
  </p>
  <p class="graf graf--p">
   被标记为可疑活动的人 — 例如祈祷的人 — 会被中国情报部门调查、并被拘留在“再教育”营。
  </p>
  <p class="graf graf--p">
   目前，约有一百万维吾尔族穆斯林人口被置于“一个笼罩在秘密中的大规模拘留营” —— 这是
   <a class="markup--anchor markup--p-anchor" data-href="https://uk.reuters.com/article/uk-china-rights-un/u-n-says-it-has-credible-reports-that-china-holds-million-uighurs-in-secret-camps-idUKKBN1KV23P" href="https://uk.reuters.com/article/uk-china-rights-un/u-n-says-it-has-credible-reports-that-china-holds-million-uighurs-in-secret-camps-idUKKBN1KV23P" rel="noopener noreferrer" target="_blank">
    联合国的描述
   </a>
   。
  </p>
  <p class="graf graf--p">
   西方政府广泛谴责新疆警方的行为。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    西方科技巨头公司一直在以建设智能城市为名义帮助建立了这个庞大的警察国家
   </strong>
   。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5683 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/数字长城.png?resize=1100%2C621&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/数字长城.png?w=1402&amp;ssl=1 1402w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/数字长城.png?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/数字长城.png?resize=768%2C434&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/数字长城.png?resize=1024%2C578&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/数字长城.png?resize=1100%2C621&amp;ssl=1 1100w" data-recalc-dims="1" height="621" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/数字长城.png?resize=1100%2C621&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    新的监控范式
   </strong>
  </p>
  <p class="graf graf--p">
   西方政府机构最近对华为等中国科技公司出口的危险性提出了警告。美国商务部还表示，华为将被禁止购买美国公司的计算机芯片等新产品。
  </p>
  <p class="graf graf--p">
   华为一直是这些措施的主要目标。去年，美国六家情报机构负责人警告美国公民不要使用华为产品。
  </p>
  <p class="graf graf--p">
   诚然，华为帮助中国政府监视自己的公民、帮助建立智能城市等等。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    但是，政府研究、公司文件、泄露的官方报告、和公共记录均表明，中国并没有自己发明智能城市的范式
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    中国能做到如此强大的警察国家是在西方政府和硅谷主要公司 ——
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://是在一些西方政府和硅谷主要公司（包括IBM和微软）的帮助下完成的。" href="http://是在一些西方政府和硅谷主要公司%EF%BC%88包括IBM和微软%EF%BC%89的帮助下完成的。" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     包括IBM和微软 —— 的帮助下完成的
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   中国智能城市项目的种子就是 IBM 于2009年首次在其“智能地球”概念中播种下的，当时
   <strong class="markup--strong markup--p-strong">
    IBM公司与中国200多位市长进行了接触，并与东北大学合作建立了联合“生态城”研究实验室，资金来自当地政府 —— 4100万美元
   </strong>
   。
  </p>
  <p class="graf graf--p">
   次年，IBM 在北京召开的一次会议上
   <a class="markup--anchor markup--p-anchor" data-href="http://www.chinadaily.com.cn/business/2010-03/26/content_9647559.htm" href="http://www.chinadaily.com.cn/business/2010-03/26/content_9647559.htm" rel="noopener noreferrer" target="_blank">
    宣布了
   </a>
   为期10年的中国智能城市发展战略。
  </p>
  <p class="graf graf--p">
   未来的城市将使用
   <strong class="markup--strong markup--p-strong">
    海量数据收集
   </strong>
   来解决交通、食品安全、制造业、水资源管理、能源和公共服务等问题。
  </p>
  <p class="graf graf--p">
   2012年IBM
   <a class="markup--anchor markup--p-anchor" data-href="https://www.ibm.com/investor/att/pdf/investor0512/presentation/06_Smarter_Cities_Growth_Markets.pdf" href="https://www.ibm.com/investor/att/pdf/investor0512/presentation/06_Smarter_Cities_Growth_Markets.pdf" rel="noopener noreferrer" target="_blank">
    投资者简报
   </a>
   “智能城市”中写道，
   <strong class="markup--strong markup--p-strong">
    该公司在中国建立了82家IBM分支机构，包括研究、创新和软件开发 —— 其中就有新疆克拉玛依新兴的“智能城市”项目
   </strong>
   。
  </p>
  <p class="graf graf--p">
   该公司早期的智能城市项目包括2012年在镇江市安装所谓智能解决方案的旗舰智能运营中心（IOC）。
  </p>
  <p class="graf graf--p">
   虽然该项目侧重于对交通、公共汽车和乘客行为的实时监控，但是，
   <strong class="markup--strong markup--p-strong">
    IBM的IOC还旨在大量整合数据以实现维稳目的。
   </strong>
  </p>
  <p class="graf graf--p">
   根据同年发布的
   <a class="markup--anchor markup--p-anchor" data-href="http://www.redbooks.ibm.com/redbooks/pdfs/sg248061.pdf" href="https://www.redbooks.ibm.com/redbooks/pdfs/sg248061.pdf" rel="noopener noreferrer" target="_blank">
    IBM手册
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    IOC 能够将大量信息整合到城市功能中，包括“维稳机构”和“其他级别的政府”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   该手册说，数据来源包括来自“天气、公民、执法、社会福利、视频” 等方面的信息，可以帮助更有效地运营城市。
  </p>
  <p class="graf graf--p">
   该手册还确定了

    <strong class="markup--strong markup--p-strong">
     情感分析
    </strong>
   </a>
   的潜力，该分析通过
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%BB%80%E4%B9%88%E6%98%AF%E7%A4%BE%E4%BA%A4%E5%AA%92%E4%BD%93%E6%83%85%E6%8A%A5%EF%BC%9F%E4%BD%A0%E7%9A%84%E6%8E%A8%E7%89%B9%E5%A6%82%E4%BD%95%E6%88%90%E4%B8%BA%E6%8A%93%E6%8D%95%E4%BD%A0%E7%9A%84/" href="https://www.iyouport.org/%E4%BB%80%E4%B9%88%E6%98%AF%E7%A4%BE%E4%BA%A4%E5%AA%92%E4%BD%93%E6%83%85%E6%8A%A5%EF%BC%9F%E4%BD%A0%E7%9A%84%E6%8E%A8%E7%89%B9%E5%A6%82%E4%BD%95%E6%88%90%E4%B8%BA%E6%8A%93%E6%8D%95%E4%BD%A0%E7%9A%84/" rel="noopener noreferrer" target="_blank">

   包括英国国防部、伦敦大都会警察局、美国宇航局和美国海军。
  </p>
  <p class="graf graf--p">
   根据美国军方太平洋司令部前分析师国家安全专家 Anders Corr 的说法，这种伙伴关系构成了严重的潜在风险。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “西方和西方公司负有最大化股东价值的信托责任。期望这些公司按照任何其他道德规范行事都是错误的”，Corr 说，他认为需要政府监督。
  </p>
  <p class="graf graf--p">
   但是实际上，恰恰相反。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    ⚠️尽管特朗普政府经常表达对中国科技公司的排斥，但是美国政府长期以来一直鼓励美国公司利用中国的数字化驱动，几乎没有任何疑虑
   </strong>
   。
  </p>
  <p class="graf graf--p">
   就在2017年夏天，美国商务部还为希望在中国投资的美国公司发布了一份
   <a class="markup--anchor markup--p-anchor" data-href="https://www.export.gov/article?id=China-Technology-and-ICT" href="https://www.export.gov/article?id=China-Technology-and-ICT" rel="noopener noreferrer" target="_blank">
    国家指南
   </a>
   ，确定了跨大数据、物联网、人工智能、云计算和“智能城市发展”的投资机会。
  </p>
  <p class="graf graf--p">
   三个月后，
   <strong class="markup--strong markup--p-strong">
    美国商务部与中国政府资助的上海浦东智能城市研究所合作，在上海举办“中美智能城市论坛”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   英国政府最近
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bbc.co.uk/news/uk-48032286" href="https://www.bbc.co.uk/news/uk-48032286" rel="noopener noreferrer" target="_blank">
    批准
   </a>
   华为提供新的5G网络设备，并试图通过其中国繁荣战略计划基金（SPF）为
   <strong class="markup--strong markup--p-strong">
    英国和中国的联合智能城市项目
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.gov.uk/government/news/uk-cities-pioneering-smart-cities-evolution-in-china" href="https://www.gov.uk/government/news/uk-cities-pioneering-smart-cities-evolution-in-china" rel="noopener noreferrer" target="_blank">
    提供直接资金
   </a>
   来吸引中国的注意力。
  </p>
  <p class="graf graf--p">
   该项目引入了英国公司“进入中国智能城市市场”，但也试图利用中国智能城市实践的研究在英国国内实施。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5681 jetpack-lazy-image" data-lazy-sizes="(max-width: 1028px) 100vw, 1028px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/酸文假醋.png?resize=1028%2C1304&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/酸文假醋.png?w=1028&amp;ssl=1 1028w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/酸文假醋.png?resize=237%2C300&amp;ssl=1 237w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/酸文假醋.png?resize=768%2C974&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/酸文假醋.png?resize=807%2C1024&amp;ssl=1 807w" data-recalc-dims="1" height="1304" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/酸文假醋.png?resize=1028%2C1304&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1028"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5681" data-recalc-dims="1" height="1304" sizes="(max-width: 1028px) 100vw, 1028px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/酸文假醋.png?resize=1028%2C1304&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/酸文假醋.png?w=1028&amp;ssl=1 1028w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/酸文假醋.png?resize=237%2C300&amp;ssl=1 237w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/酸文假醋.png?resize=768%2C974&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/酸文假醋.png?resize=807%2C1024&amp;ssl=1 807w" width="1028"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    睁大眼睛
   </strong>
  </p>
  <p class="graf graf--p">
   你会假设西方政府不知道这些技术会在中国被如何使用吗？
  </p>
  <p class="graf graf--p">
   事实上正相反。
  </p>
  <p class="graf graf--p">
   由 EUSME 中心出版的智能城市报告 —— 这是一个欧盟委员会资助的项目 —— 表明
   <strong class="markup--strong markup--p-strong">
    西方政府实际上很高兴参与建设中国的警察国家状态
   </strong>
   。
  </p>
  <p class="graf graf--p">
   该中心的目的是帮助欧洲的中小企业准备在中国开展业务。
  </p>
  <p class="graf graf--p">
   EUSME 的智能城市报告未公开发布，时间是2016年1月，由中英商务委员会的官员编写，该委员会为英国国际贸易部提供中国业务发展服务、并与英国外交部密切合作。
  </p>
  <p class="graf graf--p">
   该报告还指出，将数据收集整合到任何城市的目标是为
   <strong class="markup--strong markup--p-strong">
    包括警察在内的一系列利益相关机构
   </strong>
   提供支持“智能城市安全系统”的“人口管理”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    反乌托邦，究竟是中国制造、还是美国制造？
   </strong>
  </p>
  <p class="graf graf--p">
   像IBM和微软这样的硅谷巨头并没有透露他们在帮助建立中国智能城市监控范例方面的作用。
  </p>
  <p class="graf graf--p">
   私营公司、中央政府和市政府如何共享技术和数据的机制和任何保障措施仍然未知。
  </p>
  <p class="graf graf--p">
   隐私国际组织的 Eva Blum-Dumontet 说：”像IBM和微软等西方公司以及某些压迫性政权之间看似无关紧要的伙伴关系
   <strong class="markup--strong markup--p-strong">
    最令人担忧
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “在建设智能城市或促进政府服务信息流动的借口下，一些公司可以有效地参与侵犯人权的行为
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   但中国的警察国家才刚刚开始。未来几年内，⚠️智能城市建设将成为各地市政府的主导范式。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “不幸的是，在远东地区建造的大多数智能城市都成为了大规模监控城市，追踪你的一举一动，”加拿大安大略省前信息专员、国际智能城市理事会成员 Anne Cavoukian 博士说。
  </p>
  <p class="graf graf--p">
   华为关于“安全城市”计划的官方文献声称已将该模型出口到全球
   <a class="markup--anchor markup--p-anchor" data-href="https://archive.li/pZ9HO#selection-11355.0-11355.497" href="https://archive.li/pZ9HO#selection-11355.0-11355.497" rel="noopener noreferrer" target="_blank">
    230多个城市
   </a>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    华为安全城市解决方案的早期采用者包括具有威权主义和具有国内暴力历史的国家
   </strong>
   ，包括俄罗斯、巴基斯坦、老挝、安哥拉、亚美尼亚、阿塞拜疆、塔吉克斯坦、乌克兰、乌兹别克斯坦、吉尔吉斯斯坦和哈萨克斯坦。
  </p>
  <p class="graf graf--p">
   在塞尔维亚，华为在贝尔格莱德首都签订了公共安全合同，其面部识别摄像系统受到数字权利监管机构的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.zdnet.com/article/huaweis-surveillance-system-in-serbia-threatens-citizens-rights-watchdog-warns/" href="https://www.zdnet.com/article/huaweis-surveillance-system-in-serbia-threatens-citizens-rights-watchdog-warns/" rel="noopener noreferrer" target="_blank">
    严厉批评
   </a>
   —— 违反公民隐私法。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    到2025年，全球智能城市市场将价值2.5万亿美元。由华为等中国公司主导，很少有哪个政府不想进入该市场
   </strong>
   。⚪️
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://codastory.com/authoritarian-tech/silicon-valleys-scramble-for-china/" href="https://codastory.com/authoritarian-tech/silicon-valleys-scramble-for-china/" rel="noopener noreferrer" target="_blank">
    Western technology companies have played a pivotal role in building China’s authoritarian smart cities
