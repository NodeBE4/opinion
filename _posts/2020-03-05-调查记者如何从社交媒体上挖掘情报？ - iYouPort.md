   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%80%e4%b9%88%e6%98%af%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%83%85%e6%8a%a5%ef%bc%9f%e4%bd%a0%e7%9a%84%e6%8e%a8%e7%89%b9%e5%a6%82%e4%bd%95%e6%88%90%e4%b8%ba%e6%8a%93%e6%8d%95%e4%bd%a0%e7%9a%84/" href="https://www.iyouport.org/%e4%bb%80%e4%b9%88%e6%98%af%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%83%85%e6%8a%a5%ef%bc%9f%e4%bd%a0%e7%9a%84%e6%8e%a8%e7%89%b9%e5%a6%82%e4%bd%95%e6%88%90%e4%b8%ba%e6%8a%93%e6%8d%95%e4%bd%a0%e7%9a%84/" rel="noopener noreferrer" target="_blank">
    社交媒体情报
   </a>
   由于一直被全球当权者用来追踪异议和活动家，这个概念经常处于臭名昭著的状态。
  </p>
  <p class="graf graf--p">
   但社交媒体情报本身并不是纯粹恶意的 —— 公民记者、活动家和NGO组织都使用它寻求证据和调查线索，以维护正义。
  </p>
  <p class="graf graf--p">
   尤其是调查记者（包括公民记者），挖掘突发新闻、跟踪大型群体事件、验证热门消息的真伪等等，都需要使用社交媒体情报的相关技术和技巧。
  </p>
  <p class="graf graf--p">
   尤其是用户量最大的社交媒体平台，它们都是情报来源的金矿。
  </p>
  <p class="graf graf--p">
   从 Facebook 和 Twitter 挖掘情报的相关示例是最多的，它们面对不同的调查和追踪需求，
   <strong class="markup--strong markup--p-strong">
    不仅演示操作，更强调侦查和反侦察的思考方式
   </strong>
   。
  </p>
  <p class="graf graf--p">
   尽管 Instagram 是新的社交媒体宠儿，但同时拥有 Instagram 的 Facebook 仍然拥有两倍数量的用户群，是全球几十亿人围绕特定主题进行对话和互动的热门场所。
  </p>
  <p class="graf graf--p">
   尽管其中一些是私人团体，您可能需要被邀请或成为消息来源才能获得访问权限，但是这个世界上最大的社交网络始终是可供记者访问的大量公开信息池，
   <strong class="markup--strong markup--p-strong">
    您只需要知道在哪里找到自己想要的东西即可
   </strong>
   。
  </p>
  <p class="graf graf--p">
   波士顿大学计算新闻学实践副教授 Brooke Williams 和 Bellingcat 的首席研究员 Henk van Ess，谈论了他们如何
   <strong class="markup--strong markup--p-strong">
    使用社交媒体挖掘调查资源和情报
   </strong>
   的经验。以下对他们的答案进行了编辑，以求简洁明了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、瞄准相关性最强的可见的社区团体
   </strong>
  </p>
  <p class="graf graf--p">
   首先就是查看所有公开的组织，尤其是与您的调查目标存在关系的任何组织和个人。
  </p>
  <p class="graf graf--p">
   地理位置条件很重要，人们总是更多关心发生在自己身边的事；关心当地问题的当地人倾向于组成 Facebook 小组。
  </p>
  <p class="graf graf--p">
   例如，如果一架直升机在某个地区坠毁，那么就在 Facebook 上搜索该特定地区的社区，并使用诸如当地街道名或特定地区名之类的关键字来寻找目击者。
  </p>
  <p class="graf graf--p">
   还可以通过Google网上论坛进行搜索，在这些网上论坛中，目标群体经常会在其留言板上留下情报线索的“面包屑”。
  </p>
  <p class="graf graf--p">
   这取决于人们的隐私设置方式。你总是能找到东西，因为并非所有人都那么注意反监视措施。
  </p>
  <p class="graf graf--p">
   浏览这些群组消息时，使用“讨论记录”或“日程表”之类的关键字搜索能获得很多成功，通过这些搜索可以找到“潜在来源”的文档和联系信息。
  </p>
  <p class="graf graf--p">
   尽管您不一定能看到这些消息是发送给谁的，但发件人的电子邮件通常是可见的。你就可以沿着这条线索深入调查。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、使用为记者创建的免费搜索工具过滤 Facebook
   </strong>
  </p>
  <p class="graf graf--p">
   就如我们曾经做过的诸多演示那样：就算进行了隐私设置，社交媒体网站上仍然可以挖到大量低调而高价值的信息，从 Facebook 到 Twitter 到 Instagram 都是如此。
  </p>
  <p class="graf graf--p">
   如果您知道如何过滤掉杂音，就可以更快速地找到想要的东西，并做更多的事。
  </p>
  <p class="graf graf--p">
   还记得这项调查吗？《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e5%9c%b0%e5%9b%be%e6%98%af%e6%af%94%e8%a9%b9%e5%a7%86%e6%96%af%e9%82%a6%e5%be%b7%e6%9b%b4%e6%a3%92%e7%9a%84%e9%97%b4%e8%b0%8d/" href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e5%9c%b0%e5%9b%be%e6%98%af%e6%af%94%e8%a9%b9%e5%a7%86%e6%96%af%e9%82%a6%e5%be%b7%e6%9b%b4%e6%a3%92%e7%9a%84%e9%97%b4%e8%b0%8d/" rel="noopener noreferrer" target="_blank">

  </p>

    追踪荷兰逃犯
   </a>
   的时候使用的也是类似的方法，以及适用于 Instagram 的 Google Chrome 插件、和60多个 Twitter 用户。
  </p>
  <p class="graf graf--p">
   记者经常将 Facebook 视为一种强大的资源。虽然 Facebook 上“ 99％”的内容是宠物照片和 meme，但是，如果您可以对大量的个人帖子进行精细的分类，那么就能找到非常有用的信息。
  </p>
  <p class="graf graf--p">
   请注意：正由于 Facebook 含有大量的家庭生活和现实人际关系信息（比如宠物、旅行、家庭合影等），它也是社
   <strong class="markup--strong markup--p-strong">

   </strong>
   。

  <p class="graf graf--p">
   van Ess 创建了在线搜索工具
   <a class="markup--anchor markup--p-anchor" data-href="https://graph.tips/facebook.html" href="https://graph.tips/facebook.html" rel="noopener noreferrer" target="_blank">
    graph.tips
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.whopostedwhat.com/" href="https://www.whopostedwhat.com/" rel="noopener noreferrer" target="_blank">
    whopostedwhat.com
   </a>
   ，以便像他这样的调查人员拥有“快速的方法来过滤社交媒体”。
  </p>
  <p class="graf graf--p">
   对于专注于转瞬即逝的突发新闻或犯罪现场调查的年轻记者而言，他们可能熟悉 TweetDeck（据说是主流媒体记者必须熟练的标准工具这一），它能迅速查找目标相关的信息，为记者提供了一种便捷的追踪事件方法。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    在这里看到我们曾经分析过的做法，尤其是运算符《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bb%8e%e6%8e%a8%e7%89%b9%e4%b8%ad%e6%8c%96%e6%8e%98%e7%9c%9f%e7%9b%b8%e4%b8%8d%e9%9c%80%e8%a6%81%e5%a4%aa%e5%a4%8d%e6%9d%82%e7%9a%84%e5%b7%a5%e5%85%b7%ef%bc%9a%e4%b8%80%e4%b8%aa%e5%b8%b8%e7%94%a8/" href="https://www.iyouport.org/%e4%bb%8e%e6%8e%a8%e7%89%b9%e4%b8%ad%e6%8c%96%e6%8e%98%e7%9c%9f%e7%9b%b8%e4%b8%8d%e9%9c%80%e8%a6%81%e5%a4%aa%e5%a4%8d%e6%9d%82%e7%9a%84%e5%b7%a5%e5%85%b7%ef%bc%9a%e4%b8%80%e4%b8%aa%e5%b8%b8%e7%94%a8/" rel="noopener noreferrer" target="_blank">

   </li>

   </li>

  <p class="graf graf--p">
   您可以输入关键字，然后在“时间范围”部分中选择一个日期范围，以查找有关该关键字的所有此段时间内发布的帖子。
  </p>
  <p class="graf graf--p">
   单击感兴趣的 Facebook 个人资料，然后可以将该 Facebook 帐户的URL地址复制并粘贴到 whopostedwhat.com 上的搜索框中，以生成“ UID”编号。
  </p>
  <p class="graf graf--p">
   然后，当该用户提及该关键字时，可以在“直接来自/与之关联的帖子”部分中使用此UID来搜索实例。
  </p>
  <p class="graf graf--p">
   这些工具并非万无一失。有时搜索不会产生任何结果。有些GUI工具的确如此，这就是为什么您应该了解如何使用命令直接进行自定义的搜索，方法：
  </p>
  <ul class="postList">
