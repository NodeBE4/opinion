   <strong class="markup--strong markup--p-strong">
    —— 时间节点 ——
   </strong>

  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   如果你有很多钱，而且全部想要带走，那么时间就会相对较长 —— 你想带走的东西越多时间就会越长；如果你孑然一身，并且熟练操作，两周甚至一周内就可以消失了。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    原理
   </strong>
  </p>
  <p class="graf graf--p">
   先讲一个故事，是一个经典小说，Richard Connell 的《最危险的猎物》。
  </p>
  <p class="graf graf--p">
   故事的主人公是一个叫 Resford 的家伙。他被困在了一个小岛上；和他一起被困在这里的还有一个非常绅士但又非常疯狂的老者 —— 扎洛夫将军。
  </p>
  <p class="graf graf--p">
   将军的爱好是：狩猎活人。
  </p>
  <p class="graf graf--p">
   不久之后 Resford 就在森林里一路狂奔 —— 他成为了全世界最狗屎的野生动物园里的最新战利品。
  </p>
  <p class="graf graf--p">
   一开始他脑子里只有一个想法，那就是：和扎洛夫将军保持距离。为了实现这个目标，他一路狂奔，激励着他的是一种近似恐惧的东西。
  </p>
  <p class="graf graf--p">
   但是现在，他终于能冷静下来了。他停下了脚步，开始思考自己，思考自己的处境。
  </p>
  <p class="graf graf--p">
   他明白了，
   <strong class="markup--strong markup--p-strong">
    毫无目的地飞奔是完全徒劳的
   </strong>
   ；最终他将不可避免地和大海面对面。四面环海的岛屿是个天牢。
  </p>
  <p class="graf graf--p">
   他想明白了，
   <strong class="markup--strong markup--p-strong">
    他需要做的是给追击者留下一些可以寻觅的踪迹 —— 以将其引向错误的方向。
   </strong>
  </p>
  <p class="graf graf--p">
   Resford 偏离了崎岖的小径，投身于毫无路径可言的荒蛮之中。
  </p>
  <p class="graf graf--p">
   他精心设计了一系列圈套；他一再重复着自己的足迹。就像民间故事里总是能最终逃脱的狐狸……
  </p>
  <p class="graf graf--p">
   故事结尾不重要，重要的是：Resford 处心积虑的谋划最终取得了成效。他知道，要做到成功的销声匿迹，就必须：
   <strong class="markup--strong markup--p-strong">
    有一点狡猾、有一点欺骗，要尽你所能去除一切真实的痕迹，并同时，设置虚假的误导性痕迹，让你的追逐者偏离方向
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这就是完美隐身的关键思考方式。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这是一个三步策略：信息篡改、信息杜撰、信息重组
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    信息篡改
   </strong>
   ：首先就是我们在《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%83%85%e6%8a%a5%e5%92%8c%e5%8f%8d%e6%83%85%e6%8a%a5%e5%9f%ba%e6%9c%ac%e5%b7%a5%e5%85%b7%e6%89%8b%e5%86%8c%ef%bc%9a%e8%87%aa%e6%88%91%e4%ba%ba%e8%82%89%ef%bc%882/" href="https://www.iyouport.org/%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%83%85%e6%8a%a5%e5%92%8c%e5%8f%8d%e6%83%85%e6%8a%a5%e5%9f%ba%e6%9c%ac%e5%b7%a5%e5%85%b7%e6%89%8b%e5%86%8c%ef%bc%9a%e8%87%aa%e6%88%91%e4%ba%ba%e8%82%89%ef%bc%882/" rel="noopener noreferrer" target="_blank">
    自我人肉
   </a>
   》系列中介绍的知识。
  </p>
  <p class="graf graf--p">
   使用“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%87%aa%e6%88%91%e4%ba%ba%e8%82%89/" href="https://www.iyouport.org/%e8%87%aa%e6%88%91%e4%ba%ba%e8%82%89/" rel="noopener noreferrer" target="_blank">
    第一部分
   </a>
   ”中介绍的思考方式，以及“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%83%85%e6%8a%a5%e5%92%8c%e5%8f%8d%e6%83%85%e6%8a%a5%e5%9f%ba%e6%9c%ac%e5%b7%a5%e5%85%b7%e6%89%8b%e5%86%8c%ef%bc%9a%e8%87%aa%e6%88%91%e4%ba%ba%e8%82%89%ef%bc%882/" href="https://www.iyouport.org/%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%83%85%e6%8a%a5%e5%92%8c%e5%8f%8d%e6%83%85%e6%8a%a5%e5%9f%ba%e6%9c%ac%e5%b7%a5%e5%85%b7%e6%89%8b%e5%86%8c%ef%bc%9a%e8%87%aa%e6%88%91%e4%ba%ba%e8%82%89%ef%bc%882/" rel="noopener noreferrer" target="_blank">
    第二部分
   </a>
   ”中介绍的工具，尽可能找到所有与你自己有关的信息线索；
  </p>
  <p class="graf graf--p">
   然后运用包括“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a4%a7%e6%b8%85%e6%b4%97%ef%bc%9a%e5%8f%8d%e4%be%a6%e5%af%9f%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" href="https://www.iyouport.org/%e5%a4%a7%e6%b8%85%e6%b4%97%ef%bc%9a%e5%8f%8d%e4%be%a6%e5%af%9f%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" rel="noopener noreferrer" target="_blank">
    大清洗
   </a>
   ”在内的技术，尽可能清除掉这些信息；如果无法清除，那么就想办法给它们改头换面，
   <strong class="markup--strong markup--p-strong">
    目标是追踪者无法据此对你定位
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    信息杜撰
   </strong>
   ：也就是战略性欺骗部分了。给追踪者设置虚假的线索路径，尽可能地为自己争取时间，并同时无限地消耗追踪者的财力和精力。
  </p>
  <p class="graf graf--p">
   必须要巧妙。要知道大部分追踪者会花钱聘请专业侦查团队，或者他们本身就是政府当局，拥有自己强大的情报能力。
  </p>
  <p class="graf graf--p">
   但是
   <strong class="markup--strong markup--p-strong">
    不要被专业头衔吓住
   </strong>
   ，因为他们和你一样都是人，人类天然具备的思维弱点他们身上都有。
  </p>
  <p class="graf graf--p">
   心理战术的关键就在于：谁先拿下对手的弱点。
  </p>
  <p class="graf graf--p">
   有多少种追踪方式就有多少种反追踪方式；你要做的就是无限地消耗你的对手，直到对方因枯竭而放弃。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    信息重组
   </strong>
   ：开创新的生活，享受私密的生活，隐秘的住所，不留下一丝线索。
  </p>
  <p class="graf graf--p">
   不论是《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%a7%92%e8%89%b2%e6%89%ae%e6%bc%94%ef%bc%9a%e8%a1%8c%e5%8a%a8%e8%80%85%e5%88%9a%e9%9c%80video/" href="https://www.iyouport.org/%e8%a7%92%e8%89%b2%e6%89%ae%e6%bc%94%ef%bc%9a%e8%a1%8c%e5%8a%a8%e8%80%85%e5%88%9a%e9%9c%80video/" rel="noopener noreferrer" target="_blank">
    角色扮演
   </a>
   》还是《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%bf%ab%e9%80%9f%e5%88%9b%e5%bb%ba%e4%b8%80%e4%b8%aa%e5%81%87%e4%ba%ba%ef%bc%9f%e7%ae%80%e6%98%93%e7%89%88%e5%88%86%e8%ba%ab%e6%9c%af%e2%80%8a-%e2%80%8a%e8%b0%83%e6%9f%a5%e4%ba%ba/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%bf%ab%e9%80%9f%e5%88%9b%e5%bb%ba%e4%b8%80%e4%b8%aa%e5%81%87%e4%ba%ba%ef%bc%9f%e7%ae%80%e6%98%93%e7%89%88%e5%88%86%e8%ba%ab%e6%9c%af%e2%80%8a-%e2%80%8a%e8%b0%83%e6%9f%a5%e4%ba%ba/" rel="noopener noreferrer" target="_blank">
    袜子木偶
   </a>
   》都可以在数字世界帮你找到安全之地。而在现实中，您还需要一些其他东西。
  </p>
  <p class="graf graf--p">
   三步完成就是自由的世界。
  </p>
  <p class="graf graf--p">
   由于篇幅原因，本文只介绍第一步；后面的部分将另开新篇 —— 请注意，如上所述，我们已经介绍过在线隐身的操作方式，本系列重点关注的是在现实中需要做哪些事。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第一步：信息篡改
   </strong>
  </p>
  <p class="graf graf--p">
   任何一位不想要生活在1984里的人都应该学会信息篡改。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “知己知彼，百战不殆”。信息篡改背后的原理就在这句话中：
   <strong class="markup--strong markup--p-strong">
    抢在对手的前面了解自己的弱点，以及对手的意图，才能抢得先机，防患于未然
   </strong>
   。
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    **辨别并销毁一切外在的、与你有关的信息**
   </p>
  </blockquote>
  <p class="graf graf--p">
   如果不能销毁，你可以与信息提供者取得联系，随便撒个谎，告诉他们档案中的信息错了，你想“更正”信息。
  </p>
  <p class="graf graf--p">
   再一次，由于肯定不会有很多人去把关于自己的真相改为假象，于是理论上很多公司部门不会怀疑你的说法。
  </p>
  <p class="graf graf--p">
   其目的在于
   <strong class="markup--strong markup--p-strong">
    混淆视线、误导
   </strong>
   、将任何一个试图监视你的人都引向其他方向。
  </p>
  <p class="graf graf--p">
   效果
   <strong class="markup--strong markup--p-strong">
    最好
   </strong>
   的时候，这种做法可以让追踪者裹步难行；效果
   <strong class="markup--strong markup--p-strong">
    最差
   </strong>
   的时候，追踪者也需要反复找借口、用尽一切搜索手段，才能找到你真正的位置。这种搜索方法既耗时又费钱。如果你幸运，你的对手会因此被掏空，耐心耗尽。
  </p>
  <p class="graf graf--p">
   一个细节就是：
   <strong class="markup--strong markup--p-strong">
    别在法律的框架内思考问题
   </strong>
   。这是维权人士经常出现的错误。他们认为“这是非法的”，于是对手“做不到”。但对于反追踪来说，绝对不是如此。
  </p>
  <p class="graf graf--p">
   不论你的对手是政府当局、情报部门、还是私人侦探，在此都一样，在追踪的操作中违法的可能性非常高，因为违法是他们的优势。
  </p>
  <p class="graf graf--p">
   而作为反追踪者，
   <strong class="markup--strong markup--p-strong">
    你的工作就是尽可能发挥想象力去预测你的对手下一步会做什么，然后提前下手，给对方挖坑
   </strong>
   。
  </p>
  <p class="graf graf--p">
   上述回顾的自我人肉系列（2个）是全面且专业的，足够您使用。如果您对技术一窍不通，可以从最简单的开始。
  </p>
  <p class="graf graf--p">
   比如美国的 Zabasearch 或者中国的“找人网”，这些对方都是很
   <strong class="markup--strong markup--p-strong">
    高风险
   </strong>
   的 —— 使用独立的设备+VPN/Tor+虚拟机，去那里搜索自己，记录下找到能找到的任何东西，写在纸上而不是在线保存，然后逐一破解或混淆它们。
  </p>
  <p class="graf graf--p">
   完成后别忘了把那张纸烧掉。
  </p>
  <p class="graf graf--p">
   对中国的找人网不太了解，但美国的 Zabasearch 和 Intelius 是非常厉害的，甚至能挖到某人从未公开过的手机号和座机号码；而且信息五花八门，绝对是追踪的
   <strong class="markup--strong markup--p-strong">
    利器。
   </strong>
  </p>

   </noscript>
  </p>
  <p class="graf graf--p">
   在美国你无法完全屏蔽个人公开信息，除非你有切实的证据受到了人身威胁并据此去申请法庭指令。想必在中国也差不多。但至少你能做到
   <strong class="markup--strong markup--p-strong">
    为你的对手增加侦查的难度
   </strong>
   。
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    **去每一个你能找到的在线数据库和电话簿看一眼，找找看上面多少关于你的信息。这就是最基本的自我人肉**
   </p>
  </blockquote>
  <p class="graf graf--p">
   如果您不熟悉加密技术、虚拟机、彻底清除指纹的诸多操作，那么建议您在整个过程中使用纸和笔记录信息和进行思考，而不是使用在线工具。用完后将所有纸烧掉。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    搜索自己的名字，以及名字的各种变体，名字加地址，名字加公司，等等；这就是追踪者一开始投入时会使用的方法
   </strong>
   。尽可能删除你能找到的任何东西。
  </p>
  <p class="graf graf--p">
   然后，为上述每一个搜索条件都创建一个谷歌提醒，一旦有人在互联网上输入了某个具体的搜索条件相关信息，谷歌就会提醒你。你也可以使用这个方法确保在信息篡改后没有人再重新添加关于你的信息。
  </p>
  <p class="graf graf--p">
   搜索条件就是姓名、位置、工作单位以及不同的组合，比如这样：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    张三疯
   </li>
   <li class="graf graf--li">
    张三·疯/峰/丰/风…
   </li>
   <li class="graf graf--li graf--startsWithDoubleQuote">
    “张三疯”
   </li>
   <li class="graf graf--li">
    张三疯 + 北京
   </li>
   <li class="graf graf--li">
    张三疯 + ABC有限公司
   </li>
   <li class="graf graf--li">
    等等
   </li>
  </ul>
  <p class="graf graf--p">
   如果电子邮箱是“
   <a class="__cf_email__" data-cfemail="43392b222d2430222d25262d240325222826222727312630306d202c2e" href="/cdn-cgi/l/email-protection">
    [email protected]
   </a>
   ”，你可以这样：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    张三疯 AT fakeaddress
   </li>
   <li class="graf graf--li">
    （张三疯）（fakeaddress）
   </li>
   <li class="graf graf--li">
    zhang (DOT) sanfeng (AT) fakeaddress (DOT) com
   </li>
   <li class="graf graf--li">
    等等
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    确保删除所有照片，卸载社交媒体；在卸载之前取消掉对所有与你真实相关的人的关注；清除掉所有与你主页相关的链接。
   </strong>
  </p>
  <p class="graf graf--p">
   如果您的朋友很多，慌乱之中有可能遗漏，没关系，请使用
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/DonEvansWm/status/1163671192546320384" href="https://twitter.com/DonEvansWm/status/1163671192546320384" rel="noopener noreferrer" target="_blank">
    这里的2个工具
   </a>
   查找自己可被搜索到的社交关系网。然后清除它们。
  </p>
  <p class="graf graf--p">
   就算你逃出本国，找到你也不是那么难；因为这是社交网络时代，人们的亲朋好友同事熟人都在迫不及待地在互联网上公布含有你的照片；这绝对是
   <strong class="markup--strong markup--p-strong">
    你的灾难
   </strong>
   。
  </p>
  <p class="graf graf--p">
   请务必告诫自己周边的人要多一个心眼。最好的是你自己不要亲自公开任何真实的东西。
  </p>
  <p class="graf graf--p">
   前文中强调过，追踪者可以完全不使用数字技术也能挖到你，因为
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
    社交工程
   </a>
   学是个好东西。
  </p>
  <p class="graf graf--p">
   想想看，任何地点还有哪些关于你的个人信息？一般来说社交工程师会最先从以下这些地方寻找线索：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    电话公司
   </li>
   <li class="graf graf--li">
    移动公司
   </li>
   <li class="graf graf--li">
    家居商城
   </li>
   <li class="graf graf--li">
    电力公司
   </li>
   <li class="graf graf--li">
    健身房
   </li>
   <li class="graf graf--li">
    有线电视服务商
   </li>
   <li class="graf graf--li">
    卫星服务商
   </li>
   <li class="graf graf--li">
    互联网服务商
   </li>
   <li class="graf graf--li">
    租贷店和租贷服务
   </li>
   <li class="graf graf--li">
    杂货铺和超市（会员部）
   </li>
   <li class="graf graf--li">
    旅游公司
   </li>
   <li class="graf graf--li">
    沙龙和校友会
   </li>
   <li class="graf graf--li">
    银行
   </li>
   <li class="graf graf--li">
    信用卡公司
   </li>
  </ul>
  <p class="graf graf--p">
   你在上述这些地方是否有账户？给他们打电话，删除自己的账户，最好是能篡改部分信息。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    下一步
   </strong>
  </p>
  <p class="graf graf--p">
   进行到这里很多人也许会觉得非常凌乱，因为人们对谎言的记忆需要很大的努力。不过没关系，你完全可以提升记忆力；那就是下一部分的主题：关于如何将所有漏洞都堵上。
  </p>
  <p class="graf graf--p">
