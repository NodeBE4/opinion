   </noscript>
  </p>
  <p class="graf graf--p">
   越来越多的州和市立法机构正在采取措施遏制政府对面部识别技术的使用。
  </p>
  <p class="graf graf--p">
   但分析师表示，即使面临新的限制，该技术也很可能无处不在。
  </p>
  <p class="graf graf--p">
   延伸：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a6%81%e6%ad%a2%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%ef%bc%8c%e7%9c%9f%e7%9a%84%e5%8f%af%e4%bb%a5%e5%90%97%ef%bc%9f/" href="https://www.iyouport.org/%e7%a6%81%e6%ad%a2%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%ef%bc%8c%e7%9c%9f%e7%9a%84%e5%8f%af%e4%bb%a5%e5%90%97%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    禁止面部识别，真的可以吗？
   </a>
   》《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a6%81%e6%ad%a2%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e5%8f%af%e4%bb%a5%e5%81%9a%e5%88%b0%e4%bb%80%e4%b9%88%e3%80%81%e4%b8%8d%e8%83%bd%e5%81%9a%e5%88%b0%e4%bb%80%e4%b9%88/" href="https://www.iyouport.org/%e7%a6%81%e6%ad%a2%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e5%8f%af%e4%bb%a5%e5%81%9a%e5%88%b0%e4%bb%80%e4%b9%88%e3%80%81%e4%b8%8d%e8%83%bd%e5%81%9a%e5%88%b0%e4%bb%80%e4%b9%88/" rel="noopener noreferrer" target="_blank">
    禁止面部识别可以做到什么、不能做到什么
   </a>
   》
  </p>
  <p class="graf graf--p">
   继旧金山于5月份在全市范围内实施禁令后，马萨诸塞州萨默维尔于6月27日成为美国第二个禁止使用该技术的城市。
  </p>
  <p class="graf graf--p">
   隐私倡导者引用了“面部监视”所引发的道德问题，包括政府滥用的可能性及其在种族和性别方面的不平等表现。
  </p>
  <p class="graf graf--p">
   Forrester Research 的安全和风险团队主管 Merritt Maxim 表示，尽管面部识别有了这些新的限制，但是，
   <strong class="markup--strong markup--p-strong">
    这项技术已经变得非常普遍，足以让国家和地方政府绕避禁令。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “该技术已经开发出来了，它已经被部署用于各种不同的用例。肯定会继续下去。”
  </p>
  <p class="graf graf--p">
   根据
   <a class="markup--anchor markup--p-anchor" data-href="https://www.americaunderwatch.com/" href="https://www.americaunderwatch.com/" rel="noopener noreferrer" target="_blank">
    May Georgetown 法律中心的报告
   </a>
   ，底特律和芝加哥已经花费了数百万美元用于先进的面部监视系统；华盛顿特区、奥兰多和纽约市都启动了试点计划。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    即使在警务方面，很多美国人也对使用面部识别技术“感到满意”。（？？）
   </strong>
  </p>
  <p class="graf graf--p">
   数据创新中心今年进行的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.datainnovation.org/2019/01/survey-few-americans-want-government-to-limit-use-of-facial-recognition-technology-particularly-for-public-safety-or-airport-screening/" href="https://www.datainnovation.org/2019/01/survey-few-americans-want-government-to-limit-use-of-facial-recognition-technology-particularly-for-public-safety-or-airport-screening/" rel="noopener noreferrer" target="_blank">
    一项调查显示
   </a>
   ，只有26％的受访者认为政府应严格限制面部监控。
  </p>
  <p class="graf graf--p">
   支持限制该技术的人就更少了。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果您仍然拥有[面部识别]的一些主要消费设备，它就必须成为讨论的一部分，” Maxim 说，注意到越来越多的消费者习惯于使用面部识别登机或每天用来解锁他们的 iPhone。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “目前还不够可靠”
   </strong>
  </p>
  <p class="graf graf--p">
   根据最近的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iflexion.com/blog/facial-recognition-software" href="https://www.iflexion.com/blog/facial-recognition-software" rel="noopener noreferrer" target="_blank">
    市场报告
   </a>
   ，美国公共部门仍然是面部识别供应商向前发展的关键市场。
  </p>
  <p class="graf graf--p">
   持续的需求使得私营公司有责任考虑道德问题。比如亚马逊，仍然向政府和执法部门出售他们的技术。
  </p>
  <p class="graf graf--p">
   Axon 是美国警用监视摄像机的主要供应商，该公司已经宣布它不会销售使用“面部匹配技术”的产品 —— 或为其检测到的面孔分配数字身份 的监视产品 —— 在其道德委员会关于该技术的批评报告发布之后。
  </p>
  <p class="graf graf--p">
   但 Axon 不排除未来将该技术商业化。
  </p>
  <p class="graf graf--p">
   Maxim 表示，虽然 Axon 的决定反映了私营部门的一些参与者“从最初采取更积极的面部识别方法中退却”，但是，其他因素也可能正在发挥作用。
  </p>
  <p class="graf graf--p">
   他说，由于警察部门难以负担得起并安全地存储他们收集的巨大数据，他已经看到执法机构对身体摄像头的需求逐渐减少 —— 制造更先进的摄像头，例如采用面部监视的摄像头，是想不通的。
  </p>
  <p class="graf graf--p graf--startsWithSingleQuote">
   <strong class="markup--strong markup--p-strong">
    ‘严重关切’
   </strong>
  </p>
  <p class="graf graf--p">
   无论是什么原因，警察机构都已经制定了自己的面部识别技术禁令，即使没有州或市政府的裁决。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “奥斯汀警察局目前不使用或访问面部识别技术，并决定不这样做，直到我们征求社区对技术界的反馈并评估其对我们部门和社区的影响，”奥斯汀警察局新闻官在一份声明中说。
  </p>
  <p class="graf graf--p">
   Maxim 表示，他认为面部识别和生物识别技术面临的最严峻挑战不是来自禁令，而是可能来自法庭系统。
  </p>
  <p class="graf graf--p">
   伊利诺伊州2008年生物识别信息隐私法引发了一系列针对私营部门使用生物识别技术的法律挑战，其中包括针对 Facebook 收集其用户生物识别信息的
   <a class="markup--anchor markup--p-anchor" data-href="https://epic.org/amicus/bipa/patel-v-facebook/" href="https://epic.org/amicus/bipa/patel-v-facebook/" rel="noopener noreferrer" target="_blank">
    持续案件
   </a>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “现在，我认为这将带来更多的挑战，也会影响其他可能制定与生物识别数据相关的隐私法的国家，” Maxim 说。
  </p>
  <p class="graf graf--p">
   在萨默维尔，活动人士对这座城市的禁令表示赞赏，并表示这是对当地机构面临全国性挑战的提醒。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我对面部识别技术的使用表示严重担忧，我赞扬市议会采取这一重要行动，禁止在我们的社区中获取或使用此类技术，”Somerville 市长 Joseph Curtatone 在给 StateScoop 的一份声明中说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “作为领导者和公共安全官员，我们的工作是保护和服务我们的社区，但我不相信面部识别技术是一种有效的工具，我也不支持它在萨默维​​尔市的使用。”
  </p>
  <p class="graf graf--p">
   尽管如此，
   <strong class="markup--strong markup--p-strong">
    这种孤立的禁令更多地是面部识别监视进步的标志，而不是对其命运的威胁。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我认为监管总是会能发挥作用的，但我认为监管机构总是落后于技术发展，” Maxim 说。
  </p>
  <p class="graf graf--p">
   他说，有可能市场可能会暂时受到影响，但是最终，“精灵还是从瓶子里出来了。” ⚪️
  </p>
