  <p class="graf graf--p">
   根据英国隐私监督机构 Privacy International 的最新报告，至少有两个月经跟踪应用程序 Maya 和 MIA Fem 正在与 Facebook 和其他实体分享用户性健康的详细信息。
  </p>
  <p class="graf graf--p">
   在某些情况下，⚠️这些数据包括：用户上次发生性行为的时间、使用的避孕方式、情绪状态、以及是否排卵。
  </p>
  <p class="graf graf--p">
   在雇主、保险公司和广告商可以使用数据广泛区分或定位每个人的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89/" href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89/" rel="noopener noreferrer" target="_blank">
    监视资本主义时代
   </a>
   ，这些调查结果，
   <strong class="markup--strong markup--p-strong">
    再一次
   </strong>
   ，引发了对大多数私人信息安全性的严重质疑。
  </p>
  <p class="graf graf--p">
   这些信息通过 Facebook 软件开发套件与社交媒体巨头共享，该软件开发套件允许开发人员为特定操作系统创建应用程序、跟踪分析，并通过 Facebook 的广告网络将其应用程序货币化。
  </p>
  <p class="graf graf--p">
   Privacy International 发现，一旦用户在手机上安装了应用并打开了应用程序，
   <strong class="markup--strong markup--p-strong">
    Maya 和 MIA 就开始与 Facebook 共享数据，甚至在签署隐私政策之前
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Facebook 发言人 Joe Osborne 表示，“广告商无法访问”这些应用程序共享的敏感健康信息。他在一份声明中表示，当广告商按兴趣选择目标用户时，Facebook 的广告系统“不会利用在其他应用或网站上收集到的关于人们的活动信息”。
  </p>
  <p class="graf graf--p">
   Maya 和 MIA 等跟踪应用程序已经成为有趣的、友好的伙伴，可以深入了解生育和怀孕这个令人生畏的世界。
  </p>
  <p class="graf graf--p">
   并且，它们还可用于更一般性的跟踪健康、情绪和其他私密数据。
  </p>
  <p class="graf graf--p">
   但是，许多这类应用程序不受大多数健康类相关数据保护规则的限制。健康跟踪数据是极为敏感的私人数据。
  </p>
  <p class="graf graf--p">
   这引起了明显的隐私问题，⚠️因为一些这类应用程序已经成为雇主和医疗保险公司强有力的监控工具，雇主打着“健康”的旗号推动收集有关其雇员私人生活的更多数据。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e6%ad%a3%e5%9c%a8%e4%b8%ba%e4%b8%80%e4%b8%aa%e8%b4%b4%e8%ba%ab%e8%80%81%e5%a4%a7%e5%93%a5%e5%8a%aa%e5%8a%9b%e5%b7%a5%e4%bd%9c%ef%bc%9a%e5%9c%b0%e7%8b%b1%e7%9a%84%e6%96%b0%e5%b8%b8%e6%80%81/" href="https://www.iyouport.org/%e4%bd%a0%e6%ad%a3%e5%9c%a8%e4%b8%ba%e4%b8%80%e4%b8%aa%e8%b4%b4%e8%ba%ab%e8%80%81%e5%a4%a7%e5%93%a5%e5%8a%aa%e5%8a%9b%e5%b7%a5%e4%bd%9c%ef%bc%9a%e5%9c%b0%e7%8b%b1%e7%9a%84%e6%96%b0%e5%b8%b8%e6%80%81/" rel="noopener noreferrer" target="_blank">
     你正在为一个贴身老大哥努力工作：地狱的新常态
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   此外，正如隐私国际组织所指出的那样，这些高度敏感的数据正在被更广泛地共享，以用户几乎无法想象的程度。
  </p>
  <p class="graf graf--p">
   这类事件已多次被曝光。因此，许多女性表示他们已经制定了使用这类应用程序的策略，而不会泄露所有最敏感的信息。
  </p>
  <p class="graf graf--p">
   这些策略包括：使用假名，仅记录分散的细节，甚至输入不正确的数据。
  </p>
  <p class="graf graf--p">
   用户和专家都担心这些数据可能会暴露在安全漏洞中，或者
   <strong class="markup--strong markup--p-strong">
    被雇主和保险公司用来通过增加保险费、或拒绝提供领导职位以歧视女性
   </strong>
   。
  </p>
  <p class="graf graf--p">
   精神病学家、非营利性患者隐私权创始人 Deborah Peel 表示，人们希望他们的健康数据受到与医生办公室健康信息相同程度的法律保护，但许多应用程序不受相同规则的约束。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “大多数人都希望自己做出关于性生活的知识的决定，关于它是否可以被共享，然而，
   <strong class="markup--strong markup--p-strong">
    现在我们没有能力做到这一点
   </strong>
   ”，Peel 说。
  </p>
  <p class="graf graf--p">
   Facebook是全球最大的社交媒体平台，拥有12亿日常用户，要求用户信任它的信息比过去任何时候都要多。
  </p>
  <p class="graf graf--p">
   9月，该公司在美国推出了 FacebookDating，这是一项红娘服务，根据偏好、兴趣和 Facebook 的活动模式，帮用户表达潜在的爱情。
  </p>
  <p class="graf graf--p">
   与此同时，Facebook 近年来一直因涉及错误信息、虚假账户和违反信任的丑闻而受到抨击。包括著名的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" rel="noopener noreferrer" target="_blank">
    剑桥分析公司
   </a>
   丑闻。
  </p>
  <p class="graf graf--p">
   Facebook表示其服务条款禁止应用程序开发人员共享健康类的高度敏感数据，并且 Facebook 说它已与 Maya 和 MIA 联系，以便在声明中通知他们可能违反了哪些条款。
  </p>
  <p class="graf graf--p">
   Facebook还表示，虽然它有检测系统可以自动检测和删除应用程序共享信息中包含的社会保险号和密码等信息，但仍有必要“检测和阻止共享更多的敏感信息” ……
  </p>
  <p class="graf graf--p">
   开发 Maya 的 Plackal Tech 在其对 Privacy International 的声明中称，它将从新版本的服务中删除 Facebook 软件开发工具包。
  </p>
  <p class="graf graf--p">
   MIA背后的公司 Mobapp Development Limited 没有回应，该公司没有立即发表评论。
  </p>
  <p class="graf graf--p">
   ⚠️请注意每一次被曝光后他们都能立即做出专业级的公关，然后再继续被曝光？您是否还能数得清究竟出现过多少次曝光-公关？
  </p>
  <p class="graf graf--p">
   永远记得监视资本主义的本性。
  </p>
  <p class="graf graf--p">
   不要相信他们，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%a7%92%e8%89%b2%e6%89%ae%e6%bc%94%ef%bc%9a%e8%a1%8c%e5%8a%a8%e8%80%85%e5%88%9a%e9%9c%80video/" href="https://www.iyouport.org/%e8%a7%92%e8%89%b2%e6%89%ae%e6%bc%94%ef%bc%9a%e8%a1%8c%e5%8a%a8%e8%80%85%e5%88%9a%e9%9c%80video/" rel="noopener noreferrer" target="_blank">
    不要把您的任何真实信息放在互联网上
   </a>
   ，这才是最靠谱的解决方案。⚪️
  </p>
