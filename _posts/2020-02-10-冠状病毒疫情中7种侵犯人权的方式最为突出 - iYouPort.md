   对流行病的反应有可能影响数百万人的人权。首先是健康权，但是其他多项权利也处于危险之中。

  <p class="graf graf--p graf--startsWithDoubleQuote">
   “侵犯人权行为阻碍而不是促进了对公共卫生紧急事件的应对，并削弱了其效率”，大赦国际说。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、早期审查
   </strong>
  </h3>
  <p class="graf graf--p">
   2019年12月下旬，武汉的医生与同事分享了他们对新疫情的恐惧，症状类似于2002年在中国南部开始爆发的严重急性呼吸道综合症（SARS）。结果是。他们因 “散布谣言’’ 而被当地当局噤声并受到处罚。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “中国的医学专家试图对这种病毒发出警报。如果政府不试图将真实的危险压制到最低，世界可能有机会更及时地应对病毒的传播，” 大赦国际的 Nicholas Bequelin 说。
  </p>
  <p class="graf graf--p">
   一个月后，最高人民法院在网上发布了一篇文章，质疑武汉当局的决定。对于医生们来说，这被普遍认为是正确的。
  </p>
  <p class="graf graf--p">
   然而，中国政府的最高领导层似乎是尽了最大的努力以试图压低疫情爆发的真实严重程度，这一点从中国对世界卫生组织的积极游说中可以看出，以阻止宣布疫情爆发是全球卫生紧急状态。
  </p>
  <figure aria-describedby="caption-attachment-12771" class="wp-caption aligncenter" id="attachment_12771" style="width: 1456px">

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、健康权
   </strong>
  </h3>
  <p class="graf graf--p">
   武汉的医疗体系现在不堪重负，医疗机构和卫生专业人员正努力应对疫情不断扩大的规模。
  </p>
  <p class="graf graf--p">
   排了几个小时的长队后许多患者仍然被拒之门外。就连必要的诊断测试都不能满足所有需求者。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “中国必须确保在武汉和其他地方，所有受冠状病毒感染的人都能获得适当的卫生保健。遏制这种流行病很重要，但预防和治疗也很重要，这就是为什么健康权应成为应对措施不可或缺的一部分。” Nicholas Bequelin 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “尽管世卫组织对中国不断给予赞誉，
   <strong class="markup--strong markup--p-strong">
    但事实是，政府的反应一直是 —— 而且仍然是 —— 非常有问题的
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   当地媒体报道说，由于公共交通中断，人们无法迅速去医院，在某些情况下，甚至无法从家中转移死难者的尸体。
  </p>
  <p class="graf graf--p">
   《世界人权宣言》所保障的健康权规定了获得医疗保健的权利、获得信息的权利、禁止在提供医疗服务时受到歧视，免受未经同意的医疗的自由，以及其他重要保证。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、审查制度仍在继续
   </strong>
  </h3>
  <p class="graf graf--p">
   中国当局坚持控制新闻叙述和扼杀当局不喜欢的报道，继续推动对有时关于该病毒的合法信息的审查。
  </p>
  <p class="graf graf--p">
   自危机开始以来，当局已经审查了许多文章，其中包括中国主流媒体的文章，例如《北京青年报》和《财经》。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12772 jetpack-lazy-image" data-lazy-sizes="(max-width: 834px) 100vw, 834px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/01.png?resize=834%2C1140&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/01.png?w=834&amp;ssl=1 834w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/01.png?resize=219%2C300&amp;ssl=1 219w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/01.png?resize=749%2C1024&amp;ssl=1 749w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/01.png?resize=768%2C1050&amp;ssl=1 768w" data-recalc-dims="1" height="1140" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/01.png?resize=834%2C1140&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="834"/>

  </p>
  <p class="graf graf--p">
   Nicholas Bequelin 表示：“中国当局正冒着隐瞒信息的风险，可能会阻碍医学界解决冠状病毒问题，甚至可能难以帮助人们保护自己免受病毒的侵害。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “某些信息并非所有人都能获得的事实增加了冠状​​病毒造成伤害的风险，并延迟了有效的响应。”
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、活动人士受到骚扰和恐吓
   </strong>
  </h3>
  <p class="graf graf--p">
   试图在社交媒体上分享有关冠状病毒信息的人们也遭到了中国政府的攻击。
  </p>
  <p class="graf graf--p">
   例如，律师和公民记者陈秋实报告说，他们在武汉医院发布录像后受到当局的骚扰。
  </p>
  <p class="graf graf--p">
   武汉居民方斌在发布一段视频以示冠状病毒受害者的尸体后被当局带走。
  </p>
  <p class="graf graf--p">
   Nicholas Bequelin 说：“尽管驳斥有关该病毒的虚假陈述至关重要，但是，封锁有关该病毒的合法新闻和社交媒体内容却无济于事（只会起到相反作用）。”
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、其他国家也在做同样的事 —— 试图打击所谓的 “假新闻”
   </strong>
  </h3>
  <p class="graf graf--p">
   由于该病毒已从中国传播到东南亚的邻国，各个国家寻求控制报道范围的审查趋势也在增加。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12773 jetpack-lazy-image" data-lazy-sizes="(max-width: 842px) 100vw, 842px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/02.png?resize=842%2C764&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/02.png?w=842&amp;ssl=1 842w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/02.png?resize=300%2C272&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/02.png?resize=768%2C697&amp;ssl=1 768w" data-recalc-dims="1" height="764" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/02.png?resize=842%2C764&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="842"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12773" data-recalc-dims="1" height="764" sizes="(max-width: 842px) 100vw, 842px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/02.png?resize=842%2C764&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/02.png?w=842&amp;ssl=1 842w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/02.png?resize=300%2C272&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/02.png?resize=768%2C697&amp;ssl=1 768w" width="842"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在马来西亚、泰国和越南的人们也因发布有关疫情的所谓 “假新闻” 而被捕或被罚款。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “政府必须防止虚假信息，并提供及时、准确的健康指导。但是，对言论自由的任何限制都必须是合理的、合法和必要的。” Nicholas Bequelin 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果东南亚和其他地方的政府可以从中国处理冠状病毒危机的错误中吸取教训，那就是：限制信息和以所谓的 ‘稳定’ 为名义结束辩论只会带来严重的风险，并且可能会适得其反。”
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    6、歧视和仇外心理
   </strong>
  </h3>
  <p class="graf graf--p">
   据媒体报道，来自武汉的人 —— 甚至是那些没有症状的人 —— 也被酒店拒绝，被封锁在自己的公寓里，其个人信息在中国在线泄漏。
  </p>
  <p class="graf graf--p">
   在其他国家也有反亚裔仇外心理的广泛报道。韩国、日本和越南的一些餐馆拒绝接受中国客人，而一群抗议者警告中国客人必须离开印度尼西亚的一家旅馆。法国和澳大利亚的报纸在报道危机时也被指控种族主义。
  </p>
  <p class="graf graf--p">
   世界各地的亚洲社区都纷纷退缩，而 Twitter 标签 #JeNeSuisPasUnVirus（我不是病毒）在法国风靡一时。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “世界各国政府应采取态度制止针对中国和亚洲血统的人的种族主义。世界抗击这场疫情暴发的唯一方法是跨界团结与合作”，Nicholas Bequelin 说。
  </p>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    7、边境管制和隔离必须合理和适当
   </strong>
  </h3>
  <p class="graf graf--p">
   为了应对这种病毒，许多国家对来自中国或其他亚洲国家的人关闭了大门，而其他国家则采取了严格的隔离措施。
  </p>
  <p class="graf graf--p">
   澳大利亚政府已将数百名澳大利亚人送往圣诞岛的一个移民拘留中心，澳大利亚医学协会此前将该地的治疗条件描述为 “不人道的’’，因为难民被拘留在那里会造成精神和身体上的双重痛苦。
  </p>
  <p class="graf graf--p">
   巴布亚新几内亚对所有其他亚洲国家的人关闭了边界，不仅限于确诊感染冠状病毒的人。在巴布亚新几内亚当局的指示下，一些当地学生被禁止登机回家，从而使他们滞留在菲律宾。
  </p>
  <p class="graf graf--p">
   限制迁徙自由权的隔离，只有在合理的、有时间限制的、为合法目的而采取的、严格必要的前提下，尽可能以自愿且以非歧视性的方式，才可根据国际法辩解。隔离必须以安全和尊重的方式进行。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    被隔离的人们的权利必须得到尊重和保护，包括确保获得医疗保健、食物和其他必需品
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “政府面临严峻的形势，必须采取措施，既防止冠状病毒传播，又要确保受影响的人们能够获得所需的医疗保健协助，” Nicholas Bequelin 说。
  </p>
