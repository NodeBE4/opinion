     <p class="graf graf--p graf-after--figure" id="8559">
      说到消失你最先想到的是什么？躲债、犯罪、黑社会追杀……不不，我们想说的是，随时隐藏自己是每个人的权利，不管你为了好玩还是真的有重要需求。然而在当下这个监视时代，这项权利正在被剥夺。很多人之所以不重视隐私只是因为他们并不了解自己被抢走了哪些权利。
     </p>
     <p class="graf graf--p graf-after--p" id="8a51">
      曾经一家大牌媒体在线上支付刚刚兴起的时候做了个测试，“如何在一天之内不使用现金而保持生活正常”。这个当初很热门的测试及相关报道实际上是非常可笑的，因为它的动机严重错误 — — 人们需要知道的并不是没有现金对生活造成了哪些不便，而应该是，自己的隐私权被剥夺的事实。希望该媒体只是傻，而不是在
      <strong class="markup--strong markup--p-strong">
       故意
      </strong>
      偷换主题。
     </p>
     <p class="graf graf--p graf-after--p" id="a728">
      说实话，最近十年以来你都很难做到完全消失了，今后只会越来越难。基本上，
      <strong class="markup--strong markup--p-strong">
       你是否能消失取决于想找到你的人的技术能力
      </strong>
      ，如果你得罪的是地痞流氓街头混混，甩开他们似乎相对容易一些（要是意大利黑手党的话就算了）如果你得罪的人是当权者，想躲起来恐怕要费点力气了。
     </p>
     <p class="graf graf--p graf-after--p" id="ace5">
      不过我建议你试试，建议更多人都来试试，至少能让自己明白这个时代有多可怕，以及我们为什么要坚决抵制大数据监视社会。
     </p>
     <p class="graf graf--p graf-after--p" id="f345">
      想加入这个游戏吗？你需要了解以下几个步骤：
     </p>
     <p class="graf graf--p graf-after--p" id="102e">
      首先，要知道，真的做到消失是
      <strong class="markup--strong markup--p-strong">
       需要一段时间
      </strong>
      的，也就是说你无法立即做到这点。需要慢慢摆脱你的旧生活，这意味着随着时间的推移越来越少地接触朋友和家人。这一过程意味着逐渐让所有熟悉你的人淡忘你，直到没有人会忽然想起你以至于想要联系你。
     </p>
     <p class="graf graf--p graf-after--p" id="3586">
      这段时间内你可能需要
      <strong class="markup--strong markup--p-strong">
       撒谎
      </strong>
      ，为了完成伟大的实验，尽可能克服一下吧。要知道，之后的日子都要通过撒谎来维系，你必须习惯它。
     </p>
     <p class="graf graf--p graf-after--p" id="0afb">
      第二，
      <strong class="markup--strong markup--p-strong">
       你使用 Visa，PayPal 和 Venmo 的日子已经结束了
      </strong>
      。你现在是一个只会用现金的人，你需要把所有的钱都从银行里拿出来，做到无银行账户。
     </p>
     <p class="graf graf--p graf-after--p" id="84cb">
      这是一个需要缓慢进行的过程，以免引起注意。每次提取的钱不要很多，也不要数额相同，造成一个印象，即这是个自然的过程，你是消费者，而不是转款。否则很快会有人推测你的动机。（如果你是个富人，请放弃这个游戏吧，要么放弃你的资产）
     </p>
     <p class="graf graf--p graf-after--p" id="60ed">
      完成之后你要把现金藏在地下室里，通往地下室的路径上确保没有任何摄像头，确保没有任何人知道你的计划。
     </p>
     <p class="graf graf--p graf-after--p" id="b9b1">
      第三，抛弃
      <strong class="markup--strong markup--p-strong">
       一切社交媒体账户
      </strong>
      。记住是一切。
      <strong class="markup--strong markup--p-strong">
       删除这些帐户并不会真正抹去你的信息，所有社交媒体都拥有可以让当局获得你的历史信息的数据库。但是，尽可能地将自己从信息网格中分离出来，肯定是一种明智的选择。
      </strong>
     </p>
     <p class="graf graf--p graf-after--p" id="00da">
      Facebook 不是唯一收集个人信息的窥视者。如果 Google 是推动互联网发展的引擎，那么个人信息就是让它的石油。通过它的地图应用程序，它知道你去过哪里，你怎么到达那里的（步行，汽车，公共汽车，火车）停留多久以及是否第一次访问。该公司将您的搜索历史记录存储在你登录过的每台设备上，而算法会在您的电子邮件中搜索任何关于你的细节和动态迹象。
     </p>
     <p class="graf graf--p graf-after--p" id="627f">
      本月早些时候宣布的 Google Lens 是一款可视搜索引擎，可通过手机相机提供有关您正在查看的内容的信息。 Google 可能会知道你的年龄，性别，爱好，职业，兴趣，关系状态，你看过的每一个 YouTub 视频，你搜索过的每一张图片，你是否感到沮丧，以及为什么，它是数字万神殿中唯一无所不知的神。
     </p>
     <p class="graf graf--p graf-after--p" id="baec">
      他们使用数据库和目录来寻找目标人，大数据是社交工程学的基础。今后的文章中我们会详细介绍这些诈骗和追踪的方法。
     </p>
     <p class="graf graf--p graf-after--p" id="ab91">
      先扔掉它们，彻底的扔掉，才能继续我们的游戏。
     </p>
     <p class="graf graf--p graf-after--p" id="eeaa">
      这一过程依旧需要慢慢进行。如果你曾经发布信息很频繁，并且拥有的关注者比较多、互动频繁，那么这个过程对于你来说应该更慢，至少尽可能做到不要让任何人注意到你的动作。
     </p>
     <p class="graf graf--p graf-after--p" id="2beb">
      第四，辞职或请长假。这个过程同样需要时间，
      <strong class="markup--strong markup--p-strong">
       直到没有人会质疑你的缺席
      </strong>
      。需要做到让你的雇主和同事深深相信你真的需要很长一段时间的假期。
     </p>
     <p class="graf graf--p graf-after--p" id="2210">
      第五，收集所有关于你的证件，每个学校的学生证、身份证、驾驶执照、出生证、社保卡和银行对账单。
      <strong class="markup--strong markup--p-strong">
       收集所有将你的名字身份和照片放在一起的东西
      </strong>
      。
     </p>
     <p class="graf graf--p graf-after--p" id="5943">
      然后用你的方式处理它们。准则是从此之后不让任何人有机会获取到它们。不管是深埋还是一把火烧掉，注意一定要做的彻底。不要让任何人觉察到这一过程，否则会引起他们的兴趣，更加关注你或者更努力地去寻找你。
     </p>
     <p class="graf graf--p graf-after--p" id="e241">
      第五，
      <strong class="markup--strong markup--p-strong">
       销毁你的手机
      </strong>
      。这是绝对必须的，现在你是逃难者，不是社交爱好者，手机对你来说并没什么用。如果你实在想要的话可以用
      <strong class="markup--strong markup--p-strong">
       现金
      </strong>
      买个
      <strong class="markup--strong markup--p-strong">
       一次性手机
      </strong>
      ，并且在必要的时候毫不犹豫地扔掉它。
     </p>
     <p class="graf graf--p graf-after--p" id="c7d5">
      第六，甩掉你的车。
      <strong class="markup--strong markup--p-strong">
       很少有东西比一个人的汽车更容易追踪和辨认了
      </strong>
      。卖掉它，记住只收现金，或者烧掉车牌扔在偏远的地方。
     </p>
     <p class="graf graf--p graf-after--p" id="d8f1">
      用
      <strong class="markup--strong markup--p-strong">
       现金
      </strong>
      买一辆二手车。这辆车不应该华丽或独特，色泽应该中性大众，使用过几年的那种，有些划痕和凹痕。记住，
      <strong class="markup--strong markup--p-strong">
       你得把它变得无聊才能可靠。
      </strong>
     </p>
     <p class="graf graf--p graf-after--p" id="f13b">
      最糟糕的事莫过于在逃离的过程中车子没电了。所以你需要一个后备电源
      <a class="markup--anchor markup--p-anchor" data-href="https://deals.survivalfrog.com/pocket-jump-starter-power-bank.php" href="https://deals.survivalfrog.com/pocket-jump-starter-power-bank.php" rel="nofollow noopener noreferrer" target="_blank">
       pocket jumper
      </a>
      ，它应该小巧，便于携带，功能强大。
     </p>
     <p class="graf graf--p graf-after--p" id="9ed7">
      <strong class="markup--strong markup--p-strong">
       强调：上述所有步骤实施中都要躲避一切摄像头和空中无人机监视。
      </strong>
     </p>
     <p class="graf graf--p graf-after--p" id="402e">
      <strong class="markup--strong markup--p-strong">
       细节方法和要点
      </strong>
     </p>
     <p class="graf graf--p graf-after--p" id="62e9">
      一、查找在你出生的当天死亡的婴儿，联系政府申请出生证。这样做不会引起特别的注意，要知道很多人都会丢失自己的出生证。然后用这些资料获取驾照、社保号以及其他个人证件。
     </p>
     <p class="graf graf--p graf-after--p" id="7b41">
      当然众所周知的最佳方案是伪造死亡，在获得另一个合法身份的同时，要让掌握你的身份信息的政府认为你的真的死了。但
      <strong class="markup--strong markup--p-strong">
       当下是电子监视史无前例的高峰，对我们的游戏来说形成的威胁相当大。要知道，
      </strong>
      <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E5%8C%BF%E5%90%8D%E7%BB%88%E7%BB%93%E8%80%85-%E4%B8%BA%E4%BB%80%E4%B9%88%E8%AF%B4%E5%A3%B0%E9%9F%B3%E8%AF%86%E5%88%AB%E6%98%AF%E6%89%80%E6%9C%89%E9%9A%90%E7%A7%81%E5%85%A5%E4%BE%B5%E6%89%8B%E6%AE%B5%E4%B8%AD%E6%9C%80%E4%B8%BA%E5%8F%AF%E6%80%95%E7%9A%84%E4%B8%80%E7%A7%8D-da18b902ad37" href="https://medium.com/@iyouport/%E5%8C%BF%E5%90%8D%E7%BB%88%E7%BB%93%E8%80%85-%E4%B8%BA%E4%BB%80%E4%B9%88%E8%AF%B4%E5%A3%B0%E9%9F%B3%E8%AF%86%E5%88%AB%E6%98%AF%E6%89%80%E6%9C%89%E9%9A%90%E7%A7%81%E5%85%A5%E4%BE%B5%E6%89%8B%E6%AE%B5%E4%B8%AD%E6%9C%80%E4%B8%BA%E5%8F%AF%E6%80%95%E7%9A%84%E4%B8%80%E7%A7%8D-da18b902ad37" rel="noopener noreferrer" target="_blank">
       <strong class="markup--strong markup--p-strong">
        生物识别信息
       </strong>
      </a>
      <strong class="markup--strong markup--p-strong">
       是无法更改的，这可不止包括你的脸，还有虹膜、指纹、DNA、声音、步态。假死的难度非常之高。除非你有幸生活在一个偏远落后的地方。
      </strong>
     </p>
     <p class="graf graf--p graf-after--p" id="17fd">
      二、
      <strong class="markup--strong markup--p-strong">
       改变外观
      </strong>
      。把你平时最喜欢的经常穿着的衣服都扔掉，去一家便宜的服装店，买几件
      <strong class="markup--strong markup--p-strong">
       你永远不会选择的衣服
      </strong>
      。染头发，但千万不要选择疯狂的颜色，要
      <strong class="markup--strong markup--p-strong">
       看起来是个普通人，只是不像原来的你
      </strong>
      。再次提醒，整个过程用现金，并且去你从未光顾过的店铺。
     </p>
     <p class="graf graf--p graf-after--p" id="93bd">
      练习一些小动作，遮盖你平时使用的标志性习惯动作，练习吃你平时不怎么吃的东西，要知道你妈妈了解你的习惯和生活方式，其他所有人都能了解，只是他们没告诉你。但他们会告诉想追踪你的人。
     </p>
     <p class="graf graf--p graf-after--p" id="5dd4">
      人们已经开始寻找你了，要做到尽可能让他人很难认出你来。
      <strong class="markup--strong markup--p-strong">
       你需要让自己变得陌生。但不要让自己变得特殊，你需要融入陌生的人群才能更好地掩饰自己的身份。
      </strong>
     </p>
     <p class="graf graf--p graf-after--p" id="94e2">
      记住你必须正常，无聊，平庸，尽量不要带有特征，一套普普通通的能自圆其说的简历。那些路边的石子一直都存在着，但没人注意到它们，于是它们就仿佛不存在一样。这就是你应该追求的目标。
     </p>
     <p class="graf graf--p graf-after--p" id="1916">
      三、
      <strong class="markup--strong markup--p-strong">
       交通工具
      </strong>
      。步行和自行车是最佳选项，记住不要使用共享单车！这可以让你不容易被追踪，原则是不需要使用许可证或身份证件。
     </p>
     <p class="graf graf--p graf-after--p" id="4e6a">
      公共交通工具基本都有摄像头，并且大多需要身份证件才能搭乘，所以最好不要用，除非你已经离家非常非常远了。在任何情况下不要搭乘出租车或搭便车，最好保证交通工具内只有你一个人。
     </p>
     <p class="graf graf--p graf-after--p" id="6bc5">
      四、
      <strong class="markup--strong markup--p-strong">
       了解你的敌人
      </strong>
      。你想躲什么人很重要，你需要掌握那些最渴望追踪你的人的思考方式，他们会怎么想，会最先猜到什么，然后避免自己选择这些元素。
     </p>
     <p class="graf graf--p graf-after--p" id="74f3">
      必须说不幸的是，
      <strong class="markup--strong markup--p-strong">
       如果你的敌人是政府部门的话，他们很可能比你自己更加了解你。
      </strong>
     </p>
     <p class="graf graf--p graf-after--p" id="7b69">
      八年前，电影制作人 David Bond 拍了一部片子（Erasing David），尝试了我们这个游戏中所描述的过程 — — 摆脱当今监控社会的天罗地网。那可是八年前哦，八年后的今天这一切的难度已经增加了几十倍不止。
     </p>
     <p class="graf graf--p graf-after--p" id="eeea">
      在那部片子结束后，追踪者将 David Bond 带到了他们的监控室 — —
      <strong class="markup--strong markup--p-strong">
       一整面墙上布满了显示器，上面都是他的信息。那些数据被整合成了一个详细的人格分析，以此来预测逃离者最有可能做出什么样的选择，然后提前下手围堵。这些构成你的人格分析图解的数据都来自于公开信息，来自你平时完全没有留意到就泄漏出去了的信息。它们最终会变成等着你的手铐。
      </strong>
     </p>
     <p class="graf graf--p graf-after--p" id="4a6f">
      我们会在今后的文章中更多介绍你是如何被深度了解的，或者说你的信息是如何被利用的。本文的目标是消失游戏，好吧，总之你应该找到一些
      <strong class="markup--strong markup--p-strong">
       不可预知的地方
      </strong>
      。不一定是让你不舒服和不喜欢的地方，只是一个你以前从未与任何人说过的地方。记住是任何人。
     </p>
     <p class="graf graf--p graf-after--p" id="e7e7">
      五、
      <strong class="markup--strong markup--p-strong">
       搞点虚假线索
      </strong>
      。在你开始旅程之前可以搞点假动作，比如用银行卡买几张你根本不会去的地方的旅行手册、使用互联网查询你根本不会去的地方的机票（不需要买，只要你使用搜索引擎就够了，一切都会被记录在案）、使用轻松自然的方式和那些最容易被警方问询的身边人聊起一些你根本不会去的地方。等等，我觉得你应该知道该怎么做。
     </p>
     <p class="graf graf--p graf-after--p" id="1f60">
      最佳选择是两个极端：繁忙的大都市或者极其偏远的地区。你可以消失在人流中，或者自然的空虚之中。不过需要多一句话，后者也许不如前者，因为有些偏远地方比较封闭，人们互相之间非常熟悉，并且对外来的陌生人保持着警惕，你的“外来户”身份很快会传遍整个地区，这是危险因素。而
      <strong class="markup--strong markup--p-strong">
       大城市的人们之间陌生程度非常高，大部分情况下谁都不会注意到谁。
      </strong>
     </p>
     <p class="graf graf--p graf-after--p" id="05c4">
      六、
      <strong class="markup--strong markup--p-strong">
       创建一个历史角色
      </strong>
      。消失不止是要脱离旧的轨道。如果你想长时间消失而不被发现，这意味着你的旧有性格，你的习惯，喜恶、爱好和想法都要改变。
     </p>
     <p class="graf graf--p graf-after--p" id="79db">
      你必须学会​​自然地回答以下所有这些问题：你来自哪里？ 你之前做了什么？ 你家人在哪里？ 你喜欢什么样的食物？ 你最喜欢的饮料是什么？ 你最喜欢的作者是哪位？ 你最喜欢的音乐家是谁？ 喜欢的明星是哪款？
     </p>
     <p class="graf graf--p graf-after--p" id="28d9">
      最基本的是要做到：
      <strong class="markup--strong markup--p-strong">
       重新创造一切。
      </strong>
     </p>
     <p class="graf graf--p graf-after--p" id="105c">
      当然，
      <strong class="markup--strong markup--p-strong">
       结交新朋友最终是必要的
      </strong>
      。无论你到哪里，邻居和当地人都会成为你生活的一部分，和他们交朋友没有错，只是要确保在你的假定身份下做到这点。
     </p>
     <p class="graf graf--p graf-after--p" id="caf4">
      让人们了解到同一版本的关于你的伪装身份是很必要的，尤其是当警方开始问询的时候。所以，不要害怕遇见别人，只需要警惕不要被揭示你的真实身份。
     </p>
     <p class="graf graf--p graf-after--p" id="c15a">
      七、要
      <strong class="markup--strong markup--p-strong">
       随时保持警惕
      </strong>
      。千万不要在一段时间的安全之后就让自己陷入旧习惯！习惯很难更改，但现在你的目标是保命，是命重要还是难度重要？时刻注意自己，避免任何旧的习惯重新回到你身上。
     </p>
     <p class="graf graf--p graf-after--p" id="5e90">
      八、
      <strong class="markup--strong markup--p-strong">
       不要冲动
      </strong>
      。不要在任何时候使用任何联系方式试图和家人及旧朋友报平安，他们不再是你生活的一部分。任何形式的联系都将是你被发现和被毁灭的导火线。
     </p>
     <p class="graf graf--p graf-after--p" id="60f3">
      九、
      <strong class="markup--strong markup--p-strong">
       继续旅行
      </strong>
      。相比在一个地方停留，我们建议你继续旅行，尤其是那些欠发达的国家和地区，改名字并不那么难。变成一个平庸的背包客。再次：尽可能躲开摄像头。
     </p>
     <p class="graf graf--p graf-after--p" id="db23">
      <strong class="markup--strong markup--p-strong">
       重申强调关键细节
      </strong>
      ：用现金在你极少光顾的商店购买新的笔记本电脑和手机、使用无法追踪的短信服务、不要使用公共电子邮件服务器、旅途中不要住店、不要叫车、不要乘坐任何要求身份识别证件才能搭乘的交通工具……尤其是不要使用互联网。
     </p>
     <p class="graf graf--p graf-after--p graf--trailing" id="d00c">
      <strong class="markup--strong markup--p-strong">
       结语
      </strong>
      ：这只是一个游戏，或者说一个挑战。我们希望人们能体会到监视社会强大的追缉能力、对私人空间史无前例的侵犯。永远消失是基本不可能的，如果你能保持三个月内完全隐身，你就可以写一本书记录这个神奇的过程。我保证，它一定是非常棒的出版物。⚪️
     </p>
    </div>
