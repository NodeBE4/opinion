   我们在两年前的访谈中强调过《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%83%85%e6%8a%a5%e4%b8%8d%e6%98%af%e7%ae%80%e5%8d%95%e7%9a%84%e4%bf%a1%e6%81%af%ef%bc%9aiyouport-%e7%ad%94%e8%af%bb%e8%80%85%e9%97%ae%ef%bc%9a%e5%85%b3%e4%ba%8e%e6%88%91%e4%bb%ac/" href="https://www.iyouport.org/%e6%83%85%e6%8a%a5%e4%b8%8d%e6%98%af%e7%ae%80%e5%8d%95%e7%9a%84%e4%bf%a1%e6%81%af%ef%bc%9aiyouport-%e7%ad%94%e8%af%bb%e8%80%85%e9%97%ae%ef%bc%9a%e5%85%b3%e4%ba%8e%e6%88%91%e4%bb%ac/" rel="noopener noreferrer" target="_blank">
    “情报不是简单的信息”
   </a>
   》。情报支持的是行动，信息仅支持理解。我们始终认为政治评论值得也应该采取情报的态度，而不是吐槽的态度，这样才有可能帮助活动家在进一步的计划中确立有效的方向和策略。
  </p>
  <p class="graf graf--p">
   但是，正如您所看到的，当下的绝大多数政治评论（甚至有些自称为 “分析”的东西）都是不合格的、属于谩骂发泄的、仅仅彰显立场的，它们只是填补了媒体版面，只是表现了作者刷热点的参与性，而
   <strong class="markup--strong markup--p-strong">
    对行动指导来说基本无用
   </strong>
   。
  </p>
  <p class="graf graf--p">
   我们为什么要制造如此之多的无用的东西？让互联网变成垃圾场？
  </p>
  <p class="graf graf--p">
   有些情况下并非刻意，而是，也许很多人尚未能掌握科学的分析方法。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    分析法是大多数工作的重头戏。很多情况下您可以抓取到大量的信息，尤其是互联网时代；但是，是否能令这些信息发挥情报的作用从而对行动形成指导意义，完全取决于您的分析法是否科学合理
   </strong>
   。
  </p>
  <p class="graf graf--p">
   不论是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">

   <a class="markup--anchor markup--p-anchor" data-href="https://start.me/p/1kod2L/iyp-direct-action5" href="https://start.me/p/1kod2L/iyp-direct-action5" rel="noopener noreferrer" target="_blank">
    公民调查
   </a>
   、普通的政治评论、组织和公司的方案拟议程序、还是个人决策，都严重依赖分析的专业性 —— 你想找到一个满意的工作吗？希望联姻一位合适的伴侣吗？正在寻求合作伙伴吗？科学的分析都是您所必需的技能。
  </p>

     练习
    </strong>

  <p class="graf graf--p">
   预想一下，如何解决一些与您自己有关的问题。
  </p>
  <p class="graf graf--p">
   你家附近的公园里会修路吗？哪些同事会被提升为部门负责人？你最喜欢的球队参加本赛季的季后赛吗？你哥们新泡的妞能娶到手吗？您所在国家准备推行的新法律会遇到什么阻力？……
   <strong class="markup--strong markup--p-strong">
    确定有哪些因素会影响结果 —— 收集开源情报，并评估其可信度。
   </strong>
  </p>
  <p class="graf graf--p">
   👉这是对开源情报分析思考方式的一种很好的训练。我们多次强调过，信息收集并不是全部，它只是一个基础，开源情报最重要的部分是情报分析，是思考方式问题，而不是技术工具。
  </p>
  <p class="graf graf--p">
   您需要回答三个问题：
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    目前的情况如何？
   </li>
   <li class="graf graf--li">
    所有参与者分别有哪些能力？
   </li>
   <li class="graf graf--li">
    他们的意图是什么？
   </li>
  </ol>
  <p class="graf graf--p">
   未来会证明你的判断是否正确。这就是情报分析师的思考方式。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     事实和图景
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   👉在分析师的工作中，
   <strong class="markup--strong markup--p-strong">
    比较
   </strong>
   所有事实比单独分析每个事实更有用。
  </p>
  <p class="graf graf--p">
   比如，某国出国学习核物理的学生人数增加了一倍，但在该国国内学习这个专业的学生数量并没有太大的增幅。这种增长可能与年轻专业人士离开该国的总趋势相符。
  </p>
  <p class="graf graf--p">
   但是，如果将这一事实与其他信息相比较，可能就会看到一幅
   <strong class="markup--strong markup--p-strong">
    完全不同的图景
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    该国偏远省份的一家工厂购买了一台强大的工业离心机；
   </li>
   <li class="graf graf--li">
    通往这家工厂的铁路正在配备现代化的视频监控系统；
   </li>
   <li class="graf graf--li">
    该国最大的铁路公司下令建造运输危险物品的专用列车；
   </li>
   <li class="graf graf--li">
    该国的一座核电站已经开始额外建造一个用于冷却乏燃料元件的水池了。
   </li>
  </ul>
  <p class="graf graf--p">
   所有这些事实可以帮助你提出一个合理的假设：即 该国政府正在开发用于处理和浓缩乏燃料的技术。通过监控铁路和列车抵达工厂的记录，您就可以针对建设的进度和速度得出定量的结论。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    请注意
   </strong>
   ：并不需要对该国的核能组织进行渗透，👉 仅凭开源信息就可以得出这一结果。只需监控地方政府的委任情况，并与季节性施工人员和住在附近的低级别工作人员交谈几次，就足够了。
  </p>
  <p class="graf graf--p">
   👉 有时候分析师需要做很多观察工作。
   <strong class="markup--strong markup--p-strong">
    将最常见的数据进行比较，可能会得出意想不到的结果。
   </strong>
  </p>
  <p class="graf graf--p">
   比如，下列时间表显示了反情报人员 A（一名 43岁男子）的加班情况： 3月 2日， 3月 5日， 3月 6日， 3月 11日， 3月 15日， 3月 20日， 3月 21日， 4月 2日， 4月 4日， 4月 7日， 4月 11日， 4月 13日， 4月 20日， 5月 2日和 5月 15日。
  </p>
  <p class="graf graf--p">
   这些信息本身并没有什么意义。再看看另一个部门一位 28岁的女性情报员的加班时间表，单独看也没有什么特别的意义： 2月 27日， 3月 5日， 3月 7日， 3月 11日， 3月 15日， 3月 20日， 4月 2日， 4月 4日， 4月 6日， 4月 15日， 4月 24日， 6月 5日。
  </p>
  <p class="graf graf--p">
   👉 但是，当比较这些信息时你会发现：
  </p>
  <p class="graf graf--p">
   A： 2. 03, 5. 03, 6. 03, 11. 03, 15. 03, 20. 03, 21. 03, 2. 04, 4. 04, 7. 04, 11. 04, 13. 04, 20. 04, 2. 05, 15. 05。
  </p>
  <p class="graf graf--p">
   B： 27. 02, 5. 03, 7. 03, 11. 03, 15. 03, 20. 03, 2. 04, 4. 04, 6. 04, 15. 04, 24. 04, 5. 06。
  </p>
  <p class="graf graf--p">
   我们可以猜测 A已经与 B建立了友谊，这份友谊可能在四月初就结束了，或者进入了另一个阶段（他们在工作之余会面）。
  </p>
  <p class="graf graf--p">
   这并不是一个最终结论，而是
   <strong class="markup--strong markup--p-strong">
    一个容易验证的假设
   </strong>
   。
  </p>
  <p class="graf graf--p">

   统计学为情报分析提供了强大的数据挖掘工具。但您必须小心谨慎地使用这些工具。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    使用统计学方法需要足够大的数据量，专业化的培训，以及精心设定的、会在统计资料帮助下进行检验的假设
   </strong>

   ⚠️ 在没有合理理由推导出相关性的情况下，轻率地使用统计学可能会得出无用的或错误的结论 —— 这是很多做数据新闻的大牌媒体经常出现的错误。它们只会带来误导性。

   <li>
    《
    <a href="https://www.iyouport.org/%e4%bd%a0%e7%9f%a5%e9%81%93%e4%bb%80%e4%b9%88%e6%89%8d%e6%98%af%e7%9c%9f%e6%ad%a3%e7%9a%84%e8%b0%83%e6%9f%a5%e5%90%97%ef%bc%9f%e6%89%80%e8%b0%93%e7%9a%84%e8%b5%b7%e5%ba%95%e7%bb%8f%e5%b8%b8/" rel="noopener noreferrer" target="_blank">
     你知道什么才是真正的*调查*吗？所谓的”起底”经常是骗人的
    </a>
    》
   </li>
  </ul>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     分析是一门科学
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   分析师的工作和科学家或侦探类似。他们需要收集大量的看似互不相干的 *事实*，并在此基础上建立连贯的图景。
  </p>
  <p class="graf graf--p">
   对所有已知的事实进行比较以找到规律。这样做会花费很长时间，而且不能保证得到结果。为了让这个过程保持稳定一致，分析师会使用包括犯罪学在内的现代科学所采用的
   <strong class="markup--strong markup--p-strong">
    研究模型
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第一阶段：事实。
   </strong>
  </p>
  <p class="graf graf--p">
   收集有关这个主题的基本已知事实。尝试可视化它们，制作一个计划或思维导图。如果可能的话，研究现有的理论、意见和观点。
  </p>
  <p class="graf graf--p">
   👉 请记住，有些资料可能不完全准确可靠，有些理论也可能不完整或者是错误的。
   <strong class="markup--strong markup--p-strong">
    所以请保持始终验证的原则
   </strong>
   。（主流媒体来源的资料基本都不充分，他们采访的所谓的专家也非常片面，这些都需要小心）
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第二阶段：假设。
   </strong>
  </p>
  <p class="graf graf--p">
   做出一个假设 —— 能够以最佳方式解释事实设想。
  </p>
  <p class="graf graf--p">
   在调查过程中，调查人员会根据证据和证词判断涉事者究竟是谁，以及事件过程究竟是怎样的。
  </p>
  <p class="graf graf--p">
   👉 这个假设必须有完好的逻辑链，尽可能避免偏见。所以，调查人员应该如飞行员那样合作完成工作，互相监督彼此可能的偏颇性。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第三阶段：结论。
   </strong>
  </p>
  <p class="graf graf--p">
   想想你的假设会带来怎样的结果，还有它是否属实。这个假设的结果可能会超越现有事实的限制，比一开始已知的情况所表明的内容要多得多。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第四阶段：核实。
   </strong>
  </p>
  <p class="graf graf--p">
   形成一个假设，寻找新的事实，用这些事实去证实或反驳你做出的假设，
  </p>
  <p class="graf graf--p">
   👉 获取新事实的一种方法是进行试验。如果新的事实反驳了你的理论，排除或者修改了你的假设，那么再回到第一和第二阶段。根据新的事实形成新的假设。
  </p>
  <p class="graf graf--p">
   如果你能够确认大部分的结果，那么通常来说可以认为这个假设经过了验证。
  </p>
  <p class="graf graf--p">
   但是，这只是在一定条件下被核实了，有一天你可能会发现反驳它的其他事实。👉 通常情况下，科学家会加倍努力去推翻假设，可惜的是，侦探却往往试图证实假设。这会为 “
   <a class="markup--anchor markup--p-anchor" data-href="https://start.me/p/nRBzO9/iyp-3" href="https://start.me/p/nRBzO9/iyp-3" rel="noopener noreferrer" target="_blank">
    战略性欺骗
   </a>
   ” 提供入口。您应该采取科学家的方法。
  </p>
  <p class="graf graf--p">
   这个假设和结论应该为进一步调查提供方向。利用前文中开发乏燃料加工技术的例子，您可以构建以下的推理链：
  </p>
  <p class="graf graf--p">
   <strong>
    第一阶段：事实
   </strong>
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    研究核物理的学生人数翻了一番；
   </li>
   <li class="graf graf--li">
    一家大型化学公司已经购买了一台功能强大的离心机；
   </li>
   <li class="graf graf--li">
    通往工厂的铁路配备了视频监控系统；
   </li>
   <li class="graf graf--li">
    运输危险物品的专用列车已经订购在造；
   </li>
   <li class="graf graf--li">
    一座主要核电站已经开始额外建造一个用于冷却乏燃料元件的水池。
   </li>
  </ol>
  <p class="graf graf--p">
   <strong>
    第二阶段：假设
   </strong>
  </p>
  <p class="graf graf--p">
   该国政府正在开发处理和浓缩乏燃料的技术。建造一个额外的冷却池说明了其积聚乏燃料元件的意图。
  </p>
  <p class="graf graf--p">
   铁路上的安全措施可能意味着关键货物的运输方向和运输量的变化。专家的人数越来越多，表明核能领域正在认真执行新的政策，且这一政策将长期实施。
  </p>
  <p class="graf graf--p">
   同时这也表明该项目的特殊性。否则该国政府会邀请外国专家参与。
  </p>
  <p class="graf graf--p">
   <strong>
    第三阶段：结论
   </strong>
  </p>
  <p class="graf graf--p">
   如果假设是真的，那么：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    这家化工厂周边区域的放射物水平将会增加；
   </li>
   <li class="graf graf--li">
    将会出现具有污染性的空气；
   </li>
   <li class="graf graf--li">
    该公司将购买溶解固体放射性废物所需的硝酸；
   </li>
   <li class="graf graf--li">
    该工厂的基础建设工程将变得更加复杂，因为用于处理核废料的胶结、玻璃化和沥青化等工艺都是非常复杂的生产过程。
   </li>
   <li class="graf graf--li">
    装有特种集装箱的装甲列车将定期从核电站开至工厂。
   </li>
  </ul>
  <p class="graf graf--p">
   为了能得出这些结论，你需要做的是熟悉用于处理核废料的技术，这些信息
   <strong class="markup--strong markup--p-strong">
    都可以从开源渠道获得
   </strong>
   ，比如国际原子能机构，或环保组织NGO的网站。
  </p>
  <p class="graf graf--p">
   <strong>
    第四阶段：核实
   </strong>
  </p>
  <p class="graf graf--p">
   大众媒体或环保组织网站上可能会出现有关放射性水平增加及放射性气体排放的报道。可能需要联合当地的活动家以获得有关空气和水的样本进行化学分析。
  </p>
  <p class="graf graf--p">
   如果之前的假设和结论在第四阶段得到了确认，那么它们很可能是准确的，随后你需要据此采取行动。如果不能得到确认，你需要形成一个关于目前状况的新假设。
  </p>
  <p class="graf graf--p">
   👉 科学研究是一个正式的、创造性的过程。之所以说它正式，是因为证明或反驳假设都是基于逻辑规律的，之所以说它具有创造性，是因为每一个假设都是
   <strong class="markup--strong markup--p-strong">
    出于洞察力
   </strong>
   而诞生的。
  </p>
  <p class="graf graf--p">
   尽管如此，科学方法节省了大量的时间和资源。信息研究变得具有方向性。不会浪费时间在无关紧要的资料提取方面，因此不仅可以加速，也能减少获取信息过程中的风险。
  </p>
  <p class="graf graf--p">
   👉 使用科学方法，尝试遵循以下经过充分验证的指导方针：
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    尽快形成初步假设。毫无计划的事实收集工作会耽误解决问题的进度。
   </li>
   <li class="graf graf--li">
    批判性地对待这一假设。尝试证伪而不是证实。如果一味的将事实往假设上套，很容易误导自己。
   </li>
   <li class="graf graf--li">
    做好从头再来的准备。驳倒了假设并不意味着失败，而是迈向真相的新一步。
   </li>
   <li class="graf graf--li">
    创建一个好的假设需要时间。如果你陷入了僵局，不妨先休息一下，然后换个思考方式重新再来。
   </li>
  </ol>
  <p class="graf graf--p">
   决策需要时间才能变得成熟。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   试着更多去了解那些你通常不会关注的人。比如 观察办公室前台人员。她结婚了吗？她喜欢什么颜色的饰品？她在休息时间喜欢干什么？…… 这些信息能帮助你作出哪些关于她的私生活的假设？
  </p>
  <p class="graf graf--p">
   试着弄清楚小区保安的工作时间表；尝试推测你经常光顾的水果店老板的老家在哪。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     练习
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   试着比其他大多数人更深入地了解你的国家的政治。试着弄清楚哪些属于保密信息。收集事实，比较信息，提出假设，并进行预测。
  </p>
  <p class="graf graf--p">
   时间会告诉你，你的预测是否正确。
  </p>
  <p class="graf graf--p">
   加油！我们下周再见，将有更多实用技巧提供。⚪️
  </p>
  <p class="graf graf--p">
