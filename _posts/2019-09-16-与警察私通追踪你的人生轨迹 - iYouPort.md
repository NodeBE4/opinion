  <p class="graf graf--p">
   你知道五年前的此时你在哪里吗？你当时有 Android 手机吗？你可能已经忘了，但是，事实证明，谷歌知道 —— 而且它会告诉执法部门。
  </p>
  <p class="graf graf--p">
   在一篇新文章中，“纽约时报”详细介绍了一种鲜为人知的技术，这种技术越来越多地被执法部门用来找出过去某个特定时间段某些特定地理区域内存在的所有人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该技术正是依赖于 Google 从大多数 Android 设备以及安装了 Google 地图和其他谷歌应用的 iPhone 和 iPad 设备中，收集的详细位置数据
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这些数据存储在 Google 维护的数据库 “Sensorvault” 中，由于
   <strong class="markup--strong markup--p-strong">
    Google 无限期地存储了这些数据
   </strong>
   ，因此⚠️ Sensorvault 包含详细的位置记录，涉及全球至少数亿台设备，可追溯到近十年前！
  </p>
  <p class="graf graf--p">
   谷歌给执法部门的数据是如此精确，以至于一名副警察局长直接承认其“显示了整个生活全景图”。
  </p>
  <p class="graf graf--p">
   ⚠️即使人们没有打电话或使用那些应用程序，数据也会被收集，这意味着它可以比手机信号塔生成的数据更加详细。
  </p>
  <p class="graf graf--p">
   位置数据来自GPS信号、手机信号塔、附近的 Wi-Fi 设备和蓝牙信标。
  </p>
  <p class="graf graf--p">
   谷歌辩称，“用户选择加入的 Sensorvault 数据库位置数据收集”。但是，谷歌没承认
   <strong class="markup--strong markup--p-strong">
    用户很难拒绝选择，许多用户可能根本不知道可以退出
   </strong>
   。
  </p>
  <p class="graf graf--p">
   此外，⚠️默认情况下，Android 设备会收集大量的其他位置数据，并且
   <a class="markup--anchor markup--p-anchor" data-href="https://www.apnews.com/828aefab64d4411bac257a07c1af0ecb" href="https://www.apnews.com/828aefab64d4411bac257a07c1af0ecb" rel="noopener noreferrer" target="_blank">
    选择退出非常困难
   </a>
   。
  </p>
  <p class="graf graf--p">
   有了谷歌的帮忙，警方能够一次性从数十台到数百台设备中访问精确位置数据 —— 与真人联系在一起的设备，
   <strong class="markup--strong markup--p-strong">
    其中许多人（而且在某些情况下是所有人）与犯罪活动没有任何关系，也没有提出任何可以被怀疑的理由
   </strong>
   。
  </p>
  <p class="graf graf--p">
   所谓的地理围栏涵盖从单一建筑物到多个街区的地理区域，以及从几个小时到一周的时间段。
  </p>
  <p class="graf graf--p">
   据“纽约时报”和其他媒体报道，到目前为止，这项技术正在亚利桑那州、北卡罗来纳州、加利福尼亚州、佛罗里达州、明尼苏达州、缅因州、和华盛顿州的 FBI 和警察部门使用，尽管可能还有其他机构在全国各地使用它。
  </p>
  <p class="graf graf--p">
   但是警方并没有将这种深度监视技术用于严重或暴力犯罪的调查 —— 明尼苏达州公共广播电台
   <a class="markup--anchor markup--p-anchor" data-href="https://www.mprnews.org/story/2019/02/07/google-location-police-search-warrants" href="https://www.mprnews.org/story/2019/02/07/google-location-police-search-warrants" rel="noopener noreferrer" target="_blank">
    报道说
   </a>
   ，该技术已经被用来试图找出偷皮卡车的嫌疑人，另外还有价值650美元的轮胎。
  </p>
  <p class="graf graf--p">
   谷歌每周最多会收到180份来自警方的数据请求，而且显然正在努力跟上当权者的需求。
  </p>
  <p class="graf graf--p">
   时报说，警方获得访问权限的理由经常只是官员的断言，即“美国人拥有手机以及谷歌持有许多这些手机的位置数据”，这一事实在某种程度上甚至支持了逮捕令的可能原因。
  </p>
  <p class="graf graf--p">
   执法部门似乎正在使用三步流程来了解设备持有者的姓名（在某些情况下，单一权证授权所有三个步骤）。
  </p>
  <p class="graf graf--p">
   在第一步中，官员指定感兴趣的区域和时间段，作为回应，Google 向警方提供有关所有设备的信息，由匿名号码识别 —— 这一步可能会揭示数百个设备。
  </p>
  <p class="graf graf--p">
   之后，官员可以将他们的请求范围缩小到更少的设备上，Google 将发布更详细的数据，⚠️包括设备在原始请求区域和时间段之外的位置数据。这些数据仍然涉及多个设备，可以显示详细的行程。
  </p>
  <p class="graf graf--p">
   在最后一步中，侦探审查行程数据以查看是否有任何设备与犯罪相关，
   <strong class="markup--strong markup--p-strong">
    并且他们询问用户的姓名和特定个别设备的其他信息。
   </strong>
  </p>
  <p class="graf graf--p">
   由于几个原因，该技术存在严重问题。
  </p>
  <p class="graf graf--p">
   首先，与警察使用的其他调查方法不同，警察不会从实际的嫌疑人或目标设备开始 —— 他们是从一个地点和时间段的大范围开始；这使它成为了一种捕鱼探险 —— 而这正是第四修正案旨在防止的那种搜查。
  </p>
  <p class="graf graf--p">
   警察所拥有的唯一信息只是犯罪的发生 —— 像这样的搜索更有可能牵连到那些恰好在错误的时间错误的地点路径的无辜者。
  </p>
  <p class="graf graf--p">
   在有争议的时间段内，
   <strong class="markup--strong markup--p-strong">
    该区域内的每个设备的每个所有者都成为了嫌疑人 — 没有任何原因，除了他们拥有与 Google 共享位置信息的设备之外
   </strong>
   。
  </p>
  <p class="graf graf--p">
   其次，正如最高法院去年夏天在 Carpenter 诉美国案中所承认的那样，这样详细的行程数据可以提供“窥视一个人全部生活的私密窗口，不仅揭示了他本人的特定运动，而且还包括其家庭、政治倾向、专业、宗教信仰，和性取向。”
  </p>
  <p class="graf graf--p">
   这正是副警长在说出谷歌位置数据“显示了整个生活模式”时所认可的。
  </p>
  <p class="graf graf--p">
   第三，真正的犯罪者很可能甚至不包括在 Google 披露的数据中。
  </p>
  <p class="graf graf--p">
   这一切只是始于警方的一个预感：未知的嫌疑人拥有一部手机，而谷歌收集了数据，仅此而已。听起来就很扯对吧，因为嫌疑人有可能当时并没有携带 Android 手机或没有使用谷歌应用程序。
  </p>
  <p class="graf graf--p">
   这样的技术也揭示了我们现行的保证系统存在的重大问题。显然，“需要授权”的条件根本不足以保护人们的隐私。警方只需要一次授权就可以获得比过去更多和更详细的关于人们的信息。
  </p>
  <p class="graf graf--p">
   在其他情况下，⚠️官员甚至可以访问手机或硬盘上的所有数据；存储在 Google 帐户中的所有电子邮件（可能会追溯到数年前）；以及与社交媒体帐户相关的所有信息（包括照片、帖子、私信和联系人）。
  </p>
  <p class="graf graf--p">
   事实上第四修正案就可以起到保护作用，即 必须 “确定地描述要搜查的地方，以及要查封的人或事物。” 然而，事实上许多法院只是橡皮图章，而不会质疑请求的广泛范围。
  </p>
  <p class="graf graf--p">
   这种技术牵连着无辜的人，并对人们的生活产生了真正的影响。
  </p>
  <p class="graf graf--p">
   一旦你被错误地逮捕，很可能会在你的生活、工作、财产、和今后重新站起来的机会等各方面付出代价。纽约时报说，一名男子被拘留了一个星期，而此后几个月都没能恢复过来。他失去了工作、他的汽车被扣押搜查 ……这些都是过度广泛搜查可能导致的深远影响。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    注：
   </strong>
   希望本文能给安卓用户一些提示，甚至包括中国的安卓用户，鉴于谷歌对中国的奋力追求，很难保证他们与中国当局没有任何合作，中国没有独立媒体，人们很难了解实情。
  </p>

   在这里看到一些解决方案《

    喜欢 Android 但讨厌 Google 的隐私侵犯，可以怎么办
   </a>
   ？》；以及《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E5%8F%AF%E4%BB%A5%E9%87%87%E5%8F%96%E5%93%AA%E4%BA%9B%E6%8E%AA%E6%96%BD%E6%9D%A5%E9%98%BB%E6%AD%A2-google-%E5%92%8C%E5%85%B6%E4%BB%96%E4%BA%BA%E8%B7%9F%E8%B8%AA%E6%82%A8%E7%9A%84%E6%89%8B%E6%9C%BA/" href="https://www.iyouport.org/%E5%8F%AF%E4%BB%A5%E9%87%87%E5%8F%96%E5%93%AA%E4%BA%9B%E6%8E%AA%E6%96%BD%E6%9D%A5%E9%98%BB%E6%AD%A2-google-%E5%92%8C%E5%85%B6%E4%BB%96%E4%BA%BA%E8%B7%9F%E8%B8%AA%E6%82%A8%E7%9A%84%E6%89%8B%E6%9C%BA/" rel="noopener noreferrer" target="_blank">
