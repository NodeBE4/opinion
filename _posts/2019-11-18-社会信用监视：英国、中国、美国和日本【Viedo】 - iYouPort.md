   少数派报告风格的反乌托邦世界不是现在才开始的；有些国家已经运行好几年了。
  </p>
  <p class="graf graf--p">
   本文想说的绝不是“存在即合理”，而是，全球反抗者应该联合起来。
   <strong class="markup--strong markup--p-strong">
    生活不应该是监狱；即便真正的监狱也不会对没有犯罪的人施加惩罚
   </strong>
   。
  </p>
  <p class="graf graf--p">
   故事大概是从英国开始的。2017年5月，
   <a class="markup--anchor markup--p-anchor" data-href="https://truepublica.org.uk/united-kingdom/britains-minority-report-style-pre-crime-programme-unexpectedly-unearthed/" href="https://truepublica.org.uk/united-kingdom/britains-minority-report-style-pre-crime-programme-unexpectedly-unearthed/" rel="noopener noreferrer" target="_blank">
    英国启动了
   </a>
   第一个少数派报告风格的“预测犯罪系统”。
  </p>
  <p class="graf graf--p">
   英国在西方世界广泛部署最具侵入性的监控系统，并利用大数据赋予警察“预测犯罪”的能力 —— 现在被瞄准的是尚未违法的人。
  </p>
  <p class="graf graf--p">
   如今已经众所周知，
   <strong class="markup--strong markup--p-strong">
    世界各地的警察部门都在与私营公司合作，使用公共数据、个人信息和算法来预测
   </strong>
   最有可能发生非法行为的地方，最重要的是，“谁”最有可能犯下这些行为。
  </p>

  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bloomberg.com/news/articles/2016-03-03/china-tries-its-hand-at-pre-crime" href="https://www.bloomberg.com/news/articles/2016-03-03/china-tries-its-hand-at-pre-crime" rel="noopener noreferrer" target="_blank">
    Bloomberg
   </a>
   — 中国：中国当局的维稳努力已经扩展到曾经只存在于科幻小说中的情节：预测犯罪。
  </p>
  <p class="graf graf--p">
   共产党指示该国最大的国营国防承包商之一中国电子科技集团开发软件，整理有关所有人的工作、爱好、消费习惯和其他行为的数据，以在“犯罪”发生之前预测到它。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “在恐怖行为发生之后检查原因至关重要，但更重要的是预测即将到来的犯罪行为”，军事承包商总工程师 Wu Manqing 在2017年的一次会议上对记者说。
  </p>
  <p class="graf graf--p">
   该计划是前所未有的，目前没有隐私保护法的保障措施、也没有公民自由倡导者哪怕最小的阻力。
  </p>
  <p class="graf graf--p">
   在世界各地的所有这些例子中，法律正在迅速改变 —— 当权者异口同声地以“保护公民的安全”
   <strong class="markup--strong markup--p-strong">
    为幌子，施加最危险的压制
   </strong>
   。
  </p>
  <p class="graf graf--p">
   以下只是几个关键国家为例。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://www.thedailybeast.com/japans-terrible-anti-terror-law-just-made-the-minority-report-reality" href="http://www.thedailybeast.com/japans-terrible-anti-terror-law-just-made-the-minority-report-reality" rel="noopener noreferrer" target="_blank">
    The Daily Beas
   </a>
   t: 东京 —— 2017年6月14日，在执政联盟扼杀了标准立法议定书之后，欺骗性的“反恐法”在日本成为了法律，并避免了对这个被称为“刑事阴谋法”的法案提出更多令人尴尬的问题。
  </p>
  <p class="graf graf--p">
   该法律规定了277种罪行，允许执法人员逮捕和起诉那些被认为是“计划和准备”犯罪的人，即使这些罪行根本没有实施。
  </p>
  <p class="graf graf--p">
   讽刺的是，277种犯罪中没有任何一种与“恐怖主义”有关。
  </p>
  <p class="graf graf--p">
   从技术上讲，⚠️
   <strong class="markup--strong markup--p-strong">
    由于该立法涉及社交媒体 —— 即使只是点赞或转发了相关的推文，现在也可能成为共谋犯罪指控的理由
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://bigthink.com/think-tank/pre-crime-detection-system-now-being-tested-in-the-us" href="http://bigthink.com/think-tank/pre-crime-detection-system-now-being-tested-in-the-us" rel="noopener noreferrer" target="_blank">
    BigThink
   </a>
   <strong class="markup--strong markup--p-strong">
    —
   </strong>
   美国：由美国国土安全部牵头，一项名为“未来主动筛查技术” FAST 的倡议推出了，旨在利用传感器技术来检测“国家安全部门”定义的“恶意行为”的线索和意图。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    FAST 系统能够无接触地监测所有人的生理和行为线索。这意味着捕获数据，如心率和视觉稳定性等等
   </strong>
   。
  </p>
  <p class="graf graf--p">
   然后，通过算法实时运行以计算个人“计划犯罪的概率”。
  </p>
  <p class="graf graf--p">
   根据
   <a class="markup--anchor markup--p-anchor" data-href="http://www.nature.com/news/2011/110527/full/news.2011.323.html" href="https://www.nature.com/news/2011/110527/full/news.2011.323.html" rel="noopener noreferrer" target="_blank">
    “自然”杂志的说法
   </a>
   ，该计划的第一轮实地测试是在东北部的一个秘密地点完成的。在实验室环境中，FAST 报告的准确率为“70％”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这都是纯粹的预测犯罪技术
   </strong>
   。令人不安的是，⚠️当权者一直在努力使用“社会信用机制”这样的委婉语试图淡化其危险程度。
  </p>
  <p class="graf graf--p">
   正如 As
   <a class="markup--anchor markup--p-anchor" data-href="https://civilsocietyfutures.org/civil-society-in-an-age-of-surveillance-beyond-techno-legal-solutionism/" href="https://civilsocietyfutures.org/civil-society-in-an-age-of-surveillance-beyond-techno-legal-solutionism/" rel="noopener noreferrer" target="_blank">
    Civil Society Future
   </a>
   在一篇关于监视时代的文章中所指出的那样：“
   <strong class="markup--strong markup--p-strong">
    公民越来越多地被根据数据集合进行分类和分析，就如中国开发的社会信用评分 —— 这些分数的目的是预测未来的行为，并相应地分配或剥夺享受资源和服务的资格
   </strong>
   。
  </p>
  <p class="graf graf--p">
   换句话说，一切都必须在当权者满意的范围内。
  </p>
  <p class="graf graf--p">
   我们最初是在近五年前提醒警惕的，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%8f%88%e4%b8%80%e4%b8%aa%e8%a2%ab%e6%bc%a0%e8%a7%86%e7%9a%84%e6%b7%b1%e6%b8%8a/" href="https://www.iyouport.org/%e5%8f%88%e4%b8%80%e4%b8%aa%e8%a2%ab%e6%bc%a0%e8%a7%86%e7%9a%84%e6%b7%b1%e6%b8%8a/" rel="noopener noreferrer" target="_blank">
    又一个被漠视的深渊
   </a>
   》。
  </p>
  <p class="graf graf--p">
   五年来没能有效地抵制它的发育，现在真的没时间了；
   <strong class="markup--strong markup--p-strong">
    唯一的办法是自卫，在这里看到《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%88%86%e8%a3%82%e4%ba%ba%e6%a0%bc%e7%9a%84%e9%ad%85%e5%8a%9b/" href="https://www.iyouport.org/%e5%88%86%e8%a3%82%e4%ba%ba%e6%a0%bc%e7%9a%84%e9%ad%85%e5%8a%9b/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     自卫方法
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》。
   </strong>
  </p>
  <p class="graf graf--p">
   但自卫绝不是目标，而是手段，目标应该是结束这种噩梦。只有首先保证尽可能多的人的人身安全和自由思考的能力，才有希望作出改变。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-7074 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-22.png?resize=1100%2C489&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-22.png?w=1552&amp;ssl=1 1552w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-22.png?resize=300%2C133&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-22.png?resize=768%2C341&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-22.png?resize=1024%2C455&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-22.png?resize=1100%2C489&amp;ssl=1 1100w" data-recalc-dims="1" height="489" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-22.png?resize=1100%2C489&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-7074" data-recalc-dims="1" height="489" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-22.png?resize=1100%2C489&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-22.png?w=1552&amp;ssl=1 1552w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-22.png?resize=300%2C133&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-22.png?resize=768%2C341&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-22.png?resize=1024%2C455&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-22.png?resize=1100%2C489&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   六年前，英国和美国的监控机构 GCHQ 和 NSA 被曝光一直在收集从电子邮件、电话、私人对话、联系人、健康和财务信息到私人家中拍摄的亲密照片等等一切公民数据，那时起，越来越多的人开始迅速改变自己的在线行为。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    关于世上最大规模的曝光，完整版《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" rel="noopener noreferrer" target="_blank">
     互联网霸主的时间线 — — 他们将互联网视为间谍、破坏和战争的主场
    </a>
    》
   </li>
   <li class="graf graf--li">
    为什么曝光没能阻止监视的持续？《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/6%e5%b9%b4%ef%bc%8c%e6%94%b9%e5%8f%98%e4%ba%86%e4%bb%80%e4%b9%88%ef%bc%9f/" href="https://www.iyouport.org/6%e5%b9%b4%ef%bc%8c%e6%94%b9%e5%8f%98%e4%ba%86%e4%bb%80%e4%b9%88%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     6年，改变了什么？
    </a>
    》《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%b6%85%e8%b6%8a%e9%80%8f%e6%98%8e%e5%ba%a6%e9%9d%a9%e5%91%bd/" href="https://www.iyouport.org/%e8%b6%85%e8%b6%8a%e9%80%8f%e6%98%8e%e5%ba%a6%e9%9d%a9%e5%91%bd/" rel="noopener noreferrer" target="_blank">
     超越透明度革命
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   请注意：“改变在线行为”并非仅仅包括我们所希望的隐私安全措施，事实上，作出安全保护的人相对很少；不幸的事，
   <strong class="markup--strong markup--p-strong">
    大多数人采取的“改变”是
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E5%86%B0%E6%B2%B3%E5%B7%B2%E5%9C%A8%E4%BD%A0%E5%BF%83%E9%87%8C%EF%BC%8C%E8%BF%99%E5%B0%B1%E6%98%AF%E4%BB%96%E4%BB%AC%E7%9A%84%E7%9B%AE%E7%9A%84-%E7%9B%91%E8%A7%86/" href="https://www.iyouport.org/%E5%86%B0%E6%B2%B3%E5%B7%B2%E5%9C%A8%E4%BD%A0%E5%BF%83%E9%87%8C%EF%BC%8C%E8%BF%99%E5%B0%B1%E6%98%AF%E4%BB%96%E4%BB%AC%E7%9A%84%E7%9B%AE%E7%9A%84-%E7%9B%91%E8%A7%86/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     自我审查
    </strong>
   </a>
   。这正是监视者所希望的。
  </p>
  <p class="graf graf--p">
   就如卫报显示的那样，还是英国，另一个恐怖的技术也在同时实施中：
  </p>
  <p class="graf graf--p">
   这是一项集体捕捉公民浏览历史、记录电话交谈、语音数据以
   <strong class="markup--strong markup--p-strong">
    建立国家警察数据库的建议，并为每个公民提供所谓的“犯罪倾向”评分
   </strong>
   。这就是少数派报告。
  </p>
  <p class="graf graf--p">
   英国公司已经在建筑行业秘密
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/uk-news/2015/feb/27/on-the-blacklist-building-firms-secret-information-on-workers" href="https://www.theguardian.com/uk-news/2015/feb/27/on-the-blacklist-building-firms-secret-information-on-workers" rel="noopener noreferrer" target="_blank">
    使用黑名单多年
   </a>
   ，成千上万的人被迫停止获得劳动合同。
   <strong class="markup--strong markup--p-strong">
    黑名单是公司用来阻止他们不喜欢的人的秘密工具
   </strong>
   。效果是毁灭性的。工人不知道自己的细节哪些会被检查，也无法查看信息是否准确。
  </p>
  <p class="graf graf--p">
   人们一直不知道自己在家中拍摄的照片视频、一切对话记录和互联网浏览历史，都在被跟踪监视，直到斯诺登揭露了政府最大的秘密。现在人们知道了，就应该尽快采取行动。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    少数派报告是大数据之恶的另一个高峰
   </strong>
   ，⚠️它已经不再针对具体行为，而是针对人们的想法 —— 甚至只是当权者所认为的“人们可能会有的想法”。
  </p>
  <p class="graf graf--p">
   毫无疑问，“预测犯罪系统”技术已经到来，但它不仅仅是预测*犯罪*这样的简单目的。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “犯罪”、“信用” 这些概念都是当权者为掩盖其真实用意而采取的障眼法
   </strong>
   。因为这些概念在大众心目中具有普遍标准的道德判断，于是它们很容易唬人。
  </p>
  <p class="graf graf--p">
   通过将这项技术与其他海量数据收集相结合，它本质上显然有能力强制进行自我审查；随着时间的推移，所有所谓的社会信用系统等技术都将应用惩罚和奖励，胡萝卜和大棒，就是现在在中国进行的东西。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    事实上胡萝卜比大棒更可怕
   </strong>
   ，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%83%a1%e8%90%9d%e5%8d%9c%ef%bc%9a%e4%b8%ad%e5%9b%bd%e5%bd%93%e5%b1%80%e5%a4%a7%e4%b8%be%e5%a5%96%e5%8a%b1%e6%9c%80%e9%a1%ba%e4%bb%8e%e7%9a%84%e5%85%ac%e6%b0%91/" href="https://www.iyouport.org/%e8%83%a1%e8%90%9d%e5%8d%9c%ef%bc%9a%e4%b8%ad%e5%9b%bd%e5%bd%93%e5%b1%80%e5%a4%a7%e4%b8%be%e5%a5%96%e5%8a%b1%e6%9c%80%e9%a1%ba%e4%bb%8e%e7%9a%84%e5%85%ac%e6%b0%91/" rel="noopener noreferrer" target="_blank">
    胡萝卜：中国当局大举奖励最顺从的公民
   </a>
   》，一份北上广的户籍有可能令一切关于人权和反抗的呼吁变得虚弱甚至无效。
  </p>
  <p class="graf graf--p">
   权力倡导者必须想到办法战胜这点。⚪️
  </p>
