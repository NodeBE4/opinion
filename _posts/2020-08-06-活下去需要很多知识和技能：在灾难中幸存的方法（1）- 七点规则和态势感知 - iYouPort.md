  <p class="graf graf--p">
   IYP 不是末日论拥护者，但我们相信，如果多数人没能掌握生存智慧，每时每刻都可能是“末日” ……
  </p>
  <p class="graf graf--p">
   灾祸无处不在，并不只有病毒那一种。并且它们随机而来，没有任何预兆。它们可以是自然的，也可以是人为的，更可能的是，以自然的形式开端，并在人为的愚蠢下加剧。
  </p>
  <p class="graf graf--p">
   经历了全球半年多的灾难噩梦，想必如今每个人都能理解这点。
  </p>
  <p class="graf graf--p">
   本系列内容的构想由此而来。
  </p>
  <p class="graf graf--p">
   它最终将帮助您习得一些令人兴奋的新技能，比如，如何摆脱基胶带和绳索、如何欺骗测谎仪、如何快速分辨某人是否在撒谎，或者是否想要陷你于困境 …… 等等。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    您想要掌握这些技能需要搭配一些同样重要的东西 —— 生存智慧
   </strong>
   。
  </p>
  <p class="graf graf--p">
   即 在任何紧急状态下都能迅速作出有效的反应。
  </p>
  <p class="graf graf--p">
   要做到这点，您必需遵循几条重要的规则。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    规则1#：应变能力的训练
   </strong>
  </h3>
  <p class="graf graf--p">
   生活中很少有事先能完全准备妥当的时刻。应变能力是生存的刚需。
  </p>
  <p class="graf graf--p">
   懂得在紧急情况下该做些什么固然重要，但能挽救您的依旧是应变能力。就如
   <a class="markup--anchor markup--p-anchor" data-href="https://t.me/iyouport/6962https://t.me/iyouport/6962" href="https://t.me/iyouport/6962https://t.me/iyouport/6962" rel="noopener noreferrer" target="_blank">
    Hakin9 的训练
   </a>
   ，它注重于测试您的实战技能，而不是单纯记住书本上的知识。
  </p>
  <p class="graf graf--p">
   生活很难按计划进行，而智慧就是，随时都能充分利用现有的一切去解决迎面而来的难题。这一点至关重要。
  </p>
  <p class="graf graf--p">
   您会发现，虽然反应灵敏、身体强健、力量强劲等等都很重要，但是，如果无法对潜在的新威胁随机应变，这些辅助条件的作用最终都是局限的。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    规则2#：自救
   </strong>
  </h3>
  <p class="graf graf--p">
   您唯一应该相信的事就是自救，而不是等待他人来保护你和你的亲人的安全。这超出个人哲学的范畴，它是本系列的主旨。
  </p>
  <p class="graf graf--p">
   现实中往往存在一些最初远不至于如此，而最终由于自救能力缺乏而导致惨剧的实例。
  </p>
  <p class="graf graf--p">
   依赖自己，而非他人，这是最简单的最靠谱的途径。
  </p>
  <p class="graf graf--p">
   在您足够强大的时候，您可以帮助更多人。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    规则3#：自知，而不是自满
   </strong>
  </h3>
  <p class="graf graf--p">
   您应该能够准确评估自己的能力和现实中手边可取的条件，高估或低估都是有害的。
  </p>
  <p class="graf graf--p">
   这说起来容易，但实际上并不容易做到这点。你肯定知道
   <strong class="markup--strong markup--p-strong">
    达克效应
   </strong>
   ，越是能力不足的人，越容易轻敌并高估自己。
  </p>
  <p class="graf graf--p">
   你在互联网上经常能见到那些夸夸其谈的人，看起来就像他们能力很强一样，其实完全不然；真正技能高超的人都是沉默寡言的，因为他们能看得很深，于是更理解什么叫山外有山。
  </p>
  <p class="graf graf--p">
   您不需要追求成为无敌的终结者，只需要对自己的实际能力有一个相对准确的评估，对环境有一个更加充分的认识。
  </p>
  <p class="graf graf--p">
   保持信心，但必须灵活运用。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-13233 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/5-1.jpg?resize=1100%2C733&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/5-1.jpg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/5-1.jpg?resize=300%2C200&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/5-1.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/5-1.jpg?resize=768%2C512&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/5-1.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/5-1.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="733" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/5-1.jpg?resize=1100%2C733&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    规则4#：行动起来以拯救生命
   </strong>
  </h3>
  <p class="graf graf--p">
   灾难都是突如其来的，而幸存者是那些没有被突如其来吓呆以至于失去行动力的人。
  </p>
  <p class="graf graf--p">
   行动起来，是拯救生命的关键。虽然这也并不容易。人们真的会因惊愕而呆滞，原本学到的东西瞬间似乎被清空了一样。这非常可怕。
  </p>
  <p class="graf graf--p">
   想象一个人拿着刀子，朝你扑过来，这种情况下赤手空拳的你只有两种选择 —— 逃离，或被刺伤。如果你被吓呆了，那结果只能是见血。
  </p>
  <p class="graf graf--p">
   显然这是简化的解释。但要知道，它的适用范围非常广泛。当威胁到来时，动起来是求生的第一要务。在坠机中幸存的人都是那些能够及时离开座位并行动起来的人。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    规则5#：洞察力决定一切
   </strong>
  </h3>
  <p class="graf graf--p">
   本系列中的内容基本简单易行，您可以在扔下手机后几分钟内实践它们。
  </p>
  <p class="graf graf--p">
   这是因为洞察力很重要。
  </p>
  <p class="graf graf--p">
   您可以了解到在多种情况下需要采取的行动、如何给他人留下对您有利的印象。这是
   <strong class="markup--strong markup--p-strong">
    身体和心理两方面结合的技巧
   </strong>
   。
  </p>
  <p class="graf graf--p">
   您真的知道自己留给他人的印象是什么吗？准确评估这点非常重要 —— 这样您才能准确调试这一印象，以令其符合您在不同场合状态下的切身利益。
  </p>
  <p class="graf graf--p">
   问自己这些问题：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    我看上去像受害者吗？
   </li>
   <li class="graf graf--li">
    我的家看上去像没入居住的地方吗？
   </li>
   <li class="graf graf--li">
    我走路和说话时的姿态是看起来信心满满还是弱不禁风？
   </li>
   <li class="graf graf--li">
    周围人都是什么情况？
   </li>
   <li class="graf graf--li">
    餐厅里坐在我后手方（盲区）的那个人是否可疑？
   </li>
   <li class="graf graf--li">
    刚才路过杂货店时是否被尾随了？
   </li>
   <li class="graf graf--li">
    ……
   </li>
  </ul>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    规则6#：注意基准线
   </strong>
  </h3>
  <p class="graf graf--p">
   基准线就是所谓的常态，这是情报界的重要理念之一。
  </p>
  <p class="graf graf--p">
   除非您非常熟悉某个特定场所的常态 —— 也就是说，那里是你的地盘 —— 否则，您必须知道
   <strong class="markup--strong markup--p-strong">
    “客场” 作战不应该采取哪些方式
   </strong>
   。
  </p>
  <p class="graf graf--p">
   想必大家都看过很多游击队的故事，游击战中行动者先将对方引入自己的地盘，然后往死里折腾他们。这就是基准线技巧。
  </p>
  <p class="graf graf--p">
   那么哪些是
   <strong class="markup--strong markup--p-strong">
    你的地盘
   </strong>
   呢？您的家、您生活的社区、您工作的街区 …… 等等，都应该是，也就是说，前提是，您必须对自己生活工作的区域
   <strong class="markup--strong markup--p-strong">
    保持一种强大的了解和熟悉。这就是观察和记忆的能力
   </strong>

    规则7#：不论任何时候都要保持态势感知

  </h3>
  <p class="graf graf--p">
   这是最后一条规则，也是一个基石。简单说就如中国俗语，“先下手者为强”，您和您的对手，谁能先下手，谁就能抢得主导权。要做到这点，您就必须对周边的态势保持准确的感知力。
  </p>
  <p class="graf graf--p">
   其底线在于，如果您对周边的细微变化不敏感，那么本系列中传授的所有技能都将落空。
  </p>
  <p class="graf graf--p">
   如果您低头玩手机、对美女目不转睛、专注于与朋友闲侃 …… 所有策略都不会生效。攻击者经常会采取社交工程手段，先转移你的注意力到诸如美女等闲杂无关的东西上，然后
   <strong class="markup--strong markup--p-strong">
    在你的盲区里制造致命袭击
   </strong>
   。
  </p>

   <strong class="markup--strong markup--p-strong">
    截然不同
   </strong>
   的两码事，千万不要混淆它们。态势感知需要您 *准确* 了解周边真正发生的东西，而不是靠想象力，您需要
   <strong class="markup--strong markup--p-strong">
    尽可能将盲区降至最低，并随时准备好排除那些想要误导你的烟雾。
   </strong>
  </p>
  <p class="graf graf--p">
   态势感知能帮您在遭受袭击之前采取行动制止威胁。这需要练习，还需要保持专注。但请相信这是可以做到的，而且是挽救性命的关键。

   于是接下来我们将首先介绍这一基石：态势感知。

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    反侦查行动力的基本功
   </strong>
  </h3>
  <p class="graf graf--p">
   任何一种包含对峙技巧的训练中，第一步要学习的就是态势感知。这是
   <strong class="markup--strong markup--p-strong">
    最专业的自卫练习
   </strong>
   ，此后才是如何躲避各种袭击、如何逃脱束缚、如何追逐 …… 等等。
  </p>
  <p class="graf graf--p">
   但是能保证您活下来的基本功永远都是对态势的感知能力。
  </p>
  <p class="graf graf--p">
   抢在危机发生之前采取行动，就是态势感知能帮你做到的。如果已经到了不得不拔枪的地步，那只能说明态势感知能力缺乏。
  </p>
  <p class="graf graf--p">
   一般人作出反应所需的时间大约是
   <strong class="markup--strong markup--p-strong">
    1.5秒
   </strong>
   ，这很慢，经过训练的人肯定要做得更快。
  </p>
  <p class="graf graf--p">
   如果您不能保持警觉和清醒，就很容易遭到攻击。
  </p>
  <p class="graf graf--p">
   如果你曾观察过街头、运动场或商场里大部分人的行为，你就会发现一个共同点 — — 绝大多数人不是在用手机聊天，就是在发送和阅读文本信息。
  </p>
  <p class="graf graf--p">
   如果是在玩手机，你就会低下头，对周围的环境没有觉察，就会对身边正发生的事情茫然无知；或者也可能你没有用手机聊天，却为工作所困，压力重重；或者只是在幻想接下来的假期该怎样度过（走神儿）。
  </p>
  <p class="graf graf--p">
   在很多情况下，如果人们能发挥态势感知能力，许多伤害实际上都是可以避免的。
  </p>
  <p class="graf graf--p">
   手机只是一个太过普遍的现象，但您必须知道，

   </strong>

   <strong class="markup--strong markup--p-strong">

   </strong>
   。

  </p>

    表现
   </strong>

  </p>
  <p class="graf graf--p">
   坐在公园长椅上读书、夜晚不假思索地走入一条黑暗的小巷，很容易将自己置于易受攻击的境地。
  </p>
  <p class="graf graf--p">
   如果您对周围环境中发生的事不加留意，您就会处于弱势，不管接下来发生什么，你都会毫无防备。

    黄色态势 ——
   </strong>
  </h3>

  </p>
  <p class="graf graf--p">
   黄色态势让你能意识到是否会碰到麻烦。记住这点很重要，
   <strong class="markup--strong markup--p-strong">
    保持黄色态势能让你远离红色态势 —— 那种你肯定不愿意发生的事
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    表现
   </strong>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    表现
   </strong>
   ：橙色态势意味着您已经需要开始准备反攻了。

   <strong class="markup--strong markup--p-strong">
    红色态势 ——
   </strong>
  </h3>

    表现

  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    保持黄色态势以避免红色态势
   </strong>
  </p>
  <p class="graf graf--p">
   如果你正在执行敏感的任务，一位漂亮女士举着香烟走过来借火，这时候你需要关注的不是打火机放在哪了，也不是漂亮女士的身材，而是周围是否还有其他动静，
   <strong class="markup--strong markup--p-strong">
    尤其是您的盲区 —— 背后
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果有人从后面袭击给你套上黑头套那位女士只是帮凶。
  </p>
  <p class="graf graf--p">
   这只是一个比喻。保持黄色态势的警觉就是要这样，这是一个逐渐适应的过程，但小的改变就足以让您在正确的道路上迈出一大步。比如，不要看手机，昂首走路，随时对四周有所察觉。
  </p>
  <p class="graf graf--p">
   倾听也是非常重要的，如果危险降临，还有另外一些策略能帮你变得更加敏感。
  </p>
  <p class="graf graf--p">
   我们下次再细说 …… ⚪️
  </p>

   <em class="markup--em markup--p-em">
    —— 未完待续 ——
