  <p class="graf graf--p">
   控制媒体就能控制大众的眼睛和脑子，控制大众的脑子就能控制整个社会，这就是为什么自从有媒体的年代开始就同时有了媒体操纵。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%aa%92%e4%bd%93%e6%93%8d%e6%8e%a7%ef%bc%9a%e6%88%98%e7%95%a5%e6%80%a7%e6%ac%ba%e9%aa%97%e7%9a%84%e9%87%8d%e5%a4%b4%e6%88%8f/" href="https://www.iyouport.org/%e5%aa%92%e4%bd%93%e6%93%8d%e6%8e%a7%ef%bc%9a%e6%88%98%e7%95%a5%e6%80%a7%e6%ac%ba%e9%aa%97%e7%9a%84%e9%87%8d%e5%a4%b4%e6%88%8f/" rel="noopener noreferrer" target="_blank">
     媒体操纵：战略性欺骗的重头戏
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   而如今，每个在互联网上的人都是一个媒体，每个人都可以创造内容；而在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e6%ad%a3%e5%9c%a8%e5%a6%82%e4%bd%95%e8%a2%ab%e6%93%8d%e7%ba%b5%ef%bc%9a%e8%bf%99%e6%98%af%e6%af%8f%e4%b8%aa%e4%ba%ba%e9%83%bd%e9%9a%be%e4%bb%a5%e8%ba%b2%e9%81%bf%e7%9a%84%e6%97%a0%e5%bd%a2/" href="https://www.iyouport.org/%e4%bd%a0%e6%ad%a3%e5%9c%a8%e5%a6%82%e4%bd%95%e8%a2%ab%e6%93%8d%e7%ba%b5%ef%bc%9a%e8%bf%99%e6%98%af%e6%af%8f%e4%b8%aa%e4%ba%ba%e9%83%bd%e9%9a%be%e4%bb%a5%e8%ba%b2%e9%81%bf%e7%9a%84%e6%97%a0%e5%bd%a2/" rel="noopener noreferrer" target="_blank">
    算法专制
   </a>
   的控制下，那些符合当权者议程的内容（不论真假）被更为广泛地获取和传播。
  </p>
  <p class="graf graf--p">
   本文即将谈论的不完全是传统的断章取义、
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ae%a9%e4%bd%a0%e7%9c%8b%e8%a7%81-%e6%ac%ba%e9%aa%97%e7%9a%84%e7%a7%af%e6%9e%81%e6%96%b9%e6%b3%95%e4%b9%8b%e4%ba%8c%ef%bc%885%ef%bc%89/" href="https://www.iyouport.org/%e8%ae%a9%e4%bd%a0%e7%9c%8b%e8%a7%81-%e6%ac%ba%e9%aa%97%e7%9a%84%e7%a7%af%e6%9e%81%e6%96%b9%e6%b3%95%e4%b9%8b%e4%ba%8c%ef%bc%885%ef%bc%89/" rel="noopener noreferrer" target="_blank">
    半真相
   </a>
   、
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%93%8d%e7%ba%b5%e5%bc%8f%e5%ae%a3%e4%bc%a0%ef%bc%9a%e4%ba%94%e4%b8%aa%e8%bf%87%e6%bb%a4%e5%99%a8%e5%92%8c%e8%af%86%e5%88%ab%e7%ba%bf%e7%b4%a2/" href="https://www.iyouport.org/%e6%93%8d%e7%ba%b5%e5%bc%8f%e5%ae%a3%e4%bc%a0%ef%bc%9a%e4%ba%94%e4%b8%aa%e8%bf%87%e6%bb%a4%e5%99%a8%e5%92%8c%e8%af%86%e5%88%ab%e7%ba%bf%e7%b4%a2/" rel="noopener noreferrer" target="_blank">
    逻辑谬误
   </a>
   等
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e6%88%98%e5%92%8c%e4%bf%a1%e6%81%af%e6%88%98/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e6%88%98%e5%92%8c%e4%bf%a1%e6%81%af%e6%88%98/" rel="noopener noreferrer" target="_blank">
    信息战
   </a>
   操纵手段，而是在高
   <strong class="markup--strong markup--p-strong">
    科技协助下的、精心制作的逼真的假货
   </strong>
   。
  </p>
  <p class="graf graf--p">
   以更微妙和更具变革性的方式操作内容已经变得越来越容易。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%bd%93%e6%81%b6%e6%84%8f%e8%80%85%e6%8e%8c%e6%8f%a1%e6%8a%80%e6%9c%af%ef%bc%9a%e4%bf%a1%e6%81%af%e6%88%98%e7%9a%84%e6%9c%aa%e6%9d%a5%e7%a9%b6%e7%ab%9f%e6%9c%89%e5%a4%9a%e5%8f%af%e6%80%95%ef%bc%9f/" href="https://www.iyouport.org/%e5%bd%93%e6%81%b6%e6%84%8f%e8%80%85%e6%8e%8c%e6%8f%a1%e6%8a%80%e6%9c%af%ef%bc%9a%e4%bf%a1%e6%81%af%e6%88%98%e7%9a%84%e6%9c%aa%e6%9d%a5%e7%a9%b6%e7%ab%9f%e6%9c%89%e5%a4%9a%e5%8f%af%e6%80%95%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     当恶意者掌握技术：信息战的未来究竟有多可怕？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   互联网上的视觉内容如此丰盛，以至于当可以创建高质量（或半高质量）的 *受控视觉内容* 时，人们很难区分真伪；并且至关重要的是，这需要时间、资源和技术。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/xk4vtiXjozY" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    人们说他们已经无法相信自己看到的任何东西 ——
   </span>
  </p>
  <p class="graf graf--p">
   造假泛滥的结果就是，它形成一种近似不可理喻的氛围，导致人们看什么都像是假的，
   <strong class="markup--strong markup--p-strong">
    纠正性信息也同时被埋没
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/OfG5J83Z8UU" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    制作深度造假仍然需要些技术技能，但是这种情况正在迅速改变 ——
   </span>
  </p>
  <p class="graf graf--p">
   目前，要在视频中完全逼真的模拟真实面孔的能力仍然需要一支经过专门培训和技术培训的庞大团队。
  </p>
  <p class="graf graf--p">
   但是，现在正是从手动合成图像（耗时且需要专业技能）到自动合成（仍然粗糙但速度更快并且不需要专业技能）的过渡阶段。
  </p>
  <p class="graf graf--p">
   人们通过社交媒体创建的大量内容推动了这一过渡：因为它提供了太多的数据以训练算法，从而会快速改进深度造假的工具。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #99cc00;">
    <strong class="markup--strong markup--p-strong">
     合成媒体和深度造假目前面临的主要检测挑战是什么？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/ZaOIvrGWr9g" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    用于检测的数据的质量很差、数量不足 ——
   </span>
  </p>
  <p class="graf graf--p">
   对于检测方法，您需要一定数量和质量的培训数据或其他信息，而这在特定技术中并不总是可用。
  </p>
  <p class="graf graf--p">
   研究人员经常没有足够的数据来分析某些造假的信息，要做到准确的检测，不仅需要更多信息来训练检测工具，还需要高质量的信息。
  </p>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    部分操纵比较难以识别 ——
   </span>
  </p>
  <p class="graf graf--p">
   很难检测视频何时被 “部分操纵”（相对于完全操纵或根本没有操纵来说）。但是，
   <strong class="markup--strong markup--p-strong">
    完全操纵的视频留下了很多数字指标
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    造假的音频更难以检测 ——
   </span>
  </p>
  <p class="graf graf--p">
   要检测造假音频是一组完全不同的挑战，目前为止在研究方面基本是空白。所以，
   <strong class="markup--strong markup--p-strong">
    如果有人上传一段音频，诉说一件耸人听闻的事，你需要小心
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    制作深度造假的人和检测深度造假的人，谁会最终赢得了这场 “猫鼠”游戏？这是个问题 ——
   </span>
  </p>
  <p class="graf graf--p">
   目前，用于
   <strong class="markup--strong markup--p-strong">
    造假技术
   </strong>
   的资源（人员和投资）远多于
   <strong class="markup--strong markup--p-strong">
    检测技术
   </strong>
   的资源（这也是DARPA MediFor 计划启动的原因之一）。越来越多的研究人员承认这是一个问题。
  </p>
  <p class="graf graf--p">
   我们不应该期望在检测中达到完美的性能水平，因为我们会看到深度造假技术
   <strong class="markup--strong markup--p-strong">
    不断升级
   </strong>
   以适应检测方法，并且造假的学习过程的本质是不断改善的。
  </p>
  <p class="graf graf--p">
   相反，最好用那种能
   <strong class="markup--strong markup--p-strong">
    够使尽可能多的人难以做出逃避检测的造假产品的检测工具
   </strong>
   来定义成功。
  </p>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    技术/社交媒体平台之间也缺乏协调，无法跟踪虚假信息和被操纵的内容并将其删除 ——
   </span>
  </p>
  <p class="graf graf--p">
   我们认识到我们可以解决此问题，
   <strong class="markup--strong markup--p-strong">
    但这将同时带来审查问题，导致弊大于利
   </strong>
   。技术平台与该特定研究社区正在产生的东西之间似乎也存在脱节。
  </p>
  <p class="graf graf--p">
   平台内的确有一些模型，用于与合作伙伴一起处理有问题的内容数据，例如有关恐怖分子的宣传内容或儿童色情图像，但是我们需要探索其中哪些部分与伪造品和虚假信息有关。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/2Xjwhhkm_XQ" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    深度造假和虚假信息的检测可能无法通过技术解决。但技术会有帮助 ——
   </span>
  </p>
  <p class="graf graf--p">
   应对深度造假需要多学科的联合努力：这不仅需要将其视为技术问题，而且，还应将其视为心理学、人类学、和新闻学问题。
  </p>
  <p class="graf graf--p">
   为此，需要确保将社会心理学研究纳入检测工具的使用和开发中，并确保我们将研究重点放在视频上，而不仅仅是文本上。
  </p>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    我们需要利用面对错误信息的人们现有的专业知识和经验 ——
   </span>
  </p>
  <p class="graf graf--p">
   这包括来自现有开源情报社区以及现有媒体取证社区的专业知识。
  </p>
  <p class="graf graf--p">
   我们还需要将系统思想家、认知科学家、以及更多前线团体和受影响社区的观点，也纳入其中。
  </p>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    技术解决方案也将是多方面的，并依赖于
    <strong class="markup--strong markup--p-strong">
     在数字级别
    </strong>
    评估可疑多媒体信息，考虑视频是否在科学范围内运行以及从其他来源了解到的事件信息 ——
   </span>
  </p>
  <p class="graf graf--p">
   例如，DARPA MediFor 程序正在尝试结合多种不同的媒体完整性衡量标准，以反映包括数字完整性（例如，像素的一致性或压缩性）、物理完整性（例如，违反了物理定律的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ae%a9%e9%98%b4%e5%bd%b1%e8%af%b4%e8%af%9d%ef%bc%9a%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e5%b0%8f%e6%b8%b8%e6%88%8f/" href="https://www.iyouport.org/%e8%ae%a9%e9%98%b4%e5%bd%b1%e8%af%b4%e8%af%9d%ef%bc%9a%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e5%b0%8f%e6%b8%b8%e6%88%8f/" rel="noopener noreferrer" target="_blank">
    阴影、光线或运动
   </a>
   ）、以及语义完整性（例如，如果天气与特定日期和时间所在位置的

     </span>
    </p>
    <p class="graf graf--p">
     用来确定图像的一部分是否与其他区域进行了不同的处理。
    </p>
    <p class="graf graf--p">
     <span style="color: #ff6600;">
      需要将图像、视频链接到特定设备或设备类型的工具 ——
     </span>
    </p>
    <p class="graf graf--p">
     确定它们的传播已经通过了哪些社交网络。
    </p>
    <p class="graf graf--p">
     <span style="color: #ff6600;">
      需要专注于音频的技术包括可促进识别说话者的工具 ——
     </span>
    </p>
    <p class="graf graf--p">
     这显然是个难点。
    </p>
    <p class="graf graf--p">
     <span style="color: #ff6600;">
      需要反向视频搜索 ——
     </span>
    </p>
    <p class="graf graf--p">
     反向图像搜索已经很成熟，一般来说，开源情报社区会建议抓取视频的一帧，然后依照验证图像的方法来验证视频。但有些时候，对于那些动作很快的视频，抓取帧不很容易，并且对于不娴熟的调查人员来说，精准抓取也不很容易。
    </p>
    <p class="graf graf--p">
     因此，目前迫切需要那种能够像搜图一样简单的反向搜索视频工具。
    </p>
    <p class="graf graf--p">
     <span style="color: #ff6600;">
      需要图像标记工具——
     </span>
    </p>
    <p class="graf graf--p">
     研究人员需要在开放平台和封闭群组中都可以使用的
     <strong class="markup--strong markup--p-strong">
      标记可疑视觉内容
     </strong>
     的方法，
     <strong class="markup--strong markup--p-strong">
      并使其可以跨平台携带
     </strong>
     。
    </p>
    <p class="graf graf--p">
     跨平台非常重要。因为很多内容被一次揭穿并没有用，它会继续传播不断扩散，而揭露永远跟不上虚假的扩散速度。
    </p>
    <p class="graf graf--p">
     传播途中图像会被跨平台所清洗，因此，这里的关键是能够确定图像的来源并追溯到源。在信息错误的情况下，返回原始来源（谁上传的、何时上传的、及其他元数据）是关键。
    </p>
    <p class="graf graf--p">
     <span style="color: #ff6600;">
      需要基于区块链的工具，用于记录和跟踪原始图像并创建 “信任区域” ——
     </span>
    </p>
    <p class="graf graf--p">
     具有不同的社区作为共识节点，
     <strong class="markup--strong markup--p-strong">
      并且能够向音频视频内容添加已确认的元数据
     </strong>
     。
    </p>
    <p class="graf graf--p">
     很多记者和调查人员不会仅仅依靠这些，而是​​希望运行自己的验证流程，但这样做可以提供信任信号并实现更好的来源验证。
    </p>
    <p class="graf graf--p">
     <span style="color: #ff6600;">
      需要网络映射工具 ——
     </span>
    </p>
    <p class="graf graf--p">
     展示谁在创造、收集和分发宣传性的和操纵性的视觉内容（主要行为者）。
    </p>
    <p class="graf graf--p">
     <span style="color: #ff6600;">
      需要第三方已知的错误信息示例数据库、以及受信任的分析实体 ——
     </span>
    </p>
    <p class="graf graf--p">
     比如一种基于Web的工具，用于评估图像/图形的完整性，并为研究人员提供进行进一步分析的标志。例如
     <a class="markup--anchor markup--p-anchor" data-href="http://www.ithenticate.com/" href="https://www.ithenticate.com/" rel="noopener noreferrer" target="_blank">
      IThenticate
     </a>
     那样。
    </p>
    <h3 class="graf graf--p">
     <span style="color: #339966;">
      <strong class="markup--strong markup--p-strong">
       开源情报、社交新闻采集、以及深度造假与媒体法证研究人员今后都需要学习什么？
      </strong>
     </span>
    </h3>
    <p class="graf graf--p">
     <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/yFdE0MCqwVI" width="750">
     </iframe>
    </p>
    <p class="graf graf--p">
     <span style="color: #ff6600;">
      在研究和实践者社区之间
      <strong class="markup--strong markup--p-strong">
       更好地沟通
      </strong>
      的技巧 ——
     </span>
    </p>
    <p class="graf graf--p">
     由于多种原因，研究人员和从业人员没有互相交流。
    </p>
    <p class="graf graf--p">
     这是因为缺乏资源来支持此类对话，以及如何匹配学术界、民间社会、政府、资助者和新闻业的激励结构。以及如何理解每个部门的独特伦理关注的问题。
    </p>
    <p class="graf graf--p">
     缺乏很好地激励或协调合作结构。对于记者来说，这也是在资源和新闻编辑室预算缩减的背景下。
    </p>
    <p class="graf graf--p">
     但是，所有人都同意需要更好地联系。紧密的联系和社区将使这两个小组能够相互理解彼此的工具并以实用的方式传播研究成果。
    </p>
    <p class="graf graf--p">
     <span style="color: #ff6600;">
      需要弥合技术社区和记者社区之间的知识鸿沟 ——
     </span>
    </p>
    <p class="graf graf--p">
     有很多尚未开发的资源/功能，记者也不知道；但是，如何在工作流程中作为产品使用它们存在障碍问题；另一面，技术研究人员希望了解现实情况并处理现实数据。
    </p>
    <p class="graf graf--p">
     <span style="color: #ff6600;">
      技术社区应该尽可能共享现有的工具和新工具的进度，并提供有关特定媒体取证工具的动手培训 ——
     </span>
    </p>
    <p class="graf graf--p">
     取证界采取的措施与开源情报界所知道的东西存在明显的知识鸿沟。比如，现有的用于复制粘贴检测的​​工具，以及正在开发的用于跟踪特定相机的数字指纹的工具，修改图像的工具，以及验证内容经过了哪些社交网络的工具。
    </p>
    <p class="graf graf--p">
     就是说，在研究范围之外，许多技术尚未商业化也不大易于普通人使用。更好的数据共享将加速此过程，以完善工具以解决开源情报社区正在解决的问题类型。
    </p>
    <p class="graf graf--p">
     <span style="color: #ff6600;">
      建立多元化的取证和开源情报专业知识 ——
     </span>
    </p>
    <p class="graf graf--p">
     需要找到各种方法来支持在各种环境中拥有本地图像取证和开源情报调查专业知识，因为了解本地情况很重要，在全球范围内都会遇到错误信息和虚假信息。
    </p>
    <p class="graf graf--p">
     <span style="color: #ff6600;">
      更多的社区建设活动 ——
     </span>
    </p>
    <p class="graf graf--p">
     我们之间需要更多的联系，经常的聚会。就特定事件将研究人员和开源情报社区召集在一起，就不同的错误信息模式在全球范围内举行会议，共同探讨解决方案。
    </p>
    <p class="graf graf--p">
     也可能需要有规模较小的、更多本地的聚会，以讨论如何具体推进协作。
    </p>
    <p class="graf graf--p">
     <span style="color: #ff6600;">
      更详细的讲习班和知识交流可进一步提高理解水平 ——
     </span>
    </p>
    <p class="graf graf--p">
     有关此操作的一些想法包括：
    </p>
    <ol class="postList">
     <li class="graf graf--li">
      使用以人为中心的设计/设计思维/系统思维方式探索彼此的需求并进行深入的工作，安排结构化的时间来准备原型或想法的起点，以获得可能的解决方案。
     </li>
     <li class="graf graf--li">
      分享实际应用研究和技术的实例 —— 识别操纵、操纵目标、以及操纵方式。
     </li>
     <li class="graf graf--li">
      特定媒体取证工具的动手培训。
     </li>
     <li class="graf graf--li">
      围绕特定主题的研讨会，包括测试工具。
     </li>
     <li class="graf graf--li">
      研讨会如何为记者和公众提供实用的研究翻译方法。
     </li>
     <li class="graf graf--li">
      正在进行的知识交流，以帮助了解彼此的时间表和工作流程
     </li>
     <li class="graf graf--li">
      花一定时间与他人合作或在特定组织中帮助开发工具
     </li>
     <li class="graf graf--li">
      共享对影响研究人员和调查人员的威胁模型的理解。
     </li>
    </ol>
    <p class="graf graf--p">
     <span style="color: #ff6600;">
      需要多个论坛来进行持续的对话和提醒 ——
     </span>
    </p>
    <ul class="postList">
     <li class="graf graf--li">
      一个通知系统/电子邮件列表/ Twitter 机器人，让主要的开源情报研究人员、深度造假研究人员知道已识别出的受到操纵的视频或图像，或分享有关工具成功或失败的反馈。
     </li>
     <li class="graf graf--li">
      关于信息战发生时的警报列表，我们需要彼此的帮助。
     </li>
     <li class="graf graf--li">
      参与者可以通过这样的会议来建立主题专家的联系人资源列表。
     </li>
     <li class="graf graf--li">
      松散的团队/通过电子邮件列表/ Whatsapp群组或其他论坛，以分享知识、棘手的案例或示例、开发项目、向其他专家提出问题（注意：只要我们已同意共同的准则和道德规范，就可以）
     </li>
    </ul>
    <p class="graf graf--p">
     <span style="color: #ff6600;">
      支持有关特定问题的结构化对话 ——
     </span>
    </p>
    <p class="graf graf--p">
     有些问题已经是挑战，而其他问题在部门之间的结构化对话可能有价值的情况下将变得越来越重要：例如，关于我们如何防止恶意行为者利用知识；或关于我们如何考虑和反馈未决的规则。
    </p>
    <p class="graf graf--p">
     <span style="color: #ff6600;">
      制定共享的分类法 ——
     </span>
    </p>
    <p class="graf graf--p">
     因为并非所有人都同意关于错误/虚假信息的通用术语，因此很难创建干净的数据集进行分析。所以需要先统一概念和标准。
    </p>
    <p class="graf graf--p">
     <span style="color: #ff6600;">
      分享培训机会 ——
     </span>
    </p>
    <p class="graf graf--p">
     需要在结构化设置中相互学习。培训机会的一个示例是 UC Denver 的国家媒体取证中心提供的新的深度造假取证课程。开源情报界有很多培训机会（虽然价格不便宜）。
    </p>
    <p class="graf graf--p">
     <span style="color: #ff6600;">
      共享数据集和示例 ——
     </span>
    </p>
    <p class="graf graf--p">
     在道德要求内，共享现实世界的数据和内容示例以为研究提供信息可能非常有用。
    </p>
    <p class="graf graf--p">
     还应注意，某些数据和一些示例无法公开发布，并且不能确定哪些内容可以合乎道德地分享，以学习和理解在此问题上可以做得更好的方法。这可能需要第三方的协助。
    </p>
    <p class="graf graf--p">
     相关各方之间也需要建立
     <strong class="markup--strong markup--p-strong">
      平等的关系
     </strong>
     ，以免造成关于 *谁从数据中受益* 的权力失衡。
    </p>
    <p class="graf graf--p">
     <span style="color: #ff6600;">
      数据托管服务？ ——
     </span>
    </p>
    <p class="graf graf--p">
     集体探索的一种选择是查看数据托管服务的模型，该模型应该提供一种在道德层面适当的使用彼此数据的方式，但具有明确的保护和限制。
    </p>
    <p class="graf graf--p">
     <span style="color: #ff6600;">
      协作以创新，分享想法以进行验证 ——
     </span>
    </p>
    <p class="graf graf--p">
     例如，建立一个全国性或全球性的ENF信号收集器，这些信号随位置而变化并且可以在视频中检测到，以更好地识别真正的音频或视频记录的位置。
    </p>
    <h3 class="graf graf--p">
     最后
    </h3>
    <p class="graf graf--p">
     如上，需要做的工作非常多。尤其是技术社区应该努力与活动家和调查社区合作，以帮助技术更好地应用在现实中最需要的地方。
    </p>
    <p class="graf graf--p">
     如果您是开发者并有意愿予以协助，欢迎联系我们。⚪️
    </p>

    </p>
