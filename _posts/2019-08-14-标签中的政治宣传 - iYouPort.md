      </strong>

  <p class="graf graf--p">
   让我们从测试游戏开始：快！迅速说出一些独裁者的名字！
  </p>
  <p class="graf graf--p">
   我敢打赌你们中的大多数人只回答了几个相同的名字：阿萨德、普京、卡斯特罗、金正恩、卡扎菲、马杜罗……这不是因为他们是世界上
   <strong class="markup--strong markup--p-strong">
    绝无仅有
   </strong>
   的独裁者，远远不是，恰恰是因为这些名字是主流媒体最常标记的名字。
  </p>
  <p class="graf graf--p">
   民主是美国人的神圣理想。政治家和媒体们告诉我们，美国“代表”民主，反对各地的独裁统治，这也是美国必须继续在世界范围内以外交和军事方式参与其中的一个原因。
  </p>
  <p class="graf graf--p">
   然而，自由之家的“全球自由度报告”研究发现，49个国家 — —
   <strong class="markup--strong markup--p-strong">
    超过四分之一的世界政府 — — 是“不自由”的
   </strong>
   ，他们在其网站和
   <a class="markup--anchor markup--p-anchor" data-href="https://freedomhouse.org/sites/default/files/Feb2019_FH_FITW_2019_Report_ForWeb-compressed.pdf" href="https://freedomhouse.org/sites/default/files/Feb2019_FH_FITW_2019_Report_ForWeb-compressed.pdf" rel="noopener noreferrer" target="_blank">
    报告中
   </a>
   对这些国家的描述与“独裁统治”一词交替使用。那么为什么，大多数政治上精明的人
   <a class="markup--anchor markup--p-anchor" data-href="https://www.sporcle.com/games/TJL/finally_relevant/results" href="https://www.sporcle.com/games/TJL/finally_relevant/results" rel="noopener noreferrer" target="_blank">
    都不知道
   </a>
   所有世界上存在这些独裁者的名字？为什么他们不是家喻户晓的名字，比如阿萨德和其他人？
   <strong class="markup--strong markup--p-strong">
    是因为美国为他们中四分之三的国家提供军事援助包括培训和武器销售
   </strong>
   正如
   <a class="markup--anchor markup--p-anchor" data-href="https://truthout.org/articles/us-provides-military-assistance-to-73-percent-of-world-s-dictatorships/" href="https://truthout.org/articles/us-provides-military-assistance-to-73-percent-of-world-s-dictatorships/" rel="noopener noreferrer" target="_blank">
    Rich Whitney 的研究
   </a>
   所揭示的那样？
  </p>
  <p class="graf graf--p">
   定义和量化什么是和什么不构成独裁是一个众所周知的棘手工作，“自由之家”强烈的保守派政治偏见使其列表和判断更加值得怀疑。首先，这个自称为“非政府”的组织实际上是由华盛顿资助的，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.ft.com/content/364cda0e-c016-11da-939f-0000779e2340" href="https://www.ft.com/content/364cda0e-c016-11da-939f-0000779e2340" rel="noopener noreferrer" target="_blank">
    华盛顿于2006年聘请“自由之家”
   </a>
   在伊朗开展“秘密活动” — — 即政权更迭行动。
  </p>
  <p class="graf graf--p">
   Whitney 和许多其他负责编制该自由度评分名单的人自己也
   <strong class="markup--strong markup--p-strong">
    承认他们的方法论包括“
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/news/monkey-cage/wp/2017/11/07/why-do-we-trust-certain-democracy-ratings-new-research-explains-hidden-biases/?utm_term=.e7c10261afb8" href="https://www.washingtonpost.com/news/monkey-cage/wp/2017/11/07/why-do-we-trust-certain-democracy-ratings-new-research-explains-hidden-biases/?utm_term=.e7c10261afb8" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     预感和直觉
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ”
   </strong>
   。正如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.penguinrandomhouse.com/books/78912/manufacturing-consent-by-edward-s-herman-and-noam-chomsky/9780375714498/" href="https://www.penguinrandomhouse.com/books/78912/manufacturing-consent-by-edward-s-herman-and-noam-chomsky/9780375714498/" rel="noopener noreferrer" target="_blank">
    许多学者所指出
   </a>
   的那样，“自由之家”在支持美国同盟国的独裁和攻击尼加拉瓜等敌国方面也有着悠久的历史。这一评级具有
   <a class="markup--anchor markup--p-anchor" data-href="https://link.springer.com/article/10.1007%2FBF02687584" href="https://link.springer.com/article/10.1007%2FBF02687584" rel="noopener noreferrer" target="_blank">
    强烈的保守派意识形态偏见
   </a>
   。
  </p>
  <p class="graf graf--p">
   尽管如此，它的指数是有用的，因为它是这个问题上
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/news/monkey-cage/wp/2017/11/07/why-do-we-trust-certain-democracy-ratings-new-research-explains-hidden-biases/?utm_term=.e7c10261afb8" href="https://www.washingtonpost.com/news/monkey-cage/wp/2017/11/07/why-do-we-trust-certain-democracy-ratings-new-research-explains-hidden-biases/?utm_term=.e7c10261afb8" rel="noopener noreferrer" target="_blank">
    最常被引用
   </a>
   的来源，人们可以假设它不会错误地将美国盟友称为独裁者。
  </p>
  <p class="graf graf--p">
   当你看到被“自由之家”描述为独裁政府的政府时，不用问也肯定是在企业媒体中找到同样的描述。然而，那些被华盛顿所支持的非自由政府 — — 很少被新闻界称为独裁统治。事实上，对他们的报道都很少。
  </p>
  <p class="graf graf--p">
   让我们来看看自由之家定义的对于接受美国军事援助的四个“独裁者”的新闻报道，他们最近都出现在新闻中：喀麦隆的 Paul Biya、埃及的 Abdel el-Sisi、阿尔及利亚的 Abdelaziz Bouteflika、和哈萨克斯坦的 Nursultan Nazarbayev。
  </p>
  <p class="graf graf--p">
   1、喀麦隆
  </p>
  <p class="graf graf--p">
   86岁的 Paul Biya 是世界上任职时间最长的非皇家国家元首，他最近还赢得了第七届的任期，
   <a class="markup--anchor markup--p-anchor" data-href="https://foreignpolicy.com/2018/10/22/cameroons-paul-biya-gives-a-master-class-in-fake-democracy/" href="https://foreignpolicy.com/2018/10/22/cameroons-paul-biya-gives-a-master-class-in-fake-democracy/" rel="noopener noreferrer" target="_blank">
    外交政策杂志
   </a>
   称其为“闹剧”。由于政府的
   <strong class="markup--strong markup--p-strong">
    人权侵犯将该国推向了内战的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bbc.co.uk/news/world-africa-45723211" href="https://www.bbc.co.uk/news/world-africa-45723211" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     边缘
    </strong>
   </a>
   ，喀麦隆最近一直处于新闻中。“自由之家”认为它是世界上最不自由的国家之一。
  </p>
  <p class="graf graf--p">
   然而，即便在这样讨论时，Biya 继续被主流媒体事实地提出，而不需要称他为“独裁者”。“纽约时报”（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2018/10/06/world/africa/cameroon-election-biya-ambazonia.html" href="https://www.nytimes.com/2018/10/06/world/africa/cameroon-election-biya-ambazonia.html" rel="noopener noreferrer" target="_blank">
    10/6/18
   </a>
   ）委婉地称他为“世界上任职时间最长的总统之一”。从报道的范围来看，即使按照自由之家的标准，读者
   <strong class="markup--strong markup--p-strong">
    也不会知道
   </strong>
   他是独裁者。
  </p>
  <p class="graf graf--p">
   事实上，纽约时报报道他有20年之久了，但 Biya 从未被描述为“独裁者”、“暴君”或任何其他类似的名称。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    即便在 Biya 受到指责时，主流媒体的报道语气也不那么谴责
   </strong>
   ，而且批评得也更为温和。美国之音（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.voanews.com/a/cameroon-biya-turns-86-critics-say-time-for-change/4786641.html" href="https://www.voanews.com/a/cameroon-biya-turns-86-critics-say-time-for-change/4786641.html" rel="noopener noreferrer" target="_blank">
    2009年2月14日
   </a>
   ）指出，Biya 决定取消总统任期限制（意味着他可以终身统治），一些“批评者”称此举为“专制”。
  </p>
  <p class="graf graf--p">
   2、埃及
  </p>
  <p class="graf graf--p">
   阿卜杜勒·西西（Gen Abdel el-Sisi）将军于2013年在军事政变中上台，政变推翻了民主选举的总统穆罕默德·穆尔西（Mohamed Morsi）政府。Sisi 最近宣布了他的延长任期计划。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “纽约时报”关于这一主题的文章（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2019/02/14/world/middleeast/egypt-sisi.html" href="https://www.nytimes.com/2019/02/14/world/middleeast/egypt-sisi.html" rel="noopener noreferrer" target="_blank">
    2/14/19
   </a>
   ）指出，这将“进一步巩固他的专制统治”，甚至还指出
   <strong class="markup--strong markup--p-strong">
    他已经监禁了“数万”反对者、完全控制了互联网并接管了法院
   </strong>
   。然而，报道远远没有把他称为独裁者。事实上，该报道注意到他得到了来自世界各地的“强大支持”，并
   <strong class="markup--strong markup--p-strong">
    被美国和法国认可，被视为该地区“反对伊斯兰激进分子的堡垒”。
   </strong>
  </p>
  <p class="graf graf--p">
   其他媒体也采用了这种趋势。 CNN（
   <a class="markup--anchor markup--p-anchor" data-href="https://edition.cnn.com/2019/02/13/africa/egypt-presidential-term-amendments-sisi-intl/index.html" href="https://edition.cnn.com/2019/02/13/africa/egypt-presidential-term-amendments-sisi-intl/index.html" rel="noopener noreferrer" target="_blank">
    2/13/19
   </a>
   ）简称他为“现任总统 Abdel Fattah el-Sisi”，而NPR（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.npr.org/2019/02/14/694675332/with-constitution-changes-egypts-president-could-stay-in-power-until-2034" href="https://www.npr.org/2019/02/14/694675332/with-constitution-changes-egypts-president-could-stay-in-power-until-2034" rel="noopener noreferrer" target="_blank">
    2009年2月14日
   </a>
   ）同样称之为 “埃及总统 Abdel-Fattah el-Sissi”。
  </p>
  <p class="graf graf--p">
   虽然准确地注意到了 Sisi “前所未有的镇压异议”、其中监禁了“成千上万的人”，但 BBC 的报告（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bbc.co.uk/news/world-middle-east-47252315" href="https://www.bbc.co.uk/news/world-middle-east-47252315" rel="noopener noreferrer" target="_blank">
    2/15/19
   </a>
   ）依旧中立称他为“总统”，并且只表明他的几乎无限期地统治“被该提案的反对者”描述为“迈向专制主义的又一步。” — — 总之，主流媒体认为这样的人也远远不能成为一个完全成熟的“独裁者”。
  </p>
  <p class="graf graf--p">
   3、阿尔及利亚
  </p>
  <p class="graf graf--p">
   在他宣布再次出任总统之后，阿尔及利亚各地爆发了大规模的抗议活动，抗议82岁的 Abdelaziz Bouteflika。起义已经非常严重，他已承诺退出。
  </p>
  <p class="graf graf--p">
   尽管
   <strong class="markup--strong markup--p-strong">
    以欺诈手段上台并且用铁拳统治了20年
   </strong>
   ，媒体机构（例如 CNN
   <a class="markup--anchor markup--p-anchor" data-href="https://edition.cnn.com/2019/03/11/africa/abdelaziz-bouteflika-algeria-fifth-term-intl/index.html" href="https://edition.cnn.com/2019/03/11/africa/abdelaziz-bouteflika-algeria-fifth-term-intl/index.html" rel="noopener noreferrer" target="_blank">
    3/11/19
   </a>
   ；路透社，
   <a class="markup--anchor markup--p-anchor" data-href="https://af.reuters.com/article/topNews/idAFKCN1RE0LQ-OZATP" href="https://af.reuters.com/article/topNews/idAFKCN1RE0LQ-OZATP" rel="noopener noreferrer" target="_blank">
    4/2/19
   </a>
   ）仍然没有将他描述为独裁者，而 BBC（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bbc.co.uk/news/world-africa-47456114" href="https://www.bbc.co.uk/news/world-africa-47456114" rel="noopener noreferrer" target="_blank">
    3 / 6/19
   </a>
   ）简单地称他为“深居简出的总统”。
  </p>
  <p class="graf graf--p">
   事实上，如果你阅读卫报的报道（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2019/apr/01/algerias-president-abdelaziz-bouteflika-to-step-down-by-28-april" href="https://www.theguardian.com/world/2019/apr/01/algerias-president-abdelaziz-bouteflika-to-step-down-by-28-april" rel="noopener noreferrer" target="_blank">
    4/1/19
   </a>
   ）甚至可能误解他是
   <strong class="markup--strong markup--p-strong">
    民主人士的缩影
   </strong>
   ；“纽约时报”（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/aponline/2019/03/12/world/middleeast/ap-ml-algeria-bouteflika.html" href="https://www.nytimes.com/aponline/2019/03/12/world/middleeast/ap-ml-algeria-bouteflika.html" rel="noopener noreferrer" target="_blank">
    2009年12月3日
   </a>
   ）也
   <strong class="markup--strong markup--p-strong">
    赞扬
   </strong>
   Bouteflika 为北非国家“恢复稳定”作出贡献、并“恢复了国家军队的荣誉”。与喀麦隆的 Biya 一样，Bouteflika 在过去20年的报道中从未被描述为“独裁者”。
  </p>
  <p class="graf graf--p">
   4、哈萨克斯坦
  </p>
  <p class="graf graf--p">
   接受美国军事援助的另一位统治者是 Nursultan Nazarbayev。这位85岁的自1989年以来一直执政的人最近才宣布将下台。据透露，哈萨克斯坦首都阿斯塔纳将以他的名字命名（这种情况在民主国家并不常见 — — 不过考虑到华盛顿的名字，可以说不是例外的）
  </p>
  <p class="graf graf--p">
   Nazarbayev
   <strong class="markup--strong markup--p-strong">
    在打击言论、新闻和宗教自由方面有着
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hrw.org/world-report/2015/country-chapters/kazakhstan" href="https://www.hrw.org/world-report/2015/country-chapters/kazakhstan" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     悠久的历史
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，并对他的政治对手使用酷刑
   </strong>
   。尽管如此，他在主流媒体上依然得到了积极的表现，纽约时报（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2019/03/19/world/asia/kazakhstan-nazarbayev-resigns.html" href="https://www.nytimes.com/2019/03/19/world/asia/kazakhstan-nazarbayev-resigns.html" rel="noopener noreferrer" target="_blank">
    2009年3月19日
   </a>
   ）只是将他称为“哈萨克斯坦的长期总统”；美联社（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.apnews.com/ba03be7eb49a4794ae23b059656ca75d" href="https://www.apnews.com/ba03be7eb49a4794ae23b059656ca75d" rel="noopener noreferrer" target="_blank">
    2009年3月19日
   </a>
   ）称他是“哈萨克斯坦独立以来的唯一领导人”，还称赞他“保持稳定”；路透社（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.reuters.com/article/us-kazakhstan-president/president-of-kazakhstan-nursultan-nazarbayev-resigns-idUSKCN1R01N1" href="https://www.reuters.com/article/us-kazakhstan-president/president-of-kazakhstan-nursultan-nazarbayev-resigns-idUSKCN1R01N1" rel="noopener noreferrer" target="_blank">
    2009年3月19日
   </a>
   ）居然声称他是一位“广受欢迎”的领导人，上述主流媒体都没有使用“独裁者”的绰号。
  </p>
  <p class="graf graf--p">
   华盛顿邮报编辑委员会（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/opinions/global-opinions/kazakhstans-soviet-era-ruler-will-step-down--but-wont-be-soon-forgotten/2019/03/29/810bb2d8-4f3f-11e9-a3f7-78b7525a8d5f_story.html" href="https://www.washingtonpost.com/opinions/global-opinions/kazakhstans-soviet-era-ruler-will-step-down--but-wont-be-soon-forgotten/2019/03/29/810bb2d8-4f3f-11e9-a3f7-78b7525a8d5f_story.html" rel="noopener noreferrer" target="_blank">
    3/29/19
   </a>
   ）对他的任期发表了热烈的评价 — — 该评价让他成为一个“有远见的领导者”，一个前钢铁工人“领导前苏联国家摆脱帝国的混乱内爆”，声称他带领哈萨克斯坦“进入一个和平、繁荣的新时代“，同时“建立民族认同”并阻止任何种族冲突。
  </p>
  <p class="graf graf--p">
   这篇社论虽然注意到了他的“政治强人手腕”、并且“以更宽容的态度来看待异议和民主会更明智”，但是，该社论认为他的压制是“有道理的”，并总结说“他不会很快忘记”他的人民。
  </p>
  <p class="graf graf--p">
   双重标准 — —
  </p>
  <p class="graf graf--p">
   双重标准通过不断的媒体引用敌对国家作为“独裁者”来强调，无论标签是否有保证。“华盛顿邮报”（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/opinions/2019/01/04/cynicism-evo-moraless-reelection-bid-bolivia/?utm_term=.59ee9c8f4fcd" href="https://www.washingtonpost.com/opinions/2019/01/04/cynicism-evo-moraless-reelection-bid-bolivia/?utm_term=.59ee9c8f4fcd" rel="noopener noreferrer" target="_blank">
    1/4/19
   </a>
   ）将左派玻利维亚总统 Evo Morales 描述为“希望成为委内瑞拉式的独裁者”，而卫报（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2017/dec/03/evo-morales-bolivia-president-election-limits" href="https://www.theguardian.com/world/2017/dec/03/evo-morales-bolivia-president-election-limits" rel="noopener noreferrer" target="_blank">
    12/3/18
   </a>
   ）则警告玻利维亚成为“即将来临的委内瑞拉 — 古巴”式的“独裁统治”。
  </p>
  <p class="graf graf--p">
   尼加拉瓜的左派 Sandinista 政府也被称为“独裁政权”。 “纽约时报”（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2018/08/02/opinion/a-dictatorship-is-rising-in-my-country-again.html" href="https://www.nytimes.com/2018/08/02/opinion/a-dictatorship-is-rising-in-my-country-again.html" rel="noopener noreferrer" target="_blank">
    8/2/18
   </a>
   ）发表了一篇来自尼加拉瓜的评论文章，标题为“独裁统治再次在我的国家崛起”；许多其他网点都称他为“独裁者”（比如 经济学人，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.economist.com/the-americas/2018/07/12/daniel-ortega-is-causing-a-bloodbath-in-nicaragua" href="https://www.economist.com/the-americas/2018/07/12/daniel-ortega-is-causing-a-bloodbath-in-nicaragua" rel="noopener noreferrer" target="_blank">
    7/12/18
   </a>
   ；时代，
   <a class="markup--anchor markup--p-anchor" data-href="http://time.com/5553657/carlos-chamorro-nicaragua-press-freedom/" href="https://time.com/5553657/carlos-chamorro-nicaragua-press-freedom/" rel="noopener noreferrer" target="_blank">
    3/18/19
   </a>
   ）称其进行“可怕的镇压”（ National Review，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nationalreview.com/2019/03/nicaragua-daniel-ortega-government-crackdown-resistance/" href="https://www.nationalreview.com/2019/03/nicaragua-daniel-ortega-government-crackdown-resistance/" rel="noopener noreferrer" target="_blank">
    3/15/19
   </a>
   ） — —
   <strong class="markup--strong markup--p-strong">
    对于那些美国支持的独裁统治从来没被使用过这些词汇。
   </strong>
  </p>
  <p class="graf graf--p">
   只需粗略一瞥头条，就可以看到尼古拉斯·马杜罗、“委内瑞拉的儿童屠夫”（华盛顿考官，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonexaminer.com/opinion/on-venezuela-bernie-sanders-proves-socialisms-choking-immorality" href="https://www.washingtonexaminer.com/opinion/on-venezuela-bernie-sanders-proves-socialisms-choking-immorality" rel="noopener noreferrer" target="_blank">
    2/21/19
   </a>
   ）这类的描述：
  </p>
  <ul class="postList">
   <li class="graf graf--li graf--startsWithDoubleQuote">
    “委内瑞拉的独裁者赢得了他的头衔”（纽约时报，
    <a class="markup--anchor markup--li-anchor" data-href="https://www.nytimes.com/2019/02/27/opinion/jorge-ramos-venezuela.html" href="https://www.nytimes.com/2019/02/27/opinion/jorge-ramos-venezuela.html" rel="noopener noreferrer" target="_blank">
     2/27/19
    </a>
    ）
   </li>
   <li class="graf graf--li graf--startsWithDoubleQuote">
    “委内瑞拉的独裁者马杜罗度过了艰难的一周，但他的问题即将变得更糟”（迈阿密先驱报，
    <a class="markup--anchor markup--li-anchor" data-href="https://www.miamiherald.com/news/local/news-columns-blogs/andres-oppenheimer/article226777084.html" href="https://www.miamiherald.com/news/local/news-columns-blogs/andres-oppenheimer/article226777084.html" rel="noopener noreferrer" target="_blank">
     2009年2月25日
    </a>
    ）
   </li>
   <li class="graf graf--li graf--startsWithDoubleQuote">
    “为什么进步人士更倾向于不同意特朗普而不是反击独裁者[马杜罗]？”（华盛顿邮报，
    <a class="markup--anchor markup--li-anchor" data-href="https://news.google.com/articles/CAIiEMLjymlw7SKyPHXeVWRucrkqGAgEKg8IACoHCAowjtSUCjC30XQwzqe5AQ?hl=en-GB&amp;gl=GB&amp;ceid=GB%3Aen" href="https://news.google.com/articles/CAIiEMLjymlw7SKyPHXeVWRucrkqGAgEKg8IACoHCAowjtSUCjC30XQwzqe5AQ?hl=en-GB&amp;gl=GB&amp;ceid=GB%3Aen" rel="noopener noreferrer" target="_blank">
     3/2/19
    </a>
    ）
   </li>
   <li class="graf graf--li graf--startsWithDoubleQuote">
    “委内瑞拉的独裁者马杜罗必须滚蛋 — 即使军方必须干预”（福克斯新闻，
    <a class="markup--anchor markup--li-anchor" data-href="https://www.foxnews.com/opinion/newt-gingrich-venezuelas-dictator-maduro-must-go-even-if-the-military-has-to-intervene" href="https://www.foxnews.com/opinion/newt-gingrich-venezuelas-dictator-maduro-must-go-even-if-the-military-has-to-intervene" rel="noopener noreferrer" target="_blank">
     3/14/19
    </a>
    ）
   </li>
   <li class="graf graf--li graf--startsWithDoubleQuote">
    “马杜罗真的不喜欢被问到他是不是’独裁者’”（纽约邮报，
    <a class="markup--anchor markup--li-anchor" data-href="https://nypost.com/2019/02/26/maduro-really-didnt-like-being-asked-if-hes-a-dictator/" href="https://nypost.com/2019/02/26/maduro-really-didnt-like-being-asked-if-hes-a-dictator/" rel="noopener noreferrer" target="_blank">
     2/26/19
    </a>
    ）
   </li>
   <li class="graf graf--li">
    ……等等
   </li>
  </ul>
  <p class="graf graf--p">
   这三个当选的左翼国家元首经常被称为“独裁”，这真的是巧合吗？玻利维亚甚至没有被收入自由之家的“非自由”名单 — — 不像喀麦隆、埃及、阿尔及利亚和哈萨克斯坦。委内瑞拉和尼加拉瓜最近也加入了这一名单，尽管两国最近的选举
   <strong class="markup--strong markup--p-strong">
    都得到国际认可
   </strong>
   。
  </p>
  <p class="graf graf--p">
   虽然委内瑞拉和尼加拉瓜的政治制度有一些明显的缺点，但美国主导的美洲国家组织观察了2017年尼加拉瓜市政选举后甚至宣布“在尼加拉瓜绝大多数城市通过投票表达了民意。”（53％的投票率，执政的桑迪尼斯塔党
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Elections_in_Nicaragua#Latest_elections" href="https://en.wikipedia.org/wiki/Elections_in_Nicaragua#Latest_elections" rel="noopener noreferrer" target="_blank">
    赢得了152个社区中的135个
   </a>
   ，其中独立自由党只占其余12个）
  </p>
  <p class="graf graf--p">
   与此同时，委内瑞拉2018年的选举得到了
   <strong class="markup--strong markup--p-strong">
    150名国际观察员的支持
   </strong>
   ，其中包括西班牙的何塞·萨帕特罗和厄瓜多尔的拉斐尔·科雷亚等外国前国家元首，
   <strong class="markup--strong markup--p-strong">
    三个国际选举观察小组都赞同这一结果
   </strong>
   ，尽管美国主流媒体称他们为“假” (
   <strong class="markup--strong markup--p-strong">
    FAIR
   </strong>
   ,
   <a class="markup--anchor markup--p-anchor" data-href="https://fair.org/home/media-delegitimize-venezuelan-elections-amid-complete-unanimity-of-outlook/" href="https://fair.org/home/media-delegitimize-venezuelan-elections-amid-complete-unanimity-of-outlook/" rel="noopener noreferrer" target="_blank">
    5/23/18
   </a>
   ).
  </p>
  <p class="graf graf--p">
   即使是在“自由之家”，哈萨克斯坦也不能被做同样的描述，
   <a class="markup--anchor markup--p-anchor" data-href="https://freedomhouse.org/report/nations-transit/2018/kazakhstan" href="https://freedomhouse.org/report/nations-transit/2018/kazakhstan" rel="noopener noreferrer" target="_blank">
    即
   </a>
   “自独立以来在哈萨克斯坦举行的选举都没有被可信的国际观察员认为是’自由和公平的’。”Nursultan 被宣布赢得了98％的投票在2015年。
  </p>
  <p class="graf graf--p">
   敌对国家在美国企业媒体中的覆盖范围远远超过友好国家。在NYTimes.com数据库中搜索 Paul Biya 得出97个结果，相比之下，马杜罗为1,135，莫拉莱斯为713，奥尔特加为3,517，尽管 Biya 一直在掌权，但他的结果是前三个的总和（喀麦隆的人口为2400万，是委内瑞拉面积的四分之三，是玻利维亚的两倍多，人口密集度是尼加拉瓜人口的四倍）。
  </p>
  <p class="graf graf--p">
   在最近的一篇文章（FAIR，
   <a class="markup--anchor markup--p-anchor" data-href="https://fair.org/home/everyone-washington-supports-by-definition-is-a-moderate-centrist/" href="https://fair.org/home/everyone-washington-supports-by-definition-is-a-moderate-centrist/" rel="noopener noreferrer" target="_blank">
    3/23/19
   </a>
   ）中，我指出术语“温和”或“中间派”在媒体中使用时具有战术定义。它没有提到任何政治立场，而是用来传达合法性。因此，根据定义，
   <strong class="markup--strong markup--p-strong">
    媒体所赞同的任何人都是温和的。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “独裁者”标签也是一个强有力的线索，媒体用它来引导读者以某种方式看待某个国家或领导者；引导读者感到愤怒，
   <strong class="markup--strong markup--p-strong">
    而其他那些蔑视美国路线的专制统治者则被忽视甚至被赞扬
   </strong>
   。是否使用像“独裁者”这样的词来选择以有利于美国精英利益的方式构建一个国家印象，在单一的标签中足以传达合法性或缺乏合法性。
  </p>
  <p class="graf graf--p">
   政治宣传并没有那么容易识别，就如
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E7%8B%AC%E5%AE%B6-%E5%8C%97%E4%BA%AC%E7%9A%84%E6%94%BF%E6%B2%BB%E5%AE%A3%E4%BC%A0%E6%A8%A1%E5%BC%8F%E6%AD%A3%E5%9C%A8%E6%94%B9%E5%8F%98-%E6%88%96%E5%B7%B2%E8%B8%A9%E4%B8%AD%E4%B8%AD%E5%9B%BD%E5%BC%82%E8%AE%AE%E7%9A%84%E5%BC%B1%E7%82%B9-d1bf5ddfd412" href="https://www.iyouport.org/%e7%8b%ac%e5%ae%b6%ef%bc%9a%e5%8c%97%e4%ba%ac%e7%9a%84%e6%94%bf%e6%b2%bb%e5%ae%a3%e4%bc%a0%e6%a8%a1%e5%bc%8f%e5%8f%91%e7%94%9f%e6%94%b9%e5%8f%98/" rel="noopener noreferrer" target="_blank">
    IYP 在曝光中国官媒的内部讲座
   </a>
   中所引述的主讲人所言：“真正好的宣传在于润物细无声” — — 让你在不知不觉中被蒙骗。中国很快就需要如美国著名的 FAIR 这样的媒体评论平台，因为中国的宣传已经在演变了，变得更加隐含难以被察觉。除非有专业级的独立媒体鉴定第三方及时地揭露，才能避免公民的认知被侵害。本文仅做示范。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    D
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://fair.org/home/dictator-media-code-for-government-we-dont-like/" href="https://fair.org/home/dictator-media-code-for-government-we-dont-like/" rel="noopener noreferrer" target="_blank">
    ictator: Media Code for ‘Government We Don’t Like’
   </a>
   . The choice of whether to use a word like “dictator” frames a country in a way conducive to elite US interests, conveying legitimacy or the lack thereof in a single label.
  </p>
