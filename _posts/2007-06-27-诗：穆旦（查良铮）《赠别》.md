   <h1>
    诗：穆旦（查良铮）《赠别》
   </h1>
   <ul class="post-entry-meta">
    <li>
     By:
     <a href="http://www.ideobook.com/about-ideobook/" title="查看 爱德布克 的作者主页">
      爱德布克
     </a>
     . 2007-6-27.
     <a href="http://www.ideobook.com/372/post-views-count/" title="统计说明">
      6,339
     </a>
    </li>
   </ul>
  </header>
  <div class="post-entry-content">
   <p style="text-align: center;">
    1
   </p>
   <p style="text-align: center;">
    多少人的青春在这里迷醉，
    <br/>
    然后走上熙攘的路程，
    <br/>
    朦胧的是你的怠倦，云光和水，
    <br/>
    他们的自己失去了随着就遗忘，
   </p>
   <p style="text-align: center;">
    多少次了你的园门开启，
    <br/>
    你的美繁复，你的心变冷，
    <br/>
    尽管四季的歌喉唱得多好，
    <br/>
    当无翼而来的夜露凝重——
   </p>
   <p style="text-align: center;">
    等你老了，独自对着炉火，
    <br/>
    就会知道有一个灵魂也静静地，
    <br/>
    他曾经爱你的变化无尽，
    <br/>
    旅梦碎了，他爱你的愁绪纷纷。
   </p>
   <p style="text-align: center;">
    2
   </p>
   <p style="text-align: center;">
    每次相见你闪来的倒影
    <br/>
    千万端机缘和你的火凝成，
    <br/>
    已经为每一分每一秒的事体
    <br/>
    在我的心里碾碎无形，
   </p>
   <p style="text-align: center;">
    你的跳动的波纹，你的空灵
    <br/>
    的笑，我徒然渴望拥有，
    <br/>
    它们来了又逝去在神的智慧里，
    <br/>
    留下的不过是我曲折的感情，
   </p>
   <p style="text-align: center;">
    看你去了，在无望的追想中，
    <br/>
    这就是为什么我常常沉默：
    <br/>
    直到你再来，以新的火
    <br/>
    摒挡我所嫉妒的时间的黑影。
   </p>
   <p style="text-align: center;">
    1944年6月
   </p>
