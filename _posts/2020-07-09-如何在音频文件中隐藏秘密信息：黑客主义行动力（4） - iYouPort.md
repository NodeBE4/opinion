   许多人想知道他为什么要这样做，这种做法的安全性如何？
  </p>
  <p class="graf graf--p">
   答案是 Eliot 将这些CD上的数据隐藏起来了。
   <strong class="markup--strong markup--p-strong">
    他实际上是将音乐复制到这些CD，然后在其上嵌入加密信息，只有他自己才能恢复数据
   </strong>
   。
  </p>
  <p class="graf graf--p">
   因此，找到这些CD的任何人都只会听到音乐，并且将无法找到或检索隐藏其中的信息。
  </p>
  <p class="graf graf--p">
   这样，Elliot 收集的关于他的朋友和熟人的数据就不会受到执法人员或其他任何人的窥探。

   例如，如果您想向他人发送秘密消息，则可以将该消息放在图片、音频或视频文件中，然后通过电子邮件发送，或允许特定读者从本网站下载的文件。

  </p>

   <img alt="" class="aligncenter size-full wp-image-12535 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-8.jpg?resize=1100%2C532&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-8.jpg?w=1110&amp;ssl=1 1110w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-8.jpg?resize=300%2C145&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-8.jpg?resize=1024%2C495&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-8.jpg?resize=768%2C372&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-8.jpg?resize=1100%2C532&amp;ssl=1 1100w" data-recalc-dims="1" height="532" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-8.jpg?resize=1100%2C532&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12535" data-recalc-dims="1" height="532" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-8.jpg?resize=1100%2C532&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-8.jpg?w=1110&amp;ssl=1 1110w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-8.jpg?resize=300%2C145&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-8.jpg?resize=1024%2C495&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-8.jpg?resize=768%2C372&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/3-8.jpg?resize=1100%2C532&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   继续下载并安装到您的系统。您的视听软件可能会将其标识为病毒，但请继续并允许其安装。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第二步：Deep Sound 界面
   </strong>
  </h3>
  <p class="graf graf--p">
   完成安装后，将出现如下所示的屏幕。在屏幕的左侧显示了C盘的目录结构。通过单击目录结构上方 C: drive 旁边的向下箭头，可以浏览到系统上的其他驱动器或USB设备。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12536 jetpack-lazy-image" data-lazy-sizes="(max-width: 944px) 100vw, 944px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-6.jpg?resize=944%2C602&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-6.jpg?w=944&amp;ssl=1 944w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-6.jpg?resize=300%2C191&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-6.jpg?resize=768%2C490&amp;ssl=1 768w" data-recalc-dims="1" height="602" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-6.jpg?resize=944%2C602&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="944"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12536" data-recalc-dims="1" height="602" sizes="(max-width: 944px) 100vw, 944px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-6.jpg?resize=944%2C602&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-6.jpg?w=944&amp;ssl=1 944w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-6.jpg?resize=300%2C191&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4-6.jpg?resize=768%2C490&amp;ssl=1 768w" width="944"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   左侧的文件将是您的 “载体” 音频文件。换句话说，这些是将用来隐藏数据的文件。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第三步：设置
   </strong>
  </h3>
  <p class="graf graf--p">
   单击顶部栏上的 “设置” 图标，它将弹出如下窗口。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-12537 jetpack-lazy-image" data-lazy-sizes="(max-width: 476px) 100vw, 476px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-5.jpg?resize=476%2C449&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-5.jpg?w=476&amp;ssl=1 476w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-5.jpg?resize=300%2C283&amp;ssl=1 300w" data-recalc-dims="1" height="449" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-5.jpg?resize=476%2C449&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="476"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12537" data-recalc-dims="1" height="449" sizes="(max-width: 476px) 100vw, 476px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-5.jpg?resize=476%2C449&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-5.jpg?w=476&amp;ssl=1 476w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5-5.jpg?resize=300%2C283&amp;ssl=1 300w" width="476"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   应该设置默认语言、输出目录和输出格式。
  </p>
  <p class="graf graf--p">
   请注意，该软件包仅适用于.flac 或 .wav 文件。这意味着您不能使用.mp3 文件。大概是因为.mp3被压缩而其他两种格式没有被压缩。
  </p>
  <p class="graf graf--p">
   另外，请确保单击 “加密文件” 复选框。它使用的是最强加密算法，这可以确保您的隐藏文件已使用256位AES加密的保护。
  </p>
  <p class="graf graf--p">
   最后，添加密码，当您或这些文件的目标接收方想要恢复数据时，将使用该密码解锁。该密码越长越好。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第四步：选择音频文件
   </strong>
  </h3>
  <p class="graf graf--p">
   接下来需要选择音频文件以隐藏数据。null-byte 的大多数音乐都是.mp3 格式（可能与您一样），因此这些音乐将无法使用。
  </p>
  <p class="graf graf--p">
   null-byte 有一些 .flac 格式的 Nora Jones，将在这里使用。
  </p>
  <p class="graf graf--p">
   （通常，发烧友更喜欢 .flac，因为它的质量更好，但由于未压缩，因此它的体积要大得多。这就是为什么大多数移动音频设备使用.mp3的原因，它们的文件比较小。）
  </p>
  <p class="graf graf--p">
   双击目标音频文件，它准备好用于隐藏消息了。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12538 jetpack-lazy-image" data-lazy-sizes="(max-width: 933px) 100vw, 933px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-4.jpg?resize=933%2C623&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-4.jpg?w=933&amp;ssl=1 933w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-4.jpg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-4.jpg?resize=768%2C513&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-4.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="623" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-4.jpg?resize=933%2C623&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="933"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12538" data-recalc-dims="1" height="623" sizes="(max-width: 933px) 100vw, 933px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-4.jpg?resize=933%2C623&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-4.jpg?w=933&amp;ssl=1 933w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-4.jpg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-4.jpg?resize=768%2C513&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-4.jpg?resize=272%2C182&amp;ssl=1 272w" width="933"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   接下来，单击顶部栏上的 “添加文件” 图标。这将提示您将要隐藏的文件添加到音频文件中。
  </p>
  <p class="graf graf--p">
   在这里，null-byte 有一个名为 Shayla.doc 的文件，想把它隐藏在 Nora Jones 音频文件中。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12539 jetpack-lazy-image" data-lazy-sizes="(max-width: 797px) 100vw, 797px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7-4.jpg?resize=797%2C527&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7-4.jpg?w=797&amp;ssl=1 797w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7-4.jpg?resize=300%2C198&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7-4.jpg?resize=768%2C508&amp;ssl=1 768w" data-recalc-dims="1" height="527" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/7-4.jpg?resize=797%2C527&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="797"/>

   </noscript>
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第五步：编码
   </strong>
  </h3>
  <p class="graf graf--p">
   单击 Shayla.doc 时，它将其添加到右侧窗口。现在，需要单击顶部图标栏上的 “Encode” 图标。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-12541 jetpack-lazy-image" data-lazy-sizes="(max-width: 993px) 100vw, 993px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/8-3.jpg?resize=993%2C629&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/8-3.jpg?w=993&amp;ssl=1 993w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/8-3.jpg?resize=300%2C190&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/8-3.jpg?resize=768%2C486&amp;ssl=1 768w" data-recalc-dims="1" height="629" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/8-3.jpg?resize=993%2C629&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="993"/>

   </noscript>

  <p class="graf graf--p">
   现在，这个 Shayla.doc 文件已加密并被目标音频文件隐藏！
  </p>
  <p class="graf graf--p">
   不仅此文件的外观和声音像普通的音频文件一样，而且，如果有人希望拿到隐藏在其中的信息，他们将需要使用仅由您自己知道的密码对其进行解密。
  </p>
  <p class="graf graf--p">
   对于 Elliot 而言，他是获取了这些音频文件并将其刻录到CD了，但这不是必需的。据推测，
   <strong class="markup--strong markup--p-strong">
    他应该是担心计算机被警方查抄，于是他正在采取进一步的措施来保护这些文件
   </strong>
   。任何调查都不太可能一张张听他的音乐CD。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第六步：解码
   </strong>
  </h3>
  <p class="graf graf--p">
   最终，Elliot 或信息所针对的人将需要对隐藏的信息进行解码。只需单击音频文件，然后单击 “提取秘密文件” 图标。
  </p>
  <p class="graf graf--p">
   它将提示您输入密码。输入您在步骤3中创建的密码，点击 “确定”，隐藏的文件将出现在右侧窗口中。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12542 jetpack-lazy-image" data-lazy-sizes="(max-width: 816px) 100vw, 816px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/9-3.jpg?resize=816%2C208&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/9-3.jpg?w=816&amp;ssl=1 816w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/9-3.jpg?resize=300%2C76&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/9-3.jpg?resize=768%2C196&amp;ssl=1 768w" data-recalc-dims="1" height="208" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/9-3.jpg?resize=816%2C208&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="816"/>

  <p class="graf graf--p">
   现在，您可以使用创建消息的相应软件来读取隐藏的秘密消息。
  </p>
  <p class="graf graf--p">
   在这种情况下，它将是 Microsoft Word 或能够打开 .doc 文件的任何其他软件。
  </p>
