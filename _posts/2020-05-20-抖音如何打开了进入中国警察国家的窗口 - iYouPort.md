  <p class="graf graf--p">
   学习结束后的每个晚上，Alip Erkin 都会坐在澳大利亚悉尼的家中，并在他的 Android 手机上打开视频共享应用 TikTok。
  </p>
  <p class="graf graf--p">
   他正在寻找特别的东西：来自中国西北地区新疆的视频，这是他在2012年最后一次离开的地方。
  </p>
  <p class="graf graf--p">
   自那时以来，新疆已迅速转变成一个
   <strong class="markup--strong markup--p-strong">
    广阔的警察国家
   </strong>
   ，在那里，维族穆斯林被系统性地针对和监视，有超过一百万的人被关押在集中营和拘留中心。
  </p>
  <p class="graf graf--p">
   他滚动查看着数千英里以外的房屋的视频，怀旧和苦乐参半。但是对于现年41岁的 Erkin 说，这是对中国政府不希望他看到的地方的一瞥。
  </p>
  <p class="graf graf--p">
   打开 TikTok 应用程序后，Erkin 发现其中大部分内容对他来说都没有意义：超现实的喜剧片段、欢快的音乐演唱等等。
  </p>
  <p class="graf graf--p">
   他说： “就现实的政治情况而言，这其中的大多数视频都毫无价值。”
  </p>
  <p class="graf graf--p">
   但是，每隔一段时间，Erkin 就会能看到一段录像，
   <strong class="markup--strong markup--p-strong">
    其中揭示了中国在新疆的大规模监视镇压和洗脑政策的现实状况
   </strong>
   ：一段宣传集会的录像，维吾尔族人唱着歌赞扬中国共产党; 来自维吾尔族孤儿院的录像，其中有被拘留的父母的孩子；维吾尔族人群用普通话而不是其本土维吾尔语高呼；一座被拆除的清真寺 ……
  </p>
  <p class="graf graf--p">
   他经常会发现曾经熙熙攘攘的喀什（Kashgar）家乡如今已经变得如此空旷，杳无人烟的街道景象。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="618" src="//player.vimeo.com/video/378471127?title=0&amp;byline=0&amp;portrait=0&amp;color=8dc7dc" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   最有价值的是新疆人
   <strong class="markup--strong markup--p-strong">
    在压迫性监视制度下生活的证据
   </strong>
   ，例如，一长串维吾尔族人等待通过安检站的画面、或全副武装的新疆警察接受训练的镜头。
  </p>
  <p class="graf graf--p">
   Erkin 在浏览 TikTok 时经常会发现一些较小的陌生细节：没有戴头巾的维吾尔族穆斯林妇女；没有胡须的男人。
  </p>
  <p class="graf graf--p">
   他注意到自己的维吾尔族母语逐渐从路牌上消失了。他说：
   <strong class="markup--strong markup--p-strong">
    “维吾尔文化的这些部分已经消失了，当我看到这一点时，我会感到沮丧和愤怒。”
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “但是，即使是为了令我痛苦，我仍然试图继续浏览，只是出于寻求有用信息的目的。”
  </p>
  <p class="graf graf--p">
   他发现有趣的视频后便立即对其进行下载和存档，然后对其进行检查。然后将其重新发布到他的 Twitter 页面
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/UyghurBulletin/media" href="https://twitter.com/UyghurBulletin/media" rel="noreferrer noopener" target="_blank">
    Uyghur Bulletin
   </a>
   上。
  </p>
  <p class="graf graf--p">
   新疆处于严重的信息锁定状态。中国政府声称正在打击该地区的恐怖主义和极端主义。
  </p>
  <p class="graf graf--p">
   由于事实上被禁止与外国人联系，维吾尔族人不敢给居住在中国境外的亲人发短信或打电话。其他发布信息的方式也受到严格控制：外国记者在新疆受到严格监控，他们的相机和手机经常在每天都要经过的检查站之前清空。
  </p>
  <p class="graf graf--p">
   对于像 Erkin 这样的激进主义者来说，对信息的极权控制造成了一个问题：明显的醒目的视觉效果不足，使全世界的人们都难以掌握新疆人道主义危机的日常现实。
  </p>
  <p class="graf graf--p">
   国际社会缺乏协调一致的反应也许可以证明这一点。
  </p>

  <p class="graf graf--p">
   进入 TikTok，这是由科技巨头 Bytedance 拥有的、广受欢迎的、中国制造的视频共享应用程序。
   <strong class="markup--strong markup--p-strong">
    目前，全世界约有十亿人在使用它
   </strong>
   。
  </p>
  <p class="graf graf--p">
   维吾尔族和汉族用户拍摄的成千上万来自新疆的视频每天都上传到中文 TikTok 和其他视频共享应用程序中。
  </p>
  <p class="graf graf--p">
   庞大的视频量使当局很难对所有内容进行审查。
  </p>
  <p class="graf graf--p">
   Erkin 是维吾尔族人全球网络的一部分，维吾尔族人捕捉到这些内容，并不断在 TikTok 上全面搜索以寻找有关新疆局势的视觉线索。
  </p>
  <p class="graf graf--p">
   Aliye Yasin（为了保护自己的家人而改名）于8月开始使用该应用程序。她在晚餐时间打开 TikTok。她说：“这是中国的午夜，大多数人都在睡觉。” 她认为，包括监视新疆社交媒体内容的当局审查员也会睡觉。
  </p>
  <p class="graf graf--p">
   Yasin
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/UygursOnTikTok/status/1168643330999226371" href="https://twitter.com/UygursOnTikTok/status/1168643330999226371" rel="noopener noreferrer" target="_blank">
    发现的
   </a>
   最引人注目的视频之一是新疆警察局的官方帐户发布的。武装警察似乎主要是维吾尔族人，画面展示他们对共产党表示忠诚。标题是：“向祖国宣誓，以保护人民的和平。”
  </p>
  <p class="graf graf--p">
   另一个官方视频显示，大部分维吾尔族警察在中国国旗前跳舞。在他们头顶上方的标志表明，这是集体心理支持活动的一部分 —— 一个有说服力的细节，因为
   <strong class="markup--strong markup--p-strong">
    这些维吾尔族警察通常被要求逮捕和拘留自己的同胞
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="618" src="//player.vimeo.com/video/378487327?title=0&amp;byline=0&amp;portrait=0&amp;color=8dc7dc" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   Erkin、Yasin 和其他维吾尔族在线活动家正在使用 TikTok 内容来挑战中国官方有关其 “降低了” 在新疆的监视和拘留活动的说法。
  </p>
  <p class="graf graf--p">
   去年7月，一位
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2019/jul/31/most-people-detained-in-xinjiang-camps-have-been-released-china-claims" href="https://www.theguardian.com/world/2019/jul/31/most-people-detained-in-xinjiang-camps-have-been-released-china-claims" rel="noopener noreferrer" target="_blank">

  </p>
  <p class="graf graf--p">
   美国维吾尔族研究生 Mehmet Jan 帮助开展了该项目。他说：“我将视频分为四类。 ”根据它们是否显示政治感言、监视、清真寺毁灭、或文化灭绝来分类。
  </p>
  <p class="graf graf--p">
   学生们打算收集证据，证明新疆正在逐渐被重新编程为以北京形象为基础的地区。政府声称正在努力消除该地区的恐怖主义。
  </p>
  <p class="graf graf--p">
   学者 Rachel Harris 在四月份为《卫报》撰写的一篇
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/commentisfree/2019/apr/07/bulldozing-mosques-china-war-uighur-culture-xinjiang" href="https://www.theguardian.com/commentisfree/2019/apr/07/bulldozing-mosques-china-war-uighur-culture-xinjiang" rel="noopener noreferrer" target="_blank">
    文章中写道
   </a>
   ：
   <strong class="markup--strong markup--p-strong">
    “这不是针对暴力极端主义的回应，而是一场旨在消灭整个民族文化的协同性运动。”
   </strong>
  </p>
  <p class="graf graf--p">
   维吾尔族妇女和汉族男人之间的婚礼是最近出现在 TikTok 的一系列视频，一直是散居在外的维吾尔族人的苦恼之源，他们把这些录像视为强迫种族同化的证据。
  </p>
  <p class="graf graf--p">
   根据自由亚洲电台维吾尔语服务的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.rfa.org/english/news/special/uyghur-oppression/ChenPolicy2.html" href="https://www.rfa.org/english/news/special/uyghur-oppression/ChenPolicy2.html" rel="noopener noreferrer" target="_blank">
    一份报告
   </a>
   ，2017年，新疆政府出台了 “维吾尔族-汉族通婚和家庭激励策略”，向已婚的维吾尔族和汉族夫妇提供10,000元人民币（合1,400美元）。
  </p>
  <p class="graf graf--p">
   通婚在中国很少见：根据2010年的人口普查，
   <a class="markup--anchor markup--p-anchor" data-href="https://link.springer.com/article/10.1186/s40711-017-0059-0" href="https://link.springer.com/article/10.1186/s40711-017-0059-0" rel="noopener noreferrer" target="_blank">
    仅0.2％
   </a>
   的维吾尔族人与汉族人结婚。澳大利亚学者 James Leibold 也发现了通婚计划的视频证据。
  </p>
  <p class="graf graf--p">
   4月，他发推文说，
   <strong class="markup--strong markup--p-strong">
    中国互联网 “充斥着宣传汉族维吾尔族通婚的短片。这种殖民策略已有很长的历史 —— 使用族裔间的婚姻作为民族融合的工具。”
   </strong>
  </p>
  <p class="graf graf--p">
   北京对维吾尔族的待遇是一项影响深远的运动的一部分，该运动试图外在地摆脱穆斯林少数群体的某些身份认同，并传达出更接近于占主导地位的 “汉族文化的新疆” 的理想形象。
  </p>
  <p class="graf graf--p">
   通过被夷为平地的清真寺、破坏传统的维吾尔建筑、阻止维吾尔语的使用、以及激励通婚，人们深感关切的是，维吾尔族的许多生命已经永远丧失了。
  </p>
  <p class="graf graf--p">
   在国际社会对中国监禁超过一百万维吾尔人的强烈抗议之后，中国似乎已经加大力度，
   <strong class="markup--strong markup--p-strong">
    通过防火墙、加强审查和误导性官方言论来掩盖其在新疆的活动。
   </strong>
  </p>
  <p class="graf graf--p">
   同时，为了进一步遏制外界的批评，北京拥有的媒体充斥着描述繁华、生机勃勃的新疆的图片和故事。
  </p>
  <p class="graf graf--p">
   政府宣传人员似乎在加倍努力以控制该地区的形象 —— 经常发布自己的 TikTok 视频。
   <strong class="markup--strong markup--p-strong">
    这就是信息战
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     《
    </strong>
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%9c%8b%e6%b8%85%e4%bf%a1%e6%81%af%e6%88%98%ef%bc%9a2019%e5%85%a8%e7%90%83%e6%9c%89%e7%bb%84%e7%bb%87%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%93%8d%e7%ba%b5%e6%b8%85%e5%8d%95/" href="https://www.iyouport.org/%e7%9c%8b%e6%b8%85%e4%bf%a1%e6%81%af%e6%88%98%ef%bc%9a2019%e5%85%a8%e7%90%83%e6%9c%89%e7%bb%84%e7%bb%87%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%93%8d%e7%ba%b5%e6%b8%85%e5%8d%95/" rel="noopener noreferrer" target="_blank">
     <strong class="markup--strong markup--li-strong">
      看清信息战：2019全球有组织社交媒体操纵清单
     </strong>
    </a>
    <strong class="markup--strong markup--li-strong">
     》
    </strong>
   </li>
  </ul>
  <p class="graf graf--p">
   Twitter 用户 Eva Zheng 似乎在为中国政府工作，并定期发布 TikTok 视频，
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/evazhengll/status/1158052683598745601" href="https://twitter.com/evazhengll/status/1158052683598745601" rel="noopener noreferrer" target="_blank">
    他在8月写道
   </a>
   ：“我在这里，迷失在新疆热闹的夜市中。”
  </p>
  <p class="graf graf--p">
   Rose-Hulman 的教授 Timothy Grose 对这样的内容感到沮丧。他说：
   <strong class="markup--strong markup--p-strong">
    “拥有竞争性的视频并没有帮助。那些不信服的人继续遭到反事实的轰炸。”
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%be%9f%e8%b0%a3%e5%be%80%e5%be%80%e6%97%a0%e6%95%88%e9%99%a4%e9%9d%9e%e8%87%aa%e6%88%91%e6%89%b9%e9%a9%b3-%e8%af%86%e5%88%ab%e6%88%98%e7%95%a5%e6%80%a7%e6%ac%ba%e9%aa%97%e7%9a%84%e4%b8%bb/" href="https://www.iyouport.org/%e8%be%9f%e8%b0%a3%e5%be%80%e5%be%80%e6%97%a0%e6%95%88%e9%99%a4%e9%9d%9e%e8%87%aa%e6%88%91%e6%89%b9%e9%a9%b3-%e8%af%86%e5%88%ab%e6%88%98%e7%95%a5%e6%80%a7%e6%ac%ba%e9%aa%97%e7%9a%84%e4%b8%bb/" rel="noopener noreferrer" target="_blank">
     辟谣往往无效除非自我批驳 — 识别战略性欺骗的主要途径（6 — 下）
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="618" src="//player.vimeo.com/video/378497864?title=0&amp;byline=0&amp;portrait=0&amp;color=8dc7dc" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   为了抵制国家宣传的猛烈袭击，Alip Erkin 每个醒着的时间都在挖掘有关新疆的在线信息。他说：“这对我造成了巨大的心理伤害，我不得不有意识地减少屏幕时间。”
  </p>
  <p class="graf graf--p">
   维吾尔族澳大利亚激进主义者 Arslan Hidayat 经常为抗击数字化运动而兴奋，他经常通过 WhatsApp 将其他维吾尔族的 TikTok 视频发送给同胞。
  </p>
  <p class="graf graf--p">
   他说：“我已经与在心理层面受到困扰的人进行了交谈。” 美国学生 Mehmet Jan 承认：“这令人精神疲惫。我觉得我花太多时间上网。我不能专心其他任何事了。”
  </p>
  <p class="graf graf--p">
   TikTok 矿工还担心新疆维吾尔族人首先发布了这些视频。Jan 说：“这将被视为犯罪，就像揭露国家机密一样。”
  </p>
  <p class="graf graf--p">
   去年 8月，中国 TikTok 开始发生不寻常的事。
   <strong class="markup--strong markup--p-strong">
    新疆境内的数十名维吾尔人一个接一个地发布静音视频站在亲人的照片前。许多人在哭。
   </strong>
  </p>
  <p class="graf graf--p">
   Hidayat 说：“我认为背后有一个策划人 —— 一两个创意者。”
   <strong class="markup--strong markup--p-strong">
    那是无言的数字快闪族。人们陷入了困境，没有聚在一起，也不必解释这个概念”。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “但是每一名维吾尔族人都理解这些视频。”
   </strong>
  </p>
  <p class="graf graf--p">
   在一个视频中，一个女孩坐在四个人合影的照片前，举起四根手指。她慢慢地握紧拳头。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="618" src="//player.vimeo.com/video/378499330?title=0&amp;byline=0&amp;portrait=0&amp;color=8dc7dc" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   在中国境外，国际维吾尔族数字侨民迅速动员起来。
  </p>
  <p class="graf graf--p">
   他们将这些视频视为对大规模拘留中心的无声起义，并在 Facebook 和 Twitter 上重新发布了这些视频。
  </p>
  <p class="graf graf--p">
   Hidayat 将视频收集到一个 thread 中，创建了#WeHearU 标签。新闻媒体很快开始报道这个故事。全世界的维吾尔族和中国观看者被影片的勇气所震撼。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “就是那样。这些视频并没有真正针对任何人。这不是抗议，因为他们无法抗议中国政府。它实际上只是一条消息。”
  </p>
  <p class="graf graf--p">
   在无声视频浪潮过后的几天里，Yasin 在 TikTok 上搜索一些张贴了这些视频的维吾尔族用户。
  </p>
  <p class="graf graf--p">
   在五彩缤纷的视频应用和嘈杂的杂音中，她什么都没有找到。某些人的帐户已完全消失了。该应用程序告诉她：“您的搜索结果为空，找不到任何相关内容。” ⚪️
  </p>
