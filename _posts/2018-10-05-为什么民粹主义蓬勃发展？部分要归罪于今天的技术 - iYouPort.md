   为什么民粹主义蓬勃发展？部分要归罪于今天的技术

 <div class="entry-content">
  <p class="graf graf--p graf-after--p" id="d01a">
   世界各地的右翼民粹主义者已经兴奋好几个月了。反移民瑞典民主党在9月的大选中赢得了 17.6％ 的选票，使他们成为 Riksdag 的第三大党；在巴西，极右翼的煽动者 Jair Bolsonaro 已经成为总统；在意大利，执政党并不会损害 Lega Nord 或其联盟合作伙伴 Five Star。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-3960 jetpack-lazy-image" data-lazy-sizes="(max-width: 724px) 100vw, 724px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-355.png?resize=724%2C490&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-355.png?w=724&amp;ssl=1 724w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-355.png?resize=300%2C203&amp;ssl=1 300w" data-recalc-dims="1" height="490" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-355.png?resize=724%2C490&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="724"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-3960" data-recalc-dims="1" height="490" sizes="(max-width: 724px) 100vw, 724px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-355.png?resize=724%2C490&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-355.png?w=724&amp;ssl=1 724w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-355.png?resize=300%2C203&amp;ssl=1 300w" width="724"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--p" id="57ae">
   这是怎么回事？选民希望改变的原因有很多。但左翼和右翼都没有承认民粹主义是一种政治风格。
   <strong class="markup--strong markup--p-strong">
    民粹主义正在进行疯狂的交易，互联网新媒介已经让我们的政治文化发生了变化
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="9578">
   民粹主义这个词最近成为了一个纯粹贬义的术语，通常被自由派用来贬低他们不喜欢的一切东西。但民粹主义本身有两个主要特征。首先，它为复杂的问题提供了直接和明显的答案，这些问题通常会引起其他一些团体的注意。其次，它声称代表受压迫的“人民”。这种风格和叙事既可以为左翼使用也可以右翼使用。而社交媒体为两种攻击线提供了完美的平台。
  </p>
  <p class="graf graf--p graf-after--p" id="6fe0">
   一旦你理解了社交媒体平台是广告公司的资金来源，民粹主义这件事就很容易理解了。正如任何广告人都会告诉你的那样，情感化和大众化就能让你赢出。民粹主义信息高度适合这点。民粹主义者更加舒适地适应了当今的技术。从购物到约会，音乐再到新闻，一切都是个性化的 — 快捷、方便、随心所欲。
  </p>
  <p class="graf graf--p graf-after--p" id="0f1d">
   民粹主义者说，好消息是，我们现在可以直接与那些诚实、体面、勤奋的人一起，围绕自利的目的建立政党和媒体。这就是为什么许多民粹主义者 — 无论是 Twitter 上瘾的特朗普，还是瑞典民主党或意大利五星运动 — 都是早期采用社交媒体的，并且对这种宣传模式完全放心。
  </p>
  <p class="graf graf--p graf-after--p" id="8fa9">
   <strong class="markup--strong markup--p-strong">
    或许更准确地说，我们所有人 — 选民和领导人 — 都在我们的规范和期望中成为了民粹主义者。为了理解这一点，我们必须放弃一些固有的推卸责任的想法。其中一个典型就是认为民粹主义浪潮是由其他一些被假新闻欺骗的白痴引起的
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="f190">
   因为应该知道，
   <strong class="markup--strong markup--p-strong">
    我们每个人都围绕在那些能让自己感觉舒适的新闻的周围，而无视那些能挑战我们既有认知的真相
   </strong>
   。下意识地寻找舒适，这与假新闻不同，但效果是完全相似的：当一个共同的真相被个体所认为的现实所取代时，没有任何东西可以支持政治讨论和辩论。剩下的就只有两个群体的互相尖叫，民粹主义者可以大声尖叫。
  </p>
  <p class="graf graf--p graf-after--p" id="1018">
   你经常能听到人们强调社交媒体的回音壁，但要知道，你我和所有人一样都在被回音壁所包围。
   <strong class="markup--strong markup--p-strong">
    我们不是在倾听、交流、批判性思考，而是在自我辩护、诋毁他人
   </strong>
   。你最后一次通过在线讨论后改变了主意是什么时候？可能永远不会。谁有时间在网上进行长期、细致、互相尊重的讨论呢，以便了解事物的另一面？
  </p>
  <p class="graf graf--p graf-after--p" id="7931">
   <strong class="markup--strong markup--p-strong">
    我会打赌你从对手那里听到的越多，你就越不会同意他们
   </strong>
   。在一个印刷社会中，尽管存在各种缺陷，但至少倾向于对事实和想法进行排序和连贯。而社交媒体平台建立在不同的逻辑上：我们被淹没在不和谐的想法和故事、事实和图表的激流中。要求对所有信息合理处理已经太过分了。我们越来越无法在不检查手机的情况下集中精力超过几分钟。
  </p>
  <p class="graf graf--p graf-after--p" id="8b6a">
   <strong class="markup--strong markup--p-strong">
    经常的分心令我们难以处理复杂的想法、知识和信息，被分散注意力的国家更喜欢灰色地带的情感确定性
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="24b8">
   这些都是人类的弱点，但社交媒体以一种基本上无法预料的方式将它们变成了现代信息消费的结构特征，并将其用于赚钱。
   <strong class="markup--strong markup--p-strong">
    对于可以引导愤怒并为混乱带来秩序的党派领导人来说，这些都是完美的条件
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="9cae">
   我们都应该受到责备。我们一直在点击并分享那些情绪刺激的故事; 我们尖叫、讽刺和侮辱任何自己不喜欢的东西。我们需要保证尊重我们的对手，找到他们最好的论据，而不是最坏的。
  </p>
  <p class="graf graf--p graf-after--p" id="1ec7">
   如果我们能够削弱社交媒体所依赖的监视资本主义商业模式，那么他们就无法控制和操纵我们。技术公司必须通过采用并非完全基于注意力经济的设计和服务来帮助实现这一目标。因为如果这种民粹主义浪潮进入专制主义或更糟糕的情况，没有人会从中受益。
  </p>
