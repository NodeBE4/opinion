   </noscript>
  </p>
  <p class="graf graf--p">
   为了进一步巩固其作为惩罚性仇外心理之丑陋体现的声誉，ICE 在密歇根州建立了一所假冒的大学，以*诱使*移民试图做法律原本允许他们做的事：在赚取收入的同时留在该国求学。IYP 去年编译了这一报道，现在这件事又有了更多新的揭露。
  </p>
  <p class="graf graf--p">
   这是一所虚假的大学，它不仅拥有真正的校园，并且还获得了国家认证 — — 该虚假大学的目的是将受骗者支付的学费转化为逮捕和拘留所需的一切证据。
  </p>
  <p class="graf graf--p">
   在 ICE 开始钓鱼那些在很大程度上只是想做一些合法的事的人之前，ICE 只收取了60,000美元的申请费。受害者不会通过完成学位而获得在美国生活和工作的机会，正相反，有超过160名被骗的学生被 ICE 拘留。到目前为止，只有八个人实际上面临指控。意味着绝大多数人都是被冤枉的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在这个时代，想要让人们相信你（准确说是成功受骗），你需要的不仅仅是国家认可。您还需要社交媒体
   </strong>
   。正如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/technology/2019/apr/11/us-immigration-police-broke-facebook-rules-with-fake-profiles-for-college-sting" href="https://www.theguardian.com/technology/2019/apr/11/us-immigration-police-broke-facebook-rules-with-fake-profiles-for-college-sting" rel="noopener noreferrer" target="_blank">
    “卫报”报道的那样
   </a>
   ，在这起诱骗案件中，ICE 还创建了 Facebook 群组，同时亲自参与了这个社交工程钓鱼的刺激行动。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    超过600名学生，大多数是印度公民，都参与了该计划，卫报发现，该计划包括由美国第二大联邦调查机构 ICE 的国土安全调查（HSI）部门创建的
   </em>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/technology/facebook" href="https://www.theguardian.com/technology/facebook" rel="noopener noreferrer" target="_blank">
    <em class="markup--em markup--p-em">
     虚假 Facebook
    </em>
   </a>
   <em class="markup--em markup--p-em">
    档案。
   </em>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “与其他所有人一样，负责钓鱼的执法机关必须在 Facebook 上使用他们的真实姓名，我们在面向公众的指南页面上明确了这一政策，一位 Facebook 代表告诉“卫报”，该代表称 “不允许操作虚假账户，我们将对任何违规账户采取行动。
   </em>
   ”
  </p>
  <p class="graf graf--p">
   由于目前对被捕的八人的起诉已经推进，此案的内幕细节也逐渐曝光。
  </p>
  <p class="graf graf--p">
   ICE 的一个假帐户名为 Ali“AJ”Milani，其自称为底特律居民，其中包括一个充满南亚居民的“朋友”名单。另一个假账户是 Carey Ferrante，这是一个与 Milani 作为“朋友”关系的虚假账户，并与潜在的目标学生交换了信息。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    至少有五个 Facebook 账户被确定为 ICE 钓鱼者的虚假账户
   </strong>
   。在卫报接近 ICE 征求意见后，所有这些账户都停止使用了。
  </p>
  <p class="graf graf--p">
   虚假的 Facebook 帐户为假大学的背景故事提供支持。这场诈骗活动能取得成功的关键还在于国家机构和 ICE 共谋制造虚假认证的背后手段。这让受害者看起来会是一个在外观上非常逼真的大学。然而，美国政府坚持认为，“每一个申请学位以延长在美国停留时间的人都知道他们正在接受什么”。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    在起诉书中，政府说：“每个学生都知道大学的课程没有得到美国国土安全部的批准，这是非法的，在与他人讨论该计划时应该使用自由裁量权
   </em>
   。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种说法根本站不住脚 — — 不只有在该机构的行为直接与这一主张相抵触时。底特律自由新闻社获得的文件和声明表明，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.freep.com/story/news/2019/02/11/emails-reveal-how-university-lured-students-fake-farmington-university/2744103002/" href="https://www.freep.com/story/news/2019/02/11/emails-reveal-how-university-lured-students-fake-farmington-university/2744103002/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     政府在这场诈骗行为期间直接向学生撒谎
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   这对ICE或整个政府来说都不是好事。假的 Facebook 个人资料可能是整个谎言运动中最不起眼的那一个，但它足以表明，政府对私人和机构都没有任何尊重。
   <strong class="markup--strong markup--p-strong">
    事实显示，无论是条款和条件还是庄严的权利，如果它们阻碍了政府想做的事，它们只会被轻易忽略
   </strong>
   。
  </p>
  <p class="graf graf--p">
   对于非美国的其他国家公民来说，如果您希望获得在美国的留学机会，请务必谨慎选择；尤其是不要使用 facebook 等社交媒体中的群组获得相关推荐。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.techdirt.com/articles/20190412/17061441996/ices-fake-university-sting-operation-also-used-bunch-fake-facebook-profiles.shtml" href="https://www.techdirt.com/articles/20190412/17061441996/ices-fake-university-sting-operation-also-used-bunch-fake-facebook-profiles.shtml" rel="noopener noreferrer" target="_blank">
    ICE’s Fake University Sting Operation Also Used A Bunch Of Fake Facebook Profiles
   </a>
   . The fake Facebook profiles may be the smallest of the lies told, but it shows the government has zero respect for the private people and institutions that happen to find themselves between it and its objectives. Whether its terms and conditions or enshrined rights, if they’re in the way, they’ll just need to be ignored.
  </p>
