   与前文不同，本文中即将讲述的方法*更为*主动，间谍采取此类方法是希望能达到立竿见影的效果。
  </p>
  <p class="graf graf--p">
   即便没能立竿见影，也可以给对方制造强大的心理压力，以影响其之后的进攻。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    恐吓也许是一种假象，但是被恐吓者往往会宁信其有，这是在借助信息不对称和心理映射，激发对方产生畏惧心理
   </strong>
   。
  </p>
  <p class="graf graf--p">
   任何人都可以使用恐吓，使用者必需知道自己在对方面前的优势所在，强调这一优势以令对方害怕。我们在行动主义系列中也介绍过活动家可以使用的威胁方法，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/1%E3%80%81%E5%A8%81%E8%83%81%E6%84%9F%E5%8F%AF%E8%83%BD%E6%AF%94%E4%BA%8B%E7%89%A9%E6%9C%AC%E8%BA%AB%E6%9B%B4%E6%9C%89%E6%95%88%EF%BC%9A%E6%88%98%E7%95%A5%E6%80%A7%E7%9B%B4%E6%8E%A5%E8%A1%8C%E5%8A%A8/" href="https://www.iyouport.org/1%E3%80%81%E5%A8%81%E8%83%81%E6%84%9F%E5%8F%AF%E8%83%BD%E6%AF%94%E4%BA%8B%E7%89%A9%E6%9C%AC%E8%BA%AB%E6%9B%B4%E6%9C%89%E6%95%88%EF%BC%9A%E6%88%98%E7%95%A5%E6%80%A7%E7%9B%B4%E6%8E%A5%E8%A1%8C%E5%8A%A8/" rel="noopener noreferrer" target="_blank">

    那篇文章中
   </a>
   所强调的，活动家对威胁战术的使用必需能有切实实施威胁的能力。
  </p>
  <p class="graf graf--p">
   而在本文即将分析的威胁类型中，间谍并不需要真的实施威胁，只需要能让对方相信间谍可以实施。
  </p>
  <p class="graf graf--p">
   再次重申，本文中分析的内容最适合审讯场景，⚠️如果您是活动家、记者、人权维护者、人权律师、边缘社区等任何挑战主流政治的人，您有必要了解这些；一旦被捕，您可以做到心里有数，并做出适当的合理的反应，尽可能保全自己和队友。
  </p>
  <p class="graf graf--p">
   以下部分将更多以审讯者的角度陈述，以帮助您了解您需要应付的人的思考方式。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、巧用证据，逐步瓦解抵抗力
   </strong>
  </p>
  <p class="graf graf--p">
   在审讯中证据经常被拿来施压，但什么时候使用证据、如何令证据发挥关键作用，是间谍课程的一个重点。
  </p>
  <p class="graf graf--p">
   显然其要点也是心理战术。即 攻心术。
  </p>
  <p class="graf graf--p">
   有效的攻心术可以比酷刑的效果更高。
  </p>
  <p class="graf graf--p">
   审讯者都明白，在长时间的心理战之后，被审讯者已经疲惫不堪，其思想会徘徊在招供和不招供之间。这个时候就是施加攻心术的最佳时机。
  </p>
  <p class="graf graf--p">
   也就是说，在被审讯者思想波动幅度最大的时候，审讯者及时给对方足够的理由令其招供；令其感觉到没有其他选择。
  </p>
  <p class="graf graf--p">
   这时候有力的证据就是一张王牌。
  </p>
  <p class="graf graf--p">
   反过来看，不让你的对手拿到最致命的证据，是你能做得最好的反制策略。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1984时代，当局的证据基本都来自长期监视，不论是您的数字足迹还是您的在线和离线身份的关联，对您来说都是致命的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   所以，避免被攻心术击倒的最佳方案就是，从一开始就尽可能不要留下证据给你的对手。
  </p>

   如果您擅长以上全部方法，但最终因被亲友出卖而导致失去了反制机会，那么能保护您的方法就是撒谎。
  </p>
  <p class="graf graf--p">
   如果您平时保护工作做得很好，应该就能有信心相信审讯者手中掌握的证据不多；于是即便有人出卖您，您也能有一个大致的危险性评估。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这一评估非常重要，将决定您在遭受攻心术时的抵抗力。
   </strong>
  </p>
  <p class="graf graf--p">
   并且能决定您的撒谎计划的有效性 —— 目标是让对方的逻辑链断掉。
  </p>
  <p class="graf graf--p">
   追求零口供的被审讯者往往通过保持沉默来抵抗，理论上可行，但有些时候效果并不好。因为你越是不说话，审讯者越会觉得你隐瞒了很多。
  </p>
  <p class="graf graf--p">
   如果你能表演出一种似乎很配合的样子，反而会令审讯者放松警惕。这种时候熟知法律很重要 —— 审讯者使用的都是套证据的方式，如果最终得不到有效的口供，就无法定罪。
  </p>
  <p class="graf graf--p">
   只有证据才能让人认罪。在缺乏证据的时候，审讯者甚至会人为地创造“证据”，也就是栽赃。
  </p>
  <p class="graf graf--p">
   栽赃的信息往往不合逻辑，这种情况下你需要在律师到场之前不做任何评论。
  </p>
  <p class="graf graf--p">
   不论是真实的证据还是栽赃，审讯者都非常注重
   <strong class="markup--strong markup--p-strong">
    证据出示的时机
   </strong>
   ，他们的目标是给被审讯者制造最强大的心理压力、给自己立威造势。
  </p>
  <p class="graf graf--p">
   作为被审讯者您应该了解对手这一心理，如果您害怕就真的输了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、咆哮式审讯击毁心理防线
   </strong>
  </p>
  <p class="graf graf--p">
   理论上咆哮不是暴力，但是它能制造一种压力氛围，尤其是它持续进行时。
  </p>
  <p class="graf graf--p">
   在咆哮的过程中，审讯者往往会以最激烈的语调讲述他们获取的证据，尤其是，
   <strong class="markup--strong markup--p-strong">
    穿插引用被审讯者的心理弱点
   </strong>
   —— 比如父母、朋友、孩子。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    内心缺乏安全感的人对咆哮式审讯的氛围会非常敏感，更容易中计而产生屈服心理
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这需要审讯者事先做出社交工程调研，以掌握被审讯者最脆弱的心理敏感部位。反过来，被审讯者平时暴露自己的心理越多，这种情况下就会越危险。
  </p>
  <p class="graf graf--p">
   ⚠️人们经常觉得一些“不敏感的内容”可以在网上随便说，其实不然；所有这些信息综合在一起很容易暴露你的心理特征，令你的对手可以更准确地掌握攻心术的设置模型。
  </p>
  <p class="graf graf--p">
   还是那句话，让你的对手对你的了解越少、了解得越不真实，你就越安全。
  </p>
  <p class="graf graf--p">
   根据中情局的观点，咆哮式审讯有几点技巧：
  </p>
  <p class="graf graf--p">
   A “一怒到底”。绝不可以浅尝；否则会陷入一种尴尬的境地。一旦罪犯的感情没能调动起来，审讯者就输了。
  </p>
  <p class="graf graf--p">
   需要持久、极致，在心理上占据高地。
  </p>
  <p class="graf graf--p">
   B “要果断”。必需显得无比自信，即便审讯者自己并没有拿到切实证据，也必须表现得就像掌握全局一样，要看起来不容置疑。
  </p>
  <p class="graf graf--p">
   这种氛围将
   <strong class="markup--strong markup--p-strong">
    迫使被审讯者内心展开自我审查；一旦自我审查的意识出现，被审讯者就会立刻虚弱
   </strong>
   ，就离投降不远了。
  </p>
  <p class="graf graf--p">
   C “言之有物”。咆哮不能仅仅是愤怒，如上所述，必需在咆哮中拿出切实证据。即便证据并不充分，在情绪压力的大环境下，被审讯者往往来不及仔细思考逻辑问题，从而屈服于不完整的证据。
  </p>
  <p class="graf graf--p">
   或者说，证据不完整的时候，审讯者采取咆哮方式的可能性更大。
  </p>
  <p class="graf graf--p">
   所以被审讯者需要能保持冷静。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-6918 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-14.png?resize=1100%2C621&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-14.png?w=1254&amp;ssl=1 1254w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-14.png?resize=300%2C169&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-14.png?resize=768%2C434&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-14.png?resize=1024%2C578&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-14.png?resize=1100%2C621&amp;ssl=1 1100w" data-recalc-dims="1" height="621" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-14.png?resize=1100%2C621&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6918" data-recalc-dims="1" height="621" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-14.png?resize=1100%2C621&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-14.png?w=1254&amp;ssl=1 1254w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-14.png?resize=300%2C169&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-14.png?resize=768%2C434&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-14.png?resize=1024%2C578&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-14.png?resize=1100%2C621&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、语言攻势撕破心理伪装
   </strong>
  </p>
  <p class="graf graf--p">
   语言攻势在很多情况下都可以使用，而不仅仅是对峙过程中。
  </p>
  <p class="graf graf--p">
   在“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
    社交工程
   </a>
   ”栏目中包含很多关于如何以语言做武器的方法介绍。
  </p>
  <p class="graf graf--p">
   这里指的语言攻势是以令对方折服为目的的，它具有两个特点：
  </p>
  <p class="graf graf--p">
   A 音调方面的优势。
  </p>
  <p class="graf graf--p">
   在交锋过程中音量会被试图放大，为的是能在心理上占据优势地位。
  </p>
  <p class="graf graf--p">
   但是不能一味地放大，如果审讯者在拿出切实证据的那一刻依旧保持了极高的音量，反而会减弱证据的压力作用。
  </p>
  <p class="graf graf--p">
   有效的语言攻势是音量合理起伏的，在持续高声的过程中忽然调低音量，将更容易吸引对方的注意力，令语言更加深入内心。
  </p>
  <p class="graf graf--p">
   B 紧抓对方的疏漏。
  </p>
  <p class="graf graf--p">
   俗话说百密一疏，疏漏可能是非常致命的。但是并非每一种疏漏都容易被发觉。当审讯者通过调查抓到了对方的疏漏时，就会在语言攻势中强化运用它，以瞄准被审讯者的心理弱点。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、制造绝望
   </strong>
  </p>
  <p class="graf graf--p">
   成功进攻他人内心的关键方法就是，让对方自觉已经断了后路。
  </p>
  <p class="graf graf--p">
   绝望感会导致对方主动放弃挣扎，接下来的一切都随你处置了。
  </p>
  <p class="graf graf--p">
   审讯者追求的就是这种调动你绝望的感受，要达成这一目的，必需要有切实依据，能真正触动对方的信息。
  </p>
  <p class="graf graf--p">
   虽然这一信息很可能不值得深究，但
   <strong class="markup--strong markup--p-strong">
    只要它能制造强大的绝望感，几乎没有人会详细思考它为什么不合逻辑
   </strong>
   。
  </p>
  <p class="graf graf--p">
   绝望是能杀死人的。
  </p>
  <p class="graf graf--p">
   ⚠️一旦萌生绝望感，对方就会迅速暴露心理弱点，审讯者将及时抓住这些线索，展开更加强大的攻势。
  </p>
  <p class="graf graf--p">
   间谍在采取攻心战术时往往会遵循：
  </p>
  <p class="graf graf--p">
   A 不强行逼供。
  </p>
  <p class="graf graf--p">
   事实上外在的高压逼迫只会让对方产生更大的抵触心理。
  </p>
  <p class="graf graf--p">
   相反，聪明的暗示可以让对方知难而退。
  </p>
  <p class="graf graf--p">
   于是有经验的间谍会做得非常巧妙，
   <strong class="markup--strong markup--p-strong">
    一步步引导被审讯者走入绝望
   </strong>
   。
  </p>
  <p class="graf graf--p">
   B 要避免被对方看出己方实力。
  </p>
  <p class="graf graf--p">
   从心理学角度上，威胁的真实含义就是：实施者其实没有绝对的把握能战胜对方，他只是希望借助心理压力让对方主动投降。
  </p>
  <p class="graf graf--p">
   换句话说，如果您是被审讯者，
   <strong class="markup--strong markup--p-strong">
    遇到上述心理攻势时切不可慌乱，因为让你慌乱正是对手的目的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如上，是最常见的审讯心理战。此外其他方法还有：
  </p>
  <p class="graf graf--p">
   5、疲劳战术。这是针对心理素质很好的被审讯者所采取的方案。
  </p>
  <p class="graf graf--p">
   这样的被审讯者绝不会主动交代，审讯者采取询问的方法是无效的，尤其是被反复提及的问题，更不会回复。
  </p>
  <p class="graf graf--p">
   这种情况下审讯者会采取疲劳战术，直到对方倦怠，再准确抓住被审讯者的逻辑错误并强烈打击。
  </p>
  <p class="graf graf--p">
   6、多说不如少说。这也是一种心理技巧。目的是给对方一种神秘感，让对方摸不透审讯者究竟掌握了什么。
   <strong class="markup--strong markup--p-strong">
    这种神秘感有可能激发被审讯者的自我审查，从而进一步表现出心虚和不安
   </strong>
   。
  </p>
  <p class="graf graf--p">
   7、诱导。也就是利用语言学引诱对方说出对自己不利的信息，却不会感知到。这是社交工程伎俩，关于这点详见另外的具体介绍：
  </p>
