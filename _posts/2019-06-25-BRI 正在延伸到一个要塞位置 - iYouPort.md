  <p class="graf graf--p">
   上周的新闻中我们介绍了为什么克里米亚是一个要塞位置，详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/newsletters-06202019-8d2b6c303dc1" href="https://www.iyouport.org/%E7%BA%A0%E7%BB%93%E7%9A%84%E8%B0%88%E5%88%A4/" rel="noopener noreferrer" target="_blank">
    纠结的谈判
   </a>
   》。现在，俄罗斯国家消息人士和官员已经证实，北京和克里米亚半岛之间就主要的一带一路倡议（BRI）项目进行了“更密切的合作”，引用了中国外交官的话称，这代表了莫斯科优先事项的持续重大转变，这些优先事项历来使俄罗斯陷入与中国的经济竞争中。
  </p>
  <p class="graf graf--p">
   任何涉及克里米亚未来的 BRI 项目都将进一步巩固和完全将半岛纳入俄罗斯。
  </p>
  <p class="graf graf--p">
   星期一在俄罗斯驻北京大使馆举行的仪式上，半岛“中国同胞协会”会长 Ge Zhili
   <a class="markup--anchor markup--p-anchor" data-href="http://tass.com/economy/1049077" href="http://tass.com/economy/1049077" rel="noopener noreferrer" target="_blank">
    发表了以下正式声明
   </a>
   ：“我们的组织正在加强与克里米亚社会的合作关系，交流和友好交往“。事件并非偶然，此时是”克里米亚与俄罗斯统一五周年“。
  </p>
  <p class="graf graf--p">
   在谈到增加文化和经济联系时，Ge 继续解释目标是“帮助人们了解克里米亚是什么，开始与克里米亚合作，与半岛建立可靠和友好的关系，并在 BRI 下实施联合私人项目和道路倡议“ — — 中国超雄心勃勃的发展和投资项目可能涉及五大洲150多个国家。
  </p>
  <p class="graf graf--p">
   中国的声明说，据信当莫斯科不再追求美国制裁后的欧洲一体化，以及INF的解体和北约的扩张时，中国强调与其建立和深化“可靠的伙伴关系”。
  </p>
  <p class="graf graf--p">
   这也是传闻北京和莫斯科正在考虑建立一个潜在的更大的地缘政治联盟的关键时刻，就像华盛顿继续通过关税来打击中国出口一样。俄罗斯领导人被描述为更加热情地寻求更密切的合作，而北京已表示了耐心。
  </p>
  <p class="graf graf--p">
   但是，更大的 BRI 合作，尤其是对地缘政治敏感的克里米亚地区的合作，将是朝这个方向迈出的重要一步。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-2867 jetpack-lazy-image" data-lazy-sizes="(max-width: 938px) 100vw, 938px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-52.png?resize=938%2C692&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-52.png?w=938&amp;ssl=1 938w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-52.png?resize=300%2C221&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-52.png?resize=768%2C567&amp;ssl=1 768w" data-recalc-dims="1" height="692" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-52.png?resize=938%2C692&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="938"/>

  <p class="graf graf--p">
   虽然很少有关于新的潜在特定联合 BRI 项目的宣传，除了正在进行的项目，如 连接中国和俄罗斯在黑龙江省通江的第一条跨河铁路桥梁，以下是 RT 提出的内容：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    此外，还计划恢复克里米亚与中国南部海南省之间以及雅尔塔，三亚，塞瓦斯托波尔，大连，苏达克和黑河等城市之间的合作
   </em>
   。
  </p>
  <p class="graf graf--p">
   去年，俄罗斯总统普京宣称自己是中国“一带一路”倡议的坚定支持者，
   <a class="markup--anchor markup--p-anchor" data-href="https://news.cgtn.com/news/3d3d414e3149544f77457a6333566d54/share_p.html" href="https://news.cgtn.com/news/3d3d414e3149544f77457a6333566d54/share_p.html" rel="noopener noreferrer" target="_blank">
    他告诉广播公司 CGTN
   </a>
   ，“习主席的”一带一路“倡议融合了经济和人性的概念。该倡议的实质是发展经济和基础设施。“
  </p>
  <p class="graf graf--p">
   关于与北京的关系升温，普京当时强调，BRI 把这一潜力视为“进一步取消对经济发展和合作的限制的重要一步”。
  </p>
  <p class="graf graf--p">
   克里米亚本身未来的任何重大合作肯定会引起基辅和乌克兰西方盟友的关注，尤其是去年11月在刻赤海峡发生的危险事件以及随后的军事升级威胁之后。
  </p>
