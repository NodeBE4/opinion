  <p class="graf graf--p">
   在电影《黑客帝国》中，人类生活在反乌托邦时代，被强大的系统囚禁在虚拟世界中。
  </p>
  <p class="graf graf--p">
   他们一生都在虚拟仿真中生活，而没有任何方式能知道自己感觉到的东西实际上只是由人为编造的代码构成的。
  </p>
  <p class="graf graf--p">
   当前社会的生活几乎与此完全一样。
  </p>
  <p class="graf graf--p">
   唯一有所不同的是，让我们在 “Matrix” 中熟睡的是疯狂的寡头权势，而不是AI。而且它不是代码构成的，而是叙述。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%85%a8%e7%90%83%e6%94%bf%e6%9d%83%e5%a6%82%e4%bd%95%e5%88%a9%e7%94%a8%e6%8a%80%e6%9c%af%e5%92%8c%e8%b4%ab%e5%9b%b0%e5%a5%b4%e5%bd%b9%e4%ba%ba%e6%b0%91%ef%bc%9f%e6%9a%b4%e6%94%bf%e8%83%8c%e5%90%8e/" href="https://www.iyouport.org/%e5%85%a8%e7%90%83%e6%94%bf%e6%9d%83%e5%a6%82%e4%bd%95%e5%88%a9%e7%94%a8%e6%8a%80%e6%9c%af%e5%92%8c%e8%b4%ab%e5%9b%b0%e5%a5%b4%e5%bd%b9%e4%ba%ba%e6%b0%91%ef%bc%9f%e6%9a%b4%e6%94%bf%e8%83%8c%e5%90%8e/" rel="noopener noreferrer" target="_blank">
     全球政权如何利用技术和贫困奴役人民？暴政背后的黑手
    </a>
    》
   </li>

  <p class="graf graf--p">
   社会是由叙述构成的，就像 Matrix 是由代码构成的。身份、语言、礼节、社会角色、见解、意识形态、宗教、种族、哲学、议程、规则、法律、金钱、经济学、工作、等级制度、政治、政府 ……它们都是纯粹的精神建构，都是强加于头脑中的精神噪音构成的。
  </p>
  <p class="graf graf--p">
   如果我要你指出自己的膝盖，你可以迅速准确地做到；但是，如果我要你指出经济，最接近的可能只是使用一堆语言符号来指向一组概念。为了向我展示经济状况，您必须给我讲一个故事 — 那就是叙述。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9957 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.jpg?resize=1100%2C825&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.jpg?w=1400&amp;ssl=1 1400w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.jpg?resize=300%2C225&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.jpg?resize=768%2C576&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.jpg?resize=1024%2C768&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.jpg?resize=1100%2C825&amp;ssl=1 1100w" data-recalc-dims="1" height="825" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.jpg?resize=1100%2C825&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   任何曾经经历过片刻精神平静的人都知道，这些都不是您当前实际经验的一部分。
  </p>
  <p class="graf graf--p">
   在您的经历中，没有精神、话题、语言、礼节、社会角色、见解、意识形态、宗教、种族、哲学、议程、规则、法律、金钱、经济学、工作、等级制度、政治或政府，没有关于这些东西的喋喋不休。
  </p>
  <p class="graf graf--p">
   你摸不到它们的真实形状，你能感觉到的仅仅是关于“它们是什么”的叙述。
  </p>
  <p class="graf graf--p">
   甚至找不到一个“你”，因为事实证明它也是由叙述构成的。
  </p>
  <p class="graf graf--p">
   没有心理叙事，没有经历，唯有感觉上的印象出现在没有清晰形状或边界的对象身上。
  </p>
  <p class="graf graf--p">
   那种视觉和听觉区域的经历、空气进入和离开呼吸系统的感觉，脚在地面上或椅子上的感觉，那或多或少是总的生活
   <strong class="markup--strong markup--p-strong">
    减去叙述
   </strong>
   所得到的东西。它就是现实。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    当您加入精神上的喋喋不休时，所有这些现实都不会再能引起关注
   </strong>
   。
  </p>
  <p class="graf graf--p">
   视觉和听觉领域中的外观突然被划分、并用语言标记，并且对它们的关注仅仅取决于威胁或满足各种议程的人。
  </p>
  <p class="graf graf--p">
   你经过了数天、数周、数月或数年的时间，都没有真正注意到呼吸系统或脚在地面上的感觉，因为你的兴趣和注意力完全被吸引到了纯粹是由叙述组成的虚假世界中。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我足够好吗？我做对了吗？我需要确保完成了所有项目……如果我先做一件事，从长远来看可能会节省一些时间吗？如果我能得到自己想要的东西并实现重要目标，我会感觉很好吗？……税单要到期了。电视上有什么？哦，那些白痴，他们怎么还有脸活着？天哪，我等不及周末了 ……我希望从现在开始一切都按计划进行 ……”
  </p>
  <p class="graf graf--p">
   不断地，不断地。
   <strong class="markup--strong markup--p-strong">
    几乎所有的精神能量都进入到这些心理叙事中
   </strong>
   。
  </p>
  <p class="graf graf--p">
   是它们在统治着我们生活的一切。⚠️出于这个原因，能够控制这些叙述的人就可以控制我们。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    他们已经做到了
   </strong>
   。
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   大多数人都会试图对周围的人施加某种程度的控制。
  </p>
  <p class="graf graf--p">
   他们试图通过某种行为和言语来影响家庭家人、社会和就业圈子中其他人的想法。
  </p>
  <p class="graf graf--p">
   你的父母长辈以他们自己的经历告诉你如何才能做到 “聪明/才干”，你听到的 “这个世界” 完全是你父亲的心理活动；浪漫的伴侣将被说服他们永远不会分手，因为没有其他人会爱上他们 ……
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    他们都在不同程度上操纵着个人的叙述
   </strong>
   。
  </p>
  <p class="graf graf--p">
   然后有人发现他们实际上可以发挥自己的能力来影响人们对自己和世界的看法，并将其转化为他们自己的个人利益。
  </p>
  <p class="graf graf--p">
   邪教领袖说服追随者为他们服务，终其一生，任劳任怨；
  </p>
  <p class="graf graf--p">
   广告商说服消费者相信他们自己有一个问题或不足，只有使用“这个令人兴奋的新产品”才能解决；
  </p>
  <p class="graf graf--p">
   雄心勃勃的办公室政治玩家通过在合适的人中赢得青睐并对与之竞争的同伴使用隐秘的破坏行为来攀登公司晋升的阶梯；
  </p>
  <p class="graf graf--p">
   记者们知道如何通过推进叙事方式以令他们自己在事业上进一步发展，而此叙事方式必然必需是有利于拥有大型媒体公司的富豪们建立王国的基础……
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    他们操纵着群体的叙述。
   </strong>
  </p>
  <p class="graf graf--p">
   然后就是寡头。最主要的操纵者。
  </p>
  <p class="graf graf--p">
   这些现代世界的寡头统治者迅速学到了自文明诞生以来每个统治者都知道的秘密：⚠️只要你能控制住一个社会的人们相信某种叙述，你就是那个社会必然的统治者。
  </p>
  <p class="graf graf--p">
   身份、语言、礼节、社会角色、见解、意识形态、宗教、种族、哲学、议程、规则、法律、金钱、经济学、工作、等级制度、政治、政府：所有心理结构只会在一定程度上影响社会的信任并获得绝大多数群体的认同。
  </p>
  <p class="graf graf--p">
   ⚠️如果你能对人们所相信的那些心理构造的事物产生影响，那么你就能对社会产生影响。你就能统治它。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    寡头操纵整个社会的叙述。
   </strong>
  </p>

    为自己创造新的叙事，更好的叙事，更健康的叙事。彼此之间，关于世界如何以及我们想要什么
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这是事实：因为都是叙事，所以一切皆有可能。
  </p>
  <p class="graf graf--p">
   与那些同样走出 Matrix 的人联合在一起，你们可以确定社会将如何发展；可以重写规则。它已经开始了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    从失败的宣传机器发出的白噪声中，诞生了一个新世界，一个尊重个人自主权和自决权的世界
   </strong>
   。
  </p>
  <p class="graf graf--p">
   尊重我们进行大规模合作以创建美丽、健康、有用的系统的权利，
   <strong class="markup--strong markup--p-strong">
    而不会
   </strong>
   把一切精力都用来反驳那些顽固地宁愿被奴役的精神病态患者。
  </p>
  <ul class="postList">
   <li class="graf graf--li graf--startsWithDoubleQuote">
    <strong class="markup--strong markup--li-strong">
     “你永远不可能仅仅通过对抗现有的现实来改变这一切。要改变某些东西，你必须建立一个新的模型，使现有模型过时”
    </strong>
    — — 《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/4%E3%80%81%E5%88%AB%E6%8A%8A%E6%B3%A8%E6%84%8F%E5%8A%9B%E9%9B%86%E4%B8%AD%E5%9C%A8%E6%8A%B5%E6%8A%97%E4%B8%8A%EF%BC%9A%E9%A2%84%E8%AE%BE%E6%80%A7%E5%B9%B2%E9%A2%84%E6%88%98%E6%9C%AF/" href="https://www.iyouport.org/4%E3%80%81%E5%88%AB%E6%8A%8A%E6%B3%A8%E6%84%8F%E5%8A%9B%E9%9B%86%E4%B8%AD%E5%9C%A8%E6%8A%B5%E6%8A%97%E4%B8%8A%EF%BC%9A%E9%A2%84%E8%AE%BE%E6%80%A7%E5%B9%B2%E9%A2%84%E6%88%98%E6%9C%AF/" rel="noopener noreferrer" target="_blank">
     预设性干预战术
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   从恐惧所浸透的精神错乱的叙述中完全释放出来，让真实世界的真实美感充满您的感官。
  </p>
  <p class="graf graf--p">
