   <p class="graf graf--p graf-after--figure" id="336a">
    如果您还没有阅读或者已经忘记了我们在上一篇文章中所介绍的技巧经验，
    <strong class="markup--strong markup--p-strong">
     请在这里回顾它《
    </strong>
    <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/thetruth-57d38bb322cc" href="https://www.iyouport.org/%E5%85%AC%E6%B0%91%E5%8F%96%E8%AF%81%E6%8A%80%E5%B7%A7%EF%BC%9A%E5%A6%82%E4%BD%95%E7%95%99%E4%BD%8F%E7%9C%9F%E7%9B%B8%EF%BC%9F/" rel="noopener noreferrer" target="_blank">
     <strong class="markup--strong markup--p-strong">
      如何留住真相？
     </strong>
    </a>
    <strong class="markup--strong markup--p-strong">
     》
    </strong>
    。本文将具体就每一个步骤进行说明，您值得思考哪些问题、最好的处理方式是怎样的，以及顺手的技术储备等等，分享来自国际权利组织经验人士的意见。
   </p>
   <p class="graf graf--p graf-after--p" id="9fce">
    我们将尽可能精炼，只说要点。
   </p>
   <p class="graf graf--p graf-after--p" id="4425">
    <strong class="markup--strong markup--p-strong">
     — — 在开始拍摄之前 — —
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="8b1e">
    1、在开始拍摄之前，请确保视频中出现的内容是正确的选择; 因为您、您的组织、朋友和被您拍摄的人都面临着潜在的风险，拍摄、编辑和分发时都会出现不同类型的风险。请考虑周全，以便事先准备应对措施。
   </p>
   <p class="graf graf--p graf-after--p" id="3451">
    2、您需要有一个计划，请重
    <strong class="markup--strong markup--p-strong">
     点关注
    </strong>
    两件事：您是否有明确的、具体的变革目标？以及可以帮助您实现这一目标的明确受众？确立这两点后你就会知道该如何让拍摄成为达成这一目标的工具。
   </p>
   <p class="graf graf--p graf-after--p" id="000b">
    3、拍摄不是为了分享给朋友寻求一时的刺激转瞬即忘，它需要能
    <strong class="markup--strong markup--p-strong">
     发挥最大作用
    </strong>
    。制作完视频后，如何能在恰当的时间将视频呈现在正确的观看者面前？本身就是个学问。时机往往非常重要。您需要组织一天的行动、动员全球在线支持社区联合采取行动，或者将您的视频放在正在决定关键案例或政策制定的法官面前，以便作为关键证据对立法进行投票。
   </p>
   <p class="graf graf--p graf-after--p" id="8f04">
    4、请确保有充分的“信息”可以提供给您的受众群，它有助于为您的视频提供直接、具体的
    <strong class="markup--strong markup--p-strong">
     行动请求
    </strong>
    ：“这就是正在发生的事。行动起来！结束压迫、歧视和不公正！”
   </p>
   <p class="graf graf--p graf-after--p" id="6193">
    5、你讲的故事 — 你选择的场景、说出真相的人 — 都是传达信息的有利工具。因此，请为您的视频选择
    <strong class="markup--strong markup--p-strong">
     一个焦点
    </strong>
    — 一个充满情感、道德和说服力的故事，并以强大的个人经验以及特定的时间和地点为基础，支撑这个故事，
   </p>
   <p class="graf graf--p graf-after--p" id="3d09">
    6、许多宣传视频包括强大的、准确的事实和背景设置镜头，能很好的帮助观众了解整件事。只要你能抓住要点，就能令观众产生共鸣 — 所以，你应该让
    <strong class="markup--strong markup--p-strong">
     受到该问题直接影响的人
    </strong>
    谈论这一问题，而不是由你来代表他们做解释。要确保他们的表达能提供整个画面的有效补充。
   </p>
   <p class="graf graf--p graf-after--p" id="4983">
    7、成功的片子能
    <strong class="markup--strong markup--p-strong">
     激发观众采取行动
    </strong>
    。让您的观众感受到包容、参与、激励、充满希望或愤怒和羞辱等与自己切身利益相关的反馈，然后为这一明确的选择做好准备。
   </p>
   <p class="graf graf--p graf-after--p" id="0dff">
    8、如果您能与他人合作，将可以产生更大的影响力。视觉信息作为最直观的动力 — 指导一天的行动、请愿、游行、报告撰写或法律案件。在线随视频一起提供背景信息，并让观众清楚地了解在线和离线的
    <strong class="markup--strong markup--p-strong">
     参与
    </strong>
    方式。
   </p>
   <p class="graf graf--p graf-after--p" id="a7d9">
    9、请尽可能确保您拍摄到的人均给予
    <strong class="markup--strong markup--p-strong">
     知情同意
    </strong>
    — 这意味着他们了解自己被拍摄的风险和好处，为他们呈现视频效果，并告诉您是否需要隐藏自己的身份。您可能想要谈谈潜在的最坏情况，例如当压迫他们的人看到该视频时会发生什么，同时也需要说明您拍摄它的意义：带来改变的契机。
   </p>
   <p class="graf graf--p graf-after--p" id="c45d">
    10、你并不需要太多的设备，但你需要做好充分准备，所以最好能制定一个计划：你的故事是什么？你想怎么表述这个故事？拍摄的地点/时间/人物都是什么？谁可以帮助你？
   </p>
   <p>
    <img alt="" class="aligncenter size-full wp-image-2466 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-29.jpeg?resize=1100%2C692&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-29.jpeg?w=1546&amp;ssl=1 1546w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-29.jpeg?resize=300%2C189&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-29.jpeg?resize=768%2C483&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-29.jpeg?resize=1024%2C644&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-29.jpeg?resize=1100%2C692&amp;ssl=1 1100w" data-recalc-dims="1" height="692" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-29.jpeg?resize=1100%2C692&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
    <noscript>
     <img alt="" class="aligncenter size-full wp-image-2466" data-recalc-dims="1" height="692" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-29.jpeg?resize=1100%2C692&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-29.jpeg?w=1546&amp;ssl=1 1546w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-29.jpeg?resize=300%2C189&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-29.jpeg?resize=768%2C483&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-29.jpeg?resize=1024%2C644&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-29.jpeg?resize=1100%2C692&amp;ssl=1 1100w" width="1100"/>
    </noscript>
   </p>
   <p class="graf graf--p graf-after--figure" id="372b">
    <strong class="markup--strong markup--p-strong">
     — — 拍摄抗议示威、群体事件和警察行为 — —
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="5eb1">
    1、在不同国家这类内容的拍摄所面临的风险是不同的。在打开相机之前请确保您了解自己的法律权利，制定安全计划以保护您的安全。请记下紧急联系方式，或将其写在安全的位置。使用相机带或将相机系在手腕上，以免冲突中脱落损坏。如果做不到这点，那就不要携带这一设备。
   </p>
   <p class="graf graf--p graf-after--p" id="da0b">
    2、带上
    <strong class="markup--strong markup--p-strong">
     额外的
    </strong>
    充电器和空的存储卡。如果使用智能手机拍摄，可以通过关闭应用程序和 wifi 定位器、以及降低设备屏幕的亮度，来节省电池电量。
   </p>
   <p class="graf graf--p graf-after--p" id="ab3c">
    3、首先把镜头面向自己，对着相机说话，以确认您是拍摄者。或者，使用原始视频文件保留书面记录。如果出于安全原因需要匿名，请使用一个化名。
   </p>
   <p class="graf graf--p graf-after--p" id="d272">
    4、您需要尽可能捕获素材的日期、时间和位置，以便可以更轻松地
    <strong class="markup--strong markup--p-strong">
     验证信息
    </strong>
    。如果可能，请启用自动日期、时间和 GPS 位置捕获功能。或者，电影海报、新建或建造一半的大型建筑物、街道标志和地标，都可以。保持拍摄至少10秒。更换位置时，​​请保持相机稳定并缓慢的移动，避免抖动和缩放。
   </p>
   <p class="graf graf--p graf-after--p" id="cebc">
    5、通过广泛和中等镜头对事件进行
    <strong class="markup--strong markup--p-strong">
     语境化
    </strong>
    ；使用特写镜头捕捉细节；使用各种角度和高度来捕捉人群的规模大小和行为；捕获有关当局的存在和其形成的镜头。如果身份应该是匿名的，那么镜头应该出现在背后，只记录人们头部或脚部的背面。
   </p>
   <p class="graf graf--p graf-after--p" id="b422">
    6、在安全的情况下，通过拍摄相关人员、周围人群、受伤、弹孔或附近车辆的牌照来记录抓捕或暴力的详细信息。您也可以通过在拍摄时
    <strong class="markup--strong markup--p-strong">
     口述事件
    </strong>
    ，来对细节进行语境化。请注意一定要准确叙述，尽可能使用动词而不是形容词，避免您的个人意见进入录制。
   </p>
   <p class="graf graf--p graf-after--p" id="2fb9">
    7、如果可能的话，拍摄或录制您自己讲述的军官姓名和徽章/佩章号码。记录警官的行为，例如给予下属的命令，殴打、摧毁等等，以及创建或移动路障。记录正在向抗议人群射击的警察最好能有面部特写、以及当局持有的或使用的
    <strong class="markup--strong markup--p-strong">
     任何武器
    </strong>
    。尽可能停留10秒左右。
   </p>
   <p class="graf graf--p graf-after--p" id="65fc">
    8、通过与获取同意的抗议参与者或目击者进行访谈以提供更多的细节。请确保您拍摄到的人完全了解该视频的使用方式和位置。如果要在公开场合、在线或与法庭共享视频，请与被拍摄者讨论潜在的安全风险。如果需要
    <strong class="markup--strong markup--p-strong">
     匿名
    </strong>
    ，请在采访时拍摄受访者的手部，让拍摄对象将他们的脸裹在围巾中，或调整焦点以模糊图像。
   </p>
   <p class="graf graf--p graf-after--p" id="5d3c">
    9、您最好能有一位
    <strong class="markup--strong markup--p-strong">
     合作伙伴
    </strong>
    以帮助您保持安全，并留意周边重要情况以进行记录。如果您的合作者有相机，请尝试获取相同事件的不同角度，尽量保持彼此的视野最终都包含进去。如果您有被捕的风险但希望继续拍摄，请考虑将储存卡转交给能够最大程度保护它的朋友。
   </p>
   <p class="graf graf--p graf-after--p" id="376b">
    <strong class="markup--strong markup--p-strong">
     — — 拍摄中：安全采访，并获得被拍摄者的知情同意 — —
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="8991">
    1、请尽可能确保您、您的受访者和您所属的社区的安全；确定您是否需要隐藏身份；如果在线或公开分享该访谈，请考虑其他方面的安全隐患。这
    <strong class="markup--strong markup--p-strong">
     不是恐惧而是理智
    </strong>
    ，我们不鼓励盲目的激进，每个人的勇气都应该发挥实质作用。
   </p>
   <p class="graf graf--p graf-after--p" id="01c2">
    2、我们刚才谈到了戴口罩、面巾等方式隐藏被采访对象和您自己的脸，但是您还需要注意尽可能不要拍摄受访对象和自己身上的
    <strong class="markup--strong markup--p-strong">
     特殊标记
    </strong>
    ，比如疤痕、纹身、多孔的配饰等等，甚至还包括特殊的体态（肢体残缺等）这些细节都会暴露一个人的身份。
   </p>
   <p class="graf graf--p graf-after--p" id="8bfe">
    3、对你的受访者做点准备工作。提前安排好您的问题，以便获得所需的答案。提出以“为什么”，“如何”或“告诉我”开头的
    <strong class="markup--strong markup--p-strong">
     开放式问题
    </strong>
    。避免那些可以回答“是”或“否”的问题。这点我们在社交工程学技巧中有过详细解释，请在这里回顾《
    <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E8%AF%AD%E8%A8%80%E6%98%AF%E5%BC%BA%E5%A4%A7%E7%9A%84%E6%AD%A6%E5%99%A8-%E5%A6%82%E4%BD%95%E4%BD%BF%E7%94%A8%E5%AE%83-%E7%A4%BE%E4%BA%A4%E5%B7%A5%E7%A8%8B%E6%94%BB%E5%87%BB%E6%BC%94%E7%A4%BA%E7%BB%A7%E7%BB%AD-%E8%87%B4%E5%91%BD%E7%9A%84%E8%AF%B1%E5%AF%BC-a545ca831d24" href="https://www.iyouport.org/%E8%AF%AD%E8%A8%80%E6%98%AF%E5%BC%BA%E5%A4%A7%E7%9A%84%E6%AD%A6%E5%99%A8%e2%80%8a-%e2%80%8a-%E5%A6%82%E4%BD%95%E4%BD%BF%E7%94%A8%E5%AE%83%EF%BC%9F%E7%A4%BE%E4%BA%A4%E5%B7%A5%E7%A8%8B/" rel="noopener noreferrer" target="_blank">
     语言是强大的武器，如何使用它？
    </a>
    》
   </p>
   <p class="graf graf--p graf-after--p" id="e639">
    4、在相对安全的地方进行采访，选择一个安静的位置，避免中断。请移除位于被拍摄对象背后的可能会分散观看者注意力的物品。避免使用旋转椅。如果披露访谈的地理
    <strong class="markup--strong markup--p-strong">
     位置
    </strong>
    会使您的主题面临风险，请不要捕捉地标或其他可识别性细节。
   </p>
   <p class="graf graf--p graf-after--p" id="f032">
    5、糟糕的
    <strong class="markup--strong markup--p-strong">
     音频
    </strong>
    可能会破坏良好的视频。请使用耳机清楚地听到相机正在录制的内容。关闭您可以控制的背景噪音 — 如风扇或电子设备；关掉房间里的所有电话；不要让手机振动，因为它会干扰音频。如果可能，请使用外接式或领夹式麦克风，并在安静的地方进行录制。
   </p>
   <p class="graf graf--p graf-after--p" id="0a77">
    &gt;
    <strong class="markup--strong markup--p-strong">
     更多
    </strong>
    ：移动电话用于录音的内置麦克风意味着手机会从设备周围拾取所有声音，这会对您的录制采访产生挑战，尤其是场景比较嘈杂的情况下，比如大型群体事件、武装冲突、灾区等等。您需要尝试将受访对象引导到相对安静的地带录制采访，将您的设备进口接近说话人，令他的声音尽可能清晰。要获得最佳的录音效果，请使用外接麦克风，你会发现没有麦克风的采访效果有很大不同。当您将领夹式的麦克风固定在受访者身上时，请确保他的声音是唯一被录制的声音；手持麦克风则能给你很好的移动空间，允许一次采访多人。
   </p>
   <p class="graf graf--p graf-after--p" id="156b">
    &gt; 便携式麦克风可以在短距离内工作，因此请务必将麦克风靠近扬声器以获得最佳的声音。选择麦克风时请确保它与手机兼容。手机麦克风类型和便携式摄像机不同，但是有便宜的方法可以将您的手机连接到各种麦克风上。不论您选择哪种麦克风，都要
    <strong class="markup--strong markup--p-strong">
     在拍摄之前测试声音
    </strong>
    。您的手机可能无法在录制过程中监听声音。
   </p>
   <p class="graf graf--p graf-after--p" id="a75f">
    6、当您面对受访者时，光源应始终在您身后。避免在受访者身上留下阴影；户外情况注意太阳的位置和云。
   </p>
   <p class="graf graf--p graf-after--p" id="8c2f">
    7、使用三脚架或固定平面来稳定相机。除非绝对必要，否则切勿移动相机或变焦，尤其是在拍摄对象说话时。
   </p>
   <p class="graf graf--p graf-after--p" id="3970">
    8、您的相机应与拍摄对象处于同一水平，不要向上或向下拍摄，将您的主体从腰部或胸部拍到头顶。对于中等和宽幅拍摄来说，请将拍摄对象略微放置在画面的左侧或右侧。请勿将拍摄对象置于画面中间。
   </p>
   <p class="graf graf--p graf-after--p" id="8168">
    9、清楚地解释视频的目的、如何使用、如何共享、以及谁可以看到它。确认受访者是否能接受录制的风险，
    <strong class="markup--strong markup--p-strong">
     请要求他们用自己的语言解释明确这一风险
    </strong>
    ，把他们的知情同意一起拍摄下来。在高风险的情况下，受访者可以选择隐瞒自己的身份，也可以选择不参加。不可强求。
   </p>
   <p class="graf graf--p graf-after--p" id="dbdd">
    10、让您的受访者在访谈开始时
    <strong class="markup--strong markup--p-strong">
     说出并拼出他们的名字
    </strong>
    ，并在视频中说明他们希望自己如何被识别（例如：“我叫***，我是一名母亲、医生、幸存者等”）。如果存在安全风险并且受访对象要求保持匿名，请考虑使用假名。
   </p>
   <p class="graf graf--p graf-after--p" id="c03e">
    11、请积极
    <strong class="markup--strong markup--p-strong">
     倾听
    </strong>
    ！根据需要重新提出问题。让您的受访者说出完整的句子并将您的问题纳入他们的答案。在受访者的答案结束和下一个问题提出之间，请暂停3秒钟。这将有助于后期编辑；除非绝对必要，否则
    <strong class="markup--strong markup--p-strong">
     不要打断
    </strong>
    你的受访者。使用非语言提示帮助您的受访者保持参与感，例如点头。避免说“是”或“啊哈”。邀请您的受访者添加他们希望记录的任何其他要点或想法。
   </p>
   <p class="graf graf--p graf-after--p" id="71ea">
    12、采访后，补充镜头或 b-roll 提供了背景化细节，并有助于发展个人叙事。在可能的情况下，记录受访者日常生活的场景，例如与朋友、家人、同事或社区会议进行互动的画面。在安全的情况下，包括在访谈中讨论的具体地点或细节的镜头。比如采访对象的话中提到了天安门，您的 b-roll 就可以选取天安门的画面。
   </p>
   <p class="graf graf--p graf-after--p" id="2dc6">
    13、我们强调了需要您的受访对象知情同意，这很重要。您可以
    <a class="markup--anchor markup--p-anchor" data-href="http://www.mediafire.com/download/j92i2w8r05dphkp/Template_InformedConsent_Checklist.docx" href="https://www.mediafire.com/download/j92i2w8r05dphkp/Template_InformedConsent_Checklist.docx" rel="noopener nofollow noreferrer" target="_blank">
     <strong class="markup--strong markup--p-strong">
      在这里的链接下载
     </strong>
    </a>
    <strong class="markup--strong markup--p-strong">
     一个关于知情同意的模版
    </strong>
    ，并根据您的具体情况使用它。
   </p>
   <div class="aspectRatioPlaceholder is-locked">
    <img alt="" class="aligncenter size-full wp-image-2467 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-26.png?resize=800%2C426&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-26.png?w=800&amp;ssl=1 800w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-26.png?resize=300%2C160&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-26.png?resize=768%2C409&amp;ssl=1 768w" data-recalc-dims="1" height="426" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-26.png?resize=800%2C426&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>
    <noscript>
     <img alt="" class="aligncenter size-full wp-image-2467" data-recalc-dims="1" height="426" sizes="(max-width: 800px) 100vw, 800px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-26.png?resize=800%2C426&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-26.png?w=800&amp;ssl=1 800w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-26.png?resize=300%2C160&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-26.png?resize=768%2C409&amp;ssl=1 768w" width="800"/>
    </noscript>

   <div>
    <p class="graf graf--p graf-after--figure" id="d30e">
     <strong class="markup--strong markup--p-strong">
      — — 在团队中拍摄 — —
     </strong>
    </p>
    <p class="graf graf--p graf-after--p" id="7a02">
     您很可能不会单枪匹马，需要和朋友、团队合作，这样能
     <strong class="markup--strong markup--p-strong">
      增强
     </strong>
     大家的作品的质量，尤其是对推动变革和伸张正义的作用。朋友也许不难找，或者您有钱很容易请到帮手，但并非所有帮手都能发挥有效的作用，那是因为经过专业培训的人太少了。以下介绍一些基本的团队技巧。
    </p>
    <p class="graf graf--p graf-after--p" id="6ad9">
     1、首先要讨论并确定每个团队
     <strong class="markup--strong markup--p-strong">
      成员的角色
     </strong>
     。评估与每个角色相关的风险，例如逮捕或伤害。确认大家能接受这些风险，并制定紧急应对计划，以防暴力爆发或团队成员受伤；确定团队如何通过短信或对讲机进行通信；统一在设备上设置正确的日期和时间。
    </p>
    <p class="graf graf--p graf-after--p" id="eed9">
     2、制定一个人携带基本材料，如额外的电池、充电器、储存卡和紧急联系方式；合作伙伴还可以保持团队成员之间的
     <strong class="markup--strong markup--p-strong">
      沟通
     </strong>
     ，搜索引人注目的镜头并协助采访；每个人都应该记录日期、时间和地点，并确定谁在拍摄；捕获上下文的详细信息，例如人群的大小、警察徽章编号或车牌；遵照我们在前文中的详细讲述，有意识地拍摄并保持相机稳定；与团队保持密切联系、定期检查，以确保每个人都安全；如果其中有人被逮捕或拘留，其他团队成员应该
     <strong class="markup--strong markup--p-strong">
      尝试拍摄被逮捕事件
     </strong>
     ，并提醒立即调整合作分工网络。
    </p>
    <p class="graf graf--p graf-after--p" id="0d28">
     3、整合所有拍摄记录并传递给编辑，在必要时提供有关该事件的其他详细信息；如有必要，模糊对象的面孔以保护其身份；上传和分享素材时，请在视频的说明、标签和标题中添加上下文和详细信息。这将使您的视频更易于查找和使用。
    </p>
    <p class="graf graf--p graf-after--p" id="5859">
     4、以下是关于
     <strong class="markup--strong markup--p-strong">
      协作拍摄本身
     </strong>
     的：
    </p>
    <p class="graf graf--p graf-after--p" id="4c28">
     &gt; 尽可能详细拍摄行动，重点关注抗议的细节以及任何警察或军事暴力事件。捕获详细信息和
     <strong class="markup--strong markup--p-strong">
      可识别性信息
     </strong>
     ，如面部，警察名称，徽章，车辆和车牌。
    </p>
    <p class="graf graf--p graf-after--p" id="aceb">
     &gt; 在人群中拍摄以获得动作感。尝试将摄像机1保持在镜头中以确定上下文并提供其他角度；如果被捕或被拘留，摄像机2应确定是否立即更换摄像机1。
    </p>
    <p class="graf graf--p graf-after--p" id="915a">
     &gt; 远景捕捉事件的全部范围。如果可以安全，请记录警察或军队的位置和移动；考虑从窗户，阳台或屋顶拍摄，以获得空中视角。摄像机3应该向队友
     <strong class="markup--strong markup--p-strong">
      传达所有重大发展
     </strong>
     ，例如正在接近的军用车辆、新阵型或暴力行为。
    </p>
    <p class="graf graf--p graf-after--p" id="7e96">
     &gt; 相机具有良好的变焦，三脚架，独脚架或稳定表面支撑；拥有用于备份素材的 Internet 访问和外部硬盘驱动器。
    </p>
    <p class="graf graf--p graf-after--p" id="aa73">
     &gt; 理想情况下，编辑由两人或更多人的团队完成。一个人审查原始素材，而另一个人则专注于编辑所选剪辑、保存和共享素材。此外，建议合作伙伴与拍摄团队进行沟通，以根据需要获取额外的镜头。
    </p>
    <p class="graf graf--p graf-after--p" id="5389">
     最后，请记住我们在前文中推荐过的多个技术，可以使您的拍摄最终能成为证据的很好协助。比如 CameraV，就是在智能手机或平板电脑上捕获和共享可验证的照片和视频证据的最简单方法，尤其是
     <strong class="markup--strong markup--p-strong">
      同时保持其安全和私密
     </strong>
     。 CameraV 的开发主要为协助活动家、记者、倡导者和其他人在非常困难和高风险的情况下的工作，捕获和收集滥用和侵犯权利的视觉证据。它还可以用于您的日常生活中捕捉任何时刻，并保持安全和私密。您还可以使用它来记录您自己的遭遇的证据，事故后、房地产损坏、或任何您需要“证据”以讨回公正的地方。
    </p>
    <p class="graf graf--p graf-after--p" id="dab7">
     好啦，还是那句话，好的技术需要更多的练习。我们只能支招，而您的重视将能带给您真正的实力！一起来，做合格的公民。
    </p>
