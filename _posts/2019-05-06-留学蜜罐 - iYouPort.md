   留学蜜罐
  </h1>
 </header>

 <div class="entry-content">
  <ul>
   <li class="graf graf--p">
    <span style="color: #00ccff;">
     <strong>
      <em>
       留学移民是所有人都知道的“出路”，但是这件事有可能很危险，它不是出路，而是一个蜜罐，为了抓捕你……这是怎么回事？
      </em>
     </strong>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   根据在底特律发布的联邦起诉书，美国国土安全部在法明顿山建立了一所假大学，瞄准那些想在未经适当授权的情况下留在美国的外国学生。
  </p>
  <p class="graf graf--p">
   密歇根州东区的美国检察官 Matthew Schneider 表示，有八人因涉嫌共谋签证欺诈和庇护外国人谋取利益，而在一起移民欺诈案中被捕并被起诉。
  </p>
  <p class="graf graf--p">
   其中六人在底特律地铁中被捕，一人在佛罗里达州，另一人在弗吉尼亚州。检察官说，这些学生就读于假的大学，意图根据学生签证计划（CPT）获得工作，该计划允许学生在美国工作。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    其中这一切都是ICE干的。整件事是一个陷阱
   </strong>
   。
  </p>
  <p class="graf graf--p">
   寻求参加假冒大学的学生签证已经过期，但只要他们继续接受教育，就可以留在美国工作。只要这些学生继续努力获得批准的学位，他们就可以留在该国。有限的H-1B签证供应意味着大多数移民除了上学之外只能离开该国。
  </p>
  <p class="graf graf--p">
   政府将此描述为“未经适当授权留在美国”。但大学根据此移民计划提供了适当的授权。ICE 决定建立一所假冒的大学作为针对外国学生围剿的蜜罐。
  </p>
  <p class="graf graf--p">
   底特律自由新闻社收到的电子邮件显示了 ICE 实施的这一丑陋的和有计划的行动，不仅骗取他们的钱，还剥夺了他们留在美国的机会。该机构甚至从国家认证机构获得了虚假认证，以便更好地欺骗不知情的移民。
  </p>
  <p class="graf graf--p">
   以下是 ICE 参与收集外国学生的欺骗行为的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.freep.com/story/news/2019/02/11/emails-reveal-how-university-lured-students-fake-farmington-university/2744103002/" href="https://www.freep.com/story/news/2019/02/11/emails-reveal-how-university-lured-students-fake-farmington-university/2744103002/" rel="noopener noreferrer" target="_blank">
    部分清单
   </a>
   。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    美国国土安全部的卧底调查人员将密歇根州的 Farmington 大学注册为一所使用假名的大学。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    应国土安全部的要求，一家国家认证机构将法明顿大学列为认可，以帮助欺骗未来的学生。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    该大学也被联邦调查员放在 ICE 的网站上作为他们批准的大学根据政府的SEVIS计划（学生和访问学者计划
   </em>
   ）
  </p>
  <p class="graf graf--p">
   所有这一切都促使未来的学生相信他们正在与一所真正的大学合作，提供允许他们在美国合法留学、学习和工作的课程。除了从600多名学生那里收取了100美元的申请费外，ICE还收取了无数的学费。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “学生们告诉我，大学向他们保证会上课，一切都会好，他们遵守移民法，”纽约市律师 Prashanthi Reddy 说。“学生们向他们支付了学费，当他们意识到这是假的也已经没用了，大多数学生已经没有钱再转学到另一所大学。
   </em>
   ”
  </p>
  <p class="graf graf--p">
   ICE 的假大学告诉学生他们通过申请继续教育以遵守移民法。这得到了该机构获得的虚假认证的支持。学生们认为他们正在采取下一步措施来延长留在美国的时间 — 所有这些都看起来完全合法 — 直到很明显，假冒的大学永远不会开启课程或分发学位。
  </p>
  <p class="graf graf--p">
   虽然可能有少数人关于他们的签证状态或收取费用方面撒了谎，但大多数被 ICE 蜜罐套住的人都遵守了法律。如果政府说只要你继续接受教育，你就可以留在国内工作，申请 ICE 伪造大学的学生完全符合要求。总之，有149名被骗的学生被捕，但其中只有8人实际上面临着起诉书中详述的指控。
  </p>
  <p class="graf graf--p">
   这是一种可怕的信任滥用 — — 而且是由国家认证机构协助和怂恿的。政府多次向试图遵守法律的人撒谎，拿走他们的钱，并且最终逮捕了他们。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    I
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.techdirt.com/articles/20190212/17035541582/ice-set-up-fake-college-to-bust-immigrants-trying-to-legally-stay-country-while-they-earned-degrees.shtml" href="https://www.techdirt.com/articles/20190212/17035541582/ice-set-up-fake-college-to-bust-immigrants-trying-to-legally-stay-country-while-they-earned-degrees.shtml" rel="noopener noreferrer" target="_blank">
    CE Set Up A Fake College To Bust Immigrants For Trying To Legally Stay In The Country While They Earned Degrees
   </a>
   . This is a horrific abuse of trust — aided and abetted by a national accreditation agency.
