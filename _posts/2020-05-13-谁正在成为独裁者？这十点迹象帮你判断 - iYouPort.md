   <a class="markup--anchor markup--p-anchor" data-href="https://www.tandfonline.com/doi/full/10.1080/13510347.2019.1582029" href="https://www.tandfonline.com/doi/full/10.1080/13510347.2019.1582029" rel="noopener noreferrer" target="_blank">
    正经历着
   </a>
   专制统治的崛起，民主理想和实践的衰落。专制统治 —— 也称为独裁统治 —— 是指一个领导人或政党行使完全权力来统治一个国家及其人民。
  </p>
  <p class="graf graf--p">
   自由之家说，2008年是民主达到顶峰的一年。那时，世界上完全“自由国家”的比例最高，达到
   <a class="markup--anchor markup--p-anchor" data-href="https://freedomhouse.org/sites/default/files/Feb2019_FH_FITW_2019_Report_ForWeb-compressed.pdf" href="https://freedomhouse.org/sites/default/files/Feb2019_FH_FITW_2019_Report_ForWeb-compressed.pdf" rel="noopener noreferrer" target="_blank">
    46.1％
   </a>
   。之后便开始滑坡。
  </p>
  <p class="graf graf--p">
   民主的定义
   <a class="markup--anchor markup--p-anchor" data-href="http://doi.org/10.1007/s12116-018-9268-z" href="https://doi.org/10.1007/s12116-018-9268-z" rel="noopener noreferrer" target="_blank">
    各不相同
   </a>
   。理论上，民主国家的所有公民都应该有能力在选举中投票，选举应该是自由和公正的。独立的媒体、言论自由、和集会自由、以及法治，应该是当代大多数民主观念中的特征。
  </p>
  <p class="graf graf--p">
   在民主集中度最高的地区，民主下降
   <a class="markup--anchor markup--p-anchor" data-href="https://www.v-dem.net/media/filer_public/99/de/99dedd73-f8bc-484c-8b91-44ba601b6e6b/v-dem_democracy_report_2019.pdf" href="https://www.v-dem.net/media/filer_public/99/de/99dedd73-f8bc-484c-8b91-44ba601b6e6b/v-dem_democracy_report_2019.pdf" rel="noopener noreferrer" target="_blank">
    最为明显
   </a>
   。其中包括欧洲、北美和拉丁美洲。
  </p>
  <p class="graf graf--p">
   一个例子：美国在2018年被评为“有缺陷的民主国家”，在167个国家和地区中从第21位下降到第25位。

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    10个特征清单
   </strong>
  </p>
  <p class="graf graf--p">
   当今具有独裁倾向的领导人不仅对利用蛮力上台的感兴趣。
  </p>
  <p class="graf graf--p">
   他们甚至更聪明，更有弹性，并且可以调整自己的方法以充分利用诸如现代技术和全球化经济之类的新发展。
  </p>
  <p class="graf graf--p">
   以下是准独裁者使用的一些最新策略：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、扩大行政权力
   </strong>
  </p>
  <p class="graf graf--p">
   当今威权主义的主要力量是增强权力、同时削弱有制衡能力的政府机构，例如议会和司法机构。
  </p>
  <p class="graf graf--p">
   关键是他们使用法律手段（而不是非法手段），最终为夺权提供民主合法性。
  </p>
  <p class="graf graf--p">
   极端的形式包括修宪以取消领导人任期限制，比如中国和其他国家；以及类似土耳其进行的那种渐进式宪法改革，以扩大总统的权力。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、压制异议和任何公民的努力
   </strong>
  </p>
  <p class="graf graf--p">
   对资金的限制和其他官僚主义限制使人民追究当权者的能力
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amnesty.org/download/Documents/ACT3096472019ENGLISH.PDF" href="https://www.amnesty.org/download/Documents/ACT3096472019ENGLISH.PDF" rel="noopener noreferrer" target="_blank">
    基本丧失
   </a>
   了。
  </p>
  <p class="graf graf--p">
   超过50个国家/地区通过了扼杀公民团体的法律。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    民主国家也在这一潮流中
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://news.un.org/en/story/2019/11/1051011" href="https://news.un.org/en/story/2019/11/1051011" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     跃升
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。限制公众抗议、拘留抗议者、过度使用武力破坏和平示威，都是经常使用的手段
   </strong>
   。
  </p>

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%88%91%e8%a2%ab%e8%ad%a6%e5%af%9f%e6%8a%98%e7%a3%a8%e8%87%b3%e7%b2%be%e7%a5%9e%e5%b4%a9%e6%ba%83%ef%bc%9a%e7%8e%b0%e5%9c%a8%e6%88%91%e5%bf%85%e9%a1%bb%e7%88%ac%e8%b5%b7%e6%9d%a5%e6%88%98%e6%96%97/" href="https://www.iyouport.org/%e6%88%91%e8%a2%ab%e8%ad%a6%e5%af%9f%e6%8a%98%e7%a3%a8%e8%87%b3%e7%b2%be%e7%a5%9e%e5%b4%a9%e6%ba%83%ef%bc%9a%e7%8e%b0%e5%9c%a8%e6%88%91%e5%bf%85%e9%a1%bb%e7%88%ac%e8%b5%b7%e6%9d%a5%e6%88%98%e6%96%97/" rel="noopener noreferrer" target="_blank">
     我被警察折磨至精神崩溃：现在我必须爬起来战斗
    </a>
    》
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     中国政府说它在新疆 “反恐”
    </strong>
    ，其结果发生了什么，大家都知道《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e5%9b%be%e8%a7%a3%e6%96%b0%e7%96%86%e5%86%8d%e6%95%99%e8%82%b2%e8%90%a5/" href="https://www.iyouport.org/%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e5%9b%be%e8%a7%a3%e6%96%b0%e7%96%86%e5%86%8d%e6%95%99%e8%82%b2%e8%90%a5/" rel="noopener noreferrer" target="_blank">
     开源调查图解新疆”再教育”营
    </a>
    》
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     所谓的有组织犯罪是当权者对民间反抗者采取压制政策的典型手段
    </strong>
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%b8%ae%e6%b4%be%e6%95%b0%e6%8d%ae%e5%ba%93%e6%98%af%e4%bb%80%e4%b9%88%ef%bc%9f%e4%b8%ba%e4%bb%80%e4%b9%88%e6%8d%ae%e6%ad%a4%e5%b0%b1%e5%8f%af%e4%bb%a5%e7%9b%b4%e6%8e%a5%e7%bb%99/" href="https://www.iyouport.org/%e5%b8%ae%e6%b4%be%e6%95%b0%e6%8d%ae%e5%ba%93%e6%98%af%e4%bb%80%e4%b9%88%ef%bc%9f%e4%b8%ba%e4%bb%80%e4%b9%88%e6%8d%ae%e6%ad%a4%e5%b0%b1%e5%8f%af%e4%bb%a5%e7%9b%b4%e6%8e%a5%e7%bb%99/" rel="noopener noreferrer" target="_blank">
     “帮派数据库”是什么？为什么据此就可以直接给无辜的市民定下重罪？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   其中之一就是著名的罗德里戈·杜特尔特（Rodrigo Duterte）的毒品战，在菲律宾已至少造成数千人死亡。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    别忘了，
    <strong class="markup--strong markup--li-strong">
     谁在资助这场宏大的谋杀？
    </strong>
    —— 美国和中国《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%E8%B5%B0%E8%BF%9B%E6%9A%97%E6%9D%80%E4%B9%8B%E5%9F%8E/" href="https://www.iyouport.org/%E8%B5%B0%E8%BF%9B%E6%9A%97%E6%9D%80%E4%B9%8B%E5%9F%8E/" rel="noopener noreferrer" target="_blank">
     走进暗杀之城
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   再比如，自2016年的未遂政变以来，直到2018年，土耳其一直处于紧急状态，这使得总统雷杰普·塔伊普·埃尔多安（Recep Tayyip Erdogan）可以将大批的学者、政府官员、媒体记者、和人权倡导者扔进监狱。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    9、扩展模型和影响力
   </strong>
  </p>
  <p class="graf graf--p">
   利用国际舞台和不断发展的经济实力，像中国这样的国家正在通过“一带一路”等融资计划来扩大其影响力，以在整个亚洲乃至欧洲建设基础设施。
  </p>
  <p class="graf graf--p">
   他们正在聘请专业顾问来为外国资本提供咨询和游说，以增强其实力。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%f0%9f%94%ad-%e4%b8%ad%e5%9b%bd%e7%89%88%e8%80%81%e5%a4%a7%e5%93%a5%e7%9a%84%e4%b8%96%e7%95%8c%e4%b9%8b%e6%97%85/" href="https://www.iyouport.org/%f0%9f%94%ad-%e4%b8%ad%e5%9b%bd%e7%89%88%e8%80%81%e5%a4%a7%e5%93%a5%e7%9a%84%e4%b8%96%e7%95%8c%e4%b9%8b%e6%97%85/" rel="noopener noreferrer" target="_blank">
     中国的监视技术如何沿着一带一路铺展
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    10、独裁者互相学习并分享经验
   </strong>
  </p>
  <p class="graf graf--p">
   是的，他们互相模仿、互相帮助。
  </p>
  <p class="graf graf--p">
   这种现象被学者称为 “
   <a class="markup--anchor markup--p-anchor" data-href="https://rsa.tandfonline.com/doi/full/10.1080/21599165.2017.1307826?scroll=top&amp;needAccess=true" href="https://rsa.tandfonline.com/doi/full/10.1080/21599165.2017.1307826?scroll=top&amp;needAccess=true" rel="noopener noreferrer" target="_blank">
    专制学习
   </a>
   ”，俄罗斯、中国、伊朗、委内瑞拉、白俄罗斯、叙利亚、和其他地方的国家主管部门正在开发和互相交换模型，以遏制社会运动和所谓的“颜色革命”的威胁。
  </p>
  <p class="graf graf--p">
   国际会议和政府间俱乐部可以提供交流平台。例如，柬埔寨首相洪森（Hun Sen）成功地召集了邻国政府，以利用地区组织东盟（ASEAN）
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2019/11/07/world/asia/cambodia-hun-sen-mu-sochua.html" href="https://www.nytimes.com/2019/11/07/world/asia/cambodia-hun-sen-mu-sochua.html" rel="noopener noreferrer" target="_blank">
    帮助压制
   </a>
   反对他的统治的任何行为。马来西亚政府官员最近阻止柬埔寨反对派成员通过马来西亚返回本国。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    方向未知
   </strong>
  </p>
  <p class="graf graf--p">
   一些专家声称，世界正处在一个“临界点”，民主的减少正在推动全球专制统治的崛起。
  </p>
  <p class="graf graf--p">
   当今的社会运动激发了公民社会的希望 —— 这是民主的关键要素 —— 尽管面临压力，但仍在努力应对潮流。
  </p>
  <p class="graf graf--p">
   尽管如此，即使是今天最成熟的民主国家也沦为了准独裁者策略的牺牲品，在全球范围内加强民主暂时不可能。⚪️
  </p>
  <p class="graf graf--p">
   作者 Shelley Inglis 在联合国工作过15年，就如何加强法治、人权和民主治理向政府和民主倡导者提供建议。现在是
   <a class="markup--anchor markup--p-anchor" data-href="https://udayton.edu/directory/artssciences/hrc/inglis-shelley.php" href="https://udayton.edu/directory/artssciences/hrc/inglis-shelley.php" rel="noopener noreferrer" target="_blank">
    国际法学者
   </a>
   。
  </p>
