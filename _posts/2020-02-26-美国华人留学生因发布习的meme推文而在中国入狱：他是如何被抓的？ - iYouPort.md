  <p class="graf graf--p">
   2019年，来自明尼苏达大学的20岁中国留学生在中国被捕，被审判并被判刑，原因是他在美国学习期间使用匿名推特账户发布的推文。
  </p>
  <p class="graf graf--p">
   这名学生 Luo Daiqing 在2019年暑假回到中国家中时被捕，并在2019年11月被秘密审判后被正式判刑。
  </p>
  <p class="graf graf--p">
   该新闻由 Bethany Allen-Ebrahimian 于2020年1月22日首次
   <a class="markup--anchor markup--p-anchor" data-href="https://www.axios.com/china-arrests-university-minnesota-twitter-e495cf47-d895-4014-9ac8-8dc76aa6004d.html" href="https://www.axios.com/china-arrests-university-minnesota-twitter-e495cf47-d895-4014-9ac8-8dc76aa6004d.html" rel="noopener noreferrer" target="_blank">
    报道
   </a>
   。
  </p>
  <p class="graf graf--p">
   这是中国政府首次以这种方式起诉在国外的中国公民，所有迹象均表明，这仅仅是一个开始。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12630 jetpack-lazy-image" data-lazy-sizes="(max-width: 802px) 100vw, 802px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1.png?resize=802%2C1160&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1.png?w=802&amp;ssl=1 802w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1.png?resize=207%2C300&amp;ssl=1 207w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1.png?resize=708%2C1024&amp;ssl=1 708w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1.png?resize=768%2C1111&amp;ssl=1 768w" data-recalc-dims="1" height="1160" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1.png?resize=802%2C1160&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="802"/>

    <img alt="" class="aligncenter size-full wp-image-12630" data-recalc-dims="1" height="1160" sizes="(max-width: 802px) 100vw, 802px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1.png?resize=802%2C1160&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1.png?w=802&amp;ssl=1 802w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1.png?resize=207%2C300&amp;ssl=1 207w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1.png?resize=708%2C1024&amp;ssl=1 708w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1.png?resize=768%2C1111&amp;ssl=1 768w" width="802"/>

  <p class="graf graf--p">
   这是个可怕的问题。值得研究。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    推特账户是匿名的，中国当局如何知道账户持有者的真实身份？
   </strong>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/Mq1919801" href="https://twitter.com/Mq1919801" rel="noopener noreferrer" target="_blank">
    这个账户在这里
   </a>
   ，现在已经被屏蔽了。
  </p>
  <p class="graf graf--p">
   最初的想法可能是中国政府通过法律要求 Twitter 披露 @ Mq1919801 这个帐户的信息。但是，几乎可以肯定不是这种情况。
  </p>
  <p class="graf graf--p">
   根据 Twitter 的
   <a class="markup--anchor markup--p-anchor" data-href="https://transparency.twitter.com/en/information-requests.html" href="https://transparency.twitter.com/en/information-requests.html" rel="noopener noreferrer" target="_blank">
    透明度报告
   </a>
   ，来自全球数十个国家的7300个此类请求中，中国在2019年上半年没有任何此类请求。
  </p>
  <p class="graf graf--p">
   2018年也没有来自中国的对 Twitter 的数据要求。该请求很可能是在2019年下半年提出的，并有可能在尚未发布的 Twitter 透明度报告中呈现，但是，考虑到拘留和逮捕的时间表 —— 这是极不可能的。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%a1%85%e8%b0%b7%e5%85%85%e6%96%a5%e7%9d%80%e4%b8%ad%e5%9b%bd%e5%92%8c%e6%b2%99%e7%89%b9%e7%9a%84%e8%b5%84%e9%87%91%ef%bc%8c%e6%9c%89%e8%b0%81%e5%85%b3%e5%bf%83%ef%bc%9f/" href="https://www.iyouport.org/%e7%a1%85%e8%b0%b7%e5%85%85%e6%96%a5%e7%9d%80%e4%b8%ad%e5%9b%bd%e5%92%8c%e6%b2%99%e7%89%b9%e7%9a%84%e8%b5%84%e9%87%91%ef%bc%8c%e6%9c%89%e8%b0%81%e5%85%b3%e5%bf%83%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     硅谷充斥着中国和沙特的资金，有谁关心？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   另一个推测是，中国间谍混入了推特公司成为员工，为中国政府搜集了有关异议人士的信息。
  </p>
  <p class="graf graf--p">
   尽管这是非常惊人的，但依旧是有可能的，甚至不是外国政府第一次这样做（沙特已经这样做过）。
  </p>
  <p class="graf graf--p">
   去年，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theverge.com/2019/11/6/20952335/twitter-employees-saudi-arabia-spying-arrests" href="https://www.theverge.com/2019/11/6/20952335/twitter-employees-saudi-arabia-spying-arrests" rel="noopener noreferrer" target="_blank">
    WaPo
   </a>
   报道说，司法部已指控两名前 Twitter 员工帮助沙特政府窥视异议人士的 Twitter 帐户信息。
  </p>
  <p class="graf graf--p">
   就我个人而言，我认为这种可能性不大，因为中国不太可能为了一些有趣的 memes 投入如此高价值的资产。
  </p>
  <p class="graf graf--p">
   可以考虑的另一种推测是，使用中国国家控制的应用程序将目标手机中的图像与 Twitter 上发布的图像进行了匹配。
  </p>
  <p class="graf graf--p">
   中国许多流行的应用程序（例如微信）已经成为审查传播的重点包括memes。除了图片之外，该应用程序还可以筛选某些文本中的关键字，然后对其进行审查和报告，以供当局将来采取措施。
  </p>
  <p class="graf graf--p">
   众所周知，中国的应用会执行这种自动的图片识别、筛选和审查操作，让人们感觉已经发送成功，但对方无法收到。
  </p>
  <p class="graf graf--p">
   如果这样的应用程序可以访问包含 Luo 发布的模因的手机相册，则中国就有可能要求腾讯公司检查目标相册中与 Luo 的 Twitter 发布的图像相匹配的图像。
  </p>
  <p class="graf graf--p">
   尽管这些都是 Luo 被中国政府抓获事件在技术上可行的方式，但鉴于中国政府仍在使用的陈旧策略，还有一些话要说。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    中国仍然依靠传统的、古老的监视技术来识别持不同政见者
   </strong>
  </p>
  <p class="graf graf--p">
   不过，最有可能发生的情况是，Luo 在他自己的学校被他的身边人报告给了中国政府。在美国，受到同龄人监视的海外华人学生现在已经是一个简单的事实。
  </p>
  <p class="graf graf--p">
   2017年，美国国家学者协会呼吁一百多所大学排除孔子学院，这是中国政府资助的所谓 “文化和语言” 教育中心，被发现对海外华人学生具有影响力。参议员 Mark Warner 评论说，孔子学院 “只是中国间谍监视中国学生并追究他们的工具”。
  </p>
  <p class="graf graf--p">
   明尼苏达大学于2019年春季宣布，其校园中的孔子学院将于学期末关闭。Luo 在推特上发布习近平的 meme 时，孔子学院在明尼苏达大学仍在运作。
  </p>
  <p class="graf graf--p">
   被发现是这种间谍活动工具的另一个组织是中国学生学者协会（CSSA），该协会仍在明尼苏达大学校园内活跃。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    中国对海外华人学生的监视不仅限于​​美国
   </strong>
  </p>
  <p class="graf graf--p">
   中国在与世界各地的国际学生保持联系方面有着悠久的历史。近年来，监视和后果已大大增加。
  </p>
  <p class="graf graf--p">
   在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.smh.com.au/politics/federal/this-student-attended-a-protest-at-an-australian-uni-days-later-chinese-officials-visited-his-family-20190807-p52eqb.html" href="https://www.smh.com.au/politics/federal/this-student-attended-a-protest-at-an-australian-uni-days-later-chinese-officials-visited-his-family-20190807-p52eqb.html" rel="noopener noreferrer" target="_blank">
    澳大利亚
   </a>
   ，一名中国学生说，她参加亲民主抗议活动后，她在中国的家人遭到当局问话。在
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/StewartMcDonald/status/1220612090769166336" href="https://twitter.com/StewartMcDonald/status/1220612090769166336" rel="noopener noreferrer" target="_blank">
    苏格兰
   </a>
   ，一名中国学生也报告了同样的事。
  </p>
  <p class="graf graf--p">
   Nathan Law（罗冠聪）是香港前立法会议员，后来因参加抗议活动而被入狱，现在是耶鲁大学亚洲研究专业的研究生。他报告听说其他中国学生被告知要远离他。
   <a class="markup--anchor markup--p-anchor" data-href="https://apnews.com/7b0f328f266f46c1a498a16a5ad03fbc" href="https://apnews.com/7b0f328f266f46c1a498a16a5ad03fbc" rel="noopener noreferrer" target="_blank">
    他告诉
   </a>
   美联社：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “尽管许多中国学生就在这里学习，但有时他们依旧会受到监视。他们在这个国家也没有真正的思想和表达自由。”
  </p>
  <p class="graf graf--p">
   人权观察（HRW）也发布了有关中国政府对学术自由的威胁的报告。根据2019年HRW
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hrw.org/news/2019/03/21/china-government-threats-academic-freedom-abroad" href="https://www.hrw.org/news/2019/03/21/china-government-threats-academic-freedom-abroad" rel="noopener noreferrer" target="_blank">
    报告
   </a>
   ，该报告采访了一百多名学者：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “学者告诉《人权观察》，中国学生描述了当局对他们在中国的家人的威胁，只因为他们在教室里所说的话。来自中国的学者详细介绍了中国官员在国外直接威胁要避免在课堂或其他演讲中批评中国政府。其他人则说，来自中国的学生在教室里保持沉默，担心其他中国学生会监视他们说的话，并向中国当局报告。”
  </p>
  <p class="graf graf--p">
   Luo 应该在1月初被释放，但是没有媒体能够证实他目前的下落或健康状况。在这一消息传遍全世界的几天之内，中国政府就开始清理互联网上的相关法院文件。
  </p>
  <p class="graf graf--p">
   中国政府在世界范围内针对言论自由采取的升级行动所带来的后果，应该不仅困扰着中国公民，它足以令所有人担忧。⚪️
  </p>

   IYP 认为，
   <strong class="markup--strong markup--p-strong">
    关键问题在于，这件事不应该被强调为 “海外华人留学生”，而是所有推特华人用户，都应该警惕此案衍生的诸多可能性。可能性不高但在技术上可行的推测也应该被考虑进去
   </strong>
   。
  </p>
