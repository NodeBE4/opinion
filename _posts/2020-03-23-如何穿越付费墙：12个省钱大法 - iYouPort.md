   付费墙是有多讨厌，如今已经尽人皆知，《

    地狱般的付费墙：免费、开放和民主化的 Web 时代已成往事了吗？
   </a>
   》。
  </p>
  <p class="graf graf--p">
   但是没办法，内容提供者需要吃饭，需要活下去才能继续为读者服务 —— 像 IYP 这样自掏腰包的穷光蛋已经快撑不住了。
  </p>
  <p class="graf graf--p">
   但是IYP坚持不搞付费墙，因为我们自己就经常被其他资源的付费墙惹恼，己所不欲。
  </p>
  <p class="graf graf--p">
   尤其是那些大牌企业媒体搞的付费墙。它们原本就有着流油的资金背景和广告收入，它们还想让你掏腰包，对于老百姓来说，这不公平。
  </p>
  <p class="graf graf--p">
   绕过付费墙是可能的吗？是的。
  </p>
  <p class="graf graf--p">
   某些时候，甚至一个简单的浏览器技巧就足以帮您绕过付费墙以阅读文章、观看视频等。
  </p>
  <p class="graf graf--p">
   付费墙有两种，您最有可能解锁的是“软”的那种。软付费墙是一种让您能看到一小部分内容的拦截方法，而硬的付费墙则是，如果不付钱，你一个字都看不到。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    请注意
   </strong>
   ：我们不建议您使用付费墙拦截器，因为它们会使一家本来有潜在收入来源的公司饿死。与广告拦截器类似，您应该明智地使用它们，并考​​虑它们可能会对您喜欢的网站产生的影响。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     12种解锁付费墙的方法
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   您可以尝试多种方法来突破收费壁垒。有些方法对于有些墙来说注定无法使用，但肯定有一个或两个选项适用于您尝试访问的网站。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、如果付费墙的Firefox扩展
   </strong>
  </p>

   使用

    此 Firefox 扩展程序
   </a>
   绕过付费专区。它适用于彭博社、丹佛邮报、巴尔的摩太阳报、先驱报以及其他许多网站。
  </p>
  <p class="graf graf--p">
   只需使用该页面顶部的下载链接即可获取文件，然后将其拖到 Firefox 窗口中进行安装。
  </p>

  <p class="graf graf--p">
   从那里开始，您所要做的就是访问该站点以查看该扩展是否删除了付费墙。您也可以进入设置以在任何受支持的站点上禁用它（允许特定的付费墙）。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、寻找其他地方的文章
   </strong>
  </p>
  <p class="graf graf--p">
   复制文章标题，并将其粘贴到搜索引擎中以查找被转载的内容。
  </p>
  <p class="graf graf--p">
   这是绕过付费墙的最简单方法之一，因为很多时候，基于订阅的新闻机构的热门文章都会被完全免费复制和发布到其他地方。
  </p>

  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、试用 Unpaywall Chrome扩展程序
   </strong>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://chrome.google.com/webstore/detail/unpaywall/iplffkdpngmdjhlpjmppncnlhomiipha?hl=en" href="https://chrome.google.com/webstore/detail/unpaywall/iplffkdpngmdjhlpjmppncnlhomiipha?hl=en" rel="noopener noreferrer" target="_blank">
    Unpaywall
   </a>
   是 Chrome 浏览器的扩展程序，可解除对学术文章的付费墙的限制。它会在互联网上搜索该产品的免费PDF版本，然后为您提供查找免费版本的链接。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、重置您的浏览器 Cookie
   </strong>
  </p>
  <p class="graf graf--p">
   删除您的浏览器 cookie 或使用 “隐身模式” 绕过付费墙。
  </p>
  <p class="graf graf--p">
   如果该网站允许您查看少量文章，然后使用付费墙阻止访问，那么显然，他们是通过存储 cookie 来跟踪您浏览了多少页面的。所以，对症下药。
  </p>

    Outline
   </a>
   中输入目标页面的 URL。Outline 是用于避免干扰的，但是由于其工作原理，通常有助于避免支付墙。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-10391 jetpack-lazy-image" data-lazy-sizes="(max-width: 810px) 100vw, 810px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-3.png?resize=810%2C714&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-3.png?w=810&amp;ssl=1 810w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-3.png?resize=300%2C264&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-3.png?resize=768%2C677&amp;ssl=1 768w" data-recalc-dims="1" height="714" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-3.png?resize=810%2C714&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="810"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10391" data-recalc-dims="1" height="714" sizes="(max-width: 810px) 100vw, 810px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-3.png?resize=810%2C714&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-3.png?w=810&amp;ssl=1 810w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-3.png?resize=300%2C264&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-3.png?resize=768%2C677&amp;ssl=1 768w" width="810"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    6、手动删除付费墙
   </strong>
  </p>
  <p class="graf graf--p">
   如果可能的话，请删除付费墙弹出窗口。
  </p>
  <p class="graf graf--p">
   一些网站使用超级基本的付费墙系统，其中唯一阻止您查看页面的是弹出窗口。虽然没有退出按钮，但您可以查看页面源码并将其手动删除，以显示其背后的内容。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-10392 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-2.png?resize=1100%2C940&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-2.png?w=1278&amp;ssl=1 1278w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-2.png?resize=300%2C256&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-2.png?resize=768%2C656&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-2.png?resize=1024%2C875&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-2.png?resize=1100%2C940&amp;ssl=1 1100w" data-recalc-dims="1" height="940" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-2.png?resize=1100%2C940&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10392" data-recalc-dims="1" height="940" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-2.png?resize=1100%2C940&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-2.png?w=1278&amp;ssl=1 1278w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-2.png?resize=300%2C256&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-2.png?resize=768%2C656&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-2.png?resize=1024%2C875&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-2.png?resize=1100%2C940&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   例如，在 Chrome 中，右键单击付费墙消息，然后选择 “Inspect”，使用Chrome工具左上方的选项来查找付费专区的源代码。
  </p>
  <p class="graf graf--p">
   找到它后，右键单击该数据，然后选择 “
   <strong class="markup--strong markup--p-strong">
    Delete element
   </strong>
   ”。您可能需要删除多个东西，直到找到正确的方式来摆脱整个支付壁垒。
  </p>
  <p class="graf graf--p">
   如果付费墙使用 JavaScript，那么可能会起作用的类似选项是使用比如
   <a class="markup--anchor markup--p-anchor" data-href="https://chrome.google.com/webstore/detail/quick-javascript-switcher/geddoclleiomckbhadiaipdggiiccfje?hl=en" href="https://chrome.google.com/webstore/detail/quick-javascript-switcher/geddoclleiomckbhadiaipdggiiccfje?hl=en" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Quick JavaScript Switcher for Chrome
    </strong>
   </a>
   的工具阻止脚本。
  </p>
  <p class="graf graf--p">
   如果您不使用Chrome ，或这些方法不起作用，请尝试使用可以删除脚本的Web代理，例如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hidemyass.com/en-us/proxy" href="https://www.hidemyass.com/en-us/proxy" rel="noreferrer noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     HMA
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    7、中断页面完全加载
   </strong>
  </p>
  <p class="graf graf--p">
   在付费墙弹出之前快速暂停页面。如果运气不好无法删除或阻止 JavaScript，则可以中断页面的完全加载，以阻止显示付费专区弹出窗口。
  </p>
  <p class="graf graf--p">
   但是你必须足够快；如果页面加载速度太快或付费墙在文章出现之前就启动了，那么该功能就无法使用。要尝试此操作，只需刷新页面并在加载内容后立即按 Esc 键几次。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    8、挖掘存档站点
   </strong>
  </p>
  <p class="graf graf--p">
   诸如 Wayback Machine 和 Archive.is 之类的存档站点会保存网页以用于存档。您可以利用这些方法查找由于付费专区封锁而无法访问的文章。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10393 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-3.png?resize=1100%2C330&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-3.png?w=1140&amp;ssl=1 1140w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-3.png?resize=300%2C90&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-3.png?resize=768%2C230&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-3.png?resize=1024%2C307&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-3.png?resize=1100%2C330&amp;ssl=1 1100w" data-recalc-dims="1" height="330" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-3.png?resize=1100%2C330&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10393" data-recalc-dims="1" height="330" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-3.png?resize=1100%2C330&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-3.png?w=1140&amp;ssl=1 1140w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-3.png?resize=300%2C90&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-3.png?resize=768%2C230&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-3.png?resize=1024%2C307&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/7-3.png?resize=1100%2C330&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    9、使用“稍后阅读”工具
   </strong>
  </p>
  <p class="graf graf--p">
   将文章发送到书签或类似
   <a class="markup--anchor markup--p-anchor" data-href="https://getpocket.com/" href="https://getpocket.com/" rel="noopener noreferrer" target="_blank">
    Pocket
   </a>
   的稍后阅读服务，以绕过文章限制。
  </p>
  <p class="graf graf--p">
   但只有在您访问的前几次拥有免费访问该网站的权限时，此方法才有效。
  </p>
  <p class="graf graf--p">
   其原理与上面的 cookie 方法类似，因为该页面的内容将发送到其他地方并且不会加载到您的计算机上。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    10、将页面转换为PDF
   </strong>
  </p>
  <p class="graf graf--p">
   有很多把页面转换为PDF的转换器，其中一个就是
   <a class="markup--anchor markup--p-anchor" data-href="https://webpagetopdf.com/" href="https://webpagetopdf.com/" rel="noopener noreferrer" target="_blank">
    Webpagetopdf.com
   </a>
   。
  </p>
  <p class="graf graf--p">
   只需将付费墙页面的URL放入文本框中，进行转换，然后下载PDF，以使您可以脱机访问该文章，并且始终可以免费访问。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10394 jetpack-lazy-image" data-lazy-sizes="(max-width: 1086px) 100vw, 1086px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-3.png?resize=1086%2C512&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-3.png?w=1086&amp;ssl=1 1086w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-3.png?resize=300%2C141&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-3.png?resize=768%2C362&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-3.png?resize=1024%2C483&amp;ssl=1 1024w" data-recalc-dims="1" height="512" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-3.png?resize=1086%2C512&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1086"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10394" data-recalc-dims="1" height="512" sizes="(max-width: 1086px) 100vw, 1086px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-3.png?resize=1086%2C512&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-3.png?w=1086&amp;ssl=1 1086w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-3.png?resize=300%2C141&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-3.png?resize=768%2C362&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/8-3.png?resize=1024%2C483&amp;ssl=1 1024w" width="1086"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    11、查找登录详细信息
   </strong>
  </p>
  <p class="graf graf--p">
   如果您要处理的是 “硬” 付款墙，并且需要用户帐户，请尝试使用共享登录服务。
  </p>
  <p class="graf graf--p">
   诸如
   <a class="markup--anchor markup--p-anchor" data-href="http://bugmenot.com/" href="http://bugmenot.com/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     BugMeNot
    </strong>
   </a>
   之类的网站可能具有登录详细信息，您可以用来登录该网站，并假装是真实用户来绕过付费墙。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10395 jetpack-lazy-image" data-lazy-sizes="(max-width: 1046px) 100vw, 1046px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/9-3.png?resize=1046%2C512&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/9-3.png?w=1046&amp;ssl=1 1046w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/9-3.png?resize=300%2C147&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/9-3.png?resize=768%2C376&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/9-3.png?resize=1024%2C501&amp;ssl=1 1024w" data-recalc-dims="1" height="512" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/9-3.png?resize=1046%2C512&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1046"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10395" data-recalc-dims="1" height="512" sizes="(max-width: 1046px) 100vw, 1046px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/9-3.png?resize=1046%2C512&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/9-3.png?w=1046&amp;ssl=1 1046w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/9-3.png?resize=300%2C147&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/9-3.png?resize=768%2C376&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/9-3.png?resize=1024%2C501&amp;ssl=1 1024w" width="1046"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    12、注册免费试用
   </strong>
  </p>
  <p class="graf graf--p">
   如果可以申请试用，以便您可以绕开付费墙，那么就先使用它，然后在付费前取消试用。这绝对是一件非常容易的事。
  </p>
  <p class="graf graf--p">
   但是，如果您这样做，
   <strong class="markup--strong markup--p-strong">
    请记住务必取消试用
   </strong>
   ，否则必须付费！您可以使用诸如 “
   <a class="markup--anchor markup--p-anchor" data-href="https://privacy.com/" href="https://privacy.com/" rel="noreferrer noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Privacy
    </strong>
   </a>
   ” 之类的虚拟支付服务来确保您用于支付的卡仅涵盖试用，不会向您收取该服务的费用。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     最后
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   就是这样！
  </p>
  <p class="graf graf--p">
   以上方法会针对不同的墙有效，而不是所有方法对每一面墙都有效。所以，本文是一个手册，您可以在不同情况下尝试其中不同的方法。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    ⚠️ 请注意：不要太狠！
   </strong>
   有些内容提供者 —— 尤其是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ae%a9%e5%ae%83%e6%b0%91%e4%b8%bb%ef%bc%9a%e6%9b%bf%e4%bb%a3%e5%aa%92%e4%bd%93%e4%bd%9c%e4%b8%ba%e6%8a%97%e4%ba%89%e4%b9%8b%e6%ba%90%ef%bc%88%e4%b8%8b%ef%bc%89/" href="https://www.iyouport.org/%e8%ae%a9%e5%ae%83%e6%b0%91%e4%b8%bb%ef%bc%9a%e6%9b%bf%e4%bb%a3%e5%aa%92%e4%bd%93%e4%bd%9c%e4%b8%ba%e6%8a%97%e4%ba%89%e4%b9%8b%e6%ba%90%ef%bc%88%e4%b8%8b%ef%bc%89/" rel="noopener noreferrer" target="_blank">
