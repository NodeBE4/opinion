  <p class="graf graf--p">
   当违宪轻而易举时，宪法已经形同虚设。
  </p>
  <p class="graf graf--p">
   美国海关及边境保卫局（CBP）在
   <strong class="markup--strong markup--p-strong">
    监视记者、活动家和律师
   </strong>
   方面有一个令人惊讶的合作伙伴：墨西哥政府。
  </p>
  <p class="graf graf--p">
   墨西哥政府不会支付那堵墙的费用，但它显然愿意以其他方式提供帮助。
  </p>
  <p class="graf graf--p">
   对美国边境机构针对记者、律师、移民倡导者和活动家进行的监视和跟踪的审查证实，墨西哥政府在这场惊人的监视计划中
   <a class="markup--anchor markup--p-anchor" data-href="https://www.techdirt.com/articles/20190306/19554641751/cbp-put-bunch-journalists-immigration-lawyers-activists-secret-watchlist.shtml" href="https://www.techdirt.com/articles/20190306/19554641751/cbp-put-bunch-journalists-immigration-lawyers-activists-secret-watchlist.shtml" rel="noopener noreferrer" target="_blank">
    发挥了重要作用
   </a>
   。
  </p>
  <p class="graf graf--p">
   在2019年5月9日的一封信中，美国海关与边境保护局（CBP）外勤业务执行主任 Randy Howe 指出。“……CBP与墨西哥政府合作[以]应对发展中的威胁……” 这里指的所谓威胁是中美洲移民大篷车。
  </p>
  <p class="graf graf--p">
   CBP的信中说，“墨西哥联邦警察认定一些
   <strong class="markup--strong markup--p-strong">
    记者和摄影师
   </strong>
   可能协助移民非法越境和/或参与暴力入侵事件”。
  </p>
  <p class="graf graf--p">
   墨西哥政府官员最初表示政府没有参与这一有争议的监视行动。
  </p>
  <p class="graf graf--p">
   新闻机构和权利团体联盟在一封信中声明，要求了解为什么
   <strong class="markup--strong markup--p-strong">
    CBP和国土安全部在长期违反第四和第五修正案的情况下继续增加了对第一修正案的违反
   </strong>
   。
  </p>
  <p class="graf graf--p">
   以下是墨西哥政府的原始声明，现在它突然变得非常糟糕了：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “墨西哥政府不赞成对国内或国外任何人进行非法间谍活动。墨西哥政府不对任何人进行非法监视”…
   </em>
   …
  </p>
  <p class="graf graf--p">
   我想这一切都取决于你如何定义“非法”、或“监视”。
  </p>
  <p class="graf graf--p">
   联邦警察局负责将人员添加到CBP的监视列表中，也许这种监视在墨西哥并非非法；但是它绝对是在美国非法的。
  </p>
  <p class="graf graf--p">
   如果这不是边境另一边的法律问题，那么CBP和国土安全部基本上是
   <strong class="markup--strong markup--p-strong">
    利用
   </strong>
   外国政府进行监视，碍于法律他们不被允许在围栏的这一侧进行监视。
  </p>
  <p class="graf graf--p">
   CBP和国土安全部似乎并不关心他们自己可能已经违宪的行为。
  </p>
  <p class="graf graf--p">
   CBP 发来的信中说，该机构只是在围捕那些试图帮助移民越境的人；但不知何故，
   <strong class="markup--strong markup--p-strong">
    被瞄准目标居然包括试图帮助难民通过庇护程序的移民律师和记者。
   </strong>
  </p>
  <p class="graf graf--p">
   美国海关与边境保护局确实承认自己会定期违反宪法规定的权利，但它声称这些违规行为只是美国人生活道路上的一些“小颠簸”。
  </p>
  <p class="graf graf--p">
   他们称：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “有时候，CBP 可能会给守法的人带来一些不便，这是为了发现、阻止和减轻对我们家园的威胁。”
   </em>
  </p>
  <p class="graf graf--p">
   好吧，美国海关与边境保护局不进行非法监视（除了已经发生的非法监视之外）、并且没有针对受第一修正案保护的活动（边境附近发生的活动除外），只是为了国家安全，它忽略了美国宪法保障的权利。
  </p>
  <p class="graf graf--p">
   它正在通过外国政府的帮助完成所有这些工作 —— 该政府在首次被问及时否认与此监视有任何关系。
  </p>

    Mexican Government Pitched In To Help The CBP Spy On Journalists, Activists, And Lawyers
