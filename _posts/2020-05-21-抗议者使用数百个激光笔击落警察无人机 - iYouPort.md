  <p class="graf graf--p">
   去年一整年有几十个国家/地区实践了激烈的抗议运动，香港是其中之一，您可以在这里看到介绍《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%99%ba%e8%83%bd%e9%95%87%e5%8e%8b%e6%97%b6%e4%bb%a3%ef%bc%9a%e6%b0%91%e4%bc%97%e6%8a%97%e8%ae%ae%e6%ad%a4%e8%b5%b7%e5%bd%bc%e4%bc%8f%ef%bc%8c%e4%bd%86%e6%95%88%e6%9e%9c%e5%a6%82%e4%bd%95/" href="https://www.iyouport.org/%e6%99%ba%e8%83%bd%e9%95%87%e5%8e%8b%e6%97%b6%e4%bb%a3%ef%bc%9a%e6%b0%91%e4%bc%97%e6%8a%97%e8%ae%ae%e6%ad%a4%e8%b5%b7%e5%bd%bc%e4%bc%8f%ef%bc%8c%e4%bd%86%e6%95%88%e6%9e%9c%e5%a6%82%e4%bd%95/" rel="noopener noreferrer" target="_blank">
    ”智能镇压”时代：民众抗议此起彼伏，效果如何？
   </a>
   》。
   <strong class="markup--strong markup--p-strong">
    其中智利的抗议活动是突出的案例
   </strong>
   。
  </p>
  <p class="graf graf--p">
   智利人民针对塞瓦斯蒂安·皮涅拉总统的新自由主义政府的大规模抗议活动持续了数月，示威者与国家军事维稳部队之间的冲突一直在继续。
  </p>
  <p class="graf graf--p">
   抗议活动于10月14日开始，是对公共交通成本上涨的回应，随着社会运动的继续，在学生、工会和智利社会各界的广泛关注下，抗议活动迅速激化，人们的愤怒集中于该地区的高度不平等、不断上升的生活成本、以及从1973–1990年 Augusto Pinochet 独裁统治被继承的宪法。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    随着抗议活动的激烈进行，对示威者的镇压变得越来越残酷和高科技。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    然而，智利的抗议者以机智、毅力和同样高科技手段来抵制警察的袭击，从而做出了聪明的回应
   </strong>
   。
  </p>
  <p class="graf graf--p">
   一份录像在社交媒体上迅速传播开来，其中表明警用无人机（UAS）被阻止 —— 因为抗议者使用普通的绿色激光笔瞄准执法无人机。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种方法在绝大多数情况下都可以使用
   </strong>
   。
  </p>
  <p class="graf graf--p">
   该视频是 NextGov 报道的，最初
   <a class="markup--anchor markup--p-anchor" data-href="https://www.reddit.com/r/nextfuckinglevel/comments/dvlvwr/chilean_protesters_take_down_a_police_drone_with/?sort=confidence" href="https://www.reddit.com/r/nextfuckinglevel/comments/dvlvwr/chilean_protesters_take_down_a_police_drone_with/?sort=confidence" rel="noopener noreferrer" target="_blank">
    在 Reddit 上
   </a>
   露面；视频中显示了在激光的追踪下，无人机在示威者头顶海面上盘旋的景况。
  </p>
  <p class="graf graf--p">
   很快，出现了很多其他的激光笔，都瞄准那架无人机，当无人机试图逃脱时，这些激光笔一直跟踪聚焦它，迫使无人机跌跌撞撞地降落。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/3PKJYGoUjJo" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   在示威活动中使用激光的技巧最早不是出现在智利。在香港的示威活动中，这种技巧也无处不在，抗议者还使用激光使配备面部识别技术的监控摄像头失效。
  </p>
  <p class="graf graf--p">
   正如 Defense Maven
   <a class="markup--anchor markup--p-anchor" data-href="https://defensemaven.io/warriormaven/land/army-stryker-fires-lasers-launches-hunter-killer-drones-changes-tactics-Jf0UPr3SRkuV_G5nmPPjew/" href="https://defensemaven.io/warriormaven/land/army-stryker-fires-lasers-launches-hunter-killer-drones-changes-tactics-Jf0UPr3SRkuV_G5nmPPjew/" rel="noopener noreferrer" target="_blank">
    报道
   </a>
   的那样，美国陆军 “史崔克” 战车也配备了激光武器，可用于阻拦和击落附近的无人机。
  </p>
  <p class="graf graf--p">
   但是，智利抗议活动中使用的
   <strong class="markup--strong markup--p-strong">
    消费级激
   </strong>
   光很难像军用激光大炮那样穿透装甲或引燃人体；取而代之的是，这些激光会掩盖操作员的摄像头视线、或者禁用无人机的高度传感器，从而触发无人机的紧急自主着陆功能。
  </p>
  <p class="graf graf--p">
   虽然激光在这种基层群众起义中被证明是有效的，但一些专家表示，激光不太可能对无人机造成重大损害。
  </p>
  <p class="graf graf--p">
   Citadel 防御公司负责人 Christopher Williams 表示：该公司希望为沿美国南部边界的美国海关和边境保护局部署基于反无人机的系统。
  </p>
  <p class="graf graf--p">
   他说，“与无人机相比，使用这些类型的激光对人类更具危害性和破坏性。但还有许多其他破坏性较小的技术可提供出色的、更可扩展的、更可靠的和更具成本效益的对策。”
  </p>
  <p class="graf graf--p">
   然而，就目前而言，专为办公室演讲而非军事战术目的而设计的激光笔似乎对那些上街要求更大的社会和政治权利的抗议者来说，还算不错。
  </p>
  <p class="graf graf--p">
   您学到了吗？这很重要。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    政府使用无人机攻击和监视抗议者的现象正在全球上演
   </strong>
   。见：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%b6%88%e8%b4%b9%e6%97%a0%e4%ba%ba%e6%9c%ba%e5%af%b9%e6%8a%97%e7%a4%ba%e5%a8%81%e8%80%85%e9%a2%84%e7%a4%ba%e4%ba%86%e4%b8%80%e4%b8%aa%e5%8d%b1%e9%99%a9%e7%9a%84%e3%80%81%e6%97%a0%e6%b3%95%e6%97%a0/" href="https://www.iyouport.org/%e6%b6%88%e8%b4%b9%e6%97%a0%e4%ba%ba%e6%9c%ba%e5%af%b9%e6%8a%97%e7%a4%ba%e5%a8%81%e8%80%85%e9%a2%84%e7%a4%ba%e4%ba%86%e4%b8%80%e4%b8%aa%e5%8d%b1%e9%99%a9%e7%9a%84%e3%80%81%e6%97%a0%e6%b3%95%e6%97%a0/" rel="noopener noreferrer" target="_blank">
     消费无人机对抗示威者预示了一个危险的、无法无天的冲突时代

   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%a9%ba%e4%b8%ad%e7%9b%91%e8%a7%86%e5%a6%82%e4%bd%95%e8%bf%bd%e8%b8%aa%e6%8a%97%e8%ae%ae%e8%80%85%ef%bc%9f/" href="https://www.iyouport.org/%e7%a9%ba%e4%b8%ad%e7%9b%91%e8%a7%86%e5%a6%82%e4%bd%95%e8%bf%bd%e8%b8%aa%e6%8a%97%e8%ae%ae%e8%80%85%ef%bc%9f/" rel="noopener noreferrer" target="_blank">

    </a>
    》
   </li>

  <p class="graf graf--p">
   如上所述，技术型维稳是如今当权者的最爱，但活动家们依旧有办法利用技术对战技术。这就是21世纪新型抗议运动的特征。
  </p>
