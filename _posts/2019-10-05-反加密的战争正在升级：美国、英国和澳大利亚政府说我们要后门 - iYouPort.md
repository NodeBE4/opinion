 <div class="entry-content">
  <p>
   反加密的战争正在升级。
  </p>
  <p class="graf graf--p">
   美国、英国和澳大利亚的高级执法官员告诉 Facebook，⚠️
   <strong class="markup--strong markup--p-strong">
    他们想要后门以访问其所有平台上发送的所有加密消息
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这些政府在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.buzzfeednews.com/article/ryanmac/bill-barr-facebook-letter-halt-encryption" href="https://www.buzzfeednews.com/article/ryanmac/bill-barr-facebook-letter-halt-encryption" rel="noopener noreferrer" target="_blank">
    公开信
   </a>
   中呼吁 Mark Zuckerberg 停止 Facebook 在公司所有消息产品上引入端到端加密的计划，并要求该公司承诺它将 “允许执法部门能够以可读和可用的格式合法访问内容。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这是破坏数十亿人使用的通信工具的安全性和隐私性的惊人尝试
   </strong>
   。 EFF表示：Facebook不应该遵守这项命令（Facebook 会同意吗？
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" rel="noopener noreferrer" target="_blank">
    读过斯诺登文件
   </a>
   的人应该表示怀疑）。
  </p>
  <p class="graf graf--p">
   这封公开信与美国和英国之间新协议的签署相一致，
   <strong class="markup--strong markup--p-strong">
    该协议允许一个司法管辖区的执法部门更轻松地获取另一司法管辖区中存储的电子数据
   </strong>
   。也就是跨国维稳协议。
  </p>
  <p class="graf graf--p">
   并且，写给 Facebook 的这封信还更进了一步：⚠️
   <strong class="markup--strong markup--p-strong">
    英美澳这三个国家的执法部门和国安机构直接要求访问的内容遍及每台数字设备上的每一次对话
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这封信侧重于所谓的执法方面的挑战，但是，它忽略了引入加密后门可能造成的严重风险。
  </p>
  <p class="graf graf--p">
   许多人，包括新闻工作者、人权活动家、敏感人士、以及家暴受害者，都在使用加密技术来确保物理世界和在线世界的安全。加密对于防止犯罪分子、政府、和寡头公司的监视是至关重要的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    加密是我们可以抵御全面统治的一个希望，一个结合勇气、洞见和团结的希望  — — 我们可以摆脱无处不在的入侵，从中创造一个新的国度，将那些物质现实的控制者隔离在外，他们一直在煞费苦心地追踪我们所有人
   </strong>
   。《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%88%9b%e5%bb%ba%e5%b1%9e%e4%ba%8e%e6%88%91%e4%bb%ac%e8%87%aa%e5%b7%b1%e7%9a%84%e5%9b%bd%e5%9c%9f%ef%bc%9b%e6%8e%a8%e8%8d%90%e9%9d%9e%e6%9a%b4%e5%8a%9b%e7%9b%b4%e6%8e%a5%e8%a1%8c%e5%8a%a8%e7%9a%84/" href="https://www.iyouport.org/%e5%88%9b%e5%bb%ba%e5%b1%9e%e4%ba%8e%e6%88%91%e4%bb%ac%e8%87%aa%e5%b7%b1%e7%9a%84%e5%9b%bd%e5%9c%9f%ef%bc%9b%e6%8e%a8%e8%8d%90%e9%9d%9e%e6%9a%b4%e5%8a%9b%e7%9b%b4%e6%8e%a5%e8%a1%8c%e5%8a%a8%e7%9a%84/" rel="noopener noreferrer" target="_blank">
    创建属于我们自己的国土；推荐非暴力直接行动的终极形式 — — 为自由而战（三）
   </a>
   》
  </p>
  <p class="graf graf--p">
   并且，这些政府寻求的加密通信的后门将不仅适用于那些具有所谓法治功能的政府。因为技术是没有国界的。
  </p>
  <p class="graf graf--p">
   这些寡头公司一旦构建了这样的后门就必须将其提供给威权政权，例如，他们可能以打击恐怖主义或维稳内乱为名寻求
   <strong class="markup--strong markup--p-strong">
    间谍公民的能力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   美国司法部及其在英国和澳大利亚的合作伙伴声称支持“强加密”，但是，在这封信中所描述的对加密数据的无限制访问能力与加密的实际工作方式不兼容。⚪️
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.eff.org/deeplinks/2019/10/open-letter-governments-us-uk-and-australia-facebook-all-out-attack-encryption" href="https://www.eff.org/deeplinks/2019/10/open-letter-governments-us-uk-and-australia-facebook-all-out-attack-encryption" rel="noopener noreferrer" target="_blank">
    The Open Letter from the Governments of US, UK, and Australia to Facebook is An All-Out Attack on Encryption
   </a>
  </p>
