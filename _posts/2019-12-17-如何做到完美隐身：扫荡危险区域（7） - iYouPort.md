  <p class="graf graf--p">
   您正在使用强大且唯一的密码、您已经学会了识别网络钓鱼电子邮件的基本方法、并且您已经为每个、帐户设置了双因素身份验证。基本上，您正在处理个人网络安全101阶段。
  </p>
  <p class="graf graf--p">
   但是，随着追踪者的深入新的威胁不断涌现，您可能正在寻找其他可以保护自己的步骤。
  </p>
  <p class="graf graf--p">
   这里介绍的是一个简单的防御方法：清理您的数字垃圾。
  </p>

  <p class="graf graf--p">
   数字时代的人们很容易建立一大堆几乎未使用过的帐户，注册后很快就遗忘了。然而，这些帐户非常危险。你必须打扫干净。
  </p>
  <p class="graf graf--p">
   您需要留意的东西，例如：您从未跟进的免费试用、您已经放弃的流媒体服务、使用后发现并不是您想要的东西，等等。
  </p>
  <p class="graf graf--p">
   在某些方面，这些旧帐户除了收集虚拟尘埃或偶尔发送电子邮件唠叨之外似乎没有任何伤害。但是，从安全角度来看，有太多的休眠登录可能是非常危险的事。
  </p>
  <p class="graf graf--p">
   想想看，如果应用程序背后的开发人员突然变得流氓、或者黑客攻击服务的数据库，会发生什么。尤其是，
   <strong class="markup--strong markup--p-strong">
    任何追踪者都会采取撒网式搜罗一切关于你的信息
   </strong>
   ，任何蛛丝马迹对他们来说都是非常重要的线索。
  </p>
  <p class="graf graf--p">
   ⚠️如果数字垃圾没有清理干净，不仅会将您的电子邮件地址甚至家庭地址等重要个人信息暴露出来，并且，还可能会让恶意攻击者进入更大、更重要、更敏感的帐户。
  </p>
  <p class="graf graf--p">
   这可能是因为您使用了一项主要服务来登录较小的服务（比如用谷歌/FB账户登陆一个小游戏网页等等类似的情况），或者因为您在不同的帐户之间共享了用户名和密码 —— ⚠️这是您最不应该做的事。
  </p>
  <p class="graf graf--p">
   您闲逛后就不再使用的帐户越多，追踪者和攻击者就越有目标。
  </p>
  <p class="graf graf--p">
   因此，
   <strong class="markup--strong markup--p-strong">
    建议的做法是
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a4%a7%e6%b8%85%e6%b4%97%ef%bc%9a%e5%8f%8d%e4%be%a6%e5%af%9f%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" href="https://www.iyouport.org/%e5%a4%a7%e6%b8%85%e6%b4%97%ef%bc%9a%e5%8f%8d%e4%be%a6%e5%af%9f%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     大清洗
    </strong>
   </a>
   ：整理您留下的数字路径，并关闭所有您不经常使用的帐户，尤其是那些与真实身份想关联的账户。
  </p>
  <p class="graf graf--p">
   不幸的是，并没有一个大按钮可以让您一次性做到这一切，但是，通过一些简单的侦探工作和几分钟的业余时间，你就可以有效地擦除这些踪迹。
  </p>
  <p class="graf graf--p">
   下面将介绍完成这项
   <strong class="markup--strong markup--p-strong">
    防御工作所需要的步骤
   </strong>
   。
  </p>
  <p>

   <strong class="markup--strong markup--p-strong">
    1、找到它们
   </strong>
  </p>
  <p class="graf graf--p">
   首先，选择您实际记住的帐户和应用程序，那些仍然存在于手机屏幕上的帐户和应用程序，或者定期在收件箱中提醒您的东西，因为它们最想念您。
  </p>
  <p class="graf graf--p">
   如果实在想不起来了，去电子邮件中搜索一下也是好主意 —— 所有用过一段时间的电子邮箱，尤其是陈年的邮件，很可能会提醒您关于一些曾经的注册行为。
  </p>
  <p class="graf graf--p">
   想想您在开设账户时收到的电子邮件中最常见的字眼是什么？—— 搜索“确认您的电子邮件”、或“新帐户”、或“验证您的电子邮件地址”，等这些短语，应该会能找到东西（
   <strong class="markup--strong markup--p-strong">
    别忘了使用双语搜索
   </strong>
   ：”confirm your email” or “new account” or “verify your email address”）。
  </p>
  <p class="graf graf--p">
   您可能正处于甚至不记得手机上某些应用程序是做什么的阶段，因此，浏览您安装的所有应用程序，以识别您不再使用的东西。
  </p>
  <p class="graf graf--p">
   对于订阅项目，iOS
   <a class="markup--anchor markup--p-anchor" data-href="https://apple.co/2Th4vqI" href="https://apple.co/2Th4vqI" rel="noopener noreferrer" target="_blank">
    在这个链接上
   </a>
   有一个方便的一站式商店，只需点击一下即可取消不再想要的服务。
  </p>
  <p class="graf graf--p">
   继续，前往您的 Facebook、Twitter 和 Google 帐户，并查看您已连接到他们的所有应用程序和服务，例如，所有“使用 Facebook 登录”按钮。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    点击
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.facebook.com/settings?tab=applications" href="https://www.facebook.com/settings?tab=applications" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     此处
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    查看 Facebook，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/settings/sessions" href="https://twitter.com/settings/sessions" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     此处
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    查看 Twitter，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://myaccount.google.com/permissions?utm_source=google-account&amp;utm_medium=web&amp;pli=1" href="https://myaccount.google.com/permissions?utm_source=google-account&amp;utm_medium=web&amp;pli=1" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     此处
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    查看 Google
   </strong>
   。如果您尚未登录，则需要登录。
  </p>
  <p class="graf graf--p">
   从 Facebook、Twitter 和 Google 断开第三方应用和服务通常只需要从上面那三个页面上点击一两下，但是请记住，这实际上不会关闭您未使用的帐户 —— 这只是意味着他们无法轻易进入您的主要帐户了。
  </p>
  <p class="graf graf--p">
   最后但同样重要的是，
   <strong class="markup--strong markup--p-strong">
    查看用于管理密码和登录的任何程序
   </strong>
   ，无论是专用密码管理器还是Web浏览器。
  </p>
  <p class="graf graf--p">
   这里面应该会出现一些您已经忘记的帐户，并且还有一个额外的好处，就是，在您忘记这些帐户的情况下这些应用可以向您提供登录详细信息。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、关闭它们
   </strong>
  </p>
  <p class="graf graf--p">
   现在就可以开始实际关闭这些帐户了，如果可能的话，建议最好通过桌面Web浏览器完成。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    仅仅从手机中移除应用程序是不够的，尽管您也应该这样做。
   </strong>
  </p>
  <p class="graf graf--p">
   如果您忘记了特定网站和服务的登录凭据，则必须完成帐户提供的任何密码重置选项，否则您可能需要直接联系客户支持。这很麻烦，但为了安心，这是值得的。
  </p>
  <p class="graf graf--p">
   一旦进入这些帐户，就可以寻找任何类型的数据备份或导出选项，不论您想要保留在此僵尸帐户中积累的数据还是将其移动到其他位置。
  </p>
  <p class="graf graf--p">
   某些服务会自动查找和删除旧帐户，通常是为您搜索收件箱或社交媒体帐户。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    不过我们并不是特别推荐这些服务
   </strong>
   ；因为你无法确定这些服务对你的数据做了什么，而且你必须注册另一个数字帐户才能使用它们。
  </p>
  <p class="graf graf--p">
   坚持自己动手是最靠谱的。
  </p>
  <p class="graf graf--p">
   现在终于准备关闭僵尸帐户了。
  </p>
  <p class="graf graf--p">
   如何处理这个问题会因应用和服务而异，但你应该可以通过快速搜索轻松找到该选项。
  </p>
  <p class="graf graf--p">
   您通常需要再次输入密码才能证明您就是您所说的人。
  </p>
  <p class="graf graf--p">
   如果你完全找不到方向，

   <strong class="markup--strong markup--p-strong">
    3、注意的问题
   </strong>
  </p>
  <p class="graf graf--p">
   1⃣️ 在清理的过程中，要考虑的最重要的帐户就是您的电子邮件 —— 所有邮箱，即：您在线生活的中央数据中心。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    您的电子邮件帐户对于攻击者来说将是一个宝贵的大礼包
   </strong>
   ，因为它可能包含除了您自己之外的许多其他人（朋友、家庭成员、同事、等等）的信息。
  </p>
  <p class="graf graf--p">
   删除您不再需要的电子邮件、并将您仍希望使用的旧电子邮件导出到硬盘驱动器，是一种明智的方法，可以减少因电子邮件被攻击而受到损害的可能性。
  </p>
  <p class="graf graf--p">
   几乎所有电子邮件服务都提供导出邮件和其他帐户数据的方法，以便您在本地保存，并可以从公司的服务器中删除它。
  </p>
  <p class="graf graf--p">
   并且，不要忘记清除和删除不再使用的旧电子邮件帐户。
  </p>
  <p class="graf graf--p">
   如上所述，您还应该利用电子邮件搜索功能来梳理旧邮件。
  </p>
  <p class="graf graf--p">
   您可以批量删除特定日期之前的所有内容、或特定联系人相关的所有内容。
  </p>
  <p class="graf graf--p">
   例如，搜索旧房东或雇主的名字、删除您发送过的任何电子邮件，因为这些电子邮件中更有可能包含税务表格或工资单等个人信息。
  </p>
  <p class="graf graf--p">
   您还应该定期在电子邮件中搜索您的社会安全号码/银行卡号码/身份证号码/真实手机号的最后四位数字，并删除曾经显示过这些号码的任何邮件。
  </p>
  <p class="graf graf--p">
   如果可以，请询问这些电子邮件的发件人或收件人，要求对方也将其删除。
  </p>
  <p class="graf graf--p">
   2⃣️ 和拇指驱动器一样，您还可能会在各种服务中随机保存
   <strong class="markup--strong markup--p-strong">
    文件
   </strong>
   ，这些服务提供如 Box、Google Drive 和 Dropbox 等免费存储空间。
  </p>
  <p class="graf graf--p">
   请对那里的内容进行整理，并消除可能造成特定风险的任何东西 —— 例如
   <strong class="markup--strong markup--p-strong">
    旧的纳税申报表或财务文件、拍摄了您和重要联系人的照片、视频、描述了您的真实信息一切文档
   </strong>
   。
  </p>
  <p class="graf graf--p">
   3⃣️ 您的照片是否备份到了多种不同的服务上？清理一下；您是否还拥有两年前使用过的消息应用程序帐户？为什么2014年的卡路里计算应用程序仍在您的手机上？……取消并删除一切。
   <strong class="markup--strong markup--p-strong">
    这些类型的暴露都是不必要的风险
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️再一次提醒，不要只是从手机中删除应用程序，请确保清理并关闭您的帐户，以便它保留尽可能少的关于您的数据。
  </p>
  <p class="graf graf--p">
   4⃣️ 关闭帐户并不一定意味着公司会删除您的所有数据或删除它所了解的基本信息 —— 数据处理流程应该在应用程序的服务条款中列出 —— 它们有可能会使帐户保持活跃并可能继续收集数据。
  </p>
  <p class="graf graf--p">
   5⃣️ 智能手机的相册上是否有信用卡、身份证和驾驶执照的照片？朋友和家人在旅行和绑定时是否曾录入过您的身份信息？您是否曾经因某种申请而扫描了您的家庭成员的护照和其他文件？……
  </p>
  <p class="graf graf--p">
   回忆所有这些细节。
  </p>
  <p class="graf graf--p">
   6⃣️ 全部清理完成后就可以以全新的身份开始新生活了。
   <strong class="markup--strong markup--p-strong">
    请尽量记住这些最佳实践，因为您不可避免地会积累更多的数字残骸
   </strong>
   。
