  <p class="graf graf--p">
   在俄罗斯的任何大规模抗议活动中，细心的目光都能发现：穿着便服的警官将摄像机对准人群。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果这些警察悄悄地靠近个别示威者，这些示威者就会被立即逮捕
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在幕后，
   <strong class="markup--strong markup--p-strong">
    这些警察还负责对俄罗斯公民提起刑事诉讼
   </strong>
   ，指称这些俄罗斯公民在社交媒体上分享所谓的“极端主义言论”。
  </p>
  <p class="graf graf--p">
   这些警察工作的地方叫 Center E。
  </p>
  <p class="graf graf--p">
   Center E 非常隐秘，人们对其内部知之甚少。因此 meduza 与 Center E 的前员工弗拉基米尔·沃龙佐夫（Vladimir Vorontsov）进行了交谈。
  </p>
  <p class="graf graf--p">
   自离开该中心以来，Vorontsov 以为警察创建受欢迎的
   <a class="markup--anchor markup--p-anchor" data-href="https://vk.com/ombudsmenttttttttttt" href="https://vk.com/ombudsmenttttttttttt" rel="noopener noreferrer" target="_blank">
    在线支持组织
   </a>
   而闻名。
  </p>
  <p class="graf graf--p">
   以下是 Vorontsov 的话，使用第一人称 —— 给您一个内部视角，以观察维稳部队。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     Center E 是个什么东西？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   这个名字很容易描述：就是反极端主义中心 Anti-Extremism Center。该中心是在 UBOP（打击有组织犯罪司）的基础上于2008年9月6日成立的。
  </p>
  <p class="graf graf--p">
   在莫斯科，该中心最初成立于联邦警察总部，但由于抗议活动的频发，到2012年左右，他们扩展到了地区级警察局。
  </p>
  <p class="graf graf--p">
   如果仅查看整个城市的配备，那么可以说他们大约有100名员工，然后大致可以判断每个地区大约配10名员工。
  </p>
  <p class="graf graf--p">
   从外部看，Center E 似乎是一个庞大的官僚机构，仅在监控社交媒体共享内容和反对异议上起作用。
  </p>
  <p class="graf graf--p">
   许多人认为它是一种
   <strong class="markup--strong markup--p-strong">
    政治警察
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在某种程度上，这是正确的 ——
   <strong class="markup--strong markup--p-strong">
    我确实认为他们的员工会监视所有主要的反对派活动家。
   </strong>
  </p>
  <p class="graf graf--p">
   但从某种意义上讲，上述不够充分。例如，在AEC（Anti-Extremism Center）中，有一个部门负责协调反恐活动的打压，而该部门与抗议活动或反对派无关。
  </p>
  <p class="graf graf--p">
   其余部门也有自己的任务。在公共场合有针对宗教极端主义、民族主义极端主义、和压制极端主义的部门。
  </p>
  <p class="graf graf--p">
   有一个部门可以破坏极端主义组织的经济活动：他们处理的案件是瞄准创建了犯罪组织的人；该部门的人员都有专门的经验，他们都是训练有素的经济学家。
  </p>
  <p class="graf graf--p">
   还有一个非常奇怪的部门，他们与被禁的团体和政党打交道；
   <strong class="markup--strong markup--p-strong">
    即使在该部门内部也没人理解为什么要创建这个部门
   </strong>
   。
  </p>
  <p class="graf graf--p">
   最后，⚠️为了证明自己的存在，他们只是承担了与政治有关的一切，并试图以任何可能的方式使自己显得重要。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     Center E 的员工实际上在做什么？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   我正式在反民族主义极端主义部门任职，但这
   <strong class="markup--strong markup--p-strong">
    里的人确实可以做任何自己想做的事
   </strong>
   。有一次，我甚至逮捕了恋童癖者 —— 莫斯科郊区副首席法警安德烈·卡米诺夫（Andrey Kaminov）。
  </p>
  <p class="graf graf--p">
   早在 Maxim Martsinkevich 搜寻恋童癖者时，卡米诺夫就是其中之一。他们抓住了他，然后在互联网上发布了所有录像。
  </p>
  <p class="graf graf--p">
   然后，Life
   <a class="markup--anchor markup--p-anchor" data-href="https://life.ru/t/%D0%BD%D0%BE%D0%B2%D0%BE%D1%81%D1%82%D0%B8/114833" href="https://life.ru/t/%D0%BD%D0%BE%D0%B2%D0%BE%D1%81%D1%82%D0%B8/114833" rel="noopener noreferrer" target="_blank">
    发布
   </a>
   了视频。调查委员会根据该媒体报道开了一个刑事案件，但由于调查人员对这些激进分子一无所知，他们无法采取进一步行动。
  </p>
  <p class="graf graf--p">
   他们在AEC打电话给我，并寻求帮助。因此我们提供了帮助。根据记录，我们是解决此案的人。
  </p>
  <p class="graf graf--p">
   不过，总的来说，我们的工作结构要比这简单一些，AEC特工的权力与其他警官的权力没有太大不同。
  </p>
  <p class="graf graf--p">
   通常，每天上午8:30左右，中心负责人都会与部门负责人及其代表会面，并向他们发出当天的一般命令。
  </p>
  <p class="graf graf--p">
   然后，上午9:00左右，部门负责人与员工开会并下达命令。他们询问每个人的工作，谁在处理哪些潜在的坏人 ——
   <strong class="markup--strong markup--p-strong">
    通常，每个员工都在与某个特定的人打交道，这些目标人是从警察监控名单或警员自己的消息来源中找到的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   当然，有关目标人员的信息也可以
   <strong class="markup--strong markup--p-strong">
    通过监视社交媒体来获得
   </strong>
   。
  </p>
  <p class="graf graf--p">
   负责监视社交媒体的警员会加入他们认为是可疑的团体，并
   <strong class="markup--strong markup--p-strong">
    对操作账户进行特别的掩护（也就是
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%bf%ab%e9%80%9f%e5%88%9b%e5%bb%ba%e4%b8%80%e4%b8%aa%e5%81%87%e4%ba%ba%ef%bc%9f%e7%ae%80%e6%98%93%e7%89%88%e5%88%86%e8%ba%ab%e6%9c%af%e2%80%8a-%e2%80%8a%e8%b0%83%e6%9f%a5%e4%ba%ba/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%bf%ab%e9%80%9f%e5%88%9b%e5%bb%ba%e4%b8%80%e4%b8%aa%e5%81%87%e4%ba%ba%ef%bc%9f%e7%ae%80%e6%98%93%e7%89%88%e5%88%86%e8%ba%ab%e6%9c%af%e2%80%8a-%e2%80%8a%e8%b0%83%e6%9f%a5%e4%ba%ba/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     袜子木偶
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ）
   </strong>
   ，诸如此类。
  </p>
  <p class="graf graf--p">
   如果调查发现有证据表明所追踪的人确实在做某事，那么警员应根据该事实采取适当的措施。
  </p>
  <p class="graf graf--p">
   该中心工作的另一个组成部分是，人们不断抱怨社交媒体上发生的事。例如，在 Odnoklassniki 上，有人称别人为“Yid”，而该人觉得自己被得罪了并决定向执法部门投诉。
  </p>
  <p class="graf graf--p">
   对于此类投诉，警员必须进行检查，以确定所涉是否表现出极端主义倾向。警察本身不是语言学家，因此他们无法确定帖子中是否存在任何违法行为。
  </p>
  <p class="graf graf--p">
   直到2013年，我们几乎都在相同的总体结构中工作，但是随后，它开始迅速退化。
  </p>
  <p class="graf graf--p">
   退化与以下事实有关：我们在MVD系统中所谓的总部开始发挥更大的作用。总部是内部部门，他们本身并不能直接为打击犯罪做出贡献，但是他们可以决定谁在什么时候做什么。
  </p>
  <p class="graf graf--p">
   那时候开始，
   <strong class="markup--strong markup--p-strong">
    这些部门开始为我们介绍工作计划，普通警员开始执行一些非常奇怪的任务。
   </strong>
  </p>
  <p class="graf graf--p">
   例如，我个人被派往一个宿舍，那里的学生都是北高加索人。他们要求我与负责人谈论关于学生是否在做违法的事，即使每个人都知道没人会告诉我任何事。
  </p>
  <p class="graf graf--p">
   所以这只是一个死胡同：我去了那里，写下他们的证词说一切都还好，然后就离开了。
  </p>

  <p class="graf graf--p">
   大约在同一时间引入了许多其他耗时的手续。
  </p>
  <p class="graf graf--p">
   例如，
   <strong class="markup--strong markup--p-strong">
    每周，我不得不要求[社交媒体] VKontakte 删除五张纳粹符号的图像，而我必须自己找到每一个这样的图像
   </strong>
   。
  </p>
  <p class="graf graf--p">
   我不必找到究竟是谁发布了那些纳粹标志；平台是否确实删除了图像都没有关系。
   <strong class="markup--strong markup--p-strong">
    我只需要把单子发给他们并报告已发送，就完成任务
   </strong>
   。
  </p>
  <p class="graf graf--p">
   随着时间的流逝，越来越多的这种愚蠢的现象开始出现。
  </p>
  <p class="graf graf--p">
   话虽如此，⚠️Center E 员工在一定时期内必须抓住 **预定数量的极端分子**，这也是针对犯罪调查人员和其他MVD员工的结构。
  </p>
  <p class="graf graf--p">
   AEC具有不同的评估系统 —— 这是根据内务部长的命令建立的积分系统。简而言之，如果给定的部门抓到了罪犯，那么他们将获得加分，如果罪犯逃脱了，他们将被扣分。
  </p>
  <p class="graf graf--p">
   最重要的是，员工的义务是要将分数保持在一定数量以上。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     什么样的人在 Center E 工作？在那里工作有声望吗？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   该中心刚刚建立时，基本没有人愿意在那里工作。
  </p>
  <p class="graf graf--p">
   该中心位于UBOP的废墟之上，
   <strong class="markup--strong markup--p-strong">
    当时那里的人们从事绑架案件和敲诈勒索案件等工作，然后，他们的继任者只需点击社交媒体帖子即可
   </strong>
   。
  </p>
  <p class="graf graf--p">
   人们并不认为这是一项艰巨的工作：就像在用大炮打鸟一样。
  </p>
  <p class="graf graf--p">
   因此，在2011年，当我刚转移到那里时，出现了严重的劳动力短缺。一个100人的部门可能短缺50％或60％，而真正了解工作细节的人少于10人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    一些人被送到 Center E 来是作为惩罚的（也就是降级）
   </strong>
   。很有意思的是，分部负责人亲自跑到MVD的机构周围去招募未来的毕业生。没有任何潜在的员工在排队等候被雇用。
  </p>
  <p class="graf graf--p">
   但是，随着时间的流逝，情况发生了变化。总统开始在MVD会议上更频繁地谈论加强反极端主义努力的必要性。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    AEC 的作用开始增强，其员工开始在调查操作中获得更多特权 —— 他们能够避免官僚主义的麻烦，并跳过等待名单
   </strong>
   。现在，无论走到哪里，AEC都会被开绿灯。
  </p>
  <p class="graf graf--p">
   在这一点上，人们确实想在那里工作。这主要是因为那里的工作量没有地方警察局或刑事调查部门那样繁重。另外，工作性质有些不同。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在刑事调查中，您的工作流程是从犯罪转移到肇事者，但是，在AEC上却是另一回事：是从肇事者转移到了犯罪。甚至可能是警官发现了某个可疑人员，他们被认为可能会在未来犯罪，然后就被盯住
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是您不能提前逮捕他人，因为没人知道该人是否会真正从事非法活动。在这种情况下，很容易演变成挑衅。
  </p>
  <p class="graf graf--p">
   所以，持续监视。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     人们如何被指控极端主义？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   我将引起公众强烈反抗的所有备受瞩目的极端主义案件根据其产生方式分为两类。
  </p>
  <p class="graf graf--p">
   第一类涉及公众人物，他们在政府机构决定“将其带走”后受到指控。我从来没有接到过命令要去逮捕那些给政府添麻烦的人，但是我看到了其他AEC员工的工作方式，我认为这种事情会发生。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第二类刑事案件直接来自“指标”制度 —— 也就是中国很常见的按照指标 抓人 。
   </strong>
   从技术上讲，AEC上没有指标系统，但是警官仍然必须报告他们开立的案件数量，因此差别不大。
  </p>
  <p class="graf graf--p">
   我认为大多数情况都属于第二类。
  </p>
  <p class="graf graf--p">
   例如，当有人指控玛丽亚·莫图兹纳亚（Maria Motuznaya）时，
   <strong class="markup--strong markup--p-strong">
    没人会迫切地需要去抓她。之所以发生这种情况，是因为警察坐在那儿想：“嗨伙计，这好极了，这些照片已经被保存了，我们可以直接写报告。”
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️这就是他们证明自己存在的方式。政府提出要求，然后他们去实现它。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     Center E员工对指控他人在社交媒体上分享帖子这种事有何看法？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   ⚠️通常，人们的态度是 “这只是一项工作”。大多数人不会停下来想一想，为什么仅仅因为贴了几句话或一个视频就可以将某人扔进监狱。
  </p>
  <p class="graf graf--p">
   您可以说他们是机会主义者。
   <strong class="markup--strong markup--p-strong">
    但他们的理由是：“我会按照上级的要求去做，只要我不被老板吼就行了。”
   </strong>
  </p>
  <p class="graf graf--p">
   在大多数情况下，AEC官员是曾经在刑事调查或其他类似领域工作的警察。他们曾经从事谋杀和抢劫案，但现在他们只需要整理 VKontakte 上的共享帖子就行了。
  </p>
  <p class="graf graf--p">
   因此，
   <strong class="markup--strong markup--p-strong">
    他们所有人都希望保持自己的有掌控权的地位感，并且愿意按照上级的意愿去做。
   </strong>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     Center E 员工对政府和反对派持什么态度？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   当我在那里工作时，我从未真正听到过有关政府的负面评论。
   <strong class="markup--strong markup--p-strong">
    甚至在 Center E 的人以为1917即将到来，不久他们便会穿上Cheka制服
   </strong>
   【Cheka 契卡 俄语为ЧК（活跃于1917年-1922年）是前苏联秘密警察组织，其后继者包括现在的

    间谍机构FSB
   </a>
   】。
  </p>
  <p class="graf graf--p">
   我非常生动地记得，在普京于2012年再次当选之后，我们举行了例行计划会议，其中一名指挥官说 —— 非常严肃地说，“就这样吧。从明天开始，这个国家的言论自由已经结束了。”
  </p>
  <p class="graf graf--p">
   ⚠️别误会，在他的世界观中，情况是这样的 —— 选举已经结束；没有更多的言论自由；是时候让我们这些契卡出手赶走所有人了。
  </p>
  <p class="graf graf--p">
   但是，总的来说，人们对反对派有多种看法。有些人讨厌反对派。但是，如果从整体上看 Center E，对于大多数部门来说，这根本不是他们的工作。也就根本不会去关心。
  </p>

   </strong>

  </p>

   老板进来，说：“这样做”。没有人问任何问题 —— 他们只是去做。
  </p>
  <p class="graf graf--p">
   ⚠️当他们继续工作时，服从可能会变成个人的恶意。因为您只要开始与某人合作，就会更多地了解他们，并且可以说他们是白痴，并且他们在做坏事。

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     Center E 的警官在抗议活动中做什么？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   ⚠️他们与人群融为一体，并密切注视抗议者。他们的主要目标是搜寻最活跃的人，即 领导者、带头人。
  </p>
  <p class="graf graf--p">
   AEC官员还必须能够识别政治人物。我们都知道，
   <strong class="markup--strong markup--p-strong">
    一旦有名人出门，抗议者就会被抓捕
   </strong>
   。
  </p>
  <p class="graf graf--p">
   并不是像身穿制服的警察在指挥之下的行动，而是
   <strong class="markup--strong markup--p-strong">
    有一定程度的协作
   </strong>
   。
   <strong class="markup--strong markup--p-strong">
    AEC员工是便衣，他们可以指出他们认为应该被捕的人，然后该人就会被捕
   </strong>
   。
  </p>

   也许之后警方将不得不识别某人的身份；也许某个人引起了他们的注意，等等。没人提前知道会发生什么，总之，拍下来。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     为什么 Center E 的警员会穿着便服去监视抗议？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   **结合可见和不可见模式** 是执法行动的基本原则。俄罗斯法律甚至提供了隐身操作的能力。 Center E 员工没有义务穿制服，所以他们就不穿。
  </p>
  <p class="graf graf--p">
   也就是说，
   <strong class="markup--strong markup--p-strong">
    当这些特工站在穿着制服的人旁边并与他们交谈时，很容易被识别出来。很明显，没有普通的行人会这样做
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️尤其明显的是，因为他们通常穿着牛仔裤和漂亮的鞋子，并携带一个皮革邮差包。我称其为“ Agent Leatherbag”外观。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    他们之所以如此突出有两个原因。一是同时保持隐身状态和整个 Center E 的工作有很多障碍
   </strong>
   。
  </p>
  <p class="graf graf--p">
   举一个简单的例子，一次，我开始工作，我的浏览器告诉我，我不允许在工作时间内使用社交媒体。从外面看这只是一个愚蠢的命令，禁止所有人使用社交媒体。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    另一个愚蠢的命令是说，特工到彼得罗夫卡街的城市总部必须穿制服或穿着商务服装。
   </strong>
   因此，如果在抗议发生在彼得罗夫卡总部外面，那么AEC特工就不能穿着能融入人群的衣服。否则守卫不许他们进去。
  </p>
  <p class="graf graf--p">
   此外的原因是，有些人认为他们没有暴露自己的身份。然后其他人知道他们已经暴露自己的身份了，但是他们并不在乎。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     为什么同一批 Center E 特工连续数年出现在抗议活动中？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   我不会说有人乐于参加抗议活动，也不认为这是他们职业生涯停滞的迹象。
  </p>
  <p class="graf graf--p">
   仅仅是由于，这样做了一年以上的人自然更有经验。
  </p>
  <p class="graf graf--p">
   逻辑上认为，如果抗议活动出了事，经验丰富的特工将能够更快地适应目标，并且他们更有可能做出适当的举止。
  </p>

   <img alt="" class="size-full wp-image-9794 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-8.png?resize=1100%2C828&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-8.png?w=1156&amp;ssl=1 1156w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-8.png?resize=300%2C226&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-8.png?resize=768%2C578&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-8.png?resize=1024%2C771&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-8.png?resize=1100%2C828&amp;ssl=1 1100w" data-recalc-dims="1" height="828" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/5-8.png?resize=1100%2C828&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </figcaption>
  </figure>
  <p class="graf graf--p">
   同时，
   <strong class="markup--strong markup--p-strong">
    经验丰富的特工很难融入人群，因为每个人都已经知道他们是谁了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   从外部看可能很有趣，⚠️
   <strong class="markup--strong markup--p-strong">
    但是没有人考虑到这样的事实 —— 人群中可能还有其他人无法识别的契卡。
   </strong>
  </p>
  <p class="graf graf--p">
   您所熟悉的契卡可能距离五米开外，但是，您不知道另一个契卡在哪。甚至，⚠️可能就是著名的契卡特工正故意在抗议人群中用大动作吸引目光，以引起人们的注意。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     Center E 最著名的特工 Alexey Okopny 是个什么样的人？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   他是该中心最卑鄙的员工。他曾经在打击非法党派事务部担任高级特工【就如中国的国宝头子】。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    他们会去参加所有反对派抗议活动，以某种方式证明该部门的存在
   </strong>
   ，并在某个时候，他上了镜并出了名。另外，他们本来就不是隐身操作的光辉典范。
  </p>
  <p class="graf graf--p">
   然后，他被调到现在由他领导的公共活动部。
  </p>
  <p class="graf graf--p">
   抗议后，他开车到所有警察局，指出被捕者中最活跃的抗议者，以确保执法部门对他们进行相应的对待（更严重的惩罚）。
  </p>
  <figure aria-describedby="caption-attachment-9795" class="wp-caption aligncenter" id="attachment_9795" style="width: 798px">
   <img alt="" class="size-full wp-image-9795 jetpack-lazy-image" data-lazy-sizes="(max-width: 798px) 100vw, 798px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-7.png?resize=798%2C850&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-7.png?w=798&amp;ssl=1 798w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-7.png?resize=282%2C300&amp;ssl=1 282w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-7.png?resize=768%2C818&amp;ssl=1 768w" data-recalc-dims="1" height="850" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/6-7.png?resize=798%2C850&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="798"/>

  <p class="graf graf--p">
   反对派激进分子将他视为血腥政权的刽子手。
  </p>
  <p class="graf graf--p">
   我自己和他说话没有遇到任何问题，但我认为，
   <strong class="markup--strong markup--p-strong">
    随着时间的流逝，他与反对派的斗争已经变得个人化了
   </strong>
   。
  </p>
  <figure aria-describedby="caption-attachment-9796" class="wp-caption aligncenter" id="attachment_9796" style="width: 802px">
   <img alt="" class="size-full wp-image-9796 jetpack-lazy-image" data-lazy-sizes="(max-width: 802px) 100vw, 802px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-6.png?resize=802%2C922&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-6.png?w=802&amp;ssl=1 802w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-6.png?resize=261%2C300&amp;ssl=1 261w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-6.png?resize=768%2C883&amp;ssl=1 768w" data-recalc-dims="1" height="922" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-6.png?resize=802%2C922&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="802"/>

  <p class="graf graf--p">
   总而言之，我认为这是因为，一方面，反对派对他不利，另一方面，他的事业发展得非常顺利【他的事业发展需要反对派的活跃帮他刷存在感】；自然，他会对政府及其直接上级表示感谢。
  </p>
  <p class="graf graf--p">
   他100％忠诚。他把自己看作是一个献身的老手，一个传奇的特工，从底层开始，升至全市警察总部某个部门的高层。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     Center E 特工是否会故意捏造极端主义案件？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   假设有可能，但是为什么会这样呢？鉴于中心工作的细节，他们真的不需要捏造。另外，在技术层面上，怎么做到捏造？
  </p>
  <p class="graf graf--p">
   假设您是在谈论 VKontakte 帖子。以前，您可能只是复制某人的页面，让几个见证人聚在一起，然后跟他们讨价还价。
  </p>
  <p class="graf graf--p">
   但是现在有了
   <a class="markup--anchor markup--p-anchor" data-href="https://meduza.io/en/cards/even-without-an-internet-nuclear-option-russian-intelligence-has-been-using-an-existing-law-to-try-to-access-runet-user-data-for-months-here-s-how" href="https://meduza.io/en/cards/even-without-an-internet-nuclear-option-russian-intelligence-has-been-using-an-existing-law-to-try-to-access-runet-user-data-for-months-here-s-how" rel="noopener noreferrer" target="_blank">
    Yarovaya的法律
   </a>
   ，当每个帖子都被归档后，再捏造东西就变得不现实了。基本上，如果政府想对某些人提起诉讼，这项引起很多批评的法律实际上可以对某人有利。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     Center E 特工会入乡随俗吗？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   这取决于部门。例如，在宗教极端主义部门，我真的感觉任何人最终都可能会接受他们“在战斗”的信念。
  </p>
  <p class="graf graf--p">
   反恐部门也是如此。很好地说明了该部门刚刚成立时，他们那挂满了彩色印刷品，上面印有美国电影中恐怖分子的照片。然后，一旦搞定了哪个，他们就非常仔细地划掉了那个恐怖分子的脸。
  </p>
  <p class="graf graf--p">
   即使这个案子的处理是通过莫斯科AEC以外的特殊行动进行的，而不是他们亲手做的。我觉得这就是一种入乡随俗。
  </p>
  <p class="graf graf--p">
   但是，通常，
   <strong class="markup--strong markup--p-strong">
    AEC特工与警察部队中的其他任何人都具有相同的特征。⚠️他们的同理心和人性变得不再敏感
   </strong>
   。
  </p>
  <p class="graf graf--p">
   你已经不必为自己的行为可以决定某人的命运而承担责任，你可以使他们无法再过上正常的生活。
  </p>
  <p class="graf graf--p">
   本质上，人们的态度是这项工作是一条装配线。今天，您正在追寻某人；明天，你就会去追寻别人。因此，态度只是 “这事必须这样做”。党说：“去做吧”； Komsomol 说：“完成。”【Komsomol 是苏联列宁共产主义青年团】⚪️
  </p>
