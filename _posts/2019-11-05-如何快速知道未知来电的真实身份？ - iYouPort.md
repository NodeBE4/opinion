       有很多应用程序可以让你通过电话号码查找真实身份。但是并非每一个都那么好用。并且这类应用程序颇具争议，在开源情报方面的使用需要一些注意事项

  <p class="graf graf--p">
   TrueCaller 或 GetContact 等热门应用程序声称能够查看究竟是谁打电话给您，即使您不知道该号码；也可以向用户发出垃圾邮件或诈骗电话提醒。
  </p>
  <p class="graf graf--p">
   但是，这些应用程序通过大量收集信息以确定未知呼叫者姓名的方式很难广泛传播。
  </p>
  <p class="graf graf--p">
   这类程序的判断信息来自读取用户的联系人列表，然后将数据与正在使用相同号码的其他实例以及列出号码的 Facebook 配置文件进行交叉引用。
  </p>
  <p class="graf graf--p">
   这些应用程序大多足够聪明，可以确定“妈妈”或“爸爸”不是显示传入的未知号码的有用名称，但通常会引用联系簿中给出的名字，这些名字有可能是昵称或化名，甚至是侮辱性的。
  </p>
  <p class="graf graf--p">
   例如，2014年关于现已停产的应用程序 Noknok 的一篇文章描述了有多少女性被贴上了侮辱性的名字标签，一名女性发现自己被标记为 “Ness booty call 2”。
  </p>
  <p class="graf graf--p">
   其他类似应用程序可能会提供有关个人背景的更多信息，例如对 Jamal Khashoggi 谋杀案的调查中，就用到了这样一个应用程序来揭示关键信息：Menom3ay，这是一个在沙特流行的应用程序，允许用户查看其他用户与某些电话号码的联系。
  </p>
  <p class="graf graf--p">
   开源调查中经常会用到这样的工具，尤其是当你只拿到一个电话号码的时候。
  </p>
  <p class="graf graf--p">
   比如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bellingcat.com/news/uk-and-europe/2018/05/25/mh17-russian-gru-commander-orion-identified-oleg-ivannikov/" href="https://www.bellingcat.com/news/uk-and-europe/2018/05/25/mh17-russian-gru-commander-orion-identified-oleg-ivannikov/" rel="noopener noreferrer" target="_blank">
    在对 Oleg Ivannikov 的调查中
   </a>
   ，查到与这名男子有关的两个电话号码显示了他在乌克兰使用的合法身份 “Andrei Ivanovich”，另一个号码则透露了他在俄罗斯使用的出生名：Ivannikov。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6514 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-13.png?resize=1100%2C426&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-13.png?w=1234&amp;ssl=1 1234w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-13.png?resize=300%2C116&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-13.png?resize=768%2C297&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-13.png?resize=1024%2C397&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-13.png?resize=1100%2C426&amp;ssl=1 1100w" data-recalc-dims="1" height="426" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-13.png?resize=1100%2C426&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6514" data-recalc-dims="1" height="426" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-13.png?resize=1100%2C426&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-13.png?w=1234&amp;ssl=1 1234w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-13.png?resize=300%2C116&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-13.png?resize=768%2C297&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-13.png?resize=1024%2C397&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-13.png?resize=1100%2C426&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   使用通过电话通讯录应用程序保存的信息是数字调查中经常被忽视的资源，很大程度上是由于这些应用程序很繁杂并且都具有侵入性的隐私政策。
  </p>
  <p class="graf graf--p">
   此外，绝大多数相关应用都需要手机才能使用，而缺少可访问的Web版数据库，这可能使得研究变得更加繁琐。
  </p>
  <p class="graf graf--p">
   本指南将详细说明如何
   <strong class="markup--strong markup--p-strong">
    在相对安全的环境中
   </strong>
   使用这些应用程序 —— 通过 Android 虚拟机或未连接到调查人员真实电子邮件帐户或电话号码的一次性手机/号码。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    准备调查设备
   </strong>
  </p>
  <p class="graf graf--p">
   要为您的 PC 设置模拟 Android 手机 —— 换句话说，您在计算机上运行的程序假装自己是手机。
  </p>
  <p class="graf graf--p">
   在这里看到操作教程《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%9c%a8%e6%82%a8%e7%9a%84-pc-%e4%b8%8a%e5%88%9b%e5%bb%ba-android-%e5%bc%80%e6%ba%90%e7%a0%94%e7%a9%b6%e8%ae%be%e5%a4%87%ef%bc%9f-osint-%e5%ae%89%e5%85%a8%e6%80%a7%e5%b7%a5/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%9c%a8%e6%82%a8%e7%9a%84-pc-%e4%b8%8a%e5%88%9b%e5%bb%ba-android-%e5%bc%80%e6%ba%90%e7%a0%94%e7%a9%b6%e8%ae%be%e5%a4%87%ef%bc%9f-osint-%e5%ae%89%e5%85%a8%e6%80%a7%e5%b7%a5/" rel="noopener noreferrer" target="_blank">
    如何在您的 PC 上创建 Android 开源研究设备？
   </a>
   》
  </p>
  <p class="graf graf--p">
   您还可以使用 Bluestacks 或其他 Android 模拟器，
   <strong class="markup--strong markup--p-strong">
    其中一些具有iOS版本
   </strong>
   。
  </p>
  <p class="graf graf--p">
   并非所有这些通讯录应用程序都可以在 Android 模拟器上运行，但大多数都可以。
  </p>
  <p class="graf graf--p">
   这指的是您的
   <strong class="markup--strong markup--p-strong">
    调查相对敏感，需要对目标保密
   </strong>
   的情况下使用。如果您不担心曝光自己的真实身份、愿意使用自己的真实电话号码，请跳过这步。
  </p>
  <p class="graf graf--p">
   但必须强调，手机号能暴露很多东西，在这里看到介绍
   <strong class="markup--strong markup--p-strong">
    《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e5%8f%aa%e7%9f%a5%e9%81%93%e4%b8%80%e4%b8%aa%e7%94%b5%e8%af%9d%e5%8f%b7%e7%a0%81%ef%bc%8c%e4%bd%a0%e8%83%bd%e6%8c%96%e5%87%ba%e5%a4%9a%e5%b0%91%e6%9c%89%e6%95%88%e4%bf%a1%e6%81%af/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e5%8f%aa%e7%9f%a5%e9%81%93%e4%b8%80%e4%b8%aa%e7%94%b5%e8%af%9d%e5%8f%b7%e7%a0%81%ef%bc%8c%e4%bd%a0%e8%83%bd%e6%8c%96%e5%87%ba%e5%a4%9a%e5%b0%91%e6%9c%89%e6%95%88%e4%bf%a1%e6%81%af/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     如果只知道一个电话号码，你能挖出多少有效信息
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ？》。
   </strong>
  </p>
  <p class="graf graf--p">
   我首选的研究方法是使用一个解锁的“一次性电话” —— 这是一个旧的、脏的、破的 Nexus 5X，自2016年换了新手机以来从来没有使用过。
  </p>
  <p class="graf graf--p">
   将这个破手机重新启动，并将其链接到
   <strong class="markup--strong markup--p-strong">
    专为研究而创建的独立的 Gmail 帐户
   </strong>
   ，请注意，这里没有任何SIM卡或附加的电话号码。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6515 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.jpg?resize=1100%2C1771&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.jpg?w=1191&amp;ssl=1 1191w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.jpg?resize=186%2C300&amp;ssl=1 186w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.jpg?resize=768%2C1236&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.jpg?resize=636%2C1024&amp;ssl=1 636w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.jpg?resize=1100%2C1771&amp;ssl=1 1100w" data-recalc-dims="1" height="1771" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.jpg?resize=1100%2C1771&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6515" data-recalc-dims="1" height="1771" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.jpg?resize=1100%2C1771&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.jpg?w=1191&amp;ssl=1 1191w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.jpg?resize=186%2C300&amp;ssl=1 186w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.jpg?resize=768%2C1236&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.jpg?resize=636%2C1024&amp;ssl=1 636w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-5.jpg?resize=1100%2C1771&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   其中一些这类应用程序需要实际的电话号码才能进行SMS验证。
  </p>
  <p class="graf graf--p">
   许多人发现使用Google语音号码或其他服务可以成功为此目的生成电话号码。
  </p>
  <p class="graf graf--p">
   虽然这能适用于某些网站，但是，许多平台 —— 包括 Facebook —— 会识别到一个号码不是“真正的手机号”（即连接到 Google Voice 等服务而不是真正的SIM卡），因此会拒绝将此号码用于验证。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    再次强烈建议不要使用你的实际电话号码，更不用说使用真正的手机了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   所以我通过 Tracfone 的网站购买了一张SIM卡，在中国你可能会通过其他方式买到黑卡，非常建议线下交易。
  </p>
  <p class="graf graf--p">
   延伸：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/4%e6%ac%be%e6%9c%80%e4%bd%b3%e4%b8%80%e6%ac%a1%e6%80%a7%e6%89%8b%e6%9c%ba%e5%8f%b7%e7%a0%81%e5%ba%94%e7%94%a8%ef%bc%9a%e9%80%82%e7%94%a8-android-iphone/" href="https://www.iyouport.org/4%e6%ac%be%e6%9c%80%e4%bd%b3%e4%b8%80%e6%ac%a1%e6%80%a7%e6%89%8b%e6%9c%ba%e5%8f%b7%e7%a0%81%e5%ba%94%e7%94%a8%ef%bc%9a%e9%80%82%e7%94%a8-android-iphone/" rel="noopener noreferrer" target="_blank">
    4款最佳一次性手机号码应用
   </a>
   》
  </p>
  <p class="graf graf--p">
   此外在很多机场你都可以买到预付费的SIM卡，下次旅行的时候想着注意这些东西。您应该使用现金购买，而不是其他任何支付方式。
  </p>
  <p class="graf graf--p">
   这部旧手机非常难用，于是我使用
   <strong class="markup--strong markup--p-strong">
    手机镜像程序
   </strong>
   从电脑上控制这部手机。
  </p>
  <p class="graf graf--p">
   有很多应用程序可以做到这点，有免费的也有付费的，在这里我使用的是 Apowermirror。
  </p>
  <p class="graf graf--p">
   Airdroid 和 Microsoft Remote Desktop 都可以做到，大同小异。
  </p>
  <p class="graf graf--p">
   如果您想通过PC或Mac的鼠标和键盘控制手机，最好通过USB插入设备。
  </p>
  <p class="graf graf--p">
   设备设置将根据您使用的应用程序而有所不同，但大多数都需要您打开USB调试模式（如果您不知道如何操作，请将您的手机型号添加到
   <a class="markup--anchor markup--p-anchor" data-href="https://www.google.com/search?q=how+to+turn+on+usb+debugging+mode&amp;rlz=1C1CHBF_enUS735US735&amp;oq=how+to+turn+on+usb+debugging+mode&amp;aqs=chrome..69i57j69i60.3215j0j7&amp;sourceid=chrome&amp;ie=UTF-8" href="https://www.google.com/search?q=how+to+turn+on+usb+debugging+mode&amp;rlz=1C1CHBF_enUS735US735&amp;oq=how+to+turn+on+usb+debugging+mode&amp;aqs=chrome..69i57j69i60.3215j0j7&amp;sourceid=chrome&amp;ie=UTF-8" rel="noopener noreferrer" target="_blank">
    此搜索中
   </a>
   ）。
  </p>
  <p class="graf graf--p">
   完成设置后它应该是这样的（如果它是 Apowermirror）
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">

  <p class="graf graf--p">
   在每个摘要中，使用相同的参考手机号来查找结果：我自己的电话号码，同事的号码，Verizon Wireless 的号码，特朗普

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    GetContact
   </strong>
  </p>
  <p class="graf graf--p">
   要求：短信验证、电子邮件地址
  </p>
  <p class="graf graf--p">
   免费
  </p>
  <p class="graf graf--p">
   成功率：1/5（找到了 Borodai 的号码，以及试图搜索的所有其他俄罗斯和乌克兰号码，但是，甚至无法搜索任何+1或+44的号码）
  </p>
  <p class="graf graf--p">
   理论上 GetContact 是最好的通讯录应用程序之一，尽管它在许多国家都不起作用或被禁止。
  </p>
  <p class="graf graf--p">
   例如，俄罗斯和乌克兰的大多数电话号码都会返回相关人的名字；但是，在美国和英国（以及其他国家/地区），GetContact 甚至不允许您尝试搜索号码，可能是由于法律或隐私限制。
  </p>
  <p class="graf graf--p">
   您必须能够验证SMS消息才能使用 GetContact，这意味着您应该使用仅用于调查的废旧电话或一次性手机相关联的电话号码来激活帐户。
  </p>
  <p class="graf graf--p">
   我没有尝试使用Google语音或其他类似服务来验证 GetContact。以它的标准，不一定可以被通过。

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    结论
   </strong>
  </p>
  <p class="graf graf--p">
   开源调查对这类应用程序的使用不足 —— 主要因为大多数调查人员需要准备额外的电话号码、备用的独立 Gmail 和 Facebook 帐户，并且要有足够的耐心才能真正充分利用这些资源。
  </p>
  <p class="graf graf--p">
   安全研究人员也不喜欢这类应用程序，因为其普遍具备的侵入型和欺骗性。
  </p>
  <p class="graf graf--p">
   卡巴斯基实验室的 Leonid Grustniy 去年提供了 GetContact 商业模式之危险的基本概述，提醒那些对这些应用程序更广泛的安全含义感到好奇的人提高警惕。
  </p>
  <p class="graf graf--p">
   在这篇卡巴斯基帖子中带有 GetContact 应用权限的截图：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6546 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/33.png?resize=1100%2C2006&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/33.png?w=1360&amp;ssl=1 1360w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/33.png?resize=165%2C300&amp;ssl=1 165w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/33.png?resize=768%2C1400&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/33.png?resize=562%2C1024&amp;ssl=1 562w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/33.png?resize=1100%2C2006&amp;ssl=1 1100w" data-recalc-dims="1" height="2006" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/33.png?resize=1100%2C2006&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   在开源调查中，通常会查找有关特定目标自我发布的信息，例如可以通过商业注册文档和社交媒体配置文件收集到的信息。
  </p>
  <p class="graf graf--p">
   由于成千上万的移动应用程序从用户那里收集私人数据，开源调查的资源非常丰富，因为这些信息是公开的或无意中泄露的。
  </p>
  <p class="graf graf--p">
   显然，这种趋势将导致许多道德上的困境，即 应该和不应该将哪些信息用于研究。隐私倡导和开源调查一直是相悖的。
  </p>
  <p class="graf graf--p">
   比如有人会在 TrueCaller 或 GetContact 上附加带有您的姓名的个人电话号码，尽管您自己从不使用这些应用程序。
  </p>
  <p class="graf graf--p">
   此外，这类应用程序可能容易受到用户 GPS 数据泄漏的影响；
   <strong class="markup--strong markup--p-strong">
    如果您特别注重安全，在访问这些应用程序时，可以考虑使用 VPN 或 GPS 欺骗服务（
   </strong>
   例如
   <a class="markup--anchor markup--p-anchor" data-href="https://play.google.com/store/apps/details?id=com.lexa.fakegps&amp;hl=en_US" href="https://play.google.com/store/apps/details?id=com.lexa.fakegps&amp;hl=en_US" rel="noopener noreferrer" target="_blank">
    Fake GPS
   </a>
   <strong class="markup--strong markup--p-strong">
    ）
   </strong>
   。⚪️
