  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、Mapchecking
   </strong>
  </h3>
  <p class="graf graf--p">
   <a href="https://mapchecking.com/" rel="noopener noreferrer" target="_blank">
    Mapchecking.com
   </a>
   是使用 Google Maps API 来计算一个区域内可容纳多少人的工具，就是说，它可以告诉您在地图上绘制出的区域内最多可以容纳多少人。
  </p>
  <p class="graf graf--p">
   可以想像，这个工具对于计算参加游行/集会等现场的人数来说，会比较有用。在下面看到手动的简易计算方法：

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、Phorio.com
   </strong>
  </h3>
  <p class="graf graf--p">
   曾经需要通过 Google Maps 3D视图定位特定建筑物却失败吗？也许您可以试试这个工具。
  </p>
  <p class="graf graf--p">
   Phorio.com 包含有关建筑物的大量数据，并允许您按高度、位置、楼层数和许多其他参数来查询建筑物。
  </p>

   当您的调查线索中没有任何地标只有一栋陌生的建筑物时，将其作为重点进行查找，也许就能找出具体位置。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、五种免费的历史图像资源
   </strong>
  </h3>
  <p class="graf graf--p">
   这实际上是五种资源，而不是一种，但是它们都有相似的功能。
  </p>
  <p class="graf graf--p">
   GisGeography 概述了五个具有存档功能的不同卫星图像数据库；这些功能很有用，因为它们可以让您重新查看历史卫星图像和航拍图像，以便您可以监视随时间变化的情况。对于调查目标中有近期施工迹象的位置来说会非常有用。
  </p>
  <p class="graf graf--p">
   Google Earth 的存档功能已经众所周知，但是这里还有其他四个功能：
