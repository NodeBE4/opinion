  <p class="graf graf--p">
   我们去年在科幻故事《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%bb%bf%e8%89%b2%e5%85%94%e5%ad%90%ef%bc%88%e7%a7%91%e5%b9%bb%e6%95%85%e4%ba%8b%ef%bc%89/" href="https://www.iyouport.org/%e7%bb%bf%e8%89%b2%e5%85%94%e5%ad%90%ef%bc%88%e7%a7%91%e5%b9%bb%e6%95%85%e4%ba%8b%ef%bc%89/" rel="noopener noreferrer" target="_blank">
    绿色兔子
   </a>
   》的介绍后又
   <a class="markup--anchor markup--p-anchor" data-href="https://t.me/iyouport/6777" href="https://t.me/iyouport/6777" rel="noopener noreferrer" target="_blank">
    介绍了 “umwelt” 的概念
   </a>
   。
  </p>
  <p class="graf graf--p">
   我们每个人都有一个自己的 umwelt，它是一个气泡，我们把自己包裹在里面。每个人都认为自己看到的就是 “整个世界”。
  </p>
  <p class="graf graf--p">
   这就是为什么秘密审查可能很可怕。能突破审查的人会逐渐发现，自己再也无法和那些在审查中的人沟通了，因为这已经是 “两个世界”。
  </p>
  <p class="graf graf--p">
   在去年早些时候离开埃塞俄比亚的亚的斯亚贝巴博莱国际机场时，Moses Karanja 面临着一个尴尬的处境：他无法支付给司机的车费。
  </p>
  <p class="graf graf--p">
   当他进城时，国家控制的电信
   <a class="markup--anchor markup--p-anchor" data-href="https://ioda.caida.org/ioda/dashboard#view=inspect&amp;entity=country/ET&amp;lastView=overview&amp;from=1560888269&amp;until=1563480269" href="https://ioda.caida.org/ioda/dashboard#view=inspect&amp;entity=country/ET&amp;lastView=overview&amp;from=1560888269&amp;until=1563480269" rel="noopener noreferrer" target="_blank">
    关闭了互联网访问
   </a>
   ，使应用程序无法使用。 Karanja 和司机都不知道他的旅程应该花费多少。
  </p>
  <p class="graf graf--p">
   Karanja 是多伦多大学博士学位的学生，他掏出一些现金，并与司机达成了协议。但是在6月份该国发生了
   <a class="markup--anchor markup--p-anchor" data-href="https://news.un.org/en/story/2019/06/1041121" href="https://news.un.org/en/story/2019/06/1041121" rel="noopener noreferrer" target="_blank">
    一系列暗杀
   </a>
   之后，断网事件促使 Karanja 仔细考虑了阻塞的时间和深度。
  </p>
  <p class="graf graf--p">
   他发现，即使网络的其他部分在暗杀事件发生后的几天恢复了正常运行，但是某些服务（如 WhatsApp）仍然关闭。
  </p>
  <p class="graf graf--p">
   Karanja 是对的。与名为 “开放式网络干扰观测站” 的项目合作，该项目众包来自世界各地的互联网连接数据，他发现，最初中断后，Facebook、Facebook Messenger 和 WhatsApp 的网页版继续被阻止，这使得许多埃塞俄比亚人难以沟通。一直到8月，埃塞俄比亚人都无法使用这些服务。
  </p>
  <p class="graf graf--p">
   Karanja 的研究重点是政治与互联网的交汇处，他说，OONI 数据提供了世界各地的网络封锁状况的证据 —— 这些封锁当局不太可能承认。
  </p>
  <p class="graf graf--p">
   他说：“您一定会在特定时间的特定地点获得清晰的互联网快照。”
  </p>
  <p class="graf graf--p">
   OONI 是衡量全球在线审查制度的少数措施之一，这类审查措施并不总是像在埃塞俄比亚发生的同类事那样公然。
  </p>
  <p class="graf graf--p">
   有时政府会针对特定的网站，或者要求禁用某些视频、或过滤新闻源中的图像。所有这些加起来就是秘密的审查制度。
  </p>
  <p class="graf graf--p">
   OONI 和类似的项目记录了当权者控制公民可以说或看到哪些内容的所有尝试。
  </p>
  <p class="graf graf--p">
   对审查制度的担忧是一种全球现象，即使在自由民主国家中也是如此。
  </p>
  <p class="graf graf--p">
   作为世界上最大的民主国家，印度最近
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cbsnews.com/news/kashmir-blackout-india-leaves-muslims-cut-off-from-family-eid-al-adha-today-2019-08-12/" href="https://www.cbsnews.com/news/kashmir-blackout-india-leaves-muslims-cut-off-from-family-eid-al-adha-today-2019-08-12/" rel="noopener noreferrer" target="_blank">
    关闭了
   </a>
   克什米尔的互联网，原因是民族主义政党领导人寻求对穆斯林多数地区实施
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cbsnews.com/news/india-revokes-kashmir-special-status-amid-crackdown/" href="https://www.cbsnews.com/news/india-revokes-kashmir-special-status-amid-crackdown/" rel="noopener noreferrer" target="_blank">
    更多的控制
   </a>
   。
  </p>
  <p class="graf graf--p">
   诸如社交媒体公司删除内容、或悄悄地限制内容的可见范围之类的微妙形式，都是标准的审查制度，已经引起了包括 YouTube 表演者、人权活动家、甚至 Trump 这类保守派人士的反击，他们说社交媒体公司
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%8f%8d%e5%ae%a1%e6%9f%a5%ef%bc%9a%e5%87%bb%e9%80%80%e4%b8%83%e9%aa%91%e5%a3%ab%e4%b9%8b%e4%b8%80/" href="https://www.iyouport.org/%e5%8f%8d%e5%ae%a1%e6%9f%a5%ef%bc%9a%e5%87%bb%e9%80%80%e4%b8%83%e9%aa%91%e5%a3%ab%e4%b9%8b%e4%b8%80/" rel="noopener noreferrer" target="_blank">
    以打击假新闻为借口
   </a>
   采取不公平的压制。
  </p>

  <p class="graf graf--p">
   OONI的研究人员使用志愿者提交的一系列网络信号，这些信号对个人而言意义不大，但在组合起来时就能呈现出审查的干扰模式。
  </p>
  <p class="graf graf--p">
   这些迹象看起来像是互联网上的随机现象：404错误消息和奇怪的弹出窗口。但是，OONI的研究人员使用他们的数据来揭示审查制度背后的技术。
  </p>
  <p class="graf graf--p">
   如此，就可以映射出那些隐秘的不可见的专制举动。
  </p>
  <p class="graf graf--p">
   OONI 创始人 Arturo Filasto 说，审查制度意味着您在网上看到的内容会根据您在世界上的具体位置而有所不同。
  </p>
  <p class="graf graf--p">
   他说：“存在许多互联网的平行世界。”
  </p>
  <p class="graf graf--p">
   挑战（尤其是在威权国家中）是衡量和跟踪被阻止或删除的内容以及原因。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    记录模式
   </strong>
  </p>
  <p class="graf graf--p">
   凭借其开源的 OONI Probe 软件，OONI 项目涵盖了200多个国家，包括埃及、委内瑞拉和乌克兰。
  </p>
  <p class="graf graf--p">
   志愿者在其手机，平板电脑和 Mac 或 Linux 计算机上安装 OONI Probe 应用程序（目前所有计算机都可以使用beta版）。
  </p>
  <p class="graf graf--p">
   该应用程序会定期对预设的网站列表执行ping操作，并记录响应后发回的内容，从而发现哪些网站被阻止、限制或重定向。
  </p>
  <p class="graf graf--p">
   当互联网用户开始注意到奇怪的模式时，数据就派上用场了。 2016年，OONI 研究人员
   <a class="markup--anchor markup--p-anchor" data-href="https://ooni.torproject.org/post/egypt-network-interference/#advertisement-and-malware-injection" href="https://ooni.torproject.org/post/egypt-network-interference/#advertisement-and-malware-injection" rel="noopener noreferrer" target="_blank">
    使用
   </a>
   来自志愿者的数据以
   <a class="markup--anchor markup--p-anchor" data-href="https://cpj.org/2017/05/egypt-blocks-access-to-21-news-websites.php" href="https://cpj.org/2017/05/egypt-blocks-access-to-21-news-websites.php" rel="noopener noreferrer" target="_blank">
    调查
   </a>
   埃及正在进行的媒体审查。
  </p>
  <p class="graf graf--p">
   他们发现，当用户尝试访问由非政府组织、新闻机构、甚至色情网站运营的站点时，经常将他们重定向到奇怪的弹出式窗口。
  </p>
  <p class="graf graf--p">
   一些弹出窗口向用户显示广告，而其他弹出窗口则劫持了设备的处理能力以挖掘加密货币。
  </p>
  <p class="graf graf--p">
   这种情况在2018年仍在发生，当时试图访问包括巴勒斯坦囚犯协会和联合国人权理事会在内的网站的用户都被重定向了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    测试过滤器
   </strong>
  </p>
  <p class="graf graf--p">
   在线审查不仅限于被阻止的网站。社交媒体平台也会过滤新闻源甚至私人聊天中的内容。
  </p>
  <p class="graf graf--p">
   非政府组织人权观察称，在中国，政府要求社交媒体公司需要对其平台上出现的内容负责，并已签署保证书要监视其平台上出现的所有政府不喜欢的内容。这导致了严格限制政治话题讨论的系统。
  </p>
  <p class="graf graf--p">
   公司会从用户的聊天和新闻提要中过滤掉任何可能违反政府标准的图像。这些标准并不总是对用户透明的，而且会随着时间而变化。
  </p>
  <p class="graf graf--p">
   中国的微博相当于Twitter，已经两次试图从其平台清除 LGBTQ 内容，并且在社区反抗后再次封锁相关内容。一些内容可能会在重大事件发生前被过滤掉，之后再允许。
  </p>
  <p class="graf graf--p">
   公民实验室是多伦多大学 Munk 全球事务与公共政策学院的一个项目，其研究人员希望了解过滤过程如何在微信上工作（微信是一种拥有超过10亿用户的中文消息传递和社交媒体应用）。
  </p>
  <p class="graf graf--p">
   因此，他们使用注册到加拿大电话号码的微信帐户，向注册到中国电话号码的帐户联系人发送消息。联系人报告了他们最终能看到和看不到的东西。
  </p>

    <img alt="" class="size-full wp-image-11758" data-recalc-dims="1" height="616" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-17.png?resize=1100%2C616&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-17.png?w=1300&amp;ssl=1 1300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-17.png?resize=300%2C168&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-17.png?resize=1024%2C573&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-17.png?resize=768%2C430&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-17.png?resize=1100%2C616&amp;ssl=1 1100w" width="1100"/>
   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-11758">
    Images of Winnie the Pooh were purged from Chinese social media sites after Chinese leader Xi Jinping was likened to the cartoon bear.
    <br/>
    From left: Disney, Xinhua News Agency
   </figcaption>
  </figure>
  <p class="graf graf--p">
   研究人员发现了
   <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.ca/2019/07/cant-picture-this-2-an-analysis-of-wechats-realtime-image-filtering-in-chats/" href="https://citizenlab.ca/2019/07/cant-picture-this-2-an-analysis-of-wechats-realtime-image-filtering-in-chats/" rel="noopener noreferrer" target="_blank">
    微信如何自动执行图像过滤审查
   </a>
   的细节，并发现该公司正在更新其流程以响应当前事件。
  </p>
  <p class="graf graf--p">
   过滤不仅限于89的“坦克人”照片。还包括很多当前新闻事件的照片，例如华为首席财务官孟万洲被捕、中美贸易战、和2018年美国中期选举。
  </p>
  <p class="graf graf--p">
   这与众所周知的清除示例相符，例如 当网友将卡通熊与中国领导人习近平相提并论后，审查便要求删除小熊维尼的图像。
  </p>
  <p class="graf graf--p">
   中国的国家资本主义模式允许它以这种方式调整信息。负责公民实验室研究的博士后研究员 Jeff Knockel 表示，
   <strong class="markup--strong markup--p-strong">
    中国可以要求其境内的社交媒体公司对图像进行过滤。而其他国家/地区则必须屏蔽整个互联网或特定网站，以阻止用户看到某些内容。
   </strong>
  </p>
  <p class="graf graf--p">
   他说：“这使中国政府可以在这些平台上施加
   <strong class="markup--strong markup--p-strong">
    更精细的控制
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    追踪删除
   </strong>
  </p>
  <p class="graf graf--p">
   图像过滤审查也发生在美国和其他民主国家。
  </p>
  <p class="graf graf--p">
   以所谓的反仇恨言论的旗号，Facebook、YouTube 和 Twitter 正在开发AI算法，并雇用了大批的内容审查员，来审查其平台上显示的内容。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%b0%81%e9%94%81%e8%a8%80%e8%ae%ba%e5%b0%b1%e8%83%bd%e5%88%b6%e6%ad%a2%e4%bb%87%e6%81%a8%e5%90%97%ef%bc%9f%e4%b8%8d/" href="https://www.iyouport.org/%e5%b0%81%e9%94%81%e8%a8%80%e8%ae%ba%e5%b0%b1%e8%83%bd%e5%88%b6%e6%ad%a2%e4%bb%87%e6%81%a8%e5%90%97%ef%bc%9f%e4%b8%8d/" rel="noopener noreferrer" target="_blank">
     封锁言论就能制止仇恨吗？不
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   判断包含暴力内容的视频究竟是宣传恐怖主义还是保留为侵犯人权的证据，并不总是那么容易。结果就是，权利组织发布的暴政证据经常被删除。整个互联网变成了一片岁月静好。
  </p>
  <p class="graf graf--p">
   人权组织见证人（Witness）对全球人权活动家进行培训，以直观地看到他们的视频是如何被下架的。
   <strong class="markup--strong markup--p-strong">
    这些激进人士上传的视频是记录警察残暴镇压抗议者和对平民发动军事袭击的唯一证据。但是被删除
   </strong>
   了。
  </p>
  <p class="graf graf--p">
   叙利亚档案馆等项目在月度报告中也跟踪这些记录。该档案馆由 Hadi al Khatib 和 Jeff Deutch 在柏林创立，主要充当存储和审核视频的中央组织。
  </p>
  <p class="graf graf--p">
   该小组将叙利亚战争中的暴力视频上传到 YouTube 上，有时会被谷歌的AI自动检测删除。然后，叙利亚档案馆对视频进行认证，并将其提供给人权组织。
  </p>
  <figure aria-describedby="caption-attachment-11759" class="wp-caption aligncenter" id="attachment_11759" style="width: 1300px">
   <img alt="" class="wp-image-11759 size-full jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-15.png?resize=1100%2C734&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-15.png?w=1300&amp;ssl=1 1300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-15.png?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-15.png?resize=1024%2C683&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-15.png?resize=768%2C512&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-15.png?resize=1100%2C734&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-15.png?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="734" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-15.png?resize=1100%2C734&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   <figcaption class="wp-caption-text" id="caption-attachment-11759">
    Videos of terrorist or wartime violence are often taken off social media platforms, but they can serve as vital documentation of human rights violations. Pictured is the aftermath of a car bombing in Syria. Picture Alliance
   </figcaption>

  <p class="graf graf--p">
   2017年，叙利亚档案馆发现 YouTube 在发布有关 “消除暴力和恐怖主义宣传” 的新政策后，从 YouTube 上删除了约180个频道，其中
   <strong class="markup--strong markup--p-strong">
    包含来自叙利亚的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cbsnews.com/news/youtube-videos-syria-war-activists-human-rights-violations-war-crimes/" href="https://www.cbsnews.com/news/youtube-videos-syria-war-activists-human-rights-violations-war-crimes/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     数十万个视频
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   例如，一个视频片段显示了叙利亚四家野战医院遭到破坏的镜头，是记者拍摄的，也被删除了。Deutch 说，他的团队帮助促使 YouTube 恢复了大部分视频，但其他视频却从平台上永久丢失了。
  </p>
  <p class="graf graf--p">
   Deutch 说，除了叙利亚档案馆之外，保持视频在社交媒体平台上的访问是有价值的。YouTube 或 Twitter 上的视频具有更大的影响力，可以使国际组织意识到暴行，联合国安理会在有关叙利亚化学武器的报告中就
   <a class="markup--anchor markup--p-anchor" data-href="http://www.securitycouncilreport.org/atf/cf/%7B65BFCF9B-6D27-4E9C-8CD3-CF6E4FF96FF9%7D/s_2016_738.pdf" href="http://www.securitycouncilreport.org/atf/cf/%7B65BFCF9B-6D27-4E9C-8CD3-CF6E4FF96FF9%7D/s_2016_738.pdf" rel="noopener noreferrer" target="_blank">
    引用了 YouTube 的视频证据
   </a>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “平台本身成为了这些证据的档案，” Deutch 说。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    测量现实
   </strong>
  </p>
  <p class="graf graf--p">
   Karanja 获得了博士学位，他立即制定了出国计划，因为互联网中断使他无法与其他国家的同事保持同步。于是他飞到了邻国肯尼亚，并在那里工作。尽管如此，断网问题继续影响着他。
  </p>
  <p class="graf graf--p">
   Karanja 试图使用 WhatsApp 从肯尼亚致电他的埃塞俄比亚联络人，但该服务无法使用。因此，他不得不使用传统的蜂窝服务，费用比 WhatsApp 的费率高100倍。
  </p>
  <p class="graf graf--p">
   麻烦和费用问题困扰着 Karanja。但是他认为自己很幸运。互联网对于世界各地的日常生活和商业至关重要，非洲第二人口大国的许多人无法使用他们依赖的应用程序。
  </p>
  <p class="graf graf--p">
   Karanja说：“这是我的故事：金钱损失和不便。但还有其他人忍受得更多。”⚪️
  </p>
