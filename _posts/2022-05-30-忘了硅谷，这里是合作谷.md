  <h1 class="entry-title">
   忘了硅谷，这里是合作谷
  </h1>
  <div class="entry-meta">
   <div class="meta-item date">
    <span class="updated">
     2022-05-30
    </span>
   </div>
  </div>
 </header>
 <div class="entry-content">
  <p>
   Is Bologna on the Verge of Becoming the Italian Co-op Valley?
   <br/>
   博洛尼亚即将成为意大利的合作谷？
   <br/>
   👉
   <a href="https://platform.coop/blog/is-bologna-on-the-verge-of-becoming-the-italian-co-op-valley/">
    原文链接
   </a>
  </p>
  <p>
   作者：Pietro Ghirlanda ，2022.5.4
   <br/>
   翻译：宋健骅
   <br/>
   校译：黄孙权
  </p>
  <p>
   <em>
    Bologna向来是建筑保存的典范，在70年代，当时在左翼共产党执政下，以类型学方法，开启了西欧另一种建筑保存的经典：人、生活、建筑文化形式之共同保存。如今，它转向更为宽广的合作空间模式，这是比硅谷更具进步性的空间─社会营造，更应成为当下中国乡建宝贵的参照点。（黄孙权）
   </em>
  </p>

  </figure>
  <p>
   博洛尼亚市政府，不仅可以自上而下成为开发推动力，也能支持自下而上的基层运动。以营造出有利的社会环境并支持合作企业。将这两种战略结合起来并培养互补的举措带来更坚固的合作伙伴关系，从而加强
   <a href="https://www.tandfonline.com/doi/abs/10.1080/07352166.2019.1584531">
    当地社会生态系统
   </a>
   的紧密联系。
  </p>
  <p>
   这就是意大利博洛尼亚市为支持其
   <a href="https://www.tandfonline.com/doi/abs/10.1080/07352166.2019.1584531">
    区域性的合作运动所做的事情
   </a>
   。更具体地说，数十年来，博洛尼亚市一直是各种合作经验的孵化器和促进者，但并没有因而限制了对公众的吸引力或多利益攸关团体的启发。
  </p>
  <p>
   事实上，自第二次世界大战结束以来，所谓的“红色博洛尼亚”便开始广泛促进提升其民众的
   <a href="https://www.yesmagazine.org/economy/2016/07/05/the-italian-place-where-co-ops-drive-the-economy-and-most-people-are-members">
    政治与经济的参与能力
   </a>
   ，特别是在不断扩大的工人阶级阵营中，各级市政管理部门一直在利用和维持当地民众的能量。因此，尽管意大利共产党和全国合作社联盟（LegaCoop）的在地方的表现发挥了重要作用，但“合作组织模式”在艾米利亚-罗马涅大区的意外的成功也可以从该地区的社会文化网络中强大的“嵌入性”来解释。历史学家
   <a href="https://files.libcom.org/files/red_bologna.pdf">
    Donald Sassoon
   </a>
   说道：
  </p>
  <blockquote class="wp-block-quote is-layout-flow wp-block-quote-is-layout-flow">
   <p>
    博洛尼亚之所以成为博洛利亚，其意义在于，它的成就并不是来自高层的技术官僚费边主义决策的结果，而是一个涉及越来越广泛的人口阶层的地方民主框架的结果
   </p>
  </blockquote>
  <p>
   第三部门经济学的主要研究员 Stefano Zamagni 将这个系统称为“
   <a href="https://silo.tips/download/a-civil-economic-theory-of-the-cooperative-enterprise-stefano-zamagni-university">
    市民经济
   </a>
   ”，其指的是该国的中世纪公共文化。博洛尼亚也是世界上第一所大学的所在地（该大学成立于公元 1088 年）。鉴于私人垄断公司和国家集权管理意大利经济和政治发展方面的种种
   <a href="https://www.jstor.org/stable/j.ctt5vksfr#:~:text=Book%20Description%3A,of%20the%20world%20seem%20paralyzed.">
    功能失调
   </a>
   ，
   <a href="https://papers.ssrn.com/sol3/papers.cfm?abstract_id=1986036">
    艾米利亚-罗马涅大区
   </a>
   的替代方案主张彻底下放权力和多层次公民参与决策。因此，博洛尼亚的许多举措都是直接在
   <a href="http://dumbospace.it/">
    区级
   </a>
   而非市级运作的。
  </p>
  <p>
   这无疑是一种转化原则的方法，否则这些原则将仅仅保留在《宪法》中，例如
   <a href="https://en.wikipedia.org/wiki/Subsidiarity_(European_Union)#:~:text=In%20the%20European%20Union%2C%20the,Member%20States'%20power%20is%20insufficient./">
    欧洲辅助性原则
   </a>
   或意大利反法西斯宪法第45条（Article 45），其中规定：“共和国承认互利、非投机性质的合作的社会功能。法律通过适当手段促进和鼓励其合作，并通过适当的控制以确保其合法的性质和目的。” 除此之外，人们经常强调协作经济语境中的“
   <a href="https://repository.supsi.ch/9182/1/GdB-2016_Atti.pdf">
    地方关系性
   </a>
   ”的重要性，以及它在当今全球化世界中提供的意想不到的竞争优势（例如，为工人提供更好的就业机会，和将损害社区的外部因素内部化）。
  </p>
  <p>
   因此，
   <a href="https://resources.platform.coop/resources/reading-elinor-ostrom-in-silicon-valley-democracy-and-design-six-silberman/">
    Elinor Ostrom
   </a>
   的工作内容和 James Muldoon最近提出的平台社会主义，博洛尼亚市长Matteo Lepore将该市描绘成了未来的
   <a href="https://www.collettiva.it/speciali/idea-diffusa-4-2020/2021/04/20/news/piattaforme_digitale_bologna_fairbnb_coop_valley-193579/">
    意大利合作谷
   </a>
   。但是需要明确的是，我们正在讨论一种更公平的用来替代硅谷
   <a href="https://www.dukeupress.edu/the-politics-of-operations">
    榨取商业模式
   </a>
   的方案，该模式基于团结、互惠和亲近等价值观，而不是试图以任何方式成为下一个硅谷。
  </p>
  <p>
   从这个有利的角度来看，博洛尼亚提供了一个
   <a href="https://www.shareable.net/the-great-promise-of-multi-stakeholder-cooperatives/">
    优先考虑多方利益团体
   </a>
   导向和重新思考当前管理互联网的
   <a href="https://publicseminar.org/2018/08/own-this/">
    社会契约
   </a>
   ，并将城市公共化的既定遗产与平台合作主义全球运动的新理想相结合。数字平台可以被视为提供公用事业服务的基础设施，应由受其运营影响的所有利益相关者共同拥有和管理。比较法和数字法专家
   <a href="https://papers.ssrn.com/sol3/papers.cfm?abstract_id=3098053">
    Guido Smorto表示
   </a>
   ：“市政当局被认为拥有管理当前进程所需的一切权力。”。然而，它们并不限制新成立的合作社的自治权，因此其遵守国际合作社联盟关于
   <a href="https://www.ica.coop/en/cooperatives/cooperative-identity">
    自治和独立的第四项原则
   </a>
   。
  </p>
  <p>
   接下来，我想重点介绍在博洛尼亚生态系统中成功出现的两个平台合作社。我指的是
   <a href="https://fairbnb.coop/">
    Fairbnb
   </a>
   和
   <a href="https://consegnetiche.it/">
    Consegne Etiche
   </a>
   。
  </p>
  <p>
   Fairbnb是一家艾米利亚-罗马涅大区工人所有的合作社，该合作社提供短期的长期的社交租赁，以应对当前榨取实施者造成紧迫之
   <a href="https://link.springer.com/article/10.1007/s11625-020-00804-y">
    缙绅化（Gentrification）问题
   </a>
   。除此之外，随着其在整个欧洲的扩张，它也正在向多方利益相关者合作
   <a href="https://ia902504.us.archive.org/31/items/vera-vidal/Vera%20Vidal%20%28Mar%2014%29.pdf">
    过渡
   </a>
   。因此，平台创始人打算将当地的代表纳入其治理结构，他们将负责与Fairbnb运营地区的政府机构沟通，并激活平台的社会项目。因此，地方当局可能负责证明其可持续性并提供有利的监管以便更好地与现有竞争对手竞争。
  </p>
  <p>
   2020年4月意大利封锁期间，食品供应合作社Consegne-Etiche成立。它是在《
   <a href="http://www.comune.bologna.it/sites/default/files/documenti/CartaDiritti3105_web.pdf/">
    城市背景下数字劳动力基本权利创新宪章
   </a>
   》的框架下诞生的，目的是为无法到达商店的人们提供食品和其他商品。在这一困难时期，由市政当局和博洛尼亚大学特许成立的独立公司城市创新基金会（Foundation for Urban Innovation）举行了一次公开的在线集会，众多公共和私人行动者聚集在一起，讨论
   <a href="http://www.micheledalena.it/libro-sullimmaginazione-civica/">
    如何让城市更美好
   </a>
   。希望改变商业模式的现有合作社的社会企业家、城市设计师、当地店主、学者和零工代表参与了这一
   <a href="https://www.openddb.it/film/le-consegne-etiche/">
    共同规划
   </a>
   过程。其目标是在新冠疫情之后重新思考这座城市，打击主流的商业垄断平台，同时为公民提供
   <a href="https://www.jacobinmag.com/2020/05/food-delivery-workers-italy-bologna-coronavirus">
    必要的服务
   </a>
   。
  </p>
  <p>
   我相信，为了创建一个繁荣的数字生态系统或“合作谷”，市政机构和社交网络必须能够支持我们的愿景。这还需要超越硅谷，考虑世界其他地区的真实情况，例如博洛尼亚，它根据该地区的历史价值组织其数字经济。市政当局可以通过明确支持这些新经验和提供企业家所需的资源，营造出有利于创新的环境。那么，您所在的城市可以采取哪些措施来鼓励合乎道德的数字创业？
  </p>
