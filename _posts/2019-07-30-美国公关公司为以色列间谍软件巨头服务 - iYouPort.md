       一家美国著名的公关公司现在是以色列臭名昭著的 NSO 集团公关攻势的核心部分，发生了什么？
      </strong>
     </em>

  <p>
   <img alt="" class="aligncenter size-full wp-image-3622 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-293.png?resize=1100%2C851&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-293.png?w=1248&amp;ssl=1 1248w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-293.png?resize=300%2C232&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-293.png?resize=768%2C594&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-293.png?resize=1024%2C793&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-293.png?resize=1100%2C851&amp;ssl=1 1100w" data-recalc-dims="1" height="851" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-293.png?resize=1100%2C851&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-3622" data-recalc-dims="1" height="851" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-293.png?resize=1100%2C851&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-293.png?w=1248&amp;ssl=1 1248w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-293.png?resize=300%2C232&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-293.png?resize=768%2C594&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-293.png?resize=1024%2C793&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-293.png?resize=1100%2C851&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   一个致力于民主党政治宣传运动的公关公司其一直受雇于巴拉克奥巴马和希拉里克林顿竞选宣传活动、并自我宣称其在争取枪支管制和 LGBT 平等方面的作用，而
   <strong class="markup--strong markup--p-strong">
    现在它代表着一家以向全球诸多威权政府出售强大监控技术而臭名昭著的以色列老大哥公司 — — NSO
   </strong>
   。
  </p>
  <p class="graf graf--p">
   NSO集团正面临多起诉讼，指控其技术被用来监视记者和持不同政见者，并聘请了美国著名公关公司
   <a class="markup--anchor markup--p-anchor" data-href="http://www.skdknick.com/" href="http://www.skdknick.com/" rel="noopener noreferrer" target="_blank">
    SKDKnickerbocker
   </a>
   来帮助他们打通媒体关系，该间谍公司试图扭转其破败的形象。
  </p>
  <p class="graf graf--p">
   FastCompany
   <a class="markup--anchor markup--p-anchor" data-href="https://www.fastcompany.com/90326297/despite-ban-on-spyware-ads-google-allows-ad-blitz-by-hacking-firm-nso-group?partner=rss" href="https://www.fastcompany.com/90326297/despite-ban-on-spyware-ads-google-allows-ad-blitz-by-hacking-firm-nso-group?partner=rss" rel="noopener noreferrer" target="_blank">
    首次报道
   </a>
   的这一招聘是揭示 SKDKnickerbocker 如何工作的例证，尽管它的市场方式不同。
  </p>
  <p class="graf graf--p">
   该机构由前克林顿民意测验专家 Mark Penn 经营的私募股权公司所有，他曾为 LGBT 人权运动和生殖健康研究所
   <a class="markup--anchor markup--p-anchor" data-href="http://www.skdknick.com/our-work/" href="http://www.skdknick.com/our-work/" rel="noopener noreferrer" target="_blank">
    工作过
   </a>
   ，但该公司也吸收了像
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bizjournals.com/newyork/news/2018/12/06/amazon-hires-skdknickerbocker.html" href="https://www.bizjournals.com/newyork/news/2018/12/06/amazon-hires-skdknickerbocker.html" rel="noopener noreferrer" target="_blank">
    亚马逊
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://splinternews.com/powerful-dc-firm-begging-dems-to-be-nice-to-starbucks-e-1832369181" href="https://splinternews.com/powerful-dc-firm-begging-dems-to-be-nice-to-starbucks-e-1832369181" rel="noopener noreferrer" target="_blank">
    星巴克
   </a>
   这样的巨头客户。
  </p>
  <p class="graf graf--p">
   尽管如此，尽管 SKDKnickberbocker 曾经与墨西哥和中东的压制性政府关系密切，但他们为 NSO 集团工作的决定依旧尤为引人注目。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “SKDKnickerbocker 代表了许多有正当理由聘用我们的客户，防止和调查恐怖主义以拯救生命的工作是一个特别重要的原因，”SKDKnickerbocker 在对 intercept 的一份声明中说。“NSO 与许多其他网络技术公司的区别在于其对道德业务框架的承诺，该框架依赖于来自世界各地的具有国家安全和情报背景的人员的专业知识来评估潜在客户并审查当前客户。在我们9/11后的世界中，跟上技术先进的恐怖分子是一项重要的任务。“
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这家美国的通讯和咨询公司现在是以色列 NSO 集团公共关系攻势的核心部分
   </strong>
   ，这些公关活动是在一系列报道揭示 NSO 臭名昭著的 Pegasus 间谍软件入侵记者和活动家的电话、将他们的 iPhone 和电子设备变成行走的间谍后发起的。
  </p>
  <p class="graf graf--p">
   除了保留 SKDKnickerbocker 的服务外，NSO 集团（这家
   <a class="markup--anchor markup--p-anchor" data-href="https://www.calcalistech.com/ctech/articles/0,7340,L-3758232,00.html" href="https://www.calcalistech.com/ctech/articles/0,7340,L-3758232,00.html" rel="noopener noreferrer" target="_blank">
    价值8.3亿美元的公司
   </a>
   ）还创建了一个网站，购买了 Google 的广告，并向电视新闻工作人员敞开了大门。
  </p>
  <p class="graf graf--p">
   曾经在全球侵犯人权的阴暗狂魔居然摇身一变登上了热门节目和谷歌搜索排名第一的广告，是谁在背后支持他们以令他们有了”不会遭受制裁、还是继续捞更多钱”的信心？更多报告详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/newsletters-07032019-7293b81accc6" href="https://medium.com/@iyouport/newsletters-07032019-7293b81accc6" rel="noopener noreferrer" target="_blank">
    金榜不要脸
   </a>
   》。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他们正在试图进行名誉止损，”保护记者委员会的倡导主任 Courtney Radsch 说。
  </p>
  <p class="graf graf--p">
   然而，批评人士表示，即便NSO集团意图转向透明度，它仍然会回避有关如何评估潜在政府客户的问题，以及该公司为防止其技术被用于侵犯人权而做出的任何调查。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他们将 Pegasus 间谍软件出售给那些人权记录不佳的国家，完全可以预见，后者会滥用间谍软件来瞄准记者和人权活动家，他们的公关推动迄今为止并没有对这些问题进行有意义的解决，而是将其驳回了。”Citizen Lab 的高级研究员 Bill Marczak 表示。
  </p>
  <p class="graf graf--p">
   像许多以色列监视公司一样，
   <strong class="markup--strong markup--p-strong">
    NSO集团是由以色列军队退伍军人进入私营部门发起的
   </strong>
   。该公司的
   <strong class="markup--strong markup--p-strong">
    创始人 Shalev Hulio 和 Omri Lavie 使用了由以色列8200部队的老兵创建的技术推出了 Pegasus
   </strong>
   ，
   <strong class="markup--strong markup--p-strong">
    该部队相当于美国国家安全局
   </strong>
   ，Pegasus这是一种远程访问目标智能手机的间谍软件。具体介绍详见 IYP 类目“关于以色列和NSO”部分 – “Evil-doing”。
  </p>
  <p class="graf graf--p">
   该间谍软件允许情报机构发送带有链接的无害文本消息，如果目标点击了这些链接，则将目标的智能手机变成移动的间谍设备。该技术通过直接访问手机和捕获目标的对话、个人联系人、电子邮件、短信和图片来规避加密。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “NSO 向政府情报和执法机构许可其产品，为他们提供预防和调查恐怖主义和犯罪所需的工具，”该公司在 SKDKnickerbocker 发送到 intercept 的声明中表示。显然这只是NSO自己的狡辩口吻。
  </p>
  <p class="graf graf--p">
   然而，根据公民实验室的研究，NSO集团的产品已经被用来针对没有参与任何犯罪的民间社会成员。 2016年8月，公民实验室的研究人员
   <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.ca/2016/08/million-dollar-dissident-iphone-zero-day-nso-group-uae/" href="https://citizenlab.ca/2016/08/million-dollar-dissident-iphone-zero-day-nso-group-uae/" rel="noopener noreferrer" target="_blank">
    发现
   </a>
   ，阿联酋的人权活动家 Ahmed Mansoor 成为NSO制造的间谍软件的目标，可能由阿联酋安全部队运营。（Mansoor已向Citizen Lab发送了他收到的包含间谍软件的短信副本）
  </p>
  <p class="graf graf--p">
   自2016年8月以来，公民实验室的研究人员发现，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2017/06/19/world/americas/mexico-spyware-anticrime.html" href="https://www.nytimes.com/2017/06/19/world/americas/mexico-spyware-anticrime.html" rel="noopener noreferrer" target="_blank">
    墨西哥政府购买
   </a>
   的 Pegasus 间谍软件被用来针对调查腐败和有组织犯罪的墨西哥记者。另外，他们
   <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.ca/2018/07/nso-spyware-targeting-amnesty-international/" href="https://citizenlab.ca/2018/07/nso-spyware-targeting-amnesty-international/" rel="noopener noreferrer" target="_blank">
    确认
   </a>
   Pegasus 针对的是人权组织大赦国际的员工。
  </p>
  <p class="graf graf--p">
   即便如此，这些都还没能动摇NSO的形象。
  </p>
  <p class="graf graf--p">
   但在沙特记者 Jamal Khashoggi 被谋杀之后，NSO的形象受到了最大的打击。随后的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2018/12/02/world/middleeast/saudi-khashoggi-spyware-israel.html" href="https://www.nytimes.com/2018/12/02/world/middleeast/saudi-khashoggi-spyware-israel.html" rel="noopener noreferrer" target="_blank">
    新闻报道
   </a>
   突出了 Pegasus 如何侵入了 Khashoggi 沙特联系人的电话 — — NSO 集团协助沙特政府监视并最终杀死了 Khashoggi。
  </p>
  <p class="graf graf--p">
   NSO 集团否认与谋杀 Khashoggi 有任何关系。美国官媒
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/opinions/global-opinions/how-the-mysteries-of-khashoggis-murder-have-rocked-the-us-saudi-partnership/2019/03/29/cf060472-50af-11e9-a3f7-78b7525a8d5f_story.html?utm_term=.e54de775cff0" href="https://www.washingtonpost.com/opinions/global-opinions/how-the-mysteries-of-khashoggis-murder-have-rocked-the-us-saudi-partnership/2019/03/29/cf060472-50af-11e9-a3f7-78b7525a8d5f_story.html?utm_term=.e54de775cff0" rel="noopener noreferrer" target="_blank">
    “华盛顿邮报”报道
   </a>
   说，由于担心其间谍软件可能被滥用，该公司已冻结其对该王国的销售。
  </p>
  <p class="graf graf--p">
   该公司在对 The Intercept 的一份声明中表示，“虽然我们无法讨论特定政府或机构是否已获得NSO技术许可，但任何不属于预防或调查犯罪和恐怖活动的行为都被视为滥用行为，并将予以调查。该公司保留在必要时关闭系统的权利。“
  </p>
  <p class="graf graf--p">
   由于涉及该集团涉嫌参与跟踪 Khashoggi 的问题，联合创始人 Hulio 已经邀请主流媒体对其进行了记录采访，甚至允许 CBS 电视摄像机进入以色列的公司办公室。
  </p>
  <p class="graf graf--p">
   人权组织迫使 NSO 停止将 Pegasus 出售给专制政府，他们表示，他们怀疑一家以向侵略性政权出售侵入性工具为基础的公司是否能从根本上改变其业务。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “简单地聘请一家公关公司来清理他们的形象而不实际改变任何东西，不仅仅是不承担责任，这对那些要求真正答案和补救的人来说是一种侮辱，
   </strong>
   ”领导国际隐私防监视保护计划的 Edin Omanovic 说，“NSO集团，其所有者以及整个行业都应该去解决全球范围内造成的真正危害，而不是专注于公关。”
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    A
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2019/04/05/pegasus-spyware-pr-skdknickerbocker/" href="https://theintercept.com/2019/04/05/pegasus-spyware-pr-skdknickerbocker/" rel="noopener noreferrer" target="_blank">
    TOP PROGRESSIVE CONSULTING FIRM IS DOING PR FOR AN ISRAELI SPY COMPANY
   </a>
   . NSO Group, which is facing multiple lawsuits charging that its technology was used to spy on journalists and dissidents, hired SKDKnickerbocker to help with media inquiries as the spy firm tries to turn around its marred image.
  </p>
