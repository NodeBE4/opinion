  </p>
  <p class="graf graf--p">
   间谍采取的完全是心理战术，于是一些线索可以帮您识别有问题的渗透者。
  </p>
  <p class="graf graf--p">
   本文是第一部分，将讲述一种主动出击的方法。
  </p>

  <p class="graf graf--p">
   渗透入目标组织内部的间谍经常采取主动出击的方法，以尽快掌握主动权。
  </p>
  <p class="graf graf--p">
   由于环境不同，他们采取的具体措施各不相同，但是
   <strong class="markup--strong markup--p-strong">
    目的性都非常明确
   </strong>
   。⚠️这是与普通人的一个重要区别，普通人犯糊涂和性情偏激并不奇怪，但是不会有如此强的目的性。
  </p>
  <p class="graf graf--p">
   并且，⚠️很多时候
   <strong class="markup--strong markup--p-strong">
    线人不会与他们的目标正面接触
   </strong>
   ，而是采取迂回战术，从各个侧面包抄，然后逐一击破；目的使对手处于一种孤立无援的状态，再发起直接进攻，如此便有很大的胜算。
  </p>
  <p class="graf graf--p">
   这种手段极为常见，而且能达到出奇制胜的效果。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    即便是再强大的对手，一旦被孤立起来，就会变成一只困兽，不再具有反攻能力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   反过来说，如果你发现组织内部出现有步骤有目的的分化操作，当提醒警惕，并尽可能迅速找到背后的操纵者 —— ⚠️线人经常会尽可能让事件看起来与自己无关，他们会利用一些无辜者为自己做掩护。
  </p>
  <p class="graf graf--p">
   以下将分析这种手段如何操作，尤其是如何防御。
  </p>
  <p class="graf graf--p">
   在此之前需要提示一点点概念问题。
  </p>
  <p class="graf graf--p">
   人们使用过很多名词来形容从事间谍活动的特定人物，比如线人、线民、变节者、墙头草……等等，但必须说，理论上间谍和线人等概念不完全相同。
  </p>
  <p class="graf graf--p">
   间谍指的是专职为你的对手服务的人，他们被高度控制，是
   <strong class="markup--strong markup--p-strong">
    不容易被转化
   </strong>
   的；而线人等其他概念包括没有专职为你的对手服务的人，他们可能以各种原因变成了你的对手 —— 包括组织内部的争议演变为变节。这类人不被控制，是
   <strong class="markup--strong markup--p-strong">
    相对容易被转化
   </strong>
   的。
  </p>
  <p class="graf graf--p">
   对于后者，黑社会的山头之争中很常见。竞争对手向警察告密，为的不是帮助警察取证，而是借助警察的力量打击自己的对手。
  </p>
  <p class="graf graf--p">
   如果识别为属于这种情况，有些时候警方的间谍也会故意在山头之间挑起纷争，以促进告密者的出现。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、让对方感觉到被孤立是取胜的法宝
   </strong>
  </p>
  <p class="graf graf--p">
   这是每一个接受过相关培训的间谍都懂得的道理。
  </p>
  <p class="graf graf--p">
   就如任何一种攻击之所以能成功的关键一样，
   <strong class="markup--strong markup--p-strong">
    这种操作首先要做的就是对其目标进行全面的心理分析，并透彻理解，从而准确抓住目标人的心理弱点
   </strong>
   。
  </p>
  <p class="graf graf--p">
   换个角度看，你平时暴露自己的内心越多、在网上说的越多，你的弱点就会越明显。
  </p>
  <p class="graf graf--p">
   因为你永远不会知道有谁正在观察和揣摩你，以及他们都是什么目的。
  </p>
  <p class="graf graf--p">
   ⚠️这种心理攻击手段也经常出现在审讯中。
  </p>
  <p class="graf graf--p">
   如果被审讯者坚持零口供，审讯人就会采取瓦解士气的心理攻击策略，从侧面进攻，往往会起到意想不到的效果。
  </p>
  <p class="graf graf--p">
   *让对方感觉孤立无援是实施打击的第一步；准确掌握对方的心理弱点，是实施孤立战术的基础。
  </p>
  <p class="graf graf--p">
   中国读者应该对心理战非常熟悉了，中国古代的兵书中就强调过这点：“谋事不以力，而以攻心为上”。也就是说，心理战术的效果经常会强过武力，与武力协调使用将事半功倍。
  </p>
  <p class="graf graf--p">
   同样道理，间谍对目标人和群体展开的进攻大多运用这种方式。⚠️一个人只要从心理上被打败了，纵然是拥有庞大的兵力，也是丝毫没有战斗力的。
  </p>
  <p class="graf graf--p">
   于是，反制的方法也来自同一思路。
  </p>
  <p class="graf graf--p">
   斯诺登在第一次揭秘后就立刻面临了强大的威胁；美国下达了全球通缉令，许多国家都碍于地缘政治利益而拒绝给予斯诺登政治庇护。
  </p>
  <p class="graf graf--p">
   但是斯诺登继续曝光接下来的一系列文件，而不是就此隐藏起来。表面上看起来这似乎是“绝路”？
  </p>
  <p class="graf graf--p">
   事实上是一种反制策略。
  </p>
  <p class="graf graf--p">
   因为在第一份文件曝光后他已经成为了联邦政府和间谍机构的敌人，无论如何都不会改变这点；于是，
   <strong class="markup--strong markup--p-strong">
    在这种情况下有可能保护自己的方法就是加深这种敌对关系，而不是“回头”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果斯诺登在第一次曝光后便停止，听从国安局的“公开劝诫”、不再继续所谓的“叛国之举”的话，那么他的安全将受到极大的威胁。
  </p>
  <p class="graf graf--p">
   ⚠️他会因此迅速失去最强大的支持者 — — 这就是为什么国安局从不强调惩罚，而是公开“善意/原谅/鼓励”，以
   <strong class="markup--strong markup--p-strong">
    诱使目标心理动摇，从而迅速被孤立，之后只能是快速被击破。
   </strong>
  </p>
  <p class="graf graf--p">
   美国当局虽然发起了全球通缉，但并没有真正真正部署具体行动；美国间谍组织和情报共享网络是全球一流的，如果他们想做，绝对能做得到。
  </p>
  <p class="graf graf--p">
   但是相反，国安局并没有立刻实施追捕和暗杀，而是频频向斯诺登“示好”，他们不止一次公开宣称，只要停止揭露文件他们将撤销通缉令，并恢复护照的合法性；与此同时，国安局还对任何同情斯诺登的组织和国家施加压力，令他们放弃，以孤立斯诺登。
  </p>
  <p class="graf graf--p">
   比如从一开始便允诺为斯诺登提供庇护的冰岛和厄瓜多尔政府，被美国利用外交和贸易等手段施加压力，最终导致两国都转变了对斯诺登的态度。
  </p>
  <p class="graf graf--p">
   因为他们知道，只要斯诺登答应不再继续曝光，他就会迅速失去那些一直在帮助他和组织和政府的支持；将斯诺登孤立起来，国安局就能随意处置他了。
  </p>
  <p class="graf graf--p">
   在这场较量中，斯诺登作为表面上被动的一方，实际上从一开始就掌握了主动地位。
  </p>
  <p class="graf graf--p">
   因为他懂得国安局监视的常用手段。只要他停止曝光，很快就会淡出人们的视野，这样他就会面临更大的危险。
  </p>
  <p class="graf graf--p">
   相反，只要他能做到持续曝光，在全球人的瞩目下国安局就不敢对他下手。因为一旦国安局动手，就相当于主动承认了斯诺登对他们指责。
  </p>
  <p class="graf graf--p">
   💡我们在*行动策略*系列中介绍过如何将你的对手置于决策困境，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%8a%97%e8%ae%ae%e6%98%af%e9%9d%9e%e6%b3%95%e7%9a%84%ef%bc%8c%e9%82%a3%e5%b0%b1%e8%ae%a9%e6%97%a5%e5%b8%b8%e7%94%9f%e6%b4%bb%e6%88%90%e4%b8%ba%e6%8a%97%e8%ae%ae%ef%bc%88%e4%b8%80/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%8a%97%e8%ae%ae%e6%98%af%e9%9d%9e%e6%b3%95%e7%9a%84%ef%bc%8c%e9%82%a3%e5%b0%b1%e8%ae%a9%e6%97%a5%e5%b8%b8%e7%94%9f%e6%b4%bb%e6%88%90%e4%b8%ba%e6%8a%97%e8%ae%ae%ef%bc%88%e4%b8%80/" rel="noopener noreferrer" target="_blank">
    在这里看到
   </a>
   ，使用的是同一种反制思路。
  </p>
  <p class="graf graf--p">
   反制 即
   <strong class="markup--strong markup--p-strong">
    使用同样的策略将针对你的人孤立起来
   </strong>
   。
  </p>
  <p class="graf graf--p">
   国安局间谍全球的内幕曝光后虽然没能终止各种被曝光的监视手段，但是，绝对在地缘政治角度上产生了强大的影响，斯诺登是绝对的赢家。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、故意在对手组织内部制造矛盾
   </strong>
  </p>
  <p class="graf graf--p">
   在孤立对手的同时，间谍往往都会使用各种各样的其他手段，甚至包括约炮，都可以是策略；具体介绍参见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a7%98%e5%af%86%e8%a1%8c%e5%8a%a8/" href="https://www.iyouport.org/%e7%a7%98%e5%af%86%e8%a1%8c%e5%8a%a8/" rel="noopener noreferrer" target="_blank">
    秘密行动
   </a>
   》。
  </p>
  <p class="graf graf--p">
   二战期间著名的女间谍 Virginia Hall 年轻的时候在一次打猎的过程中伤到了左脚，导致膝盖以下被截肢，但这并没有影响她成为一名出色的间谍。
  </p>
  <p class="graf graf--p">
   军情六处招募她为外线间谍，她装上假肢后就被派往了法国里昂。
  </p>
  <p class="graf graf--p">
   在战争期间，为了安全起见，大多数间谍在同一个地方停留时间不会超过三个月，但 Virginia Hall 由于出色的间谍技能，在里昂一次停留就是十五个月。
  </p>
  <p class="graf graf--p">
   随后她又被派往伦敦，为中央情报局设立的战略服务办公室提供情报。
  </p>
  <p class="graf graf--p">
   Virginia Hall 受到全球两大情报机构的青睐，主要因为对其他间谍来说艰难的任务她每一次都能出色完成。
  </p>
  <p class="graf graf--p">
   在纳粹统治区工作的时候，当时她的任务除了搜集关于纳粹的情报之外（被动任务），还有一个主动任务，就是从内部瓦解对手，为英美联军实施最后的总攻做好准备。
  </p>
  <p class="graf graf--p">
   Hall 接受过专业训练，她知道要成功完成这项任务就必须实施巧妙的心理战术。但是当时的处境导致她根本无法主动出面。
  </p>
  <p class="graf graf--p">
   因为当时盖世太保正在四处缉拿她，并且他们还知道她是“一个跛脚女人”。这对于 Virginia Hall 来说非常不利。另一方面，在当时的纳粹军中散布不利信息也几乎不可能。
  </p>
  <p class="graf graf--p">
   Virginia Hall 在后来的回忆中写道：
   <strong class="markup--strong markup--p-strong">
    要想打击敌军，令其失去战斗力，关键在于瓦解其军心，最好的办法就是在敌人内部制造一种矛盾，从而让敌人的军队完全被孤立起来
   </strong>
   。
  </p>
  <p class="graf graf--p">
   她的方法说起来并不复杂。因为在当时，不仅是德国，所有卷入战争的国家的人民都对战争充满了恐惧和厌恶。战争导致他们流离失所甚至家破人亡。
  </p>
  <p class="graf graf--p">
   这对 Virginia Hall 来说是一个绝佳的切入点。她利用有限时间频频潜入当地驻军的城市或村庄，向当地人讲述战争的危害。借助人们的怨气，她迅速获得了热烈的支持。
  </p>
  <p class="graf graf--p">
   由于是驻军地区，纳粹军经常要与当地人接触，于是人们在不知不觉中就把这份怨气传递给了那些纳粹士兵。
  </p>
  <p class="graf graf--p">
   最初纳粹士兵只是将其视为一种简单的抱怨，并没有真的放在心上，但是听得越来越多之后，这些士兵在回到军营谈起自己的家人时，也开始为家人担忧。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    借助情感传递将这份内部分化的情绪迅速传播开了
   </strong>
   。没多久，一股厌恶的情绪开始在军中蔓延。很多士兵感觉到了战争的冷酷无情，于是纷纷逃离军营，参加了当地的反战组织。
  </p>
  <p class="graf graf--p">
   如此，整个军队逐渐成为一支被大多数人民所抛弃的军队。尽管控制权依旧被希特勒牢牢掌握，但实际上他已经被孤立起来了。
  </p>
  <p class="graf graf--p">
   因为孤立不仅仅是来自人民大众的反对，
   <strong class="markup--strong markup--p-strong">
    还来自希特勒十分信赖的军队内部。
   </strong>
  </p>
  <p class="graf graf--p">
   在纳粹投降之前，纳粹军队就几乎失去了战斗力。整个军队的士气已经被瓦解，不论战斗力有多强大都注定会失败。
  </p>
  <p class="graf graf--p">
   Virginia Hall 的侧面心理攻势起到了很大的作用。
  </p>
  <p class="graf graf--p">
   ⚠️这里的“敌”“我”并没有确定的善恶之分，默认心理战的实施者为“我”、其目标为“敌”。无论是地缘政治斗争还是国内对反对派群体的渗透，都是同样的道理。
  </p>
  <p class="graf graf--p">
   从内部瓦解对手组织的士气这一心理战术是各国间谍的必修课。
  </p>
  <p class="graf graf--p">
   如前所述，包括审讯中也一样。审讯一般都是分开单独进行的，审讯者做的就是采取心理攻势，努力孤立各方，让被审讯者认为自己的伙伴已经叛变。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-6850 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-6.jpg?resize=1100%2C605&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-6.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-6.jpg?resize=300%2C165&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-6.jpg?resize=768%2C422&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-6.jpg?resize=1024%2C563&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-6.jpg?resize=1100%2C605&amp;ssl=1 1100w" data-recalc-dims="1" height="605" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-6.jpg?resize=1100%2C605&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

    <img alt="" class="aligncenter size-full wp-image-6850" data-recalc-dims="1" height="605" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-6.jpg?resize=1100%2C605&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-6.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-6.jpg?resize=300%2C165&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-6.jpg?resize=768%2C422&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-6.jpg?resize=1024%2C563&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-6.jpg?resize=1100%2C605&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、从细微之处突破对方的心理防线
   </strong>
  </p>
  <p class="graf graf--p">
   如前所述，孤立对手是常用的办法。但是在有些时候，做到这点不会太容易。于是间谍往往会尝试突破对方的心理防线。
  </p>
  <p class="graf graf--p">
   从细微之处入手寻找切入点。
  </p>
  <p class="graf graf--p">
   还以斯诺登的经历为例。国安局间谍就是利用这种方法挑起斯诺登和阿桑奇之间的矛盾，试图将斯诺登完全孤立起来。尽管没有真正成功，但却在一定程度上离间了斯诺登和维基解密。
  </p>
  <p class="graf graf--p">
   这种心理攻势也是间谍培训中的一个重点：
   <strong class="markup--strong markup--p-strong">
    哪怕是做事十分谨慎的人，其身上都存在某种不可改变的个性或性格缺陷
   </strong>
   。即便受过专业培训，有着超强的心理素质，知道该如何更好地保护自己不被发现，但其内心深处的弱点往往很难克服。
  </p>
  <p class="graf graf--p">
   这是人性使然，没有十全十美的人。
  </p>
  <p class="graf graf--p">
   但是要发现这一弱点并不容易。中情局的反间谍专家曾经说过：“反攻应该从人性角度入手，以便找到心理突破口”。
  </p>
  <p class="graf graf--p">
   从国安局对斯诺登的手段中可以看出，间谍们将斯诺登留在美国的家人当成了切入点；与此同时他们还了解斯诺登是一个重感情的人，必定会对其家人十分看重。
  </p>
  <p class="graf graf--p">
   于是国安局的间谍便由此展开了心理攻势：一方面他们通过媒体放大“劝说”性的信息，并谎称只要斯诺登放弃曝光他们就会撤销全球通缉令；而另一方面，他们又派出特工突击斯诺登家人的心理防线。
  </p>
  <p class="graf graf--p">
   他们采取的是迂回战术：
   <strong class="markup--strong markup--p-strong">
    并没有直接用
   </strong>
   斯诺登的安全问题做武器，而是转向了阿桑奇。
  </p>
  <p class="graf graf--p">
   对于斯诺登的父亲而言，他已经从报纸上知道了自己儿子的处境，也知道阿桑奇是唯一直接帮助斯诺登的人。而且他也明白，家中的所有通信都被监听了。所以家人对斯诺登情况的了解其实并不多。
  </p>
  <p class="graf graf--p">
   于是国安局就利用这点，在斯诺登家人面前诋毁阿桑奇，谎称阿桑奇“借助斯诺登到处敛钱”。
  </p>
  <p class="graf graf--p">
   事实上这一切完全没有任何证据。国安局利用的是曼宁的例子，做了一口黑锅给阿桑奇。
  </p>
  <p class="graf graf--p">
   因为当时曼宁曝光文件后奥巴马政府立刻起诉了曼宁，虽然所有被起诉的罪名并没有被法官一一批准，但最终还是被判刑了。这些资料在各大媒体上公开，所有人都知道。
  </p>
  <p class="graf graf--p">
   但联邦调查局很快发现，曼宁的支持者非常多，人们不仅没有被当局的起诉和污蔑所动摇，并且组织起来为曼宁集资了125万美元。
  </p>
  <p class="graf graf--p">
   125万美元的确是事实，但
   <strong class="markup--strong markup--p-strong">
    国安局陈述的内容完全是虚构的
   </strong>
   ，他们将这笔钱栽赃在了阿桑奇身上，并讲给斯诺登的父亲，谎称“阿桑奇借曼宁收敛支持者的钱财”。这就是
   <strong class="markup--strong markup--p-strong">
    人格暗杀
   </strong>
   的一部分。在这里看到详细解释《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%ba%e6%a0%bc%e6%9a%97%e6%9d%80%e5%92%8c%e6%8a%b9%e9%bb%91%e8%bf%90%e5%8a%a8%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%8f%97%e7%9b%8a%ef%bc%9f/" href="https://www.iyouport.org/%e4%ba%ba%e6%a0%bc%e6%9a%97%e6%9d%80%e5%92%8c%e6%8a%b9%e9%bb%91%e8%bf%90%e5%8a%a8%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%8f%97%e7%9b%8a%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    什么是人格暗杀
   </a>
   》。
  </p>
  <p class="graf graf--p">
   在这里谎言的压力下，斯诺登的父亲果然受到了影响，开始怀疑阿桑奇 —— 斯诺登唯一的最大的支持者，他也通过律师表达了同样的质疑。
  </p>
  <p class="graf graf--p">
   国安局的目的十分明显，就是通过斯诺登的父亲离间斯诺登和阿桑奇之间的关系，
   <strong class="markup--strong markup--p-strong">
    在支持者之间制造猜忌，从而孤立斯诺登，以便进一步攻击他
   </strong>
   。
  </p>
  <p class="graf graf--p">
   后来阿桑奇的助手及时赶到香港协助斯诺登避难，国安局的心理战才未能如愿。但是，这种心理战术非常值得警惕 —— 因为并非所有被瞄准的人都具备有效的反攻机会和能力。
  </p>
  <p class="graf graf--p">
   尤其是，我们听说过中国当局擅长胁迫异议人士的家人；于是您很有必要在防卫计划中将家人被突破的可能性纳入进去。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、解读行为举止
   </strong>
  </p>
  <p class="graf graf--p">
   就如 IYP 在“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
    社交工程
   </a>
   ”栏目中多次提到过的，观察和情报搜集是发起任何有效攻势的前提。比如，在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%94%a8%e7%9c%bc%e8%af%86%e4%ba%ba%ef%bc%9a%e4%bd%a0%e6%83%b3%e7%9f%a5%e9%81%93%e8%87%aa%e5%b7%b1%e5%9c%a8%e4%b8%80%e4%b8%aa%e9%97%b4%e8%b0%8d%e7%9a%84%e7%9c%bc/" href="https://www.iyouport.org/%e7%94%a8%e7%9c%bc%e8%af%86%e4%ba%ba%ef%bc%9a%e4%bd%a0%e6%83%b3%e7%9f%a5%e9%81%93%e8%87%aa%e5%b7%b1%e5%9c%a8%e4%b8%80%e4%b8%aa%e9%97%b4%e8%b0%8d%e7%9a%84%e7%9c%bc/" rel="noopener noreferrer" target="_blank">
    用“眼”识人：你想知道自己在一个“间谍”的眼中是什么样吗？
   </a>
   》
  </p>
  <p class="graf graf--p">
   ⚠️探究到对方的心理弱点，然后针对性的发起攻势，从心理基本面上彻底搞垮对方。
  </p>
  <p class="graf graf--p">
   于是解读对方的行为举止被称为间谍能力的准绳。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    举止比言语更容易暴露一个人的内心
   </strong>
   。观察并分析举止是间谍培训中的重点部分。
  </p>
  <p class="graf graf--p">
   反过来看，间谍依靠谎言维生；虽然他们可以使用高明的谎言做到逻辑上滴水不漏，但也很难控制自己的行为举止，尤其是微表情。
  </p>
  <p class="graf graf--p">
   从心理学角度上说，即便是再狡猾的间谍，在说谎或内心情绪波动时也无法逃脱微表情停留的极限，即1/25秒。这是人的生理因素决定的。
  </p>
  <p class="graf graf--p">
   因此，⚠️细致地观察对方的行为举止就能迅速了解对方的内心世界，从而根据对方的心理特点做出不同的应对策略。
  </p>
  <p class="graf graf--p">
   做到这点并不是太难，需要很多直觉方面的练习。
  </p>
  <p class="graf graf--p">
   请注意，⚠️人们往往会因此认为，培训政府间谍的教程也适合反间谍人员学习，理论上的确如此；但是，要做到成功的反间谍，必需能在间谍基础知识的层面上
   <strong class="markup--strong markup--p-strong">
    高出一步
   </strong>
   ，否则反而更容易被识破。
  </p>
  <p class="graf graf--p">
   一个著名的案例。很多人都知道有着“21世纪间谍第一人”之称的 Robert Hansen。当 Aldrich Ames 被中情局和联邦调查局抓获后，两大情报组织发现，所有疑点都指向 Robert Hansen。
  </p>
  <p class="graf graf--p">
   他们派出了最优秀的间谍来调查此案，但是多年后一无所获。联邦调查局感觉难以置信，因为他们对 Robert Hansen 进行了多次测谎，但每一次都被他通过了。
  </p>
  <p class="graf graf--p">
   直到继续对 Robert Hansen 进行长期监视才真的发现了蛛丝马迹。
  </p>
  <p class="graf graf--p">
   本来 Robert Hansen 觉得只要能稳住一段时间就可以安全渡过难关；正因为多年来中情局和联邦调查局都没有发现任何事，他心里就有些得意了。也正因此放松了警惕。
  </p>
  <p class="graf graf--p">
   在弗吉尼亚州维也纳小镇的公园里，他的确使用了基本反侦察技术对周边做了一些调查，确认安全后他将公文包放在了一个指定地点。
  </p>
  <p class="graf graf--p">
   完成这一切很顺利。原本没什么问题，但是他发现两个可疑的人站在公园一角。当时他心里一慌，下意识地将外套衣领向上提了一下，然后放慢脚步离开。
  </p>
  <p class="graf graf--p">
   结果就是这个拉衣领和突然放慢脚步的小动作将他出卖了。
  </p>
  <p class="graf graf--p">
   那两个可疑人的确是联邦调查局特工，他们其实
   <strong class="markup--strong markup--p-strong">
    并没有
   </strong>
   发现公文包，由于跟踪 Robert Hansen 很久了，他们也在心理上放松了警惕；但是，直到看到 Hansen 的两个小动作，立刻发现了异常。
  </p>
  <p class="graf graf--p">
   随后便找到了那个公文包。
  </p>
  <p class="graf graf--p">
   作为多年从事特工工作的人，Robert Hansen 已经养成了伪装自然走路姿态的习惯，所以忽然的放慢脚步其实是他作为间谍的本能反应；然而这点恰恰被同样是间谍的联邦调查局特工捕获了。
  </p>
  <p class="graf graf--p">
   所以，必须说，了解你的对手的知识和思考方式肯定是必要的，但如果你想要做到有效的反制，就必须比你的对手高出一筹。 ⚪️
  </p>
