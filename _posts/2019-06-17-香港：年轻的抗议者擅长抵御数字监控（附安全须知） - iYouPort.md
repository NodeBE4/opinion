  <p class="graf graf--p">
   香港熟悉技术的抗议者正在试图避免监视和未来可能的起诉。他们关闭了手机上的位置跟踪、用现金购买车票、并及时清除社交媒体对话……
  </p>
  <p class="graf graf--p">
   周三，警方用橡皮子弹和催泪瓦斯打破了反对中国引渡法的抗议人群，这是香港几十年来最大规模的抗议活动。
  </p>
  <p class="graf graf--p">
   街上的许多人主要是年轻人，他们在数字世界中长大，他们都非常清楚监视的危险，并尽可能避免留下数字足迹。
  </p>
  <p class="graf graf--p">
   抗议者担心引渡法将对自由产生毁灭性影响。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “即使我们没做任何激烈的事 —— 就像在网上谈论中国那样简单 —— 因为这种监视的存在也会让当局抓住我们”，这位25岁的抗议者说。
  </p>
  <p class="graf graf--p">
   一群示威者戴上口罩、护目镜、头盔和帽子 —— 既能保护自己免受催泪瓦斯、胡椒喷雾和橡皮子弹的伤害，也能躲避面部识别。
  </p>

  <p class="graf graf--p">
   许多人表示他们在加入抗议活动之前关闭了他们在手机上的位置跟踪并加强了他们的数字隐私设置，并且在他们离开示威现场后删除了社交媒体和消息应用上的对话和照片。
  </p>
  <p class="graf graf--p">
   由于抗议者使用现金购买门票，而不是使用香港无处不在的八达通卡 —— 避免行程被跟踪，因此在城市地铁站的售票台前排起了异乎寻常的长队。
  </p>
  <p class="graf graf--p">
   在一个 WhatsApp 通常为主要工具的城市，抗议者最近几天接受了加密消息传递应用 Telegram，认为它可以提供更好的网络保护（虽然并不是），也因为它允许更大的群体进行协调（这的确是）。
  </p>
  <p class="graf graf--p">
   星期四 Telegram 宣布它已成为重大网络攻击的目标，攻击来自中国的IP。该公司首席执行官认为此次袭击

  <p class="graf graf--p">
   抗议者越来越担心在线使用某些图片如黑白紫荆花可能会引起当局的注意，并且把它们删除了。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这反映了香港公民对这个政府的恐惧，”一位29岁的在教育部门工作的抗议者表示。
  </p>
  <p class="graf graf--p">
   许多人立即删除了“可以证明你在场的证据”。
  </p>
  <p class="graf graf--p">
   由于敏感性，与媒体交谈的示威者没有提供全名，并且至少都戴着面罩。
  </p>
  <p class="graf graf--p">
   27岁的 Heung 回到抗议活动现场加入清理工作，她在 Facebook 上发帖要求帮助。但是，她很担心呼吁志愿者会将她与抗议联系起来。
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p graf--startsWithDoubleQuote">
   “也许我今晚会删除帖子，”她说。 “我不想成为他们的嫌疑人之一。”
  </p>
  <p class="graf graf--p">
   虽然香港还没有遭遇中国大陆那种强大的监视，但北京的威胁令香港人非常担忧自由受到侵害。
  </p>
  <p class="graf graf--p">
   香港浸会大学高级新闻讲师 Bruce Lui 表示，人们对安全的认识有所提高，特别是中国无处不在的监控技术以及面部识别和其他跟踪方法的广泛应用。
  </p>

  <p class="graf graf--p">
   近几年来，由于几位书商在中国面临指控，以及2017年亿万富翁商人肖建华的经历，令香港人感到震惊。
  </p>
  <p class="graf graf--p">
   批评者说，引渡法如果获得通过，将允许这些迫害案公开合法地进行。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “一个月前香港还比较平静；但是瞬间，噩梦就会成为现实。谁知道后天它是不是会变得像新疆那样，因为事情可能会迅速被改变”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们正试图通过我们的隐私设置做得更好。但我们仍然认为自己是香港人，而不是中国人，所以我们仍然认为我们有权发表意见”，抗议者说。
  </p>

  </p>
  <p class="graf graf--p">
   相比下 Wire 是比较好的。而且可以设定时间彻底清除信息。虽然容量不如 Telegram，但可以多个群组，分区管理；每个群组固定管理员，这样也能便于避免非暴力直接行动被恶意渗透。最重要的是，
   <strong class="markup--strong markup--p-strong">
    不需要手机号注册
   </strong>
   —— ⚠️使用独立的替代邮箱注册，而不是您的常用邮箱。
  </p>
  <p class="graf graf--p">
   虽然每个人都参加了抗议，但并非所有人都面临同等的危险。当局对组织动员者的盯防强度远高于参与者。
  </p>
  <p class="graf graf--p">
   所以您应该实现评估自己的状况，才能做到
   <strong class="markup--strong markup--p-strong">
    合理的防御
   </strong>
   ，在这里看到如何评估《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%BA%86%E8%A7%A3%E8%87%AA%E5%B7%B1%E7%9A%84%E7%8A%B6%E5%86%B5%E6%98%AF%E4%BF%9D%E6%8A%A4%E5%AE%89%E5%85%A8%E7%9A%84%E7%AC%AC%E4%B8%80%E6%AD%A5/" href="https://www.iyouport.org/%E4%BA%86%E8%A7%A3%E8%87%AA%E5%B7%B1%E7%9A%84%E7%8A%B6%E5%86%B5%E6%98%AF%E4%BF%9D%E6%8A%A4%E5%AE%89%E5%85%A8%E7%9A%84%E7%AC%AC%E4%B8%80%E6%AD%A5/" rel="noopener noreferrer" target="_blank">
    威胁建模
   </a>
   》。日常也要做这项工作！尤其是在“少数派报告”技术加速流行的当下 —— 是否能自由取决于您是否被当局视为“重点盯防目标”。
  </p>
  <p class="graf graf--p">
   尽可能不要使用 Twitter/Facebook 等平台进行动员，除非确实必要，动员者必须能做到
   <strong class="markup--strong markup--p-strong">
    真正的匿名
   </strong>
   ，在这里看到步骤《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%BD%BF%E7%94%A8%E7%A4%BE%E4%BA%A4%E5%AA%92%E4%BD%93%E6%97%B6%E4%BD%A0%E5%BF%85%E9%A1%BB%E6%B3%A8%E6%84%8F%E4%BB%80%E4%B9%88%EF%BC%9F%e2%80%8a/" href="https://www.iyouport.org/%E4%BD%BF%E7%94%A8%E7%A4%BE%E4%BA%A4%E5%AA%92%E4%BD%93%E6%97%B6%E4%BD%A0%E5%BF%85%E9%A1%BB%E6%B3%A8%E6%84%8F%E4%BB%80%E4%B9%88%EF%BC%9F%e2%80%8a/" rel="noopener noreferrer" target="_blank">
    使用社交媒体时的注意事项
   </a>
   》。
  </p>
  <p class="graf graf--p">
   并且动员者应该
   <strong class="markup--strong markup--p-strong">
    尽可能晚
   </strong>
   地公布具体行动时间和地点，避免被提前警戒。
  </p>
  <p class="graf graf--p">
   尽可能不要携带自己的手机，以
   <strong class="markup--strong markup--p-strong">
    避免当局黄貂鱼
   </strong>
   的使用（在群体聚集的状态下使用黄貂鱼入侵是很有可能的）在这里看到应对方案《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E5%A6%82%E4%BD%95%E5%BA%94%E4%BB%98%E6%94%BF%E5%BA%9C%E4%BD%BF%E7%94%A8%E9%BB%84%E8%B2%82%E9%B1%BC%E5%AF%B9%E5%85%AC%E6%B0%91%E8%BF%9B%E8%A1%8C%E7%9A%84%E5%A4%A7%E8%A7%84%E6%A8%A1%E9%97%B4%E8%B0%8D/" href="https://www.iyouport.org/%E5%A6%82%E4%BD%95%E5%BA%94%E4%BB%98%E6%94%BF%E5%BA%9C%E4%BD%BF%E7%94%A8%E9%BB%84%E8%B2%82%E9%B1%BC%E5%AF%B9%E5%85%AC%E6%B0%91%E8%BF%9B%E8%A1%8C%E7%9A%84%E5%A4%A7%E8%A7%84%E6%A8%A1%E9%97%B4%E8%B0%8D/" rel="noopener noreferrer" target="_blank">
    如何政府使用抵御黄貂鱼的攻击
   </a>
   》。
  </p>
  <p class="graf graf--p">
   ⚠️这里是最基本的行动者注意事项《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E9%9B%86%E4%BC%9A%E5%8F%82%E4%B8%8E%E8%80%85%E6%9C%89%E5%BF%85%E8%A6%81%E4%BA%86%E8%A7%A3%E7%9A%84%E5%9F%BA%E6%9C%AC%E5%AE%89%E5%85%A8%E6%8E%AA%E6%96%BD/" href="https://www.iyouport.org/%E9%9B%86%E4%BC%9A%E5%8F%82%E4%B8%8E%E8%80%85%E6%9C%89%E5%BF%85%E8%A6%81%E4%BA%86%E8%A7%A3%E7%9A%84%E5%9F%BA%E6%9C%AC%E5%AE%89%E5%85%A8%E6%8E%AA%E6%96%BD/" rel="noopener noreferrer" target="_blank">
    集会参与者有必要了解的基本安全措施
   </a>
   》。
  </p>
  <p class="graf graf--p">
   我们在此后还将陆续发布一系列关于相关保护性措施和行动技巧的建议，您可以在本网菜单的“How To Win”部分看到这些内容；关于防卫的技巧可以在“Survival Skills”找到。
  </p>
  <p class="graf graf--p">
   最后《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E5%AE%89%E5%85%A8%E6%89%8B%E5%86%8C%EF%BC%9A%E8%BF%99%E9%87%8C%E6%98%AF%E4%BD%A0%E9%9C%80%E8%A6%81%E7%9A%84%E5%87%A0%E4%B9%8E%E6%89%80%E6%9C%89%E5%AE%89%E5%85%A8%E4%B8%8A%E7%BD%91%E5%B7%A5%E5%85%B7/" href="https://www.iyouport.org/%E5%AE%89%E5%85%A8%E6%89%8B%E5%86%8C%EF%BC%9A%E8%BF%99%E9%87%8C%E6%98%AF%E4%BD%A0%E9%9C%80%E8%A6%81%E7%9A%84%E5%87%A0%E4%B9%8E%E6%89%80%E6%9C%89%E5%AE%89%E5%85%A8%E4%B8%8A%E7%BD%91%E5%B7%A5%E5%85%B7/" rel="noopener noreferrer" target="_blank">
    安全手册：这里是你需要的几乎所有安全上网工具
   </a>
   》⚪️
  </p>
