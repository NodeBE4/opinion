  <p class="graf graf--p">
   如果您还没有读过第一部分，可以在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%8d%e5%86%8d%e5%90%83%e7%93%9c%ef%bc%9a%e6%8a%8a%e4%bd%a0%e7%9a%84%e5%9b%b4%e8%a7%82%e5%8f%98%e6%88%90%e5%88%a9%e5%99%a8-%e5%85%ac%e6%b0%91%e8%b0%83%e6%9f%a5%e6%8c%87%e5%8d%97%ef%bc%881/" href="https://www.iyouport.org/%e4%b8%8d%e5%86%8d%e5%90%83%e7%93%9c%ef%bc%9a%e6%8a%8a%e4%bd%a0%e7%9a%84%e5%9b%b4%e8%a7%82%e5%8f%98%e6%88%90%e5%88%a9%e5%99%a8-%e5%85%ac%e6%b0%91%e8%b0%83%e6%9f%a5%e6%8c%87%e5%8d%97%ef%bc%881/" rel="noopener noreferrer" target="_blank">
    不再吃瓜：把你的”围观”变成利器 — 公民调查指南（1）
   </a>
   》。
  </p>
  <p class="graf graf--p">
   公民记者也需要很多实地采访、参加会议、或者打电话，与此同时你需要录音，以避免错过或遗忘任何细节。
  </p>
  <p class="graf graf--p">
   此后你需要整理大量的音频，将他们转换成文字，可能会花费你很多精力。尤其是，如果对方讲方言或者外语，也许整理起来会更麻烦。
  </p>
  <p class="graf graf--p">
   本文将推荐一系列工具，来自哥伦比亚大学 Tow-Knight 创业新闻中心主任 Jeremy Caplan 的总结，可以加快您的速度，其中有一些是支持多语种和方言的，方便您接触更广泛的采访对象和演讲者。
  </p>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     选项1：在线转录音频或视频
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://otranscribe.com/" href="http://otranscribe.com/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     OTranscribe.com
    </strong>
   </a>
   是最简单的免费网络转录应用。
  </p>
  <p class="graf graf--p">
   首先上传音频或视频文件，或直接输入 Youtube 链接就行。然后在键入时以您喜欢的速度播放音频。
  </p>
  <p class="graf graf--p">
   键盘快捷键使您可以轻松地暂停、后退或插入时间戳，把听不清楚的部分标记下来，回头再核查。
  </p>
  <p class="graf graf--p">
   oTranscribe 可在浏览器中运行，因此您无需下载任何软件。
  </p>
  <p class="graf graf--p">
   还有一项名为
   <a class="markup--anchor markup--p-anchor" data-href="http://Transcribe.wreally.com" href="http://Transcribe.wreally.com" rel="nofollow noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Transcribe的
    </strong>
   </a>
   类似服务也是在浏览器中运行的，在为期一周的免费试用后，每年的许可费用为20美元。
  </p>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     选项2：使用自动转录服务
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://trint.com/" href="https://trint.com/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Trint.com
    </strong>
   </a>
   会获取您的音频或视频，并且快速进行相比下很准确的转录。
  </p>
  <p class="graf graf--p">
   我用两个录音测试了该服务。第一个是五分钟的录音，声音很干净。第二个是30秒的音频文件，嘈杂难以听清。结果是，在这两种情况下，转录质量都非常好，只需要进行很少的标点编辑即可。
  </p>
  <p class="graf graf--p">
   用于编辑转录文本的用户界面直观且设计合理。您可以导出多种格式的文字（包括比如 用于 YouTube 字幕的）。
  </p>
  <p class="graf graf--p">
   30分钟的免费转录后，费用为每小时15美元，或每月40美元。
  </p>
  <p class="graf graf--p">
   此外，由两名欧洲学生开发的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.happyscribe.co/" href="https://www.happyscribe.co/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Happy Scribe
    </strong>
   </a>
   是自动转录的另一个有趣的选择。这项服务的价格仅为每分钟 0.10 美元。
  </p>
  <p class="graf graf--p">
   Happy Scribe 允许80种语言和口音。该公司报告说，已有4,000名记者和研究人员使用了该服务。
  </p>
  <p class="graf graf--p">
   上传五分钟的测试记录后，我收到的自动转录效果非常好，只需要很少的编辑即可。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://speech-to-text-demo.mybluemix.net/" href="https://speech-to-text-demo.mybluemix.net/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Speech to Text
    </strong>
   </a>
  </p>
  <p class="graf graf--p">
   IBM人工智能系统Watson实验性推出的免费转录工具 Speech to Text。用户无论主动上传文件，或是实时语音对讲，它都能帮你即时转录。
  </p>
  <p class="graf graf--p">
   这个工具适用包括中文在内的八种语言，还可以识别不同人说话的声音，并且自动分段。
  </p>
  <p class="graf graf--p">
   但是，比起 Trint 或者 Happy Scribe、Speech to Text 来说，它比较难用。因为转录的过程中无法随意暂停并编辑转录出的文本。
  </p>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     选项3：使用应用程序生成快速转录
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://www.cassette.design/" href="http://www.cassette.design/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Cassette
    </strong>
   </a>
   是一款优雅的应用程序，由 Stanford d.school 和 UCLA Cognitive Science 的设计师与 David Kelley（IDEO和d.school的共同创始人）合作完成。
  </p>
  <p class="graf graf--p">
   它可以使用多种语言（西班牙语、法语、中文、日语、葡萄牙语）工作。
  </p>
  <p class="graf graf--p">
   创始人测试的准确度约为90％，并将每小时成本定为3美元。转录的前半小时是免费的。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://anchor.fm/" href="https://anchor.fm/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Anchor
    </strong>
   </a>
   是一种越来越流行的应用程序，现在，您可以使用它来创建音频视频的文本。对于社交共享特别有用。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://itunes.apple.com/us/app/steno-live-transcription-and-recording/id1205209916?mt=8" href="https://itunes.apple.com/us/app/steno-live-transcription-and-recording/id1205209916?mt=8" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Steno
    </strong>
   </a>
   是一个iOS应用程序，您可以用来录制演讲、访谈、会议或其他任何内容。有30分钟的免费转录。之后，您只需支付$ 0.99即可获得50分钟的转录时间，或支付$ 6.99来获得500分钟的转录时间。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://cogi.com/" href="https://cogi.com/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Cogi
    </strong>
   </a>
   是另一个有用的免费应用程序，可用于录音、通话记录和转录。
  </p>
  <p class="graf graf--p">
   它具有几个方便的功能，包括“倒带”功能。
  </p>
  <p class="graf graf--p">
   缺点可能是小贵。根据您的订阅计划，从每分钟1.20美元到2美元不等。
  </p>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     选项4：录制通话，然后转录
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://nonotes.com/" href="https://nonotes.com/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     NoNotes.com
    </strong>
   </a>
   可让您记录所需的呼出电话。每月收费8美元，然后，您可以支付$ .75 /分钟，在三天内转录任何电话（或支付额外费用以获得更快的服务）。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.tapeacall.com/" href="https://www.tapeacall.com/" rel="nofollow noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     TapeaCall.com
    </strong>
   </a>
   也提供类似的通话记录功能，每年10美元。您可以使用其他转录服务之一来转录录音。TapeaCall 将它们的功能与其他一些类似应用程序进行了比较，如下：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10365 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-6.png?resize=800%2C879&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-6.png?w=800&amp;ssl=1 800w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-6.png?resize=273%2C300&amp;ssl=1 273w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-6.png?resize=768%2C844&amp;ssl=1 768w" data-recalc-dims="1" height="879" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-6.png?resize=800%2C879&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>

   </noscript>

  <p class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     选项5：使用简单的听写工具
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   如果您使用 Chrome 浏览器，那么可以试试
   <a class="markup--anchor markup--p-anchor" data-href="https://www.google.com/docs/about/" href="https://www.google.com/docs/about/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Google Docs
    </strong>
   </a>
   它在转录语音方面做得很好。
  </p>
  <p class="graf graf--p">
   只需在任何文档上转到“工具”&gt;“
   <a class="markup--anchor markup--p-anchor" data-href="https://support.google.com/docs/answer/4492226?hl=en" href="https://support.google.com/docs/answer/4492226?hl=en" rel="nofollow noopener noreferrer" target="_blank">
    语音输入”
   </a>
   菜单命令即可。
  </p>
  <p class="graf graf--p">
   要进行翻译，您可以将其与同一菜单上的Google文档自动翻译功能结合使用（工具&gt;翻译文档）。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://speechnotes.co" href="http://speechnotes.co" rel="nofollow noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Speechnotes
    </strong>
   </a>
   是免费的，并且可以很好地自动和立即转录您所说的文本。
  </p>
  <p class="graf graf--p">
   口述文字后，您可以打印，通过电子邮件发送或将其上传到 Google 文档等。另外还有一个10美元的
   <a class="markup--anchor markup--p-anchor" data-href="https://chrome.google.com/webstore/detail/speechnotesx-voice-typing/oibmlbjklbogiccfokpgddgoleeiilid" href="https://chrome.google.com/webstore/detail/speechnotesx-voice-typing/oibmlbjklbogiccfokpgddgoleeiilid" rel="nofollow noopener noreferrer" target="_blank">
    Chrome扩展程序
   </a>
   ，可让您听写其他网站。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://speechlogger.appspot.com/en/" href="https://speechlogger.appspot.com/en/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Speechlogger
    </strong>
   </a>
   是一种用于识别和/或翻译口语单词的网络应用程序。
  </p>
  <p class="graf graf--p">
   还有，
   <a class="markup--anchor markup--p-anchor" data-href="http://www.autoedit.io/" href="http://www.autoedit.io/" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     autoEdit 2
    </strong>
   </a>
   是一个开源工具，可转录视频，然后让您通过编辑文本进行编辑。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.podcasterpro.com/" href="https://www.podcasterpro.com/" rel="nofollow noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     PodcasterPro
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    用来
   </strong>
   转录播客。除了播客的 RSS URL 之外您不需要任何技术信息。
  </p>
  <p class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     选项6：只能听政府报告/党代会之类的内容
    </strong>
   </span>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “讯飞语记”，这款APP的操作很简单，点击“创建笔记”，按下语音按钮，软件自动转换文字。
  </p>
  <p class="graf graf--p">
   官方说法是 “识别准确率高达95%，且支持普通话、英语、四川话、粤语等方言”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “讯飞语记” 转文字很快，如果语速适中、吐字清晰，识别率会大大提升，但是，英语的识别成功率低于中文。
  </p>
  <p class="graf graf--p">
   它缺点是语音稍停，转换也会随即停止，因此适合转换短小、连贯、清晰的语音。最好是录音完毕后，用另一台手机转文字，时刻注意重新开启语音按钮。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    尤其提醒注意的是，音频内容会上传到服务器，所以，任何重要的采访都不建议使用它
   </strong>
   。
  </p>
  <p class="graf graf--p">
   好了，现在您可以选择适合您的工具。下一部分内容我们将继续介绍让公民掌握调查优势的一些技巧。下次见。⚪️
  </p>
