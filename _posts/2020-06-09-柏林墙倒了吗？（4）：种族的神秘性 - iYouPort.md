   <p>
    I had heard nothing about this revolt against obedience in Leipzig, of course. It is another part of postwar German history that the two Germanys never actually shared
   </p>

  <p class="graf graf--p">
   法兰克福三分之一的人口出生在国外：南斯拉夫人、土耳其人、西班牙人、希腊人、意大利人以及140个更小的民族。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    他们的共同点是，都没有公民权利
   </strong>
   ：不能投票也不能获得公务职位，他们的地位 —— 即使是出生在德国 —— 一直是临时的。
  </p>
  <p class="graf graf--p">
   在法兰克福市中心的伏尔泰咖啡馆（Café Voltaire）的吧台，Rosa Wolf 向我解释了这些事实。
  </p>
  <p class="graf graf--p">
   她为市政府的 “多元文化局” 工作，她不知道自己部门所做的工作能否阻止这个城市
   <strong class="markup--strong markup--p-strong">
    正在高涨的仇外潮流
   </strong>
   。
  </p>
  <p class="graf graf--p">
   并且，如果右翼赢得了下届选举，他们可能把她的部门关闭。
  </p>
  <p class="graf graf--p">
   当地基民盟（CDU）的一个政客已经写了一篇文章，将寻求庇护的人和毒品泛滥、骗取福利联系在一起。 Rosa Wolf 公开要求控告他种族煽动。而市长批评她（而不是那个作者）破坏作为公务人员的政治中立性。
  </p>
  <p class="graf graf--p">
   Rosa 充满活力，不拘泥于教条，而又坚定不移。
  </p>
  <p class="graf graf--p">
   我们发现彼此拥有很多共同之处。我们都是60年代的孩子，我们活的时间足够长，足以让我们看到青年时期的偶像变成了博物馆的陈列物。
  </p>
  <p class="graf graf--p">
   伏尔泰咖啡馆曾经是法兰克福左翼反文化的中心，现在它为法兰克福出版商承办书籍首发式。
  </p>
  <p class="graf graf--p">
   从前，它是策划反对美军基地的狂热游行的地方，现在，我们身后墙上挂着 Rosa Luxemburg、Friedrich Engels、Karl Marx 的印制画像，
   <strong class="markup--strong markup--p-strong">
    似乎是对失落幻象的暧昧、伤感的纪念。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们曾经想改变德国社会”，她悲伤地说，似乎知道这样的想法现在看上去有多么奇怪，“我们需要它变得不那么顺从。但麻烦在于，我们成功了”。
  </p>
  <p class="graf graf--p">
   她以扭曲的细节将自己的困难和孩子们的困难分开。
   <strong class="markup--strong markup--p-strong">
    从1968年过来的父母经常发现他们说过的话重新浮现，萦绕着他们
   </strong>
   。
  </p>
  <p class="graf graf--p">
   当然，在莱比锡我没有听到任何关于这个反抗顺从的事情。它只是战后德国历史的另一部分。
   <strong class="markup--strong markup--p-strong">
    两个德国从未共同拥有过这一历史
   </strong>
   。
  </p>

  <blockquote class="graf graf--blockquote">
   <p>
    Freud says somewhere that there is knowing and there is knowing and they are not the same. Meaning: you can confront something in your head without confronting it in your heart, your guts. Working through something is what therapy is all about: moving knowledge from your head to your body, from intention to action
   </p>
  </blockquote>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们公然谴责父母是法西斯。我们从家里搬出去。那是非常痛苦的，但是我们想，不管如何，我们会为自己创造一种新的身份”。
  </p>
  <p class="graf graf--p">
   她继续说，对顺从的反抗是非常非常好的。问题是它导致70年代旧的青少年惩罚机制的崩溃。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “所以现在，这些16岁的少年在摩恩、罗斯托克和索林根用燃烧弹袭击庇护者或移民之家”，却得不到惩罚。他们只是嘲笑警察”。
  </p>
  <p class="graf graf--p">
   现在，她悲哀地发现，
   <strong class="markup--strong markup--p-strong">
    对顺从的反抗是对德国过去的反抗，但不是真正地面对过去。“我们太乐观了，我们从来没有解决过去”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   我俩默默地沉思了一会，想着 “解决” 过去意味如何。弗洛伊德在哪里说过，认知和认知是不一样的。意思是：你可能在头脑中应对某件事，而不是在你的内心和勇气中应对它。努力解决某事是治疗方法的全部：将知识从你的头脑中移动到躯体，从意图转向行动。
  </p>
  <p class="graf graf--p">
   （译者注：还记得
   <a class="markup--anchor markup--p-anchor" data-href="https://start.me/p/1kod2L/iyp-direct-action5" href="https://start.me/p/1kod2L/iyp-direct-action5" rel="noopener noreferrer" target="_blank">
    我们在列表-5
   </a>
   的简介中写的字吗？—— 直
   <strong class="markup--strong markup--p-strong">
    接行动是一种疗愈。可以说是最好的疗愈
   </strong>
   。中国社会当前的心理问题呈现爆发式增长，人们需要思考这个问题）
  </p>
  <p class="graf graf--p">
   但没人知道，与一个民族相应的疗法可以是什么。
  </p>
  <p class="graf graf--p">
   没有人知道，一个民族如何才能努力排除一切、达到那种深刻的认知。
  </p>
  <p class="graf graf--p">
   Rosa 意识到的是
   <strong class="markup--strong markup--p-strong">
    重新统一标志着某种终结，但是，远非一种开始
   </strong>
   。整个德国社会都还在努力就这个终结达成共识。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “每一个在战后成长起来的人都曾认为这种生活将会永远持续下去。你知道，那个清晰的、有组织的德国，那个相信每一样事情都可以被组织起来的德国。1989年以来，我们已经进入了真实的世界。我们经济的增长走到了终点。随着柏林墙的倒塌，我们不再能感受到曾经在我们的小花园中感受到的那种安全了。我们知道有种东西一劳永逸地结束了”。
  </p>
  <p class="graf graf--p">
   （译者注：请注意这是来自前西德公共事业部门员工的话）
  </p>
  <p class="graf graf--p">
   随着某种东西的终结，过去重现了，但并不像人们所期望和希望的那样。
  </p>
  <p class="graf graf--p">
   Rosa 举起双手，做了个俏皮兴奋的手势。“所有这些德国人现在都说 ‘40年来我们不得不沉默，现在，我们终于可以说，我们是德国人’。然后他们说：‘而且，我们出生得太晚，没有任何东西要隐瞒’。”
   <strong class="markup--strong markup--p-strong">
    她笑了，笑那些人、笑她自己曾经相信 “反对顺从的反抗” 将会令古老的德国永远噤声
   </strong>
   。
  </p>
  <p class="graf graf--p">
   她突然说：“
   <strong class="markup--strong markup--p-strong">
    我们犯了一个错误。我们从不讨论民族。我们相信我们已经超越了民族
   </strong>
   ”。她讽刺性地指了指身后那些名人画像，“我们曾经是国际主义者，记得吗？”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “你们也犯了一个错误”，这指的是德国的邻居们。“你们禁止我们解决我们的过去、与过去妥协，像一个民族应该做的那样”。
  </p>
  <p class="graf graf--p">
   我说：“你的意思是说，世界上其他地方从不允许德国人对他们应该感到自豪的东西自豪？”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “正是这样”，她说。
   <strong class="markup--strong markup--p-strong">
    惊讶地发现，在反对顺从的革命发生25年之后，在认为 “民族” 是一种对傻子和复仇主义者的归类的25年后，她得出了这样的结论
   </strong>
   。
  </p>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    K先生的白马
   </strong>
  </h3>
  <blockquote class="graf graf--blockquote">
   <p>
    关键任务应该是恢复人们的自信。我将这一族群界定为受伤的民族 …… 我的感觉是，只有这个族群恢复其作为一个民族的天然自尊，才会得到自在。当你拿走一个人的自尊时，他会深深受到伤害，对一个民族来说也是一样的 —— Elizabeth Noelle-Neumann 舆论研究者，德国，1992
   </p>
  </blockquote>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “为什么我们是欧洲唯一不能为自己自豪的民族？” K先生痛苦地问我。
  </p>
  <p class="graf graf--p">
   这个问题在我们之间久久回荡，我已经无数次
   <strong class="markup--strong markup--p-strong">
    在法兰克福
   </strong>
   听到这个问题。
  </p>
  <p class="graf graf--p">
   我们是在法兰克福城外的一个村子里。他的小平房就在这里。他盯着我，仿佛我应该为他不被允许以德国人的身份自豪而受到谴责。问题是，我并没有说过一句话。
  </p>
  <p class="graf graf--p">
   然后他站起来，轻快地问：“想看看我的马吗？” K先生热爱他的马，一匹白色的阿拉伯种马，他养在自己房子后面的小牧场里，位于法兰克福以北的一个有人居住的小村庄。
  </p>
  <p class="graf graf--p">
   他对马的热爱 ——
   <strong class="markup--strong markup--p-strong">
    浪漫的、极端的、纯真的 —— 正像他对德国的爱
   </strong>
   。
  </p>
  <p class="graf graf--p">
   他爬上他的战马光溜溜的背，告诉我他是如何读着卡尔麦（德国通俗文学作家）关于讲德语的牛仔驰骋远方的小说长大的。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我的马”，他说，“使我的梦想成真”。然后他驰骋而去。
  </p>
  <p class="graf graf--p">
   他想做一名牛仔，他想要这里变成讲德语的狂热西部。
  </p>
  <p class="graf graf--p">
   不幸的是，他已经是一个秃头的53岁的男人，一个退休的监狱官员，这里是一片小小的泥泞空地，四周围着栅栏。他没有 “疾驰而去消失在日落之下”，而是忧郁地转了回来和我继续说话。
  </p>

   跟他这个年纪大多数的德国人一样，他自豪地告诉我，他属于第一代足够幸运的德国人，出生得够晚。

   </strong>

  </p>
  <p class="graf graf--p">
   他只记得1945年冬天燃烧在柏林的红云。俄国人进入这个城市的时候，他和母亲逃到法兰克福北部青山之中的这个村子。现在，在监狱里服务25年之后，他作为右翼共和党候选人参加了在黑塞举行的地方政府选举。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    德国政治正在向右转，推动它转向的正是像K先生这样的人
   </strong>

  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">

  </p>
  <p class="graf graf--p">
   我反击说，
   <strong class="markup--strong markup--p-strong">
    是人的原因还是体制的原因呢？
   </strong>
   他回答道，绝对是人的原因。如果他们拥有德国人的素质就好了。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们是干净的民族，一个自尊的独立的民族”，他说。
  </p>
  <p class="graf graf--p">
   他对这座郊区的木屋特别引以为豪，那是他和他的妻子亲手打造的。房子里的墙上挂有大量自然风景画（他说 “来自我的祖先”），书柜里放着《布洛克豪斯百科全书》、以及收集的席勒和歌德的作品。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这是一个谦逊的布尔乔亚对某种德意志理念的纪念
   </strong>
   。
  </p>
  <p class="graf graf--p">
   离K先生漂亮的乡间别墅10公里远的地方，有一个为避难者设立的帐篷城市。他开车去那里，我们站在围墙外面，看里面成排的帐篷，泥地里用木板铺成的走道，还有一对忧郁的非洲人，在雨中互相给对方理发。
  </p>
  <p class="graf graf--p">

  </p>

   <strong class="markup--strong markup--p-strong">

   </strong>

  </p>
  <p class="graf graf--p">
   然后，他的温和又回来了。他的肩膀随着他奇特的、阴郁的笑容而抖动。他说，难民法是有一种 “德国自大狂的典型拼凑”。还有其他国家会梦想让自己成为全世界的福利官吗？
  </p>
  <p class="graf graf--p">
   他特别欢迎那种相信德国生活方式正遭受攻击的人。他就是这样，因为他知道他的党不会输。
   <strong class="markup--strong markup--p-strong">
    避难的宪法权利已经被取消，共和党已经因为高调发声而获得信任，而其他党派只敢悄悄地说话
   </strong>
   。

   但K先生说，收紧避难程序并没有触及问题的根本。现实问题是，德国人在他们自己的国家没有像是在家一样的感觉。法兰克福有30%的人口是外国人。“你现在知道我为什么要住在乡间了？”
  </p>

  </p>

   <strong class="markup--strong markup--p-strong">

   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们是德国人，他们是土耳其人”，他说。
  </p>
  <p class="graf graf--p">
   对于K先生观察这个世界的方式而言，种族主义是过于简单的一个词。关于 “他们” 嗅闻的方式、或者 “他们” 烹调的方式，一个种族主义者通常有
   <strong class="markup--strong markup--p-strong">
    某种幻觉
   </strong>
   。
  </p>
  <p class="graf graf--p">
   K先生似乎不会受到任何恐惧症的折磨。种族主义本质（Ethnic essentialism）对于他的立场而言是一个更有想象力的词，并且可能更为准确。他相信，身为德国人界定了他可能认知、理解或同情的限度。
  </p>
  <p class="graf graf--p">
   K先生并不孤独。一整个星期里，
   <strong class="markup--strong markup--p-strong">
    我遇到的都是自由派德国人。他们不会被视为和K先生一样的死硬，但是，由于社会势利和政治信念的混杂原因，他们说着和K先生一样的话
   </strong>
   。
  </p>

   他们都觉得，伴随着避难危机，自由主义的德国意识已经到了 “关键时刻”。“不能再这样下去了”，他们都这样对我说，“我们最终会不知道我们是谁”。
  </p>
  <p class="graf graf--p">
   K先生不会认可我说共和党人一直在孜孜不倦地为暴力煽风点火，然后再极力谴责暴力。谴责电影电视中的暴力，谴责我们的青年人中的道德丧失。
   <strong class="markup--strong markup--p-strong">
    不要将它归责于我，也不要将它归责于我的历史。
   </strong>
  </p>
  <p class="graf graf--p">
   为什么德国人要为这些袭击受到谴责？为什么你们不让我们与自己和平相处？K先生再次问我，似乎我是他的罪恶的来源或是他免罪的途径。
  </p>
  <p class="graf graf--p">
   我的印象是，德国人喜欢他们自己，喜欢自己的民族是件好事，
   <strong class="markup--strong markup--p-strong">
    只要他们接受那个确实存在的德国、拥有1945年之后边界的德国，那个容纳数百万外国人的德国
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    K先生的问题不在于他是一个民族主义者，而是，他是一个民种族主义者却事实上憎恨他所居住的德国
   </strong>
   。
  </p>

    政治白日梦
   </strong>

  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   他们登上巴士，前往临时难民营。你听他们窃窃私语，用的是俄语。但如果你靠近他们，觉得窘迫的人会陷入沉默，而机灵的人则大声告诉你：“我是德国人 Ich bin Deutsch”。
   <strong class="markup--strong markup--p-strong">
    用的是德语
   </strong>
   。
  </p>
  <p class="graf graf--p">
   即使已经通过了护照检查，他们还是有些怀疑；你一定是个警察，会因为他们声称自己是德国人但实际上不是德国人而把他们扔回飞机上去。
  </p>
  <p class="graf graf--p">
   在众多汹涌通过欧洲各关口的移民人潮中，这些家庭是最令人困惑的。他们是从被摧毁的东方帝国的每一个角落回流的德意志族人。每年10万人。
  </p>
  <p class="graf graf--p">
   德国是仅有的允许他们流失的族人回归的两个现代国家之一，另一个国家当然是以色列。
   <strong class="markup--strong markup--p-strong">
    两个相信民族性在于血缘的民族，都慢慢发现，血缘纽带真的会变得薄弱
   </strong>
   。
  </p>

  <blockquote class="graf graf--blockquote">
   <p>
    Such a scene brings you face-to-face with the mystery of ethnicity.
   </p>
  </blockquote>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “必须告诉他们用完后要冲马桶！” 住在法兰克福郊区的一个俄罗斯德意志人定居点里，一个德国社会工作者这样跟我说。
  </p>
  <p class="graf graf--p">
   从未听说过室内厕所、淋浴或私人浴室的家庭们必须被重新告知，“传统的北德意志新教徒就是干净和美德的同义词”。
  </p>
  <p class="graf graf--p">
   在这个定居点的餐厅，我看到一个交通警察耐心地告诉一群六七十岁的老人如何按照交通指示灯过马路。
   <strong class="markup--strong markup--p-strong">
    老人们无动于衷地听着，仿佛还是在听党和工厂领导人的讲话。
   </strong>
  </p>
  <p class="graf graf--p">
   警察让老人们站起来，练习把头转向左边，再转向右边，以便在他们过马路时不会被他们富有的 “房东” 的宝马车撞倒。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这样一个场景让你直面种族的神秘性
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这些人看上去像俄罗斯人，他们的习惯和心智还是苏维埃式的；他们的祖先300年前离开德国，越过当时神圣罗马帝国的东部斯拉夫边界地区，定居、殖民。异族婚姻淡化了与已消逝的过去联系的纽带。
  </p>
  <p class="graf graf--p">
   然而，如果你问他们，他们的 “故乡 Heimat” 在哪里？他们会环顾这个法兰克福旅馆的狭窄房间，并骄傲地说：“这里”。对于他们中年龄最大的人而言，他们是回家来等死的。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “一直以来我的梦想是死在这里”。Olga Oschetzki 说。她那双俄罗斯农民的大手交叠在面前的油毡桌布上。
  </p>
  <p class="graf graf--p">
   她说的德语有一种美丽的古典风味。这种德语被她的家族保留至今，就像一张古老的亚麻桌布。她很自豪从未丢下她的语言，即使斯大林在1937年枪毙了乌克兰日托米尔区所有的德语教师，并因为她与德国国防兵合作，战后将她流放到西伯利亚。
  </p>
  <p class="graf graf--p">
   当我唐突地说，有人会认为她更像俄罗斯人而不是德国人，她的蓝眼睛中对我闪现出怒火。
   <strong class="markup--strong markup--p-strong">
    “我知道我是谁”，她说
   </strong>
   。
  </p>
  <p class="graf graf--p">
   德意志人是那些少有的祖父母领着他们的子孙辈回家的移民之一。祖父母还会说德语，但他们的孩子们已经不会说了。
  </p>
  <p class="graf graf--p">
   绝大多数德国人可能希望他们的俄罗斯兄弟留在他们所在的地方，但是他们也知道，
   <strong class="markup--strong markup--p-strong">
    如果不让他们来，再过一代人，可能就没有德意志人能 “带回家”了。
   </strong>
  </p>
  <p class="graf graf--p">
   Olga 和她的女儿曾梦想的故乡与她们正冒险探究的这个故乡毫不相干。她们对一切都感到惊奇，例如，在商店里有20多种肥皂，车流如此野蛮。最重要的是，他们惊讶于那些外国人。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “我本以为我是来德国，然而却好像来到了土耳其”，Olga 的女儿皱了皱鼻子，失望地说。她是用俄语说的，因为她不会说德语
   </strong>
   。
  </p>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    最后的思考
   </strong>
  </h3>
  <p class="graf graf--p">
   基于对德国种族主义历史的愧疚和补偿，德国宪法郑重承诺，对所有政治迫害的受害者给予庇护。只不过它发现，无法依照自己的良心来信守承诺。
  </p>
  <p class="graf graf--p">
   问题不仅仅在于寻求庇护者的数量，还因为，战后的自由派德国很大程度上将自己视为后民族主义的、甚至是反民族主义的国家，
   <strong class="markup--strong markup--p-strong">
    以致于它发现自己无法清楚地定义与国际移民相关的民族主义利益。
   </strong>
  </p>
  <p class="graf graf--p">
   对于像 Rosa Wolf 这样的德国自由派，难民危机是战后第一次让他们感觉要被迫放弃后民族主义国家的乌托邦，更加冷静地思考德国的民族利益。
  </p>
  <p class="graf graf--p">
   换句话说，即使是对自由派而言，某些民族主义的对话也是不可避免的。他们不得不讨论配额、限制、遣返、对德国失业率的优先考虑。所有这些话题都非常自然，并不会因为其与右翼的关联而被视为耻辱。
  </p>
  <p class="graf graf--p">
   然而，时代对于德国右派仍然不利。德国城市中形成的多元文化社会，使得把德国人视作一个单一民族的 “人民” 显得荒谬，
   <strong class="markup--strong markup--p-strong">
    而东部和西部德国人之间的差异，令每一个人都想知道这种 “人民” 的身份中到底包括了什么
   </strong>
   。
  </p>
  <p class="graf graf--p">
   抛开这些现实情况对德国身份幻想的暴力入侵，界定德国身份的法律工具仍然受到种族民族主义历史的限制。
   <strong class="markup--strong markup--p-strong">
    公民身份的标准仍然是基于 “血统决定国籍” 准则的种族传承（ jus sanguinis）。由此导致在现实和种族幻想之间的矛盾，呈现出明显的错乱。
   </strong>
  </p>
  <p class="graf graf--p">
   对于绝大多数外部人士以及许多德国人来说，一个出生在德国、成长在德国的土耳其人，
   <strong class="markup--strong markup--p-strong">
    不能
   </strong>
   成为德国公民；而一个来自西伯利亚的、从未在这个国家居住过的、没有语言竞争力的 “德意志族人”，
   <strong class="markup--strong markup--p-strong">
    可以被授予公民身份
   </strong>
   ，获得广泛的定居协助。这是荒谬的。
  </p>
  <p class="graf graf--p">
   德国公民法的荒谬性，迫使保守的德国人在一个现代的多元文化世界中，
   <strong class="markup--strong markup--p-strong">
    面对用 “人民” 确定民族身份的不一致性
   </strong>
   。同时，自由派德国人发现。基于愧疚和补偿构建的一种后民族主义身份是不够的。
  </p>
  <p class="graf graf--p">
   似乎，双方都被现实的力量缓缓推动着，放弃各自的乌托邦。一边是一个德意志人的德国乌托邦，另一边则是一个向全世界开放的德国的乌托邦。
  </p>
  <p class="graf graf--p">
   同时，伴随着统一，德国最终从一个饥饿的民族转变为餍足的民族。它的边界安定了，它失去的族人正在归来。它现在的任务不是如一些自由派提出的从共同超越民族主义转到乏味的欧洲主义，
   <strong class="markup--strong markup--p-strong">
    而是，从其种族民族主义的过去转向一个公民民族主义的潜在未来。
   </strong>
  </p>
  <p class="graf graf--p">
   换言之，
   <strong class="markup--strong markup--p-strong">
    这可能是永远埋葬德国 “人民” 理念的时刻。用现实的语言来说。这将意味着从依民族的认同转向依国家的认同；也即，从基于种族身份幻想的公民身份转向基于民主价值认同的公民身份
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这项事业的主要障碍不在少数种族自身，而是，在于那些梦想一个从未存在的德意志的保守派，以及那些认为爱国主义是傻瓜才相信的自由派。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    结果是一个僵局。一个政治和文化的虚空。那儿本应找到一个可信赖的德意志形象。在这个虚空中本应有一个民族，但现在，只有一个国家。
   </strong>
  </p>
  <p class="graf graf--p">
   这个僵局每延续一天，这个国家的权威就弱化一天，而 Leo、Leech 和他们的极右翼朋友们就变得更加强大。
  </p>
  <p class="graf graf--p">
   我登上飞机飞往伦敦，想着德国人的反讽之处，他们创造了种族民族主义的理念，实际上却一点都不喜欢他们自己。
  </p>
  <p class="graf graf--p">
   我想知道，如果德国人真正平心静气地对待自己的民族，会是怎样？爱它本来的样子，而非它可能的样子？爱它所有的历史、所有的过去？那样的民族主义会是怎样？…… 那样的爱可能吗？⚪️
  </p>
