   <li>
    <span style="color: #00ccff;">
     <em>
      <strong>
       武器有很多种，心理学就是最重要的武器之一。
      </strong>
     </em>
    </span>
   </li>

  <p class="graf graf--p">
   你会撒谎吗？也许你觉得自己很会撒谎，能做到脸不红心不跳，这很棒。但本文想要解释的撒谎可能不是生活中骗女朋友那么简单，它是一种武器，是战略性的。被称为“战略性欺骗”。
  </p>
  <p class="graf graf--p">
   IYP 在隐私保护相关的内容中一直在强调，人们在社交网络上说得太多了，在完全无意识的情况下说出了很多有可能致命的信息，这是严重失误的。
  </p>
  <p class="graf graf--p">
   有些读者询问：“那么我们还能说点什么呢？总要说些什么的吧……”。没错，应该鼓励发言，发言至少有希望凝聚共同意志，甚至有希望实现联合，这是理论层面上的；但实际层面上还要考虑到，
   <strong class="markup--strong markup--p-strong">
    您希望联合的人群很可能并没有关注您的言论，反而是希望击败您的联盟的人一直在密切关注您说了什么。这种情况下您的发言基本是在给敌人递手榴弹
   </strong>
   。
  </p>
  <p class="graf graf--p">
   那是那句话，反抗和异议完全不同；反抗是对战，不仅需要技能技巧，还需要一系列战术策略。
  </p>

  <p class="graf graf--p">
   人类有着长达五千年的战争史，战略性欺骗的历史和战争史一样久远。如果说兵力和武器占据三分之一的分量，那么谋士则占据三分之二。
  </p>
  <p class="graf graf--p">
   到21世纪高技术对战时代，欺骗的战略更是无处不在，几乎所有战争奇观中都包括谋略家的出色表演。
  </p>
  <p class="graf graf--p">
   欺骗手段历史悠久，但一直没有明确的定义。古代中国人称之为“诡道”，古代欧洲人称之为“Stratagem 谋略”；不论叫它什么，它的基础思考框架、目标和原则都是相似的。
  </p>
  <p class="graf graf--p">
   如今，
   <strong class="markup--strong markup--p-strong">
    欺骗已经远远超越了军事范畴，它就在普通生活中，是社交工程的一部分
   </strong>
   ，可以称得上社会现象。
  </p>
  <h4 class="graf graf--h4">
   <strong class="markup--strong markup--h4-strong">
    —— 概述 ——
   </strong>
  </h4>
  <p class="graf graf--p">
   通常情况下，欺骗有两层含义：1、隐真；2、示假。
  </p>
  <p class="graf graf--p">
   隐真意味着严格保密，或者成为“拒止”；示假则是欺骗。
  </p>
  <p class="graf graf--p">
   如果只是“掩饰” — 即 保密和伪装，那只能是一种
   <strong class="markup--strong markup--p-strong">
    消极的欺骗
   </strong>
   —— 通过消极的手段阻止对手接近秘密；但战略性欺骗的第二层含义则是“制造谎言”，使其成为
   <strong class="markup--strong markup--p-strong">
    积极的战术
   </strong>
   。制造谎言使对手远离事实，于是更具主动性。
  </p>
  <p class="graf graf--p">
   不论是对抗行动还是日常生活，欺骗都包括上述两方面，既有隐真，也有示假。隐真和示假侧重点不同，但相辅相成。
  </p>
  <p class="graf graf--p">
   美国蒙特雷海军研究生院教授提出过一个概念：“战略拒止和欺骗 Strategic Denial and Deception”。
   <strong class="markup--strong markup--p-strong">
    其中拒止就是信息战
   </strong>
   ，欺骗对手使其相信错误的信息、虚假的图像，误导对方情报机构和决策者，从而导致其作出符合己方需求的决策。
  </p>
  <p class="graf graf--p">
   在这里看到一个中国有可能即将实施的现代战争的典型欺骗战术，《
   <a href="https://www.iyouport.org/%E8%99%9A%E5%81%87%E5%9C%B0%E7%90%83-video/" rel="noopener noreferrer" target="_blank">
    虚假地球
   </a>
   》
  </p>
  <p>
   <img alt="" class="aligncenter size-large wp-image-3818 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-2.jpg?resize=1024%2C768&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-2.jpg?resize=1024%2C768&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-2.jpg?resize=300%2C225&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-2.jpg?resize=768%2C576&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-2.jpg?resize=1100%2C825&amp;ssl=1 1100w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-2.jpg?w=1280&amp;ssl=1 1280w" data-recalc-dims="1" height="768" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-2.jpg?resize=1024%2C768&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>

  <p class="graf graf--p">
   战略性欺骗涉及的概念和理论非常多，
   <strong class="markup--strong markup--p-strong">
    包括并不限于：情报、反情报、心理战、情报认知和分析、隐蔽、伪装、隐蔽行动（秘密行动）、非常规战争、危机管理、预警、信息战……等等
   </strong>
   。其中大部分我们都有专门介绍。
  </p>
  <p class="graf graf--p">
   IYP重点介绍并演示了很多关于心理战、社交工程、OSINT情报分析的技术和案例，其中心理学是尤其侧重的，因为你必须知道你获取的情报究竟是不是真的。
  </p>
  <p class="graf graf--p">
   是否真实与你获取该情报的困难程度完全无关 —— 这句话非常重要。
  </p>
  <p class="graf graf--p">
   ⚠️人们往往会下意识地认为“千辛万苦”获得的东西肯定是“真的”，欺骗者恰好可以利用这点；并且社交工程学利用的所有人性弱点在此都是高效的。
  </p>
  <p class="graf graf--p">
   情报人员非常容易受到战略性欺骗的攻击，这与情报获取的能力并不一致。让情报具备价值的关键在于分析，于是对情报分析师的培训一直在专注于心理学角度。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    关于情报分析心理学的知识同样应用于反社交工程攻击、真假消息辨别技巧
   </strong>
   ；也就是说，不论是攻还是防，重点都是一样的，比的是其中哪方能技高一筹。如果您对此感兴趣，我们推荐几本书和资料：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    Barton Whaley &lt;Codeword Barbarossa&gt; Cambridge: MIT Press, 1974;
   </li>
   <li class="graf graf--li">
    &lt;Stratagem: Deception and Surprise in War&gt; Cambridge, MA: MIT Press;
   </li>
   <li class="graf graf--li">
    Richard Heuer, &lt;Psychology of Intelligence Analysis&gt; Washington, D.C.: Center for the study of Intelligence, CIA, 1999;
   </li>
   <li class="graf graf--li">
    Cynthia M. Grabo, &lt;Anticipating Surprise: Analysis for Strategic Warning&gt; Washington, D.C.: Joint Military Intelligence College, 2002;
   </li>
   <li class="graf graf--li">
    Abram N. Shulsky &amp; Gary J. Schmitt, &lt;Silent Warfare: Understanding the world of Intelligence&gt; Potomac Books, Inc., 2002;
   </li>
   <li class="graf graf--li">
    Major Mark Johnson &amp; Major Jessica Meyeraan, &lt;Military Deception: Hiding The Real, Showing The Fake&gt;Joint Forces Staff College, March 7, 2003.
   </li>
  </ul>
  <p class="graf graf--p">
   战略欺骗研究在美国是一门显学，大多数资料都来自美国；当然，二战时期欧洲游击队的策略也是不俗的。
  </p>
  <p class="graf graf--p">
   美军将欺骗分为五大类：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    战略级（Strategic MILDEC）：使敌方作出有利于己方目的、策略和行动的军事决策；
   </li>
   <li class="graf graf--li">
    军种级（Service MILDEC）：保护和增强军种部队的战斗力；
   </li>
   <li class="graf graf--li">
    战役级（Operational MILDEC）：阻止敌方作出或实践正确的决策；
   </li>
   <li class="graf graf--li">
    战术级（Tactical MILDEC）：阻止敌方作出进攻相关的正确决策 ；
   </li>
   <li class="graf graf--li">
    支援作战保密（MILDEC in support of OPSEC）：削弱敌方发现己方作战保密薄弱环节的能力。
   </li>
  </ul>
  <h4 class="graf graf--h4">
   <strong class="markup--strong markup--h4-strong">
    —— 战略性欺骗和其他秘密行动的关系 ——
   </strong>
  </h4>
  <p class="graf graf--p">
   欺骗就是用己方精心设计和选取的信息操控对手的思维模式，并引导其采取错误的行动，以夺取信息权。
  </p>
  <p class="graf graf--p">
   于是欺骗往往是信息战的重要形式和核心能力；换句话说，
   <strong class="markup--strong markup--p-strong">
    只要能抵制欺骗，就能赢得信息战的大部分胜利。
   </strong>
  </p>
  <p class="graf graf--p">
   此外欺骗也与电子战、心理战、作战保密息息相关；它们互相作用，也有时会存在潜在的冲突。理清这些问题对于防骗来说是很重要的，你能知道对手在什么情况下最可能使用什么伎俩。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、欺骗和信息战
   </strong>
  </p>
  <p class="graf graf--p">
   信息战（Information Operations）是指综合使用电子战、网络战、心理战、战略性欺骗和作战保密的核心能力；在特定支援能力和相关能力的配合下，影响、扰乱、破坏或侵占对手的人工或自动决策能力，同时保护己方行动。
  </p>
  <p class="graf graf--p">
   信息战的概念来自美国，
   <strong class="markup--strong markup--p-strong">
    但如今中国采取的信息战手段已经引起全球重视
   </strong>
   。在这里看到关于中国信息战的解释 “
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Chinese_Information_Operations_and_Information_Warfare" href="https://en.wikipedia.org/wiki/Chinese_Information_Operations_and_Information_Warfare" rel="noopener noreferrer" target="_blank">
    Chinese Information Operations and Information Warfare
   </a>
   ”。
  </p>
  <p class="graf graf--p">
   信息战顾名思义，是以信息和信息的使用为武器的，于是成功的信息战离不开欺骗。
  </p>
  <p class="graf graf--p">
   利用对手的信息系统和能力，将编制好的虚假信息传递给对方决策者。从计划到实施都是在信息环境下完成的，要为对手设置一个虚假和错误的信息环境。
  </p>
  <p class="graf graf--p">
   于是，
   <strong class="markup--strong markup--p-strong">
    其中关键是了解对手的思维方式、计划习惯、以及在工作中实施信息管理的手段。反过来看，如果你不想中招，上述的一切都应该严格保密，让对手摸不清门道，他们的策略就会失败
   </strong>
   。
  </p>
  <p class="graf graf--p">
   你自己说得越多，就越危险。
  </p>

    2、欺骗和电子战
   </strong>
  </p>
  <p class="graf graf--p">
   电子战（Electronic Warfare）是指使用电磁能或定向能控制电磁频谱攻击敌人的军事行动。同样是信息作战的核心能力。
  </p>
  <p class="graf graf--p">
   利用电磁能量和电磁频谱，向对手传送错误信息、入侵对手的传输路径、阻止对手的电子战支援。
  </p>
  <p class="graf graf--p">
   欺骗为电子战提供了重要的支援，如 使用电磁冒充、模拟大规模军事部署 ，从而使对手高估己方的能力，达到威胁的作用；或者掩盖己方能力，使对手轻敌。
  </p>
  <p class="graf graf--p">
   电子战本身也可以支援欺骗行动，包括提供重要的实施渠道和反馈渠道（电磁频谱）；通过干扰或攻击对手通信和ISR系统，塑造和控制对手获取信息的能力，散播欺骗性信息。
  </p>

  <h4 class="graf graf--h4">
   <strong class="markup--strong markup--h4-strong">
    —— 情报和反情报 ——
   </strong>
  </h4>
  <p class="graf graf--p">
   战略性欺骗对抗中有欺骗者和被欺骗者两部分，而情报 工作中也有以获取情报为目的的对外情报、和以维护安全为目的的反情报。
  </p>
  <p class="graf graf--p">
   情报在战略性欺骗中具有决定性作用，主要体现在以下几个方面：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、情报是制定合理欺骗计划的前提
   </strong>
  </p>
  <p class="graf graf--p">
   欺骗是一种心理干预，这种心理干预的主体和工具就是情报机构主导下的信息系统，它的受众则是潜在的被骗者。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    欺骗者为被骗者构建了一个与真相相反的信息系统
   </strong>
   ，在此过程中，欺骗者准备描述的图像、提供的信息，
   <strong class="markup--strong markup--p-strong">
    必须在欺骗对象眼中看起来具有足够的可信度。这是成功欺骗的一个重要原则
   </strong>
   。
  </p>
  <p class="graf graf--p">
   就如小说和生活的区别，生活可以离谱，但小说必须符合大众熟知的逻辑；真相可以完全出乎意料，但欺骗必须符合目标受众的既有认知。
  </p>
  <p class="graf graf--p">
   再一次，知己知彼是重要基础。
  </p>
  <p class="graf graf--p">
   欺骗计划是不是有效、能不能打动对手的心，
   <strong class="markup--strong markup--p-strong">
    完全依赖于分析人员对对手的了解
   </strong>
   ：什么能影响对手的决策？对手倾向于如何看待时局？对手的决策流程是什么？对方最担心的是什么？……等等，都要具备充分的答案。
  </p>
  <p class="graf graf--p">
   情报工作最重要的内容是了解对手的预期和倾向。
   <strong class="markup--strong markup--p-strong">
    实验心理学证明，思维主体的倾向越强烈，就越容易曲解和误读与其不一致的观点
   </strong>
   。
  </p>
  <p class="graf graf--p">
   简单说，⚠️心理干预的方向必须迎合被骗者的偏见和心理预期。
  </p>
  <p class="graf graf--p">
   显然，若能探究对手的内心意图，就会更容易获取对手的信任。
  </p>
  <p class="graf graf--p">
   纳粹宣扬苏德友好的欺骗计划恰恰符合斯大林的先入之见。德国利用《苏德互不侵犯条约》宣扬友好关系，就在巴巴罗萨行动前不久，德国最高统帅还在大张旗鼓地迎接苏联贵宾；与此同时到处散播消息，称斯大林即将德国南部的巴登疗养，给人们一种苏德真的友好的虚假印象。
  </p>
  <p class="graf graf--p">
   简单说，⚠️欺骗者的任务就是强化被骗者的内心倾向，并尽力减少与之相矛盾的线索，使其在不知不觉中与欺骗者合作。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、情报是进行欺骗评估的核心
   </strong>
  </p>
  <p class="graf graf--p">
   成功的战略性欺骗需要密切关注欺骗对象的心理认知和思维方式；了解欺骗对象的背景情况和心理情况；了解其对力量和行动方案的认知；评估其在不同情境下的行为、以及对欺骗计划可能作出的反应。
  </p>
  <p class="graf graf--p">
   欺骗方需要对其目的、目标、对象、情节、和手段，进行
   <strong class="markup--strong markup--p-strong">
    不间断地再评估
   </strong>
   ，这一过程中情报起着关键作用。
  </p>
  <p class="graf graf--p">
   欺骗评估按内容可分为可行性评估、效果评估和风险评估三种。其中效果部分包含适应性，即 欺骗计划适应不同要求的能力，比如是否可以对预期外的状况作出安全并适宜的反应（简称 planB）。
  </p>
  <p class="graf graf--p">
   欺骗也存在风险，评估需要考虑：失败的风险、暴露的风险和非故意的风险。于是欺骗行动需要搭配稳定可靠的情报支持。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、情报也是识破欺骗阴谋的唯一途径
   </strong>
  </p>
  <p class="graf graf--p">
   反欺骗包括防御性反欺骗、进攻性反欺骗、和利用性反欺骗。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    防御性反欺骗
   </strong>
   指保护己方决策者免受对手的欺骗，从而有利于掌控局势；
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    进攻性反欺骗
   </strong>
   指迫使对手暴露其真实目的，并假装尚未发现对手的欺骗行为，诱使对方继续投入欺骗资源，继续实施事实上已经被揭穿的欺骗；
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    利用性反欺骗
   </strong>
   是指阻挠对方阴谋的同时，利用欺骗战术，影响对方决策。
  </p>
  <p class="graf graf--p">
   在反欺骗行动中，情报发挥作用的方面包括：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    了解对手的欺骗历史，包括使用欺骗的时机、模式、手段 、渠道 等等；
   </li>
   <li class="graf graf--li">
    发现对手的欺骗行为，通过反映对手已经采取、正在采取或即将采取的措施，揭露对手的欺骗企图、行动、情节和核心信息；
   </li>
   <li class="graf graf--li">
    为实施反欺骗提供依据，即 不断了解对手的情况，使己方反欺骗行动更具针对性、合理性。
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    反情报与战略性欺骗 ——
   </strong>
  </p>
  <p class="graf graf--p">
   上面推荐的一本书&lt;Silent Warfare: Understanding the world of Intelligence&gt; 中就是将欺骗作为反情报方式的。
  </p>
  <p class="graf graf--p">
   反情报就是反监视、反侦查，即 避免自己被对手更多了解和分析。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    战略性欺骗本身是一种积极的反情报工作
   </strong>
   。历史上著名的战略性欺骗都是反情报部门主导的。
  </p>
  <p class="graf graf--p">
   反情报在战略性欺骗中的作用有以下几点：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、反情报是确定欺骗渠道、评估欺骗效果的主要途径
   </strong>
  </p>
  <p class="graf graf--p">
   作为对对手的一种心理干预，战略性欺骗实践者必须确保把事先设计的虚构图像或信息传递给对手。
  </p>
  <p class="graf graf--p">
   传递假象的渠道必须通畅；欺骗者必须知道传递的信息可能无法按预期方式抵达对方手里，对方也可能不按照预期方式对信息进行解读。
  </p>
  <p class="graf graf--p">
   因此如上述所强调的，实施者应当在保持自己的一切不被对手了解到的情况下，能了解对手的一切，使信息尽可能被对手接收、解读，从而达到欺骗的目的。
  </p>
  <p class="graf graf--p">
   这一部分的工作就属于反情报工作。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、反情报是建立信息反馈渠道、控制对手情报行为的有力武器
   </strong>
  </p>
  <p class="graf graf--p">
   与信息发布渠道同等重要的是信息反馈渠道。
  </p>
  <p class="graf graf--p">
   信息发出后对手是否正确接收到了欺骗信息、对该信息做出了怎样的反应，这些都需要情报机构去收集。
  </p>
  <p class="graf graf--p">
   只有在充分反馈的基础上，欺骗者才能知道是否可以继续行事、是否应该调整欺骗计划，或者中止该计划。
  </p>
  <p class="graf graf--p">
   反馈收集可以从观察对手的行为入手，但更好的办法就是攻破对手对的情报营。
  </p>
  <p class="graf graf--p">
   二战时期，英国控制了在英国活动的德国间谍，建立了双十体系 XX System，成功了解到德国对盟军欺骗行为的认知，并据此调整了欺骗计划。
  </p>
  <p class="graf graf--p">
   当时管理双十体系的军情五处官员列出了七个目标，其中包括控制对手的谍报系统、从对手提出的问题中猜测对手的计划和意图、通过传递虚假信息对对手施加影响、以及实施欺骗的计划等等。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、反情报是欺骗成功的屏障
   </strong>
  </p>
  <p class="graf graf--p">
   欺骗是人类生活中最普遍的现象，不仅在对战中存在，在日常生活中也存在，婴儿在不会说话之前就会欺骗。
  </p>
  <p class="graf graf--p">
   比如魔术就是一种欺骗行为，如果你揭穿底牌它就无法继续下去了。战略性欺骗也是如此。如果实施欺骗过程中真相外露，一切都会结束。
  </p>
  <p class="graf graf--p">
   因此
   <strong class="markup--strong markup--p-strong">
    保密是欺骗的关键部分，是欺骗得以成功的大前提
   </strong>
   ；也就是说，在实施欺骗的过程中必须实施严格的拒止措施。
  </p>
  <p>
   <img alt="" class="aligncenter size-large wp-image-3820 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-4.jpg?resize=1024%2C579&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-4.jpg?resize=1024%2C579&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-4.jpg?resize=300%2C170&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-4.jpg?resize=768%2C434&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-4.jpg?resize=1100%2C622&amp;ssl=1 1100w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-4.jpg?w=1280&amp;ssl=1 1280w" data-recalc-dims="1" height="579" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-4.jpg?resize=1024%2C579&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>
   <noscript>
    <img alt="" class="aligncenter size-large wp-image-3820" data-recalc-dims="1" height="579" sizes="(max-width: 1024px) 100vw, 1024px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-4.jpg?resize=1024%2C579&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-4.jpg?resize=1024%2C579&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-4.jpg?resize=300%2C170&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-4.jpg?resize=768%2C434&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-4.jpg?resize=1100%2C622&amp;ssl=1 1100w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-4.jpg?w=1280&amp;ssl=1 1280w" width="1024"/>
   </noscript>
  </p>
  <h4 class="graf graf--h4">
   <strong class="markup--strong markup--h4-strong">
    —— 战略性欺骗是一门艺术 ——
   </strong>
  </h4>
  <p class="graf graf--p">
   没有任何东西能比得上战略性欺骗对情报分析难度的增加；面对对手的欺骗行为，即便是优秀的分析师也有可能出现失误。
  </p>
  <p class="graf graf--p">
   但是，想要识破战略性欺骗，首先就要深刻了解它。
  </p>
  <p class="graf graf--p">
   战略性欺骗是一门艺术，没有特定的行为规则和必胜要领，实施者总是能想办法创造出奇迹。
  </p>
  <p class="graf graf--p">
   成功的欺骗大师之间没有普遍的共性。除了创造力。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、先入之见
   </strong>
  </p>
  <p class="graf graf--p">
   首先不要把对手当成傻子，这会导致轻敌的失误。
  </p>
  <p class="graf graf--p">
   一般情况下决策者都会对局势有一个总体的认识，也就是对时局的假设性判断。如上所述，大多数成功的欺骗都是顺应被骗者先入之见的。
  </p>
  <p class="graf graf--p">
   但是，要知道，
   <strong class="markup--strong markup--p-strong">
    先入之见并非总是偏见和感性偏执的，正相反，最常见的先入之见是对手出于理性的逻辑思维
   </strong>
   。
  </p>
  <p class="graf graf--p">
   1973年第四次中东战争爆发之前，以色列摩萨德对时局有两个判断：埃及在取得空中优势前不会发动进攻；如果没有埃及的配合，叙利亚没能力发动战争。
  </p>
  <p class="graf graf--p">
   这两个判断都符合阿以双方的力量对比，是正确的局势指针。在后来的战争准备中，埃及军队就有意识地强化以色列的认知；有关军备建设落后的消息被不断放出，从而使以色列人相信埃及空军“不堪一击”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、对手的思维方式
   </strong>
  </p>
  <p class="graf graf--p">
   在制定欺骗计划时，欺骗者必须充分考虑到对方决策者的思维方式，你越是了解对手，你的成功机会就越大。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    欺骗艺术的一部分就是要学会从对手的角度思考问题 —— 即 IYP曾经强调过的，要想防止被骇，那就要像骇客一样思考
   </strong>
   。
  </p>
  <p class="graf graf--p">
   所以，诱饵的设计必须准确地掌握有关对手的思考模式以及信任状况方面的情报。
  </p>
  <p class="graf graf--p">
   思考方式往往取决于对手的性格，性格激烈、自主强的决策者往往独断专行，笃信自己的世界观，完全不接受与之相左的信息。面对这样的对手，战略性欺骗必须能按照他们的偏见制造诱饵。
  </p>
  <p class="graf graf--p">
   希特勒就是这样的人。希特勒坚信盟军登陆的最佳地点是加来海峡，盟军就不断释放加来才是真正登陆地点的假消息；在诺曼底登陆后，盟军还通过双重间谍发布假消息，把诺曼底伪装成佯攻，最终取得了完胜。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、影响对手决策的外在因素
   </strong>
  </p>
  <p class="graf graf--p">
   在决策过程中，除了决策者的主观因素外，外在因素也很重要。
  </p>
  <p class="graf graf--p">
   比如不可避免的官僚体系，这类外在因素可能会带来无法控制的意外情况。
  </p>
  <p class="graf graf--p">
   1998年印度在实施核试验之前采取了一场战略性拒止欺骗行动，印度利用云层掩护开展行动，以躲避美国的卫星侦查，并将易于泄密的电缆深埋，以防止美国情报人员获知。
  </p>
  <p class="graf graf--p">
   同时印度还使用外交手段进行欺骗，使外界认为印度并不急于进行核试验。
  </p>
  <p class="graf graf--p">
   事实上，美国在核试验前一周就获得了图像资料，但这些资料没能被及时处理，政府忙于应付其他问题，也没有提出任何相关情报需求，就连外交官也对自己说服印度的前景充满信心。
  </p>
  <p class="graf graf--p">
   在此案例中就是美国的官僚体系影响了情报的预警能力，导致欺骗成功。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    欺骗的目的是误导对手的决策 ——
   </strong>
  </p>
  <p class="graf graf--p">
   欺骗者需要编制一个完美的谎言，让对方相信，让局面陷入更深层的迷雾，使对方误入歧途。
  </p>
  <p class="graf graf--p">
   狼来了的故事在中国家喻户晓。战争中也是非常常见的。
  </p>
  <p class="graf graf--p">
   还是第四次中东战争，埃及多次征召预备役人员，然后又分批复员，频繁的动作让以色列很困惑，以色列无法确定埃及是否真的在准备战斗，于是也被迫进行了两次动员，损失了数千万美元。
  </p>
  <p class="graf graf--p">
   结果是，当埃及进行真正的动员时，以色列人已经麻木了，不愿意再“无端地消耗金钱”，埃及的欺骗就成功了。
  </p>
  <p class="graf graf--p">
   需要知道，埃及的目的是让以色列懈于防备，而不仅仅是战略误判。
  </p>
  <p class="graf graf--p">
   即
   <strong class="markup--strong markup--p-strong">
    战略性欺骗并不仅仅限于让对手错误评估局势，它的根本目的是确保对手按照己方的预期进行决策和行动
   </strong>
   。
  </p>
  <p class="graf graf--p">
   也就是说，你的目的并非让对手怎么想，而是要让他怎么做。这是欺骗的首要戒律。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    —— 未完待续 ——
   </em>
  </p>
