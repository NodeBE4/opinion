   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     TikTok已经受到年轻一代的欢迎，正在有越来越多的用户涌入该平台。该应用程序允许用户记录和共享短视频以及评论其他用户的视频。
    </span>
   </span>
  </p>
  <p class="graf graf--p">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     也存在该应用程序的浏览器版本，但是它的功能要精简简短，并且需要一点知识才能变通并充分利用它。
    </span>
   </span>
  </p>
  <p class="graf graf--p">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     调查人员应该做到这点，因为
    </span>

   <strong class="markup--strong markup--p-strong">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">

    </span>
   </strong>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    <span style="vertical-align: inherit;">

     </span>
    </span>
   </strong>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     。

   </span>

  <p class="graf graf--p">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     用户可以对视频发表评论，这可能有助于进一步识别最初发布视频的用户。
    </span>
   </span>
  </p>
  <p class="graf graf--p">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     可以查看经常在该用户的视频上发表评论的个人用户资料，查看他们是否泄漏了任何位置或标识性信息。这是一个扩展追踪。
    </span>
   </span>
  </p>
  <p class="graf graf--p">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     评论中甚至还可以显示有关该视频的更多信息，例如有人问该视频在哪里拍摄的，例如在上面的示例中，有多个评论表明该视频是在目标商店内拍摄的。
    </span>
   </span>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     <span style="vertical-align: inherit;">
      <span style="vertical-align: inherit;">
       最后
      </span>
     </span>
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     希望您现在已经准备好开始您的第一次TikTok调查了，别忘了上述要点。
    </span>
   </span>
  </p>
  <p class="graf graf--p">
   <span style="vertical-align: inherit;">
