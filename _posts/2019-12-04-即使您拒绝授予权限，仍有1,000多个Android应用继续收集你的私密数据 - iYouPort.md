  <p class="graf graf--p">
   Android 应用的权限旨在成为您的设备放弃了多少数据的守门人 —— 至少它们自己是这么承诺的。
  </p>
  <p class="graf graf--p">
   如果您不希望手电筒应用程序能够读取您的通话记录，您应该能够拒绝该访问。安全倡导者一直在这样告诉大家。
  </p>

  <p class="graf graf--p">
   但是，即使您拒绝，许多应用也会找到解决办法无视您的拒绝：
   <strong class="markup--strong markup--p-strong">
    研究人员发现
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://redirect.viglink.com/?format=go&amp;jsonp=vglnk_156629709300813&amp;key=ce074976249105acf14d8c9cf69bdcd1&amp;libId=jzjezleu01003n6p000DAou9tkzue&amp;loc=https%3A%2F%2Fwww.cnet.com%2Fnews%2Fmore-than-1000-android-apps-harvest-your-data-even-after-you-deny-permissions%2F%3Fftag%3DCOS-05-10aaa0b&amp;v=1&amp;out=https%3A%2F%2Fwww.ftc.gov%2Fsystem%2Ffiles%2Fdocuments%2Fpublic_events%2F1415032%2Fprivacycon2019_serge_egelman.pdf&amp;ref=https%3A%2F%2Ft.co%2FsdHtOSSL0E%3Famp%3D1&amp;title=%E5%8D%B3%E4%BD%BF%E6%8B%92%E7%BB%9D%E6%9D%83%E9%99%90%EF%BC%8C%E8%B6%85%E8%BF%871%2C000%E4%B8%AAAndroid%E5%BA%94%E7%94%A8%E7%A8%8B%E5%BA%8F%E4%B9%9F%E4%BC%9A%E6%94%B6%E9%9B%86%E6%95%B0%E6%8D%AE%20-%20CNET&amp;txt=more%20than%201%2C000%20apps%20that%20skirted%20restrictions" href="https://redirect.viglink.com/?format=go&amp;jsonp=vglnk_156629709300813&amp;key=ce074976249105acf14d8c9cf69bdcd1&amp;libId=jzjezleu01003n6p000DAou9tkzue&amp;loc=https%3A%2F%2Fwww.cnet.com%2Fnews%2Fmore-than-1000-android-apps-harvest-your-data-even-after-you-deny-permissions%2F%3Fftag%3DCOS-05-10aaa0b&amp;v=1&amp;out=https%3A%2F%2Fwww.ftc.gov%2Fsystem%2Ffiles%2Fdocuments%2Fpublic_events%2F1415032%2Fprivacycon2019_serge_egelman.pdf&amp;ref=https%3A%2F%2Ft.co%2FsdHtOSSL0E%3Famp%3D1&amp;title=%E5%8D%B3%E4%BD%BF%E6%8B%92%E7%BB%9D%E6%9D%83%E9%99%90%EF%BC%8C%E8%B6%85%E8%BF%871%2C000%E4%B8%AAAndroid%E5%BA%94%E7%94%A8%E7%A8%8B%E5%BA%8F%E4%B9%9F%E4%BC%9A%E6%94%B6%E9%9B%86%E6%95%B0%E6%8D%AE%20-%20CNET&amp;txt=more%20than%201%2C000%20apps%20that%20skirted%20restrictions" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     超过1,000个应用程序完全可以绕过权限限制
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，允许他们收集精确的地理位置数据和电话标识符。
   </strong>
  </p>
  <p class="graf graf--p">
   该发现强调了保持在线私密性的难度，特别是如果您使用的是手机和移动应用程序的话。
  </p>
  <p class="graf graf--p">
   科技公司往往拥有全球人的大量个人数据，包括他们曾经去过的地方、他们是谁的朋友、以及他们感兴趣的人和物都是些什么。
  </p>
  <p class="graf graf--p">
   立法者正试图通过隐私监管来实现这一目标，并且应用权限应该可以控制您放弃的数据量。
  </p>
  <p class="graf graf--p">
   Apple 和 Google 已经发布了声称能改善人们隐私的新功能，但是，应用程序仍在寻找隐藏的方法来绕开限制。
  </p>
  <p class="graf graf--p">
   国际计算机科学研究所的研究人员发现，即使在人们明确拒绝他们的许可之后，也有多达1,325个 Android 应用程序继续从设备收集数据。
  </p>
  <p class="graf graf--p">
   ICSI 可用安全和隐私研究主任 Serge Egelman 于6月底在联邦贸易委员会的 PrivacyCon 上介绍了这项研究。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “从根本上说，消费者很少有工具和线索可以用来合理控制自己的隐私并做出决定，”Egelman 在会上说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果应用程序开发人员可以绕过系统，那么向消费者寻求许可，是无意义的。”
  </p>
  <p class="graf graf--p">
   Egelman表示，去年9月，研究人员向谷歌以及FTC通报了这些问题。但谷歌表示，要到 Android Q 发布时才有希望解决。
  </p>
  <p class="graf graf--p">
   谷歌表示，此更新将通过隐藏应用中照片的位置信息来解决问题，并要求任何访问Wi-Fi的应用也拥有关于位置数据的许可。
  </p>
  <p class="graf graf--p">
   该研究调查了来自 Google Play 商店的88,000多个应用，跟踪数据在被拒绝权限时如何继续从应用中传输数据。
  </p>
  <p class="graf graf--p">
   违反 Android 权限的1,325个应用程序使用了隐藏在其代码中的变通方法，该代码将从 Wi-Fi 连接以及照片中存储的元数据等来源获取个人数据。
  </p>
  <p class="graf graf--p">
   研究人员发现，Shutterfly 是一个照片编辑应用程序，它一直在从照片中收集GPS坐标、并将数据发送到自己的服务器，即使用户拒绝授予该应用程序访问位置数据的权限，他们也能做到。
  </p>
  <p class="graf graf--p">
   Shutterfly 发言人表示，该公司只会在“明确允许”的情况下收集位置数据。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “像许多照片服务一样，Shutterfly 使用这些数据来增强用户体验，例如分类和个性化产品建议，所有这些都符合 Shutterfly 的隐私政策以及Android开发者协议，”该公司在一份声明中说。
  </p>
  <p class="graf graf--p">
   有些应用程序还读取设备SD卡上未受保护的文件，并收集他们无权访问的数据。因此，如果您让其他应用访问个人数据，其存储在SD卡上的文件夹中，这些间谍应用就可以获取该信息。
  </p>
  <p class="graf graf--p">
   据研究人员称，虽然只有大约13个应用程序这样做，但是，它们被安装的次数已经超过了1700万次。研究人员说，
   <strong class="markup--strong markup--p-strong">
    这包括百度的香港迪士尼乐园应用程序等。
   </strong>
  </p>
  <p class="graf graf--p">
   百度和迪士尼都没有回应置评请求。
  </p>
  <p class="graf graf--p">
   研究人员发现，有153个具有此功能的应用程序，包括三星的健康和浏览器应用程序，这些应用程序已经被安装在超过5亿台设备上。
  </p>
  <p class="graf graf--p">
   三星没有回复评论请求。
  </p>
  <p class="graf graf--p">
   其他应用程序通过连接到您的 Wi-Fi 网络并计算出路由器的MAC地址来收集位置数据。在充当智能遥控器的应用程序中也发现了这一点。⚪️
  </p>
