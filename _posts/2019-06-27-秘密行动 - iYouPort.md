  <p class="graf graf--p">
   秘密行动是如何运作的？事实上80％的操作是拿破仑曾经使用过的策略的复制。场景规划、系统理论的支持应用、博弈论、战争游戏理论、以及许多其他实践性战略思想和战略执行的模式也被使用。
  </p>
  <p class="graf graf--p">
   以下这些“古老的已知”行动是很好的案例研究，可以用于在秘密行动、心理战和宣传中实现80％的复制。
  </p>
  <p class="graf graf--p">
   希姆莱行动 — —
  </p>
  <p class="graf graf--p">
   希姆勒行动是党卫队为了在战争爆发时为纳粹德国的具体宣传目标服务而进行的一系列非常规秘密行动。它的目的是为了证明波兰对德国的入侵是对德国的侵略。该行动中最知名的是
   <a class="markup--anchor markup--p-anchor" data-href="http://avalon.law.yale.edu/imt/12-20-45.asp" href="http://avalon.law.yale.edu/imt/12-20-45.asp" rel="noopener noreferrer" target="_blank">
    Gleiwitz 事件
   </a>
   。
  </p>
  <p class="graf graf--p">
   而且有说服力的说法是，纳粹焚烧了他们自己的议会，即国会大厦，并把责任推在别人身上。德国国会大厦的火灾是分水岭事件，证明了希特勒夺取权力和终结自由的开端。
  </p>
  <p class="graf graf--p">
   Mainila的轰击 — —
  </p>
  <p class="graf graf--p">
   1939年11月26日，Mainila 的炮击事件是一起军事事件，苏联红军炮击了俄罗斯的 Mainila 村，宣布袭击来自芬兰，横跨附近的边境并声称人员损失。苏联领导人尼基塔·赫鲁晓夫
   <a class="markup--anchor markup--p-anchor" data-href="http://www.aroundspb.ru/finnish/docs/dir0note.php" href="http://www.aroundspb.ru/finnish/docs/dir0note.php" rel="noopener noreferrer" target="_blank">
    以书面形式承认
   </a>
   ，1939年是苏联红军炮击了俄罗斯的 Mainila 村，作为四天后发动冬季战争的基础。
  </p>
  <p class="graf graf--p">
   Gladio — —
  </p>
  <p class="graf graf--p">
   格拉迪奥是冷战期间欧洲臭名昭著的秘密行动，在九十年代，一个秘密武器藏匿、军事战略和政治阴谋的故事。秘密战争中描述了一系列事件：
   <a class="markup--anchor markup--p-anchor" data-href="http://www.php.isn.ethz.ch/collections/coll_gladio/chronology.cfm?navinfo=15301" href="http://www.php.isn.ethz.ch/collections/coll_gladio/chronology.cfm?navinfo=15301" rel="noopener noreferrer" target="_blank">
    格拉迪奥行动和北约的驻军后卫
   </a>
   。
  </p>
  <p class="graf graf--p">
   总理安德烈奥蒂承认，格拉迪奥已发展成为一个广泛的网络分支，在北约内部运作，并在1956年美国与意大利秘密机构之间的协议中受到怂恿。
  </p>
  <p class="graf graf--p">
   德国、法国、比利时、荷兰、希腊和卢森堡都承认他们维持了格拉迪奥式的网络，以准备游击队员在华沙条约入侵的条件下突击行动。许多人的代号为 Stay Behind。在希腊，它被命名为 Red Sheepskin。
  </p>
  <p class="graf graf--p">
   根据当时的新闻报道，此类行动也存在于英国，挪威，葡萄牙，西班牙，奥地利，土耳其和丹麦，甚至还存在于瑞士和瑞典等中立国家。
  </p>
  <p class="graf graf--p">
   1956年与美国机构达成协议的目的是确保西方最大的意大利共产党不能在政府中站稳脚跟。在意大利秘密机构和新法西斯主义者之间存在长期的联系，并且在70年代震撼意大利的主要未解决的恐怖主义行为都被认为是极右翼主导的，而肇事者从未被抓到过。
  </p>
  <p class="graf graf--p">
   在一些国家，Stay Behind 网络已经被关闭。希腊的社会主义者说，他们在1981年发现了该网络后来废除了它，并在1985年抄查了武器。在法国，密特朗说他取消了法国分部，但他没有说什么时候取消的。显然，大多数其他欧洲国家的秘密行动网络仍在继续。
  </p>
  <p class="graf graf--p">
   Lavon 事件 — —
  </p>
  <p class="graf graf--p">
   在埃及活动的一个以色列恐怖主义小组在包括美国外交设施在内的几座建筑物中埋下了炸弹，然后留下了“证据”，暗示阿拉伯人是罪魁祸首。其中一枚炸弹过早引爆，以便“判定”肇事者。几名以色列人后来供认，其中三人后来
   <a class="markup--anchor markup--p-anchor" data-href="http://www.ynetnews.com/Ext/Comp/ArticleLayout/CdaArticlePrintPreview/1,2506,L-3065838,00.html" href="http://www.ynetnews.com/Ext/Comp/ArticleLayout/CdaArticlePrintPreview/1,2506,L-3065838,00.html" rel="noopener noreferrer" target="_blank">
    被以色列人尊敬
   </a>
   。
  </p>
  <p class="graf graf--p">
   中情局在伊朗 — —
  </p>
  <p class="graf graf--p">
   由
   <a class="markup--anchor markup--p-anchor" data-href="https://archive.nytimes.com/www.nytimes.com/library/world/mideast/041600iran-cia-index.html" href="https://archive.nytimes.com/www.nytimes.com/library/world/mideast/041600iran-cia-index.html" rel="noopener noreferrer" target="_blank">
    纽约时报记录
   </a>
   ，为C.I.A.工作的伊朗人在1950年代发动爆炸事件，冒充共产党人所为，以使该国反对民主选举产生的总统。
  </p>
  <p class="graf graf--p">
   Northwoods 行动 — —
  </p>
  <p class="graf graf--p">
   解密文件显示，在20世纪60年代，美国参谋长联席会议签署了代号为“诺斯伍德行动”的计划，以炸毁美国飞机，并以“在美国土地上实施恐怖主义行为”的名义
   <a class="markup--anchor markup--p-anchor" data-href="http://nsarchive.gwu.edu/news/20010430/northwoods.pdf" href="http://nsarchive.gwu.edu/news/20010430/northwoods.pdf" rel="noopener noreferrer" target="_blank">
    归咎于古巴人
   </a>
   ，以证明入侵古巴是正当的。由于肯尼迪政府拒绝执行五角大楼的这些计划，因此没有进行这项行动。
  </p>
  <p class="graf graf--p">
   Macmillan 支持叙利亚暗杀阴谋 — —
  </p>
  <p class="graf graf--p">
   在伊拉克战争将近50年之前，英国和美国又在另一个阿拉伯国家寻求秘密的“政权更迭”，他们被指控通过策划入侵叙利亚和暗杀领导人来传播恐怖主义并威胁西方的石油供应。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/politics/2003/sep/27/uk.syria1" href="https://www.theguardian.com/politics/2003/sep/27/uk.syria1" rel="noopener noreferrer" target="_blank">
    文件显示
   </a>
   ，1957年哈罗德麦克米伦和总统德怀特艾森豪威尔批准了一项中央情报局 — 军情六处的计划，将假边境事件作为叙利亚入侵亲西方邻国的借口，然后“消灭”大马士革最有影响力的三巨头。
  </p>
  <p class="graf graf--p">
   部分“首选计划”内容如下：“
   <em class="markup--em markup--p-em">
    为了促进解放力量的行动，降低叙利亚政权组织和指挥其军事行动的能力，将损失和破坏降至最低，并在尽可能短的时间内取得预期成果，应作出特别努力以消除某些关键人物。他们的撤职应该在起义和干预的早期完成
   </em>
   。“
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    — — 警察渗透持不同政见的运动 — —
   </strong>
  </p>
  <p class="graf graf--p">
   不仅军方会进行秘密行动，警察也是如此。众所周知的就是他们对抗议者和政治运动的渗透。
  </p>
  <p class="graf graf--p">
   在英国，付费的线人 — —
  </p>
  <p class="graf graf--p">
   由于70年代和80年代的糟糕时期，当系统失去信誉并且皇家检察院拒绝接触任何向警方告密的案件时，规则已经收紧。 […] “2000年调查权力法”对“线人”的使用进行了监督。在公共当局授权使用隐蔽来源的情况下，该法案的规定
   <a class="markup--anchor markup--p-anchor" data-href="http://www.theguardian.com/uk/2009/apr/25/police-surveillance-sources-informants" href="https://www.theguardian.com/uk/2009/apr/25/police-surveillance-sources-informants" rel="noopener noreferrer" target="_blank">
    适用于苏格兰
   </a>
   。
  </p>
  <p class="graf graf--p">
   2009年，在英格兰，一位气候变化活动家
   <a class="markup--anchor markup--p-anchor" data-href="http://www.theguardian.com/uk/2009/apr/24/strathclyde-police-plane-stupid-recruit-spy" href="https://www.theguardian.com/uk/2009/apr/24/strathclyde-police-plane-stupid-recruit-spy" rel="noopener noreferrer" target="_blank">
    录制了证据
   </a>
   ，揭露了一些当局的线人被提供现金以获取有关团体成员和活动的私密信息。
  </p>
  <p class="graf graf--p">
   在录音带中，警察声称已经渗透到一些环保组织内部，
   <a class="markup--anchor markup--p-anchor" data-href="http://www.theguardian.com/uk/2009/may/01/liberty-climate-protesters-campaigners" href="https://www.theguardian.com/uk/2009/may/01/liberty-climate-protesters-campaigners" rel="noopener noreferrer" target="_blank">
    并表示
   </a>
   他们正在接收有关未来示威的领导者身份、战术和计划的信息。活动人员也受到广泛的监视。
  </p>
  <p class="graf graf--p">
   在苏格兰信息专员作出裁决后，斯特拉斯克莱德警方被迫公布了向“隐蔽的线人”支付款项的细节，详细情况显示该部队自2004年以来每年支付给线人的金额几乎翻了一倍，当时是145,198英镑，2004年至2008年共支付了762,459英镑。
  </p>
  <p class="graf graf--p">
   煽动 — —
  </p>
  <p class="graf graf--p">
   在抗议期间，
   <a class="markup--anchor markup--p-anchor" data-href="http://www.theguardian.com/politics/2009/may/10/g20-policing-agent-provacateurs" href="https://www.theguardian.com/politics/2009/may/10/g20-policing-agent-provacateurs" rel="noopener noreferrer" target="_blank">
    G20警察“使用卧底线人煽动人群”
   </a>
   ，这类现象并不意外。煽动就是为了栽赃，以证明镇压的合理性。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    卧底监视 — —
   </strong>
  </p>
  <p class="graf graf--p">
   Xmas gifts at all cost: 苏格兰场已经部署了秘密警察来监视一个民间活跃人士的关系网络，这些活动家反逃税的热门宣传活动有可能在圣诞节期间关闭数百家商店。尽管去年向议会保证只有穿着制服的军官在抗议活动中收集情报，但是，负责监视的便衣最初还是在10月的抗议活动中
   <a class="markup--anchor markup--p-anchor" data-href="http://www.theguardian.com/uk/2010/dec/03/uk-uncut-protests-undercover-police/print" href="https://www.theguardian.com/uk/2010/dec/03/uk-uncut-protests-undercover-police/print" rel="noopener noreferrer" target="_blank">
    被使用了
   </a>
   。
  </p>
  <p class="graf graf--p">
   European network of police infiltration: 没人知道的是，尽管这位33岁的“自由登山者” Kennedy 非常著名，但
   <a class="markup--anchor markup--p-anchor" data-href="http://www.theguardian.com/environment/2011/jan/10/mark-kennedy-undercover-cop-activist" href="https://www.theguardian.com/environment/2011/jan/10/mark-kennedy-undercover-cop-activist" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     他还是一名卧底警察
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，一直在大胆的行动，深度渗透环保活动家群体。“战术”甚至
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://www.theguardian.com/uk/2011/jan/22/undercover-police-cleared-sex-activists" href="https://www.theguardian.com/uk/2011/jan/22/undercover-police-cleared-sex-activists" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     包括滥交行为
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   2011年，
   <a class="markup--anchor markup--p-anchor" data-href="http://www.theguardian.com/uk/2011/jan/14/third-undercover-police-spy-cardiff/print" href="https://www.theguardian.com/uk/2011/jan/14/third-undercover-police-spy-cardiff/print" rel="noopener noreferrer" target="_blank">

   </a>
   ，更多关于 Kennedy 在欧洲抗议运动中的关键作用的细节也出现了。显然，他还在爱尔兰、德国和冰岛担任卧底线人。他可能已经要求公关公司 Max Clifford 出售他的故事。
  </p>
  <p class="graf graf--p">
   Blacklisting case：2015年3月，建筑工会 UCATT 暴露了警察的渗透，并于2014年12月2日提出了 FOI 请求。但是响应被不断推迟…… 2月20日，警方表示他们“无需确认或否认是否提供被要求的信息“，并列举了为什么他们无法透露包括所谓的“国家安全”在内的六方面信息 。简称：滥用法律的狡辩。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    CIA 的间谍工具包是数字化的 — —
   </strong>
  </p>
  <p class="graf graf--p">
   在过去十年中，受过专门训练的 CIA
   <a class="markup--anchor markup--p-anchor" data-href="http://foreignpolicy.com/2013/07/17/the-cias-new-black-bag-is-digital/" href="http://foreignpolicy.com/2013/07/17/the-cias-new-black-bag-is-digital/" rel="noopener noreferrer" target="_blank">
    秘密运营商已经安装了一百多个非常敏感的black bag
   </a>
   ，旨在渗透外国政府和军事通信和计算机系统，以及一些世界上最大的外国跨国公司的计算机系统。
  </p>
  <p class="graf graf--p">
   间谍软件已秘密植入计算机服务器; 安全电话线已被窃听; 光纤电缆、数据交换中心和电话交换机已被挖掘; 在这些操作中，计算机备份磁带和磁盘被盗或被秘密复制。
  </p>
  <p class="graf graf--p">
   换句话说，中央情报局建立秘密监视巨型网络的工具现在已被投入公众视野。美国情报界的消息来源证实，自9/11以来，中央情报局的秘密行动使国家安全局能够进入世界各地的极其重要的目标（不知道在美国的土地上是否也有这样的行动）
  </p>
