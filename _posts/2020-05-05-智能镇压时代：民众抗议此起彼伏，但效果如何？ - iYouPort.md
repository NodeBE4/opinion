  <p>
   <img alt="" class="aligncenter size-full wp-image-11054 jetpack-lazy-image" data-lazy-sizes="(max-width: 622px) 100vw, 622px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/抗议.jpeg?resize=622%2C1003&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/抗议.jpeg?w=622&amp;ssl=1 622w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/抗议.jpeg?resize=186%2C300&amp;ssl=1 186w" data-recalc-dims="1" height="1003" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/抗议.jpeg?resize=622%2C1003&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="622"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11054" data-recalc-dims="1" height="1003" sizes="(max-width: 622px) 100vw, 622px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/抗议.jpeg?resize=622%2C1003&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/抗议.jpeg?w=622&amp;ssl=1 622w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/抗议.jpeg?resize=186%2C300&amp;ssl=1 186w" width="622"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   除了对腐败和一系列经济和社会的不满之外，还有一个主题在全球范围内引发了大大小小的抗议，那就是，
   <strong class="markup--strong markup--p-strong">
    对人的尊严的呼吁
   </strong>
   。
  </p>
  <p class="graf graf--p">
   抗议活动反映了全球对政治制度和领导能力的信心崩溃。对尊严和社会正义的追求将黎巴嫩、伊拉克、约旦、埃及、阿尔及利亚等中东和北非国家的抗议活动，与苏丹、智利，玻利维亚，厄瓜多尔，委内瑞拉和海地等国家，及法国、津巴布韦、印度尼西亚、巴基斯坦和香港的示威活动联系在一起。
  </p>
  <p class="graf graf--p">
   2019年的全球抗议活动构成了2011年爆发的挑战与异议时代的最新阶段，在突尼斯、埃及、利比亚和也门的专制领导人垮台后，这种抗议在中东和北非的爆发最为激烈。
  </p>
  <p class="graf graf--p">
   但在这四个阿拉伯国家中，只有突尼斯从独裁体制向民主体制的过渡相对成功。
  </p>
  <p class="graf graf--p">
   地区和国内反革命势力在2013年发动了军事政变，以免除埃及第一位也是唯一一位民主选举产生的总统，从而建立了埃及独立后历史上最残酷的和压制性的政权之一。
  </p>
  <p class="graf graf--p">
   利比亚和也门在外国干预的推动下饱受内战的打击；叙利亚遭受了将近九年的内战。
  </p>
  <p class="graf graf--p">
   与该地区其他国家一样，土耳其总统雷杰普·塔伊普·埃尔多安（Recep Tayyip Erdogan）利用了2013年的盖齐公园抗议活动 —— 该党执政十年来最大的反政府示威游行，以及2016年失败的军事政变，扭转了土耳其向民主和政治多元化迈进的步伐。
  </p>
  <p class="graf graf--p">
   中东和北非撤退到了更具压制性的威权主义和专制政体，再加上俄罗斯、中国、香港和哈萨克斯坦的各种镇压行动（这是仅举几个例子），促使
   <strong class="markup--strong markup--p-strong">
    分析人士怀疑大规模抗议活动是否仍然是实现政治变革一种有效的方式
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “仅在20年前，要求进行系统性政治变革的抗议者中还有70%实现了目标，并且这一数字自1950年代以来一直在稳定增长。
   <strong class="markup--strong markup--p-strong">
    而在2000年代中期，这种趋势突然逆转。在全球范围内，抗议者的成功率下降至仅30％”
   </strong>
   ，纽约时报记者马克斯·费舍尔（Max Fisher）和阿曼达·陶布（Amanda Taub）
   <a class="markup--anchor markup--p-anchor" data-href="https://messaging-custom-newsletters.nytimes.com/template/oakv2?uri=nyt://newsletter/375eeb8b-49fd-4a0f-933c-9ef2ed0a10d8" href="https://messaging-custom-newsletters.nytimes.com/template/oakv2?uri=nyt://newsletter/375eeb8b-49fd-4a0f-933c-9ef2ed0a10d8" rel="noopener noreferrer" target="_blank">
    在专栏中
   </a>
   探讨了当前的不满情绪根源。
  </p>
  <ul>
   <li>
    《
    <a href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e8%a1%97%e5%a4%b4%e6%8a%97%e8%ae%ae%e4%b8%8d%e8%b5%b7%e4%bd%9c%e7%94%a8%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     为什么街头抗议不起作用？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   纽约时报文章的结论是基于政治学家
   <a class="markup--anchor markup--p-anchor" data-href="https://www.researchgate.net/publication/316474594_Trends_in_Nonviolent_Resistance_and_State_Response_Is_Violence_Towards_Civilian-based_Movements_on_the_Rise?te=1&amp;nl=the-interpreter&amp;emc=edit_int_20191028" href="https://www.researchgate.net/publication/316474594_Trends_in_Nonviolent_Resistance_and_State_Response_Is_Violence_Towards_Civilian-based_Movements_on_the_Rise?te=1&amp;nl=the-interpreter&amp;emc=edit_int_20191028" rel="noopener noreferrer" target="_blank">
    Erica Chenoweth 的一项研究
   </a>
   得出的，该研究表明，不自由的民主政权、专制主义者和独裁者已经变得更加善于用所谓的“智能镇压”来阻止抗议活动。
  </p>
  <p class="graf graf--p">
   然而，Chenoweth 定义的 “智能镇压” 旨在确保对精英们的忠诚，包括维稳部队和准军事代理人造成的更大程度的残酷和暴力，加强对异议者的审查和定罪，将反叛描述为外国敌对势力操纵的阴谋和恐怖主义形式，充其量不过是专制和独裁者的标准反应的升级版。
  </p>
  <p class="graf graf--p">
   很难描述在埃及发生的军事政变及其后镇压造成1000多人被杀的直接后果究竟哪里“智能”。任意拘留杰出的商人、宗教人物和活动家及其家人，这相当于沙特王储穆罕默德·本·萨勒曼（Mohammed bin Salman）的手段；在中国西北部陷入困境的新疆地区，估计有100万突厥穆斯林被大规模拘留在再教育营中；而在土耳其和埃及等国家逮捕了数万人。
  </p>
  <p class="graf graf--p">
   可能是自第二次世界大战以来的首次，这可以更好地解释抗议活动有效性降低的原因；政治学家安娜·鲁尔曼（Anne Luehrmann）和斯塔凡·林德伯格（Staffan Lindberg）称之为
   <strong class="markup--strong markup--p-strong">
    “
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.tandfonline.com/doi/full/10.1080/13510347.2019.1582029?te=1&amp;nl=the-interpreter&amp;emc=edit_int_20191028&amp;" href="https://www.tandfonline.com/doi/full/10.1080/13510347.2019.1582029?te=1&amp;nl=the-interpreter&amp;emc=edit_int_20191028&amp;" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     第三次专制化浪潮
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ” 的结果是，走向专制主义的国家数量超过了走向民主的国家数量。
   </strong>
  </p>
  <p class="graf graf--p">
   引起这一浪潮的根本原因是，一大批世界领导人的崛起，他们都拥护非自由民主、专制主义和独裁的治理原则，出于对民族或宗教团体权利的至高无上的支持，而无视基本人权和边缘群体的权利。
  </p>
  <p class="graf graf--p">
   这些领导人的崛起在许多方面都是抗议活动的反面。他们通常是政治局外人，可能会或可能不会成为精英人士，如美国的特朗普、匈牙利的 Victor Orban、印度的 Modi、巴西的 Jair Bolsonaro、菲律宾的 Rodrigo Duterte、但
   <strong class="markup--strong markup--p-strong">
    他们把自己投射为 ‘变革力量”，目的是维护精英对权力的绝对控制
   </strong>
   。
  </p>
  <p class="graf graf--p">
   他们所谓的开化主义和反动民族主义的各个方面已经得到了经常持反对态度的政治力量的允许和不同程度的支持，包括极右翼、反移民和至上主义的种族和宗教团体，以及受欢迎的左派，包括美国的一些民主党总统候选人。
  </p>
  <p class="graf graf--p">
   结果是潜在的恶性循环，在这种恶性循环中，日益受到限制的民主权利和更大的镇压使越来越多的社会群体边缘化，其中包括作为重要阶层的中产阶级以及少数群体，像香港、伊拉克、苏丹或罗兴亚人，看到自己已经没什么可失去的了，人们就会更强烈地反弹。
  </p>
  <p class="graf graf--p">
   随着激进派拥有更大的吸引力，在各方面的暴力都增加了。
  </p>
  <p class="graf graf--p">
   Chenoweth、Luehrmann 和 Lindberg 的结论将证明这一点。如果抗议是人们对政府和政治力量之无能的唯一和平解决方案，破坏抗议活动的有效性就会缩小影响变革的选择范围。
  </p>
  <p class="graf graf--p">
   从这个角度来看，学者们的结论相当于对乔治·奥威尔（George Orwell）1944年
   <a class="markup--anchor markup--p-anchor" data-href="http://gutenberg.net.au/ebooks03/0300011h.html#part21" href="http://gutenberg.net.au/ebooks03/0300011h.html#part21" rel="noopener noreferrer" target="_blank">
    的主张
   </a>
   进行了当代版的改编，“所有革命都是失败，但并非都是相同的失败。”
  </p>
  <p class="graf graf--p">
   然而，尽管学者们预测了趋势，但这可能还为时过早。
  </p>
  <p class="graf graf--p">
   可以肯定的是，对于突尼斯和苏丹的反叛运动是否会产生持久的政治变化这一问题，尚无定论。
  </p>
  <p class="graf graf--p">
   从2011年阿拉伯起义开始至今已经过去八年，抗议者一直在决心确保自己的知名度及其在社会中的地位，他们强调了经验教训，不再像阿尔及利亚和苏丹那样在一个领导人被迫做出让步或辞职后就宣布胜利，并且超越了伊拉克和黎巴嫩等易于被利用的宗派种族和宗教鸿沟。
  </p>
  <p class="graf graf--p">
   中东学者 Hanin Ghaddar
   <a class="markup--anchor markup--p-anchor" data-href="https://foreignpolicy.com/2019/10/22/iran-losing-middle-east-iraq-lebanon-protests-bad-governance/?utm_source=PostUp&amp;utm_medium=email&amp;utm_campaign=17908&amp;utm_term=Editor#39;s%20Picks%20OC" href="https://foreignpolicy.com/2019/10/22/iran-losing-middle-east-iraq-lebanon-protests-bad-governance/?utm_source=PostUp&amp;utm_medium=email&amp;utm_campaign=17908&amp;utm_term=Editor#39;s%20Picks%20OC" rel="noopener noreferrer" target="_blank">
    说
   </a>
   ：“很长一段时间以来，黎巴嫩人第一次意识到敌人在内部，就是他们自己的政府和政治领导人，而不是什么外部占领者或区域影响者……政治领导人无法控制抗议活动的进行，抗议活动发生在所有宗派和所有地区。使人们聚集在一起的是一场持续的经济危机，这场危机已经伤害了各个宗派和地区的人们。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    认识到在过渡模式到位之前需要维持街头力量，这对于提高抗议者保持其有行动效性的机会至关重要。
   </strong>
  </p>
  <p class="graf graf--p">
   抗议还能否作为一种有效工具的未来，同样取决于对共同利益的理解，这种共同利益必需超越宗派、种族和阶级成为社会结构的一部分。
  </p>
