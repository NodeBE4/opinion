  <p class="graf graf--p">
   Apple 的 macOS 平台包括终端，相当于 Windows 操作系统中的命令提示符。由于 macOS 基于 UNIX，因此终端可能会给你一个危险工具的氛围，一个错误的命令可能就会破坏你的系统。
  </p>
  <p class="graf graf--p">
   不用怕，其实有一堆无害的命令可用于自定义 macOS 的界面和其他设置。
  </p>
  <p class="graf graf--p">
   macOS上的终端在调整Mac性能方面非常有用。现在它完全取决于你想要达到的目标。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-8941 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/命令.jpg?resize=1100%2C602&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/命令.jpg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/命令.jpg?resize=300%2C164&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/命令.jpg?resize=768%2C421&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/命令.jpg?resize=1024%2C561&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/命令.jpg?resize=1100%2C602&amp;ssl=1 1100w" data-recalc-dims="1" height="602" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/命令.jpg?resize=1100%2C602&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  </p>
  <p class="graf graf--p">
   以下是可以在Mac上尝试的前10个最简单的终端命令技巧。
  </p>
  <p class="graf graf--p">
   注意：在选择/复制命令时，请确保一直向右滚动以在灰色框中选择完整命令。否则，部分命令将无法按预期工作。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、打开文件夹
   </strong>
  </p>
  <p class="graf graf--p">
   熟悉终端的最酷方法之一是使用它来打开文件和文件夹。
  </p>
  <p class="graf graf--p">
   我知道点击 Finder 也能做到。但是，使用终端，您可以打开特定文件夹或文件，而不会将文件结构暴露给任何人。
  </p>
  <p class="graf graf--p">
   假设您要打开“Documents”文件夹，您可以键入以下命令，然后按Enter键：
  </p>
  <pre class="graf graf--pre">Open ~/Documents</pre>
  <p class="graf graf--p">
   这就可以打开 Documents 文件夹。那里有一个DOCX文件，也可以用终端打开它。为此，您需要输入具有一些要求的命令 —— 相应文件的路径：
  </p>
  <p class="graf graf--p">
   Open -a “Application Name” /Path/to/File
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8942 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-55.png?resize=1100%2C557&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-55.png?w=1166&amp;ssl=1 1166w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-55.png?resize=300%2C152&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-55.png?resize=768%2C389&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-55.png?resize=1024%2C518&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-55.png?resize=1100%2C557&amp;ssl=1 1100w" data-recalc-dims="1" height="557" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-55.png?resize=1100%2C557&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8942" data-recalc-dims="1" height="557" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-55.png?resize=1100%2C557&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-55.png?w=1166&amp;ssl=1 1166w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-55.png?resize=300%2C152&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-55.png?resize=768%2C389&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-55.png?resize=1024%2C518&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-55.png?resize=1100%2C557&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   因此，要在 Mac 的 Documents 文件夹中打开DOCX文件，就输入以下命令，然后按 Enter 键：
  </p>
  <pre class="graf graf--pre">Open -a "Microsoft Word" ~/Documents/SoP_Prajith_New.docx</pre>
  <p class="graf graf--p">
   您还可以使用通配符（* .extension）而不是完整文件名。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、更改默认设置保留屏幕截图位置
   </strong>
  </p>
  <p class="graf graf--p">
   默认情况下，macOS Mojave 将屏幕截图存储在Mac的桌面上。现在，如果您启用了iCloud同步，那么它会不断上传到您的帐户。
  </p>
  <p class="graf graf--p">
   这是不安全的，也很占空间。幸运的是，您可以更改屏幕截图的保存位置。
  </p>
  <p class="graf graf--p">
   输入以下命令然后按 Enter 键：
  </p>
  <pre class="graf graf--pre">defaults write com.apple.screencapture location ~/Downloads</pre>
  <p class="graf graf--p">
   在上面的命令中，您可以提供任何特定的文件位置，如图所示：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8943 jetpack-lazy-image" data-lazy-sizes="(max-width: 761px) 100vw, 761px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-40.png?resize=761%2C396&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-40.png?w=761&amp;ssl=1 761w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-40.png?resize=300%2C156&amp;ssl=1 300w" data-recalc-dims="1" height="396" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-40.png?resize=761%2C396&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="761"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8943" data-recalc-dims="1" height="396" sizes="(max-width: 761px) 100vw, 761px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-40.png?resize=761%2C396&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-40.png?w=761&amp;ssl=1 761w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-40.png?resize=300%2C156&amp;ssl=1 300w" width="761"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   按 Enter 键后，需要重置 SystemUIServer 以使更改立即生效。所以请输入以下命令：
  </p>
  <pre class="graf graf--pre">killall SystemUIServer</pre>
  <p class="graf graf--p">
   虽然不是必需的，但建议您重新启动Mac。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、更改屏幕图像类型
   </strong>
  </p>
  <p class="graf graf--p">
   默认情况下，macOS以JPG格式保存屏幕截图。
  </p>
  <p class="graf graf--p">
   您可以更改它以将它们保存为PNG或PDF格式。以下是使用特定命令快速完成此操作的方法：
  </p>
  <pre class="graf graf--pre">defaults write com.apple.screencapture type PNG</pre>
  <p class="graf graf--p">
   之后，您需要再次终止 SystemUIServer。
  </p>
  <p class="graf graf--p">
   大多数需要进行系统更改的命令都要求关闭 SystemUIServer。所以这样：
  </p>
  <pre class="graf graf--pre">killall SystemUIServer</pre>
  <p class="graf graf--p">
   现在，您的新屏幕截图将以PNG格式保存。请注意，PNG文件通常比JPG文件更大。因此，请留意它们消耗的存储空间。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、显示隐藏文件
   </strong>
  </p>
  <p class="graf graf--p">
   您是否尝试过让 Finder 显示隐藏文件？嗯，它比在 Windows 上选择显示隐藏文件更难。
  </p>
  <p class="graf graf--p">
   但是有一个有用的命令可以在几秒钟内完成：
  </p>
  <pre class="graf graf--pre">defaults write com.apple.finder AppleShowAllFiles -bool TRUE</pre>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8944 jetpack-lazy-image" data-lazy-sizes="(max-width: 996px) 100vw, 996px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-38.png?resize=996%2C526&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-38.png?w=996&amp;ssl=1 996w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-38.png?resize=300%2C158&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-38.png?resize=768%2C406&amp;ssl=1 768w" data-recalc-dims="1" height="526" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-38.png?resize=996%2C526&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="996"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8944" data-recalc-dims="1" height="526" sizes="(max-width: 996px) 100vw, 996px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-38.png?resize=996%2C526&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-38.png?w=996&amp;ssl=1 996w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-38.png?resize=300%2C158&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-38.png?resize=768%2C406&amp;ssl=1 768w" width="996"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   之后，您需要强制关闭 Finder。
  </p>
  <pre class="graf graf--pre">killall Finder</pre>
  <p class="graf graf--p">
   现在你应该在 Finder 中看到很多隐藏文件。它们会变灰但仍然可见。如果返回更改，只需要在命令中将TRUE改为FALSE。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、关掉仪表板
   </strong>
  </p>
  <p class="graf graf--p">
   Apple推出了仪表板视图，可以快速访问计算器和便签。你肯定很少用这个功能。因此，您可以将其关闭，而不必担心错误地打开它。
  </p>
  <p class="graf graf--p">
   这是关闭仪表板的命令：
  </p>
  <pre class="graf graf--pre">defaults write com.apple.dashboard mcx-disabled -boolean TRUE</pre>
  <p class="graf graf--p">
   接下来，您必须终止 Dock 以便令更改生效。
  </p>
  <pre class="graf graf--pre">killall Dock</pre>
  <p class="graf graf--p">
   现在不必担心意外打开仪表板了。如果要再次打开它，请在命令中将TRUE更改为FALSE，即可。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    6、下载文件
   </strong>
  </p>
  <p class="graf graf--p">
   您知道可以使用终端下载文件吗？是的，您不需要让浏览器为该文件运行。
  </p>
  <p class="graf graf--p">
   这里唯一的要求是你必须拥有文件的下载链接。所以命令语法是：
  </p>
  <p class="graf graf--p">
   curl -0 downloadlink
  </p>
  <p class="graf graf--p">
   首先，导航到要下载文件的文件夹。为此，您需要切换到 Downloads 目录。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8945 jetpack-lazy-image" data-lazy-sizes="(max-width: 883px) 100vw, 883px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/4-26.png?resize=883%2C460&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/4-26.png?w=883&amp;ssl=1 883w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/4-26.png?resize=300%2C156&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/4-26.png?resize=768%2C400&amp;ssl=1 768w" data-recalc-dims="1" height="460" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/4-26.png?resize=883%2C460&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="883"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8945" data-recalc-dims="1" height="460" sizes="(max-width: 883px) 100vw, 883px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/4-26.png?resize=883%2C460&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/4-26.png?w=883&amp;ssl=1 883w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/4-26.png?resize=300%2C156&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/4-26.png?resize=768%2C400&amp;ssl=1 768w" width="883"/>
   </noscript>
  </p>
  <pre class="graf graf--pre">cd ~/Downloads</pre>
  <p class="graf graf--p">
   它看起来像这样：
  </p>
  <pre class="graf graf--pre">curl -O https://get.videolan.org/vlc/3.0.7.1/macosx/vlc-3.0.7.1.dmg</pre>
  <p class="graf graf--p">
   注意：命令中有一个字母O而不是0（零）。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    7、快速备份
   </strong>
  </p>
  <p class="graf graf--p">
   由于SSD和闪存存储，复制文件和在macOS上进行备份非常迅速。但是，复制大量数据仍需要一段时间。
  </p>
  <p class="graf graf--p">
   Terminal 提供了一个复制文件的有用命令。以下是执行此操作的方法。
  </p>
  <p class="graf graf--p">
   ditto -V /currentpath/ /new/path/
  </p>
  <p class="graf graf--p">
   测试将前文中下载的DMG复制到桌面：
  </p>
  <pre class="graf graf--pre">ditto -V ~/Downloads/vlc-3.0.7.1.dmg ~/Desktop</pre>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8946 jetpack-lazy-image" data-lazy-sizes="(max-width: 883px) 100vw, 883px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5-18.png?resize=883%2C460&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5-18.png?w=883&amp;ssl=1 883w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5-18.png?resize=300%2C156&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5-18.png?resize=768%2C400&amp;ssl=1 768w" data-recalc-dims="1" height="460" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5-18.png?resize=883%2C460&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="883"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8946" data-recalc-dims="1" height="460" sizes="(max-width: 883px) 100vw, 883px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5-18.png?resize=883%2C460&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5-18.png?w=883&amp;ssl=1 883w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5-18.png?resize=300%2C156&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/5-18.png?resize=768%2C400&amp;ssl=1 768w" width="883"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   如果要将文件复制到外部驱动器，则需要正确提供目标路径。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    8、始终在FINDER中显示文件路径
   </strong>
  </p>
  <p class="graf graf--p">
   如果您熟悉 Windows，您会注意到 macOS 上的 Finder 没有像 Windows 中的资源管理器那样显示文件路径。
  </p>
  <p class="graf graf--p">
   但是，您可以使用此命令强制Finder显示文件路径：
  </p>
  <pre class="graf graf--pre">defaults write com.apple.finder _FXShowPosixPathInTitle -boolean true</pre>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8947 jetpack-lazy-image" data-lazy-sizes="(max-width: 1019px) 100vw, 1019px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-17.png?resize=1019%2C540&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-17.png?w=1019&amp;ssl=1 1019w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-17.png?resize=300%2C159&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-17.png?resize=768%2C407&amp;ssl=1 768w" data-recalc-dims="1" height="540" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-17.png?resize=1019%2C540&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1019"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8947" data-recalc-dims="1" height="540" sizes="(max-width: 1019px) 100vw, 1019px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-17.png?resize=1019%2C540&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-17.png?w=1019&amp;ssl=1 1019w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-17.png?resize=300%2C159&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/6-17.png?resize=768%2C407&amp;ssl=1 768w" width="1019"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   点击回车后，你需要停止Finder。
  </p>
  <pre class="graf graf--pre">killall Finder</pre>
  <p class="graf graf--p">
   点击 Enter 后，Finder 将重新启动。然后，您应该在 Finder 窗口的顶部看到文件路径。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    9、停止你的MAC睡眠
   </strong>
  </p>
  <p class="graf graf--p">
   通常你下载一个大文件时你不希望Mac进入睡眠状态。
  </p>
  <p class="graf graf--p">
   如果这是临时要求，那么一个简单的命令就可以实现它。
  </p>
  <p class="graf graf--p">
   只需输入以下命令，按Enter键即可：
  </p>
  <pre class="graf graf--pre">caffeinate -u -t 600</pre>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8948 jetpack-lazy-image" data-lazy-sizes="(max-width: 685px) 100vw, 685px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-17.png?resize=685%2C460&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-17.png?w=685&amp;ssl=1 685w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-17.png?resize=300%2C201&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-17.png?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="460" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-17.png?resize=685%2C460&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="685"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8948" data-recalc-dims="1" height="460" sizes="(max-width: 685px) 100vw, 685px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-17.png?resize=685%2C460&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-17.png?w=685&amp;ssl=1 685w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-17.png?resize=300%2C201&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/7-17.png?resize=272%2C182&amp;ssl=1 272w" width="685"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在此命令中，数字600表示秒。因此，使用此命令，Mac将无法睡眠，直到满10分钟（600秒）。
  </p>
  <p class="graf graf--p">
   您可以输入您想要的秒数或单词“caffeinate”以防止Mac进入睡眠状态。
  </p>
  <p class="graf graf--p">
   但是请记住，不要关闭该终端窗口，否则该命令将停止执行，Mac将在预定义的时间内进入休眠状态。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    10、检查你的MAC正常运行时间
   </strong>
  </p>
  <p class="graf graf--p">
   想知道自从你重新启动 Mac 以来已经有多长时间了？有一个命令，用于检查Mac在没有重启或关机的情况下运行的时间：
  </p>
  <pre class="graf graf--pre">uptime</pre>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8949 jetpack-lazy-image" data-lazy-sizes="(max-width: 685px) 100vw, 685px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/8-10.png?resize=685%2C460&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/8-10.png?w=685&amp;ssl=1 685w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/8-10.png?resize=300%2C201&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/8-10.png?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="460" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/8-10.png?resize=685%2C460&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="685"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8949" data-recalc-dims="1" height="460" sizes="(max-width: 685px) 100vw, 685px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/8-10.png?resize=685%2C460&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/8-10.png?w=685&amp;ssl=1 685w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/8-10.png?resize=300%2C201&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/8-10.png?resize=272%2C182&amp;ssl=1 272w" width="685"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    使用终端是有趣的
   </strong>
  </p>
  <p class="graf graf--p">
   终端是在macOS中使用和修补的绝对乐趣。但是，严格建议您不要使用未经验证的来源或陌生人提供的随机命令，如果您不够理解它的话。除此之外，这些命令应该在最新的 macOS Mojave 更新上运行。
  </p>
  <p class="graf graf--p">
   您应该始终仔细检查涉及删除或禁用任何服务的终端命令。
  </p>
  <p class="graf graf--p">
   就是这样。您也能像专业人士那样玩终端。⚪️
  </p>
