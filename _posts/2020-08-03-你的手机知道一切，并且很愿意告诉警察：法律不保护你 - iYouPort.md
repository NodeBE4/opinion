  <p class="graf graf--p">
   根据不同人的喜好不同，这种贴身的营销可能会让有些人感觉毛骨悚然，而另一些人则觉得非常有帮助。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    但是，如果您要与政府机构抗衡，那么位置跟踪无疑是非常危险的，因为，政府机构只会把你扔进监狱，而不是给您特别优惠 ……
   </strong>
  </p>
  <p class="graf graf--p">
   您的手机正在跟踪您的一举一动，尽管受到法律保护 —— 在很多国家都有法律保护 —— 但是，中央和地方官员都正在努力寻找新的且令人不安的方式来使用该信息。
  </p>
  <ul class="postList">
   <li class="graf graf--li">

  <p class="graf graf--p">
   在法律正式实施之前很久一些警察就已经发现通过电话追踪目标人的潜力了。
  </p>
  <p class="graf graf--p">
   比如美国。密苏里州前警长 Cory Hutcheson 去年4月因在没有逮捕证的情况下非法跟踪人们的电话而被判入狱六个月，并被软禁四个月。被追踪的人中有他的前任警长、州警官和县法官。
  </p>
  <p class="graf graf--p">
   美国东部地区检察官办公室
   <a class="markup--anchor markup--p-anchor" data-href="https://www.justice.gov/usao-edmo/pr/mississippi-county-sheriff-sentenced-10-months-fraud-and-identity-theft" href="https://www.justice.gov/usao-edmo/pr/mississippi-county-sheriff-sentenced-10-months-fraud-and-identity-theft" rel="noopener noreferrer" target="_blank">
    称
   </a>
   ：“在三年内，包括当选密西西比县警长之后，Hutcheson 一直在将虚假和的欺诈性文件上传到执法数据库中，以获取200多个手机用户的具体位置。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “ Hutcheson 在没有有效法律授权、未经目标个人同意或知情的情况下，提交了数千个数据请求，并获得了数百个手机用户的实际位置数据。”
  </p>
  <p class="graf graf--p">
   Hutcheson 通过 Securus 进行了跟踪，该公司是监狱电话提供商，可在通信设备上为执法人员提供定位服务。*
  </p>
  <p class="graf graf--p">
   根据2018年 ZDNet 对这种关系的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.zdnet.com/article/us-cell-carriers-selling-access-to-real-time-location-data/" href="https://www.zdnet.com/article/us-cell-carriers-selling-access-to-real-time-location-data/" rel="noopener noreferrer" target="_blank">
    调查
   </a>
   显示，Securus 从
   <a class="markup--anchor markup--p-anchor" data-href="https://www.locationsmart.com/platform/location" href="https://www.locationsmart.com/platform/location" rel="noopener noreferrer" target="_blank">
    LocationSmart
   </a>
   获得了手机位置数据，LocationSmart 与主要的移动通信提供商合作。
  </p>
  <p class="graf graf--p">
   这项安排绕开了对政府机构直接从电信获取此类数据的
   <a class="markup--anchor markup--p-anchor" data-href="https://it.ojp.gov/PrivacyLiberty/authorities/statutes/1285" href="https://it.ojp.gov/PrivacyLiberty/authorities/statutes/1285" rel="noopener noreferrer" target="_blank">
    限制
   </a>
   。
  </p>
  <p class="graf graf--p">
   手机位置记录的第四修正案保护在2018年的 Carpenter 诉美国案中得到了加强，该案发现，“信号塔位置记录相比传统车辆的GPS带来了更大的隐私问题”。
  </p>
  <p class="graf graf--p">
   但是坏警察就是要用手边能得到的任何东西去做坏事。而
   <strong class="markup--strong markup--p-strong">
    所谓的好警察（或者至少是那些遵守规则的警察）也很感兴趣知道您在哪里，而他们的举动并不一定会让人放心。
   </strong>
  </p>
  <p class="graf graf--p">
   《华尔街日报》最近
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wsj.com/articles/federal-agencies-use-cellphone-location-data-for-immigration-enforcement-11581078600?mod=hp_lead_pos6" href="https://www.wsj.com/articles/federal-agencies-use-cellphone-location-data-for-immigration-enforcement-11581078600?mod=hp_lead_pos6" rel="noopener noreferrer" target="_blank">
    报道说
   </a>
   ：“美国移民和海关执法局……已使用手机位置数据来帮助抓捕移民。国土安全部下属的另一家机构海关与边境保护局（US Customs and Border Protection）使用该信息在不寻常的地方（例如横跨墨西哥边境的偏远沙漠中）寻找手机活动。”
  </p>
  <p class="graf graf--p">
   国土安全部从
   <a class="markup--anchor markup--p-anchor" data-href="https://venntel.com/" href="https://venntel.com/" rel="noopener noreferrer" target="_blank">
    Venntel
   </a>
   提取其使用的位置数据，
   <strong class="markup--strong markup--p-strong">
    该公司本身不是从电信拿数据的，而是从应用程序公司获取信息
   </strong>
   。
  </p>
  <p class="graf graf--p">
   考虑一下您的地图应用程序、或者移动状态下调整其天气预报的天气应用程序、或者告诉您附近有哪些供应商的食品订购应用程序 …… 这些都是老大哥。
  </p>
  <p class="graf graf--p">
   这些所谓的便捷功能需要时刻跟踪您的动向，而这些动向对包括国土安全部在内的政府间谍来说都是颇具价值的。
  </p>
  <p class="graf graf--p">
   大多数应用程序在其共享数据中都掩盖了个人身份。国土安全部购买的信息应该显示电话的位置以及手机的来回移动位置，但是与 LocationSmart / Securus 跟踪不同，它不应连接到实际身份。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    如果您将多个应用程序中的数据关联起来，或者将生活模式组合在一起，那么就很容易将地图上的移动点与人的身份重新连接起来。
   </strong>
  </p>
  <p class="graf graf--p">
   《纽约时报》关于该行业的一份报告说：“几位从事定位业务的人都表示，在这种数据中找出真实的个人身份会相对简单。”
  </p>

  <p class="graf graf--p">
   警长 Sheriff Hutcheson 对其政治敌人施加的那种个人手机跟踪方式，对于所有愿意并能够获得搜查令的政府机构都适用。
  </p>
  <p class="graf graf--p">
   每个人口袋中的电话充当位置信标，向可能想知道团体聚会地点的机构发出人员的一般动向信号，并向愿意跳过合法范围（或假装这样做）的官员广播每个人的个人位置。
  </p>
  <p class="graf graf--p">
   请记住，无论采取哪种保护措施，您的手机都是一个可怕的东西。携带手机时它会连续记录您去过哪里，并会告诉任何以正确方式提出要求的人。每一位使用智能手机的用户都应明白这点。⚪️
  </p>
