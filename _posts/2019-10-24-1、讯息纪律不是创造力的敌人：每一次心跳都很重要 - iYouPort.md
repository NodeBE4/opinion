  <p class="graf graf--p">
   快速扩大影响力是每一个行动组织最值得期待的事。为此，人们想到了很多办法。
  </p>
  <p class="graf graf--p">
   社交媒体上的群众运动是其中一个典型，也许因为它更为“直观” —— 也许人们将转发量视为了评判效应的标准。
  </p>
  <p class="graf graf--p">
   ⚠️事实上局部的影响力并不是评判效应的标准，整体才是。
  </p>
  <p class="graf graf--p">
   几年前我们分析过《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e4%b8%ad%e5%9b%bd%e7%a4%be%e4%bc%9a%e9%9a%be%e4%bb%a5%e7%bb%84%e7%bb%87%e8%b5%b7%e6%9d%a5/" href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e4%b8%ad%e5%9b%bd%e7%a4%be%e4%bc%9a%e9%9a%be%e4%bb%a5%e7%bb%84%e7%bb%87%e8%b5%b7%e6%9d%a5/" rel="noopener noreferrer" target="_blank">
    为什么中国社会难以组织起来
   </a>
   》，就是以在线组织延伸到现实中为基本设想的。
  </p>
  <p class="graf graf--p">
   很容易想到，在线运动的活力和参与者的积极性延伸到线下形成力量 —— #FreeJulianAssange 运动
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e4%b8%ad%e5%9b%bd%e7%a4%be%e4%bc%9a%e9%9a%be%e4%bb%a5%e7%bb%84%e7%bb%87%e8%b5%b7%e6%9d%a5/" href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e4%b8%ad%e5%9b%bd%e7%a4%be%e4%bc%9a%e9%9a%be%e4%bb%a5%e7%bb%84%e7%bb%87%e8%b5%b7%e6%9d%a5/" rel="noopener noreferrer" target="_blank">
    就是典范之一
   </a>
   ；如果您使用 tweetdeck 长期关注相关标签（包括 #Unity4J）您将能看到这场运动的火爆，
   <strong class="markup--strong markup--p-strong">
    尤其是持续多年的耐力。
   </strong>
  </p>
  <p class="graf graf--p">
   这不是局部影响力能准确判断的。
  </p>
  <p class="graf graf--p">
   比如，Wikileaks 官方账户的转发量和这场运动的所有参与者个人信息的转发量是天壤之别，但绝非意味着后者的“挫败”。
  </p>
  <p class="graf graf--p">
   正相反，
   <strong class="markup--strong markup--p-strong">
    后者作为基数的庞大才是这场运动能够成功的关键所在。
   </strong>
  </p>
  <p class="graf graf--p">
   作为动员者，您应该能做到让所有参与者都乐于为基数作出贡献。
  </p>
  <p class="graf graf--p">
   类似的行动还有很多。另一个最著名的例子是2015年爆发的推特行动：#LATE，“每一次心跳都很重要”。
  </p>
  <p class="graf graf--p">
   这是委内瑞拉社交媒体社会运动的口号，出现在充满活力的彩色艺术品、视频和其他多媒体传播中。
  </p>
  <p class="graf graf--p">
   这场运动是革命艺术家和基层媒体活动家团队的合作，旨在重塑玻利瓦尔革命的审美叙事，并将公社置于其“象征意义中心”。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6092 jetpack-lazy-image" data-lazy-sizes="(max-width: 880px) 100vw, 880px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/呼吸.png?resize=880%2C600&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/呼吸.png?w=880&amp;ssl=1 880w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/呼吸.png?resize=300%2C205&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/呼吸.png?resize=768%2C524&amp;ssl=1 768w" data-recalc-dims="1" height="600" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/呼吸.png?resize=880%2C600&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="880"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   委内瑞拉不断发展的公社运动为新的激进政治和经济民主奠定了基础。#LATE 运动旨在
   <strong class="markup--strong markup--p-strong">
    将革命的想象作为一种不属于政府和党派的集体愿景，而是由社会运动自下而上地塑造出来。
   </strong>
  </p>
  <p class="graf graf--p">
   ＃LATE 没有代表候选人或任何政党进行竞选，相反，是通过实施“民众运动的立法议程”来激化革命。
  </p>
  <p class="graf graf--p">
   目标是发起一项基于思想的运动，这种思想将许多集体联合起来，以面对资本主义危机，答案就是公社。
  </p>
  <p class="graf graf--p">
   这场联合运动的中心级动员者只有 70个人，他们四处走动会面人们，以传播他们的想法。
  </p>
  <p class="graf graf--p">
   ⚠️请注意，最终轰动互联网的口号“每一次心跳都很重要”并非70位策划者的预先构想，而是在他们与人们接触的过程中彼此磋商后 —— 经过辩论和投票 —— 得出的。
  </p>
  <p class="graf graf--p">
   有了口号、确立了主题，行动者们开始分头工作，创建logo、作品、网站和作品“储存库”。
  </p>
  <p class="graf graf--p">
   在这里，
   <strong class="markup--strong markup--p-strong">
    参与者们可以找到所有能用来制作*自己的艺术作品*的资源。所有内容都是可编辑的，参与者只需将徽标放在自己制作的视频或图像上 —— 这是运动的一部分
   </strong>
   。
  </p>
  <p class="graf graf--p">
   💡这是一项不属于任何人的运动 —— 完全是社会财产。这样做是为了让任何人或任何集体都可以重新使用这些作品和元素，并在他们喜欢的情况下重新创建。
  </p>
  <p class="graf graf--p">
   就如活动家所说，
   <strong class="markup--strong markup--p-strong">
    每个人都很重要，每个人都是必要的；每一次心跳都像每一个生命一样，你所做的每一件小事，尽管它们可能都很小，但是在建造这个巨大的项目中，点点滴滴都很重要。
   </strong>
  </p>
  <p class="graf graf--p">
   这场运动的最大特点之一是，💡它并非在某个封闭团体的指挥下进行的，它不属于任何一个集体或个人，所有人都在负责，同时无人负责一切。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这场发自互联网的运动实现了现实中轰轰烈烈的行动主义参与。
   </strong>
  </p>
  <p class="graf graf--p">
   引发了各种街头活动，涉及涂鸦艺术家、壁画家、印花T恤、音乐，舞蹈，马戏团、公共聚餐 …… 由100名涂鸦艺术家组成的大篷车驶入 Chacao 时，掀起了整场行动的风潮。
  </p>
  <p class="graf graf--p">
   一切都是根据人们的日常联系进行组织的：社区、工会、学生会 ……由无数个集体凝聚而成。
  </p>
  <p class="graf graf--p">
   通过这种方式，该活动几乎完全是自我管理的。
  </p>
  <p class="graf graf--p">
   你能想象到，这其中没有“挫败感”，没有人会因为“我自己的转发量不高”而产生消极退却的心理。因为
   <strong class="markup--strong markup--p-strong">
    它构建在集体共同的参与热情之上
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这是一种赋权。
  </p>
  <p class="graf graf--p">
   💡当人们感知到自己的权力时，就不再需要他人的“认可和赞赏”。
  </p>
  <p class="graf graf--p">
   不论是 #Unity4J 还是 #LATE，凝聚力都能深刻体现在作品上 —— 准确说是讯息上；同时，并没有单一的组织和策划。
  </p>
  <p class="graf graf--p">
   那么它们是如何实现令庞大的参与者团体异口同声的？（对于前者，要知道那是跨越10个国家的联动）。
  </p>
  <p class="graf graf--p">
   答案是两个关键点：讯息纪律，和赋权。
  </p>
