    我們如何預測2018年中國 – 大局前瞻

  <div class="entry-content" itemprop="text">
   <p>
    ◎ 2018年初，透視中國正確地分析、預測了幾個關於中國的宏觀大局前景和地緣政治走向。(下)
   </p>
   <p>
    <span id="more-1879">
    </span>
   </p>
   <hr/>
   <p>
    為了避開風險並在中國取得成功，企業和政策制定者需要正確瞭解宏觀大局和地緣政治趨勢。
   </p>
   <p>
    2018年，透視中國領先於媒體、觀察家和其他諮詢公司、智庫，正確地分析、預測了幾個關於中國的宏觀大局前景和地緣政治走向。雖然我們有些預測最初看上去不準確，但隨著事態的發展最終也證實我們的預測分析是正確的。
   </p>
   <p>
    在2018年預測成績單第二部分，我們回顧了5項頂級宏觀大局預測。（請參閱
    <a href="https://tw.sinoinsider.com/2018/12/%e6%88%91%e5%80%91%e5%a6%82%e4%bd%95%e9%a0%90%e6%b8%ac2018%e5%b9%b4%e4%b8%ad%e5%9c%8b-2018%e5%b9%b4%e5%b1%95%e6%9c%9b/">
     2018年預測成績單第一部分
    </a>
    和我們的
    <a href="https://tw.sinoinsider.com/our-track-record/">
     預測分析成績記錄
    </a>
    。）
   </p>
   <ol>
    <li>
     <span style="color: #000080;">
      <strong>
       不斷
      </strong>
      <strong>
       加劇
      </strong>
      <strong>
       的中共派系鬥爭
      </strong>
     </span>
    </li>
   </ol>
   <p>
    <em>
     <img alt="" class="alignright size-medium wp-image-1882" height="190" src="https://tw.sinoinsider.com/wp-content/uploads/sites/2/2019/01/Meeting01-300x190-300x190.jpg" width="300"/>
     我們的預測分析：
    </em>
   </p>
   <p>
    我們認為，中國所有問題的核心是中共高層政治，而中共高層政治的問題是派系鬥爭。
   </p>
   <p>
    因此，雖然主流觀點認為習近平是中共「幾十年來最強大的領導者」，並且在2017年第十九次全國代會後幾乎沒有遭遇挑戰，但我們在
    <a href="https://sinoinsider.com/2018/01/china-in-2018-seven-points-to-watch/#more-2895">
     2018年中國展望
    </a>
    中寫道：習近平的「政治地位仍然有些不穩固」。
   </p>
   <p>
    我們還寫道：「中國的政局將在2018年變得更加動盪。」
   </p>
   <p>
    基於觀察，我們做出了以下幾項政局預測：
   </p>
   <ul>
    <li>
     1月2日：習近平將受到不斷加劇的派系鬥爭和中共體制的挑戰。
    </li>
    <li>
     1月2日：我們再次預測王岐山將成為國家副主席。
    </li>
    <li>
     1月2日：習近平將繼續進行大規模人事調動。
    </li>
    <li>
     1月2日：習近平將繼續開展反腐運動，特別針對金融部門、軍隊及政法系統。
    </li>
    <li>
     2月18日：我們首次
     <a href="https://sinoinsider.com/2018/02/two-sessions-forecasting-state-council-npc-cppcc-national-supervision-commission-personnel-reshuffle/">
      預測
     </a>
     了中共兩會的重要官員人選（118名官員）。
    </li>
    <li>
     3月13日：我們
     <a href="https://sinoinsider.com/2018/03/personnel-watch-an-updated-two-sessions-forecast/">
      更新
     </a>
     了兩會官員人事預測列表。
    </li>
    <li>
     12月17日：
     <a href="https://sinoinsider.com/2018/12/politics-watch-xis-reform-anniversary-speech-signals-political-crisis-in-china/">
      我們說
     </a>
     習近平在改革開放40周年的講話將是中國政局危機的標誌。
    </li>
   </ul>
   <p>
    <em>
     發生了什麼：
    </em>
   </p>
   <ul>
    <li>
     上面我們所預測的宏觀局勢，隨著2018年的過去均得到證實。有關詳細資訊，請參閱我們的
     <a href="https://tw.sinoinsider.com/2018/12/%e6%88%91%e5%80%91%e5%a6%82%e4%bd%95%e9%a0%90%e6%b8%ac2018%e5%b9%b4%e4%b8%ad%e5%9c%8b-2018%e5%b9%b4%e5%b1%95%e6%9c%9b/">
      2018年中國展望成績單第一部分
     </a>
     和我們的
     <a href="https://tw.sinoinsider.com/our-track-record/">
      預測分析成績記錄
     </a>
     。
    </li>
    <li>
     我們在預測兩會官員的人事調動方面
     <a href="https://tw.sinoinsider.com/2018-two-sessions-key-state-personnel-forecast-and-results/">
      準確率超過71％
     </a>
     。
    </li>
    <li>
     3月21日：中共發佈了
     <a href="https://sinoinsider.com/2018/03/politics-watch-decoding-chinas-party-and-state-institutional-reforms/">
      黨政機構改革方案
     </a>
     。 該方案包含針對江澤民政治遺產的改革，包括解散「610辦公室」、將國安情報機構直接受控于習近平。
    </li>
    <li>
     從7月開始，中共內部反對習近平及其政策的聲音越來越多。
    </li>
    <li>
     10月7日：中紀委國家監委官網發佈消息，國際刑警組織主席、中共公安部副部長孟宏偉接受調查。 我們分析說，孟的被捕可能涉及一場反習近平的「軟政變」。（請參閱
     <a href="https://sinoinsider.com/2018/10/politics-watch-meng-hongwei-purge-hints-at-soft-coup-against-xi/">
      這裡
     </a>
     和
     <a href="https://sinoinsider.com/2018/10/politics-watch-fang-fenghuis-expulsion-meng-hongweis-arrest-and-the-factional-soft-coup/">
      這裡
     </a>
     ）
    </li>
    <li>
     10月22日至25日：習近平對廣東進行了一次異常的巡視。
    </li>
    <li>
     12月1日：在與川普總統的晚宴上，習近平承諾進行結構性改革和市場自由化。 我們分析說，自7月以來一直不斷升級的中共派系鬥爭，可能是習近平此次向川普做出不尋常讓步的決定性因素。
    </li>
    <li>
     12月18日：習近平間接地將江澤民與1992年後推動改革的鄧小平相提並論，這是另一個不尋常的舉動，證實了我們對習近平遭遇政敵強力抵抗及中國政治危機的分析。
    </li>
   </ul>
   <p>
    預測結果：
    <span style="color: #008000;">
     非常正確
    </span>
   </p>
   <p>
   </p>
   <p>
    更多詳情，請參閱我們的「
    <a href="https://tw.sinoinsider.com/personnel-forecast/">
     中共高層人事
    </a>
    」的預測分析成績記錄。
   </p>
   <p>
   </p>
   <p>
    <span style="color: #000080;">
     <strong>
      2. 中國無法承受貿易戰
     </strong>
    </span>
   </p>
   <p>
    <em>
     <img alt="" class="alignright size-medium wp-image-1884" height="202" src="https://tw.sinoinsider.com/wp-content/uploads/sites/2/2019/01/Who-won-300x202-300x202.jpg" width="300"/>
     我們的預測分析：
    </em>
   </p>
   <p>
    1月，我們注意到，2018年中國面臨灰犀牛和黑天鵝的「
    <a href="https://sinoinsider.com/2018/01/china-in-2018-seven-points-to-watch/#more-2895">
     概率上升
    </a>
    」。4月，在中美貿易關係緊張加劇之後，我們預測中國將無法承受貿易戰。
   </p>
   <ul>
    <li>
     1月2日：我們在2018年中國展望中列出了中國經濟所面臨的一些風險。
    </li>
    <li>
     4月1日：我們
     <a href="https://sinoinsider.com/2018/04/risk-watch-why-china-wont-touch-soybeans-in-trade-war/">
      預測
     </a>
     在貿易戰中，中國不會以大豆作為籌碼。
    </li>
    <li>
     4月12日：我們
     <a href="https://sinoinsider.com/2018/04/risk-watch-can-china-withstand-a-trade-war/">
      預測
     </a>
     ，基於包括農業、經濟和金融問題等7個方面的原因，中國無法承受貿易戰。
    </li>
   </ul>
   <p>
    <em>
     發生了什麼：
    </em>
   </p>
   <p>
    1月，銀監會一位高官回應了我們對於灰犀牛和黑天鵝的警告。 與此同時，在美國301調查和川普總統推動關稅之後，中國經濟惡化，金融市場暴跌。中國官方11月份的經濟數據以及在12月份急於購買美國大豆，證實了我們在2018年早些時候做出的預測。
   </p>
   <ul>
    <li>
     1月17日，中國銀監會主席
     <a href="https://sinoinsider.com/2018/01/risk-watch-decoding-china-banking-regulator-guo-shuqings-media-interview/">
      郭樹清
     </a>
     告訴官方媒體，中國金融體系脆弱，面臨灰犀牛和黑天鵝的威脅。
    </li>
    <li>
     據美國大豆出口委員會的數據，12月12日，中國購買了150萬至200萬噸美國大豆。 中國的大豆採購表明，我們4月預測中國不會以大豆做籌碼，在方向上是準確的。
    </li>
    <li>
     根據中國國家統計局12月發佈的資料：
     <ul>
      <li>
       11月份，中國規模以上工業增加值同比實際增長5.4%，為三年來最低。
      </li>
      <li>
       11月份零售總額同比增長8.1％，為2003年以來的最低水準。
      </li>
      <li>
       11月，中國出口總額同比增長10.2％，進口同比增長7.8％，均不及10月份增速的一半。
      </li>
      <li>
       中國汽車產量較上年減少3.2％。
      </li>
      <li>
       2018年的中國糧食總產量為65789萬噸，比2017年減少371萬噸（-0.6％）。
      </li>
     </ul>
    </li>
    <li>
     1月份，中國外匯儲備總額為3.161萬億美元。 截至11月份，中國的外匯儲備已下降997.6億美元至3.061萬億美元。
    </li>
    <li>
     1月3日，在岸人民幣匯率為6.5009。 截至12月19日，在岸人民幣匯率已跌至6.8911。 人民幣匯率最高（2月7日）為6.2409，最低（10月31日）為6.9771，波動幅度超過10.55％。
    </li>
    <li>
     截至12月19日，上證指數下跌22.91％，深證指數下跌32.8％。 自年初以來，中國A股總共損失了1.88萬億美元（使用12月12日的6.8911人民幣匯率計算）。
    </li>
   </ul>
   <p>
    預測結果：非常準確
   </p>
   <ul>
    <li>
     「中國無法承受貿易戰」—— 我們提前7個月預測並得到驗證。（12月官方公佈的經濟數據證實了我們4月的預測）。
    </li>
   </ul>
   <p>
   </p>
   <p>
    <span style="color: #000080;">
     <strong>
      3. 中國誤判美國
     </strong>
    </span>
   </p>
   <p>
    <em>
     <img alt="" class="alignright size-medium wp-image-1885" height="200" src="https://tw.sinoinsider.com/wp-content/uploads/sites/2/2019/01/Xi_Trump-300x200-300x200.jpg" width="300"/>
     我們的預測分析：
    </em>
   </p>
   <p>
    8月，我們注意到中國對美國關稅的回應和宣傳，顯示中共誤讀了美國對華政策的根本性轉變。 我們還注意到了美國的對華政策在意識形態上的轉向。
   </p>
   <ul>
    <li>
     8月17日：我們
     <a href="https://sinoinsider.com/2018/08/geopolitics-watch-the-ccp-misreads-americas-china-policy-revolution/">
      分析
     </a>
     說「中共應對美國對華政策的根本性轉變」要麼是「嚴重誤判形勢」，要麼是「對美國中期選舉結果下注，冀望於美國政治的改變為中國重新貿易談判創造有利條件。」
    </li>
    <li>
     8月29日：在分析美中經濟與安全審查委員會8月24日關於中共的海外統戰工作報告時，我們注意到，當美國繼續關注中共的海外滲透並積極採取行動對抗中共宣傳時，美國與中共的戰略競爭將進入意識形態領域。
    </li>
   </ul>
   <p>
    <em>
     發生了什麼：
    </em>
   </p>
   <p>
    川普在聯合國大會上發表講話、副總統邁克·彭斯強硬的中國政策演講，美國高級情報官員的言論以及美國中期選舉結果，證實了我們對地緣政治局勢變動的評估。
   </p>
   <ul>
    <li>
     9月4日：美國國家反間諜和安全中心主任威廉·埃文尼納（William Evanina）說，中國是「存在的、長期的……對我們國家安全的最大威脅」，（中國）正在投入多得可怕的資源對美國展開間諜行動。
    </li>
    <li>
     9月13日：FBI局長克里斯多夫·雷（Christopher Wray）說：「如果我審視我們的整體反間諜任務，中國是我們在這個領域中的首要任務。」
    </li>
    <li>
     9月25日：川普在聯合國表示：「實質上，不管是什麼地方嘗試了社會主義或共產主義，都導致了苦難、腐敗和衰敗。社會主義對權力的渴望導致了擴張、入侵和壓迫。世界上所有國家都應該抵制社會主義及其給每個人帶來的苦難。」
    </li>
    <li>
     10月4日：彭斯副總統在哈德遜研究所的演講中，批評中共的掠奪性貿易政策及其在美國的間諜、滲透和干預行動。
    </li>
    <li>
     11月8日：美國中期選舉結果顯示，中共針對美國農產品加征關稅未能影響選民支持川普和共和黨人。
    </li>
    <li>
     12月1日：在與川普總統的晚宴上，習近平承諾進行市場自由化和結構性改革。 中國也同意大量採購美國農產品和其他產品。
    </li>
    <li>
     12月11日：美國國務院「全球參與中心」宣佈了一項新舉措，以對抗來自中國、俄羅斯和伊朗等流氓國家的宣傳和不實資訊。
    </li>
   </ul>
   <p>
    預測結果：準確
   </p>
   <ul>
    <li>
     「中國正在誤判美國」—— 在驗證前約3個月，我們作出預測（8月17日作出預測， 在11月6日美國中期選舉中得到證實）。
    </li>
    <li>
     「美國對華政策的戰略轉變」——在驗證前一個半月，我們作出預測（8月29日作出預測，在10月4日彭斯副總統的講話中得到證實）。
    </li>
   </ul>
   <p>
    更多詳情，請參閱我們的「
    <a href="https://tw.sinoinsider.com/sino-us-relationship/">
     中美關係轉變
    </a>
    」的預測分析成績記錄。
   </p>

   <p>
    <strong>
     <span style="color: #000080;">
      4.美國正在建立新的
     </span>
    </strong>
    <span style="color: #000080;">
     <strong>
      世界
     </strong>
     <strong>
      秩序
     </strong>
    </span>
   </p>
   <p>
    <em>
     <img alt="" class="alignright size-medium wp-image-1883" height="200" src="https://tw.sinoinsider.com/wp-content/uploads/sites/2/2019/01/Trump-300x200-300x200.jpg" width="300"/>
     我們的預測分析：
    </em>
   </p>
   <p>
    7月26日，川普總統和歐盟委員會主席讓-克洛德·容克（Jean-Claude Juncker）在華盛頓宣佈新的美歐貿易協定後，我們寫道，川普正在「將美國從二戰後的世界秩序中撤出，並建立一個新的世界秩序」。
   </p>
   <ul>
    <li>
     7月27日：我們
     <a href="https://sinoinsider.com/2018/07/geopolitics-watch-with-eu-deal-trump-advances-another-step-in-reshaping-ccp-hijacked-world-order/">
      分析
     </a>
     說，川普正在通過諸如「退出跨太平洋夥伴關係和巴黎氣候協議等行動，重塑被中共劫持的世界秩序；開始與盟國和中國發生貿易衝突；美國退出聯合國人權理事會；並與臺灣建立更緊密的聯繫。」
    </li>
    <li>
     8月22日：在中共挖走臺灣邦交國之後，我們
     <a href="https://sinoinsider.com/2018/08/geopolitics-watch-is-the-prc-isolating-taiwan-or-is-the-prc-being-isolated/">
      分析
     </a>
     說：「中華人民共和國在國際上被孤立的風險更大，而不是中華民國。」
    </li>
   </ul>
   <p>
    <em>
     發生了什麼：
    </em>
   </p>
   <ul>
    <li>
     9月24日：川普和南韓總統文在寅簽署新版美韓自由貿易協定。
    </li>
    <li>
     10月1日：美國、墨西哥、加拿大就更新北美自由貿易協定達成一致，新的貿易協定被命名為「美國－墨西哥－加拿大協定」。11月30日在G20峰會上三國領導人簽署了協定。
    </li>
    <li>
     10月4日：美國副總統邁克·彭斯在哈德遜研究所的演講中指出，「中國共產黨在1949年掌權之後，開始了專制的擴張主義」，「美國將永遠相信臺灣對民主的擁抱，為所有中國人民提供了更好的道路。」
    </li>
    <li>
     10月16日：川普政府宣佈將分別與日本、歐盟和英國洽談貿易協定。
    </li>
    <li>
     10月17日：白宮宣佈美國退出萬國郵政聯盟。
    </li>
    <li>
     12月4日：美國國務卿邁克·龐佩奧在一次題為「恢復民族國家在自由國際秩序中的作用」的演講中說，美國正在「召集具有道義的國家建立一個新的自由秩序以防止戰爭，並實現更大的繁榮。」 龐佩奧說：「中國的經濟發展並沒有帶來民主和地區穩定； 它引發了更多的政治壓迫和地區挑釁。」他補充說，像伊朗和中國這樣的國家正在「破壞國際秩序」。
    </li>
    <li>
     12月12日：美國國會通過了「亞洲再保證倡議法案」，該法案旨在「為印度 – 太平洋地區制定一項長期戰略願景和全面、多方面和有原則的美國政策。」該法案中有一節涉及美國對臺灣的承諾。
    </li>
   </ul>
   <p>
    預測結果：方向性準確
   </p>
   <ul>
    <li>
     「美國正在尋求建立一個新的世界秩序」—— 在驗證前4個月，我們作出預測（7月27日我們作出預測，在12月4日龐佩奧發表的演講中得到證實）。
    </li>
   </ul>
   <p>
    更多詳情，請參閱我們的「
    <a href="https://tw.sinoinsider.com/sino-us-relationship/">
     中美關係轉變
    </a>
    」的預測分析成績記錄。
   </p>
   <p>
   </p>
   <p>
    <strong>
     <span style="color: #000080;">
      5.朝鮮半島的和平和無核化
     </span>
    </strong>
   </p>
   <p>
    <em>
     <img alt="" class="alignright size-medium wp-image-1881" height="202" src="https://tw.sinoinsider.com/wp-content/uploads/sites/2/2019/01/Kim-Trump-summit-300x202-300x202.jpg" width="300"/>
     我們的預測分析：
    </em>
   </p>
   <p>
    3月份，朝鮮領導人金正恩（Kim Jong Un）突然前往北京，這使我們修改了在1月份對朝鮮無核化的預測。 早些時候，我們寫道，金正恩「不會放棄他的核武器。」金正恩北京之行後，我們寫道，朝鮮半島的無核化與和平現在已具有非常現實的可能性。
   </p>
   <p>
    儘管川普在5月取消了川金會，但根據對朝鮮局勢的評估，我們預測，川金會仍將在6月如期舉行。
   </p>
   <ul>
    <li>
     3月26日：我們
     <a href="https://sinoinsider.com/2018/03/the-geopolitical-contest-behind-kim-jong-uns-visit-to-china/">
      預測
     </a>
     金正恩可以尋求達成協議，交換核武器以解除制裁並維持他對朝鮮的統治。 我們還預測，金正恩最頭痛的問題是：在其國內宣傳上，如何向朝鮮人民解釋他決定放棄核武器。
    </li>
    <li>
     3月28日：我們
     <a href="https://sinoinsider.com/2018/03/xi-kim-summit-foreshadows-peace-on-the-korean-peninsula/">
      預測
     </a>
     朝鮮將實現無核化，「朝鮮半島和平的可能性非常高」。
    </li>
    <li>
     3月30日：在韓國流行音樂明星和其他藝術家赴平壤演出後，我們
     <a href="https://sinoinsider.com/2018/03/politics-watch-decoding-k-pop-diplomacy-in-north-korea/">
      預測
     </a>
     金正恩可能會在朝鮮實施改革開放。
    </li>
    <li>
     4月27日：我們
     <a href="https://sinoinsider.com/2018/04/why-we-predicted-coming-peace-on-the-korean-peninsula/">
      預測
     </a>
     川普總統希望朝鮮在其第一個任期內實現無核化。
    </li>
    <li>
     5月24日：在川普取消他與金正恩的新加坡峰會之後，我們
     <a href="https://sinoinsider.com/2018/05/geopolitics-watch-in-canceling-north-korea-summit-trump-gets-two-birds-with-one-stone/">
      預測
     </a>
     川金會仍將按計劃於6月舉行。
    </li>
    <li>
     6月15日：在川普宣佈對中國商品徵收關稅後，我們
     <a href="https://sinoinsider.com/2018/06/geopolitics-watch-the-potential-dire-implications-of-us-tariffs-on-china/">
      預測
     </a>
     朝鮮無核化進程將受到影響。
    </li>
   </ul>
   <p>
    <em>
     發生了什麼：
    </em>
   </p>
   <p>
    儘管平壤出現了一些「倒退」現象，但朝鮮半島的和平與無核化進程在2018年趨向於積極的方向。金正恩3月北京之行後的主要發展包括：
   </p>
   <ul>
    <li>
     4月20日：在朝鮮勞動黨第七屆中央委員會第三次全體會議上，金正恩宣佈朝鮮將集中精力發展經濟，提高人民生活水準。
    </li>
    <li>
     4月27日：金正恩和韓國總統文在寅在非軍事區會面，並簽署了《為實現朝鮮半島和平、繁榮和統一的板門店宣言》。該宣言包括朝鮮完全無核化。
    </li>
    <li>
     5月26日：在川普取消與金正恩的峰會後，金正恩和文在寅令人驚訝地在板門店舉行了第二次會晤。
    </li>
    <li>
     6月12日：川普和金正恩在新加坡舉行會晤並簽署一份聯合聲明，重申「板門店宣言」，並立即處理朝鮮戰爭戰俘和失蹤人員遺骸工作。
    </li>
    <li>
     9月18日至20日：金正恩和文在寅在平壤舉行第三次會晤。 9月19日，金和文簽署了一項聯合協定，韓國將該協定定義為結束朝鮮戰爭的宣言。
    </li>
    <li>
     9月20日：美國國務卿龐佩奧設定2021年1月為朝鮮無核化的最後期限。
    </li>
    <li>
     11月20日：朝鮮在非軍事區拆除了10個哨所。
    </li>
    <li>
     12月13日：首爾宣佈，12月26日，朝鮮和韓國將舉行開工儀式，重新連接半島的鐵路和公路。
    </li>
   </ul>
   <p>
    預測結果：朝鮮問題仍在發展中，我們的預測方向上是正確的。
   </p>
