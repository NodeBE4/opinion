  <p class="graf graf--p">
   近日，人权观察发布了一份关于中国政府监视公民的令人不安的报告：强行安装监视应用程序。该应用程序为执法部门提供了实时监视的便利。
  </p>
  <p class="graf graf--p">
   这份68页的报告《中国的算法暴政：对新疆警方大规模监控APP的逆向工程》列出了新的证据说明了新疆的监控国家机器的运作 —— 当地1300万突厥裔穆斯林正遭到政府以“严打暴恐活动专项行动”之名加强的镇压。
  </p>
  <p class="graf graf--p">
   从2018年1月到2019年2月，人权观察对一种公安机关使用的手机APP进行逆向工程研究，该APP可以连结一体化联合作战平台（一体化平台），即 新疆警方汇集个人数据、标示潜在“威胁”人员的警务方案。
  </p>
  <p class="graf graf--p">
   通过检测这款当时公开可得的APP的内部设计，人权观察揭露了这种大规模监控系统针对的目标对象包括哪些行为和人员。
  </p>
  <p class="graf graf--p">
   人权观察也发布了这款APP的截屏图片。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-3718 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中-1.png?resize=1100%2C578&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中-1.png?w=1344&amp;ssl=1 1344w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中-1.png?resize=300%2C158&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中-1.png?resize=768%2C403&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中-1.png?resize=1024%2C538&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中-1.png?resize=1100%2C578&amp;ssl=1 1100w" data-recalc-dims="1" height="578" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中-1.png?resize=1100%2C578&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   新疆当局正在从一般民众身上收集大量不同类型的资料，从血型到身高、从“宗教气质”到政治倾向。
   <strong class="markup--strong markup--p-strong">
    该警务平台以36种人为数据收集目标
   </strong>
   ，包括停用智能手机的、避免“与邻居社交”的、以及“热中为清真寺筹募资金或物品” 的人等等。
  </p>
  <p class="graf graf--p">
   一体化平台追踪新疆每一个人。
   <strong class="markup--strong markup--p-strong">
    借助追踪手机、车辆和身分证，它可以全方位监控人们的动态，还可以记录用电和加油的状况
   </strong>
   。
  </p>
  <p class="graf graf--p">
   人权观察发现，该系统和遍布当地的检查哨共同构成了一道隐形围栏。居民的行动自由受到不同程度的限制，取决于当局认定的威胁程度，并由系统内建的编程因素决定。
  </p>
  <p class="graf graf--p">
   当局对一体化平台做了编程，
   <strong class="markup--strong markup--p-strong">
    许多稀松平常的合法活动都被视为可疑行为的指标
   </strong>
   ，例如：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    该系统能够监测出到加油站买汽油的人与该汽车的注册车主不同。这时该APP便会向附近的政府官员推送警报，通过下拉式选单显示不匹配的原因，以决定此案是否可疑而需要出警调查。
   </li>
   <li class="graf graf--li">
    该APP可以监测到人员出国旅行的时间，提示官员审问“逾期”人员或其亲友有关其旅行的详情。该APP还会指示官员检查“逾期”人员的手机是否存有“可疑内容”。
   </li>
  </ul>
  <p class="graf graf--p">
   人权观察发现，
   <strong class="markup--strong markup--p-strong">
    这种调查有时包括检查手机中是否装载51种被列为可疑的互联网工具，包括 WhatsApp、Viber、Telegram 和虚拟专用网（VPN）
   </strong>
   。
  </p>
  <p class="graf graf--p">
   一体化平台系统也会监视人们的交往关系，例如，凡是被侦测到与警察监控名单上的人员一同旅行，或者与最近新办手机门号的人员有关，都会被视为可疑。
  </p>
  <p class="graf graf--p">
   根据这些含糊笼统的标准，该系统可以生成可疑人员名单，由官员决定是否加以拘捕。有官方文件提到“该抓就抓”，显见其目标是最大限度地拘捕所有“政治上不可靠”人员。
  </p>
  <p class="graf graf--p">
   被捕人员将在基本程序保障阙如的情况下遭到审问。他们没有请律师的权利，有些人遭到酷刑或其他虐待，也无法获得有效救济。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该应用程序在很大程度上依赖于 Face ++提供的面部识别软
   </strong>
   件，Face ++是中国创业公司 Megvii 的一个部门，这种关系引发了对 Megvii 投资者的疑问。
  </p>
  <p class="graf graf--p">
   其中最著名的投资者之一是阿里巴巴集团控股公司（Alibaba Group Holding），该公司由中国最富有的亿万富翁马云（Jack Ma）共同创立，是该国企业家形象的象征。
  </p>
  <p class="graf graf--p">
   但是，⚠️
   <strong class="markup--strong markup--p-strong">
    请注意，该公司的另一位著名投资者是：亨特拜登
   </strong>
   。
  </p>
  <p class="graf graf--p">
   亨特拜登是前副总统乔·拜登的儿子，在过去十年中大部分时间都在建立海外投资和商业交易。此事可能会使他父亲竞选总统的努力变得复杂，造成一系列潜在的利益冲突。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-3719 jetpack-lazy-image" data-lazy-sizes="(max-width: 934px) 100vw, 934px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中-2.png?resize=934%2C708&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中-2.png?w=934&amp;ssl=1 934w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中-2.png?resize=300%2C227&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中-2.png?resize=768%2C582&amp;ssl=1 768w" data-recalc-dims="1" height="708" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中-2.png?resize=934%2C708&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="934"/>

  <p class="graf graf--p">
   亨特拜登在中国的投资公司，即 Bohai Harvest RST，汇集了大部分来自国有风险投资的资金，用于购买或投资美国和中国的一系列行业。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Bohai Harvest 已经将资金投入汽车公司、矿业公司和技术企业
   </strong>
   ，比如滴滴楚星科技，这是继优步之后全球最大的乘车公司之一。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2017年，Bohai Harvest
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://cn.reuters.com/article/ai-idCNL4S1N81E1" href="https://cn.reuters.com/article/ai-idCNL4S1N81E1" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     收购了
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    Face ++，这是该公司C轮投资中的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://kuaixun.stcn.com/2017/1031/13731434.shtml" href="http://kuaixun.stcn.com/2017/1031/13731434.shtml" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     4.6亿美元的一部分
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。 Bohai Harvest 的网站在其投资组合中
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://en.bhrpe.com/newsitem/278251356" href="http://en.bhrpe.com/newsitem/278251356" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     展示了
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    Face ++
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Bohai Harvest 与多家基金合作进行各种投资，
   <strong class="markup--strong markup--p-strong">
    这种纠结的业务结构使亨特拜登与中国有影响力的政府和商业人士紧密相连。
   </strong>
  </p>
  <p class="graf graf--p">
   Bohai Harvest 严重依赖国有中国银行的国际子公司为其投资提供资金，在其
   <a class="markup--anchor markup--p-anchor" data-href="http://www.bhrpe.com/newsitem/277805544" href="http://www.bhrpe.com/newsitem/277805544" rel="noopener noreferrer" target="_blank">
    网站上
   </a>
   称自己为“中国银行旗下投资平台”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该投资基金还与海航集团的子公司合作，众所周知，海航是一家备受争议的企业集团
   </strong>
   ，该集团已收购了全球各类企业的投资。
  </p>
  <p class="graf graf--p">
   正如 The Intercept 之前
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2017/06/02/hna-group-corruption-scaramucci-trump-jeb-bush-clinton-guo-wengui/" href="https://theintercept.com/2017/06/02/hna-group-corruption-scaramucci-trump-jeb-bush-clinton-guo-wengui/" rel="noopener noreferrer" target="_blank">
    报道的
   </a>
   那样，海航集团为培养与美国官员的关系做出了异乎寻常的广泛努力。该公司提出了收购前白宫官员 Anthony Scaramucci 所拥有的对冲基金的要约；保留了前美国驻华大使骆家辉的法律服务；并为杰布·布什支持的私募股权公司提供融资。特别值得一提的是，海航集团还追求与比尔克林顿的关系，宣传其与这位前总统在公司举办的慈善活动中的会面。
  </p>

  </figure>
  <p class="graf graf--p">
   在2016年总统大选后不久，中国银行将交通部长 Elaine Chao 的姐姐 Angela Chao 和参议院多数党领袖 Mitch McConnell（R-Ky）的嫂子
   <a class="markup--anchor markup--p-anchor" data-href="https://www.politico.com/story/2018/05/06/elaine-chao-father-james-transportation-department-569686" href="https://www.politico.com/story/2018/05/06/elaine-chao-father-james-transportation-department-569686" rel="noopener noreferrer" target="_blank">
    加入了
   </a>
   其董事会。
  </p>
  <p class="graf graf--p">
   与一些与政界有关的美国人一样，亨特拜登在中国的投资工具是在过去10年中达成的一系列交易的结果。
  </p>
  <p class="graf graf--p">
   2008年，在当年的总统竞选活动结束时，亨特拜登与 Joe Biden 共同创立了该公司，他是 Joe Biden 的长期筹款和法律顾问。
  </p>
  <p class="graf graf--p">
   第二年，亨特拜登、以及前国务卿约翰克里的继子 Christopher Heinz、Kerry-Heinz 的家族朋友 Devon Archer、和前 Oldaker 的合伙人 Eric Sc​​hwerin，一起使用 Rosemont Seneca 这个名字成立了几家公司。
  </p>
  <p class="graf graf--p">
   2014年，他们
   <a class="markup--anchor markup--p-anchor" data-href="https://nypost.com/2018/03/15/inside-the-shady-private-equity-firm-run-by-kerry-and-bidens-kids/" href="https://nypost.com/2018/03/15/inside-the-shady-private-equity-firm-run-by-kerry-and-bidens-kids/" rel="noopener noreferrer" target="_blank">
    开始在中国开展业务
   </a>
   。
   <strong class="markup--strong markup--p-strong">
    Bohai Harvest RST 的“RS”代表 Rosemont Seneca，“T”代表 Thornton Group。后者是一家总部设在马萨诸塞州的国际咨询公司，由詹姆斯·布尔格（James Bulger）创立，他是克里的长期盟友、前马萨诸塞州参议院议长 William Bulger 的儿子
   </strong>
   。
  </p>
  <p class="graf graf--p">
   据“华尔街日报”
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wsj.com/articles/bohai-harvest-and-u-s-investment-firms-expand-target-for-outbound-fund-1404956572" href="https://www.wsj.com/articles/bohai-harvest-and-u-s-investment-firms-expand-target-for-outbound-fund-1404956572" rel="noopener noreferrer" target="_blank">
    报道
   </a>
   ，该公司计划筹集15亿美元，利用上海的自由贸易区将人民币兑换成美元，投资于外国公司。中国的商业登记文件将亨特拜登、 Schwerin, 和 James Bulger 列为 Bohai Harvest 的主要官员。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “纽约时报”对 Hunter Biden 参与乌克兰能源公司 Burisma Holdings 一事提出了类似担忧，该公司于2014年将副总统的儿子加入公司董事会。据透露，在美国密切参与乌克兰应对俄罗斯在该地区的侵略时，该能源公司每月向亨特拜登
   <a class="markup--anchor markup--p-anchor" data-href="https://www.scribd.com/document/404001731/Rosemont-Seneca-Partners-Court-File" href="https://www.scribd.com/document/404001731/Rosemont-Seneca-Partners-Court-File" rel="noopener noreferrer" target="_blank">
    支付多达5万美元
   </a>
   。
  </p>
  <p class="graf graf--p">
   就他本人而言，乔·拜登长期以来一直是美中关系的友好代言人，甚至在其儿子投资之前就是。拜登认为，开放的中国将进一步促进贸易，为该国公民创造“通向更大政治和经济自由的道路”。
  </p>
  <p class="graf graf--p">
   的确很自由，自由到了资助老大哥的地步。
  </p>
  <p class="graf graf--p">
   人权观察表示，中国政府应该立即关闭该一体化平台，删除从新疆居民收集到的所有数据。相关各国政府都应对新疆党委书记陈全国和涉及严打行动侵权行为的其他高级官员实施针对性制裁，例如根据美国《全球马格尼茨基法案》对他们实施签证禁令和资产冻结。各国也应该适度启动出口管制机制，以防中国政府获得用于侵犯基本人权的技术。联合国会员国应推动成立国际实况调查团，评估新疆情势，并向联合国人权理事会报告。
  </p>
