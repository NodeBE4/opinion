  <p class="graf graf--p">
   掌握调查的基本原则远胜过掌握技术 —— 这非常重要。也是为什么我们需要不断强调这点，《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/osint-%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e5%ba%94%e8%af%a5%e6%98%af%e4%b8%80%e7%a7%8d%e5%bf%83%e6%99%ba%ef%bc%9a%e5%be%97%e5%88%b0%e5%ae%83%e4%bd%a0%e9%9c%80%e8%a6%81%e5%87%a0%e4%b8%aa%e6%ad%a5/" href="https://www.iyouport.org/osint-%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e5%ba%94%e8%af%a5%e6%98%af%e4%b8%80%e7%a7%8d%e5%bf%83%e6%99%ba%ef%bc%9a%e5%be%97%e5%88%b0%e5%ae%83%e4%bd%a0%e9%9c%80%e8%a6%81%e5%87%a0%e4%b8%aa%e6%ad%a5/" rel="noopener noreferrer" target="_blank">
    开源调查应该是一种心智
   </a>
   》。
  </p>
  <p class="graf graf--p">
   将正确的方法应用于问题才是首要的，而非拼命去搜罗更多的工具。
  </p>
  <p class="graf graf--p">
   这有很多原因。但最主要的原因就是，您喜欢使用的几乎所有工具和技术都是相对短暂的，并且很快就会过时。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/facebook-%e6%ad%a3%e5%9c%a8%e4%bb%a4%e7%9c%9f%e7%9b%b8%e6%8c%96%e6%8e%98%e4%ba%8b%e4%b8%9a%e9%99%b7%e5%85%a5%e5%9b%b0%e5%a2%83/" href="https://www.iyouport.org/facebook-%e6%ad%a3%e5%9c%a8%e4%bb%a4%e7%9c%9f%e7%9b%b8%e6%8c%96%e6%8e%98%e4%ba%8b%e4%b8%9a%e9%99%b7%e5%85%a5%e5%9b%b0%e5%a2%83/" rel="noopener noreferrer" target="_blank">
    Facebook 图谱搜索的消亡
   </a>
   是众所周知的例子。
  </p>
  <p class="graf graf--p">
   新技术经常令人兴奋，但是，有一些非常古老的技术由于
   <strong class="markup--strong markup--p-strong">
    高度实用
   </strong>
   ，生命力非常顽强；当这些古老的技术与现代技术相结合时，
   <strong class="markup--strong markup--p-strong">
    它们是任何开源情报调查武器库中的有用部分
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Boolean_algebra" href="https://en.wikipedia.org/wiki/Boolean_algebra" rel="noopener noreferrer" target="_blank">
    布尔搜索
   </a>
   背后的数学逻辑可以追溯到19世纪，它仍支持现代计算机技术、编程和网络搜索等。
  </p>
  <p class="graf graf--p">
   它还提供了一个逻辑结构，可用于评估命题并确定某些事物是真还是假 —— 这最终是任何类型调查的目的。
  </p>
  <p class="graf graf--p">
   本文将介绍如何使用布尔逻辑来解决侦探挑战，以及如何将它们应用于进行高度针对性的 Google Image 搜索。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9103 jetpack-lazy-image" data-lazy-sizes="(max-width: 968px) 100vw, 968px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-2.jpg?resize=968%2C780&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-2.jpg?w=968&amp;ssl=1 968w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-2.jpg?resize=300%2C242&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-2.jpg?resize=768%2C619&amp;ssl=1 768w" data-recalc-dims="1" height="780" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-2.jpg?resize=968%2C780&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="968"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9103" data-recalc-dims="1" height="780" sizes="(max-width: 968px) 100vw, 968px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-2.jpg?resize=968%2C780&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-2.jpg?w=968&amp;ssl=1 968w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-2.jpg?resize=300%2C242&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-2.jpg?resize=768%2C619&amp;ssl=1 768w" width="968"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    OR, AND, NOT
   </strong>
  </p>
  <p class="graf graf--p">
   无论多大或多小，任何调查的目标都是得出一系列可以证明是真的命题，任何不正确的命题都被消除并被证明是假。
  </p>
  <p class="graf graf--p">
   例如，如果你现在需要调查银行抢劫案，并且证人将嫌疑人描述为：25岁以下的白人男子、身穿蓝色外套、当地口音，那么你应该知道，当所有这些条件都为真时符合嫌疑人描述的人群在哪里可以找到。
  </p>
  <p class="graf graf--p">
   如果这些条件中的任何一个不属于嫌疑人，则可以将其删除（这里忽略了证人可靠性和其他现实因素的问题）。
  </p>
  <p class="graf graf--p">
   因此，如果你有一个神奇的数据库，就可以通过搜索找到谁抢了银行 —— 使用布尔逻辑设置参数来找到嫌疑人。
  </p>
  <p class="graf graf--p">
   应该是这样的：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    white male AND under 25 AND blue jacket AND local accent
   </strong>
  </p>
  <p class="graf graf--p">
   假设每个可能的人都在这个庞大的数据库中，那么嫌疑人就会藏在搜索结果中。任何不符合这些标准的人都可以被淘汰。
  </p>
  <p class="graf graf--p">
   在 Google 搜索中，AND运算符（更常见的是+号）用于实现相同的结果。
  </p>
  <p class="graf graf--p">
   搜索
   <strong class="markup--strong markup--p-strong">
    white male AND under 25
   </strong>
   将返回25岁以下的白人男性。
  </p>
  <p class="graf graf--p">
   组合多个AND查询将返回所有条件都为真的搜索结果。用以搜索特定信息。
  </p>
  <p class="graf graf--p">
   现在假设你和银行抢劫案中的另一名证人交谈过。这名证人并不认为嫌疑人穿着蓝色外套，而且她相当肯定地说那是一件绿色夹克。
  </p>
  <p class="graf graf--p">
   这使搜索变得复杂一点，因为现在不再可能确定嫌疑人穿着蓝色夹克了。
  </p>
  <p class="graf graf--p">
   这会导致一些不确定性，这意味着你必须稍微更改数据库上的搜索查询。输入OR运算符。
  </p>
  <p class="graf graf--p">
   当有多个可能的查询答案时，OR运算符有助于构造搜索项。
  </p>
  <p class="graf graf--p">
   现在你怀疑穿着什么颜色的夹克这个问题有一点不确定性，那么就可以通过添加OR运算符来修改搜索：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    white male AND under 25 AND local accent AND blue jacket OR green jacket
   </strong>
  </p>
  <p class="graf graf--p">
   所以现在这个搜索将返回所有穿着蓝色夹克和绿色夹克的嫌疑人的详细信息。出于调查目的，OR运算符允许在搜索参数中考虑一点点不确定性。
  </p>
  <p class="graf graf--p">
   这样的搜索结果中仍会呈现过多的杂音；你需要添加更多的参数来缩小结果范围。
  </p>
  <p class="graf graf--p">
   你可以在结果为真的情况下添加的条件越多，意味着有更高的机会找到嫌疑人。
  </p>
  <p class="graf graf--p">
   现在你决定再次询问证人以获取更多信息。
  </p>
  <p class="graf graf--p">
   证人们都同意一点：对于一个成年男子来说，嫌疑人的身材显得非常小，可能不超过160厘米。
  </p>
  <p class="graf graf--p">
   这个信息允许你继续调整搜索，并引入NOT运算符。
  </p>
  <p class="graf graf--p">
   这用于过滤条件不正确的结果。你可以将它应用于搜索查询，如下所示：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    white male AND under 25 AND local accent AND blue jacket OR green jacket
   </strong>
   <strong class="markup--strong markup--p-strong">
    NOT taller than 160cm
   </strong>
  </p>
  <p class="graf graf--p">
   通过将 NOT 运算符添加到已存在OR运算符的搜索中，你可以消除每个超过160cm的嫌疑人的可能结果，进一步缩小嫌疑人池。
  </p>
  <p class="graf graf--p">
   就是这么简单。
  </p>
  <p class="graf graf--p">
   因此，
   <strong class="markup--strong markup--p-strong">
    你所知道的事实越多，调查中的主题就越正确，可以应用的标准就越多，以消除不正确的答案并找到正确的答案
   </strong>
   。
  </p>
  <p class="graf graf--p">
   布尔运算符是执行此操作的最佳方法，无论是在进行搜索查询时，还是在考虑问题或进行差距分析等操作时，都非常常用。
  </p>
  <p class="graf graf--p">
   它有助于保持调查的重点，阻止人们只是猜测或只是乱撞。
  </p>
  <p class="graf graf--p">
   一切都很容易，是吧？
  </p>
  <p class="graf graf--p">
   接下来本文将展示如何应用这个想法，构建一个布尔搜索词，它将帮您找到侦探挑战的答案 —— 照片地理定位挑战。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    挑战的题目是这样的
   </strong>
  </p>
  <p class="graf graf--p">
   这个地理定位挑战采用 Philipp Dudek 发布的短视频：
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="618" src="//player.vimeo.com/video/358780957?title=0&amp;byline=0&amp;portrait=0&amp;color=8dc7dc" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   在确定该位置必须符合的标准之前，仔细听声音以及观看视频非常重要，以此来分析必须使用哪些布尔搜索运算符来查找它。
  </p>
  <p class="graf graf--p">
   应该强调一下，这种方法并不是解决这一特定挑战的唯一方法，但
   <strong class="markup--strong markup--p-strong">
    如果您能了解底层逻辑的重要性，那么当遇到困难时就可以顺利转入采取其他方案
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    使用哪种搜索标准？
   </strong>
  </p>
  <p class="graf graf--p">
   起初并不完全清楚这是什么。
  </p>
  <p class="graf graf--p">
   Philipp 正沿着某条隧道行驶。这地方是火车隧道还是地铁站？
  </p>
  <p class="graf graf--p">
   你能听到移动过程中会发出有节奏的机械声 —— 可能是火车、电梯、电动步道、还是自动扶梯？
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9104 jetpack-lazy-image" data-lazy-sizes="(max-width: 768px) 100vw, 768px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-2.jpg?resize=768%2C1024&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-2.jpg?w=768&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-2.jpg?resize=225%2C300&amp;ssl=1 225w" data-recalc-dims="1" height="1024" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-2.jpg?resize=768%2C1024&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="768"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9104" data-recalc-dims="1" height="1024" sizes="(max-width: 768px) 100vw, 768px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-2.jpg?resize=768%2C1024&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-2.jpg?w=768&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-2.jpg?resize=225%2C300&amp;ssl=1 225w" width="768"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   唯一具有视觉特色的是整个墙上的原点图案。这足以实现地理定位吗？
  </p>
  <p class="graf graf--p">
   最终会是这样。但我们可以添加的参数越多，就越容易确定。
  </p>
  <p class="graf graf--p">
   从之前的测验可以得知 Philipp 住在德国，所以也可以添加“德国”作为搜索参数之一。
  </p>
  <p class="graf graf--p">
   因此，将所有这些变量考虑在内的搜索词看起来就像下面这样。用+号替换了AND，它意味着同样的事：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Germany + tunnel OR escalator OR elevator + dots
   </strong>
  </p>
  <p class="graf graf--p">
   此搜索结合了可以确定的位置标准（Germany, dots），然后使用OR运算符搜索可能与正确位置相关的事物，但仍存在一些疑点（隧道，电梯，自动扶梯等）。
  </p>
  <p class="graf graf--p">
   搜索结果将显示这些参数的所有匹配项，因此正确的结果将存在于某处。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Google Images 还是 Yandex?
   </strong>
  </p>
  <p class="graf graf--p">
   在进行这种类型的布尔图像搜索时，搜索引擎的选择会有所不同。
  </p>
  <p class="graf graf--p">
   通常有经验的调查人员更喜欢 Yandex 用于图像搜索，因为它非常适合查找视觉匹配，因此对地理定位很有用。
  </p>
  <p class="graf graf--p">
   Google图片搜索通常更擅长确定图片显示的对象类型，然后根据相关关键字建议匹配。
