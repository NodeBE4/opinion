 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      “病态的和贫困的经济基础设施导致了病态和贫困的身体或思想”。为什么不平等和有缺陷的经济体系相互促进？
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   就像镀金涂层能让那些最无聊的东西闪闪发光一样，今天，薄薄的政治民粹主义镀金层覆盖了一个怪异的不平等的下层，这种不平等一直隐藏在明显的视野中。这种财富和权力日益集中的现象既有牛顿也有达尔文的成分。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    就如牛顿的第一运动定律所言：除非受到外力的影响，否则掌权者将继续执政。那些富有的人只有在没有任何东西偏离他们的既有路线的情况下才能获得财富；至于达尔文，在金融发展的世界里，那些拥有财富或权力的人会尽其所能保护自己财富，即使严重损失其他人的利益
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在乔治奥威尔1945年的标志性小说“动物农场”中，在反对人类农民的叛乱中获得控制权的猪最终在一条诫命的基础上对其他动物施加了独裁统治：“所有动物都是平等的，但有些动物比其他动物更平等。” 放在当下的现实中，这句话就是：“所有公民都是平等的，但富人比其他任何人都更平等（并且计划保持这种方式）。
  </p>
  <p class="graf graf--p">
   当然，不平等是有权力的人与没有权力的人之间的经济长城。对美国而言就是，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/business/2017/nov/08/bill-gates-jeff-bezos-warren-buffett-wealthier-than-poorest-half-of-us" href="https://www.theguardian.com/business/2017/nov/08/bill-gates-jeff-bezos-warren-buffett-wealthier-than-poorest-half-of-us" rel="noopener noreferrer" target="_blank">
    三个美国人现在拥有的财富与社会下半部（1.6亿人）一样多
   </a>
   ，我们生活在一个越来越奥威尔的社会里。
  </p>
  <p class="graf graf--p">
   毕竟，马克吐温和查尔斯达德利华纳已经在1873年写了一部经典小说，在他们的时代里创造了一个令人难忘的标签，至今仍令人感触深刻。“镀金时代：今日故事”描绘了内战后这个国家的贪婪和政治腐败。这是一个充斥着强盗贵族和难以理解的富有银行巨头的时期。（听起来有点熟悉吗？）
   <strong class="markup--strong markup--p-strong">
    上个世纪的镀金时代和当下时期之间的主要区别在于，当年那些强盗贵族建造了铁路等有形的东西；今天的超级富豪构建了非常无形的东西，如科技和电子平台，而一位叫总统的贪婪富豪选择了唯一的有形基础设施，那是一面一望无际的大墙
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在马克吐温的时代，美国正在摆脱内战。机会主义者正从国家受到重创的灵魂的灰烬中崛起。土地投机、政府游说和阴暗交易很快融合在一起了，形成了第一阶的不平等社会（至少延续到现在为止）。在他们的小说出版后不久，一系列经济衰退蹂躏了这个国家，随后是由投机者主导的铜市场骗局引发的1907年
   <a class="markup--anchor markup--p-anchor" data-href="https://www.federalreservehistory.org/essays/panic_of_1907" href="https://www.federalreservehistory.org/essays/panic_of_1907" rel="noopener noreferrer" target="_blank">
    纽约金融恐慌
   </a>
   。
  </p>
  <p class="graf graf--p">
   从19世纪90年代后期开始，这个星球上最有影响力的银行家J.P.摩根被多次呼吁在经济边缘拯救一个国家。1907年，美国财政部长乔治·科尔特利（George Cortelyou）应总统西奥多·罗斯福（Theodore Roosevelt）的要求向他提供了
   <a class="markup--anchor markup--p-anchor" data-href="https://books.google.com/books?id=klEeBQAAQBAJ&amp;q=%2425+million#v=snippet&amp;q=%2425%20million&amp;f=false" href="https://books.google.com/books?id=klEeBQAAQBAJ&amp;q=%2425+million#v=snippet&amp;q=%2425%20million&amp;f=false" rel="noopener noreferrer" target="_blank">
    2500万美元的救助资金
   </a>
   ，以稳定华尔街，并平息疯狂的挤兑，当时公民正试图从全国各地的银行取款。摩根做了，他通过帮助他的朋友和他们的公司同时自己捞钱。至于困扰普通百姓的银行，好吧，关门大吉了。
  </p>
  <p class="graf graf--p">
   从政府那里获得这笔赏金的主要银行家继续引发了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.federalreservehistory.org/essays/stock_market_crash_of_1929" href="https://www.federalreservehistory.org/essays/stock_market_crash_of_1929" rel="noopener noreferrer" target="_blank">
    1929年的崩溃
   </a>
   。毫不奇怪，之前就已经有很多猜测和欺诈。那些年，小说家F.斯科特菲茨杰拉德在“了不起的盖茨比”中
   <a class="markup--anchor markup--p-anchor" data-href="https://www.goodreads.com/quotes/10367-let-me-tell-you-about-the-very-rich-they-are" href="https://www.goodreads.com/quotes/10367-let-me-tell-you-about-the-very-rich-they-are" rel="noopener noreferrer" target="_blank">
    抓住了这个时代最怪诞的不平等精神
   </a>
   ：“让我告诉你，那些非常富有的人，他们与你和我不同。“当涉及到穷人和有很多人之间的巨大差距时，当然可以说今天也是如此。
  </p>
  <p class="graf graf--p">
   为了充分掌握21世纪镀金时代的不平等性，重要的是要了解财富和收入之间的差异以及各种不平等产生的差异。简而言之，收入是指您在有偿工作方面赚取多少钱或任何投资或资产回报（或您拥有的其他有可能改变价值的东西）；财富只是这些资产的总积累以及对它们的任何回报或升值。你拥有的财富越多，年收入就越高。
  </p>
  <p class="graf graf--p">
   如果您每年赚取
   <a class="markup--anchor markup--p-anchor" data-href="https://fred.stlouisfed.org/series/MEPAINUSA672N" href="https://fred.stlouisfed.org/series/MEPAINUSA672N" rel="noopener noreferrer" target="_blank">
    31,000美元
   </a>
   ，即今天美国个人的中位数工资，那么您的收入将是税后的金额（也包括联邦，州，社会保障和医疗保险）平均而言，这意味着您将获得约
   <a class="markup--anchor markup--p-anchor" data-href="https://neuvoo.com/tax-calculator/California-31000" href="https://neuvoo.com/tax-calculator/California-31000" rel="noopener noreferrer" target="_blank">
    26,000美元
   </a>
   。
  </p>
  <p class="graf graf--p">
   然而，如果你的财富是1,000,000美元，并且你把它存入一个支付2.25％利息的储蓄账户，那么你可以
   <a class="markup--anchor markup--p-anchor" data-href="https://www.synchronybank.com/banking/high-yield-savings/" href="https://www.synchronybank.com/banking/high-yield-savings/" rel="noopener noreferrer" target="_blank">
    得到大约22,500美元
   </a>
   ，并且在税后，剩下大约19,000美元，你什么都不需要做。
  </p>
  <p class="graf graf--p">
   现在排名前1％的富人平均带回家的收入是占据社会90％的最低收入阶层40倍。“财富就是力量”，正如亚当·斯密在近几个世纪前的“国富论”中经常提到的那样。遗憾的是，这句格言似乎很少过时。
  </p>
  <p class="graf graf--p">
   显然，如果你在这个国家继承财富，你就会立即在整个游戏中处于领先地位。在美国，
   <strong class="markup--strong markup--p-strong">
    所有财富的三分之一到近一半
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.brookings.edu/blog/social-mobility-memos/2015/01/30/wealth-inheritance-and-social-mobility/" href="https://www.brookings.edu/blog/social-mobility-memos/2015/01/30/wealth-inheritance-and-social-mobility/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     都是遗产
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    而不是亲手创造的
   </strong>
   。据“纽约时报”调查，例如，唐纳德·特朗普从出生开始就从他亲爱的老爸那里获得了大约
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/interactive/2018/10/02/us/politics/donald-trump-tax-schemes-fred-trump.html" href="https://www.nytimes.com/interactive/2018/10/02/us/politics/donald-trump-tax-schemes-fred-trump.html" rel="noopener noreferrer" target="_blank">
    4.13亿美元
   </a>
   （以今天的价值计算）和另外
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnbc.com/2018/10/02/trumps-small-loan-from-his-father-was-more-like-60point7-million-nyt.html" href="https://www.cnbc.com/2018/10/02/trumps-small-loan-from-his-father-was-more-like-60point7-million-nyt.html" rel="noopener noreferrer" target="_blank">
    1.4亿美元
   </a>
   的贷款。对于一个“商人”来说，开始建立帝国的道路很平坦，这个帝国已经成为总统竞选活动的平台。特朗普做到了这一点，换句话说，是老式的方式 — 通过继承。
  </p>
  <p class="graf graf--p">
   美联储是国家的中央银行，负责决定货币成本，并在2007–2008年（以及之后）金融危机之后维持华尔街的支持，最终指出了这种极端的不平等程度对于全国其他地区的影响。正如美联储主席杰罗姆·鲍威尔2月初在华盛顿市政厅
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnbc.com/2019/02/07/feds-powell-reveals-biggest-economic-challenge-over-the-next-decade.html" href="https://www.cnbc.com/2019/02/07/feds-powell-reveals-biggest-economic-challenge-over-the-next-decade.html" rel="noopener noreferrer" target="_blank">
    所说
   </a>
   ，“我们希望繁荣得到广泛分享。我们需要制定政策来实现这一目标。“ 可悲的是，
   <strong class="markup--strong markup--p-strong">
    美联储在很大程度上有助于
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://mises.org/wire/feds-easy-money-policies-arent-helping-income-growth" href="https://mises.org/wire/feds-easy-money-policies-arent-helping-income-growth" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     增加而不是减少
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    一直在扼杀金融和政治体系中的系统性不平等
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这些都不应该令人感到意外，因为大部分超额资金直接流向大银行，富豪投资者和投机者。然后他们用它来投资股票和债券市场，但不是那些对财富之外的所有公民都很重要的事。
  </p>
  <p class="graf graf--p">
   问题是：
   <strong class="markup--strong markup--p-strong">
    为什么不平等和有缺陷的经济体系相互促进？
   </strong>
   作为一个起点，能够在美联储政策的推动下投资股市的人只会以指数方式增加其财富。当然，大多数人并没有投资于股票市场，也没有投资任何东西。他们负担不起。重要的是要记住，将近80％的人口靠薪水维生。
  </p>
  <p class="graf graf--p">
   最终结果是：财政危机急剧增加，排名前1％的富人不依赖薪水来增加他们的财富。飙升的股票市场不成比例地帮助这些富人。这也是为什么美联储对华尔街银行的补贴政策只会增加极少数人的极端财富。
  </p>
  <p class="graf graf--p">
   这种不平等造成的负面清单确实很长。首先是巨大的债务。最底层的90％是全国家庭债务约四分之三的拥有者。抵押贷款、汽车贷款、学生贷款和信用卡债务累计达到创纪录的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.reuters.com/article/us-usa-economy-debt/red-flags-emerge-as-americans-debt-load-hits-another-record-idUSKCN1Q11YV" href="https://www.reuters.com/article/us-usa-economy-debt/red-flags-emerge-as-americans-debt-load-hits-another-record-idUSKCN1Q11YV" rel="noopener noreferrer" target="_blank">
    13.5万亿美元
   </a>
   。
  </p>
  <p class="graf graf--p">
   而这只是滑坡的开始。正如Inequality.org报道的那样，财富和收入不平等影响“从预期寿命到婴儿死亡率和肥胖症”等一系列问题。例如，高度的经济不平等和健康状况不佳密切相关，或者说另一种方式，不平等会损害国家的整体健康状况。根据学术研究结果，
   <strong class="markup--strong markup--p-strong">
    收入不平等使人们生病，这句话就是字面上的意思。正如一项研究
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://news.vanderbilt.edu/2014/02/10/income-inequality/" href="https://news.vanderbilt.edu/2014/02/10/income-inequality/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     所说
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，“病态和贫困的经济基础设施导致了病态和贫困的身体或思想。
   </strong>
   ”
  </p>
  <p>
   <img alt="" class="aligncenter wp-image-1370 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-144.png?resize=800%2C796&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-144.png?w=1067&amp;ssl=1 1067w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-144.png?resize=150%2C150&amp;ssl=1 150w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-144.png?resize=300%2C300&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-144.png?resize=768%2C764&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-144.png?resize=1024%2C1019&amp;ssl=1 1024w" data-recalc-dims="1" height="796" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-144.png?resize=800%2C796&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   美国擅长铸造百万富翁。从全球来看，它的财富集中度最高，为41％（另外24％的百万富翁俱乐部可以在欧洲找到）顶层的1％的富人的收入是全国平均水平的40倍，拥有该国总财富的38.6％。任何其他发达国家的最高数字“仅”为28％。然而，
   <strong class="markup--strong markup--p-strong">
    这不是美国的特例，不平等已经是一种全球趋势，中国在此已经非常明显
   </strong>
   。
  </p>
  <p class="graf graf--p">
   亿万富翁俱乐部才是真正的俱乐部。据乐施会称，最富有的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.oxfam.org/en/pressroom/pressreleases/2018-01-22/richest-1-percent-bagged-82-percent-wealth-created-last-year" href="https://www.oxfam.org/en/pressroom/pressreleases/2018-01-22/richest-1-percent-bagged-82-percent-wealth-created-last-year" rel="noopener noreferrer" target="_blank">
    42位亿万富翁
   </a>
   的总财富与全球最贫穷的50％人口相当。但请放心，在这个镀金的世纪里，即使在亿万富翁群体内部也存在不平等。毕竟，其中最富有的10个人拥有全球总财富
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theweek.co.uk/people/57553/top-billionaires-who-richest-person-world" href="https://www.theweek.co.uk/people/57553/top-billionaires-who-richest-person-world" rel="noopener noreferrer" target="_blank">
    7450亿美元
   </a>
   。名单中的下一个10位只有
   <a class="markup--anchor markup--p-anchor" data-href="https://www.forbes.com/billionaires/#2a490e60251c" href="https://www.forbes.com/billionaires/#2a490e60251c" rel="noopener noreferrer" target="_blank">
    4515亿美元
   </a>
   ，就更不用说再下一个十位了。
  </p>
  <p class="graf graf--p">
   乐施会最近
   <a class="markup--anchor markup--p-anchor" data-href="https://www.oxfam.org/en/even-it/5-shocking-facts-about-extreme-global-inequality-and-how-even-it-davos" href="https://www.oxfam.org/en/even-it/5-shocking-facts-about-extreme-global-inequality-and-how-even-it-davos" rel="noopener noreferrer" target="_blank">
    还报道说
   </a>
   ，“亿万富翁的数量几乎翻了一番，2017年至2018年间每两天就有一位新的亿万富翁产生出来。他们现在拥有的财富比以往任何时候都要多，而几乎一半的人类仍没有摆脱极端贫困，生活在每天5.50美元的生存挣扎中。“
  </p>
  <p class="graf graf--p">
   富人只会越来越富裕，而且这种差异正在以历史性的速度发生。更糟糕的是，在过去的十年中，真正的富人们获得了额外的好处。他们可以扩大因金融危机而贬值的资产，而在这块巨大财富的另一边，他们的许多同行在2007–2008年的金融危机影响中
   <a class="markup--anchor markup--p-anchor" data-href="https://money.cnn.com/2017/12/01/news/economy/recession-anniversary/index.html" href="https://money.cnn.com/2017/12/01/news/economy/recession-anniversary/index.html" rel="noopener noreferrer" target="_blank">
    仍没能完全恢复
   </a>
   。
  </p>
  <p class="graf graf--p">
   从那以后我们看到的是，金钱如何通过银行和大规模的投机继续向上流动，而那些不在金融食物链顶端的人的经济生活基本上停滞不前或变得更糟。当然，结果是，在上个世纪的大部分时间里，这种不平等程度可能是不可思议的。
  </p>
  <p class="graf graf--p">
   最终，我们都将不得不面对整个经济中的黑云。现实世界中真实的人，不是最高层的人，经历了十年不稳定的变化，而这个超越镀金时代的不平等差距肯定会塑造一个真正凌乱的世界。换句话说，没有善终。
  </p>
  <p class="graf graf--p">
   作者 Nomi Prins 是前华尔街高管，她的最新著作是《合谋：中央银行家如何驾驭世界》
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    S
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://consortiumnews.com/2019/02/27/survival-of-the-richest/" href="https://consortiumnews.com/2019/02/27/survival-of-the-richest/" rel="noopener noreferrer" target="_blank">
    urvival of the Richest:
   </a>
   Those in power will remain in power unless acted upon by an external force. Those who are wealthy will only gain in wealth as long as nothing deflects them from their present course.
  </p>
