      </strong>

  </p>
  <p class="graf graf--p">
   监视资本主义的歪点子总是能突破你的脑洞。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “免费“的一切都必须你需要付出代价。PreShow，一家来自 MoviePass 前创始人的新公司，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnet.com/tags/advertising/" href="https://www.cnet.com/tags/advertising/" rel="noopener noreferrer" target="_blank">
    希望广告
   </a>
   成为电影票的唯一价格。
  </p>
  <p class="graf graf--p">
   他们在 Kickstarter 上发布了一个活动，有一个15到20分钟的高端广告，PreShow 正在开发一款应用程序，可以为任何影院中的任何电影赢取免费的电影票。PreShow 希望让电影感觉像是免费的。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    PreShow 取决于某些人可能会认真考虑的成本，而其他人则认为是讨价还价的成本：即 面部识别
   </strong>
   。“如果不是面部识别，我认为我们仍然不能做到这一点，”PreShow 的创始人兼首席执行官 Stacy Spikes 表示。
  </p>
  <p class="graf graf--p">
   放弃密码，PreShow 的应用程序
   <strong class="markup--strong markup--p-strong">
    只能
   </strong>
   通过手机的面部识别技术解锁。当您正在观看广告以获得免费门票时，手机的相机会监控您的注意力；如果你走开甚至只是遮住脸部的一部分，广告将在五秒后暂停，只有您回来时，它才会继续。
  </p>
  <p class="graf graf--p">
   面部识别已经在您的生活中发挥着越来越大的作用，无论是好的还是坏的。这项技术可以帮助您在Google相册中找到特定朋友的所有快照; 也是一些所谓的智能家居的核心。但它也可以在你完全不知情的情况下扫描你的脸。随着面部识别的推进和传播，它已经触发了许多隐私和安全问题。
  </p>
  <p class="graf graf--p">
   但是没有面部识别，PreShow 是不可能的。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们有两个问题需要解决：我们不希望人们创建虚拟账户，而且我们在一天结束时要处理真实的货币，因此我们需要唯一的确认，”Spikes 说。 “电话级别的面部识别目前为止也只有一年半的时间。两年前你还不能做到这点。”
  </p>
  <p class="graf graf--p">
   PreShow 使用面部识别进行识别和验证 — 它需要确保您是唯一可以打开应用程序的人（并且您只有一个帐户）更重要的是，它要确认您实际上正在观看这些广告。
  </p>
  <p class="graf graf--p">
   解锁机制建立在手机现有的面部识别功能之上。在 PreShow 的演示中，Spikes 通过他的 iPhone 7 的前置摄像头解锁了应用程序。
  </p>
  <p class="graf graf--p">
   确定观看部分更难。 PreShow 构建了自己的应用程序，以便在您观看广告时，视频边缘会出现绿色边框。只要前置摄像头能够识别出您的脸部正在看屏幕，边框就会保持绿色。如果你走开，将头部远离相机，甚至只是遮住脸部的一部分，演示中的轮廓就会变为红色。经过五秒后，商业广告自动暂停。
  </p>
  <p class="graf graf--p">
   与应用程序本身一样，PreShow 的隐私细节仍在不断变化。PreShow 应用程序的私人测试版将于本月开始，Kickstarter 的赞助人是第一个使用和分享它的人。该公司尚未具体说明何时向公众提供。
  </p>
  <p class="graf graf--p">
   PreShow 表示，测试将是关键，因为它将在测试期间根据用户反馈调整条款和条件。一旦 PreShow 开始让人们进入 beta 测试，它将发送更多关于整体用户体验的细节，包括面部识别技术，它说。
  </p>
  <p class="graf graf--p">
   PreShow 定义的具体内容是：该应用程序不会记录任何人的观看，也不会向第三方分享个人身份信息。
   <strong class="markup--strong markup--p-strong">
    对于其广告合作伙伴，它将提供汇总的身份数据
   </strong>
   。
  </p>
  <p class="graf graf--p">
   请注意，
   <strong class="markup--strong markup--p-strong">
    PreShow 计划要求您确定您的年龄、地理位置和性别，以便它可以提供广告客户想要的汇总人口统计详细信息。例如，PreShow 会使用面部识别来验证您是否诚实地将自己标识为男性或女性？
   </strong>
  </p>
  <p class="graf graf--p">
   Spikes 指出，在其他方面，这种广告对您的隐私更敏感。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “今天在媒体上发生的事是该品牌将购买大量数据，然后它将交易这些数据，它将实时跟随你，它将嵌入 cookie — 这一直是隐蔽的行动，现在只是更加公开了”，他说。
  </p>
  <p class="graf graf--p">
   Spikes 是2011年 MoviePass 的联合创始人，但后来该公司破产了。与 MoviePass 一样，Spikes 希望 PreShow 可以帮助将电影拉入新的业务方式。
  </p>
  <p class="graf graf--p">
   Spikes 表示，与 MoviePass 不同的是，单个会员的电影入场费用可能远远超过其订阅带来的收入。现在消费者可以通过观看广告来支付费用，无论公司的用户群是3000或3000万。
  </p>
  <p class="graf graf--p">
   Spikes 还提高了 PreShow 获得积分的可能性，除了电影票之外，还可以用购买网球鞋等产品。
  </p>
  <p class="graf graf--p">
   应用程序中的广告将是由电影中集成的品牌制作的视频 — 就像 Dodge 将其汽车放入《速度与激情》中。它还将包括涉及品牌和电影的幕后风格视频。
  </p>
  <p class="graf graf--p">
   现在的问题在于，
   <strong class="markup--strong markup--p-strong">
    你省下一点钱的代价并不只有被强迫的广告洗脑，而且还有面部识别 — — 只能由你来决定被强制剥夺隐私权只为一张电影票，究竟值不值得
   </strong>
   。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    M
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnet.com/news/moviepass-founder-wants-to-use-facial-recognition-to-score-you-free-movies/" href="https://www.cnet.com/news/moviepass-founder-wants-to-use-facial-recognition-to-score-you-free-movies/" rel="noopener noreferrer" target="_blank">
    oviePass founder wants to use facial recognition to score you free movies.
   </a>
   Facial recognition is the linchpin to PreShow, an app that will earn you free movie tickets by watching ads — so long as you’re OK with your phone watching you back.
  </p>
