  <div class="section-inner sectionLayout--insetColumn">
   <p class="graf graf--p graf-after--figure" id="1b19">
    信息战在当下是冷战以来的再一次高峰，在这一全球大背景下，参与其中的国家都有可能采取类似的策略 — — 伪装独立的声音宣传国家政府的意识。这次明显是俄罗斯走在了“前位”，当中国还在努力遏制独立媒体的成长时，克里姆林宫正专注于伪装那些“独立声音”掩饰其信息战的真面目。这可比那些以中南海权斗传声筒为荣的中文海外八卦网站的运作要高明多了。
   </p>
   <p class="graf graf--p graf-after--p" id="2807">
    那些八卦网站不被信任不过是因为他们太不专业，制作粗糙，而“被信任”不过是因为很多中文读者严重缺乏鉴别能力、无心也无力对事实进行追踪和验证，并且，中国人无法履行公民义务，于是只剩下看热闹的心态，也就养活了那些莫名其妙的隐性宣传工具。
   </p>
   <p class="graf graf--p graf-after--p" id="af7d">
    一旦这些八卦网站变得成熟，无疑将轻而易举促进北京当局的信息战之成果。本文将展示俄罗斯在宣传方面的专业性较高的战术，以及对其揭露的线索，希望能引起中文读者的警惕，否则，它就是全球华裔即将面对的未来，且极难被揭露。
   </p>
   <p class="graf graf--p graf-after--p" id="a28f">
    <strong class="markup--strong markup--p-strong">
     更加值得警惕的是，俄罗斯的宣传机构可以借用完全真实的信息为线索，通过渲染以达成特定目的的宣传效果
    </strong>
    ，就如他们曾经在关于 Tor 的资金来源的报道中所做的那样。据称中国的外宣一直在努力做到这点。这是一种极难被识别的隐性宣传。（详见
    <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/DonEvansWm/status/969530297569210368" href="https://twitter.com/DonEvansWm/status/969530297569210368" rel="noopener nofollow noreferrer" target="_blank">
     这个 thread
    </a>
    ）
   </p>
   <p class="graf graf--p graf-after--p" id="8ec4">
    请结合我们推荐的 #OSINT 开源调查方法，发动更多人一起来挖掘。
   </p>
   <p class="graf graf--p graf-after--p" id="523f">
    <a class="markup--anchor markup--p-anchor" data-href="https://www.buzzfeednews.com/article/holgerroonemaa/russia-propaganda-baltics-baltnews" href="https://www.buzzfeednews.com/article/holgerroonemaa/russia-propaganda-baltics-baltnews" rel="noopener nofollow noreferrer" target="_blank">
     <strong class="markup--strong markup--p-strong">
      This Is How Russian Propaganda Actually Works In The 21st Century
     </strong>
    </a>
   </p>
   <p class="graf graf--p graf-after--p" id="c952">
    Skype logs and other documents obtained by BuzzFeed News offer a rare glimpse into the inner workings of the Kremlin’s propaganda machine.
   </p>
   <p class="graf graf--p graf-after--p" id="def7">
    事实证明。俄罗斯政府谨慎地资助了东欧一组看似”独立”的新闻网站，以隐蔽性输出克里姆林宫向他们提供的故事。
   </p>
   <p class="graf graf--p graf-after--p" id="89a0">
    俄罗斯国家媒体创建了秘密公司，以便为波罗的海国家的网站提供资金 — 这是俄罗斯与西方之间以及东欧和中亚其他地区的重要战场。
   </p>
   <p class="graf graf--p graf-after--p" id="e3f5">
    该秘密计划被披露，是通过 Skype 聊天记录、爱沙尼亚报纸 Postimees、以及调查性新闻出版商 Re：Baltica 通过信息自由法获得的文件，在爱沙尼亚作为对莫斯科当地个人进行刑事调查的一部分。
   </p>
   <p class="graf graf--p graf-after--p" id="061b">
    <strong class="markup--strong markup--p-strong">
     从调查人员查获的计算机中获取的 Skype 日志和其他文件揭示了俄罗斯在试图影响公众舆论并推动克里姆林宫谈话要点时使用的秘密和混淆策略。
    </strong>
   </p>
   <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote graf-after--p" id="2b92">
    <p>
     <em class="markup--em markup--blockquote-em">
      “In other words — information warfare.”
     </em>
    </p>
   </blockquote>
   <p class="graf graf--p graf-after--blockquote" id="54df">
    这些网站本身看起来是“独立”的新闻媒体，但事实上，编辑界线直接由莫斯科决定。
   </p>
   <p class="graf graf--p graf-after--p" id="ddb0">
    爱沙尼亚领先的战略传播专家 Raul Rebane 说，这个计划和其他类似计划是“在外国领土上开展的系统性信息相关活动。换句话说就是 — — 信息战。“
   </p>
   <p class="graf graf--p graf-after--p" id="c846">
    他说，波罗的海的俄罗斯宣传网络已经运营多年，并且最近变得更加激烈了。
   </p>
   <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="485a">
    “将[爱沙尼亚]从面对西方转向面向东方的压力已经增大。”
   </p>
   <p class="graf graf--p graf-after--p" id="e24b">
    莫斯科一直试图影响波罗的海国家如立陶宛、拉脱维亚和爱沙尼亚的公众舆论，这些都是北约的成员。关于波罗的海国家网站的揭露提供了一个罕见而详细的内部视角，了解这种虚假宣传活动的运作方式，以及莫斯科希望掩盖其参与此类计划的时间长度。
   </p>
   <p class="graf graf--p graf-after--p" id="29fd">
    在波罗的海地区，俄罗斯与欧盟直接接壤，北约拥有庞大的军事存在，但也许最重要的是，该地区有成千上万的俄罗斯人，主要是在爱沙尼亚和拉脱维亚。
   </p>
   <p class="graf graf--p graf-after--p" id="a4c8">
    当俄罗斯总统弗拉基米尔·普京于 2014 年从乌克兰吞并克里米亚半岛时，他是以保护“俄语发言人”的利益为借口的。所以在波罗的海国家，俄罗斯的宣传是一种真正的现实的恐惧，并解释了为什么影响公众舆论的莫斯科应该受到如此严肃的对待。
   </p>
   <p class="graf graf--p graf-after--p" id="fce7">
    调查人员获得的记录和 Skype 日志提供了一个揭露的机会，拉开了克里姆林宫的宣传机器内部工作的窗帘，这次针对的是波罗的海的俄语传声筒。
   </p>

    <p class="graf graf--p graf-after--figure" id="206d">
     2014 年10月初，亚历山大·科尔尼洛夫（Aleksandr Kornilov） — 爱沙尼亚俄罗斯同胞协调委员会成员，该组织以其俄语首字母缩写 KSORS 而闻名 — 似乎正在处理少数民族权利，如俄语教育问题，但被爱沙尼亚反间谍机构视为克里姆林宫外交政策的工具 — Kornilov 接受了立陶宛一家名为
     <a class="markup--anchor markup--p-anchor" data-href="http://delfi.lt/" href="http://delfi.lt/" rel="noreferrer noopener nofollow" target="_blank">
      Delfi.lt
     </a>
     的新闻网站的采访，关于该组织在波罗的海地区推出三个新网站的话题，全称为 Baltnews。
    </p>
    <p class="graf graf--p graf-after--p" id="6342">
     Kornilov 告诉记者说，在立陶宛、爱沙尼亚和拉脱维亚分别推出的这些网站将成为独立的门户网站，“避开政治”。
    </p>
    <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="abda">
     “门户网站将是娱乐性的; 你不会成为竞争对手，“他告诉记者。“概念是这样的 — 一个普通的、有趣的、非政治性的门户网站。”
    </p>
    <p class="graf graf--p graf-after--p" id="c97d">
     但现实却截然不同。事实上，这些网站是莫斯科资助和指导的，是俄罗斯最大的国家媒体机构的一部分。
    </p>
    <p class="graf graf--p graf-after--p" id="458a">
     在 BuzzFeed News 及其报告合作伙伴获得的 Skype 聊天记录中，Kornilov 由 Rossiya Segodnya 的一名员工订购报道的选题，后者负责运营网站和新闻机构 Sputnik、以及新闻机构 RIA Novosti，并与RT（以前称为今日俄罗斯，紧密相关） — 严格遵守经批准的主题清单。
    </p>
    <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote graf-after--p" id="1cb7">
     <p>
      <em class="markup--em markup--blockquote-em">
       “I have a task. Every day you need to report on three of the five topics that we will suggest.”
      </em>
     </p>
    </blockquote>
    <p class="graf graf--p graf--startsWithDoubleQuote graf-after--blockquote" id="2c3b">
     “San，请回复，”这是于 2015 年6月发出的消息，这是 Aleksandr Svyazin 名字的缩短版。
    </p>
    <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="489c">
     “Aleksandr, 快出来！”他后来说，越来越不耐烦了。
    </p>
    <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="d73b">
     “在了”，Kornilov 终于回应道。
    </p>
    <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="74af">
     “San，你好！我有一个任务，“Svyazin 说。“每天你需要报告我们建议的五个主题中的三个。”
    </p>
    <p class="graf graf--p graf-after--p" id="d64e">
     在收到第一个主题列表后，Kornilov 用俄语单词回答“понятно” — 意思是“理解”。
    </p>
    <p class="graf graf--p graf-after--p" id="ddb5">
     Skype 日志表明，Svyazin 是整个计划中 Kornilov 的主要联系人，因为他们在两年期间互相发送了数千条消息，并定期就网站的编辑问题进行交谈。
    </p>
    <p class="graf graf--p graf-after--p" id="7db4">
     当 BuzzFeed News 在 Facebook 上联系他时，Svyazin 否认与 Sputnik 或伪独立媒体 Baltnews 存在联系。Svyazin 在被展示 Skype 聊天记录的屏幕截图时没有回复 — 截图中他的 Skype 头像清晰可见 — 显示了他和 Kornilov 之间的交谈。然而，在随后向 BuzzFeed News 记者发布的 Facebook 消息中，Svyazin 将所有疑问都提交给了 Rossiya Segodnya 的新闻办公室。
    </p>
    <p class="graf graf--p graf-after--p" id="c14e">
     Kornilov 和 Svyazin 之间的这种交流是 Skype 日志揭示的众多信息之一，表明网站并非如表面看起来那样是独立的，正如文件所示，
     <strong class="markup--strong markup--p-strong">
      这些网站是 Rossiya Segodnya 策划、资助和管理的宣传活动的一部分，它在欧洲设立了作为莫斯科大喇叭的出口
     </strong>
     。
    </p>
    <p class="graf graf--p graf-after--p" id="23da">
     Baltnews 网站于 2014 年 10 月推出后不久 — 在拉脱维亚、爱沙尼亚和立陶宛创办了三个几乎相同的网站 — — Rossiya Segodnya 员工 Svyazin，一位
     <a class="markup--anchor markup--p-anchor" data-href="https://ru.sputniknews-uz.com/tourism/20170808/5994286/zhurnalist-rasskazal-poyezdka-afganistan-Svyazuin.html" href="https://ru.sputniknews-uz.com/tourism/20170808/5994286/zhurnalist-rasskazal-poyezdka-afganistan-Svyazuin.html" rel="noopener nofollow noreferrer" target="_blank">
      在东欧和中亚
     </a>
     各个

    </p>
    <p class="graf graf--p graf-after--p" id="6c6a">

     </strong>

    </p>
    <p class="graf graf--p graf-after--p" id="8934">

     有关该计划如何在爱沙尼亚运营和资助的文件和 Skype 聊天仅与该国有关，但在拉脱维亚和立陶宛创办的 Baltnews 版本似乎也以类似的方式运作 — 所有三个网站看起来都相同，它们被同时设置，在同一天发表很近似的报道。这三个国家的本地安全服务都认为这些网站属于同一项业务。
    </p>
    <p class="graf graf--p graf-after--p" id="072b">
     此外，根据 BuzzFeed News 获得的文件，拉脱维亚 Baltnews 分公司确实向 Barsolina Ventures 提交了至少一份月度报告。
    </p>
    <p class="graf graf--p graf-after--p" id="b6af">
     <a class="markup--anchor markup--p-anchor" data-href="http://baltnews.lv/" href="http://baltnews.lv/" rel="noreferrer noopener nofollow" target="_blank">
      Baltnews.lv
     </a>
     的财务报告显示它的年营业额为 10 万欧元，但几乎没有广告，也没有付费订阅。
    </p>
    <p class="graf graf--p graf-after--p" id="f330">
     该网站的主编 Andrejs Jakovlevs 拒绝置评，称财务状况不是编辑人员可以讨论的问题。他说与 RIA Novosti 有一个信息共享安排，但没有交换任何金钱，也没有人告诉他应该写什么。
    </p>
    <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote graf-after--p" id="e7c2">
     <p>
      <em class="markup--em markup--blockquote-em">
       “Their goal is to extend Russia’s influence within Lithuania’s information space, promote anti-western sentiments, and shape public opinion favourable to Kremlin.”
      </em>
     </p>
    </blockquote>
    <p class="graf graf--p graf-after--blockquote" id="1d21">
     但拉脱维亚的内部反间谍和安全服务（DP）称，根据其信息，
     <a class="markup--anchor markup--p-anchor" data-href="http://baltnews.lv/" href="http://baltnews.lv/" rel="noreferrer noopener nofollow" target="_blank">
      Baltnews.lv
     </a>
     的资金来自俄罗斯，使用的是第三方国家的公司。
    </p>
    <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="9832">
     “[DP] 获得的信息表明，从俄罗斯获得的资金是
     <a class="markup--anchor markup--p-anchor" data-href="http://baltnews.lv/" href="http://baltnews.lv/" rel="noreferrer noopener nofollow" target="_blank">
      Baltnews.lv
     </a>
     的主要收入来源，”DP 副主任 ĒriksCinkus 告诉 BuzzFeed News。
    </p>
    <p class="graf graf--p graf-after--p" id="7eaf">
     与此同时，在立陶宛，该国的情报机构 VSD 将
     <a class="markup--anchor markup--p-anchor" data-href="http://baltnews.lt/" href="http://baltnews.lt/" rel="noreferrer noopener nofollow" target="_blank">
      Baltnews.lt
     </a>
     和 Sputnik 确定为俄罗斯在该国传播亲克里姆林宫信息的渠道。
    </p>
    <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="1362">
     “他们的目标是扩大俄罗斯在立陶宛信息领域的影响力，促进反西方情绪，塑造对克里姆林宫有利的公众舆论，”VSD 在其最新的年度报告中写道。
    </p>
    <p class="graf graf--p graf-after--p" id="510d">
     <a class="markup--anchor markup--p-anchor" data-href="https://www.vsd.lt/wp-content/uploads/2018/03/ENG.pdf" href="https://www.vsd.lt/wp-content/uploads/2018/03/ENG.pdf" rel="noopener nofollow noreferrer" target="_blank">
      该报告称
     </a>
     ，2017 年，根据俄罗斯的命令，Baltnews 和 Sputnik 开始发表更多关于波兰社区在立陶宛的地位的文章。
    </p>
    <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="65c1">
     “他们试图煽动民族对抗，恶化立陶宛与波兰之间的关系。该报告称，这些出版物试图说服读者立陶宛歧视波兰当地社区，或者给人的印象是波兰不会放弃“对其邻国的领土要求”。
    </p>
    <p class="graf graf--p graf-after--p" id="d9b0">
     直到今年6月1日，
     <a class="markup--anchor markup--p-anchor" data-href="http://baltnews.lt/" href="http://baltnews.lt/" rel="noreferrer noopener nofollow" target="_blank">
      Baltnews.lt
     </a>
     的主编是 Anatoly Ivanov，他所经营的非营利组织 — 名为欧亚媒体实验室的财务数据显示，他正在从 Barsolina Ventures 获得大量资金。
    </p>
    <p class="graf graf--p graf-after--p" id="4611">
     根据立陶宛商业登记处的报告，欧亚媒体实验室在三年内收到约 50 万欧元。2015 年和 2016年，Barsolina 每年向欧亚媒体实验室支付 163,200 欧元。2017 年，欧亚媒体实验室报告了 185,472 欧元的收入，其中绝大部分来自 Barsolina 的“互联网网关服务”。当 BuzzFeed News 通过电话联系时，Ivanov 没有回答有关付款的任何问题。
    </p>
    <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="b999">
     “年轻人，你可能没有理解我的意思。我很忙 — 我办公室里有其他人，你可以问任何你想要的东西。如果你听不懂俄语，我可以用立陶宛语解释，“ Ivanov 说，从一种语言转换到另一种语言。“有人坐在我的办公室里。而且我不想浪费时间。“
    </p>
    <p class="graf graf--p graf-after--p" id="77a5">
     当再次询问有关50万欧元的付款时，他在挂断电话之前叹了口气，没有给出任何进一步的回复。
    </p>
    <p class="graf graf--p graf-after--p" id="3e16">
     除了提供资金外，Rossiya Segodnya 似乎还直接协调了所有三个 Baltnews 网站应涵盖的主题。Baltnews 不仅与所谓的“远离独立娱乐网站”声称一点不沾边，而且似乎还是一个更大机器中的一个齿轮。
    </p>
    <p class="graf graf--p graf-after--p" id="869a">
     2016 年5月25日，Svyazin — 可能是误操作 — 向 Kornilov 发送了一系列主题，
     <strong class="markup--strong markup--p-strong">
      这些主题似乎显示了前苏联克里姆林宫所有媒体的预期覆盖面
     </strong>
     ：波罗的海国家，吉尔吉斯斯坦，白俄罗斯，格鲁吉亚，摩尔多瓦，乌兹别克斯坦，哈萨克斯坦，阿塞拜疆和俄罗斯支持的格鲁吉亚奥塞梯分离地区。爱沙尼亚在名单中有两个单独的条目 — 一个用于 Baltnews，另一个用于本地版的 Sputnik。
    </p>
    <p class="graf graf--p graf-after--p" id="b505">
     BuzzFeed News 将主题列表与当天在这些国家发布的报道进行了比较，发现除了两个本地版本的 Sputnik 之外，其他所有版本都确实发布了规定的内容。根据指示，吉尔吉斯斯坦的斯普特尼克斯写了一篇关于当地商业大亨 Sharshenbek Abdykerimov 秘密拥有塔楼公寓的消息，当时是伦敦最高的住宅摩天大楼。
    </p>
    <p class="graf graf--p graf-after--p" id="cc2f">
     正如你看到的那样，Georgian Sputnik 写了关于
     <a class="markup--anchor markup--p-anchor" data-href="https://sputnik-georgia.ru/society/20160525/231839473.html" href="https://sputnik-georgia.ru/society/20160525/231839473.html" rel="noopener nofollow noreferrer" target="_blank">
      该国女性政治家
     </a>
     着装感的文章，包括大量的例子和照片。在摩尔多瓦，当地的 Sputnik 网站被命令涵盖本国货币对俄罗斯卢布和石油价格的依赖性。它通过发布一个标题为“对我们 Leu 的诊断：打破另一个依赖关系“的
     <a class="markup--anchor markup--p-anchor" data-href="https://ru.sputnik.md/economics/20160526/6874883.html" href="https://ru.sputnik.md/economics/20160526/6874883.html" rel="noopener nofollow noreferrer" target="_blank">
      新闻报道
     </a>
     来完成任务。
    </p>
    <p class="graf graf--p graf-after--p" id="3ccd">
     有时，在俄罗斯的其他人会指导报道。2015 年7月20日，Rossiya Segodnya 的多媒体编辑要求关于东乌克兰分离主义者的部队和武器流动的新闻报道。在接下来的几个月中，这些网站便定期对强制性主题进行类似的报道。有些订单也来自一位女士，她在 Skype 上以 Kiana Lil Minasyan 的身份向 Kornilov 介绍自己，并说她是“计划与波罗的海国家打交道的多媒体计划”的编辑。
     <strong class="markup--strong markup--p-strong">
      另一位参与指导爱沙尼亚报道的人似乎也曾前往爱沙尼亚招募当地的 Sputnik 记者
     </strong>
     （中国的外宣也在被宣传国家招募当地记者）
    </p>
    <p class="graf graf--p graf-after--p" id="e7a2">
     爱沙尼亚反间谍机构 KAPO 在 2015 年将 Minasyan 确定为爱沙尼亚 Sputnik 办事处的主管。“自 2015 年春季以来，Rossiya Segodnya 一直致力于在爱沙尼亚建立办事处，并为 Sputnik 和俄语语言门户提供服务，”KAPO 在 2015 年透露回顾并补充说，“莫斯科的 Liana Minasyan”是监督办公室的一员。去年 Minasyan 向 Re：Baltica 确认她负责波罗的海地区的所有 Sputnik 网站，但否认参与了隐性宣传机器 Baltnews 的计划。
    </p>
    <p class="graf graf--p graf-after--p" id="c0be">
     当 BuzzFeed News 再次追问时，Minasyan 说她不再与 Baltnews 或波罗的海国家合作了。“如果你愿意，我会把你的问题转发给我的前同事，”她说，并没有回复后续问题。
    </p>
    <p class="graf graf--p graf-after--p" id="ec57">
     记录显示 Kornilov 和 Svyazin 的每日 Skype 会议持续了数月，两名男子有时会争论 Kornilov 被要求提交的每周和每月的流量报告。
    </p>
    <p class="graf graf--p graf-after--p" id="c3c9">
     在一个案例中，Kornilov 说合同草案中有一个错误 — 说的不是“爱沙尼亚语”，而是“拉脱维亚语”，这是另一个迹象，表明在每个波罗的海国家都签署了类似的宣传运作合同。
    </p>
    <p class="graf graf--p graf-after--p" id="b1db">
     <strong class="markup--strong markup--p-strong">
      他们的谈话有时会超出工作范围 — — 有时他们抱怨他们的老板和他们参与的官僚机构。记录还显示他们在俄罗斯见过几次面。
     </strong>
    </p>
    <p class="graf graf--p graf-after--p" id="ace6">
     从一开始，Kornilov 就试图编织一个独立新闻网站的幻想。当立陶宛
     <a class="markup--anchor markup--p-anchor" data-href="http://delfi.lt/" href="http://delfi.lt/" rel="noreferrer noopener nofollow" target="_blank">
      Delfi.lt
     </a>
     报道了宣传机器 Baltnews 的发布时，Kornilov 将自己标示为这三个网站的出版商。波罗的海三个国家的所有域名当时都以他的名字注册，但很快就重新注册到了 Media Capital Holding BV — 就是那个由 Rossiya Segodnya 代理人在荷兰注册的公司。
    </p>
    <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote graf-after--p" id="0690">
     <p>
      <em class="markup--em markup--blockquote-em">
       “The terms are confidential. I am the editor-in-chief and communicate directly with their representative in Estonia. Right?”
      </em>
     </p>
    </blockquote>
    <p class="graf graf--p graf-after--blockquote" id="fcbd">
     在 Delfi 发布新闻报道后，Kornilov 立即就与爱沙尼亚的亲俄罗斯记者 Jevgeni Levik 进行了对话，他在 Baltnews 担任第一任总编辑，但在发布前不久被 Kornilov 解雇。
    </p>
    <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="7239">
     “我们的官方立场是这样的：我们正在与外国投资者一起开发一个互动信息项目，”根据信息自由法获得的 Skype 聊天中，Levik 在解雇之前是这样说的。
    </p>
    <p class="graf graf--p graf-after--p" id="2919">
     <strong class="markup--strong markup--p-strong">
      如果记者询问网站的投资者的话，他们还讨论了如何应对。
     </strong>
     “也许我们应该给他们公司的名字，我相信很多人不会拒绝了解它。还可以包括所有联系人？再准备一份商业计划，“ Kornilov 回答说，看似有点讽刺。
    </p>
    <p class="graf graf--p graf-after--p" id="5d7c">
     Levik 说，“如果他们问荷兰人是谁，我会回答他们是私人的，这是一个私人商业项目。这些条款是保密的。我是主编，直接与他们在爱沙尼亚的代表沟通。对吧？”
    </p>
    <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="c334">
     “是。你应该只处理你职权范围内的事。确切地说，那里的人不是私人的，它是一个投资基金会，“ Kornilov 说。
    </p>
    <p class="graf graf--p graf-after--p" id="c26d">
     当 BuzzFeed News 通过电话联系时，Levik 拒绝发表评论。“这都有四年的历史了。我只是一名记者，“他说。
    </p>

     <a class="markup--anchor markup--p-anchor" data-href="http://baltija.eu/" href="http://baltija.eu/" rel="noreferrer noopener nofollow" target="_blank">

     上进行练习。
    </p>
    <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="63c9">
     “让我们从两个不同的人的 50 条评论开始。如果一切顺利，我们可以继续在其他网站和更多的帖子上合作，“Dorofeyev 指示一个名为 Artem 的人说，此人提供巨魔工厂服务。
    </p>
    <p class="graf graf--p graf-after--p" id="78d6">
     Skype 聊天记录显示，Dorofeyev 对巨魔工厂是否可以使用不同的 IP 进行虚假评论这事比较感兴趣。该服务提供商问为什么。“
     <strong class="markup--strong markup--p-strong">
      我们需要独特的 IP，以便其他人没理由怀疑这其中有什么事。
     </strong>
     总的来说，我们需要这样一群人，“Dorofeyev 回答道。
    </p>
    <p class="graf graf--p graf-after--p" id="6a8a">
     Artem 的姓氏不为人所知，该公司的政策不允许其参与在其他网站上发表的评论。“我们拥有版权，”他说，并指导 Dorofeyev 遵守
     <a class="markup--anchor markup--p-anchor" data-href="http://smo-i-seo.ru/posting-kommentirovanie-blogov" href="http://smo-i-seo.ru/posting-kommentirovanie-blogov" rel="noopener nofollow noreferrer" target="_blank">
      条款和条件
     </a>
     。但他们仍同意继续进行
     <a class="markup--anchor markup--p-anchor" data-href="http://baltija.eu/" href="http://baltija.eu/" rel="noreferrer noopener nofollow" target="_blank">
      Baltija.eu
     </a>
     的测试。
    </p>
    <p class="graf graf--p graf-after--p" id="8bd8">
     当 BuzzFeed News 接洽时，Dorofeyev 拒绝回复关于购买网页浏览量和评论的提问。他说他只在爱沙尼亚的 Baltnews 担任日报编辑：“我写了并发表了我认为重要且有趣的新闻。”他还表示拒绝任何人告诉他应该写些什么的想法。
    </p>
    <p class="graf graf--p graf-after--p" id="ac6f">
     当被告知 Skype 聊天日志显示他参与人为增加网站流量时，Dorofeyev 努力否定这点，还打了个比方，说他虽然可以获得登月的票，但“不能证明我真的登上了月球。”他拒绝分享他的电子邮件地址，记者询问电邮是为了把他登录 Skype 的日志发送给他。
    </p>
    <p class="graf graf--p graf-after--p" id="1d9c">
     爱沙尼亚 KAPO 机构的负责人兼发言人 Harrys Puusepp 告诉 BuzzFeed 新闻，Baltnews 计划符合其他克里姆林宫宣传活动的运作方式，并展示了他们的秘密金融计划如何被用来使敌对宣传合法化。
    </p>
    <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="d1a4">
     “
     <strong class="markup--strong markup--p-strong">
      这是一个生产和融资政治宣传的恶性循环，它被伪装成当地俄罗斯人的真实声音，以便利用扭曲的公众形象来支持他们在国际平台上的外交政策目标
     </strong>
     ，” Puusepp 说。
    </p>
    <p>
     <img alt="" class="aligncenter size-full wp-image-2687 jetpack-lazy-image" data-lazy-sizes="(max-width: 672px) 100vw, 672px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/09/0-26.png?resize=672%2C420&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/09/0-26.png?w=672&amp;ssl=1 672w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/09/0-26.png?resize=300%2C188&amp;ssl=1 300w" data-recalc-dims="1" height="420" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/09/0-26.png?resize=672%2C420&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="672"/>
     <noscript>
      <img alt="" class="aligncenter size-full wp-image-2687" data-recalc-dims="1" height="420" sizes="(max-width: 672px) 100vw, 672px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/09/0-26.png?resize=672%2C420&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/09/0-26.png?w=672&amp;ssl=1 672w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/09/0-26.png?resize=300%2C188&amp;ssl=1 300w" width="672"/>
     </noscript>
    </p>
    <p class="graf graf--p graf-after--figure" id="b7c1">
     BuzzFeed News 获得的文件和 Skype 日志一直持续到 2016 年夏天 — — 爱沙尼亚警方在伪造文件的刑事调查中扣押了 Kornilov 及其同伙的计算机。公开数据显示，Baltnews 宣传机器的融资计划持续超过该日期。2016 年，在与爱沙尼亚税务机关和警方的调查陷入僵局后，Kornilov 改变了
     <strong class="markup--strong markup--p-strong">
      他用来经营 Baltnews 的非政府组织。他开始使用一个名为 Baltnewsmedia 的新 NGO，而不再是 Altmedia
     </strong>
     。
    </p>
    <p class="graf graf--p graf-after--p" id="fa17">
     根据 Baltnewsmedia 的财务记录，它在 2016 年收到了 136,800 欧元的捐款。审查没有具体说明捐款的来源，但它们每月恰好都达到了 11,400 欧元 — 即 Barsolina Ventures 过去发送给前“NGO” Altmedia 的数目总和。2017 年，捐款达到 155,508 欧元。
    </p>
    <p class="graf graf--p graf-after--p" id="e69f">
     今年 6 月1日，所有三个 Baltnews 网站都发布了同一条
     <a class="markup--anchor markup--p-anchor" data-href="http://baltnews.ee/tallinn_news/20180601/1016693677.html" href="http://baltnews.ee/tallinn_news/20180601/1016693677.html" rel="noopener nofollow noreferrer" target="_blank">
      简短的信息
     </a>
     ，称由于出版商的变更，将组建一个新的编辑团队。同一天，Kornilov 在 Facebook 上发布消息说他的团队已经结束了与 Baltnews 的联系。“感谢所有与我们在一起工作四年的人，”他写道。
    </p>
    <p class="graf graf--p graf-after--p" id="dc51">
     四天后，他与爱沙尼亚检察官办公室达成了认罪协议。虽然 Kornilov 设法避免了对税务欺诈和伪造文件的定罪，但他在出版 Baltnews 时创立和使用的非政府组织 Altmedia 被定罪了 — — 允许 Kornilov 个人保持一份干净的记录，尽管法庭判决明确表明他在安排和执行逃税计划和伪造证件方面所起的作用。
    </p>
    <p class="graf graf--p graf-after--p" id="8d3f">
     拉脱维亚 Baltnews 网站，当通过其一般电子邮件地址联系以征求意见时，他们将所有问题提交给 Rossiya Segodnya 的新闻办公室。Buzzfeednews 说，发送到那里的电子邮件仍未得到答复。
    </p>
    <p class="graf graf--p graf-after--p" id="484b">
     目前，Baltnews 仍继续在波罗的海三个国家出版。
    </p>
