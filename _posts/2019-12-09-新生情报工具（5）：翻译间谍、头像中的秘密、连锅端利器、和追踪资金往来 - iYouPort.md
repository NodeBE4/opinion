   Seeker 背后的概念很简单，就像托管网页仿冒页面获取凭据那样，请求目标人的位置。
  </p>
  <p class="graf graf--p">
   Seeker 使用 Serveo 生成链接，将它转发给目标；网站要求获得位置许可，如果目标允许，
   <strong class="markup--strong markup--p-strong">
    追踪者就可以获得
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    经度
   </li>
   <li class="graf graf--li">
    纬度
   </li>
   <li class="graf graf--li">
    准确度
   </li>
   <li class="graf graf--li">
    海拔（并非总是可用）
   </li>
   <li class="graf graf--li">
    方向（仅在用户移动时可用）
   </li>
   <li class="graf graf--li">
    速度（仅在用户移动时可用）
   </li>
  </ul>
  <p class="graf graf--p">
   除了位置信息，追踪者

   </strong>

  </p>

   <li class="graf graf--li">

   <strong class="markup--strong markup--p-strong">
    追踪资金往来
   </strong>
  </p>
  <p class="graf graf--p">
   Venmo 是 PayPal 旗下的一个移动支付服务，让用户可以使用手机或网页转账给他人。
  </p>
  <p class="graf graf--p">
   截至2018年6月，Venmo 已经拥有约2300万用户，仅2018年第二季度的交易总额就超过140亿美元。
  </p>
  <p class="graf graf--p">
   ⚠️不幸的是，Venmo 非常不安全，因为它将所有交易公开给任何人可以通过未经身份验证的API实时查看。
  </p>
  <p class="graf graf--p">
   而且默认情况下，用户个人资料也是公开的，这意味着任何人都可以在未登录应用的情况下查看用户的最近五笔交易。
  </p>
  <p class="graf graf--p">
   可以轻松找到个人的电话号码，并可用于在开源情报调查中发现 Venmo 帐户信息。
  </p>
  <p class="graf graf--p">

    <img alt="" class="aligncenter size-full wp-image-8865" data-recalc-dims="1" height="183" sizes="(max-width: 931px) 100vw, 931px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/10-1.jpeg?resize=931%2C183&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/10-1.jpeg?w=931&amp;ssl=1 931w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/10-1.jpeg?resize=300%2C59&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/10-1.jpeg?resize=768%2C151&amp;ssl=1 768w" width="931"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   ⚠️切勿随意使用在线翻译！这非常重要。即便无法公开索引，翻译工具的提供者也能看到您输入的内容 —— 想象一下，如果敏感人士使用
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%9a%90%e5%86%99%e6%9c%af%e7%9a%84%e5%b0%8f%e5%bf%83%e6%9c%ba/" href="https://www.iyouport.org/%e9%9a%90%e5%86%99%e6%9c%af%e7%9a%84%e5%b0%8f%e5%bf%83%e6%9c%ba/" rel="noopener noreferrer" target="_blank">
    隐写术
   </a>
   或者任何加密通信传递内容，而其中使用的信息并非母语；这种情况下如果接收者使用了翻译功能 …… 任何安全技术都保护不了你。
  </p>
  <p class="graf graf--p">
   这里要展示的最后一个危险例子并不是来自电子邮的件，而是来自网站。
  </p>
  <p class="graf graf--p">
   有人在网站注册了一个账户，网站自动显示已经发送了确认代码到用户的手机；而在这种情况下用户使用了翻译功能 —— 完整的电话号码就暴露了。
  </p>
  <p class="graf graf--p">
   在这里看到电话号码有多危险《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e5%8f%aa%e7%9f%a5%e9%81%93%e4%b8%80%e4%b8%aa%e7%94%b5%e8%af%9d%e5%8f%b7%e7%a0%81%ef%bc%8c%e4%bd%a0%e8%83%bd%e6%8c%96%e5%87%ba%e5%a4%9a%e5%b0%91%e6%9c%89%e6%95%88%e4%bf%a1%e6%81%af/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e5%8f%aa%e7%9f%a5%e9%81%93%e4%b8%80%e4%b8%aa%e7%94%b5%e8%af%9d%e5%8f%b7%e7%a0%81%ef%bc%8c%e4%bd%a0%e8%83%bd%e6%8c%96%e5%87%ba%e5%a4%9a%e5%b0%91%e6%9c%89%e6%95%88%e4%bf%a1%e6%81%af/" rel="noopener noreferrer" target="_blank">
    如果只知道一个电话号码，你能挖出多少有效信息？
   </a>
   》。
  </p>
  <p class="graf graf--p">
   请注意以上只是最简单的几个例子而已，在此索引中您能挖到很多神奇的东西，什么银行支付交易ID、PayPal交易等等，五花八门。
  </p>
  <p class="graf graf--p">
   永远都会有新服务出现，人们往往很少会花时间先学习一下如何正确使用它们、以及它们可能揭示的关于你的数据都包括什么。
  </p>
  <p class="graf graf--p">
   这种习惯本身非常危险！
  </p>
  <p class="graf graf--p">
   ⚠️提醒所有人：小心上传任何内容，不论使用任何数字工具都应该非常谨慎。
  </p>
  <p class="graf graf--p">
   如果您对此感兴趣，请关注本网下一篇文章：关于
   <a href="https://www.iyouport.org/%e4%b8%80%e7%9f%a5%e5%8d%8a%e8%a7%a3%e7%9a%84%e5%ae%89%e5%85%a8%e6%84%8f%e8%af%86%e5%a6%82%e4%bd%95%e5%af%bc%e8%87%b4%e4%ba%86%e6%9b%b4%e5%a4%9a%e7%9a%84%e5%8d%b1%e9%99%a9%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    在线沙箱服务（恶意软件分析）如何变成了间谍
   </a>
   的故事。小心！⚪️
  </p>
