   <li>
    <span style="color: #00ccff;">
     <strong>
      <em>
       为什么会有一个中国军事空间站藏在阿根廷？这东西究竟干什么用的、能让中国获得什么样的优势？
      </em>
     </strong>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   五角大楼高级官员越来越担心中国军方可以在阿根廷拉斯拉哈斯的一个秘密跟踪设施中监视和瞄准美国的卫星。
  </p>
  <p class="graf graf--p">
   2月的国会作证时，美国南方司令部新指挥官海军上将 Craig Faller 就警告说中国将迅速扩张到南美洲。
  </p>
  <p class="graf graf--p">
   Faller 告诉国会议员，中国积极支持古巴、尼加拉瓜和委内瑞拉的专制政府，并在整个非洲大陆采用掠夺性贷款做法。他说，北京在开发基础设施方面发挥了重要作用，例如位于巴塔哥尼亚沙漠的秘密天线。
  </p>
  <p class="graf graf--p">
   自该设施启动以来，美国情报官员一直在跟踪该设施的发展。
  </p>
  <p class="graf graf--p">
   在过去的几年里，位于该国西部200公顷的巴塔哥尼亚北建造了一个巨大的16层天线。
  </p>
  <p class="graf graf--p">
   专家说，一个8英尺高的安全围栏围绕着空间站，在阿根廷官员的有限监督下运作。
  </p>
  <p class="graf graf--p">
   安全世界基金会的空间政策和安全分析师 Brian Weeden 表示，美国军方部署的天线类似于阿根廷的这个天线。
  </p>
  <p class="graf graf--p">
   然而，美国军方官员担心该天线可用于收集有关美国间谍卫星位置和活动的敏感信息。
  </p>
  <p class="graf graf--p">
   Faller 说：北京可能违反了与阿根廷的协议条款，该条款规定其只能进行民事活动，并且可能有能力监视并可能针对美国、盟国和伙伴太空活动。他此前一直在五角大楼担任前国防部长 James Mattis 的军事助理。
  </p>
  <p class="graf graf--p">
   专家指出，北京声称仅将该设施用于和平目的可能具有欺骗性，因为中国国家航天局（目前经营该设施）与人民解放军关系密切。
  </p>
  <p class="graf graf--p">
   Frank Rose 于2014年至2017年担任美国负责军备控制的助理国务卿，目前是布鲁金斯学会的高级研究员，他表示该天线的定位为中国提供了西半球的重要太空覆盖，“这是一个覆盖某些轨道的问题。就如美国在全球拥有卫星跟踪站是有原因的 — 它可以提供全球覆盖，“他说。
  </p>
  <p class="graf graf--p">
   Evan Ellis 美国陆军战争学院战略研究所拉丁美洲研究讲师表示，天线的主要目的可能是用于太空研究。该设施的特点“与中国人所说的一致”，他说，并指出巴塔哥尼亚沙漠中的位置“确实有一定意义”，因为科学家们需要位于世界各地的天线进行外层空间观测。
  </p>
  <p class="graf graf--p">
   Ellis 表示，该设施有可能用于收集偶尔通过头顶的商用和军用卫星的敏感数据。
  </p>
  <p class="graf graf--p">
   由于缺乏对巴塔哥尼亚秘密空间站设施的开放性，中国人在西半球的确切行为尚属猜测。
  </p>
