  <p>
   多年前，中国的祝华新曾经直言不讳地讲出这样的话：“让人们在网上多骂几句他们就不会上街了”。当时我们分析过这件事。在广泛的舆论中这句话被解读为对所谓的键盘侠的嘲讽，但其实上没有这么简单。
  </p>
  <p class="graf graf--p">
   或者说，祝华新这句话远远不是全部。
  </p>
  <figure aria-describedby="caption-attachment-11679" class="wp-caption aligncenter" id="attachment_11679" style="width: 640px">
   <img alt="" class="size-full wp-image-11679 jetpack-lazy-image" data-lazy-sizes="(max-width: 640px) 100vw, 640px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/骂没有用.png?resize=640%2C608&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/骂没有用.png?w=640&amp;ssl=1 640w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/骂没有用.png?resize=300%2C285&amp;ssl=1 300w" data-recalc-dims="1" height="608" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/骂没有用.png?resize=640%2C608&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="640"/>
   <noscript>
    <img alt="" class="size-full wp-image-11679" data-recalc-dims="1" height="608" sizes="(max-width: 640px) 100vw, 640px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/骂没有用.png?resize=640%2C608&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/骂没有用.png?w=640&amp;ssl=1 640w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/骂没有用.png?resize=300%2C285&amp;ssl=1 300w" width="640"/>
   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-11679">
    一直以来，社交媒体异议群体都在做着腾讯玩家同样的事。当然，被骂的政权没有直接赚到钱，但是，他们赚到了因没有实质的抗议行动而带来的政权稳固
   </figcaption>
  </figure>
  <p class="graf graf--p">
   相关思考早已不是第一次被讲述。多年来不同领域的研究人士多次分析并质疑社交媒体促进社会运动的神话；从政治学角度上是对新媒介如何阻碍民主辩论的思考，在技术角度上是对算法专制的警告，在社会学角度上就是这篇文章尝试解释的问题。
  </p>
  <p class="graf graf--p">
   在很久很久以前，社交媒体曾经被很多人认为是政治和社会运动的新基础。但是，
   <strong class="markup--strong markup--p-strong">
    虽然在 Twitter 或 Facebook 上创建社交运动很容易，但将其转化为实际的社会运动以此来推动政策改变，却是截然不同的两码事
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在互联网彻底改变世界通信速度之前，社会运动的增长是缓慢的。蒙哥马利公交车抵制活动花了一年的时间实地地组织和动员变革；著名的民权运动花了十年的时间才促成“进军华盛顿”的示威。
  </p>
  <p class="graf graf--p">
   相比之下，像占领华尔街这样的大规模抗议活动能够迅速形成，但是，由于随后
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theatlantic.com/politics/archive/2015/06/the-triumph-of-occupy-wall-street/395408/" href="https://www.theatlantic.com/politics/archive/2015/06/the-triumph-of-occupy-wall-street/395408/" rel="noopener noreferrer" target="_blank">
    缺乏潜在的弹性
   </a>
   ，往往失去焦点、方向，最重要的是，它们有可能影响变革的推进。
  </p>
  <p class="graf graf--p">
   根据北卡罗来纳大学教堂山分校教授
   <a class="markup--anchor markup--p-anchor" data-href="https://sils.unc.edu/people/faculty/profiles/Zeynep-Tufekci" href="https://sils.unc.edu/people/faculty/profiles/Zeynep-Tufekci" rel="noopener noreferrer" target="_blank">
    Zeynep Tufekci
   </a>
   和 Tear Gas 的说法，土耳其的格兹公园抗议活动在几天之内就从无到有成为了一场大规模的运动，展示了组织动员使用数字工具的力量。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    正因为它太快了，或者说太仓促了，其结果出人意料……“当前社会运动形成的容易程度之高往往无法表明其组织能力强大到足以威胁当权者。”
   </strong>
  </p>
  <p class="graf graf--p">
   社会由共同行为驱动，即使我们不同意，共同行为依旧会形成一些东西。不仅由于推动人类进化的相互依赖，也由于我们共享空间：空气、水、土壤、道路、城镇、城市、景观 ……社区既可能也是必要的。
  </p>
  <p class="graf graf--p">
   特别是，我们的物理社区由空间组成，我们在这些空间中互动、交流，并努力相处。
  </p>
  <p class="graf graf--p">
   在公共场所我们向邻居打招呼、照看骑自行车的孩子、步行上班、给陌生人指路 ……不小心撞到人因为我们正在手机上输入“&lt;&lt; hugs &gt;&gt;”以回应朋友失恋的消息。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    学习信任和依赖邻居的最重要因素之一就是社区的街道生活
   </strong>
   。
  </p>
  <p class="graf graf--p">
   步行街区提供的日常活动也确保了我们每个人
   <strong class="markup--strong markup--p-strong">
    迫切需要的
   </strong>
   与其他人维持的定期联系。
  </p>
  <p class="graf graf--p">
   这些相互作用可能很小，通常几乎不易被察觉（比如跟邻居借根葱包饺子），但是我
   <strong class="markup--strong markup--p-strong">
    们的大脑已经发展到寻找自己的部落、自己的社区、和自己的邻居来提醒自己，我们并不孤单
   </strong>
   。
  </p>
  <p class="graf graf--p">
   研究表明，通过
   <strong class="markup--strong markup--p-strong">
    与邻居和社区之间的互动所建立的信任，显著增加了公民参与，同时增强了人们相互理解的能力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️有效的抗议不仅需要能够让人们聚集起来的权利，还需要可以进行聚集的公共空间、以及了解这些权利是什么的公民。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “公共的街道和公园一直被用于集会的目的…从古至今，街道和公共场所的这种使用已经成为基本人权、豁免、权利和公民自由的一部分” —— 这是美国最高法院于1939年作出的裁决，维护了公民在公共街道集结的权利，打击了市政要求此类集会需要事先获得许可证的要求。
  </p>
  <p class="graf graf--p">
   这些都是强有力的话语，吸取了历史的教训。
  </p>
  <p class="graf graf--p">
   当国家和地方政府开始限制集会权利必须在某些固定时间内和固定地点时，已经回到了19世纪初的状况，当20世纪中期最高法院开始允许这些限制性法规存在时，这一切对民主产生了非常实际的影响。
  </p>
  <p class="graf graf--p">
   就在2017年和2018年，
   <strong class="markup--strong markup--p-strong">
    美国有至少30个州的立法机构提出了限制集会自由的建议，并在某些情况下通过了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️有些法律禁止人们在参与抗议时戴口罩或面罩；其他的法律则试图通过加宽对“非法骚乱”的定义以削弱公众集会的集体力量。
  </p>
  <p class="graf graf--p">
   俄罗斯和土耳其等威权国家的抗议者被成百上千地逮捕，他们的集会权被剥夺。就像2015年在西班牙发生的情况一样，也许会令很多人对民主的未来感到不寒而栗。
  </p>
  <p class="graf graf--p">
   为了回应全面禁止任何形式抗议活动的官方意识，包括在国会门前举办的游行或集会，活动人士采取全息图抗议的形式。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    一万八千人将自己置身于全息影像抗议活动中，这些活动在国会门前举行预计持续数小时，同时激进派领导人也通过全息影像发表演讲
   </strong>
   。
  </p>
  <p class="graf graf--p">
   没有任何交通堵塞，依旧人声鼎沸。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4323 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、韩国-全息图.png?resize=1100%2C559&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、韩国-全息图.png?w=1408&amp;ssl=1 1408w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、韩国-全息图.png?resize=300%2C153&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、韩国-全息图.png?resize=768%2C391&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、韩国-全息图.png?resize=1024%2C521&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、韩国-全息图.png?resize=1100%2C559&amp;ssl=1 1100w" data-recalc-dims="1" height="559" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、韩国-全息图.png?resize=1100%2C559&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

    <img alt="" class="aligncenter size-full wp-image-4323" data-recalc-dims="1" height="559" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、韩国-全息图.png?resize=1100%2C559&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、韩国-全息图.png?w=1408&amp;ssl=1 1408w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、韩国-全息图.png?resize=300%2C153&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、韩国-全息图.png?resize=768%2C391&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、韩国-全息图.png?resize=1024%2C521&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2、韩国-全息图.png?resize=1100%2C559&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   与在线评论和请愿的不确定现实相结合，这是否就是我们的未来？政府机构会对全息抗议活动做出什么回应？他们真的会关心吗？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    互联网时代的点击式请愿书是快速收集成千上万个签名的完美形式，但是，它们不太可能更有效地触动政策的变更和投票意向的改变
   </strong>
   ，与一群有血有肉的人站在街头提出他们的要求相比，有天壤之别。
  </p>
  <p class="graf graf--p">
   通过不断的揭露已经越来越多人知道，恶意行为者、trolls 和假装成公民的自动机器人农场，不仅可以通过扩大分歧造成极化，而且还可以产生数以百万计伪造民意的在线评论，旨在以某种方式影响公共政策。
  </p>
  <p class="graf graf--p">
   数字技术已经开辟了难以想象的接入和与世界的连接，但是，它也使人们质疑其自身在政治抗议中的作用。
  </p>
  <p class="graf graf--p">
   当我们在人类与技术之关系的新时代中进行导航时，面对面接触和面对面的大规模抗议活动 —— 这种几个世纪以来始终作为争取完全公民权利之工具的社会运动 —— 已经必需变得更加重要。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    新的抗议时代将不得不学习如何将在线连接的简易性和速度与长期稳固的面对面组织行动相结合，从而给予身体力行的抗议以强度和持久力
   </strong>
   。
  </p>
