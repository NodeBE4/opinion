  <p class="graf graf--p">
   这是高信息技术和社交网络的数字化的时代。
  </p>
  <p class="graf graf--p">
   有人喜欢它，有人不喜欢，有人试图隐藏真实身份信息，不会出现在任何社交网络中、删除所有密码和注册。想要做到这点正在变得越来越难。
  </p>
  <p class="graf graf--p">
   然而，事实上，您可以使用 OSINT 技术找到有关每个用户的任何信息，这一事实首先得到了20世纪50年代美国间谍机构的公开证实。
  </p>
  <p class="graf graf--p">
   今天的互联网速度更快了、更容易获得想要的东西。正如您所知，你可以找到几乎一切 —— 只要您擅长搜索。
  </p>
  <p class="graf graf--p">
   但最重要的问题是，它还给我们带来了什么 —— 好处还是麻烦？
  </p>
  <p class="graf graf--p">
   毕竟，事实证明，我们没有多少选择的权利，也没有机会在网络中被忽视。甚至一个孩子都可以应对“电话号码搜索”这样的简单任务。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    每年 OSINT 技术的可能性都会快速增长、并变得真正无限！
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️想想普通用户在互联网上都干了什么。你就能知道 OSINT 可以多强大。
  </p>
  <p class="graf graf--p">
   以下是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.domo.com/learn/data-never-sleeps-5" href="https://www.domo.com/learn/data-never-sleeps-5" rel="noopener noreferrer" target="_blank">

   <a class="markup--anchor markup--p-anchor" data-href="https://www.domo.com/learn/data-never-sleeps-5" href="https://www.domo.com/learn/data-never-sleeps-5" rel="noopener noreferrer" target="_blank">
    数据
   </a>
   ，*仅仅*每分钟 —— 请注意是每分钟：
  </p>
  <p class="graf graf--p">
   – Snapchat 发布 527,760 张照片;
   <br/>
   – 超过 120 位专业人士加入领英;
   <br/>
   – YouTube 用户正在观看 4,146,600 个视频;
   <br/>
   – 向 Twitter 发送了 456,000 条推文;
   <br/>
   – Instagram 上有 46,740 张新照片出现 ……
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    是每分钟。而且仅仅是社交媒体
   </strong>
   。
  </p>
  <p class="graf graf--p">
   然后还有电子邮件、在线支付、数百个热门应用程序、云存储、工作站点和服务等等，这使得开源调查可以在搜索和处理任何信息方面拥有无限的可能性。
  </p>
  <p class="graf graf--p">
   但是，人们尚未完全考虑搜索引擎的吸引力 —— 全球每天50亿的最新数据！
  </p>
  <p class="graf graf--p">
   ⚠️ 普通的网络搜索和 OSINT 技术之间的主要区别在于方法的深度、以及各种方法和算法的可用性。
  </p>
  <p class="graf graf--p">
   这些方法和算法可以帮助我们实现任何任务：
   <strong class="markup--strong markup--p-strong">
    检测网络上用户的数字足迹、进行深度调查、追踪其数字关系链……
   </strong>
  </p>
  <p class="graf graf--p">
   可以根据搜索任务和您拥有的初始数据使用它们。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1.确定您拥有的源信息（照片、姓名、电子邮件地址、电话号码等）；
    <br/>
    2.确定您想要了解的最终目标；
    <br/>
    3.借助某些服务收集网络信息；
    <br/>
    4.分析和反向查找收到的信息；
    <br/>
    5.使用新接收的数据返回搜索；
    <br/>
    6.确认假设、或获得新假设。
   </strong>
  </p>
  <p class="graf graf--p">
   OSINT 技术与互联网上的简单搜索有何不同？
  </p>
  <p class="graf graf--p">
   💡不同在于：普通的搜索将在 OSINT 功能刚刚开始的地方结束。
  </p>
  <p class="graf graf--p">
   对于 OSINT 技术，没有统一的方法和模式化的调查算法，每个案例都是独特的，需要一个单独的方法。也就是说需要您的
   <strong class="markup--strong markup--p-strong">
    经验+创造力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这不是能够通过讲座传授的。
  </p>
  <p class="graf graf--p">
   💡 您需要不断参考成功的演示案例、积极研究新工具的使用技巧、以及最重要的是，不断尝试挑战自己。

  <p class="graf graf--p">
   什么是元搜索引擎：元搜索引擎又称集合型搜索引擎，将多个单一搜索引擎集成在一起，提供统一的检索界面，将用户的检索提问同时提交给多个独立的搜索引擎，同时检索多个数据库；并根据多个独立搜索引擎的检索结果进行二次加工，如对检索结果去重复、排序等。

    灰色文献搜索引擎
   </strong>

  <p class="graf graf--p">
   什么是灰色文献：是指介于正式发行的白色文献、和不公开出版并具有隐密性的黑色文献之间的文献，即 虽已出版但难以使用一般方式获得。
  </p>
  <p class="graf graf--p">
   目前对灰色文献的定义多取自1997年举行的“第三次国际灰色文献会议”中所提出的：不经营利出版者控制的，而由各级政府、学术单位、工商业界所产制的各类印刷与电子形式的资料。
