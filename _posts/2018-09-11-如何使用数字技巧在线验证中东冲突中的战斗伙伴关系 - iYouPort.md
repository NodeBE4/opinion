     <em>
      <strong>
       看起来合理的不一定就是真实的。中东冲突中的假消息比子弹和无人机轰炸更泛滥，给分析和追责带来了极大的难度。如今中国对中东的兴趣已经引来了观察家们的议题，然而我们发现其中有些人引用的论据却是谣传……
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4678 jetpack-lazy-image" data-lazy-sizes="(max-width: 600px) 100vw, 600px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-178.jpeg?resize=600%2C338&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-178.jpeg?w=600&amp;ssl=1 600w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-178.jpeg?resize=300%2C169&amp;ssl=1 300w" data-recalc-dims="1" height="338" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-178.jpeg?resize=600%2C338&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="600"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4678" data-recalc-dims="1" height="338" sizes="(max-width: 600px) 100vw, 600px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-178.jpeg?resize=600%2C338&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-178.jpeg?w=600&amp;ssl=1 600w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-178.jpeg?resize=300%2C169&amp;ssl=1 300w" width="600"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="9679">
   社交媒体和智能手机在整个中东的流通已经令大量公开可用的关于该地区冲突的文件公之于众。无论是视频还是照片，从战斗片段到潜在的战争罪行 — 很多事情都经常被记录下来并上传到各种网站，如 Facebook，Twitter 和 YouTube。
  </p>
  <p class="graf graf--p graf-after--p" id="3e84">
   这些照片和视频给记者、分析家和研究这些冲突的其他人，提供了一把双刃剑。
  </p>
  <p class="graf graf--p graf-after--p" id="5022">
   一方面，主要的源素材和照片可以让记者发现战争罪和其他侵犯人权的行为。然而，另一方面，被错误归因的照片和视频可能造成的是虚假报道或错误信息，以操纵观众错误的结论。
  </p>
  <p class="graf graf--p graf-after--p" id="0585">
   例如，一张关于遭受酷刑的被拘留者的视频被张贴在 Twitter 上，发布者声称该视频描绘了亲政府部队在叙利亚东部 Ghouta 折磨一名平民。由于亲政府部队多次对全国各地的囚犯施以酷刑，这种镜头一点都不稀奇。
  </p>
  <p class="graf graf--p graf-after--p" id="f783">
   但是，如果你仔细看一下这视频就能发现，某些方面似乎不太对劲。首先，囚犯和俘虏似乎正在用乌尔都语进行交流，这是一种在叙利亚不常见的南亚语言。如果囚犯确实是叙利亚人，他就不会用乌尔都语说话。利用这条线索，验证 Sy-Sy 的记者 Dirar Khatab（一个非营利组织，负责验证叙利亚冲突中的视频和镜头）
   <a class="markup--anchor markup--p-anchor" data-href="https://www.verify-sy.com/det/947/ar" href="https://www.verify-sy.com/det/947/ar" rel="noopener nofollow noreferrer" target="_blank">
    确定
   </a>
   该视频未在叙利亚发生，而是此前一年，发生在巴基斯坦。
  </p>
  <p class="graf graf--p graf-after--p" id="a4ee">
   这种虚假信息如今非常多见。记者、分析师和研究人员需要找到一些方法以验证他们正在查看的素材或照片究竟是不是发布者所描述的内容。事实上在线你就能完成这项工作，这里有大量
   <a class="markup--anchor markup--p-anchor" data-href="https://steemit.com/tools/@iyouport/2zmazk" href="https://steemit.com/tools/@iyouport/2zmazk" rel="noopener nofollow noreferrer" target="_blank">
    可用于验证信息的开源工具
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="5466">
   本指南则是为了具体介绍如何确定在中东冲突中的照片或视频中所显示的战斗人员属于哪一方，
   <strong class="markup--strong markup--p-strong">
    这是验证该地区冲突的视频和照片之真实性的关键步骤。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="4337">
   <strong class="markup--strong markup--p-strong">
    军装
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="0d2e">
   军服可以提供关于战斗人员是谁、以及他们隶属于哪个团体的线索。例如，军服上的迷彩图案通常代表其隶属于特定的国家、军事力量甚至是非国家行为者。
  </p>

  <p class="graf graf--p graf-after--figure" id="86bc">
   上图，在叙利亚一群亲政府部队的例子中，最右边的战斗人员与其他战斗人员穿着不同类型的迷彩服。你可以通过 Camopedia 轻松识别他的制服，
   <a class="markup--anchor markup--p-anchor" data-href="http://camopedia.org/index.php?title=Main_Page" href="http://camopedia.org/index.php?title=Main_Page" rel="noopener nofollow noreferrer" target="_blank">
    Camopedia
   </a>
   是世界各地迷彩图案的开源数据库。通过数据库，您可以看到战斗人员的制服似乎大致与伊朗的伊斯兰革命卫队（IRGC） Basij 的模式相匹配。这可能表明战斗人员与 IRGC Basij 部队存在联系。
  </p>
  <p class="graf graf--p graf-after--p" id="6ed1">
   除了 Camopedia 之外，“
   <a class="markup--anchor markup--p-anchor" data-href="http://camouflageindex.camouflagesociety.org/index-2.html" href="http://camouflageindex.camouflagesociety.org/index-2.html" rel="noopener nofollow noreferrer" target="_blank">
    世界迷彩图案索引
   </a>
   ”也可以成为识别关于制服的有用资源。但是，索引不包括示例图片。尽管如此，它也可能更有助于识别 Camopedia 上未能列出的更加模糊的迷彩图案。
  </p>
  <p class="graf graf--p graf-after--p" id="d61d">
   值得注意的是，单凭制服并不足以确定战斗人员的隶属关系，特别是考虑到黑市上迷彩服的泛滥。
   <strong class="markup--strong markup--p-strong">
    其他证据通常需要与制服信息一起使用
   </strong>
   ，以确定战斗人员的隶属关系。
  </p>
  <p class="graf graf--p graf-after--p" id="f895">
   <strong class="markup--strong markup--p-strong">
    袖章和徽章
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="1bde">
   代表官衔等级或单位隶属关系的军事袖章和徽章是验证或识别战斗人员隶属关系的好方法。最明显的第一步是搜索在徽章上找到的任何单词或短语，这每个人都知道；通常可以通过搜索
   <a class="markup--anchor markup--p-anchor" data-href="http://www.uniforminsignia.org/" href="http://www.uniforminsignia.org/" rel="noopener nofollow noreferrer" target="_blank">
    国际统一徽章百科全书
   </a>
   来识别没有文字的徽章。维基百科的
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/List_of_comparative_military_ranks" href="https://en.wikipedia.org/wiki/List_of_comparative_military_ranks" rel="noopener nofollow noreferrer" target="_blank">
    比较军衔名单
   </a>
   在识别徽章方面也非常有用。
  </p>

      <figcaption class="wp-caption-text" id="caption-attachment-4683">
       这名士兵袖子上的标示意味着叙利亚真主党
      </figcaption>
     </figure>
     <p class="graf graf--p graf-after--figure" id="d9c3">
      例如，上面照片中的战士，在他的制服上有一个黄色的真主党袖章。乍看似乎表明战斗员与真主党有关联。然而，这名战斗人员实际上是俄罗斯军方的成员，
      <strong class="markup--strong markup--p-strong">
       基于他的装备和制服
      </strong>
      。他穿着真主党的袖章可能是企图隐藏他真正的归属关系，或者可能是为了表明对俄罗斯正在与之并肩作战的组织表示团结。这个例子说明了袖章和徽章不是确证的方法。
     </p>
     <p class="graf graf--p graf-after--p" id="5ecf">
      <strong class="markup--strong markup--p-strong">
       武器装备
      </strong>
     </p>
     <p class="graf graf--p graf-after--p" id="88bc">
      识别战斗人员使用的武器或装备可以揭示他们的归属关系。某些武器系统更有可能在特定国家和该国的军队中找到。小型武器 — 如步枪、子机枪和手枪 — 通常不仅仅能指出战斗人员所属的军队，有时甚至可以识别士兵所属的单位。
     </p>
     <figure aria-describedby="caption-attachment-4684" class="wp-caption aligncenter" id="attachment_4684" style="width: 410px">

      塔布克狙击步枪，伊拉克 7.62x39mm 指定神枪手步枪。这条线索表明，战斗者可能在伊拉克，然后使用反向图像搜索寻找其他信息，最后确定他是伊拉克民兵的一部分。

     <p class="graf graf--p graf-after--figure" id="b3f0">
      如上面的推文所示，Sayyad 5.56 步枪是伊朗克隆的中国 Norinco CQ，这表明该战斗人员与伊朗有联系。值得注意的是，使用小型武器来确定战斗人员的隶属关系也并不是一种万无一失的办法，特别是考虑到整个中东地区小型武器的扩散状况。
     </p>
     <p class="graf graf--p graf-after--p" id="3fc0">
      理想情况下，
      <strong class="markup--strong markup--p-strong">
       多种线索结合使用是很棒的
      </strong>
      ，如制服或徽章、与小武器识别结合使用，将足够确认战斗人员的隶属关系。要识别小武器，您可以使用小武器调查的
      <a class="markup--anchor markup--p-anchor" data-href="http://www.smallarmssurvey.org/weapons-and-markets/tools/weapons-id-database.html" href="http://www.smallarmssurvey.org/weapons-and-markets/tools/weapons-id-database.html" rel="noopener nofollow noreferrer" target="_blank">
       武器 ID 数据库
      </a>
      按口径、类型、位置等数据搜索小型武器。
     </p>
     <p class="graf graf--p graf-after--p" id="ce22">
      <strong class="markup--strong markup--p-strong">
       结论
      </strong>
     </p>
     <p class="graf graf--p graf-after--p" id="80e7">
      总的来说，识别或验证中东的战斗人员从属关系时会遇到很多不同状况。有时，简单的反向图像搜索就足以确定战斗员隶属于哪一方。而其他时候，可能非常具有挑战性，需要语言和文化知识、开源研究技能、地理定位、武器识别等等。本指南仅介绍了验证战斗人员从属关系的几种方法。对于刚接触数字取证领域的初学者和其他人来说，识别战斗人员使用的制服、徽章和武器通常至少可以揭示目标人在冲突的哪一方，如果不能验证其所属的特定群体的话。
     </p>
     <p class="graf graf--p graf-after--p" id="278a">
      <strong class="markup--strong markup--p-strong">
       验证战斗人员来自哪一方可以帮助记者、执法人员和研究人员识别战争罪行的肇事者。例如叙利亚暴行
      </strong>
      。
      <strong class="markup--strong markup--p-strong">
       此外，这种验证还可用于检查可能基于不准确信息的报道
      </strong>
      。鉴于互联网上存在大量的虚假信息和错误信息，数字验证是​​对中东冲突进行开源调查时必不可少的一步。
     </p>

     <p class="graf graf--p graf-after--p" id="0519">
      ⚠️ 任何时候，制造虚假图像和信息都是严重的错误，不论你的目标有多正义，只能证明你不可信。信息环境不是中立的：敌对行动者一直在千方百计地寻找你的漏洞，借以攻击你。现代技术环境已经越来越容易验证假货，开源调查是独立媒体技能刚需。
     </p>
     <p class="graf graf--p graf-after--p" id="2aba">
      **
      <strong class="markup--strong markup--p-strong">
       (Open-source intelligence) OSINT
      </strong>
      是从公开可用来源收集数据的调查。OSINT 不同于一般的调查研究（Research），它是为特定的个人或团队量身定制特定的情报处理工作，以支持其必须做出的特定决策。不论是情报机构、调查性新闻记者还是民间独立组织**
