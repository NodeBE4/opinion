   <span style="font-size: 9pt">
    ：中国地方政府债务问题近来引起国内外广泛关注。中国社会科学院经济研究所研究员、天则经济研究所学术委员会主席张曙光
   </span>
   <span style="font-size: 9pt">
    7
   </span>
   <span style="font-size: 9pt">
    月
   </span>
   <span style="font-size: 9pt">
    19
   </span>
   <span style="font-size: 9pt">
    日
   </span>
   <span style="font-size: 9pt">
    在天则经济研究所主办的一场宏观经济形势报告会上谈及此问题，并回答了包括
   </span>
   <span style="font-size: 9pt">
    FT
   </span>
   <span style="font-size: 9pt">
    中文网在内的媒体的提问。我们特将张曙光其发言及问答中涉及地方政府债务的部分整理成文，以飨读者。
   </span>
   <span style="font-size: 9pt">
    ]
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    6
   </span>
   <span style="font-size: 9pt">
    月
   </span>
   <span style="font-size: 9pt">
    27
   </span>
   <span style="font-size: 9pt">
    日
   </span>
   <span style="font-size: 9pt">
    ，国家审计署发布《全国地方政府性债务审计结果》。该报告显示，截至
   </span>
   <span style="font-size: 9pt">
    2010
   </span>
   <span style="font-size: 9pt">
    年底，全国地方政府性债务余额
   </span>
   <span style="font-size: 9pt">
    107174.91
   </span>
   <span style="font-size: 9pt">
    亿元，占
   </span>
   <span style="font-size: 9pt">
    GDP
   </span>
   <span style="font-size: 9pt">
    的
   </span>
   <span style="font-size: 9pt">
    27%
   </span>
   <span style="font-size: 9pt">
    。其中，政府负有偿还责任的债务
   </span>
   <span style="font-size: 9pt">
    67109.51
   </span>
   <span style="font-size: 9pt">
    亿元，占
   </span>
   <span style="font-size: 9pt">
    62.62%
   </span>
   <span style="font-size: 9pt">
    ；政府负有担保责任的或有债务
   </span>
   <span style="font-size: 9pt">
    23369.74
   </span>
   <span style="font-size: 9pt">
    亿元，占
   </span>
   <span style="font-size: 9pt">
    21.80%
   </span>
   <span style="font-size: 9pt">
    ；政府可能承担一定救助责任的其他相关债务
   </span>
   <span style="font-size: 9pt">
    16695.66
   </span>
   <span style="font-size: 9pt">
    亿元，占
   </span>
   <span style="font-size: 9pt">
    15.58%
   </span>
   <span style="font-size: 9pt">
    。至此，中国地方政府债务第一次有了权威的官方数字。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    这一权威数字既有可信之处，又有可质疑之处。据审计署官员介绍，今年上半年，审计署派出
   </span>
   <span style="font-size: 9pt">
    4.13
   </span>
   <span style="font-size: 9pt">
    万名审计人员，历时三个月，对
   </span>
   <span style="font-size: 9pt">
    187.36
   </span>
   <span style="font-size: 9pt">
    万笔地方政府性债务逐笔、逐项进行了审计，工作深入细致，因而结果是可信的。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    但是，人民银行发布的《
   </span>
   <span style="font-size: 9pt">
    2010
   </span>
   <span style="font-size: 9pt">
    年中国区域金融运行报告》称，地方融资平台贷款近
   </span>
   <span style="font-size: 9pt">
    14.4
   </span>
   <span style="font-size: 9pt">
    万亿元。银监会公布的数据是，截至
   </span>
   <span style="font-size: 9pt">
    2010
   </span>
   <span style="font-size: 9pt">
    年
   </span>
   <span style="font-size: 9pt">
    11
   </span>
   <span style="font-size: 9pt">
    月，全国地方融资平台贷款约
   </span>
   <span style="font-size: 9pt">
    9.1
   </span>
   <span style="font-size: 9pt">
    万亿元。国外一些机构（如穆迪评级机构）估计的数字更高。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    上述不同统计有概念上的差异。国家审计署审计的范围包括地方融资平台债务和地方政府债券，涵盖面大于央行和银监会。但审计署审计的地方融资平台为
   </span>
   <span style="font-size: 9pt">
    6576
   </span>
   <span style="font-size: 9pt">
    个，比央行统计的数字少了
   </span>
   <span style="font-size: 9pt">
    3000
   </span>
   <span style="font-size: 9pt">
    个。根据审计署的报告，在
   </span>
   <span style="font-size: 9pt">
    10.7
   </span>
   <span style="font-size: 9pt">
    万亿元地方政府债务中，有
   </span>
   <span style="font-size: 9pt">
    8.5
   </span>
   <span style="font-size: 9pt">
    万亿元为银行贷款。如果把审计署未审计的地方融资平台包括在内，并按审计署已审计单位的平均数计算，那么，地方债务有可能接近
   </span>
   <span style="font-size: 9pt">
    15.6
   </span>
   <span style="font-size: 9pt">
    万亿元，其中银行贷款为
   </span>
   <span style="font-size: 9pt">
    12.37
   </span>
   <span style="font-size: 9pt">
    万亿元。前者超过了央行报告中所述的数
   </span>
   <span style="font-size: 9pt">
    字，后者接近穆迪公司的估计，即审计署的审计低估了
   </span>
   <span style="font-size: 9pt">
    3.5
   </span>
   <span style="font-size: 9pt">
    万亿元银行贷款。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    如果将
   </span>
   <span style="font-size: 9pt">
    10.7
   </span>
   <span style="font-size: 9pt">
    万亿元的地方政府性债务，加上近三年中央政府代地方政府发行的
   </span>
   <span style="font-size: 9pt">
    6000
   </span>
   <span style="font-size: 9pt">
    亿元债券、公共部门债务余额，以及公共部门的隐形负债，中国公共部门债务余额占
   </span>
   <span style="font-size: 9pt">
    GDP
   </span>
   <span style="font-size: 9pt">
    的比重约为
   </span>
   <span style="font-size: 9pt">
    50%
   </span>
   <span style="font-size: 9pt">
    左右，在
   </span>
   <span style="font-size: 9pt">
    60%
   </span>
   <span style="font-size: 9pt">
    的预警线之下。从总量上看，风险似乎可控，但从结构上看，风险依然不小。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    大公国际近期认为，中国地方政府债务没有太大的风险，但我认为是有风险的。主要体现为，第一，真实的债务数额到底有多少，我们心里"没底"。我们没有数据，只能按照审计署提供的数据来计算。如前所述，审计署审计的地方融资平台比央行少
   </span>
   <span style="font-size: 9pt">
    3000
   </span>
   <span style="font-size: 9pt">
    家，少了近三分之一。从这个角度来看的话，我觉得近
   </span>
   <span style="font-size: 9pt">
    15.6
   </span>
   <span style="font-size: 9pt">
    万亿元的数字也是有根据的。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    第二，中国地方债务下一步还会增加，原因就在于
   </span>
   <span style="font-size: 9pt">
    6
   </span>
   <span style="font-size: 9pt">
    月底发改委发布通知，允许地方政府通过发债来建保障房。我估计，下一步，地方政府以修建保障房的名义借债会成为一股风潮，类似的事情过去是出现过的。这个问题在下文中将详述。
   </span>
  </div>
  <div style="text-indent: 18pt">
  </div>
  <div style="text-indent: 18pt">
   <span style="font-size: 9pt">
    第三，一些人认为地方政府债务风险不大的一个理由是，中国地方财政收入增长地很快。但是，据统计，
   </span>
   <span style="font-size: 9pt">
    2010
   </span>
   <span style="font-size: 9pt">
    年底，已有
   </span>
   <span style="font-size: 9pt">
    78
   </span>
   <span style="font-size: 9pt">
    个市级和
   </span>
   <span style="font-size: 9pt">
    99
   </span>
   <span style="font-size: 9pt">
    个县级政府负有偿还责任的债务率超过
   </span>
   <span style="font-size: 9pt">
    100%
   </span>
   <span style="font-size: 9pt">
    ，即其综合财力小于当期债务，处于破产境地。由于偿债能力不足，一些地方政府开始借新还旧，甚至出现了逾期债务。这样的问题，就不是靠财政收入增长所能解决的问题了。
   </span>
  </div>
