  <p class="graf graf--p">
   中国公司在非洲扩张已经不是什么秘密，并且中国的影响力在过去十年中已经大幅增加，该报道确认了，华为一直在监视由北京支持的非洲领导人的政治对手。
  </p>
  <p class="graf graf--p">
   该报告援引了高级安全官员与这些国家的华为员工进行的协调，详细介绍了该公司的技术人员如何“帮助非洲政府
   <strong class="markup--strong markup--p-strong">
    监视他们的政治对手，⚠️包括拦截目标人的加密通信和社交媒体，以及利用手机信息跟踪他们的行踪
   </strong>
   ”。
  </p>
  <p class="graf graf--p">
   该报告记录了六起重大事件，其中华为技术人员入侵并拦截了非洲领导人和政府的通信。
  </p>
  <p class="graf graf--p">
   北京官员和华为高管都否认了这一报道，称他们没有授权对非洲领导人或政府进行任何此类黑客攻击或网络窃听援助。
  </p>
  <p class="graf graf--p">
   华为在向“华尔街日报”发表的一份书面声明中表示，该公司“从未参与’黑客攻击’活动”，并进一步表示“完全拒绝这些……指控。”
  </p>
  <p class="graf graf--p">
   以下是该报道的亮点：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    乌干达 ——
   </strong>
  </p>
  <p class="graf graf--p">
   根据乌干达高级安全官员的说法，去年乌干达坎帕拉的一组六名情报人员在努力遏制对约韦里穆塞韦尼总统33年政权的威胁。
  </p>
  <p class="graf graf--p">
   Bobi Wine 从华盛顿返回，美国支持他的反对派运动；乌干达的网络监视部门有严格的命令以
   <strong class="markup--strong markup--p-strong">
    拦截他的加密通信
   </strong>
   ，利用的是2010年的法律赋予政府的广泛权力能够‘确保其多方面的利益’。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “华为技术人员工作了两天，以帮助我们做到这点，”监控部门的一名高级官员说。
  </p>
  <p class="graf graf--p">
   华尔街日报看到，华为工程师的名字就写在内部警方文件中，请注意 ⚠️使用的是以色列制造的间谍软件以渗透到 Wine 先生的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%82%a8%e5%bf%85%e9%9c%80%e4%bd%bf%e7%94%a8-whatsapp%ef%bc%9a%e8%bf%99%e9%87%8c%e6%98%af%e5%87%8f%e5%b0%91%e4%bc%a4%e5%ae%b3%e7%9a%84%e5%8a%9e%e6%b3%95/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%82%a8%e5%bf%85%e9%9c%80%e4%bd%bf%e7%94%a8-whatsapp%ef%bc%9a%e8%bf%99%e9%87%8c%e6%98%af%e5%87%8f%e5%b0%91%e4%bc%a4%e5%ae%b3%e7%9a%84%e5%8a%9e%e6%b3%95/" rel="noopener noreferrer" target="_blank">
    WhatsApp 聊天
   </a>
   群组。
  </p>
  <p class="graf graf--p">
   当局破坏了他组织街头集会的计划，并逮捕了数十名支持者。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    赞比亚 ——
   </strong>
  </p>
  <p class="graf graf--p">
   在赞比亚，根据那里的高级安全官员的说法，华为技术人员帮助当地政府监视了反对派博客团队的手机和 Facebook 页面，这是反对派新闻网站的页面，该网站多次批评总统埃德加伦古。
  </p>
  <p class="graf graf--p">
   高级安全官员在赞比亚电信监管机构办公室的一个
   <strong class="markup--strong markup--p-strong">
    网络监控部门
   </strong>
   找到了两名华为专家 —— 是他们帮助确定了博客的所在位置，并与部署在西北部城市 Solwezi 的警察部队保持联系。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    阿尔及利亚 ——
   </strong>
  </p>
  <p class="graf graf--p">
   一名官员说，“我们讨论了针对可能威胁国家安全的反对派的黑客攻击”，并补充说，阿尔及利亚人在该领域取得了进步。
  </p>
  <p class="graf graf--p">
   该报道引用了一份乌干达和阿尔及利亚部队共同编写的机密报告，这份报告称阿尔及利亚使用的系统为“华为的智能视频监控系统”，称其为“先进系统，提供最好的监控应用之一”。
  </p>
  <p class="graf graf--p">
   当月晚些时候，警察局局长Kayihura先生签署了一项合作协议，让阿尔及利亚团队就乌干达华为实施的监控计划的推出提供建议。
  </p>
  <p class="graf graf--p">
   根据乌干达高级官员的说法，该项目的主要阿尔及利亚顾问被阿尔及利亚人描述为是“
   <strong class="markup--strong markup--p-strong">
    在华为深圳总部接受培训
   </strong>
   ”的网络专家。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    打击当地反对派博主和活动家
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “华为技术人员帮助拦截反对派博客博主的通信
   </strong>
   ，这些博主正在运行一个名为 Koswe 的新闻网站，该网站反复批评领导人 Lungu”，网络犯罪小组的两名赞比亚官员说。
  </p>
  <p class="graf graf--p">
   华为工作人员访问了该博客的 Facebook 页面，在那里他们找到了博主的电话号码，然后
   <strong class="markup--strong markup--p-strong">
    使用其他公司生产的间谍软件
   </strong>
   来查看和查找设备。
  </p>
  <p class="graf graf--p">
   4月18日，一队网络官员、警方情报官员和 Zicta 专家挤在总统官邸底层的Chanda先生办公室里。
  </p>
  <p class="graf graf--p">
   赞比亚情报官员说，当时两名华为技术人员打开他们的笔记本电脑，上面显示与目标博主的 Facebook 页面相关联的
   <strong class="markup--strong markup--p-strong">
    多个手机的实时跟踪路线，地图上还绘制了华为手机天线
   </strong>
   。
  </p>
