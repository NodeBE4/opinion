  <p class="graf graf--p">
   还记得我们的 “变得难以被追踪” 系列中强调的原则吗？👉简约主义是一种生存智慧 —— 您注册的东西、下载的东西越多，对您的攻击面就越大。所以，
   <strong class="markup--strong markup--p-strong">
    把不再用的东西全部清理掉，是一个最基本的安全措施
   </strong>
   。
  </p>
  <p class="graf graf--p">
   从互联网建立到现在，我们每个人都注册过很多东西，几乎很少有人能准确说出自己曾经都注册过什么。
  </p>
  <p class="graf graf--p">
   但是，这其中有至少一半东西如今您已经不再登陆了，今后也可能不会再使用；而您曾经交出的用于注册的电子邮件地址或电话号码，都可能会被滥用。
  </p>
  <p class="graf graf--p">
   现在该问问自己这个问题了：“我如何找到所有链接到我的真实电子邮件地址的帐户？”
  </p>
  <p class="graf graf--p">
   最好重新访问您的所有帐户以更新您的凭据（把真的改为假的）或干脆停用它们。
  </p>
  <p class="graf graf--p">
   本文将介绍一些方法，帮助您找到与您的电子邮件地址或电话号码关联的所有帐户。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、查找链接到电子邮件的帐户
   </strong>
  </h3>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12649 jetpack-lazy-image" data-lazy-sizes="(max-width: 670px) 100vw, 670px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-1.png?resize=670%2C419&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-1.png?w=670&amp;ssl=1 670w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-1.png?resize=300%2C188&amp;ssl=1 300w" data-recalc-dims="1" height="419" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-1.png?resize=670%2C419&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="670"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12649" data-recalc-dims="1" height="419" sizes="(max-width: 670px) 100vw, 670px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-1.png?resize=670%2C419&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-1.png?w=670&amp;ssl=1 670w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/1-1.png?resize=300%2C188&amp;ssl=1 300w" width="670"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   如果您经常通过电子邮件平台的快速授权按钮登录应用程序和网站，那么您就不必挖掘太多，至少可以快速查看您最近创建的帐户。
  </p>
  <p class="graf graf--p">
   对于 “使用Google注册” 等选项生成的个人资料，您只需转到电子邮件的安全设置即可。在这里，您可以浏览已连接应用程序的列表，以编辑或撤消其访问权限。
  </p>
  <p class="graf graf--p">
   要在 Google 上访问此部分，请转到 “我的帐户” 仪表板，然后单击左侧的 “安全性” 标签。滚动直到遇到 “具有帐户访问权限的第三方应用程序”，然后单击 “管理第三方访问权限”。然后，您可以点击各个应用以将其与您的电子邮件帐户断开连接。
  </p>
  <p class="graf graf--p">
   一旦摆脱了不需要的第三方应用程序，就应该考虑增强在线安全性了。
   <strong class="markup--strong markup--p-strong">
    保持简约主义大原则、并采取分身的技巧，尽可能减少风险。
   </strong>
  </p>

  <p class="graf graf--p">
   评估您使用社交媒体帐户登录的所有应用和网站。当您将第三方服务附加到您的社交账户个人资料时，风险甚至会更高。
  </p>
  <p class="graf graf--p">
   根据权限，您有可能面临着各种风险：联系人列表、个人详细信息、手机号码、和其他私人详细信息等。
  </p>
  <p class="graf graf--p">
   一般情况下您可以决定要共享或切断的内容。例如，您可以保留第三方帐户，但禁止其访问 Facebook 上您点赞的页面。

   <li class="graf graf--li">
    在 Facebook，转到 Settings &gt; Apps and Websites
   </li>
   <li class="graf graf--li">
    在 Twitter，Settings &gt; Account &gt; Apps and Sessions

  <p class="graf graf--p">
   上面的两种方法能做到的并不全面，于是您需要第三种方法。
  </p>
  <p class="graf graf--p">
   要进行彻底检查，您将需要返回到电子邮件并在收件箱中查找注册账户时使用的确认邮件。搜索常见主题行的信息。
  </p>
  <p class="graf graf--p">
   另外，您可以使用Gmail的搜索运算符和关键字来过滤特定主题行。
  </p>
  <p class="graf graf--p">
   例如，您可以输入 “subject: verify” 以提取主题行包含 “验证” 的所有电子邮件。
  </p>
  <p class="graf graf--p">
   您也可以使用
   <a class="markup--anchor markup--p-anchor" data-href="https://emailexport.io/" href="https://emailexport.io/" rel="noopener noreferrer" target="_blank">
    EmailExport
   </a>
   之类的工具来自动执行此任务。该网站可以使用功能更强大的过滤器梳理您的电子邮件，然后将结果整理到电子表格中。这项服务不是免费的，但可以为您节省很多时间。
  </p>
  <p class="graf graf--p">
   但是
   <strong class="markup--strong markup--p-strong">
    请注意
   </strong>
   。由于 EmailExport 将有权阅读您的收件箱，这本身是一个风险。对于前一百封邮件，EmailExport 是免费的，超额的数量您将必须为每250封电子邮件至少支付5美元。
  </p>

  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e6%9e%84%e5%bb%ba%e8%87%aa%e5%ae%9a%e4%b9%89%e7%9a%84%e7%94%a8%e6%88%b7%e5%90%8d%e4%be%a6%e6%9f%a5%e5%b7%a5%e5%85%b7%ef%bc%9a%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e6%8a%80%e6%9c%af/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e6%9e%84%e5%bb%ba%e8%87%aa%e5%ae%9a%e4%b9%89%e7%9a%84%e7%94%a8%e6%88%b7%e5%90%8d%e4%be%a6%e6%9f%a5%e5%b7%a5%e5%85%b7%ef%bc%9a%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e6%8a%80%e6%9c%af/" rel="noopener noreferrer" target="_blank">
     如何构建自定义的用户名侦查工具：开源调查技术
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e7%9a%84%e5%85%a8%e8%83%bd%e9%92%a5%e5%8c%99%e6%89%a3%ef%bc%9a%e4%bd%9c%e4%b8%ba%e4%b8%80%e5%90%8d%e5%85%ac%e6%b0%91%e8%b0%83%e6%9f%a5%e8%80%85%e7%9a%84%e5%9f%ba%e6%9c%ac%e8%84%91%e5%9b%be/" href="https://www.iyouport.org/%e4%bd%a0%e7%9a%84%e5%85%a8%e8%83%bd%e9%92%a5%e5%8c%99%e6%89%a3%ef%bc%9a%e4%bd%9c%e4%b8%ba%e4%b8%80%e5%90%8d%e5%85%ac%e6%b0%91%e8%b0%83%e6%9f%a5%e8%80%85%e7%9a%84%e5%9f%ba%e6%9c%ac%e8%84%91%e5%9b%be/" rel="noopener noreferrer" target="_blank">

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e8%b7%a8%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e5%b9%b3%e5%8f%b0%e8%bf%bd%e8%b8%aa%e7%9b%ae%e6%a0%87%e4%ba%ba%ef%bc%9f%e2%9a%a0%ef%b8%8f-%e8%ad%a6%e5%91%8a%e5%ae%89%e5%85%a8%e6%84%8f/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e8%b7%a8%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e5%b9%b3%e5%8f%b0%e8%bf%bd%e8%b8%aa%e7%9b%ae%e6%a0%87%e4%ba%ba%ef%bc%9f%e2%9a%a0%ef%b8%8f-%e8%ad%a6%e5%91%8a%e5%ae%89%e5%85%a8%e6%84%8f/" rel="noopener noreferrer" target="_blank">

  <p class="graf graf--p">
   如果您平时没有注意分身，也就是说在不同平台上倾向于使用相同或近似的用户名，则可以使用
   <a class="markup--anchor markup--p-anchor" data-href="https://namechk.com/" href="https://namechk.com/" rel="noopener noreferrer" target="_blank">
    Namechk
   </a>
   进行查找。
  </p>
  <p class="graf graf--p">
   该工具可以帮助您在数十个平台上扫描用户名。只需在顶部输入您最常用的ID，Namechk 就会告诉您是否以及在哪里被使用过。
  </p>
  <p class="graf graf--p">
   Namechk 插入了一系列服务，例如 Instagram，PayPal，Imgur，Foursquare、Venmo 等等。 Namechk 是免费的。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    6、检查浏览器的已保存帐户
   </strong>
  </h3>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12654 jetpack-lazy-image" data-lazy-sizes="(max-width: 670px) 100vw, 670px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/6-1.png?resize=670%2C439&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/6-1.png?w=670&amp;ssl=1 670w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/6-1.png?resize=300%2C197&amp;ssl=1 300w" data-recalc-dims="1" height="439" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/6-1.png?resize=670%2C439&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="670"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12654" data-recalc-dims="1" height="439" sizes="(max-width: 670px) 100vw, 670px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/6-1.png?resize=670%2C439&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/6-1.png?w=670&amp;ssl=1 670w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/6-1.png?resize=300%2C197&amp;ssl=1 300w" width="670"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   每当您在互联网上填写表单字段时，浏览器就会缓存您的输入。这适用于电子邮件地址，也包括密码。
  </p>
  <p class="graf graf--p">
   因此，您可以访问浏览器的设置，然后在列表中查找可能忘记的任何帐户。请注意，成功取决于您使用该浏览器已有多长时间。您可以对过去安装的每个浏览器重复此过程。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    在Chrome 上，转到：Settings &gt; Auto-fill &gt; Passwords。您可以浏览、更新、或者干脆删除它们，因为通常保存这种东西并不安全。
   </li>
   <li class="graf graf--li">
    在 Firefox 上，转到：Settings &gt; Privacy &amp; Security &gt; Login and Passwords &gt; Saved Logins.
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    重申！我们在 “
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     开源情报
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ” 专栏中介绍的大部分工具都可以用来帮您对自己使用，检查自己的漏洞，抢在攻击者到来之前弥补 —— 对抗性思考方式
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    使用密码管理器对您的在线帐户进行排序
   </strong>
  </p>
  <p class="graf graf--p">
   您不太可能完整找到您曾经创建的每个在线帐户。但是，有了这些解决方案，您就可以挖掘到其中的大多数。
  </p>
  <p class="graf graf--p">
   完成上面的操作后，最好将其迁移到密码管理器，并定期使用它轻松管理在线状态。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    不要错过《
