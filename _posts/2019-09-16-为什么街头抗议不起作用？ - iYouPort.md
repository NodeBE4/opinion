  <p>
   <img alt="" class="aligncenter size-full wp-image-4331 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02为什么抗议不起作用.png?resize=1100%2C610&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02为什么抗议不起作用.png?w=1440&amp;ssl=1 1440w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02为什么抗议不起作用.png?resize=300%2C166&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02为什么抗议不起作用.png?resize=768%2C426&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02为什么抗议不起作用.png?resize=1024%2C567&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02为什么抗议不起作用.png?resize=1100%2C610&amp;ssl=1 1100w" data-recalc-dims="1" height="610" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02为什么抗议不起作用.png?resize=1100%2C610&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4331" data-recalc-dims="1" height="610" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02为什么抗议不起作用.png?resize=1100%2C610&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02为什么抗议不起作用.png?w=1440&amp;ssl=1 1440w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02为什么抗议不起作用.png?resize=300%2C166&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02为什么抗议不起作用.png?resize=768%2C426&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02为什么抗议不起作用.png?resize=1024%2C567&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02为什么抗议不起作用.png?resize=1100%2C610&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   街头抗议活动正在此起彼伏中。
  </p>
  <p class="graf graf--p">
   从曼谷到加拉加斯、从马德里到莫斯科，大量的人群聚集在世界各大城市的街头，你几乎不会在任何一周内看不到这类消息出现在新闻中。
  </p>
  <p class="graf graf--p">
   抗议活动的原因各不相同（公共交通问题或教育问题、生活成本过高问题、公园乱砍滥伐问题，警察暴力虐待问题……等等）。
  </p>
  <p class="graf graf--p">
   通常，申诉能够迅速扩大到包括对政府或其领导人的抵抗，或对腐败和经济不平等的更普遍的谴责。
  </p>
  <p class="graf graf--p">
   反政府游行的航拍照片经常显示出一种令人热血沸腾的人山人海，你能看出人们急需改变的迫切之心。
  </p>
  <p class="graf graf--p">
   然而，令人惊讶的是这些人群的成就究竟有多少 ……？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当地热切的政治活力与这些示威活动的实际结果极不相称。令人惊讶。
   </strong>
  </p>
  <p class="graf graf--p">
   当然值得注意的例外也是存在的：比如在埃及、突尼斯和乌克兰，街头抗议活动实际上促成了政权更迭。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    大多数大规模的集会并未能在政治或公共政策方面造成重大变化。
   </strong>
  </p>
  <p class="graf graf--p">
   占领华尔街就是一个很好的例子。
  </p>
  <p class="graf graf--p">
   2011年夏天首次爆发（不是在华尔街，而是在吉隆坡的
   <a class="markup--anchor markup--p-anchor" data-href="http://www.esquire.my/InTheMagazine/Sneak-Peek/article/The-Great-Occupation" href="http://www.esquire.my/InTheMagazine/Sneak-Peek/article/The-Great-Occupation" rel="noopener noreferrer" target="_blank">
    Dataran Merdeka
   </a>
   ）占领运动迅速蔓延，并
   <strong class="markup--strong markup--p-strong">
    很快就在全球近 2,600 个城市的中心广场咆哮起来
   </strong>
   。
  </p>
  <p class="graf graf--p">
   参与的人群多种多样，基本是大杂烩，彼此之间没有正式的联系，也没有明确的等级制度，并没有明显的领导者。简称无中心。
  </p>
  <p class="graf graf--p">
   但社交网络有助于这些抗议信息的病毒式传播、并
   <strong class="markup--strong markup--p-strong">
    引发了其他地区的复制运动，因此，露营、抗议、筹款、与媒体沟通、以及与当局谈判的基本模式，在各地都比较相似。
   </strong>
  </p>
  <p class="graf graf--p">
   同样的信息随处可见：全球财富集中在1％的精英手中是不可接受的，而剩下的99％在生死线上挣扎。
  </p>
  <p class="graf graf--p">
   这种全球性的、大规模的、看似组织良好的抗议运动，本应产生更大的影响。理论上。
  </p>
  <p class="graf graf--p">
   但事实上却没有。
  </p>
  <p class="graf graf--p">
   虽然经济不平等的话题在过去几年中已经获得了发展势头，而在实践中，很难根据占领的提议找到有意义的公共政策变化。
  </p>
  <p class="graf graf--p">
   总的来说，“占领”运动不久就从头条新闻中消失了。
  </p>
  <p class="graf graf--p">
   事实上，
   <strong class="markup--strong markup--p-strong">
    政府的回应通常只不过是委婉修辞的敷衍之举，当然也不会有重大的政治改革。
   </strong>
  </p>
  <p class="graf graf--p">
   例如，巴西前总统迪尔玛·罗塞夫（Dilma Rousseff）就证实了那些走上街头的人的挫败感，她承诺会做出改变，但这些“改变”一直没有实现。
  </p>
  <p class="graf graf--p">
   土耳其总理雷杰普·塔伊普·埃尔多安（Recep Tayyip Erdogan）对他的国家抗议活动的反应更加激进。他指责反对派和抗议者是在“密谋策划针对他的所谓复杂阴谋”，并试图封锁 Twitter 和 YouTube。
  </p>
  <p class="graf graf--p">
   在墨西哥城的反暴力抗议和新德里的反腐败示威游行期间，也出现了同样的事：
   <strong class="markup--strong markup--p-strong">
    大规模的游行，但获得的成果却很少。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    为什么？这不公平。这么多极具行动热情的人怎么能收获这么少？
   </strong>
  </p>
  <p class="graf graf--p">
   在哥本哈根大学的 Anders Colding-Jørgensen 进行的实验结果中可以找到一个答案。
  </p>
  <p class="graf graf--p">
   2009年，Colding-Jørgensen 创建了一个 Facebook 小组，组织抗议在丹麦首都的一个主要广场拆除历史悠久的鹳鸟喷泉。
  </p>
  <p class="graf graf--p">
   第一周内就加入了一万人;；两周后，该团体成员数量增加到 27,000 人。这就是实验的范围。
  </p>
  <p class="graf graf--p">
   这只是实验。从来没有计划拆除鹳鸟喷泉 —— Colding-Jørgensen 只是想表明使用社交媒体创建一个相对较大的团队是多么的容易。
  </p>
  <figure aria-describedby="caption-attachment-4332" class="wp-caption aligncenter" id="attachment_4332" style="width: 1142px">
   <img alt="" class="size-full wp-image-4332 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02-中间.png?resize=1100%2C730&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02-中间.png?w=1142&amp;ssl=1 1142w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02-中间.png?resize=300%2C199&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02-中间.png?resize=768%2C510&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02-中间.png?resize=1024%2C680&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02-中间.png?resize=1100%2C730&amp;ssl=1 1100w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02-中间.png?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="730" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/02-中间.png?resize=1100%2C730&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   在今天的世界中，通过 Twitter、Facebook 或其他在线通信工具来组织抗议，有可能不难吸引人群，特别是如果行动是为了证明某些东西真的令人愤怒的话。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    问题是游行后发生的事
   </strong>
   。有时它会以与警察的暴力对抗结束，而且往往只会失败。
  </p>
  <p class="graf graf--p">
   ⚠️在大规模的街头示威活动背后，很少有一个运作良好且更具永久性的组织能够跟进抗议者的要求，并进行复杂的、面对面的、沉闷的政治工作，从而在政府中推动真正变化的产生。
  </p>
  <p class="graf graf--p">
   这是普林斯顿大学信息技术政策中心研究员 Zeynep Tufekci 所提出的重要观点，他
   <a class="markup--anchor markup--p-anchor" data-href="http://www.nytimes.com/2014/03/20/opinion/after-the-protests.html?_r=0" href="https://www.nytimes.com/2014/03/20/opinion/after-the-protests.html?_r=0" rel="noopener noreferrer" target="_blank">
    写道
   </a>
   ：
   <strong class="markup--strong markup--p-strong">
    “在互联网出现之前，为规避审查和组织抗议活动而进行的繁琐工作本身有助于建立决策基础设施和维持行动力的战略。但是，现在，运动很可能会匆匆走过这一环节，往往给自己带来不利。
   </strong>
   “
  </p>
  <p class="graf graf--p">
   许多城市的街道上都有一个强大的政治引擎。它转动高速并产生很多政治能量。
   <strong class="markup--strong markup--p-strong">
    但发动机没有连接到车轮，因此“运动”不会移动。
   </strong>
  </p>
  <p class="graf graf--p">
   💡要改变这点，就要求组织能够进行老式的和永久性的政治工作，将街头示威活动纳入政治变革和政策改革。
  </p>
  <p class="graf graf--p">
   在大多数情况下，这意味着政党。但并非意味着示威者不信任的现有政党。
  </p>
  <p class="graf graf--p">
   相反，我们需要新的或经过深刻改革的政党，这些政党可以激励那些在政治上无家可归的理想主义者以及
   <strong class="markup--strong markup--p-strong">
    专注于建立一个知道如何将政治能量转化为公共政策的政治组织
   </strong>
   的专业人士。
  </p>
  <p class="graf graf--p">
   正如许多人所指出的那样，⚠️社交媒体既可以促进也可以破坏更有效的政党的形成。
  </p>
  <p class="graf graf--p">
   我们熟悉社交媒体识别、招募、动员和协调支持者以及筹集资金的能力。但是，我们也知道，点击主义和懈怠主义会通过创造
   <strong class="markup--strong markup--p-strong">
    一种感觉良好的错觉
   </strong>
   来破坏真实的政治工作，即 将在 Facebook 页面上点击“like”或者发推
   <strong class="markup--strong markup--p-strong">
    误认为
   </strong>
   是等同于影响变化的行动。
  </p>
  <p class="graf graf--p">
   近年来人们目睹了街头游行的普及，没有计划接下来会发生什么，以及如何让抗议者参与并融入政治进程。
  </p>
  <p class="graf graf--p">
   这只是危险错觉的最新表现，即 误认为没有反对党就可以实现民主 ——
   <strong class="markup--strong markup--p-strong">
    抗议活动被更多地建立在社交媒体上，而不是持续的政治组织中，而后者才是改变社会的方式
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theatlantic.com/international/archive/2014/04/why-street-protests-dont-work/360264/" href="https://www.theatlantic.com/international/archive/2014/04/why-street-protests-dont-work/360264/" rel="noopener noreferrer" target="_blank">
    Why Street Protests Don’t Work
   </a>
   , How can so many demonstrations accomplish so little?
  </p>
