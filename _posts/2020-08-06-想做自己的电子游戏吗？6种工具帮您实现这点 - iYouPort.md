   但是如果更多人学会自己制作游戏呢？在您所在的社区内实现更迷人的技术技巧培训？会不会更好 ……
  </p>
  <p class="graf graf--p">
   人们也许对游戏开发有所误解，认为似乎只有充足预算和才华横溢的人才可以进入游戏开发领域。事实上并非如此。
  </p>
  <p class="graf graf--p">
   对于独立的小型游戏开发者而言，如今不仅是历史上最好的时机，而且，实现您的视频游戏愿景也能是很简单的事。
  </p>
  <p class="graf graf--p">
   已经有大量的游戏创建工具就摆在那里，非专业和没有技术经验的开发人员也可以使用它们来实现自己的愿景。
  </p>
  <p class="graf graf--p">
   本文将介绍六个工具的介绍，旨在从成本和技术技能两方面减轻游戏开发的辛苦。其中有些甚至可以被用来创造那种能红极一时的游戏，帮你赚钱。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、Stencyl
   </strong>
  </h3>
  <p>
   <img alt="" class="aligncenter size-full wp-image-13191 jetpack-lazy-image" data-lazy-sizes="(max-width: 1088px) 100vw, 1088px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/1.png?resize=1088%2C604&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/1.png?w=1088&amp;ssl=1 1088w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/1.png?resize=300%2C167&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/1.png?resize=1024%2C568&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/1.png?resize=768%2C426&amp;ssl=1 768w" data-recalc-dims="1" height="604" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/1.png?resize=1088%2C604&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1088"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://www.stencyl.com/" href="http://www.stencyl.com/" rel="noopener noreferrer" target="_blank">
    Stencyl
   </a>
   是基于订阅的游戏创建工具，可让您轻松地为多个平台创建2D游戏。游戏准备就绪后，您可以将其作为应用程序发布到 Android 和 iOS，但也支持所有桌面平台和浏览器。
  </p>
  <p class="graf graf--p">
   如果您喜欢手机游戏，那么您很有可能实际玩过 Stencyl 制作的游戏。
  </p>
  <p class="graf graf--p">
   使用该工具编写的许多游戏已跻身各种应用商店的前十名。 其中
   <a class="markup--anchor markup--p-anchor" data-href="https://play.google.com/store/apps/details?id=com.kappsule.wrassling&amp;hl=en" href="https://play.google.com/store/apps/details?id=com.kappsule.wrassling&amp;hl=en" rel="noopener noreferrer" target="_blank">
    Wrassling
   </a>
   是一个特别有趣的游戏，在发行时得到了很多好评。
  </p>
  <p class="graf graf--p">
   Stencyl 引以为豪的是，制作者可以完全不必为任何代码而揪头发，至少不是原始形式。在游戏逻辑方面，他们明智地使用拼合系统，将不同的逻辑参数汇总在一起，然后填写变量。
  </p>
  <p class="graf graf--p">
   因此，实际上，有很多“编码”，但是坦率地说，它就像使用乐高积木一样简单。
  </p>

  <p class="graf graf--p">
   JRPG 仍然代表着蓬勃发展的游戏类型，现在您可以使用
   <a class="markup--anchor markup--p-anchor" data-href="http://www.rpgmakerweb.com/http://www.rpgmakerweb.com/" href="https://www.rpgmakerweb.com/http://www.rpgmakerweb.com/" rel="noopener noreferrer" target="_blank">
    RPG Maker
   </a>
   轻松制作经典的《最终幻想》那类游戏。有许多版本的 RPG Maker 可选。
  </p>
  <p class="graf graf--p">
   最好仔细检查下哪个版本的工具最适合您的需求，因为特定版本具有某些特定的功能。例如，虽然 RPG Maker XP 最便宜很诱人，但有些技巧您无法使用。但另一方面，由于某些开发者对其地图系统的偏爱，XP仍然具有强大的追随者。
  </p>
  <p class="graf graf--p">
   如果您想制作特定风格的游戏（或进行一些类似的修改），那么 RPG Maker 应该是您的第一站。
  </p>
  <p class="graf graf--p">
   RPG Maker 一直是 Windows 游戏的创造者，但现在可以发布到其他平台（例如 Android），尽管需要一点调整才能使其正常工作。
  </p>

  </p>
  <p class="graf graf--p">
   Twine 突破了应该算作游戏创作工具的极限。它最初被用来建立用户自己选择的冒险型体验.
  </p>
  <p class="graf graf--p">
   Twine 是一种开源工具，基本上可以让您构建一个庞大的故事节点网络，然后将其发布为可运行的HTML应用。
  </p>
  <p class="graf graf--p">
   能将 Twine 从交互式的叙事工具提升到可以实际制作游戏的东西是在于它的脚本编写能力。用户能够构建简单的地牢爬虫并将游戏玩法添加到自己的故事中。
  </p>
  <p class="graf graf--p">
   Twine 2.0无疑是此列表中最基本的工具之一，但是如果使用得当，它可以用于获得令人难以置信的强大效果。还记得
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%97%a0%e5%bd%a2%e7%9a%84%e6%9e%81%e6%9d%83%e4%b8%bb%e4%b9%89/" href="https://www.iyouport.org/%e6%97%a0%e5%bd%a2%e7%9a%84%e6%9e%81%e6%9d%83%e4%b8%bb%e4%b9%89/" rel="noopener noreferrer" target="_blank">
    《黑镜》“潘达斯奈基” 那集
   </a>
   吗？它里面就使用了 Twine 2.0。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、Game Maker Studio 2
   </strong>
  </h3>
  <p>
   <img alt="" class="aligncenter size-full wp-image-13195 jetpack-lazy-image" data-lazy-sizes="(max-width: 1088px) 100vw, 1088px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/4-1.png?resize=1088%2C520&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/4-1.png?w=1088&amp;ssl=1 1088w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/4-1.png?resize=300%2C143&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/4-1.png?resize=1024%2C489&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/4-1.png?resize=768%2C367&amp;ssl=1 768w" data-recalc-dims="1" height="520" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/4-1.png?resize=1088%2C520&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1088"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13195" data-recalc-dims="1" height="520" sizes="(max-width: 1088px) 100vw, 1088px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/4-1.png?resize=1088%2C520&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/4-1.png?w=1088&amp;ssl=1 1088w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/4-1.png?resize=300%2C143&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/4-1.png?resize=1024%2C489&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/4-1.png?resize=768%2C367&amp;ssl=1 768w" width="1088"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Game Maker Studio 2是游戏创作软件包中的最新版本，可追溯到1999年，当时它被称为 Animo。
  </p>
  <p class="graf graf--p">
   这是一款用于制作2D游戏的工具，几乎包含制作2D风格的游戏所需的一切。一些3D的东西也是可能的，但没有2D创建系统那种通常简单直观的环境。
  </p>
  <p class="graf graf--p">
   通过 Game Maker Studio，您可以为2D对象设置动画、绘制原始2D图元并获得所需的外观。它具有许多直观的工具可以使用。
  </p>
  <p class="graf graf--p">
   在 Game Maker Studio 的帮助下有许多商业上成功的游戏，但是最著名的游戏无疑是 Undertale。
  </p>
  <p class="graf graf--p">
   该产品有许多平台支持。您可以发布到 Windows，macO，Ubuntu，HTML5，Android，iOS，PS4，Xbox One 甚至是 Switch。
  </p>
  <p class="graf graf--p">
   Game Maker Studio 不是免费的，但是有一个负担得起的 “ Creator Edition”，它限制了用户在 Windows 和 Macintosh 上进行发布。但您始终可以升级到 “Desktop” 许可，以扩展您的游戏的购买范围。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、ClickTeam Fusion 2.5
   </strong>
  </h3>
  <p>
   <img alt="" class="aligncenter size-full wp-image-13196 jetpack-lazy-image" data-lazy-sizes="(max-width: 1098px) 100vw, 1098px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5-1.png?resize=1098%2C616&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5-1.png?w=1098&amp;ssl=1 1098w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5-1.png?resize=300%2C168&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5-1.png?resize=1024%2C574&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5-1.png?resize=768%2C431&amp;ssl=1 768w" data-recalc-dims="1" height="616" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5-1.png?resize=1098%2C616&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1098"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13196" data-recalc-dims="1" height="616" sizes="(max-width: 1098px) 100vw, 1098px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5-1.png?resize=1098%2C616&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5-1.png?w=1098&amp;ssl=1 1098w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5-1.png?resize=300%2C168&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5-1.png?resize=1024%2C574&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5-1.png?resize=768%2C431&amp;ssl=1 768w" width="1098"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   由法国开发者 Clickteam 创建的 Fusion 2.5 对于想要制作2D游戏但又不想花几年时间学习编码的人来说是一个好的选择。
  </p>
  <p class="graf graf--p">
   它使用了定制的编码语言，非常简单，以至于普通用户可以在不到一小时的时间内学会它。
  </p>
  <p class="graf graf--p">
   有大量使用 Fusion 2.5 制作的商业上成功的独立游戏，其中最著名的可能是Freddy 的《五夜狂欢》。开发人员使用 Fusion 2.5 制作游戏，并使用 3ds Max 预先渲染3D图形。
  </p>
  <p class="graf graf--p">
   这个例子只是说明，只要您有创造力就可以做到什么样。
  </p>
  <h2 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    6、Construct 3
   </strong>
  </h2>
  <p>
   <img alt="" class="aligncenter size-full wp-image-13197 jetpack-lazy-image" data-lazy-sizes="(max-width: 1088px) 100vw, 1088px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/6-1.png?resize=1088%2C680&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/6-1.png?w=1088&amp;ssl=1 1088w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/6-1.png?resize=300%2C188&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/6-1.png?resize=1024%2C640&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/6-1.png?resize=768%2C480&amp;ssl=1 768w" data-recalc-dims="1" height="680" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/6-1.png?resize=1088%2C680&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1088"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13197" data-recalc-dims="1" height="680" sizes="(max-width: 1088px) 100vw, 1088px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/6-1.png?resize=1088%2C680&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/6-1.png?w=1088&amp;ssl=1 1088w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/6-1.png?resize=300%2C188&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/6-1.png?resize=1024%2C640&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/6-1.png?resize=768%2C480&amp;ssl=1 768w" width="1088"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Construct 3 是高度流行游戏开发工具的最新版本。与 Stencyl 一样，您需要每月（或每年）支付访问开发环境的费用。您制作的任何游戏都是您的财产，您可以自由出售它们以获取利润。
  </p>
  <p class="graf graf--p">
   Construct 3 仍是相当新的东西，但是 Construct 2 已经产生了一些值得注意的游戏。
  </p>
  <p class="graf graf--p">
   Construct 可以导出到 JavaScript、HTML5 和其他一些与PC和Web兼容的标准，也有原生的 Android 和 iOS 支持。
  </p>
  <p class="graf graf--p">
   可视化脚本系统可能是 Construct 的最大卖点，并且被认为是对游戏逻辑设计的一种特别直观的理解。定价也相当合理，特别是如果您考虑成果可能是一款热门独立游戏能为您带来多少财富的话。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    打造自己的大片
   </strong>
  </h3>
  <p class="graf graf--p">
   游戏世界充斥着各种小型独立开发者的故事，他们凭借一款没人能放下的游戏而大受欢迎。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    游戏好还是不好，并不在于投了多少钱；好的游戏都必需具有创新的设计，完善和引人注目的中心机制的特定组合。
   </strong>
  </p>
  <p class="graf graf--p">
   虽然掌握一些编码的技术技能总是一个好主意，但是这些直观的游戏创作工具意味着您无需费心就能制作出您一直想要的游戏。它所要做的就是创造力，和一个好点子。
  </p>
  <p class="graf graf--p">
   要不要试试看？⚪️
  </p>
