  <p class="graf graf--p">
   2014年7月，我在该网站上写了一篇博客文章，报道了互联网社区内部有关以中国 “防火长城” 为代表的 “分裂网” 兴起的争议和辩论。
  </p>
  <p class="graf graf--p">
   Bill Gates 和 Vin Cerf 当年认为，互联网过于广泛和普遍，以至于政府对互联网的限制无法成功。另一方面，思科的 Eric Sc​​hmidt 和 John Chamber 指责 NSA 的元数据监听是 “分裂网” 发展的重要因素，这将严重损害西方技术的领先地位。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    现实情况是，互联网分裂已经演变了许多年
   </strong>
   —— 因为许多政府试图阻止互联网破坏其权力和权威，其中一些早于美国国家安全局、英国间谍机构GCHQ、和中国军方搞乱互联网的行动被揭露的时间。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     《
    </strong>
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e4%b8%bb%e6%9d%83%e7%9a%84%e5%85%a8%e7%90%83%e5%b4%9b%e8%b5%b7%ef%bc%9a%e8%bf%99%e6%84%8f%e5%91%b3%e7%9d%80%e4%bb%80%e4%b9%88%ef%bc%9f/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e4%b8%bb%e6%9d%83%e7%9a%84%e5%85%a8%e7%90%83%e5%b4%9b%e8%b5%b7%ef%bc%9a%e8%bf%99%e6%84%8f%e5%91%b3%e7%9d%80%e4%bb%80%e4%b9%88%ef%bc%9f/" rel="noopener noreferrer" target="_blank">

  <p class="graf graf--p">
   今天看来，Sc​​hmidt 和 Chamber 显然是正确的。Tim Berners-Lee 也加入了这一行列，他发起了一项全球行动计划，以保护互联网免遭政治操纵。
  </p>
  <p class="graf graf--p">
   但是，我对减少这一问题的任何全球多边协议均表示怀疑。借助暗网、虚假消息、政治压制、阴谋论以及其他所有内容，分裂网已经成为魔鬼之子。
  </p>
  <p class="graf graf--p">
   我看到的唯一昏暗的光线可能是使用了VPN之类的变通办法，或者可能是昂贵到不切实际的低轨卫星互联网，这会使
   <a class="markup--anchor markup--p-anchor" data-href="https://start.me/p/nRBzO9/iyp-3" href="https://start.me/p/nRBzO9/iyp-3" rel="noopener noreferrer" target="_blank">
    政府的政治操纵
   </a>
   稍微有点难度。仅此而已。
  </p>
  <p class="graf graf--p">
   去年初，随着津巴布韦在反政府抗议活动之后切断了全国的互联网访问，权利组织 Keep It On 警告说，“绝不能让断网成为新常态。”
  </p>
  <p class="graf graf--p">
   但是，十二个月后，它真的成新常态了……
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12258 jetpack-lazy-image" data-lazy-sizes="(max-width: 780px) 100vw, 780px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-5.jpg?resize=780%2C438&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-5.jpg?w=780&amp;ssl=1 780w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-5.jpg?resize=300%2C168&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-5.jpg?resize=768%2C431&amp;ssl=1 768w" data-recalc-dims="1" height="438" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-5.jpg?resize=780%2C438&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="780"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   克什米尔新闻记者抗议印度政府于2019年10月12日在斯利那加实施的互联网封锁。
  </p>
  <p class="graf graf--p">
   追踪互联网自由的倡导组织 Access Now 说，印度控制的克什米尔地区正在进行的互联网中断
   <strong class="markup--strong markup--p-strong">
    目前是民主国家中持续时间最长的
   </strong>
   ——
   <a class="markup--anchor markup--p-anchor" data-href="https://internetshutdowns.in/" href="https://internetshutdowns.in/" rel="noopener noreferrer" target="_blank">
    超过135天
   </a>
   。
  </p>
  <p class="graf graf--p">
   只有中国和缅甸军政府时期的独裁政府才可能超过这一期限。
  </p>
  <p class="graf graf--p">
   在新德里取消了该地区的法律自治权后，印度军队涌入克什米尔，导致断网。一些克什米尔人不知道互联网被切断的原因。从那时起，由于没有互联网访问，人们在很大程度上被噤声了，该地区的人很难在彼此间传达信息。
  </p>
  <p class="graf graf--p">
   克什米尔长达4个月的互联网中断非常严重，以至于 WhatsApp 已经开始删除那里的居民的帐户，原因是他们不活跃。
  </p>
  <p class="graf graf--p">
   联合国见解和言论自由问题特别报告员 David Kaye 将这起断网描述为 “通讯围困” 和 “集体惩罚”，甚至没有指控根本性的罪行。
  </p>
  <p class="graf graf--p">
   很快，断网又蔓延到了印度其他地方，包括首都新德里部分地区在内的多个地区的当局在有争议的公民权法案引起广泛抗议的情况下削减了电话和互联网服务。
  </p>
  <p class="graf graf--p">
   最近的断网影响了数千万的人。此举引起广泛的愤慨，并遭到了政界人士甚至
   <a class="markup--anchor markup--p-anchor" data-href="https://timesofindia.indiatimes.com/city/guwahati/gauhati-hc-directs-assam-govt-to-restore-mobile-internet-services/articleshow/72886722.cms" href="https://timesofindia.indiatimes.com/city/guwahati/gauhati-hc-directs-assam-govt-to-restore-mobile-internet-services/articleshow/72886722.cms" rel="noopener noreferrer" target="_blank">
    一些法官
   </a>
   的压制。
  </p>
  <p class="graf graf--p">
   印度对中国日益严格的互联网审查制度表示欢迎，而中国官方媒体指称，这是对北京自身威权主义做法的认可。
  </p>
  <p class="graf graf--p">
   ⚠️人民日报称，印度的例子表明 “在紧急状态下关闭互联网应该是主权国家的标准做法。”
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这一切是从哪里开始的？
   </strong>
  </h3>
  <p class="graf graf--p">
   在2009年发生在中国西部省份新疆的动乱之后，中国当局切断了整个地区的互联网和电话访问权限。
  </p>
  <p class="graf graf--p">
   虽然以前在全球各地都发生过断网情况，但那通常都是偶然的，是由于海底电缆损坏所致。而新疆的断网似乎是一种反常现象，这种战术当时被认为只有中国那样的专制政权才能使用或可以使用。
  </p>
  <p class="graf graf--p">
   此后的十年中，互联网在整个发展中国家范围内掀起了一场通信革命，许多国家都在跨越旧技术，更多地采用智能手机、社交媒体和移动支付，推动经济增长并与社区建立联系。
  </p>
  <p class="graf graf--p">
   ⚠️ 但是，随着越来越多的国家看到互联网被用来组织政治变革，断网已经变得越来越普遍，这是控制动荡和扼杀异议的首选工具。
  </p>
  <p class="graf graf--p">
   断网策略已经不仅限于独裁国家了：迄今为止，最严重的罪犯是世界最大的民主国家印度。
  </p>

   <li class="graf graf--li">

     谁正在成为独裁者？这十点迹象帮你判断
    </a>
    》

  <p class="graf graf--p">
   非洲国家也接受了这一策略，津巴布韦、刚果民主共和国、乍得和埃塞俄比亚都切断了互联网访问，
   <strong class="markup--strong markup--p-strong">
    以遏制反政府的抗议活动
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这符合增加在线审查的一般模式。部分原因是由于更先进的技术的普及，使得在线监控和过滤流量更加容易，也更便宜。
  </p>
  <p class="graf graf--p">
   它也受到对互联网审查制度的看法转变的影响，该制度曾一度被视为失败的战斗。但是，中国的GFW无疑已得到证明 ——
   <strong class="markup--strong markup--p-strong">
    不仅可以控制互联网，而且这样做可以帮助支撑政权并防止反对派运动发动进攻
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%b8%ad%e5%9b%bd%e7%9a%84%e4%ba%92%e8%81%94%e7%bd%91%e7%bb%b4%e7%a8%b3%e4%ba%a7%e4%b8%9a%ef%bc%9a%e8%b0%81%e5%b0%86%e6%88%90%e4%b8%ba%e5%94%af%e4%b8%80%e7%9a%84%e5%a4%b1%e8%b4%a5%e8%80%85%ef%bc%9f/" href="https://www.iyouport.org/%e4%b8%ad%e5%9b%bd%e7%9a%84%e4%ba%92%e8%81%94%e7%bd%91%e7%bb%b4%e7%a8%b3%e4%ba%a7%e4%b8%9a%ef%bc%9a%e8%b0%81%e5%b0%86%e6%88%90%e4%b8%ba%e5%94%af%e4%b8%80%e7%9a%84%e5%a4%b1%e8%b4%a5%e8%80%85%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     中国的互联网维稳产业：谁将成为唯一的失败者？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   尽管可以说GFW开始了互联网关闭的趋势，但北京在这方面并不经常犯规。
  </p>
  <p class="graf graf--p">
   中国的审查制度通常足够复杂，以至于当局无需使用诸如关闭所有访问权限之类的钝器 ——
   <strong class="markup--strong markup--p-strong">
    它们通过内容过滤首先就可以达到相同的目的，因为人们只能知道当权者想让他们知道的东西
   </strong>
   。
  </p>
  <p class="graf graf--p">
   对于其他国家，尤其是在发展中国家，
   <strong class="markup--strong markup--p-strong">
    仅关闭互联网可能是达到这种控制水平的诱人捷径。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️断网使警察可以更自由地控制抗议活动，而无需在高度联系的社会中针对社交媒体进行严格审查，并使政府能够确保官方宣传性信息是在特定主题上唯一能被大众听到的信息。
  </p>
  <p class="graf graf--p">
   根据 Access Now 组织的记录，2018年全球有196次互联网关闭 —— 主要在亚洲、非洲和中东。仅2019年上半年就有128次断网，而2019年似乎是有记录以来最糟糕的一年。
  </p>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    主要影响
   </strong>
  </h3>
  <p class="graf graf--p">
   失去互联网访问不仅是烦人的事，而且还会在动荡时期使人们失去重要信息来源，从而使他们处于危险之中。它还限制了媒体令政府负责的能力。
  </p>
  <p class="graf graf--p">
   全球大部分经济都依赖于通信技术 —— 从高速股票交易到物流和交付服务的所有内容 —— 互联网的关闭也会对经济产生重大影响。
  </p>
  <p class="graf graf--p">
   印度国际经济关系研究委员会估计，在2012年至2017年之间，断网已经给该国经济造成超过30亿美元的损失，而此后几年又出现了更多断网。
  </p>
  <p class="graf graf--p">
   断网会影响公民在动乱或紧急情况下获取准确信息的能力。公民也会更加难以与该国其他地区或其他国家的家人和朋友联系。并且阻碍了急救人员和医疗服务提供者的工作。
  </p>
  <p class="graf graf--p">
   互联网关闭的方法，尤其是完全关闭的方法，其实非常简单。政府只需要命令互联网服务提供商（ISP）断开与外界的连接 —— 就如关闭家用调制解调器，但要覆盖整个国家。
  </p>
  <p class="graf graf--p">
   完全关闭后，通常无法解决。如果没有互联网连接，VPN等反审查工具将无法使用。
  </p>
  <p class="graf graf--p">
   提供互联网访问的替代方法很大程度上还处于婴儿期。
   <strong class="markup--strong markup--p-strong">
    卫星提供的互联网服务价格昂贵，并且还要取决于私营公司是否愿意反抗政府
   </strong>
   。
  </p>
  <p class="graf graf--p">
   网状网络的采用需要一定水平的人口密度，而这在许多地区不一定能达到这个水平，例如克什米尔。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该怎么办？
   </strong>
  </h3>
  <p class="graf graf--p">
   在许多情况下，尤其是全面关闭互联网，可能会违反1948年《世界人权宣言》和其他国际条约所保障的权利。联合国还确认了会员国有责任保护人们访问互联网的权利和在线言论自由权。
  </p>
  <p class="graf graf--p">
   Access Now 的总顾问 Peter Micek 说：“联合国应跟踪并呼吁谴责每一次关闭互联网的行为。”
  </p>
  <p class="graf graf--p">
   他补充说，断网不仅可能侵犯联合国声称保护的权利，而且还会阻碍国际援助工作、安全与发展。“我们现在不再将目光局限于国际电联这样的技术机构、资助机构，东盟和非洲联盟这样的区域机构，而是联合国秘书长本人，以明确指出断网是不可接受的措施，是对国际法的侮辱。”
  </p>
  <p class="graf graf--p">
   但是，拥有权利和行使权利却有很大的不同。在印度，一些个人因断网向法院提起了诉讼，并且正在不断努力修改该国有关互联网关闭的法律，以使其更难以实施。
  </p>
  <p class="graf graf--p">
   根据 Access Now 的报道，人们日益意识到由互联网关闭造成的损害，这导致法院面临的诉讼更加频繁。“即使法院没有直接判决原告胜诉，甚至没有审理此案，采取法律行动也可能对那些因互联网关闭而受苦的人们的人权产生积极的影响。”
  </p>
  <p class="graf graf--p">
   这些案例中有许多是由个人提起的，而不是由互联网公司提起的，即使它们通常受断网的影响最大。
  </p>
  <p class="graf graf--p">
   Access Now 说，行业可能在这场斗争中发挥更大的作用，在财务和声誉方面都起着重要作用。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    但是在许多国家/地区，法院的权力不足，不太可能对断网提出质疑
   </strong>
   。许多国家的政府也正在效仿中国，率先将互联网审查过滤和控制确立为法律。
  </p>
  <p class="graf graf--p">
   工业界已经表明自己不愿意挑战北京等政府，包括苹果在内的公司，法庭上几乎没有成功的机会。
  </p>
  <p class="graf graf--p">
   Micek 说：“工商界可以在现有协会中团结起来 …… 并要求政府遏制其破坏性趋势。披露断网的财务成本是企业可以清楚地说明对决策者的不利经济影响的明确方法。”
  </p>
  <p class="graf graf--p">
   他补充说，公司还可以拒绝进行网络中断，而以法律、游说和政策工具挑战政府。
  </p>
  <p class="graf graf--p">
   2019年是互联网关闭成为主流问题的一年，无论从经济上还是政治上对他们的诉讼都是不可否认的。
  </p>
  <p class="graf graf--p">
   然而，这些事实证明，2020 年互联网自由倡导者和反审查组织将面临更加艰巨的任务，即 如何进行反击。⚪️
  </p>

   </a>
  </p>
