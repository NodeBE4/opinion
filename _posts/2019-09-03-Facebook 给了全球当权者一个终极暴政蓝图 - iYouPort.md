      </strong>

  </p>
  <p class="graf graf--p">
   也许 Facebook 最新举措的最危险的方面是将内容审核转移到了边缘，⚠️直接在用户的手机上运行他们的算法 —— 它为所有政府提供了全球监控网络的终极蓝图。
  </p>
  <p class="graf graf--p">
   通过推动政府将监视
   <strong class="markup--strong markup--p-strong">
    不仅
   </strong>
   视为一个需要识别目标、拦截通信并集中在难以想象的大型干草堆中寻觅绣花针的活跃过程，而且，
   <strong class="markup--strong markup--p-strong">
    要直接在用户自己的智能手机上运行思想警察
   </strong>
   ，Facebook 已经证明了所有人类的未来都处于监视状态。
  </p>
  <p class="graf graf--p">
   政府对技术变革的适应力无比强大，他们长期以来一直在通过利用最新的通信方式来监视其公民并收集所有感兴趣的情报。
  </p>
  <p class="graf graf--p">
   斯诺登在2013年的披露让所有人看到了这种监控的真实程度、以及它对我们生活的深入程度。
  </p>
  <p class="graf graf--p">
   在这里看到综合版《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" rel="noopener noreferrer" target="_blank">
    互联网霸主的时间线 — — 他们将互联网视为间谍、破坏和战争的主场
   </a>
   》。
  </p>
  <p class="graf graf--p">
   然而，斯诺登的披露还揭示了当前全球规模监控的有限性，即使有NSA的资源。
  </p>
  <p class="graf graf--p">
   互联网实在太大了，无法覆盖全球所有通信的全部内容，要想能够理解全世界所说和所见的一切，所需的处理资源甚至已经超出了美国情报机构（互联网霸主）的能力范围。
  </p>
  <p class="graf graf--p">
   但是，Facebook 来了。
  </p>
  <p class="graf graf--p">
   Facebook 推出的内容匹配和人工智能内容审核算法的新举措，
   <strong class="markup--strong markup--p-strong">
    为全球各国政府提供了一个强大的新模板 —— 用于监控自己的公民、和 Facebook 遍布全球的20亿用户
   </strong>
   。
  </p>
  <p class="graf graf--p">
   突然间，⚠️Facebook 不再需要亲自渗透用户的设备或拦截他们的传输通信并解密和处理数据了，Facebook 提醒我们，
   <strong class="markup--strong markup--p-strong">
    用户将*非常乐意*允许公司直接将过滤算法下载到他们的手机上、以处理他们的所有通信
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这不仅可以在加密传输之前以及在解密之后授予对通信的访问权限来回避对加密的争论，而且，还通过将每个用户的通信的处理载入到他们自己的设备来避免可扩展性的问题。
  </p>
  <p class="graf graf--p">
   虽然 Facebook 的噩梦仍处于研究阶段，但该公司的演示已经表明，该方法在完全于设备上执行大规模内容审核的可行性方面取得了相当大的进展。
  </p>
  <p class="graf graf--p">
   对于政府来说，Facebook 的模型提供了终极监控机器的伟大蓝图。
  </p>
  <p class="graf graf--p">
   ⚠️政府可能不需要再依赖 Facebook 泄漏数据了，而是要求在其土地上使用的所有手机都安装一个监控应用程序，对本地蜂窝提供商进行身份验证、同时还对所有通过设备的通信进行实时监控、窃听麦克风、偷窥相机 ……一个24/7便携式移动监控站。
  </p>
  <p class="graf graf--p">
   甚至，⚠️
   <strong class="markup--strong markup--p-strong">
    政府还可能要求设备制造商将此类监控过滤直接嵌入到设备上的专用固件中，以防止手机在没有它的情况下在本地网络上使用
   </strong>
   。
  </p>
  <p class="graf graf--p">
   从此以后，公众本身将采购和维护这种用于压制自身人权的监控网络，这可不是1984年那种固定的政府资助的电视屏幕。它完全超越了。
  </p>
  <p class="graf graf--p">
   想象一下这样一个世界：
   <strong class="markup--strong markup--p-strong">
    每个智能手机都会监控其所有者看到或说过的每个单词，在任何一个对政府不忠诚的言论出现时提醒当局
   </strong>
   。
  </p>
  <p class="graf graf--p">
   除此之外，每个手机的麦克风全天候开放，通过板载人工智能算法进行监控，识别附近听到的每一个声音，并记录听到的每一个字，让当局可以超越电话的拥有者，把其周边所有人都装进监视区。⚠️
   <strong class="markup--strong markup--p-strong">
    当麦克风四处窃听并抄写每个单词时，加密语音通话还有什么用？
   </strong>
  </p>
  <p class="graf graf--p">
   还要将此与 Facebook 能够映射其用户的实时物理位置并创建任何给定人口的统计或兴趣习惯的详细地图之能力相结合。
  </p>
  <p class="graf graf--p">
   结果可能是有史以来最强大的监控网络。没有之一了。
  </p>
  <p class="graf graf--p">
   当然，Facebook 本身是否帮助了政府当局焕发对监控潜力的灵感？
  </p>
  <p class="graf graf--p">
   正如你一直以来所熟知的，Facebook 扫描其用户的实时对话以标记对该公司的任何负面提及、并听取可能对其有用的所有情报、策划用手机摄像头上的灰尘跟踪你 ……相比下上述可能只是一个“小小的延伸”。
  </p>
  <p class="graf graf--p">
   毕竟，该公司在今年早些时候竭尽全力地提醒公众，它要保留将其GPS、相机和照片画廊*用于其认为合适的任何目的*的权利。
  </p>
  <p class="graf graf--p">
   然而，Facebook 的努力已经远远超出了监控范围。
  </p>
  <p class="graf graf--p">
   它们为那些希望噤声所有异议言论的国家提供了终极蓝图。为什么在用户发布批评之后会对用户进行追踪和惩罚？
  </p>
  <p class="graf graf--p">
   该公司没有回复评论请求。当人们询问如何让这种邪恶的设备扫描不被压制性政府重新利用的时候，⚠️Facebook 的沉默提醒我们，它唯一的重点是利润，而不是其20亿用户的安全和人权。
  </p>
  <p class="graf graf--p">
   综上所述，无论政府转向 Facebook外包其国内和全球监控需求，还是仅将 Facebook 的模式作为自己的监控计划蓝图，都没什么区别，
   <strong class="markup--strong markup--p-strong">
    Facebook 正在迅速引领我们所有人迈向一个崭新的世界 —— 在这里，隐私人权只是久远的过去中一个古怪的微弱记忆
   </strong>
   。
  </p>
