    </a>

  <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote">
   <p>
    <em>
     “你永远不可能仅仅通过对抗现有的现实来改变这一切。要改变某些东西，你必须建立一个新的模型，使现有模型过时。” —— Buckminster Fuller
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    让人们看到正在为止奋斗的新世界是什么样，展示它将如何发展；让这样的世界感觉不仅是可能的、而且是不可抗拒的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   我们中的许多人花了很多时间和精力去试图阻止不好的事情发生，但是，很少会有人花时间勾勒出事情会如何变得更好，更不用说真正走出去创造我们想要的未来的一小部分了。
  </p>
  <p class="graf graf--p">
   还记得我们在《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E5%A6%82%E4%BD%95%E8%99%9A%E6%9E%84%E4%B8%80%E4%B8%AA%E6%B0%91%E4%B8%BB%E5%AE%9E%E8%B7%B5%EF%BC%9F/" href="https://www.iyouport.org/%E5%A6%82%E4%BD%95%E8%99%9A%E6%9E%84%E4%B8%80%E4%B8%AA%E6%B0%91%E4%B8%BB%E5%AE%9E%E8%B7%B5%EF%BC%9F/" rel="noopener noreferrer" target="_blank">
    如何虚构一个民主实践？
   </a>
   》中所陈述的问题吗？就是那样。当下中国的创作者惯于描述封建宫廷、专制苦难、贫穷和无助 ……却不会描述民主社会，这也许是反对派难以获得支持的原因之一。
  </p>
  <p class="graf graf--p">
   这里指的预设性
   <strong class="markup--strong markup--p-strong">
    不仅
   </strong>
   是文艺作品。
  </p>
  <p class="graf graf--p">
   美国民权运动的午餐柜台静坐常常被称为反抗、勇敢和最终成功抵抗种族隔离的伟大行动。
  </p>
  <p class="graf graf--p">
   绝对如此，但是这场行动也是非常具有预设性的。行动者坐在“只限白人”的午餐柜台前，要求被服务，预示着胜利、并预示了他们想要生活的世界是什么样：他们正在实践他们想要的种族融合。
  </p>
  <p class="graf graf--p">
   恶作剧、艺术干预、战术媒体、替代节日和临时社区，甚至是游击剧院 …… 都可以成为预示我们想要生活的世界的有效方式。
  </p>
  <p class="graf graf--p">
   还记得我们此前推荐过的“小房运动”吗？如果您错过了，在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%8e%e5%b0%8f%e6%88%bf%e8%bf%90%e5%8a%a8%e4%b8%ad%e6%b1%b2%e5%8f%96%e7%bb%8f%e9%aa%8c/" href="https://www.iyouport.org/%e4%bb%8e%e5%b0%8f%e6%88%bf%e8%bf%90%e5%8a%a8%e4%b8%ad%e6%b1%b2%e5%8f%96%e7%bb%8f%e9%aa%8c/" rel="noopener noreferrer" target="_blank">
    从”小房”运动中汲取经验
   </a>
   》。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    预设性干预措施是指在假设点上的直接行动
   </strong>
   —— 信念的制定或废除，拉长可能的极限。
  </p>
  <p class="graf graf--p">
   💡预设性干预的目标有两个方面：提供一个可能的、更好的、未来的，以及引人注目的一瞥，以指出我们实际生活之世界的想象力贫乏。
  </p>
  <p class="graf graf--p">
   就像埃及的解放广场和 Indignados 运动对西班牙公共广场的占领一样，
   <strong class="markup--strong markup--p-strong">
    世界各地的占领营地都是预设性干预的模版 —— 为人们创造了一个空间，让他们在微观世界中创造他们想要的社区和民主世界
   </strong>
   。
  </p>
  <p class="graf graf--p">
   同样，Burning Man 艺术节作为一个*临时自治区*，人们可以在这里实现价值观、测试想法，并实时地展望未来。
  </p>
  <p class="graf graf--p">
   Oil Enforcement Agency 是一项2006年发起的戏剧性行动，其中环保活动家戴着与 SWAT 一样的帽子和徽章，提出了一个不存在的但应该存在的措施 —— 他们模拟了政府本应该认真应对气候变化的未来。
  </p>
  <p class="graf graf--p">
   如果希望确实是我们通过锻炼出肌肉所带来的，那么预设性干预是最好的努力方式之一 —— 💡无论是通过公民不服从的预设行动、替代社区的形成、还是恶作剧的挑衅。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    ⚠️关键原则 ——
   </strong>
  </p>
  <p class="graf graf--p">
   你可以继续谈论乌托邦，关于你梦想的更美好的世界，关于事情可能会有什么不同 ……直到徒劳无功，直到你自己都对自己产生怀疑。
  </p>
  <p class="graf graf--p">
   但是，你真正需要做的是
   <strong class="markup--strong markup--p-strong">
    去创造
   </strong>
   一种你所寻求的变化的生活体验 —— 是首先你信任它，而不是你希望劝说其他人都去信任它 —— 是打破玩世不恭，激发我们的政治想象力并确认这一点的最好方法。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “是的，另一个世界是可能的”。毕竟，我们无法创造一个我们尚未想象到的世界。如果我们已经尝过它，那将更容易实现它。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    ⚠️注意事项 ——
   </strong>
  </p>
  <p class="graf graf--p">
   切勿滑入深奥的幻想世界里。
  </p>
  <p class="graf graf--p">
   预设性干预并非是让你去描绘一幅充满彩虹和独角兽的迷人画面，而是提出
   <strong class="markup--strong markup--p-strong">
    一个有远见的、可取的、超越性的片段
   </strong>
   。
  </p>
  <p class="graf graf--p">
   总之，它必须有意义。
  </p>
  <p class="graf graf--p">
   目标是实现新的，而不仅仅是“证明旧的哪里有问题”，否则又会回去单纯对抗的老路。这点很重要。
  </p>
  <p class="graf graf--p">
   下面简单说说。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4300 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/干预点.png?resize=1100%2C715&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/干预点.png?w=1290&amp;ssl=1 1290w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/干预点.png?resize=300%2C195&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/干预点.png?resize=768%2C499&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/干预点.png?resize=1024%2C665&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/干预点.png?resize=1100%2C715&amp;ssl=1 1100w" data-recalc-dims="1" height="715" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/干预点.png?resize=1100%2C715&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    <em>
     干预点是系统内的物理或概念性场所，可以通过施加压力以破坏其顺利运转而推动变革。
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p">
   干预点是系统/体制中的特定位置，其中目标操作可以有效地中断系统的功能并打开改变的途径。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    通过了解这些不同的点，组织者可以制定策略，确定最佳干预点，以产生最大的影响
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️真正有效的干预措施不仅仅是破坏一个系统，而是，对其基本假设和基本合法性提出更深层次的挑战。
  </p>
  <p class="graf graf--p">
   无论干预是针对像血汗工厂这样的物理系统，还是像种族主义、性别歧视、或市场原教旨主义这样的意识形态系统，都是如此。
  </p>
  <p class="graf graf--p">
   干预点有五种类型：生产点（例如工厂），销毁点（例如伐木开路），消费点（例如零售店），决策点（例如公司总部），和假设点（基础叙事或具有象征意义的地方）。
  </p>
  <p class="graf graf--p">
   分头说。
  </p>
  <p class="graf graf--p">
   1、 生产点
  </p>
  <p class="graf graf--p">
   生产点的行动是劳工运动的基本见解。
  </p>
  <p class="graf graf--p">
   工人组织的目标是直接影响他们所处的经济体系和该体系最脆弱的地方。
  </p>
  <p class="graf graf--p">
   罢工、纠察线、工作减速和工厂接管，都是生产点行动。
  </p>
  <p class="graf graf--p">
   2、破坏点
  </p>
  <p class="graf graf--p">
   破坏点是实际发生伤害或不公正的地方。
  </p>
  <p class="graf graf--p">
   它可能是提取资源的地方（矿产）或废物倾倒的地方（垃圾填埋）。
  </p>
  <p class="graf graf--p">
   按照其设计，破坏点几乎总是远离公众的注意力 —— 由于偏远、压迫性的假设或因无知而无形 —— 并且往往会对已经边缘化的社区产生不成比例的影响。
  </p>
  <p class="graf graf--p">
   在破坏点进行的干预可以暂时阻止破坏行为，并将更大范围的冲突戏剧化。
  </p>
  <p class="graf graf--p">
   3、消费点
  </p>
  <p class="graf graf--p">
   消费点是与不公正相关的产品或服务的互动位置。消费点行动是消费者抵抗的传统舞台。
  </p>
  <p class="graf graf--p">
   消费点通常是针对商业实体的行动最明显的干预点。
  </p>
  <p class="graf graf--p">
   当立法者不听从民意时，消费点行动也是吸引大公司注意力的好方法。
  </p>
  <p class="graf graf--p">
   4、决策点
  </p>
  <p class="graf graf--p">
   这是不言而喻的。公司董事会、或国家首府、或国际峰会；许多成功的运动在决定向关键决策者施加压力时都在此采取了某种形式的行动。
  </p>
  <p class="graf graf--p">
   5、假设点
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如上所述，假设是意识形态的基石，是政治信仰体系的 DNA。当它们可以保持未经审查时，它们会处于最佳运作状态
   </strong>
   。
  </p>
  <p class="graf graf--p">
   💡如果基本假设可以暴露出与人们的生活经验或核心价值观相悖的东西，那么整个信念系统就可以被转移。
  </p>
  <p class="graf graf--p">
   揭露和瞄准大众广泛持有的假设的行动，可以非常有效地围绕一个问题转变语话，并开辟新的政治空间。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    假设点行为可以采取许多不同的形式，例如暴露虚伪、重新解决问题、放大故事中先前被噤声者的声音，或提供替代愿景
   </strong>
   。
  </p>
  <p class="graf graf--p">
   💡将创意行动转变为真正的变革需要谨慎的策略。以目标为导向确定干预点是帮助设计与大型活动和社会变革目标相关联的行动的第一步。⚪️
  </p>
