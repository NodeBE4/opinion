  <p class="graf graf--p">
   以上工具稍微专业，适合有至少一年以上开源调查经验的玩家。如果您是新手，正在考虑从哪里开始，本文推荐的一些小东西应该可以适合您。
  </p>
  <p class="graf graf--p">
   这些工具是欧洲记者常用的；如果比喻一下的话，它们就像是开瓶器、镊子、指甲钳等挂在钥匙扣上的小玩意。基本能满足日常需求，如果您准备深度追踪一个目标人/事件的时候，可能依旧需要更专业的工具。
  </p>
  <p class="graf graf--p">
   ⚠️请注意，和所有调查工具一样，以下这些工具也有危险性，如果落入恶意者手中会产生很糟糕的结果；
  </p>
  <p class="graf graf--p">
   💡于是，即便您不是调查人员，也可以使用在这些工具
   <strong class="markup--strong markup--p-strong">
    对自己进行“调查”
   </strong>
   ，并在恶意者出手之前删除您的关键信息和踪迹。
  </p>
  <p class="graf graf--p">
   那么开始。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://hunter.io/" href="https://hunter.io/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Hunter.io
    </strong>
   </a>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    查找调查对象的联系方式
   </strong>
   是调查工作的重头戏。
  </p>
  <p class="graf graf--p">
   Hunter.io 能帮你挖出某些带特定域名的邮箱地址，比如说政府部门的邮箱。
  </p>
  <p class="graf graf--p">
   即使有时不能给出准确地址，Hunter 也会利用他们的算法提供最接近的猜测结果（例如，“{last name}{first name}@company.com”）。
  </p>
  <p class="graf graf--p">
   接着，你可以用
   <a class="markup--anchor markup--p-anchor" data-href="https://hunter.io/verify" href="https://hunter.io/verify" rel="noopener noreferrer" target="_blank">
    Email Verifier
   </a>
   来测试所得的邮箱是否是真实存在的邮箱。
  </p>
  <p class="graf graf--p">
   还可以在 Chrome 上安装
   <a class="markup--anchor markup--p-anchor" data-href="https://chrome.google.com/webstore/detail/hunter/hgmhmanijnjhaffoampdlllchpolkdnj" href="https://chrome.google.com/webstore/detail/hunter/hgmhmanijnjhaffoampdlllchpolkdnj" rel="noopener noreferrer" target="_blank">
    Hunter 的插件
   </a>
   ，这样就能在领英个人页面添加一个“查找他们的邮箱地址”（Find their email）功能。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    IntelTechniques
   </strong>
  </p>
  <p class="graf graf--p">
   IntelTechniques 能让你同时搜索数十个社交媒体和网页平台，
   <strong class="markup--strong markup--p-strong">
    “人肉”某个信源、追踪目标人物、甚至是起底某人在网络世界的一举一动
   </strong>
   。
  </p>
  <p class="graf graf--p">
   比如，在 Facebook 搜索方面，它能帮你查到某个用户点过赞的所有照片，或是所有标记了他们的视频。
  </p>
  <p class="graf graf--p">
   目前该工具的提供者已经关闭了工具箱，因为他们受到了攻击，您可以在这里看到说明：

  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://www.invid-project.eu/" href="http://www.invid-project.eu/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     InVID
    </strong>
   </a>
  </p>
  <p class="graf graf--p">
   InVID 浏览器插件是由一群欧洲记者和产品设计师共同设计的
   <strong class="markup--strong markup--p-strong">
    视频核查
   </strong>
   工具。
  </p>
  <p class="graf graf--p">
   它能够挖出视频的缩略图、某一帧画面、元数据和其他信息，来帮你判断视频的来源。
  </p>
  <p class="graf graf--p">
   不过局限性是，目前它只能用于核查 YouTube 和 Facebook 视频。

  <p class="graf graf--p">
   如果视频本身没有什么特别值得注意的地方，你可以利用反向图片搜索（Reverse Image Search）来查询视频生成的缩略图。
  </p>
  <p class="graf graf--p">

  </p>

   </strong>

    </strong>

  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-7281 jetpack-lazy-image" data-lazy-sizes="(max-width: 784px) 100vw, 784px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-7.png?resize=784%2C436&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-7.png?w=784&amp;ssl=1 784w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-7.png?resize=300%2C167&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-7.png?resize=768%2C427&amp;ssl=1 768w" data-recalc-dims="1" height="436" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-7.png?resize=784%2C436&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="784"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-7281" data-recalc-dims="1" height="436" sizes="(max-width: 784px) 100vw, 784px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-7.png?resize=784%2C436&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-7.png?w=784&amp;ssl=1 784w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-7.png?resize=300%2C167&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-7.png?resize=768%2C427&amp;ssl=1 768w" width="784"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://botometer.iuni.iu.edu/#!/" href="https://botometer.iuni.iu.edu/#!/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Botometer
    </strong>
   </a>
  </p>
  <p class="graf graf--p">
   网络水军盛行的时代，要避免在判断上出错，还需懂得
   <strong class="markup--strong markup--p-strong">
    辨认社交媒体用户的真实性
   </strong>
   。
  </p>
  <p class="graf graf--p">
   由美国大学研究者开发的 Botometer 能帮你检测某个 Twitter 账户的背后是真人还是机器操作。
  </p>
  <p class="graf graf--p">
   虽然不能保证100%的准确率，但它已是市面上最好的“机器人检测仪”，还能用来分辨真粉还是僵尸粉。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://app.burnerapp.com/" href="https://app.burnerapp.com/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Burner
    </strong>
   </a>
  </p>
  <p class="graf graf--p">
   调查记者还要特别重视个人隐私和安全问题。
  </p>
  <p class="graf graf--p">
   不论信息加密工作做得多好，你总会在网络上 — — 通常在元数据中 — — 留下个人痕迹。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://app.burnerapp.com/" href="https://app.burnerapp.com/" rel="noopener noreferrer" target="_blank">
    Burner
   </a>
   的作用是给你创建一个带加拿大或美国区号的“电话号码”（从而
   <strong class="markup--strong markup--p-strong">
    隐去了你的真实痕迹
   </strong>
   ）。
  </p>
  <p class="graf graf--p">
   不过局限是暂时不能打国际电话。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://littlesis.org/oligrapher" href="https://littlesis.org/oligrapher" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Oligrapher
    </strong>
   </a>
  </p>
  <p class="graf graf--p">
   如其名所示，
   <a class="markup--anchor markup--p-anchor" data-href="https://littlesis.org/oligrapher" href="https://littlesis.org/oligrapher" rel="noopener noreferrer" target="_blank">
    Oligrapher
   </a>
   是一个可视化工具，专门用来设计呈现
   <strong class="markup--strong markup--p-strong">
    寡头政治家之间的关联的可视化图表
   </strong>
   。
  </p>
  <p class="graf graf--p">
   它让读者使用
   <a class="markup--anchor markup--p-anchor" data-href="https://littlesis.org/" href="https://littlesis.org/" rel="noopener noreferrer" target="_blank">
    Little Sis
   </a>
   （意思是“小姐姐”，取“老大哥”的反义）数据库的数据作图，后者是一个草根监督网络，包含数百万条有关世界上的政治强人和组织实体之间的社会关联信息。
  </p>
  <p class="graf graf--p">
   如果在 Little Sis 数据库上搜不到你想找的人或公司，你还可以主动给数据库添加信息。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-7282 jetpack-lazy-image" data-lazy-sizes="(max-width: 1074px) 100vw, 1074px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-6.png?resize=1074%2C607&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-6.png?w=1074&amp;ssl=1 1074w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-6.png?resize=300%2C170&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-6.png?resize=768%2C434&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-6.png?resize=1024%2C579&amp;ssl=1 1024w" data-recalc-dims="1" height="607" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-6.png?resize=1074%2C607&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1074"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-7282" data-recalc-dims="1" height="607" sizes="(max-width: 1074px) 100vw, 1074px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-6.png?resize=1074%2C607&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-6.png?w=1074&amp;ssl=1 1074w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-6.png?resize=300%2C170&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-6.png?resize=768%2C434&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-6.png?resize=1024%2C579&amp;ssl=1 1024w" width="1074"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://rawgraphs.io/" href="http://rawgraphs.io/" rel="noopener noreferrer" target="_blank">
    Raw Graphs
   </a>
  </p>
  <p class="graf graf--p">
   Raw Graphs 是一个开源数据可视化框架，目的是帮助用户处理并可视化复杂的数据。
  </p>
  <p class="graf graf--p">
   用户只需复制、粘贴数据表格到该网页应用上，应用便会自动提供可视化方案，帮助分析并以可视化的形式标出所上传的数据集特点，最后生成一个可视化样品。
  </p>
  <p class="graf graf--p">
   不满意的话，还可以重新进行分析，尝试其他可视化形式。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://datavizcatalogue.com/index.html" href="https://datavizcatalogue.com/index.html" rel="noopener noreferrer" target="_blank">
    Data Visualization Catalogue
   </a>
  </p>
  <p class="graf graf--p">
   寻找合适的数据图表和可视化方案还可以求助
   <a class="markup--anchor markup--p-anchor" data-href="https://cn.gijn.org/2016/12/21/%25E6%25AF%258F%25E5%2591%25A8%25E6%2595%25B0%25E6%258D%25AE%25E6%2596%25B0%25E9%2597%25BB%25E7%25B2%25BE%25E9%2580%2589-50%25E5%25A4%259A%25E7%25A7%258D%25E5%259B%25BE%25E8%25A1%25A8%25E6%2580%258E%25E4%25B9%2588%25E7%2594%25A8%25EF%25BC%259F%25E6%2595%25B0%25E6%258D%25AE%25E5%258F%25AF%25E8%25A7%2586%25E5%258C%2596/" href="https://cn.gijn.org/2016/12/21/%25E6%25AF%258F%25E5%2591%25A8%25E6%2595%25B0%25E6%258D%25AE%25E6%2596%25B0%25E9%2597%25BB%25E7%25B2%25BE%25E9%2580%2589-50%25E5%25A4%259A%25E7%25A7%258D%25E5%259B%25BE%25E8%25A1%25A8%25E6%2580%258E%25E4%25B9%2588%25E7%2594%25A8%25EF%25BC%259F%25E6%2595%25B0%25E6%258D%25AE%25E5%258F%25AF%25E8%25A7%2586%25E5%258C%2596/" rel="noopener noreferrer" target="_blank">
    “数据可视化目录”
   </a>
   。
  </p>
  <p class="graf graf--p">
   这个由数据可视化爱好者、平面设计师 Severino Ribecca 建立的网站是一个持续进行的项目，现已收录了50多种可视化图表的制作工具、应用案例等信息，供用户检索参考。
  </p>
  <p class="graf graf--p">
   该网站去年还新增了
   <a class="markup--anchor markup--p-anchor" data-href="https://datavizcatalogue.com/ZH/" href="https://datavizcatalogue.com/ZH/" rel="noopener noreferrer" target="_blank">
    中文版
   </a>
   。
  </p>
  <p class="graf graf--p">
   好了。您在做调查的过程中还有其他经常使用的趁手工具吗？欢迎留言分享。⚪️
  </p>
