  <p class="graf graf--p">
   兰德公司机密报告的结论最近在“Brief”中被公布。
  </p>
  <p class="graf graf--p">
   这份机密报告解释了如何对俄罗斯发动新的冷战。显然，其中某些建议已经开始实施，但是这份系统性的曝光可以帮助人们更容易理解真正的目标和整体局势。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “迫使对手不顾一切地扩张以使他失去平衡，然后摧毁他。”
  </p>
  <p class="graf graf--p">
   这不是对柔道技巧的描述，而是兰德公司制定的针对俄罗斯的计划。兰德公司是美国最有影响力的智囊团。
  </p>
  <p class="graf graf--p">
   兰德拥有数千名专家，是世界上最可靠的情报和政治分析来源，对于美国及其盟国领导人来说。
  </p>
  <p class="graf graf--p">
   兰德公司自豪于通过迫使苏联在战略对抗中消耗自己的经济资源，制定了使美国赢得冷战的长期战略。
  </p>
  <p class="graf graf--p">
   这个模型正是兰德出版的新计划“
   <em class="markup--em markup--p-em">
    Overextending and Unbalancing Russia
   </em>
   ”的灵感来源。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4594 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-7.png?resize=1100%2C920&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-7.png?w=1458&amp;ssl=1 1458w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-7.png?resize=300%2C251&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-7.png?resize=768%2C643&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-7.png?resize=1024%2C857&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-7.png?resize=1100%2C920&amp;ssl=1 1100w" data-recalc-dims="1" height="920" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/2-7.png?resize=1100%2C920&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   据他们的分析人士称，俄罗斯“仍然是”美国在某些基本领域的强大对手。
  </p>
  <p class="graf graf--p">
   为了应对这种对立，美国及其盟国将不得不采取一项利用俄罗斯脆弱性的联合长期战略。
  </p>
  <p class="graf graf--p">
   因此，兰德分析了各种平衡俄罗斯的手段，并指出了每一种手段成功的可能性、利益、成本以及对美国的风险。
  </p>
  <p class="graf graf--p">
   兰德分析师估计，俄罗斯最大的脆弱性是其经济，因为它严重依赖石油和天然气出口。
  </p>
  <p class="graf graf--p">
   通过加强制裁和增加美国的能源出口，可以减少俄罗斯这些出口的收入。
  </p>
  <p class="graf graf--p">
   其目标是迫使欧洲减少对俄罗斯天然气的进口，并用来自其他国家的海运液化天然气取而代之。
  </p>
  <p class="graf graf--p">
   从长远来看，另一种破坏俄罗斯经济稳定的方法是鼓励“合格人员”，特别是受过高等教育的年轻俄罗斯人移民。
  </p>
  <p class="graf graf--p">
   研究指出，在意识形态和信息部门，有必要鼓励内部争议，同时破坏俄罗斯在外部的形象，将其排除在国际论坛之外，抵制其组织的国际体育赛事。
  </p>
  <p class="graf graf--p">
   在地缘政治领域，武装乌克兰将使美国能够利用俄罗斯外部脆弱性的中心点，但必须仔细计算这一点，以便让俄罗斯承受压力而不会陷入重大冲突，这样才能获得胜利。
  </p>
  <p class="graf graf--p">
   在军事部门，通过增加从事反俄职能的北约国家地面部队人数，美国可以享受低成本和低风险的高收益。
  </p>

  <p class="graf graf--p">
   研究认为，美国可以享有很高的成功概率和高效益，风险适中，特别是主要投资于针对俄罗斯的战略轰炸机和远程攻击导弹。
  </p>
  <p class="graf graf--p">
   退出中导条约并在欧洲部署新的中程核导弹指向俄罗斯将具有很高的成功概率，但也会带来高风险。
  </p>
  <p class="graf graf--p">
   通过校准每个选项以获得预期的效果 —— 兰德分析师得出结论 —— 俄罗斯最终会在对抗中付出最大的代价，但美国也将不得不投入大量资源，因此不再可用于其他目标。
  </p>
  <p class="graf graf--p">
   这也预示着美国/北约军费开支将大幅增加，不利于社会预算……
  </p>
  <p class="graf graf--p">
   这是兰德公司规划的未来。兰德公司是最有影响力的智库，换句话说，是由经济、金融和军事寡头统治的地下实体中心 —— 这决定了以上战略选择不仅是美国的，还有整个西方世界。
