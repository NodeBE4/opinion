  <p class="graf graf--p">
   由美国科技巨头谷歌和 IBM 创立的美国组织正在与一家帮助中国威权政府对其公民进行
   <strong class="markup--strong markup--p-strong">
    大规模监视
   </strong>
   的巨头公司密切合作。
  </p>
  <p class="graf graf--p">
   OpenPower 基金会是由谷歌和 IBM 高管领导的、自称旨在“推动创新”的非盈利组织 —— 该基金会已经与 IBM、中国最臭名昭著的监视技术公司之一 Semptian、以及美国芯片制造商 Xilinx 建立了深刻的合作关系。
  </p>
  <p class="graf graf--p">
   他们一直在共同努力推动一系列微处理器的发展，使计算机能够更有效地分析大量数据 —— 这对监视技术的升级来说非常重要。
  </p>
  <p class="graf graf--p">
   据消息来源和内部文件显示，⚠️总部位于中国深圳的 Semptian 正在利用这些设备增强其为中国侵犯人权的维稳机构提供的互联网监控和审查技术的能力。
  </p>
  <p class="graf graf--p">
   Intercept 引述的一名该公司员工表示，⚠️该公司的技术被用于秘密监控2亿人的互联网活动。
  </p>
  <p class="graf graf--p">
   Semptian、Google 和 Xilinx 都没有回应置评请求。
  </p>
  <p class="graf graf--p">
   OpenPower 基金会在一份声明中表示，由于反托拉斯法和竞争法，它“不会参与或寻求了解其成员的个人业务战略、目标或活动”。
  </p>
  <p class="graf graf--p">
   一位 IBM 发言人表示，他的公司“尚未与 Semptian 就联合技术开发合作”，但拒绝回答进一步的问题。
  </p>
  <p class="graf graf--p">
   熟悉 Semptian 运营的消息人士表示，⚠️Semptian 曾通过名为
   <a class="markup--anchor markup--p-anchor" data-href="https://www.research.ibm.com/labs/china/supervessel.html" href="https://www.research.ibm.com/labs/china/supervessel.html" rel="noopener noreferrer" target="_blank">
    SuperVessel
   </a>
   的协作云平台与IBM合作，
   <strong class="markup--strong markup--p-strong">
    该平台由IBM在中国的研究部门维护
   </strong>
   。
  </p>
  <p class="graf graf--p">
   参议院情报委员会副主席 D-Va 参议员 Mark Warner 告诉 The Intercept，他对这些揭露感到震惊。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “看到
   <strong class="markup--strong markup--p-strong">
    中国成功招募西方巨头公司和高科技研究人员协助他们进行信息监控工作
   </strong>
   ，这令人不安，” Warner 说。
  </p>
  <p class="graf graf--p">
   大赦国际的研究员 Anna Bacciarelli 表示，OpenPower 基金会与 Semptian 合作的决定引发了对其遵守国际人权标准的质疑。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “所有公司都有责任在整个运营和供应链中进行人权尽职调查，” Bacciarelli 说，“包括通过合作伙伴关系。”
  </p>
  <p class="graf graf--p">
   Semptian 公开表示自己是一家与互联网提供商和教育机构合作的“大数据”分析公司。
  </p>
  <p class="graf graf--p">
   然而，这家
   <strong class="markup--strong markup--p-strong">
    中国公司业务的很大一部分实际上是通过一家名为 iNext 的前线公司产生的，该公司专门向政府出售互联网监控和审查工具
   </strong>
   。
  </p>
  <p class="graf graf--p">
   iNext 在中国的办公室与 Semptian 一样，两家公司都在深圳繁忙的南山区的八楼。
   <strong class="markup--strong markup--p-strong">
    Semptian 和 iNext 也共享了200名员工和同一创始人陈龙森
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在收到关于 Semptian 在大规模监控中的角色的机密消息来源的提示后，一名记者使用假名字与该公司联系，并冒充潜在客户。
  </p>
  <p class="graf graf--p">
   作为回应，一名 Semptian 员工发送的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/6188454-aegis1.html" href="https://www.documentcloud.org/documents/6188454-aegis1.html" rel="noopener noreferrer" target="_blank">
    文件显示
   </a>
   ，⚠️该公司 — 以 iNext 为幌子 — 开发了一个名为 Aegis 的大规模监控系统，据称可以“存储和分析无限数据”。
  </p>
  <p class="graf graf--p">
   该公司在文件中声称，⚠️
   <strong class="markup--strong markup--p-strong">
    这个“Aegis”可以提供“虚拟世界的全景图”，允许政府间谍看到“每个人之间的联系”，包括“中国所有人的地理位置信息”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   根据熟悉 Semptian 工作的两位消息人士透露，⚠️Aegis 监视设备已被置于中国的电话和互联网中，使该国政府能够秘密收集人们的电子邮件记录、电话、短信、手机位置和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%b5%8f%e8%a7%88%e5%8e%86%e5%8f%b2%e8%ae%b0%e5%bd%95%e6%9b%9d%e5%85%89%e4%bd%a0%e7%9a%84%e5%86%85%e5%bf%83%e5%85%a8%e6%99%af%e5%9b%be%ef%bc%9aosint-%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e6%bc%94/" href="https://www.iyouport.org/%e6%b5%8f%e8%a7%88%e5%8e%86%e5%8f%b2%e8%ae%b0%e5%bd%95%e6%9b%9d%e5%85%89%e4%bd%a0%e7%9a%84%e5%86%85%e5%bf%83%e5%85%a8%e6%99%af%e5%9b%be%ef%bc%9aosint-%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5%e6%bc%94/" rel="noopener noreferrer" target="_blank">
    网页浏览历史
   </a>
   。
  </p>
  <p class="graf graf--p">
   ⚠️中国国安机构很可能会利用这项技术瞄准人权活动家、支持民主的倡导者以及任何习近平政权的批评者：消息人士称，由于害怕报复而不愿透露姓名。
  </p>
  <p class="graf graf--p">
   在电子邮件中，一位 Semptian 公司代表表示，⚠️该公司的 Aegis 大规模监控系统正在中国处理大量个人数据。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “Aegis 的能力是无限的，我们在中国处理超过2亿人的数据，每秒数千兆比特，” Semptian 员工 Zhu Wenying 在四月份的一则消息中写道。
  </p>
  <p class="graf graf--p">
   中国估计有8亿互联网用户，这意味着如果 Zhu 的数字准确，Semptian 的技术正在监视该国四分之一的在线人口。
  </p>
  <p class="graf graf--p">
   牛津大学互联网研究所的高级研究员 Joss Wright 说：“世界上没有多少系统可以达到这种监视范围。”
  </p>
  <p class="graf graf--p">
   Wright 说，Semptian 有可能夸大其数据。
  </p>
  <p class="graf graf--p">
   然而，他补充说，
   <strong class="markup--strong markup--p-strong">
    一个能够利用如此大量数据的系统在技术上是可行的。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这是一项巨大的监视工程”。
  </p>
  <p class="graf graf--p">
   熟悉 Semptian 在中国工作的两位消息人士表示，该公司的设备不会随意抽取并随机存储数百万人的数据。
  </p>
  <p class="graf graf--p">
   相反，该消息人士称，⚠️该设备可以监视任何通过电话和互联网网络的通信，并且它可以过滤掉与特定的单词、短语或人物相关的信息。
  </p>
  <p class="graf graf--p">
   为了回应有关该监视工具 Aegis 如何运作的更多细节的视频请求，Zhu 先生同意发送一个视频，只要卧底记者签署保密协议。
  </p>
  <p class="graf graf--p">
   Intercept 公布了一段16分钟的该视频节选，
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2019/07/11/china-surveillance-google-ibm-semptian/?comments=1#comments" href="https://theintercept.com/2019/07/11/china-surveillance-google-ibm-semptian/?comments=1#comments" rel="noopener noreferrer" target="_blank">
    在这里看到
   </a>
   ，因为其内容具有压倒性的公众价值 —— 这表明中国数百万人正在受到政府监视。Intercept 删除了可能侵犯个人隐私的信息。
  </p>
  <p class="graf graf--p">
   Semptian 的视频演示展示了监视工具 Aegis 系统如何跟踪人们的一举一动。
  </p>
  <p class="graf graf--p">
   ⚠️只要政府官员输入一个人的手机号码，Aegis 就可以显示该设备在给定时间段内的具体位置：过去三天、最后一周、最后一个月或更长时间。
  </p>
  <p class="graf graf--p">
   该视频显示了中国大陆的完整地图，并演示了如何跟踪在深圳的一个人穿越城市，从机场、公园和花园、到会议中心、到酒店、以及制药公司的办公室。
  </p>
  <p class="graf graf--p">
   ⚠️该技术还可以允许政府搜索特定的即时消息、电子邮件地址、社交媒体帐户、论坛用户、博客评论者、或其他标识符 —— 如手机 IMSI 或计算机 MAC 地址，一系列独特的数字与每个设备直接相关。
  </p>
  <p class="graf graf--p">
   ⚠️在许多情况下，该监视系统似乎可以收集通信的全部内容，例如电话录音、或文本消息的书面正文，而不仅仅是元数据，而且，可以显示电子邮件的发件人和收件人、谁在打电话给谁的电话号码。
  </p>
  <p class="graf graf--p">
   ⚠️该监视系统是否可以访问消息的完整内容，可能取决于消息本身是否受到了强加密保护。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">

     安全手册
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Semptian 的员工 Zhu 在电子邮件中写道，
   <strong class="markup--strong markup--p-strong">
    该公司为政府提供 Aegis 装置，能够监控500万人的互联网活动，费用在150万美元到250万美元之间。要窃听其他通信，成本会增加
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果添加监视电话、短信、地理位置的功能，根据网络的不同，将增加200万至500万美元。”
  </p>
  <p class="graf graf--p">
   2015年9月，Semptian 加入了 OpenPower 基金会，⚠️这是一家由科技巨头谷歌和IBM创立的美国非营利组织。
   <strong class="markup--strong markup--p-strong">
    该基金会
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://openpowerfoundation.org/about-us/board-of-directors/" href="https://openpowerfoundation.org/about-us/board-of-directors/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     现任总裁
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    是 IBM 的 Michelle Rankin，其主管是 Google 的 Chris Johnson。
   </strong>
  </p>
  <p class="graf graf--p">
   该基金会在新泽西州注册为“社区改善”组织，其目标是分享网络、服务器、数据存储和数据处理技术的进步。
  </p>
  <p class="graf graf--p">
   据
   <a class="markup--anchor markup--p-anchor" data-href="https://openpowerfoundation.org/" href="https://openpowerfoundation.org/" rel="noopener noreferrer" target="_blank">
    其网站
   </a>
   称，它希望“让今天的数据中心重新思考他们的技术方法”，以及“推动创新并在行业中提供更多选择”。
  </p>
  <p class="graf graf--p">
   ⚠️这个中国的监视巨头 Semptian 受益于与美国科技寡头公司的合作，获得了丰富的专业知识和新技术。
  </p>
  <p class="graf graf--p">
   这家中国监视公司在其网站上称自己 “积极地与 IBM 和 Xilinx 等世界级公司合作”；它声称，⚠️它是
   <strong class="markup--strong markup--p-strong">
    亚太地区唯一一家能够为其客户提供在这些美国公司的帮助下开发的新数据处理设备的公司
   </strong>
   。
  </p>
  <p class="graf graf--p">
   去年，OpenPower 基金会在其网站上表示，Semptian 正在与IBM、Xilinx和其他美国公司合作，为此
   <a class="markup--anchor markup--p-anchor" data-href="https://openpowerfoundation.org/2018-openpower-capi-and-opencapi-heterogeneous-computing-design-contest-build-your-own-super-processor/" href="https://openpowerfoundation.org/2018-openpower-capi-and-opencapi-heterogeneous-computing-design-contest-build-your-own-super-processor/" rel="noopener noreferrer" target="_blank">
    感到“高兴”
   </a>
   。
  </p>
  <p class="graf graf--p">
   ⚠️该基金会表示，它还“与中国的一些优秀大学和研究机构合作。”
  </p>
  <p class="graf graf--p">
   12月，OpenPower 的高管们在中国的北京、东城区的五星级喜来登大酒店，举办了一次峰会。
  </p>
  <p class="graf graf--p">
   一位消息人士告诉 The Intercept，
   <strong class="markup--strong markup--p-strong">
    Semptian 代表受邀参加了那次峰会。并向他们的美国同行展示了他们为“舆论监测”等目的而开发的新视频分析技术
   </strong>
   。
  </p>
  <p class="graf graf--p">
   目前还不清楚为什么美国科技巨头选择与 Semptian 合作；⚠️该决定可能已被视为与中国建立更紧密联系并进入这个东亚国家利润丰厚市场的更广泛战略的一部分。
  </p>
  <p class="graf graf--p">
   美国 OpenPower 基金会的发言人拒绝回答有关该组织与中国监视巨头 Semptian 合作的问题，只是说“通过基金会提供的技术是通用的，可在全球范围内商业化，并且
   <strong class="markup--strong markup--p-strong">
    不需要美国出口许可证
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnas.org/support-cnas/cnas-supporters" href="https://www.cnas.org/support-cnas/cnas-supporters" rel="noopener noreferrer" target="_blank">
    政策智囊团
   </a>
   新美国安全中心的兼职高级研究员 Elsa Kania 说，在某些情况下，美国和中国公司之间的商业伙伴关系和学术合作是重要的和有价值的，“但是当它是
   <strong class="markup--strong markup--p-strong">
    一家与审查或监视密切相关的公司、并且在滥用人权方面深刻同谋
   </strong>
   时，这事就非常令人担忧了”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我希望美国公司在参与之前有严格的道德审查程序，” Kania 说，“但是很多时候，
   <strong class="markup--strong markup--p-strong">
    利润高于道德
   </strong>
   ”。
  </p>
  <p class="graf graf--p">
   Semptian 成立于2003年，多年来一直是中国政府信赖的合作伙伴。
  </p>
  <p class="graf graf--p">
   中共政权授予该公司以 “国家高新技术企业”的地位，这意味着它通过了中国科技部的各种审查和审计。
  </p>
  <p class="graf graf--p">
   获得这种特殊地位的公司将获得政府赠予的优厚待遇，包括税收减免和其他支持。
  </p>
  <p class="graf graf--p">
   2011年，德国新闻杂志 Der Spiegel 就发表过
   <a class="markup--anchor markup--p-anchor" data-href="https://www.spiegel.de/netzwelt/netzpolitik/chinas-filter-dienstleister-pornos-diaeten-wir-koennen-alles-sperren-a-752941.html" href="https://www.spiegel.de/netzwelt/netzpolitik/chinas-filter-dienstleister-pornos-diaeten-wir-koennen-alles-sperren-a-752941.html" rel="noopener noreferrer" target="_blank">
    一篇文章
   </a>
   ，强调了 Semptian 与中国政府的密切关系。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    正是该监视公司帮助建立了中国所谓的防火墙GFW的一些方面
   </strong>
   ，这是一个强大的互联网审查制度，可以阻止共产党认为不受欢迎的任何网站，例如关于人权和民主的网站。
  </p>
  <p class="graf graf--p">
   Spiegel 当时就报道说，⚠️Semptian 的“网络控制技术正在中国一些主要城市使用” —— 请注意，那是2011年。
  </p>
  <p class="graf graf--p">
   到2013年，Semptian 已开始⚠️在全球推广其产品。该公司的代表前往欧洲，他们出现在巴黎东北部会议厅举行的安全（维稳）贸易展览会上。
  </p>
  <p class="graf graf--p">
   文件显示，在这一事件中，⚠️Semptian 通过展示“国家防火墙”向国际政府官员提供了复制中国互联网监视模式的机会，该公司表示：可以“阻止来自互联网的任何不良信息”。
  </p>
  <p class="graf graf--p">
   仅仅两年后，Semptian 在美国 OpenPower基金会的成员资格就获得了批准，
   <strong class="markup--strong markup--p-strong">
    该公司开始使用美国技术使其监控和审查系统变得更加强大
   </strong>
   。⚪️
  </p>
  <p class="graf graf--p">

    HOW U.S. TECH GIANTS ARE HELPING TO BUILD CHINA’S SURVEILLANCE STATE
   </a>
  </p>
