   </noscript>
  </p>
  <p class="graf graf--p">
   国际特赦组织的一份新报告中指出，美国的空袭正在杀害索马里境内的平民，报告反驳五角大楼一再声明“没有平民死亡”的谎言，即使美国的袭击至少在特朗普政府执政期间增加了两倍。
  </p>
  <p class="graf graf--p">
   大赦国际报告详述了导致14人死亡的五次空袭。
   <strong class="markup--strong markup--p-strong">
    该组织的法医调查发现了大量证据表明美国空袭导致的平民死亡案件
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这是一份详细，可信，令人深感不安的报道，”美国公民自由联盟国家安全项目主任 Hina Shamsi 在给 theintercept 的电子邮件中说。 “尽管调查条件非常具有挑战性，但国际特赦组织提供了强有力的证据，证明特朗普政府声称零平民伤亡是不可信的。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    大赦国际说，军方战术不分青红皂白的方面可能意味着一些袭击已经构成战争罪，即使美国在技术角度上没有对索马里宣战
   </strong>
   。当被问及美国是否在索马里开战时，AFRICOM 司令官 Thomas D. Waldhauser 将军在本月早些时候的国会听证会上说，“我不认为我们处于战争状态。“ 但大赦国际认为，
   <strong class="markup--strong markup--p-strong">
    美国的行为使其成为非国际性武装冲突的明确一方，因此，战争法将适用
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在特朗普2017年就职典礼几个月后，索马里的部分地区被宣布为“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2017/03/30/world/africa/trump-is-said-to-ease-combat-rules-in-somalia-designed-to-protect-civilians.html" href="https://www.nytimes.com/2017/03/30/world/africa/trump-is-said-to-ease-combat-rules-in-somalia-designed-to-protect-civilians.html" rel="noopener noreferrer" target="_blank">
    活跃敌对地区
   </a>
   ”，允许 AFRICOM 对“被认为是”激进组织青年党成员的任何人在该国进行更多的进攻性袭击，其攻击计划的机构间审查比以前要求的要少很多。
  </p>
  <p class="graf graf--p">
   2015年4月至2017年6月美国特种作战部队指挥官 Donald Bolduc 将军告诉大赦国际，他的理解是，通过新的指定，攻击定位基于四个标准的目标：年龄，性别，地点（在索马里的特定地区，该地区人口被视为“支持青年党”）以及与青年党的距离。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果 Bolduc 将军对目标标准的描述是正确的，
   <strong class="markup--strong markup--p-strong">
    那么美国就是故意杀害男性，仅仅是因为他们处于入伍年龄、并可能认识青年党成员，或者是人们正处于被认为对该团体表示同情的地区
   </strong>
   ，”Shamsi 解释说。
  </p>
  <p class="graf graf--p">
   AFRICOM 表示 Bolduc “没有准确地总结目标标准”，但拒绝提供额外信息，其借口是“运营安全原因”。“AFRICOM 仍致力于向索马里联邦政府提供援助，并对迄今为止取得的渐进进展感到鼓舞，尤其是在向青年党网络施加压力时，”一位发言人在致The Intercept的电子邮件中说。
  </p>
  <p class="graf graf--p">
   大赦国际报告2017年11月12日被杀害的三名平民 — 其中一人是13岁孩子的父亲，这些人在村庄的东南部的农田里工作，灌溉农田，以便在旱季种植农作物。男人们在晚间工作，这就是该社区的习惯，当他们在树下休息的时候被空袭杀死。村民们在早上发现了自己的尸体。
  </p>
  <p class="graf graf--p">
   大赦国际收集到18位证人一致的证词，他们说这些人都不是青年党的成员。大赦国际找不到任何暗示这些人参与过敌对行动的信息。
  </p>
  <p class="graf graf--p">
   AFRICOM 承认它在那个时间和地点进行了袭击，但坚称后续调查发现这些男子“没有睡觉并且是青年党的成员”。
  </p>
  <p class="graf graf--p">
   在大赦国际调查的另一起事件中，有三名男子驾驶汽车载着一名涉嫌青年党的人。车上的其他人为该国最大的电信公司 Hormuud Telecom 工作。一名高级地区官员告诉国际特赦组织，
   <strong class="markup--strong markup--p-strong">
    车内的大多数人都是平民。 “我不知道他们为什么被击中
   </strong>
   ，但也许这是一个错误，”他说。 “美国在这个地区犯了很多错误。”
  </p>
  <p class="graf graf--p">
   国际特赦组织武器和军事行动高级顾问 Brian Castner 对 theintercept 说：“据我们所知，
   <strong class="markup--strong markup--p-strong">
    AFRICOM 在地面调查方面没有做任何事
   </strong>
   ，我们采访过的150人中没有任何一个人曾与政府官员索马里或美国人谈过这些袭击事件。
  </p>
  <p class="graf graf--p">
   该组织称，
   <strong class="markup--strong markup--p-strong">
    在大赦国际研究期间至少又进行了76次其他袭击，表明实际的平民死亡人数可能会更高
   </strong>
   。 AFRICOM 的发言人告诉 theintercept，今年迄今为止已进行了27次空袭。
  </p>
  <p class="graf graf--p">
   索马里总统办公室没有回应对大赦国际报告发表评论的请求。
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    R
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2019/03/19/u-s-air-strikes-in-somalia-kill-civilians-in-wave-of-attacks-under-trump-amnesty-report-says/?utm_source=The+Intercept+Newsletter&amp;utm_campaign=e6b9360297-EMAIL_CAMPAIGN_2019_03_23&amp;utm_medium=email&amp;utm_term=0_e00a5122d3-e6b9360297-132064301" href="https://theintercept.com/2019/03/19/u-s-air-strikes-in-somalia-kill-civilians-in-wave-of-attacks-under-trump-amnesty-report-says/?utm_source=The+Intercept+Newsletter&amp;utm_campaign=e6b9360297-EMAIL_CAMPAIGN_2019_03_23&amp;utm_medium=email&amp;utm_term=0_e00a5122d3-e6b9360297-132064301" rel="noopener noreferrer" target="_blank">

   At least 76 other strikes were carried out over the period Amnesty researched, indicating that the civilian death toll could be much higher, the group says.
  </p>
