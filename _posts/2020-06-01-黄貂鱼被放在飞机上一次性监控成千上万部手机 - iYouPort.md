  <p class="graf graf--p">
   大家都知道黄貂鱼是什么。它伪装手机信号塔利用诱骗的诡计来监视附近所有连接到它的设备。更糟糕的是，政府间谍和警察一直把黄貂鱼放在汽车和货车里，满街跑，万一你走在路上，或者车子开过你家门口 …… 你就中招了。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     这很重要
    </strong>
    👉《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ba%94%e4%bb%98%e6%94%bf%e5%ba%9c%e4%bd%bf%e7%94%a8%e9%bb%84%e8%b2%82%e9%b1%bc%e5%af%b9%e5%85%ac%e6%b0%91%e8%bf%9b%e8%a1%8c%e7%9a%84%e5%a4%a7%e8%a7%84%e6%a8%a1%e9%97%b4%e8%b0%8d/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ba%94%e4%bb%98%e6%94%bf%e5%ba%9c%e4%bd%bf%e7%94%a8%e9%bb%84%e8%b2%82%e9%b1%bc%e5%af%b9%e5%85%ac%e6%b0%91%e8%bf%9b%e8%a1%8c%e7%9a%84%e5%a4%a7%e8%a7%84%e6%a8%a1%e9%97%b4%e8%b0%8d/" rel="noopener noreferrer" target="_blank">
     如何应付政府使用黄貂鱼对公民进行的大规模间谍行为
    </a>
    ？》
   </li>
  </ul>
  <p class="graf graf--p">
   但是现在，上述举动已经不会让任何人惊讶，因为，
   <strong class="markup--strong markup--p-strong">
    它正在扩大其追踪范围，这也使它能够一次性收集更多人的数据。
   </strong>
  </p>
  <p class="graf graf--p">
   根据《华尔街日报》的
   <a class="markup--anchor markup--p-anchor" data-href="http://online.wsj.com/news/article_email/americans-cellphones-targeted-in-secret-u-s-spy-program-1415917533-lMyQjAxMTI0NTEwNDAxMTQwWj" href="https://online.wsj.com/news/article_email/americans-cellphones-targeted-in-secret-u-s-spy-program-1415917533-lMyQjAxMTI0NTEwNDAxMTQwWj" rel="noopener noreferrer" target="_blank">
    新报告
   </a>
   ，政府一直在使用装有 *特殊电话监视设备* 的塞斯纳飞机来追踪他们想要的任何人。
  </p>
  <p class="graf graf--p">
   ⚠️请记得，监控系统的设计目的是拾取覆盖范围内**任何人**的手机信号，而不仅仅是当局想要的人。
  </p>
  <p class="graf graf--p">
   该监视设备的准确覆盖范围目前未知，但是这意味着在一次飞行中可能会收集成千上万部手机的数据。
  </p>
  <p class="graf graf--p">
   这种装在飞机上的监视系统是一个名为 Dirtbox 的2平方英尺的盒子 —— 与警察、军事和间谍机构
   <strong class="markup--strong markup--p-strong">
    已经使用了十多年
   </strong>
   的黄貂鱼基本相同或相似。
  </p>
  <p class="graf graf--p">
   这项秘密的黄貂鱼技术使警察可以伪装成合法的手机信号塔，以欺骗附近的所有手机和其他无线通信设备（例如无线网卡）连接到该黄貂鱼。于是它可以看到并记录其唯一的ID号以及指向该设备位置的信息。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     并不在意料之外
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   警察条子经常在汽车和货车上部署黄貂鱼。然后开着它满街跑。通过黄貂鱼从附近的各个位置收集的无线设备信号强度，当局
   <strong class="markup--strong markup--p-strong">
    可以精确地确定
   </strong>
   使用该设备的人的位置，而不是通过从移动网络提供商那里获取数据。
  </p>
  <p class="graf graf--p">
   定位后，特工就可以切换到
   <strong class="markup--strong markup--p-strong">
    手持设备
   </strong>
   ，该设备以相同的方式操作，但允许政府间谍自由移动以确定目标所在的确切公寓或办公室位置。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    不要忘了，
    <strong class="markup--strong markup--li-strong">
     美国技术巨头帮助中国构建了一种更厉害的便携式监视设备，政府间谍可以直接把它放在口袋里徒步追踪所有人
    </strong>
    ，见《
    <a class="markup--anchor markup--li-anchor" data-href="http://IBM和谷歌如何通过中国协助独裁政权的间谍行为？" href="http://IBM和谷歌如何通过中国协助独裁政权的间谍行为%EF%BC%9F" rel="noopener noreferrer" target="_blank">
     IBM和谷歌如何通过中国协助独裁政权的间谍行为
    </a>
    ？》
   </li>
  </ul>
  <p class="graf graf--p">
   但是，这种监视方法的主要问题之一是，这些监视设备迫使一个较大区域中的
   <strong class="markup--strong markup--p-strong">
    每一部手机
   </strong>
   都必须与其连接。因此，如果政府的黄貂鱼驶过您的办公室或住屋，你的所有设备就已经被捕获。
  </p>
  <p class="graf graf--p">
   现在是，
   <strong class="markup--strong markup--p-strong">
    政府间谍把这东西放在飞机上，更大范围地撒网
   </strong>
   。
  </p>
  <p class="graf graf--p">
   我们的《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e7%9f%a5%e9%81%93%e8%87%aa%e5%b7%b1%e6%ad%a3%e5%9c%a8%e8%a2%ab%e7%9b%91%e8%a7%86%ef%bc%9f/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e7%9f%a5%e9%81%93%e8%87%aa%e5%b7%b1%e6%ad%a3%e5%9c%a8%e8%a2%ab%e7%9b%91%e8%a7%86%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    如何发觉暗中监视
   </a>
   》的支招中强调了那种奇怪的汽车您应该警惕；但是这份报道显示，飞机飞过头顶也是应该小心的……
  </p>
  <p class="graf graf--p">
   对政府间谍来说，空中监视显然比通过货车或汽车跟踪要厉害得多，因为车辆经常不容易在繁忙的街道或农村地区行驶。飞机可以在更大范围内快速移动，并且可以捕获更多的电话。
  </p>
  <p class="graf graf--p">
   但这也意味着 Dirtbox 的信号强度很可能大于地面使用的那种黄貂鱼 —— 也就是说，这意味着它们会侵害更多的无辜者。
  </p>
  <p class="graf graf--p">
   当局的自辩称 “只保留和使用与犯罪嫌疑人有关的数据。设备确定哪个电话属于可疑电话，然后释放其他所有不相关的电话”。但是，公民自由活动家必须持续警惕，因为政府已多次尝试向法院和设备被捕获的被告
   <strong class="markup--strong markup--p-strong">
    掩盖其对黄貂鱼的使用，并很有可能也隐瞒了对 Dirtbox 的使用
   </strong>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     谁在使用 Dirtbox？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   目前尚不清楚以这种方式进行空中监视已经持续了多长时间。
  </p>
  <p class="graf graf--p">
   国防承包商哈里斯公司（Harris Corporation）于2008年从公共情报部门获得的有关黄貂鱼系统的文件显示，
   <strong class="markup--strong markup--p-strong">
    至少六年前
   </strong>
   ，该公司就以9000美元的价格出售了用于手机监控系统的机载安装套件，表明这种灾难已经持续很久了。
  </p>
  <p class="graf graf--p">
   《华尔街日报》指出，Dirtbox 监视系统由法警局使用，该局从至少五个大都市机场运行塞斯纳飞机，
   <strong class="markup--strong markup--p-strong">
    其位置允许这种间谍飞机的飞行范围覆盖全国大部分人口使用的手机。
   </strong>
  </p>
  <p class="graf graf--p">
   然而，众所周知，法警局已经将其黄貂鱼装备借给了当地警察部门，最近在佛罗里达州发生的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wired.com/2014/06/feds-seize-stingray-documents/" href="https://www.wired.com/2014/06/feds-seize-stingray-documents/" rel="noopener noreferrer" target="_blank">
    一例案件显示
   </a>
   。
  </p>
  <p class="graf graf--p">
   因此，他们也很可能将 Dirtbox 监视设备提供给了全国各地的多个机构。
  </p>
  <p class="graf graf--p">
   《华尔街日报》援引一位官员的话说，执法部门已获得法院命令进行监视。
  </p>
  <p class="graf graf--p">
   但目前尚不清楚批准这种法院命令的法官是否真的明白进行跟踪的性质以及所使用技术的功能。
  </p>
  <p class="graf graf--p">
   在某些情况下，
   <strong class="markup--strong markup--p-strong">
    警察和间谍要么绕过法院而使用黄貂鱼，要么对法官在使用该技术的关键信息方面撒谎或隐瞒，从而在没有被要求回答重要问题的情况下就获得了法院命令
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     编者注
    </strong>
   </span>
   ：IYP在多年前曾经警告过，中国当局最擅长模仿，他们模仿几乎所有恶劣的管制和压制手段；而且不仅中国，全球很多专制国家都在做同样的事。
  </p>
  <p class="graf graf--p">
   这件事在去年牛津互联网学院的年度报告中
   <strong class="markup--strong markup--p-strong">
    再次得到了证实
   </strong>
   ，在这里看到中文版《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%9c%8b%e6%b8%85%e4%bf%a1%e6%81%af%e6%88%98%ef%bc%9a2019%e5%85%a8%e7%90%83%e6%9c%89%e7%bb%84%e7%bb%87%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%93%8d%e7%ba%b5%e6%b8%85%e5%8d%95/" href="https://www.iyouport.org/%e7%9c%8b%e6%b8%85%e4%bf%a1%e6%81%af%e6%88%98%ef%bc%9a2019%e5%85%a8%e7%90%83%e6%9c%89%e7%bb%84%e7%bb%87%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%93%8d%e7%ba%b5%e6%b8%85%e5%8d%95/" rel="noopener noreferrer" target="_blank">
    看清信息战：2019全球有组织社交媒体操纵清单
   </a>
   》。
   <strong class="markup--strong markup--p-strong">
    各国当权者不仅彼此模仿，甚至互相传授/培训专制手段。
   </strong>
  </p>
  <p class="graf graf--p">
   👉这就是为什么IYP一直在转述来自各个国家的 “新型” 的管制压制 —— 因为这些手段基本很快会在其他国家被复制模仿，包括中国。尤其是，如你所知，模仿成功需要一段时间，反抗者完全可以抢在本土当局模仿成功之前找到抵御的方法。
  </p>
  <p class="graf graf--p">
   前提是：知道得越早就越有利。
  </p>

   我们把其中一些相对重要的信息总结到了 “
