<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="090b">
      以这一天为原点的五个月前，清华大学教授黄万里被打成“右派”，其父黄炎培 — — 被称作党外呼喊“毛主席万岁”第一人，主动表示与其三子断绝关系、划清界限；以这一天为原点的一天前，一届人大五次会议预备会议上，钦定“大右派”储安平的人大代表资格被罢免 — — 一个自由主义报人的“万劫不复”开始了。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="bc94">
      以这一天为原点，两个半月后，人民英雄纪念碑落成；三个月零十五天后，中共八大二次会议通过了“第二个五年计划” — — 名为“大跃进”的国难开始了。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="0479">
      这一天当天，第一届全国人民代表大会第五次会议正式开幕。人民日报的头版头条位置以《亿万人民革命干劲激动人心》为题对会议做了预告，盲马脱缰的味道已经隐隐约约地飘散了出来。
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="3d43">
      在这次会议上，时任国务院副总理李先念和薄一波先后做了关于国家预算和国家经济计划的报告，“反保守”与“增产”作为核心主题贯穿在全部的报告内容之中。而在会议正式流程之先，包括章伯钧、罗隆基，与储安平三大钦定“右派”在内的38位“右派”被剥夺了人大代表身份。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="a476">
      在1958年2月1日这一天的人民日报和天津日报的头版，都刊发了这样一条通稿 — — “全国人大常委会通过向人大工作报告”，并配以“中华人民共和国主席令”。“主席令”宣布了章乃器，章伯钧与罗隆基被分别撤销部长职务的消息。

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="ab56">
      当今天，越来越多的人越来越深刻地去怀念追溯这个“大右派”的时候，我试着摘抄几段储安平的话：
     </p>
<blockquote class="iz ja jb">
<p class="ib ic jc id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="d180">
       最后，我们愿意坦白说一句话，政府虽然怕我们批评，而事实上，我们现在则连批评这个政府的兴趣也没有了。即以本刊而论，近数月来，我们已经很少刊载剧烈批评政府的文字，因为大家都已十分消沉，还有什么话说呢？说了又有什么用处？我们替政府想想，一个政府弄到人民连批评它的兴趣也没有了，这个政府也够悲哀的了！可怜政府连这一点自知之明也没有，还在那里抓头挖耳，计算如何封民间的报纸刊物，真是可怜亦复可笑……
      </p>
</blockquote>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="541a">
      这里的“政府”指的是国民政府，这还是1949年之前是储安平主持《观察》之时，对“旧世界”的批评。而他同时也说过：
     </p>
<blockquote class="iz ja jb">
<p class="ib ic jc id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="7d86">
       因为唯有承认人民思想及言论的自由，始能真正实现民主的政治，然则吾人以此事衡量共产党，则共产党是否能容许今日生活在共产党统治区域中的人民有批评共产主义或反对共产党的自由？假如容许，则何以我们从来没有看到在共产党区域中出版的报纸有任何反对共产党或批评共产党的言论，或在共产党区域中有何可以一般自由发表意见的刊物？
      </p>
<p class="ib ic jc id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="41d5">
       老实说，我们现在争取自由，在国民党统治下，这个“自由”还是一个“多”、“少”的问题，假如共产党执政了，这个“自由”就变成一个“有”、“无”的问题了。
      </p>
</blockquote>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="3a59">
      一个洞察世事如此清晰之人，为何会选择留下？为何会落得万劫不复至此。距离这份生日报八年之后的文革元年，多次自杀未遂的储安平彻底失踪了。若是他继续完整地经历了之后的十年呢？他将又会有怎样的遭遇？
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="724d">
      自此，“反右”之事基本盖棺定论，该引的“蛇”都被引了出来，国家会议上再也不会出现黄万里或储安平之类的“异样之声”，万马齐喑，为所欲为，通往“大跃进”和“文革”道路已然坦途。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="d5af">
      在这一天人民日报的三版上，“赶超英国”的口号已经被叫嚣得肆无忌惮。
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="cbc4">
      国家的癫狂从对知识精英的摧残与驯化，转向了更加荒诞的、一场接一场的平民浩劫。“大跃进”的门开了，亩产万斤的幻觉麦香、大炼钢铁的焦糊味道，已经从门缝之中飘荡而出了。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="9e4c">
      在1958年2月1日天津日报第三版的分类广告上，一部名为“右派百丑图”的评剧标识了那个时代文艺作品的特色。

      “右派百丑图”，原著杨华生，国家级非物质文化遗产“独脚戏”传承人，著名的滑稽戏演员，2012年病逝于上海。除评剧这一形式外，“右派百丑图”还曾以连环画，小说，剧本等形式在当时出版发行。

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="080e">
      分类广告中的另一则，是“相声大会”的节目预告。相声大会历史悠久，沿用其名的常规相声演出至今仍在天津市保持活跃和正常演出。其创立人可追溯至明国初期赫赫有名的“相声八德”中的马德禄与周德山 — — 马德禄是马三立之父，周德山是马三立师傅。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="f8e9">
      曾经，京津地区耳熟能详的相声大家，多参与过其中，包括马三立、张寿臣、常宝堃、刘宝瑞等。常宝堃早在1951随慰问团年丧生于朝鲜，最初的墓地现在成为了天津市的一处高档社区；而“单口大王”刘宝瑞则和储安平一样，殒命文革之中，死因不明，尸首不见。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="3ae5">
      在这一天天津日报第四版的底栏分类广告中，有“大光明电影院”的剧目预告，这是在新式影院于2000年之后兴起前，天津市硬件设备最高端的影院。天津大光明影院由一位英籍印度人建成于1929年，略晚于更加著名的上海大光明影院，属于中国最早的一批电影院之一。直至2000年前后，我们现在熟知的“新型影院”在国内兴起，在此之前，“大光明”一直是天津市硬件设备最为精良的影院。
     </p>

      说到这家影院和我的渊源，是因为据我父母讲述，他们在当年恋爱时一同看过的第一场电影就在这家影院 — — 《庐山恋》。

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="1dc0">
      以至于后来，为了延续家族传统，我在追求姑娘时，也常常选择这家影院当做初次约会场所。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="7a7d">
      记得有一次，恰是2003年，格里高利·派克（Gregory Peck）辞世，大光明影院临时加映《罗马假日》（Roman Holiday）。我约上一位姑娘同去，意在暗示她“小记者也可以泡公主”这一电影主题。当影片落幕，姑娘非常感动且若有所思，然后在出门的时候，拒绝了我。
