  <p class="graf graf--p">
   今天，社区已经变成了一个热门的流行语。和所有其他流行语一样，它们代表的仅仅是被玩坏的东西。
  </p>
  <p class="graf graf--p">
   在市场宣传中，就像在政治言论中一样，这个词无处不在：绿色新政和扎克伯格2017年的所谓“建立全球社区”，都反复使用“社区”这个词（分别是26次和108次），但你必须知道，在这种语境下，使用“用户群”、“邻里”、或“群组”，就已经足够了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    但是，他们依旧要强加 “社区” 的概念
   </strong>
   。这是因为在这种情况下，“社区” 的概念能表现为无懈可击的善举：需要被认可的、最大化的和受保护的事物。
  </p>
  <p class="graf graf--p">
   社区被认为是一件快乐的事，各组成部分彼此同意，因为他们或多或少是具有相同价值观的同一类人。
  </p>
  <p class="graf graf--p">
   它们代表了社会关系中的“
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Goldilocks_principle" href="https://en.wikipedia.org/wiki/Goldilocks_principle" rel="noopener noreferrer" target="_blank">
    金发姑娘原则
   </a>
   ”：不是太没人情味、也不是太个人主义 —— 恰到好处。
  </p>
  <p class="graf graf--p">
   人们认为，社区是重要工作的合适规模，而不是没有责任心的个人或不露面的官僚机构；并且，社区始终为自己的成员的最大利益而行动。
  </p>
  <p class="graf graf--p">
   ⚠️由此来，该隐喻就会被盗用和污染，为国家和寡头公司等宏观社会结构的利益提供方便的掩盖。并且造成人们更广泛地对社区实际运作方式的误解。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     如果你相信全球品牌或国家是“社区”，就可以认为社区是可以注册的服务
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   社区的概念直接与社会联系在一起，这两个具有独特特征的社会单元，而不是完全分开的，代表了一系列人类互动的类型。
  </p>
  <p class="graf graf--p">
   社会关系包括维护大规模文明所需的复杂设备。
  </p>
  <p class="graf graf--p">
   百货公司的业务员、国会代表、以及可帮助您建立所选的社交媒体平台的编码员：这些人都与您
   <strong class="markup--strong markup--p-strong">
    有社会关系
   </strong>
   ；
  </p>
  <p class="graf graf--p">
   相比之下，社区是成员彼此分享的自主性的、基于共同价值观和信念的纽带。
  </p>
  <p class="graf graf--p">
   正是这种深厚的联系使得无需复杂的会计系统（金钱）就可以实现互助。
  </p>
  <p class="graf graf--p">
   邻居会借给你一杯酱油，但是杂货店不会，你不给钱的话什么也拿不走。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    社会是市场、公司、制度化宗教、和民族国家等力量所必需的唯物主义关系。另一方面，社区是依靠情感关系连接的、是提供团结感和归属感的社交纽带。
   </strong>
  </p>
  <p class="graf graf--p">
   Facebook 等平台试图利用的正是这种情感元素。
  </p>
  <p class="graf graf--p">
   20世纪主要是考虑到现代化从“社区”向“社会”的推动，而这个年轻的世纪已经尝试了多种方法来调和两者：在复杂的社会中感受到一点社区感。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “社区” 唤起了真正的人际关系，或者至少在渴望那些有效联系的社会中存在着这种可能性。
   </strong>
  </p>
  <p class="graf graf--p">
   由于大多数后期资本主义市场实体的根本目标是满足贪欲（通常是他们自己创造的贪欲），因此，
   <strong class="markup--strong markup--p-strong">
    公司和政府官员会窃取 “社区” 的概念，以吸引消费者和选民。
   </strong>
  </p>
  <p class="graf graf--p">
   正如娜奥米·克莱因（Naomi Klein）在她的《No Logo》一书中首次概述的那样，
   <strong class="markup--strong markup--p-strong">
    至少从90年代初开始，企业就致力于努力销售无形的归属感 —— 它可以比优质产品获得更多的利润
   </strong>
   。
  </p>

   <li class="graf graf--li">

   </li>

  <p class="graf graf--p">
   这具有降低成本（营销成本相对较低）和通过为品牌吸引力收取溢价来提高价格的双重好处。
  </p>
  <p class="graf graf--p">
   从耐克（Nike）到星巴克（Starbucks）的几乎使用你知道的品牌，都旨在代表一种已有的社会关系，人们可以通过购买产品来获得这种 “关系感”。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9929 jetpack-lazy-image" data-lazy-sizes="(max-width: 1000px) 100vw, 1000px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-2.jpg?resize=1000%2C668&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-2.jpg?w=1000&amp;ssl=1 1000w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-2.jpg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-2.jpg?resize=768%2C513&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-2.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="668" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-2.jpg?resize=1000%2C668&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1000"/>

  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种社区社团的**想象**是品牌忠诚度的强大力量
   </strong>
   ，这就是为什么科技公司希望在所有级别的客户体验中运用这种感觉的原因。
  </p>
  <p class="graf graf--p">
   从1984年的 Apple 到2001年的 iPod 的发明，几乎每一个 Apple 广告中，电脑和配件都是 “个人创意天才” 的必要最终要素；但是，随着苹果公司大量转向销售用于消费媒体的手持设备，他们也开始倾向于 “拥有自己的产品成为社区一部分” 的想法：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    iPhone 广告几乎完全没有提到技术，而是选择在设备捕获或传播体验时展现出那种凄美而亲密的时刻
   </strong>
   。
  </p>
  <p class="graf graf--p">
   正如90年代的星巴克所展示的那样，包容性可以证明加价的理由；苹果公司将其旗舰店称为“城镇广场” —— 要知道，这些地方是可以作为举办活动的场所，而不仅仅是产品的货架。
  </p>
  <p class="graf graf--p">
   他们的 App Store 将从 TikTok 到女同性恋约会应用程序 Her 的所有内容捆绑到名为“查找您的社区”的列表中。
  </p>
  <p class="graf graf--p">
   在最近有关防止平台上网络欺凌问题的新工具的
   <a class="markup--anchor markup--p-anchor" data-href="https://instagram-press.com/blog/2019/07/08/our-commitment-to-lead-the-fight-against-online-bullying/" href="https://instagram-press.com/blog/2019/07/08/our-commitment-to-lead-the-fight-against-online-bullying/" rel="noopener noreferrer" target="_blank">
    新闻稿中
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    Instagram 的发言听起来就像是一位关心中学生的校长
   </strong>
   ：“我们从社区中的年轻人那里听说，他们不愿意阻止、取消关注或举报欺凌行为，因为这可能会加剧紧张局势，尤其是当他们与现实生活中的霸凌互动存在时……”
  </p>
  <p class="graf graf--p">
   Amazon Ring 的监视器工具 Neighbors 应用程序呼吁用户 “可以共同创建更强大的社区”，该应用程序允许用户接收和共享警察警报、以及可疑犯罪活动的报告，并且已经
   <a class="markup--anchor markup--p-anchor" data-href="https://www.vice.com/en_us/article/qvyvzd/amazons-home-security-company-is-turning-everyone-into-cops" href="https://www.vice.com/en_us/article/qvyvzd/amazons-home-security-company-is-turning-everyone-into-cops" rel="noopener noreferrer" target="_blank">
    充分暴露了
   </a>
   其种族主义特征的高度敏感性。
  </p>

   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%9a%e9%a9%ac%e9%80%8a%e5%b8%ae%e5%8a%a9%e8%ad%a6%e6%96%b9%e6%9e%84%e5%bb%ba%e4%ba%86%e4%b8%80%e4%b8%aa%e5%b9%bf%e6%b3%9b%e7%9a%84%e7%9b%91%e6%8e%a7%e7%bd%91%e7%bb%9c/" href="https://www.iyouport.org/%e4%ba%9a%e9%a9%ac%e9%80%8a%e5%b8%ae%e5%8a%a9%e8%ad%a6%e6%96%b9%e6%9e%84%e5%bb%ba%e4%ba%86%e4%b8%80%e4%b8%aa%e5%b9%bf%e6%b3%9b%e7%9a%84%e7%9b%91%e6%8e%a7%e7%bd%91%e7%bb%9c/" rel="noopener noreferrer" target="_blank">
     亚马逊帮助警方构建了一个广泛的监控网络
    </a>
    》
   </li>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    ⚠️要运用 “社区” 隐喻，需要三个步骤
   </strong>
   ：
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    在目标人群中激发不适或恐惧，只因为他们还不是社区成员；
   </li>
   <li class="graf graf--li">
    向他们展示利益和归属条款；
   </li>
   <li class="graf graf--li">
    最后，在概述允许他们停留的规则的同时为他们提供一个席位。
   </li>
  </ol>
  <p class="graf graf--p">
   ⚠️这三个简短的步骤将个人的注意力吸引到了现代性破碎的自由观上 —— 您可以做任何事，但这也意味着
   <strong class="markup--strong markup--p-strong">
    您可能无法创造自我，然后只能接受指导，被确立一种身份，并**按要求**发挥作用
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Facebook 将自己描述为 FOMO （错失恐惧症）的解毒剂：所有朋友都聚集在这里的时尚聚会 ……
  </p>
  <p class="graf graf--p">
   该公司不仅通过突出显示您在使用它时所获得的访问权，而且，还通过他们自己的算法提醒您“有可能错过了的东西”，来使自己变得必要：活动邀请、群聊、meme引用等等 ……
  </p>
  <ul class="postList">

  <p class="graf graf--p">
   将跨越地球的数字网络（如 Facebook 或 Reddit）称为“社区”，比说组成加拿大或朝鲜的官僚机构和风俗习惯由普通百姓来控制，或多或少是现实的。
  </p>
  <p class="graf graf--p">
   本尼迪克特·安德森（Benedict Anderson）在《想象中的社区》中所做的研究，概述了印刷机之类的技术如何与早期的资本主义组织和工业革命相碰撞，以消除地区差异。
  </p>
  <p class="graf graf--p">
   这些技术帮助标准化了从语言到计时的各种文化习俗，使越来越多的人团结成一个单一的政治实体 —— 最终演变成由一套占主导地位的通用语言、一套法律、军事和警察部队、以及具体边界组成的东西。
  </p>
  <p class="graf graf--p">
   因此，
   <strong class="markup--strong markup--p-strong">
    这个想象中的 “国家” 创造了一种精神，这种精神在以前不存在的地方制造归属感。
   </strong>

  <p class="graf graf--p">

  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     可以将无定形的“社区”概念化为某种社会之外的事物
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   分层的政治实体，例如国家政党和宣传团体，与公司的工作模式非常类似，都是声称自己是有组织的、经过提炼的、和媒体就绪的社区制度联合体。
  </p>
  <p class="graf graf--p">
   他们还将这种构造投射到其他群体上。
  </p>
  <p class="graf graf--p">
   在新千年中，20世纪美国和欧洲援助机构的乐观和傲慢态度已经让位于国际发展的更协作性形式。
  </p>
  <p class="graf graf--p">
   在2001年发表的《人类参与：新暴政》的一章中，国际发展专家乌玛·科塔里（Uma Kothari）对国际援助的 “参与式发展” 方法提出了批评，来自富裕国家的援助工作者出现在贫穷国家的偏远地区 —— 也许是在一场严重的灾难之后，也许是为了进行更全面的重组 —— 并询问了 “社区” 他们需要什么。
  </p>
  <p class="graf graf--p">
   批评指出，
   <strong class="markup--strong markup--p-strong">
    在与当地人的交流中，“大多数人的日常生活的复杂性和混乱性被过滤掉了”，而 **局外人** 则提炼出社区运作方式的理想化版本
   </strong>
   。
  </p>
  <p class="graf graf--p">
   简单说：⚠️局外人将当地社区视为了一个整体，而没有考虑到权力动态和内部竞争利益。
  </p>

   <strong class="markup--strong markup--p-strong">

   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️这就是为什么我们一直要强调，
   <strong class="markup--strong markup--p-strong">

  </p>
  <p class="graf graf--p">
   矛盾的是，所有社区都以相同的方式呈现出各自的独特性：它们是相互重叠的、长期存在的紧张关系和同盟的场所，无法简化为单一名称或总体机构。
  </p>
  <p class="graf graf--p">

   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    充其量，“社区” 对话只是让有权势的人回避了棘手的话题
   </strong>

  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    通过这种方式，它避免了直面导致人们面临的真正问题的权力结构
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️就如在中国你经常看到的那样，新闻人、农民工、流民、中产阶级、学生 …… 抽象”社区”永远也解决不完的问题；但如果这些社区联合起来，大概所有人都能有所希望。为什么不呢？
  </p>
  <p class="graf graf--p">
   阿道夫·里德（Adolph Reed Jr.）在1996年发表的一篇预言性文章中，谴责了政治中对“黑人社区”的不断提及，部分原因是：提及单一的看似有凝聚力的社区意味着存在无可争议的领导职位。这不是事实。
  </p>
  <p class="graf graf--p">
   他写道，所谓的黑人社区的形象为 “白人精英承认、并与仅仅名义上且最终无法解释的‘种族领袖’进行谈判，创造了机会。”
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    这样一来，富人和有权势的人，就可以选择那些“忠实的”反对者，并切割已经被边缘化的人群
   </strong>
   。
  </p>
  <p class="graf graf--p">
   最近的一个例子是“人权运动 Human Rights Campaign”，该运动将注意力和资源转移到主要是富裕的白人男性同性恋者的利益和结婚权上，以牺牲和反对 LGBTQ 运动中更多被边缘派别的抗议为代价。
  </p>
  <p class="graf graf--p">
   Twitter 的“信任与安全委员会”由来自“社区团体”的成员组成，而他们本身就是极具竞争性的代表。
  </p>
  <p class="graf graf--p">
   例如，反诽谤联盟经常将对以色列的批评归类为 “反犹主义”，这一立场至少与大多数年轻的美国犹太人
   <a class="markup--anchor markup--p-anchor" data-href="https://www.jstor.org/stable/10.2979/israelstudies.22.3.08?seq=1#page_scan_tab_contents" href="https://www.jstor.org/stable/10.2979/israelstudies.22.3.08?seq=1#page_scan_tab_contents" rel="noopener noreferrer" target="_blank">
    相去甚远
   </a>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     社交媒体公司声称自己是 “社区的领导者”，尽管这些监视资本家的利益从来都与用户的利益截然相反
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   并且，每次调用 “社区” 概念的背后都隐含着对该组织内部权力结构的默许，无论是保守派教会、霸道父母、还是反动邻居。
  </p>
  <p class="graf graf--p">
   并且，提及社区的 “目标”、“文化”、或任何其他特征，意味着
   <strong class="markup--strong markup--p-strong">
    不必再处理
   </strong>
   其内部冲突、毒性、个体之间的压迫或胁迫动态。
  </p>
  <p class="graf graf--p">
   反过来，无定形的社区可以被概念化为某种形式游离于社会之外，或自然地与之对立，或纯粹地对它有益，
   <strong class="markup--strong markup--p-strong">
    这完全取决于宣布该“社区”存在的实体的意图。
   </strong>
  </p>
  <p class="graf graf--p">
   社区意味着超越政治的条件，即 集体自动分享价值、并就如何实现价值达成共识；
   <strong class="markup--strong markup--p-strong">
    实际上，社区真正需要的恰恰是解决这些分歧的过程和能力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9932 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/图4.jpg?resize=1100%2C1608&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/图4.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/图4.jpg?resize=205%2C300&amp;ssl=1 205w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/图4.jpg?resize=768%2C1123&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/图4.jpg?resize=701%2C1024&amp;ssl=1 701w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/图4.jpg?resize=1100%2C1608&amp;ssl=1 1100w" data-recalc-dims="1" height="1608" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/图4.jpg?resize=1100%2C1608&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9932" data-recalc-dims="1" height="1608" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/图4.jpg?resize=1100%2C1608&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/图4.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/图4.jpg?resize=205%2C300&amp;ssl=1 205w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/图4.jpg?resize=768%2C1123&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/图4.jpg?resize=701%2C1024&amp;ssl=1 701w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/图4.jpg?resize=1100%2C1608&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   社交媒体公司试图将社区的面纱包裹在先前存在的权力体系中。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这提供了某种修辞掩盖、针对实际危机的轻描淡写的回应
   </strong>
   —— “嗨，我知道 Facebook 上有很多纳粹分子，但是我们是一个社区，所以我们必须共同努力！” —— 同时，这也使公司领导者在面向平台用户讲话时具有了某种虚构的合法性。
  </p>
  <p class="graf graf--p">
   即使这些监视资本家的利益总是与用户的利益截然相反，他们也仍然可以声称自己是 “社区的领导者”。
  </p>
  <p class="graf graf--p">
   这个
   <strong class="markup--strong markup--p-strong">
    意淫出来的社区宣称
   </strong>
   最荒谬的版本是 Reddit，用户几乎普遍将站点的不同部分称为 “subreddits”，而公司本身则坚持称其为“社区”。
  </p>
  <p class="graf graf--p">
   网站行为与社区行为最不相称的部分正是由网站向潜在的新用户和广告商推广的部分，而那些确实对世界具有统一的愿景并有严格的参与规则的人通常被“隔离”，表示他们需要经过电子邮件验证的才能参与，他们被拒绝在所谓的令人反感的内容的免责声明背后，并且未包含在 Reddit 的首页上。
  </p>
  <p class="graf graf--p">
   使社区如此具有吸引力的情感纽带，甚至是人类繁荣所必需的情感纽带，已轻松地被转移到了社交媒体平台上 —— 部分是因为可以在此处找到您的朋友、家人和邻居，而且，还因为点赞、评论、转推和声誉促进，都是增强这种归属感的机制。
  </p>
  <p class="graf graf--p">
   当您只在1000人的小组讨论中听到笑话时，或在一个私密的社交媒体小组中感到被理解时，就会产生包容感；然而，平台就可以利用“社区”的概念来支撑其品牌，使它们成为有意义的联系发生的地方，社区时刻……
  </p>
  <p class="graf graf--p">
   Reddit 证明公司不希望有实际的社区，“社区”会为参与设置条件，以及几乎可以肯定会给其他所有人带来麻烦的一组规则或事实。
  </p>
  <p class="graf graf--p">
   社交媒体依靠快速的入门、易于理解的规则、和“欢迎”内容来根据风险资本的需求进行扩展。也许更重要的是，
   <strong class="markup--strong markup--p-strong">
    它表明社区不是我们所认为的纯粹的商品。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当你能找到适合自己的东西时，你会感到非常高兴，但这肯定不是可以通过算法建议可靠地传递给每个人的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   正如 Reed 所说：
   <strong class="markup--strong markup--p-strong">
    “对培养和保护谈判领域的关注越少，平衡就越多地转向胁迫。社区的修辞使前者不耐烦，其逼真的虚构使后者合理化。”
   </strong>
  </p>
  <p class="graf graf--p">
   也许我们现在需要的不是更多的社区，而是一个更加公正的社会。社交媒体使 “社区” 的构建变得更容易，但那根本不是什么社区。⚪️
  </p>
