  <p class="graf graf--p">
   对于监视的恐惧来自很多不同的方向，有些国家最担心的是政府监控、有些国家更担心寡头企业的监视资本主义、而对于韩国来说，人们最担心的是社会上的变态狂。
  </p>
  <p class="graf graf--p">
   在韩国10个城市的42间酒店客房中，有超过1,600位客人被一个偷窥网站秘密拍摄和现场直播，该网站有4,000名会员 — — 其中97人每月支付44.95美元以获得额外的功能，例如能够重播视频流，这是根据
   <a class="markup--anchor markup--p-anchor" data-href="https://www.stuff.co.nz/world/asia/111472899/four-arrested-over-south-korea-hotel-spy-camera-scandal" href="https://www.stuff.co.nz/world/asia/111472899/four-arrested-over-south-korea-hotel-spy-camera-scandal" rel="noopener noreferrer" target="_blank">
    Stuff 的报道
   </a>
   。
  </p>
  <p class="graf graf--p">
   30家酒店受到影响，但警方表示没有迹象表明这些企业参与其中。
  </p>
  <p class="graf graf--p">
   这些酒店的客人在2018年11月24日和今年3月2日之间被偷偷拍摄，而该网站为其运营者带来了超过6,200美元的收入。此案中有四人被捕。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    据称，其中一名主要嫌疑人在作为客人进入酒店后安装了相机。另一名嫌疑人被指控负责发布和管理现已被关闭的那个网站。据警方称，另外两人涉嫌购买间谍相机或资助互联网网站的运作
   </em>
   。
  </p>
  <p class="graf graf--p">
   根据国家警察厅网络调查部门的说法，
   <strong class="markup--strong markup--p-strong">
    摄像机隐藏在墙壁插座、吹风机支架和数字电视盒内
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “过去有类似的情况，非法摄像机（秘密安装）并且一直在进行秘密监视，但这是警察第一次抓住视频在互联网上直播的地方，”警方说。
  </p>
  <p class="graf graf--p">
   韩国的间谍相机和非法拍摄
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnn.com/2018/09/06/asia/south-korea-spy-cams-toilet-intl/index.html" href="https://www.cnn.com/2018/09/06/asia/south-korea-spy-cams-toilet-intl/index.html" rel="noopener noreferrer" target="_blank">
    存在严重问题
   </a>
   。 2017年，受害人向警方报告了超过6,400起非法拍摄案件，而2012年则为2,400起。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    去年，在“我的生活不是你的色情”的口号下，成千上万的女性走上首尔和其他城市的街头抗议这种做法
   </strong>
   ，并要求采取行动制止这种行为。
  </p>
  <p class="graf graf--p">
   这两名主要肇事者将面临长达7年的监禁和3000万韩元的罚款（26,581美元）。
  </p>
  <p class="graf graf--p">
   首尔市警察厅网络调查部门通过“韩国先驱报”称，“警方严格处理发布和分享非法视频的犯罪分子，因为他们严重损害了人的尊严。”
  </p>
