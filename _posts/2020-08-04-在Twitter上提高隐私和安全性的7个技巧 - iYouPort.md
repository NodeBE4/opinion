  <p class="graf graf--p">
   Twitter 很有趣，但它也会变得令人恐惧。trolls和政府间谍最喜欢在这里攻击您的推文并跟踪您的一举一动。
  </p>
  <p class="graf graf--p">
   虽然可以将整个账户设为私密以阻止外部监视，但是，这样做显然会阻碍您的信息传播 ——
   <strong class="markup--strong markup--p-strong">
    并且您不一定就能准确识别每一位申请关注者究竟是什么人、也无法预测其未来有什么意图。
   </strong>
  </p>
  <p class="graf graf--p">
   事实上您可以通过简单切换一些功能设置，即可提高在线安全性。iOS 和 Android 都可以，虽然步骤有一些不同。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、禁用照片标记
   </strong>
  </h3>
  <p class="graf graf--p">
   类似于在Twitter上@提及您，用户还可以在其发布的照片​​中为您添加标记。听起来似乎没有恶意，但是，如果用户在您不想与之公开产生关联的图片中标记您，您可能希望加强匿名性。
  </p>
  <p class="graf graf--p">
   幸运的是，您可以轻松阻止其他用户在他们的照片中标记您。
  </p>
  <p class="graf graf--p">
   您可以对其进行设置，以便没有人可以标记您，也可以仅仅阻止您不认识的人标记您。
  </p>
  <p class="graf graf--p">
   而且，如果您将其完全关闭，那么以前使用过的所有标记都会从您的帐户中消失，因此对于标记照片来说，这是一个终极开关。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13102 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-5.png?resize=1100%2C945&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-5.png?w=1208&amp;ssl=1 1208w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-5.png?resize=300%2C258&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-5.png?resize=1024%2C880&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-5.png?resize=768%2C660&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-5.png?resize=1100%2C945&amp;ssl=1 1100w" data-recalc-dims="1" height="945" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/1-5.png?resize=1100%2C945&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、禁用已读回执
   </strong>
  </h3>
  <p class="graf graf--p">
   如果您的DM已打开，则意味着任何人都可以向您发送消息。并且默认情况下，如果您阅读了该消息，那么对方会知道，该消息在其结尾处带有一个蓝色的对勾。
  </p>
  <p class="graf graf--p">
   为防止用户知道您阅读了邮件，请禁用已读回执。
  </p>
  <p class="graf graf--p">
   在 “设置和隐私” 菜单中，点击 “隐私和安全”。然后，关闭 “显示已读回执”。
  </p>
  <p class="graf graf--p">
   在 Android 应用中，先点击 “Direct Messages” 以找到该设置。
  </p>
  <p class="graf graf--p">
   现在，当您阅读某人发来的私信时，在消息下方的箭头将是灰色的（未读），而不是蓝色（已读）。
  </p>
  <p class="graf graf--p">
   如果您关闭已读回执，也不会看到其他人何时阅读您的私信。
  </p>
  <p class="graf graf--p">

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、匿名观看 Periscope 视频
   </strong>
  </h3>
  <p class="graf graf--p">
   Periscope 是内置在Twitter中的实时视频流平台。除了直播之外，您还可以观看和评论其他人的视频，但他们会知道您正在观看。
  </p>
  <p class="graf graf--p">
   如果您想继续观看直播视频，但又想保持匿名，完全可以断开与 Periscope 的连接。
  </p>
  <p class="graf graf--p">
   请注意，此选项仅存在于适用于 iOS 和 Android 的 Twitter 应用程序中。
  </p>
  <p class="graf graf--p">
   在 “设置和隐私” 菜单中，点击 “隐私和安全”。然后，关闭 “Connect to Periscope”。现在，您能够自由观看 Periscope 视频，但其他人将不知道您正在观看。
  </p>
  <p class="graf graf--p">
   只有将其重新打开，您才能做自己的直播或对其他实时视频发表评论。
  </p>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、阻止其他用户找到您
   </strong>
  </h3>
  <p class="graf graf--p">
   通过使用随机头像并将您自己的名字从帐户中删除，您可以使其他人很难在 Twitter 上找到您，但是，您也可以阻止其他人使用您的电子邮件地址或电话号码来查找您的个人资料。
  </p>
  <p class="graf graf--p">
   同样，进入 “设置和隐私”，然后进入 “隐私和安全”。再进入“可发现性和联系方式” 并关闭选项，现在那些追踪到您的电子邮件地址和电话号码的人就很难在 Twitter 上找到您了。
  </p>
  <p class="graf graf--p">
   当然，建议您尽可能
   <strong class="markup--strong markup--p-strong">
    不要
   </strong>
   使用自己真实的电邮地址和电话号码注册推特，因为该公司依旧拥有这些信息，很难说他们是否会将这些信息交给政府。
  </p>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、禁用您的地理位置
   </strong>
  </h3>
  <p class="graf graf--p">
   默认情况下，Twitter 会收集并存储使用设备的GPS和其他位置数据点跟踪的精确坐标，以向您显示本地广告以及特定于位置的趋势、内容和建议。
  </p>
  <p class="graf graf--p">
   它们还用于将位置标签添加到您的推文中，很难单独禁用。
  </p>
  <p class="graf graf--p">
   要关闭位置数据，请返回 “设置和隐私”，然后返回 “隐私和安全”。下来，点击 “精确位置”，然后将其禁用。
  </p>
  <p class="graf graf--p">
   在适用于 Android 的 Twitter Lite 中，该选项不可用，但在 Tweets 标题下有一个 “位置信息” 选项，您可以启用或禁用它，但它仅适用于您制作的 Tweet。
  </p>
  <p class="graf graf--p">
   停用 “精确位置” 会阻止您接收本地内容并将位置数据添加到推文中，但并不会阻止您获得一般广告。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13109 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5.png?resize=1100%2C1159&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5.png?w=1188&amp;ssl=1 1188w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5.png?resize=285%2C300&amp;ssl=1 285w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5.png?resize=972%2C1024&amp;ssl=1 972w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5.png?resize=768%2C809&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5.png?resize=1100%2C1159&amp;ssl=1 1100w" data-recalc-dims="1" height="1159" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5.png?resize=1100%2C1159&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13109" data-recalc-dims="1" height="1159" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5.png?resize=1100%2C1159&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5.png?w=1188&amp;ssl=1 1188w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5.png?resize=285%2C300&amp;ssl=1 285w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5.png?resize=972%2C1024&amp;ssl=1 972w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5.png?resize=768%2C809&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/5.png?resize=1100%2C1159&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    6、停止个性化和数据跟踪
   </strong>
  </h3>
  <p class="graf graf--p">
   Twitter 始终会收集您的数据，但是您可以控制它如何在线操纵您。这个方法我们以前介绍过，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%bb%95%e5%bc%80%e6%8e%a8%e7%89%b9%e7%ae%97%e6%b3%95%e5%99%aa%e9%9f%b3%e3%80%81%e4%bd%bf%e7%94%a8%e5%9f%ba%e6%9c%ac%e8%bf%90%e7%ae%97%e7%ac%a6%e6%8c%96%e6%8e%98%e6%8e%a8%e7%89%b9%e6%95%b0%e6%8d%ae/" href="https://www.iyouport.org/%e7%bb%95%e5%bc%80%e6%8e%a8%e7%89%b9%e7%ae%97%e6%b3%95%e5%99%aa%e9%9f%b3%e3%80%81%e4%bd%bf%e7%94%a8%e5%9f%ba%e6%9c%ac%e8%bf%90%e7%ae%97%e7%ac%a6%e6%8c%96%e6%8e%98%e6%8e%a8%e7%89%b9%e6%95%b0%e6%8d%ae/" rel="noopener noreferrer" target="_blank">
    绕开推特算法噪音、使用基本运算符挖掘推特数据的方法
   </a>
   》。
  </p>
  <p class="graf graf--p">
   如果您觉得上面这篇文章中介绍的方法有点复杂，那么您至少可以做到最基本的。要查看您的个性化设置和数据设置，请返回 “设置和隐私”，然后选择 “隐私和安全”。
  </p>
  <p class="graf graf--p">
   在 “个性化和数据”中，您可以选择关闭以下功能：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    Personalized ads
   </li>
   <li class="graf graf--li">
    Personalize based on your inferred identity
   </li>
   <li class="graf graf--li">
    Personalize based on places you’ve been
   </li>
   <li class="graf graf--li">
    Track where you see Twitter content across the web
   </li>
   <li class="graf graf--li">
    Share your data with Twitter’s business partners
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-13112 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/unnamed-file.png?resize=1100%2C1150&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/unnamed-file.png?w=1196&amp;ssl=1 1196w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/unnamed-file.png?resize=287%2C300&amp;ssl=1 287w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/unnamed-file.png?resize=980%2C1024&amp;ssl=1 980w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/unnamed-file.png?resize=768%2C803&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/unnamed-file.png?resize=1100%2C1150&amp;ssl=1 1100w" data-recalc-dims="1" height="1150" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/unnamed-file.png?resize=1100%2C1150&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13112" data-recalc-dims="1" height="1150" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/unnamed-file.png?resize=1100%2C1150&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/unnamed-file.png?w=1196&amp;ssl=1 1196w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/unnamed-file.png?resize=287%2C300&amp;ssl=1 287w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/unnamed-file.png?resize=980%2C1024&amp;ssl=1 980w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/unnamed-file.png?resize=768%2C803&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/unnamed-file.png?resize=1100%2C1150&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   您可以关闭单个设置，也可以通过直接切换 “个性化和数据” 设置来禁用所有设置。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13111 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/7.png?resize=1100%2C1140&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/7.png?w=1206&amp;ssl=1 1206w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/7.png?resize=289%2C300&amp;ssl=1 289w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/7.png?resize=988%2C1024&amp;ssl=1 988w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/7.png?resize=768%2C796&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/7.png?resize=1100%2C1140&amp;ssl=1 1100w" data-recalc-dims="1" height="1140" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/7.png?resize=1100%2C1140&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    7、撤消第三方应用程序访问权限
   </strong>
  </h3>
  <p class="graf graf--p">
   第三方应用程序有时会要求您使用 Twitter 凭据登录，包括 Instagram，Spotify，Imgur，Duolingo，TweetDeck，Reddit，Buffer 和 Twitterific.
  </p>
  <p class="graf graf--p">
   尽管您可以信任其中一些应用程序，但还有一些应用程序可能是不干净的，这就是为什么说在使用完毕后应撤消它们对您帐户的访问权限。
  </p>
  <p class="graf graf--p">
   是的，再次进入“设置和隐私”，但是这次，点击 “帐户”。在 “数据和权限”下，点击 “Apps and sessions” 以查看所有有权访问您帐户的第三方应用程序。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13113 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/8.png?resize=1100%2C1143&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/8.png?w=1218&amp;ssl=1 1218w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/8.png?resize=289%2C300&amp;ssl=1 289w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/8.png?resize=985%2C1024&amp;ssl=1 985w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/8.png?resize=768%2C798&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/8.png?resize=1100%2C1143&amp;ssl=1 1100w" data-recalc-dims="1" height="1143" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/8.png?resize=1100%2C1143&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13113" data-recalc-dims="1" height="1143" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/8.png?resize=1100%2C1143&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/8.png?w=1218&amp;ssl=1 1218w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/8.png?resize=289%2C300&amp;ssl=1 289w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/8.png?resize=985%2C1024&amp;ssl=1 985w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/8.png?resize=768%2C798&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/8.png?resize=1100%2C1143&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   点击应用程序以查看描述、网站和您授予的权限。要删除哪个请点击 “撤消访问权限”。除非您再次授予其访问权限，否则该应用程序将不再与您的 Twitter 帐户一起使用。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13114 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/9.png?resize=1100%2C1146&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/9.png?w=1206&amp;ssl=1 1206w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/9.png?resize=288%2C300&amp;ssl=1 288w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/9.png?resize=983%2C1024&amp;ssl=1 983w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/9.png?resize=768%2C800&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/9.png?resize=1100%2C1146&amp;ssl=1 1100w" data-recalc-dims="1" height="1146" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/08/9.png?resize=1100%2C1146&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   最后，再次重申
   <strong class="markup--strong markup--p-strong">
    以上这些步骤都是最基本的
   </strong>
   ；也就是大众应该能做到的。每一种安全都必须明确威胁模型，如果您是敏感人士、被政府长期盯梢的对象，仅仅这样做依旧是不够的，您可以参见这里介绍的步骤，见《
