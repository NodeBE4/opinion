  <p class="graf graf--p">

   <a class="markup--anchor markup--p-anchor" data-href="https://chrome.google.com/webstore/detail/twitter-archiver/pkanpfekacaojdncfgbjadedbggbbphi" href="https://chrome.google.com/webstore/detail/twitter-archiver/pkanpfekacaojdncfgbjadedbggbbphi" rel="noopener noreferrer" target="_blank">
    Twitter Archiver
   </a>

  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   授权您的 Twitter 帐户后，再次转到 Twitter Archiver 菜单并创建一个新的搜索规则。
  </p>
  <p class="graf graf--p">
   如果您已经习惯在 Twitter 上使用高级搜索页面，那么此搜索屏幕对您来说将非常熟悉。
  </p>
  <p class="graf graf--p">
   您可以在此创建提及某些搜索词的规则、查找确切的短语、通过#hashtags查找推文、@提及特定 Twitter 用户的推文 ……等等。
  </p>
  <p class="graf graf--p">
   现在，您已经创建了 Twitter 查询，单击“开始跟踪”按钮以初始化 Twitter Archiver。
  </p>
  <p class="graf graf--p">
   在内部，工作表将连接到Twitter，并提取与您的搜索词匹配的历史性推文。它将这些推文写在 Google Spreadsheet 内的单独的表格中。
  </p>
  <p class="graf graf--p">
   存档器将每小时轮询一次 Twitter，并拉入最新发布的匹配推文。
  </p>
  <p class="graf graf--p">
   您可以看上面的视频教程，它非常简单。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    不止推文，Twitter Archiver 应用程序还将导入其他数据，包括转发和收藏计数，发帖者的朋友和关注者计数，以及是否经过验证
   </strong>
   （也就是有没有蓝V）。
  </p>
  <p class="graf graf--p">
   这些数据将帮助您过滤掉垃圾 Twitter 用户，或轻松地从最有影响力的用户那里查看推文。
  </p>
  <p class="graf graf--p">
   如果您想停止存档特定搜索词的推文，请转到 Twitter Archiver，选择“要保存的搜索”菜单，您将看到现有的已保存搜索列表。从下拉列表中选择要删除的对象，然后点击“删除”按钮。
  </p>
  <p class="graf graf--p">
   很简单吧？无需创建任何 Twitter 应用，也无需摆弄Google脚本。
  </p>

  <p class="graf graf--p">
   1、如何创建复杂的搜索查询？
  </p>
  <p class="graf graf--p">
   存档器支持所有 Twitter 搜索运算符。例如，您可以将 min_retweets：5 放在“高级规则”框中，以显示该搜索词的最佳推文。在下面回顾
   <strong class="markup--strong markup--p-strong">
    运算符用法

   ：
  </p>

  <p class="graf graf--p">
   2、为什么 Twitter Archiver 访问各种服务？
  </p>
  <p class="graf graf--p">
   Archiver 附加组件连接到外部服务（Twitter API），并将推文保存到 Google Spreadsheet。另外，如果出现问题，您可以通过电子邮件将调试日志发送给开发人员，因此它需要发送邮件的权限。
  </p>
  <p class="graf graf--p">
   3、隐私政策是什么？
  </p>
  <p class="graf graf--p">
   该加载项在您的 Google 帐户中运行，并直接连接到 Twitter。它甚至不与任何人（包括开发人员）共享一个字节的数据。
  </p>
  <p class="graf graf--p">
   4、我的 Twitter Archiver 没有全面收集所有旧推文，为什么？
  </p>
  <p class="graf graf--p">
   Twitter API 并不提供所有历史推文。它只会返回过去5–7天或最近1000条已发布的推文。但是 Archiver 将获取其跟踪的任何关键字的未来发布的推文。
  </p>
  <p class="graf graf--p">
   5、我是否必须保持工作表打开才能保存推文？
  </p>
  <p class="graf graf--p">
   该存档器在Google服务器上运行，因此即使关闭Google表格也可以正常运行。您可以关闭工作表，关闭计算机，存档器也会继续工作。
  </p>
  <p class="graf graf--p">
   6、免费版和高级版有什么区别？
  </p>
  <p class="graf graf--p">
   Twitter 附加组件的免费版本使您最多可以跟踪2个搜索查询。您可以在同一个Google表格中使用高级版跟踪更多查询。
  </p>
  <p class="graf graf--p">
   免费版每小时对 Twitter 进行一次巡逻，而高级版每15分钟一次，因此它更适合于跟踪产生大量推文的关键字 —— 比如热点事件、热门标签、排名趋势前卫的话题等等。
  </p>
  <p class="graf graf--p">
   7、如何升级到 Twitter Archiver 的高级版？
  </p>
  <p class="graf graf--p">
   您可以使用此
   <a class="markup--anchor markup--p-anchor" data-href="https://www.paypal.com/cgi-bin/webscr?cmd=_s-xclick&amp;hosted_button_id=TGJ3GAFSF62SU" href="https://www.paypal.com/cgi-bin/webscr?cmd=_s-xclick&amp;hosted_button_id=TGJ3GAFSF62SU" rel="noopener noreferrer" target="_blank">
    PayPal 链接
   </a>
   升级到高级版。39.99美元这是一次性固定费用，您可以终身使用。
  </p>
  <p class="graf graf--p">
   8、我需要帮助。我应该联系谁？
  </p>
  <p class="graf graf--p">
   仅高级版提供支持。可以将您的请求发送至
   <a class="markup--anchor markup--p-anchor" data-href="mailto:amit@labnol.org" href="/cdn-cgi/l/email-protection#a8c9c5c1dce8c4c9cac6c7c486c7dacf" rel="nofollow noopener noreferrer" target="_blank">
    <span class="__cf_email__" data-cfemail="8dece0e4f9cde1ecefe3e2e1a3e2ffea">
     [email protected]
    </span>
   </a>
   。
  </p>
  <p class="graf graf--p">
   9、IFTTT 或 Zapier 也可以保存推文、为什么要使用 Twitter Archiver？
  </p>
  <p class="graf graf--p">
   Twitter Archiver 将推文的完整元数据保存在电子表格中，它以更快的速度获取推文，您可以在Google表格中直观地创建和管理搜索查询。
  </p>
  <p class="graf graf--p">
   10、如何使用其他格式保存推文（如 PDF）？
  </p>
  <p class="graf graf--p">
   由于这些推文已保存在 Google 电子表格中，因此您可以轻松将其以各种格式（包括 PDF、CSV）导出，甚至可以将数据集发布为 HTML 网页（选择 File -&gt; Publish to Web inside Google Sheets）。
  </p>
  <p class="graf graf--p">
   高级用户可以尝试使用此技术从 Twitter 创建 JSON 或 RSS Feed。
  </p>
  <p class="graf graf--p">
   不论对于香港那样的大型抗议，还是追踪信息战的动态，都是非常好用的。注意操作安全哦！希望您能找到最高价值的使用方法。⚪️
