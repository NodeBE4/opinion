  <p class="graf graf--p">
   在一次爆炸性的采访中，一名前 ISIS 指挥官承认该恐怖组织多年来一直与土耳其国家情报机构就“共同利益”领域进行合作。
  </p>
  <p class="graf graf--p">
   消息来源说，
   <strong class="markup--strong markup--p-strong">
    土耳其政府高级官员与伊斯兰国代表举行了多次会议，以协调各项活动，这也涉及为该国的外国战斗人员提供支持和安全港。据美国政府反恐顾问采访前 ISIS 官员称，埃尔多安总统“与 ISIS 携手合作”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这种关系引发了人们对土耳其在叙利亚冲突中作为北约盟友角色的质疑。
  </p>
  <p class="graf graf--p">
   作为 ISIS emir 三年的资深人士 Abu Mansour al Maghrebi，接受了国际暴力极端主义研究中心（ICSVE）主任美国政府长期反恐顾问 Anne Speckhard 教授的采访。
  </p>
  <p class="graf graf--p">
   尽管并非所有 Maghrebi 的声明都可以得到证实，但是，
   <strong class="markup--strong markup--p-strong">
    其中大多数声明都得到了其他举报人和前 ISIS 人员的声明的证实，正如之前 INSURGE 报道的那样。
   </strong>
  </p>
  <p class="graf graf--p">
   Speckhard 和 Shakovci 将 Abu Mansour 描述为驻扎在叙利亚 Raqqa 的土耳其 — 伊斯兰国外交官。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “我的问题[职责]是我们[伊斯兰国]与土耳其情报部门的关系。实际上，这是在我在边境工作时开始的，“他解释道
   </em>
   。
  </p>
  <p class="graf graf--p">
   Abu Mansour 来自摩洛哥，是一名电气工程师，于2013年前往叙利亚加入 ISIS。他在该恐怖组织的第一份工作就是涉及处理通过土耳其加入 ISIS 的外国战士。这包括与土耳其境内的伊斯兰国工作人员的网络联络，他们将战斗机从伊斯坦布尔引导到土耳其边境城镇加济安泰普、安塔基亚、桑尼乌法等。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “其中大部分是由 Dawlah [ISIS]支付的，”Abu Mansour 说，
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     虽然他说在土耳其工作的人通常是靠金钱而不是意识形态维系的。但他承认：“土耳其的许多人相信并宣誓效忠于 Dawlah。伊斯兰国的居民生活在土耳其，包括个人和团体，但土耳其境内没有武装团体。
    </em>
    “
   </strong>
  </p>
  <p class="graf graf--p">
   Abu Mansour 后来于2015年前往 Raqqa，在与土耳其国家情报高层会晤后，他为土耳其的 ISIS 战士的医疗服务提供便利。Abu Mansour 称直接接受了伊斯兰国议会议员 Mohamed Hodoud 的命令，并且还曾短暂地会见了恐怖组织难以捉摸的领导人 Abu Bakr al-Baghdadi。他告诉采访他的人：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “土耳其情报部门与伊斯兰国就边境大门、受伤人员等问题达成了一些协议和谅解。我曾与 MIT [土耳其国家情报组织]直接会面，与他们进行过多次会面
   </em>
   。“
  </p>
  <p class="graf graf--p">
   他补充说，这些定期会面发生在一系列机构会面之间，包括土耳其情报部门和土耳其军方：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “有3–5个不同的团队。大多数会议都在土耳其的军事岗位或办公室举行。有时我们每周都见面。这取决于发生了什么。大部分会议都在边境附近，一些在安卡拉，一些在加济安泰普。
   </em>
   “
  </p>
  <p class="graf graf--p">
   Abu Mansour 描述了
   <strong class="markup--strong markup--p-strong">
    在叙利亚和土耳其之间旅行完全无阻碍，导致 Speckhard 形容他实际上是伊斯兰国的“大使
   </strong>
   ”。 “我通过了边界，他们就这么让我通过了”，他说。“[在边境]，土耳其人总是送我一辆车，我受到保护。我们这边有两三个人的团队和我在一起。我大部分时间都在负责我们的团队。“
  </p>
  <p class="graf graf--p">
   尽管 Abu Mansour 否认自己是一个“大人物”，但他承认，
   <strong class="markup--strong markup--p-strong">
    他代表伊斯兰国的影响可能延伸到埃尔多安总统本人：
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “我正准备见到他，但没能见。他的一名情报人员说，埃尔多安希望私下见到你，但事情没有发生。
   </em>
   “
  </p>
  <p class="graf graf--p">
   对 Abu Mansour 的采访于2019年3月18日
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hstoday.us/subject-matter-areas/terrorism-study/the-isis-ambassador-to-turkey/" href="https://www.hstoday.us/subject-matter-areas/terrorism-study/the-isis-ambassador-to-turkey/" rel="noopener noreferrer" target="_blank">
    发布在
   </a>
   Homeland Security Today 上，这是一个政府与技术服务联盟（GTSC）杂志 — — 一个由在美国国家安全部门工作的前美国政府官员组成的行政协会。
  </p>
  <p class="graf graf--p">
   Mansour 认为，
   <strong class="markup--strong markup--p-strong">
    他的角色是协调伊斯兰国与土耳其之间的关系，“让双方都受益。
   </strong>
   ”Abu Mansour 说，
   <strong class="markup--strong markup--p-strong">
    土耳其认为伊斯兰国是扩大土耳其在叙利亚北部作为重建帝国中心影响力的战略工具：
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “我们在边境地区，土耳其希望控制其边界 — 控制北叙利亚。实际上
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     他们不仅有控制库尔德人的野心。他们想要所有的北方，从 Kessab（叙利亚最北端）到摩苏尔……这是埃尔多安的伊斯兰主义者意识形态
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    。他们想要叙利亚北部的所有地方。这就是土耳其方面[想要]控制叙利亚北部的原因，因为他们有真正的野心。实际上，我们谈到了埃尔多安在公开场合所说的内容[与他真正想要的内容]。
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     叙利亚的这一部分是奥斯曼帝国的一部分。在第二次世界大战之后达成协议之前，阿勒颇和摩苏尔是土耳其奥斯曼帝国的领土
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    。在我们的会议中，我们讨论了重建奥斯曼帝国的问题。这是土耳其的愿景。
   </em>
   “
  </p>
  <p class="graf graf--p">
   Abu Mansour 补充说，尽管这一愿景通常归于埃尔多安，但未必在土耳其政府中分享：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “我不能说这是整个土耳其政府的愿景。许多人反对将这个项目变为现实。他们说我们将试图打败库尔德工人党和库尔德人。我们害怕库尔德人之间的联盟，他们可能会成为一个库尔德国家，但他们也扩大到阿勒颇……
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     由于他们是北约国家，他们不能让北约对他们生气。因此，他们不能直接处理这种情况，但是他们想要摧毁库尔德人，因此他们和伊斯兰国一道处理这种情况并从伊斯兰国获得利益。
    </em>
   </strong>
   “
  </p>
  <p class="graf graf--p">
   伊斯兰国认为与土耳其的秘密联盟是一个“大利益”，因为“他们可以保护我们的后院。我们的边界大约有300公里。
   <strong class="markup--strong markup--p-strong">
    土耳其被认为是我们获取药物和食物的必经之路 — — 许多东西以援助的名义进入。大门是敞开的。“
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    伊斯兰国的战斗人员经常在边境的土耳其医院接受治疗。土耳其政府还向恐怖组织供水，并允许其通过土耳其出售数千万美元的石油
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们通过谈判将我们的战斗机送到[土耳其]的医院，”Abu Mansour 说。“有提供便利 — — 他们不要求查看那些即将接受治疗的人的护照。这始终是一扇敞开的大门。如果我们有救护车，我们可以毫无疑问地顺利穿越。我们可以在许多地方穿越土耳其。他们不询问官方身份。“
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    土耳其国家情报局密切参与这一进程
   </strong>
   ，他说：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “MIT 了解到每一个危急情况，他们将救护车送到边境。边境附近也有医院。接收过重症监护的人在那里接受治疗，他们[MIT]根据他们的需要将其他人送到土耳其各地。那里有非常热心的医生，叙利亚人和土耳其人，他们想要帮忙。因此，如果没有设施在边境为他们提供服务，伤员将被送往土耳其。
   </em>
   “
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     医疗费用主要由伊斯兰国支付，但“一些土耳其公立医院免费接收这些伤员
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    。这不仅适用于我们的战士，也适用于爆炸事件的受害者。我不知道有多少人在土耳其接受治疗，但这是惯例 … 我只知道这个协议为我们的伤员打开了大门，并且有救护车护送他们。这是关于我们伤员的“国际间”协议。我参与了这些协议的谈判。为了伤员、医疗和其他物资的通过，我和他们谈判。
   </em>
   “
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    土耳其供应的水源允许 ISIS 日常使用、甚至用水坝发电
   </strong>
   ：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “事实上，我们[叙利亚]与土耳其达成协议，每秒400立方米（水）进入叙利亚。革命后，他们开始将水量减少到每秒150立方米。在我们[2014年]的谈判之后，他们又恢复到了400的供应。我们需要水源用于电力和生活的重要来源。
   </em>
   “
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    据 Abu Mansour 称，伊斯兰国与土耳其的水协议“需要很长时间才能谈判成功”。
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     作为回报，伊斯兰国政府向土耳其政府保证，该国将从伊斯兰国的袭击中“获得安全稳定”。
    </em>
   </strong>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     “在谈判中，我不能说我会攻击土耳其。这是帮派的语言，但我想说我们会努力让土耳其远离战场，我们不会把土耳其视为敌人。
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    他们明白我们在说什么。我们说了很多次
   </em>
   “。
  </p>
  <p class="graf graf--p">
   Abu Mansour 进一步表示，
   <strong class="markup--strong markup--p-strong">
    土耳其是伊斯兰国石油销售的主要渠道：“大部分叙利亚石油都流向土耳其，只有少量石油流入巴沙尔政权…….这是自然发生的。有许多贸易商可以做到这一点，土耳其是唯一可以付得起油钱的市场
   </strong>
   。他们的交易员支付了进入土耳其的石油。“
  </p>
  <p class="graf graf--p">
   这些交易大部分是通过当局批准的土耳其中间人进行的：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “前往叙利亚政府的石油 — 有些是通过管道输送的，有些是通过卡车运送的。 Dawlah [ISIS] 向土耳其运送的石油由来自土耳其的交易员安排的，他们以我们的许可来取石油。交易员也来自叙利亚方面。
   </em>
   “
  </p>
  <p class="graf graf--p">
   Mansour 证实，
   <strong class="markup--strong markup--p-strong">
    通过土耳其的石油销售有助于为伊斯兰国的军事进攻提供资金。 “在叙利亚，石油足以支付武器和所需的一切，
   </strong>
   ”他说。“[我们的石油收入]每月超过1400万美元，这笔石油资金的一半就足以支付我们的武器开支所需的一切。”
  </p>
  <p class="graf graf--p">
   这些说法使人们更加确信早期的
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/insurge-intelligence/britain-s-secret-ties-to-governments-firms-facilitating-isis-oil-sales-210d21470e65" href="https://medium.com/insurge-intelligence/britain-s-secret-ties-to-governments-firms-facilitating-isis-oil-sales-210d21470e65" rel="noopener noreferrer" target="_blank">
    INSURGE 对伊斯兰国石油销售的调查
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    该调查不仅牵出了土耳其共和国的共谋，还牵出了一些伊拉克库尔德组织和西方公司之间的共谋。
   </strong>
  </p>
  <p class="graf graf--p">
   然而，Abu Mansour 否认伊斯兰国直接从土耳其获得武器或资金。相反，他称伊斯兰国通常从武装反对派团体内部获得武器：“反政府的叙利亚人民向我们提供武器; 许多黑手党和团体向我们交易武器。“
  </p>
  <p class="graf graf--p">
   其他消息来源证实了 Abu Mansour 关于土耳其军事情报部门对伊斯兰国直接支持的说法。2016年，我
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/insurge-intelligence/former-turkish-counter-terror-chief-exposes-governments-support-for-isis-d12238698f52" href="https://medium.com/insurge-intelligence/former-turkish-counter-terror-chief-exposes-governments-support-for-isis-d12238698f52" rel="noopener noreferrer" target="_blank">
    采访了
   </a>
   在2010年至2012年期间担任土耳其国家警察反恐和行动司司长的 Ahmet Sait Yayla，他在2014年之前成为公共秩序和预防犯罪司司长。
  </p>
  <p class="graf graf--p">
   Yayla 详细告诉我
   <strong class="markup--strong markup--p-strong">
    他是如何亲眼目睹他自己的警察反恐行动被破败，只因土耳其情报联络人要保护伊斯兰国的战斗人员
   </strong>
   ，他们经常让ISIS战士自由地进出土耳其，并为他们提供医疗服务就在土耳其的医院。
  </p>
  <p class="graf graf--p">
   然而，他更进一步描述了
   <strong class="markup--strong markup--p-strong">
    他是如何看到土耳其直接军事和金融赞助 ISIS 一些行动的证据的
   </strong>
   。Yayla 的详细证词表明，Abu Mansour 作为土耳其情报首席谈判代表的角色并没有涵盖某些关键的战略问题，例如直接的军事和财政支持，这可以解释为什么 Abu Mansour 不知道这一点。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    土耳其法院命令
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://shootingthemessenger.blog/2018/02/19/erdogan-censorship-demand-links-him-to-isis/" href="https://shootingthemessenger.blog/2018/02/19/erdogan-censorship-demand-links-him-to-isis/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     禁止
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    我发布去年发送给美国科技和社交媒体公司的这篇关于 Yayla 的采访。
   </strong>
  </p>
  <p class="graf graf--p">
   INSURGE 此前已经报道了西方情报来源的
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/insurge-intelligence/europe-is-harbouring-the-islamic-state-s-backers-d24db3a24a40" href="https://medium.com/insurge-intelligence/europe-is-harbouring-the-islamic-state-s-backers-d24db3a24a40" rel="noopener noreferrer" target="_blank">
    其他新证据
   </a>
   ，表明土耳其国家同意在叙利亚扩大伊斯兰国的力量。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    尽管国际法要求对被发现支持恐怖主义的实体采取坚决行动，但这些新的揭露强调了为什么西方政府忽视国家支持伊斯兰国的证据 — — 尽管国际法也不例外 。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “土耳其希望外国战士能够轻松过境……事实上，土耳其方面说，’你应该减少、改变你的方式，你进入的方式。例如，不要带一个团体进入，因为有一群人进入这样太明显了。不要携带任何武器。不要带长胡须。你们应该尽可能隐蔽地走从北到南的入口。
   </em>
   ’“
  </p>
  <p class="graf graf--p">
   土耳其情报机构再一次直接参与：“[2014年]，他们打开了一些合法的大门，让我们的人可以自由进出。但是，进入叙利亚比返回土耳其更容易。土耳其控制了这些行动。“
  </p>
  <p class="graf graf--p">
   也许 Abu Mansour 最有争议的说法是
   <strong class="markup--strong markup--p-strong">
    伊斯兰国在土耳其内部 — — 伊斯坦布尔机场、雷纳夜总会、以及安卡拉和伊斯坦布尔街头 — — 的袭击不符合伊斯兰国自己的利益，但很可能是在土耳其情报人员的命令下采取攻击行动的，那些渗透了ISIS的人
   </strong>
   ：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “我认为外部 emni 中有土耳其情报机构MIT的人。我怀疑在机场的袭击不是为了 ISIS 的利益，而是土耳其渗透 ISIS 的群体想要袭击土耳其，或者他们受到其他不希望 Dawlah 和土耳其之间关系太紧密的机构的影响。否则这没有意义，因为我们大多数人都来过那个机场
   </em>
   。“
  </p>
  <p class="graf graf--p">
   他对此的解释是，袭击命令并非来自伊斯兰国的领导，而是来自土耳其情报部门的官员：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     这些袭击土耳其的命令来自 Dawlah 内部的 MIT 人员，但不是来自我们的政治方面。他们不想摧毁埃尔多安，只是想改变他在叙利亚问题上的想法。他们希望他用他的军队攻击叙利亚，攻击 Dawlah。机场袭击是他进入叙利亚的一个很好的借口
    </em>
   </strong>
   。“
  </p>
  <p class="graf graf--p">
   可以肯定的是，没有办法独立核实 Abu Mansour 对土耳其国家情报部门的全部所有指控，但是，这些信息在另一位前ISIS特工萨 Savas Yildas 的供词中
   <strong class="markup--strong markup--p-strong">
    得到了部分证实
   </strong>
   ，他在伊斯兰国对库尔德人的攻击中被 YPG 抓获。Abu Mansour 补充说，在他被关押在库尔德 YPG 监狱期间，他听说“土耳其政府在 Raqqa 事件之后带走了40人，他们是土耳其安全机构的一部分。”
  </p>
  <p class="graf graf--p">
   新的揭露与多年的传统叙事相矛盾，传统叙事将伊斯兰国描绘为在没有重大国家支持的情况下爆发的“自发”运动。
  </p>
  <p class="graf graf--p">
   并且，更重要的是，土耳其并不是西方情报机构知道的为ISIS提供资金的唯一的国家 — — 其他同类国家还包括沙特阿拉伯和卡塔尔。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2016年12月，Wikileaks 提供的证据已经突出了“
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.thepressproject.gr/article/103870/exclusive-wikileaks-documents-relations-erdogan-ISIS-oil-smuggling" href="https://www.thepressproject.gr/article/103870/exclusive-wikileaks-documents-relations-erdogan-ISIS-oil-smuggling" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     埃尔多安与伊斯兰国之间的险恶关系
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ”。
   </strong>
  </p>
  <p class="graf graf--p">
   正如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.thepressproject.gr/article/103870/exclusive-wikileaks-documents-relations-erdogan-ISIS-oil-smuggling" href="https://www.thepressproject.gr/article/103870/exclusive-wikileaks-documents-relations-erdogan-ISIS-oil-smuggling" rel="noopener noreferrer" target="_blank">
    作家 Thanos Kamilialis 所写到的
   </a>
   ，土耳其总统雷杰普·塔伊普·埃尔多加的家人与“伊斯兰国”的石油走私事件被 Wikileaks 揭露：通过土耳其能源部长和埃尔多安的女婿贝拉特阿尔巴伊拉克之间的电子邮件。Albayrak 的电子邮件中显示，能源部长呼吁成为石油公司 Powertrance 的“非官方”所有者，该公司正在从伊拉克北部的伊希斯土地进口石油到土耳其。
  </p>
  <p class="graf graf--p">
   2015年9月底，土耳其黑客组织 Red Hack 声称它可以从 Albayrak 的个人电子邮件帐户访问近 20 GB 的数据。关于电子邮件内容的信息和文章开始上线，但土耳其司法系统决定不发布和复制电子邮件，从而暗示了其真实性。
  </p>
  <p class="graf graf--p">
   土耳其政府 —— 特别是埃尔多安家族成员 —— 在“伊斯兰国”控制地区的石油走私活动中发挥积极作用的最新指控被 Wikileaks 泄漏的土耳其能源部长的 所有电子邮件所证实。
  </p>
  <p class="graf graf--p">
   在土耳其部队于2015年11月24日击落俄罗斯飞机后，对土耳其政府——尤其是阿尔巴伊拉克 —— 的指责变得更加激烈。除了对土耳其实施制裁外，俄罗斯还指责埃尔多安和他的参与石油走私的家庭。
  </p>
  <p class="graf graf--p">
   为了支持这些指控，俄罗斯提供了卫星图像，显示了从伊斯兰国土地到土耳其的石油路线。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    挪威外交部也进行了类似的研究，得出的结论是，从“伊斯兰国”地区运往土耳其的石油以低价出售。与此同时，美国政府也提到过伊斯兰国的石油通过一个过程以土耳其为目的地。
   </strong>
   如果这些指控属实，土耳其总统就发誓要辞职。
  </p>
  <p class="graf graf--p">
   而 Albayrak 的电子邮件确实证明了俄罗斯的指控，以及将他与石油走私联系起来的各种国际媒体特征也是如此。Albayrak 似乎是石油公司 Powertrans 的非官方顾问，根据法律，该公司是唯一允许从土耳其进出石油的石油公司。在他收到的大约32封 Powertrans 电子邮件中，他被问及他对公司未来行动的意见以及他在组织结构、新高管的雇用、和工资等方面的批准。
  </p>
  <p class="graf graf--p">
   正如
   <a class="markup--anchor markup--p-anchor" data-href="http://www.worldpolicy.org/blog/2016/10/17/hacked-emails-link-turkish-minister-illicit-oil" href="http://www.worldpolicy.org/blog/2016/10/17/hacked-emails-link-turkish-minister-illicit-oil" rel="noopener noreferrer" target="_blank">
    World Policy 所分析的那样
   </a>
   ，该公司的股票奇怪地从伊斯坦布尔“旅行”到新加坡，从那里再到达维尔京群岛。公布的信息表明，Powertrans 的真正所有者现在是 Calik Holding，背后是 Albayrak。
  </p>
  <p class="graf graf--p">
   在 Wikileaks 发布的泄露电子邮件中，土耳其能源部长和埃尔多安的女婿与Calik Holding 和 Powertrans 之间的联系似乎相当清楚。几乎每一次他们之间的谈话主题都是 Powertrans，而 Yilmaz 经常要求他批准公司的员工变更，提及例如组织结构图的规划、未来的招聘和工资等。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    伊斯兰国已经 — 部分或全部 — 控制了重要的油井，两年多来，石油走私是这个恐怖主义集团的基本资金来源
   </strong>
   。 2014年，“卫报”展示过从伊斯兰国到土耳其、伊朗和约旦的石油运输路线图。
  </p>
  <p class="graf graf--p">
   2015年10月，
   <a class="markup--anchor markup--p-anchor" data-href="http://ig.ft.com/sites/2015/isis-oil/" href="http://ig.ft.com/sites/2015/isis-oil/" rel="noopener noreferrer" target="_blank">
    英国“金融时报”透露
   </a>
   ，伊斯兰国的石油平均产量为每天 34,000–40,000 桶，每桶售价为20–45美元。这意味着他们每天从石油中获得150万美元的收入，用于资助他们在军事和恐怖主义行动中的战斗人员。
  </p>
  <p class="graf graf--p">
   从ISIS控制的油井到土耳其的石油路线由俄罗斯于2015年12月公布的卫星照片显示。据俄罗斯称，石油是通过第三条路线运输的：从叙利亚东部和伊拉克西部到土耳其东南角。
  </p>
  <p class="graf graf--p">
   在这里看到视频：
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="393" src="//www.youtube.com/embed/sH9oB1YChHs" width="700">
   </iframe>
  </p>
  <p class="graf graf--p">
   国际媒体提供的照片和文件显示，伊斯兰国的石油最终出现在土耳其的各个地区，唯一一家基于土耳其的公司可以实现这种运输的就是 Powertrans。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.zerohedge.com/news/2019-04-10/erdogan-worked-hand-glove-isis-syria-claims-former-emir" href="https://www.zerohedge.com/news/2019-04-10/erdogan-worked-hand-glove-isis-syria-claims-former-emir" rel="noopener noreferrer" target="_blank">
    Inan explosive interview, a former ISIS commander has claimed that the terror group cooperated directly with Turkish state intelligence agencies for years on areas of “common interest
   </a>
   ”.
  </p>
