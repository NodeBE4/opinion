title: 英国发布香港报告：“一国两制”在港分化
---

<div id="entry">
<div class="at-above-post addthis_tool" data-url="http://zhanlve.org/?p=6974">
</div>
<p>
<img alt="" class="wp-image-6975 size-medium aligncenter" height="169" sizes="(max-width: 300px) 100vw, 300px" src="http://zhanlve.org/wp-content/uploads/2019/03/AP_17183212738034-300x169.jpg" srcset="http://zhanlve.org/wp-content/uploads/2019/03/AP_17183212738034-300x169.jpg 300w, http://zhanlve.org/wp-content/uploads/2019/03/AP_17183212738034-768x432.jpg 768w, http://zhanlve.org/wp-content/uploads/2019/03/AP_17183212738034-1024x576.jpg 1024w, http://zhanlve.org/wp-content/uploads/2019/03/AP_17183212738034.jpg 1400w" width="300"/>
</p>
<p>
</p>
<p>
<span style="font-size: 12pt;">
   英国政府周三发布最新一期《香港问题半年报告》。报告指出，香港近年来无疑正与大陆融合，这一方面能够提振区域经济；但另一方面，却对香港公民的人身自由造成了压力。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   这份盘点去年下半年的《香港问题半年报告》（Six-monthly reports on Hong Kong）是自1997年，香港主权移交中国后，英国政府发布的第44份报告。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   报告指出，“一国两制”模式在香港正出现分化。一方面，在商业和司法独立领域，“一国两制”运行良好；但另一方面，港人的基本自由却在被大陆侵蚀。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   英国外交大臣侯俊伟（Rt Hon. Jeremy Hunt MP）在报告序言中指出：“近期北京当局对香港施压，令香港公民自由及政治自由的尺度向大陆靠拢，包括一些话题实际上被禁止讨论和辩论。我对此，表示担忧。”
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   报告罗列自去年，引发媒体广泛关注的四大事件：
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   首先，去年9月，港府宣布正式禁止“香港民族党”运作。民族党于2016年成立，主张“民族自强、香港独立”。英国外交部随后发表声明对此表示担忧，并指出，这是香港回归后首个基于《社团条例》被禁的政党。声明强调，虽然英国不支持香港独立，但香港的高度自治及其享有的权利与自由，是香港赖以生存的根基，需要被充分尊重。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   其次，由于香港外国记者会去年邀请了“香港民族党”召集人陈浩天到场演讲。港府在去年10月，拒绝了记者会副主席、《金融时报》亚洲总编马凯（Victor Mallet）延长工作签证的申请。舆论普遍认为，这是当局对马凯主持演讲活动的报复。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   此外，香港高院在2017年认定，民主派议员刘小丽在就任宣誓就言行不庄重，被裁定失去议席。而当她去年底参选立法会地方选区补选时，却被选举主任取消了参选资格。理由是她支持香港自决，与“一国两制”不相容。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   最后，2014年“占领中环”运动的三名发起人戴耀廷、陈健民与朱耀明于去年11月，正式受审。他们因发起大规模街头抗议被控“公众妨扰罪”。如果罪成，三人面临最高7年监禁。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   报告结尾说，“一国两制”给香港和大陆带来了福祉。但为了保障其有效延续，当局必须尊重并维护港人的言论自由、新闻自由和结社自由。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   港府和中国外交部随后对上述报告提出抗议，强调外国政府不应以任何形式，干预香港内政。
  </span>
<br/>
<span style="font-size: 12pt;">
</span>
<br/>
<span style="font-size: 12pt;">
</span>
</p>
<p>
<span style="font-size: 12pt;">
   出处 ：RFA
