   </noscript>
  </p>
  <p class="graf graf--p">
   人们称它为“Silent Talker”。
  </p>
  <p class="graf graf--p">
   它是一个虚拟警察，现在开始如果您想来欧洲玩，必须经过这么一个新门槛 —— 在被允许入境之前，旅行者必须通过这个虚拟警察进行的测谎测试。
  </p>
  <p class="graf graf--p">
   过程是这样的。
  </p>
  <p class="graf graf--p">
   在您抵达机场之前，使用您自己的计算机登录到一个网站，在那里上传您的护照照片，你会看到一个穿着深蓝色制服的棕色头发动画男出现在你的屏幕上，就是那个虚拟警察。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “你姓什么？”；“你的公民身份和旅行目的是什么？”……“他”会问你一系列问题。
  </p>
  <p class="graf graf--p">
   您只需要口头回答就行了。虚拟警察
   <strong class="markup--strong markup--p-strong">
    使用您的设备上网络摄像头
   </strong>
   扫描您的面部和眼睛的运动，以寻找“说谎的迹象”。
  </p>
  <p class="graf graf--p">
   在测谎问答结束时，系统会为您提供一个QR码，您必须在抵达边境时向警卫展示。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    警卫使用手持平板设备扫描那个QR码，拍摄您的指纹，并获取您的面部图像，以检查它是否与您的护照相符
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果警卫手里的平板电脑显示100分，那就意味着机器告诉人类：你“是真实的”。
  </p>
  <p class="graf graf--p">
   如果你被判定为试图欺骗该系统，你就麻烦了，你会被归类为“高风险”或“中等风险”，这取决于机器发现你“错误回答问题”的数量。
  </p>
  <p class="graf graf--p">
   Intercept 的记者在今年早些时候经过塞尔维亚 — 匈牙利边境之前对该系统进行测试 —— ⚠️ 请注意，记者对机器提出的所有问题都提供了诚实的回答，但是，记者被机器视为骗子，16个问题中有4个错误答案，得分为48分。
  </p>
  <p class="graf graf--p">
   评估测谎仪结果的匈牙利警察表示，该系统建议记者应该接受进一步的检查；尽管最终并没有什么“进一步”。
  </p>
  <p class="graf graf--p">
   被视为“危险”的旅行者很可能会被拒绝入境，但在大多数情况下，旅行者永远不会知道那个愚蠢的测谎是否是他们被拒绝入境的原因之一。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    因为测试结果通常不会透露给旅行者
   </strong>
   ；只有在根据欧洲隐私法提交数据访问请求后，Intercept 才获得了记者的测试副本。
  </p>
  <figure aria-describedby="caption-attachment-7064" class="wp-caption aligncenter" id="attachment_7064" style="width: 1000px">
   <img alt="" class="size-full wp-image-7064 jetpack-lazy-image" data-lazy-sizes="(max-width: 1000px) 100vw, 1000px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-16.jpg?resize=1000%2C907&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-16.jpg?w=1000&amp;ssl=1 1000w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-16.jpg?resize=300%2C272&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-16.jpg?resize=768%2C697&amp;ssl=1 768w" data-recalc-dims="1" height="907" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-16.jpg?resize=1000%2C907&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1000"/>

  <p class="graf graf--p">
   这个虚拟警察是
   <strong class="markup--strong markup--p-strong">
    一个名为 iBorderCtrl 的项目的产物，该项目涉及匈牙利、拉脱维亚和希腊的维稳机构
   </strong>
   。
  </p>
  <p class="graf graf--p">
   该测谎仪的试验阶段已经结束，可能很快会在其他欧盟国家推出，这一潜在的发展引起了全欧洲的争议和媒体报道。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    IBorderCtrl 的这个测谎系统是由英国曼彻斯特城市大学的研究人员在英国开发的
   </strong>
   ，他们说这项技术可以通过让人们在计算机上回答问题，分析他们的面部表情、眼神和姿势的微观细节，判断谎言。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    欧盟的一项研究计划已为该项目提供了约450万欧元的资金
   </strong>
   ，该项目由13个合作伙伴组成，包括希腊安全研究中心、德国汉诺威大学、以及一堆监视技术公司，如匈牙利的 BioSec、西班牙的 Everis、和波兰的 JAS。
  </p>
  <p class="graf graf--p">
   曼彻斯特城市大学的研究人员认为，该系统可以代表“边检的未来”。
  </p>
  <p class="graf graf--p">
   在2018年6月发表的一篇学术论文中，该高校的研究人员表示，像他们的虚拟警察这样的玩意 “将适合在过境检查中检测欺骗，因为
   <strong class="markup--strong markup--p-strong">
    它们是人类信息的有效提取者
   </strong>
   ”。
  </p>
  <p class="graf graf--p">
   然而，一些学者质疑该系统的价值，他们说这完全依赖于
   <strong class="markup--strong markup--p-strong">
    伪科学
   </strong>
   来做出关于旅行者是否诚实的判定。
  </p>
  <p class="graf graf--p">
   德比大学刑事调查教授 Ray Bull 曾帮助英国警方采取更有效的审讯技巧，他专门研究欺骗行为。他告诉 The Intercept，iBorderCtrl 这个项目“不可信”，因为
   <strong class="markup--strong markup--p-strong">
    没有证据
   </strong>
   表明监测人脸上的微表情是衡量说谎的准确方法。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “他们欺骗自己认为这东西会变得非常有效，而且他们已经浪费了很多钱”
   </strong>
   ，Bull 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这项技术基于对判断人类的真实或欺骗性行为特征的基本误解。”
  </p>
  <p class="graf graf--p">
   近年来，在难民危机的情况下、以及法国、比利时、西班牙和德国发生的一系列恐怖袭击之后，欧洲的警察和维稳机构面临着越来越大的政治压力，需要更有效地追踪移民的流动情况。
  </p>
  <p class="graf graf--p">
   非洲大陆的边境安全官员表示，他们正在努力寻找更快、更有效的新方法，
   <strong class="markup--strong markup--p-strong">
    利用人工智能，检查每年进入欧盟的7亿多人的旅行证件和生物识别数据。
   </strong>
  </p>
  <p class="graf graf--p">
   欧盟的行政部门为2021年～2027年间预留了
   <a class="markup--anchor markup--p-anchor" data-href="http://europa.eu/rapid/press-release_IP-18-4106_en.htm" href="http://europa.eu/rapid/press-release_IP-18-4106_en.htm" rel="noopener noreferrer" target="_blank">
    349 亿欧元
   </a>
   用于边境管制和移民管理。
  </p>
  <p class="graf graf--p">
   与此同时，去年9月，欧洲立法者同意建立一个新的自动化系统，以监测所有游客，包括来自美国在内的免签第三国的国民，以确定是否可允许他们进入欧洲。
  </p>
  <p class="graf graf--p">
   将来，无论出于何种原因无法提前提交申请的免签证旅行者都将无法进入申根区；申根区覆盖欧洲26个国家，旅客可以自由地跨境旅行无需护照。
  </p>
  <p class="graf graf--p">
   IBorderCtrl 这种虚拟警察是一种旨在加强预检查筛选过程的技术。
  </p>
  <p class="graf graf--p">
   但透明度活动家表示，在获得有关该技术的更多信息之前 —— 例如用于制定决策的算法，不应推出这一项目 。
  </p>
  <p class="graf graf--p">
   今年早些时候，位于米兰的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hermescenter.org/" href="https://www.hermescenter.org/" rel="noopener noreferrer" target="_blank">
    Hermes 透明度和数字人权中心
   </a>
   的研究人员利用信息自由法获取了有关该系统的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/public/search/projectid:41912-iBorderCtrl" href="https://www.documentcloud.org/public/search/projectid:41912-iBorderCtrl" rel="noopener noreferrer" target="_blank">
    内部文件
   </a>
   。
  </p>
  <p class="graf graf--p">
   他们收到了数百页文件；然而，
   <strong class="markup--strong markup--p-strong">
    这些文件被大量编辑过，许多页面完全被涂黑了
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    美国的信息资源法案请求经常遭遇这种篡改，
    <strong class="markup--strong markup--li-strong">
     美国的活动家想出了一个办法可以将文件恢复原样
    </strong>
    ，见《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e9%ab%98%e6%98%8e%e7%9a%84%e5%8f%8d%e6%8a%97/" href="https://www.iyouport.org/%e9%ab%98%e6%98%8e%e7%9a%84%e5%8f%8d%e6%8a%97/" rel="noopener noreferrer" target="_blank">
     高明的反抗
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   研究员 Riccardo Coluccini 说：“他们居然通过隐瞒解决这些问题的文件来压制辩论，真的很可怕。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “理解资助过程背后的理由是绝对必要的。这些文件中写的是什么？欧盟如何证明使用这种伪科学技术的合理性？”
  </p>
  <p class="graf graf--p">
   曼彻斯特研究人员进行的一项研究用 iBorderCtrl 测试了32个人，并说他们的结果表明该系统的准确率为75％。
  </p>
  <p class="graf graf--p">
   然而，研究人员指出，参与实验的标本群体在种族和性别方面存在不平衡，亚洲人或阿拉伯人参与者的人数明显少于欧洲白人、女性人数少于男性。
  </p>
  <p class="graf graf--p">
   欧盟数据保护监管机构负责人 Giovanni Buttarelli 告诉 The Intercept，他担心这个虚拟警察 iBorderCtrl 系统可能产生基于种族原因的歧视。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “谁来设定参数以决定是不是撒谎？” Buttarelli 说。
  </p>
  <p class="graf graf--p">
   iBorderCtrl 的发言人拒绝回答相关问题。
  </p>
  <p class="graf graf--p">
   该项目的网站承认测谎系统将“对旅行者的基本权利产生影响”，但它还为自己狡辩称，由于该系统目前是测试阶段是自愿的，因此“歧视、人类尊严等问题不会发生 ”…… ⚪️
  </p>
