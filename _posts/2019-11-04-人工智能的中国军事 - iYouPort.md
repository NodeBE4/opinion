  <p class="graf graf--p">
   中国军队正在重点关注人工智能。
  </p>
  <p class="graf graf--p">
   然而，中国为军方开发“更智能”、更便宜的技术的竞赛并不是线性的，而是涉及中央政府、国内公司和国际贸易的多管齐下的战略。
  </p>
  <p class="graf graf--p">
   华盛顿智库“新美国安全中心”的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.linkedin.com/in/gregallen2/" href="https://www.linkedin.com/in/gregallen2/" rel="noopener noreferrer" target="_blank">
    Gregory Allen
   </a>
   发表了一份关于中国人工智能战略的报告，他说：
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    中国军方领导人越来越多地将智能或“智能化”军事技术称为对未来战争基础的期望。使用“智能化”一词意味着基于信息技术的军事新阶段。
   </p>
  </blockquote>
  <p class="graf graf--p">
   他还报告称，“中国国家和地方政府在实施这些计划方面的人力资源支出总额尚未公开披露，但显然
   <strong class="markup--strong markup--p-strong">
    已达数百亿美元
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   中国似乎打算出售其建立的人工智能技术，并且正在进行让人工智能相关行业取得商业成功的计划。
  </p>
  <p class="graf graf--p">
   牛津大学人工智能项目治理负责人 Jeffrey Ding 在他的“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.fhi.ox.ac.uk/wp-content/uploads/Deciphering_Chinas_AI-Dream.pdf" href="https://www.fhi.ox.ac.uk/wp-content/uploads/Deciphering_Chinas_AI-Dream.pdf" rel="noopener noreferrer" target="_blank">
    解读中国的AI梦
   </a>
   ”论文中表示，到2020年，中国的核心人工智能行业总产出和人工智能相关的行业总产值可分别
   <strong class="markup--strong markup--p-strong">
    超过225亿美元和1508亿美元
   </strong>
   ，使中国成为该领域最先进的国家之一。
  </p>
  <p class="graf graf--p">
   将其AI输出商业化可能会为其军队带来好处。Allen 称：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “中国的商业市场已经成功与中国的国家安全欲求直接相关，不仅降低了美国政府对中国施加外交和经济压力的能力，而且，
   <strong class="markup--strong markup--p-strong">
    增加了中国军事和情报界的技术能力
   </strong>
   ”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    中国的国家人工智能战略得到了私营部门的大力支持 —— 即
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%86%9b%e4%ba%8b%e5%b7%a5%e4%b8%9a%e7%bb%bc%e5%90%88%e4%bd%93/" href="https://www.iyouport.org/category/%e5%86%9b%e4%ba%8b%e5%b7%a5%e4%b8%9a%e7%bb%bc%e5%90%88%e4%bd%93/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     军事工业综合体
    </strong>
   </a>
   。UNICRI 人工智能和机器人中心负责人
   <a class="markup--anchor markup--p-anchor" data-href="https://emerj.com/ai-podcast-interviews/the-strengths-of-the-ai-ecosystem-in-china-perspectives-from-a-un-leader/" href="https://emerj.com/ai-podcast-interviews/the-strengths-of-the-ai-ecosystem-in-china-perspectives-from-a-un-leader/" rel="noopener noreferrer" target="_blank">
    Irakli Beridze 说
   </a>
   ：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “政府、私营部门和学术界之间的联盟确实促进了统一的愿景，他们认为这种愿景实际上可以为中国经济以及发展和创新带来积极的结果。”
  </p>
  <p class="graf graf--p">
   在人工智能方面，中国已经将自己视为可以迅速缩小与美国之间的差距。Allen 说出了一个关键问题：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “通过进入全球技术研究和市场，中国取得了成功。许多看似是“中国”的人工智能成就实际上是跨国研究团队和公司的成就，这种国际合作对中国的研究进展至关重要。”
  </p>
  <p class="graf graf--p">
   简单说，
   <strong class="markup--strong markup--p-strong">
    没有南方国家的大力“帮助”，中国的技术水平很可能不会有今天的高度。
   </strong>
  </p>
  <p class="graf graf--p">
   延伸《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a1%85%e8%b0%b7%e4%ba%89%e5%a4%ba%e4%b8%ad%e5%9b%bd%ef%bc%9a%e8%a5%bf%e6%96%b9%e7%a7%91%e6%8a%80%e5%af%a1%e5%a4%b4%e5%9c%a8%e4%b8%9c%e6%96%b9%e5%9c%86%e5%bd%a2%e7%9b%91%e7%8b%b1%e7%9a%84%e6%8a%95/" href="https://www.iyouport.org/%e7%a1%85%e8%b0%b7%e4%ba%89%e5%a4%ba%e4%b8%ad%e5%9b%bd%ef%bc%9a%e8%a5%bf%e6%96%b9%e7%a7%91%e6%8a%80%e5%af%a1%e5%a4%b4%e5%9c%a8%e4%b8%9c%e6%96%b9%e5%9c%86%e5%bd%a2%e7%9b%91%e7%8b%b1%e7%9a%84%e6%8a%95/" rel="noopener noreferrer" target="_blank">

   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%80%e8%87%82%e4%b9%8b%e5%8e%89%ef%bc%9a%e7%be%8e%e5%9b%bd%e4%bc%81%e4%b8%9a%e6%ad%a3%e5%9c%a8%e5%b8%ae%e5%8a%a9%e5%bb%ba%e7%ab%8b%e4%b8%ad%e5%9b%bd%e7%9a%84%e5%a5%a5%e5%a8%81%e5%b0%94%e5%9b%bd/" href="https://www.iyouport.org/%e4%b8%80%e8%87%82%e4%b9%8b%e5%8e%89%ef%bc%9a%e7%be%8e%e5%9b%bd%e4%bc%81%e4%b8%9a%e6%ad%a3%e5%9c%a8%e5%b8%ae%e5%8a%a9%e5%bb%ba%e7%ab%8b%e4%b8%ad%e5%9b%bd%e7%9a%84%e5%a5%a5%e5%a8%81%e5%b0%94%e5%9b%bd/" rel="noopener noreferrer" target="_blank">

   </a>

  <p class="graf graf--p">
   中国对“智能军事”的积极追求令分析人士担心，它可能引发人工智能军备竞赛，不仅涉及美国，还涉及其邻国，尤其是日本。
  </p>
  <p class="graf graf--p">
   Allen 表示，“人工智能系统的使用增加会导致误解和无意的冲突升级，这可能是由于缺乏有关使用此类系统的明确规范。”
  </p>
  <p class="graf graf--p">
   也就是说，中国仍然有办法克服本国人工智能生态系统的固有问题。
  </p>
  <p class="graf graf--p">
   其中包括人才缺乏、核心技术滞后、资金不平等、市场竞争、军备控制问题以及当前的外部依赖。
  </p>
  <p class="graf graf--p">
   中国军方在人工智能技术方面究竟获得了什么？本文将分别讨论中国军队的三个人工智能用例：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    自动驾驶汽车
   </li>
   <li class="graf graf--li">
    面部识别
   </li>
   <li class="graf graf--li">
    5G
   </li>
  </ul>
  <p class="graf graf--p">
   分析始于中国对自动驾驶汽车和无人机的大笔投资。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    自动驾驶汽车和无人机
   </strong>
  </p>
  <p class="graf graf--p">
   根据
   <a class="markup--anchor markup--p-anchor" data-href="https://nationalinterest.org/blog/buzz/really-big-deal-china-drone-superpower-47692" href="https://nationalinterest.org/blog/buzz/really-big-deal-china-drone-superpower-47692" rel="noopener noreferrer" target="_blank">
    The National Interest
   </a>
   ，中国是无人作战飞行器（UCAVs）的最大出口国，也被称为战斗无人机。
  </p>
  <p class="graf graf--p">
   斯德哥尔摩国际和平研究所（SIPRI）报告：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “有关UCAV扩散对和平与安全的影响的讨论很广泛。
   <strong class="markup--strong markup--p-strong">
    中国已成为 UCAV 的主要出口国
   </strong>
   。中国在2009–13年间向2个国家出口了10架UCAV，而在2014年至2018年，向13个国家出口了153架 —— 其中5个出口方在中东：埃及、伊拉克、约旦、沙特阿拉伯和阿联酋”。
  </p>

   <strong class="markup--strong markup--p-strong">
    面部识别
   </strong>
  </p>
  <p class="graf graf--p">
   中国军队中肆无忌惮地使用人工智能似乎具有重大价值。
  </p>
  <p class="graf graf--p">
   然而，这并没有阻止美国私营公司与中国研究人员合作开发基于人工智能的技术。详细《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a1%85%e8%b0%b7%e5%b7%a8%e5%a4%b4%e4%b8%ba%e4%b8%ad%e5%9b%bd%e5%86%9b%e6%96%b9%e6%9c%8d%e5%8a%a1%ef%bc%8c%e8%bf%99%e4%ba%8b%e4%bd%a0%e6%80%8e%e4%b9%88%e7%9c%8b/" href="https://www.iyouport.org/%e7%a1%85%e8%b0%b7%e5%b7%a8%e5%a4%b4%e4%b8%ba%e4%b8%ad%e5%9b%bd%e5%86%9b%e6%96%b9%e6%9c%8d%e5%8a%a1%ef%bc%8c%e8%bf%99%e4%ba%8b%e4%bd%a0%e6%80%8e%e4%b9%88%e7%9c%8b/" rel="noopener noreferrer" target="_blank">
    “硅谷巨头为中国军方服务”，这事你怎么看
   </a>
   》。
  </p>
  <p class="graf graf--p">
   其中之一是微软，微软与中国军方资助的国防科技大学（NUDT）合作开展监控技术研究。
  </p>
  <p class="graf graf--p">
   另一个是谷歌，谷歌于2017年在北京开设了人工智能研究设施。该实验室开发人工智能和机器学习工具，包括开源平台
   <a class="markup--anchor markup--p-anchor" data-href="https://www.tensorflow.org/" href="https://www.tensorflow.org/" rel="noopener noreferrer" target="_blank">
    TensorFlow
   </a>
   。
  </p>
  <p class="graf graf--p">
   这些公司对其技术的最终使用方法不承担任何责任。前谷歌云首席执行官甚至
   <a class="markup--anchor markup--p-anchor" data-href="https://www.blog.google/products/google-cloud/incorporating-googles-ai-principles-google-cloud/" href="https://www.blog.google/products/google-cloud/incorporating-googles-ai-principles-google-cloud/" rel="noopener noreferrer" target="_blank">
    直接表示
   </a>
   ，“我们无法控制我们技术的所有下游用途。”
  </p>
  <p class="graf graf--p">
   这正是与中国合作的问题。虽然这类研究的结果可供公众使用，但中国可以使用这些技术做一切，而没有顾虑。
  </p>
  <p class="graf graf--p">
   例如，来自顶级AI公司 SenseTime 和 Megvii 的面部识别技术被中国部分地区的执法部门广泛使用，特别是新疆、陕西和深圳。

  </p>
  <p class="graf graf--p">
   而
   <strong class="markup--strong markup--p-strong">
    这些技术的最终用途是军方
   </strong>
   。

  </p>
  <p class="graf graf--p">

  </p>

    常态

   <strong class="markup--strong markup--p-strong">
    5G
   </strong>
  </p>
  <p class="graf graf--p">
   数据访问和分析的相关问题是数据传输。
  </p>
  <p class="graf graf--p">
   虽然开发和获取功能强大的人工智能软件和算法无疑是中国军事人工智能战略的重中之重，但建立快速可靠的传输数据和连接设备的方式可能更为重要。
  </p>
  <p class="graf graf--p">
   延伸《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%ba%e5%b7%a5%e6%99%ba%e8%83%bd%e6%96%b0%e5%86%b7%e6%88%98%e2%80%8a-%e2%80%8a-ai%e5%92%8c5g%e5%a6%82%e4%bd%95%e5%8f%98%e6%88%90%e4%b8%80%e5%9c%ba%e5%99%a9%e6%a2%a6/" href="https://www.iyouport.org/%e4%ba%ba%e5%b7%a5%e6%99%ba%e8%83%bd%e6%96%b0%e5%86%b7%e6%88%98%e2%80%8a-%e2%80%8a-ai%e5%92%8c5g%e5%a6%82%e4%bd%95%e5%8f%98%e6%88%90%e4%b8%80%e5%9c%ba%e5%99%a9%e6%a2%a6/" rel="noopener noreferrer" target="_blank">

   </a>
   》。

  <p class="graf graf--p">
   根据前美国空军准将 Robert Spalding 的说法，如果中国成功地主导第五代（5G）移动通信，中国就可以
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%ad%a6%e5%99%a8%e5%8c%96%e5%9f%8e%e5%b8%82/" href="https://www.iyouport.org/%e6%ad%a6%e5%99%a8%e5%8c%96%e5%9f%8e%e5%b8%82/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     “武装城市”
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p">
   根据美国军方前领导人发表的公开声明，“中国设计的5G网络将向中国提供近乎持久的数据传输能力，中国政府可以随意捕获。”
  </p>
  <p class="graf graf--p">
   这是建立稳定的5G网络的激烈竞争的主要原因，尽管专家认为这种基础设施
   <strong class="markup--strong markup--p-strong">
    更容易受到黑客攻击
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但显然，中美都不在乎这点，高堡奇人追求的只是表面“优势”。
  </p>
  <p class="graf graf--p">
   主导5G网络的军备竞赛可能会对任何军队中使用AI的方式产生深远的影响。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    结论性思考
   </strong>
  </p>
  <p class="graf graf--p">
   人工智能在中国军方中的应用揭示了该技术的不确定性和破坏性。
  </p>
  <p class="graf graf--p">
   随着人工智能改变游戏规则，常规战争在今天的战场上几乎没有什么地位，中国正在充分利用这点。
  </p>
  <p class="graf graf--p">
   Allen 表示，中国是人工智能相关领域的领导者，包括研究论文、专利数量和风险投资。
  </p>
  <p class="graf graf--p">
   然而，
   <strong class="markup--strong markup--p-strong">
    值得注意的是，中国能获取这种至高无上地位的主要原因在于：中国一直在获得外国资金和技术研究
   </strong>
   。
  </p>
  <p class="graf graf--p">
   此外，AI 是双刃剑。关于人工智能和机器学习技术还有很多未知之处。
  </p>
  <p class="graf graf--p">
   中国对人工智能优势之速度的追求，几乎没有时间进行准确性和可靠性的测试，一旦发生问题可能会非常严重。⚪️
