  <p class="graf graf--p">
   The Intercept 披露，Google 执行官正在对中国的审查版搜索引擎进行秘密的内部评估。
  </p>
  <p class="graf graf--p">
   参与评估的是一小部分高层管理人员，该项目称之为 Dragonfly，旨在将有关人权、民主、宗教、和平抗议等一系列信息列入黑名单。
  </p>
  <p class="graf graf--p">
   关于谷歌的分析详见 IYP 分类知识列表2个中的“关于谷歌”部分。
  </p>
  <p class="graf graf--p">
   Google 每年都会进行绩效评估，以评估员工的产出和发展。这些评估通常以开放的同行评审方式进行：工作人员对彼此的项目进行评分，然后由管理层对结果进行评估，如果员工被认为已经准备好了在公司取得进步，他们就可以奖励员工。
  </p>
  <p class="graf graf--p">
   然而，在 Dragonfly 项目的情况下，同行评审方面已被删除，
   <strong class="markup--strong markup--p-strong">
    完全颠覆了正常程序
   </strong>
   。在一项被两个谷歌消息来源描述为极不寻常的行动中，高管们成立了一个独立的“审查委员会”小组，由高级管理人员组成，他们之前都被告知过中国的审查版搜索引擎。
  </p>
  <p class="graf graf--p">
   蜻蜓审查委员会的存在尚未向谷歌员工披露，除了极少数从事开发审查版搜索引擎的工作人员。据说该公司不到十几名高层管理人员参与了评审，其中涉及研究与 Dragonfly 相关的文件和技术工作。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “管理层决定承诺保密这些东西，”一位了解该评审的消息人士表示。他们“从[员工]审核工具中获取任何 Dragonfly 专用文件，因此此次审核仅由 Dragonfly 项目委员会决定。”
  </p>
  <p class="graf graf--p">
   据两位谷歌消息人士透露，高管们可能担心，如果对 Dragonfly 进行正常的、更开放的绩效评估过程，公司的员工就会了解该项目的内幕。
  </p>
  <p class="graf graf--p">
   据两位消息人士透露，如果有关 Dragonfly 的一些文件在公司内部得到更广泛的访问，可能会引发关于该项目的进一步争议，该项目在 The Intercept 首次曝光后引发了内部激烈的抗议和辞职潮。
  </p>
  <p class="graf graf--p">
   然而，秘密进行审查的决定本身可能会引起公司内部的愤怒。在去年对 Dragonfly 的抗议活动中，员工的一个主要抱怨是，中国计划缺乏透明度，违背了公司传统的开放式工作场所文化。在公开曝光之前，关于 Dragonfly 的知情仅限于 Google 的88,000名员工中的几百人了解 — — 约占员工总数的0.35％。
  </p>
  <p class="graf graf--p">
   面对来自公司内部和外部的压力，谷歌首席执行官桑达皮采在八月危机会议上告诉他的工作人员，他们“肯定会透明[关于蜻蜓项目]，我们当然计划更多地参与并多表达。“
  </p>
  <p class="graf graf--p">
   但是，谷歌员工本周告诉 The Intercept，
   <strong class="markup--strong markup--p-strong">
    公司老板一直拒绝向他们提供有关 Dragonfly 的信息 — — 让他们对这个项目的状态以及公司对中国的更广泛计划一无所知
   </strong>
   。
  </p>
  <p class="graf graf--p">
   去年年底，在激烈的批评声中，谷歌高管们让工程师远离了审查版搜索引擎的工作，并公开表示目前没有计划推出它。然而，三月早些时候，The Intercept 继续透露，一些谷歌员工担心审查版搜索引擎的工作仍在继续，因为该平台的部分内容似乎仍处于开发阶段。谷歌随后否认蜻蜓仍在进行中，坚持声称“没有对这样的项目进行任何工作。团队成员已搬到新项目。“
  </p>
  <p class="graf graf--p">
   谷歌此前曾于2006年在中国推出搜索引擎，但在2010年退出该国。当时谷歌联合创始人谢尔盖布林表示，停止在该国开展搜索业务的决定主要是“反对审查制度并为政治异议的自由辩护”。
  </p>
  <p class="graf graf--p">
   蜻蜓代表了这一立场的戏剧性逆转。谷歌计划推出的搜索引擎作为 Android 和 iOS 设备的应用程序，旨在遵守中国执政的共产党政权制定的严格审查规则，监控人们的搜索行为，同时还阻止数千个概念，如“诺贝尔奖”，“”人权“和”学生抗议“等。
  </p>
  <p class="graf graf--p">
   超过60个人权组织和22个美国立法者写信给谷歌批评该项目。今年2月，国际特赦组织与谷歌会面，重申了其对中国计划的担忧。“围绕蜻蜓项目的发展缺乏透明度是非常令人不安的，”国际特赦组织研究员 Anna Bacciarelli 说，“我们继续呼吁谷歌首席执行官桑达皮采公开证实它放弃 Dragonfly，而不仅仅是’暂时的搁置并秘密进行’。” ⚪️
  </p>
