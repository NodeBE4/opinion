     <em>
      消费者应该支持开源自由软件和数据管理透明化，及相关完备的法律保障。否则，就不是科技进步，而是 1984 还魂
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p graf-after--figure" id="4ab0">
   搜索是谷歌唯一明确的胜利，也是其主要的收入来源，因此当亚马逊
   <a class="markup--anchor markup--p-anchor" data-href="http://www.geekwire.com/2017/amazon-continues-grow-lead-google-starting-point-online-shoppers/" href="https://www.geekwire.com/2017/amazon-continues-grow-lead-google-starting-point-online-shoppers/" rel="noopener nofollow noreferrer" target="_blank">
    迅速超越
   </a>
   谷歌作为顶级产品搜索目的地时，谷歌的基础开始动摇。正如当时
   <a class="markup--anchor markup--p-anchor" data-href="https://techcrunch.com/2016/08/11/google-isnt-safe-from-yahoos-fate/" href="https://techcrunch.com/2016/08/11/google-isnt-safe-from-yahoos-fate/" rel="noopener nofollow noreferrer" target="_blank">
    许多人所指出
   </a>
   的那样，在线广告行业在 2010 年中期开始经历了从搜索到发现的重大转变。
  </p>
  <p class="graf graf--p graf-after--p" id="8ac2">
   虽然谷歌在垂死的搜索广告市场上保护其垄断地位，但 Facebook — 谷歌在线广告领域的最大竞争对手 — 已经逐渐在这一趋势
   <a class="markup--anchor markup--p-anchor" data-href="https://www.emarketer.com/Article/Google-Facebook-Increase-Their-Grip-on-Digital-Ad-Market/1015417" href="https://www.emarketer.com/Article/Google-Facebook-Increase-Their-Grip-on-Digital-Ad-Market/1015417" rel="noopener nofollow noreferrer" target="_blank">
    占据了
   </a>
   主导地位，并通过其 Feed 原生显示广告主导了在线广告市场。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-1272 jetpack-lazy-image" data-lazy-sizes="(max-width: 841px) 100vw, 841px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-3.jpeg?resize=841%2C378&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-3.jpeg?w=841&amp;ssl=1 841w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-3.jpeg?resize=300%2C135&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-3.jpeg?resize=768%2C345&amp;ssl=1 768w" data-recalc-dims="1" height="378" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-3.jpeg?resize=841%2C378&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="841"/>

   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="b0f1">
   2015 年底，苹果 — 谷歌在移动领域的主要竞争对手 — 为他们的手机和平板电脑添加了一项功能，允许用户屏蔽广告。
  </p>
  <p class="graf graf--p graf-after--p" id="9a43">
   运行 iOS 的设备约占谷歌移动搜索广告收入的
   <a class="markup--anchor markup--p-anchor" data-href="http://appleinsider.com/articles/15/05/27/apples-ios-drives-75-of-googles-mobile-advertising-revenue" href="http://appleinsider.com/articles/15/05/27/apples-ios-drives-75-of-googles-mobile-advertising-revenue" rel="noopener nofollow noreferrer" target="_blank">
    75％
   </a>
   ，因此，通过采取这一举措，苹果公司同时对2010年的广告拦截辩论进行了评估，并对在线广告的未来造成
   <a class="markup--anchor markup--p-anchor" data-href="https://techcrunch.com/2016/07/24/apple-lays-the-groundwork-to-kill-online-advertising/" href="https://techcrunch.com/2016/07/24/apple-lays-the-groundwork-to-kill-online-advertising/" rel="noopener nofollow noreferrer" target="_blank">
    重大打击
   </a>
   。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-1273 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-4.jpeg?resize=1100%2C405&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-4.jpeg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-4.jpeg?resize=300%2C110&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-4.jpeg?resize=768%2C283&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-4.jpeg?resize=1024%2C377&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-4.jpeg?resize=1100%2C405&amp;ssl=1 1100w" data-recalc-dims="1" height="405" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-4.jpeg?resize=1100%2C405&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>

  <p class="graf graf--p graf-after--figure" id="2633">
   一年后，随着互联网的流动，广告拦截也是如此。从2015年到2016年，在移动设备上屏蔽广告的人数
   <a class="markup--anchor markup--p-anchor" data-href="https://pagefair.com/blog/2016/mobile-adblocking-report/" href="https://pagefair.com/blog/2016/mobile-adblocking-report/" rel="noopener nofollow noreferrer" target="_blank">
    增长了102％
   </a>
   ; 截至2016年底，全球约有 16％ 的智能手机用户在移动设备上网时
   <a class="markup--anchor markup--p-anchor" data-href="https://pagefair.com/blog/2016/mobile-adblocking-report/" href="https://pagefair.com/blog/2016/mobile-adblocking-report/" rel="noopener nofollow noreferrer" target="_blank">
    屏蔽了广告
   </a>
   。对于美国的台式机和笔记本电脑用户来说，这个数字高达
   <a class="markup--anchor markup--p-anchor" data-href="https://www.emarketer.com/Article/Why-More-than-Quarter-of-US-Internet-Users-Block-Ads/1014333" href="https://www.emarketer.com/Article/Why-More-than-Quarter-of-US-Internet-Users-Block-Ads/1014333" rel="noopener nofollow noreferrer" target="_blank">
    25％
   </a>
   ，注意，这个国家占谷歌收入的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.statista.com/statistics/266250/regional-distribution-of-googles-revenue/" href="https://www.statista.com/statistics/266250/regional-distribution-of-googles-revenue/" rel="noopener nofollow noreferrer" target="_blank">
    47％
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="9857">
   最有可能阻止广告的人也是对谷歌最有价值的人群：
   <a class="markup--anchor markup--p-anchor" data-href="http://marketingland.com/ad-blocker-usage-highest-among-key-advertiser-demos-millennials-and-high-earners-143546" href="http://marketingland.com/ad-blocker-usage-highest-among-key-advertiser-demos-millennials-and-high-earners-143546" rel="noopener nofollow noreferrer" target="_blank">
    千禧一代和高收入者
   </a>
   。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-1274 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-5.jpeg?resize=1100%2C619&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-5.jpeg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-5.jpeg?resize=300%2C169&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-5.jpeg?resize=768%2C432&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-5.jpeg?resize=1024%2C576&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-5.jpeg?resize=1100%2C619&amp;ssl=1 1100w" data-recalc-dims="1" height="619" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-5.jpeg?resize=1100%2C619&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-1274" data-recalc-dims="1" height="619" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-5.jpeg?resize=1100%2C619&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-5.jpeg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-5.jpeg?resize=300%2C169&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-5.jpeg?resize=768%2C432&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-5.jpeg?resize=1024%2C576&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-5.jpeg?resize=1100%2C619&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="faa0">
   <strong class="markup--strong markup--p-strong">
    互联网用户说，他们讨厌广告
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="17ac">
   2017 年初，谷歌宣布计划在其广受欢迎的 Chrome 浏览器中制作广告拦截器。谷歌的广告拦截器
   <a class="markup--anchor markup--p-anchor" data-href="http://www.thedrum.com/news/2016/09/16/procter-gamble-unilever-google-facebook-and-more-form-coalition-better-ads" href="http://www.thedrum.com/news/2016/09/16/procter-gamble-unilever-google-facebook-and-more-form-coalition-better-ads" rel="noopener nofollow noreferrer" target="_blank">
    只会阻止
   </a>
   被广告联盟视为不可接受的广告，从而有效地
   <strong class="markup--strong markup--p-strong">
    允许该公司利用其主导的网络浏览器来加强其已占据主导地位的广告业务
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="4c10">
   即使在做出这种绝望的和合法性可疑的举动之后，谷歌很快就会清楚地知道即使广告越来越好，广告拦截数字也会继续上升。谷歌让更多的人对无广告的互联网体验有所了解了。
  </p>
  <p class="graf graf--p graf-after--p" id="d751">
   该公司发现，人们不喜欢的不只有烦人广告，而是所有广告、一般的广告。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-1275 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-6.jpeg?resize=1100%2C619&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-6.jpeg?w=1920&amp;ssl=1 1920w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-6.jpeg?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-6.jpeg?resize=768%2C432&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-6.jpeg?resize=1024%2C576&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-6.jpeg?resize=1100%2C619&amp;ssl=1 1100w" data-recalc-dims="1" height="619" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-6.jpeg?resize=1100%2C619&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-1275" data-recalc-dims="1" height="619" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-6.jpeg?resize=1100%2C619&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-6.jpeg?w=1920&amp;ssl=1 1920w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-6.jpeg?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-6.jpeg?resize=768%2C432&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-6.jpeg?resize=1024%2C576&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-6.jpeg?resize=1100%2C619&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="56eb">
   谷歌投放广告的关键平台是 YouTube，它于 2006 年收购并迅速成为其最大的实体之一。但即使世界上有
   <a class="markup--anchor markup--p-anchor" data-href="http://www.huffingtonpost.com/2013/03/21/youtube-stats_n_2922543.html" href="http://www.huffingtonpost.com/2013/03/21/youtube-stats_n_2922543.html" rel="noopener nofollow noreferrer" target="_blank">
    六分之一
   </a>
   的人每个月都会访问这个视频共享巨头，YouTube 也从未盈利。为了对抗广告拦截者的影响，YouTube 于 2015 年底推出了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wsj.com/articles/viewers-dont-add-up-to-profit-for-youtube-1424897967" href="https://www.wsj.com/articles/viewers-dont-add-up-to-profit-for-youtube-1424897967" rel="noopener nofollow noreferrer" target="_blank">
    无广告订阅
   </a>
   模式，但订阅数量并
   <a class="markup--anchor markup--p-anchor" data-href="http://www.theverge.com/2016/11/2/13498470/youtube-red-subscribers-video-content-music" href="https://www.theverge.com/2016/11/2/13498470/youtube-red-subscribers-video-content-music" rel="noopener nofollow noreferrer" target="_blank">
    不令人满意
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="33ce">
   2017 年初，随着广告投放者争议，广告客户开始退出，YouTube 已经无法克服的问题成倍增加，巨额收入来源开始
   <a class="markup--anchor markup--p-anchor" data-href="http://www.ibtimes.com/pewdiepie-starts-weekly-twitch-show-uploads-youtube-over-party-video-2523797" href="http://www.ibtimes.com/pewdiepie-starts-weekly-twitch-show-uploads-youtube-over-party-video-2523797" rel="noopener nofollow noreferrer" target="_blank">
    撤离
   </a>
   该网站。
  </p>
  <p class="graf graf--p graf-after--p" id="3fc6">
   即使那些没有阻止广告的人也已经训练自己能完全忽视它们了。研究人员称这种现象为“
   <a class="markup--anchor markup--p-anchor" data-href="http://www.tobiipro.com/fields-of-use/marketing-consumer-research/advertising/" href="http://www.tobiipro.com/fields-of-use/marketing-consumer-research/advertising/" rel="noopener nofollow noreferrer" target="_blank">
    横幅失明
   </a>
   ”。 只有
   <a class="markup--anchor markup--p-anchor" data-href="https://www.thinkwithgoogle.com/intl/en-gb/planning-tool/display-benchmarks/" href="https://www.thinkwithgoogle.com/intl/en-gb/planning-tool/display-benchmarks/" rel="noopener nofollow noreferrer" target="_blank">
    0.06％
   </a>
   的观众点击了平均横幅广告，但其中大约
   <a class="markup--anchor markup--p-anchor" data-href="http://www.goldspotmedia.com/fat-finger-report/" href="http://www.goldspotmedia.com/fat-finger-report/" rel="noopener nofollow noreferrer" target="_blank">
    50％
   </a>
   属于手滑。
  </p>
  <p class="graf graf--p graf-after--p" id="1dfe">
   调查显示，
   <a class="markup--anchor markup--p-anchor" data-href="http://www.bannersnack.com/blog/build-trust-display-ads/" href="https://www.bannersnack.com/blog/build-trust-display-ads/" rel="noopener nofollow noreferrer" target="_blank">
    54％
   </a>
   的用户表示缺乏信任是他们不点击横幅广告的原因，
   <a class="markup--anchor markup--p-anchor" data-href="http://downloads.pagefair.com/reports/adblocking_goes_mainstream_2014_report.pdf" href="http://downloads.pagefair.com/reports/adblocking_goes_mainstream_2014_report.pdf" rel="noopener nofollow noreferrer" target="_blank">
    33％
   </a>
   的用户认为他们完全无法容忍。这些数字为在线广告的可持续性描绘了一幅非常严峻的画面，尤其是想想谷歌在监视资本主义行业中的地位。
  </p>
  <p class="graf graf--p graf-after--p" id="3c37">
   谷歌巨大的搜索引擎开始肆虐。
  </p>
  <p class="graf graf--p graf-after--p" id="e6e1">
   <strong class="markup--strong markup--p-strong">
    曾经有一个机会，但谷歌错过了
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="f617">
   如果因失去大部分观众而烦恼，其余部分还不够糟糕，谷歌也未能领先于技术历史上最大的变化之一。他们认识到所谓的人工智能的重要性，但他们的方法为达到目的。由于谷歌的搜索支柱已经变得不稳定，因此公司的人工智能战略很多。
  </p>
  <p class="graf graf--p graf-after--p" id="7057">
   谷歌当时的首席执行官 Sundar Pichai 在 2016 年曾
   <a class="markup--anchor markup--p-anchor" data-href="https://blog.google/topics/inside-google/this-years-founders-letter/" href="https://blog.google/topics/inside-google/this-years-founders-letter/" rel="noopener nofollow noreferrer" target="_blank">
    预言
   </a>
   “下一个重要的步骤将是”设备“的概念逐渐消失”并且“随着时间的推移，计算机本身 — 无论其形状因素 — 将是一个智能的助理帮助你度过每一天。我们将从移动优先转移到 AI 第一世界。“
  </p>
  <p class="graf graf--p graf-after--p" id="0796">
   谷歌能够承认即将到来的趋势但仍未能领先占据它，这让许多观察家都想起了社交媒体和即时通讯这些蓬勃发展的行业中的灾难性失败。
  </p>
  <p class="graf graf--p graf-after--p" id="aa07">
   <strong class="markup--strong markup--p-strong">
    Google vs. Amazon
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="570c">
   与此同时，在 2014 年，亚马逊发布了一款名为 Amazon Echo 的产品，这是一款小型扬声器，可以放在家里为用户解答问题，执行任务和在线购物 — — 就是我们一直强调千万不要使用的东西，它很可能被用作窃听器。 Echo 赚了不少钱，因为傻子太多。而谷歌两年后发布了它的模仿产品 Google Home，但已经
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/technology/2017/jan/22/home-battleground-amazon-google-voice-technology" href="https://www.theguardian.com/technology/2017/jan/22/home-battleground-amazon-google-voice-technology" rel="noopener nofollow noreferrer" target="_blank">
    赶不及了
   </a>
   ，并且没有明确的收入策略。
  </p>
  <p class="graf graf--p graf-after--p" id="ac21">
   另一方面，Alexa — 那个 Echo 中的生活助手 — 很快被整合到多个产品和服务中，其货币化模式清晰，可行，最重要的是对未来友好。 Echo 使通过亚马逊订购产品变得更容易，每当有人使用 Echo 购买东西时，亚马逊就赚钱了。
  </p>
  <p class="graf graf--p graf-after--p" id="5ae5">
   谷歌通过将其构建到 Android 中扩展了其虚拟助手的覆盖范围，但这样做仍然无法为该技术如何产生足够的收入来维持谷歌不断扩大的昂贵创新产品提供答案。
  </p>
  <p class="graf graf--p graf-after--p" id="8c5d">
   谷歌的广告依赖于屏幕，但语音交互完全颠覆了屏幕。谷歌曾试图在 Google Home 上播放音频广告，但消费者完全无法接受。投资者开始在 2017 年表达他们的
   <a class="markup--anchor markup--p-anchor" data-href="http://www.businessinsider.com/google-ceo-sundar-pichai-responds-to-concerns-over-monetizing-voice-search-2017-1" href="http://www.businessinsider.com/google-ceo-sundar-pichai-responds-to-concerns-over-monetizing-voice-search-2017-1" rel="noopener nofollow noreferrer" target="_blank">
    担忧
   </a>
   ，但 Sundar Pichai 告诉他们不要担心，让他们假设谷歌将使用他们“古老的策略”并
   <strong class="markup--strong markup--p-strong">
    分析用户的语音搜索
   </strong>
   — —
   <strong class="markup--strong markup--p-strong">
    就是监听
   </strong>
   ，以便可以在带屏幕的设备上展示更合适的广告。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-1276 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-7.jpeg?resize=1100%2C695&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-7.jpeg?w=1200&amp;ssl=1 1200w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-7.jpeg?resize=300%2C190&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-7.jpeg?resize=768%2C485&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-7.jpeg?resize=1024%2C647&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-7.jpeg?resize=1100%2C695&amp;ssl=1 1100w" data-recalc-dims="1" height="695" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-7.jpeg?resize=1100%2C695&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-1276" data-recalc-dims="1" height="695" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-7.jpeg?resize=1100%2C695&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-7.jpeg?w=1200&amp;ssl=1 1200w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-7.jpeg?resize=300%2C190&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-7.jpeg?resize=768%2C485&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-7.jpeg?resize=1024%2C647&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-7.jpeg?resize=1100%2C695&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="c349">
   2017年初的头条新闻宣称 “Alexa Just Conquered CES The World is Next。”亚马逊随后将他们的技术提供给
   <a class="markup--anchor markup--p-anchor" data-href="http://www.zdnet.com/article/amazon-opens-echo-microphone-tech-to-third-party-alexa-devices/" href="https://www.zdnet.com/article/amazon-opens-echo-microphone-tech-to-third-party-alexa-devices/" rel="noopener nofollow noreferrer" target="_blank">
    第三方
   </a>
   制造商，使两家公司之间的距离更远了。亚马逊之前已经
   <a class="markup--anchor markup--p-anchor" data-href="http://www.investors.com/news/technology/amazon-confronts-microsoft-google-ibm-in-cloud-computing-wars/" href="http://www.investors.com/news/technology/amazon-confronts-microsoft-google-ibm-in-cloud-computing-wars/" rel="noopener nofollow noreferrer" target="_blank">
    击败了谷歌
   </a>
   ，在2016 年占据了 54％ 的云计算市场（相比谷歌的 3％）。
  </p>
  <p class="graf graf--p graf-after--p" id="5989">
   <strong class="markup--strong markup--p-strong">
    广告不是永远的
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="3c11">
   在其巅峰时期，谷歌拥有庞大而忠诚的用户群，产品数量惊人，但广告收入却是将所有产品整合在一起的粘合剂。随着数据的减少，谷歌的核心开始在其庞大的帝国的重压下垮台。
  </p>
  <p class="graf graf--p graf-after--p" id="e69f">
   自 1998 年以来谷歌一直是科技行业的推动力。但在人们鄙视广告的世界里，谷歌的监视资本主义商业模式并不是创新友好的，他们错过了几次转向的机会，最终使他们无数雄伟壮观项目不可持续。创新需要资金，谷歌的主要收入来源已经开始枯竭。
  </p>
  <p class="graf graf--p graf-after--p" id="2135">
   在短短的几年里，谷歌已经从一个有趣的，普通的动词“谷歌一下”，变成了一个巨人可以迅速堕落的警示。
  </p>
  <p class="graf graf--p graf-after--p" id="5940">
   <strong class="markup--strong markup--p-strong">
    本文希望说明
   </strong>
   ：下一个要对付的应该是亚马逊这个巨魔，该公司的恶毒在所谓的“智能家居”方面的表现尤其明显，他们把语音装置、灯泡、门锁等等一大堆玩意塞进你的家里，而你根本无法知道这些
   <strong class="markup--strong markup--p-strong">
    专有软件
   </strong>
   什么时候会变成窃听器和偷窥的眼。技术界有句流行语：“最愚蠢的人最喜欢智能设备”，那些看起来便捷省力的东西正是最危险的因素。
  </p>
  <p class="graf graf--p graf-after--p" id="5910">
   并不是拒绝科技，肯定不是，而是拒绝这种巨头垄断的暗箱技术。
   <strong class="markup--strong markup--p-strong">
    消费者应该支持开源和数据管理透明化，及相关完备的法律保障
   </strong>
   。否则，就不是科技进步，而是 1984 还魂。
  </p>
