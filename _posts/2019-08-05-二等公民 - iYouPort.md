  <p class="graf graf--p">
   反封锁、反审查、保护隐私，这是每个网络公民的基本权利和责任，然而，如今行使这一权利的人却被沦为了网络二等公民，他们被歧视、他们的行为受到限制。这是为什么？
  </p>
  <p class="graf graf--p">
   根据最近一篇研究论文中公布的数据，大约130万个IP地址 — — 包括谷歌，雅虎，Craigslist 和 Yelp — — 将 Tor 匿名网络的用户变为二等网络公民，直接阻止他们、或降低为他们提供的服务。
  </p>
  <p class="graf graf--p">
   题为“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.internetsociety.org/sites/default/files/blogs-media/do-you-see-what-i-see-differential-treatment-anonymous-users.pdf" href="https://www.internetsociety.org/sites/default/files/blogs-media/do-you-see-what-i-see-differential-treatment-anonymous-users.pdf" rel="noopener noreferrer" target="_blank">
    匿名用户的差别待遇
   </a>
   ”的这份报告称，Alexa 中有 3.67％ 的网站歧视使用已知 Tor 出口节点 IP 地址访问的计算机。在某些情况下，访问者完全被锁定，而在其他情况下，用户需要完成繁琐的人机验证、并且他们可以做的事受到限制。
  </p>
  <p class="graf graf--p">
   报告作者说，这是由于网站试图“限制欺诈和其他类型的在线犯罪”，但是不成比例地影响了 Tor 用户。在这个过程中，那些遵纪守法的 Tor 用户被降级为二等公民。
  </p>
  <p class="graf graf--p">
   作者写道：“虽然许多网站阻止 Tor 是为减少滥用行为，但是，这样做肯定会影响到那些没有其他方式访问被审查的互联网内容的国家用户。”
  </p>
  <p class="graf graf--p">
   在许多情况下，降级的体验由内容传递网络自动执行，这有助于各个网站分发内容并阻止恶意用户。最著名的 CDN 之一 CloudFlare，为每个访问的 IP 地址分配声誉分数，如果分数太低，则需要用户完成 CAPTCHA，旨在证明他们是人类而非恶意脚本。
  </p>
  <p class="graf graf--p">
   在
   <a class="markup--anchor markup--p-anchor" data-href="https://support.cloudflare.com/hc/en-us/articles/203306930" href="https://support.cloudflare.com/hc/en-us/articles/203306930" rel="noopener noreferrer" target="_blank">
    支持页面
   </a>
   上，CloudFlare 表示它没有专门针对 Tor 用户，但它继续说“由于某些人使用 Tor 网络的行为（垃圾邮件发送者、恶意软件分发者、攻击者等），Tor 出口节点的 IP 地址通常会声名狼籍。“ 该论文的研究结果引发了 Tor 倡导者和 CloudFlare 代表之间
   <a class="markup--anchor markup--p-anchor" data-href="https://trac.torproject.org/projects/tor/ticket/18361" href="https://trac.torproject.org/projects/tor/ticket/18361" rel="noopener noreferrer" target="_blank">
    长期而且经常激烈的讨论
   </a>
   。
  </p>
  <p class="graf graf--p">
   与此同时，CloudFlare 竞争对手 Akamai 的网站经常用403错误阻止Tor用户。虽然谷歌和雅虎没有完全阻止Tor用户，但是使用Tor IP地址的访问者无法使用他们的某些页面或服务。
  </p>
  <p class="graf graf--p">
   这份报告揭示了网站安全与获取信息和匿名之间的紧张关系，
   <strong class="markup--strong markup--p-strong">
    特别是那些具有苛刻的内容审查机制或密切监视公民网络浏览行为的压制性国家。
   </strong>
  </p>
  <p class="graf graf--p">
   正如作者所说，“互联网上的匿名通信对于那些受到政府限制的网民来说至关重要，但是，匿名网络的效用受到互联网上阻止或降低匿名用户请求的服务的威胁。“
  </p>

    ome websites turning law-abiding Tor users into second-class citizens
   </a>
   . Tor users blocked or faced with CAPTCHA if IP address matches known exit node.
  </p>
