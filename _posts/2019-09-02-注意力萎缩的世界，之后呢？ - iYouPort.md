   注意力萎缩的世界，之后呢？

 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      <strong>
       互联网不再是实现现实变革的希望；因为它与现实世界分离了，并且，它不断抢夺人们对真正的现实的关注能力。如果你不能集中精力在任何一件事上，不论是有重大意义的事还是简单的消遣，其结果是什么？
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4152 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/注意力萎缩.png?resize=1100%2C682&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/注意力萎缩.png?w=1142&amp;ssl=1 1142w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/注意力萎缩.png?resize=300%2C186&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/注意力萎缩.png?resize=768%2C476&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/注意力萎缩.png?resize=1024%2C635&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/注意力萎缩.png?resize=1100%2C682&amp;ssl=1 1100w" data-recalc-dims="1" height="682" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/注意力萎缩.png?resize=1100%2C682&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4152" data-recalc-dims="1" height="682" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/注意力萎缩.png?resize=1100%2C682&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/注意力萎缩.png?w=1142&amp;ssl=1 1142w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/注意力萎缩.png?resize=300%2C186&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/注意力萎缩.png?resize=768%2C476&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/注意力萎缩.png?resize=1024%2C635&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/注意力萎缩.png?resize=1100%2C682&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   根据丹麦技术大学最近的一项研究，
   <strong class="markup--strong markup--p-strong">
    由于超连接世界中的信息严重过载，人们的注意力正在不断快速萎缩
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在社交媒体、“繁忙的新闻周期”、和人们所做的其他事之间，基于所研究的几个指标得出结论，
   <strong class="markup--strong markup--p-strong">
    人们在各种主题上所花费的时间都在明显减少
   </strong>
   —— 不管是娱乐、谷歌搜索、Reddit评论、还是其他任何在线领域。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们想要了解哪些机制可以推动这种行为。我们设计了一个基于三个基本成分的数学模型：’热度’、时效性、和对新事物的渴望”，PhilippHövel 博士说，他是科克大学应用数学讲师。
  </p>
  <p class="graf graf--p">
   该模型为他们的观察做出了解释。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当互联网在更短的时间内产生出更多内容时，这种情况会快速耗尽集体的注意力
   </strong>
   。由于对新奇事物的激烈竞争，一个主题紧接着下一个主题，每一个主题的公共利益峰值都变得非常短暂。
  </p>
  <p class="graf graf--p">
   简单说，互联网上几乎没有任何内容能激励现实中的变革。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “该模型中的一个参数是输入率 — 即 信息的丰富程度。过去几十年来这个世界的连接变得越来越紧密。这意味着内容的数量一直在迅速增加，这使人们的注意力被消耗殆尽，人们对“新奇”的渴望只会使人们更快速地在不同主题之间迅速切换。
  </p>
  <p class="graf graf--p">
   该研究的指标之一是用于推广主题的流行“主题标签”，在 Twitter 的“全球50强”排名中能保持多久。
  </p>
  <p class="graf graf--p">
   2013年，主题标签能保持在前50强列表中平均 17.5 小时。 2016年，这个数字减少到了 11.9 小时。当然，2016年的美国大选是足够混乱的，若非此选举，上述数字只会下降得更快。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4151 jetpack-lazy-image" data-lazy-sizes="(max-width: 1000px) 100vw, 1000px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/数字下降.png?resize=1000%2C716&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/数字下降.png?w=1000&amp;ssl=1 1000w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/数字下降.png?resize=300%2C215&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/数字下降.png?resize=768%2C550&amp;ssl=1 768w" data-recalc-dims="1" height="716" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/数字下降.png?resize=1000%2C716&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1000"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4151" data-recalc-dims="1" height="716" sizes="(max-width: 1000px) 100vw, 1000px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/数字下降.png?resize=1000%2C716&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/数字下降.png?w=1000&amp;ssl=1 1000w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/数字下降.png?resize=300%2C215&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/数字下降.png?resize=768%2C550&amp;ssl=1 768w" width="1000"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   自2016年大选以来，新闻周期一直在持续不断的戏剧化 — 左派坚持认为特朗普是一位非正式的、不合适的总统，在弗拉基米尔·普京的帮助下当选，而右翼坚持认为左派是阴谋论家组成的智商欠费党派。
  </p>
  <p class="graf graf--p">
   除了特朗普 — 俄罗斯门歇斯底里主导的新闻周期之外，所谓的热点还可以加上恐怖主义、大规模移民、对气候变化不断发出的警钟、以及2020年民主党候选人试图互相攻击 ……难怪人们无法再像以前那样集中精力了。
  </p>
  <p class="graf graf--p">
   简单说，我们被超载了。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <strong class="markup--strong markup--p-strong">
    在集体思想中，分配的注意力是有一定规模的，并非无限，但是如今争夺关注的文化项目已经变得如此的密集。结果就是如你所见，任何新闻都无法真正保持吸引力[不管是有价值的还是没有价值的]
   </strong>
   ”，Sune Lehmann 教授说。
  </p>
  <p class="graf graf--p">
   有趣的是，在注意力减少这一领域的唯一例外似乎是维基百科和科学出版物。
  </p>
  <p class="graf graf--p">
   虽然该项研究从整体上考虑了全球的注意力，但研究下个人注意力如何突破会很有趣。
  </p>
  <p class="graf graf--p">
   根据 Lehmann 的说法; “研究数据仅支持集体关注范围正在缩小的说法。因此，作为下一步，研究这种现象对个体的影响将会很有意义，因为观察到的发展可能对个人评估他们的信息消费能力产生负面影响。
  </p>
  <p class="graf graf--p">
   例如，信息周期加速增加了新闻记者跟上不断变化的新闻环境的压力。我们希望在这方面进行更多的研究，以便可以设计出新的通信系统，目的是即使新主题以越来越高的速度出现，信息质量也不会受到影响。“
  </p>
  <p class="graf graf--p">
   注：这不是一个新话题。我们在多年前曾多次研究过信息超载的影响，详见类目“互联网时代/媒介效应”。
  </p>
  <p class="graf graf--p">
   ⚠️这一问题所导致的结果并非注意力萎缩这么表面。而是，要理解到，互联网很可能
   <strong class="markup--strong markup--p-strong">
    不再是
   </strong>
   能够动员人们行动起来实现现实变革的有力工具；并且，更重要的是，处于这种环境生态下的人是最容易被操作的人，详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" rel="noopener noreferrer" target="_blank">
    看不见的独裁
   </a>
   》。
  </p>
  <p class="graf graf--p">
   回归“慢新闻”的设想最近一两年内的支持率正在稳步高升，其原因与相关问题被大众所了解不无关系（在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bc%95%e6%b5%81%e6%9a%b4%e6%94%bf%e5%92%8c%e7%94%9f%e5%ad%98%e5%9b%b0%e6%83%91/" href="https://www.iyouport.org/%e5%bc%95%e6%b5%81%e6%9a%b4%e6%94%bf%e5%92%8c%e7%94%9f%e5%ad%98%e5%9b%b0%e6%83%91/" rel="noopener noreferrer" target="_blank">
    引流暴政和生存困惑
   </a>
   》）。但不一定只能走回头路才可以解决这个问题，如果能在不影响前进的前提下改善当下的状况，其结果也许会更好。
  </p>
