  <p class="graf graf--p">
   FP认为冠状病毒是最大的情报失败 —— 比珍珠港和9/11更严重 …… 以下是FP的文章。
  </p>
  <p class="graf graf--p">
   去年9月，我在华盛顿与一家财富百强公司的风险副总裁会面。这位高管曾经有很长的情报分析师职业经历，我问他的是所有记者遇到这样的人时都会问的问题：“你最担心的是什么？”
  </p>
  <p class="graf graf--p">
   他回答说：“一种高度传染性的病毒，始于中国某处并迅速传播。”
  </p>
  <p class="graf graf--p">
   这家高管所在的公司在整个东亚设有办事处，他解释了公司随后采取的预防性缓解措施，以应对这一潜在的威胁。
  </p>
  <p class="graf graf--p">
   自从新型冠状病毒席卷全球以来，我经常想到此人的先验风险计算。
  </p>
  <p class="graf graf--p">
   大多数领导者都缺乏进行基于风险的例行视野扫描的纪律，并且几乎很少有人能制定必要的应急计划。更有远见的领导者应该可以提前正确地识别出最大的威胁，从而抢先一步制定和实施防御计划。
  </p>
  <p class="graf graf--p">
   可以说，特朗普政府在认真对待针对冠状病毒爆发的特定的、反复的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nbcnews.com/politics/national-security/u-s-intel-agencies-warned-rising-risk-outbreak-coronavirus-n1144891" href="https://www.nbcnews.com/politics/national-security/u-s-intel-agencies-warned-rising-risk-outbreak-coronavirus-n1144891" rel="noopener noreferrer" target="_blank">
    情报界
   </a>
   <a class="markup--anchor markup--p-anchor" data-href="https://thehill.com/policy/healthcare/488763-intel-reports-going-back-to-january-warned-of-coronavirus-threat" href="https://thehill.com/policy/healthcare/488763-intel-reports-going-back-to-january-warned-of-coronavirus-threat" rel="noopener noreferrer" target="_blank">
    警告
   </a>
   以及大力推行与预期威胁相称的全国响应计划方面，都失败了。
  </p>
  <p class="graf graf--p">
   仅联邦政府就完全有资源和权力来领导公共和私人利益相关者应对病毒造成的可预见的危害。
  </p>
  <p class="graf graf--p">
   不幸的是，特朗普政府官员做出了一系列的判断（最大限度地降低了 COVID-19 的危害）和决定（拒绝采取必要的紧急行动），这些错误不必要地使美国人的安全性大大降低。
  </p>
  <p class="graf graf--p">
   简而言之，特朗普政府对美国人民施加了灾难性的战略突袭。
  </p>
  <p class="graf graf--p">
   但是，与以往不同（珍珠港、1979年的伊朗革命、尤其是9/11），当前的变革是前所未有的冷漠甚至故意的疏忽造成的。
  </p>
  <p class="graf graf--p">
   例如 9/11调查委员会报告指责基地组织通过乔治·W·布什袭击罗纳德·里根总统的政府，而日益严重的冠状病毒危机则是现任白宫的唯一责任。
  </p>
  <p class="graf graf--p">
   9/11委员会报告的
   <a class="markup--anchor markup--p-anchor" data-href="https://govinfo.library.unt.edu/911/report/911Report_Ch8.htm" href="https://govinfo.library.unt.edu/911/report/911Report_Ch8.htm" rel="noopener noreferrer" target="_blank">
    第8章
   </a>
   标题为 “系统已经亮起红灯”。引述来自中央情报局前局长 George Tenet，2001年夏天当时情报界的多种报道流表明美国境内即将发生航空恐怖袭击。
  </p>
  <p class="graf graf--p">
   尽管一些反恐官员发出了警告并做出了努力，但9/11委员会仍然认为：“几乎没有证据表明该阴谋的进展受到任何政府行动的干扰。 ……时间不多了。”
  </p>
  <p class="graf graf--p">
   《华盛顿邮报》
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/national-security/us-intelligence-reports-from-january-and-february-warned-about-a-likely-pandemic/2020/03/20/299d8cda-6ad5-11ea-b5f1-a5a804158597_story.html" href="https://www.washingtonpost.com/national-security/us-intelligence-reports-from-january-and-february-warned-about-a-likely-pandemic/2020/03/20/299d8cda-6ad5-11ea-b5f1-a5a804158597_story.html" rel="noopener noreferrer" target="_blank">
    报道了
   </a>
   情报界在一月和二月持续向白宫提供的有关冠状病毒的警告。
  </p>
  <p class="graf graf--p">
   但这些警报对高级政府官员影响不大，毫无疑问，反而是特朗普政府从1月22日开始对这种病毒的不断嘲笑影响了他们：“我们已经完全控制了它。这是一个从中国来的东西，我们可以很好地控制住它。”
  </p>
  <p class="graf graf--p">
   到目前为止，对特朗普的领导风格有三点痛苦且显而易见的发现，可以解释美国人现在所面临的日益严重的冠状病毒大流行是为什么。
  </p>
  <p class="graf graf--p">
   首先，有一个事实就是，一旦他绝对相信任何东西 —— 无论是多么糟糕的想法、信息不灵通还是不准确，他都会完全依赖于最初的印象或判断。
  </p>
  <p class="graf graf--p">
   领导者异常狂暴，过分自信。对于许多人来说，他们已经提高了权力水平这一事实证明了他们内在的智慧。但是，真正明智的领导人应该会真诚地征求反馈和批评，他们应该是积极开放的思想家，并且能够及时变通。众所周知，特朗普缺乏这些能力。
  </p>
  <p class="graf graf--p">
   其次，特朗普的判断具有很高的传播性，几乎感染了每位官员或顾问的思想和行为。而且他的不准确和糟糕的评论也具有非凡的能力，可以被军事、情报和商业领袖重新利用。
  </p>
  <p class="graf graf--p">
   而且，如果有人没能以足够的力度持续模仿总统的声明，他们就会被解雇，或者泄漏出消息说他们随时可能被解雇，最值得注意的是最近关于特朗普对过敏和传染病研究所所长 Anthony Fauci 不耐烦的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2020/03/23/us/politics/coronavirus-trump-fauci.htmlhttps://www.nytimes.com/2020/03/23/us/politics/coronavirus-trump-fauci.html" href="https://www.nytimes.com/2020/03/23/us/politics/coronavirus-trump-fauci.htmlhttps://www.nytimes.com/2020/03/23/us/politics/coronavirus-trump-fauci.html" rel="noopener noreferrer" target="_blank">
    报道
   </a>
   。
  </p>
  <p class="graf graf--p">
   第三，糟糕的判断很快就污染了联邦政府的所有决策部门，几乎没有抵抗力，甚至没有合理的质疑。
  </p>
  <p class="graf graf--p">
   通常，联邦机构由白宫认为最有能力执行政策的官员领导。这些官员通常享有一定程度的自治权，不在特朗普的管制之下。甚至从历史上看，无党派的国家安全部门或情报机构的领导职位也被思想上与白宫保持一致的人占据，而不是拥有推倒或解决非政治性员工提出的担忧所需的经验或专业知识。
  </p>
  <p class="graf graf--p">
   因此，特朗普最初的错误假设或声明会融入日常的政策实施中。
  </p>
  <p class="graf graf--p">
   邮报的同一篇报道还引述了一位匿名美国官员的惊人讲话：“唐纳德特朗普可能没想到这一点，但政府中的许多其他人都应该想到 —— 他们根本无法让他对此做任何事。系统已经亮起红灯”。
  </p>
  <p class="graf graf--p">
   鉴于特朗普很早就得出结论，冠状病毒根本不可能对美国构成威胁，而情报机构、采用流行病学模型的医学专家或公共卫生官员本可以告诉白宫以指出特朗普的错误，结果或许会有所不同。
  </p>
  <p class="graf graf--p">
   众所周知，前国家安全顾问 Henry Kissinger
   <a class="markup--anchor markup--p-anchor" data-href="https://books.google.com/books?id=F7wYAwAAQBAJ&amp;pg=PA188&amp;dq=%22You+warned+me,+but+you+didn%E2%80%99t+convince+me%22&amp;hl=en&amp;newbks=1&amp;newbks_redir=0&amp;sa=X&amp;ved=2ahUKEwiLrbq8grboAhVUlXIEHdacB6kQ6AEwAXoECAAQAg#v=onepage&amp;q=%22You%20warned%20me%2C%20but%20you%20didn%E2%80%99t%20convince%20me%22&amp;f=false" href="https://books.google.com/books?id=F7wYAwAAQBAJ&amp;pg=PA188&amp;dq=%22You+warned+me,+but+you+didn%E2%80%99t+convince+me%22&amp;hl=en&amp;newbks=1&amp;newbks_redir=0&amp;sa=X&amp;ved=2ahUKEwiLrbq8grboAhVUlXIEHdacB6kQ6AEwAXoECAAQAg#v=onepage&amp;q=%22You%20warned%20me%2C%20but%20you%20didn%E2%80%99t%20convince%20me%22&amp;f=false" rel="noopener noreferrer" target="_blank">
    曾说过
   </a>
   ，情报机构的警告未能得到人们的认可：“你警告过我，但没有说服我。”然而，特朗普政府完全与相左的观点隔绝，即便是对的，也无法被说服。
  </p>
  <p class="graf graf--p">
   在冠状病毒爆发的初期，白宫的缺乏热情将是任何现代总统任期中代价最高的决定之一。已经向这些官员发出的清晰的警告和关键决策要点，已经提前了足够多的时间，这个国家本来可以做得更好。
  </p>
  <p class="graf graf--p">
   但是，永远不要忘记他们浪费远见和时间的方式，也不要忘记他们浪费的原因：特朗普最初是错的，所以他的核心圈子在很长一段时间内，甚至直到现在，都在用夸张的言辞和不适当的政策来促进这种错误。美国人现在将付出数十年的代价。⚪️
  </p>
