   <strong>
    12、这是一些方法介绍：利用通信应用实现开源情报
   </strong>
  </p>
  <p class="graf graf--p">
   Ginger_T 研究了一大堆可用的 Messenger 应用程序，希望查看可以使用目标的手机号码来利用哪些信息进行开源情报调查。
  </p>
  <p class="graf graf--p">
   以下是哪些应用程序在全球范围内流行的图示：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12460 jetpack-lazy-image" data-lazy-sizes="(max-width: 602px) 100vw, 602px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-20.png?resize=602%2C336&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-20.png?w=602&amp;ssl=1 602w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-20.png?resize=300%2C167&amp;ssl=1 300w" data-recalc-dims="1" height="336" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-20.png?resize=602%2C336&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="602"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12460" data-recalc-dims="1" height="336" sizes="(max-width: 602px) 100vw, 602px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-20.png?resize=602%2C336&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-20.png?w=602&amp;ssl=1 602w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-20.png?resize=300%2C167&amp;ssl=1 300w" width="602"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   重要的是要了解每个应用程序的工作方式以及作为调查人员，您留给目标人的潜在追踪痕迹，要把后者减少到最低。
  </p>
  <p class="graf graf--p">
   此处显示的所有应用程序都为研究人员/调查人员提供了不同的东西，并且在创建帐户时一定程度上依赖于目标提供的信息。
  </p>
  <p class="graf graf--p">
   👉换句话说，如果您担心自己成为 *目标*，您需要在使用任何一种应用程序时都保持对提供信息的谨慎 ——
   <strong class="markup--strong markup--p-strong">
    尽可能使用假信息
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Strava，Voxer，Skype 有潜力提供位置信息和其他个人信息； GroupMe，Line，Skype，Strava，Telegram，Wire 等应用程序将为您提供创建帐户时目标注册的名字，而不是您在联系人中可能创建的名字。
  </p>
  <p class="graf graf--p">
   Apple Facetime 和 Google Duo 在个人资料照片或个人信息方面似乎提供的不多；但是，它们确实有用的地方是追踪者可以潜在地确认目标存在 Apple 或 Google 帐户。然后在顺藤摸。
  </p>
  <p class="graf graf--p">
   您有时也必须耐心，因为并非所有应用程序都立即同步，或者同步频率不一致。
  </p>
  <p class="graf graf--p">
   👉根据您能找到的尽可能多的信息，下一步就是
   <strong class="markup--strong markup--p-strong">
    应用开源情报常规步骤
   </strong>
   了：按用户名全网捕捞、按位置追踪、按头像追踪等等。
  </p>
  <p class="graf graf--p">
   不要错过：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e4%bd%a0%e6%93%85%e9%95%bf%e6%90%9c%e7%b4%a2%ef%bc%8c%e4%bd%a0%e8%83%bd%e6%89%be%e5%88%b0%e4%b8%80%e5%88%87%ef%bc%9a%e6%b2%a1%e6%9c%89%e8%b0%b7%e6%ad%8c%e7%9a%84%e5%bc%ba%e5%a4%a7/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e4%bd%a0%e6%93%85%e9%95%bf%e6%90%9c%e7%b4%a2%ef%bc%8c%e4%bd%a0%e8%83%bd%e6%89%be%e5%88%b0%e4%b8%80%e5%88%87%ef%bc%9a%e6%b2%a1%e6%9c%89%e8%b0%b7%e6%ad%8c%e7%9a%84%e5%bc%ba%e5%a4%a7/" rel="noopener noreferrer" target="_blank">
    如果你擅长搜索，你能找到一切：搜索工具大全
   </a>
   》
  </p>
  <p class="graf graf--p">
   下面归纳一下。
  </p>
  <p class="graf graf--p">
   Google Duo — App &amp; Website
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    可以确认目标人在Duo上。
   </li>
   <li class="graf graf--li">
    它将通知目标人您已将他们添加到联系人中，您现在也加入了 Duo，因此这可能是一个隐患，具体取决于您的目标是什么。
   </li>
   <li class="graf graf--li">
    但它确实提供了机会，因为您拥有与谷歌产品相关联的真实电话号码。
   </li>
  </ul>
  <p class="graf graf--p">
   Facebook Messenger — App / Website
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    可以了解到个人资料图片和来自 Facebook 简历的详细信息。有些人会倾向于在此处使用真实照片和真实资料，他们为了 “找到朋友”，于是你知道，追踪者喜欢这个。
   </li>
  </ul>
  <p class="graf graf--p">
   Facetime
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    如果您在 iPhone 上添加联系人，它将告诉您该联系人号码是否在 Facetime 上。因此可以轻松地确定您的目标人可能正在使用Apple产品。
   </li>
   <li class="graf graf--li">
    这可能是一个机会，因为您拥有与 Apple 产品相关的真实电话号码。
   </li>
  </ul>
  <p class="graf graf--p">
   GroupMe — App / Website
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    将提供个人资料照片，不是您电话簿中联系人的姓名。
   </li>
  </ul>
  <p class="graf graf--p">
   ICQ — App / Website
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    个人资料图片、昵称、姓名、BIO 和最近一次登陆。
   </li>
   <li class="graf graf--li">
    当目标人在线时它会显示。
   </li>
  </ul>
  <p class="graf graf--p">
   Line
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    个人资料图片。对方必须更改设置，才能将其添加为联系人
   </li>
   <li class="graf graf--li">
    目标人会看到您作为 “好友推荐”，具体根据对方的设置。当目标在线时它也会显示出来
   </li>
  </ul>
  <p class="graf graf--p">
   Skype — App / Desktop
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     Skype 是追踪者的最爱之一
    </strong>
    ，因为使用它的人往往会放弃更多私密信息。
   </li>
   <li class="graf graf--li">
    Skype 将显示用户名、个人资料图片，还提供目标人设置的位置信息以及出生日期等。《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%9c%a8%e5%9c%b0%e5%9b%be%e4%b8%8a%e8%bd%bb%e8%80%8c%e6%98%93%e4%b8%be%e6%89%be%e5%88%b0%e4%b8%80%e4%b8%aa-skype-%e7%94%a8%e6%88%b7%e7%9a%84%e5%85%b7%e4%bd%93%e5%9c%b0%e7%90%86/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%9c%a8%e5%9c%b0%e5%9b%be%e4%b8%8a%e8%bd%bb%e8%80%8c%e6%98%93%e4%b8%be%e6%89%be%e5%88%b0%e4%b8%80%e4%b8%aa-skype-%e7%94%a8%e6%88%b7%e7%9a%84%e5%85%b7%e4%bd%93%e5%9c%b0%e7%90%86/" rel="noopener noreferrer" target="_blank">
     如何在地图上轻而易举找到一个 Skype 用户的具体地理位置？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   Telegram — App / Desktop
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    您可以看到用户的个人资料照片，用户名和个人简介以及他们最近一次登陆的信息。
   </li>
   <li class="graf graf--li">
    还可以看到目标人加入了哪些你知道的群组、分享过什么媒体信息等等（在桌面版）
   </li>
  </ul>
  <p class="graf graf--p">
   WeChat — APP
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    如果与目标人在同一群组中就可以看到微信ID，以及目标人自己设置的位置
   </li>
   <li class="graf graf--li">
    与 telegram 类似，您可以看到 “附近的用户”，显示他们的个人资料图片和姓名。
   </li>
  </ul>
  <p class="graf graf--p">
   WhatsApp — App / Desktop
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    个人资料图片、Last seen、状态和简介
   </li>
   <li class="graf graf--li">
    即使禁用了 “Last seen” 您仍然可以看到目标人何时在线。
   </li>
  </ul>
  <p class="graf graf--p">
   Wire — App / Desktop
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    只能看到用户设置的用户名、照片和ID
   </li>
  </ul>
  <p class="graf graf--p">
   Signal — App / Desktop
  </p>
  <p class="graf graf--p">
   据我所知，除了确认有人拥有该应用程序外，很难利用Signal的任何功能。对于隐私保护者来说，这可能是最适合您的应用程序。
  </p>
  <p class="graf graf--p">
   所有工具都可以用于防御哦！⚪️
  </p>
