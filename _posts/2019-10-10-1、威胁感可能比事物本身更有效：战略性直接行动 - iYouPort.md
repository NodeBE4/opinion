  <p class="graf graf--p">
   事物的威胁性通常比事物本身更可怕，这是20世纪著名组织者之一 Saul Alinsky 的名言。
  </p>
  <p class="graf graf--p">
   💡用即将采取行动来恐吓对手，有时候会使他们屈服于你的要求。
  </p>
  <p class="graf graf--p">
   当然，⚠️这需要充分衡量你和你的对手所处的环境。
  </p>
  <p class="graf graf--p">
   1964年，当芝加哥奥黑尔机场疯狂吸取黑人和其他被压迫的芝加哥人的生命线时，Alinsky 和 Woodlawn 组织没有屈服，他们反过来威胁要“乱扔垃圾”。
  </p>
  <p class="graf graf--p">
   成千上万的积极人士通过支付一毛钱占据了整个机场的所有便盆，其他人在这些使用者后面排起了长队，活动家阻塞洗手间并迫使所有乘客将机场变成一个巨大的厕所。
  </p>
  <p class="graf graf--p">
   请注意 Woodlawns 团队从未采取完整的行动。
  </p>
  <p class="graf graf--p">
   但消息传出后，市政厅立刻召集了组织者会面，并商议满足活动家们对奥黑尔机场的要求。
  </p>
  <p class="graf graf--p">
   其实我们从小就学到了这个原理。当你需要去摸电门或火焰时，你的父亲就会举起一只手威胁着要打你。
  </p>
  <p class="graf graf--p">
   父母的姿势足以让孩子退缩。威胁做某事有可能比实际行动更有力。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这并不是说你必须首先占据“父母”这个权威的位置。
   </strong>
  </p>
  <p class="graf graf--p">
   但是，⚠️你必须抓住你的对手的弱点 —— 他们最期待的东西就是他们的弱点。在奥黑尔机场的案例中，对手的弱点就是利润 —— 秩序是保证利润的关键 。
  </p>
  <p class="graf graf--p">
   在乌干达布西亚，Masafu 医院本应该为所有当地居民提供免费的医疗服务，但是当地人已经在近十年的时间里没有​​正常运行的X光机可用。
  </p>
  <p class="graf graf--p">
   老旧的机器已经崩溃了，但是医院管理部门没有费心去修理、也没有去买一个新的回来。
  </p>
  <p class="graf graf--p">
   有传言说管理者想要挪用资金而不是把钱花在纳税人身上，这笔资金原本应该服务于整个布西亚所有居民。
  </p>
  <p class="graf graf--p">
   2017年8月，由 Frank Kasumba 和其他活动家组织的公民代表团向该区的官员发出了一封信，要求购买新的X光机，限时一个月的最后通牒。
  </p>
  <p class="graf graf--p">
   信中说，如果他们做不到，居民会入侵他们的办公场所并无限期地占用那里。
  </p>
  <p class="graf graf--p">
   在一个星期的时间里，国会议员 Nabulindo Jane Kwoba 捐赠了大约5000美元给该地区，再加上约3000美元的补充预算专门用于修理机器。
  </p>
  <p class="graf graf--p">
   💡这封信给了官员们一个价值权衡：相比下修理机器是简单的，总好过被居民占据办公室然后再出动武警维稳，那将意味着将道德制高点的位置交予了抗议者。政客们很明白这个道理。
  </p>
  <p class="graf graf--p">
   留一个小小的台阶，哪怕它很隐秘，水也会顺势流出。
  </p>
  <p class="graf graf--p">
   2015年，为了回应私营部门与以色列犯罪的共谋，BDS运动的埃及分支宣布抵制当时的电信巨头 Orange 的全国特许经营公司 Mobinil，该公司拥有3000万埃及用户。
  </p>
  <p class="graf graf--p">
   在抵制开始之前，Orange 首席执行官 Stephane Richard 专门从巴黎飞往开罗，承诺 Orange 将从以色列撤出。
  </p>
  <p class="graf graf--p">
   显然，
   <strong class="markup--strong markup--p-strong">
    他们无法承受三千万用户的损失
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Rainforest 行动网络（RAN）等组织也使用过类似的策略。
  </p>
  <p class="graf graf--p">
   例如2000年，RAN 的行动目标是让北美森林产品公司停止从原始森林中采购木材。
  </p>
  <p class="graf graf--p">
   活动家们并没有追逐每一家公司，而是将目光瞄准了市场上最大的公司（Home Depot），一旦他们赢得那场斗争，他们就可以去接下来几家排名靠前的公司，威胁要对他们做同样的事。
  </p>
  <p class="graf graf--p">
   ⚠️请注意，
   <strong>
    行使 Alinsky 的原理不适合胆小的人
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果出现对方“不吃这一套”的情况，发出威胁的个人和团体必须
   <strong class="markup--strong markup--p-strong">
    愿意并能够实施切实的威胁
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是，操作技巧绝对是关键 ——
   <strong class="markup--strong markup--p-strong">
    你必须能给你的对手以真实的压力感
   </strong>
   。
  </p>
  <p class="graf graf--p">
   就如 Jack London 说过的：“生活并不总是意味着持有好牌，但有时容易打得很差。”
  </p>
  <p>

  <p class="graf graf--p">
   ⚠️
   <strong>
    潜在风险
   </strong>
   ：
  </p>
  <p class="graf graf--p">
   人们很容易分心。
  </p>
  <p class="graf graf--p">
   如果您只是针对涉及其他人的操作发出警告，而没有行动，不久之后许多人可能会感到无聊、气馁或不确定性的恐惧，并随着时间的推移逐步退出。
  </p>
  <p class="graf graf--p">
   如果这种情况发生，您主持的运动可能会失去信誉。
  </p>
  <p class="graf graf--p">
   所以你必须有同时能加强团队凝聚力的技巧，比如让人们看到planB的激进方案的可行性，尤其是强大的成功概率。
  </p>
  <p class="graf graf--p">
   运动要随着政治戏剧的发展而不断发展。
  </p>
  <p class="graf graf--p">
   如果你为对手提供了长时间的最后通牒，就有可能让他们在这段时间内制定策略；如果你没有考虑周全，反而会让你的团队被置于不利的落后境地。
  </p>
  <p class="graf graf--p">
   所以，要参考另一条原理：把你的对手置于决策困境中，让他们能想到的每一种策略最终都有利于你，而不是他们。参见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%8a%97%e8%ae%ae%e6%98%af%e9%9d%9e%e6%b3%95%e7%9a%84%ef%bc%8c%e9%82%a3%e5%b0%b1%e8%ae%a9%e6%97%a5%e5%b8%b8%e7%94%9f%e6%b4%bb%e6%88%90%e4%b8%ba%e6%8a%97%e8%ae%ae%ef%bc%88%e4%b8%80/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%8a%97%e8%ae%ae%e6%98%af%e9%9d%9e%e6%b3%95%e7%9a%84%ef%bc%8c%e9%82%a3%e5%b0%b1%e8%ae%a9%e6%97%a5%e5%b8%b8%e7%94%9f%e6%b4%bb%e6%88%90%e4%b8%ba%e6%8a%97%e8%ae%ae%ef%bc%88%e4%b8%80/" rel="noopener noreferrer" target="_blank">
    决策困境
   </a>
   》。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    战略性非暴力
   </strong>
  </p>
  <p class="graf graf--p">
   如上所述，采取威胁的方法时您必须有可以直接行动的planB。
  </p>
  <p class="graf graf--p">
   可能更好地服务于您的目标的框架是战略性非暴力直接行动之一。
  </p>
  <p class="graf graf--p">
   💡在战略性非暴力框架内，团体应该明确就哪些策略用于特定行动达成协议。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    ⚠️请注意，这个框架具有战略性 —— 它对暴力是否合适并没有道德判断，它并不要求您承诺甘地和平主义
   </strong>
   ，但是，它指出，“这就是我们同意在这个时候共同行动的方式 ”。
  </p>
  <p class="graf graf--p">
