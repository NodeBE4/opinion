      <p class="ib ic jc id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="141f">
       <em class="fq">
        此篇讲文革结束。这里有两个课题：其一，文革是如何结束的？有个说法，“文革是以文革的方式结束的”，从周轶君提到的对“狄克”（文青时代的张春桥）的批判，可见一斑。其二，文革真正结束了吗？包括产生文革的意识形态土壤是否已被彻底铲除？导致文革的制度元素是否得到彻底清理？文革似已远去，而这样的追问，却正与日俱增。
       </em>
      </p>
     </blockquote>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="858f">
      我出生在一九七六年九月，中国现代史上最惊心动魄的时段之一。一切由9月9日毛泽东去世引发：四人帮在随之而来的内部斗争中垮台，文革结束，与此同时，苏联觊觎、台海紧张，中共接连发起地下核试验，以振声势，内外局势皆一触即发。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="1166">
      毛泽东病逝第二天，中国国内报纸上一片黑色，统一使用“极其悲痛”、“永垂不朽”的调子。嚎啕之下，或许有低语，但官方媒体上自然看不见。
     </p>

     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="01c2">
      毛泽东生前多次扶起又打倒自己提拔的“准接班人”，如刘少奇、邓小平。接班人地位被写进党章的林彪，竟然出逃，对毛泽东是重大打击。文革中指定过王洪文，很快证明不靠谱。最后，毛泽东不再将继承人的名字写进党纲，而是通过各种方式暗示华国锋接班。其中一个证明，就是他在病逝前几个月，手写一张字条给华国锋：“……照过去方针办，你办事，我放心。”华也相信这是自己将成为最高领导人的重要依据。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="e104">
      然而，在我生日前后几天，多份国内报纸不约而同在头版右上角，以黑框突出一句：“按既定方针办”。这是什么意思？像一句接头暗号，或一句密语，其实是在毛泽东身后，围绕继承人展开的斗争。
     </p>

     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="cf40">
      根据叶剑英多年后的回忆，“照过去方针办”和“按既定方针办”，字面上并没有太大差别，但“四人帮”之一张春桥说，“按既定方针办”是毛去世前几天亲口对他说的。由此推断，中共党中央对外用哪一个说法，就承认了谁是毛泽东生前指定的接班人 — — “照过去方针办”就是华国锋，“按既定方针办”则是张春桥！
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="1b16">
      这就是当时所谓“四人帮伪造遗嘱”事件。在报纸显著位置重复“按既定方针办”，“四人帮”不断标榜自己正统。也可见，当时他们对媒体颇有影响力。很快，华国锋注意到了这只字片语的改变，背后大有深意，加快了粉碎四人帮的行动。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="c880">
      1976年10月22日，中共中央正式宣布四人帮垮台。然而，多年之后，媒体才根据当事人回忆，逐步揭示：抓捕四人帮的行动，其实在10月6日一举而成，官方消息是在十六天之后才对外公布，并报道人们奔走相告、热烈庆祝。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="7ec3">
      一直以来，我有一个困惑。常听妈妈说，“我坐月子的时候，听见窗外敲锣打鼓粉碎四人帮”。以我出生的日期推算，如果10月22日官方公布消息之后，群众才上街庆祝，妈妈坐月子的时候不可能听到锣鼓喧天。真相只有一个 — — 官方宣布之前，民间早已陆续听到“四人帮”倒台的消息，提前庆祝啦！
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="9481">
      翻看当时的海外中文报纸，印证了这个猜测。10月6日至22日间，不断传出这样的消息：“上海昨更大规模示威逾百万人参加”、“中共定购卫星转播路线连续三天将发表大消息”、“毛泽东死后北京电台首次播放音乐节目”，也有直接指出“江青等四人一网成擒”等，但细节还在猜测中。
     </p>

      反观国内报纸，10月7日至22日并非无迹可寻，真正的信息都在纸背。比如，10月7日之后，头版再不见“既定方针”，代之以强调对毛泽东的继承，“战无不胜的马克思主义、列宁主义、毛泽东思想万岁！”、“继承毛主席遗志，把无产阶级革命事业进行到底！”建立毛主席纪念堂、出版《毛泽东全集》，也被认为是对华国锋等“正牌接班人”的强化。
     </p>
