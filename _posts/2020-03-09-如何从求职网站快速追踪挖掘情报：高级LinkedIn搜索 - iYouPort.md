  <p class="graf graf--p">
   求职网站是泄漏信息最多的地方之一，因为很多人寄希望于在此寻求真实的工作机会和聘请到“真实”的人，于是他们会努力将自己的真实信息放在这里。
  </p>
  <p class="graf graf--p">
   于是求职网站是开源情报调查的金矿。
  </p>
  <p class="graf graf--p">
   不仅求职者，用人单位的资料也是非常多汁的情报资源，是许多调查的关键部分。用例非常广泛，比如 公司尽职调查、聘用前筛选、内部威胁识别等等 …… 几乎是你能想到的一切。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    学习如何根据您的用例有效地查找和收集信息，以进行分析，是本文的重点关注方向。
   </strong>
  </p>
  <p class="graf graf--p">
   这里将重点关注 LinkedIn
   <strong class="markup--strong markup--p-strong">
    高级搜索和导航
   </strong>
   ，同时还会查看
   <strong class="markup--strong markup--p-strong">
    其他数据点
   </strong>
   以支持您的调查。
  </p>
  <p class="graf graf--p">
   的确有一些很棒的工具（免费的和商业性的）可用于自动执行此过程，但是，本文将着重于如何
   <strong class="markup--strong markup--p-strong">
    手动实现
   </strong>
   此过程 —— 因此我们不依赖工具，也就是说
   <strong class="markup--strong markup--p-strong">
    没有高门槛，即便您对此技术一无所知，也可以做到
   </strong>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     收集计划
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   制定收集计划是很重要的。有许多方面与调查无关需要排除，因此该结构应该是一些信息点的指南可以将强大的目标概况报告结合在一起。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10063 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-22.png?resize=1100%2C619&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-22.png?w=1488&amp;ssl=1 1488w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-22.png?resize=300%2C169&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-22.png?resize=768%2C433&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-22.png?resize=1024%2C577&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-22.png?resize=1100%2C619&amp;ssl=1 1100w" data-recalc-dims="1" height="619" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-22.png?resize=1100%2C619&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   现在您有了一些想要收集哪些东西的想法，下一步就是需要知道在哪里可以找到这些信息。
  </p>
  <p class="graf graf--p">
   将这一想法分为几个部分，并查看您可以定位的一些数据源的样本。该列表可能不全面，但肯定能提供一个起点和可重复的工作流程。
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   公司概述很重要。在深入研究其他来源之前，在这里从宏观角度了解公司。这样做的原因是，您将在这里获得可能在整个微观的调查工作中被掠过的不同信息。
  </p>

  <p class="graf graf--p">
   了解公司结构对于理解组织管理和员工结构的工作方式非常重要。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10066 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-17.png?resize=1100%2C619&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-17.png?w=1472&amp;ssl=1 1472w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-17.png?resize=300%2C169&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-17.png?resize=768%2C432&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-17.png?resize=1024%2C576&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-17.png?resize=1100%2C619&amp;ssl=1 1100w" data-recalc-dims="1" height="619" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-17.png?resize=1100%2C619&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   招聘途径 — 对于那些可能想了解如何
   <strong class="markup--strong markup--p-strong">
    在基层影响组织
   </strong>
   的人来说，这点是经常被忽略的。对教育经历的调查经常可以展示其
   <strong class="markup--strong markup--p-strong">
    文化的内在方面、外部影响的脆弱点，以及地域方面的特征
   </strong>
   。
  </p>
  <p class="graf graf--p">
   例如，如果您正在研究组织中潜在的基层影响力，并且大多数员工来自特定的教育机构，那么它可以提供针对文化主题的挖掘机会（比如校友关系和忠诚度等）。
  </p>

  <p class="graf graf--p">
   专业关系网络通常不同于私人关系网络。比较私人关系网络与专业关系网络会很有用，以识别可能
   <strong class="markup--strong markup--p-strong">
    因私人关系而影响公司运作的关键连接
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果您准备查看收购和合并的信息，了解基于私人关系影响的任何潜在风险或威胁，这一点也很重要。

    name&gt;

      Google Dorking

   <p class="graf graf--p">
    与推特一样，LinkedIn 搜索的很大一部分完全可以通过 Google Dork 获得更多的结果。下面是一些有关如何执行此操作的示例：
   </p>
   <ul class="postList">
    <li class="graf graf--li">

     <em class="markup--em markup--li-em">
      site:linkedin.com/in OR site:linkedin.com/pub intitle:”Vice President” AND intitle:”Microsoft” AND “Sydney”
     </em>

     查找在“Microsoft”工作的说“普通话”的名为“ Jane”的人 ——
     <em class="markup--em markup--li-em">

     </em>
    </li>
    <li class="graf graf--li">
     查找在“新南威尔士大学”学习过的“Microsoft”的“副总裁” ——
     <em class="markup--em markup--li-em">
      site:linkedin.com/in OR site:linkedin.com/pub intitle:”Vice President” AND intitle:”Microsoft” AND (“Education” AND “UNSW”)
     </em>
