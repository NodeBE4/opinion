  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    1970年代联邦调查局线人 Sheila O’Connor 的故事曝光，为当今的进步组织提供了经验教训，关于，国家维稳机构的渗透究竟是个什么样  ……
   </em>
  </p>
  <p class="graf graf--p">
   1972年，Sheila O’Connor 是一个忙碌的女人。她在全国律师协会华盛顿特区的办公室工作。业余时间她参加了由青年国际党（YIPPIES！）组织的示威游行，参加了反越战士兵组织的越南退伍军人联合会。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    她还有另一份工作：她是联邦调查局的带薪线人，代号为 “牧师”。
   </strong>
  </p>
  <p class="graf graf--p">
   在过去的几个月中，有关 Richard Aoki 的说法存在一定的争议，Richard Aoki 是日裔美国人60年代的激进活动家，黑豹党的秘密成员，也是伯克利加州大学种族研究斗争的主要力量 ——
   <strong class="markup--strong markup--p-strong">
    事实上他是
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://truthout.org/news/item/12555-when-celebrated-activist-turns-out-to-be-informant-making-sense-of-richard-aokis-fbi-file" href="https://truthout.org/news/item/12555-when-celebrated-activist-turns-out-to-be-informant-making-sense-of-richard-aokis-fbi-file" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     联邦调查局的长期线人
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   暂时将 Aoki 的批评者和捍卫者的论点放在一起，就可以看清楚的一点是：
   <strong class="markup--strong markup--p-strong">
    除了穿制服的执法人员之外，人们对告密者的运作方式知之甚少。
   </strong>
  </p>
  <p class="graf graf--p">
   这是有充分理由的。 Trevor Griffey 在写下有关 Aoki 的争议时
   <a class="markup--anchor markup--p-anchor" data-href="https://truthout.org/news/item/12555-when-celebrated-activist-turns-out-to-be-informant-making-sense-of-richard-aokis-fbi-file" href="https://truthout.org/news/item/12555-when-celebrated-activist-turns-out-to-be-informant-making-sense-of-richard-aokis-fbi-file" rel="noopener noreferrer" target="_blank">
    指出
   </a>
   ：“联邦调查局很少发布其任何线人的档案。” 虽然在 Aoki 案中是例外，他的一些文件已被公开，但是被公开的部分仍然被大量编辑过，提出的问题几乎与它们能回答的问题一样多。
  </p>
  <p class="graf graf--p">
   Sheila O’Connor Rees 则不同：她的那些文件几乎没有被编辑过，因此对于理解这些内幕工作来说具有巨大的价值。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Sheila 和 John Rees
   </strong>
  </h3>
  <p class="graf graf--p">
   Sheila O’Connor Rees 是两名
   <strong class="markup--strong markup--p-strong">
    自由职业间谍
   </strong>
   之一。在FBI的编码中，她是WF5728-PSI（潜在安全消息源），以及
   <strong class="markup--strong markup--p-strong">
    她的丈夫，英国国民/新闻工作者/间谍
   </strong>
   John Herbert Rees 编码为 WF3796-PSI。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    关于政府的间谍行为归档：
    <a class="markup--anchor markup--li-anchor" data-href="https://start.me/p/xbYXdR/iyp-1" href="https://start.me/p/xbYXdR/iyp-1" rel="noopener noreferrer" target="_blank">
     列表-1
    </a>
   </li>
   <li class="graf graf--li graf--startsWithDoubleQuote">
    <a class="markup--anchor markup--li-anchor" data-href="https://start.me/p/1kod2L/iyp-direct-action5" href="https://start.me/p/1kod2L/iyp-direct-action5" rel="noopener noreferrer" target="_blank">
     “维稳面面观” 列表-5
    </a>
   </li>
  </ul>
  <p class="graf graf--p">
   在这两个人中，John Rees 更加为人所知，他创办了一份名为《信息文摘》的日志。
   <strong class="markup--strong markup--p-strong">
    该日志伪装成有关反战运动和激进活动家信息的交换所 —— 并已提供给地方警察机构。
   </strong>
  </p>
  <p class="graf graf--p">
   尽管 Rees 当时不是收钱办事的 FBI 线人，但他是 DC 地铁警察的收钱办事的线人，并且在许多方面得到了他们的资助，
   <strong class="markup--strong markup--p-strong">
    于1971年在华盛顿特区建立了红房子书店。当时的目的是让 Rees 与那些想要针对的激进活动家的警察官员保持联系。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     （
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    Valentine, Paul. F. “John Rees: His Newsletter Supplied Data on Antiwar Activists” Washington Post. June 27, 1976.
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     ）
    </em>
   </strong>
  </p>
  <p class="graf graf--p">
   Sheila Rees 取了自己娘家的姓氏 O’Connor，并利用她在DC维权人士社区的联系成为了FBI的线人。
  </p>
  <p class="graf graf--p">
   如一份文件所述，O’Connor “了解并能够提供很多信息，关于革命联盟，美中友好协会、全国律师协会、基督教青年会/社会主义工人党、青年国际组织，以及其他许多党派的内幕消息”。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    （Memo to Director, FBI, From SAC WFO, 8/24/73. From the National Lawyers Guild records, TAM.191, Box 205. Tamiment Collection, New York University. All memos and reports referenced below are contained in TAM.191, Box 205）
   </em>
  </p>
  <p class="graf graf--p">
   就像我们将要看到的那样，John Rees 在“信息文摘”上的工作导致了 Sheila 作为有偿的联邦调查局线人的工作被终止，但这是将来的事。首先，她已经花费大量的时间和精力作为华盛顿特区革命联盟的成员 —— 渗透。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    革命联盟
   </strong>
  </h3>
  <p class="graf graf--p">
   今天，除了某些激进派圈子和60年代退伍军人的社区之外，革命联盟（RU）已经鲜为人知。然而，在70年代初，这是FBI的主要针对目标。
  </p>
  <p class="graf graf--p">
   在1972年的《纽约时报》上是以一种令人窒息的描述来形容人们如何看待该组织的：“革命联盟，由毛主义团体组成的联盟，目前在10个州开展活动，并致力于摧毁政府以建立共产主义社会。” FBI认为RU是一种威胁。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    （New York Times, “Maoist Drive in the U.S. Reported by the FBI.” December 12, 1972.）
   </em>
  </p>
  <p class="graf graf--p">
   在政治活动家和伯克利自由言论运动的资深人士 Steve Hamilton 和 Robert Avakian 的倡导下，RU于1968年和1969年在海湾地区开始运营。
  </p>
  <p class="graf graf--p">
   斯坦福大学英语教授 Bruce Franklin、还有他的妻子 Jane Franklin、以及前进步劳工党成员 Leibel Bergman 也与他们一起工作。
  </p>
  <p class="graf graf--p">
   到70年代初，这个主要由刚毕业的大学生和研究生组成的团体已成为该国最具影响力的组织，其成员开始融入各种工人阶级的社区和行业，并参与较大的社会运动。
  </p>
  <p class="graf graf--p">
   在1969年和1970年期间，RU组织被两名基督教传教士 Betty Sue 和 Laurence Goff 的夫妻团队渗透，他们是一对年轻的激进工人阶级夫妇。
  </p>
  <p class="graf graf--p">
   Goffs 暗示自己能够参加RU中央委员会会议。如果组织中没有分裂，那么这对夫妇可能会继续深入，并与少数派一起遣散他们。这个被称为 Venceremos 的派别主张在美国进行某种形式的城市游击战。相反，RU主张派出干部组织工人阶级，作为为未来革命做准备的战略途径。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    （US House of Representatives Committee on Internal Security. America’s Maoists: The Revolutionary Union and the Venceremos Organization. Washington, DC. US Govt. Printing Office, 1972, 27.）
   </em>
  </p>
  <p class="graf graf--p">
   事实证明，Venceremos 的生命力有限，并在几年后就褪色了。
  </p>
  <p class="graf graf--p">
   Goffs 夫妇在后来的国会作​​证中提供了一种相当令人寒心的评估，即 Venceremos 不会持续，因为
   <strong class="markup--strong markup--p-strong">
    “在该组织展示出长期威胁之前，其成员就将被逮捕或被杀害。”
   </strong>
  </p>
  <p class="graf graf--p">
   相反：
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    <em class="markup--em markup--blockquote-em">
     Goffs 认为，在这两个组织中，RU的危险性要大得多，这是因为RU制定了长期的革命计划，其运营具有高度隐秘性，并且集中精力在工人之间建立了 “群众路线”。
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    （Op. cit. 171–172.）
   </em>
  </p>
  <p class="graf graf--p">
   到70年代初，国内情报机构已经从战略上开始瞄准 “重型” 激进分子，例如RU，该组织没有走 Weathermen 和其他城市游击队的路线 —— 其成员当时在很大程度上已入狱、被迫藏匿和躲藏，或者失效。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    （Hume, Ellen. “The ‘heavy radicals’: The intellectual cadres plant ideas — not bombs; terrorists considered less important to the cause.” Boston Globe, September 27, 1975.）
   </em>
  </p>
  <p class="graf graf--p">
   相反，
   <strong class="markup--strong markup--p-strong">
    尽管RU组织既不违法也没有控从事非法活动，但是，它却是不断上升的和间谍机构的关键目标。
   </strong>
  </p>
  <p class="graf graf--p">
   因此，当革命联盟希望扩大规模并在华盛顿特区建立一个分会时，联邦调查局就很想要渗透进去。实际上，到1973年，他们已经建立了一种做到这一点的学说：
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    <em class="markup--em markup--blockquote-em">
     总部的经验已确定，让资源和信息提供者进行RU渗透的最佳时间是在新的研究小组和集体的发展初期。WFO
    </em>
    [Washington Field Office]
    <em class="markup--em markup--blockquote-em">
     应做出努力，
    </em>
    <strong class="markup--strong markup--blockquote-strong">
     <em class="markup--em markup--blockquote-em">
      以利用任何合乎逻辑的手段使消息来源与这些团体接触
     </em>
     。
    </strong>
   </p>
  </blockquote>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    （Letter to WFO et al, Re: Revolutionary Union, 2/3/1974.）
   </em>
  </p>
  <p class="graf graf--p">
   这就是 Sheila O’Connor Rees 自愿做的。我们之所以能知道这一点，是因为
   <strong class="markup--strong markup--p-strong">
    在加入RU研究小组之前，她曾在全国律师协会担任秘书，实际上是在做间谍工作
   </strong>
   。
  </p>
  <p class="graf graf--p">
   对她来说不幸的是，NLG于1977年因这种监视方式起诉了 FBI。
  </p>
  <p class="graf graf--p">
   2007年，经过多年的法律争执，法院裁定应将监视文件移交给行会，并在纽约大学（宝贵的）Tamiment 图书馆中存放一份副本。
  </p>
  <p class="graf graf--p">
   一方面，O’Connor文件是官僚主义和记事本的汇总。
   <strong class="markup--strong markup--p-strong">
    有各种备忘录记录了她收钱办事的事实（平均每月150美元）
   </strong>
   ，并证明这些付款是合理的。
   <strong class="markup--strong markup--p-strong">
    还记录了对她进行的背景调查，并包括了她作为线人发展的进度报告。
   </strong>
  </p>
  <p class="graf graf--p">
   不过，文件的核心是简短的描述性注释（由其处理者键入），说明了O’Connor 正在学习的内容和工作目标：
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    <em class="markup--em markup--blockquote-em">
     Source 已成为 DCRU 的正式成员，除了提供 DCRU 内部活动的每周摘要外，Source 还将查找任何有关 RU “秘密成员” 的信息作为她的目标之一。
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    （Informant Report — undated.）
   </em>
  </p>
  <p class="graf graf--p">
   文件还特别概述了她应该如何从事线人工作的要求。规定她 “不要保留消息来源人报告的副本，以第三方的形式提交这些报告，不要亲自与办公室联系，并在与办公室进行电话联系时使用收费电话。”
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    （To Director, FBI From SAC WFO, 11/7/1973.）
   </em>
  </p>
  <p class="graf graf--p">
   尽管联邦调查局着眼于自身的安全性，但也特别着重于处理（和破坏）其渗透的目标组织的安全性。
  </p>
  <p class="graf graf--p">
   其中一份报告描述了该组织如何 “设计电话代码，以防止警察或其他任何人窃听 DCRU 成员的电话。”
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    （Report 5/23/74.）
   </em>
  </p>
  <p class="graf graf--p">
   另一份文件则描述了其中一个成员如何通过 “鼓励在不同的地点开会” 来谈论安全性需求：
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    <em class="markup--em markup--blockquote-em">
     首先，公开，RU内部和外部的每个人都知道该人是RU成员；第二，封闭，该成员仅被其他成员所了解；第三，保密，只有某些RU成员认识此人。
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    （Report 3/28/1974.）
   </em>
  </p>
  <p class="graf graf--p">
   线人和FBI人员（特工、Robert Golden）对他们在研究小组中所听到的内容的解释也有所歪曲，很明显，该组织为在联邦调查局权限之外开展行动而采取的大部分措施在实施之前就已经受到了很大的影响。
  </p>
  <p class="graf graf--p">
   矛盾的是，该组织想避开联邦调查局和其他机构的监视，是有理有据的；然而，还有另一种讽刺。这些讨论是在不是一个而是两个警方人员在场的情况下进行的。
  </p>
  <p class="graf graf--p">
   在1974年5月的一份线人报告中，在逐项列出研究小组各成员的情况时，发现了以下令人瞠目结舌的情况：
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    <em class="markup--em markup--blockquote-em">
     Ted Falk：消息来源称，Ted Falk 是马里兰州警方一名卧底特工使用的代号，他曾渗透到 Military Law Project (MLP) 和 VVAW/WSO。
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    （Report 5/29/1974）
   </em>
  </p>
  <p class="graf graf--p">
   然而，在这一切中，发生的不仅仅是间谍与间谍之间的阴谋诡计。很明显的是一种政治意识（尽管从线人和FBI的角度来看，是歪打正着的）。
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    现在的 RU 组织认为民族问题（黑人、墨西哥裔美国人、印第安人）不再是主要的；美国的变化，现在是由工人阶级的黑人和白人来完成的。这种立场的变化使 RU 失去了两个盟友，黑人工人代表大会和波多黎各社会党（波多黎各革命工人组织，原为青年上议院党）。
   </p>
  </blockquote>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    （Report 8/7/1974）
   </em>
  </p>
  <p class="graf graf--p">
   这是在 RU 刚刚经历了一场关于如何处理黑人和其他非白人的斗争、以及这些运动与更广泛的工人阶级运动之关系的重大内部斗争的情况下发生的。
  </p>
  <p class="graf graf--p">
   这场斗争导致了一个旨在将这三个实体合并为一个单一的共产主义组织 — — 革命共产党 (一个至今仍存在的小集团) — — 的联盟的崩溃。这对本来就担心RU影响扩大的当局来说，是一个偶然的事件。
  </p>
  <p class="graf graf--p">
   除此之外，报告还记录了1974年7月的 Avakian，当时的全国发言人、即将成为党的主席的 Avakian 和其他RU领导人访问华盛顿特区的情况，当时他们正在全国范围内进行巡回演讲，作为建党工作的一部分。
  </p>
  <p class="graf graf--p">
   有一个条目记述了 Avakian 如何在美国大学谈论RU成功地进入工人阶级，以及 October League（另一个毛派组织）的局限性，“因为它是不可靠的，没有正确地遵循马克思列宁主义路线。”
  </p>
  <p class="graf graf--p">
   这再次显示出对政治细节的明显关注 —— 以及
   <strong class="markup--strong markup--p-strong">
    线人的基本政治素养，与人们普遍所认为的愤怒的煽动者的形象形成鲜明对比
   </strong>
   。
  </p>
  <p class="graf graf--p">
   获得这些信息对 FBI 来说不仅仅是一般的好奇心，而是
   <strong class="markup--strong markup--p-strong">
    为了制造分裂和鼓励政治内讧而从中收集信息
   </strong>
   —— 他们在SDS、黑豹组织和其他组织中也曾做过这样的事，并取得了很大的效果。
  </p>
  <p class="graf graf--p">
   虽然说联邦调查局是这些分裂的主要来源是错误的，但他们确实利用了一切机会来制造这些分裂，以实现自己的利益。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%E4%BB%96%E4%BB%AC%E4%B8%BA%E4%BB%80%E4%B9%88%E5%A6%82%E6%AD%A4%E7%8C%96%E7%8B%82%EF%BC%9F%E5%9B%A0%E4%B8%BA%E4%BD%A0%E4%BB%AC%E4%BA%92%E4%B8%8D%E4%BF%A1%E4%BB%BB/" href="https://www.iyouport.org/%E4%BB%96%E4%BB%AC%E4%B8%BA%E4%BB%80%E4%B9%88%E5%A6%82%E6%AD%A4%E7%8C%96%E7%8B%82%EF%BC%9F%E5%9B%A0%E4%B8%BA%E4%BD%A0%E4%BB%AC%E4%BA%92%E4%B8%8D%E4%BF%A1%E4%BB%BB/" rel="noopener noreferrer" target="_blank">
     他们为什么如此猖狂？因为你们互不信任
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   Avakian 来了之后，DC集体的工作似乎有所收敛。结果 O’Connor 被安排到其他地方继续做线人。为了保持她对 FBI 的价值，“当消息人士注意到 DCRU 中缺乏活动时，她主动请缨，在RU主导的组织 — 巴尔的摩新闻社中发挥了积极作用 ……”
  </p>
  <p class="graf graf--p">
   可惜的是这很短暂。O’Connor 的命运最终与她所渗透的组织数量有关。她的档案在1976年戛然而止，当时 FBI 得知纽约州议会正在调查 John Rees 的 “信息文摘”，并打算传唤他和他的妻子出庭作证。
  </p>
  <p class="graf graf--p">
   联邦调查局的结论是：
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    <strong class="markup--strong markup--blockquote-strong">
     这种线人的暴露，虽然不能确定她是本局的线人，但却使局里的人很容易受到大量的调查。这也会使她作为线人的价值荡然无存。
    </strong>
    WFO 在电话中表示，这个线人并不是无价之宝。
   </p>
  </blockquote>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    （SAC, WFO, to Director FBI 3/1/1976.）
   </em>
  </p>
  <p class="graf graf--p">
   用联邦调查局的说法，”该案应该结案”。O’Connor 作为线人被撤职。
  </p>
  <p class="graf graf--p">
   那时候，O’Connor 已经在众议员、约翰·伯奇协会成员 Larry McDonald 手下做了一份研究人员的工作，此前他曾被众议院内部安全委员会 (旧的众议院非美活动委员会的新名) 解雇。至于 John Rees，到了1980年代初，他在 McDonald 的私人情报机构 “Western Goals” 工作。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    曾经和现在的线人们
   </strong>
  </h3>
  <p class="graf graf--p">
   自从70年代那段激情澎湃的日子以来，情况已经发生了很大变化，但
   <strong class="markup--strong markup--p-strong">
    使用线人仍然是执法部门的主要工作。
   </strong>
  </p>
  <p class="graf graf--p">
   比如说，FBI在其
   <a class="markup--anchor markup--p-anchor" data-href="https://www.fbi.gov/about-us/faqs" href="https://www.fbi.gov/about-us/faqs" rel="noopener noreferrer" target="_blank">
    网站上
   </a>
   显著地显示了一个常见问题解答。“利用线人协助调查犯罪活动可能涉及欺骗、侵犯个人隐私，或与可靠性和动机可能受到质疑的人合作”。但是，
   <strong class="markup--strong markup--p-strong">
    这些线人基本上仍然是自由职业者。“线人是在保密的基础上向联邦调查局提供情报的个人。他们不是联邦调查局雇用的或训练有素的雇员，尽管在某些情况下，他们可能因提供信息而得到补偿。”
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    联邦调查局和警方不仅在刑事调查中继续使用线人，而且还继续利用线人渗透到行使公民权利的和平和守法组织中
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在过去的十几年里，使用线人仍然是维稳机构的主要手段。2000年费城共和党人大会上的抗议活动就是这方面的典范 ——
   <a class="markup--anchor markup--p-anchor" data-href="https://www.aclu.org/free-speech/aclu-denounces-police-infiltration-protest-groups-gop-convention-philadelphia" href="https://www.aclu.org/free-speech/aclu-denounces-police-infiltration-protest-groups-gop-convention-philadelphia" rel="noopener noreferrer" target="_blank">
    宾夕法尼亚州警察冒充 “工人” 潜入抗议者
   </a>
   （并陷害他们被捕）。
  </p>
  <p class="graf graf--p">
   还有一些人物，比如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.motherjones.com/politics/2011/08/brandon-darby-anarchist-fbi-terrorism" href="https://www.motherjones.com/politics/2011/08/brandon-darby-anarchist-fbi-terrorism" rel="noopener noreferrer" target="_blank">
    Brandon Darby
   </a>
   ，他是联邦调查局的线人，在2008年共和党全国代表大会上以活动家为目标。而后来的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.aclu.org/blog/free-speech-national-security/foia-documents-show-fbi-was-watching-occupy-protestors-some-docs" href="https://www.aclu.org/blog/free-speech-national-security/foia-documents-show-fbi-was-watching-occupy-protestors-some-docs" rel="noopener noreferrer" target="_blank">
    ”占领运动” 也不得不面对线人渗透到该运动中的幽灵
   </a>
   。
  </p>
  <p class="graf graf--p">
   线人的存在在反抗运动组织和人民中已经产生并且会继续产生明显的负面影响。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在面对这种幽灵时，反抗者往往会陷入一种过度的偏执狂，认为警察人员无处不在，使一个群体过度关注这种事（就如您在中国看到的接二连三的抓特务行动），从而表现出不必要的脆弱性。
   </strong>
  </p>
  <p class="graf graf--p">
   线人既是地形的一部分，需要被认真对待，但同时也是一种需要被纳入背景考虑的力量。
  </p>
  <p class="graf graf--p">
   正如坦普尔大学的历史学家 Heather Thompson 在谈到60年代的框架时所说的那样，
   <strong class="markup--strong markup--p-strong">
    ”FBI 试图渗透和破坏60年代和70年代的社会和政治运动的程度并不令人惊讶。难能可贵的是，尽管有如此坚决的渗透，这些运动仍然取得了如此大的成就”。
   </strong>
   ⚪️
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    （Email exchange with Heather Ann Thompson, Aaron Leonard 12/2/2012.）
   </em>
  </p>
  <p class="graf graf--p">

    A Window Into Infiltration: The FBI Informant File of Sheila Louise O’Connor
   </a>
   》
  </p>
