      <p class="ib ic jc id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="6f62">
       藏族同学赞太发现，妈妈的生日报可用“普大喜奔”和“不明觉厉”概括。大好形势报道自然“普大喜奔”；“不明觉厉”，则是说五亿中国农民声援黎巴嫩。他们高举火把游行示威，接着通宵生产，要放粮食卫星。这是一幅荒诞画，非“1958”莫属。
      </p>
     </blockquote>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="4222">
      1958年7月28日，我母亲在青藏高原呱呱坠地；如果说“大跃进”在年初吹响了前进的号角，那么在盛夏之时，全国各地报纸上已经是喜讯频传，捷报纷飞了。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="91b5">
      我的生日报选取了母亲出生当天的家乡党报《青海日报》，以及《人民日报》和不少省份的党报，通过一些直观的昔日报道以及统计，去芜存菁，来一窥五十七年前那个火热又略显荒谬的年代。
     </p>

     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="0cc2">
      翻看1958年的报纸，你很难不被铺天盖地的喜讯及铿锵有力的口号所吸引， 随机抽取当天全国各地的十份报纸共319条新闻，进行简要的归类分析，近一半新闻不是在歌颂社会主义新成就，就是喊着“打倒美帝走狗”。套用点儿时髦的网络语汇，可以总结为“普大喜奔”与“不明觉厉”。
     </p>

      </strong>

     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="7266">
      1958年7月28日的《青海日报》头版当中，“全国早稻总产八百亿斤”的新闻在相当重要的位置，而翻看《人民日报》乃至全国各地的十份党报，这条“大放卫星”的报道也是到处开花。
     </p>

     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="9276">
      这条新闻堂而皇之登上《人民日报》的头版头条，要归功于1958年开始的大跃进运动。当年的中共八大二次会议，正式通过了“鼓足干劲、力争上游、多快好省地建设社会主义”的总路线，且要求全国粮食生产总量比上一年增产80%。随着不切实际的目标，越来越夸张的数据不断出炉。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="bcea">
      大家也许对八百亿斤的全国早稻产量没有什么具体概念，我们不妨看一下2012年的国家统计局数据，当年全国早稻总产量为3329万吨，合计665.8亿斤。这意味着五十五年后，全国早稻产量仍然与当年的“卫星”数据相差百亿斤之巨。
     </p>

     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="68f7">
      青海人民在粮食生产上也不甘落后。在《青海日报》头版这篇“西宁市路家庄社争取今年亩产五千斤”中，农业二队社长和社员们声称洋芋(学名马铃薯)平均亩产达到十万斤，因此要修订社里土地亩产两千斤的原定指标，“完全有把握”完成包含“荞麦、燕麦、豌豆”等多种农作物的亩产五千斤，更要“争取达到七千斤”，指标连涨了三倍多。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="e837">
      <strong class="id jd">
       “不明觉厉”的五亿农民
      </strong>
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="bf6c">
      细心的读者会发现，上文党报头版的早稻新闻旁，频频有“五亿农民不是好惹的”（北京日报）、“五亿农民坚决不答应”（解放日报）这样的标题出现。别误会，这可不是什么农业新闻，而与当时“不明觉厉”的国际新闻有大关系。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="336b">
      1958年的中东地区很不太平。先是黎巴嫩发生武装叛乱，接着邻国伊拉克政变建立亲苏政权。黎巴嫩总统夏蒙在外忧内患之际向美国求助。当时的总统艾森豪威尔派出部队进驻黎巴嫩，促成重新选举黎巴嫩领导人。
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="dab8">
      当年7月美国部队成功登陆黎巴嫩之时，中国的工人和城镇居民举行了大规模示威，怒火很快燃烧到了广大的农村地区。《内蒙古日报》报道称，全国十四个省(区)将近一亿农民举行了示威游行和抗议集会活动；而在《人民日报》头版“五亿农民坚决支援中东兄弟”的报道里，全中国“五亿农民”都在“坚决抗议”，“要求美英侵略者立即停止在中东放火”，“愤怒的口号响遍了原野”。
     </p>

      看来，全中国最忙碌、干劲最高的恐怕是这“五亿农民”了 — — 他们白天忙着农业生产，争取着“亩产十万斤”的丰收神话；夜晚还要“举着灯笼火把示威游行”，不仅坚决要求“美国佬滚出台湾去”，还不忘声援阿拉伯兄弟，抗议美英帝国主义在中东的侵略行径。

       我的《青海日报》记忆
      </strong>

     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="bfa0">
      即使没有这份生日报，我的生命里也从来不曾缺少《青海日报》的记忆。1981年，风华正茂的母亲进入青海日报社工作，负责《青海藏文报》的校对工作。童年的寒暑假，母亲都会时不时带着我去报社上班。九十年代初的校对办公室就与印刷车间一墙之隔，墙上开了一个窗口方便两方传递印刷小样；当时的印刷工人们还在使用活字印刷，每人手捧一个木板格，拿着镊子飞快地挑选字母，排版完毕后，磨刷来回走两遍，转动手动印刷机上的把手，巨大的滚轴转一圈，墨香四溢的报纸小样就诞生了……
     </p>
     <p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="0121">
      母亲在校对编辑这个职位上一做就是三十二年，并于2014年退休。退休前一年，她在《青海藏文报》上刊登了自己的故事，讲述人生的第一张照片。
