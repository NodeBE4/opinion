 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      “我会说 Engine 似乎是一个傀儡，被谷歌操纵 ……作为一种策略这很棒。 谷歌是一只大而坏的狼，我们正在资助的初创团体说这只大而坏的狼会吃掉我们 …… 这是马基雅维利主义”…
     </em>
    </span>
   </li>
   <li>
    <span style="color: #00ccff;">
     <em>
      这篇报道简直如洪桐县一般，但究竟谁的话可信，证据已经全部摆出来了。你可以忽视那些被称之为“回应”的狡辩。
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p graf-after--figure" id="469a">
   根据一份新
   <a class="markup--anchor markup--p-anchor" data-href="https://campaignforaccountability.org/work/engine-report/" href="https://campaignforaccountability.org/work/engine-report/" rel="noopener nofollow noreferrer" target="_blank">
    报告
   </a>
   ，一个基于华盛顿的声称代表创业科技公司声音的宣传组织，实际上是谷歌的一个小傀儡。
  </p>
  <p class="graf graf--p graf-after--p" id="fd4a">
   根据该报告，创业倡导组织
   <a class="markup--anchor markup--p-anchor" data-href="http://www.engine.is/about" href="http://www.engine.is/about" rel="noopener nofollow noreferrer" target="_blank">
    Engine
   </a>
   在其董事会和顾问委员会中至少有七名前 Google 员工和顾问。该组织三位创始人之前都在谷歌工作过;
   <strong class="markup--strong markup--p-strong">
    他们创建了谷歌最终收购的创业孵化器
   </strong>
   。过去五年来，谷歌已向 Engine 提供了
   <strong class="markup--strong markup--p-strong">
    未公开的资金
   </strong>
   。这两家公司分享了一家
   <strong class="markup--strong markup--p-strong">
    名为 S-3 Group 的游说公司
   </strong>
   ，该公司曾为 Engine 和 Google 工作过。 2011年 Engine 的首次发布会让与会者通过 RSVP 转到 Google 的电子邮件地址，该地址主要是为员工保留的，与向公众提供的 Gmail 地址完全不同。
  </p>
  <p class="graf graf--p graf-after--p" id="690c">
   报告解释说，
   <strong class="markup--strong markup--p-strong">
    在众多问题上，从专利改革、到反盗版工作、还有高技术移民，到最近对“通信规范法”第230条的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://gizmodo.com/senate-passes-sesta-controversial-anti-sex-trafficking-1823916411" href="https://gizmodo.com/senate-passes-sesta-controversial-anti-sex-trafficking-1823916411" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     修改
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，Engine 的倡导和谷歌宣称的政策偏好都是一致的。谷歌甚至资助了一篇研究论文，该论文后来被发布
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="a650">
   “公职人员需要意识到，这个所谓的创业倡导组织实际上是与硅谷最重要的D.C.影响力机器绑在一起的，他们的利益往往与颠覆性企业家的利益相冲突“，发布该报告的 Daniel Stevens 说。他是
   <a class="markup--anchor markup--p-anchor" data-href="https://campaignforaccountability.org/" href="https://campaignforaccountability.org/" rel="noopener nofollow noreferrer" target="_blank">
    Campaign for Accountability
   </a>
   的人。
  </p>
  <p class="graf graf--p graf-after--p" id="1d71">
   这里面
   <strong class="markup--strong markup--p-strong">
    没有干净的手
   </strong>
   。Campaign for Accountability
   <a class="markup--anchor markup--p-anchor" data-href="http://fortune.com/2016/08/19/google-transparency-project-2/" href="http://fortune.com/2016/08/19/google-transparency-project-2/" rel="noopener nofollow noreferrer" target="_blank">
    获得的资金
   </a>
   主要来自甲骨文，而甲骨文是谷歌的对手。
  </p>
  <p class="graf graf--p graf-after--p" id="2269">
   谷歌发言人表示，该公司“很乐意支持 Engine 的工作”，以代表华盛顿政策辩论中创业公司的观点。 “虽然我们经常就政策问题达成一致，但 Engine 是一个独立的组织，就像我们支持的其他组织一样，”该发言人说。
  </p>
  <p class="graf graf--p graf-after--p" id="ae17">
   谷歌在其网站上公开了他们对 Engine 的资金支持，该发言人称：“与 Campaign for Accountability 形成鲜明对比，他们拒绝列出其企业资助者，并且一直在帮助甲骨文表达对谷歌的怨恨”……
  </p>
  <p class="graf graf--p graf-after--p" id="ab0b">
   甲骨文华盛顿办事处高级副总裁 Ken Gleuck 在上述那份报告发布后表示谷歌的指控一点根据都没有，甲骨文与该报告完全无关。 “在阅读你的故事之前，甲骨文不知道 Engine 是否存在，我们也没有任何了解或参与这份报告，”他说，“虽然我们受宠若惊，但谷歌不应该假设我们支持谷歌的每一个负面报道。如果我们整天盯着谷歌的阴暗面，我们将耗尽
   <a class="markup--anchor markup--p-anchor" data-href="https://lobbyingdisclosure.house.gov/amended_lda_guide.html" href="https://lobbyingdisclosure.house.gov/amended_lda_guide.html" rel="noopener nofollow noreferrer" target="_blank">
    20％ 的工作时间
   </a>
   。我们还能负责红色婚礼和塑料吸管吗？“
  </p>
  <p class="graf graf--p graf-after--p" id="7308">
   Engine 组织争辩说，它已经多次在政策方面与谷歌不同，尤其是在 2014 年的网络中立战期间，证明了它的“独立性”。根据“通信法”第二章，Engine 批准重新分类宽带; 谷歌则更加低调，其互联网协会的贸易组织拒绝支持 Title II。其他问题就与谷歌无关了，比如早期公司的资本形成。该组织执行董事 Evan Engstrom 表示：“Engine 制定的政策决策完全基于我们网络中创业公司的最佳利益和反馈，而不是我们资助者的职位。”
  </p>
  <p class="graf graf--p graf-after--p" id="0aa9">
   然而，
   <strong class="markup--strong markup--p-strong">
    Engine 对于科技创业社区最具存在性的威胁之一却很少表态，这一威胁就是：大公司持续收购以阻止竞争，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/List_of_mergers_and_acquisitions_by_Alphabet" href="https://en.wikipedia.org/wiki/List_of_mergers_and_acquisitions_by_Alphabet" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     特别是谷歌
    </strong>
   </a>
   （最近的“60分钟”
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cbs.com/shows/60_minutes/video/L6xIh_6f0Jo_cnOL3J2NnKZl3hpB_oe5/how-did-google-get-so-big-/" href="https://www.cbs.com/shows/60_minutes/video/L6xIh_6f0Jo_cnOL3J2NnKZl3hpB_oe5/how-did-google-get-so-big-/" rel="noopener nofollow noreferrer" target="_blank">
    报告
   </a>
   指出）。每次合并都会提升大型科技公司的主导地位，并令创新挑战者更难获得空间。那些没有被人嘲笑的人称之为“无情”和“有时不公平”; 最典型的见证如 Facebook
   <a class="markup--anchor markup--p-anchor" data-href="https://www.recode.net/2017/3/28/15079774/facebook-stories-snapchat-instagram-copy" href="https://www.recode.net/2017/3/28/15079774/facebook-stories-snapchat-instagram-copy" rel="noopener nofollow noreferrer" target="_blank">
    复制 Snapchat
   </a>
   的应用程序并将其分发到自己庞大的网络中。马里兰大学经济学家 John Haltiwanger 将科技初创公司
   <a class="markup--anchor markup--p-anchor" data-href="https://www.axios.com/startups-slump-13-years-artificial-intelligence-us-ef914164-78f7-4783-b912-2ea50a06968d.html" href="https://www.axios.com/startups-slump-13-years-artificial-intelligence-us-ef914164-78f7-4783-b912-2ea50a06968d.html" rel="noopener nofollow noreferrer" target="_blank">
    13年的衰落
   </a>
   就归咎于 Big Tech，称
   <strong class="markup--strong markup--p-strong">
    “大公司正在进行防御性创新。
   </strong>
   ”
  </p>
  <p class="graf graf--p graf-after--p" id="3fba">
   虽然 Engine 表达了对竞争的担忧 — 特别是涉及该组织所说的受益于现有公司的规定 — 在 Engine 的
   <a class="markup--anchor markup--p-anchor" data-href="http://www.engine.is/issues" href="http://www.engine.is/issues" rel="noopener nofollow noreferrer" target="_blank">
    顶级倡导问题
   </a>
   中
   <strong class="markup--strong markup--p-strong">
    没有提及
   </strong>
   合并政策，反托拉斯或垄断政治。
  </p>
  <p class="graf graf--p graf-after--p" id="13d8">
   <strong class="markup--strong markup--p-strong">
    谷歌 和 Engine 之间的关系是 D.C. 游说社区一段时间的龙门阵主题，揭示了硅谷巨头在华盛顿发挥影响力的另一种方式
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="d834">
   除了直接游说之外，谷歌还利用其巨大的战争资金来资助智囊团、学术研究和“草根”团体，然后这些团体再提供一些看起来很像“独立观点”的玩意儿。意见和建议
   <strong class="markup--strong markup--p-strong">
    恰好符合谷歌的立场
   </strong>
   。在线出版商贸易组织 Digital Content Next 的首席执行官 Jason Kint 表示，
   <strong class="markup--strong markup--p-strong">
    “Engine 是一个对谷歌有益的 D.C.组织的例子”。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="d99e">
   <strong class="markup--strong markup--p-strong">
    在大型科技公司因其掌握用户信息的规模、权力和疏忽而遭受强烈抵制的时候，像 Engine 这样的小企业家联盟去领导政治游说活动是有道理的，就像华尔街
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2018/03/02/crapo-instead-of-taking-on-gun-control-democrats-are-teaming-with-republicans-for-a-stealth-attack-on-wall-street-reform/" href="https://theintercept.com/2018/03/02/crapo-instead-of-taking-on-gun-control-democrats-are-teaming-with-republicans-for-a-stealth-attack-on-wall-street-reform/" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     利用社区银行作为盾牌
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    以推广有利于银行的政策
   </strong>
   。“你要是说 ‘这会伤害谷歌和亚马逊’ — 这个不行，会得不到结果，如果你说‘这会伤害初创公司’，就没问题了“，21世纪福克斯和美国商会的前说客 Rick Lane 说，他最近协助活动人士在第 230 条改变中与谷歌作斗争。
  </p>
  <p class="graf graf--p graf-after--p" id="4c44">
   Engine 与谷歌的关系可以追溯到它成立之初。 2011年12月，
   <a class="markup--anchor markup--p-anchor" data-href="http://www.businessinsider.com/boonsri-dickinson-dear-dc-listen-up-we-are-the-voice-of-tech-entrepreneurs-in-sv-2011-12" href="http://www.businessinsider.com/boonsri-dickinson-dear-dc-listen-up-we-are-the-voice-of-tech-entrepreneurs-in-sv-2011-12" rel="noopener nofollow noreferrer" target="_blank">
    Business Insider 写道
   </a>
   ：“Engine Advocacy 希望让硅谷创业公司在华盛顿获得发言权，直接将这一群体与提供给 Verizon、AT＆T，微软和谷歌等电信和科技巨头的大笔资金形成鲜明对比。
  </p>
  <p class="graf graf--p graf-after--p" id="0990">
   Business Insider 的报道重点介绍了2011年12月8日在旧金山举办的 Engine 成立派对。但是，该活动的
   <a class="markup--anchor markup--p-anchor" data-href="https://archive.fo/6cNz3#selection-1419.0-1423.2" href="https://archive.fo/6cNz3#selection-1419.0-1423.2" rel="noopener nofollow noreferrer" target="_blank">
    电子邮件
   </a>
   邀请表明了 Google 员工就在幕后。 邮件是这样的：“Derek Slater 曾经在 EFF 工作，但现在是谷歌的政策主管（也许你认识他？）他正开始追求一个新的政治倡导组织引领的大规模而令人兴奋的项目，” 这封电子邮件要求 RSVP 转到另一位 Google 员工的工作电子邮件地址。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-1267 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/05/0-1.jpeg?resize=1024%2C419&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/05/0-1.jpeg?w=1024&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/05/0-1.jpeg?resize=300%2C123&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/05/0-1.jpeg?resize=768%2C314&amp;ssl=1 768w" data-recalc-dims="1" height="419" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/05/0-1.jpeg?resize=1024%2C419&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>

   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="dd95">
   Slater 还在 Google+
   <a class="markup--anchor markup--p-anchor" data-href="https://archive.fo/maPkf" href="https://archive.fo/maPkf" rel="noopener nofollow noreferrer" target="_blank">
    上宣传了 Engine
   </a>
   的启动，写道：“如果你是一家科技创业公司，或者你以其他方式支持初创公司（例如，投资者、自由设计师），你需要加入 Engine Advocacy。”Google co- 创始人 Sergey Brin 在发布之日不久
   <a class="markup--anchor markup--p-anchor" data-href="https://archive.fo/kqnNY#selection-685.116-685.630" href="https://archive.fo/kqnNY#selection-685.116-685.630" rel="noopener nofollow noreferrer" target="_blank">
    就推出
   </a>
   Engine 了。
  </p>
  <p class="graf graf--p graf-after--p" id="8042">
   Engine 回应说，Slater 不是创始人，从未在集团中担任正式角色，只是帮助实现了这一目标。但
   <strong class="markup--strong markup--p-strong">
    所有三位 Engine 的创始人 — Luis Arbulu，Joshua To 和 Joshua Mendelsohn — 都是谷歌的前雇员
   </strong>
   。在离开 Goog​​le 后不久，注册了域名
   <a class="markup--anchor markup--p-anchor" data-href="http://engineadvocacy.com/" href="http://engineadvocacy.com/" rel="noreferrer noopener nofollow" target="_blank">
    EngineAdvocacy.com
   </a>
   和当前网站
   <a class="markup--anchor markup--p-anchor" data-href="http://engine.is/" href="http://engine.is/" rel="noreferrer noopener nofollow" target="_blank">
    Engine.is
   </a>
   。在新闻报道中，Mendelsohn 被称为
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bloomberg.com/news/articles/2012-12-20/josh-mendelsohn-tech-startups-washington-lobbyist" href="https://www.bloomberg.com/news/articles/2012-12-20/josh-mendelsohn-tech-startups-washington-lobbyist" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     “科技创业公司的’华盛顿游说者”
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="3a60">
   在 Engine 启动时，创始人经营着一家名为 Hattery 的创业孵化器公司; Engine 最初安放在同一个办公室里。 Engine 的政治战略家 Mike McGeary 的薪水是由 Hattery 支付的。然后，谷歌在2013年夏天收购了 Hattery，雇用了大部分员工; 倡导团体和沟通团队去了 Engine。同年返回谷歌。Engine 说，谷歌从未直接支付任何工作人员的薪水。Mendelsohn 的新公司，一家名为
   <a class="markup--anchor markup--p-anchor" data-href="https://hangar.is/" href="https://hangar.is/" rel="noopener nofollow noreferrer" target="_blank">
    Hangar
   </a>
   的公司，没有让他发表评论。
  </p>

   <a class="markup--anchor markup--p-anchor" data-href="https://www.prnewswire.com/news-releases/californians-for-consumer-privacy-calls-on-google-to-drop-opposition-to-the-california-consumer-privacy-act-and-stop-the-tracking-of-kids-300633231.html" href="https://www.prnewswire.com/news-releases/californians-for-consumer-privacy-calls-on-google-to-drop-opposition-to-the-california-consumer-privacy-act-and-stop-the-tracking-of-kids-300633231.html" rel="noopener nofollow noreferrer" target="_blank">
    一直在反对
   </a>
   投票措施，即使 Facebook 已经在批评中放弃了反对意见。
  </p>
  <p class="graf graf--p graf-after--p" id="97c7">
   这份报告总结道：
   <strong class="markup--strong markup--p-strong">
    “谷歌对 Engine 的明显创造和支持是大企业公司如何利用其财富在华盛顿实现其目标的关键指标，这令谷歌可以躲在幕后”。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="24d6">
   Engine 和 Google 都强调了 Campaign for Accountability 与甲骨文的关系，指的是关于谷歌在构建 Android 操作系统时是否违反了甲骨文专利的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wired.com/story/the-case-that-never-ends-oracle-wins-latest-round-vs-google/" href="https://www.wired.com/story/the-case-that-never-ends-oracle-wins-latest-round-vs-google/" rel="noopener nofollow noreferrer" target="_blank">
    长达八年的激烈诉讼
   </a>
   。在这一点上，高度技术性的案例似乎更像是两个硅谷竞争对手之间的激烈争斗的借口。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="72d8">
   “由甲骨文资助的这份报告试图破坏 Engine 代表创业社区所做的工作，以推进甲骨文与谷歌之间的长期争议……”Engstrom 辩解道。
