  <p class="graf graf--p">
   2019年，我们生活在这个*国家*化的时代里。至少看起来如此。
  </p>
  <p class="graf graf--p">
   你可以看到一个非常特殊的民粹主义民族主义宏大景观 —— 土耳其的 Erdoğan、巴西的 Bolsonaro、菲律宾的 Duterte、委内瑞拉的 Maduro、印度的 Modi、匈牙利的 Orbán、美国的特朗普、中国的习近平、俄罗斯的普京，等等。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    永远是同样的咆哮和虚张声势，同样的谈论恢复所谓的昔日荣耀
   </strong>
   。但某些观察人士认为，第一印象可能会产生误导。
  </p>
  <p class="graf graf--p">
   在所有的噪音和作秀中，这些观察者们都看到了国家的死亡之痛。
  </p>
  <p class="graf graf--p">
   散文家 Rana Dasgupta 认为，“
   <strong class="markup--strong markup--p-strong">
    大男子主义的吸引力在当前是一种政治风格
   </strong>
   ，巨型边界墙建筑和仇外心理、神话和种族理论、所谓的中华民族伟大复兴的幻想承诺 ——
   <strong class="markup--strong markup--p-strong">
    这些不是治疗方案，它们就是症状本身
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   不止上述这些，同样的事发生在每一个国家，因为“
   <strong class="markup--strong markup--p-strong">
    今天的所有国家都嵌入在同一个体系中，这使得所有国家都承受着同样的压力：正是这些东西正在挤压和扭曲各地的政治生活
   </strong>
   。”
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4603 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/上.jpg?resize=1100%2C660&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/上.jpg?w=2048&amp;ssl=1 2048w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/上.jpg?resize=300%2C180&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/上.jpg?resize=768%2C461&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/上.jpg?resize=1024%2C614&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/上.jpg?resize=1100%2C660&amp;ssl=1 1100w" data-recalc-dims="1" height="660" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/上.jpg?resize=1100%2C660&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <h3 class="graf graf--p">
   <span style="font-family: impact, sans-serif;">
    <strong class="markup--strong markup--p-strong">
     想象中的社区
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   无论这是否真实，国家的消亡都是应该被鼓励的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    主导政治语话的民族身份只是集体幻想：虚无缥缈
   </strong>
   —— 就如历史学家 Benedict Anderson 创造的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amazon.com/Imagined-Communities-Reflections-Origin-Nationalism/dp/1784786756" href="https://www.amazon.com/Imagined-Communities-Reflections-Origin-Nationalism/dp/1784786756" rel="noopener noreferrer" target="_blank">
    令人难忘的短语
   </a>
   ，“近代历史的萎缩想象”。
  </p>
  <p class="graf graf--p">
   人们称之为民族国家体系的政治安排于1648年在威斯特伐利亚和约中启动，显然，该政治体系的哲学基础并不长。
  </p>
  <p class="graf graf--p">
   它的基础是我们的世界被分为人民的基本单位 — 这是一个危险的错误。
  </p>
  <p class="graf graf--p">
   Anderson 在他著名的“想象中的社区”一书中探讨了这个错误的根源，并在早期的资本主义中找到了这些根源。
  </p>
  <p class="graf graf--p">
   十六世纪的欧洲包含了许多不同的人，法国人、英国人和德国人，在任何特定地区进行大规模交流都是不可能的。当时的书籍用拉丁语等古老的语言印刷，只对精英有意义。
  </p>
  <p class="graf graf--p">
   但是企业家们认为，如果广泛的理解是可能的，那么他们就能获得巨大的利润，因此，他们开始以印刷语言出版书籍和报纸，这些印刷品汇集了相关的俗语。
  </p>
  <p class="graf graf--p">
   这些新的印刷语言有助于创造 Anderson 所描述的“
   <strong class="markup--strong markup--p-strong">
    单一语言通俗阅读的公众
   </strong>
   ”。
  </p>
  <p class="graf graf--p">
   读者们逐渐意识到“想象中的社区”的存在 —— 不相识的数百万人讲着同一种语言 —— 一个国家诞生了。
  </p>
  <p class="graf graf--p">
   然而，这不可能是整个故事。Anderson 自己的例子清楚地表明，并非所有国家都通过语言联系在一起。
  </p>
  <p class="graf graf--p">
   以印度尼西亚为例：“苏门答腊东海岸的人民不仅身处近海，横跨马六甲狭窄的海峡，而且还与马来半岛西部沿海地区的人口密切相关，但他们是种族相关的，了解彼此的言论、有一个共同宗教，等等。
  </p>
  <p class="graf graf--p">
   这些相同的苏门答腊岛与位于数千英里以外的岛屿上的安汶人没有母语、种族或宗教信仰的相关性。然而……他们已经认识到，安汶人是印尼人，马来人是外国人。”
  </p>
  <p class="graf graf--p">
   Anderson 确认国家的出现在十五和十六世纪，这是正确的，但是，对于关键的连接因素的判断却是错误的。
  </p>
  <p class="graf graf--p">
   如果不是语言让一大群人成为了一个国家，那么究竟是什么原因？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    答案不可能是政治，因为民族国家体系的政治现实实际上只是一个已经存在的想法的编纂 —— 它只是一个建立在某种基础上的建构
   </strong>
   。
  </p>
  <p class="graf graf--p">
   那么那个基础是什么？
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4604 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1中国帝国.jpg?resize=1100%2C733&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1中国帝国.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1中国帝国.jpg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1中国帝国.jpg?resize=768%2C512&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1中国帝国.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1中国帝国.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1中国帝国.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="733" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1中国帝国.jpg?resize=1100%2C733&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4604" data-recalc-dims="1" height="733" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1中国帝国.jpg?resize=1100%2C733&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1中国帝国.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1中国帝国.jpg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1中国帝国.jpg?resize=768%2C512&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1中国帝国.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1中国帝国.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图1中国帝国.jpg?resize=272%2C182&amp;ssl=1 272w" width="1100"/>
   </noscript>
  </p>
  <h3 class="graf graf--p">
   <span style="font-family: impact, sans-serif;">
    <strong class="markup--strong markup--p-strong">
     以历史为基础？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   有人说这一基础是历史。这些人认为国家是延续同一悠久历史的实体。他们想象了史诗般共享历史的画卷，它将人们与时间联系起来，将二十一世纪民族国家的公民与生活在古代世界的个人联系起来。
  </p>
  <p class="graf graf--p">
   中国经常使用这些术语来描述自己 —— 我们总是能听到中国数千年历史中历代王朝的故事。
  </p>
  <p class="graf graf--p">
   今天的共产党似乎永远不会对此闭嘴。然而，仔细研究你能发现，它揭示了不同的情况。
  </p>
  <p class="graf graf--p">
   这些所谓的中国古代帝国是由在不同时期掌权的人组成的，无论他们来自哪里；元朝是由忽必烈汗在十三世纪建立的，而清朝则是由十七世纪的满族氏族爱新觉罗（Aisin Gioro）创立的。每个帝国都是截然不同的，每当君权改变时，边界就会发生变化。
  </p>
  <p class="graf graf--p">
   在公元1000年，“中国”实际上是宋朝；在公元1500年，“中国”实际上是明朝 —— 一个与宋完全不同的实体，不论你用任何衡量标准，这都是一个不同的国家。
  </p>
  <p class="graf graf--p">
   历史学家在整个千年中保持着谨慎记录的传统，这便创造了一个“持续的中国”的
   <strong class="markup--strong markup--p-strong">
    幻觉
   </strong>
   （一定程度上是由语言连续性支撑的错觉）。
  </p>
  <p class="graf graf--p">
   但是，就这些古代历史学家而言，
   <strong class="markup--strong markup--p-strong">
    他们记录的是世界历史，而不是*一个国家*的历史。它们都不叫“中国”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   📖推荐一本书，关于国家主义如何成为一种持续塑造当今中国的霸权话语《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.sup.org/books/title/?id=21568" href="https://www.sup.org/books/title/?id=21568" rel="noopener noreferrer" target="_blank">
    After Empire: The Conceptual Transformation of the Chinese State, 1885–1924
   </a>
   》
  </p>
  <p class="graf graf--p">
   在二十世纪，最后一个朝代结束了，并且人们决定需要一个民族身份才能赶上世界其他地方。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “中国”的名字笨拙地依附于许多不同帝国的悠久历史，这些帝国具有不同的身份和边界。今天，一百年过去了，那些旧的边界问题仍然没有得到解决。
  </p>
  <p class="graf graf--p">
   共产党声称台湾和西藏实际上是中国的省份，中国公民在学校里受到的教育就是这样，就好像这是一个普遍公认的事实。
  </p>
  <p class="graf graf--p">
   但是大多数生活在台湾和西藏的人对这件事的看法截然不同。
  </p>
  <p class="graf graf--p">
   所以看来，在这段时间之后，我们仍然不能就谁是中国人和谁不是中国人达成一致，仍然无法就中国的边界达成一致。
  </p>
  <p class="graf graf--p">

   </noscript>
  </p>
  <h3 class="graf graf--p">
   <span style="font-family: impact, sans-serif;">
    <strong class="markup--strong markup--p-strong">
     以地理为基础？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   其他人则认为地理是决定因素。这也经不起考证。
  </p>
  <p class="graf graf--p">
   例如，历史学家费尔南德·布罗代尔（Fernand Braudel）将印度描述为“从未形成一个整体的地区的混合物”。
  </p>
  <p class="graf graf--p">
   印度历史包括一长串独立且往往不相关的朝代、王国、公国和苏丹国，在次大陆的各个地方兴衰。
  </p>
  <p class="graf graf--p">
   印度这个名字描述了许多不同的东西 — 有时它只是一小部分聚集在印度河上游盆地周围的城市和村庄，但有时它可以向东南方蔓延到现代缅甸，并且在其界内包含巨大的巴克特里亚帝国蔓延到现代阿富汗。
  </p>
  <p class="graf graf--p">
   1858年之后，英国统治者统治了印度次大陆，但这持续了不到一百年，直到分区。
  </p>
  <p class="graf graf--p">
   当然，印度民族主义者现在声称拥有5000年的历史，但实际上这是大量地理上不同的实体的历史集合。
  </p>
  <p class="graf graf--p">
   边界的历史通常只是军事入侵和征地的历史。任何群体都有权永久管理领土的主张必须基于这样的假设，即 有一个主要的占领时刻 —— 特定的入侵或迁移或政治吸收永久地决定了土地的合法所有者。
  </p>
  <p class="graf graf--p">
   任何后来的入侵或迁移都必须予以忽视。
  </p>
  <p class="graf graf--p">
   这显然是荒谬的：不能对首次占领做出公正的判断，土地最终会被更强大的力量所统治。
  </p>
  <p class="graf graf--p">
   📖对国家神话的捍卫不仅存在于其响应人群中，也存在于同谋学者之中。推荐一本书《The Myth of Nations: The Medieval Origins of Europe》摒弃了关于欧洲国家如何诞生的民族主义神话。论述非常清晰，很有说服力。
  </p>
  <p>

  <h3 class="graf graf--p">
   <span style="font-family: impact, sans-serif;">
    <strong class="markup--strong markup--p-strong">
     以生物学为基础？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   有些人认为国家起源于民族，因此可以被认为是生物现实，但如果你试图跟随这种逻辑分析下去，很快就会遇到问题。
  </p>
  <p class="graf graf--p">
   现在居住在英格兰并且将自己视为英国人的人群中有很大一部分实际上是日耳曼部落的后裔，他们在公元五世纪入侵不列颠群岛。
  </p>
  <p class="graf graf--p">
   当然，考虑到最近英格兰和德国之间的历史性竞争，今天英国的大多数民族主义者都不会欢迎英国人是日耳曼人后裔的说法。
  </p>
  <p class="graf graf--p">
   但是，如果深入研究我们集体历史的阴暗深处就能发现这些麻烦的事实。
  </p>
  <p class="graf graf--p">
   这是因为人类不会保持静止：他们四处移动；他们互相侵略和彼此取代，随时随地传宗接代。结果是，遗传祖先无可救药地混淆了。
  </p>
  <p class="graf graf--p">
   今天，你可以使用你的唾液在实验室中获得DNA分析，当你得到结果时，可能会兴奋地在 Facebook 上宣布你是25％的英国人、10％的法国人……等等。但测试结果显示的只是你的一些祖先在遥远过去的某个地点生活过。
  </p>
  <p class="graf graf--p">
   这些测试捕捉到的是人类迁徙和混乱的巨大而复杂的历史中一个短暂时刻的快照 —— 这一时刻非常重要，因为我们碰巧有那个时刻的参考数据库，而不是所有无数其他时刻。
  </p>
  <p class="graf graf--p">
   因此，使用现有技术真正能做到的只是告诉一个人他们与西欧的一些人或过去随机点的美洲人有遗传联系等等。此外没有更具体的东西了。
  </p>
  <p class="graf graf--p">
   伦敦大学学院网站的“分子与文化进化”页面解释说，“测试公司通常会为遗传聚类分配国家标签，而基因差异频率往往会跨越国界平稳变化。因此，法国人可能被分配了很大比例的“英国”血统。Normandy 和 Kent 在遗传上是相似的，正如你对历史和地理的期望一样，因此单凭 DNA 不容易区分英国人还是法国人。”
  </p>
  <p class="graf graf--p">
   在我看来，这种区分不是*不容易*而是*不可能*。最终，将某人描述为西欧或美洲原住民是一种极端过度的简化；但是，将一个人描述为英国人或者法国人，则完全是脱离科学领域的，进入了幻想领域。
  </p>
  <p class="graf graf--p">
   正如我们在印度尼西亚的例子中看到的那样，这些民族幻想经常会跨越现有的种族边界线。
  </p>
  <p class="graf graf--p">
   尼日利亚是大英帝国的梦想，他们将居住在尼日尔河附近的各民族合并，并将其归类为两个帝国的保护地。然后将这些保护地组合在一起，此时，豪萨、伊博、约鲁巴、江户和其他许多不同的人群，惊讶地发现，他们现在都是尼日利亚人了。
  </p>
  <p class="graf graf--p">
   当然，这些愤世嫉俗的起源很快就被遗忘了，而现在，尼日利亚民族主义者正感受着与近代历史上的帝国谎言的深刻联系。
  </p>

  <h3 class="graf graf--p">
   <span style="font-family: impact, sans-serif;">
    <strong class="markup--strong markup--p-strong">
     幻想的问题
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   如上所述，*国家*并没有因语言、历史、地理或生物学的联系而形成，但是，它们确实有一个一致的因素：
   <strong class="markup--strong markup--p-strong">
    共同的信念
   </strong>
   。
  </p>
  <p class="graf graf--p">
   你可能认为这就足够了。
   <strong class="markup--strong markup--p-strong">
    如果这些人造群体的成员在自己的身份上投入了大量的情感，那么这种情感投资也许会带来实质的身份认同
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Anderson 痛苦地指出，他使用“想象”这个词指的只是 “即使是最小的国家的成员，也永远不会知道他们的大多数同伴、也不会遇到那些同伴，甚至不会听说他们，但在每个人的心中都有一个他们共融的形象。”
  </p>
  <p class="graf graf--p">
   他认为，在这个意义上想象一下，不应该是假的。但是他错了 —— 我们
   <strong class="markup--strong markup--p-strong">
    理解到这些“想象中的社区”是错误的这绝对至关重要：想象这个词是负面意义的，因为它是对现实世界的歪曲。
   </strong>
  </p>
  <p class="graf graf--p">
   这种歪曲完全具有破坏性。
  </p>
  <p class="graf graf--p">
   作为这些想象中的社区的自愿成员，我们将自己置身于皇帝和贵族无处不在的闲散想法以及他们的现代等同物上。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    我们把那些与我们没有什么不同的人描述成竞争对手和敌人，在毫无意义的战争中为了空洞的原因而战斗和死亡
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在我们温顺地接受权威人士传递给我们的身份符号时，一些东西被植入进来了 ——
   <strong class="markup--strong markup--p-strong">
    奴隶般的，几乎是自虐的
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们创造了意大利，” 政治家 Massimo d’Azeglio 在新成立的意大利王国议会第一次会议上说道，“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amazon.com/Nations-Nationalism-Since-1780-Second/dp/1107604621" href="https://www.amazon.com/Nations-Nationalism-Since-1780-Second/dp/1107604621" rel="noopener noreferrer" target="_blank">
    现在我们必须创造意大利人
   </a>
   。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种玩世不恭的态度恰好隐藏在所有民族身份的表面之下，它完全破坏了我们通过情感投资赋予身份的任何合法性
   </strong>
   。

  <p class="graf graf--p">
   在2016年的纪录片“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.beforetheflood.com/" href="https://www.beforetheflood.com/" rel="noopener noreferrer" target="_blank">
    洪水前

   ”中，演员莱昂纳多·迪卡普里奥坐下来与德里科学环境中心的 Sunita Narain 谈论化石燃料的消耗问题。他们的谈话简要介绍了
   <strong class="markup--strong markup--p-strong">
    国家身份为解决人为气候变化等全球性危机所带来的困难。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果你在过去创造了这个问题，我们将在未来创造它，”Narain 告诉迪卡普里奥， “我真的很希望美国走向太阳能，但你还没有。”……“抱歉我直说了，我知道你是美国人……但是你们的消费习惯真的会给这个星球带来一个漏洞……你们是一个石化上瘾的国家。”
  </p>
  <p class="graf graf--p">
   这样的谈话几乎没有用，因为这一慢性危害性问题本身超越了“你们”和“我们”。

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    对话完全受到民族认同的杂草和荆棘的阻碍。即使人们真的是在寻找共同问题的解决方案，他们也无法隐藏自己的*国家*骄傲，更不会“无意义地”分担责任。
   </strong>
  </p>
  <p class="graf graf--p">
   在一个需要全球共同寻找解决方案的全球性威胁问题上，这种态度将是致命的。
  </p>
  <p class="graf graf--p">
   我最近和一个（英国）人进行了热烈的交谈，他大声坚称西方政府一直是世界上最残暴的政府。当他开始称赞中国共产党时，我觉得有必要抗议了。
  </p>
  <p class="graf graf--p">
   我开始列出从毛泽东到习近平犯下的各种对中国公民的暴行，但他并不感兴趣。相反，他想知道是什么让“我们 — 西方人”有权批评“他们 — 东方人”。
  </p>
  <p class="graf graf--p">
   这个人早些时候曾说过，他非常强烈地相信个人主权的概念，我同意他的意见，但当主题转向中国时，他似乎突然对国家主权更感兴趣了。
  </p>
  <p class="graf graf--p">
   他心爱的个人主权并不适用于生活在世界另一端（东方）的个人。
   <strong class="markup--strong markup--p-strong">
    这只是我们对*国家*存在的信念所引导出的陷阱之一。
   </strong>
  </p>
  <p class="graf graf--p">
   我们的道德感受到了极大的 —— 也许是非自然的限制：我们不再关心属于不同“想象社区”的人类困境，尽管这些人类与我们基本相同。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4609 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图6.jpg?resize=1100%2C733&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图6.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图6.jpg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图6.jpg?resize=768%2C512&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图6.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图6.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图6.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="733" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图6.jpg?resize=1100%2C733&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4609" data-recalc-dims="1" height="733" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图6.jpg?resize=1100%2C733&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图6.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图6.jpg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图6.jpg?resize=768%2C512&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图6.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图6.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/图6.jpg?resize=272%2C182&amp;ssl=1 272w" width="1100"/>
   </noscript>
  </p>
  <h3 class="graf graf--p">
   <span style="font-family: impact, sans-serif;">
    <strong class="markup--strong markup--p-strong">
     国家的遗产
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   我们对民族认同的信念是危险的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    国家形成需要经常通过武力同化或排斥任意边界以外的国家
   </strong>
   。
  </p>
  <p class="graf graf--p">
   它为生活在新界限内的许多少数民族带来了灾难，正如你看到过的，在第一次世界大战后被解体的奥匈帝国形成的国家中反犹主义的兴起。
  </p>
  <p class="graf graf--p">
   在帝国主义者到来之前就存在同样的偏见，但在帝国时代它已经被有效地压制了。一旦这些国家获得独立，旧的丑陋再次浮出水面。我们在第二次世界大战后看到了同样的事，东南亚国家的反华情绪、东非国家的反印度情绪、以及西非国家的反黎巴嫩情绪；在二十世纪晚些时候出现的则是以前属于苏联和南斯拉夫领土上的种族暴力。
  </p>
  <p class="graf graf--p">
   📖为此推荐一本书《Race And Culture: A World View》这本书很全面，包含了全球十多年的研究成果，
   <strong class="markup--strong markup--p-strong">
    表明文化资本对少数民族、民族和文明的社会和经济命运的影响远大于政治、偏见或遗传
   </strong>
   。
  </p>
  <p class="graf graf--p">
   民族主义是二十世纪巴尔干地区种族清洗的根源之一，也是德国魏玛纳粹主义兴起的根源之一。
  </p>
  <p class="graf graf--p">
   在最近几十年的东欧，它一直阻碍着法治、自由市场经济、以及民主的兴起。
  </p>
  <p class="graf graf--p">
   今天，在这个拥挤的、互联的二十一世纪，我们都成为了近邻，这一问题比以往任何时候都面临着更大的威胁。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    只要我们还抱持着“国家”的幻想，我们就会继续遭受民族主义的噩梦。如果我们想要和平与进步，我们就必须首先消除这一幻想
   </strong>
   。
  </p>

    即将出版的一本书，关于全球治理。是非常有意义的思考。**
   </em>
