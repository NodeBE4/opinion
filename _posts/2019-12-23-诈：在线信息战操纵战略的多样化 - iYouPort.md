  <p class="graf graf--p">
   大规模使用网络水军的做法似乎已经过时了。心理战进攻者一直在进化其自身。
  </p>
  <p class="graf graf--p">
   今年7月的时候我们报道了俄罗斯杜马选举引发的大规模抗议、以及随之而来的
   <strong class="markup--strong markup--p-strong">
    大规模抓捕和镇压
   </strong>
   —— 这场镇压的时间与香港抗议几乎同时。在这里看到（有视频）《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%bf%911400%e4%ba%ba%e5%9c%a8%e8%8e%ab%e6%96%af%e7%a7%91%e5%8f%8d%e5%af%b9%e6%b4%be%e6%8a%97%e8%ae%ae%e6%b4%bb%e5%8a%a8%e4%b8%ad%e8%a2%ab%e6%8d%95video/" href="https://www.iyouport.org/%e8%bf%911400%e4%ba%ba%e5%9c%a8%e8%8e%ab%e6%96%af%e7%a7%91%e5%8f%8d%e5%af%b9%e6%b4%be%e6%8a%97%e8%ae%ae%e6%b4%bb%e5%8a%a8%e4%b8%ad%e8%a2%ab%e6%8d%95video/" rel="noopener noreferrer" target="_blank">
    近1400人在莫斯科反对派抗议活动中被捕
   </a>
   》。
  </p>
  <p class="graf graf--p">
   但故事并非只有这些。
  </p>
  <p class="graf graf--p">
   在莫斯科杜马选举之前不久，
   <strong class="markup--strong markup--p-strong">
    大量俄语新闻网站和社交媒体帐户
   </strong>
   开始发布有关他们所谓的 “Lanovoy’s list” 的信息。
  </p>
  <p class="graf graf--p">
   这些媒体都声称，著名的获奖演员瓦西里·兰诺沃依（Vasily Lanovoy）已编制了一份清单，关于他认为应该在每个区选出的杜马市议员。
  </p>
  <p class="graf graf--p">
   这是一项
   <strong class="markup--strong markup--p-strong">
    媒体协调运作的信息战计划
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在这里看到更多解释《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%aa%92%e4%bd%93%e6%93%8d%e6%8e%a7%ef%bc%9a%e6%88%98%e7%95%a5%e6%80%a7%e6%ac%ba%e9%aa%97%e7%9a%84%e9%87%8d%e5%a4%b4%e6%88%8f/" href="https://www.iyouport.org/%e5%aa%92%e4%bd%93%e6%93%8d%e6%8e%a7%ef%bc%9a%e6%88%98%e7%95%a5%e6%80%a7%e6%ac%ba%e9%aa%97%e7%9a%84%e9%87%8d%e5%a4%b4%e6%88%8f/" rel="noopener noreferrer" target="_blank">
    媒体操控：战略性欺骗的重头戏
   </a>
   》。
  </p>
  <p class="graf graf--p">
   不过，这样的清单并不新鲜：在反对派候选人被集体禁止参加莫斯科杜马竞选活动之后，以阿列克谢·纳瓦尼为首的自由主义者发起了“聪明的投票”运动，该运动推荐了莫斯科辖区内获准参加竞选但不受该市现行政权青睐的候选人。
  </p>
  <p class="graf graf--p">
   所谓的 Lanovoy 清单显然是 Smart Vote 策略的替代方案，但有一个问题：瓦西里·拉诺维（Vasily Lanovoy）很快宣布他根本没有创建该清单。
  </p>
  <p class="graf graf--p">
   这就是一个混淆视听的标准策略：在人们的既有认知的基础上造假，这样才更容易被大众所信任。
  </p>
  <p class="graf graf--p">
   Meduza 发现，
   <strong class="markup--strong markup--p-strong">
    张贴虚假清单的网站和社交媒体页面均由与莫斯科市政厅有密切联系的公司控制
   </strong>
   。
  </p>
  <p class="graf graf--p">
   独立调查组织对这场惊人的
   <strong class="markup--strong markup--p-strong">
    媒体协调欺诈运动
   </strong>
   进行了挖掘。在这场欺诈面前，全球瞩目的所谓巨魔工厂的信息战行动似乎已经过时了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    清单
   </strong>
  </p>
  <p class="graf graf--p">
   这一切都始于俄罗斯社交媒体网站 VKontakte 上一些似乎是普通邻居的小群体（VK 就如中国的微博，或美国的推特）。
  </p>
  <p class="graf graf--p">
   与巨魔工厂不同，这些看起来普通的在线群体并非 一夜之间忽然诞生的。
  </p>
  <p class="graf graf--p">
   而且，这些群组的名字听起来都很清白，例如 “我的家，斯特罗吉诺”、和“维希诺FM”，但是，在9月8日莫斯科市杜马选举前的几天，数十个这些群组忽然发布了相同内容的
   <a class="markup--anchor markup--p-anchor" data-href="https://aif.ru/politics/elections/stal_izvesten_spisok_lanovogo_na_vyborah_v_mgd" href="https://aif.ru/politics/elections/stal_izvesten_spisok_lanovogo_na_vyborah_v_mgd" rel="noopener noreferrer" target="_blank">
    文章链接
   </a>
   ，宣布 “Lanovoy 的莫斯科市杜马大选候选人清单已经发布。”
  </p>
  <p class="graf graf--p">
   请注意，
   <strong class="markup--strong markup--p-strong">
    该文章由莫斯科政府所有的周报 Argumenty i fakty 发表
   </strong>
   ，每个张贴该文章链接的在线团体都着重强调了据称在该团体的选区最受宠的候选人。
  </p>
  <p class="graf graf--p">
   ⚠️甚至在
   <a class="markup--anchor markup--p-anchor" data-href="https://tvrain.ru/news/smi_rasprostranili_spisok_lanovogo_k_vyboram_v_mosgordsumu_on_oproverg_chto_sostavljal_ego-493020/" href="https://tvrain.ru/news/smi_rasprostranili_spisok_lanovogo_k_vyboram_v_mosgordsumu_on_oproverg_chto_sostavljal_ego-493020/" rel="noopener noreferrer" target="_blank">
    Lanovoy 公开明确指出
   </a>
   该清单并不是他做的之后，这些帖子仍然不断出现。
  </p>
  <p class="graf graf--p">
   绝大多数时候，辟谣的作用都非常有限，这是尤其需要警惕的 —— ⚠️宣传所依赖的基本面是占据舆论场份额。政权控制的宣传战力量可以以其无处不在的出没和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%81%87%e6%96%b0%e9%97%bb%e4%ba%a7%e4%b8%9a%e8%a7%a6%e7%9b%ae%e6%83%8a%e5%bf%83-%e4%ba%92%e8%81%94%e7%bd%91%e4%ba%ba%e7%9c%9f%e7%9a%84%e5%a6%82%e6%ad%a4%e5%a5%bd%e9%aa%97%ef%bc%9f/" href="https://www.iyouport.org/%e5%81%87%e6%96%b0%e9%97%bb%e4%ba%a7%e4%b8%9a%e8%a7%a6%e7%9b%ae%e6%83%8a%e5%bf%83-%e4%ba%92%e8%81%94%e7%bd%91%e4%ba%ba%e7%9c%9f%e7%9a%84%e5%a6%82%e6%ad%a4%e5%a5%bd%e9%aa%97%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    互联网人最差的识别意识
   </a>
   ，渗入社会。
  </p>
  <figure aria-describedby="caption-attachment-9369" class="wp-caption aligncenter" id="attachment_9369" style="width: 1024px">

  <p class="graf graf--p">
   仔细观察即可发现，这种同步发帖的现象并不奇怪。
  </p>
  <p class="graf graf--p">
   所有可疑的在线小组都主要用于发布直接来自莫斯科市政厅的内容，从有关政府举办的活动的公告、到政府拥有的新闻网站的链接。
  </p>
  <p class="graf graf--p">
   这让我想起中国网络（如微信/微博），也经常能看到一大群人蜂拥而至对官方的文章链接表达关注；⚠️
   <strong class="markup--strong markup--p-strong">
    即便其中大多数意见都是反对，但是，这种蜂拥而至本身就是在推动官方 — 而不是独立机构 — 的信息成为舆论场“热门”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这是非常值得警惕的。
  </p>
  <p class="graf graf--p">
   因为当权者控制/合作寡头公司，后者控制算法，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" rel="noopener noreferrer" target="_blank">
    是算法在决定哪些内容能成为“舆论场热门”
   </a>
   —— 也就是能被更多人看到。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    但被看到并不意味着就能真正抢夺舆论场份额，除非，大众出于某种原因*主动去追随*那些垃圾，让它们转移大众对真正重要的问题的关注
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果更多人能做到对官方的发布视而不见，当权者的宣传战就不会胜利。
  </p>
  <p class="graf graf--p">
   很可惜，这不容易做到。因为，
   <strong class="markup--strong markup--p-strong">
    在人们使用社交网络的所有动机中，“个人营销”占据大部分比重；也就是说，人们只会 “关注” 所谓的热点，不论那些热点是如何被算法操弄出来；只有热点才能帮你获得更多转发量
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️这就是当下的战略性欺骗操作最常用的方法：制造无关当权者痛痒的虚假“热点”，抢占大众的注意力，以
   <strong class="markup--strong markup--p-strong">
    远离当权者真正在乎的事
   </strong>
   。
  </p>

  <p class="graf graf--p">
   得益于 Meduza 的 Ivan Golunov 在2017年进行的
   <a class="markup--anchor markup--p-anchor" data-href="https://meduza.io/feature/2017/12/25/u-nas-narod-izobretatelnyy" href="https://meduza.io/feature/2017/12/25/u-nas-narod-izobretatelnyy" rel="noopener noreferrer" target="_blank">
    一项调查
   </a>
   ，记者们已经知道这些所谓的社区团体实际上都属于一家名为 Teomedia 的公司。
  </p>
  <p class="graf graf--p">
   ⚠️这是一家专营宣传战的公司，该公司控制着 VKontakte、Instagram 和 Facebook 上的400个本地团体，在其上发布有针对性的内容。
  </p>
  <p class="graf graf--p">
   该公司的
   <a class="markup--anchor markup--p-anchor" data-href="https://vk.com/doc-130124622_499743214" href="https://vk.com/doc-130124622_499743214" rel="noopener noreferrer" target="_blank">
    合同
   </a>
   是通过另一家公司“ Sosedi Media”（Neighbors Media）制定的。
  </p>
  <p class="graf graf--p">
   这不是新鲜事。以前就有一家名为“ Sosedi”的公司
   <a class="markup--anchor markup--p-anchor" data-href="https://noodleremover.news/sobyanin-lies-48ea41d458f5" href="https://noodleremover.news/sobyanin-lies-48ea41d458f5" rel="noopener noreferrer" target="_blank">
    曾被证明可以
   </a>
   在与当地类似的 VKontakte 团体中开展亲政府运动。
  </p>
  <p class="graf graf--p">
   请注意，⚠️从 Sosedi 开始，这些团体的所有权彼此相连，形成了一个链条，直接通向国有机构的公司：莫斯科市财产局和莫斯科国立大学国家社会系统研究所（NIISS）。
  </p>
  <p class="graf graf--p">
   这考验的
   <strong class="markup--strong markup--p-strong">
    深度调查的能力
   </strong>
   —— 所有权不会在一个明显的位置结束；越是深度的信息战操控网络，其背景就越复杂。
  </p>
  <p class="graf graf--p">
   这一链条的中间部分是莫斯科信息技术（MIT），
   <strong class="markup--strong markup--p-strong">
    该公司负责了今年夏天该市大部分扩大化的宣传活动
   </strong>
   。
  </p>
  <p class="graf graf--p">
   像MIT这样的公司从莫斯科政府机构那里获得了大笔合同和补贴，上面写着：以“在城市范围内的项目和计划的开发和实施中提供信息和分析支持。”
  </p>
  <p class="graf graf--p">
   Meduza 发现，这些公司经常在伪竞争合同拍卖中相互竞争。例如，MIT工学院（其子公司拥有 Sosedi 99％的股份）最近击败了 Krasnye Palaty（其拥有 Sosedi 1％的股份），获得了3900万卢布（595170美元）的合同。
  </p>
  <p class="graf graf--p">
   事实证明，⚠️
   <strong class="markup--strong markup--p-strong">
    莫斯科市政厅媒体和广告部的补贴对于这些连锁企业来说更加有利可图，这些连锁企业已经获得了数亿卢布的“信息和分析产品的生产和分销” —— “信息和分析” 就是
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%80%e4%b9%88%e6%98%af%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%83%85%e6%8a%a5%ef%bc%9f%e4%bd%a0%e7%9a%84%e6%8e%a8%e7%89%b9%e5%a6%82%e4%bd%95%e6%88%90%e4%b8%ba%e6%8a%93%e6%8d%95%e4%bd%a0%e7%9a%84/" href="https://www.iyouport.org/%e4%bb%80%e4%b9%88%e6%98%af%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%83%85%e6%8a%a5%ef%bc%9f%e4%bd%a0%e7%9a%84%e6%8e%a8%e7%89%b9%e5%a6%82%e4%bd%95%e6%88%90%e4%b8%ba%e6%8a%93%e6%8d%95%e4%bd%a0%e7%9a%84/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     社交媒体情报
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    和
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e6%88%98%e5%92%8c%e4%bf%a1%e6%81%af%e6%88%98/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e6%88%98%e5%92%8c%e4%bf%a1%e6%81%af%e6%88%98/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     宣传战
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    的代名词
   </strong>
   。
  </p>
  <p class="graf graf--p">
   参与该计划的公司没有回应置评请求。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    莫斯科的本地宣传生态系统
   </strong>
  </p>
  <p class="graf graf--p">
   Sosedi 和与之相关的团体已经在社交媒体上广泛散播了虚假的 Lanovoy 名单，但是，他们并没有控制最初发布该消息的国有新闻媒体。
  </p>
  <p class="graf graf--p">
   ⚠️这些网站和在线报纸构成了
   <strong class="markup--strong markup--p-strong">
    另一个
   </strong>
   由政府控制的基础设施基地：莫斯科在12个地区有142个社区，每个官方部门都有一个新闻站点。
  </p>
  <p class="graf graf--p">
   这些网站共同构成了一个单一的信息生态系统，在莫斯科市政厅的控制下运作，但是俄罗斯的媒体法规和检查机构分别向每个站点颁发了许可证。
  </p>
  <p class="graf graf--p">
   ⚠️反过来，
   <strong class="markup--strong markup--p-strong">
    这意味着像 Yandex News 这样的新闻聚合器会将那些网站解释为“单独的新闻源”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这是莫斯科政府经常利用的优势，它
   <strong class="markup--strong markup--p-strong">
    通过在网络上充斥着积极的故事来稀释负面报道 —— 就如中国和其他国家政府所擅长的那样
   </strong>
   。
  </p>
  <p class="graf graf--p">
   例如，2017年3月26日，数千名抗议者在莫斯科市中心游行时，Yandex News 的
   <a class="markup--anchor markup--p-anchor" data-href="https://meduza.io/feature/2017/03/26/mnogie-rossiyskie-smi-pochti-ne-zametili-massovyh-protestov-i-top-yandeksa-tozhe" href="https://meduza.io/feature/2017/03/26/mnogie-rossiyskie-smi-pochti-ne-zametili-massovyh-protestov-i-top-yandeksa-tozhe" rel="noopener noreferrer" target="_blank">
    头条新闻
   </a>
   是关于春节的，该市的所有官方新闻网站都是如此。
  </p>
  <p class="graf graf--p">
   多个私人承包商通过生产这些官方宣传战产品而获得了很大一部分收入。
  </p>
  <p class="graf graf--p">
   简单说，
   <strong class="markup--strong markup--p-strong">
    产业化信息战
   </strong>
   。
  </p>
  <p class="graf graf--p">
   其中最大的 IMA 咨询公司经常与自己的下属实体“竞争”其合同，就像媒体服务MIT一样。
  </p>
  <p class="graf graf--p">
   这并不是两家公司之间的唯一联系：实际上，为 IMA Consulting 工作的编辑和新闻工作者只需填写模板，直接使用通过 MIT 发送给他们的语言，这些语言是由市政厅策划人员直接提供的，是比“通稿”更高级的东西（很难看出是通稿）。
  </p>
  <p class="graf graf--p">
   ⚠️这种“统一策划”的宣传方案还会做得非常隐秘，让人们误认为发布者为“独立媒体”，见早前披露报告《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bf%a1%e6%81%af%e6%88%98%ef%bc%9a%e6%8f%ad%e9%9c%b2%e6%94%bf%e5%ba%9c%e7%9a%84%e5%ae%a3%e4%bc%a0%e6%9c%ba%e5%99%a8%e5%a6%82%e4%bd%95%e4%bc%aa%e8%a3%85%e6%88%90%e7%8b%ac%e7%ab%8b%e5%aa%92/" href="https://www.iyouport.org/%e4%bf%a1%e6%81%af%e6%88%98%ef%bc%9a%e6%8f%ad%e9%9c%b2%e6%94%bf%e5%ba%9c%e7%9a%84%e5%ae%a3%e4%bc%a0%e6%9c%ba%e5%99%a8%e5%a6%82%e4%bd%95%e4%bc%aa%e8%a3%85%e6%88%90%e7%8b%ac%e7%ab%8b%e5%aa%92/" rel="noopener noreferrer" target="_blank">
    信息战：揭露政府的宣传机器如何伪装成“独立媒体”和 NGO 以混淆视听
   </a>
   》。
  </p>
  <p class="graf graf--p">
   这类统一策划的语言以每周指令的形式出现，该指令详细说明了当地新闻媒体应涵盖的内容，从莫斯科市长谢尔盖·索比亚宁（Sergey Sobyanin）计划参加的一些具体活动，到高度详细的谈话要点。
  </p>
  <p class="graf graf--p">
   2017年，目前的 Meduza 调查编辑亚历克斯·科瓦列夫（Alexey Kovalev）发现，市政厅甚至下令当地新闻网站在有关该市市长的每个报道中都分别重复“莫斯科”和“索比亚宁”两个词。
  </p>
  <p class="graf graf--p">
   莫斯科当地媒体的消息来源证实，诸如 “信息和分析产品的生产和分发” 之类的抽象用语，实际上是对 “三个莫斯科，三个索比亚宁” 之类的规则 — 非常具体的指示 — 的掩盖。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    多样化的推动方案
   </strong>
  </p>
  <p class="graf graf--p">
   对于“党管媒体”的环境，俄罗斯记者已经体验了很多年，但是，“Lanovoy 清单”事件依旧带来了一系列
   <strong class="markup--strong markup--p-strong">
    关于在线信息战新计划的启示
   </strong>
   。
  </p>
  <p class="graf graf--p">
   其中一些举措是专门为促进莫斯科政府在9月8日城市杜马选举之前的利益而特别制定的。
  </p>
  <p class="graf graf--p">
   故事的新篇章始于以下事实：
  </p>
  <p class="graf graf--p">
   ⚠️ 关于 “Lanovoy 清单” 的文章
   <strong class="markup--strong markup--p-strong">
    并不仅是
   </strong>
   在莫斯科政府控制的社交媒体群组中发布的，
   <strong class="markup--strong markup--p-strong">
    甚至不是
   </strong>
   为亲政府信息战煽动而专门创建的社交媒体帐户发布的。
  </p>
  <p class="graf graf--p">
   ·⚠️取而代之的是，诸如“地道莫斯科”、“我的莫斯科”、和“有趣的莫斯科”之类的
   <strong class="markup--strong markup--p-strong">
    受欢迎的社会团体
   </strong>
   —— 他们也散发了这个假消息和莫斯科市杜马大选的新闻简报。他们始终使用标签＃окМосгордума（#okMosgorduma）。
  </p>
  <p class="graf graf--p">
   其中一些帖子甚至还包含旨在羞辱莫斯科人的视频广告，称这些人“让别人为他们选择”，大概是遵循 Navalny 的“聪明投票”计划。
  </p>
  <p class="graf graf--p">
   在其中一个宣传视频剪辑中显示，一对夫妇允许出租车司机为他们的约会选择一个地点，而不是自己选择地点。
  </p>
  <p class="graf graf--p">
   请注意，
   <strong class="markup--strong markup--p-strong">
    与巨魔工厂信息战或国家控制的新闻媒体不同，张贴 “Date in Dyrsk” 视频的社交媒体组织并不在莫斯科市政厅的监督下行动
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这些社交媒体组织只是
   <strong class="markup--strong markup--p-strong">
    简单地得到了回报
   </strong>
   ：这些组织定期收取赞助的费用。
  </p>
  <p class="graf graf--p">
   一位小组管理员告诉 Meduza，他估计市政厅在*非国家控制的小组* 方面的支出
   <strong class="markup--strong markup--p-strong">
    每天将高达一百万卢布
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在选举前的几周内，流行的社交媒体组织并不是开始托管政府广告浪潮的唯一意外论坛。
  </p>
  <p class="graf graf--p">
   亲政府新闻网站的广告也开始出现在洪流和盗版网站上，这些网站以前是色情游戏或在线赌场的广告。
  </p>
  <p class="graf graf--p">
   点击广告后，用户会看到一些文章，这些文章抨击抗议活动“未经许可”，还称赞俄罗斯警察的“礼貌”（同时强调了西方同僚的残酷行径），并吹嘘和推荐了一些赞成政权的候选人。
  </p>
  <p class="graf graf--p">
   Meduza 甚至在 addic7ed.com 上也找到了一些此类广告，addic7ed.com 是最受欢迎的下载字幕与盗版电影电视节目的门户之一。
  </p>
  <p class="graf graf--p">
   该网站在法国注册，每月吸引大约800万访客。但整体而言，只有4％的访客从俄罗斯登录，更不用说莫斯科了。

  </p>
  <p class="graf graf--p">
   这篇调查提醒的是对信息战新战略的警惕，中国当局也很容易会采取同样的方法。中国当局的信息战手段明显正在进化，大众应该跟上这一进度。
  </p>
  <p>
   尤其请注意⚠️：你经常能听到这样的句式：“中国人都”/“中文圈都”，存在这类句式的观点都值得怀疑。
  </p>
  <p>
   因为，显然，表达者并没有了解到全体中国人/中文圈的完整生态；尤其是在互联网上，❓您真的能确定您所看到的都是独立发表意见的个人吗？甚至，您能确定对方真的是人类吗？
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-9375 jetpack-lazy-image" data-lazy-sizes="(max-width: 940px) 100vw, 940px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/中国舆论场.png?resize=940%2C274&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/中国舆论场.png?w=940&amp;ssl=1 940w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/中国舆论场.png?resize=300%2C87&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/中国舆论场.png?resize=768%2C224&amp;ssl=1 768w" data-recalc-dims="1" height="274" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/中国舆论场.png?resize=940%2C274&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="940"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9375" data-recalc-dims="1" height="274" sizes="(max-width: 940px) 100vw, 940px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/中国舆论场.png?resize=940%2C274&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/中国舆论场.png?w=940&amp;ssl=1 940w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/中国舆论场.png?resize=300%2C87&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/中国舆论场.png?resize=768%2C224&amp;ssl=1 768w" width="940"/>

  </p>
  <p>
   ⚠️请记住：当权者操纵的信息战，其根本目的并非你是让你相信他们的言论；他们是想让你被孤立，想让你绝望，让你永远也找不到战友，无法联合起来抵抗 —— 只要所有人被拆散分化，政权就能稳固。
  </p>
  <p>
   <strong>
    在互联网上，请随时怀疑那些你所见到的令人鄙夷的言论是否真实的个人，哪怕它们大规模蜂拥而至
   </strong>
   。
  </p>
  <p>
   <a href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e4%b8%8a%e7%a9%b6%e7%ab%9f%e6%9c%89%e5%a4%9a%e5%b0%91%e4%b8%9c%e8%a5%bf%e6%98%af%e5%81%87%e7%9a%84%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    互联网上有很多东西都是假的
   </a>
   ，造假的动机各不相同，其中最值得警惕是政府信息战，因为它是有计划的、协调运作的、有明确政治目的的。⚪️
  </p>
