  <p class="graf graf--p graf-after--figure" id="066f">
   最新曝光文件显示，谷歌正计划在中国推出一个审查版的搜索引擎，将被政府列入黑名单的网站，和有关人权、民主、宗教和和平抗议的内容，均抹去，The Intercept 透露。
  </p>
  <p class="graf graf--p graf-after--p" id="6121">
   根据谷歌内部文件和熟悉该计划的人士称，该项目 — 代号为 Dragonfly — 自去年春季以来一直在进行，并在 2017 年 12 月谷歌首席执行官 Sundar Pichai 和中国政府高级官员会面后加速。
  </p>
  <p class="graf graf--p graf-after--p" id="50f6">
   谷歌的程序员和工程师团队创建了一个定制的 Android 应用程序，其中不同版本被命名为“茅台 Maotai”和“龙飞 Longfei”。该应用程序已经向中国政府展示;
   <strong class="markup--strong markup--p-strong">
    最终版本可能会在未来六到九个月内推出，等待中国官员的批准。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="32a1">
   该计划中的举措代表了谷歌对华政策的巨大转变，这将是近十年来互联网巨头在该国运营其搜索引擎的第一次。
  </p>
  <p class="graf graf--p graf-after--p" id="375c">
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     延伸阅读：至于为什么，详见这篇文章《
    </em>
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E8%B0%B7%E6%AD%8C%E5%B0%86%E6%AD%BB%E4%BA%8E%E4%BB%80%E4%B9%88-c0dd7141c922" href="https://medium.com/@iyouport/%E8%B0%B7%E6%AD%8C%E5%B0%86%E6%AD%BB%E4%BA%8E%E4%BB%80%E4%B9%88-c0dd7141c922" rel="noopener noreferrer" target="_blank">

   <strong class="markup--strong markup--p-strong">

    </em>

  <p class="graf graf--p graf-after--p" id="f3db">
   谷歌的搜索服务目前无法被中国的大多数互联网用户访问，因为它被该国所谓的大防火墙阻止。谷歌为中国建立的应用程序将遵守该国严格的审查法律，限制访问一切被习近平的共产党政权认为不利的内容。
  </p>
  <p class="graf graf--p graf-after--p" id="92f9">
   中国政府在互联网上封锁了几乎所有有关政治对手、言论自由、性别权利、新闻和学术研究的信息。例如，该国禁止有关 1989 年天安门广场大屠杀的网站，包括提及“反共产主义”和“持不同政见者”的内容。在中国社交媒体微博上，禁止提及负面描绘威权政府的书籍，如乔治奥威尔的 1984 年和动物农场。该国还审查流行的西方社交媒体网站，如 Instagram，Facebook 和 Twitter，以及美国新闻机构，如“纽约时报”和“华尔街日报”。
  </p>
  <p class="graf graf--p graf-after--p" id="db92">
   <strong class="markup--strong markup--p-strong">
    最新获得的这份文件被标记为“谷歌机密”，该文件表明，谷歌的中文搜索应用程序将自动识别和过滤被防火墙阻止的网站。当一个人进行搜索时，被禁止的网站将从结果的第一页中删除，并且将显示免责声明：“由于法定要求，某些结果可能已被删除。”
   </strong>
   网站文件中引用的示例是受到审查的包括英国新闻广播公司 BBC 和在线百科全书维基百科。
  </p>
  <p class="graf graf--p graf-after--p" id="9403">
   搜索应用程序还将“列出敏感查询的黑名单”，
   <strong class="markup--strong markup--p-strong">
    以便当人们输入某些文字或短语时，“根本不会显示任何结果”
   </strong>
   。审查将适用于整个平台：谷歌的图像搜索、自动拼写检查等，这意味着搜索结果中不会出现任何共产党政府禁止的人员、信息、或图片。
  </p>
  <p class="graf graf--p graf-after--p" id="bb07">
   据知情人士透露，在谷歌内部，有关 Dragonfly 的消息
   <strong class="markup--strong markup--p-strong">
    仅限于
   </strong>
   互联网巨头 88,000 名员工中的数百名成员知晓。该消息来源在不愿透露姓名的情况下与 The Intercept 交谈，因为他们无权与媒体联系。消息人士表示
   <strong class="markup--strong markup--p-strong">
    ，他们对谷歌在审查中的角色深感困惑，这些审查行为仅仅由公司的极少数高级管理人员策划 — — 集权，完全没有受到公众监督。
   </strong>
  </p>
  <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote graf-after--p" id="4b75">
   <p>
    <em class="markup--em markup--blockquote-em">
     “I’m against large companies and governments collaborating in the oppression of their people.”
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--blockquote" id="406d">
   “我反对大公司和政府在压迫他们的人民方面进行合作，并且认为，他们所做的事的透明度应该符合公众利益，”消息人士称，他们担心“
   <strong class="markup--strong markup--p-strong">
    谷歌在中国所做的事将成为许多其他压迫性国家效仿的模版。“
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="4193">
   人权组织大赦国际的香港研究员 Patrick Poon 告诉 The Intercept，谷歌决定遵守审查制度将是“信息时代的一大灾难”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="5fd8">
   “这不仅对中国，而且对我们所有人来说都具有非常严重的影响，信息自由和互联网自由的灾难”，Poon 说，“这将为许多其他仍在中国开展业务的公司树立一个可怕的先例，他们将难以保持不屈服于中国审查制度的原则。
   <strong class="markup--strong markup--p-strong">
    世界上最大的搜索引擎在中国已经服从于审查制度了，这是中国共产党政府的胜利 — — 它发出了一个信号，表明已经没有人会费心去挑战审查制度
   </strong>
   。“
  </p>
  <p class="graf graf--p graf-after--p" id="d27d">
   目前还不清楚谷歌最终是否会推出其审查版中国搜索平台的桌面版。目前，该公司专注于最先会推出的 Android 应用程序，这是中国很大一部分人口可以访问的应用程序。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.emarketer.com/Article/More-than-95-of-Internet-Users-China-Use-Mobile-Devices-Go-Online/1015155" href="https://www.emarketer.com/Article/More-than-95-of-Internet-Users-China-Use-Mobile-Devices-Go-Online/1015155" rel="noopener nofollow noreferrer" target="_blank">
    研究人员估计
   </a>
   ，中国有 95％ 以上的人使用移动设备上网，而 Android 是目前国内最受欢迎的移动操作系统，占有 80％ 的市场份额。
  </p>
  <p class="graf graf--p graf-after--p" id="7ff6">
   被曝光的文件显示，谷歌将把搜索应用程序作为与未命名的合作伙伴公司“合资企业”的一部分，该公司几乎可以肯定就是中国公司。然而，Dragonfly 项目的大部分工作都在位于加利福尼亚州的 Google Mountain View 总部进行，该总部位于硅谷中心圣何塞西北约 14 英里处。参与该项目的其他团队来自纽约、旧金山、桑尼维尔、圣巴巴拉、剑桥、华盛顿特区、上海、北京和东京的谷歌办事处。
  </p>
  <p class="graf graf--p graf-after--p" id="6288">
   <strong class="markup--strong markup--p-strong">
    此前，在 2006 年至 2010 年间，谷歌在中国维持过一个审查版的搜索引擎
   </strong>
   。当时，该公司在遵守中国政府政策方面遭到了严厉的批评。
  </p>
  <p class="graf graf--p graf-after--p" id="46fd">
   <strong class="markup--strong markup--p-strong">
    在 2006 年 2 月举行的一次关于美国科技公司在中国的活动的国会听证会上，众议院国际关系委员会成员称谷歌为“中国政府的工作人员”，并指责其参与审查制度的行为“令人憎恶”
   </strong>
   。“谷歌严重损害了’不作恶’政策，” 众议员 Chris Smith, RN.J.说，“
   <strong class="markup--strong markup--p-strong">
    事实上，它已成为邪恶的帮凶。”
   </strong>
  </p>

  <p class="graf graf--p graf-after--figure" id="1ccf">
   <strong class="markup--strong markup--p-strong">
    舆论抨击令该公司难以招架了，2010 年3月，谷歌宣布将其搜索服务撤出中国
   </strong>
   。在当时发表的博客文章中，该公司引用了中国政府限制言论自由、阻止网站和破解谷歌计算机系统的努力，以此作为“不再继续接受审查”的原因。
  </p>
  <p class="graf graf--p graf-after--p" id="d8e5">
   谷歌联合创始人谢尔盖•布林（Sergey Brin）出生于苏联，对审查制度的担忧似乎特别敏感，他有着在一个专制政权下生活的个人经历。在谷歌于 201 0年停止其搜索服务后，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.telegraph.co.uk/technology/google/7503641/Sergey-Brin-My-upbringing-in-USSR-helped-shape-Googles-views-on-China.html" href="https://www.telegraph.co.uk/technology/google/7503641/Sergey-Brin-My-upbringing-in-USSR-helped-shape-Googles-views-on-China.html" rel="noopener nofollow noreferrer" target="_blank">
    Brin 表示
   </a>
   该公司的反对意见涉及“极权主义势力”，并补充说他希望将搜索平台拉出国外的决定将有助于形成“更开放的互联网”。
  </p>
  <p class="graf graf--p graf-after--p" id="542e">
   然而，从那时起，中国的审查和监督变得越来越普遍。2016 年，该国政府通过了一项新的网络安全法，人权观察组织
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hrw.org/news/2016/11/06/china-abusive-cybersecurity-law-set-be-passed" href="https://www.hrw.org/news/2016/11/06/china-abusive-cybersecurity-law-set-be-passed" rel="noopener nofollow noreferrer" target="_blank">
    表示
   </a>
   ，该法案 “加强了对互联网的审查、监视和其他控制。
   <strong class="markup--strong markup--p-strong">
    ”政府正在使用新的自动化系统来监控和审查互联网，并且
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.scmp.com/news/china/policies-politics/article/2064587/chinas-move-clean-vpns-and-strengthen-great-firewall" href="https://www.scmp.com/news/china/policies-politics/article/2064587/chinas-move-clean-vpns-and-strengthen-great-firewall" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     已经破解了
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    中国人用来规避限制的隐私保护技术
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="d31d">
   多伦多大学互联网研究组织 Citizen Lab 的主任 Ron Deibert 说：“要求在中国运营的公司必须准备好负责监视用户，并根据要求将用户数据交给政府机构。” “我们还总体上发现，[在中国]的互联网审查正朝着不太透明的方向发展，在所有平台上审查或删除发生时，用户很少被通知。”
  </p>
  <p class="graf graf--p graf-after--p" id="3531">
   尽管持续受到压制，但 Google 高层的观点发生了明显的变化。中国目前拥有超过 7.5 亿互联网用户，相当于整个欧洲的人口。因此，它代表了互联网巨头的潜在巨大收入来源，这可能是其决定重新启动该国搜索平台的一个重要因素。
  </p>
  <p class="graf graf--p graf-after--p" id="47c8">
   另一个原因可能是，自谷歌上次在中国运营其搜索工具以来，该公司的领导层结构发生了显著变化。联合创始人 Brin 和 Larry Page 已经较少露面，尽管他们仍然在公司的董事会任职。
  </p>
  <p class="graf graf--p graf-after--p" id="92f3">
   谷歌现任首席执行官
   <strong class="markup--strong markup--p-strong">
    Pichai 率先推出了谷歌的“中国策略”
   </strong>
   ，他是一位 46 岁的
   <strong class="markup--strong markup--p-strong">
    印度裔美国人
   </strong>
   ，他于 2015 年 10 月掌舵。在 2016 年 6 月的加利福尼亚南部会议上，Pichai 表达了他的意图：“我关心的是为全球每个角落的用户提供服务，谷歌适合所有人，“他说。“我们希望在中国为中国用户服务。”
  </p>

   <strong class="markup--strong markup--p-strong">

    <strong class="markup--strong markup--p-strong">
     <em class="markup--em markup--p-em">

    </em>

    <strong class="markup--strong markup--p-strong">
     <em class="markup--em markup--p-em">

     </em>
    </strong>

   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     （否则他们没有今天这般强大）”
    </em>
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="1013">
   <strong class="markup--strong markup--p-strong">
    2017 年 12 月，消息人士称，Pichai 前往中国并参加了与共产党领导人王沪宁的私人会晤
   </strong>
   。王沪宁是习近平的最高外交政策顾问，被称为“中国的基辛格”。据说 Pichai 认为会议取得了成功。同月，谷歌
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2017/12/13/business/google-ai-china.html" href="https://www.nytimes.com/2017/12/13/business/google-ai-china.html" rel="noopener nofollow noreferrer" target="_blank">
    宣布
   </a>
   将在北京成立人工智能研究中心。随后于 2018 年 5 月
   <a class="markup--anchor markup--p-anchor" data-href="https://money.cnn.com/2018/05/31/technology/google-in-china-files-app/" href="https://money.cnn.com/2018/05/31/technology/google-in-china-files-app/" rel="noopener nofollow noreferrer" target="_blank">
    发布了
   </a>
   针对中国互联网用户的 Google 文件管理应用程序。然后，在 7 月，谷歌在 WeChat 上
   <a class="markup--anchor markup--p-anchor" data-href="https://download.cnet.com/blog/download-blog/google-ai-guesses-what-youre-drawing-in-a-mini-app-released-for-wechat-in-china/" href="https://download.cnet.com/blog/download-blog/google-ai-guesses-what-youre-drawing-in-a-mini-app-released-for-wechat-in-china/" rel="noopener nofollow noreferrer" target="_blank">
    推出了
   </a>
   一款“Guess The Sketch”游戏，微信是一个在中国很受欢迎的中文通信和社交媒体平台。
  </p>
  <p class="graf graf--p graf-after--p" id="c7c4">
   最后的结果将是
   <strong class="markup--strong markup--p-strong">
    审查版搜索应用程序的启动 — Dragonfly 项目
   </strong>
   。据熟悉该计划的消息人士称，该应用程序发布的时间将取决于
   <strong class="markup--strong markup--p-strong">
    两个主要因素：中国政府的批准，以及谷歌对其应用程序将优于其主要竞争对手百度提供的搜索服务的信心
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="994a">
   谷歌内部人士表示，目前尚不清楚该公司何时会获得北京官员的批准，因为美中之间不断升级的“贸易战”已经放缓了这一进程。然而，谷歌的搜索引擎主管
   <strong class="markup--strong markup--p-strong">
    Ben Gomes 在上个月的一次会议上告诉员工，他们必须准备好在短时间内推出中文搜索应用程序，因为有可能“世界突然发生变化，也许[唐纳德·特朗普总统]决定他的新朋友是习近平。
   </strong>
   “
  </p>
  <p class="graf graf--p graf-after--p" id="d2d0">
   谷歌和中国外交部拒绝回应评论请求。
