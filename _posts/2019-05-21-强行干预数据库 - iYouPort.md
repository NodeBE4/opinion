  <h1 class="entry-title">
   “强行干预”数据库
  </h1>
 </header>
 <div class="entry-meta">
  <span class="byline">
   <a href="https://www.iyouport.org/author/don-evans/" rel="author" title="由McCaffrey发布">

 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      你生活的方方面面都被收集到一个大型的数据库中，如果当局通过对你的信息进行分析后认为你”有风险”，你就会被强行”干预”，干预方法包括查水表、喝茶、逮捕和强行”治疗”。这是谁干的？
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   警察、社会服务和卫生服务工作者正在使用共享数据库来追踪弱势群体的行为 — — 包括未成年人和无家可归者 — — 几乎没有监督，更没有经过同意。
  </p>
  <p>
   欢迎来到加拿大。
  </p>
  <p class="graf graf--p">
   安大略省社区安全和惩教署（MCSCS）的文件显示，至少有两个省份 — 安大略省和萨斯喀彻温省 — 维持着一个名叫“风险驱动的跟踪数据库”，用于收集高度敏感的关于人们私生活的信息。数据库中的信息包括一个人是否使用毒品、是否是攻击的受害者，还是生活在所谓的“负面社区”。
  </p>
  <p class="graf graf--p">
   风险驱动的跟踪数据库（RTD）是警务协作方法的一部分，被称为
   <a class="markup--anchor markup--p-anchor" data-href="https://news.vice.com/en_ca/article/7xqmny/heres-who-stands-to-gain-from-a-radical-policing-approach-in-canada" href="https://news.vice.com/en_ca/article/7xqmny/heres-who-stands-to-gain-from-a-radical-policing-approach-in-canada" rel="noopener noreferrer" target="_blank">
    Hub 模型
   </a>
   ，合作伙伴包括警方、学校员工、社会工作者、医疗保健工作者和省政府。
  </p>
  <p class="graf graf--p">
   任何被认为“有风险”成为犯罪分子或受害者的人的信息，都在民间机构和警察之间共享，并且
   <strong class="markup--strong markup--p-strong">
    在评估针对一个人进行的旨在降低其风险等级的“快速干预行动”时也会将其添加到数据库中。干预方式可能包括敲门查水表、喝茶聊天、强迫住院或逮捕等
   </strong>
   。
  </p>
  <p class="graf graf--p">
   分析来自RTD的数据以确定趋势 — 例如，特定区域的药物使用激增 — 目标是生成有效部署资源的规划数据，并创建可加速 Hub 模型下干预的“社区概况”，来自
   <a class="markup--anchor markup--p-anchor" data-href="https://www.publicsafety.gc.ca/cnt/rsrcs/pblctns/2015-plcy-mdps/index-en.aspx" href="https://www.publicsafety.gc.ca/cnt/rsrcs/pblctns/2015-plcy-mdps/index-en.aspx" rel="noopener noreferrer" target="_blank">
    2015年加拿大公共安全报告
   </a>
   。
  </p>
  <p class="graf graf--p">
   萨斯喀彻温省和安大略省官员表示，RTD 中的数据（有时称为萨斯喀彻温省的“中心数据库”）通过删除人员姓名和出生日期等详细信息进行“去识别”，尽管专家告诉主板说，这些数据可能永远不会被删除。
  </p>
  <p class="graf graf--p">
   主板调查 — 涉及通过 MCSCS、警察和城市文件进行梳理 — 发现，在2017年，12至17岁的儿童是安大略省几个地区数据库中最常见的年龄组，并且未经同意就进行了一些干预。
   <strong class="markup--strong markup--p-strong">
    在某些情况下，年仅6岁的儿童也受到干预
   </strong>
   。
  </p>
  <p class="graf graf--p">
   那么人们的信息是如何被添加到数据库中的？
  </p>
  <p class="graf graf--p">
   Hub 模型旨在将警察与社区成员联系起来，以评估针对潜在“风险”人群的干预措施。
  </p>
  <p class="graf graf--p">
   例如，可能会要求警察一次又一次地回应某人的破坏性但非犯罪行为。根据中心模型，相关官员可以将目标人的情况带到中心 — 可能包括儿童福利、成瘾或住房援助机构的工作人员 — 并询问其他机构是否可以进行干预。
  </p>
  <p class="graf graf--p">
   在随后的评估期间，参与者之间共享关于该人的信息并输入 RTD。当地执法人员，社会工作者和卫生工作者可以知道该人的身份，但是当他们的信息
   <a class="markup--anchor markup--p-anchor" data-href="https://www.usask.ca/cfbsjs/research/pdf/research_reports/MeasuringChange.pdf" href="https://www.usask.ca/cfbsjs/research/pdf/research_reports/MeasuringChange.pdf" rel="noopener noreferrer" target="_blank">
    被添加到
   </a>
   RTD 时，可能不包括可识别该人的详细信息。如果机构集体决定该人处于“急剧升高”的风险水平，则会采取干预措施。如果中心从业者认为某人受到高度伤害，则可以在未经同意的情况下进行干预。
  </p>
  <p class="graf graf--p">
   根据 MCSCS 的文件，截至2018年4月，在加拿大和美国的城镇中有100多个这样的中心正在运营，其中37个在安大略省（其中Hub通常被称为“情况表”）为风险驱动的跟踪数据库做出了贡献。预计到今年年底总共将有55个。
  </p>
  <p class="graf graf--p">
   安大略省和萨斯喀彻温省至少有这两个省份维持自己的 RTD 版本，但主板无法确认数据库是否在其他省份使用。
  </p>
  <p class="graf graf--p">
   萨斯喀彻温大学的学术研究员兼 RTD 的首席开发人员 Chad Nilson 博士没有回答有关该数据库的问题。只
   <a class="markup--anchor markup--p-anchor" data-href="https://webcache.googleusercontent.com/search?q=cache:rAxMLzLZU_wJ:https://www.albertacrimeprevention.com/2017-accpa-crime-prevention-conference/+&amp;cd=1&amp;hl=en&amp;ct=clnk&amp;gl=ca&amp;client=firefox-b-d" href="https://webcache.googleusercontent.com/search?q=cache:rAxMLzLZU_wJ:https://www.albertacrimeprevention.com/2017-accpa-crime-prevention-conference/+&amp;cd=1&amp;hl=en&amp;ct=clnk&amp;gl=ca&amp;client=firefox-b-d" rel="noopener noreferrer" target="_blank">
    说过
   </a>
   RTD 正在“加拿大各地”使用。
  </p>
  <p class="graf graf--p">
   对警察和社区合作伙伴委托的安大略省两个城市的情况表进行的评估显示，
   <strong class="markup--strong markup--p-strong">
    一些干预措施已经以强迫住院或逮捕而告终
   </strong>
   。对 Waterloo “情况表”的评估指出了一个人在被干预后遭受非自愿住院治疗的情况。
  </p>
  <p class="graf graf--p">
   根据 MCSCS 文件，2017年为12至17岁的儿童举办了300多次与RTD相关的谈话，为6至11岁的儿童举办了30次。安大略省2017年度 RTD 报告指出，12至17岁的儿童是数据库中“最脆弱”的年龄组，并且在几个地区的 RTD 中最为普遍。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-1108 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-107.png?resize=1100%2C700&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-107.png?w=1260&amp;ssl=1 1260w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-107.png?resize=300%2C191&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-107.png?resize=768%2C489&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-107.png?resize=1024%2C652&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-107.png?resize=1100%2C700&amp;ssl=1 1100w" data-recalc-dims="1" height="700" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-107.png?resize=1100%2C700&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-1108" data-recalc-dims="1" height="700" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-107.png?resize=1100%2C700&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-107.png?w=1260&amp;ssl=1 1260w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-107.png?resize=300%2C191&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-107.png?resize=768%2C489&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-107.png?resize=1024%2C652&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-107.png?resize=1100%2C700&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Hub 干预要求警察、教育工作者、医生和社会工作者分享关于弱势群体的极其敏感的信息 — 并将其添加到省级数据库中 — 这个过程充满了隐私侵权的可能性。
  </p>
  <p class="graf graf--p">
   RTD 中的数据可能包括一个人的年龄组、性别、位置以及用于描述个人情况的100多个所谓的“风险因素”。根据 MCSCS 文件，2017年数据库指出的中人们最常见的所谓风险因素是心理健康（包括“
   <a class="markup--anchor markup--p-anchor" data-href="http://ingersollnplc.ca/wp-content/uploads/2015/10/Oxford-Situation-Table-6-MONTH-REPORT-3-1.pdf" href="http://ingersollnplc.ca/wp-content/uploads/2015/10/Oxford-Situation-Table-6-MONTH-REPORT-3-1.pdf" rel="noopener noreferrer" target="_blank">
    疑似”心理健康问题
   </a>
   ），犯罪参与、吸毒和“反社会/消极行为”，定义为“讨厌”[或者]破坏性的“行为。
  </p>
  <p class="graf graf--p">
   当被问及情况表使用有时模糊的风险因素来证明干预的合理性时，MCSCS 重申被认为具有高危害风险的个人比如表现出“多重风险因素”，则被认为需要采取协作方式进行干预。
  </p>
  <p class="graf graf--p">
   2014年，萨斯喀彻温省的 IPC
   <a class="markup--anchor markup--p-anchor" data-href="https://oipc.sk.ca/assets/foip-investigation-105-2014.pdf" href="https://oipc.sk.ca/assets/foip-investigation-105-2014.pdf" rel="noopener noreferrer" target="_blank">
    已经对
   </a>
   该省 Hub 相关隐私侵权的行为进行了调查。该调查发现了隐私保护中的“缺陷”，并指出某些 Hub 数据库包含个人身份信息。
  </p>
  <p class="graf graf--p">
   该报告还发现，Hubs 的目标人群并未获知如何提交隐私投诉，并且
   <strong class="markup--strong markup--p-strong">
    Hub 代理商至少两次搜索 Facebook 帖子作为关于某人风险等级的信息来源
   </strong>
   。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-1109 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-108.png?resize=1100%2C748&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-108.png?w=1600&amp;ssl=1 1600w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-108.png?resize=300%2C204&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-108.png?resize=768%2C522&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-108.png?resize=1024%2C696&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-108.png?resize=1100%2C748&amp;ssl=1 1100w" data-recalc-dims="1" height="748" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-108.png?resize=1100%2C748&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-1109" data-recalc-dims="1" height="748" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-108.png?resize=1100%2C748&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-108.png?w=1600&amp;ssl=1 1600w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-108.png?resize=300%2C204&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-108.png?resize=768%2C522&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-108.png?resize=1024%2C696&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-108.png?resize=1100%2C748&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   萨斯喀彻温省 IPC 在2017年告诉 Motherboard 采取了纠正措施，例如确保 Hubs 不再使用 Facebook 数据来评估个人的风险等级，并且 IPC 对这些步骤感到满意。
  </p>
  <p class="graf graf--p">
   虽然有关部门声称这些数据是“去识别的”，但 RTD 中使用的100多个风险因素可能会描述一个人的生活私密细节，监视他们是否喝酒，是否难以找到稳定的住房，是否逃学，失业，或与所谓的“负面同伴”联系。
  </p>
  <p class="graf graf--p">
   特别需要关注的是，RTD 被用于预测性监管的目的 — 这是一种利用数据分析来识别犯罪热点的严重有争议的策略。也就是少数派报告的那种东西。一份报告指出，SPPAL “扩展了 Hub 模型的风险跟踪系统。”
  </p>
  <p class="graf graf--p">
   目前还不清楚加拿大当局使用什么算法来分析 RTD，但自动化工具也存在风险。专家指出，
   <strong class="markup--strong markup--p-strong">
    支持 PredPol 的算法是最广泛使用的预测性警务技术之一，其根本上存在缺陷，可能导致过度监管，特别是对于边缘化社区
   </strong>
   。
  </p>
  <p class="graf graf--p">
   根据主板获得的2016年协议副本，警方向 RTD 发送数据的服务必须与 MCSCS 签署协议，禁止他们在没有得到部门书面同意的情况下与媒体就数据库问题进行交谈。
  </p>
  <p class="graf graf--p">
   McPhail 表示，如果没有提高透明度，Hubs 将继续对他们在 RTD 中编译数据的弱势群体的隐私权构成风险。
  </p>
