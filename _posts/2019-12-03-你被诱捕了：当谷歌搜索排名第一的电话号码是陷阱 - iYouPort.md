  <p class="graf graf--p">
   有些人认为自己能对网络犯罪分子“免疫”。
  </p>
  <p class="graf graf--p">
   他们认为，“我甚至不在他们的雷达上”； “我有什么机会成为目标呢？我不出名、也没有几百万美元的存款” ……
  </p>
  <p class="graf graf--p">
   很遗憾他们错了。
  </p>
  <p class="graf graf--p">
   这里是一个警示性的故事：一位名叫 Bob 的绅士报告了一个最近亲历的恐怖。他在一家国土安全咨询公司工作，所以他看起来很了解在线预防措施。但还是中计了。
  </p>
  <p class="graf graf--p">
   Bob 正在使用 Yahoo Small Business 用于工作，他需要升级他的服务。
  </p>
  <p class="graf graf--p">
   他的直觉是正确的。鉴于著名的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.komando.com/happening-now/561022/yahoo-breach-settlement-117m-heres-what-happens-next" href="https://www.komando.com/happening-now/561022/yahoo-breach-settlement-117m-heres-what-happens-next" rel="noopener noreferrer" target="_blank">
    雅虎数据泄漏事件
   </a>
   ，Bob 正在进行尽职调查并及时更新软件。
  </p>
  <p class="graf graf--p">
   他有几个问题，于是去谷歌搜索雅虎的小企业帮助热线并打电话询问。
  </p>
  <p class="graf graf--p">
   他不可能知道他搜索到的电话号码是假的。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    诈骗者找到了一种方法可以将他们自己的假号码推到谷歌搜索的顶端
   </strong>
   。Bob 被骗了，还以为对方接线的人是一个令人信服的技术人员。
  </p>
  <p class="graf graf--p">
   当另一端的人询问他的登录信息，包括密码和家庭住址时，他丝毫没有质疑 ……毕竟，是 Bob 在自己打电话给他们的。
  </p>
  <p class="graf graf--p">
   帮助热线上的人告诉 Bob，他的帐户被“黑客攻击”了。但是，当他们提出通过出售645美元的防火墙套件来解决这个问题时（只能通过 Google Play Bucks 购买）鲍勃挂断了电话，关闭所有东西，包括他的Wi-Fi。
  </p>
  <p class="graf graf--p">
   当他重新启动时，Bob 在他的硬盘上发现了勒索软件，导致他无法访问计算机上的任何内容。
  </p>
  <p class="graf graf--p">
   这一点不奇怪，他已经把登陆凭据告诉给骗子了。
  </p>
  <p class="graf graf--p">
   他只能把计算机交给一些专家，后者帮助他解除了勒索软件。
   <strong class="markup--strong markup--p-strong">
    整个折腾花费了210美元，还加上伤痕累累的自负。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️雅虎并不是唯一一个成为了该骗局主角的家伙。 Facebook 最近也不得不面对欺骗许多社交媒体用户的虚假热线电话。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.komando.com/happening-now/389221/top-story-facebook-customer-service-phone-number-its-a-scam" href="https://www.komando.com/happening-now/389221/top-story-facebook-customer-service-phone-number-its-a-scam" rel="noopener noreferrer" target="_blank">
    点击此处
   </a>
   阅读 Facebook 假热线故事。
  </p>
  <p class="graf graf--p">
   简而言之：
   <strong class="markup--strong markup--p-strong">
    网络犯罪分子变得如此复杂，甚至可以欺骗专业人士
   </strong>
   。
  </p>
  <p class="graf graf--p">
   通过 Bob 的遭遇，这里有几个教训值得更多人学习。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1.知道如何以正确的方式获得帮助
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️人们都认为世界上最强大的搜索引擎应该能够清除一切潜在的恶意链接，但很遗憾，这不是它的工作方式。
  </p>
  <p class="graf graf--p">
   黑客在玩弄搜索引擎系统方面非常出色，他们只是在等着有人找到那些假的号码和电话。
  </p>
  <p class="graf graf--p">
   在某种程度上来说，中了勒索软件的 Bob 还算是幸运的，⚠️
   <a class="markup--anchor markup--p-anchor" data-href="https://www.komando.com/happening-now/551374/womans-bank-account-drained-through-a-fake-customer-support-number" href="https://www.komando.com/happening-now/551374/womans-bank-account-drained-through-a-fake-customer-support-number" rel="noopener noreferrer" target="_blank">
    在这里看到
   </a>
   在线转账服务如何让一位女士失去了所有存款。
  </p>
  <p class="graf graf--p">
   事实是，对像雅虎这样的公司的技术支持来说通常不会有简单的800号码，否则他们每天都必须拨打数千甚至数百万的电话。相反，它们通常通过电子邮件或实时聊天进行回应。
  </p>
  <p class="graf graf--p">
   所以如果你找到了一个电话号码，那就要小心了。⚠️至少应该先反向搜索一下您通过Google或任何搜索引擎找到的任何电话号码，并报告查到的诈骗。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2. 检查、仔细检查
   </strong>
  </p>
  <p class="graf graf--p">
   Bob 的对手使用了一种常见的恐吓策略：他们坚称 Bob 的电脑已经被黑了，吓唬他应该在更多的伤害发生之前迅速采取行动。
  </p>
  <p class="graf graf--p">
   Bob 肯定会担心，为了不顾一切地解决这个问题他会答应很多事；直到骗子要求一种不寻常的付款形式时（Google Play Bucks）才引起 Bob 的怀疑。
  </p>
  <p class="graf graf--p">
   Bob 的计算机遭到黑客攻击，因为他很自然地向犯罪分子提供了自己的登录信息。许多在线服务都会定期要求提供类似信息以确认客户的身份，于是人们不容易起疑。
  </p>
  <p class="graf graf--p">
   Bob 吸取了教训：他应该先确保电话另一端的人是真实的。⚠️如果骗子是非常具有说服力的社交工程师，对大多数人来说这可能就是个挑战了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3. 当所谓的技术支持人员给你打电话时要小心
   </strong>
  </p>
  <p class="graf graf--p">
   与科技公司通常不提供热线的方式相同；他们也从不会打电话给你。除非您已安排预约或就特定问题寻求了帮助。
  </p>
  <p class="graf graf--p">
   许多人没有意识到这一点，他们已经陷入了骗局。一个典型的例子是据称来自微软的一连串电话，但实际上是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.komando.com/tips/350721/scam-alert-dont-be-tricked-by-these-fake-microsoft-tech-support-calls/all" href="https://www.komando.com/tips/350721/scam-alert-dont-be-tricked-by-these-fake-microsoft-tech-support-calls/all" rel="noopener noreferrer" target="_blank">
    网络钓鱼诈骗
   </a>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%8d%83%e4%b8%87%e4%b8%8d%e8%a6%81%e7%9b%b8%e4%bf%a1%e4%bb%bb%e4%bd%95%e7%83%ad%e6%83%85%e7%9a%84%e5%ae%a2%e6%9c%8d/" href="https://www.iyouport.org/%e5%8d%83%e4%b8%87%e4%b8%8d%e8%a6%81%e7%9b%b8%e4%bf%a1%e4%bb%bb%e4%bd%95%e7%83%ad%e6%83%85%e7%9a%84%e5%ae%a2%e6%9c%8d/" rel="noopener noreferrer" target="_blank">
     千万不要相信任何“热情的客服”
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4. 如果你被勒索软件绑架了，不要惊慌
   </strong>
  </p>
  <p class="graf graf--p">
   Bob 不知道黑客造成了多大的伤害，这种经历让他非常恐惧。
  </p>
  <p class="graf graf--p">
   ⚠️请记住：网络犯罪是一项全职工作，获利可能非常大，因此职业骇客具有高度侵略性和操纵性。全球很多寡头公司 —— Equifax、雅虎 等等 —— 都已经证明被诈骗利用过。
  </p>
  <p class="graf graf--p">
   （这件事其实挺有意思：连骗子都知道大多数人只相信寡头公司。而寡头公司本身就是全世界最大的骗子好吧）
  </p>
  <p class="graf graf--p">
   面对勒索软件，Bob 的做法是正确的，那就是：
   <strong class="markup--strong markup--p-strong">
    不要掏钱
   </strong>
   ；而是把被感染的计算机带到专家那里解决问题。
  </p>
  <p class="graf graf--p">
   恐慌只会让问题变得更糟。所以从这个经验中学习：保持冷静才能渡过难关。在以下看到面对骇客时应有的做法（不论是普通骇客还是政府的入侵）：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%80%a7%e6%95%b2%e8%af%88%ef%bc%9a%e5%a6%82%e6%9e%9c%e6%9c%89%e4%ba%ba%e5%a3%b0%e7%a7%b0%e6%8c%81%e6%9c%89%e4%bd%a0%e7%9a%84%e4%b8%8d%e9%9b%85%e8%a7%86%e9%a2%91%e5%90%91%e4%bd%a0%e5%8b%92%e7%b4%a2/" href="https://www.iyouport.org/%e6%80%a7%e6%95%b2%e8%af%88%ef%bc%9a%e5%a6%82%e6%9e%9c%e6%9c%89%e4%ba%ba%e5%a3%b0%e7%a7%b0%e6%8c%81%e6%9c%89%e4%bd%a0%e7%9a%84%e4%b8%8d%e9%9b%85%e8%a7%86%e9%a2%91%e5%90%91%e4%bd%a0%e5%8b%92%e7%b4%a2/" rel="noopener noreferrer" target="_blank">
     面对勒索软件时应该怎么办
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%88%91%e7%9a%84%e8%ae%be%e5%a4%87%e5%a5%bd%e5%83%8f%e8%a2%ab%e9%bb%91%e4%ba%86%e6%80%8e%e4%b9%88%e5%8a%9e%ef%bc%9f%e2%80%8a-%e2%80%8a-%e7%bb%99%e6%95%8f%e6%84%9f%e4%ba%ba%e5%a3%ab/" href="https://www.iyouport.org/%e6%88%91%e7%9a%84%e8%ae%be%e5%a4%87%e5%a5%bd%e5%83%8f%e8%a2%ab%e9%bb%91%e4%ba%86%e6%80%8e%e4%b9%88%e5%8a%9e%ef%bc%9f%e2%80%8a-%e2%80%8a-%e7%bb%99%e6%95%8f%e6%84%9f%e4%ba%ba%e5%a3%ab/" rel="noopener noreferrer" target="_blank">
     如果你认为自己被黑了，应该怎么办
    </a>

   </li>
  </ul>
  <p class="graf graf--p">
   再一次强调：
   <strong class="markup--strong markup--p-strong">
    任何人
   </strong>
   都可能遇到这种诈骗，并不取决于您是不是名人或者是不是有钱人。
  </p>
  <p class="graf graf--p">
   所以，安全知识必须普及。希望IYP的“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e6%8a%80%e6%9c%af%e9%98%b2%e8%ba%ab-%e8%87%aa%e6%88%91%e4%bf%9d%e6%8a%a4%e6%96%b9%e6%b3%95/" href="https://www.iyouport.org/category/%e6%8a%80%e6%9c%af%e9%98%b2%e8%ba%ab-%e8%87%aa%e6%88%91%e4%bf%9d%e6%8a%a4%e6%96%b9%e6%b3%95/" rel="noopener noreferrer" target="_blank">

   </a>
   ”栏目能对您有所帮助。⚪️
