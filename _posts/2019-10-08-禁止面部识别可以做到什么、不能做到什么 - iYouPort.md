  <p class="p1">
   <span class="s1">
    旧金山成为了美国第一个禁止使用面部识别技术的城市
    <b>
     ，虽然只限于
    </b>
    执法部门、当地机构以及该市的交通管理部门。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    我对头条新闻的直觉反应是，这对个人隐私来说很好，这是旧金山监管机构真正大胆的决定。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    该条例实际上涵盖的不仅仅是面部识别，因为它的陈述是这样的：
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    “‘
   </span>
   <span class="s1">
    监视技术
   </span>
   <span class="s2">
    ’
   </span>
   <span class="s1">
    指的是，使用主要用于收集、保留、处理、或共享音频、电子、视觉、位置、热、生物识别、嗅觉的任何软件、电子设备、使用电子设备的系统、或与任何个人或团体特别相关或能够与之相关的类似材料
   </span>
   <span class="s2">
    ”
   </span>
   <span class="s1">
    。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    不是全部禁止，而是说，从闭路电视摄像机、到社交媒体扫描仪、和车牌阅读器
   </span>
   <span class="s2">
    — —
   </span>
   <span class="s1">
    都必须提交
   </span>
   <span class="s2">
    “
   </span>
   <span class="s1">
    监控技术政策
   </span>
   <span class="s2">
    ”
   </span>
   <span class="s1">
    。说明将使用该技术收集哪些信息，保留多长时间，与谁共享，公众如何注册投诉以及如果指定授权和禁止使用。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    <b>
     但是，该禁令不包括
    </b>
    旧金山的机场和海港，因为这些是由联邦机构运营的。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    <b>
     也不意味着
    </b>
    没有任何个人、公司或其他组织安装包括面部识别的监控系统，并且，
    <b>
     被禁止使用该技术的机构可以与允许使用该技术的人员合作
    </b>
    。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    例如，执法部门可以使用在家庭用监视系统上捕获的视频来协助调查。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    于是，禁令的意义实际上有多大呢？
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    对面部识别的所有反对意见中最突出的一个是，该技术被证明是不准确的，特别是在识别有色人种和女性时。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    这可能有很多原因，但一个最可能的原因应该是，用于训练算法的数据集没能代表那些特定的人群。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    这个反对理由不够好；因为至少，与所有新兴技术一样，它们会随着时间的推移而不断改进，随着更多数据被捕获并开发出新技术，这些问题将得到解决，准确性将占上风。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    而对于那些专门瞄准非暴力反抗者个异议人士的当权者来说，该技术越准确、其危害性就越大。
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    关联《警
    <a href="https://www.iyouport.org/%e8%ad%a6%e5%af%9f%e5%b0%86%e6%b4%bb%e5%8a%a8%e5%ae%b6%e7%a7%b0%e4%b8%ba%e5%9b%bd%e5%86%85%e6%9e%81%e7%ab%af%e5%88%86%e5%ad%90%e7%9a%84%e4%ba%94%e4%b8%aa%e8%8d%92%e8%b0%ac%e7%90%86/" rel="noopener noreferrer" target="_blank">
     察将活动家称为
    </a>
   </span>
   <a href="https://www.iyouport.org/%e8%ad%a6%e5%af%9f%e5%b0%86%e6%b4%bb%e5%8a%a8%e5%ae%b6%e7%a7%b0%e4%b8%ba%e5%9b%bd%e5%86%85%e6%9e%81%e7%ab%af%e5%88%86%e5%ad%90%e7%9a%84%e4%ba%94%e4%b8%aa%e8%8d%92%e8%b0%ac%e7%90%86/" rel="noopener noreferrer" target="_blank">
    <span class="s1">
     “
    </span>
    <span class="s3">
     国内极端分子
    </span>
    <span class="s1">
     ”
    </span>
   </a>
   <span class="s3">
    <a href="https://www.iyouport.org/%e8%ad%a6%e5%af%9f%e5%b0%86%e6%b4%bb%e5%8a%a8%e5%ae%b6%e7%a7%b0%e4%b8%ba%e5%9b%bd%e5%86%85%e6%9e%81%e7%ab%af%e5%88%86%e5%ad%90%e7%9a%84%e4%ba%94%e4%b8%aa%e8%8d%92%e8%b0%ac%e7%90%86/" rel="noopener noreferrer" target="_blank">
     的五个荒谬理由
    </a>
    》
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    采取行动保护个人隐私显然是朝着正确方向迈出的一步。
   </span>
  </p>
  <p class="p4">
   <span class="s3">
    这条特别的法令引起了我的注意，因为
    <b>
     作为英国人，我目睹了监控文化的深刻；在英国，你不可能做到在任何街道上行走或驾驶而不被许多监控系统捕获
    </b>
    。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    我现在是旧金山湾区的居民，但是，当我反思这项禁令时，我发现自己正在考虑执法是否应该拥有主动保护公众所需的所有工具。如果观察名单或类似的列表上有杀人犯，那我可能希望执法部门知道他们在哪。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    面部识别有很多种用途。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    许多智能手机使用该技术作为身份验证解锁、社交媒体使用它来标记照片；和中国一样，一家名为
   </span>
   <span class="s2">
    Cubic Corporation
   </span>
   <span class="s1">
    的公司开发伦敦地铁的票务系统，现在正在探索是否可以使用面部识别来验证每个旅客。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    虽然基本可以肯定，以上这三种最常见的使用方法中收集到的面部数据
   </span>
   <span class="s2">
    ——
   </span>
   <span class="s1">
    您自己的脸，您都无法完整知道它们会被转售给谁、被用于什么。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    技术的确可以用于正义的目的。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    印度证实了使用面部识别执法的一个突出例子。当时新德里的警察部队在试运行后的短短四天内发现了近
   </span>
   <span class="s2">
    3000
   </span>
   <span class="s1">
    名失踪儿童。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    该技术被用于整个城市的
   </span>
   <span class="s2">
    45,000
   </span>
   <span class="s1">
    名儿童，并确定出其中有
   </span>
   <span class="s2">
    2,930
   </span>
   <span class="s1">
    名儿童被记录为失踪。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    减少交通拥堵、让失踪儿童与父母团聚，这些都是该技术的积极用途。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    但是，很明显，问题不在于
   </span>
   <span class="s2">
    *
   </span>
   <span class="s1">
    使用
   </span>
   <span class="s2">
    *
   </span>
   <span class="s1">
    监视技术，而在于
    <b>
     滥用
    </b>
    监视技术的可能性。
   </span>
  </p>
  <p class="p1">
   <span class="s5">
    ⚠️
   </span>
   <span class="s1">
    它不会仅仅用来寻找失踪儿童，同时还可以轻松找到任何当局感兴趣的人，尤其是政府反对派；它也不会专注于解决道路拥堵问题，因为它可以轻松追踪到任何策划和参与抗议行动的人。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    相信大多数人都知道自己是否应该采取一些保护措施。而且这一评估很重要。
   </span>
  </p>

  <p class="p3">
   <span class="s3">
    保护是可以做到的。我们曾经介绍过《
    <a href="https://www.iyouport.org/%E8%AE%A9%E9%9D%A2%E9%83%A8%E8%AF%86%E5%88%AB%E5%A4%B1%E6%95%88%E7%9A%84%E6%80%9D%E8%80%83%E6%96%B9%E5%BC%8F/" rel="noopener noreferrer" target="_blank">
     让面部识别失效的思考方式
    </a>
    》
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    了解这一思考方式，您就可以更好地选择保护性措施。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    比如，一副特殊的眼镜。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    无论你是一个隐蔽的间谍还是普通的隐私捍卫者，这种东西都可以帮您拦截数字窥探。
   </span>
  </p>
  <p class="p1">
   <span class="s1">
    这一特殊的眼镜不会让你在现实生活中隐形，但是，它们的目的是阻止面部识别软件。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    VG Technologies
   </span>
   <span class="s1">
    公司已经在阿姆斯特丹的创新实验室开发了该眼镜的原型版本。这种眼镜使用在眼睛和鼻子周围的红外灯来打破用于识别面部的算法。
   </span>
  </p>

    此版本的眼镜还具有反光涂层，因此如果使用闪光灯，它可以反射强光。
   </span>

  <p class="p1">
   <span class="s1">
    其实面部识别技术已经快速变得准确，

   </span>
  </p>
  <p class="p1">
   <span class="s1">
    而且，犯罪分子也热衷于这种技术。一个澳大利亚罪犯团伙是典型，他们参观了警察毕业典礼，拍照，然后使用面部识别软件创建了自己的卧底警察数据库。
