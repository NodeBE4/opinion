  <p class="graf graf--p">
   两家主要的美国养老基金拒绝就其持有股份的中国公司发表评论，该公司就是海康威视，他们的监控设备据报在穆斯林拘留营中被使用。
  </p>
  <p class="graf graf--p">
   纽约州教师退休系统和加利福尼亚州教师退休系统都持有臭名昭著的中国海康威视的股份。
  </p>
  <p class="graf graf--p">
   立法者呼吁投资者抛售他们在该公司的股份。
  </p>
  <p class="graf graf--p">
   据说在新疆地区有多达一百万穆斯林被拘禁。
  </p>
  <p class="graf graf--p">
   纽约州教师退休系统（NYSTRS）和加州州立教师退休系统（CalSTRS）是美国最大的两个公共养老基金。
  </p>
  <p class="graf graf--p">
   NYSTRS将杭州海康威视列入其国际股权控股，截至2018年12月31日，持有26,402股股票，市值99,059.89。
  </p>
  <p class="graf graf--p">
   截至2018年6月30日，CALSTRS 持有海康威视的股份 4,356,300 股，市值为24,414,000美元。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们致力于对社会和环境负责的商业行为，并以 CalSTR 的最佳利益获取商品和服务，”该基金的网站上写道，尽管它没有特别评论海康威视。
  </p>
  <p class="graf graf--p">
   参议院外交关系委员会的高级成员马可·卢比奥（Marco Rubio）呼吁进行改革。
  </p>
  <p class="graf graf--p">
   这位佛罗里达州共和党人在他的网站上分享给特朗普总统两位内阁成员的一封信，敦促政府对
   <strong class="markup--strong markup--p-strong">
    从新疆地区的拘留中“获利”的美国个人和实体
   </strong>
   实施制裁。他提到了海康威视。
  </p>
  <p class="graf graf--p">
   西北地区是维吾尔人的家乡，维吾尔族是讲突厥语的穆斯林少数民族。他们占新疆人口的45％左右，受到中国当局的严格监视。
  </p>
  <p class="graf graf--p">
   总部位于杭州的海康威视（Hikvision）被认为是由中国政府控制的，是世界上最大的视频监控技术供应商之一。
  </p>
  <p class="graf graf--p">
   据“伦敦时报”2016年报道，
   <strong class="markup--strong markup--p-strong">
    海康威视还帮助英国客户安装了超过一百万台闭路电视监控系统，这些监视技术已安装在英国各地，包括机场和政府机构
   </strong>
   。
  </p>
  <p class="graf graf--p">
   中国面临着越来越大的压力，人权组织要求其停止对维吾尔族、哈萨克族和其他穆斯林少数民族被无限期关押，不得指控拒绝提供DNA样本的人、坚持使用少数民族语言或与官员争吵的人。
  </p>
  <p class="graf graf--p">
   人权组织还表示，海康威视为这些设施提供监控系统。
   <strong class="markup--strong markup--p-strong">
    详细报告参见
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://docs.google.com/document/d/1tmU2VmsdHRo79NniIM85yWJrS-A_GIhku_0yXZOg8Fk/edit?usp=sharing" href="https://www.iyouport.org/category/%E4%BD%9C%E6%81%B6%E8%80%85%EF%BC%9A%E8%B0%81%E5%9C%A8%E5%B8%AE%E4%B8%AD%E5%9B%BD/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     IYP 类目的“谁在帮中国”部分
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   英国“金融时报”报道称，该公司表示：“我们非常重视所提出的有关使用海康威视设备的问题，我们正在评估一系列解决这些问题的方案。”
  </p>
  <p class="graf graf--p">
   去年该公司网站上的一份声明称：“海康威视非常重视网络安全，并遵守我们经营所在国家/地区的所有适用法律法规。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们还重申，我们的产品符合行业的全球网络安全标准。”
  </p>
  <p class="graf graf--p">
   新疆正式被指定为中国境内的自治区，如南部的西藏。
  </p>
  <p class="graf graf--p">
   到目前为止，几乎没有穆斯林占多数的国家加入公开谴责这些指控。
  </p>
  <p class="graf graf--p">
   分析人士称，许多人担心来自中国的政治和经济报复。
  </p>
  <p class="graf graf--p">
   中国驳回了侵犯人权的指控。
  </p>
