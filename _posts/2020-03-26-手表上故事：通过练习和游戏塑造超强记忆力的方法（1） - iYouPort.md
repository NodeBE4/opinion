  <p class="graf graf--p">
   互联网的所谓便利性和易搜索功能也许很好，但它毁掉了很多人的天然记忆能力，
   <strong class="markup--strong markup--p-strong">
    而记忆，是您完成绝大多数重要任务都必需的基本功
   </strong>
   。
  </p>
  <p class="graf graf--p">
   互联网和应用程序提供了强大的可搜索性，也许有些人会就觉得自己不需要记住任何东西，需要的时候上网搜就行了。不，绝不是这样。

    不论是对公民调查人员来说、还是基本的反侦查、反跟踪需求，或者学习新知识的能力，您都需要记忆力作为基础
   </strong>
   。
  </p>
  <p class="graf graf--p">
   想象这样场景。您发现一辆汽车正跟在你后面，你记得曾经在哪里见过这辆汽车吗？它的车牌、外观、细节特征？如果你记得，就能迅速判断它是否危险，并在第一时间采取应对策略。
  </p>
  <p class="graf graf--p">
   对于人类跟踪者来说也是如此，
   <strong class="markup--strong markup--p-strong">
    如果你能做到过目不忘，就能用眼识破很多伪装。
   </strong>
  </p>
  <p class="graf graf--p">
   对于信息的真伪验证来说，记忆力也是格外优秀的能力：你曾经在哪里见过这张照片/视频？当时的发布者对它的描述是什么？不需要任何搜索就可以识别造假 ——
   <strong class="markup--strong markup--p-strong">
    对于在那些难以搜索的地方出现的信息尤其有效，比如加密通讯群组。
   </strong>
  </p>
  <p class="graf graf--p">
   或者你曾经见过类似的表述模式？当你再次看到它时你会立即反应过来，发布者可能是僵尸网络的一部分，或者是某个袜子木偶账户被你识破了。您可以顺着这条线索继续挖掘其幕后的操纵者。

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     关于记忆能力
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   大多数人没能充分利用自己的记忆能力，甚至很少有人知道人的记忆力可以有多强。
  </p>
  <p class="graf graf--p">
   举几个例子。俄罗斯画家 Nikolai Ge 只需要一次实地访问就可以细致地画出夏宫某个客房中全套巴洛克风格的内饰。
  </p>
  <p class="graf graf--p">
   莫扎特只需要听上一遍，就可以把复杂的乐谱全部写下来。罗马教廷曾经一度将 Gregorio Allegri 所作的《求主垂怜》的乐谱奉为秘密，直到莫扎特听过一遍后复制了它，才得以向大众传播。那时候莫扎特只有14岁。
  </p>
  <p class="graf graf--p">
   丘吉尔对莎士比亚的所有作品都能熟记，他用这些素材来练习演讲。
  </p>
  <p class="graf graf--p">
   1960年，匈牙利象棋手 Janos Flash 在不看任何一盘棋的情况下，同时进行52场比赛。这些比赛持续了超过13个小时，而他记住了52个棋盘上的所有着数。
  </p>
  <p class="graf graf--p">
   当然，并非只有那些名人拥有超强的记忆力。在一次实验中，研究人员向普通参与者展示了一万张幻灯片，随后测试他们能记住多少内容。结果发现，人们对图像辨认的准确率为80%、对于色彩比较丰富的清晰的图像准确率高达100%。
  </p>
  <p class="graf graf--p">
   因此能够看出：
  </p>
  <p class="graf graf--p">
   1、
   <strong class="markup--strong markup--p-strong">
    人类记忆的主要问题不在于记忆信息本身，而是在需要的时候从大脑中重新调用和重现这些信息
   </strong>
   。每个人都能具备绝佳的记忆力潜能，要想开发它，需要一系列技巧。
  </p>
  <p class="graf graf--p">
   2、
   <strong class="markup--strong markup--p-strong">
    大脑非常擅长记忆图像
   </strong>
   ，因此大多数记忆信息的技巧都是以想象力为基础的，将抽象的语言和数字信息转化为图像。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     记忆的类型
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   现代心理学确定了三种类型的记忆：感觉记忆、短期记忆、和长期记忆。
  </p>
  <p class="graf graf--p">
   感觉记忆 ——
  </p>
  <p class="graf graf--p">
   也就是直接来自于感官获得的信息，比如看、听、闻、尝、摸。感觉记忆的持续时间不会很久，人类保存感觉记忆的时间不会超过半秒钟。
  </p>
  <p class="graf graf--p">
   但是感觉记忆非常重要，正因为它，我们才能将所处的环境联系起来。通过感觉记忆，人们可以将电影中一系列短暂的单个画面连接为故事。
  </p>
  <p class="graf graf--p">
   短期记忆 ——
  </p>
  <p class="graf graf--p">
   那些值得关注的信息将从感觉记忆转变为短期记忆，它可以储存几分钟到几小时。
  </p>
  <p class="graf graf--p">
   比如你默念一个电话号码或验证码，一边寻找笔和纸和它写下来，这个过程中形成的就是短期记忆。
  </p>
  <p class="graf graf--p">
   长期记忆 ——
  </p>
  <p class="graf graf--p">
   重要的信息将从短期记忆变为长期记忆，它可以储存很多年。
  </p>
  <p class="graf graf--p">
   一般来说，信息的长期储存过程是在无意识的情况下发生的，这就是为什么我们经常会忘记重要的事，却会记住很久以前就应该忘掉的一些细节。
  </p>
  <p class="graf graf--p">
   当然，
   <strong class="markup--strong markup--p-strong">
    有些方法可以让人有意识地储存长期信息
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这里的训练的目标是，同时增强短期记忆和长期记忆，并学会有意识地将信息从短期记忆变为长期记忆
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果你擅长留心看到和听到的所有细节，就可以不断重新诠释这些信息，并将它们与之前掌握的信息联系起来
   </strong>
   。这对于公民调查人员来说非常重要 —— 就算您弄了一个可搜索的数据库，您也应该能记得数字组织的脉络，才有希望得出答案。
  </p>
  <p class="graf graf--p">
   💡换句话说，想要获得杰出的记忆力，您需要在日常就有意识地培养关注身边的每个小细节，并充分运用想象力将它们与已知的信息联系起来。
  </p>
  <p class="graf graf--p">
   这是需要您有意识地随手练习的。
  </p>
  <p class="graf graf--p">
   <span style="color: #ff0000;">
    ⚠️快速回答
   </span>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    您所居住的那条街上有几个面包房？它们的招牌各自是什么颜色的？
   </li>
   <li class="graf graf--li">
    您上次通过互联网联络的人的头像是什么？
   </li>
  </ul>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     专注力
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   专注力是一种强大的能力，它能有选择地感知信息，只查看和听取自己所需要的内容，忽略干扰项。
  </p>
  <p class="graf graf--p">
   你知道，互联网，尤其是社交媒体，最严重的侵害就是针对专注力的 —— 一片嘈杂中所有人被算法引诱着，迷失在一片肤浅粗糙的信息海洋里。
  </p>
  <p class="graf graf--p">
   其实社交媒体不是问题。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    社交媒体上最多的东西就是噪音。但是噪音并不会妨碍一个沉浸于阅读的人。
   </strong>
   沉浸的人可以用心感知文字，忽略外部杂音；集中注意力可以让你专注于记忆那些微妙的差别和细节，避免让大脑因为不分轻重地关注所有事物而过度负担。
  </p>
  <p class="graf graf--p">
   受过训练的注意力和注意力欠缺是两码事，因为训练可以使注意力转向指定的事物。
  </p>
  <p class="graf graf--p">
   你可以快速集中精力，在必要时长时间关注某件事，并在切换活动时轻松地转移注意力。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-10509 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-7.jpg?resize=1100%2C733&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-7.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-7.jpg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-7.jpg?resize=768%2C512&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-7.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-7.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-7.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="733" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-7.jpg?resize=1100%2C733&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    历史上一场著名的冲突
   </strong>
  </h3>
  <p class="graf graf--p">
   1954年12月10日，在布宜诺斯艾利斯举行的一场选举前会议中，选民们与阿根廷正义党候选人 Garcia Pughese 之间发生了一系列不寻找的事件。
  </p>
  <p class="graf graf--p">
   像往常一样，这次会议以候选人的演讲作为开场，结果，最后却以一场大规模的斗殴作为收场。
  </p>
  <p class="graf graf--p">
   Pughese 呼吁与社会党“对抗”。而选民们却在字面上理解了他的话。在会议举办地点也就是一家电影院里，大约300名出席会议的选民高喊口号，前往社会党选举总部。
  </p>
  <p class="graf graf--p">
   这群人手持园艺工具、石头和棍棒，打破窗户、砸烂家具、殴打他人。导致多位党代表受伤住院。其中包括社会党候选人 Gabriel Acritiso。
  </p>
  <p class="graf graf--p">
   整个斗殴过程中警方并没有太多干预，只是将其中几个人进行了短期拘留。
  </p>
  <p class="graf graf--p">
   值得注意的是，被拘留者否认他们参与了斗殴，并同时声称自己纯粹处于好奇而来到选举大会现场。
  </p>
  <p class="graf graf--p">
   就像一场集体精神错乱，忽然开始，又嘎然而止。观察人士认为正义党被选中的概率很高。
  </p>
  <p class="graf graf--p">
   而且有传闻称，正义党的竞选活动得到了德国顾问的支持，而这些顾问曾效忠于法西斯政权，并在第三帝国灭亡时逃离欧洲。但即便如此，选民对正义党的支持度丝毫没有减弱。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    练习 ——
   </span>
  </h3>
  <p class="graf graf--p">
   长时间关注一件事并不像你想象中那么容易。
  </p>
  <p class="graf graf--p">
   您可以试着仔细观察一个物体，比如一块手表。观察它的每一个细节。仔细检查表盘上的每一个分区，以及表面上的每一处划痕。
  </p>
  <p class="graf graf--p">
   你观察到一切了吗？
  </p>
  <p class="graf graf--p">
   继续尝试有没有新发现？
  </p>
  <p class="graf graf--p">
   几分钟后你就开始走神了。无法再集中在那块手表上。
  </p>
  <p class="graf graf--p">
   忽然间你注意到，自己并没有把这块手表放在心上，关于这块手表的联想已经把你的思绪带到了别处。
  </p>
  <p class="graf graf--p">
   你看到了表盘上的“11”，你忽然想到11点有一个重要的会议；接着你又顺势想到了参加这次会议的同事，又想到了那我同事曾经介绍过一本书给你……
  </p>
  <p class="graf graf--p">
   你的脑子里已经没有那块手表了。你会立刻回去再走一遍手表的旅程吗？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    记住自己的注意力从手表转移到后面一大堆联想的整个过程
   </strong>
   。回到手表引发的最后的一个联想：那本书，继续向前推进：那位同事，再继续：11点的会议。现在你再次回到了手表上。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    请保证整个路线的往返是清晰的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   💡这是一个很好的练习。通过重走联想之路，这项练习可以培养你将自身注意力转向真正需要的地方。
  </p>
  <p class="graf graf--p">
