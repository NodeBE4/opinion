  <p class="graf graf--p">
   现在，这一少数派报告式的监控系统有了更为强大的现实辅助。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    中国推出了一种新的监视相机，其分辨率是人眼的五倍，能够实时监视数千人、并识别人脸，由此引发了人们对大规模监视的新担忧
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.abc.net.au/news/2019-09-26/chinas-new-500-megapixel-super-camera/11539176?pfmredir=sm" href="https://www.abc.net.au/news/2019-09-26/chinas-new-500-megapixel-super-camera/11539176?pfmredir=sm" rel="noopener noreferrer" target="_blank">
    澳大利亚ABC新闻报道称
   </a>
   ，在中国国际工业博览会上揭晓了这种被称为 “超级摄像机’’ 的新型500万像素面部识别超级“AI”监控相机。
  </p>
  <p class="graf graf--p">
   该监控相机系统配备了最先进的面部识别工具，是由复旦大学和中国科学院长春光学精密机械与物理研究所设计的。
  </p>
  <p class="graf graf--p">
   ⚠️设计人员声称，该监视系统可以实时检测和识别成千上万的人脸或其他物体，并可以在拥挤的体育场馆等环境中即时定位特定目标。
  </p>
  <p class="graf graf--p">
   ⚠️当然，在维稳抗议活动中也同样牛逼。
  </p>
  <p class="graf graf--p">
   设计者建议，警察可以在上海市中心安置该摄像头系统并监视人群的活动，同时结合医疗记录和犯罪记录对捕捉到的人脸图像进行交叉检查。
  </p>
  <p class="graf graf--p">
   北京解放军国防大学教授李大光对《环球时报》表示，该监视系统 “很容易应用于国防、军事和公共安全[维稳]。”
  </p>
  <p class="graf graf--p">
   ⚠️像这样的监控技术在已经实行全民社会信用评分体系的中共当局手中，对于隐私权和自由而言绝不是什么好兆头。
  </p>
  <p class="graf graf--p">
   分析师萨曼莎•霍夫曼（Samantha Hoffman）表示：“党国拥有庞大的人物图像数据库，并具有将人物图像与目标真实身份联系起来的能力，因此，即使现在不然，像这样的技术在不久的将来也是很有可能的……
  </p>
  <p class="graf graf--p">
   社会信用评分制度通过限制和禁止人们购买旅行票证，以惩罚不服从行为，⚠️目前正在使用由2亿多个监控摄像头和其他跟踪工具组成的庞大网络来实施。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9528 jetpack-lazy-image" data-lazy-sizes="(max-width: 912px) 100vw, 912px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.png?resize=912%2C1228&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.png?w=912&amp;ssl=1 912w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.png?resize=223%2C300&amp;ssl=1 223w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.png?resize=768%2C1034&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.png?resize=760%2C1024&amp;ssl=1 760w" data-recalc-dims="1" height="1228" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-3.png?resize=912%2C1228&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="912"/>

  <p class="graf graf--p">
   但是，5亿像素的“超级相机”将把监视功能提升到另一个全新的层次。
  </p>
  <p class="graf graf--p">
   科学家指出，当前的监控摄像机也可以一次监视成千上万的人，但是像素很低，这意味着“您根本无法清楚地看到目标人”。
  </p>
  <p class="graf graf--p">
   ASPI的 Hoffman 说，视频中的数据将“被放入一个数据池中，再结合AI处理，可以生成用于社会控制的强大工具，包括与社会信用评分相关的工具”。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="421" src="//www.youtube.com/embed/CWS3bswxklc" width="750">
   </iframe>
  </p>
  <p class="graf graf--p">
   面部识别技术是世界各地公民自由团体和立法者之间日益激烈的辩论的主题，更不用说在中国这样的以对公民进行根深蒂固的监视而闻名的国家中引发的隐私问题。
  </p>
  <p class="graf graf--p">
   中国是世界上监控最严密的国家之一，估计有2亿个CCTV监控摄像机每时每刻监视着人们，这个数字到2020年预计将增长213％，达到6.26亿。在这里看到报告《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e5%92%8c%e4%b8%ad%e5%9b%bd%e5%9c%a8%e7%9b%91%e8%a7%86%e5%85%ac%e6%b0%91%e6%96%b9%e9%9d%a2%e5%bc%95%e9%a2%86%e5%85%a8%e7%90%83/" href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e5%92%8c%e4%b8%ad%e5%9b%bd%e5%9c%a8%e7%9b%91%e8%a7%86%e5%85%ac%e6%b0%91%e6%96%b9%e9%9d%a2%e5%bc%95%e9%a2%86%e5%85%a8%e7%90%83/" rel="noopener noreferrer" target="_blank">
    美国和中国在监视公民方面引领全球
   </a>
   》。
  </p>
  <p class="graf graf--p">
   今年10月，北京政府宣布了扩大有争议的做法的计划，要建立一个全国超过3000万家公司的公司排名系统。
  </p>
  <p class="graf graf--p">
   该算法的治理系统将以额外的特权奖励良好的行为，并限制那些分数较低的人出差或获得政府职位，该算法有望在明年正式推出。
  </p>
  <p class="graf graf--p">
   配备了面部识别、人体扫描和地理跟踪技术的CCTV监视摄像机网络使实时监控成为可能。
  </p>
  <p class="graf graf--p">
   不只是中国。
  </p>
  <p class="graf graf--p">
   在其他地方，亚马逊和 Palantir 与美国政府机构的交易
   <a class="markup--anchor markup--p-anchor" data-href="https://thenextweb.com/security/2019/09/12/ice-is-paying-an-israeli-security-company-30m-to-break-into-phones/" href="https://thenextweb.com/security/2019/09/12/ice-is-paying-an-israeli-security-company-30m-to-break-into-phones/" rel="noopener noreferrer" target="_blank">
    受到了审查
   </a>
   ；而英国高等法院最近裁定
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wired.co.uk/article/police-facial-recognition-south-wales-court-decision" href="https://www.wired.co.uk/article/police-facial-recognition-south-wales-court-decision" rel="noopener noreferrer" target="_blank">
    支持警察使用自动面部识别技术
   </a>
   搜索人群。
  </p>
  <p class="graf graf--p">
   同时，
   <a class="markup--anchor markup--p-anchor" data-href="http://ncrb.gov.in/TENDERS/AFRS/RFP_NAFRS.pdf" href="http://ncrb.gov.in/TENDERS/AFRS/RFP_NAFRS.pdf" rel="noopener noreferrer" target="_blank">
    印度已邀请投标建立一个大型监视系统
   </a>
   ，以集中在全国范围内通过监视摄像机捕获的面部识别数据，并将其与国家犯罪数据库进行匹配。⚪️
  </p>
