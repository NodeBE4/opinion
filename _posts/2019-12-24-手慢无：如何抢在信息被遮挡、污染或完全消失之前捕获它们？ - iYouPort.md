  <p class="graf graf--p">
   速度是任何类型调查中的关键因素。俗称手慢无。
  </p>
  <p class="graf graf--p">
   信息被搁置的时间越长，被遮挡、被污染、或完全消失的可能性就越大。
  </p>
  <p class="graf graf--p">
   对于任何类型的信息都是如此，无论是目击者证词、服务器日志、推文、指纹、还是调查员可能认为有用的任何其他信息。
  </p>
  <p class="graf graf--p">
   目击者会忘记、服务器日志将被覆盖、推文将被删除、指纹也将被擦除。
  </p>
  <p class="graf graf--p">
   ⚠️无论您在调查什么，知道如何迅速采取行动以在丢失信息之前做好保存和收集，是一项重要技能。
  </p>
  <p class="graf graf--p">
   <a href="https://twitter.com/jakecreps" rel="noopener noreferrer" target="_blank">
    Jake Creps
   </a>
   进一步探讨这个想法，并展示一些可以在出现问题时恢复丢失的信息的技术。值得学习。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     信息损耗
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   为了解释这是什么意思，请见下面的图：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9476 jetpack-lazy-image" data-lazy-sizes="(max-width: 640px) 100vw, 640px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-1.png?resize=640%2C400&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-1.png?w=640&amp;ssl=1 640w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-1.png?resize=300%2C188&amp;ssl=1 300w" data-recalc-dims="1" height="400" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-1.png?resize=640%2C400&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="640"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9476" data-recalc-dims="1" height="400" sizes="(max-width: 640px) 100vw, 640px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-1.png?resize=640%2C400&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-1.png?w=640&amp;ssl=1 640w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-1.png?resize=300%2C188&amp;ssl=1 300w" width="640"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   红色框代表理想的世界，在其中您可以获取所有可能的信息。
  </p>
  <p class="graf graf--p">
   也就是说，假设每个 Facebook 帖子都是公开的，每个图片都是公开共享的，没有人删除其推文，而 Google 已为每个细节编制了索引。
  </p>
  <p class="graf graf--p">
   很遗憾这样的世界根本不存在，但是为了收集开源情报资料的目的正是
   <strong class="markup--strong markup--p-strong">
    要尽可能地靠近这个理想世界
   </strong>
   。
  </p>
  <p class="graf graf--p">
   橙色框则代表您可以作为OSINT调查员实际使用的材料。
  </p>
  <p class="graf graf--p">
   这个世界并不完美，您会不断丢失信息：时间流逝、隐私设置被打开、视频被删除、网站受到审查、数据流庞大您不知所措并迷失在搜索结果的海洋中、或者长得不够帅目击者决定不再和你说话 …… 等等，总之导致您丢失很多信息的方法有很多种。
  </p>
  <p class="graf graf--p">
   最底下的黄色框是您最后需要处理的实际信息量。
  </p>
  <p class="graf graf--p">
   这是您实际用来做出报道、撰写记录片中的故事、完成报告、或起诉案件的可用材料。
  </p>
  <p class="graf graf--p">
   从橙色框过渡到黄色框再次丢失了更多的材料，其原因有一大堆：过时且不相关的、丢失了USB记忆棒、来源不再想要提供帮助、您所依赖的Internet工具不再起作用、目标已修补您正在利用的漏洞……等等。
  </p>
  <p class="graf graf--p">
   在红色框的理想世界和黄色框的最终结果之间，
   <strong class="markup--strong markup--p-strong">
    可用的OSINT材料的这种减少被称为信息损耗
   </strong>
   。
  </p>
  <p class="graf graf--p">
   💡于是，您越早入手能收集和保存到的数据就越多，最终就将获得更好更有效的结果。
  </p>
  <p class="graf graf--p">
   如果一项开源情报调查没有考虑如何很好地收集和保存证据，它看起来就会像下面这样：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9477 jetpack-lazy-image" data-lazy-sizes="(max-width: 640px) 100vw, 640px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-2.png?resize=640%2C400&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-2.png?w=640&amp;ssl=1 640w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-2.png?resize=300%2C188&amp;ssl=1 300w" data-recalc-dims="1" height="400" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-2.png?resize=640%2C400&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="640"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9477" data-recalc-dims="1" height="400" sizes="(max-width: 640px) 100vw, 640px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-2.png?resize=640%2C400&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-2.png?w=640&amp;ssl=1 640w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-2.png?resize=300%2C188&amp;ssl=1 300w" width="640"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   由于材料的最初收集和保存（橙色框）较差，因此，由于损耗而丢失了更多信息，而最终的调查产品（黄色框）则会更弱。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     示例：具有挑战性的侦探游戏
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/fiete_stegers" href="https://twitter.com/fiete_stegers" rel="noopener noreferrer" target="_blank">
    Fiete Stegers
   </a>
   发布了一个非常具有挑战性的侦探游戏。本文将以这个例子来说明，为什么以及如何迅速收集信息以防止损耗。
  </p>
  <p class="graf graf--p">
   Fiete 的题目是这样的：从他参加的活动现场实时发布了下面这张照片。你知道这张照片是在哪里拍摄的吗？
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9478 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-1.jpeg?resize=1024%2C768&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-1.jpeg?w=1024&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-1.jpeg?resize=300%2C225&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-1.jpeg?resize=768%2C576&amp;ssl=1 768w" data-recalc-dims="1" height="768" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-1.jpeg?resize=1024%2C768&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9478" data-recalc-dims="1" height="768" sizes="(max-width: 1024px) 100vw, 1024px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-1.jpeg?resize=1024%2C768&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-1.jpeg?w=1024&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-1.jpeg?resize=300%2C225&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/3-1.jpeg?resize=768%2C576&amp;ssl=1 768w" width="1024"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   结合使用 Twitter 的实时更新、Twitter 的实时地理位置数据、以及
   <a class="markup--anchor markup--p-anchor" data-href="https://onemilliontweetmap.com/" href="https://onemilliontweetmap.com/" rel="noopener noreferrer" target="_blank">
    OneMillionTweetMap
   </a>
   、
   <a class="markup--anchor markup--p-anchor" data-href="https://map.snapchat.com/" href="https://map.snapchat.com/" rel="noopener noreferrer" target="_blank">
    Snapchat Map
   </a>
   和 Facebook 的主题标签，可以非常快速地找到这个位置。
  </p>
  <p class="graf graf--p">
   请注意，这个方法在很大程度上依赖于来自各种社交媒体应用程序的短期实时数据；通过快速收集并保存数据（在这种情况下仅使用屏幕截图就可以），信息的损耗可以争取到最小化，并且可以非常快速地完成地理定位。
  </p>
  <p class="graf graf--p">
   但是，如果我是在四个月后才开始这项调查的，结果会是什么样子呢？
  </p>
  <p class="graf graf--p">
   这种情况下尝试从 Twitter 捕获实时数据已经毫无意义了，Snapchat Map 镜头也早已消失，而 Facebook 搜索机制的变化将意味着，即使不是不可能，也很难找到这场游行的准确路线。
  </p>
  <p class="graf graf--p">
   不仅是您最初所依赖的数据已经不再可用，而且，还有其他因素会导致更大的信息损失：这是一次气候维权大游行，但是自从拍摄那张照片以来，这里还有更多的同类游行出现。你可能分不清哪张照片来自哪个游行。
  </p>
  <p class="graf graf--p">
   这意味着将有数百万个搜索结果遮盖你一直在寻找的内容，本该依赖的某些 Twitter 帐户已被删除，依此类推。
  </p>
  <p class="graf graf--p">
   在Fiete发布挑战后的最初几个小时中，可用的信息库足以找到他，但是现在，从头开始将使任务更加艰巨，即使不是不可能的话。
  </p>
  <p class="graf graf--p">
   快速采取行动，尽早收集和保留开源情报信息，对最终结果将产生重大影响。接下来，本文将研究
   <strong class="markup--strong markup--p-strong">
    如何以及为何要快速保存开源情报资料
   </strong>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     保留和收集开源情报数据
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9479 jetpack-lazy-image" data-lazy-sizes="(max-width: 580px) 100vw, 580px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4.jpg?resize=580%2C324&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4.jpg?w=580&amp;ssl=1 580w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4.jpg?resize=300%2C168&amp;ssl=1 300w" data-recalc-dims="1" height="324" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4.jpg?resize=580%2C324&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="580"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9479" data-recalc-dims="1" height="324" sizes="(max-width: 580px) 100vw, 580px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4.jpg?resize=580%2C324&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4.jpg?w=580&amp;ssl=1 580w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4.jpg?resize=300%2C168&amp;ssl=1 300w" width="580"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   防止证据丢失的需求并非开源情报调查所独有。
  </p>
  <p class="graf graf--p">
   上面那张图显示了尽快保存和收集信息的相同道理。
  </p>
  <p class="graf graf--p">
   正如你所知道的，诸如血迹、指纹、头发样本、和脚印之类的重要证据对于解决严重犯罪来说至关重要。这些法医线索会很快退化或丢失，因此好的调查员会保留它们以防止丢失。
  </p>
  <p class="graf graf--p">
   这就是为什么罪案发生后，现场会迅速被警方围起来，避免不想干的人闯入破坏了上述证据。
  </p>
  <p class="graf graf--p">
   所有最敏感的区域都覆盖有帐篷。这不仅是为了让围观者远离。帐篷还可以保护非常敏感的证据碎片免受雨水、风和其他可能导致损失的因素的影响。如果一开始丢失或污染了这些证据，那么整个案件可能永远无法解决。
  </p>
  <p class="graf graf--p">
   在开源调查中，使用的原理也一样。
  </p>
  <p class="graf graf--p">
   如果您没有收集和保存正在发生或才刚发生的事件信息，证据就会丢失。
  </p>
  <p class="graf graf--p">
   ⚠️在瞬息万变的新闻世界中，珍贵的材料不仅会快速被新快讯所掩埋，甚至，可能还有其他恶意行为者在机器人的协助下推动
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e6%88%98%e5%92%8c%e4%bf%a1%e6%81%af%e6%88%98/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e6%88%98%e5%92%8c%e4%bf%a1%e6%81%af%e6%88%98/" rel="noopener noreferrer" target="_blank">
    信息战
   </a>
   ，积极地试图破坏证据、并利用阴谋论淹没原始资料。
  </p>
  <p class="graf graf--p">
   Twitter 就是一个很好的例子。如果无法快速识别和捕获主要来源，则很容易丢失它们，从而导致调查受挫。
  </p>
  <p class="graf graf--p">
   接下来本文将介绍一些实用的工具，可用于可靠的、快速地从Web捕获信息，以及其他一些工具，可帮助您收回一开始可能会丢失的信息。

   <span style="color: #ff6600;">

     保存工具
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   您可以使用多种工具来快速有效地捕获信息。
  </p>
  <p class="graf graf--p">
   最佳工具的使用确实取决于您要捕获的内容是什么，但是这里是一些不同工具的概述。

    Screenshot
   </strong>
   — 有很多不同的屏幕截取工具。
  </p>
  <p class="graf graf--p">
   Fireshot 用作浏览器插件，Windows 内置了 Snipping Tool，Mac OS 内置了 Screenshot 工具，而我最喜欢的 Linux 工具是 Shutter 和 Flameshot。
  </p>
  <p class="graf graf--p">
   Firefox 和 Vivaldi 现在都具有内置的截图工具。

  <p class="graf graf--p">

   这样一来，您就可以滤除大量无关信息的干扰，而只关注您感兴趣的时间段。只需要记住格式：DD / MM / YYYY。
  </p>
  <p class="graf graf--p">
   举例来说，假设我只是想看看 Jeffrey Epstein 从2009年开始的信息结果，我会这样做：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9493 jetpack-lazy-image" data-lazy-sizes="(max-width: 408px) 100vw, 408px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/18.png?resize=408%2C242&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/18.png?w=408&amp;ssl=1 408w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/18.png?resize=300%2C178&amp;ssl=1 300w" data-recalc-dims="1" height="242" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/18.png?resize=408%2C242&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="408"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9493" data-recalc-dims="1" height="242" sizes="(max-width: 408px) 100vw, 408px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/18.png?resize=408%2C242&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/18.png?w=408&amp;ssl=1 408w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/18.png?resize=300%2C178&amp;ssl=1 300w" width="408"/>

  </p>
  <p class="graf graf--p">
   如果您需要非常具体的内容，甚至可以设置更小的时间参数，比如减少到一天。
  </p>
  <p class="graf graf--p">
   请注意，这仍然会带来误报，因为相对较新的信息会出现在非常旧的网站上。与往常一样，您需要亲手检查所有内容。
  </p>
