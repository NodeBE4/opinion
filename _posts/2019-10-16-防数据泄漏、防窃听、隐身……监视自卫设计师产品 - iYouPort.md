  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    挫败阅读思维的摄像头
   </strong>
  </p>
  <p class="graf graf--p">
   意大利设计中心 Fabrica 的研究人员创造了一些配件，可以通过电击和闪光灯转移思想来欺骗
   <strong class="markup--strong markup--p-strong">
    神经影像监视设备
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5750 jetpack-lazy-image" data-lazy-sizes="(max-width: 468px) 100vw, 468px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-1.jpg?resize=468%2C468&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-1.jpg?w=468&amp;ssl=1 468w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-1.jpg?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-1.jpg?resize=300%2C300&amp;ssl=1 300w" data-recalc-dims="1" height="468" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-1.jpg?resize=468%2C468&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="468"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5750" data-recalc-dims="1" height="468" sizes="(max-width: 468px) 100vw, 468px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-1.jpg?resize=468%2C468&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-1.jpg?w=468&amp;ssl=1 468w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-1.jpg?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-1.jpg?resize=300%2C300&amp;ssl=1 300w" width="468"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   来自 Fabrica 的 Lisa Kori Chung 和 Caitlin Morris 设计了反NIS（神经影像监视）的部件，以检测与 CCTV 摄像机相关的监视技术何时试图读取佩戴者的脑电波。
  </p>
  <p class="graf graf--p">
   然后，它会将佩戴者的想法集中在一些无关紧要的事情上，以帮助维护隐私。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5751 jetpack-lazy-image" data-lazy-sizes="(max-width: 468px) 100vw, 468px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-2.jpg?resize=468%2C702&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-2.jpg?w=468&amp;ssl=1 468w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-2.jpg?resize=200%2C300&amp;ssl=1 200w" data-recalc-dims="1" height="702" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-2.jpg?resize=468%2C702&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="468"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5751" data-recalc-dims="1" height="702" sizes="(max-width: 468px) 100vw, 468px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-2.jpg?resize=468%2C702&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-2.jpg?w=468&amp;ssl=1 468w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-2.jpg?resize=200%2C300&amp;ssl=1 200w" width="468"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    神经影像技术目前正在迅速研究和开发中，目的是阅读和记录公众的想法，让监视者可以在行动之前扼杀它
   </strong>
   。
  </p>
  <p class="graf graf--p">
   想想看，如果你脑子里正在不由自主地构思参加抗议游行的场景，也许你还没到抗议现场，就已经被捕了。
  </p>
  <p class="graf graf--p">
   这就是为什么有了这样的设计。
  </p>
  <p class="graf graf--p">
   这个装置用于检测佩戴者何时被扫描，并提供分心，以改变佩戴者的思维模式。
  </p>
  <p class="graf graf--p">
   这是一种瞬间认知转移的方法。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “当检测到扫描时，配件会引起感官反应，牵引佩戴者的注意力，改变他们当前的大脑活动模式，并通过伪装提供隐私时刻。”
  </p>
  <p class="graf graf--p">
   这种帽子式的设备将声音脉冲通过颅骨传递到耳朵，颈圈式的会发出轻微的电击，面罩式的则发出闪光干扰佩戴者的眼睛。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5752 jetpack-lazy-image" data-lazy-sizes="(max-width: 468px) 100vw, 468px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-3.jpg?resize=468%2C614&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-3.jpg?w=468&amp;ssl=1 468w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-3.jpg?resize=229%2C300&amp;ssl=1 229w" data-recalc-dims="1" height="614" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-3.jpg?resize=468%2C614&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="468"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5752" data-recalc-dims="1" height="614" sizes="(max-width: 468px) 100vw, 468px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-3.jpg?resize=468%2C614&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-3.jpg?w=468&amp;ssl=1 468w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/配件-3.jpg?resize=229%2C300&amp;ssl=1 229w" width="468"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这意味着在扫描的那一刻，佩戴者的想法更可能被解读为“这种光太亮”或“这是一种奇怪的声音”，而不是他们的真实思想
   </strong>
   。
  </p>
  <p class="graf graf--p">
   该项目也旨在提高人们对目前与CCTV结合使用的其他监测技术的认识，如面部识别、运动检测和语音分析。
  </p>
  <p class="graf graf--p">
   Anti-NIS Accessories 是一系列拟议的设计产品，旨在作为一种服装形式，保持思想和行动的隐私。
  </p>
  <p class="graf graf--p">
   如今，闭路视频监控已经司空见惯。随着它无处不在的发展，正在开发新技术来分析所产生的大量信息。
  </p>
  <p class="graf graf--p">
   生物识别技术如FRT（面部识别技术）、步态分析和语音分析通常在事件发生后用于尝试确定所涉及的各方的身份。
  </p>
  <p class="graf graf--p">
   然而，现在各种公司正在研究算法以检测“在可疑地点”行动的人（可能基于诸如跑步、游荡和携带包裹之类的活动）。
  </p>
  <p class="graf graf--p">
   也就是说，监视技术开发正在着眼于在“事前”进行预测。
  </p>
  <p class="graf graf--p">
   于是，如何扰乱基于外部行为的意图的预测性分析算法，应该是当前防卫的重点考虑。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5758 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/自卫-2.jpg?resize=1100%2C783&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/自卫-2.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/自卫-2.jpg?resize=300%2C214&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/自卫-2.jpg?resize=768%2C547&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/自卫-2.jpg?resize=1024%2C729&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/自卫-2.jpg?resize=1100%2C783&amp;ssl=1 1100w" data-recalc-dims="1" height="783" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/自卫-2.jpg?resize=1100%2C783&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    数字自卫设备 CounterBug
   </strong>
  </p>
  <p class="graf graf--p">
   为了阻止
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%E5%85%B3%E4%BA%8E%E6%99%BA%E8%83%BD%E9%97%B4%E8%B0%8D%E5%AE%B6%E5%B1%85-%E5%9F%8E%E5%B8%82/" href="https://www.iyouport.org/category/%E5%85%B3%E4%BA%8E%E6%99%BA%E8%83%BD%E9%97%B4%E8%B0%8D%E5%AE%B6%E5%B1%85-%E5%9F%8E%E5%B8%82/" rel="noopener noreferrer" target="_blank">
    智能家居的侵害
   </a>
   ，格拉斯哥艺术学院的毕业生 Erlend Prendergast 设计了一种模块化机器人设备，可与
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%E5%85%B3%E4%BA%8E%E8%B0%B7%E6%AD%8C/" href="https://www.iyouport.org/category/%E5%85%B3%E4%BA%8E%E8%B0%B7%E6%AD%8C/" rel="noopener noreferrer" target="_blank">
    亚马逊
   </a>
   的虚拟助手 Alexa 进行对话，以保护用户免受数字监控。
  </p>
  <p class="graf graf--p">
   Prendergast 在苏格兰学校产品设计课程的最后一年学习期间开发了 CounterBug 项目。
  </p>

  <p class="graf graf--p">
   这个自发项目探讨了一些围绕
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%a5%e8%89%b2%e5%88%97%e5%88%9b%e4%b8%9a%e5%85%ac%e5%8f%b8%e7%ad%b9%e9%9b%86%e4%ba%861250%e4%b8%87%e7%be%8e%e5%85%83%ef%bc%8c%e8%ae%a9%e6%94%bf%e5%ba%9c%e5%8f%af%e4%bb%a5%e7%a0%b4%e8%a7%a3/" href="https://www.iyouport.org/%e4%bb%a5%e8%89%b2%e5%88%97%e5%88%9b%e4%b8%9a%e5%85%ac%e5%8f%b8%e7%ad%b9%e9%9b%86%e4%ba%861250%e4%b8%87%e7%be%8e%e5%85%83%ef%bc%8c%e8%ae%a9%e6%94%bf%e5%ba%9c%e5%8f%af%e4%bb%a5%e7%a0%b4%e8%a7%a3/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     国内间谍和数字监控

   存在的普遍恐惧。
  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   Prendergast 在格拉斯哥艺术学院的学位展上对 Dezeen 说，他的提议是“数字自卫的替代方法”，
   <strong class="markup--strong markup--p-strong">

   </strong>

  <p class="graf graf--p graf--startsWithDoubleQuote">

  <p class="graf graf--p">
   模块化设备的设计源于 Echo 的形状 —— 连接到 Alexa 虚拟助手的智能扬声器 —— 但是经过迭代改造，以创建比亚马逊不显眼的产品而且更友好更讨人喜欢。
