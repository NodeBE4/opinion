  <p class="graf graf--p">
   最近欧盟（EU）关于国民身份证的公告将摧毁数百万人的隐私，并创建一个近乎全球性的生物识别数据库。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “国家观察“新闻的
   <a class="markup--anchor markup--p-anchor" data-href="http://www.statewatch.org/news/2019/feb/eu-frontex-visas-ilos-id-cards.htm" href="http://www.statewatch.org/news/2019/feb/eu-frontex-visas-ilos-id-cards.htm" rel="noopener noreferrer" target="_blank">
    一篇文章透露
   </a>
   ，欧盟已经同意制作
   <strong class="markup--strong markup--p-strong">
    强制性
   </strong>
   国家生物识别身份证。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “作为欧盟‘安全联盟’的一部分正在谈判，发展迅速，理事会和议会就移民联络官新规则、欧盟签证准则、和强制性生物识别国民身份证的引入，达成了临时协议，并且理事会同意其关于新 Frontex 条例的谈判立场“。
  </p>
  <p class="graf graf--p">
   3月份，尼泊尔政府
   <a class="markup--anchor markup--p-anchor" data-href="http://kathmandupost.ekantipur.com/news/2019-03-03/data-security-concerns-over-national-identity-card-project.html" href="http://kathmandupost.ekantipur.com/news/2019-03-03/data-security-concerns-over-national-identity-card-project.html" rel="noopener noreferrer" target="_blank">
    已经宣布
   </a>
   计划推出一项影响3000万人口的国家生物识别身份证。
  </p>
  <p class="graf graf--p">
   上个月，我写了
   <a class="markup--anchor markup--p-anchor" data-href="https://massprivatei.blogspot.com/2019/02/worldwide-effort-to-restrict-everyones.html" href="https://massprivatei.blogspot.com/2019/02/worldwide-effort-to-restrict-everyones.html" rel="noopener noreferrer" target="_blank">
    一篇文章
   </a>
   ，警告人们全球当权者正在努力限制每个人的旅行权。但是整个欧洲和亚洲正在发生的事，应该让每个人都感到寒意。
  </p>
  <p class="graf graf--p">
   如果你将欧洲正在发生的事与美国的国家生物识别身份证、Real-ID 结合起来，很明显能看到，每个人的自由旅行权都处于危险之中。
  </p>
  <p class="graf graf--p">
   — — 将有5.12亿人被迫放弃自己的隐私 — —
  </p>
  <p class="graf graf--p">
   欧洲国家生物识别身份证几乎是确定无疑的。将有512.6百万公民被迫放弃他们的隐私。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “今天，理事会主席和欧洲议会的代表达成了关于改善欧洲移民联络官网络运作法规的非正式协议。现在将提交欧盟大使代表理事会进行确认。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “布鲁塞尔时报”说，当法律生效时，成员国将有八年的时间来遵守。
  </p>
  <p class="graf graf--p">
   这将如何影响美国游客？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    任何希望访问欧盟28个成员国中的任何一个国家的外国人都
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://www.statewatch.org/news/2018/aug/vis-profiling-child-fingerprinting.htm" href="http://www.statewatch.org/news/2018/aug/vis-profiling-child-fingerprinting.htm" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     将被生物识别描述
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，并通过“欧洲旅行信息和授权系统 ETIAS”进行风险评估
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在这里看到视频：
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="314" src="//www.youtube.com/embed/ujlF7D9x2a0" width="560">
   </iframe>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “由于在欧盟不需要旅行签证的国家的公民不需要经过漫长的申请签证，ETIAS 将确保这些人不是安全威胁。
   <strong class="markup--strong markup--p-strong">
    这个旅行授权系统将收集、跟踪和更新有关访客的所有必要信息
   </strong>
   ，以确定他们进入申根国家是否安全。“
  </p>
  <p class="graf graf--p">
   如果 ETIAS 听起来很像美国国土安全部的“美国旅行授权电子系统”，他们的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.tsa.gov/travel/passenger-support/travel-redress-program" href="https://www.tsa.gov/travel/passenger-support/travel-redress-program" rel="noopener noreferrer" target="_blank">
    秘密禁飞名单
   </a>
   应该给自己加一个金星，因为这正是它的模仿。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果您和您的家人计划长时间访问某个国家/地区，您将别无选择，只能将生物识别数据提交给欧盟
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “与此同时，欧盟也正在升级签证信息系统（VIS） — 目前用于持有短期申根签证的所有申请人的信息，正在扩大其中包括长期签证和居留的信息文件; 在长期签证中强制执行生物识别数据收集; 并包括6岁及以上儿童的指纹。所有签证申请人都将被描述（要了解有关前往欧洲的更多信息，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.zerohedge.com/news/2019-03-10/us-citizens-will-need-visa-visit-europe-starting-2021" href="https://www.zerohedge.com/news/2019-03-10/us-citizens-will-need-visa-visit-europe-starting-2021" rel="noopener noreferrer" target="_blank">
    请点击此处
   </a>
   。）
  </p>
  <p class="graf graf--p">
   爱尔兰时报警告说，
   <strong class="markup--strong markup--p-strong">
    12岁以上的人都将被迫携带国家生物识别身份证
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “新法案建议对那些拥有身份证的欧盟公民（12岁以上）以及非欧盟家庭成员的居留卡强制要求生物识别数据，现在包括指纹和面部图像，存储在芯片中。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “国家观察“的文章还警告说，国家生物识别身份证必须每5年更新一次，对未成年人来说这个时间更少。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “身份证的有效期最短为5年，最长有效期为10年。成员国可以为70岁及以上的人发放较长时间有效的身份证。未成年人的身份证有效期可能不到5年。“
  </p>
  <p class="graf graf--p">
   ETIAS 模型的建立紧追美国国安部的类似监视计划，令人震惊。迫使数百万人携带国家生物识别身份证之举措具有欧洲警察国家的所有特征。
  </p>
  <p class="graf graf--p">
   难道我们没有从第二次世界大战中学到任何东西吗？
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    T
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.blacklistednews.com/article/71551/the-european-unions-mandatory-national-biometric-id-card-will-affect-512-million.html" href="https://www.blacklistednews.com/article/71551/the-european-unions-mandatory-national-biometric-id-card-will-affect-512-million.html" rel="noopener noreferrer" target="_blank">
    HE EUROPEAN UNION’S MANDATORY NATIONAL BIOMETRIC ID CARD WILL AFFECT 512 MILLION PEOPLE
   </a>
   . Forcing millions of people to carry national biometric ID cards has all the earmarks of a European police state. Didn’t we learn anything from World War II?
  </p>
