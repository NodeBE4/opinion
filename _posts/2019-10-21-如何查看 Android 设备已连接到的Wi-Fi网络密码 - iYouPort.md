  <p class="graf graf--p">
   您的手机可能已连接到数十个Wi-Fi网络。学校、家、工作单位、健身房、朋友的房子、咖啡店 ……
  </p>
  <p class="graf graf--p">
   每次您输入其中一个 Wi-Fi 密码时，您的 Android 设备都会将其保存起来，以便日后轻松访问。
  </p>
  <p class="graf graf--p">
   这里唯一的麻烦在于，您无法查看设备存储的 Wi-Fi 密码。
  </p>
  <p class="graf graf--p">
   也许您想要将第二个设备连接到其中一个网络，或者朋友想要登录同一个热点，但无法访问设备存储的实际密码。
  </p>
  <p class="graf graf--p">
   因此，开发人员 Alexandros Schillings 决定通过一个应用程序来解决这个问题，该应用程序允许您查看您在 Android 设备上保存过的每个Wi-Fi密码。
  </p>
  <p class="graf graf--p">
   该应用程序需要 root 访问权限，但它可以非常轻松地查找和共享存储的 Wi-Fi 密码。
  </p>
  <p class="graf graf--p">
   看一个视频，后面是文字介绍：
  </p>
  <p>
   <iframe allowfullscreen="allowfullscreen" height="393" src="//www.youtube.com/embed/Pb0NGCKFa_M" width="700">
   </iframe>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第一步：安装 WiFi Key Recovery
   </strong>
  </p>
  <p class="graf graf--p">
   Schillings 的应用程序称为 WiFi 密钥恢复，它可以在 Google Play 商店免费获得。
  </p>
  <p class="graf graf--p">
   安装在您的 root 设备上。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <a class="markup--anchor markup--li-anchor" data-href="https://play.google.com/store/apps/details?id=aws.apps.wifiKeyRecovery" href="https://play.google.com/store/apps/details?id=aws.apps.wifiKeyRecovery" rel="nofollow noopener noreferrer" target="_blank">
     <strong class="markup--strong markup--li-strong">
      Install WiFi Key Recovery for free from the Google Play Store
     </strong>
    </a>
   </li>
  </ul>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5896 jetpack-lazy-image" data-lazy-sizes="(max-width: 1080px) 100vw, 1080px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-10.jpg?resize=1080%2C426&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-10.jpg?w=1080&amp;ssl=1 1080w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-10.jpg?resize=300%2C118&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-10.jpg?resize=768%2C303&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-10.jpg?resize=1024%2C404&amp;ssl=1 1024w" data-recalc-dims="1" height="426" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-10.jpg?resize=1080%2C426&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1080"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5896" data-recalc-dims="1" height="426" sizes="(max-width: 1080px) 100vw, 1080px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-10.jpg?resize=1080%2C426&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-10.jpg?w=1080&amp;ssl=1 1080w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-10.jpg?resize=300%2C118&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-10.jpg?resize=768%2C303&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-10.jpg?resize=1024%2C404&amp;ssl=1 1024w" width="1080"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第二步：查看已保存的 Wi-Fi 密码
   </strong>
  </p>
  <p class="graf graf--p">
   当您首次启动WiFi密钥恢复时，该应用程序将要求超级用户访问权限。
  </p>
  <p class="graf graf--p">
   点击弹出窗口中的“授予”，然后您将进入包含您曾连接过的所有Wi-Fi网络的列表，其中每个条目在“psk”字段中显示密码。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5897 jetpack-lazy-image" data-lazy-sizes="(max-width: 1068px) 100vw, 1068px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-10.png?resize=1068%2C920&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-10.png?w=1068&amp;ssl=1 1068w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-10.png?resize=300%2C258&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-10.png?resize=768%2C662&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-10.png?resize=1024%2C882&amp;ssl=1 1024w" data-recalc-dims="1" height="920" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-10.png?resize=1068%2C920&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1068"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5897" data-recalc-dims="1" height="920" sizes="(max-width: 1068px) 100vw, 1068px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-10.png?resize=1068%2C920&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-10.png?w=1068&amp;ssl=1 1068w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-10.png?resize=300%2C258&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-10.png?resize=768%2C662&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-10.png?resize=1024%2C882&amp;ssl=1 1024w" width="1068"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   如果您在拥有 Android 设备的过程中连接了许多不同的Wi-Fi网络，那么您的列表可能会很长。
  </p>
  <p class="graf graf--p">
   如果是这种情况，您可以使用“SSID Quicksearch”字段搜索特定的Wi-Fi网络。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5898 jetpack-lazy-image" data-lazy-sizes="(max-width: 1062px) 100vw, 1062px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-5.png?resize=1062%2C920&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-5.png?w=1062&amp;ssl=1 1062w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-5.png?resize=300%2C260&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-5.png?resize=768%2C665&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-5.png?resize=1024%2C887&amp;ssl=1 1024w" data-recalc-dims="1" height="920" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-5.png?resize=1062%2C920&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1062"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5898" data-recalc-dims="1" height="920" sizes="(max-width: 1062px) 100vw, 1062px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-5.png?resize=1062%2C920&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-5.png?w=1062&amp;ssl=1 1062w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-5.png?resize=300%2C260&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-5.png?resize=768%2C665&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/3-5.png?resize=1024%2C887&amp;ssl=1 1024w" width="1062"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    第三步：共享Wi-Fi密码
   </strong>
  </p>
  <p class="graf graf--p">
   如果您需要与其他设备共享其中一个密码，这里有一些选项。
  </p>
  <p class="graf graf--p">
   首先点击列表中的任何条目，然后会弹出一个小菜单。这允许您使用 Android 的内置共享系统来发送密码或整个条目。
  </p>
  <p class="graf graf--p">
   还有一个通过QR代码共享的选项，但这需要您安装一个
   <a class="markup--anchor markup--p-anchor" data-href="https://play.google.com/store/apps/details?id=com.google.zxing.client.android" href="https://play.google.com/store/apps/details?id=com.google.zxing.client.android" rel="noopener noreferrer" target="_blank">
    额外的应用程序
   </a>
   。我们解决没必要；如果你需要的话记得它是可用的。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5899 jetpack-lazy-image" data-lazy-sizes="(max-width: 1070px) 100vw, 1070px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-5.png?resize=1070%2C920&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-5.png?w=1070&amp;ssl=1 1070w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-5.png?resize=300%2C258&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-5.png?resize=768%2C660&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-5.png?resize=1024%2C880&amp;ssl=1 1024w" data-recalc-dims="1" height="920" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-5.png?resize=1070%2C920&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1070"/>

    <img alt="" class="aligncenter size-full wp-image-5899" data-recalc-dims="1" height="920" sizes="(max-width: 1070px) 100vw, 1070px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-5.png?resize=1070%2C920&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-5.png?w=1070&amp;ssl=1 1070w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-5.png?resize=300%2C258&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-5.png?resize=768%2C660&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/4-5.png?resize=1024%2C880&amp;ssl=1 1024w" width="1070"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   如果您想存储整个列表以便妥善保管，请点按设备的硬件菜单按钮（或软件导航栏中的三点菜单按钮）。
  </p>
  <p class="graf graf--p">
   选择“导出”以生成包含所有这些信息的文本文件。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-5900 jetpack-lazy-image" data-lazy-sizes="(max-width: 1066px) 100vw, 1066px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/5-4.png?resize=1066%2C920&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/5-4.png?w=1066&amp;ssl=1 1066w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/5-4.png?resize=300%2C259&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/5-4.png?resize=768%2C663&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/5-4.png?resize=1024%2C884&amp;ssl=1 1024w" data-recalc-dims="1" height="920" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/5-4.png?resize=1066%2C920&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1066"/>

  <p class="graf graf--p">
   您可以使用 Android 的共享菜单将此文件发送到另一台设备，或者如果您希望将此文本文件的副本保存到设备的存储分区，则只需点击“To SD”即可。
  </p>
  <p class="graf graf--p">
   当然，如果您的列表不是太长，您可以随时拍摄（并分享）屏幕截图。
