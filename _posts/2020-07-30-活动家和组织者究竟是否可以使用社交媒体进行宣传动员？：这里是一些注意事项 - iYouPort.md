  <p class="graf graf--p">
   社交媒体对于组织公民抵抗运动来说可能是种绝妙的工具，但是，对于行动者而言，它们也很可能是危险的地方。
  </p>
  <p class="graf graf--p">
   阿拉伯之春的灾难之后，几乎所有安全人士都会告诉你：远离社交媒体。
  </p>
  <p class="graf graf--p">
   但某些运动可能会决定冒着很高的风险去使用 Facebook 或微信这类东西，只是为了更快获得知名度和公众支持，或者只是个姿态，以此来表明他们不害怕受到威胁。
  </p>
  <p class="graf graf--p">
   而更多其他抗议运动为了试图避免风险就根本不使用社交媒体。
  </p>
  <p class="graf graf--p">
   简单说，反抗者必须做出战略性的选择。本文的目的是帮助您评估在抗议活动中使用社交媒体工具的风险。
  </p>
  <p class="graf graf--p">
   👉首先，对于抗议者来说，重要的是要了解：即使您所说的一切都处于 “封闭” 或 “私有” 的群组页面中，也
   <strong class="markup--strong markup--p-strong">
    不应
   </strong>
   被视为安全 —— 只要您使用的是寡头公司提供的服务，就是那些大多数人都能叫得上名字的服务。
  </p>
  <p class="graf graf--p">
   即便是真正靠谱的端对端加密通信（比如signal）您也要仔细审核每个人，否则很难阻止
   <strong class="markup--strong markup--p-strong">
    渗透者
   </strong>
   进入您的团队。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    telegram那种大群组极其不安全
   </strong>
   ，并且它默认不加密；微信、Facebook 就更不用说了。
  </p>
  <p class="graf graf--p">
   即使您确实信任群组中的每个人的身份，这些寡头公司也会将您的私人信息转交给政府、将个人页面和群组页面的内容移交给政府。他们经常干这种事，而且会持续干这种事。还记得吗👇
  </p>

   <li class="graf graf--li">

     他们是如何抓捕说真话的人？灾难和教训
    </a>
    》

  </ul>
  <p class="graf graf--p">
   于是，作为思考清单，
   <strong class="markup--strong markup--p-strong">
    您至少需要问自己以下这些问题
   </strong>
   ：
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    您是否仅出于组织目的就有被逮捕的风险、或人身威胁危险？如果是这样，您准备好了吗？
   </li>
   <li class="graf graf--li">
    社区成员如果给您点赞表示支持，他们是否有风险？如果有，他们准备好了吗？
   </li>
   <li class="graf graf--li">
    准备参加抗议运动的人们如果分享了任何相关帖子或评论，是否可能会在游行发生之前、期间或之后被逮捕？如果是这样，他们准备好了吗？
   </li>
  </ol>
  <p class="graf graf--p">
   ⚠️如果您知道您所在地区的人们可能仅仅因为组织抗议 —— 而不是实际抗议 —— 就会面临被捕和镇压，或者，如果您的社区成员可能只是出于对您正在组织的运动表现出兴趣 —— 而不是实际参与其中 —— 就会受到迫害，那么，社交媒体绝对不应该是您的选择。
  </p>
  <p class="graf graf--p">
   除非您愿意接受这些风险。
  </p>
  <p class="graf graf--p">
   如果您的队友担心政府的攻击，可以鼓励社区定期访问您的页面，而不必任何点赞或回复，但是您可能会发现总体参与度非常低 —— 这完全有悖于您冒险使用社交媒体的目的对吗。
  </p>
  <p class="graf graf--p">
   通过 Twitter 进行交流具有一定的附加价值。因为人们无需关注您也可能浏览您的内容 —— 我们尤其推荐不关注的方法，👉也就是说，您最好不要关注您真正感兴趣的东西，而是随便关注一些完全不感兴趣的东西，也就是
   <strong class="markup--strong markup--p-strong">
    制造噪音
   </strong>
   ，这样追踪者就很难映射您的内心全景图。
  </p>
  <p class="graf graf--p">
   只要人们不去点赞和转发您的内容，可以安全。
  </p>
  <p class="graf graf--p">
   在某些国家/地区，基于云的即时消息传递应用程序 Telegram 已成为组织游行示威的必备工具。
  </p>
  <p class="graf graf--p">
   而在一些国家，比如伊朗和俄罗斯等感觉，Telegram 的普及已经导致政府的加强监视和干预。
  </p>
  <p class="graf graf--p">
   Telegram 群组是公开渠道，其中的任何内容都可以被组内的任何人阅读。对于公共频道来说，您完全无法控制关注者，因此，任何人都可以查看您的频道的全部历史记录。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    过去已经出现了 Telegram 频道组织者的重要个人信息被移交给政府的案例，
   </strong>
   所以，如果您准备手持身份证站在中心广场上大喊出您想要说的所有话，那么没问题；如果不是这样，最好还是不要使用 Telegram 组织抗议比较好。
  </p>
  <p class="graf graf--p">
   Telegram 唯一的利好就是频道，它是公开的、可以谷歌索引的和可跨平台分享的，很像中国的公众号，对于提供知识和资源来说，是个有利的地方。此外没有优势。
  </p>
  <p class="graf graf--p">

    关闭了
   </a>
   5条不同的输油管道，以引起当局的注意 —— 故意让当局逮捕他们，为的是引起更多人关注他们的反抗事业，并在法庭上证明他们的行为对于保护人民是必要的。
  </p>
  <p class="graf graf--p">
   这些激进活动家并不是坐在管道旁等着警察来逮捕他们，他们还制作了有关整个行动的视频、网站和社交媒体帖子。
  </p>
  <p class="graf graf--p">
   👉对于使用社交媒体进行行动组织，最重要的是要记住常识：仅仅因为许多人使用某个社交媒体（Facebook/微信/Twitter）并不意味着活动家应该将其用于组织游行示威和其他抗议行动。
  </p>
  <p class="graf graf--p">
   就如公民抵抗运动中的任何战术一样，活动家和组织者
   <strong class="markup--strong markup--p-strong">
    必须采取有目的的战略方法
   </strong>
   —— 仔细比较自己希望使用的在线平台的优缺点，尤其，要详细考虑如何最好地支持可能受到威胁的抗议运动的生命和生计。⚪️
  </p>
