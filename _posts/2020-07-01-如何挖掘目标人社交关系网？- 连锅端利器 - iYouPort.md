   再次强调
   <strong class="markup--strong markup--p-strong">
    安全101
   </strong>
   ：重要的人际关系切勿在互联网上以任何形式体现！是*任何*。否则您和您的同伴都有可能陷入危险。
  </p>
  <p class="graf graf--p">
   我们曾经介绍过这种追踪方式，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e6%8c%96%e6%8e%98%e7%9b%ae%e6%a0%87%e4%ba%ba%e9%9a%90%e8%97%8f%e7%9a%84%e4%ba%ba%e9%99%85%e5%85%b3%e7%b3%bb%ef%bc%9f/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e6%8c%96%e6%8e%98%e7%9b%ae%e6%a0%87%e4%ba%ba%e9%9a%90%e8%97%8f%e7%9a%84%e4%ba%ba%e9%99%85%e5%85%b3%e7%b3%bb%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    如何挖掘目标人隐藏的人际关系？
   </a>
   》。这里是 osintcombine 介绍的一种也许是更有效的追踪方法。
  </p>
  <p class="graf graf--p">
   在社交媒体世界中，确定一个人的真正社交关系网可能是一个挑战，因为要追踪的平台太多了，而且其中很多平台并不限制谁可以关注您、或默认情况下你可以关注的人（例如 Twitter 和 Instagram）。👉这意味着朋友列表和关注者/被关注列表
   <strong class="markup--strong markup--p-strong">
    不能
   </strong>
   准确描绘真实的合作/私人关系。
  </p>
  <p class="graf graf--p">
   在这种情况下可以使用的一种方法是分析社会互动指标，以计算出个人所处的真实群体。
  </p>
  <p class="graf graf--p">
   这意味着什么？ —— 意味着捕获用户发表的帖子的点赞或评论，并根据用户与他人互动的频率将用户集群分为几类。
  </p>
  <p class="graf graf--p">
   ⚠️ 仅分析 Facebook 上一个用户的6张照片以确定谁通常点赞这些照片，基本就可以非常准确地确定目标人的共同圈子。
  </p>
  <p class="graf graf--p">
   有时，在分析伪造的个人资料时也有用，它可以揭示个人资料背后的真实人物，因为不专业的伪装会展示出较差的情报技术、并且其兴趣通常会显得很假。
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   使用上面的概念，可以发现一种简单且可重复的方法就是：
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    刮取帖子的点赞和评论，并将用户名的结果放在电子表格的单个列中。
   </li>
   <li class="graf graf--li">
    对尽可能多的照片/评论重复此操作，并继续将用户名添加到列表中，以创建一个名字的长列表，其中将包含很多重复项。
   </li>
   <li class="graf graf--li">
    利用这些重复项来计算同一用户出现的频率，以识别常规社交圈子/普通组织的集群。
   </li>
  </ol>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    特殊公式
   </strong>
   ：要计算名字从第一列出现的频率，只需将此公式放在第二列的第一个单元格中，复制即可：“
   <em class="markup--em markup--p-em">
    =COUNTIF(A:A, A1)”
   </em>
  </p>
  <p class="graf graf--p">
   可以通过在A列中写入名字列表，然后在B列中放入特殊公式来模拟此步骤。
