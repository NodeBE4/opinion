  <p>
   不要使用“TurboVPN”。该vpn声称不保留日志，但这不是事实。其隐私政策中你可以看到（下图），它收集各种用户信息，这与真正的无日志服务完全相反 —— 这个隐私政策非常难以找到，如果你使用电脑基本是找不到的。

  <p>
   Turbo 只有在使用“最快服务器”时的表现才稳定。而一旦连接中断，你的流量很容易因此暴露；同样，所有没有自定义DNS设置的VPN都无法100%保证你的安全。
  </p>
  <p>
   Turbo VPN 无法在中国下载，这可能是个障眼法。您需要小心：
   <strong>
    并不是
   </strong>
   所有被 App Store 下架的VPN都是“安全的”。
  </p>
  <p>
   我们曾经发布过《
   <a href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e9%aa%8c%e8%af%81%e6%82%a8%e7%9a%84-vpn-%e8%bf%9e%e6%8e%a5%e6%98%af%e5%90%a6%e5%ae%89%e5%85%a8%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    如何判断VPN连接安全
   </a>
   》的简易介绍；
   <strong>
    今天介绍的内容是前者的延伸
   </strong>
   。
  </p>
  <p class="graf graf--p">
   很多VPN声称自己是加密的，那么你怎样才能得知它们是不是骗子呢？
  </p>
  <p class="graf graf--p">
   其实不需要您学会怎么玩数据包嗅探器和协议分析器，一个网络实用程序就可以做到，它使您可以查看从您的计算机来回传输的数据。
  </p>
  <p class="graf graf--p">
   在未连接到VPN时检查一次，并捕获了一些数据包，然后在连接后再做同样的事。很简单就可以看到传输的数据在发送到VPN时是否已经加密。
  </p>
  <p class="graf graf--p">
   即便您没有任何技术专长也不用担心。您所需要做的就是按下一个记录按钮，滚动列表并检查一些文本。
  </p>
  <p class="graf graf--p">
   您可以忽略其他所有内容，因为除非您对计算机和网络有所了解，否则您不需要那些东西。
  </p>
  <p class="graf graf--p">
   在Mac上，使用
   <a class="markup--anchor markup--p-anchor" data-href="http://www.tastycocoabytes.com/cpa/" href="http://www.tastycocoabytes.com/cpa/" rel="noopener noreferrer" target="_blank">
    CocoaPacketAnalyzer
   </a>
   ，在PC上，使用
   <a class="markup--anchor markup--p-anchor" data-href="http://www.wireshark.org/download.html" href="https://www.wireshark.org/download.html" rel="noopener noreferrer" target="_blank">
    Wireshark
   </a>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     在Mac上验证加密
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   首先，在Mac上下载 CocoaPacketAnalyzer 并运行它。您应该看到带有四个大按钮的启动屏幕。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10979 jetpack-lazy-image" data-lazy-sizes="(max-width: 652px) 100vw, 652px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-35.png?resize=652%2C782&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-35.png?w=652&amp;ssl=1 652w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-35.png?resize=250%2C300&amp;ssl=1 250w" data-recalc-dims="1" height="782" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-35.png?resize=652%2C782&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="652"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10979" data-recalc-dims="1" height="782" sizes="(max-width: 652px) 100vw, 652px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-35.png?resize=652%2C782&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-35.png?w=652&amp;ssl=1 652w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-35.png?resize=250%2C300&amp;ssl=1 250w" width="652"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   点击 Capturing 即可开始捕获网络流量。如果您保存捕获会话，则称为 trace file，稍后您可以使用第二个按钮再次打开它们。
  </p>
  <p class="graf graf--p">
   出于本文的目的，我们只希望看到数据是否已加密，因此不需要也不会保存任何捕获数据。
  </p>
  <p class="graf graf--p">
   在执行捕获之前，请继续并单击“首选项”以设置如何捕获数据。
  </p>
  <p class="graf graf--p">
   单击顶部的
   <strong class="markup--strong markup--p-strong">
    Capture
   </strong>
   ，需要在此处检查的唯一设置是
   <strong class="markup--strong markup--p-strong">
    Capture Interface
   </strong>
   。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-10980 jetpack-lazy-image" data-lazy-sizes="(max-width: 904px) 100vw, 904px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-26.png?resize=904%2C470&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-26.png?w=904&amp;ssl=1 904w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-26.png?resize=300%2C156&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-26.png?resize=768%2C399&amp;ssl=1 768w" data-recalc-dims="1" height="470" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/2-26.png?resize=904%2C470&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="904"/>

   首先请确保按下列表框右侧的刷新小按钮。
  </p>
  <p class="graf graf--p">
   当您点击该框时，您会看到几个选项，所有这些看起来都很混乱。不要管它，您唯一需要选择的就是列出了
   <strong class="markup--strong markup--p-strong">
    IPv4
   </strong>
   并带有一些数字的那串东西。
  </p>
  <p class="graf graf--p">
   您无需选择包含127.0.0.1的那个，需要选择一个具有当前正在使用的连接的IP地址的条目。您可以通过转到“系统偏好设置”，然后单击“网络”来解决此问题。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10981 jetpack-lazy-image" data-lazy-sizes="(max-width: 938px) 100vw, 938px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-25.png?resize=938%2C406&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-25.png?w=938&amp;ssl=1 938w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-25.png?resize=300%2C130&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-25.png?resize=768%2C332&amp;ssl=1 768w" data-recalc-dims="1" height="406" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-25.png?resize=938%2C406&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="938"/>

  <p class="graf graf--p">
   单击左侧列表框中带有绿点的连接，然后检查右侧部分的IP地址字段。
  </p>
  <p class="graf graf--p">
   如您所见，它显示为 192.168.1.x，因此与程序下拉列表中的
   <strong class="markup--strong markup--p-strong">
    en0 — IP4–192.168.1.68
   </strong>
   选项匹配。
  </p>
  <p class="graf graf--p">
   现在关闭 “Capture Preferences” 窗口，以返回到主屏幕。
  </p>
  <p class="graf graf--p">
   继续并单击 “
   <strong class="markup--strong markup--p-strong">
    Capturing
   </strong>
   ”，现在将弹出一个新对话框，您可以在其中选择一些设置，然后开始捕获。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10982 jetpack-lazy-image" data-lazy-sizes="(max-width: 858px) 100vw, 858px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-21.png?resize=858%2C534&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-21.png?w=858&amp;ssl=1 858w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-21.png?resize=300%2C187&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-21.png?resize=768%2C478&amp;ssl=1 768w" data-recalc-dims="1" height="534" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-21.png?resize=858%2C534&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="858"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10982" data-recalc-dims="1" height="534" sizes="(max-width: 858px) 100vw, 858px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-21.png?resize=858%2C534&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-21.png?w=858&amp;ssl=1 858w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-21.png?resize=300%2C187&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-21.png?resize=768%2C478&amp;ssl=1 768w" width="858"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在这里您不需要进行任何更改，因此要开始使用，只需按
   <strong class="markup--strong markup--p-strong">
    Start
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是，在执行操作之前，需要注意一些事项。
  </p>
  <p class="graf graf--p">
   首先，您应该尝试关闭所有正在运行的程序和任务栏程序，并仅使浏览器窗口保持打开状态。这是为了尽可能减少数据流让任务变得简单一点。
  </p>
  <p class="graf graf--p">
   之后，立即加载一个或两个站点，然后单击“停止”。您只需要捕获几秒钟就够了。您甚至可以直接在浏览器的标签页中输入网址，那么一旦开始捕获，您只需按Enter即可加载页面。
  </p>
  <p class="graf graf--p">
   按下 “Stop” 后，您将看到一个类似于下面的窗口：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10983 jetpack-lazy-image" data-lazy-sizes="(max-width: 940px) 100vw, 940px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-16.png?resize=940%2C442&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-16.png?w=940&amp;ssl=1 940w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-16.png?resize=300%2C141&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-16.png?resize=768%2C361&amp;ssl=1 768w" data-recalc-dims="1" height="442" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5-16.png?resize=940%2C442&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="940"/>

  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   由于可能有成千上万的行，您只需快速按下向下箭头并查看底部的数据更改即可。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10984 jetpack-lazy-image" data-lazy-sizes="(max-width: 766px) 100vw, 766px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-9.png?resize=766%2C626&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-9.png?w=766&amp;ssl=1 766w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-9.png?resize=300%2C245&amp;ssl=1 300w" data-recalc-dims="1" height="626" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/6-9.png?resize=766%2C626&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="766"/>

  <p class="graf graf--p">
   如果您的VPN连接实际上是加密的，那么滚动的每一行都应显示与上图中的数据相似的样子。
  </p>
  <p class="graf graf--p">
   由于它不可读，只有一堆随机字符，因此可以判断已加密。
  </p>
  <p class="graf graf--p">
   使用加密的连接，那
   <strong class="markup--strong markup--p-strong">
    数千行中的任何一行都不应该被读取任何内容
   </strong>
   。
  </p>

  </p>
  <p class="graf graf--p">
   并非每个数据包在未加密的连接上都是可读取的，但是对于大多数数据包来说，您都能够看到实际数据，HTML代码、协议标头等。

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">

     在PC上验证加密

  <p class="graf graf--p">
   在PC上进行检查的过程与上面显示的过程几乎相同，除了您使用的是另一个名为 Wireshark 的程序。

  </p>
  <p class="graf graf--p">
   我更喜欢 Wireshark，因为您实际上可以看到每个接口上正在传输多少数据，这使您很容易看出哪个是主要连接。

   现在，您所要做的就是单击“开始”按钮（在“接口列表”按钮下方），您将可以正常使用；您无需更改任何其他选项。完成捕获后您应该会看到如下屏幕：
