   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10053" data-recalc-dims="1" height="711" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/迪拜.png?resize=1100%2C711&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/迪拜.png?w=1364&amp;ssl=1 1364w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/迪拜.png?resize=300%2C194&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/迪拜.png?resize=768%2C497&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/迪拜.png?resize=1024%2C662&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/迪拜.png?resize=1100%2C711&amp;ssl=1 1100w" width="1100"/>

  </p>
  <p class="graf graf--p">
   路透社一篇文章的部分标题写着：“中央情报局没有监视阿联酋”。作为生活在海湾国家的人，我不得不笑了。阿联酋有很多间谍活动 —— 尤其是阿联酋自己的间谍 —— 政府一直在密切监视人民的一举一动。
  </p>
  <p class="graf graf--p">
   这可能令该地区以外的许多人感到意外，但迪拜和阿布扎比的确是全
   <a class="markup--anchor markup--p-anchor" data-href="https://www.comparitech.com/vpn-privacy/the-worlds-most-surveilled-cities/" href="https://www.comparitech.com/vpn-privacy/the-worlds-most-surveilled-cities/" rel="noopener noreferrer" target="_blank">
    世界受监视最密切的城市
   </a>
   之一。
  </p>
  <p class="graf graf--p">
   阿布扎比估计有20,000个针对其150万人口的监视摄像机，而迪拜则有大约35,000个监视摄像机，监视着280万人。
  </p>
  <p class="graf graf--p">
   相比之下，华盛顿特区监视摄像机数量大约为4,000。如果论人均受到最多监视的城市，那么大多数在中国。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    这里是调查报道《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e5%92%8c%e4%b8%ad%e5%9b%bd%e5%9c%a8%e7%9b%91%e8%a7%86%e5%85%ac%e6%b0%91%e6%96%b9%e9%9d%a2%e5%bc%95%e9%a2%86%e5%85%a8%e7%90%83/" href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e5%92%8c%e4%b8%ad%e5%9b%bd%e5%9c%a8%e7%9b%91%e8%a7%86%e5%85%ac%e6%b0%91%e6%96%b9%e9%9d%a2%e5%bc%95%e9%a2%86%e5%85%a8%e7%90%83/" rel="noopener noreferrer" target="_blank">
     美国和中国在监视公民方面引领全球
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   这种监视级别可以说有一些好处，最明显的是犯罪率低。当我在迪拜任教时，来自尼日利亚的一名学生讲了一个有关她曾经如何将手提袋落在公交车站长椅上的故事。
  </p>
  <p class="graf graf--p">
   当她再次找回手提袋时她发现完好无损，里面的现金都还在。没有人感到惊讶。迪拜被认为是
   <a class="markup--anchor markup--p-anchor" data-href="https://whatson.ae/dubai/2019/07/abu-dhabi-and-dubai-ranked-among-the-safest-cities-in-the-world/" href="https://whatson.ae/dubai/2019/07/abu-dhabi-and-dubai-ranked-among-the-safest-cities-in-the-world/" rel="noopener noreferrer" target="_blank">
    世界上“最安全”的地方
   </a>
   之一，所有人都知道到处都有监视器在瞄准每个人。
  </p>
  <p class="graf graf--p">
   但是，知道有人正在偷窥你或监听你，那种感觉肯定不是什么好事。迪拜只有15％的居民是阿联酋人。其余都是外国人，和我一样，大多数人对阿联酋提供的工作和生活表示感谢。
  </p>
  <p class="graf graf--p">
   没有人愿意犯任何可能导致失业、被驱逐出境或入狱的“过错”，而且我们有充分的理由发展某种程度的偏执狂。
  </p>
  <p class="graf graf--p">
   例如，当我于2014年首次飞往迪拜从事教学工作时，移民代理立即扫描了我的眼睛（虹膜，不用担心我有多困或涂了睫毛膏）。
  </p>
  <p class="graf graf--p">
   接下来，有人告诉我必须就地购买SIM卡或手机。这也发生在我的同事身上。
  </p>
  <p class="graf graf--p">
   没有人解释为什么这样做是必要的，但是我们注意到了一些结果 —— 每当我们出国旅行时，我们都会那个被规定购买的手机上收到短信，内容如下：“阿联酋大使馆欢迎您来到德国法兰克福。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    简单说，它一直在跟踪你。
   </strong>
  </p>
  <p class="graf graf--p">
   而且，在我们返回迪拜后，也会立刻收到短信，“欢迎回到迪拜”。
  </p>
  <p class="graf graf--p">
   这些短信听起来是友好的，但是，受到监视的潜在含义却远非如此。
  </p>
  <p class="graf graf--p">
   然后是已经众所周知的在线监视。
  </p>
  <p class="graf graf--p">
   去年的轰动消息：
   <strong class="markup--strong markup--p-strong">
    阿联酋已经召集了一支美国雇佣军队伍，其中许多人曾是美国国家安全局的特工，目的是为入侵和监视阿联酋的政治批评家和记者。
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    在这里看到详细报道《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%bb%ba%e7%ab%8b%e4%b8%80%e4%b8%aa%e9%97%b4%e8%b0%8d%e5%b8%9d%e5%9b%bd/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%bb%ba%e7%ab%8b%e4%b8%80%e4%b8%aa%e9%97%b4%e8%b0%8d%e5%b8%9d%e5%9b%bd/" rel="noopener noreferrer" target="_blank">
     雄厚的财力，深藏不露：美国间谍如何协助专制国家构建一个间谍帝国
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e9%97%b4%e8%b0%8d%e5%b8%ae%e5%8a%a9%e9%98%bf%e8%81%94%e9%85%8b%e4%be%b5%e7%8a%af%e4%ba%ba%e6%9d%83%e4%b8%80%e4%ba%8b%e6%8f%90%e9%86%92%e4%ba%86%e4%bb%80%e4%b9%88/" href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e9%97%b4%e8%b0%8d%e5%b8%ae%e5%8a%a9%e9%98%bf%e8%81%94%e9%85%8b%e4%be%b5%e7%8a%af%e4%ba%ba%e6%9d%83%e4%b8%80%e4%ba%8b%e6%8f%90%e9%86%92%e4%ba%86%e4%bb%80%e4%b9%88/" rel="noopener noreferrer" target="_blank">
     美国间谍帮助阿联酋侵犯人权一事提醒了什么
    </a>
    》
   </li>
   <li class="graf graf--li">
    关于迪拜的遭遇《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%bf%98%e6%98%af%e9%ab%98%e5%a0%a1%e5%a5%87%e4%ba%ba%ef%bc%9a%e4%b8%ad%e7%be%8e1984%e5%9c%a8%e6%b5%b7%e6%b9%be%e7%9a%84%e9%bb%91%e6%89%8b/" href="https://www.iyouport.org/%e8%bf%98%e6%98%af%e9%ab%98%e5%a0%a1%e5%a5%87%e4%ba%ba%ef%bc%9a%e4%b8%ad%e7%be%8e1984%e5%9c%a8%e6%b5%b7%e6%b9%be%e7%9a%84%e9%bb%91%e6%89%8b/" rel="noopener noreferrer" target="_blank">
     还是高堡奇人：中美 1984 在海湾的黑手
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    阿联酋政府是以色列臭名昭著的网络情报公司 NSO Group 的客户，曾多次被发现在持不同政见者的电话上安装其强大的间谍软件
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    在这里看到详细报道 “
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e4%bb%a5%e8%89%b2%e5%88%97%e5%92%8cnso/" href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e4%bb%a5%e8%89%b2%e5%88%97%e5%92%8cnso/" rel="noopener noreferrer" target="_blank">
     以色列和NSO
    </a>
    ”
   </li>
  </ul>
  <p class="graf graf--p">
   事实显示，我们不仅要注意自己的电子设备和互联网活动。在我到达迪拜后不久，一个访客出乎意料地来到我的办公室。
  </p>
  <p class="graf graf--p">
   此人年纪有点大了，肯定不是学生，但是，他明确说他想旁听我的课。
  </p>
  <p class="graf graf--p">
   我解释了可用的不同课程以及它们可以提供的职业机会。他表示了感谢，然后，礼貌地提醒我
   <strong class="markup--strong markup--p-strong">
    要随身携带手机
   </strong>
   。
  </p>
  <p class="graf graf--p">
   然后，他将谈话的内容从关于课堂的闲谈转移到了关于政治事务的闲谈上。
  </p>
  <p class="graf graf--p">
   您可以想象所有这些产生的影响。
   <strong class="markup--strong markup--p-strong">
    当我和我的同事回到由雇主提供的公寓时，我们开始自我审查自己的电子邮件和写过的所有文字，就像有人在站在我们的肩膀上每时每刻都在听我们的谈话一样
   </strong>
   。
  </p>
  <p class="graf graf--p">
   我们没有写任何有颠覆性的东西 —— 或至少我没有写过 ——
   <strong class="markup--strong markup--p-strong">
    但这并没有停止我们的提心吊胆。
   </strong>
  </p>
  <p class="graf graf--p">
   在讨论政治时低声说话是很常有的事，或者根本不谈论政治。
  </p>
  <p class="graf graf--p">
   这种感觉是如此的普遍 —— 我的一名埃及学生告诉我，
   <strong class="markup--strong markup--p-strong">
    她的家人把所有手机都放到一个房间里，并关上门，然后聚集在另一个房间里进行各种对话。
   </strong>
  </p>
  <p class="graf graf--p">
   但是，如果我们想与亲人交谈，我们别无选择，只能使用手机。 Skype 或 FaceTime ？在迪拜
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnbc.com/2018/05/02/microsoft-and-apple-could-get-skype-and-facetime-bans-lifted-in-uae.html" href="https://www.cnbc.com/2018/05/02/microsoft-and-apple-could-get-skype-and-facetime-bans-lifted-in-uae.html" rel="noopener noreferrer" target="_blank">
    不允许
   </a>
   。
  </p>
  <p class="graf graf--p">
   甚至在睡着的时候也希望自己没有梦到任何颠覆性的事。我们会对自己说，我们可以这样做直到合同完成。
  </p>
  <p class="graf graf--p">
   然后，我们就可以回家了 ——
   <strong class="markup--strong markup--p-strong">
    继续由市场营销人员监视我们购买的商品，由数据经纪人监视收集并转售我们在互联网上的一举一动，并记录我们的健康历史 ……
   </strong>
  </p>
  <p class="graf graf--p">
   我们可以逃跑，但是，老大哥就在我们衣袋里。⚪️
  </p>
  <p class="graf graf--p">

    What is it like to live in a modern surveillance state? Look to Dubai.
   </a>
  </p>
