  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     基本运算符的简单示例
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   它们可以组合使用。别忘了这里介绍的技巧：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%8e%e6%8e%a8%e7%89%b9%e4%b8%ad%e6%8c%96%e6%8e%98%e7%9c%9f%e7%9b%b8%e4%b8%8d%e9%9c%80%e8%a6%81%e5%a4%aa%e5%a4%8d%e6%9d%82%e7%9a%84%e5%b7%a5%e5%85%b7%ef%bc%9a%e4%b8%80%e4%b8%aa%e5%b8%b8%e7%94%a8/" href="https://www.iyouport.org/%e4%bb%8e%e6%8e%a8%e7%89%b9%e4%b8%ad%e6%8c%96%e6%8e%98%e7%9c%9f%e7%9b%b8%e4%b8%8d%e9%9c%80%e8%a6%81%e5%a4%aa%e5%a4%8d%e6%9d%82%e7%9a%84%e5%b7%a5%e5%85%b7%ef%bc%9a%e4%b8%80%e4%b8%aa%e5%b8%b8%e7%94%a8/" rel="noopener noreferrer" target="_blank">
    从推特中挖掘真相不需要太复杂的工具：一个常用工具的全面指南
   </a>
   》。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    cybersec
   </strong>
   带有 “cybersec” 一词的推文。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    security -information
   </strong>
   带有 “security” 这个词的推文，但没有 “information” 这个词。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    HongKong lang:en
   </strong>
   “HongKong” 相关推文，仅限英文。您可以换任何语种，使用 fr，nl，ar，bg，cs 等。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    filter:verified elections
   </strong>
   经过验证的帐户发布有关选举的信息。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    from:username
   </strong>
   你想要的目标用户发送的推文。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    to:username -filter:links
   </strong>
   发送给目标用户的推文，没有链接。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    hongkong filter:retweets
   </strong>
   旨在显示 hongkong 相关内容的转推。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    hongkong exclude:retweets
   </strong>
   过滤掉转推的 hongkong 推文。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    filter:replies cybersecurity
   </strong>
   回复给 cybersecurity 推文的文本。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    exclude:replies cybersecurity
   </strong>
   没有人回复的 Cybersecurity 推文。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    security list:TIME/time-staff
   </strong>
   security lists 中的推特账户。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    security min_retweets:10
   </strong>
   Security 推文被转发了至少10次或更多。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    security min_faves:10
   </strong>
   Security 推文被 点赞/收藏 10次或更多。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    conference near:cityname
   </strong>
   列出您输入的城市附近举行的会议/研讨会。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    security near:LA within:10mi
   </strong>
   来自以 Los Angelos 为中心10英里半径内的任何人的有关 Security 的推文。可以更改城市名（美国，2个大写字母）和英里半径数字。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    security reviews since:2019–01–01 until:2020–05–01
   </strong>
   从2019年1月1日到2020年5月01日内发布的有关 Security 的推文。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    hacking filter:links
   </strong>
   hacking 主题的内容带有关于该主题的链接。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    hacking filter:images
   </strong>
   有关 hacking 主题的图片。可添加标签以限定搜索。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    hacking filter:news
   </strong>
   带有新闻来源的 Hacking 主题推文。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    videos filter:safe
   </strong>
   标记为 safe 的视频内容。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    images filter:safe
   </strong>
   标记为 safe 的图片内容。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    cybersec filter:media
   </strong>
   关于 cybersec 的媒体推文。
  </p>

   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%bb%95%e5%bc%80%e6%8e%a8%e7%89%b9%e7%ae%97%e6%b3%95%e5%99%aa%e9%9f%b3%e3%80%81%e4%bd%bf%e7%94%a8%e5%9f%ba%e6%9c%ac%e8%bf%90%e7%ae%97%e7%ac%a6%e6%8c%96%e6%8e%98%e6%8e%a8%e7%89%b9%e6%95%b0%e6%8d%ae/" href="https://www.iyouport.org/%e7%bb%95%e5%bc%80%e6%8e%a8%e7%89%b9%e7%ae%97%e6%b3%95%e5%99%aa%e9%9f%b3%e3%80%81%e4%bd%bf%e7%94%a8%e5%9f%ba%e6%9c%ac%e8%bf%90%e7%ae%97%e7%ac%a6%e6%8c%96%e6%8e%98%e6%8e%a8%e7%89%b9%e6%95%b0%e6%8d%ae/" rel="noopener noreferrer" target="_blank">
    绕开推特算法噪音、使用基本运算符挖掘推特数据的方法

  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    suggest_activity_tweet
   </strong>
   告诉推特算法停止发送 “你可能喜欢” 的推文。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    suggest_recycled_tweet_inline 阻止
   </strong>
   重复的推文。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    suggest_grouped_tweet_hashtag
   </strong>
   阻止与流行标签相关的推文。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    suggest_who_to_follow
   </strong>
   这个就不用解释了……
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    suggest_activity
   </strong>
   阻止推荐的用户活动。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    suggest_activity_retweet
   </strong>
   阻止从特定平台的转发（比如官媒等纯宣传机构）
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    suggest_timeline_tweet
   </strong>
   阻止推特算法根据您的时间轴偏好的 “建议”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    suggest_recycled_tweet
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    suggest_ranked_timeline_tweet
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    suggest_recap
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    suggest_activity_tweet_inline
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    generic-activity-momentsbreaking
   </strong>
  </p>
  <p class="graf graf--p">
