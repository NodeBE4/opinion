  <p class="graf graf--p">
   Celly 允许人们建立小型的私人社交网络称为“细胞”，“细胞”的用户们可以从任何设备快速的接入网络。随着“细胞”规模的扩大，用户们的讨论可以被控制主题或者自由聊天。
  </p>
  <p class="graf graf--p">
   一个个“细胞”就像社交单元，它们可以连在一起来组成网络，并且灵活的规模和组成可以适应不同的组织。创建者可以分散处理安全信息，同时组织内可以进行无拘无束的沟通。
  </p>
  <p class="graf graf--p">
   通过应用，用户可以用手机建立细胞并且邀请其他移动用户参与其中。Celly还可以让用户发布投票，设定通告，并且追踪话题来更好的组织信息。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <a class="markup--anchor markup--li-anchor" data-href="https://cel.ly/" href="https://cel.ly/" rel="noopener noreferrer" target="_blank">
     Celly

    (亚裔)
   </li>
   <li class="graf graf--li">
    <a class="markup--anchor markup--li-anchor" data-href="https://www.stormfront.org/" href="https://www.stormfront.org/" rel="noopener noreferrer" target="_blank">
     Stormfront
    </a>
    (白人种族主义者)
   </li>
   <li class="graf graf--li">
    <a class="markup--anchor markup--li-anchor" data-href="http://www.indianz.com/" href="http://www.indianz.com/" rel="noopener noreferrer" target="_blank">
     Indianz
    </a>
    (印度裔美国人)
   </li>
  </ul>
  <p class="graf graf--p">
   工具其实有很多，关键是您能在特定需要的条件下想起来用什么。建议最好记住几个
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/20%e4%b8%aa%e6%9c%80%e5%a5%bd%e7%94%a8%e7%9a%84%e5%9c%a8%e7%ba%bf%e4%be%a6%e6%9f%a5%e5%b7%a5%e5%85%b7%e5%92%8c%e6%95%b0%e6%8d%ae%e6%94%b6%e9%9b%86%e5%b7%a5%e5%85%b7/" href="https://www.iyouport.org/20%e4%b8%aa%e6%9c%80%e5%a5%bd%e7%94%a8%e7%9a%84%e5%9c%a8%e7%ba%bf%e4%be%a6%e6%9f%a5%e5%b7%a5%e5%85%b7%e5%92%8c%e6%95%b0%e6%8d%ae%e6%94%b6%e9%9b%86%e5%b7%a5%e5%85%b7/" rel="noopener noreferrer" target="_blank">
    比较全能
   </a>
   的工具，并同时记住一些
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%f0%9f%92%8e-%e8%b6%85%e7%ba%a7%e6%83%85%e6%8a%a5%e6%94%b6%e9%9b%86%e5%b7%a5%e5%85%b7%e5%ba%93%ef%bc%9a%e5%bc%80%e6%ba%90%e9%aa%8c%e8%af%81%e5%92%8c%e8%b0%83%e6%9f%a5%e5%b7%a5%e5%85%b7%e5%8f%8a/" href="https://www.iyouport.org/%f0%9f%92%8e-%e8%b6%85%e7%ba%a7%e6%83%85%e6%8a%a5%e6%94%b6%e9%9b%86%e5%b7%a5%e5%85%b7%e5%ba%93%ef%bc%9a%e5%bc%80%e6%ba%90%e9%aa%8c%e8%af%81%e5%92%8c%e8%b0%83%e6%9f%a5%e5%b7%a5%e5%85%b7%e5%8f%8a/" rel="noopener noreferrer" target="_blank">
    查询捷径
   </a>
   ，这样就能帮助您及时找到最适合的方法。
  </p>
  <p class="graf graf--p">
   本系列主要是介绍框架，它们表明当您从哪里开始的时候会遇到哪些路线。其实就是个经验问题，就如您看到牛排就能想到拿刀叉、看到拉面就能想到拿筷子，一样。多吃几次效果就会越来越好。⚪️
  </p>
