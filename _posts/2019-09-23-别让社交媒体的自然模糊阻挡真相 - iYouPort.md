  <p class="graf graf--p">
   从 Twitter 和 Instagram 等社交媒体捕获的图像可能是任何 OSINT 调查的重要部分，但是在 Web 浏览器中呈现给我们的图像很少与首次拍摄和上传时的图像质量相同的。
  </p>
  <p class="graf graf--p">
   不仅图像被剥离了对研究者非常有用的 EXIF 数据，与原始图像相比，它们的质量也通常会明显降低。
  </p>
  <p class="graf graf--p">
   这意味着网页加载速度更快，对普通使用来说不成问题，但是，
   <strong class="markup--strong markup--p-strong">
    如果您正在搜索照片以获取重要的调查细节，照片的分辨率将产生非常大的影响
   </strong>
   。
  </p>
  <p class="graf graf--p">
   本指南将介绍如何通过调整与照片相关的 URL 以帮助 Twitter 和 Instagram 上发布的图像恢复更高的分辨率（包括私人 Instagram 个人资料图片）。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Twitter
   </strong>
  </p>
  <p class="graf graf--p">
   这是一种非常简单的方法。在这个例子中，我将使用在阿姆斯特丹的 Hunzestraat 拍摄的照片：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4643 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1-2.jpeg?resize=1100%2C1467&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1-2.jpeg?w=1536&amp;ssl=1 1536w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1-2.jpeg?resize=225%2C300&amp;ssl=1 225w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1-2.jpeg?resize=768%2C1024&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1-2.jpeg?resize=1100%2C1467&amp;ssl=1 1100w" data-recalc-dims="1" height="1467" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1-2.jpeg?resize=1100%2C1467&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   上传高分辨率版本的图像后，Twitter 会在将其呈现给浏览器之前对其进行压缩，因此当您直接查看图像时，您看到的是默认使用 Twitter 认为的图像的“大”版本 —— 但这样不一定是最好的品质。
  </p>
  <p class="graf graf--p">
   如果你看一下 URL 就更清楚了。要获取 URL，请右键单击图像并选择“查看图像”，或将图像 URL 复制到新的浏览器选项卡中。
  </p>
  <p class="graf graf--p">
   这是上例中图像的默认 URL：
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://pbs.twimg.com/media/D6nWO_gXsAAJPUN.jpg:large" href="https://pbs.twimg.com/media/D6nWO_gXsAAJPUN.jpg:large" rel="nofollow noopener noreferrer" target="_blank">
    https://pbs.twimg.com/media/D6nWO_gXsAAJPUN.jpg:large
   </a>
  </p>
  <p class="graf graf--p">
   如果您双击此图像，就可以放大以获得更好的外观，还可以获得比这更详细的信息。
  </p>
  <p class="graf graf--p">
   请注意，默认情况下，Twitter会将 “large” 附加到.jpg扩展名的末尾，这是对所有图像的默认行为。
  </p>
  <p class="graf graf--p">
   但是，你可以调整 URL 并将“large”替换为“orig”，以获得最初上传的大小的照片链接：
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://pbs.twimg.com/media/D6nWO_gXsAAJPUN.jpg:orig" href="https://pbs.twimg.com/media/D6nWO_gXsAAJPUN.jpg:orig" rel="nofollow noopener noreferrer" target="_blank">
    https://pbs.twimg.com/media/D6nWO_gXsAAJPUN.jpg:orig
   </a>
  </p>
  <p class="graf graf--p">
   现在，当您双击放大图像时，将获得比以前更高的分辨率，这使得查找重要的小细节变得更加容易。
  </p>
  <p class="graf graf--p">
   如果您比较上面两个不同网址的结果，您可以看到差异。

    Instagram 图片

  <p class="graf graf--p">
   这个稍微复杂一点，但工作原理相同。
  </p>
  <p class="graf graf--p">
   Instagram 默认情况下会降低图像的分辨率，但通过识别 URL 并进行调整，可以获得更高分辨率的图像。
  </p>

  <p class="graf graf--p">
   Instagram 使用“og：image”作为 URL 的前缀，因此通过使用 Ctrl + F（Linux / Windows）或 Command + F（Mac），您可以快速搜索页面源代码以获取 og：image 标记并找到完整的 URL 图像。

  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4648 jetpack-lazy-image" data-lazy-sizes="(max-width: 581px) 100vw, 581px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/6.png?resize=581%2C342&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/6.png?w=581&amp;ssl=1 581w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/6.png?resize=300%2C177&amp;ssl=1 300w" data-recalc-dims="1" height="342" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/6.png?resize=581%2C342&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="581"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4648" data-recalc-dims="1" height="342" sizes="(max-width: 581px) 100vw, 581px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/6.png?resize=581%2C342&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/6.png?w=581&amp;ssl=1 581w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/6.png?resize=300%2C177&amp;ssl=1 300w" width="581"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   这是一个非常低质量的图像 — 只有150 x 150像素。在 ShitheadSteve 的情况下，如果要得到这个图像的放大版本可能就会看到更多的东西，但是当需要额外的细节时，技术完全相同。
  </p>
  <p class="graf graf--p">
   这种技术使用一种 URL 操作来访问 Instagram 为所有帐户生成的 JSON 文件 —— 甚至是私有帐户。
  </p>
  <p class="graf graf--p">
   在 JSON 文件中，有一个指向我们想要访问的全尺寸图像的链接，其方式与上一个 Instagram 示例类似。
  </p>
  <p class="graf graf--p">
   我们需要的 URL 如下所示：
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://i.instagram.com/api/v1/users/USERID/info/" href="https://i.instagram.com/api/v1/users/USERID/info/" rel="nofollow noopener noreferrer" target="_blank">
    https://i.instagram.com/api/v1/users/USERID/info/
   </a>
  </p>
  <p class="graf graf--p">
   只需要将“USERID”替换为 ShitheadSteve 的正确唯一用户ID。
  </p>
  <p class="graf graf--p">
   有几种方法可以做到这一点，但
   <a class="markup--anchor markup--p-anchor" data-href="https://codeofaninja.com/tools/find-instagram-user-id" href="https://codeofaninja.com/tools/find-instagram-user-id" rel="noopener noreferrer" target="_blank">
    CodeOfNinja
   </a>
   相比下是快速而简单的方法：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4649 jetpack-lazy-image" data-lazy-sizes="(max-width: 365px) 100vw, 365px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/7.png?resize=365%2C126&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/7.png?w=365&amp;ssl=1 365w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/7.png?resize=300%2C104&amp;ssl=1 300w" data-recalc-dims="1" height="126" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/7.png?resize=365%2C126&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="365"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4649" data-recalc-dims="1" height="126" sizes="(max-width: 365px) 100vw, 365px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/7.png?resize=365%2C126&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/7.png?w=365&amp;ssl=1 365w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/7.png?resize=300%2C104&amp;ssl=1 300w" width="365"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   很容易可以找到，ShitheadSteve 的 ID是 1364166629，所以现在我们可以将它添加到 URL 中以获取对 JSON 文件的访问权限：
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://i.instagram.com/api/v1/users/1364166629/info/" href="https://i.instagram.com/api/v1/users/1364166629/info/" rel="nofollow noopener noreferrer" target="_blank">
    https://i.instagram.com/api/v1/users/1364166629/info/
   </a>
  </p>
  <p class="graf graf--p">
   现在可以看到包含我们所追求的更高分辨率图像的 URL 的配置文件数据。在页面下方，至少有三个链接到该图片的不同分辨率版本：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4650 jetpack-lazy-image" data-lazy-sizes="(max-width: 341px) 100vw, 341px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/8-1.png?resize=341%2C255&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/8-1.png?w=341&amp;ssl=1 341w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/8-1.png?resize=300%2C224&amp;ssl=1 300w" data-recalc-dims="1" height="255" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/8-1.png?resize=341%2C255&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="341"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4650" data-recalc-dims="1" height="255" sizes="(max-width: 341px) 100vw, 341px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/8-1.png?resize=341%2C255&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/8-1.png?w=341&amp;ssl=1 341w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/8-1.png?resize=300%2C224&amp;ssl=1 300w" width="341"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   你可以看到320×320、640×640和1080×1080版本的个人资料图片的链接 —— 都比 Instagram 默认提供的150×150版本要好得多。
  </p>
  <p class="graf graf--p">
   这是一个私人账户也不要紧。只需单击 URL 即可查看图像。在这种情况下，可以在此处查看最高分辨率版本：
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://instagram.fman3-1.fna.fbcdn.net/vp/10d2eee8cc1bd4bdb2b5e4e6e8b6fd87/5D9CBE3A/t51.2885-19/20686663_183184438888795_924418902178398208_a.jpg?_nc_ht=instagram.fman3-1.fna.fbcdn.net" href="https://instagram.fman3-1.fna.fbcdn.net/vp/10d2eee8cc1bd4bdb2b5e4e6e8b6fd87/5D9CBE3A/t51.2885-19/20686663_183184438888795_924418902178398208_a.jpg?_nc_ht=instagram.fman3-1.fna.fbcdn.net" rel="nofollow noopener noreferrer" target="_blank">

   </a>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果您需要了解图片背景中的细节，或者想要近距离查看刚刚向您发送关注请求的陌生人，这种方法对您来说会非常容易。
   </strong>
