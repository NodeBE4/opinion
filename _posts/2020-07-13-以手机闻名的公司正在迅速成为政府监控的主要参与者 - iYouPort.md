  <p class="graf graf--p">
   形形色色的
   <a class="markup--anchor markup--p-anchor" data-href="https://start.me/p/RMQrDD/iyp-2" href="https://start.me/p/RMQrDD/iyp-2" rel="noopener noreferrer" target="_blank">
    监视工具
   </a>
   已安装在学校和公共房屋中、部署在街道和公交设施上，并由
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ba%ab%e4%bd%93%e7%9b%b8%e6%9c%ba%e9%80%82%e5%be%97%e5%85%b6%e5%8f%8d%e5%90%97%ef%bc%9f%e4%b8%ad%e5%9b%bd%e6%ad%a3%e5%9c%a8%e5%bc%80%e5%a7%8b%e6%b5%81%e8%a1%8c%e8%bf%99%e7%a7%8d%e4%b8%9c%e8%a5%bf/" href="https://www.iyouport.org/%e8%ba%ab%e4%bd%93%e7%9b%b8%e6%9c%ba%e9%80%82%e5%be%97%e5%85%b6%e5%8f%8d%e5%90%97%ef%bc%9f%e4%b8%ad%e5%9b%bd%e6%ad%a3%e5%9c%a8%e5%bc%80%e5%a7%8b%e6%b5%81%e8%a1%8c%e8%bf%99%e7%a7%8d%e4%b8%9c%e8%a5%bf/" rel="noopener noreferrer" target="_blank">
    警察佩戴在身上
   </a>
   。
  </p>
  <p class="graf graf--p">
   它们是由一系列 **竞争政府业务** 的技术公司开发的。是的，不仅是中国，全球都是如此。因为
   <strong class="markup--strong markup--p-strong">
    政府对每时每刻监视全社会具有浓厚的兴趣
   </strong>
   。
  </p>
  <p class="graf graf--p">
   而现在，许多这类技术都集中到了一家寻求在蓬勃发展的监视市场中占据更大份额的公司所有。
  </p>
  <p class="graf graf--p">
   摩托罗拉（Motorola）这个通常与手机和警用无线电相关的品牌，已经加入了监视技术公司之间的竞争，以提供监控公众的新方法。
  </p>
  <p class="graf graf--p">
   自2017年以来，总部位于芝加哥的科技公司（现称为 Motorola Solutions，在摩托罗拉公司拆分了其手机业务之后）已
   <strong class="markup--strong markup--p-strong">
    投资17亿美元
   </strong>
   ，以支持或收购制造警用监控摄像头的公司，训练相机发现某些面孔或行为、通过视频筛选可疑人员、并通过车牌跟踪汽车的行驶。
  </p>
  <p class="graf graf--p">
   通过将这些监视工具整合到一个公司中、并有可能将它们组合到一个产品中，
   <strong class="markup--strong markup--p-strong">
    Motorola Solutions 正在提高其在监视行业中的地位 —— 并加剧了人们对政府日益强大的权力来监视人们一举一动的担忧
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “当您的信息分散在很多代理商和实体之间时，您的隐私相比下将得到更多保护”，美国公民自由联盟的高级政策分析师 Jay Stanley 说，他在那里研究技术对隐私的影响，“但是，我不知道像整合 Motorola Solutions 这样的公司会产生什么样的协同效应，但总的来说前景不容乐观。”
  </p>
  <figure aria-describedby="caption-attachment-12574" class="wp-caption aligncenter" id="attachment_12574" style="width: 560px">
   <img alt="" class="size-full wp-image-12574 jetpack-lazy-image" data-lazy-sizes="(max-width: 560px) 100vw, 560px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-10.jpg?resize=560%2C338&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-10.jpg?w=560&amp;ssl=1 560w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-10.jpg?resize=300%2C181&amp;ssl=1 300w" data-recalc-dims="1" height="338" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/1-10.jpg?resize=560%2C338&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="560"/>

  <p class="graf graf--p">
   摩托罗系统公司没有让高管回答有关收购或收购计划的问题。该公司发表了一份声明，描述了一项计划，即 向出售给公共安全[维稳]机构的工具包中添加包括对象检测和 “异常运动检测” 在内的人工智能产品的计划。
  </p>
  <p class="graf graf--p">
   该公司表示，该系统可以帮助标记潜在的侵入者或建筑物冒烟的外观状况。该公司强调说，新工具并不是要做出自动警务决策，而是要 “帮助官员决定如何采取行动”。
  </p>
  <p class="graf graf--p">
   两年前，Stanley 在国际警察总长协会年会上首先注意到了 Motorola Solutions 的新方向。在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.aclu.org/blog/privacy-technology/surveillance-technologies/look-high-tech-gadgets-being-marketed-police" href="https://www.aclu.org/blog/privacy-technology/surveillance-technologies/look-high-tech-gadgets-being-marketed-police" rel="noopener noreferrer" target="_blank">
    博客文章中
   </a>
   ，他包括了摩托罗拉解决方案展台的照片，该展台推广了其新的警用监视摄像机系列以及分析视频的新方法。
  </p>
  <p class="graf graf--p">
   摩托罗拉系统公司进入高科技监控领域的最初并没有引起隐私研究人员的严格审查。但是随着该公司继续组装强大的监视工具，情况正在发生变化。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这些监视技术包括
   </strong>
   ：
  </p>
  <p class="graf graf--p">
   1、识别人的警用身体监视摄像机：2017年，摩托罗拉解决方案
   <a class="markup--anchor markup--p-anchor" data-href="https://newsroom.motorolasolutions.com/news/neurala-announces-14-million-series-to-expand-market-for-nasa-tested-ai-for-drones-self-driving-cars-toys-and-cameras.htm" href="https://newsroom.motorolasolutions.com/news/neurala-announces-14-million-series-to-expand-market-for-nasa-tested-ai-for-drones-self-driving-cars-toys-and-cameras.htm" rel="noopener noreferrer" target="_blank">
    投资
   </a>
   并与一家名为 Neurala 的波士顿人工智能创业公司
   <a class="markup--anchor markup--p-anchor" data-href="https://newsroom.motorolasolutions.com/news/ai-at-edge-motorola-solutions-and-neurala-to-work-together-on-intelligence-for-cameras.htm" href="https://newsroom.motorolasolutions.com/news/ai-at-edge-motorola-solutions-and-neurala-to-work-together-on-intelligence-for-cameras.htm" rel="noopener noreferrer" target="_blank">
    合作
   </a>
   ，开发了可以根据其外观
   <strong class="markup--strong markup--p-strong">
    实时搜索
   </strong>
   人或物体的警用摄像机。
  </p>
  <p class="graf graf--p">
   摩托罗拉说，这种合作关系仅限于 “早期技术可行性研究、原型和客户研究”。
  </p>
  <p class="graf graf--p">
   到2019年7月，摩托罗拉又收购了得克萨斯州
   <strong class="markup--strong markup--p-strong">
    身体摄像机
   </strong>
   制造商
   <a class="markup--anchor markup--p-anchor" data-href="https://newsroom.motorolasolutions.com/news/motorola-solutions-acquires-watchguard-inc-leader-in-mobile-video-for-public-safety.htm" href="https://newsroom.motorolasolutions.com/news/motorola-solutions-acquires-watchguard-inc-leader-in-mobile-video-for-public-safety.htm" rel="noopener noreferrer" target="_blank">
    WatchGuard
   </a>
   ，这将有助于其增加现有的身体摄像机产品线。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    不要错过《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%ba%ab%e4%bd%93%e7%9b%b8%e6%9c%ba%e9%80%82%e5%be%97%e5%85%b6%e5%8f%8d%e5%90%97%ef%bc%9f%e4%b8%ad%e5%9b%bd%e6%ad%a3%e5%9c%a8%e5%bc%80%e5%a7%8b%e6%b5%81%e8%a1%8c%e8%bf%99%e7%a7%8d%e4%b8%9c%e8%a5%bf/" href="https://www.iyouport.org/%e8%ba%ab%e4%bd%93%e7%9b%b8%e6%9c%ba%e9%80%82%e5%be%97%e5%85%b6%e5%8f%8d%e5%90%97%ef%bc%9f%e4%b8%ad%e5%9b%bd%e6%ad%a3%e5%9c%a8%e5%bc%80%e5%a7%8b%e6%b5%81%e8%a1%8c%e8%bf%99%e7%a7%8d%e4%b8%9c%e8%a5%bf/" rel="noopener noreferrer" target="_blank">
     身体相机适得其反吗？中国正在开始流行这种东西
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   2、可以跟踪人的运动的监控摄像机：2018年3月，摩托罗拉
   <a class="markup--anchor markup--p-anchor" data-href="https://newsroom.motorolasolutions.com/news/motorola-solutions-completes-acquisition-avigilon.htm" href="https://newsroom.motorolasolutions.com/news/motorola-solutions-completes-acquisition-avigilon.htm" rel="noopener noreferrer" target="_blank">
    收购
   </a>
   了一家加拿大公司 Avigilon，该公司出售监控摄像头以及用于
   <a class="markup--anchor markup--p-anchor" data-href="http://news.avigilon.com/News-Releases/News-Release-Details/2018/Avigilon-Adds-Gender-and-Age-to-Avigilon-Appearance-Search-Technology-/default.aspx" href="http://news.avigilon.com/News-Releases/News-Release-Details/2018/Avigilon-Adds-Gender-and-Age-to-Avigilon-Appearance-Search-Technology-/default.aspx" rel="noopener noreferrer" target="_blank">
    扫描
   </a>
   特定人员或车辆的监控软件。
  </p>
  <p class="graf graf--p">
   Avigilon 已签订合同，向包括田纳西州威尔逊在内的多个学区提供这些人工智能驱动的监视应用，包括乔治亚州富尔顿、和佛罗里达州布劳沃德。
  </p>
  <p class="graf graf--p">
   根据 Avigilon 的宣传材料、公共合同文件、以及对当地政府官员的采访，其客户还包括马萨诸塞州的新贝德福德住房管理局、加利福尼亚州的埃尔克格罗夫警察局、和加利福尼亚州伯克利市。
  </p>
  <p class="graf graf--p">
   3、自动车牌读取器：去年1月，摩托罗拉系统公司
   <a class="markup--anchor markup--p-anchor" data-href="https://newsroom.motorolasolutions.com/news/motorola-solutions-acquires-vaas-international-holdings-leader-in-data-and-image-analytics-for-vehicle-location.htm" href="https://newsroom.motorolasolutions.com/news/motorola-solutions-acquires-vaas-international-holdings-leader-in-data-and-image-analytics-for-vehicle-location.htm" rel="noopener noreferrer" target="_blank">
    收购了
   </a>
   位于加利福尼亚的 Vigilant Solutions 所有的 VaaS International Holdings，
   <strong class="markup--strong markup--p-strong">
    该组织与政府机构合作
   </strong>
   ，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.eff.org/pages/what-we-learned" href="https://www.eff.org/pages/what-we-learned" rel="noopener noreferrer" target="_blank">
    包括
   </a>
   美国数百个警察局
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nbcnews.com/tech/security/aclu-calls-u-s-law-enforcement-stop-sharing-license-plate-n983021" href="https://www.nbcnews.com/tech/security/aclu-calls-u-s-law-enforcement-stop-sharing-license-plate-n983021" rel="noopener noreferrer" target="_blank">
    以及
   </a>
   臭名昭著的移民和海关执法局，以收集和共享有关自动车牌读取器捕获的汽车位置数据。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    不要错过《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e6%84%9a%e5%bc%84%e8%87%aa%e5%8a%a8%e8%bd%a6%e7%89%8c%e9%98%85%e8%af%bb%e5%99%a8%ef%bc%9f/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e6%84%9a%e5%bc%84%e8%87%aa%e5%8a%a8%e8%bd%a6%e7%89%8c%e9%98%85%e8%af%bb%e5%99%a8%ef%bc%9f/" rel="noopener noreferrer" target="_blank">

  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   在去年8月1日与股票市场分析师进行的
   <a class="markup--anchor markup--p-anchor" data-href="http://investors.motorolasolutions.com/Cache/1001255394.PDF?O=PDF&amp;T=&amp;Y=&amp;D=&amp;FID=1001255394&amp;iid=4280272" href="http://investors.motorolasolutions.com/Cache/1001255394.PDF?O=PDF&amp;T=&amp;Y=&amp;D=&amp;FID=1001255394&amp;iid=4280272" rel="noopener noreferrer" target="_blank">
    电话会议
   </a>
   中，摩托罗拉董事长兼首席执行官 Greg Brown 描述了他的公司如何将这些技术结合到产品套件中，
   <strong class="markup--strong markup--p-strong">
    特别关注警察和学校的使用
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们正在建立平台，”Brown 说，“不只是视频 ……这是存储、管理、分析、机器学习和AI的集成，利用所有这些端到端经验，并提供了针对特定行业的用例进行的区分。这是一个热门市场。”
  </p>
  <p class="graf graf--p">
   摩托罗拉并非独有这种策略。包括亚马逊、微软和 IBM 在内的监控市场上更大的参与者已经扩大了在人工智能公司的股份。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    更多详细报告见
    <a class="markup--anchor markup--li-anchor" data-href="https://start.me/p/RMQrDD/iyp-2" href="https://start.me/p/RMQrDD/iyp-2" rel="noopener noreferrer" target="_blank">
     列表-2
    </a>
    <strong class="markup--strong markup--li-strong">
     “公私监控伙伴关系” 和 “老大哥的真面目”
    </strong>
   </li>
  </ul>
  <p class="graf graf--p">
   不仅是寡头。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.policeone.com/police-products/less-lethal/TASER/articles/289203006-What-TASERs-acquisition-of-2-AI-companies-means-for-the-future-of-policing/" href="https://www.policeone.com/police-products/less-lethal/TASER/articles/289203006-What-TASERs-acquisition-of-2-AI-companies-means-for-the-future-of-policing/" rel="noopener noreferrer" target="_blank">
    像 Axon
   </a>
   （以前称为 Taser）这样的新兴竞争对手也是如此，该公司已成为美国最大的警用摄像头提供商，并正在探索人工智能的新用途。
  </p>
  <p class="graf graf--p">
   公立学校是发展最快的监视市场之一。摩托罗拉利用了这一趋势，在过去两年中，Avigilon 赢得了大约十二份安装最新视频监控网络的合同。
  </p>
  <p class="graf graf--p">
   买家包括田纳西州威尔逊的学校，他们为两所最新的学校配备了摄像头，这些摄像头可以根据人们的外观自动跟踪他们穿过房间和走廊的情况。
  </p>
  <figure aria-describedby="caption-attachment-12575" class="wp-caption aligncenter" id="attachment_12575" style="width: 560px">
   <img alt="" class="size-full wp-image-12575 jetpack-lazy-image" data-lazy-sizes="(max-width: 560px) 100vw, 560px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-11.jpg?resize=560%2C235&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-11.jpg?w=560&amp;ssl=1 560w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-11.jpg?resize=300%2C126&amp;ssl=1 300w" data-recalc-dims="1" height="235" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-11.jpg?resize=560%2C235&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="560"/>
   <noscript>
    <img alt="" class="size-full wp-image-12575" data-recalc-dims="1" height="235" sizes="(max-width: 560px) 100vw, 560px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-11.jpg?resize=560%2C235&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-11.jpg?w=560&amp;ssl=1 560w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-11.jpg?resize=300%2C126&amp;ssl=1 300w" width="560"/>

    Avigilon Appearance Search technology allows users to search for a person by selecting physical descriptions, including hair and clothing color, gender and age.Avigilon
   </figcaption>
  </figure>
  <p class="graf graf--p">
   威尔逊学校副主任 Mickey Hall 表示：下一步学区的监视系统可能包括读取所有驶过的汽车的车牌，并通过人们的面孔识别行人的系统。
  </p>
  <p class="graf graf--p">
   但是，无疑，这项技术也增加了政府和私人公司积累过多权力以影响人们生活的风险，美国公民自由联盟的 Stanley 说：一旦将某人作为此类监视的目标，无论他们是犯罪嫌疑人、持不同政见者、还是被误认的其他人，当局都可以检查他们去过的所有地方，并可能发现有罪的行为。
  </p>
  <p class="graf graf--p">
   他说：
   <strong class="markup--strong markup--p-strong">
    “这是将分散的摄像机变成了大规模监视机器，使公司和政府间谍可以使用它们来重现任何人的生活。”
   </strong>
  </p>
  <p class="graf graf--p">
   Northcoast Research 跟踪摩托罗拉系统公司的分析师 Keith Housum 表示，最近与监视相关的收购仅占公司业务的一小部分，但由于投资者将其视为新的收入来源，因此有助于提振股价。
  </p>
  <p class="graf graf--p">
   Housum 补充说：“人工智能对他们的产品组合非常重要，因为收集了如此多的信息、记录了如此多的视频，所以您不可能用肉眼浏览它们。”
  </p>
  <p class="graf graf--p">
   非营利组织电子前沿基金会的资深调查研究员 Dave Maass 提倡限制政府监控，并
   <a class="markup--anchor markup--p-anchor" data-href="https://www.eff.org/pages/what-we-learned" href="https://www.eff.org/pages/what-we-learned" rel="noopener noreferrer" target="_blank">
    记录了
   </a>
   执法部门与 Vigilant Solutions 的车牌阅读器所收集数据的广泛共享，他表示，他希望看到更多的地方政府能通过要求对该技术进行更深入公众审查的法令。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “所有这些技术的融合突显出监视是一项大生意，” Maass 在电子邮件中说，“重要的是要确保我们社会中有关监视的决策不受销售和利润的驱动，而是受社区实际需求的驱动。” ⚪️
  </p>

    Motorola, known for cellphones, is fast becoming a major player in government surveillance and artificial intelligence
