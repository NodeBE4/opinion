  <p class="graf graf--p">
   Mr. Robot 第5集中出现了欺骗性短信的操作。
  </p>
  <p class="graf graf--p">
   当 Elliot 能够以社交工程的方式进入 Steel Mountain 最先进的 “难以穿透的” 存储设施时，一位负责人开始产生怀疑，并将他赶了出去，那时他还没来得及植入树莓派。
  </p>
  <p class="graf graf--p">
   他打算将树莓派放置在目标网络内部，以操纵 HVAC 系统提高存储设施的温度，以此来销毁包含世界上70％的消费者债务的记录。
  </p>
  <p class="graf graf--p">
   在那位管理人员护送 Elliot 到电梯并准备离开该设施的那一刻，她收到了她丈夫的发来的短信，很紧急，就这样分散了她的注意力。
  </p>
  <p class="graf graf--p">
   这条短信实际上并不是她丈夫发来的，而是来自 Elliot 的去社会黑客组织的其他成员。
  </p>

   在剧情中，Elliot 的去社会组织成员使用的是 Kali 来发送欺骗性短信，但是很可惜，此功能在最新版本的 Kali 中已停止使用。
  </p>
  <p class="graf graf--p">
   幸运的是它仍然位于 BackTrack 中，因此本教程将恢复到值得信赖的 BackTrack 。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    步骤1：启动 BackTrack 并启动社交工程工具包（SET）
   </strong>
  </h3>
  <p class="graf graf--p">
   首先启动 Backtrack 5，然后导航到 “应用程序”-&gt;“开发者工具”-&gt;“社交工程工具”-&gt;“社交工程工具包”（SET），然后选择 “设置”，如下面的屏幕快照所示。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13333 jetpack-lazy-image" data-lazy-sizes="(max-width: 736px) 100vw, 736px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-1.jpg?resize=736%2C476&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-1.jpg?w=736&amp;ssl=1 736w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-1.jpg?resize=300%2C194&amp;ssl=1 300w" data-recalc-dims="1" height="476" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-1.jpg?resize=736%2C476&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="736"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13333" data-recalc-dims="1" height="476" sizes="(max-width: 736px) 100vw, 736px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-1.jpg?resize=736%2C476&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-1.jpg?w=736&amp;ssl=1 736w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/3-1.jpg?resize=300%2C194&amp;ssl=1 300w" width="736"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   这将启动 SET 打开屏幕，如下所示。 SET 能够进行多种社交工程攻击。
  </p>
  <p class="graf graf--p">
   null-byte 以前的介绍曾使用社交工程工具包在 BackTrack 中投放网络钓鱼，但这次我们想要的是 “ SMS Spoofing Attack Vector”。要开始此攻击，请选择＃7。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13334 jetpack-lazy-image" data-lazy-sizes="(max-width: 547px) 100vw, 547px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4-2.jpg?resize=547%2C409&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4-2.jpg?w=547&amp;ssl=1 547w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4-2.jpg?resize=300%2C224&amp;ssl=1 300w" data-recalc-dims="1" height="409" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4-2.jpg?resize=547%2C409&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="547"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13334" data-recalc-dims="1" height="409" sizes="(max-width: 547px) 100vw, 547px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4-2.jpg?resize=547%2C409&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4-2.jpg?w=547&amp;ssl=1 547w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4-2.jpg?resize=300%2C224&amp;ssl=1 300w" width="547"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在下面的屏幕中，询问您是否要 “执行SMS欺骗攻击” 或者是 “创建社交工程模板”。选择＃1。然后它会询问您是要欺骗单个电话号码还是进行大规模攻击。这种情况下是有针对目标的，于是选择＃1 “单一电话号码”。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-13335 jetpack-lazy-image" data-lazy-sizes="(max-width: 612px) 100vw, 612px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/5-3.jpg?resize=612%2C398&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/5-3.jpg?w=612&amp;ssl=1 612w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/5-3.jpg?resize=300%2C195&amp;ssl=1 300w" data-recalc-dims="1" height="398" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/5-3.jpg?resize=612%2C398&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="612"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13335" data-recalc-dims="1" height="398" sizes="(max-width: 612px) 100vw, 612px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/5-3.jpg?resize=612%2C398&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/5-3.jpg?w=612&amp;ssl=1 612w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/5-3.jpg?resize=300%2C195&amp;ssl=1 300w" width="612"/>
   </noscript>
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    步骤2：设置欺骗性短信
   </strong>
  </h3>
  <p class="graf graf--p">
   在此示例中假装 Mary 向 John 发送了一条欺骗性短信以表示她要和他分手。这应该会使 John 感到不安，因为他疯狂地爱着她。
  </p>
  <p class="graf graf--p">
   首先，在 “Send sms to” 选项中您需要输入 John 的手机号码。然后选择＃2制作一次性使用的 SMS。最后，输入 Mary 的电话号码。确保两个号码都以 “ +” 开头。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13336 jetpack-lazy-image" data-lazy-sizes="(max-width: 543px) 100vw, 543px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/6.jpg?resize=543%2C146&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/6.jpg?w=543&amp;ssl=1 543w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/6.jpg?resize=300%2C81&amp;ssl=1 300w" data-recalc-dims="1" height="146" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/6.jpg?resize=543%2C146&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="543"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13336" data-recalc-dims="1" height="146" sizes="(max-width: 543px) 100vw, 543px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/6.jpg?resize=543%2C146&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/6.jpg?w=543&amp;ssl=1 543w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/6.jpg?resize=300%2C81&amp;ssl=1 300w" width="543"/>
   </noscript>
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    步骤3：制作文字讯息
   </strong>
  </h3>
  <p class="graf graf--p">
   也就是输入*假的* Mary 发送给 John 的短信文本。为此目的，输入如下：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “对不起，John。我遇见了另一个男人，他是我一生的挚爱。我希望我们俩能继续做朋友”。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-13337 jetpack-lazy-image" data-lazy-sizes="(max-width: 565px) 100vw, 565px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/7-2.jpg?resize=565%2C47&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/7-2.jpg?w=565&amp;ssl=1 565w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/7-2.jpg?resize=300%2C25&amp;ssl=1 300w" data-recalc-dims="1" height="47" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/7-2.jpg?resize=565%2C47&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="565"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13337" data-recalc-dims="1" height="47" sizes="(max-width: 565px) 100vw, 565px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/7-2.jpg?resize=565%2C47&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/7-2.jpg?w=565&amp;ssl=1 565w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/7-2.jpg?resize=300%2C25&amp;ssl=1 300w" width="565"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   输入完毕后，请按 Control + C 退出。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    步骤4：发送消息！
   </strong>
  </h3>
  <p class="graf graf--p">
   这将带您进入最终屏幕。在此屏幕中，您将需要为欺骗性短信消息选择媒介。在这里有4个选项。
  </p>
  <p class="graf graf--p">
   第一个是免费的，正如他们所说，它是 “buggy” 有点问题（当我运行它时，SET崩溃了）。然后，有两个付费选项，最后是 Android 模拟器。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13338 jetpack-lazy-image" data-lazy-sizes="(max-width: 559px) 100vw, 559px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/8.jpg?resize=559%2C351&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/8.jpg?w=559&amp;ssl=1 559w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/8.jpg?resize=300%2C188&amp;ssl=1 300w" data-recalc-dims="1" height="351" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/8.jpg?resize=559%2C351&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="559"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13338" data-recalc-dims="1" height="351" sizes="(max-width: 559px) 100vw, 559px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/8.jpg?resize=559%2C351&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/8.jpg?w=559&amp;ssl=1 559w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/8.jpg?resize=300%2C188&amp;ssl=1 300w" width="559"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   我选择了第三个选项 SMSGANG。他们为5条短信收取3欧元的费用，或每条约0.65美元。
  </p>
  <p class="graf graf--p">
   当您付款时（他们接受信用卡和 PayPal），他们会向您发送PIN码。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    肯定不会希望使用自己的信用卡干这种事对吧，所以不要错过《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/5%e4%b8%aa%e4%bf%a1%e8%aa%89%e8%89%af%e5%a5%bd%e7%9a%84%e4%b8%80%e6%ac%a1%e6%80%a7%e4%bf%a1%e7%94%a8%e5%8d%a1%e5%8f%b7%e6%9c%8d%e5%8a%a1%ef%bc%9a%e9%9a%90%e8%97%8f%e6%82%a8%e7%9a%84%e7%9c%9f%e5%ae%9e/" href="https://www.iyouport.org/5%e4%b8%aa%e4%bf%a1%e8%aa%89%e8%89%af%e5%a5%bd%e7%9a%84%e4%b8%80%e6%ac%a1%e6%80%a7%e4%bf%a1%e7%94%a8%e5%8d%a1%e5%8f%b7%e6%9c%8d%e5%8a%a1%ef%bc%9a%e9%9a%90%e8%97%8f%e6%82%a8%e7%9a%84%e7%9c%9f%e5%ae%9e/" rel="noopener noreferrer" target="_blank">
     5个信誉良好的一次性信用卡号服务：隐藏您的真实身份信息
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   选择＃3后，它将要求您输入“Pin码”。输入一个通过电子邮件发送给您的验证码，然后您就可以发送这个欺骗性短信了。
  </p>
  <p class="graf graf--p">
   就是这么简单！您学会了吗？
  </p>
  <p class="graf graf--p">
   👉 欺骗性短信本身不至于仅仅成为恶作剧的唯一方式是：它必须在一个逻辑完美的行动策划案过程中发挥作用。所以，您应该首先设计全套策划案，并按需求植入相关技术。好运！⚪️
