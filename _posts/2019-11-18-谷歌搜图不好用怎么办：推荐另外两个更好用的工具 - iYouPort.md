  <p class="graf graf--p">
   反向图像搜索是任何深度调查和信息验证的重要部分。
  </p>
  <p class="graf graf--p">
   有数十种专业搜索引擎可供选择，每种搜索引擎都有自己的优点和缺点，但是使用多个独立的搜索引擎逐一检查图像会非常耗时。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    搜索引擎大全《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e4%bd%a0%e6%93%85%e9%95%bf%e6%90%9c%e7%b4%a2%ef%bc%8c%e4%bd%a0%e8%83%bd%e6%89%be%e5%88%b0%e4%b8%80%e5%88%87%ef%bc%9a%e6%b2%a1%e6%9c%89%e8%b0%b7%e6%ad%8c%e7%9a%84%e5%bc%ba%e5%a4%a7/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e4%bd%a0%e6%93%85%e9%95%bf%e6%90%9c%e7%b4%a2%ef%bc%8c%e4%bd%a0%e8%83%bd%e6%89%be%e5%88%b0%e4%b8%80%e5%88%87%ef%bc%9a%e6%b2%a1%e6%9c%89%e8%b0%b7%e6%ad%8c%e7%9a%84%e5%bc%ba%e5%a4%a7/" rel="noopener noreferrer" target="_blank">
     如果你擅长搜索，你能找到一切：*没有谷歌*的强大搜索世界
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   令人沮丧的是，许多 RI 工具只允许您选取一两个搜索引擎，如 Google（有明显限制）和 Tineye（撞大运机器……）。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    还有另外两个很棒的工具，可以让您通过大量的自定义搜索囊括很多个搜索引擎
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://addons.mozilla.org/en-GB/firefox/addon/image-reverse-search/?src=search" href="https://addons.mozilla.org/en-GB/firefox/addon/image-reverse-search/?src=search" rel="noopener noreferrer" target="_blank">
    Reverse Image Search
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://github.com/dessant/search-by-image/blob/master/README.md" href="https://github.com/dessant/search-by-image/blob/master/README.md" rel="noopener noreferrer" target="_blank">
    Search By Image
   </a>
   是我最喜欢的两个。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Reverse Image Search
   </strong>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://github.com/Brawl345/Image-Reverse-Search-WebExtension" href="https://github.com/Brawl345/Image-Reverse-Search-WebExtension" rel="noopener noreferrer" target="_blank">
    Andreas Bielawski 的反向图像搜索工具
   </a>
   适用于 Chrome 和 Firefox，是一个很简单但非常有效的工具。
  </p>
  <p class="graf graf--p">
   安装完成后，只需右键单击图像，RIS 即可让您选择要用于对所选图像进行反向搜索的搜索引擎。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-7094 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-26.png?resize=1100%2C757&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-26.png?w=1104&amp;ssl=1 1104w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-26.png?resize=300%2C207&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-26.png?resize=768%2C529&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-26.png?resize=1024%2C705&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-26.png?resize=1100%2C757&amp;ssl=1 1100w" data-recalc-dims="1" height="757" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-26.png?resize=1100%2C757&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   最常见的搜索引擎几乎全包括在内，不仅有通常的谷歌和 Yandex，还有更多的利基搜索引擎，如动漫专家工具 IQDB。
  </p>
  <p class="graf graf--p">
   我最喜欢 RIS 的功能是它允许你添加任意数量的自定义搜索引擎，这样你就可以创建一个真正的瑞士军刀，用于快速反向图像搜索。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-7095 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-23.png?resize=1024%2C643&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-23.png?w=1024&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-23.png?resize=300%2C188&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-23.png?resize=768%2C482&amp;ssl=1 768w" data-recalc-dims="1" height="643" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/2-23.png?resize=1024%2C643&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>

  <p class="graf graf--p">
   该工具仍在积极开发中，目前的测试效果是完全可靠的。
  </p>
  <p class="graf graf--p">
   Firefox 版的
   <a class="markup--anchor markup--p-anchor" data-href="https://addons.mozilla.org/en-GB/firefox/addon/image-reverse-search/?src=search" href="https://addons.mozilla.org/en-GB/firefox/addon/image-reverse-search/?src=search" rel="noopener noreferrer" target="_blank">
    在这里
   </a>
   获取；Chrome 的请按照
   <a class="markup--anchor markup--p-anchor" data-href="https://github.com/Brawl345/Image-Reverse-Search-WebExtension" href="https://github.com/Brawl345/Image-Reverse-Search-WebExtension" rel="noopener noreferrer" target="_blank">
    此处
   </a>
   的说明操作。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Search By Image
   </strong>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://github.com/dessant/search-by-image/blob/master/README.md" href="https://github.com/dessant/search-by-image/blob/master/README.md" rel="noopener noreferrer" target="_blank">
    Search By Image
   </a>
   是荷兰的开源调查专家推荐的。到目前为止，它一直令我印象深刻，它正在慢慢成为我的首选工具。
  </p>
  <p class="graf graf--p">
   与RIS一样，它可以为一键式查询添加自定义范围的多种不同搜索引擎，并且它还具有一些额外的功能，可以给它带来很多的优势。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-7096 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-22.png?resize=1100%2C1307&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-22.png?w=1192&amp;ssl=1 1192w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-22.png?resize=253%2C300&amp;ssl=1 253w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-22.png?resize=768%2C912&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-22.png?resize=862%2C1024&amp;ssl=1 862w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-22.png?resize=1100%2C1307&amp;ssl=1 1100w" data-recalc-dims="1" height="1307" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-22.png?resize=1100%2C1307&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  </p>
  <p class="graf graf--p">
   通过右键单击图片然后选择要使用的搜索引擎（或者如果您愿意，可以选择所有搜索引擎）来查询图像：

   并且，我最喜欢的功能是，它能够捕获图像的一小块区域，然后对该特定区域进行反向图像搜索。
  </p>
  <p class="graf graf--p">
   当您想要关注特定细节时，这是一种很常用的RI技术，但通常涉及手动剪切图像然后将特定片段上传到搜索引擎。
  </p>
  <p class="graf graf--p">
   但使用 Search By Image 您可以非常简单地做到这点，而无需其他工具协助。这是一个非常有用的功能。
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   不得不说我对“Search By Image”印象非常深刻，虽然它仍在开发中。长期测试显示，Search By Image 的有效频率最高。
