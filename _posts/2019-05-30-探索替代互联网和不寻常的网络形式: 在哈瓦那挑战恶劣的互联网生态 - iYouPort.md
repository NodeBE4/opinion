     <em>
      哈瓦那的状况比中国严重一万倍。但哈瓦那人比中国人更会想办法，更积极努力去改变这种痛苦的局面，而不是坐以待毙。当中国社会单纯地埋冤网络封锁并心安理得地享受微信时，我们的故事至少能让某些人惭愧一分种
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   根据
   <a class="markup--anchor markup--p-anchor" data-href="http://www.itu.int/en/ITU-D/Statistics/Pages/stat/default.aspx" href="http://www.itu.int/en/ITU-D/Statistics/Pages/stat/default.aspx" rel="noopener noreferrer" target="_blank">
    国际电联的数据
   </a>
   ，有 1100 万固定互联网宽带连接（0,007％）、涉及 1,100 万古巴公民，主要是为政府机构和极端特权的个人保留的，而不是普通人。在 5,6％ 的家庭中，只有极少享有特权的人很幸运能够以大约 4–5 Kbit /s 的速度访问一个痛苦而缓慢的、政府控制的拨号互联网连接。
  </p>
  <p class="graf graf--p">
   2015年，国有
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/ETECSA" href="https://en.wikipedia.org/wiki/ETECSA" rel="noopener noreferrer" target="_blank">
    ETECSA
   </a>
   开放了35个公共 WiFi 点，古巴人可以上网了，但是每小时需要2个CUC（约2美元），这可是相当于该国社会每月平均 17 美元收入的大约10％（工作人员的平均收入 即 大多数人口）。从理论上讲，如果一个古巴公民有幸住在这些 WiFi 热点附近，并且想要在一个月内通过互联网向世界分享自己的想法，那么他们需要支付的就是 1440 CUC，网速大约可达到如罗马尼亚布加勒斯特的公民支付的 25 Mbits/s 。也就是说，
   <strong class="markup--strong markup--p-strong">
    古巴人需要工作大约6年的时间，才能连续一个月内使用这个 WiFi 上网
   </strong>
   。相比在中国买房的感觉如何？
  </p>

  <p class="graf graf--p">
   可以推测，这可能不是一个共产主义国家政府拥有的垄断所做出的最好的社会主义实践的例子，但这不是本故事的主题。
  </p>
  <p class="graf graf--p">
   我们感兴趣的是
   <strong class="markup--strong markup--p-strong">
    网络和内容分发的替代形式
   </strong>
   ，出现在集中式的和政府运行的基础设施的边缘，有时与政府设施断开连接、并作为一个完全独立的“网络服务”的存在，有时则利用任何可能的方式来恰当控制分散化的资源。
  </p>
  <p class="graf graf--p">
   在以下的3篇小故事中，我们将探讨
   <strong class="markup--strong markup--p-strong">
    一个无比孤立的社会如何设法跨越这一巨大的障碍，并建立访问、分发、分享和享受数字内容的替代方式
   </strong>
   。
  </p>

  <h4 class="graf graf--h4">
   nano-station 国家
  </h4>
  <p class="graf graf--p">
   X 先生二十五岁左右，是一位充满激情的战地游戏玩家，也是 Los Pir @ t @ s 子节点的管理员，位于哈瓦那不那么富裕的街区之一。
  </p>
  <p class="graf graf--p">
   在附近建筑物的屋顶上，他安装了一个 WiFi nano-station，允许在 UTP 电缆 100 米范围之外的其他房屋使用不同类型的 DIY 或品牌制造的 WiFi 设备，加入他的本地网络。它们共同连接成了一个由大约 20 个*互联家庭*组成的本地网络，有权共享、交流、并主要是一起玩游戏。

  <p class="graf graf--p">
   在哈瓦那的每个城区，都有一个连接的计算机、服务器、nano-station 的网络，它们共同组成了一个
   <strong>
    巨大的城市网络
   </strong>
   ，这个网络中拥有大约 12.000 个互联的家庭 — SNet (Street NetWork).
  </p>
  <p class="graf graf--p">
   延伸近 30 公里从城市西部的圣达菲到东部的阿拉马边缘，从北部的马勒孔 20 公里到南部的科托罗，有一个隐藏的电缆网络，nano-station 和中继
   <strong class="markup--strong markup--p-strong">
    完全独立于政府、国营或私营企业之外，不属于任何权力，只属于人民
   </strong>
   ，就这样形成这种令人印象深刻的网络结构。
  </p>
  <p class="graf graf--p">
   这种网络不仅存在于哈瓦那，而且存在于该岛的大部分省份。第二大省份就是马坦萨斯，有大约 2000 个连通的家庭。网络与网络之间没有连接。它们作为某种孤立的网络孤岛而存在。没有连接到彼此也没有连接到外部世界，互联网。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    隐藏的避风港
   </strong>
  </p>
  <p class="graf graf--p">
   这个庞大的基础设施的每一部分都只属于其用户。每个用户都拥有自己的网络拼图，nanostation、服务器或电缆。使用共享文件夹中的带宽或内容是免费的，任何人都可以复制和分发这些内容。没有任何付费服务，禁止广告。
  </p>
  <p class="graf graf--p">
   在如今的世界里，
   <strong class="markup--strong markup--p-strong">
    我们这些人已经忘记了如何拥有自己的基础设施，屏幕后面的一切都属于其他人，我们主要是消费者，而不是制造商，我们不断被描述，被定制、被量化
   </strong>
   — — 而哈瓦那人们享有的东西在我们看起来就像某种技术乌托邦安全避风港。在这里，
   <strong class="markup--strong markup--p-strong">
    以某种方式提醒着互联网的早期样子，那是一个梦幻般美好的时代
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是，像所有事一样，它还有另一面。
  </p>
  <p class="graf graf--p">

    游戏规则
   </strong>
  </p>
  <p class="graf graf--p">
   这个网络的存在取决于古巴国家无形的、未说出口的和微妙的东西。根据古巴的“现实”，这个网络不应该存在。即使不禁止将计算机连接到网络，将网络设备导入古巴也会被禁止。
  </p>
  <p class="graf graf--p">
   <strong>
    由于古巴本地没有任何工厂生产 nanostation 或其他网络设备，因此存在30千米宽的 12.000 个连接设备的独立网络听起来是非常惊人的
   </strong>
   。政府仍然没有采取任何行动，希望他们不会，但这种现象不太可能被当局忽视。
  </p>
  <p class="graf graf--p">
   另一方面，在网络内部也存在强有力的内部政策，规范了任何可能关于政治、色情有关的内容的分发，或“任何可能影响古巴形象的讨论”。这些政策还禁止网络连接到互联网并用于传输任何外国的电视或广播节目。SNET 用户知道，如果这个网络开始被当局视为“反革命”活动的基础，他们的宝贵资源可能就会在某一天内彻底消失。

  <h4 class="graf graf--h4">
   一个 Paquete 统治他们所有
  </h4>
  <p class="graf graf--p">
   这张照片中间的计算机包含的内容下周将成为离线互联网的主要古巴版本之一。这个东西被称为 El Paquete，整个岛上可能有数十万古巴人用来分发 1TB 的内容。每个星期一早上，从哈瓦那的这个小房间里开始，这些电影、电视剧和节目、杂志、网页、软件和书籍包由分布式网络分发，包括人物、自行车、50年代的汽车、外部硬盘和 USB，横跨整个岛屿。
  </p>
  <p class="graf graf--p">
   这种现象的规模如此之大，以至于 El Paquete 成为了古巴文化部许多会议的主题，并被描绘成“古巴公民的心态”。
  </p>
  <p class="graf graf--p">
   El paquete 还有一些与政治内容或露骨性内容有关的
   <strong class="markup--strong markup--p-strong">
    自我审查政策
   </strong>
   ，在一个互联网访问极其有限的国家，虽然没有政府控制和影响力以及经济利益，这种内容的独立分配仍然是难以被接受的，对于不利于信息自由流动的系统来说。
  </p>
  <p class="graf graf--p">
   El Paquete 不仅仅是一个分销网络，它也是一种非官方经济形式，可能支持全岛数百个家庭。在几乎每个街区都有一个这样地方，通常在一个小房间或车库里，你可以带上你的硬盘和拷贝，出个价，这就是你拥有的部分 El Paquete。
  </p>
  <p class="graf graf--p">
   完整的 El Paquete 副本的价格为 2 CUC（约等于2美元）。通常这些地方也是合法的“盗版”DVD 商店，这是一种允许古巴人按照 2011 年推出的 181 种新官方工作类型的政府名单运作的小企业。

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    来自该岛的“端对端”
   </strong>
  </p>
  <p class="graf graf--p">
   这个独立的地下网络覆盖整个岛屿，从远东的古巴圣地亚哥到西部的 Pinar del Rio，这一事实是这一现象的一个显著方面。无论是谁试图乘坐 50 年历史的老式汽车从哈瓦那前往古巴圣地亚哥，或是在 50 年代建造的唯一一条会突然断掉的“高速公路”上拖着俄罗斯版 Lada，他们都知道，这是一次冒险。
  </p>
  <p class="graf graf--p">
   El Paquete 的硬盘仍然从一个城镇到另一个城镇的各个方向，每周到达数百个社区。这种独特的古巴“人 — 电子高速公路”相当滞后。如果让哈瓦那市民每周一能享受到他们最喜欢的 dorama（韩国电视剧）的最新一集，那么距离哈瓦那以东约 800 公里的古巴圣地亚哥市民将能够在星期三，也就是2天后获得同样的享受。
  </p>
  <p class="graf graf--p">
   据我们估计，这种手牵手的速度，驱动器到驱动器、计算机到计算机的网络速度，约为 16.6 公里/小时。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    节点的力量，经济和 El Paquete 的结构变化
   </strong>
  </p>
  <p class="graf graf--p">
   所有节点都是相同的，除了其中一些节点的层次结构高于其他节点。该网络有自己的垂直层次结构。一切都从一个点开始，然后在不同的分支中增长，直到这个人工运输硬件信息的网络覆盖整个岛屿。
  </p>
  <p class="graf graf--p">
   但是，初始包的创造者有理由不觉得自己拥有该网络。此内容分发生态系统中的每个分支或单元都可以自由添加和删除内容，初始数据包的创建者或途中的任何其他节点都无法控制在其后发生的事。
  </p>
  <p class="graf graf--p">
   更高的节点，或者我们可以说更靠近分销链中心的节点，在它们下面有很多分布可以尝试在其分支机构内发展自己的微观经济，并尝试例如开发自己的广告交易。最后，古巴周边的私营、本地微型企业很少有能力在国家层面销售他们的产品或服务，因此他们选择在城市层面甚至单个街区内的 El Paquete 内部做广告，这是完全合理的。
  </p>
  <p class="graf graf--p">
   另一个有趣的观点是，由于 El Paquete 每份副本所需的时间（每份约2小时），没有人能够比其他人获得更多得收入。从这个意义上说，
   <strong class="markup--strong markup--p-strong">
    时间是在这个系统中以某种方式创造平等的一个因素
   </strong>
   。
  </p>
  <p class="graf graf--p">
   作为小中心的业主之一，在 Vedado 收集了 50 TB 的盗版游戏，El Paquete 周一早上就会出现在那里。如何做到这点？你可以尝试在下周一到来之前创造出一些东西，你可以在它的一个部分（例如游戏或软件）上发展你的小经济，或者你什么都不做也行。每个人都有同样的机会，同样的价格。简直是共产主义。
  </p>
  <p class="graf graf--p">
   一个有趣的细节是 El Paquete 的
   <strong class="markup--strong markup--p-strong">
    价格会根据当天是星期几而发生变化
   </strong>
   。在发布后的星期一，价格最高（2 CUC），直到本周末，在被复制多次后，价格下降（1 CUC）。很公平哦。
  </p>

    数据包经济
   </strong>

  <p class="graf graf--p">
   这些年来，该系统中出现了不同形式的广告和服务。
  </p>
  <p class="graf graf--p">
   如果您发送短信到某个号码，该消息的内容将以一个 jpeg 图像的形式出现在 El Paquete 文件夹“classifieds”中，该图片的一半是内容，另一半则是一些广告。并且，还有很多有趣的寄生广告形式直接嵌入到内容中。比中国的“软广”更“硬”一点，例如，在最新的 Cosmopolitan 杂志的数字版本的页面之间插入了本地的视频广告。

   <strong class="markup--strong markup--p-strong">
    El Paquete 的轮回
   </strong>
  </p>
  <p class="graf graf--p">
   El Paquete 只有一个相当短暂的存在。岛上很少有人可能拥有以前版本的数据包完整档案。甚至 El Paquete 的制造者也没有保留以前的版本，仅仅是因为硬盘驱动器的成本太高了，而普通的古巴人无法保留这些文件。
  </p>
  <p class="graf graf--p">
   它类似于普通电视或视频点播。在一周的时间内，这些文件从稀缺变为丰富，最后大部分文件在本周末被删除以替换为新文件。一些文件会继续存在，数字内容经销商或例如游戏或电影的收藏家会保留最受欢迎的电影、电视节目或游戏，并且每集或整部电影以固定价格转售。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    反馈机制
   </strong>
  </p>
  <p class="graf graf--p">
   作为每个适当的控制系统，都有一个嵌入其中重要部分的反馈。终端分发节点可以向 El Paquete 发送电子邮件并就最终用户请求的内容提出建议。根据输入、内容变化、新的电视连续剧、收藏或电影类型被添加，El Paquete 会不断变化发展。
  </p>
  <p class="graf graf--p">
   根据其中一位 El Paquete 制造商的说法，他们收到的电子邮件数量很大，他们无法响应所有请求，但这种互动仍然是这个复杂的分发系统的重要部分。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    文化的起源
   </strong>
  </p>
  <p class="graf graf--p">
   创造和策划 El Paquete 的年轻人大多是 20 多岁，这一代人在岛上经历了不同形式的内容分发时代。从 20 世纪90年代末的 VHS banks，到 21 世纪初的内部硬盘驱动器内容的离线点对点交换。断开内部硬盘驱动器并转发给朋友复制他的收藏品是过去 10 年中 El Paquete 和外部 USB 硬盘驱动器出现之前最常见的内容分发实践。
  </p>
  <p class="graf graf--p">
   内容很分散，并且您只能依赖于朋友的品味和存储内容的能力，而不是您自己的。当时互联网更难以进入，只有少数大学和机构有权访问一个非常慢的互联网。
  </p>

   内容主要来自大学网络管理员，他们在夜间下载在线内容并在接下来的几天内快速传播。内容由具有访问权限的个人策划和选择，主要是按流派分类的电影、连续剧和动漫作品，人们没有兴趣尝试获得糟糕的电视内容或中篇小说，受众主要是年轻人，需求各不相同。

  <p class="graf graf--p">
   为了使用这些 WiFi 热点，您可以在临时的露天接待台后面找到官方“电信代理”，但是，
   <strong class="markup--strong markup--p-strong">
    需要使用您的真实姓名和身份证注册您的帐户 — — 是的，就像中国的微信
   </strong>
   。然后你可以购买2个 CUC 1小时的 WiFi 信用额度，大约是古巴平均月工资的 10％。
  </p>
  <p class="graf graf--p">
   但是，在大多数官方 WiFi 热点附近，附着有许多隐形的寄生基础设施，以许多创造性的方式利用这种资源。
  </p>
  <p class="graf graf--p">
   当您接近 WiFi 区域时，某些不那么官方的“电信代理”可能会为您提供更便宜的互联网接入。如果您没有直接与他们联系，您应该搜索拥有创建海盗热点的设备的人。有时 SSID 本身就可以帮助您。例如，WiFi 网络的名称可以是“guywiththegreenshirt”或“guyunderthetree”。很有趣吧。
  </p>
  <figure aria-describedby="caption-attachment-1488" class="wp-caption aligncenter" id="attachment_1488" style="width: 1600px">

   花掉一个CUC（大约1美元），将允许在您的手机中输入 WiFi 密码，然后，您将准备好在最痛苦的慢速互联网上冲浪。
  </p>

   但有时这些盗版 WiFi 区域也出现在官方 WiFi 区域之外的地方。我们撞到了哈瓦那中心的 Malecon 附近。

  </p>

  <p class="graf graf--p">
   那些非官方的街头热点是一些短暂的街头网吧、会议场所和社交场所，人们不仅建立了互联网连接，而且在当地相互交换文件和内容。他们的设备之间发生了许多无形的活动，没有访问互联网。
  </p>
  <p class="graf graf--p">
   为了进入这个本地网，他们使用中国制造的通信应用程序
   <a class="markup--anchor markup--p-anchor" data-href="http://www.izapya.com/" href="http://www.izapya.com/" rel="noopener noreferrer" target="_blank">
    Zapya
   </a>
   ，允许用户交换文件并在本地进行群聊，而无需 WiFi 连接。
  </p>
  <p class="graf graf--p">
   有时候人们也会去那里与人聊天，免费交换文件，根本不使用互联网连接。
  </p>
  <p class="graf graf--p">
   值得一提的是，11年前的 2008 年，移动电话仅限于为外国公司和政府官员工作的古巴人使用才是合法的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    寄生网络
   </strong>
  </p>
  <p class="graf graf--p">
   销售连接的人所承担的风险是，其他人从他那里购买价值1个 CUC 的互联网接入，然后就可以以相同或更低的价格转售同一连接给其他人。这就是第一个人对别人做的事。这家伙可能正在使用他自己的 nano stations 和中继、自己动手做的基础设施，来瞄准并连接到一些酒店或官方 WiFi 区域，在那里他支付2个CUC一小时，假装是普通用户。
  </p>
  <p class="graf graf--p">
   然后，他与在街角的 20 个人分享了这一连接，他很可能会在下一个角落以同样的方式再卖给更多人。
  </p>
  <p class="graf graf--p">
   这是寄生分形网络的一种形式，网络内的网络，把碎片整理成小块。最后一个比索可能是从任何人那里收取的。
  </p>
