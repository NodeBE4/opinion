  <p class="graf graf--p">
   虽然可能永远无法像跟踪大型船只那样跟踪和识别汽车，但这篇文章提供了一些有用的提示，这些信息可以帮助您使用开源情报手段寻找特定车辆。
  </p>
  <p class="graf graf--p">
   在研究汽车和摩托车时，哪些数据是相关的？
  </p>
  <p class="graf graf--p">
   由于大多数车辆都是量产的，因此仅基于制造商、型号和颜色进行的研究可能会有些挑战。因此，您需要的应该是唯一的标识符，例如VIN或车牌。
  </p>
  <p class="graf graf--p">
   VIN（车辆识别号）是一个17位数字的代码，在制造时会分配给每个车辆。有多个付费数据库，可用于查找VIN并检索有关车辆及其历史的信息。
  </p>
  <p class="graf graf--p">
   如果您不想花钱，请尝试谷歌搜索 VIN。由于它是如此独特，因此不会出现很多误报。
  </p>
  <p class="graf graf--p">

    Autogespot

  <p class="graf graf--p">
   万一您必须弄清楚车子来自哪个国家，可以在
   <a class="markup--anchor markup--p-anchor" data-href="http://www.worldlicenseplates.com/" href="http://www.worldlicenseplates.com/" rel="noopener noreferrer" target="_blank">
    World License Plates

   上找到大量的车牌。事实证明，奥巴梅扬的法拉利在德国注册。他的车牌是 AIB-Q1414。让我们看看是否可以在 Autogespot 上找到这辆车。
  </p>
  <p class="graf graf--p">
