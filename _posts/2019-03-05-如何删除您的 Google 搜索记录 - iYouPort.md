  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      逃跑的最佳方法就是逃跑，而不是一边跑一边撒面包屑；谷歌等巨头究竟是什么鸟如今已经众所周知了，在变革成功之前您只能自己保护自己 —— 清除您在网上留下的数据面包屑。至少从谷歌开始
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p graf-after--p" id="7c7e">
   我们在前面讲述了搜索记录有多可怕。
  </p>
  <p class="graf graf--p graf-after--p" id="fd72">
   Google 会跨多个应用和服务跟踪您的搜索活动。谷歌拥有多个互联网资产，如 YouTube，谷歌地图，Chrome 网络浏览器，谷歌 Play 商店等。所有这些都用于跟踪用户活动。
  </p>
  <p class="graf graf--p graf-after--p" id="f420">
   收集的数据就是 Google 的主要收入来源，因为广告客户会使用您的数据。Google 通过收集有关个人的数据并将这些信息出售给广告商来赚钱。还包括其他莫名其妙的数据中间商，你根本不知道是谁的家伙。
  </p>
  <p class="graf graf--p graf-after--p" id="729b">
   您的“搜索活动”就是您在任何 Google 产品中输入的字词和词组，用于在网络上查找内容。即使您使用谷歌地图，您的搜索历史记录 例如 您搜索或访问过的所有餐馆，都会被 Google 跟踪到。详见我们
   <strong class="markup--strong markup--p-strong">
    仅仅使用历史记录实现的侵入性挖掘《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%EF%B8%8F-%E6%B5%8F%E8%A7%88%E5%8E%86%E5%8F%B2%E8%AE%B0%E5%BD%95%E6%9B%9D%E5%85%89%E4%BD%A0%E7%9A%84%E5%86%85%E5%BF%83%E5%85%A8%E6%99%AF%E5%9B%BE-osint-%E5%BC%80%E6%BA%90%E8%B0%83%E6%9F%A5%E6%BC%94%E7%A4%BA-d0afb124cce5" href="https://medium.com/@iyouport/%EF%B8%8F-%E6%B5%8F%E8%A7%88%E5%8E%86%E5%8F%B2%E8%AE%B0%E5%BD%95%E6%9B%9D%E5%85%89%E4%BD%A0%E7%9A%84%E5%86%85%E5%BF%83%E5%85%A8%E6%99%AF%E5%9B%BE-osint-%E5%BC%80%E6%BA%90%E8%B0%83%E6%9F%A5%E6%BC%94%E7%A4%BA-d0afb124cce5" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     浏览历史记录曝光你的内心全景图
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="73d2">
   用户可以控制和删除搜索活动。如果您不希望使用您设备的其他人看到您在网络搜寻内容，那么您需要清理搜索记录。
  </p>
  <p class="graf graf--p graf-after--p" id="9855">
   请记住，
   <strong class="markup--strong markup--p-strong">
    此信息在所有已登录的设备中共享
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="c5ac">
   这意味着，如果您通过智能手机在Google.com上搜索了某些内容并在其他位置登录到同一个 Google 帐户，这些历史就会显示在其他设备上。例如，同样登录到同一 Google 帐户的家庭计算机也包含相同的搜索记录。
   <strong class="markup--strong markup--p-strong">
    它是实时的
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="dff0">
   如何查看我过去的网页搜索记录？
  </p>
  <p class="graf graf--p graf-after--p" id="b27b">
   要查看过去的搜索活动，请在网络浏览器中访问https://myactivity.google.com/privacyadvisor/search。
  </p>
  <p class="graf graf--p graf-after--p" id="268d">
   您不必使用 Chrome，但如果您想查看和管理 Google 搜索记录，则必须登录 Google。虽然我几乎只使用 Firefox，而 Chrome 则用于编程工作，但我的活动也是被 Google 记录的。
  </p>
  <p>
   <img alt="" class="aligncenter wp-image-1623 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/03/0-6.png?resize=800%2C577&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/03/0-6.png?w=1162&amp;ssl=1 1162w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/03/0-6.png?resize=300%2C216&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/03/0-6.png?resize=768%2C554&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/03/0-6.png?resize=1024%2C738&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/03/0-6.png?resize=1100%2C793&amp;ssl=1 1100w" data-recalc-dims="1" height="577" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/03/0-6.png?resize=800%2C577&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>
   <noscript>
    <img alt="" class="aligncenter wp-image-1623" data-recalc-dims="1" height="577" sizes="(max-width: 800px) 100vw, 800px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/03/0-6.png?resize=800%2C577&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/03/0-6.png?w=1162&amp;ssl=1 1162w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/03/0-6.png?resize=300%2C216&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/03/0-6.png?resize=768%2C554&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/03/0-6.png?resize=1024%2C738&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/03/0-6.png?resize=1100%2C793&amp;ssl=1 1100w" width="800"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="36dd">
   <strong class="markup--strong markup--p-strong">
    即便您没有登录 Google 帐户，您仍然会向 Google 提供数据，但您无法控制它
   </strong>
   。当您登录 Google 帐户（Gmail，YouTube，Google Analytics或其他帐户）时，您可以从浏览器中删除搜索记录。
  </p>
  <p class="graf graf--p graf-after--p" id="b128">
   如果您还没有登录 Google 帐户，请访问https://myactivity.google.com/privacyadvisor/search。从那里，您只能查看自己的 Google 活动 — 即在Google.com和上述其他媒体资源上的搜索。要查看仅限 Google 的活动，请选择“仅限 Google”。如果您想查看所有搜索活动（Google 正在监控并保存的东西），请选择左侧按钮选择“所有搜索活动”。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-1624 jetpack-lazy-image" data-lazy-sizes="(max-width: 547px) 100vw, 547px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/03/0-7.png?resize=547%2C386&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/03/0-7.png?w=547&amp;ssl=1 547w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/03/0-7.png?resize=300%2C212&amp;ssl=1 300w" data-recalc-dims="1" height="386" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/03/0-7.png?resize=547%2C386&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="547"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-1624" data-recalc-dims="1" height="386" sizes="(max-width: 547px) 100vw, 547px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/03/0-7.png?resize=547%2C386&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/03/0-7.png?w=547&amp;ssl=1 547w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/03/0-7.png?resize=300%2C212&amp;ssl=1 300w" width="547"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="e12d">
   要删除所有搜索记录，请从活动页面开始，然后点击删除搜索活动按钮。您可以选择全部擦除或仅删除搜索历史记录的最后一小时。
  </p>
  <p class="graf graf--p graf-after--p" id="b62f">
   这会删除您的活动数据，例如您搜索的字词以及您选择的链接。
  </p>
  <p class="graf graf--p graf-after--p" id="3cbd">
   请记住，
   <strong class="markup--strong markup--p-strong">
    这并不是消除您的在线行为所有痕迹的完整方式
   </strong>
   。要真正清理它，您必须确保清理所有Internet 临时文件。如果您登录网站进行购买或查看某些内容，那么该登录历史记录也会与您保存的密码一起保存在用户名下的 Windows 系统文件中。您还需要删除Google 时间线并调整隐私设置。
  </p>
  <p class="graf graf--p graf-after--p" id="1372">
   谷歌是个强大的数据黑洞，对于社交工程攻击者来说非常重要，最简单的详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E9%AB%98%E7%BA%A7%E8%BF%90%E7%AE%97%E7%AC%A6%E8%BE%85%E5%8A%A9%E5%BC%80%E6%BA%90%E8%B0%83%E6%9F%A5-%E5%B7%A7%E7%94%A8%E6%90%9C%E7%B4%A2%E5%BC%95%E6%93%8E%E6%8C%96%E6%8E%98%E6%83%85%E6%8A%A5-osint-c64e71b05a2f" href="https://medium.com/@iyouport/%E9%AB%98%E7%BA%A7%E8%BF%90%E7%AE%97%E7%AC%A6%E8%BE%85%E5%8A%A9%E5%BC%80%E6%BA%90%E8%B0%83%E6%9F%A5-%E5%B7%A7%E7%94%A8%E6%90%9C%E7%B4%A2%E5%BC%95%E6%93%8E%E6%8C%96%E6%8E%98%E6%83%85%E6%8A%A5-osint-c64e71b05a2f" rel="noopener noreferrer" target="_blank">
    高级运算符辅助开源调查：巧用搜索引擎挖掘情报 #OSINT
   </a>
   》，如果您是一位不想被抓住把柄的社交工程师或开源调查研究人士，那么最好使用单独的账户、单独设备，并及时清理足迹。
  </p>
