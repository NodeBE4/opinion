  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://codastory.com/authoritarian-tech/global-rise-internet-sovereignty/" href="https://codastory.com/authoritarian-tech/global-rise-internet-sovereignty/" rel="noopener noreferrer" target="_blank">
    Eduard 的文章
   </a>
   从一个角度解释了“互联网主权”为什么会令民主国家和专制国家异口同声地支持，即 控制。
  </p>
  <p class="graf graf--p">
   于是他认为这是家长制的态度。当然，这里的控制是贬义的，指的是压制，就如中国所做的那样。
  </p>
  <p class="graf graf--p">
   不过对于互联网主权的概念来说，还有一种情况，就如加州的主张，即 摆脱联邦政府不合适的隐私规则所造成的危害。加州签署了自己的隐私法案，类似欧盟的政策。
  </p>
  <p class="graf graf--p">
   但是加州没有成功。美国司法部已经起诉了加利独立的网络中立法，认为“只有联邦政府才有权管理州际通信网络”。
  </p>
  <p class="graf graf--p">
   不仅于此。互联网主权的概念虽然在某些情况下没有申明，但已经实际上存在很多种形式了。
  </p>
  <p class="graf graf--p">
   比如，包括巴基斯坦和俄罗斯在内的至少十几个国家对互联网进行了审查；与此同时，欧盟内部的搜索结果可能与其他地方的搜索结果不同，因为它有被遗忘权法律，而世界各地的网络出版商仍在努力应对今年生效的欧盟隐私法规的影响。
  </p>
  <p class="graf graf--p">
   即便在美国国内也不是统一的。
  </p>
  <p class="graf graf--p">
   伊利诺伊州有一项生物识别隐私法，该法被用来涵盖 Facebook 和谷歌使用的面部扫描软件；像华盛顿这样的州也通过了网络中立规则，但加利福尼亚州以其规模和规则的全面性，特别是在一起采取时，可以使其与其他州更明显地区别开来。
  </p>
  <p class="graf graf--p">
   中国的GFW是比较极端的案例，但的确不是特殊的。
  </p>
  <p class="graf graf--p">
   在监视全球的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" rel="noopener noreferrer" target="_blank">
    棱镜计划
   </a>
   被曝光后，欧洲努力摆脱硅谷巨头控制的互联网的想法并不会令人惊讶。
  </p>
  <p class="graf graf--p">
   互联网自由倡导者担心互联网因此将切割成多个国家级的“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wired.com/story/splinternet-global-court-rulings-google-facebook/" href="https://www.wired.com/story/splinternet-global-court-rulings-google-facebook/" rel="noopener noreferrer" target="_blank">
    分裂网
   </a>
   ”。
  </p>
  <p class="graf graf--p">
   分裂也许不至于，至少还在较大程度上具有可协调性；比如加州的网络中立法和中国的审查制度就完全不同。
  </p>
  <p class="graf graf--p">
   但是无疑，这是个麻烦，因为互联网的优势被毁掉了。
  </p>
  <p class="graf graf--p">
   对于中国这种案例来说，他们不仅是希望对国内进行全面的控制，也包括在全球范围内逐渐绘制一个新的权力版图。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5736 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/普京网.png?resize=1100%2C611&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/普京网.png?w=1256&amp;ssl=1 1256w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/普京网.png?resize=300%2C167&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/普京网.png?resize=768%2C427&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/普京网.png?resize=1024%2C569&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/普京网.png?resize=1100%2C611&amp;ssl=1 1100w" data-recalc-dims="1" height="611" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/普京网.png?resize=1100%2C611&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5736" data-recalc-dims="1" height="611" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/普京网.png?resize=1100%2C611&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/普京网.png?w=1256&amp;ssl=1 1256w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/普京网.png?resize=300%2C167&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/普京网.png?resize=768%2C427&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/普京网.png?resize=1024%2C569&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/普京网.png?resize=1100%2C611&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   今年3月，约有10,000人
   <a class="markup--anchor markup--p-anchor" data-href="https://meduza.io/en/feature/2019/03/12/we-ve-got-no-alternative-but-our-hatred" href="https://meduza.io/en/feature/2019/03/12/we-ve-got-no-alternative-but-our-hatred" rel="noopener noreferrer" target="_blank">
    聚集在
   </a>
   莫斯科的 Sakharov Prospect，挥舞着一些标语，上面写着“普京网 Putin Net”，并高呼着“放开互联网！”
  </p>
  <p class="graf graf--p">
   这是对一项新立法的民意反馈，该立法推动将俄罗斯的“互联网”从万维网中分离出来，使其自给自足，并宣称这可以防范外部“威胁”。
  </p>
  <p class="graf graf--p">
   克里姆林宫的发言人迅速驳斥了抗议者的担忧，他们说这是一种“误解”，并
   <a class="markup--anchor markup--p-anchor" data-href="https://iz.ru/854888/2019-03-11/v-kremle-prokommentirovali-proshedshii-miting-za-svobodu-interneta" href="https://iz.ru/854888/2019-03-11/v-kremle-prokommentirovali-proshedshii-miting-za-svobodu-interneta" rel="noopener noreferrer" target="_blank">
    声称
   </a>
   互联网自由“不会破坏”。
  </p>
  <p class="graf graf--p">
   但是，随后，没过半个月，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.themoscowtimes.com/2019/03/18/putin-signs-fake-news-internet-insults-bills-into-law-a64850" href="https://www.themoscowtimes.com/2019/03/18/putin-signs-fake-news-internet-insults-bills-into-law-a64850" rel="noopener noreferrer" target="_blank">
    普京就签署了两项限制在线言论的法律
   </a>
   ：一项对假新闻处以罚款，另一项禁止“不尊重当局和国家的象征”。
  </p>
  <p class="graf graf--p">
   两者都是口袋罪。
  </p>
  <p class="graf graf--p">
   换句话说，俄罗斯政府正在加强对互联网的控制。
  </p>
  <p class="graf graf--p">
   俄罗斯绝不是唯一这么做的国家。
  </p>
  <p class="graf graf--p">
   俄罗斯和一系列不那么民主的国家，如中国、沙特、伊朗等等，都对互联网抱有一定的诡异期望。
  </p>
  <p class="graf graf--p">
   一方面，这些国家的当权者想要严格控制哪些信息能流入其境内；另方面在全球范围内，他们希望政府，而不是公司和NGO组织，掌控互联网。
  </p>
  <p class="graf graf--p">
   总而言之，他们的目标是试图以国界划分网络空间，因此它们有时被称为“网络主权”原则。
  </p>
  <p class="graf graf--p">
   中国是全球网络主权推动的先锋，该国的GFW构建的小世界内克隆了万维网的大部分应用。
  </p>
  <p class="graf graf--p">
   2014年，中国互联网沙皇鲁炜撰写了一篇题为“网络主权必须规范全球互联网”的专栏文章。在鲁炜的领导下，中国的网络空间机构还发布了一个备受讽刺的“网络安全”颂歌。
  </p>
  <p class="graf graf--p">
   如今有效网络主权的愿景正扩展到新的视野。
  </p>
  <p class="graf graf--p">
   自由之家去年11月的报告审查了65个国家，并发现自去年以来有26个国家的互联网自由度明显下降。
  </p>
  <p class="graf graf--p">
   与此同时，中国式的“数字威权主义”在影响力不断增强。
  </p>
  <p class="graf graf--p">
   而西方民主国家也在越来越多地反映出网络主权的谈话要点。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    早期挫折
   </strong>
  </p>
  <p class="graf graf--p">
   中国模式并不是一直在上升的。就在六年前，当中国及其盟国试图“接管互联网”时，网络主权的宣称遭遇过重大挫折。
  </p>
  <p class="graf graf--p">
   在2012年国际电信联盟会议上，俄罗斯、中国和其他一些国家试图通过一项具有象征意义的决议，这一决议将支持某些网络主权原则，并赋予联合国更大的互联网权力。
  </p>
  <p class="graf graf--p">
   但大型科技公司和美国国会发表了
   <a class="markup--anchor markup--p-anchor" data-href="https://arstechnica.com/tech-policy/2012/12/why-the-itu-is-the-wrong-place-to-set-internet-standards/" href="https://arstechnica.com/tech-policy/2012/12/why-the-itu-is-the-wrong-place-to-set-internet-standards/" rel="noopener noreferrer" target="_blank">
    强烈谴责的声明
   </a>
   ，美国和欧洲国家拒绝签署该决议。
  </p>
  <p class="graf graf--p">
   如果没有他们的签名，网络主权就无法推动。
  </p>
  <p class="graf graf--p">
   如果成功，中国及其数字政策盟友将帮助解决长期困扰专制政权的问题，即 互联网没有固定的管理者。
  </p>
  <p class="graf graf--p">
   在权力分配中，网络对民族国家没有特别的尊重；政府与公司、非政府组织、学术界和其他政党一起管理互联网，这种安排严重削弱了各国的影响力。
  </p>
  <p class="graf graf--p">
   例如，中国政府已将其令人尴尬的宣传歌曲从互联网上删除，但大量副本仍然在线。这些副本存储在ICANN（国际域名管理组织）注册的网站上。
  </p>
  <p class="graf graf--p">
   如果像中国这样的政府对这个组织拥有权力，他们就可以使这些网站完全消失。对于互联网自由的拥护者来说，这是噩梦一般的场景。
  </p>
  <p class="graf graf--p">
   而另一方面，对于网络主权倡导者而言，给予政府对全球互联网的最终发言权是终极目标。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    基层主权
   </strong>
  </p>
  <p class="graf graf--p">
   在2012年试图将互联网置于联合国控制之下的努力失败之后，网络主权辩论已转向国内政治。
  </p>
  <p class="graf graf--p">
   非党派智囊团 New America 最近的一份报告
   <a class="markup--anchor markup--p-anchor" data-href="https://www.newamerica.org/cybersecurity-initiative/reports/digital-deciders/" href="https://www.newamerica.org/cybersecurity-initiative/reports/digital-deciders/" rel="noopener noreferrer" target="_blank">
    得出的结论
   </a>
   是，各国已经意识到，
   <strong class="markup--strong markup--p-strong">
    “今天的战斗在于各国应该如何在国内塑造互联网。”
   </strong>
  </p>
  <p class="graf graf--p">
   因此，中国不再依赖国际条约，而是开始将其互联网审查制度出口到其他国家，从而自下而上地改变互联网。
  </p>
  <p class="graf graf--p">
   根据自由之家的说法，
   <strong class="markup--strong markup--p-strong">
    至少有36个政府接受了关于“新媒体和信息管理”的闭门中国培训。
   </strong>
  </p>
  <p class="graf graf--p">
   其中包括约旦、埃及、沙特阿拉伯和越南。
  </p>
  <p class="graf graf--p">
   这些培训的内容未知，但在2017年越南官员接受培训后，越南政府于2018年通过了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bloomberg.com/news/articles/2018-11-03/vietnam-says-google-and-facebook-may-have-year-to-meet-cyber-law" href="https://www.bloomberg.com/news/articles/2018-11-03/vietnam-says-google-and-facebook-may-have-year-to-meet-cyber-law" rel="noopener noreferrer" target="_blank">
    中国式的网络立法
   </a>
   。
  </p>
  <p class="graf graf--p">
   还有证据表明，一些国家，如乌干达，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bloomberg.com/news/features/2019-01-10/china-s-digital-silk-road-is-looking-more-like-an-iron-curtain" href="https://www.bloomberg.com/news/features/2019-01-10/china-s-digital-silk-road-is-looking-more-like-an-iron-curtain" rel="noopener noreferrer" target="_blank">
    使用中国制造的软件来监控其本地互联网
   </a>
   ，表面上是为了打击犯罪。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种传递模仿有助于使中国的专制互联网方法合法化，巩固其治理模式。
   </strong>
  </p>
  <p class="graf graf--p">
   New America 网络安全分析师罗伯特·莫格斯（Robert Morgus）表示，“新的国际意识形态斗争是威权主义与自由民主之间的斗争”，他将中国的网络管制模式出口定义为软实力。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “传播互联网专制模式将有助于传播专制政治手段”。
  </p>
  <p class="graf graf--p">
   正如中国在很大程度上培养和装备发展中国家一样，越来越多的迹象表明，
   <strong class="markup--strong markup--p-strong">
    西方民主国家也正在变得对互联网控制充满兴趣。
   </strong>
  </p>
  <p class="graf graf--p">
   New America 的报告载有的数据显示，在过去几年中，曾经具有开放式互联网思维和较少政府干预的国家如今已开始向“更加国家主义和更封闭的互联网方式”迈进。
  </p>
  <p class="graf graf--p">
   这就是分析师
   <a class="markup--anchor markup--p-anchor" data-href="https://www.internetgovernance.org/2017/05/19/governing-cybersecurity-or-the-internet-report-on-our-workshop/" href="https://www.internetgovernance.org/2017/05/19/governing-cybersecurity-or-the-internet-report-on-our-workshop/" rel="noopener noreferrer" target="_blank">
    所指出的
   </a>
   ：中国和俄罗斯式的偏执已经在西方产生了共鸣。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5737 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/网络主权-美国.png?resize=1100%2C827&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/网络主权-美国.png?w=1256&amp;ssl=1 1256w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/网络主权-美国.png?resize=300%2C225&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/网络主权-美国.png?resize=768%2C577&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/网络主权-美国.png?resize=1024%2C770&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/网络主权-美国.png?resize=1100%2C827&amp;ssl=1 1100w" data-recalc-dims="1" height="827" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/网络主权-美国.png?resize=1100%2C827&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5737" data-recalc-dims="1" height="827" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/网络主权-美国.png?resize=1100%2C827&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/网络主权-美国.png?w=1256&amp;ssl=1 1256w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/网络主权-美国.png?resize=300%2C225&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/网络主权-美国.png?resize=768%2C577&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/网络主权-美国.png?resize=1024%2C770&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/网络主权-美国.png?resize=1100%2C827&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “Cyber​​ 9/11”
   </strong>
  </p>
  <p class="graf graf--p">
   网络政策专家 Alexander Klimburg 在其2017年出版的“The Darkening Web”一书中写道，
   <strong class="markup--strong markup--p-strong">
    “将信息定义为武器的能力，是网络主权倡导者的最终目标”。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    因为如果信息是武器，那么审查就是国家安全的合法问题。
   </strong>
  </p>
  <p class="graf graf--p">
   作为“俄罗斯干涉2016年美国大选”调查的副作用，这项努力大部分取得了成功。
  </p>
  <p class="graf graf--p">
   在西方精英中，数字信息战现在是一个热门话题，也是一个明显的威胁。
  </p>
  <p class="graf graf--p">
   当希拉里克林顿在2016年将俄罗斯黑客行为
   <a class="markup--anchor markup--p-anchor" data-href="https://thehill.com/policy/cybersecurity/355601-clinton-refers-to-russian-election-interference-as-cyber-9-11" href="https://thehill.com/policy/cybersecurity/355601-clinton-refers-to-russian-election-interference-as-cyber-9-11" rel="noopener noreferrer" target="_blank">
    称为
   </a>
   “Cyber​​9/11”时，或当罗伯特·穆勒开始
   <a class="markup--anchor markup--p-anchor" data-href="https://www.politico.com/story/2018/02/16/mueller-indictment-russia-takeaways-353667" href="https://www.politico.com/story/2018/02/16/mueller-indictment-russia-takeaways-353667" rel="noopener noreferrer" target="_blank">
    起诉俄
   </a>
   罗斯公民涉嫌 trolling 时，这一点就很明显了。
  </p>
  <p class="graf graf--p">
   在这种氛围中，一些西方民主国家和专制国家一样，在
   <strong class="markup--strong markup--p-strong">
    努力寻求更多的控制权。
   </strong>
  </p>
  <p class="graf graf--p">
   在巴黎举行的2018年互联网治理政府间论坛上，这是一个普遍尊重政府在互联网中作用有限的机构，法国总统马克龙（他在竞选活动的最后几天电子邮件遭到
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnbc.com/2017/05/07/macron-email-leaks-far-right-wikileaks-twitter-bots.html" href="https://www.cnbc.com/2017/05/07/macron-email-leaks-far-right-wikileaks-twitter-bots.html" rel="noopener noreferrer" target="_blank">
    黑客攻击
   </a>
   ）称网络空间是一个危险的领域。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “今天的互联网会更好地被极端人士使用，”马克龙在一屋子人面前说道，“相比其他任何内容，互联网更多地被用于仇恨言论或传播恐怖主义内容。”
  </p>
  <p class="graf graf--p">
   然后，他要求政府主导打击互联网上“最糟糕的趋势”。
  </p>
  <p class="graf graf--p">
   互联网治理专家 Badiei 参加了那场巴黎会议，他表示房间里的许多人都对 Macron 的言论
   <strong class="markup--strong markup--p-strong">
    感到震惊
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果这些话是由一个压迫性政权的领导人说出来的，那么人们就会直接拉门走出去”，她说。
  </p>
  <p class="graf graf--p">
   可惜不是。而是“关于一个民主国家的总统坚持认为互联网应受到监管，
   <strong class="markup--strong markup--p-strong">
    提供的是与压迫政权极为相似的理由”。
   </strong>
  </p>
  <p class="graf graf--p">
   乔治亚理工学院的公共政策教授 Hans Klein 参加了法国的会议和在中国举办的国家赞助的乌镇互联网会议。他也指出，
   <strong class="markup--strong markup--p-strong">
    这两次会议都指出了“需要更多国家控制”的对互联网“相似的担忧”。
   </strong>
  </p>
  <p class="graf graf--p">
   这并不是说马克龙已经和中国穿一条裤子了。计算机科学教授兼互联网治理专家 Kieron O’Hara 表示，西方民主国家只是在“共同的恐惧”中与中国和俄罗斯融合了。
  </p>
  <p class="graf graf--p">
   这导致了对诸如“专制互联网”模型之类的想法异口同声。
  </p>
  <p class="graf graf--p">
   事实上，“专制互联网”甚至不是最好的能描述这件事的名字。也许“家长式互联网”更合适一些，当然，“家长作风”吸引了所有人。
  </p>
  <p class="graf graf--p">
   即 以“我为你好”为由剥夺个人意志，从而满足的仅仅是操纵者本人的优越感。⚪️
  </p>
