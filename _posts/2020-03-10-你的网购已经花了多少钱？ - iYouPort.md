  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我在亚马逊上花了多少钱？”—— 这个问题很可能会使电子商务巨头的长期客户感兴趣，甚至可能使他们感到恐惧。但了解这件事对您来说很重要，不只是钱的问题。
  </p>
  <p class="graf graf--p">
   而是数据分析的方法。
  </p>
  <p class="graf graf--p">
   本文将介绍这件事可以怎么做。
  </p>
  <p class="graf graf--p">
   这里将使用 Python 来分析 Amazon 数据。但是，
   <strong class="markup--strong markup--p-strong">
    如果您没有编码经验也请不要担心！本文将尽可能说得简单，而且只需要不到30行代码就能完成一大堆很酷的事
   </strong>
   ，例如：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    计算出您曾经在 Amazon 上花费的总金额；
   </li>
   <li class="graf graf--li">
    找到您最贵的和最便宜的订单，以及平均数和中位数订单；
   </li>
   <li class="graf graf--li">
    找出您已缴纳的税款以及您在亚马逊上的有效税率；
   </li>
   <li class="graf graf--li">
    通过条形图可视化您的每日消费习惯；
   </li>
   <li class="graf graf--li">
    自定义条形图的某些元素，例如大小和颜色，以呈现更清晰的解释。
   </li>
  </ul>
  <p class="graf graf--p">
   那么现在就开始。您会看到 Python 很好玩的地方。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     准备进行分析
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    1、下载并安装工具
   </span>
  </p>
  <p class="graf graf--p">
   为了进行分析，我们将做一些 Python 编程。但是请不要紧张，即使您一生中从未编写过任何代码，在这里也可以做到！下面将逐步介绍该过程。
  </p>
  <p class="graf graf--p">
   第一步是在计算机上安装一个名为 Anaconda 的工具。这将使您能够开始使用被称为 Jupyter Notebook 的东西进行编程。
  </p>
  <p class="graf graf--p">
   Jupyter Notebooks 是执行此类数据分析项目的出色工具，一旦掌握了这些要点，您可能会发现自己经常需要使用它。
  </p>
  <p class="graf graf--p">
   相关教程有很多，但是今天要做的事非常简单，所以只需要继续阅读即可。
  </p>
  <p class="graf graf--p">
   首先，需要访问
   <a class="markup--anchor markup--p-anchor" data-href="https://www.anaconda.com/distribution/" href="https://www.anaconda.com/distribution/" rel="noopener noreferrer" target="_blank">
    Anaconda 网站上的此页面
   </a>
   ，略微向下滚动，然后选择您的操作系统 —— Windows、Mac 或 Linux。
  </p>
  <p class="graf graf--p">
   选择了正确的操作系统后，点击
   <strong class="markup--strong markup--p-strong">
    Python 3.7
   </strong>
   版本下绿色的“下载”按钮来下载 Anaconda。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10082 jetpack-lazy-image" data-lazy-sizes="(max-width: 998px) 100vw, 998px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-23.png?resize=998%2C435&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-23.png?w=998&amp;ssl=1 998w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-23.png?resize=300%2C131&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-23.png?resize=768%2C335&amp;ssl=1 768w" data-recalc-dims="1" height="435" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-23.png?resize=998%2C435&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="998"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10082" data-recalc-dims="1" height="435" sizes="(max-width: 998px) 100vw, 998px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-23.png?resize=998%2C435&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-23.png?w=998&amp;ssl=1 998w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-23.png?resize=300%2C131&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/1-23.png?resize=768%2C335&amp;ssl=1 768w" width="998"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   下载完成后就可以像安装其他任何应用程序一样来进行安装它了：只需双击下载的文件，然后弹出安装向导以指导完成安装过程。
  </p>
  <p class="graf graf--p">
   基于本文的目的，默认安装设置会很好。不过，您需要记下要安装它的文件夹。
  </p>
  <p class="graf graf--p">
   尽管可以从任何位置保存和访问文件，但今天要将数据集和 Jupyter Notebook 的文件保存到同一文件夹中，以便于访问。
  </p>
  <p class="graf graf--p">
   安装 Anaconda 后就可以继续执行下一步了。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    2、下载亚马逊订单历史记录数据
   </span>
  </p>
  <p class="graf graf--p">
   亚马逊可以让您下载已完成订单的大量报告，尽管可用数据只能追溯到2006年。不过，如果您是普通的 Amazon 用户，那么十多年的数据肯定会包含一大堆有趣的东西。
  </p>
  <p class="graf graf--p">
   要获取数据，请在登录时
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amazon.com/gp/b2b/reports?ref_=ya_d_l_order_reports&amp;" href="https://www.amazon.com/gp/b2b/reports?ref_=ya_d_l_order_reports&amp;" rel="noopener noreferrer" target="_blank">
    单击此链接
   </a>
   ，直接进入订单报告下载页面。
  </p>
  <p class="graf graf--p">
   但是，如果这不起作用，也可以直接导航至该页面：转到 Amazon.com，然后单击右上角的
   <strong class="markup--strong markup--p-strong">
    Accounts and Lists
   </strong>
   按钮。在下一页上，找到 “订购和购物首选项” 部分，然后单击题为 “Download order reports” 的链接。
  </p>
  <p class="graf graf--p">
   如果您想通读本教程，但又不想使用自己的数据，则可以
   <a class="markup--anchor markup--p-anchor" data-href="https://www.dataquest.io/wp-content/uploads/2019/09/amazon-orders.csv" href="https://www.dataquest.io/wp-content/uploads/2019/09/amazon-orders.csv" rel="noopener noreferrer" target="_blank">
    下载这里的匿名数据集
   </a>
   。
  </p>
  <p class="graf graf--p">
   亚马逊提供四种不同的报告类型。目前，我们将要下载“订单和出货量”报告，因此请从下拉菜单中选择该选项。
  </p>
  <p class="graf graf--p">
   然后，可以为要查看的数据选择开始日期和结束日期。要查看所有 Amazon 存储的数据，需要将开始日期设置为2006年1月1日。然后点击“结束日期”旁边的 “Use today” 按钮，以自动填写今天的日期。
  </p>
  <p class="graf graf--p">
   （如果您希望查看的订单记录不是全部，则页面右侧还会显示“快速设置选项”，可用于快速输入想要查看的时间段，例如近一年或最近一个月）
  </p>
  <p class="graf graf--p">
   还可以选择在报告中添加名称 —— 这不是必需的，可以将其保留为空白，但是如果您打算下载许多不同的报告并希望轻松区分它们，那就这样做。
  </p>
  <p class="graf graf--p">
   一切准备就绪后，单击 “Request Report”，Amazon 将开始为您构建报告。
  </p>
  <p class="graf graf--p">
   您要求的数据越多、订单越多，花费的时间可能就会更长。但是在一两分钟之内就应该会弹出一个.csv文件的下载消息。请单击“保存”进行下载。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    3、重命名和移动 Amazon Order 数据
   </span>
  </p>
  <p class="graf graf--p">
   准备开始进行一些编程，然后进行分析。但是首先，请通过重命名和移动刚下载的文件使下面的工作变得简单一些。
  </p>
  <p class="graf graf--p">
   默认情况下，它可能称为
   <code>
    01_Jan_2006_to_10_Sept_2019.csv
   </code>
   但你可以更简单地重命名它为
   <code>
    amazon-orders.csv
   </code>
   .
  </p>
  <p class="graf graf--p">
   接下来将 amazon-orders.csv 文件移动到安装 Anaconda 的文件夹中，以便可以从 Jupyter Notebooks 轻松访问它。
  </p>
  <p class="graf graf--p">
   如果您在安装过程中未更改默认文件夹，则在 Windows 上为
   <code>
    C:/Users/YourUsername/
   </code>
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    4、打开一个 Jupyter Notebook
   </span>
  </p>
  <p class="graf graf--p">
   现在终于到了有趣的部分了！打开 Anaconda Navigator 应用程序。您可以通过在计算机上搜索 “Anaconda Navigator” 来找到它。
  </p>
  <p class="graf graf--p">
   在 Windows 上，您也可以在“开始”菜单中找到它；在 Mac 上，它应该在“应用程序”文件夹中。打开后，您会看到一个类似以下的屏幕：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10084 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4.jpg?resize=1100%2C824&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4.jpg?w=1161&amp;ssl=1 1161w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4.jpg?resize=300%2C225&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4.jpg?resize=768%2C576&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4.jpg?resize=1024%2C767&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4.jpg?resize=1100%2C824&amp;ssl=1 1100w" data-recalc-dims="1" height="824" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/4.jpg?resize=1100%2C824&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   单击 Jupyter Notebook 下的“启动”按钮。这将在浏览器中打开一个新屏幕。
  </p>
  <p class="graf graf--p">
   在该屏幕的右上角，单击“新建”，然后在下拉菜单的“笔记本”下，单击 “ Python 3”。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-10085 jetpack-lazy-image" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/5.gif?resize=900%2C225&amp;is-pending-load=1#038;ssl=1" data-recalc-dims="1" height="225" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/5.gif?resize=900%2C225&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="900"/>

    <img alt="" class="aligncenter size-full wp-image-10085" data-recalc-dims="1" height="225" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/5.gif?resize=900%2C225&amp;ssl=1" width="900"/>
   </noscript>
   <img alt="" class="aligncenter size-full wp-image-10085 jetpack-lazy-image" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/5.gif?resize=900%2C225&amp;is-pending-load=1#038;ssl=1" data-recalc-dims="1" height="225" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/5.gif?resize=900%2C225&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="900"/>

   </noscript>

  <p class="graf graf--p">
   已经准备就绪了！下面就进入分析。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     我在亚马逊上花了多少钱？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   Jupyter Notebooks 使您可以在单元格中编写和运行少量代码片段。在屏幕上，菜单和按钮下方，您会看到一个小方块，
   <code>
    In [ ]:
   </code>
   光标的闪烁，这是第一个单元格，是开始编写代码的地方。
  </p>
  <p class="graf graf--p">
   在此分析中将使用 Python，这是一种非常通用的编程语言，在数据分析中很流行。
  </p>
  <p class="graf graf--p">
   还将使用称为 pandas 和 matplotlib 的 Python 软件包。您可以将软件包视为类似于浏览器插件，它们是有助于扩展常规 Python 功能的工具。
  </p>
  <p class="graf graf--p">
   Anaconda 已经安装了 Pandas 和 matplotlib，因此不需要自己下载安装。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    5、将数据导入 Pandas
   </span>
  </p>
  <p class="graf graf--p">
   第一步是编写代码以打开并查看 Amazon 数据，以便可以看到正在使用的内容。
  </p>

   不过这里还有一些数据清理问题。此数据集中的价格已存储为字符串，这意味着计算机将其视为一系列字符，而不是可以加在一起的数字 —— 它们包括$，这不是数字。

  <p class="graf graf--p">
   在使用这些值执行计算之前，需要删除那些美元符号，然后将字符串值转换为浮点数（Python 可以轻松对其进行计算的数字数据类型）。幸运的是，这些操作仅需要几行代码就可以做到。

  <p class="graf graf--p">
   首先，
   <code>
    Series.str.replace()
   </code>
   这允许您在数据框的任何列中用另一个字符替换一个字符或一组字符；
  </p>
  <p class="graf graf--p">
   就本文的目的而言，这很有用，因为我们可以使用以下语法
   <code>
    df["Total Charged"].str.replace('$','')
   </code>
   。
  </p>
  <p class="graf graf--p">
   也就是告诉 pandas 在该列中将 $ 去掉。这样就能剩下的全都是数字了。然后就可以使用函数 .astype（float）将列中的每个条目转换为浮点类型，以便进行数学运算。

    df [“ Total Charged”]
   </strong>
   来指定仅在该列中进行这些更改。

2019-04-08  44.99
Name: Total Charged, dtype: float64</code></pre>
  <p class="graf graf--p">
   如您所见，.head（）的结果在这里看起来有些不同。那是因为这是一个 Series 而不是 DataFrame。
  </p>
  <p class="graf graf--p">
   确切了解其工作原理对于本教程来说并不重要，但是如果您向上滚动至以前的 df.head（）打印输出之一，则可能会注意到表格的最左侧是：0、1、 2、3、4 … 这些数字是索引标签。
  </p>
  <p class="graf graf--p">
   幸运的是，pandas 能够很好地绘制以日期为索引标签的 Series，因此您无需对数据进行任何其他操作。
  </p>
  <p class="graf graf--p">
   如果检查之前的 DataFrame 中的数字可以看到这里的总和是正确的。2019年4月1日有3个单独的订单，分别为 $ 36.93，$ 16.99 和 $ 9.99。这三者的总和为 $ 63.91，即4月1日的 day_orders 值。完美。
  </p>
  <p class="graf graf--p">
   是时候将其制作成条形图了。可以使用 Series.plot.bar（）做到这一点，并且由于该 Series 只有一列和一组索引标签，因此甚至不需要定义x和y轴，他会自动执行此操作。
  </p>
  <pre class="graf graf--pre"><code>daily_orders.plot.bar(figsize=(20,10))</code></pre>
  <pre class="graf graf--pre"><code>&lt;matplotlib.axes._subplots.AxesSubplot at 0x1c3110bbf60&gt;</code></pre>

   </noscript>

  <p class="graf graf--p">
   现在已经有了想要绘制的图表，但是您还可以做出另一个快速的改进。
  </p>
  <p class="graf graf--p">
   此图表中的不同颜色具有误导性。每个小节实际上代表的是同一事物的价值（当天亚马逊向您收取的订单总额）。但是由于每个条形都是不同的颜色，因此查看此图表的人可能会认为颜色具有一定的含义。
  </p>
  <p class="graf graf--p">
   您可以使用另一个参数（例如 figsize）来更改此设置。例如，在绘图代码中添加参数
   <strong class="markup--strong markup--p-strong">
    color =“ blue”
   </strong>
   将使图表中的每个条形都变为蓝色。
  </p>
  <p class="graf graf--p">
   该参数可以接受“蓝色”之类的基本颜色，但也可以接受其他格式的颜色信息，包括您可能在网络上经常能看到的十六进制代码。
  </p>
  <p class="graf graf--p">
   把它改成一些有趣的颜色，比如 绿色的十六进制代码是 #61D199，那么就是：
   <code>
    color='#61D199'
   </code>
   .
  </p>
  <p class="graf graf--p">
   将其放入代码中，看看它的外观！
  </p>
  <pre class="graf graf--pre"><code>daily_orders.plot.bar(figsize=(20, 10), color='#61D199')</code></pre>

  <p class="graf graf--p">
   这就漂亮很多了。
   <strong class="markup--strong markup--p-strong">
    这张图表展示了以前的分析中所没有的东西
   </strong>
   ：尽管最大的单笔订单刚好超过200美元，但在7月的一天中，有几笔订单的总金额超过了400美元。
  </p>
  <p class="graf graf--p">
   显然，您仍然可以通过很多方法来改进此图表。但是这里的目标不是创建视觉完美的图表，而是，演示
   <strong class="markup--strong markup--p-strong">
    如何使用一些快速编程以从大数据集中挖掘重要见解
   </strong>
   的方法。现在您已经了解到：
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    在此期间，我在亚马逊上花了多少钱。
   </li>
   <li class="graf graf--li">
    最高、最低和平均订单总额是多少。
   </li>
   <li class="graf graf--li">
    缴纳了多少税，以及有效税率。
   </li>
   <li class="graf graf--li">
    指出随时间波动的情况。
   </li>
   <li class="graf graf--li">
    哪天花的钱最多。
   </li>
  </ol>
  <p class="graf graf--p">
   只用了不到三十行的代码就可以完成所有这些工作。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    如何使这个项目更进一步
   </span>
  </h3>
  <p class="graf graf--p">
   想更深入吗？您可以根据自己的兴趣尝试以下几种方法来进一步扩展该项目：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    查看其他参数以添加到绘图代码中，进一步美化图形。
   </li>
   <li class="graf graf--li">
    在其他栏添加每一天的 Tax Charged。
   </li>
   <li class="graf graf--li">
    按月份或年份（而不是一天）细分您的支出。
   </li>
   <li class="graf graf--li">
    寻找一种方法来获取每天（包括没有订单的日子）到您的数据集中，以便可以更轻松地直观显示一段时间内的支出。
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    您还通过哪些其他网站购物吗？淘宝？京东？查看它们是否允许您访问任何数据，您可以做出同样的分析
   </strong>
   。
  </p>
  <p class="graf graf--p">
   基本的数据分析方法也是开源调查报告的刚需，比如分析网络水军的动态等等，希望本文的基本演示能帮助您理解这一分析方法。⚪️
  </p>
