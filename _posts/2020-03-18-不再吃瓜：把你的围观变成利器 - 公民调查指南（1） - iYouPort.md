  <p class="graf graf--p graf--startsWithDoubleQuote">
   “围观能改变” 什么？可能什么都改变不了，如果你仅仅是围观的话。
  </p>
  <p class="graf graf--p">
   当权者并不担心公民的 “围观”，他们担心的是公民由此发现真相。
  </p>
  <p class="graf graf--p">
   那么如何发现真相才能令 “围观” 具有真正的价值？
  </p>
  <p class="graf graf--p">
   开启一个新的系列内容，我们希望能回答上述问题。
  </p>
  <p class="graf graf--p">
   您每天都在围观 —— 您花钱购买手机流量、定期缴纳上网费用、把自己任何空闲的时间都贡献给 “围观” 了，您得到了什么？您关心的问题最终获得满意答案了吗？
  </p>
  <p class="graf graf--p">
   那么为什么不亲手改变这点呢？如何改变？
  </p>
  <p class="graf graf--p">
   唯一的方法是真相。尽可能多的真相。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当权者最害怕的是真相，而不是问题的曝光 —— 因为只要能让你相信他们能控制局面，他们就可以继续掩盖对他们来说最致命的真相；甚至借机进一步操控社会
   </strong>
   （利用问题/灾难笼络社会凝聚力是当权者最擅长的手段）。
  </p>
  <p class="graf graf--p">
   真相不一定是灵丹妙药，但是如果没有真相，任何问题都无从解决。
  </p>
  <p class="graf graf--p">
   是的，每一位公民都可以通过调查挖掘到真相，全世界的公民都能做。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当权者可以收买和控制职业媒体（他们做得很成功），但是他们无法控制整个社会的每一位公民
   </strong>
   。这就是为什么
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%aa%92%e4%bd%93%e5%92%8c%e6%96%b0%e9%97%bb%e8%87%aa%e7%94%b1/" href="https://www.iyouport.org/category/%e5%aa%92%e4%bd%93%e5%92%8c%e6%96%b0%e9%97%bb%e8%87%aa%e7%94%b1/" rel="noopener noreferrer" target="_blank">
    我们的导语是
   </a>
   “如果职业记者不行，那么公民记者就应该很行”。
  </p>
  <p class="graf graf--p">
   正因此，我们希望
   <strong class="markup--strong markup--p-strong">
    将曾经一直以来只有专业调查记者才能掌握的能力，普及给每一位公民
   </strong>
   —— 拥有调查技能的人越多，真相就越藏不住。
  </p>
  <p class="graf graf--p">
   不论是普通公民、非政府组织的成员、或非新闻专业的任何人，只要您希望走出 Matrix、希望看到那些强大的控制者不想让你看到的东西，您就应该掌握这些技能。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    Don’t Miss It 《
    <strong class="markup--strong markup--li-strong">
     如何从 Matrix 中醒来？
    </strong>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   世界各地都有许多令人印象深刻的公民调查成功案例，其中一些也将包含在本指南中。为所有人提振士气。
  </p>
  <p class="graf graf--p">
   数字化的普及和互联网访问的增加意味着受过专业培训的记者和公民调查员之间的界线越来越模糊，因此我们可以预期公民调查的数量将会大幅增加。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     调查是一个可以学习的技能
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   曾经一直以来，职业记者才有机会学习如何进行调查。他们花了学费，有些甚至价格不菲。
  </p>
  <p class="graf graf--p">
   而如今，所有公民都可以进行调查 —— 完全免费。
  </p>
  <p class="graf graf--p">
   这大概是互联网技术最具民主价值的（唯一）用例了。
  </p>
  <p class="graf graf--p">
   公民调查员通常从一个非常重要的优势开始 —— 动力。
  </p>
  <p class="graf graf--p">
   他们的动力可能基于特定的刺激、对政权的怀疑、或特殊的技术兴趣，等等。不论如何，目标就是优势。
  </p>
  <p class="graf graf--p">
   公民调查员的另一个优势是拥有本地经验或专业知识。思考您的想法并评估目标至关重要。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    👉从一个问题开始为研究奠定了基础
   </strong>
   。之后就是基本步骤了，包括：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    研磨问题
   </li>
   <li class="graf graf--li">
    制定一个寻求真相的计划
   </li>
   <li class="graf graf--li">
    进行研究
   </li>
   <li class="graf graf--li">
    汇总和评估您的所有发现
   </li>
   <li class="graf graf--li">
    总结您的结论。
   </li>
  </ul>

    如何掌握调查的心智

  <p class="graf graf--p">
   我们将为上述所有方面提供技巧建议，这些经验来自于全球诸多优秀的调查记者。
  </p>
  <p class="graf graf--p">
   👉和所有行动一样 —— 就如我们在 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/4%e3%80%817%e4%b8%aa%e4%bb%a5%e6%8a%b5%e6%8a%97%e4%b8%ba%e4%b8%bb%e9%a2%98%e7%9a%84%e6%a3%8b%e7%9b%98%e6%b8%b8%e6%88%8f%ef%bc%8c%e5%8d%87%e7%ba%a7%e4%bd%a0%e7%9a%84%e6%88%98%e6%96%97%e6%8a%80%e5%b7%a7/" href="https://www.iyouport.org/4%e3%80%817%e4%b8%aa%e4%bb%a5%e6%8a%b5%e6%8a%97%e4%b8%ba%e4%b8%bb%e9%a2%98%e7%9a%84%e6%a3%8b%e7%9b%98%e6%b8%b8%e6%88%8f%ef%bc%8c%e5%8d%87%e7%ba%a7%e4%bd%a0%e7%9a%84%e6%88%98%e6%96%97%e6%8a%80%e5%b7%a7/" rel="noopener noreferrer" target="_blank">

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     心态和准备问题
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   👉提出问题是发现答案的跳板。正确的问题既可以激发也可以约束调查过程。
  </p>
  <p class="graf graf--p">
   不一定是完美的问题。毕竟，您正在寻找未知的事物；它也不必很精确，但肯定不能太笼统。
  </p>
  <p class="graf graf--p">
   您真正想知道什么？你在找什么？——这个主问题可以帮助您前进。
  </p>
  <p class="graf graf--p">
   从总体问题来看，子问题很可能会接连出现。您甚至可以设定多个备选的假设。
  </p>
  <p class="graf graf--p">
   👉请注意有假设与有偏见是不同的。如果调查的依据是错误的假设或个人偏见，那么调查的基础将不会稳定 ——
   <strong class="markup--strong markup--p-strong">
    如果对正在发生的事的认识太过狭隘，很可能会阻止您看到真正的原因和事实
   </strong>
   。
  </p>

    </strong>

     <strong class="markup--strong markup--li-strong">
      什么是调查？
     </strong>
    </a>
    <strong class="markup--strong markup--li-strong">
     》这一解析非常重要
    </strong>
   </li>
  </ul>
  <p class="graf graf--p">
   因此，⚠️您的调查过程应该是质疑自己的假设、而不是试图去证实它。如果不断浮出水面的证据指向了另一个方向，那么您必须修改自己的假设。无论它多么违背您的基本立场 —— 它更可能是事实。
  </p>
  <p class="graf graf--p">
   如果没有证据足够能推翻您的假设，那么您的假设就是事实。
  </p>

   <strong class="markup--strong markup--p-strong">
    继续保持怀疑的态度，并在证据出现时保持开放的态度，至关重要。
   </strong>
  </p>
  <p class="graf graf--p">
   对调查主题存有政治观点或个人立场，非常常见，而这正是令很多调查失败的根本性原因。请记住，我们要的是事实真相，而不是去捍卫谁的立场。
  </p>
  <p class="graf graf--p">
   👉与您的调查报告相矛盾的任何证据都需要包含在您的分析中。请记住，调查可能不会总是将您带到您认为应该的地方，但这并不是一件坏事。
  </p>
  <p class="graf graf--p">
   有时事实会引导您走向意外之路。还要记住，世界是一个复杂的地方，坏人并不总是那么坏，好人也不总是那么好。

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     开始您的研究
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   👉在此开始阶段，请盘点一下您不知道什么 —— 那就是您需要查找的内容。把它们记下来。
  </p>
  <p class="graf graf--p">
   进行深入的背景研究是有回报的。
  </p>
  <p class="graf graf--p">
   许多调查人员喜欢在调查目标人之前先看资料；在面对面采访之前先钻研文件。但是例外依旧有用，例如从专家那里获得基础知识，以防止被目标人蒙骗。
  </p>
  <p class="graf graf--p">
   👉探索有关您的调查主题的文章。在阅读时，请尝试了解以下内容：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    该主题的基本事实
   </li>
   <li class="graf graf--li">
    相关的历史记录和数据
   </li>
   <li class="graf graf--li">
    该主题领域的常用语和它们的定义
   </li>
   <li class="graf graf--li">
    该领域的关键行为者是谁
   </li>
  </ul>
  <p class="graf graf--p">
   记住经典的：Who? What? Where? When? why?
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     要问的问题
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   继续问（自己）以下问题：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    公共文件/数据是否存在以及在哪里可以找到？
   </li>
   <li class="graf graf--li">
    可能存在哪些**非公开**的文件/数据？
   </li>
   <li class="graf graf--li">
    文件和数据存放在哪里？
   </li>
   <li class="graf graf--li">
    谁参与其中？
   </li>
   <li class="graf graf--li">
    这些参与者之间是什么关系？（可以创建地图）
   </li>
   <li class="graf graf--li">
    存在哪些既得利益？
   </li>
   <li class="graf graf--li">
    关于这件事人们能告诉你什么？
   </li>
   <li class="graf graf--li">
    事件实际上在哪里/哪个层面发生？
   </li>
   <li class="graf graf--li">
    有什么后果？
   </li>
   <li class="graf graf--li">
    谁受益？谁受损？
   </li>
  </ul>
  <p class="graf graf--p">
   当您深入研究时，您无疑会完善一系列目标。
  </p>
  <p class="graf graf--p">
   👉现在要做的就是制定计划的后续步骤时间表，确定优先级。
  </p>
  <p class="graf graf--p">
   许多研究人员更喜欢圈出话题，在与关键人物面对面之前收集充分的证据 —— 要问到 TA 哑口无言。
  </p>
  <p class="graf graf--p">
   👉考虑您的日程安排。考虑哪些调查工作可能需要更长的时间，例如 索取文件、数据分析、追踪，等等。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     正式走访
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   访谈是调查人员工具箱中最有效的工具之一。
  </p>
  <p class="graf graf--p">
   👉做好准备和认真倾听是成功的关键。采访是一种类似于良好对话的艺术。关于这点我们会在未来单独介绍（如果您感兴趣的话）。以下是您首先要了解的步骤：
  </p>
  <p class="graf graf--p">
   1、
   <strong class="markup--strong markup--p-strong">
    做好准备
   </strong>
   ：此步骤首先考虑您的采访对象可能知道什么，他或她的动机是什么、以及您想从中了解什么。选择一个适合受访者的场所（而不是您自己方便的场所），让他或她放心，从基础上建立信任。
  </p>
  <p class="graf graf--p">
   2、
   <strong class="markup--strong markup--p-strong">
    基本规则
   </strong>
   ：说明您是谁，对什么感兴趣以及为什么，打算如何处理这些信息、同意如何使用信息，如何归因于此？会使用此人的真实名字吗？考虑是否可能损害其安全性。
  </p>
  <p class="graf graf--p">
   3、
   <strong class="markup--strong markup--p-strong">
    好的问题
   </strong>
   ：写下您的问题并合理地组织它们是明智的。一些专业人士喜欢从相对较软的问题开始，制造一个宽松的气氛。考虑中性的开放性的问题，但不要回避针对性的询问。
  </p>
  <p class="graf graf--p">
   👉避免使用那种只回答“是”或“否”的问题；应该专注于诸如 “如何”、“为什么”和“有什么” 之类的问题。简短的单一主题问题是很好的选择。您必须认真倾听以便提出后续问题。
  </p>
  <p class="graf graf--p">
   在可能的情况下录音，或者做笔记。使用相机，包括用于文档拍摄的相机。
  </p>
  <p class="graf graf--p">
   👉准确记录研究发现是您要做的最重要的任务之一。
  </p>
  <p class="graf graf--p">
   您的信誉将取决于您的信息和文档的可靠性。
  </p>
  <p class="graf graf--p">
   👉重要的是考虑如何保护自己和资源不受伤害。基本步骤包括强密码和加密电话。

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     做研究：调查线索
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   调查人员最常犯的错误是没能彻底挖掘已经公开的全部相关信息。不要重新发明轮子！
  </p>
  <p class="graf graf--p">
   👉搜索新闻报道、学术报告、政府记录、本地档案、以及其他有可能存在相关信息的任何来源，直到您确定对那里已有的内容已经充分地了解。然后，可以在此基础上进行构建。
  </p>
  <p class="graf graf--p">
   现代技术会刺激人们产生一种 “先上网搜” 的冲动，这也许很好；但是，作为调查者您应该记住，并非所有内容都在线。例如，某些政府文件可能只能在政府办公室实际获得。
  </p>
  <p class="graf graf--p">
   👉此外，在线内容并非都可信，因此
   <strong class="markup--strong markup--p-strong">
    请仔细评估来源是否可信、以及找到科学的方法验证信息 —— 这至关重要
   </strong>
   。
  </p>
  <p class="graf graf--p">
   👉考虑可能存在所需信息的替代位置：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    除了国家政府机构的记录之外，是否还有法院记录或地方记录？
   </li>
   <li class="graf graf--li">
    谁可能收集了资料并拥有相关档案？社区活动家、业余爱好者、学者、和专业协会，都是富有成果的地方。
   </li>
   <li class="graf graf--li">
    搜寻报纸和杂志的期刊通常很有效。
   </li>
   <li class="graf graf--li">
    与图书馆管理员、档案室管理员搞好关系。
   </li>
  </ul>
  <p class="graf graf--p">
   👉好的探索需要想象力。
  </p>
  <p class="graf graf--p">
   如果一条路径被阻塞，请寻找其他相关信息来源。如果找不到所需的文档，问自己还可以如何获取信息。请保持灵活、敏捷、机智。
  </p>
  <p class="graf graf--p">
   👉本指南全面建立在 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    开源调查
   </a>
   ” 基础之上，也就是说，开源调查应该是您的基本功，而
   <strong class="markup--strong markup--p-strong">
    本指南是在您拥有这份基本功的前提下正式进入调查工作的大门。请仔细阅读 “
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">

   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     尽职调查
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   当您进行研究时，请保持谨慎。正如金融界那帮家伙所说的那样：通过确保文件真实可靠并警惕事实矛盾和矛盾之处来行使“应有的义务”。
  </p>
  <p class="graf graf--p">
   👉在整个研究过程中，请保持警惕。您听到的可能不完全都是正确的。
  </p>
  <p class="graf graf--p">
   事实的准确性需要经过科学的确认。应该对消息人士的话保持怀疑。可能需要验证照片；在最坏的情况下，文档可能都是欺诈性的 ——
   <strong class="markup--strong markup--p-strong">
    验证，不断的验证至关重要。
   </strong>
  </p>
  <p class="graf graf--p">

  </p>

   👉专注于最初的目标可以帮您提振精神。在此阶段，您可能需要起草新的问题和假设 —— 随时注意新动态的出现。

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     来源
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   除非有价值连城的文件，否则大多数调查都需要具有可靠来源和相关领域专业人士的直接知识和观点。
  </p>
  <p class="graf graf--p">
   👉通过开源调查并与人们交谈，确定可能的来源。一些想法包括：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    曾就此话题发表过演讲或写作的专家；
   </li>
   <li class="graf graf--li">
    官员，但是不要只想到高层官员，有时候底层知道得一点不少；
   </li>
   <li class="graf graf--li">
    受影响的各方；
   </li>
   <li class="graf graf--li">
    相关领域的倡导者、与该主题相关的公民组织成员，例如 民间团体或专业团体；
   </li>
   <li class="graf graf--li">
    以前与该主题相关的人员，例如前雇员。
   </li>
  </ul>
  <p class="graf graf--p">
   👉考虑可能的关系：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    谁受到不利影响、谁受到有利影响？
   </li>
   <li class="graf graf--li">
    谁获胜？
   </li>
   <li class="graf graf--li">
    谁失败？
   </li>
   <li class="graf graf--li">
    谁可能听到或看到过什么？
   </li>
   <li class="graf graf--li">
    谁有兴趣对此多说几句？
   </li>
  </ul>
  <p class="graf graf--p">
   一些研究人员创建了定义上述关系源的图谱。电子表格也可用于保留可能来源的列表。
  </p>
  <p class="graf graf--p">
   👉您使用的来源数量对于信息的完整性来说至关重要。请注意两个完全不相关的来源被认为是最小值，但
   <strong class="markup--strong markup--p-strong">
    更多则更好
   </strong>
   。
  </p>
  <p class="graf graf--p">
   更大的资源库不仅可以确保信息的可靠性，而且通常还意味着更多的细节和细微差别。您可能亲眼目睹了某些东西，这也是很有用的。
  </p>
  <p class="graf graf--p">
   👉对来源进行研究。“专家”真的有良好的信誉吗？可能是既得利益者吗？他们是谁？他们的议程是什么？ “目击者”真的看到了该事件吗，还是只听说了该事件？确认他们的陈述。
  </p>
  <p class="graf graf--p">
   👉来源可能需要一段时间的培养，特别是在涉及敏感的调查对象时。对他们的恐惧要所准备。
  </p>
  <p class="graf graf--p">
   与消息来源的会面很可能不止一次，来源是否能给你真实的东西通常需要建立在信任的基础上，所以，整个过程您都需要
   <strong class="markup--strong markup--p-strong">
    培养信任
   </strong>
   。
  </p>
  <p class="graf graf--p">
   做好笔记。在可能且合法的情况下，录音和录像，但是也要记录笔记。
  </p>
  <p class="graf graf--p">
   👉除了与人交谈外，在适当的时候进行实地访问非常重要。 “实地考察”有时会产生意想不到的结果。而且在现场您可能会遇到可以帮助您的人。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     记录您的研究
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   无论您是在网上查找还是在现实中与人交谈，保持良好的记录都是至关重要的。
  </p>
  <p class="graf graf--p">
   👉确保您可以指向信息的来源并证明其准确性，保留研究和联系人的记录。
  </p>
  <p class="graf graf--p">
   保留您的研究记录意味着保留文件并知道它们来自何处。在线文档随时可能会消失，请通过打印或下载来防止这种可能性。这意味着要跟踪您从谁那里以及何时听到的声音，并记下详细的笔记。
  </p>
  <p class="graf graf--p">
   👉保持您的调查结果井井有条。不论是纸质还是数字形式，要有准确的标签和仔细的归档。
  </p>
  <p class="graf graf--p">
   整理得井井有条的材料档案将使您更容易找到自己知道的内容、并定义您不知道的内容。建议创建一个主文件，以作为您材料所在位置的索引。
  </p>

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%bc%94%e7%a4%ba%ef%bc%9a%e5%af%b9-elon-musk-%e6%af%94%e7%89%b9%e5%b8%81%e8%af%88%e9%aa%97%e7%9a%84%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5/" href="https://www.iyouport.org/%e6%bc%94%e7%a4%ba%ef%bc%9a%e5%af%b9-elon-musk-%e6%af%94%e7%89%b9%e5%b8%81%e8%af%88%e9%aa%97%e7%9a%84%e5%bc%80%e6%ba%90%e8%b0%83%e6%9f%a5/" rel="noopener noreferrer" target="_blank">
     演示：对 Elon Musk 比特币诈骗的开源调查
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   👉下一步重新考虑事实差距、并确保事实是正确的和有据可查的。正确的名称和位置都很重要，小错误会削弱信誉。
  </p>
  <p class="graf graf--p">
   👉在编写时以及随后进行编辑时，请问自己：这份调查报告是否能使初次阅读的读者可以清楚地理解事情的来龙去脉？这个故事对读者有意义吗？哪方面的意义最重大？突出出来。
  </p>
  <p class="graf graf--p">
   👉考虑减少无关的语言和材料。编辑人员经常谈论
   <strong class="markup--strong markup--p-strong">
    节奏问题
   </strong>
   —— 也就到“叙事”是否能带动读者。照片、图表和其他可视化效果也有助于讲述故事。
  </p>
  <p class="graf graf--p">
   👉完成作品的最后几天通常意味着仔细检查所有引述的事实，并与您以前不想与之交谈的调查主题相关的人进行交流。
  </p>
  <p class="graf graf--p">
   让熟悉和不熟悉该主题的人员认真阅读您的报告。听取他们的意见。
  </p>
  <p class="graf graf--p">
   👉作为一名调查员，您可以不受检察官的高法律标准的约束，例如 “超出合理怀疑范围的证据” 或 “概率均衡” 是否以一种或另一种方式提示。但是，
   <strong class="markup--strong markup--p-strong">
    如果您的调查报告准备提交法庭、立法者和检察官，这些都必须考虑周全
   </strong>
   。
  </p>
  <p class="graf graf--p">
   👉继续查看。您的报告最后是否包含有可能被指控诽谤的信息？您的调查可能会有什么样的后果？考虑您自己的安全以及所有受影响者的安全。要尽可能让倒打一耙无法施展。
  </p>
  <p class="graf graf--p">
   总的来说，您是否满意调查结果的公平性？你能始终捍卫它吗？你必须能。
  </p>
  <p class="graf graf--p">
   调查很少是终极的。那么，作为调查员或倡导者，您可以如何跟进？
  </p>
  <p class="graf graf--p">
   以上是本系列的第一部分内容。接下来我们将介绍更多详细的知识。下次见。⚪️
  </p>
