  </p>
  <p class="graf graf--p">
   对 Instagram 进行开源情报调查的工具也有很多，其中 Jake Creps 介绍的 InstaLoader 的确是很不错的一种。它在提供大量数据的同时又可以保持用户友好性。您可以在这里看到原文《
   <a href="https://jakecreps.com/2019/07/12/instaloader/" rel="noopener noreferrer" target="_blank">
    InstaLoader – an OSINT Tool for Scraping Instagram Metadata
   </a>
   》
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    InstaLoader 执行以下操作
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    下载公开的和私有的个人信息、标签、用户故事、提要和保存的媒体；
   </li>
   <li class="graf graf--li">
    下载每个帖子的评论、地理标记和标题；
   </li>
   <li class="graf graf--li">
    自动检测个人资料名称是否有更改，并相应地重命名目标（也就是说目标即便改了名字也可以继续追踪）；
   </li>
   <li class="graf graf--li">
    允许您对过滤器以及存储下载的媒体的位置进行精细的自定义。
   </li>
  </ul>
  <p class="graf graf--p">
   InstaLoader 可让您提取主题标签、用户发布的内容、提要、标题、和保存的媒体。这是进行社交媒体调查的基础。
  </p>
  <p class="graf graf--p">
   👉这些信息很重要，追踪者和调查人员可以基于这些信息结合更多公开信息，以掌握目标人的几乎一切。
  </p>
  <p class="graf graf--p">
   接下来，您可以下载每个帖子的评论和地理标记。
   <strong class="markup--strong markup--p-strong">
    每个帖子的评论都是线索
   </strong>
   ，我们绝大多数人不会在网上无缘无故地给他人留评论对吧？于是对于追踪者来说，每一个评论都意味着 “下一条线索”。
  </p>
  <p class="graf graf--p">
   👉 如果用户对个别评论有真实的反应，那么你就可以 “透视” 该个人资料，并重复 InstaLoader 流程以构建您的调查网络。
  </p>
  <p class="graf graf--p">
   您可以潜在地从目标个人资料发现外部用户的活动。关于这一点，我们可以先讨论一个调查 Instagram 的独特问题。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     关于 Instagram
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   对开源情报调查人员来说，Instagram 存在一个问题，例如 Twitter 之类的其他社交媒体则没有这样的问题 —— 即：您无法在其个人资料页面之外看到用户活动。
  </p>
  <p class="graf graf--p">
   也就是说，您可以看到用户发布的内容，但是无法搜索他们与之交互的其他内容。
   <strong class="markup--strong markup--p-strong">
    显然，这在识别欺诈、僵尸网络帐户、潜在的人口贩子等目标的时候，就会出现问题
   </strong>
   。
  </p>
  <p class="graf graf--p">
   没办法，它就是这么设计的。由于这个问题，您需要获取尽可能多的数据，才能编写有效的报告并得出合理的结论。
  </p>
  <p class="graf graf--p">
   通过分析用户生成的内容下的评论来进行延展调查，是一种解决方案。
  </p>
  <p class="graf graf--p">
   另一种方法是分析用户遵循的主题标签，以查找外部活动，该主题是公开可用的。一旦找到另一个与之交互的个人资料，就可以在该个人资料中搜索更多信息。
  </p>
  <p class="graf graf--p">
   下面具体说说这个工具。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     InstaLoader 设置
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   InstaLoader 是我使用过的最简单的工具之一。设置仅两个步骤：
  </p>
  <pre class="graf graf--pre">$ pip3 install instaloader

$ instaloader profile [profile ...]</pre>
  <p class="graf graf--p">
   仅此而已。由于它位于 PyPi 上，因此您可以使用 pip 进行安装，即可开始使用。这里没有更多要说的。
   <a class="markup--anchor markup--p-anchor" data-href="https://instaloader.github.io/" href="https://instaloader.github.io/" rel="noopener noreferrer" target="_blank">
    这是该工具资料的链接
   </a>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     用例
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   作为一般免责声明：您绝不应该出于恶意或令人毛骨悚然的原因使用 InstaLoader。
  </p>
  <p class="graf graf--p">
   与大多数工具一样，好人和坏人都可以使用它来谋取私利。你别做坏人！
  </p>
  <p class="graf graf--p">
   👉我们介绍所有工具都是帮助您防御的，最好的情况下是帮助没有民主经验的社会尝试民主化实践 ——
   <strong class="markup--strong markup--p-strong">
    构建公民自己的调查社区
   </strong>
   ，绝不是让您侵犯他人的。否则，您会被抓住，不开玩笑。
  </p>
  <p class="graf graf--p">
   在更高的技术水平上，您可以使用 InstaLoader 提取数据、生成档案、进行链接分析等等。
  </p>
  <p class="graf graf--p">
   这里是可以永久使用 Instaloader 的几种方法。
  </p>
  <p class="graf graf--p">
   在有关
   <a class="markup--anchor markup--p-anchor" data-href="https://jakecreps.com/2019/01/02/nigerianscam/" href="https://jakecreps.com/2019/01/02/nigerianscam/" rel="noopener noreferrer" target="_blank">
    尼日利亚王子骗局
   </a>
   的文章中，就使用了 InstaLoader 来证明一个帐号不是他们所说的帐号。
  </p>
  <p class="graf graf--p">
   因为该帐户现在已删除，所以这里简单介绍一下背景。
  </p>
  <p class="graf graf--p">
   一个声称自己是尼日利亚埃米尔王子的账户说服人们购买加密货币，这是个骗局，该欺诈账户使用了该王子的照片，以便让人们信服。
  </p>
  <p class="graf graf--p">
   第一步是确定他们冒充了哪个王子，然后证明他们是欺诈行为。
  </p>
  <p class="graf graf--p">
   为此，调查确定了王子被使用的所有照片都是公开的 —— 使用 InstaLoader 下载照片和元数据，并使用反向图像搜索和 Google Dorks 查找发布照片的文章。在下面看到：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%94%bb%e9%98%b2%e7%9a%86%e5%8f%af%e7%94%a8%ef%bc%9agoogledorking-%e9%ab%98%e7%ba%a7%e8%bf%90%e7%ae%97%e7%ac%a6%e5%ae%8c%e6%95%b4%e5%88%97%e8%a1%a8/" href="https://www.iyouport.org/%e6%94%bb%e9%98%b2%e7%9a%86%e5%8f%af%e7%94%a8%ef%bc%9agoogledorking-%e9%ab%98%e7%ba%a7%e8%bf%90%e7%ae%97%e7%ac%a6%e5%ae%8c%e6%95%b4%e5%88%97%e8%a1%a8/" rel="noopener noreferrer" target="_blank">
     攻防皆可用：GoogleDorking 高级运算符完整列表
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   发现第一个危险信号。接下来就能够使用
   <strong class="markup--strong markup--p-strong">
    与该帐户有关的相同语言
   </strong>
   来查找多个帐户，以解决这个加密货币骗局。
  </p>
  <p class="graf graf--p">
   这只是您可以处理 InstaLoader 提取的信息的一个示例。显然，它可以做很多事。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     开源情报洞察力
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   InstaLoader 可能是最强大的 Instagram 开源情报调查工具之一。
  </p>
  <p class="graf graf--p">
   尽管我喜欢它的所有功能和自定义功能，但文件输出的格式确实存在点问题。
  </p>
  <p class="graf graf--p">
   我敢肯定这是可以调整的，但也是为什么在人脸识别的演示中使用了 Instalooter 而不是 InstaLoader。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e4%bd%bf%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e6%8a%80%e6%9c%af%e4%b8%ba%e5%85%ac%e6%b0%91%e6%89%80%e7%94%a8%ef%bc%9f-%e5%bc%80%e6%ba%90%e6%83%85%e6%8a%a5%e6%96%b9%e6%b3%95/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e4%bd%bf%e9%9d%a2%e9%83%a8%e8%af%86%e5%88%ab%e6%8a%80%e6%9c%af%e4%b8%ba%e5%85%ac%e6%b0%91%e6%89%80%e7%94%a8%ef%bc%9f-%e5%bc%80%e6%ba%90%e6%83%85%e6%8a%a5%e6%96%b9%e6%b3%95/" rel="noopener noreferrer" target="_blank">
     如何使面部识别技术为公民所用？- 开源情报方法
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   更具体地说，不是将所有照片都转储到一个文件夹中，而是每张照片都有其自己的带有元数据的文件夹。这对组织非常有用，虽然也有点累。
  </p>
  <p class="graf graf--p">
   由于 Instagram 由 Facebook 拥有，因此 Instagram 将来可能会收到类似的隐私保护措施；因此，很可能使 InstaLoader 之类的工具获得成功。
  </p>
  <p class="graf graf--p">
   此外，普通用户可能会转向私有帐户，从而限制了挖掘工具的使用。这些问题都要考虑到。
  </p>
  <p class="graf graf--p">
   在围绕Web抓取工具（例如 InstaLoader）构建基础结构之前，请牢记这些注意事项。⚪️
  </p>
  <p>
   《
   <a href="https://jakecreps.com/2019/07/12/instaloader/" rel="noopener noreferrer" target="_blank">
    InstaLoader – an OSINT Tool for Scraping Instagram Metadata
   </a>
   》
  </p>
