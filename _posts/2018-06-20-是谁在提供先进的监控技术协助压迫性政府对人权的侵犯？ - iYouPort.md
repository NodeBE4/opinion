 <div class="entry-content">
  <blockquote>
   <p>
    <span style="color: #00ccff;">
     <em>
      在一党专制的国家里，几乎没有机会对面部识别系统作出有效的限制
     </em>
    </span>
   </p>
  </blockquote>
  <p class="graf graf--p graf-after--figure" id="8914">
   海康威视公司赢得了一项中国政府的招标，该招标要求在中国部分地区的每个清真寺（总共近1000 个清真寺）的入口处设置面部识别摄像机，
   <strong class="markup--strong markup--p-strong">
    这种监控严重违反宗教自由和人权
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="2ecf">
   与西方不同，在中国，宗教团体受中国政府的直接控制。而现在，这些清真寺必须允许中国政府建立并使用面部识别摄像机进行监控，使当局能够对参加宗教服务的人员加强警戒，以及搜索人们出入清真寺的时间和频率。
  </p>
  <p class="graf graf--p graf-after--p" id="548f">
   这引发了一些重要问题：
  </p>
  <ul class="postList">
   <li class="graf graf--li graf-after--p" id="1fad">
    如何允许政府使用面部识别？
   </li>
   <li class="graf graf--li graf-after--li" id="1b93">
    技术提供商（如海康威视）对面部识别的使用有何限制？
   </li>
  </ul>
  <p class="graf graf--p graf-after--li" id="9d70">
   海康威视以约 4600 万美元的价格赢得了 2017 年政府合同，在新疆的墨玉县建立了广泛的监控系统。该项目被称为“墨玉县社会面防控体系 PPP 项目“。
  </p>
  <p class="graf graf--p graf-after--p" id="57e7">
   该项目是海康威视和 Dahua 自 2016 年以来在新疆赢得的
   <strong class="markup--strong markup--p-strong">
    价值
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://ipvm.com/reports/xinjiang-dahua-hikvision" href="https://ipvm.com/reports/xinjiang-dahua-hikvision" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     12 亿美元
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    的大规模监控项目
   </strong>
   之一。媒体和活动人士指责中国正在打造一个监视国家。
  </p>
  <p class="graf graf--p graf-after--p" id="13c1">
   法新社报道说，其最初的招标包括在新疆的“再教育（洗脑）”中心使用的六台摄像机，据称中国拘留了数十万维吾尔族和其他少数民族的穆斯林公民，因此墨玉的项目备受关注。
  </p>
  <p class="graf graf--p graf-after--p" id="1f28">
   <strong class="markup--strong markup--p-strong">
    面部识别技术细节
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="bb88">
   最具侵入性的标志是面部识别技术的大量使用，因为招标需要：
  </p>
  <blockquote class="graf graf--blockquote graf-after--p" id="144e">
   <p>
    <em class="markup--em markup--blockquote-em">
     967 facial recognition cameras to be placed at the entrance of each of the 967 mosques in Moyu County. 清真寺安防子系统在入口处设置,墨玉县有 967 个清真寺, 需购置摄像机 967台
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p graf-after--blockquote" id="7677">
   面部识别相机包含以下
   <strong class="markup--strong markup--p-strong">
    技术细节
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li graf-after--p" id="7367">
    最低分辨率为1920×1080，TVL （清晰度）至少为 1100；
   </li>
   <li class="graf graf--li graf-after--li" id="527a">
    能够在 — 40 至 70 摄氏度的极端温度下运转良好；
   </li>
   <li class="graf graf--li graf-after--li" id="d16d">
    加密的 SD 卡存储量为 256 千兆字节，只能从后端访问；
   </li>
   <li class="graf graf--li graf-after--li" id="e4f3">
    能够处理“大批量”的面部数据。
   </li>
  </ul>
  <p class="graf graf--p graf-after--li" id="8493">
   然而，
   <strong class="markup--strong markup--p-strong">
    招标没有提到面部识别技术所需的准确度水平，这是一个重要的问题，意味着技术的不成熟和错误识别的几率
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="b429">
   <strong class="markup--strong markup--p-strong">
    “视频会议布道需要国家批准“
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="f098">
   该项目还包括更多针对清真寺特点的视频监控和审查系统，例如视频会议系统，用于在墨玉县民族事务委员会的“视频会议中心”播放“统一”布道。简而言之，这样的制度是为了禁止任何传道人举行自己的布道。
  </p>
  <p class="graf graf--p graf-after--p" id="fda2">
   <strong class="markup--strong markup--p-strong">
    每个清真寺有 5 台以上的政府监视摄像机
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="06b8">
   海康威视项目为清真寺增加了 840 个监视摄像头（这还不包含面部识别摄像头）这一监视系统是子弹型摄像机、半球型摄像机、以及夜间全彩球型摄像机的组合。
  </p>
  <p class="graf graf--p graf-after--p" id="9e42">
   招标估计，墨玉县 967 个清真寺中每个清真寺都已经安装了 5 个监视摄像头，总共约 4,835 个摄像头。
   <strong class="markup--strong markup--p-strong">
    那些现有的摄像机将集成到新的监控系统中，即 967 个面部识别摄像机 + 840 个圆顶型/子弹型摄像机组成的系统
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="6f42">
   <strong class="markup--strong markup--p-strong">
    海康威视在洗脑中心的项目
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="ffd4">
   海康威视的招标赢得了为墨玉县“再教育中心”提供的六台全景摄像机（独立实现大范围无死角监控的摄像机）总分辨率达到 8MP：
  </p>
  <blockquote class="graf graf--blockquote graf-after--p" id="0cba">
   <p>
    <em class="markup--em markup--blockquote-em">
     内置镜头，带有8个图像采集模块，需要输出1个主视觉效果视频和 8 个辅助视觉效果视频；系统应该能够拼接八个辅助视频，形成两组 180 度视觉效果
    </em>
   </p>
  </blockquote>
  <p>
   <img alt="" class="size-full wp-image-1679 alignleft jetpack-lazy-image" data-lazy-sizes="(max-width: 364px) 100vw, 364px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-18.jpeg?resize=364%2C387&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-18.jpeg?w=364&amp;ssl=1 364w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-18.jpeg?resize=282%2C300&amp;ssl=1 282w" data-recalc-dims="1" height="387" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-18.jpeg?resize=364%2C387&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="364"/>
   <noscript>
    <img alt="" class="size-full wp-image-1679 alignleft" data-recalc-dims="1" height="387" sizes="(max-width: 364px) 100vw, 364px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-18.jpeg?resize=364%2C387&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-18.jpeg?w=364&amp;ssl=1 364w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2018/06/0-18.jpeg?resize=282%2C300&amp;ssl=1 282w" width="364"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="86f0">
   虽然招标没有列出海康威视使用的具体模型，但与此最接近的是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hikvision.com/en/Products/Network-Camera/PanoVu-camera/DS-2DP1636ZIX-D/236(5mm)#prettyPhoto" href="https://www.hikvision.com/en/Products/Network-Camera/PanoVu-camera/DS-2DP1636ZIX-D/236%285mm%29#prettyPhoto" rel="noopener nofollow noreferrer" target="_blank">
    PanoVu 360°
   </a>
   系列，如左图所示。
  </p>
  <p class="graf graf--p graf-after--p" id="d748">
   海康威视的监视系统覆盖很多“再教育营地”，还没有其他文件显示更多的“再教育营地”使用了哪家供应商的技术支持。
  </p>
  <p class="graf graf--p graf-after--p" id="0982">
   <strong class="markup--strong markup--p-strong">
    中国大规模监视系统的“重大升级”
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="0bd4">
   虽然新疆清真寺长期以来一直受到政府的严密监控 — — 包括通过使用传统的监视摄像头，但面部识别技术的使用代表了“党国监督工作的重大升级”，对中国宗教自由的“寒蝉效应”。
  </p>
  <p class="graf graf--p graf-after--p" id="df96">
   澳大利亚拉筹伯大学（La Trobe University）教授 James Leibold 解释说：
  </p>
  <blockquote class="graf graf--blockquote graf-after--p" id="07cd">
   <p>
    <em class="markup--em markup--blockquote-em">
     在新疆清真寺中使用人脸识别技术是为了控制进入条件。该技术使当局能够迅速识别被列入黑名单的成人和青年，并阻止他们进入清真寺。该技术还允许政府保存更多的系统数据，包括谁来了、进出的频率等等，将任何反对政府的态度和个人列入禁止名单。
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p graf-after--blockquote" id="d19b">
   罗斯赫尔曼理工学院的另一位新疆专家、中国研究教授 Timothy Grose 说，他听说过其他城市，如乌鲁木齐，也为清真寺入口配备了面部识别摄像头。
  </p>
  <p class="graf graf--p graf-after--p" id="3a30">
   <strong class="markup--strong markup--p-strong">
    该项目被“机密”处理
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="a6e1">
   海康威视公司拒绝就此报道发表评论。
  </p>
  <p class="graf graf--p graf-after--p" id="a99d">
   然而，海康威视此前告诉法新社，尽管进行了
   <a class="markup--anchor markup--p-anchor" data-href="https://ipvm.com/reports/xinjiang-risk" href="https://ipvm.com/reports/xinjiang-risk" rel="noopener nofollow noreferrer" target="_blank">
    省级债务审查
   </a>
   ，但仍在继续推行墨玉项目。它拒绝提供进一步的细节，声称该项目是“机密”。（此前有报道说，DaHua 和海康威视在过去两年中已经赢得了
   <strong class="markup--strong markup--p-strong">
    超过 10 亿美元
   </strong>
   的新疆监视系统项目）
  </p>
  <p class="graf graf--p graf-after--p" id="d832">
   事实上，在欧盟，
   <strong class="markup--strong markup--p-strong">
    GDPR 的第 9 条禁止面部识别等生物识别处理技术
   </strong>
   。但也确实将当局声称的“公共利益”作为例外处理了; 不过规定了监视数据必须“尊重数据保护权的本质，并提供适当和具体的措施，以保护数据主体的基本权益”。
  </p>
  <p class="graf graf--p graf-after--p" id="0560">
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     延伸阅读：当局的“公共利益”例外，相当于令面部识别技术的管制完全失效，原因请见这篇调查报告《
    </em>
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E6%9C%80%E6%96%B0%E6%8A%A5%E5%91%8A-%E6%98%AF%E8%B0%81%E5%9C%A8%E6%8F%90%E4%BE%9B%E5%85%88%E8%BF%9B%E7%9A%84%E7%9B%91%E6%8E%A7%E6%8A%80%E6%9C%AF%E5%8D%8F%E5%8A%A9%E5%8E%8B%E8%BF%AB%E6%80%A7%E6%94%BF%E5%BA%9C%E5%AF%B9%E4%BA%BA%E6%9D%83%E7%9A%84%E4%BE%B5%E7%8A%AF-a78ed7d635fe" href="https://medium.com/@iyouport/%E6%9C%80%E6%96%B0%E6%8A%A5%E5%91%8A-%E6%98%AF%E8%B0%81%E5%9C%A8%E6%8F%90%E4%BE%9B%E5%85%88%E8%BF%9B%E7%9A%84%E7%9B%91%E6%8E%A7%E6%8A%80%E6%9C%AF%E5%8D%8F%E5%8A%A9%E5%8E%8B%E8%BF%AB%E6%80%A7%E6%94%BF%E5%BA%9C%E5%AF%B9%E4%BA%BA%E6%9D%83%E7%9A%84%E4%BE%B5%E7%8A%AF-a78ed7d635fe" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     <em class="markup--em markup--p-em">
      是谁在提供先进的监控技术协助压迫性政府对人权的侵犯？
     </em>
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     》
    </em>
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="f5f7">
   美国的大型科技公司对美国监控技术的兴趣也在增加。微软和亚马逊为警方甚至军方提供大规模监视系统 — 人脸识别技术，已经在过去的几个月内饱受抨击。
  </p>
  <p class="graf graf--p graf-after--p" id="fe1e">
   在中国这种一党专制的国家里，几乎没有机会对面部识别系统作出有效的限制。
  </p>
