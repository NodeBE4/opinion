  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     图像识别
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   目前，分析人员依靠 “图像识别” 的知识来检查和验证图像。
  </p>
  <p class="graf graf--p">
   该识别通常分为六个关键区域：
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    物理的：阴影、照明和反射
   </li>
   <li class="graf graf--li">
    几何的：消失点、图像和3D模型中的距离
   </li>
   <li class="graf graf--li">
    光学的：镜头畸变或像差
   </li>
   <li class="graf graf--li">
    图像传感器：修复了图案噪声和滤色器缺陷
   </li>
   <li class="graf graf--li">
    文件格式：元数据、文件压缩、缩略图和标记
   </li>
   <li class="graf graf--li">
    像素：缩放、裁剪、克隆或重新保存
   </li>
  </ol>

   我们的调查通常是从看不见的部分开始，而不是可以看到的部分。在这里，我们将重点关注图像中捕获的元数据（上面架构中的第5级）。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     文件格式取证：元数据
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   保存图像时，文件通常包含有关图像的数据，称为元数据。
  </p>
  <p class="graf graf--p">
   用于数码相机的可交换图像文件格式中有460多个元数据标签（
   <a class="markup--anchor markup--p-anchor" data-href="http://www.cipa.jp/std/documents/e/DC-008-2012_E.pdf" href="http://www.cipa.jp/std/documents/e/DC-008-2012_E.pdf" rel="noopener noreferrer" target="_blank">
    EXIF 2.3
   </a>
   ）。该规范可帮助相机使用可以在设备之间交换的格式，例如，确保 iPhone 照片在三星设备上正确显示。
  </p>
  <p class="graf graf--p">
   可以包括图像尺寸、位置数据、较小的缩略图、甚至包括相机的品牌和型号等等。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     确定哪个相机拍了什么照片
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在最近的调查中，我们验证了一组称为 “Byethorne鸭” 的图像。
  </p>
  <p class="graf graf--p">
   RSPCA 提供给广告商的图像显示了一只鸭子，头上插着一把刀。很快就出现了指责，有人说该照片是PS的。
  </p>

  <p class="graf graf--p">
   使用传感器模式噪声和统计方法对此进行了验证。我们使用信号处理滤波器
   <strong class="markup--strong markup--p-strong">
    从每个图像中提取了唯一的指纹
   </strong>
   ，并比较了它们彼此之间的相似程度。
  </p>
  <p class="graf graf--p">
   较高的值表示它们非常相似并且很可能相关，而较低的值表示它们不相似并且不太可能相关。
  </p>
  <p class="graf graf--p">
   当我们比较五个图像指纹中的四个时，获得的值远高于2,000。考虑到它们之间的相关性，我们可以说这些图像可能来自同一台摄像机。
  </p>
  <p class="graf graf--p">
   当我们测试第五张图像时，获得的相似度值接近于零。
  </p>

   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-11252">
    Metadata of one of the images from the left above and the image from the right. These show two separate unique image identifiers which were correlated to a phone camera firmware
   </figcaption>
  </figure>
  <p class="graf graf--p">
   唯一图像ID字段还包含相机固件编号。通过与元数据中还包含的图像和传感器大小进行交叉引用，建议的结论是使用 Samsung Galaxy S7 或 S7 Edge 捕获的前四张图像，使用 Samsung Galaxy S5 捕获的第五张图像。
  </p>
  <p class="graf graf--p">
   在元数据中还显示了拍摄图像的时间，从而提供了什么时间拍摄的以及由什么设备拍摄的时间线。
  </p>
  <figure aria-describedby="caption-attachment-11253" class="wp-caption aligncenter" id="attachment_11253" style="width: 1200px">
   <img alt="" class="size-full wp-image-11253 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-1.png?resize=1100%2C708&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-1.png?w=1200&amp;ssl=1 1200w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-1.png?resize=300%2C193&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-1.png?resize=1024%2C659&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-1.png?resize=768%2C494&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-1.png?resize=1100%2C708&amp;ssl=1 1100w" data-recalc-dims="1" height="708" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-1.png?resize=1100%2C708&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="size-full wp-image-11253" data-recalc-dims="1" height="708" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-1.png?resize=1100%2C708&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-1.png?w=1200&amp;ssl=1 1200w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-1.png?resize=300%2C193&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-1.png?resize=1024%2C659&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-1.png?resize=768%2C494&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-1.png?resize=1100%2C708&amp;ssl=1 1100w" width="1100"/>
   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-11253">
    Time the photos were taken and by which camera
   </figcaption>
  </figure>
  <p class="graf graf--p">
   由于照片是由两个不同的相机在大约一小时的时间内拍摄的，因此这些图像极不可能是伪造的。
  </p>
  <p class="graf graf--p">
   RSPCA发言人证实，它从两个不同的人那里收到了鸭子的图像，与这些调查得出的结论相符。迄今为止，还没有足够的证据确定肇事者的身份。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     如何从图像中找到一个人的位置
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   相机型号不是唯一可以从元数据中获得的东西。
  </p>
  <figure aria-describedby="caption-attachment-11254" class="wp-caption aligncenter" id="attachment_11254" style="width: 1200px">

  <p class="graf graf--p">
   这种明显的隐私问题就是为什么社交媒体通常会从上传的图像中删除元数据的原因。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     图像取证的未来
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   元数据永远不会孤立使用。
  </p>
  <p class="graf graf--p">
   对图像进行身份验证以确保其未被修改，并维护监管链，对警察来说越来越重要。
  </p>
  <p class="graf graf--p">
   协助警察的工具可能包括直接内置在相机中的审计日志或插入水印。
  </p>
  <p class="graf graf--p">
   我目前正在对以前的研究进行扩展，该研究表明每个图像传感器（实际拍摄图像的电子设备）由于其对光的反应不均匀而具有
   <a class="markup--anchor markup--p-anchor" data-href="http://www.cosy.sbg.ac.at/~uhl/mmsec/LukFriSPIE06_v9.pdf" href="http://www.cosy.sbg.ac.at/~uhl/mmsec/LukFriSPIE06_v9.pdf" rel="noopener noreferrer" target="_blank">
    唯一的指纹
   </a>
   。
  </p>
  <p class="graf graf--p">
   就如《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/facebook-%e7%9f%a5%e9%81%93%e5%a6%82%e4%bd%95%e4%bd%bf%e7%94%a8%e7%9b%b8%e6%9c%ba%e9%95%9c%e5%a4%b4%e4%b8%8a%e7%9a%84%e7%81%b0%e5%b0%98%e8%b7%9f%e8%b8%aa%e4%bd%a0/" href="https://www.iyouport.org/facebook-%e7%9f%a5%e9%81%93%e5%a6%82%e4%bd%95%e4%bd%bf%e7%94%a8%e7%9b%b8%e6%9c%ba%e9%95%9c%e5%a4%b4%e4%b8%8a%e7%9a%84%e7%81%b0%e5%b0%98%e8%b7%9f%e8%b8%aa%e4%bd%a0/" rel="noopener noreferrer" target="_blank">
    Facebook 知道如何使用相机镜头上的灰尘跟踪你
   </a>
   》。
  </p>
  <p class="graf graf--p">
   总之，下次你拍摄照片时请考虑一下它可以讲述的故事。⚪️
