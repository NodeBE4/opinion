   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     日期标志
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   突出的第二件事是，尽管所有广告看起来都像荷兰语（其中有些看起来很真实），但我可以告诉你，在荷兰，我们使用不同的日期格式。
  </p>
  <p class="graf graf--p">
   如果您不了解这点也没关系，快速搜索维基百科即可。查看有关日期写法的
   <a class="markup--anchor markup--p-anchor" data-href="https://de.wikipedia.org/wiki/Datumsformat#DIN_5008" href="https://de.wikipedia.org/wiki/Datumsformat#DIN_5008" rel="noopener noreferrer" target="_blank">

   </a>

  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10445 jetpack-lazy-image" data-lazy-sizes="(max-width: 707px) 100vw, 707px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-8.png?resize=707%2C136&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-8.png?w=707&amp;ssl=1 707w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-8.png?resize=300%2C58&amp;ssl=1 300w" data-recalc-dims="1" height="136" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/3-8.png?resize=707%2C136&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="707"/>

  <p class="graf graf--p">
   再看看
   <a class="markup--anchor markup--p-anchor" data-href="https://nl.wikipedia.org/wiki/Datum_(dagtekening)#Scheidingstekens" href="https://nl.wikipedia.org/wiki/Datum_%28dagtekening%29#Scheidingstekens" rel="noopener noreferrer" target="_blank">
    维基百科
   </a>
   页面荷兰部分，显然荷兰人几乎从未在日期中使用“点”，荷兰人更喜欢用斜杠或短线或直接连写。
  </p>
  <p class="graf graf--p">
   所以，如果它是荷兰，您应该看到的是类似 7/1/2003 或 7–1-’03。大多数时候，荷兰人确实将相同的DMY表示法用于非正式用途，就像德国人经常做的那样。我从来没有见过荷兰人在写日期的时候使用“点”。
  </p>
  <p class="graf graf--p">
   但这不是依据。
  </p>
  <p class="graf graf--p">
   <img alt="" class="size-full wp-image-10446 alignleft jetpack-lazy-image" data-lazy-sizes="(max-width: 344px) 100vw, 344px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-6.png?resize=344%2C225&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-6.png?w=344&amp;ssl=1 344w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-6.png?resize=300%2C196&amp;ssl=1 300w" data-recalc-dims="1" height="225" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-6.png?resize=344%2C225&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="344"/>
   <noscript>
    <img alt="" class="size-full wp-image-10446 alignleft" data-recalc-dims="1" height="225" sizes="(max-width: 344px) 100vw, 344px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-6.png?resize=344%2C225&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-6.png?w=344&amp;ssl=1 344w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-6.png?resize=300%2C196&amp;ssl=1 300w" width="344"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   除了这些点之外，可以在照片左侧的标语牌上找到指向德国的另一条线索。
  </p>
  <p class="graf graf--p">
   仔细看，上面有 “Januar” 这个词，这是德语“一月”的意思，荷兰语的“一月”是“ Januari”。
  </p>
  <p class="graf graf--p">
   德语和荷兰语对的“六月”是一样的，都是 “ Juni”。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     关于自行车
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   许多游客知道，阿姆斯特丹尤其以其丰富的自行车而闻名。
   <a class="markup--anchor markup--p-anchor" data-href="https://awesomeamsterdam.com/facts-about-bicycles-in-amsterdam/" href="https://awesomeamsterdam.com/facts-about-bicycles-in-amsterdam/" rel="noopener noreferrer" target="_blank">
    这里有
   </a>
   关于阿姆斯特丹自行车的一些有趣的故事。
  </p>
  <p class="graf graf--p">
   是的，这里真的有很多自行车！
  </p>
  <p class="graf graf--p">
   那么照片中显示的自行车到底表示什么呢？这一切都与德国有关安全性的规范相关，这是非常具体的，但是荷兰没有这样的规范。
  </p>
  <p class="graf graf--p">
   仔细看一下照片，可以清楚地看到只有蓝色自行车的前轮有夹式煞车器。通常前轮有后轮也应该有啊，为什么呢？
  </p>
  <p class="graf graf--p">
   这与德国在刹车方面规范的几种不同的选择有关：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    前后轮都有制动器，带有独立的把手；或者
   </li>
   <li class="graf graf--li">
    前轮有一个夹式刹车器，后轮有一个倒轮闸
   </li>
  </ul>
  <p class="graf graf--p">
   原因是德国法律明确提到自行车需要有两个独立的制动系统。也就是说，万一车把上的手动操作失灵了，必须有一个后备系统，例如一个倒轮闸。完整的德语文章可以
   <a class="markup--anchor markup--p-anchor" data-href="https://www.adfc-nrw.de/kreisverbaende/kv-rheinberg-oberberg/aktuelles/aktuelles/article/welche-bremse-und-beleuchtung-muss-am-fahrrad-sein.html" href="https://www.adfc-nrw.de/kreisverbaende/kv-rheinberg-oberberg/aktuelles/aktuelles/article/welche-bremse-und-beleuchtung-muss-am-fahrrad-sein.html" rel="noopener noreferrer" target="_blank">
    在这里
   </a>
   阅读。
  </p>
  <p class="graf graf--p">
   正因为这种配置在荷兰很少见，所以对荷兰人来说，照片中这一细节很突出。荷兰的自行车有两个手闸，或者一个倒轮闸。
  </p>
  <p class="graf graf--p">
   这里需要特别注意的是，照片中可以清楚地看到那个交通护柱，根本不像传统的阿姆斯特丹版的 “ het Amsterdammertje”。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     井盖
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在照片右侧看到一个下水道井盖？显然是混凝土的。
  </p>
  <p class="graf graf--p">
   混凝土井盖在荷兰非常少见。德国倒是有这种。在网上找到一张图，上写着德语 “Kanaldeckel”，或者更笼统的 “Schachtabdeckung”。
  </p>
  <p class="graf graf--p">
   这样的井盖下面是电缆，如果需要维修的话工人可以直接在街道上打开它。
  </p>
  <p class="graf graf--p">
   Wikipedia页面上也有同样的另一个示例：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10447 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpeg?resize=1100%2C611&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpeg?w=1300&amp;ssl=1 1300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpeg?resize=300%2C167&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpeg?resize=768%2C427&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpeg?resize=1024%2C569&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpeg?resize=1100%2C611&amp;ssl=1 1100w" data-recalc-dims="1" height="611" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpeg?resize=1100%2C611&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10447" data-recalc-dims="1" height="611" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpeg?resize=1100%2C611&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpeg?w=1300&amp;ssl=1 1300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpeg?resize=300%2C167&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpeg?resize=768%2C427&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpeg?resize=1024%2C569&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpeg?resize=1100%2C611&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在荷兰，大多数井盖都是钢甚至是铸铁制造的；但是方形的井盖在荷兰绝对是很少见的，即便有，也比照片中显示的这种要小得多，而且通常不是用混凝土制成的。
  </p>
  <p class="graf graf--p">
   综上，这不是荷兰，而是德国。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     结论
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   整体上，
   <strong class="markup--strong markup--p-strong">
    这一挑战没有使用任何工具 —— 简单说它就是您在每一次进行信息验证时最初需要做的工作，用眼
   </strong>
   。
  </p>
  <p class="graf graf--p">
   显然，有关不同国家或不同文化的有用信息是很重要的，比如井盖。再比如，如果您没有注意到日期中使用小数点的特殊格式，可能也会错过正确答案。
  </p>
  <p class="graf graf--p">
   所以，
   <strong class="markup--strong markup--p-strong">
    要细致观察。尤其是发生在您所在国家的一些细节，您都应该熟知
   </strong>
   。闭上眼睛，您能说出您所在城市使用的井盖和电线杆具有什么特点吗？您经常走过的那条街上有几个面包房？他们的招牌各是什么颜色的？……
  </p>
  <p class="graf graf--p">
   绝大多数人每天都在街上走，但是很少注意到诸如广告牌、井盖、路灯等细节，
   <strong class="markup--strong markup--p-strong">
    而在调查工作中，这些细节特别重要
   </strong>
   —— 市政建设的特点是地理定位过程中的第一步，尤其是在没有明确地标的情况下。
  </p>
  <p class="graf graf--p">
   即使您对荷兰一无所知，在地图上的一次搜索中就可以确定这不是阿姆斯特丹；最常见的失误并非是找不到线索，而是，人们已经获得了一些小的提示，只是没能把它们串联起来理解。
  </p>
  <p class="graf graf--p">
   事实上这张照片是在德国汉堡拍摄的镜头，出现在德国电视剧中。
  </p>
  <p class="graf graf--p">
   总之，
   <strong class="markup--strong markup--p-strong">
    作为公民调查人员，您需要具有3项最基本的优势
   </strong>
   ：
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    首先最基本的就是
    <strong class="markup--strong markup--li-strong">
     观察和记忆的能力
    </strong>
    —— 当然，这一能力在其他很多方面都是优势；
   </li>
   <li class="graf graf--li">
    其次就是
    <strong class="markup--strong markup--li-strong">
     分析能力
    </strong>
    ，比如本文采取的排除法，再比如此前强调过的“证伪而不是证实”（如果在这个案例中您采取的是证实法，那么很容易被阿姆斯特丹的错误线索所误导）
   </li>
   <li class="graf graf--li">
    最后就是
    <strong class="markup--strong markup--li-strong">
     寻找资料证据的能力
    </strong>
    ，也就是说，当不知道答案时您应该知道去哪里找。熟记
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e4%bd%a0%e6%93%85%e9%95%bf%e6%90%9c%e7%b4%a2%ef%bc%8c%e4%bd%a0%e8%83%bd%e6%89%be%e5%88%b0%e4%b8%80%e5%88%87%ef%bc%9a%e6%b2%a1%e6%9c%89%e8%b0%b7%e6%ad%8c%e7%9a%84%e5%bc%ba%e5%a4%a7/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e4%bd%a0%e6%93%85%e9%95%bf%e6%90%9c%e7%b4%a2%ef%bc%8c%e4%bd%a0%e8%83%bd%e6%89%be%e5%88%b0%e4%b8%80%e5%88%87%ef%bc%9a%e6%b2%a1%e6%9c%89%e8%b0%b7%e6%ad%8c%e7%9a%84%e5%bc%ba%e5%a4%a7/" rel="noopener noreferrer" target="_blank">
     我们介绍的搜索资源
    </a>
    ，您就很容易找到答案，
   </li>
  </ol>
  <p class="graf graf--p">
   对于第二项优势，我们曾经提供的所有
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    调查演示
   </a>
   都是帮助这点的；对于第三项，我们提供过的所有工具都会有效。
  </p>
  <p class="graf graf--p">
   而
   <strong class="markup--strong markup--p-strong">
    对于第一项，也许才是最棘手的
   </strong>
   ；在手机和互联网的世界里，人们对现实的观察能力和对信息的记忆能力已经变得非常差，这是一个重大损失。IYP将在接下来的安排中加入对记忆力和观察技巧方面的介绍（以及练习），希望对公民调查人员/组织有用。⚪️
  </p>
