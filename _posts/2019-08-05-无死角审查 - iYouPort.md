   无死角审查
  </h1>
 </header>

   </a>

 <div class="entry-content">
  <p class="graf graf--p">
   英国政府在网络安全问题上采取了强硬立场，正在努力建立它所说的世界上第一个*让社交媒体公司受到控制*的独立监管机构。
  </p>
  <p class="graf graf--p">
   未能达到要求的公司将面临巨额罚款，被证明疏忽的高级董事将承担个人责任。他们还可能会发现用户对其网站的访问受阻。
  </p>
  <p class="graf graf--p">
   内政部和文化媒体和体育部联合宣布了旨在使互联网变得“更加安全”的新措施。监管机构的引入是在英国发布的备受期待的政府白皮书中的核心建议，该白皮书名为
   <a class="markup--anchor markup--p-anchor" data-href="https://www.gov.uk/government/consultations/online-harms-white-paper" href="https://www.gov.uk/government/consultations/online-harms-white-paper" rel="noopener noreferrer" target="_blank">
    Online Harms
   </a>
   。
  </p>
  <p class="graf graf--p">
   监管机构的任务是确保社交媒体公司“解决”一系列在线问题，就是加强审查，包括：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    煽动暴力（包括恐怖主义）内容的传播；
   </li>
   <li class="graf graf--li">
    鼓励自残或自杀的内容；
   </li>
   <li class="graf graf--li">
    虚假信息和假新闻的传播；
   </li>
   <li class="graf graf--li">
    网络欺凌；
   </li>
   <li class="graf graf--li">
    对儿童不适当的材料；
   </li>
   <li class="graf graf--li">
    儿童色情。
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    除了应用于主要的社交网络，如 Facebook，YouTube 和 Twitter 之外，文件托管网站、在线论坛、甚至消息服务和搜索引擎，也必须满足这些要求
   </strong>
   。
  </p>
  <p class="graf graf--p">
   英国首相特蕾莎·梅在一份声明中表示，“长期以来，这些公司还没有采取足够措施保护用户，特别是儿童和年轻人免受有害内容的侵害。” “我们听取了活动家和家长的意见，并且正在向互联网公司提供法律责任，以保障人们的安全。”
  </p>
  <p class="graf graf--p">
   谷歌和 Facebook 没有立即回复评论请求。
  </p>
  <p class="graf graf--p">
   英国政府正试图决定是否任命现有的监管机构、或者为此目的创建一个全新的监管机构职位。最初这项工作将由科技行业资助，政府正在讨论对社交媒体公司征税。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “在线公司的自我监管时代已经结束，行业采取的应对网络危害的自愿审查行动并未得到持续应用或远远不够。” Jeremy Wright 在一份声明中说。
  </p>
  <p class="graf graf--p">
   英国公布的措施是全球大规模加大对大科技公司监管的举措的一部分。这些努力起源于欧洲，但在美国、以及包括马克扎克伯格和蒂姆库克在内的科技公司的领导者中都取得了“进步”。要加引号，因为
   <strong class="markup--strong markup--p-strong">
    这些巨头支持政府监管有着强烈私利
   </strong>
   ，具体解释详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/newsletters-07242019-1f6bf0cda578" href="https://www.iyouport.org/%e6%8b%a5%e6%8a%a4%e7%9a%84%e8%83%8c%e5%90%8e/" rel="noopener noreferrer" target="_blank">
    拥护的背后
   </a>
   》。
  </p>
  <p class="graf graf--p">
   在英国发生巨大政治动荡之际，政府决定与硅谷科技公司站在一起，同时希望一旦该国离开欧盟，这些巨头可以继续创造当地就业机会。新监管程序的某些要素仍有争议。
  </p>
  <p class="graf graf--p">
   国会数字文化媒体和体育委员会主席 Damian Collins
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnet.com/news/facebook-broke-law-must-be-regulated-to-protect-democracy-say-politicians/" href="https://www.cnet.com/news/facebook-broke-law-must-be-regulated-to-protect-democracy-say-politicians/" rel="noopener noreferrer" target="_blank">
    最近发表
   </a>
   了一份关于将社交媒体公司称为“数字黑帮”的报道，称政府监管机构“有必要”在必要时启动调查。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “监管机构不能依赖公司的自我报告，”他说。例如，在类似克赖斯特彻奇恐怖袭击事件的情况下，监管机构应该有权调查有关该暴行的内容是如何被分享的，以及为什么不采取更多措施来尽快阻止它。
  </p>
  <p class="graf graf--p">
   行业机构 TechUK 政策负责人 Vinous Ali 对白皮书的发布表示欢迎，但在一份声明中表示，政府方法的某些要素仍然“过于含糊”，政府需要明确其希望监管机构实现的目标。她补充说，政府认为社交媒体公司对用户的“duty of care”没有明确界定，并且可以广泛解释。
  </p>
  <p class="graf graf--p">
   互联网协会代表了包括 Facebook、谷歌和 Twitter 在内的世界上最大的科技公司的一长串名单，它表示，无论规模大小，提案都适用于任何平台实施，这一点非常重要。
   <strong class="markup--strong markup--p-strong">
    简单说就是，全网审查，不留死角。
   </strong>
  </p>
  <p class="graf graf--p">
   Twitter 发言人在一份声明中表示，该公司致力于优先考虑用户的安全性，并指出该平台去年进行的70多项变更。
  </p>
  <p class="graf graf--p">
   推特说，“我们将继续参与行业和英国政府之间的讨论，以及在保持用户安全和保持互联网开放自由的性质之间取得适当平衡的工作。”
  </p>
  <p class="graf graf--p">
   Twitter 并不是唯一提出互联网开放性与英国政府计划相关问题的实体。
  </p>
  <p class="graf graf--p">
   纽约大学斯特恩商业与人权中心副主任 Paul Barrett 发现了该提案的两个潜在问题。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “首先，它引发了政府审查的幽灵，”他在一份声明中说。 “其次，它表明社交媒体公司未能进行更有力的自治，特别是在涉及所谓的虚假信息时，已经造成了政府过度扩张的风险。”
  </p>
  <p class="graf graf--p">
   数字权利组织重申了对社交媒体过于严厉的监管可能导致言论自由和隐私权侵犯的担忧。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这是对言论自由的前所未有的攻击，将使互联网巨头监控数十亿的通信并审查一切合法的言论，”Big Brother Watch 在一条推文中说。
  </p>
  <p class="graf graf--p">
   倡导组织负责人 Joy Hyvarinen 在一份声明中表示，她担心“保护言论自由甚至不如政府希望被视为’做某事’以应对公众压力的重要性”。
  </p>
  <p class="graf graf--p">
   她补充说：“互联网监管需要一种冷静的、以证据为基础的方法，以保障言论自由，而不是破坏言论自由为代价。”
  </p>
  <p class="graf graf--p graf--hasDropCapModel graf--hasDropCap">
   <span class="graf-dropCap">
    U
   </span>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnet.com/news/uk-to-keep-social-networks-in-check-with-internet-safety-regulator/" href="https://www.cnet.com/news/uk-to-keep-social-networks-in-check-with-internet-safety-regulator/" rel="noopener noreferrer" target="_blank">

   </a>
   Social media companies will face huge fines if they fail to live up to their “duty of care” to internet users.
