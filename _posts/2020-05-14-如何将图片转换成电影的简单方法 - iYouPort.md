  <p class="graf graf--p">
   活动家的在线宣传完全可以更精彩，比如电影就比图片更新吸引人，您可以尽情发挥您的创意，人们会被您的魅力所吸引，从而跟随您，至少可以促进更多注意到您希望传达的信息。
  </p>
  <p class="graf graf--p">
   您无需聘请专业视频制作人，自己就可以做，如果您认为需要为此项目花钱，那就错了。
  </p>
  <p class="graf graf--p">
   今天，您将学习如何轻松地将图片制作成电影。使用这种方法，您可以以更加愉悦的方式欣赏它，并分享以获得更大的影响力。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     你需要什么
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   USB电缆，图片（在文件夹或数码相机中）和视频编辑软件 —— 是您制作图片电影所需要的。
  </p>
  <p class="graf graf--p">
   您可以使用任何视频编辑器，但建议考虑使用
   <a class="markup--anchor markup--p-anchor" data-href="https://www.winmoviemaker.com/" href="https://www.winmoviemaker.com/" rel="noopener noreferrer" target="_blank">
    Windows Movie Maker
   </a>
   或内置程序。这样做意味着您无需下载第三方工具。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     流程
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   首先，启动视频编辑器。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11310 jetpack-lazy-image" data-lazy-sizes="(max-width: 798px) 100vw, 798px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-8.png?resize=798%2C858&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-8.png?w=798&amp;ssl=1 798w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-8.png?resize=279%2C300&amp;ssl=1 279w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-8.png?resize=768%2C826&amp;ssl=1 768w" data-recalc-dims="1" height="858" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-8.png?resize=798%2C858&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="798"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11310" data-recalc-dims="1" height="858" sizes="(max-width: 798px) 100vw, 798px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-8.png?resize=798%2C858&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-8.png?w=798&amp;ssl=1 798w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-8.png?resize=279%2C300&amp;ssl=1 279w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/1-8.png?resize=768%2C826&amp;ssl=1 768w" width="798"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   然后单击导入。
  </p>
  <p class="graf graf--p">
   如果您已经将要使用的图片加载到计算机的文件夹中，请选择从文件夹导入。
  </p>
  <p class="graf graf--p">
   如果它们仍在数码相机的存储卡中，请选择从USB设备导入。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11311 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.png?resize=1100%2C428&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.png?w=1218&amp;ssl=1 1218w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.png?resize=300%2C117&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.png?resize=1024%2C399&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.png?resize=768%2C299&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.png?resize=1100%2C428&amp;ssl=1 1100w" data-recalc-dims="1" height="428" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.png?resize=1100%2C428&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11311" data-recalc-dims="1" height="428" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.png?resize=1100%2C428&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.png?w=1218&amp;ssl=1 1218w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.png?resize=300%2C117&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.png?resize=1024%2C399&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.png?resize=768%2C299&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/12/2-5.png?resize=1100%2C428&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   勾选照片右上角的框，选择要添加到电影中的照片。然后，选择 + Add to.。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11312 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-4.png?resize=1100%2C586&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-4.png?w=1228&amp;ssl=1 1228w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-4.png?resize=300%2C160&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-4.png?resize=1024%2C545&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-4.png?resize=768%2C409&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-4.png?resize=1100%2C586&amp;ssl=1 1100w" data-recalc-dims="1" height="586" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-4.png?resize=1100%2C586&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11312" data-recalc-dims="1" height="586" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-4.png?resize=1100%2C586&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-4.png?w=1228&amp;ssl=1 1228w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-4.png?resize=300%2C160&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-4.png?resize=1024%2C545&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-4.png?resize=768%2C409&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/3-4.png?resize=1100%2C586&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   然后将出现一个窗口。它要求您指定要添加所选图片的位置。在此处，选择 New video with music。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11313 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-3.png?resize=1100%2C293&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-3.png?w=1240&amp;ssl=1 1240w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-3.png?resize=300%2C80&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-3.png?resize=1024%2C273&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-3.png?resize=768%2C204&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-3.png?resize=1100%2C293&amp;ssl=1 1100w" data-recalc-dims="1" height="293" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-3.png?resize=1100%2C293&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11313" data-recalc-dims="1" height="293" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-3.png?resize=1100%2C293&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-3.png?w=1240&amp;ssl=1 1240w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-3.png?resize=300%2C80&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-3.png?resize=1024%2C273&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-3.png?resize=768%2C204&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/4-3.png?resize=1100%2C293&amp;ssl=1 1100w" width="1100"/>

  </p>
  <p class="graf graf--p">
   然后，您将为新视频命名。输入名称后，单击创建视频。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11314 jetpack-lazy-image" data-lazy-sizes="(max-width: 758px) 100vw, 758px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/5-2.png?resize=758%2C326&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/5-2.png?w=758&amp;ssl=1 758w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/5-2.png?resize=300%2C129&amp;ssl=1 300w" data-recalc-dims="1" height="326" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/5-2.png?resize=758%2C326&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="758"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11314" data-recalc-dims="1" height="326" sizes="(max-width: 758px) 100vw, 758px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/5-2.png?resize=758%2C326&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/5-2.png?w=758&amp;ssl=1 758w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/5-2.png?resize=300%2C129&amp;ssl=1 300w" width="758"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   输入电影的基本详细信息后，即可访问情节提要。在这里，您可以使用提供给您的许多选项来编辑电影。
  </p>
  <p class="graf graf--p">
   您可以添加标题，设置电影的时长，调整大小，使用滤镜，添加文本，选择动态和添加3D效果。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11315 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/6-3.png?resize=1100%2C469&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/6-3.png?w=1230&amp;ssl=1 1230w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/6-3.png?resize=300%2C128&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/6-3.png?resize=1024%2C436&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/6-3.png?resize=768%2C327&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/6-3.png?resize=1100%2C469&amp;ssl=1 1100w" data-recalc-dims="1" height="469" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/6-3.png?resize=1100%2C469&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   在此阶段所做的不是最终的。您始终可以通过单击播放按钮查看进度。如果您要删除所做的所有修改也可以。只需点击全部删除。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11316 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/7-3.png?resize=1100%2C480&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/7-3.png?w=1232&amp;ssl=1 1232w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/7-3.png?resize=300%2C131&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/7-3.png?resize=1024%2C447&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/7-3.png?resize=768%2C335&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/7-3.png?resize=1100%2C480&amp;ssl=1 1100w" data-recalc-dims="1" height="480" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/7-3.png?resize=1100%2C480&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     提醒
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   如前所述，此处使用的视频编辑器是 Windows 内置程序。您需要注意所使用的设备。
  </p>
  <p class="graf graf--p">
   如果您使用的是Mac，也可以使用其内置的电影制作工具。尽管确切的过程可能有所不同，但概念是一样的。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     完成电影
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   如果您对这部电影100％满意，则可以整理一下。从顶部选择“完成视频”。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11317 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/8-2.png?resize=1100%2C522&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/8-2.png?w=1236&amp;ssl=1 1236w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/8-2.png?resize=300%2C142&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/8-2.png?resize=1024%2C485&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/8-2.png?resize=768%2C364&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/8-2.png?resize=1100%2C522&amp;ssl=1 1100w" data-recalc-dims="1" height="522" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/8-2.png?resize=1100%2C522&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11317" data-recalc-dims="1" height="522" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/8-2.png?resize=1100%2C522&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/8-2.png?w=1236&amp;ssl=1 1236w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/8-2.png?resize=300%2C142&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/8-2.png?resize=1024%2C485&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/8-2.png?resize=768%2C364&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/12/8-2.png?resize=1100%2C522&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   然后将出现一个窗口。它将为您提供完成项目的不同选项。选择适合您目的的一种。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11318 jetpack-lazy-image" data-lazy-sizes="(max-width: 552px) 100vw, 552px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/9-2.png?resize=552%2C870&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/9-2.png?w=552&amp;ssl=1 552w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/9-2.png?resize=190%2C300&amp;ssl=1 190w" data-recalc-dims="1" height="870" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/9-2.png?resize=552%2C870&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="552"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11318" data-recalc-dims="1" height="870" sizes="(max-width: 552px) 100vw, 552px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/9-2.png?resize=552%2C870&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/9-2.png?w=552&amp;ssl=1 552w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/12/9-2.png?resize=190%2C300&amp;ssl=1 190w" width="552"/>
