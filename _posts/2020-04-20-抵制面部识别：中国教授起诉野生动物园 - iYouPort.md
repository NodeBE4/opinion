    <span style="color: #00ccff;">
     <em>
      <strong>
       打了引号。是的，这不是抵制面部识别本身，也不是抵制面部识别最危险的用例，但的确是来自中国的一个巨大进步：人们开始反抗被强加的数据收集了
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter wp-image-10890 jetpack-lazy-image" data-lazy-sizes="(max-width: 750px) 100vw, 750px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/中国.jpg?resize=750%2C422&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/中国.jpg?w=660&amp;ssl=1 660w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/中国.jpg?resize=300%2C169&amp;ssl=1 300w" data-recalc-dims="1" height="422" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/中国.jpg?resize=750%2C422&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="750"/>
   <noscript>
    <img alt="" class="aligncenter wp-image-10890" data-recalc-dims="1" height="422" sizes="(max-width: 750px) 100vw, 750px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/中国.jpg?resize=750%2C422&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/中国.jpg?w=660&amp;ssl=1 660w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/中国.jpg?resize=300%2C169&amp;ssl=1 300w" width="750"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   和其他所有企业媒体一样，BBC拒绝承认《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%ad%e5%9b%bd%e7%9a%84%e5%8f%8d%e4%b9%8c%e6%89%98%e9%82%a6%e7%9b%91%e8%a7%86%e5%9b%bd%e5%ae%b6%e6%98%af%e7%be%8e%e5%9b%bd%e5%88%b6%e9%80%a0%e7%9a%84/" href="https://www.iyouport.org/%e4%b8%ad%e5%9b%bd%e7%9a%84%e5%8f%8d%e4%b9%8c%e6%89%98%e9%82%a6%e7%9b%91%e8%a7%86%e5%9b%bd%e5%ae%b6%e6%98%af%e7%be%8e%e5%9b%bd%e5%88%b6%e9%80%a0%e7%9a%84/" rel="noopener noreferrer" target="_blank">
    中国的反乌托邦监视国家是美国制造的
   </a>
   》。
  </p>
  <p class="graf graf--p">
   一位大学教授正在起诉一个野生动物园，以抵制其加强的人脸识别，这是中国快速使用该技术的首批重大法律挑战之一。
  </p>
  <p class="graf graf--p">
   这是一个已经引起激烈争论的问题。
  </p>
  <p class="graf graf--p">
   郭兵教授说，杭州野生动物园 “突然通过了面部识别登记成为游客入场条件的强制性要求” ，该园 “通过强制收集游客的私人数据违反了消费者保护法”。
  </p>
  <p class="graf graf--p">
   官媒《中国日报》报道，自那时以来，该公园一直在让游客在以前使用的指纹扫描系统和高科技面部识别扫描之间做出妥协。
  </p>
  <p class="graf graf--p">
   该公园只是在其入口引入面部识别技术的众多机构之一。
  </p>
  <p class="graf graf--p">
   在过去的五年中，中国一直在积极地推出面部识别技术，声称为了 “提高安全性”，又继续声称为人们的生活 “带来便利”，特别是在电子支付方面。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%be%bf%e5%88%a9%e7%9a%84%e6%9a%b4%e6%94%bf/" href="https://www.iyouport.org/%e4%be%bf%e5%88%a9%e7%9a%84%e6%9a%b4%e6%94%bf/" rel="noopener noreferrer" target="_blank">
     便利的暴政
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   但是，自从郭教授质疑它的必要性以来，关于公民大量数据被收集和储存的讨论越来越多。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这个公园发生了什么？
   </strong>
  </p>
  <p class="graf graf--p">
   郭教授是中国东部浙江科技大学的法学教授，是杭州野生动物园的季票持有者。
  </p>
  <p class="graf graf--p">
   在过去的几年中，他一直在使用指纹识别进入该公园。但是在去年10月17日，他收到一条消息，通知他公园的系统已经升级，对于游客来说，使用面部识别系统注册个人身份详细信息已经成为强制性要求。
  </p>
  <p class="graf graf--p">
   他对中国网站澎湃新闻说：“我显然对收集面部数据表示不满意”。他说，
   <strong class="markup--strong markup--p-strong">
    他愿意继续用自己的指纹进行扫描
   </strong>
   ，但被告知这是不可能的。（看起来他不是在本质上抵制生物识别数据收集）
  </p>
  <p class="graf graf--p">
   当他说他想取消他的卡时，被告知他将不会获得全额退款。
  </p>
  <p class="graf graf--p">
   因此，10月28日，他将公园告上法庭。
  </p>
  <p class="graf graf--p">
   官方的《中国日报》说，他的案子已被阜阳区人民法院受理，是“中国首例涉及面部识别的案件”。报道称此案仍在进行中。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    他会成功吗？
   </strong>
  </p>
  <p class="graf graf--p">
   牛津大学中国商法研究员 Mimi Zou 博士说，如果郭教授继续追究此案，该案很可能会被驳回。
  </p>
  <p class="graf graf--p">
   她说，目前，“没有任何具有法律约束力的文书可以直接处理他的主张”，公园强制收集他的生物特征数据作为进入条件，因此使他的同意毫无意义。
  </p>
  <p class="graf graf--p">
   她指出，在涉及中国人数据或隐私权的法律保护的商业行为者的案件中，国家监视是“房间里的大象”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “在这个领域没有个人隐私之类的东西，不仅在中国。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该案在中国网络引起广泛辩论
   </strong>
  </p>
  <p class="graf graf--p">
   如今，有关面部识别的问题已经进入法庭，中国的社交网络对此技术进行了大量讨论。
  </p>
  <p class="graf graf--p">
   微博用户注意到“现在很多地方都在强行收集个人信息”。一位用户评论说，他们担心与之相关的未来“会有风险”，这条内容已收到2,000个赞。
  </p>
  <p class="graf graf--p">
   另一位用户补充说：“在中国，人们的隐私没有受到保护，非法收集面部识别信息非常可怕。”
  </p>
  <p class="graf graf--p">
   还有一位用户说：“技术改变了生活，给人们的生活带来了便利。但是，如果发生安全漏洞，您绝对要保持谨慎。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    面部识别监视有多广泛？
   </strong>
  </p>
  <p class="graf graf--p">
   面部识别技术在中国已经存在很多年了。2017年，它被广泛纳入该国的监视网络，并因帮助该国 “识别和捕获逃犯” 而受到赞扬。
  </p>
  <p class="graf graf--p">
   去年，媒体指出，由于面部识别，警察能够在一场音乐会中从6万群众中找出一名逃犯。同年，配备该技术的警察能够识别戴太阳镜的犯罪嫌疑人。
  </p>
  <p class="graf graf--p">
   但去年开始，它在体育馆、办公楼甚至学校等私人机构中的推广更为积极。
  </p>
  <p class="graf graf--p">
   在商店和超级市场中用于支付的面部识别技术已越来越成为一种规范，取代了扫描附加在移动应用程序上的QR码的早期趋势。
  </p>
  <p class="graf graf--p">
   它甚至已经作为娱乐工具在年轻人中普及。
  </p>
  <p class="graf graf--p">
   8月下旬，一个名为Zao的移动应用程序成为头条新闻，因为它可以精美地打印某人的脸，并将其几乎无缝地贴在名人的身上，使人们看起来像是自己喜欢的电影或电视节目中的角色。
  </p>
  <p class="graf graf--p">
   但是，在Zao推出一周之内，在用户注意到该应用程序的条款和条件后，它已从在线商店中删除，条款中写着：“开发人员拥有永久使用该应用程序上创建的任何图像的全球性权利”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    不会放慢脚步
   </strong>
  </p>
  <p class="graf graf--p">
   面部识别的势头在中国不太可能放慢。官方媒体写了很多关于 “成功使用” 面部识别技术来抓捕中国的“猎狐行动”中数百名罪犯的文章。
  </p>
  <p class="graf graf--p">
   此外，中国表示将使用高度复杂的“天网”监视网络，在2020年积极扩展其监视业务。
  </p>
  <p class="graf graf--p">
   2017年，中国约有1亿7千万台CCTV摄像机；而到去年年底已经有4亿台新监视相机投入使用，其中许多相机都配备了人工智能和面部识别功能。⚪️
  </p>
