   <strong class="markup--strong markup--p-strong">
    对于黑客来说
   </strong>
   ，社交媒体的兴起是历史上最重要的技术进步之一。Facebook、Twitter、Instagram 和其他社交媒体平台使访问个人和企业信息变得异常地容易。
  </p>
  <p class="graf graf--p">
   大多数人都已经意识到在线发布个人信息的危险。
  </p>
  <p class="graf graf--p">
   凤凰城大学2016年的
   <a class="markup--anchor markup--p-anchor" data-href="http://www.phoenix.edu/news/releases/2016/04/uopx-social-media-hacking.html" href="https://www.phoenix.edu/news/releases/2016/04/uopx-social-media-hacking.html" rel="noopener noreferrer" target="_blank">
    一项研究显示
   </a>
   ，有84％的美国成年人拥有至少一个社交媒体帐户，“随着社交媒体的知名度不断提高，捕食社交媒体的犯罪分子的数量也在急剧增加。”
  </p>
  <p class="graf graf--p">
   同一份调查还显示：“拥有个人社交媒体资料的美国成年人中，有近三分之二的人表示，他们知道自己的帐户已被黑客入侵，并且有86％的人同意，由于担心黑客会访问自己的私密信息，他们限制了自己发布的内容。 ”
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    许多人忽略了在社交媒体上无意中泄漏的敏感个人信息的危险性。对于图像，尤其如此。
   </strong>
  </p>
  <p class="graf graf--p">
   👉黑客可以使用多种方法来窃取您的个人信息，而且这些方法通常是复杂的攻击，任何人都很难防范。但是社交媒体上大量的个人信息吸引了一种新型的黑客。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种新型的黑客与精通技术的网络犯罪分子看起来更像是情报分析师、或坚定的跟踪者。并且，甚至能全程合法
   </strong>
   。
  </p>

  <p class="graf graf--p">
   👉发布您的汽车照片的问题在于，您会立即暴露出大量个人信息 —— 如果照片中出现了车牌，那么就可以轻松追踪到车主。
  </p>
  <p class="graf graf--p">
   大多数地方都允许您向本地 DMV 提交信息记录请求。其中包含有关车辆登记的信息，以及公开记录的任何机动车定罪和公开事故报告。
  </p>
  <p class="graf graf--p">
   通常需要填写一张表格，并且您必须签名，以便证明提供 “允许使用” 的信息。
  </p>
  <p class="graf graf--p">
   👉追踪者一般不会使用这种方法，以避免留下记录。他们会访问各种类型的体面的公共数据聚合数据库，同样可以使用车牌确定车主身份，比如这些：

    Make/Model of Car
   </li>

    Vehicle Identification Number (VIN)

   <li class="graf graf--li">
    Owner’s Name
   </li>
   <li class="graf graf--li">
    Owner’s Address
   </li>
  </ul>
