      </strong>
     </em>
    </span>

  <p class="graf graf--p">
   在中国采访时一些社会底层的老年人告诉我们，他们不得不安装一款应用程序，并接受其人脸识别验证，因为除非这样，否则他们就无法领取养老金；
  </p>
  <p class="graf graf--p">
   IYP的《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/chinese-horror-story-e7%ef%bc%9a%e6%ad%bb%e4%ba%a1%e9%a9%ac%e6%8b%89%e6%9d%be/" href="https://www.iyouport.org/chinese-horror-story-e7%ef%bc%9a%e6%ad%bb%e4%ba%a1%e9%a9%ac%e6%8b%89%e6%9d%be/" rel="noopener noreferrer" target="_blank">
    中国恐怖故事7：死亡马拉松
   </a>
   》中描述的也是真实的社会，作为社会边缘人士的残疾人为了一年500人民币的所谓补助，被强迫手持身份证拍照，接受各种羞辱性的调查。
  </p>
  <p class="graf graf--p">
   当你买不起大house，只能住在墙壁如纸一样薄的公寓楼里时，除了学会低声说话、不放响屁、不做爱之外，有没有感到严重的侵犯和不公平？
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8932 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-26.jpg?resize=1100%2C733&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-26.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-26.jpg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-26.jpg?resize=768%2C512&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-26.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-26.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-26.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="733" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/1-26.jpg?resize=1100%2C733&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  </p>
  <p class="graf graf--p">
   在自由主义的想象中，政府的监视威胁着每一个人：*无差别*地在所有公众面前徘徊。
  </p>
  <p class="graf graf--p">
   反过来，隐私是保护每个人个性的定义，一个公民的自由空间。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “隐私是一种自我的权利，”斯诺登在2016年时说。
  </p>
  <p class="graf graf--p">
   那些压制自由讨论的独裁专制国家在独裁时期结束后却发现自己很难向民主结构转变，往往是由于这些国家中还没有出现一个私人化的空间，以及大多数人对隐私问题的了解和重视程度堪忧。《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%9a%90%e7%a7%81%e6%84%8f%e8%af%86%e3%80%81%e4%b8%ad%e5%9b%bd%e4%ba%ba%e7%9a%84%e6%80%81%e5%ba%a6%e5%92%8c%e9%80%9a%e5%be%80%e6%b0%91%e4%b8%bb%e7%9a%84%e8%b7%af%e9%9a%9c/" href="https://www.iyouport.org/%e9%9a%90%e7%a7%81%e6%84%8f%e8%af%86%e3%80%81%e4%b8%ad%e5%9b%bd%e4%ba%ba%e7%9a%84%e6%80%81%e5%ba%a6%e5%92%8c%e9%80%9a%e5%be%80%e6%b0%91%e4%b8%bb%e7%9a%84%e8%b7%af%e9%9a%9c/" rel="noopener noreferrer" target="_blank">
    隐私意识、中国人的态度和通往民主的路障
   </a>
   》
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    隐私是所有权利的“源头”。如果没有一个超出国家控制范围的“安静的空间，只属于自己的空间”，自由和民主的参与是不可能的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   以上这些解释都是对的。但是这些解释也忽略了隐私权实际上并不平等 —— 穷人、少数族裔、社会边缘人士，遭受着更多的掠夺和入侵。
  </p>
  <p class="graf graf--p">
   对于下层社会而言，隐私 —— 从未被提供过。
  </p>
  <p class="graf graf--p">
   在警务的情况下这种现实显而易见。第四修正案 —— 防止无理搜查和扣押的宪法条款 —— 规定：在人们可以合理地预期不被他人听到或看到的地方，个人对抵制国家的眼睛和耳朵（还有鼻子）拥有“合理的隐私期望”。
  </p>
  <p class="graf graf--p">
   您在自己家中拥有最多的隐私，在自己的汽车里拥有更少的隐私，而在公共街道上几乎没有。
  </p>
  <p class="graf graf--p">
   因此，
   <strong class="markup--strong markup--p-strong">
    那些能够负担得起更多私人空间的人将不可避免地拥有更多隐私
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但生活在人口密集的社区和墙壁很薄的公寓大楼里的人们，几乎找不到私人空间，他们是在公共场所过着“私人生活”。
  </p>
  <p class="graf graf--p">
   无家可归者 —— 他们必然住在公共场所，对他们的临时住所拥有脆弱的财产权利 ——
   <strong class="markup--strong markup--p-strong">
    这样的人几乎没有任何第四修正案权利。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️隐私从来不是一种普遍的权利，而是一种阶级特权：保证给那些能够买得起它的人。
  </p>
  <p class="graf graf--p">
   当一个中产阶级白人想象隐私侵犯时，他们往往会想到一些令人尴尬的个人信息披露：比如他们的性生活在网上被公开、或者他们的电子邮件地址被泄露在 Ashley Madison 入侵中。
  </p>
  <p class="graf graf--p">
   有些人可能会想到被偷窥的“令人毛骨悚然”的感觉。
  </p>
  <p class="graf graf--p">
   印第安纳大学法学院教授 James P. Nehf 的文章反映了许多学者的观点，他说：“隐私很少是生死攸关的问题。” 真不愧是知识分子。
  </p>
  <p class="graf graf--p">
   但是，要知道，
   <strong class="markup--strong markup--p-strong">
    对于穷人和边缘人士来说，隐私侵犯往往都是生死攸关的问题。
   </strong>
  </p>
  <p class="graf graf--p">
   趴在墙上等着被搜身可以很容易地避免被警察射击杀死；顺从于登记处共享所有个人数据可能会避免被逮捕或被驱逐出境。来自社工的审查可以撕裂一个家庭……
  </p>
  <p class="graf graf--p">
   在过去几年中，出现了一波新的学术，强调了普遍主义隐私语话的不足。
  </p>
  <p class="graf graf--p">
   这些学者说，为了防止有形的监视伤害，公民自由主义者必须关注这一点。
  </p>
  <p class="graf graf--p">
   隐私倡导者警告说，反乌托邦社会即将来临 — 除非对国家安全局收集的手机数据或通信的权力有适当的限制。
  </p>
  <p class="graf graf--p">
   这都是对的。然而，如果你密切关注社会历史，你会能看清
   <strong class="markup--strong markup--p-strong">
    一种完全不同的隐私图景：它并不像普遍的权利，更像是白人和富人阶层的特权
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️保护一些人免受国家监视的法律从未保护过国家寻求限制和控制的人。
  </p>
  <p class="graf graf--p">
   对于穷人和边缘人来说，噩梦般的情景一直都在那。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8933 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-15.jpg?resize=1100%2C733&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-15.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-15.jpg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-15.jpg?resize=768%2C512&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-15.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-15.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-15.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="733" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/2-15.jpg?resize=1100%2C733&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    从种植园到监狱，监视和惩罚一直作为社会控制的手段而存在
   </strong>
   。
  </p>
  <p class="graf graf--p">
   奴隶系统由三种信息技术构成 —— 奴隶通行证、逃亡奴隶海报和奴隶巡逻 —— 它们共同使得庄园主阶级能够施加独裁统治，他们的权力覆盖战前整个南部地区。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    种植园帐簿是最原始的生物识别数据库，不仅记录财务，还记录了被奴役人口的身体特征、劳动力和生殖能力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   种植园的建筑本身就促进了奴隶对监视者的可视性，从大房子的阳台到奴隶宿舍和田地。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这些通过视觉、恐怖和暴力实现的联锁系统，旨在破坏被奴役人民之间的团结，防止离家出走，最重要的是，阻止和平息反抗
   </strong>
   。
  </p>
  <p class="graf graf--p">
   和如今的大规模监视系统的运作目的一模一样。
  </p>
  <p class="graf graf--p">
   在北方城市，阻止犯罪的不是法院，而是警员、值夜班的警卫和监狱长，
   <strong class="markup--strong markup--p-strong">
    他们在选择惩罚手段时获得了广泛的自由
   </strong>
   。
  </p>
  <p class="graf graf--p">
   他们特别关注那些不守规矩的工人阶级女性，正如 Jen Manion 在她出色的2015年作品《Liberty’s Prisoners: Carceral Culture in Early America》中记载的那些女性，她们“很穷、说话很大声、性感、饮酒、或个性太独立”。
  </p>
  <p class="graf graf--p">
   任何对白人异性恋家庭“构成威胁”的人都引起了当局的注意。Manion 写道，到1807年，穷人就像他们因为没有工作或无家可归一样被关押，就像他们的样子本身有罪一样。
   <strong class="markup--strong markup--p-strong">
    贫穷本身就是犯罪
   </strong>
   。
  </p>

  <p class="graf graf--p">
   尽管废奴运动、劳工运动、民权运动和女权主义运动，取得了成功，但是，
   <strong class="markup--strong markup--p-strong">
    监狱仍然是穷人的仓库，警察国家仍然主要是调节低阶层的手段，而现代警察的监视仍然是主要针对穷人、有色人种、对抗者和边缘人士
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Eubanks 是纽约州立大学奥尔巴尼分校政治学副教授，她在全部职业生涯中一直在阐明这个社会提供的惩罚制度是如何针对那些被认为是危险的社会边缘人的穷人。
  </p>
  <p class="graf graf--p">
   《自动化不平等》这本书探索了
   <strong class="markup--strong markup--p-strong">
    19世纪的实体贫民窟如何被当今的“数字贫民窟”所取代。
   </strong>
  </p>
  <p class="graf graf--p">
   在当代福利体系中，穷人的利益价值和他们的所谓犯罪倾向被通过数据挖掘计算机算法来评估，其结果与他们的支持者的希望完全相反 —— 往往会重建和加剧不平等。
  </p>
  <p class="graf graf--p">
   Eubanks 探索了三个实验，算法如何正在取代人类或增强人类的决策 —— 印第安纳州的自动医疗补助资格流程、洛杉矶为无家可归者提供的协调准入系统、以及宾夕法尼亚州阿勒格尼县用于评估儿童被虐待和忽视之风险的预测算法。
  </p>
  <p class="graf graf--p">
   在这里看到一套
   <strong class="markup--strong markup--p-strong">
    关于算法专制的视频
   </strong>
   《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e6%ad%a3%e5%9c%a8%e5%a6%82%e4%bd%95%e8%a2%ab%e6%93%8d%e7%ba%b5%ef%bc%9a%e8%bf%99%e6%98%af%e6%af%8f%e4%b8%aa%e4%ba%ba%e9%83%bd%e9%9a%be%e4%bb%a5%e8%ba%b2%e9%81%bf%e7%9a%84%e6%97%a0%e5%bd%a2/" href="https://www.iyouport.org/%e4%bd%a0%e6%ad%a3%e5%9c%a8%e5%a6%82%e4%bd%95%e8%a2%ab%e6%93%8d%e7%ba%b5%ef%bc%9a%e8%bf%99%e6%98%af%e6%af%8f%e4%b8%aa%e4%ba%ba%e9%83%bd%e9%9a%be%e4%bb%a5%e8%ba%b2%e9%81%bf%e7%9a%84%e6%97%a0%e5%bd%a2/" rel="noopener noreferrer" target="_blank">
    你正在如何被操纵：这是每个人都难以躲避的无形专制
   </a>
   》。
  </p>
  <p class="graf graf--p">
   洛杉矶的登记处“被称为无家可归者服务”，本应该以最高效率和精确度向最需要的人分配住房资源，但这是一个蹒跚的市政官僚机构永远无法实现的。
  </p>
  <p class="graf graf--p">
   2016年8月推出的 Allegheny 家庭筛查工具（AFST）旨在消除导致社会工作者过度报告非白人家庭的人为偏见，因为虐待风险 —— 并忽视了其他地方滥用的迹象。
  </p>
  <p class="graf graf--p">
   然而，在实践中，
   <strong class="markup--strong markup--p-strong">
    自动化系统依赖于大量收集关于福利受益者生活的更加私密的细节
   </strong>
   ，对他们的创伤、应对机制、和恐惧进行编目分类和排序。
  </p>
  <p class="graf graf--p">
   从申请的那一刻起，福利申请人就发现自己陷入了严密的监视网络，受到持续的审查，推定有罪，侵入性审讯，强制他们的生育选择，以及侵犯他们的身体和决定自治权。
  </p>
  <p class="graf graf--p">
   声称“防止欺诈、节省纳税人的钱和保护儿童”，
   <strong class="markup--strong markup--p-strong">
    国家迫使贫困家庭扒光自己的一切，审问他们的每一次购买，每一个做父母的决定，每一种关系……
   </strong>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8935 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-15.jpg?resize=1100%2C619&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-15.jpg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-15.jpg?resize=300%2C169&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-15.jpg?resize=768%2C432&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-15.jpg?resize=1024%2C576&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-15.jpg?resize=1100%2C619&amp;ssl=1 1100w" data-recalc-dims="1" height="619" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-15.jpg?resize=1100%2C619&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8935" data-recalc-dims="1" height="619" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-15.jpg?resize=1100%2C619&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-15.jpg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-15.jpg?resize=300%2C169&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-15.jpg?resize=768%2C432&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-15.jpg?resize=1024%2C576&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/08/3-15.jpg?resize=1100%2C619&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    自动化破坏了正当程序，却未能解决贫困的根本原因
   </strong>
   。这些系统收集并生成的是 —— ⚠️证明现状合理、并将绝望定为犯罪的数据。
  </p>
  <p class="graf graf--p">
   Allegheny 算法将公共服务本身的使用视为儿童的风险……
  </p>
  <p class="graf graf--p">
   洛杉矶的无家可归者面临着类似的诡异选择。为了获得住房资源，洛杉矶无家可归的男女必须填写“脆弱性指数”调查。
  </p>
  <p class="graf graf--p">
   ⚠️该调查征求了一系列个人信息：社会安全号码、全名、出生日期、人口统计信息、退伍军人身份、移民和居住证明，以及在一天中的不同时间可以找到被访者的地点 ……也就是要确认实时跟踪路径。
  </p>
  <p class="graf graf--p">
   它还询问有关申请人在家庭暴力、性侵犯、自我伤害、吸毒、性工作和心理健康等方面的经历的私密问题。
  </p>
  <p class="graf graf--p">
   Eubanks 发现，在入境调查中承认“有风险、甚至有非法行为”，可以“在永久支持性住房的优先名单上获得更高的排名”。但是，这样也让你更容易受到警察的攻击。
  </p>
  <p class="graf graf--p">
   洛杉矶登记处的自我报告数据可在没有手令、没有部门监督、或没有任何形式的司法审查的情况下提供给洛杉矶警察局。
  </p>
  <p class="graf graf--p">
   最重要的是，Eubanks 证明，
   <strong class="markup--strong markup--p-strong">
    增加的数字审查和自动化进一步将穷人与富人区分开来，破坏了促进同理心的亲密互动 —— 团结一致更无从谈起
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Eubanks 写道，“就像早期的贫困管理技术创新一样，数字跟踪和自动化决策使贫困问题对中产阶级公众隐藏起来了，并为国家提供了做出不人道选择所需的道德距离。”
  </p>
  <p class="graf graf--p">
   数字贫民窟将正义问题重新定义为效率问题。算法对个人进行评分，决定如何分配稀缺资源，但这样做反而具体化了稀缺问题。
  </p>
  <p class="graf graf--p">
   加州大学洛杉矶分校法学教授 Gary Blasi 写道：
   <strong class="markup--strong markup--p-strong">
    与以往的县级贫民窟一样，自动化是管理穷人的一种手段，⚠️“这样一来我们就不必消除贫困了。”
   </strong>
  </p>
  <p class="graf graf--p">
   <img alt="" class="size-full wp-image-8936 alignleft jetpack-lazy-image" data-lazy-sizes="(max-width: 400px) 100vw, 400px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/书2.jpg?resize=400%2C600&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/书2.jpg?w=400&amp;ssl=1 400w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/书2.jpg?resize=200%2C300&amp;ssl=1 200w" data-recalc-dims="1" height="600" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/书2.jpg?resize=400%2C600&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="400"/>
   <noscript>
    <img alt="" class="size-full wp-image-8936 alignleft" data-recalc-dims="1" height="600" sizes="(max-width: 400px) 100vw, 400px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/书2.jpg?resize=400%2C600&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/书2.jpg?w=400&amp;ssl=1 400w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/书2.jpg?resize=200%2C300&amp;ssl=1 200w" width="400"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Eubanks 的书是对社会问题数字化的尖锐反驳，这种问题代表了新自由主义的福利方法。
  </p>
  <p class="graf graf--p">
   通常情况下，Eubanks 观察到，有关穷人的故事只有两部分，要么“‘你应该为穷人感到难过’、或者’你不应该难过’。”
  </p>
  <p class="graf graf--p">
   对于 Eubank 来说，
   <strong class="markup--strong markup--p-strong">
    打破自建国以来持续存在的贫困道德神话，对于纠正数字贫民窟的错误来说至关重要。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️贫困一直被认为“主要是道德缺失的后果”，这种神话使中产阶级的良知得到了安抚，证明了他们生活中永久性的极权主义监视制度的合理性。
  </p>
  <p class="graf graf--p">
   Eubank 只是拒绝那些关于隐私和监视的主流叙述的学者之一。事实上，正如她和其他学者所表明的那样，隐私的法律制度从未打算首先保护他们。
  </p>
  <p class="graf graf--p">
   波士顿大学法律学者 Khiara Bridges 在其2016年出版的“隐私权的贫困”一书中记录了申请福利的女性几乎在生活的各个方面都被剥夺了空间、决策权和信息隐私权。
  </p>
  <p class="graf graf--p">
   他们受到的侵略、有时是重新创伤以及质疑，是获得福利的先决条件。国家干涉她们生活中最私密的决定：迫使妇女分娩和终止怀孕，违背她们的意愿。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    法院一直坚持无证搜查福利受助人的住所合法
   </strong>
   —— 这可是第四修正案原则中最神圣的地方 —— 只要声称是搜寻欺诈证据。
  </p>
  <p class="graf graf--p">
   当然你可以拒绝被搜查，但也因此相当于拒绝了福利。
  </p>
  <p class="graf graf--p">
   正如法律学者 Chris Slobogin 所指出的那样，
   <strong class="markup--strong markup--p-strong">
    那些涉嫌税务欺诈的人（对纳税人最严重的罪行）一直享受着第四修正案的全面保护，而那些涉嫌福利欺诈的人则没有得到任何保护。
   </strong>
  </p>
  <p class="graf graf--p">
   政府使用的借口只有“防欺诈和保护贫困儿童”。但是
   <strong class="markup--strong markup--p-strong">
    政府也拒绝做出所有人都知道的会对贫困儿童的安全和健康实现最大贡献的事：让他们的家庭摆脱更穷。
   </strong>
  </p>
  <p class="graf graf--p">
   国家对公共利益的吝啬态度与其保护贫困儿童的承诺完全矛盾。并且，惩罚性福利国家试图“纠正”贫穷母亲的不良行为 —— 他们的道德缺陷被认为是他们的贫困原因 —— 同时拒绝为贫困儿童提供服务。
  </p>
  <p class="graf graf--p">
   消除福利制度中更多惩罚性、可疑性和侮辱性方面的最佳方法应该是，完全放弃经济情况调查的过程。取而代之的是，拥有普遍的基本收入或普遍的子女津贴制度。
  </p>
  <p class="graf graf--p">
   普遍的好处是减轻了对持久状态审查和侵入式数据收集的需求 —— 因为每个人都可以随意获取这份福利。
  </p>
  <p class="graf graf--p">
   当每个家庭都获得现金福利时，很难再将黑人母亲作为替罪羊和妖魔化 — 这是反福利言论的一个持久主题。
  </p>
  <p class="graf graf--p">
   富裕的白人最好放弃对隐私权是普遍享有的理解 — 认为其不受历史、权力、强迫的影响。这与那些忍受监视最严重后果的人所承受饿现实几乎没有相似之处。
  </p>
  <p class="graf graf--p">
   正如 Brandeis 和 Warren 在其1890年对法定隐私的阐述中所解释的那样，“to be let alone” 的权利对那些经济环境需要相互依赖才能生活的人来说几乎没用。如果你没有家，那么“家就是人的城堡”的普通法原则就毫无价值。
  </p>
  <p class="graf graf--p">
   相信隐私是一项基本权利 —— 人类尊严、自治和自决的保障 —— 的人们，也应该倡导政策来纠正财富和权力的巨大不平等，
   <strong class="markup--strong markup--p-strong">
    这种不平等使公平享受隐私成为不可能
   </strong>
   。⚪️
  </p>
  <p class="graf graf--p">

    Privacy for Whom?
   </a>
  </p>
