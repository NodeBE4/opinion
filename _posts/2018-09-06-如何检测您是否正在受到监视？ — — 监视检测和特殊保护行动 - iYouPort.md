     <p class="graf graf--p graf-after--figure" id="0877">
      本文将介绍一些技术，以检测是否有人正在对您进行监视。在我们开始之前，我想明确指出，我在这方面的知识和经验来自私营部门，而不是来自任何秘密的政府机构工作。我们将要讨论的原则和技术是可以全面有效的，尤其是在应付警务外包方面。
     </p>
     <p class="graf graf--p graf-after--p" id="2099">
      但有一点需要提醒，对于政府监控，
      <strong class="markup--strong markup--p-strong">
       您需要在对政权最有可能采取的措施做出较为充分的预测的基础上，针对性选择我们提供的方法
      </strong>
      。如何预测？如果您一直在关注本博客就能明白应该怎么做。除了
      <a class="markup--anchor markup--p-anchor" data-href="https://plus.google.com/b/104314277918209223775/104314277918209223775" href="https://plus.google.com/b/104314277918209223775/104314277918209223775" rel="noopener nofollow noreferrer" target="_blank">
       Google+ 的新闻版块
      </a>
      之外，我们的博客所提供的基本都是工具，包括一些看起来像评论的文章，事实上都是。就如中国人喜欢的俗语：知己知彼方百战不殆。只要您有心，就肯定明白这些工具的真正用途。
     </p>
     <p class="graf graf--p graf-after--p" id="388b">
      并不是说新闻版块不重要哦，它将提供最新的局势及其有效分析，对我们的工具选择来说是很有用的。
     </p>
     <p class="graf graf--p graf-after--p" id="2c8d">
      我将在这里介绍的技术已经经过测试并可以成功应用于现实生活中。在全球大部分地区都适用。因为这类监视活动的执行者会接受专门的培训，而这些培训大同小异，其不同之处仅在于结合当地地理形势等细微方面。
     </p>
     <p class="graf graf--p graf-after--p" id="cf25">
      好了，让我们开始。
     </p>
     <p class="graf graf--p graf-after--p" id="04cd">
      <strong class="markup--strong markup--p-strong">
       你应该注意哪里？
      </strong>
     </p>
     <p class="graf graf--p graf-after--p" id="8ab4">
      简短的回答是，这取决于您的位置，以及您是静态的还是移动的。下面我们分头说。
     </p>
     <p class="graf graf--p graf-after--p" id="e9a8">
      <strong class="markup--strong markup--p-strong">
       静态监视
      </strong>
      ：
     </p>
     <p class="graf graf--p graf-after--p" id="bca9">
      如果你是静止的，那么对你的监视很可能也是静态的。
     </p>
     <p class="graf graf--p graf-after--p" id="44c9">
      让我们来看看有可能找到你的两个最可能的静态位置 — 你的家和你的工作场所。这些非常重要，因为如果你已经被作为特定目标了，那么可以推测，监视者能够定期甚至经常地在这两个地方找到你。
     </p>
     <p class="graf graf--p graf-after--p" id="85e4">
      为了弄清楚监视的位置，你要做的就是让自己置身于一个潜在的监视者的角度，并试图弄清楚你的位置，以隐蔽地观察前门或通往你住所的车道，以及工作场所的主要入口或停车场入口。记得那句话吗？ — —
      <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E4%BA%BA%E6%80%A7%E5%BC%B1%E7%82%B9%E8%83%BD%E6%91%A7%E6%AF%81%E4%B8%80%E5%88%87%E9%98%B2%E7%81%AB%E5%A2%99-%E7%A4%BE%E4%BA%A4%E5%B7%A5%E7%A8%8B%E6%94%BB%E5%87%BB%E5%AE%9E%E8%B7%B5-%E4%BB%A5%E5%8F%8A%E4%BD%A0%E5%BF%85%E9%A1%BB%E5%9C%A8%E4%B9%8E%E7%9A%84%E9%97%AE%E9%A2%98-1080b745430" href="https://medium.com/@iyouport/%E4%BA%BA%E6%80%A7%E5%BC%B1%E7%82%B9%E8%83%BD%E6%91%A7%E6%AF%81%E4%B8%80%E5%88%87%E9%98%B2%E7%81%AB%E5%A2%99-%E7%A4%BE%E4%BA%A4%E5%B7%A5%E7%A8%8B%E6%94%BB%E5%87%BB%E5%AE%9E%E8%B7%B5-%E4%BB%A5%E5%8F%8A%E4%BD%A0%E5%BF%85%E9%A1%BB%E5%9C%A8%E4%B9%8E%E7%9A%84%E9%97%AE%E9%A2%98-1080b745430" rel="noopener noreferrer" target="_blank">
       <strong class="markup--strong markup--p-strong">
        像骇客一样思考
       </strong>
      </a>
      。
     </p>
     <p class="graf graf--p graf-after--p" id="6976">
      不要只是理论化，你应该实际
      <strong class="markup--strong markup--p-strong">
       尝试在现场的感觉
      </strong>
      。有没有方便的咖啡馆、繁忙的十字路口、公共汽车站或火车站、公园或城市长椅等可以让您在很长一段时间内获得良好的位置和足够的掩护？如果是，那些可能是您想要缩小注意力的位置，而不是试图覆盖整个区域。
     </p>
     <p class="graf graf--p graf-after--p" id="d308">
      您还需要注意停放在可能为监视者提供良好视野的地方的车辆（如果该区域没有如上所述的任何良好的有利位置，这一点就尤其重要）。如果存在这样的有利位置，并被流浪汉或者移动摊点所占据，那么监视者很有可能伪装为流浪汉或摊主，也就是说，第一步应该重点关注的是位置而不是看起来是否可疑的人。
     </p>
     <p class="graf graf--p graf-after--p" id="ffb3">
      <strong class="markup--strong markup--p-strong">
       一个训练有素的监视者要做的第一件事就是，把自己的真实身份融入周边环境中
      </strong>
      。还记得我们在《
      <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E7%94%A8-%E7%9C%BC-%E8%AF%86%E4%BA%BA-%E4%BD%A0%E6%83%B3%E7%9F%A5%E9%81%93%E8%87%AA%E5%B7%B1%E5%9C%A8%E4%B8%80%E4%B8%AA-%E9%97%B4%E8%B0%8D-%E7%9A%84%E7%9C%BC%E4%B8%AD%E6%98%AF%E4%BB%80%E4%B9%88%E6%A0%B7%E5%90%97-133eabea9eae" href="https://medium.com/@iyouport/%E7%94%A8-%E7%9C%BC-%E8%AF%86%E4%BA%BA-%E4%BD%A0%E6%83%B3%E7%9F%A5%E9%81%93%E8%87%AA%E5%B7%B1%E5%9C%A8%E4%B8%80%E4%B8%AA-%E9%97%B4%E8%B0%8D-%E7%9A%84%E7%9C%BC%E4%B8%AD%E6%98%AF%E4%BB%80%E4%B9%88%E6%A0%B7%E5%90%97-133eabea9eae" rel="noopener noreferrer" target="_blank">
       用“眼”识人
      </a>
      》中介绍过的观察法吗？在这里它反着用：如果你不希望被人发现或关注，那么你就应该表现得与周边和谐一致，在多方面的细节上。
     </p>
     <p class="graf graf--p graf-after--p" id="b16f">
      这属于较高水平的敌对监视，也许你在现实中遇不到这么高的水平，但如果你能以高水平的基础掌握这些观察技能，你将可以应对一切水平模式的敌对监视。
     </p>
     <p class="graf graf--p graf-after--p" id="37dc">
      <strong class="markup--strong markup--p-strong">
       移动监控
      </strong>
      ：
     </p>
     <p class="graf graf--p graf-after--p" id="f24c">
      在进行移动监控时，事情变得有点棘手，因为监视者也可能需要移动。在运动状态下，监视人员通常希望在目标后面的某个地方，并且可能在后侧面，而不是直线（那样有点傻。或者说如果你只关注直线后方，也有可能会傻）。当目标在典型的城市人行道上向前移动时，我们可以想象潜在的监视区域范围：一个是在目标后面，一个在后面但在街道的另一侧，当它向前移动时将被“拖”到目标后面。
     </p>
     <p class="graf graf--p graf-after--p" id="a5a6">
      关于移动监控最具挑战性的事之一是当目标不可避免地停止时（无论是短时间还是较长时间），监视者需要做的事情。这是因为目标从移动切换到静态，然后可能再次返回移动状态，这些过渡点对监视者来说是真正的挑战 — 因此也为您提供了探测好机会 — 因为监视者必须快速弄清楚如何继续跟踪他们的目标，而不会出现任何令人惊讶的、混乱或紧张的迹象。
     </p>
     <h3 class="graf graf--p graf-after--p" id="02f4">
      <span style="color: #339966;">
       <strong class="markup--strong markup--p-strong">
        如何发现监控
       </strong>
      </span>
     </h3>
     <p class="graf graf--p graf-after--p" id="0323">
      主要通过注意与目标的相关性来检测监视。因为在这种情况下，你自己是目标，所以挑战在于发现某人是否与你有关，即使他们故意试图从你身后或在你视野之外的角度来监视你。
     </p>
     <p class="graf graf--p graf-after--p" id="31a2">
      如果你不想透露正在寻找监视这一事实，就不得不想方设法简要地浏览上述监视有利位置（静态和移动监视点）。简单说就是，你能看到有人在看你，在你移动和停止时，或者只是在不同的地方在一段时间内以某种方式多次出现“碰巧”。
     </p>
     <p class="graf graf--p graf-after--p" id="61a4">
      听起来有点棘手，因为大多数可能下处于潜在有利位置或在街上走在你身后的人都没有试图监视你。我不想把你变成疑神疑鬼的人，但是
      <strong class="markup--strong markup--p-strong">
       你应该知道自己什么时候需要做这样的警惕
      </strong>
      。
     </p>
     <p class="graf graf--p graf-after--p" id="fa4d">
      要知道，监视人员与其他任何人的不同之处在于，他们有意地试图观察你在哪里以及你去哪里。因此，监视人员肯定会展示出与您的积极关联 — 特别是在
      <strong class="markup--strong markup--p-strong">
       时间和距离
      </strong>
      上 — 这是最重要的指标，可以将监视人员与周围的其他人区分开来。发现这些相关性并不总是很容易的，但通过练习，你肯定能做得更好。
     </p>
     <p class="graf graf--p graf-after--p" id="6488">
      以下技术旨在检测一些经典的监测指标，忘掉那些关于“可疑人”的典型特征吧，有些书描述的东西完全是错的。请记住，如果被非常高级的情报部门进行监视、或者如果您正在被一个协调一致的操作团队来进行监视，光靠这些可能不够。当然我们的读者中也不会有值得间谍机构大规模出动的人选哈。玩笑。
     </p>
     <p class="graf graf--p graf-after--p" id="98ea">
      好消息是，您可以随时随地练习这些技巧 — 逐渐变得更好，更精细地应用它们。逐渐趋向能应付更高级的和团体协作的监视状态。
     </p>
     <p class="graf graf--p graf-after--p" id="9bc4">
      <strong>
       静态监视检测
      </strong>
      ：
     </p>
     <ul class="postList">
      <li class="graf graf--li graf-after--p" id="2c6a">
       当你在家、工作地点、餐厅、酒吧等地时，试图找到来自*外面*的视角（通过窗户、入口等）。试着看看你是否可以在有利视角的位置找到任何人，或者只是在外面徘徊的人，看看谁会密切关注你可能会使用的入口/出口。
      </li>
      <li class="graf graf--li graf-after--li" id="4e31">
       当您出来时，使用自然的转头和周边视觉来查看是否有人注意到您的退出，并正在准备从静态转移到移动状态 — 从而与您自己的转换相关联。
      </li>
     </ul>
     <p class="graf graf--p graf-after--li" id="92f6">
      <strong>
       移动监控检测
      </strong>
      ：
     </p>
     <ol class="postList">
      <li class="graf graf--li graf-after--p" id="ad00">
       当你向前迈进时，试着找到创造性的方法，在你身后的区域、但在街道的另一侧，瞥一眼。这样做的一个好方法是定期侧视，并使用周边视觉来看待你身后（商店橱窗是个好东西哦）。你需要
       <strong class="markup--strong markup--li-strong">
        隐藏自己这种观察动作
       </strong>
       ，让自己看起来是对身边的东西感兴趣，而不是身后的东西，看看华丽的汽车或走过身边的有吸引力的人，等等）。尽量不要夸大这些动作，保持自然。您需要的只是极短的一个目光，即便发现了跟踪者也要当作没发现一样。这很重要。
      </li>
      <li class="graf graf--li graf-after--li" id="599f">
       使用人行横道，红绿灯或其他常见的可以短暂停留的理由，来自然环顾四周，从而获得更好的外围视图，了解您背后的情况。
      </li>
      <li class="graf graf--li graf-after--li" id="e072">
       找到合理的原因，穿过你正在走的街道的另一边。你将要做的 90 度转弯可以将你身后的区域尽收眼底，当过马路时，你可以更轻松，巧妙地覆盖周边视野。
      </li>
      <li class="graf graf--li graf-after--li" id="4c75">
       使用短暂停留（浏览商店橱窗、等待步行信号、检查手机等），试着注意你身后的人是否也突然从移动过渡到静止（从而与你的动作相关）。
      </li>
      <li class="graf graf--li graf-after--li" id="7f2b">
       使用更长时间的停靠点（走进商店、咖啡馆等）快速查看前窗或门，看看是否有人从移动过渡到静态，是不是有人正在观察您刚走进去的方向，或者有人开始走近你。然后重复上面静态监视检测部分中提到的两个步骤。
      </li>
      <li class="graf graf--li graf-after--li" id="6d47">
       使用微小的假动作来判断您是否可能暴露。保持简单和微妙。尝试执行以下操作：
      </li>
     </ol>
     <ul class="postList">
      <li class="graf graf--li graf-after--li" id="5fb9">
       走过一个带窗户的商店，然后停下来，转身回到商店。
      </li>
      <li class="graf graf--li graf-after--li" id="e6b8">
       当你坐在咖啡馆里，起身时（好像你准备离开）收拾你的东西，然后去柜台点东西，或者去洗手间，然后回到你的桌子或另一张桌子。
      </li>
      <li class="graf graf--li graf-after--li" id="ccfb">
       当然，您可以执行许多其他此类操作，并且所有这些操作的重点是：
       <strong class="markup--strong markup--li-strong">
        做一些监视者难以预料的动作
       </strong>
       ，因此更有可能导致技能较低的监管者犯下经典的错误。这些错误通常会出现引起紧张混乱、双重攻击、从静态到移动的不自然的转变，以及不自然地重新回到静态等等的形式，这将很容易帮助你找到存在相关性的机会。
      </li>
      <li class="graf graf--li graf-after--li" id="51fc">
       记住监视者总是与他们的目标相关联，问题在于他们能够隐藏它的程度，这些行动可以决定相关性能被发觉的程度。老练的间谍是很难被发现的，但一般情况下大家遇不到摩萨德那种高手。
      </li>
     </ul>
     <h3 class="graf graf--p graf-after--li" id="30cd">
      <span style="color: #339966;">
       <strong class="markup--strong markup--p-strong">
        棘手的技巧……
       </strong>
      </span>
     </h3>
     <p class="graf graf--p graf-after--p" id="0882">
      许多人喜欢好莱坞式的使用反光表面、窥视孔或小工具的想法。我不是那种东西的粉丝。并不是说通过观察反光表面来发现身后的人是不可能的，但如果过度依赖这种策略，你可能不会做得好。现在我们的
      <strong class="markup--strong markup--p-strong">
       目标是提升你不需要借助什么东西就能在现实中操作的能力
      </strong>
      ，而不是拍电影为了看起来精致。
     </p>
     <p class="graf graf--p graf-after--p" id="9735">
      此外，盯着窗户（或者更确切地说是它反射的东西）或使用笔记本电脑或手机的黑屏来反映你背后的东西，比你想象中要更明显。甚至一些老掉牙的套路也是有效的，例如故意掉东西以便在拾起时能环顾四周，或者停下来绑鞋带。别笑，这些都是有用的。
     </p>
     <h3 class="graf graf--p graf-after--p" id="6bb8">
      <span style="color: #339966;">
       <strong class="markup--strong markup--p-strong">
        监视检测路线（SDR）
       </strong>
      </span>
     </h3>
     <p class="graf graf--p graf-after--p" id="fec3">
      SDR 是您可以提前计划的预定路线，旨在巧妙地暴露敌对监视。它基本上将所有上述策略整合到一个计划的序列中，仔细选择一条路线，引诱监视人员跟随你。
     </p>
     <p class="graf graf--p graf-after--p" id="e387">
      典型的路线需要从您希望找到的逻辑位置开始（您的工作场所就很好）。
      <strong class="markup--strong markup--p-strong">
       路线不应该是您通常采用的路线
      </strong>
      ，因为如果监视人员已经明确知道您要去哪，他们就不需要跟随你了（他们可以在那里等你）。相反，
      <strong class="markup--strong markup--p-strong">
       不要让这条路线太奇怪
      </strong>
      ，因为监视人员可能不会想要跟随你，或者更糟糕的是，可能会怀疑这是一个陷阱。
     </p>
     <p class="graf graf--p graf-after--p graf--trailing" id="69ab">
      确保路线中有短暂停留和长停留（您已经知道要查找什么了，以及从哪里查看）。如果存在监视，这将增加隐蔽检测监视的可能性。
     </p>
    </div>
   </div>
  </section>
  <section class="section section--body section--last">
   <h3 class="section-divider">
    <span style="color: #339966;">
     说在最后
    </span>
   </h3>
   <div class="section-content">
    <div class="section-inner sectionLayout--insetColumn">
     <p class="graf graf--p graf--leading" id="b77a">
      另外简单说几句
      <strong class="markup--strong markup--p-strong">
       外观问题
      </strong>
      。外观在反侦察术语中被称为“封面”，包括从衣着到神态到言谈举止，都要高度符合你希望自己伪装成的那个封面的特征。绝不能够你的衣着看起来像个乞丐、你的自我介绍是“无家可归的可怜人”，而你的神态警觉地瞄向四方暗藏杀机……那样的话你就演砸了。
     </p>
     <p class="graf graf--p graf-after--p" id="4731">
      这是一件非常微妙的事，需要强烈的意识来控制它。绝大多数情况下负责这种隐蔽行动的监视方都是便衣，而不是伪装成乞丐、小贩什么的。因为没有那个必要，除非是集体行动的监视。
     </p>
     <p class="graf graf--p graf-after--p" id="5589">
      要注意，
      <strong class="markup--strong markup--p-strong">
       并不存在绝对意义上的“便衣”，人们通常说的便衣只是代表“此人没穿制服”。此人的便装也会带有浓厚的个性特征
      </strong>
      ，有可能他自己都意识不到这些。但很简单你可以判断。每个人的衣橱里都有几件根本不会去穿的衣服，你感觉它们不适合你（鬼知道当初为什么会买来）或者说不适合你的平常状态，不过请注意，
      <strong class="markup--strong markup--p-strong">
       那几件衣服就是你伪装自己的绝佳道具
      </strong>
      —— 它们不带有你最容易被暴露的个性特征。
     </p>
     <p class="graf graf--p graf-after--p" id="11fb">
      不论你是监视者，还是躲避监视的目标人，这些细节都是特别重要而需要掌握的。我们的主题是为后者服务，那么当你排查监视者时，请注意这些下意识中难以隐藏的东西。
     </p>
     <p class="graf graf--p graf-after--p" id="3ade">
      也就是说，经过练习你完全可以通过一个人的穿着细节、言谈举止来挖出其可能的真实身份信息。要知道，被派来实施这类秘密行动的人很可能都有军事或警务背景，他们经历过很多相关的训练，这些训练完全可以塑造其个性和品味，普通人都有这种经验，当过兵的人看起来不一样，没错你的直觉就是对的。
     </p>
     <p class="graf graf--p graf-after--p" id="eb66">
      并不是说看起来膀大腰圆、目光炯炯、穿着战术靴、使用坚固耐磨的手机壳和防水手表的人都是可疑人，绝对不是，上述所有观察点应该被结合起来使用。并且，充分调用你的直觉，它往往有很高的价值。
     </p>
     <p>
      如果您对反侦查感兴趣，我们在未来将会介绍更多实际操作的技巧，包括在被捆绑束缚后脱身的方法、以及能帮助躲避灾难的防身之术。
     </p>
     <p class="graf graf--p graf-after--p" id="8fd8">
      好啦，记得常练习哦，包括我们以前介绍过的观察人的技巧等，当你熟练之后就能游刃有余了。今后我们还会继续介绍相关技巧。
      <strong class="markup--strong markup--p-strong">
       反侦察是一门复杂且专业性很高的技术
      </strong>
      。⚪️
     </p>
    </div>
   </div>
