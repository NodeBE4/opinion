  <p class="graf graf--p">
   被骗意味着认知弱点被利用。
  </p>
  <p class="graf graf--p">
   骗子很擅长捕捉目标人/群体的认知弱点，他们一直专注于研究这些；而作为潜在的受骗者，我们中大多数人对自己的弱点并没有什么了解，这就很危险了。
  </p>
  <p class="graf graf--p">
   不能在造成损失后再去总结教训。
  </p>
  <p class="graf graf--p">
   欺骗非常常见。
   <strong class="markup--strong markup--p-strong">
    人类不仅是一个爱说谎话的物种，而且还具有从众的社会属性，很容易受到其他人观点的左右
   </strong>
   。
  </p>
  <p class="graf graf--p">
   我们获取信息的方式有三种：自己发现、内在吸收、外在地通过他人而得知。
  </p>
  <p class="graf graf--p">
   我们对世界的认知大多数都属于最后一种 — — 在认知世界过程的某个环节中有人告知我们、或者我们自己阅读得知，所以大多数人所获得的只是二手信息。
  </p>
  <p class="graf graf--p">
   我们依靠那些具有专业知识的人来告知。
  </p>
  <p class="graf graf--p">
   你没有亲眼见过氢原子和氧原子，但是很多文献都在介绍它们，描述了相关实验，于是你觉得自己“知道了”，从而相信它们的存在。
  </p>
  <p class="graf graf--p">
   想想看你拥有的知识，其中绝大部分都是这样得来的。
  </p>
  <p class="graf graf--p">
   你从来没有查证过它们，但是你深信：光速为186000英里/秒、巴氏消毒法真的能够杀死细菌、人体一般有23对染色体、跨越太平洋的海底光缆全长13200公里……
  </p>
  <p class="graf graf--p">
   你没有直接得知住宅楼的电梯“性能良好并维修及时”，只是因为广告上这样写着，并附有专家意见；你也没有直接了解到你挂号的医生读过医学院，只是他挂在墙上的资质证上这样写着……维基百科上告诉你的东西99.99%都没有被你亲自验证过，但是你相信它们。
  </p>
  <p class="graf graf--p">
   我们中绝大多数人
   <strong class="markup--strong markup--p-strong">
    依靠的是专家、认证书、许可证、百科全书和教科书
   </strong>
   。很多时候它们是有效的。
  </p>
  <p class="graf graf--p">
   但是，很显然，这种认知方式也很容易被骗。所以
   <strong class="markup--strong markup--p-strong">
    你还需要靠自己，依靠自己的智慧和推理能力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   那些想要骗取你的钱财、操纵你投下不利于自身的选票的说谎者，会试图用虚假的事实来蒙骗你，用没有依据的数字迷惑你，还会用经不起仔细推敲的不相干信息来使你分心。
  </p>
  <p class="graf graf--p">
   这些人经常会以专家的面目示人。
  </p>
  <p class="graf graf--p">
   💡使用我们前面分析统计数据和图表时所采用的那种方法来分析你遇到的说法与表述，是应对上述情况的正确方法。
  </p>
  <p class="graf graf--p">
   要做到这点所需要的必要技能不会超过绝大多数14岁孩子所具备的能力。所以你无需担心自己做不到。
  </p>
  <p class="graf graf--p">
   这些技能在法学院和新闻学院里都会讲授，商学院和研究生的科学课程中有时也会教授。但是，对于最需要这些技能的人来说，却很少有机会学习到。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果你喜欢收看刑事犯罪类电视剧，或者喜欢阅读调查性新闻（包括我们演示的OSINT调查技巧），你就会熟悉许多揭穿谎言的技巧
   </strong>
   — — 它们与庭审中所做的那种评估类似。
  </p>
  <p class="graf graf--p">
   法官和陪审员负责评估相互矛盾的陈述与主张，并试图从中找出真相。对于什么才能算作真正的证据，还有条例可以遵循。尽管有例外情况，但是一般来说未经验证的文件或“传闻”都不被允许用作证词。
  </p>
  <p class="graf graf--p">
   假设有人推荐你访问一个音乐网站。该网站称，每天听20分钟莫扎特的音乐会使我们变得更聪明。而另一个网站则表示，这个说法不真实。
  </p>
  <p class="graf graf--p">
   很多人都会相信前者。
  </p>
  <p class="graf graf--p">
   这里的主要问题在于，
   <strong class="markup--strong markup--p-strong">
    人脑常常根据情绪化的想法做出决定，然后再努力为已做出的决定进行辩护
   </strong>
   。
  </p>
  <p class="graf graf--p">
   人的大脑好比是一部非常强大的自我辩白机器。只需要听20分钟美妙的音乐就能使自己的智商排名突飞猛进，这当然再好不过了。
  </p>
  <p class="graf graf--p">
   就如很多人相信只要坚持不吃饭就能成功减肥、相信只要弄到枪就能实现民主……等等。事实上远比这要复杂的多。
  </p>
  <p class="graf graf--p">
   要评估这些说法还需要做更多努力。
  </p>
  <p class="graf graf--p">
   避免得出不正确的结论很有必要，因为即使最聪明的人也会被愚弄。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://www.telegraph.co.uk/technology/apple/8841347/SteveJobsregrettedtryingtobeatcancerwithalternativemedicineforsolong.html" href="https://www.telegraph.co.uk/technology/apple/8841347/SteveJobsregrettedtryingtobeatcancerwithalternativemedicineforsolong.html" rel="noopener noreferrer" target="_blank">
    史蒂夫·乔布斯（SteveJobs）

  </p>

  <p class="graf graf--p">
   记住这句话很重要：
   <strong class="markup--strong markup--p-strong">
    我们大多数人并非总是可以确定信息来源的真实性或准确性
   </strong>
   。而调查记者、情报分析师、法庭所做的主要工作就是验证。
  </p>
  <p class="graf graf--p">
   在这里看到基本验证的思考方式《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e7%9f%a5%e9%81%93%e4%bb%80%e4%b9%88%e6%89%8d%e6%98%af%e7%9c%9f%e6%ad%a3%e7%9a%84%e8%b0%83%e6%9f%a5%e5%90%97%ef%bc%9f%e6%89%80%e8%b0%93%e7%9a%84%e8%b5%b7%e5%ba%95%e7%bb%8f%e5%b8%b8/" href="https://www.iyouport.org/%e4%bd%a0%e7%9f%a5%e9%81%93%e4%bb%80%e4%b9%88%e6%89%8d%e6%98%af%e7%9c%9f%e6%ad%a3%e7%9a%84%e8%b0%83%e6%9f%a5%e5%90%97%ef%bc%9f%e6%89%80%e8%b0%93%e7%9a%84%e8%b5%b7%e5%ba%95%e7%bb%8f%e5%b8%b8/" rel="noopener noreferrer" target="_blank">
    你知道什么才是真正的*调查*吗？如何验证真伪
   </a>
   》。
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    <em>
     陷我们于困境的并非自己的无知，而是确信无疑之事其实不然。
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p">
   在故事片《大空头》的开头部分写着上面这句话。这部电影认为这句话的出处是马克·吐温，但事实上却不是。
  </p>
  <p class="graf graf--p">
   阿尔·戈尔（AlGore）在他自己出演的影片《难以忽视的真相》（AnInconvenientTruth）中也使用过这句话，而且他同样认为这句话的出处是马克·吐温。
  </p>
  <p class="graf graf--p">
   但是，查阅《武器化的谎言》（Weaponized Lies）找不到有任何依据能够证明马克·吐温曾说过这句话。
  </p>
  <p class="graf graf--p">
   两部影片的导演、编剧和制片人都没有做好准备工作 — — 他们认为自己确切地知道的事情，却根本不属实。
  </p>
  <p class="graf graf--p">
   在网络上略做搜索就能查到
   <a class="markup--anchor markup--p-anchor" data-href="http://www.forbes.com/2009/08/12/nigelreesmisquotesopinionsrees.html" href="https://www.forbes.com/2009/08/12/nigelreesmisquotesopinionsrees.html" rel="noopener noreferrer" target="_blank">
    《福布斯》杂志上的一篇文章
   </a>
   。其中写道，这句话的出处确实被张冠李戴了。
  </p>
  <p class="graf graf--p">
   文章的作者奈杰尔·里斯（Nigel Rees）通过查阅美国国会图书馆编纂的一本名言词典《恭敬引用》（Respectfully Quoted）得出了这一结论。
  </p>
  <p class="graf graf--p">
   事实上这句话是 Josh Billings 说的。
  </p>
  <p class="graf graf--p">
   Rees 写道，“你瞧，马克·吐温是比 Josh Billings 更有名的幽默家，所以那句话的出处就转移到马克·吐温头上了。”
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    沃尔特·蒙代尔（Walter Mondale）在1984年的一场总统竞选辩论中提出：“我想起了威尔·罗杰斯（Will Rogers）曾经对胡佛做过的评价。
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     他说‘让我烦恼的并不是他的无知，而是他确信自己知道的事并不属实’。
    </em>
   </strong>
   ”
  </p>
  <p class="graf graf--p">
   当面对全新或自相矛盾的表述时，人们通常会很好地参考和借鉴专家的观点和看法。那么，如何辨识专家意见？
  </p>
  <p class="graf graf--p">
   这里指的是真的专家，而不是伪装的砖家。真正的专家也是存在问题的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    专家的两种表述方式截然不同
   </strong>
  </p>
  <p class="graf graf--p">
   在评估权威人士的意见时，首先要问问自己：谁或什么使他们具有了权威性。
  </p>
  <p class="graf graf--p">
   如果权威性来自他们曾经见证了某些事件，那么这些见证又有多大的信服力？
  </p>
  <p class="graf graf--p">
   再崇高的权威也一定会出错。21世纪初，美国政府就曾误认为伊拉克存在大规模杀伤性武器；科学家们曾经都认为人类的染色体数量为24对，而不是23对……
  </p>
  <p class="graf graf--p">
   因此，
   <strong class="markup--strong markup--p-strong">
    参考社会公认权威的说法并不是评估某种表述的最后一个步骤，反而只是一个开始。
   </strong>
  </p>
  <p class="graf graf--p">
   一般来说，专家这个称呼专指这些人：接受过特殊培训的人、花费了大量的时间来拓展专业知识的人（如医生、飞行员、音乐家和运动员等），以及那些能力和知识被认为与其他人密切相关的人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    专家们的谈话方式有两种
   </strong>
   ，在第一种谈话方式中，他们回顾事实和证据，将其综合形成一种基于证据的结论。
  </p>
  <p class="graf graf--p">
   在这个过程当中，他们向你说明什么是证据，为什么它们具有相关性，以及这些证据怎样能够帮助他们得出结论。
  </p>
  <p class="graf graf--p">
   这时候你只需要小心是否有不是证据的信息被掺杂其中了。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e7%9f%a5%e9%81%93%e4%bb%80%e4%b9%88%e6%89%8d%e6%98%af%e7%9c%9f%e6%ad%a3%e7%9a%84%e8%b0%83%e6%9f%a5%e5%90%97%ef%bc%9f%e6%89%80%e8%b0%93%e7%9a%84%e8%b5%b7%e5%ba%95%e7%bb%8f%e5%b8%b8/" href="https://www.iyouport.org/%e4%bd%a0%e7%9f%a5%e9%81%93%e4%bb%80%e4%b9%88%e6%89%8d%e6%98%af%e7%9c%9f%e6%ad%a3%e7%9a%84%e8%b0%83%e6%9f%a5%e5%90%97%ef%bc%9f%e6%89%80%e8%b0%93%e7%9a%84%e8%b5%b7%e5%ba%95%e7%bb%8f%e5%b8%b8/" rel="noopener noreferrer" target="_blank">
    在这里看到如何判断什么是证据
   </a>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    科学理应遵循这种方式
   </strong>
   ，法庭审判也应当采用这种方式，而且最佳的商业决策、医疗诊断和军事战略的制定都应当采取这种方式。
  </p>
  <p class="graf graf--p">
   而第二种谈话方式中，⚠️专家只和他人分享观点。这是非常值得警惕的。
  </p>
  <p class="graf graf--p">
   要知道专家也是人，他们也具有容易被骗的人性弱点；他们还有可能因接受了金钱贿赂而仅仅为某一利益方说话。
  </p>
  <p class="graf graf--p">
   权威专家和科学家为大众编写的书籍和撰写的文章经常包含那种狂放不羁、天马行空的臆测，而人们之所以能够欣然接受，只是因为作者的专业知识和修辞给人们留下了深刻的印象。
  </p>
  <p class="graf graf--p">
   专业知识可以说是一种社会性的判断，我们经常会比较的是一个人的技能和世界上其他人的技能水平。
  </p>
  <p class="graf graf--p">
   具备相似培训水平和专业知识的专业人士不一定赞同彼此的观点。即便能够达成一致意见，他们的观点也不会总正确。
  </p>
  <p class="graf graf--p">
   成千上万的专业财务分析员所做的股票行情预测完全错误，而少数的行业新人反而能做到正确预测；就如我们在《战略性欺骗》系列中所分析的，资深级的情报分析师往往有严重的思维固化，他们更加难以发现新手很容易发现的逻辑漏洞。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5551 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/专家.jpg?resize=1100%2C763&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/专家.jpg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/专家.jpg?resize=300%2C208&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/专家.jpg?resize=768%2C533&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/专家.jpg?resize=1024%2C710&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/专家.jpg?resize=1100%2C763&amp;ssl=1 1100w" data-recalc-dims="1" height="763" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/专家.jpg?resize=1100%2C763&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    专家有什么问题？
   </strong>
  </p>
  <p class="graf graf--p">
   专业知识意味着更窄的知识面。
  </p>
  <p class="graf graf--p">
   去年开始中国国社要求被约稿人必须带有足够高的头衔，比如名校、专家身份、领域知名身份等等；这点可能也是所有商业媒体所追求的。
  </p>
  <p class="graf graf--p">
   ⚠️头衔之所以能成为吸引流量的重要诱饵 —— 而不是内容本身 —— 正因为大多数人都会下意识地视专家头衔为全能，事实上远远不是如此。
  </p>
  <p class="graf graf--p">
   当你看到纽约时报中文专栏中充满愚蠢的观点和完全没有证据的论断时就能明白这个道理；或者更简单的是，看看社交媒体上的大V们是怎么出洋相的。
  </p>
  <p class="graf graf--p">
   头衔和奖项可能意味着此人在相关一个领域有所建树，但是，绝对不代表此人在任何领域都是明白的。
  </p>
  <p class="graf graf--p">
   作为三位晶体管发明者之一的 William Shockley 获得了诺贝尔物理学奖。后来，他所大力宣扬的种族主义观点之所以能够在社会上站住脚，正是因为人们被诺贝尔奖的光环忽悠了，误认为能拿到此奖的人肯定知道一些其他人不知道的事。
  </p>
  <p class="graf graf--p">
   世界上最优秀的丰田汽车修理工有可能无法为你的大众汽车做出有效的故障判断；最优秀的税务律师也许也无法给出违约诉讼方面的最佳建议 ……
  </p>
  <p class="graf graf--p">
   最重要的在于诚实。作为专家本身，他们应该诚实地告诉人们他们做不到什么；但是，尤其是在互联网上，他们经常被光环所蒙蔽，追热点的个人营销热情导致他们参与了很多他们自己并不熟悉的领域，也因此骗了很多人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    反知识
   </strong>
  </p>
  <p class="graf graf--p">
   英国记者达米安·汤普森（DamianThompson）创造了“反知识”一词，用来指那些
   <strong class="markup--strong markup--p-strong">
    经过包装之后表面看似真实但实际上是错误的信息
   </strong>
   。
  </p>
  <p class="graf graf--p">
   科学、时事新闻、明星八卦和伪科学中都有这方面的例子。
  </p>
  <p class="graf graf--p">
   反知识包括缺少支撑性证据的表述和那些虽有证据但却与其相矛盾的表述。
  </p>
  <p class="graf graf--p">
   反知识不总是阴谋论，虽然有些时候是。
  </p>
  <p class="graf graf--p">
   英文媒体说，在中国“少有人 了解30年前的6月初发生了什么”。这是根据网络封锁和信息封锁的严重性推断出来的。但这可能不是事实。
  </p>
  <p class="graf graf--p">
   多年前开始，中国的大型门户网站内部就在6月初组织员工观看一部纪录片，那是官方角度拍摄的记录6月4日发生的事，其镜头和解说在为官方的暴行做申辩（官方将暴力归为民众）。
  </p>
  <p class="graf graf--p">
   但是民间缺乏如此充分的独立记录，或许源于当时人们没有智能手机 —— 记录事实的权力是不对称的。这也是为什么6月时我们将网站的置顶文章调整为《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%85%ac%e6%b0%91%e5%8f%96%e8%af%81%e6%8a%80%e5%b7%a7%ef%bc%9a%e5%a6%82%e4%bd%95%e7%95%99%e4%bd%8f%e7%9c%9f%e7%9b%b8%ef%bc%9f/" href="https://www.iyouport.org/%e5%85%ac%e6%b0%91%e5%8f%96%e8%af%81%e6%8a%80%e5%b7%a7%ef%bc%9a%e5%a6%82%e4%bd%95%e7%95%99%e4%bd%8f%e7%9c%9f%e7%9b%b8%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    公民取证技巧：如何留住真相？
   </a>
   》、《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ae%89%e5%85%a8%e5%9c%b0%e7%95%99%e4%bd%8f%e8%af%81%e6%8d%ae%ef%bc%9a%e7%bb%99%e7%9c%9f%e7%9b%b8%e7%9a%84%e8%ae%b0%e5%bd%95%e8%80%85%e4%b8%80%e4%ba%9b%e6%8a%80%e6%9c%af%e6%8a%80/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ae%89%e5%85%a8%e5%9c%b0%e7%95%99%e4%bd%8f%e8%af%81%e6%8d%ae%ef%bc%9a%e7%bb%99%e7%9c%9f%e7%9b%b8%e7%9a%84%e8%ae%b0%e5%bd%95%e8%80%85%e4%b8%80%e4%ba%9b%e6%8a%80%e6%9c%af%e6%8a%80/" rel="noopener noreferrer" target="_blank">
    保存真相的技术技巧
   </a>
   》、《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%8b%8d%e6%91%84%e5%8f%96%e8%af%81-2%ef%bc%9a%e8%af%a6%e8%bf%b0%e6%ad%a5%e9%aa%a4%e5%92%8c%e5%85%b7%e4%bd%93%e6%8a%80%e5%b7%a7/" href="https://www.iyouport.org/%e6%8b%8d%e6%91%84%e5%8f%96%e8%af%81-2%ef%bc%9a%e8%af%a6%e8%bf%b0%e6%ad%a5%e9%aa%a4%e5%92%8c%e5%85%b7%e4%bd%93%e6%8a%80%e5%b7%a7/" rel="noopener noreferrer" target="_blank">
    拍摄取证的步骤和技巧
   </a>
   》。
  </p>
  <p class="graf graf--p">
   掌握这些技巧在任何时候都能拿出证据以揭露谎言。
  </p>
  <p class="graf graf--p">
   简单说，人们并不是不知道发生了事件，而是会以不同的理解方式来认识这件事。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    因为看过那部官方拍摄的纪录片的人会受到反知识的困扰
   </strong>
   。
  </p>
  <p class="graf graf--p">
   反知识无处不在，⚠️而且比真相更容易被传播。其中《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%ba%e6%a0%bc%e6%9a%97%e6%9d%80%e5%92%8c%e6%8a%b9%e9%bb%91%e8%bf%90%e5%8a%a8%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%8f%97%e7%9b%8a%ef%bc%9f/" href="https://www.iyouport.org/%e4%ba%ba%e6%a0%bc%e6%9a%97%e6%9d%80%e5%92%8c%e6%8a%b9%e9%bb%91%e8%bf%90%e5%8a%a8%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%8f%97%e7%9b%8a%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    人格暗杀
   </a>
   》的操作是 一个典型。
  </p>
  <p class="graf graf--p">
   帮助反知识传播的一部分推动力正是这样的诡计 —— 即 激发人们去想象“万一它是真的呢”。
  </p>
  <p class="graf graf--p">
   很多反知识一开始就能吸引人们，所凭借的正是利用数字和统计数据装扮出来的一种*知识的外衣* —— 正如前三篇文章中所分析的。
  </p>
  <p class="graf graf--p">
   但深入研究就能发现，
   <strong class="markup--strong markup--p-strong">
    这些反知识实际上根本没有根据 — — 反知识的传播者希望的就是通过数字的呈现使你对它们产生足够的印象（或者惧怕），从而盲目地接受它们
   </strong>
   。
  </p>
  <p class="graf graf--p">
   记者 Damian Thompson 报道了这些反知识表述如何能站住脚、如何惹恼我们，使我们对已知事物产生怀疑……也就是说，这个过程直到我们运用理性分析时才会结束。
  </p>
  <p class="graf graf--p">
   Thompson 回忆了一位朋友讲起美国“9·11”恐怖袭击。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他用一番看似合理的言论抓住了我们的注意力，‘请从垂直方向观察纽约世贸中心双子塔的倒塌过程，而不要从侧面观察。飞机燃料不会产生足以熔化钢筋的热量，只有受控爆炸才能做到这一点’”。
  </p>
  <p class="graf graf--p">
   听起来像专家一样？
  </p>
  <p class="graf graf--p">
   但事实上是这样的：
  </p>
  <p class="graf graf--p">
   双子塔从垂直方向轰然坍塌，这是真实情况，人们看到过视频剪辑内容。假如那次袭击所采用的方式如他们所说的那样，你就会看到该建筑物倾倒：这一点并无明确的说明，属于隐藏前提。
  </p>
  <p class="graf graf--p">
   我们无从知晓它是否属实。仅仅由于说话者做了肯定的宣称并不意味着它就真实。还需要进一步证实这个表述。
  </p>
  <p class="graf graf--p">
   飞机燃料不会产生足以熔化钢筋的热量：我们同样不知道这句表述真实与否。而且，它还忽略了其他易燃物 — — 清洁用品、油漆、工业化学品 — — 它们可能就存放于双子塔里面，所以这些易燃物一旦着火势必会火上浇油。
  </p>
  <p class="graf graf--p">
   如果你不是专业结构工程师，也许会觉得这些假设前提似乎合理。但稍微查看一下资料就会发现，专业结构工程师并没有发现双子塔的倒塌有什么神秘之处。
  </p>
  <p class="graf graf--p">
   一种解释或理论所具有的些许漏洞并不会破坏其本身的可信度。一
  </p>
  <p class="graf graf--p">
   已经确立的理论具有成千上万的支撑性证据，少量未做解释的异常现象并不会破坏或减少它的可信度。
  </p>
  <p class="graf graf--p">
   但这些异常现象一般都占据所有阴谋思维的核心位置，比如大屠杀修正主义、反进化论主义，以及“9·11”恐怖袭击事件阴谋论。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    用反知识骗人
   </strong>
  </p>
  <p class="graf graf--p">
   就如我们的“战略性欺骗”中所强调的，欺骗的最佳方式绝不能是完全虚假，必须有部分真实，这样才更容易让对方上当。
  </p>
  <p class="graf graf--p">
   如果你想用反知识来欺骗人们，可以采用这样一种有效的手段：使用大量可以核实的事实信息，然后在其中加入一两个虚假信息。
  </p>
  <p class="graf graf--p">
   你所使用的那些真实的信息可以给人们提供真相；如果有任何人去搜索他们也一定会确认为真。
  </p>
  <p class="graf graf--p">
   所以，你只要增添一两个虚假信息来表达你的观点，很多人就会赞同你。
  </p>
  <p class="graf graf--p">
   因此，当你想要说服他人时，就可以采用这种把伪造事实或反知识与真实事实和实际知识结合在一起的方式。
  </p>
  <p class="graf graf--p">
   比如这样：
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    水是由氢元素和氧元素构成的。
   </li>
   <li class="graf graf--li">
    水的分子式是H2O。
   </li>
   <li class="graf graf--li">
    我们的身体由60%的水分构成。
   </li>
   <li class="graf graf--li">
    人类血液的92%都是水分。
   </li>
   <li class="graf graf--li">
    大脑含有75%的水分。
   </li>
   <li class="graf graf--li">
    世界上很多地方都有受到污染的水。
   </li>
   <li class="graf graf--li">
    世界上的可用水源只有1%可以饮用。
   </li>
   <li class="graf graf--li">
    如果你买了瓶装水，那么你唯一可以肯定的就是你的饮用水的品质很高。
   </li>
   <li class="graf graf--li">
    主流健康研究人员建议人们饮用瓶装水，并且大多数人都在饮用瓶装水。
   </li>
  </ol>
  <p class="graf graf--p">
   上述推理过程中的前7个论断都属实。第八条论断在逻辑上并没有遵循整个推理过程的逻辑，而第九条论断……谁是主流健康研究人员？他们自己饮用瓶装水的含义是什么？
  </p>
  <p class="graf graf--p">
   事实上在发达国家瓶装水不比自来水更安全和健康，这是
   <a class="markup--anchor markup--p-anchor" data-href="http://www.nrdc.org/water/drinking/qbw.asp" href="http://www.nrdc.org/water/drinking/qbw.asp" rel="noopener noreferrer" target="_blank">
    有大量调查作为证据的
   </a>
   。
  </p>
  <p class="graf graf--p">
   相反，如果你不想被骗，就需要时刻警惕这种手段。
  </p>
  <p class="graf graf--p">
   ⚠️当证据本身不完整时这种手段最为高效。当权者经常使用不完整的证据来支持他们的谎言。就如中国官方的“64纪录片”。
  </p>
  <p class="graf graf--p">
   这就是为什么透明度革命尤其重要；审查封锁和算法暴政尤其致命。只有我们拿到充分的证据才能反驳。
  </p>
  <p class="graf graf--p">
   顺便推荐一本书《Counterknowledge: how we surrendered to conspiracy theories, quack medicine, bogus science and fake history》。
  </p>
  <p class="graf graf--p">
   最后送您一个娱乐性的小片子，关于你的眼睛如何欺骗你。
  </p>
