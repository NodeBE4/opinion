  <p class="graf graf--p">
   Inquisitor H3x 推荐了一个工具：Tsurugi Linux，这是专为DFIR / 开源情报调查而定制和设计的发行版。它还具有恶意软件分析以及其他工具。
  </p>
  <p class="graf graf--p">
   效果很不错，您值得尝试下。他介绍了使用方法，在这里看到原文《
   <a href="https://medium.com/@inquisitorh3x/tsurugi-setting-up-your-environment-9e3a9225f380" rel="noopener noreferrer" target="_blank">
    Tsurugi — Setting Up Your Environment
   </a>
   》
  </p>
  <p class="graf graf--p">
   👉创建一个纯粹用于
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    开源情报调查
   </a>
   的环境；帮助您学习开源情报技术，而不必担心感染或破坏自己的笔记本电脑。当然，也为您提供了一定程度的隐私。
  </p>
  <p class="graf graf--p">
   现在，由于虚拟机的可用性，您不必去寻找旧笔记本电脑或完全擦除当前设备也能开始调查工作。
  </p>
  <p class="graf graf--p">
   这就是 Tsurugi 值得您拥有的原因。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Tsurugi 的优点是，它已经在虚拟机中设置了大多数基本工具，可以直接进入 OSINT 和 DFIR
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Inquisitor H3x 介绍了如何安装 Tsurugi 的分步指南；还将向您展示如何连接虚拟机和主机。这样，您可以在两者之间移动资源。这也将有助于为任何调查工作/项目保持专用的环境。
  </p>
  <p class="graf graf--p">
   Tsurugi（剣）是日本古代使用的双刃剑。

  </p>
  <p class="graf graf--p">
   然后输入 ”chmod +x *.sh”，再按enter。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13291 jetpack-lazy-image" data-lazy-sizes="(max-width: 481px) 100vw, 481px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/33-1.png?resize=481%2C131&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/33-1.png?w=481&amp;ssl=1 481w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/33-1.png?resize=300%2C82&amp;ssl=1 300w" data-recalc-dims="1" height="131" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/33-1.png?resize=481%2C131&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="481"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13291" data-recalc-dims="1" height="131" sizes="(max-width: 481px) 100vw, 481px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/33-1.png?resize=481%2C131&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/33-1.png?w=481&amp;ssl=1 481w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/33-1.png?resize=300%2C82&amp;ssl=1 300w" width="481"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   然后输入 “ ./autorun.sh”，再enter。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13292 jetpack-lazy-image" data-lazy-sizes="(max-width: 480px) 100vw, 480px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/34.png?resize=480%2C132&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/34.png?w=480&amp;ssl=1 480w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/34.png?resize=300%2C83&amp;ssl=1 300w" data-recalc-dims="1" height="132" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/34.png?resize=480%2C132&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="480"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13292" data-recalc-dims="1" height="132" sizes="(max-width: 480px) 100vw, 480px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/34.png?resize=480%2C132&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/34.png?w=480&amp;ssl=1 480w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/34.png?resize=300%2C83&amp;ssl=1 300w" width="480"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   应该显示以下窗口：
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-13293 jetpack-lazy-image" data-lazy-sizes="(max-width: 602px) 100vw, 602px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/35.png?resize=602%2C310&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/35.png?w=602&amp;ssl=1 602w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/35.png?resize=300%2C154&amp;ssl=1 300w" data-recalc-dims="1" height="310" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/35.png?resize=602%2C310&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="602"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13293" data-recalc-dims="1" height="310" sizes="(max-width: 602px) 100vw, 602px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/35.png?resize=602%2C310&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/35.png?w=602&amp;ssl=1 602w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/35.png?resize=300%2C154&amp;ssl=1 300w" width="602"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   输入密码，然后按Enter。
  </p>
  <p class="graf graf--p">
   该程序将运行并安装。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13294 jetpack-lazy-image" data-lazy-sizes="(max-width: 602px) 100vw, 602px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/36.png?resize=602%2C311&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/36.png?w=602&amp;ssl=1 602w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/36.png?resize=300%2C155&amp;ssl=1 300w" data-recalc-dims="1" height="311" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/36.png?resize=602%2C311&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="602"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13294" data-recalc-dims="1" height="311" sizes="(max-width: 602px) 100vw, 602px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/36.png?resize=602%2C311&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/36.png?w=602&amp;ssl=1 602w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/36.png?resize=300%2C155&amp;ssl=1 300w" width="602"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   完成后，别忘了按 Enter 键。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    步骤11：添加 Vboxsf 用户
   </strong>
  </p>
  <p class="graf graf--p">
   完成安装后，打开一个新终端并输入以下命令 “sudo adduser (UserName) vboxsf ”，并按 Enter。
  </p>
  <p class="graf graf--p">
   用户将被添加。
  </p>
  <p class="graf graf--p">
   完成此操作后，重新启动虚拟机。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13295 jetpack-lazy-image" data-lazy-sizes="(max-width: 602px) 100vw, 602px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/37.png?resize=602%2C158&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/37.png?w=602&amp;ssl=1 602w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/37.png?resize=300%2C79&amp;ssl=1 300w" data-recalc-dims="1" height="158" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/37.png?resize=602%2C158&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="602"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13295" data-recalc-dims="1" height="158" sizes="(max-width: 602px) 100vw, 602px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/37.png?resize=602%2C158&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/37.png?w=602&amp;ssl=1 602w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/37.png?resize=300%2C79&amp;ssl=1 300w" width="602"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13296 jetpack-lazy-image" data-lazy-sizes="(max-width: 602px) 100vw, 602px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/38.png?resize=602%2C158&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/38.png?w=602&amp;ssl=1 602w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/38.png?resize=300%2C79&amp;ssl=1 300w" data-recalc-dims="1" height="158" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/38.png?resize=602%2C158&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="602"/>

   重新启动虚拟机。
  </p>

    <img alt="" class="aligncenter size-full wp-image-13297" data-recalc-dims="1" height="310" sizes="(max-width: 602px) 100vw, 602px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/39.png?resize=602%2C310&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/39.png?w=602&amp;ssl=1 602w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/03/39.png?resize=300%2C154&amp;ssl=1 300w" width="602"/>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    步骤12：安装完成（希望如此）
   </strong>
  </p>
  <p class="graf graf--p">
   登录后，您应该在虚拟机的桌面上看到一个驱动器 “sf_您命名为位置的文件夹”。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13298 jetpack-lazy-image" data-lazy-sizes="(max-width: 600px) 100vw, 600px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/40.png?resize=600%2C402&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/40.png?w=600&amp;ssl=1 600w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/40.png?resize=300%2C201&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/40.png?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="402" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/40.png?resize=600%2C402&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="600"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13298" data-recalc-dims="1" height="402" sizes="(max-width: 600px) 100vw, 600px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/40.png?resize=600%2C402&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/40.png?w=600&amp;ssl=1 600w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/40.png?resize=300%2C201&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/03/40.png?resize=272%2C182&amp;ssl=1 272w" width="600"/>

  <p class="graf graf--p">
   现在，您将能够将数据从虚拟机保存到主机，反之亦然。
  </p>
  <p class="graf graf--p">
   您当前所在的主屏幕称为DFIR工作区。在桌面上，您会注意到一个图标 “ OSINT Switcher”。
  </p>
  <p class="graf graf--p">
   选择该选项将使您移至OSINT工作区。这里有专门针对开源情报调查定制的工具和设置。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-13299 jetpack-lazy-image" data-lazy-sizes="(max-width: 601px) 100vw, 601px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/41.png?resize=601%2C326&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/41.png?w=601&amp;ssl=1 601w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/41.png?resize=300%2C163&amp;ssl=1 300w" data-recalc-dims="1" height="326" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/03/41.png?resize=601%2C326&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="601"/>
   <noscript>
