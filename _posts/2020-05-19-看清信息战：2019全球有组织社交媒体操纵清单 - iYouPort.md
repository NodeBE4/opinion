  <p>
   但近年来在该领域出现的 “进展” 依旧是惊人的，即 “计算性宣传”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    使用算法、自动化和大数据来操纵公共生活
   </strong>
   的手段被称为 “计算性宣传”，它正在成为日常生活中无处不在的一部分。
  </p>
  <p class="graf graf--p">
   牛津互联网学院 2019年的报告分析了计算性宣传的趋势以及不断发展的工具、能力、策略和资源。以下是这份报告，非常建议您读完。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     概览
    </strong>
    ：
   </span>
  </h3>
  <ol class="postList">
   <li class="graf graf--li">
    报告展示了在
    <strong class="markup--strong markup--li-strong">
     70个国家/地区
    </strong>
    开展的有组织社交媒体操纵运动的证据，高于2018年的48个国家、和2017年的28个国家。在每个国家，至少有一个政党或政府机构使用社交媒体在国内操纵公众态度（图1）。
   </li>
   <li class="graf graf--li">
    社交媒体已成为被许多专制政权所采用为宣传工具。在26个国家/地区中，计算性宣传正以
    <strong class="markup--strong markup--li-strong">
     三种截然不同的方式
    </strong>
    用作信息控制的工具：压制基本人权、抹黑政治对手、并淹没反对意见（图2）。
   </li>
   <li class="graf graf--li">
    少数精明的国家行为者在
    <strong class="markup--strong markup--li-strong">
     对外国的影响力行动
    </strong>
    中也使用计算性宣传。 Facebook 和 Twitter 将外国影响力业务归于主要七个国家（
    <strong class="markup--strong markup--li-strong">
     中国，印度，伊朗，巴基斯坦，俄罗斯，沙特阿拉伯和委内瑞拉
    </strong>
    ），这些国家一直在使用这些平台来影响全球受众（图3）。
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     中国已成为全球虚假信息信息战的主要参与者
    </strong>
    。在2019年香港抗议之前，已经有大多数有关中国操作计算性宣传的证据都发生在微博、微信和QQ等中国国内平台上。但是，中国对积极使用 Facebook、Twitter 和 YouTube 的新兴趣应该引起民主国家的担忧。
   </li>
   <li class="graf graf--li">
    尽管如今社交网络平台比以往任何时候都要多，但
    <strong class="markup--strong markup--li-strong">
     Facebook 仍然是社交媒体操纵的首选平台
    </strong>

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     介绍
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   尽管宣传一直是政治讨论的一部分，但是，这些运动的广泛范围引起了公众极大的关注。
  </p>
  <p class="graf graf--p">
   网络部队（网军）被定义为负责在线操纵公众舆论的政府或政党参与者。调查研究并比较了全球网络部队的
   <strong class="markup--strong markup--p-strong">
    正式组织
   </strong>
   ，以及这些行为者如何
   <strong class="markup--strong markup--p-strong">
    出于政治目的
   </strong>
   使用计算性宣传。
  </p>
  <p class="graf graf--p">
   这涉及建立不断发展的宣传策略、工具和技术的清单，包括使用 “政治机器人” 来放大仇恨言论或其他形式的可操纵内容、非法收集数据或进行微目标定位、或部署的 “trolls” 军队在网上欺凌或骚扰政治异见人士和记者。
  </p>

  <p class="graf graf--p">
   报告还将跟踪开发这些技术所投入的能力和资源，以构建审视全球网络部队能力的图景。
  </p>
  <p class="graf graf--p">
   通过社交媒体使用计算性宣传来*塑造*公众态度已经成为主流，远远超出了一些不良行为者的行为。
  </p>
  <p class="graf graf--p">
   在信息量大、用户关注和信任水平有限的信息环境中，
   <strong class="markup--strong markup--p-strong">
    计算性宣传的工具和技术正成为数字竞选和公共外交的共同部分，并且可以说是必不可少的部分。
   </strong>
  </p>
  <p class="graf graf--p">
   除了建立全球范围内网络部队活动的比较图，报告还希望就此引起公众和学术界的讨论，即：我们如何定义和理解在线政治的不断变化的性质，以及应该如何使用技术来增强在线民主和自由表达的人权。
  </p>
  <p class="graf graf--p">
   在今年的报告中，牛津互联网学院研究了70个国家/地区的网络部队活动：安哥拉，阿根廷，亚美尼亚，澳大利亚，奥地利，阿塞拜疆，巴林，波斯尼亚和黑塞哥维那，巴西，柬埔寨，中国，哥伦比亚，克罗地亚，古巴，捷克共和国，厄瓜多尔，埃及，厄立特里亚，埃塞俄比亚，格鲁吉亚，德国，希腊，洪都拉斯，危地马拉，匈牙利，印度，印度尼西亚，伊朗，以色列，意大利，哈萨克斯坦，肯尼亚，吉尔吉斯斯坦，马其顿，马来西亚，马耳他，墨西哥，摩尔多瓦，缅甸，荷兰，尼日利亚，朝鲜，巴基斯坦，菲律宾，波兰，卡塔尔，俄罗斯，卢旺达，沙特阿拉伯，塞尔维亚，南非，韩国，西班牙，斯里兰卡，瑞典，叙利亚，台湾，塔吉克斯坦，泰国，突尼斯，土耳其，乌克兰，阿联酋，英国，美国，乌兹别克斯坦，委内瑞拉，越南和津巴布韦。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     全世界越来越多的计算性宣传证据
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   调查发现在70个国家/地区开展的有组织的社交媒体操纵运动的证据，高于2018年的48个国家和2017年的28个国家。
  </p>
  <p class="graf graf--p">
   这种增长的一部分来自新的加入者，他们在选举期间尝试使用计算性宣传的工具和技术、或作为信息控制的新手段。
  </p>
  <p class="graf graf--p">
   但是，新闻记者、学者和激进主义者也更好地配备了数字工具和更精确的词汇表，
   <strong class="markup--strong markup--p-strong">
    以识别、报告和发现正式组织的社交媒体操纵的实例
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在过去的三年中，我们已经能够完善语言和搜索条件以识别计算性宣传实例，并且我们发现，在过去的十年中，许多国家展示了正式组织的社交媒体操纵的要素。
  </p>
  <p class="graf graf--p">
   牛津互联网学院指出，计算性宣传已成为数字信息生态系统中普遍存在的部分。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     威权政体对社交媒体的共同选择
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在许多专制政权中，计算性宣传已成为信息控制的有力工具，
   <strong class="markup--strong markup--p-strong">

  <h3 class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     内容分析 ——
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   这是一种在传播和媒体研究中建立的研究方法。它已被用来帮助理解
   <strong class="markup--strong markup--p-strong">
    互联网和社交媒体如何与政治行动、政权转型和数字控制互相作用
   </strong>
   。
  </p>
  <p class="graf graf--p">
   进行此定性内容分析的目的是了解积极使用社交媒体操纵舆论的国家行为者的范围及其能力、策略和资源。
  </p>
  <p class="graf graf--p">
   本报告根据去年的报告对内容分析进行了建模，并使用了有目的的抽样来构建新闻文章中出现的特定变量的编码电子表格。
  </p>
  <p class="graf graf--p">
   选择了以下关键字并将其组合用于搜索：机器人、剑桥分析公司、虚假信息、假账号、信息战、Instagram、军事、误传、宣传、心理操纵、心理战、社交媒体、袜子木偶、trolls、推特、WhatsApp、YouTube。
  </p>
  <p class="graf graf--p">
   进行此定性内容分析有两个主要限制：媒体偏见和语言。
  </p>
  <p class="graf graf--p">
   为了帮助减轻偏见，报告使用了 LexisNexis 和排名前三的搜索引擎提供商 —— Google、Yahoo!、和 Bing —— 搜索各种专业、本地和业余新闻来源提供的热门话题。
  </p>
  <p class="graf graf--p">
   为了确保仅使用高质量的新闻源来构建调查的数据集，每篇文章均使用三点量表获得可信度评分。
  </p>
  <p class="graf graf--p">
   排名第一的文章来自主要的专业品牌新闻机构。排名第二的文章来自较小的专业新闻机构、本地新闻机构或专家评论和专业博客。排名第三的文章来自内容农场、个人或超党派博客。这些从样品中移出。
  </p>
  <p class="graf graf--p">
   语言是进行此分析的第二个限制。对于今年的全球清单，我们能够利用阿拉伯文，英文，法文，德文，希腊文，匈牙利文，意大利文，波斯语，波兰语，葡萄牙语，俄语和西班牙语。

   我们还与英国广播公司（BBC）监控部门合作，后者提供了一个额外的门户网站，用于收集和汇总有关网络部队活动的高质量新闻和信息，并为波斯尼亚，克罗地亚，格鲁吉亚，哈萨克斯坦，吉尔吉斯斯坦，马来西亚，北马其顿，台湾，塔吉克斯坦，土库曼斯坦，乌兹别克斯坦，提供翻译服务。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    仅依靠英语的部分
   </strong>
   包括以下国家和地区的内容：亚美尼亚，阿塞拜疆，柬埔寨，

     策略，工具和技术 ——
    </strong>

  <p class="graf graf--p">
   尽管宣传本身没有什么新意，但是，
   <strong class="markup--strong markup--p-strong">

   </strong>
   。
  </p>
  <p class="graf graf--p">
   网络部队使用虚假帐户来传播计算性宣传。

   </noscript>

  <p class="graf graf--p">
   在过去的三年中，牛津互联网学院跟踪了三种类型的虚假帐户的流行情况：bot、真人和电子人。
  </p>
  <p class="graf graf--p">
   👉Bot 是高度自动化的帐户，旨在模拟在线人的行为。它们通常用于扩大叙述或淹没政治异议。
  </p>
  <p class="graf graf--p">
   调查发现有证据表明在70个国家中有
   <strong class="markup--strong markup--p-strong">
    50个国家
   </strong>
   使用了机器人帐户。
  </p>
  <p class="graf graf--p">
   👉但是，比机器人
   <strong class="markup--strong markup--p-strong">
    更常见
   </strong>
   的是
   <strong class="markup--strong markup--p-strong">
    人工账户
   </strong>
   ，它们没有利用自动化。相反，他们通过发布评论或推文或通过社交媒体平台的私人消息传递来干扰对话。
  </p>
  <p class="graf graf--p">
   在今年的报告中，在70个国家中有60个国家被发现了人工帐户的使用。结合自动化与人工管理的电子人帐户是确定的另一种帐户类型。
  </p>
  <p class="graf graf--p">
   👉今年的报告将假冒帐户的类型增加了被黑的或被盗的帐户。
  </p>
  <p class="graf graf--p">
   尽管这些帐户本身并不是“虚假”的，但网络部队在战略上
   <strong class="markup--strong markup--p-strong">
    使用了高知名度的帐户，以散布亲政府的宣传或通过撤消合法所有者对帐户的访问权限来限制言论自由
   </strong>
   。
  </p>
  <p class="graf graf--p">
   👉少数国家行为者已开始将被盗或被黑客入侵的帐户用作其在线操纵行动的一部分，突显了
   <strong class="markup--strong markup--p-strong">
    计算性宣传与更传统形式的网络攻击之间的密切联系
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️最后，需要注意的是，并非所有用于网络部队活动的帐户都是伪造的。你知道中国的自干五和 “高级五毛”吗？
  </p>
  <p class="graf graf--p">
   不仅中国。在某些国家，例如越南或塔吉克斯坦，国家行为者鼓励网络部队使用其真实账户来传播亲政府的宣传、挑衅政治异议者或大规模报道政府定制的内容。
  </p>
  <p class="graf graf--p">
   随着社交媒体公司在减少与网络部队活动相关联的帐户方面变得更加积极主动，
   <strong class="markup--strong markup--p-strong">
    对真实帐户的共同选择可能会成为操纵者更为重要的策略
   </strong>
   。

  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   在其他情况下，网络部队则由专职人员整合到媒体和通讯领域，以控制、审查和塑造在线对话和信息。
  </p>
  <p class="graf graf--p">
   👉一些团队由管理数百个假账户的少数人组成。在其他国家（例如中国、越南或委内瑞拉），国家雇用大批人员通过在线渠道积极塑造舆论和监视对话。

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     结论
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   曾经被誉为自由和民主力量的社交媒体，如今已经……
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     《
    </strong>
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e7%a0%b4%e5%9d%8f%e6%b0%91%e4%b8%bb%e7%9a%84%e8%af%81%e6%8d%ae%e8%b6%8a%e6%9d%a5%e8%b6%8a%e5%a4%9a/" href="https://www.iyouport.org/%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e7%a0%b4%e5%9d%8f%e6%b0%91%e4%b8%bb%e7%9a%84%e8%af%81%e6%8d%ae%e8%b6%8a%e6%9d%a5%e8%b6%8a%e5%a4%9a/" rel="noopener noreferrer" target="_blank">
     <strong class="markup--strong markup--li-strong">
      社交媒体破坏民主的证据越来越多
     </strong>
    </a>
    <strong class="markup--strong markup--li-strong">
     》
    </strong>
   </li>
  </ul>
  <p class="graf graf--p">
   这份报告强调了政府机构和政党利用社交媒体传播政治宣传、污染数字信息生态系统、以及压制言论自由和新闻自由的方式。
  </p>
  <p class="graf graf--p">
   尽管社交媒体的确提供了可以提高虚假信息的规模和范围的技术媒介，但重要的是要认识到，
   <strong class="markup--strong markup--p-strong">
    计算性宣传依赖的核心问题 —— 极化、不信任或衰落民主 —— 早在社交媒体甚至互联网本身出现之前就已经存在
   </strong>
   。
  </p>
  <p class="graf graf--p">
   所以，它依旧是人的问题，而非完全由媒介决定。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    不要错过《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%ae%a8%e8%ae%ba%e7%9a%84%e7%bb%88%e7%bb%93/" href="https://www.iyouport.org/%e8%ae%a8%e8%ae%ba%e7%9a%84%e7%bb%88%e7%bb%93/" rel="noopener noreferrer" target="_blank">
     讨论的终结
    </a>
    》这是一个很有效的解释案例
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    操纵者对社交媒体技术的共同选择
   </strong>
   应该引起世界民主政体的关注，但民主社会所面临的许多长期挑战也应引起人们更广泛的关注。
  </p>
  <p class="graf graf--p">
   计算性宣传已成为数字公共领域的 “正常组成部分”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    随着新技术（包括人工智能、虚拟现实或物联网）准备从根本上重塑社会和政治，这些技术还将继续发展
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e6%8a%80%e6%9c%af%e6%9c%89%e5%88%a9%e4%ba%8e%e6%9a%b4%e6%94%bf%ef%bc%9f%e2%80%8a-%e2%80%8a-%e4%b8%80%e5%9c%ba%e4%bd%a0%e5%bf%85%e9%a1%bb%e5%8f%82%e4%b8%8e/" href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e6%8a%80%e6%9c%af%e6%9c%89%e5%88%a9%e4%ba%8e%e6%9a%b4%e6%94%bf%ef%bc%9f%e2%80%8a-%e2%80%8a-%e4%b8%80%e5%9c%ba%e4%bd%a0%e5%bf%85%e9%a1%bb%e5%8f%82%e4%b8%8e/" rel="noopener noreferrer" target="_blank">
     为什么技术有利于暴政？ — — 一场你必须参与的政治斗争
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   但是，由于计算性宣传是长期挑战民主的征兆，因此，解决方案必须考虑到这些系统性的挑战，这一点很重要。
  </p>
  <p class="graf graf--p">
   但是，还必须考虑社交媒体平台在塑造当前信息环境中所扮演的角色。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e8%99%9a%e5%81%87%e4%bf%a1%e6%81%af%e4%bb%a4%e7%8b%ac%e8%a3%81%e6%94%bf%e6%9d%83%e5%8f%98%e5%be%97%e6%9b%b4%e5%bc%ba%e5%a4%a7%ef%bc%8c%e4%bb%a4%e6%b0%91%e4%b8%bb%e5%9b%bd-2/" href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e8%99%9a%e5%81%87%e4%bf%a1%e6%81%af%e4%bb%a4%e7%8b%ac%e8%a3%81%e6%94%bf%e6%9d%83%e5%8f%98%e5%be%97%e6%9b%b4%e5%bc%ba%e5%a4%a7%ef%bc%8c%e4%bb%a4%e6%b0%91%e4%b8%bb%e5%9b%bd-2/" rel="noopener noreferrer" target="_blank">
     为什么虚假信息令独裁政权变得更强大，令民主国家变得更脆弱？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   一个强大的民主国家需要获得高质量的信息，并要求公民有能力团结起来进行辩论、讨论、审议、同情并做出让步。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    社交媒体平台真的为公众思考和民主创造了空间吗？还是它们放大了使公民沉迷、虚伪和愤怒的内容？
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e7%9c%9f%e7%9a%84%e6%83%b3%e8%a6%81%e6%80%9d%e6%83%b3%e8%87%aa%e7%94%b1%e5%90%97%ef%bc%9f%e5%a6%82%e6%9e%9c%e6%98%af%ef%bc%8c%e5%8f%af%e4%bb%a5%e6%80%8e%e4%b9%88%e5%8a%9e%ef%bc%9f/" href="https://www.iyouport.org/%e4%bd%a0%e7%9c%9f%e7%9a%84%e6%83%b3%e8%a6%81%e6%80%9d%e6%83%b3%e8%87%aa%e7%94%b1%e5%90%97%ef%bc%9f%e5%a6%82%e6%9e%9c%e6%98%af%ef%bc%8c%e5%8f%af%e4%bb%a5%e6%80%8e%e4%b9%88%e5%8a%9e%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     你真的想要思想自由吗？如果是，可以怎么办？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   我们需要能回答上述问题。⚪️
  </p>
