   技术寡头 FATBAG（Facebook Amazon Tencent Baidu Alibaba Google）和政府正在建立一个强大的监视资本主义体系。越来越多了解到事实的人陷入了焦虑和痛苦。
  </p>
  <p class="graf graf--p">
   但 Vincent Mosco 看到了其他选择：公民动员起来抵抗、实施更强大的反垄断立法、采取欧洲的替代品以打破中国和美国的寡头产品垄断世界……还有哪些可以行动的方案？
  </p>
  <p class="graf graf--p">
   以下是 Progressive Post 对 Vincent Mosco 的访谈。

  <p class="graf graf--p">
   <em>
    Progressive Post：在“Becoming Digital”一书中，你分析了不同的技术的融合（物联网、云计算、大数据），以及“下一代互联网”的发展；它带来的权力关系和利益是什么？
   </em>
  </p>
  <p class="graf graf--p">
   Vincent Mosco：下一代互联网通过将强大的技术和系统结合在一起，大规模扩展了互联网本身。
  </p>
  <p class="graf graf--p">
   物联网/智能城市在日常物品中甚至在人群中嵌入微型传感器和监控设备。
  </p>
  <p class="graf graf--p">
   这会产生更多的数据，这些数据越来越多地存储在集中化的云或管理和处理它们的数据中心中。（解释《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E5%A4%A7%E5%9E%8B%E4%BA%91%E4%BE%9B%E5%BA%94%E5%95%86%E6%97%B6%E4%BB%A3%E7%9A%84%E5%AE%A1%E6%9F%A5%E5%88%B6%E5%BA%A6%EF%BC%9A%E9%9B%86%E4%B8%AD%E5%8C%96%E6%AF%81%E6%8E%89%E4%BA%92/" href="https://www.iyouport.org/%E5%A4%A7%E5%9E%8B%E4%BA%91%E4%BE%9B%E5%BA%94%E5%95%86%E6%97%B6%E4%BB%A3%E7%9A%84%E5%AE%A1%E6%9F%A5%E5%88%B6%E5%BA%A6%EF%BC%9A%E9%9B%86%E4%B8%AD%E5%8C%96%E6%AF%81%E6%8E%89%E4%BA%92/" rel="noopener noreferrer" target="_blank">
    大型“云”供应商时代的审查制度：集中化毁掉互联网自由
   </a>
   》）
  </p>
  <p class="graf graf--p">
   大数据分析使用私密数据来绘制对象群体和人员的行为特征，以进行预测并设置决策规则，算法在越来越多地管理着我们生活。（解释《

   <strong class="markup--strong markup--p-strong">
    《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%B8%BA%E4%BB%80%E4%B9%88%E5%BF%85%E9%9C%80%E6%8B%92%E7%BB%9D%E5%A4%A7%E6%95%B0%E6%8D%AE-%E7%9B%91%E8%A7%86%E4%B9%8B%E6%81%B6%EF%BC%88%E4%BA%8C%EF%BC%89%E5%85%AC%E7%A7%81%E7%9B%91/" href="https://www.iyouport.org/%E4%B8%BA%E4%BB%80%E4%B9%88%E5%BF%85%E9%9C%80%E6%8B%92%E7%BB%9D%E5%A4%A7%E6%95%B0%E6%8D%AE-%E7%9B%91%E8%A7%86%E4%B9%8B%E6%81%B6%EF%BC%88%E4%BA%8C%EF%BC%89%E5%85%AC%E7%A7%81%E7%9B%91/" rel="noopener noreferrer" target="_blank">

   </a>
   <strong class="markup--strong markup--p-strong">
    》
   </strong>
   《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E9%9D%9E%E7%89%A9%E8%B4%A8%E5%8A%B3%E5%8A%A8%E5%92%8C%E6%95%B0%E6%8D%AE%E6%94%B6%E9%9B%86%EF%BC%9Afacebook-%E7%9A%84%E7%AE%97%E6%B3%95%E5%B7%A5%E5%8E%82-%E7%AC%AC1%E9%83%A8/" href="https://www.iyouport.org/%E9%9D%9E%E7%89%A9%E8%B4%A8%E5%8A%B3%E5%8A%A8%E5%92%8C%E6%95%B0%E6%8D%AE%E6%94%B6%E9%9B%86%EF%BC%9Afacebook-%E7%9A%84%E7%AE%97%E6%B3%95%E5%B7%A5%E5%8E%82-%E7%AC%AC1%E9%83%A8/" rel="noopener noreferrer" target="_blank">

   </a>

   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" rel="noopener noreferrer" target="_blank">

     看不见的独裁
    </strong>

   <strong class="markup--strong markup--p-strong">

   </strong>
   ）
  </p>
  <p class="graf graf--p">
   FATBAG（Facebook Amazon Tencent Baidu Alibaba Google）领导几乎所有的产品和服务、收集关于人和物的一切数据。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    ⚠️各国政府，尤其是美国和中国的政府，都支持这些技术寡头的活动，因为他们受益于通过无休止的监视来控制公民的机会，扩大军事力量（想想武器化无人机），以及他们认为这些公司对国家的经济增长至关重要
   </strong>
   。
  </p>
  <p class="graf graf--p">
   科技寡头及其政府支持者共同组成了一个强大的监视资本主义体系 ——
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%86%9b%e4%ba%8b%e5%b7%a5%e4%b8%9a%e7%bb%bc%e5%90%88%e4%bd%93/" href="https://www.iyouport.org/category/%e5%86%9b%e4%ba%8b%e5%b7%a5%e4%b8%9a%e7%bb%bc%e5%90%88%e4%bd%93/" rel="noopener noreferrer" target="_blank">
    军事工业综合体
   </a>
   。
  </p>
  <p class="graf graf--p">
   <em>
    PP：鉴于科技寡头与国家之间的利益趋同，公民可以做些什么以改变这种新专制局面？
   </em>
  </p>
  <p class="graf graf--p">
   VM：公民需要了解通信行业的企业权力历史。
  </p>
  <p class="graf graf--p">
   在不同程度上，从电报到电话和广播的每一种新媒体技术都受到企业集中化、商业化和依赖政府军事情报部门的影响。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在每种情况下，公民都通过动员反抗取得了一定胜利
   </strong>
   ，包括推动垄断企业的解体、价格和服务的监管、商业系统公共替代方案的发展、以及对通信资源军事化的限制。
  </p>
  <p class="graf graf--p">
   新自由主义的兴起使许多公民倡议得以回归，其政策包括放松管制、私有化以及通过切实的反垄断执法缓解霸权的影响。
  </p>
  <p class="graf graf--p">
   正如公民曾经一次又一次地反抗那样，如今人们需要再次组织起来以反对当前的社交媒体制度，并严格按照公共利益来规范行业，
   <strong class="markup--strong markup--p-strong">
    尤其是，努力打造为公民可以抵制集中化霸权的公共替代品。
   </strong>
  </p>
  <p class="graf graf--p">
   与过去一样，公民还需要动员起来通过限制政府监视和大幅度减少使用人工智能系统对平民进行战争的恶行，来限制军事和情报机构的侵犯人权。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    PP：你提到支撑这些新技术的“文化神话”，让人们理解它们，并减轻它们带来的恐惧。数字技术往往等同于“创新”和“不可避免的进步之旅”，这种主流观点使其难以批评。
   </em>
  </p>
  <p class="graf graf--p">
   VM：每当我们制造技术时，都会产生某种神话。这些超越了虚假的含义，只是我们在告诉自己和彼此的故事，以帮助我们应对生活中的挑战。
  </p>
  <p class="graf graf--p">
   在大企业的支持下，数字技术被赋予了一系列超级神话，包括反抗、加速进步、和创造社会转型的力量 ……每一种期待都有助于形成关于历史终结、地理终结、政治的终结、以及更广泛的神话。
  </p>
  <p class="graf graf--p">
   然而，文化总是会产生对抗主流观点的反对意见。
  </p>
  <p class="graf graf--p">
   在数字化的情况下，我将确定三个这样的愿景，从民主开始，或者尽可能多的公众参与以影响生活的决策。
  </p>
  <p class="graf graf--p">
   还有一种社会正义，它激发了一种信念，即人们和整个自然世界都应尽可能充分地存在和繁荣。
  </p>
  <p class="graf graf--p">
   特别是，
   <strong class="markup--strong markup--p-strong">
    人们有权建立自我认同，以抵制监视资本主义，包括自我的管理和自我量化。
   </strong>
  </p>
  <p class="graf graf--p">
   最后，有一种普遍性的神话，它想象用数字来促进“平等”，并创造一个广泛共享的世界观，作为人类的共同责任。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    每一个都与主流神话并存，但每个都需要扩大支持来取代主流神话，即 统治文化机构的主导思想
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    PP：通信基础设施过去曾被设想为公用事业，今天也应如此。这在实践中意味着什么？
   </em>
  </p>
  <p class="graf graf--p">
   VM：首先宣布对社交生活至关重要的数字通信，并为所有人提供特定的最低服务级别和连接性。
  </p>
  <p class="graf graf--p">
   这可能涉及围绕主要商业社交媒体公司或代替主要商业社交媒体公司创建公共选择。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    法规将应该能保证大众普遍获得通信手段和公共信息，公民控制自己数据的权利，并将其转移到一个个人选择的系统，包括由公民控制的数据信托。
   </strong>
  </p>
  <p class="graf graf--p">
   公用事业公司将严格控制商业和政府监督。它还可以确保算法的构造和应用完全透明。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    PP：在欧洲，有人认为欧盟“错过了最好时机”，也有人说欧盟应该重新获得数字主权，并创建自己的数字巨头。无论这是否可能，
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     我们宁愿认为欧盟应该努力开发一种新模式，与美国的（风险资本驱动、垄断、军事工业综合体）和中国的（国家主导的监视和控制）模式相对照
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    —— 你有什么看法？
   </em>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4564 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/阳谋.png?resize=1100%2C1233&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/阳谋.png?w=1110&amp;ssl=1 1110w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/阳谋.png?resize=268%2C300&amp;ssl=1 268w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/阳谋.png?resize=768%2C861&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/阳谋.png?resize=914%2C1024&amp;ssl=1 914w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/阳谋.png?resize=1100%2C1233&amp;ssl=1 1100w" data-recalc-dims="1" height="1233" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/阳谋.png?resize=1100%2C1233&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4564" data-recalc-dims="1" height="1233" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/阳谋.png?resize=1100%2C1233&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/阳谋.png?w=1110&amp;ssl=1 1110w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/阳谋.png?resize=268%2C300&amp;ssl=1 268w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/阳谋.png?resize=768%2C861&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/阳谋.png?resize=914%2C1024&amp;ssl=1 914w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/阳谋.png?resize=1100%2C1233&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   （用欧洲制约中美的高堡奇人想法远非首次提议，这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E5%9C%A8%E4%B8%89%E5%B2%94%E5%8F%A3%EF%BC%9A%E4%B8%AD%E5%9B%BD%E3%80%81%E6%AC%A7%E6%B4%B2%E5%92%8C%E7%BE%8E%E5%9B%BD%E6%8F%90%E4%BE%9B%E7%9D%80%E5%AE%8C%E5%85%A8%E4%B8%8D%E5%90%8C%E7%9A%84/" href="https://www.iyouport.org/%E5%9C%A8%E4%B8%89%E5%B2%94%E5%8F%A3%EF%BC%9A%E4%B8%AD%E5%9B%BD%E3%80%81%E6%AC%A7%E6%B4%B2%E5%92%8C%E7%BE%8E%E5%9B%BD%E6%8F%90%E4%BE%9B%E7%9D%80%E5%AE%8C%E5%85%A8%E4%B8%8D%E5%90%8C%E7%9A%84/" rel="noopener noreferrer" target="_blank">
    在三岔口：中国、欧洲和美国提供着完全不同的“互联网未来”
   </a>
   》；并且已经开始实施，在这里《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E9%98%BF%E5%A7%86%E6%96%AF%E7%89%B9%E4%B8%B9%E9%A2%86%E5%AF%BC%E5%AF%B9%E7%9B%91%E8%A7%86%E8%B5%84%E6%9C%AC%E4%B8%BB%E4%B9%89%E7%9A%84%E6%96%97%E4%BA%89/" href="https://www.iyouport.org/%E9%98%BF%E5%A7%86%E6%96%AF%E7%89%B9%E4%B8%B9%E9%A2%86%E5%AF%BC%E5%AF%B9%E7%9B%91%E8%A7%86%E8%B5%84%E6%9C%AC%E4%B8%BB%E4%B9%89%E7%9A%84%E6%96%97%E4%BA%89/" rel="noopener noreferrer" target="_blank">
    阿姆斯特丹领导对监视资本主义的斗争
   </a>
   》）
  </p>
  <p class="graf graf--p">
   VM：我同意世界将受益于欧洲数字技术产业的发展，但这不仅仅是因为它将与来自美国和中国的公司（高堡奇人）进行经济竞争。
  </p>
  <p class="graf graf--p">
   美国科技巨头的经济成功是以牺牲全社会和政治为代价的，其核心价值观和制度正在崩溃，部分原因是这些巨头公司的权力已经肆无忌惮。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    创建欧洲版技术公司的主要原因是为欧盟公民和世界公民提供非美国和非中国模式的替代品
   </strong>
   。
  </p>
  <p class="graf graf--p">
   值得效仿的美国体系的一个积极价值是，该国历史上对高等教育和研究的坚定承诺。不可否认，其中一些是以军事为优先事项的结果（比如谷歌的来源，在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e7%9a%84%e7%9c%9f%e9%9d%a2%e7%9b%ae%e2%80%8a-%e2%80%8a-theyre-always-watching/" href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e7%9a%84%e7%9c%9f%e9%9d%a2%e7%9b%ae%e2%80%8a-%e2%80%8a-theyre-always-watching/" rel="noopener noreferrer" target="_blank">
    谷歌的真面目
   </a>
   》）。
  </p>
  <p class="graf graf--p">
   从教育和研究的重大扩展开始的替代模式也应该承诺公民充分参与规划和政策制定，例如巴塞罗那已经实施的那样。
  </p>
  <p class="graf graf--p">
   任何新系统都必须普遍可用，
   <strong class="markup--strong markup--p-strong">
    致力于保护隐私，并设计为阻止监视资本主义的有力工具。
   </strong>
  </p>
  <p class="graf graf--p">
   它应该使用开源软件，让公民有权控制自己的数据并将其移动到他们自己选择的网络。
  </p>
  <p class="graf graf--p">
   公民控制，这点很重要，科技巨头经常在此玩花样，比如这里《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a3%b0%e7%a7%b0%e7%bb%99%e4%bd%a0%e9%9a%90%e7%a7%81%e4%bd%86%e4%b8%8d%e7%bb%99%e4%bd%a0%e8%87%aa%e7%94%b1%ef%bc%9a%e7%9c%8b-apple-%e5%a6%82%e4%bd%95%e6%8a%8a%e7%8e%a9%e6%96%87/" href="https://www.iyouport.org/%e5%a3%b0%e7%a7%b0%e7%bb%99%e4%bd%a0%e9%9a%90%e7%a7%81%e4%bd%86%e4%b8%8d%e7%bb%99%e4%bd%a0%e8%87%aa%e7%94%b1%ef%bc%9a%e7%9c%8b-apple-%e5%a6%82%e4%bd%95%e6%8a%8a%e7%8e%a9%e6%96%87/" rel="noopener noreferrer" target="_blank">
    声称“给你隐私”但不给你自由：看 Apple 如何把玩文字游戏
   </a>
   》
  </p>
  <p class="graf graf--p">
   事实上，我认为考虑这样一种公共事业模型是明智的，它将信息视作水和能源一般地重视：作为一种必不可少的资源。
  </p>
  <p class="graf graf--p">
   这些出发点需要强有力的国家干预，包括对欧盟和非欧盟公司的监管。
  </p>
  <p class="graf graf--p">
   欧盟通用数据保护条例（GDPR）标志着朝着正确方向迈出了重要一步，但是，要应对美国和中国的压力，还有很多工作要做。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4563 jetpack-lazy-image" data-lazy-sizes="(max-width: 388px) 100vw, 388px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/书-1.png?resize=388%2C600&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/书-1.png?w=388&amp;ssl=1 388w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/书-1.png?resize=194%2C300&amp;ssl=1 194w" data-recalc-dims="1" height="600" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/书-1.png?resize=388%2C600&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="388"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4563" data-recalc-dims="1" height="600" sizes="(max-width: 388px) 100vw, 388px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/书-1.png?resize=388%2C600&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/书-1.png?w=388&amp;ssl=1 388w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/书-1.png?resize=194%2C300&amp;ssl=1 194w" width="388"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    注
   </strong>
   ：我们修改了一个关键字，原文中是鼓励欧洲“自己的bigtech”，我们相信作者不是那个意思，不是说要把高堡奇人变成三国演义 —— 欧洲必须能发挥去中心化的、数据民主化的治理模式，才能真正战胜高堡奇人。
  </p>
  <p class="graf graf--p">
   顺便推荐 Vincent Mosco 这本书，上图。这本书中指出的问题并非新鲜，正相反应该是目前被大众所熟知的，此前已经有多位研究人士详细阐述过，比如《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e7%a9%b6%e7%ab%9f%e6%98%af%e4%bb%80%e4%b9%88%ef%bc%9f/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e7%a9%b6%e7%ab%9f%e6%98%af%e4%bb%80%e4%b9%88%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    互联网究竟是什么？
   </a>
   》、《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e6%8e%a7%e5%88%b6%e7%9a%84%e4%b8%96%e7%95%8c%e5%92%8c%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89/" href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e6%8e%a7%e5%88%b6%e7%9a%84%e4%b8%96%e7%95%8c%e5%92%8c%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89/" rel="noopener noreferrer" target="_blank">
    谷歌控制的世界和监视资本主义
   </a>
   》、《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%B8%8E%E5%85%B6%E8%AF%B4%E6%98%AF%E4%BA%92%E8%81%94%E7%BD%91%E7%9A%84%E6%9C%AB%E6%97%A5%EF%BC%8C%E4%B8%8D%E5%A6%82%E8%AF%B4%E6%98%AF%E4%B8%96%E7%95%8C%E6%9C%AB%E6%97%A5%E6%9B%B4%E5%AE%B9%E6%98%93/" href="https://www.iyouport.org/%E4%B8%8E%E5%85%B6%E8%AF%B4%E6%98%AF%E4%BA%92%E8%81%94%E7%BD%91%E7%9A%84%E6%9C%AB%E6%97%A5%EF%BC%8C%E4%B8%8D%E5%A6%82%E8%AF%B4%E6%98%AF%E4%B8%96%E7%95%8C%E6%9C%AB%E6%97%A5%E6%9B%B4%E5%AE%B9%E6%98%93/" rel="noopener noreferrer" target="_blank">
    新黑暗时代
   </a>
   》……
  </p>
  <p class="graf graf--p">
   但这本书不同在于，它想到了一个解决方案，而且理论上可行；虽然实际上并不容易。当下的问题在于，监视资本主义是庞大的产业、背后是巨额的难以想象的经济利润，而反监视是白衣事业，完全依靠道德感支撑；加之，欧盟的反垄断立场有点令人困惑（在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ae%bd%e5%88%ba%e6%80%a7%e4%be%9d%e8%b5%96/" href="https://www.iyouport.org/%e8%ae%bd%e5%88%ba%e6%80%a7%e4%be%9d%e8%b5%96/" rel="noopener noreferrer" target="_blank">
    讽刺性依赖
   </a>
   》）；于是，也正因此，这本书所倡导的目标应该具有重要的价值，即：公民要首先联合起来抵抗，而不是坐等权势拟定法案。⚪️
  </p>
