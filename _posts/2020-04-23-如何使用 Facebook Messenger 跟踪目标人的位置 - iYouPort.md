  <p class="graf graf--p">
   跟踪和定位个人的能力不再像间谍电影中那样困难了。智能手机之类的东西可以完成间谍工具包中的大部分能力。
  </p>
  <p class="graf graf--p">
   在数字时代，仅使用聊天应用程序就可以轻松确定某人的位置。只要您和“目标”都使用 Facebook Messenger，就可以进行位置跟踪。
  </p>
  <p class="graf graf--p">
   现在，假设您处于这种情况的另一端 —— 即 有人可能正在追踪您的位置，而您却一无所知。令人毛骨悚然，对吧？
  </p>
  <p class="graf graf--p">
   实际上，Facebook Messenger应用程序带来的风险比您想象的要大得多。
  </p>
  <p class="graf graf--p">
   对于拥有数百个 “朋友” 的人来说尤其如此，因为并非所有这些人都是您可能真正见过的人。
  </p>
  <ul class="postList">
   <li class="graf graf--li">

  <p class="graf graf--p">
   依旧有太多的人会在社交媒体上投入过多的精力，泄露的信息超出了你的想象。⚠️了解如何跟踪他人可以使您了解某人对您进行相同操作的难易程度 —— 以提高自我保护意识。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     使用 Facebook Messenger 跟踪位置
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   您甚至不需要精通技术就可以使用 Messenger 跟踪某人。精英级别的黑客无需进入 Facebook 即可看到谁已经分享了他们的位置。
  </p>
  <p class="graf graf--p">
   不需要技术就可以做到跟踪的方法有如下几个。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     1、位置分享
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   此特定功能是一种基于共识的跟踪方法。 Facebook Messenger 通过允许朋友分享他们的位置，使想要见面的人更容易联系。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    点击屏幕左下角的四个点，通过 Facebook Messenger 打开聊天对话框。
   </li>
   <li class="graf graf--li">
    点击位置
   </li>
   <li class="graf graf--li">
    从那里，只需选择“共享实时位置”，您的朋友就会看到您的当前位置。他们反过来可以与您分享他们的位置。
   </li>
  </ul>
  <p class="graf graf--p">
   ⚠️大多数人都有互惠心理，如果你先给予对方好处，对方就会感觉过意不去，更容易为你做点什么。如果你能成功设置这种心理情景，就能诱导对方分享他们的位置数据。
  </p>
  <p class="graf graf--p">
   诱导术有很多，远不止互惠心理这一种，稍不留神就容易上当。如果您希望保护自己，建议您参见我们的 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
    社交工程
   </a>
   ” 专栏。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     2、位置标签
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   在您的朋友列表中搜索目标人的位置标签，只需点击几下就可以找到他们的位置。
  </p>
  <p class="graf graf--p">
   查找位置标签的最常见方法是检查 Facebook Stories。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    如果其中一张图像上保留了位置标签，则只需点击它，选择显示位置，然后将其拉到智能手机屏幕上即可。
   </li>
   <li class="graf graf--li">
    点击标签文本，您将被重定向到Google地图，该地图将显示目标的位置。
   </li>
  </ul>
  <p class="graf graf--p">
   这不是最可靠的发现方法，因为位置不一定是最新的，而是图像最初发布的时间。还存在位置标签提供不正确位置的问题，因为发布者可以张贴他们选择的任何位置。但这仍然值得一试，因为有些人真的很傻，他们会去分享实际的位置。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     3、第三方APP
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   ⚠️这部分可就不是共识了，更多是侵犯性的，是钓鱼行为。
  </p>
  <p class="graf graf--p">
   尽管位置跟​​踪的某些应用可能会以积极的方式带来好处，但大部分情况下并非如此，这种方法在整个过程中都贴有“网络犯罪”的标签。
  </p>
  <p class="graf graf--p">
   使用免费的在线服务（如
   <a class="markup--anchor markup--p-anchor" data-href="https://iplogger.org/" href="https://iplogger.org/" rel="noopener noreferrer" target="_blank">
    IPLogger
   </a>
   ）可以帮助您查找目标人的确切位置。
  </p>
  <p class="graf graf--p">
   您所要做的就是访问
   <a class="markup--anchor markup--p-anchor" data-href="https://iplogger.org/" href="https://iplogger.org/" rel="noopener noreferrer" target="_blank">
    该网站
   </a>
   ，然后从列表中选择一个跟踪选项。
  </p>
  <p class="graf graf--p">
   输入您想要用来跟踪目标位置的URL，然后复制提供的 IPLogger 代码。
  </p>
  <p class="graf graf--p">
   从这里开始，就是想办法让您的目标人点击生成的 URL，您可以通过 Facebook Messenger 发送该URL。
  </p>
  <p class="graf graf--p">
   一旦对方点击了，你可以使用第二个生成的URL，该URL使您可以直接从浏览器查看位置统计信息。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     4、“附近朋友”功能
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   此功能要求您已安装实际的 Facebook 应用程序。然后，您可以通过进入 “更多选项” 菜单并选择 “附近的朋友” 来启用它。然后只需点按该选项即可将其打开。
  </p>
  <p class="graf graf--p">
   现在，您将可以看到当前正在使用该功能的所有其他人。每个点都会按位置列出，并显示它们之间的物理距离。
  </p>
  <p class="graf graf--p">
   最亲密的 “朋友” 将首先显示，并带有一个 Facebook Messenger 按钮，您可以用来与他们聊天。
  </p>
  <p class="graf graf--p">
   ⚠️一旦使用了该功能，即表示您同意也使用该功能的任何 “朋友” 列表中的人了解您的位置 —— 所以它非常危险。
  </p>
  <p class="graf graf--p">
   此特殊功能主要用于朋友和同事之间的现实聚会，但是，也非常可能被错误地放入朋友列表中的社交工程攻击者所使用。
  </p>
  <p class="graf graf--p">
   所以，作为防御，您最好不要使用这个东西。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     最后
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   ⚠️如您所知，只要您想查找任何人，只需要把他们放入您的朋友列表就会变得非常容易。
  </p>
  <p class="graf graf--p">
   所以，再一次提醒：千万要小心网上的任何 “朋友”，尤其是对于敏感人士来说，因为瞄准你的坏家伙比瞄准其他人的更多。
  </p>
  <p class="graf graf--p">
   与他人共享任何内容时都需要格外谨慎，尤其是在网上共享，
   <strong class="markup--strong markup--p-strong">
    多想想对方能用这些信息来做什么，而不是 “它对我来说是不是重要” —— 站在攻击者的角度上思考
   </strong>
   。
  </p>
