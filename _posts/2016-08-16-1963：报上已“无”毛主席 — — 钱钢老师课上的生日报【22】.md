<strong class="id jd">
        钱钢老师附言
       </strong>
</p>
<p class="ib ic jc id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="7532">
       语象分析显示，上世纪60年代前半期，刘少奇在党媒的传播强度曾连续3年超过毛泽东。这个传播现象折射了什么样的政治角力？它与不久后发生的文革又有何关联？邹思聪的生日报演示，提出了大可探究的课题。
      </p>
</blockquote>
<h1 class="je jf fq cd dz jg jh ji jj jk jl jm jn jo jp jq jr js jt ju jv ec" id="1f34">
<strong class="ay">
       1
      </strong>
</h1>
<p class="ib ic fq id b ie jw ig ih ii jx ik il im jy io ip iq jz is it iu ka iw ix iy fi ec" id="7e74">
      我的父亲出生于1963年4月28日。20世纪中国，4月28日曾多次成为历史转折点。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="c521">
      1906年4月28日，新闻纸第一次分属革命派和改良派，激烈论战开始； 1928年4月28日，革命党人的共产主义支流败退井冈山，党史则称之为“朱毛会师”；到1956年4月28日，已是中国最高领袖的毛泽东不顾党内不满，号召“双百方针”，之后，五十五万知识分子坠入深渊。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="2caa">
      我父亲出生在四川万县专区（今属重庆）。1963年的这一天，是否有转折性的事情发生呢？我找到了那天的《四川日报》和《人民日报》。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="4119">
      答案是，没有。这一天是星期日，《四川日报》只有两版，一个头版一个副刊。《人民日报》也只有四版。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="509f">
      四川日报这一天的头版是关于“葡萄糖”和“饭锅”的两个经济新闻，这并不符合我预想中会出现的社会主义先进性的东西，比如人民公社，钢铁产量，劳模光荣……出现这些竞争性、激励性、有商品意味的经济新闻，不是明显的政治不正确吗？
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="69d5">
      而让人疑窦更深的，则是这一条，“刘主席将访问柬埔寨越南”。在我的预设中，即便没有那些彰显社会主义优越性的新闻，也至少得有毛主席的头版头条才对，而毛主席却在报纸上消失了。
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="dfc8">
      翻到副刊这一版，我发现一个引人注目的雕塑，他的名字叫“雷锋”。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="94df">
      经济新闻，刘主席，雷锋。冥冥之间，他们会有更深层次的关系吗？
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="f250">
      我开始继续检索。这一次是最权威的，显示最高层政策方针动向的《人民日报》。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="08e4">
      从4月1日到4月28日，人民日报上占据头版头条的大部分时候是刘主席，而毛主席则时有时无。我再继续检索了当年其他月份的人民日报，结果大致一样。
     </p>

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="b1a6">
      对1963年进行统计发现，人民日报标题上“刘主席”出现了379次，而“毛主席”出现了176次，刘是毛的两倍有余。如果仅以上半年数据，则毛仅出现58次，刘却是毛的4倍多！
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="cc8e">
      是毛主席过气了吗？中学历史曾告诉我们，毛主席一直是政坛主角。教科书上的表述，显然无法解释这段历史的断裂。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="b466">
      “如果一件事情无法解释，那么一定有你所不知道的事实存在。”
     </p>
<h1 class="je jf fq cd dz jg jh ji jj jk jl jm jn jo jp jq jr js jt ju jv ec" id="3351">
<strong class="ay">
       2
      </strong>
</h1>
<p class="ib ic fq id b ie jw ig ih ii jx ik il im jy io ip iq jz is it iu ka iw ix iy fi ec" id="32c0">
      在特里尔的《毛泽东传》中，有这样一个章节《退却（1961–1964）》，第一段写道，“迫于政治和经济的现实，自1961年起，毛泽东带着满腹狐疑开始让步退却了。刘少奇、邓小平二人强调政治秩序和经济效益符合时代趋势。”
     </p>
<figure class="kc kd ke kf kg kh fc fd paragraph-image">

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="1b4c">
      时间回到1959年7月的庐山会议。大跃进第一季暂时结束，由于第一季“收视率的惨淡”，这次大会很可能终止大跃进。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="89cb">
      但由于彭德怀的一份只给毛主席看的万言书，被毛泽东印了出来分发全部与会干部批判，纠左变为了反右倾。彭德怀，张闻天，黄克诚，周小舟的“彭张黄周军事俱乐部”被打倒。大跃进第二季随即播出。
     </p>
<figure class="kc kd ke kf kg kh fc fd paragraph-image">

      </figcaption>
</figure>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="e1a5">
      大跃进导致了整个世界20世纪最大的饥荒，2000万以上人口死亡。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="a5e1">
      由于巨大的错误，毛泽东开始“过气”了。他的“过气”分步骤进行，焦灼而反复。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="8302">
      第一步开始了。1961年1月，毛泽东在北京举行的中共八届九中全会上发言，要求全党大兴调查研究之风，1961年要搞个“实事求是年”，用现在的流行语讲，这叫“请自行打脸”。会议正式批准对国民经济实行“调整、巩固、充实、提高”的八字方针。
     </p>
<figure class="kc kd ke kf kg kh fc fd paragraph-image">

<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="b38b">
      “人民公社实际上已解体为大队，农村的黑市开始半合法化。工厂中厂长负责制又复活了。偏离党的工作方法的倾向已经出现。1957年被作为毒草连根拔掉的知识分子又在刘邓的权势庭院中找到了新的土壤。”特里尔写道。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="96ce">
      步步为营，第二步在1961年初夏到来。当年5 、6月，中共中央工作会议在北京召开。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="51bc">
      在这次会上，基于实事求是的主张，中共通过了《农业六十条》、《商业四十条》、《手工业三十五条》等等，比如《农业六十条》规定，以生产队为核算单位，提供自留地。按现代经济学来讲，私人产权得到了一定程度的恢复。
     </p>

      </figcaption>
</figure>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="a328">
      6月12日，会议的最后一天，毛泽东做了总结讲话，在讲话中他做了自我批评，并且认为1959年不该把反右倾斗争搞到群众中去，提出要对庐山会议后几年来批判和处分错了的人都要甄别平反，重新教育干部。并且，不许再开展反右倾或者左倾斗争，禁止给他们戴政治帽子。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="290d">
      不止于此。中共中央工作会议结束之后，薄一波又组织制定了《工业七十条》等内容，强调了“专家的作用”和“物质刺激的作用”。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="1bfe">
      党内高层并没公开反对毛泽东，但这一次通过的所有条例政策，都在实际上反对毛主席的“三面红旗” — — 总路线，大跃进和人民公社。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="c3f3">
      高潮在1962年1月到来。第三步，七千人大会。会上，刘少奇做了主要报告，他把主要责任归到中央，强调防止粗暴的清洗和反清洗的重要性。他指出，三年饥荒的问题，70%是人祸，30%是天灾。这与毛泽东竖起的三个手指恰好相反。
     </p>

      </figcaption>
</figure>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="ef1d">
      这次的会议，“白天出气，晚上看戏，两干一稀，大家满意”。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="3cb3">
      紧接着，刘少奇主持召开高层内部的政治局扩大会议。这场“西楼会议” ，是毛“过气”的第四步。可毛泽东则自称徐霞客，他仿效这位中国最著名的驴友，一走了之。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="c6b7">
      刘在会上认为国民经济要进行大幅度调整，而因为处于非常时期，所以要退，要“退够”。但在毛泽东看来，困难时期已经挺过去了，这是“危言耸听”，把形势看的“一片黑暗”，刘少奇是“揪住不放”。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="89cc">
      我们看到，毛主席是以一种非常奇怪的方式“过气”的，他一边被党内高层承认和夸奖，一边他的主张政策却在实际上被边缘化。刘少奇，如同在报纸上观察到的那样，自1959年成为这个国家主席以来，已逐渐实际地掌握着这个国家政治秩序和经济发展的轮盘。这个国家最官方的报纸和地方的党媒，都开始把他捧为最如日中天的红人。他的团队所主张的路线，也开始在版面上大量展示。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="7443">
      经济开始恢复，政治趋于稳定。而毛主席，则在众官僚的阳奉阴违中“过气”了。毛泽东的乌托邦之怒，仿佛打在了一团深谙太极的海绵里，打得越重，陷得越深。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="359e">
      但是我父亲生日这天的报纸，隐隐地已经在告诉我们，比起毛泽东，这些实用主义的高层，实在太低估了一种精神的力量，他们以退为进，放之生长。而在不远的将来，他们都将被这股力量反噬。
     </p>

      在达到1963年的高峰后，两位领袖的名字频率在1964年出现反转，这个趋势，一直持续到进入文革，毛泽东传播强度冲天，刘少奇被打倒。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="58da">
      这转变在1962年就已开始积蓄。8月的北戴河会议，在一片“包产到户”声中，毛泽东愤怒发问“社会主义究竟存不存在阶级？”他甚至直言，“包产到户就是搞资本主义！”9月份的八届十中全会，彭德怀的再次上书申冤被毛斥之为“翻案风”，而习仲勋由于据说曾支持小说《刘志丹》，被毛认为是用“小说反党”，和彭德怀和早已开除党籍的高岗奇妙地组成了“彭习高反党集团”。
     </p>
<p class="ib ic fq id b ie if ig ih ii ij ik il im in io ip iq ir is it iu iv iw ix iy fi ec" id="1a90">
      1963年，毛泽东的“过气”只是暂时的隐退。这年，他已经找到了四个最懂他的盟友。其中三个人是江青，康生，林彪。还有一个牺牲的战士，他出现在我父亲生日这天的报纸上，雷锋。他成了“毛主席的好学生”，生前故事开始出现在人民日报，以及国人能接触的各类刊物、声音和图画里，演变成传奇。“学习雷锋好榜样”成为口号。
