   </strong>

    在你的日常生活中应用隐蔽野战技术
   </a>

  <p class="graf graf--p">
   ⚠️在技术高端到不可思议的世界里，人力监视依旧有很多优势：它灵活、直接、易于调度、稳定、杂音少……尤其是隐私保护措施已经越来越普及的情况下，人力监视尤其可以针对那些注意力完全锁定在技术领域的人。
  </p>
  <p class="graf graf--p">
   所以
   <strong class="markup--strong markup--p-strong">
    我们建议：各种手段您都应该了解
   </strong>
   。它们之间没有主次之分。
  </p>
  <p class="graf graf--p">
   在我们的国家，私家侦探和赏金猎人的行业很火爆，甚至有些时候警方都会与这些行业内的高手合作，以挖出一个藏匿很深的目标人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这些行业的人是拿钱办事的，所以您不用指望他们遵守什么道德
   </strong>
   。这点上我们对中国的情况了解很少；但是，想想看，在个人隐私信息具有昂贵价值的时代里，又有什么理由不做生意呢？
  </p>
  <p class="graf graf--p">
   私家侦探行业很疯狂，很多时候比警察更厉害；他们是靠疯狂赚钱的，⚠️他们并没有警察掌握的那些大型数据库，但他们具备
   <strong class="markup--strong markup--p-strong">
    高超的以骗制骗能力，有些时候比数据库都好用
   </strong>
   。
  </p>
  <p class="graf graf--p">
   是的，社交工程。
  </p>
  <p class="graf graf--p">
   当然，有数据库就更好了；再有点黑客技术就是好上加好。但即便毫无技术，也一样可以做到 —— 把藏匿的人挖出来。
  </p>
  <p class="graf graf--p">
   其中的关键在于：藏匿的人还不够聪明。

   现在进入正题：
   <strong class="markup--strong markup--p-strong">
    关于追踪的几个原则
   </strong>
   —— 追踪者熟知并秉持的东西，反追踪需要了解的基础。
  </p>
  <p class="graf graf--p">
   👉1、你的个人信息是一种颇具价值的商品。不论你是谁，总有人对它感兴趣，总有人想要得到它。
  </p>
  <p class="graf graf--p">
   很多对个人权利表示“无所谓”的人，基本都是因为觉得自己“没有权利”。⚠️这不是事实。这只是想要控制你的人灌输给你的概念。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    因为只有你放弃了个人权利，那些想要控制你的人才能成功
   </strong>
   。
  </p>
  <p class="graf graf--p">
   个人信息是一种至高无上的权利。也许你自己对自己的个人信息太熟悉不过了，于是觉得“它没什么稀奇的”；但是，对任何其他人来说都完全不同。这也许是很多人误解个人信息之价值的原因。
  </p>
  <p class="graf graf--p">
   不论你是罪犯还是守法公民，在此都一样。你肯定还不知道自己在追踪者眼中有多么的价值连城。
  </p>
  <p class="graf graf--p">
   ⚠️不论是想要找到你的真身、还是想要盗用你的身份，那个人绝对会不惜一切代价做到这点。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如何才能让一个人感知到个人信息有多值钱？方法很简单，那就是，让TA尝试一下私人侦探的业务。
   </strong>
  </p>
  <p class="graf graf--p">
   还记得我们在OSINT中多次强调过的吗：它既可以进攻，也可以防守。同样道理。
  </p>
  <p class="graf graf--p">
   OSINT市场已经高达数十亿美元，这可是
   <strong class="markup--strong markup--p-strong">
    仅仅
   </strong>
   基于开源的情报市场；私人侦探就更不用说了，他们游走在黑白之间，他们可以采取任何可用的手段，只要没被抓到，钱到手就是合法的。
  </p>
  <p class="graf graf--p">
   这一行业内的成功者都是不会按理出牌的人；他们经常可以自如地
   <strong class="markup--strong markup--p-strong">
    跳出常人的思考框架
   </strong>
   做出那种能让你意想不到的事。
  </p>
  <p class="graf graf--p">
   于是这就引出第二条原则：
  </p>
  <p class="graf graf--p">
   👉2、信息行业和其他行业一样。成功者懂得找到市场的空白点，而且懂得如何填补这些空白。追踪者之间的常态竞争能让强者更强。
  </p>
  <p class="graf graf--p">
   就如我们上面所说到的，这一行业拿钱办事，谁出的价码高，就接下谁的活。（这很容易理解，混过 Hack Forums 或逛过零日市场的人都知道）
  </p>
  <p class="graf graf--p">
   ⚠️如果被追踪的人是你，对手出价越高，你就离死神越近。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    永远不要停止惊讶于社交工程师的创造力，他们就是靠创造力吃饭的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   👉3、如果确实有关于你的任何精准信息，那么，只要他们有足够的时间和精力，追踪者肯定会能把你挖出来。
  </p>
  <p class="graf graf--p">
   随着技术的进步，追踪行业的钱只会越来越好赚，越来越简单和高效。
  </p>
  <p class="graf graf--p">
   他们
   <strong class="markup--strong markup--p-strong">
    不需要
   </strong>
   具备骇客能力，甚至不需要懂编程，因为最大的漏洞永远是人；社交工程之所以被成为
   <strong class="markup--strong markup--p-strong">
    人性骇客
   </strong>

  </p>
  <p class="graf graf--p">
   只要有足够的胆量和创造力，你就能拿到你想要的东西。

  </p>
  <p class="graf graf--p">
   一位多金的大客户打了一个电话给私人侦探公司的社交工程师，给了后者一个女人的名字，他们想要知道这个女人在哪。
  </p>
  <p class="graf graf--p">
   只有一个名字。
  </p>
  <p class="graf graf--p">
   这些社交工程师就开始使用常规钓鱼方法打电话给各种公司，一概声称自己是那位女士，询问对方的服务并声称自己搬家了、注册地址已变 …… 不久之后就找到了那个女人可能的藏身地点。
  </p>
  <p class="graf graf--p">
   接下来就是给这个可能藏身的地方打电话。接线的人是她家保姆。以下是对话：
  </p>
  <p class="graf graf--p">
   社交工程师：您好，我是 UPS 快递公司的***，我们有她（那个女人的名字，故意读错她的名字）的包裹，但是包裹
   <strong class="markup--strong markup--p-strong">
    进水了
   </strong>
   。我们非常抱歉！
  </p>
  <p class="graf graf--p">
   保姆：哦，那么…
  </p>
  <p class="graf graf--p">
   社交工程师：我们需要她
   <strong class="markup--strong markup--p-strong">
    签名
   </strong>
   才能处理，不知道她什么时候会回来？
  </p>
  <p class="graf graf--p">
   保姆：啊……
  </p>
  <p class="graf graf--p">
   社交工程师：那好吧，我们把包裹
   <strong class="markup--strong markup--p-strong">
    退回去
   </strong>
   就行了。
  </p>
  <p class="graf graf--p">
   保姆沉默了一会。
  </p>
  <p class="graf graf--p">
   社交工程师：我们可以把包裹
   <strong class="markup--strong markup--p-strong">
    退回去
   </strong>
   吗？
  </p>
  <p class="graf graf--p">
   保姆：不……不要，你直接送过来吧。
  </p>
  <p class="graf graf--p">
   社交工程师：好的，那么我们什么时间可以找到她
   <strong class="markup--strong markup--p-strong">
    签名
   </strong>
   ？
  </p>
  <p class="graf graf--p">
   保姆：她六点左右会回来。
  </p>
  <p class="graf graf--p">
   搞定。
  </p>
  <p class="graf graf--p">
   以上对话部分中深颜色的字意味着需要重音读出 —— 这是心理操纵的逻辑。
  </p>
  <p class="graf graf--p">
   *目标人将于某某时间到达某某地点* —— 这条信息交给那个多金的大客户，钱就到手了。
  </p>
  <p class="graf graf--p">
   那么，那个女人究竟是谁呢？
  </p>
  <p class="graf graf--p">
   如今几乎所有人都知道她，她就是莫妮卡莱温斯基（Monica Lewinsky）。
  </p>
  <p class="graf graf--p">
   你一定还记得我们正“战略性欺骗”中指出的：
   <strong class="markup--strong markup--p-strong">
    只要进行了欺骗，就存在被发现的可能。区别只是在于，发现的时间，是及时、还是为时已晚
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8651 jetpack-lazy-image" data-lazy-sizes="(max-width: 914px) 100vw, 914px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图-1.png?resize=914%2C1298&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图-1.png?w=914&amp;ssl=1 914w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图-1.png?resize=211%2C300&amp;ssl=1 211w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图-1.png?resize=768%2C1091&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图-1.png?resize=721%2C1024&amp;ssl=1 721w" data-recalc-dims="1" height="1298" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图-1.png?resize=914%2C1298&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="914"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8651" data-recalc-dims="1" height="1298" sizes="(max-width: 914px) 100vw, 914px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图-1.png?resize=914%2C1298&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图-1.png?w=914&amp;ssl=1 914w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图-1.png?resize=211%2C300&amp;ssl=1 211w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图-1.png?resize=768%2C1091&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/图-1.png?resize=721%2C1024&amp;ssl=1 721w" width="914"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   社交工程欺骗也一样，它很容易被揭穿，而且很容易漏出马脚 —— 不同的只有，是及时、还是为时已晚。⚠️只要你能争取到时间，就是绝对的胜利。
  </p>
  <p class="graf graf--p">
   Evans 介绍过一部
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/DonEvansWm/status/1157544973316739072" href="https://twitter.com/DonEvansWm/status/1157544973316739072" rel="noopener noreferrer" target="_blank">
    西班牙剧目 La Casa de Papel
   </a>
   ，我也想推荐一下，因为这部剧中涉及的“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
    社交工程
   </a>
   ”和“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e6%88%98%e5%92%8c%e4%bf%a1%e6%81%af%e6%88%98/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e6%88%98%e5%92%8c%e4%bf%a1%e6%81%af%e6%88%98/" rel="noopener noreferrer" target="_blank">
    心理战
   </a>
   ”策略，真的
   <strong class="markup--strong markup--p-strong">
    非常标准
   </strong>
   ；尤其是，你能看到，一群主演是如何与对手抢夺时间的。
  </p>
  <p class="graf graf--p">
   很荣幸，这些战术IYP都介绍过。
  </p>
  <p class="graf graf--p">
   ⚠️时间和时机是在社交工程战术中价值连城的东西。
  </p>
  <p class="graf graf--p">
   👉6、也是最后一个原则：追踪者在操作中可以犯几十个错误；但你只要犯一个错误，你就 over 了。
  </p>
  <p class="graf graf--p">
   职业社交工程可以充满娱乐性，我们
   <strong class="markup--strong markup--p-strong">
    绝对不是
   </strong>
   鼓励您加入这一职业；而是，想要通过对这一职业的解剖，帮助您避免被追踪。
  </p>
  <p class="graf graf--p">
   请务必牢记上述6个原则。这里是开篇。
  </p>
  <p class="graf graf--p">
   也就是说，如果你正在着手准备一套*退出*策略 —— 保护性策略，
   <strong class="markup--strong markup--p-strong">
    那么必须以这6大原则为前提
   </strong>
   。
  </p>
  <p class="graf graf--p">
   本系列内容就是希望帮助您制定这样的策略，您可以在任何必要的时候“重返”现实；在不必要的时候销声匿迹。⚪️
  </p>
  <p class="graf graf--p">

    —— 未完待续 ——
