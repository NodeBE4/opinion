  <p class="graf graf--p">
   Bloomberg Quint 的一篇新专栏文章指出，在中美海军于南中国海一年的紧张局势之后，美国应该真正担心北京的意图。
  </p>
  <p class="graf graf--p">
   这种“更难辨别”的潜在威胁涉及世界上大约380条水下电缆，它们承载着全球各国95％以上的数据和语音流量。中国可以悄悄地攻击这些电缆吗？如果他们想这么做，现在肯定是处于更有利的地位。
  </p>
  <p class="graf graf--p">
   中国企业集团华为技术公司已经处于更广泛的美中争议的中心，这篇文章详细介绍了涉及盗窃商业机密以及与伊朗等受制裁政权的非法交易的华为，要在全球范围内“建造或改进”近100条海底电缆。
  </p>
  <p class="graf graf--p">
   西方情报官员现在正在敲响警钟，关于北京增加访问权限以及对这种关键的全球通信基础设施的影响。顺便提一下谷歌等巨头正在将海底电缆私有化，具体详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/newsletters-08062019-453c6692952c" href="https://www.iyouport.org/%e4%b9%b0%e6%96%ad%e4%ba%92%e8%81%94%e7%bd%91/" rel="noopener noreferrer" target="_blank">
    买断互联网
   </a>
   》。
  </p>
  <p class="graf graf--p">
   报告总结了以下“西方情报专业人员”的观点：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    由于专家们有理由担心在华为的5G技术中加入间谍“后门”，西方情报专业人员反对该公司参与海底电缆改造，该项目会提供了更大的收益，因为如此多的数据依赖于如此少的电缆
   </em>
   。
  </p>
  <p class="graf graf--p">
   该报告进一步指出，去年华为海运网络与国有电信运营商的联合项目成功铺设了从巴西到喀麦隆的近4000英里的电缆，并计划更多地参与全球互联网电缆项目。这将需要合作，并在某些情况下与谷歌和 Facebook 等美国互联网巨头竞争，后者正在从构建它们的公司租赁或购买大量的海底电缆，无论是私有的还是国有的。
  </p>
  <p class="graf graf--p">
   华尔街日报上个月观察到，“中国公司华为正在将自己嵌入到有线系统中，该系统可以传输几乎所有的全球互联网数据。”
  </p>
  <p class="graf graf--p">
   在美国官方限制联邦机构使用华为的5G设备以防止中国黑客入侵的情况下，由华为构建的水下互联网基础设施可能是最敏感的“全球后门”。
  </p>
  <p class="graf graf--p">
   该报告说：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    当然，尽管美国和其他国家表示中国法律有义务将网络数据交给政府，华为否认对其正在建设的电缆项目进行任何操纵……但美国如何解决海底电缆的安全问题？仅仅基于怀疑没有办法阻止华为建立它们，或者与中国公司签订合同对其进行现代化改造。相反，美国必须利用其网络和情报收集能力来收集后门和其他安全风险的确凿证据。这将具有挑战性 — 中国企业技术复杂，与虚拟警察国家交织在一起
   </em>
   ……
  </p>
  <p class="graf graf--p">
   报告描述的一个更离奇的情况是潜艇可能从外部“窃听”电缆。长期以来，军方和情报分析人员仅就俄罗斯进行了讨论，但现在越来越多的是担心北京。
  </p>
  <p class="graf graf--p">
   其中一位分析师，美国海军上将杰米·福格戈（Jamie Foggo）被确定为职业潜艇艇员，他告诉Bloomberg Quint：“水下电缆是我们关键基础设施的一部分，对全球经济至关重要。美国必须保护他们的完整性和安全，就像我们提供公海的国际自由一样。“
  </p>
