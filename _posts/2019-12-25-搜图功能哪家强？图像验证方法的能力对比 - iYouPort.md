  <p class="graf graf--p">
   信息验证过程中对图像和视频的验证是一个重点。因为图像和视频是作假的主要渠道。调查中也是如此，图像经常包含明显的情报线索。
  </p>
  <p class="graf graf--p">
   我们介绍过一些验证视觉信息的简单方法和工具，也演示过很多验证和调查的过程，您可以在“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    开源情报
   </a>
   ”栏目中找到。
  </p>
  <p class="graf graf--p">
   但是对于初学者来说，工具越多似乎越难以选择。
  </p>
  <p class="graf graf--p">
   于是，本文将简要比较一些
   <strong class="markup--strong markup--p-strong">
    主要图像搜索引擎
   </strong>
   的反向图像查找功能 ——
   <strong class="markup--strong markup--p-strong">
    您将会发现，什么才是最好的，要取决于您的调查目标是什么，想要什么。因为这些工具各有千秋。
   </strong>
  </p>
  <p class="graf graf--p">
   熟悉它们的特征将更好的帮助您在应用方面获得准确的判断。
  </p>
  <p class="graf graf--p">
   本文不讨论图像分析的任何更复杂的方面，而是纯粹专注于您上传图像进行反转查询时的期望。关于图像情报的分析，可以参见我们的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    开源情报栏目
   </a>
   中提供的演示。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     反向图像搜索
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在安全性研究中，你会处理各种类型的数据：IOC、恶意软件二进制文件、报告等等；一种可能经常被忽略的数据类型是图像，例如 论坛帖子上的个人资料图片、恶意软件广告中的照片等等。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    通过以各种方式搜索这些图像，可以了解有关调查主题的很多信息
   </strong>
   。
  </p>
  <p class="graf graf--p">
   对于某些调查，使用常规搜索引擎（例如 Google 或 Bing）进行反向图像搜索可能就足够了。
  </p>
  <p class="graf graf--p">
   例如，如果您遇到不熟悉的logo，则简单的反向图像搜索就可能会返回有关logo图像来源的信息，例如关于品牌、甚至是位置的信息。
  </p>
  <p class="graf graf--p">
   再比如，在调查过程中遇到正在调查的恶意软件作者使用的头像。该头像似乎是特定的，但你并不熟悉它，更多地了解该头像可以帮助进一步了解该恶意软件的作者。
  </p>

   在这种情况下，上传的图片非常独特，足以让你知道
   <strong class="markup--strong markup--p-strong">
    它是与俄罗斯犯罪组织紧密相关的图片，这有助于缩小调查对象的范围
   </strong>
   —— 可能是俄罗斯人（或者至少对俄罗斯很着迷） 。
  </p>
  <p class="graf graf--p">
   有时 Google 可能无法返回足够具体的结果；在这种情况下，尝试使用 Yandex、Bing 或 TinEye 进行反向图像搜索也会很有用。
  </p>
  <p class="graf graf--p">
   有时，当尝试进行反向图像搜索时，
   <strong class="markup--strong markup--p-strong">
    以某种方式更改原始图像才更容易找到最佳结果
   </strong>
   。
  </p>
  <p class="graf graf--p">
   比如，有人发布了一张图像，并声称是原始图像，但实际上只是现有照片的翻转/反转版本。通过翻转照片然后进行搜索，您可能找到更准确的搜索结果。
  </p>
  <p class="graf graf--p">
   仔细裁剪还可能会产生更好的结果，因为照片中的其他对象可能会导致搜索引擎将注意力集中在错误的主题上。
  </p>
  <p class="graf graf--p">
   尽管图像和照片并不是研究人员在研究中发现的最常见的信息，但是
   <strong class="markup--strong markup--p-strong">
    当您尝试识别 logo 或头像、对照片进行地理定位、或识别一个目标个人时
   </strong>
   ，这些知识就会特别重要。希望本指南对您有所帮助。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     反向图像搜索平台的比较
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   用于反向图像搜索的最佳网站：以下全部都是。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9542 jetpack-lazy-image" data-lazy-sizes="(max-width: 900px) 100vw, 900px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-1.png?resize=900%2C343&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-1.png?w=900&amp;ssl=1 900w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-1.png?resize=300%2C114&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-1.png?resize=768%2C293&amp;ssl=1 768w" data-recalc-dims="1" height="343" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/4-1.png?resize=900%2C343&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="900"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Google Images
   </strong>
  </p>
  <p class="graf graf--p">
   谷歌的特点是试图识别图像的主题，而非图像本身。
  </p>
  <p class="graf graf--p">
   结果通常分为三个部分：算法认为是照片的搜索结果，视觉上相似（但不完全相同）的结果，以及包含相同图像的页面。
  </p>
  <p class="graf graf--p">
   在这三个部分上 Google 是强大的反向搜索资源。
  </p>
  <p class="graf graf--p">
   但是，不利的一面是，许多匹配项都侧重于图像中的主题，因此，如果图像搜索的主题不太清楚，您可能不会获得高质量的结果。
  </p>
  <p class="graf graf--p">
   该搜索也会把自己限制在一个主题上，因此，假设输入的是山和湖附近房屋的照片，您将不会看到有位于山湖附近的房屋的照片，而是获得视觉上相似的房屋，它不会考虑湖泊或山的背景。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Yandex Images
   </strong>
  </p>
  <p class="graf graf--p">
   与 Google 类似的 Yandex 是用于反向图像搜索的金矿。
  </p>
  <p class="graf graf--p">
   开源调查的培训经常会提及
   <strong class="markup--strong markup--p-strong">
    “Yandex 比谷歌更好”
   </strong>
   ，这是因为 Yandex 可以适合很多调查相关的情况。
  </p>
  <p class="graf graf--p">
   它的特点是提供相同图像的其他尺寸，视觉上相似的图像，以及页面上具有相似图像的许多结果。
  </p>
  <p class="graf graf--p">
   Yandex 往往是
   <strong class="markup--strong markup--p-strong">
    最适合人脸匹配和位置识别的搜索引擎
   </strong>
   ，正因此它适合开源情报调查。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Bing Images
   </strong>
  </p>
  <p class="graf graf--p">
   Bing 具有我真正喜欢的独特功能 —— 您可以在其中裁剪照片的区域并查看实时结果。
  </p>
  <p class="graf graf--p">
   这对于
   <strong class="markup--strong markup--p-strong">
    具有多个可识别主题
   </strong>
   的高质量图像来说会非常有用。
  </p>
  <p class="graf graf--p">
   此外，与 Google 相比，Bing 尝试识别照片中的元素并查找包含所有这些元素的图像。
  </p>
  <p class="graf graf--p">
   因此，一张停在树旁的古董车的图片会触发包含一棵树和一辆古董车的相关匹配，而 Google 在这种情况下会选择其中一个强烈的主题并寻找匹配。
  </p>
  <p class="graf graf--p">
   Bing 的与众不同之处还在于，它可以
   <strong class="markup--strong markup--p-strong">
    主动识别图像中的脸部、产品和其他元素
   </strong>
   。有多个显要主题的高清图像将被突出显示每个主题。显然，这样一来您不会错过东西。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    TinEye
   </strong>
  </p>
  <p class="graf graf--p">
   TinEye 最初创建唯一目的是寻找同一图像的其他尺寸，多年来，它一直提供这一功能。
  </p>
  <p class="graf graf--p">
   我认为他们在过去一年左右的时间内启用了匹配算法，现在可以做到寻找视觉上相似的图像匹配结果。
  </p>
  <p class="graf graf--p">
   这意味着您可能会在这里找到您提供的图像在其他图像中存在的状况，如果是照片，则可能会找到构图完全相同的其他照片。
  </p>
  <p class="graf graf--p">
   我想在此指出，TinEye不应该被视为这些其他搜索引擎的直接竞争对手。
  </p>
  <p class="graf graf--p">
   TinEye 的功能对于处理纯数字媒体（头像、徽标、按钮等）的情况是赢家。比如您的调查正在尝试
   <strong class="markup--strong markup--p-strong">
    追踪一个特殊的头像
   </strong>
   ，它就会很有用，即便您的调查目标在不同平台上使用了不同的ID，也容易找到。
  </p>

  </p>

  </p>

   <strong class="markup--strong markup--p-strong">
    5、HackForums
   </strong>
  </p>
  <p class="graf graf--p">
   Google 在此失败了，它只认为图片是“黑白”的。结果是随机主题的所有黑白图像。
  </p>
  <p class="graf graf--p">
   Yandex 的表现同样糟糕，但至少找到了同类主题的东西。
  </p>
  <p class="graf graf--p">
   Bing 的效果也很惨，它找到一大堆乱七八糟的照片。
  </p>
  <p class="graf graf--p">
   TinEye 相比之下简直完美，它得到了指向正确方向的多个结果。
  </p>

     总结

  <p class="graf graf--p">
   正如前文中所强调的那样，每个搜索引擎都有其优点和缺点。
  </p>
  <p class="graf graf--p">
   还请记住，这些搜索引擎都无法深度索引 Instagram、Twitter、Facebook 和其他社交媒体，也就是说，您不会以这种方式获得OSINT超级能力。
  </p>
