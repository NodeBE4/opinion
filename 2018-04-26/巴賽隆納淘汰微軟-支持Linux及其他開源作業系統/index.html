    巴賽隆納淘汰微軟，支持Linux及其他開源作業系統

    2018-04-26

    原文刊載於 P2P Foundation
    原文標題：City of Barcelona Kicks Out Microsoft in Favor of Linux and Open Source

    譯：李佳霖

    來自巴賽隆納的大消息。本文原先於ItsFoss.com網站發布。

    根據西班牙報社 El País 報導，巴賽隆納當局正逐步將開源技術移轉到政府部門的電腦系統中，首先預計以替代的開源應用程式取代所有的目前的應用程式，直到僅剩Windows這一項專有軟體，最終再用Linux取代Windows。

    巴賽隆納將在2019年春天全面實現開源

    在下年度的規劃中，巴賽隆納政府投入70％的軟體預算在開源軟體。巴賽隆納市議會科技與數位創新委員Francesca Bria表示，轉換工作會在2019年春天現任行政首長任期結束前完成。

    致力扶持在地IT人才的移轉計畫

    為完成該項計畫，巴賽隆納政府將開始向在地中小型企業外包IT項目，並延攬65位新的開發者來針對特殊需求開發軟體。其中一項主要發想是數位市場的開發——一個讓小型企業參與公共標案的線上平台。

    Linux發行版首選Ubuntu

    因巴賽隆納當局執行中的前測計畫已使用1000台Ubuntu桌面，Ubuntu會是最可能被選用的Linux發行版。報導中還提到，Outlook信箱以及微軟Exchange Server信箱系統將由Open-Xchange替代，Firefox和LibreOffice則將取代Internet Explorer與Microsoft Office。

    巴賽隆納成為首個加入「Public Money, Public Code」運動的行政區

    此舉讓巴賽隆納成為首個加入「 Public Money, Public Code 」運動的行政區。在一封呼籲用公帑開發的軟體應該免費釋出的 公開信 出現之後， 歐洲自由軟體基金會 發起了該項運動，已有超過15,000位個人及100多個組織響應。只要參與 連署請願 ，表達你對開源軟體的看法，你也可以支持這項運動。

    資金永遠是關鍵

    巴賽隆納市議會科技與數位創新委員Francesca Bria表示，從Windows轉換到開源軟體的運動提倡再利用，其中一種方式是將這些已開發的程式，應用到其他西班牙的行政區或世界上其他地方，由此可見，這項移轉計畫亦致力避免在購買專有軟體上花費大量資金。

    你怎看？

    這是一場已經獲勝、有利於開源社群的戰役，在 慕尼黑市政府決定回頭重採Microsoft系統 的同時，巴賽隆納此舉更顯必要。你怎麼看巴賽隆納政府採用開源軟體的政策？你是否預期其他歐洲城市會跟進？歡迎在留言區和我們分享。
