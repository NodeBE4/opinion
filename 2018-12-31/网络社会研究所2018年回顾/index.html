    6月10日
    合作松2 杭州拍脑会
    “拍脑”取自 “panel（论坛）”之音，我们邀请了南塘兴农合作社、永济蒲韩联合社、成都爱生活家社区、三生谷生态村、德清三林村滋农有限公司，以及几位杭州的程序员，一同参与“在地拍脑会”，为9月的合作松2活动热身，汇集各合作社的共同需求。
    除了杭州的“在地拍脑会”，黄孙权教授在台北、高雄、香港还召集了4场拍脑会。参会的团体达30多个。 </br>
    台北、高雄、香港拍脑会[/caption]
    |
    | </br>
    9月26 – 9月27日
    合作松2：透过合作社落地的技术 </br>

    主办： 中国美术学院跨媒体艺术学院网络社会研究所、台湾当代文化实验场、香港中文大学、平台合作义联盟、co-lab
    社区协力：Segmentfault
    活动召集人：黄孙权 李士杰 邱林川
    亚洲首次平台合作主义联盟（PCC）大会在香港中文大学举行，网络社会研究所所长黄孙权提议于会前两日举办一场名为“透过合作社落地的技术 – 合作松2”的黑客松：回应平台合作，回应中港台三地合作社的具体需求，为参与大会，致力于合作运动的人们献上一个携手同行的节日。
    |
    |
    9月28 – 9月29日
    平台合作主义联盟（PCC）大会

    亚洲地区实践者与世界各地行动者的首次齐聚，讨论亚洲各国的合作社状况，交流成功案例，商讨数字平台经济所面临的问题。安徽阜阳南塘兴农合作社、台湾司马库斯劳动合作社、台湾食物银行联合会分别分享第一手的组织经验，网络社会研究所所长黄孙权也在大会中发表主题演讲，报告亚洲合作社概况与合作松2成果。

    |
    |
    11月2日 – 11月13日
    网络社会学系列课程 #1
    此系列课程是为了建立网络社会学研究之基础，也是面对未来”平台社会”的理论实践之预习。培养同学参与第三次网络社会年会”智慧城网（IUF）”讨论的理论准备，并能够深化发展淘宝村初期研究与撰写。
    |
    |
    11月21日 – 11月23日
    第三届网络社会年会 “智慧都市网络”
    会议总召：黄孙权，李士杰
    学术筹备委员：高士明，管怀宾，黄孙权，李士杰，陆兴华，许煜
    主办：
    中国美术学院文化创新与视觉传播研究院
    视觉中国协同创新中心
    中国美术学院跨媒体艺术学院网络社会研究所
    第三届年会“智慧都市网络”邀请智能城市专家、国内外学者与各地市民技术组织，在地方政府-企业领导的都市化与自下而上的市民科技组织的张力中，编织构造一个能够允诺政治对话框架的数字都市未来。11月21日-22日為主題講演，11月23日由Dyne.org带来“物联网开关与社会钱包（DOWSE/ SWAPI）”工作坊。
